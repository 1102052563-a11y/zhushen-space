# 功能使用指南（小白版）

这份指南讲 4 个功能：**公共频道、ROLL 点（骰子）、私信、向量资料库**。
每个都告诉你：**是什么 → 在哪打开 → 怎么用 → 没反应怎么办**。照着点就行，不用懂代码。

> **共同前提**：这几个功能（除了纯看帖）都要让 AI 干活，所以**得先在「系统设置」里配好 API**（接口地址 + Key + 模型）。频道和私信默认就用你的**主 API**，不用另配；向量库要单独配一个"embedding 接口"（见第四节）。

---

## 一、公共频道 📡 —— AI 扮演的「玩家广场」

### 这是什么
一个热闹的契约者公共广场。里面**不是真人**，是 AI 扮演的一群虚拟玩家在发帖、卖东西、组队、聊天。你可以围观、买卖、发言、交朋友、组队。

### 在哪打开
右侧导航栏的 **📡 频道**。

### 七个频道（顶部切换）
综合 / 交易 / 组队 / 战斗 / 世界 / 情报 / 系统 —— 点不同标签看不同内容。

### 你能做的事
1. **看帖 / 刷新**：一打开就有帖子。想看新的点 **🔄**（手动刷新）。
2. **买东西**：在「交易」频道，看到别人的出售帖，点 **「购买」** 直接买（自动扣钱、东西进你背包）。
3. **卖 / 求购**：自己挂个单 → AI 扮演的人来**报价** → 你点「接受」成交。
4. **系统商店 🏪**（频道里的「🏪 系统商店」按钮）：
   - **买**：AI 一次上 20 件货，你挑着批量买。
   - **卖**：把你背包里能卖的东西，AI 一次性报价，批量卖掉。
5. **发言**：频道底部有输入框，打字发送。**你的话立刻上墙**，然后 AI 扮演的人会**陆续回你**。看到别人的帖想专门回，点帖子上的 **「↩ 回复」**。
6. **加好友 ⭐**：每个帖子上有 **「⭐ 加好友」**。加完会自动给这个人**生成一份档案**（处于"离场"状态），之后他会参与 NPC 演化（即"活"在你的世界里）。
7. **私信 ✉**：每个帖子上有 **「✉ 私信」**，单独找这个人聊（详见第三节）。
8. **组队**：
   - 看到组队帖，点 **「🤝 加入队伍」** 直接入队。
   - 任意帖子点 **「➕ 邀请入队」**，写句邀请词，**AI 会判断对方答不答应**。
   - **换世界时临时队伍自动解散**。
   - 如果你已经有"冒险团"，解散时可以把临时队友**转正**进冒险团。

### 怎么开 / 设置
**设置 → 变量管理 → 📡 公共频道**：这里有总开关、选用哪个 API、导入预设。

### 没帖子 / 没反应？
① 确认 API 配好了；② 点 **🔄** 刷新；③ 确认在设置里把频道功能**打开**了。

## 二、ROLL 点（摇骰子判定）🎲 —— 给行动加点运气

### 这是什么
当你做一件**不一定成功**的事（撬锁、说服别人、偷袭、跳悬崖…），用**骰子 + 你的属性**算出"成功到什么程度"。这样剧情有变数，而不是 AI 随口决定成败。

### 在哪打开
玩的时候点右侧导航 **🎲**；设置在 **变量管理 → 🎲 ROLL点设置**。

### 怎么用（玩的时候）
1. 打开骰子面板，**写你要做的事**（例：撬开这个宝箱）。
2. **选属性**（力量 / 敏捷 / 体质 / 智力 / 魅力 / 幸运）+ **选难度**（简单 / 普通 / 困难 / 极难 / 几乎不可能）。
   - 不知道选啥？点 **✨**（AI 建议属性和难度）。
3. **摇骰子**（d20 或 百分骰）。
4. 系统会**自动把你的技能、天赋、装备加成**，还有**和对手（在场 NPC）的实力差**都算进去。
5. 出结果（大成功 → 碾压 → … → 大失败）→ **自动填进你的输入框** → 你确认后发送给 AI 续写。

### 两种判定模式（设置里切）
- **前端确定性**：系统按数值公平计算，稳定。
- **AI 裁判**：让 AI 结合剧情来判（带骰子锚定、失败有兜底），更灵活。

### 小贴士
检定**草稿会自动保留**——你关掉页面再打开、或刷新网页，上次没发完的检定还在。

## 三、私信 ✉ —— 和某个人单独聊 + 私下交易

### 这是什么
不在公共广场，而是**一对一**找某个人聊天，还能**私下交易**：买、卖、给、要、换。

### 在哪打开
右侧导航 **✉ 私信**；或者在**频道帖子**上、**📇 NPC 列表**里点 **「✉ 私信」** 直接进。

### 能私信谁
**契约者 / 随从 / 宠物 / 没有标签的人** 都行；**土著、召唤物 不行**。

### 你能做的事
1. **聊天**：打字发，对方会**按他的人设 / 档案**回你。
2. **四种交易**（对话里的按钮）：
   - 💰 **购买**：想买他手上的东西。
   - 🤲 **给予 / 出售**：把你的东西送他 / 卖给他。
   - 🙏 **索取**：跟他要东西。
   - 🔄 **换物**：拿东西换东西。
3. AI 会**报价** → 你可以**讨价还价**（他重新报一个）→ 满意了点成交。

### 东西最后去哪了
- 你**买的 / 要来的 / 换来的** → 进**你的背包**。
- 你**给的 / 卖的 / 换出去的** → 进**对方的储存空间**。

### 没档案的人也能聊
频道里那些还没建档的发帖人，私信照样能聊，还能**一键给他生成一份完整档案**。

## 四、向量资料库 📚 —— 把原著当"活字典"喂给 AI

### 这是什么
把**《轮回乐园》原著小说 + 世界书**做成 AI 能"语义搜索"的资料库。玩的时候**每回合自动**找出和当前剧情最相关的原著片段，**偷偷塞给 AI 参考**，让剧情更贴原著、设定更准。**不用你手动写关键词**，全自动（和"叙事回忆"类似）。

### ⚠️ 这个比前三个麻烦一点：要先"建库"
建库是**一次性**的，建好就一劳永逸。需要在电脑上敲两行命令（在 `zhushen-space/zhushen-space` 这个文件夹里）：

```
$env:EMBED_KEY="你的硅基流动 key"
npm run build-vectors        ← 建小说库
npm run build-vectors-wb     ← 建世界书库
```
（两个建一个也行；建完会在 `public/` 下生成文件，跟着部署一起上线。）

### 然后在 app 里设置
**设置 → 变量管理 → 📚 向量资料库**：
1. 打开 **「启用向量资料库」**。
2. 填 **Embedding 接口**（就是查询时用的模型）：
   - Base：硅基流动 `https://api.siliconflow.cn/v1`
   - Key：你的硅基流动 key
   - 模型：`Pro/BAAI/bge-m3`　**（必须和建库时同一个模型，否则搜不到）**
3. 点 **「加载/检查索引」**，看到 **「✓ 已就绪：xxxx 段」** 就成功了。
4. 参数（可调）：
   - **topK**：每回合塞几段，默认 5。
   - **相似度阈值**：越高越严（只塞很相关的），默认 0.35。
   - **注入字数上限**：默认 2500，**怕费 token 就调小**（这是硬闸）。

### 怎么用
**设置好就不用管了**——玩的时候它自动检索、自动注入。想看看效果，点 **「🔍 测试检索」** 输一句话，看它找到哪些片段。

### 注意
- 查询那一下也会调用一点 embedding（很便宜）；不想用随时关掉开关。
- 正文里会标来源，比如 `〔原著·第X章〕`、`〔世界书·猎杀者〕`。

## 一句话总览

| 功能 | 在哪开 | 一句话 |
|---|---|---|
| 📡 公共频道 | 右侧 📡 | AI 扮演的玩家广场：看帖 / 买卖 / 发言 / 交友 / 组队 |
| 🎲 ROLL 点 | 右侧 🎲 | 给行动摇骰子，算成功程度，结果自动填进输入框 |
| ✉ 私信 | 右侧 ✉ | 单独找一个人聊天 + 私下买卖给要换 |
| 📚 向量资料库 | 设置→变量管理→📚 | 把原著喂给 AI，每回合自动找相关片段，贴原著 |

> 这几个功能都会消耗 API 额度（向量库费得很少）。觉得费，就把不用的关掉、或把向量库的"注入字数上限"调小。
