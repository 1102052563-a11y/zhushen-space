# 暗鸦星 · 暗鸦纪元 · 势力

> 本页汇总 **暗鸦星 · 暗鸦纪元** 的登场人物归属。

## 🏛 势力详解

### 暗鸦星（暗鸦纪元·原生世界）

- **圣愈教会**（二十一卷8–14章）— [暗鸦星](../世界/任务世界/暗鸦星-暗鸦纪元.md) 上**猎杀女巫的组织**、中 / 西部的实际管制者（总部圣愈城·大陆地价最高最安全的城）。900 年前由被女巫残害的平民创立——曾以「内讧散伙」为掩护隐忍：176 名成员带资金 + 研究成果隐姓埋名、强掳健康女性大量产子、把子嗣**只培养成猎杀女巫的工具**（黑铁堡）。两代人放弃亲情娱乐、信念疯狂却真改变了世界——女巫不再敢大肆抓人做活体实验、「神灵流血便无人再惧」。「**巫猎人**」之名令女巫色变（用金刚石 / 古怪道具削弱女巫·5 名巫猎人集结一般女巫只能逃）。**入会**只需身份证明、但**当巫猎人至少要亲手猎杀一名黑女巫**。[苏晓](../人物/苏晓.md) 赴圣愈城找「死亡之地」线索、并欲弄个巫猎人身份方便行事。
    - **双权分立**（二十一卷18章）：① **武力部门**——驻总部、由 **圣公会**（[卡门](../人物/卡门.md) 等 **12 名圣议员**）裁定、直管所有巫猎人；② **管理部门**——圣愈城官员、由 **教皇**（本质是国王·选举制）管军队 / 财政 / 官员。教皇高于单名圣议员、却**不得指派巫猎人**（仅圣公会能）；12 圣议员集结可罢免教皇。权 / 财 / 王权 / 神秘侧分得清清楚楚、故巫猎人间无内斗。
    - **巫猎人三级**：**白级（兔子）< 黑级（夜犬·女巫防不胜防）< 红级（血猎人·猎杀 ≥20 女巫·可查图书馆机密）**。设施：图书馆（二层）、阵营商店（四层·需黑级）、武器锻造工坊（制式武器免费 + 多肯斯兽车 / 战略级地图）。福利好（巫猎人击杀女巫得 50% 赏金 + 武器保养 / 食宿 / 工资全包·无穷困潦倒的巫猎人）。苏晓加入后成白级、拉 [美洛狄](../人物/美洛狄.md) 当助手开「猎巫之旅」。
- **西港（西湾）**（二十一卷28–31章）— 圣愈教会**西部边境的女巫聚集点 / 钉子**（三面环山峭壁数万陷阱 + 一面环海 12 瞭望塔的天然要塞）。统治者 [摩利莎](../人物/摩利莎.md)（五阶「巫女王」）**圈养人类**（买奴隶世代腐蚀精神、奉其为活神·圣愈教会领地内 80% 流窜女巫出自此）。圣愈教会鞭长莫及（心腹大患是南部「女巫之国」）。[美洛狄](../人物/美洛狄.md) 的老家。
- **牙骨部落**（二十一卷29–30章·**已被黑渊来客灭族**）— 暗鸦大陆**东部彪悍食人土著**（上千战士·35 岁即献身后代分食「与我同在」·身体素质惊人）。曾吃掉圣愈教会两万讨伐军、却敬重安葬战死的巫猎人、后与教皇签互不侵犯约。七名女巫去抢东西反被其不死不休追杀团灭；最后整族被那名虚空黝黑人形（精神凝结体）一夜屠灭。
- **三大奴隶庄园**（二十一卷12章）— **恶魔牙 / 里可 / 巴巴厄**承包暗鸦大陆 70% 的奴隶生意；恶魔牙最彪悍（**连女巫都敢卖**·武力机构生猛、多数王国对其打怵）。许多平民走投无路自卖为奴换一顿饱饭。
- **奥山王国**（二十一卷8–11章·**国王已被斩**）— 新成立 20 年的小王国（国名取自国王老家地名）、领主 **伦道夫**（粗鲁有野心·爱称「领主 / 大人」胜过「国王」）。庇护逃来的焰爆女巫 [伊萨](../人物/伊萨.md)，被杀进王宫的 [苏晓](../人物/苏晓.md) 斩杀，军权将领趁机「先王私通女巫」上位夺权。

### 大陆格局

> 出自世界页《暗鸦星 · 暗鸦纪元》。

- **中 / 西部**：[圣愈教会](index.md) 管制、法度相对完善（苏晓落点的西部常年雨季湿冷）。
- **东部**：土著势力居多、几无法律、**女巫最多的「无法之地」**。
- **北境**：冰原冻土、游猎民风彪悍、女巫少（太冷·女巫会被冻死）。**南部**：女巫地盘、人类禁区。
- **科技**：暗鸦纪元 1433 年·蒸汽机 / 火车 / 电（仅权贵用）·武器只钻研大威力霰弹枪「喷子」（金刚石子弹·因敌人是女巫·无坦克火炮）。
- **奴隶制**：恶魔牙 / 里可 / 巴巴厄三大奴隶庄园承包大陆 70% 奴隶生意（恶魔牙最彪悍·连女巫都敢卖）。许多平民走投无路自卖为奴换一顿饱饭。
