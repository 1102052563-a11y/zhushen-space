## 🏛 势力详解

### 🗺 四大区域

> 出自世界页《永光世界·厄难囚笼》。

⚠ **本世界四大区域:湿盐区、荒芜沙漠、蚀雨地、庇护城**——⚠ **三个都有格外危险的环境**。⚠ **昼光是本世界最基础的危险环境之一。**

- **荒芜沙漠**（苏晓的落点）：⚠ **外围区域只需有庇护石**；⚠ **如果是内区或中心区，庇护石也不能保证一定安全，还需要有足够多的生命力硬抗**——⚠ **也就是说，生命值上限越高，越有深入荒芜沙漠的资格**。⚠ [索恩斯](../人物/永光世界/索恩斯.md) 的警告:「**千万别去**，边缘区还好，⚠ **等你深入到一定程度，你会被拖入到炼狱噩梦**。」
- ⚠ **它其实是「沙之海」**:苏晓单手捧起一把黄沙，⚠ **除了热感外，他竟感到这黄沙的湿润感**——⚠ **当黄沙从指缝间淌没后，他手上竟沾上些水液**。⚠ **这很不妙**:如果是沙漠，用了庇护石徒步前行即可；⚠ **可如果是沙之海，那就需要以船渡海了**——⚠ **而且在这平静的沙漠之下，天知道会有什么可怕的生物**。⚠ **表面只有一层薄沙，到了十几公分以下深度水沙就稀释了很多，超过一米深后完全稀薄到和水液一样**；⚠ **随着温度持续升高，沙之海的真正样貌展现:除了表面的一层薄沙外，下面完全化为水液。**
- **湿盐区**：⚠ **在这里水很宝贵，比灵魂石更受欢迎**——⚠ **但别被这诱惑到，能别去尽量别去**；⚠ **非去不可的话，多搞些纯盐，那是好东西。**
- **蚀雨地**：⚠ **最后一座「监视者高塔」就在这里的深处**（老树族:「那些高塔被毁了不少，**唯一有可能剩下的只可能是蚀雨地的深处了**」）；⚠ 暗之女给的**灭法宝库地图**也指向这片区域。
- **庇护城**：⚠ **永光世界唯一的安稳之地**——详见下文。

### 🎟 200 个名额 · 一场被四大乐园集体拒绝的邀请

> 出自世界页《永光世界·厄难囚笼》。

⚠ 因苏晓与永光世界的**独特关联**，他可二选一:①**世界眷恋（被动）**——⚠ **你会受到永光世界的格外关注（高收益 / 高风险）**；②**邀请（权限）**。⚠ **苏晓当即无视选择 1**:⚠ **他一点都不希望永光世界眷恋自己**——⚠ **单是应对那些灭世级存在与深渊滋生物就够难顶，倘若被永光世界格外关注了，那基本距离暴毙就不远了**；⚠ **再者说，永光世界还有不少「老朋友」:蛀世、银皇后、苦痛女王、寄星蟹、食暗者。**

- ⚠ **他先想指定邀请 [神父](../人物/神父.md)**——⚠ 【提示：**你所邀请的目标，将在本次世界进度进入永光世界，无法进行邀请**】。⚠ **看到这提示，苏晓略感诧异；随即，他知道「君主宝藏」那件事是谁筹谋的了。**
- ⚠ **改为随机邀请** → ⚠ **将在天启 / 死亡 / 圣光 / 圣域四乐园随机强制邀请 80～100 名契约者、战斗天使、死亡游侠、处刑者等** → ⚠ **【警告：天启乐园已终止本次强制邀请】【警告：死亡乐园已终止本次强制邀请】【警告：圣光乐园已终止本次强制邀请】【警告：圣域乐园已终止本次强制邀请】**——⚠ **四大乐园全部当场拒绝**。
- ⚠ **轮回乐园向虚空之树发布公证审核** → ⚠ **符合初始条例 185357 条的世界开启规格** → ⚠ **本次世界进度名额上限提升至 200 个**；⚠ **但因所进入永光世界过于危险，本次所有进入者，除了灭法之影·库库林·白夜外，所有进入者的初始传送点均为「庇护城」**。⚠ **苏晓的进入位置为随机选定。**
- ⚠ **【提示（轮回乐园）：干涉中……】**——⚠ **他的进入位置依旧随机，但可拟定一个从者名额，把它的初始地点设定在「庇护城」，以此作为方向坐标**。⚠ 苏晓**没过多思索就选定贝妮**:⚠ **贝妮去永光世界有巨大劣势（没有满世界旅行的条件），并且贝妮擅长交涉——先让它在「庇护城」打探情况是最好的选择。**
- ⚠ **实际到场 225 人**（还有十几人没被传送进来，⚠ **算上那十几人，这次差不多 240 名**）——⚠ **狠人兄、黑魔、神父、所罗门、魔镰·泰莉德、噩鬼·凯因、雷法神·艾格等人全部都来了**，⚠ **一些在九阶巅峰成名许久的人也都出现在列表上**。
- ⚠ **他们真的是为「君主宝藏」来的吗？不然**:⚠ **这些人很可能是为了用【铁炼邀请函】才来本世界**——⚠ **或者说，是趁这次有更多人分担风险**；⚠ **原本还在犹豫是否要用【铁炼邀请函】的九阶强者，这次都不再犹豫**。⚠ **没有比现在更适合用【铁炼邀请函】的时刻。** ⚠ 随后的提示验证了这猜想:⚠ **【本世界内已有 20 名以上契约者、猎杀者、战斗天使等使用「铁炼邀请函」，因成功收回此等数量的「铁炼邀请函」，本次生存排行榜首位～第五名的奖励幅度将提升 50%】。**
- ⚠ **苏晓不仅没丝毫担心，反而心情很不错**:⚠ **来本世界进行试炼考核的人越多越好**——⚠ **试炼任务这东西有着绝对的个人特性**:⚠ **假设所罗门的试炼任务·第一环是寻找「亡灵心核」，那苏晓的试炼任务就绝对不可能与这「亡灵心核」有半点关系。**

### ⚠ 潜伏的第三方

> 出自世界页《永光世界·厄难囚笼》。

⚠ **眼下的局面，看似是『好队友』四人组 VS 无光神殿·四巨头**——⚠ **实际上情况比这更复杂**;⚠ **因为在这两方之间，还有个伪装成 [神殿祭祀长·厄兹勒](../人物/永光世界/厄兹勒.md) 的 [神父](../人物/神父.md)。**

- ⚠ **⚠ 而且苏晓身上已经被 [灵妖](../人物/灵妖.md) 布设了一个坐标**——⚠ **神父可以实时监测他的位置**;⚠ **⚠ 但苏晓也因「血之恶意」，能在 24 小时内感知到灵妖的方位。**
