# 势力

收录冒险团、组织与各方势力（含任务世界内的国家 / 宗门 / 团体）。

## 海贼王世界

### 哥亚王国及下属
- **哥亚王国** — 东海国家，国王 菲利普·赫伯特；下辖城门守卫队、侍卫队等，首都为王都（第2–9章）。为不让来巡视的天龙人看到城外垃圾，决意焚烧废品终点站、连同数千流民（第17、20章）。
- **哥亚王国侍卫队** — 王都治安部门；风气腐败、职位可用贝利买卖。首领由 [布兰登·欧卡](../人物/布兰登·欧卡.md) 经 [苏晓](../人物/苏晓.md)（白夜）取代（第7–17章）。

### 海贼团（焚烧废品终点站的候选）
哥亚王国欲在三伙海贼中选一伙执行焚烧、事成许以贵族身份（第18章）：

- **克利克海贼团** — 东海有名海贼团，约 600 人，团长 [克利克](../人物/克利克.md)。**被苏晓选中并利用**（第18–20章）。
- **布尔萨姆海贼团** — 哥亚王国本土海贼，实力一般；原著中焚烧废品终点站的执行者，此处因苏晓改选克利克而未被选用（第6、18章）。
- **骷骨海贼团** — 实力低微，被苏晓直接排除（第18章）。

## 东京喰种世界
- **CCG（喰种搜查官）** — 人类对抗喰种的官方组织、民众心中的「保护神」；以**库因克**（喰种赫包所制）作战，搜查官多为身体优异、靠技巧与智慧的普通人。设 **Rc 细胞检测门**辨喰种。搜查官共 **6 级**：三等 < 二等 < 一等 < 上等 < 准特等 < **特等**（特等仅 7 位）。契约者加入后有「**阵营贡献值**」——驱逐喰种积累、满则升职，亦可用于内部购物 / 制库因克。苏晓加入的一方（二卷2、7–9、13章）。
    - **零队** — 全 CCG 唯一敢深入 [24 区](../世界/地点/index.md)（人类禁区）的精锐小队，队长为「不败的搜查官」[有马贵将](../人物/有马贵将.md)（二卷37章）。
    - **特别对策班** — 为应对青铜树攻陷 11 区而组建的特别编队，由战地指挥官 [丸手斋](../人物/丸手斋.md) 统筹（二卷37章）。
    - **作战装备**：**CRc 瓦斯**（白雾·达浓度后压制喰种 Rc 活性、使赫包萎缩、放不出赫子）、榴弹发射器（CCG 武器库却不愿多供榴弹，理由牵强、苏晓存疑）、手雷（CCG 唯一允许的爆炸物，高爆手雷需层层申请）（二卷35–36章）。
    - **苏晓战绩**：八天近乎肃清整个 14 区，喰种圈称其「**14 区的黑色死神**」（黑发黑眼、从不留活口）（二卷36章）。
    - **高层黑幕**：CCG 由 **和修家**把持——局长 和修吉时、其上为总议长 [和修常吉](../人物/和修常吉.md)；二人皆属神秘组织 **V**，极度冷酷（视搜查官为可消耗的棋子）。CCG 强者中「有些身份不光彩」（暗示非人 / 喰种）（二卷70章）。
- **V（CCG 真相）** — 凌驾并**统领 CCG** 的喰种组织，**CCG 高层（和修家族）全是喰种**（二卷80–81章）：
    - **CCG 不是保护人类、而是「圈养」人类**：喰种不会彻底消灭喰种。CCG 散播「热武器对喰种无效」的谎言、自喰种搜查官学校起洗脑搜查官，逼人类只用**库因克**（喰种武器）作战——实则重机枪 / 重狙 / 坦克足以轻易屠喰。
    - 搜查官如「**清道夫**」、用库因克少量控制喰种数量；潜伏人类中的喰种则靠进食控制人类数量——双方在 V 操控下维持**畸形平衡**（喰种有食、人类不灭绝、文明不崩）。
    - 人类在 CCG 最高只能做到**特等**；察觉真相者多被灭口。青铜树 vs CCG 实为**喰种内战**（青铜树聚众占区＝挑战 V 权威）。
    - **V 喰种不会被 Rc 细胞检测门检出**（门是 CCG 所造、留有缺陷）。豢养「**清扫人**」（如 [芳村功善](../人物/芳村功善.md)）排除异己；掌握 24 区「源」与原始种真相。
- **白日庭**（待考）— 高槻泉口中 CCG 疑似培养战力 / 试验品的组织（误以为苏晓出自此处）；原文尚未展开（二卷79章）。
- **棋盘规则（人喰默契）**：青铜树虽备有大量热武器（重机枪 / RPG / 中美德产军火），却**不敢用**——几年前用过一次即遭 V / 军方小型导弹轰击；「我们不用大规模热武器、CCG 也就不用」成了双方默契的棋盘规则（二卷91章）。
- **青铜树**（青桐树）— 强大的喰种组织，首领即独眼之王 [高槻泉（枭）](../人物/高槻泉.md)；成员有多多良、野吕、雾岛绚都等，曾出动抢夺神代利世；以西街区共喰区为「兵源」招揽强喰种，因苏晓肃清而放弃该处。又**公开袭杀 11 区喰种搜查官、攻陷 11 区 CCG 分部**、使 11 区沦为喰种领地；并抓走 [金木研](../人物/金木研.md) 折磨、推其蜕变（二卷5–6、16、34、38章）。
- **月山家族** — 喰种名门，在政界、商界皆有势力、人脉广、资源丰厚（少爷月山习）；拍卖场上一名戴小丑面具、用尾赫的月山家族喰种主持人欲以身份与十亿日元免死，仍被苏晓斩杀（二卷22–23章）。
- **安定区咖啡厅** — 20 区一群「和谐派」喰种（不主动捕食、只食自杀者尸体），店长 [芳村功善](../人物/芳村功善.md) 实为 **SSS 级独眼之枭**、青铜树首领 [高槻泉](../人物/高槻泉.md) 之父（即另一头「枭」）；成员有雾岛董香、西尾锦、笛口雏实、入见萱（黑色杜宾）等。11 区作战中店长着青铜树服饰参战、为救金木研拖住 CCG（二卷40、58–60章）。
- **小丑** — 神秘喰种组织，在青铜树与 CCG 之间「打酱油」、坐收渔利。已知成员：宗太（首领·最大反派，据原著后期移植利世赫包后实力顶尖）、尼克、[伊鸟](../人物/伊鸟.md)、萝玛、呗——除宗太外均 SS 级、平时藏匿实力。亦是主线可选阵营之一（二卷2、18章）。

## 契约者冒险团
- **彼岸花冒险团** — 一支 Lv.6 的契约者永久冒险团；已知成员 [女王](../人物/女王.md)（团长·肉盾）、[黑白](../人物/黑白.md)（狙击）、[叶子](../人物/叶子.md)（奶妈），「盾 + 狙 + 奶」配置。三人持豁免卷潜入东京喰种伏击苏晓，反被全歼；黑白临终撂话「彼岸花冒险团，不会放过你」，疑为后续伏笔（二卷19–31章）。

## 进击的巨人世界（原著民势力）
- **王族 / 希娜之墙贵族** — 居最内层希娜之墙，把持墙内权力；罗塞之墙被破后，下令宪兵团死守希娜之墙护王族、并牵制调查兵团（三卷10章）。
- **达里斯·扎卡里（总司令）** — 墙内**所有兵团的最高统帅**，须发皆白、通情达理、为民着想；内心厌恶王族、知晓墙内部分真相却无可奈何。亲临希娜之墙主持「临时法庭」、裁定艾伦的归属（三卷16章）。
- **宪兵团** — 团长**奈尔·德克**（王族的狗腿子，权势超部分贵族）；守卫王族、清理墙内「不稳定因素」（先进学者 / 自由思想教派），持落后枪械、不敢与巨人拼命；曾纵容下属作恶、毁尸灭迹（苏晓的「前身」即因杀这三名宪兵被判死刑）（三卷10、16章）。
- **调查兵团** — 团长**艾尔文**；致力墙外调查、常年与巨人作战、死亡率极高，是真正能直面巨人的力量。兵长 [利威尔](../人物/利威尔.md)、第二分队长研究狂 [韩吉·佐耶](../人物/韩吉·佐耶.md) 等。艾尔文以「对人类有巨大贡献者可戴罪立功」为由，恢复苏晓「前身」的调查兵团身份（三卷5、10、16章）。其墙内地位低、生活清苦（总部老旧、伙食粗陋）。
    - **调查兵团的真相（苏晓推断·三卷36章）**：初代王成立调查兵团并非真为探索墙外（他早知墙外情形），而是给被囚墙内的平民一个「人类仍在努力探索、外面太危险」的**虚假希望**，以免其绝望反抗。墙内小岛实则广阔，调查兵团至今未抵达海洋；一旦触及真相，恐遭王族灭口。
- **训练兵团（104 期）** — [艾伦](../人物/艾伦.md)、[三笠·阿克曼](../人物/三笠.md)、阿尔敏所属；同期还潜伏着伪装新兵的巨人间谍 [阿尼](../人物/阿尼.md)（女巨人）、[莱纳·布朗](../人物/莱纳·布朗.md)（铠之巨人）、[贝特霍尔德·胡佛](../人物/贝特霍尔德·胡佛.md)（超大型巨人）等。艾尔文据「城墙两次被破的时间恰与 104 期相关」推断三名「智慧巨人」就藏身于该期成员（三卷4、14、37章）。
- **艾尔迪亚帝国 / 马莱帝国（世界史）** — 远古大陆两国：战争孤儿尤弥尔获原始巨人之力助艾尔迪亚灭马莱、建千年帝国；后马莱以离间计夺七巨人之力、墙内初代王（始祖第 145 代）带众逃岛筑三墙。详见 [生物·万族 · 巨人起源](../世界/生物万族/index.md)。如今**马莱帝国**为墙外强国、墙内人（**艾尔迪亚人**）的敌对方，派 [阿尼](../人物/阿尼.md)（女巨人）、[莱纳·布朗](../人物/莱纳·布朗.md)（铠之巨人·**副战士长**）、[贝特霍尔德·胡佛](../人物/贝特霍尔德·胡佛.md)（超大型）、贝利库（颚之巨人·已故）潜入墙内，欲**破坏神之壁、灭绝墙内人类、夺取始祖巨人之力**及岛上资源；另有野兽巨人 **吉克** 留在墙外、久未联络。马莱以「**战士**」编制管理巨人继承者（编号如「TK·317」、莱纳曾任副战士长），科技已发展出飞艇、大炮，正与「**中东联盟**」交战、资源日趋紧缺，故急于夺岛。苏晓先诈称「马莱人」骗过 [阿尼](../人物/阿尼.md)，又编造「实验战士 TK·317」身份骗取 [莱纳·布朗](../人物/莱纳·布朗.md)、[贝特霍尔德·胡佛](../人物/贝特霍尔德·胡佛.md) 信任，反向操控间谍组改图抓走艾伦（三卷22、26、27、38章）。
    - **中东联盟** — 马莱帝国故乡的一股强大势力，正与马莱交战中（三卷38章·提及）。

## 跨世界组织
- **黑帆商会（黑商）** — 一群不要命的**职工者商人**，专在衍生世界内高价兜售装备 / 物品 / 补给（也接受以物易物）。穿**绝对护甲**（6 点·被动挨揍、不能攻击，用以证明身份）；铁规：**杀害其成员者将被永久拒绝交易、并在乐园悬赏通缉**（曾以 18 万乐园币通缉一名 Lv.9 契约者）。苏晓以法师猩红卡 + 1000 币向其换得立体机动精通卷轴（三卷12章）。

## 契约者冒险团（永久）
- **彼岸花冒险团** — **全女性**永久冒险团（入团似有颜值门槛），团长 [冷月](../人物/冷月.md)、副团长 [血玫瑰](../人物/血玫瑰.md)、智囊「腹黑女（小黑）」、狙击手蜂（黑白徒弟）、奶妈糖果（接吻治疗）、女汉子主坦等约五十余人；专属团队技能「**血色追印**」（见 [机制](../总览/轮回乐园机制.md)）。为给被苏晓所杀的女王 / 黑白 / 叶子报仇，在进击的巨人世界与苏晓死战、几近团灭、求和被拒（三卷28–33章）。[女王](../人物/女王.md) / [叶子](../人物/叶子.md) / [黑白](../人物/黑白.md) 亦属此团（东京喰种伏击苏晓被全歼，黑白曾放话「彼岸花不会放过你」）。进击的巨人世界中防守一区（三卷2章）。
- **兄弟会冒险团** — 数十人永久冒险团，团长 [亚当](../人物/index.md)（高魅力法师·口碑佳）；入团即兄弟、同生共死、不收心怀叵测者。成员有法师 [尘](../人物/尘.md)、正与 Rc 细胞融合（克制吃人欲）的「烈」等——亚当给烈的「两样东西」即苏晓卖出的初代 Rc 原液 + 原始吸收。防守二区（二卷101、三卷2、8章）。
- **幻影旅团冒险团**（提及）— 以 PVP 见长的可怕冒险团，有一名用**左轮枪**的男人（出枪极快、子弹破盾、偶附真实伤害），曾重创彼岸花。腹黑女将苏晓（用刀的 PVP 强者）与其类比（三卷8章）。

> **PVP / PVE 之分**：多数契约者走 PVE（强于打衍生世界生物）；少数如幻影旅团左轮男、苏晓走 **PVP**（杀契约者比杀世界生物更强）。本世界为**和平模式**——杀死契约者无法获得猩红卡，故契约者间敌对意愿较低（三卷8章）。

> 由契约者 / 职工者组成的「**冒险团**」（临时 / 永久）属契约者玩法，见 [机制 · 冒险团](../总览/轮回乐园机制.md)；乐园内的固定大型组织尚未在已读章节出现。
