## 🏛 势力详解

### 四大势力

> 出自世界页《幽暗大陆·死寂根源之地》。

- **治愈教会**:主掌神系（圣痕之力·治愈灾祸·信奉古老神灵、重度敌对古神）——四大势力中最低调也最狠。下辖三机构:**工坊**（对外武力 + 武器开发）、**学术派**（肉体 / 心理治疗 + 圣痕研究 + 育才）、**治疗院**（「根治」= 处决冥顽罪徒·苏晓所在）。顶层为大主教·卢索、圣祭祀·安托莎（常年不出大教堂·心机深沉）。
- **蒸汽神教**:「血肉苦弱，机械飞升」——城内一切科技造物（蒸汽列车 / 电力 / 黑科技械体）的来源，最高综合战力。领袖 [机械公爵](../人物/幽暗大陆/机械公爵.md)。旗下「怒锤机构」一直想取代治疗院处理超凡案件。
- **瓦迪家族**:农耕 / 畜牧技术的掌握者——没它城内五分之一的人吃不上饭。综合战力最末但贡献巨大。
- **高墙议会**:高墙城官方机构（税收 / 财政 / 民生）·掌宪兵队与治安队。
