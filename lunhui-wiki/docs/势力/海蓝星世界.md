# 海蓝星世界 · 势力

## 🏛 势力详解

### 海蓝星 · 世界争夺战（第十九卷）

三方乐园 + 海蓝星两大国的对峙（详见 [海蓝星世界争夺战](../世界/任务世界/海蓝星-世界争夺战.md)）：

- **三方乐园（十九卷1–3章）** — **轮回乐园**（己方·崇尚力量自由·惩罚最狠·疯子高产地·组织 = 冒险团）vs **天启乐园**（老对手·惩罚温和·80% 正常人 / 妹子软萌·数量庞大·组织 = 各类组织）+ **圣域乐园**（全新对手·崇尚信仰·金黑长袍「神棍」·哪个教派强信哪个·团结爆表·组织 = 教派）。这次是 **1V2**。
- **格雅联盟（自由之国 · 己方阵营）** — 海蓝星两大国之一·多人种·民风彪悍铁血·唯才是举·科技军事碾压·名声「拥兵自重血腥野蛮」。首都**热都**。设**杀戮部队**（特殊情况应急与清理战略部队·敌后暗杀 / 情报 / 爆破·直属大总统府·苏晓所属·新任团长 [斯卡因](../人物/斯卡因.md)）、情报组织**洛丝祇之眼**（外交部高层莱宗任首领·辖热都及周边·毒美人莎琳等特务）。
- **太阳王国（敌方阵营）** — 王权 × 神权制·单人种（蓝眼）·重出身·法度严苛·科技被格雅按着打。原信太阳神**阿瑟坦勒**、改信**晨曦女神**、首都改名晨光城。**神庭** = 其武力机构（六级：修司 / 大修司〔尉〕、主教 / 晨光主教〔校〕、神使 / 大神使〔将·大神使无武力却有恐怖指挥才能〕）。**圣域乐园**疑藏身此国。**十九卷26章·权力崩盘**：掌军权的**大神使**被怒川团长 [青魇](../人物/青魇.md) 逼得清晨饮弹自尽——这位无战力却统帅能力不可替代的最高军方统帅一死，太阳王国军方瞬间动荡、前线一夜溃败、退守圣利门要塞。神庭与王权曾 200 年前内斗（神庭灭王族未遂、招致格雅大军打穿），此后改为「相亲相爱的共生」。
- **怒川冒险团（四阶大型冒险团 · 十九卷2–3章）** — 与神皇、血门齐名的四阶大型冒险团（硬实力略逊二者）·海蓝星战区轮回乐园方半数契约者属此团。团长 **[青魇](../人物/青魇.md)**（搞死旧莽夫团长上位·**四阶第一老阴哔**·走谋略路线令怒川急速崛起）；成员有**鬼夫人**（细烟杆 / 黑唇膏 / 迷香）、**人民教师**（无框眼镜 / 面瘫 / 强如怪物 · 暂代团长出面）等。**十九卷26–27章**：青魇全程不露面、带一名女性魅力契约者深入太阳王国、**逼其大神使饮弹自尽、几天搞垮太阳王国权力体系**（致前线一夜溃败）；他真正图谋是**团结轮回乐园方那群各行其是的疯子**（别有目的）。同战区轮回乐园独行侠还有 [咕噜](../人物/咕噜.md)（幻影旅团·绑圣域契约者勒索）、[所罗门](../人物/所罗门.md)（法爷炮台·杀进中型国）、**老烟**（拉 3000+ 起义军·起义对象竟含太阳王国 + 怒川自己·当先锋）等。
