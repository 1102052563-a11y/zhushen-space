# 海贼王世界 · 势力

> 本页汇总 **海贼王世界** 的登场人物归属。

## 🏛 势力详解

### 海贼王世界

### 哥亚王国及下属
- **哥亚王国** — 东海国家，国王 菲利普·赫伯特；下辖城门守卫队、侍卫队等，首都为王都（第2–9章）。为不让来巡视的天龙人看到城外垃圾，决意焚烧废品终点站、连同数千流民（第17、20章）。
- **哥亚王国侍卫队** — 王都治安部门；风气腐败、职位可用贝利买卖。首领由 [布兰登·欧卡](../人物/布兰登·欧卡.md) 经 [苏晓](../人物/苏晓.md)（白夜）取代（第7–17章）。

### 海贼团（焚烧废品终点站的候选）
哥亚王国欲在三伙海贼中选一伙执行焚烧、事成许以贵族身份（第18章）：

- **克利克海贼团** — 东海有名海贼团，约 600 人，团长 [克利克](../人物/克利克.md)。**被苏晓选中并利用**（第18–20章）。
- **布尔萨姆海贼团** — 哥亚王国本土海贼，实力一般；原著中焚烧废品终点站的执行者，此处因苏晓改选克利克而未被选用（第6、18章）。
- **骷骨海贼团** — 实力低微，被苏晓直接排除（第18章）。

### 七水之都（第六卷再入）
- **饿狼海贼团**（六卷2–4章）— 中小型海贼团，三桅快船「恶狼号」（船帆被铅弹打成筛子、副帆杆断、几近散架）；八天前与他团炮战逃脱、海上漂流，淡水 / 水果耗尽、伤口溃烂、已沦落到**吃重伤同伴**。残暴的**独眼船长**（被割过喉·靠暴力统治·遇人假称「破浪海贼团」行骗）看中苏晓的斩龙闪、欲图财——被苏晓抛肉骨头引海贼自相残杀后绕背一刀割喉**斩杀**；余下海贼反水（早恨船长）、与苏晓「井水不犯河水」、负责驾船送他往七水之都。
- **卡雷拉公司**（六卷5–6章）— 七水之都**最大势力**，名为造船厂、实则掌控全城且深得民心（连海军都难插手）；是**世界政府的附属公司、但自治**（御用造船厂）。社长兼市长 **[冰山](../人物/index.md)**（艾斯巴古·鱼人汤姆的徒弟·做帮海贼造修船的违法生意 + 担任合法市长 + 闷声发财·世界政府刻意保他不死）。**鱼龙混杂、不查员工来历**（海贼在此打工是常事），故是三阵营中**最易加入**的——苏晓即以「东海·国王护卫·白夜」身份受聘一号船坞安保入职（管事 [安格斯](../人物/index.md)）。但也**最危险**：内部潜伏 **CP9**（世界政府直属特工）——卡雷拉公司内三人 [罗布·路奇](../人物/index.md) / [卡莉法](../人物/index.md) / 卡库，城里另有第 4 人 [布鲁诺](../人物/index.md)（开酒馆·门门果实·里应外合），**会暗杀加入卡雷拉的强契约者**（已有契约者因此被灭口）。冰山要求安保「**不许杀人**」（低调）；员工的**知名度**靠看场、击退海贼、社交传闻积累（满 400 可在七水之都自由行动）。苏晓**反劫持卡莉法**（界断线 + 强喂炸弹）当筹码、令 CP9 投鼠忌器，又借「与卡莉法暧昧」的传闻把知名度刷到近 400、并勒索到六式「剃」（六卷9–12章）。
- **世界政府 / 海军 / 草帽海贼团**（七水之都三阵营之二）：世界政府统领世界 800 余年、海军是其武力机构（七水之都仅驻十几名海军·主负责紧急联络）；草帽海贼团则可能聚集 90% 以上的契约者。苏晓因魅力低、相性不合而避开此二者。
- **世界政府 · CP 部门（Cipher Pol）**（六卷17–29章·苏晓**转投此阵营**）：世界政府直属的情报机构，**CP1~CP8** 公开收集情报；**CP9 是「本不应存在的第 9 个 CP 部门」**——握有暗杀特权、身份不对外公开。**CP9 四人组**（[罗布·路奇](../人物/index.md) / [卡莉法](../人物/index.md) / 卡库 / [布鲁诺](../人物/index.md)·主力多为失双亲、经残酷训练淘汰出的孤儿）潜伏七水之都 **5 年**、本为窃取**古代兵器「冥王」的设计图纸**（图纸是古代文字·世上能识者屈指可数·故世界政府放任其流落、近年才动手·图纸其实在 [弗兰奇](../人物/index.md) 身上而非冰山）；三天前接死命令——抓捕藏匿于草帽海贼团、**能读古代文字的 [妮可·罗宾](../人物/index.md)**（得图纸 + 擒罗宾即有复活冥王之望）。名义长官 [斯潘达姆](../人物/index.md) 靠父辈人脉上位、定计划拖沓（被苏晓视为「猪队长」）。
    - **CP9 的果实能力者阵容**：斯潘达姆为 CP9 在世界政府争取了不少恶魔果实——八人中有五人是果实能力者（路奇·猫猫果实豹形、卡库·牛牛果实长颈鹿形、布鲁诺·门门果实、加布拉·犬犬果实狼形、卡莉法·泡泡果实）。**司法岛决战损兵**：六卷61–62章，CP9 的**音无猫头鹰**与弗兰奇交手落败、被苏晓补刀（殁）；**雄狮偎取**被契约者**左轮男**（幻影旅团·见下文）独立击杀（殁）。
    - **CP 高层「Zero」**：电话虫里冷气森森、能压制同级长官**斯潘达姆**的神秘人物。**看中苏晓的炼金炸弹**（隐秘 + 威力 + 远程操控·有量产战略价值），破例招他为 **CP0 正式成员（科研）**——约定每 2000 颗炸弹换一把良快刀（仅收藏权）、并喂他一颗「慢性剧毒胶囊」表忠诚（苏晓**假吞**、实则收进储蓄空间）。苏晓借此一跃成世界政府珍贵人才、CP9 四人反成他的保镖。**真相**（六卷47章·苏晓揭穿）：**Zero 其实是海军安插的「幽灵」**——招募苏晓后从不露面、把他从 CP0 倒手交给海军（与贝加庞克同理），是海军与 CP 部门暗中博弈的一招。详见 [苏晓 · 阵营转变](../人物/苏晓.md)。
- **海军 / 世界政府高层**（六卷38–47章）：世界政府的武力部门、与 CP（情报部门）各有掌权者、彼此博弈（CP 不许海军大量登司法岛）。出场要人：元帅 **[战国](../人物/index.md)**、海军英雄 **[卡普](../人物/index.md)**（路飞祖父）、大将 **[黄猿](../人物/index.md)**（光速）/ **[青雉](../人物/index.md)**（卡普旧部）、中将参谋长 **[鹤](../人物/index.md)**、上校 **[缇娜](../人物/index.md)**（槛槛果实·看守苏晓）、天才博士 **[贝加庞克](../人物/index.md)**。海军把苏晓从 CP0 接到秘密基地荒岛量产炸弹、并以「维持毒囊不破的药」企图控制他（苏晓没真吞毒、表面求解药实则不在乎）。**十八卷41章·艾斯连锁 + 顶上战争伏线**：元帅 **战国** 与海军英雄 **卡普** 密议——**波特卡斯·D·艾斯被捕**牵动全局：艾斯既是白胡子二番队长、又是卡普的孙子、更是**海贼王罗杰的亲生子**（卡普还有个亲儿子是革命军首领龙）。战国派卡普去新世界、放话「只要白胡子倒下、新世界就不再是海贼肆意之地」——顶上战争伏线张开（艾斯近期将在推进城被处死、世界政府要借此大做文章）。海军 / 海贼都察觉到「足以颠覆时代的大事」将至。本·贝克曼口中海军的「**白鲸炸弹**」（海贼别称·船底被贴炸弹引爆如被鲸鱼撞沉）即苏晓造的**特级炼金炸弹**。**十八卷51–55章·围捕苏晓**：世界政府要活捉栖身唐吉诃德家族的苏晓（觊觎阿波罗的制造方法），派出**大将 [黄猿](../人物/黄猿·波鲁萨利诺.md) + 海军英雄卡普 + 大参谋鹤中将 + 五艘军舰 + 千余海军**——这套「围捕四皇级」的阵容由卡普的副官指挥（黄猿在港口压住多弗朗明哥）。「夕阳红组合」实力恐怖：**卡普**（78 岁·没吃恶魔果实·纯体术 + 近顶级武装色 + 变态的速度 / 力量·一拳轰碎苏晓 700 强度能量盾）、**鹤中将**（74 岁·肥皂泡系果实·曾把未成七武海的多弗追得满世界逃·她的百名女子精英部队）、黄猿（光速）。苏晓**假降**（用 [纳米微生物防护服](../力量体系/装备道具/index.md) 伪装成「空间系果实能力者」），被铐海楼石镣具、押上军舰——目的地竟是**司法岛**（非玛丽乔亚 / 推进城·先经红土大陆海军分部中转）；海军视多弗朗明哥为「铲除不掉的毒瘤」。
- **天龙人 / 世界政府权力体系（十八卷48章·重大世界观）** — **天龙人 = 世界贵族**，是 800 年前组建世界政府的「二十位王」的后裔，受所有世界政府部门保护；多数天龙人无实权、只管吃喝玩乐（作用 = 吸引敌对火力 + 延续二十王血脉、危机时「废物利用」）。**对天龙人出手（击伤 / 绑架）海军必派大将增援**。世界政府权力呈金字塔，真正掌权者**并非五老星**，而是常居玛丽乔亚最深处「花之间」的 **伊姆**（既管天龙人、又管五老星）。顺序：**伊姆 > 五老星 > 全军总帅·空 > 佛之战国〔海军元帅〕> 三大将**；**天龙人**自成独立体系（受伊姆管辖、受各部门保护）；**CP 部门**受五老星 / 伊姆管辖（「半个天龙人的狗腿子」）。苏晓的四阶晋升任务「世界之敌」正是**击杀两名天龙人（查尔罗斯圣 / 罗兹瓦德圣父子）**= 与整个世界政府体系为敌（见 [德雷斯罗萨海战](../世界/任务世界/海贼王-德雷斯罗萨海战.md)）。**十八卷75–76章·伊格纳兹家族灭门**：苏晓借草帽团 + 凯米被绑把三名天龙人引进唐吉诃德拍卖会，**三枪狙杀查尔罗斯圣 / 罗兹瓦德圣 / 夏露莉雅宫**（父子女·同属 **伊格纳兹家族**嫡系）——一家死的整整齐齐、直接**导致二十王后裔伊格纳兹家族嫡系血脉断绝**（世界之源 +15%）；代价是与世界政府锁定「死敌」、与海军锁定「仇敌」。
- **CP0（统领 CP 部门 · 十八卷76章）** — 凌驾 CP1～CP9 之上的最高谍报机关，**天龙人的尖牙与坚盾**、传言强者实力不弱于海军大将。杀天龙人后世界政府不再像原著只派黄猿镇压、而是出动 CP0 三人组（含五米巨汉 [埃迪·克](../人物/埃迪·克.md)〔妮可·罗宾的童年阴影〕）月步渡海追苏晓——却追不上他装了 300 马力马达的改造渔船、被水下逃脱甩脱。
- **十一超新星（十八卷74章）** — 同期汇聚香波地群岛的十一名新生代海贼，已到五人：尤斯塔斯·基德、死亡外科医生·特拉法尔加·罗、海鸣·阿普、杀戮武士·基拉、饕餮女·乔艾莉·波妮（这些海贼比契约者低调得多·怕海军找上门·天龙人一死基德 / 罗当即开溜）。
- **梧桐树冒险团（五阶 · 十八卷48章·苏晓推断）** — [枯屠](../人物/枯屠.md)（疑为团长）/ [浅梦](../人物/浅梦.md) / 蛮石等同属的五阶冒险团。德岛海战的 6 名五阶契约者疑大多是该团成员——他们**用特殊手段进入了「阶位高于自身」的海贼世界**，代价是受限：**不能击杀影响巨大的剧情人物**（如大妈 / 凯多·否则新世界格局崩溃），故只与苏晓在海贼声望上较劲、不来德岛直接宰他。这解释了「为何强如五阶的他们不顺手杀掉大妈」。
- **四皇（新世界·十八卷2章）** — 新世界的四大掌控者（海军都不敢轻易涉足新世界）：**白胡子·爱德华·纽盖特**（最强·视船员为珍宝）、**百兽·凯多**（生存变态·爱好自杀）、**红发·香克斯**（大海般胸怀）、**夏洛特·玲玲**（能生·子嗣强者众多·痴迷甜食）。**玲玲与凯多年轻时同属 Rox 海贼团**（罗杰时代前的海上霸主）。此时（顶上战争一月前）白胡子尚在世、艾斯刚被捕，三皇借机蠢动抢地盘——详见 [德雷斯罗萨海战](../世界/任务世界/海贼王-德雷斯罗萨海战.md)。
- **唐吉诃德家族（七武海·十八卷·苏晓所选阵营）** — 首领 **[多弗朗明哥](../人物/多弗朗明哥.md)**（七武海·Joker·原天龙人·黑暗世界 BOSS·掌新世界 70% 军火 / 80% 奴隶生意）。下设四部门：**梅花**（托雷波尔·黏黏果实·管治安）/ **黑桃**（琵卡·石石果实·家族前三·多弗朗明哥护卫·苏晓落在其「黑桃号」）/ **方块**（迪亚曼蒂·最老成员·管斗牛竞技场）/ **红心**（情报·维尔戈〔潜伏海军高位〕/ 莫奈）。苏晓为避 CP 部门、选这个三方最弱势的阵营落脚。
- **革命军（二十五卷·顶上战场·苏晓五阶再入所选阵营）** — 由 **[蒙奇·D·龙](../人物/顶上战场配角.md)**（路飞之父 / 卡普之子 / 世界政府眼中「最凶恶的罪犯」）领导的**起义军**，目标推翻世界政府（故世界政府恨他远超海贼）。骨干：[克尔拉](../人物/顶上战场配角.md)（萨博弟子·情报 / 潜入）、萨博、鱼人空手道教官哈库（培养新血）。总部在新世界一座未列入海图的白色岛屿。[苏晓](../人物/苏晓.md) 五阶再入海贼时选革命军为参战阵营（图最大自由度·不指望武力支援·却能在顶上战场激活四个主线任务）——并用「替革命军调配 [圣痕药剂 / 黎明之焰](../力量体系/消耗品材料/index.md)」白嫖其稀有材料刷自己的炼金熟练度。详见 [顶上战场](../世界/任务世界/海贼王-顶上战场.md)。
- **BIG·MOM 海贼团 / 百兽海贼团（十八卷·德岛海战交战方）** — **BIG·MOM**（玲玲·进攻方·夺人造果实技术·老巢「[万国](../世界/地点/index.md)」）。**甜点四将星 = 除玲玲外最强的四人**（无明确排行·实力有强弱）；德岛只来两将星：[夏洛特·卡塔库栗](../人物/夏洛特·卡塔库栗.md)（**四将星之首**·年糕果实觉醒 + 见闻色**预知未来**·近 5 米）+ [克力架](../人物/夏洛特·克力架.md)（饼干果实 / 饼干士兵·8.6 亿·实力远逊卡塔但战争价值高·怕痛），另两将星防百兽；长子 **佩罗斯佩罗**（舔舔果实·近乎无限造糖·已被苏晓溺杀）。**百兽海贼团**（凯多）派**三灾 [旱灾·杰克](../人物/旱灾·杰克.md)**（铁颚 / 鱼人血统 / 双弯刀 / 战争狂「所到之处寸草不生」）助阵多弗朗明哥（看重人造果实·实则等多弗被打残后吞并）。BIG·MOM 断后的还有夏洛特家族**第 3 女阿曼德**（名刀「白鱼」+ 慢速「缓慢叙事曲」开颅·剑术高超残忍）。**四皇人脉对比**（十八卷33–35章）：红发人脉最广（不抢地盘、庇护各国）；白胡子地盘最广（威名使平民安居）；大妈也有长手族 / 三眼族等盟友；**唯独凯多盟友只有多弗朗明哥一人**（凯多是「狂躁症 / 爱好自杀」之徒、激怒他会亲自杀来灭团·故多弗作其盟友始终提心吊胆、却也靠凯多的资金 / 渠道 / 名声谋利）。**人造恶魔果实**的原料供应商 **凯撒·库朗**（造 SAD 原料·向唐吉诃德 / 百兽 / 大妈三方都骗经费花天酒地、还偷研人体巨大化·因凯多需要他而动他不得）。

- **暗影团（黑耗子 · 四阶刺客冒险团 · 十八卷22–23章）** — 一个由**七名刺客类能力者**组成的四阶冒险团（外人称「黑耗子」、自称暗影团）。九人在一阶时因同是刺客而结识组团、最多扩到 30 人、大浪淘沙剩 7 人（两个世界前晋四阶）；着装统一、步伐一致，**七人同时爆发足以秒掉同阶主坦**，专挑落单的剧情人物 / 独行契约者下手。德岛海战中盯上声望榜第三的苏晓（杀他可得其声望的 1/2 ≈ 1000 多点）、用影分身吸引火力突进围杀（成员有会影分身者）。另有「**送死小分队**」等数十名四阶契约者觊觎苏晓 / 重伤的克力架——但精明者（如铁链干瘦男）一眼看穿「三阶进四阶战区还刷这么多声望、必是全面发展的独行侠」而怂退。**结果（十八卷24章）**：苏晓一出别墅就用龙影闪秒掉生存力最强、探路的「影一」（只抗两刀），七人惊退；他靠别墅四周布的简易脆响触发装置 + 直感（斩断的几缕头发在感知圈里「如萤火虫」）逼出屏息潜伏的「**影三**」——影三的隐匿是花大代价买的全职猎人念能力「**神不在场的证明**」（限潜行 + 不动 + 不呼吸·被队友当弃子抛弃），现身搏命喊「给我五秒」即被斩首。苏晓共斩 2 名暗影团（约 200 声望·无猩红卡），其余 5 名隐身系一心逃则难追——苏晓由此吸取教训「**对隐身小队不能主动出击、该静候**」。
