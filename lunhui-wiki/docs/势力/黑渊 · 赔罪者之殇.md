# 黑渊 · 赔罪者之殇 · 势力

> 本页汇总 **黑渊 · 赔罪者之殇** 的登场人物归属。

## 🏛 势力详解

### 虚空种族（二十三卷·黑渊·赔罪者之殇）

> 虚空里至少 1000+ 种族博弈不知多少年，霸者如奥术永恒星 / 恶魔族。详见各人物页与 [黑渊浅层 · 赔罪者之殇](../世界/任务世界/黑渊浅层-赔罪者之殇.md)。

- **恶魔族**（虚空大种族 · 灭法之影世代盟友）— 居 [破碎大陆 · 梵落之城](../世界/地点/index.md)（418 万恶魔·守玛门莫德要塞·掌黑渊一处入口当摇钱树）。青黑皮 / 弯角 / 羊蹄 / 火焰眸；男性多为傲气战士、女性魅魔不滥媚（崇敬「强大」与「种族延续」、富有但少与外族来往）。**毒抗 + 恶魔之力 + 恶魔之焰**是种族天赋。极看重 [黑枫树](../力量体系/消耗品材料/index.md) 产物（造「恶魔之力中枢」的主材·近绝种）——故与持黑枫树的 [苏晓](../人物/苏晓.md) 力促长期交易。代表：大恶魔 [沃波尔](../人物/黑渊配角.md)、统帅 [尤利塞斯](../人物/黑渊配角.md)、魅魔外交官 [莉莉姆](../人物/莉莉姆.md)、战士 [莉莉斯](../人物/莉莉斯.md)、列车收票的 [恶魔老太婆](../人物/黑渊配角.md)。
- **羽族**（虚空种族）— 与恶魔族是宿敌（争 [灵魂果实](../世界/生物万族/index.md) 等·派战力相近者互耗、谁输都不好报复的老阴哔默契）。俊男靓女、生水晶羽片。代表：冷峻的 [残羽](../人物/黑渊配角.md)、心智老辣的小公主 [羽鸦](../人物/黑渊配角.md)。

### 赎罪者高地三势力（黑渊·二十三卷9章）

黑渊「无法地带」赎罪者高地（比梵落之城更大）的三股统治势力（互相敌视 / 无视 / 鄙视）：

- **黑泽**（外围 · 人数最多）— 拾荒人的升级版（武器更先进 / 更狡诈 / 有地盘）·掌水源、发行货币「司」·平民窟 80% 居民挂名其下。如一群鬣狗。
- **阿萨卡（中庭 · 杀手团体）** — 古老的暗杀组织·**无总部、无内部管理**（连成员都不知总部在哪·靠「终端」接暗杀 / 寻物 / 护卫等委托·完成得报酬·不怕内奸）。只收人族。[苏晓](../人物/苏晓.md) 经杀手 [枭](../人物/枭.md) 引荐加入、以此扬名引布布汪、捞资源。
- **始祖**（中心区域 · 最神秘 · 人数最少）— 与近千年的 [灵魂藤蔓](../世界/生物万族/index.md) 共生·麾下大批植物「同化者」·首领 [圣扎迦](../人物/圣扎迦.md)（灵魂亡者·被苏晓斩杀）。
