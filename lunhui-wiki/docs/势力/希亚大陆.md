## 🏛 势力详解

### 希亚大陆（原生世界·原著民势力）

（详见 [希亚大陆·灭法遗迹](../世界/任务世界/希亚大陆-灭法遗迹.md)·十七卷3–7章）

- **圣索丁王国** — 希亚大陆上与**死灵帝国**交界、**常年战争**的人类王国（**希亚世界唯一的人类王国**），国都为**圣城（王都）**。由**五个王国的领土拼合而成**、土地富饶，人口约 **3~5 亿**（已读章节·科技约中世纪欧洲水准、农耕却领先数百年——有野草级耐活的「克巴」薯类、懂耕犁与嫁接，故能养庞大人口、与死灵战千年）。**侦探是热门职业**（地位低于官员、高于商人·王族承认的大侦探可被征用）。王宫是带巨大庭院的城堡（国王 + 众王后居住 + 重臣办公一体·占地不大、不尚奢华）。王座由天降陨石锻造（漆黑）、王冠「**铁王冠**」由初代国王 **铁冠王·阿德莱德·赫伯特** 传承千年。**选新王 = 双梯队制**（防昏君）：第一梯队是老国王约 **49 名子嗣**（15 岁后外放各地管领地·有封地财权却无大军·老国王将死时召回圣城**争位**·可暗杀 / 下毒 / 夺权、不撕破脸、最终留在圣城者继位）；第二梯队是圣城培养、全国选拔的数十名继承人（非血脉·胜出赐姓「赫伯特」·近选举制）。
    - **老国王 [巴伊·赫伯特](../人物/巴伊·赫伯特.md)**（65 岁·命不久矣·45 年前杀光至亲含父王而登位）——主线「权利之冠」发布者。
    - **四心腹**（财政 / 军权 / 官员 / 极端手段·缺一不可、**平时被刻意安排互有「小矛盾」、绝不能联合**否则架空国王；**权力争夺一开打则改为碰杯示团结**帮老国王镇场）：
        - **罗文·哈维**（财务官·执掌财政·白衣金丝老头·爱调侃同僚）——在王宫故意撞苏晓肩演戏、反被真实力 89 的苏晓撞到扭腰；肃清夜用老鼠传密语「有我吗」探得自己安全，事后负责清点四名被杀官员的家财（不动产占八成·累到没睡好）。**🆕真身 = 死灵族安插数十年的间谍（十七卷23章揭）**：他一步步爬上掌财政的「王国财务官」高位、正是死灵弑君大局的一环；弑君失败后**于中午秘密逃出王都、功成身退**（由伊儿朵护送出王宫·经非士峡谷 / 苏普哈海域 / 阿昂纽河流域 / 努巴哈沙漠·已抵神之墙外）。**他更是混血中的异类（十七卷36章）**：**2/3 死灵血脉 + 1/3 人类血脉、却毫无死灵特征、外观 100% 人类**（有史第一例·2 岁觉醒死灵之力惊动死灵高层、被定向培养出财政天赋）。他在位期间财政**毫无纰漏**——这才最可怕（潜伏数十年必埋了定时炸弹）；他一走，王国财政成了「定时炸弹」（最伤前线）。
        - **鬣狗·赫士列特**（王国军团长·掌兵权·黑衣脸带伤疤·性格豪爽）——公开支持**三王子**（手握军权、谁也动不得他）；认得叛国圣战士 [约瑟·柯](../人物/约瑟·柯.md)（旧日战友·闻其死讯真心痛惜）。
        - **乔·瓦伦丁**（执政官·管大部分官员·瘦弱中年·智商顶尖却**惧内**·不敢轻易站队）——**实为忠臣**：老国王临终以「我撑不过今晚」试探，他当场真情落泪（老国王阅人无数、由此确认其忠·愧疚自己曾备好「若他神色不对就让铁之手当场格杀」）。他感念老国王 30 岁时的知遇之恩，最终与铁之手、赫士列特一同**拥立三王子上位**。
        - **[铁之手](../世界/任务世界/希亚大陆-灭法遗迹.md)**（[苏晓](../人物/苏晓.md) 现兑换的伪装身份·王族御用杀手·替国王秘密除异己·五星官阶·受众官员忌惮无人敢得罪）。
    - **王后与子嗣**：老国王有 **36 名王后**（多是权臣 / 大富商之女·只作权力网纽带、无实权）、**49 名嫡系子嗣**（私生子无继承资格）。子嗣经层层淘汰：18 弃权 + 16 打酱油（求「亲王」闲职）+ 10 有野心无实力 → **仅剩 15 名真竞争者**，其中 **5 大种子选手**（[大王子](../人物/大王子.md) / [三王子·马蒂尔](../人物/马蒂尔·赫伯特.md) / 七王子 / 二公主 / 六公主·详见 [希亚大陆](../世界/任务世界/希亚大陆-灭法遗迹.md)）。
    - **王位结局（十七卷36–40章）**：罗文叛国后**大王子退出接手财政、三王子退出去前线**（两大热门为王国而退）；苏晓奉命快刀清场——**捏死拒退的六公主**、逼退其余 26 名子嗣。最终 **[三王子·马蒂尔·赫伯特](../人物/马蒂尔·赫伯特.md) 秘密联合铁之手 + 乔·瓦伦丁 + 赫士列特上位、成新王**，老国王临终欣慰认可。
    - **王族直属护卫 / 禁卫军**：上千名金甲**直属护卫**守王宫庭院（铁栏饰「犬身狮头」兽=犬身喻官员忠诚、狮头喻王族威严）；重臣每天 6–9 点可自由出入王宫，铁之手等特殊职位 24 小时通行。另有**禁卫军**——王族专门培养的黑甲「杀人机器」（无感情、令其赴死也毫不犹豫·平均属性约 70·各 5 技能）。其能力**人数越多越强**：「**禁军**」（被动·范围内禁卫军达 100/50/20 则全体豁免 40%/30%/10% 伤害）+「**御敌**」（被动·达 100/40/10 则全体全属性 +8/6/3）——满 100 的战阵连苏晓都要暂避锋芒、须先分割削数才有机会。老国王肃清异己时**临时把直属护卫换成 10 名禁卫军**派给苏晓（暗示「一个不留」）。**禁卫军越权伏笔**：按规矩禁卫军不能对官员出手（政治立场），它们却两度越权剁碎大祭司夫妇——苏晓起疑。
    - **被肃清的官员（十七卷9–11章）**：老国王令苏晓杀鸡儆猴，除掉**王国议员伊恩·波利特 / 监察官华纳·伏恩 / 圣堂大祭司扎勃 / 首席法官埃尔维·安东尼**（详见 [希亚大陆](../世界/任务世界/希亚大陆-灭法遗迹.md)·支线「肃清」10 万币）。财务官**罗文·哈维**则靠老鼠传密语「有我吗」探得自己不在名单、安然过关（苏晓有意维系这条财政人脉）。**大祭司临终留下「王座之下隐藏着……」的伏笔**。
    - **暖床混血侍女**：铁之手宅里两名约 10 岁的小侍女（姐姐 + **丝露露**）——**死灵混血奴隶**（见 [死灵族](../世界/生物万族/index.md)），前任侍女因是别的官员安插的眼线被处理（剥皮喂蚂蚁）。
- **死灵帝国 / 死灵族** — 希亚大陆上与圣索丁王国敌对的**死灵种族**国度（领土比人类更富饶）。**首都 = 暗月沼泽**（见 [地点](../世界/地点/index.md)·建在盘结的暗月藤上·比人类王都更大·机械 / 建造技术远超人类、农耕被人类完爆）；君主 **死灵王·森迈尔**（一代代传承）。死灵族还有一种远高于王权的精神图腾「**[启迪](../人物/启迪.md)**」（百年一出·维系死灵族延续的引路萤火）——十七卷73章轮回乐园弄死了启迪、令森迈尔**停止与人族的战争、举族追杀 [苏晓](../人物/苏晓.md)**（铁之手·白夜）。死灵只攻人类运输物资困难的几段神之墙（卢利马斯雪原 / 珂珂巴亚戈壁等）、几百年没碰物资丰富的塞梅湿地段。死灵与人几乎同形、皮肤惨白、**惨绿眼睛**、智商 / 创造力 / 锻造术均超人类，喊「**嘎伊特（为了王）**」死脑筋般执着猛攻 [神之墙](../世界/地点/index.md)（原因无人能解）。军队训练有素（精锻铠甲·弯钩臂刃爬冰墙·远程武器「**死旋 / 黑色幕雨**」黑色尖锥）。详见 [生物·万族 · 死灵族](../世界/生物万族/index.md)。
