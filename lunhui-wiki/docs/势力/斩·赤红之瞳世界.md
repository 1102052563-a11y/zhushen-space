# 斩·赤红之瞳世界 · 势力

> 本页汇总 **斩·赤红之瞳世界** 的登场人物归属。

## 🏛 势力详解

### 斩·赤红之瞳世界（原著民势力）

（详见 [斩·赤红之瞳](../世界/任务世界/斩赤红之瞳.md)·四卷12–14章）

- **帝国** — 存在千年、已腐朽的大帝国，地处大陆中心最富饶处；高税暴政逼反四方、视士兵为弃子、不重热武器（因有帝具）。暗部原本分散，为对抗夜袭部队，**统一交女将军 [艾斯德斯](../人物/艾斯德斯.md) 管理**。
    - **[小皇帝](../人物/小皇帝.md) / [大臣](../人物/大臣.md)**：小皇帝幼年丧亲（双亲实为**大臣毒害**）、被先王亲信的大臣自小洗脑成傀儡、以女色腐蚀——**真正掌权者是大臣**（臃肿却智商极高、握实权、唯忌惮艾斯德斯；小皇帝的权力已被彻底架空）。**最强帝具「至高之力」须皇室血统启动**，故大臣留小皇帝当钥匙。**四卷57章，苏晓绑架小皇帝、取走至高之力启动钥匙、完成主线，断了大臣最后的底牌**。**四卷67章，苏晓放出的牛头领主把帝都化为焦土——大臣被捏死、黑化的小皇帝成牛头领主核心、帝国军与革命军（近两百万）一同被秒杀，帝国实质亡国**（艾斯德斯避战幸存）（四卷18、22–23、53–67章）。
    - **[布德大将军](../人物/布德大将军.md)**：常驻皇宫、统领近卫军、实力仅次艾斯德斯的「老顽固」；帝具「雷神愤怒·亚得米勒」（操雷电）。守皇宫时密不透风，被调去前线后皇宫空虚、苏晓才得手。
    - **皇家护卫军（敌国称「狂犬部队」）**：48 帝具初成时的 48 名帝具使、帝国基石、曾横扫两百年；后帝具屡失，到小皇帝代仅剩十余件。详见 [力量体系 · 帝具](../力量体系/装备道具/index.md)。
    - **三兽士**：[艾斯德斯](../人物/艾斯德斯.md) 的三名心腹帝具使——**利瓦**（老者管家·前任将军·队长·戒形帝具「水龙凭依·黑马林」操纵液体）、**达伊达斯**（壮汉战斗狂·「两柄大斧·贝尔瓦克」可拆合 + 追踪）、**妮乌**（巨乳伪娘·辅助帝具「军乐梦想·尖啸」笛子操控情绪/身体）。**四卷28章，三兽士在帝都港口豪华客船一役被夜袭的布兰德 + 玛茵火力反杀、全数阵亡**（与原著结局一致）（四卷22、28章）。
    - **狩人部队**（三兽士死后新建·四卷35–36章）：[艾斯德斯](../人物/艾斯德斯.md) 重组的暗部、**专门针对夜袭部队**，共 7 人（含艾斯德斯）。成员（帝具使）：[苏晓](../人物/苏晓.md)（白夜·已任第三暗部长官）、[黑瞳](../人物/黑瞳.md)（八房·赤瞳之妹·药物改造）、**波鲁斯**（赤膊壮汉·害羞·原焚烧部队·喷火帝具「炼狱招来·路比冈德」可自爆）、**威尔**（渔村海军·铠甲帝具「修罗化身·贵族战车」=恶鬼缠身升级版·逗逼正义）、**兰**（前教师·羽翼帝具「万里飞翔·莫斯提马」·**四卷43章被卡尔杀**殁）、**Dr.时尚**（同性恋科学家·「神之御手·完美者」百倍精密手指·战斗弱·**四卷43章被卡尔杀**殁）。艾斯德斯曾以肉搏试探新人；成立没几天即被幻影旅团卡尔杀掉 Dr.时尚、兰二人，艾斯德斯震怒、派军在帝都大搜捕。**总攻夜袭部队总部一役（四卷47–50章），狩人部队又折损黑瞳（被姐姐赤瞳所杀）、波鲁斯（自爆同归于尽）**——七人仅余艾斯德斯、苏晓、威尔。另有**焚烧部队**（帝国恐怖部门·烧光瘟疫村镇·波鲁斯的旧属·四卷43章曾屠特利全村）。
- **革命军** — 民间最强、最得民心的反叛军，日渐壮大、与帝国矛盾不断；原著中战力强于帝国（帝国七成大臣被其策反）。已成国家组织雏形、旗下军队近百万，正酝酿与帝国决战。**总部**藏于深山老林的庞大古堡（约 40 米高·常驻 2000+ 军 + 几百官员 + 1 名帝具使·门外百余巡逻·可调集几万）；帝具库在地下室、由臣具「[莫森尼亚·永固之门](../力量体系/装备道具/index.md)」封锁（须臣具少女开启）。苏晓靠层层追踪接头人找到总部、洗劫帝具库、杀总部军官特利与首领娜洁希坦，意在嫁祸帝国、**逼革命军提前与帝国决战**（搅乱帝都好行事）。其追踪能力最强的帝具使 [切尔茜](../人物/切尔茜.md)（变身自在·盖亚粉底）奉命追杀苏晓、却被其沿途陷阱整惨而放弃。**革命军首领震怒、下令立即集结军队进攻帝都——决战提前打响**。四卷50章，革命军**几十万大军突袭**（帝国边防被削、官员多被策反，竟毫无预警），帝国仅派出的四万军被合围；艾斯德斯撤回帝都、调集近百万军在城墙建防线，帝都变「铁桶」（粮不足、半月恐生饥荒）——一场上百万人规模的亡国决战开打（四卷37–50章）。其暗部为——
    - **夜袭部队（Night Raid）** — 革命军的暗部，专司暗杀帝国头目、筹款、刺探情报。首领 [娜洁希坦](../人物/娜洁希坦.md)（独眼独臂白发御姐·**四卷40章被苏晓腰斩**殁）；成员（帝具使）：[赤瞳](../人物/赤瞳.md)（妖刀村雨·斩伤即死·心脏右位逃生）、[雷欧奈](../人物/雷欧奈.md)（狮子王·被苏晓断臂续命）、[拉伯克](../人物/拉伯克.md)（交叉之尾·屏蔽感知丝线）、[塔兹米](../人物/塔兹米.md)（乡下正义新人）、[玛茵](../人物/玛茵.md)（浪漫炮台·南瓜·远程激光）、[希尔](../人物/希尔.md)（剪刀帝具「万物两断·消魂」·原著本应已死、契约者影响下存活）、[布兰德](../人物/布兰德.md)（王牌·铠甲帝具「恶鬼缠身」·**四卷28章被苏晓斩杀**殁）。皆国家机器培养、各有信念，苏晓认定加入帝国后与她们便是你死我活（四卷16、24、28、33章）。
        - **重创**（四卷28–33章）：豪华客船一役，夜袭虽借玛茵火力反杀帝国三兽士，却被苏晓**斩杀王牌布兰德、重创赤瞳（断肋刺心）/雷欧奈（断臂）/塔兹米**，损失惨重；战后凌晨召开「**讨伐苏晓作战会议**」。
        - **覆灭**（四卷40–50章）：首领娜洁希坦被苏晓腰斩；总部被苏晓嗅出、遭狩人部队全员总攻——**赤瞳（剧情主角）被苏晓杀死、拉伯克与艾斯德斯死战帝具被毁阵亡**；玛茵、塔兹米、雷欧奈、希尔等残部撤退。夜袭部队几近团灭。
        - **大臣借刀**：大臣献「嫁祸夜袭」之计、栽赃其刺杀忧国良臣以排除异己；娜洁希坦遂分兵护官、并教训冒名者。借夜袭名义同入本世界的契约者（代表伊凡·魅力最高·带绿色对讲机增援）想「保全夜袭战力以利最终决战、钥匙各凭本事」，却被混入的**幻影旅团 [卡尔](../人物/卡尔.md)（编号11）灭杀一批**（四卷23–24章）。
- **极北异族军** — 帝国北疆因苛税揭竿的异族；首领 [努马·塞卡](../人物/努马·塞卡.md)（北方异族王子·长枪无败绩·极北图腾）曾以 **3 万破帝国 20 万大军**。帝国遂派 50 万大军 + 王牌艾斯德斯镇压；极北军一度与超级危险种 **[芬里尔](../世界/生物万族/index.md)** 勾结（驱雪原狼群助战）。**四卷19–20 章，努马·塞卡被苏晓潜入刺杀、芬里尔被斩杀，极北军遭夜袭灭顶**。
