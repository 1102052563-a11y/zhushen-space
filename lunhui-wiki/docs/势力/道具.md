# 道具（Item · 学园都市暗部）

[魔法禁书目录](../世界/任务世界/魔法禁书目录.md) 世界里**学园都市统括理事会**麾下的**暗部组织**——专做镇压暴动、暗杀、清理外来入侵者等见不得光的脏活。**全队五人、皆为女性**（其中四人是能力者）。[苏晓](../人物/苏晓.md) 一进学园都市就被它盯上、奉命「清理」（直接击杀）——疑由理事长亚雷斯塔·克劳利派出（十三卷3–6章）。

## 五名成员

- **[麦野沈利](../人物/麦野沈利.md)**（队长 / 主力远程）：**Lv.5 超能力者**、学园都市超能者排行**第 4**，能力「**原子崩坏**」（操纵介于粒子 / 波形间的「暧昧电子」、发射高温腐蚀的光束炮 / 绿色分解屏障）。长相甜美、性格残忍。详见其页。
- **绢旗最爱**（主力近战）：**Lv.4 大能力者**，能力「**氮气装甲**」——在身体周围生成几公分厚的无形氮气铠甲（既防御又攻击·拳头能锤扁坦克）。小学生体型、却是十几岁的「人形高达」杀人机器。靠超能力（而非身体）驱动铠甲、嗑药可临时把铠甲增厚十几倍十几分钟（「超超必杀」·副作用大、滥用会缩寿 / 能力退化）。
- **芙兰达·赛维伦**：金发高中生，能力未知、靠**炸弹**作战（藏在能力里）；正面战斗力 ≈ 精通体技的女高中生（被苏晓气息外放震到咳血）。战时多去布置陷阱。
- **泷壶理后**：感知型，**嗑「体晶」**（能力体结晶·使超能力暴走增幅、但损脑、滥用成病娇 / 植物人 / 脑死）后感知范围极大——靠感知**AIM 扩散力场**（能力者独有）定位敌人。锁定苏晓后因气息外放波及 + 嗑药短时昏迷。
- 第五人未在此战露面（道具共五人）。

## vs 苏晓（十三卷3–6章）

麦野沈利一上来就用原子崩坏光束炮把苏晓楼顶轰出大洞（[苏晓](../人物/苏晓.md) 200 能量盾一击碎、却没死），随即被苏晓识破超能力者的两大软肋（[青钢影](../力量体系/技能/index.md) 因敌人无法力而近乎无效；[气息外放](../力量体系/技能/index.md) 反而克制——超能力者靠体力施法、被加 27% 体力消耗）。苏晓改用拳（能量护臂破其绿色分解屏障）一拳砸断麦野肋骨 + 贯穿身体、扭断其手臂；绢旗最爱嗑药开「超超必杀」近战、芙兰达咳血先撤、泷壶昏迷——一个照面四人已两人失能。苏晓的目的不是杀人、而是**逼幕后的亚雷斯塔·克劳利出面**（他必须留在学园都市完成「末灵法典」任务、只能设法与科学侧达成共识）。

## 突袭的结局（十三卷6–9章）
- **[麦野沈利](../人物/麦野沈利.md)（队长）殁**：被苏晓拳破防御后**割喉杀死**（苏晓得世界之源 5.6% + 淡金宝箱）。
- **绢旗最爱**：氮气装甲被苏晓一刀斩透、彻底压制，靠空间系风纪委员 [白井黑子](../人物/白井黑子.md) 公主抱**救走逃离**。
- **芙兰达·赛维伦**：先撤去布陷阱、却被 [布布汪](../人物/布布汪.md) 埋伏俘虏（堂堂 Lv.4 被一条狗擒住·屈辱）。
- **泷壶理后**：锁定苏晓后被气息外放波及 + 嗑药短暂昏迷。
- 一个照面，道具五人折损惨重——印证苏晓「全面发展 + 战斗经验」对发展极端的超能力者的碾压；而幕后的亚雷斯塔始终不为手下生死所动。

## 关系
- **学园都市统括理事会 / [亚雷斯塔·克劳利](../人物/亚雷斯塔·克劳利.md)**：其暗部打手（清理外来入侵者·首领不在乎成员死活）。
- **[苏晓](../人物/苏晓.md)**：奉命「清理」的目标（反被压制·队长麦野被杀）。
