# 轮回乐园 · 现实 · 势力

> 本页汇总 **轮回乐园 · 现实** 的登场人物归属。

## 🏛 势力详解

### 契约者冒险团

- **彼岸花冒险团** — 一支 Lv.6 的契约者永久冒险团；已知成员 [女王](../人物/女王.md)（团长·肉盾）、[黑白](../人物/黑白.md)（狙击）、[叶子](../人物/叶子.md)（奶妈），「盾 + 狙 + 奶」配置。三人持豁免卷潜入东京喰种伏击苏晓，反被全歼；黑白临终撂话「彼岸花冒险团，不会放过你」，疑为后续伏笔（二卷19–31章）。

### 现实世界

- **清理不稳定因素规划局（清道夫）** — 现实国家收编契约者后成立的机构，由契约者组成，负责管控同类（交涉 / 战斗）。对无害者**柔性招安**（签约即既往不咎、身份合法化，违约则不死不休）；成员 [魏冬](../人物/魏冬.md)（魁梧·交涉战斗）、少女小叶（撑场面）。曾在高铁站招安苏晓，并留「请其出手杀来犯的外国契约者、以机密情报为酬」的合作余地（四卷11章）。详见 [机制 · 返回现实](../总览/轮回乐园机制.md)。

### 跨世界组织

- **黑帆商会（黑商）** — 一群不要命的**职工者商人**，专在衍生世界内高价兜售装备 / 物品 / 补给（也接受以物易物）。穿**绝对护甲**（6 点·被动挨揍、不能攻击，用以证明身份）；铁规：**杀害其成员者将被永久拒绝交易、并在乐园悬赏通缉**（曾以 18 万乐园币通缉一名 Lv.9 契约者）。苏晓以法师猩红卡 + 1000 币向其换得立体机动精通卷轴（三卷12章）。**会长 = 钱大胖**（笑面虎）——商会靠早年在衍生世界高价卖药剂起家（脏钱），发家后改走正经生意 + 改善信誉（曾与神皇合资办拍卖会）；如今财力恐怖、储备甚至超大型冒险团。**十卷75–77章**：会长钱大胖下令旗下黑商 **24K**（骷髅面巾·凭专属卡片联络）务必拿下苏晓的 [卞氏匕首](../力量体系/装备道具/index.md)（金色·用作其「独办一场高信誉拍卖会、扬名 + 以金钱之力对抗旅团」的招牌）——遂带高级库存来以物易物。苏晓拿超级破坏拳卷轴 + 卞氏匕首，换得 3 把暗紫刀 + [海洋之心](../力量体系/装备道具/index.md)（淡金宝石）+ 黄金蜡（黑商被他的杀气压着、不敢狠压价）。**十七卷·与三阶最强苏晓深度往来**：① 苏晓打三阶竞技场时黑商开盘（规避惩罚硬扛·与苏晓签契约抽成 50%）；② 24K 用 [氧化裂剧变装置](../力量体系/装备道具/index.md) 换走整团星火低级装备 + 材料（苏晓拿去强化木之灵）；③ 会长**钱大胖**（笑面虎·唯一不戴面巾的黑商）因赌债欠苏晓 88 万币 + 22 颗灵魂结晶〔大〕，**贪了那 10 颗灵魂结晶**没还清，先想拿旗下战地记者 [小花鱼](../人物/小花鱼.md) 抵账卖给苏晓被拒、最终以 8 颗灵魂结晶 + 一把传说级手枪 **[D·暗杀](../力量体系/装备道具/index.md)**（从别家高价收购）了结；苏晓还警示 24K「别参合斯坦的事」、并把锻造大师 [里德](../人物/陵墓守卫者·里德.md) 介绍给乐得蹦起来的钱大胖。黑商不敢黑苏晓这种强力独行侠、转而极力结交。

- **星空座（虚空神秘组织 · 二十六卷69–71章）** — 虚空内最松散、最神秘的组织，**成员从不超 20 人**（现仅剩约 6 人·余者老死 / 意外死）、**从不争夺地盘**却没人敢惹。对意外闯入的旅人很「佛系」：完成一次**交易**就送走（如某十五眼幽鬼用 1/3 灵魂换突破上限的资质）。苏晓捏碎幽鬼世界所得的「未知眼球」误入两次——用凝合性结晶换得**先代灭法者笔记**（记上百种「吞噬之核」可吞的生物能量）。**刀魔**是唯一已知成员（杀了「狮子座」后被赶出·留个有其气息的「机偶」）；**黑雾人影 = 灭法时代的施法者**（[马文·华尔兹](../人物/马文·华尔兹.md) 的老对手 + [瑟菲莉娅](../人物/瑟菲莉娅.md) 的老师·被初代灭法之影封困于此十几年）——后被马文·华尔兹用寄存意识的阵图换出（马文留言要苏晓加入星空座、用黑枫树产物勒索成员·「有人会保你」）。成员聚会「**空座宴**」是苏晓今后用黑枫树暴富的渠道。

### 契约者冒险团（永久）

- **彼岸花冒险团** — **全女性**永久冒险团（入团似有颜值门槛），团长 [冷月](../人物/冷月.md)、副团长 [血玫瑰](../人物/血玫瑰.md)、智囊「腹黑女（小黑）」、狙击手蜂（黑白徒弟）、奶妈糖果（接吻治疗）、女汉子主坦等约五十余人；专属团队技能「**血色追印**」（见 [机制](../总览/轮回乐园机制.md)）。为给被苏晓所杀的女王 / 黑白 / 叶子报仇，在进击的巨人世界与苏晓死战、几近团灭、求和被拒（三卷28–33章）。[女王](../人物/女王.md) / [叶子](../人物/叶子.md) / [黑白](../人物/黑白.md) 亦属此团（东京喰种伏击苏晓被全歼，黑白曾放话「彼岸花不会放过你」）。进击的巨人世界中防守一区（三卷2章）。
- **兄弟会冒险团** — 数十人永久冒险团，团长 [亚当](../人物/index.md)（高魅力**木遁法师**·口碑佳）；入团即兄弟、同生共死、不收心怀叵测者。成员有法师 [尘](../人物/尘.md)、正与 Rc 细胞融合（克制吃人欲）的「烈」等——亚当给烈的「两样东西」即苏晓卖出的初代 Rc 原液 + 原始吸收。进击的巨人防守二区；**五卷被征调入寄生兽空间战场**（守飞船一面·实力居中），亚当认出苏晓「白夜」、并被神王告诫「千万别招惹裁决者」（二卷101、三卷2、五卷3–5章）。
- **幻影旅团**（PVP 永久冒险团）— 以 PVP 见长的可怕冒险团，灵感取自猎人世界的幻影旅团、规则更宽松（四卷1章）：
    - **最多 13 人**，每次进衍生世界**不一定同入一个世界**；团内**无上下级、无团长**，只有**十三个编号**（编号越靠前实力越强）、谁也不必听谁指挥；团员间结怨只需**宣战**即可动手、旁人不阻只看戏。
    - 福利：**共享情报**（但获取情报也须付出情报）、可享**冒险团技能**。
    - 已知成员：[卡尔](../人物/卡尔.md)（第 **11** 位·法师·掌 700+ 法术·**四卷46章被苏晓杀死、编号被苏晓卖掉**）；**左轮男 / 左轮牛仔**（第 **7** 位·约 30 岁·络腮胡 + 牛仔帽 + 雪茄 + 左轮枪·连卡尔都只见过他一人）——**六卷6–8章在七水之都登场、与苏晓死战**：左轮枪 **0.3 秒拔枪**、一声枪响打出三 / 五连发（手速快到枪声连成一响）；子弹**附真实伤害 + 抗性判定**（抗性过低则伤害 ×2~10·一枪打碎无抗性的「庇护」护盾）+ 银色毒弹（扩伤口）+ 子弹尾端爆炸二次加速（打头）；并能凭肩上**白头鹰**抖落的白光羽毛令自己「**隐身**」（屏蔽对手全部感知·连落地血迹与飞行子弹都「隐身」、进 5 米才现形）。他找苏晓**不为给卡尔报仇、纯「遇强者手痒想打一场」**（编号间有感应·杀过旅团成员者数个世界内可被感知）；战斗中惊觉苏晓「能打能抗就差能奶、无耻」（被近身削掉帽檐肩肉）；**最终被苏晓识破隐身（布布特尼能看见他）、引爆 7 颗埋伏的炼金炸弹炸飞一臂，再被飞掷的斩龙闪贯穿小腹，跳海循废弃建筑逃脱**（六卷9章·暂退·短期不敢再现身）；另在不同衍生世界露面的**肌肉大汉**（捏爆头·声波震碎玻璃）、**血红长发吸血鬼美女**（四尖牙·嗜血·身边跟召唤师萝莉「小六」与海量野兽）等（三卷8、四卷1、8、46章）。
        - **编号机制**：成员死亡后其编号空缺；杀掉成员者可获其「**邀请编号**」（用之直接入团、或售 5% 世界之源）；团队技能也会**自动补生新编号**（只是慢些）。苏晓杀卡尔后选择**卖掉 11 号编号**。
    - **威名**：黑帆商会自陈「一阶基本不怕九成以上冒险团，唯独旅团这种『变态 + 疯子的聚集地』及『**神皇**的狂信徒』例外」——黑商因苏晓受旅团邀请，竟主动给他原价/优惠售货、求他入团后替商会向卡尔、左轮牛仔说情（四卷8章）。
    - 苏晓**婉拒入团**：他是猎杀者，且卡尔仅排 11 已颇强、前列必更可怕，他想等实力进**前七**再考虑（四卷2章）。
