# 幽鬼大陆 · 五阶考核 · 势力

> 本页汇总 **幽鬼大陆 · 五阶考核** 的登场人物归属。

## 🏛 势力详解

### 幽鬼大陆 · 祖尔帝国（第24卷·五阶考核世界）

> 详见 [幽鬼大陆 · 五阶考核](../世界/任务世界/幽鬼大陆-五阶考核.md)。人类与幽鬼 / 妖魔 / 诡兽长期共存的原生世界。

- **祖尔帝国**（人类政权）— 17 先王和谈建国·**议会制 + 选举出的「君主」**（[哈罗德](../人物/哈罗德.md) 为现任君主·唯一凌驾议会者）·首都祖尔城·议会有 12 名议员（激进派 / 保守派对立·实为不团结向君主示好）·下设五大暴力机构。觉醒者为主战力、其子嗣（仅传一代）组成精锐战阵。
- **处刑机关**（帝国暴力机构）— 战时军团·专职**清理 / 捕捉**幽鬼妖魔（不负责侦查善后·哪危险去哪·别称「送死部队」）·战力全为觉醒者。名义军团长 [南茜夫人](../人物/南茜夫人.md)（王裔财神）、实际掌权者副军团长 [苏晓](../人物/苏晓.md)（替换利·克拉塞）·受三部门监察·总部地下室存机密 / 危险品（三把钥匙合一才能开）。
- **鬼环**（幽鬼组成的势力）— 买通帝国官员潜入祖尔城的幽鬼组织·首领 [罪恶](../人物/罪恶.md)（枪术宗师混血）·重要成员多为人类 / 幽鬼**混血「先驱者」**（气息与人类无异·如资金提供者 [埃迪爵士](../人物/幽鬼大陆配角.md)）。终极目标 = 布置「**十六眼室**」、信奉「**不朽之王**」（自诩要「解决幽鬼必须食人的问题、让人类与幽鬼和平共存」·苏晓判为给自己脸上贴金）。

### 国足三兄弟（黑暗势力 · 补充）

「黑暗势力」**国足三兄弟**（大 FFF 团）——三名骚紫西装的圣骑士壮汉·人手一把可强化的成长能量锤（一化三·强到 **+15**·+13 以上每级近半个品级）·**三人生命链接（一个不死三个全不死）**·能打能抗能互奶。敢拼（别人一个世界他们拼三个）·现为**四阶最强梯队**（实力排行仅次苏晓 / 已故斯坦·神皇都不敢小觑）。唯一克星是 [苏晓](../人物/苏晓.md)（能烧光他们法力）。第 24 卷与苏晓同入 [幽鬼大陆](../世界/任务世界/幽鬼大陆-五阶考核.md)。
