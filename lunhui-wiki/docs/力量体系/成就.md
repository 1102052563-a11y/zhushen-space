# 成就

> 随阅读更新。记录契约者达成的「成就任务」及其奖励。

## 成就任务机制
**成就任务**是达成特定条件后**自动触发**的隐藏任务（无需主动接取），完成即给额外奖励；有时其回报甚至比主线任务更丰厚（二卷36章）。详见 [机制 · 任务](../总览/轮回乐园机制.md)。

- 成就可被反复利用牟利：违规者 [12470 号](../人物/违规者12470号.md) 便靠特殊手段反复触发成就、攫取**双倍奖励**（二卷41章）。

## 已收录

### 清道夫
- **触发条件**：亲手驱逐某区域 **30% 以上**喰种，并致该区域 **90% 以上**喰种死亡或逃离。
- **达成者**：[苏晓](../人物/苏晓.md)（白夜）——肃清 14 区（亲杀 34%、间接致死 46%、逃 15%，合计驱逐 95%）（二卷36章）。
- **奖励**：1 点属性点 + 7% 世界之源。
