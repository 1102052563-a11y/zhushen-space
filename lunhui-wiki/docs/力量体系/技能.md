# 技能 · 职业

> 随阅读更新。契约者的主动 / 被动能力；解锁可来自乐园兑换、天赋衍生、任务习得或职业传承。

## 技能机制
- 技能有**等级**（LV.1 起，可至 MAX），分**主动 / 被动**（第24章）。
- **技巧性能力级别**：基础 < 精通 < **专精** < 大师——级别越高成长性越高；专精级获得概率极小（第24章）。
- **升级方式**：① 自行锻炼（实战可自然升级）；② 消耗 **灵魂结晶 + 乐园币**直接升级（无需适应、直接掌握）（第28章）。
- **解锁新技能**：部分需材料 + 乐园币（如灭法之影技能四：灵魂结晶（小）+ 1000 乐园币）（第24章）。
- **镜像授艺**：**试炼场**（分辅助模式 / 实战模式）可召唤衍生世界强者的「**镜像**」（模拟其样貌 / 性格 / 所掌握知识、非真人）陪练指点；镜像分**低级（100 币/时）/ 中级（1000）/ 高级（10000）**。如耕四郎·镜像授刀（第27章）；二卷107章苏晓随机召出**宇智波佐助**（教武器裹能量技巧）与**佐佐木小次郎**（教刀术 + 心眼运用、首次触发超极限）。**实战模式**则刷机械人偶练群战（二卷107章）。
- **剧情人物知识传授**（特殊事件）：让通晓某技能的**剧情人物当面讲解**，可触发此事件、在乐园辅助下让对应技能**快速跳级**（如卡莉法讲解「剃」、令苏晓的剃 Lv.1 直升 Lv.3）（六卷21章）。
- **真实伤害**：无视一切防御的伤害——无论敌人防御多强都会被斩伤（二卷3章）。

## 职业
**唯一职业**：极稀有的完整职业传承，会把契约者的技能列表重塑为**技能树**，并大幅改变属性。

### 灭法之影（唯一职业）
[苏晓](../人物/苏晓.md) 拒售世界树指环、完成乐园「抉择」任务所得——来自「**诞生之圣所**」六尊石雕（极强存在的意识，自称「一切施法者的天敌」）（第23–24章）。承接使苏晓 力7→12、敏7→10、智6→12（第24章）。定位：法师（「法爷」）天敌。

#### 来历 · 规则守护神（五卷26章·马文·华尔兹揭示）
苏晓解锁灭法之影新技能时被传送回**虚空中那座传承大殿**（之前看到的威严景象其实是幻境、真容已破败·内有六堆碎石=六尊石雕的真身），由其中一道蓝色虚影——前辈 **[马文·华尔兹](../人物/马文·华尔兹.md)**（完善过「绝魔体质」、是苏晓的「启灵人 / 正牌老师」、现为纯能量体）道出本源：

- **与法师的世仇**：法师每次施法都会**对世界构造造成微小破坏**；灭法之影因此而生、专司诛灭法师，**曾被称为「规则守护神」**、辉煌一时（传承约始于法师出现几百年后）。最初是法师追杀他们、后来反过来他们追杀法师，两方互相杀戮。**别称「破法者 / 破坏者」**——这是**恶魔族**对灭法之影的称呼（两族世代盟友·六卷75章揭·见下「恶魔之印」）；历代有名号者如**第三代破法者 [唐·西林](../人物/index.md)**（恶魔族的恩人）。
- **几近灭绝之由**：成为灭法之影的**资质要求远高于法师**、人数太少（法师基数庞大、杀不尽）；最终**仅剩的六名灭法之影被几万法师围攻**——他们顶着几千道禁咒级法术、杀了三四万法师，终被禁咒轰死。**六人极强**；如今传承只余苏晓一人。
- **对苏晓的评价**：他有「**恐怖到发指的靠山**」（轮回乐园），资质比马文还好，**将来或有 1% 的机会承载「能毁灭万界的能力」**。马文强调杀法师**生存力（顽强）比攻击力更重要、属性须均衡**，遂为苏晓**启灵**（一指点其眉心，将其纯净法力开始融入血肉骨骼以提升生存力·启灵剧痛深入骨髓·五卷27章）。

##### 传承遗迹 · 守护元素平衡（十七卷79–82章·灭法之影遗迹揭示）
苏晓在希亚世界进入一座**灭法之影遗迹**（详见 [希亚大陆·灭法遗迹](../世界/任务世界/希亚大陆-灭法遗迹.md)），借 [马文·华尔兹](../人物/马文·华尔兹.md) 与壁画补全了灭法者的世界观全貌：

- **传承遗迹**：每位灭法者抵达未知世界都会建造，作用有二——① 帮有潜力的人类**觉醒青钢影**、成为灭法之影；② 为新人灭法者**提供指引**。遗迹外围笼罩**雷属性青钢影能量**（电弧伤人·须以自身青钢影护体方能通过），死灵族因此千年都进不去。进入核心三殿需「青钢影能量 + 灭 / 法 / 影三石之任一」。
- **守护元素平衡 = 灭法者的真正天职**：先代灭法之影 [格林·吉莉安](../人物/格林·吉莉安.md)「守护元素平衡 427 次、避免 58 颗星球崩溃」、[马文·华尔兹](../人物/马文·华尔兹.md) 247 次 37 颗——法师施法破坏世界构造，灭法者诛法师、断绝失控法术（如希亚世界的死灵法术），本质是**维系虚空秩序（元素平衡 + 法则之力 + 星之力）**。被救的星球生灵却往往并不感恩。
- **断魂影**：先代灭法之影雕塑铭文中反复出现的名号 / 体系称谓（[阿克斯](../人物/阿克斯.md) 留下灭 / 法 / 影三殿与「断魂影之石」之名亦同源）。**绝灭之役**是灭法者一代凋零的关键战役（格林·吉莉安、马文等多死于此役）。

##### 「法爷」= 学术派法师（十二卷86章·炎辰科普）
轮回乐园里「**法爷**」这个尊称并非对所有法师而言，而是**学术派法师**的专属——是学术派法师们用实力打下来的名号。两类法师之分：

- **学术派法师（法爷·稀少）**：钻研已掌握法术的原理、加入自身体悟，走完整的「感知 TC 克元素 → 调动 → 法力转元素 → 操控 / 塑形 / 裂变 / 压缩 / 吟唱共鸣 → 释放」全流程。**同级别下比技能派强几倍乃至几十倍**。代表 [所罗门](../人物/所罗门.md)（黑暗系·苏晓直觉「就算近身他也不会狼狈」）。
- **技能派法师（多数·「渣渣」）**：只依赖固定法术技能回路、甩技能，实力强些的不过是释放更合理。如今大多数所谓「法爷」其实是这类。
- **两个法爷专属属性**（灭法之影感知不到元素、用不上）：**元素连接强度** = 法师与元素之间「线」的粗细（越粗、施法越快越稳）；**元素感知敏感度** = 能感知的元素浓度（单位「TC 克」·越高耗蓝越少·**只对法爷有效、对技能派无效**）。[月之精髓](消耗品材料.md) 二次调配的成品正是加这两项、故只值钱于学术派法爷。

苏晓自身也暗合「学术派」之道——他用灵魂结晶 + 乐园币升刀术后**仍每日冥想 1~3 小时、深度解析青钢影**（故能自创反击盾），与法爷钻研法术原理同理。

##### 青钢影 · 详解（二卷3章；二卷104章 LV.3；四卷4章 LV.4；四卷72章 LV.7）
- **消耗**：开启耗法力，每分钟 **4** 点（LV.7·升级略增耗）；法力 <1% 自动关闭。
- **原理**：消耗法力形成**特殊能量**附于武器（斩龙闪表面裹蓝白电弧状蓝光），近战命中后侵入敌体、与其能量发生「**噬灭反应**」，燃烧其法力（LV.4=19 / LV.7=28 / **LV.9=34 点**·愈发恐怖）并造成同等**真实伤害**；被燃者承受剧烈痛楚（佐助点评其本质是**纯净精神能量**）。苏晓用灵魂结晶（小）×3 把它从 LV.4 升到 LV.7（四卷72章），又用灵魂结晶（小）×2 + 2000 币升到 **LV.9**（五卷25章·**Lv.10 将迎来质变**）；每分钟耗 5 点法力。**实战效果**：一般近战法力仅 50~100，被苏晓两三刀就燃干、技能都放不出。
- **对无法力的生物**：改噬灭其它能量（**Rc 细胞、查克拉、念、灵压**等），噬灭率略低（10 → 7–9 点）。
- **青钢影坐标·感知追踪**（四卷31–32章·实战开发）：青钢影的能量本是苏晓体内法力转化、归根结底是「他的东西」；故能量侵入敌体后，**苏晓能在一定距离内感知到该敌人的方位**（如同坐标·唯他可感）——只要被他伤到、敌人短时间内就难逃。能量会随时间从敌体逐渐消散，感知随之减弱。他追杀赤瞳时正靠此能力在花街人海中锁定她（弥补了对帝都地形不熟）。
- **法力引爆**（四卷45章·对法师卡尔时开发）：用裹着青钢影能量的斩龙闪**斩在飞来的法术上**，青钢影能量与法术对冲、**令该法术技能提前激发（炸开）**——苏晓借此破解卡尔的「法术机关炮」洪流（能躲则躲、躲不开就一刀斩破）。
- **青钢影噬灭「即死毒」**（四卷48–49章·硬抗赤瞳村雨咒毒）：青钢影能量最擅**泯灭其它能量**，故能与侵入体内的村雨「咒毒」（一斩必杀的即死能量）厮杀互噬。咒毒能量等级虽高，但青钢影**只要法力不竭就源源不断**（咒毒入体时法力转化速率飙到每秒 5 点）——苏晓不切手臂、**硬抗并彻底噬灭了咒毒**（一次耗法力 276 点）。代价是过程剧痛，但他借此**练出对即死类毒素的抗性**（「斩臂如输液、硬抗如自愈、抗体更强」），今后再遇诡异即死能力更易扛。
- **极限输出**（三卷32章·借佐助所授「能量覆盖武器」之理）：把青钢影的输出频率拉满——耗法力由每分钟 3 点暴增至**每秒 30 点**，真实伤害约提升 6 倍（单刀约 96 点真实伤害）；负担大、最多撑约 5 秒（一次约耗 150 法力）。苏晓凭此一刀重创风王模式的冷月。
- **LV.10·质变「操控者」+「法力爆燃」**（六卷72章·灵魂结晶〔小〕+1000 币升 Lv.9→10）：青钢影改头换面——**消耗可自由操控**（每分钟 1~30 点法力·法力 <1% 自动关闭），每次近战**燃敌 5~50 点法力 + 同值真实伤害**（数值取决于每分钟消耗）。若全力开启（30 法力/分）运转**满 10 分钟达峰值**，则触发「**法力爆燃**」效果：**下次攻击额外造成「敌人已消耗法力值的 60%」真实伤害**（敌耗 100 法力即 +60 真伤·冷却 48h）。对刚厮杀过的法师，配合基础攻击 + 锋刃 + 青钢影 + 法力爆燃，几乎一刀必杀。
- **「震动切割」开发成功**（八卷42章·吞噬岛母巢实战）：苏晓久研「让青钢影能量高频震动产生切割力」（六卷80–81章·参考风查克拉震动空气之法）——此前因「**持续输送的能量频率不一、很快散开**」屡试屡败；后来灵光一闪——**既然无法持续输送、那就一次输送一大股能量**！代价是**单次猛耗约 1000 点法力**，但青钢影能量**成功震动起来**——斩龙闪化作一柄**高速震荡的「电刃」**：连**精英刺蝎的厚甲壳都如划纸般切开、连飞船的特殊合金都如切豆腐**，约 **10 秒**后能量消散。被七只精英刺蝎围困的绝境，他正是靠此招瞬间反杀全部、掉落三枚紫宝箱（这是他**自创的高阶杀招**，验证了「越到后期越契合」的灭法之影潜力）。

##### 至尊锋刃 · 技能五（二卷103章）

##### 灵影体质 · 技能六（五卷27章·启灵所得）

## 探测类能力
查探敌人资料（属性、技能等）的途径有三类：**① 探测类装备 ② 使徒类召唤物 ③ 侦查类技能**；权限不足时乐园不予展示。苏晓在东京喰种被敌方契约者监视后，深感「知己知彼」之要，决意回乐园后获取其一（二卷20章）。

### 基础探测（临时技能）
因猎杀者等级（LV.2）**低于所在衍生世界一级**，乐园开放的付费**临时技能**（离开本世界即消失）；苏晓花 **800 乐园币**购得（二卷20–21章）。

- **等级**：LV.2（每升 1 级 +5% 资料完整度）。
- **消耗 / 冷却**：10 法力 / 冷却 10 分钟。
- **效果**：观测任意生物的资料；**本体与目标实力差距越大、所得资料越完整**（目标弱于自己时资料近乎全显，如探得 [铃屋什造](../人物/铃屋什造.md) 全部六维与被动技能）。

## 秘术·雷属性活化（传说级 · 成长被动 · 十五卷46–47章）

开 [四代雷影·艾](../人物/四代雷影.md) 的**传说级宝箱**所得（云隐村历届雷影独有·正是雷影驾驭那恐怖速度的根源）——苏晓的**第一个传说级能力**。它是一种「**查克拉化的雷电**」（存于卡片·非能量），掌握时引入脊椎神经系统（青钢影差点把它噬灭、靠操控护住·体力须够强否则伤神经）。**满级被动（不可再升）**：**神经反射速度永久 +16%、动态视力永久 +16%**（= 真实敏捷 ×0.2%·**今后每涨 1 点敏捷再加成**）+ **雷属性抗性 +20**（附几率免疫雷系控制·简介未标的意外之喜）。前置：力 65 / 体 78 / **真实敏捷 80** / 生命 ≥2000（苏晓已满足）。神经反射速度是近战仅次于属性的关键、动态视力又强化 [直感](#)——初掌握后世界「慢放几十倍」、冥想适应后实测**已能斩飞蜘蛛女皇的子弹**（远未到极限）。长远价值或超传说级装备（成长性）。

## 刀术大师 → 刀术宗师（阶位攀升 · 十九卷92–97章）

- **刀术大师 Lv.50 = 理论上限**：升 Lv.48→50 时（每级 25 万币 + 灵魂结晶〔大〕×10），苏晓被 [黑枫树老者](../人物/黑枫树老者.md)（[马文·华尔兹](../人物/马文·华尔兹.md) 的师傅·唯一现存的灭法者·将刀魔虚影囚于体内吞噬之）召入幻境点拨抉择。老者给三条路：① 想办法**突破大师极限继续升**（下限高·初期强）；② **晋升刀术阶位**（上限高·后期强）；③ 极端之路（短时间变强几十倍上百倍、但意识可能消亡·不归路）。苏晓选 ②（核心能力当取上限）。Lv.50 后刀术大师临时锁定为 **Lv.MAX**（刀类武器攻击 +15%、刀芒威力近 +1/3）。
- **晋升门槛 = 宗师之力（完整）+ 荣耀之证明**：晋升大师级本需试炼考核，猎杀者凭高阶位能量**豁免考核**，但要付两样：**[宗师之力（完整）](装备道具.md)**（10 份任意宗师之力残卷可合成·苏晓凑齐）+ **[荣耀之证明](装备道具.md)**（试炼场「挑战模式」打赢指定镜像所得·95% 以上挑战者失败）。苏晓在试炼场连战镜像 **西尔西亚**（人类·力/敏/体 95 真实·仅一种宗师级能力「近战肉搏宗师 Lv.10」却完爆其刀术大师 Lv.MAX）——**连败八场、每场 5000 币、第八场才同归于尽式险胜**得荣耀之证明，由此领悟「大师级与宗师级差距悬殊、宗师级初始下限就极高、更靠战斗经验与时机」。
- **🆕刀术宗师 Lv.1（十九卷97章）**：消耗荣耀之证明 + 宗师之力（完整）晋升成功。**刀术宗师 Lv.1**：刀类武器攻击 +240%（因仅 Lv.1·暂低于大师 MAX）、**神经反射速度 +3%（可随等级成长）**、**完全免疫精神系 + 心灵系控制**（近战克星·之前的刀术只能免疫可判定的精神控制）、初始附加「**万物律动 Lv.MAX**」、刀芒 Lv.50（飞行 +150% / 锋刃 +30 / 破甲 +25 / 能量化实体）。晋升后他一刀斩断之前砍不动的「**莫罗复合金属柱**」（34 吨·大师 Lv.MAX 绝对斩不断）。提升费却意外便宜（Lv.1 仅灵魂结晶〔大〕×1 + 5 万币）——「厚积薄发、之后战斗力将快速提升」。
