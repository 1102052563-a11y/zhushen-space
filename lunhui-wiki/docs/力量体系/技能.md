# 技能 · 职业

> 随阅读更新。契约者的主动 / 被动能力；解锁可来自乐园兑换、天赋衍生、任务习得或职业传承。

## 技能机制
- 技能有**等级**（LV.1 起，可至 MAX），分**主动 / 被动**（第24章）。
- **技巧性能力级别**：基础 < 精通 < **专精** < 大师——级别越高成长性越高；专精级获得概率极小（第24章）。
- **升级方式**：① 自行锻炼（实战可自然升级）；② 消耗 **灵魂结晶 + 乐园币**直接升级（无需适应、直接掌握）（第28章）。
- **解锁新技能**：部分需材料 + 乐园币（如灭法之影技能四：灵魂结晶（小）+ 1000 乐园币）（第24章）。
- **镜像授艺**：**试炼场**（分辅助模式 / 实战模式）可召唤衍生世界强者的「**镜像**」（模拟其样貌 / 性格 / 所掌握知识、非真人）陪练指点；镜像分**低级（100 币/时）/ 中级（1000）/ 高级（10000）**。如耕四郎·镜像授刀（第27章）；二卷107章苏晓随机召出**宇智波佐助**（教武器裹能量技巧）与**佐佐木小次郎**（教刀术 + 心眼运用、首次触发超极限）。**实战模式**则刷机械人偶练群战（二卷107章）。
- **剧情人物知识传授**（特殊事件）：让通晓某技能的**剧情人物当面讲解**，可触发此事件、在乐园辅助下让对应技能**快速跳级**（如卡莉法讲解「剃」、令苏晓的剃 Lv.1 直升 Lv.3）（六卷21章）。
- **真实伤害**：无视一切防御的伤害——无论敌人防御多强都会被斩伤（二卷3章）。

## 职业
**唯一职业**：极稀有的完整职业传承，会把契约者的技能列表重塑为**技能树**，并大幅改变属性。

### 灭法之影（唯一职业）
[苏晓](../人物/苏晓.md) 拒售世界树指环、完成乐园「抉择」任务所得——来自「**诞生之圣所**」六尊石雕（极强存在的意识，自称「一切施法者的天敌」）（第23–24章）。承接使苏晓 力7→12、敏7→10、智6→12（第24章）。定位：法师（「法爷」）天敌。

#### 来历 · 规则守护神（五卷26章·马文·华尔兹揭示）
苏晓解锁灭法之影新技能时被传送回**虚空中那座传承大殿**（之前看到的威严景象其实是幻境、真容已破败·内有六堆碎石=六尊石雕的真身），由其中一道蓝色虚影——前辈 **[马文·华尔兹](../人物/马文·华尔兹.md)**（完善过「绝魔体质」、是苏晓的「启灵人 / 正牌老师」、现为纯能量体）道出本源：

- **与法师的世仇**：法师每次施法都会**对世界构造造成微小破坏**；灭法之影因此而生、专司诛灭法师，**曾被称为「规则守护神」**、辉煌一时（传承约始于法师出现几百年后）。最初是法师追杀他们、后来反过来他们追杀法师，两方互相杀戮。**别称「破法者 / 破坏者」**——这是**恶魔族**对灭法之影的称呼（两族世代盟友·六卷75章揭·见下「恶魔之印」）；历代有名号者如**第三代破法者 [唐·西林](../人物/index.md)**（恶魔族的恩人）。
- **几近灭绝之由**：成为灭法之影的**资质要求远高于法师**、人数太少（法师基数庞大、杀不尽）；最终**仅剩的六名灭法之影被几万法师围攻**——他们顶着几千道禁咒级法术、杀了三四万法师，终被禁咒轰死。**六人极强**；如今传承只余苏晓一人。
- **对苏晓的评价**：他有「**恐怖到发指的靠山**」（轮回乐园），资质比马文还好，**将来或有 1% 的机会承载「能毁灭万界的能力」**。马文强调杀法师**生存力（顽强）比攻击力更重要、属性须均衡**，遂为苏晓**启灵**（一指点其眉心，将其纯净法力开始融入血肉骨骼以提升生存力·启灵剧痛深入骨髓·五卷27章）。

##### 「法爷」= 学术派法师（十二卷86章·炎辰科普）
轮回乐园里「**法爷**」这个尊称并非对所有法师而言，而是**学术派法师**的专属——是学术派法师们用实力打下来的名号。两类法师之分：

- **学术派法师（法爷·稀少）**：钻研已掌握法术的原理、加入自身体悟，走完整的「感知 TC 克元素 → 调动 → 法力转元素 → 操控 / 塑形 / 裂变 / 压缩 / 吟唱共鸣 → 释放」全流程。**同级别下比技能派强几倍乃至几十倍**。代表 [所罗门](../人物/所罗门.md)（黑暗系·苏晓直觉「就算近身他也不会狼狈」）。
- **技能派法师（多数·「渣渣」）**：只依赖固定法术技能回路、甩技能，实力强些的不过是释放更合理。如今大多数所谓「法爷」其实是这类。
- **两个法爷专属属性**（灭法之影感知不到元素、用不上）：**元素连接强度** = 法师与元素之间「线」的粗细（越粗、施法越快越稳）；**元素感知敏感度** = 能感知的元素浓度（单位「TC 克」·越高耗蓝越少·**只对法爷有效、对技能派无效**）。[月之精髓](消耗品材料.md) 二次调配的成品正是加这两项、故只值钱于学术派法爷。

苏晓自身也暗合「学术派」之道——他用灵魂结晶 + 乐园币升刀术后**仍每日冥想 1~3 小时、深度解析青钢影**（故能自创反击盾），与法爷钻研法术原理同理。

- **消耗**：开启耗法力，每分钟 **4** 点（LV.7·升级略增耗）；法力 <1% 自动关闭。
- **原理**：消耗法力形成**特殊能量**附于武器（斩龙闪表面裹蓝白电弧状蓝光），近战命中后侵入敌体、与其能量发生「**噬灭反应**」，燃烧其法力（LV.4=19 / LV.7=28 / **LV.9=34 点**·愈发恐怖）并造成同等**真实伤害**；被燃者承受剧烈痛楚（佐助点评其本质是**纯净精神能量**）。苏晓用灵魂结晶（小）×3 把它从 LV.4 升到 LV.7（四卷72章），又用灵魂结晶（小）×2 + 2000 币升到 **LV.9**（五卷25章·**Lv.10 将迎来质变**）；每分钟耗 5 点法力。**实战效果**：一般近战法力仅 50~100，被苏晓两三刀就燃干、技能都放不出。
- **对无法力的生物**：改噬灭其它能量（**Rc 细胞、查克拉、念、灵压**等），噬灭率略低（10 → 7–9 点）。
- **青钢影坐标·感知追踪**（四卷31–32章·实战开发）：青钢影的能量本是苏晓体内法力转化、归根结底是「他的东西」；故能量侵入敌体后，**苏晓能在一定距离内感知到该敌人的方位**（如同坐标·唯他可感）——只要被他伤到、敌人短时间内就难逃。能量会随时间从敌体逐渐消散，感知随之减弱。他追杀赤瞳时正靠此能力在花街人海中锁定她（弥补了对帝都地形不熟）。
- **法力引爆**（四卷45章·对法师卡尔时开发）：用裹着青钢影能量的斩龙闪**斩在飞来的法术上**，青钢影能量与法术对冲、**令该法术技能提前激发（炸开）**——苏晓借此破解卡尔的「法术机关炮」洪流（能躲则躲、躲不开就一刀斩破）。

##### 灭法之影克制法师（二卷88–89章·实战寒鱼后总结）
「灭法之影」之名名副其实，对法师有全方位压制：

- **青钢影对法师有奇效**：命中其他敌人是「碰到哪、烧哪的能量」，但对法师其能量会**直冲大脑、抵脑后爆开燃烧**，剧痛远超常人——青钢影本就是为狩猎法师而开发。
- **绝魔体质**免 40% 法术伤害，给了近身的时间；**智力**属性减免**精神系控制**（如「精神·脉冲」走智力判定）；**力量**属性挣脱**物理性控制**（如冰触须，不判定、靠蛮力）；**敏捷**保证快速接近。
- 结论：只要近身，实力相近乃至略强的法师都非其对手；法师腿短、控制吃瘪后早晚被近身。智力越高，法师对他威胁越小。

##### 灵影体质 · 技能六（五卷27章·启灵所得）

> - **「刀魔」深化（马文警告的歧路）**：「**手握杀生之刃、念为魔，简称刀魔**」——灭法者堕入的歧路、**虚空金字塔顶端的狠角色**（一名壮汉的星球十年前正是被刀魔所毁）。这与马文·华尔兹「别成刀魔」的警告、古战场「蓝烟持刀人形生物」伏笔一脉相承。**已知在世灭法仅苏晓一人**（灭法者时代千年前已落幕）。

> **苏晓的完整技能盘（六卷75章·更新）**：① 青钢影 **Lv.10**（质变·操控者 + 法力爆燃）② 刀术大师 **Lv.18**（能斩刀芒）③ 绝魔体质 Lv.MAX ④ 心眼 Lv.10 ⑤ 至尊锋刃 Lv.MAX ⑥ 灵影体质 Lv.MAX ⑦ **恶魔之印 Lv.1**（+10 真伤）；特殊·心之冥想 Lv.1；六式·剃 **Lv.20 满级**（急速行者）。仍**仅青钢影 / 至尊锋刃 / 剃为主动**，余皆强化本体的被动——堆被动使他「每一刀都堪比敌人的大招」。下一个灭法之影技能要求三主属达 **50 点 + [灵魂结晶（中）](消耗品材料.md)×10**（极高）。

> **灭法之影 vs 黑暗系法师**（五卷29章）：对普通法师是「互相战争」，对**黑暗系法师**则是「赶尽杀绝、有多少杀多少」——黑暗系法师虽不破坏世界构造，却**直接破坏力更可怕**（许多烈性瘟疫与其有关）。苏晓因职业本能厌恶他们（卖精神冲击卷轴时一名黑袍黑暗系法师 10 万买走、令他天生反感），但不会主动替天行道。

> 苏晓战斗路线为**以刀术为主力**（参考海贼世界专修剑术的鹰眼、红发）（第28章）；技能四「心眼」+ 技能五「至尊锋刃」进一步强化其近战体系。

## 探测类能力
查探敌人资料（属性、技能等）的途径有三类：**① 探测类装备 ② 使徒类召唤物 ③ 侦查类技能**；权限不足时乐园不予展示。苏晓在东京喰种被敌方契约者监视后，深感「知己知彼」之要，决意回乐园后获取其一（二卷20章）。

### 基础探测（临时技能）
因猎杀者等级（LV.2）**低于所在衍生世界一级**，乐园开放的付费**临时技能**（离开本世界即消失）；苏晓花 **800 乐园币**购得（二卷20–21章）。

- **等级**：LV.2（每升 1 级 +5% 资料完整度）。
- **消耗 / 冷却**：10 法力 / 冷却 10 分钟。
- **效果**：观测任意生物的资料；**本体与目标实力差距越大、所得资料越完整**（目标弱于自己时资料近乎全显，如探得 [铃屋什造](../人物/铃屋什造.md) 全部六维与被动技能）。

## 副职业 · 炼金学（四卷69–70章）
苏晓从 [夏洛克](../人物/夏洛克.md) 遗留的布袋里得到**【炼金学（完整）】卷轴**（产地斩·赤红之瞳·**五阶卷轴**·知识类技能分一~九阶·需力 10 / 体 10 / 智 25）。使用后**炼金学归入「副职业」列表、单独计算、不影响灭法之影**——这是苏晓的**第一个副职业**。

- **定义**：炼金学起于人类对黄金的渴望、发展成对大自然的探索与对物品 / 生物的改造强化，含生物学、锻造、机关制造等（苏晓只学到很基础的部分·相当初高中知识·现实不会教）。
- **风险**：学了**生物改造学**会令一些剧情人物对他生敌意；炼金学高等级后衍生世界的**科研阵营会优先招揽、甚至有大型势力绑架**他的风险。
- **技能树（三层·各层独立升级解锁下一层）**：

        - **冲满 Lv.20**：替世界政府量产飞快刷熟练度（熟练度堆到 2073），回卧室以「500 币 + 熟练度/次」一口气从 Lv.3 升到 **Lv.20 满级**。满级三选一奖励（A 快速制造 / B 范围 / C 威力 / **D 精密操控**），苏晓选 **D 精密操控**（炸弹只是辅助、后期靠刀术，故强化「对炸弹的控制」）。
        - **解锁配方**：高级精神力炸弹、**[特级精神力炸弹](消耗品材料.md)**（绿·270 伤·可变形/分离）、**[太阳神阿波罗](消耗品材料.md)**（微型太阳·半公里·需灵魂结晶）。**「炼金药剂学」（第二层）仍须返乐园才能解锁**（满级提示）。
    - **第二层·炼金药剂学**（六卷80章解锁 → 七卷26章已升 **Lv.6**）：第一层达 Lv.10 后花 灵魂结晶（小）+10000 币解锁；七卷在钢炼世界靠**贤者之石**当材料（每瓶耗石 20 能量 + 40 法力）大量量产、把它从 Lv.1 一路冲到 **Lv.6**，解锁 [低级 / 中级炼金秘药](消耗品材料.md)（回血）、[林荫药剂](消耗品材料.md)（随身回体力 + 提神）等配方（再升还能造「大药」）。**升级解锁配方无固定规律**（有时升一级解锁、有时两三级）。另：太阳神阿波罗乐园造价 8 万币 + 灵魂结晶 + 500 法力（太贵·放弃量产）；特级炼金炸弹仅 300 币 / 颗。

## 六式（海贼王世界 · 体技）
海军与 CP 部门的高等格斗体技体系（剃、铁块、月步、纸绘、指枪、岚脚六门，及奥义「六王枪」）。学六式的真正门槛不是招式（原理都不难），而是**超凡体质**——须先经残酷训练把身体强度练到能承载体技的程度（换算成乐园属性约**力 25~30 / 敏 25~30 / 体 25~30**）。

- **剃**（苏晓所求·补缺的高速位移）：原理是**在 0.36 秒内踩踏地面 10 次以上**，靠反作用力层层叠加爆发、快到肉眼难辨（像「二次、三次…十几次加速」）。弊端：须精控踩踏力度（否则撞墙 / 撞上敌人武器送命）、且**途中无法快速变向**（只能小幅修正路线）。
- **铁块**：身体瞬间硬如钢铁、硬抗攻击（CP9 [布鲁诺](../人物/index.md) 曾以铁块硬扛炸弹爆炸毫发无伤·五卷生命归还等更高阶体技苏晓体质暂不够）。

### 苏晓的「剃」（六卷12–17章·已掌握 Lv.1）
他**反劫持 CP9 的卡莉法**、以「胃里的炼金炸弹」要挟、勒索到 **[「剃」修炼方法（完整）](装备道具.md)**（紫·评分 141·力 25/敏 30/体 25·离开本世界收回·卡莉法所制·猎杀者无法模拟）。苏晓力 32 / 敏 32 达标、体力 25 勉强够（学剃没问题、铁块 / 生命归还较难）。苦练数日 + 餐船 [药膳老者](../人物/index.md) 点拨「修体术注意侧重点、不可心急」后，**初步掌握 → 六式·剃 Lv.1**：

- **效果**：极短时间内**向前移动 5~18 米**、移动速度达敏捷属性最大值；耗 **20 体力值**、冷却 30 秒（自学掌握者可强行连用、但损伤下肢）。
- **体力值**：苏晓由此接触到一个新概念——「**体力值**」是**体力属性衍生**的隐藏数值（耐力 / 疲劳度·人人皆有·低于 2 点会脱力）。
- **升至 Lv.3**（六卷21章）：苏晓再以 2 颗炼金炸弹买卡莉法**讲解剃的技巧**，触发「**剧情人物知识传授**」特殊事件、在乐园辅助下 **Lv.1 直升 Lv.3**（移动距离与精准度大增·可用于逃跑、但仍不宜实战）；须返乐园升到 **Lv.5+** 才能融入战斗体系（高费用）。
- **升至 Lv.20 满级**（六卷72章）：海贼王第六卷再入通关后，苏晓在乐园以 **2000 币 / 级**把剃从 Lv.3 一路冲到 **Lv.20 满级**。满级三选一附加（① 超负荷 ② 自由奔行 ③ 急速行者）选 **「急速行者」**——施展时间 −50%、移动速度 +300%、但**使用间隔 +3000%**（约 10 分钟才能用一次）。满级剃：**超高速向前移动 2~50 米**（速度达身体所能承受的最大值·途中可修正方位 / 停止）、耗 **50 体力值**、冷却 10 分钟。苏晓不求用剃上蹿下跳，只为**一击接近敌人**——配合刀术，成了名副其实的「法师杀手」（冲到法师面前一刀、对方根本反应不及）。

## 自悟技能
### 心之冥想（Lv.1 · 被动）
苏晓长期每日盘坐冥想（本为清除杀念、保持冷静理性、不被杀意蛊惑而沦为杀人狂魔），其感悟达到乐园认可的程度后**自行领悟**此技能（四卷34章）。效果：小幅提升**感知力、意志力**，平复心境、增进对大自然的体悟。**仅能凭个人提升、乐园无法强化**（与「万物律动」同理）。

### 自创刀芒招式（无冷却 · 无固定数值 · 威力凭使用者把握）
苏晓用刀日久、自创过三十多种招式，**实用的仅以下几种**（开发招式极危险：他曾开发到一半把自己刺成刺猬险死·超十分之一的强者死于开发招式）：

- **环断**（八卷32章）：拔刀斩掺入刀芒、向四周扩散成**直径十几米的圆形 360° 杀伤**——围攻时清场极实用（曾一招秒杀几百刺蝎 / 契约者）。开发方向 = 增斩击力 + 扩范围；融青钢影（能量·与刀芒性质相斥）失败。
- **十字**（八卷38章）：十字形刀芒、开路用。
- **连鞘斩 ·「破空」**（十一卷53章开发、66章成型实战·苏晓自评最强招）：**居合式拔刀斩**——把约 **300 法力**化作青钢影能量、连鞘包裹斩龙闪（出鞘时迸出极强切割力的蓝光），再以极慢蓄势后骤然斩出「破空」。特点 = **速度极快**（敌在几十米外亦可瞬间斩到）+ **威力远超环断**，是苏晓「最强的攻击招式、没有之一」。**十一卷66章首次成型实战**：在地下停车场迎 [英雄王](../人物/吉尔伽美什.md) 的「天地乖离之剑·红色剑风」，他先以几十面反击盾拼成**全身能量铠甲**护体、再用「破空」冲入剑风——一道蓝光直穿剑风、斩断英雄王持剑右臂（自己也被剑风重创成独臂）。**对单 / 对众两种用法**（十二卷30章·战矮人大地骑兵团时阐明）：① **对单**——斩击力收敛、命中瞬间青钢影 + 刀芒侵入敌体、几秒后在体内形成 [影刺](技能.md) 从内重创 / 杀敌；② **对众**——完全释放斩击力，一道直线蓝斩芒掠过、沿途炸开纵横交错的斩击（黑土要塞高地战中一招把几十名矮人大地骑兵斩到仅剩 4 人）。代价是对身体负担不小（用后踉跄脱力）。**「破空」二段攻击（十三卷17章揭示·战 [神裂火织](../人物/神裂火织.md) 时用）**：破空真正最强的不是一段斩击、而是**二段**——一段命中后、青钢影会潜伏在敌体内，之后陆续在敌体内**生成能量刺、从内一根根破体而出**（须苏晓在场控制方能同时齐发·不在场则一根根刺出·敌人只能用自身能量勉强控制刺穿位置、避开要害·过程极痛），配合青钢影持续噬敌能量造真伤——即便苏晓撤走、敌人仍会被二段持续折磨（神裂火织被斩后体内不断刺出能量刺、险些丧命）。
- **寸断**（九卷25章·环断分支）：把环断的「360° 刀芒扩散」改为**只攻正前方、且不放刀芒、直接以青钢影震荡的斩龙闪（化能量刃）猛斩**（单次猛耗千点法力）——近身一击的爆发斩。九卷以此当头斩中三忍 [自来也](../人物/自来也.md)、在其胸膛留半米深伤 + 青钢影侵入令其查克拉暴走（自己也被自来也的钢铁白发刺穿身体·险中心脏）。
- **影刺**（九卷13章·青钢影衍生）：环断融青钢影失败后，改把**青钢影「包裹」在刀芒上**——刀芒生出半米长、手指粗的**淡蓝色尖刺**，穿透力极强（超过普通刀芒）。设想用法：把斩龙闪刺入敌体后令尖刺**由内向外刺出**，敌人瞬间变刺猬、近乎一击必杀（现仅半米长·增长需大量青钢影）。命名走苏晓一贯的「实用流」（不取「天马环星斩」之类）。**九卷34章首次实战**：把斩龙闪刺入 [我爱罗](../人物/我爱罗.md) 的沙球后激发影刺、十几根淡蓝尖刺将我爱罗半个身体刺穿（一臂垂落）——隔着沙球破防的开局杀招。
