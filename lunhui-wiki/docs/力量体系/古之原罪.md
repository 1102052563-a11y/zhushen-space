# 古之原罪

> ⚠ **贯穿全书的十一件「大爹级·原罪物」，其真相在终章前才被揭开:它们是陨落的界外存在。**

## 🌫 它们是什么

在星界、深渊、元素之外，还有更遥远的**混沌区域**。那些蛰伏于混沌、深渊之底的沉淀之下、以及元素之顶更上方的「无尽顶隙」中的久远存在，**别说靠近星界，连进入深渊都不可**——一旦它们有向星界靠近的架势，需要面对的就是**至高之人**。

这些沉眠至今、**等同于强大无上位阶**的古老存在，唯一忌惮的也只有至高。它们有一个共同的称呼:

> **古之原罪。**

⚠ **大爹级·原罪物，就是它们陨落所化**——每当其中之一苏醒、降临星界，那都是绝望的毁灭与灾祸，直至至高封临。**那些决定灭杀至高的「古之原罪」，现在已都陨落成了原罪物。**

!!! danger "知晓的门槛"
    ⚠ 知晓这一切的人**不足一手之数**，因为知晓的门槛是**无上之境**。<br>无上之下，知晓这些原罪存在后会逐渐被原罪侵袭——**因此都不用进行保密一类，没资格的人知晓后会自行消亡。**

## 🌍 远古星界的三大恐怖

在那遥远又恐怖的远古星界中，深渊与元素力量都比现在极端得多，三类界外存在几乎没给生灵留下喘息空间:

| 名称 | 来源 |
| --- | --- |
| **古之原罪** | 来自深渊 |
| **太古噩兆** | 来自元素——一种无形却庞大的存在 |
| **混沌灾厄** | 来自混沌区域 |

⚠ 是**守渊者**与**星使徒**交替着守望，才让一丝丝文明火种蔓延下来（详见 [守渊者与星使徒](守渊者与星使徒.md)）。

## 👑 至高之人与它们的关系

- 昔日的至高之人抵抗过元素浩劫、登上最高位后，逐渐成为**守护者**——**震慑那些古之存在的原罪们**，偶尔远远关注星界的大事件或趣事，甚至不干涉深渊/元素的自然爆发（无论进入深渊时代还是元素时代，终归是星界轮转的一部分）。
- ⚠ **一个超出无上极致的强大者，是生灵挣脱各类「界外存在」支配的唯一出路。** 封临至高需要付出因果代价——当初所有星使徒以因果牺牲，成就了其中最强者的至高力量，**这不单是力量，也是责任与背负**。
- 可他最终败给了时间。上古时代初期他陷入真正的沉眠；醒来后为了「总要有人封临这位置」而降临星界，逼迫无上强者迈向他这一步——**并在第一位挑战者身死时察觉到，无上强者的因果命运有资格被他吞噬**。详见 [至高之人](../人物/至高之人.md)。

## 💀 它们的下场与未来

苏晓在准备与至高之人决战时，把十件大爹级原罪物连同 **【无尽之环】** 一并投入 **「资源置换装置」**，转化为提升自身体魄所需的 **「自我本源」**。

- ⚠ 结果:十件原罪物全部破碎。**「古之原罪」的根源并未泯灭**，可它们积攒多年的复苏力量可谓竹篮打水——**想再度成为原罪物回到星界，估摸得是三四个大纪元后的事**。
- ⚠ 由此也能看出**「古之原罪」有多难以彻底消灭**，难怪这些界外存在活跃时，远古星界是那么危险与恐怖。

**结局**:决战两个月后，混沌之雾中的它们因「昔日的至强已身死」而蠢蠢欲动——直到新封临的顶峰强者现身，目光环视一圈，它们重新蛰伏回混沌之雾中。

⚠ 苏晓对此的态度是:他不打算主动招惹它们；他作为原罪物的封印者太清楚这些原罪的不灭特性。**而顶峰之位，也因此从不寂寞。**
