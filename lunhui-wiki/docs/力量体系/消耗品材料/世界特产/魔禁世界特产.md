# 魔法禁书目录 · 世界特产

> 魔禁世界的特产。

## 魔法禁书目录·世界特产

> 原属批次:已收录（截至第20章）

- **魔力晶碎**（淡金 · 稀有品·十三卷28章·开史提尔暗紫宝箱所得）：本身「无法使用」，唯一用途是**卖给轮回乐园换 10% 世界之源**（仅能在魔法禁书目录世界出售）。世界之源 = 少量乐园币 + 属性点，故等于白捡属性点·契约者很欢迎。苏晓直接卖掉换得 10% 世界之源。
- **魔法师的符文卡**（暗紫 · 宝箱类·十三卷23章·杀史提尔所掉）：魔禁世界魔法师的符文宝箱。
- **哈迪斯·冥界之焰**（苏晓自制 · 大型燃烧弹·十三卷36章）：苏晓战争世界后宅在专属房间研究**绿磷溶液**、再以「绿磷溶液萃取物 + 科技」结合造出的不稳定大型燃烧弹（材料含火金 / 铀浓缩物 / 裂变可燃材料 / 压缩固态氧 / 绿磷溶液萃取物等）。**绿色火焰扑不灭**（水 / 干粉 / 空气系超能力都压不住·80% 物质会成助燃物）、初始燃烧范围 5 米、完全扩散可达 5~6 公里。与 [太阳神阿波罗](../index.md) 互补：阿波罗是「瞬间狂猛」（杀敌用），哈迪斯是「持续燃烧扩散」（**烧敌人老家专用·绝户的勾当**·扩散太慢、不适合直接杀敌）。在魔禁世界他用它烧掉学园都市第二学区四分之一、逼亚雷斯塔屈服。
