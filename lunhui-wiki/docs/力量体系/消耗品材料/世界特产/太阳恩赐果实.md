# 太阳恩赐果实

## 太阳恩赐果实（暗紫 · 八卷35章）

> 原属批次:已收录（截至第20章）

苏晓从 [巴哈姆（翼龙·霸主级）](../../../世界/生物万族/index.md) 巢穴石缝里摘得的**金色椭圆果实**（拳头大·结在红色藤蔓中段·滴金色清香汁液·**暗紫色·评分 260**·类型 = 高阶植物果实 / 材料 / 恢复类食物 / 强化类物品）：

- **可兑换为世界之源**（稀有品）；脱离主体**两天后消散**（须尽快处置）。
- **用法① 直接吃**：永久 **生命 +100 / 法力 +100 / 火焰抗性 +15% / 寒冻抗性 +15%**。
- **用法② 炼制为强化类物品**：效果更强（苏晓炼金药剂学 Lv.20、懂药剂学，倾向调配而非直接吃——但需山洞 + 可信守卫专心炼制）。
- 任一用法食用 / 饮下后，**1 分钟内回满生命 + 法力**。
- 简介点破其生态：**太阳恩赐果实所在必有强大生物筑巢**，许多强大生物用其汁液喂幼崽、使其更强壮 / 成长更快（巴哈姆即如此）——疑与亚森曼部落「太阳信仰 / 血藤」秘密同源。
- **果核 = 原生之种**（八卷36章·苏晓切开发现）：果肉可食 / 可调配，**果核（原生之种）却绝不可食**。
