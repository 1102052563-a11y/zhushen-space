# 再生卵

> 虫族任务的关键物品。

## 再生卵（虫族 · 任务关键物品 · 八卷37、45章）

> 原属批次:已收录（截至第20章）

[虚空虫族](../../../世界/生物万族/index.md) 母巢里、**虫族女王身边**的卵——**与其说是卵、不如说是一种药物**（虫族特产·极宝贵·**虫巢每 50 年才产一颗**·当前共两枚）。它是**[血藤战士](../../../世界/生物万族/index.md) 不靠「回归太阳（自爆）」也能续命的生路**（却为族人所唾弃）。长老 [巴尔](../../../人物/巴尔.md) 不愿自爆而亡，遂借 [苏晓](../../../人物/苏晓.md) 之刀冒险深入母巢夺取它（已偷走一颗）；作为回报，过爆炸区后即把出红土区的「石桥罗盘」交给苏晓。八卷45章 [虫族女王亚](../../../人物/虫族女王亚.md) 把另一颗**用在了被苏晓打到濒死的盟友 [艾达](../../../人物/艾达.md) 身上**（透明卵入伤口·愈合烂掉的心脏使之重跳）。
