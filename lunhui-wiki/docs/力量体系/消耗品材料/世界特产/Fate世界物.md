# Fate/Zero 世界物

> Fate/Zero 世界的物品清单。

## Fate/Zero 世界物（十一卷11–16章）

> 原属批次:已收录（截至第20章）

- **鬼山羊皮肤**（蓝·任务物品·评分 70）：[言峰璃正](../../../人物/言峰璃正.md) 给的**柳洞寺「钥匙」**——可找到被隐藏的柳洞寺（晚 8 点后才生效）；「鬼山羊」并非生物、而是一种使魔。
- **圣水（92% 浓度）**（暗紫·消耗品·评分 255）：圣堂教会之物，可饮用（回 100% 生命 + 快速愈伤）或泼洒（对**黑暗系敌人** 500 神圣 + 190 真伤 + 强烈灼烧）——但「它的作用并不是饮用或泼洒」（暗示真正用途）。苏晓把它灌进 [黎明战吼](../../装备道具/index.md) 的空膛子弹当杀招。
- **英灵匣**（杀英灵掉落·**可随机抽取该英灵的任意一种技能或宝具**·杀英灵的额外厚礼）：品质越高、内含技能 / 物品越多。苏晓共得三只——**暗紫**（十一卷16章·杀 [迪卢木多](../../../人物/迪卢木多.md)·已开出「爱的黑痣」[封印卷轴](../../装备道具/index.md)）、**暗紫**（56章·杀 [百貌](../../../人物/哈桑·萨巴赫·百貌.md)）、**金色**（67章·杀本届最强英灵 [英雄王·吉尔伽美什](../../../人物/吉尔伽美什.md)·待开）。
- **忏悔之书（淡金 · 任务道具）**（十一卷30章·斩 [怨之孤儿·修斯](../../../人物/怨之孤儿·修斯.md) 掉落）：修斯专属掉落——**使用后天赋觉醒任务完成度 +8%**（苏晓用后 34%→42%·离开本世界即消失）。
- **圣水·中瓶（99% 浓度·淡金 · 消耗品·2 次）**（十一卷36章·圣堂教会功勋兑换）：比老神父先前给的「92% 圣水」更高级——饮用回 80% 生命 + 80% 法力 + 愈伤；或泼黑暗系敌人 800 神圣 + 300 真伤 + 强制净化 / 焚烧（评分 300·800 功勋）。
- **圣堂教会功勋（Fate/Zero 机制）**（十一卷21章起）：杀柳洞寺 [怨生物](../../../世界/生物万族/index.md) 累积的功勋（怨生物不掉宝箱·全靠功勋·苏晓炸平柳洞寺共得 **2219 点**），可在**言峰教会**老神父 [言峰璃正](../../../人物/言峰璃正.md) 处的清单兑奖（如灵魂结晶〔大〕1000 / 圣水中瓶 800 / [幽魂猎手](../../装备道具/index.md) 500 功勋）。
- **怨灵灰烬（20 克）**（金色 · 稀有道具 / 材料·十一卷48章·开修斯金色宝箱所得·评分 328·官价 13.2 万币）：[怨之孤儿·修斯](../../../人物/怨之孤儿·修斯.md) 仅存的证明。效果①（一次性）：使用后**召唤出怨之孤儿·修斯**（2 秒构建身体、存在 3~20 分钟·可精神操控其行动）——**修斯的强度与存在时间由召唤者智力属性决定**（苏晓智力 60·召出的修斯应不弱），关键时刻能救命；效果②（材料）：怨生物学高等材料，可施法 / 构建法阵 / 阻断魔力干扰。
- **生物阶梯药剂**（淡金 · 强化药剂·十一卷48章·开修斯金色宝箱所得·评分 310+·不可出售）：注射后**永久力量 +2 / 敏捷 +2 / 体力 +2 / 神经反射速度 +5%**，且**能强化 50 点以上的属性**（多数强化药剂卡在 50 点这道分水岭·此药无副作用、产出率极低）。苏晓注射后六维三主属各 +2、神经反射速度逼近 +20%（仅微微亢奋、被意志力压住）。
- **尤丕思放大镜**（蓝 · 特殊装备·十一卷48章·开修斯金色宝箱所得·评分 70·耐久 15）：圣堂教会之物。**情绪放大**（主动·耗 20 法力·冷却 1 小时）：对目标作意志力判定、未过则把其最强烈的情绪（贪婪 / 色欲 / 疯狂 / 愤怒等）放大一倍、持续 5 分钟。品级不高但功能特殊（简介调侃：别把正面对准自己、更别用它看太阳）。
- **死徒骨粉**（暗紫 · 消耗品·十一卷57章·老神父言峰璃正临终所用·评分 213·官价 360 圣堂教会功勋）：强大的「**续命**」道具——吞下后**加速身体机能坏死、待完全坏死后意识仍停留在尸身内 30 分钟不消散**；前提是须在半小时内返回轮回乐园、否则迎来的只有死亡。对独行侠意义不大、却是冒险团的爱物（团员濒死时用它续意识、30 分钟内拿到回归资格即可保命）。[言峰璃正](../../../人物/言峰璃正.md) 被腰斩后靠它撑到苏晓赶来、留下遗言后粉尘化。
