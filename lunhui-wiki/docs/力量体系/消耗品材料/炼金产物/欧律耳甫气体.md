# 欧律耳甫气体

> 苏晓炼金学产物。

## 🆕欧律耳甫气体（苏晓炼金学产物 · 非致命「毒气」 · 四十二卷46章）

> 原属批次:已收录（截至第20章）


一种炼金学气体产物，本用于**侵溶金属**——**无毒无害**，气体构成（氮 + 氢 + 甲烷 + 二氧化碳 + 氧 + 其他的特殊复合比）**与「屁」高度相似**。对常人吸入毫无感觉（人体微量金属不足以反应）；但若吸入者体内**金属物质过多**，就会引发眩晕、全身撕裂感、强烈呕吐（1~5 分钟消退）。**致命关键**：它绝不能与活力 / 生命特性的药剂结合——体内有金属者若吸入后再**注射恢复药剂**，二者剧烈反应必死。它属「非致命性危险」，科技侧侦测只会把它当成「和屁构成相似、特性不同的气体」而不预警——正是靠这一点，[布布汪](../../../人物/布布汪.md) 在时空之力交易中拔开试管（顺便真挤了个屁掩护），放翻了身体部分金属改造、又自注恢复药剂的天启机甲手 [天审](../../../人物/天审.md)（等于**用一针恢复药把自己毒死**）。
