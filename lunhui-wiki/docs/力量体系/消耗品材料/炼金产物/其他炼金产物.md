# 其他炼金产物（速查）

> 篇幅较短、未单独立页的条目，共 8 条。

## 胜利药剂（蓝 · 恢复品）

- 产地：轮回乐园 ｜ 蓝色 · 评分 **60**（蓝色为 31–70）｜ 类型：恢复品。
- **效果**：**瞬间恢复 70% 生命值**，并**解除中毒与诅咒**（可解的异常状态最高 Lv.10）。简介「胜利就在眼前，喝下它拼死一战吧」。
- 平时市价至少 1 万乐园币；**空间战争前职工者被强制降价**，苏晓仅以 **6000 币**购得（备战寄生兽空间战场）（五卷2章）。

## 命运之子的眷顾（紫 · 消耗品）

- 产地：斩·赤红之瞳 ｜ 类型：消耗品 ｜ 评分 **101**（紫色 71–150）。
- **效果**：将一件**白色 / 绿色**品级的装备或物品**提升一级**（上限 +1 级·提升后能力增强）；**须返回轮回乐园后使用**。
- 苏晓开 [赤瞳](../../../世界/任务世界/斩赤红之瞳.md) 掉的**紫色宝箱**所得（同箱还出 4300 币 + 灵魂结晶〔小〕×3）；他打算用它把绿色精品「[幸运女神之焰 +7](../../装备道具/index.md)」升为**蓝色装备**（四卷51章）。

## 芬里尔的精魄（精魄 · 战略物资）

- 产地：斩·赤红之瞳 ｜ 品质 / 类型 / 评分均显示「**？？？**」（权限不足）。鸡蛋大小的肉球。
- **精魄 = 稀缺的战略物资**，可出售给轮回乐园（标价 **20000 乐园币 + 灵魂结晶（小）×4**·苏晓判断实际远不止此价）。由超级危险种 [芬里尔](../../../世界/生物万族/index.md) 死后的蓝色宝箱产出——该箱诡异（先「空箱 + 芬里尔虚影狼头陷阱」、被黑洞吸入后才化成精魄）。苏晓不敢贸然食用（疑可增属性、但无保质期 / 安全认证）（四卷20章）。

## 炼金炸弹·太阳神（斩赤 · 道具）

- [夏洛克](../../../世界/任务世界/斩赤红之瞳.md) 藏于胸口的扁平炼金炸弹：**波及半公里**、引爆后如「太阳落地」、瞬间把土地树木高温蒸发成气体（地面玻璃化）；设定为**持有者身体机能遭破坏时自动引爆**（故苏晓没能直接斩杀夏洛克、改用炸弹反制）。苏晓另有海胆状的倒计时炸弹塞进夏洛克喉咙、令其被自己的太阳神炸碎（四卷66章）。

## 暗紫强化药剂（苏晓自制 · 八卷36章）

苏晓用 [炼金药剂学](../../技能/index.md) Lv.20、以贤者之石代材料，把**太阳恩赐果实**调配成的一瓶**暗紫色强化药剂**（评分 260+·乐园已公证·可出售但无人舍得卖）：饮后**永久 生命 +200 / 法力 +200 / 力 +1 / 敏 +1 / 智 +1 / 火焰抗性 +25% / 寒冻抗性 +25% / 神经反射速度 +4%**（神经反射对近战是「梦幻属性」）——比直接吃太阳恩赐果实强得多。苏晓饮下后法力涨到 **1363**，还触发了 [金焰女子](../../../人物/金焰女子.md) 的幻象（见其页）。

## 阳光精华（紫 · 恢复品 · 八卷45章）

[巴里](../../../世界/生物万族/index.md) 的「酋长包裹」开出的**液体形态的「阳光」**（紫·评分 97）：饮后每秒回 5 点生命、持续 3 分钟、并快速修复伤势。苏晓在生命剩 18% 的狼狈处境下饮下续命。

## 坚硬的甲壳（蓝 · 锻造材料 · 八卷45章）

精英刺蝎宝箱开出的**锻造防具上品材料**（蓝·评分 27）——存入布布汪背包待用。

## 查克拉修炼法（初级）（蓝 · 消耗品 · 九卷13章）

火影世界蓝色宝箱开出（评分 70·苏晓「闪光」开到 + 5000 币）：**使用后体质改变、可将身体能量与精神能量结合产生查克拉**（初级·效率略低）。每个忍者的宝箱都有低概率开出。在乐园交易广场**售价 20 万乐园币**（极抢手·查克拉是乐园最受欢迎的力量体系之一）——苏晓开到它视为「转运」之兆（同箱的紫箱却开成只有 1000 币的「空箱」·比闪光还罕见）。
