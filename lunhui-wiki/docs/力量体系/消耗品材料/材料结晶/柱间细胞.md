# 柱间细胞

## 柱间细胞（活性 95% · 淡金 · 材料 / 消耗品 / 血统类 · 九卷54章）

> 原属批次:已收录（截至第20章）

[苏晓](../../../人物/苏晓.md) 以卡卡西左眼 + 一桩私活从 [宇智波带土](../../../人物/宇智波带土.md) 换得的**原始柱间细胞**——初代火影千手柱间**生前**所提取（≠从尸体取的普通初代细胞·活性 / 侵蚀力极强·斑遗产，连带土皮肤一碰都险被吞噬树木化）。淡金色·评分 **305**。

- **效果**：①可用于研究；②**直接饮用** → 力 +20 / 敏 +10 / 体 +30·**300 秒后饮用者树木化**（死亡）；③**注入体内** → 60% 树木化速死 / 30% 力 +10 体 +10 + 局部树木化 / 6% 未知变异 / 3% 得千手一族体质 / **1% 得木遁血继限界**。
- **苏晓的用法**：他不要木遁，要的是细胞那**强大的生命力 / 恢复力**——拟借大蛇丸的研究资料**去除其中木遁之力**、把纯生命力融入炼金药剂（大蛇丸已成功过）。
