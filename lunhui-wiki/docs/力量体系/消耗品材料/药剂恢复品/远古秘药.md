# 远古秘药

> 炼金药剂学配方＋金色药剂。

## 远古秘药（炼金药剂学配方 + 金色药剂 · 十九卷93章·世界争夺战专属奖励）

> 原属批次:已收录（截至第20章）

苏晓海蓝星功勋兑换的**传说级炼金药剂学配方**（评分 500·掌握此配方者会成为各大势力争抢拉拢的人才）。

- **配方门槛**：炼金药剂学 Lv.20 + 掌握 ≥5 种配方 + 真实智力 ≥85 + 付 20 万币（苏晓全满足）。
- **远古秘药（金色品质药剂）**：每造耗 500 法力 + 材料（生命凝合精粹 / 虚无泉之水 / 诞生之嫩芽 / 圣橡树果实 / 水晶瓶·或 1 万币替代未知材料）。**效果 = 恢复与永久增益同步**：饮用后急速恢复 90% 生命值，并在之后 30 分钟内**永久 +20 生命值 + 1 点毒抗或随机元素抗**（评分 400）。「越喝越强」——但短时间连饮会产生药剂抗性、效果递减，苏晓评最佳用法是**受伤时饮、日积月累增身**。
