# 巨灵族

> ⚠ **⚠ 万界老好人**——⚠ **他们与小精灵族只差一点:**没找到靠山**（⚠ 小精灵族背后是灵魂书库·⚠ 像 [灾祸军团](../世界/势力/灾祸军团.md) 这个级别的势力不敢与灵魂书库叫板）。
