# 属性（六维）

契约者的六项基础属性，由 [烙印](../总览/轮回乐园机制.md) 的「个人信息」直观显示。**成年男性的标准属性为 5 点，幸运为 1 点**（第3章）。

| 属性 | 关联 |
|---|---|
| 力量 | 攻击力、负重等 |
| 敏捷 | 移动、攻击速度等 |
| 体力 | 生命值、防御力、异常抗性等 |
| 智力 | 法术伤害、法力值、感知等 |
| 魅力 | 社交、召唤等 |
| 幸运 | 物品开启、制造等（**极难提升**，乐园提示珍惜提升机会） |

> 苏晓推测：**魅力**关乎个人气质 / 亲和力（非长相美丑），杀气重者偏低；**幸运**若被扣成负数，处境堪比《死神来了》、随时有性命之危（二卷17章·推测）。

## 属性提升是非线性的
每提升 **1 点**属性都带来巨大变化，**绝非 1+1 那么简单**——苏晓力量由 6 升到 7 时「增长了一大截」（第10章）。现实世界提升属性**难于登天**：苏晓三年苦练仅 +1 力量、+2 敏捷（第15章）。

## 属性点
- 衍生世界**结算**时按评价发放的奖励之一；可提升**除幸运外**的任何属性（第22章）。
- 须在乐园的「**属性强化大厅**」内使用（见 [乐园 · 设施](../总览/乐园.md)）：付 100 乐园币开启强化仓，强化时**剧痛**，可选免费麻醉但会**影响效果**（第26章）。

## 亲和属性
- 每个契约者会选定**亲和属性**（近战选力 / 敏，法师选智……），建议优先提升；亲和属性达 **20 点**会触发额外奖励四选一（第26、二卷101章）：以苏晓力量达 20 为例——超威（力+2）/ 肌肉狂暴（+10% 攻速）/ 越战越勇（力量判定 +5）/ **不灭之息（+20% 挣脱控制几率）**（他选了末项）。
- 苏晓因 [灭法之影](技能/index.md) 传承，亲和属性**固化**为 **力量 / 敏捷 / 智力** 三种，且有特性：三者**相互差距不得超过 3 点**，否则较高属性无法在强化大厅继续提升——限制了快速成长，但促成多属性均衡发展（第26章）。

## 「80 点壁障」（十二卷83–84章）
属性涨到 **80 点**会撞上一道**硬壁障**，无法在强化大厅继续提升（[苏晓](../人物/苏晓.md) 体力率先到 80）。要突破须满足 **4 个条件**：① 达到三阶 ② 完成一次世界争夺战 ③ **突破身体极限** ④ 获得**强者之证明**。满足后再提属性可得「**80 点后的额外奖励**」（极稀有的属性）。

- **突破身体极限**：须在**现实位面**进行——向乐园申请后，乐园把契约者**五属性临时压制到 5 点、被动技能压到 Lv.5 效果**（部分技能如恶魔之印 / 气息外放直接完全封印·主动技能在现实本就不可用·似封印非剥夺·随从也被强制留在乐园），再发布任务，须以近乎普通人的状态完成历练。**任务最多发 3 次**，3 次均失败 / 放弃则突破 80 壁障告吹；时限约 3~5 天（无需大幅延长现实停留）。[苏晓](../人物/苏晓.md) 接到的是「**勇闯魔窟**」（自适应难度·潜入三国边境一座毒枭 + 偷猎者盘踞的危险小城、停留 30 分钟+·十二卷93章）。
- **强者之证明**：查询其来历需**更高的乐园权限**——须先**突破身体极限**才会获得相关情报（十二卷101章·苏晓达成）。它是**勋章类消耗品**，共三种获法：① 挑战高于自身一阶的强大剧情生物（挑战费灵魂结晶〔大〕×5·最简单也最贵）；② 参与「**黑渊历练**」（求生类·按苏晓实力生存率仅 4~6%）；③ 参与「**爱丽丝恶魔古堡游戏**」（需 [爱丽丝的邀请函](装备道具/index.md)·凑满 7 名契约者才开局·邀请函可花 20 万币 + 灵魂结晶〔中〕×15 兑·折中之选）。**此情报严格保密**：泄露会两次警告、再泄则全属性 -30% 并强制剥离相关记忆。
    - **苏晓的突破（十二卷98–101章）**：他在现实「勇闯魔窟」任务里濒死时**触发突破身体极限**（封印枷锁实为「引导术式」·激烈战斗 + 致命伤逼属性冲破压制、加速引导）→ 成功后属性恢复 + 得强者之证明情报。他正好持杀 [蝶](../人物/蝶.md) 所得的爱丽丝邀请函（80 点 + 突破身体极限后才激活可看），遂选第三条路、用掉邀请函等开局。
- **饮鸩止渴的捷径**：花 **60 万乐园币 + 灵魂结晶〔大〕×1** 买「**O 型能量机体蛋白注射液**」可硬突破 80 点——但**不可逆、无 80 点奖励、且此后每提 1 点属性要耗 9 点属性点**，后患无穷。

### 真实属性 / 主属性 · 次属性（十四卷73–75章·苏晓实证）

[苏晓](../人物/苏晓.md) 十四卷72章通关 [恶魔古堡游戏](../世界/任务世界/恶魔古堡游戏.md)、集齐四条件，**用「强者之证明」突破 80 壁障**，揭开此机制的真容（远比预想强）：

- **真实属性**：使用强者之证明后，**力 / 敏 / 体 / 智（主属性）转为「真实属性」**——① 此后世界结算**不再给普通属性点、而是直接给「真实属性点」**；② **1 点真实属性点 = 5 点普通属性点的强化效果**（且阶位更高·判定中真实属性碾压普通属性·相当于碾压 O 型注射器使用者「45 普通点」的效果）；③ **真实属性免一切环境压制**——普通属性进黑渊 / 虚空巢 / 高烈度战场会被压制 50~60%，真实属性无视。注意：**翻 5 倍是从突破后新提的属性算起**（即第 81 点起），已有的 80 点本身不翻倍；装备提供的「普通属性加成」**对真实属性失效**（堆装备撑属性者转化后只能卖装备）。真实 ↔ 普通可特殊转化、比例 **1:12**。
- **主属性 / 次属性 / 特殊属性**：传承 [灭法之影](技能/index.md) 起，苏晓的**力 / 敏 / 智即定为「主属性」**（每次升属性优先提主属性·这也是「灭法之影越到后期越强」之由）；用强者之证明后，**力 / 敏 / 体 / 智 = 主属性，魅力降为「次属性」，幸运 / 意志 = 特殊属性**（不变更）。**只有主属性达 80 才有「80 点奖励」**（故苏晓体力早先到 50 时没奖励·因当时体力非主属性）。**次属性若达 80 想再突破、须接受「二次 80 壁障考核」、难度是一次的 20 倍**——故选主 / 次属性须慎重。
- **苏晓的四主属性 80 奖励（各四选一）**：力「**赤血影**」（稀有·每损 10% 生命 → 攻速 / 移速 +8%、动态视力 +3%·叠 4 次·生命 ≥95% 失效 + 20 分钟冷却）、敏「**生灵切割**」（破防后近战附加「真实敏捷 ×3 = 240」无视防御伤·随敏捷成长）、体「**暴君**」（稀有·生命永久 +600）、智「**自然之心**」（提升感悟·对冥想 / 感悟极大加成）。仅此数项即让战力翻几倍。「能否获得强者之证明」是**强者与咸鱼最大的分水岭**。

## 装备加成是「外在」
装备提供的属性为外在加成，**本身属性不变**——失去该装备，加成的属性即减少（如摘下亡妻的项坠则力量 -1）（第10章）。

## 衍生值
- **生命值**：无法完全数据化，按受伤程度改变（第3章）。可被 [半数据化](../总览/轮回乐园机制.md) 显示为**血条**；脊柱等被毁可致「假死」（第10章）。
- **法力值**：= **智力 × 10** ＋ [噬灵者](天赋.md) 历次永久夺取的累计；**恢复速度随智力提升**（智 6 时 3/小时、智 12 时 4/小时）（第3、24章）。
