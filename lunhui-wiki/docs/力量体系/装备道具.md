# 装备·道具

> 随阅读更新。来源：乐园 / 任务掉落 / 拾取等。

## 装备品级
装备分为 **白色、绿色、蓝色……**（阶位不足时乐园不给出更高品级资料）（第4、19章）。

- **白色装备 / 道具**评分 **1–10**；评分越高价值越高；评分为 10 的白色装备会标注「**稀有**」，附加特殊属性，**甚至胜过部分绿色装备**（第4、7、15章）。
- **满评分规律**：满评分装备通常都不错（如白 10 的斩龙闪）；可推知满评分的绿 / 蓝装属性更强，且必存在更高级别装备（二卷4章）。
- **绿色装备**评分 **10–30**；评分为 30 的绿装标「**精品**」、附加特殊属性（如幸运女神之焰 30 分；恶灵咆哮 21 分为普通绿装）（二卷9、33章）。
- **首饰类**装备：稀少物品，**无耐久**、无需修理，多数任何人可穿戴（第10章）。
- **蓝色装备 / 物品**：评级在绿色之上，**评分 30–70**、非常稀少、价值远高；苏晓在 24 区后勤部首次见到蓝色物品（人体基因强化液）（二卷43、85章）。
- **库因克**（CCG 喰种武器）也分品级、可附带主动技能——如绿色精品库因克「粉碎精灵」（见下）（二卷43章）。

## 物品携带规则
- **宝箱所得**：基本都是**规则物品**，可带出任务世界（第12章）。
- **拾取 / 任务途中所得**：有**一定几率**可带出（如斩龙闪）；否则为本世界限定（如破旧的燧发枪）（第4、12章）。
- **触发类道具**：满足条件即触发效果（如引荐信）（第7章）。
- 世界内得到的部分现实物品（如缴获的匕首）无资料、不可带出、不可换乐园币（第6章）。
- **乐园内交易**：他人所有的待售物品无法带离 1 米、无法收入储蓄空间，须购买取得支配权（第24章）。

## 装备隐藏功能
花 **100 乐园币**永久开启，可自由隐藏穿戴中的装备（**武器类无法隐藏**）；隐藏后属性加成仍生效（第10章）。

## 宝箱
击杀世界内人物 / 生物可掉落宝箱。要点（第19章）：

- **品质递进**：白色 < 绿色 < 蓝色……；宝箱所开物品的等级与宝箱颜色相关。
- **开出什么与掉落者相关**：杀老虎多得虎爪 / 虎皮，杀人多得其遗物（杀黑市商人 → 引荐信、杀欧卡 → 项坠），不会从老虎宝箱里开出剑、盾。
- **白色宝箱**：最低等；曾开出乐园币、贝利、亡妻的项坠等（第5、7、10章）。
- **绿色宝箱**：杀科尔波山的主人的开出 **1000 乐园币 + [灵魂结晶（小）](消耗品材料.md) + [虎爪（绿·材料）](消耗品材料.md)**（第19章）；杀壁虎的开出 **900 乐园币 + 赫包（S）+ 恶灵咆哮（绿·戒指）**（二卷9章）。

### 开箱「闪光」（二卷33章）
开宝箱时，若开出**普通物品**则宝箱直接打开；若开出**高价值物品**，宝箱会**绿光 / 强光大盛**——苏晓称之为「**闪光**」。几率极小（他入乐园开箱无数仅遇一次），蓝色宝箱「闪光」更意味着价值数万乐园币的好物。苏晓推测这次闪光是「攒了太久的非酋运气」一次性回馈。

> 乐园币为**无形体货币**，直接存入烙印（第19章）。

### 库因克（见闻 · 零队）
CCG 用喰种赫包制成的武器，分品级、可附形态 / 技能（二卷44–46章）：

- **鸣神（S+）**：[有马贵将](../人物/有马贵将.md) 所用，三形态——炮形（大范围电流）/ 钳形（远程电球）/ 剑形（坚固近战）。
- **IXA（S+）**：有马贵将所用，多形态、攻防一体。
- **M.红月**：[尤娜](../人物/尤娜.md) 所用的库因克，狙击枪 / 镰刀双形态。
- **枭（SSS）**：CCG 制造的**最强库因克**，剑形（尖端细、护手处手臂粗、剑柄长），[有马贵将](../人物/有马贵将.md) 自用；曾许作苏晓助阵报酬，苏晓后弃之不要（二卷82、95章）。
- **赤舌（SS）**：法寺项介所用长柄大刀型库因克（由多多良哥哥的库因克制成）。
- **虚穴（S）**：法寺项介所用炮筒型库因克（羽赫制），发射可爆炸的橙黄色晶体。
- **白夜的库因克（绿 · 长刀）**：由店长（不杀之枭）的赫包所制，绿色品质、略胜斩龙闪；制造者为 CCG 实验员**地行甲乙**。苏晓叛 CCG 时从库克利亚实验室取回，但因换刀需适应（长短 / 重心 / 锋利度），暂未替下斩龙闪（二卷86章）。
- **新（铠甲类库因克）**：CCG 近期研发、篠原幸纪 / 黑磐岩等亦在用；**其材料取自被囚的喰种 [雾岛新](../人物/雾岛新.md)**——切开赫包后注高浓度 Rc 令其再生，得以量产，「新」即以他命名（二卷87章）。穿戴：力量 +3、敏捷 -3、防御 +2、移速 -30%。后颈开关启动**原形态（活化）**：力 / 敏 / 体各 +5、但**每秒 -2% HP**——「新」如发动机、以使用者血肉为燃油，**启动后须一分钟内脱下**，否则被吞噬成白骨（伤身、甚至折寿）。苏晓在 24 区危局首次借尤娜的「新」死战无眼喰种、用毕即弃（二卷49–51章）。其主材料是一位 **SSS 级赫者的赫包**，故「新」内 Rc 细胞含量是人肉的千万倍——独眼之枭曾吞下一块「新」碎片暴涨战力（二卷60章）。
