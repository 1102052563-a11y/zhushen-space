## 装备品级
装备分为 **白色、绿色、蓝色……**（阶位不足时乐园不给出更高品级资料）（第4、19章）。

- **白色装备 / 道具**评分 **1–10**；评分越高价值越高；评分为 10 的白色装备会标注「**稀有**」，附加特殊属性，**甚至胜过部分绿色装备**（第4、7、15章）。
- **满评分规律**：满评分装备通常都不错（如白 10 的斩龙闪）；可推知满评分的绿 / 蓝装属性更强，且必存在更高级别装备（二卷4章）。
- **绿色装备**评分 **10–30**；评分为 30 的绿装标「**精品**」、附加特殊属性（如幸运女神之焰 30 分；恶灵咆哮 21 分为普通绿装）（二卷9、33章）。
- **首饰类**装备：稀少物品，**无耐久**、无需修理，多数任何人可穿戴（第10章）。
- **蓝色装备 / 物品**：评级在绿色之上，**评分 30–70**、非常稀少、价值远高；评分 70 的蓝装标「**极品**」、附加特殊属性（如三笠的心意 70 分）。苏晓在 24 区后勤部首次见到蓝色物品（人体基因强化液）（二卷43、85章）。
- **紫色装备 / 物品**：蓝色之上的品级，**评分 71–150**——苏晓在斩·赤红之瞳的秘境得「帝具改造蓝本」（紫·89 分·见下「帝具」节）时**首次见到紫色品质**、由此得知蓝色以上即紫色（四卷33章）。
- **暗紫色装备 / 物品**：紫色之上的品级，**评分 151–260**（评分越高越珍贵·凤毛麟角·交易市场几乎绝迹）——苏晓六卷68–75章首次接触：**大剑豪的剑术心得**（暗紫·170）、**牛牛果实（长颈鹿形态）**（暗紫·259）（六卷68、75章）。**评分满 260 的暗紫装备会附加「特殊效果」**（如**神灵余晖**的「圣灵赐福」·见下「神灵余晖」）。
- **库因克**（CCG 喰种武器）也分品级、可附带主动技能——如绿色精品库因克「粉碎精灵」（见下）（二卷43章）。
- **完整品级阶梯（十五卷18章·苏晓总结）**：白 < 绿 < 蓝 < 紫 < 暗紫 < **蓝色套装** < 淡金 < 金 < **暗紫色套装** < **淡金色套装** < **金色套装** < **传说级**。可见「套装」整体高于同色单件、且金色套装之上即「**传说级**」——传说级单件可媲美一整套金色套装。成长类装备（如斩龙闪）不在此列、资源足够可一路升到传说级。
- **传说级宝箱 / 物品（十五卷18章·苏晓斩四代雷影首得）**：**传说级宝箱**呈暗金色（深沉厚重）——开出 80% 金色 / 10% 淡金套装件 / 8% 金色套装件 / **2% 传说级物品**。**传说级物品极稀有、一经使用即与灵魂绑定**（无法出售·死亡后有几率在猩红卡内保留）。
    - **传说之勋（传说级 · 材料 · 十五卷46章·开雷影传说宝箱所得）**：酷似黑色鹅卵石、巴掌大却重约百公斤——**锻造传说级装备的必需品**（评分 436·出售价 = 灵魂结晶〔中〕×6·苏晓首见能换灵魂结晶的物品）。产地限「开放性衍生世界 / 三阶以上战争世界」；苏晓暂留待研究——或是把斩龙闪推上传说级的关键。
    - **岚彻（淡金 · 重刀 · 十五卷45章·开达鲁伊淡金宝箱所得）**：达鲁伊的佩刀（101.6×22.3cm·攻 80~106·力 80 / 敏 50 / 刀技 Lv.25）——**查克拉传导金属 + 雷岩合金 + 陨石**打造。① 查克拉传导（被动·按注入的查克拉属性赋能：雷→锋利 +4 + 麻痹 / 火→攻 +15 + 灼烧 / 风→锋利 +5 + 斩速 +20% 等）；② 重刃（被动·命中过力量判定即击飞 + 麻痹 1~2 秒）。**苏晓不留刀、而要熔了提炼其中的「查克拉传导金属」**（火影世界最贵金属·传导查克拉几乎零损耗·云隐把这么大块打成一把刀属暴殄天物）——这堆金属足够把 [斩龙闪](装备道具.md) 推上**金色品质**、或换多把淡金武器。

### 成长类武器规则 + 魔镰·哈波迪雅（十五卷67–69章·苏晓斩 [蔷薇](../人物/黑瞳-契约者.md) 所得）

- **成长类武器铁则（苏晓由魔镰悟出）**：① 使用者死亡后**有 93% 几率掉落**（故苏晓的 [斩龙闪](装备道具.md) 也可能在他死后落入敌手）；② 每人**最多同时绑定 3 件**；③ **可熔炼、不可出售**（迄今没人舍得熔成长武器）；④ 绑定前无法收入储存空间、绑定后无法交易。蔷薇死后魔镰竟散金光（无主装备特征·普通装备会封禁属性并在半小时内消失）——这才让苏晓窥见此规则。
- **魔镰·哈波迪雅（金色 · 成长类双镰 · 强化 +9）**：产自中低烈度战争世界·费迪星，**陨落血天使的专属武器**（攻 123~167〔左〕/ 136~178〔右〕·需意志 40 / 力 63 / 体 65 / 镰刀类技能 Lv.20）。**左镰**「魔灵附体」（主动·近战技能 +8，但手持时每 60 秒强判一次属性、未过则被魔镰之灵夺意识——苏晓因排斥不可用）；**右镰**「嗜血礼赞」（被动·伤害 12% 转治疗·仅对生物）；组合技「血魔偶」（减 10% 生命上限召两具血魔偶·需「血之源」）；专属被动「血生之物」（吸生物血提品质·满 15 万血能可耗灵魂结晶〔大〕×15 升品质·现 12270/15 万）；已镶七宝石不可完整剥离。评分 386。
- **魔镰之灵·哈波迪雅**：一缕**有意识、会噬主**的武器之灵——绑定瞬间就钻进苏晓脑内想反客为主，被 [青钢影](技能.md) 当外来入侵意识弹出、惨叫缩回（每个血天使都得战胜它才能把魔镰转为专属）。苏晓假意逢迎稳住它、再扔进储存空间封印。**他的真打算是返乐园后用「[至尊锋刃](技能.md)」把魔镰喂给斩龙闪**（吞武器时极凶暴 + 青钢影可噬其意识·零风险）——根本不会拿噬主魔镰去战斗。
    - **结局·被斩龙闪吞噬（十五卷112章）**：返园后苏晓把哈波迪雅放出「透气」、用至尊锋刃祭献——哈波迪雅这才认出青钢影「**不是下位的同类、是来自虚空的怪物**」、惊觉苏晓竟是**灭法（之影）**「我居然想去算计灭法……」，自知必败却不肯在封闭空间自毁，临消散只留四个字「**你这骗子**」（苏晓当初「斩龙闪是下位武器」纯属夸大晋升需求的烟雾弹）。**她没贡献锋刃值、却触发了至尊锋刃效果2 / 3**——令斩龙闪换上「嗜血礼赞」、并觉醒**空间能力「龙影闪」**（详见上「斩龙闪·蜕变金色」），是比锋刃值更大的收益。

### 木之灵（传说级 · 万能装备 · 十五卷106章·开柱间传说级宝箱所得）

[苏晓](../人物/苏晓.md) 开 [千手柱间](../人物/千手柱间.md) 掉落的传说级顶级宝箱所得——**他的第一件传说级装备**（一团苹果大小、柔软如毛线球的翠绿缠绕树枝·耐久 103·评分 **526**〔传说级 400~530〕·需真实力 75 / 敏 70 / 体 80·绑定后与灵魂绑定·不可售、可熔炼分解）。它没有花哨能力、却条条简单粗暴，**几乎适用于所有契约者**：

- **木之活化（被动 · 防御形态）**：每分钟生命恢复 +130、生命 +1200、身体防御力 +40——单这三条就超过三件猎魔套总和，近乎给苏晓**提升了 2/3 的生存能力**（生存力超大型冒险团的主坦）。补足了「斩龙闪斩不动柱间」暴露的「敌人防御力高」短板。
- **木之野蛮（主动 · 攻击形态）**：木之灵包裹手臂 / 足部、消耗 15% 当前最大生命，下次攻击**用武器 = 力量 ×0.3、徒手 = 力量 ×3** 的攻击 / 击飞 / 击退（80 真实力量 ×3·一脚能把敌人脏器踢飞·冷却 10h）。
- **自由切换（主动）**：攻 / 防形态随意切换（冷却 1h）。
- **「万能」特性**：经乐园判定可拟形成戒指 / 项链 / 护臂 / 上下身铠甲等**任意穿戴位（喂养分还能成铠甲）**，也可拟成「**辅助装备**」（如 [命运救赎](装备道具.md)·不穿戴也享被动·辅助位共 3 件）——这一「万能」属性是传说级里仅 6% 几率出现的稀有属性。苏晓现把它拟成护臂（2/2·与王者御尊护臂）/ 下装、切防御形态生命直接 +1/3。
- **🆕强化至 +10（十七卷87–95章·氧化裂剧变装置）**：苏晓用 [氧化裂剧变装置](装备道具.md) 催化木之灵（双形态强化费贵·裂变随机·+8~10 仅 10% 几率）——运气爆棚直接出 **+10**（耐久 143·评分升至 **530** 满格传说级）。木之活化 → 每分钟回血 +150 / 生命 +1600 / 防御 +50；**木之野蛮**消耗降至 10%、徒手攻击升至力量 ×4，且 **+8 附加「断骨」、+10 附加「碎甲」**——一记侧踢令敌同时承受击退 + 断骨 + 碎甲。正是这一脚（配 [动能战靴](装备道具.md) 超负荷）成了苏晓在竞技场击败 [荒焚](../人物/荒焚.md)、逼退 [狂狐](../人物/狂狐.md) 的杀招（「挨了这一脚战斗就不用继续了」）。

### 寂灭公爵 +13（金色 · 重型狙击枪 · 十五卷110–111章·换梧桐冒险团所得）

- 镶 3 宝石：**衡序**（枪械基础技能阶位 +1·专精级以下→苏晓枪械精通 Lv.30 升**专精 Lv.30**·攻 +49%）/ **杀戮**（攻 +18~23）/ **狞笑**（爆头伤害 +25%·锁定基本只锁头）。

### Destroy·暗杀（D·暗杀 · 传说级 · 中大型手枪 · 十七卷102–103章·黑帆抵债所得）

- **数据**：攻击 **140~198**（手枪却完爆 [蜘蛛女皇](装备道具.md) 狙击枪·仅次寂灭公爵 +13）·12 发 7.62mm 氢合金弹·评分 **525**·需力 50 / 真敏 85 / 基础枪械 Lv.30。

- 简介：「当我锁定你时，你已是我的枪下亡魂——莱斯利·蒙哥马利」。子弹 7.62mm 氢合金弹**每颗 900 币**（苏晓一次性囤 300 颗）。

### 海楼石金属棍（半成品 · 仅海贼世界可用 · 十八卷27章·Dr.贝加庞克所造·寄存海贼世界）
约 1.2 米、几十公斤重的通体漆黑金属棍——掺了**海楼石**（海贼世界**恶魔果实能力者的最大克星**：一接触即快速脱力、果实临时被「封印」、元素化的身体变回血肉之躯，远胜武装色〔只捕捉本体〕与海水〔只使其无力、果实仍在〕）。当年苏晓在世界政府研究所偷藏海楼石、由 [Dr.贝加庞克](../人物/index.md) 帮他打造（半软禁期·二人私交深·黄猿盯着只准打「杀伤温和的金属棍」、不准打刀——世界政府知苏晓善刀）。**未经轮回乐园公证**（公证费太高 + 用途窄），故**只能在海贼世界使用、不可带出**（未公证物不销毁、寄存于该世界·苏晓再来即取回）。海楼石太硬、贝加庞克都无法二次改造，它**唯一能力就是「硬 + 封印果实」**。苏晓拿它当长刀使、专克自然系 / 元素化果实者——德岛海战中逼得 [卡塔库栗](../人物/夏洛特·卡塔库栗.md) 宁挨武装色一脚也不敢碰它。

### 饼干果实（超人系恶魔果实 · 紫~传说级 · 十八卷39章·开克力架宝箱所得）

一层**薄如蝉翼的透明薄膜**——由无数纳米生物组成（本身就是一种生物），原用于隔绝辐射 / 毒气 / 强酸腐蚀，因其生物特性能**隔绝海楼石**（苏晓贴身穿戴它接触海楼石镣铐时毫无脱力反应）。代价：穿戴时每分钟消耗 **10 点生命**（维持纳米生物存活）、脱离宿主后 30 分钟内自然分解。[苏晓](../人物/苏晓.md) 靠它演了一出「假降」大戏——配合龙影闪瞬移 + 物品收进储存空间「凭空消失」+ 故意装出的「脱力反应」，骗得卡普 / 黄猿 / 鹤中将**误判他是开发程度不高的「空间系恶魔果实能力者」**（海贼世界没有检测果实的装置·只能凭观察），从而安心用海楼石镣具押他、给了他海上脱身的机会。海军虽收走薄膜「研究」，但苏晓不看好他们能复制。

### 永恒水晶瓶（淡金 · 可持续恢复品 · 十八卷25章）
盛着「永恒泉」泉水的水晶瓶——**可分 3 次饮用**、每次回 **450 生命 + 300 法力**、并快速治愈脏器伤势；**自动补充**（每自然日补 5% 容量·湿润环境补 10%）。是苏晓在德岛海战这类长时间消耗战里的续航利器。

### 时空硬币（传说级 · 稀有道具 · 十六卷22章·完成隐藏任务「揍敌客家族的诅咒」A+ 四选一所得）

含时空之力的硬币（传说级·评分 **520**〔传说级 400~530〕·**仅 Lv.25 以上隐藏任务可得**·出售价「**???**」= 极高价的标志，类似当年的世界树指环）。效果：**可进入任意「试炼世界 / 战争世界 / 原生世界」**。[苏晓](../人物/苏晓.md) 在「永生之花果实 / 时空硬币 / 宗师之力(1/10) / 强制脱离豁免书」四选一中冒险选了它（宗师之力已得过、强制脱离他用不上、永生不诱人）。

!!! note "原生世界 = 野生的衍生世界"

    时空硬币能去的「**原生世界**」是苏晓最感兴趣的——它**不是轮回乐园的地盘、而是「野生的衍生世界」、从未有契约者去过**。未知即危险、却也意味着更大的机遇。是否把这枚「???」价的硬币卖给乐园（血赚）、还是留着去原生世界，苏晓视后续而定。

    - **非一次性消耗品（十七卷1章揭）**：十六卷70章苏晓嫌乐园派的 EVA（半崩溃）世界不适合自己，遂用时空硬币选「原生世界」——硬币爆开放出时空之力后**竟又重新聚拢、布满裂痕但还能再用几次**（原以为是一次性）。它带苏晓进了 [希亚大陆·灭法遗迹](../世界/任务世界/希亚大陆-灭法遗迹.md)（因价值远高于普通入场券，乐园按平衡机制补偿了 **16500 点世界功勋**）。

## 物品携带规则
- **宝箱所得**：基本都是**规则物品**，可带出任务世界（第12章）。
- **拾取 / 任务途中所得**：有**一定几率**可带出（如斩龙闪）；否则为本世界限定（如破旧的燧发枪）（第4、12章）。
- **触发类道具**：满足条件即触发效果（如引荐信）（第7章）。
- 世界内得到的部分现实物品（如缴获的匕首）无资料、不可带出、不可换乐园币（第6章）。
- **乐园内交易**：他人所有的待售物品无法带离 1 米、无法收入储蓄空间，须购买取得支配权（第24章）。

## 装备隐藏功能
花 **100 乐园币**永久开启，可自由隐藏穿戴中的装备（**武器类无法隐藏**）；隐藏后属性加成仍生效（第10章）。

## 宝箱
击杀世界内人物 / 生物可掉落宝箱。要点（第19章）：

- **品质递进**：白色 < 绿色 < 蓝色……；宝箱所开物品的等级与宝箱颜色相关。
- **开出什么与掉落者相关**：杀老虎多得虎爪 / 虎皮，杀人多得其遗物（杀黑市商人 → 引荐信、杀欧卡 → 项坠），不会从老虎宝箱里开出剑、盾。
- **白色宝箱**：最低等；曾开出乐园币、贝利、亡妻的项坠等（第5、7、10章）。
- **绿色宝箱**：杀科尔波山的主人的开出 **1000 乐园币 + [灵魂结晶（小）](消耗品材料.md) + [虎爪（绿·材料）](消耗品材料.md)**（第19章）；杀壁虎的开出 **900 乐园币 + 赫包（S）+ 恶灵咆哮（绿·戒指）**（二卷9章）。

### 开箱「闪光」（二卷33章）
开宝箱时，若开出**普通物品**则宝箱直接打开；若开出**高价值物品**，宝箱会**绿光 / 强光大盛**——苏晓称之为「**闪光**」。几率极小（他入乐园开箱无数仅遇一次），蓝色宝箱「闪光」更意味着价值数万乐园币的好物。苏晓推测这次闪光是「攒了太久的非酋运气」一次性回馈。

> 乐园币为**无形体货币**，直接存入烙印（第19章）。

## 帝具（斩·赤红之瞳 · 世界特有）
约一千年前，[斩·赤红之瞳](../世界/任务世界/斩赤红之瞳.md) 世界的帝国举国之力，以**超级危险种**（恶种恶龙、血统稀薄的恶魔等）的身体素材 + 稀有金属**奥利哈钢** + 大量失传秘术，由顶级匠师打造出 **48 件无法复制的武器，称「帝具」**；造毕后帝国兔死狗烹、处死工匠、技艺断绝，存世仅此 48 件（四卷12–13章）。

- **定位**：苏晓视帝具为「**削弱版的恶魔果实**」——各有千奇百怪的能力，部分涉及**空间**，少数甚至强于恶魔果实。
- **使用规则**（四卷21章）：帝具**择相性契合的主人**、非人人可用；**主人死后帝具能力仍在、等待下一任主人**。帝具使的通病是**身体素质相对不优**（能力强、肉身弱）。
- **猎杀者夺帝具的规则**（四卷40章）：杀帝具使可直接夺取其帝具，但**帝具无法带出斩赤世界**——若相性匹配可在本世界内使用、**回归乐园时自动变卖成世界之源**；想获得**能带离**本世界的帝具，须从杀帝具使后掉落的**宝箱**里开出。
