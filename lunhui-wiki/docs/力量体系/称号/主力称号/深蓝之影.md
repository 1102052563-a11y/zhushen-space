# 深蓝之影

> 苍龙大陆随机燃炼所得的战斗称号。

⚠ **本页合并了原页中散落 2 处的记载**，按卷序排列。

## 深蓝之影（苍龙大陆 · 随机燃炼的五星称号 · 三十九卷81章）


苏晓用攒的四星称号**随机燃炼**（不放主称号 + 额外付钱 → 出随机特性称号·300 灵魂钱币）燃出的**五星称号**（★★★★★·因随机燃炼只释放出 72% 潜力）——两效果二选一激活：**每点真实力量 → +0.5 肉体防御（上限 +94）** 或 **每点真实体力 → +30 生命（上限 +5610）**；**若提升星级可质变为「攻击特性」**（它本是「深蓝中的怒杀之影」·随机燃炼却成了防御特性）。苏晓要的是攻击特性（更快杀敌才是最好的防御）——这是他少数几个能**直接提升自身战力**的称号（多数称号如无冕之王要吃判定、行者挽歌提的是无用的魅力）。

## 🔵 深蓝之影（战斗称号 · 极限燃炼 1 次）

> 原属批次:续 · 称号极限燃炼机制、猎神者与太阳领主（52卷64章 / 53卷）


⚠ **苏晓想极限燃炼成最强称号的主目标**（⚠ **他感觉这称号继续提升今后必有关键作用，甚至比战争领主还强大许多**）:

| 深蓝之影 | 效果 |
| --- | --- |
| 适应性之力 | ⚠ 所有伤害阶位 +2 |
| **深蓝** | ⚠ **身体速度整体提升 58.76%**（⚠ 基础 20%＋真实敏捷 ×6%·高优先权） |
| 怒影 | ⚠ **对抗灭世级存在与深渊系敌人时额外 +20% 真实伤害** |

- ⚠ **⚠ 他不会拼那 8% 成功率**（⚠ **「一定要对自身心里有哔数，概率不够，数量来凑」**——⚠ 冲极限燃炼 2 次官方只有 80% 即实际拼 8%，太鲁莽）。
