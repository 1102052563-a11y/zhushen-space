## 🌑 深渊之影 → 500% 潜力（起源石·世界的隐性增益坐实）

> 原属批次:续 · 深渊之影500%潜力、猎神者极限燃炼4次与顶级至强称号（54卷66–80章）

- ⚠ **激活【起源石·世界】后，「深渊之影」的潜力疑似冲破 100% 的猜测被坐实——直接飙到 500% 潜力**——⚠ **⚠ 核心收益:基础肉体防御「等阶+7」**（⚠ 远超「团长285%」的称号潜力上限·⚠ 深渊纪元世界·埃伯亚思正是「深渊行走」特性的最强舞台·苏晓在深渊侵蚀区如鱼得水）;⚠ **⚠ 它仍是苏晓的「战斗专属称号」，与「老猎人〔满潜力单属性·无需佩戴〕」错位共存。**

## 🖤 【深渊之影】（称号·最高稀有度·潜力 600%）


| 效果 | 内容 |
| --- | --- |
| **深渊猎人（核心·被动）** | 对深渊系敌人造成的「任何伤害」**+40%**，且**击杀收益 +50%**<br>⚠ 击杀收益包括:「深渊任务·死之赋予」的任务完成度，甚至天赋能力的**「魂能·摄取量」**——而所欠缺的魂能量，将由此称号的公证方（轮回乐园）通过**收取所击杀深渊系的因果残存**来提供 |
| **盛怒之影（核心·被动）** | 豁免所有特性「黑暗腐化」的**因果侵蚀**（⚠ 不豁免其造成的物理/能量等体系损伤） |
| **深渊行走（核心·被动）** | 进入深渊后不受环境侵蚀，且**不会散发出「异乡人」的气息** |

⚠ **为何是它**:一旦「灭法天赋」真能凝聚魂能结晶，那么作为基础的凝聚耗材百分百是魂能。一般目标的魂能无论力量位阶还是因果层级都难免不够，而**通过击杀深渊系存在大量获取魂能，无疑是最高效的方法**（风险当然也奇高）。加之「深渊任务·死之赋予」必然是后续获得时空之力的主要途径——⚠ **优先级甚至比「猎杀悬赏·血契」更高**（因为不知下次任务世界是什么时候，通常触发不了猎杀悬赏，除非是大型战役）。
