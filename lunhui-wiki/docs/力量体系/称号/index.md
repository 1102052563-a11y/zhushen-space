## 正式称号系统 · 三阶竞技场称号（十七卷93–101章）

竞技场设有一套可佩戴的称号奖励机制。打上三阶竞技场**第十名**即获 **「胜利之荣耀（印记）」**（可随名次升级·三阶内有效）；击败第二名后此印记**可晋升为正式「称号」**。三阶竞技场共三种称号，**同一时间段内每种仅存一枚**（不是你的就是别人的、错过即永失·故竞技者之间不存在互刷）：

- **善战者（最强 · 三阶最强的代表）**——苏晓 [苏晓](../../人物/苏晓.md) 于十七卷100章登顶首位所得（夺自原持有者 [狂狐](../../人物/狂狐.md)）。**效果**：生命 +800 / 法力 +800 / 乐园设施费 −10% / **对契约者造成伤害 +12%**。获取门槛苛刻：须先打上第三名（拿到属性最强的「胜利之荣耀」印记）、再在**3 自然日内连挑第二、第一名**，**每 10 个衍生世界周期才出现一枚**；要**永久拥有**它须在保持首位的情况下**带它晋升四阶**（否则只是「使用权」）。它还能在战斗中**具现**（无攻防加成、但 80% 冒险团认得 = 三阶最强的招牌·见之多半逃）。
- **竞技之魂**（次于善战者）：以「龟速」打上第一名的奖励·每 5 个衍生世界周期生成一枚。
- **仰望之勋**：第二名的奖励·每 3 个衍生世界周期生成一枚。

胜利之荣耀印记随名次成长：第 10 名 = 生命 / 法力 +150·设施费 −6%；打到第三名时 = 生命 / 法力 +600·设施费 −9%（属性最强的印记形态）。打前十名竞技**禁赌斗**（因奖励机制特殊）；名次奖励另给普通属性点（10~5 名每场 +2、5~1 名每场 +3·苏晓借此「抢救」个位数的魅力属性）。**上一阶**对应物是一阶的「[竞技天王](../../力量体系/装备道具/index.md)」印记（晋阶时收回）。

## 🏅 荣誉商店 Lv.49（当前最高等级）

斩杀冥神一战让【老猎人】的荣誉经验从 0% 直接怼到 **1000%/1000%**，苏晓选择升级。

!!! note "等级上限机制"
    ⚠ **只能比历史最高纪录者高 1 级**——此前的最高纪录是 Lv.48，因此苏晓只能到 **Lv.49**；想要 Lv.50，就需要**下一个猎杀者**来做这件事。<br>⚠ 但到了最高等级，**Lv.49 与 Lv.100 已没有区别，都是权限全开**。<br>⚠ 每次荣誉商店升级都会**强制刷新**兑换列表，且无法通过「荣誉预留」等任何权限保留高价值物品。

**Lv.49 的全部权限**:

1. 在**任何区域**无次数限制召唤兑换列表并完成兑换；
2. 每 **10 个自然日**自动刷新一次（可关闭改为积攒次数），每次刷新有「中概率」触发**高效刷新**，刷出多件高价值商品甚至「星界奇物」；
3. 更多**「星界奇物」**加入刷新范畴，小概率出现，因其珍稀性**仅可通过多枚「轮回徽章」兑换**；
4. ⚠ **新增「权限类物品」刷新**——任务失败豁免权限、额外烙印等级提升权限，甚至**「任务完成凭证」**这类超高阶权限；
5. 全体猎杀者的荣誉商店兑换价降低 **0.5%**（已进入轮回徽章兑换阶段的商品不享受）；
6. 全体猎杀者获得 1~3 次免费刷新（按其等级而定），苏晓本人获 **20 次**。

⚠ **专属珍贵奖励（等级最高者独享）**:**你的荣誉商店内刷新出的「星界奇物」，仅需一枚「轮回徽章」即可兑换。**（可惜【初始的生命源泉】是升级前兑换的，否则立省 104 枚徽章。）

**对苏晓的三大用途**:① 兑换「初始源质」；② 刷出最顶尖的「星界奇物」（⚠ 荣誉商店出品的星界奇物**基本都默认最顶尖**）；③ 刷出高阶权限，比如「任务完成凭证」。
