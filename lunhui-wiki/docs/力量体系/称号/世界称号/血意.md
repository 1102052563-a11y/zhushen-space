# 血意

> 斩血魂所得。

## 血意（画中世界 · 斩血魂所得 · 四十四卷41章 · ★★★★★★★）


苏晓在沙之世界斩杀 boss **[血魂·暗魔之影](../../../世界/生物万族/index.md)** 后所得的**七星称号**（另 +3227 灵魂钱币 + 不朽级宝箱·暗魔之影 + 5.42% 世界之源）。血魂之战按特殊事件把伤害贡献百分比化（白夜 49.62% / 罪亚斯 21.59% / 伍德 20.77% / 莫雷 7.49% / 莉莉姆 0.53% / 月使徒 0%），宝箱与称号归贡献最高的苏晓。一见「血意」之名，他就隐隐觉得这称号很适合自己（他战力大量倚仗血气 / 血魂之力）。**🆕效果·血泉意涌（44卷64–65章激活）**：被动——提升鲜血系 / 血气系能力时，可大幅额外提升鲜血质量、或血气的气息强度。它不直接加战力，却极契合苏晓——他惯于去古战场吸血气来提升气息，有了血意可**事半功倍**（气息外放每升一级原本 +1 气息强度、佩戴血意时变 +2~3）。它是苏晓第 4 枚七星称号（另有 [无冕之王](../index.md) / [战争领主](../index.md) / [世界侵入](../index.md)），苏晓盘算再凑 2 枚七星（世界侵入抵两枚副称号）即可把战争领主 100% 燃炼到八星。
