# 精灵死敌

> 月光战役所得。

## 精灵死敌（苍龙大陆 · 月光战役的五星称号 · 三十九卷27章）


攻破「不动之城」月光要塞的**月光战役结算**所得**五星称号**（★★★★★）——效果单一但对口：**与精灵族战斗时压制对方全属性 + 随机封禁其一种能力**（对精灵极强、但比起提升战争领主就不值一提）。苏晓的算盘是把用不上的称号全合成——集齐后连同精灵死敌**还差 3 枚五星**，即可把 [战争领主](../index.md) 从五星燃到六星、再用 [追梦人](../index.md) 升七星。他前期专注战功榜刷 [猎龙者套装](../../装备道具/index.md)（圣灵近战套）、攒够称号后再借主线第二环期限去找虚空之树果实。（另攻新月要塞得**勇者之魂 / 要塞破坏者 / 月食**三枚四星、攻古树要塞得**精灵之颤栗**五星、燃出**血色统领** + 无特性五星称号 ×2 作素材。）
