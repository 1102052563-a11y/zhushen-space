# 团队屠戮者

> 圣杯世界的惩戒性称号。

## 团队屠戮者（圣杯世界 · 惩戒性临时称号 · 二十二卷31章）


苏晓在 [第五次圣杯战争](../../../世界/任务世界/fate-第五次圣杯战争.md) 三刀斩杀 [血门冒险团](../../../势力/index.md) 团长 [斯坦](../../../人物/斯坦.md) 后，被轮回乐园**强制佩戴**的临时称号——这是乐园对大型冒险团的**保护机制**：

- **强制佩戴·持续三个衍生世界**（无属性·无法解除 / 交易 / 销毁·三世界后自动消失·期间不能用善战者 / 旅行者称号）。
- **效果**：佩戴期间与任何契约者发生冲突时，**敌方会收到警示、并得知「团队屠戮者」称号的由来**（即此人宰过一个大型冒险团的团长）。
- **实际作用**：威慑——其他契约者一听对方杀过团长，多半转头就逃，反而帮苏晓减少了很多麻烦（他不主动惹人、违规者除外）。
