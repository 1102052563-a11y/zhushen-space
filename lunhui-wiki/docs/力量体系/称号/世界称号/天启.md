# 天启

> 塞尔星用于伪装烙印的称号。

## 天启（伪装烙印封印称号 · 四十五卷57–62章·塞尔星）


[苏晓](../../../人物/苏晓.md) 在塞尔星防御战首战重创敌军、晋「预备役战斗天使」——他伪装的天启烙印随之从「盗版」刷成「合法」，被轮回乐园封存解析、以特殊称号「天启」的形式还给他。戴上它可 **100% 伪装成天启乐园契约者**（侦测装备 / 能力都识不破·满足条件甚至能短暂进天启乐园·但不可用其基础设施）。入侵者身份暴露后此烙印本会被标注「黑户」，故被封在称号里:与天启方契约者战斗时，可生成轮回乐园主导的「**封境**」把对方拉进去——在封境内击败（不必击杀）对方，即以称号内的「初始烙印」暂时融合 / 夺取其烙印。两种用法:①杀掉对方→其烙印被轮回乐园收取分解（自己的伪装更难识破）；②留活口→用完解除称号、对方脱困拿回烙印、期间苏晓做的事全算他头上（背锅）。他借此想重开天启世界联络平台钓提款姬 / 套情报。
