# 世界侵入

> 时空之力交易奖励。

## 世界侵入（暗星·时空之力交易奖励 · 四十二卷49章 · ★★★★★★★）


苏晓完成暗星时空之力争夺（交付时空之力存储装置 + 物资箱 ×2）后所得的**七星称号·稀有**（另 +6952 灵魂钱币）——「胜利的纪念品」：

- **称号效果**：激活后可**强制侵入任何世界**！侵入后无主线任务、可获宝箱、并根据世界情况判定是否可获世界之源。**用一次即破损消失**（一次性）。
- **两种用法**：① 单独进入某世界不算太强，但「能侵入任何世界」就很强——若有非杀不可的仇敌，[航海罗盘](../../装备道具/index.md) + 世界侵入的组合即可追杀到底；② 作**副称号燃炼**极有用（内含特殊能量庞大·**一枚顶两枚同星级副称号**）——苏晓盘算把 [战争领主](../index.md) 从七星燃到八星，只需再消耗 [无冕之王](../index.md) + 世界侵入两枚七星即可（本相当于三枚）。
