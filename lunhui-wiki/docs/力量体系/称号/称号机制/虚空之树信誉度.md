# 虚空之树信誉度的最终形态

> 被单独重定为 ±10 的专属体系。

## 📉 虚空之树信誉度的最终形态


- 操作后:⚠ **信誉度达到 −?????（前所未有）**；
- 强制结算本次战役综合评价 → 获得 **1 亿点**信誉度；
- 因原有值太负，最终恒定为 **9 点**（正数）；
- ⚠ 随即虚空之树判定「你的信誉度波动过于剧烈，且你在深渊战役中有至关重要的贡献」，为他**重新评估并单开一套专属体系**:

!!! note "苏晓专属的虚空之树信誉度体系"
    ⚠ **上限 / 下限为 10 点与 −10 点**。<br>**10 点 = 极度良好，等值于 9999 万点信誉度的权限**；<br>**−10 点 = 极差，等值于 −9999 万点信誉度的裁定力度**。<br>⚠ 数值区间虽小，但波动不会像以往那么大——**与之相对，这一套专属公证代表后续肯定没有操作空间了**。

⚠ **另一条被堵死的路**:深渊战役中途，虚空之树已发出【警告:检核到猎杀者对悬赏任务的共享行为，**本次深渊战役已禁止共享悬赏任务**】——上一批那套「把悬赏共享给自己」的操作，仅此一次。（⚠ 苏晓后来还试过「申请接受一个悬赏两次」，不仅被拒，还收到两条警告，其中一条是**「不可利用队伍公证进行非常规操作」**。）
