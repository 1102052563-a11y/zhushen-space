## 🎯 「猎杀悬赏·血契」的收官


深渊战役期间触发的四个悬赏（⚠ 全部接受后**悬赏额度满额**）:

| 悬赏 | 内容 | 额度 |
| --- | --- | --- |
| **门之节点** | 破坏三座门之节点（激活费 60000 盎司） | 150 倍 → **900 万盎司** |
| **终极猎手** | 灭杀三只**超巨型·上古深渊存在**（激活费 45000） | 200 倍 → 900 万盎司。⚠ 苏晓最终**连戮四只**——「刃之域」转化出的血枪简直是这类敌人的杀手，**深渊阵营后来干脆不再往战场上派超巨型上古深渊存在** |
| **无上龙骑** | 斩灭深渊阵营的龙骑无上（激活费 250000） | 200 倍 → **5000 万盎司** |
| ⚠ **渊之主** | 斩灭深渊之主（激活费 **100 盎司**） | ⚠ **此为「猎杀名单·血契」的最终悬赏**——完成后悬赏名单被填满，获得 **「最终悬赏金」** |

!!! success "最终悬赏金"
