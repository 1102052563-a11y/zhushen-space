# 功勋值

> 深渊战役的新增机制，苏晓最终拿到 105.47 亿点。

⚠ **本页合并了原页中散落 2 处的记载**，按卷序排列。

## 🎖 深渊战役新增机制:功勋值


- **来源**:击杀深渊阵营单位。⚠ 击杀施法者之所以在深渊战役未开启时就能获得功勋值，是因为**吞噬自然元素壮大自身的施法者们，无疑是本次深渊浩劫的主要元凶**（灭掉整个奥术永恒星共得 **1295 点**，其中斩疯王 716 点）。
- **用途**:在各乐园阵营的**功勋商店**兑换真实属性点、技能升级仓使用权限等；⚠ 但战区内的功勋商店消耗**功勋**，乐园阵营内的功勋商店消耗**时空之力**（因几大乐园都在战场投入大量时空之力维持深渊净化装置）。
- ⚠ **苏晓的困境与机遇**:他的虚空之树信誉度是 **−?**，激活不了大本营的虚空之树·功勋商店，因此**花不掉功勋值**——他索性全部积攒，**留到最后来一笔史无前例的结算**。若最终这笔结算由虚空之树发放，他还打算「找机会看能否再操作下，让这收益翻倍」。
- **风海首胜的个人结算**:完整无瑕的时空石 ×10（值 500 万盎司）／星辰辉石 ×2／最高级强化石 ×4／**【机灵的叽里咕噜】**（命运奇物）／⚠ **随机最高位权限**——抽到「天启乐园所有设施费用消耗减免」，因苏晓是轮回乐园猎杀者而被转换、再被轮回乐园变更为 **「乐园设施使用费用递增减免」**（即对某一乐园设施使用后，该设施的使用费用**不再因提升幅度的升高而上涨**，超出部分**全部由天启乐园补正**）。
- **烈阳星的个人结算**:128532 点功勋值＋烙印 **+5 → Lv.110（烙印极限等级）**。⚠ 收益远逊风海——既因风海几乎是必败之局，也因为**这次对深渊阵营的打击不够大**（风海的深渊之门全开输送了差不多两天，烈阳星只全开了不到一小时）。

# 续（第54卷231–248章 · 终章）· 功勋与信誉度的终局

## 💰 105.47 亿点功勋值


**【战役任务·胜利者】**完成后:当前持有的 5859 万点功勋值 **×185 倍**（倍数按决战贡献而定）→ ⚠ **本次深渊战役总计获得 105.47 亿点**。

**这个数字的来源**（原文自列的四条战绩）:

1. 斩深渊化身，消灭深渊族裔五大家族族长；
2. 用元素虫族在风海大陆力挽狂澜，**连赢下四场主战场的胜利**；
3. 斩杀深渊无上·暗潮之影·艾薇拉；
4. 前往深渊，斩杀星界史上最强深渊之主。

⚠ **问题**:苏晓的虚空之树信誉度是 **−?** 点，**激活不了虚空之树的功勋商店**。而战役公证结束进行全部结算时他的信誉度肯定会很高——**可到那时功勋值已清零**（这是本次深渊战役独有的公证收益）。
