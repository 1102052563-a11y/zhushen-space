# 与凯撒的最后一次合谋

> 兑空虚空之树藏库的那次合谋。

## 🎭 与凯撒的最后一次合谋


⚠ **两人所做的一切都符合虚空之树的公证**，流程如下:

1. **苏晓申请「判定补正」**（理由:信誉度过低导致这笔功勋值处于资源不可用状态）→ 必定被拒；
2. **转向轮回乐园申请**——凯撒把这一招称为 **「轮回大爹为我做主」**。经此一番操作，苏晓依然用不了虚空之树的功勋商店，⚠ **但他手头的功勋值已从「仅虚空之树公证」变成「虚空之树／轮回乐园双重公证」**；
3. **把功勋值转让给凯撒**；
4. ⚠ 凯撒**一点操作都没有**——他把往日积攒的各类折扣权限全部招呼上去，硬生生把 105.47 亿点兑换光。⚠ **最终荣誉商店出现「资源售罄」的提示**——看来这是对虚空之树藏库进行了持续的暴击。

⚠ 事后原文补了一句:深渊阵营惨败后，**下一个危险其实是虚空之树的资源藏库**。
