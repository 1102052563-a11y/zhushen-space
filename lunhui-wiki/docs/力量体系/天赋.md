# 天赋

> 随阅读更新。

天赋是契约者的核心特质之一。**猎杀者**的天赋更适合战斗（第2章）。部分天赋为**成长天赋**——可随契约者成长而增强（苏晓因天赋为成长天赋，濒死时生存时间被乐园追加两小时）（第1、2章）。

## 已收录（截至第20章）

### 噬灵者（**已觉醒 · S 级**）
- **持有者**：[苏晓](../人物/苏晓.md)
- **类型**：杀戮天赋 / 成长天赋·**已于十一卷34–35章首次觉醒为 S 级**（首次觉醒的最高等级·SSS 须二次觉醒）。
- **效果（觉醒前·第3章起）**：杀死目标后永久夺取 **1–15 点法力值**；每个世界上限 **100 点**。
- **效果（觉醒后·S 级·十一卷35章）**：杀死目标永久夺取 **1–30 点法力值**（上限 +15）+ **额外恢复同等数值的法力值与生命值**；每世界夺取上限提至 **200 点**，**夺满即「完全激活」、此后杀敌恢复效果再 +50%**。——这彻底解决了苏晓「独行侠最怕人海战术、围攻时缺法力 / 受伤」的死穴：面对人海**只要体力不耗尽就永不倒下、杀越多回得越快**。
- **觉醒过程（十一卷33–35章）**：达成完成度 **95%（极为优秀）→ 天赋觉醒度 43%**（首次觉醒上限 45%·绝大多数契约者完成度不足 40%）。觉醒在意识空间里重历「**7~8 岁被卡车撞、天赋种子激活 20% 修复内脏脱离死亡**」的往事（**苏晓是「后天觉醒天赋种子」者**·比先天天赋弱、但不幸的人生反让他的天赋更强）；再走黑铁长廊到三叉路口「**命运由你自己主宰**」——三条路 = 法师（法杖）/ 法坦（盾）/ 近战（滴血利剑），他毫不犹豫选**近战**（成功 70% / 死亡 29% / 变异 1%）。觉醒时来自细胞深处的剧痛险些崩溃身体，全靠 **[灵影体质](技能.md) 与「永久加法力」天赋的共生**（淡红能量稳定身体）+ 极高意志力扛了过去。觉醒时**人极度虚弱、须在安全点**（伪 Berserker 间桐雁夜趁机发狂、被青钢影电退）。
- **触发下限**：目标太弱（如普通大鱼）达不到标准则不触发——否则踩死一窝蚂蚁即可刷满上限（第16章）。加成量按敌我实力差判定，**杀实力相当的强敌加得最多**（如寄生兽世界连杀 16 条 BOSS 级变种马来鳄，单只 +9~12、一举把法力冲到本世界上限 713）（五卷13章）。
- **纯净法力（衍生特性）**：苏晓的 [绝魔体质](技能.md) 像天生过滤器、隔绝一切元素，使其法力值**异常纯净**（法师乃至常人的法力或多或少掺杂元素）。代价是无法学法术，回报是纯净法力**让他造的 [炼金炸弹](消耗品材料.md) 威力更强**（爆炸伤害高、范围广·别人造威力要降三分之一·故他的炸弹在乐园畅销）（五卷12章）。

## 天赋觉醒（重大机制·十卷25–26章）

苏晓开出 [领主宝箱](装备道具.md) 得 **天赋之书（第二、三页）**（见 [消耗品材料](消耗品材料.md)）后，向乐园问询「觉醒天赋」，凭猎杀者权限调出资料：

- **天赋分类**：普通天赋、稀有天赋、特殊天赋、杀戮天赋等；天赋能力为 **E 级～SSS 级**，**觉醒前无等级**。
- **觉醒后**：天赋显示对应级别，并**增强现有能力或开拓新能力**。
- **觉醒四条件**：① 任意裸妆属性达 50 点（苏晓已达成）；② 获得三次 **S− 或以上**通关评价（已达成）；③ 集齐 **天赋之书（完整）**（苏晓 2/3）；④ 完成**天赋觉醒任务**（未达成）。前三条满足后才激活第四步。
- **天赋觉醒任务**：与晋升任务不同，**可在任意衍生世界执行**；难度 = 猎杀者等级 +10（苏晓 LV.14 → 任务为 LV.24·猎杀者等级越高越难，故宜在二阶至三阶尽早完成）。
- **危险性**：超过 50% 契约者无法觉醒天赋。
- **强制性**：杀戮天赋务必觉醒，乐园会在猎杀者达**三阶**后发布强制觉醒任务。
- **保密**：乐园警告「猎杀者不可透露觉醒天赋相关事宜」。

!!! note "苏晓的盘算"

    觉醒能增强 **噬灵者**（青钢影太耗法力、每世界夺 100 点已不够用）。他打算趁二阶至三阶间风险最小时完成。

!!! success "天赋觉醒任务已激活（十卷80章触发 → 十一卷5章激活）"

    全职猎人通关返园后，苏晓用 **灵魂结晶（大）×1** 换得天赋之书**第一页**、集齐三页合成「天赋之书（完整）」、达成前三项前置 → **进入 [Fate/Zero 世界](../世界/任务世界/fate-zero.md)（第11世界·特意挑的不大世界）即正式激活**（提示为**血红色**·严禁外泄）。机制：**共三环、难度持续攀升、最终环上限 Lv.26**；给三次机会（连续三个衍生世界各失败一次则天赋永久无法觉醒）。觉醒后天赋显级别 E~SSS（**最终级别看任务完成度、而非血统**）。**第一环「抵达柳洞寺」**（Lv.3·4 自然日·罚扣一次觉醒机会、本世界不再发布）——柳洞寺是冬木市地下大灵脉之上的灵地、被圣堂教会用结界藏起。

!!! success "天赋觉醒任务三环完成（十一卷20–31章）"

    - **第一环·抵达柳洞寺**（已完成）→ **第二环·灵魂灯盏**（Lv.21·进柳洞寺最深「主供殿」找到灵魂灯盏〔以灵魂结晶为灯芯、被一具白骨吸食〕·已完成）→ **第三环·破坏灵魂灯盏**（Lv.26·30 分钟·「威胁藏在暗处」）。
    - **真正的考验在「破坏之后」**：苏晓远距离炼金炸弹炸碎灵魂灯盏（连同那具普通白骨）后，柳洞寺内外瞬间罩起**10 层结界**、把主供殿变成牢笼——逼出**领主级 [怨之孤儿·修斯](../人物/怨之孤儿·修斯.md)**（永生池里被打断进化的存在）。苏晓险胜斩杀修斯，**天赋觉醒任务完成**。
    - **⚠️ 完成度机制**：任务完成后**五小时内的举动决定最终「完成度」**——完成度越高、觉醒后天赋等级越高（E~SSS）。**首次觉醒最高只到 S 级；SSS 须「二次觉醒」**（需极强运气 + 实力·非凑任务 / 道具可得）。苏晓初始完成度仅 **34%**（约 B 级），遂**用 [太阳神阿波罗](消耗品材料.md) 把整座柳洞寺夷为平地**（杀光怨生物·世界之源狂飙至 65.2%）以拉高完成度。觉醒时人会**极度虚弱、须在安全点进行**——苏晓尚未正式觉醒噬灵者。
