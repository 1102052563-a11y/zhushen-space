# 天赋

> 随阅读更新。

天赋是契约者的核心特质之一。**猎杀者**的天赋更适合战斗（第2章）。部分天赋为**成长天赋**——可随契约者成长而增强（苏晓因天赋为成长天赋，濒死时生存时间被乐园追加两小时）（第1、2章）。

## 已收录（截至第20章）

### 噬灵者（**已二次觉醒 · SSS 级**·二十八卷37–40章）
- **持有者**：[苏晓](../人物/苏晓.md)
- **类型**：杀戮天赋 / 成长天赋·**已于十一卷34–35章首次觉醒为 S 级**（首次觉醒的最高等级·SSS 须二次觉醒）。
- **效果（觉醒前·第3章起）**：杀死目标后永久夺取 **1–15 点法力值**；每个世界上限 **100 点**。
- **效果（觉醒后·S 级·十一卷35章）**：杀死目标永久夺取 **1–30 点法力值**（上限 +15）+ **额外恢复同等数值的法力值与生命值**；每世界夺取上限提至 **200 点**，**夺满即「完全激活」、此后杀敌恢复效果再 +50%**。——这彻底解决了苏晓「独行侠最怕人海战术、围攻时缺法力 / 受伤」的死穴：面对人海**只要体力不耗尽就永不倒下、杀越多回得越快**。
- **觉醒过程（十一卷33–35章）**：达成完成度 **95%（极为优秀）→ 天赋觉醒度 43%**（首次觉醒上限 45%·绝大多数契约者完成度不足 40%）。觉醒在意识空间里重历「**7~8 岁被卡车撞、天赋种子激活 20% 修复内脏脱离死亡**」的往事（**苏晓是「后天觉醒天赋种子」者**·比先天天赋弱、但不幸的人生反让他的天赋更强）；再走黑铁长廊到三叉路口「**命运由你自己主宰**」——三条路 = 法师（法杖）/ 法坦（盾）/ 近战（滴血利剑），他毫不犹豫选**近战**（成功 70% / 死亡 29% / 变异 1%）。觉醒时来自细胞深处的剧痛险些崩溃身体，全靠 **[灵影体质](技能.md) 与「永久加法力」天赋的共生**（淡红能量稳定身体）+ 极高意志力扛了过去。觉醒时**人极度虚弱、须在安全点**（伪 Berserker 间桐雁夜趁机发狂、被青钢影电退）。
- **触发下限**：目标太弱（如普通大鱼）达不到标准则不触发——否则踩死一窝蚂蚁即可刷满上限（第16章）。加成量按敌我实力差判定，**杀实力相当的强敌加得最多**（如寄生兽世界连杀 16 条 BOSS 级变种马来鳄，单只 +9~12、一举把法力冲到本世界上限 713）（五卷13章）。
- **纯净法力（衍生特性）**：苏晓的 [绝魔体质](技能.md) 像天生过滤器、隔绝一切元素，使其法力值**异常纯净**（法师乃至常人的法力或多或少掺杂元素）。代价是无法学法术，回报是纯净法力**让他造的 [炼金炸弹](消耗品材料.md) 威力更强**（爆炸伤害高、范围广·别人造威力要降三分之一·故他的炸弹在乐园畅销）（五卷12章）。

- **🆕二次觉醒（启动·二十八卷1章起·进行中）**：苏晓用 **觉醒之书·二次**（见下）启动噬灵者的二次觉醒（目标升 SSS）——**86% 契约者已放弃二次觉醒**（难度随阶位 / 等级提升）；噬灵者已达**首次觉醒极值** → 二次觉醒难度再 +30%。轮回乐园给三个匹配世界三选一，苏晓选 **[月之神](../世界/任务世界/月神大陆-黎明帝国.md)**（原生世界·有神灵存在）。本世界内**杀敌仍持续激活噬灵者攒法力**（斩鹰侯永久 +23、斩魂灵剑·曼永久 +50）。**🆕二次觉醒完成 → SSS（二十八卷37–40章）**：苏晓弑古神 [月神](../世界/生物万族/index.md) 得「噬魂之本源」、入意识空间面对「灵魂王座」二选一（**掌灵魂**速成但会剥意识 / **噬魂**靠自己变强无副作用），选噬魂、噬灵者升 **SSS**。新效果——杀敌可二选一剥「灵」（永久 1~60 法力·每世界上限 +500·并回血回蓝）或剥「魂」（永久 1~3 灵魂强度·每世界上限 +30）；**主动效果**：不剥离时可击碎敌灵魂、偷取部分技法 / 知识（每世界 3 次·不符可弃）。更关键：实测 [灵魂锁灯](技能.md) 里的灵魂晶碎可**直接吞噬提升灵魂强度**（跳过剥离选择）——苏晓由此定下「杀敌专剥『灵』攒法力、灵魂强度靠灵魂锁灯晶碎」的双收益打法（灵魂强度 60·下世界可 +28）。

### 二次觉醒天赋（启动·二十八卷）

二次觉醒是天赋从 S 级向 **SSS 级**突破的途径，须道具 **觉醒之书·二次**——**极其危险、86% 契约者放弃**，难度随猎杀者阶位 / 等级提升；若该天赋已达首次觉醒极值，二次觉醒难度再 +30%。使用后由轮回乐园匹配「觉醒世界」供选（多为原生 / 试炼 / 虚空高危世界），进入后**主线任务被「觉醒任务」替代**（分环·完成即推进觉醒），借世界历练逼出天赋质变。苏晓在第 28 卷为二次觉醒 **噬灵者**进入月之神世界（详见 [月神大陆·黎明帝国](../世界/任务世界/月神大陆-黎明帝国.md)）。

### 血之兽（**已觉醒 · SSS 级 · 第二天赋**·二十三卷31章）
- **持有者**：[苏晓](../人物/苏晓.md)
- **来历**：在 [黑渊浅层](../世界/任务世界/黑渊浅层-赔罪者之殇.md) 吃下 **[灵魂果实](消耗品材料.md)** 后觉醒的**第二天赋**——灵魂果实给四选一觉醒机会（凶兽蜚 / 土系 / 雷系 / 一只黑眸），苏晓选了透着「**同源血气**」的「蜚」石碑。本是 **SS 级**天赋、却因与苏晓**契合度极高**直接晋升 **SSS 级**（顶级天赋）。同时灵魂强度从 8 暴涨到 **47**。
- **效果（主动·大招）**：用自身气息 / 血气生成血之兽、瞬间扑向 10 米内敌人，造成 **2100 基础伤害 + 力 ×4.5 + 敏 ×3.7 + 体 ×6.8 的无视物理防御伤害**（气息外放 Lv.42 时总伤约 3607）+ 失神 / 眩晕 0.3~7.5 秒。
- **特点**：① **成长天赋**——气息 / 血气越强、基础伤害越高（[气息外放](技能.md) 升级即增幅）；② **三主属性（力 / 敏 / 体）同时加成**（苏晓四主属性里三个加持它·契合度极高）；③ **释放极快、干脆利落**（瞬发·非憋半天的华丽大招）——苏晓评它「单论杀伤力、绝影闪都达不到血之兽的强度」；④ 耗光全部气息（自行恢复）·冷却 8 个自然日（天赋能力可强行使用但损身）。这是苏晓除斩龙闪外威力最高的能力。
- **🆕血气觉醒 · 魅力 −1（三十九卷88–89章）**：苍龙大陆杀戮太多、返园后血气进一步觉醒——苏晓在专属房间里血气一度失控（[吞噬之核](技能.md) 自行开启、黑蓝色烟气涌现、把房间墙面炸裂一层，令布布汪 / 阿姆 / 巴哈 / 贝妮退避），**魅力属性达到 −1 点**（系统提示是「达到」而非「滑落」——想让魅力 −1 本身就极难）。凭 [豁免徽章](装备道具.md) 一次性豁免掉四种自身减益（召唤位 / 魅力装备 / 技能 / 魅惑抗性），只剩「守序善良阵营初始声望降低、无法正常交涉」无法豁免（徽章随即碎裂）。**魅力 −1 的永久加成**：意志力 **+10**、魅力判定阶位 **+3**、自身气息强度 **+15**、新被动 [杀意颤栗](技能.md)（对视强制意志判定·畏惧 / 昏迷）。**血气觉醒眼下只是开始**——苏晓的血气还将重觉醒（他掌握血气已久、自身血气不影响心智）。他去古战场提血气、正是为进一步驾驭这股力量。

## 天赋觉醒（重大机制·十卷25–26章）

苏晓开出 [领主宝箱](装备道具.md) 得 **天赋之书（第二、三页）**（见 [消耗品材料](消耗品材料.md)）后，向乐园问询「觉醒天赋」，凭猎杀者权限调出资料：

- **天赋分类**：普通天赋、稀有天赋、特殊天赋、杀戮天赋等；天赋能力为 **E 级～SSS 级**，**觉醒前无等级**。
- **觉醒后**：天赋显示对应级别，并**增强现有能力或开拓新能力**。
- **觉醒四条件**：① 任意裸妆属性达 50 点（苏晓已达成）；② 获得三次 **S− 或以上**通关评价（已达成）；③ 集齐 **天赋之书（完整）**（苏晓 2/3）；④ 完成**天赋觉醒任务**（未达成）。前三条满足后才激活第四步。
- **天赋觉醒任务**：与晋升任务不同，**可在任意衍生世界执行**；难度 = 猎杀者等级 +10（苏晓 LV.14 → 任务为 LV.24·猎杀者等级越高越难，故宜在二阶至三阶尽早完成）。
- **危险性**：超过 50% 契约者无法觉醒天赋。
- **强制性**：杀戮天赋务必觉醒，乐园会在猎杀者达**三阶**后发布强制觉醒任务。
- **保密**：乐园警告「猎杀者不可透露觉醒天赋相关事宜」。

!!! note "苏晓的盘算"

    觉醒能增强 **噬灵者**（青钢影太耗法力、每世界夺 100 点已不够用）。他打算趁二阶至三阶间风险最小时完成。

!!! success "天赋觉醒任务已激活（十卷80章触发 → 十一卷5章激活）"

    全职猎人通关返园后，苏晓用 **灵魂结晶（大）×1** 换得天赋之书**第一页**、集齐三页合成「天赋之书（完整）」、达成前三项前置 → **进入 [Fate/Zero 世界](../世界/任务世界/fate-zero.md)（第11世界·特意挑的不大世界）即正式激活**（提示为**血红色**·严禁外泄）。机制：**共三环、难度持续攀升、最终环上限 Lv.26**；给三次机会（连续三个衍生世界各失败一次则天赋永久无法觉醒）。觉醒后天赋显级别 E~SSS（**最终级别看任务完成度、而非血统**）。**第一环「抵达柳洞寺」**（Lv.3·4 自然日·罚扣一次觉醒机会、本世界不再发布）——柳洞寺是冬木市地下大灵脉之上的灵地、被圣堂教会用结界藏起。

!!! success "天赋觉醒任务三环完成（十一卷20–31章）"

    - **第一环·抵达柳洞寺**（已完成）→ **第二环·灵魂灯盏**（Lv.21·进柳洞寺最深「主供殿」找到灵魂灯盏〔以灵魂结晶为灯芯、被一具白骨吸食〕·已完成）→ **第三环·破坏灵魂灯盏**（Lv.26·30 分钟·「威胁藏在暗处」）。
    - **真正的考验在「破坏之后」**：苏晓远距离炼金炸弹炸碎灵魂灯盏（连同那具普通白骨）后，柳洞寺内外瞬间罩起**10 层结界**、把主供殿变成牢笼——逼出**领主级 [怨之孤儿·修斯](../人物/怨之孤儿·修斯.md)**（永生池里被打断进化的存在）。苏晓险胜斩杀修斯，**天赋觉醒任务完成**。
    - **⚠️ 完成度机制**：任务完成后**五小时内的举动决定最终「完成度」**——完成度越高、觉醒后天赋等级越高（E~SSS）。**首次觉醒最高只到 S 级；SSS 须「二次觉醒」**（需极强运气 + 实力·非凑任务 / 道具可得）。苏晓初始完成度仅 **34%**（约 B 级），遂**用 [太阳神阿波罗](消耗品材料.md) 把整座柳洞寺夷为平地**（杀光怨生物·世界之源狂飙至 65.2%）以拉高完成度。觉醒时人会**极度虚弱、须在安全点进行**——苏晓尚未正式觉醒噬灵者。

## 🆕 第三天赋【元素之王】（SSS · 增幅天赋 · 43卷32–43章）

苏晓在 [联盟星](../世界/任务世界/联盟星-死亡圣杯.md) 饮下危险物**死亡圣杯（S-002）**的水液后**临时觉醒的第三种天赋**（再用 [古老意志](装备道具.md) 触发「送别·安眠」任务后**突破为永久 SSS·极限等级**）。

| 项 | 内容 |
|---|---|
| 类别 | 增幅天赋（临时觉醒 → 完全觉醒·永久） |
| 等级 | SSS（极限等级） |
| 被动 | 所有元素系攻击伤害**阶位 +2**（突破后 +3、元素类技能上限 +Lv.10） |
| 主动·吞噬 | 可吞噬**七/八系自然元素**（火/光/水/土/暗/风/雷…），转化为对应**元素体质**：火焰君王（豁免55%物理·灼烧削治疗）、光之主（豁免63%物理·反治疗）、风暴神族（水之复苏叠生命）、岩域领主（岩护盾·生命上限+25%）…同吞七系则成「元素之王」（元素伤害20%转真实法系伤害） |
| 变态之处 | 因苏晓身为**灭法者（别称「元素守卫者」）、元素亲和力奇高**，每种形态都被其亲和力额外加成，强到「法爷看了心态爆炸」 |

!!! warning "「最强也最垃圾」的牺牲天赋"

    元素之王强到离谱，苏晓却**几乎不会用它**——他一旦吞噬元素，体内**青钢影能量会掺入元素残渣、不再精纯**，导致燃烧敌人法力数量降低、真实伤害下降，最终青钢影能力**等级持续滑落**。因此这天赋注定是他日后**牺牲掉、换取灭法者独有天赋能力**的那一个（见 [第三天赋觉醒链](../世界/任务世界/联盟星-死亡圣杯.md)）。它的真正价值反在于「宣传」：苏晓靠它 + 死亡圣杯水液 + 古老意志在世界联络平台竞拍圈了一大笔灵魂钱币。
