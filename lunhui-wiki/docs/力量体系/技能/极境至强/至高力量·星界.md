# 至高力量 · 星界

> 至高力量·星界。

## 🌠 至高力量·星界:【无尽的初始之力】


因掌握「无上体质」＋最高晋升规格而**必定获得**。三选一:元素力量（⚠ 已被占据，掌握者死亡前无法选择）／深渊力量／**星界力量**。

!!! success "【无尽的初始之力】（星界被动）"
    可选定一种能力，让其受到「星界初始之力」的持续提升，**直至提升到其等级上限**；若该能力上限后续得以提高，可再度沟通星界重复提升，**次数无限制**（但无法提升能力的等级上限本身）。

⚠ **「原初之核」的真相**:所谓原初之核，就是**星界之顶的「初始之力」破碎、散落各界后失去活性晶体化**的产物。过去提升天锋斩影需要「原初之核（主体）＋初始源质（质变剂·把原初之核回逆）＋时空之力（催化剂）」，正是因为拿不到直接的星界初始之力。如今高活性、高提升幅度、低风险的原生力量直接到手。

**与【极之影】联动出的完美循环**:

> **提升无上主属性 →「极之影」提高天锋斩影等级上限 →「无尽的初始之力」把天锋斩影免费顶满 → 综合战力提升 → 后续行动获得更多时空之力 → 再提升无上属性。**
