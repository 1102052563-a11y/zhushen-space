# 「极境之路」的终身抉择

> 极境之路上的终身抉择。

## 🌌 「极境之路」的终身抉择


⚠ 全无上主属性均达 50 点以上后，需以上古奇物 **【命运星辰】** 唤醒（虚空之树藏库 41 万盎司购得；提示说明:因当前星界环境尚未达到可孕育它的程度，**至少要到第四～第五纪元才有低概率在元素/深渊交接处孕育此物**）:

| 路线 | 内容 |
| --- | --- |
| **繁星之路** | 60／70／80／90／100 点时各获得一种「极境被动」 |
| ⚠ **唯我之界**（苏晓所选） | 60~90 点**不获得极境被动**，全部积攒到 **100 点时得到一种迄今为止的「最强极境能力」**；且⚠ **后续所提升的无上属性，对自身的提升额外提高 100%** |

⚠ 立竿见影的效果:1 点无上体力从成长 5 颗命核变成 **10 颗**、对「极境肉体防御等阶」的成长翻倍；智力对「青影王能量」与「灵魂强度」的成长同样翻倍。
