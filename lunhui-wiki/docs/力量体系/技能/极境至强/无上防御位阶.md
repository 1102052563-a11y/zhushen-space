# 无上之境的防御位阶

> 无上之境的防御位阶。

## 🛡 无上之境的防御位阶


原文在艾薇拉一战中给出的完整梯队:

| 位阶 | 说明 |
| --- | --- |
| **真实伤害** | ⚠ 最顶层梯队里的「大哥」 |
| **伤害减免 / 绝对防御** | 同属最顶层梯队，但排名靠后。⚠ **「伤害减免」的原理是以自身护体能量临时沟通星界的力量、借星界之力分担伤害**——因此**只要敌人的力量能强悍到击穿星界，就能破除它**。⚠ 坏消息:累计到 **95%** 的伤害减免会被彻底击穿；不好不坏的消息:**只有无上之境才能做到这点** |
| ⚠ **极境肉体防御等阶 / 真实抗性** | **最终极的防御位阶，不会被击穿**。前者可看作无上位阶时的「身体防御力」，后者是全抗性。⚠ 苏晓在 203 点属性时为 **+414 / 6645 点**——**朴实无华而又无解的强大** |

⚠ **「真实抗性」中唯独不包含深渊抗性与元素抗性**——这正是【无尽之环】容纳十件原罪物、把深渊抗性堆到 1688 点如此关键的原因。
