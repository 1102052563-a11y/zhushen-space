# 体魄的终点:Lv.MAX → Lv.EX

> 体魄推到 Lv.EX。

## 🏔 体魄的终点:Lv.MAX → Lv.EX


**Lv.MAX**（专属属性强化仓的极限）:⚠ **消耗掉天启乐园「物资总藏库」的 96.55%**。

!!! note "为什么这么贵"
    苏晓在**不足两天内**，达到了至高之人经历十几个、甚至几十个大纪元才累积出的高度。<br>⚠ **提升无上属性的核心原理**是「有能吸收星界始源力量的体质，以吸收它不断提升自身」。<br>⚠ **稳定与代价**:靠轮回乐园的公证＋专属属性强化仓，好处是①稳定（一次次实战表明它与「速成」无关）②**经过极致提纯与过滤的星界始源力量**。<br>⚠ **苏晓的猜测**:至高之人会败于时间，有不低的概率是他吸收的星界始源力量**含有微不可察的杂质**——少量无妨，怕的是日积月累，最终这些星界杂质与星界出现**共鸣现象**，其沉眠极可能就来自这种同化。<br>⚠ 消耗曲线:**递增而非翻倍**。若把苏晓的提升分为 10 阶段，第 3 阶段消耗就已开始不对劲，第 5 阶段起就是递增。到最后，那 35 万盎司的自付部分**已经变成零头**（200→300 点区间约 240 万盎司/点，天启需补正 8.2 亿）。

**Lv.EX**（真正的极限）:吞噬 **「黑枫母树」**（星界最后一棵）只提高了**上限**，需要一种名为 **「自我本源」** 的特殊力量来填实。⚠ 已知只有一条获取途径——吸收「星界本源」缓慢转化积攒。苏晓的绝杀是 **【资源置换装置】**:把与自身**高度因果关联**之物置换为「自我本源」。

⚠ 顺序:【无尽之环】（连同十件大爹级原罪物）→【起源石·世界】→ 厄运古戒 / 赋予之誓 / 烈阳圆盘 / 已破碎的狂猎之夜 / 铁血冥犬 / 狂笑鬼面 / 无形绝望 / 铁血暗狼 / 黑王护臂 → 最后是 **【命运逆转】**。

⚠ **所谓上限，很大程度是由「即将达到上限时最后一股吸收的自我本源有多少」决定的**——而【命运逆转】迸发的自我本源比【无尽之环】还多（它承载着苏晓一路战胜的所有强者之名），**让他在正常上限之上又高出一小截**。⚠ **与至高之人决战的胜负，搞不好就是因为这差距。**
