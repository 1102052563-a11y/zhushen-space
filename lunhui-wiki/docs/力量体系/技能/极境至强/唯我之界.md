# 「唯我之界」的两次兑现

> 唯我之界的两次兑现。

## 🏵 「唯我之界」的两次兑现


苏晓在 50 点节点选了「唯我之界」（60~90 点不给极境被动、全部积攒到 100 点），实际兑现节点为 **100 点与 300 点**（200 点没有）。

| 节点 | 能力 |
| --- | --- |
| **100 点** | ⚠ **【灵之命核】**:消耗 45 万点青影王能量，构成当前命核上限 **30%** 的「灵能命核」。**既可当命核抵御伤害，也可在下次攻击中附带**（攻击强度按灵之命核强度与附带数量而定，⚠ **可单次攻击附带全部**）。⚠ 因由自身身体能量构成，**具常驻性、无存在时间上限**，存在超过 365 个自然日后会以身体能量逐渐代谢替换。⚠ 它把苏晓庞大的青影王能量储备**实实在在转化成了战斗时的战力**，完美契合「平砍即是大招」，而**开场第一刀往往决定后续优势** |
| **300 点** | ⚠ **【极·锋刃】（被动）**:当前**总斩击威力 +35%**——⚠ **包括攻击附带的所有被动效果，甚至包括青影王·绝灭影特性，且此加成理论无上限**。⚠ 到了这等程度，别说 35%，**每提升 1% 都要付出海量资源** |
