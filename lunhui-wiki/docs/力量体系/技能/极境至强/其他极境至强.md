# 其他极境至强（速查）

> 篇幅较短、未单独立页的条目，共 3 条。

## 转化规则


| 项 | 规则 |
| --- | --- |
| 转化比例 | **100 : 1**（100 点至强身体属性凝聚为 1 点无上身体属性） |
| 强度对比 | 同属性对比下，1 点无上属性相较 100 点至强属性**仅高出 15%**——更多的是「无上属性所带来的强大适应性」与额外的「身体属性增益」 |
| ⚠ 领域压制 | **无上主属性不会受到任何领域的压制、削弱** |
| ⚠ 无上席位 | 掌握「无上体质」即**默认占据一个本纪元的「无上席位」**，需在晋升无上之境前死亡此席位才会重新空缺 |
| 被动进阶 | 所有与至强主属性相关的被动/主动技能增益，**全部对应进阶为增益无上属性** |
| 身体能量 | 法力值全部转化为**青影王能量**；1 点青影王能量 = 20 万点至强法力值 |

## 四种「基底成长」


| 属性 | 额外成长 |
| --- | --- |
| 无上力量 | 力量增益成长（在「霸主力量·极毁灭」基础上继续成长；⚠ 该被动已进阶为「体魄实战力量 = 无上力量 ×10 倍」，且**此比例不再是极限**） |
| 无上敏捷 | ⚠ **危险预感**——危险降临前**几分钟、甚至几小时**即可感受到来自因果层面的威胁 |
| 无上体力 | **真实抗性**成长 |
| 无上智力 | 「绝对特性的感知增益」成长（可提高绝对感知的范围与感知捕捉速度；⚠ 感知力本身已达极限，无法再提升）；⚠ **另每提升 1 点无上智力，永久 +3000 点元素亲和力** |

## 🌑 无尽之环 → 向深渊祭献 & 力量始源可吞灭法被动

- ⚠ **【无尽之环】用【黑暗因果】提升后特性再变**:⚠ **从「吞装备临时获特性」→「提基础生命值」→**⚠ **「击杀深渊系累积黑暗祭献值、直接向深渊本身祭献」**（⚠ 团长「这么离谱的东西要么立即扔要么重点培养」·苏晓问三老「灭法者能否向深渊祭献」得「大概率没问题」）;⚠ **⚠ 力量始源可吞灭法被动**〔完美体质/灵魂感电/适应之力保留·完美体质完全淬炼提68主属性〕、通用被动、高阶被动**（⚠ 属性一路冲2130→2487→2569·魅力却掉到-35〔杀黑暗/深渊生物也掉魅力〕）。
