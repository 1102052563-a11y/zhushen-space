# ⚔ 无上之域

> 无上之域。

无上位阶强大者的独有能力。所有人的无上之域均不相同，但必定具备三条:

1. ⚠ **可吞噬「巅峰领域」**——巅峰领域是巅峰之境**以自身对周边世界区域的影响**；而无上之域是无上之境**对所在世界区域的掌控**；
2. 对「无上位阶」之下的个体具有**绝对的力量压制力**；
3. 必定在**范围／杀伤力／增益程度**三者中有一项达到极限。

⚠ **可调节性**:这是无上之域与巅峰领域最本质的区别——前者是「自身能对周边世界造成什么影响，全看自身体系与世界反馈」；后者是「无上强者对周边世界的掌握」，**自己要掌握什么与不掌握什么，当然是可以控制的**。工具是**【命运晶尘】**（在命运因果上「洗练」掉不需要的特性）。
