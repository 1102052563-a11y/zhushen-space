## ⚡ 五种「至强·属性」被动的完全唤醒

> 原属批次:续 · 至强巅峰体系、三种巅峰能力与青钢影完全觉醒（54卷141–170章）

⚠ **身体属性从真实属性晋升为至强属性后，五种至强被动进入待激活状态**——⚠ **苏晓每唤醒一种，都会把上百平米的房间轰成血腥炼狱〔地面墙壁天棚被血气侵染成暗红〕**:

| 至强被动 | 唤醒后效果 |
|---|---|
| **至强·力量** | ⚠ **肉体实战力量 = 至强力量属性 × 2.5倍**〔晋升前为真实力量×1.3〕 |
| **至强·敏捷** | ⚠ **武器攻击力上下限 +1500 · 身体速度永久 ×8.2 · 至强本能 +60**〔每1点至强敏捷=1点至强本能·⚠ 「至强本能」代表的不止反应速度还有感知捕捉速度等，**以前需要一项项提升，现在只需提高至强本能，相关十几种特性都会增长**〕 |
| **至强·体力** | ⚠ **1点至强体力 = 50万点至强生命值成长** · ⚠ **每点 +0.12% 伤害减免**〔上限95%·⚠ **至强体力到170点即可+20%，与「霸主之力」的75%刚好凑满95%**——届时对敌就是「青钢·魂核」与「斩魂·魂核」二选一〕 |
| **至强·智力** | ⚠ **1点=10万至强法力值成长 · 主要装备栏增至20个 · 技法能力开发速度+3500% · 每提升1点永久+30元素亲和力**〔⚠ 苏晓现有1522点元素亲和力已足以让施法者们咬牙切齿;若提升10点至强智力，亲和力就飙升300点〕 |
| **至强·魅力** | ⚠ **永久豁免负魅力属性带来的一切减益 · 外放气息（血气）压制力/侵袭力 +3**〔⚠ 与强敌死战九成先进行气息交锋，气息更强的一方占据先手压制的概率更高·⚠ 有趣的是:九成至强巅峰的「至强·力量」都只有1～2条属性，而「至强·魅力」一般有6～9条;苏晓这两个恰好反过来·⚠ 他的「技能效果3〔晋升至强魅力属性后生效〕」虽因-36魅力永无触发之日，可这条被动**确确实实得到了增强，幅度还不小**〕 |
