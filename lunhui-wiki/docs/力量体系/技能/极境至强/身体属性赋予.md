# 专属身体属性「赋予」

> 专属身体属性的赋予机制。

## 🧿 专属身体属性「赋予」

> 原属批次:续 · 至强巅峰体系、三种巅峰能力与青钢影完全觉醒（54卷141–170章）

⚠ **由【绝对无暇的灵魂石】觉醒·当前1点、最高可提升至10点**;⚠ **⚠ 效果:选定四种主属性之一，把它「赋予」给主属性之外的部分属性**——⚠ **「赋予」属性的高度决定加成比例**;⚠ **⚠ 可加成的目标仅四类**:⚠ **灵魂强度 · 深渊抗性 · 真实意志力 · 多种元素属性抗性〔雷/火/寒冻等〕**（⚠ 效果差异极大:5000点真实智力赋予**深渊抗性**只提约50点〔深渊抗性本就极难提升〕，赋予**雷属性抗性**则提2000多点·⚠ 灵魂强度加成也不高，且**就算临时提到3000点也不能触发「绝对意志」的彻底唤醒**——那还得靠逐步累积）;⚠ **⚠ 苏晓的选择:真实智力 → 雷属性抗性**——⚠ **从836点提升到2994点**（⚠ 原因:他的「天怒·界雷落／奔流斩／奔流闪」都随刀术宗师同步到Lv.EX，可他从没有豁免界雷伤害的能力——**那是界雷体系中基础中的基础**;⚠ 他不是不想学，而是查看相关技能卷轴时最下面用鲜红字体写着「**因你相关天赋前置不足，无法掌握此技能**」·⚠ 他能掌握三个天怒系能力，纯粹因为它们与刀术体系息息相关·⚠ **于是他独创了一种界雷系:既然掌握不了豁免，那就不豁免了——2994点雷抗，硬抗！「几百公里粗的界雷柱落下，敌人倒下我不倒 = 我成功掌握了界雷系，且后续不用投入资源」**）。
