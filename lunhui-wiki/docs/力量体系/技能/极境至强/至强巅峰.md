## 🏔 至强巅峰 —— 至强之力的「苏醒晋升」

> 原属批次:续 · 至强巅峰体系、三种巅峰能力与青钢影完全觉醒（54卷141–170章）

⚠ **至强分两个阶段**:⚠ **第一阶段是完成至强晋升〔新晋至强～超·顶尖至强〕**——⚠ **晋升至强后身体并不能马上承受至强之力，还需要一个提升过程；当完成至强之力的「苏醒晋升」，才是真正强大的至强者，这个过程被星界各大势力称为晋升「至强巅峰」**;⚠ **⚠ 晋升前必定无法唤醒体内的至强之力，也就是攻击中不带至强之力**——⚠ **这在面对「至强生命值」时会带来最起码95.62%的伤害降低**（⚠ 举例:一名最极限的超·顶尖至强用大招轰出500多万伤害→先过苏晓的75%伤害减免只剩125万→再过「至强体力」衍生的身体防御削到约16万→再过各类防御手段削到约9万→**这9万点在真正杀伤至强生命值时，总计只能造成3942点最终伤害**）。

| 身体能量 | 晋升规则 |
|---|---|
| 至强生命值 | ⚠ **基础生命值 1:1 晋升**·⚠ **不具「至强之力特性」的攻击造成的损伤降低95.62%以上**·⚠ **仅「真实伤害」可造成全额**〔神圣伤害等次级顶尖梯队面对至强生命值时按常规攻击判定〕 |
| 至强法力值 | ⚠ **基础法力值 1:1 晋升**·上限更高 |
| 至强主属性 | ⚠ **100 : 1 晋升**〔100点真实身体属性凝聚为1点至强身体属性〕·⚠ **晋升当次额外 +20%** |
| ⚠ 阶位碾压 | ⚠ **纯力量比拼:不说30点至强力量对3000点真实力量，哪怕20点左右的至强力量也能把3000点真实力量揍到怀疑人生**——⚠ **这是力量阶位、力量凝聚程度、乃至力量调用方式的差距**（⚠ 但也不是完全打不过:真实力量堆到9000～12000点，很有可能锤得过30点至强力量·⚠ **最大的问题是上限**:真实属性5000点已是虚空之树公证认可的极限，而至强属性的50点只是开始） |
| ⚠ 不适期 | ⚠ **多数契约者晋升后会有一段不适期**:⚠ **昔日的「主动技能」核心承载不住至强身体能量，启动过程中会砰的一声炸掉，导致精神/灵魂力震荡**——⚠ **苏晓完全没这问题:主动战斗技能只有青钢影与龙影闪两种，而Lv.185的青钢影其技能核心强度估摸着直奔8000+** |
