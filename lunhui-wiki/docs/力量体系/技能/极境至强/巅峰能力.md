# 巅峰能力

> 「巅峰能力」的三种掌握方式，以及苏晓最朴实无华的那三种。

⚠ **本页合并了原「技能·功法」页中散落 2 处的记载**，按卷序排列。

## 🎯 「巅峰能力」的三种掌握方式

> 原属批次:续 · 至强巅峰体系、三种巅峰能力与青钢影完全觉醒（54卷141–170章）

| 方式 | 说明 |
|---|---|
| **自行体悟** | ⚠ 把自身的能力体系融合为一即是巅峰能力——⚠ **需长时间完善，预估半个纪元的岁月流逝** |
| **星界之顶** | ⚠ 去往「起始与终焉之地」感受自身——⚠ **初始即为最高完成度** |
| **体系传承** | ⚠ 向虚空之树支付足够时空之力，开启第一纪元初始～当前纪元、与自身同体系强者的「巅峰能力」**铭刻高碑**作为参考——⚠ **初始即为最高完成度**〔苏晓可参考的技法宗师/灭法者巅峰能力共**19种**；10万盎司一次、高碑停留12小时，一般10～11小时即可完成一种〕 |

⚠ **⚠ 所有巅峰能力都需先掌握「巅峰能力·基石」**〔整体为圆形、内部是繁琐的线体结构·没有任何特性·苏晓首次构建用了5小时、第二次4小时〕;⚠ **⚠ 铭刻高碑上的11种可参考能力**:

| # | 名称 | 效果 |
|---|---|---|
| 1 | 灵刃 | ⚠ 可无视距离进行斩击伤害〔⚠ 关键在于这算近战攻击还是刀术延伸的中远程攻击:若是近战则与苏晓极契合，若是中远程则不然〕 |
| 2 | 元素流火 | ⚠ 驾驭元素·流动心火（⚠ 苏晓是灭法者，**驾驭元素之力是自己找坑跳**，直接放弃） |
| 3 | 时间逆刃 | ⚠ 以手中利刃控制时间流速甚至短暂回溯（⚠ 看似强无敌，**实际远不如「极境·星界」**——星界一停固，时间会宛如固体般一动不动;**星界停固是时间系的绝对上位**） |
| 4 | 无尽战意 | ⚠ 与强敌战斗愈战愈强，获得全方面提升（⚠ 备选） |
| 5 | 吞星 | ⚠ **吸收星辰破碎后散溢的「星界始源力量」提升自身**〔随机提升自身能力〕——⚠ **备选**（⚠ 应为 吞星·阿卡斯 所掌握） |
| 6 | 月光秘术 | ⚠ 让「自我月亮」降临，按自我意念临时改变周边区域的基本规则（⚠ 与其战斗体系驴唇不对马嘴，**至强巅峰了不可能让体系因此变臃肿**） |
| 7 | 断魂之力 | ⚠ **造成「斩魂」后，敌人被斩伤区域的灵魂会产生「自我」并背叛、全力挣脱本体后协助你对付其本体，并达成因果层面的部分取而代之**〔所产生的灵魂体可继承原主人的部分因果存在，按占比决定存在时限，最长者活了上千年〕——⚠ **格林·吉莉安的巅峰能力**（⚠ **不仅可怕，更是恶劣**:灵魂小人挣脱后会飞到她肩头，快速介绍本体有什么能力体系与弱点；而她知道后会让灵魂小人**故意大声喊出敌人的黑历史**，甚至丢给它一个「思林特斯矮人」打造的终端，**把正与她生死战的强敌的全部秘密发到虚空的网络集群上**·⚠ 「和格林·吉莉安死战的敌人，有不少都是**先内心破防，后续才开始身体破防**的」） |
| 8 | 我之力 | ⚠ 以自身「宗师意志」为力量基础，临时使用所见所知的任何能力，且通过自我认定让自身不断强大 |
| 9 | 一念·世界 | ⚠ 将自身一切汇聚至下一次斩击:斩灭敌人则自身进一步强大，未斩灭则己身消亡、归于虚无（⚠ 过于极端;且他已有「无我之境」这种杀伤力同样拉满的能力） |
| 10 | 黑暗刀锋 | ⚠ 构成一颗吞没一切的「刀锋黑洞」（⚠ 作为掌握天锋斩影的刀术宗师，**他不缺最顶级的攻击手段**） |
| 11 | 巅峰之境／无上巅峰 | ⚠ **战胜至强巅峰的敌人时，夺取其全部「因果命运」反馈给星界，可获得等量的「星界始源力量」壮大自身**——⚠ **备选**（⚠ 应为 席曼·阿奇德 所掌握） |

## 🔥 苏晓的三种「巅峰能力」—— 最朴实无华的巅峰

> 原属批次:续 · 至强巅峰体系、三种巅峰能力与青钢影完全觉醒（54卷141–170章）

⚠ **⚠ 他的判断逻辑**:⚠ **若他的极限是至强巅峰，选「无尽战意」很不错——在现有基础上越战越强、战斗风格彻底定型**;⚠ **⚠ 可若选「吞星」+「无上巅峰」，用不了多久他就能发展到「初始状态就是无尽战意那越战越强的最强状态」，而且是日常轻松永远保持这么强大**——⚠ **二者孰优孰劣，一目了然**;⚠ **⚠ 更关键的是他的专属至强被动「力量始源」**:

| 能力 | 强弱评级 |
|---|---|
| **力量始源** | ⚠ 对「星界始源力量」的**吸收/利用率**极高、几乎达到极致——⚠ **暂定 S 级**;⚠ 可对星界始源力量的**获取比例**就比较低——⚠ **暂定 D 级〔E～S〕** |
| **巅峰能力·吞星** | ⚠ 无需任何付出即可吸纳星界始源力量——⚠ **S+ 级**;⚠ 但对其**利用**大概只有 **A** 的程度〔随机提升自身能力〕 |

⚠ **⚠ 于是他用两种巅峰能力去撬动「力量始源」的倍数**——⚠ **「能力牺牲」〔彻底牺牲一种被动技能所得的星界始源力量倍数〕的演进:晋升前1倍 → 至强巅峰2倍 → 加「吞星者」**8倍** → 再加「无上巅峰」**15倍****:

| 能力 | 说明 |
|---|---|
| **吞星者**（巅峰级·被动） | ⚠ **大幅提升通过「力量始源」所能获得的星界始源力量数量**·⚠ **附带:掌心可构成繁星印记，从星界始源力量中过滤出斑驳杂质**（⚠ **吞星·阿卡斯与席曼·阿奇德的最终死因，很可能就是身体吸纳了过多斑驳杂质**·⚠ 苏晓运气不错:轮回乐园内的星界始源力量非常纯净） |
| **无上巅峰**（巅峰级·被动） | ⚠ 同上（⚠ 「吞星者」与「无上巅峰」把「力量始源」夹在技能列表中间） |
| **力量始源**（**巅峰级**·专属被动） | ⚠ **「心之熔炉」吞噬已掌握的任何被动技能，把加成吞入体魄并吸取等量星界始源力量**——⚠ **提升:至强力量/敏捷/体力/智力、技法潜力上限、深渊抗性**;⚠ **⚠ 彻底牺牲一种至强被动 = 四主属性各 +12、技法潜力上限 +60、深渊抗性 +5** |

⚠ **⚠ 「无与伦比强大的自身体魄，就是他的巅峰能力」**——⚠ **一刀下去，对方巅峰能力制造的黑洞都能斩爆**;⚠ **⚠ 连轮回乐园都公证了这点:「力量始源」的属性标签已从「至强级」变成「巅峰级」**;⚠ **⚠ 意外之喜**:⚠ **「至强属性强化仓」所使用的提升能量，正是「星界始源力量」**——⚠ **这等于轮回乐园认证了「力量始源」这条路的稳定性与安全性:后续无需再担心该能力是否稳妥，只需看钱包能否承受它的猛烈打击**。
