# 至强体质

> 灭法者晋升至强的专属秘法。

## 🏛 至强体质（灭法者晋升至强的专属秘法）

> 原属批次:续 · 极刃·白夜（强力斩）、刀术宗师Lv.95蜕变与至强体质（53卷14–26章）


⚠ **[命定之手·阿什维斯](../../../人物/虚空/命定之手·阿什维斯.md) 告知位置:在古战场深处的石碑上（灭法阵营独有刻印）**——⚠ **⚠ 这是仅有的几种能以最高潜力上限晋升至强的秘法之一**（⚠ 门槛极高，但苏晓已完成半数以上）:

| 晋升至强梯阶 | 门槛 |
| --- | --- |
| 第三梯阶（上限最低） | ⚠ 1 份初始源质＋50 块原初碎片＋主属性 500＋核心基础能力 Lv.90 |
| 第二梯阶（大势力规格） | ⚠ 1 份初始源质＋5 颗原初之核＋主属性 700＋核心基础能力 Lv.95＋本源药剂 |
| **⚠ 至强体质（灭法阵营）** | ⚠ **5 份初始源质＋15 颗原初之核＋至少 3 种主属性 800＋核心基础能力 Lv.95＋本源药剂＋灵影体质超极限突破一次＋魔灵强度 800 以上** |

- ⚠ **⚠ 苏晓的完成度**:⚠ **初始源质 3/5、原初之核 7/15（⚠ 还需大量·⚠ 晋升至强＋三技法过 Lv.95 保守估计要 40 颗原初之核）· 刀术宗师已 Lv.95 · 本源药剂大幅超额（已饮「奇迹品级」）· 灵影体质已多次突破上限 · 魔灵强度已到 740（压都压不住）**;⚠ **⚠ 真正要在意的是把灵魂强度从 1130 提到 1200，才能压制 800 魔灵强度**;⚠ **⚠ 死亡率豁免:灵影体质 60%＋不屈之影 25%＝85%**（⚠ **就算是使用后 100% 死亡率的灭法秘法，实际也只 15% 死亡率——以灭法系标准值得一试**）。
