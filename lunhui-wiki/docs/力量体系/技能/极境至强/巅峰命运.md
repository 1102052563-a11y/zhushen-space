# 「巅峰命运」与「巅峰被动」

> 巅峰命运与巅峰被动。

## 🎲 「巅峰命运」与「巅峰被动」

> 原属批次:续 · 极境体系、极境被动与「巅峰命运」（54卷171–200章）


⚠ **⚠ 每击杀一位巅峰强者必得100点「巅峰命运」**〔击杀最强古神冥神额外+200〕——⚠ **可兑换其能力所化的「技能石」，再向虚空之树/轮回乐园申请**置换同位阶技能石****（⚠ 轮回乐园的置换列表有五页，是虚空之树的几倍）;⚠ **⚠ 掌握后的技能归入一张独立技能页，位于至强被动页之前、被称为「独有被动」**——⚠ **这是唯一一页可自行命名的技能页；苏晓命名为「**巅峰页**」，其中技能自适应更名为「**巅峰被动**」**;⚠ **⚠ 「巅峰被动」不如「巅峰技能」，但其强度也不是「至强被动」能比**;⚠ **⚠ 苏晓的分级**:⚠ **T0＝【始源体质】【极境宗师之力】· T1＝提升核心能力的杀伤力/功能性/操控性与范围 · T2＝提升肉体力量、速度、生命力**:

| 巅峰被动 | 来源 | 效果 |
|---|---|---|
| **始源体质** | ⚠ 蜘蛛夫人·【剧毒魔心】置换 | ⚠ **让全身巅峰梯阶细胞主动吞噬「星界始源之力」**——⚠ 彻底根除快速提升至强属性的速成隐患，并把吸收效率再提约**20%**〔⚠ 掌握后有「卸去一身重甲」的轻松感·⚠ **它实为上古时代极境系最繁荣时的大成之作**〕 |
| **极境宗师之力** | ⚠ 夜母·【永夜星辉】置换 | ⚠ 按当前「技法潜力上限」**大幅提升三宗师** |
| **力量吞噬** | ⚠ 【初始的生命源泉】 | ⚠ **必定改变「灭法天赋·猎影」的特性，使其在极限觉醒后获得「？？？力量」** |

- ⚠ **⚠ 两条提升路线殊途同归**〔团长语〕:⚠ **① 身体属性 ② 各类被动技能**——⚠ **本质都是提升「自我」**（⚠ 老牌至强会在漫长时间里累积几百上千种被动；而**并不是所有巅峰强者都有「力量始源」这种离谱能力**——**这能力把上一任完全体天选都眼馋到无语了好一会**·⚠ 苏晓没有时间水磨功夫开发被动，因此走「至强属性」路线并在被动上**优中选优**）。
