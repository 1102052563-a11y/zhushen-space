# 绝对意志

> 半觉醒状态的绝对意志。

## 🛡 绝对意志（至强被动 · 半觉醒）

> 原属批次:续 · 绝对意志、极境·星界与极刃三系（54卷121–140章）

- ⚠ **灵魂强度晋升时，因吸收了 [老婆婆](../../../人物/命运暴君·老婆婆.md) 所赠的高位阶「心魂刃」，苏晓可在两条路里择一**:⚠ **①「以心噬魂」〔立竿见影得多种豁免/免疫，但掌握后进入绝对上限〕 ②「以魂沐心」〔初始不弱、且能随「魂之茁壮」持续增益·占1个至强被动槽〕**——⚠ **苏晓选后者〔他的天赋之一就是提升灵魂强度〕**;⚠ **⚠ 效果**:⚠ **「灵魂强度」与「真实意志力」达成永久同步共鸣〔意志力必定不高于灵魂强度，且随灵魂强度一同提升〕**;⚠ **⚠ 【绝对意志（半觉醒）】:面对心灵系/精神系/灵魂系，以及一切幻境、内心操控、控制类的意志判定，获得 +4344 判定强度〔2172灵魂+2172意志〕**——⚠ **当两者相加达 6000 点即完成觉醒、获得 100% 契合于自身的「绝对意志」**（⚠ 100%契合的「绝对特性」无法用【不朽意志】从虚空之树换来〔换来的极限是99.99%〕）。
- ⚠ **⚠ 连锁变更**:⚠ **因与刀术宗师的「免疫精神系、心灵系控制」特性重合，该特性被【绝对意志】吸收**——⚠ **刀术宗师空出的约5%总体强度自然转化到其他方面:伤害/斩击力/破武判定阶位 +980→+1000、斩魂强度 50→60**（⚠ 「豁免是盾，能挡瞬间的猛烈冲击，挡不住如洪水般持续沁浸的心灵能力」·⚠ 这也是苏晓要「绝对不可动摇意志」的原因·都卡因始祖的【不朽意志】是「绝对不可放弃意志」，与刀术宗师「一往无前」的意志严重排斥）。
