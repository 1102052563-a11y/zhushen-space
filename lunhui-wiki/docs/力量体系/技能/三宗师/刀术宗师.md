# 刀术大师 → 刀术宗师

> 从刀术大师到宗师 Lv.110 的完整阶位攀升。

⚠ **本页合并了原「技能·功法」页中散落 6 处的记载**，按卷序排列。

## 刀术大师 → 刀术宗师（阶位攀升 · 十九卷92–97章）


苏晓的核心刀术能力在海蓝星返园后迈过一道大坎——从「大师级」晋升「宗师级」。

- **刀术大师 Lv.50 = 理论上限**：升 Lv.48→50 时（每级 25 万币 + 灵魂结晶〔大〕×10），苏晓被 [黑枫树老者](../../../人物/黑枫树老者.md)（[马文·华尔兹](../../../人物/马文·华尔兹.md) 的师傅·唯一现存的灭法者·将刀魔虚影囚于体内吞噬之）召入幻境点拨抉择。老者给三条路：① 想办法**突破大师极限继续升**（下限高·初期强）；② **晋升刀术阶位**（上限高·后期强）；③ 极端之路（短时间变强几十倍上百倍、但意识可能消亡·不归路）。苏晓选 ②（核心能力当取上限）。Lv.50 后刀术大师临时锁定为 **Lv.MAX**（刀类武器攻击 +15%、刀芒威力近 +1/3）。
- **晋升门槛 = 宗师之力（完整）+ 荣耀之证明**：晋升大师级本需试炼考核，猎杀者凭高阶位能量**豁免考核**，但要付两样：**[宗师之力（完整）](../../装备道具/index.md)**（10 份任意宗师之力残卷可合成·苏晓凑齐）+ **[荣耀之证明](../../装备道具/index.md)**（试炼场「挑战模式」打赢指定镜像所得·95% 以上挑战者失败）。苏晓在试炼场连战镜像 **西尔西亚**（人类·力/敏/体 95 真实·仅一种宗师级能力「近战肉搏宗师 Lv.10」却完爆其刀术大师 Lv.MAX）——**连败八场、每场 5000 币、第八场才同归于尽式险胜**得荣耀之证明，由此领悟「大师级与宗师级差距悬殊、宗师级初始下限就极高、更靠战斗经验与时机」。
- **🆕刀术宗师 Lv.1（十九卷97章）**：消耗荣耀之证明 + 宗师之力（完整）晋升成功。**刀术宗师 Lv.1**：刀类武器攻击 +240%（因仅 Lv.1·暂低于大师 MAX）、**神经反射速度 +3%（可随等级成长）**、**完全免疫精神系 + 心灵系控制**（近战克星·之前的刀术只能免疫可判定的精神控制）、初始附加「**万物律动 Lv.MAX**」、刀芒 Lv.50（飞行 +150% / 锋刃 +30 / 破甲 +25 / 能量化实体）。晋升后他一刀斩断之前砍不动的「**莫罗复合金属柱**」（34 吨·大师 Lv.MAX 绝对斩不断）。提升费却意外便宜（Lv.1 仅灵魂结晶〔大〕×1 + 5 万币）——「厚积薄发、之后战斗力将快速提升」。
- **🆕升至 Lv.12 · 觉醒「绝影闪」（二十卷53章·死神世界返园）**：苏晓用 10 颗灵魂结晶〔大〕连提刀术宗师，靠黑枫树加持实际从 Lv.1 冲到 **Lv.12**（每级灵魂结晶〔大〕×1 + 5 万币·Lv.10 以上每级 10 颗）。**Lv.12**：刀类武器攻击 **+336%**、神经反射速度 **+39%**、免疫精神 / 心灵控制、刀芒 Lv.60（飞行 +180% / 锋刃 +40 / 破甲 +35）。**Lv.10 觉醒终极能力「绝影闪」**——超极限速度突进、下次斩击附带「刀类攻击力 ×3」的穿透斩伤、耗 60% 现有生命（**仅生命 ≤35% 可用**·冷却 48h·杀敌则回 30% 生命并重置冷却）。苏晓评：单挑神技、群殴送死技（自残到 35% 才能放·一刀不死、自己死期就近）——竞技场实测**自割三刀到 35% → 绝影闪 2.8 秒秒杀对手**。提刀时房间内刀芒爆发把贝妮吓到投降。
- **🆕刀术宗师实战首秀（二十卷10–15章·死神世界）**：① **破镜花水月**——[蓝染](../../../人物/蓝染惣右介.md) 用完全催眠系斩魂刀镜花水月控制苏晓，被刀术宗师「完全免疫精神 / 心灵控制」的衍生效果**直接破除**（精神系能力对苏晓无效·苏晓装作被催眠骗过蓝染）；② **刀芒能量化实体**——刀术晋宗师后刀芒升为能量实体，**[青钢影](../index.md) 能量可攀附在刀芒上**（淡蓝刀芒奔涌电弧）；③ **傲歌·晶体层（青钢影分支防御）**——傲歌的第二防御形态（能量盾之外、苏晓后期开发）：把青钢影能量凝成**晶体层裹在刀身 / 体表**（耗能越多越坚固），死神世界里用它隔着斩死兆的「侵蚀黑箭」（晶体层碳化代损、护住斩龙闪本体）；④ 对身体素质流的 [死兆](../../../人物/死兆.md)（力 / 速强于苏晓）三回合压制——印证「宗师级靠技巧、初始下限极高」（这还只是 Lv.1·进死神世界前他故意没提级、怕生涩感拖累实战）。

- **🆕自创招「血刃」+ 绝影闪无冷却（二十二卷·第五次圣杯战争）**：① **血刃（苏晓自创·宗师级体悟 + 莎的虚空眼界）**——口诵「惟我不断者」、把血气包裹斩龙闪斩出**血色刀芒**，锋利度与普通刀芒相近、**速度却是普通刀芒的数倍**（设想由苏晓本人完成、再与 [莎](../../../人物/莎.md) 一同实践开发成功）；停车场一战用它划伤 [金闪闪](../../../人物/吉尔伽美什.md) 胸膛、印证「与莎合作不只提升近战、还得了虚空眼界」。② **绝影闪「猎魔时刻内无冷却」**——绝影闪是刀术衍生的**自身能力**，故在 [猎魔之王](../../装备道具/index.md) 的猎魔时刻期间**进入无冷却阶段**（自残压血到 35% 以下可连续放·只受体力限制）；斩 [斯坦](../../../人物/斯坦.md) 时连用两次绝影闪砍开其护心甲片、再补魔刃即死。

- **🆕潜能激发 + 升至 Lv.24「灵魂之刃」（二十五卷60–63章·顶上战场返园）**：苏晓把卖不掉的「枪术宗师卷轴」卖给商人 [银洛](../../../人物/index.md)（换 400 万币 + **潜能激发**被动）。**潜能激发（被动·MAX）**：大幅加速剑 / 刀 / 体 / 枪等技巧开发、提升该类天赋 10~30%、并直接把**刀术宗师提 Lv.2~5**（用它前消耗黑色能量把苏晓意识传送到某种族「圣地」溶洞·吃透明人影硬提天赋·布布汪在现实给他做心脏复苏）。刀术宗师由 Lv.12 冲到 **Lv.24**：刀类攻击 **+445%**、神经反射 **+99%**、刀芒 Lv.70；**Lv.20 终极能力「灵魂之刃」**——刀术融入灵魂、**免疫 50% 灵魂伤害 + 豁免一次可致命的斩杀 / 即死效果**（豁免后沉寂 15 天）。另在 [众生之地](../../../世界/地点/index.md) 三层（30 万币 / 时）把自创终极招「**绝影闪**」开发到 ~80%（仅 30% 时就险些一刀斩死大将青雉）。
- **🆕升至 Lv.32 · 觉醒「刃之极」（二十七卷51章·棘虫星返园）**：苏晓用世界争夺战奖励的 65 颗灵魂结晶（完整）连提刀术（Lv.30 后每级 70 颗 + 220 万币·费用再增）——从 Lv.24 冲到 **Lv.32**：刀类武器攻击 **+517%**、神经反射 **+139%**、免疫精神 / 心灵控制、初始附加「万物律动 Lv.MAX++」（突破上限两阶位）。提级时入雪山幻境见六位刀术强者毕生体悟（一人斩巨龙、一人斩穿世界）。**Lv.30 终极能力「刃之极」**——用任何刀术招式时可选择激活、招式威力提升 16%~35%（主动激活耗体力大增·5~8 次脱力；战斗中亦有几率被动触发〔无消耗 + 永久提升刀术体悟〕）。苏晓评：现在给他一把白色长刀都能打出淡金品质长刀的威力。
- **🆕升至 Lv.33（二十九卷37章·剥老圣修灵魂记忆）**：苏晓在 [贪婪之章](../../装备道具/index.md) 里终于击败第一魂灵老圣修，用 [噬灵者 SSS](../../天赋.md) 主动效果剥得「技法记忆·刀术宗师」（看一遍即领悟·≈15 灵魂结晶 + 50 万币的收益）——刀术宗师 Lv.32 → **Lv.33**（刀类攻击 **+526%**·神经反射 **+144%**）。这是二次觉醒噬灵者「主动偷技法」的实战红利。
- **🆕升至 Lv.35（二十九卷63章·30 颗灵魂结晶）**：返园用一个世界攒的 30 颗灵魂结晶（完整）连升刀术宗师 Lv.33 → **Lv.35**（刀类攻击 **+544%**·神经反射 **+154%**）——提升时入老灭法者黑枫树幻境（被一刀「斩」入升级·老灭法者肉疼自己被苏晓扯下一大块黑枫树干）。
- **🆕升至 Lv.38（三十卷73章·巫师大陆返园）**：苏晓用 54 颗灵魂结晶中的一批连升刀术宗师 Lv.35 → **Lv.38**（每级 50 万币 + 灵魂结晶〔完整〕×15·入草原幻境被三道灵魂结晶身影一刀「斩」入领悟）——刀类武器攻击 **+571%**·神经反射 **+169%**。Lv.40 后将再有一次大幅提升。
- **🆕升至 Lv.41 · 觉醒 Lv.40 终极「心·魂·刃」（三十一卷61章·深渊世界返园·浴血之试炼）**：苏晓用圣灵级辅助卷轴 **[浴血之试炼](../../装备道具/index.md)**（+ 起源石大幅增效）在试炼场关掉保命机制刷无尽极冬战场士兵 5 小时——刀术宗师 Lv.39 → **Lv.41**（刀类攻击 **+599%**·神经反射 **+185%**·万物律动 MAX+++）。**Lv.40 终极能力「心·魂·刃」（被动·独有）**：凭「心之眼」锁定敌人破绽，攻中破绽则敌方强制**无法防御 / 招架 / 移动**极短时间，期间斩击造成 **3~5 倍无视肉防伤害**（无消耗 / 无冷却 / 可连用）。苏晓本就极擅捕捉破绽（与铁羽王战中捕到三次）——得此能力后「抓住破绽 = 敌人可以考虑棺材型号了」。每个人宗师 Lv.40 终极各异（铁羽王的是适合战场的「征战」、苏晓的偏一对一）。
- **🆕升至 Lv.42（三十二卷94章·强者争霸战通关返园·30 颗灵魂结晶）**：苏晓返园用 30 颗灵魂结晶（完整）+ 70 万币升刀术宗师 Lv.41 → **Lv.42**（刀类武器攻击 **+609%**·神经反射 **+191%**·免疫精神控制 / 心灵蛊惑）——升级幻境是「世界始灭于一念之间」的黑白缠绕圆球 + 透明持刀人影。Lv.40 后每级需 30 颗灵魂结晶（完整），费用陡增但稳定累积。
- **🆕升至 Lv.46（三十三卷59章·五方混战通关返园·120 颗灵魂结晶）**：苏晓用 120 颗灵魂结晶（完整）连升 Lv.42 → **Lv.46**（刀类武器攻击 **+649%**·神经反射 **+215%**·万物律动 ×5 突破上限）——升级幻境见一名锻造灵魂刃的强壮老者（**里德**提过的锻造大师·问苏晓「因何拿起利刃」·只能见两次·评苏晓「极恶持刃者却被众多世界抱以善意·非狂魔」）。Lv.50 是下一次质变。
- **🆕升至 Lv.49（三十五卷66章·炎灵星通关返园·90 颗灵魂结晶）**：苏晓用 90 颗灵魂结晶（完整）连升 Lv.46 → **Lv.49**（刀类武器攻击 **+679%**·神经反射 **+233%**·免疫精神 / 心灵控制）——升级幻境见「屠戮了大半个世界的持刀身影」、淡蓝双眸一刀刺穿他（疑与**刀魔**同源·苏晓吞噬之核全开时双目就是这样）。**Lv.49 → Lv.50 需 100 颗灵魂结晶 + 300 万币**、是又一次质变（届时再见「锻光者」得关键之物）。
- **🆕屠戮之影 Lv.MAX+++++（三十三卷59章·3 黄金技能点）**：傲歌状态强度 **+320%**、青钢影实体外放操控距离升至 **126 米**（强度 / 杀伤随智力·现 149）。同时发现**青影王（奥义）无法用黄金技能点提升**——苏晓首个黄金点提不动的能力，至今不知其升级方式。
- **🆕猎魂之镰（被动 · 灭法者能力 · 三十卷73–75章·神皇换座狼套所得）**：[神皇冒险团](../../../人物/神皇.md) 用一大笔物资换给苏晓的 **魂镰影**技能卷轴（产地暗鸦星·先代灭法者所开发·「魂镰」二字仅灭法之影可见）——直接使用得**猎魂之镰（被动）**：攻击附带真实敏捷 ×2.1 的无视肉体防御伤害；苏晓花 126 万乐园币刷到 **Lv.MAX**，核心特性**永久 +5 真实敏捷**（敏达 137·成四主属性最高）。这是「战斗类被动技能」的典范——消耗低、看不见，但攒多了「毫不起眼的一刀下去有七八种判定」。卷轴上**「魂镰形态」的灭法者知识**（断魂影分支·以屠戮之影为载体·须先觉醒断魂影才能开发）被苏晓抄录留存。
- **🆕气息外放 Lv.68（三十卷76章·古战场提升）**：苏晓到虚空古战场用简易吞噬之核**过滤提纯血气**喂气息（吸太多被血气侵蚀险失控）——气息外放从 Lv.56 一举升 12 级到 **Lv.68**（效果蜕变中需 20 天）：意志力 +61、意志判定阶位 +2、气息可对「异物类」敌人造成意志 ×12 的灵魂伤害；血之兽基础伤害随蜕变逐步提升。
- **🆕灵影体质 Lv.MAX++++++（二十九卷64章·5 黄金技能点·提到真正上限）**：苏晓把 5 点黄金技能点全压 [灵影体质](../index.md)（噬灵者使其成成长类被动·有风险但他无视）——提升瞬间直接昏厥、醒来后如脱铁衣般轻松。**到真正上限**：技能效果 1 由「生命 = 法力 45%」升为 **「生命 = 法力 100%」**（现 +10676 生命）；技能效果 2「移速 +12% / 动态感知性 +65%」（神经反射 / 动态视力等十几种身体素质统一提升）。
- **🆕屠戮之影 Lv.MAX++（二十七卷51章·黄金技能点提升）**：苏晓用 1 点黄金属性点提升专属成长能力「**屠戮之影**」至 **Lv.MAX++**——傲歌状态强度 **+230%**、可将青钢影能量转为实体外放并在 **75 米**内操控（实体化的强度 / 杀伤 / 速度 / 穿透 / 操控随智力而定·现真实智力 120）。由此 [放逐](../../装备道具/index.md) 不再当常规武器（那会贬低其价值）、改作精疲力竭时的致命一击；晶体层（青钢影实体防御形态）也大幅增强。

⚠ **⚠ 剥离的契机**:⚠ **元素冲击把融入他右臂的「上一任深渊之主本源能量」完全剥离出去**——⚠ **失去这份外来力量的影响后，早就该独立的「极境·星界」立即完成了剥离**（⚠ 塞翁失马:也正因一直没剥离，此前几次大幅提升刀术宗师时「极境·星界」也一同得到了提升，已从初始阶段推进到**专属特性阶段**）;⚠ **⚠ 每个人的「星界能力」除「停固星界」这一固有能力外，都有自己的特性**——⚠ **苏晓的特性是**支配****:

| 项 | 内容 |
|---|---|
| 范围 / 时长 | ⚠ **20000米 · 停固5秒**〔原8秒，其中4秒被后续的特性进阶所吸收〕 |
| **支配特性** | ⚠ **期间可「限定支配」这片区域，永久性夺取强敌的一种能力**〔巅峰能力除外〕**，并经星界转化为你的「临时生命之核」**·⚠ 停固结束后，被永久剥离能力的敌人陷入**2～3秒「绝对虚弱」** |
| 对弱者 | ⚠ **永久性夺取其全部能力**〔超高概率立即死亡〕，或施加「**绝对支配**」——⚠ **48～72小时后必定死亡** |
| 临时命核 | ⚠ 仅存在24小时·⚠ 触发阈值时**优先消耗临时命核** |
| ⚠ 下一阶段进阶 | ⚠ **当敌人处于无「生命之核」状态时，可永久性夺取敌人的巅峰能力**，经「星界体系力量」纯净化后形成**永久性生命之核**〔1～？颗，按其巅峰能力强度而定〕·⚠ 需消耗5秒持续时间 |
| 冷却 | ⚠ **每1个自然日仅可使用一次**〔此为该特性所衍生的极致使用间隔〕 |
| 提升资源 | ⚠ **【星辰辉石】**·⚠ 另可靠【时·支配者】每5颗魂能结晶+1秒 |

⚠ **⚠ 实战写照**:⚠ **它对魂大人的作用是**夺走「极致升华」〔灵孔在技能列表中的技能名〕，让其强开的第13灵孔当场把自己的头炸开****;⚠ **⚠ 对冥神则是**永久性夺走终极能力「神之骸」****;⚠ **⚠ 一个不算缺点的缺点**:⚠ **敌人被停固过一次后会产生1～2点的极低抗性**——⚠ **能做到的极限只是「知道停固期间发生了什么，却依然什么都做不了」，**反而更绝望****。
