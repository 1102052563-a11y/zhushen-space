# 六式（海贼王世界 · 体技）

> 海贼王世界的六式体技。

海军与 CP 部门的高等格斗体技体系（剃、铁块、月步、纸绘、指枪、岚脚六门，及奥义「六王枪」）。学六式的真正门槛不是招式（原理都不难），而是**超凡体质**——须先经残酷训练把身体强度练到能承载体技的程度（换算成乐园属性约**力 25~30 / 敏 25~30 / 体 25~30**）。

- **剃**（苏晓所求·补缺的高速位移）：原理是**在 0.36 秒内踩踏地面 10 次以上**，靠反作用力层层叠加爆发、快到肉眼难辨（像「二次、三次…十几次加速」）。弊端：须精控踩踏力度（否则撞墙 / 撞上敌人武器送命）、且**途中无法快速变向**（只能小幅修正路线）。
- **铁块**：身体瞬间硬如钢铁、硬抗攻击（CP9 [布鲁诺](../../../人物/index.md) 曾以铁块硬扛炸弹爆炸毫发无伤·五卷生命归还等更高阶体技苏晓体质暂不够）。
