# 近战宗师

> 近战系宗师，Lv.80 → Lv.95。

## 🦵 近战宗师 Lv.80

> 原属批次:三技法宗师 · Lv.80 的极致分离（51卷17–20章）


⚠ 【**本次提升，已让近战宗师的部分特性达到极致，即将进行分离，你将永久获得以下被动特性。**】

| 分离出的永久被动 | 效果 |
| --- | --- |
| **宗师之力（被动）** | ⚠ **近战时，踢技攻击力提升 1000%**（⚠ **此为近战宗师在踢技攻击力增幅方面所能达到的极限增益**） |
| **极致反射（被动）** | ⚠ **基础·神经反射速度 +160 点，基础·动态视觉 +160 点**（⚠ **极限增益**） |

- ⚠ **【近战宗师:Lv.80（技法·被动）】**:⚠ **踢技伤害阶位 +10，肉体强韧性提升 48%。**
- ⚠ **Lv.80 终极能力·力量穿透·踢技七次强化（奥义级·被动）**:⚠ **踢技攻击判定提升至「飞升级」**——⚠ **并有概率让敌人触发「死亡回溯」效果，以此降低敌人的规避概率。**
- ⚠ **⚠ 何为「死亡回溯」**:⚠ **苏晓一脚直踹向敌人，敌人因直面死亡，而在短时间内回溯自己的一生**——⚠ **虽说只是那么一瞬间，但也算是种控制效果了。**
- ⚠ **不仅如此**:⚠ **近战宗师的各类能力，如同当初刀术宗师被选为核心技法时一样，都进阶到奥义级**——⚠ **这就是 2 点原初技能点所带来的提升**;⚠ **当然，这得是近战宗师的基础够好，否则有原初技能点也没用。**
- ⚠ **他为什么升它**:⚠ **近战宗师与血枪宗师虽没可能复刻刀术宗师的强大，可二者各有特点**——⚠ **单论极限攻击，那肯定是直踹**;⚠ **足足 80 级的近战宗师，全部加在这一脚直踹上**——⚠ **⚠ 眼下，他要将直踹再度提升一个高度，也唯有如此，才可能与无光神殿·四巨头交锋。**
- ⚠ **一个未使用的选项**:⚠ **之前苏晓为何不以直踹和突进斩的 [城主夫人](../../../人物/永光世界/城主夫人.md) 正面硬碰？**——⚠ **他能确定，自己一脚能把突进斩来的城主夫人踹没，当场秒杀这大 boss**;⚠ **但他自己也需要付出一条右腿、以及整条右臂、还有部分肩膀与躯干为代价。**
