# 自创刀芒招式

> 无冷却、无固定数值，威力全凭使用者把握。

## 自创刀芒招式（无冷却 · 无固定数值 · 威力凭使用者把握）

> 原属批次:自悟技能

苏晓用刀日久、自创过三十多种招式，**实用的仅以下几种**（开发招式极危险：他曾开发到一半把自己刺成刺猬险死·超十分之一的强者死于开发招式）：

- **环断**（八卷32章）：拔刀斩掺入刀芒、向四周扩散成**直径十几米的圆形 360° 杀伤**——围攻时清场极实用（曾一招秒杀几百刺蝎 / 契约者）。开发方向 = 增斩击力 + 扩范围；融青钢影（能量·与刀芒性质相斥）失败。
- **十字**（八卷38章）：十字形刀芒、开路用。
- **连鞘斩 ·「破空」**（十一卷53章开发、66章成型实战·苏晓自评最强招）：**居合式拔刀斩**——把约 **300 法力**化作青钢影能量、连鞘包裹斩龙闪（出鞘时迸出极强切割力的蓝光），再以极慢蓄势后骤然斩出「破空」。特点 = **速度极快**（敌在几十米外亦可瞬间斩到）+ **威力远超环断**，是苏晓「最强的攻击招式、没有之一」。**十一卷66章首次成型实战**：在地下停车场迎 [英雄王](../../../人物/吉尔伽美什.md) 的「天地乖离之剑·红色剑风」，他先以几十面反击盾拼成**全身能量铠甲**护体、再用「破空」冲入剑风——一道蓝光直穿剑风、斩断英雄王持剑右臂（自己也被剑风重创成独臂）。**对单 / 对众两种用法**（十二卷30章·战矮人大地骑兵团时阐明）：① **对单**——斩击力收敛、命中瞬间青钢影 + 刀芒侵入敌体、几秒后在体内形成 [影刺](../index.md) 从内重创 / 杀敌；② **对众**——完全释放斩击力，一道直线蓝斩芒掠过、沿途炸开纵横交错的斩击（黑土要塞高地战中一招把几十名矮人大地骑兵斩到仅剩 4 人）。代价是对身体负担不小（用后踉跄脱力）。**「破空」二段攻击（十三卷17章揭示·战 [神裂火织](../../../人物/神裂火织.md) 时用）**：破空真正最强的不是一段斩击、而是**二段**——一段命中后、青钢影会潜伏在敌体内，之后陆续在敌体内**生成能量刺、从内一根根破体而出**（须苏晓在场控制方能同时齐发·不在场则一根根刺出·敌人只能用自身能量勉强控制刺穿位置、避开要害·过程极痛），配合青钢影持续噬敌能量造真伤——即便苏晓撤走、敌人仍会被二段持续折磨（神裂火织被斩后体内不断刺出能量刺、险些丧命）。
- **寸断**（九卷25章·环断分支）：把环断的「360° 刀芒扩散」改为**只攻正前方、且不放刀芒、直接以青钢影震荡的斩龙闪（化能量刃）猛斩**（单次猛耗千点法力）——近身一击的爆发斩。九卷以此当头斩中三忍 [自来也](../../../人物/自来也.md)、在其胸膛留半米深伤 + 青钢影侵入令其查克拉暴走（自己也被自来也的钢铁白发刺穿身体·险中心脏）。
- **影刺**（九卷13章·青钢影衍生）：环断融青钢影失败后，改把**青钢影「包裹」在刀芒上**——刀芒生出半米长、手指粗的**淡蓝色尖刺**，穿透力极强（超过普通刀芒）。设想用法：把斩龙闪刺入敌体后令尖刺**由内向外刺出**，敌人瞬间变刺猬、近乎一击必杀（现仅半米长·增长需大量青钢影）。命名走苏晓一贯的「实用流」（不取「天马环星斩」之类）。**九卷34章首次实战**：把斩龙闪刺入 [我爱罗](../../../人物/我爱罗.md) 的沙球后激发影刺、十几根淡蓝尖刺将我爱罗半个身体刺穿（一臂垂落）——隔着沙球破防的开局杀招。
