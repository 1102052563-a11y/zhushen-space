# 魂雷体系的起源

> 灭法与灵魂国度的互授。

## ⚡ 魂雷体系的起源（灭法与灵魂国度的互授）

> 原属批次:续 · 灵魂秘术与青影王实战（52卷7章 / 10–11章）


⚠ **灭法·雷霆之魂（被动）:每 3 点元素亲和力 +1 点魂雷抗性**——⚠ **苏晓早疑惑灭法体系前期无魂雷、后期却突然出现**;⚠ **⚠ 真相:魂雷是灵魂国度的能力**——⚠ **先代灭法把「青影王（青钢影能量拟态雷电化）」的原理教给盟友灵魂国度→灵魂国度研发出「灵魂拟态雷电化」的魂雷体系→又把魂雷教回灭法→灭法据此开发出「灵魂感电」**;⚠ **⚠ 但青影王是顶尖梯队能力、魂雷是第一梯队，无法用第一梯队原理反推顶尖大招（逻辑性问题，非天赋强弱）。**
