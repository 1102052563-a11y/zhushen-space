# 血之恶意

> 负魅力衍生出的追猎能力。

## 🩸 血之恶意（−21 魅力衍生的追猎能力）

> 原属批次:斩始祖一战的能力清单（51卷26–29章）


⚠ **始终显示灰色、无法主动激活**——⚠ **当苏晓被 [灵妖](../../../人物/灵妖.md) 的厄运暗算、触发「不灭之运」反弹后，它自行激活**;⚠ 【**你的「血之恶意」能力已激活，猎物已锁定**】。

| 追猎特性（−21 魅力衍生） | 效果 |
| --- | --- |
| 基础 | ⚠ **移动速度提升 50%～200%**（⚠ 距离越远越快·消耗血气值） |
| **持续追踪**（被动） | ⚠ **同世界内可持续追踪目标方位**·持续 **23 小时 59 分 58 秒** |
| **血之余韵**（被动） | ⚠ **即使对方以隐匿类能力移动，依然会在沿途留下血气痕迹** |
| **战利品**（被动） | ⚠ **击杀必得猩红宝箱**，内含猎物 **15%～20% 总财富** |

- ⚠ **激活时的画面**:⚠ **格外沉闷的心跳声「咚～」**——⚠ **视野骤变:大地一片苍白，天空中犹如血焰在燃烧。**
