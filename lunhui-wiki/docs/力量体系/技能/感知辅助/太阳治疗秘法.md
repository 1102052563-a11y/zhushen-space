# 太阳治疗秘法

## 🌞 太阳治疗秘法（约 Lv.85）

> 原属批次:续 · 完美格挡（永久掌握）、自然之语与太阳治疗秘法（53卷34–40章）

- ⚠ **苏晓向四位太阳学者学习「治疗秘法」，凭药剂学天赋让其进化成「太阳治疗秘法」**（⚠ 假设原本 Lv.30，现至少 Lv.85）;⚠ **⚠ 特性:效果非常顶、愈后毫无副作用，但治疗过程「有点粗鲁」**（⚠ 太阳学者评价「治疗效果虽一般，但手法还是很温和的」→可反推有多粗暴）;⚠ **⚠ 治 [命运女神](../../../人物/虚空/命运女神.md) 魂伤时，粗暴到把她中途弄醒，这也是他后续把她坑进噩梦副本的起因。**
