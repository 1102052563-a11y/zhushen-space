# 炼金学 · 植物调和篇

> 永久增益药剂的解锁路线。

## 炼金学·植物调和篇（永久增益药剂 · 二十三卷36章解锁）


苏晓在 [黑渊](../../../世界/任务世界/黑渊浅层-赔罪者之殇.md) 开 [圣扎迦](../../../人物/圣扎迦.md) 的史诗宝箱、得「**炼金学高阶知识·植物调和篇**」（史诗级知识卷轴·前置炼金药剂学 Lv.20 + 炼金生物学 Lv.20 + 真实智 90）——由此**能制造「永久增益药剂」**（这是新里程碑：以往只能造恢复 / 临时增益药剂）。

- **难度极高**：调配永久增益药剂难度是恢复药剂的 ~20 倍以上（分离 / 调配 / 稀释·温度 / 时间 / 配比稍差即爆瓶）。Lv.1 初期成功率 0%、苏晓连失败十几次（每瓶 ~2.8 万币 + 300 法力）才出第一瓶、之后成功率快速升到 90%+。
- **配方随等级解锁**：Lv.1「活力药液」（永久 +2 体）→ Lv.3「圣痕药剂」（永久 +1 力 +1 敏 +100 生命）…品质越高的药剂制造越繁琐（详见 [消耗品材料](../../消耗品材料/index.md)）。触发幸运属性可出「完美」品质（效果翻倍）。
- **意义**：永久增益药剂是「**新人契约者的亲爸爸**」（一阶砸锅卖铁也要买）——既是苏晓的长期暴利经济来源，也让他加入任何势力都有「能造永久增益药剂」的话语权。
- **海贼世界冲到 Lv.25（二十五卷18–52章）**：知识类技能可在衍生世界内提升（每级恒定 1 万币 + 熟练度）。苏晓在 [顶上战场](../../../世界/任务世界/海贼王-顶上战场.md) **白嫖革命军材料**专练——Lv.10 解锁「黎明之焰」（+3 力 +3 体·熟练度收益是圣痕的 15 倍）、返园前停留 5 天一路刷到 **Lv.25**。Lv.20 起解锁高阶药剂「**圣龙守护**」（皮下改造出 300 强度内防护层·挡一次四阶以下诅咒 / 病毒 / 微生物）与「**活性·力**」（永久 +6 力·80 点以下生效·重复使用致力量暴走）。**目标 Lv.30**——届时可解锁 80 点真实属性以上能用的配方（苏晓自己也能服用）。等级越高配方越稀有（高到一定程度甚至无新配方·但仍要点满防今后高配方因等级不足无法掌握）。
