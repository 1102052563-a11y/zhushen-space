# 唤醒之碑 · 通用被动

> 【强者之魂】解锁的通用被动。

## 🗿 唤醒之碑 · 通用被动（【强者之魂】解锁）

> 原属批次:续 · 刀术宗师 Lv.95、双刃与深渊学（52卷52–61章）


⚠ **⚠ 平砍即是大招的流派终于可以继续堆了**——⚠ **「通用被动」基本 Lv.10 满级（满级后立即从技能列表消失，但提升永远都在）**;⚠ **⚠ 因【起源石·世界】加成，苏晓的通用被动满级达到 Lv.20，极少部分原本 Lv.20 满级的飙到 Lv.30**;⚠ **⚠ 总计几百种。**

| 通用被动（Lv.EX） | 效果 |
| --- | --- |
| **命魂强化** | ⚠ **永久 +60000 点生命值**（⚠ 1 点灭法技能点＋35000 钱币·⚠ **血赚到让人瞠目结舌**·⚠ 这般性价比在碑上也是极少的） |
| **深渊游荡者** | ⚠ **+10 点深渊抗性**（⚠ 35 万钱币·⚠ 深渊抗性越高提升难度越大） |
| **体质强化** | ⚠ **全属性 +5**——⚠ **⚠ 灭法阵营认知中的「全属性」＝力、敏、体、智、**深渊抗性****（⚠ **没有魅力属性**·⚠ **「老灭法、马文·华尔兹、席曼·阿奇德、刀魔、格林·吉莉安，这些人挨个看，哪有人能魅力属性高？」**） |
| 青钢影·强化 Ⅰ～Ⅴ | ⚠ Ⅰ:每次燃烧敌人法力值 +200 · Ⅱ:每次近战真实伤害 +400 · Ⅲ:傲歌强度 +30% · Ⅳ:提升 10 级青钢影能力 · ⚠ **Ⅴ:可对青钢影战斗形态进行一次变更**（⚠ **唤醒之碑上总计铭刻 105 种青钢影战斗形态，其中三种为灭法者们公认的最强**·⚠ 苏晓当前是「震爆」·⚠ **他暂时延后了这个选择**） |
| 其他 | ⚠ 不动如山 +60 击退抗性 · 灵体增幅 +7500 法力值 · 强大体魄 +40 肉体强度 · **界雷特性身体细胞活化**:身体速度 +0.51 倍 |

- ⚠ **⚠ 累积的意义**:⚠ **苏晓已是绝强级极限梯队——逛大聚地时，以往那些能提升 10～20 点力量属性的被动，现在只提升 0.2～0.5 点，有些干脆用鲜红字体标注「你掌握此技能无任何增益提升」**;⚠ **⚠ 「如果设身处地想想，施法者们对付人均几百种基础被动、外加高额免疫法系伤害，攻击力、速度、生命值、防御力都高到不讲道理的灭法者们，也挺绝望的」**——⚠ **唯一的好消息是每代灭法者只有几名，坏消息是这几名都是上述的实力梯队标准。**
- ⚠ **⚠ 唯一被驳回的提升**:⚠ **「噬魔体质」**——⚠ **它属于「现状态稳定，一旦有所变动立即不稳定到极致」的能力，提升等级上限有超过九成概率带来死亡性恶果**;⚠ **⚠ 轮回乐园的判定直接摒弃了这次提升，触发了猎杀烙印的保护机制**（⚠ 硬要作死也能做到，但苏晓不准备这么做）。
