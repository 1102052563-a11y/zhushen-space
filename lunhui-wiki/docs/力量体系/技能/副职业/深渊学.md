# 深渊学

> 黑暗系鉴别的进阶。

## 🌌 深渊学（黑暗系鉴别的进阶）

> 原属批次:续 · 刀术宗师 Lv.95、双刃与深渊学（52卷52–61章）


⚠ **【黑暗系鉴别】Lv.50 有三条进阶分支:黑暗物配制 / **入门级深渊学** / 灵魂药剂学**——⚠ **苏晓付 1000 盎司时空之力开启相关深渊知识资料库，选了深渊学**;⚠ **⚠ 这能力的特殊之处:可用**时空之力**而非灵魂钱币提升，且**费用不随等级递增****（⚠ 连 Lv.95 猎杀权限都查不到更详细的原因）:

| 阶段 | 每级费用 | 苏晓进度 |
| --- | --- | --- |
| 入门级深渊学 | ⚠ 0.5 盎司 | ⚠ **已满**（49.5 盎司） |
| 初级深渊学 | ⚠ 5 盎司 | ⚠ **已满**（495 盎司） |
| 中级深渊学 | ⚠ 50 盎司 | ⚠ **已满**（4950 盎司） |
| 高级深渊学 | ⚠ 500 盎司 | ⚠ **顶不住**（需 49500 盎司·⚠ 且深渊学之上百分百还有进阶梯队，若按 5000 盎司/级，升满要 **495000 盎司**） |

- ⚠ **它值这个价**:⚠ **「这是深渊，是能让虚空万界繁荣，也能让虚空万界毁灭于无尽黑暗的两种无上力量之一，想了解它，当然要付出很大代价。」**;⚠ **⚠ 实例:知识中记载的「**烟噬**」这种不死不灭·深渊滋生物，硬怼的话至强者中上游都不是对手**——⚠ **可只要造出一种超强的空间震荡装置，在其作用范围内交锋，绝强后期战力就能把它伤损到无法动弹**（⚠ **有时足够了解，比空有蛮力强百倍**·⚠ 而作为灭法者的苏晓可以直接用魔刃吞噬）。
- ⚠ **⚠ 顺带的成果**:⚠ **他靠「高级深渊学」＋顶尖药剂大师的手法，用 0.18 盎司「特异深渊能量」增益出了给 [白牛](../../../人物/白牛.md) 的【命源药剂】**——⚠ **⚠ 「放眼虚空万界，或许只有苏晓自己能调配出这瓶药剂。」**（⚠ 思路:白牛的「伤损细胞」源自渊龙的深渊之力且是**概念级**，那就**用深渊能量去滋养它，逼它作为一个兽王细胞存在**——⚠ 不能痊愈，但可在一段时间内完全遏制伤势·⚠ **白牛喝完当场呼呼大睡**）。
