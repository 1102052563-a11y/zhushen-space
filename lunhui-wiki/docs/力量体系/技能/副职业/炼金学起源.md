# 副职业 · 炼金学（四卷69–70章）

> 炼金学副职业的起点（第四卷）。

苏晓从 [夏洛克](../../../人物/夏洛克.md) 遗留的布袋里得到**【炼金学（完整）】卷轴**（产地斩·赤红之瞳·**五阶卷轴**·知识类技能分一~九阶·需力 10 / 体 10 / 智 25）。使用后**炼金学归入「副职业」列表、单独计算、不影响灭法之影**——这是苏晓的**第一个副职业**。

- **定义**：炼金学起于人类对黄金的渴望、发展成对大自然的探索与对物品 / 生物的改造强化，含生物学、锻造、机关制造等（苏晓只学到很基础的部分·相当初高中知识·现实不会教）。
- **风险**：学了**生物改造学**会令一些剧情人物对他生敌意；炼金学高等级后衍生世界的**科研阵营会优先招揽、甚至有大型势力绑架**他的风险。
- **技能树（三层·各层独立升级解锁下一层）**：
    - **第一层·炼金炸弹制造**（苏晓已 **Lv.20 满级**·六卷42章）：能造各级「[精神力炸弹](../../消耗品材料/index.md)」。**炼金炸弹只能在轮回乐园内制作**——**除非在衍生世界备齐所需材料**（六卷在海贼王世界，苏晓把材料配方交给天才博士 [贝加庞克](../../../人物/index.md) 找替代品 + [T3 药剂](../../消耗品材料/index.md) 回满法力，就在世界政府实验室里**大批量量产**·一天可造 1500~2000 颗·替代材料造的呈乳白色、威力略低但品质不降）。
        - **冲满 Lv.20**：替世界政府量产飞快刷熟练度（熟练度堆到 2073），回卧室以「500 币 + 熟练度/次」一口气从 Lv.3 升到 **Lv.20 满级**。满级三选一奖励（A 快速制造 / B 范围 / C 威力 / **D 精密操控**），苏晓选 **D 精密操控**（炸弹只是辅助、后期靠刀术，故强化「对炸弹的控制」）。
        - **解锁配方**：高级精神力炸弹、**[特级精神力炸弹](../../消耗品材料/index.md)**（绿·270 伤·可变形/分离）、**[太阳神阿波罗](../../消耗品材料/index.md)**（微型太阳·半公里·需灵魂结晶）。**「炼金药剂学」（第二层）仍须返乐园才能解锁**（满级提示）。
    - **第二层·炼金药剂学**（六卷80章解锁 → 七卷26章已升 **Lv.6**）：第一层达 Lv.10 后花 灵魂结晶（小）+10000 币解锁；七卷在钢炼世界靠**贤者之石**当材料（每瓶耗石 20 能量 + 40 法力）大量量产、把它从 Lv.1 一路冲到 **Lv.6**，解锁 [低级 / 中级炼金秘药](../../消耗品材料/index.md)（回血）、[林荫药剂](../../消耗品材料/index.md)（随身回体力 + 提神）等配方（再升还能造「大药」）。**升级解锁配方无固定规律**（有时升一级解锁、有时两三级）。另：太阳神阿波罗乐园造价 8 万币 + 灵魂结晶 + 500 法力（太贵·放弃量产）；特级炼金炸弹仅 300 币 / 颗。
    - **第三层·炼金生物制造学 Lv.1**（十二卷101章解锁·灵魂结晶〔中〕×1 + 1 万币·炼金学技能树就此全开）：能造**炼金生物**（人工召唤物·**非活物、可收进储存空间**·有「存在时间」限制〔越高级越久〕、眼中法阵失光即报废、不可充能修复·激活耗 1 法力即成召唤物）。苏晓之前嫌操控召唤物分心而没解锁，**召修斯后悟到：炼金生物当肉盾、控制权交 [布布汪](../../../人物/布布汪.md)**（布布汪 = 业余奶妈 + 顶级斥候 + 隐身召唤师），自己专心战斗——配自身 **79 智力**加成做「不怕死的可再生主坦」。
        - **炼金渡鸦**（首制·150 法力 / 乐园币代材 4000 一只）：初始属性辣鸡，但经苏晓 79 智力加成→**存在 15 天·力29 / 敏62 / 体32 / 智18 / 魅14·视觉共享 700 米〔耗 10 法力 / 分〕+ 急速飞行 586km/h**·首个充当远程侦查 / 肉盾的炼金生物（召唤上限 1/1·未学召唤类基础技能）。
        - **迷火森林速成（十四卷59–64章）**：恶魔族盟友 [莉莉姆](../../../人物/莉莉姆.md) 把苏晓带到虚空炼金师 **[树贤者](../../../人物/树贤者.md)** 的迷火森林（有古书【炼金生物起源】+ 海量材料 + 每秒回 5 法力的「魔力源泉」），让他**集中刷炼金渡鸦**。近战出身的临场反应给了炼金不小加成，他**越造越快**：10 分钟 / 只 → 5 分钟 → 2 分钟 → 1 分钟 → 46 秒 → **32 秒 / 只**（树贤者都被这「奇怪的掌握方式」惊到、却嫌「快≠精」）。两段共制 896 + 上千只，**炼金生物制造熟练度堆到 1876/10**——足够把炼金生物制造学冲到 **Lv.20 满级**（原想靠树贤者指导突破 Lv.20、被瑟菲莉娅搅局作罢）。期间他还造了枚「**阵图坐标接收器**」（贴敌身上、可发「攻击」信号让所有渡鸦群起攻之）。**炼金要诀**：核心刻法阵反成弱点（不可取）；**绝不给炼金生物独立意识**（有意识便向往自由、反杀造物主·像有灵魂的狼管家是特例）——苏晓本就让战斗交 [布布汪](../../../人物/布布汪.md) 操控、渡鸦只需力 / 速。带出的近 900 只渡鸦能否进乐园未知（若能即可拿去交易市场发财）。
        - **升满 Lv.20（十四卷77章·乐园）**：返园后花乐园币把炼金生物制造学从 Lv.1 一路升 Lv.20 满级（升级费比炸弹 / 药剂学贵得多·Lv.1~9 每级 5000、Lv.10+ 每级 1 万）——满级解锁三张内置图纸「死亡爪（敏）/ 监督者（侦查）/ 巴泽尔（力）」；要更高级图纸须另购（从 [黑帆商会](../../../人物/index.md) 黑商 24K 买金色「凛冬牛头人」图纸·5 万币 + 一号药剂）。**炼金生物制造的门槛极高**（高级生物耗 12 万~46 万币替代材料 + 灵魂结晶〔中〕×2~3 + 3000 法力·失败全报废），故鲜有契约者学。
        - **凛冬牛头人「[阿姆](../../../人物/阿姆.md)」（首只高级炼金生物·十四卷80–81章）**：苏晓亲造的体力 / 控制型炼金生物（详见 [阿姆](../../../人物/阿姆.md)·力75/敏69/体85/智30·存 256 天·6 技能·霜冻减速叠 10 层 + 终极冻结·减伤 25% + 生命 +1000 极抗揍）。**炼金要诀实证**：只给它极低智慧（智压到「1+1=3」·防其向往自由反杀造物主）；指定为召唤物受 80 智力加成；战斗控制权交 [布布汪](../../../人物/布布汪.md)（苏晓专心挥刀）。苏晓暂造不出狼管家那种能存几百年、有独立意志的炼金生物。
- 升级靠**乐园币 + 熟练度**（熟练度=实际制造次数）；知识类技能升级是「大量知识灌入脑中」、不必去强化大厅。
