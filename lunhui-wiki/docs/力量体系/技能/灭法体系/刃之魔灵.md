# 刃之魔灵 · 觉醒意识

> 刃之魔灵觉醒出独立意识。

## 🌑 刃之魔灵 · 觉醒意识

> 原属批次:续 · 噬魔体质、刃之魔灵觉醒意识（52卷14章 / 21章）


⚠ **吞噬一只至强级·不死不灭·深渊滋生物的本源能量后（相当于五份魔灵源质），刃之魔灵魔灵强度 410→560**——⚠ **按吞星·阿卡斯的比例表（800 灵魂:400 魔灵 / 1000:500 / 1100:600 / 1200:800 / 1300:1100·过 1000 点质变），560 魔灵需约 1060 点灵魂强度才能压制**;⚠ **⚠ 苏晓现有 890 点——「互相伤害」环节开始**（⚠ 靠永恒级【封魔】刀鞘顶住·后续须尽快把灵魂强度提到 1100）。

- ⚠ **魔灵的形象**:⚠ **铁砂般黑蓝色烟气构成的女性人影·骨质面具（右侧几道很淡的蓝色裂纹）·御姐身材**——⚠ **⚠ 所有灭法者的魔灵都必定不会言语、不会发声（如同定律/规则）**;⚠ **⚠ 这种能量态最强（无视物理/能量攻击），化为肉体属于自行削弱。**
- ⚠ **性格**:⚠ **高冷 + 暴戾**（⚠ 苏晓最担心它变成巴哈那种性格——他不怕侵蚀，是顶不住巴哈式精神攻击）——⚠ **它的第一种情绪是冷淡，随即因被夺七成魔灵能量、今后还要分五成而愤怒**;⚠ **⚠ 苏晓的「劝说」:「从某种程度上来讲，我可以算是你的父亲」→ 魔灵当场扑杀 → 被封魔刀鞘吸回。**
- ⚠ **⚠ 最终决断**:⚠ **到至强级中后期就是他与刃之魔灵作最终决断之时**——⚠ **魔灵胜则新的「刀魔」现世，苏晓胜则魔灵意识完全沉寂**;⚠ **⚠ 好消息:苏晓当初选择把魔灵封在斩龙闪内，故只是偶尔「互相伤害」，而非时刻侵蚀干扰**;⚠ **⚠ 且进入与强敌的战斗时，魔灵侵蚀会瞬间消退并与灭法者一同斩敌**（⚠ 这是魔灵绝不违反的本能——它比谁都不希望共存的灭法者丧命）。
