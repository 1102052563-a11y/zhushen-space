# 灭法体系的终点

> 灭法体系走到尽头的形态。

## 🗡 灭法体系的终点


| 能力 | 终局形态 |
| --- | --- |
| **青钢影 Lv.EX** | 消耗改为**每分钟 1000 点青影王能量**。每次近战攻击燃烧敌人 **50% 最大法力值**（或其他身体能量）并造成**燃烧量 ×3.25 倍**真实伤害（⚠ 至少 15 颗命核体量）；⚠ **敌人身体能量燃尽后进入「剧烈燃烧」——无需敌人身体能量即可继续燃烧，后续每刀附加可破坏至少 30 颗命核的真实伤害**（伤害翻倍。⚠ 这是因为现在开启青钢影消耗的是「青影王能量」，而青影王的特性之一就是敌人身体能量消耗越多、造成损伤越高）。⚠ 其「防御形态·傲歌」**独立成为单独能力**:可生成任意形态的晶体防御，强度与所耗青影王能量成正比（单次上限为当前青影王能量上限的 10%），已拥有迄今为止最顶尖的防御强度之一 |
| **魔灵觉醒 Lv.EX** | 刃之魔灵强度 **Lv.EX**；「替换」⚠ **无使用次数上限**、⚠ **使用距离覆盖整个星界（最远可达星界中心至深渊漂游之地）**；「力量赋予」比例升至 **70%~80%**；新增 ⚠ **「体魄赋予」——魔灵继承苏晓 50% 的四项无上属性**，且**无命核、在苏晓未死且斩龙闪未毁的情况下无法被消灭**（⚠ 苏晓死后 38000 个自然日魔灵开始沉寂，约 456000 个自然日进入深度沉寂，直至漫长岁月后彻底消亡）；新增 **「誓约·灵魂囚罪」**——魔灵位于苏晓周边 500~50000 米内（可自控）时，其意识被灵魂力量绝对压制而处于混沌状态；⚠ **脱离这个范围，它将拥有逐渐明确的自我判断力与行动力** |
| **龙影闪 Lv.EX** | ⚠ 有出乎预料的高上限:**单次消耗超过 7 万点青影王能量后效果质变**——从「无界状态」变为穿透到一种 **「虚无状态」**，⚠ **真正能豁免攻击，哪怕是来自同阶无上之境的攻击** |
| **极之影 Lv.EX** | 冲破绝对极限 **两次**；提升规律变为「每累计 10 点全无上属性提升一次」；⚠ **它是艾薇拉用「时间之眼」洞悉不到的东西**——她只看到「天锋斩影已满级」，因此判断让苏晓衰老只会削弱他 |
| **天锋斩影 Lv.EX** | 最终达到极限。中途数据:Lv.335（全技法 +7975%）→ Lv.390（**+10975%**、极刃基石 1179、⚠ **无我之境 89 倍**，达到 80 倍后提升难度「巨大幅度」提高） |
