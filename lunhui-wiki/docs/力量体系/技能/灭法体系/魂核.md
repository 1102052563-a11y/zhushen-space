# 魂核 · 格林·吉莉安的传承

> 来自格林·吉莉安的魂核传承。

## 🌀 魂核 · 格林·吉莉安的传承

> 原属批次:三技法宗师 · Lv.80 的极致分离（51卷17–20章）


⚠ **详见 [格林·吉莉安](../../../人物/格林·吉莉安.md) 页的「三颗传承结晶」一节。**

| 魂核槽占用 | 斩魂强度 |
| --- | --- |
| **1 个** | ⚠ **12～15** |
| **8 个** | ⚠ **30～32**（⚠ 她本人的过渡形态） |
| **10 个** | ⚠ **35～37**（⚠ **不推荐**——⚠ **极大概率导致斩魂·魂核永久破损**） |
| **超脱之境** | ⚠ **50**（⚠ **相当于有 36 个魂核槽才能增益出的强度**） |

- ⚠ **断魂影最多能拓展出 12 个魂核槽。**
- ⚠ **苏晓现有六颗魂核**:⚠ **青钢、急速、斩魂、刃灵、落雷、黑暗。**
- ⚠ **【黑暗·魂核（进阶·被动）】**:⚠ **临时融合所有魔能能量，让其大幅度增强你的生命力与身体防御力梯阶，以及深渊抗性自适应提升**——⚠ **但移动速度总体降低 93%～95%**;⚠ **⚠ 此移动速度削减拥有绝对优先权，无法以任何方式抵消**——⚠ **⚠ 此魂核的切换速度较慢，且需要在稳定的环境下才能切换至此魂核**;⚠ **（⚠ 极暗古堡五层的黑暗封印区，正是靠它硬闯过去的——⚠ **走出时生命值只剩 56%**）。
