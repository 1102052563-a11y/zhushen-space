# 青钢影

> 从 Lv.20 一路推到无尽 Lv.147、再到完全觉醒 Lv.185 的核心杀招。

⚠ **本页合并了原「技能·功法」页中散落 6 处的记载**，按卷序排列。

## ⚡ 灵魂感电（青钢影控制特性）

> 原属批次:斩始祖一战的能力清单（51卷26–29章）


| 项目 | 内容 |
| --- | --- |
| 效果（被动） | ⚠ **青钢影能量燃烧敌人身体能量时，15% 几率触发麻痹**（⚠ **绝对控制，无法豁免**·0.2～3 秒） |
| ⚠ 叠加机制 | ⚠ **每次未触发就 +20% 触发概率，直至 100%**——⚠ **下次附带青钢影的斩击命中必导致「绝对麻痹」** |

- ⚠ **实战**:⚠ **苏晓斩始祖的刀数早就超过六刀**——⚠ **当触发概率到 100% 后，就可以决定哪一刀附带这能力**;⚠ **他用一记加持灵魂感电的斩击麻痹始祖一瞬，接一记「太阳直踹」。**

## ⚔ 青钢影·战斗形态三选一（【青钢影·强化Ⅴ】的兑现）

> 原属批次:续 · 青钢影·极限者与凯瑟·休娜的三形态（53卷1章 / 5章）


⚠ **唤醒之碑上总计铭刻 105 种青钢影战斗形态，其中三种为灭法者们公认的最强**——⚠ **苏晓把原本的「震爆」（有明显施展前置动作·对战初始女巫时刚要用就被拍回去）换掉**;⚠ **⚠ 三种候选都带 **凯瑟·休娜** 那一代或先代灭法的开创者留言**:

| 形态 | 效果 / 开创者留言 |
| --- | --- |
| **青影化身** | ⚠ 消耗 10% 法力上限构成化身（最多五具）·⚠ 开创者=**第二纪元·灭法之影·凯瑟·休娜**:「还有什么，比以多打少更快乐呢？况且能舔老娘晶体化身的脚趾，已是你毕生最大的荣幸」 |
| **致命绽放** | ⚠ 攻击叠「死之绽放」·五层时青钢影晶体化爆发从内部刺破敌人身躯（30% 最大生命真伤·<30% 即死）·⚠ 前置=**未掌握魔刃能力**·留言:「为那些没能掌握魔刃能力的灭法所准备，没掌握魔刃的灭法之影也是灭法者，也是的！」 |
| **⚠ 极限者（苏晓所选）** | ⚠ **青钢影消耗法力值提升 5 倍、所燃烧法力值提升 2 倍**·⚠ 前置=法力 12 万＋、青钢影用「初代指骨」提升到极限潜力且 Lv.85＋·⚠ 留言:「我毕生所求，仅有极限」 |

- ⚠ **⚠ 为何选极限者**:⚠ **选它后一刀所附带的真实伤害达到 **21710 点**（原 11295）**——⚠ **⚠ 「21710 点真实伤害和 21710 点常规伤害根本是两种概念:后者对大 boss 只伤及皮肉、伤口两三公分深，前者哪怕大 boss 也要被斩断骨头、位置选好当场开膛破肚」**（⚠ **青钢影真伤在刀子入肉后立即生效、燃烧身体能量后该区域防御力骤减·从来不只是附属伤害那么简单**）;⚠ **⚠ 代价:青钢影每分钟法力消耗飙到 4100 点，苏晓 145449 法力只能开 35 分 28 秒——所以战斗中并非一直开着。**

## 🔵 青钢影 · 完全觉醒（Lv.185）

> 原属批次:续 · 至强巅峰体系、三种巅峰能力与青钢影完全觉醒（54卷141–170章）

⚠ **在至强巅峰前敢尝试彻底觉醒这星界能排进前三的核心能力，必死无疑**——⚠ **这也是三位老灭法此前从未提及此事的原因**;⚠ **⚠ 被吞噬的两项加成**:⚠ **「疾影·技能效果3〔技法型能力/身体能量/技能真实伤害+40%〕」与「青影王衍生特性·猎影〔青钢影强度提升、真伤+30%〕」被完全觉醒的青钢影**彻底吞噬**，不再显示于分支技能列表**:

| 项 | 觉醒后 |
|---|---|
| 使用条件 | ⚠ **每分钟消耗51000点法力值** |
| 主动效果 | ⚠ **每次近战攻击燃烧敌人 24% 最大法力值（或其他至强身体能量），并造成燃烧量×1.25倍的真实伤害 = 30% 最大生命值真实伤害**·⚠ 敌人承受法力燃烧后的「剧烈疼痛」 |
| 防御形态·傲歌 | ⚠ **单次法力值消耗上限 30万点 · 晶体层防御力阶位 +85** |
| ⚠ 搭配「青钢·魂核」 | ⚠ **综合强度+40%→燃烧33.6%最大法力值、真伤达 42% 最大生命值**——⚠ **⚠ 但「青钢·魂核」只能增益「燃烧敌人法力值的强度」与「傲歌晶体强度」**（⚠ 青钢影分三个状态:**状态1**=附着在刀上的能量，无真实伤害判定，对应燃烧敌人身体能量;**状态2**=斩伤敌人后开始噬灭其法力值;**状态3**=造成真实伤害·⚠ 魂核不可能增益尚未发生、处于不确定未来的状态3） |

⚠ **⚠ 意义**:⚠ **对战祖神时青钢影一刀近10万真伤，对千万级生命值的双方算不上大伤损，苏晓已把它当成最高等的「穿甲」用**——⚠ **他终于明白:并非青钢影不行，而是没到至强巅峰就无法承受完全觉醒的青钢影**;⚠ **⚠ 「断魂影」三魂核的使用占比也随之改变**——⚠ **以往「斩魂·魂核」与「噬暗·魂核」几乎各占50%、「青钢·魂核」很少用，后续注定不同**。
