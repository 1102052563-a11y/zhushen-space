# 灭法四道的定位

> 灭法四道各自的定位。

## 灭法四道的定位

> 原属批次:🗡 极之影 —— 灭法体系的第四条道路


| 道路 | 特点 |
| --- | --- |
| **断魂影** | 能触及「灵」的斩切、变化以及毁灭 |
| **破空影** | 速度最强，是锋利与极致速度的代表 |
| **噬魔影** | 吞噬魔灵，把所有提升魔灵的手段转变为提升自身，且极其难以彻底杀死 |
| ⚠ **极之影** | 前中期强度中上；到超级大后期，凭强大体魄与技法能力，达到**以手中利刃可斩灭因果**的程度 |

⚠ **为何「断魂影·魂灭」被它取代**:苏晓一路「平砍即是大招」，先能斩魂、再以此进阶到**刀斩因果**；而「魂灭」的效果本质是把斩下部分的「灵」进行因果毁灭。**如今他每一刀都自带因果斩灭，当然不需要自行降维度**——先斩「灵」再毁其因果。他的体系已达到「断魂影」所能达到的极限之上。

⚠ 这也是深渊族裔如此忌惮他的根本原因:「始源族徽」是通过因果锚定确保深渊族裔不死，而**那把一切都斩碎的极致斩击，属于绝对的力量破除一切花里胡哨的能力**——被他斩到，它们是真死。

⚠ 后世灭法者的困境:「极之影」虽有无敌般的上限，可到了超级大后期，各类能力的前置**简直让人想哭**（无上晋升前置:力/敏/体/智无上属性 80 点＋深渊之心＋元素之眼＋顶峰之路＋极致调和石……）。
