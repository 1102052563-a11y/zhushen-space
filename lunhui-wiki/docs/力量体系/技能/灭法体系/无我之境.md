# 「无我之境」的招式谱系

> 无我之境衍生的招式谱系。

## 🧬 「无我之境」的招式谱系


苏晓的核心大招是一条持续了 54 卷的进化线:

> **刃道刀·极 → 极刃·极 → 极刃·白夜 → 无我之境·白夜 → 无我之境·无夜**（天锋斩影 Lv.390 时）**→ 无我之境·永夜**（最终决战）

⚠ 另一条则是把「平砍」推到极致的产物:

!!! danger "『无我之境·青影王』"
    ⚠ 长刀乍现漆黑刀焰——**那不是刀焰，而是过于浓烈的青影王能量在质变后把接触到的世界灼穿所呈现的模样**。<br>⚠ **任何存在都不能规避，只能硬抗**:因为斩出这刀的同时，**斩击未至，斩威就已刺入敌人因果**——无论敌人用什么方法移动走，斩击落下的下一刻都会被拖回来（至高之人退到十几秒前的位置，**时间的力量被斩灭崩碎，他被强行扯了回来**）。<br>⚠ **越是简单的能力，发展到极致越无解**——而苏晓从一阶到顶峰，一直在磨砺这一刀力斩。
