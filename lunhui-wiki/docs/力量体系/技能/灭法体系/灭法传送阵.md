# 新版灭法传送阵 / 契约学

> 传送阵与契约学两条路线。

## 🌀 新版灭法传送阵 / 契约学两条路线

> 原属批次:续 · 「传递」与天怒·奔流斩满级（52卷35章 / 41章 / 44章）


- ⚠ **新版灭法传送阵**:⚠ **不再用灵魂晶核驱动，改为以**自身身体能量**构建**（⚠ **纯净灵魂能量是万能能量、与灭法传送阵契合度不高；而苏晓的身体能量与他所开发的阵法是 100% 契合**）——⚠ **成本更低、单次可传送距离更远、人数更多、穿透力与破坏力更强**;⚠ **⚠ 每次消耗 5000～10000 法力值 ＋ 10% 灵魂能量**（⚠ 每天最多十次·绰绰有余）;⚠ **⚠ 优缺参半:优点是三四阶实力乘坐也不会丧命，缺点是「像电钻一样高速转动」，乘坐体验糟糕至极**（⚠ **一种传送术式的底线是「乘坐途中不会死」，实在是让人神情复杂**）。
- ⚠ **分裂契约 VS 多层契约**:⚠ **[艾蜜尔](../../../人物/虚空/艾蜜尔.md) 暴露了「分裂型契约」的破绽——每人可签订的契约数量有上限（与运势体量有关）**;⚠ **⚠ 分裂契约每份束缚力约 5，多层契约每层约 100**——⚠ **苏晓因此决定继续发展「多层契约」，争取突破单份契约 50 层**（⚠ 对 [厄运夫人](../../../人物/女巫界/厄运夫人.md) 用出 48 层·「这次手感还可以」）。
