# 其他灭法体系（速查）

> 篇幅较短、未单独立页的条目，共 2 条。

## 🩸 其他灭法体系变动


| 能力 | 变动 |
| --- | --- |
| **青影王 Lv.EX** | 「绝灭影」从「消耗全部青影王能量」变为**固定消耗 3000 点**（此前 2000）即可发挥全部威力；伤害从已损失身体能量 ×150% → **×180% 真实伤害**。⚠ **苏晓可连用三次**——先代灭法们表示，这涉及到知识盲区 |
| **魔灵觉醒 Lv.EX** | 魂能滋养 68 次，刃之魔灵强度 **38500**；新增 **力量赋予（被动）**——可选定自身一种能力，按 45%~80% 的比例赋予魔灵。⚠ **苏晓灵魂强度 9900**，按「1 灵魂压制 5 魔灵」估算可压制约 49500 点魔灵强度，提升空间巨大 |
| ⚠ **完美循环** | 魔灵持刀在战场杀敌，其击杀**同样计入「深渊任务·守卫者」完成度，并触发「灭法天赋·猎影」的魂能吸收**——魔灵杀敌积攒魂能 → 凝成魂能结晶 → 提升「魔灵觉醒」→ 魔灵更强 → 杀敌更快。⚠ **每名灭法之影必有魔灵，因此判定中魔灵杀敌比分身/化身更高一级，和自身亲手杀敌基本没判定区别**（唯一区别:苏晓亲手杀敌**绝无可能**触发极小概率的幸运事件掉宝箱，魔灵反而可以） |
| **影之赋予 Lv.EX** | 由「魔灵唤醒」与「噬魔体质」融合而成:可消耗此能力，把已掌握的一种能力**转变为「灭法之影体系」能力**（成功率 50%）。⚠ 苏晓用它把 **【完美格挡】** 灭法体系化——从此可用魂能结晶提升 |
| **适应之力 Lv.EX** | 从「外部特性的无形护盾、只能带来一种真实抗性」转化为**自身体魄的一部分**:进入战斗状态提升 **25%** 当前的「真实全抗性」；可消耗魂能结晶提升 |
| **战斗义肢 Lv.EX** | 傲歌状态强度 +2060%、构成与攀附速度 +223%、晶体稳定性达最高值；⚠ **力量/速度 100% 还原、触觉与神经系统 99.99% 还原**；可稳定存在 568000 个自然日 |
| **至尊锋刃／吞噬之核／绝魔体质** | 均以魂能结晶推到极限（无新特性，为原特性大幅增强）。⚠ 至尊锋刃达极限后，象征它的黑色铁砂**化为暗金色** |
| **仍是无底洞的四种** | 时·支配者（5 颗/次·⚠ 超极限提升后可牺牲自身以提升「极境·星界」）／青钢影（3 颗）／魔灵觉醒（3 颗）／完美格挡（3 颗） |

## 👤 凯瑟·休娜（上一代恶劣女灭法 · 格林·吉莉安的恩师）


⚠ **名震虚空的上一代恶劣女灭法·「青影化身」形态的开创者**——⚠ **⚠ 其恶劣之程度已是世间罕见，谁知她教出的弟子 [格林·吉莉安](../../../人物/格林·吉莉安.md) 不仅战力巅峰超越了她，更把灭法阵营的品德下限拉低到一个前所未有的新下限**;⚠ **⚠ 她的出师交代极有远见与智慧**:⚠ **「为师不求你报恩，只不过你在外面惹出事端，不提及为师，为师就死而瞑目了。」**;⚠ **⚠ ⚠ 苏晓的「灭法传送阵」正是通过她的记录得知**（⚠ 但不知为何她自己却不用·⚠ **「那绝不是混蛋灭法者」**）。
