# 其他传承秘宝（速查）

> 篇幅较短、未单独立页的传承秘宝，共 3 条。

## 🗝 巨人之国·10 号宝库钥匙


⚠ 巨人之国苏晓没听过，「想来是**和地精商会类似的势力，只不过一个是做生意，一个是提供存储服务**」。⚠ **作为奥术永恒星·绝强·施法者的古亚院长，都选择在那边存东西，可见那边安全的程度**——⚠ **当然，这并不代表巨人之国比奥术永恒星安全，没那种可能**。⚠ **问题是出在古亚院长所存的东西上**:⚠ **古亚院长作为奥术永恒星的掌权者之一，是什么东西，他不能存在奥术永恒星的宝库？** ⚠ **那被存在巨人之国的东西，实在太耐人寻味了，值得专程去那边一趟。**

## 敌方契约者装备（见闻 · 二卷25–28章）

- **绿色巨盾**（评分 26）：[女王](../../../人物/女王.md) 所用门板大小金属盾，防御极高，苏晓全力一刀方能勉强破盾、修复费高昂。
- **绿色狙击辅助装备**：[黑白](../../../人物/黑白.md) 的头甲 + 右眼探测装置，集**夜视 / 热能感知 / 超声波探测**于一体——即「[查探敌人资料](../../../总览/轮回乐园机制.md)」三途径中的**探测类装备**之一例。

## 不朽水晶女神像 / 月之女神像 / 地之城钥匙（月神大陆道具 · 二十八卷18–23章）


- **不朽水晶女神像（史诗级++）**：大陆最大不朽水晶、历八代工匠 175 年雕成、几乎是月之女神化身（被亿万信徒供奉数百年→意念之力使其成超凡物·出售给轮回乐园可换 **37% 世界之源**）。布布汪「看着大就顺手偷了」、阴差阳错偷崩月神女心态、成苏晓自导自演栽赃鹰侯破案的道具。
- **月之女神像（月神女祈祷室至宝）**：月神女祈祷室内的小型月之女神像——布布汪偷的第一件「心爱之物」、用来逐步偷崩月神女心态。
- **地之城钥匙（觉醒第三环信物）**：密德尔顿家族世代守护的「地之城」钥匙、由守门人 [富兰克](../../../人物/月神大陆配角.md) 持有。斩富兰克所得（觉醒第三环三信物之一·另有灵魂语者的月之发、地龙·塔门的地龙之血）。

### 精英死灵召唤凭证（传说级 · 稀有道具 · 希亚世界所得·二十一卷35章用于西港）

苏晓在 [希亚大陆](../../../世界/任务世界/希亚大陆-灭法遗迹.md) 得到的稀有道具——使用后**召唤 3 名精英死灵族**（2 秒成形·存在 12 小时·绝对忠诚 + 独立意识 + 可执行复杂指令）。类型可选盾战士 / 狩猎者（远程弓箭）/ 生命祭祀（治疗）/ 隐之牙（刺客）·特长属性 80~82、特长技能 Lv.30~40。苏晓在西港攻坚战召出 1 狩猎者 + 2 生命祭祀组「攻坚小队」——经他 96 真实智力加成、三死灵特长属性都破 90、技能 Lv.45~50（狩猎者尤强·死亡之女 / 倒刺箭 / 死灵之力，是个小 BOSS）。

### 团队阶位晋升卷轴（史诗级 · 道具 · 二十四卷67章·开史诗宝箱 37% 所得）

使用后可**提升一个冒险团的阶位**（最高至 SSS 级）——对大型冒险团是无价之宝（团队技能 = 全团提升·比几件史诗装备更重要）。苏晓的破晓冒险团才 S 级、用着浪费，准备卖给 [神皇冒险团](../../../人物/神皇.md)（「飞升套黑到失眠、这卷轴能黑到休克」）。
