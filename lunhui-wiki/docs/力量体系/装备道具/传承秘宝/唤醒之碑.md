## 🆕 唤醒之碑 · 灭法技能系统 · 誓约物（第49卷续18 · 暗影世界 · 通关）

> 原属批次:寄生兽空间战场（战利品 / 卡拉星科技）


- **唤醒之碑（灭法传承核心 · 苏晓的终极目标）**：⚠ 一块几米高、暗蓝晶质、刻满灭法系纹路的石碑，是 [背叛者·席曼·阿奇德](../../../人物/暗影世界/背叛者·席曼·阿奇德.md) 花 1592 盎司从圣光乐园买来、临终送给苏晓的礼物。⚠ 它原本「不完整」——缺一枚**本源能量核心【？？？】**（恶魔铁匠在白色小镇代管、正在苏晓手里）;镶入后石碑化半透明浅蓝，不再需要灵魂晶核充能、只耗**灭法技能点**即可掌握上面的灭法系能力。经公证，苏晓得**九个灭法技能槽**（体力 300 可得第十）。它彻底把苏晓灭法系发展的「断掉的独木桥」变成「康庄大道」。
- **灭法技能系统（灭法被动 / 分支被动 / 灭法技能槽）**：⚠ 灭法能力分两类——**灭法被动**（灵影体质 / 绝魔体质 / 不灭体质这类·每种耗 10 灭法技能点·初始即 Lv.MAX·须有空技能槽[体魄每上一梯阶得一槽]否则掌握必死）、**分支被动**（每种 1 点·初始 Lv.1 上限 10·用来提升青钢影 / 刀术 / 青影王等已有能力，或直接 +1% 法力 / 生命 / +2 肉体强度等「提升自身」）。⚠ 分支被动有几十上百种，堆够多后**平砍即大招**（背叛者能以老年衰弱之躯把苏晓打败的秘密、也是灭法容错率高到离谱的根源:空一刀大招级平砍、再来一刀就好，平砍哪有冷却）。苏晓现可掌握的灭法被动:**适应之力**（真实抗性·敌人无法削减）、**完美体质**（四属性每 +5 额外 +1）、**技法共鸣·急速**（技法等级总和 ×0.012 倍身体速度·苏晓三宗师 210 级 = ×2.52）、**不灭体质**（生命 25% 时防御 + 四属性和 ×0.5）、**灵魂感电**（青钢影 15% 绝对麻痹）、**时·支配者**、**灭法·天雷怒**（元素亲和 ÷3 = 界雷抗）、**青影王·唤醒**（需四属性 300+ & 青影王 Lv.40）、**魔灵觉醒**（外放刃之魔灵）等，另有免费的**限定解除**（解除主属性差 3 点限制）+ **能力铭刻**。
- **刃之魔灵·外放（魔灵觉醒）**：⚠ 苏晓在背叛者身上见识了这能力的真正用法:灭法之刃放出的**黑雾人形战士**、无视一切伤害（仅攻击进实体瞬间可反击）、能与本体互换位置、附身生物、化晶体武器（青钢影·傲歌转化·可弓 / 锤 / 刀切换·还受苏晓 680 灵魂强度加成）、破格挡。苏晓已构思:用荣誉商店的「灵魂之刃」给魔灵配武器、让魔灵拟态成箭矢 / 敛匿气息潜入敌营斩杀。
- **本源能量核心【？？？】**：恶魔铁匠代管的圆形深渊色光团（碰一下电得巴哈去世），正是唤醒之碑的能量心脏、严丝合缝镶进碑顶凹槽——原来它从没遗失、一直在灭法阵营手中。
- **誓约物·生命扭转 / 誓约物系统**：⚠ 深渊商店的随机誓约物，佩戴入专属栏（誓约物仅可装一件）。它「弱点弥补」增益对苏晓的负魅力无效，可其**誓约代价**反成增益:苏晓选「濒死之咒」（濒死血线 10%→**20%**），配黑王护臂（豁免濒死减益）+ 血枪宗师「死而后生」，让他更早进入最巅峰的**第三阶段**（斩深渊大主教、险胜背叛者皆靠此）。
- **铭文基座·神祭（五插槽）**：深渊商店 128 沉淀琉璃换得，可插 5 块铭文片的铭文基座——配无上烈阳主铭文，苏晓的太阳铭文体系初成。
- **黑暗蠕动（深渊·仪式物）/ 高烈度·深渊之血**：斩深渊大主教得**黑暗蠕动**（大幅提升召唤 / 祭献成功率 + 伪装祭献物价值·苏晓视为「钓邪神神器」）；斩背叛者得**高烈度·深渊之血 ×6**（按法提升 8~15 点深渊抗性）。
- **渊陨（归宿：黑A）**：⚠ 那把「暗之邪灵」重剑最终被黑A 一把拔走——黑A 的「逆子」特性 + 吞过深渊能量，或能白嫖八重封印解除而不受邪灵侵袭。

## 🧬 唤醒之碑 · 进阶灭法系能力全览

> 原属批次:📚 续26 · 返园爆改 · 终极·基础被动与三宗师破 Lv.80（50卷65–69章）


⚠ **【复仇之魂】与【强者之魂】的分工**:⚠ **当初不知道是哪位先代灭法向虚空之树发起公证，将唤醒之碑上的所有「进阶·灭法系能力」都封印起来**——⚠ **需用一枚枚【复仇之魂】解锁**（⚠ **后续在苏晓获得唤醒之碑后，这公证转接到轮回乐园这边**）；⚠ **获得【复仇之魂】的方法很有趣:就是斩杀足够强的施法者**（例如之前斩杀古亚院长）。⚠ **其实只需两颗【复仇之魂】就能解锁所有进阶能力**——⚠ **真正的考验，是获得足够的【强者之魂】。**

- ⚠ **进阶·灭法系能力是什么**:⚠ **不是基础灭法系能力的升级版，而是有先代灭法们私货的个人特性能力**——⚠ **就比如噬魔影的专属能力「界斩」:这能力苏晓能掌握，甚至能使用出来，但他没有噬魔影那么高的暗雷抗性**，⚠ **属于这招只能用一次；用了之后，敌人是否会死不清楚，但他肯定会**。⚠ **这些能力要么就掌握不了，再或是契合度低——可一旦有高度契合的，那就是神技。**
- ⚠ **【强者之魂】与「通用被动」**:⚠ **每次以【强者之魂】提升唤醒之碑，都能让其上解锁更多「通用被动」**。⚠ **「通用被动」的效果简单粗暴:只需一点灭法技能点就能掌握**，⚠ **后续两种升级方式（灭法技能点 / 技能升级仓花灵魂钱币）**；⚠ **满级就是 Lv.10 或 Lv.20**——⚠ **奇妙的是，将其升满后这类能力会立即从技能列表内消失，但带来的永久性提升不会消失**。⚠ **技能名称相当之随意，但数量多**:⚠ **虽说单独一种效果单一，可数量多了后提升的范畴就格外广泛**（提升身体属性、生命值 / 法力值、雷属性抗性、青钢影伤害、体质阶位、身体速度、感知强度、⚠ **乃至于提升深渊抗性、以及伤害阶位**——⚠ **当然，提升深渊抗性与伤害阶位的「通用被动」升级起来非常贵，而且提升幅度不算大，可顶不住这类「通用被动」多**）。
- ⚠ **【强者之魂】的来源与掉率**（苏晓以猎杀者权限咨询到的）:⚠ **只要是击杀深渊系敌人与古神系敌人，他就有概率获得**——⚠ **敌人越强，概率就越高**；⚠ **这公证从他成为灭法之影后就已经存在**（⚠ **也就是说，哪怕苏晓在一阶时碰巧杀死一名重度虚弱＋垂死边缘＋本源严重受损＋意识沉寂的古神系，他就有概率获得**）；⚠ **仅有苏晓自己杀敌时有概率获得，想收购一类都没可能**。
    - ⚠ **六阶:0.0185%** · **七阶:0.12%** · **八阶:2.93%** · **九阶:10.5%** · ⚠ **绝强:30%** · ⚠ **至强:95%**
    - ⚠ 「看到这掉落率，苏晓理解了为何迄今为止他还没获得过哪怕一颗【强者之魂】——⚠ **掉落率太低了，不是他的问题，嗯，没错。**」
- ⚠ **现有:灭法技能点 82 点 · 闲置灭法技能槽 4 个。**

#### 断魂影·魂核体系（六种进阶能力）

| 能力 | 效果 | 苏晓的处置 |
|---|---|---|
| ⚠ **滋养·魂核**（进阶·被动） | ⚠ **掌握后此魂核不可切换出，且会占据三个魂核槽**；⚠ **会永久性强化现已掌握的所有魂核（提升 1.5～3 倍，根据魂核特性不同）** · 需 **20 点灭法技能点** | ⚠ **必学**——⚠ **但剩余魂核槽不足，无法掌握**（现 **2/6**，已掌握:**青钢、急速、斩魂、刃灵**） |
| ⚠ **魂核拓展**（进阶·被动） | ⚠ **永久性提升 3～4 个魂核槽** · ⚠ **警告:你需保证自身的体力属性在 400 点以上才可尝试掌握，且掌握过程异常凶险** | ⚠ **必学**——⚠ **最起码等突破 300 点属性壁障后** |
| ⚠ **魂核强化**（进阶·被动） | 选定一颗现有魂核进行强化，⚠ **完成后此魂核将占据三个魂核槽**；⚠ **可建立在「滋养·魂核」的提升上再度强化** | ⚠ **必学** · ⚠ **这是 [格林·吉莉安](../../../人物/格林·吉莉安.md) 所开发** |
| **黑暗·魂核**（进阶·被动） | ⚠ **临时融合所有魔能能量，大幅增强生命力与身体防御力梯阶，以及深渊抗性自适应提升**——⚠ **但移动速度总体降低 93%～95%**（⚠ **此削减拥有绝对优先权，无法以任何方式抵消**）· ⚠ **切换速度较慢，且需在稳定环境下才能切换** · 占 1 槽 | ⚠ **准备掌握**——⚠ **这不是用于战斗的魂核，而是功能性魂核** |
| **落雷·魂核**（进阶·被动） | 大幅度提升雷电抗性 · 占 1 槽 | ⚠ **准备掌握** |
| **魂核剥离**（进阶·被动） | ⚠ **可保证自身无损状态剥离一颗魂核，完成后此能力将被消耗** · ⚠ **最多可掌握六次** · 每次 **10 点灭法技能点** | 备用 |

- ⚠ **格林·吉莉安为何值得单独一提**:⚠ **其他先代断魂影开发能力，都是秉承「在掌握途中大概率不会暴毙，那就到了可接受范畴」**——⚠ **格林·吉莉安开发的能力则是:只要有 30% 的概率不会暴毙，那就试试，紧张又刺激**；⚠ **三成的成功率，在这家伙看来已经不低了。**
- ⚠ **黑暗·魂核的用法**:⚠ **在深入一些深渊重度侵蚀区时切换到这种魂核是还算不错的选择**——⚠ **虽说应对突发情况的能力锐减，但进入那种深渊侵蚀区，生命值会持续滑落**；⚠ **有些区域还会出现治疗完全遏制**（**无法以能力或药剂恢复生命值**）。⚠ **想来当初开发这魂核的先代灭法就是这种目的，因此哪怕这魂核有无法消除的强减速，也将其铭刻在唤醒之碑上。**
- ⚠ **断魂影的现阶段用法与终局**:⚠ **现阶段就是多掌握些魂核，用于应对各类情况**；⚠ **当到了至强者阶段，就要开始以「魂核剥离」剥离现有的多颗魂核，最终剩下 3～4 颗，然后把这几颗强化到极致**。⚠ **真正强大的魂核可不是占据一个魂核槽**——⚠ **迄今为止最强的一颗「斩魂·魂核」，足足占据十个魂核槽**；⚠ **魂核越强，就需要更多的魂核槽容纳。**

#### 本批掌握的两种灭法系能力

- ⚠ **灵魂感电（Lv.EX·被动）**：⚠ **大幅度加强青钢影能量的控制特性**——⚠ **青钢影能量燃烧敌人的身体能量时，有 15% 几率触发麻痹状态**（⚠ **此为绝对控制效果，无法以任何方式豁免**），异常状态持续 **0.2～3 秒**。⚠ **此效果可持续叠加**:⚠ **每次燃烧敌人的身体能量，如未触发「绝对控制」，将提高 20% 此特性的触发概率**——⚠ **直至此概率达到 100%，下次附带青钢影能量的斩击在命中敌人后将必定导致敌人陷入「绝对麻痹」状态。**（⚠ 掌握时:**蓝色电弧在唤醒之碑上涌现、奔涌到苏晓身上，这一瞬间他全身的汗毛都竖立**；⚠ **一小时后，全身升腾着灼热水汽的苏晓长舒了口气**——⚠ **事实证明，掌握多个灭法系技能后他这方面的抗性飙升，方才的过程虽难忘，但并未导致今天戛然而止**）
- ⚠ **适应之力（Lv.EX·被动·至多可唤醒一次）**：⚠ **根据敌人特性，提升 25～30 点不可削减的真实抗性！** ⚠ **「真实抗性」拥有绝对优先权，敌人无法通过任何方式削减此类抗性。**（⚠ 掌握时**美好的一天戛然而止**，⚠ **醒来已是次日清早八点**——⚠ **也幸好晋升九阶后在轮回乐园内的停留时间多了不少，否则可能直接被传送回现世**；⚠ **那感觉和掌握「完美体质」时有的一拼**）
    - ⚠ **它的价值**:「乍一看这 25～30 点真实抗性不多，⚠ **可这是最后的壁障，也是最可靠的防御**——⚠ **无论敌人采取怎样的手段，都别想让苏晓进入 0 抗性、乃至于负抗性的程度**。」⚠ **与古神系、深渊系强敌交战，抗性被削减到 0 点、乃至 −20～30 点，简直是家常便饭**；⚠ **和神父交战那就更离谱:抗性能被削减到 −150～−180 点的程度。**
    - ⚠ **神父的警报**:⚠ **这次在风海大陆神父看似全盘皆输，可这老家伙其实留了一手**——⚠ **他在部落阵营那边的布设基本都成功了，典型的闷声发大财**；⚠ **如此一来这老家伙的实力也必定迈进一大截，其削减抗性的能力应该就更离谱**；⚠ **外加这家伙和「黄金圣杯」临时共存了，这其实更离谱**:⚠ **「黄金圣杯」内能涌出巨量的神血，而神父就有饱饮神血壮大自身的能力**——⚠ **可以想象，下次见面时这家伙的实力会增长到何种程度**。⚠ **只不过，苏晓现在有了「适应之力」后完全无惧神父的抗性削减了**——⚠ **被削减到只剩 25～30 点抗性，就是他能被削减的极限**。⚠ **当然也不能因此就小觑了神父:这老家伙的手段可不止削减抗性一种**——⚠ **而且相比各类诡异能力，神父这个人本身其实更危险**。⚠ 「苏晓见过敢于反抗古神的强大者，虽说不多，但他所见闻的也有十几位之多了——⚠ **但敢窥探古神力量、并且成功夺来的，他只见过一个:就是那个头发略显花白、但梳理整齐、身穿神职人员衣着的老家伙。**」
