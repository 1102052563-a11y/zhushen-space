# 扭曲缠绕之核

> 深渊宝箱产物，与原罪之书之间的挑拨关系。

## 🌀 扭曲缠绕之核（深渊宝箱 ★）与原罪之书的挑拨

> 原属批次:续37 · 女巫界收官：狂猎之夜永恒级、黑刀、吞星佩刀与灵魂宝石（52卷51–61章）


⚠ **一颗拳头大小、由黑白两色纠缠在一起的晶核**——⚠ **⚠ 超高浓度原罪特性晶化物·封印中·唯一**;⚠ **⚠ 它的出现让苏晓确定一点:**想让一件大爹级原罪物归于永寂，唯一可行的方法是通过另一件大爹级原罪物****（⚠ 这应是虚空第一纪元最初出现的两件光/暗特性大爹级原罪物同归永寂的产物）。

- ⚠ **⚠ 缺德到让人忍俊不禁的最优解**:⚠ **他把它按向「原罪之书」第五页的**灵魂王冠**（⚠ 危险度 99.5·其他四位都是 100·⚠ 唯独它被排除在四件原罪物的彼此制衡之外，让原罪之书的「平衡感」很差）**;⚠ **⚠ 灵魂王冠吞噬后必成新晋大爹级 → ① 苏晓的封印门槛从此达到大爹级 ② **收藏家、死灵之书、猩红权杖、幽冥骨戒这四位大爹能高兴吗**——⚠ **搞不到苏晓，那就一起怼灵魂王冠好了，这属于伸手就能揍到的** ③ 灵魂王冠在吸收关键期只能先忍着挨捶**;⚠ **⚠ 结果:五件原罪物各忙各的，封印压力**从假设的 200 万暂降到 5～10 万****——⚠ **而灵魂王冠挨揍时溢出的两三成「本源原罪之力」，刚好被「原罪之书」吸收，**史诗级增强****。
- ⚠ **⚠ 【古老者的旅行笔记】**（深渊宝箱·可售灵魂书库 2000 书库银币）:⚠ **灵魂书库拥有者·古老者把毕生所听闻的大爹级原罪物全部记录在此**——⚠ **前几页是死灵之书、猩红权杖、收藏家、幽冥骨戒**;⚠ **⚠ 最后两页忽然密集起来:**太阳面具 · 深海古瓮 · 旧日眼眸 · 命运金币****——⚠ **⚠ 苏晓现持四件，若这笔记记全了，外界还有四件。**
