# 猩红权杖

> 大爹级原罪物。

## 🔴 原罪物·猩红权杖（大爹级）

> 原属批次:续29 · 斩始祖与君主宝藏（51卷26–35章）


⚠ **猩红圣殿密室展览架最上层，一根古老又形状扭曲的权杖**——⚠ **哪怕外表封了层世界树树脂，苏晓一眼就看出这是大爹级原罪物**（⚠ **[猩红君主](../../../人物/永光世界/猩红君主.md) 的象征**）。

- ⚠ **它是神父与猩红阵营联手布的阳谋核心**（详见世界页/[神父](../../../人物/神父.md)）:⚠ **包着世界树树脂的死寂烬灭被献祭给猩红权杖、正被缓慢侵蚀**——⚠ **想拿死寂烬灭就得先成为权杖持有者、中断这过程**;⚠ **⚠ 结果苏晓靠【猩红指环（伪）】走侧门提前赶到，陷阱还没激活。**
- ⚠ **苏晓把它封进「原罪之书」第五页**（⚠ 四件原罪物一反常态帮他挡住袭来的猩红触须——⚠ **因为一旦苏晓葬身于此，这四件原罪物也很难离开永光**）——⚠ **⚠ 但他并未「持有」这原罪物**:⚠ **因为它的持有者（[猩红君主](../../../人物/永光世界/猩红君主.md)）还未死、也没切断与它的因果**;⚠ **他只是暂时将其封印。**
