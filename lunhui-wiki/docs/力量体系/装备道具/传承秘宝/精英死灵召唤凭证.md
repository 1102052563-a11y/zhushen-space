# 精英死灵召唤凭证（传说级 · 召唤道具 · 十七卷44章·开死灵箱兑得）

死灵族独有的传说级道具（评分 500）——使用后**召唤 3 名「精英死灵族」**（2 秒构建身体·存在 12 小时）；可指定类型：**盾战士（近战）/ 狩猎者（远程·弓）/ 生命祭祀（治疗·奶）/ 隐之牙（刺客）**。精英死灵**绝对忠诚、有独立意识、能执行复杂指令**，特长属性 80~82、特长技能 Lv.30~40。苏晓打算召一名「生命祭祀」当奶妈、凑成「他 + 布布汪 + 阿姆 + 3 精英死灵」的合理小队闯王座之下（极珍贵·不到关键时刻不用）。**出处机制·死灵能量**：独立击杀死灵大 BOSS [敦克·尼克](../../../人物/敦克·尼克.md)（需冒险团围攻级别）后，死灵箱按贡献给 **100 点死灵能量**兑奖励列表（含死灵觉醒药剂 / 匕首投掷大师卷轴 / 黑暗者披风〔金套〕等·部分奖励为冒险团独有）。
