# 🏅 试炼之凭证（永恒级 · 灵魂试炼凭证）

⚠ **斩杀灵魂大师所得**——⚠ **灵魂国度封着三位强者之魂的容器**:

| 试炼之凭证（永恒级·项链·灵魂强度600+） | 效果 |
| --- | --- |
| 王魂之力（被动·三效果） | ⚠ **灵魂强度>700 激活血之镰（鲜红王后）·>800 激活将军战锤（巨人将军）·>1000 激活魂王大剑（魂王）** |
| 灵魂试炼（主动） | ⚠ **三重考验·每通过一重消耗一位强者灵魂**（⚠ 门槛依次 700/800/1000 灵魂强度） |
| 破碎奖励 | ⚠ **三魂全消耗后此装备破碎，并一次性永久 +100 灵魂强度** |

- ⚠ **灵魂国度**:⚠ **已灭亡的灵魂传承文明·灭法阵营的严肃盟友**——⚠ **三派系:传统灵魂秘术（魂王·金色·后期最强）/脑洞派（鲜红王后·风险大·灭国主角）/灵魂滋养者（巨人将军·把大脑变大）**;⚠ **⚠ 苏晓通过第一重（鲜红王后）得【灵魂秘术】传承知识、第二重（巨人将军·穿魂雷）提升灵魂宝石；第三重（魂王·需灵魂强度 1000）暂无法参与。**
