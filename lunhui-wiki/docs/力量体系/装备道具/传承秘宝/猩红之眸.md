# 猩红之眸

> 世界级钥匙类道具。

## 🔴 猩红之眸（钥匙类·世界级）

> 原属批次:续28 · 极暗古堡与生命晶石（51卷16–25章）


⚠ **[机械制造者](../../../人物/永光世界/机械制造者.md) 转交**:「**许久之前，有灭法者在我这寄存了些东西，你作为最后一名灭法，有资格在我这取走这东西。**」

| 项目 | 内容 |
| --- | --- |
| 产地 | ⚠ **永光世界·灭法阵营** · 品质 ⚠ **世界级** |
| 耐久度 | ⚠ **1180/1500 点**——⚠ **随着时间推移逐步下降**;⚠ **⚠ 当耐久度清零，炼狱噩梦的封印将失效** |
| 效果 | ⚠ **可开启炼狱噩梦的封印，进入此地的内部——猩红噩梦中** |
| 简介 | ⚠ 「**猩红的根源就在此处**，灭法者们将其封印在沙之海最里侧的噩梦之中。」 |

- ⚠ **奇妙的一幕**:⚠ **苏晓拿起木盒内的猩红眼眸，原本缓慢降低的耐久度立即停止滑落**——⚠ **看来只要是灭法之影带着此物，就能避免远在沙之海深处的猩红继续侵蚀将其封印的炼狱噩梦。**
- ⚠ 「**灭法者，你带着它就好，不会有危险，至少我保管此物这么多年来，没有危险**——⚠ **而且你的灭法运势，会让这颗眼眸停止被猩红侵蚀，它应该由你来保管，也只有你，有这资格。**」
