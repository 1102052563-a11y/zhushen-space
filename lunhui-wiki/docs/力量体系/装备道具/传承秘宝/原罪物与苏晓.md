# 原罪物为何总找上苏晓

> 苏晓关于「原罪物为何总找上自己」的顿悟。

## 🧠 原罪物为何总找上苏晓（苏晓的顿悟）

> 原属批次:续29 · 斩始祖与君主宝藏（51卷26–35章）


⚠ **他一直以为是自己倒霉，此刻确定不是**——⚠ **原罪物间不会互相吸引**（⚠ 他是四件原罪物持有者，很有话语权——⚠ **大爹级还会互相钳制，否则原罪之书不会这么稳定**）;⚠ **排除后只剩一种可能:灭法运势——⚠ **一整个阵营的运势集中到一人身上。**

- ⚠ **原理**:⚠ **运势庞大者（无论幸运还是倒霉），更容易遇到原罪物**——⚠ **因为只有更大的运势与因果，才能承载原罪物、成为其持有者**;⚠ **比喻:原罪物需 1000 点运势上限才能持有，寻常生灵连 0.1 点都不到**——⚠ **而运势上限几千点乃至更多者，一旦进入原罪物所在世界，双方在因果层面彼此吸引**（⚠ 见面前吸引、见面后彼此嫌弃、因果纠缠到弄不死对方）。
- ⚠ **⚠ 好消息**:⚠ **每持有一件原罪物都会占据一定运势上限**——⚠ **当运势上限被占满，自然就没可能再遇到新的原罪物**;⚠ **苏晓估测，猩红权杖或许是他遇到的最后一件原罪物**（⚠ 想到这点，他的心情明显好了几分）。
