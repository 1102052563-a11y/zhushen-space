# 收藏家

> 深渊原罪物，苏晓手里的第四件大爹级。

## 🗝 收藏家（深渊·原罪物 · 第四件大爹）

> 原属批次:📚 续25 · 风海大陆收官 · 原罪之书与荣誉商店（50卷55–64章）


⚠ 苏晓**第三次**从【深渊宝箱】开出原罪物（**纯黑色提示**）——⚠ 他这次特意**先开深渊宝箱**（以往都最后开，导致满评分·世界级装备、永恒级装备、有 30 块灵魂精魄的灵魂箱等**全部没开出来过，反而是开出了两件原罪物**）；⚠ 他还自我安慰「**基本不用担心再开出大爹级的原罪物了**」，顿时感觉**手中的深渊宝箱都顺眼了不少**——⚠ **然后就开出了收藏家。**

- **外观**：不同于死灵之书的**灰色诡秘**、幽冥骨戒的**幽绿深沉**、灵魂王冠的**鲜红＋黑暗蛊惑**，⚠ 收藏家呈现出的是**暗金色**——**一种没有固定形体，犹如铁砂与晶碎物质的暗金色**。
- **因果特性**：⚠ **获取后，此原罪物将立即与持有者融合，达成『完美的收藏家』**。⚠ **警告:与收藏家融合后，行为、意识、思维主导权均为收藏家，非此原罪物持有者，原罪物持有者将处于沉眠中。**
- **负面风险**：⚠ 具备两面性——⚠ **「满足」与「渴望」**。收藏癖得以满足则**不会对周边造成威胁**；⚠ 可一旦进入「渴望」状态，将**持续吞噬所在世界的世界之力**，⚠ **预计 20～50 个自然日，所在世界将重度枯竭、处于崩灭的边缘，导致其 99% 以上物种灭绝**。
- **原罪特性**：⚠ **收藏家可无代价持有与使用其他原罪物。**
- ⚠ **它是苏晓遇到过、意识表达最明确的原罪物**:他甚至能隐隐感觉到**对方那毫不隐藏的耐心与恶意**。⚠ 苏晓抛出的**晶质物**（原罪物酷爱吞噬之物）**没能吸引到它**；幸亏轮回乐园的**血色壁障**挡住了它的同化。⚠ **它很有耐心，而且它似乎知道不能与轮回乐园硬怼，并且很笃定苏晓早晚会离开轮回乐园**——⚠ **只要苏晓进入任务世界，第一时间就会遭到收藏家的同化**。
- ⚠ **原罪物的交流欲对照**:会表现出意识的原罪物不多，**愿意交流的就更少**——⚠ 除收藏家外**只有死灵之书会和苏晓交流**（还是因为一同收拾邪神才有的**特别简单的交流**）；⚠ **幽冥骨戒这大爹与苏晓 0 交流**（不知是不愿理他还是意识非常淡、更接近于器物，⚠ **虽说没交流，但互相嫌弃还是有的**）；⚠ **灵魂王冠的互相嫌弃就更强了**（属于**一旦有机会这原罪物就想弄死苏晓**，⚠ **苏晓对其的封印也是经常给开小灶**）。
