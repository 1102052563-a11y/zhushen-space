# 宗师之力（残卷 / 完整）+ 荣耀之证明（晋升大师级的两道门槛 · 十九卷87–97章）

**晋升基础技能「大师级 → 宗师级」的核心材料**（[刀术大师 → 宗师](../../技能/index.md) 的关键）：

- **宗师之力（残卷）**：稀有材料·**10 份任意残卷可合成「宗师之力（完整）」**。苏晓海蓝星功勋兑换「宗师之力 3/10」、加上 [战争的馈赠](../../../index.md) 转盘抽的 2/10、拍卖会用 [斧之大师 Lv.10 卷轴](../../../index.md) 换得 3 卷、零散凑齐共 10 份合成完整。残卷极珍贵（除非培养重要新团员、很少有人舍得用残卷换大师级卷轴）。
- **荣耀之证明**：在**试炼场「挑战模式」**（每次 5000 币·此权限随刀术大师 Lv.MAX 临时开启）打赢系统匹配的指定镜像所得（**95% 以上挑战者失败**）。苏晓连战镜像 **西尔西亚**（宗师级近战）八场、第八场同归于尽式险胜得之。
- 凭这两样，猎杀者可**豁免晋升试炼考核**、直接把刀术大师 Lv.MAX 晋升为 [刀术宗师 Lv.1](../../技能/index.md)。
