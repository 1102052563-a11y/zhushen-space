# 星界五件命运奇物

> 星界仅五件的命运奇物名录。

## 🔮 星界五件「命运奇物」


本批已知的两件:

- **【命运逆转】**（苏晓持有）
- ⚠ **【机灵的叽里咕噜】**——风海首胜的战胜收益之一，形态是一个**秒表**。提示:**正在偷偷观察你，预计 10~12 小时后会向你屈服；请勿强行使用**——因苏晓庞大的因果体量与此物特性，强行使用将导致其被摧毁。
