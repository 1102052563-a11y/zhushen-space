# 🆕 灵魂印记 · 神灵时代典籍（第48卷续3）

- **魂之书·灵魂印记**（神灵时代·灵魂书库顶层·原始秘术卷轴）：以灵魂能量构成一枚「灵魂印记」、巨量增幅冥想效率的秘术（印记会随灵魂强度 / 灵魂能量阶位提升而增强）。⚠ 学习门槛=**灵魂强度 600 点以上 + 灵魂能量阶位 6**，低于此贸然「冥刻」灵魂烙印会灵魂枯竭而死（野兽族「谁修行谁死」正因此，[野兽大师](../../../世界/任务世界/幽暗大陆-死寂根源之地.md) 才改良出安全的删减版）。苏晓 650 灵魂强度 + 阶位 7，安全修成——构建灵魂印记后**心之冥想升 Lv.80**、冥想效率暴涨到「一天顶别人半年」（配自然共鸣称号、起源石加成更夸张）。
- **黑王护臂 / 死寂降临**（因果物·终极能力详解）：套装终极能力「死寂降临」——开启后周边 600 米被死寂城高速同化、每秒造成最大生命 5%~23% 侵蚀伤害（移动的敌人侵蚀提升 2~6 倍）。⚠ 其本质是把死寂城的部分环境「拖」过来:故没有【庇护石】进根源·死寂城，就等同长期身处「死寂降临」领域——正是苏晓要带庇护石、了断死寂因果的原因。
- **家族徽章 / 虫之书·厄体转生 / 秘虫化石**（斩老怪物所得）：**家族徽章**代表瓦迪家族家主，在野兽族 / 狂兽族 / 流民部落有极高威望、可与恶土智慧生物交涉贸易；**虫之书·厄体转生**（神灵时代·记秽虫/永生/虫王知识）可凭它与「远古虫王」交易一次（备好其喜食饵料·回赠随缘、心情不好一口吞了你）；**秘虫化石**（不朽级材料·某被选者以圣虫剑从「罪孽集合体」斩下的心核·此人败后对永生生扭曲执念——正是老怪物）。
- **神圣分割器 · 深度世界破解**（补记）：苏晓破老怪物石椅机关的方式已「返璞归真」——直接调急冻溶液冻住石椅 + 地基、一脚踹碎（八阶快九阶不再解谜）。祭祀厅深度世界里的珍宝多不可带出，苏晓走【誓约之徽·白龙】祭献（发现祭献漫画书有时比珍宝收益还高）。
