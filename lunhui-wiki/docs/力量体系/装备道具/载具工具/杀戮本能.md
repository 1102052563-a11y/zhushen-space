# 杀戮本能（史诗级 · 腰带 · 二十五卷21章·白胡子宝库任选所得）

[白胡子](../../../人物/顶上战场配角.md) 兑现承诺、让苏晓在莫比迪克号宝库任选十件物品中**最中意的一件**——鲨鱼皮腰带（评分满 700·需真敏 95 / 意志 30·简介「杀光所有！」）。佩戴时强制一次意志力判定：成功得「**本能激活（被动）**」（大幅加速感知类能力成长 + 该类提升费用 −50%）、失败则进入五小时「本能杀戮状态」（视野血红 / 嗜杀低语·结束后永久无法再戴此装备）。苏晓在远海冒险佩戴、靠「海水触感打断本能 + 不在判定中冥想」**判定成功**——直感（直感无法在乐园内提升、只能靠他自己慢练）从此**成长速度 ×2~25 倍**（战斗 25 倍 / 危险 12 倍 / 安全 2 倍·当场把直感顶到 Lv.42）。同批选的其余九件：[月之风华](../../../index.md)（史诗宝石）/ [依恋的海螺](../../../index.md)（传说道具）/ [米奈希尔深海矿石](../../消耗品材料/index.md)×2 + 3 超人系 + 2 动物系恶魔果实。
