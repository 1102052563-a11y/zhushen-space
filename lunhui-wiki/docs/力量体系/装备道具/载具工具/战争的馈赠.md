# 战争的馈赠（传说级宝箱 · 十九卷40章·摧毁圣利门要塞奖励）

苏晓**摧毁圣利门要塞 86%、起到至关重要作用**所得的传说级宝箱类物品（一个正面带着类似轮回烙印印记的长方形木盒）——产出随机：**普通奖励** = 装备 / 宝石 / 真实属性点 / 灵魂结晶；**稀有奖励** = 觉醒之书（二次）/ 稀有血统 / 职业 / **[黑渊](../../../世界/地点/index.md) 邀请函**等（这些根本无法通过常规途径获得）。同批奖励还有 800 乐园功勋（远超他上场世界争夺战总共才得 700 多）+ 权限升至 Lv.38。**开启机制（十九卷44章·转盘三区）**：与普通宝箱靠运气不同，**战争的馈赠会按猎杀者需求过滤奖励**——用不上的选项直接变灰（如「稀有血统 / 职业」对苏晓灰显）。三区：普通奖励（装备 / 宝石 / 真实属性点等）、**稀有奖励 6 种**（觉醒之书〔二次〕/ 稀有血统 / 职业 / [黑渊邀请函](../../../index.md) / 不稳定装置 / 宗师之力〔2/10〕/ 罗兰亚特放射物）、**专属奖励**（断魂影之石·残缺 / 封印之匣）。苏晓借幸运 +1 转运、**抽中了 [黑渊邀请函](../../../index.md)**。
