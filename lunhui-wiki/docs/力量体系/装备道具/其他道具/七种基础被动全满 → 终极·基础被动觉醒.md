# ⭐ 七种基础被动全满 → 终极·基础被动觉醒

⚠ **基础被动·力之源**（对应真实力量属性，⚠ **此技能卷轴仅有你可使用**）是从【基础被动技能卷轴宝箱】开出的**第七种、也是最后一种**基础被动。⚠ **前置需求:力量属性 5 点** · **Lv.1 技能效果:斩击力判定阶位 +1**。

- ⚠ **苏晓不仅一点没感到失望，反而很希望在提升这能力途中千万别出现其他的加成**——⚠ **最好是从 Lv.1 到 Lv.EX 的所有提升与威能，全部集中到「斩击力判定阶位」上**。⚠ **为何如此？因为就在十几个小时前，苏晓刚领略过老兽王的刀术宗师**:⚠ **那一刀下来的酸爽实在太难忘，若非苏晓有「完美格挡」能力，他根本无法与老兽王正面交锋。**（⚠ 不过老兽王那 78 级刀术宗师**并不是只有这加成**，⚠ **其威能还分摊在刀术伤害、技法能力加成、以及强霸体等方面**）
- ⚠ **结果**:**Lv.MAX = 斩击力判定阶位 +8**；⚠ 再以 **7 点黄金技能点**推到 **Lv.EX = 斩击力判定阶位 +10**——⚠ **他担心的情况没出现:所有威能都提升在了「斩击力判定阶位」上。**
- ⚠ **一瓶水的比喻**:「技能就像一瓶水，⚠ **它有多少种加成，就相当于有多少个杯子**，并且要相对均匀地把所有水都倒在各个杯子中——⚠ **当杯子只有一个时，自然就能把所有水都倒入其中。**」
- ⚠ **代价**:⚠ **提升费用贵到离谱**（Lv.1 就要 1000 枚灵魂钱币，⚠ **这还是【轮回·荣耀徽章】减免 20% 后的费用**）——⚠ **他从 Lv.1 直接怼到满级，一直怼到只剩 19618 点灵魂钱币**。
- ⚠ **觉醒**:七种基础被动**并排在下方，各向上蔓延出一根黄金丝线**，⚠ **最终七根黄金能量丝线全部盘绕在最顶部**——⚠ 【提示：**你的七种基础被动能力均已达到 Lv.EX，你的「终极·基础被动」即将觉醒**】。
