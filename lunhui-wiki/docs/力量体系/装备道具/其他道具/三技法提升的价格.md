# ⚗ 三技法提升的价格

| 阶段 | 消耗 |
| --- | --- |
| **正常 Lv.80 以上** | ⚠ **30 颗灵魂晶魄 = 各提升一级** |
| **苏晓（因核心技法刀术宗师有「至刃」加成）** | ⚠ **35 颗灵魂晶魄** |
| **Lv.80 → Lv.90（三技法全）** | ⚠ **总计 350 颗灵魂晶魄 ＋ 332 万以上灵魂钱币** |

- ⚠ **这就是他对城主秘宝格外感兴趣的原因。**
- ⚠ **沉淀琉璃收支**:⚠ **极暗古堡一趟拿到 4787 颗**——⚠ **已经够买【本源果实】**;⚠ **这趟猩红城堡来的太值了**——⚠ **不仅融会贯通了晋升绝强后的强大，还有其他丰厚的收获。**
- ⚠ **他的新念头**:⚠ **是否再掌握一种高阶被动**——⚠ **其他不说，高阶被动是真的强**;⚠ **倘若给近战宗师与血枪宗师也加持上「技之升华·至刃」这等高阶被动**——⚠ **那三技法宗师发展虽消耗大，可与之对等的，是实力的显著提升。**
