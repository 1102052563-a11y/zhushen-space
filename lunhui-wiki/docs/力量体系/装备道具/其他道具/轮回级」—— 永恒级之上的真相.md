# 🏆 「轮回级」—— 永恒级之上的真相

⚠ **⚠ 苏晓把斩龙闪冲过15000评分后，「轮回级」装备/物品权限被提前解除保护性屏蔽**——⚠ **「永恒级为何被称为传统装备体系的最高品级」这句话其实没错:在**虚空之树公开的传统体系**中，永恒级就是最高，评分区间3000～6000点**;⚠ **⚠ 而永恒级之上的品级，在其他乐园阵营是不予公证承认的**——⚠ **装备评分 15000～30000 这个区间，在七大乐园各有各的叫法**:

| 乐园阵营 | 品级名 |
|---|---|
| 轮回乐园 | **轮回级** |
| 死亡乐园 | 死亡级 |
| 天启乐园 | 天启级 |
| 圣域乐园 | 圣域级 |
| 曙光乐园 | 曙光级 |
| 圣光乐园 | 圣光级 |
| 守望乐园 | 守望级 |

⚠ **⚠ 更准确的说法:被所有乐园阵营承认的传统体系最高品级，就是「永恒级」**;⚠ **⚠ 有趣的是市场价排序**:⚠ **轮回级最贵〔普遍贵出10%左右〕→死亡级→天启级→圣域级/圣光级→曙光级〔复苏时间不长、暂未纳入统计〕→**守望级公认最便宜****——⚠ **这不意味着守望级的顶尖装备弱〔同评分强度无差距〕，关键是守望乐园沉寂了，谁也不能确定守望级装备会不会因其更深度沉寂而失去公证**（⚠ 一旦失去公证，除非就在那个世界当场用掉，否则一触发回归权限就会被公正性暂存，想恢复得自掏时空之力重新公证——**一笔巨大开销**·⚠ 至强者们可都知道有「乐园争夺战」这种事:**乐园永不消亡，可对装备的公证没有永不消亡这一说**，除非达到传说中的「个体超脱」）。

- ⚠ **⚠ 五大装备体系的上限排序**:⚠ **传统体系〔绝对大哥〕→深渊级→世界级→霸主级→特质体系〔奇物/厄运物/契约物等·太过杂乱、垫底必然〕**;⚠ **⚠ 获得轮回级权限后，另外三体系的评分上限也从6000解封**:⚠ **霸主级 1～30000 · 世界级 10～30000 · 深渊级 10～30000**;⚠ **⚠ 各体系的性格**:⚠ **深渊级前中期不多见、后期不温不火，到大后期不是强力到极限就是邪门到极点，且「有多强力就要付出多大代价」**〔⚠ 凯撒的【欺诈之手】是「**获得后**〔而非佩戴后〕大幅提高贪婪」——在大后期这会让死亡率提高40%以上〕·⚠ **世界级中正平和:极限增益比深渊级弱约20%，但没有任何减益**·⚠ **霸主级是另一个极端:超过一定排名就开始需要获得其认可，前五的霸主装备搞不好会一件占据全部五个霸主装备位**。
- ⚠ **⚠ 轮回级装备的价格极端到什么程度**:⚠ **最垃圾的轮回级也有200万灵魂钱币/6600盎司的材料保底价**——⚠ **但「装备需求」中一旦出现「获得此装备的**认可**」，价格直接砍到10%；若是只有某个特定人能用的，就妥妥按材料价算**（⚠ 反之，需求里没有「认可」二字的轮回级装备**最低也是6万盎司起步**，遇到合适买家11万+·⚠ **斩龙闪就是最极端的例子**:苏晓看到的装备需求是「灭法之影」，**别人看到的是「灭法之影·库库林·白夜」**——它只有在苏晓手中才是神器）。
