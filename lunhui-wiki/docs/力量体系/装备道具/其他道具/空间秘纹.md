# 空间秘纹（淡金 · 镶嵌秘纹 · 九卷19章·晓阵营商店）

晓阵营商店里苏晓**最想要**的物品——**漩涡一族**的秘纹（**淡金色·评分 285**·淡金 = 暗紫之上的品级·评分 260~310）。烙印在装备或身体某处后获能力「**初级空间禁锢**」：封禁 50 米内空间波动、令该区域**五阶以下空间类技能失效**（30 分钟·冷却 2 天）。**能阻断飞雷神 / 宇智波带土一类的空间忍术**（但级别不够·阻不了乐园的空间传送）。**售价 3000 阵营贡献值**·且**烙印有较高失败风险、后果严重**（连长门都没用·需慎重）。**九卷75章已到手**：苏晓手术救活长门后，[小南](../../../人物/小南.md) 让他**仅花 1720 贡献值**（救命人情）买下——卷轴形态可随时使用，他正犹豫烙印部位（排除身体〔青钢影会噬灭它〕与斩龙闪〔失败破损血亏〕）。
