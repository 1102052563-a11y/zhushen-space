# 🎁 其他战利品与危险物

- ⚠ **命运主宰→永恒级**:⚠ **幸运+15·气运之力（消耗灵魂结晶临时+50%幸运+绝对运势 20 分钟）·极运（开宝箱 15% 概率「最高幅度收益」）·可吸幸运神血升品增强者之名上限**（⚠ 已刻 32 个强者之名·上限增至 41）。
- ⚠ **腐败神血石（污秽源头/封印物）**:⚠ **众神之心的朽败神血所滋生·可锻武器对神灵系（除古神）附高烈度神性腐败·可祭献「旧日之主」·长期持有会侵蚀·封入原罪之书第 8 页**（⚠ 卖回轮回乐园还得倒贴 10 万灵魂钱币封印费）。
- ⚠ **神圣·远古精灵（幸运物）**:⚠ **召唤「圣精灵」吞噬厄运并增益幸运·知其真名可立「精灵誓约」无偿相助**——⚠ **⚠ 圣精灵素质不高（历代精灵王封印她 99% 的时间·只因她一次次搞险些让精灵族噶掉的「运势仪式」）**（⚠ 一次能吞 6000+ 单位厄运不昏死·却被阿什维斯放开的苏晓厄运洪流淹没·好感度 −153·厄运残渣可喂 [厄运古戒](../../../index.md) 升级）。
- ⚠ **黑暗蠕动（深渊级·前代深渊之主脑髓）**:⚠ **双手托着可打开任何等级的深渊封禁/深渊能量覆盖·每次需付出足够黑暗否则被吞噬**（⚠ 苏晓靠它打开无光区最深的深渊封门）。
- ⚠ **时之钟（天启乐园·莫蕾的终极奖励）**:⚠ **可将指定目标的状态恢复到 3 秒前**（⚠ 苏晓用它救回卢西瓦与巴哈的性命）;⚠ **⚠ 星辰宝石**:⚠ **强化永恒级武器时确保 100% 成功、用后枯竭破碎**（⚠ 古龙祭祀场所得）。
- ⚠ **⚠ 铭文基座·神祭 碎了**:⚠ **过载使用太久，战后连碎三次成粉**（⚠ 苏晓放弃修复）。
