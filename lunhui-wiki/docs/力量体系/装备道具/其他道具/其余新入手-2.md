# 💎 其余新入手

- ⚠ **⚠ 【傲意狂歌】的进化**〔以【风海意志】提升〕:⚠ **世界级·戒指·**评分29200点（世界级排行第25位）****——⚠ **傲歌晶体构成与蔓延速度 +550%、综合强度 **12倍**、并具「剧烈侵袭特性」**;⚠ **⚠ 新增效果2「完美无瑕」**:⚠ **傲歌晶体的法力消耗+900%、构成速度降低5000%，但最终强度是当前极限的**5.5～6倍****（⚠ 战斗防御用是超级废技，但对**【战斗义肢】**与**【界级灭法传送阵】**是史诗级增强·⚠ 用它构建的晶体左臂耗时一个多小时、吃掉他全部3000万点法力，换来的是「徒手接鹿神一战棍最多小臂出现裂痕」·⚠ 苏晓由此萌生开发**「无刃流」**〔徒手如持刃，以晶体左臂使用「完美格挡」〕的想法）;⚠ **⚠ 【风海意志】**:⚠ **风海大陆成就超脱的伴生之物·可大幅提升世界级装备，装备破碎后它会重聚为初始状态**——⚠ **来自「支线任务·兽族的复仇」的【地精商会储存凭证】**（⚠ 这正是蜘蛛夫人当初的夺取目标；**老兽王早就估算到风海大陆的命运——与其落入敌手，不如化为今后斩杀敌人的利剑**）。
- ⚠ **⚠ 【隐秘古戒】**:⚠ **月女巫从夜母处所得、按约定分给苏晓的那一份**——⚠ **代表「隐秘之城·亚尔古」的城主身份:可赋予他人进入资格、册封官职〔官职者可获得部分城主能力〕；城主通过给古戒喂食资源让这座半活物的大城逐渐壮大，再靠税收从居民获取资源，良性循环**（⚠ 苏晓先送给索耶被婉拒，最终按索耶的建议**卖回给亚尔古院长——换来一把第一纪元的顶级灭法之刃**〔魔灵已逝但又被重新锻铸过的传世名刀·⚠ **灭法之刃若刀内魔灵尚存，最好别让斩龙闪吞噬:天知道未知先代灭法的魔灵会残存什么特性**〕）。
- ⚠ **⚠ 【铁血暗狼（碎片）】**:⚠ **霸主装备·**四之位**·具不灭特性·收集得越完整强度越高，彻底完整后将重登四之位·碎片间具相互吸引性**——⚠ **原是铁血猎犬一族的圣物、传承于冥界的上古时代，第一纪元被夺，第二纪元被矮人王打造成战靴**（⚠ 七只铁血猎犬本想拒绝这份委托，可**苏晓给的实在太多**——每块碎片100颗「泰坦龙心」;⚠ 而且**圣物早就被打造成战靴，那就不再是圣物了**——「战靴是一双，可铁血猎犬普遍四条腿，这新一代的圣物充满了各种恶意」·⚠ **一颗「泰坦龙心」都不私吞的噩兆，也难怪其他几只那么服气**·⚠ 当前已集7块，进度近半）。
- ⚠ **⚠ 【蠕动之卵】**〔唯一特性·幸运物·价值27000盎司〕:⚠ **顶级幸运物会**吸收持有者的厄运**从而提升其运气——这是标本兼治的顶级方案**;⚠ **⚠ 而这一枚是「邪恶幸运物」**:⚠ **它会通过所吞厄运滋养核心处的邪恶，破卵而出时就是噬主的开始**——⚠ **可这正合苏晓心意:正经幸运物「吞一点休息好几天」，而这枚**有机会就玩命地吞、毫无怨言****（⚠ 为免自身气息吓到它，苏晓先给它上了一层「封印其感知」的滤镜——把「直立血气恐怖人族」伪装成「有些血气但对它造不成威胁的刀术宗师」·⚠ 几分钟内它就从鸡蛋大小吞成直径一米、金色表面遍布黑色纹路;⚠ 「以及，这玩意诞生出的最终邪恶，还可以拿回炼金实验室研究研究」）。
- ⚠ **⚠ 【坏心眼的预言家】**〔深渊奇物·15000盎司〕:⚠ **可消耗当前运势进行占卜——**50%概率绝对准确、50%概率必定错误**，结果取决于此物的「心情」**（⚠ 占卜牌上是一位有着浅紫色眼影、表情冷漠的美丽预言家；落到真实魅力-45点的苏晓手中后，她的心情**非常不美丽**·⚠ 好在苏晓-1500万的运势完全不担心透支·⚠ 此物作为深渊奇物，对因果命运体量的抗性奇高——**毕竟有深渊因果作靠山**）。
- ⚠ **⚠ 【智者秘典】**:⚠ **封印在「灵魂大书库」的秘典，苏晓以20万书库银币暂押借出**〔归还可退〕——⚠ **记载三大类:黑暗知识／阵图术式知识／**当前星界最高深的封印学****;⚠ **⚠ 封面三张金属面孔:「禁忌智者」〔暴怒壮年男性〕「静默智者」〔老人〕「心灵智者」〔冷傲女性〕**——⚠ **本应有考验，三智者同时开口说了一个「你」字后集体沉默**（⚠ 苏晓一度以为交涉走的是魅力属性判定，那以他-36点魅力「说出个你字就没下文」属实正常又不能接受·⚠ 真相是:**三智者的考验并非为难拥有者，而是确定其命运能否承受这些或被诅咒、或过于强大的知识**——而他-1300万的运势加【起源石·世界】的命运体量，**已经代表了一切，考验没任何意义**·⚠ 静默智者的原话大意是:你现在的处境还不适合掌握我所承载的知识，你需要一个足够安稳的环境）。
- ⚠ **⚠ 【初代灭法之戒】**:⚠ **内含微量初代灭法的因果存在;若激发放出，此因果存在将消亡**——⚠ **以冥神掉落的「死之遗物」〔承载「初始之死」的首个死者〕与 [神父](../../../人物/神父.md) 交易而来**（⚠ 此物落入神父手中并非巧合而是必然:**他是当前最强的「存在体系」，只要去往其所在世界必然能察觉；而顶尖梯队的世界就那么多，他九成以上都去过**·⚠ 苏晓的判断:它与专属房间内那条【初代灭法者的右臂（低度活化）】必有莫大关联——**最稳妥的办法是把这戒指戴在那条手臂的手指上**）。
- ⚠ **⚠ 【完整无瑕的时空石】**〔「悬赏·陨灭」所得·价值50万盎司〕· ⚠ **【魅力徽章】**〔嘟嘟咕咕最心爱的珍藏·佩戴后获得800点真实魅力所对应的「交涉判定」·⚠ 因苏晓的魅力情况，从常时被动变更为**每自然日仅生效30分钟**——⚠ 「一名气场和善的刀术宗师，一切都谈好、交易也完成，结果下一秒徽章失效——**那善良阵营的成员都可能被吓得当场休克**」〕· ⚠ **【星辰辉石】×2**〔提升「极境·星界」的专用资源〕· ⚠ **【武器强化权限】**〔以「稀世珍宝·深渊魔晶」换得·可将**任何品质**的武器提升至强化+15·⚠ 轮回级斩龙闪的强化要吃「原初之核」，+11之后更是可怕的无底洞——**这省下的资源量不是一星半点**〕。
