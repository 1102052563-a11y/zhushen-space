# 📜 厄运古卷的终局（52卷42章）

⚠ **苏晓戴上【违规者】称号伪装成绝强违规者「白狼·沃伊赫」**（⚠ 轮回乐园公证的最高阶权限伪装·本是为入侵天启乐园所备）——⚠ **这让他的猎杀者判定、灭法运势等全被暂时屏蔽**;⚠ **⚠ 以这个身份签名后，厄运古卷终于「支棱起来」，海量厄运涌来把他的幸运属性从 80 点冲到 −15 点**——⚠ **【厄运古戒】如长鲸吸水般全部吸收（⚠ 它的厄运吸收早被灭法运势练到 Lv.EX）**;⚠ **⚠ 灵魂强度 890→940 后提升停滞**——⚠ **苏晓随即解除伪装，几大因果（猎杀者判定＋灭法运势＋五股原罪因果＋死寂因果＋起源石·世界的天选因果）全部回归并强烈反噬**;⚠ **⚠ 「厄运古卷」当场爆裂**——⚠ **被「原罪之书」连碎片带灰雾一并吞噬殆尽**;⚠ **⚠ ⚠ 结果:「原罪之书」的封印强度飙升了一个梯阶——似乎再来一件大爹级原罪物，也是有可能封住的。**
