# 🎰 奇异的厄运钱袋（厄运物）

⚠ **[凯撒](../../../人物/尼古拉斯·凯撒.md) 送给苏晓的「有趣东西」**——⚠ **隐隐透出血色的灵魂钱袋。**

| 项目 | 内容 |
| --- | --- |
| 产地 | ⚠ **虚空** · 类别 ⚠ **特殊装备（厄运物）** |
| 前置 | ⚠ **幸运属性 65 点以上** |
| **厄运博弈**（主动·每 10 天一次） | ⚠ **① 永久损失 10% 最大生命值**；⚠ **② 获得 1～50000 灵魂钱币 ＋ 小概率稀有资源**（时空之力/力量源质/原初碎片/灵魂晶魄等） |
| ⚠ 反常特性 | ⚠ **幸运属性所衍生的效果为负收益，厄运则衍生出正收益**——⚠ **越倒霉，高收益概率越大**（⚠ 足够倒霉就一直获得收益、不触发生命值削减） |
| **厄运侵袭**（被动） | ⚠ **触碰即绑定**——⚠ **后续每过 10 天必须用一次，否则永久损失 10% 最大生命值** |
| 评分 | ⚠ **5000 点** · ⚠ **绑定后无法出售/交易/转让/遗弃** |
| 简介 | ⚠ 「此乃与厄运之神的博弈。」 |

- ⚠ **为什么这博弈对苏晓有利**:⚠ **正常人幸运高、注定没什么厄运，大概率损失生命值**——⚠ **可苏晓有灭法运势的倒霉体质、又是药剂大师**;⚠ **被夺走的最大生命值能靠药剂补回来**——⚠ **⚠ 他第一次用就开出 49500 灵魂钱币 ＋ 2 块原初碎片**;⚠ **⚠ 这惊动了厄运之神**（⚠ 虚空一座神殿里睁眼:「又一个，胆敢挑战神灵的厄运系」——⚠ **然而这邪神还不清楚它要面对的是什么**）。
