# 🆕 击杀幽冥大帝·通关战利品（第47卷·潘多拉星🏆通关）

- **背叛者意志**（世界级·消耗品）：以【冥帝铠甲碎片】吸收幽冥大帝本源力量所化——增强 + 异变版的「世界之种」，可**人工培养世界之子**。对高潜力目标使用:21.539% 成世界之子 / 60.62% 成世界之子·伪 / 17.841% 立即死亡。⚠ 培养出的世界之子必是「二五仔」（背叛他人时产生难以想象的愉悦），但苏晓认为「明知某人必背叛，反而可加以利用」。简介「这该死的叛徒」。
- **暗黑战靴**（不朽级·战靴·苏晓新战靴）：乌鹰的【鹰领】宝箱开出。效果①暗蚀:踢技造成眩晕 / 麻木控制时，蕴藏的「扭变深渊能量」侵入敌体、后续 200 秒内力量 -3（可叠 2 次·能量耗尽需「沉淀琉璃」/「黑暗石」充能）；效果②速力增益:踢技伤害 +50% / 移速 +15%。评分 1500（满评分）·穿戴后无法交易。配苏晓「重力透 + 毁灭重炮」的直踹，暗蚀必触发。
- **幽冥之影**（血统 / 职业类·潜力 **S＋**）：深渊宝箱开出——轮回乐园结合**深渊力量 + 大帝战技 + 灭法系能力**融合出的职业传承（比大帝体系更精炼）。苏晓不愿让敌人详细了解灭法系能力，遂卖给神皇冒险团（+ 安排无息贷款、条件是只能自用不得转卖），也方便日后见见这职业的使用者。
- **烈阳圆盘**（罗盘大小·石质圆盘）：报废不朽级【黑暗救赎】召出——【黑暗救赎】可召「衔尾蛇石板」「烈阳圆盘」「暗黑面具·力量渴望」，苏晓与先古面具约定:烈阳圆盘归苏晓、暗黑面具归先古面具（面具吞暗黑面具后波动大涨）。
- **鹰戒**（无品级·纪念物）：乌鹰·索拉罗的【银枭】宝箱开出。稀有金属打造、未经引导附魔，因独特纪念意义不归入材料、故无品级但有价值。简介「看，盘旋在上空的，是乌鹰」——苏晓拟请里德打造成物。同宝箱另得**暗银残灭**（不朽级短刀）。
- **世界守卫者套装**（不朽级·集齐 100 枚「守卫者之证明」兑换）：苏晓因创建阵营无法参战功榜、改判守卫者之证明，反攻冥界一路 + 击杀大帝凑**满 100 枚**，返园可兑换此不朽级套装。
- **阵营商店产出（可带出世界）**：棘拉母巢得适应性公证后如同阵营商店，苏晓以管理权限提货——**Ⅱ型浓缩细胞液**（永久体力 +2 / 生命 +1500 / 异常抗 +4）、**电浆萃取液**（10 日内雷抗 +30~50·苏晓驭雷路一大步）、**精锐恶魔兽胚胎卵 ×30 / 恶魔焰龙胚胎卵 ×2 / 泰坦巨兽胚胎卵 ×1**（可带入非战场世界·配【太阳领主】称号更强·泰坦巨兽是战争级单位·别人只当苏晓是召唤系）。
- **赌徒之灵魂·残缺**（虚空·特殊物品）：世界级宝箱开出。激活后引来**骷髅赌徒**（画之世界那位·已骗到完整深渊之罐、让魔鬼族喜迎「完全体大爹」），可对赌或以此灵魂为酬要其履约。苏晓委托它带话给**恶魔铁匠**——到死寂城相见、交付神灵之灵魂等物。
- **生命铭文**（不朽级·铭文类）：【符文匣】/【神运钱袋】开出。插入铭文基座类装备（如【铭文基座·怒像】3~5 槽）可赋予「生命」特性。
- **铁炼邀请函**（高度稀有·树生世界结算所得）：晋升九阶、主属性达 300 点后可使用；仅可自用或售于轮回乐园（售价 6530 盎司时空之力），转让 / 丢弃将受罚。
