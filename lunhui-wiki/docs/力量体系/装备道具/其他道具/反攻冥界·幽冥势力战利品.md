# 🆕 反攻冥界·幽冥势力战利品（第47卷·潘多拉星）

- **末日君主（八星称号·脂封中）**：名望榜登顶所获·产地虚空之树·特性「极恶阵营」。①放逐豁免:遭世界排斥乃至放逐时强制抵御 3～5 自然日；②无尽恶意:身处极恶阵营时威望默认 +75、对极恶阵营首领级以下有默认震慑；③末日余存:可豁免一次主线任务失败后的强行处决惩罚（用后损毁、化为 5 枚无属性七星称号作燃炼副称号）。表面被罕有树脂封住，剥脂后激活但转为不可交易。简介「末日倾倒，无人幸免」。
- **守卫者之证明**（本世界特殊物品）：苏晓创建阵营无法参与战功排行榜，虚空之树据初始条例 1953489 弥补——在其统领阵营对敌军重创 / 击溃 / 攻占时发放。**累积 100 枚可在轮回乐园换取不朽级套装·世界守卫者**。反攻决战一路攒到 80+ 枚。
- **铭文基座·怒像**（冥界·苦修院）：四铭文槽基座（1 主 3 副）·未插铭文片前 0 评分。形象为怒目威严鬼神——泯光世界古时代苦修者以自身为容器、灵魂化「鬼」食肉身「饵料」育出「凶鬼」、再以大毅力使凶鬼醒悟为「灵神」，此为鬼神之道。市面难买，苏晓从幽冥军需处进货所得。
- **暗影双子**（不朽级武器·双长刀）：军需处进货的顶级双刀。
- **被污染的银币**（虚空·魔鬼族·消耗品）：戴身上可在 3 小时内免疫深渊侵蚀，仅在「次级深渊环境」（如通天王殿）起效——**直接进入深渊内即刻破碎失效**。苏晓以 7 颗沉淀琉璃投深渊之罐换得 7 枚，供决战大帝的四人组 + 布布汪 / 巴哈防身。
- **沉淀琉璃**：母巢大量过滤幽冥能量所累积的稀有产物，多数深渊系存在都对此有所需求，极珍贵。击退幽冥一波得 7 颗、全用于兑换被污染的银币。
- **传承之血·血天使 / 次代龙血**（血统职业类·A＋ / A－）：军需处进货的两种职业模板传承物 + 配套技能传承晶石（血天使需真实体力 80/100/150/200/270 点时陆续掌握 5 种能力）+ 恐惧者套装（圣灵级 5/5）/ 龙鳞荣誉套装（圣灵级 7/7）。
- **空间分割装置**（一次性·开冥界之门用）：近 20 米高的铁黑色导线装置，核心由苏晓以炼金学所制、外壳与部件由帝国在新星城定做。以世界之子的**命运之血**为能量供给、耗血引动本世界与冥界的空间连接，最终撑开几百米宽、过千米高的**幽冥之门**（用毕连同开门者一同被吸入门内）。
