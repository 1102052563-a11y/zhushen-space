# 界断线（超级危险种体毛 · 自交叉之尾）

- 帝具「交叉之尾」（见上「帝具」节）破损报废后，苏晓留下其中一根几十米长的特殊丝线「**界断线**」——超级危险种用于护要害的体毛，**异常锋利**（斩龙闪用很大力都难切断）、坚韧不易损、且**能屏蔽感知**（苏晓自己都差点中招）。他将其缠在小型线轴、固定于左腕衣袖下，用于布置陷阱 / 紧急割杀（虽不如完整交叉之尾灵活·但布简单陷阱足够）——绑架小皇帝时正以界断线套其脖颈「遛狗」（四卷52、54章）。
- **修复 + 强化 +10（十一卷73–74章）**：历经多场恶战、界断线从原 200 米断剩约 100 米。苏晓在乐园锻造街找邋遢老者花 5 万币**修复 + 提品质 → 暗紫色装备配件**（产地斩赤·耐久 190·强度 14·锋利 7·评分 220），又到强化大厅**强化至 +10**（几度从 +2 跌回 +1、+7 起以幻魂石护强、冲 +10 一度失败保住、最终裸强 +10 成功）：青光萦绕、**耐久 230 / 强度 34 / 锋利度 12**（+5 锋利 +1、+8 隐匿 +10、+10 耐久 +50·评分 260）——「束缚住三阶 BOSS 也只能干嚎」（[怨之孤儿·修斯](../../../人物/怨之孤儿·修斯.md) 曾挣断旧界断线、如今加力量增幅也挣不脱）。线轴 + 超小型钩锁同步 +10、移装到新的 [王者御尊护臂](../../../index.md) 上。
