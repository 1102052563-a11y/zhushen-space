# 卞氏匕首（金色 · 匕首 · 十卷72章·库洛洛宝箱·待售）

开 [库洛洛](../../../人物/库洛洛·鲁西鲁.md) **金色宝箱**所得——**正是库洛洛刺穿苏晓后心、伤他最重的那把剧毒匕首**（金色·攻 102~181·评分 380〔金色区间 310~400〕·价 16 万币）。四大被动：**毒刃**（剧毒 0.1mg 可麻痹座头鲸·中刀体力判定失败则麻痹 10 秒~3 小时、体力低则心脏麻痹致死）、**镂空刃**（刺中脏器 33% 概率令其立即枯萎）、**血精灵**（伤口流血不止·缝合 / 药剂无效·直到失血达现有生命 60% 才止·可重复触发）、**薄刃**（穿透 +10）。苏晓自评「伤他最重的不是伊路米手刀、也不是薇妮 10 万一枪、而是这把匕首」（幸亏只中一刀、又有青钢影压毒）。但它需「匕首基础技能 Lv.30」、与苏晓「右刀 + 左界断线」体系不合，**故放弃自用、挂市场出售**（落到刺客手里会是恐怖凶器）。

!!! note "金色品级（十卷·物品品级阶梯）"

    **金色 = 评分 310~400**，是已知物品品级阶梯里**淡金色之上**的更高一档（苏晓首次获得金色宝箱 / 金色装备均在斩库洛洛后）。
