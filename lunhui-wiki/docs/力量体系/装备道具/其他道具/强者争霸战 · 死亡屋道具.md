# 强者争霸战 · 死亡屋道具（第32卷·离开死亡屋即燃尽）

[强者争霸战](../../../世界/任务世界/强者争霸战-暮色之岛.md) 第二阶段「死亡屋」内的特殊道具（多为离开死亡屋即燃尽 / 收回）：

- **诅咒金币（死亡屋版 · 特殊装备）**：抛硬币赌「生 / 死」——正面随机增益（永久 +幸运〔最多 3 次〕/ 回血 / 暗影石），反面随机减益（临时 −幸运 / 被「斧哥（暴食兽）」追杀·可叠加）；无视幸运属性、冷却 30 分钟。苏晓硬抗 14 名斧哥追杀（遛 20 小时）刷得 3 暗影石 + 永久幸运 +2（→ 14）。**与月神大陆的同名 [诅咒金币](../../../index.md) 是不同物（这枚离开死亡屋即燃尽）**。
- **暗影石**：死亡屋的「货币」——可与亡者交易；对自身用则进「暗影庇护」状态、能与无法交涉的危险亡者交涉 / 交易（向无声者买白脂蜡 1 颗 1 根）。
- **白脂蜡**：可照明 90 年的蜡烛——**驱散死亡屋的「黑雾」**（黑雾会侵蚀一切·只能驱前方·驱黑雾时燃速加快、限时 20 分钟）；进黑雾深处的唯一通行手段。
- **树之生命（炼金配方 · 封之箱所得）**：每次耗 1300 法力 + 迷火木核 / 灵魂结晶〔小〕等——饮 / 注射**永久 +3 真实体力**（150 点以下生效）；比光辉药剂更受欢迎（没人拒绝涨体力）。
- **秘迹宝石（圣灵级宝石 · 圣女特蕾西所赠）**：评分 990——镶武器 +85 锋利度 + 每击削敌 8% 防御（叠加上限 30%·自适应：有盾破盾 / 有甲破甲 / 都没有破肉防）；或使用永久 +3 真实敏捷。因圣灵级宝石特性、暂镶不进已满的斩龙闪。
- **处决之刃（特殊武器 · 死亡屋）**：专用于「杀某个人」、平时拿来战斗会断——实为**制约死亡屋主人**的器物（由死亡屋生成·毁了会再生），用于防主人像爱丽丝那样肆意破坏规则。苏晓拿它没杀 [安娜](../../../世界/任务世界/强者争霸战-暮色之岛.md)、而是换了航海罗盘。
- **航海罗盘（圣灵级 · 道具 / 特殊装备 · 风海大陆产）**：海眷族专属（需魅力 151+·苏晓装不了）——但「**强掠**」可无视需求强制使用一次：**罗盘指针指向使用者心中所渴望之物**（用后永久损坏）。苏晓视若「找任务目标神器」、留作关键时刻。
- **樱风酒（圣灵级 · 羽族秘藏 · 恶魔族 + 羽族两老家伙所赠）**：评分 1000——战时饮可立即全恢复生命 / 法力 + 解七阶以下异常（可分 3 次·体力破 150 仍全恢复）；或非战时一饮而尽**永久 +2500 生命 / +1100 法力 / +2 力 / +3 敏**（苏晓直接喝了涨属性）。提示揭示「**150 点真实属性壁障 = 七阶门槛**」（突破后有 150 属性奖励效果·届时远古秘药恢复效率会暴跌）。
