# 🆕 深蓝之影（八星）· 神圣分割器（第48卷续）

- **深蓝之影（攻击型八星称号·成型）**：苏晓借本世界称号商店，用 6 枚六星称号喂【圣餐】（使其变为「必被反吞」的逆食）、令【深蓝之影】反吞【圣餐】升七星，再镶 5 枚七星副称号燃炼（13h21m）成**八星**。三项朴素强化——①**适应性之力**:刀 / 灵魂 / 能量伤害阶位 +1（三选一·永久·苏晓选**刀**，其刀伤害阶位由此叠满:刀术宗师 Lv.60 +1 / 技之升华满级 +1 / 银·月狼狼血挂饰 +1 / 深蓝之影 +1，这正是他能斩冥界大帝的底气，寻常不朽级长刀 +14 都未必行）；②**不灭**:总生命值 +30%（优先加成，取所有生命加成总和的 30%）；③**深蓝**（核心）:身体速度常时 +25%、每次受击再 +5%（可叠 3 次、满 40% 后持续到战斗结束）——深蓝能量满时可撑 12~17 个自然日的战斗，长期共鸣还会永久拔高速度潜质。简介「深蓝中的怒杀之影」。攻击 / 生存 / 速度三项皆强，正是他为闯死寂城准备的战斗称号。
- **神圣分割器**（世界级·虚空·第二纪炼金文明）：[镜中恶灵](../../../世界/任务世界/幽暗大陆-死寂根源之地.md) 交出的方块状精密造物（约魔方大小），触发连锁支线「镜子与分割」——苏晓忽略前三环、以物理手段直接完成最终环取得（故无世界之源奖励、只得恒定奖励）。⚠ 仅能在本世界内使用，离开时被强制留在本世界随机地点、折算成一定量「世界之力」补偿。
- **太阳柱 / 太阳桶**（炸瓦迪庄园用）：苏晓集四方材料一晚造出 123 颗「烈阳之怒·阿波罗」，封成一根**太阳柱**（100 颗浸溶液）+ 若干**太阳桶**（10 颗阿波罗 + 塞尔星高爆火药）+ 3 颗单发。⚠ 没造威力更大的「太阳圣剑」——那玩意曾把树生世界陆地从中间炸成两半，丢北城区会把整座高墙城炸没。
