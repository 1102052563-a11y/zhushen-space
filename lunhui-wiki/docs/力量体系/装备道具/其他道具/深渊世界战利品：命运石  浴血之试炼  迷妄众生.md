# 深渊世界战利品：命运石 / 浴血之试炼 / 迷妄众生（三十一卷·斩黎林 / 亚历山所得）

[深渊世界](../../../世界/任务世界/深渊世界-塞拉帝国.md) 的圣灵级收获：

- **命运石（圣灵级 · 宝石 / 道具 · 三十一卷19章·斩黎林众神会奖励）**：评分 927——镶装备 +5 幸运；或使用后**永久 +2 幸运 + 幸运系神灵 3 天赐福（运势大幅加成）**。苏晓得后思索「能不能打过 [幸运女神](../../../人物/幸运女神.md)」（估计打不过·须晋七阶才有望）。
- **浴血之试炼（圣灵级 · 辅助类卷轴 · 三十一卷23章·亚历山住所搜出）**：评分 998（售 59.6 万币·剩 1/2 次）——前置意志 80 + 灵魂强度 73（未达强用永久陷狂乱）；使用后须**高强度战斗才能提升基础类技能**（提升幅度按使用方式·持续时间自适应）。苏晓盘算用它在**面对大量敌人**时提升刀术宗师（一对一用 = 找死）。
- **迷妄众生（史诗级 · 手链 · 三十一卷23章·搜刮所得）**：魅力系装备（大幅延长魅惑持续 + 提真实魅力）——苏晓没魅惑能力、魅力又永远是 1，直接忽略 / 待卖。
- **灵能液体（道具 · 三十一卷22章·弑圣裔王亚历山恒定奖励·1300 克）**：进 [深渊](../../../世界/任务世界/深渊世界-塞拉帝国.md) 的关键道具——洒在皮肤上即可**破除感官误导 / 看穿隐匿监视**（苏晓靠它看穿并消去深渊里那只监视的「竖瞳」、避开危险一路无阻抵神之礼殿）。任务提示深渊极危、须备足灵能液体。
- **神之国区域奖励（灵魂钱袋 / 七阶灵魂钱币 · 三十一卷35章起）**：杀 [天巴族](../../../世界/生物万族/index.md) 掉**灵魂钱袋**（开 1~10 枚**七阶灵魂钱币**·七阶通用货币），可在神之国区域兑换列表换：**天行**（圣灵级挂饰·短时提远程基础能力两阶位至宗师·挂饰仅可戴一件·105 枚）/ **电行**（圣灵级挂饰·雷电系 +32%·103 枚）/ **神裁**（成长类装备·击杀极恶神灵〔多为古神〕成长·成长条件明确·129 枚）/ 尊贵腕饰 / 天巴族语言（1 枚·苏晓全员兑了好交涉）。苏晓借控桑冷给即将变怪物的天巴「安乐死」刷到 61 个钱袋，奔着 神裁 + 天行 去。
- **旧神骨灰（道具 · 巫师世界产·三十一卷45章用）**：使用后随机召出一名旧神（1.7 秒构建身体·存在 30~45 分钟）——苏晓在渊蛇邸用它召旧神**拉满仇恨**（旧神周旋近两分钟才被打散）、掩护自己控制沁天巴桑冷。
