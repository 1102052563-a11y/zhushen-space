# 🎭 一瞬间的骗局 · 四爹互锁（50卷64章）

⚠ 苏晓先以猎杀者权限**向轮回乐园申请公证恒定**，才把收藏家暂封入封印盒（代价:消耗 Lv.1 额外权限）。⚠ 随后是一场极限操作:

1. ⚠ 他用小刀划掌、把**鲜血 + 液态青钢影能量 + 灵魂能量**融合（**即生命力、身体能量、灵魂能量三合一**），⚠ 再**心疼地捏碎一颗「灵魂晶魄」**混入，得到一种**鲜红的浓稠液体**，倒在「原罪物笔记」的封皮上——⚠ **鲜红液体将封皮中心的灭法印记染红并融入其中**，⚠ **「原罪物笔记」竟有了几分活物的气息**（虽维持不了多久，但也足够了）。⚠ 此时封面上的灭法印记**犹如活物的心脏般、甚至有了心脏跳动的那种感觉**，印记周边也**蔓延出血管与经络等**。
2. ⚠ 苏晓**注射抑制药剂 + 收敛气息**到近乎完全消失（阿姆在旁握着中和针剂待命）。⚠ 如此一来「原罪物笔记」透出的生命气息**与苏晓有九成九的相似**，⚠ 加上苏晓本人隐匿气息、**更为重要的是这里是他的专属房间**。
3. ⚠ 他把笔记压在封印盒上，**故意在盒顶开了一个小洞**（仿佛封印盒无法再封住收藏家）——⚠ **收藏家立即挤出、直接触碰到笔记，第一时间误认为「原罪物笔记」就是苏晓，并开始进行同化**。⚠ **哪怕只持续了一瞬间它就发现不对、立即停止——但苏晓要的就是这一瞬间。**
4. ⚠ 与此同时苏晓**不惜破坏封印盒**，打开封着死灵之书、幽冥骨戒、灵魂王冠的另一个盒子——⚠ **在收藏家的主导下，正在以「原罪物笔记」封印死灵之书、幽冥骨戒、灵魂王冠**。

⚠ **收藏家面临的抉择**:①**继续和三爹硬怼**（下场想都不用想，三位不会好受，**但收藏家一定最惨，而且是这件原罪物从未有过的惨**）；②**与「原罪物笔记」同化**——⚠ 因笔记在因果层面**可以代表苏晓**，那它做的就相当于**和苏晓一同封印三件原罪物**，僵局迎刃而解。⚠ **可一旦这样做，它就等于被吸收到「原罪物笔记」、取而代之，成为三件原罪物的封印**；⚠ **更要命的是:它同化不了苏晓了**——⚠ **因为它已经同化过作为苏晓替代品的「原罪物笔记」，原罪物这东西在因果方面的限制很强，它绝不会在同化了『苏晓』一次后再同化苏晓一次**（就算它是顶尖大爹级也做不到）。⚠ **这种局面下它不仅无法同化苏晓，还得帮苏晓接着封印原罪物**——⚠ **因为它的原罪特性就是收藏，而有什么比深渊·原罪物更好的收藏品？**

- ⚠ **结果**:四种光华（灰白 / 幽绿 / 黑红 / 暗金）盛放，**专属房间的绝对安全无敌护盾**升起（苏晓、布布汪、阿姆、巴哈、连在小喵屋中睡觉的贝妮身上都浮现金色护盾）；⚠ 一切结束时**入目之处全是灰渣**——**除了荣誉商店外，房间内的其他东西都化为这种灰渣**（好在高价值物品都在炼金实验室、药剂室、种植区）。⚠ **事实证明，即使是顶尖爹级器物，也不会想和另外两件顶尖爹级器物、外加一件大爹级器物硬怼**——⚠ **它选择了后者。**
- ⚠ **成品「原罪之书」**:表面流淌暗金色花纹、约**一指厚**、封皮上有**像血管与经络结构、也有些接近深渊纹**的不规则花纹。⚠ **和预想中的不同**:「原罪之书」和收藏家**并非一体**，而是**收藏家寄宿在其中**——⚠ **如果苏晓某天死亡，那他与收藏家的因果就解除，届时收藏家就能破封而出**。
- ⚠ **页码表**:**第一页=收藏家**（暗色调 + 黑暗中透出金色的旋涡，⚠ **这正是收藏家进入「渴望」状态、要吸收掉一个世界时的模样**；⚠ **它既是被封于此，也是这本书的锁镣——正因第一页封着收藏家，这本书才能封印其他原罪物**）· **第二页=死灵之书**（纯白背景 + 灰白，⚠ **手指抚过书页都能感受到那种人皮书的质感**）· **第三页=幽冥骨戒**（一片黑暗中透出幽冥的幽绿色，⚠ **体现并不是戒指模样，而是一只只骨手组成一枚指环、最终缩小后才呈现出骨戒的模样**）· **第四页=灵魂王冠**（鲜血的猩红中一个**漆黑犹如被火焰烧过的黑色王冠**，王冠上镶着颗暗红宝石，⚠ **仿佛这宝石能放大所有欲望**）· **第五页=[熔金魔](../../../人物/熔金魔.md)** · ⚠ **全书共七页，但第八页的位置只长出了一部分**——⚠ **那感觉就像第八页被撕掉了般，实际上并非如此，是第八页在生长，只是还没满足生长出的条件而已**。
- ⚠ **它强在哪**:⚠ **「原罪之书」本身不算特别强力，关键是里面四件原罪物彼此的钳制关系**——⚠ 死灵之书被封进来后**当即反制收藏家**（两位威能相近、彼此奈何不了对方）；⚠ 幽冥骨戒的判断是**收藏家与死灵之书一同封印了它**（⚠ **幽冥骨戒是好惹的吗？这位连苏晓都懒得理睬，何时受过这等闷气**），当即与二者对峙、造成**三方都出不去**；⚠ 灵魂王冠进来时**因为是在场垫底的所以没发出什么波动，但它把力量放了出来**:⚠ **既然它被封印进来，那就都别出去了**。
- ⚠ **隐患**:苏晓有种感觉——**他能把除收藏家外的其他三件从里面拽出来，而当三件都拽出来后，届时就能把收藏家也拽出来，到那时「原罪之书」就空了**。⚠ **这不是长久之计**:**当四件原罪物彼此和解、不再互相钳制后，它们都能从里面出来**——那样苏晓就要同时封印四件原罪物。⚠ **他估测自己大概有一到两个世界进度的时间去增强它**，最理想状态是**只凭「原罪之书」就能封印所持有的四件原罪物**——⚠ 这需要**很惊人的封印系内核**，以他**初级大师级**的封印学根本做不到，⚠ **最起码是顶尖大师级的封印学才能做到**。⚠ 他决定**把大部分灵魂书库银币都用来兑换封印学**:「有四位原罪物『老师』的督促，⚠ **主要是做不到，小命都可能会没，这动力一下就足了**。」⚠ **然而苏晓此刻还不知道，他搞出的这「原罪之书」，就算虚空养爹人魔鬼族见了，都得被刷新认知。**
