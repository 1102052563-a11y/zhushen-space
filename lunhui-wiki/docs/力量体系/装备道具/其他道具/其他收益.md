# 💎 其他收益

- ⚠ **君主宝藏五五分账**:⚠ **81960 灵魂钱币 ＋ 21 颗灵魂晶魄 ＋ 4 块原初碎片 ＋ 风暴盾（永恒级·未鉴定）＋ 诅咒之书（永恒级·未鉴定）**——⚠ **一些原初碎片与灵魂晶魄因保存环境不佳损坏失值。**
- ⚠ **纯净血石**:⚠ **红眼战士系的击杀产出**（⚠ **因本源生命力被猩红同化、冷却后的凝结体**）——⚠ **可与 [神秘商人](../../../人物/神秘商人.md) 交易换深渊特性装备**;⚠ **苏晓斩 50 名红眼骑士得 528 颗、外加零散共 500 多颗。**
- ⚠ **源质宝箱**:⚠ **30% 获「力量源质」/70% 获「特殊源质」**——⚠ **苏晓最近刚用两份力量源质，暂不开**（⚠ 短期提升太多身体属性会无法适应、战力反而下滑）。
- ⚠ **神圣宝石**（世界级·6000 满评分）:⚠ **对 −21 魅力的苏晓是废物**——⚠ **只 +3 幸运、后续还 −42 幸运**（详见 [女巫·莉莉亚](../../../人物/风海大陆/女巫·莉莉亚.md)）。
- ⚠ **黑枫树边角废料**:⚠ **连废料都能引开腐烂巨人**——⚠ **食暗者震惊「你可真浪费」**;⚠ **⚠ 苏晓「因为我有棵黑枫树」，把食暗者整不会了。**
