# 🖤 【魂·王之加冕】—— 装备形态的「至强领域」

⚠ **以【永生宝石】解开【黑王护臂】最后一重封印而成**——⚠ **轮回级·护臂·产地「根源·死寂城」·装备需求「仅白王可穿戴」·评分 30000 点〔轮回级满值〕**;⚠ **⚠ 基础效果:豁免濒死状态，直至死亡**〔此加成拥有最高优先性〕;⚠ **⚠ 死寂降临（核心·主动·无消耗）**:⚠ **释放「死寂领域」高速同化周边10～500000米环境**——⚠ **敌人全属性-85%/身体防御力-95%/全抗性-95%/移动速度-120%/每秒承受最大生命值20%真实侵蚀伤害，并须过「死寂同化」「暗黑蒸发」「苍白意志」三项判定，任一不过即承受即死**;⚠ **⚠ 三重警告〔原文连写三遍〕:身处轮回乐园所公证世界内，猎杀者不可彻底展开「死寂领域」**——⚠ **完全展开会把整个世界逐步同化，且持续时间无上限:即便他已离开此界，领域仍会吸取该世界的世界之力继续存在，直到此界崩灭**;⚠ **⚠ 关键提示**:⚠ **因这「至强领域」并非他自身所拥有，本在领域交锋中处于劣势**——⚠ **但「极境体质」对它进行了「超巨幅度」增强:可**100%抵抗敌人释放的至强领域、让两者完全抵消****（⚠ 由此反推:「死寂本源」极可能就是 [永生之神](#) 昔日的「至强领域」被深渊无节制增益后的版本——**这也意味着永生之神的巅峰战力必然达到过至强巅峰**，否则幽暗大陆连向超脱迈进的资格都没有，谈何超脱失败）。
