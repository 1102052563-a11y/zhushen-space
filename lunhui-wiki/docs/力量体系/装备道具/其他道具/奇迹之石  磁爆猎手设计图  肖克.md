# 奇迹之石 / 磁爆猎手设计图 / 肖克（塞壬星·掠夺所得 · 二十九卷21–24章）

- **奇迹之石（天启乐园·无品级道具）**：掠夺契约者营地的资源箱开出的青色宝石——**一次性解除自身所有异常状态**（解除上限阶位「？？？」·苏晓权限够不到·上限未知反是好事）。被诅咒 / 远程恶意加害 / 强减益时的翻盘保命利器（无法出售）。
- **磁爆猎手设计图（史诗级·塞壬星械铠图纸）**：[铁甲熊·纳加](../../../人物/塞壬星群像.md) 偶得、当见面礼送苏晓的史诗级机械外骨骼设计图（黑底白线·十几副带文字标注的图案）。
- **肖克（塞壬星·能量货币）**：罕见能量矿石雕成的硬币——驱动机械外骨骼的必需品、避难所通用货币（亦称「肖克币」）。斩亚哈达得其全部财产 10 万枚+。
- **危机预警（史诗级·被动技能卷轴）**：资源箱开出——苏晓不发展它，而是用 [直感](../../技能/index.md) 的「兼容其他感知能力」特性把它**归纳入直感**（卷轴技能消失、直感补全变强）。
