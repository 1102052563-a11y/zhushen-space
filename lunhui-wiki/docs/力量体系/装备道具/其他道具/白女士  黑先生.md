# 白女士 / 黑先生（淡金 · 十三卷28章·开麦野 / 史提尔宝箱所得 · 魔禁世界）

开 [麦野沈利](../../../人物/麦野沈利.md) 的淡金宝箱得的一对「**辣眼装备**」（产地均为学园都市「爱与情威而刚媚药公司」·实物本是震动棒、被乐园判定成装备·装备后皆魅 +5「兴奋」）：

- **白女士**（淡金·30cm 白色「棍状治疗法杖」·270 评分·需女性 + 智 75/力 8/魅 8 + 法杖精通 Lv.25）：治愈（疗效 +8% + 智 ×5%）、心动（治女性额外 +80 + 5% 已损生命·治男性 +10% 防御 / 魔抗）——对奶妈职业提升极大（苏晓用不了·难以想象奶妈拿它施法）。
- **黑先生**（淡金·黑色带颗粒「棍棒」·攻 1~199·270 评分·需男性 + 力 75/敏 50 + 棍棒精通 Lv.25）：强攻（打男性 +17 真伤·打女性 7% 眩晕）、弱点打击（击中下体要害基础伤害翻 2~5 倍）——威力浮动大、运气成分高。
