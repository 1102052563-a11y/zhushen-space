# 起源石（特殊装备 · 竞技场首位累计奖励 · 二十八卷45章）

[苏晓](../../../人物/苏晓.md) 打上五阶乐园竞技场首位、竞技奖励累计至极限所得（轮回乐园产·耐久 7000）——被动「心之领」：随身大幅提升对世界 / 自然 / 环境 / 空间的体悟，**加速刀术宗师 / 心之冥想提升**（近战肉搏小幅）·近似随身一座「[众生之地](../../../世界/地点/index.md) 一层」。**仅使用权**（名次被同样满累计奖励者夺走则失·保持首位晋六阶才永久拥有·死亡 / 遗失乐园收回）。六阶以上原生世界崩溃成噬灭黑洞、压缩质变 0.572% 几率形成。

- **🆕起源石·世界（2/5）**（44卷·画之世界通关奖）：画卷争夺战首位得**起源石·世界**的第一块残片，返园强化期又凑到**第二块**（44卷137章）——五块残片不能拼接，须集全后在轮回乐园内以大代价、耗权限等级融合。天启乐园曾有矿工把完整的起源石·世界镶在矿铲（独有装备）上「半上交」保命。
- **🆕起源石·银皇后（完整 · 后缀起源石 · 44卷137章）**：苏晓新得的一颗银白色**完整起源石**（比单块的起源石·世界更强·耐久 7500）——装备效果「**银色七感**（被动）」:随身提升感知 / 冥想效率 / 意志力 / 精神力（心之冥想升级速度 ×4.7、直感能力升级速度 ×27.5、长期携带意志力 / 精神力渐增）。其来历骇人:导致一个世界毁灭的**银皇后**（虫族母皇·超高危险生命体）意识被星辰崩灭形成的「噬灭黑洞」吸收，几千年孤寂高压后放弃思考、思维彻底停止，最终世界残骸 + 黑洞 + 银皇后意识三者合一压成此石——**严禁尝试唤醒其意识**（母皇意识若未死会有概率被唤醒）。
