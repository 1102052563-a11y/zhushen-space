# 🆕 幽暗大陆·晋升考核相关（第48卷）

- **深蓝之影（升八星·攻击型个人战力称号）**：苏晓仅有的能提升个人战力的称号（原五星·「唯一效果」按真实力量每点 +0.5 肉体防御 或 真实体力每点 +30 生命·二选一·可质变为攻击特性）。⚠ 他在幽暗大陆故意让主线任务失败、以八星称号【末日君主】豁免强行处决换来 5 枚无属性七星副称号，正是为了把【深蓝之影】燃炼成**攻击型八星称号**去闯死寂城（八星战斗系称号能大幅提升战力）。
- **星流矿**：一种埋在地下几千米、本世界开采技术够不着的高价矿藏——竞拍平台上天启方契约者曾寄售、一块 700 灵魂钱币。苏晓的「挖矿二人组」（沉默仆从勘探 + 隧掘仆从开采）一晚就在南城区地下挖到，几乎是无本买卖，够攒出三宗师破 Lv.70 所需的「技法之魂」。
- **圣所钥匙 / 庇护石**（幽暗大陆特殊物品）：圣所钥匙是晋升任务第三环的关键物（藏在畸变的瓦迪庄园内·指向死寂城「至高圣所」）。庇护石为晋升任务各环奖励（惊变环 ×1、圣所钥匙环 ×7）。
- **世界之眼 / 世界猎手 / 世界眷恋（世界三件套·实战用途）**：苏晓用它们培养世界之子克兰克——世界之眼与克兰克双眼融合（情绪波动吸世界之力提升战力）、世界猎手存世界之力（储量随实力永久涨·目标撑到 50 盎司）、克兰克靠背叛者意志的「背刺即剧变」持续触发世界之眼。这是苏晓一探世界之力玄妙、提升「永久性增益药剂」上限的关键布局。
