# 🕳 灭都卡因的三件专属道具

- ⚠ **【神奇万能钥匙】**〔厄运物·斩双生王得〕:⚠ **能打开一切意义上的锁**——⚠ **苏晓用它开黑石壁进深渊近区〔每次开锁50%吞其厄运·10日内多用胃口翻10倍·「不知该庆幸能无代价用还是该皱眉自己厄运多庞大」〕**;⚠ **⚠ 【封印卷轴·无】**〔轮回乐园·永恒级〕:⚠ **击杀敌人后剥离其一种能力封印成技能卷轴**——⚠ **消120万生命+10万法力·苏晓用它剥都卡因始祖的「不朽意志」才真正击杀→得【不朽意志】〔绝对意志·契合度低·拟3万盎司换一种〕**;⚠ **⚠ 【元素旋刃】**〔悬赏4联动·螺旋匕首·因果破灭·耐久1/1〕:⚠ **专破深渊族裔的「始源族徽」·「所有弱点皆在始源、不死不灭不过是根源尚未破灭」**（⚠ 苏晓以自然元素造【元素旋刃（伪）】破都卡因始源族徽→灭族·真品留着对付其他深渊族裔）。
