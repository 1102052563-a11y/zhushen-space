# 🔥 铭文基座·神祭 · 炙热过载（61% → 85%）

- ⚠ **为何要过载**:⚠ **【无上烈阳】的效果虽不错，但苏晓现在是以「太阳圣剑」作为对敌手段**——⚠ **哪怕豁免 61% 伤害，他依然顶不住**；⚠ **别看古亚院长硬抗「太阳圣剑」，但古亚院长是老牌绝强者，外加对方的体魄可一点都不弱**（⚠ **这点从【重力法环】就能看出**）。
- ⚠ **换料**:四块**不朽级副铭文**当初总计花 24900，⚠ **眼下卖掉竟卖出 26800 枚灵魂钱币**——⚠ **用了一个世界进度，不仅没赔还赚了 1900 枚灵魂钱币**。⚠ 换上四块【**熔铁铭文**】（**永恒级·提升部分火系能力强度**·评分 3900～4000，⚠ **是永恒级偏下游**——⚠ **可惜:这原本应该是一块铭文石，结果被打碎，只能用碎片做成铭文片**）。
