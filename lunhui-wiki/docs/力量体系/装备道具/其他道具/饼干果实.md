# 饼干果实（超人系恶魔果实 · 紫~传说级 · 十八卷39章·开克力架宝箱所得）

[苏晓](../../../人物/苏晓.md) 用欧皇状态开 [克力架](../../../人物/夏洛特·克力架.md) 掉的传说级宝箱（闪光）所得——评分 **520**·价 25.8 万币。**食用后可制造 / 操纵饼干**（硬度取决于果实开发程度 + 个人能力·下限恒定）；通用弱点：畏海楼石、**50% 身体没入海水严重脱力 / 95% 没入淡水轻微脱力**（但苏晓分析饼干果实其实可规避——用饼干包裹全身避开海楼石、反应够快造饼干漂浮避免落海）。**苏晓不打算自己吃**（[布布汪](../../../人物/布布汪.md) 记得斯佩罗怎么死的、不想变旱狗；[阿姆](../../../人物/阿姆.md) 体内冰能量吃了恐有副作用）——他盘算把它**卖给 [夏洛特·玲玲](../../../人物/夏洛特·玲玲.md)**：托特兰 80% 岛屿、95% 建筑框架都是克力架用此果实造的高硬度饼干（既是食物 / 建材也是镇压暴动的武器），克力架一死便无人能翻修；大妈**势必不惜代价找回饼干果实**（若被外人吃下开发出来就能操控整个托特兰的饼干）——是比乐园里大型冒险团更舍得割肉的买家。难点：当面拿出果实可能让大妈直接暴走。
