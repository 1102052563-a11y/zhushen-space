# 深渊世界圣灵级装备：神裁 / 天行 / 湛蓝意志 / M·暴君（三十一卷·苏晓备战强者争霸战）

[深渊世界](../../../世界/任务世界/深渊世界-塞拉帝国.md) 通关前后入手的圣灵级 / 成长装备（多为强者争霸战做准备）：

- **神裁（传说级 · 成长指环 · 三十一卷48章·神之国兑换 129 七阶钱币）**：**魂之生（被动）= 每点灵魂强度 +80 生命 + 0.5% 神经反射**——苏晓灵魂强度 150（噬灵者攒的）故直接 **+12000 生命 + 75% 神经反射**（灵魂强度每世界还 +30·这装备只会越来越强）；成长条件「击杀极恶神灵（多为古神）」明确（苏晓宰古神不少）。评分 530。
- **天行 / 电行 / 湛蓝意志（圣灵级 · 挂饰 · 挂饰仅可戴一件）**：**天行**（短时提远程基础能力两阶位至宗师·5~30 分钟·兑换 105 钱币）；**电行**（雷电系 +32%·兑来卖给雷系法爷）；**湛蓝意志**（**所有近战总伤害 +20%**·前置宗师级 Lv.30 近战·评分 995·用魔力裂变 + 电行换得·苏晓近战主力的核心增益·与天行二选一戴）。
- **M·暴君（Metal·暴君 · 圣灵级重型狙击炮 · 三十一卷57章·交易换得）**：莱斯利·蒙哥马利的最终之作（与 [D·暗杀](../../../index.md) 同系列·评分 1000·售 63 万币）——子弹飞行速度 +600% / 2600 米几乎无下坠 / 死亡锁定（瞄 2 秒生物锁定无视 90% 障碍）/ **杀掉锁定目标即进入「高能」**：消耗自身法力源源不断造「能量风痕子弹」自动填弹（直到法力 <10%）——苏晓 13176 法力 = 一个人一把枪打出一个团的火力。唯一缺点：一发实弹 8 万币。替下 [噬灭](../../../index.md)。
- **庇护宝石（圣灵级 · 特殊装备 · 三十一卷53章·圣灵宝箱开出）**：绑一名友方为「庇护之人」、可开空间裂痕把其传送到身旁救援（庇护者可拒·1 小时后传回·须同世界）——苏晓给 [贝妮](../../../人物/狩之影.md)（寻宝常受欺负·有了它再被欺负就不必硬忍）。
- **神灵骨（圣灵级 · 合成道具 · 三十一卷52章·弑神奖励）**：凑 3 块同品质合成「**神灵之奇迹**」（可把任意史诗级装备升至圣灵级）——苏晓曾用传说级神灵之奇迹把命运主宰升传说级。
- **复苏之木（白~史诗级 · 黎林遗物 · 三十一卷53章·「恶意」）**：按使用者魅力决定治疗量的群疗道具（魅力高可回 50 米内 2500 生命）——**苏晓用只回 1 米内 2 点生命**（魅力 1·黎林公爵的「心爱之物」对苏晓恶意满满），转手给布布汪。
- **荣誉强化药剂（六阶 · 荣誉商店 · 17 钻石勋章）**：永久 +2 力 / +2 体 / 轮回烙印 +1（苏晓 Lv.58 → 59·一次结算即卡 Lv.60 晋阶前置）。荣誉商店另有霸主级成长装备 **嗜血狂笑**（109 钻石勋章·苏晓攒着）。
