# 木之灵（传说级 · 万能装备 · 十五卷106章·开柱间传说级宝箱所得）

[苏晓](../../../人物/苏晓.md) 开 [千手柱间](../../../人物/千手柱间.md) 掉落的传说级顶级宝箱所得——**他的第一件传说级装备**（一团苹果大小、柔软如毛线球的翠绿缠绕树枝·耐久 103·评分 **526**〔传说级 400~530〕·需真实力 75 / 敏 70 / 体 80·绑定后与灵魂绑定·不可售、可熔炼分解）。它没有花哨能力、却条条简单粗暴，**几乎适用于所有契约者**：

- **木之活化（被动 · 防御形态）**：每分钟生命恢复 +130、生命 +1200、身体防御力 +40——单这三条就超过三件猎魔套总和，近乎给苏晓**提升了 2/3 的生存能力**（生存力超大型冒险团的主坦）。补足了「斩龙闪斩不动柱间」暴露的「敌人防御力高」短板。
- **木之野蛮（主动 · 攻击形态）**：木之灵包裹手臂 / 足部、消耗 15% 当前最大生命，下次攻击**用武器 = 力量 ×0.3、徒手 = 力量 ×3** 的攻击 / 击飞 / 击退（80 真实力量 ×3·一脚能把敌人脏器踢飞·冷却 10h）。
- **自由切换（主动）**：攻 / 防形态随意切换（冷却 1h）。
- **「万能」特性**：经乐园判定可拟形成戒指 / 项链 / 护臂 / 上下身铠甲等**任意穿戴位（喂养分还能成铠甲）**，也可拟成「**辅助装备**」（如 [命运救赎](../../../index.md)·不穿戴也享被动·辅助位共 3 件）——这一「万能」属性是传说级里仅 6% 几率出现的稀有属性。苏晓现把它拟成护臂（2/2·与王者御尊护臂）/ 下装、切防御形态生命直接 +1/3。
- **🆕强化至 +10（十七卷87–95章·氧化裂剧变装置）**：苏晓用 [氧化裂剧变装置](../../../index.md) 催化木之灵（双形态强化费贵·裂变随机·+8~10 仅 10% 几率）——运气爆棚直接出 **+10**（耐久 143·评分升至 **530** 满格传说级）。木之活化 → 每分钟回血 +150 / 生命 +1600 / 防御 +50；**木之野蛮**消耗降至 10%、徒手攻击升至力量 ×4，且 **+8 附加「断骨」、+10 附加「碎甲」**——一记侧踢令敌同时承受击退 + 断骨 + 碎甲。正是这一脚（配 [动能战靴](../../../index.md) 超负荷）成了苏晓在竞技场击败 [荒焚](../../../人物/荒焚.md)、逼退 [狂狐](../../../人物/狂狐.md) 的杀招（「挨了这一脚战斗就不用继续了」）。
