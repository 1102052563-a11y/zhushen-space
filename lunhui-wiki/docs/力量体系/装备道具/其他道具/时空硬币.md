# 时空硬币（传说级 · 稀有道具 · 十六卷22章·完成隐藏任务「揍敌客家族的诅咒」A+ 四选一所得）

含时空之力的硬币（传说级·评分 **520**〔传说级 400~530〕·**仅 Lv.25 以上隐藏任务可得**·出售价「**???**」= 极高价的标志，类似当年的世界树指环）。效果：**可进入任意「试炼世界 / 战争世界 / 原生世界」**。[苏晓](../../../人物/苏晓.md) 在「永生之花果实 / 时空硬币 / 宗师之力(1/10) / 强制脱离豁免书」四选一中冒险选了它（宗师之力已得过、强制脱离他用不上、永生不诱人）。

!!! note "原生世界 = 野生的衍生世界"

    时空硬币能去的「**原生世界**」是苏晓最感兴趣的——它**不是轮回乐园的地盘、而是「野生的衍生世界」、从未有契约者去过**。未知即危险、却也意味着更大的机遇。是否把这枚「???」价的硬币卖给乐园（血赚）、还是留着去原生世界，苏晓视后续而定。

    - **非一次性消耗品（十七卷1章揭）**：十六卷70章苏晓嫌乐园派的 EVA（半崩溃）世界不适合自己，遂用时空硬币选「原生世界」——硬币爆开放出时空之力后**竟又重新聚拢、布满裂痕但还能再用几次**（原以为是一次性）。它带苏晓进了 [希亚大陆·灭法遗迹](../../../世界/任务世界/希亚大陆-灭法遗迹.md)（因价值远高于普通入场券，乐园按平衡机制补偿了 **16500 点世界功勋**）。
