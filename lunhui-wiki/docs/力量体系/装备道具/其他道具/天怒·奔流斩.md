# ⚡ 天怒·奔流斩（绝强级·主动 · 唯一特性）

⚠ **悬赏2·神速的奖励**（⚠ 唯一性·须持有者死后他人才可掌握）——⚠ **前置:绝强级 ＋ 可引高强度界雷 ＋ 生命 140 万＋ ＋ 雷抗 600＋**;⚠ **⚠ 效果:引下界雷 5 秒后，把界雷全部聚集到武器上进行一次超高强度攻击**;⚠ **三特性选一、永久固化**:

| 特性 | 效果 / 前置 |
| --- | --- |
| 裁决之域 | ⚠ 刺入地面引 15～30 次雷电冲击（⚠ 需地脉系 Lv.70·**未达成**） |
| 冥王 | ⚠ 突刺重炮·敌 <35% 生命有 10～25% 斩杀（⚠ 需雷系 Lv.80·**未达成**） |
| **刃宗师**（苏晓所选） | ⚠ **超高速不可阻挡突进＋下次攻击附带本次界雷 85% 雷电伤害＋斩击伤害**（⚠ 敌以武器格挡则附「破极效果」损毁其武器）·⚠ **此能力等级与刀术宗师同步并随之提升** |

- ⚠ **掌握代价**:⚠ **从 Lv.1 直接飙到 Lv.87 会把人送走**——⚠ **苏晓花 1500 盎司时空之力激活「禁锢权限」（三阶段）分段缓冲**;⚠ **⚠ 现 Lv.50·雷抗永久 +50→668**;⚠ **⚠ 战术:界雷落下前 5 秒是威力巅峰（苏晓靠高雷抗比敌人能扛），5 秒后将衰减时激活此招，把上百公里范围的界雷全聚到刀上、突进一刀奔流斩。**
