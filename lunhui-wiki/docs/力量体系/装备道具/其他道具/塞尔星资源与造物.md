# 🆕 塞尔星资源与造物（第45卷）

- **活性矿石**（塞尔星硬通货）：塞尔星独有资源，是维持 [移动要塞](../../../世界/任务世界/塞尔星-不败要塞.md) 运转 / 供其进化的能量源，也是全世界的**硬通币**（可去各大要塞城购物）。[猪头人](../../../世界/生物万族/index.md) 下矿采「粗矿」，须经洗炼、初步提纯才能被要塞吸收；要塞二次提纯得「活性能量」（可储存·随时凝成活性矿石）。产量惊人却被层层盘剥:末日要塞日产 223 公斤，八成上缴 [眷族同盟](../../../世界/任务世界/塞尔星-不败要塞.md)、再扣租赁 / 薪酬，头领净赚不到 4 公斤。
- **移动要塞**（活体·可进化）：半机械 / 半生物的**超巨型活体生命**——外表金属装甲层、下方探出几十条粗壮机械触角移动，内部有可居住空间，是抵御异兽 / 猎手 / 拾荒者的可移动家园。**有简单智慧、满足条件可进化成长**（吃活性矿石供能）。等级 T0～T5（数字越小越大）+ 三座眷族独有的「不败要塞」。凭「**要塞核心**」（一颗跳动的红宝石般心脏）操控——[苏晓](../../../人物/苏晓.md) 逼 [利·西尼威](../../../人物/塞尔星/利·西尼威.md) 交出末日要塞（T5·619 号）核心、成为新任指挥官，可下移动 / 停止 / 钻探 / 展开等指令。T5→T4 进化需大量活性矿石 + 去要塞城买「剧变型溶液」。
- **【源】石（湛蓝 · 召唤寄存物）**：[苏晓](../../../人物/苏晓.md) 兑现承诺给 [溺之首领·猎潮](../../../人物/联盟星/猎潮.md) 的天巴族圣物——因让它寄存在猎潮心脏内、令她以超越自身巅峰的实力被召唤出来，反被其吸入石内成了「寄宿者」。苏晓携此石可**召唤猎潮**（湛蓝水液涌出化肉体·带源弓与箭矢）；塞尔星再度召唤时，靠「带你回**源·神乡**（猎潮老家）」的威慑逼她现身。天巴族的蓝皮肤本非天生（幼年饮泡过源石的「源之水」到成年·近距离靠近源石才全蓝）——被召唤时源之力聚于体内、猎潮的皮肤会恢复她钟爱的白皙，这是她愿短暂合作的隐秘原因。
