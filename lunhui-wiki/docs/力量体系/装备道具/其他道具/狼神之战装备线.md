# 🆕 狼神之战装备线（第50卷35-44章）

- **无尽之环（满评分 6000 · +12 万生命）**：⚠ 苏晓原打算把它偷偷存进**海族秘宝器物库**（坐标由熔金魔提供、委托凯撒 5 万灵魂钱币搞定巡检的海族），让它慢慢吞光宝库、最后凭海族阵营商店的公证机制兑换出来。⚠ 狼神破封打乱了计划——苏晓改为向海族**明码开价**:「我这件来自深渊的器物，需要吞噬稀有武器、战甲等成长」「你们要考虑清楚，我是说，**把这无尽之环吞噬到极限为止**」。侏儒副官爽快答应（海族最不缺高阶装备），结果打脸来得飞快:「你拿出这名为【无尽之环】的东西，**已经吞噬了整整五个宝库的高品质武器、战甲，看情况，它好像还没怎么吃饱**」——苏晓直接挂断通讯继续冥想。最终评分拉满 **6000**，**+120000 点生命**，苏晓总生命达 **84 万**。⚠ 注意:随身或放**个人**储存空间均生效，放团队储存空间则无效。
- ⚠ **无尽之环的真面目**：苏晓推断「无尽」二字的真意不在吞噬——评分满后吞噬能力就消失了。真正的核心是减益**黑暗沉溺**:「预计 28~30 点深渊抗性可完全抵抗此减益」——⚠ 但很可能是**他的深渊抗性有多高，这装备的减益就有多强**;而更强的黑暗，又会衍生出更强的生命特性、给出更高的生命加成——**无尽循环**（正如简介所言:「黑暗，始终静谧，静谧衍生出沉淀，沉淀带来了腐朽，腐朽转化为养分，养分滋润了生命，在黑暗的最中心，孕育出了最纯净与茁壮的生命源质」）。**评分拉满不是成型，而是才刚激活它的真正特性。**
- ⚠ **剥离「增幅印记」**：苏晓察觉「缓慢吸收」这能力与整件装备格格不入——评分满后它增强 1.5~3 倍不假，但**吞噬效率也一并增强**:测试吞【黄金天平】碎片，苏晓预估能撑十几分钟，结果**只用了 5 秒**（「这一点都不慢的『缓慢吸收』能力」）。他遂放魔灵裹着精神力丝线探进环内黑暗，找到一枚以黑暗为能量维持的**多层复合式「增幅印记」**，用灵影线一拽——因是后加的、本就受排斥，「刚开始用力扯了下，后续是黑暗把印记给顶出来」。⚠ 剥离后无尽之环只剩「**黑暗**」与「**生命**」两种纯粹特性，增益感更自然温和，那种「黑暗随时会喷发」的不稳定感也消失了。苏晓推断:曾有位持有者想把它改造成「吞什么就有什么特性」的增强类装备，**增强搞出来了，却没遏制住吞噬性**。
- **神秘之眼（三种模式 · 起源级）**：⚠ 剥下的「增幅印记」运转模式竟与**神秘之眼**内部结构有共通性。苏晓照着它调节神秘之眼——第一次成功锁定后启动，**嘭的一声炸出强光**，围观的布布汪捂着狗眼在地上打滚、仙露露喵爪捂眼、巴哈惊呼「我淦」后煽翅撞吊灯上。⚠ 由此发现神秘之眼的三种用法:①**开锁模式** ②**爆闪模式**（埋伏或突袭时完全可以用来对敌）③**增幅模式**（照着「增幅印记」的样子调好后，整颗神秘之眼展开、化为金色、有了容纳感）。⚠ 增幅模式可**包裹一件装备并把其能力增益一个大等级**:包住【众神之眼】后，「气息感测」的判定等级**从圣灵级越过不朽级、直升起源级**（代价是每次使用都消耗众神之眼大量耐久，一个世界侦测五六次就得修——而里德不会修，让他修【狂猎之夜】已是极限，修这种超高科技造物「更可能是被邦邦两锤子给砸了」）;包住【伯格之心】则把「适应之力」抬到起源级（+85~100 点自适应抗性）。⚠ 局限:只能增益**装备**（不能增益道具），且最好选侦测这类可外放的主动效果;神秘之眼本身**没有装备槽**、无法佩戴。⚠ 苏晓一度动念「用增幅状态的神秘之眼包裹小体积的原罪物（比如幽冥骨戒）会怎样」——权衡后打消，一旁的布布汪、巴哈见他目光不再有莫名的神采，都暗中松了口气。
- **黄金圣杯（原罪物 · 神父持有）**：⚠ 神父这次**剥离失败**——并非不能强行剥离，而是代价是**他所有的神血都会在剥离途中被掠夺进杯里**。⚠ 它的特性极为特殊:**本身不会对持有者造成任何层面的损害**（不像深渊之罐会吞下历代持有者、也不像死灵之书会让人越读越疯狂畸变）。它的效果是:**持有者每杀死生灵，就会按其强弱从杯中涌出无比纯净的神血**（一种另类的祭献）。⚠ 真正的风险来自外界——**每饮一次杯中神血，持有者的「诡秘运势」就提升一些**:初期容易遇到游魂亡灵，强到一定程度就容易撞上恶灵、诡异物;⚠ 到了巅峰「可能早上出门摔个跟头都摔到空间夹缝里，然后误入漂游之间隙，转头一看，咦～，那不是**烛女**吗，它好像向我飘来了～」。为对付这些诡异存在只能更频繁地用圣杯、诡秘运势就更强、遇到的更可怕——**饮鸩止渴的死循环**，终点是漂游之间隙里那庞大的**茂生之狂乱**。⚠ 曾有位老哥硬是挺过了这一阶段、把烛女/茂生之狂乱/旧日之主见了个遍，正当他以为糟糕到极限时——他打了个喷嚏、再睁眼看到一个黑点、黑点变大又消失，等适应黑暗后，他发现自己周边漂流着**数之不清的不灭特性·深渊滋生物**:他被偶然出现的深渊通道吸进了**深渊的很深处**。⚠ 苏晓的「原罪物笔记」至此收录六件:**深渊之罐 / 死灵之书 / 灵魂王冠 / 幽冥骨戒 / 收藏家 / 黄金圣杯**——「后续应该不会再收录其他了，至少苏晓是这么希望的」。
- **银·月光套装（3/3 件 · 技法型 · 78000 声望）**：⚠ 苏晓攒了 81550 点声望换来的白银封箱（表面有淡淡血纹，似乎每次打开都需要鲜血）。⚠ 因长时间无人使用已进入**沉寂状态**，唤醒方式:①少量**银·月狼之血**（100%·但最后一只月狼死后已无纯粹月狼血）②其他狼血（20%~35%·且属性降 30% 以上）③**无上月光**（100%·作为灭法之影，苏晓能短时间驾驭**狼剑**、身处月光下即可凝聚）。⚠ 强开封箱只有 10% 成功率且属性永久降 50%。他借来海族王座上那把狼剑（狼神·希恩早年的），握上时剑身浮现青色又转瞬即逝——「**可惜，它死了**」，唤醒失败;还剩狼冢墓碑前那把。
- **安抚原罪物的晶质物（苏晓自研 · 第二杀手锏）**：⚠ 苏晓的自知之明:「据他所知，哪怕是先代灭法们，都选择不去主动招惹（原罪物），此等情况下，**想控制原罪物未免太过异想天开**」。于是退而求其次——把**黑枫树枝干**放入**高纯度深渊能量**中、再以他的药剂学催化，得到一种晶质物:对原罪物的**吸引力仅次于【原罪之芽】**。⚠ 一旦封印突然失效，关键时刻丢出几块，大概率能争取到重新布设封印的时间——这就够了。
- **界位借力 · 死灵之书当诱饵**：⚠ 乌鸦女在【封魔】刀鞘上留下的**因果系印记**（卢恩的厄运物·代价大、作用单一，⚠ **无视所有探查与感知、只有肉眼能看到，且只有实力弱于一定程度者才看得见**）——苏晓查遍能量波动/灵魂感测/显微寸镜都一无所获，最后是**仙露露**一句「你刀鞘上的是什么呀？新的纹饰吗？」点破的（女仆长也看得到;得知「是因为自己战斗力太弱才能看到」后，仙露露当场找布布汪「单挑！」，最后眼含泪水坐在墙角被女仆长安慰）。⚠ 苏晓请**女巫·莉莉亚**（因果系）把印记从【封魔】剥离、转贴到同为常用刀鞘的【**宝石誓约**】上（最大限度降低被发现动过手脚的概率），报酬是圣焰药师的**女巫特性秘药**;随后他把【宝石誓约】**送给了死灵之书**（附带条件:「至少一个月后再分解它」，另抛三块安抚晶质物）——⚠ 于是奥术永恒星的施法者们拿着追踪指环，一路追着**死灵之书**跑到了部落地盘南边的深渊侵蚀区，还煞有介事地分析「契约者去哪不能按常理理解」。「既然施法者们喜欢追踪，那就去追踪好了，**天知道死灵之书平常都去哪**。」
