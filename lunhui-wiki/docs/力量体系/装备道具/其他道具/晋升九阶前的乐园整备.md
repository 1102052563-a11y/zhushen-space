# 🆕 晋升九阶前的乐园整备（第47卷尾声·47–49章）

- **世界守卫者套装 =「世界三件套」**（不朽级·集 100 枚守卫者之证明兑换·独一）：由三件构成——**世界之眼 ×2**（魔眼·与眼球融合·情绪强烈波动时吸世界之力提升综合战力·可进化 3 次）、**世界猎手**（项坠·击杀影响世界安危之人后获世界之力）、**世界眷恋**（戒指·需魅力 150+·按魅力 30% 提升幸运）。集齐三件触发套装效果「**使命·世界**」:指定目标承受「天罚之怒」（三次大规模界雷·冷却 5 日·代价是耗 10 年量的本源生命力 = 减寿）。⚠ 苏晓魅力 -13 根本戴不了「世界眷恋」，故这套更适合发展世界之子——他拟选一名棋子放【吞噬者·黑A】共生 + 用【背叛者意志】化其为世界之子 + 再让他「无意间」得这套三件套，借以一探世界之力玄妙（吞噬者作保险、棋子反叛也不足惧）。
- **神圣唤醒**（荣誉商店兑换·为死寂城准备）：目标死亡 5 分钟内、躯体完整度 60% 以上，高概率将其从「假死状态」复生并愈合要害。⚠ 体力 200+ / 灵魂 145+ 者受致命伤后会进入 3~5 分钟「死亡状态」，此间可被救回。苏晓视其为「天助我也」的保命符。
- **龙血药剂**（八阶·荣誉商店）：最大生命 +15% / 肉体强度 +10%，持续 300 秒、无副作用。
- **铭文基座·怒像**（四铭文槽·已镶满）：以【冥帝铠甲碎片】…（误）——以冥界所得的四槽基座替换掉快爆的旧【铭文基座】，镶入 **黑暗**（主·生命 <25% 触发「黑暗余烬」30 秒·需意志力 120+）+ **生命** + **钢铁** + **烈阳**（受太阳照耀时体力 +10 / 肉体强度 +10.5%）四枚铭文。黑暗铭文（斩老骑士所得·1500++ 评分）与烈阳、钢铁（竞拍 18600 币）、生命铭文皆不俗。旧【铭文基座】的焚烬铭文一扣即爆、基座开裂，里德修不了，拟找**恶魔铁匠**。
- **枯声大剑**（世界级·苍龙大陆所得·曾被死寂侵蚀）：耐久仅 9/370。⚠ 其真正用途是「插在黑王座大陆·死寂城·至高圣所顶层某地」——这是**进入真正死寂城的钥匙**。苏晓一拳砸碎它、化成百余块碎片当「门票」（每块 100 币寄售），配合散布的「死寂城产黑枫树种」假消息（专属房间种的黑枫叶经死寂能量「增益失败」后产地隐藏为死寂城 + 死之民抓黑色树种的照片），引诱其他八阶强者一同去死寂城分担风险。
- **技法之魂·刃**（九阶物资）：能辅助刀术冲上 Lv.70 的收拢型特殊能量（对应「刃之投影」）。地精公司报价 15 万灵魂钱币。苏晓三宗师（刀术 / 近战 / 血枪）各需一种对应「技法之魂」才能破 Lv.70 关卡。
