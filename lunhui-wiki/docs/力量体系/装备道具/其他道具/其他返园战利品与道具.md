# 🎁 其他返园战利品与道具

- ⚠ **自我之力（拟态化身·100% 能力化身）**:⚠ **贪婪之章通关所得·⚠ 敌人击杀该化身会收到「完整、真实的击杀苏晓」提示+猩红宝箱（可开出【魂·王之加冕】等代表苏晓的装备）**——⚠ **能让神父这等老阴哔当场确信苏晓已死、翻盘神器。**
- ⚠ **贪婪之章（世界级·10 位魂灵具像）通关**:⚠ **苏晓开发噩梦血影代练→用【浓缩深渊源质结晶】强化成「灾厄血影」·打过神王·奥斯·托拜厄、十之位「噩梦·灭法之影」**（⚠ 通关后放弃持有权、同时解决封印浓缩深渊源质结晶的难题）。
- ⚠ **太阳圣徽（开永恒级宝箱·无上烈阳必得）**:⚠ **【古龙遗匙】＋【太阳圣徽】是开启埃伯亚思禁足塔、救白龙女、得古龙/太阳阵营全部遗产的唯一手段**;⚠ **⚠ 世界指针（[格林·吉莉安](../../../人物/格林·吉莉安.md) 遗留礼物·开雾地图·离世按开雾范围提升洞察力·给了贝妮）·⚠ 温暖阳光（艾什洛特天赋·治疗系太阳奇迹·太阳印记恢复生命）·⚠ 伪貌人偶（伪装装备·反骨到瑟瑟发抖·被原罪之书吓到忠诚度 92）·⚠ 黑暗蠕动（前代深渊之主脑髓·打开任何深渊封禁）。**
- ⚠ **狂猎之夜 → 质变**（⚠ 余烬之火吸收度 100%·防御力理论可排永恒级皮甲前三·里德修·修缮难度 +900%）·⚠ **坏笑的厄运（霸主装备·又一件缠上苏晓、熬到封印解除后主动同步契合、荣誉商店里 1 盎司甩卖·步浴血疯魔后尘）。**
