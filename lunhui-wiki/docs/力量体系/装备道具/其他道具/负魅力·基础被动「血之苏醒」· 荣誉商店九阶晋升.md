# 🆕 负魅力·基础被动「血之苏醒」· 荣誉商店九阶晋升（第48卷续10·续）

- **血之苏醒（负魅力·基础被动·史无前例）**：⚠ 苏晓委托凯撒拿【古老者卷轴】去众生之地·七层（海量高阶灵魂能量）**逆向扭变**【基础被动·秘术】，成功造出**史上首个「负魅力·基础被动」**（凯撒为此被罚丢进任务世界 + 长期禁入众生之地）。苏晓 −15 魅力 + 血气体系与负魅力契合 98%+，负魅力反成大助力。效果:①**杀敌概率恐惧周边敌人**（防 −35% / 移速 −60%·群战神技）；②血气系潜质阶位 +1；③血气贯穿 +3、血气伤害 +5%（惠及血烟炮 / 血枪 / 刃道刀·血影 / 血爆 / 血刃 / 血之兽 / 弑）。附带激发两大魅力特性:**负魅力 ×15 判定强制佩戴魅力装备**（耗耐久 25~30 倍、还会改其效果）+ **交涉威慑**（提升对方理智礼貌·魅力越低越强）。副产**开拓者纪念徽章**（+10 信誉度·卖凯撒）。
- **荣誉商店·九阶晋升**：⚠ 晋升九阶后八阶荣誉商店不再自动刷新——需以 1 枚钻石荣誉勋章换 1 枚**轮回荣誉勋章**（通用「荣誉勋章」·钻石勋章 100:1 转、余量转权限等级）完成全面晋升。九阶荣誉商店大变样:
    - **九阶补给药剂（六选二·每世界 2 份额）**：荣誉复苏（恢复 100% 生命法力 + 解战斗异常）/ 力量（真力 +8·350 秒）/ 秘银（真敏 +8 + 近战伤害阶位 +1·200 秒）/ 龙血（最大生命 +20% + 肉体 +15%·800 秒）/ 爆发（真智 +9 + 回蓝·140 秒）/ 灵魂（回灵魂能量）——苏晓选**复苏 + 龙血**（龙血 800 秒持久、且对战永生之神时正靠它保命）。
    - **灵魂补给箱**（★★=10 晶核 / 2 勋章、★=100 完整结晶 / 1 勋章·世界争夺战获胜刷新库存）、**基础被动技能卷轴宝箱**（5 勋章·力量 / 敏捷·苏晓已掌握体 / 智 / 魅 / 感知 / 灵魂五种）、**浴血疯魔**（霸主级·5 勋章）、**原罪物（伪）**（夜惑女巫公会仿制）。
    - ⚠**紧急援助（权限）**：追猎途中可用、5 秒内召来一名九阶猎杀者增援（增援者不可伤害被增援者）——5 勋章 + 80 盎司时空之力（指定阵营再加 5 勋章;可选猎杀者 / 战斗天使 / 先驱 / 死亡游侠 / 守护者 / 处刑者）。⚠ 六方里**猎杀者被选作增援次数最多**（实力受认可 + 不能攻击被增援者）、**请求增援次数最少**（苏晓一方库存已累积到满 999）。
