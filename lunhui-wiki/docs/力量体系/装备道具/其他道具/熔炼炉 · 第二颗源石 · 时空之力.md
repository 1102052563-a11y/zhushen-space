# 🆕 熔炼炉 · 第二颗源石 · 时空之力（第48卷续7）

- **熔炼炉（恶魔铁匠的旧物·锻造媒介）**：[恶魔铁匠](../../../人物/幽暗大陆/恶魔铁匠.md) 万年前在月神大陆亲手所造、后遗失、被古神越修越坏的锻造炉。苏晓「钓」来当见面礼奉还——恶魔铁匠遂免费为灭法锻造:用两颗邪神灵魂【神灵之灵魂·圣橡 / 始祖】打了一件新**刀鞘**（替换太费宝石的史诗级【宝石誓约】）、并提升霸主级装备**血羽**（代价另加一堆废装备 + 100 盎司时空之力）。⚠ 霸主级装备按 **3 / 5 / 10 精魄**分级，苏晓四件霸主里三件是 3 精魄、只有银月之刃是 5 精魄。
- **第二颗源石（圣歌团·剖心取出）**：战胜 [圣歌团](../../../人物/幽暗大陆/圣歌团.md) 后由圣心一剖心取出的源石——苏晓集齐五源石的第二块（第一块来自大主教）。同得**隐秘宝库钥匙** + **圣歌团印记**（凭此可在其势力范围自由通行）+ 10% 世界之源。
- **黑暗之源（祭祀坛·提天赋）**：死寂城内一种可提升**天赋等级**（如 8→12）的祭祀坛 / 源力，代价诡秘。
- **时空之力（封瓶·519.5 盎司）**：斩杀 / 驱逐 [龙神·迪恩](../../../人物/幽暗大陆/龙神·迪恩.md) 后掉落的一大批战利品之一——**519.5 盎司时空之力**（连同 25000 灵魂钱币、永生之神雕塑）。时空之力是苏晓跨世界运作、雇佣、兑换材料的硬通货。
- **纯白圣心·不朽项坠（五代圣女·治疗神器）**：治愈教会圣女一脉传承的顶级治疗神器（历五代圣女），死寂城线索指向的救治关键物之一（与被抓的「月光侍女」同为苏晓解死寂之毒、救回重伤者的凭仗）。
