# 🏆 黄金之力 · 黄金技能点的由来（52卷23章）

⚠ **黄金斗技场开场的古老宣言**:⚠ **「人们是何时发现的黄金力量，已无处寻觅……黄金的至高之力，可在极限之上，登峰最高极致，这是，仅次于「元素」与「深渊」两种「无上源质」之下的至高源质之力。」**

- ⚠ **⚠ 源质位阶总表（苏晓整理）**:

| 位阶 | 内容 |
| --- | --- |
| 1·**无上源质** | ⚠ **元素、深渊**（⚠ 仅此两种） |
| 2·**至高源质** | ⚠ **黄金之力**（黄金技能点的主材料）· **初始源质**（晋升至强的必须条件） |
| 3·通用源质 | ⚠ 力量源质 · 初始源质（既通用也至高）· 活力源质等 |
| 4·专属源质 | ⚠ 魔灵 / 辉光 / 黑暗 / 黄金源质（⚠ **金属系·与黄金之力无关**）/ 神圣 |
| 5·垃圾源质 | ⚠ **死魂源质**（⚠ 苏晓那份只卖了 80 盎司时空之力） |

- ⚠ **虚空之树掌握黄金之力**:⚠ **巨大黄金圆盘能将万界的黄金之力缓慢聚集**——⚠ **乐园阵营用设施将其质变为黄金技能点并储存**;⚠ **⚠ 初始条例明确规定不可凭空生成黄金之力（会影响万界体系）**;⚠ **⚠ 分配按世界争夺战战绩恒定——胜得越多，所得黄金之力比例越高**（⚠ 如轮回乐园）。
- ⚠ **⚠ 鹿神的例外**:⚠ **[鹿神](../../../人物/虚空/鹿神.md) 是虚空这么多纪元来首个能直接吸收黄金之力的个体**——⚠ **原因是曙光乐园被轰碎大半时，关于黄金之力的「转化核心碎片」被弱小时期的鹿神捡到、当超凡物给吞了**（⚠ 他真正天赋异禀的其实是消化能力）。
