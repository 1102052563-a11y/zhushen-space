# 🩸 血月女王（永恒级·唯一·评分 10200）

⚠ **由灵魂学院宝库进货的【暗月女王徽章】（永恒级·满评分·唯一·第一纪元光暗界·暗月国度）**融合【血魂之握】**而成**——⚠ **⚠ 「昔日仇敌，竟在相隔两个纪元后的今天，互相侵蚀」**（⚠ **暗月女王徽章与太阳阵营的血魂之握，是暗月与太阳两大宿敌的产物**）:

| 血月女王 | 说明 |
| --- | --- |
| **傲物** | ⚠ **至少占 3 个、至多占 5 个主装备位，占越多越强**（⚠ 当前占 4 个:项坠/戒指/护臂/挂饰·强度 120%） |
| **猩红血月** | ⚠ **立即消耗当前 95% 灵魂能量，转化为「本源生命力」——10 秒内最大生命值 +400%**（⚠ 此为「额外最大生命值」·⚠ 期间一切伤害只消耗额外血量·⚠ **最高可达 1000 万点**·⚠ 5 天冷却） |

- ⚠ **⚠ 它的用法**:⚠ **激活后血条前 1/5 鲜红、后 4/5 蓝色（伪血量）**——⚠ **临时增益的伪血量最大生命值没有任何风险与副作用**（⚠ **而永久提升生命上限会对人族体魄带来不可逆影响**）;⚠ **⚠ 它让苏晓能实打实硬抗自引的界雷**（⚠ 高雷抗＋额外血量·⚠ 且落雷后可在金色雷霆领域中冲过去捶被电麻的强敌·⚠ **「什么雷系基础/被动/操控能力一概没有，堆雷抗、堆血量就完事了」**）。
- ⚠ **⚠ 【暗月女王徽章】的祭献机制**:⚠ **女王欢悦（需献上带有足够多「你之因果」的旧装备累积祭献值）→ 女王之拥（侵蚀融合一件装备成「超极限·永恒级装备」·评分=（徽章评分＋祭献装备评分）×85%）**;⚠ **⚠ 苏晓祭献了杀戮本能/铭文基座·怒像/深海沉眠/狩环＋挽歌/灵性之刃/太阳之环/**狼血·月饰**（世界级挂饰·超强度共鸣、几乎自动祭献——⚠ 银月狼起源自光暗界、祖先追随过暗月女王）达到 100%，融合血魂之握得血月女王。**
