# 猎魔套装（暗紫 · 三件套 · 八卷80章·乐园大事件核心）

一套**只有三件**的**暗紫色套装**——**近战的「梦幻套装」**。「暗紫色套装」与「单件暗紫装备」是两个概念、不可同日而语（套装有整套加成）。八卷80章它引爆了一场**乐园级大战**：十几个大型冒险团（[神皇](../../../人物/神皇.md)、[幻影旅团](../../../势力/index.md)、血门、黑帆商会、蛮天等）为争抢它互相厮杀——其中**一枚戒指落在神皇冒险团手里**、旅团第 6 位 [咕噜](../../../人物/咕噜.md) 没抢到、却在冲突中**杀了 [茜拉](../../../人物/茜拉.md)**，直接点燃**神皇冒险团 vs 幻影旅团的「不死不休」之战**。这块「烫手山药」苏晓暂不打算掺和。**九卷83章·苏晓夺得「猎魔之戒（套装 1/3）」**：木叶被夷平后，重伤的 [咕噜](../../../人物/咕噜.md) 把猎魔戒抛向九尾战场想借刀害苏晓，被苏晓**界断线极限一钩夺回**。**十四卷43章·三件套集齐**：第三件「狂猎之裁决」（手枪）在 [恶魔古堡](../../../世界/任务世界/恶魔古堡游戏.md) 藏宝库选物时被苏晓认出取得——三件全凭抢 / 战争世界赏 / 宝库选，无一开自宝箱（神皇等四阶倾团抢都没凑齐）。**第三套装效果由此揭晓 = 可晋升金色套装**（见下「狂猎之裁决」）。

!!! note "猎魔之戒（套装 1/3 · 暗紫 · 评分 260）"

    - **单件效果**：抗拒（被动·减免 30% 异常状态持续时间）；需力 40 / 敏 40。
    - **套装效果（集齐 3 件激活）**：①猎魔人（力 +15 / 敏 +10 / 体 +10）②近战达人（近战伤害 +40% / 最大生命 +20%）③第三效果不可见。**近战梦幻套装**——苏晓终于到手一件，今后会留意另两件下落。

!!! note "猎魔之证明（套装 2/3 · 暗紫 · 评分 260 · 十二卷49章）"

    胸针款式名牌（极稀有·产地任何 Lv.20 以上世界·需力 40 / 敏 40）——苏晓**苦寻多时、在交易市场遍寻无果**的第二件猎魔套装，竟从军需官 [尼古拉斯·凯撒](../../../人物/尼古拉斯·凯撒.md) 手里得来（凯撒不识货·当报酬给了帮卡洛斯治病的苏晓·疑是某契约者死在本世界后掉落）。

    - **单件效果**：不屈（被动·生命值低于 10% 后**延迟 10 秒才进入濒死状态**·冷却 48h）——续命保命。
    - 集齐第二件后离激活「猎魔人 + 近战达人」整套加成更近一步（只差第三件）。

!!! note "狂猎之裁决（套装 3/3 · 暗紫 · 评分 260 · 十四卷43章 · 集齐！）"

    类燧发枪外形的近战手枪（约 35cm·攻 10~153·容弹 5·需力 40 / 敏 40）——**苏晓在 [恶魔古堡](../../../世界/任务世界/恶魔古堡游戏.md) 第四场积分第一、于爱丽丝藏宝库（158 万件）里凭枪上花纹与猎魔套相似而认出**（放弃了拳头大、值几十万币的太阳残骸），由此**集齐三件套**。

    - **① 特殊弹药（被动）**：只能用专属弹药——透明壳内充黄色高爆液 + 一根「**爆炸针**」。
    - **② 死刑爆炸针（主动 / 被动·无冷却无消耗）**：命中后弹内液体爆炸**削敌 60% 全部防御 0.19 秒**、爆炸针借动能近乎不可躲地刺入；可随时引爆（不引爆 40 分钟失效·强行挖出受最大生命 10% 恒定伤），每针 80~100 伤。**五针同时引爆 = 500 伤 + 易损状态**（肉体防御 −35% / 治疗 −20% / 移速 −10%·10 分钟）——苏晓评：易损叠 [气息外放](../../技能/index.md) 的 −15 防御，他一刀下去「酸爽无法描述」（近战枪械·20 米内 + 枪械精通 Lv.30 五枪不难）。
    - **套装第三效果终现 · 「猎魔的战利品」**：每场战斗判定为「猎魔状态」、杀敌攒「战利品数值」——**满 100% + 消耗灵魂结晶（大）×5 + 诞生之源（金色）×3，三件套即晋升为金色品质套装**（能力替换 / 增强、偏近战）。这正是 [神皇](../../../人物/神皇.md) / [斯坦](../../../人物/斯坦.md) 等四阶契约者不惜出动整个冒险团抢猎魔套的真正原因——**三件套的金色套装、对四阶亦可遇不可求**，而晋升条件并不算苛刻（杀敌是苏晓日常·诞生之源 = 熔炼金装必得·10~30 万币可买）。苏晓由此成了猎魔套的众矢之的（连五阶都未必抵得住诱惑）。
    - **可惜**：因苏晓尚未突破 80 点属性壁障，「猎魔人」的三属性增幅暂时无效；但「近战达人」的 **近战伤害 +40% / 最大生命 +20%** 已让他近战上一台阶（生命比主坦还高）。激活时他**摘下 [火之意志](../../../index.md)**（与 [心灵之刃](../../技能/index.md) 能力冲突·后者更强）、把狂猎之裁决定为副武器。

!!! success "猎魔之王（金色套装 · 3/3 · 十六卷64–66章 · 晋升成功！）"

    十六卷全职猎人 + 火影两世界的惨烈厮杀让**战利品数值满 100%**（千手柱间 + 蚁王两战占 28%·柱间大半）。苏晓凑齐**灵魂结晶〔大〕×5 + 诞生之源 ×3**（向职工者「机械妹」6 万 / 颗买），租 B 级锻造熔炉，以 **76% 成功率**（24% 失败将损 5 大结晶 + 19.2 万币）赌成——猎魔套晋升金色套装「**猎魔之王**」（评分 400+）。形态特殊：三件套熔成**一枚可三形态切换**的装备（戒 / 项坠 / 枪械），占一个戒栏或项栏即享全部加成。

    - **戒形态**：减免 40% 异常状态持续时间；**项坠形态**：生命 <10% 后延时 20 秒才进濒死；**枪械形态**（攻 133~153·容弹 5·杀敌生子弹）：血弹命中削敌防御、**一枪一层削弱**（叠 5 层 = 身体防御 −40% / 治疗 −30% / 移速 −15%·完爆旧「狂猎之裁决」的五枪引爆）。镶 [海之心](../../index.md)（法力 +400 / 恢复 +36）。
    - **套装效果（任何形态恒激活）**：①真实之力——**真实力 +5 / 真实敏 +4 / 真实体 +3**（晋金色后终于改回增真实属性）②女巫噩梦——近战伤害 +60% / 最大生命 +35% / 异常抗性 +28 ③**猎魔时刻**（充能型大招）：无战斗时溢出能量储入·满 **6000 法力**可开启、**100 秒内一切自身能力〔含龙影闪 / 反击盾 / 青钢影〕进入无冷却、无消耗状态**（不含装备附带技能·龙影闪因斩龙闪伴生算「自身能力」纳入）·冷却 **10 自然日**——苏晓现阶段的翻盘大招。
    - 猎魔套**对单人的加成完爆飞升套**、且高阶也无须放弃（只占一栏）。三件金套同强需 23 万币 / 次（默认三件齐强），苏晓暂未强化。
    - **首次实战（十七卷32章·斩死刑官 [敦克·尼克](../../../人物/敦克·尼克.md)）**：开「猎魔时刻」后，苏晓在 **0.7 秒内连用 12 次龙影闪 + 多层反击盾**、神出鬼没三刀斩首强于自己的领主级死灵。心得：**「无消耗」只是不耗能量、不代表不耗身体**——龙影闪连续穿透空间负担成倍攀升，以他**体力 87**（含猎魔之王 +3 真实体力）连穿 **5 次**即近脱力、须歇 1~3 秒（体力越高负担越小·体力 <30 穿一次就粉身碎骨）。猎魔时刻是他现阶段对**强于自己的敌人**的翻盘底牌（「不开就会死」）。
