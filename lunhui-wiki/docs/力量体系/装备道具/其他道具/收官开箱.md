# 🎁 收官开箱（50卷63–64章 · 六枚宝箱）

⚠ 【**深渊宝箱**】除外（不加持幸运、单独先开，⚠ **结果开出收藏家**），其余均在【**命运主宰**】的气运加持下开启。

- **【兽魂宝箱】**：⚠ **永恒命源 ×1**（**4200 点评分**·⚠ **风海大陆的独有产出，评分为 3000～5200 点**）+ **厄蛇之牙**（不朽级武器 / 材料）。
- **【法魂宝箱】**（击杀 [古亚院长](../../../人物/风海大陆/古亚院长.md) 所得）：⚠ **金色光芒盛放，闪了！**——**原初碎片 ×1** + **晚宴邀请函 ×2** + **巨人之国·10 号宝库钥匙** + ⚠ **重力法环（永恒级装备）**（⚠ **处于未激活状态，要到装备鉴定所花 15000 枚灵魂钱币激活才能看到属性**；⚠ **因获得永恒级装备，相关权限已自行向苏晓开放**）。
- **【永恒级宝箱】**（老兽王掉落）：⚠ **深渊战枪（深渊武器）**——⚠ **因猎杀者无相关技法能力，无法使用此武器，此武器已自行封存**。⚠ **每人毕生最多只能选择一次「深渊武器」**，一旦灵魂绑定**将无法以任何方式解除**；⚠ **检核到你为灭法之影，你无法持有「深渊武器」** → ⚠ **平衡机制**:可通过**向轮回乐园提交「深渊武器」**的方式，**获得价值等量的「魔灵能量」**。⚠ **当累积获得的「魔灵能量」达到或超过 100% 后可要求免费封装**，⚠ **打开封装时务必以最快速度用「灭法之刃」将其吸收，否则这些魔灵能量将会脱离掌控**。⚠ 苏晓提交后，**「魔灵能量」的累计槽达到 81.5% 后停止**——⚠ **再获得一把「深渊武器」，就能提取出一份「魔灵能量」了**。
- **【狼血宝箱】**：⚠ **传承狼血**（血统 / 职业类物品·**待激活**）——一个五公分粗、十几公分长的封瓶，⚠ **里面的月光之力格外浓郁，不用想都知道，这是月狼的力量传承**。⚠ 让 [布布汪](../../../人物/布布汪.md) 传承看似可行**其实不适合**:⚠ **布布的准则是只要开心、欢乐就行**，跟着苏晓四处逛、饿了就吃、困了就睡、睡醒后有正事做正事、没正事就玩会游戏——⚠ **让它去传承月狼的力量的确不适合**（「试想一下，**布布汪叼着狼剑、然后狗脸上浮现表情包般慌得一批的神情**，那画面，**属实让人啼笑皆非**」）。
- **【秘宝之盒】**：⚠ **曙光碎片（纪念品）**——⚠ **苏晓被【曙光碎片】所支配**。⚠ 「这宝箱在荣誉商店内 **500 盎司时空之力一枚**，能开出技能升级仓免费使用权限、原初碎片 1～3 块、满评分·世界级装备，乃至于本源果实——⚠ **可苏晓不太想兑换这宝箱，因为他感觉以 500 盎司时空之力买块【曙光碎片】，简直亏到吐血**。」
- ⚠ **总评**:「这次开宝箱**收益很大**——**法魂宝箱闪光、老兽王宝箱开出「深渊武器」、月狼宝箱更是夸张到开出月狼最后的力量传承**；⚠ **与之相对，到最后开【秘宝之盒】时，苏晓被【曙光碎片】所支配**。」
