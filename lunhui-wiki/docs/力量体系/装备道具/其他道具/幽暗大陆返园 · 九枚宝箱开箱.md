# 🆕 幽暗大陆返园 · 九枚宝箱开箱（第48卷续10 · 乐园休整）

⚠ 幽暗大陆通关返园后，[苏晓](../../../人物/苏晓.md) 一口气开了九枚宝箱（扭变的深渊 / 暗之物 / 滑腻组织球 / 苦难尽头 / 不死之虫 / 深渊 / 原罪之盒 / 世界级 / 起源级·永生）:

- **暗之吞噬（不朽级·特殊装备·评分 1500）**：⚠ 前五枚宝箱**同开时产出融合**（比「闪光」更小概率）——【暗之物】+【滑腻组织球】因特性同步度极高，融合成这件装备。前置体力 240 + 意志 160;效果「缓慢吞噬」黑暗特性之物;减益「反噬」每次使用有概率掉魅力（苏晓 −15 魅力已不在乎）。作用单一却因此极强。
- **灰烬 / 尖哮 / 万虫**：灰烬（深渊·洒出可探地面是否有「原罪物」或「深渊滋生物」经过）、尖哮（不朽级饰品）、万虫（灵魂系技能卷轴·待售）。
- **神教徽章（消耗型·可用 3 次）**：⚠ 出自【原罪之盒】。前置敏 260 + 智 260 + 灵魂 300;效果「援助」——各 50% 概率召来**圣歌团**（随召唤者真实智力 + 灵魂强度定战力·灵魂 >500 召出的队伍含**圣心一**）或**狼骑士队**（随真实敏捷 + 灵魂强度定战力·灵魂 >630 含**狼骑士队长**）。不能连续激活、不能同时召两队。同盒还开出塞满的灵魂钱袋（200~15000）、贪婪钱袋（−13000~53000·高风险高上限）。
- **神圣蜡烛（一次性）/ 世界之核残片 ×2 / 时空之力 130 盎司**：出自【世界级宝箱】。神圣蜡烛点燃驱散周边 5 米黑暗邪祟、形成庇护区至烛灭。
- **永恒权杖（起源级·斩永生之神所得·未唤醒）**：⚠ 神灵时代神教邀**虚空 / 风海大陆 / 古龙国度**的三位传说铁匠合造。耐久 5380/8600;极重——苏晓 **276 真实力量都拿不起**（靠储存空间公证才提得动）;唤醒方法需 350 盎司时空之力咨询。苏晓打算「时机恰当从储存空间放出来就能把敌人砸出个好歹」。
- **深渊宝箱（狼骑士队长掉）**：开出一件黑雾漩涡状的「爹级」深渊器物——苏晓不碰、封进炭盒锁进空房间。
