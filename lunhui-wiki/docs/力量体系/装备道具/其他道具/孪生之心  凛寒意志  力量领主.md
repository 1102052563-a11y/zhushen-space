# 💍 孪生之心 / 凛寒意志 / 力量领主

| 装备 | 说明 |
| --- | --- |
| **孪生之心**（永恒级·满评分 6000·永恒级宝箱·双生） | ⚠ **⚠ 佩戴后无法再佩戴任何饰品**（戒指/指环/项坠/徽章·⚠ 绝对优先权）·⚠ **光/暗互转**:光系可全部转暗系，暗系可全部转光系·⚠ 所有暗系能力附「暗物质」（减速＋全抗性降低）· 所有光系能力附「光焰」（⚠ **可引燃暗物质造成真实焰燃伤害＋持续降低敌人身体属性**）·⚠ 简介:「生于黑暗，向往光明，然，经世间的欲恨哀愤沁染，终又堕于黑暗。」 |
| **凛寒意志**（霸主级·**评分 6000＋＋＋＋＋**·永恒级宝箱·古龙神） | ⚠ 产地=古龙国度·永封冰原老龙的灵魂·⚠ **古龙阵营专属**（苏晓凭「誓约之徽·白龙」可用）·⚠ **护体凛寒**:为自己与周边 1000 米内友方加成光环，**承受的烈阳系伤害降低 50%**（含真实烈阳伤害·阵营相互克制特性）·⚠ **⚠ 显然是为「烈阳星·奇利亚德」准备的** |
| **力量领主**（永恒级·满评分·狩猎腰带·**滋养类**） | ⚠ 产地=虚空·银星环·厄釜莱罗（第二纪元最强工匠的得意之作·原是风海大陆兽王所订购，兽王却在交付前死于海族诡影刺客的暗杀）·⚠ **需真实力量 800 点**（苏晓靠【浴血疯魔】临时 +50% 卡上去）·⚠ **力量碎片**:每 5 天生成一块（共 6 块）——红＝真实力量／黑＝近战攻击力／⚠ **金＝掌握「泯灭之力」**（只能吸一块）·⚠ **⚠ 泯灭之力把踢技基础攻击力公式从（力×0.8＋敏×0.4）改为（力×0.6＋敏×0.6＋体×0.6＋灵魂强度×0.3）**——⚠ **苏晓的 775 点踢技基础攻击力将直接飙到 1490 点**（⚠ 而这只是最下层的基础攻击力·⚠ **滋养类装备本是至强中后期才有财力与体魄使用的东西**·⚠ 耐久无法修复） |
