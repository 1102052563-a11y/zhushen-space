# 🐿 嘟嘟咕咕的秘密任务（完成）

⚠ **三块石块的最终答案，全都离谱得超出这只小家伙的想象**:

| 石块 | 真身 |
|---|---|
| **黑石块** | ⚠ 深渊族裔·都卡因家族「始源族徽」最核心的一块 |
| **蓝石块** | ⚠ 众神「起源法典」封皮上蓝色核心碎裂后剩余的核心部分 |
| **红石块** | ⚠ **陨灭星前「世界之核」**——容纳99.9%古神始源力量后，在冥神身死、此物失去主体后凝聚而成 |

⚠ **⚠ 若说嘟嘟咕咕原本预期的「红石块」珍贵程度是8000，那苏晓拿出的这块保守估计有300万以上**——⚠ **它当场陷入难题:最心爱的珍藏宝物已不足以支付本次答谢**;⚠ **⚠ 于是它向虚空之树发出请求**——⚠ **凭借世代传承所累积的 **1485346点** 虚空之树信誉度，请求通过**;⚠ **⚠ 结果:苏晓后续触发的嘟嘟咕咕交易将**永久性增加一种自选的「星界珍宝」****——⚠ **他选了【星辰辉石】，嘟嘟咕咕为此消耗了 **1082640点** 虚空之树信誉度**（⚠ 公证完成后它疲惫到所有小骨手都无力垂下，却也特别高兴——**它终于知道该怎么回报这远超它梦寐以求的三石块**·⚠ 提示还特别说明:**嘟嘟咕咕不会因深厚友谊而放宽交易条件——这将直接导致它的存在程度降低；绝对尊重「交易准则」是其存在的基础规则**·⚠ 几只小骨手还学会了用食指和拇指捏在一起**向苏晓比心**）。
