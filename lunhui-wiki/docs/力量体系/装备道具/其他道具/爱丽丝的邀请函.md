# 爱丽丝的邀请函（品级未知 · 十一卷50章·开蝶猩红卡所得）

布布汪从 [蝶](../../../人物/蝶.md) 的猩红卡里抽出的一封**纯黑信封、火漆封口**的邀请函——当初产地 / 品质 / 效果 / 价格**全部「未知」**（权限不足 + 条件未激活·看不了），简介仅「邀请你进入……」。

**谜底揭晓（十二卷101章）**：原来它是「**爱丽丝恶魔古堡游戏**」的邀请函——这是获取「[强者之证明](../../属性.md)」（80 点壁障最后一关）的三条路之一，**须属性达 80 + 突破身体极限才能激活查看**。蝶生前正是达 80 点 + 突破身体极限、花 20 万币 + 灵魂结晶〔中〕×15 兑了它。[苏晓](../../../人物/苏晓.md) 突破身体极限后激活、得知其效果（用后等 2~3 天、凑满 7 名契约者即开「既有趣又绝望」的恶魔游戏·通关即为强者），遂**用掉它**——比起挑战剧情生物（贵）或黑渊历练（生存率 4~6%），这是折中的最优解。
