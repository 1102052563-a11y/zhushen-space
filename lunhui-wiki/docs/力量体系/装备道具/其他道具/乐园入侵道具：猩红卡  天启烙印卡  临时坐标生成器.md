# 乐园入侵道具：猩红卡 / 天启烙印卡 / 临时坐标生成器（二十九卷·入侵天启）

[苏晓](../../../人物/苏晓.md) 晋六阶考核「[入侵天启乐园](../../../世界/任务世界/塞壬星-入侵天启.md)」中的专用物：

- **猩红卡（★~★★★★★）**：入侵者杀敌方契约者几率掉落，品质按敌人实力定（杀普通天启契约者 ★★★、杀 [战斗天使维克](../../../人物/塞壬星群像.md) 得 ★★★★★）。
- **天启烙印卡（天蓝色·€符号）**：从天启契约者身上「剥离」的烙印（€ = 天启乐园·ф = 轮回乐园）——持有可临时伪装成天启契约者、进「契约者营地」、用乐园积分（仅限飞船内）。剥离须肢体接触后 64 小时（1 公里内）。
- **临时坐标生成器**：30 cm 金属管（满布空间秘纹）——竖地展开放红光、约 1 小时生成「临时分支空间坐标」，让轮回乐园借此掠夺契约者营地资源。配套金属板贴 [黑·王之陨祭](../../../index.md) 护臂用于无痕剥离烙印。
