# 🗡 晚宴邀请函（稀有物品）

⚠ **35 个自然日后，幽冥之地的暴食城堡内将举办一场晚宴，总计邀请了 20 名食客**——凭此函可进入城堡并成为食客之一。⚠ **暴食城堡为中立区域，在虚空之树的公证下，不可在此动用武力，或妄图伤及他人等。** ⚠ 简介:「**享用美食的同时，还可壮大自身的本源力量，因此所有客人都会准时到场。**」

- ⚠ 苏晓的目光环顾布布汪、阿姆、巴哈、贝妮——⚠ **下一秒，布布汪、巴哈、贝妮都指向阿姆**。⚠ 但苏晓感觉这样做**不太赚**:⚠ **这两张可以考虑给 [夏](../../../人物/夏.md) 一张**——⚠ **以夏的厨艺水平，有不低的概率可以通过品尝晚宴的美食，从而推演出这些美食的菜谱**。⚠ 「苏晓去过很多世界、各类美食都品尝过，⚠ **但这些美食中最顶尖的，也比夏差上一线**」（⚠ **这并非掺杂了个人感官，而是苏晓队所有成员的综合评判**）；⚠ 从七阶初期开始苏晓经常获得**强大野兽 boss 的腿肉、邪神心**等特殊食材，⚠ **夏最初有些不太适应，但后来习惯了——哪怕是兽形邪神里脊肉摆在她面前，她构思一会也能烹饪出一道美味佳肴来**。⚠ **一顿饱和顿顿饱的区别，谁都能分得清。**
- ⚠ 剩余一张:苏晓、阿姆、巴哈抽签（**布布汪和贝妮没战力、应对不了突发情况，不能和夏一起去**——虽说是中立安全区，**但也不能半点防范都没有**）。⚠ **最终巴哈抽到签，但它以 35 天后会突发肠胃炎为由死活不去**，⚠ **最终在巴哈的多次坚持下，这张由苏晓持有**。⚠ 夏**很痛快就答应**，并直言:「**别说白给她邀请函，就算花钱买这邀请函，她都愿意——那可是暴食城堡**。」（⚠ 单纯去享用晚餐没风险，**可如果想推演出这些美食的菜谱就有些风险了**）
