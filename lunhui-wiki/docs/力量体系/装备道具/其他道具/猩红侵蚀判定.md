# 🩸 猩红侵蚀判定

⚠ **苏晓单手探入拱门的高浓度猩红时**:⚠ 【**猩红之力与青钢影能量进行本轮判定**】——⚠ **青钢影能量优先级 10，猩红之力优先级 9.69**;⚠ **因是承受侵蚀，效果相对被动**——⚠ 【**生命值上限从 100% 滑落至 99.8%，持续 5 个自然日**（⚠ 猩红侵蚀持续的最短时限）】。

- ⚠ **⚠ 好消息**:⚠ **就算是猩红之力，也不能让他永久性损失本源生命力**——⚠ **所以这减益是临时的**（⚠ 有别于「红眼战士」的箭——⚠ **那会永久降低生命值上限，治疗无效**）。
- ⚠ **黑暗与猩红的关系**:⚠ **不是克制，而是「不相容」**——⚠ **更多的黑暗可以包裹住猩红，而猩红更多时则能侵蚀掉黑暗、让其化为猩红**;⚠ **深渊能量的本质特性之一就是黑暗，由深渊扭变出的猩红被黑暗包裹，实属正常**（⚠ 城主夫人被黑暗区封困、治疗所用黑虫抵抗猩红，都是这个道理）。
