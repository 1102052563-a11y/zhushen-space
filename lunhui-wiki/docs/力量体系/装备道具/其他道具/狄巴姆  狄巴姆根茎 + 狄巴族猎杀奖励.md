# 狄巴姆 / 狄巴姆根茎 + 狄巴族猎杀奖励（二十九卷44–48章·起源之岛）

- **超凡植物·狄巴姆（被污染世界之核·晋升最终环目标）**：污染塞壬星世界之核的超凡植物（本体在 [起源之岛](../../../世界/生物万族/index.md)·与世界之核伴生不可剥离）。「圣物仪式」植入其圣物：成功成狄巴族、失败成不死者。
- **狄巴姆根茎（史诗级·永久增益道具）**：狄巴族猎杀奖励之一——经轮回乐园提纯后可食用，**永久 +150 生命值 + 5 身体防御力**（按体质每人限 3~5 截）。苏晓吃 4 截 +600 生命 / +20 防御；随从分食（**布布汪吃一根的提升顶巴哈三根**·460 生命 / 根）。
- **焚血之刃 / 病毒阻断体质（临时被动·狄巴族猎杀奖励）**：**焚血之刃**（对狄巴族伤害 +5%~30%·苏晓拉满）/ **病毒阻断体质**（每次承受减 10~45 真实病毒伤）——离开本世界即消失的临时强化。狄巴族不掉宝箱、直接掉材料（完美感染之血等传说级）。
- **狄巴姆萃取原液（永久强化药剂·弑库斯卡奖励）**：斩 [霸主级库斯卡](../../../世界/生物万族/index.md) 用灵魂钱币换的永久强化药——**永久 +35% 神经反射速度 + 3000 生命值 + 96 异常属性抗性**（苏晓直接饮用）。同期还得狄巴姆种子（超高活性·可栽植）+ 霸主精魄。
