# 🏅 深蓝之影（九星 · 主称号）

⚠ **苏晓斩杀始祖所得的 526000 点世界声望，合成出 5 枚八星无属性称号**——⚠ **加上【凋零的世界意志】，把主称号【深蓝之影】提升到九星。**

| 【深蓝之影】★★★★★★★★★ | 效果 |
| --- | --- |
| 适应性之力（被动） | ⚠ **所有伤害阶位 +1** |
| 深蓝（核心·被动） | ⚠ **身体速度整体提升 29.75%**（⚠ 基础 10% ＋ 真实敏捷 ×5%·高优先权——⚠ 绝强的苏晓佩戴后有将近三分之一的速度提升） |
| 怒影（核心·被动） | ⚠ **对抗灭世级存在与深渊系敌人时，额外造成 10% 真实伤害** |

- ⚠ **九星≠极限**:⚠ **九星对应的是九阶～至强级，所以九阶称号间的差距特别大**——⚠ **九星称号可通过 5 枚九星副称号「极限燃炼」，只提升强度、不再提升星级**;⚠ **一次极限燃炼后成功率降到 80%，二次会导致三次跌到 20%，四次连 5% 都没有**——⚠ **⚠ 迄今为止没人能把九星称号极限燃炼五次。**
- ⚠ **苏晓准备以时空之力收购九星无属性称号，对【深蓝之影】做一次极限燃炼**（⚠ 本世界危险度让称号宝箱能开出可交易称号，对 −21 魅力的他是难得机会）。
