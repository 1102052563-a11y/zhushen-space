# 交易背后

- 【深渊宝石】在**黑暗温床**手中（【通晓之眼】给出的答案是「深渊、温床、母亲」）。苏晓以领主列车尾车厢展架上的收藏品交换——月之骨（战胜的第一名古神·月神）／王冠碎片（黑之王）／断裂枪刃（铁羽王）／船舵残骸（厄运号）／枯萎宝石（永光世界始祖额上的红宝石）／兽王冠（风海大陆老兽王）／女巫项坠（月女巫的礼物）／暗金徽章（需达成黄金阵营·永久仇恨）／旧日残阳（烈阳星「旧日」落下时的太阳核心碎片）／超脱余烬（众神界超脱向万族界所产生的大块余烬）／陨灭星·世界之核碎片（存留碎片中最大块）／**黑暗雕像（无价值·稀有物:上一任深渊之主的形象）**。⚠ 苏晓取下王冠碎片、断裂枪刃、船舵残骸、女巫项坠留下不换；**温床最终选中的，恰恰是那尊「无价值」的黑暗雕像**。
- 【元素宝石】在虚空之树藏库外（【通晓之眼】的答案是符号「ψ」）。苏晓自己申请被「信誉度过低且此物无法交易买卖」驳回，改由 **莫蕾** 出面以「灵魂宝石交换元素宝石」申请成功，换置费 50100 盎司。
