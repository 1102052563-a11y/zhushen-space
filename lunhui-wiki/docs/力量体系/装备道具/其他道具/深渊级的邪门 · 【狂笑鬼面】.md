# 🎭 深渊级的邪门 · 【狂笑鬼面】

⚠ **深渊之底产·寄生面具·使用需求「与此面具对视5秒，期间不发出狂笑」**;⚠ **⚠ 苏晓的凝视测试**:⚠ **第一次撑到2秒眼角微不可见地抽动、第二次2秒后面部开始发麻**——⚠ **他把它丢进「原罪之书」第八页关了两分钟**（⚠ 面具从狂笑逐渐变得笑不出来、开始有几分慌张——**能坚持这么久已很罕见，当初的永暗女皇进去几秒就戴上了痛苦面具**·⚠ 再取出凝视，5秒轻松通过——**它决定姑且屈服，心想不过是区区人族，等几个纪元这人族老死了，最终获胜的必然是它**·⚠ 此前它已成功害死过**35位**持有者）;⚠ **⚠ 来历**:⚠ **经茂生之狂乱确认，它是一位**上古时代·星界异存在**的遗留，那位异存在死于**元素的侵袭**——直指终结上古时代的「元素浩劫」**（⚠ 苏晓以半页「树生之页」作为解答的报酬）;⚠ **⚠ 深度唤醒**:⚠ **丢进液态深渊能量中——质地变软、生长成能完全笼罩面部并蔓延至耳后的半个头套，评分从18500飙至24680**:

| 效果 | 内容 |
|---|---|
| 梦境寄生（主动） | ⚠ **被他人的因果/气息/血迹沾染后侵入「梦界」，在目标睡眠时悄然完成面部寄生**——⚠ **完成后与被寄生者的面皮进行**因果级融合**;⚠ 唯一剥离方式:永久舍弃自身一种因果存在的同时，剥下自己的整张面孔 |
| 寄生中（被动） | ⚠ 耐久恢复到50点=初步影响潜意识选择 · 200点=可一定程度蛊惑 · **500点=可发动「狰狞狂笑」** |
| 狰狞狂笑（主动） | ⚠ **让周边5000米内所有个体强制狂笑至少3秒**——⚠ **完成本次狂笑后，「鬼面」将彻底取代所寄生者**;⚠ 完全晋升状态:每10秒可让500米内任意目标狂笑0.5～300秒，**超过5秒后每秒承受一次即死判定** |
| 狂笑因果污染 | ⚠ **狂笑具备传染性**——⚠ **所有看到/听到/以任何感知方式感受到狂笑的目标都会被传染，且新感染者同样具备强传染性**;⚠ 所有因狂笑而死者的尸体会被鬼面控制、可再度发出狂笑扩散污染 |
| 复苏厄难（核心·被动） | ⚠ **每彻底寄生一个目标，鬼面复苏一分，且**持有者获得一颗「因果之核」** |

⚠ **⚠ 苏晓的用法**:⚠ **把它寄生到五名叛逃吞噬者中智商最低、战力却仅次于黑A的**三代吞噬者·暗阳**身上**——⚠ **让暗阳自以为是大主教·萨乌洛所说的「神选者」，开始在冥神殿内部编织陷阱**（⚠ 选暗阳是因为以他的智商，「智谋」这种东西显然兼容不了他的CPU——**暗阳＋狂笑假面的组合，能把除太阳使徒外的其他三名神之子都忽悠到怀疑人生**）。
