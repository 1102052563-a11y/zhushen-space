# 🌰 至强巅峰阶段的资源观

- ⚠ **⚠ 「原初之核」取代「灵魂晶魄」成为新的头号缺口**——⚠ **提升核心技法「天锋斩影」每级=时空之力+原初之核×10+初始源质×1；轮回级装备的强化也开始吃原初之核**（⚠ 荣誉商店内7500盎司一颗且**常年0公共库存**，每次刷新瞬间被猎杀者们买空·⚠ 地精商会为满足苏晓玩了命地收购市面上的原初之核，**把这个稳定了半个纪元的至强梯阶资源价格给收涨价了**——总经理·金古沃快哭了）。
- ⚠ **⚠ 「时空之力」的行情暴涨**:⚠ **从早些年的1盎司=95枚灵魂钱币，一路涨到去年稳定的1:300**——⚠ **本次陨灭星期间已到1:400，暗地里甚至1:423**（⚠ 原因:深渊浩劫将至，消耗时空之力的群体九成以上是乐园阵营的至强梯阶，他们都知道这点·⚠ **真正引爆这次猛涨的其实就是苏晓自己**——他让第三纪元出现了超脱之界，在悄然间改变了很多事·⚠ 苏晓由此发现一条闭环:卖时空之力换灵魂钱币→买最便宜的至强被动卷轴〔约210万〕→用技能升级仓怼满〔约3200万〕→用「力量始源」全部牺牲换至强主属性）。
- ⚠ **⚠ 「神魂」的魔幻价格**:⚠ **至强神魂因融合条件苛刻、概率奇低〔万分之一都不足〕、失败生不如死、成功还有副作用，约80～100万灵魂钱币一颗，卖不动时50万都有可能**——⚠ **反观绝强神魂约20多万一枚**（⚠ 因为九成九的至强神魂被买去都是当**材料**、用来向天启乐园申请相关权限确保融合必成·⚠ 绝强神魂的传承需求低很多，而轮回乐园还有一个与契约者差不多庞大的群体——**职工者**〔商人、工匠等，混到大后期财力不差〕:一颗传承需求只是「智力属性75点」的绝强神魂，最终能在拍卖上卖出高价·⚠ **绝强神魂体系与宝石力量体系，是公证最受职工者欢迎的两个体系**）。
