## 装备品级
装备分为 **白色、绿色、蓝色……**（阶位不足时乐园不给出更高品级资料）（第4、19章）。

- **白色装备 / 道具**评分 **1–10**；评分越高价值越高；评分为 10 的白色装备会标注「**稀有**」，附加特殊属性，**甚至胜过部分绿色装备**（第4、7、15章）。
- **满评分规律**：满评分装备通常都不错（如白 10 的斩龙闪）；可推知满评分的绿 / 蓝装属性更强，且必存在更高级别装备（二卷4章）。
- **绿色装备**评分 **10–30**；评分为 30 的绿装标「**精品**」、附加特殊属性（如幸运女神之焰 30 分；恶灵咆哮 21 分为普通绿装）（二卷9、33章）。
- **首饰类**装备：稀少物品，**无耐久**、无需修理，多数任何人可穿戴（第10章）。
- **蓝色装备 / 物品**：评级在绿色之上，**评分 30–70**、非常稀少、价值远高；评分 70 的蓝装标「**极品**」、附加特殊属性（如三笠的心意 70 分）。苏晓在 24 区后勤部首次见到蓝色物品（人体基因强化液）（二卷43、85章）。
- **紫色装备 / 物品**：蓝色之上的品级，**评分 71–150**——苏晓在斩·赤红之瞳的秘境得「帝具改造蓝本」（紫·89 分·见下「帝具」节）时**首次见到紫色品质**、由此得知蓝色以上即紫色（四卷33章）。
- **暗紫色装备 / 物品**：紫色之上的品级，**评分 151–260**（评分越高越珍贵·凤毛麟角·交易市场几乎绝迹）——苏晓六卷68–75章首次接触：**大剑豪的剑术心得**（暗紫·170）、**牛牛果实（长颈鹿形态）**（暗紫·259）（六卷68、75章）。**评分满 260 的暗紫装备会附加「特殊效果」**（如**神灵余晖**的「圣灵赐福」·见下「神灵余晖」）。
- **库因克**（CCG 喰种武器）也分品级、可附带主动技能——如绿色精品库因克「粉碎精灵」（见下）（二卷43章）。
- **完整品级阶梯（十五卷18章·苏晓总结）**：白 < 绿 < 蓝 < 紫 < 暗紫 < **蓝色套装** < 淡金 < 金 < **暗紫色套装** < **淡金色套装** < **金色套装** < **传说级**。可见「套装」整体高于同色单件、且金色套装之上即「**传说级**」——传说级单件可媲美一整套金色套装。成长类装备（如斩龙闪）不在此列、资源足够可一路升到传说级。
- **传说级宝箱 / 物品（十五卷18章·苏晓斩四代雷影首得）**：**传说级宝箱**呈暗金色（深沉厚重）——开出 80% 金色 / 10% 淡金套装件 / 8% 金色套装件 / **2% 传说级物品**。**传说级物品极稀有、一经使用即与灵魂绑定**（无法出售·死亡后有几率在猩红卡内保留）。
    - **传说之勋（传说级 · 材料 · 十五卷46章·开雷影传说宝箱所得）**：酷似黑色鹅卵石、巴掌大却重约百公斤——**锻造传说级装备的必需品**（评分 436·出售价 = 灵魂结晶〔中〕×6·苏晓首见能换灵魂结晶的物品）。产地限「开放性衍生世界 / 三阶以上战争世界」；苏晓暂留待研究——或是把斩龙闪推上传说级的关键。
    - **岚彻（淡金 · 重刀 · 十五卷45章·开达鲁伊淡金宝箱所得）**：达鲁伊的佩刀（101.6×22.3cm·攻 80~106·力 80 / 敏 50 / 刀技 Lv.25）——**查克拉传导金属 + 雷岩合金 + 陨石**打造。① 查克拉传导（被动·按注入的查克拉属性赋能：雷→锋利 +4 + 麻痹 / 火→攻 +15 + 灼烧 / 风→锋利 +5 + 斩速 +20% 等）；② 重刃（被动·命中过力量判定即击飞 + 麻痹 1~2 秒）。**苏晓不留刀、而要熔了提炼其中的「查克拉传导金属」**（火影世界最贵金属·传导查克拉几乎零损耗·云隐把这么大块打成一把刀属暴殄天物）——这堆金属足够把 [斩龙闪](../index.md) 推上**金色品质**、或换多把淡金武器。

## 物品携带规则
- **宝箱所得**：基本都是**规则物品**，可带出任务世界（第12章）。
- **拾取 / 任务途中所得**：有**一定几率**可带出（如斩龙闪）；否则为本世界限定（如破旧的燧发枪）（第4、12章）。
- **触发类道具**：满足条件即触发效果（如引荐信）（第7章）。
- 世界内得到的部分现实物品（如缴获的匕首）无资料、不可带出、不可换乐园币（第6章）。
- **乐园内交易**：他人所有的待售物品无法带离 1 米、无法收入储蓄空间，须购买取得支配权（第24章）。

## 装备隐藏功能
花 **100 乐园币**永久开启，可自由隐藏穿戴中的装备（**武器类无法隐藏**）；隐藏后属性加成仍生效（第10章）。

## 宝箱
击杀世界内人物 / 生物可掉落宝箱。要点（第19章）：

- **品质递进**：白色 < 绿色 < 蓝色……；宝箱所开物品的等级与宝箱颜色相关。
- **开出什么与掉落者相关**：杀老虎多得虎爪 / 虎皮，杀人多得其遗物（杀黑市商人 → 引荐信、杀欧卡 → 项坠），不会从老虎宝箱里开出剑、盾。
- **白色宝箱**：最低等；曾开出乐园币、贝利、亡妻的项坠等（第5、7、10章）。
- **绿色宝箱**：杀科尔波山的主人的开出 **1000 乐园币 + [灵魂结晶（小）](../消耗品材料/index.md) + [虎爪（绿·材料）](../消耗品材料/index.md)**（第19章）；杀壁虎的开出 **900 乐园币 + 赫包（S）+ 恶灵咆哮（绿·戒指）**（二卷9章）。

## 帝具（斩·赤红之瞳 · 世界特有）
约一千年前，[斩·赤红之瞳](../../世界/任务世界/斩赤红之瞳.md) 世界的帝国举国之力，以**超级危险种**（恶种恶龙、血统稀薄的恶魔等）的身体素材 + 稀有金属**奥利哈钢** + 大量失传秘术，由顶级匠师打造出 **48 件无法复制的武器，称「帝具」**；造毕后帝国兔死狗烹、处死工匠、技艺断绝，存世仅此 48 件（四卷12–13章）。

- **定位**：苏晓视帝具为「**削弱版的恶魔果实**」——各有千奇百怪的能力，部分涉及**空间**，少数甚至强于恶魔果实。
- **使用规则**（四卷21章）：帝具**择相性契合的主人**、非人人可用；**主人死后帝具能力仍在、等待下一任主人**。帝具使的通病是**身体素质相对不优**（能力强、肉身弱）。
- **猎杀者夺帝具的规则**（四卷40章）：杀帝具使可直接夺取其帝具，但**帝具无法带出斩赤世界**——若相性匹配可在本世界内使用、**回归乐园时自动变卖成世界之源**；想获得**能带离**本世界的帝具，须从杀帝具使后掉落的**宝箱**里开出。
- **三大类**：① **食用 / 饮用类**（如恶魔之粹·一次性·能力一般最强）；② **武器类**（刀枪剑戟·能力五花八门，如一斩必杀、异常锋利）；③ **生物类**（一个类人形生物 / 一条狗 / 一只熊等）。综合而言一次性帝具最强。
- **皇家护卫军 / 狂犬部队**（帝国史·四卷56章）：48 件帝具初成时，48 名帝具使合为「**皇家护卫军**」（敌国称「**狂犬部队**」）——攻防 / 远程 / 大面积杀敌 / 刺杀敌首一应俱全、可组完美攻坚阵，是帝国基石，曾横扫周边两百年。后王族傲慢、帝具屡屡遗失：成立 500 年失 7 件、千年后仅剩 14 件、小皇帝在位期间又失 10+ 件（艾斯德斯出现才止住、给帝国「续命」）。
- **主线目标「帝具·至高之力」**：48 帝具中**最强**的一件，现持有者为帝国 [小皇帝](../../人物/小皇帝.md)（须皇室血统启动·外人夺钥匙也无法用、且钥匙 1~2 年可重做、真正不可仿制的是帝具本身）。其本体是一具**上百米的超大型机器人 / 炮台**（帝国炼金科技巅峰），平时半埋皇宫地下、皇室激活后破土而出，**发射的激光覆盖整个帝都、可五分钟夷平帝都**。本世界主线即夺其**启动钥匙**——**苏晓已于四卷57章绑小皇帝取钥匙、完成主线任务**（详见 [斩·赤红之瞳](../../世界/任务世界/斩赤红之瞳.md)）。

## ⚔ 斩龙闪 +21（轮回级·二之位）

本批完成登顶。三条提升线并行:

| 线 | 内容 |
| --- | --- |
| **吞噬** | 「至尊锋刃」以魂能结晶推到极限后，吞下**37 把传世名刀**（吉尔厄琉家族以三瓶【永生者药剂】为价搜罗，⚠ **已囊括星界已知九成传世名刀**）＋ 轮回级三之位 **【终焉】**，冲上 **30000 满评分** |
| **强化** | 「武器强化权限」直推 **+15**（⚠ 轮回级强化是先提升**因果层面的上限**、上限提升即代表成功，故 +1 到 +15 仅耗 60 秒出结果，后续核心提升需 6 小时）→【最高级强化石】必成 **+16** → 随机出 **95%** 连续强化成功率，在【命运逆转】加持下赌到 **+18** → 天启乐园装备强化大厅由「大师具象·凯撒」推到 **+21** |
| **镶嵌** | 【星界宝石】把「魔刃」**飞升级提升** |

**当前属性（+18 时的记录）**:耐久 9500／攻击力 5290~5560／⚠ **锋利度 34408（当前已知武器最高）**／镶嵌 3/3（灵魂宝石·星界宝石）／评分 30000++。<br>**强化效果「极之锋刃」**:每 +1 强化等级，锋利度 **+25%**（⚠ 因此强化越高收益越大:+19 即 +8602 点锋利度，+20 即 +10753 点）。

!!! note "轮回级装备强化的四条规则"
    1. 并非直接强化装备核心，而是**进行因果层面的上限提高**，此过程极为迅速；上限提升后装备强化机再据此提升装备核心。<br>2. 可连续多次强化，并在成功后单次提高多级；相对地本次提升时间会稍有延长。<br>3. 一次强化成功后可选择**连续强化**，成功率**递增**——但递增值完全随机（可能累计 0.1%，也可能 99.9%），且会概率性清空（或连续两级清空，或连续三到五级后清空）。<br>4. ⚠ **一旦强化失败将立即损毁！** 但耐久度高于 3000 点的轮回级装备可抵抗损毁，改为强化等级清 0、永久扣除一定的耐久度上限。<br>⚠ **到了强化轮回级装备的阶段，运气早就是实力的一部分**——因为这是幸运属性或运势所衍生的；倘若无视运气搞公平概率，反而是对在这方面有投入的契约者最大的不公平。
