# 血刃

> 十卷拍卖会上的未鉴定拍品。

## 血刃（未鉴定 · 拍卖会拍品 · 十卷26章）

> 原属批次:已收录（截至第20章）

[苏晓](../../../人物/苏晓.md) 以独眼巨怪眼球向 [十老头](../../../势力/index.md) 换得的拍卖会拍品（在 41 层宝库任选一件）——一把**与 [斩龙闪](../../../index.md) 款式类似、刃口血红**的长刀，因而名「血刃」。出鞘斩出凛冽破空声，苏晓凭手感判断**品质不低、至少能通过轮回乐园公正认证成装备**（拍卖会拍品需返乐园才能鉴定是否为装备 / 其品质）。已有客人预定却未付钱，被苏晓一句「没付钱就让他闭嘴」截下。

!!! note "拍卖会与「未鉴定」拍品"

    友克鑫拍卖会的藏品中**可能混有装备 / 材料、但需运气**——出宝库时只显示名称 +「品质：（未鉴定）」，须带回轮回乐园方能定品级。苏晓在宝库另见过 **【纯金处刑大刀】**（黄金大刀·未鉴定·未选）等武器、铠甲、未知药剂。真正珍贵的拍品委托人不放心存宝库、只在开拍前才送来。
