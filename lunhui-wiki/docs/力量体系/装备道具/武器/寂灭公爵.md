# 寂灭公爵 +13

> 换自梧桐冒险团的金色狙击枪。

## 寂灭公爵 +13（金色 · 重型狙击枪 · 十五卷110–111章·换梧桐冒险团所得）

> 原属批次:装备品级


[苏晓](../../../人物/苏晓.md) 用 2 枚传说之勋 + 1 颗灵魂结晶（大）向五阶梧桐冒险团副团长 [浅梦](../../../人物/浅梦.md) 换来的金色重型狙击枪（精心打造、强化至 **+13**·替下狙杀柱间时显疲软的 [蜘蛛女皇](../../../index.md)）——产地世界争夺战·门瑟星·19.67mm·攻 154~213·需力 79 / 敏 60 / 体 70 / 枪械 Lv.28·评分 400+。

- **公爵领域（主动·无冷却 / 无消耗）**：弹出自适应 2~53 倍镜·500 米内自瞄；超 500 米可花 10 秒**锁定**敌人任意部位（头 / 胸 / 臂 / 腿）、下一枪**子弹追踪**，极限射程 4762 米（+13 加 1260 米·锁定冷却 10 分 / 耗 150 法力）。
- **寂灭（被动）**：子弹穿透 +17 + 穿甲弹效果；**近战重炮（被动）**：未开公爵领域时射速 +80%。
- 镶 3 宝石：**衡序**（枪械基础技能阶位 +1·专精级以下→苏晓枪械精通 Lv.30 升**专精 Lv.30**·攻 +49%）/ **杀戮**（攻 +18~23）/ **狞笑**（爆头伤害 +25%·锁定基本只锁头）。
- **真身是「护城炮」**：原本架在城墙上、需机甲 + 缓冲装置才扛得住后坐力、专打 80 米以上魔种翼龙（一枪穿 4 米特种合金）——苏晓靠 **82 真实力量 / 82 真实体力直接无视后坐力**端着狙击。缺点是**子弹奇贵**（基础弹 1000 币 / 特殊弹 1500 币·「用乐园币轰死敌人」）、淡金以下消音器一枪就开花。**弹种**：普通弹默认带穿甲（直线穿透恐怖）；**元素弹**分火 / 雷 / 土 / 暗（分别对应力 / 敏 / 体 / 智属性·属性越高威力越大·横向破坏强·另有第五种「心灵」对应魅力被苏晓无视·十五卷117章）。
