# 屠戮

> 开修斯金色宝箱所得的副武器。

## 屠戮（金色 · 短刀 / 副武器 · 十一卷49章·开修斯金色宝箱所得）

> 原属批次:已收录（截至第20章）

圣堂教会第三任代行者首领的佩刀——陨石合金 + 高强度钛 + 月光精华 + 秘银打造的 52.8cm 短刀（攻 115~193·评分 390·官价 17.5 万币·装备需求力 30 / 敏 47 / 意志力 18 / 刀类技能 Lv.30）：① **断喉**（被动·割裂任何生物喉咙时造成基础伤害 50%~60% 的额外伤害·**无视身体防御 / 抗性**）；② **泣血**（被动·触发断喉后对敌作体力判定·未过则每秒流失最大生命 4%·持续 10 秒·此伤害有优先性；敌血 ≤30% 时持续减半但出血翻倍至每秒 8%）；③ **月之光华**（被动·锋利度 +14 / 破甲 +10 / 穿透 +9）；④ **急速切换**（主动·无消耗无冷却·武器切换速度 +70%）。苏晓留作**贴身副武器**（斩龙闪太长、贴身时只能反握割伤·有了屠戮可斩 / 割 / 捅·补上近战短板，与曾得的 [卞氏匕首](../../../index.md) 互补「即死 + 锋利」对「毒素 + 穿透」）。简介警告：意志力过低则第一个被割喉的是使用者本人——苏晓意志力远超 18 点（戴 [火之意志](../../../index.md) 无排斥即 ≥20）故无虞。
