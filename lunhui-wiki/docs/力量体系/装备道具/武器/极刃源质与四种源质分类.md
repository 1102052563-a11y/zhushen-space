# 📦 极刃源质与四种源质分类

| 层级 | 代表 |
| --- | --- |
| **通用源质** | ⚠ 力量源质 |
| **专属源质** | ⚠ 魔灵源质 · 辉光源质 |
| **至高源质** | ⚠ **共五种**（⚠ 黄金之力 · 初始源质 · 其余三种以苏晓现阶段的猎杀权限仍未知） |
| **无上源质** | ⚠ **仅两种:元素 · 深渊** |

⚠ **【极刃源质】（又称刃之源质／暗魔源质）是这二者的衍生之物**——⚠ **⚠ 刀术宗师从 Lv.94 提升到 Lv.95 的必需品**;⚠ **⚠ 传说中，最初第一位把刀术技法推到极限的人，为了更进一步选择吸收「深渊源质」，结果技法强者身死**——⚠ **可他的佩刀保留了下来并一并吸收了深渊源质:这把武器纵观三个纪元依然能排进前三**;⚠ **⚠ 它沉寂多年后重新复苏，并非神器出世，而是**炸碎开来**——史上第一份「极刃源质」就此出现**;⚠ **⚠ 源质出现很难，但只要出现第一份，后续就会成为星界法则的一部分**（⚠ 深渊源质凝聚时有极低概率化为极刃源质·⚠ **元素源质也有向此转变的概率，但不到亿万分之一，那种叫「辉刃源质」**）;⚠ **⚠ 苏晓选极刃而非辉刃:「满身血气、手持黑蓝色烟气飘散的利刃」才符合他的风格——**逆着自己的风格体系发展，一般都不会有太高成就**。**

- ⚠ **它的代价**:⚠ **等同于巫师阵营 **25～30 年**的总财富收益**（⚠ 这解释了星空研究会高层为何人均苦哔脸）。
- ⚠ **⚠ 【浓缩深渊源质结晶】**（永恒级宝箱·恶变女巫）:⚠ **两种无上源质之一的「深渊源质」**——⚠ **但它已不算源质，而是一种特别危险的结晶物，总计产生过不超十颗**;⚠ **⚠ 苏晓推测:或许每个超脱·原生世界初开时才会形成一颗**（⚠ ⚠ 警告勿使用、勿解除封印）。
