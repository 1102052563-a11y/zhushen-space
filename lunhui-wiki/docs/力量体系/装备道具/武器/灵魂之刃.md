# 灵魂之刃 · 副武器（Lv.EX）

> 灭法体系的灵魂系副武器。

## 🔪 灵魂之刃·副武器（Lv.EX）

> 原属批次:📚 续26 · 返园爆改 · 终极·基础被动与三宗师破 Lv.80（50卷65–69章）


⚠ 从荣誉商店以 **360 盎司**兑来的技能卷轴。⚠ **技能效果 1:你的傲歌能力，整体强度提升 20%** · ⚠ **技能效果 2:你可通过傲歌能力所产生的晶体构成自适应武器**——⚠ **此武器的斩击力、钝击力、穿透力等，将根据你的灵魂强度以及身体能量上限而定**。

- ⚠ **掌握过程顺利到难以想象**:⚠ **原因是在掌握这能力后要先锻炼身体能量半实体化**（**让放出的身体能量构成晶体**）——⚠ **这点苏晓早就达成**；⚠ **不仅如此，连究极进阶版的灵影线他都开发出来**。⚠ **也因如此，他很顺利就掌握了它，并且都不需要太多的适应与锻炼就能得到全部加成。**
- ⚠ **因斩杀了老兽王，现在刃之魔灵处于休眠阶段无法放出，也就暂时测试不了它。**
