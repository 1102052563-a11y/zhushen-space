# 🆕 幽冥骨戒 · 无上烈阳 · 封之刃 · 起源级刀（第49卷续16 · 暗影世界）

- **幽冥骨戒（深渊·原罪物·「爹级」）**：⚠ 苏晓从深渊宝箱开出的**第四件原罪物**（也是他手上第一件真·原罪物之外的又一件）——通体带强烈幽冥、冤魂、幽邃气息的骨戒，喷幽绿（非深渊之黑）烟雾。⚠ 其综合危险度**满值 100**（与深渊之罐、死灵之书并列，略高于灵魂王冠的 99.5）。苏晓开出即以深渊盒封起（凯撒瞬间溜出 200 米、只肯「困着等因果缓缓再想办法」，苏晓付 3 万钱币请他处理）。至此苏晓身上同时有**深渊之罐（凯撒）、灵魂王冠、幽冥骨戒**三件爹级 + 死灵之书因果，连水哥的始源魔镜都不敢招惹。
- **无上烈阳（起源级·主铭文）**：⚠ 通过陨火之地「太阳试炼」在太阳圣殿石碑取得，苏晓所见**第一块起源级铭文**。效果:**免疫 55% 太阳焰伤害**（含真实伤害），每在铭文基座插一块副铭文再提 0.1~5%、最高**免疫 75%**。仅可作主铭文。「直面太阳者，无惧太阳之烈焰。」
- **烈阳圆盘（起源级·可成长）**：⚠ 在陨坑吸满 158.59% 太阳焰后由不朽级晋**起源级**。持有效果:用太阳奇迹 / 术式 / 太阳特性装备道具爆炸物，强度或伤害 +20%（苏晓默认为阿波罗 +20%）；每小时提升 5 点评分（一天 120·25 天满 3000）、满评分后可**生成同评分的「太阳石」**（矿石 / 材料 / 消耗品）再归 1 点评分——躺在储存空间里白薅收益。成长条件是「吸收本源·太阳能量」（寻找陨落的太阳·极难）。
- **封之刃（灭法放逐灭世级之物 · 耐久 195/340）**：⚠ 灭法委托传说铁匠打造，不为杀敌，而是**放逐「灭世级」灾祸**（把目标送进单向空间通道、去往永光世界），每放逐一个灭世级族群、放逐能力更强（每次耗 1 点耐久）；它也是**开启永光世界**的钥匙（破坏它等于打开永光世界封印）。苦痛女王当年偷走它、丢在死灵之书旁，被神父带走、又辗转卖回灭法手中——苏晓用它把晋升绝强者的苦痛女王放逐走。
- **铭文之主 / 深蓝（起源级·刀类武器）**：⚠ 卖辉光遗物给晨曦神教时意外换得的两把起源级长刀（晨曦神教存不住起源级防具、却压着两把起源级刀）。
- **炽光枪 / 耀光心核 / 辉光卷轴 ×2（辉光之神遗物）**：⚠ 斩辉光之神所掉**炽光枪**（起源级·神灵武器·苏晓用它给白金主教「贯胸」伪装白夜战败）、上上任辉光之神留下的**耀光心核**、两卷记载辉光能力的卷轴——因神灵武器使用前置苛刻、在大聚地卖不上价，苏晓打包卖给新飞升的晨曦神教（各求所需）。
- **豁免徽章（★★★★★）**：⚠ 抹黑玫瑰之名所得（估值 500 盎司）。免除**魅力 / 意志力 / 幸运属性低于 0 点后的减益（−50 点内）**——苏晓负魅力 −17，从此魅力 / 幸运哪怕被砸到 −49 都运势平稳（升级了原来的 ★★ 版）。
- **灵运项坠（特级幸运物）**：⚠ 幸运女神所赠，鹌鹑蛋大、内含星辰金粒。专治「极运之后的极衰」——巨量吸收厄运、负负得正让运势平稳；给苏晓一戴就因吸他海量厄运当场炸开（他的极衰恰好度过）。
- **火金（稀有材料）**：陨火之地沙下埋着的巨块暗金色金属、纯度高到轮回乐园权限都难兑换（圣诗两次滑倒各捡一块、卖苏晓 1 万 / 2 万钱币）。
