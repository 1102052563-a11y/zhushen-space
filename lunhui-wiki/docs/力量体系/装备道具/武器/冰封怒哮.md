# 冰封怒哮

> 原「冰蓝战锤」，随从阿姆的专属武器。

## 冰封怒哮（原「冰蓝战锤」· 阿姆专属 · 十七卷67–85章）

> 原属批次:已收录（截至第20章）

[里德](../../../人物/陵墓守卫者·里德.md) 锻造的两把锤之一（**冰属性**·另一把火锤里德常用）——本是**镇压某存在**的武器、非常规兵器（约 2 米·六角锤头·水桶粗·通体冰蓝·缕缕寒气·会持续外放冰能量·常人握之会被撑爆）。[阿姆](../../../人物/阿姆.md) 因冰能量相似度 >95% 被它吸引拿起、致核心阵图过载；苏晓为它构建能量渠道（冰锤当「心脏」、冰能量循环回锤）后，它成了阿姆的**专属武器** + 变相延寿的核心——阿姆从此手持冰锤、附冰属性攻击。**🆕返园公证（十七卷85章）**：希亚通关返园后系统提示「冰蓝战锤」真名 = 「**冰封怒哮**」（战锤·未公证），苏晓花 **15 万乐园币 + 灵魂结晶〔大〕×3**（割肉价·但阿姆已与它形成能量循环、非公证不可）完成公证，阿姆正式获得其所有权（完整属性才解封）。
