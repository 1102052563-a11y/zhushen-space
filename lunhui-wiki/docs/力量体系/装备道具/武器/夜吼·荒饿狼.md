# 夜吼·荒饿狼 / 银月之刃 / 暗月星环

> 银月之刃推到「绝对极限」的一线装备。

## 🌑 夜吼·荒饿狼 / 银月之刃（→绝对极限）/ 暗月星环

> 原属批次:续39 · 血月女王、领主列车与银月之刃极限（53卷13–27章）


- ⚠ **【夜吼·荒饿狼】**（黑刀·荒饿狼给了 [巴哈](../../../人物/剧毒魔鹰·巴哈.md)·详见其页）:⚠ **耐久越低攻击力越高（50 倍速消耗·归零转入体内吸血）· 突破隐匿攻击颈后造成基础＋10% 最大生命的真实致命伤害（体力＜敏捷则强即死）。**
- ⚠ **⚠ 【银月之刃】→绝对极限**（霸主级·评分 **8000**·用「深渊监视者的灵魂残烬」在黑雾岛·狼冢提升）:⚠ **银色月光（割掌心·随机得四种加成之一:武器锋利度 +25% / 生命总量 +15% / **幸运 +50 且带「绝对运势」可在灭法运势下起效** / 大幅提升魅力）·⚠ 月之血誓:身处深渊/深渊蔓延区时周边浮现月光驱散绝对黑暗**（⚠ 简介:「我等愿以月光为你照亮最黑暗的深渊，愿你能平安无事」）。
- ⚠ **⚠ 【暗月星环】**（真品·斩高塔骑士长所得·现仅剩唯一一枚）:⚠ **第一纪元暗月阵营举全族之力开发的奇迹之物**——⚠ **可自行放大镶在深渊通道周边、缓慢收拢关闭深渊通道**（⚠ **一枚只对一个通道生效、镶上就拿不下来·⚠ 关闭本世界所有小型深渊通道要耗 6～8 个超脱之界＋第一纪元前期到第二纪元中期的全部资源**）;⚠ **⚠ 苏晓最终打算让 [艾露克露](../../../人物/虚空/自然之女·艾露克露.md) 把它带回虚空**（⚠ 用于虚空·绿茵大湿地那个无法关闭的深渊通道·⚠ **他不希望虚空近期遭大面积深渊侵袭引来大量深渊任务、对他不利**）。
