# 🎰 大转盘 —— 一次纯靠刀抵脖子的抽奖

⚠ **巫妖商人·吉欧〔自称「伟大的洞察与博弈尊者」〕的转盘宽五米以上、上千个分隔**——⚠ **正面奖励夸张到令人怀疑真假〔5000万灵魂钱币、999颗灵魂晶魄、一等奖100颗原初之核、特等大奖？？？品质装备〕，可这转盘是**两面的****（⚠ 比如「999颗灵魂晶魄」的区域，反面是**损失1099颗**——一来一回反倒得倒贴他100颗）;⚠ **⚠ 苏晓的破局**:⚠ **按下启动按钮的下一秒，长刀出鞘抵在吉欧脖颈上**——⚠ **这立即触发虚空之树惩戒、每秒滑落信誉度；但他早从「偏中立单位」的判定里算准了对吉欧出手的风险不高**（⚠ 吉欧的困惑:对方作为乐园阵营的契约者，怎么敢在他还没展露敌意、甚至没来得及作弊前就把刀抵上来·⚠ 而苏晓不仅敢，还已决定**若收益不能让他满意，就顺手欣赏下吉欧脑袋飞起来的景象**）;⚠ **⚠ 吉欧的绝望**:⚠ **他为保险起见设定「至少运势-5万以上才**有概率**转到特等奖」——可眼前这位**运势-15万**，转十次最起码八九次特等奖**（⚠ 而特等奖是他一直想用却从没用过的**夺取额度**·⚠ 他暴起血丝、十指交叉高喊着作弊操控，才把指针停在特等奖前一格——**只差0.5毫米**·⚠ 他放松地呻吟了一声，随后只想马上离开、然后把那条尿湿了的裤子换掉）;⚠ **⚠ 最终结果**:⚠ **损失300万灵魂钱币、获得450万额度储蓄卡 = 实得150万**——⚠ **抽完的瞬间吉欧向虚空之树申请当即溜了，并把苏晓拉进了黑名单**（⚠ 剩下那枚硬币被苏晓通过「眼之仪式」送给了【无暗之眼】）。
