# 🗡 封魔:最强霸主装备（霸主级·首位）

> 霸主级首位装备，觉醒【噬魔体质】的来源。

**吞噬路径**:【灭世绝刃】→【黄昏残翼】（原霸主五之位，75 万盎司购得）→ ⚠ **【黑暗源血】**（当时的最强霸主装备）→ 经三大传说铁匠再造登顶。

!!! note "为什么要吞掉灭世绝刃"
    ⚠ **装备的增益是有「使用幅度」的**。「灭世绝刃」标称「刀术技法核心综合强度 +25%」，但在天锋斩影 Lv.315 的幅度下，**实际生效只有 +0.228%**。<br>⚠ 规律:**越是到了高上限，轮回级装备的增益越强**（需霸主装备前三或轮回级前十才能带来全额增益）；而**霸主装备是首位占据一切**——到最后，只有霸主首位的【封魔】保持全额增益（当然也有它是在增益斩龙闪的原因）。

!!! tip "【封魔】（霸主级·首位·刀鞘）"
    **极致锋芒（核心·被动）**:提升所因果绑定的「斩龙闪」**30% 锋利度与耐久度**（⚠ 此提升可超出原极限）——最终把锋利度推到 **260000（绝对极限）**、耐久推到 **48000→180000**。<br>**霸主姿态（被动）**:⚠ **此霸主装备将占据全部 5 个霸主装备栏**。<br>简介:此为恶魔铁匠的得意之作，也是其最强之作。

⚠ **代价**:三大传说铁匠（熔火巨人、矮人王、恶魔铁匠）为打造【无尽之环】与再造【封魔】，全部变成皮包骨的虚弱状态——熔火巨人只剩大骨架撑着，矮人王的胳膊是一层粗糙皮肤包裹粗壮骨骼，恶魔铁匠宛如岩浆流淌的双眼都暗淡、山羊犄角出现细密裂痕。⚠ 矮人王的原话是:能帮你打造无尽之环，是我们三个老家伙的荣幸。
