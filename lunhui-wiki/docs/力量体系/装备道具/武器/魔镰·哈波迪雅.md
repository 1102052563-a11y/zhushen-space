# 魔镰·哈波迪雅

> 斩蔷薇所得，也是「成长类武器规则」的样本。

## 成长类武器规则 + 魔镰·哈波迪雅（十五卷67–69章·苏晓斩 [蔷薇](../../../人物/黑瞳-契约者.md) 所得）

> 原属批次:装备品级


- **成长类武器铁则（苏晓由魔镰悟出）**：① 使用者死亡后**有 93% 几率掉落**（故苏晓的 [斩龙闪](../../../index.md) 也可能在他死后落入敌手）；② 每人**最多同时绑定 3 件**；③ **可熔炼、不可出售**（迄今没人舍得熔成长武器）；④ 绑定前无法收入储存空间、绑定后无法交易。蔷薇死后魔镰竟散金光（无主装备特征·普通装备会封禁属性并在半小时内消失）——这才让苏晓窥见此规则。
- **魔镰·哈波迪雅（金色 · 成长类双镰 · 强化 +9）**：产自中低烈度战争世界·费迪星，**陨落血天使的专属武器**（攻 123~167〔左〕/ 136~178〔右〕·需意志 40 / 力 63 / 体 65 / 镰刀类技能 Lv.20）。**左镰**「魔灵附体」（主动·近战技能 +8，但手持时每 60 秒强判一次属性、未过则被魔镰之灵夺意识——苏晓因排斥不可用）；**右镰**「嗜血礼赞」（被动·伤害 12% 转治疗·仅对生物）；组合技「血魔偶」（减 10% 生命上限召两具血魔偶·需「血之源」）；专属被动「血生之物」（吸生物血提品质·满 15 万血能可耗灵魂结晶〔大〕×15 升品质·现 12270/15 万）；已镶七宝石不可完整剥离。评分 386。
- **魔镰之灵·哈波迪雅**：一缕**有意识、会噬主**的武器之灵——绑定瞬间就钻进苏晓脑内想反客为主，被 [青钢影](../../技能/index.md) 当外来入侵意识弹出、惨叫缩回（每个血天使都得战胜它才能把魔镰转为专属）。苏晓假意逢迎稳住它、再扔进储存空间封印。**他的真打算是返乐园后用「[至尊锋刃](../../技能/index.md)」把魔镰喂给斩龙闪**（吞武器时极凶暴 + 青钢影可噬其意识·零风险）——根本不会拿噬主魔镰去战斗。
    - **结局·被斩龙闪吞噬（十五卷112章）**：返园后苏晓把哈波迪雅放出「透气」、用至尊锋刃祭献——哈波迪雅这才认出青钢影「**不是下位的同类、是来自虚空的怪物**」、惊觉苏晓竟是**灭法（之影）**「我居然想去算计灭法……」，自知必败却不肯在封闭空间自毁，临消散只留四个字「**你这骗子**」（苏晓当初「斩龙闪是下位武器」纯属夸大晋升需求的烟雾弹）。**她没贡献锋刃值、却触发了至尊锋刃效果2 / 3**——令斩龙闪换上「嗜血礼赞」、并觉醒**空间能力「龙影闪」**（详见上「斩龙闪·蜕变金色」），是比锋刃值更大的收益。
