# 放逐

> 苏晓自塑的青钢影武器第二件。

## 放逐（青钢影武器·第二件 · 二十五卷1章·苏晓自塑）

> 原属批次:装备品级


[苏晓](../../../人物/苏晓.md) 晋升五阶后用 [青钢影](../../技能/index.md) 能量塑出的第二件本命飞行武器（继 [斩龙闪](../../../index.md) 后）——并非固定形态，而是把「屠戮之影」的能量塑成此模样、随时可重塑。平时化作右臂袖口的淡蓝纹路携带（无意触碰不伤己）。

- **状态一（无柄刺剑）**：六十公分长、晶莹剔透的无柄刺剑——**超急速 + 极致穿透**（速度 / 穿透力均胜斩龙闪一筹）、只追求刺穿头颅 / 心脏的一点破坏。裹一层青钢影能量后**可伤元素化（自然系）之敌**、且该层能量数小时不散（除非命中）。
- **状态二（数十碎刃）**：哗啦分裂成数十片锋利碎刃——穿透力稍弱（由状态一弥补）但杀伤极凶残：钻入敌体后可继续分裂成微小晶粒、顺血管涌向心脏；意志不坚者宁可自我了断也受不住那「炼狱」之苦。专治大体型敌人 / 恢复力强的敌人。
- **限制**：敌人脱离苏晓 **50 米**范围即无法操控、碎刃自动飞回——任何能力都非无解。实战首秀于 [顶上战场](../../../世界/任务世界/海贼王-顶上战场.md) 试招黄猿（穿其头颅 / 钉其元素化小腿）、秒杀 CP0 卷发特务、拷杀 [斯图茜](../../../人物/顶上战场配角.md)。
