# 海楼石金属棍

> Dr.贝加庞克所造的半成品，只能在海贼世界使用。

## 海楼石金属棍（半成品 · 仅海贼世界可用 · 十八卷27章·Dr.贝加庞克所造·寄存海贼世界）

> 原属批次:装备品级

约 1.2 米、几十公斤重的通体漆黑金属棍——掺了**海楼石**（海贼世界**恶魔果实能力者的最大克星**：一接触即快速脱力、果实临时被「封印」、元素化的身体变回血肉之躯，远胜武装色〔只捕捉本体〕与海水〔只使其无力、果实仍在〕）。当年苏晓在世界政府研究所偷藏海楼石、由 [Dr.贝加庞克](../../../人物/index.md) 帮他打造（半软禁期·二人私交深·黄猿盯着只准打「杀伤温和的金属棍」、不准打刀——世界政府知苏晓善刀）。**未经轮回乐园公证**（公证费太高 + 用途窄），故**只能在海贼世界使用、不可带出**（未公证物不销毁、寄存于该世界·苏晓再来即取回）。海楼石太硬、贝加庞克都无法二次改造，它**唯一能力就是「硬 + 封印果实」**。苏晓拿它当长刀使、专克自然系 / 元素化果实者——德岛海战中逼得 [卡塔库栗](../../../人物/夏洛特·卡塔库栗.md) 宁挨武装色一脚也不敢碰它。
