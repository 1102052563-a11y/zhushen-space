# 怒雷血枪 / 烈阳血枪

> 血枪宗师路线的两件核心兵器，构成苏晓的远程狙杀体系。

## ⚡ 怒雷血枪 / 烈阳血枪（远程狙杀体系）

> 原属批次:续36 · 王者冠冕、古龙遗匙与烈阳血枪（52卷32–46章）


⚠ **「[天怒·奔流斩](../../技能/index.md)」解封至 Lv.87 后新增的被动，让苏晓开辟了全新的远程杀招**:

- ⚠ **界雷淬锻（被动）**:⚠ **斩龙闪每次承接界雷，有 85% 概率永久 +20～30 攻击力 ＋45 锋利度**（⚠ 可多次触发）。
- ⚠ **怒雷血枪（被动）**:⚠ **把界雷聚到武器后 3 秒内，可凭血枪宗师特性将这「巨量」界雷之力从武器剥离，转化为「怒雷血枪」进行远程攻击。**
- ⚠ **⚠ 不当人的叠加组合**:⚠ **（元素亲和力或幸运属性）引界雷 → 转化怒雷血枪 → 以血气虚影的**灵魂大弓**为弓、血枪为箭 → 加持【[厄运古戒](../../../index.md)】（负 22 魅力＝9.49 倍·满额可达 **25.29 倍**·每 0.2 倍需 1000 法力）→ 再加【起源石·缓动】（超大幅提升威力/射程/飞行速度）**;⚠ **⚠ 血枪宗师的前身是「枪械宗师」（远程狙击特性），Lv.85 让他能锁定上百公里外的敌人气息并射杀**——⚠ **⚠ 用途:敌人若靠压箱底手段瞬移到百公里外躲界雷，等着他的将是「比界雷更可怕的东西」。**
- ⚠ **【烈阳血枪】首次实测（52卷45章）**:⚠ **捏碎一根液态阿波罗封存容器（消耗 20% 血气）→ 血气塑形出 4.5 米长、燃着淡金薄焰的血枪 → 灵魂大弓拉满（法力值骤降一半·血气消耗 95%）→ 从云顶之上斜射而下**;⚠ **⚠ 结果:一发轰爆万米高的腐化巢穴、**击穿地壳层形成岩浆湖**，天空城区域全部化为火海**（⚠ **威力与「太阳圣剑」不遑多让**·⚠ 天空城区域约等于一个八阶世界的十分之一大小）;⚠ **⚠ 额外加成来自【烈阳圆盘】（太阳特性物品强度/伤害 +20%）**——⚠ **⚠ 苏晓的下一个魔鬼构思:把一发「太阳圣剑」直接构成剑形当箭矢射出去。**
