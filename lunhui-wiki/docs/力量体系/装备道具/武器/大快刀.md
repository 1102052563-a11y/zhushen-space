# 大快刀

> 海贼世界的名刀评级体系与苏晓的索求目标。

## 大快刀（海贼世界名刀评级 · 十八卷32章·苏晓的索求目标）

> 原属批次:装备品级

海贼世界对名刀的官方评级体系：**良快刀 < 大快刀 < 无上大快刀**。所谓「**大快刀二十一工**」= 大快刀中排名前 21 的名刀（并非只 21 把·现存大快刀据世界政府统计至少 40 余把·名刀战斗会磨损 / 遗失 / 报废、能后续锻造才不至于绝迹）；真正稀少的是**无上大快刀**。[苏晓](../../../人物/苏晓.md) 与 [多弗朗明哥](../../../人物/多弗朗明哥.md) / [杰克](../../../人物/旱灾·杰克.md) 谈合作时，**不要钱也不要权、只要名刀**（恶魔果实也可、唯独不要人造果实）——开价「**3 颗特殊型号炸弹〔阿波罗〕换 1 把大快刀二十一工**」（材料对方出）。他没指望弄到无上大快刀，连大快刀都得靠多弗的香波地拍卖场渠道收购。名刀用途是喂养成长武器 [斩龙闪](../../../index.md)（升锋刃值 / 品质）。
