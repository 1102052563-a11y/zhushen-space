# 🗡 主武器与主要装备栏

- ⚠ **⚠ 【斩龙闪】晋升「轮回级」**:⚠ **吞下三把永恒级极限刀类武器〔神圣星辰13999／妖刀幽冥13999／冰封界14999〕**——⚠ **轮回级·耐久4050·攻击力1240～1455·锋利度+6200·刃之魔灵强度6605·镶嵌3/3〔灵魂宝石·斩魂〕·**评分15200****（⚠ 强化 ＋15→＋17 由 [咕噜](../../../人物/咕噜.md) 亲手完成，用掉苏晓84颗稀有起源石·⚠ 她的技巧:用高强化的白色垃圾装备与主装备交替强化并「强化碎」，或拿大号怀表等秒针停滞超三秒才启动·⚠ 她还承包了晋升后＋0～＋10的强化，收30颗**原初之核**——这也侧证轮回级的强化材料从「稀有起源石」升级成了「原初之核」）;⚠ **⚠ 刃口的十几处崩口**〔与屠神者、祖神厮杀所致〕:⚠ **对达到这种程度的魔刃**有利无害、是一种淬锻**——但不能过度，耐久归0会在下一次剧烈迎击后崩裂**。
- ⚠ **⚠ 主要装备栏扩至20个**〔至强·智力·Lv.EX〕:⚠ **主武器位、副武器位、项坠位、戒指位×3、护臂位×2〔同时佩戴戒指时戒指位装备隐化、无实体具现〕、防具位×4〔头部/上装+下装/腰带/鞋部〕、挂饰位、徽章位、探测类装备位、**自适应位×5****（⚠ 占据多个栏位的装备会优先占据对应位置与空余的自适应位）。
