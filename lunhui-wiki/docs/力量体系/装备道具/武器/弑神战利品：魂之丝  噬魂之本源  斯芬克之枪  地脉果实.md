# 弑神战利品：魂之丝 / 噬魂之本源 / 斯芬克之枪 / 地脉果实（二十八卷33–40章）

- **噬魂之本源（二次觉醒物·月神独有）**：斩 [月神](../../../世界/生物万族/index.md) 必得、仅此一份——[噬灵者](../../天赋.md) 天赋二次觉醒所需物品。用后入意识空间见「灵魂王座」二选一（掌灵魂会剥意识 / 噬魂靠自己变强），苏晓选噬魂、噬灵者升 **SSS**。获得它即放弃本世界之源奖励。
- **魂之丝（史诗级 · 技能卷轴 · 月神独有技能）**：斩月神后用灵魂钱币换的卷轴——具现魂之丝缠住敌人灵魂 10~600 秒（无视物理 / 能量挣脱·灵魂强度不足者无解）；依附实体物（如界断线）则灵魂杀伤 −50% / 强韧 +75%。**前置极高**：真实智力 150 + 真实体力 130 + 灵魂强度 70（苏晓暂未够）。
- **斯芬克之枪（月神大陆专用 · 密德尔顿圣物）**：密德尔顿家族打造、封存月之女神最后一丝灵魂之火的长枪（黑色雷电缠绕·仅本世界可用·须密德尔顿血脉才拔得动）。苏晓 / 巴哈强拔失败（永久失去资格），最终被密德尔顿血脉 [索尼亚](../../../人物/月神大陆配角.md) 拔出、女神灵魂之火随之熄灭、枪炸成残片。
- **地脉果实（传说级 · 道具）**：地之城 [地脉树](../../../世界/生物万族/index.md) 为求苏晓别在它脚下与月神开战而主动相赠的果实（外加「地脉汁液」=可饮可充饥的树根清水）。
