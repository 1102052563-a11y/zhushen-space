# 冒牌灭法之刃

> 一把假货的拆解，牵出灭法之刃的真实构造。

## 🗡 冒牌灭法之刃 · 拆解

> 原属批次:续28 · 极暗古堡与生命晶石（51卷16–25章）


⚠ **【灭法之刃（伪）】的用料**（⚠ **最初锻造原因:准备以假死的方式，在猎人公会那领自己的悬赏金**）:

- ⚠ **【永恒权杖】＋【超高纯度的深渊之力（大量）】＋【1200 盎司时空之力】＋【神教徽章】＋【世界之环（世界级物品）】＋【背叛者意志（世界级物品）】＋【深渊灵魂之石 ×3 颗】。**
- ⚠ **它为什么这么贵**:⚠ **原本苏晓只是想以【永恒权杖】作为材料，委托矮人王简单打造出一把冒牌灭法之刃**——⚠ **可谁知，矮人王越锻造越是顺手，就逐渐加了些时空之力**;⚠ **锻造到中途，已经陆续加入进去 1200 盎司**——⚠ **得知此事，苏晓才陆续拿出这么多资源**;⚠ **这才打造出含有很多魔灵能量的伪·灭法之刃。**
- ⚠ **拆解**:⚠ **苏晓放出刃之魔灵让其攀附在伪·灭法之刃上**——⚠ **几秒后浮现裂痕，最终破裂开，黑蓝色的魔灵能量涌出，全被刃之魔灵笼罩、吸收**;⚠ **⚠ 苏晓没有瓜分这次的魔灵能量**——⚠ **唯有刃之魔灵的魔灵强度更高，才能吞噬更强大的不灭特性·深渊滋生物**;⚠ **等刃之魔灵吞噬深渊滋生物时，会将其本源力量转化为魔灵能量**——⚠ **到时候再分一半，既能稳定提升「所有潜力上限阶位」，也可以控制刃之魔灵的强度上限以免失控。**
- ⚠ **他放弃了什么**:⚠ **如果此刻激活吞噬之核，可以夺一半魔灵能量并以吞噬之核过滤、净化后吸收掉**——⚠ **以此永久性提升自身「所有潜力上限阶位」**（⚠ **上次触发魔影体质效果，就让他「所有潜力上限阶位 +8」**）;⚠ **⚠ 眼下虽不能再次触发魔影体质**——⚠ **但苏晓有 Lv.EX 级的辅助型吞噬之核**;⚠ **从原理上来讲，魔影体质起到的效果就是过滤＋净化魔灵能量将其化为可被身体吸收的增益能量**——⚠ **吞噬之核不仅可以做到这点，还能做到更好。**
- ⚠ **⚠ 结果：刃之魔灵强度 70 点 → 190 点**（⚠ **只要魔灵强度在 300 点以下，他就压的住**）;⚠ **在苏晓晋升绝强后，刃之魔灵也有所提升**——⚠ **并非魔灵强度，而是更像是解锁了上限**（⚠ **不过眼下的程度，绝非魔灵所能解锁的最高上限**）;⚠ **也因为这上限的解锁，刃之魔灵吞噬起魔灵能量，效率是之前的几十倍乃至更强**——⚠ **几分钟时间而已，就把伪·灭法之刃内的魔灵能量吞噬一空**;⚠ **⚠ 苏晓忽然感觉刃之魔灵有点不同了**——⚠ **具体哪里不同，又难以察觉。**
