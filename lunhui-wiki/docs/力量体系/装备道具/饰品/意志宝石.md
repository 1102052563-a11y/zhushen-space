## 💎 意志宝石 / 八颗最强宝石

> 原属批次:续36 · 王者冠冕、古龙遗匙与烈阳血枪（52卷32–46章）


⚠ **月女巫的酬谢之一**（⚠ 与【原初之核】×1、一瓶未催化的 ??? 永恒级药剂同来）——⚠ **【意志宝石】永恒级·唯一·评分 6000＋＋＋**:⚠ **需意志力 500 点以上且**获得此宝石的认可**（⚠ **因苏晓魅力低于 0 点，此宝石绝无可能认可他**）;⚠ **⚠ 效果:意志之力（获得等同意志力数量的「特质属性点」，可临时加成到敏/智/魅 1:1、幸运 2:1、灵魂强度 3:1·24 小时重置）· 意志符文（48 小时生成三枚·消耗后意志特性伤害 15%～25% 转真实伤害）· 意志飞升（???）**;⚠ **⚠ 苏晓的结论:这玩意非常值钱，遇到能激活它的买家会砸锅卖铁来买——「三宗师能力所衍生的贫穷感一下就缓解了不少」。**

- ⚠ **⚠ 八颗最强宝石**:⚠ **意志、神圣、噬魂、自然、灵魂、死亡、元素、深渊**;⚠ **⚠ 已知持有者:灵魂宝石（苏晓·80%）· 死亡宝石（所罗门·未完全体）· 自然宝石（[自然之女·艾露克露](../../../人物/虚空/自然之女·艾露克露.md)）。**
