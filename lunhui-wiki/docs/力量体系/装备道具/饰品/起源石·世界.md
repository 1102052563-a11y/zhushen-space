# 🆕 起源石·世界(4/5) · 无尽本源 · 狼血 · 徽章体系（第48卷续8）

- **起源石·世界（4/5）**：从灰岩广场那棵**枯死的黑枫树母树**主干里掏出的第四块残片（比之前的稍大）——集齐五块即可凭猎杀者权限在轮回乐园内合成完整起源石。苏晓还打算试试:若它像【世界之核（残片）】那样能辅助黑枫树成长、甚至提升其品质，便安放在黑枫树旁（他黑枫树周边已插 41 块世界之核残片当肥料，此举古今独一人）。
- **古老者卷轴（灵魂书库顶层·消耗品）**：圣歌团宝库所得。耐久 1/3（无法恢复）、使用前置灵魂阶位 8。效果「**古老奇迹**」= 先选定一张技能卷轴当载体，激活后对其**逆向扭变**（扭变的程度 / 特性 / 上限由使用者提供的高阶能量决定）。「此物珍贵程度，取决于使用者的眼界与智慧」——苏晓对它另有独特想法（须活着返园才能实践）。
- **无尽本源 / 力量容器（第二纪·炼金造物）**：⚠ 黑枫树内的第二样秘藏，鹅蛋大、中空、半透明结晶壳内是粘稠黑暗 + 缩小星辰。**第二纪·炼金文明以海量「古神力量本源」制成的炼金造物**（灭法猎古神抽出的本源灭法用不上、保存不久，卖给炼金师）——古神植入神躯待适应性激活便能吞噬里面海量契合本源变极强（此说苏晓判为谣言·须外部装置才可能激活）。触之侵蚀内心、险令苏晓右臂古神化（被先古面具拦下），连几米外的古神系罪亚斯都受影响。抓握时以精神力纹印可抑制其外侵（炼金师留的「后门」）。苏晓**白送罪亚斯**当钓鱼——冥神都想要它，罪亚斯背后的上位古神更想借它拉冥神下位。
- **灵魂晶核 ×72 / 灵魂残渣 ×1852 + 大块 ×195（圣歌团宝库）**：宝库原存约 1800 颗灵魂结晶 + 200 颗灵魂晶核，因储存太久被死寂侵蚀成「灵魂残渣」，仅 **72 颗灵魂晶核**保存完好（往常拼杀一个世界进度才得十几颗，这是笔巨款）。残渣苏晓用密封箱存着，今后或有用。
- **狼血（银·月狼·血脉传承·治疗所换购）**：以狼骑士徽章换得。使用前置**深渊抗性 3 点**;人族传承后掌握**剑术大师 Lv.3~10 + 全属性大幅提升**（属性越低提升越高），并**深渊抗性永久 +5**。⚠ 布布汪深渊抗性不够 3、苏晓自己也只约 0.7~0.8 点（故都还没「半数据化」出来），这血暂放着——它揭示了 [玛丽娜](../../../人物/幽暗大陆/玛丽娜.md) 体内微量狼血的来历。
- **星核石 / 祭祀坛 / 黑暗之源 / 猎影天赋**：⚠ 催化黑暗之源的**祭祀坛**竟就在大教堂里（是大教堂的一部分），碎石下埋着一块 1 米见方的漆黑「**星核石**」（一颗星球的内核·表面星点·有隐匿波动的刻印）。祭祀坛激活后，猎杀者可以星核石为媒介、用**黑暗之源**提升灭法独有天赋「**猎影**」:**每次提升耗 5% 黑暗之源、每种天赋最多提升四次**。黑暗之源是死寂城独有资源（本源·死寂城要拿徽章去治疗所换、分支死寂城则靠杀敌获得），错过再无。
- **狼骑士徽章 / 深渊宝箱（斩狼骑士队长所得）**：狼骑士徽章 = 治愈教会十二徽章之一（属性仅「代表狼骑士队的徽章」，却是治疗所换购与开门的硬通货）；深渊宝箱有**低概率开出被封困的「爹级」器物**（惊险又刺激）。
