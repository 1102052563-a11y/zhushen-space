# 其他饰品（速查）

> 篇幅较短、未单独立页的饰品，共 2 条。

## 畸形之爱 / 庇护纪念珠 / 诅咒金币（月神大陆返园开箱 · 二十八卷38–40章）


- **畸形之爱（史诗级 · 长鞭 · 3.76 米）**：斩魂灵剑·曼的史诗宝箱开出（评分 689）——攻 210~273、易碎（耐久极低换攻 +20%）+ 神经刺激（命中放大痛觉 5 倍·几率瘫痪神经）+ 鞭痕（扇形 15 米 AoE + 几率「畸爱」失神控制）。属性 + 简介都透着污妖气质（前主马库·法兰是个虐待狂），苏晓让贝妮卖、不好意思出手给正派的神皇。
- **庇护纪念珠（契约遗物）**：斩曼的史诗宝箱另一产出——**未解除附着其上的执念前无法使用**的特殊装备（待后续研究）。
- **诅咒金币（虚空·死亡屋特殊物）**：隐藏任务随机宝箱开出——「生与死（主动·无视幸运）」抛硬币：正面随机增益（永久 +1 幸运〔限一次〕/ 每秒回 50 血 / 力敏体 +10），反面临时 **−15 幸运持续 48h（可叠加·冷却 47h）**。那「永久 +1 幸运」像美人勾引苏晓去赌、但抛出反面就被全世界针对。

## 🧥 猩红盛装 / 血羽实战 / 泯光之环


- ⚠ **猩红盛装（苏晓自创）**:⚠ **[血魂之握](../index.md) 构成鲜血战甲的阶段，以精神力操控、刺激其核心结构拟定「雷系防御」**——⚠ **永恒级装备良好的适应性自然就适应出这身高领低兜帽、腰部有抑制雷电传导术式的猩红盛装**;⚠ **⚠ 这是专门针对界雷的外部防御**（⚠ 在界雷持续倾泻下坚持五秒后完全破碎）。
- ⚠ **[血羽](../index.md) 首次实战（52卷27–28章）**:⚠ **天启六人小队围堵时，苏晓把血羽悄无声息加持到奶妈 **珞珞** 身上**——⚠ **她化身「猩红之子」（猩红双翼）追着队友治疗**;⚠ **⚠ 降雨师血条上限从 100% 变成 650% 且布满裂痕，当场喷血亡命奔逃**（⚠ 一场围堵不战自溃）。
- ⚠ **泯光之环**:⚠ **黑洞·阿兹勒三秘宝之一（另两件是黑暗之血·心灵与深渊契约）**——⚠ **由 [月女巫](../../../人物/女巫界/瑟希莉丝.md) 交给苏晓。**
- ⚠ **黑暗系鉴别**（老猎人·战利品效果从芙拉儿处夺得）:⚠ **可鉴别黑暗特性物品的来历与特性**——⚠ **⚠ 提升费用奇贵（介于普通被动与技法类被动之间），因为它 Lv.50 后可进阶为「黑暗物配制」或「入门级深渊学」**;⚠ **⚠ 苏晓决定投入资源发展**（⚠ 越到后期越能感受灭法之影与「深渊/元素」的联系·深渊方面稍有不慎就当场殒命）。

## 📦 补录（原「装备·道具」页残余条目 · 30 条）

### 昼·王之转生（传说级套装 2/3 · 项坠 · 二十六卷38章·开第2件黑王之魂宝箱所得）

「**昼之王·厄德尔斯**」（黑之王长子）的传说级套装件——银色项坠，性质是**光明**（与 [黑·王之陨祭](../../../index.md) 的暗**严重冲突**·两者的单件被动只能激活一种）。单件被动「**神圣心灵**」：生命 <20% 时 0.2 秒内回血到 50%+（每 10 小时一次·保命神技·苏晓在死寂城靠它从大 BOSS 白发女人的重击下续了命）。苏晓舍弃黑·王之陨祭的「颤栗之瞳」、激活神圣心灵。集齐 3 件（最后一件须进死寂城）才触发套装「不死意志」（生命上限 +50% / 意志 +35 / 濒死仍能战）——黑王套装这两件光暗互斥、暗示第三件起调和作用。简介「如我的牺牲可庇护帝国三百年，那么我愿坐在王座之上」。

### 意志符石（史诗级 · 装备 / 一次性道具 · 二十八卷22章·觉醒第二环奖励）

月神大陆「觉醒任务·倾听之秘（第二环）」的目标物（评分 700+·售价 39 万币·耐久 30）——原为向神灵祈祷之物，被 [灵魂语者](../../../人物/月神大陆配角.md) 倾听找到后献给月神女、藏于 3 号储藏室。布布汪以「连偷国宝逼月神女露藏处」之计偷出。**装备效果·意志守护（被动）**：随身携带可豁免部分外界意志干扰 / 入侵；**使用效果**：一段时间（30 分钟）大幅提升灵魂韧性、减免灵魂伤害 / 冲击（用后消失）。获得即激活觉醒第三环「统帅之怒」。

### 霸主宝箱产出：苏醒圣徽 / 霸主精魄（二十七卷50章·斩祭神所得）

- **苏醒圣徽（光环类装备 / 消耗品·战争世界）**：佩戴 = 触发「苏醒光环 Lv.10」（500 米内友方体力恢复速度 +20%·无消耗 24h 常驻）；使用 = 永久掌握苏醒光环 Lv.1（可升）。苏晓给 [布布汪](../../../人物/布布汪.md) 佩戴（它原本就提供冰雪女神光环·现多一道）。
- **霸主精魄（材料）**：3 颗可在轮回乐园兑换**随机霸主装备**（霸主装备无固定品级、独有、可用到死·12% 才从霸主宝箱直出）·无法出售——「运气不好用实力凑」的安慰奖。

### 守望者（成长类装备 · 项链 · 二十五卷54章·开成长类装备宝箱所得·阿姆持）

淡金色成长类项链——主动「**转换**」：把佩戴者的能量在**对立性质间转换**（光明 / 黑暗、死亡 / 生命、寒冰 / 火焰…无冷却·每次耗 1~4 耐久）。苏晓自己不敢用（青钢影能量转化不可测·作死），转手给 [阿姆](../../../人物/阿姆.md)——配合岩浆果实让阿姆获得**冰 / 岩浆双形态**，并顺带解决其「不能下水」的弱点（转冰形态即冻水面）。

### 无声之戒（金色套装件 · 「金色四耻」之一 · 二十五卷53章·开金色套装宝箱所得·已寄卖）

金色套装四耻之一（与法师残魂套争倒数第一）——副作用搞笑：佩戴让使用者**变聋 + 失明**（「无声」=听不见）。但集齐八件可「看见」地底深处 19 种颜色的稀有矿藏。轮回乐园契约者都擅杀人没人挖矿、故苏晓让 [贝妮](../../../人物/狩之影.md) 寄卖（其余顶上战场杂物已卖 173 万币）。

### 月之风华（史诗级 · 宝石 · 二十五卷22章·白胡子宝库任选所得）

苏晓获得的**第一颗史诗级宝石**——镶攻击类装备得「急速」特性、镶防御类装备得「月之护甲」特性（详细数值需镶嵌后可见）。

### 丛林猎食（传说级 · 项坠〔骨齿〕· 十九卷94章·开巴克宝箱所得·已拍卖流拍）

开 [巴克](../../../人物/巴克.md) 宝箱所得的传说级项坠（耐久 65·需真实敏 90 / 魅 80）。效果：**湿地猎手**（身处河流 / 浅水 / 泥潭则强制进入伏击、被侦测 −85%、下次攻击附敏 ×5 + 魅 ×3 的全额无视防御穿透伤害）+ **丛林伪装**（森林内可把召唤物伪装成本世界生物·同阶位无法识破）。「身处丛林则为丛林之子」——对森林系契约者是神器、但 80 点真实魅力门槛让苏晓与 [布布汪](../../../人物/布布汪.md) 都用不出威力，被苏晓默认「辣鸡」，拍卖会流拍。

### 时空主宰（传说级 · 项坠 · 二十卷1章·里德锻造）

[陵墓守卫者·里德](../../../人物/陵墓守卫者·里德.md)（希亚大陆原最强者·现于轮回乐园开「希亚铁匠铺」专心锻造）为苏晓打造的传说级项坠（评分 511·需真实智力 85 + 基础刀术 Lv.40）。里德故意只赋它**一种能力**（每件装备有「总量」·分支越多限制越多·遂把总量全凝于「空间」）：**空间斩（主动）**——消耗 10% 法力·下次斩击可穿透异空间 / 空间夹缝 / 十一次元等、造 500 + 本次斩击伤害、并**强制把敌人驱逐出所在异空间**（冷却 10 分钟）。这正补苏晓的短板：以往空间类刺客躲异空间内、他即便全开吞噬之核看到位置也奈何不得，如今一刀就能把人逼出来「按在地上摩擦」。

### 女巫指环（紫色 · 世界钥匙 · 二十卷50章·马文所赠 → 进暗鸦星）
