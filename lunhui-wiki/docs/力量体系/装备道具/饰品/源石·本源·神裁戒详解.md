# 🆕 源石·本源·神裁戒详解（第48卷续6）

- **本源 / 初始源石 / 源石**：⚠ 死寂蔓延的根源之一是一块庞大的**本源**（它产生死寂之力，否则死寂只是无源之水），神灵时代末被治愈教会封进死寂城最深的**至高圣所**（圣所钥匙开）。当年从本源上「切」下一小块凝成**初始源石**，太大、谁吞谁死，遂分成**五块「源石」**（初代被选者验证:吞源石可避免被本源吸回 + 得非凡之力 + 死后源石自剥离）。苏晓有大主教给的一块，剩四块在死寂城内**四位强者**手中——集齐五块重聚成初始源石的量，才有踏入至高圣所的资格。
- **黑王护臂（被选者证明物·吸源石）**：开死寂之门后新显特性——能**吸收源石**（源石是黑王护臂的上位级，吸够护臂会攀到上位级），提升佩戴者对死寂的抗性:吸约 3 块源石可得全天 24 小时 5 级庇护（叠庇护石约 8 级）。⚠ 唯有吸够本源之力，苏晓才能走进至高圣所（否则一进就被本源侵蚀死、庇护石无效），去做「维持现状还是挑战死亡」的抉择。
- **神裁戒 +10（不朽级·屠神器物·成长度 36.8%）**：斩罪神后属性成型——效果①**魂之生**:每点灵魂强度 +120 生命 / +0.4% 神经反射（苏晓 650 灵魂 → +65000 生命 + 230% 神经反射，均已满）；效果②**罪业之火**:近战有概率引燃敌人罪孽、灵魂燃烧（击杀新恶神则被新屠神能力替换）。成长条件:击杀极恶神灵吸其本源成长，极限**起源级**（非不朽级）。
- **自然共鸣（六星称号·极限）**：大幅提升冥想效果、冥想同时永久提升意志力。⚠ 正常冥想效率下加成平平，但配苏晓（灵魂印记 + 心之冥想 Lv.80）暴涨的冥想效率，这枚六星称号能发挥出不亚于八星辅助称号的效果。
