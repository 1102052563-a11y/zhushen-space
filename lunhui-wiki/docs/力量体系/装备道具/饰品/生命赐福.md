# 生命赐福

> 不朽级饰品。

## 💎 生命赐福（不朽级·饰品）

> 原属批次:续28 · 极暗古堡与生命晶石（51卷16–25章）


⚠ **【贪婪宝珠】的第一份战利品**——⚠ **升降梯顶部的顶栓上挂着一具干枯的尸骸**（⚠ **从残存的衣料来看，应该是这座城堡的大小姐或贵客**;⚠ **在猩红蔓延时慌忙间失足从上层的升降梯口落下，死于这顶栓的贯穿伤**）;⚠ **枯尸手中是一条线绳部分被扯断的挂坠**——⚠ **可以看出，挂坠主人在受到致命伤后，意图扯下这挂坠握在手中，以达成更快速的使用**;⚠ **苏晓包裹着晶体层的手稍微发力，包裹在项坠上已硬化的猩红结晶破碎，露出了最有价值的红宝石部分。**

| 项目 | 内容 |
| --- | --- |
| 装备效果 | ⚠ **拯救（主动）**——⚠ **立即消耗 20% 最大法力值，激活一次治疗效果，治疗当前 1.25% 最大生命值**（⚠ **绝强综合实力下，可治疗 45%～50% 生命值**） |
| 售价 | ⚠ **一般不朽级装备价格在 1900～7000 灵魂钱币**——⚠ **这东西卖 5000，属于摆在摊位上就没** |
| 买家 | ⚠ **冒险团的坦系最喜欢这东西**（⚠ **他们不怕法力值的消耗**）——⚠ **卖给神皇冒险团都不用开价，那边就会出 5300 枚以上** |

- ⚠ **没有【贪婪宝珠】他真拿不到**:⚠ **他方才相距这装备只有不到 1 米远**——⚠ **可他的感知力更擅长感知敌人，或是敌人所用出的能力。**
