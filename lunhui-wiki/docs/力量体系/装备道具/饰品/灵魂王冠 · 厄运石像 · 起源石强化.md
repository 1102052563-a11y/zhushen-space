# 🆕 灵魂王冠 · 厄运石像 · 起源石强化（第49卷续13 · 暗影世界）

- **灵魂王冠（深渊·原罪物·又称暗黑王冠）**：⚠ 苏晓从深渊宝箱开出的**爹级原罪物**——通体漆黑、存世悠久。它会向持有者投射「**万王之王**」的意识幻象（千米骸骨山 + 万族跪伏 + 多重王冠），蛊惑其戴冠、最终把持有者引向消亡（与深渊之罐「不违定律就不害人」相反）。买走它的仁兄十几天就暴毙，它便自然回来找**初始唤醒者**苏晓（深渊·原罪物皆有「找初始唤醒者」的特性·如死灵之书之于神父）。⚠ 苏晓靠意志力 200 点衍生的被动 **「无畏影」**（完全豁免原罪物与深渊滋生物造成的「意志侵袭」）扛住幻象、把它封进 [瑟菲莉娅](#) 赠的**深渊盒**（困死灵之书用的消耗品·困越久效能越弱·仅瑟菲莉娅能造）——他绝不持有原罪物，打算把它当「大礼」送给老朋友奥术永恒星。
- **厄运石像（厄运物·原罪物次级产物）**：灵魂王冠的次级产物、十公分高的无面王座石像。携带期间**幸运 −25 且持续降运势**;出售 / 破坏 / 转让都要永久扣幸运——唯独**转让给「你的仇敌」反而幸运 +2（最多 3 次）**，专为「把倒霉转嫁给仇敌」而生。⚠ 它尚未被轮回乐园公证、故影响不了「气运护身」的灭法苏晓（他本就够倒霉）——苏晓先用**圣蛇守护**（圣蛇吸厄运）把它的厄运吸掉（圣蛇难得「正常」吸了回厄运、没被撑成气球），再把它**送给仇敌 [古斯沃（耶辛格）](../../../人物/暗影世界/古斯沃.md)**、触发「转让仇敌」的幸运 +2——苏晓的**裸装幸运终于累积到 50 点**（衍生被动如幸运 20 点的「强掠之运」让他调配药剂更易出完美品级、配合七星【奇迹制造者】甚至能出「奇迹品级」）。
- **普通起源石（九阶稀缺消耗品 · 强化起源级武器）**：⚠ 苏晓晋升九阶后才发现，以前 5000 灵魂钱币随手就卖掉的普通起源石，竟是**强化起源级武器**的稀缺材料。起源级强化不同于不朽级「硬堆不朽石」，而是**蜕变性递增**——以起源之力引动装备内的起源之力、经强化机辅助完成量变:+1 到 +10 每次耗**一颗起源石**，但基础成功率反而更低（不朽 +8 约 30%、起源约 17%）。⚠ +10 以上需用**稀有起源石**（起源石·残裂 / 银皇后 / 混沌之火——越上乘成功率越高）、或直接拿**起源石·世界碎片**（必定 100% 成功）。起源石的镶嵌是「融入镶嵌部位、不可剥离」（故那位把起源石·世界镶矿铲上的仁兄，反而没人惦记来抢）。
