# 空之戒（紫 · 晓制式戒指 · 九卷18章）

[晓](../../../势力/index.md) 组织发给 [苏晓](../../../人物/苏晓.md) 的**正式成员戒指**（紫·评分 101·剧情物品·与穿戴装备不冲突·**任何情况不能摘**·上任主人是大蛇丸〔被 [宇智波鼬](../../../人物/宇智波鼬.md) 劝退〕故一直空缺·戒面「空」字）。**效果·幻灯身之术（主动）**：在特定位置构建一具幻影分身（最多 36 小时·每次耗 100 法力）。更关键的是它是**远程通讯 + 定位器**——晓成员散布世界各地、靠它**互相感应位置**、由 [长门](../../../人物/长门.md) / [小南](../../../人物/小南.md) 用「幻灯身之术」召集幻象分身、发布任务（没它晓就是一盘散沙）。苏晓戴在右手（左手是金属护臂）。未获所有权·不可出售 / 带出。
