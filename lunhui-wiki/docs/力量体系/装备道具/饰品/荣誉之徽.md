# 荣誉之徽 → 荣誉商店

> 主线任务奖励，开启荣誉商店等级体系。

## 🏅 荣誉之徽 → 荣誉商店 Lv.15（主线任务·凛冬领主奖励）

> 原属批次:📚 续25 · 风海大陆收官 · 原罪之书与荣誉商店（50卷55–64章）


⚠ **主线任务完成度 950%（严重溢出）**换来的【**荣誉之徽**】——使用后可将荣誉商店提升到最高等级。⚠ **警告:荣誉之徽极其罕有，每名猎杀者终生最多可获得一枚，如非必要请勿将此物品交易给其他猎杀者 / 请勿出售！** ⚠ 苏晓的荣誉商店由 **9 级直升 Lv.15（最高）**——他感觉**这是自己在风海大陆获得的最大收益**。

- ⚠ **核心变化**:⚠ **荣誉勋章将转化为时空之力**（⚠ **不按常规价值体系恒定，而是以 1:10 的比例兑换**——苏晓的 12 枚勋章换得 **120 盎司**）；可兑换**原初碎片、基础被动技能卷轴宝箱、满评分霸主装备、随机属性源质、随机属性本源**等（库存有限，**每次荣誉商店刷新库存也刷新**）。
- ⚠ **药剂补给额度 2/2 → 5/5**（每个世界进度免费刷新，⚠ **溢出的额度无任何补偿**）；⚠ **品质大幅度提升**；⚠ **可免费补给的药剂数量从六种提升到十二种**。⚠ 以下药剂**均有极高的加成阶位，对绝强者、至强者均有所描述的 100% 效果**:
    - **1.荣誉复苏药剂**（额度×2）：立即恢复 100% 生命 / 100% 法力、解除所有减益、快速愈合全身伤势（⚠ **可让受损的脏器与脑组织再生**）。⚠ **最大携带上限为 1**（此为最高优先度公证）；⚠ **禁止出售、转让、二次调配、无特殊理由丢弃**。
    - **2.力量药剂**（×1）：真实力量 +10%，600 秒 · **3.秘银药剂**（×1）：真实敏捷 +9%，690 秒 · **4.龙心药剂**（×1）：最大生命 +15%、肉体强度 +10%，600 秒 · **5.法魂药剂**（×1）：真实智力 +5%、每秒回 5% 最大法力（或其他类型自适应身体能量），300 秒
    - **6.狂兽药剂**（×1）：生命上限临时 −25%、真实力量 / 敏捷 +10%、获**次级霸体**，350 秒 · **7.属性药剂**（×1）：提升 20%～25% 自适应属性攻击，400 秒 · **8.升华药剂**（×1）：所有技能冷却 −35%、使用技能时 10% 概率刷新该技能冷却，700 秒
    - **9.庇护药剂**（×1）：⚠ **捏碎立即生成一个与你生命值等量的强韧护盾**，10 秒或被击碎 · **10.幸运药剂**（×2）：幸运 +10 点，**持续一个世界进度**（库存 3）· **11.灵魂药剂**（×2）：缓慢恢复灵魂能量，持续一个世界进度（库存 10）· **12.全能药剂**（×5）：⚠ **可融入 99.9% 的药剂内并完美复刻成此药剂**（⚠ **库存 0，苏晓很感兴趣却没得换**）
    - ⚠ **第 1～9 种为恒定、永久不会变更；第 10～12 种将在每次荣誉商店刷新时随机选取。**
- ⚠ **苏晓的选择**:一瓶【荣誉复苏药剂】+ 一瓶【灵魂药剂】，最后 1 个额度在【龙心药剂】与【庇护药剂】间选——⚠ **他选了庇护药剂**。⚠ **理由**:他现在有 **86 万点生命值**，而⚠ **超过 95% 的护盾其实都是 0 防御护盾**（耐久度多高就挡多少伤害，且**敌人的攻击穿透、攻击阶位高的话会对护盾造成极高伤害**）——⚠ **苏晓的刀类武器伤害阶位是 +14，他一刀斩在 0 防御护盾上能斩出 1400%～1500% 的伤害，再算上锋利度与各类穿透，基本都是一刀敌人就爆盾**。⚠ **强韧护盾则不同:有基础防御力与强韧性**，虽挡不住「伤害阶位」，⚠ **但对穿透与穿甲的抗性都不错，面对极强的攻击虽可能被打扁，但一定会尽最大可能保护里面的使用者**。⚠ **【龙心药剂】的价值**:它是【龙血药剂】的升级版，加成数值看似没龙血多**但阶位高**——⚠ **晋升绝强后用龙血药剂可能连 5% 生命值都提升不了，但用龙心药剂肯定是足额加成**。⚠ 「其实不仅是庇护药剂强力，**药剂列表内就没有弱的药剂，只是适不适合而已**——⚠ **这里面的药剂效果都很朴实，但各系猎杀者都能在里面找到适合自己的药剂**。」
