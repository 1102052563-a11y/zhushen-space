# 🆕 天赋觉醒之书 · 世界之环 · 命运主宰起源级 · 噩梦道具（第49卷续15 · 暗影世界）

- **天赋觉醒之书·灭法（灭法专属·权限物品·估值 1400 盎司时空之力）**：⚠ 斩 [告密者](../../../人物/暗影世界/告密者·索恩.md) 所得（1500 盎司结算的主体）。它**不能直接让灭法天赋觉醒**，只能触发「灭法专属天赋·猎影」的**天赋觉醒任务**——完成后猎影将觉醒至 **SSS 级（天赋上限）**，此为灭法之影的「终极能力」，众多灭法成为绝强者的秘密就在其中。⚠ 觉醒地点在**永光世界**（先代灭法囚禁灭世级族群之地·除灭法外无人会去、极安全，也是对新一代灭法的考验），那里有苏晓亲手封进去的**蛀世 / 银皇后 / 寄星蟹**等仇敌。物品**警告:以现有战力立即使用、任务完成概率极低**。
- **世界之环（世界级 · 评分 3280）**：⚠ [湛蓝熔炉](#) 融合**世界三件套 + 命运之血 + 沉淀琉璃 + 世界墨迹**而成的指环。佩戴即成**世界之子（伪）**（85% 世界之子运势 + 世界眷恋）；效果 1 按魅力 35% 加幸运（最高 +35）、效果 2 击杀影响世界安危者存世界之力、效果 3 存满世界之力后触发**阶段性全属性 +10、全能力等级 +9~12**（可多次触发）。⚠ 弊端:一旦离开提升所在的世界，力量十不存一（局限性越大、在本世界越强）——苏晓拿它当**吞噬者争夺战第二阶段的争夺品**。
- **命运主宰（起源级 · 评分 3000+++）**：⚠ 浸泡在 [幸运女神](../../../人物/幸运女神.md) 转化的幸运源血中晋升至**起源级**（原晋升需求只要「微量幸运神性之血」·苏晓直接用纯幸运神性源血）。基础幸运 +12;核心「气运之力」耗一枚灵魂结晶大、临时 +45% 幸运 + 绝对运势 20 分钟（加成随刻印的「强者之名」气运重量递增）；「极运」开宝箱 10% 必出最高价值物、未触发则累积至 100%；⚠ **强者之名上限 +3（增至 3/3）**、可继续冲更高品级；「不灭之运」遭命运/因果系暗袭时**以灭法运势增强 5~10 倍回弹施术者**（对方须硬扛或付 3 倍资源消灾），且可转移到一件无特性起源级装备上。
- **湛蓝熔炉（天启乐园世界级 · 可用 1/3 · 评分 2800 每次 -150）**：⚠ 从噩梦岛宝库缴获（某天启九阶契约者被噩梦之王所杀掉落）。激活后可**融合特性相近的装备**，融合期间加入的稀有物品 / 珍贵材料越多、成品越强——苏晓正是用它熔出了世界之环。
- **噩梦指针 / 造梦石（不朽级 · 暴食族赠物）**：⚠ **噩梦指针**是暴食族护身怀表（评分 1500·需意志 180 + 理智 350），在噩梦区域可选「宝藏」或「生路」指向、每区域最多用两次（第三次永久损坏）；**造梦石 ×3** 使用后造 3~5 小时的美梦 / 噩梦 / 炼狱噩梦、把 1~3 目标精神体拖入（死于梦境仅精神萎靡、本体不死）。
- **半融的脂肪蜡 / 神圣蜡烛（虚空异物品 / 一次性道具）**：⚠ **半融的脂肪蜡**（产地虚空夹缝·不可销毁）点燃后引来虚空异存在 **[烛女](../../../世界/生物万族/index.md)**（苏晓猎杀噩梦之王的关键·可用邪神血熄灭、仅剩一小截）；**神圣蜡烛**点燃后驱散 5 米黑暗邪祟、形成直径 5 米绝对神圣庇护区（苏晓借它在噩梦领域内自保、不知如何熄灭只能任其燃尽）。
- **铁炼邀请函 / 铁之试炼（九阶最难关卡）**：⚠ 九阶契约者的**真实力量属性**卡在 300 点是最难突破的壁障（九成契约者卡死于此），突破需极稀有的**铁炼邀请函**（不想要也只能 6530 盎司卖给轮回乐园）+ 完成**因人而异的铁之试炼**（苏晓作为灭法的铁之试炼正是去**永光世界**）。跨过此关，便逼近瑟菲莉娅、凛风王、老恶魔·沃波尔、白牛、圣女座之流。打开鹿神**黄金罐**（纯肉体力量掀盖·不许动用主动能力）便需 300 点以上真实力量——苏晓取巧以「黑暗眼」增益封盖 + **暗之吞噬**缓慢吞噬，倒出 40 盎司恶神源血、空罐委托 [凯撒](../../../人物/尼古拉斯·凯撒.md) 三成分账卖给黄金神教。
- **嗜血战甲（脱离装备品质 · 直奔准爹级）**：⚠ 本是仅次于巅峰永生之神的**深渊滋生物**、被幽暗大陆神教打造成六件套战甲（其余五件皆用于封印它）。吞了暗刃后再**吸尽百米噩梦古树的本源**，套装五件炸了四件（最后一件狼之意志已破碎）——它不再是套装、不再有装备品质，因基础是强大深渊滋生物、**向「准爹级」迈进比先古面具快得多**;再喂它一颗**原罪之芽**（深渊级·含「原罪特性」），可能下个世界就追平先古面具。
