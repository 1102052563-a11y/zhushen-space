# 黑暗宝珠 / 贪婪宝珠

> 两颗成对出现的宝珠。

## 💠 黑暗宝珠 / 贪婪宝珠

> 原属批次:续28 · 极暗古堡与生命晶石（51卷16–25章）


⚠ **来自上个纪元的宝库**（⚠ **永光世界还是超脱·原生世界时，那个文明中某个势力的一处宝库**）——⚠ **[食暗者](../../../人物/风海大陆/黑眼存在.md) 赶往黑铁城时岩石地面突然塌陷，掉到洞窟内，定睛一看，前面就是一扇宝库门**;⚠ **宝库内货架都已腐朽**——⚠ **一颗灵魂晶魄刚被单手托起就破碎，化为粗糙的晶脂残渣顺着指缝落下**（⚠ **可见这看似不算特别大的宝库，来头必然不小**）。

⚠ **两颗宝珠藏在王座扶手的暗格里**:⚠ **一颗暗金色、一颗漆黑色，质地有些类似于黑曜石但密度更高**——⚠ **而那张王座**（⚠ **犹如被火焰灼烧过、有融化痕迹**）⚠ **触发了苏晓的直感预警:不能碰。**

!!! tip "⚠ 食暗者的性格说明书"
    ⚠ **你要是告诉它，这王座是怎样罕见的宝物，它死活都不会靠近，保持防备**——⚠ **可你要是告诉它别碰，那它一定得研究用食指戳一下。**（⚠ **结果:呜呀一声，食暗者趴在石椅旁身体一下下抽搐，从眼中的白色线条看，都有点翻白眼了**——⚠ **它触发了上面的防御秘纹**;⚠ **但因秘纹存在太多万年，触发一次后许久都无法触发第二次**——⚠ **⚠ 反倒让苏晓能安全拿到这两颗宝珠**）

| 【黑暗宝珠】 | 内容 |
| --- | --- |
| 产地 | ⚠ **幽暗大陆** |
| 装备效果 | ⚠ **黑暗滋养（被动）**——⚠ **随身携带，或装备于特殊饰品栏，能以缓慢的速度滋养自身的暗特性** |
| 简介 | ⚠ 「幽暗大陆上工匠们的得意之作。」 |

| 【贪婪宝珠】 | 内容 |
| --- | --- |
| 产地 | ⚠ **幽暗大陆** |
| 装备效果 | ⚠ **贪婪之兆（被动）**——⚠ **随身携带时或装备于特殊饰品栏，如果周边 100 米内存在秘宝，此物品将放出独有波动，并放出秘宝之辉，向秘宝所在的位置飘散** |
| 装备减益 | ⚠ **贪婪厄运（被动）**——⚠ **将会大幅度降低自身的运势**（⚠ **因你的运势下限值过低，此减益对你无效**） |
| 简介 | ⚠ 「对秘宝贪心者总是会偶尔倒霉，当然，**如果你已经足够倒霉，那这宝珠也拿你无可奈何**。」 |

- ⚠ **为什么它不多余**:⚠ **「狩猎视野」介绍中虽有标注能看到秘宝，但这特性更像是附带**——⚠ **假设「狩猎视野」侦测灭世级生物的能力强度是 3000，那寻找秘宝的效能也就是 5 左右**;⚠ **而【贪婪宝珠】被制造出就是专门用于感应秘宝的。**
- ⚠ **产地为何是幽暗大陆**:⚠ **幽暗大陆也曾是超脱·原生世界，这里也是**——⚠ **两个超脱世界彼此有来往，实属正常。**
- ⚠ **【黑暗宝珠】的归属**:⚠ **巴哈的无良发作**——⚠ 「**这玩意提升暗属性，挺适合食暗者啊。**」（⚠ **食暗者对巴哈好感度 +5**）⚠ 「**不过 [暗之女](../../../人物/永光世界/暗之女.md) 也能用这东西，老大，以你和暗之女的关系，那肯定是给暗之女。**」（⚠ **好感度 −5**——⚠ **它虽然有点憨，但还是知道巴哈这家伙在故意逗它**）;⚠ **⚠ 巴哈这就是胡扯了**——⚠ **暗之女不用加强自身的黑暗特性**:⚠ **她的黑暗特性并非苦修得来，而是她在封印着一件至暗之物**;⚠ **最终苏晓把【黑暗宝珠】抛给了食暗者**——⚠ **倒不是因为其他，若非对方触碰石王座，他还真不一定会选择冒险获得这两颗宝珠**（⚠ **从食暗者身上还冒着的灰雾来看，触碰石王座可不是件轻松的事**）。
- ⚠ **后续**:⚠ **食暗者在酒馆里把【黑暗宝珠】含嘴里、还做出干呕状被苏晓看到**——⚠ 「**我牙疼，看什么！**」;⚠ **待苏晓闭目养神时，它终于把【黑暗宝珠】吐出来**——⚠ **并决定，以后再也不把这玩意含嘴里了。**
