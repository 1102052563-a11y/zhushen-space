# 魂 · 王之轮回

> 斩黑之王所得，黑王套装就此集齐。

## 魂·王之轮回（传说级套装 3/3 · 饰品 · 二十六卷58章·斩黑之王所得 · 黑王套装集齐）

> 原属批次:装备品级


黑王套装最后一件——「**夜之王**」的链饰、**特殊装备**（同 [命运主宰](../index.md)·随身携带即生效）。单件「**魂之共鸣**」让 [黑·王之陨祭](../index.md) 的颤栗之瞳与 [昼·王之转生](../index.md) 的神圣心灵**同时存在**（调和原本互斥的光暗两件）。集齐 3 件**激活套装效果**：① **不死意志**（生命上限永久 +50% / 意志 +35 / 生命 <5% 才濒死、濒死仍能战）；② **昼之时**（随从 / 召唤物 / 使魔全属性永久 +10%·永久 +3 召唤位——布布汪 / 阿姆全属性 +10%）；③ **死寂降临**（套装终极·主动·苏晓不能动、以己为心 300 米内死寂城高速同化·敌方减速 99%〔无判定〕+ 神经反射 / 动态视力 −80% + 每秒掉 7~20% 生命上限·持续 10 秒·每用一次死寂城消失一部分、净化完毕后该能力质变·CD 10 天）。这是黑王套装集齐的标志（苏晓闯死寂城斩黑之王的终极回报）。
