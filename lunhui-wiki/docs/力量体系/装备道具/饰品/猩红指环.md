# 💍 猩红指环（伪）

| 项目 | 内容 |
| --- | --- |
| 产地 | ⚠ **幽暗大陆·死寂城·灰烟工坊** · 类别 ⚠ **指环** |
| 耐久度 | ⚠ **20/390 点**（⚠ 估测还能用四五次） |
| 装备前置 | ⚠ 意志力 300＋、猩红抗性 30＋（⚠ 猩红抗性不足但意志力够，可承担侵蚀暂时佩戴） |
| 效果 | ⚠ **佩戴后可在一定程度上操控猩红之力** |
| 简介 | ⚠ 「由于暴君对猩红君主的狂热崇拜，他委托灰烟工坊仿造猩红君主的指环……暴君最终死于猩红的侵蚀之下，纵使有强大的猩红适应力，**却忘记了猩红的根源来自何处**。」 |

- ⚠ **它是苏晓进猩红宝藏区域时，被暴君骨干的意念残留「认错人」奉上的**（⚠ 因苏晓带着「灵魂王冠」）——⚠ **详见 [暴君·图尔](../../../人物/暴君·图尔.md)**;⚠ **苏晓靠它从侧后门开启猩红圣殿的腥红之雾拱门。**
