# 曙光环戒

> 曙光级装备的代表件。

## 💎 曙光级装备:【曙光环戒】


来自 [神父](../../../人物/神父.md) 的「沉厚的猩红宝箱」，重度破损、经三大传说铁匠＋里德修复（⚠ 因重度破损，修复后已损失部分特性、排行有所降低）。

!!! tip "【曙光环戒】（曙光级·三之位·30000+ 评分）"
    **装备需求**:无（⚠ 使用前需因果适配，但**可自由交易**）<br>**效果1 · 环戒锚定（被动）**:可选定当前所佩戴的装备，总计五件<br>**效果2 · 无瑕之环（核心）**:根据五件装备的**锚定评分**决定加成，并在**力量／窃夺／唤醒**三选一中定型——⚠ **一旦抉择将无法以任何方式变更**，即便失去后重新获得

**三种特性**:

| 特性 | 效果 |
| --- | --- |
| 力量 | 按五件装备的总强度提升自身战力 |
| 窃夺 | 可对任何生灵**永远窃夺其 50% 运势**并成功激怒该目标；杀死此目标则窃夺其剩余运势。⚠ 苏晓毫无兴趣——他运势已 −2000 万，就怕这提升是按自身运势算的（那样窃夺 10 万点，他就变成 −2010 万） |
| ⚠ **唤醒**（苏晓所选） | 根据**当前行为**唤醒一个此范畴内的具象，以消耗时空之力维持其存在；此具象无法交涉，只能接受一定信息，属于**除保留最擅长能力外不做任何无意义行为**的特殊存在——**「大师具象」** |

⚠ 苏晓的锚点五件:**【起源石·世界】＋【斩龙闪】＋【铁血暗狼】＋【命运逆转】＋【魂·王之加冕】**（前两件选定时环戒就亮起微光，五件选完后环戒一声嗡鸣、看起来竟有再度破碎的可能）→ **锚定评分 1587920 点**。

**升级后的效果**:根据使用者当前的行动，召唤出对应的「大师具象」——⚠ **根据锚定评分的增益，此大师具象必定是该领域的绝对最强者**（无任何直接战斗、辅助战斗甚至间接参与战斗的能力；冷却时间根据上次进行的「行动因果」衡量）。

⚠ **首次使用**:在天启乐园装备强化大厅按上强化机后激活——地上出现代表曙光乐园的「Ξ」漆黑烙印，构成的身影身材瘦弱、俨然一副猥琐小老头模样。**那是以 [凯撒](../../../人物/尼古拉斯·凯撒.md) 为锚点所构成的「大师具象」。** 结果就是斩龙闪 **+21**，以及苏晓被天启乐园强行驱逐。

⚠ 苏晓之所以敢这样用，是因为**这是曙光乐园公证的装备**——若是其他乐园阵营公证的，循规蹈矩地使用其强度必名不副实，可能连该乐园的前三十都排不进去。
