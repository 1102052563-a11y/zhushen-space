# 断魂影之石 · 残缺

> 三石合成所得的残缺影石。

## 断魂影之石·残缺（传说级 · 宝石/特殊道具 · 十七卷87–88章·三石合成所得）

> 原属批次:已收录（截至第20章）

灭之石 + 法之石 + 影之石合成而来的**传说级宝石**（评分 530·顶格）。**效果**：镶嵌至武器后**锋利度 +28**；**可继续合成**——集齐 **5 颗「断魂影之石·残缺」即可合成「断魂影之石·完整」**（残缺是镶嵌类宝石、**完整则是「力量传承宝石」**·疑承载灭法之影的某种力量传承）。苏晓已将首颗镶上斩龙闪（合成镶嵌态宝石无风险、合成后仍保持镶嵌·咨询乐园 1000 币确认）。「**断魂影**」之名屡屡出现、苏晓推测其指向「斩魂」之力（待解）。

- **🆕第四颗到手（三十一卷4–22章·深渊世界）**：[深渊世界](../../../世界/任务世界/深渊世界-塞拉帝国.md) 的圣裔称之「**魂之石**」（唯一能抑制不死者生命剥离 / 镶入灵魂）——[老公爵](../../../人物/深渊世界群像.md) 委托苏晓寻找，古神 [厄休拉](../../../世界/生物万族/index.md) 透底**它就在圣裔王 [亚历山](../../../人物/深渊世界群像.md) 的头颅里**（深渊里那具「握石骸骨」只是厄休拉随手摆的恶作剧）。苏晓斩亚历山取出**第四颗**断魂影之石——**已集齐 4 颗**，差最后 1 颗即可合成「断魂影之石·完整」、获「断魂影」能力（其分支即 [猎魂之镰 / 魂镰影](../../技能/index.md)）。
