# 浴血疯魔

> 猎杀者公认的「疯子戒」——契合度 95% 的死亡陷阱。

⚠ **本页合并了原「装备·道具」页中散落 4 处的记载**，按卷序排列。

## 😱 浴血疯魔（满评分·霸主装备 · 刷新都刷新不没的那个）

> 原属批次:📚 续25 · 风海大陆收官 · 原罪之书与荣誉商店（50卷55–64章）


⚠ 苏晓以 **10 颗特大【霸主精魄】**（[嘟嘟咕咕](../../../人物/画中世界/嘟嘟咕咕.md) 用 8 颗超大 + 26 颗新得的合成到极限，⚠ **每颗有苹果大小、看似比之前小了却更加凝实，外部黑色能量包裹、内部透出深沉的暗红色**）兑换满评分霸主装备。⚠ **检核到所提交霸主精魄的品质，本次随机兑换的霸主装备仅会在满评分·霸主装备中选择**；⚠ **如不满意可放弃，最多进行两次的随机抽取，所抽取范畴依旧为满评分·霸主装备**。

- ⚠ **抽取结果:浴血疯魔（封印全部解除状态）**——⚠ **就算是苏晓，都眼角微不可见地抽动了下**。⚠ 他忘记具体是什么时候，总之是很久之前，⚠ **这玩意就一直出现在他的荣誉商店内，刷新都刷新不没**；⚠ **直到 9 级的荣誉商店，这玩意依旧在里面，卖 3 枚荣誉勋章**（想来那是**有多重封印状态的【浴血疯魔】**）。⚠ **这次荣誉商店提升到最高级后他查看，没看到浴血疯魔，还认为终于算是把这玩意给刷新没了**——⚠ **眼下竟然用【霸主精魄】给随机兑换出来。**
- ⚠ **它的真面目**:⚠ **此霸主装备极为特殊，初始状态下为第三梯阶霸主装备**，⚠ **但可以通过向轮回乐园支付霸主精魄，陆续解除此装备的几重封印，直至其封印全部解除**——⚠ **其强度为轮回乐园的霸主装备库存中，最顶尖霸主装备之一**。
- ⚠ **「至于该如何处理，那还用问吗，当然是继续随机」**——⚠ **第二次抽取:浴血疯魔（封印全部解除状态）**。⚠ **苏晓的目光逐渐凝重**:两次随机抽取都是这玩意，虽说满评分·霸主装备的库存不多，⚠ **但再次看到这玩意，他有点眼晕**。⚠ **机会只剩最后一次**——苏晓觉得**有必要联络下凯撒让对方来救场**:⚠ **他有种感觉，要是继续这样抽取，今天怕是必定获得这【浴血疯魔】。**

## 😱 浴血疯魔 · 契合度 95% 的死亡陷阱（附:如何优雅地甩锅）

> 原属批次:📚 续26 · 返园爆改 · 终极·基础被动与三宗师破 Lv.80（50卷65–69章）


⚠ 苏晓连抽两次都是它，第三次抽之前他先想通了一件事:⚠ **哪怕这次规避了，那今后抽霸主装备时怎么办？**

- ⚠ **他的推理**:⚠ **抽取霸主装备的确是随机，但也是有优先度的随机**——例如**总计 10 件装备中，他抽到一枚近战系霸主装备的概率，肯定要比抽到霸主弓或霸主法杖的概率高出 10% 以上**。⚠ **眼下他连续两次抽到【浴血疯魔】，只可能是因为一点:他与这霸主装备的契合度最起码在 95% 以上。**
- ⚠ **为什么不能要**:⚠ **他听 [团长](../../../人物/星空座/团长.md) 提及过——在团长还在变强的那个时代，这装备就很有名:基本上，谁获得谁死**。⚠ **别忽略很重要的一点:团长说的是在荣誉商店内兑换**——⚠ **也就是说，曾获得这东西的都是猎杀者，但无一例外，获得这装备的猎杀者全部身亡。**
- ⚠ **契合度的来源**:苏晓估测**应该是自身的血气，以及 −19 点的魅力属性，导致他与【浴血疯魔】超高度契合**——⚠ **因契合度太高了，这玩意才一直被刷新在荣誉商店内。**
- ⚠ **解法·历史清单**:他**激活荣誉商店但并未查看物品列表，而是在各类权限中寻找**——⚠ **很快找到最高级荣誉商店的一项权限:一种类似于历史清单的功能**（⚠ **里面能看到以前出现在荣誉商店内的物品，例如【黑枫树·叶片】等**）。⚠ 他先以 **30 盎司时空之力兑换 3 枚荣誉勋章**，⚠ **转而就用这 3 枚荣誉勋章，在历史清单中把处于多重封印状态下的【浴血疯魔】给兑换出来**——⚠ **因为那是第三梯队霸主装备的强度，兑换价格只有 3 枚荣誉勋章。**
- ⚠ **甩锅**:一枚**犹如被鲜血染红的戒指**出现，他**以封盒啪的一声将其装上**、起身出门——⚠ **刚出专属房间，就看到满脸奸笑的凯撒**:「**我亲爱的朋友，你有什么东西要送给凯撒？**」（⚠ **贼眉鼠眼地搓了搓手，很是期待**）⚠ 「一枚戒指，**这戒指有些危险**……」——⚠ **不等苏晓说话，封盒已经到了凯撒手里**；⚠ **随着凯撒打开封盒，里面的【浴血疯魔】竟然自行飞起来、直奔苏晓而来**——⚠ **叮的一声脆响，【浴血疯魔】被凯撒用深渊之罐罩住**，⚠ **之后这厮心满意足地离开。**
- ⚠ **然后苏晓回房关好门，激活最后一次抽取**——⚠ **他能 100% 确定，这次抽不到【浴血疯魔】。**

## 💍 浴血疯魔（霸主级·猎杀者公认的「疯子戒」）

> 原属批次:续34 · 疯魔戒、封魔刀鞘与黑暗之血（52卷12–21章）


⚠ **苏晓上次得到它就想办法送走（给了凯撒），这次开【霸主宝箱】又开出来**——⚠ **它盯上苏晓，是因为「本源生命力＋理智值＋意志力＋血气值」四项相加，苏晓名列前茅**（⚠ **噬主:每吞噬一名持有者就成长，所吞者这四项越高、成长越大**）:

| 浴血疯魔（霸主级·戒指·满评分 6000） | 效果 |
| --- | --- |
| 装备需求 | ⚠ 魅力 10 点以下·生命值 50 万以上 |
| 疯魔（核心·主动） | ⚠ **真实力量、体力临时 +50%·攻速/移速 +100%·85% 攻击汲血** |
| 代价（激活期间） | ⚠ **每秒 −1% 最大理智值 / −1% 意志力 / −1% 最大生命值** |
| 装备减益·代价（被动） | ⚠ **战后按疯魔秒数，每秒永久吞噬 5000 点本源生命值 ＋ 10 点理智值 ＋ 2 点意志力**（⚠ 60 秒疯魔＝生命上限永降 30 万、理智 −600、意志 −60） |
| ⚠ 双向绑定 | ⚠ 需双方都解除誓约才可解绑 |

- ⚠ **它最棘手的一点**:⚠ **给持有者一个「虚假的更高战力上限」**（⚠ 平常绝不用、危机时刻用它翻盘→恶性循环）——⚠ **⚠ 但这招对苏晓完全无效**:⚠ **他连原罪物给的更高虚假上限都能克制住不用**（⚠ 除对付猩红君主那次·那其实是原罪物打原罪物）。
- ⚠ **⚠ 苏晓的骚操作**:⚠ **他不戴疯魔戒——而是用「战斗义肢」给魔灵造一条晶体右臂（由他的能量晶化构成、与他右臂以灵影线相连、100% 受控＝等同他的右臂），让魔灵戴上**;⚠ **⚠ 激活 1 秒疯魔状态后，一刀斩断这条晶体手臂、连戒封进原罪之书第六页**——⚠ **疯魔戒隔着「原罪之书」这道因果墙讨不到代价，就算突破了，讨要对象也是魔灵而非苏晓**（⚠ 代价讨要有时限·十多小时后账目清零）;⚠ **⚠ 用法:配合力量源质，疯魔状态下力量/体力可临时冲到 800 点**（⚠ 每次最好不超 5 秒）——⚠ **由此可提前使用「需 800 点属性的顶尖永恒级·滋养装备」，领先一个版本。**

## ⚙ 力量领主实测 / 浴血疯魔白嫖

> 原属批次:续38 · 曙光大盾、界之圆环与神裁永恒级（52卷62章–53卷11章）


⚠ **【[力量领主](../index.md)】需真实力量 800 点**——⚠ **苏晓靠 [浴血疯魔]（疯魔状态临时 +50% 力量/体力）卡上去**;⚠ **⚠ 而浴血疯魔的代价极惨（每秒疯魔时间永久吞噬 5000 生命值＋10 理智＋2 意志力·⚠ 60 秒疯魔战后就永久 −30 万生命/−600 理智/−120 意志）**——⚠ **⚠ 苏晓的白嫖操作**:⚠ **放出魔灵、用「战斗义肢」给魔灵构成晶体右臂、末端灵影线接入自己右臂（装备判定上算他的手臂）→ 操纵魔灵戴浴血疯魔、激活疯魔状态→ 增益经灵影线传到他手上→ 力量体力达 800 点极值→ 佩戴力量领主→ 一刀斩断魔灵晶体手臂→ 用原罪之书把浴血疯魔封到第七页**（⚠ **想讨代价？先突破原罪之书层层封印＋因果封困、再把魔灵反噬——被封困的浴血疯魔像饥饿的恶兽，只能不甘咆哮**）。
