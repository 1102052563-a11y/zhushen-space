# 🆕 治疗所徽章商店 · 十二徽章体系（第48卷续8·补）

⚠ 死寂城**治疗所**（虚空之树公证）的阵营商店，不走声望值，而以神灵时代硬通币「**古代金币**」+ **十二种徽章 / 苍白勋章**换购。十二徽章代表治愈教会十二高层，如今有用的剩七枚:**圣歌团 / 狼骑士 / 圣女 / 猎人（大主教）/ 月光（圣祭祀）/ 离群战士 / 罪人（罪孽集合体）**；老虫子（瓦迪·特雷奇）的秽虫徽章已作废。地位最高的**圣歌、狼骑、圣女三徽章**可换黑暗之源，还各能换特性物:

- **狼骑士徽章 → 狼血**（或 76300 古代金币）；**圣歌团徽章 → 团服 + 誓约之匕**（穿团服、把匕刺在圣十教堂门上即可挑战 30 名圣心之一、胜则顶替入团）；**圣女徽章 → 证明物**（去古代实验所找半神 [钢铁制造者](../../../人物/幽暗大陆/钢铁制造者.md)、心情好会送人去「深渊战场」）。
- 其余商品:**起源石·混沌之火**（罪人徽章 or 45000 古代金币·倾向火系）、**离群战士之魂血**（离群战士徽章 or 75000）、**技法之魂·暗**（猎人徽章·能量操控系暗系·放暗魂融敌灵魂叠几百种负面状态·值 10 万灵魂钱币）。
- **苍白勋章**：荣誉象征（非地位），1 枚可换 1000 古代金币，疑为治愈教会颁给有功苍白猎手——击杀苍白猎手有概率掉落。
