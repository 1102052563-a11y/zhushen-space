# 神秘之眼

> 已激活的神秘之眼。

## 👁 神秘之眼 · 已激活

> 原属批次:续28 · 极暗古堡与生命晶石（51卷16–25章）


⚠ **苏晓一直怀疑神秘之眼是不是少点什么**——⚠ **在遇到那只金属章鱼后，他确定了**:⚠ **神秘之眼的核心部分丢了，所以才这么难以运用。**

- ⚠ **金属章鱼**（⚠ **[机械制造者](../../../人物/永光世界/机械制造者.md) 保管的封印物之一**）:⚠ **能钻入生灵的脑髓中，操控生灵行动，以及赋予这生灵操控他人心智的能力**——⚠ **每隔几月，机械制造者的学徒们都要喂食它些罕有的金属资源**;⚠ **⚠ 它不是原装核心，但与原本的核心有一定的相似性**——⚠ **在苏晓取出神秘之眼时，神秘之眼内蔓延出那针对金属章鱼的吸附性，就代表这猜测没错。**
- ⚠ **它的死法**:⚠ **邦的一声，金属章鱼用一根机械触足把神秘之眼踢飞**——⚠ **苏晓指向被踢飞的神秘之眼让它捡回来，结果它直奔苏晓面门而来**;⚠ **当！一把晶体短刀钉在它即将途经之处**——⚠ **它再次踢了下神秘之眼，坚定地表示不会进入神秘之眼被束缚**（⚠ **苏晓也得像机械制造者的几名弟子一样好吃好喝地供着它**）;⚠ **⚠ 见此，苏晓取出「原罪之书」，不给它反应的机会就将其封到第五页**——⚠ **半小时后解开封印，它已经变得软趴趴**;⚠ **被关在原罪之书时，它的精神波动被相邻的灵魂王冠给灭杀，这玩意竟然半死了**——⚠ **苏晓取出几种金属特性的材料调配药剂慢慢倒在它身上**;⚠ **结果这危险物身躯颤动了下，然后彻底没了气息**——⚠ **⚠ 这害死过很多生灵的东西，居然就这样死透了。**
- ⚠ **对比**:⚠ **苏晓暗感可惜，他还认为这东西挺皮实**——⚠ **不得不说，熔金魔老哥真的是大有来头**;⚠ **被关在原罪之书内那么久，除了头像的痛苦面具越发生动外，竟没消散。**
