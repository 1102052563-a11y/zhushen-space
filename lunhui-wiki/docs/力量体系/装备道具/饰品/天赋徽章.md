# 天赋徽章

> 天赋类的天花板，血之兽由此蜕变 SSS。

## 🏅 天赋徽章（天赋类天花板）· 血之兽 SSS 蜕变

> 原属批次:续31 · 封噩梦、灭蛀世、开宝库（51卷46–55章）


⚠ **凯撒操作冲上生存排行榜首位换得**（详见 [称号](../../称号/index.md) / 世界页）——⚠ **用于对已达 SSS 的天赋做「适应性蜕变」**;⚠ **苏晓用它蜕变 [血之兽](../../天赋.md)**（⚠ 蜕变时以血之兽天赋吞掉噩梦血影本源为助力）:

| 【血之兽·完全蜕变】SSS | 效果 |
| --- | --- |
| 血之滋养（被动） | ⚠ **永久提升当前 30% 生命值**（⚠ 苏晓生命 110 万→**143 万**） |
| 血之怒（主动·天怒·印记） | ⚠ **血气系攻击命中有概率在敌人身上构成「天怒·印记」**——⚠ **可瞬间引下界雷**（⚠ 界雷范围固定直径 5 米·强度 = 元素亲和力 50%～70%·随印记层数） |
| ⚠ 印记概率 | ⚠ **基础 20% ＋ 当前能力消耗血气值% ×4.2**（⚠ 消耗 20% 血气即必触发·⚠ 与幸运/运势无关·绝对恒定） |
| ⚠ 层数 | ⚠ 单枚可暂不激活·攒到 3 枚同时激活引更强界雷 |
| ⚠ 终极蜕变 | ⚠ **战胜「凶兽·蜚」后可再蜕变一次** |

- ⚠ **⚠ 它把苏晓的「驭雷法」变成即战力**:⚠ **原本引界雷要以自身为坐标、和敌人同处界雷靠雷抗硬扛**——⚠ **现在血烟炮＋落雷二连**（⚠ 加大血烟炮血气消耗到 20% → 命中即引 5 米界雷 + 绝对控制雷电麻痹·中距离不波及自己）。
