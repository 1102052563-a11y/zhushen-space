# 命运主宰

> 由命运救赎晋级而来，命运系饰品线的当前形态。

## 🆕 命运主宰（传说级 · 饰品 · 二十四卷58章·命运救赎晋级·当前形态）

> 原属批次:已收录（截至第20章）

苏晓在幽鬼大陆用 [扭曲的精魄](../index.md)（紫 → 淡金·100% 成功）+ [神灵之奇迹](../index.md)（淡金 → 传说级）连升两步，把陪他从一阶用到四阶的紫色 [命运救赎](../index.md) **跨四个品阶晋升传说级、更名「命运主宰」**（金属打火机外形 + 苏晓刻的字依旧·线条更柔顺·评分 530·强化清零）。晋级瞬间「吸走方圆一公里所有生物近期运势」（连布布汪都呛到·处刑机关觉醒者 + 平民齐倒霉几小时——幸亏在人多处升级、否则只吸苏晓自己得倒霉一个月）。三效果：

- **效果1·运势掠夺（被动）**：击败 / 击杀敌人吸其少量运势储存、满 100% 永久 +1 幸运（同一敌人 160 天限掠一次·敌运势太弱不触发·每世界最多 +2 幸运）——苏晓由此盯上各世界**剧情主角**（运势强·痛殴一顿或能直接涨幸运）。
- **效果2·气运之力（主动）**：耗一枚灵魂结晶〔碎片〕**临时 +10 幸运**（持续 10 分钟·3 天 CD·期间战斗则 CD ×3）。
- **效果3·幸运女神的祝福 / 女神啜泣（被动·未激活）**：需 [幸运女神](../../../人物/幸运女神.md)「轻吻」命运主宰（得祝福）或将其浸泡在幸运女神鲜血中（得「女神啜泣」·更强）——即「跪舔女神」或「让女神跪舔」二选一。
