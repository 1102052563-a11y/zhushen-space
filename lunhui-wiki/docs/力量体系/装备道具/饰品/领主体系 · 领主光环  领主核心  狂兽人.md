# 👑 领主体系 · 领主光环 / 领主核心 / 狂兽人

- ⚠ **领主光环（第一纪元·永光世界大领主所创·E~SSS级·用「浓缩灵魂」升级）**:⚠ **六种特性「军团/资源/领袖/生产/食物」·用潜力点提升某特性→该特性后续每次升级幅度越大**——⚠ **⚠ 苏晓以争夺战首位额外获 15 点、共 30 点潜力点全砸「军团」特性**（⚠ 六角能力图被顶成「表链笔直的怀表」形状·无视天启保持均衡的提示）。
- ⚠ **领主核心（建筑类单位·E~SSS级）**:⚠ **技术断层后无法直接吸「能量晶石」、降阶到靠「进食」产出「半兽族」战斗单位**——⚠ **⚠ 两种消耗食物模式:产兵 / 提升核心分支属性〔庞大技能树〕·每升一级随机解锁1~3种军团基因图纸或强化现有基因**;⚠ **⚠ 苏晓的沐雨城领主核心从 E 升满 SSS〔共升七次·耗 1053640 单位食物〕、只发展「基础半兽人基因」。**
- ⚠ **狂兽人（完美体·半兽人）**:⚠ **携「白银领主基因」·能吞噬食物「完美分裂」自我繁殖**——⚠ **⚠ 当年拥有「狂兽人基因」的白银领主，败给了拥有「兽人战士基因」的 [鲜红领主](#)**（⚠ 苏晓有自知之明:即使他拥有狂兽人基因，略施小计就战胜鲜红领主也不可能·⚠ 故靠「死亡君主」称号而非领主光环加成军团）。
- ⚠ **食物 / 浓缩灵魂**:⚠ **领主大陆硬通货**——⚠ **「食物」需含高浓度生物能量〔每立方米>32865000Π〕、由「腐壤」〔污染兽族制成〕滋养·「浓缩灵魂」用于升领主光环。**
