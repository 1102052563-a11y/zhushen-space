# 使徒之眼

> 苏晓的首件淡金装备，侦查体系的起点。

## 使徒之眼（淡金 · 侦查类 · 九卷52章·首件淡金装备）

> 原属批次:已收录（截至第20章）

[拉米尔使徒核心](../index.md) 以 [哈瓦呐之眼](../index.md) 为载体改造而成的**龙眼大小机械眼**——[苏晓](../../../人物/苏晓.md) 的**第一件淡金色装备**（淡金 = 暗紫之上·评分 **300**·需智力 20·耐久 53/53·辅助类·官价 9.7 万乐园币）。苏晓评其价值「除 [命运救赎](../index.md) 外最珍贵」。三大主动效果：

- **生命探测（增强版）**：探测**任何生物**并获其资料（完整度看双方智力）·**无消耗**·冷却 5 分钟。
- **警戒色**：分析敌人实力、在其头顶显光点——**白**（实力悬殊）/ **绿**（我方胜率 >70%）/ **红**（对我方很危险）/ **黑**（不可对抗·胜率 <3%）·无消耗无冷却。遇「黑点」当先想撤退（苏晓此前遇黑点级 = 斩赤机械牛、钢炼大猩猩·必死）。
- **空中警卫**：升空对方圆半公里检测、可疑生物入内即预警·**可侦测隐身 / 感知屏蔽 / 地下潜行等 82 种隐匿能力**·耗 120 法力 / 次·持续 12 小时·冷却 24 小时（野外睡眠也安心）。

!!! note "🆕合成晋升传说级（二十一卷80章·当前形态）"

    暗鸦星返园后，苏晓用「装备分解/重塑/合成机」权限把使徒之眼当**主装备**、配 11 件向神皇收来的侦测类装备作副装备合成——**晋升为传说级**（无法再次合成）：性质不变、但**侦测优先度升至六阶**（六阶契约者也乐意拿史诗级装备来换）、且改为**自动吸收光 / 电 / 热 / 磁 / 生物能等十二种能量充能**（不再耗苏晓法力）；新增「**警卫之眼**」（光学隐身升空、检测方圆一公里 152 种隐匿能力）。生命探测优先度六阶、智力 ≥ 敌 2 倍即获 ≥50% 资料。评分 530。苏晓几乎等于用飞升套换了件高评分史诗级装备 + 110 万币。
