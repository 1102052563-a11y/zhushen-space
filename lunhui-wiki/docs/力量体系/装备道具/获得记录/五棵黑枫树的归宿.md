# 🌳 五棵黑枫树的归宿

> 五棵黑枫树最终各自的去向。

苏晓最终集齐星界所有黑枫树:自培的一棵、[刀魔](../../../人物/星空座/刀魔.md) 的一棵、[渊之龙](../../../人物/渊之龙.md) 的一棵、[奥术永恒星](../../../世界/势力/奥术永恒星.md) 的一棵，以及 **「黑枫母树」**。

⚠ **他没有选择垄断炒价**——因为①时间成本太高（战前各方为筹时空之力大量抛售，市场已饱和）；②战后星界迎来和平期，各大势力对壮大自身正处于**战后的初期怠惰**（⚠ 并非记吃不记打，而是**族内顶尖战力死得太多，此时显露怠惰才是智慧**，表现出斗志昂扬会让其他大势力感到威胁）。

| 树 | 去向 |
| --- | --- |
| **自培的那棵**（四棵中最优质、产出质量断层级） | ⚠ 直接售予**轮回乐园**（需经生物识别、猎杀者烙印、个人因果**三次确认**；⚠ 且享有 3 个自然日的「终止交易」申请权）。⚠ 因率先把最优质的一棵售予轮回乐园，他获得 **「分成权限」**——此树后续每次采集产出可得 **3%**，⚠ **他把这 3% 平分给布布汪、阿姆、巴哈、贝妮、棘拉各 0.6%，作为它们今后稳定又数额巨大的财富来源** |
| **其余三棵** | 委托轮回乐园代售（⚠ 苏晓自拟的方案会让收益降低 55~60%，故改为**由轮回乐园拥有 100% 代售决策权**，分成 9:1）。最终方案:⚠ **竞拍制、虚空之树公证、六大乐园参与**——天启／死亡／圣光／圣域／曙光，⚠ **以及半沉寂的守望乐园**。价值排序为渊龙底 > 黑渊 > 奥术永恒星，按 2、3、1 的顺序拍卖 |
| **黑枫母树** | ⚠ 被【猎影】**吞噬枯干**——星界最后一棵。理论上这必会导致四棵黑枫树凋零，**但根据资源最大化利用条例，四大乐园阵营的力量会当场遏制这种理论发生** |

⚠ **关于守望乐园**:它并非在乐园争夺战中惨败，只是**没落了**，加之被曙光的借位苏醒才沉寂。⚠ **乐园的沉寂不是彻底停滞，而是沉入现世之下、收拢沉淀到星界之底的时空之力，完成漫长积攒后强势苏醒**（曙光乐园那种沉寂反而是极少数个例）。守望乐园的坦系从来不弱，在之前几个大纪元中也位于前三以内。
