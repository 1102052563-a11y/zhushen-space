# 虚空 · 树生世界 / 灭法之影传承物（第三十九卷 · 苍龙返园）

> 树生世界与苍龙返园阶段的灭法之影传承物。

苏晓通关苍龙大陆返园、备战八阶晋升期间接触 / 获得的一批虚空级物品（第三十九卷88–97章）：

- **树之恩赐**（品质 ??? · 永久增益药剂 · 虚空之树果实产物）：[苏晓](../../../人物/苏晓.md) 以圣焰药师身份、耗 10 小时抽离 [虚空之树果实](../../index.md) 的纯净能量、再花 516 魂币兑三十几种植物材料调配而成（共两瓶·一份给团长）。**需分两次饮用**：首次饮 1/3 → 提升自身潜力、**恢复一切「基础恢复功能都治不好的永久性身体损伤」**（长期高强度战斗累积的暗伤）；隔 15 个自然日后**二次饮下剩余 2/3 → 生命力与身体能量永久 +20%、并有概率「冲破极限」**（对团长 / 不死老人 / 白牛这种巅峰强者、想有 1% 提升都难如登天、20% 与「冲破极限」是至宝）。可封存 5840 天。苏晓已饮首份 1/3（恢复永久损伤）、剩余 2/3 封存待八阶后突破用（他的「身体能量」= 遍布全身、可随时转青钢影能量的法力值）。**很多强者得果实却找不到炼金师，冒险自试无一例外植物化而死**。
- **神裁 −14**（圣灵级 · 指环 · 成长类 · 评分 1000+++ · 无法出售）：核心装备，被动「**魂之生**」——穿戴者**每点灵魂强度 +130 生命值 + 0.7% 神经反射速度**（现灵魂强度 380 → 已提升 **49400 生命值 + 175% 神经反射**，均逼近上限）。成长条件 = 击杀极恶神灵（多为古神）吸能量成长（成长度 62%）。**三十九卷91章苏晓用 [淬炼卷轴](../../index.md) 强化到 +14**（备战八阶提生存力）：+13 与古神战斗额外 +30 身体防御、**+14 与古神战斗可凭此戒对古神造成瞬间「神灵能量失控」**。晋品质时强化清零。「让主坦蒙圈、暗杀系落泪的生命值提升量」。
- **淬炼卷轴**（半透明卷轴 · 强化道具）：光华流淌、看一眼就令人心生渴望。使用后可**把圣灵级或圣灵级以下装备直接提升至强化 +14**、仅需几分钟。苏晓权衡后用在 [神裁](../../index.md) 上（黑王套 / 猎龙套 / 猎魔套熔锻后不可强化、斩龙闪已 +12 性价比不高）。
- **断魂影之石·残缺 ×1**（灭法之影传承之石 · 树生世界遗留）：集齐 **5 块「残缺」即可合成「断魂影之石·完整」**——一块力量传承之石，凭它可掌控**「断魂影」能力**（灭法之影独有）。苏晓已有 4 块、就差这最后 1 块（在 [树生世界](../../../世界/地点/index.md) 内、灰绅士蓄谋已久的目标之一）。
- **天赋唤醒装置**（大型装置 · 树生世界遗留 · 不可移动）：先代灭法者委托**思林特斯族**（[喔喔](../../../人物/虚空/喔喔.md) 的种族）耗时 105 年打造，灭法时代终结时从灭法大殿崩离、陨落于 [树生世界](../../../世界/地点/index.md)。身处其中可**唤醒灭法者独有的天赋能力**，但唤醒过程极危险、须**牺牲掉一种已觉醒的天赋**（如在噬灵者 / 血之兽中舍一）。苏晓另有办法觉醒第三种天赋——用原始大陆所得的 [古老意志](../../index.md)（须晋八阶、身体承受力够了才行）。
- **新月之芽 +13**（圣灵级 · 耳坠 · 女性 / 雌性专属 · 评分 1000+ · 无法出售）：[月使徒](../../../人物/index.md) 落败强制回归时所留（她当时表情快哭出来）。召唤系的「逆天」装备——被动「**怜悯之心**」每点真实魅力 +36 召唤物生命（最高 +6500）、「**月华沁心**」月色下召唤物有概率永久提升综合属性；+13 附带圣灵宝石镶嵌位 + 月光少女 / 润雨两颗圣灵宝石。因需真实魅力 182、女性专属，苏晓 105 魂币公证后卖出（3200 魂币）。
- **豁免徽章**（消耗品 · 提供永久豁免 · 已消耗）：能**永久豁免魅力 / 意志力 / 幸运属性低于 0 点后的自身减益**（召唤位削减、无法用对应特性装备 / 技能、魅惑抗性下降等），但**对外交涉的减益无法豁免**、且三属性达 −10 后失效。三十九卷88–89章苏晓 [血气觉醒](../../天赋.md) 致魅力 −1 时**一次性触发全部豁免后碎裂**（保住召唤位 / 装备 / 技能不受负魅力拖累·只剩守序善良阵营初始声望降低无法豁免）。
- **定向坐标**（圣灵级 · 一次性道具 · 售价 42 万乐园币 · 苏晓有 3 枚）：使用后构建一处小型定点空间坐标、两处坐标相连即可**超远距离互相传送**（坐标可重复使用、但无法跨世界、无隐蔽性、不可在战斗中 / 险地用）。进入很大的世界时极有用（如苍龙大陆在黑熔城 / 凛冬城 / 王都预设坐标即可瞬移三地）。月使徒落败时掉给苏晓 3 枚。
- **虚空通行证明**（消耗品 · 树生世界前十奖励）：打上 [树之竞技场](../../../世界/地点/index.md) 前十获得、可通行虚空——苏晓觉得不算太实用、直接收起。
- **简易版吞噬之核**（消耗品 · 苏晓自制 · 古战场提纯用）：苏晓把 27000 点法力转为青钢影能量构建的简易 [吞噬之核](../../技能/index.md)，主核 + 尾指粗能量线内 32 颗小核，逐级**净化 / 提纯 / 提取古战场血气**（血气侵蚀性极强、不能直接吸收）。吸满即磨损碎裂。三十九卷90章苏晓靠它在古战场把「[气息外放](../../技能/index.md)」从 Lv.1 冲到 Lv.16。
