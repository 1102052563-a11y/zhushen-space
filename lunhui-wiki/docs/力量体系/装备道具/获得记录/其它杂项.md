# 其它

> 原页「其它」段落的残余内容。

- 侍卫队配发的**火绳枪**（简陋，被苏晓弃用）（第8章）。
- 苏晓现实中的**军官长刀**（父母遗物）——大部分现实物品无法带入乐园（第1、2章）。
- **立体机动装置**（进击的巨人世界·原著民黑科技）：腰侧两个气体喷射箱 + 钩锁，钉墙 / 钉巨人后收紧钩锁拉动身体并喷气，可在高楼城镇内「准飞行」机动；连接的刀柄装 1 米多长、裁纸刀式的可换刀片（锋利但易磨损、备 6 片）。三笠等调查兵团靠它斩巨人后颈；契约者无此装备、只能用笨办法（三卷4–5章）。
- **立体机动装置精通 Lv.10（白·临时技能卷轴）**：使用即熟练掌握立体机动装置（精通级·Lv.10），离开本世界消失。乐园中本不值钱，巨人世界却翻几倍——苏晓向黑商以法师猩红卡 + 1000 币换得，使其机动力 +30%（破碎精灵 + 斩龙闪「喷气式蜘蛛侠」）（三卷12–13章）。
- **爱之腿骨**（绿·钝器·进击的巨人）：绿宝箱所得，评分 16、攻击 10~17、需力量 10；主动「挚爱之粉碎打击」（倾全力一击、伤害随力量定）。苏晓分得后收入储蓄空间待售（三卷9章）。
- **FXF6-T 防爆盾**（东京喰种世界·部下板仓缴获）：约 1.4 米、6 级防弹（除重型狙击枪外可挡各类枪械）、附防弹玻璃瞭望口；苏晓肃清持枪喰种势力时左持盾右持刀冲阵（属世界内物品，非乐园装备）（二卷36章）。
- **世界政府纪念币 ×3**（绿 · 贵重品 · 评分 12 · 海贼王）：纯金打造、面有世界政府五圆球旗徽（世界政府海德制造厂出品）；苏晓在饿狼海贼团船长室搜得（连同几十万贝利）（六卷4章）。**六卷62章被苏晓散落当诱饵**钓出财迷 [娜美](../../../人物/index.md)（娜美「海市蜃楼」幻象可屏蔽感知、唯独抵不住稀有金币的诱惑），近身擒住强喂爆胎易经丸。
- **亚当木**（海贼王世界·稀有木材）：极稀少的顶级造船木材（订购后需几天才送达）；弗兰奇正是去附近岛屿采购亚当木而暂离七水之都（六卷32章·剧情物）。
- **冥王设计图纸**（任务物 · 无品质 / 无评分）：古代兵器冥王的设计图（需掌握**古代文字**才能看懂内容）。这正是苏晓从弗兰奇体内密码锁盗走的**「某件宝物」**（介绍奇特、无品质故苏晓早判其为任务物）。海贼王主线「**司法岛决战**」要求获得它——苏晓提前到手、领先所有契约者，任务到期自动完成；但**决战开启后图纸会暴露在所有契约者的感知中**、引来厮杀（六卷35、38章）。**六卷56章他甩给草帽团的是 60% 真 + 40% 胡编的山寨版**（从真图纸两侧扯下空白纸重做·根本造不出冥王）；**六卷67–70章真图纸卖给世界政府**（战国）换大剑豪剑术心得 + 世界树枝干。
- **良快刀【血舞】**（紫 · 刀 · 评分 129 · 海贼王 · 已祭献）：海军给苏晓的 3 把良快刀中最好的一把（力 27 / 敏 25·攻 33~64）。被动「**影舞断**」：每斩削敌防御 8%、可叠 3 次（共 24%），3 叠后附「**血流如注**」（敌每秒 −2 生命 + 治疗效果 −30%）。因苏晓未获所有权、无法带出海贼世界——他遂用至尊锋刃**祭献给斩龙闪**、把「影舞断」剥来作斩龙闪的新效果2（六卷68章）。
- **大剑豪的剑术心得**（暗紫 · 知识类卷轴 · 评分 170 · 已用）：海贼王大海贼「飞天提督 [金狮子·史基](../../../人物/index.md)」留下的剑术心得。使用可**提升剑术 / 刀术 5 级**（限 Lv.20 以下）——因经乐园认证为暗紫品质，省去研读、转化为「传送进试炼场由金狮子亲自指导直到 +5 级」。苏晓六卷67章谈判从世界政府（海军）换得、六卷73章用掉（刀术大师 Lv.13 → Lv.18·并因此首次斩出刀芒）。
- **世界树的枝干**（白 · 世界级 · 已卖）：简介「浓缩的世界之源」、售价 = **20% 世界之源**（未精炼·无法带出本世界）。即世界政府所称的「亚当木核心枝干」（多在贵族 / 天龙人手中、是身份象征）。苏晓六卷67–70章谈判换得、回归前直接卖给乐园换世界之源（原想拿它「敲诈」乐园、但判断不如从贵族手中夺取靠谱）。
- **牛牛果实（长颈鹿形态）**（暗紫 · 恶魔果实 / 消耗品 · 评分 259 · 待售）：CP9 卡库的动物系恶魔果实（杀卡库的紫宝箱开出）。吃下可在人 / 人兽 / 全兽三形态切换，并**永久 +33 属性**（力+7 / 敏+8 / 体+10 / 智+3 / 魅+5）；畏海与海楼石。属性诱人，但苏晓**决定不吃 / 留售**——审美与战斗风格不合，更怕它与灭法之影职业冲突（果实力量阶位不低·噬灭时未必能保命）（六卷75章）。
- **恶魔低语**（紫 · 消耗品 · 评分 150 · 已用）：黑色恶魔木雕（卡库紫宝箱开出）。用后得被动「恶魔之力」（攻击 +3 真伤）→ 因青钢影暴走引出恶魔族、进化为 **[恶魔之印](../../技能/index.md)**（攻击 +10 真伤·见技能页）（六卷75章）。
- **王权手杖**（绿 · 法杖 · 评分 29）：布布开猩红卡所得（智 12+·加快 30% 施法速度 / 减 3 法力消耗）；苏晓用不上、法杖市价高（六卷80章卖 8000 币）（六卷74章）。
- **岚凯风衣**（暗紫 · 长风衣 · 评分 236 · 死神世界 · 苏晓首件暗紫装备）：黑底金纹立领长风衣（力 30 / 敏 30）。三被动：**夜岚**（夜间减少被发现几率·力 +4 / 敏 +5）、**防穿透**（免一定穿透效果）、**自愈**（破损后耗法力恢复）。六卷79章苏晓以拍卖牛牛果实从 [神皇](../../../人物/index.md) 处换得、当即穿上（六卷79章）。
- **胜利与战争之矛**（暗紫 · 武器 · 评分 251 · 钢之炼金术师 · 未取）：攻 100~130（力 30 / 智 40）·被动「胜利永恒」增加魔能效果 200%——「战法」职业倾家荡产也要的神器。神皇换购牛牛果实时拿出的三件暗紫物之一，苏晓未要（六卷79章）。
- **第三生命炼成法阵**（暗紫 · 消耗品 · 评分 193 · 轮回乐园 · 已用）：传奇炼金师**卡恩**之作，可把机械生物 / 能量生物等**重塑为真正的生命体**并提升其原有能力。苏晓从神皇处换得、用于让 **[布布特尼](../../../人物/布布汪.md) 从帝具进化为真生命「从者」**（六卷79章）。
- **暴食之假面（原罪套装 1/7）**（紫 · 面具 · 评分 150 · 钢之炼金术师 · 持有）：苏晓七卷15章开紫宝箱「双连闪」所得——**乐园著名的法师套装「原罪套装」之一**（共 7 件）。本件需力 5 / 敏 7 / **智 40**；技能「暴食（摄高热量食物充能）」→ 充满激活「贪食之力」（生命 +100 / 法力 +200·施法 30% 几率随机重置一个主动技能冷却·持续 20 分钟·冷却 5 小时）。**套装效果**（集齐 7 件）：① 原罪（被动）法术伤害 +60%；② 不可见。**苏晓用不上**（非法师·智门槛高），但极值钱（紫装通常 5~8 万币·原罪套装更贵·留作出售 / 凑套）。
- **嫉妒之苦难（原罪套装 2/7）**（紫 · 长裤 · 钢之炼金术师 · 持有）：苏晓七卷26章让布布开恩维紫宝箱「闪光」所得（同箱另出 12000 币 + 灵魂结晶〔中〕+ 公证贤者之石）。需力 6 / 敏 4 / **智 42**；技能「嫉妒（主动）」消耗法力上限 10% → 施法速度 +20%（7 分钟·冷却 5 小时）。苏晓集齐**原罪套装 2/7**（暴食之假面 + 嫉妒之苦难·均非法师的他用不上、凑套 / 出售用）。
- **原罪套装（背景）**：乐园最有名的**法师套装**之一·共 7 件·**出自钢之炼金术师世界**（对应七人造人 / 七宗罪）；集齐增法术伤害 60% + 隐藏效果。苏晓已凑齐 **2/7**（暴食之假面、嫉妒之苦难）。
- **高科技伪装面具**（消耗品 · 300 乐园币 · 一次性）：能伪装成他人的**容貌 + 声音**（须先探测对方脸部特征·触感与真人不同·仅能用一次）。苏晓七卷21章用它冒充研究员凯文混入第三研究所。
- **迷惑（暗紫 · 银质耳坠）**：女装大佬持有的暗紫装备·能「迷惑」目标暂时受控（成功率低·曾短暂控住钢炼黑猩猩、却因不敢反抗反被擒·七卷24章）。
- **猎魔套装（暗紫·苏晓渴望·尚未得）**：苏晓与 [神皇](../../../人物/index.md) 闲聊得知的暗紫色套装·**仅 3 件**（项链 / 戒指 / 一把手枪）·具体属性未知（七卷15章）。
- **异次元信号接收器**（蓝 · 物品）：插在手机上、能**突破位面接收信号**的特殊天线（信号差·一般没人用）；布布特尼用它玩跨位面手游（印证布布智力高·七卷15章）。
- **海楼石吊坠**（无品级 · 特殊物品）：含海楼石的吊坠；海楼石能令恶魔果实能力者瘫软无力。路奇借「赔礼」送苏晓、实为**测他是不是果实能力者**（苏晓接触无碍·非果实能力者·随手收进储蓄空间）（六卷38章）。
- **永恒指针**（海贼王世界特产）：类似指南针、但**永远固定指向某一个地点**的航海工具。苏晓杀掉饿狼海贼团船长后仍留众海贼驾船，正因船由「永恒指针」锁定七水之都方向、不怕他们改道（六卷3章）。
- **海贼世界名刀·三品级**（六卷13章）：自下而上为 **良快刀五十工 → 大块刀二十一工（21 把传世名刀）→ 无上大快刀十二工**。苏晓向卡莉法勒索名刀——大块刀难得，遂改要一把**良快刀五十工**（4 天内送到·须刀类规模如斩龙闪）；卡莉法用电话虫向 CP9 [加布拉](../../../人物/index.md) 以 3000 万贝利购其六年前所得的良快刀（坐海列车送来）。
- **六式「剃」修炼方法（完整）**（紫 · 知识类技巧 · 评分 141）：掌握需力 25 / 敏 30 / 体 25；可据此自行钻研学会「剃」、再在乐园辅助升级。**离开本衍生世界即收回**（卡莉法所制·猎杀者无法模拟制造）。苏晓勒索卡莉法所得（详见 [技能 · 六式](../../技能/index.md)）（六卷12–15章）。
