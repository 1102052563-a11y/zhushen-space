# 📚 其他重要物品

> 未归入具体类型的重要物品汇总。

| 物品 | 说明 |
| --- | --- |
| **【智者秘典】**（封印物·灵魂大书库外借） | 半米高的通体金属书籍，封面三张金属面孔——**禁忌智者（暴怒壮年男性）／老人（静默智者）／冷傲女性（心灵智者）**，对应**黑暗知识／阵图术式知识／封印学**三大类。⚠ 每隔一段时间只会放出一位智者（看模样至少要几十年才放下一位，但苏晓靠**吞噬命运**强行连开）。⚠ 它被视为封印物，除了危险知识，还因其**吞噬命运**的特性——对正在封印【命运金币】的苏晓来说，**这些被吞噬的命运并不是他的** |
| **【坏心眼的预言家】**（深渊奇物） | 占卜牌。可消耗当前运势占卜，**50% 绝对准确／50% 必定错误**，结果取决于此物的「心情」。⚠ 它按魅力下菜碟——对 −100 魅力的苏晓极度厌弃，故第一次给出「死神」。苏晓把它封进原罪之书第八页 3 秒后再翻出，它就变得目光惊惧、老实占卜了（半小时后再封一次，它被强行**「格式化」**回初见状态） |
| **【通晓之眼】** | 橘子大小的眼球，注入血气后会长出火柴棍般纤细的四肢并索取「愿望」。⚠ 旧日之神·乌波·萨拉虽已陨落，但**眼之仪式的力量并未减弱，反而更强**（这是亘古存在的力量之一）。本批用两次（问深渊宝石→「深渊、温床、母亲」；问元素宝石→「ψ」），**还剩一次** |
| **【命运星辰】**（上古奇物·41 万盎司） | 宛如绚烂晶核。注入身体能量后绽放光芒，用于唤醒无上属性 50 点后的**「极境之路」终身抉择**。⚠ 因当前星界环境未达可孕育它的程度，**至少要到第四～第五纪元才有低概率在元素/深渊交接处孕育此物** |
| **【命运晶尘】** | ⚠ 用于**「洗练」掉「无上之域」中自身不需要的特性**，让威能百分百集中。市场价 **400 万盎司/份**（这个纪元地精商会只卖出过两份），⚠ 苏晓以 **30 万/份**从前董事长伯吉·金冠处买到三份 |
| **【蠕动之卵】**（顶级幸运物） | ⚠ 本质是邪恶——鸡蛋大小的金色蛹卵，看似无害、能吸收厄运，可其核心处有**邪恶到极致的恶意**（已吞噬掉很多个持有者，包括一位老迈至强）。⚠ 苏晓的用法是**给它灌自己的厄运**:先被动吸收半小时膨胀到足球大，它开始不装了、主动吸收＋吞噬；到直径两米时触须缠住苏晓左臂彻底摊牌——苏晓这才开始以**每秒 1% 厄运量**灌注。第 1 秒它就膨大几十倍化为肉山，5 秒后挤出大本营，100 秒时已是大本营的**两倍大小**，浮现的庞大脸孔以各种语言哀嚎求饶。**嘭的一声炸开**——⚠ 归根结底它是幸运物，爆掉后变成一个超大范围的幸运 buff:**为风海大陆所有友军提供持续 48 小时、幸运/真实幸运属性 +100 点**的加成。⚠ 更奇妙的是，带来这加成的苏晓自己因 −10 真实幸运而**无法获得此增益**。<br>⚠ 本体是一只似蛇似蜥蜴的丑陋生物——**邪恶使徒**；其主上是名为「恶咒」的邪神，被苏晓以召唤术式请来后当场斩杀。⚠ 苏晓随后语气真诚地问将死的邪恶使徒「你还有其他头领或主上吗」——**这本就即将嘎儿掉的邪恶使徒四条腿一蹬，没等生命力枯竭就吓死了** |
| **【无上之力凝结露滴】** | 无上晋升奖励。可对当前所掌握的部分技能使用，令其立即晋升一个位阶。⚠ 苏晓的可选项为**青钢影／青影王／极之影**，最终用在**【极之影】**上 |
| **【极致调和石】**（虚空之树独有产物） | 风海支线任务奖励。可对「深渊之心」或「元素之眼」进行调和以提升晋升无上成功率；⚠ **亦可对二者同时调和**，在提高晋升概率的同时得到**更高的晋升上限**。⚠ 苏晓怀疑触发此任务者看到的「奖励 2」各不相同，是虚空之树的**量身定制**，而这块石头很可能是**史上首次出现** |
| **【噩梦脑髓】** | 界外异存在的部分残留。对苏晓有害无益，直接出售给虚空之树——⚠ **信誉度 +129，首次达到 130 点** |
| **【始源族徽】×4** | 卢恩（从永光世界带回）／死之眸／幻魔／冥渊（六星深渊宝箱开出）。⚠ 全部成为 [阿姆](../../../人物/阿姆.md) 的口粮——**被暴食原罪吞吃掉的始源族徽是真正的终结，永不重聚** |
