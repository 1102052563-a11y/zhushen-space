# 魔海 · 卷40 乐园收尾（第40–41卷）

> 魔海通关后返园的完整战利品与改造记录，全书最长的一份流水账。

- **上古秘药**（圣灵级·稀有·永久增益药剂 · 第40卷85章）：苏晓解译炼金秘典第四页后解锁的**第二纪炼金文明配方**——饮 / 注射后**肉体强度永久 +6、生命值永久 +2500**（真实属性破 200 后仍有效）；**完美品级还可再饮一瓶普通品级**（即一轮永久 +12 肉体 / +5000 生命）。苏晓 / 阿姆 / 巴哈皆可饮、是全队生存力的大提升（材料「活力精髓」稀有·八阶才够额度）。
- **活力复苏（完美）**（永久增益药剂·苏晓自炼 · 第41卷28–29章）：苏晓用少量 [自然恩赐](../../index.md) 稀释 + 两种温和植物材料炼成——饮 / 注射后**体力永久 +10、生命永久 +500**（体力 <80 生效·重复无效）；完美品级还在 3 天内**逐渐修复心 / 肝 / 肾等器官的永久性损伤**。苏晓拿它拿捏财神 [富兰德](../../../人物/魔海/富兰德.md)（偷注一瓶让其年轻十岁）、并与富兰德 / [默里准将](../../../人物/魔海/默里准将.md) / [凯撒](../../../人物/炎灵星/凯撒.md) 结成**药剂利益链**卖给帝国王室（一瓶定价约 20000 魔海金币）。
- **厄运之沙**（魔海世界物 · 第41卷2章）：进入 [魔海](../../../世界/任务世界/魔海-厄运与诅咒.md) 时所得——**在指定地点使用后，沉入海底的「厄运号」将从束缚中挣脱**（厄运号是苏晓强者争霸战时当过船长、并因此被魔海诅咒的活物船）；使用时还会暂时解除魔海对厄运号的封印（故厄运号破海而出那一刻处于超强状态、效果消退后回落）。
- **厄运号**（活物旗舰 · 第41卷7、12、32章）：苏晓在魔海的旗舰——一艘会「吃船」的诅咒帆船（详见 [魔海世界页](../../../世界/任务世界/魔海-厄运与诅咒.md)）。三桅→**四桅**（三阶段封印·魔海金币的力量可同化封印）。一阶封印后：生命 68000 / 防御 179 / 耐久 320 / 航速 +175% / 船员 350·主动 **毁灭重炮**（前方 75° 6000+2700 暗伤·对船 ×4.7~6.5·需泯光岛特制炮弹）。**二阶封印后（41卷32章·耗黄金骷髅 + 14500 金币）**：船长 70 米·**生命 80500 / 防御 194 / 耐久 370 / 航速 +220% / 船员上限 800**·新增被动 **诅咒洗炼**（把本世界秘宝 / 自身装备 / 道具放入船长室「诅咒魔盒」·以诅咒之力洗炼→转诅咒特性、品质大幅提升·可把夺来的世界秘宝洗成能带出的诅咒装备、或升级宝石·亦可洗炼诅咒人如 [奎克](../../../人物/魔海/奎克.md)〔风险大·多暴毙〕）。能力=船猎手 + 水下潜航 + 毁灭重炮 + 诅咒洗炼 + 终极能力封印中。**船长与船同受魔海诅咒**、解厄运号之咒即可解苏晓之咒。破三阶封印=完全体。
- **黑鸦戒**（特殊装备·不可带离本世界 · 第41卷14–20章）：[泯光岛巫祀](../../../人物/魔海/泯光岛巫祀.md) 所赠、厄运号船长专属（需意志力 ≥100 / 灵魂强度 ≥270）——戴上后融入苏晓左臂的诅咒黑纹、无固定形态。效果①**鸦群**：召唤泯光岛「黑咒乌鸦」到外界（初始 1.5 万只·需 1~60 天适应期·被魔海诅咒可加速）；效果②**诅咒石**：戒上最多镶 6 颗诅咒石、按灵魂强度递增召唤量（第 1 颗 +3 万 / 第 2 颗 +4 万 / 第 3 颗 +5 万…递增）——但每镶一颗魔海诅咒都加剧一分。**41卷33–34章苏晓靠炼金器械（灵魂能量容器截留诅咒之力）镶满 6 颗、乌鸦升至 35 万只**（诅咒黑纹蔓延至脖颈胸膛）。黑咒乌鸦=士兵类单位（遏制敌方生命恢复 + 诅咒值满即死·触发苏晓战争领主称号加成）；但满 35 万成完整鸦群、除自己人外见谁攻谁（天灾·且指向明确的攻击指令有巨大精神力负担·只能下笼统命令）。
- **海魂石**（超凡材料·魔海金币购可带出 · 第41卷18–19章）：巨型超凡海兽深海死亡后灵魂被海压 + 魔海力量双重挤压而成的晶石（女巫 / 贵族珍视·多制成超凡物 / 贵重饰品）。苏晓用它作核心放出「**魔海光环**」（直径约 1000 米·每石维持约 1 小时）——让被魔海束缚、无法登陆的**诅咒人在光环内登岛作战**（由布布汪带金属圆盘随行）。12 颗≈诅咒人可陆战 12 小时。破解「诅咒人不能登岛」的关键道具。
- **海王犄角（1/3）**（主线核心物 · 第41卷26、32章）：[伯恩男爵](../../../人物/魔海/伯恩男爵.md) 的伯恩家族传家宝——某种巨兽半截犄角（纹印缩小保存·结构似精神力共鸣装置·拼全注精神力可与某海兽共鸣、犄角之间也能「共鸣」故可追踪）。共三段（伯恩 / [西海王](../../../人物/魔海/西海王.md) / [双塔·女帝](../../../人物/魔海/双塔·女帝.md) 各持一段）。传闻「集齐 = 操控海中之王 = 号令所有海兽 = 称霸魔海」，但双塔·女帝揭穿此说片面——海中之王**只统领一个种类**的海兽，**真正的至宝是钉在它身上的「三支箭」**（先古帝国进献的财富秘宝 + 驾驭整片魔海之力 + 千年寿命），这才是众海盗拼命、厄运号诅咒人视为解咒唯一希望的原因。完成主线全环集齐犄角 = 同时通过苏晓的 **200 点属性壁障考核**。苏晓对海兽大军无感（身为厄运号船长海兽本就畏惧他），只把它当「召最强海兽 = 高品质精魄 = 升青钢影」。**41卷42–43章苏晓已集齐三段**：伯恩男爵那段（根部）+ 西海王那段（中段·西海王决战前割开侧腰塞进腹腔、被鬼影意外掏出）+ 双塔·女帝那段（尖·被 [公爵](../../../人物/魔海/公爵.md) 偷来转送苏晓）——只待找地方召来最强海兽取精魄。
- **黄金骷髅头**（魔海能量物 · 第41卷26章）：鹅蛋大小、黄金打造且内蕴魔海能量——可用于**破除厄运号封印**、效果约等于 **16000 枚魔海金币**（苏晓从伯恩家族密室连海王犄角一并搜出·大大缩短解二阶封印所需金币）。
- **法之箱**（施法者宝箱 · 第41卷24章）：女施法者 [艾娃](../../../人物/魔海/艾娃.md) 死亡掉落——金色遍布蓝纹、内蕴隐晦波动。苏晓首次获得的施法者宝箱（不急开·攒够 + 靠命运主宰开箱临时加幸运再开）。
- **满溢的灵魂箱**（隐藏任务奖励 · 第41卷25章）：完成古堡隐藏任务「邪异之物」所得，开启可获 **100 颗完整灵魂结晶**（巨额·喂噬灵者 / 升灵魂强度的大补）。
- **诅咒炸弹**（苏晓自制 · 第41卷60章）：镶诅咒石时苏晓用炼金器械（灵魂能量容器·每个 10 颗灵魂结晶大）**截留下来的诅咒之力**、封装成可投掷的容器。抛进敌人体内 / 口中引爆，对内腔造成无法豁免的诅咒侵蚀（连霸主级巨鳄都被三颗削弱内腔防御 + 阿波罗腔内收尾炸杀）。
- **源（水特性）**（气息兼容物 · 第41卷53章）：苏晓在源·神乡所得的圆形晶石——水特性、内含极庞大能量、**兼容性极强**（能把死物 / 古神血 / 邪异根须等各种骇人气息都兼容结合、隔绝外泄）。是把 [温妮](../../../人物/魔海/温妮·旺达.md) 伪装成「最强梯队女巫」的核心（阶位不够、数量来凑）。
- **欺诈者**（半活物耳坠 · 第41卷53章）：章鱼哥提供的伪装类耳坠（半个活物）——伪装 + 防物品丢失的「保险」，是温妮女巫套装的一件。
- **古旧的杀戒**（圣灵级·戒指 · 第41卷61章·蘑菇先知处）：效果极强、甚至超出多数不朽级装备——但装备途中**一旦受到攻击即永久破损、不可修复**（一次性爆发神器）。
- **裂空**（不朽级·宝石·稀有 · 第42卷24–25章）：产自八阶战争世界——镶嵌后武器锋利度 +175 / 穿透 +150 / 耐久 +25 + 中级不损特性（评分 1499·「锋利至极限，哪怕空间也足以撕破」）。苏晓 5800 灵魂钱币 + 159 光银矿从天启商人 [司羊](../../../人物/暗星/司羊.md) 处购得——这是给 [恶魔铁匠](../../../人物/暗星/恶魔铁匠.md) 镶斩龙闪的**第一颗宝石**（成就任务「曾经的友谊」需 3 颗）；却被嵌合体违规者截胡、苏晓遂反猎夺回。
- **扭曲生命铭文**（不朽级·铭文 · 第42卷23章）：苏晓威胁 [母神](../../../人物/暗星/母神.md) 逼出的好处之一——待激活状态，需先获得一件「初始无加成、但可安置不朽级铭文」的不朽级装备（可装 3~5 枚铭文·组合型·具体能力随铭文变化）。线索太少、暂收着。
- **忏罪镰**（凶诡战镰 · 第42卷31–37章）：[次代神女·沙塔耶](../../../人物/暗星/次代神女·沙塔耶.md) 的武器——一把外形简朴却极其凶诡之物，镰刃锋利、能钩断脖颈。沙塔耶当年靠它从科多学派总部杀出、猎捕灵魂钟塔 68 年。她自认「用它当报酬是恩将仇报」，故不肯以此答谢苏晓。
- **吞噬者**（苏晓炼金造物 · 团灭套餐组件 · 第42卷35章）：一团黑绿色半流体炼金生物——特性：①成长期前有隐匿性、初始被杀死即无事，杀不掉则遭殃；②敌我不分、同化触碰到黑色触须的生物为「子体」返本体吸收、**滚雪球**壮大成 30 米黑怪（喷黑色激光）；③最多存在 3 小时。与 [太阳神·阿波罗](../../index.md)（3 公里太阳焰）、[冥界之焰·哈迪斯](../../消耗品材料/index.md)（冥焰爆燃·面积远大于阿波罗）合称苏晓的「**团灭套餐**」——阿波罗炸散给吞噬者成长机会、吞噬者引敌围攻、再投哈迪斯焚烧。暗星梦境世界一役靠此套餐炸烧死 3 万 2 千余名野兽族。**🆕（44卷77–79章·初代黑A vs 二代沸红）**：苏晓共造两代——**初代吞噬者·黑A**（6A 六边形面板·凶残桀骜·极难控制·寄生 / 成长强到能刺激宿主潜力使其成长[曾想留在艾奇身上驱使他]）、**二代吞噬者·沸红**（速 / 密 2A 菱形面板·优雅听话·苏晓属意的「外附型增益生物」蓝本）。弑烈阳君主时苏晓把黑A送进其体内内噬助杀，黑A却因烈阳君主死前光焰觉醒（如泡「铚酸」）而当场逃回维生液——两只吞噬者都桀骜难驯、一恢复就想挣脱找宿主。
- **骨戒**（古神信徒标记 · 第42卷32、39章）：[默洛斯](../../../人物/暗星/默洛斯.md)、[玛西亚](../../../人物/暗星/玛西亚.md) 夫妇假死时暗塞苏晓的器物——质感似高密度几丁质、与母神那枚「残旧容器」材质接近，疑为 [大贤者](../../../人物/暗星/大贤者.md) 所造。苏晓亲身验证其真相：**所有戴骨戒的人，无论属于哪方势力，全都是古神的真正信徒（而非科多学派那种伪信徒）= 苏晓的敌人**（科多学派激进派的 [杜鲁主教](../../../人物/暗星/杜鲁主教.md) 就戴着骨戒）。
- **轮回之环**（时空之力争夺特殊装备 · 第42卷43章）：争夺时空之力时轮回乐园发放的特殊装备（可选 1~600 枚）——戴上后能**屏蔽天启乐园与圣光乐园的检核**（隐藏己方身份 / 数据），完成争夺后强制收回。可随佩戴者意愿变形（苏晓的 1 号变成遮下半脸的金属面具·脸颊有血红数字 1；沙塔耶的戴左耳）。苏晓这次为 boss 队领了 100 枚（实际只需 21 枚·有备无患）。
- **魂切**（不朽级·宝石·稀有 · 第42卷53章）：产自暗星、评分 **1496**——镶嵌后武器锋利度 +52 / 破甲性 +46，且**使用者每 1 点灵魂强度再 +1 锋利 +1 破甲**（最高 +205 锋利 / +197 破甲·满增益时额外 +12% 灵魂伤害·「刃之极锋，可斩灵魂」）。[贝妮](../../../人物/狩之影.md) 找了很久的精品，由落魄宝石商 [摩利兰](../../../人物/暗星/摩利兰.md) 以此当报酬（换苏晓查其家族厄运）——**给恶魔铁匠镶斩龙闪的第二颗宝石**（成就任务需 3 颗）。
- **破损的仪式杯**（陨灭星·仪式物品·降幸运 · 第42卷53章）：材质含古神能量 / 固态脑液 / 怨恨之人的心脏 / 仪式血 / 几丁质 / 八阶超凡鸟类胆结石——由**陨灭星学者**开发、多用于「古神之子」的孕育与「眼之生长」。破损不可用，却带**灾厄减益**:10 米内每小时临时 −1 幸运（最高 −5·脱离后恢复），多次承受会触发「灾厄之命」。它正是 [摩利兰](../../../人物/暗星/摩利兰.md) 一家厄运的根源（[灵魂钟塔](../../../人物/暗星/灵魂钟塔.md) 故意留下坑「有缘人」）；苏晓 5000 晶脂买下、留作对付古神的用途之一。
