# 🛡 其他装备

> 零散装备的汇总记录。

| 装备 | 说明 |
| --- | --- |
| **【铁血暗狼】**（霸主级·**第四位**·战靴·30000 满评分） | 集齐 20/20 碎片融合而成。装备需求:至强力量 500／至强体力 850。**狂狼噬命**:所有近战身体攻击附带「因果攻击特性」（若已足以暂时击碎敌人的「因果命运」，狂狼将噬咬其因果命运）；**狂狼之怒**:一种近战身体攻击特性（默认踢击）提升 **当前命核数量 ×0.5 倍**的攻击强度——⚠ 29 颗命核时即为**踢击总攻击强度 ×14.5 倍** |
| **【起源石·世界】** | 新增 **世界·天选（被动）**:成为「唯一·天选」，运势上限 **+205 万**（原 +5 万）→ 当前运势上限 2000 万。⚠ 「巅峰（被动）」的触发条件明确为**无上之境**（而非至强巅峰）。⚠ **死亡后必定掉落**；若击杀者不配拥有，将破碎为五块 |
| **【探索者徽章】**（权限物品） | 当所有命核破碎、生命值低于 1% 时触发「绝对无敌」并立即返回（可主动触发）。⚠ 虚空之树版本返回公证区（14.8 万盎司，用七折权限实付 103600）；苏晓把它提交给轮回乐园换一次荣誉商店刷新，用**一枚轮回徽章**换到轮回乐园公证的同款（返回专属房间）。⚠ 换的理由不是不信虚空之树，**而是他对自己 1 点信誉度太有信心了**。⚠ 任务世界、大型战役中无法激活 |
| **【赋予之誓】**（誓约物/诅咒物·灵魂大书库） | **赋予（被动）**:可将自身一种身体特性加成给从者或密切交集的友方（⚠ **若对方无法承受将立即死亡**）。**誓约代价·临时缺失**:所赋予的身体特性降低至原本的 **10%**。**诅咒代价·厄难降临**:进入厄难状态、运势持续巨量降低——⚠ **因苏晓运势已 −2000 万，此效果对他无效，甚至反噬了装备的诅咒特性**:预计佩戴 10~15 个自然日后，诅咒特性将被其运势磨灭，让此装备成为更纯粹的誓约物。简介:「誓约与诅咒？嗯，只有誓约，没有诅咒，或许今后永无诅咒。」 |
| **【猩红徽章】**（杀戮物品） | 可对乐园阵营目标使用，标记 24 小时；成功击杀被标记目标后 **100% 获得「沉厚的猩红宝箱」**（⚠ 无法对轮回乐园的猎杀者/契约者/职工者使用）。本批用于神父 |
| **【最高级强化石】**（权限物品） | 在装备强化机处使用，让当前装备强化等级**必定成功 +1**。本批共获 5 颗（1 颗以【不朽意志】卷轴换得、4 颗为风海首胜奖励） |
| **【原罪之始】** | ⚠ 大部分属性为「？？？」，只进行了最初步的公证；想完全公证需要数量非常夸张的时空之力（**从这点也能看出其价值**）。触碰瞬间苏晓就有一种「封印八件原罪物的因果重量一下转移到此物上」的莫名轻松感——收入储存空间后依然如此 |
| **【力量种子】**（星界奇物） | 栽种后以少量鲜血滋养即可快速结果；⚠ **98% 概率得到「奇迹果实」**，吃下可提升核心技法能力上限 Lv.30~35 |
| **【黄金圣杯】**（原罪物·危险度约 80） | 神父曾持有。杯中不断涌出酒液，饮下能增长神灵源血，代价是自身诡异运势越强、遇到的诅咒存在也越强，直至因此而死。⚠ 苏晓的用法是**「人工饲养 BOSS 级宝箱」**:物色一名**兽类恶神**以封印术囚困，把圣杯转让给它、让它不断提升神血以壮大击杀后的宝箱含金量（选恶神是因为其内心极度强悍与自我、被改造后大概率不会崩溃与自我放弃，从而造成损失） |
| **【遗弃者】**（大爹级·原罪物） | 一只通体漆黑、手背生有十几片苍白鳞片的**左手**，爆发出强悍的消沉、消极、堕落感。⚠ 由**九星深渊宝箱**开出，被封入原罪之书第九页，成为苏晓的**第九件**大爹级原罪物 |
