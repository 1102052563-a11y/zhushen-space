# 黄金奖励

> 黄金奖励的记录。

- **黄金荣誉卷轴**（黄金 · 贵重物 · 卷轴）：无品级、标注「黄金 / 贵重物」。**可将一项「技巧类能力」提升一个阶位**——技巧类阶位为**基础 → 精通 → 专精 → 大师 →（更高未知）**，本卷轴最高可升到「专精级」（即把专精升为大师）。是寄生兽空间战场给苏晓（裁决者表现「极佳」）的三选一特殊奖励之一（综合裁决者 + 猎杀者身份给出）。苏晓用它把 **[刀术专精 Lv.14 → 刀术大师 Lv.10](../../技能/index.md)**（攻击反增 5%·万物律动升高级）（五卷23–25章）。
