# 🧰 其他关键物品

> 剧情关键物的汇总记录。

| 物品 | 说明 |
| --- | --- |
| ⚠ **【资源置换装置】** | 「战役任务·完美谢幕」奖励。⚠ **全星界仅有一部，为「初始条例」的衍生品，具绝对唯一性，且苏晓是绝对唯一的使用者**；公证方由他选定为轮回乐园（⚠ 该乐园可借此衍生出 3~5 种本乐园独有的「乐园基础设施」）。可把各类物品按其**因果层面的强度**置换为时空之力、无上魂能等。⚠ 它过于巨大（比苏晓的专属房间大万倍），最初是外挂在专属房间上的。⚠ 附带权限 **「评估性提交」**:可向轮回乐园提交物品，由乐园按其**长期价值**单次给予通用类资源 |
| **【炽烬原核】** | 「猎杀悬赏·血契」最终悬赏金之一。⚠ **可让已达到极限强化的装备再获得 +1 强化提升** |
| **【最高级强化石】** | 主要来源是「虚空之树藏库申请权限」（⚠ 苏晓手握 **五折到两折共五份**，折扣极限为两折）；其余全凭机缘 |
| **【磨刀石】**（起源级） | ⚠ 由一名**小巨人铁匠**（矮人王的弟子）自制赠送:可对斩击类物品使用，**临时提升 5% 锋利度（可临时突破上限），持续 12 小时**。⚠ 苏晓从他身上看出了那种暗淡却沉厚、很像三大传说铁匠的因果——**这可能是第四或第五纪元的传说铁匠**（因果感很像熔火巨人）。他的因果波动晦涩又暗淡，今后的路上会经历很多艰辛苦痛，但这也是命运 |
| **【机灵的叽里咕噜】**（命运奇物） | 效果是「珍宝嗅觉」——可嗅到所在世界最高价值的几件珍宝并带持有者去找。⚠ 与苏晓的相性**根本就是没有**:面对苏晓时这只金色胡须、长相酷似老鼠的小东西会被吓得无法动弹，别说寻宝，**它不当场尿出来已是极限**，且它很怕【命运逆转】。⚠ 它把存了**一个纪元的四颗「完整时空石」**吐出来**给自己赎身**，苏晓也就放它离开 |
| **【坏心眼的预言家】**（深渊奇物） | ⚠ 苏晓用它定位「商会同盟」的方法极其质朴:**先占卜（必定失败）→ 一刀斩碎对方防追踪手段留下的因果痕迹、并让破坏力蔓延到其保护源头 → 再占卜一次**。⚠ 它「自行格式化」其实全凭演技——被封进原罪之书太恐怖，双方心照不宣地达成了这个设定。⚠ 它厌恶苏晓也不只是因为负魅力:**它曾被格林·吉莉安获得过，所以它很公平地讨厌所有灭法者**。⚠ 苏晓最后把它半刺入一具超巨型上古深渊存在的尸骸中——**深渊战场是它梦寐以求的进阶之地** |
| **【黑暗根源】** | 斩深渊之主所得，深渊独有物品，价值极高但不是苏晓需要的变强资源 |
| **【黑暗源血】**（原最强霸主装备） | ⚠ 装备需求是**魅力属性与幸运属性**（苏晓两项都低于 1 点，无法适配）。原因是它虽是黑暗特性，**但也是「黑暗宿命」一类**，确与魅力、幸运相关。⚠ **霸主装备的基础特性「霸主之傲」**:如长时间持有无法使用的霸主装备，此装备将「挣脱」你的持有——**此行为已获得虚空之树的公证**。因此苏晓必须尽快处理它（最终被【封魔】吞噬）。⚠ **任何霸主装备登上首位后，都会得到一次巨大增益**，类似强大者完成一次进阶封临，这是霸主装备独有的特性 |
