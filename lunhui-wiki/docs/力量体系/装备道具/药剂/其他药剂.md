# 其他药剂（速查）

> 篇幅较短、未单独立页的药剂，共 3 条。

## 沸血药剂（金 · 增益药剂 · 十七卷43章·开传说级宝箱所得）

评分 390 的增益药剂：饮后立回 50% 生命 + 50% 法力、并进入「**沸血状态**」（每损 1% 生命就 +1 点 / 分钟回血·最高 +150·持续 30 分钟）——**越打越能回、消耗战保命利器**。同箱另得传说之勋 + 灵魂结晶〔大〕×4。

## 永恒水晶瓶（淡金 · 可持续恢复品 · 十八卷25章）

盛着「永恒泉」泉水的水晶瓶——**可分 3 次饮用**、每次回 **450 生命 + 300 法力**、并快速治愈脏器伤势；**自动补充**（每自然日补 5% 容量·湿润环境补 10%）。是苏晓在德岛海战这类长时间消耗战里的续航利器。

## 🆕 神灵时代秘辛 · 药剂师公会 · 合成圣痕（第48卷续9·补）


- **药剂师公会 / 灵魂书库 / 合成圣痕**：⚠ 从钢铁使徒地下实验所扫得的珍藏（神灵时代「药剂师公会」整理）。神灵时代幽暗大陆有一个**灵魂书库**（古老、低调、成员稀少、从不参与争斗的特殊阵营，后消失），令「药剂师公会」（地位仅次于神教）发展极快。公会有套材料优化秘法「**合成**」——以一种名为「**环之圣痕 / 合成圣痕**」的圣痕为媒介，把三份【星辉粉末】合成一份「精炼的星辉粉末」（在炼金学看来是「师见打」的离经叛道之举）。这是**灵魂书库·顶层**的知识，也是苏晓与公爵「共享钢铁使徒知识」交易里最吸引他的部分。
