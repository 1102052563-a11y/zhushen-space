# 巫师大陆道具：魔药 / 真视之契约 / 圣银（三十卷·补充）

[巫师大陆·赫鲁](../../../世界/任务世界/巫师大陆-赫鲁.md) 的神秘学产物：

- **魔药（巫师之源）**：有潜质者饮下即成巫师（启迪魔药 → 灰巫师）——本质是把各类材料硬混（无视副作用 / 能否承受·只追求效果），故巫师疯狂。常见**死魂魔药**（白色·冷却灵魂缓解疯狂·副作用大）；苏晓炼金解析后造出碾压级金色 **冰魂魔药**（大幅降灵魂异常 / 无腥辣味·凯撒贩卖刷爆巫师会声望·抢了 [摩恩家族](../../../人物/巫师大陆群像.md) 生意）。
- **真视之契约（无风险特殊契约）**：老巫师 [修斯·阿奇德](../../../人物/巫师大陆群像.md) 所给——签订后**解除世界限制、见巫师大陆真貌**（虚假繁荣 → 破败腐朽·难度 Lv.10~52 还原为 Lv.49~56）。
- **圣银（旧神尸体提铁 + 纯银）**：巫师大陆最神圣的金属（比金 / 宝石贵）——[悲伤之女](../../../人物/巫师大陆群像.md) 的头饰即圣银，能吸纳 / 驱散「黑暗」。
- **圣银稀释溶液（史诗级 · 克古神道具）**：曙光给的「对付白山羊那份」——涂近战武器后 10 分钟内、**对古神每次攻击附 129~350 圣银真实伤害**（按智力定）。苏晓用它克 [支配者·索托斯](../../../世界/生物万族/index.md)（古神）。
- **源血·空（古神血统道具 · 给巴哈）**：斩支配者所得——剔除恶之神性的古神血统（更像职业模板·不会变古神·能力强度降低）。苏晓用它 + 不朽契约把 [巴哈](../../../人物/剧毒魔鹰·巴哈.md) 进化成「**猎空魔鹰**」（空之血脉·空间穿梭 + 断喉 + 魔鹰领域封空间）。
- **希望之心脏 / 勿忘我（弑支配者所得）**：**希望之心脏**（特殊物品·待研究）；**勿忘我**（剥 [世界之子尼亚](../../../世界/生物万族/index.md) 灵魂记忆得的剑技·一击爆发斩古神半头·因与刀术信念不合苏晓不开发）。
- **残月光（尼亚遗剑 · 限时神兵 · 三十卷55章）**：[世界之子尼亚](../../../世界/生物万族/index.md) 用以斩支配者半头的长剑（1.2 米·阿奇德锻造时混了微量圣银·宜淬高纯度圣银）——三不朽巫师淬圣银后赠苏晓，**月魂焚烧 17 小时**期间持之不被黑暗侵蚀、专斩树木化的「守暗人」无阻力，斩越多累积「黑暗」黑纹越锋利；月魂熄灭即永久损毁，苏晓在报废前用至尊锋刃吞给 **斩龙闪** 涨锋刃值（晋圣灵级的关键之一）。剑偏急速切割、对强敌有破绽（与苏晓契合度仅 60%）。
- **白山羊之母（圣灵级 · 治疗法杖 · 三十卷49章·斩黑山羊·莎布所得）**：苏晓**首件圣灵级装备**（评分 860·通体纯白 1.2 米·售 50 万币）——孕育之生命（治疗 +30% + 智 ×2.7 真实治疗·不可削减）/ 白羊之链（每秒回 107·被中断则瞬回 3745）/ **母之神**（受其治疗者会对持杖者产生强烈保护欲·意志低者甚至错觉对方是生母）。装备需求苛刻（女性 / 智 135+ / 治疗宗师 Lv.30+）；苏晓拟卖给六阶猎杀者大奶妈。
- **座狼套装 8/8（金色套装 · 远程梦幻套 · 三十卷62章·开宝箱所得）**：远程系最渴望的套装（产地即巫师世界·只整套流通）——苏晓 30 点幸运爆开宝箱所得，拟卖给神皇冒险团（远程系无人能拒）。
- **自然源泉（特殊装备 · 世界守卫者奖励 · 三十卷69章）**：用途多到无法恒定品质的水晶瓶——**生之泉**（每 15 天集 100 毫升自然能量提纯液·灌普通植物赋「灵性生命」当探子 / 灌超凡植物助生长）。苏晓给 [布布汪](../../../人物/布布汪.md) 装备去灌养专属房间的黑枫树（超凡植物·大补）。
