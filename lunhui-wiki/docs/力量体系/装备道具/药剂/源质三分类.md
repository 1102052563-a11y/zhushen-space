# 💰 源质三分类（51卷58章 · 苏晓总结）

| 类 | 说明 | 例 |
| --- | --- | --- |
| 1·**专属源质** | ⚠ 只对应传承体系可用·别人吸不仅无法提升反可能丧命 | ⚠ 魔灵/辉光/黑暗/黄金/雷息/**死魂（最不值钱）**/**神圣（最值钱）** |
| 2·**次级通用源质** | ⚠ 类别不多·极少数人用不了 | ⚠ 活力源质（→生命上限）·迅疾源质（→速度·⚠ 死亡系用活力源质比饮剧毒死得快） |
| 3·**通用源质** | ⚠ **价格最高·比其他源质高一梯阶** | ⚠ **力量源质**（→身体属性·谁都能用）·**初始源质**（→晋升至强的必须条件） |

- ⚠ **苏晓的高效吸收（11 块原初碎片晋升绝强）**:⚠ **每份「力量源质」让力/敏/体/智四主属性全 +5%**（⚠ 400 点时一份 +80 点身体属性）——⚠ **代价是要消耗更多原初碎片稳固**（⚠ 而普通 5 碎片晋升者一份只提升一种属性 +5%）;⚠ **⚠ 好处是体魄吸收更多原初之力，后期本源力量超乎寻常强大。**
