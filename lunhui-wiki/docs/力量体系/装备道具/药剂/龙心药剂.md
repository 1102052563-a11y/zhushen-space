# 扭曲的永恒之核 / 龙心药剂 / 庇护药剂

> 永恒之核一线的三种高阶药剂。

## 🌀 扭曲的永恒之核 / 龙心药剂 / 庇护药剂

> 原属批次:续36 · 王者冠冕、古龙遗匙与烈阳血枪（52卷32–46章）


- ⚠ **【扭曲的永恒之核】**（悬赏1·女巫觅心者奖励·价值 1000 盎司）:⚠ **可提交给虚空之树（1000 盎司）/轮回乐园（1500 盎司）/天启乐园（星陨熔炉）/死亡乐园（亡者圣徽）等多方**——⚠ **⚠ 但它带封印且警告「切勿解除，否则气息将引来虚空异存在」**;⚠ **⚠ 苏晓因此改主意:用它与虚空异存在交易**（⚠ **有「树生之页」换到「炼金秘典」的先例——虚空异存在不在乎「等价」，只在乎「满意」**·⚠ 苏晓估测它们对这东西的满意度最起码是树生之页的百倍）。
- ⚠ **【龙心药剂】**:⚠ 最大生命值 +15%·肉体强度 +10%·持续 600 秒（⚠ 让苏晓 183 万生命临时达到史无前例的 **210 万**）。
- ⚠ **【庇护药剂】**:⚠ **捏碎立即生成与你生命值等量的强韧护盾**（⚠ 持续 10 秒或被击碎·⚠ 即 210 万点护盾）——⚠ **这两瓶＋674 雷抗＋猩红盛装＋落雷魂核＋不灭体质，才让苏晓在超脱·原生世界内硬抗住自己引的界雷。**
