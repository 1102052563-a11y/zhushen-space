# 药剂（体系总述）

> 苏晓药剂体系的总体说明。

## 🧪 药剂


| 药剂 | 效果 |
| --- | --- |
| **【极之体壳（巅峰）】**（轮回级·稀有·28500 评分） | 永久 **+6 点全至强主属性**，巅峰品级加成再 +6（合计 **+12**）。⚠ 至强属性达 1500 点以上饮用无效；请勿重复饮用（造成不可估量的体魄损伤且无效果）。⚠ 材料成本 **1560 盎司**，售价 **10 万**（团长在巅峰联络平台代发）——第二纪·炼金文明灭亡后已太多年没出现过永久性增益秘药；⚠ **不能批量调配**，单次催化超过一份难度飙升、稳定性急剧降低 |
| **【奇迹秘药】**（轮回级·稀有·30000+） | 由【奇迹果实】开发到极致所得（⚠ 首次＋极限难度、**没有试错机会**）。永久提升当前技法核心能力**等级上限 Lv.30**，巅峰品级加成再 +15（合计 **+45**）。⚠ **需分三次饮用**，分别在无上体力达到 **20／40／50 点**后，否则将有巨大风险 |
| **【树之恩赐】**（轮回乐园/虚空之树） | 分两次饮用:首次三分之一提升自身潜力并恢复所有**永久性身体损伤**（此类损伤无法通过基础恢复功能恢复）；二次饮用剩余三分之二，生命力与身体能量永久 +20%，并⚠ **有概率冲破极限**（前置:命核低于 300 颗／青影王能量低于 10 万点；可封存 5840 个自然日）。⚠ **苏晓真正要的是「有概率冲破极限」**——他以三瓶**【全能药剂】**（可融入 99.9% 的药剂并完美复刻）复刻出四瓶连饮，把晋升成功率从 90% 推向 100% |
| **【命核秘药】**（自研·未命名） | ⚠ 灵感来自 40 点节点未选的「生命暴君的威仪」。以特异化的「星界始源力量」调配，可让已破碎的命核重聚。⚠ 缺点:**非常之贵**，且能存放的时间不长，因此对战深渊化身时只带了一瓶 |
| **【永生者药剂】** | 使用后寿命上限 **+30%** 并得到上限对应数量的寿命补充。⚠ 对苏晓而言只是每次进入任务世界时荣誉商店药剂补给权限**免费送**的 |
