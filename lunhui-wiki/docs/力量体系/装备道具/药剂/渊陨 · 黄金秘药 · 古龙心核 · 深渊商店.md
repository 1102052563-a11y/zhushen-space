# 🆕 渊陨 · 黄金秘药 · 古龙心核 · 深渊商店（第49卷续17 · 暗影世界）

- **渊陨（起源级 · 重剑 · 暗之邪灵 / 八重封印）**：⚠ 斩沙之王所得，2 米 3 长、剑柄护手倒看似三叉戟，是**虚空最重的三把武器之一**（灭法阵营委托传说铁匠、以深渊未知金属所锻，原为沙之王恩师所赠）。本不邪异，却在遭疯王意识 + 灵魂王冠侵蚀后生出贪婪生命的「**暗之邪灵**」。装备需求:真实力量 300 + 剑术宗师 Lv.65 + 意志 170；⚠ 或与其**契合度达 30% 以上可无视重量与意志苛求使用**，但会被内里的暗之邪灵驱使（每解一重封印、使用者实力也随之暴涨、却更受邪灵操控）。「生命饥渴」原罪被动:杀敌逐步解除八重封印。苏晓暂把它刺在疯人院地牢的剑基上（想试 [黑A](../../../人物/暗影世界/黑A.md) 那「逆子」特性能否白嫖封印解除而不受邪灵侵袭——黑A 果然一把拔走了它）。
- **黄金秘药（奇迹 · 世界级 · 永久增益）**：⚠ 卖黄金罐换得（凯撒十成功力要价）。原版饮后**真实力量 +10、真实体力 +10**（可多瓶、无削减、上限极高——因其主材料是被打碎的**原初碎片**，稀世之作也是暴殄天物之作）。⚠ 苏晓作药剂大师不直接喝，换「奇迹制造者」称号**二次调配成 5 瓶**（每瓶力体各 +3、最多喝 8 瓶）——用一个没用的空罐子换来 **30 点真实身体属性**，血赚。
- **古龙心核（特殊装备 · 耐久 630/1200）**：⚠ 抹窃夺者之名所得（值 200 盎司）。古龙阵营委托炼金阵营打造的**巨量能量储器**——可吸收储存「巨量」龙族 / 暗 / 血系能量（四重容量增幅叠加）。缺点是外放能量需全神贯注（鼻头一痒都可能中断），古老飞龙们气得把它藏起。⚠ 苏晓的用法:去古战场把血气存进去，从而稳步提升气息类能力、免于常去古战场冒险（那守眠者实力在绝强者与至强者间）——配合血枪宗师 Lv.60「血魂共鸣」（融合达 Lv.MAX 的气息能力、叠加血气量上限）。
- **唤醒石（灭法专属宝石）**：⚠ 水哥所赠（他四五阶买货时的添头、对旁人无用）。可**深度提升灭法系能力**（比先前那颗专唤醒魔刃的更通用，可选灵影体质 / 龙影闪 / 屠戮之影 / 青影王）。苏晓用它深度强化了**灵影体质**（生命 +20%、掌握新灭法系能力死亡率 −45%、本源生命力提升五分之一）。
- **幸运旅者（★★★★★ 称号 · 可交易）**：⚠ 苏晓开深渊商店的补偿（虚空之树信誉度 −??? 抢救失败）。微量提升运势、为佩戴者带来好运——对苏晓这灭法毫无卵用（滴幸运神血可略增其波动），但以幸运源血养到六星，在大聚地能卖高价（天启乐园契约者最爱）。
- **深渊商店（虚空之树最高阶设施之一）**：⚠ 苏晓作争夺战发起者、花 100 盎司永久开启，用「**深渊沉淀物 / 沉淀琉璃**」兑换罕有 / 独有物资、每 20~40 日刷新（保留高价值品）、可在任何虚空之树公证的世界激活。橱窗按查看者综合评价排序，好东西都在前十:**收藏家**（原罪物·9800 沉淀琉璃·融合后可无代价用其他原罪物、但意识被收藏家主导·「渴望」态会吞世界之力致 99% 物种灭绝）、**本源果实**（7680）、**原初碎片**（3820）、**原罪之芽**（1100）、**铭文基座·神祭**（五插槽·128）、**随机誓约物**（50）、**贪婪钱袋**（首次免费·开出 −1~50 万钱币·苏晓开出 −1 钱币）。
