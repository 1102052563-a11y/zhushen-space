# 🔮 魔灵源质（灭法阵营独有）· 魔灵强度 190→410

⚠ **[吞星·阿卡斯宝库](../../../世界/任务世界/永光世界-厄难囚笼.md) 3 份**——⚠ **提升刃之魔灵的魔灵强度**;⚠ **苏晓每份都以吞噬之核吸走一半魔灵能量**（⚠ **淬炼净化成无色能量吸收 → 永久提升「所有潜力上限阶位」**）:

| 用量 | 结果 |
| --- | --- |
| 第 1 份 | ⚠ 魔灵 190→245·**所有潜力上限阶位 +5**（16→21） |
| 第 2 份 | ⚠ 魔灵 245→310·**所有潜力上限阶位 +4**（21→25·⚠ 近期不能再吸、须缓半月） |
| 第 3 份 | ⚠ 魔灵 310→**410**（⚠ 860 灵魂强度足以驾驭 400＋ 魔灵） |

- ⚠ **阿卡斯的「魂灵比例卷轴」**（⚠ 半数据化·记多少灵魂强度压多少魔灵强度）:⚠ **100 灵魂→20 魔灵… 500 灵魂→260 魔灵（质变期）… 600→300… 1000→500**;⚠ **⚠ 灵魂强度过 500 算质变期·苏晓 860 灵魂能压 400 出头魔灵**（⚠ 之前一直以为极限是 300）。
- ⚠ **稳固**:⚠ **用 5 块原初碎片稳固**（⚠ 2 力量源质＋3 魔灵源质=5 块·⚠ **攒起来一次用几份最省碎片**）;⚠ **⚠ 现有原初碎片 15 块**（含阿卡斯宝库 12＋厄运钱袋等）。
