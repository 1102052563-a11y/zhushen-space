# 本源药剂（完美）

> 完美级的本源药剂。

## 🧪 本源药剂（完美）

> 原属批次:续28 · 极暗古堡与生命晶石（51卷16–25章）


⚠ **没能调配出（奇迹）品级。**

| 品级 | 能力基石的比喻 |
| --- | --- |
| **普通** | ⚠ **岩石 → 花岗岩** |
| **完美** | ⚠ **岩石 → 铁石** |
| **奇迹** | ⚠ **岩石 → 金铁**——⚠ **无论是在绝强级中能达到的极限战力，还是未来晋升至强的成功率，都占尽优势** |

- ⚠ **⚠ 每人毕生中只能饮下一瓶**——⚠ **敢尝试喝第二瓶，必死无疑。**
- ⚠ **苏晓的决定**:⚠ **不急着喝**——⚠ **这次要是能活着回到轮回乐园，可以尝试把这东西卖掉，卖成沉淀琉璃**;⚠ **如此一来不仅能再买【本源果实】进行下次调配，并且还能大赚一笔**——⚠ **这样多重复几次，总有一次能调配出（奇迹）品级。**
- ⚠ **他的调配心态**:⚠ **他并没刻意的调整状态，只是心态放稳**——⚠ **虽说这是他掌握药剂学以来最重要的几次调配之一，可他的状态却前所未有的轻松**;⚠ **这是与各类强敌搏杀所练就的坚韧心态**——⚠ **（⚠ 调配流程几小时，剩余的是让药剂自行催化，大概十几小时后就能得到成品**）。
