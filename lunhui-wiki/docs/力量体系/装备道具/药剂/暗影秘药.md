# 暗影秘药

> 暗影系永久强化秘药。

## 🖤 暗影秘药

> 原属批次:续28 · 极暗古堡与生命晶石（51卷16–25章）


⚠ **给 [食暗者](../../../人物/风海大陆/黑眼存在.md) 的补偿**（⚠ **黑暗特性浓烈**）:「**这次去探索猩红城堡，是我提议，除了你之外，其他人都有收获，我们在合作，收益不能和付出不对等。**」

- ⚠ **食暗者的挣扎**:⚠ **它的视线马上被吸引，但随后就不再关注**——⚠ **因为它知道，在『好队友』小队中，从没有无缘无故的赠予**;⚠ **它刚单手握上这药剂就撒不开手了**——⚠ **可一种『这或许有阴谋』的猜想让它想婉拒**;⚠ **怎奈，它的手突然不听控制，自行拔开瓶塞，然后吨吨吨喝下**——⚠ **它看着手中的空药剂瓶，心中开始忐忑，大脑进入最高速的运转推测这到底是怎么回事**;⚠ **怎奈，CPU 都差点运转烧了，也没想到这里面有什么阴谋。**
- ⚠ **⚠ 其实并没有阴谋**——⚠ **主要就是不让它因为这次受挫而退队**;⚠ **尤其是后续的敌人中，有一直以来的大敌 [神父](../../../人物/神父.md)。**
