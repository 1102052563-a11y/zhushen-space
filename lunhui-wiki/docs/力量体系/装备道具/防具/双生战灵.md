# 双生战灵（曙光大盾）

> 熔火巨人以曙光碎片打造的活体大盾。

⚠ **本页合并了原「装备·道具」页中散落 2 处的记载**，按卷序排列。

## 🛡 曙光碎片 → 曙光大盾（构思 · 52卷8章）

> 原属批次:续33 · 女巫界的灵魂试炼与暗杀者战利品（52卷2–11章）


⚠ **【曙光碎片】=曙光乐园的外部防御碎片**（⚠ 无坚不摧·刀斩/直踹/炼金溶液都不掉渣·高判定价值、无实际使用价值的纪念品）——⚠ **苏晓现有 2 块、并想多搞几块**;⚠ **⚠ 构思【曙光大盾】**:⚠ **10 块曙光碎片＋一件「活着的」永恒级防具（熔火巨人制）为连接基座→无属性加成，却有逆天防御力＋碾压级判定**（⚠ 物理/能量/精神/灵魂/因果/命运/一切有弹道的攻击、任何虚化透化都能挡·护甲穿透/无视防御判定再高也高不过乐园外部防御位阶）;⚠ **⚠ 拟给 [阿姆](../../../人物/阿姆.md) 用**（⚠ 嗜血战斧化战甲提升 100% 生命＋全方面吸血·再配反伤/减速/范围伤害=敌人打它等于打自己）。

## 🛡 双生战灵（曙光大盾 · 熔火巨人打造的活体防具）

> 原属批次:续38 · 曙光大盾、界之圆环与神裁永恒级（52卷62章–53卷11章）


⚠ **用 12 块「曙光碎片」＋熔火巨人的「活体防具」工艺聚合而成**（⚠ **[里德](../../../人物/陵墓守卫者·里德.md) 打不出活体防具，苏晓带他去永光世界灰暗圣所观摩 [熔火巨人](../../../人物/永光世界/熔火巨人.md)——顺便让熔火巨人传授里德些秘术好给狂猎之夜也加持这特性**·⚠ 里德:「我靠，不去，坚决不去」）——⚠ **⚠ 曙光碎片是曙光乐园的外部防御层，无法溶解、只能靠活体金属聚合**（⚠ **若整成凳子，能用到下个纪元都不成问题**）;⚠ **⚠ 直径 1.2 米·厚 10 公分·永恒级活体防具**:

| 双生战灵 | 说明 |
| --- | --- |
| **两种灵性** | ⚠ 凛寒坚固状态（暗蓝·坚不可摧）＝**阿因斯德先生** · 灼热灵活状态（炽红·半流体）＝**艾伊女士** |
| **自动切换** | ⚠ 按 [阿姆](../../../人物/阿姆.md) 生命值:>10% 为阿因斯德先生的盾形态·<10% 自动切为艾伊女士（**半流体战甲攀附体表·大幅提防御并逐渐治愈伤势**·⚠ 也可抑制切换保持盾形态） |
| **内盾** | ⚠ 阿姆的嗜血战斧（龙心斧＋嗜血战甲）多了第三用途:化半流体攀附盾内形成**内盾**（直接防御一般，但**抗冲击、震荡奇高**）——⚠ 解决持盾挡下过强攻击被震荡重创的问题 |
| **⚠ 若被轰碎** | ⚠ **重新化为一块块曙光碎片后，也能靠活体防具重新组合** |

- ⚠ **⚠ 实战首秀（53卷12章）**:⚠ **[畸变的野兽神灵](../../../人物/烈阳星/畸变的野兽神灵.md) 一拳把持盾格挡的阿姆从神灵放逐之地轰到黄昏城外城区**（⚠ **超光速级速度**·⚠ **有曙光大盾格挡阿姆才没被轰粉碎、生命剩 15%**——⚠ **曙光大盾没白打造**）。
