# 黑 · 王之陨祭

> 黑王套装的第一件，开黑王之魂宝箱所得。

## 黑·王之陨祭（传说级套装 1/3 · 护臂 · 二十五卷53章·开黑王之魂宝箱所得）

> 原属批次:装备品级


「**黑之王·阿德格什**」的传说级套装件（传说级仅五种套装之一）——通体纯黑的护臂，苏晓切掉用了很久、已金属疲劳的 [王者御尊](../index.md) 后换戴它（改色为赤红·隐藏伤势）。单件被动「**颤栗之瞳**」：与意志 <20 的敌方对视即令其恐惧（一阶契约者直接定身）。需意志 ≥60（否则被黑之王侵蚀同化）。**套装效果（集齐 3 件激活）「不死意志」**：受意志洗礼后生命上限永久 +50%、意志永久 +35、生命 <5% 才进濒死、且濒死仍能战斗到死——等于续半条命。这是苏晓下一个世界 [黑王座大陆](../../../世界/任务世界/index.md) 的主目标（有几率集齐套装并晋升其阶位）。
