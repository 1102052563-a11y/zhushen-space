# 🆕 神圣分割器（详解）· 黑王护臂真相 · 圣所钥匙（第48卷续2）

- **神圣分割器**（世界级·虚空第二纪炼金文明）：核心效果「转换」——破开封禁、让使用者进入「**伪界**」（精神/梦境/元素构建·血肉进入只能「路过」、互不影响；精神体进入才可影响其中事物）或「**深度世界**」（挂在主世界上的隐秘小号物理界位·房屋到大殿广场大小）。每次激活耗 1 盎司时空之力。⚠ 简介重磅:此物是幽暗大陆**神灵时代**大主教向炼金文明重金定做、想「分割出死寂城」却失败的——**因死寂的根源就在此地**，只能「接受命运、安坐死寂神座」或「直面并战胜无尽的死亡」。苏晓靠它绕过封死的通道斩痛苦之女、又闯进深度世界找到老怪物。仅本世界可用。
- **黑王护臂 / 黑·王之轮回**（圣灵级·套装·苏晓的「因果物」·真相）：⚠ 大主教一眼认出这是前被选者「**黑之王·阿德格什**」的护臂——阿德格什意志力能排众多被选者前三，可惜所在世界位阶太低，刚到根源之地就被死寂同化、驱逐回去。苏晓正因随身此物、与死寂因果越缠越深、被死之民盯上，才要回死寂城了断。它也是苏晓身为「被选者」的凭证。
- **圣所钥匙**（晋升任务第三环关键物）：藏在痛苦之女手中，指向死寂城「至高圣所」。苏晓斩痛苦之女取得、完成晋升任务第三环（奖庇护石 ×7）。
- **本世界宝箱一览**：斩天外生物所得——扭变深渊宝箱（羊头恶魔）、不朽宝箱「暗之物」（黑暗行者）、不朽宝箱「苦难尽头」（痛苦之女）、滑腻生物组织球（天外使者·凑齐可当宝箱开 / 可还给天外使者和好）。⚠ 提示:【扭变深渊宝箱】开扭变后的深渊特性装备，与开黑枫树种 / 原罪物 / 始源魔镜等「深渊产物」的【深渊宝箱】不同。
