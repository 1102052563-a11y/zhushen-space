# 🆕 神裁戒 · 弑罪神奖励 · 先古面具进化（第48卷续5）

- **神裁戒（吞古神本源成长·屠神器物）**：苏晓食指戒——击杀古神时自动吸收其本源力量成长。斩罪业之神·渥米普什后成长度升至 **36.8%**（其极限并非不朽级、而是能达**起源级**）。⚠ 装备效果2「**神噬**」（被动）:击杀极恶神灵后，按所杀恶神特性提供一种被动增益能力（强度随佩戴者灵魂强度而定；击杀新恶神则被新能力强行替换）——斩罪神得屠神能力「**罪业之火**」。
- **弑罪神奖励**（27.95% 世界之源 + 4270 灵魂钱币换购）：**神灵骨 ×2**（三块合成神灵之奇迹）、**罪孽之子**（S 级职业 / 血统·已祛古神特性）、**罪业心火**（技能卷轴·兼修灵魂 + 火焰可掌握）、**暗物质摄取**（深渊特性·苏晓想卖黑魔小胖子）、**原罪·天御戮杀**（奥义级·某中立神灵开创后被罪神杀夺·分「原罪」〔灵魂系 Lv.65+〕/「天御」〔刃镰战技·镰类武器宗师 Lv.65+〕/「戮杀」〔意志力 130+〕三部分）。⚠ 另有【被虫蛀的永恒木棍】——凯撒以「轮回乐园特殊权限 × 自己奇葩个人技能」把本该出现的**原罪刃镰**掉包成一根木棍分走（换购列表由此出 bug）。
- **先古面具（伪装升级·准爹级）**：冥界崩灭时吸海量深渊能量、已升至「准爹级」，伪装能力从「伪装他人样貌」进化到**连他人能力都能 100% 复刻**。对战罪神时苏晓把它扣到咕噜脸上、将附身咕噜的圣诗扯出伪装成灵魂武器——先化「元素长枪」（深渊克元素·苏晓元素亲和 978 可引界雷劈死罪神、但会污染青钢影，弃用）、后化「银项链」灵魂武器（积蓄灵魂爆发·冲击震荡 / 单体贯穿·配苏晓 650 灵魂强度可拉灵魂锁链反扯罪神）。⚠ 面具缺失的一小块仍在苏晓手中，故它即便晋「爹级」也治不了苏晓。
- **半成品庇护石 · 破晓队**：⚠ 工坊丢失庇护石秘法、百年重做都达不到半成品——凯撒把残次品丢进**深渊之罐深渊增益**补上工坊缺的关键一步，「好队友」四人遂量产半成品庇护石售卖（各分 25%·苏晓「创意入股」·首客龙神·迪恩）。苏晓另组 **SSS 级「破晓队」**（咕噜 + 圣诗加入·团队技能光辉团队 / 雷息庇佑 / 活力苏醒）。
