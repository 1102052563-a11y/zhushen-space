# 其他防具（速查）

> 篇幅较短、未单独立页的防具，共 3 条。

## 战术皮靴（紫 · 鞋子 · 八卷59章）

紫色空投箱开出的鞋子（产地「轮回乐园·第五车间」·评分 130·耐久 59·需力 15 / 敏 17 / 体 5·市价 31000 币）：**效果① 跋山涉水（被动）**——无视 30% 地形阻碍造成的移速削减（沙漠 / 沼泽等）；**效果② 跳跃增幅（被动）**——跳跃力 +25% 并减少落地硬直。轻巧合脚，苏晓当即换上（之前装备多集中于武器 / 风衣，鞋子是新补的槽位）。

## 急行战靴（暗紫 · 战靴 · 十卷72章·飞坦宝箱）

开 [飞坦](../../../人物/飞坦.md) 暗紫宝箱所得（黑色战靴·评分 240·价 65000 币）：① **急速**（跑速 +27·踢技出招 +13）；② **踢技增幅**（踢击伤害 +15% + 「力透」无视敌 30% 肌肉防御、直接作用内脏）。苏晓常以「刀架住敌兵器 + 侧踢」连招，正好大幅强化侧踢——**当即换下 [战术皮靴](../index.md)** 装上。
