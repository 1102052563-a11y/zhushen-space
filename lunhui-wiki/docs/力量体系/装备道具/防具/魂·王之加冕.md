# 魂 · 王之加冕（黑王护臂）

## 🖤 魂·王之加冕（黑王护臂）· 死寂烬灭已吞噬

> 原属批次:续29 · 斩始祖与君主宝藏（51卷26–35章）


⚠ **苏晓在猩红圣殿夺得「死寂烬灭」本体，剥离世界树树脂后，死寂烬灭犹如熔化般融入黑王护臂**——⚠ **「死寂本源」的五重封印转眼消失三重、随后一度全开**（⚠ 详见「死寂本源」小节）。

| 【魂·王之加冕】 | 内容 |
| --- | --- |
| 产地 | ⚠ **根源·死寂城** · 品质 ⚠ **起源级·套装** |
| 耐久度 | ⚠ **229/240 点**（提升 45 点） |
| 装备需求 | ⚠ **仅死寂之王可穿戴**（已达成·白王） |
| **基础效果** | ⚠ **豁免濒死状态，直至死亡**（⚠ 高优先性——⚠ **这才是黑王护臂的最强能力**·⚠ 大聚地那枚只豁免 50% 濒死减益的【灵魂绽放】就卖 195 万灵魂钱币） |
| **烬灭**（核心·主动） | ⚠ **仿刻死寂烬灭**·5 发烬灭弹（⚠ 100%→120%→150%→200%→300% 递增伤害·每小时自生成一颗）·⚠ **得本体后可将其吞噬，大幅提升综合强度** |
| **死寂降临**（核心·主动） | ⚠ 释放高浓度死寂能量，把 10～10000 米环境临时同化为「根源·死寂城」 |
| **灵魂滋养**（核心·被动） | ⚠ 灵魂力量以缓慢速度滋养意志力，永久提升真实意志力 |
| 当前封印 | ⚠ **五重封印**（⚠ 战后暂降为两重）·⚠ 减到一重需灵魂强度足够强大才可酌情考虑 |
| 评分 | ⚠ **3000＋＋＋**（随封印解除增长） |

- ⚠ **濒死状态的真相**（苏晓的顿悟）:⚠ **契约者的濒死状态是烙印的附带能力，是额外增强而非削弱**——⚠ **那 10% 濒死血条，是平常缓慢储存到烙印里的生命力**;⚠ **没有濒死状态的话契约者会直接死，而非多撑一会**——⚠ **也因此濒死状态才那么衰弱（全属性下降、视线窄到十不存一、瘫倒在地）**;⚠ **⚠ 而黑王护臂能 100% 豁免濒死减益，这在高阶极其罕见、极其珍贵。**
