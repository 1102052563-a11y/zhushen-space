# 称号

> 随阅读更新。记录契约者在衍生世界中赢得的**称谓 / 绰号**。

!!! note "关于「称号」"

    称号分两类：① **正式称号系统**（十七卷100章正式登场·**可佩戴、带属性加成、不占装备栏、每次仅可同时装备一枚**·类似装备）——目前所见均出自**三阶竞技场**（见下）；② 契约者凭战绩 / 名声**自然获得的诨名 / 称谓**（如「黑色死神」）。

## 正式称号系统 · 三阶竞技场称号（十七卷93–101章）

竞技场设有一套可佩戴的称号奖励机制。打上三阶竞技场**第十名**即获 **「胜利之荣耀（印记）」**（可随名次升级·三阶内有效）；击败第二名后此印记**可晋升为正式「称号」**。三阶竞技场共三种称号，**同一时间段内每种仅存一枚**（不是你的就是别人的、错过即永失·故竞技者之间不存在互刷）：

- **善战者（最强 · 三阶最强的代表）**——苏晓 [苏晓](../人物/苏晓.md) 于十七卷100章登顶首位所得（夺自原持有者 [狂狐](../人物/狂狐.md)）。**效果**：生命 +800 / 法力 +800 / 乐园设施费 −10% / **对契约者造成伤害 +12%**。获取门槛苛刻：须先打上第三名（拿到属性最强的「胜利之荣耀」印记）、再在**3 自然日内连挑第二、第一名**，**每 10 个衍生世界周期才出现一枚**；要**永久拥有**它须在保持首位的情况下**带它晋升四阶**（否则只是「使用权」）。它还能在战斗中**具现**（无攻防加成、但 80% 冒险团认得 = 三阶最强的招牌·见之多半逃）。
- **竞技之魂**（次于善战者）：以「龟速」打上第一名的奖励·每 5 个衍生世界周期生成一枚。
- **仰望之勋**：第二名的奖励·每 3 个衍生世界周期生成一枚。

胜利之荣耀印记随名次成长：第 10 名 = 生命 / 法力 +150·设施费 −6%；打到第三名时 = 生命 / 法力 +600·设施费 −9%（属性最强的印记形态）。打前十名竞技**禁赌斗**（因奖励机制特殊）；名次奖励另给普通属性点（10~5 名每场 +2、5~1 名每场 +3·苏晓借此「抢救」个位数的魅力属性）。**上一阶**对应物是一阶的「[竞技天王](../力量体系/装备道具.md)」印记（晋阶时收回）。

## 旅行者（暗鸦星 · 功能向称号 · 二十一卷46章）

苏晓在 [暗鸦星](../世界/任务世界/暗鸦星-暗鸦纪元.md) 超额完成「巫猎人」特殊声望任务（106/80·猎杀百余女巫）所得的**特殊称号**。**效果**（全是非战斗状态增益）：① 回法力速度 +10%；② 伤势恢复 +12%；③ 每小时 +1 魅力属性（上限 50·进入战斗或与剧情人物完成一次交涉后清零重计）。与战斗向的 **善战者** 称号性质互补——苏晓设**自由切换**（一战斗自动换善战者、战斗结束换回旅行者）。这是「抢救」苏晓个位数魅力的又一手段（虽一战斗 / 交涉就清零）。

### 🆕 升级为「行者挽歌」（三十二卷93章·强者争霸战通关后）

强者争霸战夺首位后，苏晓打上**六阶乐园竞技场首位**、得专属称号「**竞技之魂**」（特殊·可牺牲自身永久大幅强化另一称号）——他果断用竞技之魂**强化「旅行者」的魅力特性**（效果 3），「旅行者」遂晋升为稀有称号「**行者挽歌**」：① **魅力 +30**（常驻·非清零累积）；② 可抵挡一次魅力属性判定（冷却 1 自然日）。即便如此，苏晓那漆黑的魅力也只从黑色回到**淡红色**——「想挽救魅力有点难」。

## 团队屠戮者（圣杯世界 · 惩戒性临时称号 · 二十二卷31章）

苏晓在 [第五次圣杯战争](../世界/任务世界/fate-第五次圣杯战争.md) 三刀斩杀 [血门冒险团](../势力/index.md) 团长 [斯坦](../人物/斯坦.md) 后，被轮回乐园**强制佩戴**的临时称号——这是乐园对大型冒险团的**保护机制**：

- **强制佩戴·持续三个衍生世界**（无属性·无法解除 / 交易 / 销毁·三世界后自动消失·期间不能用善战者 / 旅行者称号）。
- **效果**：佩戴期间与任何契约者发生冲突时，**敌方会收到警示、并得知「团队屠戮者」称号的由来**（即此人宰过一个大型冒险团的团长）。
- **实际作用**：威慑——其他契约者一听对方杀过团长，多半转头就逃，反而帮苏晓减少了很多麻烦（他不主动惹人、违规者除外）。

## 战争领主（顶上战场 · 指挥向称号 · 二十五卷50章）

苏晓在 [海贼王·顶上战场](../世界/任务世界/海贼王-顶上战场.md) 完成主线任务3（斩大将赤犬·175% 极其优秀）所得的**顶上战争剧情事件特有称号**。平常战斗时是个无加成的「白称号」，但**指挥战争时**按己方数量阶梯加群体增益：己方 >100 → 行军速度 +10% / 士气 +5；>500 → 力敏体 +1 / 士气 +10；>1000 → 力敏体智 +2 / 士气 +20；>10000 → 全真实属性 +2（除幸运）/ 行军 +20% / 士气 +50（士气 >70 时士兵悍不畏死至数量低于 15% 才溃逃）。指挥战争时其价值高于数件史诗装备。简介「士兵们，踏平他们！」。苏晓暂无法佩戴（[杀戮本能](装备道具.md) 还在强制佩戴 / 团队屠戮者占位）。

**🆕称号燃炼升 ★★★★（三十五卷11章·[炎灵星](../世界/任务世界/炎灵星-沙皇帝.md) 战争发家）**：晋七阶后开「**称号燃炼**」权限（主称号镶炉心、消耗 1~5 副称号提升、每主称号最多 3 次、每次耗 50 灵魂钱币）。苏晓拿浴血荣耀（★★）/ 善战者（★★）/ 伸张正义（★）/ 团队屠戮者（★★★·空白称号当燃料）四枚燃炼，把战争领主从 **★★★ 升到 ★★★★**：
- **急行速率（被动）**：行军速度 +35%。
- **战争增益（择一）**：士兵 >10000 → 全真实属性 +5 / 士气 +50；士兵 >50000 → 全真实属性 +8 / 士气 +60，士气达 100 触发 **浴血奋战**（冷兵器近战有几率摄敌生命力自愈·但可能陷入极端嗜血、攻击一切非友方）。

靠这个称号，苏晓把「至少十股败军拼凑的散兵」加成成「**散装精锐**」——个体战力碾压秘鲁矮人、全灭精锐暴风锤军团。

**🆕称号燃炼升 ★★★★★（三十五卷42章·夜袭要塞刷战功）**：苏晓夜袭秘鲁边境要塞（阿波罗炸东谷 / 双塔要塞）刷够战功、用称号兑换商店的 ★★★ 称号作燃料，把战争领主升到满级 **★★★★★**（并**永久解锁称号燃炼权限**）：
- **急行速率**：行军速度 +45%。
- **领主权威（被动·增益常驻）**：麾下士兵数量达 1000 / 10000 / 50000 / 100000 时**阶梯叠加**——士气 +60 / 全真实属性 +10 / **浴血奋战**（Lv.40·摄敌生命自愈）/ **燃魂之力**（Lv.50·武器附 40 真伤）。

质变在于增益**不再需要开战才生效、只要在苏晓身边就常驻**。靠它，30 万兽骑兵在**塞戈战役**碾杀 60 万矮人（含 10 万雄羊骑兵）打出 **10 比 1**（47 万战死）——秘鲁史上最惨。这也是苏晓被传送来对付绯世的关键（轮回乐园看中的正是战争领主称号）。

## 老猎人（猎杀者专属 · 跨世界追踪 · 二十六卷31章）

苏晓在 [黑王座大陆](../世界/任务世界/黑王座大陆-死寂城.md) 以「极其优秀」评价清理掉违规者狄鸦（9744 号）后，综合评价上升所得的**猎杀者专属称号**。效果「**鲜血印记（主动）**」：凭血追踪目标——**即使猎物身处任意衍生 / 原生 / 试炼世界都能精准追踪**（无消耗·每次持续 12 小时·若猎物在别的世界则给出其所在世界坐标后结束·每个世界可用一次·进新世界刷新）。简介「在绝望中逃亡吧，然后……乖乖死在我脚下。」——猎杀违规者的利器。

## 青钢王·库库林（残阳战役·临时伪装身份 · 二十七卷42章）

并非常规称号，而是 [棘虫星·虫族战场](../世界/任务世界/棘虫星-虫族战场.md) 残阳战役（苏晓无视警告执意灭太阳部落 → 触发的「小概率事件」机密剧情）中，轮回乐园给苏晓**临时伪装的剧情人物身份**——本方阵营的**终极大 BOSS「青钢王·库库林」**（脸罩血纹黑金面具）。期间苏晓获三项临时权限：① 可替代轮回乐园**发布部分战役任务**给空降的 245 名中低阶契约者（如对米奈发的 Lv.37「深入敌后」）；② 可进行空间传送 / 封锁；③ 己方契约者挑衅 / 攻击他可当场格杀。[布布汪](../人物/布布汪.md) 同步被伪装成军需官「布布特尼」（凭战功兑换轮回乐园提供的资源·可浮动价 5%~30% 当「狗官」）。**离开本世界后伪装全部解除、临时权限收回**。名字「青钢王」让苏晓困惑（他只知核心能力青钢影、迟迟未觉醒的大招青影王）。

## 追梦人（暗狱世界 · 世界声望商店的七星称号 · 三十八卷7章）

[暗狱世界](../世界/任务世界/暗狱世界-狩猎之夜.md) 的**世界声望商店**里可兑换的**七星称号**（3000 世界声望·苏晓很想要）——效果「**牺牲（主动）**」：可选定任一枚现有六星称号、消耗掉「追梦人」把那枚六星升到**七星**并大幅强化（强于 90% 以上的七星称号）。简介「一刻不停的追逐，梦想或许就在眼前。」（世界声望商店另有 2 枚七星称号·本世界最多兑 1 枚；斩暴食暗魔得 62 声望即开启商店·刷委托 / 发布任务攒声望。）
