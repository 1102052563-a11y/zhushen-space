# 称号

> 随阅读更新。记录契约者在衍生世界中赢得的**称谓 / 绰号**。

!!! note "关于「称号」"

    称号分两类：① **正式称号系统**（十七卷100章正式登场·**可佩戴、带属性加成、不占装备栏、每次仅可同时装备一枚**·类似装备）——目前所见均出自**三阶竞技场**（见下）；② 契约者凭战绩 / 名声**自然获得的诨名 / 称谓**（如「黑色死神」）。

## 正式称号系统 · 三阶竞技场称号（十七卷93–101章）

竞技场设有一套可佩戴的称号奖励机制。打上三阶竞技场**第十名**即获 **「胜利之荣耀（印记）」**（可随名次升级·三阶内有效）；击败第二名后此印记**可晋升为正式「称号」**。三阶竞技场共三种称号，**同一时间段内每种仅存一枚**（不是你的就是别人的、错过即永失·故竞技者之间不存在互刷）：

- **善战者（最强 · 三阶最强的代表）**——苏晓 [苏晓](../人物/苏晓.md) 于十七卷100章登顶首位所得（夺自原持有者 [狂狐](../人物/狂狐.md)）。**效果**：生命 +800 / 法力 +800 / 乐园设施费 −10% / **对契约者造成伤害 +12%**。获取门槛苛刻：须先打上第三名（拿到属性最强的「胜利之荣耀」印记）、再在**3 自然日内连挑第二、第一名**，**每 10 个衍生世界周期才出现一枚**；要**永久拥有**它须在保持首位的情况下**带它晋升四阶**（否则只是「使用权」）。它还能在战斗中**具现**（无攻防加成、但 80% 冒险团认得 = 三阶最强的招牌·见之多半逃）。
- **竞技之魂**（次于善战者）：以「龟速」打上第一名的奖励·每 5 个衍生世界周期生成一枚。
- **仰望之勋**：第二名的奖励·每 3 个衍生世界周期生成一枚。

胜利之荣耀印记随名次成长：第 10 名 = 生命 / 法力 +150·设施费 −6%；打到第三名时 = 生命 / 法力 +600·设施费 −9%（属性最强的印记形态）。打前十名竞技**禁赌斗**（因奖励机制特殊）；名次奖励另给普通属性点（10~5 名每场 +2、5~1 名每场 +3·苏晓借此「抢救」个位数的魅力属性）。**上一阶**对应物是一阶的「[竞技天王](../力量体系/装备道具.md)」印记（晋阶时收回）。

## 旅行者（暗鸦星 · 功能向称号 · 二十一卷46章）

苏晓在 [暗鸦星](../世界/任务世界/暗鸦星-暗鸦纪元.md) 超额完成「巫猎人」特殊声望任务（106/80·猎杀百余女巫）所得的**特殊称号**。**效果**（全是非战斗状态增益）：① 回法力速度 +10%；② 伤势恢复 +12%；③ 每小时 +1 魅力属性（上限 50·进入战斗或与剧情人物完成一次交涉后清零重计）。与战斗向的 **善战者** 称号性质互补——苏晓设**自由切换**（一战斗自动换善战者、战斗结束换回旅行者）。这是「抢救」苏晓个位数魅力的又一手段（虽一战斗 / 交涉就清零）。

## 团队屠戮者（圣杯世界 · 惩戒性临时称号 · 二十二卷31章）

苏晓在 [第五次圣杯战争](../世界/任务世界/fate-第五次圣杯战争.md) 三刀斩杀 [血门冒险团](../势力/index.md) 团长 [斯坦](../人物/斯坦.md) 后，被轮回乐园**强制佩戴**的临时称号——这是乐园对大型冒险团的**保护机制**：

- **强制佩戴·持续三个衍生世界**（无属性·无法解除 / 交易 / 销毁·三世界后自动消失·期间不能用善战者 / 旅行者称号）。
- **效果**：佩戴期间与任何契约者发生冲突时，**敌方会收到警示、并得知「团队屠戮者」称号的由来**（即此人宰过一个大型冒险团的团长）。
- **实际作用**：威慑——其他契约者一听对方杀过团长，多半转头就逃，反而帮苏晓减少了很多麻烦（他不主动惹人、违规者除外）。

## 战争领主（顶上战场 · 指挥向称号 · 二十五卷50章）

苏晓在 [海贼王·顶上战场](../世界/任务世界/海贼王-顶上战场.md) 完成主线任务3（斩大将赤犬·175% 极其优秀）所得的**顶上战争剧情事件特有称号**。平常战斗时是个无加成的「白称号」，但**指挥战争时**按己方数量阶梯加群体增益：己方 >100 → 行军速度 +10% / 士气 +5；>500 → 力敏体 +1 / 士气 +10；>1000 → 力敏体智 +2 / 士气 +20；>10000 → 全真实属性 +2（除幸运）/ 行军 +20% / 士气 +50（士气 >70 时士兵悍不畏死至数量低于 15% 才溃逃）。指挥战争时其价值高于数件史诗装备。简介「士兵们，踏平他们！」。苏晓暂无法佩戴（[杀戮本能](装备道具.md) 还在强制佩戴 / 团队屠戮者占位）。
