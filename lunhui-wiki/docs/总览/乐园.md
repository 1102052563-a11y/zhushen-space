# 乐园（轮回乐园）

「轮回乐园」既是幕后存在，也是契约者往返衍生世界之间的**栖身之所**。本页记录它作为**场所**的样貌与设施；抽象运作机制见 [轮回乐园机制](轮回乐园机制.md)（第22–28章）。

## 外观与结构
- 形如一座巨城：大片模样各异的建筑（金属 / 砖石 / 甚至像某种生物、表面渗液），构成一条条街道；面积极大、不见尽头（第24章）。
- 顶空覆一层乳白色薄膜（如倒扣的大碗）；地面为淡黄色胶质，平坦不软（第24章）。
- 人口十万以上（无人统计——契约者死亡率极高）（第25章）。

## 中心区 · 乐园直营设施
中心地带有十几座宏伟建筑，对提升实力至关重要；苏晓初期只能看清四座，其余被**迷雾**掩盖（权限不足）（第26章）。

- **属性强化大厅**：进入后随机传送至「强化仓」，付 100 乐园币开启；在此分配属性点。强化伴随**深入骨髓的剧痛**，可选免费**麻醉**，但麻醉会**影响强化效果**（苏晓拒用）（第26章）。详见 [属性 · 亲和属性与强化](../力量体系/属性.md)。
- **试炼场**：两种模式（第27–28章）——
    - **辅助模式**：召唤「镜像」陪练，分低 / 中 / 高级，每小时 **100 / 1000 / 10000** 乐园币。镜像取自衍生世界（如海贼世界剑术宗师 **耕四郎·镜像**）；同一镜像再次召唤会**延续上次记忆**；猎杀者权限可**指定**召唤同一镜像。
    - **实战模式**：每分钟 10 乐园币，机械人偶围攻直到「战死」——**场内死亡不会真死**。
- **装备强化大厅**（尚未使用）、**竞技场**（乐园内唯一可让契约者互相分生死之处）（第26章）。

## 交易市场
- 各处都有人摆摊，但**交易广场**（圆形广场）人最多、好货也多（第26、32章）。
- **职工者摊位**比契约者摊位更可靠：常年混市、熟悉物价、不敢哄抬（自己也要进衍生世界、怕报复）；契约者则常把衍生世界所得高价出售（二卷1章）。
- **物价随需求浮动**：若近期大量契约者进入某类世界，相应道具会涨价（二卷1章）。
- **冒险大厅**：申报永久冒险团之处（见 [机制 · 冒险团](轮回乐园机制.md)）（二卷1章）。

## 两类人群
- **契约者**：执行衍生世界任务者（如苏晓）。
- **职工者**：乐园服务人员（售货 / 餐饮等）；来源是第一个世界任务失败而落选的契约者，每两个月才需进一次衍生世界。乐园**不提供兑换功能**，日常全靠职工者商铺（第25章）。代表：[希珞珞](../人物/希珞珞.md)。

## 安全与专属房间
- 除竞技场外，乐园是**绝对安全点**，契约者之间无法互相攻击（衍生世界中则可）（第25章）。
- **专属房间**：传送回乐园后所处的封闭安全屋，除本人允许任何人无法进入；身处乐园时随时可回（第22、26章）。可花乐园币**扩建空间**（乐园只扩空间、不卖家具，家具需自购，有压缩胶囊便携）（第27章）。

## 契约者的周期
每经历一次衍生世界 → 乐园停留 **3 天** → 被传送回**现实世界**约一星期 → 接召唤返回乐园 → 下一个衍生世界（第25章）。返回现实的规则见 [机制 · 返回现实世界](轮回乐园机制.md)。

## 其它
- **乐园币价值**：普通契约者经历第一个世界一般得 1000–2000 乐园币（仅够买一把武器）；苏晓因噩梦难度收益丰厚、得一万多（第26章）。
- **隐藏 / 修改相貌**：可花乐园币（希珞珞花 500）改容，以免现实遭报复连累家人（第25章）。
- **死亡率**：契约者第一个世界约 80%、第二 60%、第三 50%……（第25章）。
- **商品支配权**：乐园内他人所有的待售物品**无法带离 1 米、无法收入储蓄空间**，须购买取得支配权（第24章）。
