# 总览

《轮回乐园》是一部无限流小说：幕后存在「**轮回乐园**」将人类选中为 **契约者**，派往一个个 **任务世界** 完成任务、争夺资源——这是一套以**生存与进化**为核心的残酷竞争体系。
