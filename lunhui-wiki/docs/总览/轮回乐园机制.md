# 轮回乐园机制

> 随阅读更新；以原文为准。乐园作为**场所**的样貌见 [乐园](乐园.md)。

「**轮回乐园**」是一处神秘存在，与人类签订契约后，派其穿梭于各个「**衍生位面 / 衍生世界**」执行任务。乐园自称「无所不能」，但对猎杀者**不能**复生已故之人（第1、2章）。第17章苏晓首次听到**乐园之声**——机械音、毫无感情、冰冷残酷。

## 契约与条例
- 以一张**羊皮纸**签约，按下带血的指印即成立；签约后乐园会修复伤势、强制觉醒天赋（第2章）。
- **乐园条例**：「一切都将等价交换」（第2章）。
- **印象**：冰冷残酷但**公正**，从未言而无信，且**遵循某种规则行事**（第17、23章）。

## 共生与世界之力
- 轮回乐园与契约者是「**共生**」关系：契约者进入衍生位面探索、获取**世界之源**；乐园借此提取「**世界之力**」以**进化**并维持自身运作——故按世界之源多少给予奖励（第23章）。
- 含大量世界之力之物（如 [世界树指环](../力量体系/装备道具.md)），乐园会**主动开价收购**而非强取，正体现其受规则约束（第22–23章）。

## 契约者 与 猎杀者
- **契约者**：普通人签约后成为契约者，为乐园执行任务、领取奖励（第2章）。
- **猎杀者**：天赋特殊、更适合战斗的契约者，须有「杀人的胆量」（第2、9章）。
    - **本质**：乐园正常情况下**不能直接处决 / 处罚未触犯条例的契约者**——故设猎杀者，由其在衍生世界处理那些「**违规但未触犯乐园条例**」的契约者（如钻规则漏洞牟利者，通常强而狡猾）（第23章）。
    - 无法完成专属**试炼** → 被剥离身份、降为普通契约者；任务中死亡 → **彻底死亡**（第2章）。主线试炼成功后，猎杀者身份才正式解锁、开始发布猎杀任务（第22章）。

## 烙印
契约者手臂上的利剑状纹身，以意识激活，是与乐园交互的核心界面（第2章）。功能：个人信息 / 任务 / 储蓄空间 / 技能 / 装备（阶位不足时部分未解锁）（第2章）。

## 任务、世界之源与结算
- 任务分**主线 / 支线**（支线可在世界途中触发），标注难度（LV.）与期限（第2、11章）。**任务奖励由乐园保障**：发布方违约会被乐园**强制纠正**、操控其兑现（左大臣事后成废人）（第17章）。
- 「**世界之源**」可累计：探索、交流、改变剧情、击杀世界内人物（越强所得越多；任务目标本身不计）皆可（第2、8、22章）。
- **回乐园结算**（第22章）：
    - **综合评价**按「世界之源总和 + 完成任务数量」计算，分 **E- ~ S+**。
    - 世界等级超出猎杀者等级 5 级（噩梦难度）→ **奖励翻倍**。
    - 奖励（属性点 / 乐园币等）自动存入烙印；评价决定**升级**幅度（试炼世界单次升级上限 +1）。例：苏晓海贼王结算 A-、得 4 属性点 + 10000 乐园币、升 LV.2。
- **连锁任务**：部分世界主线为连锁式（如东京喰种），完成数次方可选择回归，完成越多评价越高（二卷2章）。
- **成就任务**：达成特定条件自动触发的隐藏任务，额外给奖、有时比主线更丰厚。如苏晓亲手驱逐 14 区 34% 喰种、致全区 95% 喰种死亡 / 逃离，触发成就「**清道夫**」（亲驱区域 30%+ 喰种且致全区 90%+ 死亡或逃离），奖 1 属性点 + 7% 世界之源（二卷36章）。
- **主线时限策略**：主线达成后可**不立即提交**——一旦提交，剩余停留时间即清零；而衍生世界是变强资源的唯一来源，故延后提交可多换自由行动时间。苏晓升上等达成主线②后即按兵不动，且因 CCG 阵营仅他一人，他不交、喰种阵营契约者的对抗类主线③也无从开启（二卷37章）。
- **平衡机制**：当有契约者加入某阵营、足以干涉阵营平衡时触发——其余契约者被强制划入对立阵营，以免一方过弱（东京喰种因苏晓入 CCG，其他契约者全划喰种阵营）（二卷9章）。
- **预知世界**：进入衍生世界前可提前获知目标世界——等级越高、提前越多（Lv.2 提前 10 分钟）（第32章）。
- **伪造身份**：为完成「加入某阵营」类主线，可花乐园币让乐园伪造合法身份（如格斗教练 50、喰种 100、喰种搜查官家属 100 乐园币）（二卷8章）。
- **伪装机制**：乐园让契约者的能力 / 物品自然融入衍生世界——储物空间取放、外来物品在世界 NPC 眼中均「理所当然」、不引诧异（二卷17章）。

- **豁免卷**：可让契约者进入低于自身等级的「低级世界」、且**没有主线任务**的道具。敌方三人女子冒险团（[女王](../人物/女王.md) / [叶子](../人物/叶子.md) / [黑白](../人物/黑白.md)）即持豁免卷进入东京喰种——无主线、遂图谋猎杀苏晓以破坏平衡、刷取大量世界之源（二卷19章）。
- **临时技能**：当猎杀者等级**低于所在衍生世界一级**时，乐园开放的付费技能，**离开本世界即消失**、价格高昂（趁火打劫）。如苏晓花 800 乐园币购「[基础探测](../力量体系/技能.md)」以查探敌方契约者（二卷20章）。
- **共生伴侣**：契约者可拥有的共生生物，附带被动能力、可远程操控并**共享视觉**，是侦查 / 情报利器（如叶子的小仓鼠「咔咔」，带 Lv.3 欺诈）（二卷19–20章）。
- **查探敌人资料**的三类途径：① 探测类装备 ② 使徒类召唤物 ③ 侦查类技能（详见 [技能 · 探测类能力](../力量体系/技能.md)）（二卷20章）。
- **临时契约**：契约者之间可签订经**乐园公证、不可违背**的契约（如约定互不攻击）；契约由发起者编写（花 50 乐园币）、附带达成奖励与违约罚（违约扣 30 万乐园币）。[女王](../人物/女王.md) 曾以「10000 乐园币精神损失费」向苏晓发起停战临时契约、被拒（二卷28章）。
- **契约者职能与队伍配合**：成熟冒险团按职能分工——**肉盾 / 坦克**（体力特长，稀少、需专门培养，十人团约 1 主坦 + 1 副坦）、**输出**（如狙击手）、**辅助 / 治疗**（奶妈，可放治疗 / 增益光环）。多人一组为作战单位；而苏晓三属性均衡、「一人即一个作战单位」（二卷26–27章）。
- **「必中」非绝对**：乐园不是游戏，技能描述里的「必中」会随实战情况改变——苏晓即凭战斗理解空翻躲过女王的「圣盾突袭」（二卷27章）。
- **隐藏区域商店**：某些特殊 / 隐藏区域设有兑换点，可用阵营资源（如 CCG 贡献值）+ 职位门槛换购稀缺物品；如 [24 区](../世界/地点/index.md) 临时分部后勤部出售人体基因强化液（蓝）、粉碎精灵（绿·精品库因克）等，他区分部并无此功能（二卷42–43章）。
- **PVE / PVP 路线**：普通契约者走 **PVE**（打世界 / 剧情）；**猎杀者**因常与契约者交手而走 **PVP**，乐园有意将其塑造为契约者克星（二卷30章）。
- **猩红卡片**：杀死契约者后**有几率掉落**的红色卡片（喰种世界为「半和平场景」、契约者死亡 **50%** 掉率）。每张有 **3 次抽取**机会，从死者的储物空间或穿戴装备中随机抽取物品 / 装备——是 PVP 的主要战利品（二卷31章）。
- **属性凌驾装备 / 技能**：身体属性是更底层的强大——女王体力达 24 即可肉身硬抗斩龙闪；苏晓由此立志把力 / 敏 / 智推上 20。同时「没有垃圾的技能 / 职业，强弱看使用者」，苏晓最强处是「敢战、不惧死」（二卷33章·感悟）。

## 衍生世界
- **可能是真实世界**：苏晓推测衍生世界极可能是真实的（因左大臣会违约，纯 NPC 则不会）；猜想其由人们的幻想衍生、最终成为真实世界（第17章·推测）。
- **时间线重置**：契约者进入时，衍生世界的时间线会被重置 / 刷新（剧情回到最初），消耗乐园「**时空之力**」；消耗带来代价（如东京喰种耗 0.03 盎司 → 该世界契约者最低评分门槛升至 C 级）（二卷1章）。
- **剧情关键人物**：击杀重要剧情人物前，乐园可能要求**取出其关键部位**（如喰种的赫包）以保剧情延续，否则世界恐生变故（二卷4章）。
- **位面之子**：衍生世界存在关键人物「位面之子」（如幼年路飞）。乐园会血红警告令猎杀者**远离**；接触会触发超高难任务（如「传奇的扼杀」LV.79）（第16章）。

## 返回现实世界
契约者在乐园停留期满后被传送回现实，约一星期后接召唤返回乐园（见 [契约者周期](乐园.md)）（第25章）。返回现实时（第29章）：

- **属性与被动技能在现实有效**——苏晓现实力量暴涨（踹飞停尸柜、徒手制服、车速 160 仍游刃有余、食量数倍）；但**主动技能无法在现实使用**。
- **乐园内所得的装备 / 物品在现实锁定、无法取出**，回乐园后恢复。
- **不可以任何方式向现实泄露乐园之事**，否则警告、无效则处决。
- **现实中的违规契约者**：乐园不支持干涉现实，但有人铤而走险牟利（如花乐园币买「心理系」广告设陷阱诱骗同类）——这类「违规但未犯条例」者，正是**猎杀者**的清理对象（第31章）。

## 冒险团
契约者的组队方式（二卷1章）：

- **临时冒险团**：进入衍生世界前临时组建（交易广场招募，须同往一个世界）；世界内不可互相攻击，但付 10000 乐园币违约金可强退。
- **永久冒险团**：到乐园「冒险大厅」申报、完成任务后获得，团长可邀人入团；等级相近的成员会进入同一世界，**有特殊加成**、提高生存率。需等级达 **Lv.10** 方可建立。
- 苏晓作为猎杀者不便入团（怕猎杀目标是队友），但对永久冒险团的加成心动，考虑日后建「一人冒险团」。

## 世界难度
进入时显示的难度，是契约者**所在区域**的难度，**并非整个世界的真正难度**——海贼世界显示 LV.6 仅指哥亚王都，而该世界存在 LV.79 的 [蒙奇·D·卡普](../人物/路飞.md)（第16章）。一般不会被送往高出自身 5 级以上的区域（猎杀者例外）（第3章）。

## 半数据化
辅助战斗能力（第2章）。要点：以**现实为准、非游戏化**；**血条**显示生命体征、受致命伤直接死亡（哪怕余 100%）；脊柱被毁可「假死」；**血量<10% 不狂化**反而虚弱；要害重伤仍会死（第2、10、12、14章）。

## 物品与宝箱
物品分**规则物品**（可带出）、**几率拾取物品**（不可带出）、**触发类道具**等；宝箱品质白 < 绿 < 蓝、所开物与掉落者相关。详见 [装备·道具](../力量体系/装备道具.md)（第4、7、8、19章）。

## 询问限制
开局阶段，乐园对契约者的提问只回答有限次数，耗尽后不再回答（第2章）。
