# 莱纳·布朗

**莱纳·布朗** 是 [进击的巨人](../世界/任务世界/进击的巨人.md) 世界**铠之巨人**的传承者，[马莱帝国](../势力/index.md) 派入墙内的间谍三人组之一、**副战士长**（三卷22、38章）。

## 简介
身材壮硕、一头黄色短发，成熟稳重、有担当；父亲是马莱人、母亲是艾尔迪亚人。13 岁起与 [阿尼](阿尼.md)（女巨人）、[贝特霍尔德·胡佛](贝特霍尔德·胡佛.md)（超大型巨人）伪装成训练兵团 104 期新兵潜入墙内，使命是攻破神之壁、夺回**始祖巨人之力**。长年潜伏使他一度模糊敌我，是间谍组里最坚定的一个，仍以「马莱副战士长」自持（三卷26、38章）。

- **铠之巨人**：周身覆盖坚硬装甲，防御力为九巨人之最；曾与兵长利威尔正面交战。在女巨人 [阿尼](阿尼.md) 被苏晓逼到晶体化时，莱纳赶到**一口吞下水晶**将她救走（三卷23、26章）。

## 被苏晓欺骗（三卷38章）
攻防战后，莱纳与贝特霍尔德跟踪苏晓，反被识破。苏晓自称是马莱支援来的「**实验战士**」、编造战士编号 **TK·317**，凭借在原著中得来的情报（马莱与中东联盟的战争、始祖巨人之力、野兽巨人**吉克**等内幕）层层取信。苏晓更谎称已感知到「始祖巨人之力不在王族、而在 [艾伦](艾伦.md) 体内」（艾伦兼有进击 + 始祖两种巨人之力），诱使莱纳放弃强攻城墙、转而策划**抓走艾伦（并按上头指示抓走「赫里斯塔」）**，并让莱纳去联络墙外的阿尼。莱纳对苏晓深信不疑（三卷38章）。

## 劫走艾伦与精神分裂（三卷39–43章）
在城墙上，莱纳精神状态失常，竟当众向 [艾伦](艾伦.md) 坦白自己是铠之巨人、邀其同走（他内心早把艾伦当弟弟）；被艾伦敷衍后，与贝特霍尔德当场变身（铠之/超大型），配合苏晓劫走「假死」的艾伦逃出城墙。途中他听从苏晓「不去巨人之森」（避开原著被擒之地）、并俘虏了 [尤弥尔](尤弥尔.md)、为 [赫里斯塔](赫里斯塔.md) 求情勿伤。

**精神分裂**日益严重：长年伪装士兵令他在「**士兵莱纳（保卫城墙）**」与「**战士莱纳（破坏城墙）**」两个人格间错乱、甚至篡改自身记忆、自说自话（教堂夜里竟畅想「加入调查兵团后升官」）。每次被苏晓厉声点破「你是马莱战士」才短暂清醒——是间谍组里最坚定、也最脆弱的一个（三卷42章）。作为间谍三人组队长，他与阿尼轮流变身载众人横穿帕拉蒂岛。

## 结局（三卷49章）
摆脱调查兵团追击后，苏晓得知主线时限不足以抵达马莱帝国（横穿帕拉蒂岛 + 渡海约需十二天），遂图穷匕见——以「我其实不是马莱帝国的人」一句翻脸，趁莱纳虚脱、来不及咬破手指变身，**一刀将其斩首**。莱纳临死前仅来得及惊呼「快散开」。其铠之巨人之力被苏晓收取为世界之源（4.2%）。
