# 志村团藏

[火影忍者世界](../世界/任务世界/火影忍者.md) 木叶暗部组织「**根**」的首领——下颚两道交错疤痕、头缠绷带遮住右眼、拄木杖、总闭眼面无表情，阴郁如「藏在地下的毒蛇」。三代火影的童年玩伴兼竞争对手，对**权力极度热衷**：四代当不成就谋五代、五代不成谋六代，一路竞选村干部皆败仍野心不死。

- **右臂的柱间细胞**：右臂内移植了大蛇丸为他植入的 [柱间细胞](../力量体系/消耗品材料.md)（不完美·缠绷带 + 三道枷锁压制其侵蚀活性）。正因此，[苏晓](苏晓.md) 拿**原始柱间细胞**做实验时被他**感知到**——遂率根十几人找上门，要抢苏晓身上的「初代细胞与研究成果」（自称是「属于木叶的东西」）。
- **政治毒蛇**：能压住他的只有二代 / 三代火影（三代主外、团藏主内、脏活归他）；三代一死他便露本性、谁当火影就想弄死谁取而代之。根是五代火影 [纲手](纲手.md) 最想铲除的组织（木叶遇袭时不见踪影、和平时出来搅局）。
- **抢柱间细胞失手（九卷64–65章）**：派根成员（山中风、油女取根等）追杀苏晓、自己解右臂封印欲亲自动手——却被赶到的纲手喝止（右臂见不得光、又不敢灭口火影），只得报「30 度方向」后退走，麻烦甩给火影派系收拾。

## 五影大会 · 别天神操盘（十五卷9–12章）

[纲手](纲手.md) 重伤无法继续任火影后，团藏当上**代理火影**——苏晓评其崛起为「**败犬的绝地反击**」（二代火影原属意的继承人本是团藏、他在考验中犹豫失了机会，遂扎根地下隐忍数十年、距六代火影只差一步）。[四代雷影](四代雷影.md) 召开五影大会（明为追责木叶），团藏却乐见——若代表木叶出席、争到「忍者联军首领」就能为转正铺路。

会上他暗用「**别天神**」（移植自宇智波止水的写轮眼瞳术·能无察觉侵入他人大脑、修改其意志·用一次需歇十几年）**操控了主持人 [三船](三船.md)**：先抖出「晓首领是宇智波斑」（以火影之名担保·搅乱四影），再借三船「中立」之口提出**五大忍村联盟、联军首领交火影**——眼看就要得手（原著中是 [青](青.md) 戳穿此局、但青早死在苏晓手中、白眼也被毁），却被苏晓的**炼金渡鸦自爆**炸断进程。他还把 [佐助](宇智波佐助.md) 当紧盯的目标（觊觎其写轮眼）。

## 石桥决战佐助 · 伊邪那岐（十五卷36–38章）

五影大会后被带土安排与升值棋子 [佐助](宇智波佐助.md) 实战（火之国石桥）。他靠**右臂**作战——那条右臂由**大蛇丸移植的柱间细胞**构成、密密麻麻镶着**十只写轮眼**，可长时间施宇智波禁术「**伊邪那岐**」（牺牲一只写轮眼即把承受的一切伤害 / 死亡转为梦境、把自己回逆到记录的初始状态·每眼撑一分钟·普通三勾玉写轮眼于他如「子弹」般是消耗品·但查克拉所限「备弹」仅十发、已耗六只）。他觊觎佐助那双能用须佐能乎的写轮眼已久。可苏晓的到场让他腹背受敌（苏晓斩雷影的战力是真切威胁、「宇智波斑」真假未明却有空间能力），止水的别天神之眼又在五影大会用过暂不可用——他想撤、却被满心仇恨的佐助死缠。最终**被佐助以幻术误导**（假装伊邪那岐还生效、实则十眼已耗尽）、双方查克拉刃互贯胸膛、败下阵。

## 里四象封印 · 自尽（十五卷38–40章·殁）

垂死的他挟持 [香磷](香磷.md) 当人质（「我并非贪生怕死、我是唯一能改变忍者世界的变革者、必须活下去」）——被佐助千鸟锐枪连人质带他一并贯穿（他已舍弃柱间细胞右臂、否则被吞噬）。逃到桥中段被带土堵住，遂启动早布于胸口的「**里四象封印术**」（同归于尽·把周围一切封入施术者尸体·施术者必死）；带土虚化下沉、佐助靠通灵鹰才逃出百米黑球。**团藏就此自尽身亡**——临终叹「**就算到最后，我也没当上火影**」，并自毁右眼止水的写轮眼 + 把大脑搅成浆糊（不留写轮眼 / 木叶机密给敌人·死无全尸却笑着死）。带土回收其尸体（剩余写轮眼）。苏晓评他「**狗急跳墙说得这么清新脱俗**」「一手好牌被搅乱」，又点破他若真抱同归于尽之心、本还有逃生机会（「狗急跳墙=示弱、疯狗才可怕」）。
