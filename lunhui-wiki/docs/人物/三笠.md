# 三笠·阿克曼

**三笠·阿克曼** 是 [进击的巨人](../世界/任务世界/进击的巨人.md) 世界的超人气角色、[艾伦](艾伦.md) 的同期挚友（三卷4章）。

## 简介
黑发黑眸的少女，战斗力在训练兵团中拔尖、立体机动装置使得英姿飒爽；对艾伦极度重视。小时杀过人、气势清冷，但被苏晓评为「充其量气势清冷、远不及他的杀伐之气」（三卷13、15章）。

被苏晓的毒计卷入：误以为艾伦被巨人吞食、悲愤欲绝，又见「艾伦兽」欲斩其后颈为之报仇（宿醉令她失手撞墙）；待艾伦本体被拽出，她虽认出巨人有艾伦的眼睛、却只能眼睁睁看苏晓与韩吉把艾伦押往希娜之墙受审。屡次想与苏晓拼命、又自知不是对手而隐忍（三卷11–16章）。

## 关系
- [艾伦](艾伦.md)——同期挚友、心中最重之人。
- 阿尔敏——同期挚友（智囊·屡为艾伦圆话）。
- [苏晓](苏晓.md)（白夜）——视为害艾伦的仇人、恨不能杀之，却畏其战力而隐忍。
