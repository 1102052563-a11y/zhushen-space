# 迷妹

[全职猎人世界](../世界/任务世界/全职猎人.md) 的女契约者——莫西干式发型、黑色皮衣、惯使一把**蛇形长匕首**，是 **「猎人同盟」**（乐园里靠杀契约者刷猩红卡、极不受欢迎的冒险团）的成员。**感知力极敏锐**（甚至胜过一些智力特长的法师）、身手凶狠（瞬身割喉）。

## 历程（十卷52–65章）

- **拍卖场攻防战**：尼拉潘围城时，她先拿出**紫色结界卷轴**垫场（撑住第一波）；后在守后门时颇为活跃。
- **加入突袭小队**：[薇妮](薇妮.md) 提出反袭杀 [红色小丑](红色小丑.md) 后，她主动入伙——六人小队（[苏晓](苏晓.md) / [国足三兄弟](国足三兄弟.md) / [薇妮](薇妮.md) / 迷妹）突围杀向小丑。途中她二十秒清掉一群拖后腿的黑帮头目、以感知力为小队探路避开尼拉潘。
- **死亡**：因不愿与旅团团长 / 成员硬碰，她自请**独自去刺杀据说已无战斗力的小丑**——却不知小丑早布下「**焰狱绝界**」（凭恶魔血统凭空生出的隐形锋利能量丝线）。她虽强行穿过丝线、伤到了小丑，最终仍被丝线绞杀、**只剩头颅**（临死表情是「没想到自己会死」的愕然）。苏晓赶到楼顶时，小丑正拿着她的头颅。

## 关系
- **猎人同盟**：所属冒险团（杀契约者刷猩红卡）。
- **[苏晓](苏晓.md)**：突袭小队临时队友（互相提防却同仇敌忾）。
- **[红色小丑](红色小丑.md)**：刺杀目标 → 反被其「焰狱绝界」所杀。
