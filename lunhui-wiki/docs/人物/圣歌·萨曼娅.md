# 圣歌·萨曼娅

一身纯白色修身长袍、素颜但女王范十足。原文对她的定位极高:

> **圣歌·萨曼娅绝对是战斗奶妈的天花板了，并且是前无古人、后也很难有来者的情况。**

若论实力，就算几大乐园阵营的巅峰强者相加，她也能排到**前五**——这还是建立在「轮回乐园的那两位（[团长](星空座/团长.md)、[苏晓](苏晓.md)）必定包揽第一、第二」的前提下。

## ⚪ 「复活」的真相

除非同为巅峰位阶，否则死亡的降临在她看来都可以推迟与逆转。条件是:**她必须在死亡降临前就已在场、死亡发生在半小时内、身体残存 20% 以上**。

看似不合理的复生能力，说穿原理并不魔幻:

1. 她拥有一种庞大的**无形巅峰领域**，范围大到能笼罩**三分之二个烈阳星**，且无形无影，作用只有一个——**固定印记**；
2. 所有位于领域内的友方目标，默认被标记一枚**「圣歌印记」**；
3. 死亡降临时，印记被动触发，把此人的**因果与灵魂**吸收进印记内，让其在半小时内持续「死亡前 1 秒的活着状态」；
4. 她再用自身超强、超大范围的治愈能力让残躯恢复；身体恢复完好后，就是因果与灵魂的回归。

!!! warning "复活的绝对前提"
    倘若死亡**真的降临**，那就没有起死回生的可能——哪怕乐园阵营与虚空之树也不能。人们所理解的起死回生，更像是**复刻出记忆与认知中的亲友**，而非因果意义上的复活。<br>一切复生必须都有一个绝对条件，无论古神也好、异存在也罢，其复生的根本就三个字:**没死透。**

## 😤 「死去活来」与人格的两面

按理说有这样一位又强大又能治疗的领袖是好事，可位于最前线的一众至强契约者此刻都在怀疑人生——他们的处境用一个词形容非常贴切:**死去活来**。

- 最开始仗着复活能力，一众至强可谓相当勇猛；可**死着死着就不勇猛了**——复活的滋味很不好受。在场一位秃顶邋遢老法师的原话是:「就像被一只无形大手揉握挤压到弹珠大小，又啪的一下被拍成薄片。」这是死亡后的体验，**复生过程就更糟糕**。
- 她知道这些吗？当然知道。她本人的态度是:垂眼俯看垃圾一样，并有几分不耐地表示——**能活过来就应该跪下亲吻我的鞋面，居然还有意见？**

⚠ 但这只是表象:

| 阶段 | 状态 |
| --- | --- |
| 战斗前 | 宛若邻家大姐姐的强大者，待人温和、语气温婉，让人安心又想多看几眼，却不敢有半分内心亵渎 |
| 战斗中 | 随着不断使用复活能力，而**死亡必定带来「负面情绪」，逆转死亡就要承受这些**——她逐渐暴躁、化身骄傲女王，内心的想法是「都是垃圾、虫豸，啊啊啊啊！！！」 |

**她其实是友善阵营**，只是表现得性格恶劣；实际上她复生了不知多少参战者。作为代价，过度使用复生能力让她头重脚轻、内心暴躁到极点，看谁都像垃圾——**可内心最真切的执念是:不会只是看着友军倒下。**

## 😐 与苏晓的一次对视

苏晓传送到烈阳星时，两人四目相对了几秒。圣歌·萨曼娅不动声色地移开视线，口中似乎还发出了「嘁」的一声；苏晓已迈出的步伐一顿，偏头看来。

她袖口下的一只拳头攥紧，但默不作声——**一种「遇到克星了」的感觉让她选择避让**。

⚠ 事实证明这是明智的:和一名虚空之树信誉度 **−?** 点的强力增援发生矛盾，并不明智。

后来当大本营穹顶被打碎、灭世之翼冲天而起时，她内心的暴躁瞬间清空、恢复到昔日的状态，第一反应是——**这次完了**。
