# 无冕领主（日阳转化水晶里的剑术宗师意识）

[暗鸦星](../世界/任务世界/暗鸦星-暗鸦纪元.md) 西港「**日阳转化水晶**」里封着的一缕**远古战士意识**——曾受人敬仰、杀敌几十万、后沦为囚徒。他是「**光阳之剑**」真正的主人（**剑术宗师**·苏晓估其剑术至少宗师级 Lv.20 以上、强于苏晓的刀术），性格纯粹刚直、鄙夷阴谋诡计、敬重对手（「无论胜败，他都是可敬的对手」）。[摩利莎](摩利莎.md) 激活日阳转化水晶得「光之女巫」之力时，他的意识趁机侵入——本想若摩利莎配得上这力量便自行消散，却见她滥用力量、近战还被打懵，遂愤而**夺走她的身体**。

## 夺舍迎战苏晓 · 排斥自败（二十一卷42–43章）

他夺过摩利莎的身体、变身成男性化的高大战士（身高 2.3 米·撕开长裙·光阳之剑在他手里不再违和）——**侦测仅得「无冕领主」之名**（使徒之眼阶位不足）。他剑术远胜摩利莎、剑芒可如实体嵌入墙壁、能凭空生成金色能量刺，把苏晓双腿 / 左臂 / 小腹贯穿。苏晓被迫开**猎魔时刻**狂斩 20 多秒才勉强适应他的神出鬼没。最终他因**灵魂与摩利莎的身体严重排斥**（全身浮现陶瓷般裂痕）撑不住、留下「可惜，有缘再见」便栽倒、身体变回重伤的摩利莎——苏晓一刀斩喉，两个意识一同灭去。

## 关系
- **[摩利莎](摩利莎.md)**：被他夺舍的「附生体」（他鄙夷她不配得此力）。
- **[苏晓](苏晓.md)（库库林·白夜）**：迫他开猎魔时刻才险胜的可敬对手（他自败于身体排斥）。
