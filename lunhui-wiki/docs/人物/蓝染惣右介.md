# 蓝染惣右介（五番队队长 · 尸魂界起义军）

[死神世界](../世界/任务世界/死神-尸魂界.md) 护廷十三队**五番队（救援队）队长**——戴眼镜、人称「老好人」，在瀞灵廷名望极高（死神遇险皆由五番队救援）。但这只是面具：他实为尸魂界的**「起义军」**，目的有二——自身达到巅峰、推翻尸魂界腐朽的封建统治体系（[苏晓](苏晓.md) 评他「死神中的起义军、很有趣」）。可惜在尸魂界的力量前他还不够强（即便日后借**崩玉**提升也难撼动尸魂界）。日后他会判出瀞灵廷、自成一股势力（带走市丸银、东仙要）。

## 识破苏晓 · 镜花水月失效（二十卷10–11章）

「同类更容易发现同类」——他是瀞灵廷里**最先确定苏晓有问题**的人（山本 / 冬狮郎尚在怀疑观察阶段，他已笃定）。借「带路去六番队」之机，他用斩魂刀**镜花水月**（完全催眠系·能让人陷入幻觉任他摆布）暗中控制苏晓。但**苏晓的 [刀术宗师](../力量体系/技能/index.md) 衍生效果完全免疫精神 / 心灵系控制**——镜花水月对他**根本无效**（苏晓努力压制衍生效果、装作被催眠 10 分钟骗过蓝染）。蓝染已察觉苏晓的伪装、却不知催眠已失效（他下次再想催眠苏晓时会很尴尬）。苏晓盘算：加入蓝染方收益最高（杀死神后有蓝染背锅、自成势力），但要先摸清蓝染识破后的反应再决定拉拢或围攻。

!!! note "镜花水月的死穴"

    镜花水月虽强、却**对瞎子无效**（日后蓝染因此拉拢瞎子东仙要）；如今又添一条——**对免疫精神 / 心灵控制的 [刀术宗师](../力量体系/技能/index.md) 苏晓无效**。

## 老阴哔结盟 · 崩玉计划 · 假死（二十卷16–26章）

镜花水月失效后，他本想除掉这个「不稳定因素」、却因苏晓的魂灵药 / 灵子球（堪称天才发明）+ 来路不明（对他这种人反而是「好人」标号）而动了拉拢心思。苏晓**直接挑明目的**、抛阿波罗 + 纯度 500 灵子球、又虚构「制约 X 契约」念能力逼他签**轮回乐园公证双向契约**（违约弹指化灰）——他权衡「百年计划付之一炬 vs 合作」后签约（苏晓自此入「蓝染阵营」、市丸银是同党）。

- **崩玉计划**：崩玉 = 简易许愿机（按渴望提升实力）。他与浦原喜助各造出一半成品、浦原把崩玉封进流魂街女婴**朽木露琪亚**体内。他遂一手布局——安排露琪亚入现世遇黑崎一护 / 被捕、**杀光中央四十六室 + 镜花水月催眠相关人**、判露琪亚死刑，只为夺其体内崩玉融合成完整崩玉（并顺手「培养」黑崎一护这个对手以求稳妥）。
- **假死遁形**：计划高潮是用镜花水月把一具秃头死神尸体伪装成自己钉上东大圣壁、由 [市丸银](市丸银.md) 扮凶手、苏晓扮好友调查——借假死从明处转入暗处。留信嫁祸 [日番谷冬狮郎](日番谷冬狮郎.md)、把崇拜他的 [雏森桃](雏森桃.md) 当棋子推向崩溃。苏晓评其「恶劣」——「与蓝染合作的人哪个有好下场」（东仙要、市丸银都死于他手），故苏晓只借合作捞斩魄刀 / 宝箱、绝不打算陪他进虚圈当炮灰。

## 夺崩玉 · 遁入反膜（二十卷45–50章）

处刑日他的计划全面收尾：本欲让双殛高温蒸发露琪亚后取崩玉，被反水的 [卯之花烈](卯之花烈.md) 搅乱，遂启用后备计划——让东仙要用空间白绸把接走露琪亚的 [阿散井恋次](阿散井恋次.md) 传送到自己面前。他刚要和恋次「叙旧」，就被不喜废话的苏晓两刀解决了恋次（蓝染不懂「反派死于话多」）。他**树木化手臂刺入 [朽木露琪亚](朽木露琪亚.md) 胸膛、取出核桃大小的半成品崩玉**（感慨「原来是这么小的东西」）。随后四道光柱落下、他遁入**反膜**（与尸魂界空间互不干涉）逃离——护廷十三队赶到时为时已晚、因三内奸（蓝染 / 市丸银 / 东仙要）潜伏多年而大败。苏晓评他「恶劣」、终究只借合作捞够好处便正常返园、没陪他进虚圈。
