# 海东（海蝎）

[苏晓](苏晓.md) 在现实世界**唯一的挚友**——一名职业杀手，绰号「**海蝎**」（惯用一种叫「**蝎子嘴**」的奇怪武器、捅出拳头大小的圆形挖肉伤口）。戴帆布手套、头戴焊工帽，性子诙谐讲义气，常对苏晓打趣「你当我儿子干爹吧」。曾与苏晓过命交情、替苏晓杀过人；二人换号码比换衣服还勤。

- **唯一的牵挂 = 7 岁的儿子**：他执行委托从不带着孩子（不让儿子沾染杀手的事），盘算着「自己凶名不及苏晓、万一哪天出事、苏晓能帮忙照看儿子」。

## 复仇与陨落（十一卷77–78章）

他带儿子来本市找久未见面的苏晓叙旧——却恰逢苏晓在轮回乐园。**他执行一桩「让目标住院一年」的委托期间、把儿子独自留在出租屋，回来时儿子已被富二代 [张旭天](苏晓.md) 的比特犬活活咬死**（稚嫩脖颈满是咬痕）。对方父亲房地产大亨张达查出他杀手身份、**雇 80 多名业余杀手 + 粗制手枪「以杀止杀」**；海东四天杀掉近 40 人、却身中三枪（肺穿、心脏破损）。

他给苏晓打了最后一通电话——**不是求苏晓帮他杀人、只是想聊聊**：欣慰地说苏晓「变得不同了、不再是满眼死气的复仇者、能潇洒地活着、真羡慕你这混蛋」，祝他保持这种心态，随后摔碎手机、在废弃厂房**自焚而亡**。苏晓循布布汪追踪赶去——「**他最好的朋友、死了**」。

## 影响 · 让苏晓「明悟」（十一卷79章）

苏晓为他**灭门复仇**：闯张达庄园、餐刀十秒杀光满屋业余杀手与张家父子（[清道夫](../势力/index.md) 西装男到场、因临市更乱无暇多管、只求别用全自动 / 爆炸物闹大、并已替海东安葬）。十天后他到城东海东父子墓前烧鸡敬烟——海东的死成为**苏晓心境的转折点**：他想通「杀手生活只是生存、自己真正的兴趣是战斗、想立于顶点获得自由」，**那股死气沉沉之气化为更纯粹、锋利、凶暴、带侵略性的气息**（练刀之人最需要的东西）。「海东，多谢，你让我明白了某些事。」

## 关系
- **[苏晓](苏晓.md)**：现实唯一挚友（互相替对方杀过人·其死促苏晓「明悟」）。
- **张达 / 张旭天**：咬死其子、雇凶杀他的房地产父子（被苏晓灭门）。
