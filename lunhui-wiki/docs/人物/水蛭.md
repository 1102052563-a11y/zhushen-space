# 水蛭（阴兽）

[全职猎人世界](../世界/任务世界/全职猎人.md) **[十老头](../势力/index.md) 麾下「十阴兽」**之一——微胖男人、阴狠疯狂，原著中是被幻影旅团「窝金」秒掉的角色。是十老头 艾尔索普 的部下。

- **偷袭丧命（十卷20–21章）**：苏晓单挑斩杀独眼巨怪、留下其会感知声音的眼球后，水蛭与搭档 [病犬](病犬.md) 找上门（图巨怪眼球 / 报独眼之利）——他爬到 16 楼墙外想偷袭，却被 [苏晓](苏晓.md) 凭听觉**隔墙一刀刺穿喉咙**（神经反射只比常人略高、躲不开苏晓的刺击），被界断线拖进屋后斩臂、最终被杀（化作紫色宝箱 +1.2% 世界之源 +11 法力）。他的死让十老头 艾尔索普 很被动（阴兽是十老头最大的底牌）。
