# 黑玛丽

[全职猎人世界](../世界/任务世界/全职猎人.md) **[十老头](../势力/index.md) 比克的女儿**——身材婀娜的黑色礼服御姐、一条手臂纹着一朵**黑玫瑰**（苏晓戏称她「黑玫瑰」·她有点天然呆），在帮派里负责财务、深受尊敬。外表高冷，**实则胆小**（被 [苏晓](苏晓.md) 盯着就手足无措）。**真实身份 = 天生念能力者**（特质系·自己尚不自知）。

- **被安插当苏晓助手（十卷23–24章）**：[比克](比克.md) 谈成与苏晓的合作后，**狠心把女儿黑玛丽安插到苏晓身旁**当助手（监视 + 联络）。她笨手笨脚地给被擒的 [病犬](病犬.md) 包扎、又跑金库取来 5 颗「永恒晶石」交付苏晓——与苏晓对视都会发怵，暗暗害怕这个「瘟神」。

## 念能力 ·「黑暗玫瑰」（十卷33–34章揭示）

拍卖会开幕夜她在一楼被 [苏晓](苏晓.md) 误斩的刀芒**腰斩**，却又一丝不挂地复活（瞳孔由黑变青）——苏晓由此识破她是 **天生念能力者**：

- **天生念能力者**：猎人世界中**先天或受刺激而觉醒念能力**的人，能力**必为特质系**（唯特质系可不经锻炼觉醒）；多被普通人当作魔术师 / 巫女 / 预言师，一般无战斗力，**本人往往不知这就是念**。
- **「黑暗玫瑰」**：右臂那朵黑玫瑰纹身即其念能力本体——**能恢复任何伤势**（腰斩 / 灼伤 / 断肢皆可），**但不能复活死人**（不是复活能力）。每次使用，玫瑰花瓣按伤势凋零：普通伤 0.3~0.5 片 / 重伤 1~2 片 / 濒死 2~3 片 / 肢残 + 濒死 3~5 片；**花瓣全部凋零或将失去能力**（现剩约 20 片）。**可对他人使用**——苏晓视之为「加强版的 [香磷](香磷.md)」，价值重大（能恢复断肢，正好用于防范被他斩断两臂一腿的 [红色小丑](红色小丑.md) 自愈）。
- 苏晓「悉心教导」（把斩龙闪架她脖子上）让她飞快掌握该能力，并严令她**对外保密**。

## 关系

- **[苏晓](苏晓.md)**：名义主人 / 雇主——起初极度畏惧，后发现「不招惹他就没危险」、渐渐放松。
