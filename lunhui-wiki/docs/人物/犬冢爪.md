# 犬冢爪

[火影忍者世界](../世界/任务世界/火影忍者.md) 木叶忍者、犬冢一族——短发、脸涂两道红痕，携一条**独眼忍犬**，是 [犬冢牙](犬冢牙.md) 的母亲。佩恩袭村时为子报仇而来。

- **三头犬 · 殁（九卷78–79章）**：她以「犬冢流人兽混合变身·**三头犬**」（自身 + 忍犬合体成黑色三头巨犬）扑向 [苏晓](苏晓.md)——自信「犬冢家族也是近战专家、不至于被瞬杀」。[卡卡西](卡卡西.md) 劝阻不及。苏晓先以环断斩碎其影分身，再反手一刀刺入地面、青钢影 + **影刺**贯穿钻地偷袭的三头犬——犬冢爪与忍犬双双惨死地下。她的鲁莽既害死了自己，也连累卡卡西被苏晓近身（间接致其后来战死）。
