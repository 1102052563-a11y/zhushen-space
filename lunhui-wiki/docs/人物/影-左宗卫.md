# 影 / 左宗卫（灵之骸召唤的队长级死神）

[苏晓](苏晓.md) 在 [死神世界](../世界/任务世界/死神-尸魂界.md) 用阵营商店换来的传说级道具 **[灵之骸](../力量体系/装备道具.md)** 召唤出的**队长级战力死神**（性别 / 样貌 / 年龄 / 能力随机·召唤无需召唤位 / 无消耗·离开死神世界即解约消失）。召出的是一名**满头黑发、破烂死霸装、邋遢颓废**的男人——气质酷似某一代「剑八」，生前真名 **左宗卫**，斩魄刀 **黄泉**（始解放出**红色血线**穿透敌人、可生擒控制）。被 [狛村左阵](狛村左阵.md) 一眼认出（旧识·以为他早已身死）。

## 忠诚度过低 → 被封为「影」（二十卷21–28章）

苏晓召出他后**无法查看其属性**（[召唤物忠诚度](../总览/轮回乐园机制.md) 仅 10~30·危险值）——他保有独立人格、爱喝酒、想见「她」一面（生前牵挂）、嗜杀。苏晓只当他是「较强的炮灰」（92 真实智力强制控制·他挣不脱），让他藏去流魂街森林待命。诱杀 [狛村左阵](狛村左阵.md) 时，他果然抗命——**不愿照苏晓的吩咐把狛村引进森林、反劝狛村离开**。苏晓当即用一枚**菱形晶体**按入其后心、**彻底封住他的个人意识**：从此他只是听令的「影」（失去左宗卫人格、却保留斩魄刀的始解 / 卍解能力·苏晓正是为这两种能力才没把他改造成纯傀儡）。

## 关系
- **[苏晓](苏晓.md)**：召唤主（强制控制·把他当队长级炮灰 / 诱饵）。
- **[狛村左阵](狛村左阵.md)**：生前旧识（认出他的斩魄刀黄泉）。
