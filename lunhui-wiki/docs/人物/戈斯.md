# 戈斯（艾丽莎的守护骑士）

[圣索丁王国](../势力/index.md) 公主 [艾丽莎·赫伯特](艾丽莎·赫伯特.md) 的**守护骑士**——银甲、约 30 岁、下巴留小胡子、配一把百斤以上的骑士重剑、气息很强的**圣战士**。奉老国王密令**誓死保护艾丽莎**（把一切恶挡在外面、不惜性命）。但在 [苏晓](苏晓.md) 眼中，这位「守护骑士」正是老国王送的**进「[王座之下](../世界/任务世界/希亚大陆-灭法遗迹.md)」开路的强力炮灰**大礼。

## 敢怒不敢言 · 沦为探路炮灰（十七卷45–51章）

苏晓拿金狮信物 + 新王书信「征调」艾丽莎（征调 = 公主死了王族也不追究），戈斯敢怒不敢言（屠戮短刀划破公主的皮、他愤怒欲狂却无可奈何·王权他不敢违）。闯王座之下时他一路被当炮灰使：苏晓**用「抽签」逼他探路**——四张纸条其实全是红色（苏晓借储存空间把自己 / 布布汪 / 阿姆的都调包成白色·阿姆没储存空间故由苏晓代开），戈斯无论怎么抽都是探路那个；苏晓更把手按在艾丽莎脖颈上、逼他乖乖去探。他探到陵墓守卫者 [里德](陵墓守卫者·里德.md) 那间锻造室。最讽刺的是——他誓死守护的公主，听了苏晓「你父王很信任你」的鬼话便一脸责任感、反过来「鼓励」他去送死，气得他想骂「你 TM 是智障吗」却不敢出口。

## 关系
- **[艾丽莎·赫伯特](艾丽莎·赫伯特.md)**：他誓死守护的公主（却眼睁睁看她被苏晓拿捏、还被她「鼓励」去探路）。
- **[苏晓](苏晓.md)（铁之手）**：把他当开路炮灰使唤的人（抽签 / 拿公主威胁）。
- **[巴伊·赫伯特](巴伊·赫伯特.md)（老国王）**：命他守护艾丽莎的主君（实则把他当送给苏晓的炮灰大礼）。
