# 顶上战场配角（白胡子 · 马尔科 · 雷利 · 旅团灰发男人 · 斯图茜 · 埃迪·克）

[海贼王·顶上战场](../世界/任务世界/海贼王-顶上战场.md) 中的剧情人物与契约者群像，集中收录。

## 白胡子·爱德华·纽盖特（四皇 · 新世界霸主）

新世界霸主之一、**四皇「白胡子」**——白色月牙胡、震震果实能力者，重情义、以「家人」对待全体船员。苏晓初见时他已**重病濒死**：肺衰竭 / 肝功能缺失 / 严重心脏病 / 脊柱变形压迫神经（低头困难）——任一项都足以致命，他却还拎着烈酒（用酒精麻痹病痛、不敢用止痛药怕肝脏崩）。震震果实的反震力是其身体崩坏主因之一。

- **被苏晓大修恢复 96.7%（二十五卷9–12章）**：苏晓连做三场手术（脊柱 / 肝 / 旧伤 → 心脏 → 其余脏器），用远古秘药加速愈合。术后他重获自由呼吸、脊柱顺滑，「只要休养两三天、开战时能把赤犬捶死」。苏晓私下告诉他：不打激烈战斗还能活 5~7 年。**赠苏晓史诗红酒 [玛歌亚加](../力量体系/装备道具.md)** 作额外报酬。
- 用真身（伪装船）留在伟大航路前半段、莫比迪克号 + 医生团留新世界迷惑大妈与凯多——这招釜底抽薪骗过了所有四皇。
- **开战立威（二十五卷25–26章）**：莫比迪克号从海下浮入马林梵多内港、白胡子开战即用震震果实震塌两侧外港、薙刀柄扫飞大将青雉落海（恢复后判若两人·连艾斯都不敢信他是重病之身）。苏晓上船与他并立船首（曾打算趁其重病时杀他·终未动手）、他出于恩情劝苏晓「革命军不会让你参合、回去吧」。
- **陨落（二十五卷36–48章·殁）**：一脚把大将赤犬踢回海军本部墙上立威，但旧病（心脏 / 肺·苏晓只压制未根治）在高强度连战中复发——独抗三大将 + 青雉黄猿 + 两五阶 + 海军 + 黑胡子团车轮围攻，为儿子们断后震塌马林梵多成深渊。留「**我是旧时代的残党、新时代没有能承载我的船**」「One Piece 在……」，**站着死**。黑胡子盖黑布欲夺其震震果实——却被苏晓预埋的「心脏维续装置」干扰、**夺果实临时失败**。尸体由红发海贼团（见下）带走安葬。

!!! note "苏晓的「心脏维续装置」"
    苏晓大修白胡子时埋下的**被动式**装置（白胡子见闻色太强、埋炸弹会被察觉，故选不可操控的被动设备）——平时维持其停跳的心脏，白胡子一死即激活、干扰黑胡子夺震震果实。苏晓借「与白胡子之死直接关联」凑够 5% 击杀贡献、完成了主线任务4。

## 马尔科 / 乔兹（白胡子海贼团队长）

- **不死鸟·马尔科**：白胡子海贼团**一番队队长**，菠萝头 + 永远睡不醒的模样、不死鸟果实（蓝色火焰再生）。与苏晓交涉大修白胡子事宜的主要对接人——从最初直呼「白夜」到术后改口「白夜先生」。夜里说梦话喊「布莱安娜」被半船人听见。
- **钻石·乔兹**：白胡子海贼团**三番队队长**、钻石果实硬汉。沉默寡言、替战死船员保管遗产送给情人。与人质 [布布汪](布布汪.md) 称兄道弟。

## 雷利（冥王 · 罗杰旧部）

**冥王·雷利**——罗杰海贼团副船长、香波地群岛隐退的涂船匠。为草帽团（尤其路飞那顶草帽）出手挡黄猿、斩碎八尺琼勾玉的余波护住草帽全员。已步入老年期，但战力仍让苏晓判不出其屠戮过多少生灵。对路过试招的苏晓颇为忌惮（「比已老去的我更危险」）。

## 旅团灰发男人（幻影旅团成员 · 健忘）

[幻影旅团](咕噜.md)（苏晓同类的 PVP 冒险团）成员——白衬衫 + 花裤衩 + 灰色马尾辫，胸口刺「魑凶什」三字，**能量丝线**能力（结界 / 吞食线丝充能 / 操控数十平民同时开口说话）。威胁度不低于黄猿。**极度健忘**（每走几步就停下补话、像来搞笑的）。

- 受 [咕噜](咕噜.md) 之托来劝苏晓加入旅团（苏晓拒）；提醒苏晓「生灵花别直接当世界钥匙、会传送到黑渊最底层、那有个叫**刀魔**的恐怖存在——刀魔已非纯粹灭法者、完全凭本能、可能前一刻护你下一刻杀你」；转达**旅团老大**想借苏晓与**恶魔族**谈一件事（苏晓可从中收费）；他本人要去对付麦哲伦、赶不上顶上战场。旅团老大很中意「灭法者独苗」苏晓。

## 阿普斯（变态契约者 · 生殖之神）

自称「阿普斯」（普里阿普斯·神话生殖之神）的契约者——暗绿短发、脸色苍白，养着一群啃尸的鬣狗状生物。专挑健康年轻女性逼问私密问题后「**种下种子**」、把她当母体（暗示繁殖类能力）。事后变成七八岁男孩离去。极度恶心变态。

## 斯图茜（欢乐街女王 · CP0 间谍 · 殁）

**欢乐街女王·斯图茜**——达坦奥岛销金窟「狂欢酒店」的老板、地下世界头目之一（皮肉生意之最）、与夏琪同代却容貌二十几岁。实为 **CP0 多面间谍**（六式·指枪 / 界断线 / 剃），上次苏晓在海贼世界被 CP 发现就是她暗中动的手脚。

- **被苏晓拷杀（二十五卷13–14章）**：[布布汪](布布汪.md) 潜入蒸汤室定位、苏晓秒杀其卷发护卫后用 [放逐](../力量体系/装备道具.md) 碎刃钻进她呼吸道逼供，从她「极力拖时间」反推出岛上藏着天龙人。问出情报后碎刃刺心击杀（得世界之源 7.5% + 金色套装宝箱 + 天赋噬灵者 +26 法力）。她死后尸体还被某存在借用一次（自拧断颈）。

## 埃迪·克 / 花（CP0 追杀小队 · 殁）

**埃迪·克**详见其专页 [埃迪·克](埃迪·克.md)（上次追杀苏晓的 CP0 头目·有追踪能力·这次被苏晓死局 + 精神污染斩杀于二十五卷16章）。**花**——其同伙女特务，脚踏鲸鲨立于海面、负责追踪定位。苏晓绑天龙人出海当饵把二人引出。

## 唐吉诃德某人（绿发天龙人 · 被绑）

达坦奥岛上的天龙人、**姓唐吉诃德**（多弗朗明哥的家族）——绿发、被海水冲乱发型仍执着整理（头可断发型不能乱），被劫后异常镇定、不叫嚣身份。苏晓狙落其坐骑怪鸟生擒、当饵引 CP0（建议苏晓找七武海多弗朗明哥联络赎金）。

## 克尔拉 / 雷兹·马克斯（革命军）

跟踪苏晓许久的革命军二人——**克尔拉**（身材丰满、粉色连衣裙的革命军成员）+ **雷兹·马克斯**（圆顶帽白西装侏儒·一生没输过的著名赌徒·离过两婚）。在苏晓绑天龙人出海时现身、有事相求（苏晓让其等他活下来再谈）。

## 蒙奇·D·龙 / 克尔拉 / 哈库（革命军）

- **蒙奇·D·龙**：[革命军](../势力/index.md) 首领、被世界政府称为「世界上最凶恶的罪犯」（革命军即起义军·目标推翻世界政府·世界政府恨他远超海贼）。红色纹身 + 深绿大衣 + 狂野黑发、冷静中透凶悍。是路飞之父、卡普之子。早就派人调查苏晓（情报等级与贝加庞克同级·一度想除掉他·后因苏晓叛出世界政府而松气）——苏晓加入革命军对他而言是「天上掉馅饼」（有没有毒不得而知，但单一个圣痕药剂就能给革命军添一大批新生力量·舍不得放弃）。
- **克尔拉**：革命军成员、丰满 + 粉色连衣裙、化妆后容貌微变（潜入任务）——萨博的弟子。本只是来谈炼金炸弹生意，却被苏晓**步步算计**：苏晓借「加入革命军挡 CP 部门」为名入伙，再用「替革命军调配药剂」为幌子**白嫖革命军的稀有材料刷自己的炼金「植物调和篇」熟练度**（圣痕药剂 → 黎明之焰），每开一份新清单就要上百份昂贵材料。她全程「被算计了」的预感越来越强、被苏晓的智商嘲讽噎得跳脚，最终「讨厌所有科研人员」。
- **哈库**：鱼人空手道教官、培养革命军新血（无官职但地位高·很多干部是他带出）——本坚决反对给学员用圣痕药剂（「只有刻苦获得的力量才属于自己」），见服药的三个孩子突飞猛进后「真香」、改口圣痕是「对天赋的激活」。

## 青雉 / 战国 / 赤犬（海军本部）

- **青雉·库赞**：海军大将、冰冻果实（自然系）——顶上战场上**单挑被苏晓大修后的白胡子**（见上）反被薙刀柄扫飞落海（由此立下「绝不能一名大将对上白胡子」的判断），转而冻结港口海面给双方造落脚点。后缠斗苏晓 + [阿姆](阿姆.md)：阿姆**克制**他（被冻只想睡觉 + 能吞噬冰能量），苏晓用众生之地启发的新刀招（刃道刀·极 / 刃风暴）擦伤其小臂。
- **战国**：海军元帅（佛之战国·人人果实大佛形态）——顶上战争的总指挥，公开宣布艾斯是罗杰之子。对苏晓的**仇恨度爆表**（苏晓的越狱 + 杀天龙人令海军今年军费被削 10%·对白胡子仇恨度若 10、对苏晓就 90+），一见苏晓上莫比迪克号就连骂「这混蛋」、下令黄猿「他进战场就格杀勿论」——最怕苏晓用阿波罗炸平马林梵多。**真目的**被苏晓识破：办处刑不为杀艾斯、而是借机弄垮白胡子 → 打破新世界平衡 → 海军入驻新世界（为此不惜烧海军公信力·随机处刑时间 / 切映像电话虫 / 岩浆拳「误杀」百余自己人由他背锅）。赤犬战死后赤膊大佛形态亲自参战稳军心。战争结束时被苏晓的阿波罗威胁 + 红发调停、只能放苏晓携艾斯离开（恨得发抖）。
- **赤犬·萨卡斯基（殁）**：海军大将、岩浆果实、作风铁血（「海贼 = 需被铲除」的偏执）——被白胡子一脚踢回海军本部墙上重伤，又被苏晓「晶体层中空手臂诱敌 + 龙影闪绕背 + 青钢影克岩浆」断其右臂、**最终被 [血之兽](../力量体系/天赋.md) 直接炸死**（苏晓得 1900 战争功勋 + 史诗宝箱 47%·主线任务3 完成度 175%）。尸体被苏晓抛向海军摧毁士气、逼战国亲自参战。

## 莫利亚 / 黑胡子 / 红发（七武海 / 新登场四皇势力）

- **月光·莫利亚（王下七武海 · 殁）**：影子果实、七武海中垫底——被黄猿坑（与黄猿夹击苏晓后黄猿撤去围白胡子、留他一人），想要苏晓的刀和尸体却被反杀（影法师挡不住·脊柱被踹断物理控制后斩首）。「一个被吓破胆的海贼根本战胜不了苏晓」。
- **黑胡子·马歇尔·D·蒂奇**：暗暗果实（自然系·无法元素化、用黑暗吸伤害但承受翻倍疼痛、无法噬灭青钢影）——隐忍几十年、专挑白胡子最虚弱时携十人现身（海军内应「泰勒」是契约者·麦哲伦被某契约者用毒「吃掉」故他能出推进城）。硬抗苏晓三颗阿波罗 + 灰酸毒不死、被斩瞎一眼，盖黑布夺白胡子的震震果实——**被苏晓的心脏维续装置干扰、夺果实临时失败**（野心已彻底放出）。曾邀苏晓入伙被拒（苏晓：「你们太丑了」）。
- **红发·香克斯**：四皇、霸王色霸气海贼世界最强——战争尾声亲临**调停**（与德岛海战调停大妈同一作风·不愿时代剧变），带走白胡子尸体安葬。战国卖他面子宣布战争结束。问苏晓「你就那么热衷战争吗」，苏晓答「我掌握的技术就是为战争而存在」。

## 阿普斯 · 续（诞生计划 / 坑队友）

变态契约者阿普斯（见上「阿普斯」条·寄生 / 催生系）在顶上战争中放出「诞生」之物（孕妇腹中爬出的透明异空间生物·曾去刺杀战国）。临时队友间内斗：他**坑死五阶契约者「黑耀」**（左轮枪手·被白胡子捏成无头尸·二人本就想互坑）、又在白胡子击杀分成中**夺得 15.7% 击杀贡献**（全场契约者最高）。战后在联络平台疯狂炫耀、被内应泰勒（10.3% 贡献）问候全家。

## 黑月 / 花影（五阶契约者情侣 · 海军阵营 · 殁）

一对 360° 无死角秀恩爱的五阶契约者情侣（黑色 / 紫色眼影），海军阵营、五阶中小有名气——**花影**主感知 / 控制 / 防御（荆棘藤蔓 + 预判报点 + 吸体力的花粉光柱），**黑月**主突进杀敌（黑色颗粒掌刀 + 半透明虚化闪避）、组合技「黑影花月 / 残月黑影」。靠秀恩爱扰乱敌人心态。围攻苏晓时（青雉被阿姆拖住）一度险些得手，被苏晓充当诱饵反杀——放逐碎刃钻入黑月体内绞脏器、断二人手臂后斩成四截。死前仍放黑色光柱想拖苏晓同归于尽（「轮回乐园的五阶契约者没有会怂的」）。给苏晓留下首次受伤（吐血）。

## 波特卡斯·D·艾斯（营救目标 · 罗杰之子）

白胡子海贼团二番队长、海贼王罗杰的亲生子、卡普的孙子——被海楼石手铐反缚跪在处刑台上待处刑。苏晓眼中他是「**断魂影之石**」（主线任务 1「营救艾斯」的奖励 = 断魂影之石·残缺×1·难度 Lv.48），救法很「粗暴」（不配合就打晕拖走）。是出了名的「作死小能手」——童年经历让他不愿承认罗杰是父亲（「我老爹只有一个，那就是白胡子」）。**被救活**：布布汪戏耍战国手铐 + 趁战国把他锤半死时狼族传家宝传送到苏晓身旁，阿姆扛着他突围、路飞 / 甚平接应——战争结束后随白胡子团回新世界（亲眼送别战死的白胡子）。

## 哮天团 / 艾露莎（契约者中型冒险团）

潜伏冰墙后观望的契约者中型冒险团——名字被女团长力排众议起成「哮天团」（被吐槽像「哮喘天团·一群病秧子」）。团长**艾露莎**（自称·团员叫她「女神经」·性感女神级容貌 + 神经质性格·毫无威信却深得人心）。侦查系妹子测出苏晓是「强到脾脏炸裂」的五阶、唯一能测的魅力属性只有 2 点（被当成弱点·实则三阶团根本冲不动）。
