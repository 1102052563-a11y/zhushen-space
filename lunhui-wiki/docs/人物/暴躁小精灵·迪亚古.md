# 暴躁小精灵·迪亚古

[苏晓](苏晓.md) 以【**精灵意志**】（一只需要戴在右手上、激活时**耐久度如流水般下滑**的手套）召唤出的小精灵——⚠ 他的**潜入与探查能力高达 SSS＋**，是苏晓手上最强的侦察手段之一。⚠ 那么问题来了:这么强的能力，为什么没人召唤他？

## 🍺 出场即巅峰

⚠ 苏晓首次召唤他，是想让他去探查永封冰原上的情况——空间旋涡开启，**嘭的一声**，一道身影从里面飞起、**在茶几上滚了几圈**，那**有点圆滚滚的肚子格外显眼**。

- ⚠ 此时的他**醉眼惺忪，一手掐腰，一手扶着肚子**:「**老子还能喝，嗝～，呕！**」——⚠ 随即**双手扶着烟灰缸一番狂吐，最后吐的肚子都重新平了**；他迷糊地环视一圈，发现了桌上的毛巾，⚠ **误将其当成被子往身上一盖，开始呼呼大睡**。
- ⚠ 苏晓、[布布汪](布布汪.md)、[巴哈](剧毒魔鹰·巴哈.md) 都看着已经**睡出鼾声**的他，算是知道了，为何没人召唤这潜入与探查能力 SSS＋ 的小精灵——⚠ **对方不仅经常口吐芬芳，还是个酒鬼**。这次探查永封冰原自然是泡汤了。

## 🕵 一瓶精灵秘药就搞定的追踪

⚠ 神父让本世界所有占卜师都躲了起来（[占卜师·佩莉儿](风海大陆/占卜师·佩莉儿.md) 与部落先知同时销声匿迹），苏晓找不到违规者 [贝芙丽](幽暗大陆/贝芙丽.md) 的踪迹——⚠ 这时候，这位酒鬼派上了大用场。

- ⚠ 银月高悬、已快到深夜时分，一道小身影**穿过玻璃**、在半空中逐渐恢复到实体后落在桌上:「啧～，**找到目标了，你怎么谢老子？**」「……」「**别不说话！我可是差点被逮住**……」——⚠ 苏晓抛出**一瓶精灵秘药**，他**当场改口**:「**老板，我们这边走，穿件厚衣服，外面下着雪呢。**」抱着秘药**眉开眼笑**。
- ⚠ 苏晓的总评是:「如果忽略这家伙的**口吐芬芳、抽烟、酗酒、见到小精灵妹子就迈不动步，以及酒后格外聒噪**外，⚠ **暴躁小精灵·迪亚古的潜入与追踪能力都毋庸置疑**。」他成功摸清:贝芙丽与**总计四名违规者队友**一同，藏身在**海族主城**——且**谨慎地没去深度探查，以免打草惊蛇**。
- ⚠ 海之心大酒店的巡查任务中，他与巴哈一组从 189 层往下查（巴哈感知违规者气息、他负责追踪）。⚠ 「巡查小队的能力布置很合理，**但性格配置不太合理**」——巴哈与他一起做事后，⚠ **音频通讯频道都快废了**，就听他们两个**互相问候**；⚠ 更离谱的是，**这两个家伙在互相问候时，并不耽误做正事**。
- ⚠ 他的**召唤姿势**也很有个人特色:苏晓激活召唤，先听闻**一声破风**，之后就是**绷的一声、利器钉在天棚上震动的声音**——那是他的**锥枪**；⚠ **至于他本人，则钉在一旁**。
- ⚠ 苏晓撤离时给布布汪、巴哈开了恶魔传送阵，⚠ **至于他，「对方有自己的撤退手段」**。

## 关系

- [苏晓](苏晓.md)：以【精灵意志】召唤他的雇主——用一瓶精灵秘药就把「你怎么谢老子」变成了「老板」。
- [巴哈](剧毒魔鹰·巴哈.md)：海之心大酒店巡查的搭档——两人把音频通讯频道变成了互相问候专用频道（且不耽误正事）。
