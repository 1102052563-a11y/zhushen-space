# 埃罗·尼克

全名 **埃罗·尼克·普里斯特利·扎卡赖亚斯·特罗洛普**——[苏晓](苏晓.md) 在轮回乐园 [试炼场](../世界/地点/index.md) 召唤的**高级镜像**（每小时 10000 乐园币）。与以往召唤的动漫角色不同，他**来自虚空**（并非任何衍生世界的人物）：

- **身份**：龙血军团·第三团战斗队长、耿勒米战斗学院剑术顾问，人称「**战场杀人狂**」。来自六阶空间战场「敦克星」、实力比大多数六阶契约者还强（属六阶空间战场的 BOSS 级·若全力杀苏晓不超 5 秒）。生平参战 106 次、负伤 106 次、杀敌 19843 名，最终被敌国 65207 人围攻、杀 1296 人后战死，英灵殿立其雕像。
- **痞气**：平民窟出身、痞里痞气；左手是 M3 合金假肢（「挚爱」的公主因他睡了她妹妹而砍断、事后又演成「人生第一次 3P」的荒唐故事）。

## 对苏晓的指导（十卷82–83章）

他用一把巴掌宽的**重剑** + 小腿暗藏的锋利金属丝带，几下就把苏晓划得见到内脏（试炼场受伤不死、但有真实痛感）。**他不教刀术招式、而是传授「大型战场的生存经验」与踢技**：战场上「活下来才是胜者」、别理远敌、靠感知守株待兔、关键时装尸体……镜像将散时，这个「战场杀人狂」竟流露出对死亡 / 灵魂消散的畏惧。
