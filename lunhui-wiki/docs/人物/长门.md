# 长门（佩恩）

[火影忍者世界](../世界/任务世界/火影忍者.md)「**晓**」组织的**首领**、**雨忍村真正意义上的首领**——一头暗红色头发的男人。

- **枯槁之躯**：身形枯槁、面色惨白、瘦得肋骨清晰可见，**背上插着一根根黑色铁棍**（带连接点），双手接在一台简陋的机械操控装置上、靠它维生，**身体一天不如一天**（常年透支身体能量）。故他**不亲自出面**，由「佩恩」代行。
- **佩恩 = 六具尸体**：长门能操控**六具尸体**、赋予各异能力，对外统称「**佩恩**」——**天道（核心·由旧同伴「[弥彦](../世界/任务世界/火影忍者.md)」的尸体制成）/ 修罗道 / 人间道 / 畜生道 / 饿鬼道 / 地狱道**。天道无论言行都即是长门本人。心腹是蓝发女子 [小南](小南.md)（与弥彦同为长门旧友）。
- **理想（此时期的晓）**：长门**厌恶战争**——认为要终结战争须先掀起更大的战争、打破忍村制度才有真正和平。晓的宗旨：聚钱→挑起并扩大战争破除旧制→**抓捕尾兽做兵器统一世界**。这一时期的晓「以极端手段创造和平」、善恶不明（后期落入 [宇智波带土](../世界/任务世界/火影忍者.md) 手中才成真正的恐怖组织）。

## 招募苏晓（九卷15–17章）
长门看中 [苏晓](苏晓.md)（白夜）「刺杀四代水影、刀术堪比**木叶白牙**」的资料，派 [宇智波鼬](宇智波鼬.md) / [干柿鬼鲛](干柿鬼鲛.md) 招他入伙、给**正式成员**身份（一枚空闲戒指）——用途是**近身牵制人柱力**（捕尾兽计划的关键空缺·原担此任的「枇杷十藏」已阵亡·组织内除飞段无人敢近身人柱力）。这恰好暗合苏晓「吸尾兽查克拉喂青钢影」的野心。长门叮嘱小南「他来路不明、交涉别靠太近」（疑其与木叶白牙有关、需观察）。

- **与鼬的约定**：[宇智波鼬](宇智波鼬.md) 加入晓时与长门有约——鼬替晓做事，作为回报，**长门在鼬活着时不袭击木叶、不抓九尾**。鼬虽因万花筒写轮眼侵蚀、视力身体大减（十几米外都模糊·不久将为弟弟赴死），但「愤怒的鼬」连长门也忌惮。故苏晓判断：**长门会在两个月内袭击木叶**（否则任务难度更高·主线最终环「凋零之木」=协助长门毁灭木叶）。

## 封印一尾 · 外道魔像（九卷40–43章）
苏晓三人活捉一尾人柱力 [我爱罗](我爱罗.md) 后运至河之国川流谷山洞，由天道佩恩主持封印：

- **幻灯身集合**：长门用「幻灯身之术」召集全晓成员投影（共七道虚影 + 苏晓三人本体），众人站上**外道魔像**的十根手指就位。
- **外道魔像 / 十尾**：天道召出的庞大雕像，**由浓郁到结成实体的查克拉构成**、九只眼睛——是**十尾的躯壳**；每封印一只尾兽便睁开一只眼，集齐九尾兽则十尾复活（**十尾真面目 = 神树·查克拉的起源**·九尾即从十尾剥离而出）。以「**封印术·幻龙九封禁**」历时**三天**把一尾守鹤从我爱罗体内抽离封入魔像——封印完成时我爱罗因尾兽被剥离而死。苏晓暗自心惊（守鹤已被他吸走大半、显得「瘦弱」），以「出手太重 / 砂忍加强了封印」蒙混过天道的疑问。
- **临时身体 / 五封结界**：面对砂忍 + 木叶的追兵，长门以「**临时身体**之术」（不出本体·每次耗约 30% 查克拉）派鬼鲛迎战 [迈克·凯](迈克·凯.md) 队、鼬迎战卡卡西队（皆败·临时身体仅本体五成实力）；又在山洞布**五封结界**（须五人同时破解五处封印才能攻入）拖住追兵，为封印争取时间。

## 重伤 · 被苏晓手术救活（九卷71–75章）
杀师（[自来也](../世界/任务世界/火影忍者.md)）后，他与一伙神秘势力（契约者）交战、被一名**四阶契约者**临死前留下**胸口约 20 公分的圆形腐烂伤**（内含寄生甲虫）险些丧命——此后身体一天不如一天、红发转白、六具佩恩只能同控三具、间歇昏迷（不敢睡眠）。只有 [纲手](纲手.md) 能救、却未必肯接受。

- **交易：救命换毁灭木叶**：[苏晓](苏晓.md)（为推进主线·长门一死任务恐失败）主动提出**全力治疗换长门答应毁灭木叶**——长门一口应下（他本就要去木叶抓九尾 + 毁掉那个「把幸福建立在他人痛苦上」的忍村）。
- **疯狂的手术**：苏晓（实则不会手术、只懂人体解剖学）以 [1 号药剂](../力量体系/消耗品材料.md) 稳住他，再调出一瓶**暗紫猛药**（香磷血液 + [柱间细胞](../力量体系/消耗品材料.md)·每秒回 130~190 生命·有树木化 / 即死等高风险·靠长门**漩涡一族体质 + 轮回眼**硬扛）→ 拔出他背后**87 根黑色金属棍**（信号接收器·禁术隐患·约 200 斤）、徒手校正严重变形的脊柱（防高位截瘫）、清创开胸**取出胸腔里酸血的智慧甲虫**（抽其神经破共生）。手术后给他**重新镶嵌 4 根金属棍**（手臂两排黑钉 + 12 颗黑耳钉·小南拦下「鼻环」之议）。
- **重获新生**：两天后醒来——**十几年来首次能平躺、自由行走**（脊柱恢复、胸腔生物消失、脏器复活），红发回暗红。虽仍需休养（起身吐血），但承诺「**恢复后第一时间去攻打木叶、带苏晓同往**」。此役也化解了双方因尾兽事件的不快。

## 毁灭木叶（九卷76–83章·主线最终决战）
康复后，他与 [小南](小南.md)、[苏晓](苏晓.md) 谋定攻木叶（先派苏晓清理巡查忍者、再以「畜生道高空抛入 + 通灵集合」六道佩恩潜入制造混乱）。小南为他用纸做假树、搭操控台。他要的不只是打垮、而是**彻底抹去木叶**：

- **神罗天征夷平木叶**：他不惜用**缩短寿命的大招**（六具佩恩力量集于天道一具），趁苏晓拖住爆走的九尾、纲手被对峙吸引之际，一记**超规格「神罗天征」把整座木叶夷为平地**（村中心化为巨坑·普通村民死亡 99%+、下忍近乎全灭）——这正达成了苏晓的主线「凋零之木」。
- **未能擒九尾**：九尾（九喇嘛夺身的鸣人）太强（一记尾兽炮就把天道轰飞插进火影岩），佩恩们与之缠斗难下；苏晓自知不敌、亦料定团藏 + 木叶上忍 / 暗部不会坐视，遂在主线完成后**返回乐园**（向天道道别「祝你好运」），把九尾的结局留给了长门。

## 结局 · 被鸣人感化而死（十五卷2章·苏晓再入火影时揭）

苏晓昏迷一月醒来后从 [小南](小南.md) 口中得知：上次进攻木叶后，**长门被木叶下忍 [漩涡鸣人](../人物/漩涡鸣人.md) 感化**——鸣人继承了 [自来也](自来也.md) 的意志、那套「以痛苦换和平、以理解止仇恨」的道理打动了长门，他遂**放弃用轮回眼踏平木叶、并就此而死**（小南只委婉说「安详地离开了」）。这等于**背叛了为尾兽计划付出甚多的其他晓成员与 [带土](宇智波带土.md)**——小南因此连雨忍村都不敢回。苏晓评：以长门恢复后的状态 + 那双轮回眼，本足以踏平木叶（五代火影 [纲手](纲手.md) 绝非其对手），他的死「很异常」。这也是「漩涡鸣人英雄」叙事的底色——鸣人「感化」长门的背后，是木叶趁机吞下雨之国经济（见 [火影忍者·忍界公敌](../世界/任务世界/火影忍者-忍界公敌.md)）。
