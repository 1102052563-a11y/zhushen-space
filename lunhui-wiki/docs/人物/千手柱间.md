# 千手柱间（初代火影 · 忍者之神）

火影世界公认的最强者之一、被尊为「**忍者之神**」——初代火影、千手一族之祖。他天生掌握独一无二的**木遁**（别人顶多生几十棵树·他能直接生出一片能杀敌的森林、巨型木人能与完全体须佐能乎正面对抗）、仙人体生命力与领袖魅力都冠绝忍界（不结印即可治愈伤口）。他与 [宇智波斑](宇智波斑.md) 是「无话不谈的挚友与战友」、又是终生宿敌——两人生前那场决战柱间略胜、重伤后不久去世，而斑活了很多年、最终觉醒轮回眼超越了他。**柱间细胞**（[苏晓](苏晓.md) 反复利用的素材源）正出自他。

## 被大蛇丸转生 · 与斑再战（十五卷92–94章）

[大蛇丸](大蛇丸.md) 复活后成佐助的「人生导师」、**秽土转生出四代火影**（初代柱间 / 二代扉间 / 三代日斩 / 四代水门）助阵忍者联军。秽土转生的柱间约有生前 7 成实力（仍是不死 + 无限查克拉 + 领袖核心）。他不愿与斑再拼死（「都死了多年、何必」），却被斑以完全体须佐能乎逼战——他靠木遁森林 / 巨型木人与斑势均力敌，却警觉到斑身旁那个气息「怎么看都不像好人」的 [苏晓](苏晓.md)。

## 苏晓的猎杀目标（十五卷93章·进行中）

苏晓盘算了一个**疯狂计划**：让带土复活斑的同时**把柱间也复活**、再由他单挑柱间（柱间能吊打一名影·苏晓生存率不超 3 成）。理由是秽土转生的柱间不死 + 无限查克拉 + 领袖魅力、战争中作用太大必须除掉；而**苏晓握有「原始柱间细胞」、自认能削弱柱间**（柱间已死太久、其一切都被研究透·柱间细胞并非完美）。他与斑约定互相吸引火力 / 断后，并已 1 公里外用蜘蛛女皇 + 消音器**狙击柱间作火力支援**（柱间木遁森林挡子弹·中三枪才被斑引离主战场）。

## 真正复活 · 被苏晓削弱 · 苏晓单挑（十五卷96–99章）

黑绝附身带土、用带土的轮回眼**把斑与柱间真正复活**（不再是秽土转生·有了心脏与呼吸）。苏晓与斑演双簧骗他、再抛出纲手的带血金发 + 照片相威胁（「杀我或被我杀·否则那女人每半小时少一个零件」）把他留下单挑——其实苏晓敬重强者、并不真拿纲手要挟。

苏晓**早用「[PSX-17 型炼金实验药剂](../力量体系/消耗品材料.md)」削弱了他**（先在纲手身上试效·斑体内也有柱间细胞故信此能力）：他的千手生命力被压成「半休眠」、木遁报废（树界降临沦为「草遁·豆芽菜」），各项属性削 15 点。[使徒之眼](../力量体系/技能.md) 探得数据——**力 74〔原 89〕/ 敏 70〔原 85〕/ 体 78〔原 93〕/ 智 73〔原 88〕/ 魅 70·查克拉量 22762（尾兽级·一尾≈3 / 九尾≈40 / 他≈15~17）**；技能含千手血脉 Lv.50（生命 +8000）/ 木遁 / 怪力·阳 / 五重罗生门 / **仙人模式 Lv.20**（属性各 +10·木遁 ×3·此技能用自然能量、**未被削弱**）。**全盛的他是四阶中上**（仙人模式体力破 100），被苏晓削到「三阶顶尖」（仙人态仍力 84 / 敏 80 / 体 88 / 智 83）——这是苏晓豁出生存率换传说级以上宝箱的「天赐良机」。

## 与苏晓的死战（十五卷98–99章·进行中）

他开仙人模式应战：用从伤口生出的树枝手臂硬接阿姆的无畏冲锋、再以**水遁·水断波**（千手扉间招式·仙人强化）把阿姆头 / 腰刺个对穿（阿姆诈尸·核心阵图未破不死、反抓他小腿打断忍术）。面对苏晓他发现——皮肉伤是小事、**青钢影 + 生灵切割造成的体内伤才致命**（恢复力被削到 26 秒才愈合一道浅伤·普通几秒就好），遂改打中距离忍术消耗战（黄泉沼 / 天泣…）拖时间等生命力解封。斩龙闪（淡金）对他仙人体的肉体「还不够锋利」、只能斩出浅痕。
