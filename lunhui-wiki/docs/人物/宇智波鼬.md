# 宇智波鼬

[火影忍者世界](../世界/任务世界/火影忍者.md)「**晓**」组织的成员——宇智波一族出身、**写轮眼幻术高手**（苏晓评其「弟控」）。九卷15章与搭档 [干柿鬼鲛](干柿鬼鲛.md) 一同在火之国边境雪原**拦下 [苏晓](苏晓.md)**。

- **眼力毒辣**：苏晓尚未看清便已警觉拔刀，鼬一眼判定「**这家伙感知很敏锐、不是浪得虚名、低级幻术对他无效**」（苏晓的高智 + 精神抗性确能豁免一般幻术·唯顶级如月读除外）。
- **来意 = 招募**：晓组织主动找上这位「刺杀四代水影、公然袭击雾忍村」的火影顶尖恐怖分子——正值苏晓被铁之国 [三船](三船.md) 拒之门外、加入势力无门之际（详见 [火影世界·招募](../世界/任务世界/火影忍者.md)）。
