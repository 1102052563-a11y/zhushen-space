# 宇智波鼬

[火影忍者世界](../世界/任务世界/火影忍者.md)「**晓**」组织的成员——宇智波一族出身、**写轮眼幻术高手**（苏晓评其「弟控」）。九卷15章与搭档 [干柿鬼鲛](干柿鬼鲛.md) 一同在火之国边境雪原**拦下 [苏晓](苏晓.md)**。

- **眼力毒辣**：苏晓尚未看清便已警觉拔刀，鼬一眼判定「**这家伙感知很敏锐、不是浪得虚名、低级幻术对他无效**」（苏晓的高智 + 精神抗性确能豁免一般幻术·唯顶级如月读除外）。
- **来意 = 招募**：晓组织主动找上这位「刺杀四代水影、公然袭击雾忍村」的火影顶尖恐怖分子——正值苏晓被铁之国 [三船](三船.md) 拒之门外、加入势力无门之际（详见 [火影世界·招募](../世界/任务世界/火影忍者.md)）。

## 钓鱼擒蛇 · 收香磷（九卷42 / 50–51章）
封印一尾的追击战中，他以「临时身体」迎战卡卡西队（败·身体每况愈下、不轻易动眼）。其后设下一场**钓鱼局**：用苏晓夺得的 [卡卡西](卡卡西.md) 写轮眼当鱼饵，引出对宇智波血统疯狂执着的 [大蛇丸](大蛇丸.md)——在大蛇丸于实验室围困 [苏晓](苏晓.md) 时率鬼鲛现身（在场「大蛇丸」实为替身、被他识破）。但他**故意放水**让真大蛇丸与 [水月](鬼灯水月.md) 逃脱——为帮**宇智波佐助**成长（佐助早一天灭大蛇丸领便当，长门就早一天袭木叶，正合苏晓主线）。事后他力排众议**留下感知型忍者 [香磷](香磷.md)**（晓下一任务要抓**四尾**、只知大致方位需感知协助）。
