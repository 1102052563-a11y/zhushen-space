# 宇智波鼬

[火影忍者世界](../世界/任务世界/火影忍者.md)「**晓**」组织的成员——宇智波一族出身、**写轮眼幻术高手**（苏晓评其「弟控」）。九卷15章与搭档 [干柿鬼鲛](干柿鬼鲛.md) 一同在火之国边境雪原**拦下 [苏晓](苏晓.md)**。

- **眼力毒辣**：苏晓尚未看清便已警觉拔刀，鼬一眼判定「**这家伙感知很敏锐、不是浪得虚名、低级幻术对他无效**」（苏晓的高智 + 精神抗性确能豁免一般幻术·唯顶级如月读除外）。
- **来意 = 招募**：晓组织主动找上这位「刺杀四代水影、公然袭击雾忍村」的火影顶尖恐怖分子——正值苏晓被铁之国 [三船](三船.md) 拒之门外、加入势力无门之际（详见 [火影世界·招募](../世界/任务世界/火影忍者.md)）。

## 钓鱼擒蛇 · 收香磷（九卷42 / 50–51章）
封印一尾的追击战中，他以「临时身体」迎战卡卡西队（败·身体每况愈下、不轻易动眼）。其后设下一场**钓鱼局**：用苏晓夺得的 [卡卡西](卡卡西.md) 写轮眼当鱼饵，引出对宇智波血统疯狂执着的 [大蛇丸](大蛇丸.md)——在大蛇丸于实验室围困 [苏晓](苏晓.md) 时率鬼鲛现身（在场「大蛇丸」实为替身、被他识破）。但他**故意放水**让真大蛇丸与 [水月](鬼灯水月.md) 逃脱——为帮**宇智波佐助**成长（佐助早一天灭大蛇丸领便当，长门就早一天袭木叶，正合苏晓主线）。事后他力排众议**留下感知型忍者 [香磷](香磷.md)**（晓下一任务要抓**四尾**、只知大致方位需感知协助）。

## 天照警告带土 · 引佐助赴死（九卷58–61章）
封印尾兽时听闻**大蛇丸被弟弟 [宇智波佐助](宇智波佐助.md) 所杀**，「阿飞」起哄要拉佐助入晓——鼬骤然开万花筒放「**天照**」黑焰烧向阿飞（被天道「神罗天征」震开），实因他已察觉带土是宇智波一族、以此**警告**。随后他配合 [苏晓](苏晓.md) 的安排：苏晓放归香磷传话「想报仇来神桥山」，把佐助引到**宇智波一族遗址**，鼬在此**刻意放水**、要让佐助亲手杀死自己——他「为村子和平可灭尽家族、唯独舍不得杀这个弟弟」，这场送死之局他已等待许久（[宇智波带土](宇智波带土.md) 断言鼬绝不会反杀佐助）。

## 与佐助决战 · 放水赴死（九卷62–63章）
在宇智波遗址与弟弟 [宇智波佐助](宇智波佐助.md) 决战（[苏晓](苏晓.md) / [带土](宇智波带土.md) / 黑白绝 / 布布汪旁观）：豪火球对喷 → **天照**黑焰焚敌 → 逼出藏在佐助咒印里的 [大蛇丸](大蛇丸.md)、以**须佐能乎（八尺镜 + 十拳剑）**「封印」之（大蛇丸分裂小白蛇逃脱、佐助由此被剥离咒印 / 大蛇丸、白得三年栽培）；佐助则以**麒麟**雷电反击。最终鼬**刻意放水**、以两指点佐助额头植入禁术、力竭撞壁而亡，留下遗言「**原谅我，佐助，这是最后一次了**」。

- **一盘隐忍多年的大棋（事后由大蛇丸 / 带土道破）**：鼬当年灭族后将佐助托付三代火影，三代被大蛇丸所杀后，他**故意让大蛇丸把佐助当「新容器」培养**（顶级老师 + 海量资源、又不会乱改造），并以写轮眼为饵借苏晓之手促成这场决战、让佐助亲手「杀」死自己——他「为村子和平可灭尽家族，唯独舍不得杀这个弟弟」。其尸体与佐助被 [带土](宇智波带土.md) 带走（带土欲收佐助为新成员、并告知真相）。
