# 结标淡希（空间系·领路人）

[学园都市](../世界/任务世界/魔法禁书目录.md) 的 **Lv.4 大能力者**（空间系）、**「领路人」**——专门接待非正常渠道进入学园都市的不速之客。双马尾、穿着暴露（牛仔服 + 胸口缠绷带露大片小腹·自称「防止女性特征妨碍战斗」）。

- **比白井黑子更强的空间能力**：不需接触、**只要在视线内的人 / 物就能移动**（最远 800 米·重量上限 4520 公斤）——比 [白井黑子](白井黑子.md)（须接触·80 米）威胁大得多。**心理阴影**：曾一次空间移动失误把自己一只脚「埋」进墙里（险些融进水泥），此后移动自身会强烈恐惧 / 呕吐（移动超四次就吐）。
- **腹黑的「借刀杀人」（十三卷9章）**：她目睹 [苏晓](苏晓.md) 割喉 [麦野沈利](麦野沈利.md) 后不敢露面，便**匿名联络 [白井黑子](白井黑子.md)**（她嫉妒白井能无忧无虑地生活、自己却困在黑暗世界）骗她来送死、想等苏晓杀光人冷静后再现身。可苏晓没全力追杀白井——正因 [结标淡希](结标淡希.md) 这种危险空间系在场，苏晓留着 [岚凯风衣](../力量体系/装备道具.md) 的**空间禁锢秘纹**没用（怕她把自己传进硫酸池 / 地下 800 米）。
- **成苏晓搭档**：她其实是来传亚雷斯塔的口信（「来见我」），随后被苏晓的凝实杀气吓服，老实领路。把苏晓、布布汪空间移动进统括理事会本部大楼见亚雷斯塔，事后又接到指令——**今后与苏晓搭档**（一脸生无可恋）。布布汪以 53 魅力 + 一记「汪之蔑视」把她噎得够呛。

## 被史提尔偷袭 + 成小队「Dark」一员（十三卷17–22章）

苏晓与神裂死战时，她本该在神裂第二次放唯闪时把神裂传送走——却被埋伏的魔法师 [史提尔·马格努斯](史提尔·马格努斯.md) **偷袭重伤（60% 烧伤）**，没能出手（苏晓因此与神裂打成两败俱伤）。靠 [冥土追魂](冥土追魂.md) 医治不留疤，但被火焰炙烤的滋味让她**记恨极深**——追猎神裂时她意外地积极，主动配合猎犬部队、还专挑更烈的酒递给苏晓水刑塞丽娜（有点恶趣味）。她正式成了苏晓小队「**Dark**」的一员（苏晓 + [布布汪](布布汪.md) + 她 + 狙击手 [叶戈尔](叶戈尔.md)）——一脸生无可恋、被布布汪挤在后排座揩油也只能忍。
