# 阿姆（凛冬牛头人 · 炼金生物）

[苏晓](苏晓.md) 通关恶魔古堡、把 [炼金生物制造学](../力量体系/技能.md) 冲到 Lv.20 满级后，**亲手制造的第一只高级炼金生物**（十四卷80–81章）——种类「凛冬牛头人」（魔兽 / 炼金造物·与人类无关）：三米多高、棕黄绒毛下肌肉扎结、头顶两根升腾寒气的弯角。本名「阿姆斯特朗」（苏晓嫌「阿姆斯特朗回旋加速喷气式阿姆斯特朗炮」太魔性、简称**阿姆**）。

## 制造与定位

- **来历**：图纸「**凛冬牛头人**」从 [黑帆商会](index.md) 黑商 24K 处购得（金色图纸·5 万币 + 1 瓶一号药剂·这种生物图纸受众小、压了很久）；制造耗 **23 万币级材料**（高级伪生骨骼 ×12 / 魔兽晶核大 / 高级生物魔导组织 ×42 / 灵魂碎屑 ×307〔灵魂结晶〔中〕×3 打碎〕等 + **3000 法力**·门槛极高·一次失败全报废）。
- **设计取向**：苏晓刻意要**体力型 + 控制型**炼金生物（关键时挡攻击、长时间缠敌），而非攻击型；并只给它**极低智慧**（能听简单命令即可·智商被压到「1+1=3」·说一口随机方言·憨厚老实）——因为「给炼金生物太多智慧会向往自由、反杀造物主」。
- **归属**：苏晓指定阿姆为**召唤物**（受苏晓 80 智力加成 → 属性更强）；战斗时**控制权交 [布布汪](布布汪.md)**（布布汪 = 业余奶妈 + 斥候 + 召唤师·苏晓专心挥刀）、平时由布布汪携带 + **负责喂养**（阿姆胃口恐怖·一顿吃掉约 180 币的食物·能消化一切有机物转能量·这也是它能存 256 天的原因）。阿姆喊苏晓「主人」、喊布布汪「二主人」。

## 属性 / 技能（十四卷81章）

`存在 256 天 ｜ 法力 600 ｜ 力 75 / 敏 69 / 体 85 / 智 30 / 魅 9`——

- **霜冻（被动 Lv.39）**：攻击附霜冻、减敌 10% 移速·可叠 10 层（每叠持续翻倍·最高 25 分 36 秒·减速恒 10%）；霜冻无冷却，随手一拳就续命。
- **无畏冲锋（主动）**：以敏捷 ×1.5 冲锋、途中减伤 50%、犄角刺穿造力量 ×2.5 伤 + 3 层霜冻。
- **狂野践踏（主动）**：连踩 4 次·每次基础攻击 ×1.2 + 1 层霜冻。
- **凛冬战吼（主动）**：10 米内 30 伤 + 3 层霜冻。
- **凛冬血脉（被动）**：**减伤 25%、减速抗性 +70%、防御 +36、生命 +1000、法力 +300**——极抗揍（85 体力 + 减伤·扛三五人集火无压力）。
- **凛冬之怒（终极 Lv.5）**：敌叠满 10 层霜冻后可远程强制**冻结** + 向周围扩散 6 米（耗 10% 生命·24h 冷却）。

战术：冲锋 → 践踏 → 战吼速叠 10 层霜冻 → 待机开终极冻结。攻击力一般、但**生存 + 控制顶级**——「只要被阿姆缠住，减速永不终止」。

## 首次随主入衍生世界（十五卷1–4章·火影）

苏晓返现实期间命阿姆「老实待专属房间」，这憨货便**真的坐在地上一动不动、滴水未进绝食等命令**（饿得牛眼发绿也不碰布布汪备的食物·直到苏晓回来下令「吃」才扑上去）。随后**首次随主进入衍生世界（火影忍者）**、对外面的世界很好奇。火之国晓据点处，苏晓揪出 [白绝](绝.md) 的细胞分身后，阿姆**抓着分身的腿狂抡摔打**（一分钟把它摔得抽搐 + 覆满霜冻冰晶·确认老实才扔回苏晓脚边），又在苏晓注射试剂时主动挡在身前「很有坦的觉悟」——初显其抗揍 + 控制的实战价值。

## 关系
- **[苏晓](苏晓.md)**：造物主（喊「主人」·绝对听令·性格难免被苏晓「带歪」、战时或显凶性）。
- **[布布汪](布布汪.md)**：战斗控制者 + 饭票（喊「二主人」·布布汪嫌这称呼像「二狗子」）。

## 五影大会 · 缠斗雷影护卫（十五卷10–15章）

铁之国雪风山等待时阿姆跟 [布布汪](布布汪.md) 玩「瞪眼对决」（谁先眨眼吃 50 斤雪·阿姆 85 体力硬扛、把布布汪瞪到含泪啃雪）。五影大会爆发后，苏晓一指令、阿姆**跃下高塔进入战斗状态**（牛眼血红、肌肉扎结、犄角寒气缭绕、如钢铁战车）——奉命**缠住 [四代雷影](四代雷影.md) 的护卫达鲁伊**，初当苏晓的「出手」棋子。

## 独挡风影 · 抗揍实证（十五卷16–26章）

五影大会苏晓斩杀雷影时，阿姆**独自拦住前来驰援的风影 [我爱罗](我爱罗.md) 一行三人**（被误认作通灵兽）——虽只挡了片刻，却全身落下深可及骨的伤口、还中了 [勘九郎](勘九郎.md) 的砂隐剧毒（脸色发青）。但这对它只算「轻伤」：靠 85 体力的恢复 + [布布汪](布布汪.md) 冰雪女神光环，伤势很快回七成。苏晓由此实测——**阿姆抗揍超乎想象**，斩龙闪破防没问题、但想断其骨几乎不可能（骨骼是硬度超金属的伪生材料）、短时间内连苏晓都杀不掉它。后又被刺客契约者**幽魂**一枪（200 币特制弹）打在额头、只砸出个铁饼（阿姆捡起当饼干嚼），引出隐匿的布布汪贴脸擒之。

## 忍界大战 · 被柱间刺穿仍「诈尸」（十五卷90–99章）

它在战场上撞我爱罗（报五影大会被暴打之仇）、助苏晓杀三影；单挑被削弱的 [千手柱间](千手柱间.md) 时，被柱间仙人模式的**水遁·水断波**把头顶 + 后腰刺了个对穿（高压水刀·头骨碎、露出内部炼金阵图）。但它**核心炼金阵图未破、并没死**——趴地装死后突然「**诈尸**」、从泥泽里伸手抓住柱间小腿、打断他的忍术（生理构造类生物、要害却不同·非破坏核心阵图 / 打爆头颅、它就能继续战斗）。此役由 [布布汪](布布汪.md) 接管它的身体控制权（配合比阿姆本意识更默契）。决战中它累计霜冻 10 层瞬冻柱间为苏晓制造杀机，最终**被柱间斩成两截（断一臂 + 失下半身）**——随主返园后等着苏晓「大修」（它对挨修比硬怼柱间更怵）。

## 蚁王篇 · 撞飞三护卫尤比（十六卷24–27章）

进全职猎人蚁王篇后它继续当苏晓的「肉盾 / 拷问机」——秒杀嵌合蚁群、抓嵌合蚁「乱砸」逼供。苏晓寂灭公爵抢三护卫 [孟徒徒·尤比](尤比·孟徒徒.md) 人头时，尤比直冲苏晓，**阿姆开冲锋迎头对撞、把强化系的尤比撞飞**（抗击打完胜·牛角刺它胸口两洞）；尤比化肉翅飞越，阿姆「很生气」——脱下 [布布汪](布布汪.md) 给它订制的 **65 码金属底大鞋**（配践踏能力）抛砸，连布布汪都本能畏惧那双大鞋。

## 出处
