# 阿姆（凛冬牛头人 · 炼金生物）

[苏晓](苏晓.md) 通关恶魔古堡、把 [炼金生物制造学](../力量体系/技能.md) 冲到 Lv.20 满级后，**亲手制造的第一只高级炼金生物**（十四卷80–81章）——种类「凛冬牛头人」（魔兽 / 炼金造物·与人类无关）：三米多高、棕黄绒毛下肌肉扎结、头顶两根升腾寒气的弯角。本名「阿姆斯特朗」（苏晓嫌「阿姆斯特朗回旋加速喷气式阿姆斯特朗炮」太魔性、简称**阿姆**）。

## 制造与定位

- **来历**：图纸「**凛冬牛头人**」从 [黑帆商会](index.md) 黑商 24K 处购得（金色图纸·5 万币 + 1 瓶一号药剂·这种生物图纸受众小、压了很久）；制造耗 **23 万币级材料**（高级伪生骨骼 ×12 / 魔兽晶核大 / 高级生物魔导组织 ×42 / 灵魂碎屑 ×307〔灵魂结晶〔中〕×3 打碎〕等 + **3000 法力**·门槛极高·一次失败全报废）。
- **设计取向**：苏晓刻意要**体力型 + 控制型**炼金生物（关键时挡攻击、长时间缠敌），而非攻击型；并只给它**极低智慧**（能听简单命令即可·智商被压到「1+1=3」·说一口随机方言·憨厚老实）——因为「给炼金生物太多智慧会向往自由、反杀造物主」。
- **归属**：苏晓指定阿姆为**召唤物**（受苏晓 80 智力加成 → 属性更强）；战斗时**控制权交 [布布汪](布布汪.md)**（布布汪 = 业余奶妈 + 斥候 + 召唤师·苏晓专心挥刀）、平时由布布汪携带 + **负责喂养**（阿姆胃口恐怖·一顿吃掉约 180 币的食物·能消化一切有机物转能量·这也是它能存 256 天的原因）。阿姆喊苏晓「主人」、喊布布汪「二主人」。

## 属性 / 技能（十四卷81章）

`存在 256 天 ｜ 法力 600 ｜ 力 75 / 敏 69 / 体 85 / 智 30 / 魅 9`——

- **霜冻（被动 Lv.39）**：攻击附霜冻、减敌 10% 移速·可叠 10 层（每叠持续翻倍·最高 25 分 36 秒·减速恒 10%）；霜冻无冷却，随手一拳就续命。
- **无畏冲锋（主动）**：以敏捷 ×1.5 冲锋、途中减伤 50%、犄角刺穿造力量 ×2.5 伤 + 3 层霜冻。
- **狂野践踏（主动）**：连踩 4 次·每次基础攻击 ×1.2 + 1 层霜冻。
- **凛冬战吼（主动）**：10 米内 30 伤 + 3 层霜冻。
- **凛冬血脉（被动）**：**减伤 25%、减速抗性 +70%、防御 +36、生命 +1000、法力 +300**——极抗揍（85 体力 + 减伤·扛三五人集火无压力）。
- **凛冬之怒（终极 Lv.5）**：敌叠满 10 层霜冻后可远程强制**冻结** + 向周围扩散 6 米（耗 10% 生命·24h 冷却）。

战术：冲锋 → 践踏 → 战吼速叠 10 层霜冻 → 待机开终极冻结。攻击力一般、但**生存 + 控制顶级**——「只要被阿姆缠住，减速永不终止」。

## 首次随主入衍生世界（十五卷1–4章·火影）

苏晓返现实期间命阿姆「老实待专属房间」，这憨货便**真的坐在地上一动不动、滴水未进绝食等命令**（饿得牛眼发绿也不碰布布汪备的食物·直到苏晓回来下令「吃」才扑上去）。随后**首次随主进入衍生世界（火影忍者）**、对外面的世界很好奇。火之国晓据点处，苏晓揪出 [白绝](绝.md) 的细胞分身后，阿姆**抓着分身的腿狂抡摔打**（一分钟把它摔得抽搐 + 覆满霜冻冰晶·确认老实才扔回苏晓脚边），又在苏晓注射试剂时主动挡在身前「很有坦的觉悟」——初显其抗揍 + 控制的实战价值。

## 关系
- **[苏晓](苏晓.md)**：造物主（喊「主人」·绝对听令·性格难免被苏晓「带歪」、战时或显凶性）。
- **[布布汪](布布汪.md)**：战斗控制者 + 饭票（喊「二主人」·布布汪嫌这称呼像「二狗子」）。

## 五影大会 · 缠斗雷影护卫（十五卷10–15章）

铁之国雪风山等待时阿姆跟 [布布汪](布布汪.md) 玩「瞪眼对决」（谁先眨眼吃 50 斤雪·阿姆 85 体力硬扛、把布布汪瞪到含泪啃雪）。五影大会爆发后，苏晓一指令、阿姆**跃下高塔进入战斗状态**（牛眼血红、肌肉扎结、犄角寒气缭绕、如钢铁战车）——奉命**缠住 [四代雷影](四代雷影.md) 的护卫达鲁伊**，初当苏晓的「出手」棋子。

## 独挡风影 · 抗揍实证（十五卷16–26章）

五影大会苏晓斩杀雷影时，阿姆**独自拦住前来驰援的风影 [我爱罗](我爱罗.md) 一行三人**（被误认作通灵兽）——虽只挡了片刻，却全身落下深可及骨的伤口、还中了 [勘九郎](勘九郎.md) 的砂隐剧毒（脸色发青）。但这对它只算「轻伤」：靠 85 体力的恢复 + [布布汪](布布汪.md) 冰雪女神光环，伤势很快回七成。苏晓由此实测——**阿姆抗揍超乎想象**，斩龙闪破防没问题、但想断其骨几乎不可能（骨骼是硬度超金属的伪生材料）、短时间内连苏晓都杀不掉它。后又被刺客契约者**幽魂**一枪（200 币特制弹）打在额头、只砸出个铁饼（阿姆捡起当饼干嚼），引出隐匿的布布汪贴脸擒之。

## 忍界大战 · 被柱间刺穿仍「诈尸」（十五卷90–99章）

它在战场上撞我爱罗（报五影大会被暴打之仇）、助苏晓杀三影；单挑被削弱的 [千手柱间](千手柱间.md) 时，被柱间仙人模式的**水遁·水断波**把头顶 + 后腰刺了个对穿（高压水刀·头骨碎、露出内部炼金阵图）。但它**核心炼金阵图未破、并没死**——趴地装死后突然「**诈尸**」、从泥泽里伸手抓住柱间小腿、打断他的忍术（生理构造类生物、要害却不同·非破坏核心阵图 / 打爆头颅、它就能继续战斗）。此役由 [布布汪](布布汪.md) 接管它的身体控制权（配合比阿姆本意识更默契）。决战中它累计霜冻 10 层瞬冻柱间为苏晓制造杀机，最终**被柱间斩成两截（断一臂 + 失下半身）**——随主返园后等着苏晓「大修」（它对挨修比硬怼柱间更怵）。

## 蚁王篇 · 撞飞三护卫尤比（十六卷24–27章）

进全职猎人蚁王篇后它继续当苏晓的「肉盾 / 拷问机」——秒杀嵌合蚁群、抓嵌合蚁「乱砸」逼供。苏晓寂灭公爵抢三护卫 [孟徒徒·尤比](尤比·孟徒徒.md) 人头时，尤比直冲苏晓，**阿姆开冲锋迎头对撞、把强化系的尤比撞飞**（抗击打完胜·牛角刺它胸口两洞）；尤比化肉翅飞越，阿姆「很生气」——脱下 [布布汪](布布汪.md) 给它订制的 **65 码金属底大鞋**（配践踏能力）抛砸，连布布汪都本能畏惧那双大鞋。

## 希亚大陆 · 牛皮糖拖死刑官（十七卷28–29章）

进 [希亚大陆](../世界/任务世界/希亚大陆-灭法遗迹.md) 后它继续当苏晓的「肉盾 / 拖延机」——审讯 [伊儿朵](伊儿朵.md) 时挂遥控闹钟捶地、配合护卫演「敌袭」迷惑俘虏；死刑官 [敦克·尼克](敦克·尼克.md) 来袭时，苏晓派它 + [布布汪](布布汪.md) 缠住另一名持宽刃短刀的死灵。它**凭凛冬血脉（减伤 25% / 防 +36 / 生命 +1000）+ 霜冻（近战附霜冻减速·最长 25 分 36 秒）+ 布布汪冰雪女神光环双倍回血**，成了「打不死、甩不掉」的牛皮糖——硬扛短刀劈胸（肌肉夹刀）、被电弧也不退，只为拖住敌人（攻击力依旧「辣眼睛」、不负责杀）。布布汪一度接管它的身体、用「若隐若现的气息」恐吓那名死灵。

## 希亚圣殿 · 融合冰蓝战锤 → 冰装甲（十七卷67–69章）

在希亚圣殿火焰区，阿姆被一把**冰蓝战锤**（[里德](陵墓守卫者·里德.md) 锻造、用于镇压某存在的冰属性武器·非常规兵器·会持续外放能量）吸引、不受控地拿起——因二者**冰属性能量相似度超 95%**。结果它**核心炼金阵图严重过载**（原本能量约 100、暴涨到 6000·几十倍）、失去意识、险些被撑爆。

苏晓作为它的制造者，剖开它后背、**给它构建了一条新的「能量渠道」**（冰蓝战锤 = 心脏·冰能量 = 血液·主渠道 = 动脉静脉·核心阵图 = 大脑·把冰能量循环回战锤、不再让核心阵图承担传输）——既解了过载，又**让大量冰能量长期保养核心阵图、变相延寿**。

!!! note "改造后的阿姆（十七卷69章·属性大涨）"

    - **存在时间：256 → 3756 自然日**（本质是「降低损耗」、非真延寿·冰能量保养核心阵图）；多出**冰能量 2000**（持战锤每秒回 5）。
    - 六维：力 **80**（+5）/ 敏 **73**（+4）/ 体 **88**（+3）/ 智 30 / 魅 **15**（+6）。
    - **霜冻（被动 Lv.45）**：减速 15%、可叠 10 层、最长 25 分 36 秒；**新增 7% 几率瞬间冰冻敌人 0.2~3 秒并直接叠满 10 层**（配合「凛冬之怒」的二次冰冻几乎冻到死）。
    - **冰装甲（终极·被动 Lv.13）**：承受任何伤害时 **40% 由冰能量替代**（受 10 伤 = 扣 6 生命 + 4 冰能量）——抗揍系数从 ~5 飙到 ~20，肉到敌人不想打它。
    - **有了武器**：从前只会抡王八拳，如今手持冰蓝战锤、附冰属性攻击（「有锤子的牛头人就是有梦想的牛头人」）。战锤完整属性需返园公证后解封。

**重要设定（炼金生物的「命」）**：炼金生物的存在时间**不取决于能量（占比 <10%）、而取决于阵图磨损 + 伪生材料**；**核心阵图 = 阿姆的大脑、存着它的记忆与意识，一旦破碎阿姆即彻底消亡**——重做一只也不是阿姆（呼应 [爱丽丝](爱丽丝.md) 重复利用狼管家核心矩阵终致其背叛）。苏晓认定阿姆「独一无二」、绝不会做替代品。

## 战锤公证 · 暗月沼泽吃霸主（十七卷84–85章）

希亚收尾，阿姆被苏晓留在死灵首都暗月沼泽的沼泽里泡着——它顺手**把方圆几百米的暗月藤啃食一空**，把当地霸主巨鳄吓得躲在藤缝里瑟瑟发抖（「直到它遇到一名『水牛』」）。返园结算时，那把冰蓝战锤露出真名「**[冰封怒哮](../力量体系/装备道具.md)**」（战锤·未公证）——苏晓花 **15 万乐园币 + 灵魂结晶〔大〕×3**（割肉价）为它公证，阿姆正式获得冰封怒哮的所有权（它已与战锤形成能量循环、非公证不可）。

## 海蓝星世界争夺战（十九卷·会合参战 → 集结冰原狼）

它在临海城市与苏晓 / 布布汪会合（先吃垮一家餐厅）——鲁贝城打残天启时随队作战：持冰封怒哮、放冰霜脉冲、破墙冲进敌方小队，扛着一发炽红箭矢只破了点皮（凛冬血脉 + 冰装甲让它「皮外伤」而已）。世界之核进雨林后，苏晓让它**北返去集结布布汪麾下那上千只冰原狼**（赶到热雨林之森附近约需两天）——为苏晓的「夺胜大计划」备一支奇兵。

## 暗鸦星 · 主坦阿姆诞生（二十一卷76–78章）

暗鸦星它吞了 [摩伊·诺拉](摩伊·诺拉.md) 守护的**紫色起源石碎片**——返园后陡然「**冰系能量匮乏**」、饿到啃光家具 + 啃自己里德造的寒冰战锤（甚至差点啃自己的核心阵图）。苏晓喂它近 80 万币的冰属性装备 + [里德](陵墓守卫者·里德.md) 弄来的「**幻光矿石**」（价值 20 枚传说之勋·当年那把寒冰战锤即此料锻成），它全吃光后吐出一团寒气、躺地大睡，**核心炼金阵图溶解、与肉身融为一体**——这是从「伪生物 / 炼金造物」向**真生物**的本质跃迁（恶魔古堡的 [爱丽丝](爱丽丝.md) 求而不得的转化·阿姆因核心阵图世代更替 + 寒冰战锤镇压的灭法之影虚空意识而办成了）。它外表不变、却能凭空生成 / 操控寒冰。

!!! note "进化后的阿姆（二十一卷78章 · 附属从者·凛冬战牛）"

    身份从「召唤物」升为 **附属从者**（可进现实世界·牛角伪装成壮汉）。
    - **冰能量 5000**（每分钟回 63·随智力）；六维 力 80 / 敏 73 / 真实体 **97**（+9）/ 真实智 **90**（+60·吸收未知残缺意识）/ 魅 19。
    - 技能 1 **凛冬战牛（主动 Lv.50）**：操控体内冰能量外放、塑形（凭空造冰武器）。
    - 技能 2 **寒冬血脉（被动 Lv.30）**：减伤 25%、减速抗性 +80%、防御 +36、生命 +1000。
    - 技能 3 **冰能量转化（被动 Lv.48）**：每点智力 +20 生命 + 0.5 防御（现 +1800 生命 / +45 防御）。
    - 技能 4 **斯提卡斯的呼唤（主动 Lv.60）**：极速生成 170 防御力的寒冰盾（耗 2000 冰能 + 1000 生命·24h CD）。
    - 技能 5 **冰装甲（终极·被动 Lv.17）**：受伤 50% 由冰能量替代（受 10 伤 = 5 生命 + 5 冰能）。
    - 技能 6 **凛冬之怒（终极·被动 Lv.10）**：瞬放全部冰能量、冻结方圆 300 米内**不分敌我**所有单位 + 已耗冰能 ×0.19 冰伤（**无冷却**）。

    **定位质变**：从前「四不像」（攻弱控弱·只强在减速），现在是彻底的**主坦 / 法坦**——打不扁、捶不烂、无 CD 硬控造冰，对上 [葬戈](葬戈.md) 也能缠很久（彻底放弃攻击、全防御）。完全放弃了原有的霜冻 / 冲锋等攻击向技能。

## 顶上战场 · 克制青雉 + 冰岩双形态（二十五卷）

- **克制大将青雉**：与冰冻果实大将 [青雉](顶上战场配角.md) 缠斗时，阿姆被冻只想睡觉、还能**吞噬冰能量**反补自己——青雉一时拿它没办法，等于帮苏晓拖住一名大将（扛着艾斯突围时全身大面积烧伤、被赤犬岩浆烤）。
- **🆕冰岩双形态（二十五卷54章·返园）**：苏晓给它戴 [守望者](../力量体系/装备道具.md)（成长项链·能量对立转换）+ 喂 [岩浆果实](../力量体系/装备道具.md)——阿姆可在**寒冰 / 岩浆**两形态间切换：冰形态 = 防御 / 控制（兼解决「不能下水」弱点·转冰即冻水面），岩浆形态 = 冲锋 / 改地形 / 强破坏（其岩浆是**炽红色**·温度胜赤犬·无赤犬那股阴沉侵蚀）。苏晓戏称它「生物自走空调」（岩浆制热 / 寒冰制冷·24 小时工作只需上百斤食物）。**镇压暴乱**时它的岩浆失手烫死了甲铁城世界的剧情主角生驹（连累苏晓被强制传送）。

## 🆕元素体质进化 · 「凛冬战牛」终极坦（三十卷70章·巫师大陆返园）

巫师大陆通关后，苏晓把斯芬克 / 莎布身上夺来的 **13 枚封之箱**卖给轮回乐园，换得**元素体质**赋予阿姆（苏晓自己绝魔体质用不了·阿姆最适合）——阿姆吐白沫倒地、岩浆 / 冰焰循环十几次后**吞掉缺陷的岩浆果实能力**、转化为生命力，彻底蜕变为更纯粹的「**附属从者·凛冬战牛**」：

- **不屈战牛（被动 Lv.70·元素体质所赋）**：减伤 30%、冰系减速抗性 +85%、防御 +70、**生命 +38000**。
- **冰能量转化 / 冰装甲 / 冰焰生命**：每点智力 +50 生命 + 0.5 防御；承伤的 50% 由冰能量替代；每损失 10% 生命就提升回血速度（最高每秒回 56）。
- **元素体质（核心 Lv.MAX）**：极强自然元素亲和力·每完成一轮元素循环就微量永久 +生命 / 体 / 智——**只会越来越抗揍、还能继续变强**。

六维定格：力 110 / 敏 95 / 体 129 / 智 128 / 魅 29。苏晓评其「拥有最肉的能力、挨最毒的打」（攻击略降·坦度面对六阶大 boss 也丝毫不虚·布布汪一头撞上去自己屁股墩、阿姆毫无知觉）。

## 🆕极寒之地 · 「无解的坦度」围猎夜惑女巫（三十二卷71–72章·强者争霸战第五轮）

第五轮极寒之地是阿姆的主场（雪里一藏就难被察觉）。围猎夜惑女巫 [桑德拉](强者争霸战群像.md) 时它当面坦——只练一招「**举锤，砸**」（练十次不够就百万次·一锤刁钻砸天灵盖），完全放弃防御硬抗桑德拉的连刺，靠三重防御 + 冰焰自愈 + [布布汪](布布汪.md) 冰雪光环死缠；危急时吞苏晓配的「远古秘药」（可食用植物胶脂壳·非玻璃）瞬间回血到 98.5%，把桑德拉的夜魔皮一次次逼到耗尽。桑德拉气到骂「无耻」——「阿姆若那么容易死，也当不了苏晓小队的坦」。
