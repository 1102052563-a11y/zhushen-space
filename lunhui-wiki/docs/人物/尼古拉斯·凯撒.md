# 尼古拉斯·凯撒

[维斯星球·多亚古大陆](../世界/任务世界/维斯星球-多亚古大陆.md) 龙腾帝国铁血军团的**军需官**（[卡洛斯](卡洛斯.md) 亲认）——**1/3 人类 + 1/3 地精 + 1/3 矮人的三族混血**（浅绿皮、尖耳尖鼻、一米来高·见鬼的身世·自嘲父母多半是 3P）。专研**炼金系炸弹学**十五年（多为偷师所得·两年毫无进展）。性格 = **吝啬、见利忘义、又贪婪又胆小、还有点自私**（缺点极多、弱点却少）——苏晓评其「龙腾帝国有你这种『公正无私』的军需官、不灭国都对不起你」。极爱财、极护他那些试管财产；吃自创壮阳「格哈姆浓汤」（野牛大肠 + 猪膀胱 + 巨魔命根子 + 季蓝花根茎·苏晓闻之欲呕）。

## 与苏晓「炼金论交」（十二卷14–17章）

苏晓在战功兑换大帐里一眼看穿他调的液体炸弹「与空气摩擦会爆燃」、点出其「材料处理无可挑剔、配比技术稀烂」——这名两年没有同行交流的混血炼金师如获至宝、热情大变。苏晓只露一手、并提出唯一条件「**别外传我懂炼金术**」（免招麻烦），凯撒一拍即合。**好感度系统首次出现**（见下）：苏晓讲解半小时把他好感度从 31 顶到 **95**，却没触发「招奴仆 / 从者」（凯撒比想象中难对付）；为压价买「圣橡树枝干」，凯撒好感度还上演 −60→−30→+90 的「大起大落」（活脱脱见利忘义）。关键价值：**他能调控战功商店的物价**（轮回乐园认证之物·连圣杯监督者老神父都做不到的事）——他对苏晓挤眉弄眼、把官价 16000 战功的圣橡树枝干给苏晓开「**9000 战功**」的内部价（背后墙上还潦草写着「公正无私、保卫帝国」）。

## 关系

- **[苏晓](苏晓.md)**：「炼金论交」的对象（好感度 95·替苏晓压价战功商店物品）。
