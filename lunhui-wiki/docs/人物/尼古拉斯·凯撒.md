# 尼古拉斯·凯撒

[维斯星球·多亚古大陆](../世界/任务世界/维斯星球-多亚古大陆.md) 龙腾帝国铁血军团的**军需官**（[卡洛斯](卡洛斯.md) 亲认）——**1/3 人类 + 1/3 地精 + 1/3 矮人的三族混血**（浅绿皮、尖耳尖鼻、一米来高·见鬼的身世·自嘲父母多半是 3P）。专研**炼金系炸弹学**十五年（多为偷师所得·两年毫无进展）。性格 = **吝啬、见利忘义、又贪婪又胆小、还有点自私**（缺点极多、弱点却少）——苏晓评其「龙腾帝国有你这种『公正无私』的军需官、不灭国都对不起你」。极爱财、极护他那些试管财产；吃自创壮阳「格哈姆浓汤」（野牛大肠 + 猪膀胱 + 巨魔命根子 + 季蓝花根茎·苏晓闻之欲呕）。

## 与苏晓「炼金论交」（十二卷14–17章）

苏晓在战功兑换大帐里一眼看穿他调的液体炸弹「与空气摩擦会爆燃」、点出其「材料处理无可挑剔、配比技术稀烂」——这名两年没有同行交流的混血炼金师如获至宝、热情大变。苏晓只露一手、并提出唯一条件「**别外传我懂炼金术**」（免招麻烦），凯撒一拍即合。**好感度系统首次出现**（见下）：苏晓讲解半小时把他好感度从 31 顶到 **95**，却没触发「招奴仆 / 从者」（凯撒比想象中难对付）；为压价买「圣橡树枝干」，凯撒好感度还上演 −60→−30→+90 的「大起大落」（活脱脱见利忘义）。关键价值：**他能调控战功商店的物价**（轮回乐园认证之物·连圣杯监督者老神父都做不到的事）——他对苏晓挤眉弄眼、把官价 16000 战功的圣橡树枝干给苏晓开「**9000 战功**」的内部价（背后墙上还潦草写着「公正无私、保卫帝国」）。

## 苏晓 VS 凯撒 · 圣橡树枝干（十二卷42–43章）

战后四个冒险团的团长（金发御姐娜莎、血狼团长顾小笙、大型团火系法爷等）都来抢「圣橡树枝干」、堵在他门口——他却**唯独给苏晓开后门**（一拳砸门、他认出是「亲爱的朋友」便放苏晓进、把其余团长关门外）。他油盐不进、连火系法爷塞的蓝装、魅力美女的攻势都不吃。逼苏晓喝雷人的「**格哈姆壮阳汤**」（拒则好感清零 + 失战功商店权·苏晓靠扯炼金话题化解），又被苏晓拿「炼金公式只讲一半」反钓得抓心挠肝、只好放苏晓抢在众团长前**以 9000 内部价兑走圣橡树枝干**（火系法爷大气放手、提日后合作；顾小笙不甘却怂）。他还藏着一管卡洛斯的「血」（受安娜逼迫研究救命之法·见 [卡洛斯页](卡洛斯.md)）。

## 「凯撒的危机」· 关门放布布（十二卷48–49章）

[安娜](安娜.md) 在他身上留了追踪烙印、逼他两个月内想出救 [卡洛斯](卡洛斯.md) 的办法（否则分尸）——走投无路的他，见苏晓炼金术远胜自己，便冒着杀头风险把卡洛斯的身世（体内未知能量·见 [卡洛斯页](卡洛斯.md)）和盘托出、求其相助。这也触发了苏晓的**特殊任务「凯撒的危机」**（Lv.23·以凯撒名义清除卡洛斯体内能量·3 天）。他还想「空手套白狼」——任务奖励只写「凯撒的感谢」（以其人品多半是破碗剩汤），被苏晓识破。

**布布汪天克凯撒**：这回苏晓带 [布布汪](布布汪.md) 来镇场——布布汪不在乎好感度、不需兑物，看凯撒不顺眼**上去就咬**（屁股 / 小腿），把这吝啬鬼咬得没脾气（武力打不过、又不敢翻脸怕学不到炼金术），硬生生被这一人一狗「拉低智商」。被布布汪咬到位后，凯撒只得掏出压箱底的暗紫胸针 **[猎魔之证明（猎魔套装 2/3）](../力量体系/装备道具.md)**（他不识货·只当魔法饰品）当报酬——苏晓佯装挑剔压价、又索要「大队长」官职作烟雾弹，最终任务奖励敲定为**猎魔之证明 + 大队长**（布布汪暗叹苏晓「好演技」）。

## 🆕 画中世界 · 苟死在 7 号房间的贪财鬼（44卷28–30章）

他也进了 [画中世界](../世界/任务世界/画中世界.md) 争夺战，却把「贪财 + 胆小 + 苟」发挥到极致——**死苟在古堡二层的 7 号房间不敢出来、连话都不敢说**（主画 / 里画世界比想象中危险，连他都虚了），只从门缝递日历纸、写字条向 [苏晓](苏晓.md) 求合作。他贪财到:苏晓弹烧烫的金币试他，他都死死攥着滚烫金币不撒手（一股烤蛋白味）。他把屋顶唾手可得的**太阳教会套装**线索让给苏晓、还罕见地开出 4001 灵魂钱币报酬（认识他这么久、一毛不拔的凯撒竟肯出灵魂钱币）——原来他没用苟命能力逃走，是**舍不得 7 号房间里某件让他垂涎的宝物**（还派蹩脚的鼻涕虫炼金生物钻紫黑半流体去屋顶取钥匙）。他透露 7 号房间「不能没人在」，想找人念「开门让我进去」来替他。苏晓找 [伍德](画中世界/沃波·伍德.md) / [罪亚斯](画中世界/罪亚斯.md) 接盘，俩人齐齐装「拉稀」推掉。

## 🆕 沙之世界 · 太阳教会后勤管理员 · 四大阵营特权（44卷44–47章）

苏晓刚加入 [太阳教会](../世界/任务世界/画中世界.md) 就在大教堂后勤补给处撞见他——他早弄到了 7 号房间的宝物、脱身跑来沙之世界捞好处，如今是教会的**后勤管理员**。他给苏晓刷出一串离谱到好感度溢出、判定错误、身份被收回又用「战争勋章」重晋裁决者的骚操作，最终把两人好感度定格 **Lv.MAX**，白送苏晓四种阵营特权（离世界即失效）:**成本价采购**（声望消耗 −25%）、**退货**（按标价而非优惠价退声望、还能用蹩脚「翻新」把用旧的装备退掉·每日限两次·有被察觉风险）、**奸商**（无视兑换的声望等级前置）、**内部采购**（能用声望买藏库非卖品:画卷残片 / 钢铁盒 / 太阳焰爆燃技术）。他仍是那个吝啬鬼:难得慷慨分苏晓一小碗回老家龙腾帝国取材复刻的**格哈姆汤**（野牛大肠 + 猪膀胱 + 巨魔命根子发酵·壮阳），苏晓婉拒。7 号房间宝物独吞一事、他弄出四特权（耗稀有的战争勋章）就算「切好蛋糕只过眼瘾、连奶油都不偷吃」的赔礼，苏晓也就此翻篇。作为**真·裁决者**（对比虚假裁决者呆毛王），他专深入战区、与己方勾结、潜入敌营挑内讧、里应外合拿胜利。

## 🆕 太阳药剂合伙人 · 联盟星被苏晓炮击的往事（44卷55–62章）

沙之世界里他与苏晓（化名「库库林药师」）合伙做**太阳药剂生意**:苏晓出主材料 [阳光颗粒](../世界/任务世界/画中世界.md) + 核心炼金技术占七成、他负责副材料（赊账）/ 售卖 / 摆平上层关系占三成。他那套「拉一名客人免 100 金币」的传销促销、把 2000 金币的药剂卖到 3000 金币的本事，尽显奸商本色。他还给苏晓刷「成本价采购 + 退货」黑声望、通报抓到月使徒的消息（把「月使徒」念成「月吃兔 / 冷吃兔」）。闲聊中他哭诉上个世界的惨事:他去了 [联盟星](../世界/任务世界/联盟星-死亡圣杯.md) 西大陆当军需官、本想卷走泰亚图文明的全部财富，却在得手前两天遭到**持续八个多小时的炮击**、超凡物品全炸成渣——他至今不知那正是当时西大陆最高指挥官 [苏晓](苏晓.md) 下的令（[巴哈](剧毒魔鹰·巴哈.md) 睁眼说瞎话糊弄他「我们上个世界去的暗星」）。他还私藏一块蛊惑力极强、疑似世界级的**墨块**（连八阶契约者拿到半小时内都忍不住吃下、他没吃纯是「舍不得、吃了心疼」）。

## 🆕 「奸商」的隐藏特性 · 代苏晓密会烈阳君主（44卷63–66章）

割 [月使徒](苍龙大陆/月使徒.md) 韭菜时，他把「中立 NPC 裁决者」的身份用到极致——临时让月使徒入教会、转让特权、再靠**奸商的隐藏特性（仅他可激活:物品归属模糊时获所有权）+ 独有能力「厚颜无耻（Lv.MAX）」+ 最终解释权**，把月使徒兑的**凝固的太阳血晶·超大块**判归苏晓，全程「三方公证的正常买卖」，苏晓作为第三方毫无把柄。太阳药剂售罄后（他一套传销促销天亮前就全卖光、印证那三成不冤），他收下苏晓给的一瓶**太阳圣药（完美）**当报酬（也当去背刺的诱饵），**代苏晓去王朝遗址圣丹城密会 [烈阳君主](画中世界/烈阳君主.md)**——以「合作争夺野兽心」为由送药、埋下切葛细胞 + 聂氧的暗手。他离岗期间，后勤军需官一职由 [布布汪](布布汪.md) 顶替。（他与 [布布汪](布布汪.md) / [巴哈](剧毒魔鹰·巴哈.md) 斗地主老出老千、把 54 张牌玩到 179 张还抓出奥特曼牌。）

## 🆕 阵营恶霸 · 打劫太阳教会（44卷75–77章）

弑君夜他全程接应:诱心腹 [孤骸·兰斯洛](画中世界/孤骸·兰斯洛.md) 上钩、开传送阵锁盘接苏晓入圣丹城、从 [布布汪](布布汪.md) 摸清的藏处偷回 7 块画卷残片。回大教堂后他重掌军需官、再演一遍「太阳血晶」骚操作（这次拉 [莫雷](原始大陆/莫雷.md) 当兑换人），让苏晓一口气兑走 21 块画卷残片 + 钢铁盒 + 爆燃纹印 + 492 太阳石。压轴是他的独有技能**阵营恶霸（主动·Lv.EX）**:双手拍柜台、地精语口吐芬芳、瞪出血丝，硬**篡改兑换列表与物价**——把不在声望商店卖的 500 颗完整灵魂结晶按 2319 声望 / 颗塞给苏晓，等于当场**打劫了太阳教会**。（他与苏晓约定这单干完请他吃「大餐」，还提前躲进沙漠风美女堆里享受。）

## 🆕 阵营恶霸的原理 · 上古秘药 · 被光焰领主一锤送飞（44卷78–80章）

他的**阵营恶霸**并非硬抢，而是「做假账」——把 500 颗完整灵魂结晶的「存在」与等价的余温太阳石掉包（30 太阳石 = 3 万声望 = 一颗灵魂结晶），再压价到 2319 声望 / 颗（价压得越低、能力持续越短·还怕吵醒地下宝库的 [太阳鸟·泰哈卡克](../世界/生物万族/index.md)），够苏晓 5 分钟兑完。事成他二选一没要灵魂结晶分成、而选了**上古秘药配方**（错过不再有）。逃脱后他现身于王宫上空——竟搂着 [光焰领主](../世界/生物万族/index.md) 那把暗金长柄钉锤问「需要武器保养吗」，被一锤震飞、连穿十几层墙壁（贪财到最后一刻还想从大 boss 身上薅点)。

## 🆕 天价赔偿账单 · 举报之王（44卷86章）

莫雷、月使徒弄丢借来的太阳头桶（本就带不出沙之世界），他化身「财政管理」上门讨赔偿:掏出一台会**模糊人对物品价值观**的 pos 机式不朽装备，从「太阳头桶绝版 + 四件套贬值 + 精神损失费 + 理智值间接损失」层层叠加、用「2 进 14.59 出」的鬼算法把账单滚到近 8 米长、**12 万灵魂钱币**，还要给她们上「地精信贷·十五期高利贷」。莫雷惊醒砍到 3000 成交，他又以中介人身份抽 3.76% 手续费、触发独有能力**抹零**多薅 7 枚。这套骚操作让他与苏晓双双被 [月使徒](苍龙大陆/月使徒.md) 举报——1953 名天启契约者骂他「诈骗 / 假冒军需官 / 用袜子丢我」，苏晓也被 1729 人举报（月使徒因此得「举报小能手」称号）。

## 🆕 主城「药剂大师」· 抢占商标（44卷111–114章）

他早苏晓三天就摸进海之底主城神恩城、伪装成**药剂大师**（其实是拿苏晓在太阳教会剩的药剂糊弄过关）——料定苏晓必来主城（海神握大量画卷残片），提前**抢占「药剂师」这个不可替代的位子主动入伙**（明面上的人脉 / 渠道 / 售卖都由他扛、既求好处又替苏晓挡暴露风险·仍是三七分）。他这三天布下大量「钉子」:靠把柄要挟买通巡夜队长 [伯纳](画中世界/波罗司神使.md) 之流，好让违反宵禁成常态。他给苏晓拉来治兽化症的**第一个大客户**——竟是 [光焰领主·驴哥（奥斯·古因）](../世界/生物万族/index.md)（委托人是想弑父的大神子奥斯·康拉德）。贪财归贪财，办事却稳（全靠怕死）。

## 🆕 扮丑老仆送毒念髓 · 中途被传送出局（44卷117–124章）

弑海神一役他扮成面色惨白的丑老仆（脸抹白墙灰·「精神污染级」丑到海神只想快点打发他）给海神送掺了能量毒素的**念髓**——那毒素待海神催动身体能量时激活、强行沉默其能量，逼海神只能近战肉搏，是暗杀成功的关键一环（康拉德为此以 10 颗灵魂结晶请他出山）。此外他把违反宵禁常态化（塞浓缩芥末冒充「回春大力丸」贿买巡夜队长伯纳）。可惜他**因做了某件违规事、被强行剥夺裁决者身份传送出局**——否则以他讹诈的本事，被算计的 [乌鸦女](画中世界/乌鸦女.md) 有五成以上概率不仅拿不到黑枫、还得倒贴钱。

## 🆕 返园强化 · 随机抽取权限骚操作 · 裁决者三贱客（44卷134–139章）

画之世界通关返园后，他又来 [苏晓](苏晓.md) 这薅羊毛——盯上苏晓要出售的 [奥术永恒星黑枫树枝干](../力量体系/装备道具.md)（4370 克），死缠着「凯撒希望这笔交易继续下去」，最终以 370 克黑枫为报酬替苏晓运作。他自曝去奥术永恒星「只是想参观黑枫树」，却因尿急掀开石台金属封盖、往人家取水的地下流井里撒了泡尿，被那帮施法者「吊起来打」（还哭诉说得声音哽咽）。他与 [阿尔托利亚（呆毛王）](阿尔托利亚.md)、癞蛤蟆并称「**裁决者三贱客**」（当年一场世界争夺战靠假情报把天启 / 圣光 / 守望三乐园同盟搅得土崩瓦解）。

- **随机抽取权限**：苏晓的这项权限凯撒自己信用度太低开不了，便拉信用度 227 的呆毛王入伙做「三方交易」。他故意**用呆毛王的信用发起重复公证、把「随机抽取权限」搞成虚空之树与轮回乐园都不认的半公证状态**（自食恶果 + 激活恶霸 Lv.EX），趁判定混乱手动把大转盘指针硬推到「铭文基座」区——苏晓由此得**铭文基座（3 铭文槽）**（第 12 次被天启乐园记录为入侵者 / 资源强盗）。
