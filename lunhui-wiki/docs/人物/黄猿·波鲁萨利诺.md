# 黄猿·波鲁萨利诺（海军大将）

海军三大将之一「**黄猿**」——蛤蟆镜 + 白色「正义」大衣 + 屎黄衣物，身高至少 3 米（现役大将无低于 3 米者），语调懒散、阴阳怪气（不了解的人会以为他在嘲讽·这只是他平常的腔调）。是「**最佛性的大将、没有之一**」，与 [苏晓](苏晓.md) 私交不错（曾在实验岛上一同钓鱼），但办起公事绝不放水。

- **闪光果实（自然系）**：直线移动近乎**光速**（光速移动时必保持元素化·那种速度连凯多的肉体强度都未必扛得住）；能凝光子成长剑 **天丛云剑**（剑锋高速震动的光粒子削铁如泥）、「光速踢」等。

## 率队围捕苏晓 → 押往司法岛（十八卷51–55章）

世界政府派出 **卡普 + 鹤中将 + 黄猿 + 五艘军舰 + 千余海军**（围捕四皇级的阵容）来德雷斯罗萨抓苏晓——黄猿在港口压住 [多弗朗明哥](多弗朗明哥.md)（暗示「下令的人你最好别知道」=远高于五老星），笑称「白夜小哥应该很想念老夫」。

- 苏晓杀出别墅、与三人在庭院对峙：苏晓掏阿波罗威慑（黄猿可元素化躲过、卡普 / 鹤只能肉身硬抗故退 300 米），但**黄猿的光速让苏晓难逃**。苏晓遂**假降**（实为海上脱身的计谋）——黄猿一记光速踢、海军给他铐上海楼石镣具。
- 黄猿一直惦记苏晓那条「很聪明的狗」[布布汪](布布汪.md)（化光上天找它、没发现融入环境的它·布布汪在苏晓身旁腹诽「死猴子」）。最终海军把苏晓押上五艘暗绿军舰、先送往司法岛（详见 [势力 · 海军](../势力/index.md)）。

## 关系
- **卡普 / 鹤中将**：同行围捕苏晓的海军前辈（「夕阳红组合」·鹤 74 / 卡普 78 / 黄猿 56）。
- **[苏晓](苏晓.md)（白夜）**：实验岛钓鱼旧识、如今的围捕对象（私交归私交、公事不放水）。
- **[Dr.贝加庞克](index.md)**：海军体系内的天才博士（苏晓的旧搭档·囚禁苏晓的外装甲即其手笔）。
