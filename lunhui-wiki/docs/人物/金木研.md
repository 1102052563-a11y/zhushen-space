# 金木研

**金木研** 是 [东京喰种](../世界/任务世界/东京喰种.md) 世界名义上的**剧情主角**（漫画原著主角），但**并非真正的「世界之子」**——蜕变前不可被杀、蜕变成喰种后死活已无关紧要（二卷1、34、64章）。

## 简介
原为普通大学生，因移植 [神代利世](神代利世.md) 的器官成为**半喰种**。其蜕变正被喰种阵营契约者刻意推动——为延续剧情、增强己方力量，他们配合 [雾岛绚都](index.md) 把金木研抓到 [青铜树](../势力/index.md) 11 区据点折磨（二卷38章）。

- 原著中折磨者为 [壁虎](壁虎.md)，因壁虎已被苏晓斩杀，改由小丑成员**妮可（尼克）**施虐（手法雷同，反复逼问「一千减七」）（二卷38章）。
- 在「调教」下完成**蜕变**（白发、单眼面具、四条赫子如神代利世，达 S+ 级），由雾岛董香、西尾锦护送逃出青铜树（二卷62章）。

## 天台之死（二卷62–64章）
逃亡途中撞见重伤的苏晓挡路，态度有恃无恐，遭苏晓暴起斩击。金木研近战极弱（断臂可恢复、却挡不住刀），眼睁睁看苏晓两刀斩杀西尾锦后精神更不稳（反复念「一千减七」）。苦战中其腿被打断、瘫于水箱下，[雾岛绚都](index.md) 赶来救走姐姐董香、却被苏晓求援吓退而弃他。最终金木研被苏晓**斩首**（二卷64章）。

- 乐园判定：杀死「剧情主角金木研（成长度 20%）」→ 世界之源 +10%、得**蓝色宝箱**、激活成就 [掠天惊澜](../力量体系/成就.md)。
- 其死触发**命运之力重新分配**：新任剧情主角变为 雾岛董香（1/2）与 [神代利世](神代利世.md)（1/2）（见 [机制](../总览/轮回乐园机制.md)）。

## 关系
- [神代利世](神代利世.md)——其器官供体（金木研因此成半喰种）。
- 妮可（尼克·[小丑](../势力/index.md) 成员）——折磨他、推动蜕变者。
- [雾岛绚都](index.md) 与喰种阵营契约者（[寒鱼](index.md) / [火爆](index.md) 等）——抓捕、推动其蜕变。
- [苏晓](苏晓.md)（白夜）——同世界、暂无交集（乐园曾警告远离命运之子）。
