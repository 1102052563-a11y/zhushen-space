# 阿德莱德·赫伯特（铁冠王 · 圣索丁开国皇帝）

千年前的人类，**圣索丁王国的初代国王、开国皇帝**，俗称「**铁冠王**」——赫伯特王朝（至今传到 [马蒂尔·赫伯特](马蒂尔·赫伯特.md) 三王子一代）的始祖。

## 投机者的开国（十七卷82章·壁画揭示）

在 [阿克斯](阿克斯.md) 断绝远古死灵族的死灵法术、覆灭希亚王朝后，阿德莱德·赫伯特冒着生命危险跟随了阿克斯几天，随后重返人类世界、**自称「阿克斯派下的使者」**——靠这名头弄来大笔财力、招兵买马，短短几年聚拢起一大股穷苦人，**推翻五大王国的暴政、建立圣索丁王国**。一个精明的投机者。

- **王都与王宫的选址**：他把王都定在希亚王朝的原址、**把王宫修在「希亚圣殿」之上**（即镇压远古死灵族的不死者陵墓）——被囚于殿中、已被夺走死灵法术、残破到「不能生也不能死」的远古死灵族对此极为愤怒。后世人类自发镇守希亚圣殿，本质是「**拍马屁**」（向开国传统与灭法者示好）。
- **铁王冠**：他传下的王冠「铁王冠」镶一颗红宝石——红宝石内竟藏着灭法之影的「**法之石**」（[苏晓](苏晓.md) 十七卷70章从三王子手中讨走、捏碎而得，见 [装备道具](../力量体系/装备道具/index.md)）。
- 「死灵族」这个称呼本质是他**沿用了远古死灵族的旧称**（现死灵族其实与远古死灵族已没太多关系·见 [生物万族](../世界/生物万族/index.md)）。

## 关系
- **[阿克斯](阿克斯.md)**：他自称是其「使者」、靠这身份开国（阿克斯本人并不在乎谁统治希亚世界）。
- **[马蒂尔·赫伯特](马蒂尔·赫伯特.md) 等赫伯特王族**：其千年后的后裔（圣索丁王国历代君主）。
- **远古死灵族 / 希亚圣殿**：他把王宫修在镇压它们的陵墓上、令其愤恨（见 [希亚大陆·灭法遗迹](../世界/任务世界/希亚大陆-灭法遗迹.md)）。
