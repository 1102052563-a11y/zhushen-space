# 朽木白哉（六番队队长 · 大贵族）

[死神世界](../世界/任务世界/死神-尸魂界.md) 护廷十三队**六番队队长**——大贵族出身。六番队名义是「贵族专属队」，实则**掌控瀞灵廷的物资 / 财力**（各项支出由其把控），是十三队中最特殊的一队（连阴鸷的涅茧利都不敢惹他——惹了他、十二番队那常年亏损的科研团队就要揭不开锅）。他平常不苟言笑、**外冷内热**。

- **义妹露琪亚**：朽木露琪亚因「赋予普通人（黑崎一护）死神之力」被囚**忏罪宫**、等待**中央四十六室**裁定。碍于白哉与朽木家的关系本不至判死刑（最多关押百年），但 [苏晓](苏晓.md) 知道——**整件事是 [蓝染](蓝染惣右介.md) 安排的、露琪亚最终会被判处死刑**（黑崎一护正随神秘店长特训、将为救她攻入尸魂界）。白哉这几天都在暗地关注判决。
- **声望商店**：苏晓加入四番队后激活的**瀞灵廷声望**，正是在白哉六番队的后勤部门（声望商店·绝对中立点）兑换物品。

## 关系
- **[苏晓](苏晓.md)（白夜）**：六番队声望商店的所属队长（苏晓评其为打不过的强队长之一）。
- **朽木露琪亚**：义妹（被囚忏罪宫·将因蓝染的布局被判死刑）。
