# 红色小丑

[全职猎人世界](../世界/任务世界/全职猎人.md) **猎杀目标「[绅士同盟](../势力/index.md)」的老大**——紫色西装、笑容温和却**情绪极不稳定 / 神经质**（兴奋时会拿小刀刺自己拇指）。掌握**空间系**能力（可空间传送、投影分身）。手下成员有 [枯尸](枯尸.md)（殁）、艳妖等（皆用代号、入伙弃原名）。

- **以投影现身（十卷5章）**：枯尸被苏晓追杀求救，他用「传送机会」赶来——却只放出**投影**（本体在一百多公里外、脚下踩着一具被他斩首的契约者）。他识破苏晓是**「追猎者」**（专猎违规者这类「不合理契约者」），还透露**曾尝试几百次想对外泄露苏晓的身份都失败**（猎杀者身份受乐园保护）。
- **「游戏」约战**：他向苏晓提议一场 Game——**他带部下追杀苏晓，苏晓若反杀他、他就告知「一个关于违规者的秘密」；苏晓若败、临死要吐露「关于追猎者的秘密」**。约「友克鑫市见」。他**没敢让本体进入自己开的空间裂缝**（否则苏晓能用 [岚凯风衣](../力量体系/装备道具.md) 的初级空间禁锢把裂缝封死、将其困死异空间）——可见其谨慎，「真正麻烦的家伙还没出现」。

## 炮台伏击 · 恶魔血统 · 被斩残（十卷8–11章）
他掌握苏晓行踪、**炸火车设伏**（吸食死者灵魂作养料·这是其能力），并花 6 万 + 12 万乐园币请旅团 [薇妮](薇妮.md) 当「炮台」远程压制苏晓，加上部下 [犀](犀.md)（被他续力连撞）、[艳妖](艳妖.md)（隐身偷袭）四打一。但苏晓界断线 + 次级无敌护盾 + 连斩**反杀了犀和艳妖**。

- **激活恶魔血统 → 地狱小丑**：他随即**祭献犀 / 艳妖的灵魂**、激活苦求十个衍生世界、祭献无数灵魂换来的**恶魔血统**（轮回乐园**五大最强生存力血统**之一·极罕见·副作用 = 被祭献的灵魂都有一次抢他身体的机会，故其神经质 / 多重人格）——改名「**地狱小丑**」，使「地狱焰」（暗红火球·烧地成岩浆·克青钢影）+ 双短刀近战（力 / 敏不在苏晓之下）+ 短距瞬移（1~10 分钟一次）。
- **被斩两臂一腿 · 攻守逆转**：苏晓注射 1 号药剂回血、以岚凯风衣「初级空间禁锢」封其瞬移、缠斗近一小时——小丑**体力不支**，被苏晓「牛头领主之怒」（力 58→73）一刀斩断臂腿、压制斩首。但乐园提示死的是「**所罗门·马奇·低等深渊恶魔**」——这只是小丑用恶魔血统弄的**替死鬼**，他本人没真死（恶魔血统不会这么轻易死）。**猎杀仍停在 4/5**；不过他被斩去两臂一腿、在猎人世界难恢复，攻守逆转——轮到苏晓追杀他了。
