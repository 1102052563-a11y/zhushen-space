# 吉克·耶格尔

**吉克·耶格尔** 是 [进击的巨人](../世界/任务世界/进击的巨人.md) 世界**野兽巨人（兽之巨人）**的传承者、[马莱帝国](../势力/index.md) 的战士；也是剧情主角 [艾伦](艾伦.md) 同父异母的**哥哥**（三卷38、44章）。

## 简介（截至三卷45章 · 多为提及）
- 是 [马莱间谍三人组](莱纳·布朗.md) 的同伴、留在**墙外**接应，截至本批苏晓一行久未与他联络（三卷38章）。
- **身世**：艾伦之父 [格里沙·耶格尔](格里沙·耶格尔.md) 与艾尔迪亚王族 **黛娜·弗里茨** 之子（故吉克亦有王族血脉）。格里沙以「艾尔迪亚复兴会」思想自幼教育吉克，并在巨人之力宿主换届时（吉克 7 岁）送他去参加选拔；但**吉克转而向马莱出卖了亲生父母**，导致格里沙夫妇被马莱清算（黛娜被变成无垢巨人流放、格里沙被「枭」所救）。这段背叛是格里沙日记里的痛悔之笔（三卷44章）。

!!! note "尚未正面登场"
    截至三卷45章，吉克只在对话与格里沙日记中被提及，本人未正面出场；其「野兽巨人」之力与具体行动留待后续章节补充。
