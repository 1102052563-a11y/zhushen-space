# 女装大佬

来抢贤者之石的**契约者**——男扮女装（时崎狂三装扮）、智力 40+ 的**念能力者**（念波动·墙 / 坚盾 / 冲击·能与苏晓正面对抗、但死斗必败）。持一枚暗紫色「**迷惑**」银质耳坠（曾短暂控住钢炼最强生物 [黑猩猩](../世界/生物万族/index.md)、却因不敢反抗反被擒）。

- **历程**：七卷22章在第三研究所的贤者之石实验室被苏晓逼交 4 块贤者之石；黑猩猩出笼后，因魅力高（15 点）被黑猩猩「看着顺眼」抓在手里逗弄，后被盛怒的黑猩猩攥成重伤抛弃（七卷22–24章）。
