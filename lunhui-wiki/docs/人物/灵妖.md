# 灵妖

⚠ **一名皮肤白皙且透出微弱荧光，蓝色短发缓缓洒落光粒的少女**——⚠ **法魂队的新成员**;⚠ **有资格加入法魂队，自然是很有手段与实力。**

## 🎲 那场抽签

⚠ **法魂队的任务在猩红城堡莫名终止**，⚠ **[艾格](风海大陆/雷法神·艾格.md)、[莉莉亚](风海大陆/女巫·莉莉亚.md)、[凯因](风海大陆/噩鬼·凯因.md) 三人都感觉特别迷茫**:⚠ **什么「城主夫人寝殿」，那是哪？现阶段的任务地点不是在黑铁城吗？**

- ⚠ **是她先站起来的**:「**欺人太甚！斩首的夜就可以抢别人的任务吗？你们就这样忍了？**」——⚠ **凯因立即赞同**:「**灵妖说的对，这事不能就这样算了。**」
- ⚠ **莉莉亚接力**:「**嗯，那这事谁去解决？总不能我们四个人一起去吧？四个打一个，传出去不好听。**」——⚠ 灵妖「**我听说～，斩首的夜挺强……**」⚠ **不等她把话说完，凯因就说道**:「**以我们四个的实力和名气，的确不应该四打一。**」
- ⚠ **收网**:「**不如这样，我们之中选出个代表，去找库库林·白夜要个说法**」——⚠ **莉莉亚当即赞同，还良心未泯的艾格犹豫了下，碍于两名队友都同意，也只能顺势同意。**⚠ 灵妖「**其实偶尔退让一下……**」（⚠ **她已经开始感觉情况不对，荧蓝色短发都不再飘散下光粒**）——⚠ 凯因:「**不行，你刚才说的对，灵妖，那家伙欺人太甚，我们四个中，随便抽签出一个，去收拾他，来，抽签吧。**」
- ⚠ **她赶紧先抽一个**（⚠ **她真怕这三个队友默契的抽走三个，给她留下必中签**）——⚠ **打开抽签结果，面色一僵，但她极好的掩饰下来**:「**你们的结果是？**」⚠ 「不是我。」「也不是我，可惜～」「我看看，嗯，不是我。」——⚠ **三人都看向灵妖，这让灵妖眼角狠狠抽动了下。**
- ⚠ **老阴哔的激将法**:「**灵妖，你不会是……怕了吧，如果怕了，其实算了也行。**」——⚠ 「**你……你放|屁。**」⚠ **话刚出口，立即后悔了**;⚠ **作为光灵族、机缘巧合成为契约者的她，显然中了激将法。**（⚠ **如果把各族的潜力属性数据化，人族就是初始所有数值都低，但成长性高、上限高**;⚠ **而像光灵族，则属于初始的天生战力就在 B～A 级的程度，与之相对，智谋方面的成长，就不像人族这么高**）
- ⚠ **她拿到的「补偿」**:「**属于我的那份任务奖励，我一定要回来，至于你们的那份……**」——⚠ 「**都归你了。**」（⚠ **凯因当即替其他两人拍板决定**）⚠ 「**一言为定。**」⚠ **灵妖脸上展现笑颜，几个光荧闪烁，就到了外面的庭院内。**
- ⚠ **三人在落地窗前的对话**:艾格「**我们这么做，是不是有点损？**」——⚠ 凯因「**呵，艾格，你认为这小丫头是和谁组队一起进永光世界的？**」⚠ 「谁？」——⚠ 「**神父。**」⚠ 「**凯因，你这事干的漂亮。**」（⚠ **雷法神·艾格的心情当即好了几分**）

## ☠ 厄运之兆 · 7560 倍魅力判定

⚠ **她此行名义上是为猩红城堡任务讨说法，其实另有目的**:⚠ **在进入本世界前，她就与 [神父](魔灵星/神父.md) 合作**——⚠ **追踪苏晓，是为了在苏晓身上悄无声息地布设一个坐标**;⚠ **而施加厄运，则不是合作内容，是因为她被抢了任务，心中非常不爽。**

- ⚠ **她的拿手好戏**:⚠ **半透明的印记飞出，飞行速度偏慢却有追踪功能**——⚠ **随着印记逐渐靠近敌人，其特性缓慢变更，最终伪装到犹如对方的气息般**;⚠ **哪怕贴附到目标衣服上，目标也难以察觉。**
- ⚠ **本该到此为止**（⚠ **至少神父的计划是如此**）——⚠ **但眼下办完正事，到灵妖处理个人恩怨的时候了**:⚠ **黑色能量丝线从她手上脱离，这是她从周边世界抽离出的厄运**;⚠ **她坐在岩石破屋的屋顶，两条小腿前后晃动，确定距离 3 公里左右后，笑的都有些露出小白牙**:「**准备倒霉吧，敢抢我任务的混蛋。**」
- ⚠ **判定结果**:

- ⚠ **此刻灵妖还不知道，她「厄运之兆」能力的极限，远不止于此。**
- ⚠ **莉莉亚早就知道会有多好笑**:凯因问「**这光灵族的小丫头，到底是什么系能力？我感觉她不像是正面战斗的光系。**」——⚠ 「**当然不是，她是厄运系。**」⚠ 「**哦？这么说？能给白夜找些麻烦？**」——⚠ **莉莉亚笑而不语**;⚠ **她都无法想象，一名厄运系在暗处对灭法之影偷偷施加厄运诅咒，那得是多么搞笑的情景。**
- ⚠ **她在等待期间的意外收获**:⚠ **她在猩红城堡外等了半天**（⚠ **虽早就完成了 300 点的属性试炼，但还没晋升绝强，因此没进去**）——⚠ **目标却忽然消失，把她气的不轻**;⚠ **湿盐地上等待的口渴感让她心情更暴躁**——⚠ **结果在周边寻找时，遇到了一个名为「理智野兽」的家伙，还从对方那接到了个支线任务，完成后，小赚了笔。**

## ☠ 死亡与真相：她其实是白金使徒的宿主（51卷26章）

⚠ **她对苏晓施加的「厄运之兆」被 10 倍反弹后**，⚠ **苏晓那始终显示灰色、无法主动激活的能力「[血之恶意](../力量体系/技能.md)」忽然自行激活**——⚠ **锁定了她这个「猎物」。**

- ⚠ **一个照面就崩了**:⚠ **一道水桶粗的血气射线扫来，她被迫中止心灵传动**——⚠ **她还不惊反喜:与近战宗师保持几公里距离，还是有很大概率逃脱的**;⚠ **可斩龙闪一记斩击，就让她整个右半边身体像陶瓷般四处飞散**——⚠ 「**这就是刀术宗师到了后期不受待见的原因**——⚠ **交手后太绝望了，既控不住，还打不动**;⚠ **关键是轰这刀术宗师半天，刀术宗师不一定死，可被刀术宗师近身来几刀，那基本就没了。**」
- ⚠ **光灵族的双生**:⚠ **她瞳孔中的光泽快速淡去又恢复**——⚠ **这是光灵族的种族特点·双生:有两颗命魂，天生两条命。**
- ⚠ **「40 万！」**:⚠ **刀锋停在她脖颈旁，凛冽的风压吹起她的短发**——⚠ **她刚要继续开口，眼睛突然瞪大、瞳孔颤抖出现重影**;⚠ **随后目光变得从容，盘坐在地上**:「**抱歉，主魂沉睡了一段时间，睡着了总是会糊涂，还认为这是自己的真身，都忘记这是用了许久的光灵族躯壳……**」
