# 石头

[全职猎人世界](../世界/任务世界/全职猎人.md) 现身的契约者——身高两米的壮汉，**原 [蛮天冒险团](../势力/index.md) 的二号主坦**（二阶竞技场约 1591 名·体力 60+），持一面带火焰图案、缘流淡金光的**塔盾**。其拉仇恨手段 =「让敌人产生强烈痛处 + 精神诱导」（塔盾火焰几无攻击力却极痛），塔盾受击可激活火焰反震顶飞敌人。

- **孤家寡人**：[蛮天冒险团](../势力/index.md) **已在火影世界团灭**（[长门](长门.md) 所灭），他成了无依无靠的散人、颓废酗酒——以「流浪主坦」身份被 [A先生](A先生.md) 拉来打 [独眼巨怪](../世界/生物万族/index.md)（顶级主坦本是各团抢手货、落到小团体令众契约者惊讶）。
- **被算计却逃生（十卷15–17章）**：A 先生小团队的阴谋本要在巨怪濒死时让奶妈停奶、把他「倒坦」害死。但 [布布汪](布布汪.md) 的「冰雪女神光环」让他在断奶的瞬间**硬扛住独眼巨怪一棍、趁乱逃出战场**——既保住性命、也粉碎了阴谋（坦一跑，散人们有了准备、没被巨怪坑死）。
