# 冒牌小丑（轮回乐园·黑乌鸦）

[维斯星球·多亚古大陆](../世界/任务世界/维斯星球-多亚古大陆.md) 世界争夺战中轮回乐园方的「疯子」之一——代号「**黑乌鸦**」。一身紫色西装、嘴上画黑色口红、脸颊被利器豁开（笑时露出整排牙龈），刻意模仿小丑做派，言行癫狂、神经质，连智商极高的 [希女王](希女王.md) 一见都断定「这家伙精神绝对不正常」。

- **黑乌鸦潜入计划（十二卷76章）**：天启大军攻黄昏宫殿时，他主动站到门口自荐当「带各位走进地狱的向导」、神经质地拖延，最后**割喉自杀放血、血中伸出冤魂之手**唬住天启——实则这是障眼法：他真正的本事是**潜入**，割喉「假死」的同时，一名戴礼帽的他混进了天启队伍（头顶光点绿变红又复绿·「潜入成功」），为苏晓、炎辰传递敌情。苏晓「悉心劝导」说服了他去「迎接客人」。
- **撕传送阵 + 加入铭门（十二卷77、81章）**：他撕开 [炎辰](炎辰.md) 提供的金色「短距离大型传送阵」、把几十名契约者瞬移到四层堵口，是大菠萝计划的一环。战后他原冒险团已团灭，欣然加入炎辰重组的铭门（「跟着你混要是无趣我随时走」）——平日还爱拿镂空匕首对镜把脸上的豁口割得更长。

## 关系
- **[炎辰](炎辰.md)**：重组铭门冒险团的团长（招他入伙）。
- **[苏晓](苏晓.md)**：大菠萝计划的主导者（「悉心劝导」他去当诱饵 / 潜入者）。
