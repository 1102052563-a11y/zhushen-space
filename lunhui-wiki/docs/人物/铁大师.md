# 铁大师

一名**契约者**，更重要的身份是「**锻造大师**」——效力于 [血门冒险团](../势力/index.md) 的「职工者」。在轮回乐园内**锻造大师实力虽不强，却被任何冒险团奉为座上宾**。

- **此行目的**：因「一些特殊原因」需要**晋升阶位**，团长斯坦不惜代价为他争取到生存试炼资格、并派 [老黑](老黑.md) 等人一路护卫（斯坦下死命令：铁大师若出意外，老黑「可以提头来见」）。故装甲车里反由这位弱小老头发号施令，他甚至有些看不起老黑这等「泥腿子」。
- **识货的眼力**（八卷49–50章）：装甲车与苏晓一行擦肩时，他一眼瞥见苏晓腰间的 [斩龙闪](../力量体系/装备道具.md)，瞳孔骤缩、贪婪毕露——失声惊呼「**成长武器、那是成长武器！**」，下车就要抢。但女契约者探查到苏晓「三维问号、技能不明」后，他**当机立断放弃、催车逃离**（识时务）。

## 结局 · 死于堵桥（八卷53章）
逃离苏晓后，[老黑](老黑.md) 一行赶往**红土区唯一出口「石桥」**，却撞上一队契约者**堵桥三天**（专杀从红土区脱出、饿到虚脱的契约者）。缺粮虚弱的血门小队被打得抬不起头、躲在装甲车后挨枪——老黑等近战后倒在血泊中，铁大师**跪地向桥对面契约者求饶仍被一枪爆头、尸体被扔下万丈悬崖**。斯坦为他晋升付出的一切付诸东流。（讽刺的是，杀他们的堵桥队随即被远处赶到的苏晓反杀。）
