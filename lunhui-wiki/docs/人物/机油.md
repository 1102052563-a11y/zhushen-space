# 机油

[轮回乐园](../总览/轮回乐园机制.md) **机械街**的契约者——除头部外几乎全身改造成机械、爱喝机油（代号由此而来），[苏晓](苏晓.md) 在竞技场结识的旧识。战斗风格是「**疯狂炮火轰炸**」、一言不合就自爆（全身都是武器）；性格豪爽。

- **牵线 + 修平板（九卷90章）**：苏晓要用 [物品改造装置](../力量体系/消耗品材料.md) 升级蜘蛛女皇，需找掌握「机械改造 Lv.17+」的契约者——机油帮他引荐 [机械妹](机械妹.md)。他还顺手帮苏晓**免费修复被佐助雷电劈坏的平板电脑**（雅特森存储卡需另买、数据他帮恢复），对苏晓「把平板狠拆」的手法哭笑不得。
