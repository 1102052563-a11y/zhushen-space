# 旗木卡卡西

[火影忍者世界](../世界/任务世界/火影忍者.md) 木叶忍者——人称「**拷贝忍者**」，白发、蒙面（写轮眼）。

- **登场（九卷38章）**：在木叶火影办公室与 [漩涡鸣人](漩涡鸣人.md)、[春野樱](春野樱.md) 一同在场，静听 [纲手](纲手.md) 与 [自来也](自来也.md) 商议砂忍村求援、五代风影 [我爱罗](我爱罗.md) 被「晓」掳走一事。

- **追击山洞（九卷43–44章）**：率小队（[漩涡鸣人](漩涡鸣人.md) / [春野樱](春野樱.md)）追到河之国山洞，与凯小队合力破「五封结界」冲入；见 [我爱罗](我爱罗.md) 已死、拦住暴怒欲冲的鸣人「这是陷阱」。后追 [迪达拉](迪达拉.md)（带我爱罗尸体）出洞，途中被 [苏晓](苏晓.md) 一道**刀芒**逼停、侧身堪堪躲过——苏晓为防「绝」监视、不肯放敌人当面过去。

## 苦战苏晓 · 被夺左眼（九卷45–46章）
追击中他心急去追鸣人、主动与 [苏晓](苏晓.md) 近身求一击必杀，反陷苦战——这位以「战斗智商」著称的拷贝忍者几秒就推断出苏晓的强项（近战 / 超远程）与弱点（中距薄弱），却仍吃了大亏：

- **招式尽出**：千鸟 / **雷切**（S 级）/ **雷盾·雷传**（雷电锁链·险斩断苏晓·只擦腿即深可及骨）/ **雷盾·影分身之术**（独创·影分身被破化雷电麻痹敌人）/ **万花筒写轮眼·神威**（空间切割·初次实战需 3 秒构成）。
- **被苏晓的「傲歌」能量盾克制**：苏晓 150 法力能量盾硬扛雷切、撤盾令卡卡西失衡反被贯肩；300 法力能量盾扛住神威空间切割（盾被搅碎、苏晓胸口仍留螺旋伤）。最终被苏晓炼金炸弹 + 贯胸 + **影刺**（十几根尖刺·卡卡西侧身只被划破心脏动脉）重创、并被**生生挖走左眼（万花筒写轮眼）**——苏晓将之留作与 [宇智波带土](宇智波带土.md) 交易的筹码（这只眼本是**野原琳**从带土身上移植给卡卡西的·是琳存在的象征·故带土此前一直没夺·见 [宇智波带土](宇智波带土.md)）。其间苏晓刀上无意沾的**蝎之残毒**也令卡卡西眩晕。
- **被凯所救**：千钧一发之际 [迈克·凯](迈克·凯.md) 开七门「夜凤」一脚踹飞苏晓救下他。**伤势极重**（胸前两处贯穿 + 失血 + 体内影刺无法取出、消散时还噬灭其查克拉、阻碍 [春野樱](春野樱.md) 治疗）——小樱直言他「今后或许不能再当忍者」。

## 出处
