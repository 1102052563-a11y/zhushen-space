# 绝（黑绝 / 白绝）

[火影忍者世界](../世界/任务世界/火影忍者.md)「**晓**」成员——猪笼草外壳、黑白两半合为一体（黑绝 + 白绝），主司情报、监视与处理首领交代的脏活。**资历最老、藏得极深**，平时不露丝毫异常——实为整个忍界的幕后黑手（伏笔）。

- **职责**：[长门](长门.md)（天道）封印一尾时命「绝」用「范围最广那招」在外围监视追兵动向；他全程汇报砂忍 / 木叶追兵的人数与来路（如认出第二波带队的 [迈克·凯](迈克·凯.md)、感知到九尾人柱力 [漩涡鸣人](漩涡鸣人.md)）。
- **与苏晓的秘密交易（九卷44章）**：[苏晓](苏晓.md) 用采集的**一尾守鹤之血**（在液 / 气间不停转化的雾状血液）作报酬，私下买黑绝的情报——找出符合苏晓所求特征的人（397 名中 3 人符合·分布火之国 / 雾忍村 / 砂忍村）。苏晓排除已去过的雾忍 / 砂忍（猎杀功能未触发）→ 锁定**火之国边境天地桥附近**（疑大蛇丸的实验室·近期「二柱子」灭大蛇丸后会空门大开），即其猎杀目标 **10001 号违规者**（火影世界的二阶猎杀任务目标·尚未现身）的所在。黑绝叮嘱「**别说我来过·我们是秘密合作**」——苏晓也提防爱乱说的白绝走漏风声。

## 两个白绝本体 + 「猪队友」（十五卷·苏晓再入火影）

苏晓厘清了「白绝」其实有**两个本体、相似度 100% 却两种人格**：① **蠢白绝**（现阶段露面的孢子分身之源·童心未泯、爱嬉皮笑脸、常不知不觉做蠢事·智商不高·简单保密就能收买）；② **逗逼白绝**（九卷「阿飞」身上那层白绝外壳·那些逗逼发言皆出自它·智商比蠢白绝高得多·带土曾披其壳伪装）。

营救迪达拉一役蠢白绝**化身「猪队友」**——把守监狱的明明是三代土影 [大野木](大野木.md)、他却报成黄土（害苏晓被迫硬抗尘盾），又不按苏晓嘱咐「先制造骚乱」、直接现身岩隐办公室（等于通报「晓来救人」、黄土立刻增援竖岩村·害苏晓险被前后夹击）、连 30 秒都没拖住。苏晓没追究（这种蠢失误他不在乎）、只敲打他别再幸灾乐祸，换他继续找外来者——蠢白绝识相地回报「**外来者在水之国附近现身又消失**」。本体黑绝仍在暗中推进辉夜复活的终极计划。

## 黑绝偷袭带土 · 复活斑与柱间（十五卷95–96章）

带土被抽走八尾、临死想用「外道·轮回再生之术」复活战场死者赎罪时，**黑绝凭超群的隐匿能力凭空附身到带土身上**（鸣人 / 波风水门事先都没感知到、被误导成「那只是空气或碎石」）、**夺走带土的写轮眼 + 轮回眼**，操控带土用轮回眼**复活了 [宇智波斑](宇智波斑.md) 与 [千手柱间](千手柱间.md)**（带土至死以为复活柱间是自己抗争黑绝的赎罪、实则是斑与黑绝的算计——斑要带土复活的从来是斑自己）。黑绝得手后即隐入地下、撇下带土等死。它的终极目标始终是复活母亲辉夜。
