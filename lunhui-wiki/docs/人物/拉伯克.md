# 拉伯克

**拉伯克** 是 [斩·赤红之瞳](../世界/任务世界/斩赤红之瞳.md) 世界 [夜袭部队](../势力/index.md) 的成员、帝具使（四卷24、27章）。

## 简介
绿发、身穿绿色风衣的帅气年轻人；平时显得好色，关键时却有底线（绝不占重伤的 [雷欧奈](雷欧奈.md) 便宜）；智商颇高（能精确计算敌人身高/跑速/方向来布线）。

- **帝具「交叉之尾」**（物品类）：一个缠着**超级危险种体毛**的线轴——其丝线兼具柔软、锋利、坚固（强度超一切已知金属），可灵活操纵；更能**屏蔽感知**（连苏晓都几乎察觉不到），布成切割人体的「**断界线**」。配合雷欧奈的恢复力**缝合断肢**，二人是「大难不死就能续命」组合。

本批中他以交叉之尾从苏晓刀下**拽走濒死的雷欧奈**、缝合其断臂；苏晓追击时险被其「断界线」割喉（及时停步）。后他听雷欧奈劝阻、未去寻苏晓，转而决定支援帝都港口豪华客船一队（布兰德/塔兹米/赤瞳/玛茵在）（四卷25–27章）。

## 结局（四卷50章）
狩人部队总攻夜袭总部时，他与 [塔兹米](塔兹米.md)、[玛茵](玛茵.md) 三人合力缠斗 [艾斯德斯](艾斯德斯.md)（凭智商勉强拖住、岌岌可危）。战后艾斯德斯手中拿着一卷**被彻底破坏的交叉之尾**——拉伯克与这届帝具默合度极高、发挥出 100% 威力，最终帝具破损，「其主人的下场可想而知」（暗示阵亡）。这卷破损的交叉之尾被艾斯德斯**转赠给苏晓**。

## 关系
- [雷欧奈](雷欧奈.md)——搭档（「续命组合」）；[娜洁希坦](娜洁希坦.md)（已殁·首领）、[赤瞳](赤瞳.md)（已殁）等同伴。
- [苏晓](苏晓.md)（白夜）——救雷欧奈、并以断界线险伤之的对手。
