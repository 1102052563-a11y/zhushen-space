# 茵蒂克丝（禁书目录·禁术目录）

[魔法禁书目录](../世界/任务世界/魔法禁书目录.md) 的**核心人物**——英国清教所属修女，自幼在英国圣乔治大圣堂长大、现今 14 岁。她拥有**完全记忆能力**，凭大脑**存储了十万三千本魔道书的全部内容**（即「**禁术目录**」），对魔法侧意义极重。

- **被「谎言」控制**：因她太重要，英国清教编了个谎言来控制她——称其「**脑容量有限、存完十万三千本魔道书后只剩一年记忆空间、每年须清除记忆一次否则会撑死**」，借此每年把她的记忆清回「刚记完魔道书」的节点。神奇的是真有人深信不疑、为「保护」她而苦斗，每年承受被她遗忘之痛（[神裂火织](神裂火织.md) 即其一）。
- **逃跑与剧情开端（十三卷12章）**：她逃跑、逃入学园都市，与本世界主角 [上条当麻](上条当麻.md) 偶遇结识、成为朋友。[神裂火织](神裂火织.md) 与魔法师史提尔·马格努斯追来想「救」她（清记忆·一年时限将近）——上条当麻误以为他们在追杀她、遂拼死保护。途中因上条的「幻想杀手」破坏了她的防御衣「移动教会」，神裂才误伤了她。
- **苏晓任务的核心**：[苏晓](苏晓.md) 受亚雷斯塔之命来击退神裂火织——他的真正目的是替科学侧把这个「禁术目录」**武力留在学园都市**（清教与科学侧交涉破裂、才派圣人来夺）。

## 关系
- **英国清教**：所属（视其为禁术目录·谎言控制）。
- **[上条当麻](上条当麻.md)**：逃入学园都市后结识的朋友（拼死护她）。
- **[神裂火织](神裂火织.md)**：想「救」她的圣人（误伤她·愧疚）。
- **[苏晓](苏晓.md) / 科学侧**：欲把她武力留下的一方。
