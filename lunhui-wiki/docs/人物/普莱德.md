# 普莱德（傲慢）

[瓶中的小人](瓶中的小人.md) 所造七名人造人之一，代表「**傲慢**」——七人造人中**最年长**（343 岁·孩童外形），苏晓评为**实力第二强**。

- **能力**：**能融入影子、用影子攻击**。
- **历程**：七卷5章在废弃研究所的阴影中与 [恩维](恩维.md) 议论苏晓（点明苏晓是瓶中小人钦点的「人柱候选」、「有弱点才好利用」）；七卷17章也曾在暗处旁观苏晓的假杀莉莎之计（评「无聊的计划·人类比你想象狡猾」）。
- **与苏晓**：苏晓知它「实力第二、却有一个**很致命的弱点**」，准备近期处理掉它。

## 出处
- 七卷5、17、19章。
