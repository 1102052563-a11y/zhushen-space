# 奥尔丁

[树生世界](../../世界/任务世界/树生世界.md) 里一名在天启乐园八阶契约者间**颇有名气**的**近战法爷**——三十出头、金蓝法袍、膀大腰圆，一根「法杖」看着就像钝器（被砸中非死即残、法系性能反被忽略）。他拉起一支**十四人临时小队**（8 名天启 + 6 名圣光·两方平时互看不爽、开局恶劣才合作），冲着 [艾朵儿·帕帕](艾朵儿·帕帕.md) 的坐标去——想生擒她、夺取「刷杀戮功勋的方法」一飞冲天。队里还有眼镜女·百莉、被害妄想症的感知系火琉、下巴纹十字的教徒、怨怒矮小的伊凡等。

## 一头撞进「好队友」的陷阱

他们明知艾朵儿身上有猫腻（特殊霸主被击杀却不换人、公告点名有人刷功勋），仍被「一飞冲天」的诱惑冲昏头。这一冲，正撞进 [苏晓](../苏晓.md)、[伍德](../画中世界/沃波·伍德.md)、[罪亚斯](../画中世界/罪亚斯.md)「好队友三人组」的钓鱼局:苏晓在异空间里用斩龙闪的「空间穿斩」隔空斩首、伍德把人转成二维神经图、罪亚斯的「寄髓虫」让教徒精神错乱举枪自尽（「给爷死！」）。小队从「14 人」被误导数到「9 人」再团灭。奥尔丁作为队长（功勋估计最多）重伤濒死后，被罪亚斯与伍德同时锁定、一根血枪 + 黑焰了结。

## 关系

- [艾朵儿·帕帕](艾朵儿·帕帕.md)：他想生擒夺功勋、反成钓他上钩的诱饵。
- [苏晓](../苏晓.md) / [伍德](../画中世界/沃波·伍德.md) / [罪亚斯](../画中世界/罪亚斯.md)：把他 14 人小队钓杀团灭的「好队友三人组」。
