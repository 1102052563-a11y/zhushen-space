# 菲洛

[树生世界](../../世界/任务世界/树生世界.md) 里一名俊美但气质弱气的少年参战者——满头浅黄碎发、笑起来阳光，颜值能排全场前两名。绰号「小洛」，是 [鳞龙·亚力克](鳞龙·亚力克.md) 护着的人。他不敢与 [苏晓](../苏晓.md) 硬拼（感知里苏晓强得像怪物），却想当「捡漏」的最后赢家——趁乱去摸紫色物资箱。

## 此生前所未有的危机

就在他即将碰到物资箱的一刻，[国足三兄弟](../国足三兄弟.md) 的钉锤【末陨】把他拖进了封闭的圆筒空间——迎接他的是三个只穿健美三角裤、手持能量锤的猛男。国足老二先用三角裤皮筋「啪」地抽自己胯部、对他一声「baby～欧耶」，直接把他的心态**当场打崩**（他一路拼杀重伤濒死过，却觉得眼前是「此生前所未有的危机」）。被能量锤锤到蜷缩在地、衣衫残破、泪花在眼角——所幸国足三兄弟秉承「能不杀就放生」，没取他性命。随后 [鳞龙·亚力克](鳞龙·亚力克.md) 俯冲救场（还失手一脚踩上他脑袋），把他拎着冲天而去。

## 关系

- [鳞龙·亚力克](鳞龙·亚力克.md)：护着他、来救他的「小洛」的同伴。
- [国足三兄弟](../国足三兄弟.md)：把他拖进【末陨】空间、锤崩他心态的三个猛男。
