# 宿命之子·尤尔

[树生世界](../../世界/任务世界/树生世界.md) 里 [精灵王·克伦威](精灵王·克伦威.md) 的**「宿命之子」**——克伦威与 795 名血脉纯净的女精灵在几十年间生了 5192 名子嗣，这些孩子生来体内带畸变的深渊之力，有三大共性:① 早智（三四岁心智初成、十岁能掌握超凡能力）② 超强深渊抗性 ③ **同源同命**（同一父亲、同一能量、灵魂波长极近、能力血脉互相兼容）。克伦威把他们赶进「猎场」互相狩猎——杀死同胞不仅夺灵魂力量、还能夺能力，几千个「小号」最终合并成一个由单独意识掌控的「满级号」，那个唯一的优胜者就是尤尔（15 岁起就在狩猎、直到猎场被清空）。

## 稚气未脱的弓术强者

尤尔外表稚嫩、性格单纯（会被 [艾朵儿](艾朵儿·帕帕.md) 无意的话伤到自尊），却背负「使命高于一切、包括生命」的宿命。他用一把暗红大弓，招式名朴实（「蓄力箭」「跃箭」「三连箭」）却凶悍——**蓄力箭从蓄力开始的瞬间就能激发、越蓄越强**（锁定即令敌人感知疯狂预警、动一下就爆头）。反常的是他不拉开距离、反而前滚近身射（怕爆发力不足），四成蓄力的一箭就把光头壮汉射爆。他首次对抗「兄弟姐妹之外的敌人」、被 [苏晓](../苏晓.md) 三人（罪亚斯生触手 / 伍德噬魂 / 苏晓身后仿佛有血兽）确认自己「加入了恶阵营」也不在意。他被父亲以残魂信托付给苏晓、加入「破晓队」进大遗迹（父亲另派蘑菇骑士为他清路却惨败）。

## 关系

- [精灵王·克伦威](精灵王·克伦威.md)：他的父亲、用五千子嗣养蛊出他这个「容器」、又以残魂信把他托付给苏晓。
- [苏晓](../苏晓.md)：带他进大遗迹的破晓队队长（他要对付圣战士·焚薇）。
- [蘑菇骑士](蘑菇骑士.md)：父亲派来为他清大遗迹之路、却惨败战死的蘑菇村强者。
