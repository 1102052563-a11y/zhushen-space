# 兽豪

[树生世界](../../世界/任务世界/树生世界.md) 开局围攻 [苏晓](../苏晓.md) 的五名主力之一——原代号「**野兽剑豪**」、久之简称兽豪，用一把无护手长刀。他是这群违规者里少见的**有骨气者**:本不愿参与「这么多人围攻一人」，但 [灰绅士](../炎灵星/灰绅士.md) 的计划深深打动了他，甚至让他生出自惭形秽之感——他们这些违规者「说好听叫追求自由、说难听就是混日子、大部分人躲着猎杀者处刑者」，灰绅士给了他一个无法拒绝的价码和目标。

## 爱刀崩口，识时务而退

他一直没上前拼近战，因为苏晓被大群违规者围攻时上去也会被波及。等他终于与苏晓刀锋相抵，两刀互磨、**他的刀当场崩出缺口**——「在他心中，他的刀就和他老婆一样」，脸一抽立即退开。他手边还有个脑子不太聪明的暗杀少女队友 [蜂](蜂.md)（他抬手按她的头）。围攻以违规者惨败告终，兽豪属于识时务撤走的幸存者。

## 关系

- [灰绅士](../炎灵星/灰绅士.md)：用打动人心的计划 + 无法拒绝的价码雇他的主使。
- [苏晓](../苏晓.md)（霸主级单位）：与他拼刀、崩了他爱刀的对手。
- [蜂](蜂.md)：他照看的暗杀系少女队友。
