# 珀·佩斯洛

[树生世界](../../世界/任务世界/树生世界.md) 寒冷墓地鬼族「地城·丘黎」出身的**新一代最强者**——一名十六七岁、容貌俊美、先天自带眼影的叛逆期少年（「他老大、天老二、地老三」）。他与妹妹 [珀·米娅](珀·米娅.md) 是鬼族这一代的**继承人候选**，属于亲近 [女王·尤罗](女王·尤罗.md) 的派系（从懂事起就是女王的小迷弟小迷妹、一直想见女王一面）。

## 朝圣路上的横祸

按家中长辈规矩，佩斯洛与米娅要正式成为继承人，须先完成**朝圣**——从寒冷墓地出发、去大树洞之底的女王寝殿。兄妹俩历经千辛万苦、绕路躲毒瘴、步行两个多月才走完三分之一，却在白色沼泽被 [传光人·安德森](传光人·安德森.md) 扣下。安德森一听「朝圣及其蕴意」就把他们扣押、逼他们「赞美太阳」。血气方刚的佩斯洛不服、跳起给安德森胸膛一拳——反震得自己右臂骨裂（还被一句「手打疼了吧」暴击自尊心）；他抽匕首刺安德森小腹，被绑起来问「你是要杀我吗」，恍惚间看到一名满身黑烟的刽子手站在身后、自己被绑上斩首台桩——当场吓尿、羞怒欲社会性死亡。安德森察觉他「好多于坏」，只往不致命处招呼、还和善地「感化」他:「你心中有邪恶，但不要怕，太阳会净化你」。他亲眼见安德森一巴掌把超凡巨鳄「死亡之口」抽成陀螺、逼得那鳄鱼委屈得哭着用前爪赞美太阳，只能认命跟着赞美（心里仍写满「不服」）。

## 关系

- [珀·米娅](珀·米娅.md)：他的妹妹、同为继承人候选（一起朝圣、一起被扣押赞美太阳）。
- [女王·尤罗](女王·尤罗.md)：他崇拜、想朝圣拜见的鬼族女王（已被苏晓斩杀·他尚不知）。
- [传光人·安德森](传光人·安德森.md)：扣押他、逼他「赞美太阳」、把他自尊心打成筛子的传光人。
