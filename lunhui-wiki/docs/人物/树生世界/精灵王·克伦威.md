# 精灵王·克伦威

[树生世界](../../世界/任务世界/树生世界.md) 南大陆最强势力**精灵族**的现任王——从诸多竞选者中脱颖而出、已在位 27 年。精灵王位自初代王 [伯莱·阿隆德](伯莱·阿隆德.md) 起就不以血脉传承，而是**激烈危险的淘汰制**（高精灵中竞选）。克伦威是历代精灵王中**首个被完全承认的**:加冕时被冠冕引来的黑森林**暗灵们全部跪伏在地**（不像伯莱·阿隆德要靠武力打退暗灵）。他既有智谋、武力也达**冕王级**。

## 只信亲眼所见

[神父](../神父.md)（带 [仙姬](../联盟星/仙姬.md)、[乌鸦女](../画中世界/乌鸦女.md) 先 [苏晓](../苏晓.md) 两天抵达贝城）想借克伦威之手除掉苏晓，却发现这王没那么好忽悠——「克伦威只会相信自己所看到的事」。他不敢直接说苏晓是灭法者（一说就会被反问「你怎么知道灭法者、怎么知道精灵族怕灭法者、难道你知道天赋唤醒装置、知道精灵族最大的秘密」，等于自爆精灵族最大机密）。所以克伦威没完全答应与神父合作，而是派中心街区巡查队长 [阿尔勒](巡查队长·阿尔勒.md) 去中心公园试探苏晓的身份与来意（如可疑可当场格杀）——正是这个「派人试探而非直接围杀」的动作，让苏晓确认「克伦威名不虚传、没被神父忽悠」。

## 关系

- [伯莱·阿隆德](伯莱·阿隆德.md)：开创精灵族淘汰制王位的初代精灵王（克伦威是首个被暗灵完全承认的继任者）。
- [神父](../神父.md) / [仙姬](../联盟星/仙姬.md)：想借他之手除掉苏晓、却没能轻易忽悠他的违规者。
- [苏晓](../苏晓.md)：被他派 [阿尔勒](巡查队长·阿尔勒.md) 试探、伪装成「药师白夜」潜入贝城的猎杀者。
