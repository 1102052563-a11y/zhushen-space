# 蘑菇骑士

[树生世界](../../世界/任务世界/树生世界.md) 蘑菇部族里一位低调却极强的骑士——椭圆大头盔、铁桶般铠甲、短粗双腿、不成比例的长臂，持双手大剑。蘑菇村虽小，却在村长 [蘑菇先知](../魔海/蘑菇先知.md) 荫庇下人才辈出，蘑菇骑士与 [女王·尤罗](女王·尤罗.md)、[精灵王·克伦威](精灵王·克伦威.md) 同属本世界金字塔尖的强者（只是更低调、略弱一筹）。

## 为宿命之子清路的悲情骑士

[精灵王·克伦威](精灵王·克伦威.md) 早知精灵族末路，除了养蛊出容器 [宿命之子·尤尔](宿命之子·尤尔.md)，还安排蘑菇骑士替尤尔**打通大遗迹之路、清理里面的强敌**（特意不让尤尔跟着以免团灭）。克伦威死后以残魂通知它火速行动，可它挑战「水生之母」的**五方力量节点**惨遭滑铁卢、力竭撤到大遗迹长廊。[苏晓](../苏晓.md) 一行到来时它已必死，[尤尔](宿命之子·尤尔.md) 单膝跪地「您辛苦了」，它憾言「精灵王的儿子都长这么高了，可惜我没能达成约定」，用生命探出的情报和盘托出后气息消散（它的死与火影自来也颇像——死于试探敌人的能力）。

## 揭出「水生之母·五力量节点」

它临终揭破关键机制:贝城的畸变是 [水生之母](水生之母.md) 的力量源泉，只要畸变还在、水生之母就杀不死。它与贝城的联系有**五个「力量节点」**（都是承受畸变的强者）:① [王后·西格莉安](王后·西格莉安.md) ② 四生恶鬼（[渔村四人](渔村四人.md) 畸变）③ [五王裔](王裔·埃里顿.md) ④ 圣战士·[焚薇](女战士·焚薇.md) ⑤ 死亡之影·[迪尤克](鬼影·迪尤克.md)。五节点彼此相连、能靠畸变力量复生，必须在 **20~25 分钟内全部清除**才会死透，之后才能杀死水生之母。

## 关系

- [精灵王·克伦威](精灵王·克伦威.md)：委托它替宿命之子清大遗迹之路的精灵王。
- [宿命之子·尤尔](宿命之子·尤尔.md)：它拼死为之开路的精灵王之子（尤尔为它跪送）。
- [苏晓](../苏晓.md)：它临终把五节点情报和盘托出的目的相似者（都要除水生之母）。
