# 无面人·佩特·佩伯

[树生世界](../../世界/任务世界/树生世界.md) 黑暗之域「黎明镇」的诡异囚徒——[苏晓](../苏晓.md) 初到黎明镇时敲门、只会一字不差复读来者话的「模仿男」，真名**佩特·佩伯**，脸上空白一片、穿一身礼服。他的诡异程度直逼 [艾莉亚](艾莉亚.md)（女王之姐）——凭一张照片就能让丝线刺入自身、飞速把容貌 / 体型 / 衣着 / 声线 / 气息**完全复刻成照片里的人**（苏晓给灰绅士照片，他当场变得与灰绅士一模一样、连声线都像调音般逼近）。

## 变身刺客

苏晓用他的画像作契约，放他去猎杀 [灰绅士](../炎灵星/灰绅士.md)。他不满足于「像」——主动要苏晓指一个「与灰绅士关系近的人」，好**先吞掉那人、完全变成他、再去接近灰绅士**（比单纯复刻更致命）。苏晓给他选定了 [兽豪](../树生世界/兽豪.md)（灰绅士麾下违规者）。他看完兽豪的照片、发出怪异笑声，奔向黑暗之域出口。他与 [猪兄·巨罗](猪兄·巨罗.md)、[迷雾](艾莉亚.md) 是苏晓同批放出去、专袭灰绅士的三尊「不是诡异到极点、就是战力强悍」的存在。

## 关系

- [苏晓](../苏晓.md)：用他的画像作契约、放他出去刺杀灰绅士的猎杀者。
- [灰绅士](../炎灵星/灰绅士.md)：他要吞兽豪、变身潜近刺杀的目标。
- [兽豪](../树生世界/兽豪.md)：他准备吞掉、完全变成以接近灰绅士的灰绅士麾下违规者。

## 🆕 死于友军的「太阳圣剑」（46卷45章）

被苏晓放出黑暗之域后他一直在找机会刺杀 [灰绅士](../炎灵星/灰绅士.md)（本要吞兽豪变身接近）。灰绅士第四阶段现身时，他与 [猪兄·巨罗](猪兄·巨罗.md) 火速赶来，却还没出手就双双死在 [苏晓](../苏晓.md) 的「太阳圣剑」里。掉落宝箱类物品「灵宴宝盒·万面」，被苏晓「含泪舔包」收下。

## 关系（补充）

- [苏晓](../苏晓.md)：放他出去杀灰绅士、却误用太阳圣剑把他炸死的猎杀者。
