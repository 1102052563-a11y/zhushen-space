# 王后·西格莉安

[树生世界](../../世界/任务世界/树生世界.md) [精灵王·克伦威](精灵王·克伦威.md) 的**首任妻子**（人生中的第一位王后）——[水生之母](水生之母.md) 的**五方力量节点之一**，也是其中最难缠的一个（[蘑菇骑士](蘑菇骑士.md) 战死的主因之一）。

## 从废物到无解的「同命相连」

她畸变前的能力堪称废物:**伤害自己、让敌人感受同等的「痛楚」**（她给自己一刀会血流不止、敌人却只是疼、无实质伤）。可畸变后质变成无解——只要看到敌人就能与之「**同命相连**」，她所承受的伤势会 **100% 同步给敌人**；更狠的是她畸变后有超强恢复力、**自愈不会同步**（她割开自己喉咙下一秒就恢复，敌人却双手捂喉喷血）。加上她是五节点之一、只要其他节点没死光、她死了也能从大遗迹的血淤里再生。这套「割自己=杀敌人、且我不会死」的能力太克制常规战法，[苏晓](../苏晓.md) 的破晓队特意安排不死的 [罪亚斯](../画中世界/罪亚斯.md) 去对付她（其他人都不行）。

## 关系

- [精灵王·克伦威](精灵王·克伦威.md)：她的丈夫、精灵王（首任妻子）。
- [水生之母](水生之母.md)：与她「同命」的畸变本源（她是其五力量节点之一）。
- [罪亚斯](../画中世界/罪亚斯.md)：破晓队里被指派专门对付她的不死古神系。
