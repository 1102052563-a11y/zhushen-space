# 珀·米娅

[树生世界](../../世界/任务世界/树生世界.md) 寒冷墓地鬼族的继承人候选之一——[珀·佩斯洛](珀·佩斯洛.md) 的妹妹，约十四五岁、同样容貌俊美自带眼影。她与哥哥是亲近 [女王·尤罗](女王·尤罗.md) 的派系、从小的女王小迷妹，一起踏上从寒冷墓地去大树洞女王寝殿的**朝圣**之路（走了两个多月、才三分之一）。

## 被扣押赞美太阳

兄妹俩在白色沼泽被 [传光人·安德森](传光人·安德森.md) 扣押、逼「赞美太阳」。相比硬顶到右臂骨裂、还吓尿的哥哥，米娅没被「物理纠正」，但也被安德森的可怕吓得泪眼婆娑、乖乖保持拥抱太阳的姿势——安德森一句「孩子，你感受到太阳了吗」，她连连点头。她比叛逆的哥哥更早认命。

## 关系

- [珀·佩斯洛](珀·佩斯洛.md)：她的哥哥、鬼族新一代最强（一起朝圣、一起被扣押）。
- [女王·尤罗](女王·尤罗.md)：她崇拜、想朝圣拜见的鬼族女王（已被苏晓斩杀）。
- [传光人·安德森](传光人·安德森.md)：扣押她、逼她赞美太阳的传光人。
