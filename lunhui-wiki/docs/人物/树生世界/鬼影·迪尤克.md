# 鬼影·迪尤克

[树生世界](../../世界/任务世界/树生世界.md) 精灵族王宫负责**反暗杀 / 潜入**的高手——原是暗杀部队的大头目，因表现优秀被提拔进王宫（唯有了解暗杀，才知道怎么防范）。三十出头、黑红面巾遮下半脸、瘦而骨架大、惨白瞳孔与人对视都令人不寒而栗，绰号「鬼影」「剥皮者」「红手」，沉默残酷、爱好是**收集眼球**。他能半化为暗绿色烟气穿墙、把敌人吸成人干化作烟丝没入手臂。

## 苏晓真正忌惮的护卫

在两名贴身护卫里，苏晓把 [焚薇](女战士·焚薇.md) 划为低威胁，却把迪尤克视为**真正的麻烦**——在这种反暗杀老手眼皮底下搞小动作难如登天，一旦被察觉蛛丝马迹就可能满盘皆输（精灵王派他来，目的之一正是防这个）。但苏晓从不小看人、也有的是办法收拾他:趁问询暗杀之事，给他下了泻药——「剥皮者」鬼影·迪尤克「屎来如山崩地裂」、脸青冒汗、夹着腿迈小碎步「出去巡逻」，把 [焚薇](女战士·焚薇.md) 看得一脸懵。后来禁卫军长庞·凯鳞带死士来灭口时，是迪尤克（深知怎么在这种局里保住自己）出手拦下、逼庞·凯鳞给个「理由」，客观上替苏晓稳住了场面。

## 关系

- [焚薇](女战士·焚薇.md)：与他一同护卫苏晓的贝城最强女战士。
- [苏晓](../苏晓.md)（药师白夜）：忌惮他的反暗杀能力、却用一支烟（泻药）把他支开的猎杀者。
- [精灵王·克伦威](精灵王·克伦威.md)：提拔他进王宫、派他护卫兼监视苏晓的精灵王。

## 🆕 死亡之影·迪尤克 · 水生之母力量节点（46卷36章）

贝城畸变后他化为 [水生之母](水生之母.md) 的**第五力量节点「死亡之影·迪尤克」**（原精灵王身边最强暗杀者的畸变形态）。[蘑菇骑士](蘑菇骑士.md) 给的情报里，他的信息最少、最难测。破晓队里本无人能应对（凯撒惜命不肯上、[奥娜](奥娜.md) 又不能与 [罪亚斯](../画中世界/罪亚斯.md) 同时出场），[苏晓](../苏晓.md) 只好用半格信号的通讯器把 [所罗门](../所罗门.md) 请来专门对付这个「死亡之影」。
