# 鹿格 / 匿名者

[树生世界](../../世界/任务世界/树生世界.md) 杀戮竞技里的一名天启乐园八阶契约者，世界联络平台代号「**匿名者**」——在天启乐园八阶以「又贱又骚气」的行事风格出名，最大特征是戴麋鹿角骨盔、披灰白长毛兽皮，活像刚从荒野归来的猎人。他滑溜得像泥鳅，各类防追踪 / 逃遁手段极强，偶尔还在世界联络平台和 [巴哈](../苏晓.md) 文斗几句。

## 被苏晓、所罗门联手「控榜」

世界进度尾声，参战者剩不足百人、排名难再大变，[苏晓](../苏晓.md) 想要第二名奖励「技法之魂」、[所罗门](../所罗门.md) 想要第四名奖励「暗黑君主·传承之力」，唯一的变数就是第四名的匿名者（可行走的 74 点杀戮功勋）。两人（联合 [凯撒](../尼古拉斯·凯撒.md)、蘑菇先知、咕噜）设了个连环局:蘑菇先知谎称打折抛售搬家把匿名者钓回蘑菇村→凯撒当街碰瓷触发支线任务「老友」（任务里的「老友」正是凯撒本人·送酒等于自投罗网）→鹿格在废弃祭祀殿送酒时被苏晓、所罗门堵个正着。鹿格审时度势、谈笑风生地答应所有非剥削条件（听说能当排行第三、风暴大剑「一看就很屌」，他主动报名），却被苏晓逼吃下一颗「慢性剧毒」丸留作把柄——**其实那是维生素 B3**（所罗门还认真提出五五分摊毒药费、问解毒法，苏晓答「给他吃钙片」）。最终排名如两人所愿:恩左第一、白夜第二、匿名者第三（风暴大剑）、所罗门第四（暗黑君主·传承之力）、咕噜第五。鹿格安然离开树生世界，却始终对那「未知猛毒」心怀敬畏。

## 🦌 天赋大揭示 · 「吸收队友运势」的移动灾星（48卷28章）

⚠ 到了幽暗大陆·死寂城，鹿格作为 [凯因](../潘多拉星/凯因.md) 队一员被审讯时，其恐怖天赋才被揭开:他的天赋能力是**吸收身边人的运势、壮大己身**——和 [豪妹](../塞尔星/豪妹.md)「影响身边人运势」的天赋像却本质不同（豪妹是让人倒霉、可被强运的人压住反成众人同走运；鹿格却是**直接抽干队友的运势导致队友暴毙**）。他从一阶到四阶送走的队友多到自己都不敢记；五阶天赋自行觉醒、临时组队都有生命危险；八阶二次觉醒达巅峰，**仅仅临时组队就有 90% 以上概率让队友因各种危险暴毙**。他最常做的事就是「含泪捡起队友的猩红卡」。⚠ 鹿格没有堕落，反而看开了:既然不能拥有队友情，就干脆把这天赋当**武器**，专门加入那些心怀叵测的临时队伍（如 [凯因](../潘多拉星/凯因.md) 的英灵殿），既获取大量资源、又不必背叛谁——因他是天启乐园契约者，进不了死亡乐园阵营英灵殿的正式编制，只能当临时成员。

## 🗄 死寂城搜刮 · 被苏晓审讯放走（48卷27–28章）

⚠ 鹿格与 [雪怪](../幽暗大陆/雪怪.md) 凭 [机械公爵](../幽暗大陆/机械公爵.md) 提供的庇护石进死寂城、又走隐秘路线直达内城，本要搜刮圣歌团宝库，却撞上刚战胜圣歌团的 [苏晓](../苏晓.md)。被倒吊审讯、见识过 [咕噜](../咕噜.md) 的仁慈之刺后，两人如实交代了公爵知道秘密通路、宝库及进宝库方式的要害情报。⚠ 鹿格识相地分两次交出 12700 + 2790 共 **15490 灵魂钱币**买命（比只有 4950 的雪怪阔气得多）——他知道上个世界苏晓收了天启三姐妹的钱却没灭口，才敢信这奸商「收钱办事」。放走后他与雪怪出宫殿撞见满地教会骑士又缩回，最后靠雪怪的圆盘机关开空间通道，随公爵、凯因一起在苏晓从空间通道现身、公爵引爆时逃遁。

## 关系

- [苏晓](../苏晓.md) / [所罗门](../所罗门.md)：联手做局把他锁在第三名、还骗他吃「维生素毒药」的两名轮回乐园契约者。
- [凯撒](../尼古拉斯·凯撒.md)：碰瓷他、当「老友」任务诱饵把他钓进局里的老演员。
- [凯因](../潘多拉星/凯因.md) / [雪怪](../幽暗大陆/雪怪.md)：与他从潘多拉星到死寂城臭味相投、「看谁先死」的坑队友二人。
- [豪妹](../塞尔星/豪妹.md)：天赋与他「像却本质相反」的天启乐园契约者（她影响运势、他直接抽干队友运势）。

## 🥾 被臭鞋炸昏 · 又一次「维生素式」勒索（48卷34–35章）

⚠ 好队友清场公爵队时，[凯撒](../尼古拉斯·凯撒.md) 往公爵队藏身的小楼里丢燃烧的臭鞋炸弹，鹿格连滚带爬冲出、被第二只臭鞋熏得当场昏迷、成了唯一被活捉的公爵队成员。⚠ 苏晓要他配合「**伪装公爵队去联合乌鸦队、沃姆队**」的局，鹿格起初硬气拒绝（连遭两次擒获），苏晓一亮**仁慈之刺**他立刻服软。为拿捏他，苏晓又故技重施:给他吞下一颗「五小时后会在胸膛里爆炸的阿波罗胶囊」——鹿格心想「这次有机会就溜、不再被维生素吓住」，却不知苏晓当着他吞的和另捏碎的根本是**脆皮水溶胶囊**（一进胃就溶、吐胃都找不到炸弹）。他遂替苏晓跑腿:通知乌鸦队、沃姆队三小时后狼冢石碑相见，还捎去一块石板当「公爵的诚意」。

## 关系（补充）

- [苏晓](../苏晓.md)：逼他吞「假爆炸胶囊」、驱他跑腿串联三队石板局的灭法者。
