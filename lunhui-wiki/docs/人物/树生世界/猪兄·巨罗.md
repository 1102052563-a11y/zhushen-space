# 猪兄·巨罗

[树生世界](../../世界/任务世界/树生世界.md) 黑暗之域「黎明镇」的极恶囚徒之一——[苏晓](../苏晓.md) 初到黎明镇时敲窗、只听到「无法交流」的暴躁猪叫声的「猪兄」，真名**巨罗**，是个屠夫。近三米高、猪头人身、穿屠宰服、粗壮手臂遍布缝合痕、身上飘散浑浊的暗黄色恶意，脾气极其暴躁。能被囚进黑暗之域，本身就说明它不是诡异莫测就是强悍凶残（女王六画像之一）。

## 换画放出去杀灰绅士

苏晓从 [咕噜](../咕噜.md) 手里得的女王六画像，对被囚的黑暗住民而言象征「自由」。苏晓用巨罗的画像作契约凭证（交付瞬间画上浮现极强契约之力·无需担心赖账），换它去猎杀 [灰绅士](../炎灵星/灰绅士.md)——一提「杀人」正中它擅长领域、语气立刻缓和，一听目标是灰绅士（不知其所在·「那需要的时间会更长」）便答应，破门而出、确认苏晓「都是极恶阵营」后点头示意、径直奔黑暗之域出口而去。它能否成功、灰绅士能否顶住这尊暴躁屠夫，苏晓也没底（「希望人没事」）。

## 关系

- [苏晓](../苏晓.md)：用它的画像作契约、放它出去杀灰绅士的猎杀者。
- [灰绅士](../炎灵星/灰绅士.md)：被苏晓派它（+ 无面人 + 迷雾）去猎杀的目标。
- [无面人·佩特·佩伯](无面人·佩特·佩伯.md) / [迷雾](艾莉亚.md)：同被苏晓放出去杀灰绅士的黑暗之域囚徒。

## 🆕 死于友军的「太阳圣剑」（46卷45章）

它被苏晓放出黑暗之域后一直在找 [灰绅士](../炎灵星/灰绅士.md)（受女王画像的契约约束）。灰绅士第四阶段终于露面时，它与 [无面人·佩特·佩伯](无面人·佩特·佩伯.md) 火速赶来袭杀，却还没来得及出手，就双双死在 [苏晓](../苏晓.md) 那把两千颗阿波罗轰出的「**太阳圣剑**」里（连灰都没剩·「厚葬友军别扯了」）。苏晓「悼念它 0.5 秒」——它产出的宝箱类物品「憎恨之骨匣·巨血手」精致贵重，苏晓「含泪舔包」收下。

## 关系（补充）

- [苏晓](../苏晓.md)：放它出去杀灰绅士、却误用太阳圣剑把它炸死的猎杀者（含泪收它的宝箱）。
