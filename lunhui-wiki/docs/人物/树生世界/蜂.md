# 蜂

[树生世界](../../世界/任务世界/树生世界.md) 开局围攻 [苏晓](../苏晓.md) 的五名主力之一——一名年龄不大、气质极冷的娇小少女，给人最强烈的感觉是「**穿透性的锐利**」。她是**中距离 + 近战暗杀系**，靠暗杀系的高速度想放苏晓风筝、中距离奇袭。有个反差:她的面无表情其实不是气质冷，而是**脑袋不太聪明**（被 [兽豪](兽豪.md) 按头时只是茫然仰头后看）。

## 奇袭反被钉墙

她原以为苏晓是纯近战系，想凭高速度中距离奇袭——结果一头撞上苏晓的血枪，被一根血枪钉在石壁上、白皙小手淌血，要不是 [马尾男](马尾男.md) 及时增援，后续还要被血枪炸。她低估了苏晓「刀术宗师 + 近战宗师却还有远程能力」的全能。围攻失败后她随幸存违规者溜走。

## 关系

- [兽豪](兽豪.md)：照看她、抬手按她头的剑豪队友。
- [马尾男](马尾男.md)：增援救她脱离血枪的指挥。
- [苏晓](../苏晓.md)（霸主级单位）：被她奇袭、反手血枪钉墙的对手。
