# 深渊守卫者

[树生世界](../../世界/任务世界/树生世界.md) 通往大遗迹途中、古老大殿里的一尊**「无法战胜之敌」**——身高 5 米、肌肉如铁铸、皮肤黑红、满头卷发、无双眼（十几个气孔代口鼻）、耳垂挂印纹金环、体型完美却无性别。虚空之树只测出 8.53% 资料:类别「深渊与？？？夹缝中的存在」、生命值 150%（始源深渊之力加持）、力/敏/体/智全 ??? 、技能全 ???。因贝城完成畸变，它从「夹缝」的天棚裂痕误入这里——树生世界战力上限虽名义八阶、却对误入的超阶存在**不立即排斥**（本世界的坑人之处）。

## 一个被吵醒的睡神

它其实对闯入者**没有敌意**，只是觉得这里深渊之力特殊、跑来接着「呼呼大睡」（[蘑菇骑士](蘑菇骑士.md) 就因它无敌意而顺利通过）。可 [苏晓](../苏晓.md) 一行闯进它的「卧室」、还有它的「敌人」——它被吵醒后二话不说、瞄准 [伍德](../画中世界/沃波·伍德.md) 隔空一拳（并非与魔鬼族有仇，而是伍德长期带 [深渊之罐](../../力量体系/装备道具.md)、沾了它气息，它在**优先猎杀深渊之罐的前持有者**）。苏晓向它丢 [茂生之狂乱](../../力量体系/装备道具.md) 树根、又祭出 [死灵之书](../../力量体系/装备道具.md)，它气息一滞、放弃扑杀苏晓、把死灵之书**特意朝伍德方向拍飞**（铁了心搞死伍德）——死灵之书就此砸进伍德胸膛、选择了伍德（伍德刚送走深渊之罐又喜迎新爹）。它随后自断右臂砸向苏晓、钻回天棚裂痕退走（裂缝愈合后再回不来）。它的遭遇与「宅家地宫的古神」颇像——睡得正香被吵醒、还被卑鄙地一直丢危险物，只能自断一臂匆忙撤走。

## 关系

- [伍德](../画中世界/沃波·伍德.md)：它铁了心优先猎杀的「深渊之罐前持有者」（把死灵之书拍进他胸膛）。
- [苏晓](../苏晓.md)：闯进它「卧室」、用茂生之狂乱 + 死灵之书把它逼退的猎杀者（它对灭法者其实不算敌对）。
