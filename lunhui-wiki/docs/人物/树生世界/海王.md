# 海王

[树生世界](../../世界/任务世界/树生世界.md) 开局围攻 [苏晓](../苏晓.md) 的违规者主力之一——穿有鳞纹的暗绿色紧身衣、头发一丝不苟。他是**水刀技法型近战**:双手探进身后涌动的海水，抽出时十指间夹满水刀，甩出残影般的**上千把水刀**如暴雨刺来，压制力极强（连苏晓都得靠黑王护臂 + 晶体层硬挡）。

## 技法型近战的宿命

他强，却撞上了「与技法型近战对拼要有短时间死于非命的觉悟」这条铁律。苏晓在时之领域（刃道刀·时）里破了他的水刀压制、突袭到他面前一刀断双臂；他慌喊「山，换！」想让坦系铁山救援互换位置未果，只能激活空间系保命手段化透明遁走。可 [苏晓](../苏晓.md) 的龙影闪能感测周边几十米空间波动——「刃道刀·流」凭空斩向空无一人之地，海王正好在那里现身、当场被斩断脖颈，满是不敢置信的头颅飞起。他死得太突然，突然到让指挥 [马尾男](马尾男.md) 心底涌现寒意。

## 关系

- [苏晓](../苏晓.md)（霸主级单位）：破他水刀压制、预判斩他首级的猎杀者。
- [铁山](铁山.md) / [马尾男](马尾男.md)：他喊话求救的主坦、同批围攻的指挥。
