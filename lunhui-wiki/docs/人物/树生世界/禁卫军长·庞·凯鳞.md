# 禁卫军长·庞·凯鳞

[树生世界](../../世界/任务世界/树生世界.md) 精灵之都·贝城的**禁卫军长**——魁梧、银黑半身甲、不怒自威的老将，名义上统管全城城卫军（实权其实握在王裔权贵手里·他只是代表）。正因这官职是「风光的空壳子」，他一直急于往上爬。他不知不觉**站队了 [神父](../神父.md)**（把能治「浊血症」的神父当救星、想借盟友的救世主表现再升一级）。

## 一夜从天堂到地狱

[苏晓](../苏晓.md) 伪装药师白夜潜入后打乱了他的算盘。神父忽悠他带死士去寝殿灭口苏晓（其实是让他把事闹大、死在寝殿），被 [鬼影·迪尤克](鬼影·迪尤克.md) 逼着要「理由」——他甩出「白夜几个月前用深渊之力污染贝城地下水」的同款脏水。正僵持时，心腹来报「神父等人已被控制（疑对地下水下毒）」，敏锐的他立刻判断大事不妙、决定带家眷逃离贝城。可他刚走到后城区街头，就撞上苏晓雇来的 [渔村四人](渔村四人.md)——为首的大背头拿着他的**画像贴脸比对**、确认后一杀鱼刀从下巴捅进天灵盖，四人一顿乱捅割喉。他至死错愕:任谁都想不到杀手会拿着目标画像来比对，更想不到四个「土包子」如此又狠又强。他一死，空出的禁卫军长位子被凯撒运作、由 [阿尔勒](巡查队长·阿尔勒.md) 顶上。

## 关系

- [神父](../神父.md)：他不知不觉站队、又被忽悠去送死的「救星」盟友。
- [苏晓](../苏晓.md)（药师白夜）：打乱他算盘、雇渔村四人除掉他的猎杀者。
- [渔村四人](渔村四人.md)：拿他画像贴脸比对、乱刀捅死他的杀手。
- [阿尔勒](巡查队长·阿尔勒.md)：他死后顶替其禁卫军长之位的巡查大队长。
