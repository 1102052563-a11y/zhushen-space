# 艾莉亚

[树生世界](../../世界/任务世界/树生世界.md) 黑暗之域「黎明镇」里独居的女子——[女王·尤罗](女王·尤罗.md) 的**孪生姐姐**。她自称「鬼族几百年内最废物的继承人」（还一副理直气壮）；若说女王是强大而平易近人，艾莉亚就是**温柔与脆弱**（一听到女王死讯就泣不成声「她总说总要有人站出来，可她最终还是死了」）。心智像小孩、还是个和妹妹一样的吃货（[苏晓](../苏晓.md) 拿五颗灵魂晶核换情报被她嫌弃、换成 [贝妮](../布布汪.md) 的甜品才吸溜着吃、友好度大增连称呼都从「灭法者」变「白夜」）。

## 深渊之女的另一面：看到与许愿

她与女王的身世揭开了深渊最深的秘密——她们的**母亲是鬼族**，某天「无意间走进黑暗、走出来时肚子已经很大」，隔天就生下姐妹俩；她们的**父亲，人们称之为「深渊」**（女王类别「鬼族 / 深渊长女」不是被侵蚀、而是**字面意义的深渊之女**）。妹妹尤罗资质惊人、两年成北大陆最强；而艾莉亚这个异常存在从诞生就有两种能力:

- **看到**:能看到许多事物（有些超出她的理解），如天赋唤醒装置在极南的「大遗迹」——正是她告诉苏晓的（每人只能让她「看到」一次·需等价交换、否则会「因连」到问者身上）。
- **许愿**:她许愿的事**一定会实现**，但事后除她本人外、所有相关者都要付出「极度惨痛的代价」。

正因如此，鬼族早年把她当「裹着柔美皮囊的怪物」丢进黑森林、后来流落黑暗之域（大概是无处可去、或作女王留下的眼线/监视者）。轮回乐园的提示早已隐喻「整个黑暗之域没有一个好人」——苏晓深知这种「一步登天」的许愿背后代价难测，只借用她的「看到」、绝不许愿。

## 关系

- [女王·尤罗](女王·尤罗.md)：她的孪生妹妹、北大陆最强者（被苏晓斩杀·她闻讯泣不成声）。
- 深渊：她与女王「字面意义」的父亲（母亲走进黑暗受孕产下姐妹）。
- [苏晓](../苏晓.md)：用甜品换她「看到」天赋唤醒装置位置的灭法者（她只借看到、不被许愿）。
