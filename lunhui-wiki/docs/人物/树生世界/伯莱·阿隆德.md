# 伯莱·阿隆德

[树生世界](../../世界/任务世界/树生世界.md) 精灵族历史上的**首位南境之王**——一生传奇:幼时丧父、与母亲姐姐相依，年少无为、壮年落魄，直到快到老年才异军突起、以不可挡之势收服几大氏族、加冕为王。他著有《天赋之苏醒》一书（[蘑菇先知](../魔海/蘑菇先知.md) 以此暗示 [苏晓](../苏晓.md) 他的发迹之秘）。

## 逆世自冕 · 打退暗灵

树生世界有个古老传统:名望实力不够就加冕，会被黑森林的**暗灵**找上门戮杀。[女王·尤罗](女王·尤罗.md) 加冕时暗灵到场即跪伏承认；而伯莱·阿隆德「**没有王之姿、却有王的野望**」——他自身不算强，靠一群实力相近的心腹，在加冕后**多次打退来戮杀他的暗灵**。他有句名言:「我有没有资格加冕为王，暗灵们没资格决定，只有我的臣民们有资格决定。」他把曾经只能缩在极南「大遗迹」的精灵族带向繁荣、雄踞南大陆一半。他中老年异军突起的真正原因，大概率是发现并激活了**[天赋唤醒装置](../../力量体系/装备道具.md)**（精灵族用灵魂石启动·催化深渊之力再吸收）——[苏晓](../苏晓.md) 怀疑，正是他开了「用天赋唤醒装置催化深渊之力」的头，埋下了后世精灵族血脉畸变、衰老症的祸根。

## 关系

- [女王·尤罗](女王·尤罗.md)：与他并列、同受敬重的北境天命女王（一个逆世自冕、一个应冠而临）。
- [精灵王·克伦威](精灵王·克伦威.md)：他开创的淘汰制王位下、首个被暗灵完全承认的继任精灵王。
- [苏晓](../苏晓.md)：怀疑他开了「催化深渊之力」祸根、循着他的《天赋之苏醒》找天赋唤醒装置的灭法者。
