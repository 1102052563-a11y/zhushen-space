# 水生之母

[树生世界](../../世界/任务世界/树生世界.md) 黑暗之域「黎明镇」的六张画像之一所记录者——[苏晓](../苏晓.md) 从 [咕噜](../咕噜.md) 手里得来的女王六画中的最后一幅画着它:一处晦暗潮湿的海边渔村，半人半鱼的渔民跪伏在地，前方海水中半浮出一只**似章鱼、身体半透明、体内遍布荧蓝色神经组织**的巨大海洋生物。画角标注三个字——「**已逃脱**」。

## 唯一逃出黑暗之域者

黑暗之域是女王亲自看管出入口的流放地，六幅画原是黑暗住民 / 管理者（[传光人·安德森](传光人·安德森.md)、[艾莉亚](艾莉亚.md)、猪兄、模仿男、[伯莱·阿隆德](伯莱·阿隆德.md)、水生之母）。唯独水生之母被标注「已逃脱」——它成了逃出黑暗之域、潜伏于海洋、受半人半鱼渔民供奉的存在（第 36 章标题「水生之母」预示它将正式登场）。苏晓打算从这些画中人（猪兄、模仿男、老精灵王、水生之母）身上获取好处、或利用它们对付灰绅士。

## 关系

- [苏晓](../苏晓.md)：得到它的画像、盘算利用它对付灰绅士的猎杀者。
- [女王·尤罗](女王·尤罗.md)：曾看管它所在黑暗之域出入口的北大陆女王（六画藏于女王遗物）。

## 🔥 浊血症与「水淤之血」的真凶（46卷29章）

水生之母正式登场的前史揭开了精灵族灭亡的最深一环。老精灵王 [伯莱·阿隆德](伯莱·阿隆德.md) 用天赋唤醒装置高度活化深渊之力（称「源水」）、争夺领土时全族滥饮，代价是肤色变浅蓝、瞳变幽蓝——这不是进化而是畸变，为「浊血症」埋下祸根。15 年前浊血症全面爆发，被囚黑暗之域的水生之母（有多重神性·邪恶与中立参半）恰在此时**逃出（重伤濒死）**，因渔村人救过它，它让渔民饮其**水特性的血**「化解」浊血症——实则是**把浊血症的上限调高**（看似治愈、实则日后爆发更烈更无解）。[精灵王·克伦威](精灵王·克伦威.md) 觊觎精灵族的信仰之力 + 大遗迹环境，率 16 人远征队以「护送」为名把上岸的水生之母**围攻生擒**、肃清渔村所有信徒（以邪教罪处死）。此后全精灵族饮下含水生之母血肉的药汤——把原本无方向的血脉畸变**导向了「水淤之血」**（深渊 + 海洋 + 水沁 + 衰老特性·树生世界最恐怖的异常状态），[阿尔勒](巡查队长·阿尔勒.md) 畸变成鱼人怪正源于此。[深渊之罐](../../力量体系/装备道具/index.md) 到贝城更催化了浊血症的总爆发。

- [伯莱·阿隆德](伯莱·阿隆德.md)：开了「源水」滥饮之头、种下浊血症祸根的初代精灵王。

## 🔥 一部「史上最倒霉霸主生物」的血泪史（46卷39章）

水生之母正式登场揭出它憋屈的全程:它原是某世界的霸主，耗尽所在世界 90% 以上资源「飞升」到树生世界、嚣张到「除虚古神外一概不虚」。可它的噩梦接踵而至——先被上一代「[黑暗之域](../../世界/任务世界/树生世界.md)看守」收买 [屠夫·巨罗](猪兄·巨罗.md)、[无面人·佩特·佩伯](无面人·佩特·佩伯.md)、[安德森](传光人·安德森.md) 三人组队收拾、「自愿」成黑暗住民；筹备多年越狱成功，一出大树洞寝殿就撞见 [女王·尤罗](女王·尤罗.md)、差点被双刀砍成「章鱼寿司」；逃亡途中又撞 [蘑菇骑士](蘑菇骑士.md) 差点被大剑怼死；好不容易逃到渔村的海里（正因被打出阴影、才对渔民友善、怕再冒出隐士强者）。结果贝城远征队又来、被 [精灵王·克伦威](精灵王·克伦威.md) **分成六块马车运回、囚在大遗迹定期割肉**。本世界金字塔尖的强者（蘑菇骑士 / 精灵王 / 鬼族女王 / 巨罗 / 无面人 / 安德森）它可以说「看到没、这些人全揍过我」。它硬苟到精灵族灭绝、终于崛起成大遗迹与贝城的主宰——

## 被 boss 队「放烟花」（46卷39章）

——却撞上 [苏晓](../苏晓.md) 的 boss 队找上门。为防它狗急跳墙毁掉天赋唤醒装置，苏晓先派 [凯撒](../尼古拉斯·凯撒.md) + [艾朵儿](艾朵儿·帕帕.md)（魅力 170）当外交安抚（凯撒借「脱鞋倒石粒」放毒气开团）。随后五 boss 齐上:所罗门黑绳捆缚、伍德幽绿焰、[尤尔](宿命之子·尤尔.md) 蓄力箭洞穿、[罪亚斯](../画中世界/罪亚斯.md)「滋生噬养」让它自噬血肉、苏晓一脚踹飞它臃肿的头部；最后苏晓把普通阿波罗塞进它口中、血枪连推炸上高空、太阳焰把它烧成灰烬——「水生之母被放了烟花」，比蜗牛哥（惨死多年的本世界最惨 boss）死得还惨。它临死那被围殴成皮球的模样，连艾朵儿都触景生情觉得它「莫名可怜」。掉落强者徽章 + 水生之物（不朽级）。
