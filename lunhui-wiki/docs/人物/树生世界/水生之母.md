# 水生之母

[树生世界](../../世界/任务世界/树生世界.md) 黑暗之域「黎明镇」的六张画像之一所记录者——[苏晓](../苏晓.md) 从 [咕噜](../咕噜.md) 手里得来的女王六画中的最后一幅画着它:一处晦暗潮湿的海边渔村，半人半鱼的渔民跪伏在地，前方海水中半浮出一只**似章鱼、身体半透明、体内遍布荧蓝色神经组织**的巨大海洋生物。画角标注三个字——「**已逃脱**」。

## 唯一逃出黑暗之域者

黑暗之域是女王亲自看管出入口的流放地，六幅画原是黑暗住民 / 管理者（[传光人·安德森](传光人·安德森.md)、[艾莉亚](艾莉亚.md)、猪兄、模仿男、[伯莱·阿隆德](伯莱·阿隆德.md)、水生之母）。唯独水生之母被标注「已逃脱」——它成了逃出黑暗之域、潜伏于海洋、受半人半鱼渔民供奉的存在（第 36 章标题「水生之母」预示它将正式登场）。苏晓打算从这些画中人（猪兄、模仿男、老精灵王、水生之母）身上获取好处、或利用它们对付灰绅士。

## 关系

- [苏晓](../苏晓.md)：得到它的画像、盘算利用它对付灰绅士的猎杀者。
- [女王·尤罗](女王·尤罗.md)：曾看管它所在黑暗之域出入口的北大陆女王（六画藏于女王遗物）。

## 🔥 浊血症与「水淤之血」的真凶（46卷29章）

水生之母正式登场的前史揭开了精灵族灭亡的最深一环。老精灵王 [伯莱·阿隆德](伯莱·阿隆德.md) 用天赋唤醒装置高度活化深渊之力（称「源水」）、争夺领土时全族滥饮，代价是肤色变浅蓝、瞳变幽蓝——这不是进化而是畸变，为「浊血症」埋下祸根。15 年前浊血症全面爆发，被囚黑暗之域的水生之母（有多重神性·邪恶与中立参半）恰在此时**逃出（重伤濒死）**，因渔村人救过它，它让渔民饮其**水特性的血**「化解」浊血症——实则是**把浊血症的上限调高**（看似治愈、实则日后爆发更烈更无解）。[精灵王·克伦威](精灵王·克伦威.md) 觊觎精灵族的信仰之力 + 大遗迹环境，率 16 人远征队以「护送」为名把上岸的水生之母**围攻生擒**、肃清渔村所有信徒（以邪教罪处死）。此后全精灵族饮下含水生之母血肉的药汤——把原本无方向的血脉畸变**导向了「水淤之血」**（深渊 + 海洋 + 水沁 + 衰老特性·树生世界最恐怖的异常状态），[阿尔勒](巡查队长·阿尔勒.md) 畸变成鱼人怪正源于此。[深渊之罐](../../力量体系/装备道具.md) 到贝城更催化了浊血症的总爆发。

## 关系（补充）

- [精灵王·克伦威](精灵王·克伦威.md)：15 年前率远征队围攻生擒它、肃清其信徒的精灵王。
- [伯莱·阿隆德](伯莱·阿隆德.md)：开了「源水」滥饮之头、种下浊血症祸根的初代精灵王。
