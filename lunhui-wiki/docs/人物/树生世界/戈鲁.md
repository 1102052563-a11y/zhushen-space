# 背叛者·戈鲁

[树生世界](../../世界/任务世界/树生世界.md) 寒冷墓地「地下聚地·斯易」里那名把任务托付给 [苏晓](../苏晓.md) 的**老鬼族**——他其实是 [女王·尤罗](女王·尤罗.md) 的**养父**，也是当年**背叛女王的主谋**。9 名老一辈鬼族觊觎女王从大树洞带回的王冠（误以为是能带来力量的「金子」），其中 3 人试探被女王连夜灭门；戈鲁遂假意「支持女王做抉择」、边吃晚餐边劝、餐后在女王最爱的【[血馨醇酒](../../力量体系/装备道具.md)】里下了针对灵魂的剧毒，联合鬼族强者斩下女王双腿、逼她重伤遁走。

## 把苏晓当刀使

女王走后鬼族抢不到王冠、寒雾弥散、又被蘑菇部族仇杀，600 万人口暴跌到 30 万、濒临灭族。垂垂老矣的戈鲁便**处心积虑地引导苏晓去「带回女王或王冠」**——他刻意把女王贬低成「只是鬼族女王、地位不高、冲动贪权不愿担代价」（全是谎言），预支给苏晓的一整套「白嫖」奖励其实各有用途:古老地图指路、[鬼族女王之血](../../力量体系/装备道具.md)追踪、先王冰魂换黑暗石、古语言载记开封眠之门、而那瓶【血馨醇酒】——他就差明说「往里下毒」（女王喜欢喝）。苏晓一路看穿了他的别有用心（门上脸孔道破真相后，支线「抉择」变更为隐藏任务「刺毒之痛」）。

## 关系

- [女王·尤罗](女王·尤罗.md)：他养大、又亲手下毒斩腿背叛的养女（北大陆最强者）。
- [苏晓](../苏晓.md)：被他用半真半假的情报 + 预支奖励当刀使、去对付女王的猎杀者。
