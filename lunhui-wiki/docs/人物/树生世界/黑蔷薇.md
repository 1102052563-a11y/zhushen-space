# 黑蔷薇

[树生世界](../../世界/任务世界/树生世界.md) 里一名倒霉的女参战者——因在世界联络平台上喷人，被**屏蔽出古精灵语**，每天只能发言 2~3 句。她在东海漂流时偶遇同样迷路的 [贝妮](../布布汪.md)、阿姆（[苏晓](../苏晓.md) 那两只走散了五六天没来会合的随从），三者同病相怜（「不只是我自己倒霉」的感觉鼓舞人心）。

## 方向游反了

她靠着一面挡路的雾墙、支付灵魂钱币向虚空之树咨询方位，结果查出这里是**极东**——他们游了好几天，方向竟然完全游反了（不是朝陆地、而是朝大海畅游到了大陆最东端的雾墙前）。这消息让阿姆当场自闭、贝妮飙泪、黑蔷薇「心态裂开」。

## 关系

- [贝妮](../布布汪.md) / 阿姆：海上偶遇、同病相怜、一起游反方向到极东的苏晓随从。
