# 小迷糊

一名**契约者**——看上去**约十二三岁的小女孩**（黑发、一口整齐小白牙、笑起来双眼弯成小月牙），自我介绍「**我叫小迷糊，辅助系奶妈，特长是跑得快**」。是 [阳之女](阳之女.md) 强逼进黑色森林的两名契约者之一（脖颈攀着一根细藤蔓作枷锁）。

- **定位**：**辅助系奶妈·增益强过奶量**（给队友叠一堆状态）；自保能力弱（被两百头狮群追得上气不接下气、靠 [无伞兄](无伞兄.md) 护着逃命）。
- **增益能力（八卷61–62章）**：**四种属性（力 / 敏 / 体 / 智）皆可增幅、每次可选其中两种**（可见能力不弱·苏晓四属性均衡发展故增哪两种都行·他选**力 + 智**〔智增法力·因其能力极耗法力〕）；另有**加速光环**等多种状态（圣洁 / 雨之回响 / 力量泉 / 永生 / 急速……一口气叠一串）。**护盾 / 防御类与苏晓的 [三笠心意](../力量体系/装备道具.md) 排斥**（不能套盾），但属性增幅与加速兼容。她会拿小本本记录队友的兼容情况、定制辅助方案——价值很快体现：苏晓全靠她的增益才能在极短时间内秒杀想炸装甲车的契约者。
- **性子**：**没什么心机、嘴贫话密**（一路和无伞兄斗嘴）；现实世界**家境不错**（张口就是「我经常开家里的跑车去飙车」），结果上手 [装甲车](../世界/任务世界/T12K星球-生存试炼.md) 一通乱挂挡、险些「灵车漂移」，被追问才招「跑跑卡丁车算不算」，当场被换下驾驶位。苏晓评她「**最好处理，或许也最不好处理**」——她无恶意、一心只想摆脱阳之女的束缚、之后有多远逃多远。

## 「被选」当队长（八卷60–61章）
四人小队（苏晓 + 老巴尔 + 无伞兄 + 她，外加布布汪）集结后议推队长——无伞兄毛遂自荐，[苏晓](苏晓.md) 与 [巴尔](巴尔.md) 却几乎同时点名「**小迷糊**」。她皱鼻嗅到一股「阴谋的味道」却推不掉（实为两只「老狼」把这只「小绵羊」推上去当探路 / 挡灾的名义首脑）。布布汪都为这可爱妹子默哀「死掉可惜了」。

## 黑色森林 · 被「队友」抛下（八卷65章）
进入黑色森林后，[苏晓](苏晓.md) 的真目标（隐藏任务第三阶段「[暗王](暗王.md)」）与 [阳之女](阳之女.md) 安排的「骷髅洞穴」不同——苏晓随 [野猪](野猪.md) 径自去寻暗王，把抱紧的「大粗腿」无情甩开。小迷糊一时没反应过来「我们不是一伙的吗」，被 [无伞兄](无伞兄.md) 一句「**从来都不是**」点醒。她神情低落、担心三人（她 + 无伞兄 + [巴尔](巴尔.md)）能否独力完成——无伞兄宽慰「依靠别人能活一时、不能活一世」，巴尔也亮出底气（力量已适应、本可与苏晓拼命）。
