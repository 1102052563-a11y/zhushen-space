# 启迪（死灵族 · 百年一出的引路者）

[死灵族](../世界/生物万族/index.md) 的「**启迪**」——一名须发皆白、颚下白须长达三米拖地、**没有瞳孔**（异于普通死灵的绿瞳）的死灵老者，住在死灵帝国首都暗月沼泽「庇护之树」主干里的小树屋（连**死灵王·森迈尔**都没资格住那）。「启迪」**百年才出一位**，是死灵族世代最崇敬的人、「黑夜中指路的萤火」：不干涉军权 / 财政，**唯一的使命是让死灵族延续下去**——若没有历代启迪，死灵族「早在千年前就已消亡」。

## 死于轮回乐园 → 全族追杀苏晓（十七卷73章）

[苏晓](苏晓.md) 在神之墙激活冒险团晋升考核、被烙下「死灵定位印记」时，**轮回乐园弄死了这位启迪**（让它临终看到「某个很恐怖的景象」）——这正是死灵族倾力追杀苏晓的根源（并非乐园直接操控死灵、而是借启迪之死驱动）。启迪临死口鼻溢血、留下遗命：

> 「**铁之手·白夜，找到他、杀掉他，只有这样我们才有未来。**」「**挖出我的心，我的心会指引你们。**」

它心脏上的印记与苏晓手背的「死灵定位印记」相似度 **>90%**——死灵王森迈尔捧着它滴血的心脏，当众宣布**停止与人族的战争、举族追杀苏晓、用他的血祭奠启迪**。成千上万死灵为启迪之死痛哭。启迪身边还有一名实力超过 [敦克·尼克](敦克·尼克.md) 的赤膊死灵（常与敦克·尼克切磋、至敦克死都没赢过一次）。

## 关系
- **死灵王·森迈尔**：死灵帝国之主（捧启迪之心、下令全族追杀苏晓）。
- **[苏晓](苏晓.md)（铁之手 / 白夜）**：被它临终遗命指定为「杀掉才有未来」的目标（实为轮回乐园借其死制造的考核）。
- **[敦克·尼克](敦克·尼克.md)**：它身边赤膊死灵的切磋对象（赤膊死灵更强）。
