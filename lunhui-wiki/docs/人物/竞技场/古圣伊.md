# 古圣伊 / 古圣一（原首位 · 「海神」· 轮回乐园猎杀者）

[苏晓](../苏晓.md) 登顶前的**五阶乐园竞技场原首位**——马尾辫、休闲装、叼吸管的随性女人，自称「宇智波·古圣伊」（竞技场名「古圣一」是拼音打错·她也是**轮回乐园的猎杀者**·与苏晓互认同类）。号称「**海神**」、几乎不可战胜：

- **蔚蓝之海（Lv.67）**：把整个竞技场封成密闭海水（敌人每秒受伤 + 体力消耗剧增）；用此能力后海水中藏一具**分身**，须找出并击杀才能破除——但攻击分身会被海水反噬、攻击者随机部位**金属化**（随机到大脑即败）。
- **金属界 / 铁锈化**：召半金属化海洋生物；让敌人身体逐步金属化生锈（无判定·关节锈死失去知觉·竞技场外近乎不可逆）。

苏晓是**第一个在竞技场伤到她的人**——靠刃道刀·万鬼逼出分身、[放逐](../../力量体系/装备道具/index.md) 刺穿分身破蔚蓝之海，忍受全身 60% 金属化，缠斗 36 分钟险胜登顶（她宁挨刀也不弃权·猎杀者本色）。

## 关系
