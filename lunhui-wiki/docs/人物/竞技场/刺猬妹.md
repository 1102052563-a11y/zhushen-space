# 刺猬妹（死亡乐园 · 灵魂系控制 · 稀有辅助）

束发瘦小女孩、精通物理 / 精神 / 灵魂三系控制的稀有辅助系（曾打到第 12 名·战力极强）。

- 两次撞 [苏晓](../苏晓.md)——灵魂系外的控制对苏晓全无效，连最稀有的**灵魂冲击在苏晓的高灵魂强度前也毫无反应**（连她自己都怀疑能力失灵），直接弃权并把苏晓列入拒战名单。
- 后用「实力差距多绝望」劝退后来挑战的死亡乐园新秀**枫叶**。

## 关系

- [苏晓](../苏晓.md)：对其能力全免疫、被她拒战的克星。

## 出处

- 二十八卷42–45章（[五阶乐园竞技场](../../总览/轮回乐园机制.md)）：两撞苏晓失效→弃权拒战。
