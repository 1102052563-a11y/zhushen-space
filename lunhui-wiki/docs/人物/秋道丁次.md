# 秋道丁次

[火影忍者世界](../世界/任务世界/火影忍者.md) 木叶忍者、秋道一族——身材微胖，**倍化之术**（身体膨胀成巨人）使用者，「**猪鹿蝶**」组合之一（与 [奈良鹿丸](奈良鹿丸.md)、[山中井野](山中井野.md)）。其父**秋道丁座**（老一辈猪鹿蝶·持巨型铁棍）随后赶到。

- **围攻苏晓（九卷67–68章）**：以倍化巨人一脚踩 [苏晓](苏晓.md)，被苏晓 200 法力能量盾扛住；其手臂被苏晓刀芒切开见骨。父亲丁座抡巨型铁棍猛砸苏晓的能量盾，却被苏晓刀芒**斩瞎一只眼**。猪鹿蝶两代合力仍留不下苏晓。

## 出处
- 九卷67–68章。
