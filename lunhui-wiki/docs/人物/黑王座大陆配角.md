# 黑王座大陆配角（肯·拉罕 · 佩黛菈 · 布卢默 · 沙耶托 · 帝国群像）

[黑王座大陆·死寂城](../世界/任务世界/黑王座大陆-死寂城.md) 中安多哈帝国的夺权者与 [苏晓](苏晓.md) 阵营人物，集中收录。

## 肯·拉罕（老神棍 · 黑之王右御 · 苏晓引路人）

[黑之王](../世界/任务世界/黑王座大陆-死寂城.md) 曾经的**右御「王怒·肯·拉罕」**——黑之王愤怒时第一个扑向敌人的人、帝国权贵闻之色变的大人物。如今是看守小教堂的「老神棍」：满口「光之神」「宽恕」、却是**最纯粹的暗系超凡者**（泯灭 + 强吸附力的暗能量·抽空敌人生命力再注回体内变成「信徒」行尸走肉）——杀人比苏晓还狠还多，嘴上忏悔手上不停。有些老年痴呆般的絮叨（吸黑色能量后能保清醒约 40 个月轮）。是轮回乐园给苏晓安排的**引路人 / 身份背书**（战力与苏晓五五开·可见违规者之难缠）。他的真正执念 = **镇压死寂城的「居民」**（黑之王留下的镇压之物快维持不住·他不在乎谁登王位、只想尽快有人坐上去镇住死寂城）；忠于黑之王。月王后被囚一事令他暴怒。

## 晨曦公主·佩黛菈（苏晓的傀儡 · 15 岁）

15 岁的晨曦公主——浅红长发、善良天真、软萌如兔（家人早死光·只剩 4 名忠心女仆·因对各方威胁小才没被弄死·却仍被刺杀过三次）。苏晓 + 肯·拉罕逼她当**王位继承傀儡**（她主要是为救 4 名女仆才同意），全程被精神疏导到「我是谁我在哪」。其软萌性格据说是被王女沙耶托送军营操练吓出来的。与裁决者 [Saber](Saber.md) 意外成了吃货好友。**死亡（二十六卷28–29章）**：被违规者**狄鸦**（见下）选作意识寄存的替身——狄鸦借「佩黛菈」身份在餐桌捅苏晓一刀露馅（佩黛菈从不敢发问、苏晓由此识破），随即**粗暴拔出寄存的意识箭、把佩黛菈的弱小灵魂扯碎致死**（临终对赶来的 Saber 说「能遇到你真是太好了」）。她其实并非纯小白兔、对王位「欲拒还迎」（苏晓点破）——这也成了 Saber 与苏晓决裂的导火索。

## 布卢默（元老首 · 老狐狸 · 囚月王后）

16 元老之首、其余 15 名元老都支持他（堪称奇迹）——鹰钩鼻、瘦削、最有权势的元老，王权四夺权者之一。极擅政斗：苏晓闯门时早布好弃子军官**艾德里安**的局（事败后艾德里安「畏罪自杀」灭口）、被反将后捏鼻交出 54 万军权（仍叮嘱「西陲塞 24 万守军绝不能调·否则野蛮人破境」·可见没被权欲腐空脑子）。**命门 = 在自家地下囚着「假死」的月王后**（黑之王最年轻的王后·肯·拉罕亲自参加过她的「葬礼」·实则被布卢默假死囚于地下百米的玻璃水箱、插着导管抽取·她知黑之王诸多秘密·苏晓夺权后逼布卢默交人）。当晚即派 14 名刺客夜袭苏晓（混杂得反常）、又把苏晓推上帝国议会想「捧杀」（让四夺权者合围他）。

## 王女·沙耶托 / 银羽公爵 / 小公爵（其余三夺权者）

- **王女·沙耶托（→ 新王）**：掌**圣城护卫军**、红眸（看人先看喉咙 / 心脏等要害）、「比男人更像男人」的女人（18 岁亲手宰过追求者·暗称「圣女·沙耶托」）——佩黛菈的童年阴影、帝国最强王族 / 最有力夺权者。恨苏晓却在权斗上奈何不了他。**成「夜之王」（二十六卷31–34章·已被同化）**：违规者清掉、傀儡佩黛菈已死、苏晓自己不当王（成王要被死寂城绑死），把王位「随手抛给」她——她先觉受辱、又被苏晓「站人族大义」的姿态压住应下（**不知成王代价**）。坐上黑王座的瞬间**被死寂城同化**、成了失去感情的人偶「**夜之王·阿德格什·沙耶托**」（与死寂城低语「约定之日三百年后」）——这就是黑之王曾说的「成王的代价」。她是继黑之王·阿德格什、昼之王·厄德尔斯之后的**第三位王**，无法治国、须左右御（伍弗 / 沃克利）辅佐。
- **银羽公爵（殁）**：约 30 岁、面带笑容的「贵族绅士」，唯一缺点是好色 + **与亲姐玛姬·维尔莉通奸**（一夜八女·「铁肾公爵」·玛姬已有孕、事后被其势力送出圣城）。被苏晓当庭揭丑（黑晶播放通奸影像 + 验出玛姬有孕）搅局，随即被苏晓用 [恶魔之罐](../世界/生物万族/index.md) 短控的心腹仆从**递淬毒匕首当庭刺杀**（嫁祸仆从·物理除掉一个夺权者）。
- **布卢默（殁·补）**：元老首（详见上「布卢默」条）——太活跃、连派 14 刺客 + 空间陷阱伏击苏晓，被苏晓**直接暗杀**（布布汪传送 + 一刀无声·世界之源 +7.6%·「掀桌子」式）。
- **小公爵（退出）**：9 岁、老气横秋却早熟（看通奸影像看入迷·偷偷和小女仆「实验」被财政大臣揪出打屁股），父亲曾是最有力夺权者、被暗杀于自家卧室——靠财政大臣 **沃克利**（见下）撑着。亲眼见苏晓两天弄死银羽 + 布卢默后**怕极了**、主动退出夺权拥护佩黛菈（「大人的世界好可怕」）。

## 沃克利（财政大臣 · 帝国财神爷）

小公爵已故父亲的遗部、忠心老官员——白发消瘦、老绅士风度、帝国财政命脉（小公爵能坐夺权席全靠他）。看淡权利（「权利解决不了我每晚起夜三次的问题」）、却绝不简单（苏晓判断他「有问题、但与违规者无关·违规者很可能就是被他赶出公爵府」）。小公爵退出后他主动拥护佩黛菈、给苏晓送十几箱金银 + 提前拨军费换「不动小公爵」。

## 巴哈 / 布嘉恩·费迪（人造生物刺客 · 殁）

回程空间陷阱伏击苏晓的西境刺客——**布嘉恩·费迪**（满脸老人斑的干瘦老头·人造生物制造者·本体不强 + 不能远程操控造物）造出的两件人造生物：**巴哈**（融在小肉山后颈的瘦侏儒·癫狂豪迈·头被刺穿仍能再生）+ **小肉山**（缠铁链 / 持千斤大砍刀的生物坦克·爆体成「血肉礼堂」九具肉山绞肉风暴 + 剧毒血）。苏晓两次试探确认侏儒非本体、绕杀老头布嘉恩、得人造生物核心 [血肉礼堂·巴哈](../力量体系/装备道具.md)（侏儒临死坦言「为创造者战斗十几年已报恩」）。

## 狄鸦（违规者 9744 号 · 天启乐园 · 殁）

本世界的**违规者 9744 号**——来自天启乐园（故能杀战斗天使）、中性化样貌 + 宝石耳钉。两大核心能力：**女武神**（红色晶体武神甲 + 水晶枪 / 双刀·属性 / 装备其实比苏晓强·已四个世界没亲手战斗）+ **意识箭**（把自身部分意识做成带倒刺的箭、射入他人灵魂寄存替代·非完全替换、故能规避追猎·拔出时灵魂受重创）。极厌恶轮回乐园（「从天堂天启跳到地狱·全是疯子杀手」）、唯一目标是夺世界之核脱离轮回乐园。替代过莱娜 → 佩黛菈、欲等苏晓得世界之核后替代其身边老头夺核。被苏晓识破后拔意识害死佩黛菈、本体被布布汪 + 8000 士兵定位、调 1.5 万攻城部队**车轮围殴 3 小时耗死**（武神甲虽硬也扛不住·临终骂「轮回乐园去死」）·被苏晓一刀斩首。

## 王后·梅瑞狄斯（蜘蛛霸主级 · 守王宫前殿）

详见 [生物万族 · 死寂城 / 王宫居民](../世界/生物万族/index.md)——五米高、蛛腹长半个女人（紫发）的**霸主级生物**，守王宫前殿（连苏晓都打不过）。布布汪偷世界之核时用「国足老二吼声喇叭」把她吓蹦起来、趁机溜过。

## 伍弗 / 十号 / 灰老鼠 / 佐斯（苏晓收的军方 / 打手）

- **统帅·伍弗（→ 帝国右御）**：敦实矮壮、管 56 万边塞军（实际军费被元老院贪到只剩一成）——苏晓砸断他一指立威、再全额拨军费收心，他当即死心塌地（利益共同体·只要连得三年全额军费就能把野蛮人 / 海族打趴）、率 1200 精锐入驻护卫。沙耶托封王后**任帝国右御**（红甲·掌军权）——沙耶托曾许他统帅 + 拉拢，他收了好处却直接送苏晓这边，仍铁了心跟苏晓（与沃克利一同辅佐被同化的夜之王治国）。
- **沃克利（→ 帝国左御·补）**：财政大臣（详见上「沃克利」条）——沙耶托封王后**任帝国左御**（白衣·掌财政 / 民生 / 官员选拔）。昨晚押对宝支持苏晓（「不信两天弄死银羽 + 布卢默的人会拱手让王位」）。
- **佐斯（假意背叛）**：黑发黑纹身叛逆少年异族（详见下）——**奉苏晓之命假意倒戈投沙耶托**（沙耶托给他介绍议政官之女丝娜联姻·他将计就计白占便宜），新王封临时在前殿配合苏晓清场（边防部队哗变冲王宫·里应外合）。
- **十号**：帝国用超凡者尸体造的**半机械生物坦克**（失语 / 无痛觉·含甲 1165 斤·蛮力撞穿 80cm 装甲板）——伍弗献的杀手锏。
- **灰老鼠**：伍弗的微胖远亲、原斥候、熟悉圣城（「最可信」的情报头子）。
- **佐斯**：黑发黑纹身的叛逆少年异族、暗系超凡者——本人战力弱但能**操控动物类仆从**（黑雾化秃鹫等·苏晓吐槽「佐斯」念快了像「作死」）。

## 维罗妮（野生超凡者 · 进死寂城的「活体消音器」）

苏晓花 500 万黑石雇的野生超凡者——21 岁、亚麻色短发、松垮垮裤子（总像要掉）。能力偏感知 / 阻断：**屏蔽周围 300 米内的声音 / 感知 / 震动 / 光感 / 大部分能量**（进死寂城的关键·避免战斗惊动全城居民）+ 强感知力；武器是三根超凡生物丝（线轮·韧 / 利胜超凡蜘蛛丝）。佩**古精灵的野性**腰带（降寿命流逝速度·副作用 = 裤子越紧越易掉、不可物理避免·入狱 12 次的奇葩记录·最坏不过「绑架案」）。被苏晓 / 老神棍演双簧「处理掉」唬住、又被 500 万黑石砸成复读机（贫穷限制了她的想象力）。一路被老神棍讲鬼故事吓、进死寂城后从抗拒到死心配合（清楚只有配合苏晓才能活）。
