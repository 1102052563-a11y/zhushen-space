# 光·暗世界群像（议会院 · 秘术师 · 鬣狼族）

[光·暗世界](../世界/任务世界/光暗世界-宿命之路.md) 中 [苏晓](苏晓.md)（炼金大师·库库林·白夜身份）遭遇的各方人物。

## 议会院（权力中心 · 安戈城）

- **安戈城**：大陆中心、议会院所在的贸易中枢——四王国货物都运到此打议会院印章再卖（化解四国民众世仇导致的贸易抵制）；表面繁荣、实则小偷泛滥（四王国故意遏制其发展防失衡）。
- **圣境老议员（普里之父 · 179 岁 · 三实权议员之一 · 病危）**：奥卡兹大帝时代沿袭至今的八议员之一（议员须弃本名、用勋徽「圣境」）——**史上猎杀鬣狼族最多者**（鬣狼高层都盼他早死）。现被幕后之人用「微型超凡生物」侵体吸生命力所害（几周内从 130 磅瘦到 60 磅·非病）——苏晓接手救治。
- **普里（代理议员 · 圣境老议员之子 · 32 岁）**：极不正经（憋着尿装死迎客、爱坐蒸汽火车游历四王国、女伴不断却风流不下流）、正事却靠谱——请苏晓救父，承诺「什么要求都行除了娶我妹妹」（19 岁妹妹是他比当议员更自豪的事）。老议员病危后接掌其权柄。
- **阿撒兹（老议员心腹 · 瘸腿老者）**：拄手杖、右腿旧伤、病恹恹——代议会院迎苏晓（顺便想拿公款挥霍被豪华餐厅挡门）；忠于老议员、养着十几只看家的马俄犬；实力深浅难测。
- **霜蛇议员 / 蔷薇议员（另两位实权议员）**：**霜蛇**（冷酷·当年力排众议推蒸汽技术·蒸汽产物都印其勋徽·疑掌控奇多尔曼公司）；**蔷薇**（女·年轻貌美·最激进·19 年前雷曼王国想独立时她集三国之力打到雷曼首都、按雷曼王的脸盖血章停战）。其余 5 名议员是凑数的（合起来才勉强算一票）。

## 秘术师 / 王国 / 公司

- **南希·埃尔娜（雷曼王国三等兵 · 迷糊却能打）**：蓝紫发马尾少女、背近 2 米高压蒸汽狙击枪——奉命秘密护送苏晓却把车票弄丢（只买到后车厢）；爱误判（把甩锅成功而笑的奇多尔曼秘术师当坏人击毙）但战力强、会顺势邀功交差。
- **奇多尔曼公司**：蒸汽火车 / 蒸汽车的制造商（标志黎明鸟）——忠于霜蛇议员、财力雄厚、深受民众欢迎；雇佣洗脱罪名的前暴徒当打手，蒸汽火车遇袭时反应最快（逼退野兽绅士图亚克）。
- **秘术师两系**：**秘纹系**（世袭家族传承秘纹·越传越强）/ **烙刻系**（秘纹刻身·不可传承·多变不稳定）；需「自由灵魂」（王权束缚会使秘纹排斥灵魂）。

## 鬣狼族（夜行猎手 · 专猎高层）

- **卡律（鬣狼 · 月光大人麾下 · 殁）**：黑礼服银耳环、彬彬有礼却野兽气息的绅士——奉「月光大人」之命劫苏晓（金属爪 / 嚼子弹吐射），被苏晓一刀斩首。
- **野兽绅士·图亚克（鬣狼通缉犯）**：上身人狼 / 下身人类的强壮鬣狼——第二波来灭口 / 劫苏晓，感知到血之兽危机 + 奇多尔曼公司赶到而撤（苏晓本想被他劫走再下手，惜未成）。
- **鬣狼族（族群）**：人 / 兽可切换的夜行族，以家族为纽带（每族数十上百脉·可为劳工 / 官员、召开家族会议即夜间猎手）——能与人婚配生纯血后裔；平均战力约七阶，专猎四王国 + 议会院高层（尤恨圣境老议员）。某「月光大人」与议会院幕后之人勾结。

## 埃尔维斯·格鲁（黑袍人 · 幕后大反派 · 王国毁灭者）

脸生黑色触手（自拔来吃）的黑袍男——本世界的幕后大敌，「忍」功登峰造极的老阴哔。

- **身世**：边塞外不属任何王国的小村落出身。17 岁发现「**秘纹 + 蒸汽技术的结合会带来绝对毁灭**」（上一代文明已为此覆灭）——向风暴王国官员举报却被当「讨食的癞皮狗」打发 15 金镑、第三次举报直接被安「盗窃」罪判 15 年，逃亡东泽王国当到二等监巡官后又神秘失踪，最终加入一个成立千年的势力身居高位。信念 = 就算变恶鬼也要收回所有秘纹、阻止秘纹与蒸汽结合（屠戮生灵皆为恶·他从不为自己辩解）。
- **手段**：借**厄运镇古神之力**——用石板裂空间放「黑色光球」笼罩四王国，剥离秘术师的秘纹（力量本就来自秘纹）、把体质弱者化成「[木眷族](../世界/生物万族/index.md)」干枯怪物；**三日内灭东泽 / 月溪 / 风暴三王国**。与**鬣狼族**合作（安戈城灭后大陆归鬣狼、建不碰秘纹的新文明）。全程隐于暗处、计划被苏晓识破就灭口所有棋子撤离，是苏晓遇过「最能忍」的老阴哔。被苏晓破坏关键一环后狗急跳墙，用「送 120 万老幼妇孺 / 派怪物谈判挑拨」等阳谋反制。

## 内奸

- **女仆·瑞丝 / 骨蜘蛛·卡罗（炼金师 · 潜伏 24 年 · 殁）**：老议员的女仆——真身是格鲁最信任的部下炼金师「骨蜘蛛·卡罗」，24 年前潜入、**自我封锁记忆 + 催眠出「忠仆瑞丝」的临时人格**（主意识睡、瑞丝真心侍奉老议员十几年），几年前主意识苏醒、开始用微型超凡生物侵老议员体、在其心脏植可远程操控的核（好操控老议员做致命决策）。被苏晓用排除法锁定、一刀斩首（意志力太强·生擒无用）。
- **铁牙·穆罗（议会院官员 · 殁）**：私改护送苏晓的秘术师队伍（换成弱鸡新晋秘术师）——事败被格鲁灭口割喉、尸旁留「灵魂灰烬」。

## 王国 / 议员

- **圣境老议员**（见上·病危）/ **霜蛇议员**（冷血务实·主政战略：开地城 / 主动迎击 / 秘术师失秘纹后全部逮捕 / 处死 5 名凑数议员）/ **蔷薇议员**（激进强势·力主「绝不削减人口、否则寒士兵心」·关键时比霜蛇更狠但不对自己人狠）——三实权议员。

- **亚伯九世（东泽王国王 · 殁）**：东泽（音乐之都·战力最弱）之王——秘术师最先被侵蚀，他变怪前令全国民众撤往安戈城地城、给守护者「苍魂」尊严（自爆免其变怪），从容赴死。

## 厄运镇（间隙之后 · 七阶高危隐藏区 · 古神所在）

焚「灵魂灰烬」开「间隙」才能进的隐秘之地（须以被腐朽者头发为坐标）——奥卡兹大帝当年正是由此建立光暗帝国、灭国之力也源于此。脱离间隙 8 米即受古神之力侵蚀（每分钟 3% 真伤·恢复失效）。

- **古神（厄运镇 · 远超七阶）**：能量恶意 / 侵蚀力远超索托斯 / 莎布（梦魇之神尼加狄在它面前「连幼童都算不上」）、疑与冥神阶位相近——**却是苏晓见过最安分的古神**（只是「存在」就衍生黑色光球 / 灵魂灰烬 / 怨愤孤儿）；反常地把古神之力分给厄运镇居民（苏晓存疑·怀疑另有隐情）。
- **不朽之人（厄运镇守门领主 · 力 167）**：破衣 + 无护手长刀 + 腰挂破损头骨的干枯强者——古神能量 61500·斩杀了先来的己方女契约者 + 死亡乐园契约者；无法离开厄运镇。
- **怨愤孤儿（天空巨眼 · 极危险）**：焚灵魂灰烬可能惊动它——半个安戈城大的愤怒巨眼、「还未出生就死」的怨念存在；轮回乐园标「极其危险」。
- **忏罪人 / 女契约者**：**忏罪人**（奥卡兹大帝当年带进厄运镇的居民·活千年、介于生死间、疯狂忏罪·被四人小队所杀）；**己方女契约者**（17 天前晋七阶来执行特殊任务·真身份 = **12059 号契约者·任务只是「开路」**〔杀不朽之人、开救赎教堂门〕·与死亡乐园契约者合作闯厄运镇被不朽之人所杀·以老议员为坐标开空间通道逃到苏晓处、临死警告后气绝——她的后续剧本由苏晓接手）。
- **不朽之人 = 阿鲁斯·莱克（救赎教堂「门之守卫」· 殁）**：纯粹的战士、救赎教堂的杀人机器（杀人时良知 / 荣誉 / 使命在心中抗争）——**拒绝祭拜古神**（拒屈服·故受古神之力侵蚀成不朽之人·愿他不朽）；守厄运镇入口、钥匙在他身上。被苏晓用青影王「灭法之刃」0.8 秒斩杀。

## 厄运镇真相（大主教千年布局 · 三十四卷38章大揭示）

- **大主教（救赎教堂最深处 · 真正幕后 · 欲成新古神）**：本世界一切的**真正缔造者**——名字早被遗忘的「黑心老爷爷」、活了千年、无法离开救赎教堂一步（出则即死），却能以精神体穿透涌动的黑暗抵达外界。千年布局：让**奥卡兹大帝**在厄运镇得力量建光暗帝国（棋子）→ 弄出「秘纹」（本属古神之物）→ 成立一个千年神秘组织（**格鲁**是历代首领·鬣狼支持只是幌子·大主教给每代首领当「黑心老爷爷」）→ 散播「秘纹会灭世」的假秘密骗首领们立誓清除秘纹。最终目的 = **战胜怨愤孤儿、成为新的古神**（不惜害死本世界 99% 生灵·「战胜恶龙的勇士自己长出了龙鳞」）。奥卡兹大帝、格鲁在他的棋局里只是陪衬棋子。
- **阿撒兹（真身 · 大主教原定的「救赎者」· 古神之力主角）**：老议员心腹的瘸腿老绅士——**这一代大主教选中的两个最优秀者之一**（另一个是「反派」格鲁）；他与格鲁都从父母遗传了古神之力、并化为可控，是这世界真正的「主角」。大主教剧本本是：阿撒兹杀格鲁得救赎石板 → 查奥卡兹陵墓进厄运镇 → 体内古神之力觉醒成「人造神选者」→ 杀不朽之人（钥匙）/ 齐斯尔曼（灰烬熔炉）/ 怨愤孤儿（古神之力现任掌控者）→ 唤醒大主教救老议员。**这套剧本被苏晓无意间顶替走**（苏晓抢先杀了格鲁、走了阿撒兹的路线）。
- **格鲁（肉体已死 · 灵魂困厄运镇）**：被苏晓一刀秒了肉体后，其意识 + 灵魂被扯入厄运镇——按大主教计划，格鲁之死是**引怨愤孤儿到某地的顶级诱饵**；他对厄运镇了解极深，再见时会很强。

## 救赎教堂（厄运镇唯一安全区 · 对抗古神之力）

- **圣女（被大主教洗礼的「奶妈工具」）**：救赎石板可三选一唤醒「修女 / 女祭司 / 圣女」，苏晓选了**圣女**（未被古神侵蚀·知隐秘较少但敢向苏晓求救）——天生的「圣枪」使用者（原力 162 / 敏 171 / 体 131），却被大主教强推入洗礼池、折断圣枪、力敏体大降（现个位数）、沁没成只会治疗的工具（智 172 / 魅 169·光愈复苏每秒回 1700~2600 生命）。是她隐晦点出「大主教」的存在向苏晓求救。
- **齐斯尔曼（守炉人 · 矮个子 · 救赎教会三基友之一）**：一米五、腰挂黄金饰品的矮小男——厄运镇少数看清真相者，怕大主教，故用镇民骸骨 + 古神之力造了个 20 米巨型骷髅当伪装（藏在其瞳焰内保持理智）。守着**灰烬熔炉**；金属手环是打开教堂石晶墙的钥匙。与苏晓从交手转联手。
- **铁匠 / 三基友**：救赎教会「三基友」= 不朽之人莱克 / 守炉人齐斯尔曼 / **铁匠**（在石晶墙后·会用灰烬熔炉锻造对付怨愤孤儿的武器·愤怒的铁匠「比小镇孤儿更可怕」）。
- **植物大 boss（木眷族最终形态 · 中立）**：世界之种被古神之力侵蚀出的高阶空间能力植物——中立、被一个无聊的「惨白小手捣蛋鬼」借用来跟布布汪 / 阿姆 / 巴哈打招呼（战力疑超怨愤孤儿·别惹）。

## 群 boss 结局 + 古神真相（三十四卷39–52章）

- **古神真相**：厄运镇的古神其实**早已死了**——它是濒死时逃到此世界、片刻吮吸就凉的重伤古神（疑近冥神阶位）；死后古神之力蔓延才成了厄运镇的不幸。**怨愤孤儿 = 它转生失败的产物**（母体人类女强者、父为古神·新生体死于胎腹）。古神之力有**四特性：光 / 暗 / 新生之火 / 心灵**——接纳者心灵会渐渐「古神化」，大主教 / 齐斯 / 怨愤谁成功都可能沦为「新古神」（或就是那古神的复生）。
- **大主教真身（蒸汽学天才少年 · 殁）**：日记揭其一生——本是孤儿院收容的**蒸汽学天才少年**（「蒸汽」一词就是他造的·祖母瘟疫死、挚爱索娅死），加入救赎教会后被古神扶成大主教、逐渐被古神之力侵蚀到**忘了自己就是发明蒸汽的少年、忘了挚爱**，只记得要成新古神（把人看成蚂蚁）。若无古神降临，他本能引领这世界的蒸汽文明。被苏晓三方合力斩杀。
- **齐斯尔曼（守炉人 · 殁）**：与苏晓联手屠大主教后翻脸（想独吞世界之种 + 怨愤心脏成新最强），放大招「黑暗埋葬」偷袭——被苏晓用猎魔时刻的 4 次龙影闪躲过、反手一刀贯首。
