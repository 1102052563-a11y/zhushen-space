## 地点 / 组织 / 设定

- **安戈城**：大陆中心、议会院所在的贸易中枢——四王国货物都运来打议会院印章再卖；表面繁荣、实则小偷泛滥。
- **霜蛇议员 / 蔷薇议员（另两位实权议员）**：**霜蛇**（冷酷·力排众议推蒸汽技术·疑掌控奇多尔曼公司）；**蔷薇**（女·最激进·19 年前按雷曼王的脸盖血章停战）。其余 5 名凑数议员合起来才勉强算一票。
- **奇多尔曼公司 / 赫菲斯**：蒸汽火车 / 蒸汽车制造商（标志黎明鸟）——忠于霜蛇议员、掌控者赫菲斯是大陆最富几人之一；灭国时全力运人。
- **秘术师两系**：**秘纹系**（世袭家族传承秘纹·越传越强）/ **烙刻系**（秘纹刻身·不可传承·多变不稳定）；需「自由灵魂」。

## 其他（一句话龙套）

- **铁牙·穆罗（议会院官员 · 殁）**：私改护送苏晓的秘术师队伍——事败被 [格鲁](光暗世界/格鲁.md) 灭口割喉、尸旁留「灵魂灰烬」。
- **亚伯九世（东泽王国王 · 殁）**：音乐之都东泽（战力最弱）之王——变怪前令民众撤往安戈城地城、自爆免变怪从容赴死。
- **忏罪人**：奥卡兹大帝当年带进厄运镇的居民·活千年、介于生死间、疯狂忏罪·被四人小队所杀。
- **己方女契约者（12059 号 · 殁）**：真身份 = 12059 号契约者、任务只是「开路」（杀不朽之人、开救赎教堂门）——闯厄运镇被不朽之人所杀，以老议员为坐标逃到苏晓处临死警告后气绝，剧本由苏晓接手。
- **植物大 boss（木眷族最终形态 · 中立）**：世界之种被古神之力侵蚀出的高阶空间能力植物——中立、被「惨白小手捣蛋鬼」借用来跟布布汪打招呼（战力疑超怨愤孤儿）。
- **幽蛇姐妹（最弱独苗 boss）**：蛇身双人上半身·被苏晓堵住后秒交神殿秘纹保命·获放过；后投靠救赎教堂替圣女。
- **白牛（天启乐园 · 坦 · 殁）/ 队长（天启乐园 · 近战 · 殁）/ 俄伊司（杀人蜂召唤物 · 伪守望乐园印记）**：白牛力体双修坦被苏晓青钢影真伤克死、自爆反炸队友；队长被杀人蜂寄生掏空；俄伊司假装与杀人蜂不相识组队、露馅致苏晓杀不朽之人奖励缩水。

## 群 boss 结局 + 古神真相（三十四卷39–52章·概览）

联手 [齐斯尔曼](光暗世界/齐斯尔曼.md) + [怨愤孤儿](光暗世界/怨愤孤儿.md) 三对一斩 [大主教](光暗世界/大主教.md) → 反杀翻脸的齐斯尔曼 → 冥神脐带弄死怨愤孤儿；[厄运镇古神](光暗世界/厄运镇古神.md) 真相（早死的重伤古神、怨愤 = 其转生失败产物）；大主教真身 = 蒸汽学天才少年；[圣女](光暗世界/圣女.md) = 索娅影子；[铁匠](光暗世界/铁匠.md) 重锻黑王护臂；[格鲁](光暗世界/格鲁.md) 灵魂被超度。苏晓 **S− 通关 + 突破 150 壁障**。
