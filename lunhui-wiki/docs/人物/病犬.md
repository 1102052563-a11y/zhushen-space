# 病犬（阴兽）

[全职猎人世界](../世界/任务世界/全职猎人.md) **[十老头](../势力/index.md) 麾下「十阴兽」**之一——干瘦、爆炸头、满口尖牙。能力特点是**速度极快 + 牙齿带神经性麻痹毒素**（被苏晓青钢影抵消）；但实力其实不强、防御也差。

- **被擒（十卷21章）**：他与 [水蛭](水蛭.md) 找上苏晓，水蛭被秒后他率几十名黑帮普通成员强攻——被 [苏晓](苏晓.md) 界断线绞成碎块，自己被追到小巷、肺被刺穿、咬下苏晓一块肉也无济于事（速度快却躲不开）。临死之际他兜里的手机响起、来电是「**十老头（主家）**」——苏晓接电话谈起赎人，遂留他一命，**还往他体内植入炼金炸弹要挟**、扯下他一只耳朵立威，押他去友克酒店谈判。他暗自发誓今后绝对远离苏晓。
