# 巴克（执着的伐木工 · 杜波·依斯的殿后猛将）

[海蓝星](../世界/任务世界/海蓝星-世界争夺战.md) 上 **神使杜波·依斯** 的贴身猛将——两米高、乱黄发 + 络腮胡、十月怀胎般的啤酒肚（满是脂肪却藏着澎湃力量）、一口**纹螺钢打造的金属尖牙**，腰挂两把伐木用的**短柄斧**。爱抽劣质雪茄、对斧头痴迷到病态（断了斧柄就咬着斧头、后腰还插着备用斧柄）。战斗时不挑衅、不带敌意，像和你闲聊——「200 多斤、能瞬间把人剁成肉泥的孩子」。

- **属性 / 能力**（使徒之眼测得）：半人类 · 神术改造·力 **98**（真实）/ 敏 90 / 体 **95**（真实）/ 智 37 / 魅 14。**神罚体质 Lv.55**（生命 +8000·肉防 +60·受伤 −25%·痛觉 −60%·免疫真实智力 99 以下的精神控制）、**斧之大师 Lv.49**（斧攻 +252%·震荡强击退·力压敌时 30% 破格挡 + 伤 +50% + 270 内脏震荡）、伐木人（三连斧·力 ×0.8 真伤）、**啃咬**（纹螺钢牙咬面门·判定失败咬碎颅骨即死·生命 >30% 豁免）、**嚼木**（吃木材每秒回 2% 生命·半小时一次）、回旋斩斧（抛斧两段·命中眩晕 / 背后中招流血）。

## 身世 · 与杜波·依斯（十九卷37章）

巴克原是太阳王国类似「杀戮部队」的士兵，厌倦杀戮后隐居深山伐木，却**杀人成瘾**——一夜醒来发现自己杀了人、桌上摆着三颗人头；山区接连命案、神庭死伤 1259 名士兵、耗时 29 天才（在他「自愿」下，理由是「没盐的食物难吃」）抓住他。临刑前神使**杜波·依斯**找上他长谈，次日一个被割舌刺瞎的「巴克」替身被处刑——真巴克从此跟在杜波身边（巴克曾救过杜波九次、杜波也容他在办公室抽烟）。

## 与苏晓五五之战（十九卷36–39章）

杜波·依斯逃跑、留巴克殿后。巴克与 [苏晓](苏晓.md) 在圣利门要塞五层**硬碰硬五五开**（力量碾压 vs 速度占优）——苏晓龙影闪贯穿其心脏、踢他撞穿墙撞上纹螺钢板，他靠神罚体质 8000 血 + 嚼木回血 + 扭脖复位硬生生扛住；最终趁要塞下沉、双方在岩浆上方一米宽落脚点厮杀时，巴克**主动跃下、抓住通红的纹螺钢框架荡进地下通道脱身**——从伤势看是他败、从「挡住苏晓」的目的看却是他胜（苏晓不追：狭窄通道里巴克占七成胜算）。

## 晚宴失主 · 转追苏晓（十九卷46–48章）

巴克随杜波·依斯赴亨纳森的晚宴、全程靠直觉提防「那只鬼」（他笃定苏晓绝不会放弃、甚至有点期待他来）——可惜雷属性子弹太快，[杜波·依斯](杜波·依斯.md) 还是被一枪轰杀（他曾警告杜波别去、杜波不听）。巴克先追逃跑的 [小助手](小助手.md)、被 [布布汪](布布汪.md) 远程一棍敲头 + 四只冰原狼拦住（攻击诡异、让他忌惮）；见东城升起求援信号弹（追苏晓的人损失惨重），他**果断放下小助手、转去追杀更想分胜负的苏晓**。途中撞见被苏晓重创（断臂 + 斩喉 + 内脏全毁）的护卫疤脸女——她吐出「晨光蝶」给他指路、并求他给个痛快，巴克沉默着一斧送她解脱。

## 终战身死（十九卷49–50章）

巴克一路追苏晓到太阳王国荒郊小河边——苏晓选这片开阔地形（埋了炼金炸弹陷阱、又刚补给完状态更好）决战。两人「比战斗智商」的近战达人对决到**生死一瞬**：巴克使出招牌**三连斧**（右臂一斧 + 左臂手刀 + 右臂最重的第三斧·力透 −70% 肉防 + 678 震荡 + 79 真伤穿透），苏晓 800 强度能量盾硬扛住前两击；最后一斧落下时苏晓**赌命预判其斜劈位置、提前冒险侧闪**——只被劈下一大片肩臂皮肉，却同时**斩透巴克持斧的小臂、再斩过咽喉**，传说级 +13 斩龙闪一刀斩断脖颈。巴克至死呲着钢牙笑（既不恐惧也不惊讶）。苏晓**破例把他埋了**（敬他那份「不在乎别人目光、只在乎与什么样的敌人交手」的执着）、插短斧作坟、并在 [命运救赎](../力量体系/装备道具.md) 上刻下第 10 个字「**巴**」。
