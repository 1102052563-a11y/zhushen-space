# 神父（多奈托·普鲁普拉）

**多奈托·普鲁普拉** 是 [东京喰种](../世界/任务世界/东京喰种.md) 世界的俄籍 [喰种](../世界/生物万族/index.md)，喰种多称其「**神父**」（二卷86章）。

## 简介
须发花白、皱纹深刻的老者，曾经营一家天主教孤儿院、暗中**吃院中孤儿**。CCG 资料定其为 **SS 级**，实则真正实力达 **SSS 级**（对战店长胜负五五）——因他只吃孤儿、不聚众也不袭击 CCG，才被低估。是 [亚门钢太朗](亚门钢太朗.md) 的**养父**，暴露身份被「击败」后关入 [库克利亚](../世界/地点/index.md)（二卷86–87章）。

笑里藏刀的老狐狸；**掌握 CCG 的把柄**（一旦公之于众，人类十年内会出现反抗军、喰种将完），故 CCG 关押他却不敢动他。与 [有马贵将](有马贵将.md) 早有约定，被苏晓等救出、加入反 V 的「**喰种世界最强小队**」（被注 Rc 抑制液、暂不能战）。

## 结局（二卷94–97章）
甲赫喰种（肩后尖锥赫子）。随队夺源，岩浆湖一战被飞溅岩浆灼伤仍强忍不出声。祭坛 BOSS 战中被初代喰种 [马尔斯](马尔斯.md) 抓住、贯穿胸口**吸干 Rc 细胞**而亡（有马念旧情未将其斩碎，反令马尔斯藉之回血）。

## 🆕 树生世界 · 记忆护密的合作者（46卷1–3章）

他与 [灰绅士](炎灵星/灰绅士.md)、[仙姬](联盟星/仙姬.md) 同桌进入树生世界，依旧一副慈蔼老神父模样。魔灵星一役他凭太阳圣地的「太阳洗礼」夺取了古神的本源力量，最终收益不比 [苏晓](苏晓.md) 少，是与灰绅士并列的难缠对手。此行他为灰绅士提供的关键能力是**记忆护密**:被他能力影响的人，一旦触及「灰绅士计划」的记忆区就会被误导——违规者艾琳诺被仁慈之刺拷问时，本想全盘招供，却只能说出「灰绅士图谋很大、要做件大事」这种听着有理、实则毫无信息量的废话。苏晓由此断定灰绅士为何非找神父合作:有这能力，哪怕召集再多人也不担心泄密。

## 🆕 议会裁定败亡 · 一场自导的「送死」（46卷24–28章）

裁定席上神父的「棋术」远胜苏晓（Lv.70 vs Lv.65），他把苏晓三天内接触的蘑菇先知、[莱戈](树生世界/莱戈.md)、渔村四人全织进「白夜用深渊之力污染地下水」的滴水不漏铁证（连魔海永生岛他与苏晓初见的影像都伪造出来·还当众爆出「北境女王已死」震动全场）。可惜他算漏了棋局之外——[苏晓](苏晓.md) 早与 [精灵王·克伦威](树生世界/精灵王·克伦威.md) 和五王裔结盟，这场裁定根本是**对神父的公开处刑**（下棋赢了又怎样、对面抽榔头锤你）。他给 [仙姬](联盟星/仙姬.md)、[冥狼](魔海/冥狼.md)、[铁山](树生世界/铁山.md) 15 分钟自行杀出结界（没卖队友、只因牵系 142 股命魂不便在密闭结界出手），三人先后战死；他自己被围后「迷途羔羊」同化黑甲卫、一化二化 3000+，最终被苏晓一颗**烈阳之怒·阿波罗**炸成灰烬（掉「死灵之书」·苏晓凭此获 3.5 万+灵魂钱币 + 189 杀戮功勋 + 127 颗畸变晶化物·深渊）。

- **真相**:苏晓事后推断，神父根本**不是敌人、而是全程伪装敌对的友军**——他被「死灵之书」缠上，故意假意投灰绅士、来虚空之树公证的树生世界找苏晓（猎杀者 + 灭法者最不受危险物待见），捆绑几万灵魂钱币 + 大量功勋 + 仙姬等人的命当诱饵，就是要苏晓杀他、把「死灵之书」接走，既帮苏晓清了一百多违规者、又坑死死敌仙姬。与圣地·奇利亚德诈死如出一辙。苏晓留着死灵之书，扬言神父再现身就把书还他（如伍德送走野爹又被送回）。

## 🕳 失联 · 「幽冥大帝」的双面牌（47卷27章）

幽冥入侵前夕，去奥凯星查源的 [神父](神父.md) 突然失联，只从古神风格的老式电话里传出一段杂音——布布汪分解后是神父的声音「幽冥……大底」。⚠ [苏晓](苏晓.md) 看穿这是神父惯用的「**双面牌**」:「幽冥大底」略改读音即「**幽冥大帝**」（幽冥势力的统领者）。若幽冥胜、神父早已投靠是受益方；若幽冥败，神父就说自己在奥凯星被困、只来得及传出「幽冥大帝存在」这条情报当好人——无论哪方赢，这老阴哔都已站在胜利者一方（不愧圣域乐园出身）。苏晓对他保持放任:早在合作前就给了他吞噬者当后手，说不定那才是最终获胜的契机。

- **真相**:苏晓事后推断，神父根本**不是敌人、而是全程伪装敌对的友军**——他被「死灵之书」缠上，故意假意投灰绅士、来虚空之树公证的树生世界找苏晓（猎杀者 + 灭法者最不受危险物待见），捆绑几万灵魂钱币 + 大量功勋 + 仙姬等人的命当诱饵，就是要苏晓杀他、把「死灵之书」接走，既帮苏晓清了一百多违规者、又坑死死敌仙姬。与圣地·奇利亚德诈死如出一辙。苏晓留着死灵之书，扬言神父再现身就把书还他（如伍德送走野爹又被送回）。
