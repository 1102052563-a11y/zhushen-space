# 亡灵妹 / 死灵妹

轮回乐园的八阶契约者、**「一人即是一个亡灵军团」**的女召唤师——三大战争世界「毒瘤级」人物之一。她是**自愿从猎杀者退阶回契约者**的亡灵系大佬（不少亡灵系强者都如此），长年停在八阶故意累积资源、什么危险世界都去过（对「深渊」印象最深、认为八阶巅峰也只是取得接触深渊的资格）。她标志性地戴一顶**超大号女巫帽**:从一阶就戴、为的是防狙击手爆头（大帽子难锁定），到高阶戴习惯了、不戴「像没穿底裤出门」。

## 潘多拉星 · 三阴哔联手

在 [潘多拉星](../世界/任务世界/潘多拉星-虫族战场.md)，她**一人单挑灭掉一座八阶虫巢**（宰了一位母皇级虫族母体·亡灵大军把敌方虫巢连尸带矿吃得精光·震动帝国），名望榜一度第二。得知她、[苏晓](苏晓.md)、[神父](神父.md) 三人都用「噩梦之始」进入本世界（后来 [凯撒](尼古拉斯·凯撒.md) 加入成四份·灾祸「超级加倍」），她提议联手——组成一支「前猎杀者 + 现猎杀者 + 违规者」前途黑暗的小队，各管一摊(她借帝国拉拢的情报渠道去查危机来源「幽冥」)。她给去奥凯星的神父塞了颗「丢出去能召唤八九千骷髅骑士」的迷你骷髅头（正常召唤系听了要流泪），还私下提醒苏晓「神父这老东西差点收我定金就携款潜逃、坏得很」。

## 关系

- [苏晓](苏晓.md) / [神父](神父.md) / [凯撒](尼古拉斯·凯撒.md)：潘多拉星「四份噩梦之始」被迫联手的队友（她管情报查「幽冥」）。

## 🆕 亡灵大军踏平深红女皇属群（47卷10章）

在潘多拉星，[亡灵妹](亡灵妹.md) 亮出真本事——手持不朽级 +14、镶三颗满评分不朽级宝石的亡灵法杖【无尽亡灵】，率**十几具百米高巨型骷髅 + 遮天蔽日的干枯翼龙 + 从地平线涌来的骨海**攻打本世界最强虫族母皇 [深红女皇](潘多拉星/深红女皇.md) 的属群（一声「亡者归来」，骨爪成片从泥土探出）。她此前已一人单挑灭掉一座八阶虫巢，此番出手让刚与深红女皇结盟的天启姐妹花隔窗目瞪口呆、直呼「召唤系公认的怪物」。

## 关系（补充）

- [深红女皇](潘多拉星/深红女皇.md)：被她的亡灵大军攻打的本世界最强虫族母皇。

## 🆕 给卡拉「修脚」 & 蹭网蹭床（47卷16–17章）

亡灵妹偶遇世界级生物卡拉、兴奋得发抖（世界级生物 = 一大堆灵魂钱币），召几十万骷髅海平推——却因敌超 5 万触发卡拉「外装甲防御阶位 +4」根本破不了防，骷髅兵没给卡拉刮痧、而是「给卡拉修了个脚」（砍甲足脱落的皮壳），遇上了究极克星。她所在的帝国阵营因遇卡拉触发了「击杀卡拉」成就任务（基础奖励**世界钥匙**·她卖苏晓的世界之核碎片都是世界钥匙开世界级宝箱所得）——这也点破了 [苏晓](苏晓.md) 的「太阳圣巢」为何无阵营职能:棘拉是黑户召唤物、苏晓 / 凯撒虚空之树信誉度负到底，得找个高信誉「法人代表」（莫雷）洗白。她还常来大本营蹭网蹭床睡觉、修补防狙大帽子（穷出来的手艺），收了苏晓 5000 灵魂钱币替他约 [莫雷](原始大陆/莫雷.md) 见面（良心疼了一下、看到入账就不疼了）。

## 关系（补充）

- [莫雷](原始大陆/莫雷.md)：她收苏晓钱约出来、被苏晓顶替赴约生擒的天启姐妹花。

## 🆕 反杀英灵殿三人暗杀队（47卷21章）

凯因图谋趁苏晓战卡拉时派八阶顶尖三人暗杀队去生擒 [棘拉](棘拉.md)（想把她变无主召唤物、取而代之当第三家）——却撞在守家的亡灵妹枪口上。她「三个打我一个，很卑鄙」一句，从空间缝隙唤出密密麻麻的死灵战士；三名隐匿暗杀系才惊觉「死灵系原来只是亡灵系能召唤的小弟品种之一」、鼻涕都要哭出来。她与 [苏晓](苏晓.md)、[凯撒](尼古拉斯·凯撒.md) 三方合谋这场局，平分凯因付的 3 万灵魂钱币（一人一万·「以后有这好事千万记得喊她」）。

## 关系（补充）

- [棘拉](棘拉.md)：她替苏晓守住、没让暗杀队生擒的虫族主宰。
