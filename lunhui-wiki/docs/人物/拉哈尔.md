# 拉哈尔（阿三）

[巴赫](巴赫.md) 那伙**「外来者」三人组**的成员——大胡子、皮肤微黑，被同伙轻蔑地叫「阿三」。号称**主坦**却**不带盾、毫无胆量**（被巴赫骂「没胆量就别往主坦发展、吃了多少团队资源」），是个**纯靠装备 + 主动技能**战斗的契约者，战斗经验在 [苏晓](苏晓.md) 眼里「上不了台面」。

- **闯祸的源头**：三人本与清道夫签约平安出境，却是**他管不住下半身、把酒店女服务员拖进房间施暴**，直接激化了与清道夫的冲突（在其本国认知里这只是「小事」）——也间接害死了自己一伙。
- **被斩杀（八卷82–83章）**：苏晓破门后他被推去试探，先被斩断双臂（连同当盾的扭曲防盗门一起斩开）、又被界断线钩锁从半空拽下，**一道刀芒腰斩**（上身落地、肠子洒地），苏晓再**一脚踩碎其头颅**。
