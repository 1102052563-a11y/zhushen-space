# 伊莉雅丝菲尔·冯·爱因兹贝伦（小圣杯容器）

[Fate/stay night 第五次圣杯战争](../世界/任务世界/fate-第五次圣杯战争.md) 三大魔术家族「**爱因兹贝伦家**」的嫡系**人造人少女**——银发红瞳、爱喝下午茶、有颗小虎牙、平日没城主威严。继承了「成为圣杯载体」的命运（**小圣杯容器**·全身被改造成魔术回路的载体）。是 [远坂凛](远坂凛.md) 要好的友人。御主一只 Berserker [赫拉克勒斯](赫拉克勒斯.md)（自恃「绝赢不了」）。

- **被苏晓物理交涉**：[苏晓](苏晓.md) 为找联络时钟塔的渠道登门——一刀斩首她的 Berserker（赫拉克勒斯瞬复活·12 命）、闪身用刀架她脖子，又强塞一颗生物炸弹「糖果」（能随时把她炸成碎肉）逼她交出老式电报机、向时钟塔举报 [言峰绮礼](言峰绮礼.md)。她虽不怕死却怕疼、只得就范。
- **小圣杯的命运**：随英灵接连退场、她体内圣杯渐满；[金闪闪](吉尔伽美什.md) 被独狼阴死后小圣杯彻底失稳、催开天之孔（苏晓本可把她放中心街强召天之孔焚城·最终她活了下来）。苏晓没杀她（杀了无容器会直接化天之孔喷污泥、场面失控对血门有利）。

## 关系
- **[苏晓](苏晓.md)（白夜）**：刀架脖子 + 喂生物炸弹胁迫她的「狂徒」。
- **[远坂凛](远坂凛.md)**：要好的友人（被苏晓逼着登门、心怀愧疚）。
- **[赫拉克勒斯](赫拉克勒斯.md)**：她的从者 Berserker。
- **塞拉**：忠诚的人造人女仆。
