# 斯卡因（加克巴克监狱典狱长 → 杀戮部队团长）

[海蓝星世界争夺战](../世界/任务世界/海蓝星-世界争夺战.md) 中格雅联盟的剧情人物——满脸伤疤近乎毁容、却在崇尚强者的自由之国颇受女性青睐的黑军装男人。原是某军团上校、因故被塞去当**加克巴克监狱**（海蓝星最危险监狱）典狱长，眼看就要官复原职。狠辣果决（当着外交官的面一枪打爆背叛自己的胖副官的头），行事铁血、信奉「战士该死在战场而非牢笼」。

- **娶了太阳王国公主**：那是他打仗时**抢来**的妻子——现任太阳王国国王险些被气死。因此斯卡因之子算太阳王国分支血脉、敌国就算生擒他也不能直接处死（只能「好吃好喝关押」）——这是他的保命牌。

## 与苏晓的交集（十九卷5–18章）

[苏晓](苏晓.md)（库库林·白夜）开局被关在加克巴克监狱，斯卡因暗中给他优待（让小队长送酒送好饭）、又与外交部高层莱宗合演「假口供」把他放出狱、给三天假。**三天后斯卡因升任杀戮部队团长**——成了苏晓在本世界的顶头上司（杀戮部队直属大总统府、实权极大）。苏晓在热都把商业区炸出百米大坑斩了门罗后，斯卡因先夸「做的不错」、转头又笑骂着要把他「关禁闭半年」。

## 关系
- **太阳王国公主**：他战场抢来的妻子（其子 = 太阳王国分支血脉·斯卡因的保命牌）。
- **莱宗**：外交部高层（亦敌亦友·二人合演假口供放苏晓出狱）。
- **[苏晓](苏晓.md)（库库林·白夜）**：麾下杀戮部队的兵（暗中优待、放其出狱）。
