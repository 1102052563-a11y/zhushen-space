# 日向宁次

[火影忍者世界](../世界/任务世界/火影忍者.md) 木叶忍者——日向一族、瞳术「**白眼**」使用者，[迈克·凯](迈克·凯.md) 的「凯小队」成员（同队天天、小李）。

- **破五封结界（九卷43章）**：晓在河之国山洞封印一尾，洞口受 [长门](长门.md) 布下的「**五封结界**」保护（须五人同时破解五处封印、否则各处互通无法强攻）。卡卡西令宁次用**白眼**透视、定位出五处封印的具体位置 → 凯小队四人 + 卡卡西配合破除结界、冲入山洞（已迟，我爱罗已死）。
