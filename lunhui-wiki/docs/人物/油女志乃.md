# 油女志乃

[火影忍者世界](../世界/任务世界/火影忍者.md) 木叶忍者、油女一族——操控**寄坏虫**（食查克拉的虫群）作战，属 [夕日红](夕日红.md) 的追击小队。

- **追击战（九卷66–68章）**：以寄坏虫掩护 [犬冢牙](犬冢牙.md) 撤退（被 [苏晓](苏晓.md) 炼金炸弹炸散）、为 [夕日红](夕日红.md) 施大幻术争取时间（被苏晓刀芒斩碎大片虫群）。混战结束后奉 [奈良鹿久](奈良鹿丸.md) 之命，带重伤断臂的 [日向雏田](日向雏田.md) 撤回木叶。

## 出处
- 九卷65–68章。
