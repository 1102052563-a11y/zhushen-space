# 艾达

一名**契约者**——手背有金色标号「**8**」（竞技天王印记·**竞技场排名第 8**），却谦和不狂傲、笑声银铃。她是**幸运属性特长者**，自嘲「假冒的幸运女神」，实则是名副其实的「**黑酋克星**」（专克 [苏晓](苏晓.md) 那种「[酋长之魂](../力量体系/技能.md)」开箱手气极差的倒霉蛋）。

- **与虫族女王的羁绊**：八卷39–40章她正在 [虚空虫族](../世界/生物万族/index.md) 母巢里——是**[虫族女王「亚」](虫族女王亚.md) 的人类盟友**（亚口中「最信守承诺的人类」「除了艾达人类没好东西」）。亚求她帮忙挡住杀进来的苏晓，她**拒绝了**（「那家伙满身血气、一看就不好惹，双方没冲突、我不想莫名结仇」）；亚答应了她某件事作为交换，她便起身去「会会」苏晓。
- **定位**：与苏晓同为乐园契约者；她信守承诺、行事谦和——是吞噬岛上少见的「中立 / 友善」契约者。

## 与苏晓一战（八卷42–44章·命运 vs 实力）
因答应虫族女王「御敌」，艾达虽觉苏晓「强得根本不像一阶契约者」，仍依约出手。她身高 1米68、19~22 岁、年轻女子，是**幸运系 + 近身抱摔型**契约者：

- **霉运透支光环**：让敌人极度倒霉——苏晓刚被笼罩就**连续倒霉 37 次**（飞船铁板砸 / 控制台爆炸 / 扳手弧线飞砸头·险些荒谬地死在精英刺蝎手里）。
- **命运迷雾光环**：金色细线连接、暂夺敌方「运势」，每秒造成 0.1~10 真伤 + 运势衰减——对苏晓竟造成**每秒 9~10 点真伤**（连艾达自己都惊·她这技能以往只 0.3~4 点）。
- **替身 / 替死技能**：被腰斩 / 穿心 / 斩首后**化作稻草人**、本体毫发无伤地从背后抱摔反击——但代价苛刻（每次耗 2% 最大生命上限 + 一枚灵魂结晶〔小〕·**每个衍生世界限 10 次**·非她运气爆棚早就破产）。
- **黑色手套**（暗紫·动物皮）：戴上即认真近战、步步为营。

**结果——「绝对的实力能抵消运气」**：苏晓以 **[命运救赎](../力量体系/装备道具.md)（气运夺取）**与她抢运势、压下霉运，再凭高敏 + 神经反射 + 锋利斩击专砍要害（青钢影让她「欲仙欲死」）将她斩首三次逼空替死次数；十分钟把整艘飞船都拆了、艾达断臂飞出、惊叹「**竞技场第九没这么强**」。最后她的运气彻底用光（金币占卜出花面）、苏晓的「霉运透支光环」也到极限失效（乐园提示「十小时内霉运已透支至极限」）——**幸运爆棚的苏晓让她只能掉头逃**，被黑白双枪「随缘射击」连续命中臀 / 腰（爆肾）一瘸一拐——其实「霉运透支光环」并非强加霉运（一阶做不到），而是**透支了苏晓之后十小时的霉运**，反令苏晓十小时内只剩「幸运」、随缘连中。她终被苏晓双枪打到濒死（喉咙 + 心脏），坦然认输（「杀掉或蹂躏我，随你」·不求饶不交装备），危急时被 [虫族女王亚](虫族女王亚.md) **冲出用「再生卵」救活**。二人随后撤回虫巢下层（苏晓与她**是「敌人」而非「仇人」**·不主动找、碰面才战）。她与虫族女王是无话不说的好友——切磋时女王被她按在地上摩擦（女王魅力高、战斗弱）。

## 与虫妹组队 · 后续（八卷48章）
被救活后，她与 [虫族女王「亚」](虫族女王亚.md)（昵称「虫妹」）在母巢下层一同看完苏晓的连环算计，感叹「两个老阴比的较量」。虫妹接到乐园招揽欲成契约者「虫族唤醒者」时，艾达**不替她做选择**（只打擦边球提醒乐园之危），待虫妹自己选定后，便**决定与虫妹组队、共渡之后的凶险**——虫妹是她「第一个朋友」。她还断言「**苏晓这种强到变态的老阴比很少见、衍生世界那么多、再碰面机会不大**」（但留了一句「很倒霉的话还是有可能」的伏笔）。两个心机都不深的姑娘逗趣斗嘴（互嘲「假冒幸运女神」「飞机场」）。
