# 炼金侍女·维尼亚

> ⚠ **第二纪元·炼金文明的产物，「初版炼金仆从」——就是曾风靡第二纪元虚空的炼金仆从最初的模样**;⚠ **⚠ 长相并不惊艳、初看只觉温和舒服，越看越是「原来是个很耐看的美人」**（⚠ 创造者严肃、审美却没得说）;⚠ **⚠ 装备特性**:⚠ **「虚假情绪」〔会按不同持有者产生不同的虚假情绪、以与机械女仆本质区分·⚠ 本质无任何情绪、最初制造时就未构建〕· 「炼金生命」〔非战斗型·绝强初期·生活系能力Lv.92〕· 「无性特征」〔可选炼金男仆〔战斗+生物学〕或炼金侍女〔炼金药剂学+更高生活能力〕、每任持有者仅可选定一次〕**（⚠ 评分6000++++++·永恒级）。

## 🧪 她的创造者 —— 炼金大师·科因·莫尔伯

⚠ **⚠ 她的真正持有者，是第二纪元·炼金文明最著名的五位炼金导师之一「科因·莫尔伯」——他还活着！**（⚠ 【科因的炼金笔记】的作者·「炼金机关学」与「炼金生物学」的当之无愧最博学者·晚年促成了【炼金秘典】的著写·秘典中机关学/生物学的传承大部分来自他）;⚠ **⚠ 「此装备处于无法获取持有权状态，上一任持有者炼金大师·科因·莫尔伯并未死亡」**——⚠ **她只是「迷茫」而非强制休眠，说明科因·莫尔伯就在本世界、只是远到她感应不到**（⚠ 苏晓由此判断:他在 **[生命海](../世界/任务世界/众神之界.md) 的尽头**·⚠ 老国王说她「来自生命海尽头、那埋藏一切秘密的可怕禁区」、三巨头一直在找她）;⚠ **⚠ 初版炼金仆从的往事**:⚠ **因性能太优秀、失去换代空间、造价又高，炼金商会高层苦恼不已；后经科因·莫尔伯的孙女出面商议许久，才有了量产的「初版炼金仆从」**——⚠ **他的要价是:炼金商会绝不可对外宣称炼金仆从出自他手，并拒收35%净利润。**

## 🍳 上了列车之后

⚠ **她对领主列车的清洁到了执着的程度**——⚠ **先用快速洗烘装置洗一整套衣物、仔细洗浴、换上烘干衣物，才拿起各种没见过但一试就会用的清洁用具开始打扫**;⚠ **⚠ 厨艺 Lv.92〔仅比 [夏](#) 的 Lv.95 低一筹〕**——⚠ **按苏晓口味逐步改善烹饪〔米饭、香草羊排、什菜汤、蒸鱼、烧烤〕**（⚠ 她给布布汪重做的奶茶好喝到苏晓都皱眉的眉头舒展、代价是黑枫茶被喝掉三分之一「以后少做，太废枫茶」）;⚠ **⚠ 她第一次给人下毒**:⚠ **把整瓶「触发型炼金猛毒」全下进给 [蕾什玛](蕾什玛.md) 的枫茶里**——⚠ **「我调制的红茶比这香醇很多、混合后会奇怪」**（⚠ 这是苏晓对界外诡异蕾什玛的三重保险之一〔原罪之书/契约放逐/触发型炼金猛毒〕·⚠ 第一次下毒的她，没以往那么平静）。

## 🔗 关联

- ⚠ **⚠ 越深入密岛，她越迷茫**——⚠ **在「污秽之地」愣愣站着、在「繁衍殿堂」被诞生之母·夏娃刻意避过〔说明她曾来过这座岛〕、还听到「维尼亚，我的孩子！」的若有若无呼唤**（⚠ 被苏晓一个响指的血气震散·⚠ 她的手掌是「生命工坊」的通行授权、按上识别装置即中止清杀程序）;⚠ **⚠ 她是老国王与苏晓的一场赌局**:⚠ **老国王以她的媒介指环为注、赌自己能否活过第二晚——赢了苏晓要保他儿子在剧变中不死**（⚠ 老国王当晚去世、苏晓赢·⚠ 老国王把她藏在几万名侍女中做十几年见不到人的活、最后一个月才带在身边·「她似乎会产生一些虚假的情绪、比如关心我的健康，不过这些都是虚假，我能确定」）。

## ✨ 新生洗礼（54卷144章 · 最初圣所·科因的个人实验室）

⚠ **苏晓在 [科因·莫尔伯](科因·莫尔伯.md) 的个人实验室最深处，看到她被浸泡在高大玻璃柱内的淡金色溶液中，发丝缓慢飘动**——⚠ **因持有者死亡，【炼金女仆】指环显示她已进入「待唤醒状态」，唤醒需完成「洗礼仪式」**;⚠ **⚠ 「悬赏4·新生:在最初圣所完成新生洗礼」**〔支付10000盎司激活·7.5倍悬赏额度〕——⚠ **苏晓终于知道这项悬赏到底是怎么回事，可怎么进行洗礼，以他「把神秘之眼开发成回旋闪光弹」的炼金机械学水平，显然弄不明白**;⚠ **⚠ 于是他戴上永恒级装备【炼金女仆】指环，对她下达强制精神指令:让她自己完成「新生洗礼」**（⚠ 她睁开双眼，目光疑惑地看着苏晓；他再度下达相同指令·⚠ 溶液排出、玻璃柱升起，全身湿漉漉的她赤着白皙脚丫走在金属地面上，停在一台宛如巨型水槽的机器前）。

- ⚠ **⚠ 三次「四目相对」**:⚠ **她抬手食指斜下指向操作键——苏晓走上前查看，「嗯，非常好，一个键位没认出来，都是炼金机械学的高阶符号」**——⚠ **四目相对十几秒后，她再度抬手，在身前空无一物的金属平面上接连按动几下**〔⚠ 苏晓照着她的位置与顺序按，顶盖打开〕;⚠ **⚠ 她走上台阶、自行躺进洗礼池；又一次四目相对，十几秒后，她似是微不可见地叹了口气**——⚠ **对某名灭法者的炼金机械学完全折服，抬手在身前的空气中凭空按动几下**。
- ⚠ **⚠ 「无性特征」的选定**:⚠ **每任拥有者仅可选定一次；且因制造者性格严肃，即便是炼金侍女状态，她依然无繁衍能力与所有繁衍器官，仅有外貌为女性特征**——⚠ **炼金男仆:更强的综合战斗力与炼金生物学辅助；炼金侍女:更强的炼金药剂学辅助与更高的生活等级能力**（⚠ 苏晓无需考虑:**炼金男仆的战斗力连认真起来的布布汪都打不过**，更别说他的「炼金生物学」的确不怎么样；反观炼金侍女不仅是调配药剂的助手，还能在领主列车上做饭）。
- ⚠ **⚠ 增益的选定**〔选定后无法以任何方式变更〕:⚠ **默认增益为「炼金机械学 + 真实魅力属性」**——⚠ **苏晓当场否了:「好家伙，这要是真加成上去，好好的顶层梯队烹饪炼金侍女，直接变成每顿都做仰望星空」**;⚠ **⚠ 最终选定:炼金药剂学 / 真实智力属性**。
- ⚠ **⚠ 新生之后**:⚠ **她从水液中坐起身，目光有些迷茫，似乎在进行「我是谁、我在哪、我在做什么」的无解三连**——⚠ **片刻后，昔日记忆全部消失，重获新生、不再牵扯因果诅咒**（⚠ 再次四目相对，依然是沉默——**这显然是秉承了苏晓的沉默寡言习惯**，新生的她也不算爱说话·⚠ 她从洗礼装置内起身后，就跟在苏晓身后）。
- ⚠ **⚠ 新生后的性格**:⚠ **忍受不了领主列车内的半点脏污**——⚠ **巴哈把从海里捞出的奥萝拉丢在地上时，她眉头微皱:她刚清理了地面**;⚠ **⚠ 她对「黑枫茶」情有独钟，且并不管这枫茶有多珍贵**——⚠ **不过只要用在自己人身上，在苏晓看来都不是浪费**;⚠ **⚠ 她端上的一桌菜，让从小吃「秘药调配锅」煮出的怪味糊糊长大的 [眼之公主](眼之公主·洛丽丝.md) 吃着吃着控制不住泪流满面**——⚠ **她的直觉告诉她:这才是食物应该有的样子与味道**;⚠ **⚠ 唯一的小遗憾**:⚠ **她做的清蒸乌古鱼相比 [夏](夏.md) 的，「不能说略逊一筹，而是稍微少了点味道上的层次」**——⚠ **苏晓正准备给出些建议，但看到她格外严肃的神情后，决定「还是吃烤羔羊排吧」**（⚠ 返回轮回乐园后她曾陷入沉睡；刚醒来时看到炼金实验室的凌乱程度，很想把这里清扫到一尘不染，但在这之前她先要作为苏晓的药剂学助手·⚠ 「枫彩蜜乳」的复刻正是由她协助完成）。
