# 炼金侍女·维尼亚

> ⚠ **第二纪元·炼金文明的产物，「初版炼金仆从」——就是曾风靡第二纪元虚空的炼金仆从最初的模样**;⚠ **⚠ 长相并不惊艳、初看只觉温和舒服，越看越是「原来是个很耐看的美人」**（⚠ 创造者严肃、审美却没得说）;⚠ **⚠ 装备特性**:⚠ **「虚假情绪」〔会按不同持有者产生不同的虚假情绪、以与机械女仆本质区分·⚠ 本质无任何情绪、最初制造时就未构建〕· 「炼金生命」〔非战斗型·绝强初期·生活系能力Lv.92〕· 「无性特征」〔可选炼金男仆〔战斗+生物学〕或炼金侍女〔炼金药剂学+更高生活能力〕、每任持有者仅可选定一次〕**（⚠ 评分6000++++++·永恒级）。

## 🧪 她的创造者 —— 炼金大师·科因·莫尔伯

⚠ **⚠ 她的真正持有者，是第二纪元·炼金文明最著名的五位炼金导师之一「科因·莫尔伯」——他还活着！**（⚠ 【科因的炼金笔记】的作者·「炼金机关学」与「炼金生物学」的当之无愧最博学者·晚年促成了【炼金秘典】的著写·秘典中机关学/生物学的传承大部分来自他）;⚠ **⚠ 「此装备处于无法获取持有权状态，上一任持有者炼金大师·科因·莫尔伯并未死亡」**——⚠ **她只是「迷茫」而非强制休眠，说明科因·莫尔伯就在本世界、只是远到她感应不到**（⚠ 苏晓由此判断:他在 **[生命海](../世界/任务世界/众神之界.md) 的尽头**·⚠ 老国王说她「来自生命海尽头、那埋藏一切秘密的可怕禁区」、三巨头一直在找她）;⚠ **⚠ 初版炼金仆从的往事**:⚠ **因性能太优秀、失去换代空间、造价又高，炼金商会高层苦恼不已；后经科因·莫尔伯的孙女出面商议许久，才有了量产的「初版炼金仆从」**——⚠ **他的要价是:炼金商会绝不可对外宣称炼金仆从出自他手，并拒收35%净利润。**

## 🍳 上了列车之后

⚠ **她对领主列车的清洁到了执着的程度**——⚠ **先用快速洗烘装置洗一整套衣物、仔细洗浴、换上烘干衣物，才拿起各种没见过但一试就会用的清洁用具开始打扫**;⚠ **⚠ 厨艺 Lv.92〔仅比 [夏](#) 的 Lv.95 低一筹〕**——⚠ **按苏晓口味逐步改善烹饪〔米饭、香草羊排、什菜汤、蒸鱼、烧烤〕**（⚠ 她给布布汪重做的奶茶好喝到苏晓都皱眉的眉头舒展、代价是黑枫茶被喝掉三分之一「以后少做，太废枫茶」）;⚠ **⚠ 她第一次给人下毒**:⚠ **把整瓶「触发型炼金猛毒」全下进给 [蕾什玛](蕾什玛.md) 的枫茶里**——⚠ **「我调制的红茶比这香醇很多、混合后会奇怪」**（⚠ 这是苏晓对界外诡异蕾什玛的三重保险之一〔原罪之书/契约放逐/触发型炼金猛毒〕·⚠ 第一次下毒的她，没以往那么平静）。

## 🔗 关联

- ⚠ **⚠ 越深入密岛，她越迷茫**——⚠ **在「污秽之地」愣愣站着、在「繁衍殿堂」被诞生之母·夏娃刻意避过〔说明她曾来过这座岛〕、还听到「维尼亚，我的孩子！」的若有若无呼唤**（⚠ 被苏晓一个响指的血气震散·⚠ 她的手掌是「生命工坊」的通行授权、按上识别装置即中止清杀程序）;⚠ **⚠ 她是老国王与苏晓的一场赌局**:⚠ **老国王以她的媒介指环为注、赌自己能否活过第二晚——赢了苏晓要保他儿子在剧变中不死**（⚠ 老国王当晚去世、苏晓赢·⚠ 老国王把她藏在几万名侍女中做十几年见不到人的活、最后一个月才带在身边·「她似乎会产生一些虚假的情绪、比如关心我的健康，不过这些都是虚假，我能确定」）。
