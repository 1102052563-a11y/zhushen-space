# 夏洛特·克力架（甜点四将星 · 饼干果实）

[BIG·MOM 海贼团](../势力/index.md) **甜点四将星之一**（除玲玲外最强的四人·8.6 亿赏金）、[卡塔库栗](夏洛特·卡塔库栗.md) 之弟——德岛海战 BIG·MOM 一方的两名将星之一（另两名将星在防百兽海贼团）。**实力远逊兄长卡塔库栗、却在战争中价值更高**。

- **饼干果实**：可无限制造、塑形「**饼干士兵**」（制造耗体力近乎可忽略），饼干铠甲普通枪械无效、刀砍只留细痕；饼干士兵「如只会近战的小型高达」、普通海贼破坏不了、连 [琵卡](琵卡.md) 都要半分钟才砸碎一具——能凭一己之力造出规模庞大的军队。
- **狡诈 + 怕痛**：战场上现身的「克力架」其实多是**饼干士兵**（连悬赏令照片都是饼干士兵的模样、非真容），本体藏在巨石后沙滩椅上遥控造兵。他与 99% 的强者不同——**极度怕痛**（连打针都抗拒）。

## 被苏晓远程狙击重创（十八卷19–22章）

[苏晓](苏晓.md) 靠情报得知其真容、又靠 [布布汪](布布汪.md)（融入环境潜到其身边、丈量距离 + 涂鸦标位）锁定本体——隔 **4 公里 + 一块巨石**用 [寂灭公爵](../力量体系/装备道具.md) 一枪轰碎其后脑头骨（脑组织暴露·克力架歇斯底里惨叫）。苏晓连补数枪要爆头，被赶来的兄长 [卡塔库栗](夏洛特·卡塔库栗.md) 用预知 + 年糕墙护住。重伤的克力架自知成了累赘、催卡塔「先解决海流氓、我能撑住」。他被苏晓打成「这是他小看唐吉诃德家族的代价」的活教材，且引来几十名契约者觊觎其人头（杀重伤的克力架可独得击杀好处）。

## 关系

- **[琵卡](琵卡.md)**：港口上与其「饼干士兵」交手的唐吉诃德家族干部。
