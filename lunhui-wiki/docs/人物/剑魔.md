# 剑魔（虚空强者 · 伏笔）

一名**虚空中的强大存在**——八卷41章经 [虫族女王「亚」](虫族女王亚.md) 之口揭露的**重大背景人物**（本人尚未登场）。

- **暴行**：他**看上了虫族女王、强行抢夺**，扬言「**老子看上的女人跑不了**」；女王拼死逃了几百年。他**一剑斩开过虫族的母星**、屠戮上百亿智慧虫族——「反抗会被屠戮一空、不反抗也会被屠杀半数以上」，是个「**杀了几十亿生物面不改色的变态 / 疯子**」。
- **伏笔意义**：剑魔与「[虚空杀神](index.md)」（六卷·杀错星球屠尽卡巴星半兽人）、本世界的 [金焰女子](金焰女子.md) 等，共同勾勒出**虚空中存在大量未知种族与恐怖强者**的背景——乐园的「夺世界」之战只是其中一隅。其与 [苏晓](苏晓.md) 是否会有交集待续。
