# 米拉（狙击契约者）

> 与 [奥利维亚·米拉](奥利维亚·米拉.md)（冰雪女王）同名异人——此「米拉」是闯入钢炼世界的**契约者**。

金色长辫的女狙击手·**现实世界本就是狙击手**，进入轮回乐园后如鱼得水、狙杀过无数契约者。

- **历程**：七卷37章卡诺镇巷战中，她藏身钟塔、用消音狙击枪**偷袭 [奥利维亚·米拉](奥利维亚·米拉.md)**（被其敏锐躲过）——这一枪苏晓事后才感知到（说明她能短暂屏蔽苏晓感知）。苏晓当即指挥十几辆坦克集火夷平钟塔，米拉牙关紧咬、抱枪逃出小镇（苏晓懒得追·距离太远）。
