# 尘

**尘** 是 [进击的巨人](../世界/任务世界/进击的巨人.md) 世界 **[兄弟会](../势力/index.md)** 冒险团的新成员、**法师**（三卷3–5章）。

## 简介
持**绿色精品法杖**、智力属性至少 20 点；风龙卷、大火球等技能威力惊人（一击秒杀十米级巨人），攻防贡献榜列**第三**。奉团长 [亚当](index.md) 之命来三区协助，凭战力一度夺得三区首领之位、收拢三十余名契约者，却因排兵布阵稚嫩、被副团长 [血玫瑰](血玫瑰.md) 比下去、不得不照其战术行事。对苏晓骤显的强大战力颇为忌惮（三卷3–7章）。

## 兄弟会全灭 → 独行 · 蚁王讨伐队再遇苏晓（十六卷32章）

数个衍生世界后，[苏晓](苏晓.md) 在全职猎人·蚁王篇的旅馆里与他重逢——此时的尘判若两人：清秀脸上**自己留下三道丑陋疤痕**（提醒自己乐园多危险、自己曾多天真）。原因是**兄弟会几乎全灭**（团长 [亚当](index.md) 死在某战争世界·队友一个个死在他眼前）——这「亲如兄弟尽丧」的痛苦让他从沉默腼腆变成「**三句话内把天聊死的冷场王 / 毒舌**」（直言「人都是虚伪的、说出事实就惹人愤怒」）。他继承了兄弟会全部资产（一人富可敌团）、并从五系技能派法师**向学术派法爷转变**（元素气息变淡）。如今他作为「一个人的冒险团」加入 [蚁王讨伐队](../世界/任务世界/全职猎人-蚁王篇.md)，对苏晓的强大彻底信服（团长生前曾说「斩首的夜强到能一人灭掉兄弟会」）。

- [血玫瑰](血玫瑰.md)——三区争首领的对手（战术不及对方）。
- [苏晓](苏晓.md)（白夜）——旧识 / 蚁王讨伐队临时队友（视其为「无法战胜」的强者）。
- 老烟枪 / 水咲——蚁王讨伐队的临时队友（常被他三句话聊死）。
