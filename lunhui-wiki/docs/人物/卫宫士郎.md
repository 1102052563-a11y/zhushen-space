# 卫宫士郎（剧情主角 · 正义的伙伴）

[Fate/stay night 第五次圣杯战争](../世界/任务世界/fate-第五次圣杯战争.md) 的**剧情主角**——卫宫切嗣的养子、励志成为「正义的伙伴」的红发少年。被 Lancer [库·丘林](库·丘林.md) 一枪刺穿心脏濒死时，[苏晓](苏晓.md) 透过空间逼他接令咒（他本能拒绝、却仍被强行烙印），成了苏晓**降临到圣杯世界的「坐标」**（实为伪 Master·相性严重不合）。他奉「人被杀就会死」的理念、绝不允许苏晓「滥杀无辜」，几次试图制约苏晓无果（连自裁威胁都被苏晓冷静教学「该怎么刺喉」噎回）。

- **被换掉令咒**：苏晓嫌他理念不合、用老神父留下的特殊十字架**剥离他的令咒、改赋给 [远坂凛](远坂凛.md)**，把他换回 [英灵卫宫](卫宫士郎.md) 的御主。苏晓自评「有了魔力供给后、我有幸成为这次圣杯战争最弱的英灵」。
- **结局**：圣杯战争收尾时被苏晓顺手清掉（剧情主角·非世界之子）——苏晓由此首次开出**史诗级宝箱（1% 含金量）**（含金量 = 世界之力含量·后缀越高越珍贵）。

!!! note "英灵卫宫（Archer）"

    Archer 职阶英灵——**来自未来、成为英灵的卫宫士郎本人**（白发黑肤·红黑紧身风衣·短刀）。本是 [远坂凛](远坂凛.md) 的从者，苏晓换 Master 时把他改作**卫宫士郎本人的从者**（他对「曾经的自己」并不认同）。战术老道、硬实力中等；自陈「不会效力于废材 Master」。曾替远坂凛跟踪苏晓、亲见其一人屠 39 人的尸场。

## 关系
- **[苏晓](苏晓.md)（白夜）**：救他一命、借他当降临坐标、又换掉他令咒、最终清掉他的「英灵」。
- **[远坂凛](远坂凛.md)**：同学（苏晓把令咒从他身上换给她）。
- **[库·丘林](库·丘林.md)**：开局追杀他两次的 Lancer。
- **英灵卫宫**：来自未来、成为英灵的他自己（被改作他的从者）。
