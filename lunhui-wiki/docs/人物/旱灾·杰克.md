# 旱灾·杰克（百兽海贼团三灾）

[百兽海贼团](../势力/index.md)（凯多）麾下**三灾之一「旱灾」**——奉凯多之命来德雷斯罗萨助阵 [多弗朗明哥](多弗朗明哥.md)（凯多极看重人造恶魔果实技术）。身高 5 米的魁梧汉子，下巴一副狰狞**铁颚**，武器是脖颈两侧象牙状的**两把曲刃弯刀（勾刃）**。**鱼人血统**——海下能呼吸、不会被淹死（但海水仍使其无力）。战争狂人、座右铭「**所到之处寸草不生**」（故称「旱灾」），强大却**极其傲慢、鲁莽**。

## 与苏晓的「炸弹换名刀」谈判（十八卷30–34章）

杰克航队赶来后逼退大妈海贼团。他得知多弗朗明哥是「与白夜一同打退卡塔库栗」、又听说白夜曾与 [Dr.贝加庞克](index.md) 同效力世界政府，一度起杀意（怕白夜也来夺人造果实），经多弗调停 + 自己尝过苏晓的炼金炸弹（曾被海军用它炸沉过船、海底躺大半个月）后改谈合作。

- **结怨与和解**：上门时一脚踩到 [布布汪](布布汪.md)（布布汪故意融入环境、调好角度「等」着被踩、为谈判铺垫）→ 苏晓拔刀、两人交手、多弗武力调停。餐桌上杰克捏碎酒杯、与苏晓对瞪——苏晓毫不虚他（阿姆堵门、布布汪守窗、随时能开猎魔时刻弄死他），杰克这才收敛。
- **被阿波罗炸过**：多弗朗明哥用阿波罗炸退大妈夜袭后，杰克认出那火焰——他驻守的岛曾被海军投过阿波罗、背上满是触目惊心的烧伤疤（「苏晓造的两种炸弹他都品尝过」）。
- **勒索失败**：杰克想白嫖 50 颗阿波罗、把苏晓当免费苦力——苏晓反开价「拿 50 颗的材料 + 17 把大快刀来」，并放话「红发、白胡子都是不错的选择」（不非得栖身百兽）。杰克勒索不成、悻悻离开，临走却私下问多弗「名刀方面你有渠道吧、我需要几把名刀」——准备与苏晓等价交易（真香）。随后因凯多有急事、暂带航队离开德岛。

## 关系
- **百兽·凯多（四皇）**：效忠的船长（凯多盟友仅多弗朗明哥一人·杰克替凯多盯住人造果实）。
- **[多弗朗明哥](多弗朗明哥.md)**：助阵的合作方（杰克想压榨、却被苏晓的存在制衡）。
- **[苏晓](苏晓.md)（白夜）**：炸弹谈判对手（傲慢勒索未果、苏晓毫不退让）。
- **凯撒·库朗**：人造果实的原料供应商（杰克对其骗经费花天酒地极不满、却动不了他）。
