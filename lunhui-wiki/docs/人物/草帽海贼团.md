# 草帽海贼团（路飞一行 · 海贼原作主角团）

海贼王原作的**剧情主角团**，香波地篇登陆 41 区。因船长 **蒙奇·D·路飞**是剧情主角，登岛即被大量契约者盯梢（有胆量承受世界排斥者甚至能杀主角）。[苏晓](苏晓.md) 借这一行人**做了袭杀天龙人的局**——他们的存在与凯米被绑、是把天龙人引进拍卖会的关键一环；[布布汪](布布汪.md) 还融入这一行人混在团队里报信（吃喝玩乐胖了三斤都没被发现）。

- **主要成员**：蒙奇·D·路飞（草帽·船长·耿直一根筋·被「为同伴着想」打动才肯撤）、罗罗诺亚·索隆（剑士·路痴）、娜美（航海士·想正常赎回凯米被天龙人五亿压价）、山治（厨子）、鱼人小八（最清楚天龙人之死的分量、力劝路飞逃）、美人鱼凯米（被彼特曼绑去拍卖、白胡子接手鱼人岛后人鱼成「稀缺资源」）、考古学家 **妮可·罗宾**（CP0 [埃迪·克](埃迪·克.md) 是她的童年阴影）。

## 被苏晓利用 + 假面引导逃离（十八卷74–77章）

苏晓**全程不与他们正面接触**——只把他们当推动剧情的棋子（谁敢搅局就一枪爆头）。三天龙人被狙杀后，草帽团一脸懵（路飞甚至不懂天龙人之死的严重性），是小八与罗宾催着才肯走；独行侠 [假面](假面.md)（她的主线任务正是「抵挡黄猿、协助暴君·熊帮草帽团逃脱」）现身引导他们朝与苏晓相反的航向撤离。

## 关系
- **[苏晓](苏晓.md)（白夜）**：把他们当袭杀天龙人的棋子（不与之正面接触）。
- **[假面](假面.md)**：主线任务是协助草帽团逃脱、现身引导他们撤离。
- **[埃迪·克](埃迪·克.md)（CP0）**：妮可·罗宾的童年阴影、率 CP0 追凶。
