# 卡普（海军英雄 · 拳骨卡普）

**海军英雄**——78 岁、灰白短发的老头，军装下躯干却极精壮、毫无老态。**没吃恶魔果实**，纯靠**体术 + 近乎顶级的武装色霸气 + 变态的速度 / 力量**，战力恐怖到能列入「围捕四皇」阵容（与鹤、黄猿的「夕阳红组合」一同围捕 [苏晓](苏晓.md)）。

- **血脉牵连半个时代**：孙子是白胡子二番队长、海贼王罗杰之子 **波特卡斯·D·艾斯**；亲儿子是革命军首领 **龙**（世界政府最大的敌人）——「论辈分卡普几乎是罗杰 / 白胡子的父辈」。艾斯被捕、即将在推进城被处死，是他这趟「至郁」的心病（战国不许他探望、答应公开后给他「检查推进城六层」的名义）。

## 海下追击苏晓 · 被重创（十八卷56–57章）

苏晓假降后在海上脱身（军舰被布布汪炸沉、苏晓海下逃），卡普独自下海追击——陆地上苏晓绝非他对手，但**海里攻守逆转**：

- 卡普没吃果实、可在海下追得比苏晓还快，却**没有水肺、只能闭气**（追击高速耗氧、最多撑 5 分钟就得上浮换气）；苏晓有水肺、能在海里待几十小时。
- **海水克制拳头**：卡普每出一拳都因海水阻力降力、且拳力先掀起急流把苏晓提前推开；苏晓被击中也只是被海水卸力推退。而**斩击能破开海水、命中不被卸力**——传说级斩龙闪一刀**斩透卡普顶级武装色霸气**、在其手臂砍出 4 公分深伤（+ 青钢影侵体剧痛），第 5 刀更让他全身麻木、险被斩喉。
- 卡普认清「海下不仅抓不到苏晓、再待下去可能葬身于此」，只得上浮换气（满身斩痕），警告鹤别让女海军们去送死。他憋屈得想起「跟罗杰打都没这么窝囊」——却忘了是他先追杀苏晓的。苏晓也因 5 天滴水未进 + 这场战斗脱力、没能趁机收拾鹤等人，只得乘救生筏离去。这是「战斗胜负不只看阶位 / 力量，环境 / 地形 / 气候同样关键」的活教材。
