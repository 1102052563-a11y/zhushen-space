# 一方通行（学园都市最强·矢量操作）

[学园都市](../世界/任务世界/魔法禁书目录.md) **毫无争议的最强超能力者**（排名第 1·Lv.5）——白发红眼、身材干瘦（像营养不良）、桀骜不驯、目光择人而噬。姓名 / 性别均「未知」（外观偏男性）。风光头衔之下却是**被木原一族圈养的「实验动物」**：连人身自由都没有（学园都市有个他无法反抗的恐怖之人），还被当作拦苏晓的「看门狗」。

## 能力 · 矢量操作（近乎无解）

**操纵物体 / 能量的动能「方向」**——可让静止物超高速移动、或改变动能方向：有人一拳打来、他能在皮肤接触瞬间把这拳的动能任意偏移、乃至**反射回敌人手臂上**（反射还会**把动能加强到 1.4~1.9 倍**）。等于一个「打不破还反伤」的乌龟壳。

- **弱点①·须皮肤接触**：他碰不到的东西就无法操纵其「矢量」。
- **弱点②·靠体力 + 大脑运算**：使用超能力要大量验算公式（高强度连续计算会脑疲劳）、且消耗微量体力——他**麻杆身材、体力极渣**。
- **唯一已知破法**：在斩中他的瞬间反向改变攻击方向——但这需对他的能力 / 习惯 / 体重 / 身高 / 发动时间了如指掌（只有其超能力开发者木原数多做得到）。
- **气息外放例外**：[苏晓](苏晓.md) 的 [气息外放](../力量体系/技能.md) 既非物体也非能量（是意念 + 精神力的产物·如同「目光」无法被反射），故**无视他的反射**、能直接伤到他。

## vs 苏晓 · 被消耗战 + 气息外放压制（十三卷30–32章）

苏晓破不了他的反射，遂改用**消耗战**：自己体力 80 + [斩龙闪](../力量体系/装备道具.md)（强化 +12·耐久高）撑得住，专挑势大力沉的斩击一刀接一刀猛攻——逼他**连续几百次高强度反射计算**（斩龙闪耐久 98→32·苏晓本人被反射弹得遍体鳞伤、生命剩 26%）。一方通行从「正面硬抗过军队」的自负、被打到太阳穴跳痛、**意志动摇** → [气息外放](../力量体系/技能.md) 被动终于触发（体力 +27% / 防御 -15 / 8% 技能失败）。苏晓再改徒手拳战（收力骗过反射）、靠气息外放令他陷「惶恐」、**超能力释放失败**的瞬间——**一拳砸碎他的下颚**（鼻子以下脸颊全碎·脖颈扭曲）。他没昏（变异智力高）、想按住苏晓胸膛同归于尽（恍惚间想起自己虐杀的上万御坂妹妹、自认罪有应得），却已眩晕——被打碎下颚 + 脊椎错位、连验算都做不了，**同归于尽失败**，被苏晓一刀斩杀。

**殁（十三卷32章）**：`【一方通行为次级剧情主角（未觉醒命运之力）·超能力者排行第一位】`——苏晓得世界之源 +10.2%（→29.22%）+ 法力 +30（→3208）+ 金色宝箱（开出「命运阻断」）+ 掠天惊澜 +10%（→40%）。苏晓评这场「畅快的战斗」：一方通行杀敌的意志不够坚定、有些迷茫（被命运圈养的痛苦），而苏晓有自己的目的与准则、不盲目乱杀。

## 关系
- **木原一族 / 木原幻生**：圈养他的「实验动物」管理者（拿他当看门狗）。
- **木原数多**：其超能力开发者（唯一懂破他能力的人）。
- **[苏晓](苏晓.md)**：用消耗战 + 气息外放压制、一拳碎其下颚的敌人。
