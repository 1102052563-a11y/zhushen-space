# 三船

[火影忍者世界](../世界/任务世界/火影忍者.md) **铁之国的首领**——铁之国是不属于忍村体系的**中立国**，国民是**武士**（用刀、擅长「能量包裹武器」之术）。

- **拒收苏晓（九卷14–15章）**：[苏晓](苏晓.md) 离开水之国后，本想加入铁之国（看中其「武士用刀 + 擅能量包裹武器」、与自己路数契合、又是中立国最稳妥）——三船虽客气接待，却**婉拒**：铁之国身为中立国，**不可能接纳苏晓这种「名声在外的刺客」**（苏晓因刺杀四代水影、公然袭击雾忍村，已是火影世界顶尖的「恐怖分子」）。这次碰壁，是苏晓转投「晓」组织的直接原因。
