# 疯医生 / 蓝大夫（前违规者 · 集六烙印者）

戴无框眼镜、文质彬彬（大卸八块溅一脸血也斯文）的疯子——曾是违规者（[苏晓](../苏晓.md) 险些接到追杀他的任务），现已「上岸」成契约者。

- **核心能力**：通过击杀其他乐园契约者**强掠烙印**，现**集齐天启 / 圣光 / 死亡 / 圣域 / 守望 / 轮回六种烙印**（手心可切换），故能**潜入任何乐园**。此能力疑似违规、需苏晓全程监督（叛变即清除）；得虚空之树公证。
- 异王把大本营当主战场时，疯医生反把四方乐园都拖下水。
- 与 [院长](院长.md)、苏晓旧敌 J 先生同出一座疯人院、同属「绅士同盟」。

## 关系

- [苏晓](../苏晓.md)：监督他违规潜入能力的人（叛变即清除）。
- [院长](院长.md)：同疯人院、同绅士同盟。
