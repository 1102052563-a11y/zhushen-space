# 癞蛤蟆舰长（暴鼠级狠角色 · 大本营飞船舰长 · 探索队）

脸盆大、戴王冠的癞蛤蟆——大本营飞船舰长，气息极强（与 [暴鼠](../强者争霸战群像.md) 一个梯队）。

- 曾是某位裁决者「小可爱」的引导人（小可爱已成裁决者·「会向他挥剑了真让人欣慰」）；负责操控弹射舱把 [苏晓](../苏晓.md) 投到敌方坐标附近。
- **真身 = 探索队**：与 1045 号、[夏](../夏.md) 是轮回乐园方的**探索队**（乘飞船到万兽大陆只是顺风车），真目的地是崩灭中的超巨型世界「拉美西斯星」——须 7 天内夺得 **[蠕行之灵](../../力量体系/装备道具/index.md)**（杀异王的关键物品）。分工 = 稳局。

## 关系

- [苏晓](../苏晓.md)：把他弹射登陆的舰长。
- [夏](../夏.md)：探索队队友（厨娘 / 交涉）。
