# 乌钟·泰尔德（最后的亚兽人 · 沐光传教场真领袖 · 殁）

传言已死的亚兽人——**灵魂（兽王·泰尔德，兽人）与肉体（乌钟，人类）被强制分离 + 重新融合**（平时分作两「人」住相邻门洞·合体即亚兽人）。

- 刺杀过 [异王](异王.md)、杀穿层层防线却倒在异王前。听闻异种将迁离本想躺平等，得知 [苏晓](../苏晓.md) 已破坏迁徙后**赌一把**：赠苏晓 [乌铁震钟](../../力量体系/装备道具/index.md) + 留守压制沃辛格城内那只巨兽（替苏晓争见异王的机会）。
- **结局（殁）**：收到苏晓让乌鸦带回的「已屠王」骨片，含笑而死（「真让人……死而无憾」）——他赌赢了。

## 关系

- [苏晓](../苏晓.md)：他赌上一切支持、并替他屠王的强者。
- [异王](异王.md)：他刺杀未遂、终被苏晓斩杀的仇敌。
- [多萝西娅](多萝西娅.md) / [幽影](幽影.md)：沐光传教场的同伴。
