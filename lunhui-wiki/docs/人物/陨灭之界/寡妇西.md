# 寡妇西 / 伊克西（蛇蝎美人 · 布局强于院长 · 殁）

不到三十岁的美人——传有 105 任丈夫皆死于新婚夜（厌恶与男性肢体接触却渴望浪漫婚礼·人称「很费丈夫的寡妇西」）。

- 布局 / 大局观**强于 [院长](院长.md)**：两人四阶曾联手进火影世界、把五大忍村 + 两大冒险团算计到「木叶都出九代火影」、月之眼计划差点成功（带土没登场就被围攻）。
- **结局（殁）**：杀异王的潜入战中，被最强 [指挥长·曼索](指挥长·曼索.md) 所杀（提着她半截尸体）。

## 关系

- [院长](院长.md)：与她互补的搭档（她布局更强）。
- [指挥长·曼索](指挥长·曼索.md)：杀她的裂变种指挥长。
- [苏晓](../苏晓.md)：核心阵容战友。
