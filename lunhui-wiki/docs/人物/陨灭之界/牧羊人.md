# 牧羊人（圣域教团首神官 · 本世界圣域乐园领袖 · 背刺成性）

金瞳的冷漠神官——「神的代行 / 地上管理者」。

- **背刺成性**：刚谎称联合圣光、转头就与死亡乐园结盟夜袭 [伊莫弥](伊莫弥.md) 的圣光；又想联合轮回去背刺死亡。
- 为试探轮回乐园态度，把侍者**乌鸦女**当弃子送出（「你的牺牲我会记住」）——乌鸦女没见到院长就被自杀小队的邪活种玩死、本体随化身之死被牧羊人冷漠抛弃。

## 关系

- [伊莫弥](伊莫弥.md)：被他背刺打残的圣光领袖。
- [苏晓](../苏晓.md) / 轮回乐园：他想背刺利用的一方。
