# 异王（沃辛格城 · 世界之核持有者 · 七阶中上 boss · 殁）

异种的绝对统领，居异种大本营「**沃辛格城**」的「西斯古殿」核心（居所外有一层理论无解的「流雾」·须用 [蠕行之灵](../../力量体系/装备道具/index.md) 腐蚀）。

- **背景**：异种本是侵略种族却想离开万兽大陆（此地资源不足、地行种会吃光有机物）——要借时空之力凝合体**举族迁去更大的世界**（兽人因此屈服）。**力量传承 37 代**（先代技法 / 战斗器官累积）；力 150 / 敏 161（半封印·七阶速度）/ 体 149、无视七阶及以下控制、免疫毒 / 火。世界之核在他身上（贝妮一直找不到）。
- **三阶段死战**：① 远程操控空气 / 岩石（要害心脏·被 [苏晓](../苏晓.md) 以血破操控 + [放逐](../../力量体系/装备道具/index.md) 侵脑，竟掏出受损大脑吃掉进二形态）→ ② 传承近战（三骨尾破空间·两颗心脏）→ ③ 力量集中到第二颗「传承心脏」气息更强。
- **结局（殁）**：抽断苏晓双臂欲捏头时，苏晓故意松开斩龙闪 + 血之兽逼退、用放逐操控脱手的斩龙闪从后心**刺穿两颗心脏**斩杀。剥得「生命洗礼」（意志力 +5）+ [魂之轻语](../../力量体系/装备道具/index.md)；异王一死沃辛格城异种随「淡红大雨」死绝。

## 关系

- [苏晓](../苏晓.md)：三阶段死战、最终斩杀他的猎杀者。
- [深红·塔莎](深红·塔莎.md) / [指挥长·曼索](指挥长·曼索.md)：他麾下的 boss / 最强指挥长。
