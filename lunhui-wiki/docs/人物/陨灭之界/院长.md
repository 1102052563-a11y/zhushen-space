# 院长（二马路精神病院院长 · 超级老阴哔 · 总布局者）

白袍老头——精通布局 / 算计 / 人心把控（大局观上等·略逊 [寡妇西](寡妇西.md)），本世界的总布局者之一。

- 出了名的记仇又欠揍（爱挑事被众疯子圈踢、断手能立刻重生），却「把他卖了还感激涕零」；常自称众契约者是他「没出院的病人」。
- 把 6 个能力太毒的独狼编成 [自杀小队](自杀小队.md)；甩锅 [无名圣徒](../强者争霸战群像.md) 背时空结晶黑锅。
- 与 [疯医生](疯医生.md)、苏晓旧敌 J 先生同出一座疯人院、同属「**绅士同盟**」（衔尾蛇）。

## 关系

- [苏晓](../苏晓.md)：核心阵容战友。
- [寡妇西](寡妇西.md)：布局强于他的搭档。
- [疯医生](疯医生.md)：同疯人院、同绅士同盟的老熟人。
