# 多萝西娅（人类 × 寄生种混血 · 沐光传教场招募者）

灰裙鹿角、气息圣洁的混血——教义是「延续人类」。

- 曾想合作的乐园（死亡 / 教团 / 圣光 / 天启）全被她「克」凉；先以「王心」为媒介托梦 [苏晓](../苏晓.md) 被识破掐碎困住，后引苏晓来传教场。
- 实权很小（只有怕光小女孩听她的）、内心因憎恨与绝望而狰狞。结局带小女孩离开塌陷的传教场（「他们的王，被一个更凶暴的人类杀掉了」）。

## 关系

- [苏晓](../苏晓.md)：她托梦招募、又引来的强者。
- [乌钟·泰尔德](乌钟·泰尔德.md)：沐光传教场真领袖。
