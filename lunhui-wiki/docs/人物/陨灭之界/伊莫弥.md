# 伊莫弥（圣光乐园领袖 · 治疗系 · 转投轮回乐园）

盘发女人——被死亡 + 圣域夜袭打残后挺身带 100 多人撤回的圣光乐园临时领袖（圣光综合战力垫底）。

- **治疗系**：大范围治疗 / 增益，还能**反治敌方**（敌人被她奶反而行动迟缓 / 肢体麻痹·治敌仅 6~8% 治疗量）。
- 看清局势后果断**与轮回乐园结盟报仇**（宁可不赢也不让背刺成性的 [牧羊人](牧羊人.md) 教团赢），借此立威、想从精英幸存者里组个五人小冒险团。三号 **绯珊** 在夜袭中被教团神棍「圣锥」秒杀。

## 关系

- [苏晓](../苏晓.md) / 轮回乐园：她结盟报仇的对象。
- [牧羊人](牧羊人.md)：背刺打残圣光的教团首神官。

## 出处

- 三十三卷（[陨灭之界·五方混战](../../世界/任务世界/陨灭之界-五方混战.md)）7–37 章：被夜袭打残→结盟轮回乐园报仇。
