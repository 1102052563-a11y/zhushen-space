# 自杀小队（6 独狼 · 能力诡异敌我不分）

[院长](院长.md) 把 6 个能力太毒、与大部队同行会误伤友军的独狼编成「自杀小队」（互相伤害就好·**毒师**抽签当队长）——首个目标是摧残人数最多的天启乐园、专打士气：

- **原地去世流（小胖子）**：菠萝头机车装——平时弱到五阶都能吊打，「死后」靠 **黑魔（恐惧传递）** 替身能力翻盘（以耳上小剪刀为载体与击杀者「换命即死」、黑色烂肉捕食三名活物后即复生·克星雪盐 / 圣属性）。
- **毒师（队长）**：玩毒的老哥、没人敢靠近（曾是某冒险团成员·操毒失误）。
- **黑寄生**：阴郁妹子·放出的寄生物连自己都控不全、不分敌我。
- **邪活种召唤**：召唤出的全是邪异活物——把教团侍者乌鸦女玩死。
- **幽灵自爆流（幽灵妹）**：软萌少女·放出的幽灵钻进半公里内一切活物体内爆炸。
- **活性生物炸弹（塔利班玩家）**：爆炸狂魔小孩·口水都能爆炸。

## 结局（血色战场折损过半）

血色战场鏖战 + 断异种补给后从 6 人折损到「死三个半」：**塔利班玩家**终极自爆化灰、**黑寄生**胸口被打穿战死、**邪活种**被地行种撕碎；**小胖子**放黑魔吃光异种补给后生死未知（黑魔仍在异种堡垒附近肆虐）；**毒师**（胸口以下没了·爬回）与**幽灵妹**（燃灵魂拖毒师脱险）重伤生还。
