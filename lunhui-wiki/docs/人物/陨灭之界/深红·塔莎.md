# 深红·塔莎（寄生种 × 人类 · 刀术宗师 Lv.35 · 三 boss 之首）

力 149 / 敏 150 / 体 145、宽刃长刀技法型——达普修要塞守卫的三 boss 之首。

- 「血吻」连斩致敌真实流血 / 治疗无效（团队坦 / 奶的噩梦）、风舞 + 魂舞 + 屠杀刃。
- 一对一根本不是 [苏晓](../苏晓.md) 对手（「三个打一个都费劲」），靠拖时间等援军；惨败后欲自刎谢罪被 [异王](异王.md) 制止。

## 关系

- [苏晓](../苏晓.md)：完全压制她的刀术宗师。
- [异王](异王.md)：制止她自刎的统领。

## 出处

- 三十三卷（[陨灭之界·五方混战](../../世界/任务世界/陨灭之界-五方混战.md)）反攻达普修要塞：守要塞惨败。
