# 幽影（三合体 · 极端暗杀者 · 憎恨一切）

人类身体 + 异种器官 + 兽人血液拼成（非混血）——能力「影之摄死」（钻敌影使本体异变·无光时战力 +70%）。

- 来自「想剥离万兽大陆大气层、让所有氧生命同归于尽」的极端组织，**憎恨一切**（既恨异种也恨人类），只想看 [乌钟·泰尔德](乌钟·泰尔德.md) 怎么死、不帮任何人。
- 结局与沃辛格城的巨兽（第六人）相战、重伤将死。
