# 巨猿·安基度（寄生霸主「白猿」· 殁）

12 米白毛巨猿——皮糙肉厚、生存力 + 自愈逆天（[苏晓](../苏晓.md) 斩二十多刀才用血之兽扑死）。

- 堪称「绝世好队友」以肉身挡苏晓、护 [毒爪](毒爪.md)；殁后掉双圣灵级宝箱（安基度 / 白猿·可合成）。攻要塞时与 [金刚王](金刚王.md) 兽化对撼。

## 关系

- [苏晓](../苏晓.md)：斩二十多刀才扑死他的人。
- [金刚王](金刚王.md)：与他对撼的黑毛巨猿。

## 出处

- 三十三卷（[陨灭之界·五方混战](../../世界/任务世界/陨灭之界-五方混战.md)）反攻达普修要塞：肉身护毒爪、被血之兽扑死。
