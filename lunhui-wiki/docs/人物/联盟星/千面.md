# 千面（违规者 14023 号）

[联盟星](../../世界/任务世界/联盟星-死亡圣杯.md) 里 [苏晓](../苏晓.md) 的**猎杀任务目标**——轮回乐园违规者、编号 **14023**。苏晓评价他是「遇到过跑路能力最强的违规者，没有之一」。

## 能力 · 两大绝活

- **超级速度 · 苟到极限**：把几乎所有资源都砸在速度与保命上——正常状态几分钟就能把追兵甩得无影无踪，可惜脚腕被机关旧危险物克制装备锁住后速度大减、耐力又差（这次被苏晓追了三个多小时就累到吐舌）。还备着短 / 长距离逃生道具、多件类 [神圣十字徽](../../力量体系/装备道具/index.md) 的保命物（极富有）。
- **千面变身**：能 **100% 转变**成他人（非伪装·连专精感知都难辨），甚至能**窃取所变目标的部分记忆**——进本世界后已靠此「吃干抹净」六个契约者小队，第七个（术士 / 雪莱 / 坦系壮男组的队）被他自曝身份、金蝉脱壳、留无辜的雪莱背锅顶追杀。
- **沙枝**：他手背上依附着一张鹅蛋大小的女性面孔「**沙枝**」，负责侦测、评估战力（一度把千面正面战力评 300、苏晓评 4 万以上），是他随身的「可移动老婆」。

## 覆灭（43卷72–74章）

他本一直苟着，苏晓也懒得理；可他忽然两次横穿加曼市、故意在苏晓的追猎范围反复进出——苏晓判断他已与 [仙姬](仙姬.md) 合作、在替仙姬探苏晓身边的防御力度。苏晓遂启动猎杀:以金斯利改造、用 [S-001](../../力量体系/装备道具/index.md) 篡改了未来的**超凡游隼荷鲁斯**（燃生命锁定追踪 14023 号）咬住他的方位。千面被机关的锁脚镣环限速，一路狂逃三小时,先在浮雕街坑了雪莱、又被 [西里](西里.md) 用磁性回震 + 脚镣 + 舰主炮打得吐血，再被 [戈·泽乌](戈·泽乌.md) 超远程狙击逼改道，终被 [苏晓](../苏晓.md) 逼进一线天山谷。他连挡下苏晓的直踹、「刃道刀·流」、血之兽枪形态（保命道具接连碎裂），最后被一记「**刃道刀·青鬼**」斩落头颅。苏晓得**钻石荣誉勋章 ×82**、又补枪打死了想投降签约的沙枝（斩草除根）。

## 关系

- [苏晓](../苏晓.md)（库库林·白夜 / 猎杀者）：追杀他三个多小时、一刀青鬼将其斩首的猎杀者。
- [仙姬](仙姬.md)（圣光乐园违规者）：疑与他合作、指使他探苏晓防御力度的同类违规者。
- **沙枝**：依附他手背、随身侦测的女性面孔「老婆」，随他一同被苏晓了结。
