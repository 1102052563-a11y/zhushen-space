# 葛韦少校

[联盟星](../../世界/任务世界/联盟星-死亡圣杯.md) 南部联盟的一名海军少校、[苏晓](../苏晓.md)（库库林·白夜）颇为信任的一位铁血军官。苏晓两次远赴 **西大陆**（泰亚图文明）都是乘他的钢铁战舰；[临时同盟](../../世界/任务世界/联盟星-死亡圣杯.md) 组建后，他升任**第二军团的战时指挥**——第二军团正是苏晓两大杀手锏之一（后期由清一色老兵组成）。

## 性格 · 「不想被自己人炸死」

作为铁血军官，他对苏晓那套「拿钢铁战舰顶舱当炼金作坊」的做派又敬又怕:苏晓在顶舱制造删减版 [阿波罗](../../力量体系/装备道具.md) 屡屡「误爆」，把士兵们炸得从舱里端枪冲出、以为遭了敌袭；葛韦干脆抱着铺盖卷搬进逃生艇「与士兵同宿」（美其名曰做表率，实为不想被自己人炸死），还委婉恳请苏晓「别再把顶舱炸了」。战斗打响后，他忠实执行苏晓「前期苟发育、后期总攻」的军团流方略——虽一度完全看不懂「怎么就总攻了、不打消耗战了」，却始终坚定传令、指挥第二军团的老兵们完成对寄虫大军的火力全开碾压。

## 🆕 帝国军人的信仰 · S-001 预言的「深海命运」（43卷59、64章）

- **「你会在加曼市引爆阿波罗吗」**：苏晓用 [烈阳之怒·阿波罗](../../力量体系/装备道具.md) 炸出直径 3 公里的金色大火球时，葛韦联想到「若这东西在加曼市爆炸会怎样」——苏晓与联盟高层因夸张的士兵死伤已近决裂。他径直问苏晓「若与联盟高层敌对，会在加曼市引爆这种炸弹吗」，心底竟隐隐**期待一个肯定的答复**;苏晓答「或许，不会？」，他没再多言，只把右拳重重按在胸口——他生于南部联盟、宣过血誓，愿为南部联盟而死，这是他的信仰。
- **S-001 预示的未来**：回南大陆后，苏晓在收容地库激活危险物 [S-001（世界之聆听）](../../力量体系/装备道具.md)——它无法预知非本世界人苏晓的未来，却预示了与苏晓有交集的葛韦「无数种未来中的一种」。那份「深海命运」的记载揭示:七年后东部联盟与南部联盟决裂血战、南部统一成**帝国**，葛韦（帝国上校·「海上屠夫」）驾驶老舰**无畏前列号**远洋送补给、遭袭沉没，被卷入海下漩涡沉入潜艇都到不了的深海。绝境中他坚持帝国军人的意志（宁自尽也不被俘、护「骤雨计划」机密），舰上生还者因黑色触须**逐渐异化成寄虫战士**（触须在黑暗中发白光照明）——他把这一切写进密压罐当遗言。这只是一种可能的未来，却让苏晓窥见了西大陆线虫之祸的深远余波。

## 🆕 撕碎「失败之命运」· 升任中将（43卷65章）

S-001 预示的未来可被「当事人亲手撕碎载体」而化解。苏晓把葛韦叫回办公室，让他撕碎那卷记载「深海命运」的羊皮纸——羊皮纸化烟消散、他背后连接千万事物的丝线应声断裂，那条凶险未来线就此被扼杀。这也间接切断了苏晓自己「失败之命运」的可能（主线第三环随之完成）。苏晓当面招揽他，被他**不讲情面地拒绝**:「我生在南部联盟，如果有一天我死了，也是为南部联盟而死。」苏晓遂反其道而行、以「机关让出两成岛屿资源 + S-001 的分量」作筹码，「逼」南部联盟高层把这位屡立战功的将才**加封为中将**（顶替年过五十的康德中将）——那条被 S-001 中断的未来里葛韦七年后仍是上校，恰因跟对了苏晓而在南部联盟受气；如今他终获应得的擢升。

## 关系

- [苏晓](../苏晓.md)（库库林·白夜 / 临时同盟副指挥官）：他信服并追随却婉拒招揽的总指挥官，两度载他远征西大陆、任第二军团战时指挥；其「深海殉国」的未来被苏晓帮他撕碎化解、又被苏晓运作升为中将。
- [西里](西里.md) / 豪祸：临时同盟第一军团（超凡者军团）的两位指挥（与他并列的军团级将官）。
