# 猎手公司

[联盟星](../../世界/任务世界/联盟星-死亡圣杯.md) **东大陆**处理危险物的势力——相当于南大陆 [收容机构](收容机构.md) / [日蚀组织](日蚀组织.md) 的对应物，但风评臭名昭著。

## 运作模式

东大陆当年也想成立机关 / 日蚀那样的组织，却因初期财力资源跟不上、大人物观望而垮台;危险物又必须有人处理，猎手公司便在这背景下成立。其理念是「**野生超凡者本身就是另类危险物**」——他们**故意通过地下渠道散播超凡知识**、诱使超凡者在民间出现，再追捕这些超凡者、用生物科技加以控制，驱使他们去应对危险物。事后可利用的危险物留下或卖给日蚀组织，无法利用的高危险物则花高价送机关收容。这套「控制」越发极端，才落得今日恶名（讽刺的是，东大陆治安反倒因此比南大陆稳定得多）。公司由**四位大股东**把持，其中 **爵士**（阴柔邪魅、以刀刃 + 命运之血剥离术示人）综合实力排第二、与 [西里](西里.md) 战力相近。

## 🆕 苏晓的「背锅侠」（43卷75–78章）

苏晓确定 [至虫](../../世界/生物万族/index.md) 在东大陆后，料定猎手公司绝不会信机关 / 日蚀的情报、必来掣肘，遂预埋一颗大雷:让 [哥雅](主角队.md) 把接近正牌世界之子的 [白发少年与艾奇](主角队.md) 以「猎手公司要收回吞噬者」的名义送去东大陆祸害猎手公司。他还编造「吞噬者是猎手公司独有技术」的假情报（实为苏晓炼金 + 古神知识所造），让主角队一路打怪拉仇恨、把猎手公司的注意力从机关引开，好降低他对付至虫时的外部风险——「让被算计的人到最后都不知被谁算计、还继续被当枪使」。真爵士曾来加曼市剥离命运之血，被机关高层假扮的「爵士」一番吓退（真爵士来则加曼市损失惨重）。金斯利率五千超凡者赴东大陆、与猎手公司「决裂待开战」，实则谁也不敢真打（踏进地盘只损颜面、无实质损失）。

## 🆕 科都大战覆灭 → 野兽公司（43卷86章）

苏晓在科都追 [至虫](至虫.md)，机关与日蚀却在猎手公司的地盘**科都**开战——这等于对猎手公司「为所欲为」，公司自然强硬入场。它先派用生物科技控制的囚奴超凡者、被机关日蚀教育得很惨；再派效忠公司、身体改造的「王牌超凡战士」（既得利益者及其子孙·命金贵·缺实战），依旧外强中干。一夜混战下来，机关日蚀元气大伤，猎手公司在科都的驻守力量**直接被打没、总部大楼断成三截**；高层还想抓至虫来研究，反被至虫带日蚀精锐灭了个七七八八（仅**爵士**逃脱）。

- **野兽公司诞生**：趁乱，被囚困、屡次抗拒控制的**野兽·克**脱困、一呼百应——他带领曾为囚奴的超凡者揭竿而起，一夜间把「半个奴隶公司」改组成**野兽公司**（收容无家可归者、应对东大陆危险物·理念是「放出心中的野性」，与机关的压抑疯狂、日蚀的烈阳冷酷并列）。此事显然有**东部联盟**蓄谋已久的推手（早知猎手公司的压迫模式无法长久），机关与日蚀的开战不过是点燃了火药桶。真是奇妙:「机关与日蚀打架，把猎手公司给打没了」。

## 关系

- [苏晓](../苏晓.md)（库库林·白夜）：把它当「背锅侠」的机关军团长——借主角队引它的火、掩护自己在东大陆猎至虫（机关日蚀在科都的开战直接导致它覆灭）。
- **野兽·克**：从囚奴中揭竿而起、把猎手公司改组成野兽公司的新领袖。
- [白发少年 / 艾奇](主角队.md)：被苏晓借哥雅之手放去东大陆、专门祸害猎手公司的两名伪世界之子。
- [收容机构](收容机构.md) / [日蚀组织](日蚀组织.md)：南大陆的对应势力（猎手公司不信任、不合作，金斯利率众与之「决裂」对峙）。
