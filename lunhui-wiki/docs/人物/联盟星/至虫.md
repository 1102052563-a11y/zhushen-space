# 至虫（线虫主体）

[联盟星](../../世界/任务世界/联盟星-死亡圣杯.md) 一切祸乱的真正源头、[苏晓](../苏晓.md) 主线第四环「寻觅」的目标——一只**虚空危险生物**，真名**至虫**。因灭法者与施法者的大量灭杀，此类虚空危险生物已极稀少，而至虫更是独一无二、无同族。

## 强度 · 与月狼的孽缘

以虚空为战力大背景、巅峰灭法者为天花板，[银·月狼·希狄](银·月狼·希狄.md) 仅比巅峰灭法者弱一线；而巅峰至虫能与月狼**正面硬拼到两败俱伤**，其强悍可想而知。它随天外陨石降临，曾把众多世界的生灵吞食殆尽——直到撞上守护此世的月狼。那一战让它元气大伤、残存一小部分，也让它认清「这个世界不好惹」、本打算沉睡恢复后离开。它万没想到 [泰亚图大帝](泰亚图大帝.md) 竟会去围攻月狼、还用它的子体把子民变成寄虫战士——它遂囚困大帝、把他养成了「复仇者的靶子」。

## 「跨界蝗虫」的狩猎流程

至虫分裂子体、发动战争，纯为**自身进化**（不同于为族群的恶魔虫族）:①派有智子体寄生各国掌权者、挑起国战；②控制遗体处理部门用尸体培养更多子体；③子体倾巢袭击城市、推平世界、吞光活物植物有机物；④召回 95% 子体互挤高温蒸发、萃取成**生命结晶**供它进化；⑤留 5% 子体开采星球地心资源、搭建「跨界级传送阵」传送到下一个世界。它所过之处不是「寸草不生」，而是留下一片高压岩球与地心黄金球飘动的混乱重力区——肉体与精神的双重毁灭。

## 寄生的进阶 · 兼容度

它择宿主的兼容度节节攀升:泰亚图大帝 **32%** → [阿陀斯·拜肯](阿陀斯·拜肯.md) **57%** → **金斯利 98.6%**。它判断只要彻底泯灭 [金斯利](金斯利.md) 的意识、完全占据其身体，就能获**物种级蜕变、重进化到完美体**。但金斯利意志力过强（解读记忆极难）、又是靠人格魅力而非暴政领导日蚀——它试图伪装金斯利掌控日蚀组织，却险被环 1~环 5 识破（时代变了、人心复杂），只得杀掉智谋薄弱的 [豪祸](主角队.md)、进科都密道藏身。

## 战斗形态（43卷87–90章 · 阿陀斯岛决战）

寄生金斯利后化四米高人形怪物:暗白皮肤、左臂垂着线虫伪装的「毛发」、眉心三圈相套金色环印、金红瞳、后肩爆出数十根 5 米长黑鳞触须（缠身则进「疯狗模式」）。侦测资料（59.5%）:力 235 / 敏 232 / 体 240 / 智 220、深渊之力 63000、借金斯利的世界之力 21350——**生存力冠绝苏晓所遇之敌**:每秒回 5% 血（有优先级）、每耗 1 点深渊之力回 5 血，重伤 20 秒即满血。武器**畸形刀·憎恨**（血肉+金属+线虫的巨型反曲刀·斩击+钝击+击退+刀肌线虫内部撕裂）攻击力 +467%。两大杀招:**进化·命劫**（弹指射「命虫」必中·15% 即死或 75% 穿透 + 虫蚀）、**进化·死亡绽放**（耗深渊之力铺 800 米「虫之领域」·按敌血量层层麻痹 / 易伤 / 逐秒即死）。苏晓靠完美反制 + 刃道刀·时 / 极 / 青鬼硬拼、[猎潮](猎潮.md) 远程连射伤它最高、[阿姆](../阿姆.md) 冰陷阱护场，却因它变态的回复力陷入苦战。

## 🆕 覆灭（43卷91章）

苏晓看破它高频自愈后有短暂「削减期」，遂以血之枪 + 斩龙闪双持 + 死寂烬灭 + 战魂·纵疯狂压制逼它进削减期。关键时刻 [金斯利](金斯利.md) 意识夺回身体、引下「最后的界雷」助攻，[猎潮](猎潮.md) 一箭穿其头颅，苏晓「天怒·奔雷落」+ 魔刃斩碎它的**寄生躯体**（此躯体有独立灵魂、但魔刃斩到什么杀什么、被金斯利隔绝的本体未被斩杀）。金斯利现身求死「它在我脑袋里，别犹豫」，苏晓两指探入其眉心夹出近 10 公分的**金色线虫本体**、甩在石板上一脚碾成碎肉。**至虫碾杀**——苏晓得 22.5% 世界之源 + 不朽宝箱·虫渊，主线第五环「灭虫」完成。至此本世界所有线虫的命源断绝、寄虫战士尽亡。它吞了太多 S 级危险物，其死也标志联盟世界一个「S 级危险物稀少的新时代」的到来。

## 关系

- [苏晓](../苏晓.md)（库库林·白夜）：主线第四环要「寻觅」并猎杀它的猎杀者——阿陀斯岛决战中的死敌。
- [金斯利](金斯利.md)：被它以 98.6% 兼容度寄生的日蚀领袖（金斯利留了后手、意志力顽抗，还没被彻底泯灭）。
- [银·月狼·希狄](银·月狼·希狄.md)：曾与它两败俱伤、让它元气大伤沉睡的守护者。
- [泰亚图大帝](泰亚图大帝.md) / [阿陀斯·拜肯](阿陀斯·拜肯.md)：它先后寄生、当作「靶子」利用的两具旧宿主。
