# 维娜（铁塔镇女医生）

[联盟星](../../世界/任务世界/联盟星-死亡圣杯.md) 极南寒地**铁塔镇**（联盟驻军把守的硫煤矿产之地）的一名超凡者医生，由当地佩德上校请来，为斩杀 [银·月狼·希狄](银·月狼·希狄.md) 后濒死重伤的 [苏晓](../苏晓.md) 疗伤。

## 外貌 · 性格

扎马尾辫、相貌中上、脸蛋带点婴儿肥，初见时一副**弱气好欺负、拘谨腼腆**的模样。可这全是伪装——她实则是个**表面腼腆、内里腹黑的「粉切黑」**，混熟后就暴露本性（穿着清凉、吃炖雪鹿肉吃到满头汗、口无遮拦地打趣苏晓「你也没什么架子嘛」），而且是个**只对同性感兴趣的女色坯**。

## 能力 · 超凡治疗

她的治疗能力相当强——双眼化为莹白、以能量顺伤口没入体内、瞬间令肌肉隆起愈合，治疗效果堪比 [阿姆](../阿姆.md) 的「大修」。唯一的问题是**治疗伴随极强痛感**（比钝刀割肉还酸爽），施展时力道又极狠，寻常患者痛到会一巴掌抽过来，所以她治疗时总空着一只手挡在脸侧防备。苏晓正因不愿自疗时忍痛（自疗需保持双手稳定、不能麻醉），才接受她连续治疗数次。她替苏晓与阿姆疗伤约四天，令二者伤势恢复大半。

## 关系

- [苏晓](../苏晓.md)（库库林·白夜）：她受托疗伤的机关军团长；苏晓重伤中始终按刀警惕、见她可疑举动便欲斩之，最终确认无恶意。
- [阿姆](../阿姆.md)：被月狼重创、由她一并治疗到接近「大修」效果的苏晓从者。
