# 阿陀斯·拜肯

[联盟星](../../世界/任务世界/联盟星-死亡圣杯.md) 千年前**阿陀斯家族**的族长——[银·月狼·希狄](银·月狼·希狄.md) 的灵魂记忆里，那个衣着华贵、蓄小胡子、眸如猎鹰、单膝跪求「深渊之孔」而被月狼婉拒的人，正是他。他也是极南寒地那一片悔罪石碑的立碑者，碑文写着:**「我们以最卑鄙的方式，谋害了最高贵的存在，所有的报应都是罪有应得，它可以屠灭所有，却没这样做——阿陀斯·拜肯。」**

## 阿陀斯家族 · 与深渊之孔的孽缘

阿陀斯家族是南部大陆王国时代把控各王国局势的旧贵族（家族的**阿陀斯三世**是史上第一个能驾驭危险物 [黑皇帝（S-003）](../../世界/生物万族/index.md) 战斗的人）。他们曾与 [泰亚图大帝](泰亚图大帝.md) 合作觊觎深渊之孔、同尝恶果——月狼死后，家族**直系血脉降生的婴孩全是半人半狼的死婴**，无论如何挽救都无法弥补，家族濒临绝种。阿陀斯·拜肯以最卑微的姿态到极南寒地立碑忏悔、甚至试图复活月狼，皆是徒劳；家族最后一员耗尽家财组建**神圣骑士团**（收容机构 / 日蚀组织的前身），希望继承月狼守卫世界的意志。

## 🆕 千年后仍存世 · 「古老存在」被唤醒（43卷61章）

苏晓封闭深渊之孔时，虚空之树公告**「古老的存在已被唤醒」**。镜头随即切到东大陆文艺之都**科都**——一间藏着名画「荆棘」的画廊里，一位苍老身影凝望画作，追忆两百余年前与画家的夜谈。他便是**阿陀斯·拜肯**:一个本该早已作古的人，却活到了千年后的今日。月光映照下他脸色惨白，**瞳孔内一条金色线虫成环形游动**——与临死的泰亚图大帝眼中那条金色线虫如出一辙。这强烈暗示:月狼灭之未尽、寄生过泰亚图大帝的**深渊线虫主体残存**，早已另择载体潜藏，阿陀斯·拜肯（或借其躯壳的存在）正是那个被唤醒的「古老存在」，也是苏晓主线第三环乃至后续要面对的真正幕后。

## 关系

- [银·月狼·希狄](银·月狼·希狄.md)：他千年前跪求深渊之孔、被婉拒、后又参与害死的守护者（立碑忏悔的对象）。
- [泰亚图大帝](泰亚图大帝.md)：当年与他合作觊觎深渊之孔、同尝恶果的泰亚图文明独裁者；二人瞳中同款金色线虫，暗示共同的寄生源头。
- [苏晓](../苏晓.md)（库库林·白夜）：封闭深渊之孔、唤醒了这位「古老存在」的猎杀者——二者尚未正面交锋（伏笔）。
