# 婻（金斯利夫人）

[联盟星](../../世界/任务世界/联盟星-死亡圣杯.md) [日蚀组织](日蚀组织.md) 领袖 [金斯利](金斯利.md) 的妻子，本名**婻**，月初刚为金斯利生下儿子。她自称「**一个普通的家庭主妇**」——的确没有本世界的超凡资质，却拥有与丈夫如出一辙的从容与心机，是个不好惹的博弈者。她怀中的婴儿也非同寻常:从不啼哭，醒着就用圆眼睛打量四周、偶尔发笑。

## 被「劫」与反制（43卷67–68章）

苏晓与金斯利合演的「夺 S-001」大戏里，苏晓需先「劫走」金斯利妻儿以占「理」（逼金斯利拿危险物 [S-001](../../力量体系/装备道具.md) 来换）。婻被 [瘦猴·西里](西里.md) 扮成日蚀「环 2」骗上车、经空间光膜掳到金斯利名下的**鹿花庄园**软禁。她的应对堪称教科书:

- **以攻为守**：一拳把 [猎潮](猎潮.md) 打进墙里、亮出日蚀独有器物 [N715-伯爵](../../力量体系/装备道具.md)（指环化杖鞭）——表明自己「虽是无超凡力量的弱女子、却非任人拿捏」，随即用这秘技**换取暂时的平安**、静待丈夫来救。她给猎潮那一拳也是精算过的（二人有私交、不会立刻遭全力反击）。
- **话疗高手**：转头就用送宝石手链、点伤药「话疗」修复与猎潮的友谊；又拿旧识「埃米莉」精准戳中西里的心事、差点把他聊到破防（被苏晓一句「闭嘴开车」喝止）。她的目的不是逃、而是争取更好的被囚待遇。
- **被算计到底**：她用 N715-伯爵 反抗、连惰性溶液都被苏晓提前备好（脱离适应者后可保存 5 天），最终连耳坠 [J615-王后](../../力量体系/装备道具.md) 也一并被「请」了出来。她清楚自己碰上的是「能与丈夫博弈的人」，索性坦然配合、静候金斯利。金斯利敢让这计划继续，正因既了解苏晓、也信任妻子。

## 关系

- [金斯利](金斯利.md)：她的丈夫、日蚀组织领袖——二人心性从容如一，金斯利后来正是借「妻儿被劫」为名奇袭机关总部夺走 S-001。
- [苏晓](../苏晓.md)（库库林·白夜）：与她丈夫合作 / 博弈的机关军团长，「劫」走她母子的人（她质问「为我们母子值得吗」）。
- [猎潮](猎潮.md)：与她有私交、被她一拳打进墙又被她「话疗」修复友谊的苏晓从者。
