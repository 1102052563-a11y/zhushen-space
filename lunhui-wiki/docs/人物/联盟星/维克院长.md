# 维克院长（收容机构监察长）

[联盟星](../../世界/任务世界/联盟星-死亡圣杯.md) [收容机构](收容机构.md) 三派系之一的首脑——**监察长**，更多被称「维克院长」。他是机构的**脸面**:对外与联盟接触、选拔培养战斗人员、做情报中转，并以「收容院」名义收容孤儿 / 残障者 / 精神病患者。按机构铁律，历代监察长**必须是普通人**（不能由超凡者担任）。他与财政总长 [休琳](休琳.md) 一样，是「一肚子坏水、藏于幕后的老阴哔」。

## 联名革职苏晓（43卷1–3章）

[苏晓](../苏晓.md)（副军团长）一耳光抽死一名联盟议员后，维克院长与休琳「带着歉意」把他联名革职、押到联盟议会大厅——把剩余 11 位议员吓得一致表决「这是机构内部的事、赶快把副军团长带回去」。他把想趁苏晓倒台上位的另一名副手也顺手处理掉（剁了喂给危险物 A-252）。他表面主张求稳，这次却没直言反对苏晓脱困、甚至暗暗支持——因为他和苏晓一样看穿了联盟封锁海运背后有蹊跷（有人想绕过收容机构截留海上的危险物）。苏晓评价:自家这边维克院长和休琳都是藏于幕后的老阴哔，比理念极端却耿直的日蚀组织还难对付。

## 关系

- [苏晓](../苏晓.md)（副军团长）：被他与休琳联名革职、却在脱困上暗暗放行的下属。
- [休琳](休琳.md)：与他一同把持机构幕后的财政总长。
- [收容机构](收容机构.md)：他作为监察长执掌的势力脸面。
