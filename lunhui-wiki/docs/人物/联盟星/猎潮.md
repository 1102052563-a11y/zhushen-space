# 猎潮（溺之首领 · 天巴族第一美人）

源·神乡「**天巴族**」的**溺之首领**、公认的天巴族第一美人（靠武力保证的名号）——蓝中透白光粒的皮肤（天巴族饮源之水后的异变、非血脉）、瞳孔中心黑边缘透蓝，持**源弓**射修长箭矢。她的「溺」能力极强:**每箭附带敌人生命值最大百分比的无视防御伤害**（哪怕放在八阶都强悍到离谱·当年把阿姆射到自闭、巴哈射成「跑地鸡」、布布汪屁股中箭至今怕天巴族）。

## 被苏晓召唤（43卷5–6章）

苏晓进 [联盟星](../../世界/任务世界/联盟星-死亡圣杯.md) 后阿姆 / 巴哈 / 贝妮被传送分散，急需可靠帮手，遂动用一直没舍得用的 [猎潮之残魂](../../力量体系/装备道具.md) 把她召出——还往她新构的心脏里植入了**源（水特性）**（天巴族的力量源泉·大幅提升其战力与存在时间）。她一现身就朝苏晓射箭泄愤（她最后的记忆停在被苏晓用死寂烬灭轰爆手臂、打碎头颅、击穿胸膛），却被召唤契约束缚、挣不脱。苏晓告诉她:天之宫已被他炸平、源在她心脏里、守源人们或死或被他收编——她悲愤却只能适应。作为**临时召唤物**（可存 15~30 天·只要水特性的「源」不被破坏、肉体就能重构），她在本世界战力属上游梯队，被苏晓当核心战力用（冬泉镇一役她进灾厄铃铛的领地打散其一部分、身受重伤——却嫌被当「炮灰」）。

## 🆕 boss 杀手 · 解约留守 · 被寄存【源】（43卷87–91章）

她是苏晓的**「boss 杀手」**核心战力:阿陀斯岛决战中冒死连射伤 [至虫](至虫.md) 最高（对至虫的伤害比苏晓还高些），却因能力极端——一旦被至虫近身或瞪一眼就中线虫濒死，全靠 [阿姆](../阿姆.md) 冰陷阱 / [巴哈](../剧毒魔鹰·巴哈.md) 掩护才没死。这些天她忍辱负重当「女秘书」，只为保住心脏里的【源】、也怕偷袭苏晓失败连累天巴族被屠。

至虫死后，苏晓**解除了她的召唤契约**、放她留在联盟星本世界——却用一份拆成 **32 层的嵌套契约**把水特性【源】反「寄存」在她体内（她检查了半天没看出问题、签下后才发现被算计，身体大半化水吸进【源】内险死）。她愤怒扬言「我下次就算死，也不会再帮你战斗！！」。实则苏晓是留个「可再召唤的工具人」——下个世界起仍能把她召出（用巴哈的话:再「收拾」老实了就能用）。

## 关系

- [苏晓](../苏晓.md)（白夜）：炸平天之宫、把「源」植进她心脏、又召她当临时战力的召唤主（她戒备又不忿、却挣不脱召唤契约）。
- 天巴族 / 天之宫：她的故乡与族群（已被苏晓炸平·她盼有朝一日回源·神乡）。

## 🆕 塞尔星再召唤 · 高级炮灰（45卷5–6章）

苏晓在 [塞尔星](../../世界/任务世界/塞尔星-不败要塞.md) 急需战力，又从 [【源】石](../../力量体系/装备道具.md) 里把她召了出来。经联盟星被吸入源石的教训，她这次备好了「**不战斗就没价值、就不会被利用**」的无敌之盾（她连至虫都敢死战、不惧死），打定主意一动不动。可 [苏晓](../苏晓.md) 两句话就破了她的盾:一是亮出契约（违约不会死、只会**变苍老**）、并戏称她将从「天巴第一美人」沦为「**天巴老斑鸠**」——爱美如命的她当场被拿捏得说不出「不介意」；二是威胁「带你回**源·神乡**（她老家）」——她亲眼见过苏晓在联盟星把西大陆炸沉，深信「没有苏晓不敢做的事」，惊得不轻。最终她扎起单马尾、无奈入队当「**高级炮灰**」（契约时效一个世界进度·活下来别再签第二份）。

## 🆕 塞尔星 · 洛亚什遇伏濒死 · 被封回源石（45卷28–29章）

她奉苏晓之命把 [利·西尼威](../塞尔星/利·西尼威.md) 送到审判所所在的海滨城 [洛亚什](../../世界/任务世界/塞尔星-不败要塞.md)，办完事在七星级酒店享受生活（住宿护肤三餐全挑最贵的·回去找苏晓报销），却被一个脸带金属纹的圣光乐园妹子短刀捅穿后心、搅碎心脏（她以源之水替身脱险、反手一箭把对方钉墙、用苏晓给的**液体金属虫**箭矢审讯——却问不出所以然）。混乱中她被一记看不见的重炮轰掉右侧半月形腹部、重伤逃亡，一口咬定是「审判所」干的（其实是苏晓伪装天启、令她被误当「天启召唤物」闯进圣光地盘所致——**这场误会点燃了天启 VS 圣光的大乱战**）。她濒死逃回大本营，苏晓给她做手术稳住伤情后、把她**封回 [【源】石](../../力量体系/装备道具.md)**（借「重伤无法履约」重置契约·她还得再当一次高级炮灰才能自由）——本世界内不再召她（伤要养 10～15 天·不划算）。

## 🏚 与三件原罪物当邻居的日子（50卷55–56章）

⚠ 苏晓其实一直记着承诺:他原本认为猎潮能凭【源】复生，但越到高阶越发现这不可能——⚠ 「无论怎么说，**猎潮都给他当了两个世界的秘书，承诺还是要尽量遵守的**」。⚠ 他的打算是:**若有机会接触到九阶世界的世界之源，就想办法让那世界之源把【源】吸收掉、让猎潮从【源】内脱离**——⚠ 如此一来她就从**依附【源】存在，变成依附世界之核存在**，虽仍非真正复生，**但可以在一个九阶的广袤世界内随意活动**（这也是苏晓所付出的、比较不错的报酬了）。⚠ **怎奈，天巴第一美人的耐心不太够。**

- ⚠ **恶邻编年史**:她在【源】内沉睡其实还挺舒服（外加【源】的力量滋养，让她沉睡中也在缓慢变强）——⚠ 可这舒适日子没过多久，她发现储存空间内来了个**恶邻**（好在被封在炭盒内、还是碎片）。「想来也是，**隔壁邻居是深渊之罐碎片，搁谁也睡不安稳**」。⚠ 没多久这邻居走了，她心中暗道「**灭法，算你还有点良心**」，继续沉睡——⚠ 可突然有一天她被惊醒:**来了一个被封印的新邻居，而且那邻居的封印越来越不牢固，到最后几乎失效**——⚠ **那是炭盒内的「死灵之书」**。
- ⚠ 「和原罪物做邻居的体验感可想而知」，但她也是经历过各类艰难险阻的人，适应了一段时间后**勉强适应了这新邻居**。⚠ **就在她认为来一位原罪物邻居已经是极限时，第二位新邻居到了——没错，又是一件原罪物**（这让她在心中各种『祝福』了苏晓很久；⚠ 这都让她失笑:**她以前在天巴族的经历加一起，都没这段时间惊险与刺激**）。⚠ **就在她认为和两个原罪物当邻居已是极限时，第三位原罪物邻居来了**——她虽心情复杂，**但也差点笑出声，因为同时持有三件原罪物，属实让她眼界大开**。
- ⚠ **意外的好处**:没多久她发现，与三个原罪物当邻居**也不是无法接受**——⚠ **她感觉自己各方面的抗性都在一点点提升**，短时间内还不算强，**可长此以往，这也是种锻炼**。

## 🗡 越狱翻车 · 被魔灵能量洗礼（50卷55–56章）

⚠ 她可能是感觉苏晓要违背约定、一直把她困在【源】内当工具人，⚠ **这次竟在一定程度上控制【源】，将其触碰到「伪·灭法之刃」的封核**——在两者互相吸附下，⚠ **她成功从【源】内脱困、冲破了苏晓布设的封印，进入到了「伪·灭法之刃」内**。⚠ **然后车门就被焊死了**:从诸多痕迹看，刚进去的她就已发现不对、准备退走，可魔灵能量已侵袭而来，将她的灵魂笼罩其中。

- ⚠ **不能立即抢救**:苏晓握着归鞘的伪·灭法之刃，明显还能抢救一下，⚠ **但要等她被魔灵能量完全侵蚀后才能救**——⚠ 并非故意坑她，而是**她的灵魂在被侵蚀途中与那些能量紧密相连，生硬向外拖拽可能把她的灵魂扯成两截**。真正有效的方式是等侵袭结束、能量的缠绕性与吸附性小了，再把她扯出来。
- ⚠ **抢救**:苏晓在铁匠工坊布下大量剥离特性的印记，从右手指尖放出**五根加入了灵魂能量的灵影线**（如此一来能量特性的灵影线就有缠束住灵魂的特性）——⚠ 那感觉「就像用发丝逐渐刺入到果冻中，刺入的越多，越难以控制方向」；⚠ **一种灵魂的呼救感从魔灵能量内传来**（这显然是她还有意识、发现有外来能量进入），让灵影线的方向更明确。
- ⚠ **那只手**:灵影线先缠上一物、险些将其拖出——⚠ **那是一只有着纤葱般五指、皮肤白皙的手，手背上有很淡的黑色纹路**（不像后加的纹饰，⚠ **而是有种浑然天成感，似乎这只手本就是如此**），黑色指甲不算长却给人种**修长、唯美感**。⚠ **苏晓鬼使神差地抬手与之十指相扣、握在一起**——⚠ **之后模糊又遥远的妩媚笑声在耳中传来**；⚠ **可下一秒苏晓的手骤然发力，那只白皙的手被握成黑烟**，构成似手非手的黑烟形态、**似是想抓住苏晓的袖口但最终没成功**，重新没入到魔灵能量中。⚠ 猎潮被救出后，苏晓抓起她的手腕检查:**她的指甲是正常颜色，手背上也没有那浑然天成的淡淡黑色纹印**——⚠ **那么，方才拖出来的那只手，是谁的？** 她的回答是:「**怎么可能，那里面漆黑一片，除了我被困在里面，没有其他的灵魂波动，这我确定。**」
- ⚠ **脱困后的变化**:她被扯出时**伴随着黑色液质飞溅**，下意识大口呼吸、口鼻间滴出的黑色水液遇空气即蒸发消散——「**呼！呼！差点就被，同化掉了，你储存空间里的都是什么鬼东西！**」⚠ 因被魔灵能量洗礼，她的模样有所变化:⚠ **眼底化为黑色，瞳孔外圈是蓝色，越向中心越透出莹白，头发也漆黑**，在黑色眼底与秀发的衬托下**肌肤显得更白皙几分**——⚠ **只不过，这些美感已在苏晓眼中自行过滤**。
- ⚠ **她能撑过来的原因**:⚠ **正是因为和三件原罪物当了这么久的邻居，她灵魂的多种抗性大幅度提升，才撑过了魔灵能量的侵袭**。⚠ 就算如此，脱困后的她也**感到昏昏欲睡**，并在返回【源】前**多次强调，她不想再去苏晓的储存空间内**。⚠ 苏晓给了她三个选择:**收入个人储存空间 / 收入团队储存空间 / 就此分别**——⚠ 她选择了**团队储存空间**（那里足够安全），只想**沉睡一段时间，其他事后续再谈**。
- ⚠ 她走后，苏晓就可以安心用斩龙闪吞噬伪·灭法之刃了——⚠ 届时**刃之魔灵的魔灵强度将从 70 点至少飙升到 170 点以上**。
