# 猎潮（溺之首领 · 天巴族第一美人）

源·神乡「**天巴族**」的**溺之首领**、公认的天巴族第一美人（靠武力保证的名号）——蓝中透白光粒的皮肤（天巴族饮源之水后的异变、非血脉）、瞳孔中心黑边缘透蓝，持**源弓**射修长箭矢。她的「溺」能力极强:**每箭附带敌人生命值最大百分比的无视防御伤害**（哪怕放在八阶都强悍到离谱·当年把阿姆射到自闭、巴哈射成「跑地鸡」、布布汪屁股中箭至今怕天巴族）。

## 被苏晓召唤（43卷5–6章）

苏晓进 [联盟星](../../世界/任务世界/联盟星-死亡圣杯.md) 后阿姆 / 巴哈 / 贝妮被传送分散，急需可靠帮手，遂动用一直没舍得用的 [猎潮之残魂](../../力量体系/装备道具.md) 把她召出——还往她新构的心脏里植入了**源（水特性）**（天巴族的力量源泉·大幅提升其战力与存在时间）。她一现身就朝苏晓射箭泄愤（她最后的记忆停在被苏晓用死寂烬灭轰爆手臂、打碎头颅、击穿胸膛），却被召唤契约束缚、挣不脱。苏晓告诉她:天之宫已被他炸平、源在她心脏里、守源人们或死或被他收编——她悲愤却只能适应。作为**临时召唤物**（可存 15~30 天·只要水特性的「源」不被破坏、肉体就能重构），她在本世界战力属上游梯队，被苏晓当核心战力用（冬泉镇一役她进灾厄铃铛的领地打散其一部分、身受重伤——却嫌被当「炮灰」）。

## 🆕 boss 杀手 · 解约留守 · 被寄存【源】（43卷87–91章）

她是苏晓的**「boss 杀手」**核心战力:阿陀斯岛决战中冒死连射伤 [至虫](至虫.md) 最高（对至虫的伤害比苏晓还高些），却因能力极端——一旦被至虫近身或瞪一眼就中线虫濒死，全靠 [阿姆](../阿姆.md) 冰陷阱 / [巴哈](../剧毒魔鹰·巴哈.md) 掩护才没死。这些天她忍辱负重当「女秘书」，只为保住心脏里的【源】、也怕偷袭苏晓失败连累天巴族被屠。

至虫死后，苏晓**解除了她的召唤契约**、放她留在联盟星本世界——却用一份拆成 **32 层的嵌套契约**把水特性【源】反「寄存」在她体内（她检查了半天没看出问题、签下后才发现被算计，身体大半化水吸进【源】内险死）。她愤怒扬言「我下次就算死，也不会再帮你战斗！！」。实则苏晓是留个「可再召唤的工具人」——下个世界起仍能把她召出（用巴哈的话:再「收拾」老实了就能用）。

## 关系

- [苏晓](../苏晓.md)（白夜）：炸平天之宫、把「源」植进她心脏、又召她当临时战力的召唤主（她戒备又不忿、却挣不脱召唤契约）。
- 天巴族 / 天之宫：她的故乡与族群（已被苏晓炸平·她盼有朝一日回源·神乡）。

## 🆕 塞尔星再召唤 · 高级炮灰（45卷5–6章）

苏晓在 [塞尔星](../../世界/任务世界/塞尔星-不败要塞.md) 急需战力，又从 [【源】石](../../力量体系/装备道具.md) 里把她召了出来。经联盟星被吸入源石的教训，她这次备好了「**不战斗就没价值、就不会被利用**」的无敌之盾（她连至虫都敢死战、不惧死），打定主意一动不动。可 [苏晓](../苏晓.md) 两句话就破了她的盾:一是亮出契约（违约不会死、只会**变苍老**）、并戏称她将从「天巴第一美人」沦为「**天巴老斑鸠**」——爱美如命的她当场被拿捏得说不出「不介意」；二是威胁「带你回**源·神乡**（她老家）」——她亲眼见过苏晓在联盟星把西大陆炸沉，深信「没有苏晓不敢做的事」，惊得不轻。最终她扎起单马尾、无奈入队当「**高级炮灰**」（契约时效一个世界进度·活下来别再签第二份）。
