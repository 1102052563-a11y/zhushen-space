# 收容机构（敬畏神秘的收容者）

[联盟星](../../世界/任务世界/联盟星-死亡圣杯.md) 两大危险物势力之一——由神圣骑士团分裂而来（第三研究所 → 收容机构），[苏晓](../苏晓.md) 进入本世界的初始阵营。信条:「**我等隐于黑暗之中，皆为无名之人，敬畏神秘，将危险藏于地下之暗**」——主张只**收容 / 消灭**危险物、绝不滥用。

## 三派系

- **监察长**（[维克院长](维克院长.md)）：机构的脸面，对外与联盟接触、选拔培养战斗人员、收容孤儿 / 残障 / 精神病患者。历代监察长**必须由普通人担任**（黑历史:某年机构险些把联盟最高掌权者们控制起来，自那以后不许超凡者当监察长）。
- **财政总长**（[休琳](休琳.md)）：机构的「财神」，掌管名下所有不动产、贸易。
- **暴力机关「机关」**（[苏晓](../苏晓.md) 掌控）：没有正式名字（前身叫第三军团 / 守墓人 / 守夜人·为隐蔽干脆只叫「机关」），40% 以上成员是超凡者，专职处理烈性超凡事件与危险物的收容 / 灭杀。副军团长是高危身份（曾一个月换过三位）。

## 「屁股下的核弹」

机关总部地下收容着 **862 种危险物**——这正是收容机构特权的来源:万一寒了机关的心、无人维持地下收容库，这些危险物跑出来或被夺走，联盟就会原地爆炸。联盟议员曾提议处理掉这些危险物、派 300 名超凡者接手（机关众人乐得夹道欢迎、吃瓜看戏），结果有人关了 **S-009** 的维暖装置放它出来，300 人**死 279、废 15、仅逃 4 人**。从此联盟绝口不提替代机构，还每年拨款——意思是「收好这些危险物、千万别放出来」。

## 关系

- [苏晓](../苏晓.md)（副军团长）：机关的掌控者、机构三派系之一。
- [维克院长](维克院长.md) / [休琳](休琳.md)：另两派系首脑（联名革职苏晓的老阴哔）。
- [日蚀组织](日蚀组织.md)：同源于神圣骑士团、理念敌对的另一势力。
