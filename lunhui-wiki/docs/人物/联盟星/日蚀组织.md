# 日蚀组织（利用危险物的极端者）

[联盟星](../../世界/任务世界/联盟星-死亡圣杯.md) 两大危险物势力之一——由神圣骑士团分裂而来（永夜教会 → 日蚀组织），以 **金斯利** 为首。信条:「**存在即是合理，危险物必定有其价值**」——寻找、**利用**、消灭危险物，**绝不收容**、更为极端（不惜以惨痛代价利用危险物）。它与 [收容机构](收容机构.md) 同源、权势相近、理念敌对。

## 争夺联盟倾向

它其实不算反派阵营（目的与收容机构相近、只是理念不同，比藏于幕后的维克院长 / 休琳还耿直些）。眼下联盟更倾向拥护收容机构——但日蚀在暗中发力:领袖**金斯利**的外甥正参与议员选拔、内投选票很高，几天后就可能填补 12 议员的空位。苏晓判断:一旦把联盟彻底得罪、倒向日蚀，不出几个月日蚀就会彻底崛起、收容机构处境将愈发艰难。这也是苏晓虽被囚、却不能硬闯出去打联盟脸面的原因。

## 关系

- [收容机构](收容机构.md)：同源于神圣骑士团、理念敌对的对手势力。
- [苏晓](../苏晓.md)（副军团长）：收容机构一方，暂避开与日蚀的正面冲突、先完成主线。
- 金斯利：日蚀领袖（了不得的人物·外甥竞选议员）。
