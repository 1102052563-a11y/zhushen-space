# 仙姬（圣光乐园的顶尖违规者）

**圣光乐园**的违规者——身穿白色光粒汇聚的长裙、裙摆拖地、所覆盖处探出一条条半透明手臂（各抓着评分 1200+ 的不朽级武器）。她是**八阶最顶尖梯队**的战力（晋八阶已久·非灰绅士 / 神父那种七阶暴发的底蕴型），也是要在 **树生世界** 搞事的三名头目之一（另两个是 [灰绅士](../炎灵星/灰绅士.md) 与神父）。她冷傲、务实——一句「最强也死得最快」，转手就斩了东部联盟最强超凡者佩肯·西蒙威。

## 联盟星 · 与苏晓的隔空交锋（43卷27章）

在 [联盟星](../../世界/任务世界/联盟星-死亡圣杯.md)（全开放·原生世界）的世界之源排行榜上，她以 **27.5%** 高居首位（[苏晓](../苏晓.md) 第二·19.7%）——两人都想要首位奖励 [树之芽](../../世界/地点/index.md)（开启众生之地·七层），敌对已定。她在世界联络平台上与苏晓对话时坦言:她不像灰绅士 / 神父那样狡猾、与苏晓本无仇怨（「脑子正常的人都不想平白多个敌人」），只是「立场不同——你是猎杀者、我是违规者」，还留下一句「**树生世界见，我的确想见识下灭法有多强**」。

- **被扣「全民公敌」**：她一度想赶去南部大陆用隐秘手段格杀苏晓（顾虑苏晓刚晋七阶就杀了绯世）。苏晓抢先使阴招——让机关全面通缉她，使她与**南部联盟 / 东部联盟 / 收容机构 / 日蚀组织四大势力全部敌对**、被强制佩戴减益称号「**全民公敌**」（交涉 −30 好感、无法与 98.7% 商人交易、无法免费用公共设施）。这会大幅拖慢她的世界之源获取——苏晓无法在实力上稳胜她（单挑九成会败），便用这招削她、保住自己冲首位夺树之芽的机会。他真正怕的，反倒是她够理智、选择忍让拖过这个世界进度。

## 🆕 被灰绅士拉入西大陆助战（43卷32、50章）

被扣全民公敌后，她一度在死亡圣杯水液的竞拍中出到 15500~18200 灵魂钱币（最终 18200 拿下·并不当冤大头）。真正把她卷入战争的是 [灰绅士](../炎灵星/灰绅士.md)——他以「奇术师」马甲在西大陆阵营潜伏，用「两块足够纯净的深渊之力凝结体」+ 一枚**紧急脱离卷轴**为诱饵，把她召去助西大陆阵营对抗苏晓的军团流（条件:事成后深渊之力凝结体一人一块）。她一改平常风格、对灰绅士口吐粗鄙（「你这微笑的混蛋，衮远点」），显然被他算计过、碍于树生世界还要合作才没翻脸。她浑然不知这正是灰绅士的布局:**无论她成败，都会因此与苏晓彻底敌对**，成为灰绅士在树生世界的一枚棋子。

## 🆕 奇袭机关总部（43卷62–63章）

眼看西大陆战局崩盘、苏晓的世界之源已反超自己，她没在西大陆坐等被战争领主的「远古战兽」收拾（苏晓特意为她留了这份厚礼却始终没对上），而是**趁苏晓率军远征、南大陆防御空虚，奇袭加曼市的机关总部**——硬闯进去、又杀出来，庭院里几名超凡者被斩成碎片。她的目标不是危险物本身，而是**资料库**:盗走了 S-009、[S-109（凝视之眼）](../../世界/生物万族/index.md) 等多份危险物情报，还顺带撬开收容地库（危险物 **S-005** 趁乱自己跑了）。此举既是单纯在**捞世界之源**（免得首位奖励树之芽被苏晓独得），也为她与灰绅士后续的图谋添了筹码。

## 关系

- [苏晓](../苏晓.md)（库库林·白夜 / 猎杀者）：联盟星世界之源榜的头号竞争对手、树生世界的敌人（被他扣「全民公敌」削弱、被灰绅士拉去助西大陆阵营、又趁其远征奇袭其机关总部盗资料捞世界之源）。
- [灰绅士](../炎灵星/灰绅士.md) / 神父：与她并列的、要在树生世界搞事的三名头目；灰绅士以深渊之力凝结体 + 脱离卷轴把她拉入西大陆、暗中布局让她与苏晓死敌。

## 🆕 塞尔星争夺战 · 按捺不住的杀意（45卷1章）

作为圣光乐园违规者，她参与了 [塞尔星](../../世界/任务世界/塞尔星-不败要塞.md) 世界争夺战，却和 [灰绅士](../炎灵星/灰绅士.md) 一起窝在虚空·奥术永恒星海港的沙滩椅上——穿比基尼、戴茶色墨镜晒太阳，一边等 [苏晓](../苏晓.md) 上钩。她冲动好斗、耐性差:一度想直接杀去**现实世界**把白夜一劳永逸解决，被灰绅士按住并点破「在现实你所有引以为傲的主动能力都用不出分毫」，她沉默良久、无言以对。她与灰绅士赌 5000 灵魂钱币「白夜不会来」（灰绅士偏赌他会来）。
