# 艾奇（吞噬者寄体 · 伪世界之子）

[联盟星](../../世界/任务世界/联盟星-死亡圣杯.md) 加曼市索娅大酒店的一名小厮——黑褐短发的年轻人，住廉价公寓、有个温柔女友、被楼上的胖邻居吵得睡不好觉、心底满是负面情绪。正因如此，他被 [苏晓](../苏晓.md) 选中当**吞噬者的寄体**:苏晓放出植了黑暗眼的 [吞噬者](../../力量体系/装备道具.md)、寄生进他体内「人造世界之子（伪）」。寄生后他不再唯唯诺诺、肌肉线条分明、夜里「重拳出击」惩恶（初次差点把胖邻居打死才回神）——他以为自己得了天命，却不知身后站着苏晓。

## 苏晓手中的棋子（43卷5、17章）

- **打小怪掉神装**：苏晓给他铺好了「主角剧本」——先安排富商奥利弗（[维克院长](维克院长.md) 施压·自愿被打断腿换民生生意）演一出西雅·索娅家族被棘花报社爆炸牵连的戏，让艾奇「英雄救美」卷入棘花报社被炸案；再让几名戴 **[裂杀](../../力量体系/装备道具.md)**（S 级危险物残骸打造·大小随手掌调节·恰配他战斗形态变大的手）的「小怪」来送，被他打败后「掉落」裂杀给他当武器。
- **两个伪世界之子叠加**：苏晓还偷偷往吞噬者残片里喂了 [命运之种](../../力量体系/装备道具.md)（斩正牌世界之子道格拉斯所得）强化他的伪世界之子特性。他与 [金斯利](金斯利.md) 培养的「[白发少年](主角队.md)」成了「相见恨晚」的挚友、组成主角队一同调查棘花报社被炸案、乘渔船出海找美人鱼——他和白发少年都不知道，自己只是苏晓与金斯利对弈的两颗棋子。

## 🆕 吞噬者第五阶段 · 命运之血被抽（43卷76–78章）

西大陆战争后他体内的**吞噬者**因长期厮杀、吞噬大量超凡血肉而急速成长——他越来越阴冷、能嗅到令他择人而噬的特殊气味，正是逼近**第五阶段**的征兆。[哥雅](主角队.md) 借「吞噬者五阶段必疯、艾奇即将狂化」的假说催化局势:用「混合型惰性气体」激他狂化、又送上被割伤的 [奈奈尼](主角队.md)——狂化的吞噬者·艾奇一口咬掉奈奈尼三分之一躯干，彻底点燃 [白发少年](主角队.md)。二人决战:吞噬者·艾奇进入第五阶段、左臂**开出五只「黑暗眼」**（远超寻常寄体的三只·气息暴涨），双臂化黑色弯刃，对轰白发少年觉醒的金色雷电。就在两人气运于巅峰对撞时，「爵士」（机关高层假扮）从二人眉心**剥离出超越极限的命运之血**（归苏晓·世界级物品）。⚠ 真相:吞噬者是苏晓以炼金 + 古神知识所造、真弱点是「混合型惰性气体」而非什么「五阶段必疯」——若艾奇让吞噬者成长到极限，他会成为**完美共生体**;苏晓还能凭眉心蓝印记对他下三次绝对指令（含自毁）。决战后他与白发少年冰释前嫌，把仇恨全记到 [猎手公司](猎手公司.md) 头上、一同前往东大陆。

**🆕 初代吞噬者被收回（43卷86章）**：三人赶到东大陆才发现猎手公司已一夜覆灭（改组成野兽公司）、报仇扑空。就在风烟港，[猎潮](猎潮.md) 以源之力擦身而过、**将他体内的初代吞噬者剥离收回**（此时吞噬者被最高浓度混合惰性气体压得沉睡）。资料显示:**初代吞噬者**（黑暗 / 撕咬 / 暗系侵蚀 / 超·吞噬 / 宿主保护 / 高潜力·破坏力A/速度A/成长A+），艾奇只把它成长到巅峰的 **21%**、只开发了黑暗眼的部分能力、未进入「重瞳」阶段。它自我意识过强、几乎时刻抗拒创造者命令（若养到巅峰会成自由个体），故苏晓考虑改削弱的**二代吞噬者**（水 / 沸腾 / 血系）另择宿主、7~15 天就练出强于现阶段艾奇的战力。至此艾奇作为苏晓棋子的使命告一段落。

## 关系

- [苏晓](../苏晓.md)（幕后）：把吞噬者寄进他体内、给他铺主角剧本、推他去争美人鱼、又借决战抽走他命运之血的「造物主」（他毫不知情·被蓝印记锁三次绝对指令）。
- [白发少年](主角队.md)：金斯利培养的另一个伪世界之子、他「相见恨晚」的挚友兼主角队队友。
- 西雅·索娅：他救过的索娅大酒店富商之女（苏晓借她把艾奇卷入棘花报社案）。
