# 西里（瘦猴般的机关副手）

[联盟星](../../世界/任务世界/联盟星-死亡圣杯.md) [收容机构](收容机构.md) 暴力机关「机关」里的一名成员——瘦猴般、脸上交错着伤疤、站姿痞里痞气，是 [苏晓](../苏晓.md)（副军团长）最得力的两名副手之一（另一名因苏晓倒台时想联合维克院长上位、被剁了喂危险物 A-252）。他是个「讲究人」（约特殊服务还先付定金）、近三年没休过假。

## 被推上「临时副军团长」（43卷3–4章）

苏晓被囚看押所时，西里带着丰盛餐食和女郎来给长官解闷。苏晓一番「我很看好你、你能成大才」的暖心话把他捧得眼眶发热——转头就给他披上军团长大衣、任命他为「**临时副军团长**」，再把他塞进囚困装甲、代自己继续被关在看押所里顶缸（西里还惦记着预约的服务给了定金没消费）。这是苏晓的折中之计:让联盟议会看到「副军团长仍处于被囚状态」这个态度、给足台阶，自己则脱身去友克市低调完成主线。西里虽被坑，却也因此换来 6~8 个月的**带薪休假**（近三年没休假的他觉得「美差来得太突然」），怨言瞬间烟消云散。

## 关系

- [苏晓](../苏晓.md)（副军团长）：把他捧上临时副军团长、再塞进装甲顶缸的长官（半月前还一耳光把他抽进墙里）。
- [收容机构](收容机构.md) / [维克院长](维克院长.md)：他名义上要打理的机构 / 上级。
