# 泰亚图大帝（泰亚图·奥蒂）

[联盟星](../../世界/任务世界/联盟星-死亡圣杯.md) 世界里，与南部大陆隔海相望的**泰亚图大陆**（未知大陆）之上、**泰亚图文明**的独裁统治者，本名**泰亚图·奥蒂**。他是本世界一切灾厄的真正源头——**窃取深渊之孔、弑杀守护者银·月狼**的野心家。在官方史册里，他是「击败恶狼、镇守可毁灭世界之物的帝王」，真相却截然相反。

## 身份 · 泰亚图文明

- **比南部大陆更古老的文明**：早在南部大陆还处于冷兵器铠甲的「王国时代」、由 [阿陀斯家族](主角队.md) 把控各王国局势时，泰亚图文明就已兴盛多年、并拥有了火器。连**神圣骑士团**（[收容机构](收容机构.md) 与 [日蚀组织](日蚀组织.md) 的前身）的首位骑士团长，都是泰亚图文明派来的将军。
- **停滞退化的原始部落**：如今泰亚图文明退化成了一片阿兹特克风的原始部落，供奉大帝、以活祭掳掠危险物（他们掳走美人鱼，正是为献给大帝吞食）。文明整体陷入贫瘠——这正是大帝为稳住深渊之孔而付出的代价。

## 弑月狼 · 窃深渊之孔的真相

据 [银·月狼·希狄](银·月狼·希狄.md) 的灵魂记忆（苏晓从月光剑读出）还原：

- **贪婪深渊之孔**：大帝曾亲赴极南寒地「探访」月狼，月狼劝他「珍惜文明、莫窥深渊」。大帝表面退去，心中却无法忍受一个自己不可对抗的强大外族存在于世，更觊觎深渊之孔那「令万灵跪扶」的力量。
- **借线虫围杀**：一颗天外陨石带来了**深渊线虫主体**（吸收深渊之力繁殖、吞噬过众多世界生灵、难缠程度近乎中高位虚空异存在）。正是大帝**命人取走陨石里的线虫主体、以子民喂养**，用它拖住月狼。待月狼灭掉线虫却身受重伤、又遭深渊之孔反噬濒死时，大帝**以人海战术围攻数年、终将重伤的月狼耗死**，夺走了封着深渊之孔的那块坚冰。
- **恶果反噬**：月狼死后大帝无法掌控深渊之孔，帝国数日内分崩离析、子民变得野蛮嗜血；他自己则**永远不敢站在月光下**（月光唾弃他、如撕血管掀头盖骨般折磨）。他后悔却死不认账，反在极南寒地立石碑「沐浴在我之荣光下的疆域，皆臣服于我，不需野兽守护」以振威严（此碑被 [阿姆](../阿姆.md) 一脚踹倒、[巴哈](../剧毒魔鹰·巴哈.md) 吐了口痰）。

## 🆕 苏醒 · 吞食金斯利（43卷45章）

[金斯利](金斯利.md) 率三艘钢铁战舰 + [日蚀组织](日蚀组织.md) 环1~环16 远征泰亚图大陆，却因**泰亚图大帝苏醒**而全军覆没——大帝**一口将金斯利吞入腹中**（后证金斯利实以手段存活）、日蚀强者尽墨，仅 [华兹沃](华兹沃.md) 一人扯下巴忍受线虫寄生逃生。苏晓由此判断：**吸收了深渊之力的大帝很可能并未真正死亡，深渊之孔就在他那里**——主线第二环「封闭深渊之孔」的答案，指向去泰亚图大陆宰了这位大帝。

## 🆕 众叛亲离 · 帝王宫殿的困兽 · 殁（43卷57–62章）

同盟军平推到古老王城时才揭开他的窘境:他早已**众叛亲离**——手下**三骑士**（银甲骑士银莫、赤甲骑士等心腹）见「大帝的荣光随月狼之死泯灭」，暗中投靠了 [金斯利](金斯利.md)（后被金斯利坑死）；他自己更是**被困在帝王宫殿的王座上、连起身都做不到**（全靠月狼遗留的结界与深渊之力撑着宫殿），根本无法调度那庞大的寄虫战士军团反攻——所以数十万寄虫战士自始至终群龙无首、如被塞进马蜂窝的爆竹般乱冲。

- **被以最粗劣的方式围杀**：舰主炮 + 阿波罗把古老王城炸成直径 3 公里的巨坑、帝王宫殿被炸得倒立坑中却因深渊之力不碎。待老兵与上万超凡者冲进内殿、打破宫殿结界，他才得以起身——一身如半融石油的铠甲、惨白双眼、由内而外腐朽。他战力虽仍骇人（[阿姆](../阿姆.md) 被拍飞、[猎潮](猎潮.md) 被震退、子弹被无形铁板弹回射杀开枪老兵），却在舰主炮齐射 + 万人围殴下节节败退，被十几杆长兵器架住钉在地上。[苏晓](../苏晓.md) 亲自上前、一刀斩落他的头颅——**「你，是，谁」成了他将死不瞑目的遗言**。苏晓得 **12.55% 世界之源** + 不朽级宝箱 + 特殊物品 [暗蚀虫·帝恨](../../力量体系/装备道具.md)。
- **真相 · 他只是一枚「靶子」**：他之所以敢去围杀高不可攀的月狼，是因为体内寄生着**深渊线虫主体的残存**（月狼灭其大半后余下一小部分寄生于他）——他被力量与贪婪诱惑、渴望复制那「能与月狼对抗」的强大。可线虫主体残存醒来后**根本看不上他、反把他囚困**，只把他当成「万一有人发现月狼之死、来复仇时的绝佳靶子」。苏晓这一炮火洗地，不但轰烂了靶子，还把靶子背后线虫主体在王城地下累积的**高度异化寄虫战士**全烧成灰、连深渊之孔一并破坏——那存在血亏到极点。

## 关系

- [银·月狼·希狄](银·月狼·希狄.md)：被他借线虫与人海战术害死的守护者；月狼的灵魂记忆揭穿了他的真相。
- [阿陀斯·拜肯](阿陀斯·拜肯.md) / 阿陀斯家族：当年与他合作觊觎深渊之孔、同尝恶果（直系血脉生半人半狼死婴而绝种）的南部大陆旧贵族族长（立悔碑者·千年后仍存世·瞳中同款金色线虫）。
- [金斯利](金斯利.md)：远征泰亚图大陆时被他吞食（诈死）的日蚀领袖。
- [苏晓](../苏晓.md)（库库林·白夜）：读出月狼记忆、锁定他为深渊之孔持有者、决意组军团流洗地泰亚图大陆讨伐他的机关军团长。
