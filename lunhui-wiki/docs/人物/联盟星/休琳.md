# 休琳（收容机构财政总长）

[联盟星](../../世界/任务世界/联盟星-死亡圣杯.md) [收容机构](收容机构.md) 三派系之一的首脑——**财政总长**，机构上下所有人的「财神」。机构名下的一切不动产、贸易都由她掌控（她那边拨的经费甚至不及联盟每年拨给机关的多）。她与监察长 [维克院长](维克院长.md) 同为「藏于幕后、一肚子坏水的老阴哔」。

## 坚持「严办」苏晓（43卷1–3章）

[苏晓](../苏晓.md) 抽死联盟议员一事，是她与维克院长联名把苏晓革职、押去议会大厅的。当 11 位议员吓得表决「赶快把副军团长带回去」时，休琳女士偏要「严厉处理这件事」、一番扯皮把苏晓拖在看押所——既是给联盟一个态度（只要副军团长还被囚，那些议员就不会较真），也顺势拖住了机关近半的效率。这背后其实是有人设的局:先收买议员把手伸进危险物领域、再借苏晓杀人做文章、趁机绕过收容机构截留海上的危险物。

## 关系

- [苏晓](../苏晓.md)（副军团长）：被她与维克院长联名革职、坚持「严办」的下属。
- [维克院长](维克院长.md)：与她一同把持机构幕后的监察长。
- [收容机构](收容机构.md)：她作为财政总长执掌的钱袋子。
