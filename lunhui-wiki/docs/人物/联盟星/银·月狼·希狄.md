# 银·月狼·希狄

[联盟星](../../世界/任务世界/联盟星-死亡圣杯.md) 极南寒地的守护者，名为**希狄**，属**银·月狼**一族——**灭法者的盟友、「元素守卫者」**。苏晓的天赋任务 [「送别·安眠」](../../世界/任务世界/联盟星-死亡圣杯.md) 的目标，正是为这只以死亡之躯挣扎千年的月狼带来最后的安眠。

## 银·月狼一族 · 灭法者的盟友

- **月光的化身**：月狼一族生前爱洁，以**月亮、银白毛发、湖泊**为象征；傲气、独行、力量、敏锐是它们的代表。每只月狼都配一把**月光剑**（青色月光凝成的水晶大剑），如同灭法者的灭法之刃。全族**共七只、已全部陨逝**，希狄是最后的守望者。
- **与灭法者的盟约**：当年月狼一族与来猎古神的**灭法者**在此世相遇、并肩作战，结下盟约。灭法者一系没有驾驭月光的体质，否则其能力中「绝对有月光这一派系」——两族能力互通、情谊坚如磐石。这也解释了 [苏晓](../苏晓.md)（灭法者/元素守卫者）为何元素亲和力奇高。
- **千年镇深渊之孔**：自深渊之孔出现于此世，希狄便一直镇守极南寒地、轻易不离开，以一己之力压制那能「搅碎世界」的黑洞漩涡，守卫本世界的自然元素平衡。

## 死亡 · 被深渊之力支配

生前的希狄在本世界近乎**无敌**（其奥义「**辉光之月**」是人海战术的天然克星）。它并非败于正面交锋——[泰亚图大帝](泰亚图大帝.md) 借天外陨石带来的深渊线虫主体拖住它，待它灭掉线虫却重伤、又遭深渊之孔反噬濒死时，以人海战术围攻数年将它耗死，并夺走它封印深渊之孔的坚冰。死后它的残躯被**深渊之力**支配、意识狂乱如野兽，化身「灾祸」，吞噬曾守卫之物——但**它的残魂始终在拒绝深渊之力**（只用月光与灭法传承之力）。

## 与苏晓的决战（43卷39–42章）

侦测到的资料（因环境仅得22.5%）：生命值93.72%、月光之力42520/68000、深渊之力87000/87000，力/敏/魅约230/232/218（死亡高额削减），持**元素吞噬者/狼血战意/月光/元素眷恋/剑术宗师/月狼化**等技。战斗于极南寒地湖心岛：

- **月之青钢**侵蚀全场，苏晓以灭法者身份豁免，[阿姆](../阿姆.md) 持寒冰盾一照面即碎、侧腰被削去半圆，[巴哈](../剧毒魔鹰·巴哈.md) 在异空间被随之而入的火元素烤成焦炭。
- 血量跌破60%后**彻底月狼化**：人立而起、召来青色月光剑，力与美结合、剑术宗师直觉极强，接连贯穿苏晓身体、又靠**吞噬之核**吸木系元素3秒回满血打消耗。
- 月狼一剑**贯穿苏晓心脏**濒死；苏晓靠晶体层包心脏停跳自保 + 魂之丝缝合脏器 + 血之兽排淤血扛过必死，终以**斩碎怨灵小红（S-173灾厄铃铛怨灵）激活青影王加持斩龙闪**（月狼豁免青钢影真伤、却豁免不了青影王真伤）+ 血气压制，一刀刺穿月狼胸膛。巴哈一句「大狗」唤醒了它对一位已逝女灭法者的回忆，它按头嘶吼、痛苦挣扎。
- 临终它看到了天空的月光与手持长刀的灭法者，轻声「啊～，满足了」，安然闭眼。**月狼，已安眠。**

## 遗泽

- 苏晓获**世界之源+32.7%**（死多年削减后仍是巨款）、不朽级宝箱银色月光、[元素结晶](../../力量体系/装备道具.md)（捏碎幸运永久+3）。
- 随身戒指 [【银月之刃】](../../力量体系/装备道具.md) 在葬身地献肉后**成长**为一把精巧小刀（「割刃」增益·灭法者可无代价使用）。
- 苏晓临时觉醒的第三天赋 [【元素之王】](../../力量体系/天赋.md) 在此**突破极限成永久 SSS**。
- 苏晓从月光剑读出它的灵魂记忆，揭穿 [泰亚图大帝](泰亚图大帝.md) 弑月狼夺深渊之孔的真相。

## 关系

- [苏晓](../苏晓.md)（灭法者/元素守卫者）：与它同源盟友、却不得不亲手送它安眠的灭法者；战后为它下葬、插月光剑为碑、献肉喂养。
- [泰亚图大帝](泰亚图大帝.md)：借线虫与人海战术害死它、夺走深渊之孔的独裁者。
- 灭法者一系：它千年前并肩作战、结下盟约的战友（灭法时代已终结、月狼一族亦仅余它一只）。
