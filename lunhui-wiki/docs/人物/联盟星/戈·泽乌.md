# 戈·泽乌

[联盟星](../../世界/任务世界/联盟星-死亡圣杯.md) [临时同盟](../../世界/任务世界/联盟星-死亡圣杯.md) 五万多名老兵中约 300 名狙击手的**最强者**，在总攻中担任 [苏晓](../苏晓.md) 的贴身狙击护卫——凡可能威胁到苏晓的敌人，他都会一枪轰碎。

## 身份 · 能力

- **厥颅独耳的异乡人**：名字带浓重异邦风格，他是**厥颅人**（一个海岛小国）——那里的男性 16 岁要割下自己的左耳献给虚像神灵萨萨耶。异族、独耳、沉默寡言、年近四十仍单身，这些标签让他在军中饱受饿狼般的异样目光，也让他愈发沉默。
- **枪械宗师 Lv.51**：他并非靠 [战争领主](../../力量体系/称号.md) 加成才强——本身就是**专精于狙击枪的枪械宗师 Lv.51**（战争领主只能对他微量加成、无法让他突破）。他怀里那把两米多长、枪管手臂粗、遍布螺旋稳固槽的蓝火药狙击枪，与其说是枪，「更像一门可移动的狙击炮」。蓝火药武器精准感人，可在他手中就是一座能点名清除高价值目标的活炮台。

## 关系

- [苏晓](../苏晓.md)（库库林·白夜 / 临时同盟副指挥官）：他贴身护卫的总指挥官（专职轰碎一切威胁苏晓的敌人）。
