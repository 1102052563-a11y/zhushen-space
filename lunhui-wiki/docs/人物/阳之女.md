# 阳之女（血藤之母）

吞噬岛红土区**真正的幕后存在**——一株**体积巨大、盘踞整个红土区域地下**的神奇植物（红土区那片赤色土地正是因她而生），介于「动物与植物之间、偏向植物」，已不知存在了多少年。[亚森曼部落](../世界/生物万族/index.md) 世代将她供奉为「**血藤之母**」，而她自称「**阳之女·太阳的化身**」（讨厌「血藤之母」这个「愚蠢的称呼」）。

- **化身形态（八卷51章·后山古圣地）**：[巴尔](巴尔.md) 为断臂少年主持血藤仪式、植入生根的原生之种时，少年突然睁眼、燃起金色火焰、容貌体态变成一名**二十多岁的普通女人**（不算美也不丑）——这便是阳之女借仪式现身。古老石台在她脚下崩裂、长出**藤蔓王座**，细藤攀身缝合伤口，双眼如太阳之火般燃烧金焰。

## 亚森曼部落的真相 · 共生契约
- 亚森曼人当年作为「**看守者**」被虚空王者派驻吞噬岛，在红土区发现了她（血藤），双方先祖**达成合约**：把她的种子「**原生之种**」移植入体，亚森曼人即获**植物 + 火焰之力**（成为 [血藤战士](../世界/生物万族/index.md)）；**代价**是阳之女会**抽离所有血藤战士的生命力**（故血藤战士短命、归宿是「回归太阳」自爆）。
- 苏晓此前吃下的「**太阳恩赐果实**」就是阳之女的果实，**果核即原生之种**——通常她会自行回收，但因**畏惧巴哈姆翼龙的晶焰**而不敢收回那一颗，才让 [苏晓](苏晓.md) 捡了便宜。
- 仪式「**成功**」诞生老巴尔那种血藤战士，「**成功一半**」则诞生巴里那种**无痛觉、神情呆滞、残忍嗜血**的半成品（巴里日记里「我进行了仪式……意识正被某种东西侵蚀，那是一个全身金色火焰的女……」即指她）。

## 与苏晓的谈判（八卷51–52章）
她认得苏晓是「**灭法之影**」（[灭法](../力量体系/技能/index.md) 体质·见 [机制](../总览/轮回乐园机制.md)）——**发自骨子里讨厌灭法之影**：她最脆弱的本体曾被一名手持长刀的「灭法疯子」拔出、在手里甩了甩，只因她太弱才没被杀。

> **那名「灭法疯子」= [马文·华尔兹](马文·华尔兹.md)**（苏晓的启灵人·初代灭法之影）——八卷75章马文亲述：几百年前他来红土区找材料，把自称「神」、要他供奉的阳之女连根拔起甩几圈，**还带走了她体内九成以上的能量**（本想等她恢复再夺、被法师围攻没回去）。这才是阳之女**对灭法之影有阴影、不敢对苏晓动手**的真正根由。马文点拨苏晓「等够强可去她那夺能量·植物能量能滋养青钢影」。她先是高傲招揽（「效忠于我、饶你一死」），被苏晓一句「**太阳表面约 6000 度，你与太阳相比我只看到渺小**」噎住；苏晓更点破「你出来见我、没直接动手，说明你**有求于我**」，她遂卸下伪装谈正事：

- **她在畏惧「寒灾」**——「**黑色森林那些不死者要『出笼』了、怪物们都要复苏、寒灾即将到来**」；据她观察契约者一般在岛上停留十天后消失，而**寒灾五天后就开始**，初始阶段就足以致苏晓于死地（详见 [黑色森林·寒灾](../世界/任务世界/T12K星球-生存试炼.md)）。
- **为何找苏晓合作**：因他是「灭法之影」——**偏激但实力够强**（哪怕还在成长期）。她声称能**保证苏晓安全进入黑色森林**（苏晓的隐藏任务恰需进黑色森林）。此时乐园提示「**隐藏任务第一阶段『太阳部落』已完成、激活第二阶段**」。

## 合作条件 · 改造原生之种（八卷52–53章）
她**无法离开红土区**（这副化身只能维持几小时即崩溃），故不能亲入黑色森林，而是提出条件——**让苏晓带上她的人**：[巴尔](巴尔.md) + **另外两名契约者**（她早已联合的契约者·苏晓略头疼但接受，因生存试炼无猎杀任务、不会暴露）。

- **改造原生之种**：黑色森林必须靠原生之种「**门票**」续命——她把苏晓的黑色原生之种**改造成淡红、生翠绿枝芽的形态**：进林后随身携带、**每两小时捏开一颗**、每颗护身约两小时，**十颗用光前必须离开，否则就是埋骨之地**。讨价还价后给了苏晓**十五颗**（她强调改造原生之种绝不简单）。
- **「她的人」= [小迷糊](小迷糊.md) + [无伞兄](无伞兄.md)**（八卷60章）：阳之女联合的另两名契约者，脖颈各攀一根**控制藤蔓**（强逼他们进黑色森林·她**本就打算让二人去送死**）。她还让无伞兄带一颗**未处理的原生之种**——那是给黑色森林边缘 [巨人](../世界/生物万族/index.md) 的过路费（换黑森林地图）。她**也曾召唤过巴尔**、被巴尔拒绝。
- **对巴尔的掌控**：她在巴尔心中「就是神、无法反抗的存在」（巴尔表面恭顺称「阳之女神」、领命「谨慎使用」原生之种）——但出洞后巴尔私下吐露真心：「**弑神是不错的选择，现在还太早，先度过寒灾再说**」（详见 [巴尔](巴尔.md)）。
