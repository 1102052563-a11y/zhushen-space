# 狩之影（招财猫 · 苏晓的寻宝召唤物）

[苏晓](苏晓.md) 在 [海蓝星·热雨林之森](../世界/任务世界/海蓝星-世界争夺战.md) 收服的**第二只随从 / 召唤物**——原是雨林里的极地首领生物 **维斯蒙丛林猫「狩之影」**（详见 [生物·万族](../世界/生物万族/index.md)）：高约 1.4 米、连尾近 5 米的黑白虎纹巨猫，**敏捷 99（真实·本世界极值）**、速度快到苏晓的直感都难捕捉。最致命的隐藏能力是 **双生（Lv.62）**——可暗绿光一闪**缩小到家猫大小**（全属性 −70%、体型 −96%）。性格桀骜、极其记仇。

## 从死敌到「招财猫」（十九卷69–84章）

它被苏晓砍掉一块肉后死缠不放、全天跟踪（速度快到苏晓不开猎魔时刻都奈何不了它）——最后**踩进苏晓预设的二次纤维丝陷阱被吊起**、又被苏晓连抽几个大嘴巴子 + 喂炼金炸弹收服。精通炼金生物学的苏晓没杀它（单属性达世界极值太可惜）、而是用**柱间细胞 + 虚空生物遗传因子（带空间特性）**改造它（耗约 12 万乐园币）、保留它的主观意识却以柱间细胞波动**牢牢控制**（它逃出去也无法恢复原体型）、逼它签下**召唤契约**（苏晓魅力破 20 多出的召唤位）。

- **定位 = 寻宝「招财猫」**：苏晓**不让它战斗**（它宁死不为敌对过的人拼命）——而是利用它鬼魅的速度 + 极强搜寻能力 + 雨林本能，让它在每个衍生世界里**自由活动、专门寻找稀有资源**（矿产 / 稀有植被 / 增益果实，甚至潜入剧情人物家中偷装备 / 财物 / 任务道具），世界结束前带着搜来的物资与苏晓会合——为每个世界增收。布布汪、阿姆各有战斗 / 侦查之责，正好缺这么个寻宝的。它一不老实就被「[布布](布布汪.md) 揍它」（布布汪 + 阿姆的「布姆组合式陀螺暴风旋转」）治得服服帖帖。

## 召唤物「贝妮」(十九卷89章)

返园后查看召唤物详情，它得了个名字「**狩之影·贝妮**」（布布汪嫌「真娘炮」）。改造后部分战斗能力被剥离、换成更强的探测 / 逃生：

- **属性（召唤生物·常保家猫「双生形态」）**：法力 500·力 72 / 敏 **99（真实）** / 体 80 / 智 50 / 魅 50（双生形态下力 / 敏 / 体 / 智削减·魅力增益）。
- **能力**：猫之本能（神经反射 +65% / 动态视力 +60%）、野性本能（捕捉隐身 / 潜行）、**双生**（切换猫 / 森豹形态）、**珍宝之眼 Lv.50**（探测 10 公里内高价值物品·6% 几率·冷却 24h）、**贼猫 Lv.40**（猫形态奔跑 +300%）。它偏爱常保家猫体型（属性弱但反应更快），与布布汪、贝妮的「喵 / 汪星语」竟能互通互喷。苏晓给一狗一牛一猫各发 1 万乐园币零花钱。

## 关系
- **[苏晓](苏晓.md)**：从死敌到主人（被炼金改造 + 召唤契约收服为寻宝召唤物）。
- **[布布汪](布布汪.md) / [阿姆](阿姆.md)**：苏晓的另两只随从（曾与它打成绒毛横飞·如今负责「揍它」让它老实）。
