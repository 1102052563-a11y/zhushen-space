# 暗狱世界群像（世界人物索引）

[暗狱世界·狩猎之夜](../世界/任务世界/暗狱世界-狩猎之夜.md) 中 [苏晓](苏晓.md)（伪装成霸主级「神灵猎人·库库林」）遇到的守望公会、契约者与暗魔。主要角色已拆为独立页，下按分组索引；龙套见文末。

## 契约者

- [圣鸳](暗狱世界/圣鸳.md)（魅力系御姐 · 撞上「库库林免疫魅力」的铁壁 · 莫雷小队交涉担当）。
- [莫雷（原始大陆页）](原始大陆/莫雷.md)（战斗天使 · 天启乐园 · 又来「躲某个人」·却把库库林当剧情人物投了庇护屋）。

## 守望公会 / 本土 / 猎人

- [南瓜·塔·巴尔德](暗狱世界/南瓜·塔·巴尔德.md)（三神灵猎人之一 · 缺心眼却强大 · 苏晓临时队友）。
- [蒂法尼](暗狱世界/蒂法尼.md)（女猎人 · 液体化不死 · 记忆剥离 · 瓶装随苏晓行动）。
- [警探·道格拉斯](暗狱世界/警探·道格拉斯.md)（索多市警探→「暗界」幕后主脑 · 已亲取遗物）。
- **鬣狐·亚斯（三神灵猎人之一 · 叛徒 · 未露面）**：守望公会三神灵猎人的第三位——「打酱油」的实为**守望公会内应叛徒**（地位高但不知太多秘密·派银发老猎人剧毒短刀暗杀苏晓）；苏晓 + 塔·巴尔德推断出他、决定先斩后奏。

## 暗魔 / 幽魂

- [豪鬼](暗狱世界/豪鬼.md)（四阶段暗魔 · 硬闯守望公会总部 · 手刃 157 老猎人 · 比泰密莎强一筹）。
- [厄梦](暗狱世界/厄梦.md)（暗狱罕见幽魂 · 最强梯队 · 率 10 暗魔到雾城抢遗物）。
- **暗魔（灰 / 红 / 黑 / 有称号四阶段）**：从暗狱涌出的七阶战力智慧怪物——皮色越深越强越聪明（灰皮憨、黑皮狡诈）；四阶段的有专属称号（屠杀太多平民被公会单列）。
- **幽魂**：稀少·寻常攻击无效·惧灵魂伤害（+150%~300%）·可悄无声息控制任何人（如「胖小丑」= 天赋者被幽魂侵入放大天赋 + 操控·被巴哈战·「用屁点火」喷它）。
- **暴食暗魔（三阶段 · 殁）**：把巴特少校一家连尸吃光、蹲案发现场偷袭猎人的乖戾暗魔（胸口挂 16 银 1 黑 2 白猎人牌·杀过十几名猎人）——被苏晓一见「神灵猎人」牌吓逃、腰斩斩杀（得 62 声望 + 声望商店开启）。
- **红女妖（三阶段暗魔 · 有称号）**：能伪装成貌美女性、专杀男性的雌性暗魔——莫雷小队接下的一等超凡事件（Lv.65）。

## 契约者 / 其他（补）

- **圣堂冒险团 · 团长阿德利 / 兰斯（天启 · 165 人）**：天启乐园大型冒险团（守望公会阵营·被公会当「精英炮灰」）——团长**阿德利**（如「有情有义」的阿德利企鹅·憨式坦诚交涉「交朋友要坦诚」·硬着头皮接下 31 个委托、被 153 个吓退）；感知系**兰斯**先来投石问路。莫雷小队高效完成「红女妖」（+7 声望）。
- **融合之根源 / 魂 / 暗界**：暗狱深处那颗遍布触手、见啥吞啥的**黑色巨口**=暗狱吞世界的本体（意识「**魂**」维系它稳定、无视世界壁障）；「**暗界**」= 暗狱吞世界后的新世界（[道格拉斯](暗狱世界/警探·道格拉斯.md) 所命名）。暗魔 / 幽魂势力**并非铁板一块**——只有一部分拥护豪鬼 / 魂 / 道格拉斯。
- **守望遗物（守望公会创始人遗物）**：守望公会创始人 / 第一位猎人的猎人牌·世界之力最多（藏晨曦市·位置无人知·暗狱势力夺到它可能就够了）。另有圣猎 / 灵响 / 黄昏等神灵遗物（各 1120~1280 声望·苏晓收集当诱饵 + 换声望）。
- **老猎人·亚当（狩猎 34 年 · 被融合之根源侵蚀 · 殁）**：常年黑风衣、锯刃弯刀 + 飞刀 / 弩箭 / 生命回流的老猎人——本被求「安眠解脱」，却被「融合之根源」偶发性侵蚀成漆黑暗魔化，被苏晓斩杀（揪出根源线索）。
- **警官维娜（道格拉斯老友之女 · 殁）**：首次接触超凡的年轻女警官——被觉醒真身的 [道格拉斯](暗狱世界/警探·道格拉斯.md) 所杀。
- **银发叛徒老猎人（守望公会内应 · 殁）**：受某守望公会高层指使、在去雾城的火车上假意送牌行剧毒短刀暗杀苏晓，失败自刎——揭出守望公会里有「那个人」。
- **美女蛇情侣（圣域 · 魅力系）**：一对圣域情侣契约者（女方魅力系·浅红蛇眸）——接下「遗物收回」去邻市曼斯市（大概率撞上豪鬼）。
- **亚莉克希亚 / 失控老猎人**：狩猎 34 年即将失控的黑衣老猎人求「猎人·亚莉克希亚」到市外古堡杀他解脱、传承衣钵（「心火未熄，必会迎来黎明」）。
- **警官维娜（道格拉斯老友之女）**：首次接触超凡事件、被暗魔气息熏到作呕的年轻女警官。
- **臃肿男（前庇护屋管理人）**：金属臃肿铠甲、爱喝假酒的老猎人——把「庇护屋（十六）」交接给苏晓、叮嘱对新人友好些。
- **除虫公司**：守望公会的情报部门（白橡胶衣 + 防毒面罩·处理超凡现场 / 强制疏散·听说「那位」库库林会来就火速撤走）。

## 关系
