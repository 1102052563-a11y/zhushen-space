# 暗狱世界群像（世界人物索引）

[暗狱世界·狩猎之夜](../世界/任务世界/暗狱世界-狩猎之夜.md) 中 [苏晓](苏晓.md)（伪装成霸主级「神灵猎人·库库林」）遇到的守望公会、契约者与暗魔。主要角色已拆为独立页，下按分组索引；龙套见文末。

## 契约者

- [圣鸳](暗狱世界/圣鸳.md)（魅力系御姐 · 撞上「库库林免疫魅力」的铁壁 · 莫雷小队交涉担当 · 决战差点被苏晓「顺手」秒掉）。
- [莫雷（原始大陆页）](原始大陆/莫雷.md)（战斗天使 · 天启乐园 · 又来「躲某个人」·却把库库林当剧情人物投了庇护屋 · 决战想跟血神合作被坑飞）。
- [阿德利](暗狱世界/阿德利.md)（天启 · 圣堂冒险团团长 165 人 · 受道格拉斯 / 魂收买背刺 · **率 105 人围杀苏晓被死寂烬灭团灭·殁**）。

## 守望公会 / 本土 / 猎人

- [南瓜·塔·巴尔德](暗狱世界/南瓜·塔·巴尔德.md)（三神灵猎人之一 · 缺心眼却强大 · 苏晓临时队友 · **护香夫人破阵界后在头盔下战死·殁**）。
- [鬣狐·亚斯](暗狱世界/鬣狐·亚斯.md)（三神灵猎人之一 · 被诬叛徒实为「猎人存续论」· **被魂吞为古神火种 / 被塔·巴尔德斩杀·殁**）。
- [蒂法尼](暗狱世界/蒂法尼.md)（女猎人 · 液体化不死 · 记忆剥离 · 被阿德利泼暗魔血袭击后逃走 · 遣白衣女鬼传话）。
- [警探·道格拉斯](暗狱世界/警探·道格拉斯.md)（索多市警探→「暗界」幕后主脑→**半个世界之子·被苏晓斩首完成掠天惊澜·殁**）。

## 守望公会四高层（守护者）

- [香夫人](暗狱世界/香夫人.md)（外交 / 药剂 · 剧毒之身 · **成人造世界之子破阵界子体结构** · 残局仅存三高层之一）。
- [蝮蛇](暗狱世界/蝮蛇.md)（人员调动 / 情报 · 腹黑冷血 · 掰断指癖 · 被阿德利袭击失一臂 · 残局仅存）。
- [图书馆·康芒斯](暗狱世界/图书馆·康芒斯.md)（219 岁活体图书馆 · 献魂运剥离仪式造世界之子 · 残局仅存）。
- **戈（财政 · 大胡子壮汉 · 殁）**：四高层之一——为造人造世界之子主动试植气运之石，当场爆体身亡（「除了那老年痴呆，我们三个都是能被替换的人」）。
- **金瑟（守护者 · 殁）**：守望公会第五位高层——世界公证前夜死于幽魂近乎自杀式的袭击。

## 暗魔 / 幽魂 / 古神

- [血神·格赫普斯](暗狱世界/血神·格赫普斯.md)（**本世界最终 boss** · 被魂复活的陨灭星古神 · 世界之核阵界无敌 · 蚀血 / 血共鸣 / 命种复生 · **被苏晓斩尽命种·殁**）。
- [魂](暗狱世界/魂.md)（暗狱三巨头之一 · 古神信徒 · **复活血神的真正主谋** · 血神死后自爆·殁）。
- [豪鬼](暗狱世界/豪鬼.md)（四阶段暗魔 · 硬闯守望公会总部 · 手刃 157 老猎人 · 比泰密莎强一筹 · **躯体被魂夺作血神神躯·殁**）。
- [厄梦](暗狱世界/厄梦.md)（暗狱罕见幽魂 · 最强梯队 · 率 10 暗魔到雾城抢遗物 · **被灵魂锋刃斩杀·殁**）。
- **暗魔（灰 / 红 / 黑 / 有称号四阶段）**：从暗狱涌出的七阶战力智慧怪物——皮色越深越强越聪明（灰皮憨、黑皮狡诈）；四阶段的有专属称号（屠杀太多平民被公会单列）。
- **幽魂**：稀少·寻常攻击无效·惧灵魂伤害（+150%~300%）·可悄无声息控制任何人（如「胖小丑」= 天赋者被幽魂侵入放大天赋 + 操控·被巴哈战·「用屁点火」喷它）。
- **暴食暗魔（三阶段 · 殁）**：把巴特少校一家连尸吃光、蹲案发现场偷袭猎人的乖戾暗魔（胸口挂 16 银 1 黑 2 白猎人牌·杀过十几名猎人）——被苏晓一见「神灵猎人」牌吓逃、腰斩斩杀（得 62 声望 + 声望商店开启）。
- **红女妖（三阶段暗魔 · 有称号）**：能伪装成貌美女性、专杀男性的雌性暗魔——莫雷小队接下的一等超凡事件（Lv.65）。

## 契约者 / 其他（补）

- **圣堂冒险团 · [阿德利](暗狱世界/阿德利.md) / 兰斯（天启 · 165 人）**：天启乐园大型冒险团（守望公会阵营·被公会当「精英炮灰」）——团长 [阿德利](暗狱世界/阿德利.md)（详见其页·率 105 人围杀苏晓被死寂烬灭团灭）；感知系**兰斯**先来投石问路。团内核心：**星妹 / 十字星**（意念子弹·眉心白十字纹身·信念越强威力越大·全团最强·围杀失败自尽·殁）、奶妈 **柏柏莎**（天使降临·中断则对友军造 27% 真实治疗伤害的「毒奶」·殁）。莫雷小队高效完成「红女妖」（+7 声望）。
- **暗影小队 · 七凶徒（莱茵各监狱重刑犯 · 破暗狱之门）**：香夫人 / 蝮蛇从莱茵各监狱调来破暗狱之门的 7 名超凡重刑犯（合计刑期 56 万年·莱茵想杀又舍不得杀的人）——队长 **死灵·尤尤尔**（绿瞳·最强最阴·擅破除超凡机关·十公分小号本体探路）、**沉默者·索尔兹伯里**、**纽扣·桑尼**（女·想坑队友逃跑未遂）、**大嘴·基特**（宽舌·殁·冲门时被机关银色流体吞噬）、人猿·洛克奇、巨山·列科夫、缝合人·沃莫里。尤尤尔破尽机关、苏晓一脚踹碎门。
- **融合之根源 / 魂 / 暗界**：暗狱深处那颗遍布触手、见啥吞啥的**黑色巨口**=暗狱吞世界的本体（意识「**魂**」维系它稳定、无视世界壁障）；「**暗界**」= 暗狱吞世界后的新世界（[道格拉斯](暗狱世界/警探·道格拉斯.md) 所命名）。暗魔 / 幽魂势力**并非铁板一块**——只有一部分拥护豪鬼 / 魂 / 道格拉斯。
- **守望遗物（守望公会创始人遗物）**：守望公会创始人 / 第一位猎人的猎人牌·世界之力最多（藏晨曦市·位置无人知·暗狱势力夺到它可能就够了）。另有圣猎 / 灵响 / 黄昏等神灵遗物（各 1120~1280 声望·苏晓收集当诱饵 + 换声望）。
- **老猎人·亚当（狩猎 34 年 · 被融合之根源侵蚀 · 殁）**：常年黑风衣、锯刃弯刀 + 飞刀 / 弩箭 / 生命回流的老猎人——本被求「安眠解脱」，却被「融合之根源」偶发性侵蚀成漆黑暗魔化，被苏晓斩杀（揪出根源线索）。
- **警官维娜（道格拉斯老友之女 · 殁）**：首次接触超凡的年轻女警官——被觉醒真身的 [道格拉斯](暗狱世界/警探·道格拉斯.md) 所杀。
- **银发叛徒老猎人（守望公会内应 · 殁）**：受某守望公会高层指使、在去雾城的火车上假意送牌行剧毒短刀暗杀苏晓，失败自刎——揭出守望公会里有「那个人」。
- **美女蛇情侣（圣域 · 魅力系）**：一对圣域情侣契约者（女方魅力系·浅红蛇眸）——接下「遗物收回」去邻市曼斯市（大概率撞上豪鬼）。
- **臃肿男（前庇护屋管理人）**：金属臃肿铠甲、爱喝假酒的老猎人——把「庇护屋（十六）」交接给苏晓、叮嘱对新人友好些。
- **除虫公司**：守望公会的情报部门（白橡胶衣 + 防毒面罩·处理超凡现场 / 强制疏散·听说「那位」库库林会来就火速撤走）。
- **亚莉克希亚（上任世界之子 · 老猎人亚当之徒 · 殁）**：戴白乌鸦面具、身材偏瘦的资深女猎人——**本世界上一任「受世界青睐之人（世界之子）**」，是老猎人·亚当想传承衣钵的弟子；却在魂的计划中**死在师父亚当手里**（苏晓亲见其半个头颅）——她一死、气运之力立即转嫁给濒死的道格拉斯（半个世界之子）。印证「七阶世界之子不与之同行就极难察觉真身、且最怕至亲之危」。
- **阿蜜雅（莱茵王国高官嫡女 · 事件代表 · 在世）**：来守望公会「镀金」的年轻贵族女官——被香夫人指派、带批文去各监狱把 7 名重刑凶徒押来（一路强忍干呕、崩溃）；因表现被委任为莱茵王国在本次事件中的代表、地位水涨船高。
- **鸟嘴医生（香夫人助手 · 东古联邦恶名医生 · 在世）**：戴乌鸦嘴面具的医生——医术高超（猎人断手断脚只要带回残肢都能治），但**只能治猎人、治普通人 100% 治死**。协助押送凶徒、剥离并暂存道格拉斯体内共生幽魂（造人造世界之子的关键一步）。
- **白衣女鬼（弱小善良的幽魂 · 在世）**：喜欢吓人、却夜间帮迷路平民指路、从没做过坏事的弱幽魂——受逃走的蒂法尼所托，深夜到庇护屋给苏晓传话（差点被顺手斩了、赶紧亮银色脚链表明身份），带走苏晓给蒂法尼的药剂。
