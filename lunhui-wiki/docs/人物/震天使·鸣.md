# 震天使·鸣（精英死灵 · 无敌防御）

[希亚圣殿](../世界/地点/index.md) 「**震天殿**」的守卫——比 [长刀使·空](长刀使·空.md) 更强的精英死灵：身高约 **5 米**、秃头、赤膊、下身大裤衩、满身扎结肌肉，手持两根 3 米长的「**鼓锤**」。使徒之眼**只能侦得它的名字**（被未知能量隔绝、其余全无·苏晓首次遇到这种情况）。

## 战鼓增益 + 近乎无敌的防御（十七卷60–62章）

它不近身、而是远程**敲击虚空中的「无形战鼓」**：

- **震天（音波）**：穿透能量盾、造 40 真伤 + 105 震荡 + 眩晕判定（苏晓豁免眩晕）。
- **战鼓增益**：它敲的是「战鼓曲」、越激昂则**全体死灵士兵实力涨到 2~3 倍**——若与 500 死灵军配合本会极强（被苏晓开局一枪 + 抛阿姆缠住打乱）。
- **虚无鼓锤（近战）**：锤中即 **100 灵魂伤 + 80 真伤 + 120 震荡 + 高强度痛苦**——布布汪被锤了屁股一锤就惨嚎着一瘸一拐逃，阿姆挨多锤后满头大包、间歇翻白眼。

它**力量 / 敏捷 / 战斗经验都辣眼睛、唯防御「迷之无敌」**：苏晓的 **寂灭公爵 +13** 轰它脑门竟毫发无伤（蚁王挨这一枪都得碎脑壳）；青钢影也噬不进、界断线勒不入——全因体表一层网格状灰色能量护盾。

## 破防 → 被斩杀（十七卷62–63章）

苏晓发现它**仍需呼吸**（是「活着」的悠久生物），遂用界断线 + 阿姆把它生擒、按地上喷**红色毒雾**（9 罐·喷个不停）；它闭气苦撑约 1 小时后无力、护盾终于淡去破碎，苏晓一刀贯心、几刀斩要害**将它杀死**（掉**[远古死灵精粹](../力量体系/消耗品材料/index.md) ×5**）。真相：它那「无敌护盾」的能量**并非自身、而是连着震天殿偏殿的一处灰色能量柱装置**——布布汪潜进偏殿啃坏了那东西，护盾才消失。

## 关系
- **[苏晓](苏晓.md)（铁之手）**：闯震天殿的入侵者（用毒雾耗破它的无敌护盾后斩杀）。
- **[长刀使·空](长刀使·空.md)**：同陵墓的守卫死灵（仅一墙之隔）。
- **[死灵贤者](死灵贤者.md)**：偏殿能量柱被毁后唤醒的中立石像。
