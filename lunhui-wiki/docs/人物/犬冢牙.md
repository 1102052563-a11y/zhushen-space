# 犬冢牙

[火影忍者世界](../世界/任务世界/火影忍者.md) 木叶忍者、犬冢一族、原著「十二小强」之一——与忍犬**赤丸**搭档作战（牙通牙 / 人兽混合变身·叁头狼）。属 [夕日红](夕日红.md) 的追击小队。

- **战死（九卷67章）**：追击 [苏晓](苏晓.md) 时，他与赤丸（化人形）双双「牙通牙」冲锋，被苏晓 200 法力「傲歌」能量盾挡下；苏晓一刀**斩杀忍犬赤丸**（犬冢一族失忍犬便废了 95% 忍术）→ 再一刀贯穿犬冢牙小腹、施**影刺**。恰在此时 [日向雏田](日向雏田.md) 的「八卦空壁掌」把苏晓击飞——**体内的影刺被硬生生拖出**、掏空犬冢牙内脏三分之二，犬冢牙惨死（雏田「间接杀了队友」）。苏晓杀他得**世界之源 +4.6%（→36.2%）+ 紫宝箱**——印证「实力足够，原著十二小强乃至主角都可杀」。
