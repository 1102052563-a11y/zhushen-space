# 黑蜂（天启乐园最强 · 世界之核持有者）

[海蓝星世界争夺战](../世界/任务世界/海蓝星-世界争夺战.md) 中**天启乐园方最强的契约者**——31 岁男性、双臂为金属构造、声音带金属质感。**空间系能力**（可瞬移、化漩涡来去），并能**召唤未知品种的黑色蜜蜂**（带烈性剧毒·疑似元素类剧毒）。喜怒无常，却不会丧心病狂到袭击己方契约者。爱好：极辣食物、站在高处、26~30 岁成熟女性（[所罗门](所罗门.md) 在战争频道置顶的情报）。上一场世界争夺战中、黑蜂所在的世界正是天启乐园方获胜。

## 持世界之核（十九卷52章）

一支天启乐园佛系小队在西斯巴宁的及路矿山挖矿、意外挖出**世界之核**（拳头大透明晶体、看几秒会眩晕），不敢沾手便交给黑蜂。世界之核现机制：持有方须在 15 天内把它送到「**极地·热雨林之森**」（蓝海星中心）形成世界坐标、守护成功即获胜——天启乐园由此成为持核方、也成了轮回 / 圣域两方共同的攻击目标。黑蜂自信不虚轮回与圣域，计划**联合圣域乐园先解决最棘手的轮回乐园、再分胜负**（无奈格雅大军压境、圣域方一时离不开太阳王国）。他坐镇西斯巴宁鲁贝城——这也成了 [苏晓](苏晓.md) 等轮回乐园契约者夺核的目标地。

## 「一神带多坑」的阴谋 + 战死（十九卷55–61章）

黑蜂是召唤师 + 空间系法师的结合体（只能召一种生物：**元素蜂**·靠吞食强敌渐强），有胆有谋。世界争夺战中他**确认了轮回乐园方的任务惩罚 = 强行处决**（生擒七名轮回契约者拷问、前六人宁死不说·第七人坦言「强行处决」）——明白了这群人「无退路、不胜即死」才如此疯狂，圣域乐园中期惩罚也变强行处决。于是他**骗了三方**：把轮回、圣域都引来鲁贝城死磕、又把己方契约者当「拖住轮回的坑」骗来，自己想趁乱带世界之核空间瞬移去热雨林之森、独吞「一神带众坑」的夸张奖励。

可惜算计落空——[青魇](青魇.md) 看穿一切设局：[人民教师](人民教师.md) 无视其蜂毒屡次锤他半死，[苏晓](苏晓.md) 用「初级空间禁锢」封死方圆 50 米空间让他无法瞬移逃走，最终他被苏晓、人民教师、[咕噜](咕噜.md)、[所罗门](所罗门.md)、柑橘妹**五人合围**——柑橘妹一脚踢碎其金属臂、所罗门暗元素枯萎其腿、咕噜匕首破脊椎、人民教师一拳轰碎头颅。**世界之核随即被布布汪从抢到核的柑橘妹手里「偷」给了苏晓**。

## 关系
- **天启乐园**：其方最强契约者（持世界之核·骗己方当「坑」）。
- **[人民教师](人民教师.md)**：屡次锤他半死的怒川怪物（其第二号仇人·最终一拳轰碎其头）。
- **[所罗门](所罗门.md)**：在战争频道置顶其情报的轮回乐园法爷。
- **[苏晓](苏晓.md) 等轮回乐园契约者**：因持核而成众矢之的（苏晓首站直奔其所在的鲁贝城夺核）。
