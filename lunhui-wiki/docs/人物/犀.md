# 犀

[全职猎人世界](../世界/任务世界/全职猎人.md)「[绅士同盟](../势力/index.md)」成员——身材壮硕、皮肤黝黑、头生一根犀牛般的犄角。**一招鲜「角撞」专精型**：其他能力都不行、唯独把「角撞」练到极致，死在这招下的契约者不计其数。

- **角撞 Lv.29（天赋技·主动）**：用尽全力冲锋、把全身力量集中到犄角撞击敌人——伤害 = 300 + 力量 ×12 + 50% 犄角攻击力（Lv.10 强力击飞 / Lv.20 冲锋加速）；耗 20% 生命、冷却 2 天、用后脱力 5 分钟。撞在体格单薄的契约者身上几乎一击秒杀。
- **靠老大续力连撞 · 战死（十卷9–10章）**：炮台伏击 [苏晓](苏晓.md) 时，他本来角撞每战只能用一次，却被老大 [红色小丑](红色小丑.md) 用紫色能量**连续续力、连撞三次**（撞飞苏晓两次、破其能量盾、令苏晓呕血）。但第二次被苏晓**界断线缠住犄角**反带踉跄、第三次撞上苏晓的**次级无敌护盾**力竭——苏晓趁机一刀斩出，**斩下他半个脑袋**。死后灵魂被红色小丑祭献以激活恶魔血统。
