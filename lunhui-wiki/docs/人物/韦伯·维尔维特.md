# 韦伯·维尔维特

第四次圣杯战争中 Rider [征服王·伊斯坎达尔](征服王·伊斯坎达尔.md) 的**御主**——一名**面色白皙、声音弱气**的少年魔术师。站在人高马大的征服王身旁，活像个「**王妃**」（虽是男性·「这么可爱当然是男孩子」）。乘征服王战车赶到英灵齐聚现场时被颠得直喊「我要吐了」，又被 [苏晓](苏晓.md) 凶暴的气势吓得双腿发颤——全靠征服王挡在身前护着。剧情存在感主要依附于其英灵。

- **被迫用光令咒 · 死里逃生（十一卷60–61章）**：苏晓为清令咒找上他俩的乡下藏身处、以他的小命为筹码逼谈判——他在 [征服王](征服王·伊斯坎达尔.md) 劝说下选择信任、**用掉全部 3 枚令咒**（第三枚卡壳半天、憋出「你要在这次圣杯战争中活下去」），换得苏晓如约放生。一番生死惊吓后他暗暗萌生「**我也要成为征服王那样霸气说一不二的人**」的志向。

## 关系
- **[征服王·伊斯坎达尔](征服王·伊斯坎达尔.md)**：他的英灵（君臣相得）。
- **[苏晓](苏晓.md)**：英灵齐聚现场被其气势震慑。
