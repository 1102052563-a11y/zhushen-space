# 卡门（圣愈教会圣议员 · 苏晓的接引人）

[暗鸦星](../世界/任务世界/暗鸦星-暗鸦纪元.md) [圣愈教会](../势力/index.md) **圣公会十二圣议员之一**——白袍、严重驼背、只剩两颗门牙的老头，杵着一根其实是武器的金属拐杖。处事极老辣：一见 [苏晓](苏晓.md) 就察觉到他浓重的血气与危险（直呼「世上原来真有这种怪物」），却**不阻拦、反而直接收他入会成巫猎人**——「诚心来投欢迎、图谋不轨也无所谓·把威胁放在眼前总比让它处于暗处好」。他是「同类相吸」识破苏晓的存在（他自己也是类似的「怪物」）。自任苏晓的**接引人**。

## 暗中观察苏晓

他安排孪生姐妹 [雪莉 / 雪伊](卡门.md) 监视苏晓（详见下），判断苏晓「想探索女巫的秘密、目标是成为血猎人」——他不忌惮（「想探索秘密并不危险、更何况是关于女巫的秘密」）。苏晓一口接下全部委托（含五阶女巫 [摩利莎](摩利莎.md) 那张 7000 金币的）把他和雪伊都惊到。

!!! note "孪生姐妹 雪莉 / 雪伊"

    圣愈教会的一对孪生姐妹：**雪莉**（委托处前台·先天女神范·温和）与 **雪伊**（图书馆管理员·跳脱女神经·话痨）——长相相同性格相反。雪伊在图书馆一边「陪」苏晓看书一边替卡门观察他（被苏晓五花大绑塞桌下）。两人是卡门安插观察苏晓的眼线。

## 破例擢升红猎人（二十一卷45章）

苏晓覆灭西港、斩杀巫女王 [摩利莎](摩利莎.md) 归来（西港是圣愈教会鞭长莫及的「毒刺」），卡门大喜——巫猎人本有三年考核期、但他**破例直接擢升苏晓为红猎人**（红宝石指环·可自由出入图书馆查机密 + 用阵营商店）。他坦言不信苏晓来自北境、但「以整个西港 + 摩利莎的命作投名状」足够让他冒险担责：「我们能信任你吗——这要你自己判断……如果想判出圣愈教会，记住先杀掉我这老头子」（成红猎人没漂亮话、只把丑话说前头·一旦苏晓反叛、接引人卡门要豁命去猎杀他）。

## 「夕阳红敢死队」赴南部（二十一卷50–58章）

雪伊力促苏晓猎 [摩伊·诺拉](摩伊·诺拉.md) 后，卡门亲率一支**四名圣议员级老巫猎人**组成的敢死队（[布布汪](布布汪.md) 戏称「夕阳红小队」）深入南部、去尘泥祭坛破 [厄运](厄运.md) 的身体、为苏晓杀诺拉开窗口。这些老头都已培养出继承人、不惧死亡（「死在南部是职责、活着回来是血赚」），战法老辣（用一种「化身」之力·过度则脸现纹路至太阳穴即大限）。四人都是被女巫命名的狠角色：

- **剔骨人·卡门**（瘸子·综合型老阴哔·杖鞭〔银纹〕）；
- **溅血熊·约瑟亚**（独眼·正面硬怼型·扛门板霰弹炮·**已自尽**——黎明之门突围中「化身」用到太阳穴极限、举枪自尽善后）；
- **破膛·克洛菲勒**（醉鬼·侦查 / 器械 / 爆炸物 / 暗杀·黄纹·苏晓赎美洛狄时见过的醉酒看守）；
- **女巫坟·冈特**（最狠·远程弓弩 / 拷问专家·女巫宁死不落其手·已患老年痴呆、握枪时才清醒）。

教皇亲自送行。黎明之门遇百名女巫伏击，靠苏晓 + 布布汪音响诈军突围（约瑟亚阵亡）。

**尽数殉职（二十一卷65章）**：尘泥祭坛一战，卡门、克洛菲勒（醉鬼）、冈特（女巫坟）三人**任「化身」纹路蔓延至太阳穴、化身成恶鬼**（头脸血肉模糊），冲进祭祀厅杀光厄运的护卫、破坏 [厄运](厄运.md) 的身体使其沉寂——为苏晓杀 [摩伊·诺拉](摩伊·诺拉.md) 开出窗口。完事后卡门全身骨骼寸寸崩裂而亡（弥留间忆起身世：农夫之子帕特里克·卡门·幼年家人变怪物啃他成瘸子·立誓杀三百女巫报仇）。夕阳红敢死队**全员殉职**——「巫猎人代表了什么」，他们用命作答。雪莉得讯落泪、却仍递出下一张悬赏令：故人已逝、猎巫不止。

## 夕阳红敢死队四人（结局）
- **剔骨人·卡门**（瘸子·综合型·**殁**·骨骼崩裂）；
- **溅血熊·约瑟亚**（独眼·**殁**·黎明之门突围举枪自尽）；
- **破膛·克洛菲勒**（醉鬼·**殁**·尘泥祭坛化身耗尽）；
- **女巫坟·冈特**（老年痴呆·最狠·**殁**·尘泥祭坛化身耗尽）。

- **夕阳红敢死队**：约瑟亚（殁）/ 克洛菲勒 / 冈特三名老圣议员战友。
