# 爱德华·艾尔利克

[钢之炼金术师世界](../世界/任务世界/钢之炼金术师.md) 的**剧情主角**——国家炼金术师「钢之炼金术师」（个子矮·机械义肢）。

- **身世**：童年与弟 [阿尔冯斯](阿尔冯斯·艾尔利克.md) 一同进行禁忌的「人体炼成」复活母亲失败——他失去一条腿（后又以一臂为代价把弟弟的灵魂固定到铠甲上），从此当国家鹰犬、踏上找回身体之旅。父 [冯·霍恩海姆](冯·霍恩海姆.md) 抛妻弃子，故他憎恨父亲、改用母姓。
- **处境**：他与弟弟、父亲、师傅 [伊兹米](伊兹米·卡迪斯.md) 皆被 [瓶中的小人](瓶中的小人.md) 选为「**人柱**」祭品；他已见过与父亲同貌的瓶中小人（=主谋）。在瓶中小人面前炼金术会失效，故他来 [布里克兹要塞](../世界/地点/index.md) 找「带小熊猫的小女孩」学**炼丹术**。
- **与苏晓**：相性不和（爱德华「不杀」信念被苏晓视为拖后腿），故苏晓把主角队**排除出结盟选项**；但同为反瓶中小人，七卷数次同场（围观苏晓的「爆炎地狱」、在要塞印证国家秘密）。
