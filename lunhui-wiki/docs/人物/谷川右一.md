# 谷川右一

[火影忍者世界](../世界/任务世界/火影忍者.md) 雾隐之村的**体术型上忍**——在追杀 [苏晓](苏晓.md) 的队伍里遥遥领先众人、跑得比布布汪还快。曾以「**水遁·居丸**」（凝聚查克拉的粘稠水球）泼在地上让布布汪打滑、险些留住苏晓（被布布汪一记「360 度原地漂移」化解）。

- **被一枪轰杀**：苏晓倒骑布布汪、拎出重狙 **[蜘蛛女皇](../力量体系/装备道具.md)**——他威胁最大、又在全速追击中无法躲避高速狙击弹，被**一枪打穿胸膛**（惨白肋骨崩飞、胸口轰出大洞），是苏晓在火影世界的**第一个击杀**（**世界之源 +2.1%**·首次得世界之源·掉蓝色宝箱）。印证「火影强者身体强度普遍不高、苏晓的枪很克他们」。
