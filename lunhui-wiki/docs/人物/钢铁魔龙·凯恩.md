# 钢铁魔龙·凯恩

> ⚠ **身高两米五以上，身形高大却比例协调，满头灰白头发倒梳，一双暗金色竖瞳，全身金属战甲**——⚠ **风格沉稳威严的龙系战士，代表死亡乐园出战本次世界争夺战**;⚠ **⚠ 他的战力，是苏晓公认「综合实力能排进前三」的敌方小队队长**（⚠ 争夺前 4 块破碎本源时凯恩队理应来争，却被神父暗中阻止，苏晓才判断出神父在替违规者们铺路）。

## 🐉 大器晚成的魔龙

⚠ **他从「一阶」苦苦发展，一路熬到「绝强顶尖」，是名副其实的大器晚成**——⚠ **正因如此，眼下终于能和其他乐园绝强巅峰的代表同台交锋，就算沉稳如凯恩，也难免龙血燃起、心中激奋**;⚠ **⚠ 战斗形态**:

- ⚠ **⚠ 人形态**:⚠ **龙鳞大剑（可炸裂化为巨大龙爪）· 龙骑枪 · 龙鳞大盾 · 虚幻龙尾（横扫破碎大片空间、能把物品扫飞几千公里并靠龙鳞感测其位置）**;⚠ **⚠ 细密龙鳞可攀附体表防御、也可包裹物资。**
- ⚠ **⚠ 钢铁魔龙形态**:⚠ **高跃冲出大气层、调转方向化为「钢铁陨星」俯冲砸落**——⚠ **一下就把广袤的植物大陆中带砸出深不见底的沟壑、大峡谷化为超巨型陨石坑**（⚠ 落地时周边重力加大·⚠ 正因这大招波及范围极广，他从不带队友来、怕误伤自己人）。

## ⚔ 首轮大乱斗（54卷10章）

⚠ **在大峡谷争夺 300 颗灵魂晶魄、原初之核、初始源质、破碎本源时，苏晓、豪梼、凯恩、[泰莉德](魔镰·泰莉德.md)、神父五双眼睛互相牵制**——⚠ **凯恩并未与豪梼合谋，一度被豪梼的重拳、泰莉德的必中魔镰缠住**;⚠ **⚠ 他吞下神父下了手段的物资后当场喷血、一口咬断自己的右臂排毒**（⚠ 神父让谁碰物资都没好下场）:

- ⚠ **⚠ 连环钢铁陨星大招**:⚠ **他两度冲出大气层、化为钢铁陨石砸落，把泰莉德、豪梼都轰得七荤八素**——⚠ **苏晓却迎着陨落冲上去，一记「刃道刀·时」让俯冲的凯恩慢了一瞬**;⚠ **⚠ 就这短暂的缓慢让他毕生难忘**:⚠ **苏晓一记蓄满力的侧踢精准命中他的头颅，把他从 I 轨迹踢成 L 轨迹、砸入几公里外的高坡**（⚠ 他恍惚看到过世多年的二舅姥爷在对面朝他招手、注射永恒级恢复药剂才驱散幻觉、喷出一大口血）。
- ⚠ **⚠ 结局**:⚠ **首轮白夜队以 5 块破碎本源排名第一，凯恩队等七队均 0 块**——⚠ **凯恩心态跟得上、务实退走**（⚠ 与豪梼那种「要么活着赢一切、要么死在争夺战」的偏执不同，凯恩是踏实沉稳的对手）。

## 🔗 关联

- ⚠ **⚠ 与苏晓无私仇，是纯粹的阵营对手、绝强顶尖的同台竞技者**——⚠ **苏晓评价他是本次争夺战里少有的「正常」强敌**（⚠ 相比神父那种远远看着都精神污染的诡异、豪梼那种情绪越强越强的猛兽，凯恩是可以正面掰手腕的类型）。
- ⚠ **⚠ 他与苏晓、豪梼，被视为本次世界争夺战的「三强」。**

## 🐉 主动退场、格局与至强席位（54卷24章）

⚠ **领主大陆决战，苏晓灭尽领主势力时，凯恩队正在北境亡灵族那边——却主动退场**（⚠ 可见凯恩和亡灵族关系不错·⚠ 若他借亡灵族生死试探苏晓的态度，亡灵族会迎来灭顶之灾，苏晓并不介意灭掉亡灵族）;⚠ **⚠ 凯恩主动退场，等于打开了格局**——⚠ **既不想在必败境地摆出丑陋嘴脸，也给了亡灵族一个机会**（⚠ 让苏晓对他高看一眼）;⚠ **⚠ 「这个大器晚成的家伙，无论胜败，本世界后都晋升，后续在至强行列的强者中，必定有其席位。」**（⚠ 死亡乐园的阵容「有凯恩在，不一定稳赢，但一定不会团灭」·⚠ 第三轮死灵妹被吊后，苏晓与凯恩、泰莉德、豪梼几人的强度让绅士队违规者彻底自闭）。

## 🐉 深渊亲卫长血战与「带队活着离开」（54卷42–50章）

⚠ **他与水哥、泰莉德等被卷入深渊亲卫长死局**——⚠ **他被深渊亲卫长一手刺穿胸膛、撕下大半身躯、所有能力被「深渊压制」变灰，靠苏晓斩断亲卫长手臂+水哥水液才被拖走**;⚠ **⚠ 战中他还展开钢铁龙翼护住魔镰队大奶妈神女**（⚠ 让神女「后续这场战斗一定最优先奶你」）;⚠ **⚠ 临时大队 27 人只剩苏晓、神父、豪梼、泰莉德、凯恩、水哥六人**。

- ⚠ **⚠ 沉稳务实的队长**:⚠ **曙光废墟决战，他的凯恩队只有 2 块破碎本源、这次只有第一没有第二**——⚠ **他果断放弃争胜:「我接下来的任务，是带你们尽可能获得收益的同时，带你们活着离开」**（⚠ 队友血姬〔战力第一线、智谋拖后腿〕撺掇去抢太阳怒兽的击杀奖励，被凯恩、泰莉德一致否决·⚠ 「你猜这东西是谁对应的黑暗混种，我不信那家伙没在暗处盯着」）;⚠ **⚠ 苏晓多次给他丢「活力原液」〔总计16瓶〕、让他感慨后续交锋会「不太好意思动手」。**
