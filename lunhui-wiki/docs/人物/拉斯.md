# 拉斯（愤怒）= 大总统金·布拉德雷

[钢之炼金术师世界](../世界/任务世界/钢之炼金术师.md) [瓶中的小人](瓶中的小人.md) 所造**七名人造人**之一，代表「**愤怒**」——化名**金·布拉德雷**、现任亚美斯多利斯国**大总统**，**最强的人造人**。

- **形象 / 能力**：后天人造人（60 岁·左眼戴黑眼罩·腰挂 6 把刺剑）；青年时被注入液态贤者之石 + 瓶中小人的「愤怒」情绪而成——**完全吸收了体内贤者之石**，故无人造人的恢复力（贤者之石耗尽即死期）、但**体术 / 剑术 / 指挥 / 统帅 / 人格魅力俱顶尖**。
- **真实身份**：本是孤儿，被一个组织收养、培训为「**总统候选**」，从众多候选中脱颖而出、再经注入贤者之石的最后一关成为人造人。与其他人造人不同，**他保有人类的全部情感**——书中评「若没有瓶中小人，他本可把亚美国治理得强盛至极、是天生的领导者」。

## 历程 / 殁
- 七卷13–14章下水道现身、「笑里藏刀」押苏晓返中央市试探弱点（被滴水不漏化解）。
- **七卷39–41章·最终单挑**：苏晓潜入总统府地下时正撞上拉斯，二人「都有死的觉悟」展开近战死斗。苏晓先斩断其刺剑（斩龙闪锋利度发威·6 把刺剑被逐一斩成锯齿 / 断裂）、再斩去其**左臂**与左眼「最强之眼」（失去恐怖动态视力后挡不住苏晓的刀）；拉斯则以刁钻刺剑刺穿苏晓锁骨 / 胸腹十几道贯穿伤。最后一击两人擦身而过——苏晓胸口被斩到见肋骨（生命跌至 **2%** 濒死），拉斯则被**斩首**（无头躯体仍持断剑屹立片刻才倒）。**世界之源 +10.2% → 38.5%·噬灵者 +15 → 法力 989·掉落暗紫色宝箱（苏晓初次获得这一品级宝箱·返乐园时尚未开启）·成就断罪 6/7**。
