# 自来也

[火影忍者世界](../世界/任务世界/火影忍者.md) **木叶「三忍」之一**——身材高大、满头松散白发垂到腰间、脸上两道红痕、戴写着「油」字的护额，面带笑意。是鸣人此时的师傅（鸣人随他修炼三年）。

- **登场 · 调查晓（九卷24章）**：他正在**暗中调查「晓」**。[蝎](蝎.md) + [迪达拉](迪达拉.md) 为截杀河之国大名在附近停留两天、被他捕捉到踪迹。[苏晓](苏晓.md) + 迪达拉狙杀完大名一行、黏土鸟降低查看战果时，自来也以「乱狮子发之术」破土偷袭（白发如钢丝缠人）——被迪达拉引爆黏土鸟震开。三人随即认出这位三忍（蝎一眼识破·迪达拉只闻其名）。
- **应对**：[蝎](蝎.md) 主张拖住敌人、[迪达拉](迪达拉.md) 制黏土鸟空中支援，**苏晓则想先近身试探火影世界强者（影级 / 三忍级）的真实实力**。

## 三忍之力 / 被苏晓重创（九卷25章）
自来也技艺全面、结印快如闪电——土遁·土流壁 / 黄泉泽（软泥困足）、火遁·炎弹 / 大炎弹（喷油持续燃烧）、**乱狮子发之术**（白发可伸长、坚如钢铁、缠人 / 化钢针「乱发刺」/ 包身防爆）。围攻中除苏晓外无人能伤到他。

- **苏晓「一刀换重创」**：苏晓硬顶火焰冲近（靠 [三笠心意](../力量体系/装备道具/index.md)「庇护」护盾扛火），以「**寸断**」（[环断](../力量体系/技能/index.md) 分支·青钢影震荡斩龙闪化能量刃·猛耗千点法力）当头斩下——**他被十几根白发刺穿身体（险中心脏），却也在自来也胸膛斩出半米深伤口、青钢影侵入令其查克拉暴走吐血**。自来也「大意了」、白发挑起苏晓、掌心凝出四代招牌「**螺旋丸**」欲杀他——见苏晓含笑（料定他会用次级无敌护盾反斩）遂撤招、改用大炎弹逼退（苏晓被 [迪达拉](迪达拉.md) 黏土鸟接走）。[蝎](蝎.md) 由此明白佩恩为何招苏晓（近战强过晓内任何人）。

## 「此生最憋屈一战」(九卷25–26章)
自来也白发缝合伤口、连续通灵——**蛤蟆健**（二层楼大·U 形钢叉 + 巨盾·忠诚战死不退）、其兄**蛤蟆文太**（红皮叼烟斗·五米大短刀·更生猛）。但苏晓三人祭出「无耻的 C 阵型」：[蝎](蝎.md) 召出傀儡「**三代风影**」（铁砂·自来也年轻时险死于其手）近身缠斗、[迪达拉](迪达拉.md) 空中轰炸、**苏晓在 1 公里外用蜘蛛女皇狙杀（重狙能射穿自来也的钢铁白发与蛤蟆通灵兽·专打断他的忍术）**——感知极强的自来也也难招架，直呼这是「此生最憋屈的一战」。被逼入绝境，他进入 **仙人模式**（鼻头变大、眼底红痕·肩上是妙木山的**深作仙人 / 志麻仙人**两只蛤蟆仙人）——正面迎接迪达拉的杀招「**十八号（C3）**」轰炸（曾炸平过一个村庄·方圆四百米玻璃化大坑·他靠及时通灵的「胃囊」硬扛、内脏震伤）。

## 被打跑 · 调查受挫（九卷26–27章）
仙人模式的自来也强得恐怖（蛙震声波碎绯流琥、一脚踢碎傀儡），但在苏晓三人「狙击 + 傀儡 + 轰炸」的多层无赖配合下**最终被打跑**（苏晓界断线 + 白蜘蛛炸弹 + 迪达拉黏土炸弹围攻·他躲进通灵青蛙逃走·胸口刀伤崩裂）——这位「三代火影之徒、四代之师、五代之友、六代之师公、七代之人生导师」竟被三名晓**成员**（非首领）打退。他通灵**蛤蟆寅**（身负记录重要信息的卷轴·含解九尾封印的钥匙）向五代火影**纲手**传信（苏晓三人的情报 + 应对策略），自承「情报缺乏、最初就落了下风、晓成员都这么强、首领只会更难对付」，打算养伤后**亲自潜入雨忍村**（怀疑那封闭的忍村就是晓大本营）。

## 殁于雨忍村（九卷61章·侧面交代）
养伤后他果然**亲自潜入雨忍村**调查晓的大本营——**死在了那里，被自己的弟子 [长门](长门.md)（佩恩）所杀**（[苏晓](苏晓.md) 经衍生世界联络平台得知）。这位悲情人物「一生什么都差一点」：出征前与 [纲手](纲手.md) 饮酒时，纲手已透露「这次回来就嫁给他」——他带着「查完雨忍村就回老家结婚」的信念去，却长跪于雨忍村。杀师之后的长门更趋极端（已至「噬师」之境）。这场战死也扫清了苏晓主线「毁灭木叶」的一大间接阻碍（长门遂将转向木叶）。
