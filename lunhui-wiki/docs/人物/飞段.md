# 飞段

[火影忍者世界](../世界/任务世界/火影忍者.md)「**晓**」组织成员、**邪神教（杰修教）信徒**、**不死之身**——与 [角都](角都.md) 组成「**不死二人组**」（也被吐槽为「僵尸二人组」）。

- **性情**：晓里画风「最活泼」的活宝——**豪放、啰嗦、狂妄**，与晓一贯的严肃格格不入。九卷20章幻灯身集合时凑上来**拉 [苏晓](苏晓.md) 入邪神教**（喋喋不休·每来新人必拉·无一例外失败），被苏晓冷淡回绝、一句「智障」气得他大呼小叫「邪神大人旨意要弄死你」（只是性格如此·非真要杀）。
- **不死特性**：原著中他**近身人柱力不会死、但会被揍得很惨**——这正是晓「近身牵制人柱力」长期缺人的写照（苏晓被招入正是来填补这空缺）。

## 封印二尾 / 分头抓尾兽（九卷55章）
封印二尾时他与 [角都](角都.md) 是本体到场（自承「二尾是我抓的」、还想找 [布布汪](布布汪.md)「复仇」石头剪刀布之仇）。天道因五大忍村已警觉而下令全员加速抓尾兽——派**飞段 + 角都去抓八尾**。但二人前途堪忧：八尾人柱力所属的云忍村，四代雷影身体强度极高（连开万花筒的「二柱子」都险被踹死），飞段能否取到其血液成疑。

## 殁 · 被血门围杀（九卷58章·侧面交代）
去抓八尾途中，他与 [角都](角都.md) 被**[血门冒险团](../势力/index.md)**（约一百名二阶契约者 + 团长 [斯坦](斯坦.md)）围攻致死——契约者熟知二人资料，飞段「近身不死」的优势废了一半。晓众人最初误以为是四代雷影 / 八尾人柱力所为。
