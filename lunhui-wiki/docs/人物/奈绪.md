# 奈绪

[火影忍者世界](../世界/任务世界/火影忍者.md) 水之国的**歌姬**（类似现实的歌舞演员·有些歌姬掌握无杀伤力的「忍术」来表演精彩节目）——天生丽质、纤弱少女、一双炯炯有神的大眼睛，**清纯又性感**，是颇有名气的名伶。

- **登场（九卷9–10章）**：她正乘一艘**直属水之国大名府**的大帆船、从盛产歌姬的「**雾雨镇**」出发去**火之国木叶忍村**演出（木叶有庆典）。[苏晓](苏晓.md) 假扮苦力混上这艘船，雾忍**暗部**追来——苏晓为不在船上开战（怕破坏船、无法离开环海的水之国），**挟持奈绪当人质**：往她嘴里塞了颗炼金炸弹（骗她是「舒筋壮骨丸」），威胁「**敢开船就崩**」，逼船留下。
- **「人质 + 船票」**：苏晓上岸引开 / 斩杀暗部后回船，奈绪吓得只能照他的话开航。航行中她怯怯来问「吃了什么」，苏晓直言是**炸弹**、承诺「**送我到火之国（目的地）就取出**」。她吓得腿软，却也只能照办——苏晓恰好也去火之国，等于白嫖了一程船。她想确认「到火之国你保证不伤害我？」被苏晓一个眼神吓退（苏晓魅力仅 5、不擅与剧情人物交涉）。
