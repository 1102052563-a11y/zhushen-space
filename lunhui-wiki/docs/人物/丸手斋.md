# 丸手斋

**丸手斋** 是 [东京喰种](../世界/任务世界/东京喰种.md) 世界 [CCG](../势力/index.md) **七位特等搜查官之一**（二卷8章）。

## 简介
CCG 的**战地指挥官**：指挥才能远超战斗才能、一般不亲身参战，尤其擅长大型战役，凡 CCG 有大型战斗任务多由他现场指挥（二卷37章）。

- 二卷8章：经视频破例首肯 [苏晓](苏晓.md)（伪造身份的「白夜」）加入 CCG（待核实战力）。
- 二卷37章：在 20 区分部主持「**特别对策班**」会议、为进攻 11 区作战前动员，慷慨陈词之际被迟到的 [铃屋什造](铃屋什造.md) 一脚推门打断（二卷37–39章）。

## 关系
- [苏晓](苏晓.md)（白夜）——首肯其入 CCG。
- [铃屋什造](铃屋什造.md)——其动员讲话被铃屋推门打断。
- [篠原幸纪](篠原幸纪.md)、[亚门钢太朗](亚门钢太朗.md)——同属 CCG。
