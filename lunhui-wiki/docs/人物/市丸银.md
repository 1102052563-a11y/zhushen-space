# 市丸银（三番队队长 · 蓝染同党）

[死神世界](../世界/任务世界/死神-尸魂界.md) 护廷十三队**三番队队长**——满头浅紫色短发、永远笑眯眯眯眼的男人，钟爱柿子饼。是 [蓝染](蓝染惣右介.md) 的核心同党之一，中层死神眼中**诸队长里最忌惮的存在**（[雏森桃](雏森桃.md) 形容他「像一条正在微笑的毒蛇」、第一次见他就渗冷汗）。其斩魄刀**神枪**能瞬间伸长到夸张的长度刺穿墙壁与目标（嘲讽能力满级·「蓝染死了」也用幸灾乐祸的语气）。

## 接纳苏晓 · 配合假死（二十卷18、23–25章）

[苏晓](苏晓.md) 与蓝染签下合作契约后，市丸银是第三个知情者（蓝染引荐「新的合作者」）。三人在裏廷间密会敲定**蓝染假死计划**——市丸银扮演「杀害蓝染的凶手」（最苦逼、首要怀疑对象），苏晓扮蓝染好友「调查死因」。计划执行时他差点趁机「错手」杀掉冲上来的 [雏森桃](雏森桃.md)（被自己副队长**吉良伊鹤**挡刀 + 苏晓到场拦下而未得手）。事后苏晓与他擦肩而过、用灵子在其袖口写下「**是时候了**」——他立刻会意「近期有人要倒霉」（从初见就看穿苏晓绝非表面那般随性）。

## 调戏死囚露琪亚 · 真实目的（二十卷41、45章）

处刑日他在忏罪宫吊桥上拦下死囚 [朽木露琪亚](朽木露琪亚.md)「恶劣」地调戏——放灵压假意要救她、把怂掉的忏罪宫看守吓得想逃，临了又笑「骗你的」，硬生生把她已看淡的生死重新搅成对死亡的恐惧（「像递给沙漠渴死者一杯水、刚要喝又告诉他是硫酸」）。但其真实立场比表面复杂：他**不属于任何一方**——潜伏蓝染身边只为**找机会杀蓝染 + 帮青梅竹马松本乱菊夺回崩玉**（蓝染当年为半成品崩玉伤过松本乱菊）。正因 [苏晓](苏晓.md) 灵子脉冲时留了松本乱菊一命，他认苏晓比蓝染可信、私下与苏晓有往来（替苏晓找死兆 / 人·虽没上心）。

## 关系
- **[蓝染](蓝染惣右介.md)**：表面核心同党、实则潜伏在侧伺机杀他（为松本乱菊夺崩玉）。
- **松本乱菊**：青梅竹马（蓝染为崩玉伤过她·市丸银为她卧底复仇）。
- **[朽木露琪亚](朽木露琪亚.md)**：处刑日被他恶劣调戏的死囚。
- **[苏晓](苏晓.md)（白夜）**：因苏晓留松本乱菊一命而认其可信的合作者。
- **[苏晓](苏晓.md)（白夜）**：蓝染引荐的新合作者（彼此心知肚明）。
- **吉良伊鹤**：三番队副队长（关键时刻替他挡下雏森桃的刀）。
