- **极冰领主（北境凛冬城 · 六王之一）**：冰裔之主、全身铠甲无人见其真容——十几年前妻女被座狼族咬死、悲怒交加致**铠甲融在身上**（似脱离活物范畴）；被苏晓几刀斩碎「亚塔德之壁」刺激到亲率 10 万精英出凛冬城（攻破冰蹄要塞时都没这反应）。
- **六族同盟其余四王（术式虚影议会）**：**多因王**（人族 · 隐性以他为首 · 老迈却说一不二 · 曾靠割地 + 联姻差点同化极冰 / 精灵 · 疑藏兵远多于展现）· **矮人王**（黑络腮胡 · 不满多因独大 · 银勋部队名为防北境）· **蛮祭祀**（野蛮人 · 阴阳怪气 · 六族最苦逼）· **异族长老**（龙人 / 蜥族 · 天生结晶之力 · 最古老神秘）——听闻焱谷 / 月光失守、「他回来了」而震动。

## 其他（补）

- **妖鬼·谭雅坦 / 鬣狐鬼姬（阿伽门内奸 · 殁）**：混沌军团的妖鬼女仆、专好精灵美男子（焱谷战中「消化」偷袭沃波尔的精灵战士）——真身竟被**鬣狐鬼姬**（当年混沌军团自立为王的古树王 / 鬼姬之一·借同族高位附身谭雅坦）占据、是 [阿伽门](苍龙大陆/黑暗君主·阿伽门.md) 的第二名眼线（供炎兽人灵魂原料）；被 [沃波尔](苍龙大陆/沃波尔.md) / [尤亚](苍龙大陆群像.md) 设酒局杀掉（破坏阿伽门计划）——真谭雅坦脱身、苏晓拒招募（太弱·体力 −3 不值）。
- **无尽炼狱**：封印着黑暗君主·阿伽门的地方，[希拉](苍龙大陆/希拉.md) 由此脱困而来（10 年前老蛇人试图解封阿伽门失败、却放出了希拉）。
- **术式 / 超凡体质（世界设定）**：本世界信仰「术式」而非神灵（利于统治·不会神权大于王权）；国民生来有超凡体质、2 岁定去留（约 10% 保留），术式非力量来源、而是「引出力量的方式」，核心术式决定战斗风格乃至性格；魔法师是研究术式的学者（只能承受治疗 / 生命术式），战斗型术式须战士强体或刻印铠甲。
- **北境（极北 · 三方纷争）**：苍龙大陆最寒冷的地带、矿产富饶几乎不产农作物——**极冰领主**的**凛冬城**（冰裔·权力比多因王都更集中）vs **座狼族**（数量多·擅群体配合）vs **白熊族**（个体强）三方争斗至今。**亚塔德之壁**（400~500 米天然冰墙·6 小时愈合·隔开北境与战区）+ **冰蹄要塞**（亚塔德之壁三要塞之一·被苏晓堆尸攻破·守将副军卫被打懵）。
- **世界之子 / 世界之使命（伏笔）**：[精灵女王](苍龙大陆/精灵女王.md) 笃信「世界在庇护六族同盟、被世界眷恋之人（英雄）已在路上、杀战争恶魔是世界赋予他的使命」——本世界世界之子 [赞格](苍龙大陆/赞格.md) 正带主角队赶来。苏晓佩 [掠天惊澜](../力量体系/称号.md) 进世界即被世界之子强烈敌视（杀之有掠天惊澜压制反噬）。
- **古树要塞（活物要塞）**：精灵族前线要塞之一——原是岩石结构，因城墙术式变化 + 精灵族喂大量生命宝石而**活了过来、产生模糊意识**、树根缠敌吸生命、与精灵族共处御敌（其他要塞想拆得重建·它却是活物）。苏晓让 [布布汪](布布汪.md) 挖 U 形地道潜入、往其中枢倒 [血神·格赫普斯](暗狱世界/血神·格赫普斯.md) 神血、令其**狂化无差别抽死自己人**——「善水者溺于水」，成了守军惨败的根源。
- **六族同盟崩解 → 三族同盟（39卷38章）**：矮人 / 野蛮人 / 蜥族迫于多因 + 北境压力对混沌军团主防线总攻一轮后**退出六族同盟**（六族同盟名存实亡）；**多因帝国与北境（极冰领主）即将开战**——共同的敌人混沌军团反成两方争相拉拢的对象（狼骑兵军团让谁都不敢让它成孤军）。
- **三大精锐的食物链（39卷52章）**：**重锤部队**（北境·30 万·四米超重甲 + 黑羽翼会飞 + 三颗心脏·非完全活物·单挑本世界最强·俯冲砸一锤即飞的无赖战术）**克** **圣徽骑士团**（多因·62 万·[图尔特](苍龙大陆/图尔特.md) 统领·骑兵冲锋 + 术式引爆铠甲）**克** **晨辉部队**（精灵·精灵女王直属·弓箭连恶魔焰龙都虚·克重锤）——三者相互克制成完美食物链。
- **蜥族衰败史（39卷50章）**：龙人 / 蜥族曾崇尚「晶体」力量、以地标「莫迪法克之眼」雄踞东境、自居「行走的守护者」维系大陆平衡（多因术式疑源自蜥族结晶力量的二次改进）——却因沉醉神秘学 + 与多因「和平共处」，荣光自然陨落、莫迪法克之眼崩倒，如今缩成东泽边缘的中小种族。是「与多因和平共处的下场」的典型（北境冰裔正是看到这下场才绝不与多因共存）。
- **货币（圣银币 / 帝国金币）**：**帝国金币**（多因硬通币·沉金 + 术式防伪·大陆通用）；**[圣银币](../力量体系/装备道具.md)**（多因术式货币·仅术式系超凡者可用·外界无法流通、却对乐园人有致命吸引·1 枚 = 15000 乐园币）——[迪福顿](苍龙大陆/迪福顿.md) 随手当「纪念品」送出后悔断肠。
- **燃火之地 / 炼狱之底 / 炎兽人（39卷63–64章）**：北境凛冬城冰封殿深处、极冰王座后封着的**燃火之地**（岩浆浓烟破碎大地·火之侵蚀每秒 10 真伤）——**炼狱之井 / 炼狱之底**封着黑暗君主 [阿伽门](苍龙大陆/黑暗君主·阿伽门.md)（十米尖锥石柱 + 上千层术式）。**炎兽人**（岩浆 + 熔石 + 火焰能量构成、灵魂多为十几年间战死的灰兽人·无智慧只依命令 / 本能行事）是阿伽门在此培育的大军——守门的 [奥卡姆](苍龙大陆/奥卡姆.md) 是极特殊的有智慧个体；[月使徒](苍龙大陆/月使徒.md) 正靠杀炎兽人刷 6 倍战功。
- **混沌军团崩溃真相 / 虚空之树果实（39卷62–64章）**：**果实真相** = 精灵女王石板画的「果实」实为一块**木雕**（触感如金属·两年前极冰领主赠多因王换东西）——**虚空之树果实在多因王都**。**崩溃真相** = 前任战争恶魔功高盖主要取阿伽门而代之，阿伽门遂与**多因王合谋**（割三分之一大陆立血誓）把「阿伽门 + 战争恶魔」双人大礼包一起封印（战争恶魔封环封树 / 阿伽门封炼狱），混沌军团权力崩溃、[阿卡迪亚](苍龙大陆/阿卡迪亚.md) 力挽狂澜。详见 [黑暗君主·阿伽门](苍龙大陆/黑暗君主·阿伽门.md)。
- **炎兽人 / 炎魔 / 火焰巨魔（39卷65–73章）**：**炎兽人**（岩浆 + 熔石 + 火焰能量 + 战死灰兽人灵魂构成·无智慧只依命令 / 本能·灰兽人灵魂的最后尊严让它们用武器）是 [阿伽门](苍龙大陆/黑暗君主·阿伽门.md) 在燃火之地培育的大军；受 [战争领主](../力量体系/称号.md) 加持后晋升 **炎魔**（力 142 / 火之生命 + 灼痛 + 共生武器·怕低温〔全属性 −25%〕）；上百炎魔合体成 12 米 **火焰巨魔**（蛇人三人一组操控）。苏晓借巴巴托斯龙焰布下的超巨型炼金阵图、让 470 万炎魔 + 火焰巨魔**全体扯胸自爆成「岩浆之花」**（炎魔是阿伽门的力量、绝不能留给他反噬自己）。
- **王都伊焚林格 / 城卫军 / 冰翼 / 原厥（39卷74–76章）**：多因王都**伊焚林格**——城墙 276 加固术式 + 4 主术式（铁壁 / 苍龙力 / 阳炎 / **正义公爵**〔触城墙即死〕·当年 47 万精灵攻城留 17 万尸）；**50 万城卫军**（前线 8~10 年 + 68% 死亡率训练选拔·全金属 300 公斤穿透术式战矛）。多因的**冰翼**（抽亚塔德之壁冰能量的翼龙状能量生物·低温削炎魔）是大决战杀手锏。苏晓用 [战争领主](../力量体系/称号.md) 终极召来**远古战兽·[原厥](../力量体系/装备道具.md)**（古老巨猿·死 4759 年·曾灭拜阳王朝·30 秒·徒手拆掉三面城墙）破城。
