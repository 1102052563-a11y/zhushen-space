# 厄运（摩伊家族三妹 · 不死的怪胎咒术女巫）

[暗鸦星](../世界/任务世界/暗鸦星-暗鸦纪元.md) **摩伊家族的三妹**——五阶段**咒术系女巫**，**天生的怪胎**（怪到连身为女巫的父母都不愿承认、故取名「厄运」·正式姓名而非绰号）。她是全书「**衔尾蛇诅咒**」的源头：泄露「死亡之地」消息者会被她**超远距离**的诅咒化为飞灰（[伊萨](伊萨.md) 即死于此·[美洛狄](美洛狄.md) 险因此丧命）——拥有已知最远距离的诅咒类能力。她居南部**尘泥祭坛**。

- **不死性 = 她的化身能力**：她的化身属性即「**不死**」（咒术系可办到）。她把这不死性**施加在姐姐 [摩伊·诺拉](摩伊·诺拉.md) 身上**——只要厄运处于「生」的状态，诺拉就不死。她本人也极难杀（曾被圣愈教会「杀掉」、半小时后即复生）；**破坏其身体只能让她沉寂半小时**（这半小时是杀诺拉的唯一窗口）。暴露踪迹后她的战力会大幅下降。
- 苏晓评她「已不能称之为人类或女巫」（类似钢炼人造人的不死、却更无解）；苏晓不信有无限复生、只是没人找到正确方法。

## 身体被破 · 姐姐殒命（二十一卷65章）

[卡门](卡门.md) 的夕阳红敢死队（化身成恶鬼）在尘泥祭坛冲进她的祭祀厅、破坏了她的身体使其陷入**沉寂**——这半小时窗口里 [摩伊·诺拉](摩伊·诺拉.md) 失去不死性、被苏晓斩杀。但厄运本人只是沉寂、并未真死（半小时后会复生）；苏晓没时间等她复生再杀，故隐藏任务「摩伊家族覆灭」未完成（少拿 1 黄金技能点）。代价是夕阳红敢死队为破她的身体全员牺牲。

## 关系
- **[摩伊·诺拉](摩伊·诺拉.md)**：长姐（厄运赋她不死性·厄运被破身使诺拉可被杀·「杀诺拉必先杀厄运」）。
- **[卡门](卡门.md) 的夕阳红敢死队**：破她身体、为苏晓杀诺拉开窗口的老巫猎人（全员牺牲）。
- **[摩利莎](摩利莎.md)**：次姐（已被苏晓斩杀）。
- **[卡门](卡门.md) 的夕阳红敢死队**：奉命去尘泥祭坛破她身体、为苏晓杀诺拉开窗口。
- **[苏晓](苏晓.md)（库库林·白夜）**：早已领教其衔尾蛇诅咒、要破她不死性的猎人。
