# 兰爵

[天启乐园](希女王.md) 一方的**机械侧高科技强者**——光头壮汉、一身皮衣，腰带上挂满高科技产品（两把能量枪、三颗电浆手雷及诸多闻所未闻之物）。他是被「莫名其妙」征调进 [希女王](希女王.md) 团队的契约者、**不认希的号令**（自认无须遵从、能与希翻脸）。狡诈、贪财、独来独往；自称「心软」，因 [琉璃](银发少女.md) 央求 + 收了她一颗「红水晶」好处，便赶赴黑土战区**寻杀 [苏晓](苏晓.md)**（「全力战斗的话、不是他死就是我亡」）。

## 「异常」的真相（十二卷31章·希女王的推断）

[希女王](希女王.md) 以恐怖智商看穿他的反常——他**从进入战争世界起就一直死盯「世界之核」的去向**（为套情报甚至不惜暂时听命于希），如今却为一颗红水晶不惜与希翻脸赶去黑土战区。希由此推断：**兰爵与所有独行者都不同**——他独行、能力全面、**总在主动寻找「能力异常」的契约者并杀掉他们**（乐园中常有他「无缘无故与异常契约者结怨」的传闻）。这恰与苏晓「独行 / 能力全面 / 常与异常契约者敌对」的特征重合——希隐隐觉得自己已触到「**某种不可触碰的禁忌的危险边缘**」（疑指向猎杀者 / 灭法之影一脉的隐秘）。

## 「肃清使」= 天启乐园的猎杀者（十二卷33章）

兰爵自报「**肃清使**」——这正是**天启乐园版的 [猎杀者](../总览/轮回乐园机制.md)**（负责清理触犯禁忌的契约者），与苏晓「同一类人」。两个「同类」一相遇便确认了彼此的特殊：① **猎杀者 / 肃清使之间能直接对话、不被乐园屏蔽**（苏晓听得到兰爵的话）；② 这类身份的发展方向是「**人对人（PVP）**」而非普通契约者的「对剧情人物 / 精英生物（PVE）」；③ 危险度是普通契约者的几倍乃至几十倍、但成长更快、且**不能与人组队 / 多次合作**、还得清理千奇百怪的「异常者」。他奉**强制任务**寻杀苏晓（乐园开战期·这世界结束前两人只能活一个），并刻意**等了两小时**待苏晓消耗体力（状态约 8/10）才出手——「这是战术、别说我卑鄙」。

## 能力 · 机械召唤师（十二卷33–37章）

属性力 33 / 敏 72 / 体 67 / 智 73 / 法力 1700——三属性（敏 / 体 / 智）同发展的**远程机械召唤师**，生存力强、还带一身高科技（双能量枪、电浆手雷等）：

- **机械之心 Lv.32**：把昆虫类生物**改造成机械召唤生物**（二次活化 / 远程控制 / **引爆**）——机械蜜蜂（自爆导弹）、萤火虫（指令·守护四层荧光护盾）、蜻蛛 / 甲虫 / 蚯蚓 / 蝎子等，「几百只几百只地召唤」、地下还预埋大量伏兵，堪称「爆炸狂魔」。
- **召唤大师 Lv.30**（被动）：召唤物力 / 敏 / 体 +20%、召唤近乎瞬发；多重 / 超多重召唤 + **召唤物融合技**。
- **电浆球**（能量枪）：篮球大小、可追踪、真正作用是放电**麻痹控制**（攻击力一般·恶心难缠）。
- **幽灵蜂**（最强武器·一根黑色金属箭·融合蜂 / 甲虫 / 蚁的稀有昆虫·强度 20 / 穿透 32）：吹口哨操控、速度比苏晓快 3~5 倍、**几乎无视护盾**、能在**实体 ↔ 虚拟（指令·幽灵）**间切换穿透防御、毒（王蜂之毒·叠 3 层质变·削神经反射 / 防御）、还能（指令·蚁）裂成上万小个体散开。它若死、兰爵会被扣属性。
- **指令系**：守护 / 急速 / 替换（中招瞬间与召唤物互换位置逃命）/ 幽灵 / 穿梭 / 散 / **自杀**（召唤物集体自爆同归于尽）。
- **义肢**：被苏晓斩断左臂 + 肩后，他往**心脏**注射恢复品止血、再以几十只机械蝎子拼成黑色义肢手臂（五指为蝎尾毒针）——因是义肢非再生，斩龙闪「万仞宗」未发动。

## vs 苏晓 · 猎杀者大战（十二卷34–37章·未分胜负）

两名「杀人大师」死战：兰爵以电浆球追踪麻痹 + 幽灵蜂穿盾刺杀压制（苏晓多次靠「[看到](苏晓.md)」式预判 + 反击盾 + [界断线](../力量体系/装备道具.md) 缠住幽灵蜂、一脚把它踢飞出 100 米外令其指令失效）；苏晓 [剃](../力量体系/技能.md) 瞬移冲入兰爵的「指令·自杀」爆炸群、**斩断兰爵左臂 + 肩膀**（兰爵替换 + 甲虫自爆火球·苏晓双腿烧伤、注射 1 号药剂续命）。挨苏晓一刀就被青钢影噬 160 法力 + 240 真伤 + 灵魂感电麻痹——兰爵全靠「替换」屡屡保命。

**结局 · 机械之心保命（十二卷37–38章）**：苏晓**不顾追来的幽灵蜂、直冲兰爵连斩**——幽灵蜂（指令·超音速）穿透苏晓胸膛（刺中心脏附近·偏移没穿要害·幽灵蜂随之强制休眠 72 小时），苏晓忍着血洞连斩兰爵颈 / 胸 / 腹，**把兰爵斩成六段、斩下头颅**。兰爵却没死——他亮出保命大招「**指令·机械之心**」：他的**心脏本是机械构造**，被斩首 / 斩碎后全身机械化、只要机械心脏不被破坏就暂不死（维持 2 小时·机械甲虫把断躯拼成人形）。他命甲虫去自爆杀趴地的苏晓，反被苏晓的炼金炸弹白蛙逼退，只得让机械甲虫拖着断躯 + 休眠的幽灵蜂**狼狈逃离**。苏晓自评「这一战是我赢了」（自己只胸口一洞、兰爵被斩成六段）——但身负重伤、无力追杀，只得装尸体疗伤。

## 第二回合 · 见修斯而逃（十二卷63–64章）

养好伤后他用机械蜜蜂给苏晓送战书——约在古遗迹外围荒草东侧「分生死」（不愿在混乱战场交手·怕被人拖住给苏晓近身机会）。他**独自赴约**（连唯一作伴的野生花豹都放走·他只能转化昆虫为召唤物、偏又讨厌虫子）。可苏晓亮出「大礼」——金色道具召出的 **[怨之孤儿·修斯](怨之孤儿·修斯.md)**（领主级 BOSS·灵魂感染光环·力 / 敏 / 体皆 70+）。兰爵一算「修斯 + 苏晓」的组合（1V1 苏晓他都未必稳赢），**当即转身就逃**——这架打不得。

## 决战 · 机械虫王豪赌而亡（十二卷64末–66章）

逃不掉（苏晓亮 [蜘蛛女皇](../力量体系/装备道具.md) 重狙·要逃出射程得挨 5~7 枪、那只骷髅生物会缠上来），兰爵祭出**最不想用、98% 是最后一次的能力**——让「**机械虫王**」寄生自己（体 / 智 / 意志三判定·任一失败即死·成功则**永久力 -15 / 体 -20 / 生命 -200 / 智 +15**·法力临时 +3000·**30 分钟内解除召唤物数量与种类上限**·虫王死后还要再过一次体力判定否则即死）。他用「**终极指令·嵌合**」把库存上万只机械昆虫（12604 只）熔合成 **25 根幽灵蜂 + 原本 1 根 = 26 根幽灵蜂**，同时操控（心跳 300+/ 分·七孔流血）与苏晓拼命——「绝不妥协、逼急了鱼死网破」，行事反倒更像轮回乐园人。

苏晓把修斯控制权交 [布布汪](布布汪.md)、自己专心近战：26 根幽灵蜂（虚化穿盾·**毒素叠层质变**·削神经反射 / 防御·每秒真伤·持续时间翻倍）险些杀掉苏晓（身上十余血洞、三处伤及内脏）。决胜在 **[界断线](../力量体系/装备道具.md) 配合**——苏晓抬手把界断线钉在修斯背后、布布汪控制修斯绕兰爵缓步走（佯作谨慎）令线缠住兰爵腰；修斯肌肉弯刀砸下、兰爵正要侧身脱身，苏晓猛扯界断线（**兰爵力仅剩 18〔33-15〕vs 苏晓力 72**）**直接把他扯倒**、肌肉弯刀砸断其肋。兰爵蓝甲虫连环自爆（炸断修斯腿 + 自己一臂·又塞独臂入修斯口炸碎修斯头），双臂尽断仍不弃；苏晓 [剃](../力量体系/技能.md) 冲来踹断其小腿、**斩龙闪一刀斩首**。兰爵自始至终没用「指令·替换」逃命——同时控 26 根幽灵蜂已无余力。死前他笑「你这家伙……很久没遇到你这样的敌人」。

**结算（兰爵死·十二卷66章）**：`【猎杀者成功击杀敌方肃清使·彻底死亡、编号消散·此行为得大力嘉奖】` → [苏晓](苏晓.md) **得 100 点功勋 + 灵魂结晶〔大〕×3 + 金色品质装备宝箱 ×1 + 资料查询权限阶位 +1**。
