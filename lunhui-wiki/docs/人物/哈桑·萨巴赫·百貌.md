# 哈桑·萨巴赫（百貌）（Assassin）

第四次圣杯战争 **Assassin（暗杀者）职阶**英灵——**性别不详**、患有**多重人格精神障碍**的暗杀者。

## 能力 · 分裂而非分身

- **战斗力不算顶尖**，但多重人格非但不是弱点、反成其能力：他能把不同人格**独立出来并赋予各自的肉体**——这已不是「分身」、而是「**分裂**」，每个人格都有独立意识与身体、可单独行动。
- **共能分裂 80 个人格**（能力名「**妄想幻象**」）。弱点：分裂时**魔力总数不变**——分裂得越多、每个个体越弱（基础筋力 C / 耐久 D·分裂 20 人后还要除以 20、辣鸡至极）；人格分身死后其英灵能量仍会回归本体。本体混在分身之中、杀掉本体则余分身不足为患。
- **气息遮断（A+ 级）**：完全隐藏自身气息——但「**提前**使用」才有效，攻击时等级骤降；他被苏晓盯上后才用、又是在 [使徒之眼](../力量体系/装备道具/index.md)「空中警卫」侦测期间用，故根本逃不过。综合战力是七英灵里**最水**的（连分身都不用、正面也被苏晓 / 英雄王 / 征服王吊打），仅敏捷 A 拿得出手。

## 历程 · 登场到陨落（十一卷51–56章）

他奉命**刺杀 [Saber](Saber.md) 的代理御主 [爱丽丝菲尔](爱丽丝菲尔.md)**（[卫宫切嗣](卫宫切嗣.md) 之妻），被 Saber 的直感察觉、险些得手却被阻止，引来了 [征服王](征服王·伊斯坎达尔.md)、[英雄王](吉尔伽美什.md)，拉开「英灵齐聚」序幕。

- **被苏晓锁定追杀**：英雄王被时臣令咒召回后，苏晓锁定这名「最好对付」的 Assassin。征服王提议罢战、他也想撤、却已被苏晓气息锁定；他用气息遮断逃，反被苏晓的 [使徒之眼](../力量体系/装备道具/index.md)「**空中警卫**」（A 级·侦 82 种隐匿）牢牢标定。
- **被组合技狙穿**：他分裂 20 人格四散逃——苏晓登信号塔，以 [蜘蛛女皇](../力量体系/装备道具/index.md)（连射）+ 使徒之眼 + 光学红外 8 倍镜「透视」+ 枪术精通 Lv.30 + 粉碎被动，**隔楼隔墙一枪碎一个**分身。
- **被吞噬**：本体现身（解除分身后速度骤增、暴露身份），被苏晓以 [剃](../力量体系/技能/index.md) 瞬移贴身、斩成「人棍」、再以 [吞噬之核](../力量体系/技能/index.md) 吞噬——**世界之源 +7.3%→77.5%·噬灵者 +26 法力 2274·得英灵匣（暗紫）·青钢影 Lv.28→29·青影王觉醒 2%→3%**。圣杯就此盛入第三名英灵灵魂（兰斯洛特、迪卢木多、百貌）。
