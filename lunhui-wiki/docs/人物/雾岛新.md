# 雾岛新

**雾岛新** 是 [东京喰种](../世界/任务世界/东京喰种.md) 世界的 [喰种](../世界/生物万族/index.md)，[雾岛绚都](index.md) 与雾岛董香的**父亲**（二卷87章）。

## 简介
蓝色头发、长相帅气的阳光暖男，待人和气如沐春风，实力不明（能被 CCG 耗重资做实验、可见不弱）。原著情报称其「可能已死」，实则被**生擒囚于 [库克利亚](../世界/地点/index.md)**——铠甲库因克「[新](../力量体系/装备道具.md)」的材料正取自其体内、并以他的名字命名：疯狂科学家不彻底剥离其赫包，而是切开后注高浓度 Rc 细胞令赫包**再生**，使「新」得以**量产**（二卷87章）。

被 [有马贵将](有马贵将.md) 救出、加入反 V 的「最强小队」；表态「有马是救命恩人、绝不推辞」。不过 [苏晓](苏晓.md) 对他**最为防范**——被 CCG 当「赫包生长器」囚禁多年，心理未必还如表面阳光（二卷91章）。

## 结局（二卷94–95章）
甲赫喰种（双手化青黑利爪、质地同库因克「新」）。夺源途中过岩浆湖时疑似**故意出声**惊动湖中「章鱼怪」（用赫子护腿硬扛岩浆、面不改色），险害全队，立场可疑。抵达祭坛后率先冲击苏醒的初代喰种 [马尔斯](马尔斯.md)，被其一拳爆头、吸干 Rc 成无头干尸——**当场身亡**。

## 关系
- [雾岛绚都](index.md) / 雾岛董香——子女。
- [有马贵将](有马贵将.md)——救命恩人。
- [苏晓](苏晓.md)（白夜）——结盟伙伴（被苏晓暗中列为最需防范之人）。
- CCG（V）——曾被囚作「新」的赫包来源。
