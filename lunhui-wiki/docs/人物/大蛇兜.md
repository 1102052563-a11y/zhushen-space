# 大蛇兜（药师兜）

火影世界臭名昭著的「**二五仔**」——本是木叶间谍，被 [蝎](蝎.md) 抓住、体内留毒后转作蝎的间谍潜入大蛇丸身边，大蛇丸替他解毒后又成大蛇丸部下（双面、三面间谍都做过、童年在逃亡与背叛中度过）。

!!! note "早期·药师兜（九卷）"

    早年（九卷）他只是 [大蛇丸](大蛇丸.md) 的得力部下「**药师兜**」——戴圆框近视镜、灰发扎起、脸带若有若无的假笑、擅医疗，此时实力不强（甚至不是 [卡卡西](卡卡西.md) 的对手），其药物维系着大蛇丸被三代封印的手臂。九卷47–50章他在天地桥蹲守、误把 [苏晓](苏晓.md) 当代 [蝎](蝎.md) 赴约之人；苏晓厌其假笑、一道刀芒逼他现出大蛇丸，又**以界断线 + 炼金炸弹白蛇挟持他当人质**（先割其一耳·大蛇丸竟投苦无想灭他口），还拿他试炼金秘药（耳伤瞬愈）。这段「割耳之恨」令他对苏晓印象极深。后来他得大蛇丸血液 + 蛇仙人之力进化成「大蛇兜」、实力暴涨（见下）。大蛇丸被佐助「杀掉」后，他**把大蛇丸的血液注入自身**、凭自身科研水平 + 大蛇丸遗留资料**得到了大蛇丸的力量**，又拜会白蛇仙人习得「龙地洞」**仙术**（类妙木山的自然能量）——遂进化成「**大蛇兜**」：科研水平与大蛇丸不相上下、简直是大蛇丸翻版，脸上不再挂假笑、皮肤苍白如碎陶瓷、还长出一条活蛇尾巴。他对忍术的「真理」（构成 / 原理 / 由来 / 运行方式）有近乎偏执的兴趣——钱财 / 女人 / 权力都打动不了他，唯独珍贵的实验资料无法拒绝。

## 入伙晓 · 秽土转生（十五卷71–72章）

他循着情报网摸到晓组织的隐秘基地、当面找 [带土](宇智波带土.md)（戳穿其「宇智波斑」是假身份）提出**联手**——亮出底牌「**秽土转生**」：当场转生出 [长门](长门.md) / [角都](角都.md) / [蝎](蝎.md) / [宇智波鼬](宇智波鼬.md) 四具尸体（这是只有二代火影、大蛇丸与他三人掌握的禁术·他更「青出于蓝」·哪怕只转生出 70% 实力也够呛）。带土起初想杀他灭口，他随即又秽土转生出**宇智波斑本人**——直击带土死门（带土全靠「斑」的身份招摇撞骗）。带土被将军、终于答应联手（报酬「**宇智波佐助**」战后再给·兜想窥宇智波一族全部秘密）。

!!! warning "真相：他是被苏晓引来的"

    大蛇兜的投诚全在 [苏晓](苏晓.md) 算计之内——鼬的后事是苏晓与带土处理、苏晓知道埋尸处，便在那里布好监视设备，引这「盗墓贼」主动找上门（比原著早了一个月）。有了秽土转生大军，**带土与五大忍村的开战已无可阻挡**（最多十天内爆发），苏晓的主线任务「挑起忍界大战」就此稳了。

## 协助苏晓猎杀外来者（十五卷73–74章）

苏晓以「**部分原始柱间细胞资料**」为筹码（兜唯一无法拒绝的珍贵实验资料）委托他找 [外来者](外来者.md)——他**秽土转生出二百多名感知型忍者**（含二代土影无、日向日差〔宁次之父〕、加藤断、中吉、油女取根等），配合白绝分身铺成天罗地网；又顺手用高超医术**治好了苏晓的左臂骨折**。他棋盘上每颗棋子都是一名转生强者，运筹七天后的开战。

## 关系
- **[苏晓](苏晓.md)**：合作兼利用——被苏晓设局引来，又收苏晓的原始柱间细胞资料替他找外来者（苏晓当年擒过他、割下他一只耳朵·令他印象深刻）。
- **[带土](宇智波带土.md)**：以秽土转生「宇智波斑」逼其联手的盟友（要佐助为报酬）。
- **大蛇丸**：力量与衣钵的来源（他是大蛇丸的翻版）。

## 出处
