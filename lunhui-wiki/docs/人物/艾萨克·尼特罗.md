# 艾萨克·尼特罗（猎人公会会长）

全职猎人世界**猎人公会的会长**——身穿宽松白袍、冲天辫、耳垂极大、和蔼可亲的老者，实则是连 V5 组织都要忌惮其个体实力的强者（猎人公会的诸多特权正源于此）。已 110 岁高龄、却**充满活力、风趣、爱挑战**（实力只剩巅峰约一半·念能力者衰老会大幅掉力·全盛时与 [马哈·揍敌客](马哈·揍敌客.md) 交手未分胜负）。他是强化系，与马哈是同档对手。

## 组「蚁王讨伐队」 + 接纳苏晓（十六卷30–31章）

V5 派他来解决嵌合蚁。他在东果陀国王都附近寻找「志同道合」的强者组队——[苏晓](苏晓.md) 凭马哈的二星猎人执照 + 「熔灰 / 异植物 / 黑念」等枯枯戮山的关键词取信于他（尼特罗只在意苏晓是人类且够强、不在乎善恶），将其纳入**蚁王讨伐队**（原著本是他单挑蚁王·现凑成 **5 人围攻**·胜算更大）。他得知马哈死讯后判断「揍敌客家族那边的危险或许不比这边低」。其手下 **莫老五**（白发圆墨镜·综合素质强、无短板的猎人）与 **诺布**（空间系·胆小怕事·原著被三护卫普夫的气息吓破胆）先去王都侦查。

## 关系
- **[苏晓](苏晓.md)**：蚁王讨伐队的合作者（凭马哈的执照与情报取信·一名「很安全的合作者」）。
- **[蚁王·梅路艾姆](蚁王·梅路艾姆.md)**：讨伐目标。
- **[马哈·揍敌客](马哈·揍敌客.md)**：同档老对手（全盛交手未分胜负·马哈多次邀他合作被拒）。
- **莫老五 / 诺布**：手下的猎人（先行侦查王都）。
