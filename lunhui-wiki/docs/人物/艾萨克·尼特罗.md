# 艾萨克·尼特罗（猎人公会会长）

全职猎人世界**猎人公会的会长**——身穿宽松白袍、冲天辫、耳垂极大、和蔼可亲的老者，实则是连 V5 组织都要忌惮其个体实力的强者（猎人公会的诸多特权正源于此）。已 110 岁高龄、却**充满活力、风趣、爱挑战**（实力只剩巅峰约一半·念能力者衰老会大幅掉力·全盛时与 [马哈·揍敌客](马哈·揍敌客.md) 交手未分胜负）。他是强化系，与马哈是同档对手。

## 组「蚁王讨伐队」 + 接纳苏晓（十六卷30–31章）

V5 派他来解决嵌合蚁。他在东果陀国王都附近寻找「志同道合」的强者组队——[苏晓](苏晓.md) 凭马哈的二星猎人执照 + 「熔灰 / 异植物 / 黑念」等枯枯戮山的关键词取信于他（尼特罗只在意苏晓是人类且够强、不在乎善恶），将其纳入**蚁王讨伐队**（原著本是他单挑蚁王·现凑成 **5 人围攻**·胜算更大）。他得知马哈死讯后判断「揍敌客家族那边的危险或许不比这边低」。其手下 **莫老五**（白发圆墨镜·综合素质强、无短板的猎人）与 **诺布**（空间系·胆小怕事·原著被三护卫普夫的气息吓破胆）先去王都侦查。

## 百式观音 · 正面缠蚁王（十六卷40–42章）

他的最强招式是「**百式观音**」——身后浮现一具十几米高、生有几十条手臂的金色雕塑（「观音」二字纯属个人爱好命名·与观音无关）。它最可怕的不是力量、而是**速度快到连凭直感战斗的苏晓都躲不开**（蚁王凭视力更不可能闪避）。但他亲手一式 / 九十九式拍上 [蚁王·梅路艾姆](蚁王·梅路艾姆.md)、蚁王竟几乎毫发无伤——他暗叹「自己果然老了」（强化系却无法正面压制·实力剩巅峰一半）。他最先察觉蚁王**学习能力恐怖、越战越强**，及时改作「挡住它的诱饵」、让真正的杀招（苏晓 / 尘的远程）去重创蚁王。原著中他对蚁王没办法、只能同归于尽；如今有了 4 名帮手、局面大不相同。

## 击杀蚁王中的取舍（十六卷43–45章）

击杀阶段他的「百式观音」三十四式刺中蚁王肋下（虽伤不动、却不断削其体力、把它打飞制造机会）、还被蚁王弄断一条手臂。蚁王陷入「困兽犹斗」最危险阶段后，他与众人一同**退开远远围着**——他清醒地意识到：原著中他是出于无奈才与蚁王同归于尽，如今有 4 名帮手、不必付出惨烈代价就能解决蚁王，何必送命？（以他现在的身体状态再活几十年没问题、若能再突破甚至可能恢复全盛）。最终蚁王被苏晓远程爆头击杀，他全身而退。
