# 所罗门（黑暗系学术派法爷）

轮回乐园里罕见的**学术派法师（真正的「法爷」）**——黑暗系。戴黑色尖顶帽、一身黑袍、衣领遮住大半张脸，只露出**一双纯黑无瞳的眼睛**；体内充斥黑暗系魔能、几乎与黑暗融为一体（[苏晓](苏晓.md) 在他身上**感觉不到丝毫人类气息**）。所到之处玻璃凝霜、温度骤降（被餐厅店长嫌弃是「自走空调」）；养一条**变种黑曼巴蛇**当宠物（活跃期收不起来）。**惜字如金**，一句话很少超过五个字。

- **真·法爷的分量**：他这种「学术派法师」会钻研法术原理、融入自身体悟，同级别下比只会甩固定技能回路的「技能派」强几倍乃至几十倍——轮回乐园里「法爷」这个称呼正是学术派打下来的（炎辰科普）。[苏晓](苏晓.md) 作为灭法者感知敏锐，直觉**就算与他近身、他也不会像普通法师那样狼狈**。
- **称苏晓「天敌」**：一见苏晓就吐出「天敌」二字——他显然认得灭法之影对法师的克制（尤其 [青钢影](../力量体系/技能.md)）。
- **月之精髓交易（十二卷88章）**：[炎辰](炎辰.md) 牵线、想把苏晓二次调配的 [月之精髓](../力量体系/消耗品材料.md)（法系强化药剂·元素连接强度 +3）卖给他，他只肯出 30~35 万「经费紧张」、任炎辰抬价也雕塑般不松口，最后作势离场未成交。

## 海蓝星「暗天」· 围杀黑蜂（十九卷54–62章）

第二场世界争夺战他作为轮回乐园方散人法爷参战——正是鲁贝城上空那个降下「**暗天**」的暗元素法师：暗元素凝成法师袍，以巨型暗元素骑枪「**黑骑士**」轰塌 ZF 大楼、再化黑洞「**无**」把方圆数百米分解成原子，**推倒了城市标志「鲁贝之拥」雕塑**、宣告「轮回乐园来了」。他面对任何敌人都全力以赴（连鬼夫人都佩服）、在战争频道置顶过黑蜂的情报。最终他与苏晓、[人民教师](人民教师.md)、[咕噜](咕噜.md)、柑橘妹**五人合围**斩黑蜂（暗元素枯萎其腿）。事后他主动向「天敌」[苏晓](苏晓.md) 提出**合作**（灭法之影与法爷竟能携手·二人象征性握手）。**打残天启 + 布大型法阵（十九卷63–66章）**：他与苏晓一队清扫城南——他带着黑暗化身、死亡领主、地狱骑士（五米黑火战马）三召唤物，苏晓近战挡前、他后方狂轰（法爷 = 学者·能力无冷却·虽专修暗元素、暗系召唤 / 控制 / 诅咒 / 削弱 / 攻击却样样精通），两小时合杀**天启 97 人**（约其总数的八分之一·苏晓帮他挡了无数攻击、他破例向苏晓道别 + 赠猩红宝箱 + 恢复药剂）。随后他先去热雨林之森**布设大型法阵**（约 3 小时·为守世界坐标做准备）。

## 🆕 树生世界 · 霸主级单位 · 轮回三大穷的法爷（46卷1–3章）

他这次突然决定参与树生世界（之前还说不想来），一身宽松黑袍金边兜帽、死亡系大法爷既视感，被虚空之树公证为**第四名「霸主级单位」**（前猎杀者/契约者）。最出圈的是他的「穷」:轮回乐园有三大穷——**技法、死灵、法爷**，所罗门一人占了两样半。速降舱要 100 灵魂钱币，他直接从舱门免费跃出——不是抠门，是**真穷**，全身资产总计 **38 枚灵魂钱币**（曾穷到把死亡领主剑上的宝石抠下来卖、地狱骑士都不敢召唤战马怕被卖）。[苏晓](苏晓.md) 对此的评点很妙:对方不开口借，缄口走开才是尊重——「一个强壮的瘸子，并不希望别人主动搀扶他」。

## 关系（补充）

- [苏晓](苏晓.md)：树生世界同为霸主级单位的老朋友（尊重他的「穷」而不主动施舍）。

## 🆕 树生世界 · 灵魂晶核钓鱼（46卷4–7章）

物资箱争夺战他与 [冥狼](魔海/冥狼.md)、兽豪对拼却「拼得并不狠」（竞技才刚开始）。真正出圈的是他在世界联络平台上**用灵魂晶核当鱼饵「钓鱼」**——挂出「出售 7 颗灵魂晶核」，把凑上来交易的参战者一个个宰掉，前后钓杀 20 多人，逼得天启乐园的卡罗罗在平台哀求「大哥别钓了、能不能换种鱼饵、尊重下我们这些鱼的智商」。杀戮榜上他一度稳居第三（32 点）。穷法爷终于靠这招回了口血。

## 关系（补充）

- [苏晓](苏晓.md)：同为霸主级单位、树生世界各凭本事的老朋友。

## 🆕 树生世界 · 卖门密令的穷法爷（46卷10章）

他与咕噜、国足三兄弟从蘑菇先知那接了拜访女王的任务、各持一条关键情报——他手里的是**第二扇封眠之门的密令**。苏晓一句「3000 灵魂钱币」，穷法爷几乎没犹豫就「成交」（咕噜急得喊「你不能被灵魂钱币迷惑」也没用），把密令写在纸条上给了苏晓，转身赶往他心仪的资源产地。他没跟着去围攻女王——那不是穷法爷该趟的浑水。

- [苏晓](苏晓.md)：花 3000 灵魂钱币买他手里第二封眠门密令的老朋友。

## 🆕 破晓队 · 亡灵大军毒瘤（46卷35–36章）

树生世界里所罗门的存在感拉满:贝城畸变后他上午召唤**亡灵大军**正面冲击前城区、把那里的怪物打死八成以上（军团流杀伤不亚于苏晓在塞尔星的太阳军团·只是持续几小时）。八阶若评「世界争夺战最不想遇到的对手」，亡灵妹第一、他第二、[苏晓](苏晓.md) 与黑魔三四（若苏晓也玩军团流、就与亡灵妹并列两大毒瘤）。他在蘑菇村小市场摆摊（货值 3~3.5 万钱币·不穷只是资源消耗大），受苏晓两支生命秘药「门票」投资。攻略大遗迹时被苏晓用半格信号的通讯器请来加入**「破晓队」**（艾朵儿付 300 钱币临时扩队位）、带地狱骑士 / 死亡领主 / 渴血死神三大召唤物专门对付水生之母的「死亡之影·迪尤克」节点——他进队后见到「运势 E-~S+」的小队还狐疑了一下。

## 🆕 死亡之影节点 · 「比手下都能打」的亡灵大法师（46卷38–40章）

对付死亡之影·[迪尤克](树生世界/鬼影·迪尤克.md) 节点时，所罗门坐在石椅上、左手红酒杯右手古书，让地狱骑士 / 死亡领主 / 渴血死神 + 高阶亡灵去混战，还嫌「太慢」（苏晓那边更强的四生恶鬼都搞定了）——他悟出的道理是「**亡灵大法师=比手下所有亡灵都更能打的大法师**」，每次迪尤克冲到他跟前都被他亲手安排（急得血族女仆想发帖「我家饲主明明是召唤师却比所有召唤物强该怎么办」·在守旧的亡灵圈这是没脸出门的事）。水生之母一死他第一个「告辞」撤走（怕那三个队友开始黑吃黑连他一起围），干脆不趟这浑水。

## 🆕 杀戮榜首位 · 明哲保身留蘑菇村（46卷42章）

## 🦴 「我去过」 · 永光世界的组队邀约（50卷66章）
