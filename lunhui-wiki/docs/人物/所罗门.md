# 所罗门（黑暗系学术派法爷）

轮回乐园里罕见的**学术派法师（真正的「法爷」）**——黑暗系。戴黑色尖顶帽、一身黑袍、衣领遮住大半张脸，只露出**一双纯黑无瞳的眼睛**；体内充斥黑暗系魔能、几乎与黑暗融为一体（[苏晓](苏晓.md) 在他身上**感觉不到丝毫人类气息**）。所到之处玻璃凝霜、温度骤降（被餐厅店长嫌弃是「自走空调」）；养一条**变种黑曼巴蛇**当宠物（活跃期收不起来）。**惜字如金**，一句话很少超过五个字。

- **真·法爷的分量**：他这种「学术派法师」会钻研法术原理、融入自身体悟，同级别下比只会甩固定技能回路的「技能派」强几倍乃至几十倍——轮回乐园里「法爷」这个称呼正是学术派打下来的（炎辰科普）。[苏晓](苏晓.md) 作为灭法者感知敏锐，直觉**就算与他近身、他也不会像普通法师那样狼狈**。
- **称苏晓「天敌」**：一见苏晓就吐出「天敌」二字——他显然认得灭法之影对法师的克制（尤其 [青钢影](../力量体系/技能.md)）。
- **月之精髓交易（十二卷88章）**：[炎辰](炎辰.md) 牵线、想把苏晓二次调配的 [月之精髓](../力量体系/消耗品材料.md)（法系强化药剂·元素连接强度 +3）卖给他，他只肯出 30~35 万「经费紧张」、任炎辰抬价也雕塑般不松口，最后作势离场未成交。

## 关系
