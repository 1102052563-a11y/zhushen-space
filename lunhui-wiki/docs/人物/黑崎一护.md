# 黑崎一护（死神世界男主角）

《死神》原著**男主角**——被 [朽木露琪亚](朽木露琪亚.md) 赋予死神之力的「普通人」，斩魂刀斩月（缠绷带的大刀）。**实力派**（智商不低、却算不过 [苏晓](苏晓.md) / [蓝染](蓝染惣右介.md) 这种老阴哔）。为救被判死刑的露琪亚，他随浦原喜助特训（斩魂刀曾被斩断、险死、重获死神之力）后攻入尸魂界。[布布汪](布布汪.md) 早已**打入他的队伍监视他**（随他进尸魂界·浦原想收养布布汪被咬·布布汪跳深坑帮他加快特训）。

## 「那多谢你」· 被苏晓利用（二十卷47章）

双殛处刑被卯之花烈搅乱后，黑崎一护与 [阿散井恋次](阿散井恋次.md)、四枫院夜一三人冲来救露琪亚。苏晓现身、他不认得苏晓、却认得苏晓身旁帮过他大忙的布布汪——夜一来不及捂嘴，他已脱口一句「**那多谢你**」（认了苏晓这个「合作者」）。这一句让旁边的恋次惊到不敢置信（你知道那家伙在瀞灵廷干了什么吗）。他终究没看穿苏晓 / 蓝染的连环局、眼睁睁让恋次接走露琪亚送命。

## 关系
- **[朽木露琪亚](朽木露琪亚.md)**：赋予他死神之力的死神（他攻入尸魂界要救的人）。
- **[阿散井恋次](阿散井恋次.md)**：露琪亚的青梅竹马（同行救人）。
- **[布布汪](布布汪.md)**：打入其队监视他的「狗合作者」（他认得、帮过他大忙）。
- **[苏晓](苏晓.md)（白夜）**：被他一句「那多谢你」认作合作者、实为利用他的老阴哔。
