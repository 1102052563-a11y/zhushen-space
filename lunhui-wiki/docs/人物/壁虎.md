# 壁虎

**壁虎**（本名大守八云，CCG 称其「杰森」）是 [东京喰种](../世界/任务世界/东京喰种.md) 世界的 [喰种](../世界/生物万族/index.md)，原著中「调教」金木研者（二卷1–2章）。

## 简介
面具西装男，**虐待狂**，惯用长柄铁钳（喰种手术用具）施虐。赫子为**鳞赫**（赫包近肾脏、恢复力强、攻强防弱）（二卷2–3章）。

## 经历（截至二卷3章）
奉命来抓神代利世（要活的），与利世对峙间被 [苏晓](苏晓.md) 盯上——苏晓首开 [青钢影](../力量体系/技能.md)，一刀斩断其鳞赫、燃其 Rc 细胞致其惨叫，第二刀斩飞其一臂半耳，两刀将其斩至半死、战力骤降（二卷3章）。
