# 老公爵·艾德里安·阿尔弗（分尾蛇 · 搅屎棍 · 苏晓的合作者）

五公爵中最老的圣裔——枯瘦如柴、随身盘着两任亡妻被神术缩小并包浆的头骨。贪婪善变、翻脸无征兆，享受不死（图谋极大、大到不惜与所有公爵敌对来掩饰目的）。

- 诈病引仇敌 [黎林](黎林公爵.md) 的人来刺杀、反被 [苏晓](../苏晓.md) 强闯压制——谈成让苏晓当「**代行公爵**」（换苏晓帮他找「魂之石」= [断魂影之石](../../力量体系/装备道具.md)）。
- 塞拉灭亡后**戴人皮面具混平民里安然逃脱**（五公爵里最不起眼却活到最后的老狐狸）；又追到深渊 / 神之国，**忽悠 + 抓住 [伊威尔](伊威尔.md)**（想借她报复 [渊之蛇](渊之蛇.md)·靠天巴语脱身反锁她）。

## 关系

- [苏晓](../苏晓.md)：合作者 / 让其当代行公爵。
- [黎林公爵](黎林公爵.md)：他诈病引来的仇敌。
- [伊威尔](伊威尔.md)：他忽悠 + 反锁、想借以报复渊之蛇的噬神者。
