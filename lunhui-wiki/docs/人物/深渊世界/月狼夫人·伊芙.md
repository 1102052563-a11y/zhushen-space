# 月狼夫人·伊芙（月狼 · 自私狠辣 · 殁）

慵懒华贵、看似三十出头的美妇——「狠如蛇蝎」、极度自私。

- 亚兰斯危局选「与城共存」守后方（实因贪生不敢上前线）；城破后被冰皇帝 [雷德·巴纳德](冰皇帝.md) 当众斩首处决——临刑预言「有个不讲道理的怪物〔指 [苏晓](../苏晓.md)〕会回来把你们赶尽杀绝」。

## 关系

- [雷德·巴纳德](冰皇帝.md)：当众处决她的北部联盟新王。
- [苏晓](../苏晓.md)：她临刑预言的「怪物」。

## 出处

- 三十一卷（[深渊世界·塞拉帝国](../../世界/任务世界/深渊世界-塞拉帝国.md)）1–32 章：守后方→城破被冰皇帝处决。
