# 黎林公爵（森林 · 众神会真首领 · 为塞拉粗暴夺权 · 殁）

鹰钩鼻、不苟言笑的古板公爵——非初代圣裔（生来即是、世袭爵位）；**撑着塞拉帝国对抗北部联盟**（没他帝国早被打穿）、30% 官员愿拥他为圣裔王。

- **真相**：他**重建了 [老公爵](老公爵.md) 50 年前组建、被怒狮剿灭的「众神会」**，趁铁羽王将死想**杀光其余公爵 + 圣裔王、粗暴夺权救塞拉**（连杀怒狮 / 白龙·武器为爵徽「黎明树」+ 植物生命力）。
- 被 [苏晓](../苏晓.md) 识破登门智取斩杀（生来即公爵·从没打过你死我活的仗·技法被碾压·临终「我所做的一切都是为了塞拉，真正理解我的却是你」）。可敬的对手。

## 关系

- [苏晓](../苏晓.md)：识破并智取斩杀他的人。
- [老公爵](老公爵.md)：众神会的初代组建者。
