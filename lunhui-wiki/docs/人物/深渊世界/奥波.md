# 奥波（守源人 · 胖巨人 · 第六环引导人）

蹲守深渊神之礼殿 / 渊蛇邸的秃头胖巨人（真实力 150+·不伤活物·见人就逃）。

- **守源人**：与 [渊之蛇](渊之蛇.md) 约定「只要他守着源之潭，挚友古龙之子洛斯就不会受伤」，已苦等死去的洛斯几百年（其实早猜到洛斯遭不幸·洛斯尸骸就在他守望的神之礼殿内）。
- [苏晓](../苏晓.md) 拔刀「友善交涉」+ 告知洛斯死讯（奥波泪崩）后**激活第六环**；奥波善良、怕自己「带来灾难」不敢去人多的永月城。

## 关系

- [渊之蛇](渊之蛇.md)：与他约定守源的深渊圣灵。
- [苏晓](../苏晓.md)：告知洛斯死讯、激活第六环的人。

## 出处

- 三十一卷（[深渊世界·塞拉帝国](../../世界/任务世界/深渊世界-塞拉帝国.md)）33–46 章：守源→被告知洛斯死讯激活第六环。
