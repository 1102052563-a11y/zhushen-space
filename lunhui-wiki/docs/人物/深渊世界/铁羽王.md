# 铁羽王（塞拉真正的王 · 非圣裔 · 二度被斩）

塞拉帝国「真正的王」——出身战场士兵、一切都是杀出来的，手持勾刃长枪、**非圣裔**（约 260 岁·没老死已是奇迹）。

- 曾只身跃入深渊与 [深渊之蛇](渊之蛇.md) 鏖战两天、刺瞎其一眼；一人镇得北部联盟不敢攻亚兰斯城。垂垂老矣、每出手一次就更近死亡；现身时顺手宰了四名死亡乐园契约者（如杀鸡屠狗），后在木屋后**自挖坟、裹草席、平静老死**。
- **结局**：被 [渊之蛇](渊之蛇.md)「映召」出全盛之姿（枪术宗师 Lv.43·战王免控 + 「征战」越杀越强·钩刃长枪能把青鬼甩回来）与 [苏晓](../苏晓.md) **巅峰技法对决**——互卖破绽、瞬息定生死；苏晓靠超微 [放逐](../../力量体系/装备道具.md) 攀刀尖偷袭头颅 + 魔刃终结（残血 16.8% 险胜·「真是畅快」）。苏晓**第二次葬了他**（断刃枪当碑·命运主宰刻「铁」字）。

## 关系

- [渊之蛇](渊之蛇.md)：被他刺瞎一眼、又映召他全盛之姿的深渊圣灵。
- [苏晓](../苏晓.md)：与他巅峰对决、二度葬他的刀术宗师。

## 出处

- 三十一卷（[深渊世界·塞拉帝国](../../世界/任务世界/深渊世界-塞拉帝国.md)）9–61 章：镇北部→自挖坟老死→被映召后与苏晓巅峰对决再殁。
