# 冰皇帝·雷德·巴纳德（北部联盟新王 · 铁血却不嗜杀）

灭塞拉帝国、登东大陆的北部联盟最高统治者——魁梧、胡须扎成粗辫、铁血却不嗜杀。

- 攻塞拉非为复仇（那只是鼓舞士气的借口），真正目的是**让苦寒之地的子民离开极地、占东大陆求活**（且 [铁羽王](铁羽王.md) 一死「不战之约」即失效）。城破后宣布平民自由、私产不可侵犯；当众处决 [月狼夫人·伊芙](月狼夫人·伊芙.md)。也清楚还有深渊 / 神之国 / [渊之蛇](渊之蛇.md) 等大敌。

## 关系

- [月狼夫人·伊芙](月狼夫人·伊芙.md)：被他当众处决的塞拉公爵。
- [铁羽王](铁羽王.md)：镇得他不敢攻城、一死约定即失效的塞拉王。
