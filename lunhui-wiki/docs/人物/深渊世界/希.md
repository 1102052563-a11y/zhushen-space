# 希（无编号契约者 / 超脱者 · 水晶女皇 · 苏晓旧敌 · 殁）

[苏晓](../苏晓.md) **二阶时的旧敌**——银发、爱猫、有 9~10 条命的女人（当年正面一对一苏晓打不过她、最终靠 [阿波罗](../../力量体系/消耗品材料.md) 硬炸死·能力为「**水晶**」）。

- 她没死透、在天启乐园外**以中立神灵的「源」为食**疯狂成长、已**杀了 76 名猎杀者**；本世界加入北部联盟、来神之国想吞古神厄休拉的源——却被恢复巅峰的 [伊威尔](伊威尔.md) 完克。
- **结局**：草原被苏晓斩杀（完成主线 2）。希近乎全能（**晶体化**免致命 + 自愈 / 同化 / 晶女皇召水晶龙 / 晶焰燃生命灵魂 / 晶爆），但水晶困不住刀术宗师——苏晓以血气分身 + 龙影闪绕到死角、连破晶体化（要害分离即真死）斩杀（「刃道刀·绝幽」破其最后一搏）。死前用未驾驭的「神灵泣」也没翻盘；背后那个「神秘存在」仍是悬案。

## 关系

- [苏晓](../苏晓.md)：她二阶旧敌、已「杀」她 10 次的毕生之敌。
- [伊威尔](伊威尔.md)：完克她的第一噬神者。
- [米糖](米糖.md) / 老狼 / 康巴：她雇佣的希阵营队友。
