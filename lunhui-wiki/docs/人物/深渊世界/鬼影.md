# 鬼影（女刺客 · 众神会 · 被苏晓反利用）

自幼受训、毕生使命就是刺杀怒狮公爵的女暗杀者——「向死而荣」、信背后组织以「神灵名义」诛圣裔（死后回归神灵怀抱）。

- 刺杀时**不滥杀无辜**（只击晕仆从）成了破绽；[苏晓](../苏晓.md) 10 分钟从手印 / 蔷薇花香 / 腰伤推理出她的体征、生擒后**操控分解的放逐微操她的表情伪装「叛变」**（钓 [众神会](黎林公爵.md) 组织出手·心态崩）。

## 关系

- [苏晓](../苏晓.md)：推理生擒、反利用她钓组织的人。
- [黎林公爵](黎林公爵.md)：重建众神会的真首领。

## 出处

- 三十一卷（[深渊世界·塞拉帝国](../../世界/任务世界/深渊世界-塞拉帝国.md)）9–23 章：刺杀怒狮→被苏晓反利用伪装叛变。
