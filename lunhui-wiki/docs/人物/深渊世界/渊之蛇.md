# 渊之蛇（深渊守护圣灵 · 映召已故强者 · 殁）

蓝皮人族的守护圣灵——巨大黑鳞分尾蛇（被 [铁羽王](铁羽王.md) 刺瞎一眼·收天巴族献的「源」能量·守着死亡乐园的时空枢纽换庇护物品）。

- **真正的能力 = 「映召」已故强者**（本体凭巨躯战斗反而弱）：[伊威尔](伊威尔.md) 当年的对手是它映召的古龙之子洛斯、铁羽王的对手是初代圣裔王——故所有与「渊之蛇」战斗者的遗言都对不上号。
- [苏晓](../苏晓.md) 杀穿其映召的铁羽王后、随手捏死缩小的本体（蛇尸丢给古神厄休拉·得圣灵级宝箱·破时空枢纽🏁通关）。

## 关系

- [铁羽王](铁羽王.md)：刺瞎它一眼、又被它映召的塞拉王。
- [伊威尔](伊威尔.md)：被它蛊惑成噬神者、又找它决战的第一噬神者。
- [苏晓](../苏晓.md)：杀穿映召、捏死本体的人。

## 出处

- 三十一卷（[深渊世界·塞拉帝国](../../世界/任务世界/深渊世界-塞拉帝国.md)）47–61 章：映召铁羽王→本体被苏晓捏死。
