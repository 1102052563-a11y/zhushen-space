# 圣裔王·亚历山（笨拙的战王 · 剑术宗师 · 殁）

塞拉帝国之王——银灰发、宽松白衣如睡衣、毫无气场的中年男，开会都能秒睡。

- **真相**：他不是深藏不露、只是觉得身边争权夺利的一切「太无聊」；笨拙到学礼仪要练一月，却**唯独握剑那刻第一次笑了**——是个纯粹的「战王」。曾**弑父**（父亲临终却以他为傲）、封剑去完成父亲期望当圣裔王、又因愧疚踩住野心的怒狮；采纳 [黎林](黎林公爵.md) 大半意见、并不昏庸。
- **剑术宗师 Lv.37 + 11 种被动 + 不死性 + 圣灵级战剑**——塞拉除 [铁羽王](铁羽王.md) 外最强者（睡觉能梦中修行剑术）。被 [苏晓](../苏晓.md) [狼族传家宝](../../力量体系/装备道具.md) 拖到荒野单挑 + 小队围攻斩杀（[阿姆](../阿姆.md) 硬接 + [巴哈](../剧毒魔鹰·巴哈.md) 暗杀 + 苏晓主力），取其心脏 + 头颅里的**第四颗 [断魂影之石](../../力量体系/装备道具.md)**。

## 关系

- [苏晓](../苏晓.md)：拖到荒野围攻斩杀、取魂之石的人。
- [铁羽王](铁羽王.md)：塞拉除他外最强者。
- [伊威尔](伊威尔.md)：吃了他的心脏（古龙之子血脉）恢复神智的噬神者。
