# 伊威尔（史上第一噬神者 · 一切的起始之人）

被 [深渊之蛇](渊之蛇.md) 蛊惑、**史上第一个斩神夺力的噬神者**、活得最久的圣裔（屠戮上万圣裔 + 无数神灵·命镰「夺生命力之镰」斩中圣裔即秒杀）。

- 后从所杀神灵记忆得知**圣战是真的、神灵并不恶、弑神只源于人心贪婪**，悔悟弃猎；找深渊之蛇决战首尝败绩、跪在深渊前永世忏悔、生命力飘散成白痴——吃下 [圣裔王·亚历山](圣裔王·亚历山.md) 的心脏（古龙之子血脉）恢复神智、又屠万名北部士兵恢复巅峰。
- 是任务「一切的起始之人」（[苏晓](../苏晓.md) 交亚历山心脏 + 她掏眼球给指引光球）；**完克 [希](希.md)**（30 秒切一种元素形态·把希小队打到撤退）。追到神之国前守村反被天巴族群殴，目睹这里就是「神之国」彻底崩溃；被同期噬神者 [老公爵](老公爵.md) 忽悠 + 反锁。

## 关系

- [渊之蛇](渊之蛇.md)：蛊惑她成噬神者的深渊圣灵。
- [希](希.md)：被她完克的水晶女皇。
- [老公爵](老公爵.md)：忽悠 + 反锁她的同期噬神者。
- [苏晓](../苏晓.md)：给她亚历山心脏、换指引的人。
