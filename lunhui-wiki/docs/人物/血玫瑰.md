# 血玫瑰

**血玫瑰** 是 [进击的巨人](../世界/任务世界/进击的巨人.md) 世界 **[彼岸花冒险团](../势力/index.md)**（全女性永久冒险团）的副团长（三卷2章）。

## 简介
酒红色长发披肩、深黄色皮甲、身材修长妩媚的御姐；手持一把布满螺旋纹路的**蓝色品质长枪**，主力量、副敏捷（跟不上奇行种的鬼魅速度）。惯以媚态拉拢战力（尤好勾搭法师），却也**深谙排兵布阵**——攻防战中主动到三区组织防御、把队形安排得滴水不漏，令抢了首领位的兄弟会新人尘相形见绌（三卷2–5章）。

## 吞噬岛复仇 / 关键一枪（八卷12、27–29章）
彼岸花在巨人世界险被苏晓灭团后**重整为六人精英小团**（详见 [冷月](冷月.md)），八卷11–12章六女与苏晓同批被分到 [二阶生存试炼·吞噬岛](../世界/任务世界/T12K星球-生存试炼.md)、谋为「三十二名姐妹的血仇」复仇。八卷27–29章正面恶战中，她全程「按兵不动」、一度被误以为**卖队友**——实则她是全队唯一让苏晓感到「**隐隐危险**」的人，**一直在酝酿招式**。当苏晓的断头台几乎勒死 [罗娜](罗娜.md) 时，她**一枪（血色长枪·音啸破空·差点穿苏晓的头）逼退苏晓、抓走救下罗娜**——这一手沉稳老辣，印证了她「副团长」的分量。

## 殁 · 泣血玫瑰（八卷30章）
救下罗娜后，她开启职业**终极技能「泣血玫瑰」Lv.3**（燃血爆种·耗灵魂结晶〔小〕×2 + 90% 法力·力 +13 / 敏 +12 / 体 +10 / 魅 +10·**用后无法停止、不死即亡**·全队唯她知此招）——力敏由此达 50、更强更快，却仍因**枪术 Lv.9 远不及苏晓刀术 Lv.21** 而一枪都刺不中。最后她**亲手掏出自己的心脏、握碎、把血涂在螺旋长枪上**爆发突刺，被苏晓的 **[神灵余晖](../力量体系/装备道具.md)「圣灵赐福」（1 秒次级无敌护盾）**硬挡——长枪与她的身体随即**寸寸碎裂如瓷娃娃**而亡。临终对苏晓笑道「**轮回乐园对我们是地狱、对你或许才是乐园**」「我一点都不恨你」。她的牺牲为时已晚——其余四女随后被苏晓与布布汪逐一杀尽。
