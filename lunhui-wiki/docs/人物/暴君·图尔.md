# 暴君·图尔（不死暴君）

**天启乐园**八阶顶尖梯队的**坦系**契约者——35 岁男性，绰号「**不死暴君·图尔**」，[天审](天审.md) 小队成员。领域全开后可变成身高近四米、皮肤灰黑、肌肉泛钢铁质感的巨汉。他在天启乐园名声赫赫，但这名声「不是打出来的，而是因为**挨打**」。

## 能力 · 越挨打越强

- **打击适应性**：随受击次数增多 / 生命值下降，肉体防御力、生命恢复速度持续提升——**越挨打越抗揍**，身体防御力能高到匪夷所思。
- **生存力远超防御力 · 血统复活**：只要不把他彻底烧成灰烬（连一块血肉都不剩），他就能靠血统之力恢复。从四阶到八阶「死」过很多次，每次被锤到自闭又都能复原——这才是他最强之处。曾被旅团成员宰了扔岩浆湖、被绯世用金属高跟鞋踢成墙上均匀肉泥，均未真死。
- **弱点·治疗越多越浪**：天审的吐槽——坦系有治疗就降智商涨勇猛（开领域再降 50% 智商、每多一名治疗系再降 10% 智商 + 涨 30% 勇猛）。故天审反话「祝你好运」想让他别浪，可惜结界内五名治疗系让他彻底「迷失自我」。

## 时空之力交易 · 挨打典范（42卷44–49章）

在 [暗星](../世界/任务世界/暗星-世界入侵.md) 时空之力交易中，他被 [异端处刑队](暗星/异端处刑队.md) 十二人围砍——处刑队把他斩到 3~4 米高空、十二把大剑轮番漩斩 / 跳斩 / 翻身力劈（像街舞团），他越挨砍越麻木、终至一根手指都动不了，全靠结界内三名治疗系续命才没死。战斗尾声，治疗系跑光后他被斩成粉碎、异端处刑队还点火烧了他残骸——**结果他居然没死**，靠血统之力又恢复过来。[苏晓](苏晓.md) 估计今后八阶世界争夺战大概率还会撞见这位「一天不挨揍浑身难受」的老兄。

## 🆕联盟星 · 被国足三兄弟锤自闭（42卷49→43卷18章）

暗星时空之力交易后，他在 [联盟星](../世界/任务世界/联盟星-死亡圣杯.md) 友克市郊外又栽了跟头——他来抢 [国足三兄弟](国足三兄弟.md) 刚击杀的八阶野生小 boss 的奖励、全开重力领域压制三人，却被国足老二一个「双手比心」的骚姿势打出**物理强制眩晕**（心脏似被攥住），随即被三兄弟三角站位一顿「80/欧拉/木大」鬼畜乱锤，抱头蜷缩、动弹不得，只想知道「自己遇到了三个什么东西」。这位「越挨打越强」的不死暴君，撞上专治单体、能物理眩晕的国足三兄弟，也只有挨锤的份。

## 🆕 西大陆阵营 · 军团流的活靶子（43卷50–56章）

被国足锤过后，他又跟着圣光乐园毒奶 [光沐](魔海/光沐.md) 钻进西大陆这「高收益隐雪区」、加入了泰亚图文明阵营的契约者堆（图谋圣灵级套装 [虫厄共生](../力量体系/装备道具.md) 等阵营商店好物）。可他一路倒霉:先被苏晓下令「谈判诱敌」引出的舰队**五小时环岛炮击**炸得灰头土脸（他嘴硬「挠痒一样」，转头就躲进地下几百米），又眼睁睁看西大陆边缘被炮火炸沉。他坚信寄虫战士数量是同盟军几倍、「这次稳了」，还怂恿死亡乐园杀手 [水哥](联盟星/水哥.md) 去暗杀苏晓（被拒）。他不知道苏晓在憋大招——等 5 万老兵 + 40 万士兵触发战争领主「血·魂之力」，等着他这个「越挨打越强」的活靶子的，是子弹附真实伤害的军团流平推。总攻中他与 [光沐](魔海/光沐.md) 在王城地宫为争夺 [虫厄共生](../力量体系/装备道具.md) 套装迟迟不走，[巴哈](剧毒魔鹰·巴哈.md) 背着 [布布汪](布布汪.md) 从空中往他脸上狂扔删减版阿波罗——他硬顶着一颗颗太阳焰爆炸、怒吼「渣渣」死不服输，**最终在金色烈焰中逐渐化为焦炭、变成灰烬**。「越挨打越强」的不死暴君，恰恰倒在了他唯一的软肋「彻底烧成灰烬」上（原文未给明确击杀提示、生死存疑）。

⚠️ 附带一提，他每「死」一次智商都会被削——[光沐](魔海/光沐.md) 就据此断定他那句「一会看我眼色行事」（对盲人 [水哥](联盟星/水哥.md) 说）的蠢话，正是不死能力反复削智的后果。

## 关系

- [苏晓](苏晓.md)（库库林·白夜）：打劫时空之力交易的敌方猎杀者（顺手在他脖子上留了道刀痕验证「打击适应性」）；联盟星西大陆阵营里被他的军团流吊打的活靶子。
- [国足三兄弟](国足三兄弟.md)：在联盟星把他物理眩晕 + 围殴到自闭的逗逼锤子组合。
- [光沐](魔海/光沐.md) / [水哥](联盟星/水哥.md)：联盟星西大陆阵营里同为契约者的临时盟友（光沐拉他来隐雪区、他怂恿水哥暗杀苏晓被拒）。
- [天审](天审.md) / [异端处刑队](暗星/异端处刑队.md)：天审小队队长 / 把他斩碎焚烧却没弄死的科多学派处刑队。

## 🆕 树生世界 · 又双叒死了（46卷12章）

多次「死而复生」的他（暗星被处刑队围攻致死、联盟星遭舰炮 + 阿波罗、塞尔星被眷族宪兵围攻致死，只要留下部分血肉组织就能复生·代价是脑子不太好使）也来到树生世界，被 [伍德](画中世界/沃波·伍德.md) 用深渊之罐坑进 [女王·尤罗](树生世界/女王·尤罗.md) 寝殿。他一见 [苏晓](苏晓.md) 就表情一僵、决定「惹不起躲得起」。作为天启坦系他被圣诗增益后一度「老子已无敌」，却被女王一记光刃眩晕 + 力量共振震得龟裂、黑刃钉地扭爆——又一次当场身死（「快看，暴君又死了」）。
