# 迟玫（空间系女刺客）

轮回乐园契约者、**空间系女刺客**——黑斗篷兜帽、惨白脸化黑眼影、容貌中上却气质独特（没香水味、只有淡淡血腥气）、三阶竞技场排行 1876 名（三阶中的狠角色）。她的隐匿不是隐身、而是**进入异空间**（每次有约 3 秒延时、且会大幅降速·只能当高等隐身用、近战中用不了）。

## 偷袭苏晓 → 被「玩坏」（十六卷28–29章）

她误以为正在远程狙击的 [苏晓](苏晓.md) 是个不擅近战的狙击手，**酝酿许久刺出力求秒杀的一刀**（淬毒 + 背刺 / 暗影刃 / 要害贯穿 / 绝杀刺 / 死亡之手五重效果·一刀刺穿 500 强度反击盾）——却没能秒掉苏晓（直感早有预警）。她躲进异空间不敢动，被苏晓空间禁锢 + 满地陷阱困住；最惨的是 [布布汪](布布汪.md) 在她可能现身处埋了 **14 颗高仿「磁导雷」**（其实是炼金炸弹假货·每颗成本不到 30 币·苏晓指点下精确计算连导角度）——她一出异空间就**连扎三颗雷**、被炸断一腿一臂、左眼视力降 90%，崩溃大笑「这个变态！」，最终捏碎项链脱身逃走。从此发誓远离苏晓。

## 关系
- **[苏晓](苏晓.md)**：偷袭未遂、反被「玩坏」的苦主（发誓再不招惹他）。
