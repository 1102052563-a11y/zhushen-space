# 神皇

[神皇冒险团](../势力/index.md) 的**老大**（团长）、[茜拉](茜拉.md) 的**亲哥哥**。出场常穿**全身铠甲 + 头盔**（怕被认出）；私下贪杯、酒后常做荒唐事（带上百名部下去「大保健」、欠下属乐园币……被茜拉当众揭老底），但能撑起这么大的冒险团、是**天生的领袖**（衍生世界中气魄惊人·茜拉口中「最想跟随的团长」）。苏晓评他「**有舍有得、知道什么对团队更重要**」。

- **与苏晓**：早有交易之缘（六卷曾买苏晓的恶魔果实）。八卷5章苏晓出售钢炼战利品时，神皇**用暗紫武器鞘 [神灵余晖](../力量体系/装备道具.md) 换走两件原罪套装 + 人造人血统**（原罪套装对走法师路线的茜拉 / 团队更重要、人造人血统「适合坦都那家伙」）——明知神灵余晖救过自己几次仍忍痛割爱，足见其格局。
- **拍卖会**：八卷6章起与**黑帆商会**联办乐园拍卖会（中心会堂·租金 2 万/时·拍装备 / 技能 / 血统 / 恢复品 / 灵魂结晶），卖给苏晓前排门票；他能租到中心会堂，是因「洗劫一个衍生世界、得高级物品卖给乐园」换来了租赁权限。手下 [疯奶](index.md) 常念叨着要杀苏晓。

## 妹妹之死 · 对幻影旅团宣战（八卷80章）
他一向理智、不是冲动之人，这次却**疯了一般**——亲妹妹 [茜拉](茜拉.md) **被幻影旅团第 6 位 [咕噜](咕噜.md) 所杀**（争抢「[猎魔套装](../力量体系/装备道具.md)」三件套之一时·套装戒指本在神皇冒险团手里），他遂与 **[幻影旅团](../势力/index.md) 正式「不死不休」开战**。此战把 **神皇、旅团、血门、黑帆商会、蛮天等十几个大型冒险团**全卷进对那套「猎魔套装」的争夺，成了一块**烫手山药**（夏当茶余饭后讲给苏晓·苏晓暂不打算掺和）。

## 买飞升套 · 五心腹具体登场（十六卷60–63章）

苏晓开宝箱得**飞升套部件 [神圣挽歌之戒](../力量体系/装备道具.md)**、只发邮件给神皇一家（给足面子），神皇遂带心腹赴交易市场割肉买下——**飞升套实在太适合大型团的奶妈**（首席大奶妈 [抹茶](../势力/index.md) 哭着要当「飞天奶」）。他懂「面子是互相维持出来的」、一上来就报诚意价，最终以「宗师之力（1/10）+ [狂逆兽](../力量体系/装备道具.md) + [动能战靴](../力量体系/装备道具.md)」+ 苏晓补 15 颗灵魂结晶〔小〕成交。神皇冒险团是**利益链 + 感情牌并用、恩威并施**（与血门 [斯坦](斯坦.md) 的纯利益链不同——他与二号 [狂狐](../势力/index.md) 意见不合就私下「拳脚论道」、最后讲理者胜）。五心腹 = 狂狐 / 抹茶 / 老山羊 / 疯奶 / 已故的妹妹 [茜拉](茜拉.md)（他身为四阶仍复活不了她）。
