# 安娜（暗夜精灵公主）

[维斯星球·多亚古大陆](../世界/任务世界/维斯星球-多亚古大陆.md) 的**暗夜精灵公主**——常以**隐身**状态伴在铁血军团军团长 [卡洛斯](卡洛斯.md) 身旁（部落首领传言「那两个家伙有一腿」）。精通炼金 / 魔法，是卡洛斯黑土要塞攻坚战的关键助力。

**暗夜精灵族**（帝国特殊兵种·多任斥候 / 暗杀者·**逾七成为女性**）：天生皮肤与毛发能**吸附光线、长时间隐身**（一生大半时间隐身、因女性太美易招觊觎）——但**隐身时不能穿衣**（穿衣则隐身大幅失效），故隐身斥候皆赤身；随身只能带一两枚**炼金腐蚀炸弹**（藏在发辫里·腐蚀金属·专破部落的攻城器械）。她们是战争器械的最大威胁，故部落总把器械藏偏僻处、少留士兵、布炼金陷阱（既火力支援又作诱饵）。安娜身为「公主」是这一族的高位者。

## 黑土要塞之战 · 隐身助攻（十二卷22章）

卡洛斯花 9 万金镑从炼金协会购入暗藏「**热冷换置炼金阵图**」的特殊石块、混进堆要塞的石坡里——待石块被绿磷火烤化成岩浆、阵图就位，卡洛斯一声「安娜，去吧」，由隐身的她**激活那道方圆十几公里的连锁炼金阵图**：岩浆喷涌至与要塞齐高，再骤然冷热交替、迸出远超临界点的**过热蒸汽**，借西风吞没黑土要塞、**「清蒸」死了驻守的三万部落军**（兽人 / 地精 / 矮人 / 巨魔无一幸免）。部落方首领因「暗夜精灵公主一定在卡洛斯附近」而忌惮其能力、曾密谋借此除掉卡洛斯。

## 痴情而狠辣 · 为救卡洛斯（十二卷38章）

[卡洛斯](卡洛斯.md) 恶战重伤、命不久矣，她哭着誓「**一定有办法**」——美丽容貌下藏着惊人的狠辣：她**亲手杀了暗恋她、给卡洛斯下毒药剂的地精混血道格拉**（道格拉自以为是她「青梅竹马」·她冷道「暗夜精灵的孩童一同长大、按他的逻辑我有一百多个青梅竹马」），又暗暗**盯上军需官 [尼古拉斯·凯撒](尼古拉斯·凯撒.md)**——限他两个月内想出救卡洛斯之法、否则分尸活埋。这源于**暗夜精灵女性的天性**：一旦认定所爱便**「终身绑定」、只有丧偶没有分手**（想解绑？去死才解绑·美丽的花带刺）。
