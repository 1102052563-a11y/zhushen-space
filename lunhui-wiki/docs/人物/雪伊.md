# 雪伊（圣议员 · 伪装图书管理员的女巫坟级强者）

[暗鸦星](../世界/任务世界/暗鸦星-暗鸦纪元.md) [圣愈教会](../势力/index.md) 的**十二圣议员之一**——表面只是图书馆里跳脱话痨的「女神经」管理员（扎斜马尾·爱看小黄书·自称「女巫克星雪伊·克里斯汀」），真身却是圣愈教会最强武力梯队的一员（实力强于圣议员卡门·单独对上五阶女巫不虚）。右臂是构造精密的**义肢**（与摩伊·诺拉交手所失）。她与委托处前台 **雪莉**（先天女神范·温和）是孪生姐妹——这对身份耐人寻味的姐妹让苏晓联想到「五阶白女巫」的可能（待考）。

## 摩伊·诺拉的宿敌 · 促成猎杀计划（二十一卷48–50章）

她是 **[摩伊·诺拉](摩伊·诺拉.md) 的多年宿敌**——「只要我在中原，南部某人就不敢北上一步」（她在牵制摩伊·诺拉、故无法离开中原区域）。她曾多次率队袭杀诺拉、把诺拉「打成肉泥她都能爬出来」、为此失去右臂；正是她以血的经验确认「**破坏 [厄运](厄运.md) 身体使其沉寂半小时 = 杀诺拉的唯一窗口**」。她暗中观察苏晓查机密、一拳试探出他「圣愈教会最强梯队」的战力后，**力促苏晓去猎摩伊·诺拉**、并派活不久的老圣议员们（夕阳红敢死队）去破厄运（赌赢则压制南部·赌输也损失有限）。

## 关系
- **[摩伊·诺拉](摩伊·诺拉.md)**：多年宿敌（互搏多次·为此失右臂）。
- **雪莉**：孪生姐姐（委托处前台·女神范）。
- **[卡门](卡门.md)**：同为圣议员（她派夕阳红敢死队赴南部）。
- **[苏晓](苏晓.md)（库库林·白夜）**：被她试探战力、力促去猎摩伊·诺拉的红猎人。
