# 糖果

[彼岸花冒险团](../势力/index.md)（全女性冒险团）的**奶妈**（治疗位）。三卷围杀 [苏晓](苏晓.md) 时曾以「**接吻治疗**」为被斩穿小腹的团长 [冷月](冷月.md) 回血（三卷28–33章）。八卷11–12章随重整后的六人精英团参加 [二阶生存试炼·吞噬岛](../世界/任务世界/T12K星球-生存试炼.md)，谋向苏晓复仇。这名柔弱少女为团队付出极多（所得资源全用于增强治疗能力）。八卷27–30章恶战中她一直拼命给队友加状态续命，最终在掩护撤退时被 [布布汪](布布汪.md) 一口**咬断喉咙而亡**（彼岸花六女全灭）。

## 出处
- 三卷28–33章、八卷12、27–30章。
