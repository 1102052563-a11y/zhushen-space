# 克利克

**克利克** 是 [海贼王](../世界/任务世界/海贼王.md) 世界 [克利克海贼团](../势力/index.md) 的团长（第18章）。

## 简介
身材高大、身穿黄金战甲，走动时有铠甲碰撞声；狂傲自大，刀上淬有剧毒。东海有名海贼，麾下船队约六百人（第18、19章）。

## 与苏晓的交集（被利用）
苏晓（化名白夜）以哥亚王国「贵族身份」为饵，选中克利克海贼团执行焚烧废品终点站，要其自备炸药，并留下世界政府盖章、不可单方毁约的封爵凭证（第18–19章）。

实为骗局：事成后苏晓不开城门、将海贼团困于火海，再借一名死刑犯投下炸药诱其炸开城门攻入王都——意在引强敌冲击王宫、调开国王身边守卫（第20章）。克利克识破后枪杀「救命恩人」、率众杀向王宫（第20章）。
