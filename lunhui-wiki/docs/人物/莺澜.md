# 莺澜

[火影忍者世界](../世界/任务世界/火影忍者.md) 中现身的契约者——**血门冒险团团长 [斯坦](斯坦.md) 的妻子**。长相娇美却悍妇气息扑面，「能徒手拆高达」、暴力至极，连斯坦都怕她。

- **门内的「清道夫」**：外人面前很给斯坦面子（斯坦说一不二），私下才对他泼辣。血门冒险团成员都怕这位嫂子——若有人背叛冒险团，向来由她出手**清理门户**。
- 鬼眼背叛事发后，斯坦下令她「做掉鬼眼」，她不再多问、起身去执行。
