# 莺澜（奈洛伊 · 血蜘蛛）

[血门冒险团](../势力/index.md) 团长 [斯坦](斯坦.md) 的**妻子**、血门创始三巨头之一——长相娇美却悍妇气息扑面，「能徒手拆高达」、暴力至极，连斯坦都怕她。**名称说明**：九卷（火影）原文称其「**莺澜**」；十七卷96章再登场时称「**奈洛伊**」、外号「**血蜘蛛**」（血门内除斯坦外都直呼「嫂子」）——同为斯坦之妻、清理叛徒的暗刃，应为同一人（名称差异疑为原著前后行文不一）。

- **门内的「清道夫 / 暗刃」**：外人面前很给斯坦面子（斯坦说一不二），私下才对他泼辣。她是血门的**「眼睛」与「暗刃」**——情报、处理背叛血门的成员都由她经手；成员都怕这位嫂子。鬼眼背叛事发后，斯坦下令她「做掉鬼眼」，她不再多问、起身去执行（九卷31章）。

## 血门三巨头 · 与斯坦的「内部矛盾」（十七卷94–97章）

血门金字塔顶端是**斯坦（核心·掌团队重要物资）、[荒焚](荒焚.md)（管下属冒险团·压三阶）、奈洛伊（眼睛 + 暗刃）**三人共建——三人都与血门血脉相连、牵一发动全身，**即便奈洛伊「绿」了斯坦、斯坦也不能除掉她**。这场「斯坦被绿」风波的内情：

- **利用无伞兄**：[无伞兄](无伞兄.md) 其实是被奈洛伊**利用的棋子**（她与无伞兄保持「长期情人」关系、给斯坦扣绿帽子）。夫妻俩感情早已不和、只因利益捆绑还在一起（血门冒险团相当于「夫妻共有财产」、划清界限就等于血门内部分裂）。
- **杀弟之恨 + 提离婚**：斯坦曾**亲手杀了奈洛伊在现实世界作恶的弟弟**（无伞兄的「大舅哥」死得很惨），奈洛伊为此当众向斯坦提「离婚」、要他「以后少提这件事、团队才是首位」。二人互看不爽、却不可能真敌对——**一旦有外敌出现就化为「雌雄双煞」**。苏晓正因这圈子太乱（大坑），坚决不掺和血门与无伞兄的事。
- 苏晓挑战荒焚时，她与斯坦等血门骨干都到竞技场观战、并向黑帆商会重注押荒焚胜。

## 关系
- **[斯坦](斯坦.md)**：丈夫 / 血门核心（互看不爽、利益捆绑、外敌当前仍是雌雄双煞·她杀弟之恨未消、提离婚）。
- **[荒焚](荒焚.md)**：血门副团长（称她「嫂子」、忌惮其算计）。
- **[无伞兄](无伞兄.md)**：被她利用的「长期情人」棋子（引发「斯坦被绿」风波）。
