# 莺澜（奈洛伊 · 血蜘蛛）

[血门冒险团](../势力/index.md) 团长 [斯坦](斯坦.md) 的**妻子**、血门创始三巨头之一——长相娇美却悍妇气息扑面，「能徒手拆高达」、暴力至极，连斯坦都怕她。**名称说明**：九卷（火影）原文称其「**莺澜**」；十七卷96章再登场时称「**奈洛伊**」、外号「**血蜘蛛**」（血门内除斯坦外都直呼「嫂子」）——同为斯坦之妻、清理叛徒的暗刃，应为同一人（名称差异疑为原著前后行文不一）。

- **门内的「清道夫 / 暗刃」**：外人面前很给斯坦面子（斯坦说一不二），私下才对他泼辣。她是血门的**「眼睛」与「暗刃」**——情报、处理背叛血门的成员都由她经手；成员都怕这位嫂子。鬼眼背叛事发后，斯坦下令她「做掉鬼眼」，她不再多问、起身去执行（九卷31章）。

## 血门三巨头 · 与斯坦的「内部矛盾」（十七卷94–97章）

血门金字塔顶端是**斯坦（核心·掌团队重要物资）、[荒焚](荒焚.md)（管下属冒险团·压三阶）、奈洛伊（眼睛 + 暗刃）**三人共建——三人都与血门血脉相连、牵一发动全身，**即便奈洛伊「绿」了斯坦、斯坦也不能除掉她**。这场「斯坦被绿」风波的内情：

- **利用无伞兄**：[无伞兄](无伞兄.md) 其实是被奈洛伊**利用的棋子**（她与无伞兄保持「长期情人」关系、给斯坦扣绿帽子）。夫妻俩感情早已不和、只因利益捆绑还在一起（血门冒险团相当于「夫妻共有财产」、划清界限就等于血门内部分裂）。
- **杀弟之恨 + 提离婚**：斯坦曾**亲手杀了奈洛伊在现实世界作恶的弟弟**（无伞兄的「大舅哥」死得很惨），奈洛伊为此当众向斯坦提「离婚」、要他「以后少提这件事、团队才是首位」。二人互看不爽、却不可能真敌对——**一旦有外敌出现就化为「雌雄双煞」**。苏晓正因这圈子太乱（大坑），坚决不掺和血门与无伞兄的事。
- 苏晓挑战荒焚时，她与斯坦等血门骨干都到竞技场观战、并向黑帆商会重注押荒焚胜。

## 第五次圣杯战争 · 断臂逃命（二十二卷28–31章）

随 [斯坦](斯坦.md) 进 [第五次圣杯战争](../世界/任务世界/fate-第五次圣杯战争.md) 推进人造圣杯计划。决战中她化「上半身人 + 下半身蜘蛛轻机甲」（甲足淌酸性剧毒），与斯坦「雌雄双煞」二打一围攻开了猎魔时刻的 [苏晓](苏晓.md)——却被苏晓近乎无间隔的 [龙影闪](../力量体系/技能.md) 瞬移耍得团团转、断右臂、用一张羊皮纸（替死符）才捡回一命。**眼睁睁看苏晓三刀斩杀斯坦后、她吓得逃了**（按脾气本该战死·却对这「手持长刀、气息如恶兽」的男人心生恐惧）。团队主线失败时她用团队仲裁官权限发「即刻退团」公告、保住 76 名团员不被处决（被斯坦斥「妇人之仁」）。斯坦一死、血门将分崩离析，她还要面对斯坦结下的诸多仇敌。

## 「送死队」感测手 · 暗星深潜（42卷37–38章）

在 [暗星](../世界/任务世界/暗星-世界入侵.md)，一名叫「**奈洛伊**」的**六阶**契约者随 [无伞兄](无伞兄.md)、[小侦探](小侦探.md) 组成的「送死队」被传送来解谜——原文点明「这个名字在契约者间不算响亮，但她还有另一个称呼，就是**斯坦的前妻**」，故与本页同为「斯坦（前）妻·奈洛伊」（**注**：此处称其六阶、名声不响，与她作为血门三巨头之一的高调形象略有出入，疑为原著前后设定/时间线的差异；姑且并记）。她在送死队中担任**感测手**——在精神世界内能发现别人留意不到、却意义重大的隐秘细节与物品（透出微光）。三人靠她的感测 + 无伞兄的不死铺路 + 小侦探的推理，深潜梦境世界、解除「时间停滞」。到时空之力交易结束时，三人在精神世界里累计「死亡」**825 次**、彻底自闭（她呆滞诡异地傻笑）——苏晓的血气投影更把大贤者/老骑士/沙塔耶的投影全吞噬，精神世界只剩那道血气投影、进去 10 秒必死。
