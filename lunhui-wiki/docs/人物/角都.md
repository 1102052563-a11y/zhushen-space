# 角都

[火影忍者世界](../世界/任务世界/火影忍者.md)「**晓**」组织成员、**财迷**——一双绿眼、靠夺取他人心脏维持的「**不死」之身**，与 [飞段](飞段.md) 组成「**不死二人组**（僵尸二人组）」。

- **管钱 / 杀候补**：负责晓的**钱财筹备**（天道交代他「抓紧筹钱、最近开支不小」·常去黑市用尸体换钱）。晓正式编队后，许多候补成员死于内斗——**角都尤甚**（与他组队的人先后被他所杀·直到遇上「打不死」的飞段、屡次未果反丢了两颗心脏）。这也是 [小南](小南.md) 警告 [苏晓](苏晓.md)「小心搭档别被他们杀了」的由来。
- **不和**：与 [干柿鬼鲛](干柿鬼鲛.md) 互看不爽（被鬼鲛叫「僵尸二人组」便绿眼冒杀气·两人早想打一场）——晓「三句不离杀同伴」的日常。

## 封印二尾 / 抓八尾（九卷55章）
封印二尾时与 [飞段](飞段.md) 本体到场。天道全员加速抓尾兽，派**角都 + 飞段去抓八尾**——前途堪忧（八尾属云忍村·四代雷影身体强度极高·连开万花筒的「二柱子」都险被踹死）。苏晓还想起传言「角都曾在『八百里』外向初代火影柱间投手里剑」（实则二人实力天差地别）。

## 殁 · 被血门围杀（九卷58章·侧面交代）
去抓八尾途中，他与 [飞段](飞段.md) 被**[血门冒险团](../势力/index.md)**（约一百名二阶契约者 + 团长 [斯坦](斯坦.md)）围攻致死——上百名二阶契约者足以耗死多心脏的角都。这是火影世界契约者们的又一笔收获。
