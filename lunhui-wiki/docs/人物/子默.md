# 子默

**[血门冒险团](../势力/index.md) 的种子团员**——一个看似 12~13 岁、却举止老成的**天才少年法师**。拥有稀有天赋「**空冥幻想**」（对五种元素先天亲和·学五系元素能力威力 +20%），是难得的「天生法师」。

- **顺风顺水的二阶之路**：第一个衍生世界就被大型冒险团团长看中天赋、破例提为**种子团员**（资源倾斜、编入完善团队、法师 / 牧师大姐姐们格外疼爱），一阶到二阶一路顺遂、前途无量。九卷被同团成员（马脸男子、尖嘴猴腮「老九」）邀来河之国「捞外快」——护送河之国大名（任务难度 Lv.14·奖励丰厚）。
- **死于狙杀（九卷22–23章）**：他一眼认出天上的白色大鸟是 [迪达拉](迪达拉.md) 的黏土鸟、惊觉遇上「晓」——[苏晓](苏晓.md) 在 1 公里高空以蜘蛛女皇 + 制空权**把护卫 / 大名 / 同伴 11 人逐一爆头**，子默藏土沟装死、用 [衍生世界联络平台](../总览/轮回乐园机制.md) 求援却无人敢来（团长斯坦「不会为与晓作对的人出头」·[咕噜](咕噜.md) 还嘲讽挑拨）。最终他被苏晓**打断腿、轰破护盾、爆头**——在没有主坦保护下，第一次尝到轮回乐园的残酷（种子团员的身份、冒险团的名号全无用）。

- **身后的「索赔」（九卷93章）**：火影通关后，团长 [斯坦](斯坦.md) 亲赴乐园找苏晓**索赔 10 万乐园币**（培养子默花了约 50 万·「死人没价值·死掉只说明子默不行」）——把这天才少年的死纯当一笔生意账，被苏晓一口回绝。
