# 黑暗君主·阿伽门（混沌军团统领·被斩首囚回无尽炼狱）

苍龙大陆混沌军团的**绝对统领**——黑暗纪 01 年，灰兽人用一片黑色铠甲残片、从**无尽炼狱**将他唤醒，由此组建了混沌军团（灰兽人 / 古树族 / 蛇人 / 妖鬼四族），在烟囱山地底繁衍军队、与六族同盟开战千年，一度占据近半大陆。他并非无目的地开战——「他在寻找着什么」。苏晓所在的**「混沌军团·阿伽门」**阵营即得名于他。

## 兴衰（世界背景）

- **黑暗纪 1263 年**：六族同盟全线溃败之际，奇迹出现——阿伽门被一位**人类英雄**斩下首级、重新囚回无尽炼狱，混沌军团群龙无首。
- **黑暗纪 1300 年**：紧接着「战争恶魔」（[苏晓](../苏晓.md) 所替换的身份）被封印进王都偏郡的环封树，六族同盟才彻底转入反攻。
- **10 年前**：[老蛇人](老蛇人.md) 试图重新解封阿伽门失败——阿伽门没救出、却放出了自称其女的 [希拉](希拉.md)。

黑暗君主之下设四位**大领主**（战争恶魔 / [老蛇人](老蛇人.md) / [阿卡迪亚](阿卡迪亚.md) / [希拉](希拉.md)），其黑金座椅已很久无人敢坐。谁主导了那两次「太过突然」的封印（阿伽门 + 战争恶魔）、以及本世界的「背叛者」之谜，是本卷核心悬念。

## 关系

- [苏晓](../苏晓.md)（战争恶魔）：名义上苏晓伪装的身份是他「最信任的部下」。
- [希拉](希拉.md)：自称是他的女儿。
- **人类英雄（未名）**：黑暗纪 1263 年斩下他首级的传奇存在。

## 出处

- 三十九卷 1 章（世界简介·黑暗纪 01~1300 年）：被唤醒建军团→被人类英雄斩首囚回无尽炼狱。
