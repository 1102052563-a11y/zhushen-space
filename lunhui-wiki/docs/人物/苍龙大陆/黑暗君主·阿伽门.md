# 黑暗君主·阿伽门（混沌军团统领·被斩首囚回无尽炼狱）

苍龙大陆混沌军团的**绝对统领**——黑暗纪 01 年，灰兽人用一片黑色铠甲残片、从**无尽炼狱**将他唤醒，由此组建了混沌军团（灰兽人 / 古树族 / 蛇人 / 妖鬼四族），在烟囱山地底繁衍军队、与六族同盟开战千年，一度占据近半大陆。他并非无目的地开战——「他在寻找着什么」。苏晓所在的**「混沌军团·阿伽门」**阵营即得名于他。

## 兴衰（世界背景）

- **黑暗纪 1263 年**：六族同盟全线溃败之际，奇迹出现——阿伽门被一位**人类英雄**斩下首级、重新囚回无尽炼狱，混沌军团群龙无首。
- **黑暗纪 1300 年**：紧接着「战争恶魔」（[苏晓](../苏晓.md) 所替换的身份）被封印进王都偏郡的环封树，六族同盟才彻底转入反攻。
- **10 年前**：[老蛇人](老蛇人.md) 试图重新解封阿伽门失败——阿伽门没救出、却放出了自称其女的 [希拉](希拉.md)。

黑暗君主之下设四位**大领主**（战争恶魔 / [老蛇人](老蛇人.md) / [阿卡迪亚](阿卡迪亚.md) / [希拉](希拉.md)），其黑金座椅已很久无人敢坐。谁主导了那两次「太过突然」的封印（阿伽门 + 战争恶魔）、以及本世界的「背叛者」之谜，是本卷核心悬念。

## 混沌军团崩溃真相（重大揭示 · 39卷63–64章）

苏晓循冰封殿后的**燃火之地 / 炼狱之底**找到被封印的阿伽门（十米尖锥石柱 + 上千层术式·几百年都挣不脱），揭开全卷最大真相：

- **功高盖主**：前任战争恶魔（苏晓替换的身份）战无不胜、名望与功绩后期已压过阿伽门，士兵们都感觉沐浴在战争恶魔而非阿伽门的荣光下——阿伽门通过能力偷听到战争恶魔**要取而代之当新黑暗君主**。
- **与多因王合谋**：阿伽门震怒又忌惮（翻脸七成是自己被搞下去），恰逢**多因王秘密联络**「帮你除掉战争恶魔、但苍龙大陆三分之一领土归多因、立血誓」——阿伽门为独占世界 + 保命同意了。于是「**阿伽门 + 战争恶魔双人大礼包**」一起被多因王封印（战争恶魔封环封树 / 阿伽门封炼狱），混沌军团权力体系崩溃、三大领主各自为政（[阿卡迪亚](阿卡迪亚.md) 后来力挽狂澜）。
- **阿伽门的野心**：他要统治全大陆、建多种族共存国度、全力发展**神秘学**以抵御外世界危险（外来强者 / 恶神吮吸世界·他曾目睹一种「将万物化为死寂的力量」毕生胆寒——疑即灭法之影）。
- **苏晓的应对**：阿伽门认不出苏晓是假的、仍视其为「忠诚的战争」，命他烙火焰印**吞噬多因、把多因王带来**——苏晓用灵魂结晶承接烙印（= **操控燃火之地炎兽人大军**的关键），暗知「**阿伽门出封印后必须死一个**」（阿伽门背叛过战争恶魔），已备阿波罗「欢迎」它重获自由。
## 史上最惨大 boss · 殁（39卷82章）

苏晓假意履行阿伽门之命（用多因王血 + 残火灰烬解封）——阿伽门刚挣脱封印、还没来得及「君临苍龙大陆」，就撞上早备好的**45 颗阿波罗连炸 + 死寂降临 / 死寂烬灭**（他唯一恐惧过的**死寂之力**·速度被死寂拖成龟速活靶子·眉心一枪）。他至死不甘「我才刚挣脱封印」——**史上最惨大 boss**（比月神还惨、月神当初还险些杀掉苏晓）。斩杀得遗骸宝箱·死寂 + 16.22% 世界之源、主线最终环完成。苏晓早知「阿伽门出封印后必死一个」（他背叛过战争恶魔·出来第一件事就是杀苏晓重掌军团）。
