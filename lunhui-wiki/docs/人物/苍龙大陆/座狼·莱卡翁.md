# 座狼·莱卡翁（北境座狼族独眼老阴哔·会人言的试验品）

瞎了一只眼、毛发暗淡脱落、腹侧能看到肋骨的老座狼——**北境座狼族真正的决策者**（族中另有一只毛发洁白似雪、精神力极强的**座狼领袖**负责血脉共鸣指挥群狼，但拿主意的是莱卡翁）。他胸腹刻着**术式**：曾被多因帝国的魔法师买去当**试验品**、由此获得了不低于人类的智慧、还能**口吐人言**。座狼族血脉本就有超凡特性、智慧远超普通狼。

## 处境 · 老阴哔的算计

座狼族被极冰领主逼到绝境（十几年前极冰领主的妻女被座狼族咬死、他从此铠甲融身、不死不休）：一旦混沌军团被灭、极冰领主抽回兵力就会来清算座狼；投混沌军团（风中残烛）= 战后被六族同盟剿灭；投六族同盟又没人肯为它得罪极冰领主。莱卡翁是**狼族里的老阴哔**——他既想搭混沌军团这条线求生、又**不能明面上投靠**（否则座狼族战后必被围剿）。

## 与苏晓的「假抓捕」合作（39卷21–22章）

[苏晓](../苏晓.md)（战争恶魔）看穿了这点——率几十万兽人大军来「**假装抓捕**」座狼：莱卡翁先摆出拒绝姿态、放三只强壮座狼去「求援」做样子，苏晓一刀架住座狼领袖脖颈生擒，给所有座狼**戴上金属环镣**（实为炼金加固阵图的幌子·屁用没有）——这样座狼族对外就是「**被战争恶魔强行抓捕、被迫沦为爪牙**」的受害者，战后六族同盟最多让它们付点代价、不会派大军围剿。莱卡翁配合演到底（噗通倒地装被控制·「混沌军团你们不会有好下场」），签下**狼血之誓**，54 万座狼 + 灰兽人组成 **[狼骑兵](../../力量体系/装备道具.md)**。他被关铁笼里当「人质」随军，越看越心惊——这合作者准备得太充分了（连狼甲鞍甲都早备好·不联合就真抓捕、不服就杀到服）。

## 关系

- [苏晓](../苏晓.md)（战争恶魔）：看穿并配合他「假抓捕」求生算计的合作者（准备得令他心惊）。
- **座狼领袖**（白狼）：精神力极强、以血脉共鸣指挥群狼的名义首领（演技比莱卡翁差很多）。
- **极冰领主**：座狼族的死仇（妻女被座狼咬死）。
