# 佩佩尼（法师学徒·受幸运女神祝福·苏晓的采血诱饵）

束着淡金发、身穿学徒法师袍的少女——多因帝国的宫廷法师学徒。她跟导师（宫廷老法师）到王都偏郡禁区偷看环封树术式，撞上灰兽人袭击 + [苏晓](../苏晓.md)（战争恶魔）破封脱困，同门尽死、自己因**心脏长在右侧**躲过必死一击而幸存。她满脑子「主角开局」的少女妄想（幻想获得力量传承、当挽救苍龙大陆的女英雄、邂逅英俊帝国军官），却一路被苏晓的血腥现实反复暴击、胆量飙升、逐渐认命。

## 「幸运」之谜（受幸运女神祝福）

苏晓侦测发现佩佩尼的幸运属性异常——初始 5 点（远高于常人）、经历险象环生后竟涨到 **12 点**。她并非世界之子、身上也无隐藏力量，这突兀的提升指向**幸运女神**：苏晓两度想「顺手」宰了她，都莫名在大晴天遭雷劈（第一次可推给术式、第二次无从解释）。他由此判定**幸运女神拿佩佩尼当保命符 / 诱饵**（幸运女神被先代灭法吓得躲藏、伤好后找上了苏晓来采血）——于是佩佩尼被留作**采血诱饵**（苏晓能随时得知她位置、也能让她短时间暴毙）。

## 经历（39卷2–13章）

- **脱困人质**：被苏晓空间阵图带出王都偏郡（「落汤鸡」呕吐）、指错方向反而指对了烟囱山。
- **越狱暗桩**：苏晓放她伙同一名大贵族越狱回多因（幸运女神只是能量投影、佩佩尼失去人质价值后，放出去反而可能有意外收获·苏晓借此暗查虚空之树果实）——她越狱时还把倒下的牢门费劲立回去、缩回角落（「外面全是灰兽人不安全」）。

## 关系

- [苏晓](../苏晓.md)（战争恶魔）：她口中的「导师」（先古面具伪装）、实为掌控她生死的猎杀者、拿她当采血诱饵。
- [幸运女神](../幸运女神.md)：暗中祝福她、拿她当保命符的神灵。
