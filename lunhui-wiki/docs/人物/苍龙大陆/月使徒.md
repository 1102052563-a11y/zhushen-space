# 月使徒（天启乐园·召唤系·可移动军团·战功榜劲敌）

天启乐园的七阶契约者——**超强召唤系**，据说是「一个可移动的军团」。她十七八岁的模样、气息清澈得几乎没有杀意，因为她**与敌人战斗基本不亲自出手**：先把自己藏进空间系使魔体内，然后就等外面的敌人被大群召唤兽围攻致死。召唤系的日常枯燥（「就不能给我点惊喜吗～」）。

## 经历（39卷11–17章）

- 在战区北部的冰洞孵化召唤兽——已召 17560 只、想先攒到 **5 万只**当给 [灰绅士](../炎灵星/灰绅士.md) 的「惊喜」。
- 战功榜上一度稳居前二（榜眼）——直到看见 [苏晓](../苏晓.md)（白夜）攻破焱谷要塞、单场狂揽 **19 万战功登顶**，她在冰原惊呼「为什么会有人获得 19 万战功啊！」

## 关系

- [苏晓](../苏晓.md)（白夜）：战功榜上把她挤下首位的轮回乐园对手。
- [灰绅士](../炎灵星/灰绅士.md)：她想给「惊喜」、疑有默契 / 合作的对象。
- [焰火师](焰火师.md) / [牧枭](../原始大陆/牧枭.md) / [咕噜](../咕噜.md)：同轮战功榜竞争者。
