# 王剑·盖特（多因大决战总指挥·图尔特之父·格雷厄姆骑士家族）

年过六旬却依旧强壮、气息威严的军团长——多因帝国「**格雷厄姆骑士家族**」家主、多因王最信任的几人之一，也是**多因与混沌军团大决战的战场总指挥**（虚空之树公告称他为「**王剑·盖特**」·领袖级单位）。他是军团长 [图尔特](图尔特.md) 的父亲（六名子嗣中最引以为傲的就是图尔特）。

## 骑士精神 · 大决战（39卷70–73章）

图尔特战死后（遗体被苏晓完整送回），盖特没把它当私仇——「图尔特是以骑士之名战死、遗体被完整送回代表战争恶魔承认他值得尊重、这是**荣誉之死**」（格雷厄姆家族的骑士精神：参与杀戮就要有面对死亡的准备）。多因王命他「为次子复仇」，他率 600 万大军与混沌军团决战：

- **战术老辣**：靠**冰翼**（抽亚塔德之壁冰能量的能量生物·低温削炎魔 25%）一度把 470 万炎魔打到割麦子般倒下、多因气势如虹。
- **识破陷阱**：正当众副指挥官欢呼胜利，盖特却察觉不对、下令退军——可为时已晚，苏晓的**470 万炎魔 + 火焰巨魔全体扯胸自爆成「岩浆之花」**（多因大部队死 80%·一换一）。盖特「仿佛苍老了十几岁、鬓角白了」，明白「**从最开始敌人就没准备决战、整个战场都是陷阱**」；发现狼骑兵已绕后直扑王都，急令全军回援却只能吃灰（步兵追不上骑兵）。

## 关系

- [图尔特](图尔特.md)：他最骄傲的次子（战死·他视之为荣誉之死不寻私仇）。
- [苏晓](../苏晓.md)（战争恶魔）：大决战的对手（被其「整个战场都是陷阱」的谋划坑得大败）。
- [多因王·格莱斯顿](多因王·格莱斯顿.md)：他世代效忠、最信任他父子的君主。
