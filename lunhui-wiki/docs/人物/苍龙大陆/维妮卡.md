# 维妮卡（塔尔塔·维妮卡·赞格之妹·感知系·被苏晓血气烙印）

银发少女——衰败的**塔尔塔探险家族**仅存的两兄妹之一、世界之子 [赞格](赞格.md) 的妹妹。她是**感知系**：将精神力融入水中、以水为媒介移动，达成远程侦查（除非在沙漠、否则难避周边有水）——被赞格派去**监视战争恶魔**。

## 被苏晓血气反噬 · 血气烙印（39卷29章）

维妮卡远程侦查 [苏晓](../苏晓.md) 时，苏晓以血气侵入她的精神力（血气无侵蚀性、却令精神力暴走失去理智）——这就是「**感知连逆**」（超远程感知者精神力受影响会反馈本体）。她当场失控、割伤胖矮人；[赞格](赞格.md) 徒手夺刀 + 按住她（骨断多处）；她手心涌出全身三分之一鲜血化血气、身后浮现操控她的**血兽虚影**（苏晓远隔几十公里只能影响、无法操控）。虚影消散后她昏倒——**背脊皮肤上悄然留下一道血色印记**（苏晓可远程感知其位置、成了后来揪出主角队潜入的关键）。事后靠 [狄牧耶](狄牧耶.md) 的森林女神之力痊愈，只记得那是「噩梦」、选择性遗忘。

## 冰封殿 · 液体化自尽（39卷59–62章）

她随哥哥 [赞格](赞格.md) 转投北境、参与冰封殿 7v1 弑父夺权 + 围攻 [苏晓](../苏晓.md)——全程被苏晓的血气压得精神模糊、双腿打颤（血气烙印的旧噩梦）。见兄长赞格被斩杀后，她突然「有骨气」、全身**液体化爆开自尽**（放逐刺了个空），不愿落入苏晓之手。
## 关系

- [苏晓](../苏晓.md)（战争恶魔）：以血气反噬她、在她背脊留下追踪烙印的猎杀者。
- [赞格](赞格.md)：徒手救她、护她的世界之子哥哥。
- [狄牧耶](狄牧耶.md)：治好她的森林女神（绿茶婊）。

## 出处

- 三十九卷 29–37 章：远程监视战争恶魔被血气反噬→留血气烙印→随赞格潜入救白狼。
