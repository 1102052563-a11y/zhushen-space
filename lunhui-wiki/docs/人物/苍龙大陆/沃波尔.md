# 沃波尔（原灰兽人「乌古」·被苏晓改造+造神的兽人英雄）

满脸疤痕、比其他灰兽人聪明的灰兽人——原名 **乌古**（灰兽人语「铁矿」）。[苏晓](../苏晓.md)（战争恶魔）征召败兵后挑头目，他一挺身「我是最强的乌古」、几秒揍趴周围几名灰兽人自荐，被委任为**领主**（灰兽人军官体系：大领主 > 领主〔各统 15~30 万〕> 掠夺者）。他献上多因帝国印徽的精密战场地图、坦言「我们能战斗的战士，不多了」。

## 造神 = 兽人英雄（39卷10章）

苏晓独挑上万帝国军获胜后，把功劳**全安到没逃的乌古头上**——当场斩杀一名说「没看到」的灰兽人立威，逼所有人「都看到了、只有乌古一人杀退上万帝国军」。理由是：苏晓身为战争恶魔杀敌对名望加成几乎为 0（初始身份太高），而灰兽人**急需一个能喊「跟我冲」、带头冲锋的英雄**（大领主坐镇后方才能让士兵安心死战）。乌古由此被赐名「**沃波尔**」（灰兽人语「烈阳」）、野心被点燃、成为苏晓的**傀儡先锋**。

## 改造 · 战功

- **等离子切割锯改造**（39卷13章）：苏晓用炼金手段给他做痛苦改造（哀嚎整夜），换来「能冲锋在最前方而不死」的实力——焱谷要塞战中挡精灵战士偷袭（十字剑刺胸「你在给我挠痒吗」）、护住降兵。
- **战前动员**：焱谷战前照巴哈写的稿子动员卡壳，急中生智喊「抢来粮食、吃软烂炖肉、塞满煮土豆」把兽人士兵全喊饿了、士气爆棚（「跟我冲」达成）。给饿肚子的小灰兽人分炖肉（「我的崽子没死也有你这么高了」）。
