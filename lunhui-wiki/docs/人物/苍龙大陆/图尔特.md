# 军团长·图尔特（多因帝国圣徽骑士团统领·忠诚重于信念的传承骑士）

身穿金纹铠甲、披蓝色披风、棕发自然卷、利剑上有洗不掉血痕的军官——多因帝国**第二军团·骑兵先遣团军团长**，统领多因两支最尖锐部队之一的**圣徽骑士团**（62 万编制）。他生来是大贵族，却当过底层士兵、在军团战旗下宣誓「誓死保卫多因帝国」——**平常待人随和、战时极为铁血**，出了名的**护犊子**（惹他手下最多遭惩戒），受爱戴亦受敬畏，把麾下士兵当兄弟。见到恶魔焰龙 [巴巴托斯](../苍龙大陆群像.md) 不惊反艳羡（谁不想当龙骑士）。

## 忠诚 vs 信念（39卷51章）

图尔特**早知多因高层想在灭北境后背刺盟军（狼骑兵军团）**——互相定位的两颗术式宝石本就是多因高层设的圈套，他得知后暴怒，但「信念与忠诚有时矛盾，他选择了忠诚」（世代为多因效力）。即便如此，作为最前线的指挥，他对 [苏晓](../苏晓.md) 展露出十足诚意：主动坦白亚塔德之壁是苏晓打穿的（尴尬）、把圣徽骑士团当**剿灭重锤部队的诱饵**（自己军团当先锋伏击、让狼骑兵趁机突袭）——「没人是傻子、要诚意才不内讧」。

## 圣徽骑士团 · 重锤之战（39卷38–52章）

圣徽骑士团用术式破开亚塔德之壁攻入北境、一路杀穿断补给，却被北境**重锤部队**（会飞的四米超重甲巨怪）从天而降捶懵——图尔特负伤（换别的部队早全军覆没）。他与苏晓联手：圣徽骑士当诱饵伏击北境后勤部队、引出重锤部队决战（重锤俯冲砸一锤即飞、圣徽死伤惨重·骑兵利剑术式击中即引爆重锤铠甲、却挡不住三颗心脏的巨怪）。
