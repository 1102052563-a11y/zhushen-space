# 希拉（混沌军团四大领主·自称阿伽门之女·来自无尽炼狱）

外表与人类相近、脖颈 / 脸颊 / 手背布满黑色纹路的女子——[混沌军团](../../世界/任务世界/苍龙大陆-背叛者之末路.md) 四位大领主之一。她**来自无尽炼狱**（封印着黑暗君主·[阿伽门](黑暗君主·阿伽门.md) 的地方）：10 年前 [老蛇人](老蛇人.md) 试图解封阿伽门失败，阿伽门没救出来、希拉却借机脱困。她**自称是黑暗君主·阿伽门的女儿**，加上「无尽炼狱」这重身份，才当上大领主——但**基本没有实权**，主要是给阿伽门颜面（毕竟她的确有可能是阿伽门之女·一切关于她都是神秘未知）。

## 性格

有点爱挑衅、却也懂分寸——见 [苏晓](../苏晓.md)（战争恶魔）便试探「听我父亲说你是他最信任的部下」，被 [阿卡迪亚](阿卡迪亚.md) 提醒「别挑衅白夜、他会杀你（有五任大领主死在他手中）」；战争议会里她只顾修指甲、深知自己**别跟着搅局**就是本分（她某种程度上代表无尽炼狱、这对混沌军团就足够了）。

## 疑为阿伽门的外界监视者（39卷64章暗示）

苏晓在燃火之地推断：燃火之地的**炎兽人大军**（岩浆 + 熔石 + 火焰能量 + 十几年间战死灰兽人的灵魂构成）需要「足够强大的妖鬼收拢战死者灵魂、以特定方式送入燃火之地」才能形成——而自称阿伽门之女、多年来「只负责美如画」的希拉，位置就很微妙：她更像是**阿伽门安插在外界的监视者**（替被封印的阿伽门源源不断供应灵魂原料），绝非真的无所事事。
## 被揭穿关押（39卷80章）

苏晓斩多因王 / 取果实后揪出阿伽门的两名眼线——希拉正是其一（自称阿伽门之女、多年「只负责美如画」，实为**阿伽门安插在外界的监视者**、替其供应炎兽人的灵魂原料）。她被狼骑兵围住、锯刃大刀架脖时坦然「杀我是没问题的」，供出**另一名眼线 = 妖鬼女仆谭雅坦（实为鬣狐鬼姬附身）**后，被押走关进秘密监牢。
## 关系

- [黑暗君主·阿伽门](黑暗君主·阿伽门.md)：她自称的父亲、封印她 / 她脱困之处的军团统领。
- [苏晓](../苏晓.md)（战争恶魔）：她试探挑衅、却被警告别惹的大领主同僚。
- [老蛇人](老蛇人.md)（解封阿伽门失败却放出她的人）/ [阿卡迪亚](阿卡迪亚.md)：另两位大领主同僚。
