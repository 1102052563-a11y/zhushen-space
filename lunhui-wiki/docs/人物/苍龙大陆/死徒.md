# 死徒（轮回乐园最疯梯队·黑白蔷薇自爆流·大决战来袭）

笑眯眯、很有礼貌的少年——轮回乐园里**最疯狂梯队**的契约者（与黑血、小胖子等同级）。多因与混沌军团大决战时，他专程来找苏晓（白夜）「打一场」：「这次不是世界争夺战、不用一致对外，有失礼的地方多见谅」——先用白色珊瑚状物质困住苏晓右臂打招呼、又抛金币自嘲「我今天死的概率很高」。

## 黑白蔷薇 · 被踢飞（39卷71–72章）

死徒祭出 **黑白蔷薇**（[苏晓](../苏晓.md) 也造不出的**科技侧 + 神秘侧结合爆炸物**·遭电磁冲击后延迟 2.5~3 秒爆炸·直径 2000 米白光球 + 黑洞般重力吸碾·500 灵魂钱币一颗的常规同归于尽补给）——苏晓在他自爆前用龙影闪绕后、一脚把他上踢飞到半空，避开爆炸（得**猩红之勋 ×3**）。这一爆把附近的 [牧枭](../原始大陆/牧枭.md)、[咕噜](../咕噜.md) 都顶飞（咕噜自己也备着两颗黑白蔷薇·1000 钱币一颗兜售）；也吓得再没契约者敢来袭击战功榜首位的苏晓。

## 关系

- [苏晓](../苏晓.md)（白夜）：他专程来「打一场」、被一脚踢飞的轮回乐园同乐园强者。
- [咕噜](../咕噜.md) / [牧枭](../原始大陆/牧枭.md)：认得他、被其自爆波及的同轮契约者。
