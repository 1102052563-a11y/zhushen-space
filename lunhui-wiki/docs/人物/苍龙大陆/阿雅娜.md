# 阿雅娜（极冰领主长女·心机蛇蝎美人·弑父夺权·殁）

身穿紫色高叉长裙、皮肤白到不正常、红唇妖艳的女人——**极冰领主的长女**（极冰领主三子嗣：长子 [冰铠](冰铠.md)、长女阿雅娜、次女 [安塔·埃米](../苍龙大陆群像.md)）。她驾驭**极光的力量**（水桶粗光柱 + 光傀偶驱动身体闪避），实力能进本世界前三梯队。极冰领主真正视为亲人的一对母女早年被座狼所杀、从此他冰冷如铠——三个子嗣在他眼里根本不算子嗣（长子被当看门狗、次女变成怪物），阿雅娜的皮下被剥开刻满印记再缝合，她选择戴上「面具」与父亲同流合污。

## 弑父夺权 · 结局（39卷58–62章）

她与被虐得更惨的兄长 [冰铠](冰铠.md)、以及世界之子 [赞格](赞格.md) 主角队、军团长 [图尔特](图尔特.md) 等**八人联手弑极冰领主夺权**（想让她当北境女王、再联合圣徽夺回凛冬城）——却在快胜时撞上杀进冰封殿的 [苏晓](../苏晓.md)。她放了咕噜、假意投降拖时间失败；七个打一个被苏晓按地上捶（她与晶体长老最有默契）。最后她求生欲爆棚、以「主掌调动补给、可让城外冰裔退军」保命，供出**虚空之树果实真相 = 一块木雕**（触感如金属·两年前她父亲赠多因王换东西）——话音刚落被苏晓一记手刀斩喉。苏晓评价：这种「聪明 + 隐忍 + 愿屈从死敌的蛇蝎美人」远比绿茶婊危险、留着必阴沟翻船（她眼中瞬间爆发的仇恨让苏晓心悸）。

- [冰铠](冰铠.md)（长兄）/ [安塔·埃米](../苍龙大陆群像.md)（幼妹）：同样被极冰领主残害的兄妹。
- [苏晓](../苏晓.md)（战争恶魔）：杀她兄长、逼她供出果实真相后斩杀她的猎杀者。
- [赞格](赞格.md) / [图尔特](图尔特.md)：与她联手弑父的世界之子 / 圣徽军团长。
