# 瑟莱芙雅（多因帝国公主·主角队「红发女」·殁）

满头酒红色波浪长发、身穿紧身皮甲、身材劲爆的神秘女人——[赞格](赞格.md) 探险小队（主角队）的成员「**红发女**」，几个月前才加入、一切都很神秘。她身手矫健、用两把形状奇异的宽刃短刀近战，是队里的暗杀 / 战斗担当。真实身份是**多因帝国的公主·瑟莱芙雅**（隐藏在世界之子的主角队里）。

## 结局（39卷37章）

随 [赞格](赞格.md) 术式潜入新月要塞救座狼领袖白狼时，被苏晓凭 [维妮卡](维妮卡.md) 背脊的血气烙印当场逮住——她率先拔双刀冲向苏晓，被 [苏晓](../苏晓.md) 一个瞬移**斩喉贯穿**（双手死死抓着斩龙闪刀锋、「快……走」话没说完就倒地）。击杀提示揭穿她的真身：**多因帝国公主·瑟莱芙雅**（6.52% 世界之源 + 圣灵级宝箱）——多因的公主竟混进了世界之子的主角队，耐人寻味。

## 关系

- [苏晓](../苏晓.md)（战争恶魔）：斩杀她、揭穿其多因公主身份的敌人。
- [赞格](赞格.md)（世界之子）：探险小队队长（她在他面前被斩杀、他嘶吼着想救却晚了）。
- **多因帝国 / 多因王**：她隐藏的真实出身。

## 出处

- 三十九卷 29–37 章：主角队神秘红发女→救白狼被苏晓斩喉→揭真身多因公主·瑟莱芙雅。
