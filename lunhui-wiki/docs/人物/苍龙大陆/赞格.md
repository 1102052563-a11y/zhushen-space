# 赞格（塔尔塔·赞格·苍龙大陆世界之子·探险小队队长）

满头银色短发的年轻男人——**本世界的世界之子（受世界眷恋之人）**，出身**塔尔塔探险家族**（世代在大陆各地探险、发现众多古迹陵墓、著有《苍龙纪元兴衰历》·北境 / 多因 / 艾尔大森林都给这家族薄面）。家族已衰败、只剩他与妹妹 [维妮卡](维妮卡.md) 两人 + 一座古堡。他组建**探险小队**（隐性队长）：赞格 + 妹妹维妮卡 + 胖矮人（术式师）+ 红发女（[多因公主·瑟莱芙雅](瑟莱芙雅.md)）——即苍龙大陆的「**主角队**」。

## 世界之子的表现（39卷29章）

维妮卡被 [苏晓](../苏晓.md) 血气影响失控（血兽虚影操控·全身骨断）时，赞格**徒手抓住刀锋夺下短刀、单手把失控的妹妹按在地上**——[狄牧耶](狄牧耶.md) 能感觉到「世界在眷顾这个男人」。他意志坚定、临危不乱、在死亡压迫下**体内力量完全迸发**（面对苏晓的刃道刀·时也能鬼使神差低头躲过斩击）；谋而后动、懂得给自己和队友留退路。

## 与战争恶魔为敌 · 断臂（39卷29–37章）

赞格坚信「正因为杀战争恶魔不是他的责任、是六族同盟的敌人，他才选择与之为敌」（见过被攻占要塞的炼狱之景·认定混沌军团 = 混乱杀戮）。被绿茶婊 [狄牧耶](狄牧耶.md) 忽悠去**新月要塞救「可怜的座狼领袖白狼」**（想让座狼倒戈）——四人术式潜入，却被苏晓凭血气烙印当场逮住：眼睁睁看红发女 [瑟莱芙雅](瑟莱芙雅.md) 被斩喉、自己**被斩断右臂**才靠预设的撤退术式逃走。苏晓放他走——「世界之子会主动送上门、只是时间问题」。

## 关系

- [苏晓](../苏晓.md)（战争恶魔）：世界赋予他「杀战争恶魔」的使命、却被斩断右臂放走的宿敌。
- [维妮卡](维妮卡.md)：他的感知系妹妹（被苏晓血气烙印）。
- [瑟莱芙雅](瑟莱芙雅.md)（红发女 / 多因公主）：探险小队成员、在他面前被苏晓斩杀。
- [狄牧耶](狄牧耶.md)：忽悠他去救白狼的绿茶婊森林女神。
- **精灵女王**：笃信他会来终结战争恶魔的六族同盟之首。
