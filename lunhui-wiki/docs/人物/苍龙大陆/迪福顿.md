# 外交官·迪福顿（多因帝国外交官·拉拢狼骑兵军团的谈判者）

笑眯眯的多因帝国中年外交官——六族同盟崩解、多因与北境即将开战之际，代表**多因王**来拉拢 [苏晓](../苏晓.md)（战争恶魔）的谈判者。他是普通人却是人精（战争年代能爬到这位置的都是人精），口才滔滔、擅画饼——「多因取中部 + 北境、混沌军团得大半南境 + 西境、换苍龙大陆的和平」。凭术式之力能快速往返前线。

## 被巴哈 / 苏晓反复拿捏（39卷43–49章）

- **谈判被吊打**：他滔滔不绝半小时、苏晓一个字不说，最后由 [巴哈](../剧毒魔鹰·巴哈.md) 出面把他拿捏得死死的（多因拖不起·每拖一天主战场都在死人）——达成「西境 17 座要塞全归混沌军团、狼骑兵深入北境断补给 + 62 万圣徽骑士团同行」。
- **圣银币之悔**：他为展示多因术式、随手甩出一枚 **[圣银币](../../力量体系/装备道具/index.md)**（多因术式货币·外界无用）当「纪念品」——殊不知它对乐园人有致命吸引（1 枚 = 15000 乐园币）；苏晓当即狮子大开口「1500 万帝国金币 + 15000 枚圣银币」剿重锤部队，他悔得想抽自己嘴巴。
- **破壁之悔**：他冒失去性命的风险终于把狼骑兵拖入战场（成就感满满）、还得亲口「感谢」苏晓破坏了亚塔德之壁——却隐隐觉得这不符合「上面那位」的意愿。

## 关系

- [苏晓](../苏晓.md)（战争恶魔）/ [巴哈](../剧毒魔鹰·巴哈.md)：被他们反复拿捏、被迫层层加码的谈判对手。
- [图尔特](图尔特.md)（军团长）/ [奥芙](奥芙.md)（医师长）：同属多因 / 圣徽体系的同僚。
- **多因王**：他代表、忠实执行「联合后背刺盟军」意志的君主。
