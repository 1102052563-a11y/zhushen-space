# 阿卡迪亚（混沌军团四大领主·战争部队指挥·灰兽人×冰裔混血）

头生双角、沉默寡言的大领主——[混沌军团](../../世界/任务世界/苍龙大陆-背叛者之末路.md) 四位大领主之一，**灰兽人与冰裔的混血**，负责指挥所有混沌军团的战争部队。以这种「杂血」身份能登上大领主之位，足见其能力与贡献（混沌军团讲能力不讲出身、再低微只要有本事都能上位）。

## 性格 · 立场

态度微妙——看似力挺归来的 [苏晓](../苏晓.md)（战争恶魔）、实则**不愿盲目押上全部兽人部队**（混沌军团已赌不起、输一次就是亡族灭种）。他清醒地认识到混沌军团「担负不起大规模战争、战士不能饿着肚子上战场」，却也在苏晓身上看到一丝希望——最终交出约 **35 万兵力**（30 万灰兽人 + 500 古树族 + 500 妖鬼）作豪赌前的试探。堂堂混沌军团要因「没粮食」而崩溃，让他苦笑「因饥荒战败，太丢人了」。他也清楚苏晓「有五任大领主死在他手中」、谈不拢可能被当场斩杀，故言辞谨慎。

## 关系

- [苏晓](../苏晓.md)（战争恶魔）：他谨慎配合、拨兵支持的大领主同僚（暗自防着被斩）。
- [老蛇人](老蛇人.md) / [希拉](希拉.md)：另两位大领主同僚。
