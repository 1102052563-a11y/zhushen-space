# 老蛇人（混沌军团四大领主·财政物资·和气却极记仇）

全身缠着灰布的老蛇人——[混沌军团](../../世界/任务世界/苍龙大陆-背叛者之末路.md) 四位大领主之一，负责**战争物资与财政**（没有他，军团只能空手打仗）。他是四大领主里最没「领主威严」的一个，看似和气好说话，实则**极记仇**（踩他一脚能记住六七年）。十几年过去、三大领主里只有他没更替。

## 经历（39卷11–18章）

- **让权迎归**：[苏晓](../苏晓.md)（战争恶魔）归来时，老蛇人与另两位大领主直接让权、把苏晓迎进黑环要塞休息（省去夺权麻烦）。
- **10 年前解封事故**：他曾试图重新解封黑暗君主·[阿伽门](黑暗君主·阿伽门.md)，失败了——阿伽门没救出来、却让 [希拉](希拉.md)（自称阿伽门之女）从无尽炼狱脱困。
- **筹粮 / 后勤**：混沌军团断粮在即（总储备才 6 万吨·再撑一个月就崩），他只凑得出「20~30 万人吃 4 天」的军粮支持苏晓出征（在黑熔城征粮被饿极的灰兽人大妈挠了一脸）；焱谷要塞大捷缴 41620 吨谷物运回、他发 3000 吨给平民庆祝（黑熔城久违地恢复生气）。

## 关系

- [苏晓](../苏晓.md)（战争恶魔）：让权迎归、为其筹粮备战的大领主同僚（帮他布置炼金实验室 / 安排佩佩尼牢房）。
- [阿卡迪亚](阿卡迪亚.md) / [希拉](希拉.md)：另两位大领主同僚。
- [黑暗君主·阿伽门](黑暗君主·阿伽门.md)：他试图解封却失败的军团统领。
