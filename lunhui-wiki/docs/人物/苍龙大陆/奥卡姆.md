# 背叛者·奥卡姆（炎兽人·炼狱守门者·当年秘辛的见证者）

皮肤干瘪、下巴胡须如蛇般扭动、眼中燃火、全身遍布疤痕找不到一块完整皮肤的**炎兽人**（称「炎兽人」更贴切）——守在冰封殿后的**燃火之地**入口。他是**炎兽人中极特殊的有智慧个体**，也是黑暗君主 [阿伽门](黑暗君主·阿伽门.md) 的旧部；当年他号称「**暴怒·奥卡姆**」，曾独自杀入敌军把被围的 [老蛇人](老蛇人.md) 救出（战后身上找不到一块完整皮肤）、无畏过。

## 谜题 · 臣服（39卷63章）

他给闯入者出「**一杯永生 / 一杯死亡·只有一次机会**」的谜题——[布布汪](../布布汪.md) 先被困住，[苏晓](../苏晓.md) 一到就用「让出题人自选」破题（两杯都是剧毒、一起喝则不死），奥卡姆当场把两杯灌下。苏晓由此**解开背叛者·奥卡姆的谜题、得「炼狱之庇护」**（免火之侵蚀·因苏晓是混沌军团绝对掌控者、火之魔灵对他极其畏惧、可完全豁免）。奥卡姆一见苏晓（战争恶魔）就惊愕畏惧跪地「不是我背叛」——他是**当年秘辛的见证者**（知道阿伽门为何被封、战争恶魔为何被封在环封树），却一个字都不敢提（提一字就被烧成灰烬），只劝苏晓「现在一切不是很好吗、您还是离开吧」。他为苏晓带路到炼狱之井（封印阿伽门处）。

## 关系

- [苏晓](../苏晓.md)（战争恶魔）：他畏惧臣服、为其带路见阿伽门的旧主（他谜题被苏晓破解）。
- [阿伽门](黑暗君主·阿伽门.md)：他效忠、守护其封印的黑暗君主。
- [老蛇人](老蛇人.md)：当年他拼死救出的大领主。
