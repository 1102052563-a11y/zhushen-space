# 狄牧耶（森林女神·顶级绿茶婊·维京之海背叛者之一·知虚空之树果实）

满头绿色长发、衣物由植物构成、身处其附近即获天然治疗的存在——「**狄牧耶**」是精灵语音译，意为**森林女神 / 自然精灵 / 兽语者**（须极强自然亲和力·除恶特性野兽外所有动物都亲近她）。她治好了被苏晓血气反噬的 [维妮卡](维妮卡.md)。她是**本世界顶级的「绿茶婊」**——演技无可挑剔、连精灵女王都忌惮她。

## 真身 = 虚空之树果实的背叛者之一（39卷36–37章·重大揭示）

狄牧耶正是 [苏晓](../苏晓.md) 进入本世界要找的**「背叛者」之一**——当初带原生世界土著从苍龙大陆脱离、抵达维京之海（[永恒意志](../../力量体系/装备道具.md) 所记的圣光乐园叛徒事件）争夺**虚空之树果实**的几人之一。她望着银月心念「那颗果实到底在哪、还在北境吗、或者已经到了多因？当初背叛那个人、或许是错误的选择」——**坐实虚空之树果实曾在她们手中、如今已辗转到多因王都**。

她用绿茶婊话术**忽悠世界之子 [赞格](赞格.md)**去新月要塞救「可怜的座狼领袖白狼」（暗示救出白狼座狼就会倒戈助他对抗战争恶魔）、给赞格能听懂兽语的宝石——把主角队推向苏晓的刀口（赞格断臂 / 瑟莱芙雅殁）。在她的观念里战争只是立场敌对（若混沌军团败、下场只会更惨）。

## 关系

- [苏晓](../苏晓.md)（战争恶魔 / 猎杀者）：进世界要找的「虚空之树果实背叛者」之一（她尚不知库库林就是来夺果实的猎杀者）。
- [赞格](赞格.md)（世界之子）：被她绿茶话术忽悠去救白狼、断臂的棋子。
- [维妮卡](维妮卡.md)：被她治愈的赞格之妹。
- **精灵女王**：忌惮她的六族同盟之首。
