# 多因王·格莱斯顿（苍龙大陆真正的枭雄·永生者伪·多因术式巅峰）

坐木质轮椅、盖毯子、老眼浑浊却与之对视便觉压迫的苍老国王——多因帝国的**格莱斯顿王**。他是本卷真正的幕后枭雄：靠割地 + 联姻 + 谋略搅乱各方，**与黑暗君主 [阿伽门](黑暗君主·阿伽门.md) 秘密合谋**（割三分之一大陆立血誓）把「阿伽门 + 战争恶魔」双人大礼包一起封印（这才是苏晓一进世界就被封在环封树的根源）；动辄「送公主联姻」（多因公主多为速成培养的貌美心机女·[瑟莱芙雅](瑟莱芙雅.md) 亦然、他根本不在乎被杀）；灰绅士潜入王宫也险被他烧死赶走。

## 永生者（伪） · 多因术式巅峰（39卷77章·使徒之眼侦得 30.5% 资料）

- **类别：永生者（伪）**——他很可能已活了不知多少年（多因或从没换过国王）、能力是**庞大生命力 + 焚尽一切的火焰**（炼狱之力·焰）。六维力 190 / 敏 192 / 体 187 / 智 191 / **魅 159**，永生之念（生命 +37000·减 75% 火伤·八阶判定）、火之裁罚（+715 火真伤 + 灼血）。
- **技法型的可怕**：**战斗宗师**（擅 6 种武器）+ **自行开发能力 12 种**（原 295 种·经实战与漫长岁月磨砺舍弃 283 种、集大成于至强）——没有大范围压制、没有侵蚀，只有火焰 + 一把武器 + 岁月磨出的**极致技法**（远程「空炎回响」斩、37 米外平平一剑就能斩伤苏晓）——比血神那种天生强大的古神更棘手。
- **猎神（多因术式巅峰武器）**：无固定形态、随意识变形（十字剑 / 手杖…）、温度越高越坚硬锋利沉重——它是**多因「术式」之力的化身**：多因当年从蜥族窃取结晶之力（发现结晶之力 = 向「神灵」祭献所得）、怕神权大过王权（如被神弃毁灭的拜阳王朝），遂**人造神失败、却造出「以器物凝聚信仰之力 + 印记共鸣」的术式体系**，「猎神」正是那汇聚信仰之物——多因的野心是把猎神打造成弑神武器、有朝一日猎杀降临此世的神灵。

## 王宫决战 · 殁（39卷77–79章）

多因王的招牌是**回响斩**——他能与空气中飘飞的燃屑**瞬间位置互换**（0.001 秒），借此达成「远程斩击」（37 米外平平一剑就斩伤苏晓）；[苏晓](../苏晓.md) 靠砸墙 / 踩地掀飞燃屑破解、并全程用青钢影侵墙破掉四壁术式。猎神多形态狂变（十字剑 / 战镰 / 金属伞 / 三爪 / 龙卷风）+ 迷惘系风刃，苏晓靠刃道刀·时 / 极 / 流 + 双刀（青影王 + 魔刃）+ 血之兽血气尖刺**斩下他的头颅**。多因王至死不敢活化体内炼狱之力（会变怪物），头颅燃成灰烬——「御火者，终被火焰焚尽」。掉圣灵宝箱·永生者 100% + 21.73% 世界之源 + **残火灰烬**（苏晓用它 + 多因王血解封阿伽门）+ 猎神（圣灵武器·需信仰虔诚者·苏晓卖掉）。苏晓惋惜技法型的灵魂记忆最宝贵、却随灰烬烧没了。
