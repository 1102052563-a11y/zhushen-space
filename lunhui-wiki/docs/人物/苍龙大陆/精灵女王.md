# 精灵女王（艾尔大森林之主·六族同盟六王·笃信世界之子将至）

满头金发、身穿白纱裙、皮肤白皙、仿佛全身透出温润光芒的女精灵——**精灵族女王**、艾尔大森林（南部）的统治者，[六族同盟](../苍龙大陆群像.md) 六王之一。柔声细语、定力极强，但**语气温柔时最危险**（不高兴谁都可能死·连精灵长老 [鲁思](鲁思.md) 都惧她）。她的王座设在被瀑布与山脉环绕、美轮美奂的**艾兰耶尔城**。

## 立场 · 笃信世界之子（39卷27–28章）

面对 [苏晓](../苏晓.md)（战争恶魔）连破焱谷 / 月光两要塞、直逼艾尔大森林，众精灵长老主张「不能向恶徒屈服」（心里却知道必须服软），精灵女王却**不慌**——她拒绝了苏晓「把月光要塞卖给她」的羞辱性提议（桌下的手已握紧「欺人太甚」），却笃信一句话：

> 「不用惊慌，**世界在庇护我们。被世界眷恋之人已经在路上，杀掉战争恶魔是他身为英雄的使命，世界赋予他的使命**。」

这正是**本世界世界之子（英雄）**的伏笔——世界站在六族同盟一方、正派它的「天命英雄」来终结战争恶魔。她清楚混沌军团战败是必然、却也承认「战争恶魔疯了、是移动的天灾、六族同盟没人想和一个疯子同归于尽」，遂把 [鲁思](鲁思.md) 派去死守 50 万人的古树要塞（多因增援已不可能、只能靠自己）。

## 投影亲临 · 投降 · 虚空之树果实（39卷34–41章）

- **投影试探**：她以能量投影亲临平原战场、用带极强治疗效果的能力试图影响 [苏晓](../苏晓.md)（战争恶魔）——却因他魅力仅 0.1 而**完全无效**，投影被放逐一击穿透。
- **误判理智**：多因与北境即将开战、还来拉拢狼骑兵，她笃定「只要战争恶魔还有一点理智就不会再攻艾尔大森林」——结果苏晓照打不误，她端着的红茶「突然就不香醇了」。
- **投降 · 交全部资源**：50 万狼骑兵攻破艾尔大森林（她投降免平民被屠），交出全部资源（215 大 + 147 完整灵魂结晶 + [守望者之剑鞘 / 枯萎的厄运](../../力量体系/装备道具/index.md) 等）；才知苏晓根本看不上艾尔大森林（要的是南部肥沃战区）。
- **虚空之树果实的关键线索**：苏晓索要「一颗果实」，她取出一块画着果实的石板——**那颗多因王 / 极冰领主 / 蜥族长老 / 传说英雄都争夺过的果实，如今在多因王都！**（正是 [狄牧耶](狄牧耶.md) 等背叛者辗转带去的虚空之树果实）。

## 关系

- [苏晓](../苏晓.md)（战争恶魔）：直逼她大本营、被她视为「移动天灾」、笃信将被世界之子终结的敌人。
- [鲁思](鲁思.md)：她的精灵长老、被她派去守古树要塞。
- **世界之子（英雄·未名）**：她笃信世界派来杀战争恶魔的「被世界眷恋之人」。
- **[六族同盟六王](../苍龙大陆群像.md)**：多因王 / 极冰领主 / 矮人王 / 蛮祭祀 / 异族长老等同盟之首。
