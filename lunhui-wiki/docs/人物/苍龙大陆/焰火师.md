# 焰火师（圣域乐园·点歪的火系法爷·万物皆可爆·战功榜劲敌）

圣域乐园的七阶契约者、苏晓口中的「超级神棍」——他原本是名**火系法爷**，可随着阶位提升、技能却越点越偏：法系能力掌握得稀烂，却**极为擅长制造精神力炸弹**。到如今，焰火师已能**将自己触碰到的任何东西都变成爆炸物**——包括敌人的身体。制造爆炸物的手法越来越高端。

## 经历（39卷5–17章）

进入苍龙大陆打战功排行榜——是 [苏晓](../苏晓.md) 判定「需要警惕」的三名对手之一（另两名是 [灰绅士](../炎灵星/灰绅士.md) 与 [月使徒](月使徒.md)）。战功榜上排在前列（焱谷战役后一度第四·15166 战功）。

## 关系

- [苏晓](../苏晓.md)（白夜）：把他列为需警惕对手的轮回乐园强者。
- [灰绅士](../炎灵星/灰绅士.md) / [月使徒](月使徒.md) / [牧枭](../原始大陆/牧枭.md) / [咕噜](../咕噜.md)：同轮战功榜竞争者。

## 出处

- 三十九卷 5–17 章：战功榜劲敌（圣域·万物皆可爆的爆破流）。
