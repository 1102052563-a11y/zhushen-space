# 大王子（圣索丁王国·王位大热门）

[圣索丁王国](../势力/index.md) 老国王 [巴伊·赫伯特](巴伊·赫伯特.md) 的**长子**、本次王位争夺「**五大种子选手**」之首——**43 岁**（一说 47）、下巴留一撮小胡子、相貌俊朗、待人**温和如沐春风、彬彬有礼**。他是**第一个抵京**的子嗣，且**握有老国王秘密赐予的「免死权」**（一次不杀的特权），是这场权力争夺中的**大热门**（连布布汪都「赌五毛」断定他会是新王）。

## 实力画像（罗文·哈维的资料）

- **财力 ★★★★★**（满）：所治理的领土极为繁荣、近乎成为**王国的经济命脉**——这是他最强的底牌。
- **人脉 ★★★★ / 领导 ★★★ / 综合 ★★★★**：沉稳老练、从不动怒。
- **拉拢策略**：他认为要赢只需拿下老国王的三名心腹——**财务官 [罗文·哈维](../势力/index.md)、执政官 乔·瓦伦丁、军团长 鬣狗·赫士列特**（父王的左膀右臂）；至于**「铁之手」（[苏晓](苏晓.md)）他视作「裁定方」、不在争取之列**（「他的支持太梦幻」）——殊不知苏晓的主线正是要选一人扶上王位。

## 旧识 · 首席法官埃尔维

被肃清的首席法官**埃尔维·安东尼**（详见 [希亚大陆](../世界/任务世界/希亚大陆-灭法遗迹.md)）临死前**赌「大王子继位」**，并忆起大王子曾对他说「逃吧，埃尔维，如果我胜了，你就来做我的臣子」——可见大王子早在暗中经营、收揽人心。

## 为王国退出 · 接手财政（十七卷36章）

得知罗文·哈维叛国后，他**主动退出王位角逐、单膝跪请接手财政权**——脸上那如沐春风的笑容消失、只剩不甘。他深知罗文留下的财政「定时炸弹」一旦引爆、最先垮的是前线（神之墙若破便是全人类的存亡）；想接财政就必须退出争位（否则便是「趁国难上位」），他**看得比王位更远**。老国王早料到他会如此（正因这份品格才赐他免死权）。结果王位被秘密上位的 [三王子](马蒂尔·赫伯特.md) 摘得。

## 关系
- **[巴伊·赫伯特](巴伊·赫伯特.md)**：父王（赐其「免死权」）。
- **三王子 / 七王子 / 二公主 / 六公主**：王位争夺的竞争对手（其余四名种子选手）。
- **[苏晓](苏晓.md)（铁之手）**：他视作不可争取的「裁定方」（实为决定他能否继位的关键·苏晓主线候选之一）。
- **罗文·哈维 / 乔·瓦伦丁 / 鬣狗·赫士列特**：他要拉拢的老国王三心腹。
