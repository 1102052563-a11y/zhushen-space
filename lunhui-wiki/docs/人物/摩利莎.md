# 摩利莎（巫女王 · 西港五阶元素女巫）

[暗鸦星](../世界/任务世界/暗鸦星-暗鸦纪元.md) **西港（西湾）的统治者**——五阶段元素系女巫「**巫女王**」（悬赏 7000 金币·死活不论·苏晓的悬赏令里赏金最高者）。她把西港经营成女巫在 [圣愈教会](../势力/index.md) 边境的钉子：**圈养人类**（买奴隶世代腐蚀其精神、让他们甘为女巫劳作、奉摩利莎为活着的神）——圣愈教会领地内 80% 流窜女巫都出自西港。西港三面环山（峭壁数万陷阱）一面环海（12 座女巫瞭望塔），守备森严如铁桶。

- **与美洛狄的恩怨**：[美洛狄](美洛狄.md) 是西港人、逃离时**偷走了摩利莎传承九代的传家宝**（后又弄丢、疑在南部）——故美洛狄极怕在西港撞见摩利莎。
- **与「死亡之地」线索**：摩利莎名字里的「摩」一度让苏晓怀疑她就是 [伊萨](伊萨.md) 临死那句「只有摩……」所指——但 [美洛狄](美洛狄.md) 险因泄露而被衔尾蛇咒杀，证实**死亡之地与摩利莎无关**（只是名字相似）。

## 光之女巫 · 西港陷落（二十一卷33–43章）

苏晓烧城逼出女巫后，摩利莎**见死不救**、用 99% 部下女巫的性命 + 元素聚集激活家传的「**日阳转化水晶**」，借太阳之力永久晋升成「**光之女巫**」（人类 / ???·法力 9600·力 / 敏 / 体 99 真实·智 100 本世界极值·光属性免疫 + 元素豁免 60% + 物理豁免 40%·主战手段「**耀环**」30 道无差别切割〔连命中 4 道则第 5 道伤害 ×3·专坑主坦〕+ 大招「**炽阳**」5 颗光之耀无差别灭杀）。但她原本是近战女巫、刚得新形态便迷之自信跟苏晓近战——被苏晓（力/敏/体同 99）一套踹腹斩喉 + 布布汪偷袭打懵。

- **败因 = 无冕领主夺意识**：日阳转化水晶里封着 **[无冕领主](无冕领主.md)** 的意识——它鄙夷摩利莎「滥用力量 / 近战弱鸡、不配得此力」，在战斗中夺走她的身体（她变身成男性化的无冕领主）。无冕领主（剑术宗师）远强于摩利莎，苏晓被迫开**猎魔时刻**险胜——无冕领主因灵魂与摩利莎身体严重排斥（全身陶瓷裂痕）自败、身体变回摩利莎。重伤的摩利莎想谈判，被苏晓一刀斩喉清杀。
- **西港陷落**：苏晓斩门触发禁制、**破坏日阳转化水晶 = 覆灭西港女巫聚集点**，得开拓之源 +20%（累计 26.3%）+ 圣愈声望 12700 + 灵魂结晶大 ×10 + 传说级宝箱·猎。

## 关系
- **[美洛狄](美洛狄.md)**：西港出身、偷其传家宝后逃离的「叛徒」（被她抓为人质·靠卡伊保命）。
- **[无冕领主](无冕领主.md)**：封在日阳转化水晶里、夺走她身体的剑术宗师意识。
- **[卡伊](卡伊.md)**：西港二号人物（被她见死不救寒心、还她断臂而去）。
- **[苏晓](苏晓.md)（库库林·白夜）**：攻陷西港、将她（与无冕领主）斩杀的巫猎人。
