# 凛冬（天启乐园·冰系刺客）

[天启乐园](../世界/任务世界/维斯星球-多亚古大陆.md) 一方、[希女王](希女王.md) 的心腹**冰系刺客**——年轻独眼男人，身高仅 1 米 6，却被形容有「**6 米 1 的霸气**」。手中可凝寒气、催生冰锥，攻击附冰晶冻结（一度把 [炎辰](炎辰.md) 的火系魔枪裹上冰晶、打退几步）。和「炎辰」一样**用代号、不改名**——「听代号就知道，我们注定是对手」。

- **希的安排**：契约者大乱斗中由希专门调来**克制火系的炎辰**（情报上炎辰是「火系法坦 / 法师」、冰系正好相克）。开打时确实压制了一段（炎辰火焰被寒气克）。
- **殁（十二卷68章）**：他没料到炎辰真身是「**炎魔枪骑士**」——炎辰脱袍亮出魔法回路、把法杖拧成火系魔枪、召焰虎「皮亚克」、智力转力量化身近战，**5 分钟便把凛冬轰成下半身消失、上身满是高温灼烧的斩痕**（伤口被烫得不流血），最后焰虎踩住、炎辰一枪贯穿其喉。

## 关系
- **[希女王](希女王.md)**：主帅 / 调度者（被派去克制炎辰）。
- **[炎辰](炎辰.md)**：注定的对手——以为克他、反被其真身（炎魔枪骑士）5 分钟秒杀。

## 出处
- 十二卷67–68章。
