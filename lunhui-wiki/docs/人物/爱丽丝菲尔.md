# 爱丽丝菲尔

[Fate/Zero 世界](../世界/任务世界/fate-zero.md) 中 [卫宫切嗣](卫宫切嗣.md) 的妻子——白发美妇人。她**并非人类，而是「爱因兹贝伦家族」（三大魔术家族之一）制造的人造人**，且身负更悲剧的使命：她是家族为**圣杯降临所准备的「容器」**。

- **与圣杯绑定**：圣杯战争中**每有一名英灵战死，她就越发虚弱**——她越虚弱、就代表圣杯降临越近。从诞生起便注定悲剧。
- 随 [卫宫切嗣](卫宫切嗣.md) 与 [Saber](Saber.md) 行动，并曾代理御主（被 Assassin [百貌](哈桑·萨巴赫·百貌.md) 列为暗杀目标、靠 Saber 救下）。

## 目睹丈夫惨死（十一卷62–63章）
苏晓潜入爱因兹贝伦城堡外狙杀 [卫宫切嗣](卫宫切嗣.md) 时，她被 [Saber](Saber.md) 抱着冲去救人，却只来得及**亲眼看着丈夫被巨弹打成碎肉**（温热碎肉溅了她一脸）、情绪彻底崩溃尖叫痛哭。她哭喊着要 Saber「别管我、去给切嗣报仇」，Saber 却为护她而放弃追击——她跪地痛哭，眼睁睁看苏晓全身而退。

## 关系
- **[卫宫切嗣](卫宫切嗣.md)**：丈夫（被苏晓当面狙杀）。
- **[Saber](Saber.md)**：同阵营（御主死后仍誓死护她）。
