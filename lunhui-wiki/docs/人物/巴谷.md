# 巴谷（现实·毒枭）

[苏晓](苏晓.md)「勇闯魔窟」任务（突破身体极限）所在的**三国边境「小城」**（地图上不存在的无法之地）的新霸主——一名毒枭。前一阵突然现身小城、**横扫其他势力、迅速统治这里**（统治时间不长、还没来得及维持秩序·故城里见财起意的亡命徒横行）。

- **「怪物」手下的真相（十二卷100章）**：手下**强得不像普通人**——杀人不眨眼、肌肉因**注射药物**而变形隆起、**体力超 5 点**（中五枪不死）。真相是：他威逼一名进了轮回乐园的**化学系高材生契约者**（用其父亲的断肢相胁），逼高材生硬扛乐园警告、在现实造出「**肌肉爆发生长**」药剂（任何人可用·体力 +7~10·副作用极重·透支生命力、用者活不过 3 个月）。那名高材生（伪装成「佣兵守卫」）便是地下室里用试管炸弹炸苏晓的人。
- **废品厂制毒窝**：他的老巢是小城边缘的**废品厂**——表面破败、实为**大规模制毒**窝点（酸味 + 淡黄烟囱烟雾·守卫几十人 + 热感应警戒 + 地下制毒车间〔防化服工人调配白色粉末〕）。[苏晓](苏晓.md) 的任务指定点恰在废品厂地下室、须撑满 30 分钟——遂与巴谷的守卫展开血战。

- **殁（十二卷98章）**：苏晓在地下室被高材生的试管炸弹炸到濒死、却因此**触发「突破身体极限」、封印枷锁尽碎、属性恢复**——他一跺脚震塌整座废品厂，巴谷被建筑残骸活活压死（临死只来得及说一句「怪物……」）。苏晓事后查了巴谷书房的资料、弄清守卫超人之谜，又顺手清掉巴谷别墅里想趁机上位的管家（防患于未然·小城一个错误抉择就会引来连串麻烦）。

## 关系
