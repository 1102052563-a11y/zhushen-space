# 爱丽丝（恶魔古堡女王）

[恶魔古堡游戏](../世界/任务世界/恶魔古堡游戏.md) 的**女王、古堡内一切的主宰**——在这座封禁古堡里**力量近乎神灵、不可对抗**。她和她的恶魔古堡已存在无数岁月、接待过众多游戏参与者，这个过程重复了太久，可她**孜孜不倦地欣赏参与者的喜怒哀乐**，并把自己制造的宝物散落在古堡各处当作「答谢」。

- **核心规则**：玩 [爱丽丝的邀请函](../力量体系/装备道具.md) 召来的「恶魔游戏」、通关者可得「**强者之证明**」。但**不可讨好她**——只有「**公平的游戏**」才能让她露出笑颜；**一定要让爱丽丝微笑，若她愤怒或伤心，后果不堪**。同时她也是「最有可能杀掉你的人」。
- **她的造物 / 奴隶**：「**狼管家**」（公平的裁决者·中立）、「**古堡恶魔奴仆**」（古堡原主人·被她夺走三分之一大脑而失语·成奴隶苦力·中立）。

## 幻象现身 · 「善意下的恶毒」（十四卷15–19章）

第二场「美食天国」她以**幻象 / 分身**现身（白裙、金发波浪、皮肤散淡金光）——**自带魅惑光环**（在场所有人都不受控想亲近她·[苏晓](苏晓.md) 想掳走她、金丝眼镜男跪迎其为女王·**苏晓的 [心灵之刃](../力量体系/技能.md) 都免不了她的魅惑**）。她表面温和善良、自称「离了古堡只是个弱小女人」，背地里却脸色一冷骂金丝眼镜男「马屁精」。

她最骇人的是**能在抱着善意的同时做出毛骨悚然的恶毒之事**（苏晓评：像「捅你一刀、好心帮你止血、说几句安慰话、止血后再捅一刀、还亲切说『这是为了你好』」）——亲手给每人斟的红酒是「血腥玛瑙」（20 分钟无法治疗 + 受伤翻倍）、端上的牛排是剧毒的毒厄之龙肉，斟酒时却是发自内心的笑。她已**无法用善恶评价**。她说自己造古堡不为取乐、而是「强者诞生之地」、今后或还需勇者帮助——真假难辨。

## 分身 · 全能战力（十四卷23–24章·使徒之眼实测）

第三场「欺诈」她以**分身**伪装成游戏者潜入（黑雾 + 高跟鞋·**无气息却是血肉之躯**）。分身仅「本体 3% 实力」、却已是**全能**：`力 / 敏 / 体 / 智 / 魅 全 80 ｜ 伪神力 960`，技能含**近战 / 元素 / 魔法 / 召唤四专精 Lv.40**——

- **古堡神灵意志 Lv.40**（主动·仅古堡内）：耗伪神力具现淡金~金色的武器 / 防具（同时具现超 5 件则降为淡金）——她能瞬间切换水晶盾（近战防御）/ 黑木弓（远程·冰 / 风穿透 / 火 / 暗元素黑洞箭）/ 法杖（召唤「暗影兽」·她见过的动物都能召·力敏体恒 60）/ 树木法杖（藤蔓控制），堪称四位一体的全能战士。
- **噩梦之瞳 Lv.40**（古堡内无视一切控制）+ **恶魔古堡女王 Lv.40**（古堡内生命恢复 +1000%·金色具现物短时破不掉）。

[苏晓](苏晓.md) 自评：他四属性 80 已是极限、却对这种「**全能**」的敌人望尘莫及（基础技能光刀术大师就够他倾家荡产）。但分身**「战斗时机把握辣鸡」**、被苏晓的直感一一躲过、斩六刀报废一件金色铠甲——在古堡内她「不是无法对抗」（印证血书）。

## 个人恩怨 · 虚空台（十四卷26章揭）

她第一黑夜专挑苏晓下手、其实出于**私怨**——她曾**险些被灭法之影吊死在「虚空台」**（与莎独白「你已忘了差点被谁吊死在虚空台」相互印证）。这也解释了她为何畏惧「那个存在」（灭法者一脉）、却仍把灭法者传承的苏晓放进古堡。可惜动不了苏晓（有「那个存在」庇护·她只能用游戏正规手段、不能破坏规则），遂转去「享受游戏」——专挑推理高手侦探小萝莉折磨（扒光当抱枕）、杀掉搅局的猫女、让多人「违规」禁足，一夜把契约者方逼入大劣势。

## 伪装国足老大 · 换号牌诡计（十四卷28–29章）

第三场「欺诈」她伪装的是**国足老大**（5 号）——靠音乐骚扰逼猫女（6 号）与「国足老大」换房号牌（换号是她的主意·真国足老大进 6 号房就被传送走、由她替代），借此把嫌疑搅浑、又能与「国足」抱团避险。她的阴谋是：第二夜杀掉猫女（已是 5 号）后**取回原属自己的 5 号号牌**、让众人误验两次猫女从而彻底洗清「国足老大」——可惜幸运通关者（真国足老大）画的「O＊」线索（破碎眼镜 + 破碎蛋·唯她关联两者）太精准，被苏晓质疑 5 号一举揭穿。游戏结束她以分身现身、与苏晓气息外放对峙后放话「祝你好运」上四层。

## 身世 · 被灭法之影差点吊死（十四卷30章·重大世界观）

她本名「**爱丽丝·安吉莉亚**」——金发女子，**喜用他人骨骼造城堡、融合了「第十七魔网」、破坏三颗星球的元素平衡**（致两颗崩溃、一颗半废弃）。多年前她被一名**黑发女 [灭法之影](../力量体系/技能.md)**（腰挂长刀·脚踝藏锯片）抓住、装进铁笼要吊死在「**虚空台**」（古老王国的处刑台）。那灭法之影的话点破了灭法者的真正职责：「**你杀几十亿人、对灭法之影只算个人爱好；但破坏元素平衡不行——维系虚空秩序的是元素平衡 + 法则之力 + 星之力**」（灭法者 = 虚空秩序的维护者）。爱丽丝求饶、愿切断魔网做事被拒，绝望中誓言「总有一天建立自己的古堡、让你体会这种绝望」（灭法之影提到「某个恶魔老兄已建古堡、在其中相当于伪神」）——这正是她造恶魔古堡、当古堡内「近乎神灵」女王的由来。当时一颗几万米陨石袭来、黑发灭法之影以「**青钢·刹**」破之、拔刀迎敌，爱丽丝这小角色被丢在一旁。

## 关系
- **灭法之影一脉**：她曾差点被其吊死在虚空台的「天敌」（畏惧·却把传承者苏晓放进古堡）。
- **狼管家 / 恶魔奴仆**：她的造物 / 奴隶（裁决者 + 苦力）。
- **[苏晓](苏晓.md)**：为「强者之证明」入局的契约者（须让爱丽丝微笑方能通关）。
