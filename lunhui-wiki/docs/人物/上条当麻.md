# 上条当麻（幻想杀手·本世界主角）

[魔法禁书目录](../世界/任务世界/魔法禁书目录.md) 世界的**主角**——刺猬头黑发少年，学园都市的 **Lv.0 无能力者**，却是个「运气极差、桃花运爆棚」的少年（幼年起住院 138 次·出门能集齐踩狗屎 / 被抢 / 车祸 / 花盆砸 / 丢钱·练出超常危机感）。自称「伪善者」、实为热血善良少年。是不是「位面之子」一类，[苏晓](苏晓.md) 尚待考证。

- **幻想杀手（Imagine Breaker）**：他右手具备一种近乎无敌的能力——**凡是右手接触到的超能力 / 魔法 / 一切「异能之力」都会立即失效并消除**。靠这只右手，他昨晚打败了一名闯入学园都市的魔法师。
- **剧情开端**：特殊女孩 [茵蒂克丝](茵蒂克丝.md)（逃入学园都市）与他「偶遇」结识、成为朋友；昨晚那名魔法师有一名**同伴**（[神裂火织](神裂火织.md)），强到上条当麻应付不了。
- **被压制 + 被苏晓拎走打昏（十三卷12–13章）**：他不肯交出茵蒂克丝、硬冲 [神裂火织](神裂火织.md)，被压倒性碾压（右臂十几道伤、被踩在脚下·神裂手下留情没杀他、还反过来劝他快逃）。[苏晓](苏晓.md) 赶到后**没杀他**——只要懂魔禁世界的契约者都不会动上条当麻（他**死亡前会爆发出秒杀任何契约者〔含苏晓〕的战斗力**·击杀风险远在收益之上），且亚雷斯塔重视他、要求带他到安全处。苏晓把他拎起抛给 [结标淡希](结标淡希.md)、被一记军用手电筒打昏看管（他还天真地求结标淡希「阻止你同伴」、被反呛「那家伙是怪物、你想害死我吗」）。

## 关系
- **[茵蒂克丝](茵蒂克丝.md)**：逃入学园都市、与他结识的特殊女孩（拼死想保护她）。
- **[神裂火织](神裂火织.md)**：追杀茵蒂克丝的魔法侧圣人（上条对付不了的「同伴」）。
- **[苏晓](苏晓.md)**：奉亚雷斯塔之命来处理神裂火织的契约者（与上条的主线交汇）。
