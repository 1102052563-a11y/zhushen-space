# 上条当麻（幻想杀手·本世界主角）

[魔法禁书目录](../世界/任务世界/魔法禁书目录.md) 世界的**主角**——刺猬头黑发少年，学园都市的 **Lv.0 无能力者**，却是个「运气极差、桃花运爆棚」的少年（幼年起住院 138 次·出门能集齐踩狗屎 / 被抢 / 车祸 / 花盆砸 / 丢钱·练出超常危机感）。自称「伪善者」、实为热血善良少年。是不是「位面之子」一类，[苏晓](苏晓.md) 尚待考证。

- **幻想杀手（Imagine Breaker）**：他右手具备一种近乎无敌的能力——**凡是右手接触到的超能力 / 魔法 / 一切「异能之力」都会立即失效并消除**。靠这只右手，他昨晚打败了一名闯入学园都市的魔法师。
- **剧情开端**：特殊女孩 **茵蒂克丝**（被魔法师追杀、逃入学园都市）与他「偶遇」结识、成为朋友；昨晚那名魔法师有一名**同伴**（[神裂火织](神裂火织.md)），强到上条当麻应付不了——今晚两人已遭遇、正对峙 / 战斗。[苏晓](苏晓.md) 受亚雷斯塔之命要去**击退 / 击杀**这名同伴。

## 关系
- **茵蒂克丝**：逃入学园都市、与他结识的特殊女孩。
- **[神裂火织](神裂火织.md)**：追杀茵蒂克丝的魔法侧圣人（上条对付不了的「同伴」）。
- **[苏晓](苏晓.md)**：奉亚雷斯塔之命来处理神裂火织的契约者（与上条的主线交汇）。
