# 干柿鬼鲛

[火影忍者世界](../世界/任务世界/火影忍者.md)「**晓**」组织的成员、[宇智波鼬](宇智波鼬.md) 的搭档——雾忍出身的**叛忍**，背着一米多长、缠满绷带的宽身忍刀「**鲛肌**」（能吞噬查克拉·会颤动示警）。

- **拦人 / 招募（九卷15章）**：与鼬一同在火之国边境拦下 [苏晓](苏晓.md)。晓「新人入伙照例打一场」，鬼鲛刚要出手，苏晓却**干脆答应加入**（他被铁之国 [三船](三船.md) 拒了、又被雾忍暗部追杀，加入晓正合心意）。一路同行，**鬼鲛被近在咫尺的苏晓那股「随时斩下头颅」的锋利感压得极不舒服**，由此明白首领为何招他。
- **忠诚 / 狠辣**：他虽是叛忍，却是**最称职、最忠诚**的忍者——一旦认同某势力便绝不背叛（原著后续被擒时，为不泄露情报、自缚水牢 + 通灵鲨鱼吞噬自己，连敌人都敬佩）。
- **鲛肌示警**：进雨之国时，苏晓本能用青钢影噬灭雨中查克拉，鲛肌随之颤动——鬼鲛替苏晓打了圆场（「鲛肌感应到某种能量而已」）。

## 抓回八尾 + 引荐蔷薇（十五卷50–52章）

他奉带土之命去抓八尾人柱力 [奇拉比](奇拉比.md)——沿用「**藏身忍刀鲛肌内、放白绝分身扮自己**」的稳妥打法，却遇鲛肌「叛变」（鲛肌有自我意识、更爱八尾的查克拉）；幸而苏晓斩杀了四代雷影、令奇拉比失了义兄援军（原著正是雷影驰援秒掉那具假鬼鲛），他便现身横跨铁之国一路追、终生擒奇拉比。途中救下被武士追杀的「黑眸职工者妹子」（与他上次入火影有旧、开过好感度）、由她助阵追上奇拉比，遂把她**引荐进晓**（伪职工者 + 特殊血继限界）——殊不知这正是疑似苏晓猎杀目标的 [蔷薇](黑瞳-契约者.md)。他是个「不背叛所在势力」的忠诚叛忍，回基地后没多问苏晓等人去救迪达拉的事。
