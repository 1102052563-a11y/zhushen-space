# 枭（阿萨卡杀手 · 苏晓的接引人）

[黑渊](../世界/任务世界/黑渊浅层-赔罪者之殇.md) 赎罪者高地 **[阿萨卡](../势力/index.md) 杀手团体的第 17 位**——黑色短袍 + 白色合金面具（带伪声功能）。能在丝线网上以「忽隐忽现」的高等移动技巧行走（非空间能力）、双短刀。表面是有逼格的杀手、实则**异常健谈**（杀手身份让他没机会说话、快憋出病）。主动找上满身血气的 [苏晓](苏晓.md)、引荐他加入阿萨卡——动机是当接引人能从苏晓完成的暗杀任务里抽成。

- **加入仪式约 2 秒**：枭随手帮苏晓填资料（年龄随便填）、丢来黑袍 + 面具 + 终端 + 地下住所地图就送客。阿萨卡**无总部、无内部管理**（连枭都不知总部在哪）——「能听闻、看不见、寻不到，需要时它会找上你」才是真正的杀手团体，故不怕内奸。
- 听苏晓自称来自「罗坦蓝大陆」、又怀疑「我在黑渊战力会比较靠后」，差点被气得吐血（苏晓其实是黑渊浅层高层梯队战力）。

## 关系
- **[苏晓](苏晓.md)（白夜）**：他引荐入阿萨卡的「最强外乡人」（抽成对象）。
- **[阿萨卡](../势力/index.md)**：所属杀手团体（第 17 位）。
