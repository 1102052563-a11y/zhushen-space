# 罪恶（鬼环组织首领 · 枪术宗师混血）

[幽鬼大陆](../世界/任务世界/幽鬼大陆-五阶考核.md) 由幽鬼组成的势力 **[鬼环](../势力/index.md) 的首领**——皮衣皮靴、举杯敬「不朽之王」的强者。综合素质等同**十三眼幽鬼**、加 **枪术宗师 Lv.27** 的技巧，使其拥有凌驾十三眼幽鬼的战斗力（[苏晓](苏晓.md) 不开猎魔时刻很可能落败·全靠克制其能量武器才打到最后）。始终保持人类形态战斗（未幽鬼化）、对疼痛忍受恐怖。是**大 BOSS 级**（需一群晋升期四阶契约者围攻那种）。

- **死刑·暮（能量长枪）**：白绿色能量具现的长枪·**无视防御**（消耗自身能量把一点防御削到极致再刺穿·阿姆的寒冰盾 + 苏晓都挡不住·把阿姆扎成刺猬）·可一次具现十根悬浮飞射 / 折返。破碎后能立刻重新具现。另有「魂」（震荡灵魂的吸力强控）。**致命弱点 = 能量武器**：[青钢影](../力量体系/技能/index.md) 最擅噬灭能量、一刀即碎死刑·暮（这是苏晓能打的根本）。
- **野猪庄园重伤逃走（二十四卷15–18章）**：苏晓阻其当晚布成「十六眼室」、用 **[血之兽](../力量体系/天赋.md)** 把他扑成重伤（满身碎骨）+ 绝影闪补刀，他自爆能量枪逃走。靠枪术宗师 Lv.20 衍生的「不死」保命。逃后重伤、连派人通知情报中转站**乔安娜**（见下）都不敢——鬼环情报网就此被苏晓逐个掐断。

## 乔安娜（情报中转站）

鬼环的情报中转站负责人——伪装成渔港街 37 号占卜屋的女占卜师（前身交际花·仅二眼普通幽鬼·对枪械近乎无视）。鬼环半数以上情报经她中转（防底层暴露被一锅端）。苏晓循名册找上门、三枪爆头击杀（中断鬼环情报网 87 小时以上）·掉秘典·魔女。

## 重伤潜伏 · 与 J 先生合作（二十四卷28–43章）

野猪庄园逃走后重伤潜伏——大本营「不朽城堡」被苏晓阿波罗炸平时，十二眼幽鬼 [月蚀](幽鬼大陆配角.md)（眼室计划提出者·暗恋他）牺牲自己保住眼室「核」、用青蓝粘液把他裹住从地洞送走逃命。此后他藏身**修·洛斯南勒的陵墓**（君主 [哈罗德](哈罗德.md) 祖先之墓·等于刨了哈罗德祖坟）养伤、与神秘契约者 **J 先生**（见 [挽歌页](挽歌.md)）临时合作。养着次级霸主生物「宠物」[扭曲](../世界/生物万族/index.md) 守眼室。国足三兄弟来盗修·洛斯南勒遗骨时与他撞上、他怕暴露交手没多久就撤。

## 真目的揭晓 · 救赎与复仇（二十四卷50–57章）

眼室 / 不朽之王 / 助南茜当女王全是**幌子**——罪恶从一开始就知道眼室在祖尔城召不出不朽之王（哈罗德太可怕）。一连串操作（布眼室 / 三道黑光柱）只为**吸引处刑机关**，他借 J 先生窃取格斯记忆得知地下室路径、**用小王扑克牌伪装成苏晓潜入处刑机关地下室**，真目的只是救出**童年如父**的濒死十六眼幽鬼 [法迪尔](幽鬼大陆配角.md)（罪恶本名「小波伊尔尼」·法迪尔幼时待他如子）。这就是他的「**救赎与复仇**」——救法迪尔是救赎、找哈罗德报仇是复仇（但被苏晓 + 哈罗德锤到几乎放弃复仇）。苏晓 + 哈罗德全程以为他有宏大阴谋、反被这「加农炮打蚊子」的小目标秀到。事后法迪尔给他一颗**三金瞳黑眼球**（与苏晓在眼室得的「未知眼球」同款·嘱他找另一颗）。

## 回头寻仇 · 殒命（二十四卷63–64章）

救出法迪尔、又惦记着找哈罗德复仇，他在苏晓夺时空之核途中**杀回来**——单挑苏晓本占上风（死刑·暮五枪贯穿苏晓胸膛），但苏晓不与他硬拼到底：先血之兽换断他一臂、再假逃把他引到 [阿姆](阿姆.md) 埋伏处。阿姆**凛冬之怒**冰封 + 寒冰盾拖坦，苏晓嗑远古秘药回血打车轮战、反复换人毒打，终于把他耗到濒死。临了他坦言来寻仇是「为了心中得到安宁」，苏晓收枪与他堂堂正正一刀对冲——绝影闪斩首（本名波伊尔尼·掉**双宝箱**〔史诗 37% + 秘典卡·杀戮与救赎〕·世界之源 15.9%）。苏晓念其是对手、本想水葬。
