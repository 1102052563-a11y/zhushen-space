# 晨曦女神（太阳王国信仰之神）

[海蓝星](../世界/任务世界/海蓝星-世界争夺战.md) 上**太阳王国**所信奉的神明——原本太阳王国信太阳神**阿瑟坦勒**，后因太阳神不再赐神力而被王室与神庭弃信、改信晨曦女神（首都遂改名晨光城）。这类「神」是某种**高维意识**的载体（非肉身存在）。

## 「降临虚影」(十九卷34章)

太阳王国神庭的军官可**割喉献血**召其降临——四名军官以血为媒，半空凝出一道**四米高的白色女性虚影**（相貌模糊、唯双手最清晰），本质是**一大股生命能量**汇聚（几乎无法被物理攻击伤到、除非攻击强度超过某个极限）。降临时罩出「晨曦之神注视领域」：

- 领域内敌人**力 / 敏 / 体 / 魅各 −20**（[苏晓](苏晓.md) 凭真实属性豁免力敏智、仅魅 −20）、**重力压制移速 −50~60%**、**每秒损 6~12% 生命**（青钢影豁免一半）。寻常四阶契约者靠近不出 5 秒即丧命。
- 虚影由四名军官的光链牵着、移动范围有限、约持续 3~5 分钟；弱点是**阴影**。[小助手](小助手.md) 的好友们正是死在它手下（她因此患过抑郁）。

## 被苏晓的刀魔咬掉头颅（十九卷34–35章）

苏晓识破它「只是一大股生命能量」、硬冲到虚影身前、用青钢影按住其小腹发动 [吞噬之核](../力量体系/技能/index.md)——**伴生的「刀魔虚影」**（血红眼、嘴比头还大、满口尖牙）破天荒地主动现身、**一口咬掉并吞下晨曦女神虚影的头颅**，「神」当场惨叫殒灭（这次降临的虚影被毁）。刀魔吞下这股生命能量后能量基数大涨（从原需 Lv.20 青钢影压制升到需 Lv.35）。

## 关系
- **太阳王国 / 神庭**：信奉它的国家（军官割喉献血召其降临）。
- **[苏晓](苏晓.md)（灭法之影）**：其降临虚影被苏晓的刀魔咬掉头颅吞噬。
