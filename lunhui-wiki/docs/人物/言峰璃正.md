# 言峰璃正

[Fate/Zero 世界](../世界/任务世界/fate-zero.md) 冬木市「言峰教会」的**老神父**——满头白发、中气十足、对外形象正直谦逊。真实身份是隶属 **[圣堂教会](../势力/index.md)「第八遗迹部」**、受委派**监管第四次圣杯战争**的监督者。他不是魔术师（精通八极拳、体格健硕，但战力不算强），其子 [言峰绮礼](言峰绮礼.md) 是本届 Assassin 的御主——父子俩既是监督者、又是参战者，且已与 [远坂时臣](远坂时臣.md) 结盟。

## 被苏晓套话 / 胁迫（十一卷9章）

苏晓为找被圣堂教会藏起的**柳洞寺**（[天赋觉醒任务](../世界/任务世界/fate-zero.md)第一环），戴高科技面具、抹去手背令咒（伪装普通人）到言峰教会、以「**忏悔杀人**」为名进告解室单独套话。老神父先当他是寻刺激的妄想症青年、后越听越胆寒（直觉「隔壁的人能瞬间秒杀我」），想掏手机联络儿子绮礼报信——被苏晓用 [界断线](../力量体系/装备道具/index.md) 缠住脖颈、切碎手机，逼问「如何进入现在的柳洞寺」。他出于善念本不愿苏晓去那「地狱」，最终在威胁下决意开口。

**转中立 · 「投资」苏晓（十一卷10–11章）**：他是柳洞寺**八名「钥匙」持有者之一**——苏晓一接触他，乐园便判定他**转为中立、好感固定「友善」**（此后双方不可再互相攻击 / 威胁 / 欺骗）。柳洞寺现被一群「疯子」（信仰诡异）以城市安危为筹码占据，他本想派儿子绮礼去清除那里某些东西、却舍不得（老来得子）；于是「好不容易来个不怕死的」苏晓让他喜出望外，便豪气地**「投资」**：送 [柳洞寺钥匙「鬼山羊皮肤」](../力量体系/消耗品材料/index.md) + [圣水](../力量体系/消耗品材料/index.md) + 女代行者遗物 [黎明战吼](../力量体系/装备道具/index.md)，并领苏晓走密道从私人密室取物，临走还好言相劝「活着回来、小心你认为可靠的人」。条件 = 苏晓活着出柳洞寺、打开其驱逐灵体结界。「不算好人、但也不坏」。

## 修斯之死 · 功勋兑换 · 契约（十一卷36–37章）
苏晓炸平柳洞寺、斩杀领主级 [怨之孤儿·修斯](怨之孤儿·修斯.md) 后回到言峰教会——告知老神父「修斯已死」时，老神父捂脸哽咽落泪：**修斯曾是他的恋人**（圣堂教会的女代行者·本应教会内部处置、却被魔术协会以「怨生物有研究价值」插手才酿成此祸）。他触发 **「圣堂教会功勋兑换」**机制，让苏晓用炸柳洞寺攒的 2219 功勋兑得灵魂结晶〔大〕+ 圣水中瓶 + [幽魂猎手](../力量体系/装备道具/index.md)（还自嘲着打了 95 折）。他**痴迷于夺取大圣杯**（监督过第三次圣杯战、亲见过圣杯·明知第四次圣杯已被恶念污浊仍不甘放弃）、想拉苏晓入伙助他夺杯，被苏晓以「夺圣杯者都得死」劝拒。最终他应苏晓之托**牵线爱因兹贝伦家 / [卫宫切嗣](卫宫切嗣.md)**（自己是监督者、说话有分量），并逼苏晓签下「自我强制证文」（誓约·不联合爱因兹贝伦对付他）——却不知此誓约只对「体内有魔力的本世界土著」有效、对苏晓形同废纸。

## 死于亲子之手 · 临终托付（十一卷57章）
苏晓接到他求救来电赶到言峰教会——只见教堂被刻意破坏掩饰、老神父**被腰斩成两截**、拖着半截身体爬出很远。他靠吞下 [死徒骨粉](../力量体系/消耗品材料/index.md)（续命道具·身体坏死后意识停留 30 分钟）撑到苏晓来。他只说凶手是「一个『魔物』罢了」（哀莫大于心死、不愿点破是**亲生儿子 [言峰绮礼](言峰绮礼.md)** 所杀·手臂上历届圣杯战争积存的 41 枚令咒也被绮礼夺走）。他认出苏晓是 Berserker（「魔术师怎么能杀掉修斯、炸平柳洞寺」），本想留几枚令咒给苏晓「御主」助阵、却已被夺空。临终他看透圣杯战争的荒谬（「魔术师把英雄召唤出来当工具厮杀、是用人性侮辱已故英灵」），**托付苏晓「别让那些贪婪的魔术师得逞」**、并认同苏晓先前「不该追求圣杯」之言。粉尘化前留下半句「**最终、我居然开始不了解你、我的……**」（未尽之言是「亲生子嗣」）。他的死**触发了苏晓的隐藏任务「[英灵的荣耀](../世界/任务世界/fate-zero.md)」**。苏晓放火焚教堂送别（教会阵营商店须他亲启、其宝库带不走）。
