# 大蛇丸

[火影忍者世界](../世界/任务世界/火影忍者.md) 木叶「**三忍**」之一——脸色惨白、披黑色长发，气质已极近蛇。曾是「晓」成员、后叛出；为追求**永生**不择手段（拿数千上万人做实验），喜把实验室建在阴冷地下；被 [宇智波鼬](宇智波鼬.md) 秒杀过后，对**宇智波血统**有近乎疯狂的执着。

- **被封印的废人期**：双手的灵魂被三代火影封印，**无法使用大部分需结印的忍术**，只能靠软体术 + 通灵术（潜影蛇手）战斗——而软体术要近身，近身正是 [苏晓](苏晓.md) 的杀场。苏晓判断：单挑此时的大蛇丸，纵不能杀死他、也能把他斩碎十几次（其生存力极变态）。部下 [药师兜](药师兜.md) 的药物维系着他被封的手臂。
- **抓住 10001 号违规者（九卷48章）**：苏晓苦寻的猎杀目标 **10001 号违规者**（盯上大蛇丸实验室、只等他被「二柱子」杀掉好趁虚而入）反被大蛇丸提前五天抓住（为此死了几十名部下），刨胸截腿当了试验品。
- **交易与翻脸（九卷48–50章）**：他垂涎苏晓夺得的 [卡卡西](卡卡西.md) 写轮眼与特级炼金秘药，以违规者为筹码与苏晓周旋；苏晓挟持兜、以**尾兽之血**（科学家的终极诱惑）换得违规者并斩首。随后大蛇丸**翻脸设局**（关铁门 + 放试管人造兵 + 部下 [水月](鬼灯水月.md) / [香磷](香磷.md)）想用药剂 + 写轮眼换苏晓的命——岂料整件事是 **[宇智波鼬](宇智波鼬.md) 的钓鱼局**（写轮眼当饵）：在场「大蛇丸」是替身（女忍假扮），真大蛇丸早被鼬识破、与水月化水**逃脱**（鼬为帮**宇智波佐助**〔「二柱子」·近期将灭大蛇丸〕成长而放水）。

## 死讯（九卷58–59章）
逃脱后不久，[小南](小南.md) 向晓众人通报「大蛇丸死了」，[黑绝](绝.md) 点破凶手是 **[宇智波佐助](宇智波佐助.md)**（鼬之弟·正是被苏晓帮带土「培养成仇恨棋子」的目标）——这也印证了「二柱子近期灭大蛇丸、其实验室空门大开」的剧情走向。
