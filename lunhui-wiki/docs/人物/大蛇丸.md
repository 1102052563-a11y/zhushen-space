# 大蛇丸

[火影忍者世界](../世界/任务世界/火影忍者.md) 木叶「**三忍**」之一——脸色惨白、披黑色长发，气质已极近蛇。曾是「晓」成员、后叛出；为追求**永生**不择手段（拿数千上万人做实验），喜把实验室建在阴冷地下；被 [宇智波鼬](宇智波鼬.md) 秒杀过后，对**宇智波血统**有近乎疯狂的执着。

- **被封印的废人期**：双手的灵魂被三代火影封印，**无法使用大部分需结印的忍术**，只能靠软体术 + 通灵术（潜影蛇手）战斗——而软体术要近身，近身正是 [苏晓](苏晓.md) 的杀场。苏晓判断：单挑此时的大蛇丸，纵不能杀死他、也能把他斩碎十几次（其生存力极变态）。部下 [药师兜](大蛇兜.md) 的药物维系着他被封的手臂。
- **抓住 10001 号违规者（九卷48章）**：苏晓苦寻的猎杀目标 **10001 号违规者**（盯上大蛇丸实验室、只等他被「二柱子」杀掉好趁虚而入）反被大蛇丸提前五天抓住（为此死了几十名部下），刨胸截腿当了试验品。
- **交易与翻脸（九卷48–50章）**：他垂涎苏晓夺得的 [卡卡西](卡卡西.md) 写轮眼与特级炼金秘药，以违规者为筹码与苏晓周旋；苏晓挟持兜、以**尾兽之血**（科学家的终极诱惑）换得违规者并斩首。随后大蛇丸**翻脸设局**（关铁门 + 放试管人造兵 + 部下 [水月](鬼灯水月.md) / [香磷](香磷.md)）想用药剂 + 写轮眼换苏晓的命——岂料整件事是 **[宇智波鼬](宇智波鼬.md) 的钓鱼局**（写轮眼当饵）：在场「大蛇丸」是替身（女忍假扮），真大蛇丸早被鼬识破、与水月化水**逃脱**（鼬为帮**宇智波佐助**〔「二柱子」·近期将灭大蛇丸〕成长而放水）。

## 「死」而不死 · 与苏晓交易（九卷58–64章）
- **公开死讯**：[小南](小南.md) 通报「大蛇丸死了」，[黑绝](绝.md) 点破凶手是 [宇智波佐助](宇智波佐助.md)（其实大蛇丸藏身佐助的咒印里没真死）。
- **被鼬「封印」、分裂逃生**：鼬决战中从佐助咒印逼出他、以须佐能乎十拳剑封印——但他**分裂出一条小白蛇逃脱**（咒印持有者不死光、他就能复活）。
- **沦为苏晓的谈判筹码**：那条藏其灵魂的小白蛇被 [苏晓](苏晓.md) 收走。大蛇丸认栽（明白整盘是鼬的局——鼬故意让他把佐助当「新容器」全心培养），与苏晓做交易：**给苏晓「去除原始柱间细胞木遁成分」的方法、提供一处实验室**，换苏晓**对外宣称他已死**（晓已抓 7 尾·他更愿「死」掉好做最后的赢家）。说完地址后白蛇自爆——其得意弟子 [药师兜](大蛇兜.md) 已继承他大量能力、成为他最好的新容器。

## 苏晓的「背锅侠」（十五卷·火影再入）

正因大蛇丸什么都研究、堪称「天生的背锅侠」，[苏晓](苏晓.md) 每当要向剧情人物透露原著秘密（直接透露会被世界反噬）就**把信息来源全推给「大蛇丸的研究资料 / 实验室」**——迄今甩锅多次。忽悠 [宇智波斑](宇智波斑.md) 时那些大筒木一族、神树、原始柱间细胞的「资料」，也都借大蛇丸之名抛出（苏晓笑称大蛇丸若知道，定会全速赶来弄死他、并向斑澄清「与我无关」）。

## 复活 · 佐助的「人生导师」 + 秽土转生四代火影（十五卷92章）

算准时间，假死的大蛇丸**已经复活**——还成了 [佐助](宇智波佐助.md) 的「人生导师」，帮佐助**秽土转生出四代火影**（初代 [千手柱间](千手柱间.md) / 二代千手扉间 / 三代猿飞日斩 / 四代波风水门）助阵忍者联军、对抗已成十尾人柱力的 [带土](宇智波带土.md)。四名火影皆不死 + 无限查克拉，一场强者大乱斗就此展开。
