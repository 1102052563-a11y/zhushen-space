# 土御门元春（多重间谍）

[魔法禁书目录](../世界/任务世界/魔法禁书目录.md) 学园都市里的 **Lv.0 无能力者**，却是同时效力科学侧与魔法侧的**双重间谍**——发展成两边的情报中转站，谁都不动他、也都不太信任他（比单方间谍聪明得多）。金发刺猬头、花衬衫露胸肌、戴太阳镜（不让人看穿眼神·间谍的本能）。战斗力不强，但**心思缜密、处事冷静**；可为朋友奋不顾身、为守护没有血缘的妹妹能与全世界为敌。他是 [苏晓](苏晓.md) 试炼任务「末灵法典」的**线索人**（亚雷斯塔的重要棋子·不许动）。

## 提供末灵法典线索（十三卷26章）

[苏晓](苏晓.md) 为见他、不惜替亚雷斯塔去杀魔法侧圣人。他起初推说「末灵法典或许已不在世上」想搪塞，被苏晓「敢动你身边人就一起下地狱」的威胁试探（苏晓其实无意用其亲人要挟·那样情报不可靠），又被苏晓「拿不到法典我必死、为它什么都肯做」的决绝逼到墙角，终于松口——只求**提供情报后两清、再无瓜葛**（间谍的生存之道）：

- **末灵法典真相**：又称「邪灵魔法书」，记载一种**凌驾于元素之上的「崩坏」之力**——被它破坏的东西**不可逆、等于被清除「存在」**（400 年前一魔法师首用即自爆毁一城·300 年前另一魔法师自爆把湖变成「气态水」〔接触即分解·飘散伤人·十字教十几年才平息〕·那湖至今是填不平的天坑〔填充物过午夜即消失〕）。每有人掌握它的技术、它就神秘失踪很久。
- **线索 = 木原一族**：他不知确切位置，但传闻**有人发现了末灵法典、准备拿它做些什么**——那人是「**木原**」（不是 [苏晓](苏晓.md) 见过的木原数多、是「更老一辈」的·指**木原一族**）。说完便走、心中默念「木原一族，别怪我，这是你们自己的选择」。
