# 米拉（血怨 · 四阶元素血之女巫）

[暗鸦星](../世界/任务世界/暗鸦星-暗鸦纪元.md) 的**四阶段元素系黑女巫**——有专属称号「**血怨**」（女巫里的顶尖梯队·悬赏 580 金币·曾毁灭 4 座人类城市）。她操控**鲜血与生命力**：化身「怨恨」（与红甲血骑士化身**融合**·人形怪兽·力 94 / 敏 95 / 体 99 真实 / 智 99 真实·近战肉搏专精·每秒回 15 生命·鲜血化躲致命攻击·能磁力般物理操控敌人血液致其血液倒流）。她**逼迫三名三阶女巫用改造黑鱼虫荼毒尼克多多城**收集生命力，是为了**晋升五阶段女巫**做准备。

## 火车决战 · 晋五阶失败殒命（二十一卷25–27章）

苏晓循「生命力气雾」追到她乘的火车上。她虽强（操血液 + 吸生命力回血 + 肉球再生 + 化身融合），却有死穴——**怕火**（被子弹打碎颅骨不吭声、被火烧却惨叫）。苏晓以霰弹喷子 + D·暗杀压制（无声领域 + 震爆弹剥夺其视听）、寂灭公爵火元素弹 + 炼金炸弹烧她的血肉 / 肉球，逼她耗光 60% 生命力再生；她想晋升五阶（白发返老还童的少女态）却**晋升失败**、血肉爆开。最终被苏晓抛刀斩碎 + D·暗杀爆头 + 炼金炸弹烧尽那摊鲜血清杀。苏晓得**开拓之源 3.5%（累计 6%）+ 770 圣愈教会声望 + 高品质传说级宝箱 + 咒术抗性 +15%（达 65%）**。

## 关系
- **[苏晓](苏晓.md)（库库林·白夜）**：火车上将她清杀的巫猎人。
- **三名三阶元素女巫**（贵妇 / 皮衣 / 金发）：被她驱使荼毒尼克多多城的下属（均被苏晓斩杀）。
