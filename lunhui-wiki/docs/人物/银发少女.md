# 琉璃（银发少女·希之妹）

[维斯星球·多亚古大陆](../世界/任务世界/维斯星球-多亚古大陆.md) 世界争夺战中、[苏晓](苏晓.md) **首个正面交手的天启乐园契约者**——真名**琉璃**，银发、琥珀色眼瞳、身材娇弱的少女，**[希女王](希女王.md)（姐姐）的妹妹**（自承「我们很像、有血缘关系」）。性子轻佻爱装、嗅着挺香、贫嘴皮实（人称「皮皮虾附体」），却一身玄之又玄的「**狂想者**」系能力。

## 能力 · 「狂想者」幻象 / 时间一脉

- **幻象欺骗（非精神控制）**：能伪造对手的**视觉与感官**——登场即用「音爆冲锋 + 一拳打碎空间 + 拳穿心脏」的假象唬人（实则用暗紫手枪偷袭后心、再补匕首），但这**不是精神类幻术**（故苏晓的 [心灵之刃](../力量体系/技能/index.md) 免不了它·只能靠 [界断线](../力量体系/装备道具/index.md) 结界 + 触感识破）。
- **幻身**：分出多道幻影分身（子弹 / 刀芒穿身而过·真身另在）。
- **半透明穿梭**：身体半透明、**穿梭于幻象与现实之间**（攻击穿体而过·类似 [带土](宇智波带土.md) 的空间能力·但本质是幻 / 实切换）。
- **狂想者·迷森**：抱住对手「如抱恋人」、令其精神麻痹沉迷虚幻——**被心灵之刃免疫**（苏晓一记膝撞破招）。
- **狂想者·刻刻帝**（保命大招）：周身 3 米内**时间变慢**；副作用是**年龄退化、不可逆**（须回天启乐园恢复）——她为逃命用了一次、当场缩成 7、8 岁小萝莉（身体素质不降·借脖子变细从苏晓手里挣脱）。
- 装备：一把暗紫品质黑手枪、能包裹斩龙闪使其无法斩击的「银色水晶」、一条不到万不得已不用的珍贵「水晶吊坠」（疑保命 / 逃生神器）。

## 历程（十二卷8–10章）

她趁苏晓在黑土平原酣战时偷袭——却踢到铁板：苏晓识破其幻象、一记肘撞顶在她胸口（「**很软很软**」）、一刀擦过她背 + 险些斩头，吓得她连「天上地下唯我独尊」「狂想者」各种花招齐出、几度死里逃生（被界断线结界困、被剃瞬移追上）。最终她**靠「刻刻帝」时间术变小逃脱**、队友（感知女 + 狙击男 + 法阵男）以珊瑚定身弹接应（苏晓抗性高、只被定 0.4 秒）。苏晓认定她是个「喜欢装哔」的对手、不让她「装了就跑」、徒步紧追。

## 海皇道具 · 重伤被救（十二卷11章）

变小后的她（「银发萝莉」）靠体型在人潮中钻、加队友土系契约者远程改地形骚扰苏晓，竟越逃越远、还回头拍小屁股挑衅。苏晓以**环断清场 + [蜘蛛女皇](../力量体系/装备道具/index.md) 狙击**逼她出底牌——她捏爆一件**金色品质一次性道具「海皇」**（金色护盾 + 十米宽五米高的金色巨浪·触之即化血雾·苏晓全靠 500 能量盾 + 次级无敌护盾 + 距离才没被秒）。可苏晓**半空回身一狙**（枪术精通 Lv.30）正中她——**把她肚脐以下整个打碎、仅剩上半身**（双腿尽失），全靠队友拼死救走。队友决定送她去**别的战区找姐姐 [希女王](希女王.md) 救治**（希不在黑土战区）。她重伤垂死仍贫嘴「我嘘嘘的地方是不是被打碎了、那个变态」。

## 不甘报复 · 引兰爵杀苏晓（十二卷31章）

被希女王救回、下半身恢复后，她**心有不甘**——明知打不过苏晓、却「有筹码」：她**私下把姐姐团队里的异常强者 [兰爵](兰爵.md) 拿红水晶买动**、求他去黑土战区杀苏晓（其实是兰爵主动找上她）。事发被希女王看穿，她「认错态度极好、就是不改」（坚定「我不甘心」），希反顺水推舟、放她去做「她最擅长的事」（埋伏 / 暗算）——她颓废一扫而空、露出坏笑。提起苏晓仍心有余悸（「能打能抗、远近皆可、免疫控制、简直怪物·我最多抗他两刀、甚至一刀就被秒」）。

## 「噩梦女神」· 虚幻镜身骚扰（十二卷43–46章）

希女王放她去做「最擅长的事」——便是用能力「**虚幻镜身**」（无攻击力、能穿墙遁地的虚幻分身·造价不到一颗糖果·本是侦查能力）**专司「骚扰」**：自封苏晓的「**噩梦女神**」，飘到苏晓的房间里五音不全地循环唱儿歌（「我的小花花」）、敌人睡觉就唱、吃饭就装吐、上厕所就凝视……不让人安生。她曾凭这招把一名三阶契约者折磨到签下「互不招惹」条约。可苏晓靠 [心之冥想](../力量体系/技能/index.md) 硬扛三小时神色不变、还反夸「你唱的不错」、起身去接 [布布汪](布布汪.md)（带回了世界之核 + 天启美猫 [贝斯妮](贝斯妮.md)）。

- **贝斯妮的软肋**：贝斯妮是琉璃**可托付性命的好友**（琉璃看似跳脱、真朋友极少）。布布汪叼来昏死的贝斯妮时她瞬间失神——苏晓抓住这点，以贝斯妮要挟她拿「希的全部资料」（实为试探贝斯妮的分量）。她去找希商谈、却**慢了一步**：苏晓认定天启情报不可靠（两乐园无法签临时契约）、当她面**割喉杀了贝斯妮**（尸体化光消散）。她悲愤大骂苏晓「冷酷无情的混蛋」，却也无意中漏了话——「**果然不出姐姐的预料、你不会放过贝斯妮**」（贝斯妮之死竟在希的算计中·此话让苏晓套实了「希 = 贝斯妮的主人 + 九条命」）。最惨的是：她自以为「虚幻镜身」无懈可击，苏晓却**用 [吞噬之核](../力量体系/技能/index.md) 的蓝烟直接冲击她的精神**（之前一直按兵不动、装作被她骚扰得没辙）——她当场陷入「被几十个大汉……」般的痴女幸福呆滞、口水直流，被队友连扇耳光才唤醒、**记忆出现断层**（精神受重创）。醒来才惊觉自己「输得很惨、被利用、还害姐姐少一次重生机会、重要情报也被套走」，屈辱得直骂「为什么每个人都比我聪明」。

## 杀人小队 · C 战略团灭（十二卷69–70章·殁）

[希女王](希女王.md) 派她去解决在天启后方屠奶妈的「刺客」（不知正是 [苏晓](苏晓.md)）。她带齐**五人杀人小队**（自任控制 / 指挥·土系沉稳中年男主防御 + 控制·感知女连通五人精神链接·巨锤壮汉**道恩**〔单属性力 80·一锤秒杀脆皮〕·远处一名暗恋她的狙击手）——专为对付强契约者而非剧情人物的配置。一照面她就犯「苏晓恐惧症」想撤，被土系契约者劝住、放出「C 战略」：

- **C 战略**：以「狂想者·迷幻之境」全感官迷惑苏晓（视 / 听 / 嗅 / 感知尽欺·仅触觉残留）、土系花岗岩把苏晓连同 [反击盾](../力量体系/技能/index.md) 钉死在地、道恩巨锤猛砸 + 狙击手连射——三锤几乎砸破反击盾。
- **翻盘点 = [布布汪](布布汪.md)**：危急关头 [布布汪](布布汪.md) 趴远处石柱、把敌人**方位信息**传给被迷得「等还差 230 米」喃喃自语的苏晓——苏晓一枪 [蜘蛛女皇](../力量体系/装备道具/index.md) **把琉璃打飞**（左臂 + 小半身破碎、迷幻之境随之破灭）。清醒后的苏晓 [剃](../力量体系/技能/index.md) 腰斩感知女、凭神经反射躲狙击枪、一刀破脑斩杀道恩、「环断·血刃」斩死土系契约者、再蜘蛛女皇五枪轰碎 300 米外狙击手——**整支小队团灭**。
- **殁**：最后他走向半身破碎、爬不起来的琉璃，一句「**永别了，儿歌美少女**」、挥刀斩落。苏晓事后感慨「别人遇的萌妹子都是友军、我遇的十个有六个想杀我」、却仍不改行事风格。她的死是 [希女王](希女王.md) 与苏晓的死仇之一。
