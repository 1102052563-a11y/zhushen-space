# 金发骑士少女（裁决者预备役）

[轮回乐园](../总览/index.md) 的一名女契约者——满头**蓬松金发 + 一撮呆毛**、翠绿眸子、双手持**骑士剑**，气质酷似某「呆毛王」。她正在**虚空·黑渊浅层**接受成为 **[裁决者](../总览/轮回乐园机制.md) 的预备役试炼**（十六卷55章·插叙·非苏晓视角）。

> **注**：她疑似此前在乐园「曾为凑齐『骑士王』八件套向苏晓买护臂」的那名金发少女（见 [神皇冒险团](../势力/index.md) 条·待原文确认）；此处只记已读到的黑渊片段，不抢跑。

## 黑渊浅层试炼（十六卷55章）

- **环境**：虚空黑渊浅层——怪石林立的黑色戈壁，「**煞风**」过处能瞬间剥离生物血肉只剩白骨（对高阶生物只是「略感凉爽」）。她已在此磨砺许久，三天前遭遇一名**饿到失去理智的流放者（亚人种）**，二选一（杀对方 / 被吃光）下她斩杀了对方、但头部受重击，留下「起床懵逼」的后遗症。
- **能力**：她能开启一种「**灰色视野**」——睁眼后世界变灰、地上浮现彩色脚印辨别生物强弱：**红 = 极危险、白 = 可猎捕（食物）、黑 = 无法对抗**。战斗风格经地狱级实战后从「重心在腰」改为「重心下沉」。
- **善良的羁绊**：她与黑渊地下的 **嘞嘞斯诺族**（藏在地下、帮过她很多次的小鼠人）关系不错；常对弱小之人伸出援手——这正是考官眼中她「不够格」的原因。

## 裁决者预备役 · 考官的刁难

负责考核她的是一只**戴王冠、握羽毛笔、抱厚笔记本的青色癞蛤蟆**（考官）。它在笔记里评：**再试炼 107 天她就够格成为正式裁决者，但「性格还有问题」——依然会对弱者施以援手，而裁决者需要的是「绝对公正」、与她的骑士精神相冲突**；同批参加试炼的预备役**已全部死光、仅剩她这根独苗**（若一批里出不了一名正式裁决者，癞蛤蟆「没好果子吃」）。

为「让她重新尝到绝望的滋味」、磨去她的骑士精神，癞蛤蟆从笔记本里召来了 **1045 号**——一个**被封印的「屠夫」**（以杀戮为唯一乐趣、爱屠戮偏远星球种族、因此被封印），让他去黑渊「吓唬」她。

## 关系
- **1045 号（屠夫）**：被癞蛤蟆派来吓唬她的封印杀戮狂。
- **癞蛤蟆考官**：负责其裁决者试炼的王冠青蛙（嫌她太善良）。
- **嘞嘞斯诺族**：黑渊地下帮过她的小鼠人。
- **[苏晓](苏晓.md)**：疑为曾向他买「骑士王」套装件的金发少女（待确认）。
