# 卯之花烈（四番队队长 · 初代剑八）

[死神世界](../世界/任务世界/死神-尸魂界.md) 护廷十三队**四番队（医疗队）队长**——面容和善、温柔有礼的御姐。但这只是表象：她加入瀞灵廷前是名**恶徒**，更惊人的身份是**初代剑八**（每代剑八基本是十三队最强队长·她战力在队长中排前几），如今颇有「放下屠刀、改行救人」之意。[苏晓](苏晓.md) 评她**心机至少 80**（冬狮郎才 10）、是死神世界里**最难对付**的存在之一（比十二番队那名疯狂科学家涅茧利更棘手），身上有「不是杀几百几千人能有」的浓重血气。

## 抢人 · 笑里藏刀（二十九卷7–9章）

苏晓抛出混了灵子的炼金药剂钓出冬狮郎后，卯之花烈奉命来**判断苏晓的目的与能力**——她设过陷阱试探（「这些材料不是尸魂界有的东西」诱苏晓自曝来历、好令门外死神拿下他），被苏晓以「家传、用尸魂界材料替代即可」轻巧化解。确认苏晓的药剂价值后她**直接「抢人」**（药剂归医疗·她管辖名正言顺）、把苏晓收入四番队（临时成员→正式成员·激活瀞灵廷声望）。她与苏晓的相处是经典的**「两人面带笑容对视、却随时可能厮杀成一团」**——把旁边的松本乱菊吓出一身冷汗。她看破苏晓拿多余材料研究炼金炸弹却**不说破**（心知肚明即可·反让苏晓更放心）。她与苏晓有句招牌对白：苏晓一谈炼金「原理」，她便摆手「**请你说人话**」（耿直）。
