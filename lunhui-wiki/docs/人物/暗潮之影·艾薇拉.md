# 暗潮之影·艾薇拉

> ⚠ **深渊阵营除深渊之主外最强的存在**，也是苏晓在成为无上之境后遇到的第一个同位阶对手。

## 🌑 来历

- **黑暗温床的最强女儿**，[深渊化身](深渊化身.md) 的妹妹一系；不少人认为她是当代深渊之主最大的竞争者。
- ⚠ 她是通过**上古时代残存的「元素之眼」**晋升的无上，实力比旧王、[团长](星空座/团长.md) 稍弱。
- 她曾与**上一代天选**交锋、受伤太重而长期沉眠——⚠ **那一战正是当年上一代天选与旧王在最终决战时未能分出胜负的原因**。
- 她与当代深渊之主早就可以夺下深渊高座，却一直在等——等施法者们对力量的贪婪、等虚空的「元素真空现象」引发爆发式的深渊浩劫。最终它们成功了。
- 唤醒她的是 [暗域统帅·阿尔瓦·厄索](暗域统帅·阿尔瓦·厄索.md)——他交出了深渊之主赐予的指环来加速这一过程，理由只有一句:我们需要她。

## 🧒 她的童年（第4133章）

在意识深处，她曾回到那个破败的地下聚落:天空透出白炽色的强烈炙烤感，聚落的黑暗正在蒸发；再往下就是连通深渊之底的暗流地心。**全家唯一没患灼热病的她，缩在仅剩的那点阴影里**——前方是几具升腾着黑暗气息的白骨，它们死前很平静地坐着或躺着，**没有来争抢这唯一的生机**。

⚠ 她的执念也由此而来:「它们追随、相信我们，相信深渊时代会降临，所以我……不能输。」

## ⏳ 「时间暗潮」

她的力量体系是**时间**，而她所掌控的「时间暗潮」是**「时间长河」的阴影面**——另一种意义上的时间系主干。

!!! note "为何她不受因果反噬"
    传统时间系能力若敢这样干扰时间流动，对应的因果反噬将非常可怕。<br>⚠ 「时间暗潮」却没有这个问题——**因为整个「时间暗潮」都属于她**。

**消耗规律**（相对值）:逆转主时间轴「此刻」= 10；改变过去 = 50，且越往过去呈百倍千倍递增；**干涉未来 = 1~3**。⚠ 因此她的能力**更倾向于改变未来**。

| 能力 | 说明 |
| --- | --- |
| **时间插片** | ⚠ 她的招牌。在正面对斩后，她的时间线中「2 秒后会突袭到敌人身后」——她**直接来到未来 2 秒后的那个位置**，满蓄势的一枪瞬间刺出。**这比瞬移快多了，是绝对的 0 帧出手** |
| **瞬回** | 回溯自我的时间，回到十几秒前所在的位置（并非空间能力） |
| 时切 / 时之回响 / 反逆 / 因果逆还 / 时停刻 | 时间特性斩击、三种反伤＋一种强行停止敌人能力 |
| **时之门 / 时·旅行** | 划开世界跃入其中（⚠ 被苏晓凭纯粹斩击力**斩开并追进冥界**） |
| **时追忆** | 从时间长河中召来手持深渊大剑的高大黑影 |
| **自我共存** | ⚠ 最强能力。让「使用前 0.0001 秒的自我」「0.0001 秒后的自我」与现在的自我同时存在（三者**没有任何区别**、且处于**三个不同的时间维度**）。⚠ **必须先杀过去、再灭未来，最后才能杀死现在的她**——这个顺序是绝对的。代价是危机解除后，过去与未来的自我会反噬真实自我 |
| **让敌人衰老** | 高风险高收益。⚠ 她洞悉到「天锋斩影已满级、技法宗师只会老去不会更强」，却**洞悉不到【极之影】的存在** |
| **深渊凝视** | 深渊系的绝杀，一次让苏晓**骤然破碎 420 颗命核**（近四分之一） |

**武器**:赤红战枪 **【血泣誓约】**（陪伴多年，被苏晓斩至只剩下半部分）→ **【满月双子】**（来自混沌、拥有最强的「噬灭」特性）→ 从时间长河抽出的锐剑与深渊长枪 → **【时光低语】**（把「时间」实质化所构成，被斩中将陷入永无止境的时间轮回致因果崩溃）。另有 **【永夜】**（30 多厘米长、遍布孔洞的漆黑尖刺，刺中即被放逐到混沌深处）与 **【猎首】**（从时间长河窃取，每人一生只能射出三箭，⚠ **无论挡住还是命中，三箭都是因果级命中**）。

## ⚔ 三次交锋

### 第一次:遗忘之地的救场（54卷234章）

她把太阳队全员与「死前五秒的自己」重叠——太阳·塞斯当场失禁、距离老死只剩 5 秒。苏晓靠留在他队里的魔灵「替换」瞬间赶到，一刀斩断全场与衰老相连的因果。

⚠ 她判断出苏晓的魔灵**并非完全体**（似乎只有上半身），而按她与上古灭法者交手的经验，**不完全体的魔灵是绝不能用「替换」的**——她当年正是利用这点险些格杀一名上古灭法者。可她不知道，苏晓的魔灵强度已达 4 万点出头，**这在灭法阵营辉煌的上古纪元也足以让最顶尖的先代灭法陷入知识盲区**。

### 第二次:五五之数（54卷236章）

双方各自乘龙（巴巴托斯 vs 渊之龙）在虚空上空大战一小时。她的评估是:苏晓的无上之力强到离谱，**尤其是技法与体魄——体魄强大得比她更像深渊系无上**。而苏晓的评估是:正面对战他略强，但**想在混乱战场上斩杀这个时间系无上，成功率两成都没有**。

### 第三次:斩灭（54卷238–239章）

两战之间苏晓把全属性从 203 推到 316 点。她的直觉判断是「敌人上次没全力出手，是在麻痹我」——⚠ **因为另一种可能（在如此短时间内实力暴涨到这种程度）会让这场必胜的深渊战役一下就有了风险**。

- **8 秒死亡时区**:她被一刀轰穿黑暗堡垒后回溯，却发现回溯目标区间内的 8 秒（尤其是其中 3 秒）**全是绝对死区**，再触碰必死；
- **时间被斩碎**:她所有影响时间的能力都会被苏晓的斩威斩碎。⚠ 原文的注脚是——「时间被斩碎，这放在以往说出来都会惹人发笑的情况，此刻真实发生了」；
- **近战阴影**:她被斩到「快出近战阴影」，甚至一度徒手迎向刀锋，企图把斩龙闪**回溯到还不是「传世名刀」的时候**（那样她就能立刻崩断它）——被一脚直踹打断，撞上界壁又弹回来，⚠ **她真的是被踹懵了**;
- **十五次死而复生**:每次都是身躯、灵魂、因果的全面死亡再逆转，风险与代价极高；她与「未来艾薇拉」甚至开始出现简短交流，**代表两者正从一体变成两个个体**；
- **终局**:『无我之境·白夜』一刀斩下后，她并没被斩飞多远——反噬她的时之门忽然不再反噬她。⚠ 因为**她的存在因果被斩碎了**。左腿、右腿、双臂、躯干、连杂乱的半透明犄角都如晶体般碎裂。她留下的最后三个字是**「原来，是这样。」**

!!! danger "为什么「不死」的她真的死了"
    ⚠ 在绝对的斩击力下，**存在本身被斩碎**。别说过去、现在、未来三个状态，就算再多几个时间点，**任何时间的艾薇拉都只有一个结局:立即消亡**。
