# 香磷

[火影忍者世界](../世界/任务世界/火影忍者.md) [大蛇丸](大蛇丸.md) 的部下——满头红色长发、前凸后翘，**感知 + 医疗型**忍者（感知力极强、但缺乏逃脱手段）。

- **被苏晓「气息」吓到（九卷50章）**：大蛇丸设局对付 [苏晓](苏晓.md) 时她与 [水月](鬼灯水月.md) 被叫来助阵，却一见苏晓便瞳孔颤抖、毫毛直竖——直言「这家伙不知杀了多少人、气息都成红色、简直是刽子手」。
- **被胁迫加入鼬小队（九卷50–51章）**：[宇智波鼬](宇智波鼬.md) 的钓鱼局令大蛇丸逃脱、她吓瘫在地。鬼鲛 / 苏晓都主张杀她，**鼬以「下次任务需要感知型忍者」留下她**（晓接下来要抓**四尾**、只知大致方位需感知协助）。苏晓**强行喂她吞下一小块炼金炸弹**作控制手段（代替会削弱其感知的幻术）。她极识相、答应替晓做事、求事后放她并清除记忆。

## 抓四尾诱饵 → 被放归引佐助（九卷56–60章）
抓四尾时她被 [苏晓](苏晓.md) 当**诱饵**（割伤 + 开枪逼她真喊救命）骗出和善的 [老紫](老紫.md)；战后苏晓左手被烫伤，**咬她脖颈吸生命力**疗伤（她的能力：被咬即向对方传导生命力·原著二柱子咬太多致女儿长得像她）。封印结束后，苏晓**放她走**（让她吐出腹中炼金炸弹、给冬衣 / 地图 / 食物）——条件是回大蛇丸南方据点继续看守，并向「来找她的人」传话：「**想找鼬报仇，可来火之国神桥山找白夜**」（实为引 [宇智波佐助](宇智波佐助.md) 上钩）。她照办，遂带佐助一行来到神桥山。

- **入鹰小队 · 献血（九卷70 / 72章）**：大蛇丸「死」后她归入 [宇智波佐助](宇智波佐助.md) 的「鹰小队」。[苏晓](苏晓.md) 找她抽 100CC **富含生命力的血液**（她按苏晓事先嘱咐在抽血时外放生命力·漩涡一族失生命力能自恢复）作公平交易的报酬——这管血后来成了苏晓**救活长门的暗紫猛药**的关键材料。

## 被佐助弃 → 被苏晓收为「移动恢复药剂」（十五卷38–43章）

苏晓再入火影时她已随 [佐助](宇智波佐助.md) 的鹰小队助战，在石桥上协助佐助斗团藏——却被垂死的团藏挟为人质，**佐助毫不犹豫一记千鸟锐枪连她带团藏一并贯穿**（划破她心脏上侧）、随即弃她等死、头也不回去追团藏。心寒的她（「我真是个白痴、救了他那么多次」）靠**漩涡一族顽强生命力**苟住——被 [布布汪](布布汪.md) / [阿姆](阿姆.md) 发现，苏晓将她**救活并收编**：用白色小蜘蛛炼金炸弹堵住她心脏伤口（七成本会死）。

苏晓看中她 = **可移动的超级恢复药剂**：她有**漩涡一族稀薄的「仙人体」**（继承仙人「生命能量」多·故漩涡 / 千手两族同源、漩涡擅长封印 / 生命力·她的治疗因仙人生命能量比五代火影 [纲手](纲手.md) 更强·被咬一口即愈、≈ 一号药剂 ×20~30·零投入）、又是感知型。规矩：**每天最多咬 5 次**（超则效果减弱·10 次以上她会死）。她的**个人好感度**被苏晓激活（现 12 点·她极度惧怕苏晓→好感下限被压到 2·上限因苏晓低魅力降至 80），等于被「**刀架脖子**」、火影世界内不敢背叛（详见 [火影忍者·忍界公敌](../世界/任务世界/火影忍者-忍界公敌.md)「好感度机制」）；布布汪魅力高、其对她好感度反达 72（上限 130）。从此「跟着苏晓风险 ≥80」的她哭丧着脸入伙。

## 关系
- **[苏晓](苏晓.md)**：心理阴影 + 救命恩人 + 主子（收她当移动恢复药剂·好感度 = 刀架脖子）。
- **[宇智波佐助](宇智波佐助.md)**：旧主——**被他当人质刺穿心脏后弃如敝屣**，自此寒心。
- **[布布汪](布布汪.md)**：对她好感度更高（魅力高）·激活其光环保命。
