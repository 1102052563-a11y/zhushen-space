# 阿散井恋次（六番队副队长 · 露琪亚青梅竹马）

[死神世界](../世界/任务世界/死神-尸魂界.md) 中满头红发、缠绷带的死神——与 [朽木露琪亚](朽木露琪亚.md) 自幼在流魂街一同长大、又同期入真央灵术院，是她的**青梅竹马**。头脑耿直、非智谋派（在他认知里只有 [苏晓](苏晓.md) 是恶党、其余该是同阵营、想不通为何突然内讧）。随 [黑崎一护](黑崎一护.md)、四枫院夜一冲来救露琪亚。

## 接人送命（二十卷48章）

反水的 [卯之花烈](卯之花烈.md) 判断在场唯一对露琪亚绝无恶意的就是他、遂把露琪亚托付给他「带她走」。他抱着露琪亚冲入森林、却被蓝染同党**东仙要**用空间白绸传送到 [蓝染](蓝染惣右介.md)、[市丸银](市丸银.md)、苏晓面前——蓝染刚要「叙旧」，不喜废话的苏晓**一刀穿胸 + 一刀穿喉**两下要了他的命（掉一枚金色宝箱），蓝染随即取走露琪亚体内崩玉。「反派死于话多」、苏晓没给他半点反应机会。

## 关系
