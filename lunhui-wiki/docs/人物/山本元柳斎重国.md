# 山本元柳斎重国（护廷十三队总队长）

[死神世界](../世界/任务世界/死神-尸魂界.md) 护廷十三队**一番队队长、总队长**——光头老者、白须扎起、花白长眉垂至下颚，「年龄几乎等同尸魂界的历史」。看似年迈，却像**一座随时可能爆发的火山**，严肃、沉稳、深不可测。对所有四阶契约者而言他都是**怪物级**——[苏晓](苏晓.md) 自评不被他秒掉已是万幸（「那不是开猎魔时刻能否战胜、而是能否逃掉的问题」）。

- **斩魂刀「流刃若火」**：最强炎系斩魂刀（平时藏于上粗下细的木杖内）。**始解**火焰达太阳表面温度，**卍解**升至太阳核心温度——苏晓直言「横竖都是死、计算太阳多少度是浪费时间」。

## 收编苏晓（二十九卷9章）

魂灵药横空出世后，他召六队长议事；议中苏晓的实验室爆炸（暗里研究微型阿波罗），他只用鬼道在苏晓房屋周围布**结界**加固防御、未加追究。一句「你不错」（既像夸药剂、又像点破伪装），便把苏晓的实验室**安排在一番队驻地、自己房间隔壁**——既给苏晓在瀞灵廷自由行动的身份，也把这名可疑的「科研人员」放在眼皮底下看管。苏晓只受他与 [卯之花烈](卯之花烈.md) 两人管辖。

## 关系
- **[苏晓](苏晓.md)（白夜）**：收编入瀞灵廷的可疑「科研人员」（实验室安在自己隔壁看管）。
- **[卯之花烈](卯之花烈.md)**：四番队队长（与他同为苏晓的直属管辖者）。
