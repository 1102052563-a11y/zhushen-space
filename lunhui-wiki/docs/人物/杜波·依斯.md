# 杜波·依斯（太阳王国神使 · 苏晓战争主线目标）

[海蓝星](../世界/任务世界/海蓝星-世界争夺战.md) 上**太阳王国神庭的神使**——军中地位仅次于大神使（神使分两类：靠资历擅指挥的、与靠个人武力上位的「门面」）。儒雅、极厌烟味，却唯独容忍贴身猛将 [巴克](巴克.md) 在他办公室里抽劣质雪茄（因巴克曾九次救他性命）。是 [苏晓](苏晓.md) 的**战争个人主线任务击杀目标**（杀之奖黄金技能点 ×2·苏晓每 3 小时可锁定其位置一次）。

## 弃要塞而逃（十九卷36–40章）

圣利门要塞被苏晓夜潜炸塌时，杜波·依斯**抢先从五层密道乘单向飞机跑道的飞机逃走**、留巴克殿后挡苏晓。机敏的 [小助手](小助手.md) 没能在要塞拿下他、索性**乘在飞机下方**追踪（吞下苏晓给的信号发射器、刻字「已乘在飞机下方」报信）。苏晓遂靠追踪小助手的方位、绑车深入太阳王国境内（安贝拉城）继续追杀——他判断杜波不可能一程飞抵首都晨光城。

## 晚宴遇刺身亡（十九卷43–48章）

逃回安贝拉城后，他作为**大神使候选人之一**（现任大神使只是临时充场面），渴望赴大富商 **亨纳森**（无偿给神庭提供能武装十几万人的资金）的私人晚宴、捆绑大金主以争上位——尽管巴克警告他别去（圣利门要塞失守、他本就上位无望）。苏晓抓住这唯一机会布局：[布布汪](布布汪.md) 在后厨炸炼金炸弹制造混乱、[小助手](小助手.md) 扮「大小姐」混入会场喷煤油引燃，苏晓在斜对面酒店架 [寂灭公爵](../力量体系/装备道具.md)——从制乱到子弹穿墙仅 2.53 秒，**一枪 19.67mm 雷属性子弹轰碎了杜波·依斯的上半身**（巴克与护卫疤脸女都没能挡下这么快的雷属性弹）。苏晓由此**完成战争个人主线任务**。

## 关系
- **[巴克](巴克.md)**：他的贴身殿后猛将（被他从刑场救下·誓死替他挡苏晓·他死后巴克转去追杀苏晓）。
- **亨纳森**：邀他赴晚宴的安贝拉城大富商 / 神庭金主（晚宴正是苏晓的袭杀场）。
- **[苏晓](苏晓.md)（白鬼）**：奉战争主线任务追杀他的杀戮部队狙击手。
- **[小助手](小助手.md)**：乘飞机下方一路追踪他的格雅观察手。
