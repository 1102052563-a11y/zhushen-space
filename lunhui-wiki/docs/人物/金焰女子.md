# 金焰女子（虚空意识 · 伏笔）

[二阶生存试炼·吞噬岛](../世界/任务世界/T12K星球-生存试炼.md) 背后的一道**重大伏笔**——八卷36章 [苏晓](苏晓.md) 饮下用「[太阳恩赐果实](../力量体系/消耗品材料.md)」调配的强化药剂时，**眼前浮现的虚幻投影**：身穿金色长裙、裙带飘逸、**全身燃烧金色火焰**的女人（更像一道「**意识**」而非人类）。

- **登场**：她居高临下喝令「**凡人，跪下**」，并升起足以烧穿空气的烈焰要焚尽苏晓——可苏晓体内**升起一股蓝色火焰**（[灭法之影](../力量体系/技能.md) / 青钢影）将其灭去，金焰女子**眼中涌现一丝畏惧**，呢喃「灭……」，改口「**离开我的星球，这里不欢迎你们这些灭法的疯子**」，随后化作藤蔓向苏晓攀附（幻象即散）。
- **含义（推断）**：苏晓判断这是**某个意识**、且自己**恰是它的克星**（故它形成幻象恐吓他）。结合「太阳的信仰 / 血藤战士 / 原生之种 / 太阳恩赐果实」一连串线索，金焰女子疑是**红土区「太阳信仰」背后的存在 / T12K 星球的幕后意识**，与亚森曼部落的秘密同源；它**惧怕「灭法」**，正与苏晓的职业 [灭法之影](../力量体系/技能.md)（恶魔族称「破法者」）相呼应（伏笔待续）。
