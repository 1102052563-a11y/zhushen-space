# 信长·哈查马

[全职猎人世界](../世界/任务世界/全职猎人.md) **[幻影旅团](../势力/index.md)（蜘蛛）成员**——**强化系念能力者**，擅用刀、是**聚合拔刀术高手**，旅团主要战斗成员。性格直来直去，与同为强化系的 [窝金](窝金.md) 是挚友，二人配合后战斗力飙升（默契度旅团第一，但非旅团最强组合）。

## 历程 · 山谷之战 · 死亡（十卷39–40章）

随旅团在独眼巨怪山谷围困黑帮。[苏晓](苏晓.md) 在 700 米外用 [蜘蛛女皇](../力量体系/装备道具.md) 连射模式发动**三连狙杀**、专挑信长（理由：信长与窝金配合后较强、且用刀之人遇狙击的本能就是劈飞子弹）：

- 第一枪被信长本能侧身、以刀劈挡——长刀崩出大缺口、持刀臂严重脱臼，崩飞的残刃还穿透了 [小滴](小滴.md) 的小腹（足见 17.36mm 弹的恐怖力量）；
- 第二枪被堪堪躲过（炸出大坑）；
- 第三枪预判其移位，**轰碎信长整个左胸 + 炸飞一条手臂**。信长临死前仍喊出「敌人在东面山上」，为 [玛奇](玛奇.md)、[侠客](侠客.md) 等争取了躲避时间。

**结算：[苏晓](苏晓.md) 得世界之源 +7.4%（→29.7%）、旅团声望 −300（→−200·仇敌）、掉落暗紫宝箱**。

## 关系
- **[幻影旅团](../势力/index.md)**：强化系主战成员。
- **[窝金](窝金.md)**：挚友、默契搭档——窝金为他的死激怒、拼死复仇。
- **[苏晓](苏晓.md)**：被其狙杀。
