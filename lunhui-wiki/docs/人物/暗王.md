# 暗王·米纳斯

[黑色森林](../世界/任务世界/T12K星球-生存试炼.md) **深处**沉睡的恐怖存在——[苏晓](苏晓.md) 隐藏任务**第三阶段（最终阶段）「暗王」**（Lv.**19**）的目标：**在暗王彻底苏醒前将其消灭**（连锁任务·按完成度给奖·无惩罚）。它其实是**被虚空流放到吞噬岛的犯人**之一（黑色森林里的强者皆是此类·见 [生物·万族](../世界/生物万族/index.md)）。

## 本体 / 数据（哈瓦呐之眼·八卷66章）
全身漆黑、肌肉虬结的**三米壮汉**——秃头、周身飘黑雾、膝肩扣着镶铆钉的粗大铁环。沉睡于黑雾之源中的一口**深竖井**（黑色森林安全点·黑雾之源侵蚀不到此处）。

- **名称**：暗王·米纳斯（首领级生物！）｜**类别**：变异恶魔（**苏醒 29%**）｜生命 70%｜法力 310｜**力 50 / 敏 50 / 体 50** / 智 31 / 魅 4。
- **技能1 领主权威**（被动）：生命 +2000、力 +16 / 敏 +13 / 体 +15。
- **技能2 恶魔之魂**（主动）：完全恶魔化、属性大增、全抗 +40%（**未完全苏醒无法使用**）。
- **技能3 怒吼**（主动）：震荡敌人大脑的声波·300 伤 + 2 秒眩晕（未苏醒时威力大打折扣）。
- **技能4 不死者头领**（被动）：召唤、指挥 **[不死者](../世界/生物万族/index.md)（共 67.5 万）**（未完全苏醒无法使用）。
- **技能5 升空**（主动）：跳上高空全力下砸·力×4 + 体重×0.2 + 高度×0.6。
- **技能6 暗黑陨石**（终极技）：召唤 3×3 米陨石高空砸落·500 + 力×2 + 智×5 伤害 + 恶魔之焰灼烧（10 秒·每秒 30）。
- **⚠️ 每小时苏醒 2%·苏醒度超 70% 即完全苏醒**（届时可恶魔化·秒杀苏晓不在话下）。它寿命悠久、每次沉睡后都有**虚弱期**——苏晓正是抓住这虚弱窗口才有一线生机。

- **⚠️ 苏醒即升级**：任务信息明确警告——**若暗王完全苏醒，任务难度将飙升至 Lv.28**。如今它仍处**半苏醒**状态，[野猪](野猪.md) 称「半苏醒的暗王，你（苏晓）有可能杀掉」；可一旦全醒，苏晓「就死定了」。苏晓自承在黑色森林**尚未用出全部实力、当前也还不是暗王的对手**。
- **「同化」之源**：暗王能把深入黑色森林的生灵（包括契约者）**同化**成人不人鬼不鬼的妖兽形态——[野猪](野猪.md)（被同化的契约者）正是「它的杰作」，[托马](托马.md)（变成硬壳王八）亦然。被同化者**失去痛觉、近乎不死**（剁成十几段也不死）、靠吃 [不死者](../世界/生物万族/index.md) 存活。
- **与不死者**：黑色森林内**数以万计的不死者**听其驱使 / 因它而生；小队进林即被几万不死者远远包围（那些东西畏惧某种存在、不敢靠近）。
- **线索**：黑色森林的亚森曼遗迹石碑残文「**第十三星·寒之·试验·失败……**」、老巴尔所知的四个关键词「**寒灾·不死者·暗王·同化**」，都指向暗王与这座「牢笼」的来历（待揭）。
