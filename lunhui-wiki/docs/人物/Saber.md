# Saber（阿尔托利亚·潘德拉贡）

[Fate/Zero 世界](../世界/任务世界/fate-zero.md) 第四次圣杯战争的 **Saber（剑之骑士）英灵**——真名**阿尔托利亚·潘德拉贡**，即古不列颠传说中的**亚瑟王**（骑士王）。金发碧眼、头顶一缕标志性「呆毛」、容貌纯净；性格**忠诚正直、谦逊有礼、认真**，恪守骑士品格（连「在手画令咒欺骗他人」都因有违「诚实」而拒绝）。持圣剑 **Excalibur**（宝具 A++）。御主是 [卫宫切嗣](卫宫切嗣.md)——二人**相性严重不和**（她重骑士道、切嗣信奉「牺牲少数救多数」的暗杀功利主义）。

- **能力**：耐力 / 敏捷 / 魔力皆 A（受强御主增幅）、宝具 A++；**A 级「直感」**——可短时间预见未来（如提前预判对手斩击角度）。
- **十一卷17章**：她察觉到苏晓这「奇怪的英灵 / 御主组合」（看穿其令咒是画上去的、身份对调），自评若与苏晓单挑约 **五~六成胜算**（宝具占优、刀术能应对），**唯独最忌惮苏晓手上「能吃掉我」的蓝色烟气**（[吞噬之核](../力量体系/技能.md)）。苏晓也视她为本届少数「难对付、甚至能杀掉自己」的英灵之一。
- **拒斥苏晓 · 英灵齐聚（十一卷50–51章）**：[卫宫切嗣](卫宫切嗣.md) 电话联络苏晓谈联手时，她直言「与这种恶徒合作风险很大」、斥苏晓「**邪魔外道**」——被苏晓以「**骑士等于蠢**」的歪理激将（骑士美德自相矛盾云云），她虽看穿是激将法、却仍因「骑士」一词受辱而动怒。随后废弃工厂，她从 [哈桑·萨巴赫（百貌）](哈桑·萨巴赫·百貌.md) 的暗杀下护住代理御主 [爱丽丝菲尔](爱丽丝菲尔.md)、与之交手，引出「英灵齐聚」之局；苏晓气势压场时她一个健步冲到爱丽丝菲尔身前守护。她还为不识苏晓职阶的 [征服王](征服王·伊斯坎达尔.md) 解惑（「他是 Berserker」）。英雄王被令咒召回、[征服王](征服王·伊斯坎达尔.md) 提议罢战后，她附议「混战御主难活」、**带着代理御主退入废弃工厂**。苏晓评她「圣剑 + A 级直感、不好对付」、是本届少数能威胁到他的英灵。

- **护主不得、御主身死（十一卷62–63章）**：苏晓潜到爱因兹贝伦城堡外狙杀 [卫宫切嗣](卫宫切嗣.md)——她的 A 级直感在使徒之眼探测城堡的瞬间预警、抱起 [爱丽丝菲尔](爱丽丝菲尔.md) 飞奔去救御主，**仍慢了一步**（赶到时御主已被巨弹打碎）。苏晓随即一枪逼来、她以圣剑劈开子弹护住爱丽丝菲尔——却被苏晓的「选择题」困住：追击则爱丽丝菲尔死。正直的她**选择守护卫宫切嗣的妻子、放苏晓离去**（「我不能看着你死、你是我 Master 的夫人」）。正牌御主已死、虽有代理御主爱丽丝菲尔（非七御主之一）续命，**她终将消散**。

- **最后一面 · 化为「裁决者（预备役）」（十一卷67章）**：苏晓与英雄王死战后濒死、回归前 2 分钟，她从暗处现身——**没有趁人之危杀这个重伤的胜者**（恪守骑士道·苏晓暗布界断线 + 炼金炸弹防她、又嘴上激她「骑士精神束缚行事、冲上前砍敌脑袋才是王道」）。她斥之「卑鄙」、并立誓「**任何人的生命都不可亵渎、无论善恶、有机会一定用行动告诉你**」。苏晓「谢」她废话两分钟（拖到回归）后消失。她随即在消散之际被一条「**规则之力锁链**」刺中——轮回乐园判定她**潜力 S~SSS 级**，将她**转化为「裁决者（预备役）」**（本世界线由其他同职阶英灵代替、特性更改至「唯一」），传送往「**虚空黑渊战场**」历练 **397 个自然日·死亡概率 98.12%**（同批选 65 人、黑渊生存率 0.53%）；若完成历练则晋升正式裁决者、监督 1~9 阶空间战场。她一睁眼便置身黑岩深渊、目睹吸食黑龙的「拉里因族」人形生物与一名**一指爆杀强者、绝望级的喝酒男人**——领略了远超圣杯战争的虚空恐怖（重大世界观一瞥：虚空黑渊 / 裁决者机制 / 顶尖虚空强者）。

## 黑渊重逢 · 裁决者「小可爱」（二十三卷3–33章）

被送往黑渊历练的她，在 [黑渊浅层 · 赔罪者之殇](../世界/任务世界/黑渊浅层-赔罪者之殇.md) 中已能独立做事——成了虚空强者 [癞蛤蟆 + 1045 号](黑渊配角.md) 培养的裁决者「**小可爱**」（癞蛤蟆找人教她本领、救她几十次命、是她真正意义上的师傅·她态度虽差却离不开）。她剑术还粗糙、持「契约之剑」宝具；曾被癞蛤蟆设局让酒馆酒客围攻历练，也曾把昏迷的 [莉莉斯](莉莉斯.md) 送到 [苏晓](苏晓.md) 处。后因 1045 号去抢 [恶魔之罐](../世界/生物万族/index.md)，她一行被 [恶魔老太婆](黑渊配角.md) 一并吞掉——靠苏晓回赠黑枫树皮的人情、加上她身上与苏晓「**同源**」的气息（灭法者一脉·她正是从苏晓的圣杯世界进度里走出、跨维度成「唯一」存在），被老太婆吐出放过。重逢时她仍称苏晓「恶徒」、却欠下救命之恩（被布布汪 / 癞蛤蟆起哄「肉偿」、她沉默离去）。

## 正式晋升裁决者 · 师傅离去（二十五卷59章）

历练有成后，[癞蛤蟆](黑渊配角.md) 在一处中烈度战场正式授予她 **ф 型印记**——她从「预备役」**正式成为 [裁决者](../总览/轮回乐园机制.md)**（轮回乐园第二梯队·「**旗帜**」），职责是在战场督战、阻止契约者「越境」逃离战区（越者杀无赦），遇异常 / 高危违规者则**交给第一梯队的猎杀者**（如 [苏晓](苏晓.md)）去清理。获印记后她的**英灵性质消失、成了真正存在于世的人类**。

癞蛤蟆随即功成身退（独白透露自己其实是个**「从良」的异常**——它一直舍不得把这「小可爱」丢进黑狱、培养了这么久终于放手·去找老恶魔们喝酒）。她郑重躬身道谢，癞蛤蟆留言「遇到像我这样的异常别试图战斗、像我这种从良的不多」后消失。

## 协助苏晓清违规者（二十六卷·黑王座大陆）

作为裁决者提前进入 [黑王座大陆](../世界/任务世界/黑王座大陆-死寂城.md)——再遇 [苏晓](苏晓.md)（被勾起圣杯世界的惨痛回忆、一度拔剑、误以为苏晓在欺负小公主 [佩黛菈](黑王座大陆配角.md)）。经黑渊磨砺后她已悟「亚瑟王不懂人心」、不再事事亲力亲为，主动表示**只在「清违规者」一事上协助苏晓**（她更懂权斗·确保苏晓与契约者间不起冲突）；却嘴硬「能做到的话尽管来」回应苏晓的威胁。与软萌的佩黛菈意外成了**吃货好友**（聊圣城美食）。后佩黛菈被违规者狄鸦拔意识害死（苏晓始终在利用她当傀儡）——她为挚友之死与苏晓决裂、清完违规者后要约苏晓去乐园竞技场单挑（「你利用我的挚友、她的亡魂在哭泣」），话没说完就被 [癞蛤蟆](黑渊配角.md) 抓回——师傅嫌她「守护骑士」的表现不合格、要把她**回炉重造**（又一次丢回黑渊）。
