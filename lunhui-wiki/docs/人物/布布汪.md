# 布布汪（撕空恶犬·布布特尼）

[苏晓](苏晓.md) 的**头号随从**——昵称「**布布汪**」（大名「撕空恶犬·布布特尼」）。本页追踪它的来源、能力、各世界历程与逐世界结算快照。

!!! note "随从建页规则"
    苏晓身为**猎杀者**不便加入冒险团（怕猎杀目标是队友），但仍可能通过道具、技能、契约获得随从 / 召唤物。**每个随从 / 召唤物各单开一页**（与「人物建页铁律」一致），在 mkdocs.yml「主角与随从」分组下逐条列出；目前在册仅 **布布汪** 一员，将来新增随从即在其下添加新条目。

## 撕空恶犬·布布特尼（生物类帝具 · 紫色）
- **来源**：用紫色「[帝具改造蓝本](../力量体系/装备道具.md)」合成而来（四卷41–42章）——主材料 = 帝国生物类帝具「魔兽变化·百臂巨人」（小白狗）+ 超级危险种 [芬里尔的精魄](../力量体系/消耗品材料.md)（默合度 96%）；副材料 = 7 件物品类帝具。归属者：苏晓。
- **外形**：黑白花色的**狼狗合体**巨兽（约 1 米多高 / 3 米多长·比虎更大），与巨狼芬里尔有几分神似；有点「二货」（让它向前冲会耿直撞墙），却对苏晓绝对忠诚、会**撕碎任何敢靠近苏晓的生物**（「恶犬」名副其实）。
- **核心形态**：可收起为拳头大小的**冰蓝色球体「布布特尼的核心」**（紫·评分 130·生物类帝具·售 36000 币·极便携，原著从未出现），随时召唤 / 收回，便于在帝都藏匿。需正常进食维持活力（约 3 天一次·饭量大）。
- **属性**：生命 100% ｜ 法力 260 ｜ 力 12 / 敏 15 / **体 20 / 智 26** / 魅 10（体力、智力**比苏晓还高**）。
- **技能**：
    - **全速冲锋**（主动）：移速 +100%（约等于 +30 敏的速度）·可载重·耗 10 法力/分。
    - **冰雪女神的馈赠**（光环）：每秒回友军 1 点生命·无消耗·随智力提升。
    - **忠心护主**（主动）：耗 50% 法力、3 秒内回主人 50% 生命·冷却 48 小时。
- **定位**：极品**辅助类**帝具 = **坐骑 + 山寨奶妈**（追杀 / 逃跑 / 持续回血一体）；战斗力不强（无攻击技能），战时多在旁待命、苏晓不顺时可骑它逃命。嗅觉远超普通犬类，是绝佳追踪工具（苏晓憾「若早有布布、雷欧奈赤瞳跑不掉」）。
- **历程**：合成当日即认主（四卷42章）；随苏晓骑乘赶路、在特利村废墟刨出幸存者、循塔罗牌气味助苏晓追查卡尔、嗅出夜袭部队总部（四卷42–44章）。在帝都须收成核心藏匿（其形似芬里尔、恐被艾斯德斯认出·且不在帝国 48 帝具记录内）。**苏晓 vs 卡尔一役，布布躲在暗处偷偷以「冰雪女神光环」+「忠心护主」为苏晓回血**——这是苏晓扛住卡尔自爆、最终反杀的关键（「因为我带了条狗」）（四卷46章）。在皇宫地下的 [比鲁马克炼金试验场](../世界/地点/index.md) 中，布布**不免疫炼金毒素**（每秒 -1 真伤·但靠自身光环可自愈），作为苏晓的**坐骑**载着他与小皇帝逃过蓝色小鸟群、聚合虫；对凶悍的地穴人看守也敢引走缠斗，并几次凶相毕露「提醒」不老实的小皇帝（四卷59–63章）。
- **寄生兽空间战场（五卷·随苏晓出战）**：在裁决者身份下被乐园免费**伪装成漆黑的「地狱三头犬」**（颈侧多生两头·仅契约者视角可见·能力不变），驮苏晓巡战场、拦逃兵、深入废土设世界坐标。受命去诱**变种马来鳄**群时连嘲讽带尿都引不动、反被鳄群追得「憋着半泡尿狂奔」（一度被吓尿·心里苦）；首领巨鳄偷袭时想助战被苏晓喝退（战斗力弱·留在「冰雪女神光环」覆盖范围内默默替主回血），决战迪卡时也帮苏晓挨了两拳、半死续命（五卷4–23章）。随核心带回乐园后与苏晓一同被绿光治愈。
- **世界争夺战·被委以重任（十二卷3章·[维斯星球·多亚古大陆](../世界/任务世界/维斯星球-多亚古大陆.md)）**：进二阶世界争夺战后，它**竟被编入 50 人的敏捷类契约者「斥候小队」、负责寻找战争任务核心「世界之核」**（在帝国首都圣龙城·与在黑土平原前线的苏晓分隔两地）——这「重任」让苏晓大跌眼镜。另：它已自悟隐匿技能「**你看不到我**」（按地形伪装成柯基 / 沙皮狗 / 胖橘猫等·现 864 种），并沉迷一款能连现实网络的手游（战绩 0-10-0·被苏晓嫌「好菜」）。
- **「世界之核」抢夺战 · vs 美猫贝斯妮（十二卷11–12章）**：「世界之核」（指甲盖大的亮晶晶石头）**无法放进任何储存空间、只能随身携带**——布布汪靠「你看不到我」神出鬼没地从天启乐园契约者手里**偷走世界之核**（一度让乐园公告在「己方得核 / 敌方夺回」间反复横跳·把苏晓看懵、生怕它「一激动把核啃了」）。它的死对头是天启乐园的侦查美猫 **贝斯妮**（会人言的布偶猫·从布布汪口中夺回过核）——一猫一狗在圣龙城地下逃生通道里上演「争霸战」：布布汪叼着她脑袋左右抡甩、把她揍哭、夺回吞进猫肚的世界之核、还顺手「拐走」这只美猫。它**只肯把世界之核交给苏晓一人**（不信同阵营的斥候）。脸上那几道猫抓痕便是「战功」。
- **乐园停留趣闻（五卷25–28章）**：精力旺盛、把 15 币的沙发啃穿藏身其中；苏晓带它去职工者宠物口粮店买烤骨头（**乐园有召唤兽口粮店**——召唤系职业若魅力低、召唤兽太饿会「噬主」，故需备口粮），女职工者一句「给它绝育治啃家具」吓得布布夹腿小碎步逃。布布魅力高达 10、兽语可与苏晓部分交流（苏晓兽语 Lv.6）。
- **二阶生存试炼·吞噬岛（八卷13–17章）**：背着蜘蛛女皇跳伞，空中遇五米钢羽黑鹰时**学起鹰叫「交涉」、竟与黑鹰打成一片**（化敌为「同伴」）；落地倒栽葱挂树被苏晓刀芒救下后啃树泄愤；红土区饿到「吃草也不错」。最惊人的是——**它耳濡目染学会了用蜘蛛女皇开枪**（一枪打爆偷袭苏晓的红皮土著、却把苏晓吓得以为遭敌方狙击手；这把重狙后坐力让它生命 −15%、胸口白毛渗血）。「布布汪学会用枪、已经不是拖鞋可以制裁的了」。八卷31章奉命「打入敌阵突围」、与苏晓走散后在红土区里**茫然乱窜、掉进沼泽险被吞**、沾成一身灰泥，直到八卷35章苏晓敲骨为号才**伏地鬼祟地赶来会合**（领教了「没有主人在身边的红土区」有多恐怖），被苏晓拎去山泉洗澡、饱餐一锅蘑菇蔬菜肉汤。
- **吞噬高等生物进化（八卷45章·「高级生物」技能首次见效）**：此前布布汪吃红土区野兽的血肉都「无效」（且挑食·吃生肉如吃药），直到苏晓逼它（脱鞋威胁 + 它含泪「实力演技派」狂吃）**啃下半只精英刺蝎**——「高级生物」技能终于触发：**力 +2 / 敏 +1 / 体 +3 / 魅 +2·「全速冲锋」微量加强**（同类生物只生效一次）。**意义在技能可被强化**——日后若能强化「冰雪女神光环」，它就从「弱鸡奶妈」晋升「业余奶妈」。「高等生物」定义模糊，此后苏晓杀的生物都让它尝尝（懵懂的布布汪还不知今后要吃多少奇怪东西）。

### 进化为真生命「从者」（六卷79章 · 第三生命炼成法阵）
苏晓在交易市场用拍卖牛牛果实换来暗紫色消耗品「**第三生命炼成法阵**」（传奇炼金师卡恩之作·可把机械 / 能量生物**重塑为真正的生命体**并提升其能力）。布布特尼一直渴望摆脱「收起来就是个玻璃球」的帝具身份、想真切感受主人摸头的体温——它一口咬碎法阵，化作一颗三米高的灰茧，**6 小时后破茧而出**：外表毫无变化、二货意识依旧，却已是**有血有肉的高等生命体**（阶位远超魔兽·从此具备成长潜力·味觉 / 触觉完整）。返回现实世界时，作为苏晓的「从者」**也一并回到现实**（其回血光环在没有恢复品的现实弥足珍贵）。

- **进化后属性（从者）**：生命 100% ｜ 法力 **270** ｜ 力 23→**24** / 敏 27→**29** / 体 25 / 智 27 / **魅 43**（苏晓裸魅的 14 倍·堪称「犬类最帅二货 / 妹子杀手」·但魅力不影响主人苏晓）｜ **十卷20章吞独眼巨怪血肉「高级生物」技能再进化**：力 +1 / 敏 +2（含泪吃下三大碗·它已挑食：生的 / 恶心 / 可疑 / 太咸的都不吃）。
- **六大技能**（核心技能保留并增强）：
    - **全速冲锋**（主动）：移速 +150%·可载重·耗 12 法力/分。
    - **冰雪女神的馈赠**（光环）：每秒回友军 1 生命·无消耗·随智力提升。**十卷17章「商业头脑」初显**：杀独眼巨怪的团战中，它先免费放光环给众契约者尝甜头、再改成「**每分钟 10 乐园币付费激活**」收钱（十名契约者乖乖付费），还自拟临时契约**分给苏晓六成**——其智商 / 鬼点子被一群勾心斗角的契约者彻底激活。
    - **忠心护主**（主动）：耗 50% 法力、3 秒回主人 **60%** 生命·冷却 24 小时。
    - **高级生物**（主动）：可**吞食高等生物的血肉进化**（成长潜力）。
    - **忠心耿耿**（被动）：忠诚度锁定至最高·任何情况无背叛可能。
    - **你看不见我**（被动·自悟）：可依任何地形隐藏（已掌握 678 种·曾把整间专属房间啃成废墟后蹲吊灯上让苏晓遍寻不着）。
- **十卷·拍卖会期间（十卷25–30章）**：与苏晓在友克鑫拍卖会潇洒度日——爱干净每天洗澡、双爪捧热牛奶慢喝、玩手游。**首功**：拍卖会开幕夜在一楼巡逻时**听到地下室异响、发现潜入的 [幻影旅团](../世界/任务世界/全职猎人.md)**，让苏晓得以及时下去拦截 [飞坦](飞坦.md)（旅团并非被苏晓自己察觉）。飞坦放小型太阳前被苏晓提前打发去远处「烤火」避难（赶路掉进灰堆、成了只满身黑灰的「黑狗」、抖一身黑烟，气鼓鼓）；事后又**循气味带苏晓追到飞坦藏身的民宅**，助其斩杀飞坦。**山谷决战（十卷38–43章）**：苏晓狙杀旅团时它背朝主人蹲守、警戒身后；苏晓与 [窝金](窝金.md) 死斗、重伤断臂后，它全速冲进山谷「上车救驾」、驮着苏晓撤到山谷边缘的汽车旁。**决战前夜（十卷47–50章）**：靠嗅觉替走错路的苏晓 / 黑玛丽找到新拍卖场会客厅；还回敬了来找苏晓合作的 [薇妮](薇妮.md) 一记「汪之蔑视」（把这位炮台小姐气得不轻）。**拍卖场攻防战（十卷52–57章）**：开放「冰雪女神光环」权限给守场的众契约者群体回血（实用得人人满意）；战后向苏晓汇报后门战况（赖「兽语 MAX」沟通）。**突袭小丑 · 间谍 MVP（十卷59–62章）**：开「你看不到我」混入尼拉潘大军、**打入尼拉潘族高层当间谍**，探回关键情报——小丑与团长在半公里外酒店、酷拉皮卡杀了旅团三人、友克鑫尚有 2700+ 尼拉潘（头目级 5 / 首领级 1）、小丑断肢未复无战力——这些情报让小队从被动反转为主动突袭，是这一战毋庸置疑的 **MVP**。（注：吞独眼巨怪血肉后它属性剧增·魅力高达 **53**，而「你看不到我」的强弱正取决于魅力——故该技能效果惊人。）**返园开宝箱（十卷71–73章）**：把物品获得权转给苏晓、帮忙开那一百多枚宝箱（开出蓝材料时乐得摇头晃脑）；又被苏晓闭眼升刀术大师时的斩击吓得戴安全帽躲墙角；事后领 5000 币零花钱去置办（被它「一点点啃短」的）家具。**返现实 / 进 Fate/Zero（十卷83–十一卷1章）**：随苏晓返三线城市老家，坐高铁被托运、晕车晕得生无可恋；随后随苏晓一同从空间裂缝走进 [Fate/Zero 世界](../世界/任务世界/fate-zero.md)（苏晓以伪英灵身份入世）。**Fate/Zero·樱的依靠（十一卷3–9章）**：它 53 点的高魅力对被虫池折磨绝望的小女孩 [间桐樱](间桐樱.md) 形成「心灵慰藉 + 依靠」——樱一步不离地黏着它、躲苏晓凶气也往它身后躲（连拧不开的水瓶都求它用前爪拧）；苏晓遂让它专门负责带樱隐蔽。它也嫌改造成「伪 Berserker」的间桐雁夜满身油漆 + 黑烟味太冲。**柳洞寺（十一卷19–22章）**：随苏晓闯进柳洞寺结界后，奉命在山下看护 [间桐樱](间桐樱.md) 与伪 Berserker 间桐雁夜（苏晓嘱「危险时卖一个〔雁夜〕保一个〔樱〕」、若雁夜死就立刻带樱去与苏晓签英灵契约）。**忽悠平头哥（十一卷32章）**：被苏晓一脚踢飞、回头来寻仇的蜜獾怨生物「平头哥」，竟被布布汪靠「兽语交流 + 零食诱惑」一通**大忽悠**劝住、化干戈为玉帛（苏晓险些打不过它·全靠布布汪化解）。还在苏晓觉醒天赋发狂的间桐雁夜面前炸毛护主、把雁夜拖出房间。

## 逐世界结算快照
> **铁律**：苏晓**离开某任务世界**时，名下每个随从 / 召唤物**各自单列**（一个角色一份·绝不与他人合并·新随从另开页），记录其**离世瞬间定格**——格式与 [苏晓主角结算快照](苏晓.md) 一致（项 / 值表：形态/品级·属性·技能·本世界作用·去向）。苏晓**首入海贼王 / 东京喰种 / 进击的巨人**期间名下无随从，故无快照。
