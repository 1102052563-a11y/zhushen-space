# 布布汪（撕空恶犬·布布特尼）

[苏晓](苏晓.md) 的**头号随从**——昵称「**布布汪**」（大名「撕空恶犬·布布特尼」）。本页追踪它的来源、能力、各世界历程与逐世界结算快照。

!!! note "随从建页规则"
    苏晓身为**猎杀者**不便加入冒险团（怕猎杀目标是队友），但仍可能通过道具、技能、契约获得随从 / 召唤物。**每个随从 / 召唤物各单开一页**（与「人物建页铁律」一致），在 mkdocs.yml「主角与随从」分组下逐条列出；目前在册仅 **布布汪** 一员，将来新增随从即在其下添加新条目。

## 撕空恶犬·布布特尼（生物类帝具 · 紫色）
- **来源**：用紫色「[帝具改造蓝本](../力量体系/装备道具.md)」合成而来（四卷41–42章）——主材料 = 帝国生物类帝具「魔兽变化·百臂巨人」（小白狗）+ 超级危险种 [芬里尔的精魄](../力量体系/消耗品材料.md)（默合度 96%）；副材料 = 7 件物品类帝具。归属者：苏晓。
- **外形**：黑白花色的**狼狗合体**巨兽（约 1 米多高 / 3 米多长·比虎更大），与巨狼芬里尔有几分神似；有点「二货」（让它向前冲会耿直撞墙），却对苏晓绝对忠诚、会**撕碎任何敢靠近苏晓的生物**（「恶犬」名副其实）。
- **核心形态**：可收起为拳头大小的**冰蓝色球体「布布特尼的核心」**（紫·评分 130·生物类帝具·售 36000 币·极便携，原著从未出现），随时召唤 / 收回，便于在帝都藏匿。需正常进食维持活力（约 3 天一次·饭量大）。
- **属性**：生命 100% ｜ 法力 260 ｜ 力 12 / 敏 15 / **体 20 / 智 26** / 魅 10（体力、智力**比苏晓还高**）。
- **技能**：
    - **全速冲锋**（主动）：移速 +100%（约等于 +30 敏的速度）·可载重·耗 10 法力/分。
    - **冰雪女神的馈赠**（光环）：每秒回友军 1 点生命·无消耗·随智力提升。
    - **忠心护主**（主动）：耗 50% 法力、3 秒内回主人 50% 生命·冷却 48 小时。
- **定位**：极品**辅助类**帝具 = **坐骑 + 山寨奶妈**（追杀 / 逃跑 / 持续回血一体）；战斗力不强（无攻击技能），战时多在旁待命、苏晓不顺时可骑它逃命。嗅觉远超普通犬类，是绝佳追踪工具（苏晓憾「若早有布布、雷欧奈赤瞳跑不掉」）。
- **历程**：合成当日即认主（四卷42章）；随苏晓骑乘赶路、在特利村废墟刨出幸存者、循塔罗牌气味助苏晓追查卡尔、嗅出夜袭部队总部（四卷42–44章）。在帝都须收成核心藏匿（其形似芬里尔、恐被艾斯德斯认出·且不在帝国 48 帝具记录内）。**苏晓 vs 卡尔一役，布布躲在暗处偷偷以「冰雪女神光环」+「忠心护主」为苏晓回血**——这是苏晓扛住卡尔自爆、最终反杀的关键（「因为我带了条狗」）（四卷46章）。在皇宫地下的 [比鲁马克炼金试验场](../世界/地点/index.md) 中，布布**不免疫炼金毒素**（每秒 -1 真伤·但靠自身光环可自愈），作为苏晓的**坐骑**载着他与小皇帝逃过蓝色小鸟群、聚合虫；对凶悍的地穴人看守也敢引走缠斗，并几次凶相毕露「提醒」不老实的小皇帝（四卷59–63章）。
- **寄生兽空间战场（五卷·随苏晓出战）**：在裁决者身份下被乐园免费**伪装成漆黑的「地狱三头犬」**（颈侧多生两头·仅契约者视角可见·能力不变），驮苏晓巡战场、拦逃兵、深入废土设世界坐标。受命去诱**变种马来鳄**群时连嘲讽带尿都引不动、反被鳄群追得「憋着半泡尿狂奔」（一度被吓尿·心里苦）；首领巨鳄偷袭时想助战被苏晓喝退（战斗力弱·留在「冰雪女神光环」覆盖范围内默默替主回血），决战迪卡时也帮苏晓挨了两拳、半死续命（五卷4–23章）。随核心带回乐园后与苏晓一同被绿光治愈。
- **世界争夺战·被委以重任（十二卷3章·[维斯星球·多亚古大陆](../世界/任务世界/维斯星球-多亚古大陆.md)）**：进二阶世界争夺战后，它**竟被编入 50 人的敏捷类契约者「斥候小队」、负责寻找战争任务核心「世界之核」**（在帝国首都圣龙城·与在黑土平原前线的苏晓分隔两地）——这「重任」让苏晓大跌眼镜。另：它已自悟隐匿技能「**你看不到我**」（按地形伪装成柯基 / 沙皮狗 / 胖橘猫等·现 864 种），并沉迷一款能连现实网络的手游（战绩 0-10-0·被苏晓嫌「好菜」）。
- **「世界之核」抢夺战 · vs 美猫贝斯妮（十二卷11–12章）**：「世界之核」（指甲盖大的亮晶晶石头）**无法放进任何储存空间、只能随身携带**——布布汪靠「你看不到我」神出鬼没地从天启乐园契约者手里**偷走世界之核**（一度让乐园公告在「己方得核 / 敌方夺回」间反复横跳·把苏晓看懵、生怕它「一激动把核啃了」）。它的死对头是天启乐园的侦查美猫 **贝斯妮**（会人言的布偶猫·从布布汪口中夺回过核）——一猫一狗在圣龙城地下逃生通道里上演「争霸战」：布布汪叼着她脑袋左右抡甩、把她揍哭、夺回吞进猫肚的世界之核、还顺手「拐走」这只美猫。它**只肯把世界之核交给苏晓一人**（不信同阵营的斥候）。脸上那几道猫抓痕便是「战功」。**十二卷44章**它风餐露宿几天、瘦了一圈，终于带着**世界之核**回到黑土要塞交给苏晓——还顺手把死对头美猫 [贝斯妮](贝斯妮.md) 又叼了回来（一路被它含着甩、贝斯妮趁机往它嘴里吐口水报复·得知真相后它狂干呕），交苏晓处置（贝斯妮随后被苏晓割喉杀死）。
- **专业斥候 · 新技能「汪星人的秘密窥视」（十二卷62–63章）**：抢核归来它自悟新技能「**汪星人的秘密窥视**」（被动·30 米外目视敌人时减 80% 被感知几率·一旦被敌察觉视线即刻转入「你看不到我」状态）——与「你看不到我」配合堪称无耻的侦查 combo（窥视 → 被发现 → 同化消失 → 再窥视·把敌人折腾够呛）。契约者大乱斗中它替苏晓**当斥候**先去侦查 [兰爵](兰爵.md) 的约战地，用文字通讯器回报敌情（其间还不忘嚷嚷「我想尿尿～」被苏晓喝令「憋着」）。它战斗力虽弱，却是「业余奶妈 + 专业斥候」。
- **接管修斯 + 界断线配合斩兰爵（十二卷65–66章）**：决战 [兰爵](兰爵.md) 时苏晓**把 [怨之孤儿·修斯](怨之孤儿·修斯.md) 的控制权转交给它**（苏晓要专心近战、无法一心二用），它趴草丛凝神操控三米高的修斯（刚接手时下意识让修斯伸舌四肢着地、被苏晓踹起来、还盯着「鸡腿」〔肌肉弯刀〕看半天）。决胜一招全靠默契——苏晓抬手把界断线钉在修斯背后，它立刻领会、控制修斯**绕兰爵缓步走**令界断线缠住兰爵腰，给苏晓「力 72 对 18 扯倒兰爵」的机会（修斯最终被兰爵的甲虫炸碎头颅、召唤随之消散）。
- **护主疗伤 + 探路屠奶妈（十二卷66–67章）**：苏晓斩兰爵后身负重毒重伤，它**冰雪女神光环**降三分之一生命流失、再用「忠心护主」**奶回苏晓 60% 生命**，又提前找好山洞（揍跑里头那只「棕熊老哥」）、护苏晓疗伤一小时。重返战场后继续探路、专找落单的天启治疗 / 辅助类契约者给苏晓割喉。
- **学「气息追踪」（十四卷14章·恶魔古堡）**：苏晓花一堆灵魂结晶向囚徒 [莎](莎.md) 学「直感」时，布布汪**只用一些零食**就跟莎学了「**气息追踪**」（被动 Lv.3·锁定已知气息追踪、极限 5 公里·3 小时内未追到 / 超 5 公里则丢失）——气息比气味更难隐藏（除非像苏晓那样会控制气息），让它的追踪本领更上一层。苏晓不禁感慨「人与狗的差别为何如此之大」。
- **吞毒龙肉 · 属性大涨（十四卷16–18章·美食天国）**：第二场「美食天国」吃毒厄之龙肉时，苏晓把大半块毒龙肉切给它代吃（喂了三瓶 1 号药剂压着）——它「高级生物」技能触发、**吞下远高它阶位的「超高等生物（龙）」血肉、属性大幅提升**（吃完还嫌不够、惦记「下次见巨龙该油焖还是炭烤」）。这次它的提升幅度远超苏晓本人——消化后实测：**力 26→37 / 敏 32→44 / 体 30→50 / 智 27→37 / 魅 53→75**（仅吃约 200~300g 龙肉、总涨 75 点·近乎苏晓三个衍生世界的成长·若放开肚皮吃恐五维全 80）。吞噬高等生物血肉是它（无法获属性点 / 无法乐园强化）唯一的提升途径；连「忠心护主」也升到 Lv.3（回 66% 生命·冷却 20h）。其 **75 点魅力**是苏晓 8 点的近 10 倍（苏晓郁闷「没有对比就没有伤害」）。苏晓打算返园后教它近战 / 控制炼金生物（它惦记着哪天见到巨龙该怎么吃）。
- **石柱传方位破琉璃 C 战略（十二卷70章）**：[琉璃](银发少女.md) 五人杀人小队用「迷幻之境」把苏晓全感官迷惑、巨锤几乎砸破反击盾的危急关头，它趴在远处石柱上、把敌人**方位信息**悄悄传给被迷的苏晓（苏晓据此「等还差 230 米」精准蓄势），助苏晓一枪打飞琉璃、翻盘团灭整队——它则缩回头不再露面。退守石桥阶段又靠探路掩护苏晓避开上百天启契约者、冲回己方阵营。
- **乐园停留趣闻（五卷25–28章）**：精力旺盛、把 15 币的沙发啃穿藏身其中；苏晓带它去职工者宠物口粮店买烤骨头（**乐园有召唤兽口粮店**——召唤系职业若魅力低、召唤兽太饿会「噬主」，故需备口粮），女职工者一句「给它绝育治啃家具」吓得布布夹腿小碎步逃。布布魅力高达 10、兽语可与苏晓部分交流（苏晓兽语 Lv.6）。
- **二阶生存试炼·吞噬岛（八卷13–17章）**：背着蜘蛛女皇跳伞，空中遇五米钢羽黑鹰时**学起鹰叫「交涉」、竟与黑鹰打成一片**（化敌为「同伴」）；落地倒栽葱挂树被苏晓刀芒救下后啃树泄愤；红土区饿到「吃草也不错」。最惊人的是——**它耳濡目染学会了用蜘蛛女皇开枪**（一枪打爆偷袭苏晓的红皮土著、却把苏晓吓得以为遭敌方狙击手；这把重狙后坐力让它生命 −15%、胸口白毛渗血）。「布布汪学会用枪、已经不是拖鞋可以制裁的了」。八卷31章奉命「打入敌阵突围」、与苏晓走散后在红土区里**茫然乱窜、掉进沼泽险被吞**、沾成一身灰泥，直到八卷35章苏晓敲骨为号才**伏地鬼祟地赶来会合**（领教了「没有主人在身边的红土区」有多恐怖），被苏晓拎去山泉洗澡、饱餐一锅蘑菇蔬菜肉汤。
- **吞噬高等生物进化（八卷45章·「高级生物」技能首次见效）**：此前布布汪吃红土区野兽的血肉都「无效」（且挑食·吃生肉如吃药），直到苏晓逼它（脱鞋威胁 + 它含泪「实力演技派」狂吃）**啃下半只精英刺蝎**——「高级生物」技能终于触发：**力 +2 / 敏 +1 / 体 +3 / 魅 +2·「全速冲锋」微量加强**（同类生物只生效一次）。**意义在技能可被强化**——日后若能强化「冰雪女神光环」，它就从「弱鸡奶妈」晋升「业余奶妈」。「高等生物」定义模糊，此后苏晓杀的生物都让它尝尝（懵懂的布布汪还不知今后要吃多少奇怪东西）。

### 进化为真生命「从者」（六卷79章 · 第三生命炼成法阵）
苏晓在交易市场用拍卖牛牛果实换来暗紫色消耗品「**第三生命炼成法阵**」（传奇炼金师卡恩之作·可把机械 / 能量生物**重塑为真正的生命体**并提升其能力）。布布特尼一直渴望摆脱「收起来就是个玻璃球」的帝具身份、想真切感受主人摸头的体温——它一口咬碎法阵，化作一颗三米高的灰茧，**6 小时后破茧而出**：外表毫无变化、二货意识依旧，却已是**有血有肉的高等生命体**（阶位远超魔兽·从此具备成长潜力·味觉 / 触觉完整）。返回现实世界时，作为苏晓的「从者」**也一并回到现实**（其回血光环在没有恢复品的现实弥足珍贵）。

- **进化后属性（从者）**：生命 100% ｜ 法力 **270** ｜ 力 23→**24** / 敏 27→**29** / 体 25 / 智 27 / **魅 43**（苏晓裸魅的 14 倍·堪称「犬类最帅二货 / 妹子杀手」·但魅力不影响主人苏晓）｜ **十卷20章吞独眼巨怪血肉「高级生物」技能再进化**：力 +1 / 敏 +2（含泪吃下三大碗·它已挑食：生的 / 恶心 / 可疑 / 太咸的都不吃）。
- **六大技能**（核心技能保留并增强）：
    - **全速冲锋**（主动）：移速 +150%·可载重·耗 12 法力/分。
    - **冰雪女神的馈赠**（光环）：每秒回友军 1 生命·无消耗·随智力提升。**十卷17章「商业头脑」初显**：杀独眼巨怪的团战中，它先免费放光环给众契约者尝甜头、再改成「**每分钟 10 乐园币付费激活**」收钱（十名契约者乖乖付费），还自拟临时契约**分给苏晓六成**——其智商 / 鬼点子被一群勾心斗角的契约者彻底激活。
    - **忠心护主**（主动）：耗 50% 法力、3 秒回主人 **60%** 生命·冷却 24 小时。
    - **高级生物**（主动）：可**吞食高等生物的血肉进化**（成长潜力）。
    - **忠心耿耿**（被动）：忠诚度锁定至最高·任何情况无背叛可能。
    - **你看不见我**（被动·自悟）：可依任何地形隐藏（已掌握 678 种·曾把整间专属房间啃成废墟后蹲吊灯上让苏晓遍寻不着）。

## 智取迪达拉 · 「本汪到此一游」（十五卷55–56章）

竖岩村救援战里它立了头功——苏晓在地下监狱正面缠住三代土影 [大野木](大野木.md)（用身体替它趟出安全路径），布布汪靠「你看不到我」隐匿神技摸进牢房，**趁大野木放尘盾时把昏沉的 [迪达拉](迪达拉.md) 救走、还在地上刻下「本汪到此一游」**挑衅，把大野木气到血压飙升（人被劫走、怎么劫走的都没看清）。撤退路上它一边狂奔一边回头看追兵、撞岩石 5 次撞山壁 8 次堪称「老司机」。

## 忍界大战 · 接管阿姆 + 得万花筒写轮眼（十五卷90–108章）

整场忍界大战它是苏晓最大的助力——感知 + 业余奶 + 操控 [阿姆](阿姆.md) 作战（配合比阿姆本意识默契）+ 关键偷袭（咬柱间屁股·硬如卡车轮胎硌疼牙），却几乎不亲自上场。它去取轮回眼时**顺手得了带土那只万花筒写轮眼**（隐藏任务第一环奖励），返园后苏晓不要、丢给它研究（它的高等生物能力或能从中得点好处·可惜它「去恨谁」是个问题——啃不动的奇异兽骨头吗）。返园后它凭小金库自费乐园币恢复自身重伤、还帮苏晓垫付了恢复费。

## 蚁王篇 · 14 颗磁导雷玩坏女刺客（十六卷28–29章）

空间系女刺客 [迟玫](迟玫.md) 偷袭苏晓失手躲进异空间后，布布汪在她可能现身处**精确埋下 14 颗高仿「磁导雷」**（其实是炼金炸弹假货·每颗成本不到 30 币·在苏晓指点下连导角度都精算过）——迟玫一出异空间就连扎三雷、被炸断一腿一臂、左眼报废，崩溃逃走。这种「二货也能办的精算活」全靠苏晓在背后指点。

## 希亚大陆 · 智商在线 + 戏耍俘虏（十七卷14、27–28章）

进 [希亚大陆](../世界/任务世界/希亚大陆-灭法遗迹.md) 后它把「**战斗时不二、平时才二**」演到极致：杀圣战士 [约瑟·柯](约瑟·柯.md) 时，它**潜入城堡找书房、靠「律法书 + 中老年男性气味少处」判断出首席法官埃尔维的气味、5 分钟把逃跑的他绑回**（凭抽象羽毛笔画像本找不到人）。审讯死灵 [伊儿朵](伊儿朵.md) 时更献上「**布布汪的欢乐时光**」——挂着审讯架钥匙串装蠢、勾伊儿朵伸手够钥匙（永远差 2~3 公分）、半分钟一次「自由与囚笼」的大起大落，把对方意志一点点磨碎（5 小时后伊儿朵才发现它「智商不低于人类」）。它的交涉哲学：**用高魅力把剧情人物的智商拉到自己这条水平线、再公平较量**（「死亡如风、常伴汪身」）。死刑官 [敦克·尼克](敦克·尼克.md) 来袭时它接管 [阿姆](阿姆.md) 的身体、用「若隐若现的气息」恐吓敌人。它还露了手**自悟侦查技能「汪星人的秘密窥视」**（被动·**30 米外目视敌人时减 80% 被感知几率·一旦被察觉即转入「你看不到我」状态**）——靠它潜进布瓦村、十分钟定位伪装农夫的幕后主使 [鲁珀](鲁珀.md)。

## 德岛海战 · 全能搅屎汪（十八卷·海贼世界）

「你看不到我」被动在海贼世界发挥到极致——它能融入环境贴到任何人身边而不被察觉（连卡塔库栗都只能靠见闻色「预知」、连感知系奶妈小馒头都察觉不到它）：替苏晓**热感应定位藏匿的克力架**、潜入大妈首都制造骚乱、给斯佩罗收尾。海下更是「电动小马达」（两条后腿涡轮增压·白毛被女海军误认成海王类血统的鲨鱼），帮苏晓炸沉押送军舰、海下脱身。脱身后解锁**「狗拉皮艇 Lv.MAX」**（拉着苏晓的橡皮艇游向香波地·被许诺「到了带它吃大餐」）。**招牌绝活「狗爪手枪 biu」**：抬起肉嘟嘟的狗爪摆成手枪、口中「biu」一声，把 21 区想杀它的变态契约者**蝎子整个上半身轰碎**（暴力美学·「来生见到人类要记得跑」）——子弹能连穿三层墙、几乎团灭那一整桌四阶契约者小队（独留奶妈 [小馒头](小馒头.md)）。**偷罗杰佩枪**：21 区火药酒屋几十名独狼僵持争夺 [罗杰的佩枪](../力量体系/装备道具.md)（值 10000 海贼声望）时，布布汪潜进酒屋、就蹲在佩枪旁哈气（无一人感知到它·「差点唱无敌是多么寂寞」）——苏晓在 1200 米外狙击制造混乱、它趁乱**叼走佩枪并嫁祸给 [岚佐](岚佐.md) 背锅**。唯一翻车：海里玩太嗨、把苏晓托它保管的多弗电话虫泡进了海水（差点被苏晓烤成焗蜗牛、所幸救活）。

## 海蓝星 · 冰原狼王（十九卷4、25章·世界争夺战）

第二场 [世界争夺战](../世界/任务世界/海蓝星-世界争夺战.md) 中，布布汪又被分进侦查队、被传送到一片冰原、混入一群战斗力爆表的**冰原狼群**（白毛与狼相似·它却「稳如老狗、心中慌得一匹」）。靠苏晓派 [阿姆](阿姆.md) 来帮衬，它在狼群里**拉帮结伙、势力位居第二**，最后**一声「汪」就让上千头冰原狼集体低头、成了狼王**——把另一支被上百冰原狼围困的轮回乐园侦查小队（含苏晓熟人**水咲**）看得目瞪口呆（「这是……成了狼王了？？」）。它**靠智慧 + 权谋 + 75 魅力 + 模样相似**统治狼群（并非武力·救了重伤被弃的疤眼冰原狼 + 食物 + 给它忽悠来漂亮母狼·再借疤眼的武力强拐打手·头狼最终被围攻致死）。

**安贝拉城刺杀建功（十九卷42–48章）**：当上狼王后它**带四只冰原狼小弟火速与苏晓会合**（阿姆留守冰原），疤眼最强（力 / 敏 / 体 85~90·90 敏真实·首领级 BOSS）。它潜进神庭分部、还**和任务目标杜波·依斯一起吃了顿早餐都没被察觉**——交出「神庭分部详细草图 + 杜波·依斯日常作息」（连他是左撇子、上几次厕所都记），把小助手「连续打脸」。袭杀杜波那晚，它在后厨炸炼金炸弹制造混乱、给小助手递煤油罐；事后还远程一棍敲巴克的头 + 派冰原狼拦住他，**救下被追杀的小助手**（让疤眼驮她遁走）。

## 热雨林之森 · 咸鱼翻身（十九卷69–72章）

进了六星极地 [热雨林之森](../世界/地点/index.md) 后，布布汪迎来「咸鱼翻身」之地——几十小时里靠「高级生物」能力**吞食了十六种极地生物的血肉**（苏晓只取血肉、不与极地生物拼命，灵感来自记仇死缠的丛林猫「狩之影」）。属性大幅暴涨、更**魅力达 80 临界点触发「转真实属性」**：作为「从者」（仆从类最高阶位·高于召唤物的第三阶位），它可借苏晓的四种真实属性、不经考核选两种转真实——默认转**魅力**、自选转**敏捷**（「不够快怎么浪起来」）。

- **现属性**：从者·撕空恶犬·法力 400·力 69 / **敏 82（真实）** / 体 72 / 智 40 / **魅 86（真实）**。**8 种能力达上限**（不能再新增·只能进化 / 替换 / 晋升）：冰雪女神的馈赠 **Lv.5**（每秒回 3 生命 + **每分钟回 5 法力**·真实魅力加成·对缺蓝的苏晓很重要）、「忠心护主」晋升为 **生命之力 Lv.30**（远程回 50 + 魅 ×4 生命·冷却仅 5 分钟·从「业余奶」升「三流奶」）、高级生物 MAX、忠心耿耿 MAX、你看不到我（928 种隐藏）、汪星人的秘密窥视、气息追踪 Lv.10、以及新自悟的「**呜嗷**」（脱离隐身长嚎扰敌心智 / 吸引仇恨·无冷却——「隐身奶 + 跳出来嚎一嗓子再隐身」恶心到敌人）。靠属性暴涨它开始在前面探路、大大加快了深入雨林的速度。

## 冰原狼群灭圣域 · 卸任狼王（十九卷79–84章）

苏晓的夺胜「大计划」由它实现——阿姆集结的 **1276 只极地冰原狼**杀进雨林，配合轮回乐园方把人数最多的圣域方（353 人）团团包围（狼群执行力不弱于士兵：分股切割、长嚎迷惑数量）。布布汪还**现身战团中心狼嚎**、把神奈引开（用「呜嗷」+「你看不到我」凭空消失耍她）。此战圣域近乎全灭，代价是冰原狼折损近半（1276 → 756）。战后它**卸任狼王**——冰原狼要回冰原延续种族（疤眼接任），它平静长嚎送别（早料到会分别）。随后它与记仇死缠的极地首领 [狩之影](狩之影.md) 丛林猫缠斗（凭 82 真实敏 + 恢复 + 隐匿能与它打成「绒毛横飞」、靠苏晓寂灭公爵助阵）；丛林猫被收服后，它和阿姆负责用「布姆组合式陀螺暴风旋转」治这只新「招财猫」。
