# 血神·格赫普斯（暗狱世界最终boss·被复活的古神）

近四米高、通体外骨骼包裹的古神——胸膛是肋骨状外骨骼、后背一根根近一米长骨刺、双眼漆黑瞳焰血红。它是**魂**耗费几十年、以「世界之核 + 暗狱 + 本世界」三重融合为局，借豪鬼躯体 + 道格拉斯 / 鬣狐·亚斯灵魂为养分**复活的古神**，坐在黑色树根盘结的神座上重新接管世界。它曾是**陨灭星**的古神，畏惧险些让它彻底消亡的**冥神**，暗狱正是它初降临时濒死、无节制「吮吸」世界所致（同 [月神](../月神大陆配角.md) 疑同源·巴哈屡拿月神死状嘲讽它）。

## 能力（使徒之眼侦得 31.7% 资料）

- **六维**：力 190 / 敏 190 / 体 195 / 智 ?? / 魅 1（真实属性·全面压制苏晓）；神灵能量（古）63210/69000。
- **高位神灵（被动 Lv.78）**：生命 +98500、免疫控制、每秒回 0.5% 生命。
- **世界之核·阵界（无敌防御）**：吞世界之核后周身生成「**阵界**」——如鲁珀特之泪，本体无敌、须破十公里外的「子体结构」才整体崩碎（须世界之子配合破坏，否则受世界排斥）。
- **蚀血·暗（被动 Lv.70）**：把任何鲜血转化为「蚀之血」、任意塑形（尖刺 / 弓箭 / 血兽 / 汲血恶魔刃枪）——每击 2760 黑暗侵蚀 + 3275 灵魂伤害 + 基础伤害、附灵魂麻痹（对灵魂 >300 者无效），敌血 <65% 时 10% 即死。
- **魂·驭血（主动 Lv.70）**：900 米内所有单位体内鲜血持续燃烧（每秒 176~350 燃血伤害·无差别）。
- **血共鸣（主动 Lv.72）**：无判定、无视豁免、瞬发**强制击退**所有体内有血的敌人（冷却 3 秒·耗 5000 古神能量）——克近战的杀招。
- **汲血恶魔（血枪）**：血神的神血 + 初始神力所铸圣灵级刃枪（评分 1000+·血之渴望每秒 −50 生命换敏体、攻击回 82% 生命、蚀 6.5% 即死）——即死本质是**灵魂斩杀**。
- **命种·不断复生**：吞噬的众多灵魂困在体内激活「命种」，正常击杀会复生几十次；须在死亡瞬间**斩尽命种**才能彻底杀死。

## 经历（38卷27–52章）

- **复生**（27–28章）：魂当场吞豪鬼 / 道格拉斯 / 亚斯，血神以豪鬼躯体复生、吞世界之核成阵界无敌体；道格拉斯五十米雷电柱同归于尽亦被弹开、塔·巴尔德一击掉 81% 血。
- **决战**（48–52章）：苏晓借守望公会「以命填门」+ 香夫人（人造世界之子）持黑暗处刑破掉阵界子体结构后，单挑血神——三道「刃道刀·流 / 血刃」斩痕叠于一点破外骨骼防御 + 阿波罗自爆炸空胸膛 + 青影王一镰、**灵魂之刃豁免一次汲血恶魔即死**。追杀到晨曦市（血神想去屠 370 万人口的城转化蚀血回血），最终一刀**斩尽命种**、彻底陨落。
- **掉落**：5800 世界声望 + 圣灵级宝箱·豪（97%）+ 1025 灵魂钱币奖励列表（[神灵骨](../../力量体系/装备道具.md) / [熔炼炉](../../力量体系/装备道具.md) / [汲血恶魔](../../力量体系/装备道具.md) / 血之意愿·不灭）；[神裁戒](../../力量体系/装备道具.md) 成长至 62%。
