# 圣鸳（魅力系御姐 · 撞上「库库林免疫魅力」的铁壁）

前凸后翘、彬彬有礼的**圣光乐园**魅力系御姐——[莫雷](../原始大陆/莫雷.md) 那队契约者的**交涉担当**（与剧情人物交涉自带 1 被动 + 1 称号 + 2 装备的初始好感 buff）。

- **魅力全废**：她想靠魅力特性从「庇护屋管理人」[苏晓](../苏晓.md)（伪装的霸主级「神灵猎人·库库林」）身上捞好处——却撞上系统铁壁：「与库库林交涉时**所有魅力特性无效**、精神干涉 / 误导均判定为敌对」，激活魅力能力反而差点被归为「精神误导」。她硬着头皮正常侦测，因**魅力是库库林的 860 倍**只探出基础资料（霸主级·力 181 / 敏 181 / 体 180 / 智 180·法力 27900），当场石化——只好拉全队自发大扫除、拿实际行动挣回守望公会声望。她被「物理推选」出来当交涉代表（「关门，放圣鸳」）。

## 关系

- [苏晓](../苏晓.md)（神灵猎人·库库林）：她魅力全废、只能实打实办事的庇护屋管理人。
- [莫雷](../原始大陆/莫雷.md)：同队的战斗天使。

## 出处

- 三十八卷 3–4 章：投庇护屋→魅力交涉碰壁→大扫除挣声望。
