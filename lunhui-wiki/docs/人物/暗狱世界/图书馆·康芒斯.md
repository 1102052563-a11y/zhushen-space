# 图书馆·康芒斯（守望公会四高层·219岁活体图书馆）

已活 **219 岁**、仿佛随时咽气、全身插满透明导管、座椅后连着维生器械的老头——守望公会四高层（守护者）之一。他的大脑无价：**守望公会所有机密、战斗知识、武器结构图纸，乃至 1400 余年的超凡事件历史，都记在他脑中**，是活着的图书馆——只要他呼吸或心跳停止，头颅会**瞬间炸开**（防知识落入敌手）。

## 性格 · 能力

- **活体知识库**：什么都知道，但「你不问，我很难回忆起——我是图书馆，不是先知」；偶发老年痴呆（中午非说该吃早餐），却总在关键时刻掏出解法。
- 也是最先说破「世界之子」概念的人之一（让布布汪怀疑他是契约者伪装）。

## 经历（38卷33–46章）

- **破暗狱之门献策**：单凭猎人破门要死 300~1000 人——他改献「让死灵·尤尤尔等擅长破除超凡机关的凶徒去」的思路。
- **人造世界之子的关键**（45章）：从脑中翻出一个已灭亡民间组织留下的「**魂运剥离仪式**」——虽仪式本身行不通，却启发百人学者团改良成「共生 + 移植」方案，最终造出人造世界之子香夫人。
- **结局**：蝮蛇带他撤离决战现场；世界通关后与香夫人、蝮蛇一同收拾守望公会残局。

## 关系

- [苏晓](../苏晓.md)（神灵猎人·库库林）：屠神计划的高层配合者、破门与造世界之子的献策者。
- [香夫人](香夫人.md) / [蝮蛇](蝮蛇.md)：守望公会同僚高层（残局仅存三人）。
