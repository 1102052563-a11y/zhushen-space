# 南瓜·塔·巴尔德（守望公会三神灵猎人之一·缺心眼却强大的实干派）

臃肿全身甲、戴南瓜形头盔、持一米五螺旋剑（当锥枪用、能钻穿暗魔）的猎人——守望公会**三神灵猎人之一**（另两位 = 苏晓伪装的「魔刃·库库林·白夜」、叛徒「鬣狐·亚斯」）、脖挂乳白猎人牌。集**强大与缺心眼于一身**：豪爽、坦诚（「在你用刀刺穿我胸膛之前，我都会相信你」）、爱用吸管隔着头盔喝酒。

- **迷路的实干派**：他一直满世界巡查暗魔 / 幽魂到底想干什么（是暗狱阴谋的最早察觉者）。奉公会命来与 [苏晓](../苏晓.md)（白夜）会合——为了快，他不坐火车、**顺着铁轨跑**（比火车快），却一路跑迷路、冲出东谷联邦跑进北方极寒地、靠冰裔送回。会合后送来「黄昏遗物」、组临时队，还抢着「作为迟到的惩罚」独自捶爆巨型暗魔（南瓜头盔垂下就打鼾）。
- **屠神战 · 战死**（38卷47–50章）：屠神时他带破阵队、护 [香夫人](香夫人.md)（人造世界之子）杀入暗狱最深处大殿——独自斩杀数百守殿暗魔（含被古神操控的老友 [鬣狐·亚斯](鬣狐·亚斯.md)、听其「别把暗魔杀绝、但一定要杀恶神」遗言）身负重伤，护香夫人破掉阵界子体结构后，**举着珍藏的存酿在南瓜头盔下静静战死**（「香夫人，我们干杯」·头盔下悄然渗血·像每次思考着睡去一样）。苏晓评价「巴尔德完成了他想成就之事」，把神灵猎人身份牌交还香夫人。

## 关系

- [苏晓](../苏晓.md)（魔刃·库库林·白夜）：同为神灵猎人、组临时队的可靠队友。
- **鬣狐·亚斯**：另一名神灵猎人（叛徒·未露面）；他与苏晓推断亚斯才是内应。
