# 魂（暗狱三巨头之一·古神信徒·复活血神的幕后）

身穿黑袍、看不清容貌、**存在感极低**（目光一移开就会忽略它）的存在——暗狱三巨头（[豪鬼](豪鬼.md) / 魂 / [道格拉斯](警探·道格拉斯.md)）之一，也是整场阴谋**真正的主谋**。它出身低贱，是「神灵」（一个超脱特性世界来的古神）改变了它的命运、给了它力量与资格，让它得以与豪鬼、道格拉斯这种大人物合作——**乃至利用他们**。

## 阴谋（38卷27–28章揭晓）

魂花几十年布下**血神复生局**：① 找到本世界的**世界之核**、用神灵留下的知识让世界之核与本世界融合；② 神灵临死用半死躯体温养出「**融合之根源**」、让**暗狱**也加入融合——三者互相干涉、融合极慢，为它争取时间收集世界之力与强者遗骨。复活古神的五条件：**融合之根源（寄存物）+ 世界之力（不被排斥）+ 强大灵魂作火种（原定道格拉斯·后改亚斯）+ 强大躯体（豪鬼）+ 吞噬大量生命力**。它故意操控融合之根源侵蚀老猎人·亚当，就是为把灵魂远强于道格拉斯的**苏晓**引来当更好的火种。

## 能力

- 幽魂本源 + 古神加持：袖口涌黑红色触手、可操控融合之根源的黑色流体；复活血神后转化为「古神信徒」（肉体被构建、灵魂异变、双眼漆黑）、实力再升。
- 弱点：终究是幽魂——被 [阿姆](../阿姆.md)（龙心斧 + 源之晶石 + 寒冰盾生命回震）**开战 30 秒打到自闭**，连跑都跑不了。

## 经历

- **主导复生**（27–28章）：当着苏晓的面吞掉豪鬼 / 道格拉斯 / 亚斯，唤醒血神·[格赫普斯](血神·格赫普斯.md)；却被古神「命血·聚合」无差别洗地波及（它想搞神教的美梦破灭·心里暗骂古神漠视生灵）。
- **殁**（54章）：与阿姆缠斗中，血神被苏晓斩尽命种、古神之力失控，魂**原地自爆**、惨死阿姆眼前——**未留任何击杀奖励**（阿姆很郁闷）。

## 关系

- [血神·格赫普斯](血神·格赫普斯.md)：它信奉、耗几十年复活、却被漠视利用的古神。
- [豪鬼](豪鬼.md) / [警探·道格拉斯](警探·道格拉斯.md) / [鬣狐·亚斯](鬣狐·亚斯.md)：被它当复生养分的暗狱同伙。
- [苏晓](../苏晓.md)：它想夺其强大灵魂当火种、却成为埋葬它整个计划的人。

## 出处
