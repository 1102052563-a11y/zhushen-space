# 香夫人（守望公会四高层·毒体外交官·人造世界之子）

年过三十、女性魅力完全熟透的金发贵妇——守望公会四名高层（守护者）之一，负责**外交与补给药剂调配**，助手是恶名昭著的**鸟嘴医生**。她本是世袭公爵的未婚妻，因灵魂特殊、会散发出**只有暗魔与幽魂能闻到的异香**，导致全家与未婚夫尽数惨死（未婚夫为护她而死）；众人本称她「公爵夫人」，因她厌恶这称呼（勾起亡夫）改称「香夫人」。

## 性格 · 能力

- **剧毒之身**：日常不经意碰到都可能中毒，喷的香水分昏睡型 / 致死型（心情差就是致死型·沁入怀中深吸即死）——曾有失控老猎人绑架她图谋不轨、半路被毒死；她在等某天被暗魔吞食时**毒死那暗魔**。
- **手段狠辣的理想主义者**：年轻时也心怀正义，后来明白「想对付怪物，只有比怪物更狠、更卑鄙」——正是她拍板从各监狱调 7 名重刑凶徒去破暗狱之门（逼莱茵王国嫡女阿蜜雅押送）。记仇（被苏晓「你衣服上有毒」嫌弃后一直笑里藏刀）、怕痛（植入气运之石时割破手背就喊疼）。

## 经历（38卷33–54章）

- **对接屠神计划**：为苏晓（库库林）筹集炼金材料（借布布汪掩人耳目·让他背锅调药）、张罗黑暗蔷薇药剂发放。
- **成人造世界之子**（45–46章）：图书馆·康芒斯的「魂运剥离仪式」改良成「共生 + 移植」——把道格拉斯共生幽魂封入蓝黑宝石移植体内，独眼男 / 戈依次试都爆体，**香夫人成功**（能模糊感应到道格拉斯的金色气息）、学巴哈对蝮蛇竖中指。
- **破阵界子体结构**（47–50章）：塔·巴尔德护她深入暗狱最深处大殿，她持老猎人亚当的锯刃弯刀 [黑暗处刑](../../力量体系/装备道具.md) 拍碎阵界子体结构（原本拿不动的沉重武器·此时轻若无物·似有两股意识托起）——转身却见塔·巴尔德在南瓜头盔下悄然战死，她坐下陪他「干杯」。
- **结局**：世界通关后，守望公会四高层折了戈 / 金瑟、三神灵猎人只剩战死的塔·巴尔德，仅存香夫人、蝮蛇、康芒斯收拾残局。苏晓返园前把神灵猎人身份牌交还给她。

## 关系

- [苏晓](../苏晓.md)（神灵猎人·库库林）：并肩屠神的核心搭档（互相「嫌弃」却默契）。
- [蝮蛇](蝮蛇.md) / [图书馆·康芒斯](图书馆·康芒斯.md)：守望公会同僚高层（残局仅存三人）。
- [警探·道格拉斯](警探·道格拉斯.md)：她移植其共生幽魂、成为与他相连的「人造世界之子」。
- 塔·巴尔德：护送她破阵、在她面前战死的神灵猎人。
