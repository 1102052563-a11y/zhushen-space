# 蝮蛇（守望公会四高层·人员调动·腹黑冷血男）

气息宛如眼镜蛇、给人阴冷感的中年男——守望公会四高层（守护者）之一，负责**人员调动与情报**，没人知道他真名，都叫他「蝮蛇」。他只是普通人，却理解力惊人（巴哈模糊描述「受世界青睐之人」，他几乎当场说破「世界之子」概念，让巴哈怀疑他是契约者伪装的）。有个瘆人癖好：思索时把自己的**尾指一点点掰断**（还面无表情掰回去、掰断第二次）。

## 性格 · 能力

- **腹黑冷血**：下令「除白夜、巴尔德、香夫人外，未经批准不得出总部，违规者死」时眼神瞬间变成冷血动物；但欣赏世界之子候选人亚莉克希亚，也敢自请带队去破阵界子体结构送死（「连我这下令的都怕了，猎人们没理由去送死」）。
- 组织力强：用「富商幸运庆典」（100 万席勒选运气最好的人）+ 莱茵 / 尼耶尔公国情报渠道多线寻找世界之子。

## 经历（38卷33–54章）

- **寻世界之子**：拿出候选人亚莉克希亚（已被师父亚当所杀·殁），改办富商幸运庆典找幸运儿——被阿德利袭击时**丢了一条手臂、险被暗魔咬掉脑袋**、扶持的富商惨死、庆典会场被炸。
- **决战后**：带图书馆·康芒斯撤离；世界通关后与香夫人、康芒斯一同收拾守望公会残局。苏晓来杀道格拉斯时，他点破「你运气也不好，或许负负得正」——反倒启发苏晓意识到自己忽高忽低的运气是杀多了世界之子所致。

## 关系

- [香夫人](香夫人.md) / [图书馆·康芒斯](图书馆·康芒斯.md)：守望公会同僚高层（残局仅存三人）。
- [警探·道格拉斯](警探·道格拉斯.md)：他判定其为「半个世界之子」、主张不能带其入暗狱。
