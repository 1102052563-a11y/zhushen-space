# 警探·道格拉斯（索多市警探 · 超凡事件对接人）

蓝黑制服的中年警探——莱茵王国索多市的警官，负责把辖区里的**超凡事件第一时间转交给庇护屋**（王国把平民与超凡分割：警官接手后通知守望公会派猎人）。

- 打通「庇护屋（十六）」的电话、对接管理人 [苏晓](../苏晓.md)（神灵猎人·库库林）——嘴上说「希望永远都不用打给你」、心里清楚这不可能。他多年验证出一个结论：**「猎人很可怕，可猎人有多可怕，就有多可靠」**。刚对接没多久就又急着来电（银行被盗、失窃 210 万席勒等接踵而至的超凡 / 疑似超凡事件）。

## 关系

- [苏晓](../苏晓.md)（神灵猎人·库库林）：把超凡事件转来的对接警探。
