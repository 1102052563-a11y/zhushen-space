# 警探·道格拉斯（索多市警探 · 超凡事件对接人）

蓝黑制服的中年警探——莱茵王国索多市的警官，负责把辖区里的**超凡事件第一时间转交给庇护屋**（王国把平民与超凡分割：警官接手后通知守望公会派猎人）。

- **明面**：打通「庇护屋（十六）」电话、对接管理人 [苏晓](../苏晓.md)（神灵猎人·库库林）——嘴上说「希望永远都不用打给你」、心里清楚不可能。有个怪癖：每次点两支烟（只点一支就觉得「像和某个人共吸一支」很别扭·伏笔）。
- **真身 = 暗界的主导者（重大反转 · 三十八卷13章）**：他其实是给「**暗界**」（暗狱吞世界后诞生的新世界）**命名、并封印自我融入人类社会**的存在——是唯一认可暗界、把苏晓的「白夜」化名从守望公会机密里透给豪鬼的「那个人」。[豪鬼](豪鬼.md) 因「毁灭」克不住苏晓「锋利」，把他从青蓝烟球里**唤醒**、让他想起真名——他当场发作（青色瞳孔·牛喘）、随即杀了看着他长大的女警官**维娜**。

- **亲取遗物（三十八卷21–23章）**：拿到了雾都遗失多年的**第三枚神灵猎人遗物**；亲赴晨曦市地库找**守望遗物**（连杀十几人·「莱茵也没落了」）——正与苏晓「带敌人找遗物」的布布汪撞上。

## 关系

- [苏晓](../苏晓.md)（神灵猎人·库库林）：明面对接、暗里要联手拖住 / 除掉的目标。
- [豪鬼](豪鬼.md)：唤醒他的四阶段暗魔（豪鬼在暗狱时只管战斗、动脑的事靠道格拉斯）。
