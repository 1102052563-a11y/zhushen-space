# 豪鬼（四阶段暗魔·硬闯守望公会总部·手刃157老猎人）

满头狂乱红发、尖爪、胸膛两道剑疤、爱痛饮的强壮暗魔——**四阶段暗魔**（有称号 = 屠戮过太多平民的最强暗魔）。曾**硬闯守望公会总部**、重伤至半死仍全身而退，至今已有 **157 名老猎人**死在他手中。他嗜战、把苏晓（神灵猎人·库库林）当「老友」（只有跟苏晓等寥寥数人厮杀才找得到感觉）。[苏晓](../苏晓.md) 判定他**比上个世界的泰密莎还强一筹**。

- **技法型暗魔**：看似豪放、实为技法型（近战宗师 Lv.50+）——毁灭·斗士（拳砸地生黑刺）、毁灭·震界王（黑红能量灌右臂）。在女猎人 [蒂法尼](蒂法尼.md) 的占卜屋抢先来杀她、与苏晓激战正酣时，**被两条缠着黑色触手的能量手臂强行拖回了暗狱**——说明他背后有势力操控（且能「危险预知」般提前出手），他自己并非老阴哔（否则不会孤身闯公会总部）。

## 关系

- [苏晓](../苏晓.md)（神灵猎人·库库林）：把他当「老友」、想痛快一战的对手。
- 未知势力：操控他抢猎人牌、危急时把他拖回暗狱的幕后黑手。

- 三十八卷 7–9 章：抢杀蒂法尼→与苏晓激战→被黑触手拖回暗狱。
