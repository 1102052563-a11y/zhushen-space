# 鬣狐·亚斯（守望公会三神灵猎人之一·被诬叛徒）

脸上遍布疤痕、额头镶满金属钉扣、声音沙哑（曾被割喉过）的猎人——守望公会最强梯队「**三神灵猎人**」之一（另两位 = 苏晓伪装的「魔刃·库库林·白夜」、缺心眼的南瓜·[塔·巴尔德](南瓜·塔·巴尔德.md)）。苏晓 + 塔·巴尔德一度推断打酱油的他就是「叛徒」（能派银发老猎人暗杀苏晓、地位高但不知太多秘密），塔·巴尔德为此心沉——他与亚斯有不浅的交情。

## 真相 · 理念（38卷27 / 50章）

亚斯确实倒向了古神势力，但动机并非贪婪，而是一套黑暗的**「猎人存续论」**：他不愿看到守望公会现役 12537 名猎人有悲惨下场，认定「**永远不要把暗魔杀绝——只有它们存在、作为人类共同的外部死敌，各王国才能团结；暗魔死绝，没有外敌，各国迟早爆发战争、死更多人**」，而「猎人们死在战斗中，才是归宿」。他被魂当作复活血神的**灵魂火种**吞噬（原定火种是道格拉斯、因被鸟嘴医生唤醒而改用亚斯）。

## 经历

- **登场 · 被吞**（27–28章）：现身暗狱黑山、站在豪鬼身旁；魂反手用黑色触手把他吞为古神火种（他没挣扎、笑着受死，令魂心生一丝不安）。
- **殁**（50章）：血神复生后曾操控被吞的亚斯守暗狱大殿——塔·巴尔德在大殿肩抵肩、螺旋剑刺穿其胸膛斩杀。亚斯遗言托付塔·巴尔德「永远别把暗魔杀绝……但一定要杀掉恶神」。

## 关系

- [南瓜·塔·巴尔德](南瓜·塔·巴尔德.md)：交情不浅的同僚神灵猎人、最终亲手斩杀他并听其遗言。
- [苏晓](../苏晓.md)（神灵猎人·库库林）：与塔·巴尔德一同推断他是叛徒、决定先斩后奏。
- [魂](魂.md)：把他吞为血神复生火种的古神信徒。

## 出处

- 三十八卷 21–50 章：被推断为叛徒→暗狱被魂吞为火种→被塔·巴尔德斩杀留遗言。
