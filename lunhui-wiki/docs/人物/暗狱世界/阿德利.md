# 阿德利（圣堂冒险团团长·天启乐园·背刺者）

天启乐园「**圣堂冒险团**」（165 人大型冒险团·守望公会阵营·被当精英炮灰）的团长——一身紧绷得随时要爆的西装、大背头，人称「有情有义」的阿德利，以憨式坦诚交涉著称（「交朋友要坦诚」），却是个见利忘义之徒。他与团队在守望公会阵营混得不好，一被 [道格拉斯](警探·道格拉斯.md) / [魂](魂.md) 以好处诱惑就倒戈。

## 能力（指挥型团长）

- 不亲自参战，提供**九种集体增益状态**、指挥能力极强——擅长三重领域（重力 / 水 / 岩砂）压制的围攻阵型，对付瞬间爆发型的鬣狐·亚斯很有效。
- 团队核心战力：**星妹 / 十字星**（意念子弹·眉心白十字·信念越强威力越大·最强）、奶妈 **柏柏莎**（天使降临·可造真实治疗伤害）、光头战锤男、暗杀系等。

## 经历（38卷10–40章）

- **登场投庇护屋**（10章）：率圣堂冒险团来投、被苏晓借机批量派发 31 + 153 个委托攒声望。
- **背刺**（31–32章）：受道格拉斯收买，跑路前捞最后一笔——袭击庇护屋代理管理人蒂法尼、把 32 个委托全按 E 评价乱刷（甩锅给已死的契约者情侣）。
- **围杀 · 殁**（37–40章）：靠魂给的「普通神灵猎人」假情报，率 105 人在异空间三重领域围杀伪装的苏晓——反被双宗师 + 猎魔时刻空间穿透 + **死寂降临·烬灭**（每杀一人 +28 攻的死寂烬灭枪）反屠过半（星妹自尽 / 奶妈柏柏莎「毒奶」痛击队友 / 阿德利本人「我知道怎么对付古神，别杀」话说一半被烬灭弹爆头）。苏晓由此获该团 20% 团队物资 + 42000 团队经验（破晓冒险团逼近 SSS 级）。

## 关系

- [苏晓](../苏晓.md)（神灵猎人·库库林）：被他假情报诱来围杀、反被团灭。
- [警探·道格拉斯](警探·道格拉斯.md) / [魂](魂.md)：收买 / 误导他背刺守望公会的暗狱势力。
- [莫雷](../原始大陆群像.md)：同为投庇护屋的契约者（莫雷选择正道换契约、他选择背刺送命）。

## 出处

- 三十八卷 10–40 章：投庇护屋→受收买背刺袭蒂法尼→假情报围杀苏晓被死寂烬灭爆头。
