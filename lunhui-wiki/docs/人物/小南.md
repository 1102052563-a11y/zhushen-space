# 小南

[火影忍者世界](../世界/任务世界/火影忍者.md)「**晓**」组织的成员——蓝发女子，[长门](长门.md) 最信任的心腹、与已故的「弥彦」同为长门的旧友（三人是患难与共的同伴）。

- **照料长门**：常年照看身体枯槁、靠机械维生的长门，对其日渐衰败的身体深感忧心。
- **迎接苏晓（九卷17章）**：奉长门之命，与「天道」（弥彦尸体·即长门本人代行）一同下楼**迎接新成员 [苏晓](苏晓.md)**——长门特意叮嘱她「（白夜）刀术能比肩木叶白牙、来路不明、交涉时别靠他太近」。

## 谈待遇 · 编入小队（九卷17–18章）
她代长门与 [苏晓](苏晓.md) 敲定加入事宜——苏晓干脆答应（「为保命 + 利益」），只要求晓**帮他解决雾忍暗部的追杀**（其他好处等他做出贡献再说）。她借「闲聊爆破知识」**打探**苏晓底细，被苏晓一通化学公式 + 爆破细节听懵、却由此确认他「不是说大话」。随后她发给苏晓 [空之戒](../力量体系/装备道具/index.md) 与黑底红云风衣，并**把他编入一个二人小队当第三人**（搭档在河之国·实为让那两人**监视、观察**他），临走叮嘱「不可泄露组织信息、小心你那两个搭档别被他们杀了」（晓正式编队后许多候补成员死于内斗·尤以角都 / 蝎为甚）。

## 守护重伤的长门 · 当手术助手（九卷71–75章）
[长门](长门.md) 被四阶契约者重创后命悬一线，她遍寻医疗忍者无果（只有纲手能救、又难接受），心急如焚。她奉长门之命去餐馆见 [苏晓](苏晓.md)（长门判定「白夜虽图尾兽、却比鼬 / 迪达拉可信」），拿出寄生甲虫的卵试探 → 引苏晓见长门。苏晓提出「救命换毁灭木叶」后，她**全程当手术助手**（找齐材料·脖颈受伤仍坚持·递械擦汗·一度被甲虫酸血溅脸）——为救挚友长门，她不在乎「强者尊严」。手术中她**否决了苏晓给长门套「鼻环」的提议**（改成 12 颗黑耳钉、长门没成「牛头人」）。长门十几年来首次平躺、康复在望，一向喜怒不形于色的她难掩笑容。

!!! note "并非「辅助」"
    晓内一度以为这位蓝发御姐是「辅助」型——直到她**险些炸死「阿飞」[宇智波带土](宇智波带土.md)**，众人才知她绝非善茬（详见 [宇智波带土](宇智波带土.md)）。

## 长门死后 · 照顾苏晓 + 翻脸逃走（十五卷1–2章）

[长门](长门.md) 被鸣人感化而死后，小南没敢回雨忍村（无法向其他晓成员交代「长门被一名下忍感化」）。出于**报答苏晓当年救长门一命**之恩，她在雨之国一处偏僻村庄照料昏迷一月的苏晓。可当苏晓笑着点破「**阿飞 / 宇智波斑（即 [带土](宇智波带土.md)）呢？**」、暴露出自己是「带土的手下」时，她瞬间收起感激、咬牙切齿——苏晓若把「长门背叛晓」的细节捅出去，她就是晓的追杀目标。她当即俯身夺门而逃（苏晓没追·只为打探情况、且忌惮她的起爆符战术）。

## 藏轮回眼 · 被佐助生擒（十五卷70–73章）

[长门](长门.md) 死后，她**带着轮回眼（连同长门尸体）藏匿于雨之国**——靠雨之国某家族庇护、又深知白绝的侦查能力，从不久居一地、还杀掉过白绝的几具分身，令白绝只能远远跟、随时可能跟丢。最终晓派 [迪达拉](迪达拉.md) + [佐助](宇智波佐助.md) + 黑绝围捕，**佐助以幻术将她生擒、套出轮回眼下落后并未杀她**（带土对佐助这份「手下留情」颇为不满）；轮回眼落入黑绝之手运回——月之眼计划又近一步。
