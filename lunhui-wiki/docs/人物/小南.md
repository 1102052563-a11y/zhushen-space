# 小南

[火影忍者世界](../世界/任务世界/火影忍者.md)「**晓**」组织的成员——蓝发女子，[长门](长门.md) 最信任的心腹、与已故的「弥彦」同为长门的旧友（三人是患难与共的同伴）。

- **照料长门**：常年照看身体枯槁、靠机械维生的长门，对其日渐衰败的身体深感忧心。
- **迎接苏晓（九卷17章）**：奉长门之命，与「天道」（弥彦尸体·即长门本人代行）一同下楼**迎接新成员 [苏晓](苏晓.md)**——长门特意叮嘱她「（白夜）刀术能比肩木叶白牙、来路不明、交涉时别靠他太近」。

## 谈待遇 · 编入小队（九卷17–18章）
她代长门与 [苏晓](苏晓.md) 敲定加入事宜——苏晓干脆答应（「为保命 + 利益」），只要求晓**帮他解决雾忍暗部的追杀**（其他好处等他做出贡献再说）。她借「闲聊爆破知识」**打探**苏晓底细，被苏晓一通化学公式 + 爆破细节听懵、却由此确认他「不是说大话」。随后她发给苏晓 [空之戒](../力量体系/装备道具.md) 与黑底红云风衣，并**把他编入一个二人小队当第三人**（搭档在河之国·实为让那两人**监视、观察**他），临走叮嘱「不可泄露组织信息、小心你那两个搭档别被他们杀了」（晓正式编队后许多候补成员死于内斗·尤以角都 / 蝎为甚）。
