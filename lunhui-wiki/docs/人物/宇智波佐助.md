# 宇智波佐助

[火影忍者世界](../世界/任务世界/火影忍者.md) 宇智波一族——[宇智波鼬](宇智波鼬.md) 的弟弟，[苏晓](苏晓.md) 戏称的「**二柱子**」。一副高冷模样、佩**草雉剑**、此时是**三勾玉写轮眼**（尚未开万花筒）。曾拜入 [大蛇丸](大蛇丸.md) 门下、一心向兄长复仇。

- **杀大蛇丸（九卷58–59章）**：他**杀掉了大蛇丸**（黑绝向晓众人点破凶手是「宇智波佐助 = 鼬的弟弟」）——长门由此警觉宇智波兄弟若各开万花筒、互换眼可觉醒永恒万花筒乃至轮回眼。
- **被苏晓引去见鼬（九卷59–61章）**：苏晓放走 [香磷](香磷.md)、托她传话「想找鼬报仇可来火之国**神桥山**」——佐助遂率 [香磷](香磷.md)、[鬼灯水月](鬼灯水月.md)、[重吾](重吾.md) 四人寻来。佐助想以三勾玉幻术 + 逼问拿苏晓情报，却**幻术对苏晓无效**（智力 50）、近身一招就被苏晓环断 + 刀尖抵下颚完全压制（「就算鼬也不敢主动靠近我」）。苏晓直接告知鼬在「宇智波一族遗址」、带他前往——只让佐助一人进、其余被鬼鲛 / [宇智波带土](宇智波带土.md) 拦下。一场鼬刻意放水送死的兄弟对决就此拉开（详见 [宇智波鼬](宇智波鼬.md)）。

## 出处
- 九卷58–61章。
