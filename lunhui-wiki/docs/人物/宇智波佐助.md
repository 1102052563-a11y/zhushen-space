# 宇智波佐助

[火影忍者世界](../世界/任务世界/火影忍者.md) 宇智波一族——[宇智波鼬](宇智波鼬.md) 的弟弟，[苏晓](苏晓.md) 戏称的「**二柱子**」。一副高冷模样、佩**草雉剑**、此时是**三勾玉写轮眼**（尚未开万花筒）。曾拜入 [大蛇丸](大蛇丸.md) 门下、一心向兄长复仇。

- **杀大蛇丸（九卷58–59章）**：他**杀掉了大蛇丸**（黑绝向晓众人点破凶手是「宇智波佐助 = 鼬的弟弟」）——长门由此警觉宇智波兄弟若各开万花筒、互换眼可觉醒永恒万花筒乃至轮回眼。
- **被苏晓引去见鼬（九卷59–61章）**：苏晓放走 [香磷](香磷.md)、托她传话「想找鼬报仇可来火之国**神桥山**」——佐助遂率 [香磷](香磷.md)、[鬼灯水月](鬼灯水月.md)、[重吾](重吾.md) 四人寻来。佐助想以三勾玉幻术 + 逼问拿苏晓情报，却**幻术对苏晓无效**（智力 50）、近身一招就被苏晓环断 + 刀尖抵下颚完全压制（「就算鼬也不敢主动靠近我」）。苏晓直接告知鼬在「宇智波一族遗址」、带他前往——只让佐助一人进、其余被鬼鲛 / [宇智波带土](宇智波带土.md) 拦下。一场鼬刻意放水送死的兄弟对决就此拉开（详见 [宇智波鼬](宇智波鼬.md)）。

## 胜鼬 · 知晓真相（九卷62–63 / 69章）
与鼬决战中，他靠咒印肉翅 + 麒麟雷电苦撑，鼬却**刻意放水**、在逼出并封印他体内的 [大蛇丸](大蛇丸.md)（佐助由此被剥离咒印 / 大蛇丸、白得三年栽培）后，以两指点其额头植入禁术、力竭撞壁而亡，留遗言「**原谅我，佐助，这是最后一次了**」——佐助惨胜（实为鼬送死）。其后他与鼬尸体一同被 [宇智波带土](宇智波带土.md) 带到晓据点；带土向他揭露鼬「一切都是为保护你」的真相，他**信念动摇、恨意涣散、怀疑人生**。见到 [苏晓](苏晓.md) 时仍嘴硬问「我和鼬谁更强」，被苏晓 / 带土齐斥为「最愚蠢的问题」（苏晓自称三勾玉的佐助二十分钟可宰、却不愿与鼬战斗）。一枚带土口中「满心仇恨且强的棋子」初具雏形。

- **开万花筒 · 组鹰小队（九卷69–70章）**：得知真相、情绪崩到极点时**开出万花筒写轮眼**（六芒星·恨意转向「木叶高层」立誓杀之）后昏倒数日。醒来组「**鹰小队**」（[鬼灯水月](鬼灯水月.md)、[香磷](香磷.md)、[重吾](重吾.md)）；[宇智波带土](宇智波带土.md) 将派他去捕八尾（原著走向是捕捉失败）。

## 入晓 · 抓假八尾 + 誓杀团藏（十五卷6–9章）

苏晓再入火影时，佐助已是晓正式成员（带土的「重要棋子」·率鹰小队 [水月](鬼灯水月.md) / [香磷](香磷.md) / 重吾·后三人非正式成员）。他领命首次捕尾兽——自以为抓到了 [奇拉比](奇拉比.md)、险些战死（脏器被打飞 40%+），却被 [苏晓](苏晓.md) 青钢影一验、[带土](宇智波带土.md) 拧脖现出章鱼触须，坐实**抓来的是替身**（首次捕尾兽被骗·苏晓「我亲手抓过的尾兽比你见过的更多」的嘲讽让他难堪、「二柱子不愧是二柱子」）。他已知 [鼬](宇智波鼬.md) 的真相（鼬为护他自我牺牲、皆因 [团藏](志村团藏.md) 逼迫），故得知团藏现任代理火影后**杀意冰冷、一心要趁五影大会杀团藏**（被苏晓利用——佐助在哪、[漩涡鸣人](漩涡鸣人.md) 必追来、九尾自送上门）。
