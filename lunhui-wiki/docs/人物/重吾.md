# 重吾

[火影忍者世界](../世界/任务世界/火影忍者.md) [宇智波佐助](宇智波佐助.md) 的同伴之一——与 [香磷](香磷.md)、[鬼灯水月](鬼灯水月.md) 同属佐助身边的小队（皆出自 [大蛇丸](大蛇丸.md) 麾下）。

- **登场（九卷60–61章）**：佐助循 [苏晓](苏晓.md) 留下的「神桥山」线索来寻鼬复仇时，重吾作为四人小队成员一同到场。佐助单独进宇智波遗址见 [宇智波鼬](宇智波鼬.md) 时，他与水月、香磷一道被 [干柿鬼鲛](干柿鬼鲛.md)、[宇智波带土](宇智波带土.md) 拦在外面。

## 出处
- 九卷60–61章。
