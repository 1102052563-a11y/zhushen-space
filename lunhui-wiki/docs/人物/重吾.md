# 重吾

[火影忍者世界](../世界/任务世界/火影忍者.md) [宇智波佐助](宇智波佐助.md) 的同伴之一——与 [香磷](香磷.md)、[鬼灯水月](鬼灯水月.md) 同属佐助身边的小队（皆出自 [大蛇丸](大蛇丸.md) 麾下）。

- **登场（九卷60–61章）**：佐助循 [苏晓](苏晓.md) 留下的「神桥山」线索来寻鼬复仇时，重吾作为四人小队成员一同到场。佐助单独进宇智波遗址见 [宇智波鼬](宇智波鼬.md) 时，他与水月、香磷一道被 [干柿鬼鲛](干柿鬼鲛.md)、[宇智波带土](宇智波带土.md) 拦在外面。

## 天秤一族 · 咒印之源 + 重伤（十五卷13–14章）

五影大会上 [佐助](宇智波佐助.md) 被武士群围攻，重吾因「某个约定」近乎死忠于他、出手相助——揭其底细：他是**天秤一族**的特殊体质者，能吸收自然能量（仙之力）**半仙人化**（右臂化灰黑巨爪、右脸生灰斑、右瞳变黄·力 / 查克拉暴增 + 身体硬化）；**大蛇丸的「咒印技术」正源自重吾**（咒印 = 自然能量的粗浅用法）。但全身仙人化会令他暴走弑杀、故只局部化。可惜在 [四代雷影·艾](四代雷影.md) 面前不堪一击——**被全开雷遁之铠（活化 60%）的雷影一记侧拳把上半身打飞、当场重伤濒死**（雷属性给雷影恐怖的速度 + 切割力·重吾根本看不清他的动作）。
