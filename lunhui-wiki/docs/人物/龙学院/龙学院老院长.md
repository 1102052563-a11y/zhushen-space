# 龙学院老院长

[龙学院](../../世界/任务世界/龙学院-瓦伯雷.md) 深居简出的院长——一个不知活了多少年的老怪物，皮肤干枯、嘴唇开裂、瘦到皮包骨，那双**结晶眼球**般的眼睛让人发自内心打怵。传闻他就是龙学院的创立者，正在**窥探永生的秘密**。大部分学院决策由导师们定，无事无人敢去打扰他。

## 认出灭法者 · 选择和解

多年前 [格林·吉莉安](../格林·吉莉安.md) 来龙学院「交流学习」，因学院傲气把她惹毛，她**把学院塔楼从根打断、摧毁了龙学院三分之二**，临走还放话会让更多灭法者来「交流」。老院长愁得不轻，率多名导师历时多年研制出「**青钢影能量感应装置**」——却一直没派上用场，直到 [苏晓](../苏晓.md) 到访这天才启动。他起初怀疑苏晓是不是灭法者（居然这么守规矩），直到苏晓被 [利奥波特导师](利奥波特导师.md) 挑衅后当即毒倒宫廷骑士、那份无畏强权的凶悍让他确认「是那伙强盗没错」。老院长权衡后选择息事宁人:亲自出面、以苏晓的「太阳之书后半部」换出龙学院不传之秘「能量化结晶」「灵魂与结晶」（所谓不传之秘，出价够高就外传了）。他还随口点破利奥波特对太阳神族的偏见「人心中的成见会蒙蔽智慧」。

## 大书库失窃 · 赔笑收场

隆重开启只有他持钥匙的大书库四层，门后却被 [凯撒](../尼古拉斯·凯撒.md) 盗空。事后他回溯「时光流」查到戴深渊之罐的凯撒→找 [魔鬼族](../../世界/生物万族/index.md) 讨说法（碰上刚迎新爹「死灵之书」正憋气的魔鬼族一顿硬顶）→又惊动地精公司（凯撒通缉 23 万），最终只能加一大批灵魂石通缉 + 送魔鬼族 / 地精入学名额，赔着笑脸收场。

## 关系

- [苏晓](../苏晓.md)：他认出的灭法者、选择和解并交易结晶不传之秘的对象。
- [格林·吉莉安](../格林·吉莉安.md)：当年把龙学院捶毁三分之二、逼他研制灭法感应装置的女灭法者。
- [利奥波特导师](利奥波特导师.md)：他麾下势大欺客、被他点破成见的导师。
- [凯撒](../尼古拉斯·凯撒.md)：盗空他大书库四层、让他血压破六位数的地精。
