# 尼塔

[龙学院](../../世界/任务世界/龙学院-瓦伯雷.md) 的一名学徒——戴茶色兜帽、有一双美丽琥珀色眼睛的少女，[利奥波特导师](利奥波特导师.md) 的门下。导师不愿亲自冒险面见「太阳疯子」，便把她当炮灰派去接待 [苏晓](../苏晓.md)（她自知学徒身份配不上、一路满怀歉意「小嘴抹了蜜」、差点把病重的导师直接说没）。

## 从炮灰到内奸

亲眼看着苏晓、[巴哈](../苏晓.md) 毒倒门外的宫廷骑士后，尼塔吓得惶恐——但苏晓没杀她，反而塞给她一颗**增益秘药**（精神力正常疯长·只透支些体内能量）当作「你刚才没喊出声的报酬」，巴哈顺势把她说成「同伙」。在「被逮住就死咬你是同伙 / 帮带路则算被胁迫」的二选一下，尼塔选择当**内奸带路党**、卖掉自己的导师（导师让她来当炮灰、本就没真看好她）。苏晓还给她一瓶「**J4 型药剂**」（服用周期 5~7 周·期间痛不欲生·能逐步改变超凡资质、后天提升才能——打破了她「才能天生不可提升」的认知），换她一路引路（因苏晓伪装成宫廷骑士、其他骑士看一眼就放行）。她带苏晓上到导师居住区、又到大书库——却撞见大书库四层已被 [凯撒](../尼古拉斯·凯撒.md) 盗空的名场面。

## 关系

- [苏晓](../苏晓.md)：她被迫带路的「敌人」、却给她增益秘药和 J4 药剂改命的收买者。
- [利奥波特导师](利奥波特导师.md)：把她当炮灰派出去、被她转头卖掉带路的导师。
