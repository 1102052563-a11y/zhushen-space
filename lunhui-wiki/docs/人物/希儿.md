# 希儿

⚠ **她平常看上去是害羞、文静的眼镜妹**——⚠ **有时还梳着白色单马尾，一副她有社交恐惧症的模样**;⚠ **可在看到敌人的血后，她就逐渐眼冒红光**——⚠ **猎杀者中的毒奶，可以想象希儿有多危险。**

## 🔪 那两个晚上

⚠ **[狠人兄](狠人兄.md) 对自己的搭档希儿是一点其他想法都没有**——⚠ **或者说，刚开始合作时还是有些的。**

- ⚠ **第一晚**:⚠ **在一个五阶世界内，希儿突然从隔壁的帐篷内醒来，梦游着走进狠人兄的帐篷里，捅了他好几十刀，然后又回到自己帐篷**——⚠ **次日清晨，醒来的希儿才发现队友被袭，然后还帮狠人兄治疗。**
- ⚠ **在那时，狠人兄一度怀疑希儿是不是人格分裂**——⚠ **但碍于轮回乐园这边的治疗系不好找，更别说是猎杀者中的治疗系，就只能先忍了**;⚠ **外加他猜测，这种事应该只是偶然。**
- ⚠ **第二晚（当天晚上）**:⚠ **狠人兄在被捅刀子的剧痛中醒来**——⚠ **因为伤的过重，没机会还手就昏过去**;⚠ **然后再次醒来时，刚睁眼就看到正给他释放治疗能力的希儿**——⚠ **狠人兄险些当场又昏过去。**
- ⚠ **从那之后**:⚠ **哪怕希儿的容貌姣好、气质文静，也再也无法吸引狠人兄了**——⚠ **那张满是愉悦笑容、戴着圆框眼镜、还沾染星星点点血迹的脸，已经难以在脑海中抹去。**

## 🏅 【天赋徽章】的竞争者

⚠ **在生存排行榜首位奖励【天赋徽章】的争夺中**——⚠ **狠人兄与希儿是最棘手的竞争者**;⚠ **两人中，看似是狠人兄更危险，其实不然**——⚠ **狠人兄是比希儿略强一筹，但要是说危险程度，其实是希儿更高。**

- ⚠ **她也在世界联络平台上**:⚠ **当匿名者一口气开出 116140 枚一星称号的公告刷出时**——⚠ **希儿（轮回乐园）:「**奇怪的见识增长了。**」
