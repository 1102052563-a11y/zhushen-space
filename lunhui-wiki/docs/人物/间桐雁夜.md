# 间桐雁夜

[Fate/Zero 世界](../世界/任务世界/fate-zero.md) 的魔术师——满头白发的男人，**[苏晓](苏晓.md)（伪英灵 Berserker）名义上的 Master**。

- 为救「两个人」，他不惜把自己弄成一副惨样、参与第四次圣杯战争——体内被植入间桐家的**刻印虫**（在血管里游动、每动一毫米都剧痛，左半身已失去知觉）。
- 他召唤出的真 Berserker 是 [兰斯洛特](兰斯洛特.md)，却在召唤仪式出错后被 [苏晓](苏晓.md) 顶替——**苏晓作为伪英灵不受其令咒控制**，间桐雁夜只是「魔力供给者」、控制不了苏晓（甚至怕惹怒他被杀）。
- 他对折磨自己一年的「父亲」[间桐脏砚](间桐脏砚.md) 恨之入骨——见苏晓一刀斩了间桐脏砚，他扭曲狂笑。**苏晓暂留他一命**（圣杯世界魔术师不多·留作魔力供给者；他一死、苏晓在本世界停留就只剩最多 2 天）。

## 被改造成「伪 Berserker」（十一卷5–6章）

间桐脏砚一死、刻印虫失控，雁夜本就死定了（虫子破坏的身体组织无法修复、虫消失反而加速其死亡）。苏晓**蛊起他对 [远坂时臣](远坂时臣.md) 的复仇执念**（夺妻〔葵〕坑女〔樱〕之恨）、把他「废物利用」改造一番——注入 [兰斯洛特](兰斯洛特.md) 的**狂化之血**（保不死）+ 超浓度 **Rc 细胞** + 稀释 **柱间细胞**，再以青钢影噬灭这些能量做「粗暴增强」。结果他变得**极其抗揍却失去理智**（黑烟缠身、肌肉树木化、双目赤红、惧怕青钢影），模样酷似狂化的兰斯洛特——苏晓遂给他套上黑漆铠甲扮 **Berserker**、自己手背画令咒扮御主（全场最奇特的一队）。他虽失智，却仍受两个执念牵引：**保护 [间桐樱](间桐樱.md)**、**弄死远坂时臣**——可凭这两点略加引导。身体在缓慢崩溃、命不久矣，但苏晓自评「让他有了复仇的可能、好过原著中毫无意义地死」。

## 复仇执念发作（十一卷38–51章）

被 [蝶](蝶.md) 的「爱意俘虏」短暂魅惑操控、参与对苏晓别墅的袭击（蝶死后解控）。**英灵齐聚一战，他一眼认出 [英雄王·吉尔伽美什](吉尔伽美什.md) 是仇人 [远坂时臣](远坂时臣.md) 的英灵**——全身黑烟地站在公路旁、对着英雄王嘶吼「**时臣！！**」（夺妻坑女之恨发作）；英雄王斥他「疯狗」、射宝具欲杀，**被 [苏晓](苏晓.md) 一道刀芒救下**。但**苏晓与英雄王酣战之际、他却悄然消失不知去向**（狂化后唯一念叨的只有「时臣、樱」二词）。

## 死亡 · 被借刀伪造「同归于尽」（十一卷57–59章·殁）
他其实是循着 [英雄王](吉尔伽美什.md) 的气息追去（而非直接找时臣）——恰逢 [言峰绮礼](言峰绮礼.md) 在英雄王协助下背刺弑杀 [远坂时臣](远坂时臣.md)。雁夜赶到、扑向时臣尸体，**被英雄王斩首**；绮礼遂把他的无头尸塞进喷泉水池、让他的手掐着时臣的喉咙、另一只手捏爆时臣下体，**伪造出「雁夜与时臣同归于尽」的假象**（实则时臣的致命伤是背后一刀刺破心脏、非雁夜所为）。苏晓循布布汪指引找到现场、看破这桩借刀杀人。他一死、苏晓名义上「失去御主」（但主线已完成、无所谓）。临死仍紧攥着仇人——总算以另一种方式偿了「夺妻坑女」的执念。
