# 暴鼠（裁决者）

轮回乐园的一名**裁决者**——穿坎肩款黑色合金战斗服、腰挂能量霰弹枪，每次出场都拎着一瓶酒往口中灌（据它说是追随某位大人征战留下的习惯）。它不太厚道、爱看热闹，是 [阿尔托利亚（呆毛王）](阿尔托利亚.md) 的酒友前辈、也是 [癞蛤蟆](癞蛤蟆.md) 的酒友。

## 一句玩笑坑了呆毛王（42卷77章）

呆毛王被黑暗物质侵蚀，起因正是暴鼠——它与癞蛤蟆喝酒闲聊时随口说了句「要不去渊龙底弄点黑枫树枝干」，说者无心、听者有意，耿直的呆毛王当真跑去接受了龙之试炼、被黑暗物质重创。事后它一边不厚道地看呆毛王笑话、一边张罗请 [苏晓](苏晓.md) 治疗:它给的报酬是一截女灭法者 **格林·吉莉安** 的佩刀刀尖（苏晓想拿去喂斩龙闪）。

## 关系

- [阿尔托利亚（呆毛王）](阿尔托利亚.md)：被它一句玩笑坑去渊龙底、又被它张罗治疗的裁决者。
- [癞蛤蟆](癞蛤蟆.md)：一同关照呆毛王的裁决者酒友。
- [苏晓](苏晓.md)（白夜）：它以灭法者佩刀刀尖为报酬请来治疗呆毛王的猎杀者。
