# 深渊世界群像（老公爵 · 黎林公爵 · 月狼夫人 · 圣裔王 · 希）

[深渊世界·神灵挽歌](../世界/任务世界/深渊世界-塞拉帝国.md) 中 [苏晓](苏晓.md) 接触的圣裔（不死人）权贵、乐园契约者与猎杀目标。

## 塞拉帝国五公爵（圣裔 · 不死人）

塞拉帝国权力中心「圣廷」由圣裔王 + 五公爵决策——五公爵都是活了几百上千年的老阴哔，彼此联合又算计。

- **老公爵·艾德里安·阿尔弗（分尾蛇 · 搅屎棍 · 苏晓的合作者）**：五公爵中最老的圣裔——枯瘦如柴、随身**盘着两任亡妻被神术缩小并包浆的头骨**。贪婪善变、翻脸无征兆，享受不死而非厌倦（图谋极大、大到不惜与所有公爵敌对来掩饰目的：曾把优秀子嗣过嫁给其余四公爵逼对方喊他「祖父」）。诈病引仇敌黎林的人来刺杀、反被苏晓强闯压制——谈成让苏晓当「**代行公爵**」（既保权力又得武力 + 让苏晓帮他找「魂之石」= [断魂影之石](../力量体系/装备道具.md)）。塞拉危局时被派去前线督战、传出「战死」讯（实为见局势不可为就**跑路苟活**）；塞拉灭亡后**戴着人皮面具混在平民里安然逃脱**（「能苟到帝国灭亡还安然无恙」——五公爵里最不起眼却活到最后的老狐狸）。又追到深渊 / 神之国，**忽悠 + 抓住伊威尔**（想借她报复渊之蛇·靠天巴语跟天巴族谈笑脱身），图谋极大。
- **黎林公爵（森林 · 众神会真首领 · 殁）**：鹰钩鼻、不苟言笑的古板公爵——非初代圣裔（生来即是、世袭爵位）；**撑着塞拉帝国对抗北部联盟**（没他帝国早被打穿）、30% 官员愿拥他为圣裔王。真相 = 他**重建了老公爵 50 年前组建、被怒狮剿灭的「众神会」**，趁铁羽王将死想**杀光其余公爵 + 圣裔王、粗暴夺权救塞拉**（连杀怒狮 / 白龙·武器为爵徽「黎明树」+ 植物生命力）。被苏晓识破登门智取斩杀（生来即公爵·从没打过你死我活的仗·技法被碾压·临终「我所做的一切都是为了塞拉，真正理解我的却是你」）。可敬的对手。
- **月狼夫人·伊芙（月狼 · 自私狠辣 · 殁）**：慵懒华贵、看似三十出头的美妇——「狠如蛇蝎」、极度自私。亚兰斯危局选「与城共存」守后方（实因贪生不敢上前线）；城破后被冰皇帝 **雷德·巴纳德**（见下）当众斩首处决（临刑预言「有个不讲道理的怪物〔指苏晓〕会回来把你们赶尽杀绝」）。
- **怒狮公爵·吉伯特（怒狮 · 殁）**：握帝国 60%+ 军权的公爵——握重兵却不作为（北部联盟压境仍在酒庄吃喝）；被 **鬼影**（见下）等众神会刺客毒杀于城外葡萄酒庄（黎林借「不死寡妇 / 绯红恩赐酒」之名清掉这无能军头·死得不冤）。
- **白龙公爵·克米伦（殁）**：三百多岁仍痴迷女人、家里蹲不作为、不肯立继承人的公爵——被黎林所杀（临死前察觉黎林不对、秘密给老公爵送黎明树枝示警，成苏晓锁定真凶的关键）。

## 铁羽王（塞拉真正的王 · 非圣裔 · 二度被斩）

塞拉帝国「真正的王」——出身战场士兵、一切都是杀出来的，手持勾刃长枪、**非圣裔**（约 260 岁·没老死已是奇迹）。曾只身跃入深渊与**深渊之蛇**鏖战两天、刺瞎其一眼（「老实趴在深渊，你这爬虫」）；一人镇得北部联盟不敢攻亚兰斯城。垂垂老矣、每出手一次就更近死亡，故只要北部联盟 / 深渊不来就不露面；现身时**顺手宰了四名死亡乐园契约者**（如杀鸡屠狗）。在木屋后**自挖坟、裹草席、平静老死**（北部联盟当日破境）。**结局**：被 [渊之蛇](../世界/生物万族/index.md)「映召」出全盛之姿（枪术宗师 Lv.43·战王免控 + 战场「征战」越杀越强·钩刃长枪能把青鬼甩回来）与苏晓**巅峰技法对决**——互卖破绽、瞬息定生死；苏晓靠超微 [放逐](../力量体系/装备道具.md) 攀刀尖偷袭其头颅 + 魔刃终结（残血 16.8% 险胜·「真是畅快」）。苏晓**第二次葬了他**（断刃枪当碑）、命运主宰刻「铁」字。

## 渊之蛇（深渊守护圣灵 · 殁）

蓝皮人族的守护圣灵——巨大黑鳞分尾蛇（被铁羽王刺瞎一眼·收天巴族献的「源」能量·守着死亡乐园的时空枢纽换它的庇护物品）。**真正的能力 = 「映召」已故强者**（本体凭巨躯战斗反而弱）：伊威尔当年的对手是它映召的古龙之子洛斯、铁羽王的对手是初代圣裔王——故所有与「渊之蛇」战斗者的遗言都对不上号。苏晓杀穿其映召的铁羽王后、随手捏死缩小的本体（蛇尸丢给厄休拉·得圣灵级宝箱）。

## 鬼影 / 众神会刺客（向死而荣）

- **鬼影（女刺客 · 众神会 · 被反利用）**：自幼受训、毕生使命就是刺杀怒狮公爵的女暗杀者——「向死而荣」、信背后组织以「神灵名义」诛圣裔（死后回归神灵怀抱）；刺杀时**不滥杀无辜**（只击晕仆从）成了破绽。苏晓 10 分钟从手印 / 蔷薇花香 / 腰伤推理出她的体征、生擒后**操控分解的放逐微操她的表情伪装「叛变」**（钓组织出手·心态崩）。
- **野鸳 / 十二人众（众神会成员）**：**野鸳**（众神会探子·目睹「叛变」的鬼影）；**十二人众**（12 名圣剑士·剑术大师 Lv.30~40·身穿重甲 + 互相用「光明」圣力治疗·围攻苏晓被斩·最后集体自尽**召唤出古神 [厄休拉](../世界/生物万族/index.md)**）。

## 圣裔王·亚历山（笨拙的战王 · 殁）

塞拉帝国之王——银灰发、宽松白衣如睡衣、毫无气场的中年男，开会都能秒睡。**真相**：他不是深藏不露、只是觉得身边争权夺利的一切「太无聊」；笨拙到学个礼仪要练一月（跳舞曾把女士裙子扯到底），却**唯独握剑那刻第一次笑了**——是个纯粹的「战王」。曾**弑父**（父亲临终却以他为傲）、封剑去完成父亲期望当圣裔王、又因「父之龙」的故事愧疚。其实并不昏庸（采纳黎林大半意见·踩住野心的怒狮）。**剑术宗师 Lv.37 + 11 种被动 + 不死性 + 圣灵级战剑**——塞拉除铁羽王外最强者；睡觉时能在梦中修行剑术（天赋）。被苏晓 [狼族传家宝](../力量体系/装备道具.md) 拖到荒野单挑 + 小队围攻斩杀（坦 [阿姆](阿姆.md) 硬接 + [巴哈](剧毒魔鹰·巴哈.md) 暗杀 + 苏晓主力·「不死剑术宗师」也扛不住默契围攻），取其心脏 + 头颅里的**第四颗 [断魂影之石](../力量体系/装备道具.md)**（魂之石）。苏晓剥得其「技法记忆·剑术宗师」（修行方式太奇特·不适用）。

## 希（无编号契约者 / 超脱者 · 水晶女皇 · 殁）

苏晓**二阶时的旧敌**——银发、爱猫、有 9~10 条命的女人（当年正面一对一苏晓打不过她、最终靠 [阿波罗](../力量体系/消耗品材料.md) 硬炸死，其能力为「**水晶**」）。她没死透、在天启乐园外**以中立神灵的「源」为食**（炼狱般历练）疯狂成长、已**杀了 76 名猎杀者**。本世界加入北部联盟、来神之国想吞古神厄休拉的源——却被恢复巅峰的**伊威尔**（见上）完克。最终在草原被苏晓**斩杀**（完成主线 2）：希近乎全能（**晶体化**免致命 + 高速自愈 / **同化**地面 / **晶女皇**召水晶龙 / **晶焰**燃生命与灵魂 / 晶爆·燃魂），但水晶困不住刀术宗师——苏晓以血气分身 + 龙影闪绕到死角、连破其晶体化（要害分离即真死）斩杀。死前用未驾驭的「神灵泣」也没能翻盘。背后那个「神秘存在」仍是悬案。

> 战斗细节：希更擅中距离（近战吃亏）；苏晓「刃道刀·绝幽」（眼前世界变黑、三道斩痕分裂成一大片）破其最后一搏。

## 乐园契约者（盟友 / 资源）

- **瑟薇（轮回乐园契约者）**：低胸裙、气质妩媚的「自己人」——出金镑赎苏晓出狱、交一等勋后完成自己的主线最终环、离场。
- **天启乐园矿工队（6 人 · 被「可持续压榨」）**：在亚兰斯城墙下偷挖 T19-亚合矿（蓝矿 / 生命矿）的天启乐园矿工——矿工帽队长（阅历丰富、果断说实话求活）+ 礁山（2.5 米肌肉壮汉、胆小心善）+ 黑皮矿工妹等。被苏晓逮住后立「五五分成」继续挖（预估给苏晓 2000+ 蓝矿）。
- **半人鳄（猎杀者 · 委托人 · 现实世界）**：冷血如鳄的中年猎杀者——被希打得很惨、奉轮回乐园之命把希的 DNA（一缕银发）送到苏晓现实饰品店，委托他「趁希没完全成长就宰了」。

## 冰皇帝·雷德·巴纳德（北部联盟新王）

灭塞拉帝国、登东大陆的北部联盟最高统治者——魁梧、胡须扎成粗辫、铁血却不嗜杀。攻塞拉非为复仇（那只是鼓舞士气的借口），真正目的是**让苦寒之地的子民离开极地、占东大陆求活**（且铁羽王一死「不战之约」即失效）。城破后宣布平民自由、私产不可侵犯，却也清楚还有深渊 / 神之国 / 渊之蛇等大敌。当众处决月狼夫人。

## 伊威尔（史上第一噬神者 · 一切的起始之人）

被深渊之蛇蛊惑、**史上第一个斩神夺力的噬神者**、活得最久的圣裔（屠戮上万圣裔 + 无数神灵·命镰「夺生命力之镰」斩中圣裔即秒杀）。后从所杀神灵的记忆里得知**圣战是真的、神灵并不恶、弑神只源于人心的贪婪**，悔悟弃猎；找深渊之蛇决战首尝败绩（甘愿赴死却未得解脱）。如今**跪在深渊前向所杀神灵永世忏悔**、生命力飘散成只会念叨的白痴——吃下圣裔王**亚历山**（见上）的心脏（古龙之子血脉）后恢复神智、又屠万名北部士兵恢复巅峰二十岁模样。是任务「一切的起始之人」（苏晓交亚历山心脏 + 她掏眼球给指引光球）；同时**完克希**——30 秒切一种元素形态（风电超高速 / 生死即死 / 光生治疗 / 地死死灵 / 死风瘟疫），把希小队打到自闭撤退。追到神之国前守村**反被天巴族群殴**（单挑无敌、群殴无解·靠雷+风形态逃命），目睹这里就是「神之国」彻底崩溃（自己被渊之蛇利用屠尽神灵、让旧敌占了神灵乐土）；被同期噬神者**老公爵**（见上）以「让渊之蛇付出代价」忽悠（实为利用·后双双被天巴关进牢笼·老公爵靠天巴语脱身、把她加固反锁）。

## 希阵营（北部联盟 · 来吞神源 · 几近团灭）

- **米糖（圣光乐园违规者 · 欢乐智障萌妹 · 签约改行）**：希雇佣的感知系违规者——像出笼萨摩耶、智商捉急说话欠揍，却极能苟（隐匿大招开 6 小时无敌但不能动）。希死后被苏晓小队围住，**跪地签 78 条轮回契约求活**、奉上买命钱（14 灵魂结晶〔完整〕+ 16 传说之勋 + 3 史诗宝石）、给苏晓「封灵之匿」灵隐 buff（潜渊蛇邸用），最后被苏晓扯掉近一月记忆放走（她决定「改行」）。
- **老狼（希的队友 · 殁）**：随性帅气、欠希人情来还债的男人——能提纯空气让人「氧中毒」、令生物「氧化生锈」；被阿姆（越战越抗揍·氧化反令其回血）+ 巴哈联手捶死。
- **康巴（希的主坦 · 殁）**：沉默寡言的中年汉子——4 米巨汉 + 5 米塔盾硬抗；被沁天巴族**五箭秒杀**（沁的源之力顺武器侵入致脱力、远比远程的「溺」恐怖），为米糖逃脱争取 2 秒。

## 奥波（守源人 · 胖巨人 · 第六环引导人）

蹲守深渊神之礼殿 / 渊蛇邸的秃头胖巨人（真实力 150+·不伤活物·见人就逃）——**守源人**：与渊之蛇约定「只要他守着源之潭，挚友古龙之子洛斯就不会受伤」，已苦等死去的洛斯几百年（其实早猜到洛斯遭不幸·洛斯尸骸就在他守望的神之礼殿内）。苏晓拔刀「友善交涉」+ 告知洛斯死讯（奥波泪崩）后**激活第六环**；奥波善良、怕自己「带来灾难」不敢去人多的永月城。
