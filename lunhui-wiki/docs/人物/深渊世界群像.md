# 深渊世界群像（老公爵 · 黎林公爵 · 月狼夫人 · 圣裔王 · 希）

[深渊世界·神灵挽歌](../世界/任务世界/深渊世界-塞拉帝国.md) 中 [苏晓](苏晓.md) 接触的圣裔（不死人）权贵、乐园契约者与猎杀目标。

## 塞拉帝国五公爵（圣裔 · 不死人）

塞拉帝国权力中心「圣廷」由圣裔王 + 五公爵决策——五公爵都是活了几百上千年的老阴哔，彼此联合又算计。

- **老公爵·艾德里安·阿尔弗（分尾蛇 · 搅屎棍 · 苏晓的合作者）**：五公爵中最老的圣裔——枯瘦如柴、随身**盘着两任亡妻被神术缩小并包浆的头骨**。贪婪善变、翻脸无征兆，享受不死而非厌倦（图谋极大、大到不惜与所有公爵敌对来掩饰目的：曾把优秀子嗣过嫁给其余四公爵逼对方喊他「祖父」）。诈病引仇敌黎林的人来刺杀、反被苏晓强闯压制——谈成让苏晓当「**代行公爵**」（既保权力又得武力 + 让苏晓帮他找「魂之石」= [断魂影之石](../力量体系/装备道具.md)）。
- **黎林公爵（森林 · 众神会真首领 · 殁）**：鹰钩鼻、不苟言笑的古板公爵——非初代圣裔（生来即是、世袭爵位）；**撑着塞拉帝国对抗北部联盟**（没他帝国早被打穿）、30% 官员愿拥他为圣裔王。真相 = 他**重建了老公爵 50 年前组建、被怒狮剿灭的「众神会」**，趁铁羽王将死想**杀光其余公爵 + 圣裔王、粗暴夺权救塞拉**（连杀怒狮 / 白龙·武器为爵徽「黎明树」+ 植物生命力）。被苏晓识破登门智取斩杀（生来即公爵·从没打过你死我活的仗·技法被碾压·临终「我所做的一切都是为了塞拉，真正理解我的却是你」）。可敬的对手。
- **月狼夫人·伊芙（月狼 · 自私狠辣）**：慵懒华贵、看似三十出头的美妇——「狠如蛇蝎」、极度自私（不威胁到自己就不冒险）。众人称她「月狼夫人」而非公爵（似有隐情）。亚兰斯危局选「与城共存」守后方补给（实因贪生不敢上前线）。
- **怒狮公爵·吉伯特（怒狮 · 殁）**：握帝国 60%+ 军权的公爵——握重兵却不作为（北部联盟压境仍在酒庄吃喝）；被 **鬼影**（见下）等众神会刺客毒杀于城外葡萄酒庄（黎林借「不死寡妇 / 绯红恩赐酒」之名清掉这无能军头·死得不冤）。
- **白龙公爵·克米伦（殁）**：三百多岁仍痴迷女人、家里蹲不作为、不肯立继承人的公爵——被黎林所杀（临死前察觉黎林不对、秘密给老公爵送黎明树枝示警，成苏晓锁定真凶的关键）。

## 铁羽王（塞拉真正的王 · 非圣裔 · 老死）

塞拉帝国「真正的王」——出身战场士兵、一切都是杀出来的，手持勾刃长枪、**非圣裔**（约 260 岁·没老死已是奇迹）。曾只身跃入深渊与**深渊之蛇**鏖战两天、刺瞎其一眼（「老实趴在深渊，你这爬虫」）；一人镇得北部联盟不敢攻亚兰斯城。垂垂老矣、每出手一次就更近死亡，故只要北部联盟 / 深渊不来就不露面；现身时**顺手宰了四名死亡乐园契约者**（如杀鸡屠狗）。最终在木屋后**自挖坟、裹草席、平静老死**——他一死，北部联盟当日破境、塞拉再无屏障。

## 鬼影 / 众神会刺客（向死而荣）

- **鬼影（女刺客 · 众神会 · 被反利用）**：自幼受训、毕生使命就是刺杀怒狮公爵的女暗杀者——「向死而荣」、信背后组织以「神灵名义」诛圣裔（死后回归神灵怀抱）；刺杀时**不滥杀无辜**（只击晕仆从）成了破绽。苏晓 10 分钟从手印 / 蔷薇花香 / 腰伤推理出她的体征、生擒后**操控分解的放逐微操她的表情伪装「叛变」**（钓组织出手·心态崩）。
- **野鸳 / 十二人众（众神会成员）**：**野鸳**（众神会探子·目睹「叛变」的鬼影）；**十二人众**（12 名圣剑士·剑术大师 Lv.30~40·身穿重甲 + 互相用「光明」圣力治疗·围攻苏晓被斩·最后集体自尽**召唤出古神 [厄休拉](../世界/生物万族/index.md)**）。

## 圣裔王·亚历山（笨拙的战王 · 殁）

塞拉帝国之王——银灰发、宽松白衣如睡衣、毫无气场的中年男，开会都能秒睡。**真相**：他不是深藏不露、只是觉得身边争权夺利的一切「太无聊」；笨拙到学个礼仪要练一月（跳舞曾把女士裙子扯到底），却**唯独握剑那刻第一次笑了**——是个纯粹的「战王」。曾**弑父**（父亲临终却以他为傲）、封剑去完成父亲期望当圣裔王、又因「父之龙」的故事愧疚。其实并不昏庸（采纳黎林大半意见·踩住野心的怒狮）。**剑术宗师 Lv.37 + 11 种被动 + 不死性 + 圣灵级战剑**——塞拉除铁羽王外最强者；睡觉时能在梦中修行剑术（天赋）。被苏晓 [狼族传家宝](../力量体系/装备道具.md) 拖到荒野单挑 + 小队围攻斩杀（坦 [阿姆](阿姆.md) 硬接 + [巴哈](剧毒魔鹰·巴哈.md) 暗杀 + 苏晓主力·「不死剑术宗师」也扛不住默契围攻），取其心脏 + 头颅里的**第四颗 [断魂影之石](../力量体系/装备道具.md)**（魂之石）。苏晓剥得其「技法记忆·剑术宗师」（修行方式太奇特·不适用）。

## 希（无编号契约者 / 超脱者 · 主线猎杀目标）

苏晓**二阶时的旧敌**——银发、爱猫、有 9~10 条命的女人（当年正面一对一苏晓打不过她、最终靠 [阿波罗](../力量体系/消耗品材料.md) 硬炸死，其能力为「水晶」）。她没死透、在天启乐园外**以中立神灵的「源」为食**（炼狱般历练）疯狂成长、已**杀了 76 名猎杀者**，还在入侵圣光乐园时偷袭过苏晓。背后有**某个神秘存在**支持（屡次隐匿她踪迹、把她当「亲女儿」养、却屡被轮回乐园夺走时空之力）——苏晓判断希不是重点、她背后的存在才是。本世界主线 2 即猎杀她（[贝妮](狩之影.md) 负责找她踪迹·希爱猫故贝妮可卖萌潜伏）。

## 乐园契约者（盟友 / 资源）

- **瑟薇（轮回乐园契约者）**：低胸裙、气质妩媚的「自己人」——出金镑赎苏晓出狱、交一等勋后完成自己的主线最终环、离场。
- **天启乐园矿工队（6 人 · 被「可持续压榨」）**：在亚兰斯城墙下偷挖 T19-亚合矿（蓝矿 / 生命矿）的天启乐园矿工——矿工帽队长（阅历丰富、果断说实话求活）+ 礁山（2.5 米肌肉壮汉、胆小心善）+ 黑皮矿工妹等。被苏晓逮住后立「五五分成」继续挖（预估给苏晓 2000+ 蓝矿）。
- **半人鳄（猎杀者 · 委托人 · 现实世界）**：冷血如鳄的中年猎杀者——被希打得很惨、奉轮回乐园之命把希的 DNA（一缕银发）送到苏晓现实饰品店，委托他「趁希没完全成长就宰了」。

## 关系

- **[苏晓](苏晓.md)（库库林·白夜 / 代行公爵）**：老公爵的合作者 / 代行公爵；斩众神会首领黎林、斩圣裔王亚历山；与古神 [厄休拉](../世界/生物万族/index.md) 立「灭法者之约」换情报；希的毕生之敌（已「杀」她 10 次）。
- **塔西娅（圣裔小萝莉）**：老公爵记不清第几代的子嗣——偷听苏晓与老公爵谈话被吓哭（两人都在笑她却发自内心地怕）。
- **北部联盟（极地民 · 攻塞拉）**：苦寒之地善战的极地民族——新生儿要冻一天才喝母乳、骑披甲极地野猪。铁羽王一死即 15 万极地骑兵破境直扑亚兰斯城；**希**（见上）已加入其中。
