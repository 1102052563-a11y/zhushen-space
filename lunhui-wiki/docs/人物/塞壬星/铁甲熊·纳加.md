# 铁甲熊·纳加（9 号避难所新首领 · 同盟）

长相粗犷却心细如发的猎手队长——为还断牙救命之恩（他半盲那只眼是替断牙挡的）巧妙保下必死的断牙、送 [苏晓](../苏晓.md)「磁爆猎手」史诗设计图当见面礼。

- 被苏晓扶上首领之位（接替 [亚哈达](亚哈达.md)·治疗药换满世界找狄巴姆·野心渐长）。

## 关系

- [苏晓](../苏晓.md)：扶他上位、结盟的伪装药师。
- [亚哈达](亚哈达.md)：被斩后由他接替的原首领。

## 出处

- 二十九卷（[塞壬星·入侵天启](../../世界/任务世界/塞壬星-入侵天启.md)）17–33 章：保断牙 + 送设计图→被扶为首领。
