# 哀嚎女王（洛丹城 · 不死者小 boss · 殁）

废城里的精英 [不死者](../../世界/生物万族/index.md)——灰白肤、血红眼、满头垂地白发（可金属化护体）、有智慧、压迫感强。

- 本想为部下报仇杀游侠小队，一对上 [苏晓](../苏晓.md) 三随从就只想逃——被 [阿姆](../阿姆.md)（火时定身 + 抡砸）、[巴哈](../剧毒魔鹰·巴哈.md)（一次性注 2300 毒）、[布布汪](../布布汪.md)（跳板配合）按住围殴致死（「史上最惨 boss」）。

## 关系

- [苏晓](../苏晓.md) 及三随从：把她围殴致死的人。

## 出处

- 二十九卷（[塞壬星·入侵天启](../../世界/任务世界/塞壬星-入侵天启.md)）1–16 章：欲报仇→被三随从围殴致死。
