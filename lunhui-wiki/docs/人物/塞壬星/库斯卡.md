# 库斯卡（霸主级 · 狄巴姆融合体 · 本世界最强 · 殁）

起源岛温房深处守护超凡植物·狄巴姆的**霸主级生物·狄巴姆融合体**——本世界最强（力 / 体 150 本世界极值·徒手战斗宗师·完美植入 +8 万生命 / 免伤 35%·死亡咆哮双重音波控制 + 双生共存每秒回 10% 霸体）。

- 出场逼格极低（从藤蔓球摔下来），却强到「比月神还强一截」；是 [莫格罗](莫格罗.md) 的旧仇（当年莫格罗跪着逃命·维罗妮卡之死的纠葛）。
- 被 [苏晓](../苏晓.md) **三连阿波罗 + 阿波罗·改 + 死寂降临「正义痛击」**轰残、再血之兽 + 绝影闪 + 魔刃斩杀（不获世界之源·得灵魂钱币 100 + 狄巴姆萃取原液 / 种子 / 霸主精魄·死前道「斩的漂亮、多谢你」获解脱）。

## 关系

- [苏晓](../苏晓.md)：轰残并斩杀他、送他解脱的猎手。
- [莫格罗](莫格罗.md)：他的旧仇（进化反目后撞上他阵亡）。
