# 莫尼卡（天启乐园 · 光杆辅助 · 被苏晓当幌子）

红裙、波浪长发、被队友嫌弃的光杆辅助系——苦求组队（「你加入我们就有两个人了」）被 [苏晓](../苏晓.md)「你太弱了」噎到。

- 被苏晓利用：带路进契约者营地、敲开舰长室、目睹苏晓一刀斩舰长后被逼着出去喊「舰长被袭击」当幌子（全员恶人既视感·一脸「我到底做了什么」）。

## 关系

- [苏晓](../苏晓.md)：利用她当带路 / 幌子的人。
