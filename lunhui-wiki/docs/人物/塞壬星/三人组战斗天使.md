# 三人组战斗天使（克罗 / 皮衣男 / 天使妹子）

临时三人一队追杀 [苏晓](../苏晓.md) 的天启战斗天使（实力五阶顶尖梯队·前 30）：

- **克罗（殁）**：壮汉·摔投绝技「力霆·撼地」——被苏晓借力反摔「开大招自己摔自己」后斩杀。
- **皮衣男**：能 65% 同步复刻斩龙闪 / 刃道刀 / 青鬼——但无苏晓体质硬撑会肌肉断裂。
- **天使妹子**：淡雅圣光系·「天启之光」光柱——把苏晓视作大魔王、遇见就喊「跑！」。

苏晓**选择性放水**钓出暗处刺客后斩克罗；皮衣男 + 天使妹子从此见苏晓掉头就跑（砍出阴影）。

## 关系

- [苏晓](../苏晓.md)：追杀他、被放水钓刺客的三天使。

## 出处

- 二十九卷（[塞壬星·入侵天启](../../世界/任务世界/塞壬星-入侵天启.md)）34–48 章：追杀苏晓→克罗被斩、余两人逃。
