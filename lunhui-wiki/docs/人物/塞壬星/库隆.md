# 库隆（天启坦 · 指挥 · 攻 C 形走廊）

脸带斩痕的坦系契约者——指挥契约者营地 329 名天启契约者三波猛攻 C 形走廊（十坦盾墙 + 生命之链 / 近战 + 海量控制 / 集束爆炸物）想夺回舰长室。

- 全被 [苏晓](../苏晓.md) 凭地形 + 侧踢「一脚秒坦」挡下（48 人永远留在长廊）；被苏晓资料（六阶战力 / 独赢世界争夺战）打到「终于懂轮回乐园为何那么强」。后 [米奈](米奈.md) 到场时他自愿让出指挥权。

## 关系

- [苏晓](../苏晓.md)：三波攻不进、被其战力折服的对手。
- [米奈](米奈.md)：他自愿让出指挥权的天启首领。

## 出处

- 二十九卷（[塞壬星·入侵天启](../../世界/任务世界/塞壬星-入侵天启.md)）17–33 章：三波攻 C 形走廊被挡。
