# 战斗天使维克（天启乐园 · 追杀苏晓 · 殁）

追杀「高危违规者」[苏晓](../苏晓.md) 的**天启乐园战斗天使**（相当于轮回乐园猎杀者）——褐皮衣、寸发、醉酒登场。

- 说辞极熟练（劝违规者「去挖矿十个世界就能重新当契约者」·半真半假）；液态金属臂铠、圣血（金瞳爆发）、捏卷轴定身。
- 实力其实平平（非乐园竞技场前百名），被苏晓侧踢踢爆侧腰、[放逐](../../力量体系/装备道具.md) + 龙影闪穿空间斩喉秒杀（得 ★★★★★ 猩红卡·入侵度 +5%·杀战斗天使会引来更多天使）。

## 关系

- [苏晓](../苏晓.md)：追杀他、反被秒杀的战斗天使。
