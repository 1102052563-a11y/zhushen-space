# 火焰颅·莫格罗（火颅帮首领 · 同盟 → 进化暴食君主反目 · 殁）

统领 17 号避难所 28 年、没人战胜过的火颅帮首领——半脸金属面具 + 大量导管、骑 200 吨「不死蜥」、重度血焚 + 不死者性质「以毒攻毒」才活到现在的狠人。

- 摧毁过 12 处避难所、却是**为子嗣 / 部下甘愿赴死的守信枭雄**（带全帮来换药、做好死在苏晓刀下的准备）——[苏晓](../苏晓.md) 认其诚意结盟，用治疗药配方 + 9 号避难所换他**带路去狄巴姆本体（被污染的世界之核）**。
- **进化暴食君主·反目（29卷43章起）**：登起源之岛后暴露另有所图——**吞不死虫进化成「暴食君主」（裂变体Ⅵ型终极形态）**、拉怪冲散苏晓小队、屠戮天启契约者、入侵起源岛中心。
- **阵亡（29卷51章）**：温房内撞上旧仇 [库斯卡](库斯卡.md)（当年维罗妮卡之死 / 他跪着逃的纠葛）——明知不敌仍冲上去拖住，被库斯卡 + 苏晓第二颗阿波罗焚成灰（伤主要库斯卡造成·苏晓未获奖励）。

## 关系

- [苏晓](../苏晓.md)：从结盟到反目的枭雄（人格魅力评价极高）。
- [库斯卡](库斯卡.md)：他的旧仇、拖住其阵亡的对象。
