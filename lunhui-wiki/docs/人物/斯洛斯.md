# 斯洛斯（慵懒）

[瓶中的小人](瓶中的小人.md) 所造七名人造人之一，代表「**慵懒**」——3 米肌肉壮汉·独眼（另一眼是红节点）·智低生性愚钝、梦想一直睡觉。

- **能力**：**人造人中速度最快**（全速冲撞化残影、破空气浪·快到自己都刹不住）；子弹 / 实心炮弹都打不破他的皮。任务是在亚美国地下挖**国土炼成阵的圆形能量通道**。
- **殁**：七卷30章在 [布里克兹要塞](../世界/地点/index.md) 地下武器库现身；苏晓赤膊、摘下双臂负重金属环锁、让 [布布特尼](布布汪.md) 拉仇恨，斩龙闪「如切豆腐」斩之。斯洛斯为报兄弟（[嫉妒](恩维.md) / [暴食](格拉特尼.md)）之仇压制了慵懒、全力一拳把苏晓砸到生命 27%（苏晓由此记下「**不可因一时顺利小看任何敌人**」）；最终被青钢影「法力爆燃」击碎贤者之石斩杀（剧情关键·世界之源 +5.7% → 28.3%·紫宝箱·断罪 5/7·七卷31–32章）。
