# 蒙奇·D·路飞

**蒙奇·D·路飞**（前文记为「草帽小孩」）是 [海贼王](../世界/任务世界/海贼王.md) 世界、原著的主角，被轮回乐园判定为**位面之子**（第16章）。

## 简介
黄皮肤、黑发、漆黑眼睛，头戴草帽，在 [科尔波山](../世界/地点/index.md) 长大。此时尚幼，已具橡胶般弹力（被踢飞会弹跳），但果实能力基本未开发、对攻击免疫不强；性格直率（第16章）。

## 与苏晓的交集
- 第14–15章：在灌木丛目睹 [苏晓](苏晓.md) 猎杀科尔波山的主人。
- 第16章：被苏晓踢出、盘问（苏晓本意是观察恶魔果实能力者）。此时轮回乐园发出血红警告「猎杀者正接触**位面之子**，请远离」，并检测到超强生物 [蒙奇·D·卡普](index.md)（LV.79）靠近、触发高难任务「传奇的扼杀」；苏晓果断扔下路飞、放弃任务离开。

## 关系
- [苏晓](苏晓.md)——被其踢飞、盘问。
- 蒙奇·D·卡普——其祖父（LV.79 超强生物；苏晓接触路飞时触发相关警告）。
