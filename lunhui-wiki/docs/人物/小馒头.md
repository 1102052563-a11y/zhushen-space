# 小馒头（糖可欣 · 野生奶妈契约者）

[海贼王 · 德雷斯罗萨海战](../世界/任务世界/海贼王-德雷斯罗萨海战.md) 香波地篇登场的**四阶契约者**——真名糖可欣（坚持要别人叫她「小馒头」），一名**团队奶妈（治疗 + 感知）**，性格软、跳脱、话痨，杀人时还会紧张。原是某中型冒险团的奶妈 + 感知，团队覆灭后沦为「无家可归的野生奶」、被一伙契约者强行签约抓去当「工具奶」（要抽她 70% 任务收益）。

- **资料**：12709 号·Lv.33（四阶）·法力 2870·力 31 / 敏 27 / 体 43 / 智 86（真实）/ 魅 56 / 幸 3。天赋 **圣之女**（治疗时有几率施加双倍 / 三倍 / 五倍治疗）；装备金色治疗法杖**天使之权** + 淡金 TT-M 驱动戒 / 蓝海之拥项坠；技能 缓速创伤愈合 / 圣力洗礼（回 900 + 10% 已损生命）/ 龙魂咆哮（增友方力敏）。

## 被苏晓收作临时辅助（十八卷61–63章）

[布布汪](布布汪.md) 一记「狗爪手枪」灭了她那伙在 21 区抓活人厮杀刷声望的小队，仅她幸免。[苏晓](苏晓.md) 验明她是奶妈后留她一命、用爆炸耳钉逼她签**强制契约**（海贼世界结束前无偿辅助苏晓·**收益互不干涉**·17 条强制条款·违约全属性 −30% + 扣装备 / 剥技能）。她爽快接受——因苏晓不压榨她的收益（辅助苏晓杀敌还能分 10% 声望）、远胜旧队抽她 70%。她话痨地爆出「**罗杰佩枪 = 10000 海贼声望**」的隐藏任务大料、又认出苏晓就是「1V1 灭掉星火冒险团」的三阶大佬白夜（吓得节操碎一地）。此后跟着苏晓在火药酒屋外辅助（被苏晓用胶带封嘴防她答布布汪的话）。

## 关系
- **[苏晓](苏晓.md)（白夜）**：签约收她当临时辅助的契约者（手段粗暴但不压榨·她遂不背刺）。
- **[布布汪](布布汪.md)**：灭她旧队、她感知都察觉不到的「汪大佬」。
