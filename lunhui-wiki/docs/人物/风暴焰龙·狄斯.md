# 风暴焰龙·狄斯

[苏晓](苏晓.md) 在暗影世界亲手培育的**九阶霸主级**龙骑坐骑，名**狄斯**。它并非天生，而是一头被 [棘拉](棘拉.md) 赋予**无主领主级灵魂**的**恶魔焰龙**，注入起源级满评分的**暴风生物核心（[驱动核心·暴风领主](力量体系/装备道具.md)）**、再灌以黑暗羽 / 风暴龙血 / 生命力后完成的一次超强**蜕变**——从「配不上领主灵魂」的基础形态，一跃成为遍体**黑蓝色大羽毛**、竖瞳威严冷傲的中大型焰龙。

!!! note "与旧坐骑「恶魔焰龙·巴巴托斯」不是同一头"
    早年的 **恶魔焰龙·巴巴托斯** 是棘拉的**首领级**衍生灵魂，离开虫巢大本营 / 棘拉所在的世界后战力大跌、只能当飞行坐骑。狄斯用的是**无主领主级灵魂**——不受虫族本能牵制、到哪个世界力量都不降，苏晓在其灵魂上烙有印记以操控。羽翼质感也远比巴巴托斯的硬龙皮 + 骨刺适合乘骑。

## 桀骜与驯服 · 界雷立威

⚠ 蜕变后的狄斯桀骜不驯，最初想挣脱灵魂烙印、不愿被乘骑。苏晓一度**主动破碎其灵魂烙印**放它自由——它狂喜欲走，却因龙类的多疑而犹豫;苏晓遂翻开夏的龙类菜谱念「龙腹肉肥而不腻、小火慢炖」、点破它的**风暴核心**是本世界不超五指之数的稀世珍宝、又断言「没我的灵魂烙印，你飞不过六小时就被端上餐桌或关进疯人院地牢」。带它飞一圈联盟（沿途被高空警卫结界锁定、四名飞行强者盘查、险被泰莎的铁血重炮轰下），狄斯的龙目彻底迷茫——**这世界危险到超乎它想象**，遂乖乖吞回灵魂烙印。此后苏晓又以【雷之灵】引界雷锻炼其**界雷抗性**，它便听话许多。

- **六维（真实属性）**：力 281 / 敏 293 / 体 260 / 智 249 / 魅 2
- **核心能力**：超高速飞行（掠空之翼 → 死亡之翼袭杀）+ **风暴龙焰**（风 + 太阳焰 + 灵魂三重攻击特性 · 附治疗遏制 48h -80% · 三重抗性判定全不过则防御 -90% + 10% 概率即死焚灰）

- **用途**：龙骑状态对付无飞行能力的敌人 / 空中主力;猎辉光之神时因怕运势不稳、苏晓不以龙骑状态出战
