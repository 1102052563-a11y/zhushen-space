# 窝金

[全职猎人世界](../世界/任务世界/全职猎人.md) **[幻影旅团](../势力/index.md)（蜘蛛）成员**——满头白牙的壮汉，**强化系念能力者**，一身钢铁般的肉体。袭击拍卖会时与 [侠客](../势力/index.md)、[信长](../势力/index.md)、[玛奇](../势力/index.md) 等负责真正抢夺拍品。

## 能力

- **钢铁之躯**：身体强度极高，**子弹、火箭弹打在身上都无效**（被乱枪扫射只是被肉体摊开）；能**用牙咬住射向头颅的子弹**（留下牙印）。
- **怪力**：一记耳光就能把人脖颈扭成 90°；抓着人腿一甩便把脑浆甩成浆糊，「虎入羊群」般冲进人群屠戮——平时穷凶极恶的黑帮成员在他手里如同布偶。

## 历程（十卷37章起）

旅团在独眼巨怪出没的山谷被上百名黑帮成员「围困」（实为旅团居高临下戏耍）。窝金主动跳下、独自迎战——黑帮头目朝他头开枪反被他咬住子弹、一耳光打死，随即**单人屠戮整群黑帮**，把围攻打成溃散。

## 山谷之战 · 死亡（十卷38–43章）

十老头派 5 名阴兽（蚯蚓 / [病犬](病犬.md) / 豪猪 / 蝙蝠 / 刺猬）来试探旅团，窝金一人压制群殴：**生撕阴兽蝙蝠**（活扯双肉翅 + 拽头·「和撕蝴蝶差不多」）、再用捏碎的蝙蝠头骨当霰弹**轰杀阴兽刺猬**（前后不到 5 秒）。

[苏晓](苏晓.md) 到场后，窝金成了苏晓的眼中钉：

- **被克制**：苏晓持刀 + [反击盾](../力量体系/技能.md)，对窝金这种纯肉体念能力者天克——挡下窝金重拳的瞬间以 [影刺](../力量体系/技能.md) 贯穿其胸口。
- **「最强组合」自动操作**：危急下 [侠客](侠客.md) 把天线插入窝金口腔、耗尽自身念能量，让窝金进入 **「自动操作」**状态（金色念气护体如「超级赛亚人」·力 / 速 / 念 100% 凭本能发挥·攻击不经大脑·持续 9 分钟·事后须卧床一周）——窝金一拳把阴兽蚯蚓**轰成渣**、又一拳把苏晓打飞撞穿山体。
- **死亡**：苏晓饮 1 号药剂回血，与窝金硬碰硬——把全部法力灌入斩龙闪后发先至，**擦身斩断窝金一臂**（自己也故意断右手·有 [黑玛丽](黑玛丽.md) 在可补）；继而膝砸其肩、**双指挖出窝金双眼**（自动操作靠视觉锁敌，挖眼即令其强制终止）→ 斩断脖颈。**结算：[苏晓](苏晓.md) 因杀「自动操作（能力大增）」的窝金获奖励加成，世界之源 +10.2%（→39.7%）、旅团声望 −700（→−900·死仇）、掉落淡金宝箱**。窝金一死，[侠客](侠客.md) 的念能量随之全部消散、当场暴毙。

## 关系
- **[幻影旅团](../势力/index.md)**：强化系主战成员；与 [信长](信长.md) 是性格相投的挚友（最默契搭档）、与 [侠客](侠客.md) 是「最强组合」；另有 [飞坦](飞坦.md)、[富兰克林](富兰克林.md)、[小滴](小滴.md)、[玛奇](玛奇.md) 同行。
- **[苏晓](苏晓.md)**：把信长之死的仇全记在他身上、拼死来战，最终被斩杀。
- **[酷拉皮卡](酷拉皮卡.md)**：原著中的死敌（窟卢塔族灭族之仇）。
