# 卡洛斯（边塞血狮）

[维斯星球·多亚古大陆](../世界/任务世界/维斯星球-多亚古大陆.md) 龙腾帝国**铁血军团**的副军团长——身材魁梧、一身血色铠甲、血腥气不弱于 [苏晓](苏晓.md)，人称「**边塞血狮**」。出身卑微（**贵族与妓女之子**·被同僚鄙夷），却**镇守黑土要塞二十年**（从小队长做起·无人比他更懂这座要塞）、战功无数。性子刚硬、深谙「慈不掌兵」（一声令下便有数十万生灵战死、却毫不手软）。

## 转正军团长（十二卷4–5章）

顶头上司 **马克侯爵**（铁血军团军团长·世袭废物·把镇守边塞的苦差都丢给卡洛斯）逼他**一周内夺回半月前失守的黑土要塞**、却只肯给八万兵（另二十万被抽去支援希尔大峡谷的公爵）。卡洛斯怒而要求至少二十万兵、险些与侯爵翻脸。关键时刻**王命传到**：查实马克**贪墨黑土要塞翻修费**（14 年前翻修案）、当场**降马克为伯爵**、**卡洛斯转正铁血军团军团长** + 增援 30 万。他随即统军压向黑土要塞——明知攻要塞将以海量人命为代价，仍判断「宁可部落驻守要塞、也怕它们从内部炸毁」，下令强攻，冷兵器大战就此爆发。开战中他在高点督战、**注意到一公里外杀出堆积如山尸体的 [苏晓](苏晓.md)**（以为是新兵），赞其勇猛、下令「若这新兵能活到明早、提他为小队长」。

他指挥「箭雨清场 → 军阵冲锋 → 箭雨再清场」的循环战术、稳步取得优势。

## 赏识苏晓 · 破格提拔（十二卷13章）

战后他被 [苏晓](苏晓.md) 煮的「秘制海鲜汤」香味吸引、盘坐过来豪饮（毫不忌惮被毒），自报家门「我叫卡洛斯、你的军团长」。苏晓直言「我会杀人、军队刚好发挥我的才能」（而非「为帝国而战」），卡洛斯反而大笑赏识——**他只看能力、不在乎身份种族**（自己便出身卑微），当场**破格提拔白夜（苏晓）为小队长**（管最多 200 名打散的散兵·自己去找·身份凭证战后再说）。苏晓由此获得龙腾帝国官职、激活战功系统（战功 1436·凭杀千余士兵所得）。苏晓暗忖「这人有独特人格魅力、实力强大、我不是他的对手」。

## 亲率冲锋 · 夜攻黑土要塞（十二卷17章）

30 余万增援到位后，他连夜集结大军强攻黑土要塞——**亲自带头冲锋**（「『给我冲』和『跟我冲』是两个概念」·这是世袭废物马克伯爵永远做不到的事）。战前动员真挚不矫饰：「**害怕不丢人……这不是为了保卫帝国、狗屁、是为了给你们所爱之人安定幸福的生活·跟老子杀上去**」，士气大振。他的战术狠绝——**用尸体 + 石块堆到与要塞等高**才可能攻上（否则在矮人「护城重轮弩」下凭攀梯必死）。为防自己冲锋战死动摇军心，他备下后手：让情同手足的副官**罗纳**持炼金道具**伪装成他**、在后方稳住军心 + 接任指挥。

## 「岩浆蒸汽」铁血计 · 清蒸三万部落军（十二卷21–23章）

攻坚战初期帝国军在部落「护城重轮弩」+ 绿磷火下死伤逾八万、军心溃散——这其实全在卡洛斯算中。他战前花 **9 万金镑**从炼金协会购入暗藏「**热冷换置炼金阵图**」的特殊石块、混进堆要塞的石坡（部落用绿磷火把石坡烤成岩浆坡时、阵图随之就位）。待时机成熟，他令隐身的暗夜精灵公主 [安娜](安娜.md) 激活阵图、副官罗纳掌石板控制：**岩浆喷涌至与要塞齐高 → 骤然冷热交替迸出过热蒸汽**，借西风**「清蒸」死了驻守要塞的三万部落军**（兽人 / 地精 / 矮人 / 巨魔无一幸免·要塞上出现「四道清蒸菜」）。代价是大量帝国士兵被岩浆吞噬——他**明知却不提前告知**（防计划暴露·「能让士兵牺牲、但不会让他们白白牺牲」）。其名言尽显铁血：「**血债血偿、老子先祖建造的要塞、部落杂碎休想夺走**」「战争中没有卑鄙、死人不会说敌人卑鄙」。蒸汽散后他重整二十万军、浇水冷却出直通要塞顶的斜坡，与部落精英部队展开**高地战**。

## 重伤将死 · 身后布局（十二卷38章）

高地战中他**单挑牛头人首领阿姆斯特 + 冰霜巨魔（巨魔贤者）「一死两伤」**——斩杀冰霜巨魔、阿姆斯特断一只牛角逃走，自己腹部被牛角顶了个血洞。更阴的是：部落联盟的地精发明家千方百计让他人把一瓶**下了毒的炼金药剂**送到他手里（赌他受伤时服下·这才敢派两首领来杀他），被他识破踩碎。但旧伤新创叠加，他**自评活不过三个月**——已开始考虑身后事：认定副官罗纳「很适合接替军团长」，让 [安娜](安娜.md) 把阿姆斯特的断角转交其父（化解旧怨）。他右眼偶现血红光芒、如炼狱恶魔，喝止落泪的安娜「**就算老子死了、也别在老子尸体前哭**」。安娜誓「一定有办法」——并暗暗盯上军需官 [尼古拉斯·凯撒](尼古拉斯·凯撒.md)（限他两个月内想出救命之法、否则分尸）。

## 关系
- **龙腾帝国 · 铁血军团**：副军团长 → 军团长（镇守黑土要塞 20 年）。
- **马克侯爵 / 伯爵**：原顶头上司（世袭废物·贪墨翻修费被降级）。
- **副官罗纳**：相处十几年、情同手足（备作其战死替身 / 接任者）。
- **[安娜](安娜.md)**：暗夜精灵公主·恋人（为救其性命不惜疯狂·暗夜精灵「终身绑定、只有丧偶没有分手」）。
- **[苏晓](苏晓.md)**：其麾下「三等兵」→ 破格提拔的小队长（赏识其能力）。
