# 莉莉姆（恶魔族·魅魔外交官）

[虚空角斗场](../世界/任务世界/恶魔古堡游戏.md)（恶魔古堡第五场）上主动找到 [苏晓](苏晓.md) 的女恶魔——与人类女性相似度超 90%（只额头小犄角 + 尖耳泄底）、自称**莉莉姆**、**恶魔族高等族系「魅魔族」**、芳龄 125 岁（约合人类 26 岁）。她不嗜战、爱好和平，在暴脾气的恶魔族里是异类，故担任**魅魔外交官**（负责与各族和谈、避免恶魔族与人开战）。

## 恶魔族盟友信使（十四卷54章）

她从背后抱住苏晓搭讪、险些被屠戮短刀刺穿下颚，靠一句「**我是恶魔族**」止住——她父亲常讲「灭法之影与恶魔族的盟友关系多么可靠」，她从没见过活的灭法之影。她以非参赛者身份混进选手席传话（来得晚是受 [爱丽丝](爱丽丝.md) 阻挠），核心是**两条善意警告**：

- **「踏足虚空太早会死」**：以苏晓现在的战斗力进虚空必丢命，他原来待的地方（轮回乐园）更安全。
- **「远离奥法永恒星」**：**奥法永恒星 = 施法者的圣地**（一颗超大的永恒星球），以苏晓灭法之影的身份踏足那里必尸骨无存。

她最在意的是苏晓**有没有传承「灭法的信念」**——听他说没有便如释重负（「老一辈总想恢复荣光、互相打来打去，还是和平好；你若传了灭法意志、振臂一挥会有很多老恶魔愿以盟友身份随你杀向奥法永恒星」）。可当苏晓露出对奥法永恒星的兴趣、以及那桀骜犀利的眼神（**与她家古堡里「初代灭法」壁画上的目光一模一样**），她又惊觉：这是个「**拥有刀魔般狂暴能力、内心却冷静的灭法者**」，比传了信念的还可怕。

## 恶魔族的「试探」 · 带苏晓见树贤者（十四卷57–64章）

她转达父亲的答谢之意：父亲当年曾被某位灭法者在战斗中救过一命、一直无从报答（灭法者消亡太快），如今让莉莉姆问苏晓「有什么要求（能力 / 珍贵资源）」。这其实是**恶魔族对苏晓性格的试探**——苏晓没索要宝物（「宝物我之后自己去抢」让她直翻白眼·骂灭法之影是土匪），而是请她**帮忙找一个能速成炼金生物制造学的机会**、还主动提出回报，借此表态「想做能合作的盟友、不占免费便宜」。

为此她**自费**（满脸肉疼·念叨要回家找二哥报销）布起「**恶魔虫洞**」远距离传送阵（倒五星法阵·用恶魔之力非魔法），把苏晓传到偏僻星球的「**迷火森林**」去见炼金师 **[树贤者](树贤者.md)**（她幼时长住之地·树贤者疼她如准传人）。她不擅长远距离传送、阵很不稳，自己一路被颠得狂吐、却惊见苏晓「**对恶魔虫洞的抗性怪物级**」（因常被轮回乐园远距传送·已生抗性）。

关于**黑渊**她守口如瓶（怕苏晓的探索欲驱他去送死）——只透露自己进黑渊留下一道横贯胸口的疤（靠父兄护着才活·当时的她都比现在的苏晓强）、「实力提升几十倍才够独探黑渊」「不同人进入黑渊各有不同收获」。她越发觉得苏晓那桀骜眼神像她家古堡里的初代灭法壁画、是个比传了信念者更可怕的存在。

## 黑渊驰援 · 妹妹莉莉斯（二十三卷4–32章）

苏晓进 [黑渊浅层](../世界/任务世界/黑渊浅层-赔罪者之殇.md) 做晋升任务时，她（受大恶魔 [沃波尔](黑渊配角.md) 安排）**派亲妹妹 [莉莉斯](莉莉斯.md) 去增援**——给苏晓一颗定位晶体、又用「映空镜」远程叮嘱「我妹看着呆、其实不傻、别抛弃她、她会帮你战斗」（被莉莉斯一脚踹碎镜子）。她自己脖子上还留着被莎咬的牙印「耻辱」。事后她联络苏晓**力促「黑枫叶长期等价交易」**（恶魔族视黑枫叶如绝种珍宝·能造恶魔之力中枢）——笑容逐渐变态地要把妹妹「附赠」（「998 片黑枫叶我妹带回家」），核心是表达长期合作的诚意。

## 关系
- **[莉莉斯](莉莉斯.md)**：亲妹妹（战斗系恶魔·派她赴黑渊增援苏晓兼历练）。
- **[沃波尔](黑渊配角.md)**：安排增援事宜的大恶魔。
- **[苏晓](苏晓.md)**：她奉恶魔族之意传讯 / 试探 / 牵线（带他见树贤者）的「世代盟友」（灭法之影）；对其「没传灭法信念」颇有好感，却又怕他重振灭法；黑渊后力促长期交易。
- **[树贤者](树贤者.md)**：她幼时长住迷火森林、视为和蔼长者的炼金师（疼她如准传人·她却嫌炼金麻烦）。
- **恶魔族 / 灭法之影**：世代盟约（95% 恶魔族暴脾气、一言不合就动手，或正因此与同样极端的灭法之影结盟）。

## 黑枫树交易 + 查刺客（二十九卷·补充）

苏晓晋六阶后，莉莉姆急需 **黑枫树枝干**（30 克·救某位重病至亲）——苏晓自家种的几乎零成本、远程交易过去。莉莉姆回礼：① 查出刺杀苏晓的是元素族刺客**尤·阿奇德**（阿萨卡组织报复 + 奥术永恒星 [瑟菲莉娅](瑟菲莉娅.md) 雇佣·已被恶魔族解决）；② 早先送的**红宝石项坠**=感应虚空空间波动 + 进虚空后供她定位 + 联络器；③ 转达恶魔族长老 **沃波尔** 的回礼——一把封着「疯魔」的**灭法者的刀**（「那原本就是灭法者的东西」·待苏晓够强、不会被刀里疯魔吞噬时再送）。
