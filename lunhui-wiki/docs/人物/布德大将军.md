# 布德大将军

**布德大将军** 是 [斩·赤红之瞳](../世界/任务世界/斩赤红之瞳.md) 世界帝国**常驻皇宫、统领近卫军**的大将军（四卷54章）。

## 简介
实力仅次于 [艾斯德斯](艾斯德斯.md)（二人未正式交手·苏晓估约「四六开·布德四、艾斯德斯六」）。性格**古板**、一心为帝国服务，对大臣也毫不客气、被艾斯德斯与大臣私下称作「**老顽固**」；恪守「武将不干涉朝政」的祖训，故对大臣架空小皇帝、艾斯德斯只顾打仗的乱象**放任不管**。

- **帝具「雷神愤怒·亚得米勒」**（元素类·与艾斯德斯的恶魔之粹同类·操纵**雷电**）：他在皇宫时，会在小皇帝寝宫附近布**静电网**，外人想潜入无异于做梦。

## 与苏晓的关系（间接）
正因布德大将军坐镇皇宫、皇宫密不透风，苏晓迟迟不敢潜入夺钥匙。直到**革命军兵临城下、布德大将军被调去帝都城墙前线**（帝国阵营的契约者也随之上前线），皇宫防御出现空档，苏晓才得以混入绑架小皇帝、完成主线（四卷54章）。

## 关系
- [艾斯德斯](艾斯德斯.md)——帝国另一王牌（实力略胜他）；大臣 / 小皇帝——他效忠的帝国（虽看不惯乱象却不干政）。
- [苏晓](苏晓.md)（白夜）——其离岗给了苏晓潜入皇宫之机（未正面交手）。
