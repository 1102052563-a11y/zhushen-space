# 赫里斯塔

**赫里斯塔** 是 [进击的巨人](../世界/任务世界/进击的巨人.md) 世界训练兵团 104 期成员，表面只是一名普通女兵，实则身负**墙内王族血脉**（三卷41章）。

## 简介
金发、体型娇小的「傻白甜」少女；她是墙内**王族与佣人**所生的私生女，因未经婚姻诞生而受王族排挤，却是**嫡系王族血脉**——有资格成为墙内之王。**关键**：唯有具王族血统者才能彻底发挥**始祖巨人之力**，故她是马莱间谍组奉命要一并抓走的目标（「赫里斯塔」即上头交代要抓的人）。

性格矛盾、一味为他人着想，因童年不幸而几乎没有「活下去的目标」、甚至觉得「就算死也无所谓」。被 [尤弥尔](尤弥尔.md) 深爱。

## 被俘（三卷41–43章）
[尤弥尔](尤弥尔.md) 趁城墙混乱变身颚之巨人、将她吞入口中跳下城墙，本想带她投奔马莱。苏晓一行将尤弥尔弄残后，[莱纳·布朗](莱纳·布朗.md) 出面求情「把她绑起来、别伤害她」（莱纳对她亦有爱慕），苏晓遂将她五花大绑、堵口带走（因她体内无巨人之力、不能靠断肢控制，只能捆绑）。途中莱纳还喂她吃烤肉。

## 关系
- [尤弥尔](尤弥尔.md)——深爱她的颚之巨人女子。
- [莱纳·布朗](莱纳·布朗.md)——对她心怀爱慕、求情勿伤她。
- 墙内王族——其生父一系（私生女、受排挤却为嫡系血脉）。
- [苏晓](苏晓.md)（白夜）——掳走她的猎杀者（视她为始祖巨人之力的关键筹码）。
