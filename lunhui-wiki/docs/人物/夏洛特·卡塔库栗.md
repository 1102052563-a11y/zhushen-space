# 夏洛特·卡塔库栗（甜点四将星之首 · 年糕果实）

[BIG·MOM 海贼团](../势力/index.md) **甜点四将星之首**——大妈麾下除玲玲本人外**最强的人**、近乎「仅次于四皇的强者」。身高近 5 米、体型匀称，赤色短发、毛皮领巾遮住下半张脸，大臂红色纹身；只是站着便透出冷酷凶悍。性格看似孤傲冷峻，**实则极在乎部下与亲人**（与生母玲玲不同）。

- **年糕果实（超人系·已觉醒）**：能造 / 操控年糕（覆武装色霸气后硬度远超金属）、**近乎无视物理攻击**（除非攻击附武装色）；觉醒后能把周围一切物体「**年糕化**」、改造战场环境。
- **🔥 见闻色霸气 = 预知未来**：他的见闻色强到**能预知短暂的未来**——这是 [苏晓](苏晓.md) 眼中「最无解」的能力。苏晓自评：正面打克力架有四五成胜算（开猎魔时刻克力架九成死），但**对卡塔库栗，若猎魔时刻在冷却就有九成会死**，故根本不会让自己陷入与他正面单挑的境地。

## 德岛海战 · 救弟克力架（十八卷21–22章）

[苏晓](苏晓.md) 隔 4 公里狙击重创其弟 [克力架](夏洛特·克力架.md)（后脑头骨被轰碎）——卡塔库栗**预知到子弹落点**、瞬间从 6 号登陆点抽身赶来，以年糕墙 / 年糕护腿连续挡下苏晓的致命狙击。苏晓与 [多弗朗明哥](多弗朗明哥.md)（线线果实觉醒·也已觉醒超人系）联手「**围伤打援**」：苏晓锁定多弗朗明哥引其注意 → 布布汪巨石写「拖住卡塔库栗」→ 多弗拖住卡塔、苏晓继续狙克力架。沙滩半边年糕化、半边丝线化，二人缠斗——卡塔一度把多弗打飞撞进巨岩、又用武装色三叉戟一击秒掉一名来捡便宜的透化系四阶契约者。但**护着克力架就无法全力战多弗**，是他的两难。

## 三人围攻 · 苏晓 + 多弗联手（十八卷25–29章）

苏晓亲临 7 号登陆点加入战团（多弗朗明哥已处下风·卡塔速度 / 力量 / 果实 / 三色霸气 / 预知全面占优·难击中本体）。这场缠斗揭出卡塔库栗与年糕果实的诸多细节：

- **年糕果实（超人系）**：[恶魔果实价值](../力量体系/技能.md)通常自然系 > 超人系 > 动物系（非绝对·超人系上限最高、顶级强者多是超人系如白胡子），年糕果实带「**类元素化**」能力，弹力 / 粘合 / 可塑被卡塔开发到极致——身体能年糕化避开斩击、把腿扭成「U 形」绕过武器、用年糕束缚敌人、临时把周围环境年糕化（觉醒）。
- **🔑 预知的弱点**：他**能预知未来、却不一定能改变未来**；尤其**无法预知「必定发生、自己躲不开」的事**（如多弗附武装色的那一脚·他躲不开就只能靠见闻色「感知」而非「预知」）。苏晓 + 多弗正是靠**前后合围 + 让卡塔分心护重伤的克力架 + 防范隐形的 [布布汪](布布汪.md) / [阿姆](阿姆.md)**，逼出这种「必定发生」的破绽。
- **怕两样东西**：① 苏晓的 [海楼石金属棍](../力量体系/装备道具.md)（海楼石是果实能力者的最大克星·宁挨武装色一脚也不碰它）；② 苏晓的**青钢影**——卡塔预知到「一旦碰上青钢影能量，会短时无法年糕化 + 剧痛，甚至失控任打本体」（详见下「青钢影克果实者」）。故卡塔视苏晓**比多弗危险几个档次**、一直率先攻击他。
- 战况胶着：卡塔一拳打裂苏晓的能量盾 / 险些打裂其肺、又把苏晓与多弗轮番打飞；但他护着克力架就放不开手脚，三人难解难分。

## 围岛 → 奉母命撤围（十八卷36–39章）

阿波罗的威胁下，卡塔库栗改以「最稳妥、己方损失最大」的方式打——**从巨岩区围岛**（分散 3~5 人小队拉成环形战线），把德雷斯罗萨围得水泄不通（再给他一天就能攻下）。但 [大妈](夏洛特·玲玲.md) 的老巢被苏晓 + 多弗炸了、后院的未知势力又闹得地盘岌岌可危——大妈遂电令他停战撤围。卡塔不甘（「克力架就死在我面前、这仇不会这样算了」），却也敏锐察觉母亲的反常 = 地盘已危到极点，只得遵命。私下扯下领巾时露出俊朗面容 + 满口鲨鱼尖牙（4 颗犬牙尤其狰狞）。他把多弗朗明哥、白夜、杰克的名字咬得咯咯作响。

## 谈判桌上 · 亡弟果实之痛（十八卷44–45章）

歇斯海域谈判时他站在母亲身后。[苏晓](苏晓.md) 当场掏出 [饼干果实](../力量体系/装备道具.md)——卡塔库栗**前所未有地暴怒**（连大妈都没见过他这般失态）：这是亡弟 [克力架](夏洛特·克力架.md) 吃下的果实，他记得当年茶会上那个吃下果实、塞给他一块「很硬、不好吃」饼干、打针时哭着要离家出走的小屁孩。当苏晓以「捏碎即爆的阿波罗」逼大妈时，是卡塔库栗**预知到「苏晓真的敢引爆」**、一把抓住要冲上去的大妈、劝她坐下来谈。
