# 棘拉

[苏晓](苏晓.md) 麾下的**虫族女王**（虫族母体）——外形近乎人类少女、脖颈两侧有几条浅紫色平滑甲壳。她本是「棘虫星」的原生虫族（名字即由此而来），原本天赋点也是「5 点战斗 / 2 点资源 / 3 点族群延续」的普通配置，却在**幼生期被苏晓逮住、重新洗成「10 点全加战斗」的极端战争族群**——正因如此，她把老家棘虫星祸害到惨状、又几乎推平七阶世界原始大陆（虚空之树多次警告苏晓把她接走·被无视）。她极度嗜睡爱宅、走出虫巢晒个太阳就算「郊游」、离开大本营就是「出国旅行」级别，但只要苏晓开口就绝对服从（幼生期不听话被吊起来抽疼哭都没用）。

## 恶魔虫族 · 关键属性（47卷·潘多拉星）

被召来潘多拉星后因世界体系不同**被削弱一次**（从七阶顶尖跌到七阶出头），但换来的是更高的成长上限（可冲击她从未达到的「主宰级」）。她的族群是**恶魔虫族**——虫族单位除英雄级外**几乎无个体意识**、只听最高意识、是纯粹的战争武器（利在纯粹、弊在失去统御即陷混乱无差别攻击）。核心能力:

- **虫族意志（EX）**:对下属虫族绝对掌控。
- **进化·初（EX）**:催生一阶虫族「螳甲」（螳甲可建虫巢）。
- **恶魔虫族（MAX）**:受恶魔之力影响·本属群虫族单位带恶魔之力的辅助 / 战斗能力（恶魔兽、[恶魔焰龙·巴巴托斯](../力量体系/index.md)、利维坦等）。
- **召唤属从契约（EX）**:⚠ 击杀主导者（苏晓）前，任何人无法与她签从者 / 召唤契约（轮回乐园公证·绝对优先）。
- **感知泯灭（Lv.70）**:无视七阶及以下的感知与侦测。
- 六维:力 36 / 敏 37 / 体 190 / 智 210 / 魅 206（体智魅为真实属性·力敏偏弱「打不过贝妮和布布汪」）。

## 潘多拉星 · 魔鬼开局

苏晓给她吃下 [深渊石](../力量体系/装备道具.md)（大幅提升虫族进化能力与潜力上限·棘拉委屈巴巴噎着吞下），又灌进 785 颗活性结晶作启动资金——她的螳甲（含少数有个体意识的老螳甲**布鲁**）几小时内就把虫巢从一阶怼到**七阶**，让苏晓的主线任务「萌芽」以 **695% 完成度**一口气跳到第五环。别的契约者从一阶虫巢起步，苏晓这边直接七阶虫巢 + 完善战斗单位开局。她只会自己挖矿（力敏太弱挖一年也挖不出啥）、麾下虫族不会挖矿，于是苏晓改走「吃肉」路线——放恶魔兽平推邻居虫族女王抢生物能。目标:把她从女王级喂到**主宰级**，才有资格和帝国掰手腕。

## 关系

- [苏晓](苏晓.md)：幼生期把她洗成极端战争族群、召来潘多拉星建虫巢的契主（绝对优先的召唤属从契约锁定人）。
- **布鲁**:她属从里少数有个体意识的老螳甲（甲壳满是伤痕·见苏晓会高兴转圈）·与恶魔兽·亚巴顿、恶魔焰龙·巴巴托斯同类。
- [布布汪](布布汪.md) / [阿姆](阿姆.md) / [巴哈](剧毒魔鹰·巴哈.md)：苏晓小队伙伴（棘拉的力敏弱到打不过布布汪）。

## 🆕 晋升母皇级 · 八阶虫巢 · 融宿虫普罗斯（47卷6–8章）

灭怒甲虫巢、吞下 10 万工兵后，[苏晓](苏晓.md) 让棘拉晋升到**母皇级**（虫巢随之升到**八阶**·需 85000 点生物能·因吞深渊石大幅降需求）。八阶母巢诞生第四位英雄级单位——研究室里的**融宿虫**（虫族科学家·据基因储备开发虫族建筑），它强烈的自我意识让它主动申请命名、自称**普罗斯**（前三位英雄级:螳甲布鲁、首领级恶魔兽亚巴顿、首领级恶魔焰龙巴巴托斯）。普罗斯把菌毯扩到直径 5 公里，建起代表性的「棘星螺旋塔」（140 米·母巢 +12 万生命 / +196 外防）、地窝、生物反应垛、杀戮哨塔。她还怕晋升被邻居虫族母皇**卡拉**感应到（同区两母皇精神力会相叠），所以先攒 34 万工蝎底蕴。⚠ 她从母皇级冲**主宰级**需骇人的 **870 万点生物能**（87 万单位生命矿石）——这是苏晓十天内要凑齐、以顶住「幽冥」入侵的死线。她还有个反差萌:纯食肉、迷之**恐高**（向下看超 30 米就软成面条）。

## 关系（补充）

- **普罗斯（融宿虫）**:她八阶母巢诞生的第四位英雄级单位、自己命名的虫族科学家。

## 🆕 晋升虫族主宰 · 母巢终极进化（47卷17章）

集齐 870 万点生物能后（灭盖伊 + 卡拉源矿分巢 + 公司 60 万单位矿石），[苏晓](苏晓.md) 让棘拉晋升到**虫族主宰级**——成为本世界唯一的主宰级（卡拉那种世界级生物不算在内）。母巢随之终极进化（长到 200 米·棘星螺旋塔 230 米）:精英恶魔兽深度增强、菌毯从「只吞生物」拓展到**吞万物**（虫族建筑 / 岩石 / 树木 / 金属矿石都能吸收提元素）、新解锁运输型虫族**宿主**（60 米水母状·载 5~7 倍货 / 抵 25~30 只孢子坦克 / 自带高级隐匿），最关键的是**唤醒了两位首领级英雄的灵魂**——首领级恶魔兽·亚巴顿、首领级恶魔焰龙·**巴巴托斯**（苏晓最强龙坐骑·飞行力巅峰·龙焰强抑生命恢复·龙骑状态能陪苏晓扛界雷持「界雷枪」·尚待母巢培育匹配肉体）。晋升后棘拉短暂休眠。主线任务「崛起」第五环完成、得灵魂晶核 ×4。

## 关系（补充）

- **亚巴顿 / 巴巴托斯**:随她晋主宰级、母巢进化而被唤醒灵魂的两位首领级英雄单位（巴巴托斯是苏晓对抗卡拉的关键龙坐骑）。
## 👑 晋升女皇级 · 进化之路开启（47卷36–37章）

击退幽冥后，[苏晓](苏晓.md) 用【起源石·银皇后】当引导物让棘拉晋升**女皇级**——唤醒起源石里数千年不灭的虫族女皇「**银皇后**」意识（棘拉的同源远祖·棘虫星曾吃光一个世界的强者），让棘拉全程感知起源石质变、学会「怎样走向女皇级」（银皇后想夺棘拉躯体、却先撞上苏晓 620 灵魂强度的灵魂之印被弹开、又被苏晓的「加强版世界公敌核心」引世界意识强制放逐进永光世界）。晋升后外形 / 虫巢 / 恶魔兽都无变化，真正的收获是**进化之路**:菌毯升级到能微量摄取敌人战死飘散的斑杂灵魂能量、经母巢「魂核」转化成「**进化点**」——可给虫族单位 / 母巢 / 她自己**永久加点**（模仿苏晓的猎影天赋·加恶魔兽整个群体、加母巢爆兵、加自己精神力……），从此**越战越强、质量也涨**。⚠ 缺点:腐化者灵魂太弱吸不到，得杀幽冥 2 级（灰甲勇士 / 灵魂扭曲者）/ 3 级（幽冥战士 / 灵魂巫师）兵种才有进化点。她还是恐高（骑巴巴托斯吓得逼它落地狂奔）、爱睡懒觉不认自己是当事人。

## 关系（补充）

- **银皇后**:起源石里数千年不灭的虫族女皇、棘拉的同源远祖（唤醒助棘拉晋级后被放逐永光世界）。
