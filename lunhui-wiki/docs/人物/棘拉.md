# 棘拉

[苏晓](苏晓.md) 麾下的**虫族女王**（虫族母体）——外形近乎人类少女、脖颈两侧有几条浅紫色平滑甲壳。她本是「棘虫星」的原生虫族（名字即由此而来），原本天赋点也是「5 点战斗 / 2 点资源 / 3 点族群延续」的普通配置，却在**幼生期被苏晓逮住、重新洗成「10 点全加战斗」的极端战争族群**——正因如此，她把老家棘虫星祸害到惨状、又几乎推平七阶世界原始大陆（虚空之树多次警告苏晓把她接走·被无视）。她极度嗜睡爱宅、走出虫巢晒个太阳就算「郊游」、离开大本营就是「出国旅行」级别，但只要苏晓开口就绝对服从（幼生期不听话被吊起来抽疼哭都没用）。

## 恶魔虫族 · 关键属性（47卷·潘多拉星）

被召来潘多拉星后因世界体系不同**被削弱一次**（从七阶顶尖跌到七阶出头），但换来的是更高的成长上限（可冲击她从未达到的「主宰级」）。她的族群是**恶魔虫族**——虫族单位除英雄级外**几乎无个体意识**、只听最高意识、是纯粹的战争武器（利在纯粹、弊在失去统御即陷混乱无差别攻击）。核心能力:

- **虫族意志（EX）**:对下属虫族绝对掌控。
- **进化·初（EX）**:催生一阶虫族「螳甲」（螳甲可建虫巢）。
- **恶魔虫族（MAX）**:受恶魔之力影响·本属群虫族单位带恶魔之力的辅助 / 战斗能力（恶魔兽、[恶魔焰龙·巴巴托斯](../力量体系/index.md)、利维坦等）。
- **召唤属从契约（EX）**:⚠ 击杀主导者（苏晓）前，任何人无法与她签从者 / 召唤契约（轮回乐园公证·绝对优先）。
- **感知泯灭（Lv.70）**:无视七阶及以下的感知与侦测。
- 六维:力 36 / 敏 37 / 体 190 / 智 210 / 魅 206（体智魅为真实属性·力敏偏弱「打不过贝妮和布布汪」）。

## 潘多拉星 · 魔鬼开局

苏晓给她吃下 [深渊石](../力量体系/装备道具/index.md)（大幅提升虫族进化能力与潜力上限·棘拉委屈巴巴噎着吞下），又灌进 785 颗活性结晶作启动资金——她的螳甲（含少数有个体意识的老螳甲**布鲁**）几小时内就把虫巢从一阶怼到**七阶**，让苏晓的主线任务「萌芽」以 **695% 完成度**一口气跳到第五环。别的契约者从一阶虫巢起步，苏晓这边直接七阶虫巢 + 完善战斗单位开局。她只会自己挖矿（力敏太弱挖一年也挖不出啥）、麾下虫族不会挖矿，于是苏晓改走「吃肉」路线——放恶魔兽平推邻居虫族女王抢生物能。目标:把她从女王级喂到**主宰级**，才有资格和帝国掰手腕。

## 🆕 晋升母皇级 · 八阶虫巢 · 融宿虫普罗斯（47卷6–8章）

灭怒甲虫巢、吞下 10 万工兵后，[苏晓](苏晓.md) 让棘拉晋升到**母皇级**（虫巢随之升到**八阶**·需 85000 点生物能·因吞深渊石大幅降需求）。八阶母巢诞生第四位英雄级单位——研究室里的**融宿虫**（虫族科学家·据基因储备开发虫族建筑），它强烈的自我意识让它主动申请命名、自称**普罗斯**（前三位英雄级:螳甲布鲁、首领级恶魔兽亚巴顿、首领级恶魔焰龙巴巴托斯）。普罗斯把菌毯扩到直径 5 公里，建起代表性的「棘星螺旋塔」（140 米·母巢 +12 万生命 / +196 外防）、地窝、生物反应垛、杀戮哨塔。她还怕晋升被邻居虫族母皇**卡拉**感应到（同区两母皇精神力会相叠），所以先攒 34 万工蝎底蕴。⚠ 她从母皇级冲**主宰级**需骇人的 **870 万点生物能**（87 万单位生命矿石）——这是苏晓十天内要凑齐、以顶住「幽冥」入侵的死线。她还有个反差萌:纯食肉、迷之**恐高**（向下看超 30 米就软成面条）。

## 🆕 晋升虫族主宰 · 母巢终极进化（47卷17章）

集齐 870 万点生物能后（灭盖伊 + 卡拉源矿分巢 + 公司 60 万单位矿石），[苏晓](苏晓.md) 让棘拉晋升到**虫族主宰级**——成为本世界唯一的主宰级（卡拉那种世界级生物不算在内）。母巢随之终极进化（长到 200 米·棘星螺旋塔 230 米）:精英恶魔兽深度增强、菌毯从「只吞生物」拓展到**吞万物**（虫族建筑 / 岩石 / 树木 / 金属矿石都能吸收提元素）、新解锁运输型虫族**宿主**（60 米水母状·载 5~7 倍货 / 抵 25~30 只孢子坦克 / 自带高级隐匿），最关键的是**唤醒了两位首领级英雄的灵魂**——首领级恶魔兽·亚巴顿、首领级恶魔焰龙·**巴巴托斯**（苏晓最强龙坐骑·飞行力巅峰·龙焰强抑生命恢复·龙骑状态能陪苏晓扛界雷持「界雷枪」·尚待母巢培育匹配肉体）。晋升后棘拉短暂休眠。主线任务「崛起」第五环完成、得灵魂晶核 ×4。

## 👑 晋升女皇级 · 进化之路开启（47卷36–37章）

击退幽冥后，[苏晓](苏晓.md) 用【起源石·银皇后】当引导物让棘拉晋升**女皇级**——唤醒起源石里数千年不灭的虫族女皇「**银皇后**」意识（棘拉的同源远祖·棘虫星曾吃光一个世界的强者），让棘拉全程感知起源石质变、学会「怎样走向女皇级」（银皇后想夺棘拉躯体、却先撞上苏晓 620 灵魂强度的灵魂之印被弹开、又被苏晓的「加强版世界公敌核心」引世界意识强制放逐进永光世界）。晋升后外形 / 虫巢 / 恶魔兽都无变化，真正的收获是**进化之路**:菌毯升级到能微量摄取敌人战死飘散的斑杂灵魂能量、经母巢「魂核」转化成「**进化点**」——可给虫族单位 / 母巢 / 她自己**永久加点**（模仿苏晓的猎影天赋·加恶魔兽整个群体、加母巢爆兵、加自己精神力……），从此**越战越强、质量也涨**。⚠ 缺点:腐化者灵魂太弱吸不到，得杀幽冥 2 级（灰甲勇士 / 灵魂扭曲者）/ 3 级（幽冥战士 / 灵魂巫师）兵种才有进化点。她还是恐高（骑巴巴托斯吓得逼它落地狂奔）、爱睡懒觉不认自己是当事人。

## 😱 她为什么进不了「虫族意识聚集处」

⚠ **苏晓发现棘拉在他调配台前鼓捣、以为她研究药剂学**——⚠ **翻开笔记，标题赫然是「薯片的最佳炸制方法」**（⚠ 一副「我是不是很厉害」的骄傲小表情）;⚠ **被苏晓遣走首领级虫族后，她骄傲表情僵住、低头玩手指摆烂**:「我其实很喜欢药剂学的，然后学习药剂学时饿了，我就想着学以致用……」——⚠ **觉醒 Lv.100 醒来第一句话是「我饿了」**;⚠ **她还得了新能力:可在苏晓附近飘飞。**

## ⚡ 蜕变为「元素虫族」（54卷221–222章）

⚠ **本批最大的转折点**:苏晓在风海大陆看出，恶魔虫族之所以不能拉出去对抗深渊，**不是战争能力不够，而是「因果位阶」不够、过于被深渊体系压制**——虫族个体一旦被深渊侵蚀就会失控，甚至一路蔓延到棘拉本人；届时深渊虫族降临，那就更没得打。

他的思路是:**让棘拉成为自然元素体系的虫族**。因为深渊与元素是**同位阶**的力量——「说守护者能压制深渊能量那是扯淡，可只是阻挡、不让其侵蚀所保护者，一定可以」。

### 13 万点元素亲和力的由来

⚠ 这是苏晓自己都没料到的数字堆叠（第4113章）:

1. 灭法之影本身元素亲和力就高，但极限约 2000~3500 点；
2. 「至强·智力」被动:每提升 1 点至强智力，永久 +30 点元素亲和力 → 上限约 8000+；
3. ⚠ **上个大纪元，灭法者被称为「元素监守者」**——这在因果层面让灭法之影体系拥有极高的「元素亲和力**上限**」，可达 1~2 万；
4. ⚠ 他作为多件大爹级原罪物的封印者、又是当前纪元最后一名灭法者（灭法运势都在他这），**总运势 −2000 万点**带来了「元素亲和力上限」的惊人拓展；
5. ⚠ 最后一环:晋升无上属性时，「至强·智力」进阶为 **「无上·智力」——每提升 1 点无上智力，永久提高 3000 点元素亲和力**。45 点无上智力 × 3000 = 13.5 万。

**上限（庞大运势）＋ 实质（无上·智力）= 129982 点。**

### 【赋予之誓】的极限承压

这件灵魂大书库的誓约物（苏晓卖掉整座「虚空大书库」换来的）原理是**「力量概念」的置换转移**——不是先吸纳再转移，而是把苏晓的元素亲和力与棘拉**互换**，此后自身只作为**锚点**存在。

- 苏晓选择赋予的瞬间，就听到誓约物内部传来咔嚓一声脆响——**它承受了它不应该承受之重**（元素亲和力这东西本不该按「万」为单位计算:哪怕元素体系的强者，大后期也就几千，达到 8000 已非常惊人）。
- 苏晓当即把【赋予之誓】封印进「原罪之书」**第九页**，破碎戛然而止、**固定在 28.56%**。
- ⚠ 代价:一旦从原罪之书取出，这件星界最强几件誓约物之一会**立即破碎**；而当「力量赋予」解除时，苏晓的元素亲和力很可能**永久性降低 50% 以上，甚至 90%+**。他并不在意——就算降 99%，也还剩 1300 点；而「界雷」在深渊里本就引不下来（界雷是存在于星界的最强雷霆力量，深渊不仅不属于星界内，甚至笼罩着星界）。

### 蜕变结果

棘拉坐进元素秘术阵图中心，219 份「元素源质」作为勾勒媒介驱动秘术，构成三米高的**元素能量茧**。破茧后:虫族外甲壳彻底消失、满头紫色长发几乎垂到地面、身高约一米五，皮肤表面是作为最后防御的**无形精神镀层**，淡金色如眼影般的妆容让她有种说不出的威仪感——⚠ 下一秒打哈欠的模样让威严感尽失。

⚠ **她在向自己所仰慕与信赖之人的种族模样进化。**

!!! tip "元素虫族的三种基础天赋"
    1. **元素力量（被动）**:⚠ **将不会被任何力量侵蚀意志**，其主宰拥有对下位个体绝对的掌控力。<br>2. **恩赐生命（被动）**:获得更高的成长上限，但虫族精英个体**绝无法达到「巅峰位阶」**（无法获得巅峰席位，虫族母巢也绝无此类特性）；⚠ 唯一例外是特殊个体 **元素焰龙·[巴巴托斯](巴巴托斯.md)**——在凝聚足够元素力量后有 **0.0000000001%** 概率晋升到此位阶，且晋升失败必定彻底消亡、无法以任何方式重新培育。<br>3. **元素母巢（被动）**:虫巢可摄取「自然元素能量」提升，但需转化出对应数量的自然元素，否则将成为**元素之敌**、导致棘拉的元素亲和力大幅降低。

⚠ 元素虫族有**独立的衡量标志**:Lv.1~Lv.100 的虫族个体等级，因其群体性战斗方式，**无法与九阶／绝强／至强等战力位阶进行比对**。（且个体等级高于 Lv.95 者，存在时间必定不超过 30 个自然日，甚至更短。）

### 🧬 菌毯:整套体系的地基

第三条天赋的「元素反噬」隐患，苏晓用一个词解决:**菌毯**。

- 深渊与元素的关系是:深渊更强大时会侵蚀＋吞噬自然元素；而**元素力量更强时，则是包裹与同化深渊能量、将其转化为自然元素**。
- 于是把菌毯拉满，让它悄然包裹吸收战场上被击杀的深渊生物尸体，其体内的**「二代深渊能量」**（即经生命力特性活化过的深渊能量）就被同化为 **「次级元素能量」**——⚠ **这些自然元素是己方母巢自己转化产出的，甚至都还没进入星界环境，根本不存在「元素反噬」。**
- ⚠ 强度换算:若自然转化的元素力量强度为 100，这种「二代元素力量」约 **70~80**；但对为战争而生的虫族母巢，这已是非常精良的细糠——爆兵时用 40~50 强度的都算奢侈。

!!! note "【活性菌毯 Lv.130（满级）】"
    菌毯构成后，将吸引周边**巨大范围**内的自然元素依附其上，从而获得同化「深渊能量」的特性并让其向元素能量转变；但因固有特性，**仅可同化经生命力活化的「二代深渊能量」**。转化的同时也会摄取并储存生命力，转化为生物能储存于母巢内。

### 🚀 反攻

- **守护者的突破**:达到 185 次提升上限后，苏晓在战役联络平台征集提升方法，从各战区参战者免费寄来的珍宝中挑出【因果回响】（可回溯一小时内的因果变化）＋【元素之语·破限】（50% 冲破上限／50% 毁灭）。他先用后者观察破限原理，在虫族基因核心即将破碎时用前者回溯，随后以**药剂学把黑枫树产出液化、再以封印学精密勾勒**，让液态黑枫树产出攀附包裹「守护者核心」的每个细微部分作为**消耗性护层**——⚠ 一次次强行提升失败的代价由黑枫树产出承受。总计消耗 **3.55 个单位黑枫树产出**后破限成功。
- ⚠ 他为何不叫凯撒来操作:「牟利的事当然可以操作，但**这种需要在关键时刻委以重任、承受重压的环节，不能有任何操作**，就得实打实的资源猛堆——唯有堆得够结实，承受重压时才不会崩塌。」
- **爆兵**:母巢向地下发展（虚空之树批准了地下区域使用权限），并不断建立**次级虫巢**——虫巢越多爆兵越快、灭世之翼越多杀敌越多、菌毯转化的元素晶核越多，**滚雪球**。
- **收官**:灭世之翼吞下海量元素晶核，喷出掺杂「次级元素能量」的七彩龙焰。这不是用来直接同化深渊能量的，而是**让深渊蔓延核心区「蜂窝化」**——若把深渊侵袭比作一块钢铁、本世界元素能量比作火焰，那么这些次级元素能量是把这块黑暗金属变成内部满是蜂窝孔的铁渣，**既易燃，其内部的次级元素能量还是下方火焰最佳的助燃养分**。风海大陆的深渊侵蚀度由此从 96.2% 一路压到 **4.98%**。

⚠ 苏晓对棘拉未来的规划:一旦她成为轮回乐园的**契约者**（虚空之树已提出可从从者进阶为契约者甚至死亡游侠/战斗天使/先驱/处刑者，**被她拒绝——她只想当灭法者·库库林·白夜的从者**），麾下所有虫族个体就会有完善的召唤物公证，届时可与苏晓组队分战场功勋。

⚠ 她的元素亲和力**只会在一段时间内保持 13 万**——但足够她从「打破平衡的战力」成长为「决定主战场胜负的关键」。
