# 小花鱼（战地记者 · 直播主职工者）

轮回乐园里无人不知的一名**职工者**——17.8 岁的萌妹子，职业奇葩到「**战地记者**」：几乎没有攻击能力、**只有生存力**，靠「**记录战斗**」而非战斗本身变强。配合一件**灵魂绑定装备**，她能把所亲历衍生世界的**实时画面直播 / 传输回轮回乐园**供他人观看（免费观看 + 她爆笑的风格 = 人气极高，连五阶契约者都听过她）。是出了名的「爱作死」的妹子。

- **直播过的名场面**：金木研 VS 大守八云（[壁虎](壁虎.md)）、卡巴内尸潮、寄生兽进食、[赤瞳](赤瞳.md) VS [黑瞳](黑瞳-契约者.md) 两姐妹对战等——许多契约者正是通过她的直播知晓这些衍生世界的战况。
- **黑帆商会资助**：她是 [黑帆商会](../势力/index.md) 资助 / 旗下的人。

## 险被卖给苏晓（十七卷102章）

黑帆会长 [钱大胖](../势力/index.md) 欠 [苏晓](苏晓.md)（白夜）赌债 12 颗灵魂结晶〔大〕还不上，竟想**把小花鱼「卖」给苏晓抵 4 颗的债**（暗示其姿色、任苏晓摆布）。小花鱼被「赞助商」强行找来、察觉被卖险些哭出声——但苏晓**当场拒绝**（小花鱼是黑帆的人、不可信；苏晓只信 [布布汪](布布汪.md)、[阿姆](阿姆.md)、已故的海东），改向钱大胖要一把高品质手枪 + 8 颗灵魂结晶。两分钟后小花鱼被告知可以走了——虚惊一场、保住了清白。
