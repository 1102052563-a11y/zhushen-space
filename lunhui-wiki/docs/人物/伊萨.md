# 伊萨（焰爆女巫 · 火元素黑女巫）

[暗鸦星](../世界/任务世界/暗鸦星-暗鸦纪元.md) 的**焰爆女巫**——亚麻色长发、戴女巫帽、赤瞳的火元素黑女巫，与白女巫 [琼·艾塞亚](琼·艾塞亚.md) 合伙把马德高经营成奴隶城（她负责武力震慑）。实力在女巫中算「中上」。被艾塞亚传信示警后逃往**奥山王国**、躲在领主 **伦道夫** 处避难。

## 火元素战力 · 被苏晓练手（二十一卷9–10章）

苏晓杀进奥山王宫、斩了庇护她的伦道夫，再与她交手摸清女巫战力——她的**化身**是头生犄角、背生双翼的橘红色火焰恶魔（具现后近乎实体、且**与本体不同意识**），能瞬间抽空半公里内火元素、用「**元素爆炸**」轰出直径十米深三十米的岩浆大坑（把整座王宫由上至下炸穿）。但 [苏晓](苏晓.md) 用龙影闪躲过爆炸、青钢影刀芒斩碎她的火焰爪，确认「中上女巫尚且如此·身处爆炸中心绝没好果子吃」后直接制服她。

## 半句线索 · 被灭口（二十一卷10–11章）

苏晓没酷刑、反而给她喂水喂糖，用 [无尽黑暗](../力量体系/装备道具/index.md)（黑之狱·3 秒黑暗 = 意识里数十日）把她折磨到崩溃求饶——她吐出半句关键线索「我和艾塞亚都不知道死亡之地、**只有摩……**」，话没说完就**被某个超远距离的咒术系女巫灭口**（以自身为中心一切枯萎化飞灰·苏晓抓布布汪跳窗逃过）。原来女巫们泄露「死亡之地」必死——南部有个老女人设了「**尘霜诱惑**」咒杀泄密者。苏晓只得到一个「**摩**」字。

## 关系
- **[琼·艾塞亚](琼·艾塞亚.md)**：合伙贩奴的白女巫（长期勒索艾塞亚财物）。
- **[苏晓](苏晓.md)（库库林·白夜）**：生擒拷问她的猎杀者（她被灭口、苏晓仍占 95% 击杀判定·得开拓之源 + 传说级宝箱）。
