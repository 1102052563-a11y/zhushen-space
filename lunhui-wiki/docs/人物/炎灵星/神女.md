# 神女（茂生之神代行 · 秘鲁 · 投诚）

青绿眸、白皙美貌的少女——[茂生之神](茂生之神.md) 的代行。

- 以「预示你的死亡」唬人，却被 [温·艾尤儿](温·艾尤儿.md) 一巴掌打懵、又被 [苏晓](../苏晓.md) 当场吞掉她召来的神灵化身。
- **被神抛弃后崩溃投诚**（女人遭背叛后很可怕），供出秘鲁计划。是苏晓找到茂生之神本体的关键（藏在塞戈纳城平民中·[赖顿·红须](赖顿·红须.md) 曾不惜屠平民护她）。

## 关系

- [茂生之神](茂生之神.md)：抛弃了她的神。
- [苏晓](../苏晓.md)：吞其召来的化身、逼她投诚的猎手。
- [温·艾尤儿](温·艾尤儿.md)：生擒并「教育」她的女军官。

## 出处

- 三十五卷（[炎灵星·沙皇帝](../../世界/任务世界/炎灵星-沙皇帝.md)）：藏平民中被擒→召神灵化身被吞→被神抛弃投诚。
