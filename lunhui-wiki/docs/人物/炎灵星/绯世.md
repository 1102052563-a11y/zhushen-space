# 绯世（七阶最强违规者 · 来自轮回乐园 · 将至）

[莫蕾](莫蕾.md) 口中的**七阶最强违规者**、与 [苏晓](../苏晓.md) 同出轮回乐园——若 [灰绅士](灰绅士.md) 战力是绝望级、绯世更在其上（莫蕾自评战力 5、绯世 100）。

- 即将进入本世界，手下 [锯毒](锯毒.md) 惧怕的正是她的到来（锯毒疯狂清理世界即因恐惧她）。
- **共 4 名违规者**（锯毒已在世界内、绯世等 3 名将陆续抵达）——苏晓战争发家、升 [战争领主](../../力量体系/称号.md) 称号正是为迎战她做准备。
- **🆕逼近登场（三十五卷38 / 45章）**：审违规者套出她 **6~8 天后抵达**本世界（[锯毒](锯毒.md) 等三人正是她派来肃清世界 + 找 [达涅斯之戟](../../力量体系/装备道具.md) 的）；已在别的开放原生世界现身抢**世界之核**（淡红长袍 + 兜帽 + 黑色指环·凭空捏爆整支七阶冒险团 275 人·「核，在哪」）。苏晓撤到「安眠之地·灵魂树」（违规者的降临坐标）布满炼金阵图，备下 27 万兽骑兵的「欢迎仪式」等她。
- **结局（殁·三十五卷49–51章）**：从灵魂树降临即中「几公里大炼金阵图 + 空间回廊（空间道具原地折返）+ 电浆网 + 魔鹰领域」逃不掉——被 **27 万兽骑兵人海围杀**。她凶残优雅、清场三小时、吞敌回血、放同归于尽的「赎印」都没死透（意志极强·「八阶看到这阵仗都脑瓜子嗡嗡」），终被十六长枪钉地、苏晓一刀斩首。死前不甘「我，绯世，怎么可能死在这」。掉 137 钻石勋章 + **★★★★★★★ 猩红宝箱**——**四违规者全清**。（怪队友 [锯毒](锯毒.md) 的骚操作也坑了她。）
