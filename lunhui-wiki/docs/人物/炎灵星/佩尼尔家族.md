# 佩尼尔家族（末代旧王一脉 · 已灭 · 假王伏笔）

炎灵星原本的正统沙皇帝姓氏——**早被弑王杀绝**（详见 [炎灵星弑王史](../../世界/任务世界/炎灵星-沙皇帝.md)）。

- 当今 [沙皇帝](沙皇帝.md) 是「假王」，其父与 [乌诺](乌诺.md) 是弑王叛徒；半神 [尼波索](尼波索.md) 亲手处决末代佩尼尔沙皇帝。
- 秘鲁矮人至今敬重佩尼尔旧王、鄙夷当今假王。[苏晓](../苏晓.md) 伪装的**勇颅族**当年冲在弑王最前、事后反被灭族。
- **[佩尼尔之环](../../力量体系/装备道具/index.md)**（圣灵级·需佩尼尔嫡系后裔）落到苏晓手中——简介「将假王拖下王座」，是弑王史的关键伏笔道具。

## 关系

- [沙皇帝](沙皇帝.md)：取代其一脉的「假王」。
- [尼波索](尼波索.md) / [乌诺](乌诺.md)：弑王功臣 / 叛徒。

## 出处

- 三十五卷（[炎灵星·沙皇帝](../../世界/任务世界/炎灵星-沙皇帝.md)）：由赖顿·红须 / 尼波索之口揭史→佩尼尔之环出土。
