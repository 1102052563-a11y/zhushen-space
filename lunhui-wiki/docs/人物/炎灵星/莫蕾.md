# 莫蕾 / 莫雷（天启乐园战斗天使 · 战力担当 · 逗比）

粉发白肤少女——天启乐园的战力担当（脑回路跳脱、其他方面靠谱）；每颗环绕她的赤红水滴 = 一名死在她手里的违规者、「灵魂颂唱」凝成枪。

- **变青蛙逃命**：在秘鲁国被 [锯毒](锯毒.md) 逼到险被操控，靠「变青蛙（顶荷叶蹲水塘学蛙叫）」逃命（「帅不过三秒」）。
- **传纸条求救 + 警告绯世**：在 [布布汪](../布布汪.md) 身上定坐标、传青蛙纸条向 [苏晓](../苏晓.md) 求救，并警告七阶最强违规者 [绯世](绯世.md) 将至（「你有一点良心就来救我，否则以后想起我良心会痛」）。自评解印八成后能与锯毒五五开、但绝对顶不住绍世。
- 后被同乐园的 [灰绅士](灰绅士.md) 救走、组队几天（几次误以为对方要动手、吓出冷汗）。

## 关系

- [苏晓](../苏晓.md)：想合作杀违规者被婉拒；在布布汪身上定坐标要苏晓抗后果。
- [灰绅士](灰绅士.md)：救她的战斗天使同伴（真身吓人）。
- [锯毒](锯毒.md) / [绯世](绯世.md)：追杀她的违规者 / 她惧怕的最强违规者。

## 出处

- 三十五卷（[炎灵星·沙皇帝](../../世界/任务世界/炎灵星-沙皇帝.md)）：开场提合作→秘鲁变青蛙逃→传纸条求救警告绯世。
