# 赤沙（第五半神 · 沙皇帝王影护卫）

红衣缠布条、气息「撕裂般」的影子护卫——[沙皇帝](沙皇帝.md) 身边的王影，六位半神中排第五。

- 引 [苏晓](../苏晓.md) 进罗赛宫见沙皇帝；感知到苏晓的血气 / 斩龙闪锋利而不安（苏晓判断「交手六成胜算」·由此推知半神没传说中那么强、世界 Lv.66 上限只是个体极限）。
- 「王就是他活下去的理由」——对沙皇帝绝对忠诚。曾与乌诺王卫军一起教训闯宫的 [锯毒](锯毒.md)。

## 关系

- [沙皇帝](沙皇帝.md)：他誓死守护的王。
- [苏晓](../苏晓.md)：让他感到危机的引路对象（六成胜算）。

## 出处

- 三十五卷（[炎灵星·沙皇帝](../../世界/任务世界/炎灵星-沙皇帝.md)）：引路见沙皇帝→教训闯宫的锯毒。
