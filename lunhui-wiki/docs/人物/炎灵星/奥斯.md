# 奥斯（副统帅 · 从死人堆爬出的败将）

曾是「提着敌将头颅怒吼的海西堡·奥斯」、如今发福斗志尽失——因拍马屁拍歪被 [乌诺](乌诺.md) 贬。

- 乌诺委派他接替被毒杀的 [阿瓦雷](../炎灵星群像.md) 当 [苏晓](../苏晓.md) 副统帅；亲眼见前任一小时暴毙后极其识相、只听命不多言。
- 替苏晓截溃军、押俘虏、关军官、统计战果，渐成得力副手（行事越来越「懂事」得让苏晓都摸不透）。
- **🆕真相 = 深藏不露的幸存者（三十五卷51章）**：他其实是 [乌诺](乌诺.md) 派的眼线（苏晓早知却留着他·杀了也是乌诺的人）——但他早备好退路、还暗中投靠了沙都另一位大人物，把妻女托 [乌拉泰](../炎灵星群像.md) 从沙都救走。绯世死后被苏晓点破，**自尽化作沙土飘走**（临别劝「千万别回沙都」），是权力角逐里活到最后的老油条。[温·艾尤儿](温·艾尤儿.md) 接他的副统帅。

## 关系

- [苏晓](../苏晓.md)：又怕又忠的顶头上司。
- [乌诺](乌诺.md)：把他贬下来、又委派他的旧主。

## 出处

- 三十五卷（[炎灵星·沙皇帝](../../世界/任务世界/炎灵星-沙皇帝.md)）：接任副统帅→带兵作战→截溃军 / 押俘虏。
