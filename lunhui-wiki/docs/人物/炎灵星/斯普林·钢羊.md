# 斯普林·钢羊（秘鲁王）

现任秘鲁王（王权非世袭·曾是战士）——野心显露，要的不是沙焰领土、而是黄沙下的资源与超凡生物；[赖顿·红须](赖顿·红须.md)、[斯普林·铁羊](斯普林·铁羊.md) 皆其亲系。

- 看神灵不顺眼（若非顾忌早想除掉秘鲁神灵 [茂生之神](茂生之神.md)）。
- 被茂生之神警告 [苏晓](../苏晓.md) 是「杀过 5+ 古神的毁灭一切之人」后（只信一部分），不惜代价请异族刺客 [红姬](红姬.md) 来暗杀苏晓。

## 关系

- [苏晓](../苏晓.md)：欲除之的心腹大患（永久死敌）。
- [茂生之神](茂生之神.md)：庇佑秘鲁的中立神灵、却被他看不顺眼。
- [红姬](红姬.md)：他雇来暗杀苏晓的异族刺客。

## 出处

- 三十五卷（[炎灵星·沙皇帝](../../世界/任务世界/炎灵星-沙皇帝.md)）：赎表弟→受神灵警告→请红姬暗杀苏晓。
