# 凯撒（尼古拉斯·凯撒 · 前沙焰财政官 · 领路人 / 军需官）

小老头模样、笑起来就是奸笑的贪婪之徒——却「贪财不贪墨」，是 [苏晓](../苏晓.md) 的老搭档（多次给苏晓当开路先锋·胆小奸诈却常做最危险的事、连伤都没受过、家底相当足）。

- **前沙焰财政官**：差一步就爬到「司政」（沙焰财政官），被王后姐姐 [卡罗丝](卡罗丝.md) 排挤出沙都（吃了记错律法的没文化亏）。
- **本世界职责 = 军需官**：苏晓让他重操旧业管钱（卖巨岩龙 4250 金币起家、抽 5%），发安家费 / 悬赏、销赃乌诺金库偷来的珍宝、去秘鲁王处交涉勒索灵魂结晶（多余归他）。毕生三爱：钱、更多钱、炼金学（半吊子炼金术·见苏晓的洛氏阵图知识差点疯癫）。

## 关系

- [苏晓](../苏晓.md)：老搭档、军需官，二者互利。
- [卡罗丝](卡罗丝.md)：把他排挤出沙都的仇人。
- [奥斯](奥斯.md) / [焚克](../炎灵星群像.md)：军中同僚 / 手下。

## 🆕魔海·泯光岛杂货铺老板（四十一卷12–13章）

苏晓晋八阶后进入 [魔海](../../世界/任务世界/魔海-厄运与诅咒.md)，在厄运号的补给地「泯光岛」再度撞见这位老友——凯撒自称「洗心革面又成了半个裁决者」，用**某件契约者绝无可能获得的道具**、凭旧契约「**协助式进入**」魔海（无主线 / 无世界之源 / 杀敌不掉宝箱·但他毫不在意·九成资源都砸在苟命上）。经乐园多重判定，他成了**泯光岛杂货铺的老板**（苏晓是厄运号船长）：卖厄运号专用特制炮弹与各类船饰品（厄运血帆 / 怒鲨撞角 / 无畏船舵 / 诅咒之心等）。

- **「我亲爱的朋友，好久不见」**：一见苏晓便菊花般灿烂开笑、张口就是「凯撒对你的思念，就像巴乌凸鸟拉稀时的粪便一样滔滔不绝」。泯光岛被虚空之树公证、货价不能降，他却主动开「**赊账**」（好感度一度冲破上限又被判定归零、来回折腾后被系统「恒定为 100 点·友谊如顽石」，触发隐藏成就「诅咒之拯救」→ 苏晓获成就权限「**赊账**」）。苏晓当场把杂货铺库存几乎清空、欠下 **−295720 枚金币**（离世后权限移除）。
- **业界「任劳任怨」的骚操作**：巴哈由此明白他为何反复被剥夺裁决者身份——曾把 29 名天启乐园契约者「协助」逃进高一阶位的超高危区团灭、又把区内大 boss 忽悠进轮回乐园大赚，前后没违背任何初始条例、却已是**第 19 次**丢掉裁决者身份。唯一克星是 [布布汪](../布布汪.md) 的「降智型交涉」（高魅力把凯撒智商拉到与狗同级、逼他每次都拿出十足诚意）。

### 「无私」加入药剂利益链（四十一卷29、32章）

苏晓与财神 [富兰德](../魔海/富兰德.md) 结成活力复苏药剂利益链，缺个游走各方的中间人——一听有药剂交易，凯撒眼睛「亮得像探照灯」，「无私」且「坚决」地拒绝金币分成（「你居然认为凯撒会贪图金币？这是对凯撒品格的践踏！这次凯撒无偿帮你」——可惜最后没忍住奸笑出声）。他只用半小时就凭空间物品**脱离泯光岛**、还在泯光岛黑雾里布下临时传送阵（外人夺其空间物品闯泯光岛会落在黑雾里送死）。利益链里他不要分成、专收「交易途中的额外收益」——以他的骚操作，实得收益仅次于苏晓。
