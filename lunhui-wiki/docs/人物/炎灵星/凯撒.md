# 凯撒（尼古拉斯·凯撒 · 前沙焰财政官 · 领路人 / 军需官）

小老头模样、笑起来就是奸笑的贪婪之徒——却「贪财不贪墨」，是 [苏晓](../苏晓.md) 的老搭档（多次给苏晓当开路先锋·胆小奸诈却常做最危险的事、连伤都没受过、家底相当足）。

- **前沙焰财政官**：差一步就爬到「司政」（沙焰财政官），被王后姐姐 [卡罗丝](卡罗丝.md) 排挤出沙都（吃了记错律法的没文化亏）。
- **本世界职责 = 军需官**：苏晓让他重操旧业管钱（卖巨岩龙 4250 金币起家、抽 5%），发安家费 / 悬赏、销赃乌诺金库偷来的珍宝、去秘鲁王处交涉勒索灵魂结晶（多余归他）。毕生三爱：钱、更多钱、炼金学（半吊子炼金术·见苏晓的洛氏阵图知识差点疯癫）。

## 关系

- [苏晓](../苏晓.md)：老搭档、军需官，二者互利。
- [卡罗丝](卡罗丝.md)：把他排挤出沙都的仇人。
- [奥斯](奥斯.md) / [焚克](../炎灵星群像.md)：军中同僚 / 手下。

## 出处

- 三十五卷（[炎灵星·沙皇帝](../../世界/任务世界/炎灵星-沙皇帝.md)）：会合领路→军需官→交涉赎金→销赃采购→挑拨军卫长。
