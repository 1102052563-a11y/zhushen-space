# 温·艾尤儿（军卫长 · 爱权女汉子）

曾是「扛着自己帐篷行军的女兵」、被 [苏晓](../苏晓.md) 记住——攻塞戈纳城时立功生擒 [神女](神女.md)、被破格提拔到**军卫长**（沙都编制）。

- 粗暴爽利：抽神女大嘴巴子、搂肩威胁（「沙焰的女人好粗鲁」）。
- 后被派带 2000 兵护送神女回沙都、当苏晓在沙都的眼线（很危险，也是机遇）。

## 关系

- [苏晓](../苏晓.md)：破格提拔她的统帅。
- [神女](神女.md)：她生擒并「教育」的俘虏。
- [焚克](../炎灵星群像.md)：一同立功的军官（爱财，被分到凯撒手下）。

## 出处

- 三十五卷（[炎灵星·沙皇帝](../../世界/任务世界/炎灵星-沙皇帝.md)）：生擒神女→升军卫长→护送神女回沙都当眼线。
