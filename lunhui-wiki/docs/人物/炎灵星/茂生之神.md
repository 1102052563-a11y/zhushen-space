# 茂生之神（秘鲁中立神灵 · 代表「复苏」· 化身被吞）

十几米高、顶着「毯子」的模糊人形、树枝状爪——万物复苏的恩赐之神，只与两国之王平起平坐。

- **化身被吞**：[苏晓](../苏晓.md) 激怒 [神女](神女.md) 召来它的化身，被苏晓吞噬之核全开吞噬掉（比古神化身「好吃」·[青钢影](../../力量体系/技能/index.md) 因此升 Lv.59）；逃跑时**抛弃了神女**。
- **警告秘鲁王**：转而警告 [秘鲁王](斯普林·钢羊.md)——苏晓是「杀过 5+ 古神、能毁灭沙焰与秘鲁的毁灭一切之人」，要不惜一切在激怒他前杀掉他。
- **主线「神泣」要生擒的正是它本体**。
- **结局（殁·三十五卷47章）**：因惧怕绯世想在前线积蓄力量（弄「食魂石像」被斯普林·铁羊拒 + 揭它当年跪求秘鲁王屠佩尼尔家族），又察觉苏晓已吞其化身而想逃——被秘鲁王当叛徒交出、[苏晓](../苏晓.md) 吞其本体**完成主线「神泣」**（黄金技能点 ×5·并从铁笼里带出 [达涅斯之戟](../../力量体系/装备道具/index.md) 残片）。临终「你们也会被吞噬，谁都逃不掉」。

## 关系

- [苏晓](../苏晓.md)：吞其化身、要生擒其本体的猎手。
- [神女](神女.md)：它的代行、却被它抛弃。
- [斯普林·钢羊](斯普林·钢羊.md)：它庇佑（又看它不顺眼）的秘鲁王。

## 出处

- 三十五卷（[炎灵星·沙皇帝](../../世界/任务世界/炎灵星-沙皇帝.md)）：化身被吞→抛弃神女→警告秘鲁王。
