# 锯毒（镶钻金牙违规者 · 毒 / 控制 / 侵蚀 · 绯世手下）

金属镶钻牙——能超远程操控本世界活物、但操控掌权者 / 契约者需 2~5 秒身体接触（操控不了契约者 / 从者）；擅追踪，把 [莫蕾](莫蕾.md) 从秘鲁追杀到沙焰。

- [苏晓](../苏晓.md) 刚进世界就被其操控的秘鲁奴隶下毒偷袭（青钢影驱毒），留「斯芬克不会白死」的字误导拖延（**斯芬克**实死于苏晓·与此人无关）。
- **跑去沙都竟想操控 14 岁 [沙皇帝](沙皇帝.md)**、被 [赤沙](赤沙.md) + [乌诺](乌诺.md) 王卫军教训、遭全沙焰通缉；卷末操控了乌诺。
- **是 [绯世](绯世.md) 的手下、对绯世不忠又惧怕她**——因恐惧绯世到来，改为疯狂猎杀世界里的战斗天使 / 游侠 / 先驱想「清理干净」。
- **结局（殁·三十五卷37–38章）**：与 [魔术师](../炎灵星群像.md) / [阿蜜莎](../炎灵星群像.md) 操控乌诺、设局想泼脏水 + 引爆乌诺体内生物炸弹炸死三掌权者嫁祸苏晓——却被沙都三掌权者早识破、[赤沙](赤沙.md) 清掉炸弹、8 万护卫军围杀；锯毒 / 阿蜜莎被擒，苏晓地下监牢审杀之（11001 号死亡乐园）、套出绯世 6~8 天后到。

## 关系

- [苏晓](../苏晓.md)：他偷袭又误导的猎杀者。
- [莫蕾](莫蕾.md)：被他追杀的战斗天使。
- [绯世](绯世.md)：他惧怕又不忠的上级违规者。

## 出处

- 三十五卷（[炎灵星·沙皇帝](../../世界/任务世界/炎灵星-沙皇帝.md)）：下毒挑衅→追杀莫蕾→闯沙都操控沙皇帝失败→操控乌诺。
