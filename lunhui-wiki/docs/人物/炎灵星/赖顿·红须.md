# 赖顿·红须（断角部队指挥官 · 秘鲁王表弟 · 三次被擒）

酒红络腮胡扎辫、1 米 45 却「5 米 41 气势」的矮人军官——[秘鲁王斯普林·钢羊](斯普林·钢羊.md) 的表弟。

- 率秘鲁精锐**断角部队**（破沙焰防线的先头军），被 [苏晓](../苏晓.md) 生擒后一心求死（被 [凯撒](凯撒.md) 臭袜子熏到招供）；有骨气、敬 [佩尼尔](佩尼尔家族.md) 旧王、鄙夷假王。秘鲁王用 40 颗完整灵魂结晶赎回他。
- 攻塞戈纳城时第二次被擒（投降是为拖延、护 [神女](神女.md)）；门之战役敲晕 [斯普林·铁羊](斯普林·铁羊.md) 送走、替兄接手战场，第三次被擒（「代兄而死不丢人」）。
- 从他口中揭出「假王 [弑王史](../../世界/任务世界/炎灵星-沙皇帝.md)」。

## 关系

- [斯普林·钢羊](斯普林·钢羊.md) / [斯普林·铁羊](斯普林·铁羊.md)：他的王兄 / 他替死的堂兄。
- [苏晓](../苏晓.md)：三度生擒他的宿敌（「我不会再被他算计」）。

## 出处

- 三十五卷（[炎灵星·沙皇帝](../../世界/任务世界/炎灵星-沙皇帝.md)）：首战被擒爆弑王史→塞戈纳城再擒→门之战役替兄三度被擒。
