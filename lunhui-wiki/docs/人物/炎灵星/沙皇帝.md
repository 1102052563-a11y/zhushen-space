# 沙皇帝（14 岁 · 体虚 · 全国主心骨 · 疑为「假王」）

白衣、脸色苍白、被美艳 [王后](../炎灵星群像.md) 迷得出黑眼圈的少年王——却是所有掌权者 / 军队 / 神灵都拥护的主心骨、王位无人能撼（威严一发赤沙都不敢直视·「一头幼狮」）。

- **与苏晓的旧交**：竟与 [苏晓](../苏晓.md) 伪装的「勇颅族后裔」身份**6 年前有私交、还被他揍过一拳**（关系亲近·苏晓却不知当年互换了什么礼物）。是苏晓在沙都名义上的支持者。
- **弑王史阴影**：秘鲁矮人称其为「假王」——正统 [佩尼尔家族](佩尼尔家族.md) 沙皇帝早被杀绝，其父当年**战胜并屠灭了「吞噬神灵的佩尼尔家族」**（联合 [乌诺](乌诺.md) / 勇颅族 / 秘鲁王）（详见 [炎灵星弑王史](../../世界/任务世界/炎灵星-沙皇帝.md)）。
- **🆕真相 = 沙都布局者（三十五卷44章）**：所谓「三方内斗制衡」全是**苦肉计**——[乌诺](乌诺.md) 其实一直效忠他（私下称他「王」）、借调给苏晓的 30 万战兽是他借乌诺之手送的；[卡罗丝](卡罗丝.md) 演戏多年。他让 [王后·卡美拉](../炎灵星群像.md)（当年被剥脸换美颜送到身边）去杀姐姐卡罗丝、再命 [赤沙](赤沙.md) 灭口王后。借苏晓打残秘鲁扫外患，好从「假王」变真王。
- **🆕真·结局 = 反被反杀、却借复生成小孩胜出（三十五卷52 / 58章）**：他自以为稳操胜券，却不知卡罗丝姐妹本就是**佩尼尔家族后裔**（血脉未断）、以苦肉计诈死——姐妹俩吞神灵复生后**反杀了他和乌诺**（乌诺护主被撕成两截）。但沙皇帝**竟又借茂生之神的「复生」重生成两三岁小孩**（真正的最终胜者·「幸好库库林对王位不感兴趣」）——是套娃权谋里活到最后的人。

## 关系

- [苏晓](../苏晓.md)：6 年前的「旧交」、他布局里最信任的一环（借苏晓打秘鲁扫外患夺权）。
- [乌诺](乌诺.md)：明面敌对、实则一直效忠他的「猛犬」。
- [卡罗丝](卡罗丝.md) / [王后·卡美拉](../炎灵星群像.md)：他要清除的演员 / 剥脸送到身边又要灭口的王后。
- [佩尼尔家族](佩尼尔家族.md)：他父亲屠灭的正统旧王。

## 出处

- 三十五卷（[炎灵星·沙皇帝](../../世界/任务世界/炎灵星-沙皇帝.md)）：罗赛宫会面→让苏晓自夺军权→隐秘增援 14 万兵。
