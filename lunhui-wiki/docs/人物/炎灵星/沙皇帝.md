# 沙皇帝（14 岁 · 体虚 · 全国主心骨 · 疑为「假王」）

白衣、脸色苍白、被美艳 [王后](../炎灵星群像.md) 迷得出黑眼圈的少年王——却是所有掌权者 / 军队 / 神灵都拥护的主心骨、王位无人能撼（威严一发赤沙都不敢直视·「一头幼狮」）。

- **与苏晓的旧交**：竟与 [苏晓](../苏晓.md) 伪装的「勇颅族后裔」身份**6 年前有私交、还被他揍过一拳**（关系亲近·苏晓却不知当年互换了什么礼物）。是苏晓在沙都名义上的支持者。
- **弑王史阴影**：秘鲁矮人称其为「假王」——正统 [佩尼尔家族](佩尼尔家族.md) 沙皇帝早被弑王杀绝，当今沙皇帝之父与 [乌诺](乌诺.md) 是弑王叛徒（详见 [炎灵星弑王史](../../世界/任务世界/炎灵星-沙皇帝.md)）。

## 关系

- [苏晓](../苏晓.md)：6 年前的「旧交」、名义支持者。
- [乌诺](乌诺.md) / [卡罗丝](卡罗丝.md)：他用卡罗丝拴住猛犬乌诺、维持王权 / 财政 / 军权的制衡。
- [佩尼尔家族](佩尼尔家族.md)：他这一脉取代的正统旧王。

## 出处

- 三十五卷（[炎灵星·沙皇帝](../../世界/任务世界/炎灵星-沙皇帝.md)）：罗赛宫会面→让苏晓自夺军权→隐秘增援 14 万兵。
