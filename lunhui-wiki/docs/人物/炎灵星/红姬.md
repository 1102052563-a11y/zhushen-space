# 红姬（异族刺客 · 金属丝线 · 已死）

来自「大海以西古文明大陆」的异族（那片废土文明与古神鱼死网破而毁·**圣魂修行的起源**）——与 [秘鲁王](斯普林·钢羊.md) 是雇佣关系。

- 红面具黑衣、擅金属丝线暗杀，潜入塞戈纳城杀了 [焚克](../炎灵星群像.md)、隔离石楼夜袭 [苏晓](../苏晓.md)，却被苏晓反手斩杀（掉圣灵级宝箱 + 噬灵者 +60 法力）。

## 关系

- [斯普林·钢羊](斯普林·钢羊.md)：雇她的秘鲁王。
- [苏晓](../苏晓.md)：她暗杀失败、丧命于其刀下的目标。

## 出处

- 三十五卷（[炎灵星·沙皇帝](../../世界/任务世界/炎灵星-沙皇帝.md)）：夜袭塞戈纳城杀焚克→被苏晓斩杀。
