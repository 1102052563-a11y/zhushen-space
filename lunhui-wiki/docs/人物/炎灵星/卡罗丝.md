# 司政·卡罗丝（真身 = 佩尼尔·阿莉雅 · 吞噬神灵的姐妹 · 殁）

把 [凯撒](凯撒.md) 排挤出沙都的狠角色、[王后](../炎灵星群像.md) 的姐姐——掌沙焰财政，与 [乌诺](乌诺.md) 同派（财政是「绳子」、军权是「猛犬」）。

- 暗地里养劫掠团做脏活（沙都外那条劫掠巨岩龙就是她的人·被苏晓灭了留活口回礼）；明面上在沙都不显敌意。捏断踩到裙摆的女仆脖子、心狠手辣。
- 神女被擒后异常积极主张调兵给苏晓（似对神女别有所图）；身后曾浮现「满是尖牙的巨口」。
- **🆕真身 = 佩尼尔·阿莉雅、复仇 16 年的姐姐（三十五卷52–58章）**：她真名 **佩尼尔·阿莉雅**——被 [沙皇帝](沙皇帝.md) 之父屠灭的正统王族 [佩尼尔家族](佩尼尔家族.md) 后裔（血脉根本没断）。为复仇苦熬 16 年，用「置之死地而后生」的苦肉计：故意演「可控的反派掌权者」求「败」（让沙皇帝放松警惕）、又**剥下亲妹 [卡美拉](../炎灵星群像.md) 的脸换美颜、送她去仇人身边潜伏**找赤沙弱点。苏晓打秘鲁扫外患后，她**吞茂生之神、妹妹吞神女而血脉觉醒复生**，反杀沙皇帝 + 撕碎乌诺。
- **结局（殁·三十五卷55–57章）**：作为本世界最强 boss（吞两神灵 + 唤醒「众王」燃灵魂·力 173 / 敏 168 / 体 170），属性能力全面碾压苏晓，却被苏晓**技法吊打**（完美反制削力 + 青影王五灭法之刃 + 界断线魂之丝 + 刃道刀·极）斩首。掉 [众王大剑](../../力量体系/装备道具/index.md)。
