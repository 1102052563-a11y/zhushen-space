# 司政·卡罗丝（王后姐姐 · 财政官 · 反派）

把 [凯撒](凯撒.md) 排挤出沙都的狠角色、[王后](../炎灵星群像.md) 的姐姐——掌沙焰财政，与 [乌诺](乌诺.md) 同派（财政是「绳子」、军权是「猛犬」）。

- 暗地里养劫掠团做脏活（沙都外那条劫掠巨岩龙就是她的人·被苏晓灭了留活口回礼）；明面上在沙都不显敌意。捏断踩到裙摆的女仆脖子、心狠手辣。
- 神女被擒后异常积极主张调兵给苏晓（似对神女别有所图）；卷末身后浮现「满是尖牙的巨口」——疑另有隐藏势力 / 被违规者盯上。

## 关系

- [苏晓](../苏晓.md)：暗中作对（劫掠团 / 拖延军权）的敌对掌权者。
- [乌诺](乌诺.md)：同派系盟友。
- [凯撒](凯撒.md)：被她排挤出沙都的手下败将。

## 出处

- 三十五卷（[炎灵星·沙皇帝](../../世界/任务世界/炎灵星-沙皇帝.md)）：养劫掠团→早宴拖延军权→力主调兵给苏晓→身后现巨口。
