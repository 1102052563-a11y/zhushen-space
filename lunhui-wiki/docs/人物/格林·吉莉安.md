# 格林·吉莉安（先代女灭法之影 · 至尊锋刃开发者）

灭法之影遗迹传承大殿内一尊女性雕塑的主人——一位前辈 **[灭法之影](../力量体系/技能.md)**，雕塑铭文记其功绩与遗言：

> 「**格林·吉莉安。断魂影。至尊锋刃开发者。死于绝灭之役。守护元素平衡 427 次，避免 58 颗星球上生灵死于星球崩溃。**」
> 「**不许碰我的雕塑！否则砍死你——格林·吉莉安。**」（雕塑是一段有声留言·性格意外恶劣）

她开发的「**至尊锋刃**」正是 [苏晓](苏晓.md) 斩龙闪「至尊锋刃效果」的源头（嗜血礼赞等）。

## 即「将爱丽丝砍出阴影」的黑发女灭法之影（十七卷80章·身份揭晓）

[苏晓](苏晓.md) 在大殿认出这尊女雕塑的容貌——与恶魔古堡宝库壁画上**那名将 [爱丽丝](爱丽丝.md) 装进铁笼、要吊死在「虚空台」的黑发女灭法之影**（腰挂长刀 · 脚踝藏锯片）是同一人。当年正是格林·吉莉安抓住破坏三颗星球元素平衡的爱丽丝、点破「灭法者 = 虚空秩序的维护者」，逼得爱丽丝立誓造古堡复仇——爱丽丝至今对铁笼 / 吊死心有余悸，皆源于她。

## 关系
- **[爱丽丝](爱丽丝.md)**：被她抓去差点吊死的「金发女子」（爱丽丝造恶魔古堡、当女王复仇的执念之源）。
- **[马文·华尔兹](马文·华尔兹.md)**：同为先代灭法之影（断魂影一脉）。
- **[苏晓](苏晓.md)**：后世灭法之影传承者，斩龙闪「至尊锋刃」即承她所开发。
