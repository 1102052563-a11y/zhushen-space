# 纲手

[火影忍者世界](../世界/任务世界/火影忍者.md) **五代火影**——木叶「三忍」之一，与 [自来也](自来也.md) 同辈。身穿宽松绿衣难掩豪乳，虽年龄已大却容貌如二三十岁的成熟女人；精通怪力与医疗忍术。参加过第二次忍界大战，对砂忍村印象不佳。

- **登场（九卷37–38章）**：[我爱罗](我爱罗.md) 被掳后砂忍村向木叶求援，凭和平条约木叶不会坐视不理。她从秘书静音处接到袭击砂忍村三人（[蝎](蝎.md) / [迪达拉](迪达拉.md) / 不用忍术的「白夜」[苏晓](苏晓.md)）的情报，本以为「白夜」只是晓的候补成员——直到刚回木叶的自来也解开衣服露出**胸前半米多长、深可及骨**的伤口，告知「这家伙可不是候补成员」，她才正色重视。此前自来也已通灵蛤蟆寅向她传过苏晓三人的情报。
- 此时木叶火影办公室内还有 [漩涡鸣人](漩涡鸣人.md)、[春野樱](春野樱.md)、[卡卡西](卡卡西.md)。

## 团藏死后复任火影（十五卷52章）

苏晓 + 长门攻木叶后她重伤、火影之位被 [团藏](志村团藏.md) 趁机夺去（任「代理火影」）。其实她早凭**仙人体**（千手 / 漩涡同源·生命力强）养好了伤、只是养病期间被团藏架空——**团藏一死（被佐助 + 苏晓一伙逼到里四象封印自尽），她便名正言顺重新接任五代火影**，去收拾团藏留下的烂摊子。这印证「五大忍村底蕴深厚」：数日内连失三影（雷 / 水 / 火代理）却各自迅速稳住阵脚，反推着忍界大战临近。

## 率木叶追苏晓 · 加藤断之痛（十五卷77–78章）

苏晓在木叶附近狙杀外来者、枪声引来她率心腹（静音 / 春野樱 / 奈良鹿久 + 暗部）追击——她咬牙切齿喊出「白夜」（上次苏晓与长门一同灭了木叶）。但拦在苏晓身前的秽土转生者，竟是她祖父开发的禁术转生出的**前恋人加藤断**（灵化之术·她琥珀眸骤缩、回忆涌上）。作为一村之影、她强压私情亲手「打碎」加藤断（等封印忍者来）、只敢派暗部远远跟苏晓；面对苏晓后续的二代土影无（尘遁）、三 / 四代雷影、连被苏晓斩杀又被转生来的 [照美冥](照美冥.md) 等接连殿后，木叶死伤不断——她**重木叶安危胜于个人恩怨、果断停止这绝望的追击、回防木叶**。
