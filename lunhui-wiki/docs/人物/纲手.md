# 纲手

[火影忍者世界](../世界/任务世界/火影忍者.md) **五代火影**——木叶「三忍」之一，与 [自来也](自来也.md) 同辈。身穿宽松绿衣难掩豪乳，虽年龄已大却容貌如二三十岁的成熟女人；精通怪力与医疗忍术。参加过第二次忍界大战，对砂忍村印象不佳。

- **登场（九卷37–38章）**：[我爱罗](我爱罗.md) 被掳后砂忍村向木叶求援，凭和平条约木叶不会坐视不理。她从秘书静音处接到袭击砂忍村三人（[蝎](蝎.md) / [迪达拉](迪达拉.md) / 不用忍术的「白夜」[苏晓](苏晓.md)）的情报，本以为「白夜」只是晓的候补成员——直到刚回木叶的自来也解开衣服露出**胸前半米多长、深可及骨**的伤口，告知「这家伙可不是候补成员」，她才正色重视。此前自来也已通灵蛤蟆寅向她传过苏晓三人的情报。
- 此时木叶火影办公室内还有 [漩涡鸣人](漩涡鸣人.md)、[春野樱](春野樱.md)、[卡卡西](卡卡西.md)。

## 出处
- 九卷27章（自来也向她传信·提及）、九卷37–38章（登场）。
