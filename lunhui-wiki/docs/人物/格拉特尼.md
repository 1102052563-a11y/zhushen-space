# 格拉特尼（暴食）

[瓶中的小人](瓶中的小人.md) 所造七名人造人之一，代表「**暴食**」——秃头小胖子（百余岁·智低·与 [拉丝特](拉丝特.md) 最亲·吃生物前会先问拉丝特）。

- **能力**：瓶中小人造「真理之门」的**失败品**——肚子是一个**无限大、无出口的吞噬空间**（**被吞 = 全属性 −10**·连斩龙闪都扛不住空间切割）；胸口有「**真理之门之眼**」喷真空波吞噬一切（攻击使其闭合→吞噬范围缩小）、绝招「黑洞」放强吸力。
- **殁**：苏晓**绕到背后、用白色应急武器逐刺那只眼**（眼闭合则吞噬范围缩小·消耗 6 把白武器）；其放「黑洞」危急时被 [布布特尼](布布汪.md) **遁地咬屁股拖倒**（眼闭吸力消），被苏晓斩杀（剧情关键·世界之源 +5.6% → 12.1%·紫宝箱·七卷9章）。
