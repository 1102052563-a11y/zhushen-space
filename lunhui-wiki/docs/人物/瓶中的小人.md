# 瓶中的小人

[钢之炼金术师世界](../世界/任务世界/钢之炼金术师.md) 的 **BOSS / 幕后主谋**（非人类）。

- **来历**：400 年前古王国「赛尔克塞斯」王室追求永生，用主角父亲 [冯·霍恩海姆](冯·霍恩海姆.md) 的血合成出它；它教霍恩海姆炼金术、却**骗王室布大型炼成阵、把全国民炼成贤者之石**，借此自炼成人造人（貌似霍恩海姆）、并令霍恩海姆永生（只要它不死）。
- **势力**：一手发展壮大 **亚美斯多利斯国**、军方高层皆其鹰犬；麾下有 [七名人造人](拉斯.md)（以自身情绪 + 贤者之石所造）。
- **终极目的（成神）**：把全国炼成贤者之石、是为了**成「神」**——疑是拿贤者之石与「真理之门」交换 / 夺取真理之门（钢炼世界得真理之门即可称神）。它体内贤者之石终会耗尽、那即是死期；目前武力尚不足以强掌全国、故人造人皆暗中行动（**致命弱点 = 见不得光**）。
- **与苏晓**：苏晓的**晋升任务「窃取神之力（伪）」**目标就在它体内；但它在本国近乎无解、原著靠众人围攻才落败——苏晓**不正面硬碰**，而是逐个削弱 / 猎杀它麾下的人造人、并联合反它的势力（马斯坦 / 冰雪女王）。

## 成「神」与陨落（七卷42–44章）
- **国土炼成阵启动**：五名人柱（爱德华 / 阿尔冯斯 / 霍恩海姆 / 伊兹米 / 马斯坦）凑齐后，血红炼成阵笼罩全亚美国（5000 万人被抽魂），它吞入约 **2000 万灵魂**、化为顶破总统府的巨人；天上「**真理之门**」开启、露出「**真理之眼**」，它强行把真理之眼拖入体内、**成为「神（伪）」的容器**（钢炼世界的伪神·无自我意识）。但启动时机不对（非日全食 + 仅 2000 万 / 5000 万灵魂），炼成阵并不完整。
- **阴谋背后的阴谋**：[冯·霍恩海姆](冯·霍恩海姆.md) 早把自己体内几十万人份贤者之石布在全国、备下**第二道「逆向」炼成阵**——待它成神瞬间发动，**把那 2000 万灵魂尽数放回各人身体**。它由此**空有神之力却失去驱动神之力的能量**（贤者之石），连行动都吃力，被五人柱（已恢复炼金术）反压。
- **被夺神之力 / 陨落**：苏晓藏在坑洞旁伺机，趁界断线被它崩断、五人柱缠斗之际**一刀斩下它的头颅**，再把任务物品 **魔晶瓶** 插进其脖颈、**收取神之力（伪）**（约 30 秒收满）——它就此定格如雕塑。苏晓「用剧情人物消耗 BOSS、最后摘桃子」的计划完成；收取完成即触发**晋升任务达成**。
