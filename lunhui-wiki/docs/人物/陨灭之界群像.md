# 陨灭之界群像（院长 · 疯医生 · 寡妇西 · 金刚王 · 飞船班子）

[陨灭之界·五方混战](../世界/任务世界/陨灭之界-五方混战.md) 中 [苏晓](苏晓.md) 同阵营的轮回乐园「疯子契约者」核心阵容，及飞船职工、敌方群像。

## 轮回乐园方核心阵容

- **院长（二马路精神病院院长 · 超级老阴哔）**：白袍老头——精通布局 / 算计 / 人心把控（大局观上等·略逊寡妇西）。出了名的记仇又欠揍（爱挑事被众疯子圈踢、断手能立刻重生），却「把他卖了还感激涕零」。常自称众契约者是他「没出院的病人」。本世界的总布局者之一。
- **疯医生 / 蓝大夫（前违规者 · 集六烙印者）**：戴无框眼镜、文质彬彬（大卸八块溅一脸血也斯文）的疯子——曾是违规者（苏晓险些接到追杀他的任务），现已「上岸」成契约者。核心能力 = 通过击杀其他乐园契约者**强掠烙印**，现**集齐天启 / 圣光 / 死亡 / 圣域 / 守望 / 轮回六种烙印**（手心可切换），故能**潜入任何乐园**。此能力疑似违规、需苏晓全程监督（叛变即清除）；得虚空之树公证（因轮回乐园未派七阶强入·公证倾向轮回方）。
- **寡妇西 / 伊克西（蛇蝎美人 · 布局强于院长）**：不到三十岁的美人——传有 105 任丈夫皆死于新婚夜（厌恶与男性肢体接触却渴望浪漫婚礼·人称「很费丈夫的寡妇西」）。布局 / 大局观强于院长，两人四阶曾联手进火影世界、把五大忍村 + 两大冒险团算计到「木叶都出九代火影」、月之眼计划差点成功（带土没登场就被围攻）。
- **金刚王（战争巨兽 · 正面冲锋）**：智谋欠缺但极适合战场的契约者——变身后是「不怕死的战争巨兽」，负责正面带队冲锋。
- **小屁孩（钻体操控者 · 12~13 岁外形）**：两百多疯子里极少数自认「正常人」的契约者——能力是钻入敌人体内操控对方（自己也觉得这能力「有点不正常」）；曾跳起来踹院长的脸，被记仇的院长缠上。

### 自杀小队（6 独狼 · 能力诡异敌我不分）

院长把 6 个能力太毒、与大部队同行会误伤友军的独狼编成「自杀小队」（互相伤害就好·**毒师**抽签当队长）——首个目标是摧残人数最多的天启乐园，专打士气：

- **原地去世流（小胖子）**：菠萝头机车装——平时弱到五阶都能吊打，「死后」靠 **黑魔（恐惧传递）** 替身能力翻盘：以耳上小剪刀为载体与击杀者「换命即死」、黑色烂肉「黑魔」捕食三名活物后他即复生（克星雪盐 / 圣属性）。
- **毒师（队长）**：玩毒的老哥、平时一副要把自己毒死的样、没人敢靠近（曾是某冒险团成员·操毒失误）。
- **黑寄生**：阴郁妹子·放出的寄生物连自己都控不全、不分敌我。
- **邪活种召唤**：召唤出的全是邪异活物（连自己都不知会召出啥）——把教团侍者**乌鸦女**玩死。
- **幽灵自爆流**：软萌少女·放出的幽灵钻进半公里内一切活物体内爆炸（敌我不分）。
- **活性生物炸弹**：爆炸狂魔小孩·口水都能爆炸、随时自爆。

## 飞船班子（轮回乐园大本营 · 职工 / 副官）

- **癞蛤蟆舰长（暴鼠级狠角色）**：脸盆大、戴王冠的癞蛤蟆——大本营飞船舰长，气息极强（与 [暴鼠](强者争霸战群像.md) 一个梯队）。曾是某位裁决者「小可爱」的引导人（小可爱已成裁决者、在五阶战区·会向他挥剑了「真让人欣慰」）。负责操控弹射舱把苏晓投到敌方坐标附近。
- **1045 号（黑甲副舰长）**：全身焚烧过的黑甲与肉身融为一体、沉默寡言的男人——飞船副舰长。
- **珞珞（酒保 · 职工者）**：偏男性打扮的少女酒保，爱睡过头（睡迷糊把鞋摆吧台上往里倒酒）；怕疼、不渴望力量。
- **[夏](夏.md)（厨娘 · 职工者）**：负责飞船饮食的优秀职工者——苏晓的旧识（参与强者争霸战后，巴哈 / 贝妮也能到现实 / 大本营了）。

## 敌方群像

- **白发妹（天启乐园 · 感知系 · 殁）**：苏晓 1 对 53 开局的天启乐园小队成员——侦测装置生物波动上限 8000 被苏晓「爆表」吓到，藏情报想先卖队友再逃（很「天启乐园风格」=先保自己活）；最终全队仅逃 3 人、她跪地求死被一刀斩首（误把苏晓当七阶契约者）。
- **天启乐园 53 人小队**：刚抵达世界、想去集结点会合——主坦持史诗级 +11 双宝石盾（被苏晓一刀破）、核动能狙击手（枪管被苏晓弹回的金属针炸膛）等；「契约投降」（付资产 + 装备求饶）对苏晓无效，**杀 50 逃 3**。
- **圣域教团**：飞船着陆前就找上轮回乐园方大本营对峙（实力 + 胆量罕见）——背刺能手、打起来悍不畏死，是己方大敌（老阴哔斯芬克也出自圣域乐园）。

### 圣光乐园（综合战力垫底 · 转投轮回乐园）

- **伊莫弥（圣光乐园领袖 · 治疗系）**：盘发女人——被死亡 + 圣域夜袭打残后挺身带 100 多人撤回的临时领袖；**治疗系**：大范围治疗 / 增益，还能**反治敌方**（敌人被她奶反而行动迟缓 / 肢体麻痹·治敌仅 6~8% 治疗量）。看清局势后果断**与轮回乐园结盟报仇**（宁可不赢也不让背刺成性的教团赢），借此立威、想从精英幸存者里组个五人小冒险团。
- **绯珊（圣光乐园三号 · 殁）**：六阶顶尖战力——夜袭战中被教团神棍的「圣锥」秒杀，圣光士气大挫。

### 圣域教团（背刺能手 · 牧羊人麾下）

- **牧羊人（教团首神官 · 本世界圣域乐园领袖）**：金瞳的冷漠神官——「神的代行 / 地上管理者」。背刺成性（刚谎称联合圣光、转头就与死亡乐园结盟夜袭圣光；又想联合轮回去背刺死亡）；为试探轮回乐园态度，把侍者**乌鸦女**当弃子送出（「你的牺牲我会记住」）。
- **乌鸦女（牧羊人侍者 · 殁）**：可化灰乌鸦的侍者——替教团出面交涉（死亡 + 圣域结盟也是她谈的）；奉命来联合轮回乐园，没见到院长就被**邪活种**玩死（本体随化身之死一同被牧羊人冷漠抛弃）。

### 天启乐园（背锅侠）

- **[无名圣徒](强者争霸战群像.md)**（旧识·见强者争霸战）：本世界天启乐园主心骨——被院长甩锅举着时空结晶背了大黑锅、四散被追杀，愤而前四天放弃争夺保战力。异种大军来袭时**用资源优势抛矿石「买命」拖住异种**（每小时来袭就抛两箱稀有矿·异种竟撤·异王认得矿石价值），借此筛掉怕死的（830→300 精英）、留七阶援军不露面、等反击——稳局手段漂亮。第二次在世界争夺战撞上苏晓（强者争霸战 + 竞技场后第三次）。
- **彩虹鸟（天启乐园 · 战斗天使 · 逗比）**：双马尾少女、无名圣徒队友——很强的逗逼（坚信「童子尿是邪物克星」）。

### 死亡乐园（已出局）

- 大部队被苏晓的阿波罗 + 疯子们灭于野外（含中型「赤铁冒险团」被猎团吞噬）；残党守大本营时一名**被强制征召的七阶契约者**见来的是轮回乐园疯子，直接弃征召任务撤走；临时空间坐标稳定器被苏晓破坏 → 死亡乐园付 7500 盎司时空之力强启飞船**出局**（这次没出双子星那种领袖、比教团好对付）。

### 自闭兄弟（轮回乐园方 · 不善交流的好人）

被某种能力影响、排斥交流的孪生兄弟（不入临时联盟·只按自身意愿行动）——其实是好人：哥哥用简陋如烧火棍的枪「三点一线即死」（无子弹飞行时间·打爆来袭的珀琉爆桶）、弟弟被寡妇西逗到脸红（这种少年在轮回乐园稀有如熊猫）；为全军造了消耗品木盾。

### 本土土著（暗中敌人 → 主战场大敌）

万兽大陆原住民——**异种**（地心苏醒的智慧种族）与**屈服的兽人**已联手（巴哈发现异种 / 兽人尸体混在一起），早在五方乐园抵达前 5 天就到陨落区、已得 **12+ 颗时空之力凝结体**；占地利、数量多，等五方互耗削减后下场。人类与亚兽人仍反抗。死亡乐园出局后，异王把轮回乐园方大本营当作**主战场（血色战场「图耶古勒」）**主攻——派裂变种来宣战「天外人滚出我们世界」，**疯医生**反把四方都拖下水。

### 异种（万兽大陆霸主 · 战争种族）

- **异王**：异种的绝对统领（统御异种高层、高层指挥地行种·王不能有失否则地行种失控）。**异种本是侵略种族却想离开万兽大陆**——此地资源不足、地行种会吃光一切有机物，故要借时空之力凝合体**举族迁去更大、超凡生物更多的世界**（兽人因此屈服：盼异种快滚别再吃兽人）。认定轮回乐园方是唯一值得赌上族群命运决战的强敌；世界之核在异王处（随从贝妮一直找不到）。
- **裂变种 / 寄生种（高等智慧）**：**裂变种**有肉翅、角质防御、能飞 / 钻地伏击 / 放紫色追踪火团（人形、不丑反有威严·雌雄皆有）；**寄生种**是顶尖暗杀潜入者——寄生夺身 + 神经连脑（当年正是它伪装人类高层断了人族文明）·能化血刺 / 岩石禁锢，是最棘手的异种。
- **地行种（次级智慧 · 主力炮灰）**：四爪奔跑、力 105 / 敏 107、地心生物（火减 67%）——「战斗族群」被动：数量超 2 万只就有一属性转真实属性。悍不畏死却有痛觉与情感（雌雄结终生配偶、丧偶则孤独终老），「躲回地心沉睡比死更可怕」故一切为种族延续而战。
- **珀琉爆桶（兽人战争武器）**：大肚木桶——落地化「火焰蟒蛇」360° 喷火两次（兽人投靠异种前装冰脉对付异种、投靠后改火系·异种火抗高故对异种伤害有限）。

#### 达普修要塞守卫（地下五层 · 守 9 颗陨星石）

- **特殊异种（陨星石载体 · 雌雄同体）**：身上镶着异种收集的 **9 颗时空结晶（异种称「陨星石」）**——故能提前半小时感知结晶落点（误差 < 30 米）；雌雄同体一分为二（一处镶石），外人一碰石就以雄体之死把石传给大后方雌体。被苏晓团队（巴哈 2 秒空间夺 + 幽灵妹侵体阻传送）破掉。
- **深红·塔莎（寄生种 × 人类 · 刀术宗师 Lv.35 · 三 boss 之首）**：力 149 / 敏 150 / 体 145、宽刃长刀技法型——「血吻」连斩致敌真实流血 / 治疗无效（团队坦 / 奶的噩梦）、风舞 + 魂舞 + 屠杀刃。一对一根本不是苏晓对手（「三个打一个都费劲」），靠拖时间等援军；惨败后欲自刎谢罪被异王制止。
- **巨猿·安基度（寄生霸主「白猿」· 殁）**：12 米白毛巨猿——皮糙肉厚、生存力 + 自愈逆天（苏晓斩二十多刀才用血之兽扑死），堪称「绝世好队友」以肉身挡苏晓护毒爪；殁后掉双圣灵级宝箱（安基度 / 白猿·可合成）。
- **毒爪（裂变种 · 空间限制者）**：粗右臂射剧毒 / 孢子黑骨刺——能**单体封锁苏晓的龙影闪（穿空间）10 分钟**，最棘手；全程猥琐躲精英堆后，被苏晓猎魔时刻 + 龙影闪斩首（只掉「未知生物组织」+ 85 功勋·气息像被严重透支潜力）。
- **重甲部队（异王杀手锏 · 专守要地对付强者）**：2500~3000 斤全身重甲 + 重武器的裂变种——锤地放「土黄冲击波」叠「大地震撼」减益（移速削 / 麻痹 / 脱力·满层强制眩晕）、巨斧异种如陀螺旋转碾人；无痛觉、不在乎误伤友军、专在建筑内耗死强者。秒了自闭兄、把苏晓也逼到受伤（建筑内硬拼极不智）。

## 强攻队（破要塞城门 · 吸火力）

- **金刚王 · 兽化**：变 12 米黑毛巨猿、吞黑能量球喷漆黑光柱崩城墙（防守时不用·会成活靶子）；与寄生巨猿安基度对撼。
- **萝尼（重炮少女 · 最疯一档）**：吟唱「共鸣」短棍变战镰、「黑暗镰」劈裂城墙——爱小动物、战斗中还忙着跟地行种尸体合影比剪刀手的疯丫头。
- **重炮男**：沉默寡言、一记「重炮」冲拳震崩 60 米城墙（开战 2 分 31 秒破第一层防御）。

## 自闭兄弟结局

潜入达普修要塞时，**自闭兄战死**——为让弟弟与全队活，在地下四层用螺旋钉吸引重甲部队、争取到关键的 5 秒（留言「要成功啊，白夜，哥哥他，不在了」）；自闭弟（植物化吃尸 / 屏蔽感知）以植物形态护送苏晓断后后撤离。

## 探索队（飞船班子的真身）

**癞蛤蟆 / 1045 号 / [夏](夏.md)**（+酒保珞珞留守）并非单纯飞船职工——前三人是轮回乐园方的**探索队**，乘飞船到万兽大陆只是顺风车，真正目的地是崩灭中的超巨型世界「**拉美西斯星**」（时空之力凝合体即它崩溃陨落而来）：须 7 天内夺得 **蠕行之灵**（**杀异王的关键物品**·得手后传回舰长室）。分工 = 夏交涉 / 1045 号战力 / 癞蛤蟆稳局；珞珞「真的只是个酒保」。

## 自杀小队伤亡（血色战场）

血色战场鏖战 + 断异种补给后，自杀小队从 6 人折损到「死三个半」：**塔利班玩家（活性生物炸弹）**终极自爆化灰、**黑寄生**胸口被打穿战死（坦然赴死）、**邪活种**被地行种撕碎死透、**小胖子**放黑魔吃光异种补给后生死未知（黑魔仍在异种堡垒附近肆虐·敌友不明）；**毒师**（胸口以下没了·爬回）与**幽灵妹**（化幽灵体·燃灵魂拖毒师脱险）重伤生还。

## 绅士同盟 · 疯人院渊源

院长、疯医生、以及苏晓的旧敌 **J 先生** 都出自**同一座疯人院**、同属「**绅士同盟**」（标志 = 衔尾蛇）。J 先生曾独立出去一段、险些被苏晓宰了。「这疯人院人才辈出」。

## 关系

- **[苏晓](苏晓.md)（库库林·白夜）**：本世界以「六阶烙印 + 七阶战力」获特殊任务、乘飞船弹射登陆 1 对 53 杀 50；与院长 / 疯医生 / 寡妇西 / 金刚王组成核心阵容，负责监督疯医生的违规潜入能力。
