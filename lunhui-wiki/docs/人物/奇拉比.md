# 奇拉比（八尾人柱力）

云隐村忍者、**八尾（牛鬼）人柱力**、[四代雷影·艾](四代雷影.md) 的义弟——也是 **现今最强的人柱力（没有之一）**：他与八尾**已达成合作关系**，能在完全尾兽化的同时保持神智清醒、且八尾会协助他战斗；战斗经验远胜寻常忍者。

## 替身骗过佐助（十五卷6–7章）

[宇智波佐助](宇智波佐助.md) 领鹰小队首次捕尾兽，自以为抓到了奇拉比、险些战死（脏器被打飞 40%+）——实则**被奇拉比用「章鱼触须替身」骗得团团转**（替身太逼真、连感知型 [香磷](香磷.md) 多次确认查克拉都没发现异常、抓过尾兽的鬼鲛 / 迪达拉也没察觉）。[苏晓](苏晓.md) 凭青钢影一探就识破「抓了个假货」（「我亲手抓过的尾兽比你见过的更多」），[带土](宇智波带土.md) 拧断「奇拉比」脖颈 → 现出章鱼触须，坐实佐助被耍。**真奇拉比本体安然脱身**（晓改派鬼鲛去抓）。原著中佐助也正是被八尾所骗。

## 关系
- **[四代雷影](四代雷影.md)**：义兄（情逾亲兄弟·为其「被抓」震怒召开五影大会）。
- **[宇智波佐助](宇智波佐助.md)**：抓他的鹰小队队长（实则只抓到替身、被骗）。
- **晓组织 / [苏晓](苏晓.md)**：欲捕八尾的猎尾兽组织。

## 本体被鬼鲛生擒（十五卷50章）

替身骗过佐助后，真奇拉比本体仍被 [鬼鲛](干柿鬼鲛.md) 盯上——鬼鲛藏身鲛肌内放白绝分身扮自己，鲛肌「叛变」更爱八尾查克拉、险些坏事；但**苏晓斩杀了义兄 [四代雷影](四代雷影.md)、令他失去原著里那记关键援军**（原著雷影驰援秒掉假鬼鲛），加上鬼鲛得「黑眸职工者妹子」[蔷薇](黑瞳-契约者.md) 助阵横跨铁之国穷追，他终在海边被生擒、扛回晓组织基地。八尾被抓、晓离复活十尾又近一步。

## 出处
- 十五卷6–7章（替身骗佐助·本体脱身）、十五卷50章（本体被鬼鲛生擒带回晓）。
