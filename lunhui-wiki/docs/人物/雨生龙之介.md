# 雨生龙之介

第四次圣杯战争中 **Caster（魔术师）的御主**——一名**喜欢杀人、痴迷于在他人临死时体验「死亡」滋味**的少年（尤好解剖小孩）。他无意间召唤出 Caster 职阶英灵 **吉尔斯·德·莱斯**（书房恶魔·原著与他到处杀人、寻找圣女贞德），自己却对圣杯战争一无所知、基本没参与其中。藏身冬木市下水道一带（留下浓重海腥味 + 碎尸现场）。

## 死亡（十一卷59章）

苏晓为完成隐藏任务「[英灵的荣耀](../世界/任务世界/fate-zero.md)」（杀御主清令咒）追查御主，让 [布布汪](布布汪.md) 循其血衣海腥味找到他的小木屋——撞见他正解剖一名被绑的小女孩。苏晓**不屑动刀、用一条炼金炸弹塑形的白蛇钻入其口中、再炸碎木屋**，将这名「毫无警惕、太弱」的御主连同其手术刀一并炸成齑粉（无任何奖励掉落）。其从者 Caster 不在场、亦未被诱出。隐藏任务御主减至 3 人、令咒 33 枚。

## 关系

- **[苏晓](苏晓.md)**：清令咒路上被随手炸杀的「杂鱼御主」。
