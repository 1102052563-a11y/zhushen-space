# 茜拉

[神皇冒险团](../势力/index.md) 的**二号人物**·绝美到令人窒息的女法师（黑蓝色法师袍·实力强·走原罪套装法师路线），也是团长 [神皇](神皇.md) 的**亲妹妹**。性格直率、口无遮拦——是个「**话题终结者**」（三句话就能把气氛聊死、还当众揭神皇「大保健」「欠钱」的黑料，气得神皇牙痒却拿亲妹妹没办法）。

- **与苏晓**：八卷5章替神皇来交易钢炼战利品（苏晓忌惮她单独前来必生死擂、神皇遂同行「不让她说话」）。她已集齐**原罪套装 3 件**（集 3/5/7 各解锁一条套装属性·满 7 = 法术伤害 +60%），故神皇势在必得苏晓那两件。成交后她**一度起意想抢回 [神灵余晖](../力量体系/装备道具.md)**（被苏晓察觉杀意当面警告），临走对苏晓竖中指、记错名字喊他「杀意蓬勃兄 / 杀气腾腾兄」。

## 结局 · 死于咕噜之手（八卷80章·侧面交代）
夏向苏晓转述的「乐园大事」：茜拉**被幻影旅团第 6 位 [咕噜](咕噜.md) 所杀**——起因是十几个大型冒险团争抢一套**「[猎魔套装](../力量体系/装备道具.md)」**（只有三件的暗紫色近战梦幻套装·其戒指在神皇冒险团手里、咕噜没抢到）。茜拉与咕噜本就有旧怨（八卷拍卖会上二人就剑拔弩张），这回咕噜直接要了她的命。她一死，**团长亲哥 [神皇](神皇.md) 暴怒、与幻影旅团正式「不死不休」开战**（详见 [神皇](神皇.md) / [势力](../势力/index.md)）。
