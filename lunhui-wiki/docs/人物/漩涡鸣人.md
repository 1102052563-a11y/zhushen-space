# 漩涡鸣人

[火影忍者世界](../世界/任务世界/火影忍者.md) 木叶忍者——**九尾人柱力**，原著火影世界的「剧情主角」。金发青年，[自来也](自来也.md) 的弟子（随其修行三年）。

- **此时的状态（九卷38章）**：经过与自来也三年修行已比少年时成熟一些，遇事虽仍有些冲动，但不再大惊小怪。在火影办公室与 [纲手](纲手.md)、[春野樱](春野樱.md)、[卡卡西](卡卡西.md) 一同得知砂忍村五代风影 [我爱罗](我爱罗.md) 被「晓」掳走的消息（鸣人与我爱罗同为人柱力）。
- **与一尾的封印对比**：他童年时根本不知道自己体内有九尾，封印远比我爱罗压制一尾守鹤的简陋封印高明——后者需我爱罗以意识时刻压制、甚至不能睡眠。

- **追击与暴怒（九卷43–44章）**：随卡卡西小队追到河之国山洞，破「五封结界」冲入时 [我爱罗](我爱罗.md) 已死。见挚友尸体、又见 [迪达拉](迪达拉.md) 坐在尸体上挑衅，鸣人**情绪失控、瞳孔转红、牙齿变尖、显出尾兽化痕迹**（他与我爱罗同为人柱力、同有悲惨童年、互为挚友）。被 [卡卡西](卡卡西.md) 拦下「冷静，这是陷阱」（晓正盼他靠近好顺手擒九尾）。迪达拉遂带我爱罗尸体引他追出山洞——苏晓评「该直接割我爱罗的头，九尾人柱力必冲过来」。

- **爆九尾尾兽炮（九卷46章）**：他被 [迪达拉](迪达拉.md)（坐我爱罗尸体挑衅 + 带尸引诱）彻底激怒，**爆出九尾、放尾兽炮**——把迪达拉炸断一手一脚（迪达拉狼狈逃命去找角都缝合）。苏晓评：迪达拉自己作死招惹九尾，鸣人若真爆九尾、连长门都不一定是对手，遂果断不掺和、提前撤离。

## 毁灭木叶之战 · 九尾爆走（九卷80–83章）
亲眼目睹卡卡西被 [苏晓](苏晓.md) 斩首，鸣人情绪彻底崩溃、**九尾爆走**——从一尾直接暴涨到四尾、又一分钟内冲到六尾、七尾、八尾，本人意识被九尾（九喇嘛）夺取（六尾就压得天道打、八尾时九喇嘛借身开口）。其间他**只认死苏晓一个目标**疯狂追杀（苏晓凭刀术 + 经验勉强撑过，靠牛头领主之怒 + 怪鸟脱身）；最终被天道「神罗天征」夷平木叶时一并卷入，与佩恩战成一团。**他体内的九尾仅是二分之一**（另一半被四代封印），却已强到苏晓认定「九尾与其他尾兽是两个概念、八只尾兽合起来未必是完全体九尾对手」——苏晓自知不敌、放弃捕九尾返回乐园。

## 感化长门 · 仙人模式追佐助（十五卷·苏晓再入火影）

第九卷后他成了「木叶英雄」——**感化了 [长门](长门.md)**（继承自来也意志、令其放弃毁灭木叶而死），但这「英雄」叙事的底色是木叶趁机吞下雨之国经济（见 [火影忍者·忍界公敌](../世界/任务世界/火影忍者-忍界公敌.md)）。十五卷他已掌握**仙人模式**（踩通灵蛙文太·伴深作 / 志麻两只蛤蟆仙人），率几十人追击团藏战场——苏晓本想趁忍界大战前抓他（守序善良的位面之子·这种主角好交涉、身边聚集大量友好契约者），但苏晓与他早是**死敌**（抓我爱罗 / 斩卡卡西 / 攻木叶 / 断雏田臂 / 斩犬冢牙）、且契约者太多，遂改派 [佐助](宇智波佐助.md) 当炮灰挡他。鸣人与开第三形态须佐能乎的佐助激战——他**九尾查克拉太暴躁、幻术对他无效**；他视佐助为最好的挚友、痛心其已变。
