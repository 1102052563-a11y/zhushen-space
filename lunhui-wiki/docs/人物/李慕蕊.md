# 李慕蕊

现实世界一名 **17 岁的叛逆期少女**——别墅主人之女、**将军 [方老](方老.md) 的外孙女**（母亲是将军之女）。富豪千金却是个**宅妹**，痴迷超能力影视与动漫，**一心想成为「不平凡的人」**，脑洞奇大（人称「脑洞少女」）。

- **撞见凶案（八卷83章）**：[苏晓](苏晓.md) 夜袭别墅区清剿三名外来者时，她**本该被清道夫疏散、却被疏漏**，躲在隔壁别墅衣柜里**目睹苏晓斩首 [巴赫](巴赫.md)**（第一次知道「人被斩首血会喷几米、无头尸还会抽搐」）。她先把苏晓当超能力者、又脑补成武侠片，**恐惧之余更多是对神秘的向往**。
- **求拜师**：苏晓拉开衣柜，她竟「噗通」跪下「**收我为徒吧，我能吃苦，说不定很有悟性**」（把苏晓都看得脸颊抽搐）。苏晓把她绑住塞毛巾、一度吓唬「考虑要不要灭口」。
- **化险为夷**：[魏东](魏冬.md) 查出她身份特殊（将军外孙女·灭口会招军方疯狂报复·且对苏晓无益），苏晓遂叫来她家人——正是旧识方老。苏晓戴**半脸面具**全程没露真容给她。最终方老严令全家**绝不外传**此事。

## 寻到饰品店再拜师（二十卷57章）

死神世界返园后，苏晓在现实世界开的饰品店里又撞见她——她特意找上门、戴茶色墨镜假装顾客，选了条「相对便宜」的项坠（被苏晓黑了 50 万压岁钱）。见吧台那把一米五重型狙击枪她吓得不轻，却仍坚持喊苏晓「师傅」（「我都磕头了，一日为师终生为父」）。苏晓一通「不买就宰了你」也没吓退她，只得打电话叫她爷爷 [方老](方老.md) 把她接走——几名壮汉把她拖上黑色 SUV，她还喊「师傅我还会回来的」。苏晓断定她不会再来（方老绝不许宝贝孙女再踏入「恶兽的巢穴」）。苏晓评：她这是对神秘的渴望、可这少女的性格第一个考核世界就会凉透。
