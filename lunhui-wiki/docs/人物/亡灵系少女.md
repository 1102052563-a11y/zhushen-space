# 亡灵系少女（八阶顶尖亡灵系）

[轮回乐园](../总览/乐园.md) 里一名八阶最顶尖梯队的**亡灵系**契约者——戴一顶直径一米多、软趴趴的灰色圆顶软布帽（让人怀疑她走路看不看得见），有一双墨绿色眸子。战斗时她是「亡灵大军踏平一切」的死亡化身，可一旦事关灵魂钱币就莫名卑微（第40卷78章 / 41卷77–85章）。

## 与苏晓的交易 · 世界之核残片渠道

- **卖卷轴起**：她最初卖给 [苏晓](苏晓.md) 一张 [基础被动·体魂](../力量体系/技能.md) 卷轴（换太阳三件套）。此后成了苏晓稳定的**世界之核残片供货商**——她从相熟的亡灵系契约者处筹货（每颗约 400 灵魂钱币）转卖苏晓（每颗 550~600·净赚差价·「够她几只永久性召唤物饱餐几顿」）。10 块一批 5000 灵魂钱币成交。
- **「你千万别死呀」**：她反复叮嘱苏晓别死在任务世界——「如果你死了，就没有金主收世界之核残片了」。她的主武器是 **无尽亡灵 +12**（不朽级·满镶嵌 3 不朽宝石·二次强化），证明亡灵系不是穷、而是消耗资源太恐怖。

## 关系

- [苏晓](苏晓.md)（白夜）：她的世界之核残片大主顾（叮嘱他千万别死）。
