# 10777 号违规者（融合的不死意识）

[Fate/stay night 第五次圣杯战争](../世界/任务世界/fate-第五次圣杯战争.md) 的**猎杀任务目标**——一名「外来入侵契约者」（从其他乐园偷渡来·因初始条例轮回乐园无法强制处决·须 [苏晓](苏晓.md) 代劳清理）。样貌普通、超速再生、意识混乱：在冬木市下水道大肆抓捕上百平民（不分男女老幼·想起初恋就抓女学生、想起祖父就抓老头），对一切生命包括自己都极度淡漠。

- **离奇身世**：他本是**天启乐园的 21478 号裁决者**，在某战争世界执行裁决时与轮回乐园小队交手，杀光对方后，拥有不死能力的**轮回乐园 10777 号契约者**暴走、把一片含 16 名契约者 + 这名裁决者的空间与自身融合。两个意识进入同一具超速再生的身体、撕扯融合，最终只剩一具躯壳与混乱意识——故他既是 10777 号、又是 21478 号。
- **被苏晓清理**：超速再生让 D·暗杀爆头都瞬愈，苏晓改用克再生的 [灵魂行者](../力量体系/装备道具.md)（灵魂撕裂）把他斩成几截、再用凝固汽油高温压制再生——临死他终于想起「我是谁」、含笑消散（被自身能力战胜的可怜人）。奖励 **黄金荣誉勋章 ×7**（如此高奖正因他是天启乐园裁决者）。

## 关系
- **[苏晓](苏晓.md)**：奉猎杀任务清理他的猎杀者。
- **天启乐园**：他原本所属（21478 号裁决者）。
