# 艳妖

[全职猎人世界](../世界/任务世界/全职猎人.md)「[绅士同盟](../势力/index.md)」成员——浓妆艳抹的女子，肩上常停一只**麻雀**（替她传递 / 侦查情报），毒舌。掌握一种**比隐身更高阶的「透化」能力**——能在所有人的感知中**凭空消失**（连苏晓也难以锁定），持双手匕首近身偷袭。

- **炮台伏击战中战死（十卷8–10章）**：随老大 [红色小丑](红色小丑.md) 设伏 [苏晓](苏晓.md)（红色小丑炸火车 + 花钱请旅团 [薇妮](薇妮.md) 当「炮台」压制）。她隐身偷袭刺穿苏晓胸口、又刺穿其手掌——却被苏晓「抓到了」一把攥住，连斩五刀**斩成支离破碎的尸体**。死后她的灵魂与 [犀](犀.md) 一同被红色小丑祭献、用以激活恶魔血统（甘愿赴死·可见红色小丑的人格魅力）。
