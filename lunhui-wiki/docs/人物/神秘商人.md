# 神秘商人

⚠ **一道背着大木箱、头戴古怪头套的身影**——⚠ 【**提示:神秘商人已因为鲜血祭坛的激活，来到此区域，预计在此区域停留 5～8 天。**】

## ⚖ 中立与规则

| 规则 | 内容 |
| --- | --- |
| **公证** | ⚠ **神秘商人为虚空之树所公证的中立单位**——⚠ **如尝试攻击，将承担多种恶果** |
| **无视魅力** | ⚠ **他将无视魅力属性的衍生效果**——⚠ **不会被高魅力属性所影响，也不会因低魅力属性而态度冷淡**（⚠ **对 −21 魅力的苏晓是好事**） |
| **收什么** | ⚠ **在永光世界，他只收血石**（⚠ 【纯净血石】——⚠ **因本源生命力被猩红所同化、冷却后的凝结体**） |
| **卖什么** | ⚠ **可在他处购买深渊特性装备，或其他罕有装备** |

## 🏃 被噩梦血影吓跑，又被拉回来

⚠ **苏晓刚出猩红圣殿，就看到他匆匆跑过**:「**年轻人，我们之后有机会再交易，老夫要尽快离开这。**」——⚠ 「因为一个噩梦化身？」⚠ 「**哦？你也看到那可怕的家伙了？**」——⚠ 「咳～，只是有所耳闻。」

- ⚠ **他的敏锐**:「**年轻人，你不对啊，那噩梦化身的气息，好像和你……**」——⚠ **说到此处，他的目光有几分震惊**;⚠ **哪怕他见多识广，也的确没见过战力那么恐怖的噩梦化身**——⚠ **他匆匆的步伐，直接改为小跑。**
- ⚠ **一句话把他拉回来**:⚠ **苏晓激活手上的监视者指环**——⚠ **这让他倒着跑回来，带着几分惊喜**:「**灰暗圣所？我和那老树是老朋友，不介意的话，我想去拜访下老朋友。**」
- ⚠ **进门的礼遇**:⚠ **刚回灰暗圣所，没等苏晓拔刀，一把锐剑的剑尖已经抵在他的眉心处，随后移开**——⚠ **是 [暗之女](永光世界/暗之女.md)**;⚠ **苏晓看了眼暗之女，暗之女收起锐剑后双手缚于小腹前，略对苏晓低头施礼，随后就回到座椅处慢慢品尝枫茶。**
- ⚠ **他的开场**:「**上次来这，还是几十年前，年轻人，要和我做生意吗，在永光世界，我只收血石。**」

## 🖤 深渊扭变结晶

⚠ **他展开一直带着的大木箱，各类被缩小后的商品展示出来**——⚠ **其中一块拳头大小的黑色【深渊扭变结晶】，吸引了苏晓的目光**;⚠ **苏晓的灵感飙升，他感觉此物十之八九能增益【贪婪之章】**（⚠ **二者的波动很相似**）——⚠ **详见 [噩梦血影](噩梦血影.md) 的「新准原罪物构思」。**
