# 鲁珀（弑君大局的幕后主使）

[死灵族](../世界/生物万族/index.md) 中**地位不低、却几乎没有战斗力的智谋型人物**——其强大全在**智商与布局能力**。他正是 [小公主弑君大局](../世界/任务世界/希亚大陆-灭法遗迹.md) 的**幕后总策划**：明明人不在王都，却把整个王族闹得鸡犬不宁、几乎一举颠覆 [圣索丁王国](../势力/index.md)（若非老国王临场应变，计划已然成功）。他的能力是「**伪装（成人类）**」——藏身王都附近的**布瓦村**、扮成一名满脸络腮胡、汗臭味十足的酣睡光棍农夫，连苏晓「若非得到准确情报，对面走过都不会起疑」。

## 被擒（十七卷35章）

[苏晓](苏晓.md) 从弟子 [伊儿朵](伊儿朵.md) 口中套出他的位置后，连夜赶到布瓦村——先让 [布布汪](布布汪.md)（「汪星人的秘密窥视」侦查）定位，再破墙一拳砸昏。鲁珀反应极激进（摸枕下绿色金属球武器·被苏晓抛飞出屋、在夜空炸出 300 米火球），随身还带爆炸物 + 三根毒针 + 手腕皮下毒囊（随时准备赴死、防泄密）。苏晓给他注射了**猎人世界枯枯戮山的麻痹毒液**（不致死但麻痹极强·防他醒来自尽），把他交给老国王——**完成隐藏支线「王族之秘」、并激活后续任务**。

## 关系
- **[敦克·尼克](敦克·尼克.md) / 萨甲 / [伊儿朵](伊儿朵.md)**：他的三名死灵帮手（前两者已被苏晓所杀·伊儿朵被俘并供出他）。
- **[苏晓](苏晓.md)（铁之手）**：生擒他的人（交给老国王换隐藏支线完成）。
- **[巴伊·赫伯特](巴伊·赫伯特.md)（老国王）**：他弑君的目标（反被老国王智慧化解）。
