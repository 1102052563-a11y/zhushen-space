# 左轮男（幻影旅团·编号7）

**幻影旅团**（专猎契约者的 PVP 冒险团·苏晓视为「同类」）成员、**旅团编号 7**，牛仔打扮、口叼雪茄、用左轮手枪。是 [苏晓](苏晓.md) 在乐园的**宿敌**之一。

- **与苏晓**：六卷七水之都期间二人交手过——苏晓凭「灵影体质」高生命硬抗、**把他「爆了膀胱」**（左轮男对此记恨）；卡尔（旅团成员）的编号也是苏晓出售的。
- **与神皇**：与 [神皇](神皇.md) 一方仇怨不可和解（杀过神皇 34 名部下）。八卷9章他持门票「不请自来」乐园拍卖会（15 号座），还带上了疯丫头 [咕噜](咕噜.md)；神皇当众与他互呛，黑帆商会则吓得龟缩后台。

## 出处
- 六卷5–8章（七水之都·提及）、八卷9章。
