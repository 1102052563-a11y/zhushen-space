# 机械妹

[轮回乐园](../总览/轮回乐园机制.md) **机械街**的契约者——蓬松金发、带黑眼圈（常熬夜），住处堆满机械（含一架拆到一半的机器人），掌握高阶**机械改造技能**。由旧识 [机油](机油.md) 引荐给 [苏晓](苏晓.md)。

- **改造装备（九卷90–94章）**：苏晓找她用 [物品改造装置](../力量体系/消耗品材料/index.md)（需机械改造 Lv.17+）+ 灵魂结晶〔中〕×2 作材料**升级蜘蛛女皇**（紫 → 暗紫·评分 260+·攻 60~134·17.36mm）。机油介绍苏晓时一句「他把我斩成六段后认识的」把她惊到。她为赶这单**四天没睡**、收 3 万乐园币手续费（激活临时契约交付），完工后秒睡。其工坊名「**爆炸小妹工坊**」。
