# 柯特·揍敌客（女装大佬）

[揍敌客家族](桀诺·揍敌客.md) 的成员、[桀诺](桀诺·揍敌客.md) 的孙辈——绰号「**女装大佬**」（容貌出众·一段「黑历史」常被打趣）。

## 东果托国的黑历史（十六卷8章）

[苏晓](苏晓.md) 在揍敌客家族餐桌上提到「东果托国」（嵌合蚁出没之地）时，柯特咬牙变脸——他曾去那个自治国执行任务，**那名国王看上了柯特的姿色、令他差点气得杀掉国王**（这段往事让布布汪听得津津有味）。席巴在苏晓食物下毒被点破时，柯特还一脸尴尬「怎么可能下毒」、转头默默吃饭。

## 关系
- **[桀诺·揍敌客](桀诺·揍敌客.md)** / **[席巴·揍敌客](席巴·揍敌客.md)**：祖父 / 长辈（同为揍敌客家族）。
- **[苏晓](苏晓.md)**：来揍敌客家族的雇主（在场闲谈）。
