# 涅茧利（十二番队队长 · 疯狂科学家）

[死神世界](../世界/任务世界/死神-尸魂界.md) 护廷十三队**十二番队（技术开发队）队长**——身披白羽织、戴黑白生物合成面具、手掌呈树脂般不正常白色的**疯狂科学家**，阴鸷、为「探索未知」可放下尊严。他对 [苏晓](苏晓.md) 又妒又贪：一心想**生擒苏晓、拷问出魂灵药 / [灵子球](../力量体系/消耗品材料.md) 技术**（他自己从食物里提取灵子的纯度不到灵子球的 0.15/10、远远做不到）。他的一身实力**七成在斩魄刀「疋杀地藏」上**。

## 斩魄刀失窃 · 疲于奔命（二十卷33–34章）

[布布汪](布布汪.md) 潜入戒备森严的十二番队实验楼（鬼道机关 / 灵子警报 / 暗哨全程「无视」），趁他研究灵子球时**把他的斩魄刀疋杀地藏与自己一同融入环境、悄无声息偷走**（死神与斩魄刀有灵魂联系、几百年来从没人想过会失窃）。涅茧利由此发现：自己与斩魄刀的**感知距离约 2.3 公里**——布布汪用斩魄刀逐步遛他（0.5→1→2.3 公里·留言「放弃吧、你不行的」）、把他耍得团团转、气到嘴唇发抖。丢刀是奇耻大辱（怕昔日仇人趁机寻仇），他迟迟不敢声张、发动全队大规模搜寻。事后他力主**审问苏晓**（实为想囚禁拷问）。

## 关系
- **[苏晓](苏晓.md)（白夜）**：想生擒拷问其药剂术的觊觎者（互看不爽·力主审问苏晓）。
- **[布布汪](布布汪.md)**：偷走他斩魄刀疋杀地藏的「真凶」（他全程没察觉）。
