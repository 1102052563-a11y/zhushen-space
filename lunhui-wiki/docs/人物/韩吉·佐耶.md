# 韩吉·佐耶

**韩吉·佐耶** 是 [进击的巨人](../世界/任务世界/进击的巨人.md) 世界 [调查兵团](../势力/index.md) 的**第二分队队长**、对巨人有狂热研究热情的女狂人（三卷5、13章）。

## 简介
戴眼镜、行事豪爽；对巨人的身体构造痴迷到近乎疯狂——见到会攻击同类的「[艾伦](艾伦.md) 兽」，第一反应是「**一定要生擒**」研究（怀疑它感染了能令巨人互攻的病毒、可用于以巨人制巨人）。战斗力虽不及兵长利威尔，但更懂巨人。

罗塞之墙攻防战中本守二区，发现其它战区被破便来救场。与苏晓（其身份是熟识的「前调查兵团成员」）并肩用立体机动装置压制艾伦兽、拽出艾伦本体，并力主把艾伦带去希娜之墙交团长定夺（三卷13–16章）。

## 开始怀疑苏晓（三卷37章）
攻防战后的总部晚餐上，韩吉佐耶疑心苏晓与女巨人接触过多，故意弄断椅腿、掀翻餐桌**试探**——[艾伦](艾伦.md) 受惊几乎变身（被苏晓锁喉、利威尔压身合力制服）。当夜她又独自来到男宿舍，借「询问女巨人资料」试探苏晓护臂的来历（苏晓答「废弃民宅捡的」）。苏晓由此判断：艾尔文与韩吉佐耶已开始怀疑他的真实身份，唯利威尔暂时蒙在鼓里（三卷37章）。

## 升任代理团长（三卷46章）
调查兵团追击叛逃的苏晓一行至玛利亚之墙外时，苏晓用破碎精灵「凝视射击」**一枪爆掉团长艾尔文的头**。艾尔文当场身亡、调查兵团群龙无首。[利威尔](利威尔.md)（战力强但无指挥才能）当即指定韩吉·佐耶**代理团长**，由她接过指挥、率众继续追击。

## 关系
- 艾尔文——[调查兵团](../势力/index.md) 原团长（其上司，**三卷46章被苏晓爆头身亡**）；[利威尔](利威尔.md)——兵长（战力强于她，推她代理团长）。
- [苏晓](苏晓.md)（白夜）——熟识的「前调查兵团成员」、得力搭档，但已开始**怀疑**其身份（与巨人接触过多）。
- 利威尔——同僚兵长（战力强于她，被她劝阻与苏晓互掐）。
- [艾伦](艾伦.md)——她眼中价值连城的研究对象（会攻击巨人的「奇行种」）。
