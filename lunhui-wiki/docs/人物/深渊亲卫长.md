# 深渊亲卫长

> ⚠ **上一任深渊之主麾下有两位左膀右臂:偏谋略的「深渊总督」，与武力代表「深渊亲卫长」**——⚠ **他是后者，持有昔日深渊之主的武器「深渊大剑」**;⚠ **⚠ 身高近十米的四臂漆黑巨汉·下颚触须·木质化犄角·浑身睁开大小各异的眼睛·八颗心脏**（⚠ 曾被苏晓的师傅「灭法者·[马文·华尔兹](#)」重创、沉寂多年·大部分深渊系能力都处于损伤状态）。

## 🕳 被封印于不死君主体内

⚠ **他就是被封印在 [不死君主](不死君主.md) 体内、逐渐侵蚀不死君主心智的那个存在**——⚠ **这是白金使徒与席克托合谋的核心**:⚠ **一个曾拥有、一个正拥有「禁忌印记·命定印记」的两人，初见就心照不宣**——⚠ **他们要把「本时代的天选」[苏晓](苏晓.md)、以及 [神父](神父.md)、[豪梼](豪梼.md)，一起埋葬在这囚笼世界**（⚠ 不死君主一放开封印、深渊亲卫长破封而出、本世界开始崩溃、谁也逃不出去）;⚠ **⚠ 临时大队 27 人，被它以「黑金属陨星」连环轰炸后只剩苏晓、神父、豪梼、泰莉德、凯恩、水哥六人**。

## ⚔ 无解的暴力（战斗方式）

⚠ **它没有花哨能力，是「力量与速度的结合体」，越是这种简单的强大越无解**——⚠ **本世界压制它无法达到至强、却也因它破封而崩溃**;⚠ **⚠ 恐怖机制**:

- ⚠ **⚠ 深渊压制**:⚠ **与它身体接触，敌人所有能力会变灰、进入「深渊压制」不可用状态数小时〔接触越久压制越久〕**（⚠ 巅峰时它周边几公里内别想用任何能力）;⚠ **⚠ 850% 生命值上限+断颈 1 秒再生+越挨打防御越强**·⚠ **无所不吞噬巨口〔下臂手心裂开成尖牙圆口·连斩龙闪都能咬断〕· 黑暗吞噬漩涡· 暗域·噬时〔停滞时间〕。**
- ⚠ **⚠ 绝强与至强的鸿沟**:⚠ **九阶→绝强战力提升是 100，绝强→至强是 30000~50000**——⚠ **苏晓的至强级大招「极刃·白夜」对它平A，只换来「你力量差了点」的评价**（⚠ 它随攻击越战越强、把坦系·哈姆迪的塔盾夺去化陨星、把凯恩撕下大半身躯）。

## 💀 死于晋升失败（54卷 · 第3936–3938章）

⚠ **它的关键弱点是「不灭躯体」需要苏晓的刃之魔灵去压制**——⚠ **它故意等苏晓用魔刃压制它、借机力量飙升五成、一口咬断苏晓的整条右臂**;⚠ **⚠ 苏晓的反杀**:⚠ **右臂被吞前，他已把【黑暗蠕动】〔前代深渊之主的脑髓液〕攀附在袖口手臂上封好、让深渊亲卫长「品尝了它昔日主上的味道」**——⚠ **深渊亲卫长因此被强行催动「晋升深渊之主」**;⚠ **⚠ 可上一任深渊之主还没消亡、加之它非巅峰状态，晋升成功率是 0**——⚠ **它死于晋升失败，来自深渊最强梯队的不死不灭也因「晋升会自行临时脱离不死状态」而失效**（⚠ 临死反扑即死+斩杀，苏晓靠 2 颗刃之心豁免）;⚠ **⚠ 苏晓被吞的右臂，融入了一股极纯粹顶尖的深渊之力。**
