# 艾斯德斯

**艾斯德斯** 是 [斩·赤红之瞳](../世界/任务世界/斩赤红之瞳.md) 世界**帝国的王牌女将军**、帝国暗部的统一管理者（四卷12、14章）。

## 简介
身材高挑、满头**蓝发**垂腰，即便在零下数十度的极北也只穿性感清凉的白色军装；遇强敌便极度兴奋的**抖 S 女王**（眼中似有「冰焰」燃烧）。身手迅敏，惯用**细剑**。她奉小皇帝之命率帝国 50 万大军 + 王牌镇压极北异族军、夺回了极北要塞（四卷14章）。

- **冰帝具「恶魔之粹」**：一种由危险种鲜血制成的**鲜血类帝具**，饮下后若能承受血中疯狂意识，即获**操控寒冰**之力。她在身体素质上不及海贼世界的冻冻果实能力者青雉，却有一点青雉无法比拟——**能冻结空间**（可凭空生成冰刃、冰刺、冰墙；招式如「白冰碎刺」「白冰平原」）；极寒气候更适合她战斗，可一掌**秒杀数万**冲锋的巨狼成冰雕。本身六维属性应不逊于苏晓、甚至更高（四卷12、14–17章）。

## 顶级将才 · 军中声望（四卷15章）
虽是抖 S、嗜杀虐敌，治军却堪称**顶级将才**：帝国赏赐的金钱全用于犒赏士兵、与兵同食、不开小灶、麾下从不受贿、与平民融洽——军中声望极高，**若举旗造反成功率逾九成**。腐朽帝国能撑到今日，很大程度靠她麾下的数十万军队（加上帝国手握众多帝具）。对认可的部下极为宽宏关切（部下犯错只稍加处罚），认人不认来历——「只要是强者一律接纳，若不怀好意就剥皮扔火里」。

## 收苏晓为副官（四卷14–18章）
苏晓以「云游旅者」之名到极北要塞求加入帝国军、被疑为探子，与艾斯德斯交手（她兴味大起、互有试探）。苏晓借「五万极北军 + 雪原狼来袭」化解、随她退守要塞。艾斯德斯带他去除掉与极北军勾结的超级危险种 **[芬里尔](../世界/生物万族/index.md)**（巨狼·速度快到残影）——苏晓潜入偷袭、最后**飞掷斩龙闪贯喉**钉死芬里尔、救下被其抓伤的艾斯德斯。艾斯德斯遂任命苏晓**顶替老副官杰罗姆、做她的直属副官**（既用其能、也便监视这名「空降的他国暗部」）。苏晓自报伪装身份「白夜·帝国东方帕克斯亚森人·前杀手/佣兵/军队暗部（要员暗杀·斩首）」，理想「给钱给地位就杀人」——正中艾斯德斯下怀（视为捡到宝的无家可归前暗部）。

## 识破苏晓 · 决裂（四卷57章）
苏晓绑架小皇帝、潜入祀堂取钥匙时被艾斯德斯率几百侍卫（含威尔）当场包围——**破绽是威尔灵敏的鼻子嗅出苏晓手上有黑瞳的特有香味**（苏晓曾谎称没见过黑瞳），艾斯德斯又早在皇宫布下眼线。她沉声「我曾……很信任你」，见苏晓**第一次露出微笑**（「我其实不是暗部」）而恍然——遂翻脸「不是你死就是我亡」、脚下冰晶升腾。但苏晓已抢先取钥匙、**当众完成主线任务**（钥匙破碎、大臣气急败坏）。她对生死并不看重（嘲讽怕死的大臣「最多被革命军围杀」），与苏晓的决战一触即发——印证苏晓早先「与艾斯德斯必有兵戎相见之日」的预判。

## 关系
- 帝国小皇帝 / 大臣——其效忠的帝国（实权在操控小皇帝的大臣手中·大臣唯一忌惮她）；[布德大将军](布德大将军.md)——皇宫近卫统帅、实力略逊她的同袍。
- [苏晓](苏晓.md)（化名白夜）——她收为直属副官的强者（杀芬里尔救她一命、又奇袭斩杀努马·塞卡）；她不知苏晓真实身份与图谋。
- 杰罗姆——年迈的前任副官（被调帝都后勤养老）。
