# 艾斯德斯

**艾斯德斯** 是 [斩·赤红之瞳](../世界/任务世界/斩赤红之瞳.md) 世界**帝国的王牌女将军**、帝国暗部的统一管理者（四卷12、14章）。

## 简介
身材高挑、满头**蓝发**垂腰，即便在零下数十度的极北也只穿性感清凉的白色军装；遇强敌便极度兴奋的**抖 S 女王**（眼中似有「冰焰」燃烧）。身手迅敏，惯用**细剑**。她奉小皇帝之命率帝国 50 万大军 + 王牌镇压极北异族军、夺回了极北要塞（四卷14章）。

- **冰帝具「恶魔之粹」**：一种由危险种鲜血制成的**鲜血类帝具**，饮下后若能承受血中疯狂意识，即获**操控寒冰**之力。她在身体素质上不及海贼世界的冻冻果实能力者青雉，却有一点青雉无法比拟——**能冻结空间**（可凭空生成冰刃、冰刺等）（四卷12、14章）。

## 与苏晓初遇（四卷14章）
苏晓以「云游旅者」之名到极北要塞求加入帝国军，被士兵当作极北军探子围捕。艾斯德斯现身、拔细剑亲自试招——苏晓以斩龙闪挡住其一剑，她兴味大起（「居然挡住了」），随即凭空生出一排冰刃刺向苏晓（交手进行中）。

## 关系
- 帝国小皇帝——其效忠的幼主；帝国暗部由她统一掌管。
- [苏晓](苏晓.md)（旅者身份）——初遇即试招开打的强敌。
