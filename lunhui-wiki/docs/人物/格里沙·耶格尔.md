# 格里沙·耶格尔

**格里沙·耶格尔** 是 [进击的巨人](../世界/任务世界/进击的巨人.md) 世界 [艾伦](艾伦.md) 与 [吉克·耶格尔](吉克·耶格尔.md) 的父亲、墙内医生兼巨人学研究者，也是**进击巨人之力**的前任传承者。其经历由苏晓在艾伦家地下室翻出的**三本日记**揭明（三卷44–45章）。

## 生平（地下室日记 · 三卷44–45章）
- **童年**：生于 [马莱帝国](../势力/index.md) 的艾尔迪亚人「隔离区」；幼时带妹妹溜出隔离区看飞艇，妹妹被马莱人**放狗咬死**——这成为他一生的恨与愧。
- **复兴会**：成年后加入秘密组织「**艾尔迪亚复兴会**」，得知会中神秘成员「**枭**」竟是马莱政府内部人员（复兴会因此屡屡逢凶化吉）。在会中结识并娶艾尔迪亚王族 **黛娜·弗里茨**，生长子 [吉克·耶格尔](吉克·耶格尔.md)；他以复兴会思想教育吉克，并在巨人之力换届时送 7 岁的吉克去参加选拔，结果**被吉克出卖**、夫妇遭马莱清算。
- **传承进击巨人之力**：黛娜被变成无垢巨人流放帕拉蒂岛，格里沙则被「枭」所救。枭真名 **艾伦·克鲁格**（即当年抓他兄妹的两名士兵之一、掌进击巨人之力），将进击巨人之力**传承**给格里沙（格里沙化无垢吞食了他）；枭还告知他「**初代王与体内始祖巨人之力签订『不战契约』、永不与马莱交战**」的真相。
- **潜入墙内**：格里沙借复兴会所学潜入墙内、借平息瘟疫建立威望；娶第二任妻子**卡尔菈·耶格尔**，生次子并取名「**艾伦·耶格尔**」（为纪念已故的枭·艾伦·克鲁格）。日记末他向次子艾伦忏悔、立誓不再向其灌输思想。
- **结局**：玛利亚之墙被破当日，他给艾伦注射巨人药剂、被艾伦吞食，将**进击 + 始祖巨人之力**传给艾伦，并留下钥匙嘱其回地下室寻真相（详见 [艾伦](艾伦.md)·三卷18章）。

!!! note "钥匙'并没什么卵用'"
    格里沙嘱艾伦贴身保管五年的地下室钥匙，被苏晓一脚踹开门后随手抛还——真正的收获是地下室里能**出售为世界之源**的实验物品与一瓶蓝色 [巨人精炼提取液](../力量体系/消耗品材料/index.md)（三卷43–45章）。

## 关系

- [吉克·耶格尔](吉克·耶格尔.md)——长子（黛娜所生），出卖了他。
- 黛娜·弗里茨（艾尔迪亚王族·前妻）/ 卡尔菈·耶格尔（艾伦生母·后妻）。
- 「枭」艾伦·克鲁格——进击巨人之力上代传承者、复兴会内的马莱政府人员，救他并传力。
