# 阿尼·利昂纳德

**阿尼·利昂纳德** 是 [进击的巨人](../世界/任务世界/进击的巨人.md) 世界**女巨人**的传承者，[马莱帝国](../势力/index.md) 派入墙内的间谍之一（三卷14、22章）。

## 简介
黄发、刘海遮住半边脸、总是一副漠不关心高冷模样的少女；13 岁起潜入墙内训练兵团（与莱纳·布朗、贝特霍尔德·胡佛组成「墙内智慧巨人三小组」），任务是破坏神之壁、杀光墙内人类。长期与墙内人共同生活令她意志动摇——见同期被巨人吞食会崩溃，是个「不合格的间谍」（三卷22、26章）。

**女巨人**形态约 14 米：力量略逊艾伦、但速度快几倍，是「人类杀手」，配合本体格斗术更强；有**硬化**（受击部位结晶、巨人世界刀刃难破）与**晶体化**（把本体包进水晶棺自保）能力（三卷23–26章）。

## 与苏晓的交手（三卷22–26章）
为阻止艾伦堵洞而现身，被苏晓断头台几乎勒死、被迫变身。苏晓以立体机动 + **蓝色斩龙闪**（连硬化都斩透）反复斩断其腿、以心眼闭眼感知躲其攻击，硬生生把女巨人压榨到只剩头与一臂；阿尼自我晶体化求生。危急时苏晓一番**诈术**（自称也是「马莱人」、称她父亲被关「隔离区」、让莱纳与贝特霍尔德来找他）骗得阿尼信以为真、自解水晶，被赶到的铠之巨人莱纳吞下带走。

## 后续（三卷38章）
因身份在苏晓面前暴露，阿尼无法再于墙内潜伏，暂时躲藏于墙外。苏晓用诈术骗取 [莱纳·布朗](莱纳·布朗.md) 信任后，让莱纳联络阿尼，将她拉入「抓走 [艾伦](艾伦.md)」的计划。劫走艾伦逃出城墙后，阿尼变身女巨人赶来汇合，此后与莱纳轮流变身载众人横穿帕拉蒂岛、逃往墙外；她对苏晓仍保持戒备距离。其外表坚强、内心却脆弱（曾在苏晓面前落泪）。同伴 [贝特霍尔德·胡佛](贝特霍尔德·胡佛.md) 暗恋她，但她最多视其为同伴（三卷40–43章）。

## 逃出生天（三卷49章）
苏晓翻脸屠杀间谍组（莱纳、贝特霍尔德、艾伦、尤弥尔接连被杀）时，因阿尼当时离苏晓较远、且女巨人速度快，她在 [尤弥尔](尤弥尔.md) 拼死变身颚之巨人拖延、吐牙逼停苏晓的掩护下，**抓起 [赫里斯塔](赫里斯塔.md) 变身女巨人逃脱**——成为巨人世界结局中**唯一幸存的智慧巨人传承者**（随后世界崩坏、苏晓回归，她的下落未再交代）。

- [苏晓](苏晓.md)（白夜）——交手完全不敌、又被其诈术所骗。
