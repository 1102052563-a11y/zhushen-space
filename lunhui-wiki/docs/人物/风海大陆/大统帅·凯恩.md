# 大统帅·凯恩

兽族**两位大统帅之一**——须发皆白、身形偏瘦、鹰视狼顾、拄手杖，一双看似浑浊实则暗藏犀利的眸子，气场强到让周边亲卫都不敢与他对视。他是现任兽王（钢羽家族）**左右手中的「右手」、绝对心腹**，掌管兽族广袤地盘上的一百多位领主。他是苏晓所伪装身份**怒泽·戈温领主**多年的死对头。

## 🕸 把戈温收拾得明明白白的因果系操盘手

⚠ 凯恩权谋深不可测，「几十个戈温也斗不过一个凯恩」。戈温两次想复仇——一剑砍下「凯恩」的脑袋、晚宴上格杀「凯恩夫人」——都因凯恩身边一名**因果系高手**（在戈温的传家宝**骨齿项坠**上做了手脚·厄运物·注定戈温杀不死凯恩），砍中的都是海族长老会**深海女妖**伪装的替身。凯恩借此既清除了潜伏身边的海族隐患、又把戈温送进监牢、还把戈温家族收拾到几近退出兽族权贵层。⚠ 凯恩**从没打算弄死戈温**（害死一名领主是大事、会引众领主兔死狐悲、还得罪垂老而威仪犹存的兽王），「让他一直在牢里，才是我想看到的」。

## 🎭 假死破局 · 与苏晓合作（50卷4章）

⚠ 戈温入狱后，**七大兽族家族同盟**（战熊 / 胡狼 / 蛇巫 / 占星 / 风暴 / 铁纹…）想拉钢羽家族下王位，遂在戈温的牢饭里下**无解猛毒**——若戈温毒死狱中，凯恩百口莫辩、兽王必须给态度。⚠ 凯恩亲赴牢房，坦言这是「二对七、有心无力」的败局，心口刻印悄然生效、准备**假死**在此以瓦解家族同盟的算计。可他没想到，牢里的「戈温」是灭法苏晓——苏晓早把厄运骨齿项坠冲进马桶、亮出灭法印记与骇人气息。⚠ 凯恩起初冷淡（「这是兽族自己的事、和你们乐园无关」），但一听苏晓是「炸了施法者资源星的灭法」、且此行有**狩猎奥术永恒星绝强**之意，态度立刻转暖——双方有共同敌人，合作趋于稳定。他反复试探苏晓会不会深度干预兽族内政（确认「不会」后满意），最终谈成:**苏晓牵制风海大陆所有施法者，换领主之位 + 一块封地**。他让戈温「把领主之位世袭给侄子」（省去册封麻烦），又抛给苏晓一个**领主威仪**阵营任务（声望 >100 才给封地）——既要苏晓吸引施法者仇恨，又不轻易掏封地，老奸巨猾。

## 关系

- [苏晓](../苏晓.md)（灭法·伪装戈温）：他多年死对头的「继任侄子」、与他谈成「牵制施法者换封地」的乐园合作者。
- 怒泽·戈温：被他用因果系收拾到家破、送进监牢、又「假死」利用的老仇敌（真身已被苏晓顶替）。
- 兽王（钢羽家族）：他誓死效忠的君主——他的假死布局，正是为兽王瓦解七大家族的逼宫。
- 七大兽族家族同盟：要拉钢羽家族下王位、给戈温下毒陷害他的对手。
