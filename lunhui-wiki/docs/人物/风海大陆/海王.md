# 海王

风海大陆**海族阵营的最高统治者**——身着暗金色鳞甲、满头如细蛇般的灰发，端坐王座。海族由**海王 / 海神教 / 长老会**三方组成:海王权力最大，长老会（暗部 + 处罚 + 监管机构）次之，海神教（信奉**初代海王**为神、即现任海王先祖，故动不得根）再次。

## 🌊 与奥术永恒星的交易

⚠ 海王为在与兽族的持久战中占上风，**囚了三大传说铁匠之一的矮人王**——用矮人王打造的、能增长自然元素的器物，向**奥术永恒星**换来施法者的强援（如 [瑟菲莉娅](../瑟菲莉娅.md)、[魂手·法斯特](魂手·法斯特.md) 等），一度把兽族压得节节败退（兽族遂去找陨灭星结盟）。他手握的**初始印记**（曙光乐园钥匙）是苏晓此行的最终目标。⚠ 他也懂得分寸:海族找奥术永恒星当帮手，但绝不会蠢到引狼入室、把施法者们都招来本世界。

## 🐚 遥询白蹄港

⚠ 白蹄港失守时，海王经**感知水母**遥询珀涅罗，语气平缓、**未责罚**这位镇守多年的心腹——他甚至担心珀涅罗在主战场遇险，特意把亲卫长**黑兆·阿波努斯**调到其身边保护。他没有王座上常见的暴虐，是个懂得用人、维系三方平衡的统治者。

## 关系

- [海族统领·珀涅罗](海族统领·珀涅罗.md)：他镇守白蹄港多年的绝对心腹——失守亦未责罚。
- 海神教（信奉初代海王）/ 长老会：他麾下 / 制衡的另两方势力。
- 矮人王：被他囚禁、以器物换奥术永恒星助力的三大传说铁匠之一。
- [苏晓](../苏晓.md)（灭法·伪装戈温）：抢走白蹄港初始印记大块碎片、牵制其施法者盟友的乐园入侵者。
