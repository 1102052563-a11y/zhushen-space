# 灵魂死神（殁）

一位存在了**数千年**的灵魂系神灵——身高十几米以上、身披勾金线的黑色大袍，松垮兜帽下是浓郁的黑暗与**五只透出灵魂蓝白色的眼睛**；空间漩涡周边探出一只只骨手扣住边缘、顶部一排骷髅头，宛如一道通往幽冥的死亡之门。他身份众多:**黑暗神教三大无上神祗之首的「死神」**、部落阵营**牧魂部落**所信奉的神灵，苏晓推测他还另有身份。

## 🎣 「灵魂凝视」——垂钓千年的诱饵

⚠ 他毕生所图是**掠夺灵魂系强者的灵魂源质**，以此封临「绝强」乃至加冕「至强」（神灵系前中期极强、后期晋升难如登天，他已累积很久）。⚠ 他的手段极为阴险:让麾下信徒故意散播**「灵魂凝视」技能卷轴**——这能力性价比高到离谱（豁免控制后反制施术者，若伤害超敌人最大生命 60% 即触发**灵魂斩杀**），引诱灵魂系超凡者掌握并大力发展；等其实力养肥，灵魂死神再来收割灵魂源质。⚠ 每次「灵魂凝视」触发斩杀，其实就是灵魂死神亲自来割韭菜——[苏晓](../苏晓.md) 每次用都心中发寒、常把它调成沉寂状态，直到猎杀名单上出现「灵魂死神」才想通这一切。

- ⚠ 讽刺的是，苏晓那张卷轴是**团长得到后让咕噜到交易街低价卖给他的**，且**经轮回乐园公证**——灵魂死神根本无法激活副作用、摄不走他的灵魂源质，等于苏晓**白嫖了这个陷阱**千年。

## ⚔ 被两次「无伤近身」锤杀（50卷32–33章）

⚠ 他极谨慎（历来只让信徒出面、连一个部落倾力祭献都钓不出他），却栽在**弑神队**手里:[凯撒](../尼古拉斯·凯撒.md) 的**地精跳**（人罐合一 + 欺诈者头裹 + 黑暗蠕动伪装）钓出了他——他单手托起盛满「灵魂源质」的仪式容器、赞一句「你很虔诚」，全程不出空间漩涡，随即容器炸开，苏晓一记蓄满力的**直踹**（近战宗师 Lv.74 + 青钢影 + 神父的古神削弱 + 巴泽的最强巫毒）踹碎他胸膛的外骨骼——「那一瞬间它感觉自己的身躯就像被一颗星球撞了般」。

- ⚠ 他至死不解:自己以「灵魂力场」干涉空间、把「无形魂链」布满百米，苏晓凭什么还能与刃之魔灵**互换位置**？答案是——那**任何系都不是**，它就是最纯粹的「位置互换」（灭法之影三大最强能力之二，另两种是「魔刃斩杀」与一种苏晓尚未唤醒全貌的能力）。他因此**两次被无伤近身**:一次挨满蓄力的『刃道刀·极』斩断手腕，一次脑袋挨满蓄力侧踢。
- 他还被**青鬼刀链**的新战术阴到:巨型青鬼故意让他躲开，魔灵收回刀链时，50 道小号青鬼以更快速度、更强切割力从背后斩穿他。
- ⚠ 最终苏晓在**未进入第二阶段（不灭体质）**的情况下就把他锤死。这并非他不抗揍——他确有九阶顶尖梯队实力、神体强大、中距离强势、压制力极强——而是苏晓得**唤醒之碑**后综合战力有了质的飞跃，且他全程处于神父的抗性削减与巴泽的巫毒之下。

## 💀 被夺魂 · 饮血 · 噬肉 · 窃财

⚠ 弑神队「一点都不浪费」:苏晓以提取装置抽走 **2000 多盎司液态「灵魂源质」**、[神父](../神父.md) 用黄金圣杯抽干神灵源血畅饮、[巫毒术士·巴泽](巫毒术士·巴泽.md) 嘴角裂到耳根大口吞噬神灵血肉、凯撒扛着大布袋卷走他空间器物里的全部身家——地上只剩一颗两米高、五个眼洞、遍布裂痕的神灵颅骨。⚠ 欲神与蛇神经空间镜像目睹这一幕（本想坐收渔翁之利），当场倒吸凉气、匆忙关闭镜像——**蛇神当夜就逃出了风海大陆**。

## 关系

- [苏晓](../苏晓.md)（灭法·猎神者）：斩杀他、夺其灵魂源质与神魂的死敌——反被他的「灵魂凝视」白嫖了千年。
- [神父](../神父.md) / [巫毒术士·巴泽](巫毒术士·巴泽.md) / [凯撒](../尼古拉斯·凯撒.md)：分食他神血、血肉、财富的「弑神队」另三人。
- 欲神 / 蛇神：黑暗神教另两位无上神祗——目睹他的下场后心态裂开（欲神随即被人马族重骑兵射爆，蛇神连夜逃离本世界）。
