# 兽王

风海大陆**兽族阵营的最高统治者**——身高近三米、披钢羽披风、头戴王冠，须发皆白、骨架很大、皮肤已显灰白，需两名仆从搀扶才能落座，接连干咳、要喝药剂才缓得过来。⚠ 但**哪怕垂垂老矣、神情平和，依旧无人敢与之对视**；他的亲卫队人均九阶巅峰、内含一名**绝强者**。他的王座以**世界树的树桩**雕凿而成，铁黑色如被火焰烧灼过，意义非凡。[苏晓](../苏晓.md) 第一眼见到他就觉不可思议:此等生命状态下，为什么还能活着。

## 👑 一眼看穿虫族的老王（50卷27章）

⚠ 一百四十多位领主为讨要**菌毯**，与大统帅·凯恩合谋把事闹到兽王面前。大议厅上兽王**睁着眼睛就睡着了**，等领主们汇报得差不多、凯恩一提菌毯，他刚好醒来、只说了一句「白夜，我听说，你最近开发出了种菌毯」——关于菌毯的讨论就此为止（兽王从不直接下令，那一句已足够明显）。⚠ 真正的重头戏在散会前的闲聊:
- 「**你那菌毯，很像虫族的产物。**」——一句话点破苏晓最大的秘密。
- 他还知道灭法的底细:「为了应对去永光的历练，你也是煞费苦心，要我说，你们灭法派系的先代灭法们，精神都有些问题，哪有让后辈去永光历练的，那是历练吗」；又调侃永光世界的宝库「我感觉那宝库里什么都没有，别怀疑，马文·华尔兹和格林·吉莉安那伙人干得出这种事，那伙人里我看阿卡斯还算靠谱，但这靠谱的家伙死得也最早」。
- ⚠ 他拍着苏晓的肩膀定下底线:「**尽快发展虫族吧，但别在这里孵化战斗虫族，哪怕是一只，都不行**」「菌毯的来历，我帮你保守，世间的事其实就是这样，各取所需就好，谁还没有那么一两个秘密」。

## 📜 一张纸条 · 恩威并施

⚠ 兽王临走时借亲卫塞给苏晓一小截羊皮纸:「**今日勿离永环城，海族、施法，多名绝强伏杀**」——正是这张纸条让苏晓避开了四名绝强者的必死伏杀。⚠ 这既是对苏晓领主身份的认可、合作的诚意，也是**恩威并施的警告**:施法者与海族在兽族地盘上的一举一动都逃不过他的眼耳——同理，若苏晓违背承诺让虫族暗中爆兵，也同样逃不过王下禁卫。可这警告裹在救命情报里，就格外容易接受了。

## 关系

- [苏晓](../苏晓.md)（灭法·凛冬领主）：他一眼看穿其虫族、却选择替他保密并示警的合作者——底线是「别在这孵化战斗虫族」。
- [大统帅·凯恩](大统帅·凯恩.md)：他的绝对心腹、左右手中的「右手」——由凯恩出面分配菌毯，既拉拢领主、又让七大家族不敢再明争暗斗。
- 七大家族：暗中联手要拉钢羽家族下王位的对手——菌毯之功让众领主倒向兽王，令他们不敢造次。
