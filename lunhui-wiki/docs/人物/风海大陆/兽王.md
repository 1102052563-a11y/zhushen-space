# 兽王 / 老兽王（殁）

风海大陆**兽族阵营的最高统治者**——身高近三米、披钢羽披风、头戴王冠，须发皆白、骨架很大、皮肤已显灰白，需两名仆从搀扶才能落座，接连干咳、要喝药剂才缓得过来。⚠ 但**哪怕垂垂老矣、神情平和，依旧无人敢与之对视**；他的亲卫队人均九阶巅峰、内含一名**绝强者**。他的王座以**世界树的树桩**雕凿而成，铁黑色如被火焰烧灼过，意义非凡。[苏晓](../苏晓.md) 第一眼见到他就觉不可思议:此等生命状态下，为什么还能活着。

## 👑 一眼看穿虫族的老王（50卷27章）

⚠ 一百四十多位领主为讨要**菌毯**，与大统帅·凯恩合谋把事闹到兽王面前。大议厅上兽王**睁着眼睛就睡着了**，等领主们汇报得差不多、凯恩一提菌毯，他刚好醒来、只说了一句「白夜，我听说，你最近开发出了种菌毯」——关于菌毯的讨论就此为止（兽王从不直接下令，那一句已足够明显）。⚠ 真正的重头戏在散会前的闲聊:

- 「**你那菌毯，很像虫族的产物。**」——一句话点破苏晓最大的秘密。
- 他还知道灭法的底细:「为了应对去永光的历练，你也是煞费苦心，要我说，你们灭法派系的先代灭法们，精神都有些问题，哪有让后辈去永光历练的，那是历练吗」；又调侃永光世界的宝库「我感觉那宝库里什么都没有，别怀疑，马文·华尔兹和格林·吉莉安那伙人干得出这种事，那伙人里我看阿卡斯还算靠谱，但这靠谱的家伙死得也最早」。
- ⚠ 他拍着苏晓的肩膀定下底线:「**尽快发展虫族吧，但别在这里孵化战斗虫族，哪怕是一只，都不行**」「菌毯的来历，我帮你保守，世间的事其实就是这样，各取所需就好，谁还没有那么一两个秘密」。

## 📜 一张纸条 · 恩威并施

⚠ 兽王临走时借亲卫塞给苏晓一小截羊皮纸:「**今日勿离永环城，海族、施法，多名绝强伏杀**」——正是这张纸条让苏晓避开了四名绝强者的必死伏杀。⚠ 这既是对苏晓领主身份的认可、合作的诚意，也是**恩威并施的警告**:施法者与海族在兽族地盘上的一举一动都逃不过他的眼耳——同理，若苏晓违背承诺让虫族暗中爆兵，也同样逃不过王下禁卫。可这警告裹在救命情报里，就格外容易接受了。

## 🪶 一片钢羽 · 生死之约（50卷59章）

⚠ 就在苏晓把虫族的族群等级提升到 **Lv.95**（并选定「主宰意志」路线、上限 Lv.100）后没多久，一件来自永环城的「礼品」摆在了他面前——⚠ **一片钢羽，老兽王披风上的钢羽**。

- ⚠ 这是**兽族·钢羽家族的习惯**:在他还是战王时，**要和某个对手分生死时，会取下披风上的一片钢羽寄送给对手，当他格杀这名对手时，再取回**。收到钢羽，即代表**与兽王的合作正式终止**。
- ⚠ 他之所以选择此时死战，是因为他**感觉到暮冬城那虫族母巢已发展到超出掌控的地步**。⚠ 而他的判断并非妄想:如果没有这场最终之战，为了夺最后一大块「初始印记」，⚠ **苏晓的确会让虫族开始爆兵、直奔永环城而去**。他的原话是:「你会的，你会为了这印记，让你培养出的虫族女王，对兽族展露獠牙，⚠ **你已经把那虫族女王，培养成足以覆灭兽族的存在，那是专注到只为征战而存在的族群，兽族的丰裕生命力，是那些虫族滋长的温床**，滋长到，数量足够吞没兽族，撕碎海族。」
- ⚠ **他的账算得很清楚**:与虫族开战的代价太大（乃至比与海族开战大很多）；直接交出 35% 印记看似代价最小，⚠ 但那代表着**几百万盎司时空之力**——若他成为最终胜者，扣除公证费与各类折损后，**整个兽族大概能得到五分之一**（假设储存装置内有 200 万盎司，兽族就能得 **40 万盎司**）。⚠ 若格杀苏晓，虫族会立即被风海大陆排斥走，**这种天赋点全点在战争能力上的族群，一旦失去猎杀者的庇护，会在很短时间内消亡**——以最小代价解决兽族的危机。⚠ 反之若他输，苏晓会尽快离开风海大陆（这是他设的第二重保险）。
- ⚠ 而**最关键的一点，他不会直接说出来**:他征战一生、带领兽族建立起永环城，⚠ **时至今日却垂垂老矣到即将死于病榻之上**——⚠「与其继续苟延残喘数月，乃至数年，**老兽王更想拿起昔日的兵刃，在生命之火将熄前，让其最后爆燃一次**。」
- ⚠ **他不会有半点放水**:前往旧都之前，他把兽族存藏的【**极纯·深渊之血**】注入到了心脏中（注入前已让多个封印师对其心脏进行层层力量封印）。⚠ **这不是向深渊借取力量，而是要以这浓烈的深渊阻断因果**——⚠ 换句话说，**哪怕兽王格杀苏晓，苏晓所持有的三件原罪物，也不会根据因果找上兽族**。这就是他的态度:**双方各持筹码，押上一切死战一场，胜者获得所有**。

## ⚔ 旧王殿的最后一战（50卷60–61章）

⚠ 兽族**旧都**已是一片破败——想当初他登上王位时，兽族的首都还是此地，是他带领兽族一步步建立起如今辉煌的永环城；⚠ **可到了生命的最后，他选择了回到这里，那座他被加冕王冠的王殿内**。大量世界之力汇聚在上空，把夕阳渲染成金色的云雾**呈漩涡状缓慢转动**，而漩涡的正下方，正是旧王殿。⚠ 他从王座上站起身时，那**原本就破败老旧的王座因他的气息唤醒，开始浮现大片龟裂**；他下意识用手扶正头冠，**却摸了个空**，转而握上依在扶手旁的战刀。

### 三阶段与破局

⚠ 苏晓的判断:他巅峰时是**至强者初期**，如今战力**在绝强不到、但超出九阶的程度**，且**状态并不稳定，介于九阶顶尖到半个绝强者之间、波动幅度还不小**——⚠ 「如果老兽王稳定保持这种程度，那以他累积几千年的丰厚战技，**苏晓不是对手**」，正因不稳定，才有的打。

- ⚠ **能力体系简单粗暴**:自身体魄强大 + 判定高到离谱的力斩——⚠ 「这是在千军万马中杀出来的战王，能力当然不花哨，**任凭对手怎样花里胡哨，一战刀力斩下去，敌人不死也脱层皮**」。**一阶段攻高＋血厚，二阶段全程霸体＋攻高＋血厚**；⚠ 至于第三阶段的**黑暗兽王**，「简直是离谱，⚠ **已经不用考虑怎么对付第三阶段的老兽王，而是绝对不能让对方有机会进入第三阶段，否则的话，苏晓今天必死无疑**」。
- ⚠ **每刀都必须硬挡**:因为他的**每一刀斩出，都会撕破所斩过之处及附近范围的空间，并以撕破的空间刃斩伤试图规避斩击的敌人**——半个王殿被他的强力斩变成**死亡切割区**（一块碎石飞过苏晓身后半米处，就被悄无声息地切成片状、再切成碎粒）。⚠ 他没有空间系能力，**却可以通过斩击将一片区域内的空间震碎**，与苏晓的「雷抗驭雷」有异曲同工之妙。
- ⚠ **[阿姆](../阿姆.md) 的贡献**:阿姆两次举斧劈来、两次被一刀斩成「**一束带着冻气的光**」轰进墙里，第三刀被斩到濒死（靠巴哈激活药剂操控装置、捏碎心脏内预埋的生物胶囊才拉回来）。⚠ **但阿姆可没白挨揍**——通过它的伤损，苏晓摸清了一种没侦测到的能力:⚠ **老兽王的每刀都是一重判定，第一刀普通重斩、第二刀附带强悍震荡的强力斩、第三刀恢复普通重斩、第四刀再次强力斩**。（事实证明，阿姆如果有宗师能力的话，那**或许不是挨揍宗师，而是装死宗师**，装的和真的一样）
- ⚠ **破局点·招架槽**:苏晓发现他**硬抗威力更强的「刃道刀·流」，却闪避了威力较弱的「超·血烟炮」**——而这一闪身，让苏晓感受到的**蓄势压迫感骤然从 50% 滑落到 25%**。⚠ 由此推断:**强霸体这无解能力必有代价，那就是老兽王有类似「招架槽」的特性**——平常用来舒缓强霸体带来的过于刚硬，⚠ **但战斗时一旦积满，他就可能陷入短暂的破防状态**。⚠ 「他知道怎么赢了，**不能退，敌人硬，那就压迫力给满，直到敌人崩招架**」——刚被打入冷宫的**青鬼**因此被重新拽出来（斩碎青鬼时他的招架槽涨得比斩碎「流」时都多）。⚠ 这也是与他战斗的**最大陷阱**:「老兽王刚猛的刀术会让人下意识望而生畏、选择稳妥的防御状态，⚠ **可一旦这样选择，大概率会被兽王压制到死为止**。」
- ⚠ **二阶段·生命焰**:他解开了封印多年的力量——「**能赢下我，这力量就归你所有，我封印这力量很多年，没想到，如今我会衰弱到，借助它和敌手搏杀**」。⚠ 那金红色火焰**会点燃生命力**:生命力越强点燃得越慢，⚠ **可一旦被点燃，越强大的生命力也会燃烧得越旺盛**。他的心脏跳动到炽红、隔着胸膛都能看见，鲜血都要沸腾起来——⚠ **他已经太多年没有这感觉，这让他眼中的浑浊荡然无存**。
- ⚠ **反被捶通**:「苏晓在与老兽王的战斗中变强了？不，⚠ **应该说，他是被强霸体状态的老兽王给捶通了**」——进入本世界前苏晓短时间内掌握的强大能力太多，「**只用出了八分风采，剩余的两分威能则沉淀起来，沉淀带来停滞，停滞导致淤堵**」。⚠ 面对狼神没被捶通，是因为实力差距太大；**对上老兽王，则是遇到了旗鼓相当的对手**（他技法水平高但体魄已经不行，苏晓体魄年轻有活力耐力十足但技法经验逊色一筹）。⚠ 老兽王的生命焰在灼烧苏晓的同时，**也让他对「时·支配者」能力的『消化』速度加快**——苏晓由此**首次触发了「时·支配者」**。⚠ 他也因此**一生征战中首次在「熔铁之躯」的情况下被敌手压退**（苏晓的「完美反制」在他战刀上崩出一块缺口），脸上是**一闪而逝的诧异**。
- ⚠ **终局·拔枪失败**:生命值跌破 30% 后，天地都黑暗了，一处**深渊漩涡**在他身旁出现，中心处是一把**钩镰枪**的末尾——⚠ **他进入第三阶段的未知条件就是拔枪**。⚠ 他的刀术宗师只有 Lv.78，**可渊之战枪却高达 Lv.85**（超过 Lv.80 的技法能力就是另一种阶位了）；这把昔日的主武器**因他过于衰老已无法继续压制，只能封于深渊漩涡中**，每次使用都要重新拔枪。⚠ [巴哈](../剧毒魔鹰·巴哈.md) 硬扑上去、每秒承受惊人的黑暗伤害、全身飙血，硬是让他的拔枪速度**迟缓了些**（单手拔不动、改双手拔）；⚠ 苏晓趁这空挡捏碎【黑暗蠕动（深渊·仪式物）】炸开高浓度深渊能量、缓住黑暗洪流，双血魂加持的**超·血烟炮**轰在他身上——⚠ **深渊漩涡开始暴走，被拔出过半的深渊战枪全部被拖了回去，最终深渊漩涡消散**。他哇的一声吐出一大口鲜血，⚠ **却并未熄灭斗志，一刀将苏晓斩退**。
- ⚠ **最后一击**:苏晓用「**死寂烬灭**」连开五枪（⚠ 前三枪都被他斩碎，第四枪命中肩头，第五枪被他用刀柄挡住）——⚠ **来自死寂城的「死寂烬灭」堪称生命力的克星**，他熊熊燃烧的生命力**突然出现萎靡的势头**。随后是液态阿波罗裹腿的**太阳直踹**洞穿他的躯干（把他背后的半截披风轰飞、钟塔被轰碎、旧王都的大钟落地发出巨响）。⚠ 最终在时间「缓滞现象」中，苏晓一刀斜斩过他的整个胸膛——⚠ **几乎同时，老兽王的战刀也力斩而下，把苏晓的整条左臂、整个左肩、乃至小半边躯干都斩了下来**。苏晓靠晶体手臂撑地再度站起，⚠ **他刚起身，老兽王轰然倒下**。
- ⚠ **遗言**:「**白夜，假如有天兽族做了蠢事，饶他们一次。**」「嗯。」——⚠ 得到回应后，他眼中失去光泽。⚠ 苏晓与兽族的合作**堪称从一开始就尔虞我诈，但与老兽王的这场死战却很纯粹**；况且**就凭老兽王的那把钩镰枪，苏晓也不会拒绝这承诺**——⚠ **那把钩镰枪，他见过一次，那是在对战狼神时**（当时从天而降贯穿狼神胸腹钉入地底的正是它）。
- ⚠ **身后事**:苏晓让阿姆在附近挖了个坟墓。当天空放晴、一缕雨后的阳光照在旧王都的中心处，⚠ **在旧王座原本的位置，是一座新埋的坟墓**——⚠ **一把长刀插在坟前，长刀的末柄上还绑着半截有铁羽的披风，在这长刀的刃口上有一道最为显眼的崩口**，一股微风吹过，绑在刀柄上的钢羽披风**叮叮作响**。

- ⚠ **他的继承者 [骑士长](骑士长.md) 也是赢家**:苏晓离开本世界后没可能带走「水晶森林」，⚠ **届时兽族就有了晶脂产出**。「苏晓没睚眦必报到，战胜老兽王后再与兽族开战，⚠ **他与老兽王已各达目的，双方互相留个体面，是很不错的结局**。」
