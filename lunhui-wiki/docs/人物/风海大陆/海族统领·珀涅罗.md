# 海族统领·珀涅罗

镇守海族登陆要地**白蹄港**多年的最高指挥官——皮肤青白、头顶光秃的海族统领，是**海神的绝对心腹**。白蹄港本是主战场范围内兽海互夺之地，之所以被海族占据多年，**七成原因是珀涅罗的个人能力**（连海王都不因失守责罚他）。

## 🏰 被奇袭破防的能臣

⚠ [苏晓](../苏晓.md)（伪装领主·怒泽·戈温）以**渊龙长子 + 数百名人均大头目级的领主队**从天而降、直捣白蹄港中心建筑（指挥中心 + 物资库 + 全城术式中枢所在），打了珀涅罗一个措手不及:
- 他先靠黑兆的空间器物撤到「先祖灯塔」、又立刻传送回白蹄港外区稳住十个海族军团，可**始终不敢下令轰毁中心建筑**——因术式中枢就在其最下层，一旦被毁，全城城墙与海下术式尽废、白蹄港形同平坦荒野。
- ⚠ 他赌「蛮兽不识术式」，结果苏晓不仅认得、还**直接把术式中枢关停再砸毁**，令他血压飙升。他向海王如实禀报「敌人奇袭到了宝库里」时羞愧到额头贴地，却**替苏晓隐瞒了渊龙长子与精锐队的存在**（保全颜面）。
- 谈判是双方各怀鬼胎的拖延:珀涅罗拖时间等最精锐的暗杀小队 + 空间塔封锁到场，苏晓则拖满 40 分钟公证、拿稳初始印记碎片——最后苏晓一击砸爆封锁空间塔、扬长而去。

## 关系

- 海王：他效忠的君主——失守白蹄港时海王未责罚他（多年镇守之功）。
- [苏晓](../苏晓.md)（灭法·伪装戈温）：以渊龙长子空降、破他术式中枢、抢走初始印记大块碎片的死敌。
- 侏儒副官 / 黑兆·阿波努斯 / 女妖·狄塔：他麾下代表海神教 / 海王亲卫 / 长老会的三名副手。
