# 城主·芬里斯

暮冬城封地九名城主之一——狼族，原本与领主管家串通一气、站在架空小领主那一边。管家被除后，他与另一名兽人城主主动找上门归顺 [苏晓](../苏晓.md)。苏晓不但没清算，反而重用他:因为他**敢站出来、敢做事**（比那七名藏身暗处、静观其变的城主强），且他管的大城**粮食库存比核心大城暮冬城还多**（各凭本事搞来）。

## 🐺 狼狈为奸的情报头子

⚠ 苏晓在暮冬城的三名心腹各代表一种力量:[斑狐族·皮鲁](皮鲁.md)（财神爷·前永环城财政官）、[厄格因](厄格因.md)（军团·五成以上将领附庸他）、以及芬里斯（**地头蛇·封地八成情报渠道在他手**）。苏晓刻意让厄格因与芬里斯**狼狈为奸**（「厄格因是天生的狼、芬里斯就是同流合污的狈」）、组成封地上最难缠的同盟，好摸清各种秘密、震慑其余城主——只要不让皮鲁 / 芬里斯 / 厄格因三人合流，局面就压得住。

## 🕵 办的事

- ⚠ 是他点破「区区一个管家竟敢作对」背后有**部落阵营暗中扶持**，引出老领主留下的家传秘宝（内镶两块原初碎片）的传闻，让苏晓立刻上心。
- 与厄格因联手查清了闯入封地的是部落最强巫毒术士 [巫毒术士·巴泽](巫毒术士·巴泽.md)（信奉灵魂死神）。
- ⚠ 末了替苏晓给封地上的**黑暗神教**送「见面礼」（三个木盒·实则一盒装着巴泽长孙灵魂药剂的主材「灵性萃取物」）——苏晓借此栽赃黑暗神教「劫了药材」，把巴泽的杀意引向黑暗神教这颗毒瘤。

## 关系

- [苏晓](../苏晓.md)（领主·灭法）：他归顺、重用他当情报头子的领主大人。
- [厄格因](厄格因.md)：被苏晓刻意撮合、与他狼狈为奸的军团首领（狼与狈）。
- 黑暗神教：他奉命去送「见面礼」交涉、警告其收敛的封地毒瘤。
