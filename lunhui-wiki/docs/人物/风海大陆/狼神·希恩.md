# 狼神·希恩（殁）

风海大陆**兽族与海族共尊的狼神**——本是一头**银·月狼**（深渊监守者）。早在本世界遭深渊侵袭时，尚未被加冕「狼神」之称的希恩就来到此地，在兽族与海族最危难之际**独自抗住了那一波深渊侵袭**，也因此再也遏制不住体内积攒已久的深渊与黑暗，终被深渊侵蚀。⚠ 最后关头，迫于无奈的兽族与海族**在它失去意识前、经它首肯**将其封印——正因有矮人王为它打造的**月光大剑**一同陪葬，它才可能被封住。此后两族尊其为狼神。多年后封印意外破损，靠至强者**蜘蛛夫人**重新封印。

## 🌑 被深渊彻底吞没的悲剧

⚠ 它不仅有月狼被深渊彻底侵袭后的强大战力，更**扯碎并吞入了一只不灭特性·深渊滋生物、与之融合**，成了不死不灭的存在。它的战力保守估计已达**绝强者上游~顶尖梯队**——神秘之眼只侦测到 15% 资料就已让在场绝强者「对绝强者的实力上限有了新认知」:

- **狼血·战歌（Lv.89）**：生命 +156000、**无视所有控制效果**、意志 +150。
- **月光（Lv.89）**：生命 +72000、身防 +280、物理减伤 24%、能量减伤 27%。
- **元素·眷恋（Lv.EX）**：元素亲和力 **+920**。**元素吞噬者（深渊·Lv.EX）**：可吞噬四种自然元素。
- **剑术宗师（Lv.89）**：⚠ 苏晓发展这么久，刀术宗师才 Lv.74。
- **狼剑·辉光之月（奥义Lv.EX）**：附带敌人最大生命 8.5% + 6700 月光伤害，并令其每秒降 10 点身防、持续 5 秒（削减可持续 1~3 小时）。
- **狼剑·月影王（奥义Lv.EX）**：剑类武器伤害阶位 **+8**、所造成伤害 **30% 为真实伤害**。
- **狼剑·不灭意志（奥义Lv.EX）**：生命降至 45% 时，按四项真实属性总和 ×0.45「巨量」提升身防（与苏晓的「不灭体质」同源）。
- **野性隐匿（Lv.86）**：气息平缓时隐藏战力强度——正因如此，先来侦测的黑兆才误判它「只是绝强上游」。

## ⚔ 封印大殿的死战（50卷41–44章）

⚠ 它被黑暗神教（实为奥术永恒星栽赃）放出后，封印内的**深渊之孔**持续外涌深渊能量、短则几天长则半月就会化为深渊通道，届时深渊能量几天内就能笼罩大半个风海大陆——兽族与海族因此**暂时停战**，组建五人强者队深入浮光岛。

- **第一阶段**是**附在最外层的不灭滋生物**在用它的身体战斗:被瑟菲莉娅的黎元素光柱、链钉、法球压制，骑士长锥枪贯腹、黑兆十几把短刀钉脊、苏晓超·血烟炮命中头颅，最后瑟菲莉娅的奥义『**辉光**』把它生命打落 70%、苏晓的狂噬魔刃没入其胸膛——⚠ 斩掉的只是那只深渊滋生物，**狼神本尊这才登场**（身躯凝练、伤势尽复、生出鹿角般的木质化犄角、双瞳化为青黑竖瞳，从黑暗中抽出 2.5 米长的**深渊大剑**）。
- ⚠ **第二阶段**它把强者队按在地上教育:骑士长（绝强上游~顶尖）被它一脚踹跪、大剑对穿肩腰、剑柄砸凹头甲、一脚镶进墙里——⚠「骑士长不是弱，而是**被捶到什么能力都用不出来**，这就是为何到了后期技法型遭人恨」。黑兆被单手捏爆头颅（靠绝强的「一年三条命」复生）、神父被一剑贯穿轰成均匀分布在墙上的血肉、瑟菲莉娅被拖进重度深渊侵蚀区。它甚至会「**与断掉的暗月大剑互换位置**」躲开瑟菲莉娅的『辉光审判』，气得她差点吐血:「**可恶的灭法，怎么什么能力都教给盟友！**」
- ⚠ 转机是苏晓**引下界雷**（封印大殿被劈碎、界雷柱比浮光岛直径还粗、几十公里外的巴哈只看得见雷柱、连大统帅·凯恩都被震撼、混在船员里想抢人头的契约者当场散伙回去睡觉），把岛上黑暗生物清空（苏晓收到海量击杀提示），也对黑暗特性强烈的狼神造成高伤——代价是苏晓被骑士长的「雷陨」连着狼神一起对穿、当场昏厥。
- ⚠ 终局:一把**钩镰枪**从天而降（「已彻底超出绝强者所能达到的范畴，但不知为何又失去了至强者那种难以抵挡之势」）贯穿它的胸腹钉入地底；苏晓抓住这唯一的机会，血之兽炸开 + **液态烈阳之怒·阿波罗**裹住右腿的**太阳直踹**（太阳焰呈平放沙漏型贯穿，狼神在最细处）把它踹到单膝跪地，随即一刀魔刃斩杀成功。⚠ 弥留之际它**短暂恢复清醒**，看到苏晓时疑惑了一瞬，转而释然、接受了这一结局，化为黑色灰烬消散——**至死仍紧握着手中的深渊大剑，直到手臂化为灰烬散落**。

## 🗡 两把狼剑与两只月狼

⚠ 浮光岛与月狼因缘不浅:**岛上封印着一只（希恩）、埋葬着一只**。故本世界有两把狼剑——一把是希恩早年的狼剑（插在海族王座靠背上，苏晓借来试过，「可惜，它死了」，无法凝聚无上月光）；另一把属于灭法与施法者混战时重伤来此、终死于此地的那只银·月狼，插在**狼冢**墓碑前。而矮人王为希恩打造的月光大剑，则与它一同被封印，最终被深渊侵染成**暗月大剑**、在死战中被瑟菲莉娅炸到破败不堪。

## 关系

- [苏晓](../苏晓.md)（灭法）：斩杀它的灭法之影——⚠ 万界中**只有月狼和灭法能消灭不灭特性的深渊滋生物**，月狼绝迹后就只剩灭法（另一位是[魔镰·泰莉德](魔镰·泰莉德.md)，但她拒绝出手:那不是种能力，而是种代价）。
- [兽王](兽王.md) / 海王：曾在最危难时被它所救、又不得不将它封印、如今只求让它安息的两族君主。
- 骑士长 / [瑟菲莉娅](../瑟菲莉娅.md) / 黑兆 / [神父](../神父.md)：与苏晓组成强者队、把它按到斩杀线的另四人。
