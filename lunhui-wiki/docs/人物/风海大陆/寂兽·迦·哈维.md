# 寂兽·迦·哈维

[苏晓](../苏晓.md) 在黑暗戈壁**地下城**以 1500 金币一名的高价赎聘来的三名大头目之一——实为战力已达**首领级**的将领。他有二分之一熊族、四分之一兽族、四分之一人族血脉，脸庞尤为刚毅。招募这么多牛鬼蛇神后，哈维是苏晓终于发现的**靠谱部下**。

## 🐻 忠诚95 的老实人 · 却当不了二当家

⚠ 哈维一进领主队**忠诚度就高达 95 点**（感激苏晓帮他摆脱地下城的困境、准备安心效力），与厄格因的 −20 形成鲜明对比。他的短板是**谋略只有 B 级**（不擅阴谋诡计，这正是他被俘、沦落地下城的主因）。

- **统领潜质（被动·S级）**：同厄格因，可将统领的能力 95% 加成给麾下士兵。
- **能力数据**：统领力 S / 谋略 **B** / 战力 S+ / 远征力 A / 忠诚 **S**——除谋略与远征外全面优秀。
- ⚠ 但他**当不了领主队二号人物**:作为外来赎聘的将领，若苏晓有了封地、需率军远征，厄格因的远征之才（S+++）远超哈维。苏晓刻意的稳定格局是**厄格因与哈维各执己见、互相制衡**。

## 🔨 太阳重锤砸渊龙

⚠ 哈维手持长柄战锤，能汇聚风暴与太阳能量化为**百米级太阳能量巨锤**，是七战渊龙长子的主力输出（一记记「烈阳重锤」把渊龙砸得连吐黑血）。破白蹄港时，他带树茂族 / 战熊族 / 泰坦族等猛男体型族群向地下物资库推进。

## 关系

- [苏晓](../苏晓.md)（领主·灭法）：帮他摆脱地下城困境、令他死心塌地效力（忠诚95）的领主大人。
- [厄格因](厄格因.md)：与他统领力并列 S、却反骨 E 级的另一位大头目——被苏晓刻意安排互相制衡（「今后这是他们内心想法最一致的一次」= 都想退出渊龙的沉眠之地）。
