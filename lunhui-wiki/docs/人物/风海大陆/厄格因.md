# 厄格因

[苏晓](../苏晓.md)（伪装兽族领主·怒泽·戈温）在铁堡城贫民窟招募到的一员猛将——身高两米五、狮鬃般的狂乱头发披散在背、上身疤痕累累，是**纯血兽人**（黑暗时代就存在、继承了上个纪元霸主·纯血人族些许血脉的种族），容貌更近人族，一双暗金色竖瞳。他的统领之才在铁堡城数一数二，是苏晓领主队实际的**二号人物**。

## 🐍 脑后生反骨的「二五仔」

⚠ 厄格因沦落贫民窟的原因，是他的**统领才能强到差点带着麾下兽族战士把军团首领砍了**——只因他自己想当首领。他被恶齿、蝰蛇引荐时**一听「领主」二字、契约内容都没细看就刷刷签名加入**，动机是「没封地的领主·领主徽章归属模糊·能者居之」，盘算着多拉自己人、不断试探苏晓的手段，一旦发现苏晓外强中干就显露反心。⚠ 他的**忠诚度锁死在 −20 点**（历任领主手下都如此），全靠 [太阳领主称号](../../力量体系/装备道具.md) 的「精英单位忠诚度 +70」压着，才不至于当场捅刀。

- **统领潜质（被动·S级）**：可继承所属统领的光环 / 战争类能力，将其 **95%** 加成于麾下士兵——意味着日后若有兽族军团，厄格因能替不在场的苏晓给整个军团带去 [战争领主称号](../../力量体系/装备道具.md) 95% 的加成。
- **能力数据**：统领力 S / 谋略 S+ / 战力 S+ / 远征力 **S+++** / 忠诚 **E**——率军远征之才远超哈维、蝰蛇、恶齿，唯忠诚垫底。

## 🐉 七战渊龙长子 · 破白蹄港

⚠ 厄格因是领主队的实战核心:七次围攻**渊龙长子**都由他断后 / 带头（第七次干脆不退、原地驻扎逼降），最终在巨铠掰开龙颚时降服渊龙；奇袭**白蹄港**时，他带蛇人 / 血兽 / 狂兽等攻杀型成员直捣 21 层指挥中心。他对**维护苏晓领主威严**极上心（谁敢质疑无封地领主的实权，他会主动弄死那人）——不是忠诚，而是深知「苏晓越有权势、他这二当家的位子越稳」。苏晓刻意维持的稳定局面是:**厄格因与哈维各执己见、蝰蛇居中当墙头草**。

## 📈 反骨疯长 · 凛冬军团的主战场传说（50卷25–28章）

⚠ 被苏晓任为军团首领、放到远离暮冬城的主战场后，「一直拴着厄格因的锁链已经解开了」，这家伙的**反骨涨势惊人**:

- **首战**就以两个军团硬撼海族**五个军团**（凭战争领主 95% 加成 + 菌毯增益打得有来有回并占上风），不仅把五个军团打退，还**收拢两个半溃散的兽族军团、二合一纳入麾下**。海族长老会当晚就派刺客来杀他——他喝完酒睡进几百名大头目级兽族的集体营帐，刺客不敢进，「来都来了」就顺手把 [恶齿](恶齿.md) 做了（厄格因看到尸体「『伤心』到强忍着才没笑出声」）。
- ⚠ 麾下军团**越打越多**:6→7→8→9→10 个，且「不知道是有意还是巧合，他们每次都是趁苏晓睡觉后偷偷发展」。他在主战场**私招其他领主麾下的军团**，招得十几位领主的执事官堵到暮冬城领主庄园门厅要说法（别人的将领越打越少求补充，自家这几个越打越多）。**凛冬军团**由此成了兽族阵营几大主力之一——不同于战熊军团的稳重、钢羽军团的稳扎稳打、铁纹军团的不动如山，凛冬军团**完全是肉食动物、激进又凶悍**，打着打着就被人马族重骑兵拉扯成尖锥阵形往前捅。
- ⚠ 一众领主误以为这变化全是**菌毯**之功（实为战争领主称号所致），对菌毯的渴求因此更加迫切——苏晓乐得不解释。

## 🪵 800 棍杖 · 芬里斯的「自动翻译」

⚠ 苏晓把他从主战场召回、当着十六名执事官的面发落。他跪地嘟哝「我没做过，那些都是溃军」，苏晓怒斥「还敢狡辩，你才出去两天就给我惹了这种大祸，要是继续放任你，我怎么和在场的十几位交代，来人，拖下去斩首，立即斩了」——⚠ 满座皆惊，唯有 [城主·芬里斯](城主·芬里斯.md) 听得明明白白，自动翻译成:「（做得好，但别多说话）（才出去两天就这么有出息，做的好）（不用在乎这几个，老子给你顶着）（芬里斯，快给厄格因求情）（马上求情）」。芬里斯酝酿两秒情绪，噗通跪下声泪俱下地演——十几名执事官只是来讨说法，哪敢真让苏晓处死主战场主力，台阶既已搭好只能顺坡下。⚠ 最终判罚 300 棍杖，可 [蝰蛇](蝰蛇.md) 因「兄弟情谊深厚」不顾他人反对**极力替他求情**（期间还差点笑出声），生生让他从 300 棍挨到了 **800 多棍**。

## 😰 反骨缩回去的两个瞬间

⚠ 第一次:苏晓宴请八城主、当晚就收到管家首级，厄格因在走廊望着夜景，感到这位「不善谋略」的领主大人并非自称的那样，「已经生出一截的反骨，突然缩了回去」。⚠ 第二次:苏晓边掰灵魂结晶加热咀嚼、边说「你在那边也代表我，如果你替我丢脸，我就把你的灵魂做成明早的餐点」，那咔吧咔吧的咀嚼声让他额头彻底贴在地毯上。⚠ 出发打黑暗神教前，苏晓一句「听说你最近和白龙领主来往密切？」+「我不是，一直很相信你吗」的和善笑意，让他冷汗直冒:「大人，您的恶意，涌出来了。」——**但只要没有反骨，这就不是厄格因了**。苏晓由衷认为他今后有当领主的资格:能屈能伸、狠辣果断、最重要的是有野心，且目标明确、一步步向上爬。

## 🪖 一张照片压住的反骨（50卷59章）

⚠ 水晶森林一战，兽族以**死伤过半**的代价夺下这片资源区、让海族在至少几年内没有觊觎此地的资格。⚠ 战后他正就着火烤的海兽肉享用午餐，[蝰蛇](蝰蛇.md) 愁容满面地提起「二哥（[寂兽·迦·哈维](寂兽·迦·哈维.md)）现在生死不知」——⚠ 他吮吸着食指上带血水的焦香油脂，意犹未尽地回:「**如果哈维死了，不是更好吗。**」（一听蝰蛇说出「棍杖」二字，他原本美好的心情**突然晴转多云，还伴随着隐隐的腰腿疼**）

- ⚠ **他的反骨长到了顶点**:「蝰蛇，我们真的赢了吗？真的把海族赶出水晶森林？」——⚠ 「依我看，海族虽然被打退了，但**水晶森林的情况依然岌岌可危，我们得驻守在这，长时间驻守在这**」。蝰蛇犹豫着问「如果领主大人召我们回去怎么办？」——⚠ 「蝰蛇，你要知道，**领主大人的决策，有时候也不一定完全正确**，我相信，只要我们的坚持是对的，大人不会责罚我们，你说……对吗。」⚠ 「但凡蝰蛇现在敢说出半个不字，**厄格因都会让他出不了水晶森林**」。
- ⚠ **苦心经营到现在**:他终于**不用再被那犹如他梦魇般的男人所支配**——「对方对他有知遇之恩，却又是他最恐惧的存在；⚠ 眼下，这让他最恐惧的存在，**只能用他故意留下的破绽**，这真是，让他脸上难以控制地浮现笑容」。⚠ 他打开那个必须送到他手中的信封，本以为是自己**故意暴露的把柄**——⚠ **倒出的却不是信件，而是一张照片**。他把照片向外抽，⚠ **可刚抽出一半，就把照片塞了回去**。
- ⚠ 蝰蛇试探性询问「你这是？」——⚠ 「**我们攻下水晶森林这好消息，我应该亲自回去禀告大人。**」「嗯？你刚才还说……」「**我刚才什么都没说，不行，我作为大人一手提拔上来，他最忠诚的部下，我现在就得回去见他。**」——⚠ **蝰蛇人都蒙圈了**，还不等他开口，厄格因已大步离开，去往最近的传送阵回暮冬城。（⚠ **那张照片，是霜雪城的俯视照**——含义很明显:**霜雪城的城主之位**。）

## 😐 到手的城主之位飞了 · 却拿到领主戒

⚠ 他为什么急吼吼地赶回？因为**他对权力无比痴迷**:「看似厄格因现在率领凛冬军团、非常有实权，其实不然——⚠ **再怎么说，厄格因都只是名统帅而已**，外加他麾下的军团每天都要消耗数量恐怖的补给，⚠ **这也代表，厄格因就算反骨生的再长，也不敢叛出兽族**。」⚠ **领主是第一梯队的掌权者、是兽王直接管辖；城主是第二梯队；至于统领，则到了第三梯队**——⚠ 倘若他在掌控几个精锐军团的情况下成为凛冬封地的城主，**那他的权势会达到直追领主的程度**。

- ⚠ 书房内，他满脸严肃、丝毫不见之前反骨疯长的模样:「大人，**我其实什么都不想要，能追随大人，我已经心满意足**。」⚠ 苏晓开口:「我和霜雪城的老城主谈过了，他年老体衰，难以继续胜任城主之位，所以我决定……」——⚠ **听到此处，厄格因脸上已控制不住地绽放笑容**（他先打下白蹄港、之后又攻袭浮光岛，**不就是为了这一刻吗**）——⚠ 「**所以我决定，让蝰蛇去担任霜雪城的城主。**」⚠ **脸上笑容僵住的厄格因，忽感一阵落寞，转而笑了笑，低垂着眼帘没再说什么，目光暗淡。**
- ⚠ **转折**:苏晓提起魔角封地的老领主因**多年来暗中勾结海族长老会、被老兽王下令处死**，那边情况不太稳定——⚠ 「我作为大统帅，理应亲自前往，**但我最近很忙**」；⚠ 「厄格因，我听说，**水晶森林那边的情况不太稳定，你也挺忙？**」——⚠ **这是他之前亲口说出的话，他背后的冷汗都下来了**:「**绝对没这种事**，属下估测，最近几年，海族都不敢窥望水晶森林。」「哦，是吗。」
- ⚠ **苏晓目光和善地看着他**——⚠ **就是这种目光，这让厄格因在多少个噩梦中惊醒、被此支配的目光**；⚠ 他有种感觉:**哪怕他今后同样登上大统帅之位，再被这目光盯上，他同样会心中打哆嗦**——⚠ **已经有挥之不去的阴影了**。⚠ 「那就先由你做这领主吧。」——⚠ **领主戒被抛在桌上，他犹豫了两秒、确认这不是试探后才拿起**，⚠ **这让他心中的落寞感荡然无存，而是野心熊熊燃烧着**。
