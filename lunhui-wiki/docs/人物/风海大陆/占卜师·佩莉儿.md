# 占卜师·佩莉儿

[大统帅·凯恩](大统帅·凯恩.md) 的**养女**（已故过命交情老友留下的独女，凯恩这么多年视若己出、当亲孙女般）——身穿黑色纱裙、全身衣裙犹如黑雾，那绝美的容颜让人难以移开视线。⚠ 她是**本世界最强梯队的占卜师**，且魂毒造诣与部落赫鲁伊 [巫毒术士·巴泽](巫毒术士·巴泽.md) 的巫毒**完全是一个梯阶**——正因如此，凯恩始终不同意苏晓见她:**既担心苏晓算计她，也担心她把苏晓给毒倒**。

## 🔮 占不透的灭法 · 敬而远之

⚠ 她与苏晓的初见是在凯恩宅邸的甬路上擦肩而过——⚠ 苏晓难以移开视线是因为「这是名占卜师，**很强的占卜师**」；而她作为占卜师，**下意识感觉这与她擦肩而过之人危险**，情不自禁加快了脚步，当天就联络凯恩让他提高对苏晓的防范。⚠ 她曾试着占卜这名灭法的因果与命运——**结果是占卜失败，她什么都没看到**。⚠ 她不是懵懂的小姑娘、不会因好奇就接近一个人:「**一切她无法看透命运的人，她都会远离，这样才安全**」——怎奈事与愿违。

## 💸 选了财运当占卜代价的「明灯」

⚠ 这位堂堂顶尖占卜师，**住所却是租的**:

- 她占卜一次收 5000 索铎，可每月也就能占卜几次，且**大部分有占卜需要的人根本没必要找太顶级的占卜师**（找亲人下落、寻遗失物，上百金币一次的水平就够用）。同行还越来越多。
- ⚠ 占卜师若占卜财运敛财，**因果上早晚会还回去**（那些靠占卜赌博、投资敛财的占卜师，没一个有好下场）。
- ⚠ **最要命的是**:作为大统帅家的掌上明珠，她当初选择占卜代价时**选的是财运**——这让「我们的佩莉儿大小姐历来都财运不佳」。父亲曾送她一栋酒庄，可作为**投资界「明灯」**的她，很快把那葡萄庄园「赠予」了银行。她已 30 岁，找父亲要钱实在难以开口;她厌烦兽族的权力争斗到极点，只求清静。

## 🎴 一千万索铎 + 一副占卜牌（50卷51章）

⚠ 苏晓要拉她入伙伏杀绝强·施法者，她三次拒绝:「不要再派你的手下来找我，我不会帮你对付绝强者」「白夜领主，这不是钱财的问题」——

- 苏晓开价**十万索铎**，她仍拒；⚠ 苏晓直接抛出「**一千万索铎**」——通讯器那边突然安静，几秒后她叹气:「白夜领主，**请不要拿我开玩笑**。」苏晓答「我很少开玩笑」，随即挂断，让 [皮鲁](皮鲁.md) 去筹钱（对已攻占大半「水晶森林」的凛冬封地，一千万金币已可轻易拿出）。
- ⚠ 她收到暗金储蓄卡时，一度真以为里面存了一千万，随即**笑自己这想法荒谬**——直到老女仆来报「**永环银行的行长来找你**」，她才愣住:「行长？不就欠他们几百索铎吗，行长怎么会找上门，等等……」
- ⚠ **真正打动她的不是钱**，而是随后送来的**一副占卜牌**——由**矮人王的首席弟子 [秘纹师](秘纹师.md) 亲手打造**（可遇而不可求），木盒上写着「**赠予本世界最强占卜师·佩莉儿**」，下角是矮人王、秘纹师及几名大师级工匠的印记。
- ⚠ 她把储蓄卡给老女仆:「咳～，去把我欠几个银行的几千索铎还上，再把酒庄赎回来，**你自己留下几万给家人，剩下的，给那些需要这些钱财的人吧，太多的钱财，会带来许多不必要的因果**。」老女仆劝「有了这些钱，你今后就不必为钱财烦恼……」，她只淡淡「哦」了一声——**全程没再看那张储蓄卡一眼**，白皙的食指抵在占卜牌的木盒上，最终拿起联络器拨通。

## ☠ 三重毒的最后一环

⚠ 苏晓拉她入队**并非侧重占卜**（事关灭法者与施法者的因果，她的占卜结果仍是一片迷雾），而是要她的**魂毒**——⚠ 「并非是因为魂毒有多强，而是**魂毒刚好和剧毒与巫毒，组成一个无解的循环**」:[古亚院长](古亚院长.md) 喝抑制炼金剧毒的**灵魂能量**抑制剂 → 魂毒效力猛增；喝魂毒解毒剂 → 巫毒杀上来；喝抑制巫毒的**植物类**秘药 → 炼金剧毒又冲上来。这套循环虽不致命，却把这位绝强者的速度**削减到前所未有的程度**，为苏晓的极限一击创造了条件。

## 关系

- [大统帅·凯恩](大统帅·凯恩.md)：她的养父（已故过命老友的独女）——始终不许苏晓见她，警告苏晓「我小女儿最近要是出了什么事，我和你没完」。
- [苏晓](../苏晓.md)（灭法·白夜领主）：她占卜不透、下意识想远离、最终却被一千万索铎 + 一副占卜牌请进伏杀小队的领主。
- [秘纹师](秘纹师.md)：为她亲手打造那副占卜牌的矮人王首席弟子——那才是真正打动她的东西。
- [巫毒术士·巴泽](巫毒术士·巴泽.md)：与她魂毒/巫毒同一梯阶的部落赫鲁伊——二人的毒与苏晓的炼金剧毒构成无解循环。
