# 女巫·莉莉亚

圣光乐园的契约者——一身女巫装、气质慵懒丰满（衣着并不暴露却让人浮想联翩），手托修长烟斗、红唇间呼出幽紫色烟雾。她是**占卜系 + 因果系**双修，能与 [噩鬼·凯因](噩鬼·凯因.md)、[雷法神·艾格](雷法神·艾格.md) 临时组队，实力与手段可想而知——可惜这三人**恰好全被 [苏晓](../苏晓.md) 天克**，人称「刮痧天团」（凯因吐槽:因果系的莉莉亚**可能连痧都刮不了**）。

## 🌫 与空气斗智斗勇的两个月（50卷29章·名场面）

⚠ 初见时莉莉亚始终眼中含笑地盯着苏晓，末了终于问出憋了很久的话:「被称为『斩首的夜』的你，那次为什么会饶我一命？」——她说**两个月前主动挑战过苏晓、试图窥探他的因果，险些因果反噬而亡**，苏晓却放过了她。可苏晓对此**毫无印象**:

- 「我这个人的记忆力，很一般。」「但那就是两个月前的事。」「**这话虽说有些冒犯，但你会感觉到一粒尘埃落在你身上吗。**」
- ⚠ 她当场破防:「等一下，让我缓缓——**一直以来找你麻烦的因果系，你都没感应？**你不知道，你在我们因果系的圈子里很出名？**四阶到八阶中，每个阶位最有实力的那几名因果系，都被你教育过**，你真的丝毫没有感觉？」——苏晓的沉默即答案。原来她那次是**在和空气斗智斗勇**。凯因与艾格借口出去透气、门外传来快活的笑声，气得她决定让这两个家伙倒霉两天。

## 🔮 灭法的因果 · 结成合作

⚠ 莉莉亚的推断极有价值:**灭法之影是元素监守者，因果体量庞大到让因果系绝望**——「你这传闻中最后一位灭法之影，整个灭法阵营的因果，应该都在你这」（这也是灭法运势不佳的代价:有得有失）。她隐隐感到只要窥其一角，自己在因果系的成就将不可想象，但她不自大、也不想找苏晓的弱点，转而提出**更安全的合作方向:研究施法者的因果**——「施法者和灭法的因果运势有一点相同，就是很难窥探，但施法者那边给人的感觉是……分散」，她能窥探同阶位的施法者。苏晓与奥术永恒星是死敌、常能得到施法者的魂血，那对她就是无价之宝；她作为乐园阵营也不惧奥术永恒星报复。⚠ 她提的唯一条件是「**为了我的生命安全，我不占卜你的未来，至少你涉及到原罪物的未来，不能占卜**」——她还好心提醒苏晓「小心神父那老家伙，他身上带着不得了的东西（原罪物）」，结果听到苏晓淡淡回一句「**我有几件原罪物**」，目光更加迷茫。

## 🃏 三次占卜

- ⚠ **四名绝强者**（免费·作为「曾放过她一命」的回报）:占卜牌翻出女贤者（[瑟菲莉娅](../瑟菲莉娅.md)）、老态学者（古亚院长）、全身铠甲的巨型鱼人（大战士·扎卡瓦），第四张却是**黑蓝色漩涡**——重抽、掀开整副牌，牌面全是漩涡:「代表占卜失败了，原因是，你近期内的运势……比较一般。」（苏晓反倒安心:运势不佳是小事）
- **蛇神方位**（收费）:凭巴哈取来的蛇鳞当媒介，她占出「**蛇神已经不在风海大陆了，昨晚就离开了**」——她很想知道苏晓等人到底做了什么，能把一位恶阵营神灵吓得连夜逃到别的世界。

## 关系

- [苏晓](../苏晓.md)（灭法·斩首的夜）：她挑战过、险些反噬而亡、对方却「一粒尘埃都没感觉到」的目标——如今是共同研究施法者因果的合作者。
- [噩鬼·凯因](噩鬼·凯因.md) / [雷法神·艾格](雷法神·艾格.md)：与她组「刮痧天团」、在门外笑到她想让他们倒霉两天的队友。

## 🔮 地下世界里的神圣宝石（51卷33–34章）

⚠ **法魂队三人来到了地下世界**（⚠ **苏晓之前截了他们的猩红城堡任务，他们完成后触发了地下世界的相关任务**）——⚠ **苏晓约她见面，因为她是因果系的占卜师，或许知晓更多君主宝藏的秘密。**

- ⚠ **她以某种非空间的远程移动现身**（⚠ 层层波纹蔓延）:「**白夜，你怎么没去夺神圣宝石？这次来争夺君主宝藏的人，都去夺神圣宝石了，那宝石就在暴君的宫殿里。**」——⚠ **她诧异于苏晓竟没听过【神圣宝石】**:「**如果我能在君主宝藏中随便选一件宝物，我肯定会选这颗宝石，这可是 6000 点评分的世界级宝石。**」
- ⚠ **【神圣宝石】的真相**:⚠ **虽是颗宝石，但不是用来镶嵌，而是对自身使用**——⚠ **用后像被动能力般一直加持，直到持有者死亡才脱离、等待下一任**;⚠ **效果:幸运属性 +30 点，以及提升相当于自身魅力属性 2 倍的幸运属性，最高能带来 200 点的幸运属性加成。**
- ⚠ **⚠ 对苏晓是废物**:⚠ **世界级·满评分宝石，对 −21 魅力的苏晓只提升 3 点幸运，后续还 −42 点幸运属性**——⚠ **他兴趣一扫而空**（⚠ 「白夜，那可是神圣宝石，真的不去争夺？」——⚠ 「没兴趣。」）;⚠ **她若有所思地看着他去救队友的背影**——⚠ **⚠ 而暴君宫殿抢神圣宝石，很可能是神父与猩红阵营布设的陷阱。**
