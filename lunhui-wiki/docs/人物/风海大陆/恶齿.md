# 恶齿

[苏晓](../苏晓.md) 招募的领主队成员——酷似狼人的**魂狼族**，一看就是凶狠的亡命徒。他桀骜不驯、够狠、能力也杰出，但**不够老辣与沉稳**（苏晓一杯酒就把他看穿:一饮而尽的痛快劲儿）。

## 🦴 不要脸皮的狠角色

⚠ 恶齿够狠、狡诈，却**很怕厄格因**，也是个不要脸的活宝:身中蝰蛇的毒时，为求解药「大哥求你了、感谢你大恩大德」张口就来（「当场认父都是小事」）。开价也没数——只敢要月薪 5000~8000 灵魂钱币（被蝰蛇鄙夷「看你那点出息」），事后恨不得抽自己嘴巴。

- **统领潜质（被动·B级）**：可将统领能力 50% 加成给麾下士兵。
- **能力数据**：统领力 B / 谋略 A / 战力 A / 远征力 C / 忠诚 B——最没话语权的一个。
- 忠诚度 15 → 放血气后到 30 点（比蝰蛇的飙升平淡得多）。

## 🐟 生吞海兽垂体「补肾」

⚠ 恶齿最出圈的名场面:伏击运输队时，蝰蛇随手毒死一只海兽，恶齿竟捡起那海兽的**深海垂体生吞**、还说「这东西生吞补肾」——补没补肾犹未可知，**当场中蛇毒（神经性毒素）**，佝偻在合金箱上抽搐哼唧，最后靠苏晓丢来的解毒剂捡回一命，被阿姆拎到远处去灌药（「最近几小时不想看到这蠢材」）。

## ☠ 死得最冤的军团首领（50卷25章·殁）

⚠ 恶齿被苏晓任为军团首领、率一个军团出战主战场，却在**抵达前线仅半个晚上**、连一场仗都还没打就送了命。经过堪称离奇:[厄格因](厄格因.md) 首战以两个军团把海族五个军团打退、还收拢两个溃散兽族军团纳入麾下，海族长老会当晚就派**暗杀者**来清除这号悍将——可厄格因当晚喝完酒**直接睡在几百名大头目级兽族的集体营帐里**，暗杀者又不傻，哪敢往里钻。⚠ 于是那暗杀者秉承着「**来都来了**」的理念，转头把住在首领营帐里的恶齿给暗杀掉了。

- ⚠ 更讽刺的是，厄格因在号角声中冲到恶齿营帐、看到尸体后，「『伤心』到强忍着才没笑出声」——恶齿一死，他那个军团就不会被调回暮冬城。最终 [哈维](寂兽·迦·哈维.md) 牵制厄格因，[蝰蛇](蝰蛇.md) 趁机把这个军团收拢到自己麾下（蝰蛇既心惊胆战又必须这样做:他怕厄格因，但更怕厄格因一口把他吞下——主战场上，那条一直拴着厄格因的锁链已经解开了）。

## 关系

- [苏晓](../苏晓.md)（领主·灭法）：把他从铁堡城捞进队、又救他毒命、最后任他为军团首领的领主大人。
- [厄格因](厄格因.md)：他很怕的二当家——他死后「伤心到强忍着才没笑出声」的那位。
- [蝰蛇](蝰蛇.md)：一同杀光应聘者、又互相下毒（拿解药调戏他「求我就给你」）的损友——他一死，蝰蛇就把他的军团收进了自己麾下。
