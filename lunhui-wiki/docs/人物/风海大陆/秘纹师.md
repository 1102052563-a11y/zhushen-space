# 秘纹师

三大传奇铁匠之一**矮人王**最看重的**首席弟子**——一米七五左右的人族，黑眼圈、邋遢头发、衣领袖口满是腐蚀孔洞、左手三根手指是机械义体，典型的痴迷手艺、不修边幅之人。他被海族请去加固白蹄港宝库防御，结果被 [苏晓](../苏晓.md) 用**恶魔传送阵**连同物资一起劫了过来（一路被卷得以为自己死定了，力荐苏晓「换一种传送阵」）。

## 🔧 秘纹造诣超越恩师的体弱天才

⚠ 他幼年时家国因兽海战争覆灭、吸入了战争武器弥散的液质雾，导致多种脏器活力大减——注定他**无法胜任高强度锻造**（矮人王遍访名医也没能治好）。但他在**秘纹**一道上达到了连恩师都无法涉及的高度:矮人王那些惊世之作上的秘纹皆出自他手（外圈印徽是矮人王、内部圆标即他）。难得的是他从不膨胀、对恩师如父般尊敬，生命里除了秘纹再无其他。⚠ 苏晓发现他「只会铭刻秘纹、不会打造装备」后本要「和善」地把他送出庄园（好让埋伏的厄格因等人在庄园外「合法」绑走他敲海族赎金），他却死抱桌腿不走、用一个秘密换五天停留。

## 🌀 泄露虚空深渊危机

⚠ 秘纹师抖出的秘密极重:**虚空出现了一处一次比一次大的特殊深渊通道**（从核桃大小长到 3 米、间隔越来越短·很可能成为史上最大深渊通道），根源是奥术永恒星吞噬了太多自然元素。奥术永恒星靠多年前从灭法处窃得的方法反复封堵，只是饮鸩止渴，真正的解法是提升虚空自然元素总量——这才是**奥术永恒星肯放下身段来风海大陆、帮海族锤兽族**（以换取矮人王的元素器物）的真正原因。苏晓由此联想到瑟菲莉娅带格林·薇去的、被黑雾笼罩重兵把守的**绿茵大湿地**，判断那深渊通道七成在那。此后秘纹师对苏晓从敌意转为「亦敌亦友、可能联手」的试探，反复力劝苏晓去海族救出矮人王，报酬**最高开到三块原初碎片**（师徒俩「最近打造了一件作品、手头很紧」）——把苏晓馋得领主印徽差点戳穿桌子。

## 关系

- 矮人王：他如父般尊敬的恩师、被海族囚困的三大传奇铁匠——他求苏晓救出的对象。
- [苏晓](../苏晓.md)（灭法）：把他连物资一起劫来、又「和善」地想送他出门敲海族赎金、他反过来想拉拢救恩师的领主大人。
