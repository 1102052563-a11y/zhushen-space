# 魔镰·泰莉德

圣光乐园的契约者——身着黑色神职长袍、手持魔镰的绝美强者，是**从深渊中爬出的深渊系强者**（右臂、右手乃至脖颈右侧都缠着裂痕般的暗蓝色命纹，是无法抹除的侵蚀痕迹，也是强大灵魂的证明）。她人狠话不多、能动手从不废话，实力已达**九阶最极限**（迟迟不冲绝强者，多半是在积攒原初碎片或等契机，所需碎片数量与苏晓相近）。

## ⚔ 与苏晓的魔镰之战（50卷20章）

⚠ 泰莉德也盯上了 [小领主·古尔薇](小领主·古尔薇.md) 家传秘宝里的两块原初碎片（在寻宝队狐族身上留了深渊追踪印记），本想坐收渔翁之利，却因苏晓与巫毒术士的猛毒离谱到能渗进她的暗空间、险些毒发。她与苏晓正面一战:
- **魔镰·瞬时**（残影换位）+ **魔镰·命陨**（令百米内时间空间尽皆静止、一镰勾落苏晓头颅）——却被苏晓的「**位置互换**」（与刃之魔灵换位·连时间停滞都无法阻止的质朴能力）化解。
- **魔镰·汲血**（核心能力）:以魔镰为中心 300 米摄取敌人生命力、再回血自身。因苏晓有 72 万生命，她能每秒回血 1300~1400 点，一度变成速度 S / 攻击 S+ / 生存 S+ 的克星形态。
- ⚠ 但战局逆转:她的魔镰内蓄着大量「不灭特性·深渊滋生物」的本源能量（汲血时必然外放），苏晓把**刃之魔灵攀附在斩龙闪上与之对斩**，一场架下来**刃之魔灵强度从 62 涨到 72（+10）**、吸走了她攒了很久的约十分之一本源能量。她心痛之下主动退走，退场时那幽幽的一眼「仿佛要把苏晓永远记住」。
