# 魔镰·泰莉德

圣光乐园的契约者——身着黑色神职长袍、手持魔镰的绝美强者，是**从深渊中爬出的深渊系强者**（右臂、右手乃至脖颈右侧都缠着裂痕般的暗蓝色命纹，是无法抹除的侵蚀痕迹，也是强大灵魂的证明）。她人狠话不多、能动手从不废话，实力已达**九阶最极限**（迟迟不冲绝强者，多半是在积攒原初碎片或等契机，所需碎片数量与苏晓相近）。

## ⚔ 与苏晓的魔镰之战（50卷20章）

⚠ 泰莉德也盯上了 [小领主·古尔薇](小领主·古尔薇.md) 家传秘宝里的两块原初碎片（在寻宝队狐族身上留了深渊追踪印记），本想坐收渔翁之利，却因苏晓与巫毒术士的猛毒离谱到能渗进她的暗空间、险些毒发。她与苏晓正面一战:

- **魔镰·瞬时**（残影换位）+ **魔镰·命陨**（令百米内时间空间尽皆静止、一镰勾落苏晓头颅）——却被苏晓的「**位置互换**」（与刃之魔灵换位·连时间停滞都无法阻止的质朴能力）化解。
- **魔镰·汲血**（核心能力）:以魔镰为中心 300 米摄取敌人生命力、再回血自身。因苏晓有 72 万生命，她能每秒回血 1300~1400 点，一度变成速度 S / 攻击 S+ / 生存 S+ 的克星形态。
- ⚠ 但战局逆转:她的魔镰内蓄着大量「不灭特性·深渊滋生物」的本源能量（汲血时必然外放），苏晓把**刃之魔灵攀附在斩龙闪上与之对斩**，一场架下来**刃之魔灵强度从 62 涨到 72（+10）**、吸走了她攒了很久的约十分之一本源能量。她心痛之下主动退走，退场时那幽幽的一眼「仿佛要把苏晓永远记住」。

## 🍁 一杯枫茶 · 找错人的联合（50卷55章）

⚠ 她**不计前嫌**地登门找上苏晓——热茶的水汽缭绕飘起，二楼内厅中两人隔着茶几对坐，⚠ **桌上的热茶只被拿起一杯**。「茶快凉了，难不成怕我在茶里下毒？」「**嗯。**」——⚠ 她**毫不掩饰**地嗯了声，这让气氛有点僵。⚠ 「毕竟是**孤身行走在黑暗中的女强者**，虽说挽救的人多到数之不清，但**并不擅交际，也不擅长算计一类**——⚠ **魔镰·泰莉德是纯粹的强大与专注**。」

- ⚠ 苏晓一句「**这是枫茶，浪费**」刚要伸手，⚠ **她就端起枫茶饮了口，紧皱的眉头舒展了些**。「哦？现在又不怕我下毒了？」——⚠ 「**灭法，毒杀，深渊魔镰，很罕见。**」言罢又饮一口，⚠ **枫茶带来的独特舒缓感，让她的气息放松了几分**。
- ⚠ **她的提议**:「**我们一起，把狠人驱赶出永封冰原。**」「好提议。」「你同意了？」「**不同意。**」——⚠ 她皱起眉头，不理解为何苏晓不同意对付最难缠的 [狠人兄](../狠人兄.md)。⚠ **她的理由**:「因为他是猎杀者……」（话到一半想到眼前这位也是，⚠ **她改口**）「因为他是**虚空之树信誉度 −3800 点以上的猎杀者**，⚠ **我以前做过一些错事，虚空之树信誉度负到 400 点**，我很清楚，**负将近 4000 的人，到底会有多能玩弄规则**，有他在永封冰原，我们可能连参与那印记碎片争夺的机会都没有。」
- ⚠ **她挑中苏晓的理由**:有能力把狠人兄赶出永封冰原的人本就不多，最佳人选是 [雷法神·艾格](雷法神·艾格.md)（正经战斗天使、信誉度 3000 点以上）——⚠ **怎奈艾格不知去向**。之后的选择是苏晓、[神父](../神父.md)、黑魔、所罗门、亡灵妹、游侠四兄弟——⚠ **她艰难地选择一番后，找到了虚空之树看似最高的苏晓**（「我用了些特殊方式，查询了你的虚空之树信誉度，**至少你的信誉度不是负数**」）。⚠ **此刻她还不知道，坐在她对面的猎杀者，并不是没负虚空之树信誉度，而是负到已经查询不出来了**。⚠ 苏晓凝视着她，把她看得有些疑惑、皱眉不自在——⚠ 而一旁的 [布布汪](../布布汪.md)、[巴哈](../剧毒魔鹰·巴哈.md) **面容严肃，一个用翅膀搓脸，一个用狗爪抠牙——因为只有这样，才能忍住不笑**（阿姆正在擦拭拆开的古董钟，神色如常）。
- ⚠ **不过某种角度上，她其实找对人了**:「正所谓，**只有魔法才能打败魔法**，而**面对虚空之树很低的敌人，那就去找更低的，操作更多样，更让人目瞪口呆**。」

## 🖤 「别和你的那位前辈一样，堕入深渊」（50卷56章）

⚠ 无规则公证的公告突然出现，终止了这场还未开始的合作。⚠ 起身离开前，她说:「**灭法，别和你的那位前辈一样，堕入深渊。**」

- ⚠ 即将到厅门前，她的脚步一顿，**没转身、只是稍微侧头**问道:「**你，消灭了几只。**」——⚠ 苏晓自然知道她在问什么（**消灭过几只不灭特性·深渊滋生物**）:「**三只。**」
- ⚠ 「**如果遇到很强的滋生物，我可以临时做你的搭档，但仅限对付滋生物。**」——⚠ 她留下这话就抬步离开，在方才停留之处的半空中**留下一枚黑色印记，这印记自行凝结，构成一张黑色卡片**（被苏晓以灵影线一扯抓在手中）。
- ⚠ 苏晓由此有种感觉:**她即将晋升绝强了，可能这次离开风海大陆，对方就会晋升绝强**。⚠ 而在浮光岛的**狼冢**前，苏晓发现了**一束就在附近采的野花**，以及**一丝有几分深渊感的气息残留**——⚠ **是魔镰·泰莉德来过此地**。（她曾拒绝出手消灭不灭特性·深渊滋生物:那不是种能力，**而是种代价**。）
