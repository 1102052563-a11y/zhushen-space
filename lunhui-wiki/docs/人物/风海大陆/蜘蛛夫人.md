# 蜘蛛夫人

> ⚠ **万界大名鼎鼎的「剧毒美夫人」**——⚠ **[风海大陆](../../世界/任务世界/风海大陆-初始印记.md) 原住的**至强巅峰****;⚠ **⚠ 她被囚于灵魂书库前，是与猩红君主并列的存在**——⚠ **「她当初要不是被困在灵魂书库内，那之前到永光世界，苏晓要对付的终极 boss 就不只有猩红君主一位，还得加上蜘蛛夫人。」**

## 🕷 她为何「磨平了性子」

⚠ **外界都以为她被囚后端庄、惬意、优雅，对曾经恩怨早就不在乎**——⚠ **⚠ 事实恰恰相反:护短与睚眦必报，是她到至强顶尖梯队都还保持的行事风格**（⚠ 10 岁时因小镇地痞打了她弟弟，她在那群地痞住所下方下水道等了一星期，用蛛毒把十几名地痞全毒死）;⚠ **⚠ 被海族与兽族这样背刺，她居然选择谅解？只有一种可能:她在假装对风海大陆的善意，就等着脱困的那天。**

- ⚠ **⚠ 她与海族、兽族的真实关系**:⚠ **外界以为海族兽族都尊重她、风海大陆大事都她做主，其实她与两族关系早就不好**;⚠ **⚠ 起因是海族与兽族旷日持久的战争**——⚠ **她是在风海大陆晋升的至强，想迈出超脱极致一步只能留在此地等契机（原住世界的馈赠）**;⚠ **⚠ 可对身处原住世界、感知几乎与世界相连的她而言，两族的战争就像「两家在砸墙装修的邻居」**——⚠ **这装修持续几千年后，她彻底暴躁，多年积累的修行前功尽弃**;⚠ **⚠ 她一暴躁，海族与兽族都麻了**（⚠ **理论上她的灵魂剧毒可以把整个风海大陆都笼罩几年**）。
- ⚠ **⚠ 她被囚的真相**:⚠ **海族与兽族因血脉诅咒无法停止纷争，当代海王与兽王秘密商议后决定暗算她**——⚠ **把她囚困在灵魂书库内**;⚠ **⚠ 「风海大陆这么多年没出过很顶尖的强者，除了战争导致死伤率高，是否还因为——曾出过一个准备弄死海族与兽族的顶尖至强后，两族这么多年都在防着这种强者再出现。」**
- ⚠ **⚠ 倘若她真的脱困，第一个倒霉的一定是风海大陆**——⚠ **⚠ 「后续三个超脱之界的局面一定很精彩:女巫界失去了主心骨月女巫，陨灭星的冥神与夜母即将开战，风海大陆即将迎来蜘蛛夫人的报复。」**

## 🕸 头冠事件（52卷62章）

⚠ **她和 [熔火巨人](../永光世界/熔火巨人.md) 是一个时代崛起的老朋友**——⚠ **她无法离开灵魂书库，便让书库顶层的代理管理员 **小精灵·波波利** 想办法把一大箱材料送到熔火巨人手中打造头冠**;⚠ **⚠ 波波利不敢去已成苏晓地盘的永光世界（随便一只昆虫鸟兽都能把他当晚餐），只好联络三弟 [暴躁小精灵·迪亚古](../暴躁小精灵·迪亚古.md) 帮忙**——⚠ **迪亚古嘲讽二哥一顿，但也带材料去了，并把此事经情报渠道告知苏晓**;⚠ **⚠ 苏晓全当不知道**（⚠ 熔火巨人在永光世界的灰暗圣所内，谁敢去那边劫走或要挟熔火巨人，恶魔虫族了解一下）。

## 🕸 脱困 —— 十万年的等待终于到头（53卷17章）

⚠ **⚠ 她终于逃出了灵魂书库。**——⚠ **[熔火巨人](../永光世界/熔火巨人.md) 打造的那顶漆黑王冠（镶着一颗遍布裂痕的红色宝石）从风海大陆送到——⚠ **别说小精灵们的检查，就算是锻造大师也甭想从这王冠上发现端倪**;⚠ **⚠ 她握起王冠嘟哝「无论多么古老的存在，总有沉睡的时候」，一捏，王冠炸裂，只剩那颗红宝石被她捏在指间**——⚠ **宝石内绽放的猩红在这固态级空间封禁的灵魂书库内，炸出一道空间破洞**;⚠ **⚠ 她的甲足刺入破洞周边、注入剧毒，把空间侵蚀成一个通道，优雅地迈着甲足走进黑暗**——⚠ **⚠ 大书库内只剩倒下的书架与破碎的书籍。**

- ⚠ **⚠ 她合上的最后一本古籍，封皮刻着「爱与原谅」**——⚠ **她尝试过在内心中原谅海族与兽族的背叛，可越是试着原谅，内心就像有魂焰在燃烧，不断侵蚀她的理性与那为数不多的善良**;⚠ **⚠ 「唯有海族与兽族绝望的灵魂哀嚎，才能让她的内心平息。」**
- ⚠ **⚠ [团长](../星空座/团长.md) 与 [鹿神](../虚空/鹿神.md) 就在百米外旁观，选择不干涉**（详见团长页）——⚠ **⚠ 「这早就不是我们的时代了，况且我们当初……败了。」**;⚠ **⚠ 而 [暴躁小精灵·迪亚古](../暴躁小精灵·迪亚古.md) 特意来灵魂书库，只为确保他二哥波波利无事**——⚠ **他知道蜘蛛夫人要脱离囚笼了，这是谁也无法阻止的**（⚠ **亦如烈阳、永光、古龙这些超脱之界的衰落·⚠ 风海大陆作为第二纪元的超脱之界，也将在第三纪元迎来衰落之初始，这初始就在灵魂书库内——这是命定，是因果，是星界的循环往复，或许能勉强延后，但绝无可能阻止**）。

## 🕷 复仇成真 —— 风海大陆的深渊时代（53卷35-39章）

⚠ **十万年的等待到头，她走进黑暗、回到风海大陆复仇**——⚠ **至强巅峰的剧毒之雾下，兽族主城·永环城化为一片死域**（⚠ 世界树断裂燃烧·尸骸填满中心城区·鲜血顺排水孔涌出，宛如捏碎了整个兽族的心脏）;⚠ **⚠ 她单挑能力在至强巅峰中偏后，但猛毒太适合灭杀大势力——若她想，半月就能让整个风海大陆没有活物。**

- ⚠ **⚠ 她放出了「先祖」**:⚠ **第一纪元初期在浑浊黑暗中诞生的五位始祖异兽之一、最古老也最强大的异兽**——⚠ **百万米级体型、遮天蔽日、无数百米粗的黑色触须**;⚠ **⚠ 兽族/海族自食恶果**（⚠ 两族各留一个深渊通道以维持血脉诅咒·利用「先祖异兽」尸体被动吸收深渊能量结晶、还定期产出「黑暗晶石」——直到先祖活了过来）。
- ⚠ **⚠ 她一并屠灭了海族主城·亚托古城**（⚠ 剧毒海水化为幽紫色·海洋生物骸骨漂浮）——⚠ **她的目标是当初四名背叛者之一、也是策划者的海王**;⚠ **⚠ 「我也曾舍命守护风海之界不被深渊侵袭……可是啊！你们……还有海族，竟胆敢背叛我！让我在那该死的书库里，就像只被关到盒子中的蜘蛛」。**
- ⚠ **⚠ 海王的秘密**:⚠ **海王其实是「深渊族裔」、深渊家族的「第四之位」**（⚠ 用「古瓮」永远镇压家族其他成员·「你不是海族，就算成了海族之王也不是海族，你逃不掉」）;⚠ **⚠ 一名诡异老人从石门内的黑雾走出，海王生命力肉眼可见地枯萎、家族封印崩裂、海量深渊能量涌出**——⚠ **海族与兽族的时代就此结束，风海大陆的深渊时代降临**（⚠ 苏晓的安全区「暮冬城」因此成为风海大陆仅剩的安全区、价值水涨船高）。

## 🕷 堕入深渊族裔与风海大陆的深渊时代（54卷50章）

⚠ **风海大陆剧变后，她已堕入深渊、成为「深渊族裔」**——⚠ **上身人族、下身蛛腹甲足、及腰奶白长发、血红竖瞳、近五米身高·漆黑眼底与甲足黑色代表她已投身深渊**（⚠ 寻常强者没资格成为深渊族裔·她本就是「至强巅峰」、放眼虚空万界都是顶尖强者）;⚠ **⚠ 她在兽族主城原址·永环城，杀了昔日与苏晓并肩战狼神的骑士长**（⚠ 骑士长是风海大陆最后一个纯血兽族·她本欲拉他一同投入深渊「你父亲当初没背叛我」）。

- ⚠ **⚠ 与苏晓的死约**:⚠ **骑士长临死前告知，他已把兽族的所有秘宝通过暮冬城的渠道转送给 [苏晓](../苏晓.md)，作为代价，要苏晓帮兽族宰了蜘蛛夫人**——⚠ **「哦？灭法者白夜吗，看来下次见面，要取下你的脑袋了」**;⚠ **⚠ 风海大陆的「深渊时代」正在降临**（⚠ 苏晓判断风海剧变实由「阿兹古克家族」〔深海主宰·五大深渊族裔之一〕主导·⚠ 他计划晋升至强后第一时间去风海大陆、趁阿兹古克家族尚未彻底恢复时出手·⚠ 深渊大主教也在风海大陆）。
