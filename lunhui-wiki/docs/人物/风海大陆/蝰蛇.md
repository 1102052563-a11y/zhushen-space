# 蝰蛇

[苏晓](../苏晓.md) 招募的领主队成员——有一半人族血脉、四分之一蛇族血统，皮肤生着蛇鳞般纹路、给人邪异感，原是**铁堡城四十二城区的纹身店主**。他睚眦必报、擅长挑拨，却又是个**光速墙头草**——这就是蝰蛇。

## 🌾 光速墙头草

⚠ 苏晓初次登门递上晶体卡（考验:明早前自行到铁堡城），蝰蛇先是把卡丢进抽屉「会丢命的事老子才不去」，又几度动摇；同伙**魂狼族·恶齿**偷走卡想独吞机遇，苏晓便悄无声息在蝰蛇工作台上留下**印着「3」的新卡 + 纸条「明早前别迟到」**（意即能割他喉），把他吓出一身冷汗、乖乖赴约。⚠ 苏晓一眼看穿他:小聪明、对自身能力自信、凶狠狡诈**但不坚定、爱当墙头草**。日后若厄格因与恶齿起分歧，蝰蛇必定两头骑墙、从中挑拨，再第一个跑去向苏晓「打小报告」。

- **统领潜质（被动·A级）**：可将统领能力 75% 加成给麾下士兵。
- **能力数据**：统领力 A / 谋略 **S+** / 战力 A / 远征力 B / 忠诚 B——谋略最为出众。
- ⚠ 忠诚度提升最惊人:放血气那一刻，从 **10 点飙到 85 点**（墙头草性格使然、跟着强大领主有前途）。放毒是他的看家本事（自己牙里就有蛇毒、往酒里呲一点确认没事才喝）。

## 🩸 与恶齿的「毒杀」旧账

⚠ 蝰蛇与恶齿一见苏晓招募机会难得，竟**把其他应聘者全杀了**、搞出「要么选我俩、要么谁都招不到」的架势，气得 [巴哈](../剧毒魔鹰·巴哈.md) 差点弄死他俩，最后靠给两人下毒逼签契约（签名后毒解）、又让他俩**倒贴薪金**（杀了昨天费九牛二虎招来的人才）。

## 🏰 墙头草的终点 · 霜雪城城主（50卷59章）

⚠ 水晶森林战后，[厄格因](厄格因.md) 反骨疯长、当着他的面说出「**领主大人的决策，有时候也不一定完全正确**，我相信，只要我们的坚持是对的，大人不会责罚我们，**你说……对吗**」——⚠ **但凡他现在敢说出半个不字，厄格因都会让他出不了水晶森林**。⚠ **蝰蛇这墙头草何等机灵**，马上开始附和:「**对对对，大人会体量我们的难处。**」

- ⚠ 转眼厄格因就抱着那张霜雪城照片大步跑回暮冬城「禀告大人」，把他扔在原地一脸问号:「**？**」⚠ 书房内，苏晓说「我和霜雪城的老城主谈过了，他年老体衰，难以继续胜任城主之位，所以我决定……」——⚠ **正在后面咕嘟咕嘟大口喝茶的蝰蛇，呛得茶水都从鼻孔里喷出来**:「**所以我决定，让蝰蛇去担任霜雪城的城主。**」
- ⚠ 「**大人，您是说，我吗。**」「对。」——⚠ **得到明确答复的蝰蛇，完美复刻了厄格因方才脸上绽放笑容的神情**。⚠ **城主是兽族的第二梯队掌权者**（领主第一梯队、由兽王直接管辖；统领只到第三梯队）——⚠ 这个曾经的**铁堡城四十二城区纹身店主**，就此跻身兽族掌权者的第二梯队。（而当初生生把厄格因从 300 棍求情到 800 多棍的账，⚠ 这下算是**连本带利地收了利息**。）

## 关系

- [苏晓](../苏晓.md)（领主·灭法）：把他从纹身店诓来、又用毒治服的领主大人——他忠诚度飙升最快的一位。
- [恶齿](恶齿.md)：一同杀光应聘者、又互相下毒使绊子的损友兼同伙。
- [厄格因](厄格因.md)：领主队实际二当家——蝰蛇甘居其下、日后注定在厄格因与恶齿间骑墙挑拨。
