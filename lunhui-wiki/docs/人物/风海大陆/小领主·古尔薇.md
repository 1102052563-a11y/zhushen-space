# 小领主·古尔薇

[苏晓](../苏晓.md)「协助管理」的暮冬城封地名义领主——一个约 10 岁、气质忧郁、爱穿金黑色哥特连衣裙的人族小女孩。她**双亲遇害后就再没开口说过话**，靠比手势、写小纸条（一手小学生字迹，如「什么时候吃饭啊，好饿」）与人交流，心智却远超同龄人（超脱世界的小孩「都这么有演技」）。

## 🕯 被架空的傀儡 · 还是深藏不露？

⚠ 她的管家勾结两名城主、几乎架空了她手中实权，甚至软禁她于领主庄园。⚠ 但苏晓看穿她并非纯粹的傀儡:她**在后街区能反过来护着女仆**（凶徒、帮派成员见了她都退避低头），可见其身怀一种「只对后街凶徒用过」的能力——苏晓推断多半是**灵魂系或声波系**（她开口时周边世界会轻微扭曲，且能力的先决条件正是「说话/发声」，才落下不开口的习惯）。她表面懵懂、一有风吹草动就用衣兜里的宝石扣传信，实则冷静得可怕。

## 💎 家传秘宝 · 两块原初碎片

⚠ 她的家传秘宝（一枚项坠）核心镶着**两块原初碎片**——因家族从未出过接近绝强者的家主，这至宝一直没被启用、只当作父亲遗物戴着，直到女仆偷走它才传开。她感应不到秘宝确切位置，却能凭随身的一小块残片判断大致方位;她的三名亦友亦仆（[头狼·约克](约克.md) / 逆耳狐·银羽 / 石树·奥）组成的寻宝队因此成了最接近秘宝的人。⚠ 秘宝被寻回后当场引来[魔镰·泰莉德](魔镰·泰莉德.md)、巫毒术士、苏晓三方争夺，被苏晓的刃之魔灵夺走、项坠损毁，苏晓将两块原初碎片拿走、告知了她真相（她惊呆地明白了原委，还想知道「为什么会败得这么彻底」——才发现自己外衣被布布汪装了监听装置）。她收下苏晓给的钱袋后，噔噔噔跑上楼取来一颗灵魂结晶（小）塞给苏晓。

## 关系

- [苏晓](../苏晓.md)（灭法·伪装戈温/白夜）：名义上「协助」她管封地、实则借她封地藏虫巢的领主大人——夺走了她的家传秘宝（原初碎片），也除掉了架空她的管家。
- [头狼·约克](约克.md) / 逆耳狐·银羽 / 石树·奥：她的三名亦友亦仆、组成寻宝队替她找秘宝的好友。
- 管家（殁）：架空她实权、软禁她、勾结两城主的「忠诚」代理人——被八名城主取了首级献给苏晓。
