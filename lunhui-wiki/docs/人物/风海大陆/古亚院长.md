# 古亚院长（殁）

奥术永恒星**仲时学院的院长**——须发花白的**老牌绝强者**，主修**重力系**（背后会浮现虚影般的**长者之书**）。他阅历极丰厚、遇事从不慌神，是四绝强伏杀苏晓那局的策划者之一。

## 🧠 最了解乐园阵营的施法者

⚠ 他对乐园阵营的了解远超其他施法者。出发围杀苏晓前，他的一番话让不少施法者眼中的胜券在握**收敛了几分**:
- 「**别因为这灭法不是绝强就大意，他是轮回乐园的猎杀者。什么是猎杀者？狩猎那些违规的契约者，从一阶狩猎到九阶。我们要去围杀的，不是猎物，是名看着年轻、其实狩猎经验丰富的老猎人。**」
- 他还科普过契约者的公证机制（杀敌得宝箱、轮回乐园这方面奖励最高），一名年轻暗系施法者听得眼馋「这么爽？我都想去当契约者了」——⚠ 他笑着回:「**轮回乐园的契约者，从一阶到九阶，不足 1% 的生存率，你还去吗。**」
- ⚠ 他行事极谨慎:战后会用重力把敌人箭矢上**沾染的自己的血迹剥离并泯灭掉**——「这老施法者的阅历很丰厚，**会最大程度避免他人获得他的血，从而进行因果等方面的施术**」。

## 💀 三重毒循环 · 被算计到死（50卷52–53章）

⚠ 苏晓这局的真正猎物就是他。瑟菲莉娅被八个凛冬军团围困后，古亚院长的判断堪称理性:只要他带人成功增援，就有很大把握击溃那八个军团;⚠ 那样一来，哪怕当场围不住那灭法，对方也会因「为私怨折损兽族八个军团」而在兽族失势、被高层架空，后续便可在本世界追杀他到天涯海角。
- ⚠ 他的**路线选择也极稳妥**:主战场的传送阵天知道被动过多少手脚，「先祖灯塔」有不低概率被动手脚，「浮光岛」刚被凛冬军团打过也不稳妥——他退让到防御固若金汤、**近年从未被兽族攻袭到**的「**暗礁岛**」。⚠ 可惜苏晓根本没预判他的路线:**「先祖灯塔」「浮光岛」「安眠岛」「暗礁岛」四处最近的海族大城，同时都是这一幕**（因海王刚收拾完海神教、海族高层人人自危、全面收缩战线，让这些大城潜入了大量兽族眼线）。传送台上叠了两层阵图:**普通传送阵 + 灭法传送阵**。他以击退能力把五十多名施法者全顶飞、独自被卷走。
- ⚠ **太阳圣剑**（600 颗液态阿波罗 + 烈阳圆盘 20% 增益 + 体积小 50% 却威力更强）在他头顶的岩层内引爆——他单手虚握把几十公里内的重力全压向爆点，仍被烧得法袍破损、须发焦糊、整条左臂漆黑、右手法戒碎裂。⚠ 「如果是一般绝强者面对『太阳圣剑』，**有六成概率身死**」，但他是奥术永恒星的老牌绝强者，绝无可能因此而死。
- ⚠ 千余名人马族重骑兵的烈阳箭雨，只用**上百箭**就射爆了他的盾墙——但他背后浮现长者之书，一轮魔能冲击就把 **400 多名重骑兵炸成血雾**；他食指点在射到眉心前的螺旋箭上，重力令其扭曲压缩成**弹珠大小的黑色重力球**，弹指飞出、沿途一条直线尽碎。最终**上千名重骑兵的全部输出，只有三根分裂箭**。以他为中心方圆几十公里的地面被夯得平坦如洗，岩浆被强行冷却压成黑色石层，数不清的碎石因重力漂浮静止在半空。
- ⚠ 真正杀死他的是**三重毒的无解循环**:炼金剧毒（喝**灵魂能量**抑制剂压下 → 魂毒效力猛增）→ 魂毒（喝解毒剂压下 → 巫毒杀上来）→ 巫毒（喝**植物类**秘药压下 → 炼金剧毒又冲上来）。⚠ 「人老奸马老滑」的他每一步都掏出了压箱底的解药，却每一步都精准踩中下一个循环——**速度被削减到前所未有的程度**。⚠ 更绝的是：太阳圣剑的爆炸让方圆十几公里空间遍布裂痕，**没可能使用空间道具**；而乐园产地的保命道具，他**没有乐园烙印、根本用不了**。
- ⚠ 终局:苏晓以**极限血枪**（满血气 119% + 满灵魂能量 + 双血魂 + 仙露露奥义 + 巴泽的「远古狩猎」+20% + 莫蕾借出的星辉项坠/灵能挂饰 + 【厄运古戒】6 万法力换来的究极加成）一击射来。他察觉避无可避、从五公里外反向冲来，最后所能做的**只有单手按上枪尖**——魔能炸散，以他的右臂为起始点裂痕蔓延，手臂破碎、血枪贯首。⚠ 爆炸形成直径上百公里的**血色圆球**，持续十几秒后消散，笼罩其中的一切都消失了，**只剩犹如破布袋般向下坠落的古亚院长**。那巨大的半圆形凹坑随即涌满水液——「看起来用不了太久，这里就会变成一片湖泊」，水底的他气息快速衰弱，直至完全消失。
