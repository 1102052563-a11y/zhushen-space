# 头狼·约克

[小领主·古尔薇](小领主·古尔薇.md) 的三名**亦友亦仆**之首——年轻的灰狼族，自称「未来头狼」。他是个**与生俱来的冒险者**，贪的不是秘宝的价值，而是寻宝的过程与找到时的成就感。寻宝队三人以他为首，逆耳狐·银羽（狐族·心思颇多·出身贫民窟·常穿银羽衣）居中出谋，石树·奥（树茂族·耿直）压阵。

## 🗺 最接近秘宝的寻宝队

⚠ 因古尔薇能凭秘宝残片判断大致方位，约克三人成了**最接近那件家传秘宝（内镶两块原初碎片）的人**。他们查出秘宝被偷走后藏于某处地宫，可当银羽调查到新领主 [苏晓](../苏晓.md)「一人屠灭一整个海族军团、还能在主城和大统帅·凯恩互相算计」后，主张「战力谋略都斗不过、不如保持我们在暗他在明的局势」。⚠ 他们在地宫寻回秘宝时，同时招来了 [魔镰·泰莉德](魔镰·泰莉德.md) 与部落 [巫毒术士·巴泽](巫毒术士·巴泽.md) 的截杀，被巴泽的剧毒饲从追得夺路而逃，最终靠站在黑暗中的苏晓（刃之魔灵）逼退巫毒术士、才捡回性命。约克全程不知道，自己早被 [苏晓](../苏晓.md) 用布布汪的监听装置盯着——正是那个「幕后黑手」。

## 关系

- [小领主·古尔薇](小领主·古尔薇.md)：他效忠、替其寻宝的领主小姑娘、亦友亦仆。
- [苏晓](../苏晓.md)（灭法）：暗中盯着寻宝队、坐收秘宝（原初碎片）的「幕后黑手」，也是从巫毒术士手里救下他们的人。
