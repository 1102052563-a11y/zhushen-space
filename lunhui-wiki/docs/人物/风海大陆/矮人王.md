# 矮人王

万界**三大传说铁匠之一**（另两位是**恶魔铁匠**与**熔火巨人**，「其中就没有弱的」）——体格魁梧、头发胡乱披散、胡须扎成须辫。⚠ 苏晓见他的第一眼总觉得哪里不对，很快反应过来:这位**身高五米以上**的巨汉，怎么会被称为「矮人」？——转念一想便通了:若矮人王是**异兽时代**的族群，那就说得通了（异兽们都是千米、万米乃至十万米级的体型，五米出头**的确是矮人**）。

## 🔨 被软禁的传说铁匠

⚠ 他并不弱——「三大传说铁匠中就没有弱的，**孱弱的体格怎么可能在那足以灼伤强者的世界火炉旁工作**？更别说有时锻造到痴迷，长年累月地在那火炉旁」。他之所以被海族软禁，**更多是因为海族掌控了他的弟子们**。⚠ 他为海族培养了大量大师级工匠（这正是主战场上海族武器战甲品质高出兽族几个等次的原因），这让海族哪怕软禁着他，也得客客气气。⚠ 他能打造**增长自然元素的元素器物**——正是这些器物，换来了奥术永恒星对海族的助力，也让海族成了烫手山芋的持有者。

## 🕯 一句话认出的旧识

⚠ 苏晓仰头看他，他低头看苏晓，目光里没有获救的喜悦，**更多是缅怀感**。他开口第一句是:「**那老山羊打造的刀鞘，看着挺不错，非常不错**」——一眼认出【封魔】刀鞘出自恶魔铁匠之手。接过斩龙闪抽刀一看:「**这武器锋利的，看来你一路走来，满是艰辛和强敌啊**」。他抚过锻造台叹「很久没打造战甲了，你穿战甲吗？」——苏晓一句「不穿」，让旁边的 [秘纹师](秘纹师.md) 疯狂挤眉弄眼暗示他改口（矮人王帮忙打造战甲的机会**可遇而不可求**）。⚠ 他一眼认出 [凯撒](../尼古拉斯·凯撒.md):「**你是那个坑了地精公司十几次的欺诈者？**」（凯撒无辜摊手:「那不是坑，那是生意上的合作，做生意嘛，总是有盈有亏」——矮人王并不信）。

## ⚔ 意外锻造出「伪·灭法之刃」（50卷49章）

⚠ 苏晓请他打造**冒牌的灭法之刃**（用来骗猎人公会那 260 万悬赏）。他听完这构思不仅没意外，**反而感觉「这很灭法」**，只说「我的技艺还没高到能打造出灭法之刃，况且，**那不是能打造出的武器**」——但伪造可以试试。以苏晓的【**永恒权杖**】为主材、凯撒提供高纯度深渊之力充当魔灵气息。

- ⚠ 锻造过程失控:「越是锻造，矮人王越感觉这把半成品的利刃不对劲，**那种莫名的浑然天成感**，让他都有点沉迷到这次锻造中」——他索性往里加了 **1200 盎司时空之力**与各类高价值材料（这是除秘纹师那三块原初碎片外，他额外的答谢）。苏晓也跟着投入【神教徽章】【世界之环】【背叛者意志】【深渊灵魂之石×3】，凯撒一咬牙一跺脚掏出「分量大到让矮人王都侧目」的超高纯度深渊之力。矮人王遂遣散大部分弟子、只留几名技艺高超者。
- ⚠ 成品是【**伪·灭法之刃·魔灵意志（未唤醒状态）**】:攻击力 1~10、装备需求写着「**无任何人可正常使用此武器**」，评分却是 3000++++（起源级上限 3000）。⚠ 骗猎人公会的计划彻底失败，收获却更大——刀内**意外诞生的魔灵意志**，若被斩龙闪内的刃之魔灵吞噬，**魔灵强度最起码提升 100~130 点**（现魔灵强度才 87 点）。简介写着:「**意志不强大者，无资格拥有魔灵；灵魂不强大者，无资格让魔灵唤醒意志。**」

## 🛠 无特性起源级装备

⚠ 苏晓另委托他打造**无特性装备**（用来承接「掌控者」灌注的厄运）。矮人王的建议是:打造可以，但**最好以一件苏晓用过许久的装备作为契合核心**，以免打造出的无属性装备被环境或其他器物影响、出现自适应能力。苏晓遂选【**灵魂锁灯**】为核心（噬灵者天赋成熟后已不需它配合），最终打造出一件**评分 2000 点的无特性起源级装备**。⚠ 矮人王拒收苏晓自备的材料（这次算白送），但明言:下次再来打造虽不用出锻造费，**材料一类的费用得准备好**。

## 关系

- [苏晓](../苏晓.md)（灭法）：把他从海族救出的灭法之影——他一眼认出其刀鞘出自「那老山羊」之手，又替他打出了伪·灭法之刃与无特性装备。
- [秘纹师](秘纹师.md)：他最看重的首席弟子——为救他，秘纹师向苏晓开出了三块原初碎片的报酬。
- 恶魔铁匠 / 熔火巨人：与他并列的另两位传说铁匠（前者是「那老山羊」，打造了封魔刀鞘与骑士长的「雷陨」）。
- 海王 / 奥术永恒星：软禁他多年、靠他的元素器物做交易的两方——他一被劫走，这桩交易的根基就断了。
