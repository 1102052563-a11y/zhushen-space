# 雷法神·艾格

天启乐园的**战斗天使**——满头白发、气质与颜值罕有敌手（对路过异性眨左眼)，是**法系 + 雷系 + 灵魂系**三系兼修的九阶顶尖战力，也是天启乐园年度十大沙雕事件榜的常客。他接下 [海族统领·珀涅罗](海族统领·珀涅罗.md) 悬赏「除掉白夜」的委托，兴致勃勃地想找这位「斩首的夜」较量一番——却不知命运比他想象的更奇妙。

## ⚡ 三重克制 · 被刮痧的怀疑人生（50卷21–22章）

⚠ 艾格偷袭起手就放**界雷枪**（Lv.50+ 奥义·界雷+法系+灵魂三特性·十几公里玻璃化巨坑），结果只对苏晓造成 **3940 点混合伤害（最大生命 0.48%）**——一记大招给对方「刮了下痧」。他随即被苏晓的血枪远程压着打（近战系在远程压他这个中远程职业），逃又逃不掉（苏晓的追踪 + 肩上仙露露的超远程锁定 = 无解）。他祭出杀手锏**魂雷狱**（每秒 700~1500 + 最大生命 5% 的雷+法+灵魂伤害、每秒触发 3~9 次），却撞上苏晓的**究极克星组合**:绝魔体质 60% 法减、544 雷抗、灵魂之刃 20~50% 灵减、710 灵魂强度、伯格之心自适应抗性……层层削减后每次只有个位数~十几点伤害。⚠ 半小时后艾格被斩断手臂、彻底服软——他败得「不是实力差距，而是被克制到怀疑人生」。

## 🎭 保命权限 · 被钓来当邪神祭品

⚠ 艾格起初「淡然等死」，是仗着**天启乐园的保命权限**（须在因果层面死于敌手才触发、自我了断无效）。苏晓看穿后用「仁慈之刺」封住他，逼问三个诛心问题（「一个和你无冤无仇的人突然用雷劈你、你会怎么想」），把他问到沉默。⚠ 更惨的是他被苏晓当成**召唤邪神·腐化始祖的活祭品**（被魔灵操控、捧着伪装成巨大心脏的死灵之书献祭，亲眼看死灵之书反噬吞掉邪神）。最终他赔 **20 万灵魂钱币 + 签下苏晓那份「40 层薄如蝉翼」的契约**才被放走，一瘸一拐地走远。他事后与 [噩鬼·凯因](噩鬼·凯因.md)、女巫·莉莉亚（同被苏晓天克的「刮痧天团」）组队，任务物品却因苏晓覆灭邪神阵营、被原罪物卷走而泡汤。
