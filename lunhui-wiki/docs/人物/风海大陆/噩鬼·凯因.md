# 噩鬼·凯因

死亡乐园的契约者——**魂鬼能力（灵魂系）**，人如其名，是个「吃队友」的老手。他早年被 [苏晓](../苏晓.md) 收拾到怀疑人生:先是差点被界雷团灭，之后他召来灵魂之主等灵魂强者，随之「一记究极灵魂系大招打出三位数」的一幕让他毕生难忘——从此他见了苏晓就绕道，宁可不看热闹也不想再打照面。

## 😈 吃队友的噩鬼

⚠ 在白蹄港指挥所听到 [海族统领·珀涅罗](海族统领·珀涅罗.md) 发布「除掉白夜」委托时，凯因就满心恶意地盼着「主修法系+雷系+灵魂系的 [艾格](雷法神·艾格.md) 接了这单会有好戏看」——因为这三系恰好被苏晓天克。艾格果然接单挨了揍、赔钱回来，凯因还幸灾乐祸地拆穿他被勒索了 15~20 万。⚠ 艾格骂他「明知白夜克制灵魂雷系却半个字不提醒队友、真是人如其名吃队友的噩鬼凯因」，凯因却答「你说错了——我不仅知道他克制灵魂系，还知道他克制雷系」。⚠ 这临时组队的三人（凯因·灵魂系 / 艾格·法雷灵三系 / 女巫·莉莉亚·占卜+因果系）**恰好全被苏晓天克**，是个「联手对战苏晓简直就是刮痧天团、连因果系莉莉亚可能连痧都刮不了」的组合。他们的世界任务物品在邪神阵营，最终因苏晓覆灭邪神阵营、被原罪物（死灵之书）卷走而泡汤。

## 关系

- [苏晓](../苏晓.md)（灭法）：早年把他界雷团灭、灵魂反杀到毕生难忘、他从此绕道走的克星。
- [雷法神·艾格](雷法神·艾格.md) / 女巫·莉莉亚：与他组「刮痧天团」临时队、被他坑得不轻的队友。
