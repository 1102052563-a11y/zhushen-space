# 魂手·法斯特（殁）

奥术永恒星派驻风海大陆、随海族运输队押运能量水晶的**施法者**——遍布黑色符印的光头，是**灵魂系 + 法系**的九阶顶尖梯队战力。他被调来风海大陆的缘由，正是暗恨某个灭法**炸了繁茂星**，害他从舒服日子被发配到这处处凶险的战争世界。

## ⚔ 被灭法正面碾杀

⚠ [苏晓](../苏晓.md) 率领主队伏击运输队时，法斯特一度以隔空捏杀（把血兽族拧成血肉麻花）、重力压制、灵魂激流展现施法者威压，逼退兽族。但对上苏晓本人，他的克制被逐层拆解:

- **灵魂风暴**（毕生最强一击）打在苏晓身上，被绝魔体质 + 灭法叠满的 **60% 法系减免 + 500 多点法系抗性 + 710 灵魂强度**层层削到只剩个位数伤害——苏晓**直接从灵魂风暴里走了出来**，令他目瞪口呆。
- 他把九成本源生命力转成灵魂能量、召出**战魂骑士**孤注一掷，被苏晓一记『**刃道刀·极**』连人带骑士轰进几公里的玻璃化大坑，消瘦成皮包骨、须发枯白。
- ⚠ 他败得心服口服（「败得，真彻底」），随即被斩芒封喉。同行的**乌鸦女**（黑袍人）则被苏晓抓活口、塞了两件原罪物送往奥术永恒星。

## 关系

- 奥术永恒星 / 魂大人：他效忠、也在利用他的势力（同行黑袍人乌鸦女:「奥术永恒星只是一直在利用我而已」）。
- [苏晓](../苏晓.md)（灭法）：他暗恨「炸了繁茂星」害他被发配、最终正面将他碾杀的死敌。
- [乌鸦女](../画中世界/乌鸦女.md)：与他同队押运、他战死后被苏晓抓活口的黑袍人。
