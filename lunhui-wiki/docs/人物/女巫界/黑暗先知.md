# 黑暗先知

> ⚠ **[女巫界](../../世界/任务世界/女巫界-巫师界.md) 大名鼎鼎的「最强污秽者」**——⚠ **一位游历四方、吸收所见苦痛、病痛、厄运等不幸的古老旅者**;⚠ **⚠ [星象塔](../../世界/势力/星象塔.md) 现存成员之一，持有「黑暗之血·智慧」。**

## 🌑 他为何跪伏在灭法者面前

⚠ **原本苏晓以为这是主线第四环要死战的对手**——⚠ **黑暗先知却主动跪伏在地、双手托举半流体的「黑暗之血·智慧」献上**:

> ⚠ 「**尊敬的灭法者，原罪封守者，死寂之王，我这污秽之人能见到您，无上荣幸，请您收下这些暗之血，请您，让我们的女巫解脱吧。**」

- ⚠ **他活了太久、欲望早已麻木**:⚠ **对钱财权力视若无物，所期望的只是让那些心有善良的生灵不要、或少承受苦难**;⚠ **⚠ 他把原本会蔓延开、足以让亿万生灵死去的「异种深渊能量」封印到自身、以己身为容器**——⚠ **再继续寻找那些可能蔓延的「异种深渊能量」**（⚠ 身死后化为枯树、只在地毯上留下一个直径两米的黑色灰烬印记）。
- ⚠ **他见过格林·吉莉安**:⚠ **一位在狂喷深渊能量的巨大深渊通道前、翘着二郎腿坐在深渊巨兽首级上的女灭法**（⚠ 不远处是断颈涌黑血的无头巨兽尸体·⚠ 「强大又魅力十足」）——⚠ **随后就爆出这女灭法骚扰当代月女巫、意图不轨，被愤怒的巫师们压着脾气请出巫师界**;⚠ **⚠ 也因此，听闻有人封印五件原罪物＋死寂本源，他先是诧异，得知此人是灭法者后便不那么意外了。**

## ⚠ 异种深渊能量（二代深渊能量）

⚠ **渴望力量者吸收深渊之力变强，最后失控时，体内按其灵魂、意志、身体能量适配过的深渊能量爆发开——即为「异种深渊能量」**（⚠ 专业称呼）——⚠ **它具有污染性与滋生性**;⚠ **⚠ 树生世界的精灵之都·潘达兰、风海大陆的黑暗地带，都是被它污染的区域。**
