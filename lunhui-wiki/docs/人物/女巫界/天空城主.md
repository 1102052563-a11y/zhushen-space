# 天空城主

> ⚠ **[女巫界](../../世界/任务世界/女巫界-巫师界.md) 隐藏的第三位至强者**（⚠ 至强初期·⚠ **一个世界出三位至强已是相当了不起**——⚠ 风海大陆出了一位至强的蜘蛛夫人，这么多年都是那边的光辉事迹）;⚠ **⚠ 身高三米五、胳膊比常人腿还粗的魁梧巨汉。**

## 🧒 三个玩泥巴的孩子

⚠ **⚠ 只有 [月女巫·瑟希莉丝](瑟希莉丝.md) 与 [会长·珀·耶恩](珀·耶恩.md) 知道的秘密**:⚠ **这三人都在天空城·底城长大**——⚠ **月女巫和会长是两个落魄巫师家族的成员，天空城主是孤儿院长大**;⚠ **⚠ 小时候一次偶然的经历，让三人在城东的巨大排污渠附近偶遇，从此成了一起玩泥巴、一同在钢铁管道上疯跑的小伙伴**;⚠ **⚠ 长大后分别、又在巫师公会内重逢、再分别，直到多年后三人都已登上巅峰。**

- ⚠ **他的「反派」戏码**:⚠ **天空城在他手上「恶变」概率创历史新高，可这不是他无能——是整个巫师体系出了问题、所有巫师的恶变概率都在飙升**;⚠ **⚠ 月女巫的对策是完善「疯狂序列」与「污秽序列」这种饮鸩止渴的秘法，且**只让天空城掌握**——⚠ **快恶变的巫师为活命只能前仆后继投奔天空城**;⚠ **⚠ 如此一来天空城的恶变率飙升，再经月女巫手下引导，让世人以为那是天空城巫师「自己故意为之」，抛去这部分统计，外界巫师的整体恶变率就保持了平稳**（⚠ **他替整个巫师阵营背了这口锅**）。
- ⚠ **他的选择**:⚠ **「我不管星象塔、女巫公会、古王城、灭法者这四方怎么交锋，哪怕四方全部打到重伤，巫师大陆依旧是巫师阵营的——可如果让黑暗神教趁机得逞，那就糟了」**;⚠ **⚠ 他在餐厅结账时故意掉一枚硬币，趁店长抬头看时一把捏爆其头颅**（⚠ 「这不就是黑暗神教的杂碎嘛」·⚠ 无头尸体因体内无人控制的黑暗能量而畸变）。
- ⚠ **⚠ 落败**:⚠ **他对上神父＋白金使徒＋深渊大主教三人，死得不冤**——⚠ **他反对月女巫「把天空城居民疏散到巨铠城」的命令，正是他已落败的信号**;⚠ **⚠ 月女巫与会长没来救援——冥神已离开陨灭星，这是月环城与天空城之间的抉择，三人都选择保住月环城（巫师阵营的心脏）。**
