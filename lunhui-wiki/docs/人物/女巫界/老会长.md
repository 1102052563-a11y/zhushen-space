# 老会长

> ⚠ **[星空研究会](../../世界/任务世界/女巫界-巫师界.md) 的上一任会长**——⚠ **太强了:不仅培养出至强的 [会长·珀·耶恩](珀·耶恩.md)，还把会长之位传给这位深受自由巫师崇敬的新会长**（⚠ 在此之前，女巫公会与星空研究会从没受过自由巫师拥护）;⚠ **⚠ 他不倚老卖老（对月女巫直称「月女巫大人」），也因此在十年前的无月暗杀之夜没挨背刺。**

- ⚠ **被原罪之书冲击（52卷15章）**:⚠ **苏晓翻开原罪之书时他撞倒座椅、掏药瓶吞小药片顶着，看到大爹级原罪物一件件亮相脑瓜嗡嗡**——⚠ **他是最先追问「我们应该拿出什么诚意，你才放过女巫界」的人**（⚠ 被苏晓「以封印五件原罪物的封印学水平担保、万无一失」说服）。

## 出处（第52卷）

- 第52卷15章（**星空研究会上一任会长·培养出至强会长珀·耶恩+把会长位传给深受自由巫师崇敬的珀·耶恩·不倚老卖老对月女巫直称大人·十年前无月暗杀之夜因此没挨背刺·被原罪之书冲击撞倒座椅+吃药顶着·追问苏晓拿什么诚意放过女巫界·被苏晓封印学担保说服**）。
