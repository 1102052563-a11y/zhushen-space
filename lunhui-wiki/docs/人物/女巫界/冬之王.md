# 冬之王

> ⚠ **[女巫界](../../世界/任务世界/女巫界-巫师界.md) 大陆最北「永冬之城·隆卢」之主**（⚠ 冰裔·古老族群）——⚠ **一位「战王」，长处不在武力而在率军征战，其「凛冬军团」连巫师阵营都异常忌惮**;⚠ **⚠ 巫师时代前，北境的冬之王与南边大沼泽的古王打得不可开交，后被崛起的巫师打退到苦寒的永冬城。**

## ❄ 命不久矣的贪欲

⚠ **他已命不久矣**——⚠ **年老带来的衰弱与死亡临近，让这昔日王者贪欲疯长、睿智被欲望与苟且偷生侵蚀千疮百孔**;⚠ **⚠ 他执意与黑暗神教合作**（⚠ 次子 [厄姆](厄姆.md) 一直反对）——⚠ **因为 [黑暗双子](黑暗双子.md) 身上就有他能续命的黑暗本源。**

## 🎣 苏晓故意留的活口（52卷11章）

⚠ **决战中，冬之王带百余亲卫躲在暗处想渔翁得利**——⚠ **被苏晓与双生领主联手先清算，双生领主一记「黑暗降临」把百名精锐亲卫全灭**;⚠ **⚠ 苏晓一脚把冬之王踹飞、由 [巴哈](../剧毒魔鹰·巴哈.md) 的空间漩涡救走**——⚠ **⚠ 苏晓故意不让他死在此战**:⚠ **他有七成以上把握，冬之王也被 [白金使徒](../暗影世界/白金主教.md) 吞噬了命运**（⚠ 从冬之王说出「保护本王」后自己都诧异的神情判断·以他的命运/因果之强，这是白金使徒经营了几个世界进度才达成）;⚠ **⚠ 让冬之王缓过来→整个永冬城成为白金使徒/神父/深渊大主教的新仇家**（⚠ 破坏敌方「让永冬城陷入混乱」的计划转折点）。

## ❄ 恢复清醒 · 送出【初始源质】（52卷14章 / 17章）

⚠ **苏晓找 [命运女神](../虚空/命运女神.md) 拿到「命运晶石」，在永冬城王宫当着冬之王的面击碎，用无特性的命运之力临时补上他被吞噬的命运**——⚠ **冬之王脸颊剧烈颤动、牙齿咬得咔咔响，随后神情平静「就像做了一场很真实的梦……我的时间的确不多了」**;⚠ **⚠ 昔日那位睿智的北境之王回来了**——⚠ **他平静接受将死（无特性命运之力飘散后即死期），趁这段时间安排好继承人（长子次子通过考验）、避免永冬城陷入混乱、避免十几代冬之王的累积土崩瓦解。**

- ⚠ **复仇的决心**:⚠ **他眼中那双凛冬狼王般的眼睛，有着势必向 [白金使徒](../暗影世界/白金主教.md)/[神父](../神父.md)/[深渊大主教](../永光世界/深渊大主教.md) 复仇的决心**——⚠ **「这次欠了你个大人情，最晚明天还你」**;⚠ **⚠ 次日他派仆从给苏晓送来【[初始源质](../../力量体系/装备道具/index.md)】×1**（⚠ 晋升至强所需的顶尖资源·世界树树脂琥珀封混沌源质·气魄过人）;⚠ **⚠ 他一恢复，整个永冬城/北境就成了三方老阴哔的死敌**（⚠ 苏晓感觉到来自白金使徒与神父的压力骤减）。
