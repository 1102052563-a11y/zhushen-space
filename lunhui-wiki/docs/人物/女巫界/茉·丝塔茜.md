# 茉·丝塔茜

> ⚠ **[阿格耶](阿格耶.md) 的冰裔未婚妻**——⚠ **也是 [凛冬之剑·厄姆](厄姆.md) 从小照料的远亲**;⚠ **⚠ 永冬之城·隆卢被毁灭时，除厄姆外唯一的幸存者**——⚠ **⚠ 这绝非侥幸，而是神父等人故意留下的活口。**

## 🕸 三人相爱相杀的棋局

⚠ **[白金使徒](../暗影世界/白金主教.md) 对命运的操控之精准，从这三人的关系可见一斑**:⚠ **冬之王处死了阿格耶全家 → 厄姆是阿格耶的血仇死敌;茉·丝塔茜是厄姆的远亲、阿格耶的未婚妻;而在厄姆眼中，投靠神父等人换力量的阿格耶是「仇敌的狗腿子」，还跟着他的茉·丝塔茜则是「叛徒」**;⚠ **⚠ 而茉·丝塔茜自己，甚至还不知道永冬城已被毁灭。**

- ⚠ **街角的重逢**:⚠ **三人「遵循命运的引导」逐渐靠近彼此**——⚠ **她在街角看到厄姆，泪水涌上眼圈、抛下纸袋跑过去拥抱这照料过她的远亲**;⚠ **⚠ 她鬼使神差说出的第一句「走，我们去找阿格耶，我们一定能说清彼此的误会……」，早已被人提前设定好**——⚠ **这句话让厄姆想起阿格耶正是那个害他父王被算计的家族余孽，脸上浮现病态扭曲的笑容。**
- ⚠ **结局**:⚠ **她被倒吊在吊灯上、全身竖向割痕，生前遭凌辱与折磨、受尽苦痛后失血过多而死**（⚠ 厄姆:「你要是早回来半刻，还能在她死前，让她看到你最后一眼，这是她最后的愿望」）;⚠ **⚠ 而厄姆事后被增生组织吞没时，看到吊灯上她的尸体，完全理解不了自己上午为何如同失了理智的野兽般摧残这近乎视为义妹的远亲——「这一切，就像他的潜意识被人操控了般」。**
