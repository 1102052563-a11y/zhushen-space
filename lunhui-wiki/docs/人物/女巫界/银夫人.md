# 银夫人

> ⚠ **[夜惑女巫公会](../../世界/任务世界/女巫界-巫师界.md) 的首席长老**——⚠ **由曾大力支持 [月女巫·瑟希莉丝](瑟希莉丝.md) 上位的「白银家族」族长担任**（⚠ 因瑟希莉丝是拎前任月女巫头颅上位、首席长老无法由上代月女巫担任）;⚠ **⚠ 她代表所有巫师家族的利益（钱财、资源、可观战力），是制衡瑟希莉丝的一股力量**——⚠ **瑟希莉丝正是通过银夫人才能掌控所有巫师家族。**

- ⚠ **右臂的疤痕**:⚠ **她右手臂上一道道骇人疤痕，是 [杀手三兄弟](杀手三兄弟.md) 在无月暗杀之夜「搞错目标」所赐**（⚠ 险些让她死在那夜）。
- ⚠ **被原罪物吓崩（52卷15章）**:⚠ **得知苏晓带来原罪物已经不淡定，听说「五件」直接被打击到坐下、抓着扶手发抖「一定不能让他死在巫师界」**——⚠ **看到大爹级·收藏家时血压当场飙升。**
