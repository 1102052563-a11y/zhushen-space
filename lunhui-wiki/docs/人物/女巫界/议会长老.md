# 议会长老

> ⚠ **天空城顶城·古老祭祀场上空飘浮的三道黑色漩涡**——⚠ **天空城最初的监视者**;⚠ **⚠ 它们不在乎权利、财富，仅看重誓约、因果等**（⚠ 帮谁全看谁与它们签订了初始契约）;⚠ **⚠ 在深渊侵袭本世界的时期「议会」就已存在，时至今日早已衰落，但三位「议会长老」在天空城的地位无法撼动**（⚠ 连城主也要保持足够敬意·有说法称「三位议会长老恒古不变，天空城的城主却每隔几百年换一位」）。

## 🌑 与莎的交易（52卷30章）

⚠ **帮 [莎](../莎.md) 掌握「最纯粹深渊之力」的正是这三位**——⚠ **代价是莎要替 [星象塔](../../世界/势力/星象塔.md) 的前任双子守卫（[黑暗双子](黑暗双子.md)）报仇。**

## 🗣 它们的预言（52卷30章）

> ⚠ 「**古之王将会苏醒，我们，要干涉吗。**」
> ⚠ 「**不，已经……是时候了。**」
> ⚠ 「**恶变将要满溢而出。**」
> ⚠ 「**她已至极限，一切皆为命定。**」
> ⚠ 「**包括灭法的到来？**」——⚠ 「**或许，是吧。**」
> ⚠ 「**他可是天选，我们怎敢窥探他之命定。**」
> ⚠ 「**不，他只是拥有了「世界石」的天选，还不是唯一。**」

- ⚠ **⚠ 「世界石」的天选**指向苏晓的 [起源石·世界](../../力量体系/装备道具/index.md)（⚠ 简介正是「你拥有了『资格』，却还不是唯一」）——⚠ **「古之王将会苏醒」「恶变将要满溢而出」则是后续剧情的明确预告。**
