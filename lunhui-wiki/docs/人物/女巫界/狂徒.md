# 狂徒

> ⚠ **[黑洞·阿兹勒](黑洞·阿兹勒.md) 的心腹**——⚠ **名为心腹，其实两人实力不分伯仲**（⚠ 都是 [深渊大主教](../永光世界/深渊大主教.md) 一具化身的养子）;⚠ **⚠ 黑暗神教三大顶尖爆炸物制造者之一**（另两位:永冬城一名王室成员、巨铠城的疯狂女巫莉莉丝·薇儿）。

## 🔥 野兽与爆炸物大师

- ⚠ **形貌**:⚠ **身材健硕、赤膊上身、满头乱发狂野披散·满嘴交错的尖牙如野兽利齿**——⚠ **路过肉摊顺手抓生肉仰头就嚼**;⚠ **⚠ 一个彻头彻尾的疯子，却能制作出极精密的爆炸物**（⚠ [神速](神速.md):「难以想象你这种粗鲁的野兽，居然能制作出那么精密的爆炸物」）。
- ⚠ **战绩**:⚠ **曾炸毁月环城一整个城区**——⚠ **古王城、巨铠城等要地都被他袭击过、损失惨重**;⚠ **⚠ 准备充足、知道对手的情况下，战力不弱于 [黑暗双子](黑暗双子.md)**——⚠ **属于「约架幻神级、遭遇战比较一般」**。
- ⚠ **宝石炸弹**:⚠ **能操控心灵让无辜者在毫不知情下携带伪装成宝石/金属饰纹的爆炸物**——⚠ **52卷16章伏击苏晓与 [瑟琳](瑟琳.md)（女仆装仆从的宝石炸弹·火焰→重力→空间多层叠加爆炸）**;⚠ **⚠ 生命炸弹曾引爆星辰女巫·尤里娅的生命力。**

## 🎭 对神父的忌惮 / 蜂巢剧场（52卷17章 / 20章）

⚠ **他忌惮神父「简直像人形古神，不，他就是古神」**——⚠ **[神父](../魔灵星/神父.md) 一句话他就替其跑腿传话给 [神速](神速.md)**;⚠ **⚠ 20章他与黑洞·阿兹勒在蜂巢剧场以「血肉替身」现身拖住苏晓、引爆全场**——⚠ **切身感受到直面灭法之影「终极反派大 boss」般的血气压迫**（⚠ 立即引爆、不敢耽搁）。

## 出处（第52卷）

- 第52卷12章 / 16–17章 / 20–21章（**黑洞·阿兹勒心腹实则实力不分伯仲·深渊大主教一具化身的养子·黑暗神教三大顶尖爆炸物制造者之一·身材健硕赤膊满头乱发+满嘴尖牙野兽+顺手抓生肉嚼·疯子却能造精密爆炸物·曾炸毁月环城一整个城区·准备充足战力不弱于黑暗双子·约架幻神遭遇战一般·宝石炸弹操控心灵让无辜者携带·伏击苏晓瑟琳·生命炸弹炸死尤里娅·忌惮神父像古神·蜂巢剧场血肉替身炸落星城10城区近半**）。

## ⚔ 黄金斗技场决战 · 殁（52卷29章）

⚠ **他靠「冠军戒指」跳过第七场直接晋级第八场，与苏晓在古战场·巨兽埋骨地决战**——⚠ **他毫无大 boss 风范:坐在怪石上大口吃烤兽腿「这得有五成概率是老子的最后一顿了，总得吃饱不是」**，还试着用一颗「原初之核」买下冠军之位。

- ⚠ **他的真正出身**:⚠ **他曾只是个街头混混，偶然得到一份爆炸物术式图纸，借此不断向高处爬**——⚠ **只要能战胜挡路的强敌，无论多卑鄙无耻的手段他都视为可行策略**;⚠ **⚠ 他从未忘记自己出身卑微，所有的一切都要亲手去夺。**
- ⚠ **[深渊契约](../../力量体系/装备道具.md) 形态**:⚠ **四米以上人形怪物·金属般乌光的皮肤·大臂与后肩的气孔喷吐以生命力为燃料的烈焰·心脏如引擎般炽红**——⚠ **核心优势是「根据敌人攻击特性产生临时进化」**（⚠ 被斩几次后，苏晓的刀再斩其手臂已有明显滞塞感·他本想一直拖到能反杀）;⚠ **⚠ 他的代价之一是「百倍苦痛」——这让青钢影能量附带的剧痛首次起到惊人效果（疼到差点休克）。**
- ⚠ **被界雷终结**:⚠ **苏晓穿上自创的「猩红盛装」（由 [血魂之握](../../力量体系/装备道具.md) 的鲜血战甲以精神力操控拟定雷系防御而成），以 1180 元素亲和力引下几十公里粗的界雷柱**（⚠ 狂徒:「这……真的是绝强者有资格使用的能力吗？」）——⚠ **随后「[天怒·奔流斩](../../力量体系/技能.md)」把周边界雷尽数聚到刀上、一刀将其斩成两截**（⚠ 斩龙闪耐久骤降五分之一）。
- ⚠ **⚠ 被至亲背叛的终局**:⚠ **他还剩一招同归于尽的「华丽谢幕」，却因灵魂能量消耗过多而无力施展**——⚠ **可他忽然发现自己的灵魂被引燃了:是 [芙拉儿](芙拉儿.md) 远程激活了这能力**;⚠ **⚠ 他先是愤怒（来自至亲的背叛与算计），随后释然——因为他的最终目的也是背叛**;⚠ **⚠ 「呵。」他在最后笑了，目光有几分无神地看着天空，自爆殆尽、尸骨无存**（⚠ 苏晓因此无法剥离其灵魂记忆）。
