# 初始女巫·蒂塔佩娅 / 蒂因什

> ⚠ **⚠ 这个超脱之界为何不叫「巫师界」，而叫「[女巫界](../../世界/任务世界/女巫界-巫师界.md)」？**——⚠ **因为在这超脱之界，有位初始女巫叫**蒂塔佩娅**。**

## 🌑 恶变的真相与月之圣殿

- ⚠ **恶变的病根**:⚠ **使用元素巫术，必定要让自然元素途经身体**——⚠ **自然元素内含有的微量深渊之力难以避免地沉淀在体内，长久淤积后引发「恶变」**;⚠ **⚠ 更可怕的是:恶变能量**不会因巫师的死亡而消散**——⚠ **今后巫师越多，恶变能量就越多，直到环境中都充斥恶变能量**;⚠ **⚠ 到那时，只要进行巫师体系的传承，哪怕只是成为巫师学徒，都有不低的概率恶变——巫师阵营等同于覆灭。**
- ⚠ **消除的代价**:⚠ **每名巫师恶变后所产生的恶变能量都有细微差别，每一种都要用对应的资源、方式、术式处理**——⚠ **有巫师长老计算过:处理 10000～15000 份各种恶变能量，就需要这个超脱之界九成以上的资源**（⚠ 完全消耗不起）。
- ⚠ **⚠ 她的选择**:⚠ **她消耗海量资源在月亮上建立了「月之圣殿」——只要到了夜晚，圣殿就会吸收本世界的恶变能量**;⚠ **⚠ 可圣殿需要一个核心，而这世上没有任何东西会主动吸收恶变能量**——⚠ **既然没有，那就制造一个:**她自行选择恶变，随后命人将自己封印到月之圣殿内，成为这圣殿的核心**;⚠ **⚠ 这就是初代月女巫为何忽然消失、甚至没时间培养继承者**（⚠ 那时本世界环境中的恶变能量已经不低了）;⚠ **⚠ 巫师史上恶变的巫师何止亿万，本世界的环境中却毫无恶变能量——全被「月之圣殿」吸收了。**
- ⚠ **配套的一切**:⚠ **[星象塔](../../世界/势力/星象塔.md) 的成立，其实就是在守卫月之圣殿与里面的初始女巫**（⚠ 这也是月女巫·[瑟希莉丝](瑟希莉丝.md) 曾是星象塔成员的原因）;⚠ **⚠ 「星象圆盘」＝前往月之圣殿的钥匙**;⚠ **⚠ 而五份「黑暗之血」，其实就是**恶变后的初始女巫本人的部分源血**——⚠ **既是圆盘的一部分，也是让后世人有守卫圆盘的力量**（⚠ 这也解释了黑暗之血里那离谱的恶变能量浓度从何而来·⚠ **也解释了 [古王](古王.md) 为何肯当考验者之一保管一份**——⚠ 初始女巫有让这位古老王者这样做的资格）。
- ⚠ **⚠ 圣殿的封印**:⚠ **是灭法阵营独有的「超·界级封禁术式」的浓缩版**（⚠ 后者针对一个世界，前者针对一栋建筑）——⚠ **殿内能发挥的最强战力为**绝强级**，高于绝强者连门都进不去、用星象圆盘都不会有反应**;⚠ **⚠ 这是她当初防范自己被恶变侵蚀到失去理智后破坏圣殿、也防范心怀不轨的至强者从内部打碎封印所设的保险**（⚠ **月之圣殿可以再建，但真的还会有第二位「初始女巫」吗**）。

## 🖤 黑女巫·蒂因什

⚠ **被恶变侵蚀这么多年，她产生了第二个人格**:⚠ **白女巫＝曾经的初始女巫、女巫界的守护神;黑女巫＝生自恶变、渴望冲破封印降临女巫界的黑暗意识**;⚠ **⚠ 黑女巫曾告诉瑟希莉丝:**待她冲破封印，女巫界别想有一个活着的生灵**。**

- ⚠ **她已经赢了**:⚠ **「母亲已经不再了，没人能再阻止我，这属于母亲的世界，当然也要继承给我。」**——⚠ **初代月女巫·蒂塔佩娅早就不在了，眼下这已是另一个人，是获得蒂塔佩娅所有东西的**初始女巫·蒂因什**;⚠ **⚠ 她完美继承了对抗深渊至今的全部战斗经验与能力，「说她是绝强级的天花板，也毫不夸张」**——⚠ **⚠ 如同优雅又残忍的狩猎者，是微笑着的黑暗意志化身。**

## ⚔ 苏晓凭什么能赢（52卷53章）

⚠ **⚠ 他不是靠正面战胜——他从一开始就没打算正面赢。**

- ⚠ **他的自知之明**:⚠ **女巫界多年来一个个绝强巅峰的强者来此挑战，全部落败**——⚠ **苏晓的战力还没达到那些强者的程度，「他凭什么能赢呢？」**;⚠ **⚠ 真正打动他的，是月女巫无意间说的那句:**月之圣殿的封印是灭法阵营独有的「超·界级封禁术式」的浓缩版**。**
- ⚠ **⚠ 放逐术式**:⚠ **在其他环境下，他成功放逐初始女巫的概率连一成都不到**——⚠ **可身处这灭法阵营独有的封禁领域内，成功率至少八成以上**;⚠ **⚠ 阵图从哪来？**苏晓、[阿姆](../阿姆.md)、[巴哈](../剧毒魔鹰·巴哈.md) 溅得遍地都是的鲜血**——⚠ **战至双臂尽失、阿姆被斩首、巴哈被斩落一翼时，场地上的血迹已构成一个与【封之刃】刻印一模一样的巨大血之阵图。**
- ⚠ **⚠ 灭法阵营最强囚笼·永光世界**:⚠ **在那边等着她的是 1500 万精英恶魔兽、120 万只恶魔焰龙、15 万只泰坦巨兽**（⚠ 不够还能爆兵）——⚠ **⚠ 「以往灭世级存在被放逐过去是囚困，现在是被围攻到死」**;⚠ **⚠ 加上永光世界的「超·界级封禁术式」余威仍在，让她无法战力全开。**
- ⚠ **终局**:⚠ **苏晓以「指定召唤」召来一只孢子坦克——它现身即爆碎，吐出生命值仅剩 **0.65%**、身负上千层「电浆侵蚀」与龙焰「易伤状态」的初始女巫**;⚠ **⚠ 她单膝跪地、拄着战镰才没倒下——没半句废话，升腾着黑蓝色烟气的长刀斩过。**
- ⚠ **她临终的怒吼**（放逐时）:⚠ **「我一定……回来！让你，死无葬身……之地！！」**
