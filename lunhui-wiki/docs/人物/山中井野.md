# 山中井野

[火影忍者世界](../世界/任务世界/火影忍者.md) 木叶忍者、山中一族——金发美女，**心转心之术**（意识入侵敌身夺取控制）使用者，「**猪鹿蝶**」组合之一（与 [奈良鹿丸](奈良鹿丸.md)、[秋道丁次](秋道丁次.md)）。其父**山中亥一**（老一辈猪鹿蝶）随后赶到。

- **被青钢影驱逐 · 精神受创（九卷67章）**：围攻 [苏晓](苏晓.md) 时她以心转心之术钻入苏晓意识，却被其体内的**蓝色青钢影能量驱赶出来**、心转心之术被打断（「他身体里有种蓝色能量」）——退出后满眼血丝、精神受创（用精神硬抗青钢影是作死·非十天半月难恢复）。其父山中亥一同样在再度尝试时被青钢影反噬。
