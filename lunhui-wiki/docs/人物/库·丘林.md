# 库·丘林（Lancer · 光之子）

[Fate/stay night 第五次圣杯战争](../世界/任务世界/fate-第五次圣杯战争.md) 的 **Lancer（枪兵）**——北爱尔兰阿尔斯特地区的英雄「光之子」，更多人称「库兰的猛犬 / 狗哥」（自称不伤任何犬类）。蓝色软甲、血红长枪、耳坠挂坠；颜值在本届从者中数一数二（「枪兵的实力普遍不怎么样、但一个比一个俊美」）。御主是监督方 [言峰绮礼](言峰绮礼.md)，被当放养型情报 / 暗杀棋子使用。

- **被苏晓教育**：开局追杀 [卫宫士郎](卫宫士郎.md) 时撞上 [苏晓](苏晓.md)，几招就被按地暴揍、还被 [吞噬之核](../力量体系/技能.md) 抽走大量英灵能量（苏晓没灭口、怕刚开战死英灵会让血门追查·只让他回去告诉言峰绮礼「我会去拜访那弑父的兔崽子」）。战力因此严重下滑。
- **被阿姆摔死**：奉令咒强制与 [阿姆](阿姆.md) 战（耻于打一名重伤的对手），却被进化后的凛冬战牛**冰封 + 寒冰刺贯穿 + 房屋大小冰锤**砸到重伤，最后被阿姆抓着右腿「像布布汪看电影学的招式」反复摔打致死——苏晓的从者击杀 Lancer、获**世界之源 7.3% + 金色套装宝箱**。

## 关系
- **[言峰绮礼](言峰绮礼.md)**：御主（监督方·把他当放养棋子）。
- **[苏晓](苏晓.md)**：开局把他教育到怀疑人生、抽走其英灵能量的「无名英灵」。
- **[阿姆](阿姆.md)**：摔死他的凛冬战牛（他承认是值得尊敬的战士）。
- **[吉尔伽美什](吉尔伽美什.md)**：同阵营（停车场二打一围攻阿姆 / 远坂凛）。
