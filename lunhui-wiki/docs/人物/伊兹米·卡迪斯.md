# 伊兹米·卡迪斯

主角 [爱德华·艾尔利克](爱德华·艾尔利克.md) 兄弟的**师傅**（炼金术不弱·是 [瓶中的小人](瓶中的小人.md) 国土炼成阵的「五人柱」之一）。

- **身世**：曾因流产而**人体炼成未出生的孩子**失败、失去部分内脏并永久不孕，遂在小镇半隐居、断然拒绝国家招揽。丈夫是熊般壮汉（近身可正面战人造人）。
- **历程**：七卷11章 [布布特尼](布布汪.md) 曾装流浪狗潜入她夫妻的肉铺卧底（凭 43 魅力骗吃骗住），记住来探师的阿尔冯斯气味、循味找到 [格利德](格利德.md) 的藏身处。
