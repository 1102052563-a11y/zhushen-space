# 比克

[全职猎人世界](../世界/任务世界/全职猎人.md) **[十老头](../势力/index.md)**（掌六块大陆黑帮的十名首领·办友克鑫拍卖会）中的**代表人物**——寸发、嘴上一道疤、惯性摸疤思考，是个**人精**。[黑玛丽](黑玛丽.md) 的父亲。

- **与苏晓谈判（十卷23–24章）**：苏晓杀了 阴兽 [水蛭](水蛭.md)、擒了 [病犬](病犬.md)，他出面谈判——本想用「填补阴兽空缺」把苏晓收为部下、再慢慢控制，被苏晓拿病犬性命威胁、直接回绝（只肯「合作」、且帮做事须付报酬）。他最先看清苏晓是「**瘟神**」（说翻脸就翻脸、比黑帮更极端），力主「破财免灾」雇佣苏晓——最终十老头**集体出资**雇苏晓护场到拍卖会结束、报酬定为**每天 5 颗「永恒晶石」（即灵魂结晶〔中〕）**、恩怨一笔勾销。他还**把女儿黑玛丽安插到苏晓身旁**当助手（够狠）。
- **拍卖会的隐患**：他几天前收到密报「**幻影旅团（蜘蛛）盯上了这次拍卖会**」——这正是他愿高价雇苏晓（疑为念能力者）护场的真正动机（蜘蛛真来则是天赐良机、不来就当破财免灾）。
