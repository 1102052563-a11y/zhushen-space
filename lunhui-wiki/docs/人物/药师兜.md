# 药师兜

[火影忍者世界](../世界/任务世界/火影忍者.md) [大蛇丸](大蛇丸.md) 的得力部下——戴圆框近视镜、灰发扎起、脸带若有若无的假笑，**双面间谍**、擅医疗。

- **此时尚弱**：这一时期的兜实力不算强、甚至不是 [卡卡西](卡卡西.md) 的对手（后期得大蛇丸血液 + 传承蛇仙人之力才真正变强）。其药物维系着大蛇丸被三代封印的手臂，故对大蛇丸极重要。
- **与苏晓的交集（九卷47–50章）**：他在天地桥蹲守、误以为 [苏晓](苏晓.md) 是代 [蝎](蝎.md) 来赴约（晓曾与大蛇丸有「咒术交换」之类往来）。苏晓厌其假笑、一道刀芒逼他现出大蛇丸；交易中苏晓**以界断线 + 炼金炸弹白蛇挟持他为人质**（先割其一耳、大蛇丸竟投苦无想灭口）。后被苏晓拿来试炼金秘药外用效果（耳伤瞬愈、他却硬撕血痂保留断耳能接合的可能）。大蛇丸翻脸时他早早溜走。

## 出处
