# 青丈（竹村族长 · 刀术强者 · 藏秘密的老人）

须发皆白、暗紫宽袖长衣（袖口六道圆形绣刺）、皱纹里透严厉的竹村族长——**极强的刀术强者**（[苏晓](../苏晓.md) 判断他对上上个世界的 [佩尼尔姐妹](../炎灵星群像.md) 胜率六成以上）。

- 不显敌意、也不排外；让苏晓「五天后再走」（无月之夜有怪事）。孙儿 **小青石**（10 岁能一刀劈断木桩·父母青火 / 竹鬼已故）。
- **藏秘密**：家中供台后有暗门通地下——地下藏着**他女儿青火 / 女婿竹鬼当年培育「月树」失败**的遗迹（想彻底解决源的侵蚀与血脉传承·[枯萎的月树种子](../../力量体系/装备道具/index.md)）。无月夜里他在地下黑暗中与苏晓刀战（刀术相当·怕塌洞适可而止）。竹村传承**天巴族**文明（村民衣着 / 灵魂结晶挂饰）。
- **战死（三十六卷9章·保卫战）**：蓝皮天巴族袭村，他斩杀 62 名天巴 + **前代天巴首领**（斧心龙心的巨斧持有者）后，被最后一击轰穿心脏——盘坐平静赴死（「终究是老了」）。遗言托 [苏晓](../苏晓.md)**带 [琉](琉.md) 去塔林王国找源之水**（琉是塔林王族·「别利用她·她封住的东西很可怕」）、赠月树根木盒（装【源】防天巴感知）+ 源特性刀「[风逝](../../力量体系/装备道具/index.md)」（作琉王族身份的凭证）。

## 关系

- [苏晓](../苏晓.md)：暗中与他刀战、互相试探的旅人。
- [琉](琉.md)：带苏晓来竹村、被他称「大人」的白衣少年。
