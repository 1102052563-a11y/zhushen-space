# 琉 / 艾德里·贝芙（塔林王族 · 被青丈养大 · 封着「可怕之物」）

宽松白衣、放牛戏耍却谈吐清澈有礼的「少年」（实为 17 岁·女·男装）——把来路不明的 [苏晓](../苏晓.md) 带来竹村（对村民有信心）、自称「我很强的」。

- **真身 = 艾德里·贝芙**：塔林王国**艾德里王族**（家徽古龙·房里藏 2 米壮汉古龙印徽铠甲）——幼年被 [青丈](青丈.md) 带到竹村养大（族长称她「大人」却用「你」）、极厌恶塔林的血腥王位（「富丽堂皇的炼狱」·不愿用本名）。青丈战死后遗言托 [苏晓](../苏晓.md) 带她回塔林找源之水。
- **封着「可怕之物」= 王血的「种子」（三十六卷21–22章揭示）**：青丈临终警告「别利用琉·她封住的东西很可怕」——真相是她是**王血的种子**（不能继承王血·王血断绝时她体内封的东西会觉醒、生出一丝新王血给下一代王）；把她变成「**初王**」是两国强者必追杀的禁忌，故她被藏在与艾德里世代同盟的「青之一族」竹村由 [青丈](青丈.md) 守护。晕船、路痴（把北当西）、单纯。

- **进神乡 · 女装暴露 · 被擒（三十六卷27–33章）**：她是**开神乡之门的关键**（交巨人指环给 [巨人王](巨人王.md) 确认艾德里血脉）；进神乡后男装身份被戳穿（头发变长、声音女性化、大心脏 + 恐高 + 路痴），干脆认了。天巴族不杀她只想抓她（艾德里王族 = 天巴开门恢复自由的机会）——她主动当诱饵嘲讽引敌，最终被 [沁之首领](沁之首领.md) 一弹指打飞昏倒擒走。

- **继承火【源】· 成「源之女」（三十六卷47章·世界结局）**：斩初王后，苏晓把火特性【源】交给她——作为与初王血脉最相近的**王血种子**，她一口吞下、成了**第二任守源人「源之女」**（对闯入源·神乡的七阶契约者而言，她 = 七阶大 boss，与八阶的 [巨人王](巨人王.md) 双镇神乡）。她坐在深渊旁守着熟睡的巨人王，留在了这个世界。

## 关系

- [苏晓](../苏晓.md)：带她回塔林、青丈托付的护送对象（苏晓要找源之水）；最终把火【源】交给她的人。
- [初王·艾德里·德温](初王·艾德里·德温.md)：她继承的火【源】的前任守源人（血脉最相近）。
- [青丈](青丈.md)：养大她、为她战死的族长。
- [沃奇夫](沃奇夫.md)：护送她回塔林的莽夫。
