# 巨人王（远古巨人 · 艾德里王族盟友 · 神乡之门守卫）

30 米高、镶黑金属甲片、提十几米长弯曲大刀的**远古巨人**——战力**远超本世界极限**（使徒之眼侦测失败）。

- **艾德里王族的盟友**：远古巨人族与艾德里王族曾结盟对抗古老神灵（贝妮在古遗迹石碑查到）。族人死绝、万念俱灰后，他守在迷雾沼泽尽头的**神乡之门**、践行与艾德里王族的承诺（脚下踩着族人遗骸·「巨人和你们是朋友」）。
- **开神乡之门**：[琉](琉.md) 交出**巨人指环**（[王冠](../../力量体系/装备道具.md) 的真身）确认艾德里血脉后，他用龙心 + 指环转动锁盘（王剑对太阳 / 天巴族徽对晶石 / 巨人盔对天之门）、硬顶落雷推开 50 米金属巨门——开完门指环化碎片飞回王都。他也进神乡（守门已无意义、要亲自清除神乡里的东西）、终获自由。苏晓借此得 [艾德里·王者套装](../../力量体系/装备道具.md) + 巨人之友 ★★★★ 称号。
- **老友重逢 · 开火源封印 · 去沉睡（三十六卷41–42章）**：布布汪把他招来、和苏晓成了朋友（烤巨蟒、痛饮巴哈递的烈酒·嗅觉退化、关节生锈、太孤单——整个巨人族只剩他自己）。为助苏晓冲 S+，他吞下**沁之魂**（暂熄体内燃烧之血）、用琉的血**开启火特性源之晶石的封印**（黑红太阳注岩浆填平深渊、化作「熔山」）——使命至此了结，他去寻一场长达数月的沉睡（靠沉睡自愈神陨之战的旧伤）。苏晓离开后，他与 [琉](琉.md) 一起留守神乡——对任何闯进源·神乡的契约者而言，熟睡的他 = **八阶大 boss**（连八阶契约者都会被他捶到怀疑人生），与七阶的源之女琉双镇此地。

## 关系
