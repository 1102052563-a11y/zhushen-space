# 老国王·艾德里·卢卡斯（塔林之王 · 淡漠 · 殁）

头发花白、身材偏瘦、对一切都淡漠的塔林国王——725 王后 / 267 嫡系，把血腥的王位厮杀视作对艾德里王族的试炼、既放任又鼓励。

- **偏向大王子**：因守塔人的特殊性只能暗中偏帮最满意的 [大王子·沃尔什](三王子.md)——故成了苏晓 + [二王子·泰德](二王子·泰德.md) 眼中「**最大隐患**」。厌弃泰德（骂其早产「失败品」）；见 [青丈](青丈.md) 的刀便知「天巴开始收回源之力」、下令午后处死苏晓。
- **结局（殁）**：苏晓与泰德「决裂」被举报下狱后，靠布布汪定位、**空间传送到老国王卧室一刀斩杀**（世界之源冲到 79.45%·王血 4/12）——老国王临死竟含笑「多谢你」（完成使命、解脱）。苏晓随即用 [先古面具](../../力量体系/装备道具.md) 伪装成他。

## 关系

- [苏晓](../苏晓.md)：判他死刑、反被其斩杀 / 顶替的人。
- [二王子·泰德](二王子·泰德.md)：被他厌弃的「失败品」儿子。
