# 二王子·泰德（贫民窟的银狐 · 怕死老阴哔 · 与苏晓合作）

真名 **艾德里·泰德**——银甲、7 岁被赶出王宫、在贫民窟长大的王子，人称「银狐」。爱吃（每天五顿·怕被王血吸干生命力·随身带肉干）、话痨爱跑题、怕死到病态，却是个战力不俗的**老阴哔**。

- **诅咒之血**：王血在吸他生命力（左臂已干枯·不成王活不过 40·成王得所有王血则能活 200 年）——这是他拼命夺位的唯一动机。
- **设局与合作**：先设局把 [苏晓](../苏晓.md) 与**四王女**引到一起对撞（想让二者两败俱伤·让礼帽男替死）；败露后干脆**与苏晓合作**（王血归他 / 王冠归苏晓 / 源之水随便拿），供出王血 / 王冠 / [琉是种子](琉.md) / **初王禁忌**等秘辛，带苏晓斩其生擒的五王女。他恩师正是 [青丈](青丈.md)（亦师亦父）。
- **决裂计**：与苏晓假装「分赃不均决裂」+ 假打一场 → 举报苏晓引出守塔人，好合谋借老国王除掉大王子 / 蜘蛛夫人。
- **加冕新王（三十六卷25–26章）**：苏晓假冒老国王主持冕王、在戴王冠的大王子头上一刀斩杀之，转而让泰德**加冕为塔林新王**（王血归泰德·集齐所有王血）。泰德没拼命除掉这可怕盟友（收益不对等），把 [王冠](../../力量体系/装备道具/index.md) 让给苏晓、供出源·神乡在科多艾森；却也留一手（暗告守塔人首领·若苏晓下令即处死他）——苏晓故以炼金剧毒逼他吐出圣守国纯净**源之水**、拿解药换。互相提防的「勉强算朋友」。

- **世界离场定位（三十六卷47章）**：苏晓通关后，泰德作为塔林新王留在世界外层——对之后进入源之世界（但没深入神乡）的契约者而言，他 = 能**燃烧「死之特性」王血**的**外层最强 boss**。
