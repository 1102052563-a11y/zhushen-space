# 溺之首领·猎潮（神乡天巴族溺特性首领 · 殁）

蓝袍、背源弓、容貌出众的女性天巴族——神乡天巴族**两首领之一（溺 / 远程特性）**；地位与战力都在 [沁之首领·尼罗](沁之首领.md) 之下（溺首领只能用源弓、沁首领可选两种武器）。

- **「耿直陷阱」擒敌**：她临时提出诱捕 [苏晓](../苏晓.md) 的计划（获全族支持）——故意让两名天巴演戏「传源」、透露 [阿姆](../../力量体系/装备道具.md) 关押处，把混进天之宫的苏晓骗进百余天巴 + 5 精英的包围绝境。可她不知苏晓怎么收拾老国王、大王子的（在苏晓看来这陷阱「耿直、演技一般」）。
- **三枪暴毙（殁）**：苏晓放出 **[死寂降临](../../力量体系/技能.md)**（600 米死寂同化）+ **死寂烬灭**枪反杀——她刚拉开源弓就被一枪断臂、一枪腐朽头颅、一枪穿心，「三枪，只是三枪而已」这大 boss 就死透。死后得圣灵级宝箱 + **溺之魂**（用后 +10% 世界之源 / 可打造源特性远程武器）。

## 关系

- [苏晓](../苏晓.md)：设局擒他、反被死寂烬灭三枪射杀。
- [沁之首领·尼罗](沁之首领.md)：另一位、更强的天巴首领。

## 出处

- 三十六卷（[源之世界·源神乡](../../世界/任务世界/源之世界-源神乡.md)）36–37 章：耿直陷阱→被死寂烬灭三枪射杀。
