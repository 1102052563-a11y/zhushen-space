# 盲眼老人（树之契约商人 · 巴之魂兑换商店）

神乡荒野小木屋里、双目惨白、干瘦到只剩骨头、介于生死之间的老头——**树之契约（与虚空之树）的签订者**：留在本世界、以特定货币换取世界外的物品。

- **巴之魂兑换商店**：只收「**巴之魂**」（天巴的灵魂 + 源之力结合物）——47 种货品含神灵骨（史诗 / 圣灵）/ 神灵能量凝结体 / 弓箭宗师卷轴 / **秘技·天之流**（天巴刀术秘技·450 巴之魂）。[苏晓](../苏晓.md) 兑神灵能量凝结体喂 [神裁](../../力量体系/装备道具/index.md) 到 62% + 史诗神灵骨。
- **绝不可攻击**：树之契约签订者只要不被主动攻击就不能出手，一旦被打则爆锤（站起来是巨人王梯队的战力）；与天巴两首领约定「木屋内不许动武」（是安全避难所）。

## 关系

- [苏晓](../苏晓.md)：用巴之魂在他这兑换物资的客人。
- 天巴族：也与他交易、护他木屋中立的族群。
