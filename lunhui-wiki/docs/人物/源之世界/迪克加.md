# 迪克加（塔林税务官 · 贪婪领路人 · 下错注的倒霉蛋）

四十岁不到、油光满面、贪婪又狡诈的塔林税务统计官（管收税·油水足却无实权）——被 [苏晓](../苏晓.md) 威逼利诱拉进王权厮杀（黄金 + 点破他「一直没走是没想好怎么捡地上的 50 金镑」）。

- **领路人**：苏晓料定他不会真忠于自己、而会拿黄金打通渠道**转投某王位继承人**、把苏晓的情报卖过去——正好替暗中跟踪的布布汪 / 巴哈**带路**。
- **下错注**：果然投靠 [三王子](三王子.md)（断他一指作考验、三王子曾下令把他喂狗）——却因苏晓**当晚就斩了三王子**，[阿姆](../阿姆.md) 一巴掌把躲在仓房的他打晕扛回（「你们这么猛怎么不早说」）。后又被二王子那边抓走审问、又被苏晓小队保下（「这么称职的领路人不好找」）。在王族厮杀里两头挨的倒霉蛋。

## 关系

- [苏晓](../苏晓.md)：把他当「领路人」、又两次保下他的人。
- [三王子](三王子.md)：他舍命投靠、却当晚就被苏晓斩了的新主子。

## 出处

- 三十六卷（[源之世界·源神乡](../../世界/任务世界/源之世界-源神乡.md)）15–17 章：被苏晓收买→投三王子→被喂狗。
