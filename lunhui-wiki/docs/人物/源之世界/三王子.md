# 三王子（诅咒之血 · 阴柔权谋 · 6 王位继承人之一）

脸色偏白、阴柔含笑、心思深沉的塔林王子——**6 名幸存王位继承人之一**、[苏晓](../苏晓.md) 猎王的目标。

- **背叛与代价**：曾背叛过愿为他死的亲密兄长——**挖出自己左眼**（假眼）与兄长陪葬；信条「背叛要付出代价、包括我在内」。诅咒之血在身、喜怒无常。
- **手段**：能操控替身（本体藏城堡地下）、养重度烧伤的女杀手 **暗女·多明尼卡**、后院养双头犬。收 [迪克加](迪克加.md) 当副从（断指考验）。
- **结局（殁）**：苏晓斩穿 50 黑甲护卫（魔甲护卫·得掠夺者称号）杀到地下本体——因苏晓**完全免疫精神控制、天克其精神系能力**，被一刀斩喉（真名 **艾德里·佩格**·苏晓得世界之源 15.2% + 王血 1/12）。城堡被炸。暗女被 [阿姆](../阿姆.md) + [沃奇夫](沃奇夫.md) 所杀。

## 关系

- [苏晓](../苏晓.md)：杀上门夺其王血 / 世界之源的猎手。
- [迪克加](迪克加.md)：投靠他、又被他喂狗的税务官。
