# 初王·艾德里·德温（守源人 · 神乡最强者 · 王血源头）

3 米多高、全身破朽铠甲已如皮肤、持一把腐朽缺尖大剑、头甲下双眼只有暗红火焰的巨汉——**守护火特性源之晶石的「守源人」、天巴族公认的「神乡最强者」**。

- **「初王」的真相**：他**从未坐过王座、甚至没去过塔林**——「初王」只是塔林王族一厢情愿的尊称（王都最高的雕塑就是他）。他本是**卑微之人**，自决定守源那一刻起便不再卑微；塔林近一亿人几百年的意念凝成信仰之力反馈到他身上，让他成了无冕之王。**艾德里王族一代代残酷相传的「王血」= 他与源融合的血**（唯最强者才配传承）。历代天巴强者（初代天巴、九代青沁、沁之首领·尼罗…）都来熔山挑战过他、皆败——四块留言石铭刻着从「卑微」到「伟大」的敬意。
- **战力（约 58% 资料）**：守源人（力 182 / 敏 170 / 体 180 / 智 157·源之力 92900·无法交涉）——**守源人**（+87000 生命·免控·火 / 斩 / 钝伤 −40%）+ **初王**（+32000 生命·防 +32.7%·精神反噬）+ **艾德里疯剑**（剑攻 +712%·不可中断）+ **死亡抓握**（冲撞眩晕、抓颈钉地炎爆）+ **源·焚爆**（生命 50/20/10/3% 各触发一次·430 米范围）+ 源之觉醒（终极·50% 以下）。半技法型 + 超强生存力 + 源之力大范围攻击，[苏晓](../苏晓.md) 评比大主教更强。
- **决战 · 险胜被斩（殁）**：初王「**源之觉醒**」后直起弓背、大剑燃「源之火」、生命值每滑到 50/20/10/3% 触发一次四段「源焚爆」自愈——是苏晓历来遇到**生存力 / 防御力最强的敌人**。苏晓靠**五灭法之刃融合攀刀**（[青影王](../../力量体系/技能.md)·单刀 9750 真伤 + 高概率麻痹）+ 血之兽·狩枪破膛 + [阿姆](../../力量体系/装备道具.md) 一斧劈断其持剑臂 + 「刃道刀·魔刃镰」割首**险胜斩杀**（苏晓一度被斩断左臂左小腿·得世界之源 27.3% → 145.15% + 圣灵·无冕宝箱 → **[王之魂](../../力量体系/装备道具.md)**）。
- **一生真相（灵魂记忆·守源人 · 完整）**：他本是战场杀 16 战士被俘、被烙满铠甲当仆从卖的「**笼中野兽**」——一生只两人待他有善意：赠名「德温」并放走他的**老铁匠**、善待他的神灵后裔少女**艾德里·希耶儿**（偷跑出神乡·被其兄「人族都该死」连累）。希耶儿死后，笼中野兽背起她尸体、胸嵌【源】、屠尽 12575 名入侵者成守源人（姓取「艾德里」+ 名取「德温」）。熔山**等了 447 年**继承人、感将「崩朽」——直到等来苏晓，「他可以休息了」。苏晓观其记忆得 +8 意志力、取走火【源】，尸身化灰。

## 关系

- [苏晓](../苏晓.md)：为冲 S+ 评价、开火源封印挑战他的猎杀者。
- 天巴族（[沁之首领·尼罗](沁之首领.md) 等）：世代来熔山挑战、公认他为神乡最强者。
- 塔林艾德里王族：以他的血为「王血」、尊他为初王的后世王朝。
