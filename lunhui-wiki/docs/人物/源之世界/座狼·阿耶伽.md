# 座狼·阿耶伽（最后一位古老神灵 · 守门源犬 · 殁）

6 米灰毛、金属利爪、蓝竖瞳的**古老神灵·座狼**——**本世界最后一位古老神灵**。背上插三根装浅蓝液体的玻璃柱（曾被囚·中位神灵 + 沁能力·力 165 / 敏 167 / 体 153）。

- **天巴族的失败品**：天巴族屠光古老神灵后，想拿它做「源 + 神灵能量结合」实验、失败（源化更猛）——被扔去地之宫看守通往外部的出口（沦为「守门源犬」）。
- **主动求死（殁）**：因神灵的骄傲不愿当敌人的看门狗、也压不住源化，**主动压制源化让 [苏晓](../苏晓.md) 一刀斩杀**（斩头后还自我对冲生命力生怕不死·世界之源 17.3%）。死前只关心「塔林 / 圣守还在吗」（更憎恨把它变成这样的天巴族）、遗愿托栽「**月树之种**」（触发隐藏任务·奖狼魂石圣灵宝石）——[神裁](../../力量体系/装备道具.md) 因它是极恶神灵而成长。

## 关系

- [苏晓](../苏晓.md)：它主动求死、成全它的人。
- 天巴族：把它变成失败品守门犬的仇敌（详见 [沁之首领](沁之首领.md)）。

## 出处

- 三十六卷（[源之世界·源神乡](../../世界/任务世界/源之世界-源神乡.md)）29–30 章：地之宫守门→主动求死→遗愿月树之种。
