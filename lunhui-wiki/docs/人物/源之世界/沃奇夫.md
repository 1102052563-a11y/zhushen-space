# 沃奇夫（外乡莽夫 · 井下战神 · 龙心斧持有者 · 殁）

戴破脚盔、腰挂两把短柄斧、打扮与竹村迥异的外乡人——竹村里被小孩捉弄也一言不发的沉默憨憨，实为**塔林（西塔）派来护送 [琉](琉.md) 回国的人**。

- **井下战神**：血脉能力让他战斗时感觉不到痛（血似沸腾）——躲进枯井把追来的天巴族劈得怪叫连连（狭窄空间里没人能胜他·「能动手绝不废话」）。
- **龙心斧**：背着天巴族至宝之一 **龙心**（斧心嵌一颗缩小的古龙心脏·真劈杀过古龙·把「沁」发挥到极限）——守塔人见此斧即认他 / 琉的身份。脖颈血管渐化蓝（有天巴血脉）。
- **追求成为「天巴」（三十六卷27章）**：他其实没家人、一心想借龙心变强、成为「**猎龙天巴**」——坦言若败给龙心的侵蚀就让苏晓杀他（「别让我这满嘴谎言的人拖累任何人」）。护送琉的使命完成后随队进神乡。
- **成为第一个「炎天巴」· 永眠熔山（三十六卷42章·殁）**：进神乡后独自追逐力量、最终未能驾驭龙心的侵蚀——却在熔山之巅**斩杀了黑眼女人**（用龙心斧劈死她）、两种源之力共存于体、成了**史上第一个「炎天巴」**（皮肤化暗红）。濒死之际把最后的**火特性源之力**加持给苏晓（1 小时火伤 −20%）、坦言「我这满嘴谎言的人也能成为天巴……已经没有遗憾」、永眠于「属于奇夫的永眠之地」（[阿姆](../../力量体系/装备道具/index.md) 将他冰封熔山、接管了龙心斧）。

## 关系

- [苏晓](../苏晓.md)：同行护送琉的强者（提供枯井外的战力）。
