# 沁之首领·尼罗（最强天巴 · 神乡天巴族沁特性首领 · 殁）

3 米高、蓝袍、鬓角扎细辫、持近 1.5 米长刀（刀上洗不掉血印）的天巴族——神乡天巴族**两首领之一（沁 / 近战特性）**、**当代最强天巴**（[苏晓](../苏晓.md) 评比 [青丈](青丈.md) 更强·天巴战力排序 寻常 < 精英 < 溺之首领 < 沁之首领·沁首领可选龙心斧或沁心刃两种武器）。

- **初遇擒琉**：苏晓刚斩尽 14 名巅峰期宗师级天巴族、就被他找上——一刀斩开巴哈胸腹（比族长狠）、带 6 精英 + 几百天巴族。他要抓 [琉](琉.md)（艾德里王族 = 天巴开神乡门恢复自由的机会·故不杀只抓）、留不下苏晓——苏晓带巴哈暂撤，琉被他一弹指打飞擒走。
- **抱源焚身 · 被斩（殁）**：天之宫被苏晓一颗 [阿波罗](../../力量体系/装备道具.md) 炸没时，他搂着【源】、被众多同族拼死护住只烧了很短时间、仍重伤（生命值 47.2%）。他**扯下衣袍、舍弃了自己「身为人的最后一颗心脏」**、让源沁溺其中彻底变怪物（源侵蚀等级暴涨 Lv.9）——用自创刀术「**巴之流·沁**」（对 [秘技·天之流](../../力量体系/装备道具.md) 的二次开发）与苏晓对拼，仍被苏晓斩下头颅。死后得**沁之魂**（交巨人王可暂熄其燃烧之血、开火源封印·概率达 S+ 评价）。

## 关系

- [座狼·阿耶伽](座狼·阿耶伽.md)：天巴族屠光的古老神灵之一。
