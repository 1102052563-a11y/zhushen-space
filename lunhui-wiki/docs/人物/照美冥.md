# 照美冥

[火影忍者世界](../世界/任务世界/火影忍者.md) **雾隐之村「主和派」的首领**——棕色长发、身材丰满、成熟端庄的女忍者，**实力强大、掌握两种血继限界（溶遁·沸遁）**，不久后将成为**五代水影**（眼下已有水影之权、只差名分）。

- **两面性**：**对敌人**——够狠、理性、实力强（绝不受个人善恶情绪左右、一切以忍村安危为先）；**对部下**——成熟温柔友善、偶尔发小脾气。
- **政治困境**：四代水影 **枸橘矢仓**（三尾人柱力）死后，雾忍村分裂为**主和派**（照美冥·主张停内乱 + 对外通商·因水之国地处海岛、又穷又封闭）与**主战派**（占七成忍者·主张停内战转而劫掠他国·无顶尖首领·四代曾是其首）。两派都想停内战、却为水影之位寸步不让。照美冥**顶尖战力强**（自身 + [青](青.md) / [长十郎](长十郎.md) 等辅佐），却**缺足够忍者支持**、又不能杀光主战派（否则雾忍空前虚弱）——故才想利用 [苏晓](苏晓.md) 这种「外人武力」清除异己。

## 与苏晓的交易（九卷4–6章）
苏晓的乐园身份正是「刺杀四代水影失败的刺客」，故照美冥对他**高度警惕**（四代尸上的刀伤为证·一见面 [青](青.md) 就察觉苏晓的杀意）。她**不愿付一分钱报酬**（雾忍太穷·秘术 / 忍术 / 查克拉传导金属皆不外传），只压价成「**帮我杀三个（主战派上忍头目），事成放你安全离开雾忍村**」——明面合作、**实则全程监视、做完即灭口**（她绝不信一个杀手）。苏晓将计就计、暂时答应，却**一件都不打算做**（识破卸磨杀驴·「受威胁就玩命」是他的作风）。

## 识破 / 反应（九卷6–7章）
苏晓白天狂吃雾忍特产水果、把炼金炸弹藏进咬孔的果核里**躲过青的白眼透视**，当晚假装去刺杀目标、实则**沿路布满炼金炸弹**。照美冥从「水果 / 果核不对」**察觉异常、赶来拦截**——一招「**溶遁·溶怪之术**」喷出铺天盖地的岩浆，可惜**慢了太多**（她事后懊悔「最好的选择是把他烧死在牢里、根本不该放他出来」）。苏晓引爆全村炼金炸弹（**雾忍村十分之一房屋被炸 / 引燃**），她遂登高指挥忍者一致对外、追杀苏晓。

## 一石二鸟 · 失青之痛（九卷7–9章）
她放苏晓出来本就是**借刀一石二鸟**——既让苏晓的炸村「外敌」之祸**逼主战派放下内斗、一致对外（露出破绽）**，又顺势清理了「晓」组织。她与苏晓「互相算计、目标却不冲突」（他要离开、她要打破内战僵局上位）。命 [青](青.md) 处理掉「已没用」的苏晓——

- **痛失青**：苏晓反杀了青（详见 [青](青.md)）。她**面上平静、袖中拳头紧握、心如滴血**（青是她最得力、最忠诚的助手·侦查全村顶尖），将青的白眼与遗体**厚葬、以英雄家属待遇安置其家人**。
- **借势上位**：她把「村周安全」职务交给主战派的 [启介长老](启介长老.md)（以「外敌当前内部矛盾先放下」为名·实为让主战派紧张露破绽）——若顺利，「**不超半个月她就是新一任水影**」。
- **三度躲过狙杀**：苏晓打回马枪、在 1.4 公里外用蜘蛛女皇**三枪狙杀她**（玻璃窗碎、办公桌near破），均被她侧身跃开躲过（她绝非动漫里那点性格的「冰山一角」·苏晓评她是「足以担负水影之位的女人」）。她随后把苏晓资料**公布给其他四大国**、标注「**有能力刺杀影级**」，派暗部穷追。

## 正式上位 · 五代水影（九卷11章）
苏晓炸村的「外敌之祸」加快了她的上位——雾忍把**四代水影之死归罪于苏晓**（对外宣称：四代受苏晓刺杀身负刀伤、后为保护村子与外敌作战「刀伤复发阵亡」），「一个忍村不能没有影」，**照美冥遂正式成为五代水影**，雾忍村内战很快平息、向好发展。这也间接印证了 [鬼灯残月](鬼灯残月.md) 当初「她现在不是、以后也不可能是五代水影」的断言其实落空（或别有所指·伏笔）。
