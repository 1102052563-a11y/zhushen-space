# 摆烂少女·德洛娜

> ⚠ **[大贵族·卢西瓦](大贵族·卢西瓦.md) 的妹妹**——⚠ **一副生无可恋、被迫营业的神情:「你们好，我是心甘情愿来帮忙的，请你们随便使唤……我兄长没强迫我来，真的。」**

## 😑 摆烂光环

⚠ **她的能力没有具体名称，称其为「摆烂光环 / 消极光环」也可以**——⚠ **激活后，她周边范围内所有被她敌对的目标，都会陷入重度的消极与迷茫**;⚠ **⚠ 这能力与生俱来，无法通过锻炼提升，只会随她长大而增强。**

- ⚠ **实战**:⚠ **地下监狱的守卫队长撕毁奥古斯的批文、挡住去路**——⚠ **她叹口气、自暴自弃地坐在地上摆烂**;⚠ **⚠ 几秒后，半数守卫也席地而坐唉声叹气，其中一名前不久刚分手的年轻人更是单手掩面而泣**——⚠ **「走吧，他们都不想理我们了。」**;⚠ **⚠ 一路走过牢房，沿途死囚犯都开始用床单上吊。**

## 👁 邪祟转生的传说

⚠ **上一任游猎团军团长（她与卢西瓦的父亲）前半生都在探索无光区、接触太多黑暗与深渊**——⚠ **某次身受重伤后传位给长子卢西瓦**;⚠ **⚠ 关键在于:这位军团长回黄昏城没多久，他那被灵魂学院首席医师诊断为「没有生育能力」的第三任妻子，竟有了身孕**（⚠ 一切正常反而让全家暗暗担心·⚠ 德洛娜出生的消息在几年后才很低调地公布）;⚠ **⚠ 有个说法是:老军团长在无光区带回来了某种邪祟，德洛娜就是这邪祟的转生**——⚠ **⚠ 整个家族都对她很忌惮，唯独兄长卢西瓦会强行把她拎出庄园让她逛逛。**

- ⚠ **⚠ 苏晓的旁观**:⚠ **他从封印区带着眼中蓝芒走出时，德洛娜的双眼瞬间化为漆黑，像是隐藏在体内的力量即将被唤醒**——⚠ **可片刻后眼底的漆黑快速褪去**（⚠ 苏晓并未深究她的秘密）。

## 🖤 黑暗觉醒的真相（53卷16章）

⚠ **⚠ 苏晓向 [卢西瓦](大贵族·卢西瓦.md) 揭开了她的秘密**:⚠ **她父亲（上一任游猎团军团长）当初在无光区不是被深渊能量侵蚀，是被**少量深渊源质**侵蚀**——⚠ **深渊源质会把灵魂同化成坐标，让深渊内的诡奇存在借此转移、寄生**;⚠ **⚠ 而那未知诡奇「更中意你父亲的下一代因果，也就是还在母胎中的德洛娜——这种能融入物质世界的机会，它不会错过：它没寄生，是**和德洛娜同化了**」**;⚠ **⚠ 「她的确是你妹妹，这点毋庸置疑，那属于诡奇的一部分，被人剥离封印，但随着她被动的黑暗觉醒，那部分很快会冲破封印，反客为主。」**

- ⚠ **⚠ 无法阻止，是必然**——⚠ **但苏晓能让她成为赢家:她与那未知诡奇可以互相吞噬**（⚠ **想成为赢家总要付出代价：或卢西瓦付，或拼她命格是否够硬，大概千万分之一获胜概率**·⚠ 苏晓开价先是一份力量源石、后改原初碎片，卢西瓦付了 1 颗原初之核＋20 块原初碎片）;⚠ **⚠ 每次苏晓带蓝芒出现，她双眼都会瞬间化漆黑——那隐藏在体内的力量即将被唤醒**。
