# 永夜女皇（永暗女皇）

> ⚠ **游猎团在无光区遭遇的未知存在**——⚠ **[卢西瓦](大贵族·卢西瓦.md) 的第三大队一瞬间全灭、连死者的七大姑八大姨等远亲都在同一刻化为血雾、幸存者还会逐渐遗忘死者**;⚠ **⚠ 这让常年探索无光区、极限承受「侵袭级」的游猎团，判断出这是他们对付不了的「直面级」深渊，果断封闭天壁入口。**

## 🖤 一个渣男的报应 —— 格林·吉莉安的旧爱

⚠ **她原本是人族，是先代女灭法 [格林·吉莉安](../格林·吉莉安.md) 第 129 个深爱着的人**——⚠ **也是把渣过无数妹子的格林·吉莉安反过来渣得戴上痛苦面具的人**;⚠ **⚠ 她堕入深渊，并非受格林·吉莉安刺激，而是她老家所在的世界被 [星界吞噬者](星界吞噬者·新生代.md) 吃掉、亲人一个不剩**（⚠ 格林·吉莉安不止一次劝阻，可她说「谁说我是渴望力量才堕入深渊？我只是更喜欢黑暗的静谧」，最终仰躺入深渊大漩涡）;⚠ **⚠ 堕入深渊后意识一分为二，本体意识败给深渊滋生意识，她成了彻彻底底的深渊族裔**（⚠ 最后关头听到格林·吉莉安「你想想，你还有我啊」的挂念，寒若坚冰的脸上浮现过些许笑容）。

- ⚠ **⚠ 格林·吉莉安这次以残魂来烈阳星，就是要在封印撑不住前，主动打开封印、把她放逐回深渊**（⚠ 与其等她冲破封印出现在无光区外，不如借太阳神族遗留的无光区结界再困她一段时间·⚠ 「作为灭法者，我必须这样做」）。

## ⚔ 概念性不死与诡异能力

⚠ **她的不死并非复生/转生/再生，而是「对她的攻击无效」**——⚠ **格林·吉莉安以「鲁珀特之泪」作比:攻击锤头面需千百倍强度才能击碎（=绝对不死），可找到弱点、轻轻掐断细尾就整颗爆裂**;⚠ **⚠ 使用能力时必须从黑暗虚无脱离、与敌人处于同一空间维度，那一瞬间就是她的破绽**（⚠ 刀术宗师斩魂+伍德侵袭+神父古神之暗都能伤到她）:

| 能力 | 说明 |
| --- | --- |
| **黑暗绞杀** | ⚠ 让受害者的灵魂、命运像拧麻花般被拧到极限直至「存在」破裂·⚠ **绞杀之力顺血脉命运线蔓延到死者所有亲系（都炸成血雾）、死者被逐渐遗忘**·⚠ 一次最多锁 5 个目标·⚠ 3~5 分钟冷却（⚠ 所以绝不能与她一对一） |
| **沁暗** | ⚠ 对视者的灵魂被吞入黑暗沁溺而死·⚠ 冷却长达几个自然日 |
| **红雾领域** | ⚠ 吞噬生命·⚠ **1 秒吸取苏晓 273 万血量的 75%**（⚠ 白金使徒在其中 1 秒死 265 次） |

## 🕸 封入原罪之书（53卷51-52章）

⚠ **苏晓的破局关键是「[原罪之书](../../力量体系/装备道具.md) 的被动护盾」**——⚠ **成功封印六件大爹级原罪物后，原罪之书衍生出六层特殊护盾，可豁免六次黑暗绞杀**（⚠ 这也是永夜女皇要先把苏晓传送到最东侧、需七次绞杀才能杀他的原因）;⚠ **⚠ 苏晓拉神父/白金使徒/黑魔入无光区，把「四分之一的倒霉概率」摊薄成「七分之一」**;⚠ **⚠ 红雾领域被苏晓召来的无穷「[工蝎](../../力量体系/装备道具.md)」（消耗生命力·冲了三个多小时）活活撑爆**;⚠ **⚠ 最终苏晓以放逐术式引她现身、刃之魔灵替换硬吃「沁暗」，把她拖入原罪之书第 8 页封印**（⚠ 放逐是计划 1、封印是后备计划·「对付这等诡异存在，不备三四种后备计划就太鲁莽」）;⚠ **⚠ 封印完成的瞬间，神父/白金使徒/黑魔的背刺齐至，被苏晓反手放逐到永光世界。**
