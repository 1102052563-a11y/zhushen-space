# 星界吞噬者·新生代

> ⚠ **注意，此非 [永光世界·无光神殿四巨头的星界吞噬者](../永光世界/星界吞噬者.md)**——⚠ **那是被苏晓宰掉的「上一任」**;⚠ **⚠ 这是从星辰巨树诞生没多久的新一代**（⚠ 形象如十岁左右小男孩·脸上手上遍布陶瓷般裂痕·刚诞生就从传承之力接收到「找灭法者报仇」的记忆）。

## 🌌 星界规则的一部分

⚠ **星界吞噬者是星界规则的一部分**——⚠ **星界内一些废落的星辰需要被吞噬，否则会阻碍新星辰诞生、还会腐朽滋生可怕之物**;⚠ **⚠ 每一代星界吞噬者都是不同个体、可善可恶**:

| 走向 | 结果 |
| --- | --- |
| **只吞噬废星/即将爆裂的星辰** | ⚠ 逐渐向**星辰之龙体系**转变（⚠ 获得可化身星辰之龙的变身类能力） |
| **沉迷星辰之力、吞噬健全星辰** | ⚠ 逐渐化为**灭世级存在** |

- ⚠ **⚠ 它初始就有「顶级至强」的上限级**——⚠ **哪怕什么都不做，每天也都在「恢复」力量（而非提升）**;⚠ **⚠ 能否达成「至强巅峰」则看每一代自身。**
- ⚠ **⚠ 善恶不定的证据**:⚠ **在灵冕城会笨拙地给流浪狗喂肉饼、还懂得付钱等找零**——⚠ **可在暗月噩梦，又把一伙探险者骗去玩「抛球游戏」全部杀光**（⚠ 谁躲过它抛的暗物质弹球就放谁走·结果无一幸免）。

## 🍶 被苏晓「控着愤怒值揍」（53卷40-41章）

⚠ **[特殊任务·星界规则](../../世界/任务世界/烈阳星·奇利亚德.md)（Lv.94）**:⚠ **让现今的星界吞噬者向星辰之龙体系演变，而非成为灭世级存在**（⚠ 奖励星界宝箱·⚠ 苏晓请战斗天使·泽丽娜代接、再由 [凯撒](../尼古拉斯·凯撒.md) 转移到自己任务列表）;⚠ **⚠ 苏晓的策略是「控着愤怒值揍」**:

- ⚠ **⚠ 先用四箱药剂拉好感度**（⚠ 黑暗药剂×24＋奇迹药剂×24＋暗之生命秘药×20＋星空药剂×22·成本低廉·把它好感度从 −50 拉到 68）——⚠ **它一边默念「不能被仇敌诱惑」一边目光越发清澈，主动给自己找台阶「我就当以前的事没发生过」**;⚠ **⚠ 之后再控着愤怒值揍它（别让愤怒值破 100 转仇恨值），让它体会成为灭世级的险恶，把它掰向星辰之龙体系。**

## 🩸 死城博弈的第一个受害者（53卷42章）

⚠ **面对 [众神化身](众神化身.md)，它认出杀死星辰之龙的仇敌、灵魂记忆中浮现第一纪元一位星辰之龙被扯成两截吞掉一半的画面**——⚠ **暴怒扑杀，一拳轰在众神化身面甲上连裂痕都没有，反被单手抓起、砸进地里、接连拍击**;⚠ **⚠ 恢复理智后带着颤音「对，对不起啊，我不是故意跳起来打你……」**——⚠ **被苏晓的「刃道刀·流」划出一道血痕才被随手丢在地上，直挺挺躺着涕泪横流**（⚠ 从星辰巨树诞生没多久·淌眼泪不丢人）;⚠ **⚠ 它是苏晓眼中的「星辰宝箱」——若能让它向星辰之龙体系发展，就能完成任务开出星界宝箱。**
