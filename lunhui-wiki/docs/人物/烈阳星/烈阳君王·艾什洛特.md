# 烈阳君王·艾什洛特

> ⚠ **⚠ 他活着，天空中那颗越发暗淡的太阳，才会继续存在。**——⚠ **从太阳与古龙战争结束后，[烈阳星](../../世界/任务世界/烈阳星·奇利亚德.md) 的维续全凭太阳神族一代代传承「烈阳之血」，否则那颗暗淡烈阳将陨落而下。**

## ☀ 传承「烈阳之血」的可怕代价

⚠ **传承会导致传承者的自身源血逐渐燃尽**——⚠ **太阳与古龙战争刚结束时，本世界的太阳神族还剩一族嫡系、上百族分支**;⚠ **⚠ 而现在，整个太阳神族只剩烈阳君王·艾什洛特一人**（⚠ **这就是传承「烈阳之血」的可怕代价**）;⚠ **⚠ 更狼狈的是:他的子嗣在血脉传承上都是「残次品」，没资格传承烈阳之血，有些甚至天生体态畸形。**

- ⚠ **⚠ 完美族裔的由来**:⚠ **放眼几个纪元，所有超脱世界的主角种族都有各自因对抗深渊、达成超脱所遗留的缺陷**（⚠ 兽族与海族的血脉诅咒、巫师的恶变……）——⚠ **唯独太阳神族没有这样的缺陷**;⚠ **⚠ 原因是他们的烈阳帮他们承受了超脱后的代价，亦如他们守卫这颗烈阳般**（⚠ 那些原本该由太阳神族承受的深渊能量侵蚀，被这颗烈阳全部吸收）;⚠ **⚠ 也因此，万界中所有敢自称「烈阳君王/太阳君王」的王者皆是伪王，因为没有资格**——⚠ **只有艾什洛特能称烈阳君王:他以自身承载烈阳之血让烈阳星稳定，也承先祖荣光。**

## 🏛 他的抉择与黄昏城的三派

⚠ **他基本不离开中心城区，甚至几年不露面一次**——⚠ **黄昏城平常由一位位大贵族组成的王城议会管理**;⚠ **⚠ 他只信任两个人:当代太阳战士首领，以及 [老贵族·奥古斯](老贵族·奥古斯.md)。**

- ⚠ **⚠ 他要抉择的问题**:⚠ **是否让外族传承「烈阳之血」**——⚠ **外族真有能力传承来自烈阳星鼎盛时期、从太阳上引下的传承圣物吗？就算有资格，他会允许祖先的荣光被扭曲神灵或灵魂学者亵渎吗？**;⚠ **⚠ 黄昏城的三派**:⚠ **旧日守护者（太阳战士组成·他的中坚力量·只要他不让外族传承就是最忠诚拥护者）· 大书库（几位贤者与太阳学者·认为本世界所有强者都有资格参与传承·某种程度是反对者） · 旧贵族（掌整个黄昏城六成财富·表面中立·实则全力支持君主·由奥古斯领衔）**;⚠ **⚠ 而真正的暗线是「他与大书库·[老怪物](../../世界/任务世界/烈阳星·奇利亚德.md)」的明争暗斗**（⚠ 君主阵营离不开大书库掌握的「野兽心脏」变强秘术核心，大书库又绝不敢公然挑衅——没君主阵营保卫黄昏城，大书库灭亡是早晚的事）。
- ⚠ **⚠ 成王之路**:⚠ **大书库估他再撑上百年已是极限，必须提早选出「烈阳之血」新一任传承者**——⚠ **由黄昏城、诸神教、灵魂学院各出一队人，去夺回散落各处、已被篡夺的一颗颗「[太阳源石](../../力量体系/称号.md)」，集齐者即为新王**;⚠ **⚠ 他默认此事，说明他的身体状态已远不如传言中乐观。**
