# 财政大臣·黑伯莎

> ⚠ **黄昏城真正实行君王职能的，是老贵族·奥古斯与财政大臣·黑伯莎**——⚠ **烈阳君王·艾什洛特与世隔绝、地位更接近黄昏城的神祇而非君王**;⚠ **⚠ 尤其是后起之秀黑伯莎，一生中只见过烈阳君王几面，甚至没被对方说过哪怕一句话。**

## ⚖ 激进派 —— 新晋权贵的领袖

⚠ **黄昏城·君主阵营的内部权力划分**:⚠ **王下三位——[老贵族·奥古斯](老贵族·奥古斯.md)（保守派·旧贵族利益）、财政大臣·黑伯莎（激进派·高层官员与新晋权贵/新晋家族利益）、太阳战士首领（根本不表态·统领旧日守卫者军团）**;⚠ **⚠ 黄昏城能撑到今天，至少有黑伯莎两成功劳**（⚠ 严谨的老夫人·初见皱纹）:

- ⚠ **⚠ 她对「太阳陨落」的第一反应是怀疑**——⚠ **在她看来，「烈阳之血失去传承者则太阳陨落」更像是太阳神族彻底衰败前的挣扎，以及旧贵族想借此从她手中夺权的手段**;⚠ **⚠ 她对太阳源石毫无兴趣**（⚠ 拥有太阳源石的强大者没一个有好下场）——⚠ **在她视角里，苏晓夺回越多太阳源石，越是帮旧贵族壮声势。**

## 🗡 围杀苏晓，与双双身死的结局（53卷36-37章）

⚠ **她亲自在现场指挥了围杀苏晓的阵容**——⚠ **近乎出动黄昏城五成力量（游猎团、城卫军精锐、宪兵团精锐、治安部门精英、旧贵族强者），等同撕破脸皮**;⚠ **⚠ 苏晓一句「我在暗月噩梦找到了把大剑——太阳王的烈阳大剑」，让围杀当场叫停**（⚠ [烈阳大剑](../../力量体系/装备道具.md) 进献烈阳君王后，旧贵族将稳压新晋权贵）:

- ⚠ **⚠ 苏晓的阳谋:让 [枭](../枭.md) 暗杀奥古斯、嫁祸黑伯莎**——⚠ **奥古斯遇刺、君主圣徽遗失，所有旧贵族都会咬定是黑伯莎干的，不惜一切弄死她**（⚠ 打破了双方互相威胁、互相平衡的关系）。
- ⚠ **⚠ 讽刺的结局**:⚠ **听闻奥古斯身死、君主圣徽遗失后，黑伯莎意识到旧贵族会疯狂到歇斯底里，当机立断要逃往曦光城**——⚠ **却在餐厅门口，被跟随了她几十年的心腹一刀刺穿心脏**;⚠ **⚠ 「奥古斯大人交代我，代他和您道别，永别了」**——⚠ **至死她都不敢置信，这心腹居然是奥古斯的人**（⚠ 比听到奥古斯身死时还错愕）。
