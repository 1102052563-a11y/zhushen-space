# 深渊次子·乌瓦琉托

> ⚠ **一只 [不死不灭·深渊滋生物](../../世界/任务世界/烈阳星·奇利亚德.md)**——⚠ **血夜降临时，它向灭法者 [苏晓](../苏晓.md) 报复而来，认定他为宿敌**（⚠ **本次敌对关系:本次血夜结束前必定与他分出生死**）;⚠ **⚠ 它召唤了整片大区域内的所有黑暗生物、暗之滋生物向苏晓围杀。**

## 🌑 钢铁意志试炼

⚠ **它的袭来触发了「钢铁意志试炼」（★★★★★难度）**——⚠ **苏晓选了它而非更稳的「勇敢试炼」，被传送到灵冕城外 150 公里的古遗迹独自迎战**（⚠ **原因很简单:他很想试试刀术宗师大招「强力斩」的威力**）。

- ⚠ **⚠ 身高十米以上、上半身人形、下半身腿足如野兽般弓曲、头颅右侧生满木质化犄角、右臂满是深渊之眼**;⚠ **⚠ 它秉承「能围攻绝不单挑」的策略，扑杀上前时周边黑暗生物一拥而上。**
- ⚠ **⚠ 被一刀秒杀（53卷18章）**:⚠ **它哪里知道，苏晓斩龙闪上爆发的金色光华根本不是能力，而是「强力斩」出刀前的迹象**——⚠ **苏晓「朴实无华、却仿若经历亿万次重复与磨炼的一刀」斩下:方圆上百公里空间如爆裂的玻璃满是裂痕，近三分之一的黑暗生物被碎裂的空间搅碎**;⚠ **⚠ 深渊次子只能抬右手爪去迎（闪避、空间能力全无意义），右臂化为血烟，这一刀落在它身上——**整个右侧身躯连右侧头颅近半一并消失**，它强悍的再生力毫无意义，栽倒炸散**（⚠ **代表★★★★★难度大 boss 就这样被秒了·连苏晓自己都有瞬间的诧异**）。
- ⚠ **⚠ 这一刀斩下后，苍穹之上的黑云被斩开、构成一道映下白光的沟渠——夜晚暂时化为白昼**（⚠ **亦如「强力斩」在技能列表内的名称:极刃·白夜**）。
