# 我爱罗

[火影忍者世界](../世界/任务世界/火影忍者.md) **五代风影**——约十七八岁、满头红色碎发、额头一个「爱」字，背负一米高的沙葫芦。是体内封印**一尾守鹤**的人柱力，[勘九郎](勘九郎.md) 与 [手鞠](手鞠.md) 的弟弟（同为四代风影之子）。是 [苏晓](苏晓.md) 加入「晓」后**活捉人柱力计划**的第一个目标。

- **砂之操控**：注入查克拉的沙子可凭空漂浮（飞行）、结沙盾 / 大沙球防御、**砂缚柩**（黄沙缠身收紧勒断骨头）、**沙柱**（地面顶飞敌人）、**沙之铠**（查克拉沙覆盖全身如陶瓷甲）。空中布**沙结界**难被察觉。
- **「近战辣鸡」的人柱力**：原著就显——尾兽化之前他近战极弱。**体内的一尾拖累他不能发挥全力**：一旦尾兽化会立刻失控，故苏晓近身后他毫无还手之力。
- **被攻略经过（九卷34–36章）**：先被苏晓 [影刺](../力量体系/技能/index.md) 隔沙球刺穿半身（一臂垂落）+ 侧踢碎沙之铠；姐姐手鞠出手解围、却被苏晓挟持当人质逼他分心 → [迪达拉](迪达拉.md) 趁机黏土轰炸将他炸得半死。
- **被吸干本源 · 被掳走（九卷36–37章）**：苏晓跳入沙球、以**吞噬之核**按住他脸，强吸出一尾守鹤的查克拉与**本源之力**（守鹤本源被抽四分之三）→ 苏晓青钢影由 Lv.10 暴涨至 **Lv.20**；我爱罗失去意识，被掳上黏土鸟带走。砂忍村因内部不团结（忍者怕他这个「怪物」不敢支援）+ 高端武力太少而惨败，建筑被炸毁五分之二，转向木叶 [纲手](纲手.md) 求援。

- **续命 → 被封印致死（九卷39–43章）**：撤退途中他被苏晓吸得气若游丝，苏晓注射**初级炼金秘药**续命（验证了乐园药剂对剧情人物有效）+ 麻醉剂控制。运抵河之国川流谷山洞后，[长门](长门.md)（天道）召出**外道魔像**、以「幻龙九封禁」历时三天将一尾守鹤从其体内剥离 → **尾兽被剥离的瞬间，五代风影我爱罗死亡**（砂之铠碎裂剥离、生命气息消散）。木叶 + 砂忍追兵赶到时已迟（弟弟同伴 [漩涡鸣人](漩涡鸣人.md) 见尸暴怒、几近尾兽化）。

## 复活 · 五影大会驰援（十五卷11–19章）

九卷被剥离一尾身亡后，果如苏晓预料**被祖母级的 [千代](千代.md) 以秘术复活**（千代为此付出性命）。十五卷他已坐稳**五代风影**，赴铁之国五影大会——会上被 [大野木](大野木.md) 当众嘲讽「身为影却被掳走」下马威，他不卑不亢以风影之位回敬。五影大会爆发后他率护卫 **[勘九郎](勘九郎.md)（兄）/ [手鞠](手鞠.md)（姐·见苏晓咬牙切齿·当年正因她被掳、我爱罗硬扛攻击救她才中招被擒）** 驰援雷影，却**被 [阿姆](阿姆.md) 拦在半路**（误以为是通灵兽）、没能及时赶到。等他下到一层，[四代雷影](四代雷影.md) 已被斩首；他本欲留下苏晓，被带土「佐助已去找其他影 + 天照黑焰痕迹」逼得果断撤退（身为风影、他不再被个人仇恨左右）。

## 第二次死于苏晓之手（十五卷91章）

忍界大战中他随四影被斑揍得半死，又被 [苏晓](苏晓.md) 收拾——被阿姆冲撞缠住、[布布汪](布布汪.md) 咬臂、阿姆斩臂后扔到苏晓面前，最终被**斩龙闪贯穿心脏而亡**（生平第二次死在苏晓手中）。临死他抓着苏晓裤管问「为什么要做到这种程度」，苏晓答「因为立场不同」、还透露「我之前想站你们这方、却被照美冥拒绝了」。苏晓由此再得一枚淡金宝箱（斑协助·削减 40%）。
