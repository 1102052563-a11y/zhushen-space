# 我爱罗

[火影忍者世界](../世界/任务世界/火影忍者.md) **五代风影**——约十七八岁、满头红色碎发、额头一个「爱」字，背负一米高的沙葫芦。是体内封印**一尾守鹤**的人柱力，[勘九郎](勘九郎.md) 与 [手鞠](手鞠.md) 的弟弟（同为四代风影之子）。是 [苏晓](苏晓.md) 加入「晓」后**活捉人柱力计划**的第一个目标。

- **砂之操控**：注入查克拉的沙子可凭空漂浮（飞行）、结沙盾 / 大沙球防御、**砂缚柩**（黄沙缠身收紧勒断骨头）、**沙柱**（地面顶飞敌人）、**沙之铠**（查克拉沙覆盖全身如陶瓷甲）。空中布**沙结界**难被察觉。
- **「近战辣鸡」的人柱力**：原著就显——尾兽化之前他近战极弱。**体内的一尾拖累他不能发挥全力**：一旦尾兽化会立刻失控，故苏晓近身后他毫无还手之力。
- **被攻略经过（九卷34–36章）**：先被苏晓 [影刺](../力量体系/技能.md) 隔沙球刺穿半身（一臂垂落）+ 侧踢碎沙之铠；姐姐手鞠出手解围、却被苏晓挟持当人质逼他分心 → [迪达拉](迪达拉.md) 趁机黏土轰炸将他炸得半死。
- **被吸干本源 · 被掳走（九卷36–37章）**：苏晓跳入沙球、以**吞噬之核**按住他脸，强吸出一尾守鹤的查克拉与**本源之力**（守鹤本源被抽四分之三）→ 苏晓青钢影由 Lv.10 暴涨至 **Lv.20**；我爱罗失去意识，被掳上黏土鸟带走。砂忍村因内部不团结（忍者怕他这个「怪物」不敢支援）+ 高端武力太少而惨败，建筑被炸毁五分之二，转向木叶 [纲手](纲手.md) 求援。
