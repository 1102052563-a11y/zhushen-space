# 库洛洛·鲁西鲁

[全职猎人世界](../世界/任务世界/全职猎人.md) **[幻影旅团](../势力/index.md)（蜘蛛）的团长**——出乎意料地是个相貌英俊的年轻人（并非想象中的粗犷大汉）。在旅团中拥有绝对核心地位，**智商极高、行事缜密**，是 [苏晓](苏晓.md) 自评「最不好对付」的旅团成员。

## 能力 ·「盗贼的秘笈」

- **特质系念能力**：一本念能力日记本——**满足四个条件可偷取他人的念能力**据为己用（① 亲眼见到对方念能力 ② 问出念能力相关问题并得对方回答 ③ 让对方手按书封手印 ④ 三项在一小时内完成）；使用偷来的能力时须**手持秘笈、翻到对应页**。被偷者将**永远无法再使用自己的念能力**（对念能力者「生不如死」）。**限制极大**（激战中难凑齐条件、除非对手很蠢），故偷不到苏晓（苏晓动手从不废话、且其能力非念能力体系）。他曾偷得瞬移「操控者之期望」（原主斯莉莉）等。
- 十卷他刚偷得一种**预言能力**，得出一首**预言诗**（预言了窝金 / 侠客之死、「锁链人带走重要月份」〔小滴被酷拉皮卡的锁链擒走〕、「炼狱的使者来临」「疯笑的小丑与猎人对峙」「**垂死倒在无情的猎人脚下**」等）。又用它给团员占卜（[苏晓](苏晓.md) = 诗中「无情的猎人」）——正因这句预言，他**雇了 [揍敌客家族](../势力/index.md) 的伊路米**专程来杀苏晓以避祸。近身肉搏也不弱（速度旅团前三）。

## 历程（十卷46–51章）

- **回归 · 识破西索**：旅团折损四员（飞坦 / 信长 / 窝金 / 侠客）后，库洛洛回到旅团残部据点，宣布**放弃拍品、改以「杀掉 [苏晓](苏晓.md)」为唯一目标**。他识破内奸 **西索**——西索用念「轻薄的假象」（把气变成纸般薄的质感皮）篡改了自己的预言诗、伪装成「救世主」，被前来的临时成员 [薇妮](薇妮.md) 当场揭穿；库洛洛驱散假象、读到真相（西索勾结 [酷拉皮卡](酷拉皮卡.md)），**下令全团追杀西索**，西索撞墙逃走、沦为过街老鼠。
- **与红色小丑联手**：为复仇，库洛洛**与 [红色小丑](红色小丑.md)（绅士同盟）结盟**——红色小丑召来上千异界生物当炮灰，约定 9 点拍卖会开场时合力强攻拍卖场、共杀苏晓。他还布下苏晓「想不通」的局（让 [芬克斯](../势力/index.md) 在拍卖场外游而不入引诱）。

- **拍卖场之夜（十卷56–57章）**：与 [红色小丑](红色小丑.md) 在拍卖场外的楼顶坐镇——他**看穿了小丑「炼狱伯爵虚影只能唬人」的虚张声势**，但因双方仍互有价值（小丑握 [尼拉潘](../世界/生物万族/index.md) 弱点、库洛洛握战力 + 旅团成员）而暂未翻脸。小丑则反算计：**库洛洛「太过相信预言 / 命运（念能力）」正是其弱点**。他拒绝了为小丑接驳断肢（手脚已不鲜活、做不到）。
- **酒店伏击 · 被斩杀（十卷62–65章）**：他识破小丑「拿他当诱饵」、反将计就计——料定苏晓会来刺杀小丑，遂在尼拉潘攻城时按兵不动、携 [揍敌客家族](../势力/index.md) 长子 **伊路米** 在酒店四楼设伏。他流着泪向苏晓确认「窝金 / 飞坦 / 信长 / 侠客是不是你杀的」后动手，与伊路米合围。苏晓以反击盾衍生招「**[赌一分钟](../力量体系/技能.md)**」（踩盾多段跳）绕后斩其背、青钢影噬其念；最终苏晓被伊路米的控制犬咬住、又被库洛洛剧毒匕首刺中后心之际，**花 10 万乐园币买 [薇妮](薇妮.md) 一枪**贯穿三人，反手以斩龙闪刺穿自身刺向库洛洛——库洛洛瞬移逃命却被 [岚凯风衣](../力量体系/装备道具.md) 的「初级空间禁锢」封住，被斩龙闪自下颚破脑**斩杀**。**结算：[苏晓](苏晓.md) 得世界之源 +14.9%（→54.6%）、旅团声望 −4000（→−4900·永久血仇）、掉落金色宝箱（苏晓首个金色品质·淡金之上）**。
