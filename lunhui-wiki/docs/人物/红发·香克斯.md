# 红发·香克斯（四皇）

新世界四皇之一「**红发**」——独臂、眼上有疤、腰插西洋剑、满头深红发，**霸气侧漏**；大海般的胸怀，多数势力都会给他几分薄面。他不抢地盘、人不犯我我不犯人，**最希望新世界安定下来**（他要的不是地盘、而是「时间」·因为「有足以颠覆这个时代的大事要发生」）。

## 德岛海战停战的公证人（十八卷41–45章）

[多弗朗明哥](多弗朗明哥.md) 与 [大妈](夏洛特·玲玲.md) 谈判赔偿时，红发**应邀做公证人**（半小时前被多弗联络即同意、不索酬·因他比谁都希望新世界平静）。会面定在歇斯海域（近白胡子地盘）、他的爱船 **雷德·佛斯号** 上。

- **病猫状态**：当公证人前夜他的好友 **鹰眼**（鹰眼·快一年没见）来访、老友相聚喝得酩酊大醉，谈判当天宿醉得像只病猫、靠副船长本·贝克曼提醒才强压不适。
- **霸气镇场**：大妈在谈判中暴起霸王色霸气，红发一句「冷静一些」+ 释放威压便压下她的气势（三船海贼倒地、琵卡忘了呼吸·但苏晓评其霸气「没想象中那么恐怖」、以青钢影硬抗）。
- **公证人的分寸**：他只调停冲突、不掺和具体交易；当 [苏晓](苏晓.md) 在他船上掏出阿波罗威胁大妈做饼干果实交易、险些殃及雷德·佛斯号时，红发明确「你们私下的交易与红发海贼团无关」——既不出手帮苏晓（苏晓非其船员 / 好友）、也乐见大妈把苏晓引到她自己的船上去谈（红发最想看到的结果）。
- **被苏晓的「艾斯情报」勾住**：苏晓临走抛出「推进城关的那个〔波特卡斯·D·艾斯〕近期会被处死、世界政府要大做文章」——红发眸子放光（他只知艾斯被捕、不知详情），副船长本·贝克曼为他点破苏晓的来历（世界政府前科研人员 / 武器专家 / 海军「白鲸炸弹」即其所造）。

## 关系
- **本·贝克曼**：副船长（叼烟 + 燧发枪·智谋远胜红发·替他权衡苏晓可信度）；**拉基·路**（胖子团员·爱吃肉）。
- **鹰眼**：老友（来访喝酒、害红发宿醉当公证人）。
- **[夏洛特·玲玲](夏洛特·玲玲.md) / [多弗朗明哥](多弗朗明哥.md)**：他调停的谈判双方。
- **[苏晓](苏晓.md)（白夜）**：在他船上威胁大妈、又用艾斯情报勾起他兴趣的契约者（本·贝克曼评「比海贼更疯狂」）。
