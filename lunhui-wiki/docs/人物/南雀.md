# 南雀（二番队暗杀部队 · 伪装跳脱的监视者）

[死神世界](../世界/任务世界/死神-尸魂界.md) 一名跳脱开朗、关键时刻「胆小」、自称因此不得重用的女性死神——表面是奉命**侍奉 [苏晓](苏晓.md) 生活起居**的小萌妹（每天来收药剂、备午餐、藏门后只探头防爆炸）。实则是 [碎蜂](碎蜂.md) 亲手培养的**二番队暗杀部队卧底**，真正任务是**监视苏晓**。

## 被苏晓看穿 · 反成最好的伪装（二十卷22、31–32章）

苏晓**从一开始就看穿她**——她的气质与 [小助手](小助手.md) 相似度超 40%（同样**漠视敌人生命**、是杀过很多死神 / 虚的暗杀者），喝水前会用唇舌大面积接触试毒（专业训练才有的习惯）、血气压得极淡。山本元柳斎与卯之花烈派她侍奉苏晓，本身就说明对苏晓未松警惕。但苏晓**不杀她**——反让 [布布汪](布布汪.md) 在她饮食里下了三颗**追踪器**（24 小时掌握其位置），又跟踪到她向 [碎蜂](碎蜂.md) 汇报。如此一来南雀反成苏晓**最好的不在场证人**（在她视角苏晓始终在房里饮酒、永远不知苏晓夜里出去猎杀）。她对碎蜂爱慕极深（被碎蜂按头一句「辛苦了」就脸红到扯破衣襟）。

## 关系
- **[碎蜂](碎蜂.md)**：队长 / 培养她的人（爱慕极深）。
- **[苏晓](苏晓.md)（白夜）**：监视对象（反被其看穿利用作不在场证人·气质似 [小助手](小助手.md)）。
- **[布布汪](布布汪.md)**：在她饮食里下追踪器的「真凶」。
