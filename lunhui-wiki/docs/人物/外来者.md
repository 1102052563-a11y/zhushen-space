# 外来者（TQ-9670 号）

[苏晓](苏晓.md) 在第十五卷火影世界的**猎杀任务目标**——代号「**TQ-9670 号外来者**」。他是**其他乐园派来潜入轮回乐园衍生世界的契约者 / 间谍**（疑天启乐园·苏晓一直找不到），因没有轮回乐园的「伪装机制」，能力在剧情人物眼里显得异常、行事小心却不计后果、还总带点「倒霉」（被世界轻微排斥）。

## 画像 · 滑溜的泥鳅（十五卷74章）

经白绝长时间追踪 + 大蛇兜的转生感知忍者网，终于看清其样貌与能力：

- **外形**：人类、男性，20~25 岁、棕色短发、身高 1.73~1.78 米，**右臂纹一只瞳孔**、相貌阳光英俊。
- **远距离传送能力**：能在**预设的固定坐标**间瞬移（已确认 35 个坐标·火之国 26 / 土之国 3 / 风之国 2 / 雷之国 2 / 水之国 2）；向某坐标移动后 2~3 天内无法再用该坐标，使用间隔 1~2 小时。**这正是他「3 小时从土之国跳到水之国」（斜跨整个火影版图 + 大片海洋）的真相**。
- **战力 / 软肋**：战斗力最多三阶初期（白绝分身曾围攻过一次），但**逃跑能力连四阶契约者都未必有**——苏晓判定他「实力不强、却像泥鳅一样滑溜」，清理的难点不在战力、而在如何找到并追踪。
- **高魅力**：现身雷之国时勾搭五代雷影的妹妹（一名女下忍）套晓组织情报、轻易撩动其心绪——疑有很高的魅力属性。

苏晓只带 [布布汪](布布汪.md) 去追（人多无益）、计划先展现碾压性近战、逼他「一见苏晓就逃」，再设局清理。

## 真正的能力 + 目的（十五卷75–76章）

带土（空间专家）由波动判定：他**并非空间能力、而是「精神传输」**——把肉体与精神同化成完全的精神体、再无视物理规则地在预设的「**精神节点**」间「精神跳跃」（置换类·非破开空间）。每次传输后冷却 47 分钟、**70 小时内不能向同一节点传两次**（否则精神反震、破坏神经元变白痴）。他牺牲了战斗力换这身极强机动（实力仅三阶初期·苏晓一斩就断他一臂）。

他的**真目的是「废掉整个火影世界」**：勾搭五代雷影之妹（高魅力奴役）在云隐村布下天启乐园的战略物资「**希尔之晶岩（扭曲之源）**」——集齐四颗组成扭曲法阵、能把世界变成排斥空间之力的危险废土。轮回乐园拿下衍生世界后会大量复制它、世界一旦不可修复便是巨大损失——他正是**天启乐园派来恶心轮回乐园的间谍**（故这次猎杀奖励近乎平时 5 倍）。苏晓审出晶岩、用炼金炸弹反逼那名女下忍拆阵交货。

## 殒落 · 一枪爆头（十五卷76–77章）

苏晓不与他正面纠缠、而是织就天罗地网把他逼成困兽：① **白绝伪装成苏晓**反复现身恐吓（白绝分身有苏晓 80% 身体素质、却无灭法能力·只吓不战·拿捏 1.5 小时的传输间隔逼他不停逃、避免节点冷却循环）；② 大量秽土转生感知忍者分区围猎（加藤断〔灵化之术〕/ 油女取根 / 日向日差 / 二代土影无…）。被追十几小时、几乎传遍各国、失血 + 灵化之术伤令他精神恍惚。苏晓则赶到精神节点最密（11 个）、地势平坦无险可逃的**木叶附近**埋伏八小时——以「**恶魔凝视**」（[恶魔之印](../力量体系/技能/index.md) Lv.10·子弹附 80% 真实伤害）的子弹 + 蜘蛛女皇，**1737 米外一枪打爆他上半身**。猎杀任务完成、**评价 A+**（大幅拉高世界综合评价）、得黄金荣誉勋章 ×5。
