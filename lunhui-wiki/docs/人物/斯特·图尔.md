# 斯特·图尔（盲圣刃）

[黑渊](../世界/任务世界/黑渊浅层-赔罪者之殇.md) 中名号赫赫的「**盲圣刃**」——157 岁、独眼且双瞳惨白的瘦削老者，看似一阵风就能吹走，背一只腐朽木箱（**生灵匣**·内藏 16 把史诗级长刀·拔刀如孔雀开屏）。虽目不能视、却连颜色都能分辨（靠一种浑浊的「念 / 磁场」感知）。[苏晓](苏晓.md) 接的第一个阿萨卡委托即「杀斯特·图尔」（酬劳一把史诗级狙击枪 [噬灭](../力量体系/装备道具/index.md)）——背后是某个女人执着了几十年的旧怨。

## 能力 · 险胜（二十三卷14–15章）

- **刀术宗师 Lv.17**、近战属性 98~100（**敏捷卡 100·破极值**）——速度快如鬼魅、几米移动直追龙影闪。
- **盲鱼**：从生灵匣抽出的佩刀（不杀生不归鞘）·品质不低于斩龙闪、比拼速度更快。
- **血泣流**（真实敏捷破壁障后的能力）：对躯干造成割裂伤即触发流血、按伤口大小每秒掉 15~100 生命、**持续 3 小时 25 分**（伤口极难愈合）——这才是苏晓最凶险处，故他斩杀后匆匆撤离、连吞 4 瓶远古秘药才扛过流血。

两名用刀宗师相遇、退者死。苏晓以「战斗中弃刀 + 侧踢踹飞 + 龙影闪绕后」的「无耻」打法（两用刀者交手不必刀下分生死）斩断其持刀臂、再割喉。他一身实力都在刀上、失刀即死。掉落 [生灵匣](../力量体系/装备道具/index.md)（史诗级·苏晓拿来喂斩龙闪）。斩盲圣刃让「杀手·白夜」之名传开、引 [布布汪](布布汪.md) 来会合。

## 关系
- **[苏晓](苏晓.md)（白夜·杀手）**：奉阿萨卡委托斩杀他的对手（两用刀宗师的生死局）。
- **[羽鸦](黑渊配角.md)**：他临死前正护送的羽族小公主（与少年随从·苏晓目标非她故放过）。
