# 蝶

[血门冒险团](../势力/index.md) 的**头号控制师**——紫发妖艳少妇、**魅力特长 / 特质系念能力者**（念能量是肉眼可见的**粉红色**、总量远超幻影旅团团长或窝金）、**三阶顶级契约者**。受斯坦之命与 [荒焚](荒焚.md)、[小胖子](小胖子.md) 进 Fate/Zero 追杀契约者、主司**追踪 + 控制 + 削防**。她贪图苏晓的人头（斯坦许的重赏）、撇下荒焚带小胖子单独行动。

## 能力（多数被二阶世界压制、但装备不受压制）

- **爱意俘虏 LV.43**（魅惑·被压至 LV.35）：粉红心形念能量·让人疯狂爱上她——但被苏晓 [刀术大师 Lv.30「心灵之刃」](../力量体系/技能.md) 免疫。
- **心形念能量**（削防）：飘空、追击、一触即碎，**大幅削减目标防御**（连 400 点反击盾都被削脆·触到人体则人如玻璃般易碎）；可分裂成几十枚。
- **大量高品级装备**（碾压苏晓）：**动能装甲**（无形护盾·防御极强、能挡锋利度 +32 的斩龙闪·每 60 秒有 3~5 秒空档）、金色护盾（可见·8~10 分钟一次·持续 2 秒）、增力 / 敏项链（短时力量超苏晓 5~10 点）、**艾克马秘刃·花舞浪漫**（膝撞附眩晕 2 秒）。
- **近战不弱**（三阶辅助的近战仍能与苏晓不落下风·但缺乏血肉拼杀的历练、**怕疼**）。

## 死亡（十一卷40–42章）

她主动近战苏晓——苏晓凭心灵之刃免疫魅惑、靠青钢影克她念能量（一刀噬其 140 念 + 一长串真伤），又用战修斯后新创的「[能量外骨骼](../力量体系/技能.md)」（像须佐能乎）化解她的背摔。她有装备碾压、却败在**怕疼 + 苏晓抓住她「动能装甲」60 秒一次的 4 秒空档**——苏晓被她断头台勒到脑供血不足之际，左手反握斩龙闪后刺、**刺穿她眼窝 → 贯脑 → 斩喉**杀之（暗紫恢复药剂从喉伤口漏出、没救回）。**杀这名三阶顶级契约者，猩红卡掉落率提升、掉 1 张猩红卡**。蝶一死、三人组就失去了追踪苏晓的能力。

## 关系

- **[荒焚](荒焚.md)** / **[小胖子](小胖子.md)**：同组（撇下荒焚单独行动）。
- **[苏晓](苏晓.md)**：将她斩杀。
