# 法师贤者·瑟菲莉娅

来 [虚空角斗场](../世界/任务世界/恶魔古堡游戏.md) 观战的**法师贤者**——虚空中很强大的存在、守序阵营的大人物，红色法师袍 + 兜帽。她是 [灭法之影](../力量体系/技能/index.md) 的**世仇**（与灭法同代的「老不死」之一·正是 [苏晓](苏晓.md) 最忌惮的那类敌人）。

## 出手与被逼退（十四卷54章）

[苏晓](苏晓.md) 一登场，瑟菲莉娅便起身、指尖凝起暗红诅咒「**灼热之痛**」锁定他——这种层级的出手连主办方都拦不住。千钧一发之际 **[鼠大人](鼠大人.md)** 持古旧猎枪（枪管布满符文·上膛即空间扭曲）顶住她，把两人一起传送到角斗场外：

- 鼠大人一枪**穿透空间**（音爆层层扩散、空间崩碎）逼她以「禁」字符文勉强挡下、**指尖渗血受伤**；
- 鼠大人再上膛、声称「这一发是**源爆弹**」（她领教过其威力），终于逼她收手回座。

事后她「给举办方一些薄面」、短时间不再出手——作为守序大人物，她**最多用隐匿手段悄然刺杀苏晓、不会在明面动手**（被鼠大人评为「虚伪的女人」）。这印证了莉莉姆的判断：苏晓真正的危险不是新生代、而是这些与灭法同代、活了千年的「老不死」。

## 追杀至迷火森林 · 身世（十四卷59、63章）

苏晓第二轮结束、随莉莉姆传送去 [迷火森林](树贤者.md) 速成炼金术时，瑟菲莉娅**追踪空间波动赶到**、张口就要放「炎火」——却被 [树贤者](树贤者.md)（迷火森林主人·领地内实力翻几十倍）的树人挡下，她不敢在他地盘造次（如同他人不敢闹她的魔法塔），只能悻悻撤走，并道出**身世与立场**：

> 「我是瑟菲莉娅，一名**被灭法之影毁灭家园、无处归去的火系法师**。我们个人之间没任何恩怨，但你成为灭法者那一刻，你就是我必杀的目标——杀了我、或被我所杀。」

她没为杀苏晓而拼死在树贤者地盘，因为**虚空中还有另一名更强的灭法者（疑「刀魔」一脉）她杀不掉**。她外号「**疯女人**」——曾**两次伏杀刀魔**（一次险死、一次被剥皮扔进黑渊），可见其执念与狠绝。事后她与树贤者交涉、暂不再纠缠（苏晓只要返回轮回乐园她便鞭长莫及）。

## 雇凶刺杀苏晓（二十九卷·补充）

苏晓晋六阶后，瑟菲莉娅**雇佣元素族刺客「尤·阿奇德」**（前阿萨卡组织成员）在苏晓返园传送途中偷袭——被黑王套装血壁挡下、青钢影耗 8000 法力噬灭其黑能量，刺客随后被恶魔族解决。据 [莉莉姆](莉莉姆.md) 情报，奥术永恒星对苏晓分两派：**瑟菲莉娅主张「将灭法者扼杀在萌芽」灭苏晓**，[风王](风王子.md)（风王子之父）主张「灭法时代已过、人不犯我我不犯人」。瑟菲莉娅活了上千年、在奥术永恒星权力不小（传言曾被一名女灭法者「吊起来抽」过）。是苏晓现阶段无法对抗的两大敌人之一（另一为爱丽丝）。

## 🆕弟子狄琳之死 · 与苏晓正式敌对（三十二卷·强者争霸战）

瑟菲莉娅几百年来「克灭法者」的研究全栽在苏晓手里：她把弟子 **[狄琳](强者争霸战群像.md)** 当人体实验品培养出**黎元素**（土 + 光剥离重铸的异元素·克灭法者）+ **吸青钢影的特制耳钉**——两样都在强者争霸战被苏晓两招破掉。她亲临第四轮角斗场观战、暴鼠安排苏晓对上狄琳，狄琳被**一分钟钉死**（临死「这是你最后送我的礼物」）。瑟菲莉娅当众被打脸（视狄琳如「个人财产」·别人侮辱狄琳就要「永灭」），与苏晓**彻底结仇**，却碍于虚空之树 + 轮回乐园多方公证不能出手。奥术永恒星 **至高之人**（戴马文 / 吉莉安 / 初代灭法骨戒）态度 = 除掉这仅存的灭法者；[凛风王](强者争霸战群像.md) 当众斥其「拿弟子做实验、法之魂早被灭法者踩灭」。

## 🆕悬赏苏晓人头（三十三卷60章·陨灭之界返园后）

奥术永恒星至高之人下令「清除最后的灭法者」后，瑟菲莉娅亲赴虚空「流放之地」（恶魔族 / 羽族 / 奥术永恒星都管不到的混乱凶犯流放地）——出 **50 克黑枫树枝干**（黑枫树枝叶平时按毫克算·她按克付）悬赏 [苏晓](苏晓.md) 的头颅，雇那里不怕死的凶犯去刺杀（凶犯首领波勒鲁动心成交）。她借流放之地的人「试试水」（有些乐园干涉的世界 / 虚空外世界她本人进不去）。

## 🆕魔海伏杀（四十一卷·疑似幕后·补充）

苏晓晋八阶后进入首个原生世界 [魔海](../世界/任务世界/魔海-厄运与诅咒.md)，一落地便撞上圣光乐园「毒奶」[光沐](魔海/光沐.md) 的伏击——光沐已在魔海**蹲守苏晓三个世界进度**、并笃定「厄运号船长晋八阶后第一个世界只可能是魔海」。苏晓复盘现阶段三方敌人（灰绅士 / 神父 / 奥术永恒星）后判断：灰绅士与神父不会这么草率、奥术永恒星至高之人出手必是「屠尽全世界活物 + 高阶施法者围杀」的天崩地裂手笔——**唯有作风「小家子气、动辄派暗杀者 / 弟子白给」的瑟菲莉娅最符合、且多半是她私自行事**（未得奥术永恒星授意）。这与她一贯的隐匿刺杀路数一脉相承。

## 🆕 画中世界观战 · 「野兽」乌鸦女（44卷32章）

[画中世界](../世界/任务世界/画中世界.md) 画卷争夺战里她在莫乌斗技场观战席——奥术永恒星的炎启·索耶格、洛希被苏晓接连斩杀，新生代施法者又羞又怒，唯她格外淡定（相比强者争霸战亲眼看弟子 [狄琳](强者争霸战/狄琳.md) 被苏晓公开处刑那次的奇耻大辱，这已是小场面）。她身旁坐着奥术永恒星的杀手锏 [乌鸦女](画中世界/乌鸦女.md)——一头被她们生命学派长辈养成的「野兽」、对她毫无敬意、句句带刺。

## 🆕 被「太阳圣剑」惊到 · 奉命去请「药师圣焰」（46卷45章）

此时她的定位是**奥术永恒星高层之一**——身穿金白法袍、戴兜帽只露下半边脸的**法师贤者**，掌**黎元素**（能衍生近战能力）。她与同僚 [凛风王](凛风王.md) 素来不睦（没有至高之人威压早就闹翻）。奥术永恒星是树生世界·杀戮竞技的参与方之一，能通过 [乌鸦女](画中世界/乌鸦女.md) 了解世界内情况。

[苏晓](苏晓.md) 在树生世界用两千颗阿波罗轰出的「**太阳圣剑**」，让奥术永恒星高层（含元素波动强到看不清模样的**至高之人**、凛风王等）如临大敌——他们担心这种爆炸物若在即将召开的「**奥法庆典**」当天投向奥术永恒星、后果不堪设想，苏晓的敌对等级因此飙升（但那爆炸物弊病多、引爆慢，至高之人暂未下令动手）。⚠ 与此同时，瑟菲莉娅正操碎心地去办另一件事:一种极适合施法者饮用的药剂在虚空流传开、连长期合作的炼金大师·[树贤者](树贤者.md) 都惊叹调不出，奥术永恒星急欲把调药的**「药师圣焰」**请来供起来——那药师与黑暗世界皇帝 **[白牛](白牛.md)** 合作、消息含糊。她打算赶在奥法庆典前几天把药师请到、当天抛橄榄枝。她甚至被凛风王调侃「去请药师时别摆死人脸、你除了年龄大还很有姿色」、被至高之人示意要微笑相迎，只好回家练练久违的微笑。**⚠ 她做梦也想不到，要迎接的「药师圣焰」正是苏晓本人**。

⚠ 而在另一头，白牛心知圣焰就是灭法者苏晓，表面「被那女人逼得没法」、实则把她的原话原封不动转达给苏晓——等于借她的手把苏晓引荐进了奥术永恒星（详见 [白牛](白牛.md)）。

## 🔮 「坏心眼的预言家」占卜出的那张牌（54卷209章）

苏晓在决战前用【坏心眼的预言家】占卜——这张深渊奇物占卜牌上是名衣着贵气、有着浅紫色眼影、表情冷漠的美丽预言家。以苏晓 −100 点的魅力，这位美丽预言家非常厌弃他，那种「绝不会帮助你」的神情已写在脸上。

第一次翻牌，结果是**「死神」**——分明代表他即将迎来死亡。

于是苏晓把【坏心眼的预言家】封印进「原罪之书」第八页，安静等待 **3 秒**。再翻出来时，这位美丽预言家的心眼也不坏了，而是**目光惊惧**。

> 「我以灭法之影的名誉担保，帮我占卜一次，让你离开。」

这次的牌面是:一身金色华丽装束、一手执掌权杖、一手拿着书籍的美丽占卜家。苏晓没看懂（占卜学涉猎较少），贝妮弄回一本占卜体系基础书籍后他很快查到——那身装扮意味着:

> **贤者。**<br>奥术永恒星、贤者。<br>**法师贤者·瑟菲利雅？**<br>——**后续去奥术永恒星，第一个宰了这施法者。**

⚠ 至于「占卜一次就放它离开」的承诺:**灭法之影的名誉担保，早在第二纪元就人尽皆知**。（半小时后苏晓再把它封回第八页、翻开时，它又变回了看魅力下菜碟的厌弃模样——在强大的原罪力量冲击下，这本就是死物但有一定智慧的占卜牌被成功「格式化」了。别说它，就连蛀世被关久了也都被强行「格式化」。）

## ☠ 化身之死（54卷211章）

星辰广场上，她与凛风王一同现身，从后方射出黎元素构成的金色锐刺直取苏晓眼睛。双方对视时，她眼中不再是上次的恨意与不甘，而是**迷茫**——上次见面苏晓已比她强大，可总归都是至强；这次，作为至强巅峰的苏晓居然来单挑整个奥术永恒星。

> **此刻的法师贤者·瑟菲莉娅，比当初被某个渣女抛弃时候，内心更加迷茫。**

咔吧！黎元素武器被苏晓**单手抓住**。

⚠ 这一幕引发的记忆，比直接斩她一刀更刻骨铭心:**回来了，完全回来了——第二纪元时被先代灭法者们支配的恐惧，全部回来了。** 不仅是她，所有经历过第二纪元「灭法 VS 施法」战役的老施法者，目睹苏晓徒手抓住元素武器的一幕，全都回忆起了昔日的恐惧。

天启者·伽罗娜被一脚侧踢炸碎后，其骨骼碎片刚好把瑟菲莉娅的身体洞穿出多个血洞。苏晓单手掐着她的脖颈，目光疑惑——无论是感知中还是亲手触碰到的灵魂气息，都从方才的完整变成**只剩小半**。这是化身一类的能力。

傲歌晶体在他手上蔓延、侵蚀她的脖颈。被扼住咽喉、全身青钢影能量窜动、身体完全麻木的她，双眼中**没有半分恐惧**:

> 「你……**会来，给我们，陪葬**……」

咔吧！苏晓没半句废话地掐碎了她的脖颈。

⚠ 苏晓的注脚:说起来，他和奥术永恒星的仇恨越来越大，**真就多亏了这位法师贤者**。（当初负责弄死苏晓的正是她，那时苏晓才三阶；若非轮回乐园的世界阶位压制，他必死在三阶世界。）

## 💀 真身之死 —— 亲手打开深渊之门（54卷214章）

真身在虚空「绿荫大湿地」的**特殊深渊通道**前。她双手抓着封住通道的**「暗月星环」**两侧——星环上已遍布裂痕——

嘭！星环破碎，后方的深渊之门逐渐化为实体。

此刻她的眼睛格外明亮，**笑得很美**，目光灼灼地看着苏晓，等待欣赏敌人的愤怒。她亲手破坏了自己一直以来的封禁，并知道深渊之门即将打开、星界将要进入深渊时代——

> 就如猜测的那般:**奥术永恒星这辆没有刹车却全速行驶的列车，要将所有人拖入地狱——要么一直作为霸主，要么带来毁灭。**

然而她并没从苏晓脸上看到半点愤怒，而是平静，**甚至没去看她**，只仰头看着深渊之门，与门内的深渊之主用深渊语互致问候（见 [深渊之主](深渊之主.md)）。

**被无视的愤怒**，让被深渊重度侵蚀的她黎元素作势爆发——

> 铮！刀芒一闪而逝。

作为新晋至强、连命核都没有，加之失去三分之一的生命本源与因果，想抗住这一刀斩喉实在过于魔幻。她双手捂着喉咙，眼前视线快速模糊，**最终看到的，是那灭法者一步步走远的背影，直到眼前化为一片黑暗。**

> ⚠ **昔日高高在上的法师贤者·瑟菲莉娅，最终还是死于灭法者的刀下。**
