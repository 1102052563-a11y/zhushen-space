# 迈克·凯（迈特凯）

[火影忍者世界](../世界/任务世界/火影忍者.md) 木叶上忍——**纯体术大师**，率「凯小队」（成员 [宁次](宁次.md)、天天、小李）追击掳走五代风影的「晓」。

- **忍界真正的强者**：他**没有血统、没有家传秘术、甚至不会忍术**——纯靠少年起至今从未停歇的苦练成才。开**八门遁甲**后实力与这一时期的强者彻底脱节：「**开八门一脚踹飞六个佩恩都不夸张**」（虽随后会因反噬而死）。[干柿鬼鲛](干柿鬼鲛.md) 称他为「那头珍兽」、与他交过手。
- **追击战中（九卷40–43章）**：晓在河之国封印一尾，[长门](长门.md) 用「**临时身体**之术」（不出动本体·消耗约 30% 查克拉）迎敌——派鬼鲛迎战凯队、鼬迎战卡卡西队，**两战皆败**（临时身体仅本体五成实力）。最终凯小队抵山洞外，由 [宁次](宁次.md) 白眼定位、四人合力破长门的「五封结界」。
- **苏晓的评价**：[苏晓](苏晓.md) **不愿与这种「随时爆豆」的近战狂战斗**——凯开七门后他能否活下来都未知；但苏晓「青钢影克查克拉 / 近战」之能恰好克制凯，纵使凯踢死他、自己也得断胳膊断腿。

- **七门「夜凤」救卡卡西（九卷46章）**：[苏晓](苏晓.md) 险些斩杀重伤的 [卡卡西](卡卡西.md) 时，凯赶到、**开七门以一脚「夜凤」将苏晓踹飞**（连破苏晓三层 300 法力能量盾、最后一层仅轻裂）救下卡卡西。此时凯因先前与鬼鲛 / 结界敌作战已大耗体力。苏晓极忌惮他更高一级的「**夜凯**」（开死门·秒杀级·连次级无敌护盾都扛不住），故见好就收、撒炼金炸弹骑布布汪撤退。
