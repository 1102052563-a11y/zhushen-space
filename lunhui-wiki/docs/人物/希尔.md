# 希尔

**希尔** 是 [斩·赤红之瞳](../世界/任务世界/斩赤红之瞳.md) 世界 [夜袭部队](../势力/index.md) 的成员（四卷33章）。

## 简介
天然呆的眼镜娘、满头紫色长发。持帝具「**万物两断·消魂**」——一把至少一米长的**大剪刀**（剪刀类帝具）、异常锋利、可两断万物。

!!! note "蝴蝶效应·改写生死"
    按原著，到本时间点希尔已经死去；但在**大量契约者乱入**的影响下（蝴蝶效应），这位天然呆眼镜娘**并没有死**——印证苏晓「不可再按原著剧情判断事态」的判断（四卷33章）。

## 关系
- [娜洁希坦](娜洁希坦.md)（首领）及 [赤瞳](赤瞳.md) / [雷欧奈](雷欧奈.md) / [拉伯克](拉伯克.md) / [塔兹米](塔兹米.md) / [玛茵](玛茵.md) 等——夜袭同伴（出席「讨伐苏晓作战会议」）。
