# 雏森桃（五番队副队长 · 被蓝染利用的兔子）

[死神世界](../世界/任务世界/死神-尸魂界.md) 护廷十三队**五番队副队长**——棕色短发、看上去弱气的少女，实力不弱、对队长 [蓝染](蓝染惣右介.md) 极度崇拜与依赖。[苏晓](苏晓.md) 称她「天真的兔子、可惜被一只狼蛊惑」——蓝染**刻意把自己培养成她的精神依靠**，正是为了在假死布局里利用她。她与十番队队长 [日番谷冬狮郎](日番谷冬狮郎.md) 是青梅竹马（关系在友情之上）。

## 蓝染假死 · 被当棋子（二十卷19、24–25章）

她见证苏晓与「蓝染」饮酒（还送过 [南雀](../世界/任务世界/死神-尸魂界.md) 手链），全然不知这二人将成尸魂界的心腹大患。蓝染假死时**留给她一封信**（内容：自知命不久矣 / 让她「坚强」/ 嫁祸 [日番谷冬狮郎](日番谷冬狮郎.md)）——这封信正是用来**害死她**的工具。见「蓝染」尸体被钉东大圣壁，她精神崩溃、被信与冬狮郎旧言（「小心三番队、特别是蓝染独自外出时」）误导，认定 [市丸银](市丸银.md) 是凶手、提刀冲杀——险被市丸银借机「错手」杀掉，靠三番队副队长吉良伊鹤挡刀 + 苏晓到场拦下、冬狮郎赶到才保住性命。

## 关系
- **[蓝染](蓝染惣右介.md)**：崇拜的队长、却把她当假死布局的棋子（精神依靠 = 操控手段）。
- **[日番谷冬狮郎](日番谷冬狮郎.md)**：青梅竹马（关系在友情之上·暗中护着她）。
- **[市丸银](市丸银.md)**：误认的「杀蓝染凶手」（提刀冲杀险被反杀）。
- **[苏晓](苏晓.md)（白夜）**：以为是帮她的一方（实为蓝染同党·拦下她其实是顺势保她以利布局）。
