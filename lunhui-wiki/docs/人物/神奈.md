# 神奈（圣域乐园头号人物 · 金袍占卜师）

[海蓝星世界争夺战](../世界/任务世界/海蓝星-世界争夺战.md) 中**圣域乐园方的头号人物**——身穿金色长袍、慵懒从容、手不离一副**占卜 / 塔罗牌**的女子。她曾潜伏在太阳王国**神庭**里（金袍兜帽形象·处决不再听话的工具人却不受任何惩罚），是与 [青魇](青魇.md) 在太阳王国军权体系里互相搅局的对手（她骂青魇「搅屎棍」、青魇回怼「那你岂不是就是屎」）。

## 率圣域赶赴热雨林之森（十九卷52、66章）

世界之核现世后，神奈不再纠缠太阳王国、率圣域大队（坐一只十几米高的金属蜘蛛行进舱·内载上百契约者）赶往热雨林之森。她损兵不少——手下圣焰被海王类吞没、泉 / 百魂 / 灰鸦 / 詹姆布等接连被斩首 / 3000 米外狙杀（只剩始终与目标保持五公里的「毒刺」还活着），打听到这些多半是 [苏晓](苏晓.md)（「白鬼」）所为时捏碎了酒杯。她为苏晓**占出「死神 / 树 / 战争 / 雪狼」四张塔罗牌**，对「死神」「雪狼」之意颇感费解。天启乐园方被轮回打怕、不肯去热雨林之森，她遂认定「胜者只有一个、这样更好」。

## 关系
- **圣域乐园**：其方头号人物（率大队夺核）。
- **[青魇](青魇.md)**：太阳王国军权体系里互相搅局的对手（互骂搅屎棍）。
- **[苏晓](苏晓.md)（白鬼）**：连杀其多名手下的「白鬼」（她为其占卜·颇感费解）。
