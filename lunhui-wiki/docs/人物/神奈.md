# 神奈（圣域乐园头号人物 · 金袍占卜师）

[海蓝星世界争夺战](../世界/任务世界/海蓝星-世界争夺战.md) 中**圣域乐园方的头号人物**——身穿金色长袍、慵懒从容、手不离一副**占卜 / 塔罗牌**的女子。她曾潜伏在太阳王国**神庭**里（金袍兜帽形象·处决不再听话的工具人却不受任何惩罚），是与 [青魇](青魇.md) 在太阳王国军权体系里互相搅局的对手（她骂青魇「搅屎棍」、青魇回怼「那你岂不是就是屎」）。

## 率圣域赶赴热雨林之森（十九卷52、66章）

世界之核现世后，神奈不再纠缠太阳王国、率圣域大队（坐一只十几米高的金属蜘蛛行进舱·内载上百契约者）赶往热雨林之森。她损兵不少——手下圣焰被海王类吞没、泉 / 百魂 / 灰鸦 / 詹姆布等接连被斩首 / 3000 米外狙杀（只剩始终与目标保持五公里的「毒刺」还活着），打听到这些多半是 [苏晓](苏晓.md)（「白鬼」）所为时捏碎了酒杯。她为苏晓**占出「死神 / 树 / 战争 / 雪狼」四张塔罗牌**，对「死神」「雪狼」之意颇感费解。天启乐园方被轮回打怕、不肯去热雨林之森，她遂认定「胜者只有一个、这样更好」。

## 雨林扎营 · 静观其变（十九卷76–77章）

进雨林后神奈不学轮回乐园那群莽夫硬拼——她率人数最多的圣域方（353 人）在水源旁扎近 200 顶帐篷、用金属蜘蛛运石料**抢建防御基地**（为最后 5 小时坚守做准备），并下令静观其变、少惹是非。她看穿轮回乐园把世界坐标「转让」给天启是「聪明而又愚蠢」的麻痹之计。她极忌惮 [青魇](青魇.md)（始终不知此人是谁、却被他接连搞死太阳王国两任大神使、堪称耻辱）。一次午茶时差点被 [苏晓](苏晓.md) 凭空出现的子弹爆头（老管家挡下）、又被一名轮回刺客「原琥珀」近身（被她念动力凝住、自爆逃走）——这些只是轮回乐园**麻痹式骚扰**的开端（平均 10 分钟一次远程偷袭，把圣域施工队烦不胜烦）。

## 穷途末路（十九卷79–81章）

苏晓的杀招其实不是契约者——而是 [布布汪](布布汪.md) 麾下 **1276 只极地冰原狼**。圣域营被狼群 + 轮回乐园方包围、近乎全灭（353 人战后仅 16 人逃），神奈引开布布汪未果（它「凭空消失」）、虽亲手捏爆冲进来的冰原狼却挡不住狼海。她率残部逃命、却被轮回乐园方独狼**一个接一个单挑式追杀**（死一个上一个、绝不互助——她由此真正见识了「轮回乐园的规则：强者生弱者死」）。圣域最终只剩她 + 圣焰 + 被苏晓擒走的老管家：她甚至**亲手了结了两名重伤的死忠部下**（部下故意说侮辱话、好让她不手软·爆成血雾、魂之力归她）——这是她最忠诚的一批人。她念念不忘那个查不出身份、接连搞死太阳王国两任大神使、又设局灭她全军的对手（青魇）。

## 找苏晓同归于尽 · 自爆殒命（十九卷86–87章）

走投无路的神奈不肯尊重老管家的牺牲（老管家当初是为掩护她撤退才被苏晓生擒）、反而**锁定老管家死后飘散的灵魂气息、追到苏晓处想拉他同归于尽**。她的能力是把**灵魂能量具现化**、触之即被冻结（无需判定·克制一切近战）——这本是苏晓的克星，但他一见她「心有死志」便果断开**猎魔时刻**（她捕捉不到神出鬼没的苏晓），被斩断右臂 + 挨十几枪后，她释放出最后的灵魂能量**自爆**（30 米内尽化粉尘）殒命，只掉了枚猩红宝箱给苏晓。

## 关系
- **圣域乐园**：其方头号人物（率大队夺核·人数最多 353·全灭）。
- **[青魇](青魇.md)**：太阳王国军权体系里互相搅局的对手（互骂搅屎棍）。
- **[苏晓](苏晓.md)（白鬼）**：连杀其多名手下的「白鬼」（她为其占卜·颇感费解）。
