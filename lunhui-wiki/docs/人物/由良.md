# 由良

[火影忍者世界](../世界/任务世界/火影忍者.md) 砂忍村**上忍**——留小胡子、头发遮住右眼，负责村子防御、人脉极广，本无资格参与砂忍高层会议（凭防御头目身份列席）。

- **不自知的间谍（九卷32–33章）**：他实为 [蝎](蝎.md) 多年前用**大蛇丸的咒术**布下封印、植入砂忍村的一枚棋子——平时**他自己都不知道是间谍**，故从不受人怀疑。蝎在抓捕 [我爱罗](我爱罗.md) 的任务前解除封印 → 砂忍高层会议上他脑中封印碎裂、记忆涌回 → 他**杀掉守卫、为苏晓三人打开砂忍村唯一入口「一线天」**，并以紫烟为号引路、单膝跪拜「蝎大人」。
- **结局**：蝎心存死志，要为「自己的事做个了断」，任务后托 [苏晓](苏晓.md) **清理掉这枚棋子**（苏晓答应）。
