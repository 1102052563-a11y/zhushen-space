# 琼·艾塞亚（生命女巫 · 实力演技派）

[暗鸦星](../世界/任务世界/暗鸦星-暗鸦纪元.md) 边陲**马德高人类聚集地**的统治者——水蜜桃般丰满的白女巫（预言系·真实魅力 90·化身是一朵白茉莉花），能力是**控制植物生长**（不能速生·但波及面积极广·养活周边十几座城·换得马德高的「中立地」地位）。战斗力极弱（女巫中的「老弱病残」·黑女巫都嫌她给女巫丢人），但她是个**实力演技派**——撒谎手法是「九句真话掺一句假」，初见 [苏晓](苏晓.md) 时先装强横、被拔刀就秒怂跪地求饶。

## 伪善的奴隶城 · 出卖伊萨断后（二十一卷3–5章）

她口口声声「一人之力养活十几座城」、实则与黑女巫 [伊萨](伊萨.md) 合伙把马德高经营成**奴隶城**（她供粮收买周边王国、伊萨武力震慑、家家挂的是奴隶号牌·城堡地下满是囚笼血污）。被苏晓逼问「死亡之地」时，她一面用 [无尽黑暗](../力量体系/装备道具.md) 的威胁挤出半句、一面**给伊萨传信让她趁机逃**、自己留下用拖延 + 假情报（先供出伊萨、再反复问苏晓何时动身去对付伊萨、实为探苏晓还能被她拖多久）断后。苏晓识破后斩了她——「她只隐瞒了伊萨的位置与死亡之地」（得开拓之源 0.3% + 金色宝箱 + 激活「巫猎人」声望任务 1/10）。

## 关系
- **[伊萨](伊萨.md)**：合伙贩奴的黑女巫「好友」（其实被伊萨长期勒索·临死还为她传信断后）。
- **[苏晓](苏晓.md)（库库林·白夜）**：识破其伪善、斩杀她的猎杀者。
