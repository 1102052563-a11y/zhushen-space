# 君主·哈罗德（祖尔帝国君主 · 老阴哔）

[幽鬼大陆](../世界/任务世界/幽鬼大陆-五阶考核.md) **祖尔帝国的君主**——选举制选出、唯一能凌驾于议会之上的人。约 40 岁、短络腮胡、脸色苍白却目光锐利。表面风趣、没有君主的威严，[苏晓](苏晓.md) 一见就判断出这是**超级老阴哔**。

- **遇刺**：被「挚爱」第二任妻子**伊芙**一刀刺穿胸口（与苏晓 / 南茜同遭鬼环操控的觉醒者刺杀·唯苏晓全身而退）。
- **冷酷至极**：为查刺杀真相（关系到地下室钥匙），他**囚禁伊芙全家**（妹妹尤尔 / 双亲 / 弟弟 / 远亲整整齐齐）。苏晓 + 邦妮诈出真相——是伊芙之妹**尤尔**与一只幽鬼相恋、那把削铁如泥的匕首即幽鬼所赠（伊芙被幽鬼能量操控行刺）。查明后他一句「**全部处理掉**」屠了爱妻满门（只「让伊芙走得体面些」）——曾午后陪他下棋的天才少女 [邦妮](伊瑟薇·邦妮.md) 目睹后腿软（那个被她连赢 12 局的风趣君主，竟一句话灭妻满门）。
- **借刀杀人**：他默许苏晓私自围剿鬼环（只给口头嘉奖）；又借苏晓之手杀掉一直想除却不能亲自动的政敌——红矿毒枭 [埃迪爵士](幽鬼大陆配角.md)（混血先驱者）。「这条线以下的官员你看情况处理」。处刑机关地下室钥匙三把分掌（君主 / 南茜 / 副军团长各一）。

## 关系
- **[苏晓](苏晓.md)（库库林·白夜）**：他默许 / 借用的副军团长（借刀清政敌·盯防其与南茜的关系）。
- **伊芙 / 尤尔**：被幽鬼操控行刺的爱妻与其妹（连同全家被他下令处决）。
- **[南茜夫人](南茜夫人.md)**：处刑机关名义军团长·王裔。
- **[埃迪爵士](幽鬼大陆配角.md)**：借苏晓之手除掉的红矿毒枭政敌（混血先驱者）。
