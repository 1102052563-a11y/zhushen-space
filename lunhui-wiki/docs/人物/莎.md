# 莎（恶魔古堡囚徒·半精灵战斗族）

[恶魔古堡游戏](../世界/任务世界/恶魔古堡游戏.md) 一层关押的 48 名囚徒之一——黑发女人、**眼白黑 / 瞳孔白**（与常人相反）、八块腹肌的女汉子（半精灵·战斗族）。野性凶猛如随时暴起的雌豹，是个**近战高手**。被女王 [爱丽丝](爱丽丝.md) 囚禁封印多年，靠摄入「纯净灵魂」逐步剥离封印、恢复自由（一旦试图擅离房间会被狼管家清理）。

## 能力（封印后属性）

`力 80 / 敏 80 / 体 80 / 智 80 / 魅 76 ｜ 生命 35%·法力封印中`——即便如此（已是封印后），[苏晓](苏晓.md) 自评若她不被封印、两人交手他「不被秒杀也是几招内被击毙」。技能高得吓人、但大多被封印 80~96%：近战大师 Lv.67、徒手战斗大师 Lv.63、**野兽直感 Lv.59**、战斗精通 Lv.53、专属主动「**狂蝎摆尾 Lv.69**」（不封印能踢死苏晓绰绰有余）等。她徒手能撕裂苏晓 300 强度的能量盾、指尖快到斩龙闪只斩飞两片指甲。最克她的是 [爱丽丝](爱丽丝.md) 的一道「怪响」——一响她就软倒虚脱、痛苦捂耳（苏晓只觉刺耳）。

## 与苏晓 · 公平交易学技（十四卷8–9章）

她本是「无法交涉」的固执囚徒（既想解封、又不愿听爱丽丝摆布·标准是「实力够强 + 看着顺眼」·苏晓属「够强但不顺眼」），被苏晓接连塞 20 封 [安莉亚的介绍信](../世界/任务世界/恶魔古堡游戏.md) 烦得放他进门暴打——交手后却改主意肯交涉。她**吃纯净灵魂（灵魂结晶）像吃糖豆、享受到颤抖**（被囚多年早丢了强者尊严·苏晓直觉自己像在「投食」）、每吃一颗就剥离一丝封印、恢复一点实力。**公平交易**：付纯净灵魂可学她一种能力——苏晓不贪那些 Lv.69 的生猛主动技（不适合刀客·近战大师倾向拳法亦不合），只选 **「野兽直感」（战斗时增强感知·非预警）+「战斗精通」**。她嫌苏晓「依靠外物（刀）的弱者」——可她锁骨上一道 30cm 刀疤，泄露她曾被刀术强者教育过。

## 教苏晓「直感」+ 伏笔独白（十四卷10–14章）

她以「**战斗是一门学科**」的老道经验教 [苏晓](苏晓.md) 觉醒「[直感](../力量体系/技能.md)」（蒙黑布压制基础感知挨打·点破他一直只用「最基础感知力」、白瞎了心眼这进阶感知）。修炼**有死亡风险**（透支感知力打基础·她原世界禁修·她 17 岁初学时痛哭成泪人）——靠苏晓心眼打底 + 「爱丽丝的馈赠」增益，16 小时初成、再升 Lv.3。她还白嫖了一堆 [布布汪](布布汪.md) 的零食、顺手教布布汪「气息追踪」。临别她把脸贴近苏晓**忠告「小心狼管家；恶魔奴仆能帮你——你身上有恶魔之力、与它同源」**。

**伏笔独白**：她感知到苏晓是「**那些疯子（[灭法之影](../力量体系/技能.md) 一脉）的传承者**」+ 带恶魔之力——这是她「**自由的契机**」。她推断**爱丽丝竟把这种人放进古堡、是在畏惧「某个存在」**（疑指 [虚空灭法者 / 古战场那一脉](../力量体系/技能.md)·「你已忘了差点被谁吊死在虚空台」）；她想**逃到那个存在的地盘获得自由**——「为了自由、付出什么都值得」。

## 重获自由 · 离场（十四卷71–72章）

苏晓通关前夕一层暴动，他来到莎的房间、以「**放你出去 + 放出全部房客**」为条件与她交易（用炼金强酸从外侧削弱木门 / 金属门禁制助攻·从外破坏效率是内侧的几百倍）。莎一拳轰破木门、仰天大笑「老娘出来了」，随即逐间砸门放出其他房客。最终数十名房客合力轰破一层金属门禁制、尽数逃出古堡——**莎就此重获自由、离场**（她原本就把苏晓视作「自由的契机」、这一愿望终于借苏晓之手达成）。

## 重逢 · 苏晓的近战导师（二十一卷81章）

暗鸦星返园后，苏晓用 **混沌勋章** 在试炼场召「宗师级 Lv.45+ 近战肉搏导师」，召来的竟是**已解封的莎**（一传送来就偷袭把苏晓按地上、笑「奇缘」）。她已剪短发、封印全解、实力大涨；但**因得罪奥术永恒星的人、侧腰中了一道腐蚀生命力的黑色伤**（活不久了）——遂与苏晓做「契约外的交易」：教他近战、换苏晓**半年内带她进轮回乐园避难**（像里德先例·苏晓凭猎杀者权限可办到）。苏晓用青钢影**噬灭大半伤口能量**给她续命（灭法者全清反而会让她死更快）。十小时特训里他的**近身肉搏专精从 Lv.40 升到 Lv.44**——莎的招牌正是侧踢类的「狂蝎摆尾」（解封后 Lv.69），与苏晓的木之灵侧踢路子相合。

## 关系

- **[苏晓](苏晓.md)**：先付纯净灵魂学其「直感」、后在暗鸦星返园成其近战导师（教近身肉搏·求带入乐园避难）。
- **奥术永恒星**：得罪的虚空势力（侧腰中腐蚀伤·苏晓为她续命）。
- **[苏晓](苏晓.md)**：付纯净灵魂学她「直感」的契约者（嫌他用刀、却肯与他交易·视其为「自由的契机」·末了被他放出古堡）。

## 黑渊重伤求救 · 入轮回乐园（二十三卷4–34章）

被奥术永恒星的人一路追杀、重伤 + 中毒 + 魔能侵蚀 + 大面积骨折，她拼死赶到 [黑渊浅层](../世界/任务世界/黑渊浅层-赔罪者之殇.md)、在阿萨卡发委托联络上 [苏晓](苏晓.md)。苏晓以「帮她弄到 [灰烬之芽](../世界/任务世界/黑渊浅层-赔罪者之殇.md)」为条件**给她做手术解毒疗伤**（她在手术台上疼到求饶差点哭出声·「这事谁提谁死」·治疗比拷问更可怕的名单上又添一人）——恢复至**五阶战力**（全盛约七阶·非进乐园难复巅峰）。随即并肩杀入始祖地下城、抱摔拖坦决战 [圣扎迦](圣扎迦.md)。事后苏晓用「生灵花」把她传送进轮回乐园（坑·她被传到黑渊最底层生灵花海、刀魔近在 20 米·吓得不轻）——她由此成为轮回乐园的**「引导者·莎」**、住试炼场，苏晓持「莎之勋」可每 15 天免费找她学一次近战（特训凶狠·把苏晓近身肉搏专精拉到 Lv.50·锤得他几乎丧命却进步飞快）。

## 🆕牵线暴鼠 / 癞蛤蟆求救 Saber（四十一卷82章）

魔海通关返园后，她（此时已恢复、正在雪山苦修）替与她同权阶的**暴鼠 / 癞蛤蟆**向 [苏晓](苏晓.md) 带话——暴鼠 / 癞蛤蟆培养的裁决者 **Saber（阿尔托莉雅·潘德拉贡·圣杯世界）** 去渊龙底接受龙之试炼、掉进黑泥被**黑暗物质**侵蚀（几个月内必死）。她说漏了「癞蛤蟆与刀魔的恩怨」被苏晓叫停（听了会死）。苏晓开价 **格林·吉莉安佩刀的刀尖**（史上最恶劣的女灭法者）才肯出手祛除黑暗物质（凭炼金师功底·分 3~4 次·过程比治她本人更痛苦的「撕扯拉扯」·防止把灵魂 / 意识扯出）——下个世界进度结束后动手。（顺带证实：她当年潜力透支后的神秘恢复，正是苏晓这位「药师」所治。）

⚠ **有段时间未见，莎的气息已格外黑暗、有强烈侵略性**——⚠ **她穿黑色背心，后背与双臂有黑色纹线，眼底与瞳孔都漆黑一片**;⚠ **⚠ 她一向追求力量，可到八阶巅峰后到了极限**（⚠ 这是她曾不顾身体伤损、不断强行突破极限的代价·若非苏晓多次治疗，她甚至命不久矣）——⚠ **对一个以追求力量为目标的强者，这很痛苦，因此她选择了深渊系。**

- ⚠ **她不是鲁莽直接吸收深渊能量**:⚠ **显然是获得了某种深渊系的力量传承体系**——⚠ **以此突破了自身极限、得以继续追求强大**;⚠ **⚠ 就算如此，风险依旧极高。**
- ⚠ **苏晓的态度**:⚠ **不干涉，最多提醒**（⚠ 「你或许会失控」·莎「我不会·放心吧，老娘命硬的很，况且，在追求力量的道路上死掉，又有什么可惜的呢?」）——⚠ **每个人都有自己的选择**;⚠ **⚠ 苏晓抛给她一瓶能暂时遏制深渊之力侵蚀的药剂，她只触碰就知效果很强，未道谢、如往常般背对苏晓挥挥手消失在街角。**

## 🎬 一场演技大比拼（52卷32章 / 36章）

⚠ **苏晓一眼看穿这不是死战**——⚠ **判据:① 主线任务对她给出了「击败**或**击杀」的额外选项（前两环都明确标注「击杀」）② 她逮住 [瑟琳](女巫界/瑟琳.md) 却手下留情（内心若已被黑暗吞没，瑟琳必死）③ 她主动找上门太反常**（⚠ 她迷醉于死战，却没有找死的爱好）;⚠ **⚠ 结论:她在装作「承受深渊力量后判断力大减、被力量所操控」的模样，因为暗中还有双眼睛盯着她**——⚠ **⚠ 「这次不是死战，是一场演技大比拼。」**

- ⚠ **她的时间系能力**:⚠ **「加速」＋「回溯」**——⚠ **每次激活可对自己进行约 1.5 秒的时间加速（能把 1.5 秒压成几毫秒的突进）**;⚠ **⚠ 起算 3 秒内还可「回溯」，把自身时间逆回 3 秒前**（⚠ 既位移、也消除这 3 秒内承受的伤势与负面状态）——⚠ **通俗讲就是能「存档」与「读档」**（⚠ 3 分钟冷却）;⚠ **⚠ 她的炽红晶质双臂是体系核心:血液温度越高，力量/速度/防御越强。**
- ⚠ **苏晓的「稍微放水」**:⚠ **没解封印环、没用弑/流/极、没用灵魂感电、没激活时·支配者、没用极刃·世界、没用飞升踢、没用死寂烬灭、没用青影王、没用天怒·奔流斩**——⚠ **「嗯，没错，稍微放水而已。」**
- ⚠ **⚠ 她在薅羊毛**:⚠ **她意志力极强的三个原因——技法宗师／天赋就是「巨量」提升意志力且随实力成长／长期接受苏晓的治疗**;⚠ **⚠ 这次「死战」侥幸活下来，三位 [议会长老](女巫界/议会长老.md) 只会拿出更多「纯净深渊之力」让她继续变强**——⚠ **等羊毛薅够，她再剥离体内未吸收的「黑暗之血·意志」脱身**（⚠ **她渴望力量但不傻:纯净深渊之力是无主之物、两种无上力量之一，而吸收黑暗之血面对的必定是死局**）。
- ⚠ **交出黑暗之血（36章）**:⚠ **她在天空城顶城的楼顶抛给苏晓一个封印晶体瓶——【黑暗之血·意志】，主线第三环完成**;⚠ **⚠ 抬手左右摆了摆便顺势坠落、转而出现在街道上**——⚠ **她走向顶城边缘，因为下方的底城深处有什么东西在吸引她**;⚠ **⚠ 她还托黑雾身影送来【爱丽丝的挚爱耳坠】——不是要苏晓顾及旧情，而是在表达:她还有清晰的自我意识。**
