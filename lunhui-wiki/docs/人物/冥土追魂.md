# 冥土追魂（神级医生·学园都市善势力）

[魔法禁书目录](../世界/任务世界/魔法禁书目录.md) 学园都市里**相对温和的「善」势力**首脑——青蛙脸的秃顶中年医生（圆脸微胖·憨厚温和），**医术高超到不可思议**（能凭局部麻醉做心脏手术·接断臂在他看来只是小手术）。

- **学园都市的另一极**：很多年前 [亚雷斯塔·克劳利](亚雷斯塔·克劳利.md) 被魔法侧重伤活埋，**正是冥土追魂把他挖出救活**；学园都市的组建他也是关键人物。原著里敢直接阻碍亚雷斯塔计划的人不多、他是其一（亚雷斯塔只淡淡一句「我有善良心肠的『敌人』」便作罢）——他不参与管理、却一旦出声连亚雷斯塔都不能无视。他视学园都市为「科学发展的先驱」（而非亚雷斯塔眼中的「大型试验场 / 疯狂计划的基石」），不愿亚雷斯塔死（否则这里立刻沦为战场）。**学园都市并非亚雷斯塔一家独大**——他与亚雷斯塔是「善 / 恶、温血 / 冷血」的两极、亦敌亦友、互不干涉却共享技术。
- **为苏晓接断臂 + 试探拉拢（十三卷18章）**：[苏晓](苏晓.md) 与神裂火织死战断臂、由亚雷斯塔引荐来此——他**完美接回苏晓的断臂**、愈合胸伤（其医院无滞空回线·亚雷斯塔监控不到）。双方互相试探，他暗示苏晓去「第十二学区神庙」（拉拢之意·想要苏晓这员强战力）。苏晓婉拒：处事风格差太多、以自己的气息 + 6 魅力加入「善」阵营不明智，更愿与「恶」的亚雷斯塔合作。

## 关系
- **[亚雷斯塔·克劳利](亚雷斯塔·克劳利.md)**：当年被他救活的「敌人」（善 / 恶两极·亦敌亦友）。
- **[苏晓](苏晓.md)**：他接好断臂、试图拉拢的强力契约者（被婉拒·愿合作不愿入伙）。
