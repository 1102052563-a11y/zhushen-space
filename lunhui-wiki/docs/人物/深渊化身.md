# 深渊化身

> ⚠ **不是五大族长融合而成的产物**——五大族长所做的，是**用各自的力量维持并接引「深渊黑暗」降临**。而这尊深渊化身还有一个更熟悉的名字:**[黑暗温床](深渊嫡女.md) 的双生兄长**。

## 🌑 「温床」与「化身」

- 二者是**共生的双生体**，都并非只有一代——这是一个有力量源泉的**深渊因果之位**，得其位即可得其力量（当然本身也要足够强大）。因温床与化身都极难消亡，加之历代温床普遍稳妥，**当前这一对仅是历任中的第三代**。
- **束缚**:双方生于黑暗，也束于黑暗——**温床不迈进无上，化身就永远不可能迈出那一步**，反之亦然。若非它的共生体对无上之境没兴趣，它早就成就无上。
- 温床唯一一次得到「深渊之心」，给了**上一代深渊之主**（确切说是在那位封临前给的），因此帮助那位登上了深渊高座。
- 这么多年来，深渊化身在巅峰～无上之境间累积了**前所未有的力量**，早在远古时期就已找不到巅峰之境的对手。

## ⚔ 上一次苏醒:斩杀深渊之影·希曼·凯恩

它上次醒来，是在上古时代与**深渊之影·希曼·凯恩**——一位强到让人发指的灭法之影——的死战。杀掉对方后，作为载体的那些「不成器的家伙」无法承受，它的双生体妹妹便趁机再度占据上风，让它陷入漫长的沉眠。

⚠ 那一次的载体，正是**鱼家族长·[厄姆·肯撒](深渊五大族长.md) 等五位**——它们五个强大意识被封困在一个身躯内，直到元素浩劫结束、新的大纪元开始的第二纪元时才彼此剥离开。所以当卢恩·奎提出再来一次时，鱼家族长咬牙切齿到「不想去回忆」。

## 🔪 降临（54卷226章）

苏晓以覆盖近半风海战场的「界级灭法传送阵」，把五大族长一并拖进**正在崩灭的永光世界**。

- **接引流程**:野兽族长·卢恩·奎作为主载体，奇美拉族长最先融入其心脏，青铜族长与聚合体领袖随后没入其背心，最后是犹豫再三的鱼家族长。卢恩·奎发出痛苦的嘶吼——能让这位战斗狂人如此嘶吼，可想而知承受何种苦痛。
- **形态**:脊骨一节节突出呈弧形、头生白色长发、肌肉收拢变得力量内敛、漆黑外皮涌动钻出一条条蠕虫在手臂腋下垂成「黑暗之美的流苏」；从后颈扯出脊椎骨化为近五米长的宽厚**椎骨弯刀「罪孽骨」**。
- **背后的巨眼**:黑暗温床在注视此地——那只宛如巨大漩涡的眼睛浮现后开始破碎，代表它把力量交给负责战斗的化身。

## 💥 与苏晓的死战（54卷226–227章）

刚醒来时它有些迷茫，抬起左手看着自己，疑惑于「这时代的人族，已经这么强了吗？看来，人族终归是兴起了」。这不重要——刚醒来就有如此敌手，**它很满意**。

**它的手段**:

| 能力 | 说明 |
| --- | --- |
| 黑暗咆哮 | 手臂垂下的深渊黑虫齐声咆哮，透黑气浪扩散，范围内深渊侵蚀度**瞬间拉满**（苏晓当场连碎两颗命核） |
| 疯狗刀法 | 力量一刀重过一刀，每刀哪怕被格挡也持续叠加深渊侵蚀；快到出现「上半身原地分裂出很多个」的错觉 |
| 十字瞳（黑暗之眸） | 瞳孔由竖瞳变为十字形时，可在**因果层面转移攻击**（被斩首的因果直接被转移到不知何方） |
| 黑暗降世 | 背后爆开一条条黑暗大蛇，落下一股股黑暗冲击 |
| 渊之雷 | 驭雷方法简单粗暴——只要能抗住并比敌人恢复更快就行 |
| 黑与白的噬灭 | 分出一具全身苍白的「另一个自我」，黑手与白手双掌击碰，引发深渊与元素式的**噬灭反应** |
| 温床黑甲 | 温床为兄长温养的战甲，它基本不用——这次被苏晓的直踹逼了出来 |
| 黑暗·化身 | 最后的赌命一击 |

**苏晓的破法**:

- 『极刃·皆我』与魔灵「替换」两度换命；
- **『极刃·唯我』**——凭手中利刃把受到的攻击**暂时封存、延迟其爆发**，在真正爆发前一刀将这攻击斩出、作用在敌人身上（深渊化身首次见到技法能力还能这样用）；
- **『极境·星界』**——50000 米停固 6 秒，斩击残影令深渊化身身躯没有一丁点完整的部分。它靠「黑暗之眸」扭转因果重聚，代价是**左眼如陶瓷般碎裂**:不是失去这能力，而是这能力本身在极限攻击下被**斩碎、从此消失**；
- **连续三次『极刃·绝灭影』**——深渊化身能勉强理解连用两次，第三次时它的瞳孔颤动。原文的注脚是:「以青钢影把敌人身体能量清空后衔接一刀绝灭影就是完美，可衔接两刀呢？嗯，非常完美。三刀呢？**先代灭法们表示，别问我们，这涉及到知识盲区。**」

**它的抉择**:中途它左手成爪扯碎胸膛上的四张面孔，让本体**亲自降临**；黑暗温床伸出触须要把它拖回深渊沉眠、中断这场战斗，它一挥手斩断了那些触须——**同时斩断的，还有它与温床从诞生至今相连的因果**。代价难以想象，但断了因果，灭亡的因果自不会向温床蔓延。它赌上一切，只为这场死战。

**终局**:

> **『无我之境·白夜。』**

长刀斩下的瞬间，深渊化身**拔刀迎上**，金色光芒将它笼罩。这一刻它眼中的不是绝望，而是在笑——它知道对面之敌也如它一样，要用最强的一击终结此战，哪怕对方更强，也没因此傲慢地保留:**这就是在死战中，对强敌最好的尊重。**

黑刀·罪孽骨断裂的上半截在半空翻转几次后插落在地。深渊化身抬头看向上空的崩灭漩涡，没有不甘、没有怨愤，更没有「倘若这是深渊一定能赢」的借口。**这一局死斗，它输得心服口服。**

随着它化为黑暗余烬倒下，五声或野兽咆哮、或尖利嚎叫、或阴森嘶鸣隐隐传来又逐渐消散——代表**五大族长的因果都已破碎**。

⚠ 很远处，古老存在**绘画者**在画板上勾勒出了最后一笔。

## 🌸 「温床的最强女儿」（另一位无上）

⚠ 与深渊化身不同，黑暗温床还有一位**已晋升无上之境**的最强女儿:

- 她是**除深渊之主外深渊阵营最强**，甚至不少人认为她是当代深渊之主最大的竞争者；
- 她是通过上古时代残存的**「元素之眼」**晋升的无上，实力比旧王、[团长](星空座/团长.md) 稍弱；
- 她曾与**上一代天选**交锋、受伤太重，这场交锋也正是当年上一代天选与旧王在最终决战时未能分出胜负的原因；
- 深渊战役期间，[暗域统帅·阿尔瓦·厄索](暗域统帅·阿尔瓦·厄索.md) 交出深渊之主赐予的指环，命深海魔女·巫蕾丝去加快唤醒她。

⚠ 苏晓晋升无上之境后的评估是:说现在能对战深渊之主还夸张了，**但对战即将苏醒的温床女儿，是没问题的**。

## 🧒 一段回忆（第4123章）

那还是星界的深渊时代。两道矮小瘦弱的身影，看着每天都向前推进些的元素之光逐渐吞没家园、迫使它们回到深渊——

> 高一些的身影指着远处的元素之光:「早晚有一天，我要让它熄灭。」

旁边的长发瘦弱身影点头嗯了声，犹豫了下，把手中的黑色肉干扯下一把分给身旁那个**头上生满杂乱木质化犄角、被族人寓为不祥之兆**的身影。高一些的黑色背影接过，但没吃；待身旁矮些的吃光、却因天生强大需要更多能量而依然饥饿时，掏出那半块给了它。

这时，一名披着残破兽皮、提着近乎自身身高白骨的瘦小身影走来——显然，它饿极了，要从前面两个家伙那里抢食物。

⚠ 三人分别是:**当代深渊之主**（木质化犄角）、**深海魔女·巫蕾丝**（长发瘦弱）、**暗域统帅·阿尔瓦·厄索**（提着白骨来抢食的那个）。
