# 首席神职·伯格曼

> ⚠ **[罪亚斯](#) 的老丈人、奥娜的父亲**;⚠ **⚠ 形态**:⚠ **身材偏瘦、大骨架、身高一米九以上·鹰钩鼻·满是咒文刺青的秃头·满口金属牙·眼底漆黑而眼瞳暗黄**——⚠ **气质宛若随时可能死掉**;⚠ **⚠ 昔日是位文武双全的古神学者**:⚠ **属于「当无法以知识解决问题时，他也略懂一些拳脚」**——⚠ **直到他遇上了夜母麾下的大主祭·乌露洛伊**（⚠ 婚后他常偷瞄自己身高五米、体重9572公斤的「娇妻」感慨:「咱们的初次见面，我到如今都感觉，我的生命力顽强，没被你单手掐死。」——**发现妻子目光不善后便识趣闭嘴，毕竟没生气的妻子最起码是人形态的，而不是大量黑紫色触手构成的古神大主祭**·⚠ 甚至让人怀疑，当初他是不是被乌露洛伊逮去改良夜狼家族基因的——**从奥娜的颜值来看，改良得非常成功**）。

## 🍷 死中求活的一杯酒（54卷171章）

⚠ **他被夜母阵营派去在 [眼之公主](眼之公主·洛丽丝.md) 脑中种下「寄髓虫」**——⚠ **苏晓当场以魂体左臂探入眼之公主脑中揪出虫子，再以血气线虫顺着因果反向追踪，找上了他**;⚠ **⚠ 他的反应是摘下兜帽，气息没有半点恐惧，只有释然**:⚠ **「看来，我今天没可能活着离开了。」「作为死前的愿望，我能喝一杯吗。」**;⚠ **⚠ 他走进街边酒馆，里面的酒客忽然齐齐起身、迎着他向馆外走去**——⚠ **他到吧台内认真挑选了很久:**生命中的最后一瓶酒，当然要认真****（⚠ 苏晓一进门就在桌上放了一瓶元素佳酿，他眼睛一亮:「那混小子送过我一瓶，啧～，滋味至今难忘啊，人生最后一瓶美酒，这瓶佳酿配得上。」）;⚠ **⚠ 这副赴死姿态，实际上是死中求活**——⚠ **作为大主祭的丈夫，他太清楚失去价值后将面临什么**（⚠ 这一家子六个人，五个对夜母的信仰都非常不虔诚；而大主祭·乌露洛伊对任何家人都没有亲情、是家族至上，对应的『回报』是她的五名亲人对她也都没有过多亲情）。

- ⚠ **⚠ 他提供的关键情报**:⚠ **能激活「深渊之眼仪式」的，放眼整个陨灭星只有两人——大学者·艾德罗与 [传奇巫师·索耶](传奇巫师·索耶.md)**;⚠ **⚠ 关于艾德罗为何不可能帮忙**:⚠ **第二纪元时，一名女灭法有事找到这位德高望重、眼之仪式体系最巅峰的人物，对方展现出了足够的高傲**——⚠ **结果被那女灭法装进沙袋打了一上午；打完之后，他在生命安危与无偿帮助之间，明智地选择了后者**（⚠ 苏晓的回应是「必要的话，我也可以把他装进沙袋」，吓得伯格曼连忙推荐了索耶——**「那老东西现在是徒有其名，索耶才是现在的眼之仪式最高学者」**）。
- ⚠ **⚠ 他的开价**:⚠ **「把我的孙子·格里芬送到万族界，而且，他要在这超脱之界的**世界因果上有一席之位**。」**——⚠ **此刻他的眼睛非同寻常地明亮:他已做好给夜母阵营、甚至给陨灭星陪葬的准备，但不能让孙子也死在这**（⚠ 至于宝贝孙女 [耶萝·伊莉亚](#)，他早就让顶尖占卜家洞悉过——**她的命运不要去胡乱干涉，那或许是下一个纪元最大的几个因果之一**）;⚠ **⚠ 「我这老东西，已经准备好，给那帘幕后的漆黑烂肉陪葬了，呵呵呵……」**（⚠ 交涉结束后他脸上的笑容才收敛，长舒一口气、感受已湿透的后背，拿起酒瓶灌了半瓶压惊——**「只要踏错一步，他的生命立刻终结；这就是能和鹿神死战的灭法之影所拥有的压迫感」**）。

## 🎁 苏晓的兑现

⚠ **苏晓联络地精商会总经理·金古沃，通过商会渠道把格里芬送走**——⚠ **他自己送更简单快速，问题是「到了后格里芬还有没有气息，得看格里芬的生命力」；反观地精商会的渠道全程要4天，但胜在安全舒适**;⚠ **⚠ 关于「世界因果一席之位」**:⚠ **苏晓的办法是亲自去一趟万族界，凭「原罪之书」的因果体量暂时接触到「万族界·世界因果」，然后**薅下一大块给格里芬吃了****（⚠ 伯格曼听完沉默了至少一分钟——他原本的想法只是让孙子「受到些世界因果的洗礼」、能触碰到就行·⚠ 他随即追问「你不怕万族界的因果反噬吗」，结果这一问直接问到苏晓的知识盲区，苏晓沉默几秒后反问:**「什么是因果反噬。」**——这句话的威力堪比顶级「沉默系能力」，让伯格曼又一分钟没吭声·⚠ 他转念一想也是:这是「原罪之书」的持有者、星界唯一还活着的灭法之影、万族界的缔造者、死寂白王、永光主宰——**怎么可能会被因果反噬呢**）;⚠ **⚠ 后续加码**:⚠ **他又替苏晓办成了两件事〔告知蜘蛛夫人在隐秘之城并与夜母结盟；半小时内把一部通讯器送到冥神面前〕**——⚠ **报酬涨到「让你孙子多吃掉一份因果，占两个万族界的因果席位」**（⚠ 他的原话是「我早晚死在你们这一人一鸟手里」·⚠ 而他妻子乌露洛伊被苏晓斩杀后，他在通讯里**没有半分悲伤感**）。
