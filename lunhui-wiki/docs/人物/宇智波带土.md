# 宇智波带土（阿飞）

[火影忍者世界](../世界/任务世界/火影忍者.md)「**晓**」组织真正的幕后黑手之一——表面以吊儿郎当的「**阿飞**」身份混入（戴橙色面具、自称「宇智波斑」），继承了宇智波斑的庞大遗产。九卷52章他作为 [苏晓](苏晓.md) 夺得的卡卡西左眼写轮眼的「买家」登场（已用 [蝎](蝎.md) 遗下的戒指成为晓正式成员、与 [迪达拉](迪达拉.md) 组队）。

- **「阿飞」的真相**：少年时他右半身被巨石砸碎，由宇智波斑注入**柱间细胞**保命、并用一具**漩涡形白绝**包裹体表代行动作。「阿飞」插科打诨的声音其实是这具**漩涡白绝**在说话（带土本体平时不出声、只偶尔用空间能力）。晓内只有 [长门](长门.md)、[小南](小南.md) 知道阿飞的身份。
- **空间忍术（神威）**：可令自身**虚化**（攻击穿体而过）、瞬间空间转移、**把目标吸入异空间**（被吸入即死/困死）。柱间细胞令其生命力极强、大幅减轻万花筒瞳术对身体的负担。漩涡白绝的木遁因与带土查克拉轻微排斥而只能用简单的（「树界降临」缩成一两米小树）。
- **与苏晓近战吃瘪**：交手中带土靠虚化游斗，却始终被苏晓的「[傲歌](../力量体系/技能.md)」能量盾 + 影刺逼停（一旦恢复实体就被刺穿）——他评「与你近战不是明智的选择」，并感慨「调查过你却查不到、你像凭空出现」。
- **交易原始柱间细胞**：他本可强夺卡卡西左眼却没动手——因那只眼是**野原琳**从他身上移植给卡卡西的、是琳存在的象征（且他另有可发动「伊邪那岐」保命的写轮眼）。最终他以**原始柱间细胞**（斑的遗产·活性 95%）+ 一件「私活」换苏晓手中的血轮眼；私活 = **帮他把那对兄弟中的弟弟培养成「满心仇恨、又足够强的棋子」**（即图谋利用宇智波佐助）。交易后他直接消失（其实根本不在乎柱间细胞、皮肤一碰就险被吞噬树木化）。

## 旁观兄弟对决（九卷59–61章）
封印尾兽时他（阿飞）起哄要拉佐助入晓，被鼬「天照」黑焰烧（其身份被鼬看破）；事后他用本体声音与苏晓交代「**之后的事交给你，作为柱间细胞的报酬**」——即让苏晓促成 [宇智波佐助](宇智波佐助.md) 杀鼬。他笃定鼬绝不会反杀佐助（「鼬为弟弟可灭尽家族、唯独舍不得杀他」）。佐助赴宇智波遗址决战鼬时，他与 [干柿鬼鲛](干柿鬼鲛.md) 拦下 [香磷](香磷.md) / 水月 / [重吾](重吾.md)、和苏晓 / 黑白绝一同**旁观**（约定「我们两清」）。

## 收佐助 · 揭真相（九卷62–63 / 69章）
鼬 vs 佐助决战时，他与苏晓打赌「佐助胜则鼬双眼归我」（鼬之眼关系到他一个**宏大计划**），并和黑白绝 / 布布汪一同旁观、解说战况。鼬力竭而亡后，他**肩扛佐助、单手按鼬尸体**用空间能力一并带走（约定鼬尸他处理、大蛇丸的小白蛇归苏晓）——意图把 [宇智波佐助](宇智波佐助.md) 收为晓的新成员。在晓据点他向佐助层层揭露鼬「为保护你而扮恶」的真相，把佐助说得信念崩塌、怀疑人生（苏晓在旁「静静看你装逼」）。

## 招揽苏晓 · 自称宇智波斑（九卷69–70章）
他向佐助揭真相、逼佐助**开出万花筒写轮眼**（六芒星·恨意转向木叶高层）后，转而招揽 [苏晓](苏晓.md)：直言不让苏晓参与捕八尾 / 九尾（苏晓抓的尾兽都半死不活、长门忍耐到极限），提出**「帮我做事」并自称「宇智波斑」**（点破晓真正的首领既非他也非佩恩、而是已死的「弥彦」·这是他与佩恩的约定）。苏晓以「需观察验证」未即答应。他透露**佩恩（长门）与神秘势力战斗身受重伤**（试探苏晓），并察觉「苏晓的目的不止捕尾兽、与长门有关」。其那只写轮眼是**空间系**（本是他自己的眼）；他在犹豫是否移植另一只写轮眼以恢复须佐能乎等瞳术（左右眼一防一攻威力极强）——这一切都服务于他的「**月之眼计划**」（伏笔·他利用长门、斑利用他、黑绝利用斑·层层相利用）。

## 接任晓首领 · 三方月之眼计划（十五卷5–8章·苏晓再入火影）

[长门](长门.md) 被鸣人感化而死后，带土**正式接任晓组织首领**（仍打「宇智波斑」旗号·已移植 [卡卡西](卡卡西.md) 的写轮眼〔神威空间漩涡绞杀力〕）。他主动邀苏晓当首领（论资历苏晓不输任何人）、被苏晓婉拒（苏晓选拥护·当首领=拉仇恨当活靶子）——晓内带土真正信任、能共同执行计划的仅苏晓。揭开 **三方共谋的「月之眼计划」**：带土（造有野原琳的幻术世界）、宇智波斑（利用带土当祭品复活自己）、黑绝（终极是复活母亲辉夜）——三方初 / 中 / 后期相同（轮回眼 → 十尾 → 无限月读），唯终极目的不同（详见 [火影忍者·忍界公敌](../世界/任务世界/火影忍者-忍界公敌.md)）。

计划当前卡在**轮回眼**（长门尸体 + 轮回眼被失踪的小南藏起·下落只她知）。带土把 [佐助](宇智波佐助.md) 当重要棋子培养（靠仇恨催熟）；验出佐助抓来的奇拉比是替身后拧断其脖颈、现出章鱼触须。白绝传回「五影大会」情报后他果断改计划——全员扑铁之国五影大会、伺机袭杀火影 / 雷影 / 水影搅乱局势（暂不想让五大忍村真联合·轮回眼半年内不会坏、五影大会更紧迫）；这与黑绝「轮回眼优先」的意见起了分歧。

## 五影大会 · 宣战 + 收买苏晓（十五卷10–15章）

会前他找苏晓商定——五影大会无论结果都会达成「灭晓」共识，故他要**趁机向五大忍村宣战**（先发制人·宣战≠开战·晓在暗、开战时机由晓定·气势不能输；这比原著「得大蛇兜相助后才宣战」早得多）。他全程不愿亲自出手（**土影曾与真宇智波斑交手过**·一旦动手有暴露伪身份的风险·获轮回眼前绝不轻易出手）。眼见刚觉醒须佐能乎的 [佐助](宇智波佐助.md) 要死在雷影手下、这枚「升值的棋子」不能丢，他**用一卷柱间细胞资料（部分·余者五影大会后给）换苏晓出手保佐助**——其实苏晓真正盯的是落单的水影 [照美冥](照美冥.md)。

## 摊牌月之眼计划 · 第四次忍界大战宣战（十五卷21–24章）

雷影死后他以神威空间现身会议室（虚化无视油女毒虫苦无）、自称「宇智波斑」、向四影**摊牌「月之眼计划」全貌**：九尾兽是「十尾」被六道仙人分散的查克拉、十尾躯壳即月亮；他要复活十尾、成其人柱力、把瞳术映上月亮发动**无限月读**、把全人类合为一体（藏一句私心——造一个有野原琳的世界）。他坦言抓尾兽 / 聚五大忍村忍者，都是为凑发动无限月读的海量查克拉。四影一致反对、拒交剩余两尾——他遂宣布「**第四次忍界大战，就此宣战！**」（比原著「得大蛇兜相助后才宣战」早）。他临时改主意把雷影头颅物归原主（佐助的须佐能乎更值得养·让佐助与团藏实战即可），与苏晓分头：苏晓盯照美冥、他盯团藏、火之国据点会合。他视晓成员皆为完成计划的棋子。
