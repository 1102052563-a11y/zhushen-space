# 南茜夫人（处刑机关军团长 · 王裔毒蛇）

[幽鬼大陆](../世界/任务世界/幽鬼大陆-五阶考核.md) [处刑机关](../势力/index.md) 名义上的**最高统领（军团长）**——约 30 岁、黑纱裙 + 薄面纱、气质出众的**嫡系王裔**。无战斗力、却绝非花瓶（小看她的上两任副军团长下场都很惨）。她凭王裔身份为处刑机关争取「额外」经费，是处刑机关的**「财神」**；[苏晓](苏晓.md) 评她是「**母毒蛇**」——只要他露丝毫破绽，她就会死死咬住、置之死地。

- **从内斗到合作**：王裔失踪案（前任副军团长杀王裔）后，她本想换掉副军团长、第一时间带觉醒者抄苏晓住宅扑空，又半夜登门试探——见苏晓滴水不漏只得作罢。她自己也遭鬼环觉醒者刺杀（浴室中险死、靠诡兽替死），遂转为与苏晓合作「找出鬼环杀干净」。
- **正确的合作方式**：苏晓不需要帝国功绩（职务特殊、无法升迁），遂把杀敌功劳全甩给她换 **2 万金勒**（来路要干净）+ 让她出面背锅善后；她得功绩、苏晓得钱与行动自由——各取所需。她机敏地察觉祖尔城局势异常、果断停内斗（处刑机关此刻更需要苏晓的武力镇场）。
- **风险红线**：君主 [哈罗德](哈罗德.md) 暗中盯死「正副军团长是否滚床单」——一旦二人有肉体关系即代表处刑机关可能脱离掌控、哈罗德会不惜代价除掉二人（处刑机关政变是颠覆帝国的危险信号）。

## 真面目 · 鬼环合作者 · 伏诛（二十四卷48章）

[苏晓](苏晓.md) 一进世界就让 [布布汪](布布汪.md) 在她家中 / 衣物上装窃听器、最终揪出她的真面目：她**一年前就与 [鬼环](../势力/index.md) 合作**——约书亚·埃迪出钱、她凭王裔身份**帮幽鬼打通入城渠道**（否则罪恶 / 月蚀带不进那么多幽鬼）、并**陷害前任副军团长利·克拉塞**（职务犯罪案的幕后·做得天衣无缝）。她劝罪恶把眼室布在祖尔城「除掉哈罗德」，自己好顶位**当女王**（鬼环并不想颠覆帝国体系、她才是真有野心者·「有时人比幽鬼更可怕」）。苏晓用「永久增益药剂能让她返老还童」诱她吐露全盘——她想拉苏晓入伙（让位换琼斯的议员位），却不知君主 [哈罗德](哈罗德.md) **藏在落地镜后听了三小时**。哈罗德假意放她去风海城养老、实则当场令人**一枪打爆她的脸**（为追随多年的琼斯议员报仇·她临死还想保住容颜服毒而不得）。
