# 老黑

[血门冒险团](../势力/index.md) 的**中层成员**——一名络腮胡中年男人（皮肤略黑、故称「老黑」，非黑人），自有几分傲气。一行六人（五男一女）乘空间压缩携带的**三排座装甲车**，在 [二阶生存试炼·吞噬岛](../世界/任务世界/T12K星球-生存试炼.md) 中行动。

- **憋屈差事**（八卷50章）：此行实为给 [铁大师](铁大师.md) 当保镖——团长斯坦的死命令压顶（铁大师出意外就提头来见），逼得他事事听这弱小老头指挥、忍气吞声（论实力他能把铁大师锤成肉饼，却半根毫毛都不敢动）。一行人已一天没进食、断了粮，正急着离开红土区。
- **与苏晓擦肩**：本想抢 [苏晓](苏晓.md)、[布布汪](布布汪.md) 身上的肉干淡水，被铁大师喝止；后铁大师反要下车抢斩龙闪，又是老黑冷汗直冒地拦住。女契约者探出苏晓「三维问号」的可怕资料后，他**当机立断带队逃离**。苏晓懒得追击（生存试炼猩红卡掉率低、对方缺粮无油水），只留一句「车不错，希望之后能见面」。

## 结局 · 死于堵桥（八卷53章）
逃离苏晓后赶往红土区唯一出口「石桥」，却撞上一队契约者**堵桥三天**（专杀从红土区脱出、饿到虚脱的契约者）。缺粮虚弱的血门小队躲在装甲车后被压制，敌方冲上来近战——**老黑等数十秒内尽数倒在血泊中**，[铁大师](铁大师.md) 也被爆头扔下悬崖。一行人全军覆没。（杀他们的堵桥队随即被赶到的苏晓反杀。）
