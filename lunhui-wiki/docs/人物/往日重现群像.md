# 往日重现群像

**【往日重现】** 是 [团长](星空座/团长.md) 耗尽毕生财富准备的底牌之一，形态是一把十多公分长的钥匙——出现的瞬间，周边的时间、因果、存在等力量都剧烈波动。

⚠ 它带回的不只是第二纪元的 [先代灭法者们](先代灭法者们.md)，还有整个**第一纪元的星界霸主**。

## 🐗 大领主的狂兽人军团

虚空大本营正面的上百个金属升降巨门同时开启后涌出的，不是一支狂兽人大军，而是**狂兽人浪潮**——它们身下战兽的利爪带有空间特性，能抓着空间在半空中如履平地般狂奔，因此是**铺天盖地**地冲锋。

**狂兽人存在的意义只有两条**:忠于大领主；杀戮殆尽大领主的一切敌人。

| 机制 | 说明 |
| --- | --- |
| 爆兵来源 | 大领主们在虚空大本营内建立了 **「领主核心」**；「领主要塞」吸收所在世界的**无主生命力**构成「能量晶石」以驱动它 |
| ⚠ 为何只发展狂兽人骑兵 | 既因狂兽人与战兽的战斗基因强大，也因这会**提升几倍的开发效率**；⚠ 第一纪元几位最强大领主在没决裂前，**会毫无保留地共享「狂兽人基因」的研究成果** |
| ⚠ 顶级生命工程 | 战场上第一只被杀的狂兽人**当场死而复生、气息更强一筹**——它们的本源生命力已被开发成另一种东西，如同把铁胚百炼成钢，**必须不断杀死它们，直到生命源核到极限才会破碎** |
| 适应性 | 失去坐骑的狂兽人会当场爆出肉翼冲天而起；吞吃死于深渊侵蚀的同类可**临时飙升深渊抗性**，随后被周边同类吞吃、并通过后颈腺体分泌生物催化物质弥散开——⚠ **这种临时抗性会在狂兽人之间宛如孢子传播般扩散** |

## 🐉 古龙之王·阿兹尼德

万米级黑龙，第一纪元最强梯队之一。⚠ **其神话生命可怕到，哪怕部分龙血未死，都化身「古王」存在到当前纪元**（苏晓曾在女巫界与之交锋）。

- **战力**:鲜红的雷电龙息喷吐而下，上千米直径所过之处，深渊种族尽数化为岩石化的无机物；⚠ **那灰白色宛如岩石质感的不朽龙皮坚固到离谱**——古老飞龙最强的一点就是龙躯的防御力。
- ⚠ **他有生前完整的理智**，记忆截止于第一纪元与太阳王死战后意识死亡的那一刻。他对「死后的世界」的感受是:**什么也没有**——没有黄泉九幽，没有炼狱轮回，只是一片感受不到的虚无。
- 他睁眼那一刻就知道这不是复生，而是**重现**；也知道重现一次已是奇迹，意图再度重现纯粹是妄想。
- ⚠ **他为何愿意帮忙**:若这是星界势力间的交锋，他懒得理会——**可这次是有血仇的深渊军团**。

⚠ 一个耐人寻味的对照:古老飞龙们有着最强的防御力，风格却激进弑杀，从开始到结束都没关注过己方防线，只顾着在战场上大杀四方。

## ☀ 四代太阳王与太阳军团

- **为何是四代而非初代**:若论强大当然是初代太阳王，⚠ **但要说率领太阳神族远征，四代太阳王当仁不让**（初代太阳王的神话生命更强悍，根本没死透，而是沉寂到第三纪元、随「旧日」一同落山；他是否会登场未知）。
- **登场**:太阳神族们拔出大剑，却并不使用更强悍的太阳体系能力——那必会误伤友军。可他们的「朴素」是这样的:⚠ **敌方所在区域，一颗巨型太阳陨落而下**。太阳焰焚烧中，深渊军团尽成灰烬；随后太阳军团的大剑整齐前指，**迎面而来的太阳之焰全部聚集到他们的大剑上**。
- ⚠ **深渊三堡垒之所以那么坚固**（被太阳砸落只出现部分破损与燃烧且逐渐恢复），正是**第一纪元时被锤炼出来的**，普遍拥有极高的太阳伤损抗性。

**他们的选择——磐石**:太阳军团只有几千之众，却牢牢抵在己方正面防线的中心处。⚠ **在太阳战士们的默认里，所有对抗深渊军团的星界生灵都是友方**，而敢站在最前线直面深渊的勇敢者，**每一个灵魂都很宝贵**。

- 受致命伤而无法冲回大本营的参战者，退到太阳军团后方就能获得宝贵的喘息空档；那些人的共同感受是——**这些太阳老哥真是太可靠了**。
- ⚠ **不灭的编制**:每当有太阳战士身死，一缕阳光会映下，一位展现出赞美太阳姿态的太阳战士会从「往日」中站起替代他，**让太阳军团始终保持几千人的规模**。
- 一名叫暗鬼的绝强契约者被战锤打碎下巴、口腔涌血时，一只戴着护臂的大手按上他的肩膀，温暖的太阳之力涌入——⚠ **这等治愈已超出九成的专业治疗系能力**。

!!! quote "两个宿敌的对照（第4129章）"
    **更强大的防御为了更肆意的杀戮，更强大的攻伐为了更坚毅的守护。**<br>——只能说，古龙与太阳这两方宿敌会互相征战一个纪元，是一点也不奇怪。
