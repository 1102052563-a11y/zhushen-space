# 国王·罗阿尔德·圣·阿德兹

> ⚠ **联合帝国的第三十任国王，年过40才继位、如今已98岁**——⚠ **身形消瘦到皮包骨、气息如风中残烛、命不久矣，可那双眼睛一点也不浑浊、很亮且神采奕奕**;⚠ **⚠ 他是一位与「机械神教〔众神〕、王族〔灵魂钟塔〕、三大臣〔卢恩家族〕」三大势力周旋了一生、不仅没被架空、还彻底掌控了王城的明君**（⚠ 苏晓评「肉身D、权谋 SSS＋＋＋＋＋＋＋级别」）;⚠ **⚠ 继位后帝国财政翻七倍、战争频率锐减90%、废除多个殖民/不平等条约**——⚠ **一个若隐藏阴谋一生、直到垂垂老矣仍没展现阴谋的人，总归要「论迹不论心」**。

## 🃏 濒死打出的一张手牌（54卷111章）

⚠ **他邀苏晓一面——不是召见、是「邀请」〔给面子就去〕**;⚠ **⚠ 他强撑最佳状态见这满身血气的强大「凶兽」**——⚠ **苏晓一开场就表明「把众神全宰了」**（⚠ 若他是众神傀儡就绝不会有这次见面·⚠ 他一见【起源法典】就认出「第一纪元失踪的法典、古龙们封藏的」、却不敢碰「别让众神发现我的因果不对」）;⚠ **⚠ 他任命苏晓为「帝国特殊事物部」部长〔空缺十几年·直属国王·前任部长被谋害〕**——⚠ **这是他的阳谋:被动把苏晓搅进帝国这滩浑水**（⚠ 他要的是保护帝国和追随他的亿万子民、不在乎钟塔/众神/卢恩最终覆灭与否·⚠ 苏晓与他是天然盟友）。

## 🕯 拒绝「偷生」 · 选择赴死

⚠ **他见完苏晓后，墙上挂钟化为金属脸孔〔众神〕来劝降**:⚠ **「只要你向我妥协、无需臣服无需听从、只需妥协，你的生命会永无止境」**——⚠ **他直视金属脸孔:「你们这些神灵，力量强大、生命漫长，但内心……太渺小了」**（⚠ 计划是钓那只「老野兽」卢恩家族·却没上钩）;⚠ **⚠ 苏晓拿【永生者药剂】给他续命——他希冀地看着、却虚弱地摇头**:⚠ **「我会死，不仅因身体和灵魂油尽灯枯，是钟塔、神教、黑暗野兽对我的忍耐已到极限。我死了、我们或许能胜；我偷生、我们的胜算太低了」**——⚠ **「你赶快把那东西收起来，我怕自己忍不住偷生，谁都不想死啊，我也不想」**（⚠ 笑得自己连连干咳·⚠ 谁都不想死、可这明君选择用自己的死，为帝国搏一线胜算）。

## 👑 真身 —— 帝国皇帝·罗阿尔德·圣·阿德兹（54卷144–153章）

> ⚠ **帝国有几十任国王，但「帝国皇帝」只会有这一位。**

⚠ **他的「死」是计划的一部分——他的确死去了，否则怎么可能骗过那几位强敌**;⚠ **⚠ 复生的方式:寄居在辅相·恩姆的身体里**——⚠ **「因果符文」被封印、全世界剧震的那一刻，正坐在王位上听辅相低声诉说的新国王·拜拉席·圣·阿德兹忽然瞳孔一缩再放大，目光变得沉稳威严:「恩姆，是时候了，去找利坦维格，和他说，今天就是我们等待的机会。」**（⚠ 辅相·恩姆身体一僵、心跳失控——**这种语气、这种神态，正是他追随了几十年的帝国皇帝**·⚠ 恩姆言罢起身就走，新国王迷茫地问「你去哪」，他只停下脚步、没转身，沉声道:**「去履行，我最重要的职责。」**·⚠ 走出王殿后他抬头看了眼太阳，很想现在就回家最后看一眼亲人们，可惜他不能）。

- ⚠ **⚠ 「昔日败者组」的三人合谋**:
    - ⚠ **① 帝国皇帝本人**——⚠ **壮年时曾尝试改变帝国被动的局面，结果没能撼动三巨头分毫，还付出了近乎所有子嗣的性命**（⚠ 他最满意的长女已死——那会是让帝国更进一步的女皇陛下；最后仅剩的子嗣，作为国王方面不如他所选继承人的十分之一）。
    - ⚠ **② [万象始源·哈罗塔](万象始源·哈罗塔.md) 本尊**——⚠ **被自己的老师〔祖神〕暗算、险些彻底陨灭，仅剩少量神血，宛如无主幽魂般靠寄生其他生物存活，还时常陷入沉眠**（⚠ 是老国王昔日的左右手之一、**上一任「帝国特殊事物部」部长**发现了这位众神的处境·⚠ 它就是王殿二楼那面墙壁上化为金属脸孔的挂钟，见到恩姆的身体时说出的名字是**「罗阿尔德」**:「你真的成功了。」）。
    - ⚠ **③ 前地精商会董事长·伯吉·金冠**——⚠ **惨遭凯撒多次坑害〔最惨一次被骗到内裤都不剩〕、因此被股东大会罢免**;⚠ **⚠ 他许久前曾孤身前往密岛**——⚠ **旁人误以为他想凭密岛珍宝重回商会，现在看来这分明是三人合作的一部分**（⚠ 决战中他一身笔挺西装、神情紧张地扛着**生命权杖**奔跑——**祖神从此刻丝毫感应不到被偷走的权杖，仿佛它已经不存在**）。
- ⚠ **⚠ 他的真正目标**:⚠ **不是复国、不是长生，而是替深渊浩劫前的星界万族，造一个能容纳万族的超脱之界**——⚠ **打动 [老婆婆](命运暴君·老婆婆.md) 的正是这番话**（⚠ 详见 [老婆婆](命运暴君·老婆婆.md) 页·遗言）;⚠ **⚠ 本世界为何最有资格超脱**:⚠ **第二纪元时炼金师们主动打开超巨型深渊通道、本世界被深渊侵袭95%以上并成功抵抗——涌入的九成以上是「初始状态深渊能量」，等于整个世界都在被最纯粹的深渊力量增益**（⚠ 帝国在本世界壮大2000多年，本世界依然有九成以上未被开发的区域·⚠ 超脱只差临门一脚，被「因果锁环」死死卡住）。
- ⚠ **⚠ 落幕**:⚠ **世界之核归位、王城回溯成机械污染前的模样后，他从王前大道附近的坐姿起身——他方才与周边跪伏的金属身影不同，始终平静地盘坐着，哪怕承受身体金属化的剧烈痛苦**;⚠ **⚠ 他看着重新巍峨的王殿与上空逐渐浮现的七彩旋涡，留下最后一句评价**:⚠ **「果然，正如那位占卜家所说——一个世界有灾祸时，灭法者是最可靠的救赎者；不过，当一个世界安定时，灭法者就可能成为灾祸本身。」**——⚠ **言罢瞳孔咔吧一声裂开，意识彻底破碎，辅相·恩姆的身体倒下**（⚠ 原本认为自己必死的恩姆突然睁开双眼——**帝国皇帝给自己忠诚的部下留下了活路**·⚠ 他还给大统帅·利坦维格、副部长·托,恩塔等心腹都留了活路，可惜在这种大事件中有半数人身死；⚠ **唯一没留活路、压上一切的，只有帝国皇帝自己**）。
