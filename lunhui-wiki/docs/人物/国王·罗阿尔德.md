# 国王·罗阿尔德·圣·阿德兹

> ⚠ **联合帝国的第三十任国王，年过40才继位、如今已98岁**——⚠ **身形消瘦到皮包骨、气息如风中残烛、命不久矣，可那双眼睛一点也不浑浊、很亮且神采奕奕**;⚠ **⚠ 他是一位与「机械神教〔众神〕、王族〔灵魂钟塔〕、三大臣〔卢恩家族〕」三大势力周旋了一生、不仅没被架空、还彻底掌控了王城的明君**（⚠ 苏晓评「肉身D、权谋 SSS＋＋＋＋＋＋＋级别」）;⚠ **⚠ 继位后帝国财政翻七倍、战争频率锐减90%、废除多个殖民/不平等条约**——⚠ **一个若隐藏阴谋一生、直到垂垂老矣仍没展现阴谋的人，总归要「论迹不论心」**。

## 🃏 濒死打出的一张手牌（54卷111章）

⚠ **他邀苏晓一面——不是召见、是「邀请」〔给面子就去〕**;⚠ **⚠ 他强撑最佳状态见这满身血气的强大「凶兽」**——⚠ **苏晓一开场就表明「把众神全宰了」**（⚠ 若他是众神傀儡就绝不会有这次见面·⚠ 他一见【起源法典】就认出「第一纪元失踪的法典、古龙们封藏的」、却不敢碰「别让众神发现我的因果不对」）;⚠ **⚠ 他任命苏晓为「帝国特殊事物部」部长〔空缺十几年·直属国王·前任部长被谋害〕**——⚠ **这是他的阳谋:被动把苏晓搅进帝国这滩浑水**（⚠ 他要的是保护帝国和追随他的亿万子民、不在乎钟塔/众神/卢恩最终覆灭与否·⚠ 苏晓与他是天然盟友）。

## 🕯 拒绝「偷生」 · 选择赴死

⚠ **他见完苏晓后，墙上挂钟化为金属脸孔〔众神〕来劝降**:⚠ **「只要你向我妥协、无需臣服无需听从、只需妥协，你的生命会永无止境」**——⚠ **他直视金属脸孔:「你们这些神灵，力量强大、生命漫长，但内心……太渺小了」**（⚠ 计划是钓那只「老野兽」卢恩家族·却没上钩）;⚠ **⚠ 苏晓拿【永生者药剂】给他续命——他希冀地看着、却虚弱地摇头**:⚠ **「我会死，不仅因身体和灵魂油尽灯枯，是钟塔、神教、黑暗野兽对我的忍耐已到极限。我死了、我们或许能胜；我偷生、我们的胜算太低了」**——⚠ **「你赶快把那东西收起来，我怕自己忍不住偷生，谁都不想死啊，我也不想」**（⚠ 笑得自己连连干咳·⚠ 谁都不想死、可这明君选择用自己的死，为帝国搏一线胜算）。
