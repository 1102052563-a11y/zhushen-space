# 言峰绮礼

[Fate/Zero 世界](../世界/任务世界/fate-zero.md) 老神父 [言峰璃正](言峰璃正.md) 之子——本届圣杯战争中 **Assassin（暗杀者）的御主**，与父亲同属 [圣堂教会](../势力/index.md)、又已与 [远坂时臣](远坂时臣.md) 结盟（演「英雄王杀 Assassin」的戏时，Assassin 实未死）。

- **性格缺陷（伏笔）**：他不算恶徒或邪道，但**无法对世人所谓「美丽」的事物感到美丽，反而对丑陋之物难以忘怀、见他人不幸便莫名畅快**——这一扭曲属性尚未觉醒、却已不远（埋下原著大反派的种子）。
- **暗藏的杀心（十一卷54章揭示）**：原著情报显示，他**与父亲老神父其实不是一个阵营**——只是老神父单方面把他当成「最值得信任的继承人」。原著中老神父被枪杀、绮礼见尸毫无表情、反而**惋惜「老神父为何不是死在我手里」**（因老神父不了解他）。他已被父亲安排**拜入 [远坂时臣](远坂时臣.md) 门下为徒**（以稳固联盟），却暗生杀师之意；其英灵 [吉尔伽美什](吉尔伽美什.md) 又觉他比时臣有趣、不时蛊惑他弄死时臣。

## 提前黑化 · 弑父弑师（十一卷57–59章）
**[蝶](蝶.md) 间接催熟了他的恶兽**：蝶追杀契约者时、在他体内偷埋了一枚**魅惑能量种子**（想离开圣杯世界前控制他捞外快）——蝶被苏晓斩杀后，**那枚种子破裂**，反而**唤醒了他心中沉睡的「恶兽」**：他第一次不再按他人意愿活、而要按自己的意愿做事。

- **弑父**：按自我意愿做的第一件事，竟是找 [言峰璃正](言峰璃正.md) 长谈后**将亲生父亲腰斩弑杀**（老神父惊愕「不了解这 27 岁的儿子」）、夺走其手臂上 41 枚令咒。
- **弑师**：在 [吉尔伽美什](吉尔伽美什.md) 协助下、**从背后捅 [远坂时臣](远坂时臣.md) 一刀刺破心脏**（凑齐「弑父 + 弑师」双成就·愉悦至病态）；又把追来扑向时臣尸体的 [间桐雁夜](间桐雁夜.md) 借英雄王之手斩首、伪造成「雁夜与时臣同归于尽」的假象。
- **英雄王改投门下**：[吉尔伽美什](吉尔伽美什.md) 觉他比时臣有趣得多（不是满脑子圣杯），**改作他的从者**，唯一要求是「找到 Berserker、他必须死在我手中」。眼中尚有蝶念能量的淡粉余光——他却**拒绝祛除**（「正是它让我看清自己」）。这是埋下原著大反派的关键转折。

## 被暗杀 · 死的老惨（十一卷63–64章·殁）
作为隐藏任务「英灵的荣耀」最后一名御主，他**持有惊人的 27 枚令咒**（原著中可把令咒当战斗魔力供给挥霍）。他与苏晓互相追踪、终在冬木市步行街擦肩——苏晓戴一次性伪装面具扮游客、收敛气息问路试探，他**警惕性极高**（退后一步避开藏在袖口的 [屠戮](../力量体系/装备道具.md)、还一眼看破对方露出学生证的「学生」身份），没让苏晓得手。但他放松吃麻婆豆腐时，被扮服务生的苏晓**在冰橙汁的冰块里藏毒**（饭菜与橙汁都无毒、毒随冰块融化才渗出·避开了他的银针验毒）——中剧毒后他刚要喊令咒，就被苏晓 [屠戮](../力量体系/装备道具.md) 短刀**刺喉贯脑 + 割喉**（断喉 + 泣血 + 月之光华三效齐发），**连令咒都没用出就死了、死得「老惨」**。世界之源 +2.9%→83.5%·苏晓斩下他那条 27 令咒的手臂打包（隐藏任务「英灵的荣耀」就此达成）。
