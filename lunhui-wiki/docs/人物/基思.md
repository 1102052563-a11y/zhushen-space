# 基思（盛宴女巫 · 咒术系食人魔）

[暗鸦星](../世界/任务世界/暗鸦星-暗鸦纪元.md) 的**盛宴女巫**——咒术系三阶段黑女巫（悬赏 128 金币·容貌绝美·咒术系天然优势），臭名昭著的**食人魔**（最爱吃白女巫的血肉·死在她手里的白女巫 / 巫猎人不在少数）。藏身焚炉人类聚集地（一名白女巫领主的合法领地·她在地下经营骇人的「地狱厨房」）。

## 被苏晓秒杀（二十一卷20章）

苏晓成为巫猎人后的**第一个猎杀目标**。他用使徒之眼定位、布陷阱 + 界断线，独自进她的仓库地下巢穴——一声巨响后，断臂的她惊恐冲出求救、又被界断线扯回斩杀（尸体瞬间干枯成皮包骨，这才是她真容）。这是苏晓首次接触咒术系女巫（[伊萨](伊萨.md) 即死于咒术女巫之手）。击杀后苏晓首获**咒术抗性**奖励（克咒术系女巫的诅咒 / 侵蚀·防被远程灭口），并激活巫猎人声望任务进度。

## 关系
- **[苏晓](苏晓.md)（库库林·白夜）**：成为巫猎人后秒杀她的猎人。
- **艾妮·克劳迪娅**：焚炉聚集地的白女巫领主（基思藏在其领地·美洛狄曾想冒充的「大人物」就是她）。
