# 冷月

**冷月** 是 [进击的巨人](../世界/任务世界/进击的巨人.md) 世界 **[彼岸花冒险团](../势力/index.md)**（全女性永久冒险团）的团长（三卷8章）。

## 简介
身着洁白女性法师袍、倾国倾城、声音柔和而清冷的女子；攻防战中攻防贡献榜列前茅（曾居榜首、被苏晓反超）。沉稳、护短、善安抚团员（不愿打击士气），与腹黑女是过命搭档（三卷8章）。

经副团长 [血玫瑰](血玫瑰.md) 与智囊腹黑女的分析，对苏晓（用刀、一刀秒契约者、无大范围技能仍强的 PVP 型强者）极为警惕，定下「**别招惹那个男人**」的方针（三卷8章）。

## 三形态与围杀苏晓（三卷28–33章）
冷月可在**法师 / 弓箭手 / 剑士**三种形态间切换（持蓝色法杖 / 金属弓 / 双刺剑、蓝色高跟鞋）：法师有风刃 / 风王怒 / 风之息，弓箭手有附追踪 + 爆炸 + 控制的「月华」箭，剑士的双刺剑附「月之寒 / 月寒毒」；另有「风王模式」御强力风刃。

为给被苏晓所杀的女王 / 黑白 / 叶子报仇，她激活 [彼岸花](../势力/index.md) 专属技能「**血色追印**」、率五十余人结界围杀苏晓。然而苏晓免控、闭眼凭心眼应对，主坦、控制系尽被秒杀；冷月以三形态苦撑（一度被斩穿小腹、靠奶妈糖果「接吻治疗」回血），仍被苏晓的**青钢影极限输出**一刀重创、濒死。她提出「和平模式无收益」的停战被苏晓**断然拒绝**（战局未完）。

## 重整精英团 · 吞噬岛复仇（八卷12章）
险些被苏晓灭团后，冷月幡然醒悟「**人多没用、要的是精英**」——巨人世界后彼岸花的普通成员纷纷离去，从中型冒险团衰落成小冒险团，她却借机精简、让资源分配更充足。此后几个衍生世界她们一改中规中矩、深挖世界、几度濒临团灭却快速变强，并（在腹黑女谋划下）**卡在同一时间点参加晋升任务**。如今只剩**六名精英女将**：团长冷月、副团长 [血玫瑰](血玫瑰.md)、智谋腹黑女、狙击手蜂、奶妈糖果、新招募的主坦 [罗娜](罗娜.md)。八卷11–12章六人与苏晓同批被分到 [二阶生存试炼·吞噬岛](../世界/任务世界/T12K星球-生存试炼.md)、在飞机上认出他，冷月立誓「**这次不会再输、三十二名姐妹的血仇一定要报**」，定下上岛后伺机猎杀苏晓。

## 吞噬岛惨败（八卷27–29章）
[腹黑女](腹黑女.md) 因窥破苏晓身份触禁忌、自断手甩印记后强制退团、深入红土区中心；冷月率团去寻她，却与同样在找腹黑女的苏晓**狭路相逢**。一场恶战印证了「**技巧与经验的差距**」——冷月三形态（风王 / 弓箭手 / 银月剑士·现银月形态也能用风王技能）虽敏捷比苏晓还高一点，仍被苏晓**剃 + 抛刀（界断线缠柄）+ 牛头之怒（力封顶 50）**逐一拆解：先断她一臂、再没入肩头、割开喉咙、斩断腿筋——**双臂尽断、喉咙被割、瘫倒认命**（重伤·若不及时救治会失血而亡）。她至死想不通「敌人为何强这么多」「[血玫瑰](血玫瑰.md) 究竟怎么回事」（不信血玫瑰会背叛）。**八卷30章血玫瑰开「泣血玫瑰」拼死掩护、终究未能逆转**——苏晓收尾时一刀刺穿冷月心脏，**冷月殁**（彼岸花六女全灭）。
