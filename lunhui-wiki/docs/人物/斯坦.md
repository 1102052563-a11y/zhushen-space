# 斯坦

[火影忍者世界](../世界/任务世界/火影忍者.md) 中现身的契约者——**血门冒险团团长**。赤膊壮汉、满身疤痕，脸上一道从眼底蔓延到脖颈的刀疤（进轮回乐园前留下）；发怒时双眼变金、闪耀黄金光芒，身周空间浮现金色裂痕。

- **凭武力统领**：曾与幻影旅团第二位交手、卸掉对方两条胳膊一条腿；在团内说一不二「谁不听话就宰了谁」（与靠人心统领的 [神皇](../势力/index.md) 截然相反）。畏惧晓内**那个能单挑木叶的怪物**（即 [苏晓](苏晓.md)）。
- **怕老婆**：妻子 [莺澜（奈洛伊·血蜘蛛）](莺澜.md)「能徒手拆高达」、暴力至极；外人面前很给他面子，私下却泼辣（她是血门的「眼睛」与「暗刃」）。
- **水之国阴谋的幕后委托人（九卷28–31章）**：「**挑起水之国内战**」的委托正是血门冒险团布的局（多层保险、想让「晓与雾忍狗咬狗」从中得利）——被苏晓识破、委托人凭空消失任务作废。其部下**鬼眼**（绿火眼）被**神王**收买、悄悄背叛（隐瞒「加入晓的契约者」与种子团员 [子默](子默.md) 被杀的调查）→ 斯坦察觉后下令莺澜「做掉鬼眼」。他随即决定撤出契约者扎堆、三大冒险团互耗的火之国，并放弃寻找难找的**咕噜**（幻影旅团成员·此前抢走猎魔戒）。

## 乐园上门索赔（九卷93–94章）
火影通关后，他在轮回乐园**亲自找上 [苏晓](苏晓.md)**——出示子默被狙杀的第一视角录像，承认任务冲突不重要，但**「死人没价值」**，要苏晓**赔偿 10 万乐园币**（培养种子团员 [子默](子默.md) 花了约 50 万·这是生意非私怨）。他礼貌、狠辣、智商在线（自陈本想报复、但派人杀不掉苏晓还费钱、索赔最划算；还点破吞噬岛装甲车的渊源、笑称二人「天生犯克」）。苏晓一口回绝「一分没有」，他便笑着「那就算了」、拿出**金光追踪罗盘**离去——但苏晓的虚假编号 13013 令追踪无从下手，遇上的概率也不大，此事最终不了了之。

## 「斯坦被绿」风波 · 杀妻弟（十七卷94–97章）

返园后牵动各大冒险团的「斯坦被绿」事件揭底：妻子 [奈洛伊](莺澜.md)（血蜘蛛）与 [无伞兄](无伞兄.md) 保持「长期情人」、给斯坦扣绿帽（实为奈洛伊利用无伞兄）。斯坦原话「你找乐子我不管、可你找了个契约者还要长期保持，脑子进水了吗」——他**亲手杀了奈洛伊在现实世界作恶的弟弟**（即无伞兄的「大舅哥」、死得很惨），奈洛伊为此提离婚。但血门是夫妻共建、利益捆绑，二人**不可能真敌对**（外敌当前即化「雌雄双煞」）。此事让 [无伞兄](无伞兄.md) 人头被悬赏 **100 万乐园币**（面子之争·神皇 / 血门 / 红枫各冒险团借机博弈），[苏晓](苏晓.md) 因与血门是死敌、却坚决不掺和这个「大坑」。苏晓挑战 [荒焚](荒焚.md) 时，斯坦亲到三阶竞技场观战、重注 80 万押荒焚胜（结果荒焚败）。

- **血门金字塔**：斯坦（核心·掌团队重要物资）/ [荒焚](荒焚.md)（管下属冒险团·压三阶招新人）/ [奈洛伊](莺澜.md)（眼睛 + 暗刃）三巨头共建；骨干有**铁鼠**（四阶爆破坦）、**灰烟枪**（四阶远程枪手·两枪秒过四阶副坦）。血门是**四阶大型冒险团**、与苏晓「不死不休」。
