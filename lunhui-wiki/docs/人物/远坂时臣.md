# 远坂时臣

[Fate/Zero 世界](../世界/任务世界/fate-zero.md) **三大魔术家族之一「远坂家」的家主**——第四次圣杯战争中 [吉尔伽美什](吉尔伽美什.md)（Archer）的御主。远坂家人丁稀少，但在普通人与魔术师中都地位很高。

- **「合格的魔术师、不合格的父亲」**：他把女儿 **樱**（[间桐樱](间桐樱.md)）过继给间桐家——结果樱被 [间桐脏砚](间桐脏砚.md) 丢进虫池折磨一年（[间桐雁夜](间桐雁夜.md) 因此恨他入骨，视为「夺妻坑女」之仇·「都是时臣的错」）。苏晓的评价是「不幸源于当事者实力不足，想护女就该去杀那只老虫子」。
- **结盟布局（十一卷7章）**：他与 [言峰璃正](言峰璃正.md) / [言峰绮礼](言峰绮礼.md) 父子早在战前结盟，并让 [吉尔伽美什](吉尔伽美什.md) 演了出「**杀掉 Assassin**」的戏（实则 Assassin 未死·用来迷惑其他御主）——自认稳操胜券。他对镇不住的吉尔伽美什卑躬屈膝，却暗中盘算**用令咒逼其自尽、独吞圣杯**。

- **「躺枪王」 · 令咒撤英雄王（十一卷53–54章）**：英灵齐聚战中，他眼见英雄王被苏晓斩断一臂、险些被「连鞘斩」绝招命中，**果断用一道令咒强制把吉尔伽美什召回**（不想损失英灵）——却惹得本就高傲的英雄王暴怒、弄死他的心更坚定（「这还真应了间桐雁夜那句『都是时臣的错』」）。苏晓评他是本届**头号「躺枪王」**：被 [间桐雁夜](间桐雁夜.md)（执着复仇）、拜入他门下的弟子 [言峰绮礼](言峰绮礼.md)（暗生杀意）、自己的英灵 [吉尔伽美什](吉尔伽美什.md)（被英雄王蛊惑、又因令咒生恨）三方同时惦记要命，「所剩时间应该不多」。

- **「躺枪王」的结局 · 弑师之祸（十一卷57–59章·殁）**：苏晓那句「所剩时间不多」一语成谶——他的弟子 [言峰绮礼](言峰绮礼.md)（恶兽被蝶的种子破裂催醒）在 [吉尔伽美什](吉尔伽美什.md) 协助下，**从背后捅他一刀、刺破心脏弑杀之**（绮礼凑齐「弑父 + 弑师」双成就）。死后他被恰好赶到的 [间桐雁夜](间桐雁夜.md) 扑住掐喉、捏爆下体，被绮礼伪造成「雁夜与时臣同归于尽」的假象（真正致命伤是背后那刀）。他的妻子 **远坂葵**（[间桐雁夜](间桐雁夜.md) 的青梅竹马、[间桐樱](间桐樱.md) 与 [远坂凛](远坂凛.md) 的母亲）与女儿在世——苏晓事后把 [间桐樱](间桐樱.md) 送回了远坂葵身边。
