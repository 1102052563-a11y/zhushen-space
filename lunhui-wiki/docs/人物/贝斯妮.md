# 贝斯妮

[天启乐园](希女王.md) 一方的**侦查类契约者**——一只会说人言、蔚蓝色篮宝石眼、「眉清目秀」的**布偶猫**美猫（雌）。在 [维斯星球·多亚古大陆](../世界/任务世界/维斯星球-多亚古大陆.md) 的「世界之核」抢夺战中，她是 [布布汪](布布汪.md) 的**死对头 / 宿敌**。

- **核心战绩**：天启乐园契约者夺得「世界之核」（无法放进储存空间·只能随身）后，被布布汪用「你看不到我」神不知鬼不觉偷走——是**贝斯妮率先察觉异常、从布布汪口中夺回了世界之核**（代价是被布布汪含在嘴里甩得晕头转向）。
- **被反复吊打**：一猫一狗在圣龙城地下通道「争霸」，她侦查特长、身体素质远逊布布汪——被布布汪叼着脑袋甩、撞墙、揍哭（蔚蓝大眼泪汪汪），她把世界之核吞进猫肚也没用、又被布布汪夺回，最终被「拐走」。叫嚣「来一场公平决斗」也只换来「喵星人连爪」对「汪」的又一顿胖揍。
