# 枭·芙莉亚

**幽冥王下四骑士**之一——身材苗条的女骑士，戴仅口鼻三孔的白色骨质面具，⚠ 竟是**盲眼**。她不统军队、也不擅带兵（让她打仗会戴痛苦面具），强在**个体战力**、各类手段专门**针对个体强者**、是顶尖暗杀者。她在太阳圣巢外潜伏一整晚窥伺 [苏晓](../苏晓.md)（隐隐觉出这猎物不对、对方无意透出的血气就令人心惊），是苏晓可选主线任务「识时务者」的目标。

## 暗杀棘拉 · 折在阿姆手里

⚠ [乌鹰·索拉罗](乌鹰·索拉罗.md) 双线开战——趁苏晓率军攻白金之都、在太阳圣巢开两道黑暗之孔倾泻腐化者，枭·芙莉亚借总攻潜到母巢暗杀 [棘拉](../棘拉.md)。她本已成功潜入，却倒霉地被一只泰坦巨兽被灵魂巫师轰爆的电浆冲击波从幽冥之雾里顶出暴露；换常人早撤，她却拔双刃杀进母巢——直撞上苏晓留的后手:守家的 [阿姆](../阿姆.md) 和 [艾塞亚](艾塞亚.md)。她单挑艾塞亚本不至死，却遇上克星阿姆:阿姆的「寒冻 + 龙心斧」减速效果优先度极高（羽神、罗格什、至虫都栽过），被减速的她再被艾塞亚夹击，终被阿姆**冻成冰雕**。等苏晓回来处置时，她感知到苏晓气息、自知无生路，为免被套情报**自我了断炸成碎冰**。艾塞亚欣赏她的骨气、亲自葬了她（[阿姆](../阿姆.md) 得主线击杀提示 + 不朽宝箱·银枭）。

## 关系

- [乌鹰·索拉罗](乌鹰·索拉罗.md)：与她同列的王下四骑士（他双线开战放她去暗杀棘拉）。
- [阿姆](../阿姆.md)：她的克星（减速效果冻住她）、拿到她击杀提示的凛冬战牛。
- [艾塞亚](艾塞亚.md)：与阿姆一起守母巢击败她、欣赏其骨气葬了她的救世者。
