# 帝国君主·奥尔丁

**帝国**（奥凯星）的最高统治者——身材消瘦到皮包骨、骨架奇大，明明只有普通人的体魄（因常年操劳、身体还不如壮实些的普通人），可与他对视时仍让人下意识退避:**因他的灵魂极其强大**。他坐镇会议室三主位之一，是这场浩劫里帝国一方真正的主心骨。

## 跨星球撤职桑德 & 主持大局

⚠ 幽冥入侵下，[苏晓](../苏晓.md) 早判断第三舰队高层出了投靠幽冥的叛徒（[桑德将军](桑德将军.md) 已现感染征兆）——救世者 [卡拉](卡拉.md) 被坑成灭世巨怪、对 [深红女皇](深红女皇.md) 的暗杀都出自第三舰队之手，这在灾祸将至时无异于自掘坟墓。远在母星的奥尔丁一举**跨星球把桑德将军撤职**（其侄 [莱茵·戈德](莱茵·戈德.md) 接任第三舰队临时统领），证明帝国这体量的势力绝非因一名叛徒就垮的猪队友。他放话要亲赴潘多拉星主持大局，仅是这风声就把公司吓得连夜捐 290 万单位生命矿石 + 700 公斤稀有金属跪舔（可见其积威）。帝国正不惜代价从 469 亿人口里选 5000 万迁来潘多拉星（最后净土）。三方会议上他与公司大股东**艾泰奇·福克**共主持，苏晓当场接下「除掉卡拉」的活（事后收资源、报酬少了就让福克「自己体会后果」）。

## 关系

- [桑德将军](桑德将军.md)：被他跨星球撤职的第三舰队原最高指挥官（疑幽冥叛徒 / 感染者）。
- [莱茵·戈德](莱茵·戈德.md)：接替桑德、任第三舰队临时统领的帝国之手。
- [苏晓](../苏晓.md)：与他 / 福克在三方会议共商、接下除卡拉之约的「太阳圣巢」之主。
