# 乌鹰·索拉罗

**幽冥势力**的**王下四骑士之首**——身穿暗金全身甲、头甲缀羽、坐白骨高座、左插双手大剑右置金属大弓的狠角色。幽冥的权力构成很简单:绝对统治者「**幽冥大帝**」之下是**王下四骑士**——**乌鹰·索拉罗**（大帝的猎鹰·能发现猎物、杀死猎物、带回有价值部分·最受大帝信任）、**黄金狮·缪**（殿前护卫长 / 禁卫军统领）、**枭·芙莉亚**（不统军·个人战力针对个体强者）、**魔蛇·古摩**（军师·过于神秘、疑已死，故常被称「王下三骑士」）。索拉罗身旁有名蓝黑贴身铠甲的苗条女战士（唯独对他温柔）。

## 急攻太阳圣巢 · 敬重灭法者

攻陷白金之都后，索拉罗没按常理屯兵，而是急攻 [苏晓](../苏晓.md) 的太阳圣巢——因他要抢 [莱克利](莱克利.md) 体内融合了幽冥力量的**命运之血**（得手可在本世界开永久空间通道让幽冥主力大军直入·他也知本世界有世界意识、要抢在其运作前拿下）。他先化太阳焰龙为幽冥焰龙当坐骑（被苏晓紫水晶戒指挡下精神冲击、焰龙自爆），再以数不清的腐化者组成几公里粗的流柱空袭 + 亿万腐化者围城，还放冥龙鲸、灵魂扭曲者、给腐化者加幽绿之力增幅。他一眼认出苏晓是「**灭法**」（部下说灭法已亡故，他却断言「新的灭法会出现·哪怕是敌人，灭法都值得尊敬——尊敬他、然后打败他、给他最高敬意的死亡」），隔空对苏晓点头示意、颇有风度。

## 关系

- 幽冥大帝：他效忠、最受其信任的绝对统治者。
- [苏晓](../苏晓.md)：他敬重为「灭法」、却要夺其保护的世界之子、并打败他的对手。
- [莱克利](莱克利.md)：他不惜代价急攻太阳圣巢要夺的「预言之人」命运之血宿主。

## 🏛 「世界之门」& 一人安排帝国的战绩（47卷31–32章）

⚠ 凯撒探得情报:索拉罗选白金之都当驻地，是因那里布设了**「世界之门」**——幽冥核心技术的大型阵图，一旦建成，幽冥**主力军团**将正式攻入本世界（约 4 天建成·已建 2 天·他急夺 [莱克利](莱克利.md) 命运之血正是为加速它）。此前整场入侵，幽冥只出了他这一位王下骑士 + 十几名中层:他先在殖民星蛊惑邪教反叛引发死伤、以幽冥能量把死者化腐化者、占灰猎星、给奥凯星施压；再用「先炮灰骗帝国自信、后放本土组建的幽冥战士 / 灵魂巫师主力痛击」把帝国吓得放弃母星逃来潘多拉星（其实帝国当时硬打本能赢·却被热咖啡养没了血性）。他杀过的世界之子已达两位数——其中一个叫**鲁珀**的少年杀穿千军万马、险取他性命、笑着**斩下了他右臂的「存在」**（再生 / 移植全无效·只能用铠甲手臂替代）。到潘多拉星他察觉苏晓「可能反把他安排了」，从不托大、遂重资建世界之门以备退路。

## 关系（补充）

- 鲁珀:曾杀穿千军斩下他右臂「存在」、让他印象最深的世界之子。

## ⚔ 双线开战 → 世界之门被踹碎 · 撤回冥界（47卷34–35章）

⚠ 决战日索拉罗祭出精妙布局:趁苏晓率全军攻白金之都、故意放「世界之门快建成」的情报引他倾巢而出，同时在太阳圣巢开两道黑暗之孔倾泻腐化者、并放 [枭·芙莉亚](枭·芙莉亚.md) 潜入暗杀 [棘拉](../棘拉.md)（苏晓的后手是留 [艾塞亚](艾塞亚.md) + [阿姆](../阿姆.md) 守母巢）。他还打穿黑暗之孔（万米·殖民星腐化者把白金之都淹成大水池、保世界之门）——却栽在苏晓用战争领主称号终极能力召出的**远古金属生命「蛀世」**上:蛀世群吞光腐化者 + 合金城墙 + 一切，把白金之都夷为平地。索拉罗没死、以某种手段撤回冥界（代价是入侵本世界的布设完全失效、两界能量共鸣中断），走前对苏晓道「灭法，很遗憾这次我们是敌人」。[苏晓](../苏晓.md) 一脚踹碎**世界之门**、把幽冥势力击退（得 18% 世界之源 + 95000 名望登顶）。

## 关系（补充）

- [苏晓](../苏晓.md)：他敬为灭法、双线布局却败于蛀世、被踹碎世界之门后撤回冥界的对手。
- [枭·芙莉亚](枭·芙莉亚.md)：他放去暗杀棘拉、却折在阿姆手里的王下四骑士。

## 🌇 秃鹰的末路 · 战死王殿前（47卷38–41章）

⚠ [苏晓](../苏晓.md) 靠 [莱克利](莱克利.md) 的命运之血耗血开启冥界之门、反攻死者之城，被入侵是冥界千年来头一遭。开战之初索拉罗以四军团（冥界主力军 + 秽树人 + 龙血 + 死灵）迎战，一度把恶魔兽压得节节败退。转折来自 [棘拉](../棘拉.md) 晋女皇级后开发的**进化点**:恶魔兽经进化点升为「精锐恶魔兽」（尾刃能贯穿 45 米秽树人头颅）、泰坦巨兽升满后以百米电浆炮雨洗地。四军团随之崩溃——[龙血领袖·卢恩](龙血领袖·卢恩.md) 死于电浆炮雨、[烟公主](烟公主.md) 的死灵军团被专挑锤爆、[扭曲战铠](扭曲战铠.md) 战至最后一人被一雷枪贯首。索拉罗曾对老友扭曲战铠说「这次我们一起战胜那个灭法」，也曾放走背锅的 [凯因](凯因.md) 三人、对 [烟公主](烟公主.md) 打感情牌劝她脱离冥界。⚠ 决战末，他全身浴血、双手大剑折断、战弓崩裂、左臂已断，仍单手持长柄战刀立于通天王殿正门前——「休想靠近……我王半步」。他挺身冲向龙首上的苏晓、被恶魔兽群完全笼罩，半小时后仰面倒在雨中，这位维系冥界至今的「秃鹰」战死于此。苏晓从他尸旁走过，停步于通天王殿正门，感知里面浓郁的深渊之力——只有战胜 [幽冥大帝](幽冥大帝.md)，才是彻底战胜幽冥。
