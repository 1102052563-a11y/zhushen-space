# 怒甲

[潘多拉星](../../世界/任务世界/潘多拉星-虫族战场.md) 上一座七阶「怒甲虫巢」的**虫族首领**（层级与虫族女王对等，只是对外性别不同；本世界虫巢并非只有母体能建，爆兵靠虫巢，首领 / 母体只是金字塔顶尖、负责决策与精神网管控）。形体类人、四臂、金黑骨甲。名字听着刚猛，实则**阴险谨慎**——是收保护费和借刀杀人的老手（[苏晓](../苏晓.md) 灭掉的蜘蛛虫巢，每天开采的生命矿石有 80% 上贡给它当保护费）。

## 借刀杀人 → 被活捉当插件

苏晓灭蜘蛛虫巢、断了怒甲的矿源，怒甲又接了 [神父](../神父.md) 委托灭苏晓，遂使阴招:培育出与 [卡拉虫巢](../../世界/任务世界/潘多拉星-虫族战场.md) 麾下「角犬」同形的虫族、袭击苏晓的护送队后留下这些尸体，想诱初来乍到的苏晓去触犯冷酷睚眦必报的虫族母皇卡拉的威严（借刀杀人）。苏晓识破、反用怒甲驱使的伪角犬尸体套话，假意谈「五五分成、8→5 天后给第一批矿」——其实是「5 天后我灭了你」。怒甲不信这交易、连夜派 2 万虫族战士 + 黑铠巨人 + 角犬夜袭苏晓大本营，反被菌毯「越打越多」的恶魔兽拖垮；苏晓则趁其虫巢空虚孤身杀入，用刀术宗师豁免其精神控制、激活「灵魂凝视」，一拳把满身骨甲却不扛打的怒甲打晕活捉（怒甲被俘时暴怒，苏晓名望 −270 → −320「恶名初显」）。怒甲被当作**珍贵插件**给苏晓的大本营母巢扩充基因库（提升工蝎开采能力），其虫巢的 10 万工兵 + 生命矿石尽归苏晓。

## 关系

- [苏晓](../苏晓.md)：识破其借刀杀人、灭其虫巢、把它活捉当插件的灭法者。
- [神父](../神父.md)：委托它灭苏晓的老阴哔。
- [蜘蛛女王](蜘蛛女王.md) / 蜘蛛妹：怒甲收保护费的下游（蜘蛛妹本是蜘蛛女王属群被赶出者·怒甲是其靠山）。
