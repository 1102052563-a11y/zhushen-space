# 龙血领袖·卢恩

**幽冥势力·龙血军团**的领导者——目光阴冷的蛇人。龙血一族号称传承龙血，但血脉纯度早已减低，说是「蛇人」更贴切:整体人形、上半身是水桶粗的蟒蛇身、有双臂、头上一小段尖角。这**快消失的麟龙角是龙血族最后的荣誉**，没了它就真的退化成蛇人——而卢恩恰恰**没有独角**，却依然坐上龙血族领袖之位，可见其手腕。它主管冥界军需与物资，战略上略低 [乌鹰·索拉罗](乌鹰·索拉罗.md) 一级、听其指挥。

## 装备精良、实力平平

龙血战士们身穿暗金色铠甲、手持精良枪矛，属于**自身实力一般但装备特别好**的类型。反攻决战中，卢恩眼见 [烟公主](烟公主.md) 的死灵军团被 [棘拉](../棘拉.md) 的精锐恶魔兽按在地上毒打，果断收起原本嚣张的态度、不想步其后尘。⚠ 军需处被 [苏晓](../苏晓.md)、[凯撒](../尼古拉斯·凯撒.md)、[亡灵妹](../亡灵妹.md) 三人盗空后（他主管军需难辞其咎），额头急得直冒汗。

## 关系

- [乌鹰·索拉罗](乌鹰·索拉罗.md)：他效忠、听其指挥的幽冥总指挥。
- [烟公主](烟公主.md)：与他有旧怨、曾在王宴当众把他毒打的死灵军团领袖。
- [苏晓](../苏晓.md)：把他军团团灭、以电浆炮雨要他性命的太阳圣巢领主。

## ☄ 拥抱天空 · 死于电浆炮雨（47卷41章）

反攻平原战鏖战至下午，卢恩麾下的龙血军团被打光——他全身浴血、喘气如牛，深知自己若倒下龙血军团便不复存在。他环顾周边的恶魔兽，认清这些绝非只知杀戮的野兽，而是「**有指挥、有纪律、极擅配合的战争生物，比冥界骑兵更纯粹的战争族群**」。死战正酣，敌军后方骤然轰来一颗直径百米的电浆球，半空炸裂成数不清的电浆炮雨——⚠ 它并不知道，这仅是**一只泰坦巨兽**的一发轰击，而此刻太阳圣巢有**一百多只泰坦巨兽**。仰头望着遮天蔽日落下的电浆炮雨，卢恩笑了，双手持刀、举臂做出拥抱天空的姿势，在第一波电浆雨中殒命。龙血一族至此团灭。
