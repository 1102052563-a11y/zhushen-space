# 桑德将军

[潘多拉星](../../世界/任务世界/潘多拉星-虫族战场.md) 上最强势力——帝国**第三舰队的最高指挥官**。一名头发花白却精气神压过年轻人的强壮老人，穿偏灰迷彩军装、坐在主舰指挥区的全息沙盘前。他曾击溃三颗殖民星的土著、满手鲜血，却不同于寻常将领:**懂得到一个陌生的地方要先尽可能了解这里**（关注潘多拉星的土壤解析、植物生命数据）。身为最高指挥官，他不能表现出哪怕一秒的虚弱。他有一名不属军方体制、游走于军 / 政 / 公司三方的红框眼镜女秘书副手，走到哪都能把事办妥。

## 与苏晓的初次交锋

M952 号试验所被 [苏晓](../苏晓.md) 摧毁、几乎杀光所有人，只留下一名女医生与其男助手（因替苏晓指路而幸存）。桑德将军对试验所被毁毫不惊讶，却惊讶于「苏晓竟留活口」——他没料到苏晓正是**在那两名活口身上留了追踪手段**，好顺藤摸到第三舰队大本营。当苏晓的纳米级监听装置刚靠近他，就被帝国科技拦截销毁（帝国科技之强可见一斑）；[布布汪](../布布汪.md) 潜入主舰内网写病毒、还让第三舰队史无前例地「断电三秒」，警报大作、布布屁滚尿流逃出——「帅不过三秒」。桑德将军由此成为苏晓在潘多拉星必须持续摸底、日后正面碰撞的头号本土强敌。

## 关系

- [苏晓](../苏晓.md)：摧毁其试验所、在活口身上留追踪、潜入其主舰的帝国三等重刑犯（他的头号麻烦）。
- [布布汪](../布布汪.md)：潜入其主舰内网、把第三舰队搞断电三秒的科技汪。

## 🆕 妥协 · 放其他舰队入驻（47卷7章）

原本 [桑德将军](桑德将军.md) 阻止其他舰队入驻潘多拉星（为第三舰队部下争功绩、凝聚人心，「帝国白狮」已不在乎个人升迁）。但随着局势变化，布布汪窃听第三舰队秘密频道得知:桑德将军**妥协了、同意其余几支舰队陆续入驻**。一旦如此，帝国将展现压倒性优势、逼得本世界虫族从松散同盟抱成一团——这是苏晓必须抢在前面发展的又一压力源。

## 关系（补充）

- [苏晓](../苏晓.md)：靠布布汪窃听得知其妥协、据此加速发展的对手。
