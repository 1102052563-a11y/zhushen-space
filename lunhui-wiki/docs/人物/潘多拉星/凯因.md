# 凯因

死亡乐园的八阶顶尖契约者、大型冒险团**「英灵殿」**的领袖——身材偏壮、有莫名威严感的**法坦**。他与副团长阿隆（力坦·两人每次冲在最前）是英灵殿的两大灵魂人物。在 [潘多拉星](../../世界/任务世界/潘多拉星-虫族战场.md) 的名望排行榜上，他靠整个冒险团把名望机会集中给他而**稳居首位（2400 点）**——上个世界水哥夺树生世界杀戮榜首、让死亡乐园的「万年老二」们扬眉吐气，这次英灵殿颇有夺第一的气魄。他在帝国、公司两阵营都混得风生水起，本能参与到 C5N2 超导体护送事件。

## 「儒雅随和」的背锅兄

⚠ 他被 [凯撒](../尼古拉斯·凯撒.md) 选作**劫金属骗局的「预备役背锅兄」**（凯撒的理由:「和老子名这么像、有难同当」）。护送战中他亲眼看着 [苏晓](../苏晓.md) 用暗刃三连秒杀公司三王牌、又一刀秒了自己的坦系副团长阿隆（八阶顶尖坦系被秒得他不敢信、轻踢「阿隆别玩了起来」），当即意识到自己「早就不知不觉成了锅中人」。他快速判断莱茵·戈德必先护家人、不会帮他，遂「展现儒雅随和」拽着手下（含被传绯闻实则只是尽团长照顾之责的黑丝御姐团员）撞穿舱壁撤退——把这口大锅结结实实背下。

- 阿隆：他被苏晓一刀秒杀的坦系副团长。

## 🆕 被迫暗杀深红女皇 · 遭天启 96000 悬赏（47卷15章）

上次背黑锅后，[凯因](凯因.md) 很快重回名望榜首位、在帝国那边重新经营起来。⚠ 他让团员搞到帝国机密——**放弃母星、放弃所有殖民星、全面驻守潘多拉星**的密令，才惊觉幽冥入侵有多可怕。帝国以「不干就继续让你背飞船被劫的锅」逼他去**暗杀虫族领袖深红女皇**（帝国想扶卡拉当第三家、让英灵殿背锅）。暗杀险成、深红女皇差点殒命，却因 [天启姐妹花](深红女皇.md) 火速救援收场——英灵殿有个叫**雪怪**的猛男当众开团单挑，被 [豪妹](../塞尔星/豪妹.md) 险些当场宰了（豪妹与凯因单挑都五五开）。天启三姐妹靠「小富婆之力」当场开出 **96000 灵魂钱币悬赏**英灵殿全员（杀团员 3000 / 副团长 7000 / 团长凯因 15000），逼得英灵殿成员不敢单独出门。

## 🩸 图谋太阳圣巢 · 反被雷柱团灭部下（47卷21章）

⚠ 凯因得知苏晓要对付卡拉后，公布消息、率英灵殿（含感知 + 暗杀双修的**银雉**·精神剧毒隔百米「蜇」人）埋伏高泽湖——但他真正的图谋是**杀苏晓、夺太阳圣巢**:他探知苏晓靠永久召唤物 [棘拉](../棘拉.md) 发展虫族，遂两手准备:① 派八阶顶尖三人暗杀队去生擒棘拉（成无主召唤物就能取而代之当第三家）；② 花 3 万灵魂钱币从「公司合伙人」买 7 万角犬放进湖里，触发满卡拉的「世界之青睐」（外装甲 +4 + 超速回血），想把钻进卡拉体内的苏晓永远困死、自己摘桃子。⚠ 致命的是:那位「公司合伙人」正是 [凯撒](../尼古拉斯·凯撒.md)——整件事是苏晓与凯撒合谋（3 万钱币里 1 万落苏晓手、亡灵妹也分 1 万，还反杀了那三人暗杀队）。苏晓故意让卡拉满血好在其体内引下十几公里雷柱，凯因身后**百余名部下当场被雷霆撕成飞灰**。「花钱来找死的，苏晓只遇到过凯因这独一份」。

## 👹 英灵殿魂鬼覆灭 · 噩鬼凯因的真面目（47卷22–24章）

⚠ 十几公里雷柱后，凯因、银雉、黑巫师、小迪四人靠 SSS 级冒险团核心团队技能没死透、却**变成「魂鬼」/ 英灵**——代价是**透支了本世界内没参与「高泽湖计划」的四十多名英灵殿成员的生命**（190 多人一战死 150、又坑死 40 多，全变魂鬼）。这群「史上最阴间冒险团」为触发团队技能复活而执意报复灵魂强度 620、掌握断魂影的 [苏晓](../苏晓.md):用「时间错感反应」道具把苏晓灵魂拖进树生世界的一座废弃**灵魂斗技场**，还花大代价召来苏晓在树生世界挨个捶过的**灵魂之主**等六灵魂守卫（六人一见苏晓就变脸溜走、还在跑前替凯因断了部分连接）。苏晓靠斗技场赋予的「震退」+ 灵魂结晶枪反将魂鬼团灭。⚠ 结局最阴间:魂鬼从 40 多死到只剩凯因 / 银雉 / 小迪三个，凯因当场**吞掉银雉、小迪自救**（魂体裂痕愈合），露出「噩鬼·凯因」的狰狞真面目、喃喃「又要招募新一批团员了」。
