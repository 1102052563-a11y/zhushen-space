# 蜘蛛女王

[潘多拉星](../../世界/任务世界/潘多拉星-虫族战场.md) 五虫族母皇同盟之一——她的战斗兵种 / 虫族建筑中上、工兵虫族顶尖（挖矿能力最强），但真正出名的是**放高利贷**:外借生命矿石，几天后收回本金 + 远超本金的高息，利滚利吃到借款方矿脉枯竭、再灭掉对方。她妩媚如蛇蝎美人、声音勾人（魅力系能力），来 [苏晓](../苏晓.md) 大本营时用的是**精神分身培育出的第二具全人形身体**（原为潜入帝国而造·遭背刺）。之前被苏晓灭掉的虫族女王「蜘蛛妹」，正是从她属群里被赶出去的。

## 想割「韭菜」反被反薅

苏晓急需生命矿石给 [棘拉](../棘拉.md) 晋升，主动找上这放高利贷的、以己方大型矿脉作抵押借款。蜘蛛女王「这片韭菜割定了」，却与苏晓上演一出**双向演戏**:她怕自己名气吓跑苏晓、欲拒还迎；苏晓怕暴露爆兵能力（会让她不敢借）、故意削减工蝎数量装穷、还搬出「公司愿借 20 万利息减半」的假合同逼价。[巴哈](../剧毒魔鹰·巴哈.md) 一顿彩虹屁 + 布布汪的真实魅力 237 唱双簧，最终她拍板「15 万生命矿石今晚送到」，还破天荒对骗她的苏晓一队生出一丝愧疚。⚠ 她全然不知苏晓**根本没打算还这笔高利贷**——正愁没人主动来打，巴不得她武力讨债好抢资源。她还随口透露附近古代遗迹「有邪神降临」，成了苏晓连夜去猎神的线索。

## 关系

- [苏晓](../苏晓.md)：她想割的「韭菜」、实则设局借她高利贷不还、还反将一军的灭法者。
- [棘拉](../棘拉.md)：苏晓要靠她的借款晋升到主宰级的虫族女王。
- 蜘蛛妹：从她属群被赶出、后被苏晓灭掉活捉的虫族女王。
