# 左大臣

**左大臣** 是 [海贼王](../世界/任务世界/海贼王.md) 世界 [哥亚王国](../世界/地点/index.md) 的重臣，国王身边的人物（第9、11章）。

## 简介
满头白发、面容严肃的老者，位高权重、神态自若；身边有一名体术高手贴身护卫。**贪婪、好收藏**——黑市商人卡洛斯长期向他供奉藏品；亦经营欢乐街的生意（第9、11章）。

## 与苏晓的交集
因苏晓冒充卡洛斯的侄子，左大臣召见「白夜」、要其代卡洛斯继续供货；并以**侍卫首领**之位为饵，发出支线任务「杀科尔波山的主人、取其牙齿」（第11章）。

## 违约与下场
任务完成后，左大臣意图赖账、否认许诺过侍卫首领之位，旋即因「**违逆轮回乐园条例**」被乐园**强制纠正**——被操控着改口、运作苏晓上任侍卫首领（第17章）。代价是精神疑似被毁：此后双眼无神、形同废人（第17、19章）。此事向苏晓印证了乐园「冰冷残酷但公正、从未言而无信」（第17章）。

## 关系
- [苏晓](苏晓.md)（白夜）——向其发布支线任务；违约后被乐园强制纠正、兑现侍卫首领之位。
- 卡洛斯（黑市商人）——其供货人；苏晓冒充卡洛斯之侄接近左大臣。
- 菲利普·赫伯特（国王）——其效力的哥亚王国国王。
