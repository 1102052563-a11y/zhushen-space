# 虫族战场配角（棘拉 · 暮光 · 亚克洛 · 元素星 · 四方乐园）

[棘虫星·虫族战场](../世界/任务世界/棘虫星-虫族战场.md) 中的契约者与苏晓的虫族母体，集中收录。

## 棘拉（苏晓的虫族母体 · 临时召唤物）

[苏晓](苏晓.md) 在棘虫星捡到的一阶**幼生虫族母体**——被原女王赶出自立门户、却快饿到吃土的「不争气」小母体（甲壳包裹、眉清目秀近似人类小女孩、魅力 63·消化力惊人）。苏晓喂饱她、用 [恶魔之罐](../世界/生物万族/index.md) 注恶魔之力提忠诚、喂 [枯萎之种](../力量体系/装备道具.md)（伪主角种子·得微量命运之力降进化风险）、连用 15 张临时召唤契约才契约成功（苏晓魅力太低）——取名「**棘拉**」（其族群即称棘拉）。

- **定位**：苏晓**自建虫族大军**的核心。母体负责产「螳甲」（工兵·建巢 / 搬运）、指挥虫族、与虫巢共同晋阶（母体晋阶 = 虫巢晋阶·解锁更多兵种）。本身战力弱鸡（家里蹲、死活不出虫巢·「这是母体的责任」），靠下位虫族战斗。
- **进化**：吞血肉攒「生物能」、满 2000 晋二阶（解锁狩猎齿 / 孢子坦克 / 监视巢穴·能吞别的虫巢获基因记忆 + 基因片段）。忠诚度随恶魔之力侵蚀攀升（已 98 点·除「出虫巢」外言听计从）。
- **晋三阶（女王级）+ 觉醒恶魔兽（二十七卷18章）**：吃布布汪从四阶母皇偷的高纯度 [生命石](../世界/生物万族/index.md) 晋三阶，头部甲壳褪去、露出白皙脸颊（恶魔族 + 虫族混合外观）。因苏晓注的恶魔之力到三阶才发威，**觉醒微量世界之力加持的战斗虫族「[恶魔兽](../世界/生物万族/index.md)」**（近人形·四阶战力·苏晓虫族大军的主力）。虫巢长到 60 米（三阶上限）、能转中纯度生命石。苏晓还造**意识传导装置**（机械臂铠 + 导线）可强控棘拉 / 经她操控数千虫族（后恶魔之力够用就不常用了）。

!!! note "棘拉的结局（二十七卷48章·留在原生世界）"
    通关时苏晓**中断与棘拉的召唤契约、但不抽离她体内的恶魔之力**，把她留在棘虫星（原生世界·时间线不会被轮回乐园重置）自行发展——盼她将来晋四 / 五阶成自己的「战争兵器」。临别她终于读懂苏晓那句「如果你足够强，我让你看到不一样的世界」。带不走的不是嫌弃，是棘拉变强的方式只能是「时间 + 发展」，不适合随队闯衍生世界。

## 玛格（冷酷四阶母皇 · 联军盟友）

草原边一座**四阶段虫巢**的母皇——全身翠绿甲壳、背生六根蜷曲甲足、体态近人、眼底蔚蓝瞳孔淡红，生人勿近的高冷做派（「这才是虫族母皇该有的模样」）。她是猎食型虫巢，靠**放养附近低阶虫巢、每 300 天清理一次**累积生物能（布布汪偷的 11 颗生命石就出自她家），同时常年牵制东边的太阳部落。苏晓以「太阳部落要联亜蛙部落先灭你」说动她，**联手左右夹击灭掉太阳部落**（她出 34 万虫族·死 21 万惨胜）。战后立刻头也不回撤走、连地盘都不谈——像捡到巨款的孩子怕被抢。

## 红厄（温和四阶母皇 · 大肥羊）

自称「这片大陆最古老的虫族·嗒啐红厄」的**四阶段虫巢**母皇——红色甲壳、头生刀锋利甲（无发）、骑七米宽的甲壳巨蝶「厄圭蝶」分身现身。与玛格相反，她靠**猎兽 / 找资源**温和发展（甚至帮巨兽杀天敌取而代之）。来求购 [恶魔兽](../世界/生物万族/index.md) 的「遗留个体」，开口「臣服于我，我的下位个体」很霸气，却被巴哈一套「交易法第 35746 条 + 电视购物·组合拳」+ 布布汪端茶递水果绕懵，最终以 **32 颗高纯生命石**买下遗留个体（走时满脸问号）——殊不知没恶魔之力培育的恶魔兽是残次品，恶魔之力苏晓另开价 500 颗。

## 古拉巴（太阳部落大首领 · 残阳战役终极 BOSS · 殁）

[太阳部落](../世界/生物万族/index.md) 的大首领——铁塔般强壮的老者、花白发辫一米长、满身暗红火纹，持一把一米五长、刃口参差的陨石「圣物」长刀，能喷火 / 放灼火斩芒、速度快到力量型不该有。残阳战役三大终极 BOSS 之一（另两个是苏晓 / 玛格）。守焚火山至最后，与苏晓单挑——苏晓评其「刀太慢、没技法、只凭力量速度，像挥棍棒的洞熊」，以反刃 / 刃道刀·时等技法碾压，最终斩首（古拉巴心服口服、亲手把刀刺地请苏晓取其首级）。死后苏晓得其陨石长刀 **[陨之火刃](../力量体系/装备道具.md)**（史诗刀）+ 储藏洞 251.3 kg [火金](../力量体系/装备道具.md) + 太阳焰心脏。

## 米奈（残阳战役·临时事件契约者）

残阳战役期间被轮回乐园空降的 245 名中低阶契约者之一（四阶上游·本世界战力前十）——五人临时小队的临时决策者，十七八岁的朝气妹子。被队友怂恿来「勾引」终极 BOSS 青钢王（苏晓伪装身份）触发任务，结果接到 Lv.37「深入敌后」战役任务（对四阶契约者堪称团灭难度），当场认怂被苏晓「我很看好你」打发走。她隐隐察觉「这青钢王有点奇怪」（苏晓其实是真人伪装·不在乎暴露）。代表了那批「跟在恶魔兽屁股后捡战功」的奖励世界契约者群像。

## 暮光（共生盟团长 · 轮回乐园方首领）

轮回乐园方大部队的首领——**共生盟**（轮回乐园中型冒险团·团员个个不正常·如成长版铭门）的团长，传言是女性（满身鳞甲的壮男「团长」只是能力外现）。主张不硬拼、先混入土著势力补人数差——率众加入最凶残的**哈萨罗部落**（掠食），与壮硕原始人喝血盟誓入伙（杀人不眨眼·眼含红光）。苏晓认同其策略、不与之争大部队指挥权（自走虫族路线）。

## 亚克洛（死亡乐园首领 · 惩戒骑士）

死亡乐园方推选的首领——**惩戒骑士**（相当于轮回乐园的猎杀者）。上半身机械改造、能从掌心喷爆燃高温火焰（「机械 + 高温」的浪漫·战斗如死亡之花）。一巴掌烧化轮回乐园生化女。轮回乐园方 80% 的死伤出自死亡乐园之手；他与苏晓交手胜负各半（被祭神打断）。后率众加入**太阳部落**（火）。

## 元素星（死亡乐园 · 五孪生兄弟）

死亡乐园的五胞胎契约者——**冰星 / 电星 / 光星 / 风星 / 魂星**（相貌全同·**共享视觉 / 能力 / 思维**·战斗无需交流）。围攻苏晓时被其刀术连斩**风星 / 电星 / 光星**（冲得快死得快）。剩冰星 / 魂星。

## 阿芙萝（超级大奶妈 · 圣光乐园智囊）/ 沐雨师（天启乐园智囊）/ 千花（殁）

- **阿芙萝（超级大奶妈 · 圣光乐园 · 必杀）**：背生六道金色光翼、持长法杖、自带圣光环的最强治疗系——能**同时给上千人加治疗 + 增益**（致命伤秒愈 / 断肢再生 / 全员属性 + 勇气），一度把节节败退的三乐园联盟硬顶回去。被苏晓列入必杀；曾被轮回乐园紫发女巫打落。是三乐园联盟智囊三巨头之一（擅判生命能量波动·识破雾光虫巢是被强控的）。
- **沐雨师（天启乐园智囊）**：水 + 冰系（超低温）、如沐春风好说话——三乐园联盟的大脑之一（识破雾光虫巢非近期进化、推断苏晓难再控母体）。天启乐园「局势顺就冲得快、逃也最快」、靠出钱出稀有矿石抱死亡乐园大腿。
- **千花（圣光乐园暗杀手 · 殁）**：1 米 5、黑长直、超短裤的顶尖暗杀系——被阿芙萝加 20+ 种增益（隐匿 / 速度 / 护盾 / 空间瞬移）后潜入雾光虫巢。判断刺杀苏晓成功率不足两成，转而**暗杀掉雾光女王**（达成三乐园「灭苏晓虫族」的目标）——随即被苏晓 [血之兽](../力量体系/天赋.md) 连同护她的主坦一并秒杀（上半身落地·临死觉得苏晓像「终极反派大 BOSS」）。

## 寒焰 / 紫发女巫（圣光 / 轮回 辅助系）
- **寒焰（圣光乐园 · 殁）**：蓝焰战锤、冰冷火焰——以为锤死了苏晓、反被斩断双臂 + 放逐贯穿身亡。
- **紫发女巫（轮回乐园 · 辅助系）**：女巫尖帽 + 细烟斗、说话极慢顿挫——稀有的野生辅助系。用「死亡 / 仇恨 / 疯狂」之力把圣光超级大奶妈从高空打落（自己也鼻血 / 力竭）。
