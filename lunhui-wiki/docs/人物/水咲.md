# 水咲

轮回乐园契约者、[蚁王讨伐队](../世界/任务世界/全职猎人-蚁王篇.md) 五成员之一——身材极为性感（单薄胸衣 + 十几公分超短牛仔裤）、腰间挂两把短弯刀的**近战**契约者。得 [尼特罗](艾萨克·尼特罗.md) 认可、是三阶高手。

她是个**心机婊**：占有欲 / 主导欲极强、爱挑拨离间、精神不太正常。最大爱好是在现实世界**结交软妹子**（对女人没兴趣·只为用花言巧语抢走她们的男友、再「好心」安慰、欣赏她们痛苦无奈的表情）。她一眼认出 [苏晓](苏晓.md) 是「斩首的夜」（那个喜欢砍人头的疯子）、当众警惕拆台想挑拨——被尘怼回后又立刻态度急转、对苏晓抛媚眼示好。

## 关系
- **[苏晓](苏晓.md)**：蚁王讨伐队临时队友（认出他是「斩首的夜」·又忌惮又想拉拢）。
- **[尘](尘.md) / 老烟枪**：同队队友（爱挑拨二人·被尘戳穿）。
- **[艾萨克·尼特罗](艾萨克·尼特罗.md)**：讨伐队会长。
