# 水咲

轮回乐园契约者、[蚁王讨伐队](../世界/任务世界/全职猎人-蚁王篇.md) 五成员之一——身材极为性感（单薄胸衣 + 十几公分超短牛仔裤）、腰间挂两把短弯刀的**近战**契约者。得 [尼特罗](艾萨克·尼特罗.md) 认可、是三阶高手。

她是个**心机婊**：占有欲 / 主导欲极强、爱挑拨离间、精神不太正常。最大爱好是在现实世界**结交软妹子**（对女人没兴趣·只为用花言巧语抢走她们的男友、再「好心」安慰、欣赏她们痛苦无奈的表情）。她一眼认出 [苏晓](苏晓.md) 是「斩首的夜」（那个喜欢砍人头的疯子）、当众警惕拆台想挑拨——被尘怼回后又立刻态度急转、对苏晓抛媚眼示好。

## 双重能力 · 围攻蚁王（十六卷39–42章）

她其实是**双重能力发展（近战 + 召唤）**——召唤物是一具**十米高的冰霜巨人**（持续放寒气·能削减 [蚁王](蚁王·梅路艾姆.md) 的速度，因此「不能被打碎」）；近战用两把裹着透明水能量的细弯刀（带「水毒」）。她的压箱底招「**咲蚩魍魉厄**」（需承受两次「死亡」的剧痛才能完成）可爆开蚁王肩伤、并以蚁王体内水分化水蛇缠住它约 0.3 秒（蚁王异常抗性太高）。围攻时她屡次被蚁王尾巴 / 拳头抽碎身体、却没真死——靠 [老烟枪](老烟枪.md) 的「烟替身 / 虚假术」替换、奶妈奶回（每次都要熬死亡前的剧痛）。她藏着真正的压箱底招没拿出来（蚁王战只是第一阶段·之后还要抢心脏）。

## 助攻击杀蚁王 + 抢心脏（十六卷43–53章）

击杀蚁王时她以冰霜巨人残骸 +「**海之冰**」（强度极高·会干扰友军攻击）强行冻结蚁王与会长 3 秒——这正是苏晓 / 尘同步集火的窗口。心脏争夺战中她以**寒冰手臂穿透蚁王尸体掏心脏**、却被 [尘](尘.md) 一枪打断；最终 2/3 心脏被苏晓抢走，她与 [老烟枪](老烟枪.md) 得 1/3。她忌惮苏晓近战（「危险不低于蚁王」），见好就收散去。事后揭出：她其实**愿意接手沙暴献的星火冒险团**，奈何那 30 克心脏是与老烟合出的、老烟急需即时好处、二人合作又不稳，交易遂告吹。
