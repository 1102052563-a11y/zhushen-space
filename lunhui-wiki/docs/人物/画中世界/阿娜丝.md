# 阿娜丝（菲蕾德翠卡·维尔莉·塔薇·阿娜丝）

[画中世界](../../世界/任务世界/画中世界.md) 主画古堡二层「庇护厅」中、轮回乐园印记安全屋「房间III」自带的**女仆幽魂**——身着红色华美长裙、彬彬有礼、气质温柔（[巴哈](../剧毒魔鹰·巴哈.md) 见她第一眼就喊「小红二号」）。全名菲蕾德翠卡·维尔莉·塔薇·阿娜丝，被 [苏晓](../苏晓.md) 简称阿娜丝。

## 魂侍者 · 安眠曲

她是这画之世界王朝消亡前「为战士们而出现」的**魂侍者**。核心价值是在客人安眠时，用「**安眠曲**」（并非歌声、而是一种安抚心灵与灵魂的能力）驱散「野兽」的侵袭——即**加速理智值的恢复**（画中世界的核心属性，被特殊敌人攻击、甚至只是被其气息波及都会掉理智值；苏晓与噩梦之王交战、对方没命中他也掉了理智）。

## 「狂兽症」· 画之世界的病

她道出了画中世界沦落的真相——这世界病了，患上名为「**狂兽症**」的病症:人的理智值缺失时，意识会**兽化**（样貌不变、内心却沦为野兽、会被「消除」）。少数强者理智值极低却硬扛着心灵兽化、处于自我对抗；与他们战斗时，他们会通过身体能量把兽化**传导蔓延**给对手——**心灵兽化越深越好战、嗜血、强大**，到巅峰甚至连肉体都会兽化。加上「烈阳之地」「水之底」也出了问题，画之世界才只剩这一座古堡、原住民全躲进里画世界。

## 关系

- [苏晓](../苏晓.md)（库库林·白夜）：她服侍的客人——靠她的安眠曲加速恢复被画中世界削减的理智值。
- 画中世界的居民 / 战士：她作为魂侍者曾守护、如今仍安抚其心灵的对象。
