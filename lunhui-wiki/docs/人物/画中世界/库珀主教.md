# 库珀主教

[画中世界](../../世界/任务世界/画中世界.md)·沙之世界 太阳教会的一位**主教**，五十岁出头、下巴蓄小胡子、双眼神采奕奕。他是**太阳教会最有成就的药剂配方开发者**，却也是**最不受信任的药剂师**——因为他亲手调药时总爱**夹私货**，教徒们宁可等他的新配方经旁人完善后、由别的药剂师来调，也不敢喝他本人调的。公告板上「尝试库珀主教的实验性药剂」的委托，正是「已经很久没人敢喝」。

## 温房里的会面

[苏晓](../苏晓.md)（化名库库林药师）用「七种药剂预支报酬」的可疑委托把太阳教会高层逼到墙角后，本该有两位主教、四位执事、十二位神官在大教堂五层的**温房**（2000 多平米·仿奇利亚德太阳神族技艺、以一米厚人造玻璃墙置换阳光）与他会面、裁决去留——结果**只来了库珀主教一人**，这本身就说明高层拿他没办法。库珀问苏晓「太阳是什么」（苏晓答「艺术？」、他纠正「是奇迹」），随即宣布不追究其来历、正式**任命苏晓为太阳教会药剂师**（赠红里金日纹的白袍），一句「时间会证明一切」揭过。他还邀苏晓一同开发药剂，被婉拒。他这份务实，来自这个野兽横行、或许百年内就破碎的世界——不极端无法生存。

## 关系

- [苏晓](../苏晓.md)（库库林药师 / 白夜）：他一手任命的药剂师（不知其真实图谋）。
- 太阳教会高层：因他一人赴约放行苏晓，可见高层对这药剂师的无可奈何。
