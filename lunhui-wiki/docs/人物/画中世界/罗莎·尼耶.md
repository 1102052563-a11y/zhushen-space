# 罗莎·尼耶

[画中世界](../../世界/任务世界/画中世界.md) 整个「画之世界」的**创造者**——前任**绘画者**。她的全名一直被血迹遮盖（屋顶诊疗单、太阳教会「调查罗莎」的委托里都只写「罗莎……」），[苏晓](../苏晓.md) 直到在噩梦·古堡病房密室取得**绘画者之血**，才得知她全名**罗莎·尼耶**——因为这是她亲手绘出的世界，随她死去，这世界便不再允许出现她的名字，凡有人写出，都会被血迹抹去。

## 绘画者的机制与她的抉择

**画之世界是绘画者画出来的**（凭「世界画布」= 一种世界之核·集齐 [画卷残片](../../力量体系/装备道具.md) 即可绘世）——绘画者尊贵却并不强大、也非无所不能，且**一生仅能创造一幅画卷、同一时刻只能存在一位**。当她绘的画卷被疯狂（心灵兽化）染黑、世界破碎，她便成了「无用之人」；而新的绘画者（[大小姐](大小姐.md)）已诞生。为不让新绘画者因「唯一性」而夭折，她放出自己全部**源血**、想用来唤醒新绘画者——却被一名古堡医生带走、注入一名强大兽化者体内，反而催生出**史上唯一的七阶段兽化者**（即 [老骑士](大骑士.md)·5 号病患）。唤醒失败后，她只能**留下所有源血、结束自己的生命**，避免新绘画者夭折。她的血至今鲜红如初、富有生命力。

## 关系

- [大小姐](大小姐.md)：她之后诞生的**新绘画者**——因缺她的源血作唤醒物、如今只是「半个绘画者」、无法用世界画布绘世。
- [老骑士（大骑士）/ 5 号病患](大骑士.md)：意外被注入她源血、蜕变成七阶段兽化者、又被她的血治愈理智的人。
- [苏晓](../苏晓.md)（白夜）：在古堡病房密室取得她「绘画者之血」的猎杀者（可交大小姐 / 5 号迹王 / 迹王殿 / 太阳教会·或售轮回乐园换世界级宝箱）。
