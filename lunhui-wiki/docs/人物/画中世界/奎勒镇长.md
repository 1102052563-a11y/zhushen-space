# 奎勒镇长

[画中世界](../../世界/任务世界/画中世界.md)·沙之世界里几百人小镇**永望镇**的镇长，全名**尤·福·奎勒**。[苏晓](../苏晓.md) 接的首个太阳教会阵营任务「永望镇的异响」把他牵出来:苏晓敲门时，门缝里只露出一只**浑浊棕黄、遍布血丝、瞳孔扩散**的眼睛，语气却平静无比——这精神状态与语气的巨大反差，让苏晓一眼看出他已是**兽化者**。

## 兽化中仍存理智的好人

他向**残忍型**的野兽转变（人形、头颈臂生满黑毛、头部呈狼相），已达**三阶段兽化**——这个阶段多数平民早已失智，他却仍保有一丝理智。苏晓判断:他之所以兽化，多半是**一直忧心小镇安危、被永望镇的「异响」悄然影响所致**，是个意志坚定的好人。被锯刃刀刺穿门板重创后，他一边大口撕咬吞食自己的脏器（「真是……诱人的味道」），一边在剧痛下短暂回神，用爪尖点着自己的头留下遗言:**「不是我，原因不是我，它在这里」**——随即神情转凶，被苏晓一刀斩落半人半狼的头颅。他的妻子、儿子儿媳都已被他吃剩残骸。这句遗言（夜间、脑部、来源不明之声）正是苏晓破解「异响 = 噩梦」的关键。

## 关系

- [苏晓](../苏晓.md)（白夜）：接太阳教会任务来永望镇、以「锯刀疗法」了结他并读出其遗言线索的猎杀者。

## 🆕 一家四人的绝笔 · 灰笔（44卷51章）

苏晓进入噩梦·永望镇后，在镇长家三楼的枯尸旁读到他留下的长篇绝笔，才知这位普通人一家四口的全部努力:永望镇被**噩梦缠住**——现实是本体、噩梦是投影，投影里的东西（如兽化）会映射回现实，就算逃出小镇，夜里睡去意识仍会回噩梦、身体自行往镇子走。他没有超凡之力、没有坚定意志，是**当脑颅医生的儿子用扁针从他眼窝刺入、一小块一小块切除他的大脑**，才让濒溃的理智一次次喘息、不让挚爱的小镇沦为野兽。他的妻子研究出「噩梦与现实的唯一连接是**意志**」，一家人切掉了太多大脑、濒临极限，遂决定最后一次进噩梦、以命验证「**平行推进**」之法:噩梦里留一个极理智者引导、现实里的帮手去打醒怪物本体。为让两界能沟通，他们把**妻子、儿子、儿媳在噩梦中的躯骸烧成粉、压成三根手指长的「灰笔」**——用它在噩梦里写字，现实中相同位置就能看到。他们自己终究失败（理智不够、无法「见证抹杀」怪物），却把这三根灰笔和方法留给了后来的太阳教徒。他在绝笔里恳求:别迁怒永望镇居民、一切是他的责任，还有那句「人，总是希望被记住，不是吗」。这个 294 年的小镇，是他们一家人拼死也不肯交出的家。
