# 奎勒镇长

[画中世界](../../世界/任务世界/画中世界.md)·沙之世界里几百人小镇**永望镇**的镇长，全名**尤·福·奎勒**。[苏晓](../苏晓.md) 接的首个太阳教会阵营任务「永望镇的异响」把他牵出来:苏晓敲门时，门缝里只露出一只**浑浊棕黄、遍布血丝、瞳孔扩散**的眼睛，语气却平静无比——这精神状态与语气的巨大反差，让苏晓一眼看出他已是**兽化者**。

## 兽化中仍存理智的好人

他向**残忍型**的野兽转变（人形、头颈臂生满黑毛、头部呈狼相），已达**三阶段兽化**——这个阶段多数平民早已失智，他却仍保有一丝理智。苏晓判断:他之所以兽化，多半是**一直忧心小镇安危、被永望镇的「异响」悄然影响所致**，是个意志坚定的好人。被锯刃刀刺穿门板重创后，他一边大口撕咬吞食自己的脏器（「真是……诱人的味道」），一边在剧痛下短暂回神，用爪尖点着自己的头留下遗言:**「不是我，原因不是我，它在这里」**——随即神情转凶，被苏晓一刀斩落半人半狼的头颅。他的妻子、儿子儿媳都已被他吃剩残骸。这句遗言（夜间、脑部、来源不明之声）正是苏晓破解「异响 = 噩梦」的关键。

## 关系

- [苏晓](../苏晓.md)（白夜）：接太阳教会任务来永望镇、以「锯刀疗法」了结他并读出其遗言线索的猎杀者。
