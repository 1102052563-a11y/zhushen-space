# 大小姐

[画中世界](../../世界/任务世界/画中世界.md) 主画世界古堡一层墙角的白发小女孩——年约十四五岁，下半张脸戴着**打满银铆钉的银质脸罩**（华美中透残酷），坐在高脚椅上对着画板作画。她是本世界**画卷争夺战的中枢与裁决关键**。

## 中立核心 · 友善度

- **绝对中立·不可伤害**：轮回乐园明确警告——大小姐是中立单位、不受魅力属性交涉影响（[苏晓](../苏晓.md) −9 魅力对她无初始敌意）；若伤害她，将成为**噩梦之王、烈阳君主**等强大存在的绝对死敌、并被画中世界排斥。苏晓试过把 [烈阳之怒·阿波罗](../../力量体系/装备道具.md) 塞进她衣兜（想在主画布置），被她一句「不可以」制止，只得作罢——她是「暂不考虑物理交涉」的关键人物。
- **画卷残片的收纳者**：参战者向她上交 [画卷残片](../../力量体系/装备道具.md) 即计入排行（每块 +5 友善度）；友善度超 20 可进古堡二层。她收下残片、也随手给苏晓、[布布汪](../布布汪.md) 画肖像画（被塞阿波罗后哑然地盯着苏晓看、继续作画）。
- **象征**：她本身像是这濒死世界残存意志的化身——那句反复的「保持理智，这很重要」，与她安静作画的姿态，构成画中世界最诡谲的底色。

## 关系

- [苏晓](../苏晓.md)（库库林·白夜）：代表轮回乐园来夺画卷的参战者——想讨好她冲排行、却被禁止在她身上 / 主画动手脚。
- 噩梦之王 / 烈阳君主：伤害大小姐会招致的强大存在（各里画世界的主宰）。
