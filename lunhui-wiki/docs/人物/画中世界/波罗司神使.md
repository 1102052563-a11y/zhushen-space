# 波罗司神使

[画中世界](../../世界/任务世界/画中世界.md) 第三个里画世界「海之底」中，**六号庇护城**的神使，全名**波罗司·涅罗**。他体重六百斤以上、身高 2 米 7，左脸有条穿满小铁环的蜈蚣状疤痕，三大爱好是**好色、美食、人体器官收集癖**。他在外名气不大、为人低调，可六号庇护城每年拿到的粮食物资配给最多——这恰说明海神在**刻意喂饱他、不让他搞事**。他能用手段让部下当场暴毙（当年海神就这样控制他），神使之位坐得很稳。

## 沦为「傀儡神使」

[苏晓](../苏晓.md)、[伍德](沃波·伍德.md)、[罪亚斯](罪亚斯.md) 三人初到六号庇护城，急需一个能瞒过主城海神的可信身份，遂拿他开刀:苏晓一进门就秒了他的光头女护卫，巴哈的异空间把小楼与世隔绝，三人围殴重创他、清光其护卫（[黑A](../../力量体系/装备道具.md) 夺舍夺舍了憨批石斑鱼脸清尸、伍德的欺诈卷轴把打烂的现场「倒放」复原）。罪亚斯用**寄髓虫 + 冥神信徒手段**篡改他的**认知**（不是奴役、而是让他打心底认定苏晓三人是「结识多年、回来帮他的好兄弟」·稳定得多）。随后太阳鸟追来，罪亚斯又篡改他的认知让他**以为是自己招惹了太阳鸟**——于是他心甘情愿调集近千名部下当炮灰围攻太阳鸟、替三人背下这口惊天大锅。

## 关系

- [苏晓](../苏晓.md) / [伍德](沃波·伍德.md) / [罪亚斯](罪亚斯.md)：把他做成傀儡、借他身份潜往主城搞海神的恶阵营三人组。
- 海神：喂饱他、不让他搞事的主城之主——他早想摆脱海神的掌控。

## 🆕 甩锅上缴 · 女儿兽化引狼入室（44卷109–110章）

太阳鸟一役后他被认知篡改、心甘情愿替苏晓三人善后:把招来太阳鸟的锅甩给那名喊「誓死追随」的大嘴海族（被海神心腹 [黑角·罗厄](命祭司·索菲娅.md) 乱刃砍死），又把苏晓提供的一枚**太阳焰·爆燃纹印**当「赃物」上缴海神——海神收了很满意、既往不咎。他向海神递交的人员清单把苏晓三人写得极简（显出「生怕被海神挖走」），部下们也集体咬定没见过白夜。可他那位得了**兽化症**的女儿成了破绽:[命祭司·索菲娅](命祭司·索菲娅.md) 从女儿「病情好转」顺藤摸瓜，挖出了能治兽化的苏晓——他这傀儡神使，反倒把三条大鱼送进了海神的门。
