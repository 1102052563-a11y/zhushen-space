# 罪亚斯

[画中世界](../../世界/任务世界/画中世界.md) 中**陨灭星**（古神的老巢）的参战者——身穿白色神职长袍、脖颈戴满是眼球的十字架、双手背上移植着眨动的眼睛、腮下随时生出可扯下来嚼吃的黑色触手。能在古神老巢陨灭星混得好，他绝对是**「信仰疯子 + 老阴哔」**双料。

## 陨灭星的癫狂

他落座就微笑祈祷、随手扯下腮下触手嚼食、全身生出扭动的黑色触手；[巴哈](../剧毒魔鹰·巴哈.md) 在他身上感到强烈危险。[苏晓](../苏晓.md) 估测其**实力与自己相近**（拼命五五开）。他最标志性的癫狂是**挖眼生吃**——逮住 [天羽](天羽.md) 后三指成爪抠出眼球塞嘴爆浆、满脸享受，还想去抠第二只（被苏晓一斧敲墙警告才收手），一句「小哥你可真好吃」道尽陨灭星的血腥暴戾。他对苏晓身上那股「对古神 / 恶神的杀意与恶意」极为忌惮，怀疑苏晓到底杀了多少古神。

## 噩梦之地 · 三方联盟

他与魔鬼族 [沃波·伍德](沃波·伍德.md) 早有约定（伍德把从上任猎命人骗来的「黑暗住民」交给他、他借此成了「半个生存者 / 背叛者」）。他对生存游戏那一块画卷残片毫无兴趣——真正图谋是**保证不被虚空之树驱逐、想办法让本体脱困后回噩梦世界弄死 [噩梦之王](../../世界/生物万族/index.md)、夺更多画卷残片**（与苏晓、伍德想法一致）。三人遂结**三方临时联盟**:先合作清场灭噩梦之王、最后「三选一」分胜负（谁敢中途搞幺蛾子、其余两人群起拍碎其脑壳）。他负责坑洛希 / 炎启那边。

## 关系

- [苏晓](../苏晓.md)（库库林·白夜 / 猎命人）：三方联盟中实力与他相近的盟友——苏晓那股杀古神的气息让他忌惮。
- [沃波·伍德](沃波·伍德.md)：早有约定、把他拉成背叛者的魔鬼族骗术师（被伍德当面说「心怀鬼胎」）。
- [天羽](天羽.md)：被他挖眼生吃、留作活口的羽族渣男。
