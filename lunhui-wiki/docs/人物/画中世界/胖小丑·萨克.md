# 萨克（胖小丑）

[画中世界](../../世界/任务世界/画中世界.md) 噩梦之地游乐场门口的胖小丑，本名**萨克**，浮夸花哨、爱丑化自己麻痹他人，持一把曾割开无数游戏者咽喉的匕首「**嘲笑**」。他与 [噩梦之王](../../世界/生物万族/index.md) **只是合作、不俯首称臣**，故被噩梦之王当**炮灰**推去门口拖时间。

## 被吞匕首的领路人

面对 [苏晓](../苏晓.md)、[沃波·伍德](沃波·伍德.md)、[罪亚斯](罪亚斯.md) 三个凶神，他不想替噩梦之王送死、当场倒戈:主动供出游乐场里有三块 [画卷残片](../../力量体系/装备道具/index.md)。伍德用巧妙手段限制他的战力——不签契约（免他绝望）、而是「请他表演吞刀」逼他**把自己的匕首吞进胃囊**（熟悉之物、心理上比签约易接受、且限制行动力却不生绝望），苏晓再给他喝下缓痛的药剂。此后他老实给三人领路，供出骨屋赌徒藏着三块画卷残片、以及那间「太阳灼痕大石屋」（[嘟嘟咕咕](嘟嘟咕咕.md) 所在）。他还在苏晓的血气威压 + 唱红脸下，「想起」自己捡到的**学者木棍**藏在大石屋暗格里、乖乖交出。被这三个家伙的气息一衬，他觉得自己「好像也不算坏」。

## 关系

- [苏晓](../苏晓.md) / [沃波·伍德](沃波·伍德.md) / [罪亚斯](罪亚斯.md)：逼他吞匕首、领路的三方联盟（苏晓唱红脸给药、伍德唱白脸限战力）。
- [噩梦之王](../../世界/生物万族/index.md)：只与他合作、把他当炮灰推出去送死的噩梦世界主宰。
