# 奥斯·康拉德

[画中世界](../../世界/任务世界/画中世界.md)·海之底主城**神恩城**的「大神子」——[海神](../../世界/生物万族/index.md)（奥斯·亚特兰蒂）的长子，也是**主城内唯一敢和海神叫板**的人。他心狠、敢干，怀着一个明确的梦想:**弄死自己的父亲**（[巴哈](../剧毒魔鹰·巴哈.md) 一句「原来是个带孝子」）。海神虽是主城绝对的掌权者，却也压不住这个儿子——两方势力在主城盘根错节、僵持不下（谁也不敢发动大规模乱战、否则主城会变废墟、上百年缓不过来），这正是 [苏晓](../苏晓.md) 潜入后最想借力的「突破口」。

## 「奥斯」姓氏与王裔之谜

他的姓氏「奥斯」是画之世界**王裔**的姓氏（[烈阳君主](烈阳君主.md) 奥斯·瓦伦丁、海神奥斯·亚特兰蒂皆姓奥斯）。可 [光焰领主](../../世界/生物万族/index.md)（驴哥）的存在证明「奥斯一族的最后血脉已断绝」——苏晓由此推断:**海神是鸠占鹊巢**，占了奥斯一族建立的海底主城、再冒姓奥斯给自己和子嗣安上王裔之名（免落人话柄）。耐人寻味的是，替 [光焰领主（驴哥）](../../世界/生物万族/index.md) 发布「治疗委托」的，正是这位大神子康拉德——他与那个追杀弑王者的光焰领主之间，藏着尚未揭开的关系。

## 关系

- [海神（奥斯·亚特兰蒂）](../../世界/生物万族/index.md)：他一心想弄死的父亲、主城之主。
- [苏晓](../苏晓.md)（白夜）：潜入主城、想借他与海神的矛盾搞海神的猎杀者。
- [光焰领主（驴哥）](../../世界/生物万族/index.md)：他暗中发布治疗委托的对象——关系成谜。

## 🆕 弑父的真相 · 继承海神诅咒（44卷116–120章）

他为弑父隐忍了大半生:幼年发现海神会轮番清理长大的优秀子嗣，便先装愚笨保命；15 岁「失手」毁容自己半张脸、放出「海神迫害亲子 + 乱伦」的劲爆谣言；27 岁按海神授意创立一个「可控的反叛势力」诱杀真反叛者——却借此真的壮大自己，终于站起来。他集齐 [苏晓](../苏晓.md)（斩海神）+ [索菲娅](命祭司·索菲娅.md) / [黑角·罗厄](命祭司·索菲娅.md) / 潜影 / 休鲁大师（海神的心腹尽数倒戈）+ [凯撒](../尼古拉斯·凯撒.md)（扮老仆送掺毒念髓强行沉默海神身体能量）+ [伍德](沃波·伍德.md) / [罪亚斯](罪亚斯.md)（挡两神官），5 分钟暗杀成功。苏晓一刀斩死海神寄体后，**海神的本源神灵能量转移进了他体内**——他将逐渐被同化、日后也会渴望成为圣神、也会残杀自己的子嗣（王裔诅咒的轮回）。他姓氏「奥斯」是祖先鸠占鹊巢盗来的，家族世代因此受海诅咒折磨。他不是好儿子、却是不愿子嗣陪葬的好父亲；活下来就有打破诅咒的希望。
