# 洛希

[画中世界](../../世界/任务世界/画中世界.md) 中**奥术永恒星**的参战者之一——身穿黑红法袍、戴兜帽、只露白皙下半脸的女施法者。她是**掌控者·哈曼的嫡系孙女、生命学派的「大小姐」**（哈曼是奥术永恒星元老、灭法时代曾与今日的至高之人平分秋色）。

## 被误解的实力派

奥术永恒星新生代对她的印象都是「靠资源堆、不常露面、傲慢」——实则不然。她自幼异于常人的早熟、修行魔能不忘实战、实战经验丰富，只因直系亲属太多（同父同母的兄姐弟就六人、哈曼的生命学派人丁兴旺），她**隐忍太久、急需一鸣惊人**换取更多资源。这次进画中世界，正是她等待的机会——她一度想凭**除掉灭法者 [苏晓](../苏晓.md)** 来一鸣惊人。

## 与苏晓的博弈

初见苏晓，她与之对视后立刻移开目光。苏晓由此判断:她有阅历但离「老阴哔」还嫩、想对付自己却不会立刻出手、是有备而来且自信能在初期抢到最多 [画卷残片](../../力量体系/装备道具.md)——一个「画卷残片提款姬」（疑有锁定残片的能力）。她还保管着虚空之树租借的 **[洞察眼](../../力量体系/装备道具.md)**（斗技场十余万观众的直播视角全靠她）。噩梦之地里她表现出对逃杀规则的老到理解（教 [炎启·索耶格](炎启·索耶格.md)「遇猎命人就躺地装死」），撞见猎命人苏晓时果断弃搭档独逃、瞬间躺地隐匿脱身——比只会花哨操作的炎启·索耶格「教科书」得多。

## 🆕 被捕兽夹「千层饼」耍到自闭（44卷13–14章）

她被 [沃波·伍德](沃波·伍德.md) + [罪亚斯](罪亚斯.md) 堵在无岔路的迷宫段、又遭猎命人苏晓追杀——在闷热低氧的迷宫里被追了三个多小时（她体力仅 30 点·全速跑到脱水、眼中无光），倔驴脾气地割动脉自尽而非被擒（这一幕经洞察眼反馈、让斗技场的新生代施法者与有荣焉）。复活后她自信「已找到三处锁盘、回去找队友 2 小时决胜负」，却在初生广场出口踩进苏晓布的**捕兽夹「千层饼」局**:前跳、后跳、两侧跳全被算死（假捕兽夹诱她判断、真捕兽夹夹住手脚），[巴哈](../剧毒魔鹰·巴哈.md) + [布布汪](../布布汪.md) 还一路补夹——她苦撑的「四名队友是希望」话音未落，就看到队友全被苏晓等吊着丢来。最终她和全部生存者一起被扔进初生广场，眼睁睁看苏晓赢下生存游戏。

## 关系

- [苏晓](../苏晓.md)（库库林·白夜 / 猎命人）：她想借除掉的灭法者、画卷残片的头号竞争对手；噩梦之地被他追杀三小时、又被捕兽夹千层饼耍到擒获。
- [沃波·伍德](沃波·伍德.md) / [罪亚斯](罪亚斯.md)：堵她、当「背叛者」的魔鬼族骗术师与陨灭星疯信徒。
- [炎启·索耶格](炎启·索耶格.md)：与她同批代表奥术永恒星的搭档（被苏晓斩杀·她弃他独逃）。
- 掌控者·哈曼：她的祖父、奥术永恒星生命学派元老。
