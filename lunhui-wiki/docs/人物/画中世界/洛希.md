# 洛希

[画中世界](../../世界/任务世界/画中世界.md) 中**奥术永恒星**的参战者之一——身穿黑红法袍、戴兜帽、只露白皙下半脸的女施法者。她是**掌控者·哈曼的嫡系孙女、生命学派的「大小姐」**（哈曼是奥术永恒星元老、灭法时代曾与今日的至高之人平分秋色）。

## 被误解的实力派

奥术永恒星新生代对她的印象都是「靠资源堆、不常露面、傲慢」——实则不然。她自幼异于常人的早熟、修行魔能不忘实战、实战经验丰富，只因直系亲属太多（同父同母的兄姐弟就六人、哈曼的生命学派人丁兴旺），她**隐忍太久、急需一鸣惊人**换取更多资源。这次进画中世界，正是她等待的机会——她一度想凭**除掉灭法者 [苏晓](../苏晓.md)** 来一鸣惊人。

## 与苏晓的博弈

初见苏晓，她与之对视后立刻移开目光。苏晓由此判断:她有阅历但离「老阴哔」还嫩、想对付自己却不会立刻出手、是有备而来且自信能在初期抢到最多 [画卷残片](../../力量体系/装备道具/index.md)——一个「画卷残片提款姬」（疑有锁定残片的能力）。她还保管着虚空之树租借的 **[洞察眼](../../力量体系/装备道具/index.md)**（斗技场十余万观众的直播视角全靠她）。噩梦之地里她表现出对逃杀规则的老到理解（教 [炎启·索耶格](炎启·索耶格.md)「遇猎命人就躺地装死」），撞见猎命人苏晓时果断弃搭档独逃、瞬间躺地隐匿脱身——比只会花哨操作的炎启·索耶格「教科书」得多。

## 🆕 被捕兽夹「千层饼」耍到自闭（44卷13–14章）

她被 [沃波·伍德](沃波·伍德.md) + [罪亚斯](罪亚斯.md) 堵在无岔路的迷宫段、又遭猎命人苏晓追杀——在闷热低氧的迷宫里被追了三个多小时（她体力仅 30 点·全速跑到脱水、眼中无光），倔驴脾气地割动脉自尽而非被擒（这一幕经洞察眼反馈、让斗技场的新生代施法者与有荣焉）。复活后她自信「已找到三处锁盘、回去找队友 2 小时决胜负」，却在初生广场出口踩进苏晓布的**捕兽夹「千层饼」局**:前跳、后跳、两侧跳全被算死（假捕兽夹诱她判断、真捕兽夹夹住手脚），[巴哈](../剧毒魔鹰·巴哈.md) + [布布汪](../布布汪.md) 还一路补夹——她苦撑的「四名队友是希望」话音未落，就看到队友全被苏晓等吊着丢来。最终她和全部生存者一起被扔进初生广场，眼睁睁看苏晓赢下生存游戏。

## 🆕 刮痧女技师 · 殁于沙之世界（44卷32–33章）

沙之世界里她与搭档 [炎启·索耶格](炎启·索耶格.md)（善阵营）迎战恶阵营的苏晓——索耶格化 2 米 3 壮汉、炎系近战大招被苏晓一刀斩首后，她成了最后一个。她本以为凭身后几百道**元素环刃**（每道仅 30 公分、追求飞行速度与切割力）能对付「灭法者跑不过近战」的绝境，谁知撞上苏晓的**绝魔体质**（免疫 40% 法系伤害、每受一次法术攻击 +2% 免疫至 20%）+ 灵影体质 / 神裁 / 体魂 / 伯格之心堆起的海量血量与抗性——她那招「环刃舞·骨葬」把几百环刃分裂成上万道刮向苏晓，每道也只造成 **7~10 点元素伤害**，被网友（斗技场观众）封为「**刮痧女技师**」。苏晓硬扛环刃风暴、逼近后一记晶体层横抡打碎她手臂、一刀斩其胸膛、贯穿咽喉与心脏。她至死没求饶、没怂（还留下遗言「你这灭法的混蛋太谨慎了、让人讨厌不起来、但你一定还单身、活该」），赢得苏晓的尊重。她一死**奥术永恒星阵营临时淘汰**（还剩第三名额 [乌鸦女](乌鸦女.md) 在候补），掉一枚秘法宝箱。

## 关系

- [苏晓](../苏晓.md)（库库林·白夜 / 猎命人）：她想借除掉一鸣惊人的灭法者——噩梦之地被追杀三小时、沙之世界又被斩杀（她的「刮痧」环刃被苏晓的抗性硬扛）。
- [乌鸦女](乌鸦女.md)：她死后接替入场的奥术永恒星第三名额（真正的杀手锏）。
- [沃波·伍德](沃波·伍德.md) / [罪亚斯](罪亚斯.md)：堵她、当「背叛者」的魔鬼族骗术师与陨灭星疯信徒。
- [炎启·索耶格](炎启·索耶格.md)：与她同批代表奥术永恒星的搭档（被苏晓斩杀·她弃他独逃）。
- 掌控者·哈曼：她的祖父、奥术永恒星生命学派元老。
