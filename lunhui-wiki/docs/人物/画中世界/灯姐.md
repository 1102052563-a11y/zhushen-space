# 灯姐

[画中世界](../../世界/任务世界/画中世界.md) 主画世界·**噩梦·古堡病房**深处的守卫，本是**72 号病患**——一个心灵兽化被罗莎血抑制、却被「海之怨怒」把头肿成大肉瘤的女病患。古堡医生把她的大脑切割改造:脖颈打铆钉、头颅罩上金属探照灯、面部聚起十几颗眼球，放出的**浊光**经探照灯聚焦、强度**比 [肿胀之眼](../../世界/生物万族/index.md) 还强几倍**（被照到即头晕耳鸣、重影、全身无力、理智狂掉）。她一身破烂长袍如布条垂下、两条沾血大长腿踩金属高跟鞋（她生前极爱这双鞋），迈着诡异大步「哒哒」巡视——恐怖与性感诡异地搭上了边。[苏晓](../苏晓.md) 称她「灯姐」。

## 无解的守卫 · 改造的用意

她是**这噩梦里一对一无敌**的存在（苏晓凭龙影闪 + 敲声调虎离山、才险险从她身后溜进密码门），改造者甚至让她**惧怕太阳之力**（97% 太阳教徒都会太阳奇迹·这是改造者对自己人的保护:凭太阳之力过她这关 = 有资格知晓密室秘密，很「太阳教会」的风格）。改造她的真正用意，是**防止 [老骑士](大骑士.md)（5 号病患·七阶段兽化者）回病房夺走 [绘画者之血](../../力量体系/装备道具/index.md)**——有古堡病房场景加持，她能与兽化的老骑士「碰一下」。她把混进来的 [神隐](神隐.md) 用尖锥麻痹、钩在后腰当挂件。她无疑是个可怜人，却也是探索者绕不过的死关。

## 关系

- [苏晓](../苏晓.md)（白夜）：靠调虎离山从她眼皮底下溜进密码门的猎杀者。
- [老骑士（5 号病患）](大骑士.md)：改造她就是为了拦住他回来夺绘画者之血。
- 古堡医生 / 太阳教会：把她（72 号病患）改造成守卫的人——那位医生在诊疗单里问「把你改造成怪物，恨我吗」。
