# 餐刀姐

[画中世界](../../世界/任务世界/画中世界.md) 主画世界·古堡二层庇护厅 1 号房间的房客——[苏晓](../苏晓.md) 因她「狠中透惧」的嗓音，脑补出一名披头散发、眼窝深陷、身穿松垮衣袍、手持餐刀、神经有点衰弱的三十多岁妇人（一敲门就用一把加持了未知能量、刺穿门板的**噩梦餐刀**回敬）。实则她并非普通房客，而是**大小姐的仆从**:主业侍奉大小姐、副业给 2/3/4/6 号房客每两天送一餐（食物来自某个里画世界）。她是庇护厅六名房客里**唯一能从里侧开门**的人。

## 刀子嘴豆腐心

她一身键盘侠资质、装刺猬装得凶，恶意其实不强、偏善阵营。被苏晓一手「松手让她后仰撞头」整到怀疑人生后，态度反倒软了。苏晓丢进一颗**温热的太阳石**（对只剩一座古堡、外面全是紫黑半流体、早没了太阳的主画世界，这是稀世珍宝·她原有的那颗又小又劣），她便回赠一支**萃取后的镇静剂**（王朝技术遗留·圣灵级·可恢复 300~390 理智值）。她还替大小姐把关:证实苏晓知道密纹码后半（「14」是倒数第三、二位），并提醒他**进第四幅画之前一定要用到密纹码**，否则就白得了。

## 关系

- [大小姐](大小姐.md)：她侍奉的主人（她那些待晒的衣物、太阳石都是大小姐的）。
- [苏晓](../苏晓.md)（白夜）：拿温热太阳石换她镇静剂 / 密纹码的猎杀者——被她的餐刀刺门、也被他整到撞头。
