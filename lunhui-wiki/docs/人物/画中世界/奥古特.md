# 奥古特

[画中世界](../../世界/任务世界/画中世界.md)·沙之世界 太阳教会的一名教徒，两米五的五大三粗壮汉，几年前号称**红腕·奥古特**——擅用锯斧、左手天生神力，能轻松扯开敌人喉咙、单手刺进敌人内腔掏脏器。多年狩猎心灵野兽积下重伤:24 根肋骨没一根是顺滑的、每根都断过又没校正就愈合，脏器一团糟，生命上限已从 100% 掉到 87.9%——他一瘸一拐来求医，只盼能看着五岁的儿子长大。

## 灵影线的第一位「贵客」

他是 [苏晓](../苏晓.md)（化名库库林药师·靠「愈者」权限治教徒赚声望）遇到的第一个重伤大客户。苏晓一套「伸左手 + 说变性趣闻」分散他注意力、一发注射器弩箭麻翻他、让 [巴哈](../剧毒魔鹰·巴哈.md) 按他拇指强签「手术同意书」，再用**灵影线**（青钢影微米能量丝线）开胸缝合脏器、清淤血——过程粗暴到他一进门一瘸一拐、出去时被抬着横躺，惨叫响彻走廊。但治疗效果确实优异（恢复 95.72%），苏晓也借他一人摸清了「伤越重、声望越多」（这一台手术得 7620 声望）。苏晓由此感慨:太阳教会每名教徒都是一具「可移动的声望值」。

## 关系

- [苏晓](../苏晓.md)（库库林药师 / 白夜）：给他做灵影线开胸手术的药剂师——效果虽好、过程「生不如死」。
