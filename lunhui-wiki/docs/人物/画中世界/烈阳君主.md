# 烈阳君主

[画中世界](../../世界/任务世界/画中世界.md) 第二个里画世界「沙之画」的**掌管者**——曾经强大、英俊、阳刚的太阳之君，统领着一个庞大的**新王国**（结构不像王国、更像利益集合体:没货币、没法律，只因这个狂兽横行、饿殍遍野的世界根本立不起法度）。他手握大量 [画卷残片](../../力量体系/装备道具.md)，是 [罪亚斯](罪亚斯.md) / [伍德](沃波·伍德.md) 等人打算去死磕、夺取残片的目标（而 [苏晓](../苏晓.md) 选择闷声在太阳教会发财、隔岸观火）。他比 [噩梦之王](../../世界/生物万族/index.md) 更强——噩梦之王一度想带画卷残片来投靠这位「隔壁邻居」，还被他有点看不起。

## 被太阳教会锤到心态崩了的「苦主」

他这辈子说过最后悔的一句话是「你们脑子有问题吗？」——某次他在不知对方身份时杀了一名（没戴标志性头桶的）**[太阳教会](../../世界/任务世界/画中世界.md)** 信徒，太阳信徒找上门只要他「表达哀痛」，他这一句顶回去，两天后就迎来蜂拥而至的围攻。此后半年他共遭 **2947 次暗杀、85 次围攻**（平均每天不到一小时就被暗杀一次），强大英俊的君主都憔悴了、心态几近崩溃，却始终没搞懂这是多大的仇。直到一名信徒捧着那罐骨灰进王宫要他道歉，他疲惫地致了歉，两位主教便宣布停战——原来太阳教会疯狂弄他，不为利益名誉，只为讨一句歉（按其信条:蒙受屈辱而死者死后无法向太阳飞升，这是头等大事）。从此他严令手下**离那群「太阳疯子」远点**:不贪财、不好色、不慕名利、一心崇拜太阳的疯子，在他看来才是最可怕的。

## 关系

- [太阳教会](../../世界/任务世界/画中世界.md)：忌惮至极的宿敌——被其 2947 次暗杀折磨到不敢再招惹。
- [苏晓](../苏晓.md)（白夜）：选择在太阳教会闷声发财、隔岸观火其领地被袭的契约者。
- [罪亚斯](罪亚斯.md) / [伍德](沃波·伍德.md)：打算去他地盘死磕、夺画卷残片的老阴哔。
- [噩梦之王](../../世界/生物万族/index.md)：想带残片来投靠他的「邻居」（被他看不起）。

## 🆕 苏晓的背刺布局 · 凯撒来谈「野兽心」（44卷66章）

[苏晓](../苏晓.md) 从治疗教徒时旁敲侧击得知:他的力量**天生源自太阳系血脉**（非信仰）——故一瓶不朽级的**太阳圣药（完美）**对他是无法拒绝的礼物。苏晓遂派 [凯撒](../尼古拉斯·凯撒.md) 代自己到圣丹城密会他:以「合作争夺**野兽心**」为由送上该药，让他相信苏晓愿为野兽心随时叛出太阳教会、与他站一队（他日后必能查到教会高层对苏晓的排斥、更信这点）。这是**软硬兼施**:太阳圣药 + 药剂师身份是利诱、野兽心是合作理由，而药里混的**切葛细胞**是保险（16~17 天内一旦摄入聂氧，即狂裂增生成一团烂肉球）。苏晓合作三目的:①借他之手压制 / 消灭难缠的 [伍德](沃波·伍德.md) 与 [罪亚斯](罪亚斯.md)、②趁机多夺画卷残片、③**以最低风险反过来灭掉他、夺其世界之源与宝箱**。

## 🆕 真名奥斯·瓦伦丁 · 被攻心离间的理想之王（44卷71–74章）

密会 [苏晓](../苏晓.md) 时他自报是**「王朝」瓦伦丁一族的王裔**（真名**奥斯·瓦伦丁**·天生太阳血脉），故敢在画中世界自称君主。他并非纯粹的暴君:傲慢是他的面具（已用它坑死众多强敌），实则心思缜密、判断精准，更有一份**理想**——用画卷残片把沙之世界填补到完整独立、压制疯狂与兽化、让此地享千年安宁，故新王国有众多强者死心塌地追随他。他背后还有一位效忠的长辈智者「**阿泽乌**」为他出谋划策（求稳、他却急于给手下看到光明未来——这正是二人的裂缝）。苏晓一场攻心:抛 9 块 + 太阳教会 21 块 + 伍德罪亚斯共 11 块「41 块画卷残片」的诱饵、以「你是不是阿泽乌的**傀儡**」点燃他的**自满 / 猜忌 / 分歧 / 急于求成**四层隔阂，还预付残片「投资他的理想」——无形之刃切割其君臣之心。他喝下苏晓送的太阳圣药（切葛细胞已入体），布布汪则潜踪跟他摸清了画卷残片的藏处。

## 🆕 三方隔空爆锤 · 王宫盛宴（44卷73–74章）

被 [苏晓](../苏晓.md)（太阳教会）、[罪亚斯](罪亚斯.md)（操控野兽群）、[伍德](沃波·伍德.md)（被弃人聚集地新领袖）三方暗通、火力全倾在他那位阿泽乌智者身上（凯撒搅屎 + 伍德策反红蜂夫人 + 罪亚斯控布劳布卢兄弟），智者被隔空爆锤到心力交瘁、明哲保身闭门不出。他又遭死亡乐园盲眼杀手 [水哥（恩左）](../联盟星/水哥.md) 伏击（四从卫废两个），被逼得不敢出圣丹城。眼看罩不住局面，他定于当晚在圣丹城王宫设宴、以灵魂石为筹码请伍德 / 罪亚斯暂时停战——这场「王宫盛宴」，正落进苏晓「午夜吃大餐」的收网局。

## 🆕 王宫盛宴 · 一世英名秀死于自己的鸿门宴（44卷75–77章）

他设的「王宫盛宴」本是精心的鸿门宴——伍德、罪亚斯若只来一个就和谈、全来就用只有王族血脉能启动的**王朝机关**一网打尽（铲除顺序:伍德→罪亚斯→水哥→苏晓→凯撒→天启姐妹花）。他甚至先支开苏晓去「药师交流会」、以为拴住了这最大变数。可他万没想到:那个从背后一手贯穿他心口的「心腹 [孤骸·兰斯洛](孤骸·兰斯洛.md)」，是 [苏晓](../苏晓.md) 杀了兰斯洛、用 [先古面具](../../力量体系/装备道具.md) 顶替的。掀桌开战、太阳之力迸发、正高呼「我奥斯·瓦伦丁必挽救这世界」的瞬间被刺穿——晶体手臂送进**初代吞噬者**，加上他先后九次摄入的**切葛细胞**（八次微量·最后一杯酒是巨量）遇聂氧反应，身体再生限制彻底关闭，1.5 秒增生成一个直径五米、如海胆般刺穿四周部下的**黑肉球**，在「我，败了？」的错愕中暴毙。他死于**身体增生 + 生命透支 + 能量透支 + 吞噬者内噬**，从头到尾没能出一招——苏晓布局这么久，绝不给这位沙之世界最强者一丝出手的机会。（讽刺的是他其实智谋中上、武力顶尖、更怀理想，却因傲慢面具与急于求成，被苏晓攻心攻到「自己把自己秀死」。）
