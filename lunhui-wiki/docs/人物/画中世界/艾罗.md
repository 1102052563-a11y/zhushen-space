# 艾罗

[画中世界](../../世界/任务世界/画中世界.md)·沙之世界 太阳教会的一名教徒——原本有点娘炮、弱气，却是个战斗时够狠的**出色野兽猎人**，也负责去 [凯撒](../尼古拉斯·凯撒.md) 的后勤补给处**核对账目**（她清楚凯撒的账「看着没问题、实则小毛病不断」，却怎么都查不出来，这感觉「比变成异性更糟」）。

## 每天早起先确认性别

某次委托情报有误，她本以为是去打心灵野兽，却撞上一只从噩梦世界误入沙之世界的**异种母体生物**（生命 / 繁殖 / 母体 / 蜕变特性、能产卵孵大量下位个体）。她在完全不了解敌人的情况下以重伤为代价将母体灭杀，战后从那母体的粘液 + 血液混合物里爬出、没当回事——直到回教会养伤半个月、发现自己的「胸肌」越发发达，才惊觉**基因层面出了问题**。从此她每天早起的第一件事，是**确认自己今天的性别**、再决定穿哪身衣物（那天恰是「艾罗女士」）。[苏晓](../苏晓.md) 昨晚在医疗室无偿治过她——这也是苏晓借「愈者」权限广施恩惠、让教会高层更难动他的一环。

## 关系

- [苏晓](../苏晓.md)（库库林药师 / 白夜）：昨晚在医疗室无偿治过她的药剂师。
- [尼古拉斯·凯撒](../尼古拉斯·凯撒.md)：她核账的对象——账目查不出问题却明知有鬼。
