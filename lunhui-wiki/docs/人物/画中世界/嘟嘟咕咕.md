# 嘟嘟咕咕

寄宿在一间**大石屋**内的友善小家伙——只闻其声（清澈稚气的「嘟嘟」「咕咕」「亲亲亲」）、只见几只婴儿手大小的莹白骨手从墙内石盘探出。它是**画之世界的友善单位**，与卖水老太婆、盲眼老人、蘑菇贤者一类相似——[苏晓](../苏晓.md) 判断它近期刚与虚空之树签了契约（这次交易直接给灵魂钱币、上次还用「粗制灵魂凝结物」当货币）。它并不属于噩梦世界，是随那间被圣地·奇利亚德太阳灼出熔痕的大石屋一同「穿梭」而来（苏晓认得——这石屋曾在圣地·奇利亚德庇护过他免遭太阳灼烧）。

## 交易 · 什么都收

- **收废品式交易**：它几乎什么都要（苏晓塞了一堆待淘汰物:命运金锭、洗发水、木雕、名表、纸抽……），收下后回赠灵魂钱币 / 灵魂结晶 / 灵魂晶核 / 霸主精魄等；遇到它喜欢的暗特性 / 特殊物品，回赠会格外丰厚（[扭变的深渊能量凝结体·残片](../../力量体系/装备道具.md) 换得 **6 颗灵魂晶核**、命运金锭换一大堆灵魂钱币）。它也会「二选一」（2 块画卷残片 或 一颗拳头大的霸主精魄）任性给。
- **增益祝福**：交易够慷慨后，它抓住苏晓的手加持**祝福**（幸运临时 +6·力/敏/体 +5 + 生命上限 +15%·各持续 12 小时），且允许下次再来 / 别处偶遇时重新获得。
- **怂萌 · 有交易上限**：它怂到极点——苏晓 −9 魅力属性把它吓得直说「你坏、坏坏坏」、几次拒绝交易，苏晓做出拔刀姿势它立刻「亲亲亲」认怂同意。交易有上限，濒临时它用手势提醒，强行超限会让它伤心、并给苏晓带来厄运（幸运永久降低）；上限似乎每隔一段时间 / 每个世界刷新。它还好心提醒苏晓「[黑暗物质](../../力量体系/装备道具.md) 是坏东西、别碰」。

## 关系

- [苏晓](../苏晓.md)（库库林·白夜）：与它交易两次的老主顾——魅力太负把它吓怕、靠拔刀「逼」成交，却也换得丰厚回赠与增益祝福。
- 圣地·奇利亚德：它曾寄宿之地（随大石屋一同「穿梭」到噩梦之地）。
