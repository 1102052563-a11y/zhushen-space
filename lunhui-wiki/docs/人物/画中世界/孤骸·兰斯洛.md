# 孤骸·兰斯洛

[画中世界](../../世界/任务世界/画中世界.md)·沙之世界 [烈阳君主](烈阳君主.md) 的心腹之一——身形消瘦、气息很冷、沉默寡言，双臂能弹出厚重而锋利的臂剑。正因他沉默孤僻、又是烈阳君主的贴身心腹，[凯撒](../尼古拉斯·凯撒.md) 才盯上他、向 [苏晓](../苏晓.md) 推荐把他当作弑君计划的「关键切入点」。

## 沦为计划的钥匙

王宫盛宴前一个多小时，他接烈阳君主密令匆匆赶回城北，路上被凯撒引诱、被 [巴哈](../剧毒魔鹰·巴哈.md) 用炼金阵图拖进异空间。面对赤膊持刀、血气凶暴的苏晓，他自知绝望、开场就自爆生命力换取一搏，仍被苏晓一刀斩碎手臂、再一刀斩落头颅（他临死只求留个名「我是孤骸·兰斯洛」，不愿默默死去）。他值 **2.71% 世界之源**（小 boss 级、却没掉宝箱）。苏晓用采血针取其 300 毫升血、凭 [先古面具](../../力量体系/装备道具.md) 伪装成他的容貌 / 气息，混进王宫盛宴——晚宴上正是这张「孤骸·兰斯洛」的脸，一手从背后贯穿了烈阳君主的心口。

## 关系

- [烈阳君主](烈阳君主.md)（奥斯·瓦伦丁）：他忠心追随的主君——却被顶替他容貌的苏晓当众「背刺」（烈阳君主至死错骂「兰斯洛，你这吃里扒外的狗东西」）。
- [苏晓](../苏晓.md)（白夜）：斩杀他、夺其容貌行刺的猎杀者。
- [尼古拉斯·凯撒](../尼古拉斯·凯撒.md)：把他选定为弑君切入点、诱他上钩的奸商。
