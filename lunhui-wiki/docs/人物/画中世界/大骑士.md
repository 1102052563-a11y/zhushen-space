# 大骑士

[画中世界](../../世界/任务世界/画中世界.md) 里来自另一个里画世界「**古城**」的骑士——身着（被阿波罗炸得）破烂铠甲、身高约三米、背披残破红披风，手中大剑却始终雪亮。他是画卷世界的**高战力单位**、剑术宗师 Lv.60 打底（剑势刚猛老练、沉稳），年龄至少五十以上。

## 骑士的荣誉与背负

轮回乐园提示点明他的品格:不轻易信人，可一旦联合便绝不猜忌、不背刺。他为了**故乡古城的人民**而来夺 [画卷残片](../../力量体系/装备道具.md)——「人们在等待骑士，但骑士不可空手而归，或牺牲，或带回希望」。也正因此，他从开始夺画卷残片起，就自认已失去骑士之荣:噩梦之王喊「我和大骑士的决斗你们两人偷袭，卑鄙」想离间他，他却一剑斩向噩梦之王脖颈——故乡的人民在等他带画卷残片回去，比虚名的骑士决斗更重。

- **可联合的强援**：[苏晓](../苏晓.md) 可选择联合他共战噩梦之王——胜后须分他三分之一画卷残片；若战败，他会舍身掩护苏晓撤退、并打开「画之门扉」（大概率通里画世界·古城、小概率通主画）。他正是噩梦之王所忌惮、也要夺画卷残片的那个人（苏晓、伍德、罪亚斯凭噩梦之王的行为模式反推出了他的存在）。
- **厄梦镇决斗**：苏晓用 [烈阳之怒·阿波罗](../../力量体系/装备道具.md) 夷平厄梦镇后，他与噩梦之王在废墟中激战（互有攻防·硬抗了阿波罗的爆炸），后 [沃波·伍德](沃波·伍德.md) + [罪亚斯](罪亚斯.md) 加入围攻噩梦之王，苏晓则架起 [J·恶魔](../../力量体系/装备道具.md) 狙击炮远程助攻。

## 🆕 守古城 268 年 · 被偷袭撤退（44卷25章）

决战中 [噩梦之王](../../世界/生物万族/index.md) 想用「你我决斗、他们偷袭卑鄙」离间他，他却一剑斩向噩梦之王脖颈——从他动手夺画卷残片起，就自认已失骑士之荣，故乡的人民在等他带残片回去。他不肯说古城的存在（怕引来这些强者、古城居民太脆弱），只推说「为了更强」。[罪亚斯](罪亚斯.md) 趁其分心、用埋在地下的触手从腿甲裂缝刺入偷袭，他大口吐出满是触手的血。他一度想「干脆死在这就能解脱、休息」（**已守古城 268 年**），但一张张古城居民童真 / 绝望 / 期待的面孔在脑中闪过，他暴喝一声、拔剑插地、带伤撤退——正常交战他本有四到六成把握格杀罪亚斯，可先挨阿波罗、又遭奇袭，已无力再战。

## 关系

- [苏晓](../苏晓.md)（库库林·白夜）：可联合共战噩梦之王的强援（战利品分他三分之一 / 战败为其打开画之门扉）。
- [罪亚斯](罪亚斯.md)：趁他分心用触手偷袭、逼他带伤撤退的陨灭星疯信徒。
- 噩梦之王：与他争夺画卷残片、他不惜舍骑士之荣也要斩杀的敌人。
- [沃波·伍德](沃波·伍德.md) / [罪亚斯](罪亚斯.md)：与他一同围攻噩梦之王的（各怀鬼胎的）临时同阵者。
