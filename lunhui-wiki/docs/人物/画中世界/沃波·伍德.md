# 沃波·伍德

[画中世界](../../世界/任务世界/画中世界.md) 中**魔鬼族**的参战者——身穿西装、头是一颗镶满米粒黑宝石的骷髅头、眼洞燃深邃瞳焰，属魔鬼族里战力极强的战力代表分支。他是**虚空臭名昭著的骗术师**，曾凭一份契约骗走羽族三处大型高震钢矿脉。

## 恭维即武器

伍德最可怕的不是骗、而是**恭维**——「把虚假恭维成真实」，能把死的东西说到「活过来」（弄成亡灵生物）。他一见 [苏晓](../苏晓.md) 便先奉上一段让人心生好感的恭维（避免刚见面就被一斧劈了），得知苏晓是轮回乐园的便利落收起契约（都是玩契约的行家、互相试探多余）。虚空里能在嘴皮子上跟他碰一下的、也就凯撒。对付他最有效的方式就是「打嘴」。

## 噩梦之地 · 背叛者 · 三方联盟

他在噩梦之地当上了「**背叛者**」（无固定阵营·条件满足可转阵营·所在阵营胜则胜）——正是他**忽悠上一任猎命人（黑暗住民）签了契约、骗走猎命人的套装与武器**（苏晓的猎命人身份即由此而来）。他主动找苏晓合作:交出 1 号锁盘位置、点破奥术永恒星「炎启 + 洛希只是试探、真正杀手锏是乌鸦女」等情报当人情，与苏晓、[罪亚斯](罪亚斯.md) 结成**三方临时联盟**（先清场堵初生广场 → 灭 [噩梦之王](../../世界/生物万族/index.md) → 最后三选一分胜负）。他自认这场生存游戏只是热身，负责用恭维话术拖住 / 坑掉 [天羽](天羽.md) 等生存者。

## 关系

- [苏晓](../苏晓.md)（库库林·白夜 / 猎命人）：他主动结盟的对象——苏晓的猎命人套装正是他从上任猎命人手里骗来间接给的。
- [罪亚斯](罪亚斯.md)：三方联盟另一员（陨灭星疯信徒·被伍德当面说「心怀鬼胎」险闹翻）。
- [天羽](天羽.md)：被他恭维话术拖住、坑进包围的羽族渣男。
