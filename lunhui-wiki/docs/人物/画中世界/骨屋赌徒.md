# 骨屋赌徒

[画中世界](../../世界/任务世界/画中世界.md) 噩梦之地游乐场深处、由骸骨堆砌的「骨屋」中那具坐镇赌桌的**骷髅荷官**——洗牌码牌娴熟无比，是一个**不可力敌、极古老的纯粹赌徒**（[沃波·伍德](沃波·伍德.md)、[罪亚斯](罪亚斯.md) 都感其不好惹，与噩梦之王「不在一个次元」）。它手握三块 [画卷残片](../../力量体系/装备道具/index.md)，只与人**小赌**。

## 以命弈命 · 双面牌

它的赌局「**以命弈命**」——以「年」为筹码押上寿命 / 命魂（它的命魂存量高达 **1,695,234 年**）。它主掌的赌局公平，只是规则特殊:所用**双面牌**（一面普通牌面、一面隐藏牌面，可翻面改写胜负），为的是加大博弈的紧张感。作为纯粹的赌徒，它为赌失去了太多——至亲、同族、身躯、三分之二的灵魂，早已麻木于「输」。它曾被灭法者女强盗 **格林·吉莉安** 抢走赌注赶走过，也认得灭法者身份。

## 与伍德的深渊之罐赌局

魔鬼族 [沃波·伍德](沃波·伍德.md) 拿 **[深渊之罐](../../力量体系/装备道具/index.md)** 上桌——伍德的真实目的是**输掉、甩掉这祸害魔鬼族几百年的东西**。巧的是骷髅赌徒抽屉里正有一个与之完全相同的**深渊罐盖**（它也被这半个「爹」祸害不轻、盼着摆脱）。两个老赌徒用双面牌反复翻面「争输」:伍德最终把恢复完整的深渊之罐「赠」给了骷髅——「你赢了，但你也输了」。此后它又与伍德、罪亚斯赌了几轮，赢走三块画卷残片（罪亚斯输到只剩四角裤）。[苏晓](../苏晓.md) 判断赌局全凭运气、无法作弊，「不参与就是赢」，全程旁观。

## 关系

- [沃波·伍德](沃波·伍德.md)：与它「争输」、把深渊之罐甩给它的魔鬼族骗术师。
- [苏晓](../苏晓.md)（库库林·白夜）：拒绝与它对赌、全程旁观的灭法者（它感慨「上一个拒赌的灭法者是格林·吉莉安」）。
