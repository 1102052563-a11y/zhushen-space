# 神王·奥斯·托拜厄

[画中世界](../../世界/任务世界/画中世界.md) 一切的源头——**旧世界的世界之子**、**奥斯一族（王裔）的始祖**、王朝的第一任君主，人称**神王**。由 [迹王·卢修曼](5号老人.md) 在海底宝库向 [苏晓](../苏晓.md) 揭出这段远古真相。

## 屠万神 · 画出新世界

**旧世界**曾是九阶中梯队的原生世界，繁荣于神灵、也亡于神灵——那里神灵多到纪年称「万神纪元」，其中最强的是**太阳、地脉、海洋、天空、心灵**五大神祗（[太阳鸟·泰哈卡克](../../世界/生物万族/index.md) 是太阳神祗转生、[海神](../../世界/生物万族/index.md) 是海洋神祗）。众神为争信仰之力打破誓约、万神联合与五大神祗开战，把世界拖向灭亡。世界之子奥斯·托拜厄挺身而出，联合尚存理智的人族与海族、极族等组成同盟，**把旧世界的神灵屠了个干净（含五大神祗）**，成了最后的胜利者。他收集**万神的源血**，在旧世界崩灭前，拿着打碎的**旧世界·世界之核**制成的画布 + 万神血制成的墨，找到另一位世界之子 [罗莎·尼耶](罗莎·尼耶.md)，请她画出了**画之世界**（八阶·比旧世界低一梯阶）。

## 自我牺牲的开辟者

大迁徙时他建立王朝、当上第一任君主，却**没有随族人迁入画之世界**——他太强，去了会引发问题，且他是「因世界危难而生、也要因世界崩灭而死」的世界之子。他只把弟弟 [奥斯·古因（光焰领主）](../../世界/生物万族/index.md) 送了过去（残忍地斩下弟弟下半身、又将战马头颈与之融合，把过强的弟弟永久削弱到能进画之世界的上限）。他留下的王朝与家族在画之世界称霸，直到兽灾爆发——**兽灾的根源正是画卷的墨迹**:墨是万神源血所化，其中将亡的**心灵神祗**在消亡前让墨迹蔓延出「疯狂」、不断侵蚀画卷，是来自神灵时代的报复。

## 关系

- [罗莎·尼耶](罗莎·尼耶.md)：他请来画出画之世界的绘画者（初代绘画者）。
- [光焰领主·奥斯·古因](../../世界/生物万族/index.md)：他最信任的弟弟——被他削弱后送进画之世界。
- 奥斯一族王裔（[烈阳君主](烈阳君主.md) 等）：他的子嗣血脉——因末代的畸形与病态，酿成兽灾横行的今天。
