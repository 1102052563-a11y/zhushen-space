# 乌鸦女

[画中世界](../../世界/任务世界/画中世界.md) 里**奥术永恒星**藏着的**杀手锏**——被 [魔鬼族·伍德](沃波·伍德.md) 点破:炎启·索耶格与 [洛希](洛希.md) 只是试探，「乌鸦女才是那边的杀手锏」。她此刻还在虚空莫乌斗技场的观战席，坐等奥术永恒星第三个入场名额（前两名 [炎启·索耶格](炎启·索耶格.md) / [洛希](洛希.md) 已被苏晓斩杀淘汰）。

## 被养成的「野兽」

她最摄人的是那双眼:眼底漆黑、瞳外圈白、正中一道深邃到摄人心魄的黑色中心瞳。她被 [法师贤者·瑟菲莉娅](../瑟菲莉娅.md) 等生命学派长辈**养成了一头「野兽」/ 战斗兵器**——对瑟菲莉娅毫无敬意、句句带刺:「你们几个把我培养成野兽、还想让我懂礼貌？谁抓着链子我就听谁的、帮他咬断别人喉咙，这不就是你们想看到的吗？我是野狗、是牲畜、是雌性野兽。」她态度散漫（把腿翘到前排施法者少年的椅背上），却对苏晓极为忌惮——直言「不怎么想和那血兽拼命，那是杀出来的强敌、特别难对付」，甚至盼着洛希也死掉、好让自己没机会出场。

## 关系

- [法师贤者·瑟菲莉娅](../瑟菲莉娅.md)：把她养成「野兽」的生命学派长辈——她对其毫无敬意、以「抓链子的人」相称。
- [苏晓](../苏晓.md)（库库林·白夜）：她极为忌惮、不愿拼命的「血兽」强敌（奥术永恒星派她入场就是冲他而来·伏笔）。
- [炎启·索耶格](炎启·索耶格.md) / [洛希](洛希.md)：先她入场、已被苏晓斩杀的奥术永恒星前两名参战者。

## 🆕 入场 · 被恶阵营三人组算计到血崩（44卷115–125章）

奥术永恒星的最后名额、终于入场——她自称「奥术永恒星派来的疯狗」，战力强、不择手段、痴迷灵魂结晶 / 晶核、脑子因受过重击「不太好使但很能打」。[水哥](../联盟星/水哥.md) 交割委托后、她又出 12 万灵魂钱币（[苏晓](../苏晓.md) 在游侠公会的赏金）雇水哥杀白夜，可她进世界后连苏晓的面都没见着、还被安上「刺杀海神」的黑锅遭全城通缉。争夺战尾声她用**虚空之树契约预支排行奖励**买下 [伍德](沃波·伍德.md) / [罪亚斯](罪亚斯.md) 的残片、几度冲上首位（34 块），却被苏晓一次次留希望再碾压（最终苏晓 63 块）——更被三人当面「分赃」她那份还没到手的奖励（她实得 0 克黑枫）。这份被算计到血崩却转头就忘大半的豁达，注定她难缠。她还险些把 [法师贤者·瑟菲莉娅](../瑟菲莉娅.md) 被女灭法者玩弄抛弃的黑历史当众直播抖出去。

## 🆕 树生世界 · 霸主级单位 · 与仙姬联手追杀苏晓（46卷1–7章）

她作为**奥术永恒星（虚空）**的强者来到树生世界，被虚空之树公证为**第三名「霸主级单位」**。画之世界她曾被 [苏晓](../苏晓.md)、[罪亚斯](罪亚斯.md)、[伍德](沃波·伍德.md) 联手算计、第二名奖励被三人截走，回虚空后又因深渊之罐险些杀了伍德、被法师贤者·瑟菲莉娅等人拦下逼她道歉、还被凛风王关半个月禁闭。可这次重逢她一反常态——对苏晓眨眼、带一丝色气，「会咬人的狗不叫」，让人愈发警惕。她是**野兽与不拘小节**的性格（吃相狼吞虎咽、暴饮暴食小腹仍平坦、口无遮拦），奥术永恒星把她培养成「消除一切对奥术永恒星不利因素的疯狗」——只要目的达成、什么手段都行。

- **与仙姬一拍即合**：古都时她本想动手，见苏晓 + 伍德 + [奥娜](../树生世界/奥娜.md) 组队就没冒进，一路跟到寒冷墓地。她与 [仙姬](../联盟星/仙姬.md)（能感测断魂影之石）、[冥狼](../魔海/冥狼.md) 联手——她掏出一根**透红的初代遗骨**（灭法者死前用本源能量凝成、若被苏晓得到可提升核心能力上限），凭它能追踪苏晓本人。她还笑谈灭法时代「奥术永恒星被灭法者打得娇喘连连」，性格若非阵营敌对并不惹人厌。
- **白色沼泽惨败**：她随仙姬队北上追杀，却在苏晓布下的炼金剧毒沼泽里连中五种、乃至十几种慢毒（一度被「猛毒·狂犬」逼得攻击欲near失控），80 违规者只 9 人活着溜回。撤离时她五指抓刺进仙姬侧腹（未在空间卷轴留血印、只能抓着仙姬才被带走）。

## 🐦 乌鸦队之首 · 又一次「似曾相识」的被算计（48卷34、36章）

⚠ 进死寂城后乌鸦女成了「**乌鸦队**」之首（[月光侍女](../幽暗大陆/月光侍女.md)、被公爵夺舍的 [克兰克](../幽暗大陆/克兰克.md) 以她为首、只因她如今很不好惹）——她的核心目的仍是**摆脱身上的死灵之书**，否则本世界时限一到就会被虚空之树连人带书传送回奥术永恒星。她手里有一块教会石板、还在与公爵队谈合作。⚠ 狼冢石碑前三队合拼四块石板时，苏晓召来的**死灵之书**卷走全部石板、假意没入她体内——所有人都盯着她，她自己却也一脸茫然「那四块石板哪去了」，一种「**似曾相识、好像又被算计了**」的感觉难以抑制地涌上心头（正是白树林被苏晓塞死灵之书那次的翻版）。她被沃姆队当成夺板真凶、杀意暴涨地追杀而去。
