# 怨之孤儿·修斯

[Fate/Zero 世界](../世界/任务世界/fate-zero.md) 柳洞寺「永生池」之主、**领主级 [怨生物](../世界/生物万族/index.md)**（生命同化度 97%·只差一点就「进化为完全体、成为神」）。她原是 [言峰璃正](言峰璃正.md) 口中那名「早已死亡」的**女性代行者**（[黎明战吼](../力量体系/装备道具/index.md) 的原主·曾用那把枪救过无数人）——实则没死、堕落成了泡在永生池里的怪物（背后另有故事，苏晓无心深究）。

## 形态与能力

- **诱惑态**：暗绿长发的绝美裸女、媚惑勾引（被青钢影 / [心灵之刃](../力量体系/技能/index.md) 免疫魅惑后现原形）。
- **战斗态**：3 米以上的白骨怪（骨内仍有内脏 / 眼球 / 大脑·像被活剥血肉），武器是裹着鲜血肌肉的「**肌肉弯刀**」（无刃钝器·一击势大力沉、还能**吞噬命中者血肉**·砸盾会震伤持盾者）。属性力 70 / 敏 69 / 体 ?? / 智 65；一身**最低也是 Lv.20 起步的技能**：**灵魂感染**（光环·侵蚀 30 米内一切生物灵魂·每分钟造成最大生命 5% 真伤·站着不动都能耗死人）、高强度骨骼（防 +25）、生命同化、生命阶梯、永生者等（多为 ?? 未知）。

## 死亡（十一卷28–30章）

苏晓炸碎灵魂灯盏后，10 层结界把主供殿封成牢笼、逼出修斯。修斯先招募苏晓当奴仆（被拒），遂死战——其灵魂感染光环逼得苏晓只能**撤防、靠感知躲钝器、抓空隙反击**（生命被光环狂掉）。最终苏晓**用「牛头领主之怒」（力 +15）超越修斯力量 + 全法力灌青钢影**，一刀砍断肌肉弯刀、斩穿其肩到腹，骑身捶上百拳爆其一眼，再把 [黎明战吼](../力量体系/装备道具/index.md)（灌了圣水·她生前的配枪）枪管插进眼洞、一枪把圣水炸进她大脑——黑暗系的她被神圣的圣水净化、化回女人模样而死。**结算：[苏晓](苏晓.md) 得世界之源 +13%（→42.8%）+ 7000 币 + 灵魂结晶〔中〕×2 + 忏悔之书（任务道具）+ 金色宝箱**。苏晓自评修斯实力本在他之上、只因「刚进化、不熟悉自己的能力 / 重武器」才被翻盘。

## 死后被召唤为「召唤物」（十二卷63章）

苏晓斩她时掉的金色道具「[怨灵灰烬](../力量体系/消耗品材料/index.md)」、十二卷在维斯星球派上用场——一次性**召出怨之孤儿·修斯当临时召唤物**（强度 / 存在时间随苏晓智力·智力 79 判定下存在约 18 分钟、领主级·力 70 / 敏 73 / 体 76·七大技能·**灵魂感染光环每分钟削敌 5% 最大生命真伤**仍在）。曾把苏晓揍个半死的领主级怨生物，如今成了他随手召来的「大礼」——一现身就吓得 [兰爵](兰爵.md) 转身逃命。决战时苏晓把它的控制权交给 [布布汪](布布汪.md)（自己专心近战），它成了苏晓 [界断线](../力量体系/装备道具/index.md) 缠倒兰爵的「工具」（绕兰爵缓步走令线缠其腰）；最终头颅被兰爵的甲虫自爆炸碎、召唤随之提前消散。

## 关系
- **[苏晓](苏晓.md)**：杀她的伪英灵（用她生前的枪 + 圣水了结·后以「怨灵灰烬」把她当召唤物召回）。
- **[言峰璃正](言峰璃正.md)**：**昔日恋人**（老神父·得知她死讯哽咽落泪·只盼她「没受太多痛苦」）。
