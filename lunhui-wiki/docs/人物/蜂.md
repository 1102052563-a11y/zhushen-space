# 蜂

[彼岸花冒险团](../势力/index.md)（全女性冒险团）的**狙击手**。三卷围杀 [苏晓](苏晓.md) 时其子弹被苏晓轻易躲过（苏晓凭心眼免控、闭眼应对）。八卷11–12章随重整后的六人精英团参加 [二阶生存试炼·吞噬岛](../世界/任务世界/T12K星球-生存试炼.md)，谋向苏晓复仇。

## 殁（八卷28章）
她师傅 [黑白](黑白.md) 即死于苏晓之手、仇不可和解。吞噬岛上苏晓为清理走漏其身份的 [腹黑女](腹黑女.md) 撞上彼岸花六女、开战——作为狙击手的蜂被苏晓「**剃**」瞬移近身**斩断双臂与狙击枪**，糖果的护盾治疗被斩龙闪「影舞断 / 血流如注」克制（治疗 −30%）而无力回天，终被刺穿心脏。苏晓还把她的尸体抛向冲来的队友、连同炼金炸弹引爆。
