# 冯·霍恩海姆

主角 [爱德华·艾尔利克](爱德华·艾尔利克.md) 兄弟的**父亲**。原是古王国「赛尔克塞斯」的奴隶「**23 号**」；[瓶中的小人](瓶中的小人.md) 用他的血合成、又给他取名并重炼身体，使他**永生且容颜永驻**（只要瓶中小人不死、他就一直活着）。

- **历程**：永生后流浪 380 年、与一名女子成家生下爱德华与阿尔冯斯，后又厌倦生活、抛妻弃子再次流浪——故爱德华憎恨他、改用母姓「艾尔利克」。
- 也是瓶中小人「国土炼成阵」的**五人柱**之一（与两兄弟、马斯坦、伊兹米并列）。
