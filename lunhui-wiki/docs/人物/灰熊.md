# 灰熊（口腔分析独行侠）

[维斯星球·多亚古大陆](../世界/任务世界/维斯星球-多亚古大陆.md) 世界争夺战中轮回乐园方的**独行侠**——满口洁白平齿、性格豪爽的狠人。他的能力是「**口腔分析成分**」（用嘴品尝来分析物质成分·极罕见），为此什么稀奇古怪的东西都尝过（包括用口腔分析尿液成分·把奶糖问的「那东西什么味」答成「有点咸」）。一脚跺地能溅起泥土，实力不弱，是己方一员大将，[炎辰](炎辰.md) 评价他「实力不错、可惜太鲁莽」。

- **被吓跑的「正常人」（十二卷75章）**：水晶球灭掉 403 人后，他特意来向 [炎辰](炎辰.md) 辞行——「我是个正常人，不想和一群疯子并肩作战」。炎辰笑着放行「想走就走」，可他穿过七层走廊时，两侧站满神色疯狂 / 冷漠的契约者凝视他，吓得后背冷汗浸透（只要 [苏晓](苏晓.md) 或炎辰一声令下，这群疯子就会把他碎尸万段）。他直觉苏晓与炎辰「一定会做骇人听闻的事」，遂离开黄昏宫殿、与 [艾达](艾达.md)、虫妹组临时小队打游击（目的与大部队一致·只是不愿与疯子同处）。中意奶糖那一型。

## 关系
- **[艾达](艾达.md) / 虫妹**：临时游击小队队友。
- **[炎辰](炎辰.md)**：己方主帅（评他鲁莽·放他离开却暗藏杀机）。
- **[苏晓](苏晓.md)**：让他直觉「血腥味比疯子更重」的强者。
