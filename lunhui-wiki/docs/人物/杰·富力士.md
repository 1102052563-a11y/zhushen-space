# 杰·富力士

全职猎人世界的**剧情主角**——一名已掌握念能力的小屁孩、[奇犽·揍敌客](奇犽·揍敌客.md) 的耿直好友。[苏晓](苏晓.md) 视他与奇犽为猎人世界的「两个命运之子（伪）」（两小屁孩合体才是真正的命运之子）。他**天生感知极敏锐**、像在森林里长大的幼兽。

## 跟来帮忙（十六卷14章）

[席巴](席巴·揍敌客.md) 把奇犽揪回枯枯戮山时，他主动提出跟来帮忙（言明「帮不上忙也绝不添乱」·席巴对他态度还算和善）。可一见 [马哈·揍敌客](马哈·揍敌客.md)（黑念·死亡气息）与 [苏晓](苏晓.md)（血红·锋利凶暴）——他凭天生敏锐当即判定「自己很可能帮不上忙」、神色低落地被安排暂住古堡（由 [柯特](柯特·揍敌客.md) 招待·年龄相仿）。耿直如他：见奇犽家的古堡只说「你家好大……但也好破」。

## 关系
- **[奇犽·揍敌客](奇犽·揍敌客.md)**：好友 / 搭档（互相信任·「命运之子」组合）。
- **[苏晓](苏晓.md)**：气息令他本能畏惧的强者（被婉拒帮忙·留在古堡）。
