# 格利德（贪婪）

[瓶中的小人](瓶中的小人.md) 所造七名人造人之一，代表「**贪婪**」——年轻男·小圆墨镜·笑面；**已背叛瓶中小人一世纪**（贪婪相性使其不效忠任何人）。

- **能力「最强之盾」**：把体内碳分子提取到皮肤形成炭层、硬度超乎想象，**恢复力是人造人中最强**；统领一队 [合成兽](../世界/生物万族/index.md)（罗亚〔牛〕/ 多鲁契德〔狗〕/ 马黛儿〔蛇〕）。
- **历程**：为求永生（人造人体内贤者之石约 200 年会耗尽、须父亲大人重炼），绑了主角弟弟 [阿尔冯斯](阿尔冯斯·艾尔利克.md) 逼问「灵魂提取固定术」。
- **殁**：苏晓硬抗它几十颗炸弹（连死 16 次）、再用**徒手近战**（力 38·肘击 / 跺断肋骨 / 膝砸头入地）打瘫、塞炸弹爆杀（剧情关键·世界之源 +5.1% → 17.2%·断罪 3/7·**未掉宝箱**·临死想爆料「主谋是…」被打断·七卷13章）。
