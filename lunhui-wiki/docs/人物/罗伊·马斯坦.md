# 罗伊·马斯坦

**火焰炼金术师**·上校（与化名国家炼金术师的苏晓平级）。因被 [瓶中的小人](瓶中的小人.md) 看中天赋而被留任中央市，实则**暗中谋反**；是瓶中小人「人柱」候选之一。

- **能力 / 弱点**：戴印有炼金阵的白手套、用火焰战斗（堪称人造人克星）；**雨天则火星都擦不出、形同废物**。
- **与苏晓**：七卷3章目睹苏晓的「爆炎地狱」杀 [艾扎克](艾扎克·迈克度加尔.md)。七卷17章苏晓**「假杀真救」其副官 [莉莎·霍克艾](莉莎·霍克艾.md)**、向他揭露国家秘密——他遂成为苏晓在本世界的**第一个盟友**（反瓶中小人）。其好友 [马斯·休斯](马斯·休斯.md) 死于 [恩维](恩维.md) 之手、人柱身份 + 这桩血仇坚定了他谋反的决心；与苏晓约定动手清理中央的高级将领（靠莉莎「鹰眼」狙杀）。

- **最终决战**：作为五名「人柱」之一被瓶中小人**强制人体炼成**、通行费是**视力**——七卷44章起**双目失明**。也正因与苏晓合作过、最了解其作风，他是唯一看穿「白夜把瓶子塞进瓶中小人体内必有图谋」、急呼众人阻止苏晓的人（已晚——苏晓反手挟持其副官 [莉莎](莉莎·霍克艾.md) 当肉盾从容脱身）。
