# 圣扎迦（始祖首领 · 灵魂亡者 BOSS）

[黑渊浅层](../世界/任务世界/黑渊浅层-赔罪者之殇.md) 赎罪者高地中心区域 **[始祖](../势力/index.md) 势力的首领**——肩披兽皮、身材魁梧如壮年雄狮的「灵魂亡者」。守护着一株近千年的 **[灵魂藤蔓](../世界/生物万族/index.md)**（体内更植入活体藤蔓·与之共生），麾下是大批由灵魂藤蔓同化的「**同化者**」（植物形态·惧火 / 辐射）。言谈有礼、却被 [苏晓](苏晓.md) 连日用 [阿波罗](../力量体系/消耗品材料.md) 轰祖坟 179 座 / 同化者 27 万、炸到「被炸习惯了」、心态血崩——还被苏晓伪装阿萨卡杀手反向「接委托」耍了一道。

## 能力（二十三卷25章 · BOSS 模板主坦）

`生命值 30000+ ｜ 法力 2800 ｜ 力 102 / 敏 99 / **体 110**（真实·破极值）/ 智 73 / 魅 28`——六种偏生存能力的怪物坦：

- **灵魂死者**：生命 +12000、防御 +58、免疫 27% 伤害（受灵魂伤害则翻倍·这是死穴）。
- **集体意识**：体内器官各有独立意识（心脏等可自行闪避 / 参战）。
- **软组织装甲 / 原始呼吸**：皮下软组织进化为植物、防御 +67、生命 +5800；生命 <40% 进入植物形态（总防御 +50%·恢复 +300%·越打越硬）。
- **鲜血撕咬（血之口）**：外放血液成尖牙之口撕咬·吸 80% 伤害回血。
- **植物君主**：可进行第三次进化 / 变异（待激活·未及发动即死）。

## 决战殒命（二十三卷24–27章）

[苏晓](苏晓.md) + [莎](莎.md) + [莉莉斯](莉莉斯.md) 三人围攻——D·暗杀五枪爆头只削 4% 血（防御变态），莎抱摔拖坦、莉莉斯贴身导恶魔之焰「内燃」削其 24% 生命、苏晓**一刀穿三人**（贯莉莉斯肩 + 圣扎迦喉 + 莎胸）。最终苏晓把他砍到生命 ≤25%、用 [斩龙闪](../力量体系/装备道具.md) 的**魔刃即死斩杀**（刃之魔灵侵入意识空间斩魂·灵魂亡者灵魂强度不及刃之魔灵）将其清理——这是苏晓 400% 难度晋升任务真正的拦路 BOSS。

## 关系

- **[莎](莎.md) / [莉莉斯](莉莉斯.md)**：围攻他的拖坦 + 内燃手。
