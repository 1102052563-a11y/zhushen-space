# 背叛者·席曼·阿奇德（深渊之影）

[苏晓](../苏晓.md) 猎杀名单上的**第六名、也是最强的叛徒「背叛者」**——本名**席曼·阿奇德**，人称**深渊之影**。一身灰白宽松衣衫、满头灰白长发、戴圆框眼镜，虽老迈却格外有男性魅力，一双锐利的眼睛，一眼便知是**刀术宗师**。他曾是灭法阵营的一员（比欺骗者所属的时代更早入阵），却因**堕入深渊**、与灭法信念决裂而被列为叛徒（灭法阵营不禁一定程度使用深渊力量，但完全投身其中即视为叛徒）。他在北境帝国建立前就是这世界无人敢惹的强者，隐居**巴尔大森林**、以「石刻师」身份示人，还教出了**北境大将军**等**五名弟子**。

## 🎣 布下整局 · 引苏晓来送唤醒之碑

⚠ 苏晓一直想不通:背叛者为何用虚空之树公证、花 **1592 盎司**从圣光乐园买下**唤醒之碑**（这引来了追踪唤醒之碑的苏晓、触发猎杀名单），买来后又对苏晓的连环屠戮无动于衷。真相由 [席尔维斯](掌控者·席尔维斯.md) 的灵魂记忆揭开:⚠ 背叛者命不久矣，唯一的软肋是**五名如子的弟子**——他们迟早会向往虚空万界，而一旦去了那边，八成会撞上「轮回乐园的那名灭法」，那时弟子们「深渊倾向的灭法特质能力」会被苏晓一眼认出，凶多吉少。于是背叛者布下一局:⚠ **主动买来唤醒之碑引苏晓入世界**，让苏晓一路清掉欺骗者、噩梦之王、黑玫瑰、沙之王等本世界的野心家（等于替他弟子扫清了未来的强敌），再以一场「灭法前辈观察 + 决战 + 传承唤醒之碑」的体面了结——让苏晓收下他备好的厚礼、知晓五弟子是谁后，今后即便不助、也不至加害。他直言告诉弟子「我要去和灭法打最后一场、给此生做个终结」，好让他们死后不来寻仇。席尔维斯本是这局的「引路人」，却因融合白金主教、憎恨秀成深渊大主教、又自作聪明贸然与苏晓死磕，把自己秀死了。

## ⚔️ 灭法之影 vs 深渊之影 · 死战（49卷51–52章）

⚠ 背叛者在列车站主动露面表态，又一刀灭了想坑北境大将军的两名北境高层、一人一刀屠了整个附虫族（因其对他说「等你死了你弟子没一个能活」），随后约苏晓在巴尔大森林深处决战。⚠ 佩戴**灭法之刃**的他悬赏一度飙到 **35000 盎司**（因本世界战力峰值抑制 + 深渊之孔已关 + 老年衰弱，实时降到 1800~3000 盎司）。他的刀术**质朴凝练、平砍即大招**（灭法堆满被动的必然），三刀之内就重伤阿姆、拍死巴哈、险些灵魂斩杀圣诗。⚠ 最骇人的是他**外放刃之魔灵**:黑雾人形、无视一切伤害（仅攻击进实体瞬间可被反击）、能与本体**互换位置**（本体对斩、魔灵远程蓄势、蓄满瞬间换位秒放）、附身其他生物、化晶体武器（弓 / 锤 / 刀随意切换）。他还有暗涌 / 黑暗巨爪 / 深渊放逐（把圣心一放逐走）等手段。⚠ 苏晓一度被斩断左臂、濒死，连保命的**漂游之饵**都被背叛者斩断空间丝线（只卷走了布布汪等四人）——**这一战苏晓败了**。可背叛者早在苏晓以灵魂王冠杀沙之王后，就查明苏晓身负**四件原罪物气息**（灵魂王冠 / 幽冥骨戒 / 死灵之书 / 深渊之罐），深知**杀死原罪物持有者会继承其因果**——若杀了苏晓，四件原罪物的因果会灭了他视若性命的弟子与整个北境。⚠ 于是他决战中始终有所保留、拒绝以深渊力量强撑，在最后一刻按住苏晓面门却无力刺下、**油尽灯枯而亡**。

## 🎁 死后的馈赠

⚠ 苏晓虽「败」，却从这场教学般的死战里大赚:见识了**刃之魔灵**的真正用法，抹名领 **2600 盎司**（时空石碎片 ×200 + 高烈度·深渊之血 ×6），完成主线最终环「深渊之影」得**起源石·无尽**，并在战斗场地地下取得梦寐以求的 **[唤醒之碑](../../力量体系/装备道具.md)**（连同三根初代指骨、两颗唤醒石、一颗灵魂晶魄，及一封信）。⚠ 背叛者以「灭法最后的独苗」相称、把唤醒之碑当礼物送他，只在信末提及五弟子——「今后哪怕不是友，最好也不要为敌」。他一生**起于灭法、终于灭法**，死在与灭法的决战中，算是潇洒的了结。

- 五名弟子（北境大将军 / 面具女 等）：他视若亲子、为之布下整局扫清未来强敌的能力传承者。
