# 沙之王（倒戈者）

[苏晓](../苏晓.md) 猎杀名单上的**第五名叛徒**「倒戈者」——统治整个沙漠之国的**暴君**，人称**沙之王**（原悬赏 800 盎司时空之力）。他身高 3 米以上、赤膊、右臂覆金色鳞甲、酒红色短发，一双乌黑到令人跪拜的眼睛，气场骇人。

## 背叛救命恩师的倒戈者

⚠ 沙之王曾是路边饿到濒死的孩童，被 [马文·华尔兹](../马文·华尔兹.md)（灭法者）一句「小兔崽子，跟老子走，管饱、有肉吃」救下、收作灭法学徒。可当灭法与施法者的巅峰之战败局已定（灭法之影太少），他**以最干脆的方式背叛灭法阵营、要站在赢的那一方**——用利剑刺穿身负重伤的马文·华尔兹后心，那错愕痛惜的目光成了他一次次的噩梦。「倒戈者」之名由此而来。他来到本世界后，以**控制淡水**的方式，把松散的沙漠部落议会（圣沙堡）逐渐变成自己独裁的君主制。

## 眼界与野心 · 目标「至强者」

⚠ 沙之王绝非无能暴君——麾下大臣（右御·卡伽、左御·佩温、亲卫队长·索瓦）死心塌地，「丰水都」兵强马壮、暗中驻军，打造出能与联盟 / 北境帝国争锋的**沙漠军团**。⚠ 而这军团的真正用途，是**挑动联盟与北境帝国开战、渔翁得利**，最终统治沙漠、联盟、凛冬之地三大地盘——他要的从不是统治，而是以统治者所能支配的巨量资源，去冲击 **「至强者」**（冥神、魂大人、鹿神那一级别；比黑玫瑰所求的「绝强者」更高）。他身边还有一名沉睡的**王妃（强大占卜师）**、以及一位使他魂伤难愈的旧伤。

!!! note "绝强者 / 至强者（幸运女神口中的分层）"
    **绝强者** = 达到九阶巅峰实力者（凛风王、圣女座那一层，黑玫瑰的目标）；**至强者** = 实力超出九阶者（冥神、刀魔、不死老人、鹿神那一层，沙之王的野心）。再上则是团长、至高之人。

## 👑 戴上灵魂王冠 · 堕为疯王（49卷34–35章）

⚠ 苏晓的猎杀手段一如既往:遇事不决就送**原罪物**。他让 [凯撒](../尼古拉斯·凯撒.md) 以【欺诈者头裹】**替换了军需官·加布奇的存在**（概念上顶替、当了十几年军需官）、借右御·卡伽之口把 **[灵魂王冠](../../力量体系/装备道具.md)** 献给沙之王。⚠ 沙之王一眼认出是原罪物、一度暴怒下令斩杀凯撒 + 丢弃王冠，可终究被「契合度高 = 莫大机遇」的念头蛊惑，深夜取来戴上——瞬间吞掉最忠心的**右御·卡伽**的命源（本源生命力 + 本源力量），一举冲破卡了他近百年的瓶颈（自觉再进两步就能追平背叛者）。⚠ 他就此堕为**疯王**:身高涨到 3 米 5、双眼全黑、酒红短发变作有生命般的漆黑长发。他误以为是自己「驾驭了王冠」（把记忆空白期自行合理化，正是灵魂王冠最可怕之处——从不强控、只让持有者自以为掌控），亲手捏死想投奔银面的**左御·佩温**（鹿角组织·银面同款银面具），把献冠有功的**凯撒升为左御**、以家人要挟亲卫队长·索瓦。⚠ 因灵魂王冠加持，猎杀名单上「倒戈者·沙之王」的悬赏从 **800 盎司一跃升至 1300 盎司**、更名**疯王**。他狂言「这世界除了那深渊之影，没人能杀我，辉光不行，那个自称席尔维斯的深渊滋生物也不行」。

## ⚔️ 圣沙堡死战 · 500% 生命值成陷阱（49卷36–38章）

⚠ 灵魂王冠让沙之王个体战力暴涨（悬赏 1300），却也让他两天内吞尽左右御与几百亲卫、势力分崩离析（王冠对君王是「1000% 杀伤加成」，高度契合的是**收拾**他而非使用）——反倒比从前好对付。⚠ 他手持虚空最重三把武器之一的起源级重剑 **[渊陨](../../力量体系/装备道具.md)**（灭法委托恶魔铁匠以深渊金属所锻·恩师所赠），资料显示疯王状态 500% 生命值、真实力量 300（本世界极值），奥义被动**不灭之影**（每降 1% 生命 +防御）、**灵魂狂乱**（承受灵魂伤害 +75%）。⚠ 苏晓识破:500% 生命值是**吞命源满溢的陷阱**——循规蹈矩打到 100% 以下他会进巅峰状态，倒不如**给他喂更多治疗、让生命力溢出到爆炸**。他把 [圣诗](../../人物/塞尔星/圣诗.md) 塞进沙之王阵营当治疗师、用 **[血羽](../../力量体系/装备道具.md)** 让圣诗的治疗对沙之王变 5 倍、又借死战让圣诗在沙之王身上叠满 3 层灵魂印记。

## ☠ 被灵魂怒涌奶爆 · 抹名（49卷38章）

⚠ 圣诗被驱逐出沙之王阵营、与苏晓同阵营后，对沙之王施展奥义 **「灵魂怒涌」**（15 秒内每秒恢复 20% 最大生命 = 1500% 生命值）——本就本源生命力满溢的沙之王当场躯干膨胀、胸口炸洞、增生血肉暴涌，最终整个人化成**百米大小的肉球爆开**，王冠飞出、他干瘦濒死。⚠ 濒死时他想拿「背叛者的消息」换活路，苏晓一发**血烟炮轰碎其头颅**（不与濒死叛徒废话），抹去猎杀名单「倒戈者」之名——结算 **1300 盎司时空之力**（时空石碎片 ×60 + 技能升级仓免费使用权限［值 700 盎司］）+ 击杀奖励 2900 钱币 + 16.8% 世界之源 + 起源级重剑**渊陨** + 起源级宝箱·疯王。⚠ 苏晓随后以**噬灵者**剥离他的灵魂记忆（不摄取·因混杂原罪物 / 深渊气息），窥见黑暗教堂内**沙之王、深渊首领·席尔维斯、背叛者**三人会面的往事——下一个目标就此清晰:深渊首领·席尔维斯。灵魂记忆还揭出他背叛恩师的定格:一剑背刺马文·华尔兹，反被恩师一嘴巴子 + 一刀伤到濒死、却终究被恩师留了活口。
