# 神秘者·黑玫瑰

[苏晓](../苏晓.md) 猎杀名单上的第六名叛徒「**神秘者**」，本世藏于**圣兰王国**、人称**黑玫瑰**（悬赏 600 盎司时空之力）。经「势力百科全书」[老院长](老院长.md) 的情报，她的真实分量才显露:⚠ **她其实是圣兰王国的「幕后女王」**。

## 圣兰王国的真正掌权者

⚠ 圣兰王国如今**新王年幼**、权柄由三派联合把持——几名老国王遗留的**权臣**、**王后**、以及**晨曦神教大祭司**。⚠ 但王后不过是**傀儡**:是黑玫瑰亲手培养、扶持上位的;黑玫瑰**通过掌握王后来掌控几名权臣**（王后许诺权臣「拥护她就继续手握重权」），神权一侧的晨曦神教又给足配合。⚠ 于是「说黑玫瑰是圣兰王国的女王，一点问题没有」——圣兰王国历史上从没有国王能做到她这种掌控程度。这也解释了晨曦神教为何敢向联盟扩张、又为何撑 [古斯沃（耶辛格）](古斯沃.md) 夺权。

## 六叛徒中权谋最强

⚠ 作为叛徒，黑玫瑰**战力在欺骗者、窃夺者、告密者之上**（六叛徒中上游），而**权谋很可能是六叛徒中最强的**。苏晓定下的猎杀次序是:先取藏在噩梦岛的**告密者·索恩**，再赴圣兰王国取黑玫瑰，最后去沙漠之国找**倒戈者·沙之王**——黑玫瑰这一战，注定是权谋与战力的双重硬仗。

## 给告密者的信 · 苏晓的下一个猎物（49卷18–22章）

⚠ 苏晓在噩梦岛 [告密者](告密者·索恩.md) 的地下宝库里，找到一封落着**黑玫瑰图印**的信:「背叛者把[唤醒之碑](../../世界/任务世界/暗影世界-背叛者猎杀.md)弄到了这世界，这可能会引来麻烦，我们几人去找他无异于找死，你曾是他的手下，你去才有些可能。」——由此坐实:**窃夺者死于背叛者之手**，而黑玫瑰与沙之王**大概率知晓背叛者的踪迹**（否则她不会知道唤醒之碑被弄来本世界）。⚠ 据 [幸运女神](../幸运女神.md) 的本世界强者排行，黑玫瑰**位列第八**（垫底，但权谋最强）。苏晓判断:一旦硬闯圣兰王国，会同时对上**王族、黑玫瑰的隐秘势力、晨曦神教**三方，必是惨胜甚至必败;于是改为「单个收拾」——先猎杀晨曦神教信仰的 [辉光之神](辉光之神.md)（本世界第二强、身边没保护者），再回头收拾失了神权靠山的黑玫瑰。为此他还扣先古面具让 [龙神·迪恩](../幽暗大陆/龙神·迪恩.md) 伪装白夜高调赴圣兰，把黑玫瑰的视线钉在假白夜身上。

## 🎭 分身议会 · 追踪本体（49卷28章）

⚠ 苏晓弑辉光之神、大祭司背刺古拉公爵、拉拢小国王后，去王国议会与黑玫瑰摊牌。可议桌对面的黑玫瑰只是一具**能承载部分精神力的血肉分身**（她行事缜密，绝不以本体来到围满禁军之地——除非禁军全归她掌控，而这些禁军恰由古拉公爵的亲侄子调遣）。⚠ 苏晓故意与分身「废话一整天」等术式激活:黑触须缠住分身、让本体与分身的联系更紧密，[凯撒](../尼古拉斯·凯撒.md) 以新得的罗盘**追踪定位到本体**（王都后区一座庄园），随即血枪爆破了分身。

## 💔 神秘者的真相 · 一体两魂（49卷28章）

⚠ 苏晓循踪抵达**苦难之巢**，却在血肉墙内找到一个双手被缚、油尽灯枯的**紫瞳女子**——她才是**真正的黑玫瑰**，猎杀名单上「神秘者」600 盎司悬赏的正主。真相由她道来:很多年前，一个充满好奇的天才少女用先祖的危险知识把自己**逆向传送进了「深渊」**，回来时已被深渊侵蚀到濒死，恰被 **银·月狼·希狄 + 老灭法** 所救（保住命却无法根除与命源融合的深渊力量）。⚠ 她选择随他们去灭法阵营，成了应对深渊侵袭的专家、常与月狼结伴奔赴深渊爆发之地救人。可长年**过度压制体内深渊力量**，竟让她的命源**一分为二**——诞生出另一个气息、灵魂、记忆都相同、但性格想法迥异的「她」;这另一个她压制力惊人，瞬间包裹压制了黑玫瑰本人、接管了身体，**背叛到施法者那边**（把先代灭法与施法者双方都秀懵）。⚠ 所以她**既算背叛灭法、又不算**——这正是「神秘者」这奇特称呼的由来。真身黑玫瑰以将死之躯被封困续命至今、劝苏晓「等你更强些再来」，随即化尘散去。苏晓以她几滴血抹名，领 **600 盎司**（时空石碎片 ×10 + **豁免徽章（★★★★★）**·免魅力 / 意志 / 幸运低于 0 点后 −50 内的减益，苏晓当即用掉）。

## 👹 叛徒人格「苦痛女王」· 被放逐永光世界（49卷28章）

⚠ 那个活跃在外、晋升为**绝强者**的「黑玫瑰」，其实是命源分裂出的叛徒人格——她来本世界的目的，从不是掌控圣兰王国，而是借这饱受神灵摄取、王族压迫的王国滋生的海量**厄难**冲击绝强者（与辉光之神收集信仰、她收集厄运，一体两面）。她凭「苦难之巢」摄取王都后区所有生灵的厄难、蜕变时的余波把后区变成炼狱、侦测其名已成**苦痛女王**（灭世级灾祸）。⚠ 苏晓没硬拼（现阶段必是死伤惨重的惨胜），而是祭出 **[封之刃](../../力量体系/装备道具.md)**（灭法开启 / 放逐灭世级之物·苦痛女王当年偷走、丢在死灵之书旁，却被神父带走、又辗转回到灭法手中）——蓝色空间漩涡把她拖进单向通道、放逐到**永光世界**（若遇上被苏晓封进去的银皇后、蛀世，倒有共同话题）。
