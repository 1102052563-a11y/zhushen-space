# 白金主教

**太阳神教**在联盟一侧的全权代表——一身红袍、戴**白金面具**（自加入教会起就戴、从没摘过），言行随性、不着调，干过「站议会院当避雷针顶赞美太阳」之类啼笑皆非的事。⚠ 可**真正了解他的人都知道他深不可测**:[苏晓](../苏晓.md) 与他见面时，那种「若真死战、彼此各占五成胜算」的感觉，比对联盟最强的 [泰莎](泰莎.md) 还强——是苏晓在本世界**最忌惮的存在**。他手下有近四米高、持水桶粗权杖的**野兽骑士**，和一个爱看倒着的书、把飞龙气到龙目怒瞪的**红瞳女**。

## 联合苏晓 · 取代修道院

⚠ 太阳神教约见地点接连被 [古斯沃（耶辛格）](古斯沃.md) + 晨曦神教查封，白金主教索性带着野兽骑士、红瞳女**亲赴黄昏疯人院**，谈「取代人手凋零的修道院、成为凶犯们新的矫正 / 感化部门」（这是苏晓「被迫」联合太阳神教、反击晨曦的正当理由）。苏晓惯用「借钱→送钱」笼络这穷分部，双方一拍即合。⚠ 他还替苏晓照看那颗发育飞快的**恶魔焰龙**（用金白锁链束缚）;苏晓把他那辆「除喇叭不响哪都响」的破车车门拆下丢水沟、扔他一把新车钥匙，他反倒喜提新车、还主动要「借用」苏晓的传送阵（首位主动想用苏晓传送阵的人），一被传送就吓得面具都歪了。

## 深不可测的战力

⚠ 白金主教答应与苏晓**同赴噩梦岛**（「收了你那么多药剂、去趟噩梦岛只是小事」），还提醒苏晓「至少准备 300 海盗金币买骨帆船、否则黑暗海域船沉都回不来」。他、红瞳女、野兽骑士三人都是战力担当——苏晓评估:北境帝国最强大将军约与泰莎相近，而白金主教这种「平常做事不着调」的，才是真正的威胁。

## 关系

- [苏晓](../苏晓.md)：与他联合对抗晨曦神教、约定同赴噩梦岛的盟友——也是苏晓最忌惮、死战各五成胜算的存在。
- 太阳神教 / 野兽骑士 / 红瞳女：他代表的神教、以及麾下两名战力担当。
- [古斯沃（耶辛格）](古斯沃.md) / 晨曦神教：查封他与苏晓约见地点、被两人合力反击的对手。
