# 北境公主（水晶姬）

五吞噬者之一 **水晶姬** 的宿主——北境帝国的公主，一头**白中隐隐透浅蓝的长发**、右耳银色耳坠，优雅中带点小慵懒，说话总是「果然只有需要我时才想起我」这类多愁善感、有点不着调的调调，是唯一把 [巴哈](../剧毒魔鹰·巴哈.md) 都「整不会」的人。⚠ 她正是此前给苏晓寄**唇印邀请函**「你终究还是来了」的女子。

## 🃏 看似废柴 · 实为钞能力翻盘的赢家

⚠ 水晶姬的特性是**与宿主融合（非寄生）、起点更高**（宿主够强可开局四阶段），偏偏选了实力平平的北境公主、开局只一阶段。可这场吞噬者争夺战里，比她强的暗阳出局、黑A 怀疑人生、太阳使徒隐忍、艾丽莎全程死战，唯独她每天规律惬意地生活、抽空提升一点，被其他四人「选择忽略」，反倒得了个完美收场。⚠ 她真正的底牌是**钞能力**:爷爷临终留给她**暗影商会 20% 的股份**（大股东）——而「暗影世界」在轮回乐园的公证名正是暗影世界，暗影商会八成就是从本世界崛起、把生意做到各乐园阵营的巨头（苏晓买斩龙闪的同品质武器多在此）。她藏了多少一次性道具没人知道，真打起来能「一把一把往外扔道具、以钞能力灭杀对手」。

## 🚚 谈判入伙 · 挖矿队智囊 + 北境向导（49卷49–50章）

⚠ 苏晓去北境找背叛者，必须由她作**引荐人**（否则连灭四五名本世界顶尖强者的苏晓，北境君主绝不敢放进北境）。她大姐悬赏 3 亿古朗要活捉她（王裔间的恩怨），回北境有风险，可她还是心动于「游历万界」。⚠ 她把「我战力不如黑A、谋略不如太阳使徒」的谦辞挂嘴上，转身却亮出完整方案:进八阶世界后先由她**交涉**、不成再让艾丽莎斩敌方高层 / 黑A 群战 / 太阳使徒攻心，采完矿再凭暗影商会渠道**高价 15%~30% 卖出**——一跃成为**挖矿队**不可或缺的一员。她选苏晓这边的理由很实在:「你们不在乎这世界的强权、不会和我姐姐合谋，合作的小老头是大名鼎鼎的欺诈者，而且白夜院长强啊，找这么强的靠山不好吗」——哪怕得知苏晓与奥术永恒星敌对（对方有三四名绝强者、一名至强者），她也不在意（「我是他敌人的合作者，奥术永恒星懒得为收拾我掉身价」）。⚠ 她还认得背叛者:那是教出北境大将军的**石刻师**。

## 关系

- **水晶姬**：与她灵魂意识融合、起点本可极高却因她实力平平而开局垫底的吞噬者。
- [苏晓](../苏晓.md)：她谈判加入的挖矿队「靠山」、去北境找背叛者的引荐对象。
- 大姐（北境王裔）：悬赏 3 亿古朗要活捉她的姐姐——她因重亲情被父王偏爱、惹来兄姐忌惮。
- 爷爷 / 暗影商会：临终留她 20% 股份的最疼她的长辈、以及她赖以钞能力翻盘的巨头商会。
- 背叛者（[席曼·阿奇德](背叛者·席曼·阿奇德.md)）：她认得的「石刻师」——北境大将军的恩师。
