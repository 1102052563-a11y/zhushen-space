# 欺骗者·彼司沃

[苏晓](../苏晓.md) 猎杀名单上的**第一名叛徒**「欺骗者」——本世转生为一个名叫**彼司沃**的诈骗犯（诈骗 7000 多万古朗、被判送黄昏疯人院·悬赏 200 盎司时空之力·六叛徒中最低）。他其实是**虚空灵族的[转生者](../../世界/生物万族/index.md)**（转生能力与生俱来·灵魂强韧到能鸠占鹊巢附入新生命），本世尚未觉醒宿世记忆时只是个提心吊胆、怕监狱的骗子。

## 两度改投的叛徒真相

⚠ 苏晓起初以为他是「投靠奥术永恒星才获转生混血」，觉醒后真相浮现:他**原本就是灵族**。当年**灵族与奥术永恒星交恶、遭法师贤者瑟菲莉娅筹划报复**，灵族为求生**投靠了灭法阵营**（灭法虽少、却有思林特斯矮人、恶魔族等盟友）;可到灭法阵营面临败局时，欺骗者**代表灵族又改投了奥术永恒星**（那时奥术永恒星刚灭思林特斯矮人、正需展现「不赶尽杀绝」以安抚恶魔族等灭法盟友，遂接纳灵族投靠）。⚠ 讽刺的是，灵族最终仍被施法者们**抽干转生魂血而灭亡**——欺骗者背叛了灭法，却没能保住灵族。他常想起当年**灭法者·阿卡斯**带他去过的各个世界;提心吊胆转生了这么多世、活得漫无目的，见到找上门的新一代灭法苏晓，反而安心求死。

## 被设局擒获 · 觉醒与招供

⚠ 苏晓的设局堪称教科书:先抽走彼司沃律师**弗恩**的一滴血（血枪宗师无伤抽血）、支开弗恩，再让拟态囚犯 [女妖](女妖.md) 用那滴血**拟态成弗恩律师**、去索托市审判所——以盖着「黄昏疯人院 + 猎手部队」印章的**精神评估证明**，让彼司沃「主动要求进黄昏疯人院」（老法官几十年首见有人主动要去疯人院）。⚠ 入地牢三层后，苏晓以**灵体封困术式**（契约羊皮纸 + 灵魂晶核·把转生者灵魂锁在肉体内、死也逃不掉魂体）+ 仁慈之刺贯喉逼他**觉醒宿世记忆**，又两根仁慈之刺压得刚觉醒的他毫无还手之力。他把知道的全招了（当初意志坚定就不会当叛徒）——供出其余五叛徒:**告密者·索恩**（噩梦中）、**神秘者·黑玫瑰**（圣兰王国）、**倒戈者·沙之王**（最西的沙漠之国）、**背叛者**（无人知其名讳来历·最强·他躲都来不及）；⚠ 并点破**窃夺者已死在背叛者手中**（其灵魂残屑仍在·苏晓可凭之领 500 盎司悬赏）。⚠ 苏晓暂不杀他（刃之魔灵正消化深渊滋生物本源、暂时斩杀不了），撂下「过几天再来杀你」。

## ☠ 被斩 · 猎杀名单首名抹除（49卷15章）

⚠ 待刃之魔灵消化完深渊滋生物本源，苏晓回到地牢三层。彼司沃正念叨着「这几天我经常想起阿卡斯先生……」——话没说完，苏晓一刀斩断他喉咙、**黑蓝色灭法之刃烟气顺伤口涌入斩杀其灵魂**（巧舌如簧的欺骗者，对上「从不听敌人废话、也不给敌人说话机会」的苏晓，是彻底的克星、连一句蛊惑都没来得及说）。⚠ 苏晓以他的血抹去猎杀名单上「欺骗者」及其后一长串转世曾用名——**成猎杀名单首名被抹除的仇敌**。悬赏结算:原 50 盎司因血契多倍变 **200 盎司时空之力**（时空石碎片 ×15［值 150 盎司］+ 起源级宝石盒［必出满评分起源级宝石·值 50 盎司］）;另有击杀奖励 **11.9% 世界之源 + 转生匣**（低概率转生魂血 / 高概率灵魂系能力）。苏晓照例往牢房丢一颗普通阿波罗（遇过死两次仍活的神父后养成的习惯）。
