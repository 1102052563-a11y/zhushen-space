# 迪尤尔

满脸大胡子的黄昏疯人院**安保部门总队长**——表面是疯人院的人，实则是**「猎手部队」那边的钉子**，且**不站 [苏晓](../苏晓.md) 这边、支持已去议会院活动的老副院长**（那老头想凭议会院人脉把新院长苏晓搞下去）。

## 逼签卸任 · 翻脸比翻书快

⚠ 苏晓上任要快速掌控疯人院，第一步就是清走迪尤尔、换上自己人。迪尤尔起初拿捏「我们大人（猎手部队领袖）那边我不好交差」摆架子、点烟试探——苏晓一把**斩龙闪出鞘**，他表情一僵、立刻笑着签了卸任文件（好汉不吃眼前亏）。⚠ 这其实是他的**投石问路**:唯有苏晓给出「态度」（亮刀），他才好回猎手部队交差、说自己是被强势新院长逼走的，否则灰溜溜回去日子不好过。苏晓看破这层、还暗叹「这种翻脸比翻书快、只要油水足脏活累活都干的混账，要不是猎手部队的人，说什么都得挖过来」，最后打发他去财务部**领半年薪资**走人。

## 油滑的后手

⚠ 清走迪尤尔有弊端:安保部门会短期受影响、地下囚犯蠢蠢欲动（也正因此，苏晓才顺势下地下牢房立威）。但迪尤尔的油滑之处在于——他看似与新院长彻底闹翻、撤走手下，实则**正门、所有岗哨塔、地下三层的安保力量一点没动**，反而在周边岗哨塔加派了人手（既给苏晓面子、又不真把疯人院的安全弄砸）。

## 关系

- [苏晓](../苏晓.md)：逼他签卸任书、看破他「投石问路」、还暗叹「可惜是猎手部队的人」的新院长。
- 猎手部队 / 领袖泰莎：他真正效力的隐秘组织（他是安插在疯人院的钉子）。
- 老副院长：他支持的、想把苏晓搞下去的政敌（此时在议会院活动）。
