# 老院长

黄昏疯人院一层安保室里那位端着热茶、盯着监控、穿着安保制服却有些邋遢的**看门老大爷**——⚠ 其真实身份是疯人院的**上上任院长**，退休后闲得无聊才来看门。这次被迫退休的老院长，当年正是他一手培养出来的，由此可见这老狐狸在疯人院的资历有多深。

## 老狐狸的识时务

⚠ 苏晓深夜来安保室，老院长一眼看穿他「把安保总队长 [迪尤尔](迪尤尔.md) 清出去是鲁莽、要搞大事」，当即表态「我有种预感你要搞大事，为免被牵连，我还是回去养老吧」。⚠ [苏晓](../苏晓.md) 顺势跟他做交易——「走前给我引荐几名人才」，老院长起初推诿「我只是个老家伙」，苏晓一句「那明天我把你孙女调到疯人院来、放 [艾琳诺](艾琳诺.md) 手下做事」，老狐狸动作一顿、笑着松口「随你吧，你就算娶了我孙女我都不管，刚好你们年龄相近」，答应「晚些派人送来几份简历」。临走他仔细审视苏晓片刻、满意点头——把黄昏疯人院交到这样一个「既有实力、做事又不死板」的人手里，他放心了。

## 🏃 预谋已久的跑路 · 保险箱补偿（49卷4–6章）

⚠ 老院长嘴上说「大家统一战线应对副院长 [古斯沃](古斯沃.md)（耶辛格）的报复」，转头却**带上全家 + 全部资产连夜跑去小国**——报复还没开始人就没影了，[泰莎](泰莎.md) 在他住处接电话时一句「老东西已经跑了」道破。⚠ 这是**预谋已久**的跑路。他给苏晓留了个叮嘱「别打开」的保险箱、还附信「看在传位情面帮忙看管」——苏晓（用神秘之眼开锁）撬开一看，是**商盟银行储物箱钥匙** + 一张纸条:「白夜，我已带家人离开，这份礼物就当补偿，别来找我」——翻译过来即「我跑路了、你和耶辛格的事老子不管了、这是精神补偿、别追杀老子」。这老狐狸把院长位交给外人、又留补偿求不追杀，滑不留手到底。

## 关系

- [古斯沃](古斯沃.md)（耶辛格）：他忍让 / 提防了几十年、临跑路把烂摊子甩给苏晓去应对的死对头副院长。
- [泰莎](泰莎.md)：在他跑路后的住处接苏晓电话、道破「老东西已跑」的猎手部队领袖。
- [苏晓](../苏晓.md)：他嗅出「要搞大事」而主动退避、留保险箱补偿求不追杀、又被其「调孙女」拿捏着引荐人才的黄昏疯人院新院长。
- 被迫退休的老院长：他一手培养出来的、这次被迫交位给苏晓的继任者。
- [艾琳诺](艾琳诺.md)：他孙女将被调去其手下做事的现任副院长（同属「老院长一派」）。
