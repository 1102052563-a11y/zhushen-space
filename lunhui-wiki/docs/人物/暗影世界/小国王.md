# 小国王

圣兰王国现任国王——一个**不足十岁、灵魂却强壮如几十岁**的聪慧少年。历任圣兰国王都被 [黑玫瑰](神秘者·黑玫瑰.md) 除掉（活不过 40 岁），王族为拯救命运用了下策:⚠ 他的**灵魂承自父亲**（准确说是**摄取了父亲的见闻**、而非被父亲的意识占据——「我父亲只是想让我好好活着，不是要借我重新活过来」）。黑玫瑰虽知情，但杀这傀儡的麻烦太多、辉光之神也不允许（血誓之威神灵都不敢试），只得留着他当充数的傀儡。

## 站队苏晓 · 求活的聪明人

⚠ 在 [苏晓](../苏晓.md) 弑辉光之神、大祭司背刺古拉公爵后，苏晓让布布汪投影去见小国王:「你是黑玫瑰的死敌？那我们是朋友。」——小国王当即结盟。王国议会上，他隔着议桌对黑玫瑰的分身扬起下巴、不再扮傀儡。⚠ 黑玫瑰（苦痛女王）晋升后，小国王坚持跟着苏晓一队走:「现在那老神棍和王后都想我死，但只要我撑到事态平息，他们又会被迫把我托上王位，跟着你们，我活下来的可能最大。」他还领苏晓去了王宫最隐秘的**地下密室**（继承父亲记忆才知的祭祖之地），苏晓在此布下 32 颗灵魂晶核的阵图、暂时压住了地下**深渊通道余存**里怒吼「灭法」的万千生灵。

## 关系

- [苏晓](../苏晓.md)：认作「黑玫瑰死敌即朋友」、他为求活紧紧跟随的灭法者盟友。
- [黑玫瑰](神秘者·黑玫瑰.md)：除掉他历代先王、把他当傀儡、又被苏晓联手扳倒的幕后女王。
- [大祭司·特里维康](大祭司·特里维康.md) / [白金主教](白金主教.md)（伪·古拉公爵）：与他同站苏晓一侧的翻盘同盟。
