# 红瞳女·希莉德

[白金主教](白金主教.md) 麾下两名战力担当之一——一双红瞳、爱看倒着的书、随处赞美太阳（在车厢里舒展不开双臂会把 [布布汪](../布布汪.md) 与 [阿姆](../阿姆.md) 挤到两边偏头）。⚠ 她是**太阳阵营传承的独苗**，各类别人梦寐以求的太阳秘术，她小时候是学到哭鼻子的。太阳圣殿石碑上「太阳大主教·席尔维斯 / **红瞳女·希莉德** / 野兽骑士·加尔」三名字，正指她与白金主教、野兽骑士三人。

## 🩸 被黑暗神教掳走 · 疑深渊首领之女（49卷39–43章）

⚠ 白金主教在幽魂城前不辞而别（最后被人看到独坐郊外公园长椅、看夕阳），而红瞳女没与他同行——她被黑暗神教**主祭·豪德斯**掳走（豪德斯控制一群小孩冲向她、逼她不敢出手，再把她吞进虫腹带回）。⚠ 深渊首领·席尔维斯下令要「安稳带回」她，见豪德斯把她吞进虫腹，一手将豪德斯抽成粉碎。⚠ 主教·血妖审视她眉眼，严重怀疑她是**教皇席尔维斯的亲女儿**——只敢关在偏殿、给她一桌秘术卷轴让「傍晚前学会（否则今晚没饭吃）」、还允她在教堂范围内由教堂骑士陪着「随便逛逛」。⚠ 红瞳女性子刚烈:假装昏厥、暴起给深渊首领·席尔维斯脸上一记勾拳（自损骨裂的「伤敌 0、自损 999」），转身就往外奔逃，被教堂骑士与血妖拦回。她的真实身世与白金主教一行的来历，成了本世界又一处未解之谜。

## 关系

- [白金主教](白金主教.md)：悉心教导她太阳秘术的师长——太阳圣殿石碑上与她、野兽骑士同列的太阳大主教。
- [掌控者·席尔维斯](掌控者·席尔维斯.md)：令主祭「安稳带回」她、疑似是其亲生父亲、被她一记勾拳打脸的深渊首领。
- 主祭·豪德斯 / 主教·血妖：掳她的黑袍主祭（因把她吞虫腹被席尔维斯抽碎）、逼她学秘术又疑她是教皇之女的女主教。
- 野兽骑士：与她同列石碑、后从众人记忆中消失的太阳阵营同伴。
