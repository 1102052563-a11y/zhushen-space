# 珀金市长

联盟·库斯市的市长——一个**非超凡者**、人到中年、诸事操劳而身材发福的实干派。⚠ 他被称作库斯市的「**财神爷**」:对同驻本市的**猎手部队**与**黄昏疯人院**历来大方（苏晓刚上任、他后天就要拨来 600 多万古朗）。他上任 + 猎手部队老领袖退位（继承人泰莎接任），是几年前联盟迎来「**最安定时期**」的两件关键事之一。也正因不能得罪这位财神爷，连猎手部队领袖泰莎都对他客气。

## 半夜查地基 · 一顿佳酿定立场

⚠ [苏晓](../苏晓.md) 打穿地下监牢地基、电话里报出「见水了……地下水」，珀金市长当场挂电话、穿衣上车火速赶来（那地基当年造价惊人、整个联盟能修的人很少）。苏晓早备好丰盛夜宵 + **元素佳酿**等他——一口温润好酒下肚，市长大半夜赶来的闷气就消了大半。⚠ 他下地下监牢一看:凶犯个个温顺、每见他目光就挤笑脸（苏晓立威的成果），关注点却全在「修这地洞要花多少古朗」。他嘴上「市财政没钱了」，却早已跟圣都招呼、调好维修人员和重力金属（后半夜三点前到）——典型的**嘴上哭穷、事事办妥**。⚠ 临走车上他自语:「那两只老狐狸，这次选了个杀伐狠戾的，挺好，能让猎手那些人安分点，而且白夜自酿的酒真不错。」——苏晓由此摸清了这位财神爷的立场:惹不起、但只要不越界、他总把事办利索。

## 关系

- [苏晓](../苏晓.md)：黄昏疯人院新任院长——用元素佳酿 + 妥当态度换得这位财神爷认可、也摸清其立场的对象。
- 猎手部队 / 黄昏疯人院：他一向大方拨款的两个直属部门（他的拨款是两边都不愿得罪他的原因）。
