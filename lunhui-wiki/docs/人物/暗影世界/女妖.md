# 女妖

黄昏疯人院**地下三层**的五名重囚之一——一个能力极其危险的女人。⚠ 她的本领是「**拟态**」:**凭他人的细胞，就能获得对方已掌握的专业知识与能力**（太强的能力不行）、并变成对方的模样。正因如此，她曾**伪装成一位联盟大议员、堂而皇之走进议会院**达成自己的目的，被判 **1 万多年**刑期、关进疯人院地下三层。她始终没打消越狱的念头。

## 苏晓擒彼司沃的关键棋子

⚠ 苏晓设局擒 [欺骗者·彼司沃](欺骗者·彼司沃.md) 时，女妖是关键一环:苏晓抽走彼司沃律师**弗恩**的一滴血、丢给女妖，女妖含血入口便读出「男性细胞、律师的细胞记忆」、拟态成弗恩的模样与专业。⚠ 她（伪装的弗恩律师）赶赴索托市审判所，在老法官即将宣判时递上盖着黄昏疯人院 + 猎手部队印章的**精神评估证明**，让彼司沃「主动要求」被判进黄昏疯人院。作为报酬，苏晓答应她**每周能在疯人院大院自由活动两小时**——她欣然成交（这既是改善、也更方便她的越狱计划）。她还在地牢三层配合演戏、变回弗恩模样吓崩溃刚入狱的彼司沃。⚠ 苏晓从不一开始就亮全部筹码，而是先压低开价、到紧要关头再抛个对方想不到的高价——女妖正是被这套拿捏。

- [欺骗者·彼司沃](欺骗者·彼司沃.md)：被她拟态其律师弗恩、一步步送进疯人院地牢的首名叛徒。

- 怒鲨 / 狮王 / 心灵大师 / 憎恨：与她同囚地下三层的四名重囚（皆未打消越狱念头，唯憎恨例外）。
