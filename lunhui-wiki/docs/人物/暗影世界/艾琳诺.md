# 艾琳诺

黄昏疯人院两位副院长之一——不到三十岁的未婚女士，戴眼镜、穿订制职装，一副淑女气质，声音「听着让人酥酥麻麻」。她当初以**远超入职要求的专业水平和超凡资质**加入疯人院，联盟不少权贵都惋惜「这种人才该去议会院、而非那可怕的黄昏疯人院」。

## 淑女皮相下的抖S

⚠ 别被她的淑女形象骗了——她是个**超级抖 S**:天生一个毛病，**看到别人痛苦（且痛苦不能是她造成、她须以旁观者身份）就会难以克制地愉悦**。老院长起初也惋惜「这么好的年轻人不该来」，两天就收回想法，认定「她不该穿医生白袍、该穿病号服」。她最初当牙医时**不打麻药就拔牙**、吃了官司被传唤审判所（家里赔钱 + 本人道歉才作罢）。可她确实适合黄昏疯人院——那些凶徒见了她先兴奋嗷嗷叫，一对上她的眼睛就少有人敢挑衅。如今凶犯的**矫正、感化、精神评估**全由她手下负责，她每天去「视察工作」。⚠ 她的精神评估是囚犯命运的分水岭:评估「因病犯罪」转疗养院、评估「纯粹内心邪恶」则直接下地下牢房。

## 一把斩龙闪逼出的表忠

⚠ 新院长 [苏晓](../苏晓.md) 上任第一件事就是摸她的底——把归鞘的斩龙闪往桌上一放、面无表情地问「我和那老头，你支持谁，大胆说，我不会把你怎样」。艾琳诺神情一肃、答得干脆:「当然是支持你，我是老院长一派系、都是自己人，所以把刀收起来吧。」——她本不太看好这位新院长，但经此一逼、已逐渐看清局势、认清苏晓是个既有实力又不好惹的主。苏晓还把 [老院长](老院长.md) 的孙女调来、安排在她手下做事。

## 关系

- [苏晓](../苏晓.md)：黄昏疯人院新任院长——用斩龙闪逼她表忠、她遂支持的上司。
- [老院长](老院长.md)：她所属「老院长一派系」的旧领袖（上上任院长的培养者）。
- 黄昏疯人院凶犯：被她矫正、感化、精神评估、又对她眼神发怵的地下囚徒们。
