# 辉光之神

**晨曦神教**所信仰的神灵，端坐于圣兰王国的**神域**（一处独立空间·似神殿）内。它是 [苏晓](../苏晓.md) 猎杀 [神秘者·黑玫瑰](神秘者·黑玫瑰.md) 计划中的第一个目标——因为它与黑玫瑰串通一气，「敌人的朋友，就是新的敌人」。

## 本世界第二强 · 凭信仰累积的伪善恶神

⚠ 据 [幸运女神](../幸运女神.md)（也是神灵、来过本世界很多次），辉光之神比众人预估的强得多:苏晓原以为它与沙之王实力相近，其实它稳居**本世界实力第二**（仅次于背叛者、在深渊首领·席尔维斯之上）。⚠ 幸运女神更一语道破:「凭信仰之力累积神血的神灵，都不是友善神灵」——辉光之神连中立都算不上，**根本是恶神**。它是圣兰王国晨曦神教壮大、几任国王活不过 40 岁、大祭司只遵神谕不理王命的幕后神权支柱，与掌王权的黑玫瑰形成神权 / 王权的合谋。它几百年没出手，实力情报极难调查。

## 🎯 被猎神 · 强制祭献 + 运势碾压（49卷22–24章）

⚠ 硬碰辉光之神并不明智（后头还要对付沙之王与背叛者）。苏晓借 [九星称号·猎神者](../苏晓.md) 定下逆向思路:不与晨曦神教死磕，直接猎杀它信仰的神灵。手段是一套精密合谋——

- **强制祭献**:[凯撒](../尼古拉斯·凯撒.md) 头戴【欺诈者头裹】冒充虔诚信徒祈祷、蒙骗过神，再人罐合一强开空间漩涡「强行祭献」，往辉光之神神域里塞进**恶臭增益过的破鞋**（气到神域震动怒喊）、一颗颗**烈阳之怒·阿波罗**、以及**灵魂王冠**（塞进去又被暴怒弹回、被苏晓深渊盒接住封回）。
- **运势碾压**:[幸运女神](../幸运女神.md) 以**燃烧 500 多滴幸运神血**为代价，临时把苏晓的运势拉满、同时压低辉光之神的运势;苏晓再以刚晋升起源级的**[命运主宰](../../力量体系/装备道具/index.md)** + 幸运属性引【界雷】——引雷引到一半停手，界雷劈谁**全看运势**。灭法「水罐砸水杯」的运势差一旦被灌满，界雷必然精准劈向负运势的辉光之神。

（49卷24章末，幸运女神已把运势尽数灌入苏晓胸膛、脸色惨白，这场猎神之战正式打响。）

## 关系

- [苏晓](../苏晓.md)：把它列为猎杀黑玫瑰第一步、用强制祭献 + 运势碾压猎神的灭法者。
- [神秘者·黑玫瑰](神秘者·黑玫瑰.md)：与它神权 / 王权合谋、共掌圣兰王国的幕后女王（「敌人的朋友」）。
- [幸运女神](../幸运女神.md)：戳穿它「凭信仰累积神血 = 恶神」、燃 500 滴神血为苏晓拉运势压它的友善神灵。
- [尼古拉斯·凯撒](../尼古拉斯·凯撒.md)：冒充其信徒、强制祭献往它神域里塞破鞋阿波罗的地精裁决者。
- 晨曦神教大祭司：只遵它的神谕、不理小国王王命的神权代理人。
