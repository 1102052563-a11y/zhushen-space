# 告密者·索恩（噩梦之王）

[苏晓](../苏晓.md) 猎杀名单上的**第二名叛徒**「告密者」——本世藏于**黑暗海域中心的噩梦岛**，人称**噩梦之王（噩梦之主）**（基础悬赏 400 盎司时空之力，六叛徒中偏低）。他曾是最强叛徒 **背叛者** 的**旧部下**，如今独踞噩梦岛，掌控这片被深渊侵袭过的黑暗海域。

## 只在噩梦岛上强大的叛徒

⚠ 告密者是六叛徒里最「跛脚」的一个:黑玫瑰掌控圣兰王国、沙之王统治沙漠之国，而告密者**只能待在噩梦岛上**——离开这里，实力就当场拉胯。因这缘故，轮回乐园给出的悬赏分三档:在黑暗海域边缘的**骷髅岛**杀他 = 400 盎司（基础）、在黑暗海域内区杀他 = 700 盎司（补正）、在其老巢**噩梦岛**上杀他 = **1500 盎司**（超补正·已与背叛者的赏金持平）。⚠ 这也反证:身处噩梦岛的告密者被加成到「比背叛者更难对付」的程度，才值 1500 盎司。当地传闻甚至把噩梦之王吹得强过深渊首领·席尔维斯与辉光之神，但那份资料满是传闻色彩、不足信。

## 噩梦领域 · 培植古树 · 黄金罐

⚠ 告密者能在噩梦岛上无敌，靠的是**噩梦领域**——这其实并非他所创，而是当年**深渊能量侵袭噩梦岛**留下的产物，他只是占据了此地;领域内有他花百余年布置的噩梦术式，以及一棵吸收噩梦本源、约百米高的**噩梦古树**（他想以古树过滤、结出本源果实，从而获得不受限的强大力量、好离开孤寂的噩梦岛）。⚠ 他的地下宝库藏着鹿神遗留的**[黄金罐](../../力量体系/装备道具/index.md)**（存恶神源血）、一座天启乐园的**湛蓝熔炉**，以及 [黑玫瑰](神秘者·黑玫瑰.md) 写给他的信——信中说「背叛者把[唤醒之碑](../../世界/任务世界/暗影世界-背叛者猎杀.md)弄到了这世界」、让曾是背叛者手下的他去交涉（由此坐实:窃夺者死于背叛者之手，黑玫瑰与沙之王大概率知晓背叛者踪迹）。

## ☠ 被斩 · 单挑到了烛女（49卷17–18章）

⚠ 告密者比苏晓本人更希望他尽快登岛——好凭噩梦岛之利将这尚未完全成长的灭法杀死。他在噩梦中策反了苏晓故意带来的航海士 [怒鲨](怒鲨.md)，编出「击碎三块界石可破噩梦领域」的鬼话，想把苏晓骗进领域单挑。⚠ 可苏晓将计就计:让怒鲨提着一盏点了 **[半融的脂肪蜡](../../力量体系/装备道具/index.md)** 的提灯——引来虚空异存在 **[烛女](../../世界/生物万族/index.md)**。噩梦领域一封闭，反成困死一切生灵的囚笼，苏晓以**神圣蜡烛**的庇护圈自保，任由烛女去「单挑」噩梦之王。一场深处的惨嚎后，烛女提回了告密者沾血的头颅。⚠ 苏晓以其血抹去猎杀名单上的「告密者」——**成第二名被抹除的仇敌**，超补正结算 **1500 盎司时空之力**（时空石碎片 ×10 + **[天赋觉醒之书·灭法](../../力量体系/装备道具/index.md)**，后者估值 1400 盎司）。

## 关系

- [苏晓](../苏晓.md)：想把他骗上噩梦岛单挑、反被他引来烛女猎杀、抹去其名领 1500 盎司的灭法者。
- **背叛者**：他昔日的主子——六叛徒中最强、最神秘的那一位（他只是背叛者的旧部下）。
- [怒鲨](怒鲨.md)：他在噩梦中策反、当作杀苏晓棋子、最终连怒鲨一起被烛女吞噬的航海士。
- [神秘者·黑玫瑰](神秘者·黑玫瑰.md)：写信让他去找背叛者交涉唤醒之碑的圣兰幕后女王。
- [烛女](../../世界/生物万族/index.md)：苏晓引来「替他单挑」、把他一击猎杀的虚空异存在。
