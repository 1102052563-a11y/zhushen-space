# 掌控者·席尔维斯

盘踞在联盟境外西境大沼泽「**幽魂城**」（魂鬼一族入侵本世界时所建的旧主城、被联盟 + 北境帝国收拾后沦为无法之地、如今是**黑暗神教的老巢**）地下宫殿里的存在——**黑暗神教的领袖**，又称**深渊首领·席尔维斯**。⚠ 它**上半身是健壮却病态惨白的人族躯体**（满头黑发自行飘散）、**下半身如黑泥般的粗壮蛇身**盘卧高台;黑泥身躯上会不断有一只只**环圈交叠成瞳孔的眼睛**睁开又闭合。它睁眼的模样极其诡异——不像自然睁眼，倒像被两只无形的手生硬扯开眼皮。

## 与灭法的诡异牵连

⚠ 席尔维斯给人强烈的「被操控 / 不协调」感:脸颊生硬抽搐、左右眼睁开后大小不一、上半身人类肉体与灵魂似有分离。它令一名黑暗神教主教「**去找到、告诉背叛者:他等的灭法，来了**」。⚠ 说话间它下半身黑泥上一道**已愈合的刀伤突然炸开、蓝色电弧奔涌**，它痛苦地把右臂刺进自己黑泥身躯、抓住体内一把**蓝色线丝连遍全身的长刀**的刀柄向外抽——越抽越痛、上半身都出现重影（人类肉体与灵魂分离），终究承受不住松手，长刀自行没回黑泥。它嘶吼着「**灭法！！**」震动地宫。⚠ 这把「体内蓝丝长刀」与它对灭法的执念，强烈暗示席尔维斯（乃至黑暗神教）与灭法阵营 / 斩龙闪·刃之魔灵一脉有某种未解的深层渊源——而 [苏晓](../苏晓.md) 刚把黑暗神教推上全联盟的对立面。

## 关系

- 黑暗神教：它执掌的、崇拜深渊力量、盘踞幽魂城、被苏晓嫁祸后遭全联盟痛击的教派。
- 「背叛者」（六叛徒之一）：它派主教去通知「灭法来了」的对象——猎杀名单上最强、最神秘的那名叛徒。
- [苏晓](../苏晓.md)：那名「找来的灭法」——刚把黑暗神教引成众矢之的、又要循噩梦线猎杀告密者的新一代灭法。
