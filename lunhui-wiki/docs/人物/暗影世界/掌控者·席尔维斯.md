# 掌控者·席尔维斯

盘踞在联盟境外西境大沼泽「**幽魂城**」（魂鬼一族入侵本世界时所建的旧主城、被联盟 + 北境帝国收拾后沦为无法之地、如今是**黑暗神教的老巢**）地下宫殿里的存在——**黑暗神教的领袖**，又称**深渊首领·席尔维斯**。⚠ 它**上半身是健壮却病态惨白的人族躯体**（满头黑发自行飘散）、**下半身如黑泥般的粗壮蛇身**盘卧高台;黑泥身躯上会不断有一只只**环圈交叠成瞳孔的眼睛**睁开又闭合。它睁眼的模样极其诡异——不像自然睁眼，倒像被两只无形的手生硬扯开眼皮。

## 与灭法的诡异牵连

⚠ 席尔维斯给人强烈的「被操控 / 不协调」感:脸颊生硬抽搐、左右眼睁开后大小不一、上半身人类肉体与灵魂似有分离。它令一名黑暗神教主教「**去找到、告诉背叛者:他等的灭法，来了**」。⚠ 说话间它下半身黑泥上一道**已愈合的刀伤突然炸开、蓝色电弧奔涌**，它痛苦地把右臂刺进自己黑泥身躯、抓住体内一把**蓝色线丝连遍全身的长刀**的刀柄向外抽——越抽越痛、上半身都出现重影（人类肉体与灵魂分离），终究承受不住松手，长刀自行没回黑泥。它嘶吼着「**灭法！！**」震动地宫。⚠ 这把「体内蓝丝长刀」与它对灭法的执念，强烈暗示席尔维斯（乃至黑暗神教）与灭法阵营 / 斩龙闪·刃之魔灵一脉有某种未解的深层渊源——而 [苏晓](../苏晓.md) 刚把黑暗神教推上全联盟的对立面。

## 👁 成苏晓的下一个目标（49卷38–39章）

⚠ 苏晓从 [沙之王](沙之王.md) 的灵魂记忆里，看到黑暗教堂内**沙之王、席尔维斯、背叛者**三人会面的景象，就此锁定次序:先取深渊首领·席尔维斯，再对付最终的背叛者。⚠ 主线最终环「深渊之影」揭示:背叛者正**沉眠中**，而幽魂城十有八九藏着**深渊之孔**（深渊通道的雏形），关闭它能大幅削弱背叛者——这条线索也指向幽魂城。苏晓的打法是把整件事做成**联盟 vs 幽魂城**（黑暗神教百年作恶、联盟苦无人牵头讨伐，唯身为疯人院院长、与黑暗神教天然死敌的苏晓敢带头），自己作联盟本次的代表、拿议会院的巨额酬金，光明正大地攻打席尔维斯的老巢。⚠ 苏晓评估:对战深渊首领·席尔维斯有**六成胜算**（灭法者对深渊特性强敌天然占优、不易被深渊侵蚀）；真正让他心悸的是背叛者。

## 🐍 脱去灭法之刃 · 登场（49卷43章）

⚠ 幽暗大教堂地下宫殿里，那把曾刺入它黑泥下半身、令它痛苦的**灭法之刃已然脱去**（不知是自行抽离还是有人相助），失去束缚的席尔维斯气息比先前更强大、更黑暗，人族上半身的表情也不再那般生硬诡异。它座下有**三主教 + 两长老**:主教·怨鬼（鬼魂状）、主教·黑虫·厄诺德（千疮百孔、体内蠕动黑粘虫）、主教·血妖（完美贵妇形象·正是疯人院凶犯 [女妖](女妖.md) 的**亲生母亲**·「女妖」称呼由此而来）。黑暗神教等级森严:新晋信徒 < 正式信徒 < 虔诚者 < 教士 < 灰袍主祭 < 黑袍主祭 < 主教 / 长老 < 深渊首领·席尔维斯。⚠ 它一手拍碎了擅自行动、又把要「安稳带回」的 [红瞳女·希莉德](红瞳女·希莉德.md) 吞进虫腹的黑袍主祭·豪德斯（「我说了两次，你把她吞到虫腹，就该死」），并派 [黑A](黑A.md) 代它进家族宅邸争夺战出战（以一池深渊能量为酬）。

## 🕯 惊天真相 · 太阳大主教·席尔维斯的三分之一（49卷46章）

⚠ 地下祭祀场的壁画揭开了席尔维斯的来历:当年黑暗神教凭联盟与北境的千年血战崛起、搞出一道**深渊通道**，是**太阳大主教·席尔维斯**（陨火之地石碑最上那名字）以太阳神教近乎覆灭为代价冲散了它，只剩他一人活下来、以自身囚困本世界第一只**不灭特性·深渊滋生物**。⚠ 他撑不下去时，寻得一把**灭法之刃**刺穿自身、同时把身体 / 灵魂 / 意识**一分为三**:代表**太阳**的是 [白金主教](白金主教.md)（复兴太阳神教）、代表**深渊**的是**深渊首领·席尔维斯**（插着灭法之刃继续封印深渊滋生物、当黑暗神教领袖压住教众下限）、代表**混沌**的是 **憎恨**（一道保险——防止深渊首领去找白金主教融合、放出深渊滋生物）。只有三者重新聚合，才能拔出灭法之刃、还原成太阳大主教。

## 👑 融合成深渊大主教 · 企图回深渊（49卷46章）

⚠ 有人（水哥所受托）把深渊首领体内的**灭法之刃直接拔了出来**——可他与不灭深渊滋生物早已半融、意志被深渊彻底侵蚀、渴望深渊、并因久被灭法之刃束缚而敌视灭法。脱束后他第一时间找上同源的 [白金主教](白金主教.md)（白金主教毫无还手之力、当场被深渊滋生物意识侵袭击败、**被融合**——这正是他不辞而别、去看最后一次日落的缘由），又借水哥救出**憎恨**融合。⚠ 他不再想还原成太阳大主教，而是要凑齐残缺灵魂、成为**深渊大主教·席尔维斯**，好打开祭祀场的**深渊之孔**、把它撑成深渊通道、自己回归深渊——至于本世界是否化为死寂黑暗，与他无关。

## ⚔️ 被苏晓奇袭斩杀（49卷47章）

⚠ 苏晓识破他借家族宅邸争夺战吸引全城视线、闭门不出的真意，遂**返璞归真、直接奇袭**:凯撒以【欺诈者头裹】替换幽暗大教堂后勤官、引爆一颗被增益到「太阳圣剑」气场（实则爆竹威力）的假阿波罗吓空全教堂，苏晓独闯地下、跃入深渊能量池抵达祭祀场。⚠ 深渊大主教是恐怖的**深渊召唤系**（力297敏295体300智300·黑暗印记 / 暗环 / 深渊之孔召黑暗生物 / 深渊滋生物），近战克星、险些锤死苏晓。可他有四大劣势:刚融合、能力不熟、没吞完白金主教（太阳类恢复 / 退敌手段弱一截）、不灭深渊滋生物兽性正暴躁。⚠ 苏晓靠**濒死死而后生 + 龙影闪 + 月之刃 + 布布汪催发的针对深渊生物的炼金猛毒（早埋在憎恨身上）** 险胜，最后两刀「一刀斩敌、一刀封魔」——斩杀深渊大主教、并以斩龙闪的**刃之魔灵吞噬封印了那只不灭深渊滋生物本源**。苏晓随后关闭了深渊之孔。
