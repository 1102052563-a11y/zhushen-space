# 副院长·古斯沃

黄昏疯人院两位副院长之一（另一位是 [艾琳诺](艾琳诺.md)）——一个**眼窝深陷、鹰钩鼻、作风如秃鹰**的老家伙。（⚠ 原著第 4 章称其「**古斯沃**」、第 5–6 章又称「**耶辛格**」，实为同一人·疑作者笔误，本页统一用「古斯沃」。）⚠ 他与退休的老院长的恩怨要追溯到上一任院长 [老院长（老狐狸）](老院长.md):老狐狸从竞争者中夺得院长位后，为免争位惨剧重演，培养了两名传人——即今天的**老院长**与**古斯沃**。当年古斯沃**败给老院长、忍了几十年**。

## 忍无可忍的夺权

⚠ 黄昏疯人院院长这个位置绝不简单:落到联盟大家族手中能做很多事，甚至以此为台阶登上**四大议员**之位（联盟权力之巅）。老院长身体每况愈下、按理该把位子交给古斯沃——可老院长偏偏**把它交给了一个联盟内没有权势、却实力强大者**（[苏晓](../苏晓.md) 所替代的正是这位仁兄；白夜成为新院长，实为轮回乐园的干涉）。⚠ 没人知道老院长为何这样做，但这毫不影响忍了几十年的古斯沃倾泻怒火。

## 联合晨曦神教 · 钻规则空子

⚠ 古斯沃表面是跑到圣都向大议员告状，实则是**联合了晨曦神教**——当年联盟与北境帝国承认四神教时明令「神教不得干政」，但黄昏疯人院是特殊部门、加上**晨曦神教总部在圣兰王国**（不在联盟 / 北境境内），才让他钻了空子。⚠ 老院长这等能人都连夜跑路（其住处如今是猎手部队领袖泰莎在接电话、笃定「老东西已经跑了」），足见古斯沃极难对付、这次得到晨曦神教支持后来势汹汹。这反倒给了苏晓「**被迫反击、正当联合太阳神教**」的绝佳理由——苏晓越看合照里这鹰钩鼻老家伙越顺眼，还盘算着晨曦神教高层里或许就藏着他要找的背叛者。

## 🩸 稳健狠厉的老阴哔 · 绑老院长挑衅（49卷9–10章）

⚠ 古斯沃（耶辛格）作风稳健又狠厉:面对老院长「把院长位让给外人、逼他二选一」的阳谋，他不两头开战，而是**先去「安排」跑路的老院长**、再回头收拾苏晓。⚠ 他绑了**老院长一家**，把一条被绑者的手臂 + 全家照片装进木匣、由一只只黑蜜蜂送到疯人院——赤裸裸的挑衅，也是让苏晓「失院长之位」的陷阱。⚠ 可他准备收拾苏晓时发现事情不对:苏晓竟联合猎手部队领袖 [泰莎](泰莎.md)、把地牢三层囚了多年的深渊滋生物消灭了（他深知泰莎没这手段、必是苏晓灭的还把功劳让给泰莎）——这意味着**疯人院与猎手部队达成了从未有过的合作**，对议会院是挑衅、局势陡然微妙。他的靠山「晨曦神教向联盟扩张」的图谋（总部在圣兰王国、想抢黄金神教的联盟地盘），正是老院长失议会院支持、他得以夺权的根源。

## ☠ 议会院暴毙 · 嫁祸黑暗神教（49卷12章）

⚠ 苏晓早在【厄运石像】上埋了「**切葛细胞**」——耶辛格触碰、甚至被石像砸伤手臂时便沾上了（切葛细胞本身无害、存人体十几天，一旦摄入「聂氧」就狂野裂变、把人一分钟内长成巨大烂肉球，苏晓曾用它秒杀画之世界的烈阳君主）。⚠ 议会院裁定席上，老院长带狮王 + 两名鬼帮成员作证「耶辛格勾结黑暗神教绑架老院长、想放出憎恨」（脏水泼得众人发笑）；就在这节骨眼，苏晓扣合怀表放出**聂氧**——耶辛格**当场爆成烂肉球、血肉溅满周边几米**。⚠ 泰莎笃定是苏晓做的却拿不出证据，还从其血肉里「抽出」一条黑色小虫、佐证「像黑暗神教的手段」；老院长高呼「黑暗神教杀人灭口」，黄金神教代表顺势鼓噪（暗谢黑暗神教替它挨打）——大议员遂下令**痛击黑暗神教**，黄金神教 / 猎手部队 / 议会院武力 / 疯人院乃至太阳、晨曦神教全体出动。这位忍了几十年、以为能从「战力强谋略一般」的苏晓手中夺位的老阴哔，就此稀里糊涂地成了苏晓政治大棋里的一枚弃子。

## 关系

- [老院长（老狐狸）](老院长.md)：培养他与老院长的前前任院长——古斯沃与老院长数十年的竞争皆源于此。
- 退休老院长：他忍了几十年、却把院长位交给外人的死对手——他先绑其一家「安排」了这老家伙。
- [苏晓](../苏晓.md)：顶替「实力强大者」当上院长、被他绑老院长挑衅、也「被迫」联太阳神教反击的新院长。
- 晨曦神教：他钻「总部在圣兰王国」的空子联合来对付苏晓、图谋向联盟扩张的神教（苏晓疑其高层藏着背叛者）。
