# 艾丽莎

联盟·猎手部队领袖 [泰莎](泰莎.md) 的**妹妹**——摩诺家族出身，从小活在姐姐的光环下（姐姐登上联盟最强个体战力、与北境大将军齐名，而她的天分「如萤火与繁星」），逐渐养成叛逆、独立又不服输的性子，做梦都想超越姐姐。⚠ 她其实是五吞噬者之一 **沸红** 的宿主——「命运中的伙伴」沸红共生进了她的心脏（她起初以为自己被邪灵入侵，还给了自己脑袋一拳）。上身运动装配超短裤 + 微笑脸塑料胸针的混搭穿法，别有一番她自己的美感。

## 🗡 与黑A 的死磕 · 提刀追杀

⚠ 沸红前期发育最快、是唯一能吞噬其他吞噬者的一支，最忌惮的对手正是初代吞噬者 [黑A](黑A.md)。艾丽莎的日常一度是:上午打不过黑A、心中憋闷，傍晚又打不过、更郁闷，晚上越想越气、提着刀袋出门，把黑A 一路追到郊外才心满意足回家睡觉。她提刀追杀黑A 的莽劲，注定与「怕生 / 害羞」无缘。

## 🎁 苏晓的「亲大爹」扶植 · 代联盟出战（49卷43–45章）

⚠ 吞噬者争夺战拓展公证后，苏晓选沸红作联盟阵营的核心，把艾丽莎请来幽魂城。沸红叮嘱她「这是亲大爹，不用客气，只管要」——苏晓当即把冥想 / 感知、刀术开发 / 对敌、十几名刀术宗师晚年之作三大堆刀术心得**全送给她**，又给她一打**五次改良·完美**级的永久增益药剂（上古魔剂 / 黎明之焰 / 圣龙守护 / 树之生命…摩诺家族出身的她都没见过这阵容），把她激动到「有种偶遇大爹的感觉」。⚠ 她因实力暴涨略有膨胀（偷偷溜出去砍了名黑暗神教教士），被苏晓授意 [阿姆](../阿姆.md) 一顿暴揍敲打——她那套「刮痧刀法」连阿姆的皮都没破（阿姆伤口飘白气秒愈），当即认清现实。⚠ 争夺战开幕，她**一个人就是联盟阵营一整队**，代苏晓进家族宅邸；苏晓嘱她进庭院后对付鬼族队。她与黑A（黑暗队）、太阳使徒（潜商盟队）、犹格·迪娅（犹格队）、鬼族队五方齐入迷雾。

## 🏰 家族宅邸建功 · 斩女伯爵（49卷48–49章）

⚠ 争夺战里艾丽莎全程与各路怪物、敌方小队死战:第一阶段**斩杀宅邸主人·女伯爵**、拿到地宫钥匙（苏晓作胜出阵营领袖选了 2 盎司深渊沉淀物）;地宫里她与 [黑A](黑A.md) 合斩大 boss**家族始祖**，却被半路杀出的太阳使徒（潜商盟队）抢走古老纹章。⚠ 她没得古老纹章，却从地宫最里带出一个装着 7 盎司**深渊沉淀物**的黑色陶罐（一路上所有怪物都为这罐子追杀她）。⚠ 她姐姐 [泰莎](泰莎.md) 事后对苏晓一记完美裸绞、质问「你为什么选我妹进噩梦古宅」——见了那一长串完美药剂 + 宗师刀术秘传的补给清单，泰莎既心疼又想给自己也白嫖一套（被苏晓无视），最终还是没再溺爱庇护妹妹（「不经历外界凶险，永远成不了强者」）。

## 关系

- [泰莎](泰莎.md)：她一直想超越、却觉遥不可及的姐姐——联盟最强个体战力（近来发觉妹妹「叫姐姐了」而起疑、事后裸绞质问苏晓为何选妹妹参战）。
- **沸红**：共生进她心脏、发育最快、唯一能吞其他吞噬者的初代级吞噬者伙伴。
- [苏晓](../苏晓.md)：沸红口中的「亲大爹」——送她刀术心得 + 一堆完美药剂、又让阿姆揍她敲打的创造者。
- [黑A](黑A.md)：她最忌惮、日常提刀追到郊外的「兄长」吞噬者。
- [阿姆](../阿姆.md)：奉苏晓命把她一顿暴揍、让她认清「刮痧刀法」现实的凛冬牛头人。
