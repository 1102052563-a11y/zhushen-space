# 希女王（天启乐园）

**[天启乐园](../世界/任务世界/维斯星球-多亚古大陆.md)**（与轮回乐园争夺世界的对手乐园·疑同出一源）一方的统御者——**满头银色长发、琥珀色眼瞳的绝色美女**。天启乐园的契约者发色 / 瞳色都很特殊（绿发紫瞳、白发红瞳等），与地球人迥异。

- **登场威慑（十二卷2章）**：星空中转站双方骂战正酣时，她一声「退下」走出人群，**天启乐园所有人竟齐刷刷退后一步**——这让 [苏晓](苏晓.md) 心头一凛：己方（轮回乐园）无人有此威慑，意味着**敌方千余人很可能同属一个组织（教团）**、组织度远胜己方这群「乌合之众」。她扫视轮回乐园众人、轻蔑一笑「运气不错，遇到一群乌合之众」；对己方法爷的叫板只回一句「你？不够资格」。其部下扬言「想和我们希女王战斗、至少出动一个教团」。

## 关系
- **[天启乐园](../世界/任务世界/维斯星球-多亚古大陆.md)**：教团首领（统御千余契约者）。
- **[苏晓](苏晓.md)**：二阶世界争夺战的敌方核心（尚未交手）。
