# 希女王（天启乐园）

**[天启乐园](../世界/任务世界/维斯星球-多亚古大陆.md)**（与轮回乐园争夺世界的对手乐园·疑同出一源）一方的统御者——**满头银色长发、琥珀色眼瞳的绝色美女**。天启乐园的契约者发色 / 瞳色都很特殊（绿发紫瞳、白发红瞳等），与地球人迥异。

- **登场威慑（十二卷2章）**：星空中转站双方骂战正酣时，她一声「退下」走出人群，**天启乐园所有人竟齐刷刷退后一步**——这让 [苏晓](苏晓.md) 心头一凛：己方（轮回乐园）无人有此威慑，意味着**敌方千余人很可能同属一个组织（教团）**、组织度远胜己方这群「乌合之众」。她扫视轮回乐园众人、轻蔑一笑「运气不错，遇到一群乌合之众」；对己方法爷的叫板只回一句「你？不够资格」。其部下扬言「想和我们希女王战斗、至少出动一个教团」。

## 能力 · 恐怖的智商（十二卷31章）

她操控**透明水晶**——身旁常漂浮数颗，随其情绪变色（不悦转红、冷静转蓝、无波动则无色），能以水晶**封冻成片尸堆**。但她最骇人的是**头脑**——

- **识破兰爵**：[兰爵](兰爵.md)（被征调入她团队）反常地一直死盯「世界之核」去向、为套情报甚至听命于她，却忽为一颗红水晶不惜翻脸赶去黑土战区。她由此**推断兰爵与所有独行者都不同——总在主动寻找「能力异常」的契约者并杀掉**（乐园常传他无故与异常契约者结怨）；这与苏晓的特征重合，让她警觉自己已触到「**某种不可触碰的禁忌的危险边缘**」（疑指向猎杀者 / 灭法之影一脉的隐秘·选择「最好不去深究」）。
- **冷静的布局**：明知妹妹 [琉璃](银发少女.md) 私下把红水晶给了兰爵换其出手杀苏晓，她非但不罚、反顺势促成（「兰爵败也好、苏晓败也罢，双方都不是省油的灯、对我们都是双赢」），又另派琉璃去做「她最擅长的事」（埋伏 / 暗算）。对妹妹「认错态度极好、就是不改」也只能无奈纵容。
- **「九条命」 = 多次重生（十二卷46–47章）**：她的爱猫 [贝斯妮](贝斯妮.md) 有「九条命」、且**与她灵魂绑定、可把命共享给她**——苏晓据此推断**希本人能死后复活 8 次、第 9 次才真死**（贝斯妮被苏晓杀一次、希便少一次重生机会·贝斯妮则化作小奶猫重生）。这是她最棘手的底牌。
- **「标号」与全局规划（十二卷47章）**：她按敌人特点 + 威胁度给轮回乐园方契约者编号——苏晓 =「**近战 1 号**」（威胁度第一）、火系法爷 =「**法师 1 号**」、希尔大峡谷自称「幸运女神」的女契约者亦在列。她虽坐镇希尔大峡谷却对三大战区了如指掌，已周密布局：派心腹「**凛冬**」去对付法师 1 号，又付出大量高级恢复药剂雇人（[兰爵](兰爵.md) / [琉璃](银发少女.md)）对付近战 1 号、不行则亲自出手。她警觉**轮回乐园的契约者「十个里至少三个是疯子」**（培养手段极端·疑任务惩罚是「死亡」·绝境会同归于尽），告诫部下别轻敌。

- **「兽人炮灰」阳谋（十二卷59–61章）**：契约者大乱斗开战，她（全员战功转让·官职够调兵）先**驱一万多名兽人士兵打头阵当炮灰**——明知会被几百契约者杀光，只为让兽人**替己方契约者挡 90% 远程攻击**、减天启死伤。这是不怕识破的阳谋（炎辰看破也无解）。她还操控水晶护盾挡狙、本人坐镇森林不轻易露面（怕被集火）。

- **坐镇前线 · 折损凛冬（十二卷67–68章）**：她身处最前线杀敌（透明水晶网络一触轮回乐园契约者便层层判定、失败即晶体化「碎」），却被独臂团长 + 死了 168 次的 [黑血](黑血.md) 死死拖住（她反倒欣赏这群「不惧死」的疯子·自认骨子里也有一颗不惧死亡的心、这才是她与其他天启契约者的不同）。她派去克制 [炎辰](炎辰.md) 的冰系刺客 **[凛冬](凛冬.md)** 反被炎辰真身（炎魔枪骑士）5 分钟秒杀。同时她察觉后方有刺客连杀己方治疗 / 辅助类契约者（屡设真实侦测护卫仍挡不住）、派妹妹 [琉璃](银发少女.md) 去解决——却不知那刺客正是头号目标 [苏晓](苏晓.md)。
