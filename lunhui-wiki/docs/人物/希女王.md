# 希女王（天启乐园）

**[天启乐园](../世界/任务世界/维斯星球-多亚古大陆.md)**（与轮回乐园争夺世界的对手乐园·疑同出一源）一方的统御者——**满头银色长发、琥珀色眼瞳的绝色美女**。天启乐园的契约者发色 / 瞳色都很特殊（绿发紫瞳、白发红瞳等），与地球人迥异。

- **登场威慑（十二卷2章）**：星空中转站双方骂战正酣时，她一声「退下」走出人群，**天启乐园所有人竟齐刷刷退后一步**——这让 [苏晓](苏晓.md) 心头一凛：己方（轮回乐园）无人有此威慑，意味着**敌方千余人很可能同属一个组织（教团）**、组织度远胜己方这群「乌合之众」。她扫视轮回乐园众人、轻蔑一笑「运气不错，遇到一群乌合之众」；对己方法爷的叫板只回一句「你？不够资格」。其部下扬言「想和我们希女王战斗、至少出动一个教团」。

## 能力 · 恐怖的智商（十二卷31章）

她操控**透明水晶**——身旁常漂浮数颗，随其情绪变色（不悦转红、冷静转蓝、无波动则无色），能以水晶**封冻成片尸堆**。但她最骇人的是**头脑**——

- **识破兰爵**：[兰爵](兰爵.md)（被征调入她团队）反常地一直死盯「世界之核」去向、为套情报甚至听命于她，却忽为一颗红水晶不惜翻脸赶去黑土战区。她由此**推断兰爵与所有独行者都不同——总在主动寻找「能力异常」的契约者并杀掉**（乐园常传他无故与异常契约者结怨）；这与苏晓的特征重合，让她警觉自己已触到「**某种不可触碰的禁忌的危险边缘**」（疑指向猎杀者 / 灭法之影一脉的隐秘·选择「最好不去深究」）。
- **冷静的布局**：明知妹妹 [琉璃](银发少女.md) 私下把红水晶给了兰爵换其出手杀苏晓，她非但不罚、反顺势促成（「兰爵败也好、苏晓败也罢，双方都不是省油的灯、对我们都是双赢」），又另派琉璃去做「她最擅长的事」（埋伏 / 暗算）。对妹妹「认错态度极好、就是不改」也只能无奈纵容。
- **「九条命」 = 多次重生（十二卷46–47章）**：她的爱猫 [贝斯妮](贝斯妮.md) 有「九条命」、且**与她灵魂绑定、可把命共享给她**——苏晓据此推断**希本人能死后复活 8 次、第 9 次才真死**（贝斯妮被苏晓杀一次、希便少一次重生机会·贝斯妮则化作小奶猫重生）。这是她最棘手的底牌。
- **「标号」与全局规划（十二卷47章）**：她按敌人特点 + 威胁度给轮回乐园方契约者编号——苏晓 =「**近战 1 号**」（威胁度第一）、火系法爷 =「**法师 1 号**」、希尔大峡谷自称「幸运女神」的女契约者亦在列。她虽坐镇希尔大峡谷却对三大战区了如指掌，已周密布局：派心腹「**凛冬**」去对付法师 1 号，又付出大量高级恢复药剂雇人（[兰爵](兰爵.md) / [琉璃](银发少女.md)）对付近战 1 号、不行则亲自出手。她警觉**轮回乐园的契约者「十个里至少三个是疯子」**（培养手段极端·疑任务惩罚是「死亡」·绝境会同归于尽），告诫部下别轻敌。

- **「兽人炮灰」阳谋（十二卷59–61章）**：契约者大乱斗开战，她（全员战功转让·官职够调兵）先**驱一万多名兽人士兵打头阵当炮灰**——明知会被几百契约者杀光，只为让兽人**替己方契约者挡 90% 远程攻击**、减天启死伤。这是不怕识破的阳谋（炎辰看破也无解）。她还操控水晶护盾挡狙、本人坐镇森林不轻易露面（怕被集火）。

- **坐镇前线 · 折损凛冬（十二卷67–68章）**：她身处最前线杀敌（透明水晶网络一触轮回乐园契约者便层层判定、失败即晶体化「碎」），却被独臂团长 + 死了 168 次的 [黑血](黑血.md) 死死拖住（她反倒欣赏这群「不惧死」的疯子·自认骨子里也有一颗不惧死亡的心、这才是她与其他天启契约者的不同）。她派去克制 [炎辰](炎辰.md) 的冰系刺客 **[凛冬](凛冬.md)** 反被炎辰真身（炎魔枪骑士）5 分钟秒杀。同时她察觉后方有刺客连杀己方治疗 / 辅助类契约者（屡设真实侦测护卫仍挡不住）、派妹妹 [琉璃](银发少女.md) 去解决——却不知那刺客正是头号目标 [苏晓](苏晓.md)。

- **vs 炎辰 · 复活与「死亡水晶」杀招（十二卷71–73章）**：[炎辰](炎辰.md) 的火系魔枪温度超过水晶熔点、能力克制她，半小时内**杀了她一次**——但她**原地复活**（复活瞬间现猫的虚影〔[贝斯妮](贝斯妮.md) 的九命〕、状态恢复 80%+）。她最骇人的杀招也由此暴露：**平常的水晶化只要抗性够就能避免，唯独她「死亡那一瞬会完全水晶化并爆开」，被溅中者无需判定、持续晶体化恶化**（炎辰正因杀她一次被反噬成重伤垂死）。她没追退守石桥的炎辰、只留一句「或许这次见面就是永别」。
- **痛失琉璃（十二卷71章）**：妹妹 [琉璃](银发少女.md) 死于苏晓之手。身为领袖她当众不露悲色、独自躲到石柱后垂泪——「琉璃走好……对付疯子，就要用更疯狂的手段」。值得注意：她转身离去时，**身旁漂浮的水晶里浮现出淡淡的黑色**（伏笔）。一轮混战后双方死伤：轮回乐园 658 存活 / 137 阵亡、天启 783 存活 / 202 阵亡——天启虽占人数 + 治疗优势却死伤更重，她忧心的是手下死太多会有人放弃任务。

## 能力本质 · 晶能（十二卷74章揭露）

她的水晶来自「**晶能**」——由法力转化的**无活性能量**（1 法力 → 10 晶能·放出体外即凝成水晶）。无活性能量胜在**极其稳定、不会暴走**，正合她「讨厌一切不可控之物」的性格。她晶能储量惊人，能凭此完成下面的「超大型水晶球」杀招。

## 千米水晶球 · 全灭 403 人（十二卷74章·全局阳谋的终章）

她从一开始就借古遗迹地形布下这盘大棋：兽人炮灰、迷宫死战、乃至重伤 [炎辰](炎辰.md)（对战时**故意让炎辰杀她一次**），全是为最终一击铺垫——只有炎辰重伤、轮回乐园方失去主心骨，她的杀招才敢实行。轮回乐园方被赶出二区、密集挤在三石桥 + 黄昏宫殿一带后，她**连死 3 次、抽干自身 3 条命 + 几十名法系契约者的法力**，催出**直径千米的实心水晶球**把数百契约者像琥珀里的昆虫般同化——**一举灭掉 403 人**（镇守石桥者几乎死绝）。

- **代价与后续**：此役轮回乐园方仅剩 **255 人**、与天启 **783 人**拉成三倍差距（往后将是三打一）；但她的复活只剩 **5 条命**（已死 4 次：炎辰处 1 + 水晶球 3），且这种不顾己方死伤的狠招让她**威望大减**。她也算漏了一点——被水晶球灭掉的多是「精神还算正常、时刻警戒」的契约者，**活下来的几乎全是疯子**（大敌当前还在宫殿里睡觉的那种），反被逼成了最难缠的对手。

## 殁 · 太阳火地狱耗尽九命（十二卷77–79章）

中了苏晓与炎辰的「大菠萝计划」——黑乌鸦 / 冒牌小丑潜入诱敌、空城逐层深入，最终被苏晓（堵四层通五层）与炎辰（金色传送阵堵四层通三层）困进黄昏宫殿四层密室。苏晓引爆预埋的**太阳神「阿波罗」**（密闭环境发挥 500% 威力·6000 真实灼烧 + 2000 后续·三、四阶硬抗都死）——她当场被气化（连同 307 名天启契约者）。两人的真正目的是**让她在四层火焰地狱里死满九次**：

- **火焰地狱里的反复横死**：四层 2000 度高温 + 岩浆 + 无氧 + 毒气 + 辐射，她每次复活活不过 2 秒（「晶能控制 Lv.39」被高温压制 -70%、「水晶环绕」失效），刚破茧就再被烧死 / 被苏晓抛来的第二颗阿波罗（膨胀成百米火球）炸死。她临终回想入世时那句「运气不错、遇到一群乌合之众」，无奈苦笑「**倒霉，遇到一群疯子**」。
- **最后一搏失败**：最后一命她开终极能力「**水晶源 Lv.10**」（锁定身体状态·免疫除真实伤害外一切伤害·持续 5 秒）+ 吞「晶女王」药瓶冲上五层楼梯——却被苏晓的**青钢影反击盾墙**挡住（晶能同化不了「能量杀手」青钢影）、一脚把盾墙踢飞撞退她；水晶源时效一过高温复侵。她**宁可被烧死也不死在苏晓刀下**，单膝跪进岩浆引火自焚。
- **被未知空间波动卷走 = 死**：将焚成灰时一股**未知空间波动**带走她半焦的残躯（苏晓得 5 功勋）。苏晓疑天启作弊保她、举报至 [虚空之树](../世界/任务世界/维斯星球-多亚古大陆.md)——公证结果是那波动**与天启乐园相差 98.73%、是「未知空间之力入侵」**（非天启偏袒），虚空之树裁定其「对轮回乐园造成战争阻碍」、把己方「大型空间定点坐标」形成时长**砍半**（再四天多即可夺下世界）。希就算万一不死也已离世、且受轮回 + 天启两方通缉——苏晓据此确认「原来她只能复活四次、不是八次」。她至死也没能如愿冲上五层与苏晓「正大光明打一场」。
