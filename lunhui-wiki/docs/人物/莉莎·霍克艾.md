# 莉莎·霍克艾（鹰眼）

[罗伊·马斯坦](罗伊·马斯坦.md) 上校的**副官**·金发女狙击手。**天生异禀、视力为常人 8 倍**（故称「鹰眼」），参与过伊修瓦尔歼灭战、狙杀过几百人、心理素质超过一些契约者；其父是马斯坦的师傅、两人关系非同寻常。

- **历程**：[恩维](恩维.md) 伪造「莉莎投敌」照片、委托苏晓杀她，意在逼苏晓与马斯坦结死敌。苏晓**将计就计**——夜闯她家不杀反而亮出国家秘密、与她结盟，再导演一场以假乱真的「杀莉莎」骗过人造人，最后放出活着的莉莎引来马斯坦、促成结盟（七卷16–17章）。开战时由她「鹰眼」负责狙杀军方高级将领。
- 养有幼年军犬「**疾风号**」（黑白花色小狗·曾对潜入的苏晓示警护主）。
- **最终决战**：七卷44章苏晓收取神之力后被识破，**反手用界断线把她从洞口拽下、挟为肉盾**全身而退（众人忌惮误伤她而不敢攻击·失明的马斯坦怒吼「爆炎炼金术师·白夜！」）；脱身时苏晓从她身上扣下一枚可重复使用的跟踪器，她则狠咬了苏晓手背一口（留下一排牙印·随苏晓被传送回乐园）。
