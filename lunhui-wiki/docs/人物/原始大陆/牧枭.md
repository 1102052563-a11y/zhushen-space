# 牧枭（圣光乐园·原罪·刑镰·被误认追猎者的战力担当）

赤脚踩金属丝、袖口弹出尖细臂刃的女契约者——**圣光乐园**七阶强者，战力与美食家 / 坦克男 / 加斯克同级。人人都以为她这实力必是追猎者，**她偏偏是生存者**（这点验证后连 [幻术师](幻术师.md) 都差点笑出声）。

- **战力担当**：她口才差（三句话就冷场、故拒当临时首领），却看出幻术师不自私、甘当他的战力担当。核心能力「**原罪·刑镰**」——割掌放血凝成血色巨镰，一镰「嫉妒罪业」加深伤害、重创雌雄双猿的白猿。她曾在湖边混战夺下一枚白色物资箱。
- 与 [苏晓](../苏晓.md) 有旧怨（「被他信任可是会死的·我不想和他有一点关系」），双猿战时被幻术师派去暂时打退苏晓。
- **四修原罪使者 · 灰绅士线索（三十七卷22–23章）**：她是**力 / 敏 / 智 / 魅四修的「原罪使者」**——**原罪·暴食**（暴食君主可「吃掉死亡 / 伤痛」续命·被苏晓四颗阿波罗炸到剥离烙印时靠它捡回一命、暴食君主随之永灭）、**原罪·欲拥 / 欲血舞**（血雾虚影 / 瞬杀）、敏 + 魅衍生「血之舞」（叠加真实流血）。逃命属七阶顶尖（灵魂结晶保命术 + 血丝瞬移）——被苏晓刃道刀·流（风痕）斩掉 70% 生命仍逃脱，留血字点破**幻术师是灰绅士人偶**：她真身是圣光乐园类似猎杀者的存在、这次是**借幻术师循线追灰绅士**（故被设为生存者），选择继续留在世界里（却被「国足三兄弟」缠上「铲除邪恶」）。

- **原罪·长眠 · 配合诈死（三十七卷33章）**：被 [纳吉·加](纳吉·加.md) 拉来联手时，用斩杀能力「**原罪·长眠**」（深红侵蚀·「不死的美食家太傲慢·傲慢也是原罪」）配合 [美食家](美食家.md) 演了一场诈死——她连日苦难（先怕灰绅士人偶、又被苏晓爆锤、还被国足三兄弟缠上）总算出了口气。
- **借战争弥补损失 · 回归（三十七卷59章）**：她真正的目的是**借这场大战弥补损失**——她曾被「圣光乐园处刑者」一类的大人物盯上、为抹除那「追踪印记」付出极大代价。战争尾声她果断回归（临走还调侃自己与苏晓「有一腿」逗纳吉·加·随即改口永别）。

## 关系

- [苏晓](../苏晓.md)：避之唯恐不及的旧识。
- [幻术师](幻术师.md)：她力挺、甘当战力的凝聚者。

## 苍龙大陆 · 被苏晓招募当间谍（三十九卷44–46章）

在 [苍龙大陆](../../世界/任务世界/苍龙大陆-背叛者之末路.md) 里，牧枭化名「**派翠西亚**」加入多因帝国、成圣徽骑士团医师长 [奥芙](../苍龙大陆/奥芙.md) 的心腹（负责为圣徽后方押运珍贵物资、借拦截战捞战功）。她进营帐撞见伪装成混沌军团大领主的 [苏晓](../苏晓.md)、想装不认识溜走被 [巴哈](../剧毒魔鹰·巴哈.md) 叫住——苏晓没揭穿（她在原始大陆只被他阿波罗炸过一次·无深仇），反而**招募她当间谍**：给她定点远程通讯石盘、以「未知大贵族赏识」为名把她调回**多因王都**（去盯苏晓在黑熔城放走的中毒大贵族、给药缓解其炼金剧毒当威慑）。牧枭虽觉不靠谱、却「这种机会不赌一下不如去挖矿」——她认定 [苏晓](../苏晓.md)「强到离谱、还是轮回乐园的老阴哔」，决定绝不招惹他。
