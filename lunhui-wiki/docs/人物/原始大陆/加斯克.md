# 加斯克（死亡乐园·狂兽·完全兽化的不死怪·追猎者）

会突然多长手臂、在人形 / 怪兽形间快速转换的**死亡乐园**契约者——「**狂兽·加斯克**」，原始大陆公认最强五人之一（[苏晓](../苏晓.md) / 美食家 / 加斯克 / 坦克男 / 神父）。完全兽化后力量碾压全场、物资乱战一人夺 3 箱。

- **狂化成不死怪**：生命之龙精魄乱战里他吞下精魄、又被老对手**神父**下的「**解离型活力萃取物**」（能无限刺激细胞再生、直到生命枯竭）催化——狂化成六米高、金属钩爪、不死般（半身被轰碎 0.5 秒再生）的灰皮怪物，反手就去揍下毒的神父。他是货真价实的莽夫（比莽第一）却也理智（第三天没到决战日就撤）。

## 关系

- [苏晓](../苏晓.md)：抢精魄时把它抛给苏晓坑人、反被吞走 4/5 能量。
- 神父（圣域）：给他下解离型萃取物、被他狂化后反揍的老对手。

## 出处

- 三十七卷 24–26 章：物资乱战 → 吞精魄 → 神父下药狂化成不死怪。
