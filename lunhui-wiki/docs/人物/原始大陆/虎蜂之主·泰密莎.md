# 虎蜂之主·泰密莎（原始大陆两大霸主之一·虎蜂阵营金字塔顶端）

原始大陆「真正世界」两大势力之一 **虎蜂阵营** 的**霸主生物、金字塔顶端**——住在一座 200 米高、已不像蜂巢更像黑石建筑的主巢里。她之下是三位蜂女王（[阿芙拉](蜂女王.md) / 卡萝 / 另一位）、再之下是虎蜂族。进化远比 [阿芙拉](蜂女王.md) 完美、能掌控古精灵的强大能力。

- **全面开战**：[苏晓](../苏晓.md) 靠 [棘拉](../虫族战场配角.md) 的虫族军 5 分钟踏平她一座边缘蜂巢——虎蜂阵营太久没损失过蜂巢，泰密莎被从沉睡中唤醒、与两位蜂女王一起驱虎眼蜂强行进化补员（几百万虎蜂族），亲自化一只精致虎眼蜂来见苏晓、不问缘由**直接宣战**（「开战吧，人类」）。
- **手下心腹**：蜂女王·**卡萝**、**佩内洛普**（两心腹之一·卡萝亲临前线、收编叛徒冰爵）。
- **亲临战场 · 仍败退（三十七卷46–50章）**：她金黑外甲如修身铠、上半脸覆薄甲——**徒手把巴哈扔下的阿波罗压缩成核桃大捏爆**；用古精灵秘术「**金色印记**」（吸够灵魂能量激活·剧痛给全族 buff）+ **火焰王子**（20 米喷火柱巨兽）+ **生物巨炮**（一炮吞 1700 虎蜂族）三重杀手锏猛攻。可苏晓的**七阶虫巢**爆出七阶恶魔兽 + 恶魔焰龙 + 首领级个体，鏖战 10 小时她仍不得不退。**「不能共存，只能战争」**——她转去迪亚部落找「圣巫」联合、共抗虫族。

- **八阶道具连击 · 被炸杀（殁 · 三十七卷63–64章）**：她躲进几千年的古精灵遗殿想跟苏晓决一死战，却成了「阴沟翻船」的反面教材——苏晓明明比她弱一筹，却掏出**一整套八阶道具连击**把她按死：深渊凝望（重力压制到连她这徒手捏爆阿波罗的强者都动弹不得）、轻语的阿特柔斯（削防）、恶运缠身 + 运势剥离（幸运压到 −45、遭世界排斥·全速突袭反被古精灵机关「罪刑审判」的穿刺钉扎中）、虚弱恩赐、极冰射线（叠满减益后一发 3.6 万冰伤）——最后用「天堂之门」地雷炸得只剩一小块面甲。她至死用左手按住裂开的面甲、不愿被人看见容貌。杀她得世界之源 15.7% + 霸主宝箱（开出 [女王之傲](../../力量体系/装备道具.md)）。
