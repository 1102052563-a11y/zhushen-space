# 阿西瓦（战争天堂·女技法型·20人追猎小队队长）

戴黑手套、爆发力十足、目光有侵略性的短发女人——名字在某低阶语里是「**战争天堂**」，本身是**技法型**、最擅长与契约者对战。她率一支 20 人追猎小队去猎杀 [苏晓](../苏晓.md)，布阵老练（先控坦系领域、摔死坦当盾、叫控制系别秀存在感）。

- **追猎翻车**：她指挥严密，可苏晓的战斗经验 / 直觉 / 感知都强于她——冰爵冲阵即被界断线锁死、一刀多重伤害打崩，她亲自用「瞬时·命锁」缠住苏晓却被左手放逐刺穿侧腰。20 人小队被打崩、只逃回 6 人。她果断断后、把苏晓引向 [纳吉·加](纳吉·加.md) 的祭坛（她早知苏晓最强、故意瞒着、让群体只当他排第四）。事后被纳吉·加勒令「处理掉」——**被最信任的旧识冰爵偷袭 + 领域压制、惨死于瘟虫手（殁）**；冰爵自己则签了近乎卖身契的契约、转投蜂女王·卡萝当舔狗（傲骨十足的阿西瓦为何与他为友，成了谜）。

## 关系

- [苏晓](../苏晓.md)：追猎他反被打崩的技法型队长。
- [纳吉·加](纳吉·加.md)：她畏惧却不得不投靠、最终要她命的领袖。

## 出处

- 三十七卷 29–34 章：20 人追猎小队→被苏晓打崩→引狼入祭坛→被纳吉·加清算。
