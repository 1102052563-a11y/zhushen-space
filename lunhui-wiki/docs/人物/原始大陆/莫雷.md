# 莫雷（战斗天使 · 天启乐园 · 炎灵星死里逃生 · 极怕苏晓）

**天启乐园**的战斗天使——[苏晓](../苏晓.md) 的炎灵星旧识。在上个世界进度 [炎灵星](../../世界/任务世界/炎灵星-沙皇帝.md) 差点被违规者**灰绅士**掏心做傀儡（127 种异常缠身），全靠她几个世界前花 150 枚天使圣徽买的「异常触发营救权限」被天启救回——从此对**灰绅士**和**苏晓**这两个「怪物」怕到死也不想再遇。

- **默契互不干涉**：中转站被苏晓一手搭肩威慑、瞬间头皮发麻——她何等聪明，当即表态「炎灵星没机会合作、这世界更没机会」（= 我不透露你情报作筹码、你也别来找我），苏晓「这样最好」。她与「包子脸小女孩」在中转站装死敌、其实是**好友**（一天启一轮回·演戏给旁人看）。
- **怕苏晓怕到发抖**：她与包子脸、「学姐」组队去偷蜂蜜时撞见苏晓又在拉拢土著（虎蜂阵营），瞳孔发抖——回忆起炎灵星苏晓率几十万兽骑兵、黑龙喷黑焰踏平十几万人要塞的炼狱景象，只想找地方躲起来，「和这怪物进同一个世界没好事」。

- **树之竞技场再遇秒投降（三十九卷92章）**：苏晓返园在 [树之竞技场](../../世界/地点/index.md) 车轮战冲名次，杀到前百名内时匹配到莫雷（战斗天使·树之竞技场排名靠前）。她出场逼格满满地凝出赤红水滴长枪，一看是苏晓当即僵住、本能地把枪藏到身后、脸有些发烫——「被苏晓与灰绅士打出阴影的她」心里发虚，但作为战斗天使「投降是不可能投降的」。结果一番刀光弹雨后，被青钢影能量侵体「实在太疼」，还是选择了投降。

- **魔灵星世界争夺战 · 故意放苏晓进古都（四十卷21–24章）**：她作为天启乐园方参战 [魔灵星](../../世界/任务世界/魔灵星-世界争夺战.md) 世界争夺战（她道破本轮参战七阶契约者「树之竞技场前五十名有一半以上都在这个世界」）。天启与死亡乐园临时联手在冈卢古都外围埋伏、围杀来会合的己方契约者时，莫雷亲眼看着 [魔女](../魔灵星/魔女.md) 把世界之核交给苏晓，却**故意力排众议放苏晓进古都**——她不敢向队友挑明「敌人有多强」（怕他们跑路），只用「先让轮回乐园布坐标当炮灰、封禁解除后再合作」「大不了挖矿回本」的说辞糊弄过去，把队友们哄住（她上次被苏晓打断腿的经历太难忘）。

## 关系

- [苏晓](../苏晓.md)：炎灵星见识过其恐怖、达成互不透露的默契；三十九卷树之竞技场再遇投降；四十卷魔灵星故意放他进古都（不敢正面为敌）。
- [魔女](../魔灵星/魔女.md)：魔灵星里把世界之核交给苏晓的轮回乐园己方（被莫雷目睹）。
- 灰绅士 / 绯世：炎灵星「神仙打架」里让她怀疑人生的怪物们。

- 四十卷21–24章：魔灵星世界争夺战·天启方参战·故意放苏晓进冈卢古都·哄住队友。
