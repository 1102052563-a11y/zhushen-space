# 蜂女王·阿芙拉（虎蜂区·又怂又倔的究极家里蹲·三蜂女王之一）

金黑甲壳、金色瞳孔、身高约 1 米 6、脱去甲壳几乎与人族无异的存在——盘踞在五层楼高超巨型蜂巢中心蜂茧里的**蜂女王·阿芙拉**（精灵语「守望之人」）。她**并非霸主生物、只是有成为霸主的资质**（脆弱到呼吸正常空气都要适应）——真正的虎蜂区霸主是「**虎蜂之主·泰蜜莎**」。

- **究极家里蹲**：她是**三位蜂女王之一（中立）**，却因又怂又倔、毫无野心（唯一愿望是搜罗古精灵书慢慢解读），被霸主泰蜜莎**半放逐**出来——她反而乐得无限制宅在蜂茧里读书。所谓「残暴女王」全是 [雨喉部落](../原始大陆群像.md) 偷蜂产出被蜇死后单方面脑补的黑锅。
- **耿直的交易**：[苏晓](../苏晓.md) 带猿心 + [阿波罗](../../力量体系/装备道具/index.md) 上门，她耿直到把**全部** [自然恩赐](../../力量体系/装备道具/index.md)（2231 克）都拿出来换、还倒贴虎蜂乳 / 刺蜂毒 / 黑毒蜂巢（可扔雨喉）；但死活不肯让虎蜂帮忙战斗、也死活不给古精灵书（连吃猿心的「完整」愿望都只是想安心当家里蹲）。触发支线**「阿芙拉的愿望」**（换黑鳞得一本古精灵书）、苏晓由此入**虎蜂阵营**。

## 关系

- [苏晓](../苏晓.md)：用猿心 + 黑鳞换她物资 / 古精灵书的交易对象（他判定操控她比杀她还难十几倍）。
- 虎蜂之主·泰蜜莎：把她半放逐、让她畏惧到发抖的真霸主。

## 出处

- 三十七卷 4–18 章：布布汪探巢→猿心交易→虎蜂阵营 / 阿芙拉的愿望。
