# 圣巫·古芜（迪亚部落权力顶端·能与古老存在沟通）

不生寸发、纹满暗金色纹路、肩披霸主雄狮皮的男人——原始大陆两大势力之一**迪亚部落**的**圣巫**（能力代代相传·据说能与古老存在沟通获强大力量）。只是坐着就有震彻灵魂的压迫感（一句话能让蜂女王·卡萝眩晕栽倒）。他之下是四名萨满祭司 + 众分支部族。

- **联合抗虫族**：[虎蜂之主·泰密莎](虎蜂之主·泰密莎.md) 踏平了 [苏晓](../苏晓.md) 的威胁后来找他联手——两大宿敌破天荒合流（连契约者阵营声望都互通）。他态度强硬（「在战士们死尽前，没人能夺走我的领地」·与泰密莎当面互不相让、却也不真内讧），还**逼独立势力灰岩族 / 蜥族参战**（不来就战后灭它们），凑成对虫族的「四打一」。
- 主持「库库林·白夜歼灭战会议」——只要杀了苏晓，虫族必消亡。他还藏了张底牌「**乌雨**」（黑绿雨·让恶魔兽岩石化崩碎、连恶魔焰龙都被侵蚀），一度让战局逆转；却被苏晓当场召唤 **60 万恶魔兽**反攻打崩。
- **战至最后（殁）**：迪亚部落全线溃败时，古芜站在恶魔焰龙头上、亲手扯裂龙头、皮肤遍布裂痕殊死挣扎——终被十几道龙焰烧死坠落（菌毯没分解他的尸体·连敌人都认可他誓死不逃）。他一死，本就由多部落拼成的迪亚大聚地立刻分崩离析。杀他得世界之源 10.2% + **誓约之荣光**（[古老意志](../苏晓.md) 天赋任务的钥匙·[纳吉·加](纳吉·加.md) 图谋之物）。

## 关系

- [苏晓](../苏晓.md)：联军要歼灭的头号目标。
- [虎蜂之主·泰密莎](虎蜂之主·泰密莎.md)：破天荒联手的宿敌。
- [纳吉·加](纳吉·加.md)：混进迪亚部落高层、被他气势震慑的契约者。
