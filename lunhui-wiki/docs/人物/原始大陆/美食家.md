# 美食家（狂血 · 轮回乐园 · 树之竞技场53名 · 吃人肉的追猎者）

棕黄竖瞳、满嘴尖牙、笑起来很「熟悉」的男人——**轮回乐园**七阶契约者、**树之竞技场第 53 名**、「狂血的美食家」。是原始大陆生存试炼里的**追猎者**，也是货真价实的轮回疯子（把敌人的血肉搓成肉球吃、还嫌难吃）。

- **追猎者内讧**：与 [苏晓](../苏晓.md) 等被少年月光拉去「组队」，一照面五人全是追猎者——他先与苏晓「所见略同」看破各人乐园，苏晓抽身后，他**当场瞬杀月光（死亡乐园）**、又盯上要逃的紫罗兰（圣光），扬言三个全宰，纯粹是「感觉不爽」而非图收益。后来也想去虎蜂巢捞好处、被几万只精英虎眼蜂追得狂奔逃走。

- **诈死 · 半个违规者（三十七卷33–35章）**：被 [纳吉·加](纳吉·加.md) 逼着自证时，与 [苏晓](../苏晓.md) 烙印通讯挑好「用哪把武器打你」，配合 [牧枭](牧枭.md) 的斩杀能力「原罪·长眠」上演一场**诈死**——击杀提示、猩红卡掉落一应俱全，连纳吉·加都信了（触须化占了个契约者的身体挨围杀）。诈死代价是变回一米一的小孩、两天才恢复。他自称「半个违规者·正在从良」（越高阶越觉得违规者是坑），约苏晓死亡林地外见、去夺第 7 日紫箱。

## 关系

- [苏晓](../苏晓.md)：同为追猎者、同属轮回乐园、互换编号临时合作。
- 月光 / 紫罗兰：他要清掉的他乐园追猎者。
