# 幻术师（天启乐园·疑似开挂的凝聚者·梦幻开局的生存者）

戴无框眼镜的男人——**天启乐园**契约者、原始大陆生存试炼里最会「聚人」的核心。人格魅力与领袖才能出众，短时间就聚起 30+ 契约者（拉拢来的强者连 [牧枭](牧枭.md) 都甘当战力担当、自己居中协调），若放任发展可能拉出百人队。

- **疑似开挂**：他一系列行为宛如开挂——偶然救下 [雨喉部落](../原始大陆群像.md) 族人入其阵营、又能**在广袤水源区精准找到一个个小队**。[苏晓](../苏晓.md) 判断他有件「能纵观全局、标出每名契约者位置」的物品 / 借来的能力，但**使用间隔长 / 不稳定**（外挂「到期」了就得靠围杀霸主给众人尝甜头稳住人心）。
- **雌雄双猿的翻车**：他带 32 人去沼泽区围杀霸主巨猿（设的是「等追猎者来渔翁得利」的陷阱）——外挂重启时靠幻觉迷住黑白双猿十几秒，却被苏晓召来的**[生灵屠夫](../../力量体系/装备道具/index.md)** 搅局、被黑猿一把捏摔、队伍崩盘。战后名号彻底歇菜（被「幻术师想独吞物资」一嗓子搞得只剩 11 人）。他用「灵魂催眠」报复抓走 [希贝儿](希贝儿.md)、被苏晓斩了施术者。
- **真身 = 灰绅士的人偶（殁 · 三十七卷20–23章）**：苏晓端雨喉部落时用**四颗改良阿波罗**把他连同萨满祭司一起炸死——奇的是苏晓反获**钻石勋章 ×10**，揭出他其实是违规者**灰绅士的「工具人 / 人偶」**：外挂 = 灰绅士的远程操控 / 能力赋予（被中断后时灵时不灵、他自己也慌），捞的好处全归灰绅士。他至死保持风度、被第四颗阿波罗前忍不住爆了句粗口。[牧枭](牧枭.md) 正是趁灰绅士失控他时接近、想循线追灰绅士。

## 关系

- [苏晓](../苏晓.md)：被他「帮忙」搅黄围杀、又斩其报复施术者的死敌。
- [牧枭](牧枭.md)：甘当他战力担当的圣光强者。
- 雨喉部落：他结盟的土著势力（供出巨猿情报 / 暗箭助战）。
