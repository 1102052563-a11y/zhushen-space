# 纳吉·加（生存者同盟领袖·瘟疫之主·图谋第三次觉醒天赋）

路人脸、却让人本能恐惧的男人——**生存者同盟**的领袖（背上血肉会涌动·爱嚼火红木炭·手下「[虫男](../原始大陆群像.md)」真名「瘟虫」）。相貌天生，气势却是后天经历磨出来的；连追猎者们都对他发怵。

- **垄断「出牢笼」的钥匙**：他抢先探明「试炼区 = 牢笼」，靠古精灵祭坛（雨喉地下异空间·祭献装备换白琥珀）**垄断通过死亡林地的入场资格**——以「一小队杀一名追猎者」为门槛，驱使生存者同盟四处清剿追猎者、连轮回生存者也要铲除，好独占后续物资（他能抽 10~15%）。识破 [苏晓](../苏晓.md) 伪装成感知系「暗木」潜入、逼 [美食家](../原始大陆群像.md) 自证（一口叫破「斩首的夜 / 绅士的人偶 / 断罪牧枭 / 狂食美食家…」）。
- **第三次觉醒的野心**：他读到古精灵记载——**天赋成长到极限后有唯一方法可强行「第三次觉醒」**（契约者公认天赋只能觉醒两次）——认定这场神仙打架就是那机遇；水源区还藏着一只旧八阶 boss，他要伺机瞬杀（需不受打扰地布置）。发现石碑被苏晓毁去三分之二时暴怒（背肉涌动、险些屠尽所有人），随即悬赏追猎者人头、下令处理掉 [阿西瓦](../原始大陆群像.md) 和冰爵。他与违规者**灰绅士**有旧交情、正是趁其人偶幻术师死后来补那个空位。

- **图谋落空 · 被斩（殁 · 三十七卷58–59章）**：决战里他一脚把叛徒冰爵踹进恶魔兽群喂虫，最终迪亚部落惨败、被苏晓强大精神力锁定。他自曝是**违规者**（付大代价来参与试炼、图谋的正是古芜掉落的「誓约之荣光」/ 古老意志），想拿「灰绅士 / 违规者情报」跟苏晓签约换命——苏晓「不好奇」，一刀斩了他。（他其实一直没料到对面不是隔千里智斗的老阴哔、而是握几十万虫族的召唤师。）

## 关系

- [苏晓](../苏晓.md)：被其伪装潜入、又毁其祭坛石碑、最终一刀斩杀的头号变数。
- [美食家](../原始大陆群像.md) / [牧枭](../原始大陆群像.md)：逼美食家自证、与牧枭联手「杀」美食家。
- 灰绅士：有旧交情、想接他人偶死后的空位。
