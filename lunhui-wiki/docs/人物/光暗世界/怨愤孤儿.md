# 怨愤孤儿（天空巨眼 · 古神转生失败的产物 · 殁）

半个安戈城大的愤怒巨眼、「还未出生就死」的怨念存在——轮回乐园标「极其危险」（焚灵魂灰烬可能惊动它）。

- **真身**：[厄运镇古神](厄运镇古神.md) 转生失败的产物（母体人类女强者、父为古神·新生体死于胎腹）——是古神之力的现任掌控者。
- **结局**：靠「新生之火」全盛复活（生命回 92%）、拖 [苏晓](../苏晓.md) 入黑白领域——却被苏晓早先甩上的 [冥神脐带](../../力量体系/装备道具.md)「污秽之呓语」侵体弄死（苏晓炼金销毁脐带、掐断与冥神的联系防其降临）。

## 关系

- [厄运镇古神](厄运镇古神.md)：孕育它（转生失败）的古神。
- [大主教](大主教.md)：想战胜它成新古神的野心家。
- [苏晓](../苏晓.md)：用冥神脐带弄死它的人。

## 出处

- 三十四卷（[光·暗世界](../../世界/任务世界/光暗世界-宿命之路.md)）39–52 章：全盛复活→被冥神脐带弄死。
