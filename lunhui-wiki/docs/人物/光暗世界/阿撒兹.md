# 阿撒兹（老议员心腹 · 真身 = 大主教原定的「救赎者」）

拄手杖、右腿旧伤、病恹恹的瘸腿老绅士——代议会院迎 [苏晓](../苏晓.md)；忠于 [圣境老议员](圣境老议员.md)、养着十几只看家的马俄犬；实力深浅难测。

- **真身大揭示（三十四卷38章）**：他是 [大主教](大主教.md) 这一代选中的「两个最优秀者之一」（另一个是「反派」[格鲁](格鲁.md)）——都从父母遗传了古神之力并化为可控，是这世界真正的「**主角**」。
- **原定救赎剧本**：阿撒兹杀格鲁得救赎石板 → 查奥卡兹陵墓进厄运镇 → 古神之力觉醒成「人造神选者」→ 杀 [不朽之人](不朽之人.md)（钥匙）/ [齐斯尔曼](齐斯尔曼.md)（灰烬熔炉）/ [怨愤孤儿](怨愤孤儿.md) → 唤醒大主教救老议员。**这套剧本被苏晓无意间顶替走**（苏晓抢先杀了格鲁、走了他的路线）。

## 关系

- [圣境老议员](圣境老议员.md)：他忠心侍奉的主人。
- [格鲁](格鲁.md)：大主教选中的另一「最优秀者」（反派角）。
- [大主教](大主教.md)：为他写下救赎剧本的幕后。
- [苏晓](../苏晓.md)：无意间顶替走他剧本的人。

## 出处

- 三十四卷（[光·暗世界](../../世界/任务世界/光暗世界-宿命之路.md)）1–38 章：迎苏晓→真身揭为原定救赎者。
