# 埃尔维斯·格鲁（黑袍人 · 幕后大反派 · 王国毁灭者）

脸生黑色触手（自拔来吃）的黑袍男——本世界表面的幕后大敌，「忍」功登峰造极的老阴哔（[苏晓](../苏晓.md) 遇过「最能忍」的）。

- **身世**：边塞小村落出身。17 岁发现「**秘纹 + 蒸汽技术的结合会带来绝对毁灭**」（上一代文明已为此覆灭）——举报无门反遭构陷判刑、逃亡、最终加入一个成立千年的势力身居高位。信念 = 就算变恶鬼也要收回所有秘纹、阻止秘纹与蒸汽结合。
- **手段**：借**厄运镇古神之力**放「黑色光球」剥离秘术师秘纹、把弱者化成 [木眷族](../../世界/生物万族/index.md)，**三日内灭东泽 / 月溪 / 风暴三王国**；与 [鬣狼族](鬣狼族.md) 合作。
- **真相**：他其实是 [大主教](大主教.md) 千年剧本里的**棋子**（历代组织首领·被灌「秘纹灭世」的假秘密）。被苏晓一刀秒了肉体、灵魂困厄运镇当引 [怨愤孤儿](怨愤孤儿.md) 的诱饵；[阿撒兹](阿撒兹.md) 是与他并列的另一「最优秀者」。最终苏晓焚古神驱壳时用 D·暗杀打碎、给他「死在强敌手中」的体面。

## 关系

- [大主教](大主教.md)：把他当棋子的真正幕后。
- [阿撒兹](阿撒兹.md)：与他并列的另一「救赎者 / 反派」候选。
- [苏晓](../苏晓.md)：秒他肉体、又给他体面之死的人。
