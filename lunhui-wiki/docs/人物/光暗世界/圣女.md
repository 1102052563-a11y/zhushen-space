# 圣女 = 索娅（领主之女 · 大主教挚爱的影子 · 被洗礼成工具）

救赎石板可三选一唤醒「修女 / 女祭司 / 圣女」，[苏晓](../苏晓.md) 选了**圣女**（未被古神侵蚀·敢向苏晓求救）。

- **天生的「圣枪」使用者**（原力 162 / 敏 171 / 体 131），却被 [大主教](大主教.md) 强推入洗礼池、折断圣枪、力敏体大降（现个位数）、沁没成只会治疗的工具（智 172 / 魅 169·光愈复苏每秒回 1700~2600 生命）。是她隐晦点出「大主教」的存在向苏晓求救。
- **真相**：大主教因她的眼睛太像已死的挚爱「**索娅**」，才立她为圣女、又矛盾地把她洗礼成工具（洗礼时神灵之身竟感到心痛）。大主教死后她把半截圣枪赠苏晓「让它继续战斗」。

## 关系

- [大主教](大主教.md)：把她当挚爱影子、又洗成工具的幕后。
- [苏晓](../苏晓.md)：选择唤醒她、她向之求救的契约者。

## 出处

- 三十四卷（[光·暗世界](../../世界/任务世界/光暗世界-宿命之路.md)）23–52 章：被唤醒求救→身份 = 索娅影子→赠半截圣枪。
