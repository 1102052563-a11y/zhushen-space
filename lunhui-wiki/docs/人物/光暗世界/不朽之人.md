# 不朽之人 / 阿鲁斯·莱克（厄运镇守门领主 · 救赎教堂「门之守卫」· 殁）

破衣 + 无护手长刀 + 腰挂破损头骨的干枯强者（力 167·古神能量 61500）——守 [厄运镇](../../世界/任务世界/光暗世界-宿命之路.md) 入口、钥匙在他身上、无法离开厄运镇。

- 本名**阿鲁斯·莱克**：纯粹的战士、[救赎教堂](齐斯尔曼.md) 的「门之守卫」/ 杀人机器（杀人时良知 / 荣誉 / 使命在心中抗争）——**拒绝祭拜古神**（拒屈服·故受古神之力侵蚀成不朽之人·愿他不朽）。斩杀了先来的己方女契约者 + 死亡乐园契约者。
- 被 [苏晓](../苏晓.md) 用青影王「灭法之刃」0.8 秒斩杀。掉圣灵级长刀「无主之刃」。

## 关系

- [苏晓](../苏晓.md)：0.8 秒斩他、取厄运镇钥匙的人。
- [齐斯尔曼](齐斯尔曼.md) / [铁匠](铁匠.md)：救赎教会「三基友」的另两位。

## 出处

- 三十四卷（[光·暗世界](../../世界/任务世界/光暗世界-宿命之路.md)）23–38 章：守厄运镇门→被苏晓灭法之刃斩杀。
