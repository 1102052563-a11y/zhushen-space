# 卡律（鬣狼 · 月光大人麾下 · 殁）

黑礼服银耳环、彬彬有礼却野兽气息的绅士——[鬣狼族](鬣狼族.md) 一员、奉「月光大人」之命劫 [苏晓](../苏晓.md)。

- 金属爪 / 嚼子弹吐射的战法，火车上第一波来劫苏晓，被苏晓一刀斩首。

## 关系

- [鬣狼族](鬣狼族.md)：他所属的夜行猎手族群。
- [苏晓](../苏晓.md)：斩他的目标。

## 出处

- 三十四卷（[光·暗世界](../../世界/任务世界/光暗世界-宿命之路.md)）1–9 章：火车劫杀苏晓被斩首。
