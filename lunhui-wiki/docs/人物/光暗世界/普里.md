# 普里（代理议员 · 圣境老议员之子 · 32 岁）

极不正经（憋着尿装死迎客、爱坐蒸汽火车游历四王国、女伴不断却风流不下流）、正事却靠谱。

- 请 [苏晓](../苏晓.md) 救父 [圣境老议员](圣境老议员.md)，承诺「什么要求都行除了娶我妹妹」（19 岁妹妹是他比当议员更自豪的事）。
- 老议员病危后接掌其权柄，是苏晓在议会院的主要合作方。

## 关系

- [圣境老议员](圣境老议员.md)：他病危的父亲。
- [苏晓](../苏晓.md)：请来救父、追查幕后的合作者。
- [阿撒兹](阿撒兹.md)：父亲的瘸腿心腹。
