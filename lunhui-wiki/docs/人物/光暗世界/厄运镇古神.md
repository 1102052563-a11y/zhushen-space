# 厄运镇古神（远超七阶 · 其实早已死）

能量恶意 / 侵蚀力远超索托斯 / 莎布（梦魇之神尼加狄在它面前「连幼童都算不上」）、疑与冥神阶位相近——却是 [苏晓](../苏晓.md) 见过最安分的古神（只是「存在」就衍生黑色光球 / 灵魂灰烬 / [怨愤孤儿](怨愤孤儿.md)）。

- **真相（三十四卷39–52章）**：它其实**早已死了**——濒死时逃到此世界、片刻吮吸就凉的重伤古神；死后古神之力蔓延才成了厄运镇的不幸。[怨愤孤儿](怨愤孤儿.md) = 它转生失败的产物（母体人类女强者、父为古神·新生体死于胎腹）。
- **古神之力四特性**：光 / 暗 / 新生之火 / 心灵——接纳者心灵会渐渐「古神化」，[大主教](大主教.md) / [齐斯尔曼](齐斯尔曼.md) / 怨愤谁成功都可能沦为「新古神」。奥卡兹大帝当年正是由厄运镇得力量建光暗帝国。
- **🆕复生（三十六卷1章·跨世界伏笔）**：厄运镇只剩飘散的古神能量。**流放之地的雇佣杀手四人小队追杀 [苏晓](../苏晓.md)、却因其 13013 号虚拟防追踪编号被传送进厄运镇禁地**（「生者止步」）——古神吞了他们灵魂、又欲吞苏晓的血滴，血滴被**青钢影电弧炸毁**、古神只得一丝青钢影电弧，靠「**新生之火**」复生（怒吼「灭法者！！」）。这只本已死透的重伤古神，又回来了。
