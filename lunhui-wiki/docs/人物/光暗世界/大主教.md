# 大主教（救赎教堂最深处 · 真正幕后 · 蒸汽学天才少年 · 殁）

名字早被遗忘的「黑心老爷爷」、活了千年、无法离开救赎教堂一步（出则即死），却能以精神体穿透黑暗抵达外界——本世界一切的**真正缔造者**。

- **千年布局**：让奥卡兹大帝在厄运镇得力量建光暗帝国（棋子）→ 弄出「秘纹」（本属古神之物）→ 成立千年神秘组织（[格鲁](格鲁.md) 是历代首领·鬣狼支持只是幌子）→ 散播「秘纹会灭世」的假秘密骗首领清除秘纹。最终目的 = **战胜 [怨愤孤儿](怨愤孤儿.md)、成为新的古神**（不惜害死本世界 99% 生灵·「战胜恶龙的勇士自己长出了龙鳞」）。[格鲁](格鲁.md) / [阿撒兹](阿撒兹.md) 都是他棋局里的棋子。
- **真身（三十四卷39–52章）**：本是孤儿院的**蒸汽学天才少年**（「蒸汽」一词就是他造的·祖母瘟疫死、挚爱 [索娅](圣女.md) 死），加入救赎教会后被古神扶成大主教、逐渐被古神之力侵蚀到忘了自己是发明蒸汽的少年、忘了挚爱，只记得要成新古神。被 [苏晓](../苏晓.md) 联手 [齐斯尔曼](齐斯尔曼.md) + 怨愤孤儿三对一斩杀。

## 关系

- [格鲁](格鲁.md) / [阿撒兹](阿撒兹.md)：他棋局里的两枚「最优秀者」棋子。
- [圣女](圣女.md)（= 索娅影子）：他矛盾地立为圣女又洗成工具的少女。
- [苏晓](../苏晓.md)：顶替阿撒兹剧本、最终斩杀他的人。
