# 杀人蜂（死亡乐园 · 四人小队真正的可怕者 · 殁）

娇小远程女契约者——真身是主人，靠**寄生**（钻进队长体内再爬出·每次身体变小 / 吸生命力回春）续命。

- 计划先与队友捞好处再阴他们独吞；[白牛](../光暗世界群像.md) 自爆的火焰烧到她头才翻车，仍带召唤物 [俄伊司](../光暗世界群像.md) 深入厄运镇（是 [苏晓](../苏晓.md) 在镇内的隐患）。
- 后被苏晓刺心斩杀。

## 关系

- [苏晓](../苏晓.md)：镇内隐患、最终刺心斩她的人。
- 俄伊司 / 队长 / 白牛：四人小队队友（她的寄生宿主 / 独吞对象）。

## 出处

- 三十四卷（[光·暗世界](../../世界/任务世界/光暗世界-宿命之路.md)）23–52 章：闯厄运镇→寄生续命→被苏晓刺心斩杀。
