# 铁匠（救赎教会三基友之一 · 灰烬熔炉锻造者）

救赎教会「三基友」= [不朽之人](不朽之人.md) 莱克 / [守炉人齐斯尔曼](齐斯尔曼.md) / **铁匠**（在石晶墙后·会用灰烬熔炉锻造对付 [怨愤孤儿](怨愤孤儿.md) 的武器·愤怒的铁匠「比小镇孤儿更可怕」）。

- [大主教](大主教.md) 死后恢复神智，用**灰烬熔炉**替 [苏晓](../苏晓.md) 重锻黑王护臂（成 [黑王护臂](../../力量体系/装备道具.md)）——有恩必报、只看结果。

## 关系

- [苏晓](../苏晓.md)：替他重锻黑王护臂的报恩对象。
- [不朽之人](不朽之人.md) / [齐斯尔曼](齐斯尔曼.md)：救赎教会「三基友」的另两位。

## 出处

- 三十四卷（[光·暗世界](../../世界/任务世界/光暗世界-宿命之路.md)）39–52 章：恢复神智→重锻黑王护臂。
