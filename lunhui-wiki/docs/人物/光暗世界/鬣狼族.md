# 鬣狼族（夜行猎手 · 专猎高层）

人 / 兽可切换的夜行族，以家族为纽带（每族数十上百脉·可为劳工 / 官员、召开家族会议即夜间猎手）——能与人婚配生纯血后裔。

- 平均战力约七阶，专猎四王国 + 议会院高层（尤恨 [圣境老议员](圣境老议员.md)）。
- 某「月光大人」与议会院幕后 [格鲁](格鲁.md) 勾结（安戈城灭后大陆归鬣狼、建不碰秘纹的新文明）——实则鬣狼支持只是 [大主教](大主教.md) 千年布局的幌子。代表成员 [卡律](卡律.md) / [野兽绅士·图亚克](野兽绅士·图亚克.md)。

## 关系

- [格鲁](格鲁.md)：与鬣狼「月光大人」勾结的幕后。
- [圣境老议员](圣境老议员.md)：鬣狼最恨的猎狼者。

## 出处

- 三十四卷（[光·暗世界](../../世界/任务世界/光暗世界-宿命之路.md)）1–22 章：劫火车→与格鲁合作图安戈城。
