# 迪达拉

[火影忍者世界](../世界/任务世界/火影忍者.md)「**晓**」组织成员、岩忍村叛忍（护额带划痕）、**黏土爆破使**——19 岁的金发少年（扎短辫、左眼被金发遮、说话带「嗯」口癖）。是 [苏晓](苏晓.md) 在晓的**临时搭档**之一（与 [蝎](蝎.md) 同队·骑白色黏土大鸟出行）。

- **黏土炸弹 / 艺术观**：以「**爆炸的瞬间**」为艺术（与蝎「永恒艺术」理念相左）；单纯直率却带残忍。师傅是**三代土影大野木**（若不叛村·四代土影或是他而非黑土）。
- **与苏晓的炸弹之交**：他嗅出苏晓身上「奇怪的炸弹味」、一度警惕苏晓与岩忍有关——见到苏晓的 [炼金炸弹](../力量体系/消耗品材料.md)（与黏土炸弹相似·苏晓送他一颗中级）后发现**可融精神力操控**（变蛇变鸟·比黏土更省力），两人交流炸弹心得却**都不透露核心配方**（初见无信任·苏晓求黏土「飞行」秘诀、迪达拉笑而不答）。布布汪看迪达拉的眼神「像看同类」。
