# 迪达拉

[火影忍者世界](../世界/任务世界/火影忍者.md)「**晓**」组织成员、岩忍村叛忍（护额带划痕）、**黏土爆破使**——19 岁的金发少年（扎短辫、左眼被金发遮、说话带「嗯」口癖）。是 [苏晓](苏晓.md) 在晓的**临时搭档**之一（与 [蝎](蝎.md) 同队·骑白色黏土大鸟出行）。

- **黏土炸弹 / 艺术观**：以「**爆炸的瞬间**」为艺术（与蝎「永恒艺术」理念相左）；单纯直率却带残忍。师傅是**三代土影大野木**（若不叛村·四代土影或是他而非黑土）。
- **与苏晓的炸弹之交**：他嗅出苏晓身上「奇怪的炸弹味」、一度警惕苏晓与岩忍有关——见到苏晓的 [炼金炸弹](../力量体系/消耗品材料.md)（与黏土炸弹相似·苏晓送他一颗中级）后发现**可融精神力操控**（变蛇变鸟·比黏土更省力），两人交流炸弹心得却**都不透露核心配方**（初见无信任·苏晓求黏土「飞行」秘诀、迪达拉笑而不答）。布布汪看迪达拉的眼神「像看同类」。

## 围攻自来也 / 砂忍村 / 被九尾炸断肢（九卷24–46章）
他参与围攻三忍 [自来也](自来也.md)（黏土鸟空中轰炸 + 杀招「十八号 C3」）、空袭砂忍村活捉 [我爱罗](我爱罗.md)（与苏晓研究出「让苏晓临时操控注入查克拉的黏土鸟」之技·撤退时「十八番」炸毁砂忍村五分之二）。

- **被九尾尾兽炮炸断一手一脚（九卷43–46章）**：封印一尾后他带我爱罗尸体坐尸挑衅、彻底激怒九尾人柱力 [漩涡鸣人](漩涡鸣人.md)——**被爆出的九尾尾兽炮炸断一只手 + 一条腿**狼狈逃命（[苏晓](苏晓.md) 半路捡走他·他念叨断肢还在森林里、要去找 [角都](角都.md) 缝合）。

## 五影大会被俘 · 师徒纠葛（十五卷15–24章）

五影大会中迪达拉负责引开土影、却**被俘**（土影一族太懂他——爆遁是土遁分支、被同级雷遁克制）；被黑土塞进布袋扛回（嘴用石灰凝之术封住·身为 S 级叛忍丢尽脸面）。审问时他装糊涂「不认识白夜」「间歇性失忆」，硬骨头一如所有晓成员。**[大野木](大野木.md)（土影）其实有意保他**——迪达拉是他「最聪慧的弟子、部下的遗孤」，若非他太叛逆、土影候选人本该是迪达拉而非黑土；土影用「他自爆能再夷平木叶」挡回团藏的索人。迪达拉笃定晓组织会来救他（晓急需战力、同小队队友还活着）。
