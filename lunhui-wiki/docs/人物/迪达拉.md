# 迪达拉

[火影忍者世界](../世界/任务世界/火影忍者.md)「**晓**」组织成员、岩忍村叛忍（护额带划痕）、**黏土爆破使**——19 岁的金发少年（扎短辫、左眼被金发遮、说话带「嗯」口癖）。是 [苏晓](苏晓.md) 在晓的**临时搭档**之一（与 [蝎](蝎.md) 同队·骑白色黏土大鸟出行）。

- **黏土炸弹 / 艺术观**：以「**爆炸的瞬间**」为艺术（与蝎「永恒艺术」理念相左）；单纯直率却带残忍。师傅是**三代土影大野木**（若不叛村·四代土影或是他而非黑土）。
- **与苏晓的炸弹之交**：他嗅出苏晓身上「奇怪的炸弹味」、一度警惕苏晓与岩忍有关——见到苏晓的 [炼金炸弹](../力量体系/消耗品材料/index.md)（与黏土炸弹相似·苏晓送他一颗中级）后发现**可融精神力操控**（变蛇变鸟·比黏土更省力），两人交流炸弹心得却**都不透露核心配方**（初见无信任·苏晓求黏土「飞行」秘诀、迪达拉笑而不答）。布布汪看迪达拉的眼神「像看同类」。

## 围攻自来也 / 砂忍村 / 被九尾炸断肢（九卷24–46章）
他参与围攻三忍 [自来也](自来也.md)（黏土鸟空中轰炸 + 杀招「十八号 C3」）、空袭砂忍村活捉 [我爱罗](我爱罗.md)（与苏晓研究出「让苏晓临时操控注入查克拉的黏土鸟」之技·撤退时「十八番」炸毁砂忍村五分之二）。

- **被九尾尾兽炮炸断一手一脚（九卷43–46章）**：封印一尾后他带我爱罗尸体坐尸挑衅、彻底激怒九尾人柱力 [漩涡鸣人](漩涡鸣人.md)——**被爆出的九尾尾兽炮炸断一只手 + 一条腿**狼狈逃命（[苏晓](苏晓.md) 半路捡走他·他念叨断肢还在森林里、要去找 [角都](角都.md) 缝合）。

## 五影大会被俘 · 师徒纠葛（十五卷15–24章）

五影大会中迪达拉负责引开土影、却**被俘**（土影一族太懂他——爆遁是土遁分支、被同级雷遁克制）；被黑土塞进布袋扛回（嘴用石灰凝之术封住·身为 S 级叛忍丢尽脸面）。审问时他装糊涂「不认识白夜」「间歇性失忆」，硬骨头一如所有晓成员。**[大野木](大野木.md)（土影）其实有意保他**——迪达拉是他「最聪慧的弟子、部下的遗孤」，若非他太叛逆、土影候选人本该是迪达拉而非黑土；土影用「他自爆能再夷平木叶」挡回团藏的索人。迪达拉笃定晓组织会来救他（晓急需战力、同小队队友还活着）。

**被押岩隐 · 晓将营救（十五卷48章）**：白绝目睹他**被三代土影 [大野木](大野木.md) 生擒**（土影护卫用一种**雷遁秘术卷轴克住他的爆遁**、令他连自爆都没用出来·这才被擒），现押在岩隐村附近。带土把营救迪达拉与「外来者情报」一并拿来与苏晓交易——苏晓与迪达拉多次并肩、私交不错，遂同意一同去救。

## 被「不让睡觉」折磨 + 获救（十五卷50–59章）

竖岩村地下监狱里他没受刑、伙食还极好——岩隐对他用的是「**连续不让睡觉**」（困了就扇耳光 / 针刺 / 泼冰水）：几天下来他神志恍惚、把幻听当成苏晓 / 带土的声音、几次险些说漏晓的机密（他自知撑不了多久）。苏晓攻村时他昏昏沉沉听不懂话，**[布布汪](布布汪.md) 趁大野木放尘盾时摸进牢房把他救走**（留「本汪到此一游」）。获救后他直接昏睡不醒（多日无眠 + 忽冷忽热患重感冒·苏晓抽肿他的脸都没弄醒），撤退路上由 [阿姆](阿姆.md) 扛着——只要他一醒、晓就有飞行能力甩脱岩隐追兵。

## 萌生退意 + 力挺苏晓（十五卷60–62章）

获救后他开始动「**退出晓**」的念头：他当年入晓只为「证明爆遁才是真正的艺术、胜过 [鼬](宇智波鼬.md) 的写轮眼」——可鼬已死（对手没了）、长门也死、他看不上 [佐助](宇智波佐助.md)、又不认同带土那套「月之眼」（实现了他还怎么探索艺术）；当初他选的首领是长门、不是带土，已没理由留下（暂被「斑来了」按住·叛晓只能硬叛、晓从无人退出过）。投票审 [蔷薇](黑瞳-契约者.md) 入伙时他**力挺苏晓投了反对票**（两人是出生入死的旧搭档·交情非带土 / 鬼鲛可比）；蔷薇逼人退场时他本不愿走（苏晓救过他一命），被苏晓以「这是我的个人恩怨」劝离。

## 还人情抓小南 · 「炸死」脱身（十五卷70–73章）

为还带土在竖岩村的救命人情（他叛忍却不忘恩、此前已动退意），他**主动请缨 + 拉佐助一同去抓 [小南](小南.md)**（苏晓给佐助补了压制柱间细胞的药丸）；黑绝也亲自随行（关乎轮回眼）。任务中他**与小南死战、随后被数不尽的起爆符「炸死」**——白绝事后隐约感知到他的查克拉还在又很快消失（战场在河边）、黑白绝推测他**七成以上未死**。带土遂放弃追究（早知其退意·救命人情已两清、再纠缠就要敌对）。这很可能是他借机脱离晓的金蝉脱壳。
