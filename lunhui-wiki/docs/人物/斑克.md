# 斑克（虚空虫族·生命进化系）

[苏晓](苏晓.md) 在 [虚空角斗场](../世界/任务世界/恶魔古堡游戏.md)（恶魔古堡第五场）休息室里撞见的**虚空虫族**——青白色甲壳、双腿与头部全包在甲壳内、双臂是两把带崩口的甲壳臂刃；不用语言、以特殊频率声波交流（起初被苏晓误以为在生气、其实是受伤后紧张想赶人）。**生命进化系**（非法系），自称无聊才来参赛、一直压制实力，可见其嗜战。

## 灭法之影的「试探对象」（十四卷46–48章）

苏晓借斑克试探灭法之影在虚空的处境——亮出青钢影的淡蓝瞳孔后，斑克先僵住、再缩到墙角瑟瑟发抖（融入血脉的本能敬畏），确认苏晓无恶意后却又彻底放松。由他口中苏晓厘清了关键世界观：

- **「灭法从不欺骗虫族」**：灭法之影与虫族是**中立 / 盟友**关系（斑克对灭法极信任）。
- **灭法 = 绝对中立**：「规则与元素的天平监督·**元素之内，灭法至高无上**·虚空不会允许灭法完全消失」（数量只削不灭、是虚空定律）；谁触禁忌灭法就临时变中立邪恶 / 混沌邪恶。
- **情报**：本届有灭法世仇**法师贤者瑟菲莉娅**来观战；**已知在世灭法仅苏晓一人**（灭法者时代千年前已过、斑克生来没见过、只在传说里听过）；角斗场提前发了印着苏晓照片的传单「消逝千余年的灭法者再现」（爱丽丝所为）。

确认苏晓无敌意后，斑克便跟在他身旁——有这样一个了解虚空、又无敌意的向导，对苏晓只有好处。他也是苏晓在首轮观察到的**强敌之一**（天生双臂刀·一直压制实力·目的不明）。

## 关系
