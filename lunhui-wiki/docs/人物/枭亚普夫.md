# 枭亚普夫（普夫 / 苍蝇男）

[蚁王·梅路艾姆](蚁王·梅路艾姆.md) 的**直属三护卫之一**——金发、头顶生触角（特性其实更像蝴蝶、被老烟戏称「苍蝇男」）。三护卫里**能力麻烦（能分裂、极难彻底消灭）、但身体并不算强**；智商与生存能力都很高。在原著里，[诺布](艾萨克·尼特罗.md) 仅感知到它的气息就被吓破了胆。

## 被围攻（十六卷37章·进行中）

蚁王讨伐队按分工由 [尘](尘.md) / [水咲](水咲.md) / [老烟枪](老烟枪.md) 三人围攻它（三人围杀把握更大）——它**胸口被蚁王吃掉大部分脏器、压制得无法分裂**（这正是好下手的时机）。老烟以巨烟布下中西结合的「八卦法阵」屏蔽它、水咲用两把裹着水能量的细弯刀、尘戴黑手套（替代法杖）以五元素球施法（冰 → 暗「暗天」），围杀正酣。

## 殒落 · 被「暗天·噬」抽空（十六卷38章）

[尘](尘.md) 一颗暗法球（吸附力极强）飞到它头顶、施「**暗天·噬**」——形成只抽离其体内**某种元素能量**的黑洞（不吸物质、专抽元素），元素一被抽空、它的念能量因「元素真空」失衡崩溃，身体**瞬间干瘪成一具干尸**而亡（尘的法爷理论：能量都依附于元素、元素真空令念失衡）。三护卫至此死了两个（尤比 / 普夫），只剩 [尼飞比特（猫女）](尼飞比特.md)——亦已被苏晓所杀。
