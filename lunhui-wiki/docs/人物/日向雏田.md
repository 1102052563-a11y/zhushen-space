# 日向雏田

[火影忍者世界](../世界/任务世界/火影忍者.md) 木叶忍者、日向一族——瞳术「**白眼**」使用者（360° 视野·一公里全览），原著的女主角。柔拳法 / 八卦六十四掌 / 回天 / 八卦空壁掌。属 [夕日红](夕日红.md) 的追击小队。

- **追踪与重伤（九卷66–68章）**：她的白眼是小队能追 [苏晓](苏晓.md) 四小时的关键（连续使用负担极大、需间歇闭眼）。混战中——她的「八卦空壁掌」把苏晓击飞，却**间接把 [犬冢牙](犬冢牙.md) 体内的影刺拖出、害死了队友**；随后她「回天」防御被苏晓青钢影一刀**斩破、断去一条小臂 + 横跨胸腹的刀伤**。苏晓本要斩她喉咙（「敌人是女主角就手软是脑残行为」），却被 [奈良鹿丸](奈良鹿丸.md) 的影子模仿术 + [山中井野](山中井野.md) 心转心一度拦住。她最终被 [油女志乃](油女志乃.md) 带回木叶（断口整齐·纲手医疗忍术可接回）——**成为苏晓在火影世界少数没杀成的主要剧情人物**（杀仅次于男一男二的女主角，除非大举攻木叶否则机会不大）。

## 出处
