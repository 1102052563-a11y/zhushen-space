# 手鞠

[火影忍者世界](../世界/任务世界/火影忍者.md) 砂忍——四代风影之女、[我爱罗](我爱罗.md) 与 [勘九郎](勘九郎.md) 的姐姐（三姐弟中最年长），约十九岁、金发、身材丰满，惯使一把一米多长的**大铁扇**。

- **风遁中距离战**：**镰鼬 / 大镰鼬**（铁扇挥出强风+隐藏其中的密集真空刃）、**风砂尘**（卷黄沙）。最忌与敌人拼体术近身。
- **被苏晓擒为人质（九卷35–36章）**：见 [我爱罗](我爱罗.md) 遇袭出手解围，却被 [苏晓](苏晓.md) 用 [界断线](../力量体系/技能.md) 借周围建筑结成的「八角笼」结界困住——真空刃被青钢影斩飞、铁扇被斩龙闪一刀斩断、锁骨被斩开深可及骨。苏晓掐住她后颈、以斩龙闪**缓缓划开她喉咙的皮肤**逼她向我爱罗呼救，成功令我爱罗分心。她被我爱罗用沙救走（实为苏晓故意放水），自此对苏晓留下心理阴影。战后弟弟被掳走、挚友战死，她向 [纲手](纲手.md) 求援、誓要晓付出代价。
