# 狂狐（神皇冒险团二把手 · 三阶竞技场原第一名）

[神皇冒险团](../势力/index.md) 的**副团长（二号人物）**、[神皇](神皇.md) 的五心腹之一——一身黑白相间绒衣、戴**狐狸头绒布兜帽**（「狂狐」之名疑由此而来），扛一把**传说级战镰**。是 [疯奶](../势力/index.md) 的师傅（轮回乐园基本没有师徒名分、二人以兄弟相处）。与 [苏晓](苏晓.md) 同属交好阵营（神皇冒险团与苏晓私交不错）、竞技中虽不留手却不结仇。

## 「最恶心的对手」· 竞技场登顶之战（十七卷100章）

苏晓打到三阶竞技场第二名后挑战首位狂狐——苏晓评他是**入园以来「最恶心的对手」：能打、能奶、能抗，「除了不会生孩子之外似乎什么都会」**：

- **开场无耻流**：开战即用一枚**传说级戒指**放出金色高压脉冲挡住苏晓突进，趁机在 5 秒内给自己叠 **30 多种增益状态**（含 5 种持续回血、1 种回法力、各类强化防御 / 速度），顺手还给苏晓套了 3 种减益（降速 / 增空气阻力 / 加速流失体力）——套完战力从「8」飙到「70 以上」。
- **续航惊人**：靠传说级战镰 + 海量回血 / 回蓝增益，与苏晓**鏖战 7 小时**（[疯奶](../势力/index.md) 看睡着、改玩联机游戏，最后观众只剩呼呼大睡的 [布布汪](布布汪.md)）。狂狐法力被苏晓青钢影耗空后才显疲软；苏晓则靠斩龙闪「嗜血礼赞」回血，双方僵持到苏晓停留时间将尽。
- **败因 = 那一脚**：苏晓的 [木之灵](../力量体系/装备道具.md) 主动效果（断骨 + 碎甲侧踢）二次冷却恢复、再次摆出躬身姿势——狂狐深知「**这一脚绝对扛不住第二次**」，当即弃权消失。苏晓一脚踢空、却就此**登顶三阶竞技场首位**，狂狐滑落第二、被迫交出 [善战者称号](../力量体系/称号.md)。

## 资料
- **战镰**：传说级战镰（与斩龙闪互相奈何不得·僵持）。
- **三主属性**：真实力 / 体 / 智应俱高（与荒焚相仿的三阶顶尖·具体未明示）；强在「能打能奶能抗」的全能续航而非爆发。

## 关系
- **[神皇](神皇.md)**：团长（二人意见不合就私下「拳脚论道」、讲理者胜）。
- **[疯奶](../势力/index.md)（战斗奶爸）**：徒弟（以兄弟相处）。
- **[抹茶](../势力/index.md)**：同团首席奶妈（吐槽狂狐把她当「挂件」挂在镰刀上「保护」）。
- **[苏晓](苏晓.md)**：交好阵营、竞技场登顶战的对手（败而不结仇）。
