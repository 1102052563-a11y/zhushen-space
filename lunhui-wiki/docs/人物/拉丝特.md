# 拉丝特（色欲）

[瓶中的小人](瓶中的小人.md) 所造七名人造人之一，代表「**色欲**」——黑波浪长发美女（超 200 岁·锁骨下有红色衔尾蛇 = 人造人标志）。

- **能力「最强之矛」**：手指可化作 3 米长的利刃、刺穿大多数物质，**擦伤即触发即死判定**；绝招「万矛穿心」「刺刑」。攻击力为七人造人之最，但防御 / 速度 / 恢复平平。
- **殁**：在炼金阵节点城里欧尔的教堂被苏晓盯上——苏晓界断线缠住其核心贤者之石、三笠的心意护盾硬挡「万矛穿心」（太阳穴见骨），最终以**青钢影「法力爆燃」**（408 真伤）击碎其贤者之石**斩杀**（剧情关键·世界之源 +5.3% → 6.5%·紫宝箱·七卷8章）。
