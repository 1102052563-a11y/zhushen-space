# 恶魔老太婆

> ⚠ **恶魔族的至强巅峰**——⚠ **不少大势力高层都认为，恶魔族正是在她的余荫下，才能靠接连当墙头草兴盛到今天**;⚠ **⚠ 外形**:⚠ **身披灰袍·严重驼背·枯槁的手臂·内部宛如有岩浆在燃烧的双眼〔眼底漆黑、瞳孔如同眼睛在燃烧〕·脖颈上用粗绳挂着铜铃，走起路来发出当啷当啷、让人灵魂眩晕的声响**;⚠ **⚠ 她的能力可以抹去距离、空间，甚至因果——「距离对于她来讲没有意义」**（⚠ 她能让整列高速行驶的领主列车不知不觉越来越慢，最终无故障停下）。

## 🤝 陨灭星的一次对峙（54卷166章·第4060章）

⚠ **她走上领主列车，开门见山:「灭法小子，你现在也不想和我这老婆子交手吧，所以，让我带走她，是对你对我都好的选择」**;⚠ **⚠ 她对双方战力的判断非常坦率**:⚠ **「你能和鹿神打到这种程度，我能击败你，但你如果一心逃脱，我的确杀不掉你」**;⚠ **⚠ 苏晓的回应是最平静的语气说出最残忍的话**:⚠ **「你带走她，我去虚空，把恶魔族宰得一只不剩。」**——⚠ **面对其他人，恶魔老太婆会阴森地回一句「你敢！」；可面对苏晓她没这么说，因为这灭法者是真的敢——把一族杀到一个不剩，本就是他常用的手段**（⚠ 气氛安静到令人窒息，半分钟对峙后她按在眼之公主肩上的手放下，笑得非常诡异，慢慢走下列车）。

- ⚠ **⚠ 她为何退让**:⚠ **并非怂了——她洞悉到了那跃跃欲试要将苏晓吞没的「原罪因果」**——⚠ **这种情况下，她不愿与一个不足一月就要迎来大劫难的灭法者拼个你死我活**;⚠ **⚠ 苏晓的惋惜**:⚠ **他从桌底扯下一张契约羊皮纸——只差那么一会儿就能签订成功**（⚠ 看来这老恶魔有这方面的预警能力，所以才果断离开·⚠ 同一套桌下契约，此前刚在 [幸运女神](幸运女神.md) 身上大获成功）。
- ⚠ **⚠ 副作用**:⚠ **眼之公主在她的威压下恐惧到忘记呼吸**——⚠ **也正是这次对峙，让眼之公主从「找机会逃走」彻底转为「说什么都不下这列车」**。

## 📜 恶魔族两个纪元的墙头草史（苏晓的复盘）

⚠ **第一纪元:恶魔族是太阳阵营的盟友**——⚠ **因太阳阵营与古龙阵营一同衰亡，它们暗中勾结古龙阵营的龌龊事没被外界知晓**;⚠ **⚠ 第二纪元:恶魔族是灭法者的同盟**——⚠ **看到这里聪明人就已看出端倪:哪方实力强，它们就是哪方的盟友**;⚠ **⚠ 它们万万没想到，奥术永恒星背后隐藏的领袖竟是至高之人**——⚠ **直接滑跪太丢脸，于是买通虚空各大报社媒体大肆报道被施法者杀死的恶魔战士**（⚠ 必须承认，在灭法VS施法的前中期恶魔族**确实出了力**，中下层成员没少死；但暂没听闻有高阶恶魔死于那场纪元之战·⚠ 这造成当时恶魔族内部近乎割裂:**中下层恶魔真的是灭法盟友**，因为他们痛恨那些把自己性命当筹码的高层，而高层则是纯粹的墙头草）。

- ⚠ **⚠ 苏晓的结论**:⚠ **那些「真正的血誓盟友」和月狼一样，早已全部身陨**——⚠ **他在中期时感受到的恶魔族「友谊」，并非因为他灭法者的身份，归根结底是恶魔族极其需要黑枫树产出，且确信哪怕杀死或生擒他也夺不下那棵树**;⚠ **⚠ 到了绝强、至强后，恶魔族那边一点音讯都没了——那些老恶魔心里也是有数的**（⚠ 若还敢强调「灭法阵营的盟友之谊」，那就是找死）;⚠ **⚠ 对照组**:⚠ **思林特斯矮人是灭法者真正的盟友，同遭奥术永恒星清算打压，曾是一线势力的它们现在近乎灭亡；态度偏向灭法的星族与魂族也被慢慢收拾，从超一线差点滑落成一线**——⚠ **唯独恶魔族清算＋打压了这么多年，还是超一线，反而完成了足够的积累**（⚠ 「不脏手地清除内部异己，还乘上胜利一方的大船」·⚠ 现在看来它们的野心是:**第三纪元，属于恶魔族**）。
