# 风王子（奥法永恒星·风水融合法师）

[虚空角斗场](../世界/任务世界/恶魔古堡游戏.md) 上 [苏晓](苏晓.md) 标记的劲敌之一——来自施法者圣地「**奥法永恒星**」的初级法师、**凛风王的次子**、虚空新一代里名气不小。让他扬名的不是实力、而是**把风元素与水元素「完美融合」**（[莉莉姆](莉莉姆.md) 蔑称他「书呆子 + 温室花朵」）。

## 风水融合 · 灭法必杀（十四卷58、66–67章）

- **科研派法师**：他长年宅在家族法师塔钻研元素变化（掌握法术后离塔不超五次），其父忍无可忍才把他扔进死亡率极高的角斗场「见世面」、并委托好友 [瑟菲莉娅](瑟菲莉娅.md) 暗中保护（王子般待遇）。
- **风 + 水的完美融合**：这在灭法之影看来「**不是破坏元素平衡那么简单、而是从本质上改变体内的元素构造**」——若在灭法者的时代，他是**必杀目标**。融合产物「**极纯水**」：硬度远超金属 + 风元素赋予的强穿透 + 两元素相合产生的低温（中 5 滴以上即冻成冰坨）；他还能**远程操控**被打飞的水滴变向偷袭，并给水滴附「轻灵 / 急速 / 无声 / 破空祝福」四增益——科研派法师不甩死板火球水弹，专做这类阴狠精准的杀招。
- **死斗混战**：最终十人混战里，他拉拢石头人 / 铁男 / 红蝎子四人联盟围杀苏晓（判定苏晓是场上最强）——却被苏晓一记「破空」先斩杀铁男、打乱阵型。

## 关系
- **[苏晓](苏晓.md)**：角斗场劲敌 / 死斗混战中合围他的法师（灭法之影本是他这类「改变元素构造者」的天敌）。
- **[瑟菲莉娅](瑟菲莉娅.md)**：其父之友、奉命暗中保护他。
