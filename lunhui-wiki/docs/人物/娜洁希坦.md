# 娜洁希坦

**娜洁希坦** 是 [斩·赤红之瞳](../世界/任务世界/斩赤红之瞳.md) 世界**革命军暗部「夜袭部队（Night Raid）」的首领**（四卷23–24章）。

## 简介
独眼独臂的**白发御姐**——断臂以一条墨绿色金属手臂代替、瞎掉的眼睛戴黑色眼罩；常夹着黑色香烟。守序中立、行事沉稳，下令多以**商谈**而非强命的方式。是少数清楚 [艾斯德斯](艾斯德斯.md) 恐怖的人。

得知大臣**嫁祸夜袭部队**（栽赃其刺杀忧国良臣、以离间民心、引夜袭出动）后，她不愿再失去这些日后建新帝国的人才，遂分兵护官、并教训冒名者「杀手的规则」：布兰德 + 塔兹米一队、[赤瞳](赤瞳.md) + [拉伯克](拉伯克.md) 一队、[雷欧奈](雷欧奈.md) 去打探艾斯德斯动向（她忧心艾斯德斯之危）（四卷24章）。

## 死亡 · 求助途中遭遇（四卷34–40章）
「讨伐苏晓会议」后，她自知夜袭现有战力已不足以对抗帝都暗部，遂连夜**离队外出求助**、并刚在革命军本部获得人形生物帝具「[电光火石·须佐之男](../力量体系/装备道具/index.md)」的使用权（尚在调试）。命运弄人——她返回**革命军总部**地下室找管帝具的「臣具少女」时，**恰好撞上正洗劫帝具库的苏晓**；虚弱的她来不及说话即被苏晓**一刀腰斩**，临终强喊吸引同伴、令须佐之男拖延，旋即气绝。她作为「剧情关键人物」死亡，给苏晓 **3.9% 世界之源**（→12.2%）、掉落宝箱（四卷39–40章）。
