# 埃迪·克（CP0 · 妮可·罗宾的童年阴影）

世界政府最高谍报机关 **[CP0](../势力/index.md)** 的成员——身高约五米的白袍面具巨汉。CP0 统领整个 CP 部门、既是**天龙人的尖牙也是坚盾**，传言其中强者实力不弱于海军大将。埃迪·克曾为追查 **妮可·罗宾** 的踪迹**屠戮过一个小国**，是罗宾的「童年阴影」。

## 渡海追杀苏晓（十八卷76–77章）

[苏晓](苏晓.md) 三枪狙杀三天龙人后，世界政府不再像原著只派黄猿镇压、而是出动 CP0 三人组（5 米的埃迪·克 + 3 米者 + 1.6 米的女特务）追凶。女特务靠**透支体力的能力**定位苏晓在海上，三人**踩月步渡海**追击——却追不上苏晓那艘装了 300 马力马达的改造小渔船。渔船散架后苏晓转「水下逃脱」、骑海王类潜遁，两名特务下水搜寻无果，埃迪·克只能骂一句「混蛋」。

## 关系
- **[草帽海贼团](草帽海贼团.md) / 妮可·罗宾**：罗宾的童年阴影（曾为追她屠灭小国）。
- **[苏晓](苏晓.md)（白夜）**：奉命追杀的「杀天龙人凶手」（海上渡海追击未果）。
