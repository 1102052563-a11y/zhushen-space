# 埃迪·克（CP0 · 妮可·罗宾的童年阴影）

世界政府最高谍报机关 **[CP0](../势力/index.md)** 的成员——身高约五米的白袍面具巨汉。CP0 统领整个 CP 部门、既是**天龙人的尖牙也是坚盾**，传言其中强者实力不弱于海军大将。埃迪·克曾为追查 **妮可·罗宾** 的踪迹**屠戮过一个小国**，是罗宾的「童年阴影」。

## 渡海追杀苏晓（十八卷76–77章）

[苏晓](苏晓.md) 三枪狙杀三天龙人后，世界政府不再像原著只派黄猿镇压、而是出动 CP0 三人组（5 米的埃迪·克 + 3 米者 + 1.6 米的女特务）追凶。女特务靠**透支体力的能力**定位苏晓在海上，三人**踩月步渡海**追击——却追不上苏晓那艘装了 300 马力马达的改造小渔船。渔船散架后苏晓转「水下逃脱」、骑海王类潜遁，两名特务下水搜寻无果，埃迪·克只能骂一句「混蛋」。

## 顶上战场重逢 · 殒命（二十五卷15–16章）

[苏晓](苏晓.md) 五阶再入海贼（[顶上战场](../世界/任务世界/海贼王-顶上战场.md)）后，把埃迪·克视为「卡喉之刺」（不除掉这个有追踪能力的 CP0 就没法安心参与顶上战争）——故意拷杀 [斯图茜](顶上战场配角.md)、绑走绿发天龙人出海当饵，引埃迪·克来救。

- **能力揭晓**：埃迪·克实力仅比大将弱一筹、但能力比大将诡异得多（暗杀更强）——「生命归还·气态」可缩小体型、用红色气泡把敌人体内**水分束缚剥离**（强行挣脱即水分与身体分离）；全身裹「气态血肉」（使徒之眼只能侦测到外层「气态生物结构」、穿不透）；被斩下的血肉能化气泡复用。
- **苏晓的死局**：把天龙人锁在甲板下的麻醉剂蓄水池里——埃迪·克为完成「救天龙人」任务被迫登船，水中麻醉剂经皮渗入瓦解其防御；苏晓又用传说级道具 **[精神污染](../力量体系/装备道具.md)** 让他短时思维混乱、冒然行动。最终 [放逐](../力量体系/装备道具.md) 洞穿其手心 / 心脏、斩龙闪贯穿胸腹，他自扯左臂仍救出了天龙人（「任务……完成」）、被苏晓一刀斩喉（世界之源 +5.3% 累至 16%·噬灵者 +30 法力·史诗级宝箱）。从登船起他就知道「无关实力、敌人早备好死局」。

## 关系
- **[草帽海贼团](草帽海贼团.md) / 妮可·罗宾**：罗宾的童年阴影（曾为追她屠灭小国）。
- **[苏晓](苏晓.md)（白夜）**：奉命追杀的「杀天龙人凶手」（海上渡海追击未果）。
