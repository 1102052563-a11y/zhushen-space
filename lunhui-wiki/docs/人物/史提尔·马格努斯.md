# 史提尔·马格努斯（魔法侧·条码脸魔法师）

[魔法禁书目录](../世界/任务世界/魔法禁书目录.md) **魔法侧**（英国清教）的魔法师——红发、脸上纹着**条形码**、叼烟。外表老成（25~30 岁、身高 2 米），实则**只有 14 岁**。与 [神裂火织](神裂火织.md) 同属一个组织、一同来「救」（清记忆保护）逃跑的 [茵蒂克丝](茵蒂克丝.md)；昨晚他先被主角 [上条当麻](上条当麻.md) 的幻想杀手「教育」过一次。

- **隐忍偷袭（十三卷17章）**：苏晓与神裂火织死战时，他早早赶到却**没露面**——而是在暗处等时机、布术式，**偷袭重伤了空间系的 [结标淡希](结标淡希.md)**（60% 烧伤）。正因结标淡希被他打伤、没能在神裂第二次放唯闪时把神裂传送走，苏晓才得以与神裂打到两败俱伤。
- **察觉「己方有问题」（十三卷22章）**：战后他与重伤的 [神裂火织](神裂火织.md) 藏进第十学区少年感化院东侧的废弃仓库。他点烟掩盖血腥味、隐隐察觉**不对劲不是出在敌人、而是出在己方**（魔法侧内部有问题的伏笔）——而苏晓的小队此时已循着塞丽娜供出的位置摸来。

## 能力 + 殁（十三卷22–23章）

精通**火系魔法**（魔禁法师以**符文为主导、法力为辅**·不沟通元素·消耗小、擅阵地战·没符文就成废渣）——最强能力是召出四米高的火焰人形「**猎杀魔女之王**」（3000 度高温）、仓库墙壁地面贴满火属性符文。可苏晓与 [结标淡希](结标淡希.md) 用**「B 计划」**破局：结标淡希军用手电筒红光在他胸前标点（给空间移动设基准）、瞬间把他**空间移动到苏晓刀下 2 米**——他来不及反应、被斩龙闪一刀斩首（猎杀魔女之王、魔法陷阱全没派上用场·死得不明不白）。苏晓得世界之源 +3.2%（→8.8%）+ 法力 +30（→3176）+ 暗紫「魔法师的符文卡」。

**他是被英国清教抛弃的弃子**：他不肯接受「撤退、暂放弃茵蒂克丝」的命令、没有败者的觉悟，主教萝拉·斯图亚特遂果断弃他、放任苏晓杀他（详见 [神裂火织](神裂火织.md) / 势力博弈）。
