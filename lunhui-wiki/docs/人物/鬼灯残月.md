# 鬼灯残月

[火影忍者世界](../世界/任务世界/火影忍者.md) **雾隐之村「鬼灯一族」**的忍者——白发、满口鲨鱼般的尖利牙齿，靠坐在 [苏晓](苏晓.md) 隔壁的牢房里。**鬼灯一族的能力是「水化」**（身体液化）。

- **入狱缘由**：他**刺杀「照美冥」（主和派）失败**被擒——自称「一对一绝对能赢、是长十郎突然出现才坏事」。他对照美冥的容貌身材念念不忘，却极度蔑视她（断言她「**现在不是、以后也不可能是**五代水影」·与苏晓「五代水影」的猜测相左·伏笔）。
- **被封水化**：脖颈戴一枚**铁质抑制项圈**（几根导管深刺入喉·一旦动用水化，封印术感应到便释放雷盾），加之被关押前已极虚弱、周围又有感知型忍者看守，**逃不出去**。
- **性情**：话痨、阴冷、不是善茬——饿到**吞食狱友尸体**（向苏晓讨同牢已死的「川木」的胳膊吃）。苏晓借他这张话痨嘴**套取雾忍村的情报**。

## 与苏晓的「缘分」
两人都是「刺杀水影失败」的犯人——苏晓（[白夜](苏晓.md)）是**刺杀四代水影失败**、残月是**刺杀照美冥失败**。残月一听「白夜」之名便认出他「就是那刺杀四代水影的疯子」。

## 结局 · 被处死（九卷4章）
五名雾忍（含两上忍）押他出地牢处死。临别他向苏晓托付一事——**「若你能出去，告诉鬼灯水月：就说我已经死了、不用来找我报仇；好处他会给你」**（鬼灯水月 = 其族人·伏笔），随即被押走赴死。苏晓没答应、却记下了这条线索。
