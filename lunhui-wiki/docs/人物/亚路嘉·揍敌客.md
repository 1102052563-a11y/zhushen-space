# 亚路嘉·揍敌客（与拿尼加共生）

[揍敌客家族](桀诺·揍敌客.md) 的**四子**——被 [马哈·揍敌客](马哈·揍敌客.md) 称为「怪胎」：他**一半是马哈的血脉、一半是「黑暗」**（类似黑暗大陆「灾难」的某种存在）。他本身并不可怕，可怕的是体内与他**共生**（而非寄生）的另一个存在——**拿尼加**，一个来自黑暗大陆、性别为女性的未知「生物」。

## 与揍敌客诅咒的关联（十六卷11章·据马哈所述）

[马哈·揍敌客](马哈·揍敌客.md) 自黑暗大陆起死回生后悠久不死、实为**剥夺至亲寿命**——而**亚路嘉的诞生正是这诅咒之源**（马哈是在亚路嘉诞生后才确知自己为何不老）。但马哈并不想消灭亚路嘉或拿尼加，而是要**切断某种联系**。关键在于：**亚路嘉的某个后代能向亚路嘉「无条件许愿」**——这是马哈「死去的希望」，也是 [苏晓](苏晓.md) 触发的隐藏任务「揍敌客家族的诅咒」的核心（详见 [全职猎人·蚁王篇](../世界/任务世界/全职猎人-蚁王篇.md)）。

## 登场 · 随队深入枯枯戮山（十六卷14–18章）

被刻意隐瞒于家族（连家族秘密都不让他知）。他**装扮、长相酷似「小萝莉」（戴发卡、长发、宽松衣物），其实是个常宅在家、对一切都好奇的男孩**。作为马哈「切断诅咒」行动的特殊人员被席巴带来，由马哈亲自抱护——进入枯枯戮山后他**反常地不惧高温（体质最弱却毫无汗珠）、却一路昏昏欲睡**。

## 关系
- **[马哈·揍敌客](马哈·揍敌客.md)**：高祖父——视他为「怪胎 / 诅咒之源」、却也借他寻求解脱。
- **拿尼加**：与他共生的黑暗大陆未知生物（女性）。
- **[苏晓](苏晓.md)**：隐藏任务里要去接触 / 处置的关键人物（尚未正式见面）。
