# 奈良鹿丸

[火影忍者世界](../世界/任务世界/火影忍者.md) 木叶忍者、奈良一族——**影子模仿术**使用者，以高智商与战场指挥见长。与 [山中井野](山中井野.md)（心转心）、[秋道丁次](秋道丁次.md)（倍化）组成环环相扣的「**猪鹿蝶**」组合。其父**奈良鹿久**（老一辈猪鹿蝶之一）随后赶到增援。

- **围攻苏晓（九卷67–68章）**：作为援军核心，他指挥猪鹿蝶配合——影子模仿术束缚 [苏晓](苏晓.md)、井野心转心夺识、丁次倍化巨人踩踏；但苏晓青钢影外放噬灭影子内查克拉**挣脱影子模仿**、心转心被青钢影驱出（[山中井野](山中井野.md) 精神受创）、能量盾扛住倍化巨人。鹿丸清醒判断「与他近战必死、改用远程」，但仍奈何不了苏晓。最终苏晓撒白蜘蛛炼金炸弹群炸撤退，鹿丸劝众人「暗部都追不上、别追了」收兵。
