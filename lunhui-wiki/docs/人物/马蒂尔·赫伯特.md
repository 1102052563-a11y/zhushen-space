# 马蒂尔·赫伯特（三王子·新王）

[圣索丁王国](../势力/index.md) 老国王 [巴伊·赫伯特](巴伊·赫伯特.md) 的**三子**、王位争夺「五大种子」之一——**40 岁**、穿轻便甲胄、神色阴郁、**狠辣果决、说一不二**（铁血形象）。他的领土邻近 [神之墙](../世界/地点/index.md)、常年驻守前线，**深受各战区军团长拥护**（人脉 / 领导皆五星·军方亲和力极高）；军团长 [鬣狗·赫士列特](../势力/index.md) 一向力挺他。

## 失势 → 秘密上位（十七卷36–40章）

他本是最有希望继位的两人之一，却因 [小公主遇刺案](../世界/任务世界/希亚大陆-灭法遗迹.md) 一度失势、心有不甘。罗文·哈维叛国后，他一度**主动退出角逐、请缨去前线镇守西南**（让老国王颇感意外）——但这其实是局的一部分：他**秘密联合了「铁之手」[苏晓](苏晓.md) + 执政官 乔·瓦伦丁 + 军团长 赫士列特**。在老国王命不久矣的当夜，上百禁卫军杀散议事厅外守卫、把他迎入议事厅——老国王见来的不是众望所归的大王子、而是三王子，**非但不怒、反而无比欣慰**：「**马蒂尔，你成为了我最优秀的子嗣**」（能在失势后暗中串联三名最有实权的重臣上位，正是帝王之才）。他由此成为圣索丁王国的**新王**。

> 苏晓助他上位的真实目的并非效忠，而是**完成主线「权利之冠」**（选一名子嗣使其继位）；同时他要新王的「赫伯特血脉」替自己**打开「[王座之下](../世界/任务世界/希亚大陆-灭法遗迹.md)」的机关**、找灭法遗迹的「灭之石」。

## 新王 · 与苏晓的默契（十七卷41–45章）

老国王临终点破他「用 5 年才渗透禁卫军、当初我只用 2 年」（这老狮子看透了所有子嗣、唯独看不透苏晓）。苏晓掐死老国王时他**不是不想杀、而是不敢杀**（老狮子太可怕），由此明白这两人「一个看透一切、一个没有不敢杀的人」。他没与苏晓翻脸——**需要铁之手的身份震慑权臣**（完全掌权要半个月到一两个月）；在那之前双方**互不干扰**（苏晓只要不碰军权 / 财政、就算把公主肚子搞大他都不眨眼，甚至会借机剥其铁之手职、改授闲职）。这正是苏晓选他的原因：他**手握大军、懂军队**，新王是他则前线稳、死灵短期打不过来，军团长 [赫士列特](../势力/index.md) 等必受重用。改口极快的 [乔·瓦伦丁](../势力/index.md) 转投他门下。苏晓临走撂下一句「近期我会进王座之下、**借你妹一用**」（指公主 [艾丽莎·赫伯特](艾丽莎·赫伯特.md)）——为稳住苏晓，他牺牲一个妹妹眼都不眨。

## 即位之初的烂摊子（十七卷69章）

他坐上修好的王座（王座下是通希亚圣殿的深井·别扭却也没钱修新议事厅），翻着战报眉头紧锁——**王国财政是个空壳子**（罗文留的烂账）、**前线连连告急**（死灵族如疯了般猛攻神之墙·赫士列特那边尚稳、另一条战线吃紧）。正当此时，议事厅的王座被苏晓一行从「王座之下」破开顶飞——他们带着冰蓝战锤等满载而归。

## 关系
- **[巴伊·赫伯特](巴伊·赫伯特.md)**：父王（临终欣慰认可他为最优秀的子嗣）。
- **[大王子](大王子.md)**：竞争对手（曾并列大热门·互看不顺眼·大王子为补财政而退出）。
- **[苏晓](苏晓.md)（铁之手）**：秘密盟友（苏晓助其上位以完成主线·并借其血脉进王座之下）。
- **乔·瓦伦丁 / [鬣狗·赫士列特](../势力/index.md)**：拥立他的执政官 / 军团长。
