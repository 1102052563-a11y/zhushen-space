# 小助手（贝蒂·安妮特 · 苏晓的战场观察手）

[海蓝星世界争夺战](../世界/任务世界/海蓝星-世界争夺战.md) 中 [苏晓](苏晓.md)（库库林·白夜）的战场观察手——格雅联盟十二团**最优秀的观察手**，真名贝蒂·安妮特。1 米 5 出头、棕色短发、偏中性打扮（夜里常被当成少年）、满身火药味、是个**话痨 + 活力少女**（能徒手击杀四米高棕熊）。靠肉眼即可把敌人位置精确到 10 米误差内（不停地「南，173 度，670 米，风速 3.2……」报数，苏晓只回一句「闭嘴」）。

- **「克搭档」**：她曾与 27 名狙击手合作、那些人「坟头草都三尺高了」——为此一度抑郁，被团长一顿「长官的毒打」（打断两根肋骨 + 小臂骨折）后反而康复、重新活力满满。听说要给「白鬼」当观察手时她欢呼雀跃（白鬼 = 杀戮部队最优秀狙击手·与第一军团暗杀部队的宁科夫齐名）。

## 随苏晓深入敌后（十九卷21–29章）

苏晓本拒绝观察手、被斯卡因硬塞了她。她会太阳王国方言（苏晓只会听不会说）、这成了她「赖」着跟苏晓深入敌营的筹码。一路被苏晓嫌弃「跟着我不超两天你就会死」、却也屡屡被他从炮火 / 鳄鱼潭里捞出来；她则贡献了精准测距、白旗诈降等本事。在圣利门要塞攻坚战中继续给苏晓做观察手。

## 夜潜圣利门要塞（十九卷30–34章）

夜潜要塞时她大显身手：用太阳王国方言 + 蓝美瞳色诱一名副修司带路、在厨房果断捅死（「抱歉，我不是你的菜」——她还嫌对方想睡她「当然要多捅几刀」）；混进士兵宿舍演技不漏破绽；提着高温通红、随时会炸的 N 型反坦克雷往敌群扔（「请你们吃铁菠萝」）。**心理阴影**：她当年患抑郁症的真正原因，是同小队的好友都死在 [晨曦女神](晨曦女神.md) 的降临虚影手下——再见那虚影她不惜下决心冲到苏晓身前殿后掩护（却被苏晓的刀魔三两下解决）。她还跟苏晓立了个「任务成功送把特种纹螺钢匕首」的约定。

## 乘飞机下方追神使（十九卷36–40章）

苏晓在五层缠住巴克时，她追逃跑的神使 [杜波·依斯](杜波·依斯.md)——没能拿下、索性**乘在杜波的飞机下方**一路追踪（吞下苏晓给的信号发射器、在要塞顶刻字「已乘在飞机下方」报信、靠某种法子没被高空低温冻死）。最终把苏晓引向太阳王国境内的安贝拉城（她藏在城市下水道）。

## 关系
- **[苏晓](苏晓.md)（库库林·白夜 / 白鬼）**：她崇拜并死缠的「白鬼」狙击手（苏晓嫌她拖后腿、却也护着她）。
- **[杜波·依斯](杜波·依斯.md)**：她乘飞机下方一路追踪的太阳王国神使。
- **[晨曦女神](晨曦女神.md)**：杀死她同小队好友、令她患抑郁的太阳王国之神（其降临虚影）。
- **[斯卡因](斯卡因.md)**：把她硬塞给苏晓当观察手的杀戮部队团长。
