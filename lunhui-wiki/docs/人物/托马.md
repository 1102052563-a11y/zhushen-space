# 托马

[黑色森林](../世界/任务世界/T12K星球-生存试炼.md) 里另一名**被 [暗王](暗王.md) 同化的契约者**——据 [野猪](野猪.md) 所述，他原是「一名两米多高的壮汉」，被同化后**变成一只「硬壳王八」**。

- **处境**：与野猪同属那批深入黑色森林失败、被同化的契约者（共五人·三个撞黑雾自杀·只剩野猪与托马）。托马**已到崩溃边缘、近期正「花式自杀」**（被同化者失痛觉近乎不死、求死不能）。
- **危险性**：野猪庆幸 [苏晓](苏晓.md) 一行遇到的是自己而非托马——「如果遇到的是托马那家伙，他一定想方设法杀掉你们」。

> 尚未正面登场，仅由野猪提及。
