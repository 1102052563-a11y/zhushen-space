# 银皇后

⚠ **她的身体富有流线的美感，形体与人族相似**，⚠ **身体的部分位置有银色生物外甲**；⚠ **头发就像一把把后曲的刀锋般**；⚠ **右手背上，有一只深邃的暗绿圆瞳**。⚠ **一双银色竖瞳**——性格**孤傲**。

## 🌍 她的黑历史：把自己的世界撑裂了

⚠ **在很久之前，银皇后也是诸多虫族女王中的一员**——⚠ **所有虫族女王都有一个期望，就是成为万界虫族的主宰**；⚠ **曾经的银皇后，几乎要做到这点。**

- ⚠ **虫族的铁律**:⚠ **无论怎么发展族群，都不可让自己所掌控的世界崩灭**——⚠ **虫族的观点是延续与进化，而非毁灭。**
- ⚠ **她的困境**:⚠ **她夺下这世界，是与多名虫族女王交锋的成果**——⚠ **可当她成为最后的胜者时，这世界已经因为虫族大战而千疮百孔**;⚠ **换个世界固然可以，但抛弃这曾让她崛起的世界，必然会让万界的其他虫族不耻**——⚠ **更别说成为虫族主宰了**（⚠ **哪怕这世界原本就没有其他生灵，只有虫族**）。
- ⚠ **虫族主宰不只是个头衔**:⚠ **除了棘拉这种特殊虫族女王，其他虫族女王都要听从主宰的指令**——⚠ **需要让所有虫族女王都承认她，她才能成为主宰**;⚠ **虫族之间其实没有尔虞我诈，只有进化的强弱之分，因此才能达成这等主宰之位的抉择。**
- ⚠ **她的「机智」**:⚠ **她想到世界的衰败无非是世界之力大量透支——只要把世界之力补充回来，让此地生机勃勃，再建立宏伟的虫族建筑，主宰之位不就稳了**;⚠ **于是她建立起虫巢巨舰，在世界间的虚无之间行进，专找那些有世界之力、但没有生灵且即将崩灭的世界**（⚠ **不去世界之力充沛、生灵众多的，那会引来大量麻烦**）。
- ⚠ **结局**:⚠ **在几百年的努力后，她所掌控的那个世界里面充满了世界之力**——⚠ **然后那个世界裂开了**;⚠ **具体因为什么，银皇后自己也不知道**——⚠ **她明明查阅了大量的记载，说这方式可以让世界恢复生机。**
- ⚠ **她的意识被世界崩灭时的「噬灭黑洞」挤压到起源石之中**——⚠ **因在虚无之间飘游了几千年，她最终因孤寂感放弃了思考**;⚠ **随后 [苏晓](苏晓.md) 以「起源石随机抽取权限」获得【起源石·银皇后】**，⚠ **因回到物质世界，她的意识逐渐苏醒**（⚠ **是 [棘拉](棘拉.md) 建立虫巢时飘散的生物能唤醒了她**）。

## ⚔ 半米距离的和解（51卷21–22章）

⚠ **她每天都恨苏晓恨得牙根痒痒**——⚠ **不仅因为被放逐到此，还因为之前完全落入对方的陷阱**;⚠ **对于性格孤傲的银皇后而言，这是此生之耻。** ⚠ **就在她想着「倘若那灭法敢出现在这，她就将对方灭杀当场」并为此忍俊不禁时**——

- ⚠ **[布布汪](布布汪.md) 现身，黑蓝色烟气构成空间坐标**;⚠ **从烟气浮现到空间传送，全程不超 1.5 秒**——⚠ **苏晓以蹲姿落在她的寝床上，距离不超半米。**
- ⚠ **她的心态彻底崩了**，说出了她自己都不信的话:

> ⚠ 「**我虽是虫族女王，但近身战能力其实很强，所以我劝你不要轻举妄动。**」

- ⚠ **她保住孤傲的最后一分努力**:落座后解释「**请不要误会，我并不是怕了。**」——⚠ 苏晓「**嗯，我知道。**」（⚠ **她已经准备好身死于此，结果对面这灭法者的态度，却让她大感意外**）
- ⚠ **有时，彼此间的矛盾需要用距离来调节**:⚠ **之前她恨苏晓恨得咬牙切齿，可现在两人相距半米，她甚至会对苏晓微笑**——⚠ **这矛盾调节的，可谓是相当之好。**

## 🗣 「我说，你听」——负 21 魅力的交涉

⚠ 「**我说，你听，如果我说的对，你就回答对，反之，你就回答不对，有问题吗？**」——⚠ 「**有问题。**」⚠ **铮～长刀出鞘，抵在银皇后白皙的脖颈上**:「**现在呢？**」——⚠ 「**看起来……没问题了。**」

| 苏晓的论证 | 她的回答 |
| --- | --- |
| ⚠ 不是我用「起源石随机抽取权限」，你现在还在轮回乐园的物资库存里 | 对 |
| ⚠ 我获得起源石是否触及你的利益？是我把你变成起源石的吗？ | 不是 |
| ⚠ 到这个时间点，我没对你造成任何损害，反之是我把你兑换出来 | 对 |
| ⚠ 是棘拉虫巢飘散的生物能把你唤醒 | 对 |
| ⚠ 你被唤醒后第一时间隐匿、伺机加害于我 | ⚠ 「这……**对**。」（⚠ **突然莫名的来了几分心虚**） |
| ⚠ 我发现你的恶意却没格杀你，而是放逐——你认为我当初没能力格杀你吗 | ⚠ 「你……**有这能力**。」 |

- ⚠ **结论**:「**我之前说，你应该感谢我，我这话有问题吗。**」——⚠ 「**有问题！我到了这里后，多次险些丧命。**」⚠ 「**嗯，这是当然，不过我为什么把你放逐到这？我这个人，记忆力不太好，你帮我回忆回忆？**」——⚠ 「是因为……」（⚠ **说不下去了**）⚠ 「**你意图谋害我，我把你放逐到永光，随后你在这里有几次险些丧命，这么说来，我们之间的恩怨就扯平了，你说对吗。**」——⚠ 「**似乎，对……吧。**」
- ⚠ **⚠ 谁说 −21 点魅力属性没有交涉作用？**——⚠ **眼下的银皇后，显然已被苏晓精妙的口才说服。**
- ⚠ **她被气到的名场面**:苏晓实话实说「**放逐你来永光，是提前来帮棘拉找资源地，你选的地方很不错，做得好。**」（⚠ **毫不吝啬夸奖**）——⚠ **银皇后错愕了下，随后气的瞳孔都开始缩小**:「**你，立即，马上，离开我的领地。**」⚠ 苏晓取出【封之刃】作势要走:「可惜，**原本想在与你合作后，送你离开这里**。」——⚠ **她立即犹豫了**:「**等等，你真的准备，在合作后放我出去？**」⚠ 「**当然，否则继续把你关在这，累积对我的仇恨度吗。**」——⚠ **听到此言，银皇后的怒气＋2 点**;⚠ **她都不知为何，对面这灭法者，明明每句话都是事实，却都能把人气的半死**:「**你平常都是这样和别人交涉的？你不怕挨打吗。**」⚠ 「**并不，多数人与我交涉时，礼貌又谦逊。**」——⚠ 「**怎么可能，他们……额～，好像，的确会是这样。**」（⚠ **她想起方才对方把刀架在她脖颈旁**）
- ⚠ **她离死亡最近的一次**:⚠ **苏晓布设阵图时她曾试探「你，不怕我无视契约反悔？只是一次死亡的话，我承担的起，我的命魂，不止一颗」**——⚠ **或许银皇后永远都想不到，这是她漫长的生命中，距离彻底死亡最近的一次**;⚠ **但凡她敢下达围杀苏晓的命令，0.1 秒后，苏晓能让银皇后立刻死透。**
- ⚠ **苏晓为何笃定她不会反悔**:「**你是虫族吗。**」——⚠ 「当然。」⚠ 「**虫族为了延续与进化，不会出尔反尔，否则，你认为我上次为什么不直接灭杀你。**」

## 🕯 无光神殿的拜访

⚠ **神殿祭祀长·[厄兹勒](永光世界/厄兹勒.md)**（⚠ **其实是 [神父](魔灵星/神父.md) 冒充**）来找过她:「**到现在为止，你们无光神殿，还没找到那灭法者的踪迹？**」——⚠ 「**当然不是，其实我的手下，很快就能找到那灭法者，但我并不想找到他。**」

- ⚠ 「**这么说，你想背叛无光神殿？**」——⚠ 「**只要能离开这里，值得**……**我们庇护城的居民，没得选，要么加入，要么成为饵食。**」
- ⚠ 「**我听说，你和那灭法是熟识？**」——⚠ 「你……听谁说的。」⚠ 「**苦痛女王，虽然它已经被 [蛀世](永光世界/蛀世.md) 吞食掉，但在那之前，我们的合作一直很愉快。**」（⚠ **听到蛀世，银皇后目光严肃了几分**——⚠ **她与蛀世交锋过一次，结果是派出的虫族军团很快消失，就如同被那区域的世界一口吞食了般**）
- ⚠ 「**我和那灭法者，是仇敌。**」——⚠ 「**仇敌也算熟识，如果可能的话，我们应该和那灭法合作。**」⚠ 「**不可能。**」——⚠ 「**怎么不可能，只要能离开这囚笼，一切，都有可能。**」（⚠ **此言让银皇后沉默**）
