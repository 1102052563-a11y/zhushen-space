# 富兰克林

[全职猎人世界](../世界/任务世界/全职猎人.md) **[幻影旅团](../势力/index.md)（蜘蛛）成员·编号 7**——身材高大、耳垂极长的壮汉。**放出系念能力**：十指打开后可发射念弹。性格成熟稳重，是旅团的**和事老**。

## 历程 · 袭击拍卖会（十卷27–30章）

奉团长 [库洛洛·鲁西鲁](../势力/index.md) 之命，与 [飞坦](飞坦.md)、[小滴](小滴.md) 凿墙潜入友克鑫拍卖会地下室（任务是吸引黑帮注意、掩护窝金等四人抢拍品）。

- 力主**不杀那名「全知全能契约者」**、改为拷问取情报（默许飞坦动手）。
- 危急时一脚把队友 [小滴](小滴.md) 踢飞、救她避开 [苏晓](苏晓.md) 的斩首一刀。
- 见苏晓强、留飞坦断后，自己带小滴先撤、去与窝金等人会合（确认宝库已空）。

## 关系
- **[幻影旅团](../势力/index.md)**：编号 7 成员；与 [飞坦](飞坦.md)（编号 2）、[小滴](小滴.md)（编号 8）同行。
- **[苏晓](苏晓.md)**：敌对。
