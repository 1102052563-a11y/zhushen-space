# 咕噜（幻影旅团·编号6）

**幻影旅团**（苏晓的同类·PVP 冒险团）成员、**旅团编号 6**。外表是个**红色童装、叼棒棒糖的小女孩**（手背纹着阿拉伯数字「6」），却**喜欢吃糖、更喜欢杀人**——死在她手中的契约者**超过三位数**；喜怒无常、发起神经连同旅团的 [左轮男](左轮男.md) 都可能被她杀掉。

- **与茜拉**：三个衍生世界前曾与 [茜拉](茜拉.md) 交手、**一只手已抓住茜拉的心脏**（「你的心脏真是柔软呢」），靠 [神皇](神皇.md) 及时赶到才没得手——故八卷9章拍卖会上她凑到茜拉身后一句「大姐姐，我能杀掉你吗」吓得茜拉色变。
- **与苏晓**：八卷9章随左轮男「不请自来」乐园拍卖会，对苏晓（爆了左轮男膀胱、卖了卡尔编号的人）和那只「好萌」的二哈 [布布汪](布布汪.md) 起了兴趣。
