# 长十郎

[火影忍者世界](../世界/任务世界/火影忍者.md) 雾隐之村的一名**戴眼镜的少年忍者**，**[照美冥](照美冥.md) 的护卫**——背着一把双柄、鱼型的奇形兵器「**鲆鲽**」（**雾隐「七忍刀」之一**）。

- **性情**：平时容易紧张、有些不自信，但护卫照美冥时面色平静、尽职——面对「刺杀四代水影的恶徒」[苏晓](苏晓.md) 也强自镇定。
- **作用**：照美冥与苏晓在水影办公室谈判时，长十郎**一直站在苏晓身旁**——职责是**万一苏晓暴起、只需挡他片刻，照美冥即可出手解决**。残月被擒也与他有关（[鬼灯残月](鬼灯残月.md) 自称刺杀照美冥时「是长十郎突然出现才坏了事」）。
