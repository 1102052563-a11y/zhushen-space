# 老紫

[火影忍者世界](../世界/任务世界/火影忍者.md) **四尾人柱力**——约 50 岁、满脸火红络腮胡、红发短辫、云游僧打扮，岩忍护额无划痕（非叛忍·只是与三代土影理念不合离村）。**操控岩浆（熔盾）**——四尾擅熔盾，故藏身于土之国大川谷地下火山的**岩浆湖**附近（那是它的天然主场）。

- **和善的人柱力**：成为人柱力前是感知型忍者、能从查克拉性质感知善恶；查克拉平和暖人。当年因**不愿把一名少年培养成下届人柱力**（人柱力命运悲惨）而拒绝三代土影、离开岩忍村——有同情心，自知「不是合格的忍者」，遂抛去忍者身份当云游僧。长期苦修使他自信变自大、不知世间有「晓」。
- **被攻略经过（九卷56–57章）**：[苏晓](苏晓.md) 三人不去岩浆湖、而用 [香磷](香磷.md) 当诱饵（苏晓割伤、开枪逼她真喊救命）——老紫感知到善良的香磷遇险、被骗出主场。战中苏晓近身（老紫感知不到其体内查克拉、误以为他只是刀术好）一刀斩破尾兽衣 + 斩开脊背、[宇智波鼬](宇智波鼬.md) 万花筒幻术定身、[干柿鬼鲛](干柿鬼鲛.md) 水遁克熔盾——三人「近战 + 坦克 + 远程辅助」围攻下，老紫没用出真正实力就落败。
- **被吸本源 → 封印致死**：苏晓以**吞噬之核**吸四尾的本源之力（四尾不求饶、奋力挣扎仍被困）→ 青钢影由 Lv.20 连升至 **Lv.26**（四尾骨瘦如柴）。老紫被运至铁之国，与三尾 / 六尾一同被外道魔像封印 → **尾兽被剥离后死亡**。
