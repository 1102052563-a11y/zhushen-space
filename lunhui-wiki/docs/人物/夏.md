# 夏

**夏**（单字名）是 [轮回乐园](../总览/乐园.md) 内一间小餐厅的厨娘、**职工者**（四卷1章）。

## 简介
二十多岁，身材不高却不错，长相不算漂亮却越看越顺眼；性格善良、有点话痨。她的餐厅只有三张桌、装饰温馨暖色调，因位置偏僻客人不多——苏晓（化名白夜）在乐园的伙食多在此解决，二人因此熟识。她进入的衍生世界难度已升至 **Lv.5**（职工者中不算低），曾在喰种世界险被 A 级喰种吃掉。

得知苏晓在衍生世界杀过约两千人后仍待他如常，认定常照顾自己生意的他「是个大方的好人」；见他长期绷紧、神经疲惫，**送他一台画质精美的解谜求生游戏平板**助其放松。苏晓则承诺「若日后进入同一衍生世界，保她一命」（四卷1–2章）。

## 能力 ·「魅力厨娘」（十卷70章揭示）

夏不爱战斗、只钻研菜谱（常找 [布布汪](布布汪.md) 当陪练自保），却有一身**烹饪奇能**——

- **魅力厨娘 Lv.20（被动）**：烹调特殊食物时，有几率为食物附加三种效果中的二种——① 体力恢复 ② 美味佳肴 ③ 此次食物**进化为恢复类食物**（可回生命、可出售）。Lv.10 附加「**菜谱开发**」（自创新菜谱）。
- **Lv.20 附加「美食女神」**：自身食用各种「**食物类恢复品**」可**增强战斗力**（仅对她自己有效）——白品 +1 三维 / 绿 +2(智+1) / 蓝 +4(智+3) / **紫品 力敏体智 +7、魅 +3、持续 3 小时**（紫以上按紫计、技能升级可提加成上限）。

十卷苏晓全职猎人通关返园后到她店里大吃一顿，发现这些菜竟能**恢复强化属性的疲劳、小幅增身**——美味之余还有奇效，只收他 395 币成本价（嫌「常让布布当陪练不好意思」）。她婉拒一切大型冒险团的招揽，只图安静钻研、潇洒度日。

## 现实世界遇险 ·「救命」蓝色邮件（十九卷98–99章）

夏与同伴「学姐」（眼镜妹·成熟丰满·36D）接了一份**强制任务**、在**现实世界**执行时撞上一伙二阶契约者「杀手集团」追杀，一路躲进边境草原的牛棚（被不友好的奶牛一直踢），向 [苏晓](苏晓.md) 发出一封只有「救命」二字的**蓝色邮件**（邮件颜色对应发送位置·蓝色 = 现实世界·详见 [轮回乐园机制](../总览/轮回乐园机制.md)）。千里之外的苏晓赶不及，便就近调来 [黑血](黑血.md)（铭门冒险团）——黑血把两名追杀者「一不小心吃了」，随后 [炎辰](炎辰.md) 率铭门赶到善后、把她们送往附近城市。事后夏请苏晓到她餐厅吃大餐答谢救命之恩。此事也印证了**契约者一般不在现实世界冲突**（夏因接强制任务才例外遇险）。

## 关系
- [苏晓](苏晓.md)（白夜）——常客、熟人；苏晓难得与之闲聊放松，并许下「同世界则保命」之诺；十九卷其现实遇险，苏晓调 [黑血](黑血.md) / [炎辰](炎辰.md) 相救。
- [布布汪](布布汪.md)——常找它当陪练（战斗力因此大进、但仍逊于厨艺）。
- [黑血](黑血.md) / [炎辰](炎辰.md)（[铭门冒险团](../势力/index.md)）——十九卷在现实世界救她性命的恩人（苏晓所托）。
