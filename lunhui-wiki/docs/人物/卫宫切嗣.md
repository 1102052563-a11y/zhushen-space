# 卫宫切嗣

[Fate/Zero 世界](../世界/任务世界/fate-zero.md) 中 [Saber](Saber.md) 的御主——三十岁左右、一身黑（惯于夜间暗杀魔术师），人称「**魔术师杀手**」。**信奉「以最小损失取胜、牺牲少数救多数」的功利主义**，行事狠辣、不择手段（与重骑士道的 Saber 相性极差）。

- **起源弹**：他曾取出自己几根肋骨、磨粉后以灵魂工程凝成弹头——被这种子弹命中的魔术师即便不死、**体内魔术回路也会被大量破坏**。
- **战术家**：[苏晓](苏晓.md) 接触老神父后被他盯上，他主张「**先观望、让 Lancer 阵营与 Berserker 阵营先冲突**」、坐收减员之利；并**一眼看穿苏晓「御主 / 英灵身份对调、令咒是画上去的」**。他战略性撤退正是想让苏晓多杀几名英灵、减少竞争对手。
- 妻子 [爱丽丝菲尔](爱丽丝菲尔.md)（圣杯容器·人造人）、忠诚的女助手（被他从战场救下、近乎其「武器」）随行。
- **固有时制御**：卫宫家代代探索的**时间操作魔术**——加速自身个人时间流速（血红蛋白高速燃烧供能）、可三倍速 / 四倍速爆发闪避，是他最强的保命手段（但身体是普通御主、透支极伤）。

- **谈联手 · 错失时机（十一卷50章）**：荒焚一伙三阶契约者退场后，他才**用「未知号码」打电话**给苏晓谈联手（沉默对峙三分钟才开口·他不在乎名声、只看结果）——苏晓用「**让出圣杯归属**」为饵诱他现身（其实三阶威胁已除、苏晓不再需要他）。席间 [Saber](Saber.md) 斥苏晓「邪魔外道」、被苏晓「骑士等于蠢」激将；他听出苏晓并无诚意、挂断。苏晓评他「**考虑太久、错过了一个机会**」（一天前来电的话尚有合作可能）。

## 死于更强的狙击手（十一卷62–63章·殁）
他没有远离冬木市、反而藏在 **爱因兹贝伦家城堡**（机关重重 + 人造人女仆 + Saber 守备）内——被苏晓用临时技能「大规模探测」锁定。苏晓不正面闯城堡，而是布下狙击阵（1 公里外古树 + 清理城堡警戒用的炼金生物、耗时数小时）：以 [使徒之眼](../力量体系/装备道具.md) 探位、[蜘蛛女皇](../力量体系/装备道具.md)（暗紫）+ 枪术精通 Lv.30 **穿墙狙击**。他凭战斗直觉躲到承重墙后、连开**固有时制御·三倍速 / 四倍速**堪堪偏头（仍被擦掉半边脸颊），却被苏晓的特殊弹药（尾翼减速「震撼弹」·音波震荡）打懵——**上半身、下半身接连被 17.36mm 巨弹打碎而死**（这位惯于狙杀魔术师的「魔术师杀手」、终究死在更强的狙击手枪下）。苏晓给 [Saber](Saber.md) 出了选择题（追击则代理御主 [爱丽丝菲尔](爱丽丝菲尔.md) 死），正直的 Saber 选择守护爱丽丝菲尔、放他离去。世界之源 +3.1%→80.6%。
