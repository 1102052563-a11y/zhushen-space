# 温莉·洛克贝尔

主角 [爱德华·艾尔利克](爱德华·艾尔利克.md) 的**青梅竹马**（为爱德华维护机械义肢的技师）。

- **历程**：七卷36章苏晓手中拿着一条**温莉的挂坠**离开布里克兹要塞——[阿尔冯斯](阿尔冯斯·艾尔利克.md) 认出后，兄弟俩误以为苏晓加害了温莉、暴怒欲追，最终强压怒火改去**寻找温莉**、再赶往中央市阻止战争。（此为苏晓有意留下的「钓饵」，引兄弟与父亲卷入最终之战。）

## 出处
- 七卷36章（提及）。
