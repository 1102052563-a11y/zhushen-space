# 铁鼠（血门主坦 · 重力侏儒）

[血门冒险团](../势力/index.md) 的**主坦**、[斯坦](斯坦.md) 的心腹之一——全身包裹铠甲的侏儒，门牙突出如啮齿动物（人称「铁老鼠」），在团内话语权极高。战斗经验老辣、防御与防击退能力是四阶顶尖。

- **重力 / 臂盾**：主能力是**重力场**（半径十几米地面凹陷·身处其 5 米内重力数倍 + 吸附力·苏晓速度降 60%+）+ 一对传说级 +11 臂盾（直径半米·花血门大笔团资），还能召磨盘大的**暗重力拳**砸人。这套重力能力其实是 [斯坦](斯坦.md) 通过金属甲片「借」给他的（即斯坦暗金属领域的分支）。
- **被斩杀**：圣杯世界肃清队一战，他想缠住 [苏晓](苏晓.md)，却被「刀刀破盾」的 [斩龙闪 史诗级 +10](../力量体系/装备道具/index.md) 连斩——臂盾十几刀就被斩碎、最后被一刀斩颈瘫坐而亡（没等到斯坦来援·也没来得及传回敌情）。「打团先秒坦」的反面教材：主坦最悲剧的不是被破防、而是敌人根本不理他。

## 关系
- **[斯坦](斯坦.md)**：团长 / 主公（重力能力实为斯坦甲片所赋）。
- **[苏晓](苏晓.md)**：刀刀破盾斩杀他的死敌。
- **[奈洛伊](莺澜.md) / [老烟枪](老烟枪.md)**：血门同僚。
