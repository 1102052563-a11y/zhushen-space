# 假面（小野猫 · 四阶独行侠）

香波地篇的**四阶独行侠**——黑色短发、浓黑眼影的女子，穿一身**猫科软金属轻型外装甲**（黑亮紧身衣·头盔顶两只猫耳、尾椎一条猫尾），自称「**豹子**」、最忌讳别人叫她「猫 / 小野猫」（否则要把「某段录像」公之于众）。一只「渴望变成豹的野猫」，狡诈贪婪、享受「猫抓老鼠」的快感。

- **探测标记**：稀有能力——把小型卫星送上香波地上空，**20 公里内被她探测到的敌人都会被标记 + 追踪**（外装甲探测模式下世界变黑白、生物呈红色·按生物能反应锁定目标）。
- **外装甲战斗**：弹道感知（预判普通子弹落点、在落点加强防御）、隔热、伸缩金属爪；**超燃状态**（防御 −30%、力量 / 速度 / 反应 +35%·持续 15 分钟·爪由猫爪变虎爪）；**强制导性催眠**（凭语言 / 特殊波长引导敌人·只要与之交谈 5 句以上、催眠成功率升至 55%+·削敌感知与神经反射）；还能召**虚无猫**（虚影猫扑挠敌人、走魅力属性判定）+「燃烧」火焰等。

## 香波地追杀苏晓（十八卷66–69章）

她与 [岚佐](岚佐.md)、铁坦克临时合作追杀偷走罗杰佩枪的「狙击手」[苏晓](苏晓.md)。一路靠标记追踪、却**反复算计队友**：把坏掉的平板「卖」给岚佐骗钱、想甩开众人独吞佩枪（「旅团的疯子也没什么了不起」）。单挑苏晓时——她的强制导性催眠被「**从不与敌废话**」的苏晓躲过；近战中又栽在 [D·暗杀](../力量体系/装备道具.md) 手里（无声 + 屏蔽感知、她的弹道感知预判不到落点·连中数枪、头部一枪让她暂聋）。她虽生存力极强（换别的四阶早被打死），却被苏晓 [战术探测仪](../力量体系/装备道具.md) 隔墙狙、超燃硬冲也被斩龙闪 + 能量盾压制。

**败北逃命（十八卷70–71章）**：苏晓反扑——斩飞她头盔上的猫耳（实为信号接收 / 增幅装置），她以「震荡」（脉冲震荡敌体内水分子）反制，却被苏晓**界断线**从医院顶层硬拽下来、一刀险些斩首（靠外装甲伸长缩头才保命）。多亏一名嚼口香糖的狙击手「看在狂狐老哥面子上」狙伤苏晓救场、又被岚佐坍塌的医院掩护，她才捡回一命。眼睁睁看苏晓 1 对 43 用猎魔时刻屠场、连岚佐都被杀两次后，她蜷在病床下「吓死本喵了」——自认再遇苏晓也只会被「按在地上锤到死」。
