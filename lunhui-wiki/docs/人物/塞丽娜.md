# 塞丽娜（艳花之蛇·魔法师酒吧老板）

[魔法禁书目录](../世界/任务世界/魔法禁书目录.md) 第十学区里的**魔法师**——全名「塞丽娜·加布里埃·维吉妮亚」（一长串名字·苏晓差点报成别名），绰号「**艳花之蛇**」。紫色连衣裙、黑唇膏、左耳金蛇坠，像一条色彩斑斓的毒蛇。她能力**偏向情报搜集与侦查、正面战斗弱**（施法媒介是一本笔记），开「火之森」酒吧（借迎宾优惠的人流掩护、给魔法侧的人提供庇护所）。

- **被水刑逼供（十三卷20章）**：苏晓循线找来，几十名无能力者集团的酒客持先进枪械围攻——被苏晓的信号发射器 + 狙击手 [叶戈尔](叶戈尔.md) 的重狙清场（一枪穿三人）。塞丽娜腿被轰碎，嘴硬不肯说神裂下落。苏晓**用衣服蒙她头、往布料上灌烈酒令她呛水般窒息**（比死更难受），逼出 [神裂火织](神裂火织.md) 的藏身处（第十学区少年感化院东侧废弃仓库）。事后猎犬部队善后、一个活口不留——她被灭口（其「研究价值」则被苏晓用来跟木原数多换取狙击手 [叶戈尔](叶戈尔.md)）。

## 关系
- **魔法侧**：情报 / 侦查魔法师（给魔法侧的人提供庇护）。
- **[神裂火织](神裂火织.md)**：被她供出藏身处的圣人。
- **[苏晓](苏晓.md)**：水刑逼供她的猎手。
