# 陵墓守卫者·里德（霸主级 · 中立锻造者）

[希亚圣殿（不死者陵墓）](../世界/地点/index.md) 内的**陵墓守卫者**——身高约 1 米 5、敦实、**镶满铁甲**，双眼是两个锈迹斑斑的「老式望远镜探头」，头部曾被劈成三截、用铆钉 + 大号订书钉钉合（**换头位置即可切换死灵语 / 人类语**）。他守在陵墓深处一间火红的锻造室、打着一把半成品斩马刀。**阶位是「霸主级生物」**（高于死刑官 [敦克·尼克](敦克·尼克.md) 的领主级），却是**中立阵营、友善对待任何种族**（登场便笑着「人类，你们好」）。

## 资料（使徒之眼 · 中立单位）

类别：**死灵族 1/2 + 人类 1/2**；法力 9860；**力 / 敏 / 体「???」、智 98、魅 53——五属性全是真实属性**。

- **技能1·霸主权威（被动 Lv.62）**：生命 +10 万 / 法力 +7000 / 力 +18 / 敏 +15 / 体 +20。
- **技能2·锻造（被动 Lv.58）**：可锻造**白色 ~ 传说级**品质装备。
- **技能3·捶打（主动 Lv.55）**：每次锤击附 200 真伤 + 目标最大生命 30%（体力 95 以下生效）+ 1200 伤。
- **技能4·中立（被动 Lv.MAX）**：中立阵营、友善待任何种族。
- 技能 5~10（钢铁身躯 / 威压 / 荒蛮 / 碎裂大地〔终极〕等）：均「???」未显。

!!! warning "战力悬殊"

    苏晓自估**最多挨他两锤就会死**，斩杀更是「别开玩笑」（他如金刚石般坚不可破）。布布汪也判断：领主级的敦克·尼克若与里德战斗，**不超两锤就被锤成碎肉**——霸主级与领主级之间是云泥之别。好在里德是中立、不主动为敌。

## 被忽悠送进轮回乐园（十七卷53–54章）

斩不动他、又怕他守在陵墓口碍事，苏晓改用「**等价交换**」忽悠他——里德被困这几十平米石屋几百年（最初自愿、后只想离开、形同囚牢·寂寞到曾凭肉体冲破大气层想离世），早渴望出去。苏晓拿出 [时空硬币](../力量体系/装备道具.md)、画图大谈「虚空」（里德鸭子听雷却能判真假），承诺「帮你进虚空」。里德很穷、只肯给情报作交换：**影之石已失踪多年、灭之石在陵墓深处某处、法之石他是少数知道下落的人**；他还提过自己锻过两把锤——火锤（在用）和**冰锤**（搁陵墓里、镇压一个不安分的家伙）。

苏晓布下**空间阵图**（虚空角斗场结识的女魅魔莉莉姆所教·灭法之影的友军）、设坐标恶魔古堡，趁里德试探时**激活吸力把他吸进裂缝**（借口「你身上有东西干扰阵图」、不撕破脸）。里德临入裂缝时感知到另一端「惊悚的未知存在」、骂着「无耻」被吸走——**其实是被送进了轮回乐园**（乐园会招纳剧情人物、不会拒绝里德这种锻造者·相当于苏晓付时空之力「邮费」、乐园中转截胡、收件人爱丽丝白等）。

> 【为轮回乐园获得一名锻造者】苏晓得 **3 真实属性点 + 4 万币 + 权限 +Lv.2 + [里德之勋](../力量体系/装备道具.md)**（可在锻造者里德处每 15 天无偿锻造一次装备·「请忽略下次见面时里德的态度」）。里德将以**职工者**身份在乐园安身（恨苏晓恨得牙痒、却也算被还了有限的自由）。
