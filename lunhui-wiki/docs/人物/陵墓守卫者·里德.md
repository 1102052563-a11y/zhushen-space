# 陵墓守卫者·里德（霸主级 · 中立锻造者）

[希亚圣殿（不死者陵墓）](../世界/地点/index.md) 内的**陵墓守卫者**——身高约 1 米 5、敦实、**镶满铁甲**，双眼是两个锈迹斑斑的「老式望远镜探头」，头部曾被劈成三截、用铆钉 + 大号订书钉钉合（**换头位置即可切换死灵语 / 人类语**）。他守在陵墓深处一间火红的锻造室、打着一把半成品斩马刀。**阶位是「霸主级生物」**（高于死刑官 [敦克·尼克](敦克·尼克.md) 的领主级），却是**中立阵营、友善对待任何种族**（登场便笑着「人类，你们好」）。

## 资料（使徒之眼 · 中立单位）

类别：**死灵族 1/2 + 人类 1/2**；法力 9860；**力 / 敏 / 体「???」、智 98、魅 53——五属性全是真实属性**。

- **技能1·霸主权威（被动 Lv.62）**：生命 +10 万 / 法力 +7000 / 力 +18 / 敏 +15 / 体 +20。
- **技能2·锻造（被动 Lv.58）**：可锻造**白色 ~ 传说级**品质装备。
- **技能3·捶打（主动 Lv.55）**：每次锤击附 200 真伤 + 目标最大生命 30%（体力 95 以下生效）+ 1200 伤。
- **技能4·中立（被动 Lv.MAX）**：中立阵营、友善待任何种族。
- 技能 5~10（钢铁身躯 / 威压 / 荒蛮 / 碎裂大地〔终极〕等）：均「???」未显。

!!! warning "战力悬殊"

    苏晓自估**最多挨他两锤就会死**，斩杀更是「别开玩笑」（他如金刚石般坚不可破）。布布汪也判断：领主级的敦克·尼克若与里德战斗，**不超两锤就被锤成碎肉**——霸主级与领主级之间是云泥之别。好在里德是中立、不主动为敌。

## 被忽悠送进轮回乐园（十七卷53–54章）

斩不动他、又怕他守在陵墓口碍事，苏晓改用「**等价交换**」忽悠他——里德被困这几十平米石屋几百年（最初自愿、后只想离开、形同囚牢·寂寞到曾凭肉体冲破大气层想离世），早渴望出去。苏晓拿出 [时空硬币](../力量体系/装备道具.md)、画图大谈「虚空」（里德鸭子听雷却能判真假），承诺「帮你进虚空」。里德很穷、只肯给情报作交换：**影之石已失踪多年、灭之石在陵墓深处某处、法之石他是少数知道下落的人**；他还提过自己锻过两把锤——火锤（在用）和**冰锤**（搁陵墓里、镇压一个不安分的家伙）。

苏晓布下**空间阵图**（虚空角斗场结识的女魅魔莉莉姆所教·灭法之影的友军）、设坐标恶魔古堡，趁里德试探时**激活吸力把他吸进裂缝**（借口「你身上有东西干扰阵图」、不撕破脸）。里德临入裂缝时感知到另一端「惊悚的未知存在」、骂着「无耻」被吸走——**其实是被送进了轮回乐园**（乐园会招纳剧情人物、不会拒绝里德这种锻造者·相当于苏晓付时空之力「邮费」、乐园中转截胡、收件人爱丽丝白等）。

> 【为轮回乐园获得一名锻造者】苏晓得 **3 真实属性点 + 4 万币 + 权限 +Lv.2 + [里德之勋](../力量体系/装备道具.md)**（可在锻造者里德处每 15 天无偿锻造一次装备·「请忽略下次见面时里德的态度」）。里德将以**职工者**身份在乐园安身（恨苏晓恨得牙痒、却也算被还了有限的自由）。

## 乐园重逢 · 认证锻造者 + 长期合作（十七卷90–91章）

苏晓回乐园持 [里德之勋](../力量体系/装备道具.md) 找里德，两人正式「重逢」——

- **被乐园半路截下**：苏晓当初定的坐标本是「虚空·恶魔古堡」，里德却在半路被轮回乐园截下、收进乐园。里德咨询过、确认苏晓没骗他（终点确是虚空），但他**猜到苏晓早知道自己会被截**，因此见面只表现愤怒（「你，这，混，蛋！」）、并无深仇——对住惯漆黑陵墓的他，乐园「已堪称天堂、有无限可能」，头颅的裂痕也愈合了些（进乐园的好处）。
- **乐园认证锻造者 + S 级熔炉**：里德已是**轮回乐园认证的「锻造者」**（自陈「猎杀者是武力机构、我是后勤·锻造者 / 裁决者 / 猎杀者互不敌对」），并与乐园等价交换得到一台**S 级锻造熔炉**（S 级每小时租赁费 40 万币·B 级买价 160 万·S 级几乎天价）——能锻传说级装备、地位不低。他对锻造有近乎偏执的完美追求（以微米 / 显微级查看刃口、修一次斩龙闪报价 75800 币·他人仅 6000）。
- **7.5 折长期合作**：苏晓可凭里德之勋每 15 自然日免费打造 / 修一次装备，但里德不甘单方面被压榨——双方谈定苏晓的**稀有金属 / 矿产 / 锻造材料按市场价 7.5 折优先卖给里德**（从六折砍价到七五折）。里德也确实「快真正活过来了」（注：他在希亚是个缺德角色·曾夺某国王王后在王宫淫乐、杀生无数）。

## 锻造水平瓶颈 + 喔喔 + 重锻陨之火刃（二十七卷55章）

苏晓返园找里德保养装备时——

- **卡在史诗级瓶颈**：里德靠海量契约者求锻装备练手、锻造水平飞涨到能锻**史诗级**，却卡在史诗级无法寸进（赤膊铁铸肌肉·头部裂痕基本愈合·更敦实强大）。
- **捡了个喔喔（思林特斯族）**：里德半个月前去虚空进货「捡」回的小亚人——身高 1 米、青绿皮肤、瘦弱、水汪汪大眼异常蠢萌（每挥铁锤都使尽全身力气）。**思林特斯族 = 虚空最擅制造的种族之一、几百年前被施法者灭族**（苏晓一眼认出·叮嘱别去奥术永恒星）。里德嘴上「看着顺眼捡的」、实为抢来的不听话奴隶被飞船压力门挤过脑袋（苏晓打趣让她叫自己「父亲」）。
- **重锻陨之火刃换人情**：里德见 [陨之火刃](../力量体系/装备道具.md) 的珍贵材料惊呆（「哪只狒狒雕出来的」），破例**主动包材料免费重锻**——条件是苏晓日后去虚空替他还个人情：教里德一位虚空朋友酷爱钻研刀术的女儿一次刀术知识（苏晓是他认识的唯一刀术宗师）。

## 关系
- **[苏晓](苏晓.md)（铁之手 / 灭法者）**：把他忽悠进乐园的人类；乐园重逢后成**长期合作的锻造伙伴**（里德之勋免费打造 + 7.5 折收材料 + 重锻陨之火刃）。
- **喔喔（思林特斯族·里德捡养）**：虚空灭族种族的幸存幼女、里德的小帮工。
- **[戈斯](戈斯.md)**：被苏晓推去探路、最先撞见里德的骑士。
- **[敦克·尼克](敦克·尼克.md)**：同为死灵相关强者，但里德的霸主级远在其领主级之上。
