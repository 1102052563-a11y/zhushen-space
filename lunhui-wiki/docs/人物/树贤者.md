# 树贤者（虚空顶尖炼金师）

[莉莉姆](莉莉姆.md) 带 [苏晓](苏晓.md) 去拜访的**虚空顶尖炼金师**——隐居在一颗偏僻星球的「**迷火森林**」（火红树木·绿萤飞舞·实为一大片炼金防御禁制·林中上万「树人」皆其炼金造物·含炼金生物制造学）。本人身材干瘦、面容枯槁如将枯的老树、**没有双腿**（身体接在一台炼金轮椅机关上）。在虚空能被尊称「**贤者**」者，必在某领域造诣极高。

## 顶尖炼金师 · 还人情（十四卷59–64章）

他与莉莉姆的父亲是旧友、欠其一个人情（莉莉姆幼时曾在此长住、视他为和蔼长者·她一身高等炼金物品皆出其手）。他一眼看穿苏晓是 [灭法之影](../力量体系/技能/index.md)、且青钢影纯度极高，主动问出苏晓自称是 **[马文·华尔兹](马文·华尔兹.md) 的弟子**（莉莉姆都惊到——「做他徒弟还能活、佩服你的耐痛能力」）。为还恶魔族人情，他给苏晓一本古书 **【炼金生物起源】** + 一间完善炼金实验室 + 海量练手材料（房中还有「魔力源泉」每秒回 5 法力），任他速成 [炼金生物制造学](../力量体系/技能/index.md)。

他通过透明石板暗中观察苏晓——见苏晓只读古书一小时就备份开造、把他归入「**急于求成**」之列，却又被苏晓**越造越快（渡鸦从 10 分钟 / 只飙到 32 秒 / 只）、且越造越精良**的怪异天赋惊到（虽不看重「快」·认为一个高级炼金生物胜过几万低级）。他是个**态度不明的中立者**（疑两不得罪），故 [瑟菲莉娅](瑟菲莉娅.md) 追到迷火森林时他出面挡下（领地内实力翻几十倍·瑟菲莉娅不敢造次），又在事后与瑟菲莉娅交涉。苏晓临走说「或许还有见面机会」，他却答「你若成长起来、我们一定不会再见面」（被那「提刀笑着请教炼金学」的既视感吓到）。

## 关系
- **[莉莉姆](莉莉姆.md)**：最中意的「准传人」（可惜她嫌炼金麻烦、毫无兴趣）；其父的旧友。
- **[苏晓](苏晓.md)**：因恶魔族 / 马文·华尔兹之故传他炼金生物制造学的炼金师（交易性质·不会教更多）。
- **[瑟菲莉娅](瑟菲莉娅.md)**：领地内挡下她对苏晓的追杀、事后交涉。
