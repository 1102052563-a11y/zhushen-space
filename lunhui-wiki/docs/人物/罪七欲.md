# 罪七欲（异常违规者·七欲之镰）

[苏晓](苏晓.md) 在 [魔法禁书目录](../世界/任务世界/魔法禁书目录.md) 世界的**猎杀任务目标「异常违规者」**——苏晓遇过的违规者里**最神秘、也最反套路**的一个。能力「**七欲之镰**」可操纵 / 放大目标的七情绪（愤怒 / 傲慢 / 暴食 / 怠惰 / 嫉妒 / 色欲 / 贪婪）；现身时操控七名超能力者（额头各带一情绪符号·如提线木偶·头顶悬一个七眼大头婴儿）。曾用七欲之镰隔空「试探」过苏晓一次（见 [七欲之镰伏笔](../世界/任务世界/魔法禁书目录.md)）。

## 真相 · 画风突变的「求和」（十三卷33–34章）

他原本只是乐园里小有名气的**职工者情报商人**——某个衍生世界被一只**虚空生物寄生**（那虚空生物自大、无视乐园三次警告、连累他一同成了违规者）。后来虚空生物在一场与猎杀者的战斗中死掉，他重夺主导意识、却**继承了被拔高的天赋（三属性达 80）**——可他战斗经验几乎为零、心理还停在职工者状态，自称「纸老虎」「违规者中的二哈」。他怕极了猎杀者（近距离接触苏晓时腿直抖），故特意等苏晓与一方通行两败俱伤后才现身，全程**用情报当武器、神秘 → 慢慢怂 → 吐苦水 → 完全怂 → 拍马屁**，只为求和——他真正想要的不过是「摆脱违规者身份、继续安心卖情报」。

- **交易 · 重做契约者（十三卷34章）**：苏晓向乐园举报后，乐园给了他一个机会（因非他主观违规 + 战斗力不错）：**永久清除一切猎杀者 / 违规者相关记忆、重做契约者**（代价：每个衍生世界接 2 条主线·奖励 / 世界之源 / 币 / 属性点皆减半·直到以 A 级评价经历四个世界）。他万分不舍那些「瑰宝级」的禁忌情报（其爱好与苏晓对变强的热衷相通），但更怕被疯子追杀，最终**同意**——被传回乐园、清空记忆离场（下次见面要重新自我介绍）。他还顺手用情报拖住了亚雷斯塔派来阻止苏晓的人，间接帮苏晓杀成了一方通行。

## 关系
- **[苏晓](苏晓.md)**：来猎杀他的猎杀者（反被他求和·达成交易、苏晓的猎杀任务完成度 94%）。
- **虚空生物**：曾寄生他、连累他违规的「傻哔」（已死）。
