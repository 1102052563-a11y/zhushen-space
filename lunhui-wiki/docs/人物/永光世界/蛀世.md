# 蛀世

⚠ **几个月前，狂茂之地来了个堪称是灭世级生物中疯子般的存在——蛀世！**

## 🌳 是虚空之树把它放逐来的

⚠ **要知道，当初召唤出蛀世后，并非 [苏晓](../苏晓.md) 把这族群放逐**——⚠ **而是触发了虚空之树的公证，是虚空之树将蛀世放逐到永光世界**;⚠ **可以想象，这玩意的可怕程度。**

- ⚠ **它原本为何沉寂**:⚠ **蛀世虽不是来自超脱·原生世界，或是虚空**——⚠ **但它在原本世界沉寂的原因，并非被战胜**;⚠ **而是它把那世界的一切都吃光了**——⚠ **外加有虚空之树公证的战力上限，让蛀世无法突破到蛀穿世界壁障的程度**;⚠ **最终才因没物体吞噬，很多年后逐渐沉寂。**
- ⚠ **放逐＝给它更高的上限**:⚠ **被放逐到永光世界，对于其他灭世级族群来讲，这是危险与囚困**——⚠ **可对蛀世而言，这给了它更高的上限。**
- ⚠ **它已经进化到连极端自然元素都能吞噬**——⚠ **在狂茂之地，无论多凶狠的族群也不会大量吞噬植物**（⚠ **那会成为本区域内所有灭世级族群的敌人**）;⚠ **其中蛀世例外。**

## ☠ 它的战绩

- ⚠ **[银皇后](../银皇后.md) 与它交锋过一次**:⚠ **结果是，派出的虫族军团很快消失**——⚠ **就如同被那区域的世界一口吞食了般**;⚠ **提及蛀世时，银皇后的目光会严肃几分。**
- ⚠ **苦痛女王已被它吞食掉**（⚠ 这是 [厄兹勒](厄兹勒.md) 亲口说的——⚠ **「虽然它已经被蛀世吞食掉，但在那之前，我们的合作一直很愉快」**）。
- ⚠ **苏晓的判断**:⚠ **不知为何，他忽然有种，除了无光神殿·四巨头外，蛀世也是同样可怕的敌人**——⚠ **对付蛀世唯一的方法，就是召来 [棘拉](../棘拉.md)、发展虫族**;⚠ **唯有爆兵出足够强悍的焰龙军团，以无死角的焚世龙焰，才可能对付蛀世。**
- ⚠ **它是攒兵期的最大威胁**:⚠ **这些灭世级族群中没有弱的，但相对而言，已经能吞噬极端自然元素的蛀世，才是这方面的终极大 boss**——⚠ **攒够生物能前，绝不能被蛀世留意到。**
- ⚠ **[暴躁小精灵·迪亚古](../暴躁小精灵·迪亚古.md) 已被派去追踪蛀世的方位。**

## 关系

- [苏晓](../苏晓.md)：召唤出它的人——结果虚空之树亲自出手，把它放逐到了永光世界；如今苏晓要在这里再对付它一次。
- [棘拉](../棘拉.md)：对付它的唯一答案——需要爆兵出足够强悍的焰龙军团，以无死角的焚世龙焰。
- [银皇后](../银皇后.md)：与它交锋过一次、派出的军团如被世界一口吞食的人。
- **苦痛女王**：已被它吞食掉——曾是厄兹勒（神父）愉快的合作方。

## ⚔ 50 小时决战与封印（51卷51章 · 传说任务的一半）

⚠ **蛀世的本相**:⚠ **单个个体和寻常飞虫接近、铁灰色、三对金属质感虫翼**——⚠ **飞行振鸣的嗡嗡声若几十万只齐震，足以让大部分生灵动弹不得乃至致死**;⚠ **⚠ 到了永光世界，蛀世的数量要以亿计算。** ⚠ **它没有本体与分裂体之分**——⚠ **哪怕只剩一只，也能通过分裂快速壮大、短时间恢复巅峰规格**;⚠ **⚠ 这就是它「无法被消灭」的根源**（⚠ 唯一制约:它不能距离彼此太远）。

- ⚠ **它并非在成长，而是在「恢复」**:⚠ **它原本就是最顶尖猎食者级别的存在**——⚠ **之前被虚空之树的世界公证压制到八阶**;⚠ **被放逐到永光、有了充足自然元素后，它重新解锁了曾经的上限**（⚠ **这也是寄星蟹、狂茂族都不与它冲突的原因——它本就高它们一筹，它们不服气都不行**）;⚠ **⚠ 它进化出「吞噬极端自然元素」的特性后，甚至到了灭世级存在的一个新梯队，即以它命名的「蛀世级」。**
- ⚠ **它小觑了虫族**:⚠ **它明知狂茂之地的虫族情况，却仍选择完成沉睡蜕变、把自身提升到最强**——⚠ **⚠ 正是这份轻视，给了苏晓机会。**

### 蛀世 vs 恶魔焰龙

⚠ **它吞龙焰、靠吸收龙焰的能量快速分裂**——⚠ **半边天空都化为它的铁灰色**（⚠ 庇护城被它的嗡鸣震成粉碎·⚠ **本世界契约者收到虚空之树警告刷屏，十分之九选择离开**）;⚠ **它聚成翼展上百公里的金属飞龙、喷「金属龙焰」（松散的蛀世）啃食恶魔焰龙**——⚠ **一度有全灭恶魔焰龙的征兆。**

- ⚠ **压制它的关键**:⚠ **等级 3 的满级恶魔焰龙喷「耀金色龙焰」**——⚠ **能把蛀世炙烤成液态**;⚠ **可蛀世反手汇聚成几米圆核爆炸、专找喷耀金龙焰的焰龙啃食**;⚠ **⚠ 它甚至构成「蛀世黑洞」，把恶魔焰龙、泰坦巨兽、连同己方虫巢和银皇后虫巢一同拔起吞噬。**
- ⚠ **它输在「跑不掉」**:⚠ **以它的速度，它无法主动脱身**——⚠ **这正是苏晓主发展恶魔焰龙的原因（能抗能打能追、配战争领主 90% 飞行速度）**;⚠ **⚠ 苏晓早把主虫巢转移到元素区（世界之力更浓、易守难攻），从四面八方悄悄增援满级恶魔焰龙**——⚠ **与蛀世战斗，焰龙的战斗经验疯长**;⚠ **50 小时后，60 万只满级恶魔焰龙的耀金龙焰把蛀世从上百米压到只剩最后一只。**

### 封入原罪之书

⚠ **最后一只蛀世被龙焰顶在半空、只变炽红却烧不成灰**——⚠ **且此处空间已遍布裂痕，再烧空间就会炸开、大概率把蛀世卷走传送到未知之处。**

- ⚠ **苏晓停龙焰、单手抓住蛀世、甩进封印盒**（⚠ 十秒布 15300 层封印——⚠ **突破了他之前十秒 15000 层的极限**）——⚠ **可封印盒加上万层封印也不能永远困住它**;⚠ **⚠ 苏晓把蛀世连同封印盒封进「[原罪之书](../../力量体系/装备道具/index.md)」最后一页**（⚠ **和熔金魔不同，蛀世被封进书页后依然用复眼盯着苏晓，没有丝毫退让与畏惧**——⚠ 「灭法者，你无法消灭我，更无法永远封印我」）。
- ⚠ **它对传说任务的分量**:⚠ **消灭它占 50% 完成度、封印占 15%**——⚠ **苏晓封进原罪之书，判定猛增，获得 30% 任务完成度**;⚠ **⚠ 这让「[永光领主](../../世界/任务世界/永光世界-厄难囚笼.md)」传说任务完成度达 80% 以上。**

## 关系（补）

- [苏晓](../苏晓.md)：召唤出它、又用 60 万满级恶魔焰龙耗死它、最后把它封进原罪之书最后一页的人——但从未认为自己能真正消灭它。
