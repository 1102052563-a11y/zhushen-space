# 猩红长老·虫什（殁）

⚠ **二层是一个书库**——⚠ **只不过有很多书架倒下，各类书籍堆的四处都是，有些更是有虫蛀的痕迹**;⚠ **位于书库最里侧，一名身着暗金色古旧长袍、满头花白长发的老者正站在那。**

## ⚡ 「闯入者，都要死于……」

⚠ **四目相对，老者逐渐飘浮而起，他周边涌现猩红的闪电球，气势疯长**:

> ⚠ 「**闯入者，都要死于……**」

- ⚠ **不等他把出场台词说完**——⚠ **苏晓已放出魔灵并构成魔灵箭，身后血气虚影出现，灵魂大弓拉满，一箭射去**;⚠ **嘭的一声，魔灵箭射中一面静电壁障，猩红的静电转瞬隐没，化为黑烟的魔灵作势向下方落去。**
- ⚠ **就在这时，苏晓骤然与魔灵互换位置**——⚠ **更致命的是，他在换位置前，已开始蓄势「刃道刀·弑」**;⚠ **满蓄势的「刃道刀·弑」斩出，呼的一声！血色匹链斜斜斩出**——⚠ **外加有血魂加持，所斩出的血色匹链呈现出暗红色，里面遍布星星点点的火星**;⚠ **一刀下去，所斩过之处的空间出现一道道细密黑痕**——⚠ **猩红长老构成的几十层各类护盾，当即全部爆盾。**
- ⚠ **「刃道刀·血刃」**:⚠ **不给半点喘息的机会，苏晓已化为一道直线血色斩芒**——⚠ **随着血色斩芒掠过，猩红长老的胸膛前喷溅出大片血迹**;⚠ **他的眼睛已经瞪圆，死死盯着苏晓**——⚠ **几乎同时，苏晓感觉到自己全身的鲜血温度飙升，仿佛他的所有鲜血都会在下一秒炸开。**
- ⚠ **龙影闪 → 一脚侧踢，将猩红长老踢成一道光，斜斜砸向书库的角落**——⚠ **半躺在地的他抬手虚握，因预想到敌人因鲜血而炸开的景象，眼中有了几分被爆锤后格杀敌人的快意**;⚠ **他不信，一个近战系在连续突进两次后，还能在短时间内使用这么离谱的突进能力。**
- ⚠ **然而**——⚠ **刃之魔灵的烟气在他背后飘散**:⚠ **方才那一脚侧踢大幅度收力，目的就是为了精准度**——⚠ **让斜斜向下飞的猩红长老穿过刃之魔灵的黑烟，从而在对方身上附着一定量的刃之魔灵**;⚠ **这些魔灵，已经足够发动位置互换。**
- ⚠ **此刻，作为中远程强者的猩红长老，深深体会到了施法者与灭法之影战斗时的心情。**
- ⚠ 「**等……**」——⚠ **铮～！一道斩芒划过，敌人首级落下，就是如此的干脆利落。**

## 🐛 那 3 秒的空白

⚠ **1 秒，2 秒，3 秒，击杀提示未出现**——⚠ **苏晓开启「狩猎视野」环视周边**:⚠ **一条猩红的液渍在地上向远处蔓延，此刻正没入到十几米外的书堆中**;⚠ **留下这猩红液渍的东西，完全屏蔽感知的捕捉。**

- ⚠ **「血烟炮」瞄准书堆**——⚠ **吱的一声虫类惨嘶后，一只水蛭般的黏虫被轰杀成灰烬。**
