# 永暗之主

**庇护城**——永光世界唯一的生灵大型聚集地——**就是它的作品**。⚠ 而它是**灭世级存在**、**无光神殿四巨头之一**，是本世界**至强实力**的四个「囚徒」之一。

## 🏙 它为什么要建一座城

⚠ 苏晓一走进庇护城就察觉不对:⚠ **这庇护城不对，很不对，尤其是顶部**——⚠ **其他人察觉不到，可他作为灭世级存在的死敌·灭法之影，第一时间就察觉到:这庇护城的顶部有灭世级存在的气息**，虽然很淡，**但他不会感知错**。

- ⚠ **这气息的伪装极高明**:「那股气息很淡，⚠ **就算被感知到，也会被误认为这是神灵的气息**——⚠ **除灭法者或深渊监视者外，其他人基本没可能察觉到不对**」；⚠ **而且这气息格外强大，那种吞噬一切光芒的力量**。
- ⚠ **谜题的答案**:苏晓之前一直奇怪，⚠ **永光世界的这些可怕怪物，怎么会忍住不来此地猎食生灵？大部分灭世级存在都很渴望鲜活的生命力**——⚠ **现在看来，这一切都合情合理:因为庇护这里的，就是本世界最强的灭世级存在之一·永暗之主。**
- ⚠ **无光神殿是它所创立**——⚠ **如此想来，这势力基本就代表了一些灭世级存在或深渊滋生物**；⚠ **搞不好，这势力的最高掌权者不只有永暗之主，而是还有其他强大的灭世级存在或深渊滋生物**（事实正是如此:⚠ **四巨头 = 永暗之主 ＋ [深渊大主教](深渊大主教.md) ＋ [始祖](始祖.md) ＋ [星界吞噬者](星界吞噬者.md)**）。
- ⚠ **现在的唯一好消息是:永暗之主不在庇护城**——⚠ **否则庇护城顶城的那气息不会这么淡。**

## 🗓 它的编年史

- **极昼纪元 185 年**：⚠ **无光神殿日渐崛起，庇护城的雏形逐渐展现**——⚠ **城外的极昼之光代表死亡，唯有灭世级存在与深渊滋生物们能无惧那可怕的极昼之光。**
- **极昼纪元 279 年**：⚠ **庇护城彻底建成**——⚠ **在永暗之主的庇护下，庇护城成为永光世界唯一的生灵大型聚集地**；⚠ **而信奉永暗之主的无光神殿，也在永暗之主的授意下成为庇护城的管理者。**
- **极昼纪元 350 年**：⚠ **大部分封印被破除**，那些被灭法者、深渊监守者囚困的可怕存在们陆续破封而出——⚠ **来自上个时代的恐惧，让庇护城岌岌可危。**
- **极昼纪元 352 年**：⚠ **庇护城被毁灭八成以上，直至永暗之主降临。**
- **极昼纪元 405 年**：⚠ **庇护城与几位最可怕的存在签订誓约，彼此互不招惹**——⚠ **至此，庇护城终于迎来稳定。**

## 🕳 逃出去的想法最坚定的那个

⚠ **四巨头里，永暗之主的刑期最短**:⚠ **[始祖](始祖.md) 十几万年 ＞ [星界吞噬者](星界吞噬者.md) 九万年出头 ＞ [深渊大主教](深渊大主教.md) 差不多五万年 ＞ 永暗之主几千年**。⚠ **也正因如此，永暗之主逃出去的想法最坚定**——⚠ **又是建立庇护城，又是创建无光神殿这势力**。

- ⚠ **而当它以无光神殿的神位去拉拢深渊大主教、始祖、星界吞噬者时，三方的态度很有趣**:⚠ **深渊大主教与星界吞噬者的态度是「随便吧，反正也都出不去」**；⚠ **始祖最提不起兴趣**。
- ⚠ **它麾下的成员一直在做的事**:「永光世界的各个封禁**无人修缮**——不对，⚠ **并非只是无人修缮**:⚠ **这么多年来，无光神殿的骨干成员们，一直在永光世界各处破坏这些封禁**」；⚠ **他们是本世界最强三个灭世级存在、外加一个本世界最强深渊存在的手下，自然不在乎更多灭世级存在被放出来。**
- ⚠ **它的鹰犬**:「这里中城区的**无光神殿成员**，⚠ **这是永暗之主麾下的鹰犬**」；⚠ **而庇护城的居民与一些强者，他们其实没什么坏心思**——⚠ **最多是有些强者，被外界的强大怪物打击到心灰意冷**。⚠ **土著民根本不知道是什么在庇护这里，误认为永暗之主是神灵存在**:⚠ **街道上的行人神情自然，街边的小贩烹煮着小吃，更暗的小巷内，几名少年扒手正满脸得意地清点刚入手的财物。**
- ⚠ **庇护城的三层结构**:**下城**（平民居住区，**占整个庇护城面积的七成以上**）· **中城**（**无光神殿所把控，经过批准的平民可进入**）· **顶城**（⚠ **代表平民对永暗之主的朝拜之处**——⚠ **虽说永暗之主并不在那，却也无人敢踏足其中**）。
- ⚠ **两个陷阱**:⚠ **庇护城的庇护石都被无光神殿动了手脚**（[索恩斯](索恩斯.md) 与 [罪亚斯](../画中世界/罪亚斯.md) 都特意警告过「不要用庇护城的庇护石」）；⚠ **庇护城有「空间震膜」，能感应到几乎所有传送方式**——⚠ **唯一的例外是 [尤莎](尤莎.md) 那处古旧的传送阵**。⚠ 「想进入这里简单，**可想要离开，那就困难重重**。」
- ⚠ **它现在最想要的东西**:**君主宝藏**——⚠ **无光神殿是盯上君主宝藏的第三伙势力**（另两伙是「好队友」四人组、以及冥神麾下的鹰犬＋神父）。⚠ **因为那宝藏里有 [月之辉](../../力量体系/装备道具.md)**:⚠ **能自由出入永光世界的钥匙**。

## 关系

- [苏晓](../苏晓.md)（最后的灭法之影）：一走进庇护城就隔着整座城认出它真身的死敌——「除灭法者或深渊监视者外，其他人基本没可能察觉到不对」。
- [深渊大主教](深渊大主教.md) / [始祖](始祖.md) / [星界吞噬者](星界吞噬者.md)：它拉进无光神殿的另三位巨头——前两位的态度是「随便吧，反正也都出不去」，始祖最提不起兴趣。
- **庇护城的居民**：把它误认成神灵、在它「庇护」下活着的土著——却也活不过 40 岁。
