# 暗之女

一名身穿**黑色长裙、身形柔美**的女人——⚠ 她的**双眼一片漆黑**，虽气质柔和，⚠ **可当她目光幽幽地盯着人看时，会让心志不坚者感觉到莫名的瘆得慌**。⚠ 「可以说，**暗之女总是很安静**——**安静、柔和，但强大**。」

## 💠 那双晶体义肢

⚠ 除了那双漆黑的双眼外，她的**双臂为黑蓝色的晶体义肢**:⚠ **靠近肩膀处的晶质已完全漆黑，越向手部则越是透蓝，指尖处还存留着唯美的浅蓝感**。⚠ 这双手臂的**所有细节都很精细，但破损程度也不低**——尤其是**左臂，袖口下的手腕处已经破损小半**。

- ⚠ [苏晓](../苏晓.md) 一触碰就摸清了来历:⚠ **这两条晶体手臂，应该是某位先代灭法，以傲歌能力把青钢影能力晶体化所构成的**——⚠ **这么多年过去，这两条晶体手臂的结构竟还如此稳定，可见当初那位灭法之影的青钢影能力有多强大**。
- ⚠ **修复**:苏晓以 **9500 点青钢影能量**构成晶体（**这是他能生成的最高强度傲歌晶体**），以这些**还呈现液态的傲歌晶体**修复她的双臂；⚠ 不仅如此，他还**构成几十根灵影线，让这些保持高活性的灵影线没入到她的双臂内，当作这两条手臂的神经，并与肩膀的断臂处神经相连**。⚠ **她的肩膀不自然地动了下，随后恢复安静。**
- ⚠ **技艺对比**:「如果说以傲歌晶体构成义肢，⚠ **以苏晓现在的青钢影能力等级，的确不如先代灭法**；⚠ **可如果说晶体手臂的完善程度，苏晓是不虚任何先代灭法的**——⚠ **这点，经过了无良灭法导师，以及老灭法的承认**。」
- ⚠ **她的反应**:完善结束后，她**抬手用手指触碰了下脸颊**——⚠ **随后指尖的触感，让她漆黑的双眼中有了几分意外**。⚠ **这是见面以来，她除了安静之外，首次有其他情绪表达。** ⚠ 她**慢慢活动自己的十指**，片刻后**对苏晓略低头施礼**:「**感谢您，灭法大人。**」

## 🕯 职责的两次改变

⚠ **在曾经**，暗之女的职责是**为灭法者、深渊监守者提供「庇护」这增益状态**，以及在需要时**保证监视者高塔的安危**。⚠ **但在后来，多数监视者高塔被毁、一名名暗之女死去后**——⚠ **最后的暗之女来到了灰暗圣所，将这最后的圣所激活**；在那之后，[老树族](老树族.md)、[索恩斯](索恩斯.md)、还有**卷轴大师**才来到此地。⚠ **从这之后，暗之女的职责改变，变成尽可能清理本世界的灭世级存在**——⚠ **她一直在履行这职责**。

- ⚠ **她的战力**:苏晓看了眼传送阵附近**那对被黑暗灼烧到只剩白骨的颅骨**——⚠ **保守估计，这对颅骨在活着时都是绝强级**；⚠ **而将它们变成这幅模样堆在这的暗之女，其综合战力可想而知**。（⚠ 她回圣所后**将一把归鞘中的锐剑刺入到一座雕塑的凹槽内**——⚠ **这雕塑上有着十几把归鞘中的利剑，其中有款式酷似狼剑的大剑，也有永光监视者用的机关剑**）
- ⚠ **誓约**:她来到苏晓前方，**柔声说道**:「**灭法大人，您是来履行作为灭法者的职责吗。**」⚠ **几米外的索恩斯已经开始疯狂挤眉弄眼了**——⚠ **因为在永光世界沦落到此等局面后，暗之女的存在已经不同于以往**:⚠ **如果苏晓应允她的「誓约」，那么她的职责将恢复到最初**——即为**保证灰暗圣所的安全**，以及**为灭法之影加持「庇护」状态，让灭法之影完全无惧永光世界的极昼之光**。
- ⚠ **她只是安静地等待**。片刻后苏晓问:「**灭法在这的职责是什么？**」——⚠ 「**斩灭逃出牢笼的囚徒。**」⚠ 「是吗，**那我应该没问题**。」——⚠ 「**祝您成功，灭法大人。**」

## 🌑 庇护

⚠ 她请苏晓在一张不大的圆形石桌对面落座，随即**抬起手、双手虚握**——⚠ **透蓝的晶体手指上涌现黑色能量，在虚握的双手间构成一颗能量黑球**；⚠ **过度浓郁的黑暗，导致中心处出现莹白色微光**，这微光内**蔓延出一条丝线**，蔓延到苏晓的手臂上。

- ⚠ 【**你已得到暗之女的「庇护」增益，后续 10 个自然日内，你将无惧极昼之光的照灼。**】⚠ 不仅如此，**这股庇护能量还继续蔓延，以丝线模样蔓延到 [布布汪](../布布汪.md)、[阿姆](../阿姆.md)、[巴哈](../剧毒魔鹰·巴哈.md) 身上**。
- ⚠ 苏晓这才明白，**为何之前索恩斯给自己庇护石时，说自己应该用不到这东西**——⚠ **暗之女加持一次「庇护」状态就能坚持 10 天，并且这状态并非只能加持一次，快要到时间时可以再来加持**。

## 🗝 君主地宫的钥匙与宝库地图

⚠ 修复双臂后，她从石桌抽屉中取出**一块羊皮纸地图**放在桌上——⚠ 这触发了**支线任务·宝藏**:⚠ **在苏晓完成特定事件、或斩杀永暗之主 / 深渊大主教 / 始祖 / 星界吞噬者之一时，她将把所寻找到的灭法宝库位置，以宝库地图的方式交给他**。⚠ **警告:请勿尝试强行从暗之女手中夺取宝库地图，否则，她将采取相应措施，给予你绝不会致命，但需休养多日的反击。**

- ⚠ 苏晓以索恩斯的地图作参照，很快发现**这份宝库地图位于蚀雨地所在的区域**——⚠ **这应该不是巧合，是暗之女知道最后一座监视者高塔在蚀雨地的深处**。（⚠ 本世界的**灭法宝库不少**，但那都是**灭法者们的个人资产**；⚠ **真正惊人的资源财富都在灭法阵营的阵营宝库内**——⚠ **阵营宝库倒是只有一个，问题是不知道是否已被灭世级存在们发现**）
- ⚠ **她的情报**:「灭法大人，**无光神殿已经在寻找君主宝藏的位置**，您有必要在他们之前更先找到这宝藏——⚠ **这宝藏中，有能打开永光封禁的方法**，⚠ **不过这没有开启「界级术式」重要**。」苏晓答「我知道这事，就目前的情况看，**这事上我们没什么优势**」——⚠ 「**有的，灭法大人。**」
- ⚠ 她说话间，**将一把古朴又手感沉重的钥匙放在石桌上**——⚠ **这赫然是君主宝藏的开门钥匙**。⚠ **没有这东西，别想进入君主地宫**:据罪亚斯所说，⚠ **君主地宫的入口和君主地宫不在一个空间内，地宫位于一个独立存在的空间**；⚠ **永光世界因有整体的空间封禁，这类独立存在的中小空间格外难以探查——或者说基本没办法探查，空间系来了也没用**。⚠ **因此找到君主宝藏的唯一方式，就是找到入口处，然后凭这把钥匙打开封印的空间之门。**
