# 神殿祭祀长·厄兹勒

⚠ **永暗之主手下的头号鹰犬**——⚠ **在无光神殿内部，四巨头之下就是他**;⚠ **带着些许花白胡茬的老脸**——⚠ **说话时脸上常有几分笑容。**

!!! danger "⚠ 他不是他"
    ⚠ **说到最后，神殿祭祀长·厄兹勒脸上的笑容越发慈蔼**——⚠ **慈蔼到，逐渐开始有神父的面部轮廓**;⚠ **准确的说，这就是 [神父](../神父.md)**——⚠ **真正的神殿祭祀长·厄兹勒，早已被他杀死，并取而代之。**

## 🐍 「我并不想找到他」

⚠ **他去见了 [银皇后](../银皇后.md)**:「**到现在为止，你们无光神殿，还没找到那灭法者的踪迹？**」——⚠ 「**当然不是，其实我的手下，很快就能找到那灭法者，但我并不想找到他。**」

- ⚠ **他的言外之意分明是**:⚠ **因本世界的封禁开启，最强只能达到绝强顶尖**——⚠ **无光神殿那四位无上存在，也从不可战胜变成有可能被战胜**;⚠ **因此现在找到那灭法之影，肯定是让他们这些鹰犬一拥而上，而非那四位无上出手格杀灭法者**——⚠ **因此他一点都不想找到灭法者，更不想试试灭法者的强大是否属实。**
- ⚠ 「**这么说，你想背叛无光神殿？你在无光神殿的地位，只在永暗之主、始祖等之下，你这样做，真的值得？**」——⚠ 「**只要能离开这里，值得，带着钥匙的人来了这里，我以前从没和他敌对，所以我为什么要选择和这灭法敌对，给那些灭世存在当炮灰**;⚠ **别说我是无光神殿的人，我们庇护城的居民，没得选，要么加入，要么成为饵食。**」
- ⚠ **他的情报来源**:「**我听说，你和那灭法是熟识？**」——⚠ 「你……听谁说的。」⚠ 「**苦痛女王，虽然它已经被 [蛀世](蛀世.md) 吞食掉，但在那之前，我们的合作一直很愉快。**」
- ⚠ **他的推理**:「**我们能离开这囚笼的唯一方法，只在君主宝藏里，现在我们已经找到入口**——⚠ **只不过，我怀疑那入口的钥匙在那灭法者手中**;⚠ **否则他不可能这么从容，让我们察觉不到他的半点踪迹，甚至，他都不来窥探我们。**」
- ⚠ **他找银皇后的真实理由**（⚠ **摸了摸下巴花白的胡茬，语气带着几分不确定**）:「**这虫族女王是那灭法者放逐到此，如果是敌对，有机会放逐这虫族女王，为什么不直接灭杀她？虫族女王并非不死不灭，所以我猜，那灭法者放逐这虫族女王另有目的，只是，我还没想到原因。**」——⚠ **红衣祭祀猜「会不会是那灭法者的从属」**;⚠ **他否定了**:「**不像**——⚠ **其实还有种可能，那灭法者的确有虫族从属，但不是银皇后，他把银皇后放逐到这里，只是让银皇后提前找一处能发展虫族的区域？**」（⚠ **说到这，脸上的笑容越发浓郁，这让他看起来有种莫名的诡谲感**）⚠ **——他猜对了。**

## 🕸 策反红衣祭祀

⚠ 「**对啊！厄兹勒你说得对，不行，我们得立即把这件事报告给……**」——⚠ 「**报告给谁？**」（⚠ **似笑非笑地看着红衣祭祀**）

- ⚠ 「当然报告给四位无上存在。」——⚠ 「**然后呢？那四位无上存在会离开顶城，到这狂茂之地灭掉银皇后的虫巢？**」⚠ 「这……」（⚠ **红衣祭祀无话可说**:⚠ **如果事态这么发展，肯定是他们带着大量祭祀与神殿骑士、神殿战士来此对付虫族**）
- ⚠ 「**你不想丢掉性命，对吗。**」——⚠ 「对。」⚠ 「**所以，同病相怜的我们，才应该彼此信任，而不是给那四个无上存在做爪牙，你说，对吗。**」

## 🔍 苏晓的直觉

⚠ **苏晓查看 [贝妮](../布布汪.md) 传回来的资料时**——⚠ **无光神殿内部相当复杂，不过在四巨头之下，是神殿祭祀长·厄兹勒**;⚠ **看到此人的照片，苏晓以前明明没见过此人，却感觉此人有种熟悉感**——⚠ **具体是哪方面的熟悉，一时间真就难以判断。**

- ⚠ **最终他选中了这个人**:⚠ **该让谁来「发现」那三颗【生命晶石】？**——⚠ **答案是神殿祭祀长·厄兹勒**;⚠ **因为此人，越看，越像是神父。**
- ⚠ **眼下的局面，看似是「好队友」四人组 VS 无光神殿·四巨头**——⚠ **实际上情况比这更复杂**;⚠ **因为在这两方之间，还有个伪装成神殿祭祀长·厄兹勒的神父。**
