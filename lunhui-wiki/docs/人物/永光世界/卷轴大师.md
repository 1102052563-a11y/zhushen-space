# 卷轴大师

⚠ **一名有些驼背的老者**——⚠ **左眼上镶着机械寸镜**（完全替代了左眼的功能）;⚠ **脸上与皮肤满是皱纹，但对知识的掌控让他的精神力并不衰老**——⚠ **四条手臂代表他应该不是人族**;⚠ **花白、有点蓬乱的头发被扎成粗辫，身上的长披肩布料略显破烂。**

## 🏛 他是谁

⚠ **[索恩斯](索恩斯.md) 之前提过，灰暗圣所的卷轴大师已经出门有几个月之久**——⚠ **他就是那位卷轴大师。**

- ⚠ **他被软禁的经过**:⚠ **几个月前，他误入了地下世界**——⚠ **结果被猩红阵营的成员堵截、抓住**;⚠ **就算猩红阵营一向以高傲、残忍著称，但活捉他后也只是将他软禁**——⚠ **这显然是贪图他所掌握的术式**;⚠ **（⚠ 苏晓在地下世界治疗所见过他的精妙圆形术式——⚠ **由成千上万个微型术式构成、放眼万界这种级别的术式大师不超五指之数**;⚠ 神父书房桌上那些半成品卷轴，也是他的**）。
- ⚠ **是噩梦血影间接救了他**:「**那化身在猩红圣殿大肆破坏，间接帮我脱困，我这个人，很讨厌欠人情。**」

## 🕵 他的试探

⚠ 「**灭法，你是谁的弟子？如果不是师徒关系，是谁作为导师，引导你掌握灭法系能力？或许，你的导师是我的老朋友。**」——⚠ **苏晓有着被先代灭法们坑的丰富经验，当即感觉这问题不普通**;⚠ **他答「阿卡斯」**（⚠ **吞星·阿卡斯是灭法者中为数不多可靠的人**——⚠ **卷轴大师略有几分失落:「你要是马文·华尔兹的弟子就好了」**）。

- ⚠ **失落的真相**:「**这混账欠了我 600 多万灵魂钱币。**」——⚠ 「你和 [格林·吉莉安](../格林·吉莉安.md) 熟吗？」⚠ 「不熟。」——⚠ 「**可惜，她还欠我 200 多万灵魂钱币。**」（⚠ **说这话时目光都犀利了几分**）

## 关系

- [苏晓](../苏晓.md)：被他试探师承的最后一名灭法之影——他把噩梦血影间接救他的人情，用净化卷轴还清。
- [索恩斯](索恩斯.md)：同在灰暗圣所的伙伴——是他先向苏晓提起「卷轴大师出门几个月」。
- [马文·华尔兹](../马文·华尔兹.md) / [格林·吉莉安](../格林·吉莉安.md)：分别欠他 600 万、200 万灵魂钱币的两位「老朋友」。
- **猩红阵营**：软禁他数月、贪图他术式的势力——被噩梦血影扫平后，他才得以脱困。
