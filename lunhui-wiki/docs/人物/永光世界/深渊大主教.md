# 深渊大主教

**无光神殿四巨头**里唯一不是「灭世级存在」的那个——⚠ **它是本世界最强的「深渊存在」**，也是**至强实力**。⚠ **苏晓通过灵魂书库的记载，知晓了它几分底细。**

## ♾ 它为什么杀不死

⚠ **这家伙非常可怕**:⚠ **有黑暗神教的世界，一定会有深渊大主教**——⚠ **又或者说是意志分身**。⚠ **每个世界的深渊大主教都有不同，但他们有一个共同点:他们的出现，是因为永光世界这深渊大主教的意识分身。**

- ⚠ **它不是「被选中者」，而是「被推成」的**:⚠ **有时并非是那些世界中的强者愿意成为深渊大主教，而是在命运与因果的趋势下，他们成为了深渊大主教**——⚠ **就像 [暗影世界](../../世界/任务世界/暗影世界-背叛者猎杀.md) 的 [深渊大主教·席尔维斯](../暗影世界/掌控者·席尔维斯.md)**。
- ⚠ **而眼下永光世界的深渊大主教，才是深渊大主教的终极本体**——⚠ **若非永光世界的封禁，这终极本体是不可消灭的**:⚠ **哪怕费尽千辛万苦将其杀死，用不了多久，他就会凭借某个世界内的深渊大主教复生。**
- ⚠ **死循环**:⚠ **只要还有黑暗教，就会有新的深渊大主教分支个体出现**；⚠ **只要还有分支个体，深渊大主教的终极本体就永远不会被消灭**。⚠ **更确切的说法是:只要还有深渊，就会有人渴望那强大又黑暗的力量，黑暗教自然会出现**——⚠ **深渊大主教难以消灭的原因就在这。**

## 🔒 先代灭法的解法:换个牢笼

⚠ **所以先代灭法们，才将深渊大主教囚禁在永光世界，这个没有深渊力量的世界内。**

- ⚠ **先代灭法者们付出不小的代价，才将它囚禁在此**——⚠ **先代灭法者们不能随意尝试，能否在此杀死深渊大主教**；⚠ **而深渊大主教自己也不敢轻易实验，是否能以死亡的方式重生到永光世界外**。⚠ **局面就这样暂时僵住。**
- ⚠ **结果是:灭法们囚困住深渊大主教，而深渊大主教有的是时间等待**——⚠ **它甚至可以等到永光世界崩灭再出去。**
- ⚠ **它的恐怖之处不只在「不死」**:⚠ **深渊大主教的深渊光环——靠近它一定范围内的目标，一旦生命力低于某个临界点，就会被它瞬间吸取**。⚠ **这也是为何苏晓判断「真要对付四巨头，不能靠 [棘拉](../棘拉.md)」**——⚠ **这类灭世级存在最不怕人海战术。**
- ⚠ **它对越狱的态度**:当 [永暗之主](永暗之主.md) 以无光神殿的神位来拉拢时，⚠ **它与 [星界吞噬者](星界吞噬者.md) 的态度是「随便吧，反正也都出不去」**。⚠ **它已在此被囚约五万年**（四巨头中排第三，仅次于始祖的十几万年与星界吞噬者的九万年出头）。

## 🕳 唯一没中毒的那个（51卷26–32章）

⚠ **它的样子**:⚠ **身着黑色主教长袍、容貌与人族很相近**——⚠ **那遍布白色环圈的漆黑眼睛，让人只是与之对视一眼，就如同堕入深渊，被黑暗与绝望所笼罩、吞没。**

- ⚠ **生命晶石毒杀里唯一的例外**:⚠ **[永暗之主](永暗之主.md)、[始祖](始祖.md)、[星界吞噬者](星界吞噬者.md) 都中了猛毒**——⚠ **唯有它没感察到自身有半点中毒迹象，与之相反，因刚吞噬大量生命本源能量，它的状态非常好**;⚠ **⚠ 这就是猛毒的第三种特性——九成以上概率，对深渊系存在无效。**
- ⚠ **它反而成了「催化剂」**:⚠ **它吸收「生命晶石」后升腾的生命之力，吸引来了木系与水系的自然元素，进而带动土系，最终引发元素汇聚现象**——⚠ **这一大片区域内的极端元素之力都在向它靠拢**;⚠ **而永光世界自然元素浓郁，猛毒又以自然元素为第二反应物**——⚠ **⚠ 结果就是身中猛毒的其他三巨头，体内猛毒剧烈增强。**
- ⚠ **它百口莫辩**:⚠ **众所周知，元素和深渊是两种互相对立、互相制约的力量**——⚠ **可现在深渊大主教居然在悄然聚拢元素力量，这实在太可疑**;⚠ **永暗之主寒声「你，现在就离开这」，星界吞噬者一拳轰来**——⚠ **它由衷希望在这的是永暗之主，好联手除掉暗之女**;⚠ **可惜来的是脑子一根筋的星界吞噬者。**

## 🗝 无法拒绝的诱饵

⚠ **它击退星界吞噬者、成为绝望沼泽一战的胜者**（⚠ 对身处永光的它实属困难·面色苍白到渗人、主教袍破烂）——⚠ **它从黑暗中取出 [罪亚斯](../画中世界/罪亚斯.md) 承诺给它的【封之刃】**:「**终于能，离开这牢笼。**」

- ⚠ **它看穿了阳谋，却还是接**:「**这就是那灭法的阳谋，那灭法感觉杀他的概率不高，因此才出此阳谋，而那猛毒，更像是铺垫**——⚠ **让四巨头的合作出现裂痕的铺垫**;⚠ **这把已经调节好、最多让一个人使用的【封之刃】，才是他无法拒绝的诱饵。**」
- ⚠ **⚠ U 型空间通道**:⚠ **它激活【封之刃】、冲入空间通道**——⚠ **它以为出口通往外界，脸上不禁浮现笑容**:「这外界，无论怎么看都顺眼，尤其是远处那有点眼熟的庞大建筑。」——⚠ **然后笑容僵住:它越看，越感觉这像是永光世界的庇护城**;⚠ **原来苏晓的【封之刃】开的是 U 型通道——⚠ **把 U 型通道按在球形的空间封禁上，入口在永光、出口也在永光的某个地方**。**
- ⚠ **概念类不死不灭**:⚠ **苏晓对它没太大把握格杀**——⚠ **深渊大主教是概念类的不死不灭，比永暗之主还棘手**;⚠ **在完成天赋任务第一阶段前，苏晓不会让四巨头中的任何一个有机会逃出永光世界。**
- ⚠ **它后来奇袭了猩红阵营老巢**:⚠ **地下世界南侧涌现浓郁黑暗，将那片星空染黑**——⚠ **这是它与永暗之主奇袭猩红阵营的手笔。**

## 🗡 螳螂捕蝉的最终赢家（51卷40–41章 · 逃出永光）

### 🌑 它就是黑暗神教的无上神灵

⚠ **万界中任何世界的黑暗神教，那些祭献给「深渊」的秘宝、灵魂、乃至本源灵魂能量，其实全部是祭献给深渊大主教**——⚠ **它就是黑暗神教的无上神灵**;⚠ **哪怕它被囚永光，黑暗神教依然活跃**（⚠ **苏晓晋九阶后经历的暗影世界、风海大陆都有黑暗神教**——⚠ **风海那个「深渊大主教」其实是被真正意识影响的分支个体，称「深渊主教」更妥当·真正的只有一个，就在永光**）。

### ♟ 三方博弈里的真正阳谋

⚠ **它的计划**:⚠ **等星界吞噬者被算计而死 → 假意与永暗之主联手 → 用最强背刺武器「[唾弃](../../力量体系/装备道具.md)」背刺永暗之主**——⚠ **⚠ 但灭杀永暗之主不是目的**:⚠ **它是要以「唾弃」引发的无与伦比的攻击，把本世界满目疮痍的空间封禁暂时轰开一道裂痕、从这裂痕逃出永光。**

- ⚠ **它为何等这么久**:⚠ **「唾弃」的机会只有一次——⚠ **一旦暴露持有「唾弃」，其他三巨头都会远离它、就没有能背刺的强者、发挥不出最强二段**;⚠ **它用一次【封之刃】确认了本世界空间封禁已不像上万年前那般坚不可摧**（⚠ **尤其有一股猩红力量一直在侵蚀空间封禁**），才下定决心。
- ⚠ **它与苏晓的交手**:⚠ **它提永暗之主头颅正要倒进空间裂缝**——⚠ **苏晓一把【封之刃】卡进裂缝（只维持几秒）＋一记突刺 ＋ 一脚直踹破黑暗壁障**;⚠ **苏晓「刃道刀·魔刃·暗渊·汲渴」双斩魂 ＋ 魔刃斩下它整条左臂**（⚠ **黑焰燃尽断臂——⚠ 先斩本源灵魂再斩杀，这条手臂再也无法恢复、只会逐渐腐化**）;⚠ **它也回敬苏晓三根黑色触须（左肩/右胸/右肋）**——⚠ **随裂缝愈合消失在其中**:「**今后再见，白夜。**」

### ⚠ 它成了苏晓此生大敌

⚠ **它成功逃出囚笼、恢复至强级只是时间问题**——⚠ **苏晓「深渊大主教绝对是他此生的大敌」**;⚠ **有了它回归的黑暗神教，不是之前能相比的**——⚠ **⚠ 苏晓预感用不了太久就会与它再见，在那之前他必须封临至强。**
