# 月巫

一名**身穿淡紫色长裙、有着满头银色长发**的女人——⚠ 在监视者高塔一层被召来的七名违规者中**最引人注目**。⚠ **尤其是那温婉的气质，会让意志不坚者下意识怀疑:这真的是违规者？或许，不，一定是判定出了问题。**

## 🌙 真正强大的魅力系，是故意降低魅惑性

⚠ **她是顶尖魅力系能力者**——⚠ **她并不是凭借主动能力魅惑他人**。⚠ 「**真正强大的魅力系，不会刻意增强自身的魅惑性，反而是要故意降低**——⚠ **真正可怕的魅力系，是让人与之见面后，下意识心生好感。**」

- ⚠ **她的机制**:「试问，两人见面时，⚠ **对于月巫这等气质温婉、颜值逆天的异性，心生几分好感与幻想是很正常的事**——⚠ **谁都不会对于内心这几分好感有所戒备。**」⚠ **问题就出在这**:⚠ **月巫能让他人对她的那几分好感，不知不觉间变成一颗种子**；⚠ **然后这种子逐渐生根发芽，最后茁壮地生长在一个人的意识与判断中**——⚠ **到了那时，此人会对月巫言听计从。**
- ⚠ **她的定位**:⚠ **如果灰绅士没死的话，那现在 [神父](../神父.md)、[灰绅士](../炎灵星/灰绅士.md)、月巫三人，就是违规者中绝强之下的天花板**。⚠ **值得一提的是，月巫来自圣光乐园。**
- ⚠ **现场案例·魂鬼**:同为七名违规者之一的**魂鬼已经中招了**——⚠ **正飘浮在月巫身后**；⚠ **从那眼神来看，没可能凭自身能力摆脱这种状态**。

## 💄 「你老姐我怎么可能那么轻易被蛊惑了心智」

⚠ [贝芙丽](../幽暗大陆/贝芙丽.md) 警惕地看了眼月巫——⚠ **她可是知道，对方的能力从不是针对异性**:⚠ **只要对月巫有一点好感，那就会中招**。

- ⚠ **如此警惕着，贝芙丽打量了下月巫的衣裙**——⚠ **这衣裙的款式让她感觉很赞**。⚠ **如此想着，贝芙丽调转视线**，⚠ **可不知为何，她的脑海中总是想着那身连衣裙**；⚠ **之后脑中还自行补全月巫的模样**——⚠ **对方那温婉、亲和的气质，让她感觉月巫越来越顺眼**；⚠ **或许，之后可以与对方合作**——⚠ **她下意识又向月巫看去。**
- ⚠ **啪～！** ⚠ **一只小手抽在贝芙丽脸上，是挨了 [小戈](../幽暗大陆/小戈.md) 一耳光**——⚠ **贝芙丽当即打了个机灵**:⚠ **她明明已经内心防范，却还是险些被月巫的能力所影响**；⚠ **对方的手段，实在太隐秘与猝不及防。**
- ⚠ 小戈在小队频道里问「老姐，你干嘛呢，**是不是差点中招？**」——⚠ 「**放|屁，你老姐我怎么可能那么轻易被蛊惑了心智。**」⚠ 「**我刚才把你的表情照下来了，你要不要自己看看。**」「删了。」「**就不。**」（⚠ **小戈笑着露出满嘴整齐的小白牙**）⚠ 「**三。**」「**二。**」「**已经删了。**」（⚠ **碍于血脉压制**）

## 🫱 「我也很珍惜这缘分」

⚠ 轮到 [凯撒](../尼古拉斯·凯撒.md) 上场时，几名违规者都看向她——⚠ **作为能和神父一个梯队的违规者，此等关头肯定是月巫上场**。

- ⚠ 「**好吧，但输了可别怪我，我刚好也想领略一下，欺诈者·凯撒的手段。**」（⚠ **双手十指交叉、带着几分慵懒感地伸了个懒腰**）——⚠ **白骨男、驭虫师、贝芙丽都偏头躲开视线**；⚠ **刚从月巫这逃过一劫的魂鬼，立即也移开视线。**
- ⚠ 「**我哪有你们想的那么可怕，不信你们可以问魂鬼，是不是我主动解除的能力**——⚠ **虽然我们是临时队友，但我也很珍惜这缘分。**」⚠ **听闻此言，魂鬼沉闷的没说话，只是看了眼空空如也的储存空间**；⚠ **不过他装备栏内的装备，却是一件没动**——⚠ **这就很奇妙:魂鬼虽心中恨极了月巫，却又感觉，承担丢掉性命的风险尝试报复，还不如远离对方**（⚠ **毕竟一身精心打造出的装备和能力都还在**）。
- ⚠ **她的弃权**:走上竞技场地，看着对面**头戴深渊之罐的凯撒**——⚠ **她虽依旧气质优雅，心中却是如临大敌**。⚠ **可谁知道，下一秒凯撒身后的黑雾就像泄气般快速消散，凯撒头上的深渊之罐也消失**——⚠ **露出他贼眉鼠眼、还带着几分猥琐的神情**。⚠ **如此差距，让月巫诧异了那么一瞬间**；⚠ **随后，她的面色透出几分苍白感，并立即说道:「我弃权。」**
- ⚠ **凯撒的算盘**:⚠ **因月巫的弃权，刚准备脱鞋试试能否以此翻盘的凯撒，就被传送到己方观众席上**——⚠ **这让凯撒笑的格外奸诈**:⚠ **只要对面的月巫出手，凯撒会立即弃权**（**战斗可不是他所擅长的**）。

## 🗼 十二层的留言
