# 灵魂女仆

⚠ **她是苏晓在极暗古堡见到的第一个「敌人」**:⚠ **吱嘎～房门自行打开，寒雾弥散进来**——⚠ **一道身着透白的血色长衣、满头长发自行飘散、双眼中一片猩红的幽魂飘入房间**;⚠ **这显然是来夺苏晓的性命。**

## 😨 被吓退的鬼

⚠ **苏晓从床上站起身，血气放出，周边的家具、墙壁、天棚噼啪开裂**——⚠ **随着他单手按在腰间的刀柄上，刀术宗师的气场将周边一片区域笼罩**;⚠ **血气、灵魂气息、以及刀术宗师独有的气场，让她一下停在那**——⚠ **双方对峙后，她的眼神竟然逐渐清澈起来，慢慢倒着向门外飘去。**

- ⚠ **她为什么退**:⚠ **苏晓高达 860 点的灵魂强度，还有刀术＋斩魂·魂核的双斩魂效果**——⚠ **他对幽魂、怨灵类敌人造成的伤害会提升 500%～1200% 左右**;⚠ **再加上双斩魂效果，能不被他一刀秒掉的幽魂类敌人，绝对是同阶翘楚。**
- ⚠ **能把鬼物类吓成这幅模样的，属实不多见**:⚠ **她突然打了个寒颤**——⚠ **不仅是因为对面的是刀术宗师，而是对方微笑的看着它，似是克制住了品尝下它灵魂的味道，所以拿出块灵魂晶石正咔吧、咔吧的吃着。**
- ⚠ **她不敢再靠近的理由**:⚠ **并非她不想继续上前，而是苏晓的血气简直就是她的克星**——⚠ **就像在积雪上泼炽红的铁水般。**

## 🧪 一瓶药剂洗回生前

⚠ **苏晓取出几瓶药剂，在她疑惑的目光中完成了药剂调制**——⚠ **随后将这半透明药剂泼在她身上**;⚠ **嘶啦一声，她发出一声尖哮，随后身上的些许猩红快速褪去，逐渐成为正常的灵魂体**——⚠ **奇妙的是，她身上的衣裙竟逐渐呈现出女仆装扮，似乎是变回了生前的模样。**

- ⚠ **她的演技**:⚠ **逐渐飘起，距离地面一米高处停下**——⚠ **那迷茫的目光，仿佛根本不记得之前所发生的一切**;⚠ **她眼中带着胆怯的问道**:「**这位大人，是您救了我吗。**」
- ⚠ **这一幕实在太像机缘巧合下招募到女仆一名**——⚠ **倘若是世界之子般的热血少年，还真就没法招架住这女仆攻势**;⚠ **尤其是从猩红幽魂向无害灵魂体转化时，那特别自然的女仆便装**——⚠ **不知道的，还认为这是受到了洗礼，整个灵魂都向友善转化。**
- ⚠ **苏晓看破不说破**（⚠ **他对这种特别有演技的有点感兴趣，因为足够好的演技，有时可以发挥出很大作用**）——⚠ **或许是心中感觉到已被识破，她也就没顺着思路演下去，安静的飘浮在那，被动等待事态变化。**
- ⚠ **测谎**:⚠ **苏晓直视她的眼睛**——⚠ **虽说现在已经没有「猩红美瞳」，但双眼会透出更浓郁的灵魂颜色**;⚠ **如若说谎，会考虑更多，这会导致对方眼中的灵魂颜色更深。**
- ⚠ **她说的话**:「**大人，你只能出去或者继续深入，下面的地宫是绕远路的入口，您为什么要去那？**」——⚠ **由此苏晓确认，[布布汪](../布布汪.md)、[巴哈](../剧毒魔鹰·巴哈.md)、[罪亚斯](../画中世界/罪亚斯.md) 他们并没进入极暗古堡，而是被挡在了下方的迷宫内**;⚠ **至于他自己为何能进来，这可能与他之前在那个灭法宝库内破除了黑暗封印有关。**
- ⚠ **她的耐心**:⚠ **方才还在升降梯内乖巧等待，苏晓从检修口回来时，已经不知道跑哪去了**——⚠ **这小女仆演技是不错，就是耐心差了点**;⚠ **不过当苏晓抵达一层、并未去按动开门机关时，一根灵魂形态的手指暂时具现按压在上面**——⚠ **灵魂女仆不知何时又出现了，她的魂体比方才要透明些。**
- ⚠ **她的知识边界**:⚠ **她生前只是这城堡的一名侍女**——⚠ **连进入这座猩红内堡的资格都没有，更别说来到这猩红亲卫长都没资格来的第六层**;⚠ **因此她讲的城主故事，只是版本之一——真正的版本，在六层城主书房的那本笔记里。**
