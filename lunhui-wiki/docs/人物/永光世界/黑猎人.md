# 黑猎人（殁）

监视者高塔一层回合战中，**违规者阵营商议后决定的首发**——⚠ 他**一个闪身就到了黄沙场地上**:身着**短风衣**、戴着**头罩**，⚠ **手上漆黑一片，还有粘稠的液质从他手部渗出**；⚠ **滴落在地上的黄沙后，发出嘶啦一声腐蚀声**。⚠ **黑暗液质在他周边飘浮，整个黄沙场地上的空气都因此而扭曲。**

## 💀 三次误判

> ⚠ 「**很久之前就听过斩首的夜，没想到今天能交锋——可别说，我的能力卑鄙。**」

- ⚠ **误判一·攻击强度**:战斗开始，**魔灵从斩龙闪内放出**，几乎同时**血气虚影在苏晓身后出现，拉满灵魂大弓，将化为黑烟箭矢的魔灵射出**——⚠ **轰！魔灵箭被硬化的黑色液质挡住，犹如铁砂般炸散**。⚠ **这让屏障后的黑猎人心生疑惑**:⚠ **这名气不小的斩首的夜，攻击速度是很强，可攻击强度，真的不怎么样。**
- ⚠ **误判二·「看不起人」**:下一瞬他就注意到——⚠ **屏障前方的魔灵忽然消失，几十米外的苏晓骤然出现在此**；⚠ **并未拔刀，而是一脚朴实无华的直踹**。⚠ **这让黑猎人心生几分怒意**:⚠ **斩首的夜历来以刀术宗师闻名，此时与他对敌竟都不拔刀——这也太看不起人。**
- ⚠ **误判三·屏障**:虽心中这样想着，⚠ **但黑猎人依旧加强了身前液质屏障的防御力**——⚠ **然后在他的能量感应中，这屏障突然消失了**。⚠ **凛冽的劲风迎面而来，把他的头发呼的一声吹成倒竖发型**；⚠ **身上的起源级布甲级防具，竟发出撕布般的声响，裂开一道道细痕。**
- ⚠ **终局**:⚠ **黑猎人的瞳孔已缩小到极致，他立即启动保命装备准备传送到百米外**——⚠ **可惜在这「飞升级」攻击判定的一脚直踹下，那保命指环咔崩一声破碎**。⚠ **啪叽～** ⚠ **一面透出金色的无敌屏障出现在了贝芙丽、小戈、驭虫师等人前方**——⚠ **位于这无敌屏障上，均匀地分布着黑猎人。**
- ⚠ 【提示：**你已击杀违规者 18＊＊＊50 号，你获得荣誉勋章 ×2 枚**】——⚠ **没有猩红卡掉落。**

## 😰 他给对面留下的心理阴影

⚠ **贝芙丽、小戈、白骨男、驭虫师、魂鬼彼此对视，心中竟有些庆幸**:⚠ **幸好不是擂台战啊，否则都可能会出现一脚一个的骇人场面。**

- ⚠ 「**问……问题不大，每个人只能打一轮。**」——[小戈](../幽暗大陆/小戈.md) 开口；⚠ **听闻此言，白骨男、驭虫师、魂鬼纷纷附和**:⚠ **都已经开始竞技了，直接投降属实不甘心**——⚠ **这种被监视者高塔召来的机会，很难遇到。**

## 关系

- [贝芙丽](../幽暗大陆/贝芙丽.md) / [小戈](../幽暗大陆/小戈.md) / [月巫](月巫.md)：与他同批被召来的六名违规者——他成了他们「幸好不是擂台战」的理由。
