# 老镇长·利戈迦（殁）

⚠ **一名身着黑袍、拄着拐杖的老者**——⚠ **满脸慈蔼笑容，脸上的皱纹让他看起来更慈眉善目**:「**年轻人，你是来找那些宝藏的吗。**」

## 🍲 「贪婪的人，都走不出这小镇」

⚠ **他把苏晓带到地下的君主宝藏存仓处**——⚠ **用拐杖一点墙壁，一扇岩石暗门升起**;⚠ **可当苏晓进入后，他就在暗门外的大石屋内，往一口混杂着猩红的浓汤里搅动**:

> ⚠ 「**年轻人，不要急，等过去几十年，上百年后，你就会用你的所有财富，祈求我放你出来**——⚠ **虽说，我就算收了你的财富，也不会放你出去，呵呵呵呵……**」

- ⚠ **他见过太多来寻宝的人**——⚠ **只不过，这些人无人能离开此地**;⚠ **既带不走那些被诅咒的秘宝，还要将性命搭上。**
- ⚠ **他放心的理由**:⚠ **就算是中游梯队的绝强者，也对这加持了猩红力量的暗门没办法**——⚠ **他已经感察过，来人虽血气惊人，但应该晋升绝强不久。**
- ⚠ **翻车**:⚠ **咚！！一声炸响，不等他反应，他已经没入到侧面的墙壁内**——⚠ **破损到只剩三分之一的暗门轰破墙壁飞出，将小镇中心的怪兽雕塑击碎大半**;⚠ **他透过墙壁的破洞，看着破碎大半的怪兽雕塑，满是皱纹的老脸狠狠抽动了下**——⚠ **苏晓**:「**这么说，你是准备把我困死在这条长廊里。**」（⚠ 踏了踏有点发麻的右脚——⚠ **这暗门的确挺坚固，踹的有点脚疼**）

## 😨 被血兽吓冷静

⚠ **铮～斩芒凭空出现，利戈迦的头颅被斜斜斩下一块**——⚠ **他再也不能保持伪装，化为全身皮肤灰白、有着一双猩红眼睛的大头怪物**;⚠ **伤口处涌现猩红线虫**——⚠ **他满嘴尖牙咬到咔咔作响，当即准备扑杀苏晓**;⚠ **可就在这瞬间，他忽然感知到，前方这人族身后似有一只庞大的血兽，正露出交错的尖牙、狞笑着等待它主动扑上前**——⚠ **直面血兽的恐惧，让他心中的愤怒荡然无存**;⚠ **在苏晓 −21 点魅力属性的「友善交涉」下，他迅速冷静下来。**

## 🕯 他讲出的真相（然后死于一句谎言）

⚠ **这片区域其实是一处独立存在的空间**，⚠ **大小约是庇护城的二分之一**——⚠ **上层除狂笑镇外，周边全是枯死的树木以及游荡的腐烂巨人**;⚠ **详见 [暴君·图尔](../暴君·图尔.md) 与世界页的地下世界一节。**

- ⚠ **他也曾是暴君的手下之一**:⚠ **暴君的寄生物失控时，他与几名同僚见势不妙，就炸毁了通往下层君主地宫的空间之门**——⚠ **将那些畸变的怪物都困在下层**;⚠ **他虽逃过一劫，但很快发现，因体内的寄生物，他已经无法离开这片区域**——⚠ **这些完全活化的寄生物只能在有猩红之力弥散的环境内存活**;⚠ **一旦出了此地，寄生物会快速死去，与之共生的他也会死。**
- ⚠ **误入狂笑镇的人为何无法离开**:⚠ **就是因为先被此地的寄生物寄生，随后被迫与之共生**——⚠ **而他们之所以会狂笑，是寄生物逐渐侵袭到大脑所致。**
- ⚠ **他的死因**:「可能性很低，除非我带你去那处破碎的空间之门……」——⚠ **话说到一半，几道斩芒闪过**;⚠ **到死为止，他都没理解，为何苏晓这么干脆利落就出手**——⚠ **啪叽一声，他的血肉炸裂开，里面一只流体般的寄生物迎面扑来，还在半空中就被斩成两截**（⚠ **附带斩魂特性的攻击，就算这寄生物没要害，也被当场格杀**）。
- ⚠ **苏晓斩他的原因**:⚠ **对方在撒谎，而且是几句真话中隐藏着一句谎言**——⚠ **如何去往下方的地宫，苏晓已经想到:被那贪婪之口吞掉，自然会抵达君主地宫内。**

## 关系

- [苏晓](../苏晓.md)：把他困人的暗门一脚踹飞、又被他一句谎言送了命的人。
