# 星界吞噬者

**无光神殿四巨头**之一——⚠ **本世界最强的三个灭世级存在之一**（另两位是 [永暗之主](永暗之主.md) 与 [始祖](始祖.md)，第四位 [深渊大主教](深渊大主教.md) 则是本世界最强的**深渊存在**）。⚠ **它已被囚禁九万年出头**，在四巨头中**刑期排第二**（仅次于始祖的十几万年）。

## 📏 它是本世界的「恐怖标尺」

⚠ 苏晓刚到永光世界不久，就在一处沙丘庇护所外撞见一只**深海邪灵**:⚠ **那赫然是一只巨大的眼睛**——**如怨灵般完全苍白、遍布黑红的血丝**；⚠ **那感觉就像有一个高大的厉鬼正站在外面的沙水中，一手扒着沙丘庇护所，一只眼睛死死盯着庇护所内的活物**。⚠ **满是倒刺的长舌在庇护所外部舔过，让庇护所内的术式一阵明暗不定**；⚠ **它两只「大手」一捏，庇护所四壁都开始凹陷下来**。

- ⚠ 苏晓对这东西的评价，正是以星界吞噬者作参照:⚠ **「这玩意非常不好惹，虽说达不到星界吞噬者那一级，但也只弱一梯队而已」**——⚠ **而刀术宗师克制这类怨魂、厉鬼，是建立在战力差距不是特别大的情况下**。
- ⚠ 苏晓最终只能**启动「灭法传送阵」硬撤**:⚠ **传送的空间冲击轰开了刺入庇护所的舌头与一根利爪**，⚠ **那满是倒刺的舌头好像还断了一截**——⚠ **一条几米长、有着倒刺的断舌被一同带回灰暗圣所的庭院，如同被剁下一段的水蛭般生命力顽强、肆意扭动，让人汗毛直立**；⚠ **上方映下的微光竟能灼烧这断舌，并在短时间内将其灼烧成灰烬蒸发**。⚠ 【提示：你已消灭深海邪灵（**1/30**）】· ⚠ 【你获得 **2680 点世界声望**】· ⚠ 【**你已被深海邪灵所憎恨**】。
- ⚠ **换句话说:一只「比星界吞噬者弱一梯队」的东西，就已经让九阶巅峰的苏晓只能开传送阵跑路。**

## 🤷 「随便吧，反正也都出不去」

⚠ 当 [永暗之主](永暗之主.md) 以无光神殿的神位去拉拢三方时，⚠ **深渊大主教与星界吞噬者的态度是:「随便吧，反正也都出不去」**——⚠ **只有被关了几千年、刑期最短的永暗之主，逃出去的想法最坚定**。

- ⚠ **它们的手下一直在做的事**:⚠ **这么多年来，无光神殿的骨干成员们，一直在永光世界各处破坏那些封禁**——⚠ **他们是本世界最强三个灭世级存在、外加一个本世界最强深渊存在的手下，自然不在乎更多灭世级存在被放出来。**
- ⚠ **这也是永光世界越来越危险的根因之一**:⚠ **绝强者实力的灭世级存在，本世界有一群**——⚠ **没错，就是一群**；⚠ **风海大陆都没多少的绝强者，这里却很多**。⚠ **这是攒的**:灭法时代前哪个世界都有灭世级存在，⚠ **但在灭法时代结束后，九成以上的灭世级存在都在永光世界关着呢。**
- ⚠ **而先代灭法的后手，恰好是它们的克星**:⚠ **只要开启监视者高塔的「超·界级封禁术式」，就算是深渊大主教、星界吞噬者、始祖、永暗之主这种至强实力的灭世级存在，其实力也会被压制到绝强级**——⚠ **这并非是针对单独个体，而是直接向下拉整个世界的战力上限。**

## 关系

- [苏晓](../苏晓.md)（最后的灭法之影）：把它关在这里的阵营的最后一人——他刚到本世界就以「比星界吞噬者弱一梯队」来给一只吓退他的深海邪灵定级。
- [永暗之主](永暗之主.md) / [始祖](始祖.md) / [深渊大主教](深渊大主教.md)：无光神殿的另三位巨头——它与深渊大主教是那两个说「随便吧，反正也都出不去」的。
