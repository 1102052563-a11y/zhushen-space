# 熔火巨人

万界**三大传说铁匠**的最后一位（另两位是 [恶魔铁匠](../幽暗大陆/恶魔铁匠.md) 与 [矮人王](../风海大陆/矮人王.md)）——⚠ 苏晓 [猎杀名单·血契](../../力量体系/装备道具/index.md) 上那笔 **2500 盎司时空之力**悬赏的目标:「**找到熔火巨人，并成功营救**」。

## 🔨 一座失去色彩的工坊

⚠ 监视者高塔**第十三层**内**没有色彩**——⚠ **入目之处皆为黑白两色，仿佛此地的色彩都被抽离**；⚠ **这是一处破败的工坊，因十三层内没有色彩，让这里显得更孤寂**。⚠ **当！当！当……** 锻造锤的敲打声传来——⚠ **唯有铁匠炉里燃烧的火焰，让周边的事物恢复了些许色彩**。

- ⚠ **外形**：一道**十几米高的身影**坐在那，拿着个**小一号的锻造锤**敲打一块粗坯（⚠ **这锤子在他手中看起来小，其实锤头都有近半米长、水桶粗**）。⚠ **皮肤粗糙、胡须编成须辫垂下、头发蓬松地披散在身后**；⚠ **右臂上镶满金属甲片，看起来就像熔在了皮肤上一样**。
- ⚠ **口音**：⚠ **说话偶尔会顿一下，似是口舌曾受伤，说话不算太清楚**——「好久不见的……**朋友**，你们……去哪了。」
- ⚠ **他的性格**:⚠ **三大传说铁匠中，相比恶魔铁匠的孤傲、强大，矮人王的威严、沉着，熔火巨人是最和善的。** ⚠ 苏晓没回答他那句「你们去哪了」，他只是点了点头:「**不想说……也没关系，虽然我们是……朋友，但每个人……都有……属于自己的秘密。**」⚠ 说完又专心于锻造台上的铠甲粗坯。

## ⏳ 「等过几千年……上万年」

⚠ 「你是被困在这？」——⚠ 「**原来不是，后来这里的传送阵坏掉了；不用……担心我，等过几千年……上万年，这座高塔没被虚空树公证的部分，就会塌掉，到时就能出去。**」

- ⚠ **他是真的不着急**:⚠ **对于其他生灵，几千年上万年很漫长**——⚠ **可作为最后一位火巨人族，熔火巨人睡一觉都可能是几百年**；⚠ **因此这对他来讲，并不算漫长的时间。**
- ⚠ 苏晓只留下一句「**过会带你出去**」，随后在工坊里侧找到那处**已经坏掉的定向传送阵**——⚠ **直接传送出去的功能没可能修好，但向十四层传送还是没问题的**（⚠ **这方面的术式，特意加强过**）。⚠ 完成十四层的一切后，苏晓**回到十三层，叫醒正在打鼾的熔火巨人**，先带他传送到十四层，再与 [暗之女](暗之女.md) 一同传送到高塔最底层——⚠ **营救完成。**
- ⚠ **他怀念的东西**:在灰暗圣所里，他**目光出神地坐着**——⚠ 「**嗯～，好怀念永暗树的果实。**」（⚠ 苏晓回忆了下，**还真没听过永暗树**）

## 🛡 【余烬之火】× 【狂猎之夜】

⚠ 苏晓取出 [【狂猎之夜】](../../力量体系/装备道具/index.md) 和老兽王遗留的 **【余烬之火】**放在工作台上——⚠ **看到这两件物品，熔火巨人一时间犯了难，最终点了点头**:⚠ **他有办法用【余烬之火】增强【狂猎之夜】**。⚠ **关于防具方面，熔火巨人是三大传说铁匠中最擅长的。**

## 关系

- [苏晓](../苏晓.md)（最后的灭法之影）：把他从困了不知多少年的高塔十三层带出来的「朋友」——他则接下了用【余烬之火】提升【狂猎之夜】的活。
- [恶魔铁匠](../幽暗大陆/恶魔铁匠.md) / [矮人王](../风海大陆/矮人王.md)：与他并列的另两位传说铁匠——⚠ **相比恶魔铁匠的孤傲强大、矮人王的威严沉着，他是最和善的**。
- **火巨人族**：⚠ **他是最后一位**——这也是他觉得「几千年上万年」不算漫长的原因。
