# 始祖

**无光神殿四巨头**中**刑期最长**的那一个——⚠ **它足足被囚禁在此已有十几万年**：⚠ **永光世界作为囚笼没多久，始祖就被关进来。**

## ⏳ 十几万年，一栋建筑那么大

⚠ 四巨头的刑期排序:⚠ **始祖（十几万年）＞ [星界吞噬者](星界吞噬者.md)（九万年出头）＞ [深渊大主教](深渊大主教.md)（差不多五万年）＞ [永暗之主](永暗之主.md)（几千年）**。⚠ 而始祖的刑期不只是最长的——⚠ **要知道，它被封印时，活动范围不是一个世界这么大，而是在只有一栋建筑大小的地方，封印十几万年。**

## 🌀 愤怒 → 忏悔 → 麻木 → 疯狂

⚠ 这是全书对一个灭世级存在的心理曲线，写得最完整的一次:

- ⚠ **最开始，它愤怒又孤傲。**
- ⚠ **但在十万年过去后，它开始为曾经的恶忏悔**——⚠ **后悔曾吞噬掉近百个世界的所有生灵。**
- ⚠ **紧接着，始祖都被关的麻木。**
- ⚠ **直到最后，这灭世级存在有点疯狂了。**

## 💀 「弄死苏晓」＞「逃出去」

⚠ 也因此，⚠ **始祖是无光神殿的四巨头中，最恨灭法之影、也最疯狂的一个**——⚠ **如果在逃出本世界与弄死苏晓间选择，始祖会毫不犹豫地选择后者。**

- ⚠ 当 [永暗之主](永暗之主.md) 以无光神殿的神位去拉拢四巨头时，⚠ **始祖最提不起兴趣**（深渊大主教与星界吞噬者的态度好歹还是「随便吧，反正也都出不去」）。
- ⚠ **好消息是，现在这些灭世级存在，都不知道苏晓已经来此。** ⚠ 这也是苏晓一进本世界就佩戴【违规者】称号伪装的原因:⚠ **他作为灭法者进入本世界，身份一旦暴露，立即会引来所有灭世级存在与深渊滋生物的追杀**；⚠ **而只要他出手战斗、并且敌人是灭世级存在或深渊滋生物，他的身份会立即暴露。**

## 关系

- [苏晓](../苏晓.md)（最后的灭法之影）：它十几万年恨意的最终收件人——「如果在逃出本世界与弄死苏晓间选择，始祖会毫不犹豫地选择后者」。
- [永暗之主](永暗之主.md) / [深渊大主教](深渊大主教.md) / [星界吞噬者](星界吞噬者.md)：无光神殿的另三位巨头——它是其中对越狱最提不起兴趣的一个。
