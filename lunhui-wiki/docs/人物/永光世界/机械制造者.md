# 机械制造者

⚠ **并非所有灭世级存在都对生灵抱有极端的恶意**——⚠ **机械制造者就是特例中的特例**。⚠ **它对生灵没有直接的恶意**；⚠ **在黑铁城内，有着一个与永暗之主、深渊大主教等同样强大的灭世级存在，其名为机械制造者。**

## 😤 「在我的视线中消失，我随时等你来报复」

⚠ 它之所以被关在永光世界，得从一件事说起——⚠ **机械制造者凭自己造出的战斗机械，在它所在世界内无敌**；⚠ **但因为反感外出，对外界的情况了解的不多。**

- ⚠ 然后**有名绝强级灭法去那世界处理深渊之孔**，⚠ **双方因误会发生矛盾**——⚠ **机械制造者揍了那灭法一顿**，⚠ **然后还来了句:「在我的视线中消失，我随时等你来报复，记得把你的朋友也带上。」**
- ⚠ **次日清早**，机械制造者睁开眼时——⚠ **发现几名神情『和善』的男女，正围在他的机械床槽周边，满眼『善意』地看着它。**
- ⚠ **然后几位至强最巅峰的灭法，把机械制造者带到了永光世界**——⚠ **倒是没动手，而是要求机械制造者，让其建筑所有监视者高塔，此事就作罢。**

## 🔧 「创造的天堂」

⚠ **到后来，机械制造者感觉这里也还不错，就暂时没离开**——⚠ **既有灭法阵营的书库，还有各类外面找不到的罕有材料**；⚠ **简直是创造的天堂。**

- ⚠ **等它从自己的创造中回过神时，已经过去很多年**——⚠ **然后它得知了一个坏消息:灭法阵营消亡了，月狼们也都死去，永光监视者们更是不知所踪。** ⚠ **一个很严重的问题出现:机械制造者出不去了。**

## 🏙 黑铁城的靠山

⚠ **眼下给黑铁城撑腰的就是机械制造者**——⚠ **虽说这边只有它一个，可它是真的在乎黑铁城**。⚠ **反观无光神殿，那边虽说有四巨头，但四巨头更多是把无光神教当工具用**——⚠ **因此无光神殿不会轻易招惹黑铁城**；⚠ **眼下的黑铁城，已成为庇护城都不会主动招惹的势力。**

- ⚠ **黑铁城的由来**（从苦痛神教说起）:⚠ **苦痛神教的准则是「相信苦痛能激活潜力」**——⚠ **说的直白些，就是以外部环境刺激自身的适应力，从而产生对应的进化**；⚠ **也因此，这虽是个教派，却不信奉任何神灵**。⚠ **最初的苦痛神教，在不加持任何庇护的情况下直接走出阴影、暴露在极昼之光下**，⚠ **硬抗极昼之光炙烤的痛苦，想凭借坚定的意志力以及强烈的外部刺激，逐渐诞生昼光抗性**。⚠ **结果是成功了**——⚠ **这些成员成为苦行者，行走在荒芜的大地上**；⚠ **可惜，这种成果没能维持多久**:⚠ **一些苦痛者，其身躯开始硬化、或者说是金属化，最终在金属化的侵蚀中死去。**
- ⚠ **转机**:⚠ **苦痛者的意志力普遍比较高**——⚠ **在荒野上前行的苦痛者随后发现了一处遗迹**:⚠ **那是上个纪元的遗迹，本世界还是超脱·原生世界时，一个机械文明留下的遗迹**。⚠ **也是这个契机，苦痛神教发生改变**:⚠ **苦痛者用金属锻造出部分躯体，用这锻造出的金属躯体，换下他们已经金属化、死亡化的躯体部分**；⚠ **到了最后，他们连心脏等都换成机械器官**——⚠ **更极端些的，连部分大脑都用机械取代。** ⚠ **苦痛神教逐渐灭亡，黑铁城正式崛起**；⚠ **昔日的苦痛者也改为自称进化者**——⚠ **很有「血肉苦弱，机械飞升」的势头。**
- ⚠ **怒兽列车**：黑铁城几百年时间累积出的成果——⚠ **地下隧道网**（索恩斯给的地图上那**几十颗野兽头标志**都是列车站台）。⚠ **单节列车，车头怎么看都像金属构成的活物**（⚠ **一种坚固的经络组织蔓延到后方的车厢上**）；⚠ **速度稳定在几倍音速**。⚠ **安全度相当不错**:⚠ **如果在地表行进，哪怕能无视昼光，也可能遇到强大的野怪**——⚠ **反之，这有黑铁城维护的地下隧道，最起码比在地表行进安全百倍不止**。⚠ **不过没任何一处列车站台，敢于涉足湿盐区、蚀雨地、荒芜沙漠这些超危险区域**（只抵达附近）。
- ⚠ **票价里的善意**：⚠ **按距离收取黑铁币**（庇护城附近到蚀雨地边缘过五站 = 50 黑铁币）——⚠ **可如果要去黑铁城，这最便宜:无论从哪出发，都是 5 枚黑铁币。** ⚠ **可以看出，这是黑铁城对本世界内居民的善意。**
- ⚠ **黑铁城的样子**：⚠ **深入地下几万米，地面距离上方的穹顶有万米以上的高度**——⚠ **甚至有飞艇模样的巨型飞行器在半空中飘飞**；⚠ **作为地下城并没有阴暗、潮湿感，城市上方透出微光**。⚠ **面积差不多有风海大陆永环城 2 个城区大小**（⚠ **别认为这很小:永环城的一个城区范围，比一些中小王国的面积都要大几分**）。⚠ **街边有不少商贩，说明此地的治安还可以，至少是有法度约束之地。**
- ⚠ **进出规则与庇护城截然相反**：⚠ **想进入黑铁城要经过几层关卡**（⚠ **一队目光锐利的巡视警卫上前，虽还算客气，但言语间充满陷阱，尽最大可能试探来历与目的**）；⚠ **反之，想要离开的只需接受简单的检查即可**——⚠ **庇护城则是进入容易，想出来，常规方法几乎不可能。**
- ⚠ **它的批准**：[食暗者](../风海大陆/黑眼存在.md) 去黑铁城**见了它一面**，表示**自己虽然是灭世级生物，但不吞噬生灵，准备留在黑铁城**——⚠ **得到机械制造者的允许后**，食暗者才开始在城内寻找能解谜之人。

## 关系

- **那名绝强级灭法**：因误会被它揍了一顿的倒霉蛋——它还嘴硬「记得把你的朋友也带上」，结果第二天床边围了一圈至强巅峰的灭法。
- [苏晓](../苏晓.md)（最后的灭法之影）：把它关进来的阵营的最后一人——它建的监视者高塔，如今成了苏晓晋升绝强的试炼场。
- [永暗之主](永暗之主.md) / [深渊大主教](深渊大主教.md) / [始祖](始祖.md) / [星界吞噬者](星界吞噬者.md)：与它同等强大、却把无光神教当工具用的四巨头——⚠ **而它是真的在乎黑铁城**，这正是无光神殿不敢轻易招惹黑铁城的原因。
- [食暗者](../风海大陆/黑眼存在.md)：向它报备「我不吞噬生灵、想留在黑铁城」并获批准的同类。
