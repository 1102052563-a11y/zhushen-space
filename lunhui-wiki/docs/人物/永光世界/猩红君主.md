# 猩红君主

⚠ **本世界的几大区域中，沙之海位于北边**——⚠ **沙之海的最中心，就是炼狱噩梦**;⚠ **很多本世界的原住民认为炼狱噩梦是危险之地**——⚠ **其实这更像是一层封印，封着里面的猩红噩梦**;⚠ **那才是真正可怕的存在——猩红君主。**

## 🩸 猩红之力的根源

⚠ **[机械制造者](机械制造者.md) 转交的【猩红之眸】上说，猩红的根源就位于猩红噩梦中**——⚠ **而暴君仿造的【[猩红指环（伪）](../../力量体系/装备道具.md)】所提及的「猩红君主」，就是这猩红的根源。**

- ⚠ **真正的猩红之力，并不显得诡异与渗人**:⚠ **猩红君主所掌握的猩红之力，是霸者般的强横与侵袭**——⚠ **也可以是鲜血与火焰相加的力量**;⚠ **而到了 [暴君·图尔](../暴君·图尔.md) 那边，猩红之力的侵袭就变成侵蚀，或是诡异的寄生物。**
- ⚠ **猩红之力的来历（苏晓的推测）**:⚠ **[暗血城主](暗血城主.md) 父子准备在本世界开启深渊通道，结果通道打开、涌现的却是猩红之力**——⚠ **这让人怀疑，猩红之力是否是深渊能量所扭变而出**;⚠ **被深渊能量增益属于次级深渊特性，可扭变则近乎与深渊能量同梯阶**（⚠ **就算低，也只是 100 点与 95 点这种差距**）;⚠ **深渊、元素是两种最强的源质力量**——⚠ **如果猩红之力真的是由深渊能量所扭变而来，那这种力量将相当可怕。**
- ⚠ **暗血城主父子放出猩红并非巧合**:⚠ **而是猩红君主正酝酿着冲破封印的手段**——⚠ **哪怕处于噩梦封印中，猩红君主依旧有大量的猩红之力蔓延出**;⚠ **这代表它能通过某种方法，不断凝聚猩红之力，即使身处封印中。**

## 🔒 三重封印

⚠ **苏晓的判断**:⚠ **单是沙之海的噩梦之地，都不足以封印住对方**——⚠ **还需要天空中的血月，以及这个地下世界，才能让对方被困在封印中。**

- ⚠ **地下世界的真相**:⚠ **能打造出这地下世界的，只有灭法阵营**——⚠ **他们委托思林特斯矮人打造出这地下世界**;⚠ **从噩梦封印内抽离的猩红之力，都释放到此地，然后通过此地的术式，让这些猩红之力结晶、最终将其封存起来**——⚠ **因为一旦被猩红君主积攒到足够的猩红之力，噩梦封印很可能困不住他**;⚠ **因此不仅要封困，还要不断抽离这可怕存在的猩红之力。**
- ⚠ **天空的血月**:⚠ **地下世界上方的星空间有着圆形的血月**——⚠ **血月上有着暗影，就像一根权杖般**;⚠ **权杖上方的血月部分，血色格外明显，隐约构成王冠形状。**
- ⚠ **它是先代灭法封印的最可怕存在**:⚠ **可怕到，就算 [无良导师](../马文·华尔兹.md) 与 [老灭法](../星空座/老灭法.md) 都没提及这存在**——⚠ **更不会告诉苏晓开启那封印的方法，避免苏晓去挑战。**
- ⚠ **⚠ 连四巨头都忌惮它**:⚠ **[始祖](始祖.md)、[星界吞噬者](星界吞噬者.md) 的封印，是最先挣脱封印的 [永暗之主](永暗之主.md) 所打开**——⚠ **可哪怕到了今天，无光神殿·四巨头都没去打开猩红君主的封印**;⚠ **这有一种可能，就是四巨头同样忌惮猩红君主，不敢将其放出来。**

## 👑 曾经的猩红帝国

⚠ **地下世界内的 [猩红阵营](../../世界/任务世界/永光世界-厄难囚笼.md) 不是暴君死后畸变出的怪物**——⚠ **而是有着纪律等级、独特语言、建筑风格、力量传承的成熟势力**;⚠ **苏晓的猜想是:这是猩红君主曾经所统治的帝国，悄然来到了永光世界，准备打开噩梦之地的封印。**

- ⚠ **他们的目标**:⚠ **救出猩红君主，让这位恐怖君主再度统领他们**——⚠ **让万界重新见识到猩红军团的姿态**;⚠ **以及一整个世界被猩红笼罩、压碎，最后只剩结晶的世界之力，镶在血纹圆盘上**——⚠ **给予所有猩红之力掌控者无穷无尽的生命源质，让其近乎永生。**
- ⚠ **和暴君的对比**:⚠ **和猩红君主相比，暴君哪怕戴上「灵魂王冠」那都是弟弟**——⚠ **倘若被猩红君主戴上「灵魂王冠」，那将是一场浩劫。**
- ⚠ **[猩红权杖](../../力量体系/装备道具.md)**:⚠ **这大爹级原罪物，正是猩红君主的象征**——⚠ **可现在他们的君主身处噩梦封印中，整个猩红阵营无人有资格拿起它**;⚠ **详见世界页。**
