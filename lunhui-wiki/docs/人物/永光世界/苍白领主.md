# 苍白领主

⚠ **寄星蟹巢穴地下、一处遍布微光丝线的区域中心**——⚠ **一颗苍白的海胆状晶石，宛如有生命力般一下下跳动**（⚠ **看到就想到跳动的心脏**）;⚠ **爆碎开后，里面是一团跳动的血肉**——⚠ **苏晓单手拿起时，眉心一阵阵刺痛**（⚠ 【警告:你正在受到精神力侵蚀！】）。

## 🦀 寄星蟹的真相：聚是灾祸，散是珍馐

⚠ **先代灭法们真正要封印的，并非寄星蟹，而是这【苍白领主】。**

- ⚠ **没有苍白领主的寄星蟹**:⚠ **是超凡特性的高阶蟹类生物，但对正常世界根本不感兴趣**——⚠ **它们只喜欢寄生在已凋零、腐朽、没有生灵的世界里，慢慢蚕食**;⚠ **平常没有族群意识**（⚠ 公蟹争配偶扬钳打架、争食物干架是常事）;⚠ **⚠ 只要数量不超危险范畴，这玩意是无害的。**
- ⚠ **蟹肉的美味**:⚠ **蚕食凋零世界后蟹肉异常美味**（⚠ 曾有「蟹猎手」靠猎捕它们发财）——⚠ **可若蚕食了正常世界，血肉就腥臭难闻、还带侵蚀力**;⚠ **⚠ 苏晓的见闻由此提升:今后在荒芜世界遇到寄星蟹，第一反应不是跑，而是赶快追、带回去让 [夏](../夏.md) 清蒸。**
- ⚠ **有了苍白领主，寄星蟹才成灭世灾祸**:⚠ **一个有强大精神控制力 ＋ 不死不灭 ＋ 无限分裂繁衍，一个有啃食世界的本领**——⚠ **苍白领主让寄星蟹快速分裂繁殖，寄星蟹能啃食世界又让繁殖速度更骇人。**
- ⚠ **它才是灭法阵营的仇敌**:⚠ **一直是它在影响所有寄星蟹的判断，让它们认定「灭法者是威胁生存的敌人，一定要消灭」**——⚠ **⚠ 连本世界以外的寄星蟹，也会一定程度受它这种影响**;⚠ **⚠ 只要消灭它，寄星蟹就变回正常高阶生物，遇到强者只会像正常生物那样逃走。**（⚠ 这解释了苏晓以前在恶魔古堡、黑渊见到的寄星蟹为何时强时弱）

## 🔒 被封入死灵之书

⚠ **先代灭法们把它封在此地，是因为无法消灭它**——⚠ **这么多年来，其他灭世级族群的不死不灭都受永光囚困影响，唯独苍白领主所受影响不大。**

- ⚠ **苏晓的判断**:⚠ **封印不可行**（⚠ 思林特斯矮人的封印学都困不住猩红君主，先代灭法未必更强）· ⚠ **消灭没明确方法** → ⚠ **想办法吞噬是最稳妥的**;⚠ **⚠ 但怕它的精神操控会影响恶魔虫族。**
- ⚠ **对策**:⚠ **苏晓取出「[原罪之书](../../力量体系/装备道具.md)」第二页的「死灵之书」**——⚠ **死灵之书一出现，苍白领主立即蔓延出暴躁乃至狂暴的精神力**（⚠ 苏晓大脑如被精神尖针刺入）;⚠ **死灵之书的半透明触须把苍白领主缠绕拖入其中**——⚠ **⚠ 死灵之书反馈来几分满意、还构成一个虚空数字「1」**（⚠ 苏晓后续计划的成功率至少提升三成）;⚠ **消灭它即算彻底灭掉寄星蟹这灭世级族群**（⚠ 传说任务完成度达 46.9%）。

## 关系

- **寄星蟹**：被它当工具的高阶蟹类——失去它后，寄星蟹就是无害的（还挺好吃）。
- [苏晓](../苏晓.md)：识破「寄星蟹不是真正敌人」、把它封进死灵之书的人。
- **死灵之书**：吞下它、还反馈来一个虚空数字「1」表示满意的原罪物。
