# 猩红长男

⚠ **一名肥硕又高大的超级大胖子**——⚠ **他身上肥肉可谓是层层叠叠，稍有移动就会甩动这些肥肉**;⚠ **这已经不是肥胖，而是以此畸变出了强大能力。**

## 💰 差 30 秒就跑掉了

⚠ **苏晓踹开四层门扇时，他正抓着一大把灵魂晶魄，向一个大皮袋内塞**——⚠ **一旁的传送阵已经激活**;⚠ **幸好来的及时**——⚠ **这极暗古堡探索的，不担心打不过 boss 级敌人，而是更担心这类敌人带上家底就跑。**

- ⚠ **三层的同行已经跑成功了**:⚠ **苏晓连斩两层的 boss 后，占领第三层的「理智野兽」惊的不轻，果断跑路**——⚠ **苏晓抵达三层时，只见各类古旧家具被翻乱、收藏品摔得粉碎、保险箱式的墙上暗格被打开、门都没来得及关**;⚠ **传送阵还热着，分明是刚使用过。**
- ⚠ **他的反应**:「**你，你是灭法者？**」——⚠ **苏晓没说话**;⚠ 「**啊～，吓死我了，一层的哈格姆和二层的虫什，不到五分钟就全被宰了，是灭法者啊，太好了。**」
- ⚠ **苏晓依然没说话，似是感觉到危险，他赶紧解释**:「**我虽然因为猩红畸变了，但成功和猩红共存，所以理智清楚**——⚠ **是 [索恩斯](索恩斯.md) 让我在这守着，避免五层的 [城主夫人](城主夫人.md) 冲破黑暗的封困，她是这里畸变程度最深、最可怕的猩红畸变者。**」
- ⚠ **为了证实所言**:⚠ **他还拿出一枚只剩半截的监视者指环**——⚠ **虽说这指环已经不能用，但也能代表身份**;⚠ **外加说出了索恩斯的特征、以及双方约定的诸多细节**——⚠ **显然是确有其事。**
- ⚠ **在正常的任务流程中**:⚠ **快要击败猩红长男时，他会透露自己并非猩红城堡这一阵营的，而是在与永光监视者·索恩斯合作**——⚠ **负责看守通往寝殿的入口**;⚠ **既是避免有人把城主夫人放出来，也防范城主夫人破坏寝殿正门后从黑暗区域内冲出来。**

## 关系

- [索恩斯](索恩斯.md)（杂货哥）：他的合作方——那枚只剩半截的监视者指环，是这段合作的凭证。
- [城主夫人](城主夫人.md)：他在这里防着的人。
- [苏晓](../苏晓.md)：不到五分钟斩掉一层与二层 boss、把他吓得赶紧自报家门的人。
- **理智野兽**：三层的同行——比他跑得快；后来还给 [灵妖](../灵妖.md) 发了个支线任务。

## 💀 暗血城主二子的真相 · 濒死交易（51卷37–38章）

⚠ **苏晓重回地下世界废墟时，在一处密室里发现濒死的猩红长男**（⚠ **被噩梦侵袭、身上血烟升腾、伤口无法愈合**）:「**猩红君主逃出来了，我亲眼看到了**」→「**我知道猩红君主的弱点，快救我**」→「**抱歉我刚才撒谎了，那应该不是猩红君主，更像是噩梦**」。

- ⚠ **他的求生天赋**:⚠ **上次在猩红城堡，他谎称是 [索恩斯](索恩斯.md) 的小弟、拿出半截监视者指环骗过苏晓**——⚠ **真相是:某次索恩斯偶遇他、见他用猩红之力想除掉他，是他哭求「我是猩红城堡的、与地下世界猩红阵营没关系、愿看守城堡怪物」才饶他一命**;⚠ **⚠ 他这「绝境求饶 ＋ 不结仇的谎言」的本事，是一种天赋。**
- ⚠ **苏晓的怀表酷刑**:⚠ **取出 [圣女座](../星空座/圣女座.md) 送的蒸汽朋克怀表、按下十秒倒计时**——⚠ **急促的机械声给他巨大精神压力**;⚠ **他在生命最后几秒智商最高光——⚠ **挤出情报:「星界吞噬者被那噩梦重创了，一定受了很严重的伤」**（⚠ **可信度九成以上**）;⚠ **苏晓喂药救回他，又弹进一颗「猛毒」**（⚠ **其实是维生素丸——⚠ **任何药师都查不出、成本低，真猛毒比永久增益药剂还贵、保存又短**·⚠ 阿姆用终端给他放了段猛毒爆发的影像镇场）。

### 🩸 他是暗血城主的二子

⚠ **苏晓的推断**（后经神父书房印证）:⚠ **当初 [暗血城主](暗血城主.md) 拼死断后，让「畸变但没失去理智的二子」带走最小的幼子**——⚠ **那个二子，就是猩红长男**;⚠ **他一直守猩红城堡四层（[城主夫人](城主夫人.md) 寝殿入口），对外说是防怪物逃出，其实是守住自己变成怪物的生母**——⚠ **不让生母杀食无辜，直到有人能战胜她、让她永远安眠。**

- ⚠ **他为何自称「猩红长男」**:⚠ **猩红城堡的猩红爆发，是城主长子一意孤行所致**——⚠ **二子在短时间内失去双亲和所有弟弟妹妹，对兄长的恨意极深、又永远无法宣泄**;⚠ **他索性当这兄长从未存在过，自称为「猩红长男」。**
- ⚠ **他掌握开启猩红通道的方法**:⚠ **父亲与兄长的知识，很可能分享给了他**——⚠ **⚠ 这正是 [神父](../魔灵星/神父.md) 约他在地下世界见面、想帮他引荐加入猩红阵营换取「开启猩红通道方法」的原因。**
- ⚠ **他被苏晓当棋子**:⚠ **苏晓先让他逃去庇护城引出星界吞噬者、后又变卦让人截住他**——⚠ **最终他去庇护城的路成了斩杀星界吞噬者的诱饵局**（详见 [星界吞噬者](星界吞噬者.md)）。
