# 尤尤安（旅团一阶新人 · 放养工具人）

轮回乐园 [幻影旅团](../势力/index.md) 新加入的**一阶契约者**——性别难以判断（脑部曾受不可逆损伤、常出现认知 / 记忆错误，连自己的性别有时都要「手动确认」，最后确认是男孩子）。表面唯唯诺诺、老实巴交，实则**蔫坏**:平常唯唯诺诺、关键时刻重拳出击。他被 [苏晓](苏晓.md) 选作低阶资源的长期交易对象。

## 「黑心老爷爷」的交易（42卷81–82章）

苏晓开暗星宝箱得了 **狼蛛血脉**（前期无解、门槛极低、可让一阶契约者一路碾压到七阶初期的血统），却对血脉本身没兴趣——他学 [马文·华尔兹](马文·华尔兹.md) 当年当「黑心老爷爷」的手法:找旅团里潜质一般、但有远见的一阶新人，用血脉换取只在低阶产出、高阶再难获得的独有物资（基础·被动卷轴等）。他托旅团 **魔女** 联络了三名低阶团员（暗 / 舞妹 / 尤尤安），由 [巴哈](剧毒魔鹰·巴哈.md) 用**三张全空白的纸签**抽签——名为抽签、实为考验「谁敢承担与八阶契约者交易的未知风险」。尤尤安「抽中」（其实全空白·敢赌者胜），当场签约、拿狼蛛血脉换给苏晓一张 **基础被动·灵想**（精神力卷轴）。

- **被彻底「放养」**：苏晓给他配齐了里德（打装备）、[贝妮](狩之影.md)（买药剂）、黑帆商会黑商（收购）三条渠道，之后**完全放养、绝不再见面**——「最强烈的情绪是恐惧、最大的恐惧来自未知」，保持距离让对方摸不透自己，才能省去麻烦。他就此成了苏晓远程供货的「工具人」，就算哪天死了、损失也在可接受范围（当作累积放养经验）。

## 关系

- [苏晓](苏晓.md)（白夜）：用狼蛛血脉换他低阶特有资源、又把他彻底放养的「黑心老爷爷」。
- **魔女**（幻影旅团）：牵线介绍他的旅团成员（诅咒 / 魅力系）。
- [马文·华尔兹](马文·华尔兹.md)：苏晓这套「黑心老爷爷」手法的师承来源（间接）。

## 🆕 50 万投资案 · 500 万到账的震撼（45卷101–102章）

塞尔星结束后她通过烙印联络 [苏晓](苏晓.md)，怯生生申请一笔投资:她已晋升二阶、凭一阶良好表现拿到「生存试炼」资格，首位奖励是能开出七种顶尖基础被动（力/敏/体/智/魅/感知/灵魂）的「基础被动技能卷轴宝箱」——按契约，她所得的一切基础被动资源都归苏晓。她扭捏半天，末了才敢开口要 **50 万乐园币**「……可以吗，不行也能少一些」。苏晓嫌她啰嗦直接挂断，转头**打了 500 万乐园币** + 30 颗灵魂结晶 + 一大堆完美永久增益药剂，外加十几颗炼金炸弹和一颗最初版阿波罗（二阶被炸原地去世级），把她吓得险些「被富有之力击跪」、脱口而出「你难道是我失散多年的亲生父亲吗」。苏晓的条件很硬:必须「一定」夺到宝箱，失败就要做**三个世界进度的「素体」**（他有几种设想需要素体·「不会死，只是过程不算美好」）。
