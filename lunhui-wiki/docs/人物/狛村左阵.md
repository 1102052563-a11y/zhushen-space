# 狛村左阵（七番队队长 · 狼头半兽人）

[死神世界](../世界/任务世界/死神-尸魂界.md) 护廷十三队**七番队队长**——身高近三米、**人身狼头的「半兽人」**（非人类灵魂转变的死神·实为狼·平时戴竹制面罩遮容）。重义、谨慎、战斗经验极丰富。是 [苏晓](苏晓.md) 在瀞灵廷开猎后**选定的第二个猎杀目标**（第一个是白道门的兕丹坊）。

- **属性（队长级垫底）**：法力 2600·力 95（真实）/ 敏 85 / 体 98（真实）/ 智 60 / 魅 25。技能：死神体质（生命 +3000 / 法力 +2000 / 吸纳灵子）、白打（高级体术）、鬼道（仅擅破道）、先代狼族、斩魄刀**始解·天谴**（巨大化部分灵体）、斩魄刀**卍解·黑绳天谴明王**（召出巨型黑甲武士·动作与本体同步·有生命链接·武士受伤本体同位受对等伤）、人化之术（终极技能·暂不可用）。苏晓评：**最弱的队长都这么强、山本元柳斎该恐怖到何种程度**。

## 被「蓝染遇害」线索诱出（二十卷27–30章）

[布布汪](布布汪.md) 扮大老鼠、衔着半副沾血的「蓝染同款破眼镜」闯进狛村左阵居所——这关键线索把谨慎的他**单独引出**到流魂街。苏晓先用召唤死神 [影](影-左宗卫.md)（其旧识左宗卫）诱他深入、再现身亮明猎杀者身份。狛村震惊于「提供灵子球的孱弱科研人员」竟比他更强（细思极恐——他也吞过灵子球）。开战后他亲身领教**死神与刀术宗师的「刀术素养」鸿沟**——他的格挡在苏晓眼中「像在挥棒」（死神多靠灵压 / 鬼道 / 卍解、不屑技巧），被技巧型苏晓一脚踢退、刀芒逼出始解、不得不放**卍解黑绳天谴明王**（巨型黑甲武士）。苏晓却归刀掏出 [寂灭公爵](../力量体系/装备道具.md) 狙击炮——因天谴明王与本体**生命链接**（武士受伤本体同位受对等伤），三发元素弹（土 / 土 / 雷）打崩天谴明王、狛村本体胸口凹陷 + 喉咙被穿。苏晓没斩头、先用 [斩龙闪](../力量体系/装备道具.md) 黑色幽光**吞噬他的斩魄刀**（涨锋刃值 9%），再一刀枭首。**苏晓得世界之源 12.9%（累计 13.7%）+ [噬灵者](../力量体系/天赋.md) 永久 +30 法力（上限 6125）+ 传说级宝箱**（瀞灵廷声望反被扣 2 万、惹来通缉）。临死前他匆忙在兕丹坊尸体下刻「左宗卫·复活·黄泉·流魂街」示警。

## 关系

- **[影 / 左宗卫](影-左宗卫.md)**：被其认出的生前旧识。
- **沙也加**：他亲自选入七番队的女性成员（被影生擒作诱饵）。
