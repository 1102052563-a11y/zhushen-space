# 西奈（神皇冒险团新成员 · 六阶治疗系 · 弱气少女）

[轮回乐园](../总览/乐园.md) 里 [神皇冒险团](神皇.md) 新加入的成员——一名性格弱气的**六阶治疗系**契约者（战斗时并不弱气）。上个世界她原属的小团队被团灭、绝望之下来投靠神皇冒险团抱大腿（第40卷78章）。

## 经历

- **被「财富之力」击倒**：她做好了被剥削 / 迫害的心理准备（那张当筹码的技能卷轴已易手三次、上任持有者是个幸运到让人嫉妒却最终身死的二阶契约者），却发现这个对外极狠的六阶冒险团**内部氛围轻松**（狂狐是酒鬼、抹茶「有点没节操」、神皇常为团员琐事发愁），大有归属感。
- **卖卷轴给苏晓**：她与团队各占 50% 的一张卷轴 = **[基础被动·体魂](../力量体系/技能.md)**（体力 201 真实可解锁高阶·提升生命 + 肉体潜力上限）——[苏晓](苏晓.md) 用「太阳三件套」换走（她全程被苏晓掏出的太阳勇士套装 / 太阳契约的价值震到靠椅背上不动·抹茶掐她大腿让她「有点骨气」）。

## 关系

- [神皇](神皇.md) / 抹茶 / 狂狐：她投靠的神皇冒险团成员（抹茶像大姐姐般照看她）。
- [苏晓](苏晓.md)（白夜）：用太阳三件套换走她卷轴（基础被动·体魂）的交易对象。

## 出处

- 第40卷78章（神皇冒险团新加入的六阶治疗系弱气少女·卖基础被动·体魂卷轴给苏晓换太阳三件套）。
