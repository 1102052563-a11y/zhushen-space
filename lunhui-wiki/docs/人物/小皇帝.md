# 小皇帝

**小皇帝** 是 [斩·赤红之瞳](../世界/任务世界/斩赤红之瞳.md) 世界帝国的名义皇帝，实为 [大臣](大臣.md) 操控的傀儡（四卷22–23章）。

## 简介
十三、四岁、身材瘦弱、营养不良的少年，一身紫衣不伦不类、略显娘炮，毫无威严（说话带童音如小学生朗读）。幼年丧亲（双亲实为大臣所毒害），自小被大臣洗脑、言听计从；无心治国、被大臣诱导沉迷女色（纵欲过度致瘦弱·苏晓推测）。

**关键价值**：他能启动最强帝具「**至高之力**」——该帝具**须皇室血统才能启动**，否则只是废铁。大臣正因此留他做「钥匙」、不敢杀。**本世界主线任务即夺取「帝具·至高之力」的启动钥匙**（现持有者即小皇帝），故他是众多契约者的争夺焦点。苏晓初见即考虑擒他，因不确定钥匙是否在其身上而暂缓（四卷22章）。

## 被绑架取钥匙（四卷53–57章）
苏晓查得他**每夜与一群妃子纵欲**（淡绿头发、十五岁、全靠大臣每天喂「[皇室秘药](../力量体系/消耗品材料.md)」温养才没精尽人亡——大臣正是借女色诱他沉沦成废人）。革命军兵临城下他却毫无危机感（只因大臣一句「艾斯德斯会打退贼人」）。苏晓用安眠药迷晕他、以界断线套颈「遛狗」、逼他带路去祀堂取**至高之力启动钥匙**（他爽快答应、因笃信「钥匙须皇室血统、外人拿了也没用」，还试图策反苏晓）。

途中闹剧揭穿其**权力早被架空**：他怒令侍卫拿下拦路的队长，侍卫竟无人听命（队长靠演戏化解）——他几乎察觉「没有大臣，没人听我的」。在祀堂面对列祖灵位，他自问「我是不是没威严的皇帝」，苏晓直言「你是个昏君」。最终苏晓当众取走并完成主线、钥匙破碎，大臣冲出怒骂他「废物·不能启动至高帝具要你何用」——他这才惊觉一直最亲近的大臣的真面目（四卷53–57章）。

> 他是**单纯到可悲的昏君（而非暴君）**；帝国本是农耕小部落起家、千年扩张成大陆最强、靠 48 帝具「皇家护卫军」横扫两百年，到他这代却帝具尽失、亡国在即——「前人栽树后人乘凉，他已不是乘凉、而是在吃树」。

## 跳洞逃生 · 向导（四卷58–63章）
被大臣当众抛弃后，小皇帝万念俱灰、立誓**复仇**，主动向苏晓透露一条生路（他第一次触碰至高之力钥匙时偶然发现、一直没声张），换苏晓带他逃。苏晓遂踩塌祀堂竖井、带他跳入皇宫地下的 **[比鲁马克炼金试验场](../世界/任务世界/斩赤红之瞳.md)**。由于他身负**皇室血统、天生免疫场内的炼金毒素**，又凭「奇怪的直觉」辨得安全方向，成了苏晓在试验场里的**向导**（也几次被苏晓 / 布布特尼威胁着老实带路、识读皇室文字石碑）。苏晓向他挑明三点：他当初不是为救苏晓、而是为自保；不带他走他会被大臣艾斯德斯处死；如今他是俘虏、指路才放他走。

## 黑化 · 化为牛头领主之核心（四卷66–68章）
逃亡途中，他在紧急通道前临阵退缩、被苏晓强按手入金属门激活了通道；随后**没跟上、被诞生之圣所放出的 [牛头领主](../世界/生物万族/index.md) 吞融**（腰以下融入其额头/大脑、成为驱使它的「核心」）。承受剧痛与大臣背叛之恨，他**彻底黑化、产生灭世倾向**——下令牛头领主杀大臣、杀白夜、**毁灭整个帝国**（牛头领主遂一小时夷平帝都、捏死大臣）。最终苏晓按下早埋在他身上的微型炸弹、**炸毁牛头领主的「核心」（即他·1/5）**，他这单纯到可悲、又被仇恨吞噬的昏君就此落幕。
