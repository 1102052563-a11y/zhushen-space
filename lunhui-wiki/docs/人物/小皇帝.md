# 小皇帝

**小皇帝** 是 [斩·赤红之瞳](../世界/任务世界/斩赤红之瞳.md) 世界帝国的名义皇帝，实为 [大臣](大臣.md) 操控的傀儡（四卷22–23章）。

## 简介
十三、四岁、身材瘦弱、营养不良的少年，一身紫衣不伦不类、略显娘炮，毫无威严（说话带童音如小学生朗读）。幼年丧亲（双亲实为大臣所毒害），自小被大臣洗脑、言听计从；无心治国、被大臣诱导沉迷女色（纵欲过度致瘦弱·苏晓推测）。

**关键价值**：他能启动最强帝具「**至高之力**」——该帝具**须皇室血统才能启动**，否则只是废铁。大臣正因此留他做「钥匙」、不敢杀。**本世界主线任务即夺取「帝具·至高之力」的启动钥匙**（现持有者即小皇帝），故他是众多契约者的争夺焦点。苏晓初见即考虑擒他，因不确定钥匙是否在其身上而暂缓（四卷22章）。

## 关系
- [大臣](大臣.md)——毒害其双亲、洗脑操控他的真正掌权者。
- [艾斯德斯](艾斯德斯.md)——帝国王牌（对小皇帝半蹲行礼、并不真敬）。
- [苏晓](苏晓.md)（白夜）——觊觎其「至高之力钥匙」的潜入者。
