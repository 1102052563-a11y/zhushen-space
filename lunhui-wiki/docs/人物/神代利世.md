# 神代利世

**神代利世** 是 [东京喰种](../世界/任务世界/东京喰种.md) 世界的 [喰种](../世界/生物万族/index.md)，原著中与「命运之子」金木研关系密切的关键人物（二卷1章）。

## 简介
紫发美女，身材窈窕却有惊人怪力；食量极大、人称「**大喰**」。被契约者评为 **S 级到 SS 级**喰种（二卷1–2章）。后腰脊柱末端可生出深红色**尾赫**（四根，如长尾）（二卷2章）。

## 经历（截至二卷3章）
开局在某大楼吃人，与奉命来抓她的 [壁虎](壁虎.md) 对峙；保护膜消失后被 [苏晓](苏晓.md) 一脚踢飞、撞地翻滚，被评「出乎意料的弱」，欲逃却被苏晓锁定（二卷1–3章）。
