# 神代利世

**神代利世** 是 [东京喰种](../世界/任务世界/东京喰种.md) 世界的 [喰种](../世界/生物万族/index.md)，原著中与「命运之子」金木研关系密切的关键人物（二卷1章）。

## 简介
紫发美女，身材窈窕却有惊人怪力；食量极大、人称「**大喰**」。被评为 **S 级到 SS 级**喰种；后腰可生四根深红**尾赫**（二卷1–2章）。

## 经历（截至二卷6章）
开局在某大楼吃人；被 [苏晓](苏晓.md) 一脚踢飞后试图以媚惑偷袭，反被苏晓一刀贯穿心脏重伤。乐园提示「杀重要剧情人物前需取出其赫包以延续剧情」，苏晓欲取其赫包未果——[青铜树](../势力/index.md)（高槻泉等）赶来抢人，利世趁乱破窗跳楼逃走，青铜树的野吕追去（二卷4–6章）。

## 新任剧情主角（二卷64章）
金木研被苏晓斩杀后，命运之力重新分配——神代利世成为本世界**新任剧情主角之一**（与雾岛董香各占 1/2）（见 [机制 · 命运之力](../总览/轮回乐园机制.md)）。

## 关系
- [苏晓](苏晓.md)（白夜）——被其踢飞、刺穿心脏重伤，趁乱跳楼逃脱。
- [壁虎](壁虎.md)——奉命前来追捕利世的喰种。
- [高槻泉（枭）](高槻泉.md) 与青铜树——率众前来抢夺利世（野吕追击她）。
