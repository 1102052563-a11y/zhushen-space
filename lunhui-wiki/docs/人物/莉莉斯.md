# 莉莉斯·艾亚（恶魔族战士少女 · 苏晓的黑渊盟友）

[黑渊浅层 · 赔罪者之殇](../世界/任务世界/黑渊浅层-赔罪者之殇.md) 中由大恶魔 [沃波尔](黑渊配角.md) 派来增援 [苏晓](苏晓.md) 的**恶魔族战斗系少女**——19 岁、皮肤微黑、山羊犄角、青绿色竖瞳、羊蹄、凹凸有致透着野性美，是魅魔 [莉莉姆](莉莉姆.md) 的妹妹（纯血恶魔·血脉珍贵·黑泽 / 阿萨卡都不敢动她）。性格人狠话不多、易怒、还是个吃货（一来就猛炫黑龙肉）；战斗智商高、却涉世未深（被苏晓埋沙坑教训、被 [莎](莎.md) 各种「女流氓」调侃）。

## 能力（二十三卷7章 · 使徒之眼测）

`恶魔之力 8980 ｜ 力 103（真实·破极值）/ 敏 99 / 体 99 / 智 50 / 魅 72`——力量已突破 99 壁障。技巧性略逊苏晓、是恶魔学「学渣」但近战极勇猛：

- **近战武器大师 Lv.50**：长刀 / 战斧 / 战镰 / 重剑四器精通（武器攻击 +195%·战镰附流血 / 战斧附焰破甲 / 重剑附眩晕骨折）。
- **恶魔之焰（主动 Lv.40）**：操控恶魔之焰·苦练使火焰伤害 + 灼烧时长 + 附着性大增（决战 [圣扎迦](圣扎迦.md) 时贴身导入「内燃」削其 24% 生命）。
- **战兽军团（终极 Lv.5）**：召 20~30 头恶魔战兽冲杀（学渣·耗光全部恶魔之力才放得出）。
- **体质锤炼 / 寒冬血脉类被动**：防御 +46、移速 +27、异常抗性 +35、各类伤害 −10%、生命 +3000。

## 黑渊并肩（二十三卷6–33章）

她奉命「跟着苏晓直到他安全离开黑渊」+ 历练。一来就被癞蛤蟆抽飞送到苏晓处（污点·绝不许提）；与苏晓、莎组队杀入始祖地下城决战 [圣扎迦](圣扎迦.md)（被血之口咬伤、被圣扎迦掐喉·苏晓一刀穿三人救场）；单挑把羽族公主 [羽鸦](黑渊配角.md)「只打脸」按地暴锤。事成苏晓塞给她一块黑枫树皮（够十份恶魔之力中枢·有她姐一份）。她对犬科好感高、却被 [布布汪](布布汪.md) 三观尽毁。

## 关系
- **[莉莉姆](莉莉姆.md)**：姐姐（魅魔·派她来 + 映空镜叮嘱苏晓「别抛弃她」·口头要把她「附赠」给苏晓）。
- **[苏晓](苏晓.md)（白夜）**：黑渊盟友 / 师从对象（一同决战圣扎迦·得其黑枫树皮）。
- **[沃波尔](黑渊配角.md)**：派她来历练的大恶魔长辈。
- **[尤利塞斯](黑渊配角.md)**：恶魔统帅·她口中的「师傅」（为她抢灵魂果实）。
