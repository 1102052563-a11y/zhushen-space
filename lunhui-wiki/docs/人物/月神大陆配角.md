# 月神大陆配角（菲洛米娜 · 老圣王 · 月神女 · 鹰侯 · 灵魂语者）

[月神大陆·黎明帝国](../世界/任务世界/月神大陆-黎明帝国.md) 中的帝国权斗群像与超凡者。

## 菲洛米娜（小副官 · 蒸汽炸弹专家 · 苏晓的副官）

第九部队的二号人物、苏晓的旧**副官**（编号 T17.2229）——十八九岁、茶色短发、一脸没睡醒、精致容貌却言行神经质的爆破天才（曾间接 / 直接致死 15680 人·叛国 + 多重谋杀罪·与苏晓一同关在地下监狱）。最大特点是**护食**（吃饭时谁靠近就发威胁低哼·布布汪差点被她咬一口），认人靠**闻气味**（嗅出苏晓真身后被一脚踢飞还乐）。行事风格极简：接令、执行、杀敌、撤离。**白磷蒸汽炸弹「火汽蹄」**用得出神入化（鹰侯晚宴仅 4 颗炸弹屠尽满堂宾客），随身藏几十颗违禁炸弹。对苏晓忠诚（入狱坚称罪名是自己所为没甩锅）。

## 老圣王（黎明帝国·官员最高统治者 · 超级老阴哔）

黎明帝国的**老圣王**——灰白长发、面容枯槁、盖毯子像普通老头，眸子却精光内蕴、一收和蔼就不怒自威。相当于国王（但受月神女制衡）。和蔼是「吃人不吐骨头的和蔼」、被苏晓评为「老狮子 / 哈罗德梯队的超级老阴哔」。爱养花（古董花盆被苏晓捏裂心痛到手抖）。停战后决意铲除所有对手（含月神女），力保苏晓出狱、把「乌鸦眼」控制权交他、与月神女**以不谋而合的默契联手除鹰侯**——谁的人先杀鹰侯，谁在战后权力分割中占主导。给苏晓 [魂源（高纯度）](../力量体系/装备道具.md) 作酬。名言「勿要寻找，勿要倾听，勿要亵渎神灵」。

## 月神女（黎明帝国·信仰最高代表 · 稀有类老阴哔）

黎明帝国的**月神女**——月之神在帝国的代表、信仰方面最高代表（地位仅次老圣王、能批量造月使徒故权势不低）。约三十岁、黑色长发、丰满白皙、「越看越顺眼」的独特气质，**说翻脸就翻脸、温和外表下狠辣至极**。出场让苏晓「扶我一下、腿跪麻了」、是苏晓见过最奇特的老阴哔类型。设局想拉拢 / 除掉苏晓（拉拢不成就杀），却被苏晓用机械蜜蜂窃听 + 反套路化解、还误以为内鬼是部下索尼亚（实为被苏晓利用）；事后认栽「这一局输了」、羡慕老圣王捡到苏晓这样的人。因「勿亵渎神明」是三巨头中唯一不渴求意志符石者；月神修道院长期庇护灵魂语者。

## 鹰侯（黎明帝国·军方代表 · 殁）

黎明帝国第三巨头**鹰侯·埃尔维斯（阿特利）**——掌帝国 80% 军权、中气十足、有五位妻子、滔天权势与花不完的资产。常年战乱让他地位水涨船高，停战后却面临「兵权交老圣王还是月神女」的死局。**绑架灵魂语者 = 拎起导火索坐上炸药桶**（还嫌死得不够快）。在为小女儿办的「最后晚宴」上被苏晓**刃道刀·时一刀钉死在墙上、斩首**（苏晓得世界之源 9.7%·隐藏任务鹰之陨完成）——临死只有不甘。是「饿则啄肉、饱则飞走」的鹰、第九部队则是开始「吃人肉」的魔鹰。

## 灵魂语者·尼迪亚（每代仅一位 · 神棍）

被鹰侯绑到首都的**灵魂语者**——灰白蓬发、衣衫褴褛、永远低头呓语（似多种声音混合·皮肤直接触碰她会让人产生「开颅 / 全身血管如花盛开」的恐怖幻觉）。每代仅一位、能「**倾听**」找到老圣王 / 鹰侯渴求的「某地钥匙·意志符石」、受月神修道院庇护故三巨头不敢杀（杀她是大丑闻）。表面慈蔼引导者「尼迪亚大人」，**实为吃人肉的恶徒**（小副官认出她——几年前在列歐西港机密任务里目睹她用大锅煮一村平民充食·脸 / 手背有菱形刺青）。被苏晓推理逼出线索、死猪不怕开水烫、有恃无恐（谎称体内有慢性剧毒拒绝逃跑·实为笃定修道院会救）。把利用她正义感聚起的**灵魂守卫**成员当弃子（「民间超凡者散漫不可靠」）。

## 灵魂守卫：兰 / 红发少女（被利用的民间组织）

灵魂语者用正义感 / 归属感聚起的民间超凡者组织——专处理超自然异常事件、救过不少平民（剥皮人 / 倒垂鼠绑架 / 红靴子连环杀人案）、成员多年轻、口号「灵魂至高无上」（被洗脑）。

- **莱斯·兰（二号首领 · 被擒）**：约 20 岁、金发、气息如阳光、拳裹金光的近战超凡者；曾以偷窃为生（会撬锁）。率百余成员强攻第九部队驻扎地救尼迪亚，被苏晓 10 秒团灭、生擒（看穿尼迪亚在利用自己却仍尊重她「带我们救过很多平民」）。
- **红发少女（被擒）**：火红长发、皮甲、双刺剑、脸带小痘痘的灵魂守卫主力；速度快但对苏晓只算「快」、被一脚踹进墙；目睹尼迪亚把同伴当弃子时三观崩塌泪流。

## 其他超凡者

- **魂灵剑·曼（鹰侯护卫 · 殁）**：寸头、能凭魂能生成「剥离」能量黑剑（插地形成吸力点、拉扯敌人灵魂 / 意识·使用副作用是瞳孔缩小）。鹰堡退场战与苏晓交手，四剑围出口字阵想扯裂其意识，被放逐 + 晶体层 + 侧踢轰碎下半身斩首（苏晓得世界之源 1.2% + 史诗宝箱）。
- **亚伦·鲁（剥面人 · 在逃）**：地下监狱三犯人之一、能剥下他人面皮伪装成那个人（多次冒充黎明 / 西莫罗帝国官员·经济损失 26.8 亿金镑）。混乱中越狱、伪装成被擒的兰、骗灵魂语者临摹下她脸 / 手背的关键刺青后溜走。
- **索尼亚（帝国之花 · 月神女部下）**：酒红长发、交际花、舔狗资质满点；无意中把苏晓的机械蜜蜂带进祈祷室成了泄密源（自己浑然不知）、险被月神女清理（因「跟了七年」捡回一命）。
- **富兰克（服装店主 · 发际线）**：地下监狱明面掩护点的店主、为头发发际线和睡眠惆怅、被苏晓 + 小副官两个「瘟神」吓到求生欲爆棚。
- **幽灵（乌鸦眼大头目）**：第九部队情报部门「乌鸦眼」的头目、如幽灵般融入黑暗·一个半小时就查出灵魂语者下落（打脸月神女「乌鸦眼永远找不到」）。
- **违规者 9013 号（在逃）**：从一阶活到五阶、全凭一个「怂」字的违规者——把资源全堆在逃跑上，速度快到苏晓全息地图都追不上（红点瞬间拉成直线消失）。郊外感叹「这次中头奖、那气势至少是老猎人」、果断开溜。
