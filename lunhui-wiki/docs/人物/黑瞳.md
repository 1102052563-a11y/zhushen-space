# 黑瞳

**黑瞳** 是 [斩·赤红之瞳](../世界/任务世界/斩赤红之瞳.md) 世界**帝国狩人部队**的帝具使，[赤瞳](赤瞳.md) 的**亲妹妹**（四卷35章）。

## 简介
黑发美少女、一身黑色学生装；最大特点是**贪吃且护食**（常提着点心袋·谁动她爱吃的就用水灵灵的眼睛盯着）。与姐姐赤瞳长相相似，命运却大不相同——她所在的部队对她做了**药物改造**：除非被斩首或搅碎心脏，否则受多重伤都能继续战斗，但身体机能因此出现一定崩溃、需**一直嗑药维持**。

- **帝具「死者行军·八房」**（刀形）：可**记录所斩杀的 8 个生物**（人或动物皆可），战斗时召唤出来——被召唤者有**不死性**（破坏头部仍能战斗、被八房碰到才化为普通尸体）；弊端是召唤后会临时减弱使用者能力、快速耗体力。八房全开时正面单挑夜袭全员都没问题，但持久力差。

苏晓将她列为本世界**最想交手的三人之一**（与艾斯德斯、赤瞳并列·都是刀剑高手·切磋利于刀术成长）。她对苏晓很感兴趣（用饼干换他讲与姐姐交手的经过）（四卷34–36章）。

## 死亡 · 姐妹对决（四卷47–48章）
狩人部队剿杀夜袭总部时，她与姐姐 [赤瞳](赤瞳.md) 正面对上。她八房全开召唤「死者」围攻（村雨咒毒对已死的「死者」无效），看似占优，实则**狂嗑十几倍剂量的药**续命、药效一过战力暴跌、神经系统崩溃。她其实**没有杀姐姐的觉悟**（演技差、只想「有同伴在身边就好」）；最终下半身失去知觉、被村雨贯胸——死在姐姐怀里。临终她见伪装同僚的苏晓现身，惊觉不对、叮嘱「姐姐逃」。苏晓事后把姐妹二人合葬。
