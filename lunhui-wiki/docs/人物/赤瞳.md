# 赤瞳

**赤瞳** 是 [斩·赤红之瞳](../世界/任务世界/斩赤红之瞳.md) 世界 [夜袭部队](../势力/index.md) 的成员、帝具使，是国家机器培养出的红瞳杀手（四卷16、24章）。

## 简介
持帝都赫赫有名的妖刀帝具「**一斩必杀·村雨**」——只要被村雨斩伤即死；但这「一斩必杀」**并非规则之力**，而是刀上附着一种**咒毒**，被斩伤后咒毒侵体致死，原理与苏晓的 [青钢影](../力量体系/技能/index.md) 类似（四卷24章）。她与同伴各有信念、皆国家机器所育——苏晓认定一旦自己加入帝国阵营，与赤瞳等夜袭女性便是你死我活（四卷16章）。

她是位黑长直少女、赤红双眼（名字由来）、平时有些呆萌；战斗经验极丰富，常以**被动防御 + 抓敌破绽补刀**作战（村雨只需一击即可一斩必杀，故此打法极「赖皮」）。

## 与苏晓死战 · 心脏右位逃生（四卷28–32章）
豪华客船一役她正面对苏晓——苏晓判其**身体素质与他相近、战斗经验丰富、但刀术更弱**。她持村雨令苏晓不敢被斩伤（怕咒毒），又有玛茵远程、布兰德拼死支援，仍被苏晓**三连击**（顺势卸力致其失衡 + 扫腿 + 膝撞 + 肘锤断肋）放倒、一刀**贯穿左胸刺心**。岂料赤瞳是几百万分之一的**心脏长在右侧**者、未被杀死，十秒后竟站起——苏晓由此判定**赤瞳才可能是本世界真正的「小气运之子」主角**（原著夜袭多半死绝、独她存活）。她燃烧体内能量恢复、喊暗号让玛茵掩护撤退，凭对帝都地形的熟悉、靠青钢影能量消散前的窗口逃入平民窟花街、再钻地道脱身（苏晓追丢、却另得帝具改造蓝本）。

## 终局 · 抛弃人身的死战（四卷48–49章）
妹妹 [黑瞳](黑瞳.md) 死后，赤瞳为布兰德报仇、又看穿苏晓「不忠帝国、另有目的」，遂**抛弃人身、开村雨大招**——划破手心引咒毒入体、承载村雨所记录的亡魂之愤怒仇恨化成力量（燃尽己身），速度力量暴涨，并以「**杀气虚影**」（留杀气幻影、本体奇袭）这一终极能力斩中苏晓左手背、注入咒毒（「葬送」）。

但苏晓以**防御 + 高智力心眼**化解其暴涨攻势（赤瞳力速涨、刀术却没涨），又用「假震荡弹」（奥利奥 / 洗发水 / 士力架反复戏耍后再来真的）破其节奏；至于注入的村雨咒毒——被苏晓体内的**青钢影能量（最擅泯灭其它能量）硬生生噬灭**（耗法力 276 点·苏晓不切手臂、硬抗这「即死毒」反练出抗性）。最终苏晓一刀**贯穿赤瞳右胸心脏**将其杀死（赤瞳用了大招、即便不死也只剩几年寿命）。

她是**本世界的剧情主角之一（2/3·成长度 100%）**——杀她给苏晓 **15% 世界之源**（→27.2%）+ 紫色宝箱，并推进成就任务「[掠天惊澜](../力量体系/成就.md)」。临终她道「总有人代表民众意志、心中正义打败你」，苏晓答「强者即正义」。苏晓收走村雨 + 八房、合葬赤瞳黑瞳姐妹。
