# 白牛

> ⚠ **虚空地下世界的皇帝，苏晓少数几位真正的至交之一**;⚠ **⚠ 形态**:⚠ **身披纯黑色兽皮·头戴带有两只牛角的野兽头骨·魁梧到需要加大号沙发才坐得下**;⚠ **⚠ 他与几名心腹共有一个宗旨**:⚠ **可以死，但一定要辉煌过——哪怕只是短暂封临巅峰**（⚠ 「在白牛踏入黑暗世界的那一刻、在他还是那个街头寂寂无名的强壮小头目时，他就注定有今天没明天」）。

## 🐂 他的兴衰

⚠ **⚠ 他曾真的爬上过至强巅峰**——⚠ **靠的是那极端又猖狂的封临方式；正常情况下，他不因体内「万兽力量」崩溃而死于巅峰之境都不合理**;⚠ **⚠ 有趣的是，救他一命的正是重创他的人**:⚠ **渊之龙造成的「伤损细胞」把他打到跌落至强前中期，后续侵蚀下直接掉到绝强极限——**而这不是实力滑落，是永远失去****;⚠ **⚠ 可「伤损细胞」与「万兽力量」持续彼此侵蚀，反而让他虽持续掉血却活得更长**——⚠ **这也是他寻访星界各大名医却治不好的原因:伤势结构太复杂**（⚠ 最终是苏晓的超级大修加各类顶尖药剂，让他奇迹般痊愈）;⚠ **⚠ 他的死期为何越来越近**:⚠ **「能来给我卖命的都不是好东西——不惹事又有实力的强者，正经大势力抢着招募，怎么可能加入到我麾下」**——⚠ **这些年他「亲爱」的手下们帮他惹上的仇家，让他只能去追求【元素魔核】**。

- ⚠ **⚠ 关于他的战力争议**:⚠ **满血无伤的白牛，本应属于「冥神、不死老人、渊之龙」那一梯队的至强巅峰**（⚠ 这是作者在第4070章后附的「战力设定说明」中亲自澄清并承认的:白牛这个人物因跨越篇幅过长而写出了偏差，当初设定他被渊之龙重创正是为了拉高他）。

## 🌌 星空座时期 · 万兽王（三十六～四十六卷）

披兽皮、戴双牛角黑色兽头骨、超 5 米高、万兽随行的强壮男人——**万兽王**、[星空座](../世界/势力/星空座.md) 成员（**星空之环 4·戴左尾指**）、体内有「兽王魂」。**虚空黑暗势力的统治者**（虚空 65% 以上见不得光的产业都与他有关·得罪过羽族 / 恶魔族 / 星族至今仍活着的亡命徒）。

- **震慑奥术永恒星**：[瑟菲莉娅](瑟菲莉娅.md) 率上千八阶施法者追杀苏晓到空座宴场地时，白牛一句「5 秒之内滚出我的视线」就把她逼退（星空座场地内无人能杀成员·除非星空座全死绝）。
- **空座宴交易**：需求「**命源**」（吃了能压制被**渊之龙**所伤的旧伤几年·「早晚宰了那条渊之龙」）——用团长给的命源满足。他对黑枫树产出的需求仅次于恶魔族与奥术永恒星（有兽王魂·直接吃黑枫叶 / 枝干即可），用**初代灭法的半根指骨**（能真正唤醒灭法者体内的「影」）换走了 [苏晓](苏晓.md) 几乎全部黑枫树产出。

### 圣格亚偶遇 · 争果实败给团长（三十九卷97章）

苏晓随 [里德](陵墓守卫者·里德.md) 到虚空偏远之地 [圣格亚](../世界/地点/index.md) 指点朋友女儿刀术时，白牛循**星空环**（星空座成员在一定范围内可互相感知·摘下即隐匿）找上门——全身缠满绷带、一进屋就把 [沃迪加](虚空/珀黎·沃迪加.md) 的豪宅震得几近崩碎（他 5 米多高、进门要扯开门框上方墙壁）。

- **来收拾羽族**：他与 [羽族](../势力/index.md) 有仇、双方常「互相问候」；本批圣格亚正因羽族与恶魔族开战（争归属权），白牛便亲自来「收拾羽族」。
- **争虚空之树果实败给团长**：白牛坦言自己**争夺 [虚空之树果实](../力量体系/装备道具/index.md) 只差一点就得手**，却因 [不死老人](星空座/不死老人.md)「关键时刻靠不住、背地里收了团长好处」而落败，郁闷地吐槽「别看团长平时和善，他这人坏得很」，末了自认「败了就败了吧，技不如人」。此时果实已在苏晓这（团长得手后交苏晓这位「圣焰药师」调配成 [树之恩赐](../力量体系/装备道具/index.md)）。
- **委托调配 + 敲打沃迪加**：白牛本想借星空环之便当面委托苏晓帮他调配药剂（需自备配方 / 材料），但察觉苏晓不愿在虚空公开炼金师身份，便顺水推舟改口「配方弄不到」，约定下次**空座宴**再谈。临走还让手下都盯着记 [沃迪加](虚空/珀黎·沃迪加.md) 的长相以作警告——防他是奥术永恒星眼线、危及星空座好不容易来的炼金师。

### 空座宴 · 命源换承诺 · 圣焰药师合作（42卷86–88章）

暗星返园后的空座宴上，苏晓给了白牛一颗**命源**（他上次也从团长处换过一颗、能压制渊之龙旧伤几年）——白牛「最近交了好运」，遂许下重诺:**下次空座宴带来「初代灭法遗骨」**（他知道某势力有、找不到就去明抢·星空座是苏晓获取初代遗骨的唯一渠道）。这次他先付了 3 颗灵魂晶核 + 一把 **[世界钥匙](../力量体系/装备道具/index.md)**（偶然得的乐园之物·进入世界前用、可进资源丰厚的任务世界）换走苏晓 2000 克黑枫树；还委托苏晓调配恢复永久损伤的药剂（当年与渊之龙厮杀重伤、妹妹赶来增援也被伤·药剂用于救妹妹·他自己的旧伤积压太久几乎无解）。他试吃了刀魔的黑枫树枝干、越嚼脸色越怪（比苏晓那份质量差、刮嗓子）。

- **圣焰药师的虚空渠道**：贝妮当场与白牛谈妥——苏晓以「**圣焰药师**」身份、借白牛（虚空地下黑皇帝）的渠道在虚空售卖药剂，不出半个月「圣焰药师」之名将传遍大半个虚空。届时奥术永恒星势必坐不住、邀这位「远超树贤者的炼金大师」去作客——一名灭法者被众施法者在奥术永恒星迎接，将是史无前例的奇景（苏晓甚至能点名让法师贤者瑟菲莉娅带他观光）。

### 空座宴 · 引荐「圣焰」赴奥法庆典的高明使坏（46卷50–51章）

白牛与苏晓合作卖施法者药剂，把「**圣焰药师**」的名号打响——奥术永恒星的 [瑟菲莉娅](瑟菲莉娅.md) 便来拉拢圣焰、想请其赴「奥法庆典」。白牛心知圣焰就是灭法者苏晓（施法者拉拢灭法者「这世道太疯狂」），表面「言辞拒绝、被那女人逼得没法」，实则把瑟菲莉娅的原话原封不动转达给苏晓、等于把苏晓引荐进奥术永恒星——**既吃透药剂生意的好处、又替苏晓送上门去、还能不与奥术永恒星死磕**（事后查清是瑟菲莉娅强逼引见，奥术永恒星只能捏鼻子认）。本次空座宴他给苏晓「初代遗骨」（卖命源的报酬），另求苏晓帮他那位白发妹妹治伤——苏晓当场给了压制伤势之痛的药剂、并允诺日后以圣焰身份来虚空顺路医治（「治出心理问题概不负责」）。

## 🧊 交付「顶峰之路」（54卷183章·庇护城）

⚠ **他找上苏晓时，开口的第一句是「因果层面上，这里安全吗」**——⚠ **苏晓取出【起源石·世界】，他脸上的微醺笑意才完全消失**;⚠ **⚠ 「白夜，我能相信你吗。」「不能。」**——⚠ **他被噎得差点没接住，随即笑着摇头:「是啊，能靠得住的人，只有自己。对了，尼琳，死了。」**（⚠ 他以稀松平常的语气说出此言，仿佛死去的不是他最珍视的至亲之一·⚠ 苏晓的回答也很平静:「哦。」——**他见过太多生死**）;⚠ **⚠ 他割开自己的左臂、深可及骨，从血肉已攀附生长其上的位置活生生扯出一卷皮质卷轴**——⚠ **【顶峰之路】:一份记载「同时以『深渊之心』与『元素之眼』晋升无上之境」的秘法，且经过轮回乐园公证确实有效**;⚠ **⚠ 「它属于你了。」**——⚠ **递出后他起身就走，没半分犹豫；一切决定都在来之前做好了**（⚠ 苏晓没有给报酬——**因为这是白牛的选择，而非交易**）。

- ⚠ **⚠ 他被算计了**:⚠ **是 [元素学者·赫洛斯](元素学家·赫洛斯.md) 让他看到了「元素魔核」的力量，此后他就无法拒绝**——⚠ **他赌上一切深入险地去活化那颗魔核，结果妹妹死了，最终只得到这张卷轴**（⚠ 团长当年在空座宴上听到他要【元素魔核】时，说过一句「白牛，冷静，你不应该渴望它」·⚠ 而元素学者对团长的解释是:**「他的命运要耗尽了，你应该早就知道，没人能改变其他人注定的命运。」**）;⚠ **⚠ 交出卷轴后，元素学者在庇护城南街的小巷口拦下他，把已活化的【元素魔核】放在了他掌心**——⚠ **他一度犹豫是否回去问问白夜此事何解**——⚠ **最终他把魔核抛入口中吞下，熊熊的野心之火在心中升腾**。

## 🔪 「记得吗，我曾经说过。」（54卷200章）

⚠ **奥术永恒星〔怒海·赫尔斯带队〕对他发起围杀**——⚠ **多名至强施法者围攻，换作其他人一对一都必败；他靠着「地下皇帝不擅逃命早就死在仇家手上」的本事成功脱身，但重创在所难免**;⚠ **⚠ 他退到风海大陆的深渊侵蚀区，八名心腹寻来**——⚠ **然后是三当家·泰斯从背后捅穿他的后心、二当家·卢修斯一匕首开喉、其余心腹一拥而上**;⚠ **⚠ 两把武器的来历极为致命**:⚠ **泰斯手中的是背刺武器中**整个星界排名第三、名为「背叛」**的短刀；卢修斯手中的是**「开喉者」——仅弱于「唾弃」，甚至强过当前未完成苏醒的唾弃****;⚠ **⚠ 他对泰斯说的最后两个字不是辱骂，而是**「快、逃。」****——⚠ **因为「背叛」褪去伪装的瞬间，他就认出了这把武器的因果特性:一切早在泰斯握上它时就已注定**（⚠ 泰斯的家人早在被威胁之初就被当着他的面一个个杀死；他懵在原地，随即被卢修斯一刀开喉）。

⚠ **⚠ 卢修斯的那句「我曾经说过」，回到很多年前的虚空·玛门莫德要塞后城区**:⚠ **还是小头目的白牛孤身杀进敌对帮派地盘，打断了卢修斯原本老大的脖颈，然后坐在街边台阶上对鼻青脸肿的卢修斯说「以后认我当老大？反正那头肥猪也断气了」**——⚠ **卢修斯没说话，白牛便说「那就当你答应」**;⚠ **⚠ 卢修斯当时的回应是**:⚠ **「如果你有一天，带我走向死亡，我杀了你，取代你。」**——⚠ **白牛的回答是「哦？哈哈哈，好啊，看你有没有这能耐了」，然后大手按上他的肩膀**（⚠ 两人就这样一路向前，身边的手下越来越多、白牛的身高逐渐提高、脸上的豪迈向沉稳暴恶转变，卢修斯则越发老成阴鸷·⚠ 让两人一跃成为虚空三流势力的，正是他们策反旧老大的心腹、把旧老大围捅死在其家中餐厅的主位上，之后其家人一个活口不留——**因果轮回**）。

- ⚠ **⚠ 死后暴起**:⚠ **倒在泥水中已身死的白牛，右手手指勾动了一下**——⚠ **一只大手掐上卢修斯的喉咙，把他举离地面三米以上；其余几人当场身体破碎四散**;⚠ **⚠ 这不是他恢复了实力，是「巅峰万兽力量」彻底失控**——⚠ **这种连掌握者都反噬的力量，在他还活着时不至于如此爆发；可他一死，这曾被滋养至巅峰之境的力量便当即爆发出来**（⚠ 这也正是这么多年各大势力不敢对他动手的原因:**他们在担心他能否以性命为代价短暂重回巅峰**——答案是不能；**可如果白牛身死，失控的巅峰万兽力量却一定会爆发出来**）;⚠ **⚠ 他捏爆卢修斯、把飞起的【元素魔核】一口咬碎吞掉**——⚠ **让即将衰退的万兽力量暂时有了源泉**;⚠ **⚠ 他还有三天**:⚠ **因果已破碎，三天后必死，无人可救**——⚠ **站在倾盆大雨中的他反而释然:老友的女儿已托人送到万族界〔那是白夜的地盘，施法者不敢踏足〕，至亲的妹妹死了，最信任、视作家人的手下也死了**;⚠ **⚠ 「现在，它什么都失去，无牵无挂——那么应该担心的，就是让它失去这些之人。」**（⚠ 深渊浩劫将至已让虚空各大势力绷紧神经，属于点火就着；**而现在的白牛简直是个炸药桶——虚空的乱战即将开始**）。

## ☠ 「我，已经死了」——最后的隆重退场（54卷205章）

三天寿命的最后，他拨通了苏晓的通讯器。声音低沉，甚至带着几分死气:

> **白牛**:「我要去趟奥术永恒星，有办法吗。」<br>**苏晓**:「就算至高之人不会出手，这也是自寻死路。」<br>**白牛**:「**我，已经死了，呵呵呵。**」<br>（苏晓沉默了几秒）「有办法。」<br>**白牛**:「我就知道你有办法。看来，真像团长说的那样，我的命运耗尽了——**可就算退场，也要隆重**。」

⚠ 他的语气中没有对死亡的恐惧。或许他早就想到自己的下场:成为地下世界的皇帝，仇家实在太多，加之手下作恶多端必会引来更多仇恨；**如此数量的仇恨，有好下场反而是种怪事**。

苏晓让贝妮把一枚**傲歌晶体**送到大聚地交给白牛委托之人——那是超级增强版的灭法传送阵。

### 三小时后:奥术永恒星·星辰广场

咚！一声传送震响传开。几公里外的**空间干扰塔**明暗不定，最终熄灭却并未破碎——⚠ **这是首个抗住灭法传送阵的空间干扰塔**。

白牛在周边施法者们愕然的目光中站上星辰广场。他一手爪掠过，身旁几名施法者年轻一代成员**当即化为碎片**；随后看向上方的**永恒魔能塔**——那代表奥术永恒星脸面之物、每次「奥法庆典」报纸头版封面的建筑——悍然跃起冲破层层结界，一拳轰上，大量裂痕浮现。

破风声袭来，赫然是 [凛风王](凛风王.md)。白牛的目光迎上，**咧嘴笑了**。

### 他真正的目的

⚠ 白牛从没想过以「巅峰万兽力量暴走」的状态能极限一换一奥术永恒星的巅峰强者——那太看不起施法者体系，也太便宜奥术永恒星。

**他要做的，是让星界所有人都知道:神其实并没流血。**

- 「星辰广场」就在【元素超导塔】下方，而**至高之人就在【元素超导塔】内**——⚠ 白牛可不是随便找地方传送过来的，他明确提及:**距离【元素超导塔】越近越好**。
- 正常情况下，他这一系列行为必然遭到至高之人的瞬间碾杀。在即将被引爆的虚空，**唯有如此威慑，才能让后续的所有事都不发生**。所有人都知道这点，包括奥术永恒星的施法者们——**他们没有立即一拥而上围杀白牛，其实就是在等待**。
- **然而，至高之人并未出手，甚至懒得看白牛和一众施法者一眼。**

> 「啊哈哈哈！」<br>满天远程法术、被压制围攻中浴血奋战的白牛，发出**豪情狂鬼般的狂笑声**——这让一众奥术永恒星高层面色变得非常难看。

没多久，星界各大势力的领袖都通过各自的情报渠道知晓此事（这种事根本瞒不住，何况就发生在奥术永恒星经常对来客开放的星辰广场）。⚠ 几乎所有人都生出同一个想法:**立即联手，围杀奥术永恒星**——这关乎所有人的安危，如果杀光这些施法者，深渊末日浩劫是否能来得更晚些？面对这等生死存亡，星界的大佬们愿意赌一局。

而对于奥术永恒星那位一众星界大势力无法对抗的神，白牛的举动已经证明了一件事:

> **神，不在乎。**

⚠ **称霸本纪元的最强霸主势力，即将要面临星界各大势力的围攻。毋庸置疑的是，奥术永恒星这次要面对的最强之敌，名为——灭法之影·库库林·白夜。**

### 他留下的遗产

| 项 | 去向 |
| --- | --- |
| 【顶峰之路】秘法卷轴 | ⚠ **他为苏晓拼命换来的**（追求「元素魔核」并祭献掉第一颗所得），成为苏晓晋升无上之境的核心秘法之一 |
| 「元素魔核」（第二颗·已活化） | 被他自己一口咬碎吞掉；⚠ 奥术永恒星追杀他，目的就是要确定这颗魔核是「活化」还是「被祭献」 |
