# 蔷薇（契约者·黑瞳·镰刀吸血）

[苏晓](苏晓.md) 第十五卷再入火影时、同入的 157 名契约者之一——黑色露背皮衣、黑色短发、一双黑宝石般妖异眸子、**背纹一朵彼岸花**的女契约者（最初以代号「**黑瞳**」登场·与斩·赤红之瞳世界的 [黑瞳](黑瞳.md) 同名异人；真名 / 假名「**蔷薇**」）。她最强的不是战斗力、而是**随心切换气质**的本领——双手沾满鲜血、却能笑得像朵洁白蔷薇（在铁之国是清理异教徒的「杀戮天使」、转脸又是清纯活力少女）。**疑即苏晓的猎杀目标「TQ-9670 外来者」。**

## 登场 · 血·芳华（十五卷20、24章）

她在铁之国雪原**七分钟杀光一支 23 人的契约者冒险团**，逼唯一逃出的幸存者「学狗叫」、学得不像便挖其眼。随后她**伪装成「女职工者」、向苏晓装可怜诈救**（被铁之国武士追·想拉苏晓下水）——苏晓没理会（人不犯我我不犯人）；她跑远后自嘲「不擅长布局」，摘兜帽露出真容、亮出**金色品质短柄战镰「血·芳华」**（约 50cm·幽青寒光·握柄镶五颗金色宝石）：一招「血·芳华」便把六名追兵武士斩成碎肉，且**所有鲜血凭空漂起、被战镰吸收**（「贪吃的家伙、都快喂不起你」）——一件嗜血的吸血神兵。她随即盯上雪地里的几把武士刀、似在盘算新策略。

## 潜入晓 · 暗算苏晓（十五卷51–52章·真名蔷薇揭）

她借「**黑眸职工者妹子**」的身份在铁之国被武士追时遇见 [鬼鲛](干柿鬼鲛.md)、助他横跨铁之国追上八尾人柱力 [奇拉比](奇拉比.md)（与鬼鲛上次入火影时结识、开了好感度），遂被鬼鲛**引荐进晓组织**（伪称感知型职工者 + 一种特殊血继限界·来「抱大腿」）——真名自报「**蔷薇**」。但她进基地后撞见苏晓在晓内**地位极高**（带土 / 鬼鲛皆敬他三分），警觉「怎么有种主动送上门的感觉」。其内心独白暴露真意：她**布局能力是「小学生级」**（自嘲「牺牲色相蠢出天际、幸好没人听到」、佩服曾经擅阴谋的队友「小黑」），遂弃掉宏大计划、改回简单粗暴的本色——「**火影世界结束前，不是你死就是我亡**」。她瞄准的「你」正是苏晓；而苏晓也一眼觉她「不对」、只因她在晓内、随时有办法收拾（哪怕她与鬼鲛好感度高）。

## 真相 · 彼岸花前任团长来复仇（十五卷61–63章）

晓组织投票（带土新设的「半数过」制）否决她入伙、她以 KMTA 放射物项链（半公里灭活物）逼带土 / 佐助 / 迪达拉退场，与苏晓单独摊牌——她的真身是 **[彼岸花冒险团](../势力/index.md) 的前任团长**！苏晓在进击的巨人世界**斩尽了彼岸花**（[黑白](黑白.md) / [女王](女王.md) / [叶子](叶子.md) / [冷月](冷月.md) / [糖果](糖果.md) / 小黑〔[腹黑女](腹黑女.md)〕/ [血玫瑰](血玫瑰.md) 等全死在他刀下），唯独她因**早已退团**而幸免——她查了很久才确认是苏晓所为，遂一路追到火影世界、借鬼鲛潜入晓「来找你清算旧账」。

!!! quote "「团长，你退团吧」——彼岸花众人的成全（十五卷63章·回忆）"

    当年彼岸花众逼她退团：她**战斗天赋约 500~700、队友只有 5**（九人面对卡巴内险些团灭、唯她杀出），队友（[冷月](冷月.md) / 小黑等）不愿再「拖累」这天生的独行者——「独行奖励更高、你的天赋不该被冒险团束缚」。小黑临别还嘱她「**独行别轻信任何人、多属性发展〔力 / 敏 / 体为主、智 / 魅适当〕、遇冒险团拉拢别直接拒绝也别真加入**」。她（6 岁揍哭 13 岁亲哥的暴力软妹）含泪退团——这份「同生共死的情谊」正是她复仇的根源。她不擅布局（自嘲「小学生级」），故弃阴谋、改简单粗暴：「不是你死就是我亡」。

## 战斗 · 双魔镰 + 血舞者（十五卷62–65章）

普通状态下她已与苏晓势均（速度不输他·力量 83 略胜·心狠手辣）。武器是**两把短柄魔镰（可噬主的成长武器·镶五宝石）**：右镰增流血伤害、左镰附流血 + 吸血回血；宝石效果「**恶鬼獠牙**」（防 −10）/「**痛苦女王**」（四阶以下治疗 −30%）/「**鲜血礼赞**」（失血 +60% / 流血伤 +100%·可叠）——划一刀就叠满减治疗 / 增流血。变身「**血舞者·断罪**」后黑发变血色长发、脸生血纹、双镰变长喷血雾，可抛镰生「**血魔偶（血色虚影）**」（高等分身·受击不消失·近战不弱本人）、大招「红莲」（变身限定）。直感中她「似是两个人」、口呼「哈波迪雅」（疑双人格 / 噬主魔镰之灵·伏笔）。意志力极强（气息外放被动判定失败·否则压不住魔镰）。开打即用 [阿姆](阿姆.md) / [布布汪](布布汪.md) 缠她、苏晓自战两具血魔偶（青钢影 + 生灵切割能压制血魔偶）；阿姆被她过肩摔 + 「红莲」踩进地里、布布汪被她「小拳拳」揍肿嘴。

## 关系
- **[苏晓](苏晓.md)**：血海深仇——他斩尽了她的彼岸花冒险团（除她外），她追到火影来复仇（「不是你死就是我亡」）。
- **[彼岸花冒险团](../势力/index.md)**（[冷月](冷月.md) / 小黑 / [血玫瑰](血玫瑰.md) 等）：她创立并曾带领、后被众人成全退团的旧部（全死于苏晓·进击世界）。
- **[干柿鬼鲛](干柿鬼鲛.md)**：引荐她进晓的「老相识」（摊牌时清空了与她的好感度）。
