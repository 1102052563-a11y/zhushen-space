# 腹黑女（小黑）

[彼岸花冒险团](../势力/index.md)（全女性冒险团）的**智囊**·团长 [冷月](冷月.md) 的过命搭档（只忠于冷月）。**智商高达 156**——武力不算优秀，但**推断能力极强**，是彼岸花从中型团重整为精英小团的幕后谋划者（让全团卡在同一时间点参加晋升任务、深挖衍生世界）。彼岸花众人中，她**最了解 [苏晓](苏晓.md)**。

## 误触禁忌（八卷22章·重要）
吞噬岛上她复盘与「刀男」苏晓的恩怨、凭高智商推理出**苏晓每次进衍生世界都有「特殊任务」（猎杀特定契约者）、实力增长异常快、对战契约者的能力超群**——话还没说完，**轮回乐园直接屏蔽了她的声音、手背浮现要成形的蓝色印记（禁忌惩罚）**。她由此惊觉**苏晓的身份是极机密的「禁忌」**（即「猎杀者」·一旦曝光会被其他契约者围杀，但泄露者自身也要受乐园惩罚）——遂当机立断**改口「推断错误、放弃复仇」**（她原本坚持复仇，是因与苏晓已结死仇、单方面不追究也不行；但触了禁忌后明智收手）。此事点明：**「猎杀者」身份受乐园禁忌保护**，连推理出来都会遭惩罚。

**反成催命符（八卷26章）**：她虽噤声放弃复仇，**身份却已被她想透**——乐园遂判定苏晓身份暴露（即便与苏晓本人无关），在她身上打下**「追猎（蓝）」印记**、令苏晓 **12 小时内清理她**（否则剥离猎杀者身份）。此时她已与队友走散、独自一人（她是彼岸花的**结界师**·智商高但大局观一般），在稀疏森林里靠小型结界险险挡住绿毒蛇——苏晓正循追猎印记骑布布汪连夜赶来（约 30 公里外）。

**自残脱身（八卷27章）**：以她的智商，一眼认出手背蓝印的含义——遂**甩开队友、硬抗「全属性 −3」强制退团**（不连累冷月等），更**自断那只带追猎印记的手**（印记在四肢可强行剥离）、孤身**深入红土区中心**（中心猛兽盘踞、却能甩开苏晓·她甚至想躲进黑色森林）。苏晓赶到时**追猎印记只剩一条断手**——他只得付 5% 世界之源催生第二枚印记（赌它这次长在头 / 躯干、让她无法再剥离）。这份当机立断的狠辣，正是「智囊」的底色（追她的途中苏晓反而**撞上了来寻她的冷月等五人**）。
