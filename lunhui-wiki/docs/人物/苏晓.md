# 苏晓

**苏晓** 是《轮回乐园》主角，身份为 **猎杀者**——天赋特殊、更适合战斗的特殊契约者（见 [轮回乐园机制](../总览/轮回乐园机制.md)）。已完成首个任务世界 [海贼王](../世界/任务世界/海贼王.md)、获唯一职业 [灭法之影](../力量体系/技能.md)，现进入第二个任务世界 [东京喰种](../世界/任务世界/东京喰种.md)。

## 简介
入乐园前是现实世界的**复仇者**：三年弃学、专习杀人技巧（第1章）；刺杀仇人后被持枪保安「假死」、与轮回乐园签约（第1章）。信奉「**弱小即是原罪**」，视现实为牢笼、立志在乐园不断变强（第10、16章）。

## 性格
冷酷狠辣、为达目的可牺牲他人（第20章）；嗜战、享受生死一线（第13–15章）；越历磨难越强（强化拒用麻醉）（第26章）。

苏晓凭原著情报**回避「巨人之森」**（原著被擒之地），率众横穿**帕拉蒂岛**直奔艾伦故乡，踹开其家**地下室**——出售「进击巨人之骨髓 / 始祖巨人之骨髓」等只能换世界之源的实验物品，一举获 **14% 世界之源**（回归任务需 15%、即将达成），又在暗格寻得蓝色 [巨人精炼提取液](../力量体系/消耗品材料.md)（返乐园花 13000 币完善后可 +8 属性）。借父亲 [格里沙·耶格尔](../人物/格里沙·耶格尔.md) 的日记，艾伦的身世（一家出身马莱、二胎、同父异母兄 [吉克·耶格尔](../人物/吉克·耶格尔.md)、初代王「不战契约」）大白（三卷42–45章）。出墙时遭七八百骑兵的调查兵团追上，苏晓判断敌方连日奔波已成强弩之末，用破碎精灵「凝视射击」**一枪爆头狙杀团长艾尔文**——调查兵团群龙无首（利威尔无指挥才能、推韩吉佐耶代理团长），追击战就此展开（未完）。详见 [进击的巨人页](../世界/任务世界/进击的巨人.md)。
