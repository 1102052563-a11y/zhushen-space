# 苏晓

**苏晓** 是《轮回乐园》主角，身份为 **猎杀者**——天赋特殊、更适合战斗的特殊契约者（见 [轮回乐园机制](../总览/轮回乐园机制.md)）。已通关 [海贼王](../世界/任务世界/海贼王.md)（A-）、[东京喰种](../世界/任务世界/东京喰种.md)（S-）、[进击的巨人](../世界/任务世界/进击的巨人.md)（A+），获唯一职业 [灭法之影](../力量体系/技能.md)；现于乐园停留、即将进入第四个任务世界 **[斩·赤红之瞳](../世界/任务世界/index.md)**（四卷）。

## 简介
入乐园前是现实世界的**复仇者**：三年弃学、专习杀人技巧（第1章）；刺杀仇人后被持枪保安「假死」、与轮回乐园签约（第1章）。信奉「**弱小即是原罪**」，视现实为牢笼、立志在乐园不断变强（第10、16章）。

## 性格
冷酷狠辣、为达目的可牺牲他人（第20章）；嗜战、享受生死一线（第13–15章）；越历磨难越强（强化拒用麻醉）（第26章）。

随后帝都港口一役，苏晓单挑来收帝具的夜袭部队：以震荡弹开局、用**极限青钢影一刀斩杀王牌 [布兰德](../人物/布兰德.md)**（+3.6% 源·法力 517、收三件物品类帝具 + 蓝箱），又三连击放倒 [赤瞳](../人物/赤瞳.md)、贯心一刺却因她**心脏长在右侧**（几百万分之一）未能杀掉（苏晓由此判赤瞳才是本世界「小气运之子」主角）。玛茵、塔兹米跳河，赤瞳逃入平民窟——苏晓追击中**开发出「青钢影坐标感知」**（被其能量伤到的敌人短时难逃），追至花街地道、劈开锈死的帝国金属门、冒「卡尼亚特毒气」夺得**紫色「[帝具改造蓝本](../力量体系/装备道具.md)」**（合成最强帝具的图纸·**首见紫色品质 71–150**）。世界之源累至 8.3%。夜袭部队连夜召开「**讨伐苏晓作战会议**」（娜洁希坦称他是「第二个艾斯德斯」、令四人以下遇之即撤）。

此后苏晓加入艾斯德斯新组的暗部「**[狩人部队](../势力/index.md)**」（针对夜袭·同僚有 [黑瞳](../人物/黑瞳.md)〔赤瞳之妹〕、波鲁斯、威尔、兰、Dr.时尚），并接掌**第三暗部长官**；他还每日冥想**自悟「[心之冥想](../力量体系/技能.md)」**（清杀念、保冷静）。为「搅浑帝都的水」、逼帝国与革命军提前决战，他暂离皇宫——靠顶尖追踪（层层换接头人、耗两天）摸到**革命军总部**古堡，绑军官特利以**心理战**逼供（不折磨肉体、专摧意志），随后绑走能开臣具「莫森尼亚·永固之门」的臣具少女、**洗劫革命军帝具库**（得 5 件帝具含 1 件生物类·**至此凑齐「帝具改造蓝本」全部合成材料**），退出时偶遇并**一刀腰斩夜袭首领 [娜洁希坦](../人物/娜洁希坦.md)**（+3.9% 源·她刚获人形生物帝具须佐之男）。世界之源累至 **12.2%**。

逃出革命军总部后（沿途陷阱整惨追兵切尔茜），苏晓用「帝具改造蓝本」**合成出生物类帝具召唤物「[撕空恶犬·布布特尼](../人物/苏晓的随从与召唤物.md)」**（狼狗坐骑 + 奶妈·他的第一个随从 / 召唤物）。他大闹革命军直接逼得**革命军首领震怒、提前进攻帝都**（决战将至）。骑布布返帝都途中见特利全村已被焚烧部队屠尽（仍信守承诺把宝石抛进废墟）。回帝都得知狩人部队的 Dr.时尚、兰被**幻影旅团 [卡尔](../人物/卡尔.md)** 所杀；苏晓借布布的嗅觉循卡尔留下的塔罗牌气味，竟**嗅出了夜袭部队总部**。艾斯德斯遂率狩人部队全员剿杀夜袭，苏晓负责拖住赤瞳——却在森林中先撞上卡尔（700+ 法术的双枪法师），二人就此死战（苏晓以绝魔体质 + 青钢影「**[法力引爆](../力量体系/技能.md)**」硬扛其法术机关炮·战斗进行中）。猎杀目标残缺编号「**113？？号**」违规者仍未现身。详见 [斩·赤红之瞳页](../世界/任务世界/斩赤红之瞳.md)。
