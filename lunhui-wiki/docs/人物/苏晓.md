# 苏晓

**苏晓** 是《轮回乐园》主角，身份为 **猎杀者**——天赋特殊、更适合战斗的特殊契约者（见 [轮回乐园机制](../总览/轮回乐园机制.md)）。已完成首个任务世界 [海贼王](../世界/任务世界/海贼王.md)、获唯一职业 [灭法之影](../力量体系/技能.md)，现进入第二个任务世界 [东京喰种](../世界/任务世界/东京喰种.md)。

## 简介
入乐园前是现实世界的**复仇者**：三年弃学、专习杀人技巧（第1章）；刺杀仇人后被持枪保安「假死」、与轮回乐园签约（第1章）。信奉「**弱小即是原罪**」，视现实为牢笼、立志在乐园不断变强（第10、16章）。

## 性格
冷酷狠辣、为达目的可牺牲他人（第20章）；嗜战、享受生死一线（第13–15章）；越历磨难越强（强化拒用麻醉）（第26章）。

## 历程（按任务世界）

### 海贼王（LV.6 · 噩梦 · ✅已完成，评价 A-）
化名白夜：入侍卫队 → 杀欧卡 → 完成左大臣支线升侍卫首领 → 设局引克利克海贼团攻王宫 → 腰斩国王完成主线（第7–22章）。详见 [海贼王页](../世界/任务世界/海贼王.md)。

### 乐园停留 & 返回现实（第26–31章）
- 乐园：属性强化、试炼场习刀、升级刀术专精、布置专属房间（第26–28章）。
- 现实（化名**苏笑**）：醒于停尸间、夺回家传佩刀、成通缉犯，逃亡至沿海 **DL 市**租房隐居；识破一条「十万/月」租房广告实为**契约者**（疤脸大汉）以心理系手段设的陷阱（智力 >20 可免疫），予以无视（第29–31章）。

### 东京喰种（LV.3 · 困难 · 进行中）
化名白夜，与大批契约者同入。开局即一脚踢飞 [神代利世](神代利世.md)、首开 [青钢影](../力量体系/技能.md) 两刀斩 [壁虎](壁虎.md) 至半死；选边 **CCG**，另背猎杀任务（清理 12470 号契约者）。详见 [东京喰种页](../世界/任务世界/东京喰种.md)。
