# 巴尔（老巴尔）

[二阶生存试炼·吞噬岛](../世界/任务世界/T12K星球-生存试炼.md) 红土区 **[亚森曼部落（红皮土著）](../世界/生物万族/index.md)** 的**长老**——头戴羽饰、笑起来满脸褶子「像一朵盛开的菊花」，却是个**老狐狸**（表面热情、暗里一直打探 [苏晓](苏晓.md) 的来历）。

- **背景**：亚森曼部落是这颗 T12K 星球红土区的**原住民**（与外来入侵的 [刺蝎（虚空虫族）](../世界/生物万族/index.md) 已战争三百余年）；巴尔所在的是**大本营部落**（石屋林立、几百名女性 + 大量幼童·一夫一妻制·每胎生 2~3 个），战场上拼杀的多是外派精英。**他会说苏晓听得懂的通用语**。
- **与苏晓**：八卷22–23章战后现身，以食物 + 淡水相赠、欢迎这位「外来的强者」——苏晓由此获**亚森曼部落好感：友好（21/500）**。巴尔想把苏晓拉入土著阵营，更想用他那枚**罗盘**（类「杰克船长的罗盘」·指向红土区唯一的石桥、一天即可出红土区）做交易：食物水白送、**罗盘却要价码**。
- **交易条件 / 私心**（八卷24章）：他亮出吞噬岛地图（靶子结构）摊明罗盘价值，开出条件——**三天后部落与虫族开战（出动 1.5 万），要苏晓陪他去虫族老巢「抢一样东西」**（巴尔亲自同往·显是**见不得光的私事**·才找外人帮忙）。他即苏晓所杀吹箭酋长 **[巴里](../世界/生物万族/index.md)** 的**哥哥**——巴里是其弟兼竞争对手，得知弟弟被杀，巴尔难掩喜色「我『亲爱』的弟弟，你终于死了」，还旁敲侧击问苏晓在巴里部落有无见过「透明晶体（灵魂结晶）/ 黑色种子」。
- **血藤战士 / 求再生卵（八卷36–38章·真相）**：他确是**[血藤战士](../世界/生物万族/index.md)**——**血藤战士的「归宿」是回归太阳（俗称自爆）**，巴尔不愿这般死去，才冒险去虫族母巢夺**「再生卵」**（血藤战士的一条生路·却为族人所唾弃，故他不敢让部族知道、也不敢多带人探路〔内部觊觎其首领位者众〕，只能借外人苏晓的刀）。72 个儿子已死 60（10 在战场·2 残疾管事），战争由其第四子统领。过爆炸区时他**自断五指化「藤蔓小人」探雷**、不断灌淡红液体（再生 / 续命）、周身温度越升越高（苏晓警惕他随时自爆）。
- **早知苏晓身份**：他会通用语、显然**与契约者接触过多次**——因**吞噬岛不同于衍生世界**（乐园对此地只有干涉权、不能重置时间线），亚森曼世代生存于此、契约者只是一批批过客（详见 [机制](../总览/轮回乐园机制.md)）。其同行者 [卡洛](卡洛.md) 一句话漏出「契（约者）」即印证此点。

## 攻守易势 · 吸收再生卵（八卷48章）
围杀 [卡洛](卡洛.md) 后，巴尔向苏晓兑现承诺、带他目睹仪式。**他即将自爆**（体温逼近临界），当场**捏碎「再生卵」滴在腹腔内那颗赤红圆球上**（自爆之源），随即周身长出血藤、化作几米高「**藤茧**」——破茧而出后体温降到 50 度、容貌复原，自爆危机解除。

- **被苏晓反制**：藤茧期间巴尔极脆弱（苏晓一刀可解决），苏晓趁机**将剩余特级炼金炸弹化成粉尘混入空气、植入巴尔体内**（血液、骨骼里都有、成千上万颗微型炸弹·根本取不干净）。此前是巴尔用罗盘威胁苏晓，**如今攻守易势**——巴尔察觉却无可奈何，只得乖乖履约「带你目睹亚森曼部落的全部秘密」。
- **原生之种=黑色森林「门票」**：他点破**原生之种是血藤战士的根本、也是进入「黑色森林」的门票**——没有一定数量的原生之种，踏入黑色森林必死无疑。

## 亚森曼部落真相 · 看守者（八卷49–51章）
返部落途中巴尔倾吐了部落的根底（自知部落能否挺过这场「**寒灾**」未卜，故无妨说出）：

- **亚森曼人原是虚空种族**——是「**虚空中一位王者派下的看守者**」，奉命驻守吞噬岛、看守某种东西，已守了**几千年**；困守孤岛无法离开，文明与科技逐代退化，越来越像原始人。
- **几百年来一直是亚森曼主动攻打虫族**（并非虫族入侵）——只为夺取虫族母巢产的**再生卵**（血藤战士的唯一生路）。巴尔这已是**第六次来偷再生卵**。
- **血藤战士并非为对抗虫族而生**——而是为应对「**寒灾**」（没有寒灾就不会有血藤战士）。部落定居环境恶劣的红土区（四面悬崖、仅一条石桥·天然堡垒），正是为防黑色森林之变。
- 在后山「**太阳部落古圣地**」，巴尔**亲手为一名断臂少年主持血藤仪式**（开膛、植入生根的原生之种 + 灵魂结晶〔中〕、血藤接驳血管供血）——却意外**唤出沉睡的 [阳之女](阳之女.md)**（血藤本体·部落世代供奉的「血藤之母」），巴尔这才知自己一直借用的「力量之源」竟是这般存在，惊惧跪伏。

## 真面目 · 假神棍真枭雄（八卷52–53章）
阳之女要求苏晓带上巴尔同入黑色森林，巴尔在女神面前**恭顺至极**（领命「谨慎使用原生之种」、口称「女神的命令不容违背」）——但一出山洞，他的真性情就显露：苏晓讥他「拙劣的演技」，他坦言「**弑神是个不错的选择，现在还太早，先度过寒灾再说**」。**懂隐忍、趋利避害、阴险狡诈、手段狠辣**——这才是真正的老巴尔（为权力连亲弟弟巴里都能除掉）。

- **被阳之女种下「一次性能力」**：阳之女给了他一种**在黑色森林内有巨大用途的一次性能力**——但这能力**极不稳定**，使他现在**体温超过 80 度就危险**（远低于血藤战士平时的自爆阈值）。因此**进黑色森林前他全程不出手**，一路战斗都由苏晓负责，把这张底牌留到林中关键时刻。
- 他背上大量肉食淡水随苏晓出发，经罗盘指引赶往红土区唯一出口「**石桥**」（过桥后再走三天即达黑色森林·此时已是求生试炼第五天）。

## 进黑色森林 · 一次性能力（八卷62–65章）
进入黑色森林靠他**张口吐一分钟白火**在入口黑雾烧出窟窿（只维持十几秒·吐完萎靡）。林中他殿后、背最重的物资。**地图真相**也由他道破——那张兽皮地图**实为亚森曼祖先用鲜血与生命换来**（非巨人所绘·传说阳之女曾用它讨好巨人族），且黑色森林面积不与吞噬岛挂钩、**横穿即可离岛、不必返回**（与阳之女说法不同）。

- **「一次性能力」摊牌**：苏晓随 [野猪](野猪.md) 去寻 [暗王](暗王.md)、与众人分道后，巴尔向 [无伞兄](无伞兄.md) / [小迷糊](小迷糊.md) 亮出底气——「这是我和白夜**第二次合作**，如果拼命的话，我会死，可他也活不了」；之所以一路低调不出手，是因**体内被苏晓埋了几万颗炼金炸弹**（且阳之女种的黑森林专用一次性能力已「适应得差不多」）。他与苏晓「**各求所需**」地愉快分别（苏晓助他入林、他为苏晓清障引路），带无伞兄、小迷糊去 [阳之女](阳之女.md) 的「骷髅洞穴」。
- 他始终在**伺机弑神**——苏晓断言「若发现阳之女的弱点，第一个动手的绝对是老巴尔」。

## 结局 · 死于骷髅洞穴（八卷71章·侧面交代）
与 [无伞兄](无伞兄.md)、[小迷糊](小迷糊.md) 去阻止黑雾消散的「骷髅洞穴」之行**失败**——这位枭雄一世的老狐狸，最终**被同行的「奶妈」小迷糊毒死**（小迷糊真身是刺客·见其 [人物页](小迷糊.md)）。无伞兄断双臂重伤、靠在他尸体旁苦笑「小迷糊，原来你才是藏得最深那个」。他终究没能等到「弑神」、也没度过这场寒灾。
