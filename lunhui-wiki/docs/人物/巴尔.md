# 巴尔（老巴尔）

[二阶生存试炼·吞噬岛](../世界/任务世界/T12K星球-生存试炼.md) 红土区 **[亚森曼部落（红皮土著）](../世界/生物万族/index.md)** 的**长老**——头戴羽饰、笑起来满脸褶子「像一朵盛开的菊花」，却是个**老狐狸**（表面热情、暗里一直打探 [苏晓](苏晓.md) 的来历）。

- **背景**：亚森曼部落是这颗 T12K 星球红土区的**原住民**（与外来入侵的 [刺蝎（虚空虫族）](../世界/生物万族/index.md) 已战争三百余年）；巴尔所在的是**大本营部落**（石屋林立、几百名女性 + 大量幼童·一夫一妻制·每胎生 2~3 个），战场上拼杀的多是外派精英。**他会说苏晓听得懂的通用语**。
- **与苏晓**：八卷22–23章战后现身，以食物 + 淡水相赠、欢迎这位「外来的强者」——苏晓由此获**亚森曼部落好感：友好（21/500）**。巴尔想把苏晓拉入土著阵营，更想用他那枚**罗盘**（类「杰克船长的罗盘」·指向红土区唯一的石桥、一天即可出红土区）做交易：食物水白送、**罗盘却要价码**。
- **交易条件 / 私心**（八卷24章）：他亮出吞噬岛地图（靶子结构）摊明罗盘价值，开出条件——**三天后部落与虫族开战（出动 1.5 万），要苏晓陪他去虫族老巢「抢一样东西」**（巴尔亲自同往·显是**见不得光的私事**·才找外人帮忙）。他即苏晓所杀吹箭酋长 **[巴里](../世界/生物万族/index.md)** 的**哥哥**——巴里是其弟兼竞争对手，得知弟弟被杀，巴尔难掩喜色「我『亲爱』的弟弟，你终于死了」，还旁敲侧击问苏晓在巴里部落有无见过「透明晶体（灵魂结晶）/ 黑色种子」。
- **疑为血藤战士**：72 个儿子里 60 个已战死（10 在战场·2 残疾管事）。其**气息时而一股、时而两股**（似体内寄存某种东西），眼中藏着「被逼到绝境的疯狂」——苏晓怀疑**他自己也是「[血藤战士](../世界/生物万族/index.md)」**（随时可能自爆），部落后山的秘密就握在他手里。
