# 佐尔夫·J·金布利（红莲炼金术师）

「**红莲炼金术师**」——苏晓化名「爆炎炼金术师·白夜」入狱时的**狱友**（战争狂·爱杀人）。两人炼金术性质相似、战时合作过。

- **特点**：他并非出不去、而是**不想出**——享受狱中悠闲，安心等大炼金阵画好（杀人狂的耐心）。
- **与苏晓**：七卷14章苏晓被大总统 [拉斯](拉斯.md) 试探「有无亲人」时，只承认金布利这个「**朋友**」（既不露真弱点、又有个挡箭牌）。
