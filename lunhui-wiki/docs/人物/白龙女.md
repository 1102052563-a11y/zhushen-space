# 白龙女

**古龙国度·埃伯亚思**中沉睡的龙裔——[苏晓](苏晓.md) 用 98 枚钻石荣誉勋章从荣誉商店兑换「埃伯亚思进入凭证」（前置:未崇拜 / 赞美过太阳·否则被古龙敌视）后所见之人。身高不足三米、身材匀称、着冷白长裙覆半透明纱幕，说一口古龙语 + 奇利亚德语，端庄严肃、疏远高冷（在苏晓 −7 魅力面前尤甚）。她并非坐骑（苏晓一度失望），却是能授人「龙之力」的古龙遗存。

## 古龙阵营 vs 太阳神族

她的存在揭开了一段隐秘世界观:**古龙阵营（埃伯亚思）与太阳神族（[圣地·奇利亚德](../世界/任务世界/魔灵星-世界争夺战.md)）是绝对死敌**——古龙的力量源自黑暗 / 混沌，太阳神族则崇拜太阳。两大阵营原本强到「就算不及陨灭星那一梯队、也只弱一线」，却曾血拼一场、两败俱伤:古龙被彻底打废（如今只剩她一个），太阳神族也元气大伤、举族退迁到魔灵星、圣地退减成八阶危险区（其末路并非全因 [暗黑面具](../力量体系/装备道具.md)）。苏晓由此推断出太阳系职业（[太阳契约](../力量体系/装备道具.md)）、末代太阳王族不死者·休西罗 / 永生者·席兀等的来龙去脉。石桥上刻满奇利亚德语的胜利宣言（「古龙尽灭于我等之手」）与警告，正是太阳战士留下的。

## 与苏晓的交易 · 龙之力

苏晓确认她非坐骑后本想拿了徽章就走，她却端着龙裔的架子不肯结缔——被苏晓用 [罪落天遗](../力量体系/装备道具.md) 骨棍「以理服人」两下后（挨打不动、手背浮龙鳞），她「好龙不吃眼前亏」地同意结缔 [誓约之徽·白龙](../力量体系/装备道具.md)（因苏晓曾吸收龙血之力、契合度 +19）。相处中透露:

- **幸运女神的旧友**：她能接触外界——**虚空的幸运女神「幸运」偶尔来探望她**（苏晓神色如常记下这条线索，为日后捕捉命运 / 幸运女神埋伏笔）。
- **晶体操控**：她一指令水晶覆满全屋、瞬间凝出与苏晓等身的水晶塑像，说「这种控制力你也能做到、想知道方法吗」——苏晓极想学（能大幅强化他青钢影晶体层防御），却被她「时间到了哦」一句强制传送打断，报了挨骨棍之仇。
- **一桩托付**：她请苏晓若遇到还活着的古老飞龙，务必转告「埃伯亚思还存在、没被太阳神族彻底毁灭」。

## 关系

- [苏晓](苏晓.md)（库库林·白夜）：用骨棍「建立友谊」、与她结缔誓约之徽·白龙的人族猎杀者（苏晓惦记着再来学她的晶体操控）。
- **幸运女神「幸运」**：偶尔来探望她的虚空幸运女神旧友——苏晓暗记的重要线索。
- 太阳神族 / [圣地·奇利亚德](../世界/任务世界/魔灵星-世界争夺战.md)：与古龙阵营血拼两败俱伤的死敌。

## 🆕 「邪神」的祭献 · 被解谜游戏折磨到自闭（44卷56–60章）

[苏晓](苏晓.md) 在沙之世界靠不停向 [誓约之徽·白龙](../力量体系/装备道具.md) 祭献物品来刷品质、套太阳阵营回赠——这一切她都在埃伯亚思的禁足塔里「收货」，体验到一种**仿佛自己成了「等人祭献珍宝的邪神」**的奇妙感（孤单太久、这抽奖般的期待让她心情大好）。她收到苏晓祭献的戒指还很欢喜（「原来知晓吾喜欢何物」、闷气都散了），却也接连被祭献来的**狮树枝**（见太多、失望）、**爆炸物**（当场炸得靠结晶护体·「那些邪神也挺不容易」）搞得哭笑不得——殊不知灭法者本就爱用可定位的祭献物**钓邪神来揍**（灭法者公式:邪神=资源=我的）。最绝的是苏晓祭献了一台存着**皮胖那款「反人类」解谜游戏**的平板电脑当诱饵——端庄的龙之女硬撑着「一款游戏而已、不值得生气」，结果被虐了整整 **16 小时到自闭**（借口从「纱幕挡视线」到「网卡了」一应俱全）。这正是苏晓「先让她尝甜头、再套出晶体力量与坐标」计划的开端。
