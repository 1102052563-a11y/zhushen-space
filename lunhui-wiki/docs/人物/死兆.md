# 死兆（违规者10073 · 渴战逗逼少女）

[死神世界](../世界/任务世界/死神-尸魂界.md) 中 [苏晓](苏晓.md) 的**猎杀目标**——黑色短发、身穿敞开黑皮衣、露出平坦小腹与缠胸绷带的少女。轮回乐园直接标明其代号、并警示「**极度危险·已击杀两名三阶猎杀者**」（刚晋四阶没多久）。她与多数违规者截然不同：不躲不藏、**主动找猎杀者当目标**（视作「考验」·死了也不可惜），战斗时骚话不断、多变跳脱、是个十足的**话痨逗逼**（与一言不发的苏晓堪称两极）。

## 战斗风格 · 身体素质流

死兆是典型的**身体素质流**——把资源全倾斜在六维上，**力量 / 速度都强于苏晓**（爆发速度甚至不弱于 [贝妮](狩之影.md)），却**栽在技巧**上（与苏晓刚成 [刀术宗师](../力量体系/技能/index.md) 的差距，正如苏晓当初对宗师级镜像西尔西亚）。她另有违规者的诡异之处：**瞳孔涌血芒、以自身鲜血催动力量**（短时间力量剧增三成、代价是大量失血、脸色苍白）。

!!! note "虚空武器「沙耶诺德」"

    死兆的搭档兵器 **沙耶诺德**——来自**虚空**、能**变形为任意武器**（长弓「同化」射侵蚀黑箭 / 长枪「龙卷」/ 拳套 / 长柄战斧「吸血模式」/ 双倒刺刺剑「射杀」吸血 / 盾），还能**吸血**（被双刺剑刺中一剑即被吸走大量鲜血·苏晓中一刀瞬间被吸 20%）。它一眼**认出苏晓是灭法之影「魂影 / 断魂影」**、对那个「时代」颇为唏嘘（自陈与灭法之影交手「一个回合后他们就停手检查我死没死、没死顺手补一刀」），力劝死兆快逃——却被倔驴般的死兆拒绝。它与死兆是「同生共死」的欢乐损友组合（互相吐槽、死兆扬言把它泡粪坑）。它**没有卍解能力**。

## 死亡之钟结界 · 落败逃走（二十卷12–15章）

她追到一番队驻地两公里处、用「**世界阻断**」造出**死亡之钟结界**（20 分钟倒计时·世界炭化成黑白）单挑苏晓（为避开山本元柳斎等队长）。交手中她虽力速占优、却三回合挨三下狠的（被刀柄钝击破盾砸后颈、被青钢影刀芒所伤）；遂瞳孔血芒催力、用双刺剑吸血组合技伤了苏晓一刀。但结界倒计时一到，**苏晓判胜**——从她储存空间抽走金色特殊装备 [无尽黑暗](../力量体系/装备道具/index.md)（她气到差点哭）。她转身就逃（速度快到苏晓懒得追·挨了几枪 + 寂灭公爵一发雷弹仍逃脱），自我安慰「这次当打招呼、我感觉要升级了、获得大量经验值、绝不是白挨揍」。苏晓评她虽强于预想、却不是藏在暗处的老阴哔、而是个逗逼（这反而让他安心）。

## 反派埋伏 · 沙耶诺德终极形态（二十卷41–44章）

逃走后她在流魂街 80 区荒野孵化沙耶诺德的**新生体「小皮蛋」**（虚空战争兽幼生体·呆呆的、长满尖牙），并吸干几十名契约者的生命力（皮包骨干尸）布下杀局。苏晓循「诡异波动」找来，她**精心设伏**——小丑弹簧炸弹（苏晓靠龙影闪躲过）。苏晓反用 [布布汪](布布汪.md) 的虚空道具 **[负离子电浆超导装置](../力量体系/装备道具/index.md)**（巴纳小人族囚禁结界）把她困进直径 30 米的电浆网。走投无路的她不再跳脱、沉默地把沙耶诺德的**终极形态之一「破碎世界之枪·丧钟」**（黑色左轮·填弹 5 发）开到极致：

- **世界·刻**：开枪不出子弹、却让周围世界化为黑白——**时间系能力**（「在我的世界里没人比我更快」·把苏晓相对放慢）；
- **世界·崩坏**：射出乌光子弹、所过之处空间碎裂炭化（在她的凋零空间里轰窟窿·动能不弱于寂灭公爵）。
她以自身鲜血催动（短匕首刺心脏 + 黑胶遮半脸供血），却仍被近战专家苏晓压制——挨踢、被斩断左臂 + 锁骨。最后苏晓用 [青钢影](../力量体系/技能/index.md) 刀芒**麻痹她 0.3 秒**、抓住这一瞬抛出 [斩龙闪](../力量体系/装备道具/index.md) **穿其喉**、再用 [木之灵](../力量体系/装备道具/index.md)「木之野蛮」一脚把她踹上电浆网（脖颈以下尽失知觉），一句「永别，死兆」一枪爆头。她坦然看着初阳「原来我会在清晨战死」。沙耶诺德（有意识无身体·不肯投诚·想拉苏晓陪葬）随之化炽红自爆，苏晓早用龙影闪避开；新生体小皮蛋也被能量冲击搅碎。苏晓由此**清理违规者完成**、得黄金荣誉勋章 ×10 + 荣誉商店权限提升。

## 关系
- **[苏晓](苏晓.md)**：猎杀她的灭法之影「魂影」（她对其又恨又「欲罢不能」·誓要亲手杀他或被反杀·设伏决战）。
- **沙耶诺德**：她的虚空武器搭档（同生共死的欢乐损友·终极形态丧钟 / 破碎世界之枪·新生体小皮蛋）。
