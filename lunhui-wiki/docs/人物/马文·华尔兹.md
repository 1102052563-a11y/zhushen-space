# 马文·华尔兹

[苏晓](苏晓.md) 的**「启灵人」/ 正牌老师**——一位前辈 **[灭法之影](../力量体系/技能.md)**，曾**完善过「绝魔体质」**，如今已是一道**蓝色虚影（纯能量体）**，驻于虚空中那座灭法之影**传承大殿**（殿内六堆碎石原是六尊石雕真身）。每当苏晓满足条件解锁灭法之影新技能，便被传送回此与他相见。

## 启灵 · 灭法之影本源（五卷26–27章）
- 他向苏晓道出灭法之影的本源（**规则守护神**一脉），点评苏晓「有恐怖到发指的靠山（[轮回乐园](../总览/轮回乐园机制.md)）、资质比自己还好、将来或有 1% 的机会承载『能毁灭万界的能力』」。
- 强调杀法师**生存力（顽强）比攻击力更重要、属性须均衡**，遂为苏晓**启灵**（一指点眉心、把纯净法力融入血肉骨骼提升生存力·剧痛深入骨髓）——苏晓由此得「**[灵影体质](../力量体系/技能.md)**」（法力转生存力）。
- 性子不太靠谱、爱「忘事」：启灵前没提「会痛到昏厥」，传授吞噬之核时又漏说关键步骤（见下）。

## 揭秘阳之女 · 传授「吞噬之核」（八卷75–76章）
苏晓三主属性破 50、解锁灭法之影**第八个技能**时再见马文——

- **阳之女真相**：几百年前马文去吞噬岛红土区找材料，撞见自称「神」、要他供奉的藤蔓 [阳之女](阳之女.md)——他把她**连根拔起、在手里甩了几圈**，她跪地苦求饶命，他赶时间便放过、却**带走了她体内九成以上的能量**（本想等她恢复再夺、之后被法师围攻没回去）。这正是阳之女**对灭法之影有阴影、不敢对苏晓出手**的根由。他还点拨苏晓「等够强可去阳之女那夺能量·植物能量能滋养青钢影」。
- **传授吞噬之核**：他带苏晓到一池**液态青钢影能量**前，道破灭法之影「智力属性 = 承载强化青钢影的能力」之谜，命苏晓**跳进池中构建「吞噬之核」**（**85% 死亡率**·苏晓灵魂强度高故实际 50%+）。他却**漏说「构建 70% 即可粉碎」**，害苏晓硬构建到 100%、能量球爆裂重组——歪打正着让苏晓得了**初代灭法之影级的恐怖吸力**（详见 [吞噬之核](../力量体系/技能.md)）。
- **灭法之影传承之难**：他点明「那枚破戒指的价值配不上灭法之影——是**灭法之影选择了苏晓**；传承最难的不是天赋、而是**不怕与所有法师结仇的胆量**，大部分传承者都主动放弃」。

## ⚠️ 「刀魔」之诫（八卷76章·重大伏笔）
临别前（「**空间之力来了，下次见、那应该是很久之后的事**」）他反复叮嘱苏晓：**吸收能量时务必保持清晰、千万别成为「刀魔」**——「成为刀魔后**不可逆**……别成为下一个我、做一个灭法之影，**刀魔没前途**」。由其语气，**马文自己就是一名「刀魔」**。苏晓据此推断：**刀魔疑是灭法之影的一条分支 / 吸收能量时需承担的风险**（待解·苏晓暗自警惕）。他还提醒「最好打败对手后再吸收其能量」。

## 出处
