# 奇犽·揍敌客

[揍敌客家族](桀诺·揍敌客.md) 的**五子**、全职猎人世界的**剧情主角之一**——12 岁、刺猬头白发、皮肤白皙、天然惹人好感的小朋友，实则是个**有些危险的小杀手**（经家族残酷培训·**能无视 60% 以上毒素、无视电击**·既调皮又冷酷）。[苏晓](苏晓.md) 视他与 [杰·富力士](杰·富力士.md) 为猎人世界的「两个命运之子（伪）」（两小屁孩合体才是真正的命运之子）。

## 被卷入「揍敌客诅咒」之行（十六卷14章起）

他本在东果托国边境游历成长，却被父亲 [席巴](席巴·揍敌客.md) + 兄长 [伊路米](伊路米·揍敌客.md) 28 小时揪回枯枯戮山——成了 [马哈·揍敌客](马哈·揍敌客.md) 那趟「切断揍敌客诅咒」深入枯枯戮山行动的**特殊人员**（与 [亚路嘉](亚路嘉·揍敌客.md) 一同·两人「绝不能有事」），由苏晓负责保护。伊路米警告他「之后会见到一个人，不要违抗他、否则会死」（指苏晓）——他一见苏晓与马哈的气息（一黑一血红）就识趣地躲到苏晓身后、礼貌道谢，机灵不失礼。火山口岩浆、亞巴耶溶洞一路被苏晓拎着后颈带过（常被勒得脸发青）。

## 关系
- **[苏晓](苏晓.md)**：保护他过枯枯戮山的「猎杀者」（伊路米叮嘱他万勿违抗）。
- **[杰·富力士](杰·富力士.md)**：耿直的好友（主动跟来帮忙·两人是「命运之子」搭档）。
- **[席巴](席巴·揍敌客.md) / [伊路米](伊路米·揍敌客.md) / [桀诺](桀诺·揍敌客.md) / [马哈](马哈·揍敌客.md)**：父 / 兄 / 祖父 / 高祖。
