# 苏晓的随从与召唤物

本页**专门追踪 [苏晓](苏晓.md) 在各任务世界中获得的随从、契约生物、召唤物**——包括跟随他的 NPC、可召唤的战力、生物类道具 / 帝具召唤体等。每个随从单列一节记录其来源、能力、归属；**每个任务世界结束时，为在册随从补一份「结算快照」**（见末尾）。

!!! note "为何单列此页"
    苏晓身为**猎杀者**不便加入冒险团（怕猎杀目标是队友），但仍可能通过道具、技能、契约获得随从 / 召唤物（如召唤类首饰、生物类帝具、收服的 NPC 等）。这些附属战力与苏晓本体分开记录，避免淹没在 [苏晓](苏晓.md) 主页里。

## 在册随从 / 召唤物

### 撕空恶犬·布布特尼（生物类帝具 · 紫色）
- **来源**：用紫色「[帝具改造蓝本](../力量体系/装备道具.md)」合成而来（四卷41–42章）——主材料 = 帝国生物类帝具「魔兽变化·百臂巨人」（小白狗）+ 超级危险种 [芬里尔的精魄](../力量体系/消耗品材料.md)（默合度 96%）；副材料 = 7 件物品类帝具。归属者：苏晓。
- **外形**：黑白花色的**狼狗合体**巨兽（约 1 米多高 / 3 米多长·比虎更大），与巨狼芬里尔有几分神似；有点「二货」（让它向前冲会耿直撞墙），却对苏晓绝对忠诚、会**撕碎任何敢靠近苏晓的生物**（「恶犬」名副其实）。
- **核心形态**：可收起为拳头大小的**冰蓝色球体「布布特尼的核心」**（紫·评分 130·生物类帝具·售 36000 币·极便携，原著从未出现），随时召唤 / 收回，便于在帝都藏匿。需正常进食维持活力（约 3 天一次·饭量大）。
- **属性**：生命 100% ｜ 法力 260 ｜ 力 12 / 敏 15 / **体 20 / 智 26** / 魅 10（体力、智力**比苏晓还高**）。
- **技能**：
    - **全速冲锋**（主动）：移速 +100%（约等于 +30 敏的速度）·可载重·耗 10 法力/分。
    - **冰雪女神的馈赠**（光环）：每秒回友军 1 点生命·无消耗·随智力提升。
    - **忠心护主**（主动）：耗 50% 法力、3 秒内回主人 50% 生命·冷却 48 小时。
- **定位**：极品**辅助类**帝具 = **坐骑 + 山寨奶妈**（追杀 / 逃跑 / 持续回血一体）；战斗力不强（无攻击技能），战时多在旁待命、苏晓不顺时可骑它逃命。嗅觉远超普通犬类，是绝佳追踪工具（苏晓憾「若早有布布、雷欧奈赤瞳跑不掉」）。
- **历程**：合成当日即认主（四卷42章）；随苏晓骑乘赶路、在特利村废墟刨出幸存者、循塔罗牌气味助苏晓追查卡尔、嗅出夜袭部队总部（四卷42–44章）。在帝都须收成核心藏匿（其形似芬里尔、恐被艾斯德斯认出·且不在帝国 48 帝具记录内）。**苏晓 vs 卡尔一役，布布躲在暗处偷偷以「冰雪女神光环」+「忠心护主」为苏晓回血**——这是苏晓扛住卡尔自爆、最终反杀的关键（「因为我带了条狗」）（四卷46章）。在皇宫地下的 [比鲁马克炼金试验场](../世界/地点/index.md) 中，布布**不免疫炼金毒素**（每秒 -1 真伤·但靠自身光环可自愈），作为苏晓的**坐骑**载着他与小皇帝逃过蓝色小鸟群、聚合虫；对凶悍的地穴人看守也敢引走缠斗，并几次凶相毕露「提醒」不老实的小皇帝（四卷59–63章）。
- **寄生兽空间战场（五卷·随苏晓出战）**：在裁决者身份下被乐园免费**伪装成漆黑的「地狱三头犬」**（颈侧多生两头·仅契约者视角可见·能力不变），驮苏晓巡战场、拦逃兵、深入废土设世界坐标。受命去诱**变种马来鳄**群时连嘲讽带尿都引不动、反被鳄群追得「憋着半泡尿狂奔」（一度被吓尿·心里苦）；首领巨鳄偷袭时想助战被苏晓喝退（战斗力弱·留在「冰雪女神光环」覆盖范围内默默替主回血），决战迪卡时也帮苏晓挨了两拳、半死续命（五卷4–23章）。随核心带回乐园后与苏晓一同被绿光治愈。
- **乐园停留趣闻（五卷25–28章）**：精力旺盛、把 15 币的沙发啃穿藏身其中；苏晓带它去职工者宠物口粮店买烤骨头（**乐园有召唤兽口粮店**——召唤系职业若魅力低、召唤兽太饿会「噬主」，故需备口粮），女职工者一句「给它绝育治啃家具」吓得布布夹腿小碎步逃。布布魅力高达 10、兽语可与苏晓部分交流（苏晓兽语 Lv.6）。

## 结算快照（按任务世界）
> 规则：苏晓**离开某任务世界**时，若名下有随从 / 召唤物，在此为每个随从记录其**离世瞬间定格**（名称 / 来源 / 等级或属性 / 能力 / 归属与去向：随苏晓带出·留在本世界·阵亡·解约等），并注明「此为离世瞬间、与当前不同」。

- **海贼王 / 东京喰种 / 进击的巨人**：均无在册随从，无快照。
- **斩·赤红之瞳**（✅通关·四卷68章）：
    - **布布特尼**——离世定格：生物类帝具召唤物（紫·核心评分 130）·属性 力12/敏15/体20/智26·技能全速冲锋 / 冰雪女神光环 / 忠心护主；本世界全程是苏晓的坐骑 + 奶妈（卡尔战救主、试验场代步、嗅出夜袭总部）。
    - **去向：随苏晓带出**——它以「核心」形态存于储蓄空间、**回归乐园后仍在册**（似不受「帝具不可带出斩赤」之限·核心是乐园承认的生物类帝具实体）。回乐园后跳上苏晓的床、啃哑铃、被嘱「常识」，正式成为苏晓跨世界的常驻随从。
