# 茉莉（战神奶 · 战斗奶妈 · 被苏晓收买成内线）

红法袍紫眼影、战斗系 + 治疗系混合的女违规者（原是用花语作代号的「小白花」奶妈）。

- 被 [苏晓](../苏晓.md) 的 [血羽](../../力量体系/装备道具.md) 天克（治疗反成真伤）逼得**只能偷偷奶苏晓**（开发「近战奶」法杖打人回血·秀到苏晓），还得演「女战神」掩饰。
- 苏晓用「假心之毒」（其实是压缩晶体层·最难解的「心之毒」）收买她成内线（喂 [违规者联盟](斯芬克.md) 情报）。

## 关系

- [苏晓](../苏晓.md)：血羽天克她、收买她成内线的人。
- [斯芬克](斯芬克.md)：她被策反出卖的违规者联盟大佬。

## 出处

- 三十卷（[巫师大陆·赫鲁](../../世界/任务世界/巫师大陆-赫鲁.md)）31–72 章：被血羽天克→被收买成内线。
