# 支配者·索托斯（古神 · 「它们」· 殁）

「它们」= 从崩溃的巨大世界流亡来此、由「概念神」退化成的**古神**（重伤·吮吸此世界自疗）——索托斯是人形 + 黑触手下身、持「扭曲恶意」剑的古神。

- **支配**能力可无限制穿梭多重空间（古神殿内提速 300%）、攻击附空间切割伤·力 140 / 敏 152 极致机动但防御垫底（极强与极弱结合体）。曾被 [尼亚](尼亚.md) 斩去半头颅（永久 −13% 生命）。
- 把 [苏晓](../苏晓.md) 扯进古神殿想单挑、反被四打一斩杀（死前畏惧「灭法者……抱歉」）。

## 关系

- [尼亚](尼亚.md)：斩他半头颅的世界之子。
- [苏晓](../苏晓.md)：四打一斩杀他的猎手。
- [阿撒托斯](阿撒托斯.md)：与他同为「它们」的幕后古神。

## 出处

- 三十卷（[巫师大陆·赫鲁](../../世界/任务世界/巫师大陆-赫鲁.md)）17–44 章：扯苏晓进古神殿→被四打一斩杀。
