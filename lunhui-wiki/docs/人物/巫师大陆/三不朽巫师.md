# 三不朽巫师：曙光 / 古斯 / 劳伦特（曾经伟大的可怜人）

巫师会三位不朽巫师（六阶顶尖）——皆**天生巫师 + 曾为旧神**，为让世界恢复正常反抗「它们」、失败后被囚旧神教堂、死亡被推迟（活在痛苦中）。老巫师评其「曾经伟大的可怜人」。

- **劳伦特（摩恩·劳伦特 · 精神领袖）**：上个时代真主角——19 岁聚起曙光 / 古斯、耗 20 年创立**巫师会**、凑齐魔药配方传子嗣（[摩恩家族](摩恩家族.md) 始祖）；率 7.6 万巫师讨伐支配者大败、被触须刺穿变成「一坨」。**最惨却最 MVP**：识破幕后、关键时刻让 [尼亚](尼亚.md) 得以出战。
- **古斯（曾旧神 · 倒吊 · 擅战）**：燃烧自身淬圣银的莽汉（90% 重度烧伤·倒吊缓痛）——嘴臭心善、多次劝 [苏晓](../苏晓.md)「别送死」；受苏晓所托盯住曙光 / 劳伦特。
- **曙光（制造者 · 偏执）**：制造 [悲伤之女](悲伤之女.md)、培养 [尼亚](尼亚.md) 的不朽巫师——偏执坚信「只有强大个体能对付古神」、为尼亚的暗无天日童年深深愧疚。
- **结局（30卷55章）**：为淬高纯度圣银——**劳伦特 / 曙光主动入熔炉焚身**（劳伦特吞下古斯的两腿一臂同焚·曙光临终赞「原来最无畏的是你，古斯」），古斯独臂离去**继续培养新希望**（最艰难的抉择）。
