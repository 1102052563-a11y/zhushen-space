# 老巫师·修斯·阿奇德（友方引路人 · 真身 = 阿撒托斯意识寄体 · 殁）

基尔城巫师会分部里满身灰褐根系、坐轮椅的老巫师（双腿被 [悲伤之女](悲伤之女.md) 砸断）——表面是少数还清醒、看清世界真相的友方。

- 给 [苏晓](../苏晓.md) [真视之契约](../../力量体系/装备道具.md)（解除世界限制见真貌）+ 暗门传送阵直送赫鲁；盼苏晓「杀光『它们』」。
- **真身**：幕后古神 [阿撒托斯](阿撒托斯.md) **意识的上一届寄体**——修斯家族与 [摩恩家族](摩恩家族.md) 同源、传承阿撒托斯意识 + 明面恭维索托斯。被苏晓用「能量性剧毒」当酒灌下毒杀前诈出全部真相（家族同源、意识 / 兽性分裂、寄体机制）。

## 关系

- [苏晓](../苏晓.md)：被他引路、又诈供毒杀他的人。
- [阿撒托斯](阿撒托斯.md)：寄在他身上的古神意识。
- [摩恩家族](摩恩家族.md)：与他同源的一脉。

## 出处

- 三十卷（[巫师大陆·赫鲁](../../世界/任务世界/巫师大陆-赫鲁.md)）1–30 章：引路→真身揭为寄体→被诈供毒杀。
