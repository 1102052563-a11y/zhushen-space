# 世界之子·尼亚（希望之光 · 斩支配者半头颅而殉难 · 殁）

[曙光](三不朽巫师.md) 找到的天生巫师女童（带「它们」的神性）——12 岁起藏巫师会地下地牢、18 岁成年后比任何巫师都强。

- 带一把混圣银的剑「残月光」+ 人偶 [悲伤之女](悲伤之女.md) 去讨伐 [支配者·索托斯](索托斯.md)，**斩下支配者半个头颅而殉难**（绝技「勿忘我」·用后自尽以免古神血被夺）。
- 后世传颂「希望之光·尼亚」，但她本人只是个「想吃甜面包、比较强的普通人」——从只知杀戮的野兽被曙光救赎成人、为救赎者的愿望而战。其技法记忆「勿忘我」被 [苏晓](../苏晓.md) 剥得。

## 关系

- [曙光](三不朽巫师.md)：救赎并培养她的不朽巫师。
- [悲伤之女](悲伤之女.md)：陪她出战、为她长眠的人偶。
- [索托斯](索托斯.md)：被她斩去半头颅的古神。
