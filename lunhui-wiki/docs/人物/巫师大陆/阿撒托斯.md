# 阿撒托斯（幕后古神 · 意识 / 兽性分裂 · 意识被扯进贪婪之章）

真正的幕后黑手——本是「概念神」、重伤后**意识与兽性本体分裂**（兽性本体太强不接纳原意识）。

- 其**意识**借 [老巫师修斯](老巫师修斯.md) 家族 / [摩恩家族](摩恩家族.md)（同出一脉·修斯传承意识 + 恭维索托斯·摩恩当棋子）暗中操控一切：[劳伦特](三不朽巫师.md) 创会 / 讨伐 / 培养 [尼亚](尼亚.md) 都是为造「能战古神的容器」好让它夺回本体。
- 被 [苏晓](../苏晓.md) 不动声色拼出真相、设宴**扯进贪婪之章**喂生灵屠夫魂灵解决（意识）；**兽性·阿撒托斯本体仍在**（怕意识 + 兽性重合成七阶）。

## 关系

- [苏晓](../苏晓.md)：拼出真相、扯走其意识的人。
- [老巫师修斯](老巫师修斯.md) / [摩恩家族](摩恩家族.md)：承载其意识 / 当棋子的两脉。
- [索托斯](索托斯.md)：与他同为「它们」的古神。

## 出处

- 三十卷（[巫师大陆·赫鲁](../../世界/任务世界/巫师大陆-赫鲁.md)）17–44 章：幕后操控→意识被扯进贪婪之章。
