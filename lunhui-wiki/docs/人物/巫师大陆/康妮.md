# 康妮（斯芬克心腹 · 刀术宗师 · 亡魂 · 殁）

包子脸熊孩子——[斯芬克](斯芬克.md) 唯一不舍得牺牲的心腹、「没有依托就无法活下去的亡魂」。

- 双黑短刀（麻痹 + 介于史诗 / 圣灵级·星陨熔炉锻）、敏 132 凭空斩、纯黑眼瞳；瞬秒过墨镜男。
- 与 [苏晓](../苏晓.md) 单挑——力 / 敏更高却因**没在众生之地修行过**技法被碾压（同为刀术宗师实战差太多）、被苏晓断臂后斩杀。

## 关系

- [斯芬克](斯芬克.md)：唯一舍不得牺牲她的大佬。
- [苏晓](../苏晓.md)：技法碾压、斩杀她的刀术宗师。

## 出处

- 三十卷（[巫师大陆·赫鲁](../../世界/任务世界/巫师大陆-赫鲁.md)）45–72 章：与苏晓单挑被技法碾压斩杀。
