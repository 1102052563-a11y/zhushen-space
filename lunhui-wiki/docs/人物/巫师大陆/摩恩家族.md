# 摩恩家族（巫师会魔药调配者 · 阿撒托斯棋子）

巫师会负责调配 [魔药](../../力量体系/装备道具.md) 的家族（衔尾蛇徽印·住巫师会九层·始祖是 [劳伦特](三不朽巫师.md)）——与 [老巫师修斯](老巫师修斯.md) 家族同出一脉，是幕后古神 [阿撒托斯](阿撒托斯.md) 的棋子。魔药生意被 [苏晓](../苏晓.md) 的冰魂魔药抢光、派人除 [凯撒](../巫师大陆群像.md) 反被苏晓截杀。

- **摩恩·迪（秘法巫师 · 反伤 + 换位置 · 逃）**：精神错乱的秘法巫师——**攻击他所受伤害会放大数倍反噬攻击者** + 与队友**瞬间换位置**（「自带反伤甲的精神病巫师」）；被苏晓看穿换位后靠换位逃走。
- **摩恩·德尔（灰袍巫师 · 枯萎焰 · 殁）**：「枯萎焰」幽绿火无判定燃敌灵魂 + 肉体（对苏晓只成「给灵魂取暖」的增益）；逃跑踩中布布汪埋的磁爆猎手、被一刀斩首。
- **摩恩·埃尔莎（女巫师 · 扯灵魂）**：双手合十「扯出敌人灵魂」（对灵魂强度 113 的苏晓扯不动）；德尔死后落单待斩。
- **摩恩·阿尔弗思（最高掌权者 · 殁）**：黑袍族长·下令隐蔽除凯撒——被苏晓斩杀。

## 关系

- [苏晓](../苏晓.md)：抢其魔药生意、灭其家族的克星。
- [阿撒托斯](阿撒托斯.md)：把摩恩家族当棋子的幕后古神。
- [劳伦特](三不朽巫师.md)：摩恩家族始祖。
