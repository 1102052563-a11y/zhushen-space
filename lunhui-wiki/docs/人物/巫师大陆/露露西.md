# 露露西（圣光乐园 · 冰系 · 求合作被拒）

途经艾森拉来赫鲁求与 [苏晓](../苏晓.md) 合作的圣光乐园冰系妹子——谦和柔弱、带五女小队（含顶级大奶妈）。

- 打白山羊败、想用情报 / 治疗换合作，却被巴哈一通「情报全过时」怼到怀疑人生；苏晓因**时空之力交易的内奸未明**（可能是圣光乐园方）直接拒绝。

## 关系

- [苏晓](../苏晓.md)：她求合作、却被婉拒的对象。

## 出处

- 三十卷（[巫师大陆·赫鲁](../../世界/任务世界/巫师大陆-赫鲁.md)）31–44 章：求合作被拒。
