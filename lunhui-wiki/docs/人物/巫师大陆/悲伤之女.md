# 悲伤之女（旧神人偶 · 中立 · 长眠陪尼亚）

不朽巫师 [曙光](三不朽巫师.md) 为世界之子 [尼亚](尼亚.md) 所造的人偶——褐裙、淡金麻花辫、戴圣银头饰、皮肤柔软却无体温。

- **靠钥匙上发条**（一次活动七天）、有颗可拆的**水晶心**（装上心会学撒谎 / 想变活人·故只给半颗诱惑她）；知无不答、轻柔无波却藏着对初代主人尼亚的悲伤。
- **能驱散「黑暗」**（把侵蚀 [苏晓](../苏晓.md) 的黑暗吸入圣银头饰）、不伤任何活物——赫鲁城中立的关键人物。
- **结局（30卷55章）**：淬圣银时主动摘下圣银头饰投入熔炉、最后捧着一颗心脏靠在尼亚雕像上**自愿长眠陪伴尼亚**（满足地闭眼·证明人偶从始至终都有「心」、根本不需要水晶心）。

## 关系

- [尼亚](尼亚.md)：她初代主人、为之长眠的世界之子。
- [曙光](三不朽巫师.md)：造她的不朽巫师。
- [苏晓](../苏晓.md)：她的临时主人（驱散黑暗）。

## 出处

- 三十卷（[巫师大陆·赫鲁](../../世界/任务世界/巫师大陆-赫鲁.md)）1–55 章：中立助苏晓→自愿长眠陪尼亚。
