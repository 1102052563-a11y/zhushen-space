# 尤弥尔（颚之巨人）

**尤弥尔** 是 [进击的巨人](../世界/任务世界/进击的巨人.md) 世界**颚之巨人**的传承者、艾伦的训练兵团 104 期同期。**注意：她并非巨人之力始祖那位「尤弥尔」**，只是被冒用始祖之名的另一人（三卷40章）。

## 简介（身世 · 三卷40–41章）
七十年前，她本是 [马莱帝国](../势力/index.md)「艾尔迪亚族收容所」里一名无名的流浪小女孩；当地一处**邪教**看中她的艾尔迪亚血统，收留她、取名「尤弥尔」、谎称其为**始祖女神**以蒙骗信徒。十几岁时邪教据点被马莱政府查获，她为救信徒而承认自己「就是尤弥尔」——马莱不经审判，直接给她与众信徒注射**巨人药剂**变成**无垢巨人**（这是艾尔迪亚人犯罪的惩罚），统一流放到帕拉蒂岛。她以无垢巨人之身游荡整整六十年，机缘巧合吞下莱纳等人的一名同伴、获得**颚之巨人之力**而恢复人形（仍是十几岁的身体）。此后她潜入墙内以偷窃为生，并爱上了一个女人——**[赫里斯塔](赫里斯塔.md)**。

- **颚之巨人**：约 7 米、模样凶恶、四肢着地如野兽奔跑，最显眼的是一口咬合力惊人的**尖牙**；不擅长距离奔跑。
- 性格爽朗、毒舌、腹黑，却心地善良；性取向只钟情女性、对男人态度冷淡。

## 被苏晓俘虏（三卷41章）
她吞下赫里斯塔、趁乱跳下城墙追上苏晓一行，自称「来投诚」。但苏晓深知她在原著中**阵营摇摆不定**（一会帮莱纳、一会倒戈调查兵团），不给她临时倒戈的机会——直接提刀强攻、令莱纳的铠之巨人协助，斩断她双臂、再被铠之巨人咬伤以阻其变身，将她弄残后俘虏带走。此后她常以毒舌点破莱纳的精神分裂，被苏晓警告「再放嘴炮就割舌头」（三卷41–42章）。

## 关系
- [赫里斯塔](赫里斯塔.md)——她深爱的女人（同被苏晓一行掳走）。
- [莱纳·布朗](莱纳·布朗.md) / [贝特霍尔德·胡佛](贝特霍尔德·胡佛.md) / [阿尼](阿尼.md)——同为马莱出身的同期；她吞食过莱纳的一位同伴而得颚之巨人之力。
- [苏晓](苏晓.md)（白夜）——识破其摇摆本性、强行弄残俘虏，被她视为「比莱纳危险百倍」的男人。
