# 切尔茜

**切尔茜** 是 [斩·赤红之瞳](../世界/任务世界/斩赤红之瞳.md) 世界**革命军的帝具使**（四卷40–41章）。

## 简介
长发美少女、含棒棒糖、戴耳机、英伦式制服；**追踪能力为革命军最强**。持帝具「**变身自在·盖亚粉底**」（化妆盒形态）——可**随心意变成任何东西**（如变成一只小花猫潜行追踪）。

苏晓洗劫革命军帝具库、杀首领娜洁希坦后，革命军高层派她追杀苏晓、「带回尸体」。但苏晓沿途布置了二十多个陷阱（兽夹等），把她整得狼狈不堪（耳机丢失、险被夹断腿），终于精疲力竭放弃追踪——她不知再往前还有更多陷阱等着（四卷41章）。

## 关系
- 革命军——所属（追踪苏晓的任务）。
- [苏晓](苏晓.md)（白夜）——她奉命追杀、却被其陷阱整惨而放弃的对手。
