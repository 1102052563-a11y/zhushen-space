# 亚雷斯塔·克劳利（科学侧首领）

[魔法禁书目录](../世界/任务世界/魔法禁书目录.md) 世界**科学侧的首领**、[学园都市](../世界/任务世界/魔法禁书目录.md) 的理事长——疑为**四阶顶级（被世界阶位压制后仍不可招惹）**。满头白色长发（直立可垂到脚踝）、绿眸，长期**倒立浸泡在维生溶液玻璃罐**里、靠机械替他发声，已**不能算人类**（能力 / 实力 / 寿命 / 性格 / 目的皆不明·疑终极目的是「毁灭魔法」）。其监控手段「**滞空回线**」（肉眼难见的水母状监视器）遍布全城。

## 身世（十三卷10章）

曾被誉为「二十世纪最伟大的魔法师」（也有人骂他是下三滥）——他**祭献自己的妻子**召唤出守护天使「爱华斯」、女儿死也无动于衷、甚至拿同龄孩童做实验，终于站上魔法师巅峰。可他随即**抛弃魔法、转修当时被斥为异端的科学**（临走还朝魔法师的尊严吐了口痰），激怒所有魔法师——被英国清教一名魔法师「杀死」活埋，幸得一名医生挖出救活、此后只能困在维生装置里。「死过一次」让他对死亡产生恐惧，故把统括理事会本部大楼造成核弹都炸不毁的无窗堡垒、永不离开。

## 与苏晓的合作（十三卷10–11章）

苏晓为完成「末灵法典」任务、必须先与他达成共识（否则在满是监控的学园都市寸步难行）。苏晓伪装成「**魔法、怪异生物清理与调查组织（第三代猎魔家族）**」成员、亮出 [青钢影](../力量体系/技能.md)（魔法师天敌·赌亚雷斯塔解析不了它），并献上**最诱人的筹码——寻法典期间替他做事、不问缘由（时间 / 地点 / 目标给齐即可·杀人救人都做）**。亚雷斯塔（手下如一方通行 / 上条当麻多只能「利用」而无法直接下令）大为心动，**接纳苏晓加入科学侧**（开放阵营商店 / 阵营贡献值·见土御门元春时贡献值至关重要），但要他**见土御门元春前先办一件事**（土御门是其重要棋子、不许动）——即去击退追杀 **茵蒂克丝**的魔法师同伴 [神裂火织](神裂火织.md)。苏晓故意露出「易怒多疑」的不完美一面让滞空回线看到、以免「完美得可疑」。

## 翻脸 + 被「哈迪斯」逼怂（十三卷33–37章）

苏晓杀 [一方通行](一方通行.md)、夺末灵法典超出他的计划掌控（[罪七欲](罪七欲.md) 用情报拖住了他派去阻止的人），他遂翻脸、派几百架无人机 + 暗部 + 猎犬部队围杀苏晓。可苏晓造出绿磷燃烧弹 **[哈迪斯·冥界之焰](../力量体系/消耗品材料.md)** 烧掉第二学区四分之一、再威胁「第七学区还埋了一颗」——第七学区聚居七成学生、是他投入最多心血的学区（烧二区 = 剁他一只手·烧七区 = 砍他脑袋）。这个「20 世纪最伟大的魔法师」、学园都市大 BOSS，**一旦关乎自身关键利益就当场怂了**：30 秒内撤回所有追杀部队、把九成滞空回线调去第七学区戒备。苏晓由此安然离开学园都市（与他就此结仇·烧二区之恨几乎不可化解）。他的庞大计划看似滴水不漏，却忽略了「契约者」这个超出其理解的变量。

## 关系

- **木原数多**：直属部下（猎犬部队领导·暴虐天才·协助开发一方通行能力·替他向苏晓派活·车里载着御坂美琴的克隆体）。
