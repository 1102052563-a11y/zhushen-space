# 汉克

**汉克** 是 [海贼王](../世界/任务世界/海贼王.md) 世界、[哥亚王国](../世界/地点/index.md) 侍卫队的一名年长侍卫（第7章）。

## 简介
圆滑世故、惯于看上司脸色、收受好处。带化名「白夜」的 [苏晓](苏晓.md) 入侍卫队，被苏晓以香烟与贝利收买，成为其在王都的**情报来源与助手**（第7–9章）。苏晓升任侍卫首领后留他在身边办事（第17–19章）。

## 结局
第20章被苏晓派去伪装海贼、随 [克利克海贼团](../势力/index.md) 行动。当苏晓按计划**不开城门**、把海贼团困于火海时，迁怒的克利克下令将汉克绑起、抛入火海烧死——汉克至死才明白自己被苏晓当作弃子（第20章）。

## 经历
- 第7章：在侍卫队总部接待苏晓。
- 第8–9章：默许其夺斩龙闪、被灌酒套出情报。
- 第17–19章：作苏晓助手，联系克利克海贼团。
- 第20章：被牺牲，烧死。

## 出处
- 首次登场：第7章；阵亡：第20章。
