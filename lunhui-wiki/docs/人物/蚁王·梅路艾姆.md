# 蚁王·梅路艾姆

[全职猎人·蚁王篇](../世界/任务世界/全职猎人-蚁王篇.md) 的**终极 BOSS**、嵌合蚁的王——[苏晓](苏晓.md) 主线任务「蚁王的心脏」（Lv.30）的目标。比原著**提前 4 天诞生**，诞生即拥有强大实力 + 极高智商。

## 占领王都 · 谨慎扩张（十六卷30章）

它已占领东果陀国王都、准备把全国居民汇聚来当**食物储备**——却深知「不能随意扩张、要先稳定」（智商极高）。血脉记忆告诉它：嵌合蚁一族是**被从黑暗大陆赶出来的**（在那里嵌合蚁体型被压到 10 厘米、连基本生存都难），如今这片人类大陆相比之下简直是天堂——它因此疑惑「黑暗大陆的强大生物为何不来入侵」。它遇到的人类都弱小，更让它不解人类为何能在此繁衍生息。

## 念能力 · 比原著更强（十六卷33章）

其念能力为**特质系**——可通过**吞噬念能力者的血肉来增强自身的念、还能剥夺心仪的能力**（原著中吃了三护卫尤比 / 普夫的细胞便得二者能力、且到它手中还被增强）。**它已比原著更强**：两天前一名强化系契约者（纯粹肉体强化）袭杀三护卫「猫女」被反杀、又被蚁王吞食——蚁王虽没得特殊能力、却获得了**更强大的身体**（正因这变数，水咲 / 尘才主动找老烟、苏晓才与尼特罗等组「蚁王讨伐队」5 人围攻）。

!!! warning "V5 的底牌"

    V5 组织只派猎人公会（会长 [尼特罗](艾萨克·尼特罗.md)）来解决、态度随意（嵌合蚁不足为惧·免动军队破坏环境）。但若事态不可控、V5 出动军队，**东果陀国会从版图上消失、化为辐射地带**——苏晓深知不能让嵌合蚁发展到失控（否则只能弃主线用豁免权）。

## 现身 · 被 5 人围攻（十六卷39–42章）

讨伐队炸碎王宫逼它现身——**浅绿皮肤、天生的「圆顶头盔」甲壳护头、紫色轻甲、圆筒形针尖尾巴**，体态近人。它对两名护卫战死毫无感情波动、视人类为「低等生物」、嚣张地把苏晓等当「这世界最强的人类」（打算缩小领地）。一甩尾就抽碎了尘那颗百米石球（全力招被秒）。

**它的强与短板**：身体强到「合金打造」般、**抗击打力恐怖**——硬抗 [尼特罗](艾萨克·尼特罗.md) 最强招「**百式观音**」的「一式」/「九十九式」竟几乎毫发无伤。但它**能力极单一、没有自己的战斗风格**（只靠力量 + 速度 + 抗击打·诞生才几天、战斗经验「辣鸡」）——**真正可怕的是学习能力**（从硬抗到用甲壳挡 + 调角度减伤·越战越强）；吞了那名强化系契约者后身体强度又 +10%。

**首次重创**：尼特罗等三人只是「挡住它」的诱饵，真正的杀招在远处——水咲压箱底招爆其肩伤 + 体内水分成水蛇缠 0.3 秒、[尘](尘.md) 蓄力的雷电长矛刺穿其胸、[苏晓](苏晓.md) 800 米外寂灭公爵狙其头甲壳，**蚁王首次重伤**（胸口手指粗洞、头甲壳 1/3 被打飞）。它这才醒悟这是精心布置的围杀、真威胁来自远处的苏晓 / 尘，遂收起「学习」念头全力突围；首次主动攻击却被苏晓**预判狙击**（自己撞上子弹·小腹碗口洞）。

## 殒落 · 困兽犹斗 → 被爆头（十六卷43–45章）

重伤后它陷入「**困兽犹斗**」（生死危机下会拼命脱身、甚至拖人垫背），是围杀最危险的阶段。水咲以**冰霜巨人 + 海之冰**强行将它与会长冻结 3 秒——[苏晓](苏晓.md)（火属性元素弹·避开尘的暗属性免互相干扰）与 [尘](尘.md)（毁灭指）算准时机同时命中：解冻一刻攻击已抵面前、无从闪避，**尾巴大半被毁灭指轰飞**（失平衡感）、火属性弹轰碎头顶「圆顶帽」甲壳——**秃头**（之前苏晓那枪已碎其 1/3 头甲壳）。它困兽欲突围，做冲锋蓄力动作时**被苏晓预判狙击打穿大腿**（雷属性元素弹·拳头粗洞）。

最终它选错突围方向冲向 [老烟枪](老烟枪.md)（它最初傲慢地「学习」而非速杀远程的苏晓 / 尘、为此付出代价）——老烟以烟雾将它包裹、在烟内用压箱底本领**削弱它 24% 身体防御力**（双臂被它扯断为代价）。**苏晓的第三枪**（附 [恶魔之印·远程效果](苏晓.md)〔本次攻击附加最终伤害 80% 真实伤害〕的最后一颗暗弹）**直接轰碎蚁王头颅**——蓝血喷溅、无头身体踉跄扑倒。

> 【你杀死了蚁王·梅路艾姆。】获**世界之源 20.6%**（→共 45.9%）。蚁王死后心脏被瓜分（详见 [蚁王心脏争夺战](../世界/任务世界/全职猎人-蚁王篇.md)）。
