# 迪斯科（香波地奴隶拍卖场代言人）

香波地群岛上的海贼原作角色——头戴黄色圆顶帽、脚步虚浮的中年男人，[多弗朗明哥](多弗朗明哥.md) 的部下，替其打理岛上的奴隶拍卖场，某种程度上是多弗在香波地的「代言人」。集胆小、奸诈、欺凌弱小于一身（恶名昭著、除天龙人与海军外谁都不放眼里），但确有一项真本事——**商业头脑 + 完善的地下情报渠道**（手下有捕奴队）。年过 40、飘了多年。

## 替苏晓查鉴定师（十八卷72–73章）

[苏晓](苏晓.md) 要完成罗杰佩枪的「销赃」隐藏任务、急需找到藏匿的鉴定师托德·安德森，便通过 [多弗朗明哥](多弗朗明哥.md) 牵线（多弗已掌握人造恶魔果实技术、看不上贩奴生意，遂爽快应下、没向苏晓讨好处）。迪斯科被支使着满头大汗赶到 70 区旅馆——他畏惧「与四皇开战的狠人」白夜、办事却很利索：靠捕奴队迅速查出托德·安德森（49 岁·21 区暴动后躲到 35 区情人家）的下落交差。

## 关系

- **[苏晓](苏晓.md)（白夜）**：经多弗牵线、替其查鉴定师下落（畏其威名、不敢怠慢）。
