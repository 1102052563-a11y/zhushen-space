# 门罗（圣域乐园王牌 · 幸运女神信徒）

[海蓝星世界争夺战](../世界/任务世界/海蓝星-世界争夺战.md) 中**圣域乐园方的王牌之一**——金色瞳孔、声音磁性的短发男子，信仰**幸运女神**（圣域乐园多为墙头草信徒、他却专信一神）。打牌（扑克）即施法，更像独行侠而非大团成员。被苏晓击杀时也写作「罗门」。

- **强运领域（约 105 米结界）**：将周围运势凝于己身、全身金光「言出法随」——他攻击命中前一瞬把所受伤害**转嫁到领域内的碎石 / 家具**（看似无敌）；身处领域内的敌人**运势大幅下降**、周遭一切物件都生出「恶意」干扰其感知。配合扑克牌预示（梅花 5 = 五成胜算、黑桃 10 = 十成把握躲、红心 3 = 3 分钟愈合等）。
- **言出法随的代价**：用运势改现实时（「哑弹」让炼金炸弹失效、「失明」烫干敌眼球水分、「迟缓」改重力、「晕眩」空气锤砸后脑）**无法同时保持「攻击转嫁」**、且两次施法都站着不动——这正是其致命弱点。被苏晓承受的攻击若超领域上限、领域即崩。

## 被苏晓斩杀（十九卷16–18章）

苏晓在狐狸占卜屋单挑门罗——以双眼为赌注摸清其能力弱点（施法时不能转嫁），又靠「**被轮回乐园传送的锤击感比你的『晕眩』更狠**」**硬扛空气锤一秒都没晕**，趁其施法破绽用龙影闪绕后、**12 刀连斩击碎领域 + 金光大招**，将这名四阶王牌斩杀。门罗至死想不通苏晓为何没被空气锤捶晕——「死于攻击敌人后脑」。掉落苏晓**首个 [猩红宝箱](../力量体系/装备道具/index.md)**。

## 关系
- **圣域乐园**：所属敌对乐园（其王牌之一）。
- **[苏晓](苏晓.md)（白夜）**：海蓝星世界争夺战的对手（被其识破弱点、硬扛大招后斩杀）。
