# 征服王·伊斯坎达尔（Rider）

第四次圣杯战争 **Rider（骑兵）职阶**英灵——史上的**征服王伊斯坎达尔**（亚历山大大帝）。**满头酒红发、连络腮胡都是酒红色**的彪形大汉，披红色披风，给人粗犷大气之感，**带点二愣子潜质**；御主是怯弱可爱的少年魔术师 [韦伯·维尔维特](韦伯·维尔维特.md)。

## 实力 · 王者的群战

- **七英灵实力可排前几**——但他的强大**不在个人单挑、而在群战**：宝具能召唤生前麾下汹涌如潮的「**王之军势**」。
- 登场宝具即一辆**会在空中疾驰的战车**（[苏晓](苏晓.md) 凭敏捷徒步即可跟上）。
- 性情豪爽、爱凑热闹——废弃工厂一战他**主动现身、一出场就挑衅在场所有英灵**（「被这场战斗吸引来的英灵绝不止我一个」），活脱脱一个「二愣子」，却也借此把躲藏的英灵逼出来。

## 历程（十一卷51章·英灵齐聚）

[Saber](Saber.md) 与 Assassin [哈桑·萨巴赫（百貌）](哈桑·萨巴赫·百貌.md) 在废弃工厂交手时，他驾战车赶到、为这场打斗喝彩并挑衅众人——引出了傲慢的 [英雄王·吉尔伽美什](吉尔伽美什.md)。当吉尔伽美什气势压场时，他**挡在御主 [韦伯](韦伯·维尔维特.md) 身前护其周全**。

- **提议罢战 · 君子之风（十一卷53–54章）**：英雄王被时臣令咒召回后，他直言苏晓改造御主（[间桐雁夜](间桐雁夜.md)）成怪物的做法「不敢苟同」，又被苏晓一句「**在了解事情全部前妄加评论、这就是征服王？**」说得轻咳认错。他主张「混战难免草率、今晚到此为止」、获 Saber 与百貌附和——可百貌随即偷用气息遮断（疑再行暗杀），他当即蓝色电光一闪、**召出战车宝具「神威车轮 + 飞蹄雷牛」、单手抓起御主韦伯撤离**。其宝具登场印证了「群战之王」的轰隆声势。

- **谈判保御主 · 被放生（十一卷60–61章）**：苏晓为隐藏任务「英灵的荣耀」（清光御主令咒）找到他与御主的乡下藏身处——他感知到 Berserker 来袭、护住 [韦伯](韦伯·维尔维特.md)，却深知「王之军势」人海战术短时间挡不住苏晓那变态的速度、御主必死。苏晓也无意硬拼（杀韦伯导致他消散、对苏晓无收益），遂来**谈判**：只要韦伯**用掉全部 3 枚令咒**便放他们走。他劝服韦伯信任自己、用掉三令咒（其一竟是「不准再拍我肩膀」）——苏晓如约在韦伯低头的瞬间离去。一场凶险对峙以「君子协定」收场，他与韦伯都活了下来。

## 关系
- **[韦伯·维尔维特](韦伯·维尔维特.md)**：御主（被他当「跟班王妃」般使唤、实则真心相待·谈判时拼死维护）。
- **[英雄王·吉尔伽美什](吉尔伽美什.md)**：被其斥为「背着我自称为王的鼠辈」、针锋相对。
- **[苏晓](苏晓.md)**：英灵齐聚战中的同场对手。
