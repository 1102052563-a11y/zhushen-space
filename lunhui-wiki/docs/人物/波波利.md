# 小精灵·波波利

**灵魂书库兑换处**柜台里看书的小精灵——⚠ 他是 [暴躁小精灵·迪亚古](暴躁小精灵·迪亚古.md) 的**兄长**。⚠ 见 [苏晓](苏晓.md) 来，他**坐直身形、笑着问道**:「**白夜，家弟最近没给你惹什么麻烦吧。**」「没。」「**那就好，那就好，你这次来是？**」

## 📚 「这不巧了吗」

⚠ 苏晓这次来是要**把现有的书库银币全部投入到封印学方面**（以应对原罪之书和四件原罪物当前的情况）——⚠ 波波利的第一反应是劝阻:「**你不是取走封印大全没多久吗？那是很高等的封印学知识了，你先把那些掌握了……**」

- ⚠ 「**已经掌握。**」——⚠ 「**什么？！**」波波利**惊了**，⚠ **他上下打量苏晓，感慨道**:「**难不成，你是封印学方面的惊世奇才？**」⚠ 「并不是……」——⚠ **苏晓没藏拙，他的确不是，他只是掌握封印学的动力比较强而已，四件原罪物级别的动力。**
- ⚠ 「**嘶～，你这的确掌握的太快了，这次你弄到多少书库银币？**」「**1580 枚。**」⚠ 苏晓说话间**取出「封印大全」退还，这让他又得到 101 枚书库银币**——总计达到 **1681 枚**。
- ⚠ **柜台里的波波利沉思了一会，忽然问道**:「白夜，方便告诉我，**你是要封印什么吗？封印敌人？还是一些危险的生物？或是封印一些诡异的秘宝？**——⚠ **不同类型的封印，有不同的侧重点。**」「**原罪物。**」「**这样吗，你等会。**」
- ⚠ 他飞走，没一会**抱回来一个很有历史感的金属箱**（⚠ **这箱子如同是由熔化的暗银浇堆成**）:⚠ 「**高阶封印学，别听这个名字简单，这些知识的估价达到 3800 书库银币**——⚠ **当然了，我最近在整理顶层的知识，所以刚好把它分成了前、后两部分，又恰巧，前面这部分的兑换价是 1680 枚书库银币**——⚠ **你说，这不巧了吗。**」
- ⚠ 苏晓**放下书库银币与几瓶元素佳酿**，⚠ **小精灵·波波利的眼睛马上就亮了**——⚠ **显然是品尝过这美酒**。

## 🍺 弟弟送的那瓶酒

⚠ 不用想也知道，是 [暴躁小精灵·迪亚古](暴躁小精灵·迪亚古.md) **把苏晓分给他的几瓶元素佳酿，赠送给了自己兄长一瓶**。

- ⚠ **每次提及小精灵·波波利，暴躁小精灵·迪亚古都是满脸的厌恶与满嘴的不屑**——⚠ **结果这嗜酒成性的家伙，得到绝顶好酒后，竟不是自己独享，而是给自己老哥送去**。
- ⚠ **不用想都知道迪亚古送酒时的神情和语气**:⚠ 「**『嘁，老子最近戒酒，送你了』，说完，转身咽了下口水，然后大步走开。**」

## 关系

- [暴躁小精灵·迪亚古](暴躁小精灵·迪亚古.md)：他的弟弟——那个提起他就满脸厌恶、却把最好的酒偷偷送来的酒鬼。
- [苏晓](苏晓.md)：兑换处的老主顾——他因苏晓「取走封印大全没多久就全掌握了」而惊呼「难不成你是封印学方面的惊世奇才」（并不是，只是有四件原罪物级别的动力）。
- [古老者](古老者.md) / [灵魂长者](暗星/灵魂长者.md)：书库的拥有者 / 顶层那位酣睡多年的管理者。
