# 唐吉诃德·多弗朗明哥（七武海 · Joker）

[海贼王 · 德雷斯罗萨海战](../世界/任务世界/海贼王-德雷斯罗萨海战.md) 中 [苏晓](苏晓.md) 选定加入的势力首领——**唐吉诃德家族**之主、新世界声名赫赫的海贼。苏晓选这个三方中最弱势的阵营，正是看中他的特殊身份。

## 为何苏晓选他（十八卷3章）

- **原天龙人 → CP 部门不敢碰他的地盘**：这是苏晓的核心考量。司法岛事件后苏晓与世界政府彻底翻脸、被悬赏 **2 亿贝利（只活捉·因炼金炸弹技术有战略意义）**——CP 部门敢潜入四皇领地，却**不敢轻易涉足多弗朗明哥的地盘**（牵涉天龙人等太多问题）。加入他的势力，正好避开 CP 纠缠。
- **「聪明的海贼」**：成名后基本不在海上劫掠、而是和海贼们做生意（军火 / 奴隶 / 恶魔果实 / 淡水运输·凡违法的都参一手）。靠身世取得**七武海**之位、摇身变「合法海贼」（他视七武海为一种「特权」而非「世界政府的狗」）。
- **黑暗世界 BOSS「Joker（小丑）」**：坐拥德雷斯罗萨大岛、掌新世界 **70% 军火 + 80% 奴隶生意**、在香波地群岛有拍卖场、与四皇凯多合作；财力在新世界排得上前三。缺点是**太念旧情**（三名最高干部里只有琵卡得力）。得力干将**维尔戈**已潜入海军身居高位。

## 德雷斯罗萨海战中的处境（十八卷2–3章）

玲玲为夺他手中的**人造动物系恶魔果实技术**（核心 = 科学家 M·凯撒·库朗 + 德岛地下咚塔塔小人族），派长子卡塔库栗来攻——多弗朗明哥综合实力打不过卡塔库栗，幸得合作者凯多派**三灾·旱灾杰克**助阵（凯多极看重人造果实）。他本人是恶魔果实能力者，暴风雨中藏身船舱。

## 唐吉诃德家族结构
顶层是多弗朗明哥，下设四部门：**梅花**（干部托雷波尔·黏黏果实·用鼻涕战斗·管治安）/ **黑桃**（干部琵卡·石石果实·家族前三·多弗朗明哥护卫·辖古拉迪乌斯 / Baby-5 / 巴法罗·苏晓落在其「黑桃号」）/ **方块**（干部迪亚曼蒂·最老成员·管斗牛竞技场）/ **红心**（情报·见不得光·维尔戈 / 莫奈）。详见 [势力 · 唐吉诃德家族](../势力/index.md)。

## 与苏晓的「合作」谈判（十八卷13–16章）

苏晓登德岛后到王之高地见多弗朗明哥（近 3 米·金发·**除睡觉外永远戴太阳镜**遮眼不让人猜心思·**线线果实已开发到觉醒程度**·标志性「咈咈咈咈」笑）。苏晓送上 [斯佩罗](夏洛特·佩罗斯佩罗.md) 的首级当见面礼、还把「救了你部下 [琵卡](琵卡.md)」按成救命之恩——多弗朗明哥几度动了杀心（无形丝线缠向苏晓、被苏晓界断线缠回）。苏晓掏出 **[阿波罗](../力量体系/消耗品材料.md)** 当筹码（别墅内炸了只活得下两人），双方挑明利用、谈成合作：

- **合作内容**：苏晓帮唐吉诃德家族造**阿波罗 + 炼金炸弹**（多弗朗明哥见过特级炼金炸弹·海军靠它潜海炸船底·视苏晓为「瘟神」）；**爆炸物所有权归苏晓**、唐吉诃德家族有使用 / 征调 / 出售权——多弗朗明哥不懂苏晓为何执着所有权（实为苏晓借家族用炸弹杀敌、自己刷海贼声望）。多弗朗明哥则用「七武海身份替苏晓挡海军 / 世界政府 / CP」作回报。
- **想要技术被忽悠**：多弗朗明哥想直接要造炸弹的技术，苏晓的炼金阵图 + 文字他连标点都看不懂（自陈「Dr.贝加庞克 都破解不了、你绕不过我这一环」）——多弗朗明哥当面撕了图纸、改谈纯合作。
- **天龙人身份 + 人造果实真相**：苏晓点破他「原天龙人」（自陈在世界政府是中高层、参与过和平主义者计划〔熊的改造〕+ **人造恶魔果实完善**）——并道破：凯撒·库朗造的只是「让身体部分动物化」的半成品垃圾，**真正的开发者是 Dr.贝加庞克**（近乎能仿完整动物系果实·缺陷是削寿 / 暴走）。苏晓其实造不出人造果实（只擅长武器）、是忽悠住了多弗朗明哥。

## 困局 · 猪队友（十八卷16章）

德岛海战中唐吉诃德家族三最高干部除琵卡外尽墨（托雷波尔失踪、迪亚曼蒂重伤），普通干部死的死俘的俘（拉奥·G 战死、古拉迪乌斯重伤等）——契约者加入后大妈势力七成把握获胜。多弗朗明哥遂放弃海战、**固守德雷斯罗萨**（巨岩区是天然屏障·打登陆战），「带着一群猪队友能撑到现在全靠他个人能力」；还要提防「盟友」凯多想看他被打残后吞并他。他派女装大佬干部**德林杰**监视苏晓。
