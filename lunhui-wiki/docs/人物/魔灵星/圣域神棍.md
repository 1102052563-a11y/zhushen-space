# 圣域神棍（圣域乐园八阶契约者双人组 · 操控神灵化身 · 皆殁）

[魔灵星](../../世界/任务世界/魔灵星-世界争夺战.md) 世界争夺战里**圣域乐园**方的两名八阶契约者——**信奉同一位神灵**（故二人绝不内讧、这点最麻烦）。他们的特点是可**祭献 / 召唤出神灵化身**，是苏晓最忌惮的一类敌人（谁也不知他们能召出神灵化身还是难缠的「异诡之物」）。本批两名圣域神棍先后殒命（40卷9、14–15章）。

## 能力

- **神灵化身**：能操控自己信仰的神灵之化身（信的神灵**阶位远高于八阶**）——具现为**荧蓝色的十米高能量虚影**，可凭空伸出大手捏碎目标、头顶具现六米长的能量长枪（利于穿透）；本体可如幻影般穿透墙壁。召唤需付代价、且第二个神棍要**约 30 小时**才能唤出化身。
- **能量 + 精神系控制 / 替死**：水下的神棍用「能量 + 精神系」锁链穿透苏晓身体束缚（被刀术宗师豁免）；还有**替死能力**（帮同伴豁免灵魂斩杀·被斩时化背生双翼虚影替死）。
- **圣域打法**：敌对时惯于击杀后补刀、扔爆炸物（苏晓深知这点、故意用血气把藏暗处的神棍逼出来）。

## 经历 · 二人皆殁

- **水下神棍 · 殁**（40卷9章）：首个节点在 5 区河底，这名神棍好死不死地以能量 + 精神系控制苏晓——被 [刀术宗师](../../力量体系/技能.md) 豁免 + [灵魂凝视 Lv.70](../../力量体系/技能.md) 反制（苏晓灵魂强度 410），当场被灵魂斩杀（同伴用替死能力也只多撑一刻、终被补刀）。
- **另一神棍 · 唤化身后反被蛛卵吸干 · 殁**（40卷14–15章）：死了队友的第二名神棍，花 30 小时唤出神灵化身、当众捏死了蒸汽联盟 [铁颅·基普林](铁颅·基普林.md) 立威，随即想拉苏晓「联手先除掉天启那怪物」——却在无意间**泄露了「节点的秘密」**（杀光敌方全部契约者才是最终获胜者、虚空之树会把败方临时坐标剥离成奖励给胜者）。他打算借苏晓拖住 [尤妮丝](怕死鬼.md) 后再用神灵化身除掉苏晓，却不知白天照面时早已被尤妮丝种下蛛卵——刚召来化身就被蛛卵从口鼻眼耳爬出**吸成皮包骨**而亡（苏晓趁机将其神灵化身八成神灵能量吸入吞噬之核）。

## 关系

- [苏晓](../苏晓.md)（斩首的夜）：水下被其灵魂反制斩杀 / 另一人被其吸走神灵能量。
- [尤妮丝](怕死鬼.md)：第二名神棍算计她、反被她的蛛卵吸干。
- [铁颅·基普林](铁颅·基普林.md)：被第二名神棍的神灵化身捏死的蒸汽联盟代表。
