# 海伊左（死亡乐园契约者 · 树之竞技场前十 · 阴柔智囊 · 剧毒之蛇 · 殁）

[魔灵星](../../世界/任务世界/魔灵星-世界争夺战.md) 世界争夺战里**死亡乐园**方的**树之竞技场前十名**之一——满头长发、极其阴柔阴冷的男人（不是娘炮、只是「阴柔 + 阴冷、宛如海中的剧毒之蛇」）。与 [怒犬](怒犬.md) 是死亡乐园联军的一文一武（他主智谋）（第40卷27–30章）。

## 特点 · 智囊

- **见多识广的判断力**：一眼认出「黑魔」[小胖子](小胖子·黑魔.md) 的危险（听闻过传闻·「那东西不能杀」），第一时间让狙击手 [胡豺](海伊左.md) 快逃（可惜已晚）。
- **识破太阳圣殿机关**：凭治愈神教的机密档案（记载太阳圣殿是冈卢王朝历练「太阳勇士」之地）**推理出二层 50 个无门房间 = 50 重试炼 = 50 名轮回契约者的 1v1 强制决斗**（己方 8 人心灵共振小队进二层后 7 人随机消失死亡、非幻境），提出「选 50 名最强者走最前、把敌人这机关变成其墓场」的破解法，得到怒犬 / 莫雷认可。他是联军里最冷静的头脑。

## 经历

- **联军攻圣殿**（40卷27–30章）：与怒犬、[莫雷](../原始大陆/莫雷.md) 率联军攻太阳圣殿；被黑魔吞掉数十人、又被一层守卫 [复仇男爵 / 木乃伊](复仇男爵.md) 的再生 + 幻觉耍得团团转（连连感叹「还没见到敌人就被耍得团团转」）。靠他识破二层机关，联军才决定以怒犬为首、50 强者 + 百人跟进的阵型强攻。
- **一眼认出白夜 · 殁**（35–37章）：见 [苏晓](../苏晓.md) 的碾压式打法、一眼认出「我们遇到轮回乐园的白夜了、宰了绯世的那个」。六层围战中他酝酿海系大招（海水凝巨型章鱼），却在苏晓即将激活「死寂降临」时被吓退——最终被苏晓「刃道刀·流」斩杀（临死化竖瞳反扑重击苏晓胸腹）。他战至只剩独臂仍不失冷静、与怒犬同为死亡乐园「哪怕死也不让敌人好受」的硬骨头。

## 关系

- [苏晓](../苏晓.md)（白夜）：太阳圣殿防御战的敌方智囊。
- [怒犬](怒犬.md)：同为死亡乐园树之竞技场前十的联军搭档（一文一武）。
- 胡豺（死亡乐园狙击手·殁）：射杀小胖子触发黑魔、被海伊左劝逃却已晚、被黑魔吞噬的部下。
