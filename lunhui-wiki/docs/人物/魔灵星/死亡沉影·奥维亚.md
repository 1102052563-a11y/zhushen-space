# 死亡沉影·奥维亚（治愈神教最强者 · 烈阳圆盘持有者 · 中立善阵营 · 殁）

[魔灵星](../../世界/任务世界/魔灵星-世界争夺战.md) 东大陆**治愈神教的最强者**——整个治愈神教都掌握在她手中（权力比魔法协会会长 [罗格什](罗格什.md) 更集中、地位仅在治愈神教所信仰的神灵之下）。外形是一名满头雪白长发、黑烟构成缥缈长裙（裙摆拖地两米）、薄红唇似笑非笑、眼角黑眼影的知性妩媚美人；年龄看着二十出头。她是这个杀戮横行的世界里**为数不多的中立·善阵营强者**（「死亡沉影」这恶徒既视感的称号、只因其信仰的黑暗神灵、与她本人无关）（第40卷44–45章）。

## 身份 · 烈阳圆盘 · 太阳门小队

- **第五把钥匙·烈阳圆盘**：她持有太阳门「五把钥匙」里最重要、也是最后一把的**烈阳圆盘**——它一直被藏在治愈神教大教堂最里侧的石雕刻图里（拧下来是个碗口大石盘）。[神父](神父.md) 苦心积虑寻它、还在这大教堂献祭了 200 多契约者，却抠遍刻图都没发现它就在眼皮底下（被休西罗、巴哈笑懵）。钥匙不怕遗失——被钥匙认可者、哪怕钥匙丢了也能感应到其位置。
- **被逼加入 → 主动合作**：太阳门小队（苏晓 + 神父 + 罗格什 + 休西罗）以 5726 秘术师压境、摆出「不给烈阳圆盘就让治愈神教消失」的架势逼她——她当机立断**主动加入小队**（「翻脸比翻书还快」的反被她将了一军）。
- **筹码·太阳药剂**：她掌握**太阳药剂**的调制方法——**进入太阳门者必须先喝太阳药剂，否则十几秒就会「变成那里的一部分」而死**（罗格什当年正是喝了太阳药剂才得以触碰暗黑面具）。她以此为最大筹码、答应 3 天后带足太阳药剂、一同去冈卢古都太阳之梯。

## 动机 · 暗黑面具真相

奥维亚的目的与众人相反——**若真能找到暗黑面具，她会毫不犹豫地毁掉它**（那是「灾祸之源」、没了它西大陆不会那般贫瘠、世界都会不同）。她揭示了**暗黑面具的真相**：

- **魂能之源**：所有势力唯一已知的情报是——**魂能来自暗黑面具**。它并非产出魂能、而是**吸收掉这个世界所有自然生成的魂能（超凡之力）**；[黑潮](../../世界/任务世界/魔灵星-世界争夺战.md) 就是它吸满饱和后、在西大陆随机区域喷出魂能的现象（故西大陆资源贫瘠）。
- **诅咒**：轻微触碰即对人产生难以磨灭的影响；冈卢王朝的君王曾戴上暗黑面具——**137 年后、在无强敌 / 无天灾 / 无内乱的情况下、王朝在那位君王死去的当天骤然灭亡**（原因无人知晓）。暗黑面具虽是死物、却不断诱惑强者「永远消失在太阳门内」。

## 烈阳圣地 · 断臂（第40卷47–49章）

进入 [烈阳圣地·奇利亚德](../../世界/任务世界/魔灵星-世界争夺战.md) 后，她兑现承诺发太阳药剂（每人 1 瓶·由 [红瞳女](红瞳女.md) 背包带着·一旦有人夺就全毁）；却被 [罗格什](罗格什.md) 隐瞒圣地危险——一只可借蒸汽 / 烟雾移动的「王族侍女（污秽天使）」凭空出现、指尖一触就**炸断了她的整条左臂**（伤口如煤炭断面、无血、极难恢复）。她虽瞬秒了王族侍女、也意识到「这里已完全突破七阶、是八阶区域」。罗格什翻脸后她与红瞳女随神父触手一同脱身。她始终不肯拿出烈阳圆盘（知其作用、疑是她获胜的关键）。
