# 嘟嘟咕咕（烈阳圣地的友善单位 · 「收废品」的小骨手 · 疑外世界存在）

[烈阳圣地·奇利亚德](../../世界/任务世界/魔灵星-世界争夺战.md)（八阶危险区）荒村地下暗室里的一个**友善单位**——藏在墙内、只伸出几只婴儿手大小的荧白色小骨手，说话是「树树树」「亲亲亲」「亮晶晶闪光光」这类清澈叠词。**圣地内此类友善单位极其稀少**（危险物横行之地难得一见）。[苏晓](../苏晓.md) 推测它并非本世界存在、而是荒村村民濒临绝望时用未知方法召来的**外世界 / 更隐秘之地的存在**（第40卷52章）。

## 交易 · 「收废品」

- **魅力门槛**：因苏晓魅力 −1，它起初拒绝交易、还朝苏晓丢石头挑衅；苏晓拔刀要「揪出来砍了」，它立刻「亲亲亲」改口（「因不可抗原因同意交易」）。
- **只要亮晶晶的废品**：它想要「亮晶晶、闪光光」「神秘秘、烧烧烧」的东西——苏晓把储存空间里用不上的杂物（捏碎的酒瓶玻璃碎屑、[永恒意志](../../力量体系/装备道具/index.md) 圣灵宝石、白琥珀、巴之魂、暗行迷惑、布布的芥末薯片、巴哈的闲置太阳镜…）统统拿来换，它则回以**灵魂结晶〔完整〕×26 + 灵魂结晶〔小〕×15 + 一大堆「粗制的灵魂凝结物」（每颗约值 950~1072 魂币）+ 光之水**（圣灵级·可洗去同品阶物品的封印）——「它简直就是收废品的，什么都要」，苏晓这波血赚。它有物品价值概念（收玻璃碎屑时还愧疚地说「坏人坏坏坏」、仍给足诚意）。
- **交易有限 · 慷慨得祝福**：与它交易过多会招厄运（幸运永久下降·荒村村民正因交易过度而绝望终结）。它感念苏晓的慷慨、给了**双重增益祝福**（幸运临时 +5 / 真实力敏体临时 +4·各 12 小时·结束后可再来领）。

## 关系

- [苏晓](../苏晓.md)（白夜）：用一堆废品换得海量灵魂结晶 + 祝福的「大慷慨买家」。
- 荒村村民：濒临绝望时召来它的人（因交易过度招厄运、以相对安详的方式迎来终结）。
