# 毒锐剑·蒂芙（死亡乐园八阶契约者 · 敏智双特长 · 殁）

[魔灵星](../../世界/任务世界/魔灵星-世界争夺战.md) 世界争夺战里**死亡乐园**方的八阶契约者——扎马尾辫、右手戴五枚镶细链的指环（分别代表力 / 敏 / 智 / 体 / 魅五种属性、可扭动转化）。她是**敏、智双特长的技法型**「毒锐剑」，七阶时颇有名气、晋八阶后经历过三个世界（处于八阶初期）。与 [光头男](光头男.md) 组成死亡乐园双人组，来杀「斩首的夜」[苏晓](../苏晓.md)（40卷6章）。

## 能力

- **五链五属性转化**：右手五指环各连一细链、代表五属性，可将自身转化为某一属性特长——曾把自己转成**魅力特性（单属性魅力 208）**去侦测苏晓（代价是敏 / 智降到 170 以下）。
- **双刺剑 · 剧毒**：双手翠绿细刺剑、滴浅绿腐蚀液（落地嘶嘶腐蚀）；招式「**极刑**」可把大范围绿毒斩击匹链瞬间压缩成发丝粗的丝线（操控度与压缩力惊人）。战斗风格擅长挡住一刀后进行超高速连续斩击、以剧毒麻痹敌人。
- **对技法型的血泪认知**：她五阶时曾在一场 19 打 1 里见识过一名天启技法型契约者（刻「€」印记的刺剑）的「优雅屠杀」，痛定思痛才开始练剑术专精、却卡在 Lv.19——深知技法型是「契约者单体最强梯队」。

## 经历 · 殁

- **魅力侦测的困惑**（40卷6章）：她转魅力特性侦测苏晓，因苏晓**魅力是 −1 点**（[血气觉醒](../../力量体系/天赋.md) 所致）、特性不同，只测出「七阶契约者 + 魅力 −1」、其余全失败——她无法理解「魅力怎么达到 −1」（比转真实魅力还难），一度以为是圈套。
- **被一脚踹到失禁**：见苏晓只是七阶、本欲逃，仗着队友休西罗牵制才留下开打；结果苏晓一刀「青鬼」就斩势远超她预料、一脚直踹后腰把她踹成倒 C 形、吐着脏器碎片撞穿数墙——她跌进废弃教堂，羞愤于「被踹到失禁」（对八阶契约者是奇耻大辱）。
- **殁**：撤退时被界断线绷紧拦下，一招「[刃道刀·极](../../力量体系/技能/index.md)」斩断她的刺剑、直接斩穿她圣灵级贴身甲与心脏——她致死不信「这是七阶契约者」。苏晓得 5 功勋 + 10 猩红之勋（激活血色武器兑换列表）。

## 关系

- [苏晓](../苏晓.md)（斩首的夜）：来杀他、反被其一脚重创、终被斩杀。
- [光头男](光头男.md)：死亡乐园的同组队友（同批来杀苏晓、亦殁）。
