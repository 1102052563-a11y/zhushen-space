# 神父（圣域乐园契约者 · 献祭流 · 祭掉本方 200 契约者 · 阴险幸存者）

[魔灵星](../../世界/任务世界/魔灵星-世界争夺战.md) 世界争夺战里**圣域乐园**方的一名契约者——[苏晓](../苏晓.md) 在 [原始大陆](../../世界/任务世界/原始大陆-狩猎之地.md) 见过一面（能力当时未知）、评价其**「阴的很」**：原始大陆时苏晓召虫族把敌方压着打、最终仅几人得利，神父就是那不显山不露水、活到最后的得利者之一。苏晓认为**神父比月使徒更棘手**（第40卷21–23章）。

## 献祭流 · 队友祭天

世界争夺战开局后不久，圣域乐园方人数从 270 **暴跌到 25 人**——种种迹象表明是神父干的：

- **祭掉本方 200 多契约者**：某夜圣域方集合准备奇袭轮回乐园，却遭「未知袭击」；现场大殿留下**五座半没入墙体、胸口有太阳环印的石雕**（上方刻五种代表物：圆盘 / 权杖 / 棘鞭 / 衔尾蛇 / 火焰，再上是一张眉心流黑血的面具刻图），地面刻出注满鲜血的面具池，血字写着「**神父**」——是他以「献祭流」祭掉了圣域自己 200 多名契约者以换取强大力量（「队友祭天，法力无边」）。
- **规避处决之谜**：世界争夺战中献祭哪怕一名队友都会被本乐园强制处决，神父却用某种方法规避了这一点（具体不明）；其余 24 名圣域契约者在世界联络平台**始终一言不发**（疑为其同谋、或已被囚 / 昏迷）。他甚至没清除大殿血字（疑那座大殿对他很重要、不能破坏）。
- **世界联络平台**：曾在平台上就苏晓的动向发过「不妙」二字。其真实实力与能力仍是谜——苏晓判断，若他真献祭了 200 多契约者，所得力量会非常强，故暂时不去招惹、专注夺世界之核。

## 现身 · 五把钥匙 · 疑违规者（第40卷37章）

太阳圣殿决战尾声，神父终于现身——头发花白、身穿黑色长衣、戴十字架挂金链，在一层顺手放倒二十多名天启契约者，却对 [莫雷](../原始大陆/莫雷.md) 说「今天还不是你的死期」放她们逃走。他一路走上六层、在 [苏晓](../苏晓.md) 身旁坐下，道破他真正的目的——**并非夺世界之核，而是收集「五把钥匙」**：

- **五把钥匙 = 冈卢五物**：**蛇之板（衔尾蛇石板·在苏晓手中）/ 审判权杖（在魔法协会会长 [罗格什](面罩男.md) 手中·魔法协会保存多年的钥匙）/ 忏罪棘鞭（神父已找到）/ 烈阳圆盘（下落不明）/ 心之焰（在烟之都黑塔组织）**——「你、我、罗格什各有一把，剩下两把一起想办法」。罗格什正是 [休西罗](休西罗.md) 之父。
- **疑已是违规者**：他「很能隐忍、无论哪个世界都不是最强、实力排前几名」，直到这次世界争夺战才「积蓄好、露出狰狞的一面」——苏晓严重怀疑他已是违规者。他还提醒苏晓「月使徒大概召唤出 9 万只召唤物、小心些」后离去。
- **献祭圣域方**：世界争夺战结束时圣域乐园**存活 0 人**——全被神父献祭 / 消耗光（初期圣域方一直在旁看轮回与死亡死磕、然后就「没了」）。（六层的「毁灭区」爆炸物多是空壳幌子·神父来六层强行激活它们「开路」；天启隐匿系 [幽影](海伊左.md) 潜入六层时被「他自己带进圣殿的」圣域之人吞噬。）

## 关系

- [苏晓](../苏晓.md)（白夜）：原始大陆见过一面的阴险对手、太阳圣殿决战尾声与之并坐谈「五把钥匙」（各持一把·未联手）。
- 圣域乐园方契约者：被他献祭掉的 200 多名「队友」（世界争夺战结束时圣域存活 0 人）。
- [罗格什](面罩男.md)（魔法协会会长·[休西罗](休西罗.md) 之父）：持第二把钥匙「审判权杖」。
- [月使徒](../苍龙大陆/月使徒.md)：苏晓评「神父比月使徒更棘手」的参照对象。
