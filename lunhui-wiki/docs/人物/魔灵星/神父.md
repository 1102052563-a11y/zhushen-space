# 神父（圣域乐园契约者 · 献祭流 · 祭掉本方 200 契约者 · 阴险幸存者）

[魔灵星](../../世界/任务世界/魔灵星-世界争夺战.md) 世界争夺战里**圣域乐园**方的一名契约者——[苏晓](../苏晓.md) 在 [原始大陆](../../世界/任务世界/原始大陆-狩猎之地.md) 见过一面（能力当时未知）、评价其**「阴的很」**：原始大陆时苏晓召虫族把敌方压着打、最终仅几人得利，神父就是那不显山不露水、活到最后的得利者之一。苏晓认为**神父比月使徒更棘手**（第40卷21–23章）。

## 献祭流 · 队友祭天

世界争夺战开局后不久，圣域乐园方人数从 270 **暴跌到 25 人**——种种迹象表明是神父干的：

- **祭掉本方 200 多契约者**：某夜圣域方集合准备奇袭轮回乐园，却遭「未知袭击」；现场大殿留下**五座半没入墙体、胸口有太阳环印的石雕**（上方刻五种代表物：圆盘 / 权杖 / 棘鞭 / 衔尾蛇 / 火焰，再上是一张眉心流黑血的面具刻图），地面刻出注满鲜血的面具池，血字写着「**神父**」——是他以「献祭流」祭掉了圣域自己 200 多名契约者以换取强大力量（「队友祭天，法力无边」）。
- **规避处决之谜**：世界争夺战中献祭哪怕一名队友都会被本乐园强制处决，神父却用某种方法规避了这一点（具体不明）；其余 24 名圣域契约者在世界联络平台**始终一言不发**（疑为其同谋、或已被囚 / 昏迷）。他甚至没清除大殿血字（疑那座大殿对他很重要、不能破坏）。
- **世界联络平台**：曾在平台上就苏晓的动向发过「不妙」二字。其真实实力与能力仍是谜——苏晓判断，若他真献祭了 200 多契约者，所得力量会非常强，故暂时不去招惹、专注夺世界之核。

## 现身 · 五把钥匙 · 疑违规者（第40卷37章）

太阳圣殿决战尾声，神父终于现身——头发花白、身穿黑色长衣、戴十字架挂金链，在一层顺手放倒二十多名天启契约者，却对 [莫雷](../原始大陆/莫雷.md) 说「今天还不是你的死期」放她们逃走。他一路走上六层、在 [苏晓](../苏晓.md) 身旁坐下，道破他真正的目的——**并非夺世界之核，而是收集「五把钥匙」**：

- **五把钥匙 = 冈卢五物**：**蛇之板（衔尾蛇石板·在苏晓手中）/ 审判权杖（在魔法协会会长 [罗格什](面罩男.md) 手中·魔法协会保存多年的钥匙）/ 忏罪棘鞭（神父已找到）/ 烈阳圆盘（下落不明）/ 心之焰（在烟之都黑塔组织）**——「你、我、罗格什各有一把，剩下两把一起想办法」。罗格什正是 [休西罗](休西罗.md) 之父。
- **疑已是违规者**：他「很能隐忍、无论哪个世界都不是最强、实力排前几名」，直到这次世界争夺战才「积蓄好、露出狰狞的一面」——苏晓严重怀疑他已是违规者。他还提醒苏晓「月使徒大概召唤出 9 万只召唤物、小心些」后离去。
- **献祭圣域方**：世界争夺战结束时圣域乐园**存活 0 人**——全被神父献祭 / 消耗光（初期圣域方一直在旁看轮回与死亡死磕、然后就「没了」）。（六层的「毁灭区」爆炸物多是空壳幌子·神父来六层强行激活它们「开路」；天启隐匿系 [幽影](海伊左.md) 潜入六层时被「他自己带进圣殿的」圣域之人吞噬。）

## 确认违规者 · 目的暗黑面具 · 太阳门小队（第40卷39–45章）

世界争夺战结束后，神父的身份与动机彻底明朗——**他已彻底成为违规者**（属于「一到六阶是契约者、七阶将晋八阶时突然转违规者」这类蓄谋已久的特殊违规者）：

- **目的 = 暗黑面具**：他坦言真正想要的是**暗黑面具**（要开启太阳门、集齐五把钥匙才能拿到）。他**驾驭古神之力**（非盲目信仰古神——「力量没有善恶之分」、每位古神陨落时夺其力量）；气息带一丝古神味却无古神的诡谲感、疑曾与高位古神接触。曾与 [光暗世界](../../世界/任务世界/光暗世界-宿命之路.md) 的大主教约伯亚有交情、警告苏晓杀约伯亚会招来「几乎不会陨落的古神」。
- **太阳门小队**：他提议苏晓、[罗格什](罗格什.md) 三人暂时联合、共选另两名钥匙持有者（免得五个老阴哔齐聚开门「天崩地裂」）。带众人去魔法协会总部会合罗格什（此前他曾带「羔羊」夜袭魔法协会、交手后与罗格什转为合作）；又甩锅让 [休西罗](休西罗.md) 先探治愈神教大教堂找烈阳圆盘——却在那儿（他献祭 200 契约者之地）**抠遍石雕刻图都没发现烈阳圆盘就藏在其中**（被 [奥维亚](死亡沉影·奥维亚.md) 当场拧下、笑懵众人）。
- **羔羊 / 献祭**：他能把神职人员转化为长黑触手 / 羊角的「**羔羊**」（大教堂一战一名女神职人员异变、把治愈神教红袍神使吸干）。

## ⚠ 他现在有多危险

⚠ **上个世界进度，看似是最终胜过了神父**——⚠ **但别忘记，神父虽失去了所有，可作为古神体系能力的他，与原罪物·黄金圣杯进入了共存阶段**;⚠ **让一个人形古神能得到海量的纯净神血后，结果可想而知。**

- ⚠ **哪怕神父现在就晋升绝强，而且在绝强级有不弱的实力，苏晓也不会感到一丁点的意外**——⚠ **好消息是，神父应该还没完成试炼突破**;⚠ **否则的话，那边的问候早就来了。**

## 🐛 食暗者是他的克星

⚠ **当初在风海大陆，[食暗者](../风海大陆/黑眼存在.md) 为何找上苏晓？因为苏晓是灭法者？这只是原因之一而已**——⚠ **是神父想窃夺食暗者那的「印记碎片」，但又不能与绝强者实力的食暗者交锋，才出此对策。**

- ⚠ **食暗者加入『好队友』小队后，在苏晓的讲解下已经反应过来是怎么回事**——⚠ **心中正憋着恨呢，就等找到神父。**
- ⚠ **试问，神父最不愿意应对什么样的敌人？**⚠ **那就是食暗者这种憨憨**——⚠ **只要食暗者遇到神父并开始追杀，根本不会再听神父忽悠半个字**;⚠ **哪怕食暗者没可能真正杀死神父，但也能让神父的计划受阻。**
- ⚠ **这也是苏晓给食暗者送【暗影秘药】的原因**——⚠ **避免这憨批因这次没收获而生气退队**;⚠ **后续还要与神父交锋，这憨憨或许能成为杀手锏。**

## 🎣 生命晶石的收货人 · 晋升绝强（51卷26–34章）

⚠ **苏晓选定由「[厄兹勒](../永光世界/厄兹勒.md)」（神父假身）来「发现」三颗【生命晶石】的计划，成功了**——⚠ **神父的演技，历来无可挑剔，属于是苏晓、[伍德](../画中世界/沃波·伍德.md)、[格林·吉莉安](../格林·吉莉安.md) 同一级别的**;⚠ **伪装成神殿祭司长·厄兹勒，完全秉承了这大祭司以往的行事风格。**

- ⚠ **配套的骗局**:⚠ **他要求「三小时内，从黑铁城请一位工匠大师来开这宝盒」**——⚠ **而那三名有能力开盒的工匠大师，苏晓等人早就全部提前秘密合作了**（⚠ **不知无光神殿会找哪一个，就都谈妥，完美解决概率问题**）;⚠ **工匠大师当众胡诌「这是图布雅文明的产物，擅长生命力量和灵魂力量」**——⚠ **给三颗「生命晶石」提前铺垫出天然来历**（⚠ **就好像没人会怀疑铁匠家里为什么会有一把铁锤**）;⚠ **⚠ 轻易得来的东西惹人怀疑，藏在宝库、封着机关宝盒、还威逼利诱工匠大师才得到的宝物，才让人放心。**
- ⚠ **他没吞下贪婪**:⚠ **看到三颗生命晶石，说不动心是假的**——⚠ **但转瞬间他就泯灭那贪婪**:⚠ **这些晶石出现的时机可疑，而且还是三颗，这个数量实在耐人寻味**;⚠ **⚠ 心中有了某种猜测后，他对这三块晶石已经没有半点兴趣**——⚠ **因为「圣焰药师的猛毒，就算作为人形古神的他，也要给足警惕」**;⚠ **他把宝盒献给四巨头，四巨头中招，他全身而退。**
- ⚠ **他放出了始祖的死讯**:⚠ **通过他伪装的厄兹勒身份，他冒着被四巨头惩戒的风险，把「始祖死于灭法之影·白夜之手」的消息放了出去**——⚠ **这是他即将搞事的信号。**

## 🌑 他晋升绝强了

⚠ **在地下世界的混战中，神父晋升绝强了**——⚠ **一只由灰白人皮构成、生满黑色触手的巨手，抓上 [星界吞噬者](../永光世界/星界吞噬者.md) 的面门**;⚠ **就算是星界吞噬者，也因这诡异到极点的攻击，气息沉寂了几秒**——⚠ **这大敌，终于开始展露其强大手段。**

- ⚠ **他现在有多可怕**:⚠ **上个世界进度他虽失去所有，可作为古神体系的他与原罪物·黄金圣杯进入共存阶段**——⚠ **一个人形古神得到海量纯净神血，结果可想而知**;⚠ **苏晓发现，死在这老家伙手中的违规者，似乎一点不比自己斩杀的少**——⚠ **这次连 [黑魔](../index.md) 的代号都在联络列表里变成了半透明**（⚠ 从未出现过的情况）。

## 🕸 他真正的图谋：与猩红阵营虚与委蛇

⚠ **苏晓的推理**:⚠ **神父早就与猩红阵营有所合作**——⚠ **他之前散播的「[君主宝藏](../../世界/任务世界/永光世界-厄难囚笼.md)在永光世界」消息，配的那张照片，正是地下世界上层那部分秘宝**;⚠ **这张照片是猩红阵营许诺给神父的好处**——⚠ **这次被拖进永光，神父干脆谎称这就是「君主宝藏」放出去。**

- ⚠ **这也解释了他为何从不向四巨头透露苏晓的情报**:⚠ **因为神父并不是要投靠四巨头**——⚠ **他是在与猩红阵营虚与委蛇、互相利用，同时随时准备彼此背后捅刀子。**
- ⚠ **死寂烬灭的陷阱是他和猩红阵营联手布的**（详见世界页/装备页）:⚠ **他知道苏晓有黑王护臂、必对死寂烬灭很感兴趣**——⚠ **但他从始至终都没想过这陷阱会成功**;⚠ **⚠ 他真正的目的，是让苏晓与猩红阵营互相牵制**——⚠ **唯有如此，神父才能在此地完成他真正的目的。**
