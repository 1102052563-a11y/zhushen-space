# 神父（圣域乐园契约者 · 献祭流 · 祭掉本方 200 契约者 · 阴险幸存者）

[魔灵星](../../世界/任务世界/魔灵星-世界争夺战.md) 世界争夺战里**圣域乐园**方的一名契约者——[苏晓](../苏晓.md) 在 [原始大陆](../../世界/任务世界/原始大陆-狩猎之地.md) 见过一面（能力当时未知）、评价其**「阴的很」**：原始大陆时苏晓召虫族把敌方压着打、最终仅几人得利，神父就是那不显山不露水、活到最后的得利者之一。苏晓认为**神父比月使徒更棘手**（第40卷21–23章）。

## 献祭流 · 队友祭天

世界争夺战开局后不久，圣域乐园方人数从 270 **暴跌到 25 人**——种种迹象表明是神父干的：

- **祭掉本方 200 多契约者**：某夜圣域方集合准备奇袭轮回乐园，却遭「未知袭击」；现场大殿留下**五座半没入墙体、胸口有太阳环印的石雕**（上方刻五种代表物：圆盘 / 权杖 / 棘鞭 / 衔尾蛇 / 火焰，再上是一张眉心流黑血的面具刻图），地面刻出注满鲜血的面具池，血字写着「**神父**」——是他以「献祭流」祭掉了圣域自己 200 多名契约者以换取强大力量（「队友祭天，法力无边」）。
- **规避处决之谜**：世界争夺战中献祭哪怕一名队友都会被本乐园强制处决，神父却用某种方法规避了这一点（具体不明）；其余 24 名圣域契约者在世界联络平台**始终一言不发**（疑为其同谋、或已被囚 / 昏迷）。他甚至没清除大殿血字（疑那座大殿对他很重要、不能破坏）。
- **世界联络平台**：曾在平台上就苏晓的动向发过「不妙」二字。其真实实力与能力仍是谜——苏晓判断，若他真献祭了 200 多契约者，所得力量会非常强，故暂时不去招惹、专注夺世界之核。

## 关系

- [苏晓](../苏晓.md)（白夜）：原始大陆见过一面的阴险对手、苏晓视其为比月使徒更棘手的隐患。
- 圣域乐园方契约者：被他献祭掉的 200 多名「队友」。
- [月使徒](../苍龙大陆/月使徒.md)：苏晓评「神父比月使徒更棘手」的参照对象。
