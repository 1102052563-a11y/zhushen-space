# 艾妮儿（罗格什之女 · 休西罗之妹 · 暗黑面具的共存体 · 青春永驻）

[魔灵星](../../世界/任务世界/魔灵星-世界争夺战.md) 魔法协会会长 [罗格什](罗格什.md) 之女、[不死者·休西罗](休西罗.md) 之妹——一名绑鱼骨辫、眼角有小黑痣、青春活力的少女。她称罗格什「父亲」、称休西罗「哥哥」。**气息随情绪波动**（时而弱如常人、时而突然拔高）（第40卷41–42章）。

## 身份 · 暗黑面具的共存体

- **青春永驻 = 停止生长**：休西罗离家十几年，回来时艾妮儿却仍是他离家时十几岁的模样——因为**她是「暗黑面具的共存体」**（罗格什年轻时触碰暗黑面具、把这份「触碰」遗传给一双子女；艾妮儿由此在十几岁停止生长、休西罗则得「不死之痛」）。罗格什九年前起研究、两年前得出此结论。
- **黑秽之物将流尽**：罗格什称她「黑秽之物快流尽了」——那些恶意 / 贪婪 / 怠惰 / 傲怒（从「共存体」眉心孔洞淌出的黑色之物）流尽后她会变成什么，**无人知晓**；「在她看来、你我都只是陌生人」（罗格什警告休西罗别被她的亲情表演骗到）。她曾拿书砸罗格什（被其长发缠住）、为「父亲这样说哥哥」落泪——真假难辨。
- **诱饵「9 号」**：罗格什曾造一名顶着艾妮儿容貌的「半成品」女秘术师**9 号**当诱饵引休西罗回总部（9 号身上能刺出利刃、容貌可变、事后变回平凡脸退回沉眠之所）——用以说明「浸泡实验器具的 1 号（真艾妮儿）」与半成品的区别。

## 🆕真父 = 席兀（第40卷58–59章揭）

艾妮儿与哥哥休西罗其实是**奇利亚德最强者 [席兀](席兀.md)（永生者·戴暗黑面具者）的亲生子嗣**——[罗格什](罗格什.md) 并非其生父、而是**把兄妹俩从席兀手中夺走的仇人**。席兀正因思念被夺走的这一双子嗣、才痛苦到戴上暗黑面具沉痛枯坐；罗格什则「吃」席兀的孩子夺回踏足圣地的资格、还拿「艾妮儿命不久矣」激席兀。她的「暗黑面具共存体」体质，或与生父席兀（暗黑面具持有者）一脉相关。

## 关系

- [席兀](席兀.md)：她的**亲生父亲**（奇利亚德最强者·戴暗黑面具者·因思念她与休西罗而戴面具）。
- [罗格什](罗格什.md)：把她从生父席兀手中夺走的仇人（伪称其父）、九年来研究她的秘密、盼暗黑面具能让她「恢复」。
- [休西罗](休西罗.md)：其兄、被罗格什以她为筹码劝入太阳门小队（「你妹妹需要你、有希望就别放弃」）。
