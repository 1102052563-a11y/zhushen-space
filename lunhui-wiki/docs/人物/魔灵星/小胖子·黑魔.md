# 小胖子·黑魔（轮回乐园契约者 · 死后触发的吞噬型天灾 · 「不完全算友军」）

[魔灵星](../../世界/任务世界/魔灵星-世界争夺战.md) 世界争夺战里**轮回乐园**方的一名恐怖契约者——外号「**黑魔**」。外形是穿机车装、肩带尖钉、菠萝头、刺钉鞋、右耳挂金属小剪刀、双手插兜、周身金属挂件叮铃作响的小胖子，脸上挂着若有若无的笑意。连己方的疯子们都对他极度忌惮——[魔女](魔女.md) 称他「**不能完全算是友军**」（第40卷27–28章）。

## 能力 · 黑魔（死后触发的吞噬）

黑魔的能力是「打不得、杀不得、逃不掉」的无解存在：

- **宿主死亡后触发**：小胖子这具「宿主」被杀死后，能力才启动——受害者与周边一定范围内的**所有敌人立即不可移动、不可发出声音、不可呼吸**。
- **多级感应逐步吞噬**：被拖住后，黑魔以**热感 → 生物感应 → 水成分感应 → 气息感应**多个阶段逐级搜寻，从口鼻 / 肩膀等处涌出**黑色半流体**把猎物一点点吞噬、化为黑色流体涌回小胖子处让其「头部 / 身体再生」。
- **不分敌我**：一旦完全脱离宿体，黑魔**不分敌我、任何活物都是猎物**——故 [小凶](小凶.md) 警告己方绝不能去与黑魔联手。
- **吞噬后狂暴**：吞掉大量猎物后需狂暴几天。

## 经历

- **拖住天启死亡联军**（40卷27–28章）：世界争夺战开战前，他主动去古都外围「拖住」天启 + 死亡乐园联军。死亡乐园狙击手 [胡豺](海伊左.md) 一枪爆头「杀死」他触发黑魔——把联军近百人钉在原地不能动 / 不能呼吸；他还呆板地耍弄莫雷（假装放她走、又「我在骗你」），扯开自己整张脸让黑色半流体四涌。联军 592 人被他吞掉 **49 人**（剩 543 人进古都），代价是他要狂暴几天。

- **紧咬月使徒 · 走进太阳圣殿**（33、38章）：吞掉联军后黑魔在古都游荡、又盯上了天启召唤师 [月使徒](../苍龙大陆/月使徒.md)——吞了她几千只月兽（「可能月兽比较好吃」）、把她逼得四处逃命；与她从早撼到中午、数不清的召唤物褪去后双方黑泥停手达成某种共识。世界争夺战尾声，这个「大部分时间敌我不分」的黑魔竟主动走到太阳圣殿下、仰头与 [苏晓](../苏晓.md) 对视片刻、点了下头便走进圣殿（令巴哈与己方契约者都大为警惕）。
