# 罗格什（魔法协会会长 · 本世界权力最大者 · 审判权杖持有者 · 休西罗之父）

[魔灵星](../../世界/任务世界/魔灵星-世界争夺战.md) 东大陆**魔法协会**的会长——**本世界权力最大、也是最强的存在之一**。外形是一名赤脚、白衣、满头黑发垂到膝盖（发梢锐利如针）的男人，看起来并不冷酷、却令了解他的人既敬畏又恐惧。他能以**长发操控书籍 / 缠物**、所经之处**烛火尽被吸灭**、声音带独特魅力。是 [不死者·休西罗](休西罗.md) 之父、[面罩男·巴斯](面罩男.md) 的会长（第40卷41–45章）。

## 身份 · 审判权杖 · 暗黑面具

- **第三把钥匙·审判权杖**：他持有太阳门「五把钥匙」之一的**审判权杖**（魔法协会保存多年的「钥匙」·因他有资格使用才找到）。凭一己之力，多年来他也只集齐这一把——五钥匙极特殊、唯有资格使用者才能感知，故连他这般权势也难独力收集。
- **触碰过暗黑面具**：年轻时还是普通秘术师的他喝过**太阳药剂**、触碰过**暗黑面具**——自身看似无恙，却把那次「触碰」**遗传给一双子女**（[休西罗](休西罗.md) 与 [艾妮儿](艾妮儿.md)），导致艾妮儿十几岁即停止生长、休西罗则得了「不死之痛」。他九年前起研究、两年前得出结论：**女儿艾妮儿是「暗黑面具的共存体」**。他渴望暗黑面具的程度不亚于神父。

## 手段 · 城府

- **设局引休西罗**：他派一名顶着艾妮儿容貌的「半成品」女秘术师（**9 号**）当诱饵，算准休西罗一见此脸必杀回总部、乖乖送来「心之焰」——原计划休西罗回来即是其死期、由更听话的艾妮儿持钥匙；因 [神父](神父.md) 与苏晓带钥匙上门而改为合作。
- **压治愈神教**：为夺烈阳圆盘，他带 **5726 名秘术师**压境治愈神教大教堂、配合太阳门小队逼 [死亡沉影·奥维亚](死亡沉影·奥维亚.md) 就范（「若她不兑现承诺，就让治愈神教在这世上消失」）。对 [苏晓](../苏晓.md) 的评价：**开启太阳门前不会有问题、但那之后是「需赌上一切应对的大敌」**。

## 烈阳圣地 · 12 岁的真相 · 翻脸（第40卷47–49章）

太阳门开启、进入 [烈阳圣地·奇利亚德](../../世界/任务世界/魔灵星-世界争夺战.md) 后，罗格什的城府与迫切暴露无遗：

- **12 岁的真相**：他自称「上次来是从太阳里掉出来、当时有 35246 人」——实为撒谎。真相是当年某大人物绑来 **35 万平民**一批批送进烈阳圣地（用活人挡「太阳」直射、每个活人撑 2 秒、「蚂蚁过河」），**年仅 12 岁的罗格什是唯一撑到有建筑处的幸存者**。他把 [面罩男·巴斯](面罩男.md) 当试验品扔进太阳门测生存时间（7 秒）；又验证了休西罗 / 艾妮儿（暗黑面具共存体）能不喝太阳药剂进圣地。
- **懂烈阳圣地的语言 + 三钥匙之用**：**审判权杖**能以投影阴影指路（他一直隐瞒圣地情报、[奥维亚](死亡沉影·奥维亚.md) 被王族侍女削断左臂后才知他「你没问」式的坑）；他还懂圣地土著「太阳勇士」的语言、能命其跪拜。
- **翻脸被斩喉**：苏晓早察觉他「进圣地的意向太迫切、仿佛进来就必胜」，判断他会翻脸——罗格什果然用未知语言号令太阳勇士，[苏晓](../苏晓.md) 与神父抢先出手（刃道刀·时 / 极斩其喉 + 神父触手缠腿），他被 20+ 名太阳勇士（八阶精英级）护住、捂着喷血的喉咙而笑（疑之前的王族侍女 / 痛苦怨妇也是他故意引来）。布布汪潜伏在旁盯着他。

## 真相 · 夺子的仇人 · 与神父同盟（第40卷55–59章）

苦痛地牢里布布汪窃听到的通讯 + 乌奇 / 席兀的证言，揭开了罗格什的真面目：

- **与神父早已同盟**：进太阳门前罗格什就与 [神父](神父.md) 结盟、四处找钥匙持有者（钥匙相近会互相感应）。约定神父负责杀 [奥维亚](死亡沉影·奥维亚.md) 夺烈阳圆盘；他给神父一件避「太阳」灼照的活物器物作诚意。太阳勇士误伤神父只因不知这层计划。**痛苦怨妇掠走休西罗是意外**（休西罗是他计划里关键一环），故他急召太阳勇士撕破脸。
- **败于席兀 · 被剥太阳力**：他曾是**唯一有资格挑战奇利亚德最强者 [席兀](席兀.md) 的人、却败在其手下**；席兀剥夺了他的太阳力、令他多年无法再踏足圣地。他一直想尽办法重回奇利亚德。
- **夺子的仇人 · 「吃孩子」重回圣地**：**[休西罗](休西罗.md) 与 [艾妮儿](艾妮儿.md) 其实是席兀的亲生子嗣、被罗格什夺走**（罗格什并非二人生父）。他能重回圣地、唯一的可能是**「吃」了席兀的孩子**夺回资格。他到圣祭堂摘席兀的暗黑面具、被席兀一把捏碎手腕，却低语「你的子嗣休西罗、艾妮儿已长大、一个已死、另一个命不久矣、你去救他吧」——**拿子嗣激席兀起身**。

## 戴上暗黑面具 · 夺休西罗之力（第40卷63–65章）

在王宫偏殿，罗格什被苏晓 / 神父斩喉后靠黑发刺入自身操控残躯行动，被暴怒的 [休西罗](休西罗.md)（已掌控体内力量）一顿爆锤（「为艾妮儿 / 为巴斯」）近乎打死——却在最后一刻**戴上三分之一的暗黑面具**（面具破碎过、他先用面具眉心的「黑孔」），以黑孔的**强大吸力把休西罗吸进面具内、夺走其力量**。随即伤势尽复、拼合全面具（成功「只差一步之遥」·却感寂寞）。神父操控的「羔羊」来道贺、他淡然应对（用「灭法者的魔刃」提醒神父别乱来）。之后他藏进王宫、**要从暗黑面具里抽离休西罗的意识**才算彻底完成计划——他藏身在**环水祭坛**（当年败于席兀之地）；苏晓由 [噩梦王子](噩梦王子·埃律西克.md) 口中得知其位置、还放了一头顶配「大斧哥」去找他拼命。**苏晓推断罗格什求面具是想夺休西罗 / 席兀「无代价戴面具」的能力**（席兀是唯一戴面具没性情大变者）。

## 关系

- [席兀](席兀.md)：奇利亚德最强者、他唯一曾有资格挑战却败下的对手、被其剥夺太阳力；他夺走席兀的一双子嗣、如今又来摘其暗黑面具。
- [休西罗](休西罗.md)：他名义上的嫡子 / 实为夺来的席兀之子——被他戴面具用黑孔吸进面具内夺走力量。
- [噩梦王子·埃律西克](噩梦王子·埃律西克.md)：供出他藏身「环水祭坛」的画廊游魂。
- [面罩男·巴斯](面罩男.md)：跟随多年的秘术师、被他当试验品扔进太阳门测生存时间（他点破巴斯实为复仇而潜伏）。
- [死亡沉影·奥维亚](死亡沉影·奥维亚.md)：太阳门小队钥匙持有者、被他隐瞒圣地情报（她因此断左臂）。
- [休西罗](休西罗.md)：不成器却有顶尖秘术师资质的嫡子（「没遗传到智慧」）；设局引其送钥匙、又劝其加入太阳门小队救妹妹。
- [艾妮儿](艾妮儿.md)：暗黑面具共存体的女儿（他九年来的研究对象）。
- [苏晓](../苏晓.md)（白夜）：太阳门小队的合作者、也是他眼中「开门后需赌命应对的大敌」。
- [神父](神父.md) / [死亡沉影·奥维亚](死亡沉影·奥维亚.md)：太阳门小队的另两名钥匙持有者。
- [面罩男·巴斯](面罩男.md)：其麾下秘术师（在他面前抖得渗冷汗）。
