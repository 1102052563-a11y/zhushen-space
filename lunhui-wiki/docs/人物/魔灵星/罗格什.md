# 罗格什（魔法协会会长 · 本世界权力最大者 · 审判权杖持有者 · 休西罗之父）

[魔灵星](../../世界/任务世界/魔灵星-世界争夺战.md) 东大陆**魔法协会**的会长——**本世界权力最大、也是最强的存在之一**。外形是一名赤脚、白衣、满头黑发垂到膝盖（发梢锐利如针）的男人，看起来并不冷酷、却令了解他的人既敬畏又恐惧。他能以**长发操控书籍 / 缠物**、所经之处**烛火尽被吸灭**、声音带独特魅力。是 [不死者·休西罗](休西罗.md) 之父、[面罩男·巴斯](面罩男.md) 的会长（第40卷41–45章）。

## 身份 · 审判权杖 · 暗黑面具

- **第三把钥匙·审判权杖**：他持有太阳门「五把钥匙」之一的**审判权杖**（魔法协会保存多年的「钥匙」·因他有资格使用才找到）。凭一己之力，多年来他也只集齐这一把——五钥匙极特殊、唯有资格使用者才能感知，故连他这般权势也难独力收集。
- **触碰过暗黑面具**：年轻时还是普通秘术师的他喝过**太阳药剂**、触碰过**暗黑面具**——自身看似无恙，却把那次「触碰」**遗传给一双子女**（[休西罗](休西罗.md) 与 [艾妮儿](艾妮儿.md)），导致艾妮儿十几岁即停止生长、休西罗则得了「不死之痛」。他九年前起研究、两年前得出结论：**女儿艾妮儿是「暗黑面具的共存体」**。他渴望暗黑面具的程度不亚于神父。

## 手段 · 城府

- **设局引休西罗**：他派一名顶着艾妮儿容貌的「半成品」女秘术师（**9 号**）当诱饵，算准休西罗一见此脸必杀回总部、乖乖送来「心之焰」——原计划休西罗回来即是其死期、由更听话的艾妮儿持钥匙；因 [神父](神父.md) 与苏晓带钥匙上门而改为合作。
- **压治愈神教**：为夺烈阳圆盘，他带 **5726 名秘术师**压境治愈神教大教堂、配合太阳门小队逼 [死亡沉影·奥维亚](死亡沉影·奥维亚.md) 就范（「若她不兑现承诺，就让治愈神教在这世上消失」）。对 [苏晓](../苏晓.md) 的评价：**开启太阳门前不会有问题、但那之后是「需赌上一切应对的大敌」**。

## 关系

- [休西罗](休西罗.md)：不成器却有顶尖秘术师资质的嫡子（「没遗传到智慧」）；设局引其送钥匙、又劝其加入太阳门小队救妹妹。
- [艾妮儿](艾妮儿.md)：暗黑面具共存体的女儿（他九年来的研究对象）。
- [苏晓](../苏晓.md)（白夜）：太阳门小队的合作者、也是他眼中「开门后需赌命应对的大敌」。
- [神父](神父.md) / [死亡沉影·奥维亚](死亡沉影·奥维亚.md)：太阳门小队的另两名钥匙持有者。
- [面罩男·巴斯](面罩男.md)：其麾下秘术师（在他面前抖得渗冷汗）。
