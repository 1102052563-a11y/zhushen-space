# 卡奇（轮回乐园契约者 · 笑眯眯的幸运光环少年 · 金沙化身）

[魔灵星](../../世界/任务世界/魔灵星-世界争夺战.md) 世界争夺战里**轮回乐园**方（己方）的一名契约者——一个始终笑眯眯、眼睛像闭着般、口中总在咀嚼东西的少年。[苏晓](../苏晓.md) 与他同乘一班轨道列车时结识（第40卷23–24章）。

## 特点 · 能力

- **幸运光环**：他自称「只要和我同处一个交通工具，好运就会……」——话没说完，列车就被死亡乐园的远程光柱轰成两截（好运光环当场翻车、被巴哈吐槽「你的运气也不怎样」）。
- **金沙化身**：身体可化为**金色沙粒融入空气**遁走——苏晓放话「10 秒内消失」时，他感知到危险便化金沙脱身。
- **被三初始传送点坑**：他解说了魔法协会的三个初始传送点——**东大陆**（运气好）、**南岛 / 北群岛**（偏远、危险度与东大陆相近），并被魔法协会坑了一笔传送费。

## 经历

- **与蓝骑士的个人恩怨**（40卷23–24章）：他杀了死亡乐园 [蓝骑士](蓝骑士.md) 的一名队友，被蓝骑士三人组 + 2 公里外远程契约者追杀。他强调「这是我个人的事」、让苏晓别插手（还调侃「等我死了才轮到他对付你」）；死亡乐园远程光柱转而轰苏晓后，形势从「立场敌对」变成苏晓的「个人恩怨」，卡奇便化金沙识趣脱身（把蓝骑士留给苏晓收拾）。

## 关系

- [苏晓](../苏晓.md)（白夜）：同乘列车结识的己方契约者、把蓝骑士甩给苏晓解决。
- [蓝骑士](蓝骑士.md)：与他有「个人恩怨」的死亡乐园契约者（卡奇杀了其队友）。
