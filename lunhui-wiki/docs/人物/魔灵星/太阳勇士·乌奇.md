# 太阳勇士·乌奇（被囚的独角勇士 · 席兀之兄弟 · 苏晓的圣地领路人）

[烈阳圣地·奇利亚德](../../世界/任务世界/魔灵星-世界争夺战.md) 苦痛地牢里被囚的一名**特殊太阳勇士**——身高仅 1 米 3（身材比例匀称、圣地某种族）、下巴黑须、暗金色眼、头生金属独角、扛一把 1 米 2 长的大剑（一剑能把大腿粗的铁柱熔成铁水）。暴脾气、火爆记仇，却不蠢。他是圣地最强者 [席兀](席兀.md) 的**兄弟**、众多太阳勇士的教导者——因与席兀有约「不会离开这」而甘愿被囚（第40卷54–60章）。

## 经历 · 苏晓的领路人

- **被苏晓交涉唤出**（54、59章）：他是首个见到 [苏晓](../苏晓.md) 未直接动手的太阳勇士（囚久了寂寞、乐意交涉）。苏晓兑换「奇利亚德语」后与他攀谈——他道破圣地的三大追求（永生 / 暗黑面具 / 先知石板〔外乡人叫「衔尾蛇石板」〕）。当巴哈亮出苏晓与 [罗格什](罗格什.md) 的合照、证明「罗格什已重回圣地」，乌奇勃然大怒（席兀早已剥夺罗格什太阳力、他重回只可能是「吃了席兀的孩子」）——金色烈焰暴涌、一剑熔断铁柱冲出牢房，誓要去找罗格什拼命。
- **旧部尽叛 · 领路**（59章）：他想召集旧部却发现太阳勇士们**尽数投靠了罗格什**（近百人反追他）；他遂给苏晓当**圣地领路人**（去王城弄死罗格什顺便解闷）——并帮苏晓破解城墙上「炎灵」箭雨机关（虽最终靠布布汪搬对机关杆），把王城怪物引去城门坑守卫。
- **警告衔尾蛇石板**（60章）：他警告苏晓「衔尾蛇石板（先知石板）是活的、有预言能力、别完全信」——他自己就被它骗过（还被它「吃了剑」）。

## 关系

- [席兀](席兀.md)：他的兄弟、圣地最强者（因与其有约而守在苦痛地牢）。
- [罗格什](罗格什.md)：他誓要拼命的仇敌（重回圣地 = 吃了席兀的孩子）。
- [苏晓](../苏晓.md)（白夜）：交涉唤出、给其当圣地领路人 / 打工仔的外来者。
