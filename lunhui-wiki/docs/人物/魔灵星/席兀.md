# 席兀（奇利亚德最强者 · 永生者 · 暗黑面具持有者 · 休西罗与艾妮儿的真父）

[烈阳圣地·奇利亚德](../../世界/任务世界/魔灵星-世界争夺战.md) 的**最强者**——一名满头金色狮鬃般狂发、肌肉如花岗岩、身高三米以上的强壮男人，**永生者**。他坐在奇利亚德王城「圣祭堂」最深处、脸上戴着**暗黑面具**（骨白色、眉心一道食指粗的黑孔、黑血般的液体不断从孔中滴落进石盆），宛如雕塑般一动不动、沉痛枯坐（第40卷58–59章）。

## 身份 · 悲剧

- **奇利亚德最强 · 永生者**：他是圣地最强、[乌奇](太阳勇士·乌奇.md) 的兄弟；[罗格什](罗格什.md) 曾是**唯一有资格挑战他的人、最终仍败在他手中**（罗格什年轻时对他就有发自内心的恐惧）。他还**剥夺了罗格什的太阳力**、令罗格什永远无法再踏足奇利亚德。
- **戴上面具的原因 = 思念被夺走的子嗣**：他**太想念自己的一双子嗣**——[休西罗](休西罗.md) 与 [艾妮儿](艾妮儿.md) 才是他的亲生儿女，是**罗格什把他们从他手中夺走**（罗格什实为夺子的仇人、并非二人生父）。席兀因这份思念痛苦到不想动、不想言语、不想看到任何人，终究戴上了暗黑面具。
- **罗格什的激将 / 「吃孩子」重回圣地**：罗格什能重回奇利亚德，唯一的可能是**「吃」了席兀的孩子**（借此夺回踏足圣地的资格）。罗格什到圣祭堂摘他的面具、被他一把捏碎手腕（力量恐怖到不可置信）；罗格什却低语「你的两个子嗣休西罗、艾妮儿已长大成人、其中一个已死、再晚另一个也命不久矣、你作为真正的父亲去救他吧」——以子嗣激他起身。席兀缓缓站起、俯视罗格什。

## 关系

- [休西罗](休西罗.md) / [艾妮儿](艾妮儿.md)：他的亲生一双子嗣（被罗格什夺走·艾妮儿是暗黑面具共存体）。
- [罗格什](罗格什.md)：夺走他子嗣的仇人、败在他手下并被他剥夺太阳力、如今来摘他面具 / 拿子嗣激他。
- [乌奇](太阳勇士·乌奇.md)：他的兄弟（因与席兀有约而守在苦痛地牢不离开）。
- [苏晓](../苏晓.md)（白夜）：闯入圣地、正逼近王城核心区的外来者。
