# 面罩男·巴斯（魔法协会秘术师 · 人形全彩打印机 · 敬畏休西罗）

[魔灵星](../../世界/任务世界/魔灵星-世界争夺战.md) 西大陆烟之都的**魔法协会秘术师**，名叫**巴斯**——戴面罩、皮衣、脖颈金属项圈镶飘着铁砂状魂能的玻璃管，气息与 [休西罗](休西罗.md) 相当（烟之都强者极多、七阶中游契约者到此连小头目都未必打得过）。**手心镶着一块带时针 / 分针的表盘**。

## 能力 · 人形全彩打印机

他能从自己耳中抽出一支笔、以残影般的手速在笔记本上飞快画写，撕下一页即成一张**宛如打印机印出的「特许证明」**——工整字体 + 姓名 / 来历 / 身高 + **全彩上半身照片**（连 [苏晓](../苏晓.md) 发梢的水珠都画得一清二楚），巴哈怀疑他「上辈子是台打印机」。凭此特许证明，苏晓可全天进入 4~9 区（魔法协会地盘·原本黑塔成员晚 6 点后才可入）的大部分场所。

## 经历 · 对休西罗的敬畏

- **递特许证明**（40卷10章）：苏晓布下首个临时坐标出水后，面罩男带三名秘术师现身「查岗」，却礼貌地递上特许证明、邀苏晓去 4 区魔法协会分部；对休西罗抓走本会女秘术师毫不在意。苏晓让他转告会长**罗格什**「有个人在挂念他」。
- **被踢仍恭敬**：休西罗因他「态度太恭敬」一脚把他踹穿数墙，他歪着头校正好头颅、只说「失礼了」；三名秘术师欲以水刺围杀休西罗，被他一声「住手」喝退。他对休西罗**毕恭毕敬**——直言「还没见到你和那位大人的最终一战，我怎么能死」「望您安好，休西罗大人」，暗示休西罗身负一段与某位大人物的宿命对决。

- **五方齐聚被逼退**（40卷12–13章）：第三节点争夺时五方齐聚，被 [天罚](天罚.md) 用「与黑塔有恩怨、魔法协会要干涉吗」一句话架住——若干涉则魔法协会 / 黑塔 / 治愈神教三方结仇、他担不起，只得撂下「永远记住你今天说的话」带秘术师退场。
- **落柱潮夺魂能 · 与苏晓交易**（40卷18章）：落柱潮爆发，他率 153 名秘术师在 5 区血腥厮杀夺得 6275 单位原始魂能。苏晓却抛给他一盏装了 **5000 单位魂能的旧油灯**（布布汪在落柱潮里苟着偷吸的）当见面礼、微笑着开出条件——面罩男当即警觉（对他而言苏晓是要赌上一切才敢对付的绝对强敌）。

## 关系

- [休西罗](休西罗.md)：极度敬畏的对象（想见证他与「那位大人」的最终一战·而休西罗正是会长罗格什的独子）。
- [苏晓](../苏晓.md)（白夜）：以特许证明「招待」的黑塔成员；落柱潮后被其用 5000 单位魂能开出交易。
- **罗格什**：魔法协会会长、巴斯的上级、[休西罗](休西罗.md) 之父（暂未登场·被苏晓「挂念」）。
