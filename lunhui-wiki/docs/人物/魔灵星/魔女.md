# 魔女（轮回乐园契约者 · 旅团成员 · 老化致死能力 · 说话断续）

[魔灵星](../../世界/任务世界/魔灵星-世界争夺战.md) 世界争夺战里**轮回乐园**方（己方）的契约者——头戴女巫帽、抽长烟杆吐紫烟的旅团成员，刚晋升七阶不久就来参战。她**说话总是断断续续、要停顿好几次**（这是她能力的副作用），[苏晓](../苏晓.md) 嫌听着累、干脆坐到她背上交流（第40卷12、24章）。

## 能力 · 老化致死

- **加速老化**：她的能力可让目标**快速老化而死**——本批奉团长之命清理旅团的叛徒时，亲眼看着对方「快速地老去、死于她的能力」。副作用是让她自己说话也变得断续迟缓。

## 经历

- **清理旅团叛徒**（40卷24章）：团长发现旅团 5 号灰发男与违规者勾结、**已成半个违规者**，命魔女处决他。她与灰发男关系还算不错、亲手将其老化处死，眼中带着伤感。
- **转交世界之核**（40卷24章）：她拿到本世界的**世界之核**、却因古都附近有死亡乐园 [怒犬](../../世界/任务世界/魔灵星-世界争夺战.md) / 海伊左 + 天启 [莫雷](../原始大陆/莫雷.md) 带大量人马把守而难以进古都布设——遂**把世界之核交给苏晓保管**（用食指一推飘给他），由苏晓杀进古都的太阳圣殿。

## 关系

- [苏晓](../苏晓.md)（白夜）：同为轮回乐园己方、把世界之核托付给他布设。
- 旅团 5 号灰发男（叛徒·已老化处死）：她奉命清理的、关系还不错的旅团同伴。
- 团长：下令让她清理叛徒的旅团首领。
