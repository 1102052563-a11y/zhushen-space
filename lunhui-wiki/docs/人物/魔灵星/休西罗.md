# 不死者·休西罗（黑塔组织 · 八阶战力 · 魂印能力）

[魔灵星](../../世界/任务世界/魔灵星-世界争夺战.md) 西大陆烟之都的 [黑塔组织](../../世界/任务世界/魔灵星-世界争夺战.md) 成员——下半边脸戴皮质黑色骷髅面具、灰色瞳孔（不犀利却危险）、满嘴宛如鲨鱼牙般的尖牙。是**凯撒（己方契约者）忽悠来还人情的强力临时队友**，与 [苏晓](../苏晓.md) 组成临时小队、帮他找「节点」（第40卷5章）。他自称「不是好人、但承诺会遵守」，厌恶魔法协会（把抓来的女秘术师当与魔法协会谈判的筹码、扬言卖给炼金联盟）。**具八阶战力**。

## 能力 · 魂印（伤害同步 → 魂影斩魂）

休西罗的能力堪称「一对一无解」：

- **标刻魂印**：战斗开始前可**悄无声息地给一名活着的敌人标刻「魂印」**（最大目标 1）。标刻成功后，**只要被标刻者攻击休西罗，就会承受与休西罗相同的损伤**（其他敌人打休西罗则被标刻者不受伤）；范围 100 米。
- **成长 → 魂影**：魂印会「发芽成长」，最终从被标刻者背后长出一道与之相连的**黑灰色魂影**（手持倒曲短弯刀、尾端锁链连于胸口）——魂影一刀割喉即**斩杀敌人的「魂」**（[光头男](光头男.md) 正是这样脸上扭曲着死去、致死不信自己死在「七阶土著」手中）。
- **近乎不死**：可硬核地拍碎自己头颅 / 断臂来**解除精神控制**（断颈处涌血长出新头颅）；断肢会以与被标刻者同步的速度再生。**弱点**：伤势通过灵魂反馈 / 治疗有约 **0.1 秒延时**（[光头男](光头男.md) 用机械眼细胞级观察发现·可抓延时空当攻击使伤害不反馈）；且「强行扯下魂印」会让他看到敌人记忆并受其影响一段时间（曾扯下某女强者魂印后穿长裙上街、成人生三大黑历史之一）——他并非真正意义上的不死。
- **构建魂能生物**：能用魂能构建出仿真的「人」——**波兄弟**（一高瘦一矮胖·靠「耳光」触发伤害转嫁·被光头男「斩杀」后化黑铁砂消散）即是他用魂能构建的、当作战斗灵感的产物。

## 经历

- **组队与试探**（40卷5–6章）：迟到 27 分钟来见苏晓，亮出波兄弟、又扛来抓获的女秘术师；被苏晓看破波兄弟有诈（休西罗使眼色「看透别说破」）。
- **解控 + 斩光头男**（40卷6–8章）：一度被苏晓的仇家（死亡乐园光头男）精神控制、靠拍碎自己头颅清醒；随后与光头男单挑，装作满身破绽、任其一刀刀砍（伤害同步反馈），最终以**魂影割喉斩杀光头男的魂**。事后吐大口血却嘴硬「昨天吃鱼卡到刺」、不肯用珍贵的恢复药剂。因临时队伍，苏晓分得 60% 奖励（3 功勋 + 9 猩红之勋）。
- **护航布坐标**（40卷9–10章）：跟苏晓赶去 5 区节点、率先跳河探路（河底藏几百道气息）；节点布设时在岸上与巴哈防备天启二人组。
- **敬畏「那位大人」**：魔法协会的 [面罩男·巴斯](面罩男.md) 对他毕恭毕敬、直言「还没见到你和那位大人的最终一战、怎么能死」——暗示休西罗身负一段与某位大人物的宿命对决（他自己也留了一句「每个人都有秘密」）。淋到**黑潮雨**会立刻戴帽缩手躲避。
- **🆕真身 = 魔法协会会长罗格什的独子**（40卷13章·[天罚](天罚.md) 经 [红瞳女](红瞳女.md) 查出）：招惹休西罗等同**同时惹上魔法协会与黑塔**——这解释了面罩男·巴斯为何对他毕恭毕敬，也是「那位大人 = 罗格什（其父）」的伏笔。但休西罗清醒地不去东大陆复仇：「魔法协会记录在册的秘术师有 12850 名，我又不是天选之人，去了根本见不到罗格什、就会被别的秘术师宰了封成瓶装送到他面前」——他不是世界之子（巴哈原以为他该是顶级世界之子待遇）。
- **预示黑潮**：他能预示黑潮的降临（40卷12章·预示 5 区次日中午的「落柱潮」）——这是极危险的秘密（一旦被渴望原始魂能者知晓便无处可逃）；因坚守「信守承诺」的底线（近乎不死之躯的代价），他才向苏晓透露。
- **败给红瞳女**（40卷13–14章）：与治愈神教 [红瞳女](红瞳女.md) 交手时，因一见面就落入其幻境、只能追着她满街跑、沮丧到跪地捶地（十分丢人）；被红瞳女的手杖插头顶、又靠布布汪一大嘴巴抽醒才解控（碎颅解控对红瞳女无效·会被直接拖入必死的梦境）。事后他把红瞳女的红宝石手杖洗过还她。

## 🆕身世大揭 · 不死之痛 · 加入太阳门小队（第40卷41–45章）

苏晓通关世界争夺战、激活八阶晋升考核「太阳门」后，休西罗带**心之焰**（第五把钥匙之一）杀回魔法协会总部找父亲 [罗格什](罗格什.md) 算账——他的身世与「不死」的真相就此揭开：

- **被 9 号诱回**：罗格什派一名顶着其妹 [艾妮儿](艾妮儿.md) 容貌的「半成品」女秘术师**9 号**当诱饵（休西罗死活舍不得卖掉这张「与死去妹妹一模一样的脸」、正中下怀），把带着心之焰的休西罗引回总部——原本这是罗格什给他安排的死期（由更听话的艾妮儿持钥匙），却因苏晓 / 神父带钥匙上门而改为合作。
- **不死之痛的真相**：罗格什年轻时触碰过**暗黑面具**、把这份「触碰」遗传给一双子女——妹妹艾妮儿是「暗黑面具的共存体」（十几岁即停止生长），而休西罗的「不幸」则是**「不死之痛」**：他有顶尖秘术师资质却「没遗传到智慧」，眉心有个看不见的孔洞、会「淌出」黑色秽物（他的恶意 / 贪婪 / 怠惰 / 傲怒）——**黑秽流尽之日、便是他彻底变成「每天笑呵呵的智障」之时**（巴哈直白翻译：「你爹在说你蠢」）。他智商被罗格什、苏晓、神父全面碾压、屡屡被绕懵却又靠「我休西罗绝不会被骗」的迷之自信硬撑。
- **加入太阳门小队**：他嘴上「今天就算死在这也绝不合作」、5 分钟后端着红茶改口「暂时……合作吧」——**为救疑将异变的妹妹艾妮儿**、也因分不清罗格什的话真假，加入了太阳门小队（苏晓 + 神父 + 罗格什 + 奥维亚 + 休西罗五钥匙持有者之一：心之焰）。
- **背锅侠**（43–44章）：神父甩锅让他先探治愈神教大教堂（找烈阳圆盘），他被认出是「围杀尤妮丝那一战」的参与者、遭几百名神职人员 + [红瞳女](红瞳女.md)「喜提巨锅」箭雨招呼（幸有不死之身）。

## 关系

- [苏晓](../苏晓.md)（白夜 / 斩首的夜）：为还凯撒人情组队的临时队友、替他解决光头男；后同为太阳门小队成员；他只信任「在烟之都没坑过他」的苏晓。
- [罗格什](罗格什.md)：其父、魔法协会会长（设局引他送心之焰、劝他加入小队救妹妹）。
- [艾妮儿](艾妮儿.md)：其妹、暗黑面具共存体（他加入小队的主因）。
- 凯撒（己方契约者）：休西罗欠其人情、故帮苏晓。
- [面罩男·巴斯](面罩男.md) / 魔法协会：厌恶的对象；巴斯却对他极其敬畏。
- [光头男](光头男.md)：被其魂影斩杀的死亡乐园八阶契约者。
