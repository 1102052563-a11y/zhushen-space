# 红瞳女（治愈神教 · 蒙眼红瞳 · 幻境→梦境的清理武器）

[魔灵星](../../世界/任务世界/魔灵星-世界争夺战.md) 烟之都**治愈神教**的一名女性——身穿黑色宫廷长裙、目盲、双眼蒙黑布条（揭开是一双红宝石般的眸子）、脚踩高跟鞋、手持镶宝石的手杖当导盲棍。治愈神教把她定位为「**负责清理异端的武器**」，本批被 [天罚](天罚.md) 请来当强援对付 [休西罗](休西罗.md)（40卷12–14章）。她说一种除自己外无人能懂的独特语言（用其强大能力有代价）。

## 能力 · 幻境与梦境

红瞳女的能力极其可怕——**不是用她的精神力影响敌人，而是通过敌人自己的精神力反噬敌人**：

- **第一阶段·幻境**：受影响者会消沉、恍惚，用自己的精神力构成幻境（[布布汪](../布布汪.md) 中招后自己成了「可移动的幻境场」）。
- **第二阶段·困**：变得极困，一旦睡着就被**拖入梦境杀掉**——「幻境不可怕，可怕的是梦境，只要被拖入梦境，必死」。故 [休西罗](休西罗.md) 中招后只能几步一耳光硬撑不睡（碎颅解控对她无效、只会被直接拖入梦境）。

苏晓推测：刀术宗师能否豁免这种能力尚待实战——若是精神系控制则可豁免，若是增益类则未必。

## 经历

- **戏耍休西罗**（40卷13–14章）：与休西罗的「战斗」其实是她穿高跟鞋在前面走、休西罗在后面像无头苍蝇追、沮丧到跪地捶地；被休西罗骂到生气，才冒险把手杖插他头顶、踢他一脚离开。
- **治愈神教和解**（40卷14章）：天罚死后，治愈神教出面**将天罚与尤妮丝逐出教门**（称二人「滥用权职」）、并送苏晓一件圣灵级奖励 [纯净的魂源](../../力量体系/装备道具.md)（可将传说 / 史诗级物品升圣灵级）致歉——苏晓先按「无关个人恩怨」退回、后又收下，还借此让治愈神教欠他一个承诺。
- **交付吸魂能油灯**（40卷18章）：白天她给苏晓送来治愈神教神使、又取来「13 号货品」一盏能自动吸收原始魂能（最多 5000 单位·贪婪·活物）的旧油灯——兑现苏晓用「一个承诺」换来的「收取原始魂能的手段」。她还收下了休西罗洗干净还她的红宝石手杖。

- **守大教堂 · 被苏晓擒（40卷43–44章）**：太阳门小队为夺烈阳圆盘闯治愈神教大教堂，她率几百神职人员 + 银链大网守卫、用手杖波动令 [休西罗](休西罗.md) 精神恍惚（休西罗的「究极克星」再现）——却被 [苏晓](../苏晓.md) 界断线缠颈、三根晶体刺钉锁骨 / 后腰生擒。待 [死亡沉影·奥维亚](死亡沉影·奥维亚.md) 加入小队后，她随奥维亚一同退下。

## 关系

- [休西罗](休西罗.md)：交手 / 被戏耍的对象、后互还手杖（休西罗评她「白天看还挺漂亮」）；大教堂再遇仍是他的克星。
- [死亡沉影·奥维亚](死亡沉影·奥维亚.md)：治愈神教最强者、她的上级（随其退下）。
- [天罚](天罚.md) / [尤妮丝](怕死鬼.md)：请她来的天启二人（事后被治愈神教逐出教门当「歉礼」）。
- [苏晓](../苏晓.md)（白夜）：治愈神教向其致歉 / 交易的对象。
