# 小凶（钉脸小男孩 · 轮回乐园契约者 · 剥皮小魔鬼 · 增益 / 断头台锁技）

[魔灵星](../../世界/任务世界/魔灵星-世界争夺战.md) 世界争夺战里**轮回乐园**方的契约者——一个脸上打满金属钉、爱舔钉舌的小男孩，人称**小凶**。外表是小孩、本质是个会把敌人「一点点剥皮、让其哀嚎着死去」的小魔鬼（第40卷12、27章）。

## 能力

- **增益 / 断头台锁技**：能给队友加持增益状态——他给临时领袖 [升天兄](升天兄.md) 加增益时，用「**断头台锁技**」勒着对方脖子（勒到升天兄翻白眼放屁），一副恶趣味。
- **剥皮**：升天兄亲眼见过他把敌人一点点剥皮致死的残忍手段。

## 经历

- **忌惮黑魔**（40卷27章）：连这个残忍的小魔鬼都对来「拖住敌人」的「黑魔」[小胖子](小胖子·黑魔.md) 怕到瞳孔紧缩、躲到升天兄背后——他强调黑魔「不能完全算友军」、警告己方绝不能去与黑魔联手（黑魔脱离宿体后不分敌我）。

## 关系

- [升天兄](升天兄.md)：被他用断头台锁技勒着「加增益」的临时领袖。
- [小胖子·黑魔](小胖子·黑魔.md)：连他都忌惮、警告队友别靠近的己方「黑魔」。
- [苏晓](../苏晓.md)（白夜）：同为轮回乐园己方。
