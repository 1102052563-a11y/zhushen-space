# 天罚（天启乐园八阶契约者 · 刀术宗师战斗狂 · 殁）

[魔灵星](../../世界/任务世界/魔灵星-世界争夺战.md) 世界争夺战里**天启乐园**方的八阶契约者——一名神色如常的壮男，晋升八阶后已经历九个任务世界，初始势力为治愈神教（凭提前用的道具地位不低）。与队友 [尤妮丝](怕死鬼.md)（伪装成「胆小少女」）组成天启双人组。**战斗形态是身高超 5 米、生满褐色角质的小巨人**，是**刀术宗师**、一个「除了斩人没别的爱好」的战斗狂人——「心细如发」却因缺乏高强度残酷战斗而稍逊。

## 能力

- **世界链束缚**：因八阶战力过强，进入七阶世界后被世界之力用一根「**世界链**」束缚（链上遍布裂痕、随时间推移逐渐解除）——「能被束缚一根世界链，是对他实力的肯定」（侧面印证 [苏晓](../苏晓.md) 战力超七阶、故延时进入且不受排斥）。
- **圣灵级 +13 三宝石合金大刀**：战斗形态持一把三米长、巴掌宽的科技感合金长刀（淡金花纹·可充能）——他是**刀术宗师**，自创绝技「**地牙**」（大刀刺地爆出几百道淡金刀芒·大范围）；大刀还能分裂攀附成臂刃。身披两件不朽级保命装备。

## 经历 · 殁

- **围观首个节点战 · 暂避**（40卷9–10章）：目睹苏晓水下瞬杀圣域神棍后，判断「能对付、但束缚链没断」，撂下「10 小时后再交手」离开；他察觉到 [尤妮丝](怕死鬼.md) 身后一闪而逝的锁链虚影、追问她（暗示尤妮丝绝非表面那么简单）。
- **五方齐聚 · 摆平各方**（40卷12–13章）：第三节点也出现在 5 区、五方（魔法协会 / 蒸汽联盟 / 治愈神教 / 天启 / 黑塔）+ 圣域神棍齐聚。天罚老谋深算——借「与黑塔有恩怨（非与休西罗）」逼退魔法协会 [面罩男·巴斯](面罩男.md)（否则三方结仇）、又付 12 万黑晶买通蒸汽联盟 [铁颅·基普林](铁颅·基普林.md)，为单挑苏晓扫清干扰。
- **决战苏晓 · 殁**（40卷13章）：与苏晓这场「锋利 vs 狂暴」的刀术宗师对决——苏晓刺穿他的巨大刀芒、以刃道刀·血刃 / 绝幽绕后、一脚直踹削去一块头骨（靠不朽级装备保命）；他抖出绝技「地牙」几百道刀芒，仍被苏晓贯穿后心、又一刀从眼窝没入头颅。他**靠刀术强者的意志力硬撑不死**，只为等队友 [尤妮丝](怕死鬼.md) 偷布出第三节点坐标——见坐标成功，才含笑留下「**天启，也有强者**」而亡（苏晓得 5 功勋 + 37 猩红之勋 + [强者残辉](../../力量体系/装备道具/index.md)×3）。他至死不知自己是被尤妮丝当弃子送死。

## 关系

- [苏晓](../苏晓.md)（斩首的夜）：惺惺相惜的刀术宗师对手、斩其于夕雾街。
- [尤妮丝](怕死鬼.md)（怕死鬼）：同组天启队友、实则把他当弃子送死以独吞坐标奖励。
- [面罩男·巴斯](面罩男.md) / [铁颅·基普林](铁颅·基普林.md)：决战前被他分别逼退 / 买通的两方势力。
