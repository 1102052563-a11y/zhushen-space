# 兔糖（天启乐园契约者 · 软萌小奶妈 · 共生愿·回逆）

[魔灵星](../../世界/任务世界/魔灵星-世界争夺战.md) 世界争夺战里**天启乐园**方的一名软萌小奶妈——[莫雷](../原始大陆/莫雷.md) 的临时队友，总喊莫雷「大佬」（第40卷29章）。她与莫雷都是树之竞技场前五十内的熟人（在世界联络平台上「仰望莫雷大佬」）。

## 能力 · 共生愿·回逆

- **共生愿·回逆**：一种范围救援 / 撤离能力——双手间凝出一个大泡泡、把范围内的人像装进袋子一样一起扯出来。太阳圣殿一层的液态火焰机关烧起时，她站在正门外用此能力**把冲进一层的 150 名联军扯出来 126 人**（其余被烧死），是联军的关键奶妈 / 救场手。
- **监测生命体征**：莫雷让她「时刻留意我的生命体征」——她负责在混乱的圣殿攻防里给队友兜底。

## 关系

- [莫雷](../原始大陆/莫雷.md)：她的临时队长（她的「莫雷大佬」）。
- [苏晓](../苏晓.md)（白夜）：太阳圣殿防御战的敌方。
