# 学者·里曼斯（荒村的干瘪学者 · 被太阳降智 · 殁）

[烈阳圣地·奇利亚德](../../世界/任务世界/魔灵星-世界争夺战.md) 最外围**荒村**里、一栋依城墙而建的大石屋中，一名全身裹满风化布条、干瘪臃肿的老者——**荒村的学者里曼斯**。他早已被那颗「太阳」灼照到**思维固化、降智**（说话干哑断续、意识不清），却也因此无视了苏晓 −1 魅力的负面影响（第40卷51章）。

## 经历 · 殁

- **透露暗门 · 求解脱**：他误以为 [苏晓](../苏晓.md) 一行喝了「治愈神教的秘药」（太阳药剂）、感慨「他们终于成功了」。他以「告诉你一个秘密」换「把我丢到外面」——指出大石屋左侧墙壁内有**暗门**（通向地下、藏着 [嘟嘟咕咕](嘟嘟咕咕.md)），叮嘱丢他时「不要碰到我、会被灼伤、连椅子一同丢掉」。
- **化为灰烬**：巴哈把他连躺椅抬到阳光下——他身上布条瞬间燃尽、留下枯瘪的脸，留下一句「希望你们能活下去，太阳的追逐者们」便化为灰烬。荒村人「不后悔追随王、只是渴望太阳」，这一切是代价——他不需要同情、也不需要挽救。（他留下的木棍成了苏晓敲开暗门的钥匙。）

## 关系

- [苏晓](../苏晓.md)（白夜）：以暗门秘密换解脱的荒村学者（其木棍成了苏晓开暗门 / 唤醒嘟嘟咕咕的工具）。
