# 尤妮丝（怕死鬼真身 · 天启乐园八阶战斗天使 · 蜘蛛系母猎手）

[魔灵星](../../世界/任务世界/魔灵星-世界争夺战.md) 世界争夺战里**天启乐园**方的八阶契约者——真名**尤妮丝**、是一名**战斗天使**。她一路伪装成怯懦无能的「**胆小少女 / 怕死鬼**」（被队友 [天罚](天罚.md) 直呼怕死鬼），实则是**深藏不露、极耐心谨慎的蜘蛛系母猎手**、本批苏晓遇到的最难缠对手。

## 伪装 · 真身揭露

- **一路装怂**（40卷9–13章）：她表面上一遇危险就腿软、哭求逃跑 / 放弃任务去挖矿，故意让 [天罚](天罚.md) 独自去争第三节点送死（「一个人分坐标奖励更好」）。破绽藏得极深——被石板绊倒时身后闪过几十根「世界链」虚影、撤退时嘴角翘起哼歌。
- **真身 = 战斗天使尤妮丝**（40卷14–15章）：当有人布出 3 个临时坐标、坐标奖励达极限后，她如陨石般砸落现身——粉色短发、手背 / 小臂 / 额头浮现精雕细琢的**浅红色蜘蛛眼**、气息骤变、震碎周身「世界链」。她对苏晓眨眼「忍到现在，好辛苦」。

## 能力 · 蜘蛛系

- **蛛卵种植（核心杀招）**：能悄无声息地把「蛛卵」漂流植入敌人体内（有 1~2 小时独立期、可用高品阶道具 / 药剂察觉清除、逾期即与宿主共生），之后随时操控蛛卵卵化成小蜘蛛**从口鼻眼耳爬出、把宿主吸成皮包骨**——[圣域神棍](圣域神棍.md) 白天与她照面时已被种卵、想算计她时反被瞬间吸干（殁）。风险：蛛卵漂流途中若被摧毁，她本人精神力遭不可逆重创（「像烧红烙铁插进脑子」）——故对刀术宗师苏晓（近距离感知敏锐、蛛卵漂到十几米就被斩）她只能亲自现身。
- **蜘蛛甲足 + 猛毒**：凭空刺出近三米长、金属质感的蜘蛛甲足（速度极快、穿透力强、可穿透宿主与目标）；甲足带**猛毒**（每秒 59 点、持续 12 小时、疑可叠加）——苏晓靠青钢影压制 + [天然蜂胶](../../力量体系/消耗品材料.md) 才解掉。
- **蜘蛛丝**：周身遍布屏蔽感知、极锋利的蛛丝（巴哈贴身偷袭被割得飙血）。
- **不朽级保命底牌**：血盆大口空间转移（评分 1498 的不朽级保命道具）——被苏晓的「普通阿波罗」诈成「[烈阳之怒·阿波罗](../../力量体系/消耗品材料.md)」吓得消耗掉一枚（心在滴血）。

## 经历

- **偷走第三节点**（40卷13章）：趁苏晓与天罚死战、以幻术 / 精神系影响布布汪（虚空之树的节点提示对幻术 / 精神系不公平），**在节点里偷布临时坐标**、抢走天罚拼死等来的第三节点。
- **蛛卵杀圣域神棍 + 与苏晓对峙**（40卷14–17章）：现身秒杀 [圣域神棍](圣域神棍.md)；与苏晓爆发一场极凶险的攻防——苏晓用普通阿波罗诈她消耗不朽级保命道具、双方均全身而退。此后苏晓反客为主，让布布汪贴身尾随、四处张贴她的照片令她疑神疑鬼（「狩猎游戏」）。**雷引**道具令她被雷反复劈中暴露位置。
- **黑潮夺第五节点**（40卷18章）：趁 5 区落柱潮爆发、众人疯抢原始魂能之机，悄然占据第五节点。**节点态势 → 轮回 3 / 天启 2 / 空 1**。

## 关系

- [天罚](天罚.md)：同组天启队友、被她当弃子送死（她好独吞坐标奖励）。
- [苏晓](../苏晓.md)（斩首的夜）：最难缠的对手、双方猫鼠互猎中。
- [圣域神棍](圣域神棍.md)：被她蛛卵吸干的死者。
