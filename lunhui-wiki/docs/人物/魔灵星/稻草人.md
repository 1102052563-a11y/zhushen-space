# 稻草人（轮回乐园契约者 · 太阳圣殿三层守卫 · 灵魂的舞者）

[魔灵星](../../世界/任务世界/魔灵星-世界争夺战.md) 世界争夺战里镇守**太阳圣殿三层**（太阳光束杀戮区）的**轮回乐园**方契约者——一个瘦到病态、宛如稻草人般的女子，说话是机械电子音、动作怪异。自称「**灵魂的舞者**」。与「黑魔」[小胖子](小胖子·黑魔.md) 并称让天启死亡联军惊悸的两大存在（第40卷29–33章）。

## 能力 · 灵魂之舞 / 灵魂之声

- **索求舞伴**：她会向敌人索要「舞伴」——**要求得到满足（有人陪她跳一支舞），她就只能靠「诱导」坑敌；若得不到满足（找不到舞伴），则会有更可怕的「惊喜」**。她一舞选中了死亡乐园 [海伊左](海伊左.md)（[怒犬](怒犬.md) 把海伊左甩过去陪跳），跳完落泪「你居然真的相信我」。
- **灵魂之声**：跳舞时她能以「灵魂之声」向舞伴单独传递信息（旁人听不见）——她给海伊左传的是**假情报**（「他们都在五层、四层有杀戮机关、尽快冲过四层」），诱使联军冲进太阳光束区送死。
- **假投敌 + 杀戮之舞**：她当着敌人的面「投敌」——跑到四层砸碎机关晶体、被己方契约者「轰成粉碎」（做戏），实则激活三层太阳光束；随后破碎的身体又聚合、翩翩起「杀戮之舞」，把冲进三层的联军**成批切成碳化尸体**。「难得找到一个愿和她共舞的人、她和我们一样想做什么就做什么」。

## 经历

- **三层诱杀**（29–33章）：靠灵魂之声的连环诈术 + 杀戮之舞，在太阳圣殿三层大量收割冲上来的天启死亡联军——是这座「恐怖片」式防御要塞里的关键一环。怒犬「卖队友」把海伊左甩去陪舞，反而算是救了海伊左和莫雷的命（满足了稻草人、她便只用诱导而非「惊喜」）。

## 关系

- [苏晓](../苏晓.md)（白夜）：同为轮回乐园己方（守三层杀戮区）。
- [海伊左](海伊左.md)：被她选为「舞伴」、传假情报诱杀其联军的死亡乐园智囊。
- [小胖子·黑魔](小胖子·黑魔.md)：与她并称让联军惊悸的两大己方存在。
