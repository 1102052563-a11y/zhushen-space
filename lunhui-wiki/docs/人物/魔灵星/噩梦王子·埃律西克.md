# 噩梦王子·埃律西克（画廊管理者 · 斗兽棋收藏家游魂 · 殁）

[烈阳圣地·奇利亚德](../../世界/任务世界/魔灵星-世界争夺战.md) 王城**画廊**的管理者——身穿冈卢王族正装、声音温和磁性的男人，自称被困此地的**游魂**（实则有一具赢来的、生命力很强的肉体）。他是个**什么都收藏的收藏家**：宝物、珍奇异兽、乃至强者——画廊里 219 幅画作，其实都是**被封进去的活人**（在此对他用武力，这 219 幅画作会一齐成为敌人）（第40卷62–64章）。

## 斗兽棋 · 十赌十诈

- **死灵金币 · 只赢不还**：他靠**斗兽棋**赌局吸取对手的一切——赌注用「死灵金币」（画廊吞噬珍宝凝成·传说级·每枚约值 76200 乐园币）；一旦参与、在一方输光前不可离开。他连椅子都是能让人放松警惕的「罗米克王国宝物」，各种做手脚让自己占优。他连自己的**游魂**都能压上赌桌。
- **反被苏晓算计**：[苏晓](../苏晓.md) 深知「十赌十诈」，用**虚空的 T559-A 号微型终端 ×3 + 纳米具像膜**（眼球隐形眼镜）作弊——前三局故意输、诱噩梦王子上钩，之后连胜把死灵金币 + 灵魂结晶尽数赢回、还额外赢来上万死灵金币。最后一局苏晓逼他**压上自己的游魂**，32 秒破局——噩梦王子游魂输光、当场死亡（得 10.2% 世界之源、12728 死灵金币转 6465 魂币）。「赌徒的下场很凄惨，他连自己的魂都输掉了」。

## 母亲 = 画廊真·超级 boss

画廊里那名一直坐高椅、以 2 米长金色发辫当画笔、把痛苦画像修改成安详的**作画少女，其实是噩梦王子的母亲**——才是画廊真正的超级大 boss，**只比巅峰时期的永生者 [席兀](席兀.md) 弱一筹**（在画廊内交手可能更强）。她与噩梦王子都没接受「太阳」的力量；儿子被苏晓「杀」死，她只淡淡看一眼便继续作画、不予理会。

## 关系

- [苏晓](../苏晓.md)（白夜）：用虚空微型终端作弊反杀他、逼他压上游魂的对手（苏晓由此从他口中问出罗格什藏身「环水祭坛」）。
- **噩梦王子之母**（画廊真 boss·作画少女·比巅峰席兀弱一筹）：她的儿子。
- [罗格什](罗格什.md)：噩梦王子供出其藏身处（环水祭坛·罗格什当年败于席兀之地）。
