# 鼬鼠商人·帕格·波波图加

> ⚠ **一名形象酷似大号鼬鼠、背着大包裹的行商，继承父辈的手艺、制作各类机械商品**——⚠ **卖给噩梦庇护所的梦魇一族、或余灰城地城深处的王族，换取干净的食物与淡水**;⚠ **⚠ 他记忆力有问题**:⚠ **只能记住不超过一小时内发生的事，每过一小时记忆就模糊一次**（⚠ 只有知识记忆会被永久记住·⚠ 他随身带笔记、自动记录他的「感受经历」，翻开看到「白夜朋友」几个字就坚定地点头）。

## 💔 孤独与善良

⚠ **鼬鼠族对他态度冷淡甚至恶意，他早忘了因为什么才遭厌恶**——⚠ **他一般遇不到活人，有往来的客人也不屑于骗他、甚至不理会他**;⚠ **⚠ 他曾把一只随时想吃掉他的猛兽·古魈当成「唯一的朋友」**（⚠ 每次去 3 号噩梦庇护所前给它带肉·「孤单就像能吞噬掉一切的怪兽，会先从心脏位置吞噬」）——⚠ **古魈被苏晓三箭射死后，他悲痛欲绝，还两度放「七彩斑斓空间屁」把苏晓、死灵妹传送到 Lv.85 的骸骨岛**（⚠ 这一屁把 [死灵妹](死灵妹.md) 送进了苏晓的灵魂锁网陷阱、坑了她整个世界）。

- ⚠ **⚠ 善良到把巴哈整不会**:⚠ **巴哈本想戏弄他、谎称传送走的死灵妹是「我们的朋友、已经死了」——他竟当场信了、瘫坐在地「我把人害死了」**（⚠ 对凶恶之徒巴哈可以更残忍、对无耻之徒能更无耻千百倍、唯独这种单纯善良又容易信人的，巴哈是真的难搞·差点「直接宰了」）。

## 📚 地城王族与大书库

⚠ **他的祖祖辈辈都是为地城王族效力的能工巧匠**——⚠ **王族没落后，付给他的报酬是一堆「他一个字都看不懂」的古籍**（⚠ 包括第一纪元「铭刻学」、[神秘之眼](../力量体系/装备道具.md) 的仿造体「炼金之眼」等）;⚠ **⚠ 正是他引荐苏晓见了地城王族长者、买下整座「大书库」**（⚠ 七八成是第一纪元羽族的传承古籍·苏晓借此得【大师级封印学】、把封印学提升到 Lv.94·是苏晓「世界咆哮计划」翻盘的关键）。

## 🏡 结局

⚠ **苏晓问他「希望在没有诅咒蔓延的区域生活吗」，他很认真地点头「很希望」**——⚠ **随后余灰城五大势力〔阵界师公会/慈悲圣殿/狩猎教会/黑暗神教/朝圣者〕与地城王族，全体经界级传送阵迁往被苏晓清空的领主大陆**（⚠ 帕格作为其中一员、终于能离开这死透又畸变的诅咒之地）;⚠ **⚠ 而他身上那股「像诅咒却没侵蚀力」的莫名气息，也引起了苏晓的注意。**

## 🔗 关联

- ⚠ **⚠ 他分给苏晓的一颗深渊软糖/灵魂糖果，让这个从没吃过甜食的鼬鼠族「心情美好地绽放、愣在那有点死机」**——⚠ **他是这场冰冷争夺战里，少有的一抹温暖底色。**
