# 玛奇

[全职猎人世界](../世界/任务世界/全职猎人.md) **[幻影旅团](../势力/index.md)（蜘蛛）成员**——松散辫子的女团员，**变化系念能力者**。旅团辅助人员，但能力实用且关键。

## 能力

- **念线**：把念能量化为细长的丝线，可用于**攻击**（缠绕分尸）或**治疗**——能将血管、骨头、神经、肌肉 **100% 完美缝合**（旅团的「医疗兵」，团员负伤多靠她接续）。

## 历程 · 山谷之战 · 撤退（十卷38–43章）

随旅团在独眼巨怪山谷围困黑帮。战中以念线**分尸阴兽病犬**、又切喉重创阴兽豪猪。[窝金](窝金.md)、[信长](信长.md)、[侠客](侠客.md) 接连战死后，她与 [小滴](小滴.md) 面对苏晓的威胁果断**撤离**——小滴愿留下断后赴死、催她先走（「你的能力更重要、要把战况报给团长」）。她因「苏晓说能保豪猪一命 = 对方有治疗能力、追不得」而决意撤退，最终二人一同逃离（苏晓状态太差未追）。是这场战斗中**少数幸存的旅团成员**，将带回旅团损失四员战力的噩耗。

## 关系
- **[幻影旅团](../势力/index.md)**：变化系辅助 / 医疗成员（能力对旅团至关重要）。
- **[小滴](小滴.md)**：一同撤离的同伴。
- **[苏晓](苏晓.md)**：敌对——逃过其手。
