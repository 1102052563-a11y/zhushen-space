# 暗鸦（初代女巫 · 暗鸦纪元开创者）

[暗鸦星](../世界/任务世界/暗鸦星-暗鸦纪元.md) 的**初代女巫**、史上最强女巫（没有之一）——一名**被不幸逼疯**的麻衣女人，在雨夜里跪倒在草原那块**起源石**（紫色晶体）上，体内分裂出的「另一个自我」吸收了起源石之力而觉醒。她抬手便让静止的雨点爆开、把黑夜白昼随意翻转、凭意识汇聚出**百万只乌鸦**（精英个体化作她的黑紫长袍），自取名「暗鸦」。

## 67 天征服一颗星球

帝国军队尽数葬身于她的乌鸦群下、年轻的王被斩首——她**用王的头盖骨饮酒**、自封这片大陆的新女王。她是「天灾、力量的象征、圣之女、女巫」——却**只存活了 67 天**，便如烟花般爆开消逝，只留下一块石板。她死后，自称女巫的女性陆续出现（第二位是汽之女巫珍娜·金斯利），后世女巫尊她为**神**、四处寻找她觉醒所凭的紫色晶体「**起源石**」。**暗鸦纪元**由她而始；「暗鸦」从此既是她的名字、也是女巫中**最高荣誉的称号**（仅最强女巫可得）。

## 关系
- **后世女巫**：尊她为神、寻她的起源石（女巫的转化 / 化身之力皆源于起源石）。
- **[苏晓](苏晓.md)**：进入暗鸦星时通过世界简介「亲历」了她的觉醒（仅旁观）。
