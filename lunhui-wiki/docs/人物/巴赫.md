# 巴赫

现实世界一伙**「外来者」（国外偷渡入境的契约者）三人组的头目**——白人、壮硕如北极熊、一双满是血丝的眼睛。三人在国外犯罪后逃出境，组了个**类佣兵团**接「脏活」换钱（契约者中难得的「为钱不择手段」之辈）。

- **闯祸**：三人偷渡入境后住进五星酒店、与当地 [清道夫](魏冬.md) 签约「五天内出境」，却因同伙 [拉哈尔](拉哈尔.md) **强奸酒店女服务员**激化矛盾、与清道夫开战——一阶维安契约者根本不是三名二阶的对手，[魏东](魏冬.md) 最信任的手下也死了。
- **依赖装备 / 主动技能**：他选「增属性」装备（现实力量 45·乐园内带装备 56），这正是其死穴——**现实世界禁用主动技能、装备被强收进储蓄空间**，他只剩约「一阶较强」的肉身。
- **被斩杀（八卷83章）**：[苏晓](苏晓.md)（堆被动技能·现实战力极强·刀芒非主动技能可用）破门来杀——巴赫弃同伴逃跑，以尼泊尔军刀格挡被刀术大师 Lv.21 的唐红险些斩断，角力落败时滚出拉了环的手雷想同归于尽，被苏晓一脚踹飞、踩断肋骨、临死喊「他们给你什么好处、我出十倍」无果，**被斩首**。
