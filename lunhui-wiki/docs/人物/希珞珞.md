# 希珞珞

**希珞珞** 是 [轮回乐园](../总览/乐园.md) 的一名**职工者**，[苏晓](苏晓.md) 回归乐园后结识的第一人（第24–25章）。

## 简介
白发蓝眼的小萝莉外貌（曾假扮「康纳」卖萌招揽客人），性格活泼机敏。职工者身份——经历过两次衍生世界，自认战斗力低微、全靠出钱请「法爷」保护（第24–25章）。

## 与苏晓的交集
收苏晓 50 乐园币，在冷饮店「猫与狗」以一块乐园公证的透明平板，为其讲解轮回乐园的运作（职工者、契约者周期、安全规则、隐藏相貌、设施、死亡率等）；并提醒他衍生世界中勿招惹「法爷」——殊不知苏晓正是法师天敌 [灭法之影](../力量体系/技能.md)（第25章）。

## 关系
- [苏晓](苏晓.md)——收费为其讲解轮回乐园；后亦同入东京喰种世界。
