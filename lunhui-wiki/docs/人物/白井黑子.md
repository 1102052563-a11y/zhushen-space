# 白井黑子（空间系·风纪委员）

[学园都市](../世界/任务世界/魔法禁书目录.md) 的 **Lv.4 大能力者**（空间系）、**风纪委员**（管校园治安·常越权到校外维持治安）——国中生、双马尾、对超能者排名第 3 的「电磁炮」御坂美琴有近乎变态的爱意（视所有男性为「没进化干净的类人猿」）。**天性善良、从未杀过人**、甚至不愿无故伤人。

- **空间能力（需皮肤接触）**：靠接触把人 / 物**空间瞬移**——每次最远 80 米、可携重量上限 130.7 公斤、**瞬移后会停顿片刻**。狠起来能把金属棒传送进敌人膝盖骨（对她已是极限狠手）。
- **被「借刀杀人」（十三卷8章）**：[结标淡希](结标淡希.md) 匿名打电话、骗她来「救人」（实为送死）——她用空间能力救走重伤的 [道具](../势力/道具.md) 成员绢旗最爱，却撞上 [苏晓](苏晓.md)。一照面她就知道双方不是一个层次（自己「制服敌人」、对方「杀敌清障」满眼杀意），果断带绢旗最爱**空间瞬移逃命**。苏晓没全力追杀她（杀暗部的麦野无妨、但杀「阳光下」的风纪委员等于正面挑衅某人·且他要防真正危险的 [结标淡希](结标淡希.md)），任她「比泥鳅更滑溜」地溜走。

## 关系
- **绢旗最爱 / [道具](../势力/道具.md)**：被她空间救走的暗部成员（其实素不相识）。
- **[结标淡希](结标淡希.md)**：同为 Lv.4 空间系（匿名骗她来当「刀」借刀杀苏晓）。
- **[苏晓](苏晓.md)**：放她逃走的危险男人（她直觉不可与之为敌）。
