# 三合体（厄塔斯 / 违规者 / 小橘）

[苏晓](苏晓.md) 在 [全职猎人·蚁王篇](../世界/任务世界/全职猎人-蚁王篇.md) 的**猎杀目标**——「**史上最奇葩的违规者**」：一具身高三米五、浅黑肌肉、独角利爪的强壮身体里，**塞了三个意识**：

- **厄塔斯**（虚空生物·占意识 5/3 主导·居头部）：**咔伽族**——虚空的**战斗种族**，天赋是坚硬身体 + **「静止时间」**（实为用特殊能量把周围几十米空气加大阻力、瞬间变「固态」·2~3 天一次、每次约 1 秒·施术者自己能在固态空气中行动）。它本是族中天才（约五阶）、成年后该入黑渊浅层历练，却被「空间爆震漩涡」卷进衍生世界。雌雄同体、爱吃生肉。
- **违规者**（男·占右手 5/1·右手心长一张嘴·爱喝茶）：曾被猎杀者追杀过、险死、自此主张「即时享乐」（一度想强暴小橘）。
- **小橘**（职工者妹子·占左手 5/1·用意识震动空气发声·爱甜食·软妹）：拥有「第三人称视角」能力、名义上是身体最高层、实为最底层（常被另两个意识欺负）。

三意识被时空之力强行挤进虚空生物那具（因漩涡 + 排斥 + 轮回乐园 2% 处决而残破）的身体——虚空生物**靠违规者 + 小橘的精神波动才免于被轮回乐园强行处决**，故不能动那两个意识。日常「一手生肉、一手冰淇淋、面前一杯热茶」，奇葩到辣眼睛。三意识争夺身体导致它时而冲、时而停。

## 殒落 · 战斗种族的求死（十六卷2–4章）

苏晓追上时，厄塔斯接管身体死战——斩龙闪金色断其利爪、**龙影闪瞬移到背后斩颈**（差点斩首），它靠「静止时间」勉强续命；苏晓界断线 + **木之灵攻击形态徒手力量 ×3 一记侧踢**把它踢穿数栋楼、踢烂小腹。它身负不可逆重伤仍掰断肋骨再战（战斗种族「为战斗而生、愿为战斗而死」·能与传说中的灭法之影一战、战死也甘愿），终被苏晓上百道刀芒斩成碎肉。**虚空生物 + 违规者意识被清理、职工者小橘被传回乐园清记忆存活**——苏晓借此把猎杀任务做成 **S+**（5 分 36 秒·开局即完成·全靠龙影闪）。苏晓欣赏这「耿直」的对手、不因立场对立而厌恶。
