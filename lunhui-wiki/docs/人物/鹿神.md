# 鹿神

> ⚠ **星界「平头哥」——生死看淡、不服就干**;⚠ **⚠ 外形**:⚠ **身高五米却有着五十米级的气场·全身淡金色长毛〔手肘等处略有流苏感〕·下巴是随意修剪的胡须·脸戴灰白色骨质面具·头生麋鹿角·身披破旧兽皮·背后是根三米多长、鹅蛋粗的金属黑棍**（⚠ 那是深渊武器「梼守」——⚠ **九成九的深渊武器都有噬主情况，唯独这把非常愿意协助鹿神战斗**·握柄两侧会蔓出黑色触须缠在他小臂上）;⚠ **⚠ 金色瞳孔的由来:长期吸收初始状态的「黄金之力」**——⚠ **已知强者中只有他敢这么做**（⚠ 黄金技能点其实是弱化后的「初始状态黄金之力」，并非乐园故意削弱，而是**谁加点谁死**、不弱化就称不上奖励；未经弱化的初始黄金之力虽危险，却拥有持续突破上限的威能）。

## ⚔ 与苏晓的第一战（54卷·陨灭星·艾索戈壁）

⚠ **他抬手食指指向领主列车，意思很简单:交出眼之公主**——⚠ **双方无需多言，血海气场与金色神威轰然相撞**（⚠ 血海更胜一筹——那是苏晓的「至强·魅力」所带来的气息阶位提升，提升气息阶位的手段在星界很少见）;⚠ **⚠ 两个近战系至强巅峰的对轰**:⚠ **「极刃·白夜」对「贯日」，一击就让周边上千公里的空间宛如防爆玻璃般密布碎痕，范围内三十多个邪神信徒聚集处、至少十几万名信徒同一瞬间炸成血雾**（⚠ 苏晓被轰退一大步、喷出一大口鲜血、五脏六腑仿佛开裂——但**鹿神才是真正意外的那个**:以往同阶敌人敢正面接他一棍的九成会被轰飞，能扛住的除了虚空的刀魔就只有眼前这灭法者）。

- ⚠ **⚠ 交手细节**:⚠ **「完美格挡」把鹿神的重击悍然弹开、让其持棍右臂与小半个身体瞬间麻木并被迫退半步** · ⚠ **泯灭血枪刺穿其胸膛却被黄金力量物质化而无法引爆** · ⚠ **觉醒后的青钢影所造成的「强烈疼痛」让鹿神眼角一阵颤抖〔能不狂吼一声就抗住的都是狠人〕** · ⚠ **短短几分钟对轰几千回合** · ⚠ **「极刃·青鬼」在其左眼留下斜跨的斩痕**。
- ⚠ **⚠ 徒手搓太阳**:⚠ **他一棍撼地，让几百公里内的大地喷出柱状岩浆、让苍穹遍布千米长的裂痕淌出「世界之力」，再把地底灼热与世界之力一并汇聚——全程不到1.5秒**——⚠ **手中托起的不是大火球，而是一颗真真正正的小型太阳**（⚠ 苏晓以「无我之境·极」一刀将其斩开、两半飞过身侧时快速消散〔两股极限力量相抵的噬灭现象〕·⚠ 这一刀差点把作为至强巅峰的苏晓精气神抽干，也让他把「灵魂强度提升到3000点」提上日程）。
- ⚠ **⚠ 收场**:⚠ **苏晓以「极境·星界」停固后一刀斩过其脖颈，鹿神生命值骤降一大截但当即以自愈力复原**——⚠ **「再生」与「自愈力」不同:再生出的先是烂肉再重塑，自愈力则与生俱来、无风险无副作用、体力足够就能一直被动持续；而自愈力强到胜过再生的，星界独此一份**;⚠ **⚠ 苏晓的战斗直觉告诉他:趁机杀死鹿神也不会让对方彻底死去，反会引发更强的未知危险**——⚠ **于是改为一脚直踹把人送走**（⚠ 鹿神飞出十几秒后从高空坠落，犁出十几公里长的沟渠；他起身后环顾四周确认无人，才单手按胸「哇」的喷出一大口鲜血、片刻后气息平稳·⚠ 没有追击是因为**他在第二纪元末期体验过领主列车有多快**）。
- ⚠ **⚠ 苏晓的判定**:⚠ **「以现在的自己，如果是今天分生死，必不是鹿神的对手」**——⚠ **但这场交锋让他的至强巅峰战斗经验飙升**（⚠ 简单比喻:和其他至强巅峰打到分生死约提升1000，和鹿神这场没分胜负就飙升8000+·⚠ 这也直接催生了苏晓在陨灭星的核心战略:**吞至强被动提升→找鹿神打一场消化→回兽镇再吞**，让鹿神每次见面都发现对手比上次更强）。

## 🗺 「极境系」的代表

⚠ **至强巅峰分两大体系:「领域系」〔约九成〕与「极境系」〔约一成〕**——⚠ **两者无强弱之分，只有是否适合自身**;⚠ **⚠ 有一点是客观事实:「极境系」是否更能打因参考基数太少而无法考证，但「极境系」是真的愿意和其他人打**——⚠ **比如鹿神，比如巅峰期的格林·吉莉安，再比如青年期的席曼·阿奇德**（⚠ 至强领域对至强极境胜算基本五五开:领域带来无与伦比的压制力，极境能豁免六成左右的领域效果，加上极境自身的单体杀伤威慑或增益，双方持平·⚠ 苏晓正是参考了这一点选择「极境系」，再用完全解封的【魂·王之加冕】把「领域」也一并补齐——**全都要**）。
