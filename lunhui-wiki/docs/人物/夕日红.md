# 夕日红

[火影忍者世界](../世界/任务世界/火影忍者.md) 木叶上忍——黑发御姐、赤红双眸的**幻术型忍者**。九卷追击 [苏晓](苏晓.md) 时率小队（[油女志乃](油女志乃.md)、[犬冢牙](犬冢牙.md)、[日向雏田](日向雏田.md)）出动。

- **追击与战死（九卷66–68章）**：因 [日向雏田](日向雏田.md) 白眼追踪，小队咬住苏晓四小时。她深知小队实力悬殊（卡卡西的惨状在前）、不敢贸然接触，却仍被苏晓剃瞬移找上。她数次施幻术（魔幻·树缚杀）皆被苏晓的高智 + 青钢影破解（幻术回馈微弱）。在猪鹿蝶等援军赶到的混战中，她想以最强幻术「**落奈之术**」（漫天花瓣）困住苏晓为众人争取机会——却在喊出「趁现在」的瞬间被苏晓的**界断线一丝斩首**（苏晓的精神抗性 + 智力 50 令幻术失效）。
