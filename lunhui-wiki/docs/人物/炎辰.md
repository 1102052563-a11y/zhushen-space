# 炎辰（炎魔枪骑士）

二阶契约者、**「铭门冒险团」团长**——[维斯星球·多亚古大陆](../世界/任务世界/维斯星球-多亚古大陆.md) 世界争夺战中**轮回乐园方唯一的大型冒险团**首领。即星空中转站用火元素摆「赤炎中指」、又与苏晓争过 [圣橡树枝干](../力量体系/消耗品材料.md) 的那名**火系法爷**。用真名「炎辰」（不必用假名·家人都已死）。

- **真身 = 炎魔枪骑士（不是法师、也不是法坦·十二卷68章）**：他对外的「火系法师 / 法坦」全是迷惑——真实职业是极罕见的「**炎魔枪骑士**」（稀有度仅次于灭法之影·主属性体力 / 智力）。靠几种被动支撑近战：「法杖形态转换 Lv.26」（金色法杖可拧成两米火系魔枪）、「长兵器精通 Lv.30」（+92% 长兵器攻击）、「武器属性附魔 Lv.10」（武器附 100 火伤 + 可叠灼烧）、「炎魔枪骑士 Lv.32」（乘骑时免疫 10% 任意伤害·力 / 敏 / 智 / 体可任意转换）。入乘骑状态便把智力转成力量（力达 80）、化身近战推土机。坐骑是召出的**焰虎「皮亚克」**。所谓火球火箭法术只是障眼法——「不会近战的法师不是合格的法师」。
- **伪装的「法坦」**：对外智力 80 / 体力 71、持金色法杖 + **三件暗紫色法系套装**，实力在二阶能排前几。
- **身世与「德国骨科」**：孤儿院长大、继承一大笔财产，仇人亲属为夺财绑架虐待他——他**亲手杀光这些「亲人」、只留亲妹妹**（杀时只觉畅快·视他们为陌生人 / 仇人）。妹妹 **茜茜**（双马尾少女·一把血色战镰·杀人狂·连自己人都可能砍）是他唯一的亲人、也是其**心腹兼床伴**（妹控）。
- **正直部下**：手下一名 OL 装魅力美女（魅力特长·三观正常的「正常人」）对老大的妹控与茜茜的疯狂深感绝望。

## 世界争夺战 · 守核（十二卷58–59章）

铭门冒险团约 186 人是本战场轮回乐园方主力。开战前他在黄昏宫殿七层与各团长**商定「死守战略」**（收缩防线守黄昏宫殿 + 第二区 + 三石桥·不成功便成仁·没退路）。他赏识苏晓（白夜）拿出的古遗迹详图、邀其共商被婉拒（苏晓不擅指挥）。得知苏晓要对付的「难缠敌人」正是曾差点阴死他（锁骨留疤）的机械召唤师 [兰爵](兰爵.md)，便答应在战争频道为苏晓通报兰爵踪迹——还半真半假地嘀咕「别死了、我妹很漂亮」（怕死后妹妹跟别人、被「死后绿」）。

## 大乱斗 · 揭真身秒凛冬（十二卷67–68章）

契约者大乱斗中他被 [希女王](希女王.md) 专门派来的冰系刺客 **[凛冬](凛冬.md)**（身高 1 米 6、却有「6 米 1 的霸气」·与「炎辰」代号注定是对手）缠上、火焰被冰系克制一时落于下风。炎辰索性**脱袍露出满身魔法回路、亮出真身**——把法杖拧成火系魔枪、召焰虎「皮亚克」、智力转力量（力 80），骑虎如推土机般冲锋，**5 分钟把凛冬轰成下半身消失、一枪贯喉**（「只会偷鸡摸狗的刺客、对付远程法师你或许擅长，但老子不是法师」）。战罢他向 [希](希女王.md) 的方向赶去——「我隐藏这么久、是时候给你个惊喜了，希」。期间他一度被希塞来的「过去记忆」幻觉冲击（面容狰狞、斥「别给我看这些」），旋即压下、判定为幻觉。
