# 炎辰（炎魔枪骑士）

二阶契约者、**「铭门冒险团」团长**——[维斯星球·多亚古大陆](../世界/任务世界/维斯星球-多亚古大陆.md) 世界争夺战中**轮回乐园方唯一的大型冒险团**首领。即星空中转站用火元素摆「赤炎中指」、又与苏晓争过 [圣橡树枝干](../力量体系/消耗品材料.md) 的那名**火系法爷**。用真名「炎辰」（不必用假名·家人都已死）。

- **真身 = 炎魔枪骑士（不是法师、也不是法坦·十二卷68章）**：他对外的「火系法师 / 法坦」全是迷惑——真实职业是极罕见的「**炎魔枪骑士**」（稀有度仅次于灭法之影·主属性体力 / 智力）。靠几种被动支撑近战：「法杖形态转换 Lv.26」（金色法杖可拧成两米火系魔枪）、「长兵器精通 Lv.30」（+92% 长兵器攻击）、「武器属性附魔 Lv.10」（武器附 100 火伤 + 可叠灼烧）、「炎魔枪骑士 Lv.32」（乘骑时免疫 10% 任意伤害·力 / 敏 / 智 / 体可任意转换）。入乘骑状态便把智力转成力量（力达 80）、化身近战推土机。坐骑是召出的**焰虎「皮亚克」**。所谓火球火箭法术只是障眼法——「不会近战的法师不是合格的法师」。
- **伪装的「法坦」**：对外智力 80 / 体力 71、持金色法杖 + **三件暗紫色法系套装**，实力在二阶能排前几。
- **「妹控」表象（对外传言）**：对外人称他**杀光自己亲人、强行囚禁妹妹的变态妹控**。妹妹 **[茜茜](茜茜.md)**（双马尾少女·一把血色战镰·杀人狂）是他唯一的亲人、随他征战。**真相见下方电疗揭露——他其实并非妹控。**
- **正直部下**：手下一名 OL 装魅力美女（魅力特长·三观正常的「正常人」）对老大的妹控与茜茜的疯狂深感绝望。

## 世界争夺战 · 守核（十二卷58–59章）

铭门冒险团约 186 人是本战场轮回乐园方主力。开战前他在黄昏宫殿七层与各团长**商定「死守战略」**（收缩防线守黄昏宫殿 + 第二区 + 三石桥·不成功便成仁·没退路）。他赏识苏晓（白夜）拿出的古遗迹详图、邀其共商被婉拒（苏晓不擅指挥）。得知苏晓要对付的「难缠敌人」正是曾差点阴死他（锁骨留疤）的机械召唤师 [兰爵](兰爵.md)，便答应在战争频道为苏晓通报兰爵踪迹——还半真半假地嘀咕「别死了、我妹很漂亮」（怕死后妹妹跟别人、被「死后绿」）。

## 大乱斗 · 揭真身秒凛冬（十二卷67–68章）

契约者大乱斗中他被 [希女王](希女王.md) 专门派来的冰系刺客 **[凛冬](凛冬.md)**（身高 1 米 6、却有「6 米 1 的霸气」·与「炎辰」代号注定是对手）缠上、火焰被冰系克制一时落于下风。炎辰索性**脱袍露出满身魔法回路、亮出真身**——把法杖拧成火系魔枪、召焰虎「皮亚克」、智力转力量（力 80），骑虎如推土机般冲锋，**5 分钟把凛冬轰成下半身消失、一枪贯喉**（「只会偷鸡摸狗的刺客、对付远程法师你或许擅长，但老子不是法师」）。战罢他向 [希](希女王.md) 的方向赶去——「我隐藏这么久、是时候给你个惊喜了，希」。期间他一度被希塞来的「过去记忆」幻觉冲击（面容狰狞、斥「别给我看这些」），旋即压下、判定为幻觉。

## vs 希 · 杀她一次反被晶体化（十二卷71–73章）

秒杀凛冬后他奔赴 [希女王](希女王.md)——魔枪的火温超过水晶熔点、能力**克制**希，开战半小时一直压着打、还**杀了希一次**。可他骇然发现希竟**原地复活**（复活时现猫的虚影、状态恢复 80%+），且火属性附魔持续耗法力、自己快撑不住，遂下令**退守三石桥**。希没追、只笑「或许这次见面就是永别」——因为杀死希一次的代价，是被希死亡瞬间**完全水晶化爆开**的水晶溅中：他胸口三片 + 整条左臂 + 一侧肺逐渐晶体化（如跗骨之蛆、不需判定、持续恶化），命悬一线（奶妈**奶糖**判他只剩约 10 小时）。

**托付情报（十二卷73章）**：自知可能撑不过，他把用惨痛代价换来的**希的杀招情报**全盘交给苏晓——「希平常的水晶化只要抗性足够就能避免，**真正可怕的是她死那一瞬完全水晶化并爆开**，碰到就会变成我这样、不需判定、持续恶化」。又交代「我若挺不过，希只能由你（白夜）对付，那些团长不是她对手、幸运女和虫女最多拖住她、灰熊太鲁莽」。临危仍调侃自己枪兵身份「枪兵果然不靠谱、看来我今天要幸运 E 了」（Fate 梗）。

!!! note "头号奶妈「奶糖」"

    炎辰手下的头号治疗师 **奶糖**（婴儿肥少女·现实是实习医生·把治疗技能 + 医术结合）——长相可爱爱卖萌，动起手术刀却是「**女版汉尼拔**」（曾切下炎辰心脏、咽口水好奇「煮着吃还是炒着吃」），苏晓戏称她「杨教授」。她搞不定炎辰的晶体化，只得请来苏晓的「电疗」（用 [青钢影](../力量体系/技能.md) 噬灭炎辰体内致晶体化的能量·高死亡率）。

## 「电疗」揭真相 · 他并非妹控（十二卷74章）

苏晓用 [青钢影](../力量体系/技能.md) 噬灭炎辰体内的「晶能」（[希](希女王.md) 的能力·法力转化的无活性能量、放体外即成水晶）救他——靠**奶糖**续命，炎辰**求生意志惊人地活了下来**（左臂恢复血肉、只余少许晶体化痕迹·过程中失禁）。但青钢影侵入其脑、破坏了一角**封印术**（似火影世界的封印），漏出他真正的记忆，苏晓由此推断出**炎辰其实没有妹控癖好**：

- **真相**：妹妹 [茜茜](茜茜.md) 幼年患精神分裂、裂出**反社会 + 暴力人格**，某日**亲手杀了父母**（父亲临死遗言「帮帮你妹妹、别恨她、这不是她的意愿」）。茜茜原本活泼的人格在弑亲那刻便**死去**，剩下的只是个女疯子。炎辰**花 10 万乐园币把惨痛记忆封印、并虚构出一段「妹控」记忆**——封印后只留一个意念：「别碰妹妹、那很危险、她缺一个人格、要修复才能得到完整的妹妹」。他扮「妹控」，既是封印的保护机制，也因他相信轮回乐园这种奇迹之地或许能**修复茜茜死去的人格**（这是他战斗的动力之一）。苏晓无意挖人隐私，只在意「炎辰不死 = 能挡希 = 是最佳合作者」。

## 「大菠萝计划」· 反派 BOSS×2（十二卷74–77章）

醒来后他与苏晓警觉：希为何不趁他垂死攻石桥？——果然希在等更好的机会。希随后**死 3 次、以自身 + 几十名法系契约者的法力**催出**直径千米的实心水晶球**、把三石桥与黄昏宫殿连同数百契约者一并同化（**灭 403 人**），轮回乐园方只剩 255 人、与天启 783 人形成三倍差距。残存的几乎全是「疯子」——炎辰与苏晓索性退守黄昏宫殿、共谋**「大菠萝计划」**（用苏晓造的太阳神「阿波罗」全灭杀入宫殿的敌人·苏晓的计划「更丧心病狂」故采用之）：放冒牌小丑 / 黑乌鸦潜入天启队伍、空城诱敌逐层深入、二层埋炼金炸弹……一旁的 OL 美女直觉这俩像「两个反派大 BOSS 在密谋邪恶计划」。此时妹妹 [茜茜](茜茜.md) 也随他在四层（他给她梳头·茜茜好奇打量苏晓）。
