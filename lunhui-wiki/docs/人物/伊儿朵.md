# 伊儿朵（死灵暗杀者）

[死灵族](../世界/生物万族/index.md) 的女暗杀者——身高约 1 米 4、灰白皮肤、**翠绿眸**、尖耳、四颗犬牙（除肤色外容貌近人类少女），红披风红兜帽 + 白面具、双手缠灰白绷带各握一把**绿色死灵之力长匕首**。她是「死刑官」[敦克·尼克](敦克·尼克.md) 的**弟子**，奉命潜入 [圣索丁王国](../势力/index.md) 王都为 [弑君大局](../世界/任务世界/希亚大陆-灭法遗迹.md) 灭口（在销金窑斩了二公主的眼线、报数「卡巴可呐〔第 99 个〕/ 呜克拉七七〔你是第 100 个〕」）。

## 实力强、实战弱（十七卷25–26章）

她**实力不弱、战斗经验丰富，但「实战应变」极差**（有高明的师傅、却没久经沙场）——苏晓抛出**磁核光爆弹（闪光弹）**，她不识高科技、第一反应是「观察是否有毒 / 会爆」而非格挡后跃，直接被闪瞎（体力 75·失明约 2 分钟）。苏晓**为生擒她收起斩龙闪**（90% 手段会致死），改用 [木之灵](../力量体系/装备道具.md) 拳 + 炼金炸弹 + [动能战靴](../力量体系/装备道具.md) 侧踢，把她连揍三分钟。她以绿色死灵能量盾（**机理类同青钢影「傲歌」、但粗浅**）硬扛——惊呼「**傲歌不可能是蓝色的！**」，泄露了**死灵族对灭法之影有所了解**；终被苏晓一拳击昏生擒。

## 审讯室 · 意志被消磨（十七卷26–28章）

苏晓不用刑、专攻心理：每隔一小时唤醒、喂巨辣食物、不许睡眠，并**故意喊出她的名字 + 套出「师傅」二字**（她以为死灵内部出了内奸）——苏晓借此拼出师傅 = 死刑官 [敦克·尼克](敦克·尼克.md)。最折磨她的是 [布布汪](布布汪.md)：挂着审讯架钥匙串装蠢、勾她伸手够钥匙（永远差 2~3 公分）、半分钟一次「自由与囚笼」的大起大落，把她意志一点点磨碎（5 小时后她才发现布布汪「智商不低于人类」、是在戏耍她）。

## 招出幕后主使（十七卷33–34章）

师傅 [敦克·尼克](敦克·尼克.md) 战死后，苏晓把他的断臂（虎口十字青疤）抛到她面前——她崩溃、却被苏晓「敦克·尼克还没死、你每废话一句就拆他一处零件」的谎言拿捏。苏晓恩威并施（强调「是我抓你们、不是王族；王族抓必死、我只要幕后主使、招了就放你们走 + 互留把柄」），先给「自由的希望」、再扯脸威胁。她其实不是铁骨铮铮（最怕师傅被肢解送到面前），在苏晓**循序渐进的委婉引导**下（一步步「假设藏在哪个村」），终于招出幕后主使 = 死灵 **[鲁珀](鲁珀.md)**（藏在布瓦村、伪装人类、能力是「伪装」、附详细地址相貌），只求让师傅死得体面些。她被留下（由阿姆看守·死灵来救则就地除掉）。
