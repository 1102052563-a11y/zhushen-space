# 强者争霸战群像（双子 · 殇月 · 尼塔 · 阿雷克 · 蒙德 · 鼠大人）

[强者争霸战](../世界/任务世界/强者争霸战-暮色之岛.md) 中 [苏晓](苏晓.md) 在第七争霸区（及他处）遭遇的各阵营强者。

## 鼠大人（暴鼠 · 接引人 · 中立）

鼠头人身、扛能量霰弹枪的虚空强者——第七争霸区的接引人（虚空之树征召）。[布布汪](布布汪.md) 旧识（虚空角斗场时受轮回乐园委托硬挡奥术永恒星法师贤者 [瑟菲莉娅](瑟菲莉娅.md)、帮苏晓挡住她）。**嗜酒如命**——被布布汪用一瓶海贼世界「玛歌亚加」（凭酒品硬顶到史诗级的酒）换得隐晦情报（「天气太热 / 海」·暗示与海有关的后续）。

## 死亡乐园双子（夺冠热门 · 与苏晓瓜分海岛）

两名容貌全同、剃光头、头布满条形编码、眉心「￡」死亡印记的少年——**死亡乐园六阶最强者、六阶乐园竞技场第三位**、本届夺冠大热门。说话一人接一人、不分你我；老阴哔，与苏晓**瓜分暮色之岛**（左 / 右各清场·箱子各凭本事·不在第一阶段内耗）。把陨灭星世界之子阿雷克当「见面礼」让给苏晓。

## 殇月（羽族 · 装死求生 · 被放过）

黑羽衣、蓝眼影、不苟言笑的女性羽族（非最年轻一代·才有资格参战）——战斗素养高（羽族「艾文艾学院」教育·懂建掩体 / 防大威力远程），却带不动一盘散沙的杂牌队友。被 [M·暴君](../力量体系/装备道具.md) 清场时靠羽衣硬挡一枪、**装死求生**（兄长传授的战场绝技）——苏晓识破却因**羽族 + 恶魔族的庇护**（单羽族身份必死）放她一马（只割一缕发当凭证·贴灰石确认她活着）。死亡逼近时她被「活着真美好」治愈了不苟言笑的性子。

## 尼塔（吟语族 · 伪名「瓦戈」· 诱饵 / 墙头草）

脸有疤的吟语族——**能拟态任何人的模样 / 体型 / 气息**（甚至衣着）。开场伪装成虚空之树接引人「瓦戈」拉满仇恨、骗走奖励 [封之箱](../力量体系/装备道具.md)；被苏晓识破击杀其伴生化身后归还封之箱、求合作当**诱饵**（把 30+ 参战者引到荒草地任 M·暴君清场）。**吟语族**：古老中小种族·因拟态天赋被虚空当「珍稀商品」禁买卖、千年前为自保投靠奥术永恒星（年年纳贡）；处世宗旨「成功率没 20% 不赌命」。

## 阿雷克（陨灭星世界之子 · 殁）

陨灭星（古神统治的超脱·原生世界）的世界之子——**被古神力量侵蚀**（无神源·黑触手 + 胸口蠕虫独眼·变 3 米怪物·古神所赐强铠 + 魂之咆哮 / 死亡领域）；以「能与神灵直面交谈」为荣、早已彻底放弃反抗（那世界的人「双腿」早被打断·无劳伦特式的反抗者）。被苏晓**心·魂·刃**破其最强一拳后斩杀（**掠天惊澜任务 +13% → 96%**·因不在陨灭星收益打折）。其不肯彻底「神灵化」是身为人最后的不屈。

## 其他争霸区群像（一卷开场群英）

- **随缘施法者（第二争霸区 · 学术派法师）**：黑金法袍、慵懒、说话不自信的年轻施法者——**认同灭法之影的理念**（不维持元素平衡虚空早晚完）、跟苏晓「没仇、连面都没见过」、不想掺和千年前的恩怨（「随缘·遇到再说」）；却已用大范围能力杀光第二争霸区所有人（施法者强在范围）。
- **蒙德（恶魔族 · 越挨揍越强的莽夫）**：锯断双犄角 + 寸发的恶魔族——曾冲到羽族后方被老羽族拽着犄角 / 长发毒打四小时（黑历史·故锯角剪发）；**越挨揍越强、濒死最恐怖**。[莉莉姆](莉莉姆.md) 的「冤家」（互相偷袭 + 告状的循环·把莉莉姆打到打石膏、自己被他爹打到坐轮椅）。
- **魔鬼商人（魔鬼族分支 · 镶宝石骷髅头）**：西装 + 镶满黑宝石的骷髅头——魔鬼族战力代表，爱「交易 / 签契约」（摘自己脑袋当筹码·与魔鬼族交易非同小可）；在第七十九争霸区找夜惑女巫「做交易」。
- **夜惑女巫（超脱·原生世界 · 控尸）**：紫裙尖顶帽——来自夜惑女巫统治、排斥虚空与乐园的世界；**杀谁就能控其尸 + 用其生前全部能力**（驱「鬣狗」追杀）。

## 狄琳（奥术永恒星施法者 · 瑟菲莉娅弟子 · 苏晓的「敌人」）

红法袍、满耳钉、中性短发的女施法者——**法师贤者 [瑟菲莉娅](瑟菲莉娅.md) 的弟子**（异元素气息·与瑟菲莉娅同源·苏晓青钢影对她活跃）。童年被「打着为你好之名虐待」的导师折磨成地狱（恨的是导师 + 灭法之影、非苏晓本人）；奉导师命杀苏晓，却也坦承「与你无冤无仇、只是立场不同」「杀出来的强大可敬」。有轻微洁癖（童年阴影）。**黎元素（针对灭法者）**：瑟菲莉娅把土 + 光元素剥离重铸的「黎」异元素（金属性 / 锋利 / 接触生命力起魔导反应放高温·**且青钢影不会主动噬灭未经魔力干涉的自然元素**）——她整个童年的苦难就为得这能力来克灭法者，却在环瘠岛被苏晓**两个照面就崩碎了黎元素武器**（青钢影并非不能噬异元素、只是平时不主动·一加力就崩）、信念与心态一起血崩；想开后果断撤（「我还不想死、成奥术永恒星的耻辱」）。

## 维京之海群像（第三轮 · 海盗世界）

第三轮「维京之海」是海盗世界——苏晓循古神线索接触的海盗 / 信徒。

### 三大传说海盗团（维京之海霸主）

维京之海大半海域由三大传说海盗（各 5000+ 海盗·船长六阶顶尖 + 心腹六阶 boss 级）说了算：

- **黑海女帝·阿加蒂（最富 · 血鸦 · 精明野心家）**：竟是女人（嫌「黑海女帝」诗人吵故称黑海王·出名之战在黑夜的海上）——三大海盗里**最富**（走海运 / 贩海兽·守规矩〔不滥杀 / 不大量贩奴〕·疑暗控某王国）。能力「**血鸦 / 红鸦症**」（在人体种「种子」·宿主活不过三年、发作化血鸦·苏晓杀过她一只血鸦套情报）。精明 + 野心拉满：算到苏晓难惹便「这事没发生过」灭口目击者、白送古神右臂（图谋用它控泯光神教·想称霸维京之海）、谈判从中作梗想坑苏晓换怒海之魂炼却失算。二副 **疯海牛·图尔特**（干瘦老头·曾与海兽角力）。
- **红胡子·伊格纳缇（最强 · 瘸腿枭雄）**：黑帽黑大衣、金属锥假腿、络腮胡（杀人血溅胡子得名）的瘸子——三大海盗里**总体战力最强**（主黑产·武器 / 野鸳），凶残却胸襟宽广、唯一敢怼骨羊。因恐怖岛死伤近百部下 + 3 心腹被其他参战者策反叛逃而暴怒；与苏晓谈判时试出他「不是来抢地盘的」便踩台阶、还以家乡朗姆酒「只与朋友共饮」、给了恐怖岛海图（[怒海之魂炼](../力量体系/装备道具.md)·15 年仅一份）。
- **骨羊·凡俄西（最疯 · 单挑最强）**：彻头彻尾的疯子、喜怒无常滥杀成性、无心腹——**单挑无人能敌**（红胡子 / 阿加蒂都不是对手）；主营奴隶抓捕（维京之海八成奴隶经其手·仇家无数）。听闻三海盗会面也要掺一脚（行事无法预测）。

- **殇月兄长（羽族 · 夺冠五大热门）**：殇月的嫡亲兄长——本届强者争霸战夺冠五大热门之一（自称「比殇月强 16.752 倍」·扬言宰了蒙德）；第二个抵达恐怖岛级别的强者。父亲顶压力把兄妹俩都送来历练（「羽族不需要温室的花朵」）。
- **费尔德（黄昏之神狂信徒 · 殁）**：原渔夫、神秘天赋低 + 智商捉急的古神狂信徒——被据点头目骗来环瘠岛「传教」、刚到就挨枪重伤垂死。苏晓用 [无尽黑暗](../力量体系/装备道具.md) 项圈 + 假冒信徒套出关键情报（**黄昏之神 / 弗因丁王国 / 海妖港 / 灵祭**）后斩杀。
- **梦魇之神·尼加狄（黄昏之神 · 古神 · 殁）**：海上传说「黄昏的余光、夜晚的君王、绝望的起源」的真身——3.5 米外骨骼古神（智力 156 破壁障·为用夺来的中立女神武器「光之枪」不惜把右臂改造成全骨）。**泯光神教**（教义「抹除太阳、永夜降临」·盘根错节于各王国 / 海盗·想以此世界为起点打通虚空传教）的幕后古神，藏在无风孤岛、靠**梦境操控**（泯光领域剥感知 / 梦境化身神出鬼没 / 把人拖进梦境世界稳杀）操控一切。被苏晓循血滴 + 鲜血印记找上——**刀术修心使苏晓内心强到拖不进梦境**（古神的核心能力卡壳），反被「一颗真实石子破梦境化身」+ 心·魂·刃 + 血之兽斩杀（得史诗级神灵骨 / 圣灵级 [阿斯萨耶](../力量体系/装备道具.md)·[神裁](../力量体系/装备道具.md) 升级）。
- **黑袍人（孤岛引路人 · 失败的反抗者）**：枯骨之躯、死神模样的引路者——本是农夫，两位所爱之人被尼加狄夺走，悲伤到灵魂与骨骼共鸣、血肉剥离，平静地找古神落败却不甘心，从此为每个来此的人渡船引路（连亡妻女笑颜和自己的名字都忘了）。苏晓斩尼加狄后把孤岛 / 残党交给他处理（他眼中瞳焰由白转血红、如接引亡者的死神）。
- **休格（随缘施法者 · 再放苏晓一马）**：黑金法袍、慵懒的奥术永恒星施法者（黑焰能灼烧灵魂·与蒙德同梯队·比狄琳强一截）——认同灭法者理念、与苏晓无仇，在环瘠岛**替狄琳挡场又随缘放走她**（黑闪劈走狄琳）、还替 [风王子](风王子.md) 转告苏晓「近期别去恶魔古堡」；「第四 / 五轮见、在这两败俱伤想想就亏」。

## 死亡屋亡者群（第二阶段 · 痛苦女王的囚徒）

死亡屋（[爱丽丝恶魔古堡](../世界/任务世界/恶魔古堡游戏.md) 的高配版）由 **痛苦女王·安娜**主持，内有诸多被囚的「亡者」（非死人·杀死亡游戏参与者越多、被锁链囚得越死、还会降智）。

- **痛苦女王·安娜（死亡屋主人 · 病娇 · 中立公正）**：黑裙苍白、不时干咳（蒙德反复「你快病死了」她也只无奈微笑）的死亡屋主人——绝对强大却**尽力维系公平**（危险的是死亡屋本身、非她）。并非自愿当主人（「死亡屋需有人看守、或是你或是我」）；女灭法者 **格林·吉莉安**（爱把敌人关笼子的那位）送过她指环（留言「祝你晚点病死」·待她比待 [爱丽丝](../世界/任务世界/恶魔古堡游戏.md) 友善）。
- **圣女·特蕾西（亡者 · 灵魂割离的战争武器 · 解脱）**：金发白裙、脊背刺入 60+ 根黑锁链（杀人无数被囚到极致）——翡廷神权帝国灭亡前把她改造成战争武器：**灵魂上半被劈成两半**（左侧保理智 / 右侧专战斗·可被咒语操控）、念对咒语降伏念反则暴走（海胆瞳 / 咧到耳根的尖牙大嘴）。看似柔弱胆小、侦测全是问号的危险类型；其实并不可怜（至亲被术士烧死后她意识清醒地屠了半个世界泄愤、还吃了唯一好友）。苏晓用暗影石解开她的锁链、她**自愿投入黑雾化灰解脱**（留下白脂蜡 / 秘迹宝石 / 处决之刃）。
- **大祭祀·奴埃（亡者 · 亵渎死亡而降智 · 殁）**：来自卢阳大陆的大祭祀（兼礼仪教师）——被囚 2201 天、杀 30 名拜访者后锁链缩到 3 米且**忘了自己是谁**（杀人越多越降智）；死灵术式需「读条」蓄能，被苏晓「布布汪开关门打断读条 + 巴哈对喷 + 刀芒攒射」的战略生生耗死（死前还以为他在装死）。
- **无声者（亡者 · 缝嘴老太婆 · 卖蜡指路）**：缝着嘴、提油灯、背蜡箱的驼背老太婆——气息恐怖（斧哥 / 猎龙蛛在她面前如小动物）；在黑雾里**卖白脂蜡**（1 颗暗影石 1 根），且会硬扛灵魂缝合之痛给买家指路（觉得一根蜡不值一颗暗影石·这是她的补偿）；遗憾自己没能成死亡屋主人。
- **守雾人（死亡屋核心成员 · 黑雾管理者）**：黑袍、暗哑嗓的死亡屋重要人员——管黑雾里的二层木楼、用「遗忘骨牌」换暗影石（苏晓引摩匹来 + 1 颗暗影石换 2 块骨牌通关）；曾嚣张，被痛苦女王**安娜**（见上）当苏晓面扯掉大半躯干教做人（跪地求饶）。
- **摩匹（硬核信徒 · 殁）**：棕熊般强壮、瞎一眼、穿牧师袍的老头——力大到**直接扯断囚锁链**；信奉一名神灵、每年勒索其分享神力（那神灵被囚后「欣慰得想哭」）；降智（谁跟他搭话就会被他当神灵信奉）。被苏晓当筹码引去守雾人处、被守雾人捏碎头颅（罗盘归苏晓）。
- **死亡屋怪物**：**猎龙蛛（安尤之喉）**（黑渊生物·12 复眼 + 倒齿甲足·实为引路友军·甲足刻「生·死」·爱吃「翠绿果实」的贪吃鬼）；**斧哥（暴食兽）**（4 米无头怪 + 剔骨大斧·诅咒金币反面引来追杀·速度慢但近战无解·苏晓遛了 20 小时）；亡者迷宫的**分裂怪**（斩开就分裂·杀不死）。

## 轮回乐园同伴（本届另两名参战者 · 均出局）

苏晓本届的两名轮回乐园同伴（轮回名额 3 / 3）——均在第二阶段出局，「疯子乐园」作风十足。

- **牧花（战斗奶妈 · 猎杀者 · 抢世界之核出局）**：六阶乐园竞技场曾第 34 名（实为打到前十后嫌无聊故意掉名次虐菜）——第二阶段抢夺类考核因运气不佳（参战者绝望时毁了 6 枚记忆芯片），她**屠光同区域参战者、直冲世界之核抢来**，虚空之树险些判她违规、最终被淘汰回园（成功夺一个原生世界）。
- **巨杰士（即将晋七阶的独狼 · 战死）**：被 17 人围攻战死——但敌方只活下来 1 人（另一人没被传走就被他的能力侵蚀而死·那座岛被打出阴影）。

## 奥术永恒星阵营（第四轮 · 与苏晓正式敌对）

第四轮角斗场观众席上的奥术永恒星高层——斩狄琳后，苏晓与奥术永恒星**正式敌对**（此前恩怨只来自瑟菲莉娅一人）。

- **狄琳（瑟菲莉娅弟子 · 殁）**：第四轮被暴鼠（豁出引导人身份 + 星空座大人物撑腰）安排第二场对上苏晓——化身「跑酷流弓兵」满场逃想拖 5 分钟弃权，黎元素盾 / 螺旋剑 / 黎羽翼一一被青钢影崩碎、连吸青钢影的特制耳钉（瑟菲莉娅成果·能在青钢影侵脑前吸掉）也扛不住，**一分钟被钉死**。临死「这是你最后送我的礼物」（对瑟菲莉娅）。
- **奥术永恒星至高之人**：元素波动强到看不清模样 / 性别——食指戴**马文·华尔兹臂骨 + 格林·吉莉安腿骨 + 初代灭法头骨**所制的骨戒（与灭法者渊源极深）；一句「都闭嘴」让瑟菲莉娅 / 凛风王立刻俯首。
- **凛风王 / 风王子（主和派 · 父子）**：**凛风王**（正派威严·**千年前对抗灭法者的主力之一**·主和派）当众斥瑟菲莉娅「拿弟子做人体实验、法之魂早被灭法者踩灭」；**风王子**（睡衣家里蹲·与苏晓 / 休格有交情·曾托休格转告苏晓「别去恶魔古堡」·苏晓的盟友 / 中立方）。
- **法师贤者·[瑟菲莉娅](瑟菲莉娅.md)**（见其专页）：苦研几百年「克灭法者」的黎元素 / 吸青钢影耳钉全被苏晓两招破、弟子狄琳被一分钟钉死——当众被打脸，与苏晓彻底结仇却碍于多方公证不能出手。

## 其他对手（第四轮角斗场）

- **双子星（死亡乐园）**：第四轮积分第一（运气好 + 实力强）；危险度仅次苏晓的夺冠热门。
- **夜惑女巫（超脱·原生世界）**：魅惑诡异、靠敌人痛苦取乐（30 秒掏心脏）——危险度第三；盯上了蒙德（俩人不在一个频道）。
- **守望乐园小胡子（守卫者 · 法爷）**：精明法爷——第三回合最后一场对苏晓，用守望乐园特产 **[光之沙漏 +13](../力量体系/装备道具.md)**（守望乐园道具可强化）买 5 分钟弃权（不想冒险打第五轮·太亏）。被苏晓「能量阻断」三次电晕成木棍（直挺挺倒地·凛风王看了直拍额头「千年前我也这样」）后弃权——记下苏晓「有能力杀我却守承诺」的人情。

## 十强榜 · 第五轮（极寒之地 · 三十二卷66–73章）

第四轮积分前十晋第五轮（乐园 5 / 虚空 4 / 超脱 1）：**双子星（死亡乐园）· 蒙德（恶魔族）· 夜女巫（夜惑公会）· 沃尔迪（魔鬼族）· 白夜（轮回乐园）· 无名圣徒（天启乐园）· 休格（奥术永恒星）· 泉羽（羽族）· 亚穆（圣域乐园）· 阿芙妮（圣光乐园）**。第五轮 = 极寒之地存活赛（5 天·人手一枚霸者勋章·被夺 10 分钟即淘汰·暴风雪冻灵魂、场地向巅峰山脉收缩）。**泉羽 + 休格已结盟**、沃尔迪坑阿芙妮签了魔鬼契约当炮灰。

- **亚穆（圣域乐园「先驱」· 殁）**：温和神官打扮、腰挂矿铲——实为拿活精灵淬粉召邪祟的阴险货。第五轮被苏晓盯上，想以假交易（拿真霸者勋章当诱饵）暗中蓄势偷袭，反被苏晓秒翻脸 + 巴哈空间背刺 + **[布布汪](布布汪.md) 趁乱偷换「[败者勋章](../力量体系/装备道具.md)」**（刻木纹 / 磨砂仿手感的晶体勋章·上有布布汪歪字「败者勋章」），勋章丢失 10 分钟**首个被淘汰**（×2 勋章归苏晓）。
- **夜惑女巫·尤·西斯·桑德拉（超脱·原生世界 · 殁）**：榜上只标「夜女巫」（真名涉灵魂·只至亲知）——**26 只「女巫眼」**的「女巫眼强盗」（女巫眼 = 存特殊能量的器官·常人仅 3 只·她全靠抢）。靠痛苦取乐、改造身体成夜战怪物（角质护臂 / 心脏掠夺 / 诅咒）；保命绝技「**夜魔皮**」= 每用一次耗一只女巫眼的替死术。第五轮被苏晓团队半包围围猎——[阿姆](阿姆.md) 无解坦度死缠 + 巴哈 三连击放血 + 苏晓刀芒磨耗，夜魔皮被耗尽后斩首（×3 勋章归苏晓）。
- **无名圣徒（天启乐园 · 矿工壮汉）**：腰挂矿铲、十字枪可变高科技矿铲——第三轮第二个抵恐怖岛、第五轮一人刨开沃尔迪天灵盖把他淘汰、把阿芙妮打到跪地求饶（「富到流油的矿工」≠ 弱小）。
- **沃尔迪（魔鬼族 · 淘汰）**：西装 + 镶黑宝石头骨的「奴隶主」——休息区一句对话就钻空子让**阿芙妮**签下魔鬼契约（当炮灰）；曾想拉苏晓交易被无视、被巴哈反递「签轮回契约吗」吓退。第五轮被无名圣徒打到舍弃勋章淘汰（其契约随之解除）。
- **阿芙妮（圣光乐园 · 折扇妹）**：说话爱拿小折扇挡嘴、贱兮兮——死活不肯跟苏晓对上（「五个打一个」）；被沃尔迪契约捆当炮灰、撞上无名圣徒后被打跪求饶。
- **冰精灵（极寒之地土著 · 能量生灵）**：半透明似幽灵的罕见生灵、被布布汪冰雪光环吸引——苏晓假装要炸她家园逼她去找其他参战者「求救」，再贴追踪器当眼线（她先后引来亚穆、夜惑女巫）。后来鼓起勇气把冰洞里的「信号发射装置」（其实是苏晓的道具障眼法）砸了，发现啥也没发生、收获满满自豪感跑没影了。

## 第五轮决战（寒霜森林 + 围攻蒙德 · 三十二卷74–81章）

- **冰霜守卫（寒霜森林 · 世界之核守卫 · 107 名）**：全身惨白、灰冰甲 + 臂刃的人形生灵——都是「罪人」，在身体与灵魂被处死前换一次永世看守世界之核赎罪的机会（故记仇）。守着雪原地下 500 米寒冰大殿的**世界之核**；靠「霜印」以**声音**超远程追踪入侵者（人均斧哥战力、为首者 3 米高更强）。不主动招惹就不危险——**双子星**误触被擒去给巨人墓磕头才放走；[布布汪](布布汪.md) 靠融入环境 + 霜印 / 世界之核双重干扰偷出核、遛飞 20 守卫。
- **双子星（死亡乐园 · 殁）**：两兄弟（弟精神力强、兄身体强）——融合成「**双生形态**」才完整（满头条码组双倒五芒星）；招式「双生·陨 / 混沌〔大范围意念巨拍·秒坦 80–95%〕/ 龙哮」、近战克制极强（最难近身的对手）。被苏晓硬抗两次「双生·混沌」+ 青钢影侵体后一刀贯首斩杀；临死激活心脏炸弹（黑紫丝线·触之即蒸发），苏晓靠猎魔时刻 + 龙影闪穿空间躲过。掉 [猩红宝箱](../力量体系/装备道具.md)（可当挑战书直跳六阶乐园竞技场第 2 名）。
- **蒙德（恶魔族 · 纯血战斗恶魔 · 铁憨憨）**：同辈防御最强、**伤越重力越强**（濒死最猛）——第四天被虚空队 + 无名圣徒组合四打一围攻（休格灵魂黑焰占六成伤），宁站着死不跪着生、把霸者勋章吞进肚也不交。苏晓雪中送炭与他临时结盟（蒙德对付休格）。
- **休格（奥术永恒星 · 修灵魂的黑焰师 · 六强最不危险却深不可测）**：看似火系、实为**修行灵魂**——核心是能灼烧灵魂的「黑焰」（前期弱到掉渣 / 后期越来越强·克纯肉体的蒙德）；人品奇好（曾有同伴为掩护他撤退而死）、与**泉羽**结「虚空队」。
- **泉羽（羽族 · 殇月兄长 · 羽界丝线）**：透明丝线 + 黑羽的「羽界」束缚术——夺冠五大热门；与蒙德有血仇（挚友塔努一个半月前被蒙德当面扯下头颅），强忍私怨参与围攻。
- **无名圣徒（天启乐园 · 科技流土豪矿工）**（承上）：靠**圣灵级动能机甲 +12**（抗斩 / 抗高温 / 抗腐蚀·唯弱穿透）+ **圣灵级十字枪 +14 镶三圣灵宝石**（可变矿铲）+ 核心能力「**心灵意感**」（把机甲活化、发挥 300% 威力）+ 海量一次性道具（几分钟用掉 19 种·最差也史诗级）作战；圣灵级穿甲弹轰掉苏晓近半血 + 斩断其左小臂。讨厌技法类敌人（克他·曾被夺武器），被苏晓夺走十字枪后只得交霸者勋章换回武器。
- **阿芙妮（圣光乐园 · 治疗系 · 折扇妹）**（承上）：与无名圣徒组队当奶妈——被苏晓的 [血羽](../力量体系/装备道具.md) 依附（每 25 秒须治疗一方否则即死·治疗全转「真实伤害」毒奶），奶谁谁倒霉（把自己的召唤植物妹子奶到忠诚 −3 反攻自己）；布布汪还夺走她的治疗大招光球去奶苏晓。全程在演、留着随时丢勋章逃。

## 结局（三十二卷82–92章 · 争霸战终）

- **阿芙妮（殁）**：被**无名圣徒**用「双层契约」反制（她在契约上动手脚反触发隐藏条款被夺 42.8% 资产 + 强制虚弱）→ 当场捏断脖颈、勋章被夺。
- **无名圣徒（第五名退场）**：靠和轮回乐园老阴哔学来的「双层契约」坑死阿芙妮、邀苏晓合作被婉拒，以第五名被淘汰（天启乐园奖励丰厚）；其武器 [天陨之劫 +14](../力量体系/装备道具.md) 被苏晓夺下公证。战后稳坐**六阶乐园竞技场首位**，苏晓拿双子星的猩红宝箱来挑战，他直接弃权拉黑（「你当我是智障吗」）。
- **泉羽（羽族 · 殁）**：与蒙德的血仇了结于此——重伤的铁憨憨蒙德反杀了泉羽。
- **蒙德（末位 · 第三名）**：吞勋章卡肚被苏晓开刀取出、杀泉羽后提交勋章取**末位**得起源石（恶魔族老家伙们需要次位的黑枫树枝干续命，他乐得拿起源石）。与苏晓结下「雪中送炭」的交情。
- **休格（次位 · 第二名 · 逃）**：被苏晓斩得死去活来、想魂仿先代灭法者反引来刀魔夺刀，见苏晓**吃灵魂石**当场认怂、交勋章取次位逃走（黑枫树他在奥术永恒星努力些就能拿）。回头把那把**先代灭法者的刀卖给 [瑟菲莉娅](瑟菲莉娅.md)**（坑她——这刀会招来苏晓和刀魔）。
