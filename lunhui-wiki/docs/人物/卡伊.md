# 卡伊（西港二号人物 · 有理想的金属女巫）

[暗鸦星](../世界/任务世界/暗鸦星-暗鸦纪元.md) **西港的二号人物**（仅次于巫女王 [摩利莎](摩利莎.md)）——黑色短发、双耳扎满耳钉、气息冰冷如金属的少女。四阶段顶尖元素女巫（**金属系**·能操控各类金属针〔附高温 / 毒素〕、肢体可半金属化、徒手接箭），擅旷野不擅室内战。她有几分女汉子风范、更难得是一名**有理想的女巫**——认为女巫能与人类共处、组成放弃战争的帝国共同开发大陆（可惜不懂人心：强大的异常个体终会被人类群体出于恐惧排斥）。她对 [美洛狄](美洛狄.md) 有旧情（美洛狄曾对她有恩）。

## 寒心割席（二十一卷34–39章）

西港攻坚战中她是主战力（守二层、操金属针压制苏晓·被布布汪偷袭吓到）。美洛狄被掳为人质时，她**坐在美洛狄背上、又力主留其当人质**，其实是在摩利莎手下**保美洛狄一命**（有恩必报）。可她看清摩利莎**用 99% 部下的命换力量、见死不救**的真面目后彻底寒心——当众怒骂摩利莎「无情无义的畜生」、**咬断自己被摩利莎挡箭保下的那条左臂还给她**、与西港割席而去（这条命她早能撤、只是道义不允许）。她的离去间接放了苏晓一马。

## 关系

- **[苏晓](苏晓.md)（库库林·白夜）**：攻陷西港的巫猎人（她未与之死战、愤而离去）。
