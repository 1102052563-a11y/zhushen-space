# 鹿神

> ⚠ **虚空「记仇排行榜」第五位**——⚠ **灵系的牌面**;⚠ **⚠ 51卷62章在幽冥之地暴食城堡的晚宴上，作为食客之一登场**（⚠ 与苏晓无正面冲突）。

## 🦌 身份与实力

- ⚠ **形貌**:⚠ **身材高大·头生公麋鹿角·披着披风般的兽皮·脸上、脖颈、手臂等处有很多伤疤。**
- ⚠ **早年间的死战狂人**:⚠ 「**早年间，鹿神不是在与强敌死战，就是在前往死战的路上**」——⚠ **是灵系的牌面**;⚠ **⚠ 近些年晋升至强了，就很少找强者干架。**
- ⚠ **至强级**:⚠ **他已是至强级神灵**——⚠ **虚空记仇排行榜第五位**（⚠ 榜上另有第五名之外的强者，如「记仇」这一榜单本身，便是虚空强者们爱恨情仇的缩影）。

## 出处

- 第51卷62章（**虚空记仇排行榜第五位·灵系牌面·身材高大+头生公麋鹿角+披风般兽皮+脸颈手臂多伤疤·早年间不是在与强敌死战就是在前往死战的路上·近些年晋升至强后很少找强者干架·出席暴食城堡晚宴**）。
