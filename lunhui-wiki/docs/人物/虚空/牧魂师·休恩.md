# 牧魂师·休恩

> ⚠ **奥术永恒星的强大利刃**——⚠ **排除异己、秘密处理那些敢于挑战奥术永恒星之人**;⚠ **⚠ 他与 [自然之女·艾露克露](自然之女·艾露克露.md) 同为本届 [黄金斗技场](../../世界/任务世界/女巫界-巫师界.md) 奥术永恒星的两名参战者**（⚠ 朴素黑袍·自然卷长发·病态苍白·右手缠绷带）。

## 💀 叛徒之子的一生

- ⚠ **父亲是深渊学者**:⚠ **他父亲曾是与瑟菲莉娅、凛风王、古亚院长同级的灵魂学者，与魂大人同出一门**——⚠ **却为奥术永恒星的未来钻研深渊学、并提出「若继续这般吞噬自然元素，今后必定还会有灭法者出现」**;⚠ **⚠ 这直接触及 95% 以上施法者的核心利益，双方决裂，那位深渊学者惨败身死。**
- ⚠ **被丢去灰铁星**:⚠ **休恩与弟弟 [休格](../强者争霸战/休格.md) 从小都是导师口中的天才**——⚠ **父亲死后他被通知离开「仲时学院」、去灰铁星学会杀戮、生存与强大**;⚠ **⚠ 魂大人当时的处境只能二选一（保住休恩或休格），她选择了后者。**

## 🤝 一场施法者与灭法者的交易（52卷28章）

⚠ **他在树皮下留了只有极少数人能看懂的「灵魂印记」——那是伪原罪物「[厄运卷轴/厄运古卷](../../力量体系/装备道具/index.md)」的标识**（⚠ 放眼万界能读懂此印记者不超五指之数）;⚠ **⚠ 苏晓循印记找来后，他不顾艾露克露劝阻、率先全身黑焰冲向苏晓——却在交手瞬间黑焰戛然而止，被一刀斩杀**（⚠ **真相:他直面苏晓时，他持有的「厄运卷轴」被吓懵，压制咒毒的那部分力量骤失，咒毒瞬间爆发**）。

- ⚠ **他求的四件事**:⚠ **① 为「掩护同来的朋友脱身、与灭法者厮杀到最后而死」——让弟弟休格摆脱「叛徒之子」的身份**（⚠ 兄长已为对付灭法者而死，再纠结其弟身份便不妥）;⚠ **② 他身中烈度奇高的咒毒、每分每秒都在难以想象的痛苦中（这痛苦转移给他人半秒就足以让人歇斯底里），他想解脱，却内心傲气、不想死在寻常敌人手中**;⚠ **③ 摆脱缠了他多年的「厄运卷轴」——给这伪原罪物找个「野爹」，没有比灭法者·白夜更合适的人**;⚠ **④ 身揣至宝赴死，让苏晓拿到【法魂宝箱】，是难以拒绝的诚意。**
- ⚠ **⚠ 苏晓的评价**:⚠ **「很果决的老阴哔」**——⚠ **他死后本源力量在极短时间内被因果之力抽空（转给了弟弟休格:两兄弟的父亲留下遗产与诅咒——一个死去，另一个掠夺走对方的所有）。**

## 出处（第52卷）

- 第52卷28–29章（**奥术永恒星的黑色利刃+排除异己的秘密武器·深渊学者之子·父亲提出「继续吞噬自然元素今后必定还会有灭法者」触及95%施法者核心利益而惨败身死·休恩被丢去灰铁星学杀戮·魂大人只能二选一保住了弟弟休格·树皮下留厄运卷轴的灵魂印记引苏晓来·率先冲向苏晓却因厄运卷轴被吓懵致咒毒爆发被一刀斩杀·四重目的:让弟弟摆脱叛徒之子身份/解脱咒毒之苦且不死于寻常敌手/给厄运卷轴找野爹/身揣至宝当诚意·死后本源力量被因果抽空转给休格(父亲留的遗产与诅咒)**）。
