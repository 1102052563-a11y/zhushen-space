# 腐败领主

> ⚠ **[灾祸军团](../../世界/势力/灾祸军团.md) 的领袖**——⚠ **他的诞生极富传奇性。**

## ⚔ 大统领之死

⚠ **第二纪元初期，他还是一个九阶顶尖帝国的大统领**（⚠ 那里比风海大陆更先遭深渊侵袭）——⚠ **当帝国其他军团被打得溃不成军时，他这边已集结精锐、组织起有力且准确的反攻**;⚠ **⚠ 就在他的精锐骑兵团已反攻到一处深渊通道下方时，他接到了如遭雷击的命令:立即返回王都，掩护王室与大贵族们退走、前往联盟星。**

- ⚠ **他拒绝了**:⚠ **舍弃帝国战旗，率部从最前线退下，退守帝国人口最多的贸易之城·伯圣敦**——⚠ **镇守此地十五年，直到手下战士阵亡到仅剩一成**;⚠ **⚠ 几个月后，他站在高耸如山的巨兽尸堆上，失血九成以上，一言不发地坐在巨兽头颅顶，慢慢被周边聚拢而来的黑暗吞没。**
- ⚠ **⚠ 大统领死去了，腐败领主现世**:⚠ **他心中如九幽之寒的怨愤，让那个世界的初始深渊能量开始有了异变方向——变得更具侵蚀性、蔓延性、同化性、适应性**;⚠ **⚠ 他昔日的部下们逐渐站了起来，成为第一批腐化者**——⚠ **「灾祸军团」出现。**

## 🎯 他为何来女巫界

⚠ **他与神父、[白金使徒](../暗影世界/白金主教.md)、[深渊大主教](../永光世界/深渊大主教.md) 达成了合作**——⚠ **[厄运夫人](../女巫界/厄运夫人.md) 窥听到:神父三人要的几百万盎司世界之力，是为了去一个叫「**曙光乐园**」的地方**;⚠ **⚠ 而去那边的名额，有腐败领主一个——这正是他愿意冒险入侵女巫界的原因**（⚠ 否则他一定不会来招惹这些特别记仇的女巫）;⚠ **⚠ 他把三从卫中的两个都派去潜伏进巫师阵营，说明他并未失去理智——没有占领女巫界的想法，而是达成目的后立即退走。**
