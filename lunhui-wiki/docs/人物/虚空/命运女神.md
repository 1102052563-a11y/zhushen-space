# 命运女神

> ⚠ **[幸运女神](../幸运女神.md) 最好的朋友**——⚠ **主掌命运的神灵**;⚠ **⚠ 52卷13章由幸运女神引荐（引荐费 3 瓶苏晓的「神灵药剂」）与苏晓合作**（⚠ 奶白色长发·金红瞳·手背淡金印纹）。

## 🔮 关于「命运不可恢复」

- ⚠ **面对苏晓的紧张**:⚠ **她初见就想留好印象，却是主掌命运却怕这位「面对古神都拔刀」的狠人**——⚠ **「你就是……那位倒霉到超出她神职领域的灭法者吗，真是强大的运势」**（⚠ 布布汪巴哈强忍笑·阿兰娜疯狂默念不能笑）。
- ⚠ **命运的真相**:⚠ **命运虚无缥缈、绝对专属、不可恢复**——⚠ **[白金使徒](../暗影世界/白金主教.md) 能「吞噬命运」，在她看来违反命运系常识、不应存在**;⚠ **⚠ 被吞噬的命运无法用他人命运替补（会引恶果），只能夺回本人的命运才可能恢复**;⚠ **⚠ 折中办法:用「无特性的命运之力」临时加持被吞噬者，能恢复一段时间，随其飘散最终命运枯竭而亡。**
- ⚠ **命运晶石**:⚠ **她拿出一颗存有「无主命运」的中空晶石**——⚠ **击碎后可加持附近被窃命运之人（维持不久）**;⚠ **⚠ 「开个价」→「你付过了……你在未来付过了」**（⚠ 她并非窥探自己未来·而是选择现在拿好处、让药剂大师欠个人情）。

## 🌑 洞悉苏晓的少许命运

⚠ **她私下向幸运女神透露，她洞悉了苏晓的少许命运**——⚠ **「我看到，他站在黑暗中，直面将要吞噬一切的黑暗。」**（⚠ 与 [希儿](../希儿.md) 的警告一同指向终极大敌「至高之人」）。
