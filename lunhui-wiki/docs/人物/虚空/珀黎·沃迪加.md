# 珀黎·沃迪加（虚空富商 · 里德老友 · 周旋羽族与恶魔族之间）

**圣格亚**的虚空富商——身材偏胖、十根手指都戴满戒指、笑起来眼睛眯成一条线的中年人（「珀黎」为虚空东南域的姓氏风格）。是 [陵墓守卫者·里德](../陵墓守卫者·里德.md) 的老友、长期与里德做矿产 / 锻造材料生意，本批 [苏晓](../苏晓.md) 履行「教里德朋友女儿刀术」的人情、正是应他之邀而来（第三十九卷94–97章）。

## 身份与处境

- **看似小角色、实则有靠山**：自陈「外人看我光鲜、实际我就是个小角色」，但他能在 [羽族](../../势力/index.md) 与 [恶魔族](../../势力/index.md) 两大种族间长袖善舞、还敢热情邀请与「奥术永恒星」敌对的灭法者 [苏晓](../苏晓.md) 到家中，全因**他有不止一个靠山**——「机遇与风险并存」是他的经商哲学。
- **圣格亚这个「马蜂窝」**：圣格亚位于羽族与恶魔族地盘的交叠点、双方都划入各自版图，本批**两族已因圣格亚归属权开战**（实为两族老不死秘密筹划的一盘棋）。开战后其他大势力都不愿踏进这个是非之地，反而给了沃迪加这种中间商生存空间——他要小心翼翼维持与羽族 / 恶魔族的关系，「都不能疏远、也不敢太亲密」。
- **豪宅与产业**：招待苏晓的是一栋带 5 公顷庄园的三层豪宅（人工林 / 庭院 / 犬舍 / 葡萄园 / 人工湖 / 花田），而这**只是他众多地产之一**。手下有讲究到极致的老管家与女仆 [提丝](提丝.md)。

## 「你确定？」——把女儿交给苏晓

沃迪加请苏晓在小辈比试前**指点女儿 [珀黎·佩萝](珀黎·佩萝.md)**，报酬都不必（里德已付）。苏晓两度反问「你确定，让我指点你女儿？」，沃迪加却神色如常、举杯一饮而尽——他**根本不清楚这句「当然确定」对女儿意味着什么**（苏晓的「指点」= 让布布汪 / 巴哈拿真刀把佩萝往死里练、还差点溺死她）。结果五天下来佩萝确实从刀术专精突破到 [刀术大师](../../力量体系/技能/index.md) Lv.6。

被 [白牛](../白牛.md) 造访时，沃迪加吓得连大气都不敢出（他根本没资格见白牛这级别的大人物）；白牛临走还让手下都盯着记他长相以作警告（防他是奥术永恒星眼线、危及星空座好不容易来的炼金师苏晓）——被这群虚空恶名昭著之辈惦记，令沃迪加真会失眠。

## 关系

- [陵墓守卫者·里德](../陵墓守卫者·里德.md)：多年老友兼生意伙伴（矿产 / 材料）。
- [苏晓](../苏晓.md)（白夜）：应邀来指点其女儿刀术的灭法者「刀术宗师」。
- [珀黎·佩萝](珀黎·佩萝.md)：其女儿、被苏晓五日调教的刀术天才。
- [白牛](../白牛.md)：造访其庄园的星空座大人物、令他恐惧。
- [提丝](提丝.md)：其庄园的强迫症女仆。
