# 提丝（沃迪加庄园女仆 · 强迫症 · 丈量狂）

[珀黎·沃迪加](珀黎·沃迪加.md) 庄园的短发女仆——目光犀利、腰间右侧挂着直尺 / 皮尺 / 伸缩尺，是个**超级强迫症**。她比 [珀黎·佩萝](珀黎·佩萝.md) 年长十岁、近乎看着佩萝长大，情同姐妹（第三十九卷94–97章）。

## 强迫症的丈量狂

- **0.12 公分也要较真**：[苏晓](../苏晓.md) 一行乘 [里德](../陵墓守卫者·里德.md) 的空间穿行装置抵达时，提丝一进门就指出「位置偏差了」——她扯开伸缩尺量出装置停靠位置与预估偏差 **0.12 公分**、会带来 0.17%~0.19% 的启动风险（乐观是有人受伤、悲观是有人遭遇不测）。她拼尽全力去推那纹丝不动的装置、脸涨得通红，最后靠巴哈鹰爪一勾才卡回金属轨、她如释重负地躬身道谢。
- **每天给佩萝「丈量」**：叫佩萝起床后，她抽出皮尺在佩萝身上一一丈量——连头发长度、衣裙褶皱幅度都不放过，佩萝早已习以为常。
- **善良的旁观者**：她对客人的第一印象排序是布布汪 > 巴哈（帮过她）> 苏晓，觉得三位客人「还挺友善、不是坏人」——她根本不了解，带他们来的里德本身就不是好人、苏晓更是把佩萝往死里练。看着佩萝被血气威压、被真刀所伤、被溺水苦修，她「心如刀绞」，却只能听命递上苏晓给的疗伤药剂。
