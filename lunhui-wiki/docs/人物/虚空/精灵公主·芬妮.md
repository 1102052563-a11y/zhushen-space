# 精灵公主·芬妮

> ⚠ **精灵之国的五公主**（⚠ **精灵之国是依附灵魂书库的势力**——⚠ 说无人敢惹夸张了，但只要不是交恶到极点，虚空、万界中没有任何大势力会对精灵之国出手）;⚠ **⚠ 她是暴躁小精灵·[迪亚古](../暴躁小精灵·迪亚古.md) 的妹妹**（⚠ 双方关系一言难尽:彼此暗暗关心着亲人，见面却一言不合就互揭伤疤、不欢而散）;⚠ **⚠ 黑色哥特裙＋暗紫色眼影，眉眼间给人种不友善感，仿佛惹了她就会被口吐芬芳。**

## 🎲 她的「命运」培养法

⚠ **她把 [艾蜜尔](../虚空/艾蜜尔.md) 从一个「连 50 点属性壁障都过不去、本应死在二阶」的低阶契约者，一路带到九阶**——⚠ **她的理论**:

> ⚠ 「**如果你得过且过，那你的命运线会按照预定的轨道继续，可如果你突然作死，而且反复作死，你的命运就充满无限可能。**」

- ⚠ **她的建议永远是「继续努力作死」**（⚠ 笑得别提多开心）——⚠ **这次让艾蜜尔召人去洗劫天启三姐妹，正是这套路的一环**;⚠ **⚠ 至于「你怎么知道这猎杀者收了灵魂钱币就不会杀人？」——⚠ 「我猜的。」**
- ⚠ **⚠ 她见到了这个虚空纪元的「天选」**:⚠ **看着地上「灭法传送阵」留下的痕迹，她眼中浮现莫名的兴奋**——⚠ **虽然对方还不是唯一**（⚠ 与 [议会长老](../女巫界/议会长老.md) 那句「他只是拥有了『世界石』的天选，还不是唯一」呼应）;⚠ **⚠ 她还提到:她的「怨种三哥」认识虚空中最强契约大师·骗术师 [沃波·伍德](../画中世界/沃波·伍德.md)，兄长还认识另一位契约大师，却不肯说是谁、只称对方「大人」。**
