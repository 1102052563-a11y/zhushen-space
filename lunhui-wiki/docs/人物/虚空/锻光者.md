# 锻光者

> ⚠ **一名下巴处的胡须扎起很粗麻花辫、脸上皮肤粗糙、只是站在那就如同一座矮山的强壮老者**——⚠ **他在黑暗中的岩石锻造台前，一锤锤敲打着如心脏般跳动的白光。**

## 🔨 古老契约

⚠ **苏晓在刀术宗师突破 Lv.95 时第三次见到他**——⚠ **每名技法宗师都能吞噬一把属于自己的魂刃，从而得到独属于自身的能力**;⚠ **⚠ 苏晓之所以肯松手把魂刃交出去，是因为他在锻光者的锤柄末端看到了几个很小的印记**:⚠ **灭法印记 · 月狼印记 · 漫游印记 · 魔镰印记。**

> ⚠ 「**我会遵循古老契约，为灭法者锻出更强的魂刃，还好，锻魂刃的技艺，铭刻在我的灵魂上，否则我或许会忘记，回去吧，等魂刃锻好，无论你在哪，我都会找到你，我有些陌生的……老朋友。**」

- ⚠ **⚠ 「年轻人，你要记住，你只能见我两次。」**——⚠ **这是他亲口对苏晓说过的话**;⚠ **⚠ 倘若苏晓不是灭法者，他一生中真的只能见到锻光者两次**;⚠ **⚠ 第三次见面时，锻光者分明已经不记得苏晓了**（⚠ **第一次见面他甚至没认出苏晓是灭法者**）——⚠ **但他记得古老誓约，也记得如何锻造与增强魂刃**（⚠ 技艺刻印在灵魂上，在其生命之火彻底熄灭前不存在技艺下降的问题）。
- ⚠ **⚠ 这就是先代灭法们强大的多种原因之一**:⚠ **经锻光者打造与增强的魂刃，肯定比普通魂刃强一截，吸收这等魂刃所得的能力肯定也更强**;⚠ **⚠ 苏晓的魂刃被上面千锤百炼的光团攀附后，从虚幻的半透明长刀，凝实成浅蓝色的半成刀坯**——⚠ **他估测最起码要等下个世界进度前期才能收到，吸收后刀术宗师 Lv.95 才是战力全开。**
