# 喔喔（思林特斯族遗孤 · 制造天赋虚空顶尖 · 里德收养）

[陵墓守卫者·里德](../陵墓守卫者·里德.md) 收养的小亚人——身高约 1 米、青绿皮肤、瘦弱、一双水汪汪的大眼异常蠢萌（每挥一次铁锤都要使尽全身力气）。**思林特斯族 = 虚空最擅制造的种族之一**，几百年前被施法者灭族，喔喔是幸存的遗孤（[苏晓](../苏晓.md) 一眼认出、叮嘱她千万别去「奥术永恒星」）。

## 制造天赋顶尖、锻造水平极低

里德二十七卷「捡」回喔喔时说是「看着顺眼」，实为从虚空飞船压力门下抢来的不听话奴隶。到本批（第三十九卷94章），里德终于气到「脑淤血」——

- **顶尖的「制造天赋」**：思林特斯族的制造天赋是**虚空顶尖水平**，这个种族当年为**先代灭法者们制造了很多东西**（比如 [树生世界](../../世界/地点/index.md) 里那台「[天赋唤醒装置](../../力量体系/装备道具/index.md)」，正是先代灭法者委托思林特斯族耗时 105 年打造的）。
- **却「锻造水平如此之低」**：可她跟里德学的**锻造**却学得一塌糊涂——里德痛骂到她蹲墙角抽泣，甚至对苏晓说「白夜，你把她领走吧，收养她我至少短命 200 年」。喔喔真正的天赋在「制造 / 造物」而非「锻打」，跟错了师父。
- **空间穿行装置的制造者**：里德往返虚空「圣格亚」进货用的那台十秒即达的**空间穿行装置**，正是喔喔造的——里德夸「虽然看着破，但绝对没问题、结实得很」。

苏晓深知思林特斯族的价值，一旦把喔喔引荐给 [马文·华尔兹](../马文·华尔兹.md) 与老灭法者那边，「绝对会提供庇护」——对灭法者阵营而言，一个顶尖制造种族的遗孤是极宝贵的。
