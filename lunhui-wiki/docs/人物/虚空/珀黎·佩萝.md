# 珀黎·佩萝（沃迪加之女 · 刀术专精天才 · 苏晓五日调教到刀术大师）

[珀黎·沃迪加](珀黎·沃迪加.md) 之女——一名娇气却修行刻苦的虚空人族少女，师从闭门修行的老者 [白须](白须.md)。半月后要参加虚空**小辈间的比试**，其父请来 [苏晓](../苏晓.md)（白夜）指点五天（第三十九卷95–97章）。

## 「憧憬与差距」——见到偶像

佩萝早听闻父亲请来一位刀术强者、心心念念、脑补对方「强大又年轻帅气、还带点沧桑」。真见到苏晓时，只**一个对视**，她眼前的会客厅便消失、化作尸横遍野的战场，一道双眼透红的身影坐在尸堆上、血月在后、一头血兽从侧后方将她吞掉——这是苏晓 [血气觉醒](../../力量体系/天赋.md) 后「杀意颤栗」的威压。佩萝手中礼盒当场跌落、瞳孔紧缩、几欲转身就逃，却在冥冥中忍住（怕失去机遇），哭腔着报出自己名字。巴哈评价「不错嘛，居然没逃」。

## 刀术天才 · 走错路却连破瓶颈

苏晓以 [使徒之眼](../../力量体系/装备道具.md) 查其资料，发现她的刀术极为「匪夷所思」：

- **刀术专精 Lv.67**——按常理，刀术专精到 Lv.40 就撞瓶颈、突破后应晋升为 [刀术大师](../../力量体系/技能.md) Lv.1；佩萝却**硬把专精的极限一次次冲破**（Lv.41 → Lv.51 → Lv.60 → Lv.67），毅然走在「专精」这条错误的路上、拒绝转「大师」。
- **「理论强到飞起、实战一坨狗屎」**：刀术专精哪怕升到 Lv.70，也不如刀术大师 Lv.5——两者不是一个阶位。这不是她的错，而是**教她的师父 [白须](白须.md) 本身就是个不擅实战、只会闭门钻研刀术的人**（苏晓由此判断白须的「实力」= 与自己交手活不过 5 秒）。能在错误的路上连破瓶颈，反证佩萝在刀术上**天赋极高**。

## 苏晓的「地狱五日」调教

苏晓的教法简单粗暴，目标是先让她能晋升刀术大师：

1. **让布布汪 / 巴哈拿真刀教她「必杀」**——第一课：只要敌对，立刻对准颈部斩、其次心脏（打蛇不死反受其噬）；佩萝惊觉自己苦修多年竟连狗都打不过。
2. **文克达水晶柱溺水修行**——把她关进盛满蒸馏水的水晶柱、青钢影刺激她吐光空气，濒死之际她本能地抽刀斩破「硬过大部分金属」的文克达水晶逃生（激发潜能）。
3. **巴哈带真刀的杀意苦修**——累到脱力歇一小时再继续、只要还在呼吸就练。

五天后，**佩萝的刀术达到 [刀术大师](../../力量体系/技能.md) Lv.6**（有专精 Lv.67 打底、这并不意外）。
