# 启介长老

[火影忍者世界](../世界/任务世界/火影忍者.md) 雾隐之村的一名**老年忍者**，是**「主战派」的核心人物**——与 [照美冥](照美冥.md) 的「主和派」属敌对关系。

- **被照美冥借势（九卷8–9章）**：[苏晓](苏晓.md) 炸村后，照美冥**主动把「村子周围安全」这一重要职务交给启介长老**（理由「现在有贼人破坏村子、内部矛盾先放下」）——他一时想不通照美冥为何愿放手。他刚带人去村边缘，身旁感知型忍者便将他扑倒、躲过苏晓打来的一发狙击（脸盆大的土坑炸开）。照美冥此举既显「大局为重」、又借「外敌」让主战派紧张露破绽（为她上位铺路）。

## 出处
