# 人民教师（怒川冒险团最强 · 双拳怪物）

[怒川冒险团](../势力/index.md)（后文亦写作「荒川」）的成员——无框眼镜、白衬衫扎领带、一副「人民教师」打扮（这种打扮的人「不是老阴哔、就是强到如怪物般」，他属后者）。**他才是怒川冒险团最强的战力**（团长 [青魇](青魇.md) 走谋略、武力并非最强）。面瘫、近乎**没有任何善恶观**——只做自己想做的事，时而救人、时而所作所为连老阴哔青魇都胆寒；连主干成员 [鬼夫人](../势力/index.md) 都不敢违逆他。

- **双拳怪力**：仅**双拳**拥有匹敌乃至超过 [巴克](巴克.md) 的怪力（其余身体部位不具备）——一拳捶地如掀海水、落空的拳风化作空气炮能轰穿居民楼十几米。**专攻敌人后脑**（一拳轰后脑·画面感十足）。

## 围杀黑蜂（十九卷55–61章）

海蓝星世界争夺战中，他在鲁贝城屡次截住天启最强的 [黑蜂](黑蜂.md)——**无视黑蜂的烈性蜂毒、仅凭两只拳头把他锤了个半死**（黑蜂只能靠空间能力一次次逃命，把人民教师列为「仅次于希」的第二号仇人）。最终在青魇的局里，他与 [苏晓](苏晓.md)、[咕噜](咕噜.md)、[所罗门](所罗门.md)、柑橘妹**五人合围**被苏晓「初级空间禁锢」困住的黑蜂——一拳轰碎其头颅，天启乐园的希望黑蜂战死。

## 真名宗木 · 打残天启（十九卷63–65章）

打残天启乐园的合围战中，他与 [咕噜](咕噜.md)、柑橘妹一队清扫城西落单小队。他的真名是**宗木**——但他「弑父、已没资格用生父所赐的名字」、故只用代号。鬼夫人战死后，他给青魇下了句最公道的注脚（鬼夫人生前多次要求死后火葬·尽管烧起来有股怪味）；社恐少女雨沫只信任他与青魇。

## 关系
- **[青魇](青魇.md) / [怒川冒险团](../势力/index.md)**：所属冒险团（他是其最强战力·支持青魇的指挥）。
- **[黑蜂](黑蜂.md)**：屡次锤他半死的天启最强（黑蜂的第二号仇人）。
- **[苏晓](苏晓.md) / [咕噜](咕噜.md) / [所罗门](所罗门.md) / 柑橘妹**：海蓝星临时五人小队的队友（合围斩黑蜂）。
