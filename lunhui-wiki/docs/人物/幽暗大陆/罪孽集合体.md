# 罪孽集合体

死寂城**四门试炼**的终门「罪之试炼」之主——一团黑泥般的半流体集合体，同化历代强者残缺灵魂，能分裂为狼骑士、圣歌团成员乃至神灵时代大战士的不完全战力形态。它把历代被选者的**证明物**全收集到赎罪殿偏殿，也把 [鬼老头](鬼老头.md) 同化进自身（鬼老头一度挣脱意识、以治疗所店主身份活动）。（48卷39–40章）

## 能力与战力

- **分裂个体**:可化为 12 名狼骑士 + 30 名圣歌团等（无狼骑士队长、圣歌团无法融成真正圣心一）。
- **主魂**:真正核心；可在复生强者之间**自由转移**；不灭主魂则残缺灵魂源源复生。
- **复生**:每隔几分钟以残缺灵魂复生一名强者（苏晓一战累计击杀约 16 狼骑 + 35 圣歌 + 11 大战士）。
- **二阶段**:半人半兽八臂骨刺、全身痛苦面孔；精神+灵魂双重尖哮。
- **消亡三斩杀**:灵魂扼杀 / 众生注视 / 罪孽审判——苏晓分别以灵魂强度、意志力、刀术宗师豁免；「噬死而生 Lv.EX」因而未激活。（48卷40章）

## 被斩

⚠ [苏晓](../苏晓.md) 凭神裁戒**罪业之火**对「罪孽」必触发的克制，以伤换杀削减数量；二阶段时鬼老头干枯手臂从脖颈探出抓主魂、苏晓罪焰斜斩+刺头颅将其跪灭。掉落**源石** + **原罪之盒**（无世界之源·特殊个体）。黑雾门消散、通往至高圣所方向的金属门升起。（48卷40章）

## 关系

- [苏晓](../苏晓.md)：通过罪之试炼、将其斩灭的灭法被选者。
- [鬼老头](鬼老头.md)：被它同化又短暂挣脱、临终助苏晓抓主魂的旧时代魂体。
- [狼骑士队长](狼骑士队长.md) / [圣歌团](圣歌团.md)：其分裂个体所借用的残缺灵魂原型（队长本人未复生出）。
