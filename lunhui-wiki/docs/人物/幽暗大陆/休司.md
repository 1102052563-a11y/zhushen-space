# 休司

苏晓从治疗院 172 名新成员里挑中的少年——身穿高领长衣（衣领挡到鼻梁）、清秀腼腆、意外脸皮薄的年轻人。他是**墙外流民部族的弃婴**，被治愈教会探考队捡回城内抚养；⚠ 他**无法开口说话**——那个流民部族会割下婴儿整条舌头（在他们看来「言语是对神灵的不敬、味觉是诱人堕落的魔鬼」），故他靠小册本 / 手语交流。资料评价:性格稳定、判断力 / 执行力 / 应变力全是 A＋。他没有亲人，赚来的钱都捐给北城区的孤儿院与养老院。

## 「空间鬼门」

⚠ 苏晓招揽休司（每天 3000 金镑·远高于新成员年薪 4560）不是图他战力（他自称「打不过克兰克、甚至打不过莉斯」），而是图他诡异却稳定的空间能力:摘下右手黑手套、露出一只发青的**鬼手**当门把手，就能在空气中「嘎吱」拉开一扇**空间鬼门**——门框周边挤满蠕动的眼球与冤魂等污秽物维持稳定，能连通他曾去过的地方（不是随心所欲的空间移动，胜在稳定）。这能力帮苏晓大幅省时、赶路。⚠ 北城区惊变时，唱歌谣的天外存在以「扭变生命力为植物力」杀人，休司当机立断**双手拍破自己耳膜**（丧失听力就不被诱发性能力影响）——苏晓因此对他评价高了几分。
