# 镜中恶灵

一面「魔镜」（顶级凶宅里的落地圆镜）内的恶灵——形象为**人形黑烟**，四处作恶、以智慧生灵的**绝望为食**壮大自身灵体。它能把接触者的意识囚入镜中「伪界」、自己夺取对方身体的控制权（还能追踪对方生活轨迹、遇熟人时短暂「回忆」其身份）。⚠ 它盯上了刚买下顶级凶宅的 [莉斯](莉斯.md)（治疗院新任院长）:囚莉斯意识入镜、占她的身、扬言要「一点点堆叠绝望」地杀光她的朋友同僚——可当它顺着莉斯的轨迹走到治疗院门口、看到「库库林·白夜」的门牌时，恶鬼当场想「鬼生走到了尽头」，因为它绑架的竟是治愈教会治疗院副院长的助手。

## 撞上血兽 · 沦为炮灰

⚠ 它冒充莉斯走进办公室，一开门就被 [苏晓](../苏晓.md) 那「要屠戮多少生灵才有」的血气逼得灵体剧痛（对灵体如侵蚀极强的血焰），与苏晓对视时更看到他身后一只满嘴尖牙的血兽——当场「规规整整跪地」求饶「我是善灵」。苏晓看穿它送上门的蹊跷（像异星战场的莱克利、疑是这半死世界推动的「援助」），给它**植入一根晶体核**（想炸就能把它魂体炸碎），命它十分钟内把莉斯换回来（靠 [休司](休司.md) 的空间鬼门代运），从此把这恶灵当**炮灰**使——去偷危险之物 / 打探凶险情报都不心疼。⚠ 它「镜中伪界」是镜 + 精神 + 梦境三位一体、苏晓进不去也不愿剥离精神体进入。后来它果然与公爵那边也搭上关系（坐实「本世界援助」），被苏晓斩魂之刃穿透伪界威胁，交出了世界级道具**神圣分割器**。

## 关系

- [苏晓](../苏晓.md)：把它血气吓跪、植入晶体核收服、当炮灰使唤、逼它交出神圣分割器的血兽。
- [莉斯](莉斯.md)：被它囚入镜、夺身、扬言杀其亲友的治疗院院长（正主）。
- [休司](休司.md) / [布布汪](../布布汪.md)：用空间鬼门运它 / 奉命盯梢它的苏晓部下。
