# 查曼

**治疗院**在狂兽入侵后仅剩的少数幸存者之一——一身老旧黑风衣、黑色圆礼帽、手持斧柄可伸缩的利斧（斧刃上是洗不掉的红痕），行将就木、手臂脖颈脸上遍布疤痕、头发稀疏、走路都不快的老头。⚠ 别被这副老胳膊老腿骗了:他抽出战斧后会「如疯狗般跳起来狂抡」，是治疗院内出了名的「**老疯狗**」。

## 顾念旧情的收尸人

苏晓（治疗院副院长）那时伤势暴毙概率九成以上，查曼是提着还没展开的**裹尸袋**、准备来给他收尸的——见苏晓竟醒了，先愣一下、慌忙把裹尸袋藏到背后、长叹一声，能来收尸已是很顾及昔日情分（当然，苏晓也没排除「就是这老头下毒、先放身边观察一天」的可能）。此后他受苏晓之托，守着弥留中的女战士 [玛丽娜](玛丽娜.md)（苏晓输人造仿生血把玛丽娜从鬼门关拉回）。治疗院 118 名老成员心理问题严重、几近全数该被处理，查曼是熬过来、还能并肩的极少数。

## 关系

- [苏晓](../苏晓.md)：他准备收尸、却见其醒来、此后听命守护玛丽娜的治疗院副院长。
- [玛丽娜](玛丽娜.md)：与他同为治疗院幸存者、被他守护的重伤女战士。
