# 蛇夫人

⚠ 神灵时代与 [大主教](大主教.md)（狩猎）、[圣祭祀](圣祭祀.md)（月亮）、[瓦迪·特雷奇](瓦迪·特雷奇.md)（万虫）、钢铁使徒（钢铁心）并列的五位大人物之一——椅背印记「**无限之蛇**」，一手缔造了**高墙议会**。她也是一位著名的**占卜师**（暗星·白色小镇那位「蛇夫人」正是她的名号被引用——「占卜的准不准不重要，怎么抓住那条未来线让结果实现才最重要」「占卜师戴兜帽既有美感又避免被记住长相」皆出自她）。

## 「另有要务」的活祭

⚠ 几百年前围杀 [罪业之神·渥米普什](封印古神.md) 时，蛇夫人不参战——她要负责「后续更重要的事」:正是**她的本体持续影响、让罪神一直吮吸这个世界**（配合封印，才把死寂之力吸走、护住高墙城）。她很久前分裂出一个独立分身留在高墙城（即 [烟夫人·阿娜丝](烟夫人·阿娜丝.md) 一脉），本体则封在死寂城入口的封印内。⚠ 但当 [大贤者·图尔兹](大贤者·图尔兹.md) 解开封之门时，最先坠落的是**一条全身腐败、盘结在一起的大蛇**——蛇夫人本体早在多年前就死透了（连她的分身都早已与本体失去精神纽带）。罪神也正因如此挣脱了封印。

## 关系

- [大主教](大主教.md) / [圣祭祀](圣祭祀.md) / [瓦迪·特雷奇](瓦迪·特雷奇.md)：与她并列神灵时代五人、共建高墙城的老伙伴。
- [烟夫人·阿娜丝](烟夫人·阿娜丝.md)：她很久前分裂出、留在高墙城的独立分身（已与本体失去纽带）。
- [罪业之神·渥米普什](封印古神.md)：靠她本体的持续影响才一直吮吸世界的被封古神（她死后罪神挣脱）。
