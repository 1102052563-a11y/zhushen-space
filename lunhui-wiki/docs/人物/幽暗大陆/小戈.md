# 小戈

一名**坐在中型作战机甲肩膀上的小男孩**——[神父](../神父.md) 这次找来的两名新队友之一。到死寂城后他的科技类感测装置基本失灵，能用范围只剩几米，因而对 [贝芙丽](贝芙丽.md) 能感知几十米外的人感到意外。（48卷38章）

## 出场要点

- 开口就问神父「你不会在算计我吧」，被贝芙丽一句「你来这就是被他算计的途中」怼回。
- 目睹神父与苏晓互抛魂血/晶体短刀又双双收回的「朴实无华」圈套时，嘴角抽动。
- 神父见到「好队友」四人组后转身就走，他与贝芙丽诧异目送。（48卷38章）

## 👦 真身 · 贝芙丽的亲弟弟（50卷59章）

⚠ 一个连乐园阵营都不知道的秘密:**[贝芙丽](贝芙丽.md) 其实是他的亲姐姐**——⚠ 这也代表，**他并不是圣域乐园的违规者，他其实是轮回乐园的违规者**，只不过**有着圣域乐园违规者的烙印而已**（⚠ 不用想都知道，**他曾与 [灰绅士](../炎灵星/灰绅士.md) 合作过**）。

- ⚠ **姐弟两人故意隐瞒这关系**，原因很实在:**在乐园阵营，有个完全可信的队友，真的是太难得了**——⚠ 也正因如此，**这姐弟联手坑死了不少契约者与违规者**。
- ⚠ **血脉压制**:他在贫民区小旅馆里玩掌机，只因**撇了撇嘴**，坐在木桌前的贝芙丽**一下站起身**——⚠ 他眼中立即带上几分**害怕**。这不是畏惧或恐惧，**而是来自血脉压制的害怕**:⚠ **他从小就叛逆，不怕自己的父母，就怕自己姐姐**——因为他知道，⚠ **他父母打他，无论怎么样都会手下留情，可他姐姐揍他，那真是往死里揍**。
- ⚠ **他的「大方」露了馅**:苏晓找上贝芙丽夺印记时，她脱口而出「**小戈之前突然这么大方，我就知道没好事，果然，你和神父的较量，这块「初始印记」很重要**」。

## 关系

- [贝芙丽](贝芙丽.md)：外人都不知道的**亲姐姐**——他不怕父母就怕她（「他姐姐揍他，那真是往死里揍」）；姐弟隐瞒关系联手坑死了不少契约者与违规者。
- [神父](../神父.md)：雇他入内城的雇主（他本人也怀疑被算计）。
- [灰绅士](../炎灵星/灰绅士.md)：他那枚圣域乐园违规者烙印的来源——「不用想都知道，小戈曾与灰绅士合作过」。
- [苏晓](../苏晓.md)：神父要交易、工坊黑暗中的刀术宗师。
