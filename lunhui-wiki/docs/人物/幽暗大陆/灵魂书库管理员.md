# 灵魂书库管理员

赎罪殿偏殿里侧那面「不规则扭转」的墙后，是通往**灵魂书库**入口长廊的通道。长廊尽头木桌后坐着一名**章鱼头老者**——头部半透明幽蓝、身穿大袍、背后两扇对开巨门——态度温和却不容无凭证者入内。（48卷40章）

## 灵魂书库（本批可知）

- 苏晓踏入长廊时已**脱离幽暗世界**；书库像收录千万世界知识的独立小世界。
- 入口有气鼓鼓的飞行小精灵（巴哈连丢石块砸肿其中一名脑门）。
- 需**凭证**方可入书库；无凭证则管理员合拢古籍，将访客所在空间退后送回。
- 本世界药师高造诣者中，小部分人曾有入书库资格；死寂城内仍可能遗留凭证。（48卷40章）
