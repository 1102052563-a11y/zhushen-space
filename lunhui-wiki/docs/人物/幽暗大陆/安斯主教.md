# 安斯主教

**治愈教会**的高层之一——深红长袍、面容温和、声音如沐春风的中老年人，与 [苏晓](../苏晓.md)、[大贤者·图尔兹](大贤者·图尔兹.md) 三人在治愈教会内平级。他是出了名的**老好人**、擅长左右逢源、八面玲珑、见人说人话见鬼说鬼话。⚠ 治愈教会高层分三类:靠硬实力上位的（苏晓、图尔兹）、天生有地位的历代圣女、以及安斯这种**左右逢源型**——出大事时不可靠，但平常经营教会的好名声不可或缺（让苏晓、图尔兹这种硬核人物去给信徒主持礼拜根本不靠谱，安斯却能做得很好）。

## 走过场的劝阻

苏晓要炸瓦迪庄园时，安斯主教电话里刚要展开「我们是守序阵营、做事要冷静」的温和劝导，就被苏晓直接挂断（那不是申请、是通知）。到了结界现场，他一边「支持你、你这边我放心」地捧场，一边告诉苏晓一个「坏消息」:学术派一名贤者预言到他是被选者、[大贤者·图尔兹](大贤者·图尔兹.md) 已亲自去教堂找大主教和圣祭祀（被大主教拒见）。⚠ 苏晓提醒自己：别太信任这老好人——他和学术派那边的关系肯定也不错。太阳焰柱化火雨险些烧掉半个高墙城时，他的脸都快绿了（幸得苏晓以烈阳圆盘吸回）。

## 关系

- [苏晓](../苏晓.md)：被他捧场又通风报信、却提醒自己「别太信任」的治疗院副院长。
- [大贤者·图尔兹](大贤者·图尔兹.md)：与他关系不错、同为治愈教会平级高层的学术派院长。
