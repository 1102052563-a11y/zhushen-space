# 鬼老头

死寂城**治疗所**里、木栏橱窗后一名戴着小圆眼镜的小老头——[苏晓](../苏晓.md) 一眼看去，一度怀疑这是不是 [凯撒](../尼古拉斯·凯撒.md) 的分身。他咧嘴一笑，嘴角咧到两侧耳下、满口发黄尖牙交错，不用感知都知道是类似**怨魂**的存在。⚠ 他是 [钢铁制造者](钢铁制造者.md) 那一时代的人——正是他与「神教」联手，把巅峰半神钢铁制造者**阴了**、封困于古代实验所（作为代价，他的肉体被打碎、只剩灵魂体）。钢铁制造者太记仇，多少年过去，见到他还情绪不稳定。

## 治疗所徽章商店

⚠ 鬼老头看守的是虚空之树公证的**治疗所阵营商店**（详见 [装备道具](../../力量体系/装备道具.md)）:不走声望值，而是以神灵时代硬通币「**古代金币**」+ **十二种徽章 / 苍白勋章**换购稀有物资。凭 [圣歌团徽章](../../力量体系/装备道具.md) 苏晓被暂认定为「治愈教会成员」、与治疗所保持中立略偏友善。地位最高的圣歌 / 狼骑 / 圣女三徽章可换黑暗之源，还能换各自特性物（狼骑士徽章换狼血、圣女徽章换「证明物」、圣歌团徽章换团服 + 誓约之匕）。苏晓在此换走黑暗之源 30% + 狼血。

## 阿姆的下落 · 接下委托

⚠ 鬼老头「最近心情好」，主动告诉苏晓一桩「新奇事」:不久前有个「长犄角、挺壮」的倒霉蛋落进钢铁制造者的陷阱，被送进了「**深渊战场**」——那正是苏晓失散的从者 [阿姆](../阿姆.md)。钢铁制造者见阿姆「由炼金生物转变成自然生物」（首见·「侍女」最初灵感就来自炼金生物），才破例开通往深渊战场的通道把他丢进去；阿姆离开只有两法:完成制造者考验、或破坏古代实验所（后者会困死自己）。⚠ 苏晓便**委托身为魂体、能去深渊战场的鬼老头**，转交 90 瓶【活力原液】+ 5 颗灵魂晶核（报酬）+ 一瓶【62.57 盎司世界之力】给阿姆——阿姆注入世界之力后，或能像世界之子那样「死战中不断变强」。鬼老头本想赖账，但苏晓灵魂力量一压、加上「灭法记仇、追杀仇敌十几年」的名声，他立刻答应「四小时内送到」。

## 关系

- [苏晓](../苏晓.md)：他的顾客与委托人——用灵魂威压 + 灭法记仇名声镇住他老实办事。
- [钢铁制造者](钢铁制造者.md)：被他联手神教阴掉、封困至今、见他仍情绪不稳的旧时代半神「老朋友」。
- [阿姆](../阿姆.md)：他受托去深渊战场转交物资与世界之力的对象（那「长犄角的倒霉蛋」）。
