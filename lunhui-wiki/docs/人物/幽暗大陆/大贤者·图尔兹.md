# 大贤者·图尔兹

**治愈教会·学术派**（圣痕学院）的最高——**大贤者**，一身白袍、满头苍白长发、蓄黑长须、眼窝发黑凹陷，看着极严肃、不好相处。他对**灵魂学、神学、圣痕学**都有极深造诣，是这两方面的「百科全书」，与 [苏晓](../苏晓.md)、[安斯主教](安斯主教.md) 三人在治愈教会内平级（谁也管不了谁）。[休司](休司.md)、[莉斯](莉斯.md) 都是圣痕学院毕业的学生，见他就像见母校校长。⚠ 别看他不苟言笑，实则是个「学识渊博的老阴哔」:他并不在乎个人名声，看重的是让圣痕学院有更大名气、把良才留住不被蒸汽神教 / 高墙议会截胡。

## 被选者的反对派

⚠ 图尔兹与 [苏晓](../苏晓.md) 是**不可调和的对手**:学术派早在多年前就集体反对、并废除了「被选者」传统（他们认为解决问题靠被选者不行，大教堂 11 层那些失败被选者的遗骸就是铁证；学术派自有一套立足之法维系高墙城）。当圣痕学院一名贤者预言到苏晓已成被选者，图尔兹的态度就变了——苏晓要进根源·死寂城，必然波及学术派最根本的立足之法。他力排众议供苏晓爆炸物材料、事先组建结界班（名义阻止、实为把爆炸稳定化博名声），可当苏晓抛出**太阳柱**，他两度以暗金能量大手想抓停未果——最后只能超负荷运转结界、放太阳焰柱冲天泄能，自己累得不想留下有失颜面。

- [安斯主教](安斯主教.md)：与他关系不错、同为治愈教会平级高层的老好人。

## 🕯 封罪神的主导者 · 一生的执念（48卷19章）

⚠ 图尔兹并非天生保守——几百年前高墙城最艰难时，他这个年轻人力主「立即封锁死寂城入口、放弃被选者传统」，遭狂热拥护者险些处死、被 [大主教](大主教.md) 保下，20 年后当上圣痕学院院长。他冒死离开本世界（本世界底蕴够）去外世界游历，从一个「暗无天日却再无空间震爆」的世界得到灵感——主导了**封印一位古神吸走死寂**的计划:诱来八阶最顶尖战力古神 [罪业之神·渥米普什](封印古神.md)。⚠ 围困它时四大势力九成超凡者战死、他本已备好战死，却被大主教一拳打晕保下，醒来战斗已结束、从此接管治愈教会。他一生都不认可「被选者」这条路（大教堂 11 层的失败者遗骸就是他的论据），与身为被选者的 [苏晓](../苏晓.md) 不可调和。

## ☠ 圣光普照 · 舍命一击（48卷20章）

⚠ 解封时罪神挣脱、速度恐怖到不讲道理、压缩巴掌大暗物质防御无解。图尔兹积蓄半天，以「苍白眼」令罪神短暂僵直、又在半米内引爆掌心龙眼大的**圣光普照**圆核——刺目白光把方圆炸成十公里巨坑，**破了罪神的体表防御、蒸发其暗物质、令其速度锐减**（这正是后续能围杀速度恐怖的罪神的关键前提）。代价是他自身皮肉血肉尽数枯化如枯枝、随即被罪神的原罪刃镰尾锥**刺穿心脏而死**（心脏是他的力量源泉、致命弱点）。⚠ 他明知击败罪神多难、也知同伴还要深入死寂城而他不会去（他到死仍不认可被选者），却仍以命换下这最关键的一击——[苏晓](../苏晓.md) 让巴哈带出了他的遗骸。
