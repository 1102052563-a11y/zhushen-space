# 泽卡亚

[神女](神女.md) 的护卫之一——扎单马尾、气质锐利、颜值堪比 [莉斯](莉斯.md) 的美少年（一眼看去像少女）。神女被治疗院绑走后，他在几名圣痕学院导师的**拱火**下来营救:计划是自己故意冲袭 [苏晓](../苏晓.md) 的办公室、拖住对方，同伴趁机救人。⚠ 计划很「完美」，现实很骨感——他一进办公室感知全开，就撞见了 [罪亚斯](../画中世界/罪亚斯.md)（幽绿骷髅）、[伍德](../画中世界/沃波·伍德.md)（触手怪）、苏晓（血兽）、[凯撒](../尼古拉斯·凯撒.md) 四大「好队友」同框的毕生难忘景象，咽着口水想「说好的单挑呢」，转眼就被自己背上生出的黑触手捆成毛毛虫。他和另外三名护卫（含一对孪生姐妹与司机）被一锅端进治疗院地牢——圣女一脉派来「营救」，实则是三个憨憨 + 一个送人头，只求神女别死。

## 关系

- [神女](神女.md)：他冒死营救、却把自己搭进地牢的主子。
- [苏晓](../苏晓.md)：他奉命冲袭拖住、却被好队友一锅端擒住的治疗院副院长。
