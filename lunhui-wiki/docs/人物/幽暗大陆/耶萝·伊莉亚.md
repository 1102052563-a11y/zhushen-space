# 耶萝·伊莉亚

[罪亚斯](../画中世界/罪亚斯.md) 与 [奥娜](../树生世界/奥娜.md) 的**女儿**——12 岁、扎鱼骨辫、遗传奥娜长相更多的乖巧小姑娘，家教极好、举止慢吞吞地讲究优雅。她外婆是个「名讳不能轻易说出」的超级狠人。这次父母与外婆放她独自来幽暗大陆**送信 + 见世面**（外婆没派她去更凶的九阶世界，只让来相对安定的高墙城送信），她靠一种占卜物确定了 [苏晓](../苏晓.md) 的方位找上门。

## 完美的喊爹保命

⚠ 罪亚斯太清楚苏晓猎古神的本事与对非友军古神系的态度，出门前反复叮嘱女儿:一旦感知到空间波动、或看到血色 / 蓝色电弧，就立刻喊「**罪亚斯是我爸爸**」。事实证明这招神准——苏晓的斩龙闪刀尖已抵到她眉心、几乎触到皮肤时，正是这句话救了她的命。她母亲奥娜还教她「若那位白夜大哥哥对你笑，就赶快跪成一团求饶，不丢人」，于是苏晓一露和善笑容，她当场「跪成一团」碎碎念「我没有钱也没有灵魂晶核、我家可穷了」。除了眼睛东张西望，伊莉亚极乖巧——但揍起那个「和年轻罪亚斯一个模子」的熊孩子弟弟来，从不手软（弟弟谁都不怕、独怕这会往死里揍的姐姐）。

## 关系

- [罪亚斯](../画中世界/罪亚斯.md) / [奥娜](../树生世界/奥娜.md)：她的父母（古神系「好队友」夫妇），教她喊爹 / 跪地保命。
- [苏晓](../苏晓.md)：她来送信、险些挨一刀、受托照料两天的「老友」。
- [布布汪](../布布汪.md)：会照料罪亚斯之女、蹲在她身旁的头号随从。
