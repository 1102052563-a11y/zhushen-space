# 圣歌团

**治愈教会**曾公认最能打的两大分支之一（另一是剑狼队 / 狼骑士）——统领大部分教会骑士的核心守备力量，如今守在死寂城内城的**圣十教堂**、保管着**四强者**中的一颗**源石**。她们是神灵时代活到今日的**永生者**，全员为「圣心」（按排名加冕、以数字为号；神灵时代同时有 30 名圣心）。⚠ 挑战她们=非生即死（教堂门闸落下、顶部巨钟倒计时约 30 分钟，被选者活到钟停即算通过、可临时带走源石；败则死）；此前从没有被选者真正击败过她们，最多得其认可暂取源石。

## 五名圣心

- **圣心一**:大姐大、C 位、比众人高半头——螺旋长枪、**枪术宗师 Lv.70**、金蓝双特性强防御铠甲。⚠ 她的螺旋枪枪尖断了 1 公分，是弱点。
- **圣十七**（圣心十七）:前排、菱盾 + 长刀、战盾宗师 Lv.68 + 刀术宗师 Lv.52、金色全甲，擅控制 / 防御。
- **圣心十**:苗条却右臂镶铸金属、灵魂战锤。
- **圣心七**:身材娇小如「凝视灵魂的黄金毒蜂」、灵魂大弓（苍白猎手都是她们教出的猎人）。
- **圣三十**（圣心三十）:小小妹、金袍兜帽、水晶权杖、远程增益（水晶爆弹）。

## 「圣心·聚合」与三重削弱

⚠ 五人对单独强者近乎无解:三近战 + 两远程（灵魂战锤 / 灵魂弓的灵魂攻击杀死过无数挑战者）。被打散后她们会施「**圣心·聚合**」——五人化金流合为**圣心一**（鱼骨辫末端一分为五、各系 1/7/10/17/30 的指环），状态全恢复巅峰、由圣心一主导、其余四人辅助。⚠ 但她们经历**三重削弱**（幽暗大陆自行降维 + 死寂爆发 + 常年身处死寂城），终极技「神圣怒放」、奥义「烈阳裁伐·极昼」皆因死寂无法使用。苏晓这个「究极克星」以血气压制 + 晶体躯壳金蝉脱壳 + 空间穿透（斩龙闪穿盾）+ 斩魂拳砸碎灵魂战锤 + 暗黑战靴直踹，把聚合的圣心一踹散——⚠ **首次真正击败圣歌团**。她们躬身认败:交出源石（圣心一剖心取出）、隐秘宝库钥匙、圣歌团印记（凭此可在其势力范围自由通行）+ 10% 世界之源。
