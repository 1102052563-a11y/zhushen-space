# 钢铁使徒

⚠ 神灵时代高墙城**五位缔造者之一**、旧治愈教会**十二位高层之一**（椅背印记为「钢铁心 / 钢铁印记」），也是**蒸汽神教的初代开创者**。他是半神 [钢铁制造者](钢铁制造者.md) 的学生，一心向科技侧发展、信奉「血肉苦弱、机械飞升」。⚠ 但治愈教会的圣痕修行的正是肉体与灵魂——他自己放弃血肉没人管，却要求部下也放弃血肉，理念与教会彻底对立，遂被**逐出治愈教会、背上「罪人」之名**（[治疗所](../../力量体系/装备道具.md) 里代表他的「罪人徽章 / 钢铁徽章」带着悬赏意味，能换起源石·混沌之火，等同旧教会的通缉赏格）。

## 独立建教 · 被四老合谋重创

⚠ 神灵时代末（治愈教会巅峰期）钢铁使徒位高权重，直到他执意独立、成立起与治愈教会理念对立的**蒸汽神教**（若非当时时局太需要它，大主教与圣祭祀早会出手剿灭）。到高墙城建立后，[大主教](大主教.md)、[圣祭祀](圣祭祀.md)、[蛇夫人](蛇夫人.md)、老怪物 [瓦迪·特雷奇](瓦迪·特雷奇.md) 四人**合谋忽悠钢铁使徒去围攻罪神**——在四人的「刻意关照」下，他虽没死，机械核心却严重受损，此后长眠不醒，原就不太正常的思维变得更加难以捉摸。他在与罪神之战中**最先落幕**、核心损毁，蒸汽神教后来还是靠大主教帮衬才成立。

## 「具量」不死 · 被公爵反算计送死

⚠ 钢铁使徒的独有手段是「**具量**」:将灵魂融入机械结构、达成「核心不灭则身不死」。几天前 [机械公爵](机械公爵.md) 为自救，**故意把钢铁使徒的机械核心植入自己体内、将其唤醒**——钢铁使徒的意识苏醒后占据了公爵躯体，把公爵灵魂「具量」进机械义体成「公爵核心」，自以为处理掉了公爵。⚠ 可这一切正中公爵下怀:公爵早被苏晓下了**炼金合成物**（微生物特性、不断分裂、专门软化消融金属，对满身机械的他比剧毒更致命），这具躯体已是死局。钢铁使徒思维混乱，浑然不知自己是被公爵推出来的**替死鬼**。

## 被斩

⚠ 死寂城内，占着公爵躯体的钢铁使徒来找 [苏晓](../苏晓.md) 夺教会石板。苏晓早看破他气场不对（攻击手段比圣歌团强一筹、接近狼骑士队长），罕见地战前开口、连刀都不拔，只为拖延时间让**催化剂**起效——连问三个「公爵为什么」逼他露怯，随后一刀贯穿其核心。合金躯体因炼金合成物**锈化风化成红色粉尘**，钢铁使徒到死都没想明白「只休眠了些年、世界怎么变化这么大」。掉落 11% 世界之源 + 机械核心（半损）+ **罪人徽章**。他留下的地下小实验所（藏神灵时代药剂师公会资料等），被公爵拿来与苏晓做「共享知识」的交易筹码。
