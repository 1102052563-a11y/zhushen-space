# 迪肯·恩

**奥术永恒星**派来幽暗大陆的施法者大队头领——身披法袍、脸色病态苍白、气息阴冷的男人，人狠话不多。他统领一百余名施法者（在虚空同阶罕有敌手），本随 [乌鸦女](../画中世界/乌鸦女.md) 一同为「死寂城的黑枫树」而来。⚠ 他与乌鸦女的分工微妙:乌鸦女因身份特殊（非法系、奥术永恒星的「疯狗」）地位独特，围杀 [苏晓](../苏晓.md) 的计划多由她提出。

## 「动手」话音未落的背刺

⚠ 乌鸦女联合被苏晓一方「搞定」的学术派（[大贤者·图尔兹](大贤者·图尔兹.md) 那边），把苏晓围在城南农场木屋——迪肯·恩一声「少废话，动手！」话音刚落，学术派的**络腮胡导师**（地位仅次图尔兹）就一刀**神灵特性武器**从背心刺穿他的胸膛。这一刀点破了真相:学术派早已被 [罪亚斯](../画中世界/罪亚斯.md) 出城两天后「搞定」、当了苏晓一方的临时打手，百余施法者反被治疗院 / 学术派 / 工坊 / 圣女一脉的埋伏团团包围。⚠ 更妙的是苏晓一句「乌鸦，你做得不错，来领赏吧」——迪肯·恩当即确信乌鸦女「叛变、和那灭法有一腿」（乌鸦女两次追杀苏晓都毫发无损而归的旧传言成了铁证），围杀反成了让施法者内讧、消耗战力的局。

## 关系

- [乌鸦女](../画中世界/乌鸦女.md)：与他同来、被苏晓一句话坐实「叛变」嫌疑的奥术永恒星「疯狗」。
- [苏晓](../苏晓.md)：他率众围杀、却反被里应外合包围 + 挑内讧的灭法者。
- 络腮胡导师：一刀神灵武器背刺他的学术派导师（地位仅次图尔兹）。

## ☠ 白树林殿后 · 「现在是我们的时代」（48卷17章）

⚠ 被络腮胡背刺的短刀「不能拔」（拔则暴毙或异化狂兽），迪肯·恩只得下令撤退——百余施法者靠懒散施法者的**陨石巨人 + 魔能爆发 + 大规模空间传送**逃出城。可 [苏晓](../苏晓.md) 是猎杀者、追猎是本行:靠 [罪亚斯](../画中世界/罪亚斯.md) 给的黑铁指环（与那把定位短刀共鸣）+ [布布汪](../布布汪.md) 问路，追到恶土白树林。⚠ 苏晓一人（+ 布布汪光环 + 巴哈）就把百余施法者反包围:青钢影「能量阻断」让他们每次施法有 58% 概率灵魂感电麻痹、随即被血枪射杀，绝魔体质 + 灭法分支让法系伤害几乎无效。施法者四散奔逃，迪肯·恩选择**留下殿后**拖住苏晓，笑着露出沾血的牙「嘿，你好」，两分钟后被长刀穿胸跪地——临死仍笑「你赢不了我们，现在是我们的时代」，被苏晓一刀了结。（一名小个子施法者回来替他殿后，一并被杀。）

## 关系（补充）

- [苏晓](../苏晓.md)：追至白树林、以灭法克施法者、斩杀他这殿后者的猎杀者。
