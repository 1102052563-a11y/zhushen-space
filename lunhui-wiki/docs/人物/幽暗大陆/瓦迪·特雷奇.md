# 瓦迪·特雷奇

⚠ 神祭日惊变的**幕后真凶**——瓦迪家族的**初代家主**、藏在瓦迪庄园「深度世界」祭祀厅里的**「老怪物」**。他一双完全灰黑色的眼、气息死气沉沉却无半点迟暮感，脸上还长着当代家主 [瓦迪·利法克](../../世界/任务世界/幽暗大陆-死寂根源之地.md) 那颗独特瞳孔的眼球（他吞噬了历代家主）、随手抠出咀嚼。⚠ 真相是:瓦迪家族从创立至今**家主从未换过人**——一直是这个老怪物，每一代「新家主」都是他吞噬、取而代之的伪装。他背后石座椅背刻着「**万虫**」印记。

## 神灵时代的五人之一

⚠ 苏晓借大教堂 12 层的五座石座（椅背印记:狩猎〔[大主教](大主教.md)〕、月亮〔[圣祭祀](圣祭祀.md)〕、无限之蛇、万虫、钢铁心）串起了真相:**神灵时代有五位大人物一同建起高墙城**——猎人（大主教）+ 月亮（圣祭祀）创立治愈教会、万虫（老怪物）夺得瓦迪家族、钢铁（蒸汽神教初代领袖）、无限之蛇（高墙议会缔造者）。五人都因与**永生之神**的关联而获近乎永生的体质，贯穿整个墙纪元的历史。⚠ 世界简介载「瓦迪·特雷奇墙纪元 147 年生」——实为老怪物在其成年后夺舍了这个有生意头脑的普通人、取而代之组建商盟。他的本体不明（刚夺永生、众神之眼只能读出乱码）。

## 「老怪物」的永生赌局

⚠ 不久前永生之神出了问题，连累五人都**失去永生体质、即将寿终**。[大主教](大主教.md)、[圣祭祀](圣祭祀.md) 坦然接受永眠，唯独这老怪物对生命执念可怕——遂在神祭日窃取永生之神的少量神力，引来同特性的存在:核心目标是**夺 [痛苦之女](../../世界/任务世界/幽暗大陆-死寂根源之地.md) 的不死永生** + 让**黑暗行者**顺血脉侵灼瓦迪家族全员灵魂（灭掉那支世代秘密培养、只为杀他的「猎人队」）。小花花、羊头恶魔、天外使者都是意外的「附赠品」。他一度成了赢家（除隐患 + 续永生），却撞上登门的灭法者 [苏晓](../苏晓.md)——苏晓炸平庄园斩了痛苦之女、又用神圣分割器闯进深度世界找上门。

## 关系

- [大主教](大主教.md) / [圣祭祀](圣祭祀.md)：与他同为神灵时代五人、口中「教堂那两个老朋友」（坦然赴死、与他执念相反）。
- [苏晓](../苏晓.md)：斩了他要夺永生的痛苦之女、闯进深度世界找上门来的灭法者「被选者」。
- 瓦迪·利法克（当代家主）：被他吞噬、脸上长其眼球的历代伪装身份之一。
