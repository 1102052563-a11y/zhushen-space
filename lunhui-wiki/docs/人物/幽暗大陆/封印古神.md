# 罪业之神·渥米普什

⚠ 治愈教会锁困在死寂城**封之门**后、逼其「吸走死寂」几百年的一位**古神**——它正是高墙城为何不被死寂侵染、以及 [苏晓](../苏晓.md) 一进本世界就感知「世界被古神吮吸却毫无迹象」的答案。它整体 3 米 5 人形、上半身生满光洁黑鳞、面部是纤长白色骨面具（像外骨骼）、黑雾般长发与触手如诡异拖地长裙，持一把 2 米 6 的**原罪战镰**（暗系金属 + 骨骼 + 黑暗血肉 + 固态灵魂）。⚠ 它如今已**不完全是古神，而是半个古神、半个深渊存在**——不知从何摄取大量深渊之力、还适应了深渊能量，比原本更强。战前它开口只吐两字「灭法」。

## 「以古神吸死寂」的疯狂之法

⚠ 治愈教会想出的疯狂对策:古神能吮吸世界、让世界暗无天日——幽暗大陆本就因死寂暗无天日，那么**封住一位古神、逼它持续吮吸这个世界**，它吸走部分生机的同时也把高墙城内的死寂之力一并吸走！几百年前 [大贤者·图尔兹](大贤者·图尔兹.md) 主导此计:诱骗渥米普什的神使、往其体内灌友善神灵能量挑衅，引来这位**八阶最顶尖战力古神**。⚠ 围困它的代价惨烈:四大势力九成以上超凡者战死、它一度杀进高墙城吸走三成平民、五位创建者〔钢铁使徒核心近碎、老怪物灵魂几乎粉碎、[圣祭祀](圣祭祀.md) 差点狂兽化、[大主教](大主教.md) 再用不了初始圣痕、[蛇夫人](蛇夫人.md) 另有要务〕尽数重创，最后靠大主教的能力困住它。⚠ 它吸了几百年死寂却不消亡、封印近年反越来越强（学术派因此死活不肯开门），故 [罪亚斯](../画中世界/罪亚斯.md) 促成用「眼之仪式·滋生眼」替代封古神的方案后，众人才来解封围杀。

## ☠ 被「好队友」+ 图尔兹舍命围杀

⚠ 开门时先落下的是早已死透的 [蛇夫人](蛇夫人.md) 本体（大蛇），渥米普什则从天棚黑液中落下——它挣脱封印已久。它速度恐怖到不讲道理、压缩巴掌大暗物质防御无解。围杀靠众人层层配合:苏晓龙影闪穿透空间硬接 + 刃道刀·时减速 + 金蝉脱壳、[烟夫人·阿娜丝](烟夫人·阿娜丝.md) 化身替死 +「无妄」禁锢 1.5 秒、[阿姆](../阿姆.md) 冰封、巴哈鹰爪、伍德骷髅——⚠ 最关键是 **[大贤者·图尔兹](大贤者·图尔兹.md) 的舍命一击**（苍白眼僵直 + 龙眼圆核「圣光普照」）破了它体表防御、蒸发暗物质、速度锐减（图尔兹自身枯化、被它刃镰尾刺穿心脏而死）。随后罪亚斯黑粘虫〔灵魂 + 炼金剧毒〕+ 伍德幽焰 + 凯撒人罐合一「负增益 Lv.EX」（灵魂伤害 ×5 / 毒 ×2 / 幽焰 ×4.2）四连击。⚠ 苏晓收尾:暗黑战靴连踹 + 圣诗（先古面具伪装的灵魂武器）灵魂锁链反扯、踹上高空 + 太阳桶引爆 + 烈阳圆盘吸焰，最终贯胸斩首——**罪神被围杀**（苏晓的【神裁戒】吸其本源、成长度升 36.8%、得屠神能力「罪业之火」）。击杀得 27.95% 世界之源 + 换购列表〔含奥义级「原罪·天御戮杀」等〕。
