# 玛丽娜

**治疗院**在狂兽入侵后仅剩的重伤幸存者——请别被这甜美名字骗了:玛丽娜女士身高 **2 米 37、体重 415 磅**、体脂率不超 5%。苏晓的临时记忆里，她**生撕一只狂兽**的场面格外清晰，是战到最后才倒下的猛女。⚠ 她原出身高墙议会**宪兵部**，因看不惯宪兵队军官欺压平民、当场重拳把那军官打得半死，被治疗院副院长（苏晓前主人）无意间遇到、吸纳进治疗院，此后一直称苏晓「长官」。

## 从鬼门关被拉回

狂兽入侵后她重伤弥留:脸色蜡黄、呼吸时有时停。[苏晓](../苏晓.md) 给她做静脉注射、又输了几大袋「人造仿生血液」（某天才发明·不分血型、能临时替代鲜血传氧并缓慢化为造血养分）——她生命力极强，输了几百毫升就恢复了神采、能作势起身。苏晓安排 [查曼](查曼.md) 继续守着她，确保情况稳定。

## 关系

- [苏晓](../苏晓.md)：把她从宪兵部吸纳进治疗院、又输仿生血把她救回的「长官」。
- [查曼](查曼.md)：与她同为治疗院幸存者、奉命守护她的战斧老兵。

## 🌙 银·月狼之血 · 灭法的老友既视感（48卷18章）

⚠ 苏晓对玛丽娜的印象比查曼、休司、莉斯都更好，像「许久没见的老友」——直到得知她一个爱好:每到夜深，她都爱独自坐在寝室楼顶、沐浴月光，说月光下让她「感到安心」，却不知为何。⚠ 苏晓由此参透:这与玛丽娜本人无关，而是她体内传承的**银·月狼之血**。当年**灭法之影与银·月狼以元素力量为凭立下盟友誓约**，如今遇到传承狼血之人，苏晓自然生出老友般的既视感。可惜玛丽娜体内狼血不多，连「月狼化」都做不到（只能「人狼化」）、更用不了月光之力。⚠ 苏晓推测本世界大概率有**月狼墓（狼冢）**——他的成长武器【银月之刃】需抵达银·月狼葬身地献上鲜肉方能成长（已成长一次、共三次），故打算问 [大主教](大主教.md) 狼冢所在。玛丽娜没随苏晓去死寂城（「你的事之后再说」）。

## 关系（补充）

- [苏晓](../苏晓.md)：因她体内银·月狼之血对她生出「老友既视感」、想为她寻月狼墓的灭法者（灭法之影 = 银·月狼的盟友）。
