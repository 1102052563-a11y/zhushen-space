# 玛丽娜

**治疗院**在狂兽入侵后仅剩的重伤幸存者——请别被这甜美名字骗了:玛丽娜女士身高 **2 米 37、体重 415 磅**、体脂率不超 5%。苏晓的临时记忆里，她**生撕一只狂兽**的场面格外清晰，是战到最后才倒下的猛女。⚠ 她原出身高墙议会**宪兵部**，因看不惯宪兵队军官欺压平民、当场重拳把那军官打得半死，被治疗院副院长（苏晓前主人）无意间遇到、吸纳进治疗院，此后一直称苏晓「长官」。

## 从鬼门关被拉回

狂兽入侵后她重伤弥留:脸色蜡黄、呼吸时有时停。[苏晓](../苏晓.md) 给她做静脉注射、又输了几大袋「人造仿生血液」（某天才发明·不分血型、能临时替代鲜血传氧并缓慢化为造血养分）——她生命力极强，输了几百毫升就恢复了神采、能作势起身。苏晓安排 [查曼](查曼.md) 继续守着她，确保情况稳定。

## 关系

- [苏晓](../苏晓.md)：把她从宪兵部吸纳进治疗院、又输仿生血把她救回的「长官」。
- [查曼](查曼.md)：与她同为治疗院幸存者、奉命守护她的战斧老兵。
