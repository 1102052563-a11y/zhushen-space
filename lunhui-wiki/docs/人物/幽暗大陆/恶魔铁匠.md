# 恶魔铁匠

一位魁梧的锻造大师——暗红粗糙的皮肤、扎成粗须辫的大胡子、头顶弯曲的公羊角、一双内部仿佛有岩浆燃烧的眸子，站着就像一座古老而愤怒的火山。他能**熔锻神性灵魂**（多数锻造师做不到），且技艺高超到**不挑材料**——任何有超凡特性的装备 / 器物都能当材料。苏晓凭骷髅赌徒传话、以【誓约之物】约他在死寂城的**安眠庭院**（旧治愈教会的大教堂）相见。⚠ 多年前另一位灭法 **[马文·华尔兹](../马文·华尔兹.md)** 也以誓约之物约见过他（在「界之底」）——苏晓机敏地称与马文「不熟」，才没挨那把炽红锻锤（否则十有八九得挨一锤）。

## 为灭法免费锻造

⚠ 苏晓摆出【熔炼炉】（恶魔铁匠万年前在月神大陆亲手所造、后遗失、被古神越修越坏）+ 两颗邪神灵魂【神灵之灵魂·圣橡 / 始祖】（苏晓「钓」来的）+ 1 万灵魂钱币。恶魔铁匠退回钱币「给你们灭法锻造，我不收钱」，留下熔炼炉修好，给苏晓两选:两邪神灵魂**锻两件新装备**或**改良两件现有装备**。苏晓选:①打一件新**刀鞘**（替换太费宝石的史诗级【宝石誓约】）；②提升霸主级装备**血羽**（霸主装备按三 / 五 / 十精魄分级，苏晓四件霸主里三件是三精魄、只有银月之刃是五精魄）——代价另加一堆没用的装备 + 100 盎司时空之力。他还告知苏晓**狼冢**（月狼、跟灭法的「大狗」）的方向。

## 关系

- [苏晓](../苏晓.md)：他免费为之锻造刀鞘、提升血羽的灭法者（凭誓约之物约见）。
- [马文·华尔兹](../马文·华尔兹.md)：多年前同样以誓约之物约见过他的另一位灭法（苏晓的领路人 + 导师，但苏晓机敏地说「不熟」）。
