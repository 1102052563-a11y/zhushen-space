# 黑羽

治疗所内一名身披黑色羽衣、体型估测为女性的「乌鸦医生」——实为**猎人公会**猎人，接委托暂时顶替一名疯癫乌鸦医生。持造型奇异、约 60 公分的短刀，刀柄有圆形印徽；[苏晓](../苏晓.md) 获九星称号【猎神者】后，在称号简介中认出该印徽=猎人公会。（48卷45章）

## 猎人公会（本批可知）

- 与地精商会类似，往来乐园阵营 / 虚空 / 超脱世界 / 高阶原生世界，但更隐秘低调，**仅受理与发布委托**。
- 持【猎神者】称号后默认持有「猎人铭牌」，可在猎人公会接/发委托。（48卷44–45章）

## 猎刃

⚠ 她其实也能治疗（「能不能治活看当事人生命力」），但更自认猎人。委托早完成，因听闻「**猎刃**」在死寂城才不走——猎刃=三位传说铁匠中**矮人王**所造，神教重金委托、历任首席猎人武器。今日在大教堂找到猎刃，**只剩刀柄**（大主教与强敌厮杀中破碎）。（48卷45章）

## 关系

- [苏晓](../苏晓.md)：对视数秒后她上楼、不交手；猎人公会印徽因猎神者称号被他认出。
- [大主教](大主教.md)：猎刃原持有者/使用者（刃碎于厮杀）。
- [鬼老头](鬼老头.md)：她从对方处听闻猎刃在死寂城。
