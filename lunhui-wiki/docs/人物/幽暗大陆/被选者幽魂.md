# 被选者幽魂

大教堂 11 层的一位黑红色**强大幽魂**——他是曾**深入根源·死寂城、重伤而归的前被选者**，死后因灵魂力量强大，魂体一直留存至今（在大教堂 11 层不知待了两百多年，无聊得很）。⚠ 大教堂 11 层的长廊两侧金属门里，存着历代**失败被选者**的骨灰或遗骸（找不回的用武器 / 贴身物替代）——他们都是一个个时代的最强者，却在深入死寂后惨败、乃至再没回来。这位幽魂便是其中魂体独存的一位，精通「**魂圣痕**」（以魂丝操控为核心的灵魂系能力）。

## 收下莉斯当徒弟

⚠ 幽魂本想收 [苏晓](../苏晓.md) 为徒，掂量后作罢（「我们谁教谁，我有点拿不准」）。苏晓便把 [莉斯](莉斯.md) 推荐给他——起初他嫌莉斯不太适合魂圣痕，直到听说「她家很热闹:一只恶灵、一个天外生物（小花花）、还有古铜镜危险物」，顿时来了兴趣（惊叹莉斯竟能在这么热闹的家里活下来）。⚠ 苏晓的安排很精妙:把这位治愈教会前辈幽魂送去莉斯家当师父，正好**制约**莉斯家那三名危险房客（古老魔镜、[镜中恶灵](镜中恶灵.md)、小花花），让它们互相牵制不出问题；也算给「今后只能住治疗院宿舍」的莉斯一个补偿——这样的师父可是许多人梦寐以求的。莉斯家由此凑齐「四名房客」。

## 关系

- [苏晓](../苏晓.md)：本想拜他为徒、又给他安排住处与徒弟的被选者（灭法者）。
- [莉斯](莉斯.md)：他收下的徒弟（虽起初嫌她不适合魂圣痕）。
- [镜中恶灵](镜中恶灵.md) / 古老魔镜 / 小花花：与他同住莉斯家、被他制约的三名危险房客。
