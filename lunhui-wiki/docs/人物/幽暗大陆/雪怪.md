# 雪怪

编号 17＊＊08 的**死亡乐园契约者**，一个看着膀大腰圆、却把「**见风使舵、欺软怕硬**」发挥到淋漓尽致的家伙。他叼着雪茄、一口能吸半根还不吐一点烟（让人怀疑他上辈子是台抽油烟机）。他看着没什么特殊，却「以他人想不到的方式」活到现在——就他那嘴贱程度还没被打死，也算奇迹（曾在世界联络平台骂 [豪妹](../塞尔星/豪妹.md) 被捶得半死）。⚠ 连他的**气息都只有半个**，却依旧跑得飞快——能在有 [凯因](../潘多拉星/凯因.md) 与 [鹿格](../树生世界/鹿格.md) 的小队里活到现在，雪怪显然也有特殊本领，这小队真是人才辈出。

## 凯因队的一员 · 被审讯放走

⚠ 雪怪与 [鹿格](../树生世界/鹿格.md) 在上个世界潘多拉星就与 [凯因](../潘多拉星/凯因.md) 因各种原因臭味相投组队，是「看谁先死」的坑队友组合。三人凭 [机械公爵](机械公爵.md) 提供的庇护石进死寂城、又从一条隐秘路线直达内城，本是来圣歌团宝库搜刮，却撞上刚战胜圣歌团的 [苏晓](../苏晓.md)。被倒吊审讯时，[咕噜](../咕噜.md) 拿晶体飞刀、又用「**仁慈之刺**」（让嘴硬敌人变得好交涉的炼金酷刑道具）扎他一下，他这八阶契约者当场「我服了，我全说」。他如实交代了公爵知道秘密通路、宝库及进宝库特殊方式等要害情报。⚠ 苏晓放他走，他还识相地交出仅剩的 4950 灵魂钱币买命（拜天启三姐妹活下来的先例所赐，他才敢信苏晓收钱不灭口）；出宝库撞见满地教会骑士又缩回来，最后靠雪怪的圆盘机关开空间通道逃走，公爵引爆后他半个气息仍跑得飞快。

## 关系

- [凯因](../潘多拉星/凯因.md) / [鹿格](../树生世界/鹿格.md)：与他臭味相投、「看谁先死」的坑队友二人（同队从潘多拉星到死寂城）。
- [机械公爵](机械公爵.md)：给他们庇护石、领他们进死寂城、又拿他们当替死鬼的合作者。
- [苏晓](../苏晓.md)：审讯、放走、还宰了他 4950 灵魂钱币的灭法者。
- [咕噜](../咕噜.md)：拿仁慈之刺扎他逼供、让他「全说」的暗杀系。

## ☠ 逃不过的一刀 · 初始神殿的瓜葛（48卷35章）

⚠ 好队友清场公爵队时，[凯撒](../尼古拉斯·凯撒.md) 的臭鞋炸弹把众人熏出，雪怪照例撒腿就逃——他以为自己不是追杀首选（教会石板在公爵、凯因又太强），却没料到「好队友」的规矩是**该拼命时不含糊、要捏就挑软柿子捏**。[罪亚斯](../画中世界/罪亚斯.md) 追上这个「扮猪吃虎、生存力极强」的家伙:以不灭特性专治这类难缠敌人，一刀斩下他的头颅。⚠ 罪亚斯事后提及雪怪与「**初始神殿**」（几个高位邪神组建的势力）有瓜葛、自己似乎被这势力「标记」了——[苏晓](../苏晓.md) 却宽他的心:初始神殿那几名柱神「不是死了，就是被我带回去当食材」，不会有人来报复。
