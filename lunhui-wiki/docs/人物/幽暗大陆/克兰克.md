# 贵公子·克兰克

[机械公爵](机械公爵.md) 的**长子**——27 岁、未婚，天赋普通却头脑过人，17 岁便凭才智在**蒸汽神教**身居要职。在「以强者为尊」的蒸汽神教里，一个超凡资质平庸却又极优秀的公爵长子，简直是矛盾的结合体。⚠ 他最特别之处在于:**对财富、美色、地位一概无感**——一切在他眼里都平淡如水。

## 苏晓选定的「世界之子胚子」

⚠ 正因克兰克对一切都索然无味，[苏晓](../苏晓.md) 看中了他:准备用【吞噬者·黑A】+【背叛者意志】+【世界三件套】，把他培养成一名**世界之子**当火力吸引器（在前面吸引神祭日惊变的火力）。「背叛者意志」的传承者在背叛他人时会产生难以想象的愉悦——对克兰克这种万事平淡之人而言，一旦尝到这种愉悦，绝对会沉迷其中。苏晓因此调侃:一出「父慈子孝」的戏码就要上演了，正在机械公爵手下做事的贵公子，搞不好就要变成「戴孝子·克兰克」。（这也算是回敬机械公爵今晚的趁火打劫。）

## 关系

- [苏晓](../苏晓.md)：选定他做世界之子胚子、要把他培养成火力吸引器的灭法者。

## 🩸 被改造成世界之子 · 六重变强 buff（48卷5–6章）

⚠ 苏晓在音乐厅把演奏钢琴的克兰克麻醉带走，在治疗院地下据点给他做了「大手术」:①以炼金生物学让**吞噬者·黑A**融入其脏器系统共生（大幅提升消化 / 生存力）；②对昏迷的他用**背叛者意志**——他没死，即成**世界之子**；③把**世界之眼**与他双眼融合。可克兰克意志强得离谱、判断冷静到近乎冷血，反把黑A的意识**困在自己意识空间内**（历代吞噬者里只有四代太阳使徒能与他较量意志）——黑A此时更像苏晓留的「后手」（一旦克兰克与苏晓为敌就凶残爆发致其死地）+ 加速成长的「潜力映射」引擎。苏晓给他叠满**六重变强 buff**（脏器共生黑A / 圣焰炼金药剂套餐 / 黑A潜力映射 / 公爵长子优先资源 / 世界之子 / 世界猎手+世界之眼），目标是让他在几天内成为强世界之子、把【世界猎手】的世界之力储量从 1.5~2 盎司撑到 **50 盎司并装满**——这才是苏晓在本世界的最大预期收获。⚠ 妙处在于:背叛者意志让克兰克**每次背刺他人都情绪剧烈波动、触发世界之眼变强**。他成世界之子却没引来任何天地异象或世界意识青睐（幽暗大陆的异常）。术后他撞窗逃回蒸汽神教，苏晓一通电话让公爵还了 400 尾款、也算「改善父子关系」的开场。

## 🗡 觉醒 · 在蒸汽神教内部「杀到超神」（48卷14章）

⚠ 世界之子克兰克彻底觉醒——他把 [苏晓](../苏晓.md) 给的六重 buff + 背叛者意志用到极致，在**蒸汽神教内部一路背刺、杀到超神**:十位神使已有三位被他背刺致死、一人昏迷。妙的是他接管了怒锤机构、又以「查凶手」之名**追查他自己**（让凶手查凶手），演技超群到连父亲 [机械公爵](机械公爵.md) 都没想到这些事是长子干的。他的姐姐 [克萝](克萝.md)（公爵长女）发现他成了自己的「同类」（都是被公爵改造胚胎的半成品）——两人明争暗斗、要在弑父前先分胜负，克萝已慌到暗中联络苏晓求联手除他。⚠ 蒸汽神教随 [机械公爵](机械公爵.md) 站到「保守派」（[大贤者·图尔兹](大贤者·图尔兹.md) 阵营）、与苏晓敌对——但苏晓早留了这枚「带孝子」克兰克，让蒸汽神教内部越乱越好。

## 🦊 吞妹伪装 · 怂了求别追杀（48卷18章）

⚠ 觉醒后的克兰克成了「狐狸」——他从一开始就没想反扑那只盯着他的「血兽」（苏晓），目的只有一个:**逃**。趁野兽大师入城、苏晓收拾施法者的空档，他吞噬了妹妹 [克萝](克萝.md)、**伪装成克萝**（他细胞与克萝同样有复制分裂性）留在试验所迷惑苏晓；还**复刻出另一个自己**骗过黑A的共生、把复刻体化成液态浓缩细胞把黑A困在容器里（纯个人天赋、无法复制）——甚至硬生生把【背叛者意志】从自己身上剥离出来（苏晓都没想到他这么是个人才）。⚠ 他没被力量迷惑，做出巨大割舍:把狩猎所得的 **62.57 盎司世界之力 + 世界三件套（世界之眼已进化 3 次）+ 背叛者意志 + 珍藏的圣歌团徽章**全留在合金箱里，附一封「彩虹马屁 + 我怂了求你别追杀 + 用徽章换我妹妹的命」的信——他对苏晓惧怕到有阴影。可他不知道:苏晓有【老猎人】称号（凭血跨世界追踪），随时能找到他。苏晓放他一马（今后或能合作、逼急了反成死敌），他则继续以「克萝」身份躬身「白夜先生您慢走」——真正的克萝早被他吞噬。苏晓感觉如今的克兰克已能和 [机械公爵](机械公爵.md) 拼一下。

## 🩸 死寂城 · 父子对决落下风 · 三人组一员（48卷28章）

⚠ 克兰克也进了死寂城:在死寂城入口打开后，他与父亲 [机械公爵](机械公爵.md) 上演了一幕「父慈子孝」的**父子对决**——从他后来现身的模样（左眼电子眼、右眼放射状瞳孔、双臂皆机械义体）看，他**落了下风**。⚠ 此后他与畸变的 [月光侍女](月光侍女.md)、[乌鸦女](../画中世界/乌鸦女.md) 组成进城三队人里最弱的**三人组**:月光侍女纯粹看热闹、乌鸦女想摆脱死灵之书、而克兰克**时刻盼望父亲公爵暴毙**。他躲在诞生圣所暗门后，等苏晓离开才与乌鸦女一同现身——父子相残的执念，仍是他进死寂城的主线。

## 💀 躯体被父夺占 · 一具身躯里的一家人（48卷35–36章）

⚠ 那场「父慈子孝」的父子对决，其实是公爵在**为自己挑选载体**——公爵故意让克兰克活下来、还摧毁他半边身体逼他换上机械义体，正是要一具「高契合 + 无炼金合成物 + 经机械改造」的躯壳。斩杀 [钢铁使徒](钢铁使徒.md)（占公爵旧躯送死）后，[机械公爵](机械公爵.md) 的灵魂便**夺占了克兰克这具身躯**:乌鸦队里的「克兰克」左眼瞳孔紧缩、垂头再抬头目光深邃，已是公爵主导。⚠ 更离奇的是这具躯体的来历——它原是公爵**妻子**的身躯（妻子因公爵改造未出生的子嗣心灰意冷、把身躯遗送给女儿 [克萝](克萝.md) 并保留灵魂）;克兰克当初为逃苏晓追杀夺下它时，发现妹妹克萝与母亲的灵魂都在;如今公爵再夺，**长子克兰克、长女克萝、妻子三人的灵魂竟齐聚在同一具躯体的意识空间里**。公爵由此起了「把一家人灵魂都具量重塑、团聚重生」的念头——克兰克是否还活着尚不可知，但已不再是这具身躯的主导。
