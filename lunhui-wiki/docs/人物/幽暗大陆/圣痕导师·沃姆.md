# 圣痕导师·沃姆

学术派的一名**圣痕导师**——身形瘦到皮包骨、以绷带缠身，目光犀利咄咄逼人。他曾因理念与 [大贤者·图尔兹](大贤者·图尔兹.md) 截然相反而长期敌对、被囚困;⚠ 图尔兹舍命重创罪神而死后，高墙城陷入混乱，本该由 [烟夫人](烟夫人·阿娜丝.md) 接任领袖，学术派新晋导师却**放出了被囚的沃姆**（图尔兹手下的老导师们都随他战死罪神了）。

## 与图尔兹截然相反的路线

⚠ 图尔兹主张**死死封住死寂城、废除被选者传统**（认定初代圣女、狼骑士、圣歌团都不可信，这么多年死寂城问题依旧没解决，就该换新方向）。沃姆则是另一极端:主张**继续深入死寂城**，找齐四块**教会石板**，以上面记载的**五大初始圣痕 + 神灵印记**壮大战力、并用神灵印记**封临出新的半神**;在半神率领下不断向死寂城发起挑战——「将死寂城封禁看似能得一时安宁，但死寂侵蚀掉封禁是早晚的事」。他没留在高墙城当领袖，而是带上愿追随的几名强者、用城中剩余的庇护石来到死寂城内城，自成「**沃姆队**」（五人白袍兜帽）。他执着于教会石板，因手中已有一块。

## 被苏晓设局 · 结死仇

⚠ 苏晓以先古面具**伪装成公爵**，联合乌鸦队、沃姆队谎称「三队合拼四块石板」。沃姆本已交代:四块拼齐立即动手抢——圣痕与神灵印记无法复刻，只有掌握完整石板才有意义、绝无共享可能。可拼合瞬间，苏晓召来的**死灵之书**卷走四块石板（假意没入乌鸦女体内、实则进苏晓储存空间）。混乱中苏晓（伪装公爵）一拳抡杀沃姆一名新晋导师、血肉碎骨飞射沃姆（被他单手静止），当着他的面把仇结死——这断绝了公爵日后联合沃姆的可能。沃姆一时猜不透谁是真赢家，最终「谁跑谁心虚」、去追杀逃走的乌鸦队。

## 关系

- [大贤者·图尔兹](大贤者·图尔兹.md)：他的死对头、理念截然相反（图尔兹封城废被选者、沃姆封临半神强攻死寂城）。
- [苏晓](../苏晓.md)：伪装成公爵、当着他面杀其部下把仇结死、还夺走全部教会石板的灭法者。
- [乌鸦女](../画中世界/乌鸦女.md) / [月光侍女](月光侍女.md)：被他与「公爵」两方联手时又生变故、最终被他追杀的乌鸦队。
- [烟夫人](烟夫人·阿娜丝.md)：本该接任高墙城领袖、却因学术派放出沃姆而生变的高墙议会强者。
