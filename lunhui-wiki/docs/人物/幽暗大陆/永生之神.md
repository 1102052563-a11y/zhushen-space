# 永生之神

**治愈教会**信奉的「**古老神灵**」——本世界「古老神灵」不是一类神，而**仅指永生之神**这一位。传说古时代信奉这位神祇即可永生；更实在的说法是，是它的庇护驱走了城内的死寂、让土地重新肥沃、淡水甘甜、果实丰盈，[高墙城](../../世界/任务世界/幽暗大陆-死寂根源之地.md) 人口才从灾难时的十几万恢复到超千万。⚠ 在高墙城可以不信任何势力，但**绝不能对永生之神有半点不敬**（曾有人自称神使，当晚就被四方势力联手送上火刑架）。它平日被封于中心广场那座十几米高的石像内——石像看起来并不和善，反而像人立而起的**半人半兽**存在。

## 神祭日的复活与自封

⚠ 神祭日惊变中，[瓦迪家族](../../世界/任务世界/幽暗大陆-死寂根源之地.md) 窃取永生之神的部分神力（用来召唤界外存在），并突降**血雨**试图打破它的封印。石像当着所有人的面活了过来、仰天咆哮——它散发的**混沌恶意狂乱暴戾到很多古神都无法匹敌**，[苏晓](../苏晓.md) 隔几百米都被那种狂乱影响，直言这是他见过最狂乱暴戾的一位神灵系。可它并未挣脱脱困:反而**双手撕开自己的胸膛、化为大片金色血珠**倒卷上天——这不是攻击，而是它以残存的意识力量**重新关上了被人打破的封禁**。[机械公爵](机械公爵.md) 因此想起祖父那句话:「如果没有神灵，我们早就成了徘徊在死寂中的躯壳。」它与「死寂」的深层关系（大主教「别追求永生，和永生相对的是死寂」的点拨）仍是本世界的核心谜团。

## 关系

- 治愈教会：以它之名立教、世代供奉它、以它庇护立国的势力。
- 瓦迪家族：窃取它部分神力召唤界外存在、并企图打破其封印的高墙城势力。
- [苏晓](../苏晓.md)：隔空感受到它狂乱暴戾、断言它是所见最狂暴神灵系的灭法者（古神猎人）。
