# 野兽大师

墙外恶土**野兽族**中威望极高的一位——一只会口吐人言、披挂老旧野兽族布饰的苍老老狼。野兽族一向排外，外人踏入其地盘都会遭围攻，是 [被选者幽魂](被选者幽魂.md)（生前是屠尽野兽族六七成的「杀神」远征大队长）带着莉斯家三名危险房客「请」他来的。⚠ 他掌握一种**以灵魂力量增益冥想效率**的独特冥想秘术——灵魂越强、冥想效率越高——本是他借鉴残本古籍、苦研十几年改良而成。

## 「谁修行谁死」的原始秘术

⚠ 外界传这秘术是野兽大师所创，实则他只是**改良者**:这秘术有原始版本【**魂之书·灵魂印记**】（神灵时代之物、藏于灵魂书库顶层），效果强到离谱、上手简单，最大弊病是「**谁修行谁死**」（早上看、上午练、下午凉、傍晚入土、全村吃席）——野兽族亲测多次、修一个死一个，久而久之无人问津。野兽大师年轻时是天才，把它改良成安全可练的删减版（冥想效率 +5~8 倍），代价是威力大减。⚠ [苏晓](../苏晓.md) 靠乐园权限看出:原始秘术的死亡门槛其实是**灵魂强度 600 点以上 + 灵魂能量阶位 6**（他 650 点灵魂强度 + 阶位 7，正好能安全修行）。苏晓以从龙学院所得的《兽之灵魂》等三本古籍，从野兽大师手里换来【魂之书·灵魂印记】卷轴——野兽大师更需要能荫庇部族的《兽之灵魂》，成交得干脆，苏晓反倒血赚到良心稍痛。
