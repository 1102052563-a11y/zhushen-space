# 神女

**圣女一脉**的当代继承者——[圣祭祀](圣祭祀.md) 的后裔。高墙城的圣女允许婚配（结婚后称呼从「圣女」变「神女」、待女儿成年再传回「圣女」），这位 32 岁的成熟女士已**离过三次婚**、多情之名满城皆知，却仍有众多追求者。⚠ 她其实是普通人——远远看过 [苏晓](../苏晓.md) 一眼就被那无形血兽吓得心底颤栗、想逃；被绑到苏晓面前时，一句「你知道我是谁吗」出口就后悔了。

## 心脏 · 死寂城的门锁

⚠ 苏晓用「私奔」为由（让 [休司](休司.md) 与她共餐、[烟夫人](烟夫人·阿娜丝.md) 引荐、再半途劫走），把她绑进治疗院。面对「仁慈之刺」的和风细雨式威胁，她比严刑拷打更受不了，全盘托出**死寂城入口的开门法**:当初治愈教会封住死寂城时，初代圣女以自身全部**次级神血**化作封禁之锁，唯有与其**血脉共鸣者**才能临时打开——⚠ 而她**还在跳动的心脏**正是那把钥匙，只需带她到入口处即可。（圣所钥匙则是开死寂城内部一处要地，非开这扇门。）她一度在护卫帮助下逃出治疗院，尝过炼金慢毒后又被送回、乖乖住下不逃了。

## 关系

- [圣祭祀](圣祭祀.md)：她的先祖（圣女一脉之祖）——大主教特意请苏晓「别动圣女一脉」。
- [苏晓](../苏晓.md)：绑她、逼问出死寂城开门法、令她心脏成开门钥匙的治疗院副院长。
- [休司](休司.md)：被苏晓派去接近她、外界宣称「与她私奔」的哑巴空间系少年。
- [烟夫人·阿娜丝](烟夫人·阿娜丝.md)：牵线引荐、把她带到劫持地点的高墙议会战力。
