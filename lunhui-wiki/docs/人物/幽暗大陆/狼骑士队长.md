# 狼骑士队长

死寂城**狼冢**里、坐在高大坟墓前石碑下的一道银黑全身甲身影——**最后的狼骑士**、狼骑士队的**队长**。他的铠甲原是象征月光的银色，被深渊侵蚀成银黑、凹凸不平；护手末端镶着 [苏晓](../苏晓.md) 要找的**源石**。他传承了**银·月狼的力量乃至狼血**（[苏晓](../苏晓.md) 的部下 [玛丽娜](玛丽娜.md) 体内那微量狼血，同源于此）。

## 狼冢的真相 · 镇守深渊通道

⚠ 圣歌团的成立是为对抗死寂，**狼骑士队的成立则是镇守此地的深渊通道**——狼骑士背后的高大坟墓只是掩饰，里面**封禁的深渊通道**才是重点。因没有彻底关闭深渊通道的方法，周边才被侵蚀成一片黑色遗存、街道铺满死之民与树蚀的骸骨。狼骑士队一般有 **15 名成员 + 1 名队长**，其余皆已亡故，只剩这位无理智的队长，无时无刻不被死寂与深渊双重侵袭。银·月狼是**深渊的监守**（任何遭深渊侵袭的世界都是它们要去的地方），是灭法世代的血誓盟友——狼骑士队正是这一使命的传承。

## 恐怖的狼剑术

⚠ 单挑的话，狼骑士队长**能完虐任何一名圣歌团成员**，要圣歌团三十圣心「聚合为一」才能与队长拼。他的**狼剑术**毫不花哨、简单纯粹却强悍到不讲道理:

- **越斩越强**:第一剑威力若为 10，第二剑 13~15、第三剑突破 20、第四剑起进入**持续强霸体**（应对之法唯有一直保持压制、绝不能让他斩起来）。
- **群攻矫健、单挑如困兽**:群战时敏锐迅捷，单独迎敌则如穷途困兽、不退半步而将敌人斩退。
- **穿透空间**（与苏晓同款瞬移）、徒手捏碎奥义（一把捏爆咕噜融合 3 不朽 + 8 圣灵短刀的「极限锋刃」）、狼嚎斩杀技（黑蓝烟气·似魔刃）。
- 因**灭法与银·月狼的血誓**，被深渊侵袭到无理智的他，感知中仍不把苏晓完全算作敌人——故苏晓走到他十几米前都无事，反倒是踏入环墙的 [咕噜](../咕噜.md) 一进场就被他瞬移一剑刺穿腹部举起。

## 死战 · 被斩

⚠ 苏晓深知此战 **10 分钟内不结束就是自己的葬身之地**。他先借 [巴哈](../布布汪.md) 异空间潜入坟墓、破开深渊通道封印把**先古面具**丢进去（先古面具狂吸深渊能量、切断队长与深渊通道的连接，否则杀不死他）。随后死战:弃掉不朽 +10 的狂猎之夜（在狼剑下如纸），以刃道刀·极对斩、刃道刀·血爆炸碎其左臂（很伤刀·每世界限用两次），[咕噜](../咕噜.md) 用团长给的起源级道具（金色纹印大幅减伤）挡下狼嚎斩杀技。最终两人各以武器贯穿对方（苏晓胸腹被狼大剑刺穿、狼骑士胸膛被斩龙闪贯穿），苏晓徒手把肝脏塞回胸膛、血烟炮 + 青鬼连轰，一刀刺穿队长头颅——⚠**「最后的狼骑士，已斩」**。掉落 15.72% 世界之源 + 黄金技能点 + **狼骑士徽章** + 深渊宝箱，源石被苏晓从剑柄抠下。

## 关系

- [苏晓](../苏晓.md)：以死战险胜、斩其头颅、夺其源石与狼骑士徽章的灭法被选者（因血誓一度不被他视作敌人）。
- [咕噜](../咕噜.md)：一进环墙就被他瞬移一剑刺穿腹部的暗杀系倒霉蛋。
- [玛丽娜](玛丽娜.md)：与他同源「银·月狼狼血」的苏晓部下（微量狼血）。
- 银·月狼（月光之主）：他力量与狼血的源头、灭法的血誓盟主、深渊的监守。
