# 月光侍女

死寂城内城**圣十教堂**右侧门后「**诞生圣所**」中，一根顶到天棚的玻璃柱里、泡在暖白色溶液中的银白长发女人。她皮肤白皙娇嫩、一双琥珀色眸子让人下意识心生好感——这是**月光侍女**，治愈教会人工创造、供被选者依仗的**顶级治疗遗产**。诞生圣所里还吊着几千个几米高的玻璃罐（内是骸骨）、两侧高台盛放大量眼球手臂，唯她这根玻璃柱最特殊。

## 治愈教会的救命底牌

⚠ 用大主教的原话:只要被选者**还没死、并回到月光侍女附近**，无论受多重的伤，月光侍女都能替他抢救一下。没有她，被选者的死寂城之路会更艰难、甚至不可能完成。她是治愈教会留下的重要遗产——[大主教](大主教.md) 特意嘱咐 [苏晓](../苏晓.md) 进城后要看她是否还在。

## 从无情到畸变

⚠ 最初的月光侍女没有情感（治愈教会不会给她这多余的东西），却给了她**智慧**——有了智慧，情感便如雨后嫩芽破土。她在一代代帮助被选者时，难免被这些「每个时代的最强者 / 领袖」的人格魅力吸引，其中有些成了她的朋友、更多是她所尊敬之人。可这些被选者**九成九都战死**，只有寥寥几个走出死寂城、且再也没回来。不断失去朋友 + 孤身一人的漫长等待，终于让她**从内心开始畸变**，继而身体畸变:脸颊笑着撕裂开、露出一直裂到耳下的嘴与满口交错尖牙、指尖探出利爪（可轻易在玻璃柱内壁划出裂痕）、琥珀色竖瞳。她之所以不挣脱封印她的容器，只是不想和外面的「血气怪」厮杀。

## 灭法的血誓相护

⚠ 她从灵魂书库的书籍上认得「灭法」——一眼看破苏晓也是灭法、是**月光之主（银·月狼）的血誓盟友**。她说自己拥有的**月光力量在抗拒与苏晓为敌**（「这就是血誓吗，真奇妙」）。她感慨苏晓「来晚了几百年」——几百年前她还只是月光侍女时，见到他一定会说「被选者大人，欢迎您的到来」。⚠ 苏晓循大主教指引来找她当治疗后盾，却在她指向放她出来的金属拉杆时，一发血烟炮炸毁拉杆机关、**拒绝放她**（不与这邪门畸变生灵纠缠）。她转而指出斜后方超大玻璃罐里的**半成品**——[灰色侍女](灰色侍女.md)，苏晓遂带走了那个「需要被治疗的治疗者」。

## 关系

- [苏晓](../苏晓.md)：她认作「月光之主血誓盟友」的灭法被选者——来晚几百年、还拒绝放她出来。
- [灰色侍女](灰色侍女.md)：她指给苏晓的「半成品」、月光圣女的最初版（被苏晓带走当治疗）。
- [大主教](大主教.md)：造她、留她当被选者救命底牌的治愈教会掌权者（她已不再是教会成员）。
- 月光之主（银·月狼）：她月光力量的源头、灭法的血誓盟主——血誓令她无法与苏晓为敌。
