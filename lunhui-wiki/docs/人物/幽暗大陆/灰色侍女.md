# 灰色侍女

一名身穿灰色长袍、戴银色面具、侧坐在超大号玻璃罐内的身影——她是治愈教会制成的**半成品**，或者说是**月光圣女的最初版**。是 [月光侍女](月光侍女.md) 指给 [苏晓](../苏晓.md) 的（「那里有个半成品，她的生命力可真顽强，明明是个半成品」）。苏晓打碎玻璃罐把她拎出时，她气息已很微弱——本想找个强力治疗者，结果找到个**需要被治疗的治疗者**。

## 半成品的顶尖治疗

⚠ 苏晓把她安置在大教堂、注射浓缩生命力溶液救回后，她反过来成了破晓队的救命恩人。她**不能言语**，双手叠于小腹前对人躬身施礼；能力低调到「不仔细感知很容易忽略她的存在」。她的治疗过程**动作笨拙**（用镊钳生拔苏晓左臂血肉里填充的晶体与灵影线、笨到痛觉神经快到极限），却**治疗能力不比月光侍女差**——同样是「只要还有一口气，就能抢救回来」。或许正因这份笨拙，她才被旧教会判定为「半成品」。

## 斩狼骑士后的救场

⚠ 苏晓与 [狼骑士队长](狼骑士队长.md) 死战重伤昏迷、遭深渊能量轻度侵蚀，是灰色侍女守在石床边:一手托着苏晓右臂输入温和能量修复脏器，又用镊钳拔他左臂的晶体填充物（她的银面具底部因此被黑色侵染=吸纳了苏晓体内的深渊能量）。她接连治好苏晓、重伤的 [巴哈](../布布汪.md)（缠满绷带蹲吊灯上思考鸟生）、[咕噜](../咕噜.md)（绷带 + 右臂石膏）。苏晓事后感叹:比起没找来的月光侍女，能带回**灰色侍女**其实更省心（少很多后顾之忧）——先去圣十教堂是正确选择。

## 关系

- [苏晓](../苏晓.md)：把她从玻璃罐带出救活、又被她反过来治愈重伤的灭法被选者。
- [月光侍女](月光侍女.md)：她的「成品」版、指给苏晓的畸变前辈（同为治愈教会诞生圣所的治疗造物）。
- [巴哈 / 布布汪](../布布汪.md)：她一并治好的、斩狼骑士队长后重伤的苏晓随从。
