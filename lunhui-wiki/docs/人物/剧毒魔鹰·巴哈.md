# 剧毒魔鹰·巴哈（暗杀系炼金生物·第四从者）

[苏晓](苏晓.md) 在黑王座大陆通关后、用银洛凑的 14 种史诗级材料 + 黑王世界所得人造生物核心 [血肉礼堂·巴哈](../力量体系/装备道具/index.md) 重做出的**第四个炼金生物**（继 [布布汪](布布汪.md) / [阿姆](阿姆.md) / [贝妮](狩之影.md) 后）——定位「炮灰 + 暗杀」却因核心是成品而智商高（不低于人类）。八十公分高的**黑蓝色鹰 / 秃鹫结合体**，四爪如钩刃（后爪中空可注毒）。**性格跳脱话痨**、几分钟就和布布汪这老司机厮混到一起、对阿姆有天生好感（前世搭档是小肉山·与阿姆体型 / 沉默相近）。

!!! note "缺陷 · 语言能力时好时坏"
    核心阵图有部分损坏（苏晓只修好大半）——**语言能力时灵时不灵**：刚冒出一句骚话（「食屎啦你」「啦啦啦蓝天」）就又变回「嘎嘎嘎」。苏晓觉得它不能说话最好（话痨）；实战中若突然恢复语音就是个把敌人骂到心态崩的大喷子（双马尾少女即被它骚扰到心态炸）。

## 属性 / 技能（二十六卷66章）

- **存在时间** 656 自然日；力 80 / 敏 **125**（真实）/ 体 85（真实）/ 智 27 / 魅 45；法力 = 毒能量 3770。
- **钢躯铁羽**（被动）：生命 +1800 / 法力 +3500 / 羽护甲 +42。
- **空中突袭**（主动）：超音速俯冲·敏 ×3 + 130 真实毒伤。
- **魔鹰**（被动）：可**无声飞行**（降速 60% + 屏蔽大部分感知）。
- **背袭 + 断喉**（被动 / 主动）：背后突袭真实伤害 + 用双爪扯开敌喉（200~700 要害伤 + 毒 + 失血）。
- **剧毒影**（终极·被动·可成长）：每击附 130 真实毒伤、可一次性灌 2300 真实毒伤（虚弱 1~5 秒）；**摄取高阶生物毒素可升级**（现附加：致死 / 麻痹 / 真实伤害）。
- **舍命忠诚**（被动 MAX）：对创造者绝对忠诚、愿战死（前世即为创造者战死）。

## 乐园竞技场首战（二十六卷68–69章）

苏晓首次让全从者上阵打**乐园竞技场**——「召唤师 + 狙击手」战术：阿姆正面突进、布布汪隐身断敌能力、贝妮侧翼、**巴哈高空待机从背后暗杀**、苏晓远程 [D·暗杀](../力量体系/装备道具/index.md) / [噬灭](../力量体系/装备道具/index.md) 点射。巴哈一爪扯喉 + 挖眼秒掉死亡乐园格乌（「敢打我们的颜值代表，食屎啦你」）。靠这套苏晓**69 连胜·排名 47305 → 159 名**（冲击前五十的「强者争霸战」资格）。

## 进化「猎空魔鹰」（三十卷26章·空之血脉）

巫师大陆里，苏晓用斩支配者所得的 [源血·空](../力量体系/装备道具/index.md)（剔除恶神性的古神血统）+ **不朽契约**（签为第二从者）把巴哈进化成 **猎空魔鹰·巴哈**（职业「空之血脉」）——从「会暗杀的吉祥物」蜕变成真正的暗杀者：新增**空之魔鹰**（无冷却空间穿梭·来无影去无踪）+ **断喉**（背袭后双爪扯喉 + 注毒 + 重度失血）+ **魔鹰领域**（六根空间之羽全碎·封锁周边 1000 米六阶空间能力 6 分钟）；剧毒影附「致死 / 麻痹 / 真实伤害」。**最关键：存在时限消失**（彻底成独立生物·不再受召唤时限约束）。弑支配者一战即靠空间穿梭 + 断喉 + 腚后雷（磁爆猎手）立功。

## 空间专精反杀西海王（四十一卷40–41章）

[魔海](../世界/任务世界/魔海-厄运与诅咒.md) 里 [西海王](魔海/西海王.md) 抛银币把苏晓拉进「决斗异空间」想单挑——却踩在巴哈的专精上：巴哈用「基础空间公式」把整队 [诅咒人](魔海/厄运号诅咒人群像.md) 一起放进异空间、再封死出口（「决斗=单挑=你单挑我们所有人」），把西海王的决斗秘宝反变成他自己的牢笼。它一边用空间穿梭在西海王（对它仇恨值近千·必杀那只鸟）的爪下自由进出、一边看戏诛心（「西海王，你这人头送的，牛哔啊」）。这一战靠它的空间封锁 + 诅咒人配合，两小时耗死了「越伤越强、不死般」的西海王。

## 🆕 噩梦·永望镇的现实帮手(44卷51–54章)

苏晓精神体进噩梦时，它守在苏晓肉身旁、与 [布布汪](布布汪.md) 组成「平行推进」的现实端:凭它魔鹰的目力捕捉噩梦里苏晓用灰笔写下的米粒小点、抛闪光弹扫描全镇找出蜈蚣浮雕并抓碎、拽下肿胀之眼地道的铁门。它还调皮地给布布汪喷制冷喷雾「加智力 buff」。这一战它的战力已隐隐向苏晓小队第二把交椅靠拢。

## 🦅 绝望试炼 —— 新的「吞心之恶」（54卷157–165章）

⚠ **⚠ 「绝望试炼」是万族界超脱后 SSSS 结算给出的从者权限**——⚠ **苏晓无法参与，只有布布汪、阿姆、巴哈能上**（⚠ 贝妮战力太弱被权限默认禁止，棘拉则因整体战力过于超极限也被禁止）;⚠ **⚠ 死亡率**:⚠ **阿姆 23.59% · 巴哈 76.562% · 布布汪 99.85%**——⚠ **试炼持续8～10天:通过者说是脱胎换骨也不夸张，通不过，必死**;⚠ **⚠ 「老大，我去。」**——⚠ **巴哈落在阿姆肩头开口，那双透出暗蓝的鹰眼中，是和以往不一样的神采**。

- ⚠ **⚠ 为什么是它**:⚠ **对战祖神时，布布汪堪称辅助MVP〔最后硬控祖神3秒、在这种级别的战斗中说是决胜点也毫不夸张〕，且它的贡献更多体现在平时〔进入众神之界没几小时就把本世界的「网络集群」黑了，三大势力的各类资料都给弄出来〕**;⚠ **⚠ 阿姆更不用说:祖神的压箱底大招之一被它一口吞了，哪怕躺尸到战斗即将结束才站起来，可它那记蛮牛冲锋把祖神从王前大道一路顶回高座——祖神就是在那里被斩杀的**;⚠ **⚠ 反观巴哈，对战祖神中并非不够悍勇，而是过于激进就会被瞬秒**——⚠ **以巴哈的骄傲，别说失败率76.562%，就算99.99%它也有胆量去**;⚠ **⚠ 苏晓的选择**:⚠ **破晓队最重要的一条规则是「尊重每一名成员的选择」**——⚠ **只要他开口，巴哈一定会听从、不会去；但那也等于彻底打碎了巴哈的骄傲**（⚠ **没有了内心骄傲的巴哈，不会有多强的求生欲、会感到自身是破晓队的累赘**；后续某场危急到极点的死战中，必能看到它与敌人同归于尽的身影）;⚠ **⚠ 七小时后，从者烙印储存空间内装满各类超高品级药剂的巴哈，使用了「绝望试炼」**——⚠ **一张漆黑、遍布参差不齐尖牙的巨口出现，一口把它吞下**（⚠ 每名从者面对的试炼都不同;巴哈的试炼中，苏晓明确感应到了**古神体系的力量**——这很正常:巴哈的空间系力量有很大程度来自古神体系，只不过它并非古神系，而是通过吸收古神系的力量，获得了偏黑暗特性的**空间暗杀系**）。

| 项 | 内容 |
|---|---|
| 试炼评分 | ⚠ **SSS（E～SSS级）**——⚠ **舍命达成了本次试炼的最佳** |

> ⚠ **「魔鹰领域·吞心恶鬼（至强技能·Lv.EX）」:巴哈从此刻燃烧生命值，维持 1000×1000 米范围的魔鹰领域；在此领域内，巴哈将强行夺取敌人的一种能力作为自身能力使用。**
> - ⚠ **夺取能力后，巴哈的主属性〔敏捷〕将发生转变，但转变后的新主属性无法超过当前敏捷属性的最高值。**
> - ⚠ **如成功杀死敌人，巴哈将彻底吞噬此能力，从而永久性增强自身。**

（⚠ 对战强敌时堪称神技:**「巅峰能力」它一定夺不来，可夺取到一种「至强能力」也至关重要**，尤其是夺到那种能力体系中的小核心能力，必然让敌人很难受）;⚠ **⚠ 救场能力大幅提升**:⚠ **它的「魔鹰领域」更接近有实质的领域——散出钢铁鹰羽，在1000×1000米范围内定立几百个空间坐标**——⚠ **这意味着它能对领域内所有己方目标进行「空间拖拽」**（⚠ 举例:激战中阿姆重伤濒死，只要阿姆在领域内，巴哈就能瞬间把它拖入异空间，待恢复过来再重回战斗）。

- ⚠ **⚠ 归队**:⚠ **它在苏晓的炼金实验室玻璃柱内融合力量，布布汪、阿姆、贝妮都守在门外**——⚠ **一句「已经在融合力量」让三只明显放心了不少**（⚠ 试炼期间阿姆吃了几个大肉包就没胃口、布布汪该出去浪的时候也在家还时常叹气·⚠ 苏晓一句「没死」之后，阿姆立刻把整盘大肉包一手捞起全塞嘴里）;⚠ **⚠ 归队后的第一件事**:⚠ **和布布汪一起鬼鬼祟祟地鼓弄那把「功率位数多了个0」的电动牙刷**——⚠ **结果是巴哈头顶电动牙刷、布布汪头顶各类拆卸工具，一起在墙壁前面壁**。
