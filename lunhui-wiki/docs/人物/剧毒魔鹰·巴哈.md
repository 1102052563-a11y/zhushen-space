# 剧毒魔鹰·巴哈（暗杀系炼金生物·第四从者）

[苏晓](苏晓.md) 在黑王座大陆通关后、用银洛凑的 14 种史诗级材料 + 黑王世界所得人造生物核心 [血肉礼堂·巴哈](../力量体系/装备道具.md) 重做出的**第四个炼金生物**（继 [布布汪](布布汪.md) / [阿姆](阿姆.md) / [贝妮](狩之影.md) 后）——定位「炮灰 + 暗杀」却因核心是成品而智商高（不低于人类）。八十公分高的**黑蓝色鹰 / 秃鹫结合体**，四爪如钩刃（后爪中空可注毒）。**性格跳脱话痨**、几分钟就和布布汪这老司机厮混到一起、对阿姆有天生好感（前世搭档是小肉山·与阿姆体型 / 沉默相近）。

!!! note "缺陷 · 语言能力时好时坏"
    核心阵图有部分损坏（苏晓只修好大半）——**语言能力时灵时不灵**：刚冒出一句骚话（「食屎啦你」「啦啦啦蓝天」）就又变回「嘎嘎嘎」。苏晓觉得它不能说话最好（话痨）；实战中若突然恢复语音就是个把敌人骂到心态崩的大喷子（双马尾少女即被它骚扰到心态炸）。

## 属性 / 技能（二十六卷66章）

- **存在时间** 656 自然日；力 80 / 敏 **125**（真实）/ 体 85（真实）/ 智 27 / 魅 45；法力 = 毒能量 3770。
- **钢躯铁羽**（被动）：生命 +1800 / 法力 +3500 / 羽护甲 +42。
- **空中突袭**（主动）：超音速俯冲·敏 ×3 + 130 真实毒伤。
- **魔鹰**（被动）：可**无声飞行**（降速 60% + 屏蔽大部分感知）。
- **背袭 + 断喉**（被动 / 主动）：背后突袭真实伤害 + 用双爪扯开敌喉（200~700 要害伤 + 毒 + 失血）。
- **剧毒影**（终极·被动·可成长）：每击附 130 真实毒伤、可一次性灌 2300 真实毒伤（虚弱 1~5 秒）；**摄取高阶生物毒素可升级**（现附加：致死 / 麻痹 / 真实伤害）。
- **舍命忠诚**（被动 MAX）：对创造者绝对忠诚、愿战死（前世即为创造者战死）。

## 乐园竞技场首战（二十六卷68–69章）

苏晓首次让全从者上阵打**乐园竞技场**——「召唤师 + 狙击手」战术：阿姆正面突进、布布汪隐身断敌能力、贝妮侧翼、**巴哈高空待机从背后暗杀**、苏晓远程 [D·暗杀](../力量体系/装备道具.md) / [噬灭](../力量体系/装备道具.md) 点射。巴哈一爪扯喉 + 挖眼秒掉死亡乐园格乌（「敢打我们的颜值代表，食屎啦你」）。靠这套苏晓**69 连胜·排名 47305 → 159 名**（冲击前五十的「强者争霸战」资格）。

## 进化「猎空魔鹰」（三十卷26章·空之血脉）

巫师大陆里，苏晓用斩支配者所得的 [源血·空](../力量体系/装备道具.md)（剔除恶神性的古神血统）+ **不朽契约**（签为第二从者）把巴哈进化成 **猎空魔鹰·巴哈**（职业「空之血脉」）——从「会暗杀的吉祥物」蜕变成真正的暗杀者：新增**空之魔鹰**（无冷却空间穿梭·来无影去无踪）+ **断喉**（背袭后双爪扯喉 + 注毒 + 重度失血）+ **魔鹰领域**（六根空间之羽全碎·封锁周边 1000 米六阶空间能力 6 分钟）；剧毒影附「致死 / 麻痹 / 真实伤害」。**最关键：存在时限消失**（彻底成独立生物·不再受召唤时限约束）。弑支配者一战即靠空间穿梭 + 断喉 + 腚后雷（磁爆猎手）立功。

## 空间专精反杀西海王（四十一卷40–41章）

[魔海](../世界/任务世界/魔海-厄运与诅咒.md) 里 [西海王](魔海/西海王.md) 抛银币把苏晓拉进「决斗异空间」想单挑——却踩在巴哈的专精上：巴哈用「基础空间公式」把整队 [诅咒人](魔海/厄运号诅咒人群像.md) 一起放进异空间、再封死出口（「决斗=单挑=你单挑我们所有人」），把西海王的决斗秘宝反变成他自己的牢笼。它一边用空间穿梭在西海王（对它仇恨值近千·必杀那只鸟）的爪下自由进出、一边看戏诛心（「西海王，你这人头送的，牛哔啊」）。这一战靠它的空间封锁 + 诅咒人配合，两小时耗死了「越伤越强、不死般」的西海王。
