# 黑白

**黑白** 是 [东京喰种](../世界/任务世界/东京喰种.md) 世界中 [女王](女王.md) 三人女子冒险团的成员、敌方契约者（二卷19章）。

## 简介
一身紫色紧身衣、紫色长发扎成马尾、身材丰满的女性；背着一把至少一米长的重型狙击枪，是队中的**狙击手**（队伍主输出）。**极擅隐藏**——藏于暗处时几乎无人能察觉（不开口便难被发现）。性格爱调侃，常拿叶子的身材打趣（「飞机场」）。伏击苏晓前主动去寻找制高点（二卷19章）。

## 装备 · 战斗
- **绿色狙击辅助装备**：头顶一小片头甲 + 右眼一副放红光的精密探测装置，兼具**夜视 / 热能感知 / 超声波探测**，是狙击手的理想装备（亦为「探测类装备」一例）（二卷25章）。
- 700 米外开枪狙击苏晓，但苏晓凭变态感知（被瞄准处心口隐痛）一一**预判躲弹**，令她屡射不中、直呼「可恶的智力特长者」（二卷25章）。
- 伏击中被苏晓派去的剧情人物 [铃屋什造](铃屋什造.md) 缠住、无法继续支援女王（二卷27–28章）。

## 关系
- [女王](女王.md)——队长。
- [叶子](叶子.md)——队友，常调侃其身材。
- [苏晓](苏晓.md)（白夜）——伏击目标。
