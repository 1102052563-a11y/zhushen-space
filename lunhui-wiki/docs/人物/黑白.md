# 黑白

**黑白** 是 [东京喰种](../世界/任务世界/东京喰种.md) 世界中 [女王](女王.md) 三人女子冒险团的成员、敌方契约者（二卷19章）。

## 简介
一身紫色紧身衣、紫色长发扎成马尾、身材丰满的女性；背着一把至少一米长的重型狙击枪，是队中的**狙击手**。**极擅隐藏**——藏于暗处时几乎无人能察觉（不开口便难被发现）。性格爱调侃，常拿叶子的身材打趣（「飞机场」）。伏击苏晓前主动去寻找制高点（二卷19章）。

## 关系
- [女王](女王.md)——队长。
- [叶子](叶子.md)——队友，常调侃其身材。
- [苏晓](苏晓.md)（白夜）——伏击目标。
