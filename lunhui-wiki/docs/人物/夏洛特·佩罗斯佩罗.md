# 夏洛特·佩罗斯佩罗（斯佩罗 · 玲玲长子）

四皇 **夏洛特·玲玲的长子**——德雷斯罗萨海战中 BIG·MOM 海贼团一方的主将，奉母命来夺多弗朗明哥的人造恶魔果实技术。一根三米多高、如麻杆般干瘦的男人，舌头堪比蜥蜴。

- **舔舔果实（超人系）**：用恶魔果实造糖浆 / 棒棒糖形手杖，**糖浆硬度堪比钢铁**（覆武装色霸气更甚）、能瞬间修复、把敌人「**糖果化**」——唯一弱点是**怕火 / 高温**（遇热变软融化·火拳艾斯是其克星、不止一次把他按在地上锤）。
- **变态嗜好**：把敌人糖果化后**慢慢舔舐品尝**（人类血肉变糖果「味道不错」）。喜欢玩弄对手、嘴炮不断；作为玲玲长子战斗经验极丰富、有见闻色霸气（在苏晓出海面瞬间就感知到他）。

## 被苏晓三人组拖海溺杀（十八卷7–9章）

德雷斯罗萨海域，斯佩罗用糖浆把十艘残船粘成「小船岛」、与黑桃干部 [琵卡](琵卡.md) 厮杀（占上风）——[苏晓](苏晓.md) 在水下斩刀芒把小船岛弄沉。斯佩罗用糖浆造船撤离，苏晓三人组**7 秒完美配合**把他拖进海里：苏晓炸糖果船、踩 [阿姆](阿姆.md) 背冲出海面（被斯佩罗糖果手杖抽飞、小腹破碎、却已用界断线缠住其手腕）→ 阿姆「[冰封怒哮](../力量体系/装备道具.md)」砸船 + 苏晓侧踢踢飞糖果船 → [布布汪](布布汪.md) 一口咬住斯佩罗手臂（「你好啊，卖糖的，来海下玩玩呗」）→ 界断线把他扯入海中。**恶魔果实能力者入海即废**，苏晓等其溺得脱力昏迷、收线斩喉。

- **结算**：**独立击杀「海战区首领」夏洛特·佩罗斯佩罗** → 1000 点海贼声望 + **世界之源 14.8%（首领加成 5%）** + 噬灵者 +30 法力（上限达 5456）+ **第三级别传说级宝箱**。海战因其战死判唐吉诃德家族胜。苏晓**公开了击杀消息**（海贼威望大涨·代价是可能引来玲玲部下「格外关注」）。

## 关系
- **夏洛特·玲玲（四皇）**：母亲（其长子）。
- **[苏晓](苏晓.md)（白夜）**：将其拖海溺杀的契约者（苏晓海贼世界首个名角色击杀）。
- **[琵卡](琵卡.md)**：交战的唐吉诃德家族黑桃干部（占其上风、却因苏晓搅局双双落水）。
