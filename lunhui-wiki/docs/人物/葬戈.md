# 葬戈（巴京斯族虚空杀手 · 又强又蠢）

来自**虚空巴京斯族**的杀手——四米高（小巨人级·一般巴京斯族三米左右）、全身黝黑如铁铸、惨白瞳、来自黑渊中层的**战斗 / 杀手种族**（能极快适应恶劣环境、无氧 / 千度火焰下都能生存、却怕低温）。巴京斯族在虚空「**出了名能打、出了名穷、也出了名头脑简单**」（不会劳作生产、只靠佣金为生）。葬戈是奥术永恒星雇来追杀 [苏晓](苏晓.md)（灭法之影）的——实为 [风王子](风王子.md) 「精挑细选」派来的炮灰（五名巴京斯族传送中死了四个、最强最耿直的葬戈活下来）。

## 又强又蠢 · 被困北境（二十一卷61–72章）

他战力极强（一掌捏裂阿姆头骨、阿姆一锤只歪歪头），却**蠢得令人发指**：被苏晓一路布的陷阱坑十几次（火堆埋炸弹·布布汪还插木牌「火堆内有炸弹勿碰」他照样去碰）；在迷雾山谷被 [布布汪](布布汪.md) + [阿姆](阿姆.md) 轻松骗进 [负离子电浆超导装置](../力量体系/装备道具/index.md) 困住；追到北境极寒冰山被布布汪「智商碾压」遛住（布布汪一边给他烤火 / 喂肉干套情报、一边不让他上山）。苏晓返园后他才想起**传送装置早已损坏、回不去虚空**——只能困在零下几十度的北境发抖（「我是战士、我莫得感情」）。他临死式的执着倒赢得几分敬意。

## 关系
- **[苏晓](苏晓.md)（灭法者）**：追杀的目标（被其陷阱 + 布布汪耍到怀疑「狗生」）。
- **[布布汪](布布汪.md) / [阿姆](阿姆.md)**：把他骗进电浆网、智商碾压的一狗一牛（套走他饮食 / 战斗经历等情报）。
- **[风王子](风王子.md)**：派他来的虚空旧识（实为还苏晓人情、派能进此世界的炮灰交差）。
