# 西海王（四大海盗之一）

[魔海](../../世界/任务世界/魔海-厄运与诅咒.md) 四大传说海盗之一——戴黑色海盗帽、满脸大胡子（下巴须编成几根细辫）、狠厉残暴，坐拥四桅杆大船「暴风号」。他因 [苏晓](../苏晓.md) 夺走 [海王犄角](../../力量体系/装备道具/index.md)（1/3）而找上门，是**主线第二环「西海王之怒」**的正面敌人（第41卷29–31章）。

## 能力 · 两大棘手手段

- **祭运（指环）**：他食指戴一枚古朴指环，能对大片海域施加「祭运」——范围内幸运属性不足 50 点者，幸运被**恒定降至 −12**（苏晓幸运仅 30、无法豁免·战斗中极致命）。
- **海战鼓舞（首领级）**：士气加持下麾下海盗全员**力 / 敏 / 体真实 +12、幸运 +8、生命 +4600**；士气破 85 还触发「海之憎恨」——攻击有概率令目标 0.5~1.2 秒**定身**。
- **一次性抗诅咒秘宝**：他不惜代价从一名 [女巫](../../世界/任务世界/魔海-厄运与诅咒.md)（暴风号上·能拖厄运号上海面）手中弄到克制诅咒人的秘宝——水下一击**灭掉百余名诅咒人**（只能用一次·女巫每交易一次折寿·最多两次）。

## 海战试探（41卷30–31章）

他率 15 艘三桅 + 1 艘四桅围攻厄运号，苏晓靠 **毁灭重炮 + 水下潜航打游击**、连轰沉他 3~4 艘三桅船（每艘 105 传说度）后撤走。这其实是他的**试探**——试苏晓对海王犄角的态度：肯交则拉拢共对付双塔·女帝，不交则说明苏晓也要犄角、冲突难免。他笃定「厄运号船长的魔海诅咒撑不久」，遂打算龟缩大本营、拉拢公爵、等苏晓诅咒爆发。海王犄角之间能「共鸣」（他正是靠此追踪到厄运号）。

## 🆕地盘被洗 · 血蹄岛决战覆灭（41卷36–41章）

苏晓不与他海上死磕，而是**专洗他的地盘**痛击其士气——突袭最繁荣的**巴图岛**（毁灭重炮把港口炸平·斩其心腹卡隆·+12.9% 世界之源 + 852 传说度），又连洗大小 27 海盗岛 + 12 贸易岛。西海王苦心经营 20 年的产业尽毁、却龟缩大本营（丢尽脸面·部下以为他怕了）。

- **血蹄岛决战 · 血燃不死**：西海王在大本营血蹄岛（红巨石屏障·易守难攻）率 950+ 精英海盗死战。他的能力简单粗暴却可怕——**受伤越重、力量 / 速度 / 生命恢复越强**（燃烧鲜血获力·打到后期每秒回 4~5% 生命、半分钟没人伤到就回满、化红皮外骨骼怪物）。
- **被巴哈反锁异空间**：他抛银币把苏晓拉进「决斗异空间」单挑——却被 [巴哈](../../人物/剧毒魔鹰·巴哈.md) 用空间专精把整队诅咒人一起放进来、再封死出口（「决斗=单挑=你单挑我们所有人」），反把这决斗秘宝变成他自己的牢笼。[阿芒斯 / 鬼影 / 绞刑手 / 海怒等诅咒人](厄运号诅咒人群像.md) 一套 1.5 秒配合围攻，苦战两小时才把这「不死」的大海盗彻底耗死（绞刑手被他当武器怼进地里、多名诅咒人破碎）。苏晓得 **+15.9% 世界之源 + 不朽级宝箱 + 2600 传说度**。

## 关系

- [苏晓](../苏晓.md)（库库林·白夜 / 厄运号船长）：洗光其地盘、血蹄岛决战耗死他的对手。
- [巴哈](../../人物/剧毒魔鹰·巴哈.md)：反锁其决斗异空间、令他「送人头」的空间专精魔鹰。
- **双塔·女帝** / [公爵](公爵.md)：他拉拢 / 竞争海王犄角的其他大海盗。
- [伯恩男爵](伯恩男爵.md)：曾与他秘密合作（想私吞伯恩手中那段犄角）。
