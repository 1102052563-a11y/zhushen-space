# 双塔·女帝（四大海盗之一）

[魔海](../../世界/任务世界/魔海-厄运与诅咒.md) 四大传说海盗之一——黑色长发、真实魅力属性 200 点以上的绝色女海盗，与帝国及其他三团皆有瓜葛、左右逢源。她持有 [海王犄角](../../力量体系/装备道具/index.md) 的**最后一段**，是 [苏晓](../苏晓.md) 主线终点绕不开的一环（第41卷32章）。

## 「双塔」· 附体现身

- **从不亲身露面**：她本人所在极难寻，每次现身都是**以意识 + 媒介临时占据部下的身体**（此次占据一名手背有「两座黑塔」纹身的修女）。纹身透光即她在附体。她处境最好（谁称霸魔海对她都不利），故对海王犄角不算太渴望。

## 揭示「三支箭」真相（41卷32章）

公爵化身半人半狼上门逼她交犄角（当场秒杀她派来试探的三名黑衣女 + 一名埋伏的八阶契约者），她道出**海王犄角的深层秘密**：

- 传闻「集齐犄角操控海中之王、统领所有海兽」是**片面的**——那海中之王其实**只能统领一个种类的海兽**；
- **真正的至宝是钉在海兽身上的「三支箭」**：先古时代几个帝国进献的所有财富与秘宝、**驾驭整片魔海的力量**、以及**千年的生命**。这才是西海王敢拿命 + 折寿去赌、公爵也心动、厄运号上的「恶鬼」们（诅咒人）比谁都渴望（那是厄运号解咒的唯一希望）的原因。

说完她与公爵**暂时结盟**共对付苏晓。

## 🆕想白送犄角求自保（41卷38章）

西海王覆灭的消息传来后，她握着手中那段（花 2 枚魔海金币买来的）海王犄角只觉烫手、生怕成下一个目标。她盘算把犄角**假托「交易」送给 [公爵](公爵.md)**（1 万金币就卖）——既保脸面、又把祸水引向公爵（厄运号船长有魔海诅咒、不会浪费时间与她死磕；若公爵倒了、其地盘半数会落她手）。结果公爵**连白送都不要了**（同样怕成苏晓的下个目标）。

## 🆕被公爵算计 · 犄角被偷（41卷42–43章）

她与 [公爵](公爵.md) 的「结盟」是场骗局——公爵**派人偷了她手中那段海王犄角**（在犄角上做了手脚·非厄运号的诅咒人所为）、又拉她当幌子登血蹄岛压场。她借心腹 [剖心·列蒂西雅](公爵麾下五大头目.md) 的身体附体现身时，被公爵一句「带上你的人从我地盘离开」当众羞辱——才惊觉自己被算计到明明白白（「被踹下船还捅了一刀」）。她愤而撤走、却没走远，就地观望、等着苏晓与公爵因分赃不均而火并（可惜二人利益一致没内讧）。

## 🆕「双塔」真相 · 女帝觉醒（41卷55–58章）

「双塔·女帝」原来是一对**双胞胎姐妹**——平常四面逢源、示弱忍气吞声、被公爵算计的那个「女帝」其实是**妹妹**；真正的「女帝」（姐姐）一直沉睡。永生岛乱战成了她觉醒的舞台：

- **幕后操盘手**：她早在 [黑巫萨](黑巫萨.md) 之子 [小黑巫萨](小黑巫萨.md) 身上埋下操控的「种子」——借他之手弑父、吞掉黑巫萨的整个势力；又操控他在永生岛揽下毒虫区排除异己。她「示弱」是伪装，关键时刻反手一耳光差点抽下小黑巫萨的脑袋（震惊全场）。
- **姐妹合一 · 觉醒**：她让妹妹「死去」（「有你在，我就不完整」）——妹妹含泪自尽，姐姐由此觉醒，斩下小黑巫萨首级（「随便找个地方把这废物埋了」）。**「已经没有双塔，只剩女帝」**——五名海盗头目跪迎「女帝大人，欢迎您回来」。觉醒后的她吞并了黑巫萨势力、实力大增，成了苏晓 + 公爵都要联手才能对付的强敌。她事后借口懊悔「妹妹当初该把犄角送厄运号船长、而非公爵」。
