# 蘑菇先知（孢子森林 · 绝对中立单位）

[魔海](../../世界/任务世界/魔海-厄运与诅咒.md) 的永生岛真菌湿地深处、「孢子森林」里的古老存在——蘑菇头、绿胡须、拄荧绿手杖的蘑菇人。它是**绝对中立单位**（已与虚空之树签约·猎杀者不得伤害），把守着通往永生岛中心区的必经之路（第41卷61章）。

## 超低价商人 · 因灭法者身份「打折」

- **与世无争的古老商人**：类似 [苏晓](../苏晓.md) 在月神地宫遇到的卖水老太婆、源·神乡的盲眼老人——古老但不强大、杀它极难。腥鳞鳄由它这类「先知」照拂（伤它则孢子森林喷孢子把一切变蘑菇 + 孢子地雷）。它卖 107 件物品、**价格远低于契约者成交价**：神灵骨（不朽·2500）、世界钥匙（2100）、**古旧的杀戒**（圣灵·效果超多数不朽级、但装备中一旦受击即永久破损）、**命源**（5000·白牛来者不拒的高阶物）等——四件恰好掏空苏晓大半灵魂钱币（共付 11300）。
- **「打折」的真相**：它嘴上说「见惯世界崩灭、这些珍宝对我无用、灭法者屠恶护世也不错」——其实是场面话。真相是它三个月前还在别的世界做买卖、被 **[马文·华尔兹](../../力量体系/技能.md)** 揍过一顿、还差点被丢进废墟世界、并被警告「以后见灭法者卖便宜点、否则炖汤喂狗」。苏晓认出它头顶伤痕正是马文佩刀的符印、道「替我多谢他」，它便闭口不言了。

## 关系

- [苏晓](../苏晓.md)（灭法之影 / 白夜）：因灭法者身份得它超低价「打折」的买主。
- **马文·华尔兹**（灭法之影传承者）：揍过它、逼它给灭法者打折的老熟人。
