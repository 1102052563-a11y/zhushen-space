# 冥狼（轮回乐园八阶违规者 · 孤狼）

[魔海](../../世界/任务世界/魔海-厄运与诅咒.md) 里同处一个世界的一名**轮回乐园八阶违规者**——作风极其独特，是各乐园契约者「群起而攻」的对象（第41卷8章·魔海传说度排行**第六名·1 点**）。他与 [苏晓](../苏晓.md) 同属轮回乐园阵营、互不为敌。

## 独特作风

- **只要不主动招惹他，他绝不主动出手**：哪怕阵营冲突，只要对方不惹他便相安无事。他「除轮回乐园的契约者不杀，其他人都是他的猎物」，而轮回乐园的猎杀者也不追杀他（毒奶光沐都为此叹气「想办法把冥狼弄出这世界吧」）。
- **与团长的渊源**：苏晓从**团长**处得知——团长对违规者向来见一个杀一个，唯独对冥狼印象极好、甚至邀请他加入旅团（借此摆脱违规者身份）。冥狼婉拒，此后**遇到己方（轮回乐园）的违规者便见一个杀一个、不讲情面**。
- **放走阿姆与贝妮**：他在海上遇到落单的 [阿姆](../阿姆.md)、贝妮（自报「我叫冥狼，八阶违规者」），却只自言自语一阵便离去、未下杀手。

## 🆕永生岛 · 「晦气」（41卷55–58章）

他进魔海本是冲着另一高危区域**大漩涡**去的、没找到、循苏晓发布的坐标（花 150 灵魂钱币）来了永生岛。他是传说度榜上个体战力最强者（[亚历克](亚历克.md) 的能力被他克制、故亚历克不愿与他正面硬怼）——在永生岛沙滩上向亚历克提议合作（曾把亚历克斩成两截·亚历克恩怨分明不太计较），却因苏晓（厄运号船长）现身而各自走人。他撞见苏晓与公爵联手、又见三大海盗势力填了上万海盗性命把永生岛稳成「战争区域」，直呼「晦气」——担心自己杀了几天最后是帮别人「打工」。

## 关系

- [苏晓](../苏晓.md)（白夜）：同属轮回乐园、互不招惹的同乡违规者；苏晓这次没接肃清违规者任务，也不去找他麻烦。
- [亚历克](亚历克.md)：曾被他斩成两截、又想在永生岛与他合作的死亡乐园猎杀者。
- **团长**：曾赏识并邀他入旅团的轮回乐园强者。

## 🆕 树生世界 · 站队灰绅士 · 电弧莽夫（46卷4–8章）

他来到树生世界，出乎意料地**站在了 [灰绅士](../炎灵星/灰绅士.md) 那边**、与 [仙姬](../联盟星/仙姬.md) 一同行动（他本有原则:己方契约者不招惹他就不下杀手，但某次与团长见面后开始见违规者就杀）。他在魔海世界显然藏了拙——此时化身**人狼**（上半身格外强壮、双腿显细、长毛间闪蓝白色电弧），战斗方式莽到不敢置信:「我可以被攻击，但也必须还回去」，挨打时身躯纹丝不动，一记脸盆大的手爪拍中、敌人挡住也会浑身电弧倒飞。他与 [苏晓](../苏晓.md) 本无交情（只买卖过原初之水），如今立场敌对、压上性命对付这大敌。白色沼泽追杀里，他被炼金猛毒毒到肤色发绿、又强忍「猛毒·渴水」不敢喝水，最终随仙姬撤退。

## 关系（补充）

- [灰绅士](../炎灵星/灰绅士.md) / [仙姬](../联盟星/仙姬.md)：树生世界他罕见站队、并肩追杀苏晓的阵营与队友。
- [苏晓](../苏晓.md)：本无交情、如今压上性命要对付的大敌。
