# 奎克（诅咒人 · 疯狗）

[魔海](../../世界/任务世界/魔海-厄运与诅咒.md) 里 [厄运号](../../世界/任务世界/魔海-厄运与诅咒.md) 上一名恢复意识后敢挑战船长的诅咒人——自称「生来就是疯狗、谁都咬」。他是 [苏晓](../苏晓.md) 用**诅咒魔箱（诅咒洗炼）**强化出的战时杀手锏（第41卷34章）。成为诅咒人不过几个月（上次厄运号活跃时入伙）。

## 战力（诅咒人的技法样本）

奎克六维力 203 / 敏 187 / 体 198（真实属性·靠战争领主 +20 加成一度高过苏晓本人），技能揭示诅咒人的强度来自「生前的技法」：

- **疯狗刀术**（Lv.50+3）：刀类攻击 +492% / 攻速 +32% / 附 527 诅咒伤害 / 每击攻速再 +3%（上限 +41%）。
- **拒死身躯**（Lv.55+10）：生命 +6200 / 身防 +27 / 物理伤 −16% / 黑暗伤 −19% / 诅咒伤 −95%。
- **凶狞**：近战命中临时降敌 1 点真实体力（叠 8 次·持续 300 秒）。
- **海底绝恨**：凭建筑 / 树木 / 岩石无视物质超高速潜游。
- **怒鲨**（Lv.MAX）：化身海中黑影（真实力量转置为真实敏捷）。
- **血·魂之力**（战争领主称号加成）：武器附浴血 + 燃魂、攻击附 120 真伤 + 摄敌生命自愈。

## 「试炼」· 疯狗形态

奎克想挑战苏晓换撕毁血契，苏晓改让它进「试炼」= **诅咒魔箱洗炼**：30 秒 → 犬化，60 分钟 → 化作 5 米以上的诅咒疯狗冲出（撑裂魔箱、诅咒人围捕两小时才逮回）——这正是它「理想中的模样」。此后它忠诚度降到 30 以下、平时须关金属笼、只在战斗时放出当杀手锏（一旦异动，苏晓可令其黑影化拖回泯光岛黑雾）。**另有 3 名诅咒人效仿进魔箱、当场暴毙**（洗炼有巨大风险）。

## 关系

- [苏晓](../苏晓.md)（厄运号船长）：把它洗炼成疯狗杀手锏的主人。
- **其他诅咒人**：畏惧它、也艳羡它能进食（普通诅咒人吃了化灰渣吐出）。
