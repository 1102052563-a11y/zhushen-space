# 亚历克（鳞龙 · 死亡乐园猎杀者）

[魔海](../../世界/任务世界/魔海-厄运与诅咒.md) 里死亡乐园的**猎杀者**（同 [苏晓](../苏晓.md) 一样负责肃清违规者）——背生龙翼、手臂遍布金黑鳞片、金色竖瞳、气息宛如鳞龙，故称**鳞龙·亚历克**。魔海开局他便高居**传说度排行榜首位（197 点）**（第41卷4、8章）。

## 血脉 · 品性

- **龙血追踪**：方圆几十公里内的契约者他都能锁定，这是体内龙血之力；只要时间未过太久，杀他血亲之人一出现在他面前、体内鲜血就会共鸣。
- **寻兄之仇不迁怒**：他弟弟是「**死亡狂兽·加斯克**」——死在苍龙大陆的一片冰原上、为轮回乐园契约者所杀。亚历克开出「200 颗完整灵魂结晶 / 2 万灵魂钱币」悬赏凶手情报，找上苏晓询问。苏晓不知情、拔刀戒备（自忖胜算不高），亚历克却坦然离去——「**他从不迁怒于谁，那是内心软弱之人才做的事**」，还赞了苏晓杀绯世「杀得漂亮」，撞坏苏晓的船后主动抛出一颗灵魂结晶（大）赔偿。「这两兄弟虽是敌人，但人品还行。」（巴哈语）

## 🆕永生岛 · 绕开厄运号（41卷55章）

他循坐标来永生岛，在沙滩遇见 [冥狼](冥狼.md)（曾被冥狼斩成两截·他恩怨分明不太计较、还动过合作念头）。但一得知白夜就是厄运号船长，他二话不说展翅飞走（「白夜、冥狼，有缘再见」）——他目睹过太多被人海战术围攻的惨状，来永生岛前就打定「遇苏晓绕着走」。

## 🆕 联盟星 · 助演「夺 S-001」大戏（43卷66、69章）

到了 [联盟星](../../世界/任务世界/联盟星-死亡圣杯.md)（原著此卷写作「**麟龙·亚力克**」），他已混成机关中高层、世界之源排行榜**第三（38.6%）**——却被异军突起的 [水哥](../联盟星/水哥.md)（恩左）紧咬、「痛失万年老二之位」。苏晓曾帮他提升阵营声望、他欠一次人情。苏晓兑现承诺时点名要他「明天傍晚来进攻机关总部」，他起初以为送死断然拒绝（「你刚打沉西大陆把活物全弄死、以人格担保让我有点……」），直到苏晓开价 **45000 阵营声望**才谈细节，还特意拉上圣光乐园毒奶 [光沐](光沐.md) 当帮手。他与光沐并不参与危险物 [S-001](../../力量体系/装备道具.md) 的争夺（那是苏晓与 [金斯利](../联盟星/金斯利.md) 之间的事），只负责扮演攻打机关总部一层的「第三方」、掩护金斯利夺走 S-001。

## 关系

- [苏晓](../苏晓.md)（斩首的夜 / 白夜）：擦肩而过的死亡乐园猎杀者、得知他是厄运号船长后果断绕开；联盟星里欠人情、被雇来假攻机关总部掩护金斯利夺 S-001。
- [水哥](../联盟星/水哥.md)：联盟星世界之源榜上把他挤下「老二之位」的死亡乐园同僚。
- [光沐](光沐.md)：被他拉来一起假攻机关总部的圣光乐园毒奶（一听「进攻机关总部 + 白夜」当场告辞）。
- [冥狼](冥狼.md)：曾斩他两截、永生岛想与他合作的轮回乐园违规者。
- **死亡狂兽·加斯克**：他战死于苍龙大陆冰原的弟弟。
