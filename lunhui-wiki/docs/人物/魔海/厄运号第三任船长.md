# 厄运号第三任船长·无名之人（世界之子 · 魔海最终boss）

魔海世界史上真正的**救世之人**——一名混沌·恶阵营的**「世界之子」**，也是 [苏晓](../苏晓.md) 解厄运号诅咒时唤出的**最终 boss**（第41卷70–72 章起）。他没有名字、没有丰碑，甚至没人知道他存在过。

## 救世 · 沉入海底的成果

- **冲大漩涡救世**：他为让魔海退潮、打败了堕落神灵 [奥格司](堕落神灵·奥格司.md)（把其驱壳带到永生岛封存），带着亲如兄弟的船员和最爱的女人（[泯光岛巫祀](泯光岛巫祀.md)）冲入大漩涡杀掉饮原初之水淹世的那个人——魔海海平面由此降低、露出陆地。他用仅存意识镇压化为凶船的厄运号，是苏晓仅见的一位真正的世界之子（「坚韧、凶狠、从黑暗中扯出光明、自身沉入黑暗」）。
- **复活再战**：苏晓在游魂海域倒下 [原初之水](../../力量体系/装备道具/index.md) 洗去厄运号诅咒、解开全部封印时，他被血月中的黑暗流体唤出（四次预示：老太婆之口 / 封印心跳声 / 黑鸦戒简介 / 苏晓的黑暗之梦）。他来此是**要把一切与厄运号有关之物沉入海底**（凶船无需被铭记、责任已尽、沉海才是归宿）——也要在灵魂彻底消亡前**再战最后一场**，为被责任贯穿的一生画上句号。

## 战力（众神之眼侦测 · 41卷71章）

身高 2.3 米、黑红大衣、满头乱发、一米三阔刀（所屠生灵不下六位数、皆海上作恶者）。真实六维**力 216 / 敏 213 / 体 215 / 智 203 / 魅 157**（均破 200·被海王犄角压制中）。**孤高之傲**（豁免所有八阶及以下判定 + 超爆发力）+ 使命·意念（生命 +31000·意志坚定）+ **刀术宗师 Lv.59**（+715% 攻击·可杀强敌永久涨等级 / 力敏体）+ 全套刀术宗师衍生（心灵震割免精神控制 / 超感 / 震声五感 / 自然共鸣 / **刃震**：对斩多重震荡·67% 麻木 / 12.5% 眩晕·震荡伤害=力量 ×4.7 无视防御）+ **自开发 29 种刀术**（17 被动 12 主动）+ 奥义「世界·奔流震」。风格=大开大合、刚猛到极致的正面硬撼、唯斩龙闪能挡他的阔刀。

- **压制 30 分钟**：苏晓用 [海王犄角](../../力量体系/装备道具/index.md)（放弃主线三~五环奖励换来）切断本世界对他的加持、抵消祭命的幸运削减、**限制其战力 30 分钟**（战胜可永久 +2 幸运）。开战他一刀把 [阿姆](../阿姆.md) 斩成两段（刀带震荡·阔刀不可防御·阿姆天克）、险剃了 [巴哈](../../人物/剧毒魔鹰·巴哈.md) 平头，赞苏晓「不错的敌手」。

## 🆕决战落幕 · 安眠（41卷73章）

30 分钟限时的最终决战——他世界·停滞（10 米半径尘封世界 1.5 秒·苏晓靠空间穿透险避）+ 阔刀震荡几乎无解，苏晓靠 [刃道刀·时](../../力量体系/技能/index.md) / 心魂刃捕破绽 / 海洋复仇水刺架身 / 放逐内燃 / 血之兽 / 刃道刀·绝影层层拼杀，最终在厄运号从中断裂的一刻对斩——苏晓被斩掉左侧五分之一躯干（靠 [庇佑徽章](../../力量体系/装备道具/index.md) 锁血才没同归于尽），第三任船长脖颈也被斩断。他接住自己掉落的头颅、道一句「**不过……刀术是你胜了**」，随即被苏晓「刃道刀·绝影」的球形斩击 + 三道交错巨斩收尾。巴哈问他名讳，他答「那只是无关紧要的东西」——最终与厄运号一同沉入海底，安眠（脑中浮现当年陪他冲大漩涡的一张张面孔）。苏晓斩他得 **+19.35% 世界之源 + 含金量 51% 不朽级宝箱**（他已完成宿命之路、掉的不是命运宝箱）。
