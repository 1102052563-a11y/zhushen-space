# 狂欢（天启乐园）

[魔海](../../世界/任务世界/魔海-厄运与诅咒.md) 里天启乐园的契约者——神情懒散、有些欠揍。魔海开局他身上某样「东西」值一大笔灵魂钱币、被各方乐园契约者疯狂追捕（世界联络平台上人人求他的坐标·连 [亚历克](亚历克.md) 都被请去「安排」他）。

## 经历

- **把「东西」丢进大漩涡**：他被追得没办法、干脆把那样东西丢进了大漩涡（[冥狼](冥狼.md) 亲眼所见）——「杀我没用了」，这才在永生岛现身时被打得鼻青脸肿却保住命。
- **侦测堕落神灵资料**：围攻 [堕落神灵·奥格司](堕落神灵·奥格司.md) 时，他花 50 灵魂钱币把发言权限提高、在世界联络平台**公布了堕落神灵的部分资料**（并非无私·不解决 boss 九成人得死在这）——让众人知道这终极大 boss 的恐怖能力。

## 关系

- [苏晓](../苏晓.md)（白夜）：同在永生岛围攻堕落神灵的契约者。
- [冥狼](冥狼.md) / [亚历克](亚历克.md)：见证 / 被派去追他的人。

## 出处

- 第41卷53、63、65章（天启乐园契约者·被追杀 → 把「东西」丢进大漩涡自保·永生岛侦测并公布堕落神灵资料）。
