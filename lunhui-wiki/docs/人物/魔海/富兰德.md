# 富兰德（黎金银行副行长 · 「财神」）

[魔海](../../世界/任务世界/魔海-厄运与诅咒.md) 圣默罗岛「黎金银行」的**副行长**——大腹便便、年过四十、好色体虚的中年商人，却是 [默里准将](默里准将.md) 出手阔绰（愿掏 5 万金币买公爵性命）背后的「**财神**」（第41卷27–28章）。

## 身份 · 财富与生存之道

- **真正的银行拥有者**：他名义是副行长，实为黎金银行的真正主人；那「行长」是他推出去挡海盗绑架的高危挡箭牌。银行私有制近十几年才兴起（帝国大力支持），他是第一批捞金者、做得最好的一个。
- **资助海军的巧劲**：他不敢直接资助皇家海军（会被扣上通敌谋逆的死罪），而是**以帝国税务的名义**把金币先交帝国、再转到各海军将领手中——既缓解帝国财政、又不犯帝国威严、还结交了海军高层，帝国与海军都乐得给他大开方便之门。他深知在拳头为王的魔海上，太多财富对一个普通商人是灾难。

## 恩威并施 · 被拿捏（41卷28章）

苏晓要凑解厄运号二阶封印所差的金币，盯上了这位「财神」——用**梦魇之神神血**让富兰德整夜噩梦（被吓晕 12 次 / 吓尿 2 次），同时**偷偷给他注射 [活力复苏](../../力量体系/装备道具/index.md) 药剂**（永久 +体力 / 生命、修复器官）。次日富兰德发现自己年轻了十岁、留下的字条写着「以你的体质活过 120 岁没问题、代价是噩梦相伴」。他领教了这「付出与收获」的高明手段（远比绑票威胁高），登上厄运号谈交易——**30000 魔海金币买断噩梦**（他财力雄厚、一口答应，把巴哈都惊到「这哪是富商，分明是财神爷」）。

## 药剂利益链 · 权力野心（41卷29章）

苏晓当场再给他两瓶 [活力复苏](../../力量体系/装备道具/index.md)、提议**长期贩售药剂**——富兰德眼睛发亮（「一瓶至少定价 20000 金币、那些大人物不会拒绝」）。他甚至主动分利、织成一张利益网：

- **苏晓 70%（药剂提供者）｜[默里准将](默里准将.md) 20%（关系网）｜富兰德 10%（谈价）｜[凯撒](../炎灵星/凯撒.md)（游走各方 + 额外收益）**。
- 他要的**不是金币而是权力**——凭活力复苏 + 黎金银行副行长身份为跳板去接触**帝国王室**、搏一个人上人（自知无超凡潜质、唯权力可攀）。苏晓判断他将来会是魔海的大人物之一（「厄运号船长 + 皇家海军准将」两重身份压着、帝国王室也不敢轻动他）。

## 关系

- [默里准将](默里准将.md)：他资助 / 出谋的皇家海军将领（药剂利益链占 20%·关系网）。
- [苏晓](../苏晓.md)（白夜 / 厄运号船长）：拿捏他、又与他结成药剂利益链的合作者。
- [凯撒](../炎灵星/凯撒.md)：利益链里游走各方、与他交接药剂 / 金币的地精裁决者。
