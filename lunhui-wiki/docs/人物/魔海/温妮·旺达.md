# 温妮·旺达（三流女巫 · 厄运号的清扫女巫）

[魔海](../../世界/任务世界/魔海-厄运与诅咒.md) 里一名金发、姿色出众的**三流女巫**——本身只会「让易燃物燃烧」「烟雾占卜」（准率奇高全靠察言观色），却是 [苏晓](../苏晓.md) 破解「诅咒人无法登岛」难题的关键人物（第41卷16–20章）。

## 身世 · 被厄运号「引来」

- **诅咒研究组织末代传人**：多年前一名从厄运号脱身的女巫，返陆后创立研究「诅咒人 / 魔海诅咒」的女巫组织；历经五名女巫、耗时 319 年、传承至今只剩温妮一人（研究方向已从「解除诅咒」变为「驾驭诅咒之力」）。她因此每每上岸久居便噩梦缠身（梦见黑雾中挂油灯的大船），被迫每在一艘海盗船待约 2 个月就换船漂泊。
- **被苏晓循名而来**：苏晓以一小瓶古神血为代价从 [泯光岛巫祀](泯光岛巫祀.md) 处问得她的名字，驾厄运号找到她所在的伯克船——诅咒人屠尽 [伯克船长](伯克船长.md) 一船海盗、独留她被掳上厄运号。

## 能力 · 「超忆症」与灵想

- **超忆症 → 灵想**：她天生患**超忆症**（无差别记住一切细节），被恩师女巫「封印」大脑一部分、化隐患为能力——平时记忆差如鱼（不在意的事几秒即忘），需要时启动「**灵想**」可翻看自身记忆深处（连母乳味道、幼年退牙都能回忆）。恩师传承的诅咒知识就存在她脑海深处、靠灵想调取。她的门牙里藏着恩师给她装的一颗**放光断齿**（被苏晓掰下）。
- **⚠ 更深的秘密**：苏晓用梦魇之神·尼加狄的神血影响她时，发现她被操控下记忆力只剩 6~7 秒（异常），逼问出她不仅能翻看**自己**的记忆、还能**翻看别人的记忆**——这才是苏晓真正看中的价值。

## 经历 · 收编

- **「让海水淹没沙」**：她交出关键设想——用一件带魔海气息的物品作核心，造出「**魔海光环**」覆盖到陆地、让诅咒人在光环范围内登岛活动（五名女巫 319 年才摸索出核心材料）。核心材料 = **[海魂石](../../力量体系/装备道具/index.md)**。
- **被玩弄的谈判**：她想留一手（在丹加岛下船装买矿石、实则苟起来），却被苏晓用梦魇神血操控、稀里糊涂真去买回 12 颗海魂石又被带回厄运号（醒来抱着木盒和金币一脸懵）。最终她认清「和厄运号船长没有谈判、只有赶尽杀绝」，主动留在厄运号「负责清扫」、承认能翻看他人记忆——「她感觉自己从此可能要变成坏人了」。

## 🆕伪装「最强梯队女巫」· 忽悠三海盗（41卷53–55章）

苏晓要把三大海盗都骗去永生岛替他取原初之水（趁乱夺水），便把温妮包装成气息骇人的「最强梯队女巫」——章鱼哥的黑袍 / 手套 / **欺诈者**（半活物耳坠·防丢），[凯撒](../炎灵星/凯撒.md) 的戒指 / 烂肉 / 深渊半宝石，再由苏晓塞进 **[源](../../力量体系/装备道具/index.md)（水特性·兼容死物气息）** + 六位古神神血（月神 / 支配者索托斯 / 暗之源阿撒托斯 / 梦魇神尼加狄 / 大主教 / 血神）+ 死寂铠甲片 + [蠕动的触须](../../力量体系/装备道具/index.md) + 茂生之狂乱根须炭盒——温妮往那一站，全城行人吓晕、孩童吓尿。她全程是船长室里众人喂词的「工具人」（第一视角投影给大家看）。报酬是章鱼哥的**水晶球**（497 年前强大女巫的灵魂传承·经梦魇神血她可在梦中习得·成真本事）。

- **忽悠三海盗取原初之水**：她挨个找上门（灵魂箭 + 「事成各分一半原初之水·现付一半灵魂或对等灵魂石」）——[公爵](公爵.md) 付 47 颗完整灵魂结晶、[双塔·女帝](双塔·女帝.md) 付 38 颗；[黑巫萨](黑巫萨.md) 已被亲子杀死（改找 [小黑巫萨](小黑巫萨.md)、对方头铁翻脸、她丢下灵魂箭就跑）。「装哔这么累」是她的血泪心声。

## 🆕告别厄运号 · 女巫生活（41卷70章）

苏晓驶向游魂海域解厄运号诅咒前，途经一座贸易岛把温妮放下——她的「女巫生活」就此开始（打算买栋小木屋当占卜女巫）。走在码头上被几名海盗调戏，她一个响指血浆四溅（「怎么说我也算半个厄运号船员，太没礼貌了」），从容消失在人潮中——早已不是当初那个瑟缩的三流女巫。

## 关系

- [泯光岛巫祀](泯光岛巫祀.md)：供出她名字的泯光岛老太婆（同为诅咒相关的女巫系）。
- [伯克船长](伯克船长.md)：她最后栖身的海盗船船长（因她被厄运号灭船）。
