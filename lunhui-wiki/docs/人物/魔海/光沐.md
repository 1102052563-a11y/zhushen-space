# 光沐（毒奶 · 圣光乐园治疗系）

[魔海](../../世界/任务世界/魔海-厄运与诅咒.md) 里圣光乐园的**治疗系**契约者——黑色高开叉长裙 + 高跟鞋，手持黑白缠绕的治疗系法杖。她表面是「奶妈」，实则是把 [苏晓](../苏晓.md) 引进魔海的幕后棋子之一（第41卷4章）。魔海开局传说度排行**第三名（50 点）**。

## 能力 · 「毒奶」

- **过度治疗（狂茂生长）**：她的治疗一旦让目标生命值满至 100% 并超出 **1.5 秒**，就会生命力溢出、经她引导使敌人体内满溢的生命力**无节制血肉增生**——皮肤被撑透明、最终化作大肉球原地爆炸。她一上来就「治疗」敌人（眼镜男三人小队），把三人尽数奶爆成漫天血肉。

## 阴谋 · 老千

- **在魔海蹲守苏晓三个世界进度**：她受某方（对话暗示是**奥术永恒星 / [法师贤者·瑟菲莉娅](../瑟菲莉娅.md)**）好处驱使、笃定「苏晓强者争霸战当过厄运号船长、晋八阶后第一个世界只可能是魔海」，故提前埋伏。
- **坑蒙拐骗**：她收了公爵亲信 [铠甲男](公爵亲信.md) 的 3000 灵魂钱币「定金」、转头矢口否认（「契约？下次看清楚再签，这是跟轮回乐园那些老阴哔学的」），害对方追杀苏晓不成、白丢一大笔钱。

## 🆕永生岛 · 双毒组合（41卷53–57章）

永生岛乱战中她与作死的 [甜橙](甜橙.md) 签约组成**「双毒组合」**（一个玩毒 / 一个毒奶·连亚历克都说双毒能硬撼冥狼）。她相对冷静（拦不住甜橙作死挑战白夜），亲眼看着甜橙被苏晓抓去解析细胞 38 小时、意识转入迷你分裂体后一接收记忆就尖叫栽倒。她判断「苏晓能排传说度第一必有特别原因」、不敢轻易招惹。

## 🆕 联盟星 · 发现西大陆 · 被灰绅士玩弄（43卷50–53章）

到了 [联盟星](../../世界/任务世界/联盟星-死亡圣杯.md)，最先发现 **西大陆**（泰亚图文明）这块「高收益隐雪区」的正是她——她本想吃独食，摸清死亡率太高后才拉上 [暴君·图尔](../暴君·图尔.md)、[水哥](../联盟星/水哥.md) 等一批契约者加入西大陆阵营。可她撞上了轮回乐园违规者 [灰绅士](../炎灵星/灰绅士.md)（马甲「奇术师」）:被「傀偶同步」控得如遭鬼压床、被拉进「时运金币」双层契约游戏一路惨输、险些被剥走手背的圣光烙印——幸亏她杀过五名以上违规者、触发圣光乐园保护机制护住烙印，才被灰绅士「信守承诺」放走。她这才彻底认清「轮回乐园的老阴哔名不虚传」，怂到不敢直视对方。她原以为苏晓的军团流「稳输」、暗自幸灾乐祸，殊不知等着西大陆阵营的是白夜式军团流的碾压——总攻中她在地宫七层被删减版阿波罗逼到墙角，只差 3417 点阵营声望就能换到 [虫厄共生](../../力量体系/装备道具.md) 套装，最终不得不**捏碎水晶圆盘脱离西大陆**（阵营声望清零、血亏）。逃到友克市 / 加曼市后，她在世界联络平台哀嚎那句名言:**「我讨厌，炮弹。」**「友克市、加曼市这么好的地方，我居然在西大陆死磕。」

## 关系

- [甜橙](甜橙.md)：她的「双毒组合」搭档（作死之王）。
- [苏晓](../苏晓.md)（白夜）：她蹲守三个世界要害、又在永生岛见识其手段的目标；联盟星里她发现西大陆、拉人加入泰亚图阵营与苏晓的军团流为敌。
- [灰绅士](../炎灵星/灰绅士.md)：在联盟星把她「傀偶同步」戏耍、图剥其圣光烙印的轮回乐园违规者（被保护机制挡下才放她走）。
- [暴君·图尔](../暴君·图尔.md) / [水哥](../联盟星/水哥.md)：她在联盟星西大陆阵营拉来的临时盟友。
- [瑟菲莉娅](../瑟菲莉娅.md) / 奥术永恒星：暗示中给她好处、指使她伏击苏晓的幕后。
- [公爵亲信·铠甲男](公爵亲信.md)：被她骗走 3000 灵魂钱币定金的冤大头。
