# 甜橙（天启乐园 · 双毒组合）

[魔海](../../世界/任务世界/魔海-厄运与诅咒.md) 里天启乐园的近战系契约者——身高仅 1 米 45、满头短发、穿着清凉的作死小妹，与 [光沐](光沐.md) 组成传说度榜上的**「双毒组合」**（一个玩毒、一个毒奶）。[苏晓](../苏晓.md) 遇到过的「最能作死的人、没有之一」（第41卷55–57章）。

## 能力 · 细胞化 + 剧毒

- **细胞化「妹汁」**：她能把身体细胞化、转成荧绿色剧毒半流体——这形态无法豁免攻击、却**没有要害**（「从妹子变成妹汁」）；还能**分裂**，把部分灵魂 + 天启烙印转移到分裂体上，本体一死意识就转到迷你分裂体（可返园恢复）。细胞有剧毒、放毒时敌人毫无察觉就中减益。
- **作死拉满**：她笃信「不签契约、不买命，那太丢人」，主动挑战传说度榜首的白夜（想夺传说度），被苏晓倒吊在椰子树上仍嘴硬求速死（好让意识转分裂体）。

## 被苏晓「解析」（41卷57章）

苏晓看中她细胞化的奇妙原理（可用于造一只**只会「靠微量身体组织无休止追踪敌人」的无意识流体炼金生物**），把她抓上厄运号解析细胞 **38 小时**（全程不施痛处、只取细胞 + 稳精神）——甜橙从嘴硬骂人、到崩溃求饶、到「你让我做什么都可以」。事后本体被阿姆冰封处决、意识转入 20 公分高的迷你分裂体（一接收这 38 小时记忆当场尖叫栽倒）。

## 🆕永生岛之死（41卷69章）

体型恢复的她在永生岛围攻堕落神灵后、随各方被苏晓召出的 [戾巴](../../力量体系/称号.md)（世界之蛇）蛇潮追杀——她细胞化想逃、还冲苏晓喊「你也得死」，被黑蛇大蛇吞噬 + 侵蚀，最终被 [放逐](../../力量体系/装备道具.md) 一道残影贯穿头颅（「败了啊」）真正斩杀。她的作死之路至此终结。

## 关系

- [光沐](光沐.md)：她的「双毒组合」搭档（毒奶）。
- [苏晓](../苏晓.md)（白夜 / 厄运号船长）：她作死挑战、反被抓去解析细胞 38 小时的对象。
