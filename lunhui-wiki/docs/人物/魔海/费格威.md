# 费格威（帝国执政官）

[魔海](../../世界/任务世界/魔海-厄运与诅咒.md) 帝国的**执政官**——胡须花白的老绅士，代表王室、帝国的大小事宜大多由他决断。他在圣罗兰城帝国议厅登场（第41卷38章），是 [默里准将](默里准将.md) 的上级。

## 质问默里 · 只求「别让厄运号袭港」

厄运号一日之内专挑西海王的地盘连洗——**大小 27 座海盗岛、12 座贸易岛全遭袭击**（暗礁岛 1295 名海盗无一幸免却独留平民、巴图岛港口被炸平、晶环群岛遭半小时炮击）。费格威在议厅当众点名默里：

- **戳穿默里与厄运号的勾连**：默里假意「最痛恨海盗」，费格威一句「在你夹着尿布时我就是执政官、小崽子你是我看着长大的、少和我说屁话」逼他承认「是有合作」。
- **只求维稳**：他不追究、反而要默里**继续保持合作、无论如何别让厄运号来袭帝国港口**（「不是尽力，是必须做到」）。他心底毛骨悚然的是——这一代厄运号船长不同于以往的疯子船长：诅咒人能登岛、袭击极有目的性；一旦那船彻底挣脱束缚达最强，某个「一口吞下三桅船的怪物」就会出现。

## 关系

- [默里准将](默里准将.md)：他看着长大、逼其继续「管住厄运号」的下级。
- [苏晓](../苏晓.md)（厄运号船长）：让他毛骨悚然、只求别来袭港的存在。
