# 艾娃（女施法者 · 黎元素弟子）

[魔海](../../世界/任务世界/魔海-厄运与诅咒.md) 沉船岛上潜伏的**白袍女暗杀者**——戴兜帽、有一双藏着魔法阵图的**金色瞳孔**，是 [法师贤者·瑟菲莉娅](../瑟菲莉娅.md) 门下的**黎元素弟子**、也是这场魔海伏杀的实际主谋（第41卷23–24章）。她是 [苏晓](../苏晓.md) 击杀 [狄琳](../强者争霸战群像.md) 后遇到的第二个「黎元素适应者」。

## 身份 · 如何进入原生世界

- **施法者混入原生世界之谜**：苏晓身为灭法之影、一见她便感知到「奥术永恒星施法者」独有的元素气息（体内青钢影都活跃起来）。施法者本无法进入全开放·原生世界——苏晓推断她是**与 [灰绅士](../炎灵星/灰绅士.md) 达成合作、由灰绅士剥来一枚烙印植入她体内**（烙印波动与灰绅士的「人偶」疯医生相似度 95%+），才得以进入魔海。
- **执念 · 证明黎元素**：黎元素是瑟菲莉娅耗时多年人工合成、专为克制青钢影的战斗元素，却在狄琳身上被苏晓一刀破防、宣告失败。瑟菲莉娅对黎元素彻底失望后，曾疼爱的艾娃从天堂跌入地狱、生出「用黎元素击败灭法者、把头颅带给导师」的执念——哪怕导师早劝她「那个人不是新一代的你能对付的」。这场伏杀多半是她**擅自策划**（灰绅士只出谋划策、不直接参与，像是给苏晓传递「井水不犯河水」的信使）。

## 覆灭（41卷24章）

苏晓早有防备（[衔尾蛇石板](../../力量体系/装备道具.md) 预警 + [金克](金克.md) 供出「有人针对他」），且是按「对付瑟菲莉娅本人」的规模布防的——艾娃被 350 诅咒人 + 8.5 万黑咒乌鸦的黑色浪潮围杀，黎元素放出金色箭矢阵图重创大批诅咒人 / 乌鸦（诅咒人战死 103、乌鸦死 5180），仍杀不出重围，喉咙被刺穿、至死没想通「到底哪里出了问题」。**掉落「法之箱」**（金色蓝纹·施法者宝箱·苏晓首次获得）。

## 关系

- [瑟菲莉娅](../瑟菲莉娅.md)：她的导师（对黎元素失望后她生执念）。
- [灰绅士](../炎灵星/灰绅士.md)：给她植入烙印、助她进原生世界的幕后（借此向苏晓传威慑）。
- [苏晓](../苏晓.md)（灭法者 / 白夜）：她执意伏杀、反被淹杀的目标。
- [萤火](萤火.md) / [贪食（铠甲男）](公爵亲信.md)：她雇来 / 联合的两名契约者杀手。
