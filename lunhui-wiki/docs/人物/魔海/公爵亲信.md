# 公爵亲信（全身铠甲男 & 泡泡小男孩）

[魔海](../../世界/任务世界/魔海-厄运与诅咒.md) 里效力于大海盗**公爵**的一对心腹搭档，专程截杀 [苏晓](../苏晓.md)（第41卷4–5章）。这已是**铠甲男第三次**来魔海（「算上这次」）。

## 铠甲男

- **全身铠甲、鲜红双眸、金属质感嗓音**，坐镇沉船岛码头。核心是**伤害反噬**能力（小男孩打他一拳，疼得对方小脸皱起），但**水中速度极慢**、不便追击。他曾与厄运号交手、笃信「苏晓晋八阶后第一个世界只可能是魔海」，故守株待兔。

## 泡泡小男孩

- 铠甲男的搭档，自嘲「战斗力很垃圾」，擅长坑蒙拐骗——**女装扮小女孩把矿工骗出来杀掉**、赚了 3000 灵魂钱币，转头又被 [光沐](光沐.md) 用假契约骗光（气得暴跳如雷、拿黑枫树枝干 3100 克的「法师贤者·瑟菲莉娅见面礼」说事）。身体可化为**泡泡**飘移。他是「公爵的亲信」这一身份成了搭档的杀手锏。

## 追杀落空

- 二人本要在沉船岛外海截杀苏晓，苏晓靠 [衔尾蛇石板](../../力量体系/装备道具.md) 预警「不要登岛」而绕行避开；铠甲男虽感知到「危险」，却因水中速度慢 + 公爵的四桅船未到而未追。小男孩另调一艘两桅船、坑了四名天启乐园契约者（[金克](金克.md) 一伙）去当追兵，「虽杀不掉你，但你短时间别想发展」。

## 🆕铠甲男名「贪食」· 循鲜血印记追杀（41卷23–26章）

温妮·旺达翻看 [萤火](萤火.md) 的记忆时套出铠甲男的名字——**贪食**（与萤火的「嫉妒」同为一套代号），几小时前刚去见了 [公爵](公爵.md)。他是 [艾娃](艾娃.md) 联合的三名杀手之一（贪食 + 萤火 + 女施法者艾娃）、同时又是公爵一方。苏晓攻沉船岛时他不在岛上（去见公爵而躲过一劫）。

- **被鲜血印记牵着走**：萤火死后触发的「鲜血印记」在贪食面前浮出一颗透红血珠、指向「白夜」的方位——他遂驾一叶小帆船、在暴风雨的未知海域死追不放（「你逃不掉的！就算追到天涯海角也不放过」）。殊不知那「白夜」是苏晓用 [先古面具](../../力量体系/装备道具.md) 伪装的诅咒人——诅咒人正往最凶险海域游、把贪食引向漫漫海上漂流（不出几天多半沉船）。他因**水中速度极慢**、始终隔十海里追、无法直接感知，只能凭方向硬追。

## 🆕铠甲男「贪食」身世 · 追到厄运号（41卷43–44章）

贪食一路追着萤火的鲜血印记（追的其实是苏晓伪装的诅咒人）横穿暴风雨海域——储存空间食水被封、大船被巨浪拍散、他饿到嚼海水泡过的木板，也死追不放（「你逃不掉的、就算追到天涯海角」），直到发现「好像追错人了、萤火你死前到底看到了什么」，终于逼近真正的厄运号。他的身世由此揭开：

- **圣域乐园的八阶顶尖梯队铁憨憨**：26 岁、未婚、代号**贪食**，从一阶到八阶经历 72 个世界、信条「生死看淡、不服就干」。他习惯组 2~3 人小队、却**极费队友**（一路死了 103 名队友·平均每世界 1.3~1.4 名·但从不卖队友），在圣域乐园臭名远扬到没人肯与他组队；萤火正是他好不容易找到的「聪明队友」（明知萤火动机不纯也不在乎、只要头脑能用）。他曾与「**[神父](../神父.md)**」组队闯过 3 个五阶原生世界（神父后来停留七阶不晋级、憔悴离去——**苏晓的老对手神父的伏笔**）。苏晓评价：贪食的危险度**远比西海王高**（铁憨憨的正面突杀能力迷之强悍）。
- **能力 = 沉寂领域 + 伤害反噬 + 重型长枪破碎**：他捏碎卷轴冰封 2 公里海域、吃圣灵级护臂补充、持 2.5 米锥枪，放出 **「沉寂领域」**（500 米·压制**战斗系主动技能**及不朽级以下装备的主动能力）——诅咒人的看家本领（鬼影潜匿 / 阿芒斯死亡焰）与 [巴哈](../剧毒魔鹰·巴哈.md) 的空间穿梭全被封。唯苏晓几乎不受影响（青钢影 / 青影王是高阶位豁免·只有 Lv.16 的气息外放被压）。

## 🆕贪食决战覆灭（41卷45–46章）

贪食是纯正的**枪骑士技法型**——凝环锁（锁定不动）、枪骑士·不动（举枪吸伤豁免）、复仇（把吸收的伤害转成无视防御的冲击、曾秒八阶坦系）、震压 / 战意 / 锥穿；配 2.5 米锥枪，在沉寂领域里几乎能单挑 800 诅咒人。他的**伤害反噬**弹回 85.7% 的法系 / 黑暗 / 精神伤害，唯独怕**真实伤害**（无法反噬）、怕速度型 + 高频控制。

- **苏晓险胜**：诅咒人围攻被他「复仇」反爆没用。苏晓靠速度突袭斩铠甲衔接处（斩龙闪一刀 1374 真实 + 1406 灵魂伤害·但不朽级铠甲强到只能斩衔接·右臂被反噬最多赌 5 刀）、[刃道刀·时](../../力量体系/技能.md) 慢敌、雨滴凝针刺眼、近身宗师直踹——却判断硬拼必败（伤害反噬无解）。
- **黑咒乌鸦落柱收尾**：苏晓捏碎黑球召 20 米粗的黑咒乌鸦落柱轰贪食（战争领主加持下每只乌鸦附 120 真实伤 + 诅咒能量·伤害反噬也炸乌鸦但数量太多），贪食被诅咒之力**完全侵蚀即死**（代价：苏晓的诅咒黑纹蔓延过肩、包裹了小半边身躯）。
- **真相 · 契约作废**：贪食临死道出——奥术永恒星用「**3100 克黑枫树枝干**」悬赏苏晓、他受诱惑害死了队友萤火。巴哈「话疗」戳穿：雇主根本不是瑟菲莉娅、而是她第四梯队的弟子 [艾娃](艾娃.md)（艾娃拿不出 3100 克黑枫·实为冒瑟菲莉娅之名）——雇主已死、契约早失效、他白等白死了队友。贪食死后必掉 **[画卷残片](../../力量体系/装备道具.md)** + 猩红宝箱。

## 关系

- [苏晓](../苏晓.md)（白夜）：追杀他、反将他以黑咒乌鸦诅咒侵蚀致死的目标。
- [神父](../神父.md)：贪食曾并肩闯 3 个五阶世界的旧队友（伏笔·苏晓的老对手）。
- [公爵](公爵.md)：他们效忠的四大海盗之一（贪食去见过他）。
- [艾娃](艾娃.md) / [萤火](萤火.md)：贪食所属的三人杀手组（艾娃主谋、萤火代表嫉妒、贪食代表贪食）。
- [光沐](光沐.md)：骗走小男孩 3000 灵魂钱币定金的老千。
