# 厄运号诅咒人群像

[苏晓](../苏晓.md) 麾下 [厄运号](../../世界/任务世界/魔海-厄运与诅咒.md) 的船员——一群介于生死之间、被厄运号奴役的**诅咒人**。解二阶封印后它们恢复意识，各自亮出生前的看家本领；「诅咒人的强不在数量，而在平均战力」。以下为其中的精英个体（疯狗 [奎克](奎克.md) 另有专页）。战力排行大致：阿芒斯 → 疯狗奎克（完全疯化时略强但敌我不分）→ 鬼影 → 绞刑手 → 海怒。

## 精英个体

- **阿芒斯·默里**（贵族剑术 · 总指挥）：[默里准将](默里准将.md) 的先人、意识最清醒的诅咒人，苏晓不在场时由它指挥作战。西洋剑附「**死亡焰**」——伤口会燃起灰火逐渐烧尽血肉、只剩白骨后它还能操控骸骨扑敌自爆。苏晓许诺帮它做完事后还它自由、让它像常人般再活 30 天。
- **鬼影·内厄**（潜匿刺杀）：下半脸戴金属面具、几乎不说话。能在空气中进入「水下潜匿」超高速移动、用双钩刺匕首（连锁链）割喉贯颅，真实流血伤害持续近一小时、极难止血。
- **绞刑手·沃**（握力撕肉）：双手握力惊人、十指如剃刀般从敌人身上扯下血肉。西海王决战中被西海王抓着当了十几分钟「武器」大头朝下往地里怼、左半身破碎没了脖子仍不死。
- **海怒**（海压水球）：真名已遗忘。平时低调受调侃只笑笑，战时极顶——把敌人周围空气化为海水 + 海压水球困死。
- **撕裂·科尔里奇**：另一名生前很强的精英诅咒人。

## 机制 · 诅咒人

- **忠诚 / 意识**：苏晓魅力让诅咒人基础忠诚 70+、厄运号加成后固定 95~98（相比黑雾漆黑一片、厄运号是它们梦寐以求之地）；不听话可被令黑影化拖回泯光岛黑雾。它们畏惧泯光岛与厄运号船长两个存在。
- **战争领主加成**：靠 [黑咒乌鸦](../../力量体系/装备道具/index.md) 数量触发苏晓 [战争领主](../../力量体系/称号/index.md) 全档加成（全真实属性 +20 等），诅咒人真实属性一度高过苏晓本人。
- **配合质变**：恢复意识后懂得配合——西海王决战中十几名诅咒人一套围攻（潜匿割喉 + 死亡焰 + 海压 + 钩链束缚 + 贯颅）耗时不超 1.5 秒。

## 关系

- [苏晓](../苏晓.md)（厄运号船长）：奴役 / 统率它们的主人。
- [奎克](奎克.md)：同为诅咒人、被洗炼成疯狗杀手锏的特例。
- [西海王](西海王.md)：被这群诅咒人在异空间围攻耗死的大海盗。
