# 黑蔷薇（轮回乐园 · 毒舌大喷子）

[魔海](../../世界/任务世界/魔海-厄运与诅咒.md) 里轮回乐园的八阶契约者——中性美、双手插兜、一身运动装，是**大名鼎鼎的毒舌喷子**（苏晓晋八阶前就听过其名号），也是永生岛乱战中的一方战力（第41卷61–69章·传说度**第四名**·仅次亚历克 160 点）。

## 能力 · 又能喷又能打

- **口才碾压**：气死人不偿命——把作死的 [甜橙](甜橙.md) 喷到自闭（「你的脑容量最多 2KB、她至少 1TB」是光沐的评价）；在世界联络平台得罪一大票人、组团报复的（含前五十名）反倒来送传说度；她口无遮拦常被虚空之树限制发言（每天只能喷两句、每次写「小课文」）。
- **战力接近亚历克**：之前一直藏拙、后期杀起来才冲到传说度第四，还有**斩杀类能力**（本想在堕落神灵战后趁乱夺宝箱撤退，却因斩杀有短暂判定被苏晓捷足先登）。围攻 [堕落神灵](堕落神灵·奥格司.md) 时活到最后一批。

## 经历

围攻堕落神灵后她想抢原初之水、却忌惮苏晓再召 [戾巴](../../力量体系/称号.md)（世界之蛇）而作罢（「就算死也不想被黑蛇淹没」）。最终与 [冥狼](冥狼.md) / [亚历克](亚历克.md) / [光沐](光沐.md) 组团向苏晓买原初之水（每人 1000 灵魂钱币 / 1.5 升·苏晓每人多送 0.5 升）——交易时一句「我真是谢谢你啊」道尽复杂心情。

## 关系

- [苏晓](../苏晓.md)（白夜）：被抢先夺宝箱、又向其买原初之水的同乡契约者。
- [甜橙](甜橙.md)：被她喷到自闭的作死小妹。
- [冥狼](冥狼.md) / [亚历克](亚历克.md) / [光沐](光沐.md)：一同买原初之水的临时伙伴。
