# 伯恩男爵（哈伯德·伯恩 · 沉船岛管理者）

[魔海](../../世界/任务世界/魔海-厄运与诅咒.md) 里 [沉船岛](../../世界/任务世界/魔海-厄运与诅咒.md) 的实际管理者——蓄小胡子、身着贵族服、年过三十仍富男性魅力、举止透着贵族的温雅与严苛（第41卷19章）。他正是 [苏晓](../苏晓.md) 主线第一环「恶名昭著」的目标（夺**伯恩家族传家宝**）。

## 身份 · 沉船岛

- **「男爵」的由来**：爵位制早废，但他的曾曾曾祖父曾是**厄运号第十五任船长的副船长**（虽不到 1 个月就被诅咒侵蚀而死）——就凭这份「厄运号副船长」的凶名，海盗们敬畏他，他也乐得被称「伯恩男爵」。
- **与公爵的分成关系**：沉船岛名义上是大海盗**公爵**的地盘，实由伯恩男爵打理、手下数百侍卫；他占收益 55%、公爵抽 45%（公爵只派海盗、不管理）。公爵之所以不吞掉他，是因「灭他自管」的代价高于收益——这份分成其实是公爵的一种认可。他近来因公爵有与帝国开战之势、沉船岛恐沦为缓冲带补给岛而忧愁。

## 处境（41卷19章）

- 沉船岛古堡里，[公爵亲信](公爵亲信.md)（铠甲男 + 泡泡小男孩）与瑟菲莉娅雇来的**第三名白袍女暗杀者**（[艾娃](艾娃.md)）正在他这里蹲守苏晓。他不知道的是，一只双目漆黑的黑咒乌鸦已落在庭院树上——厄运号即将来袭。

## 覆灭 · 自轰头颅（41卷21–22章）

- **驱海盗当炮炸**：厄运号来袭时，他一枪轰烂一名嗤笑海盗立威、逼码头百余海盗替「伯恩家族」迎战皇家海军（谎称乌鸦是海军胁迫女巫召来），自己借祭献来的庇护溜回古堡地下二层、等公爵增援里应外合。
- **寄虫侍卫不堪一击**：伯恩家族多年拿奴隶试验、造出模仿诅咒人的「**寄虫侍卫**」（虫化尸体·血肉化尖刺 / 加重长剑），本想借此成魔海大势力——结果对上苏晓的诅咒人被力量碾压、几斧劈成两截，认知崩溃。
- **「无敌」的自杀**：面对 [巴哈](../../人物/剧毒魔鹰·巴哈.md) 与 [阿姆](../阿姆.md) 的 boss 战邀战，他皮肤下涌起千万虫子、却抬起圣灵～不朽级的连发燧发枪**给自己脑袋来了一枪**——「只要我死了，你们就永远无法战胜我」，一枪轰烂头颅自尽（让巴哈措手不及）。传家宝随后被苏晓从书房密墙搜出 = **[海王犄角](../../力量体系/装备道具.md)（1/3）**（生命力共振断指解锁）。

## 关系

- [苏晓](../苏晓.md)（厄运号船长）：奔他传家宝而来、即将攻岛的敌人。
- **公爵**：他效忠 / 分成的大海盗（沉船岛名义主人）。
- [公爵亲信](公爵亲信.md) / 白袍女暗杀者：在他古堡里蹲守苏晓的杀手。
