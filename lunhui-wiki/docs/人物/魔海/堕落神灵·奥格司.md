# 堕落神灵·奥格司（永生岛终极大boss · 异神）

[永生岛](../../世界/任务世界/魔海-厄运与诅咒.md) 中心区洞窟里守着原初之水的**终极大 boss**——平常是靠坐水池旁的枯尸，一旦有人取水便苏醒、化身十米以上、遍体罪孽之火的人形神灵怪物（第41卷64–67章）。它是「神灵化途中出现偏差」的**异神**。

## 身世 · 从殉道主教到堕落神灵

- **虔诚的殉道者**：奥格司原是信奉海神的虔诚主教，游历各海域劝大海盗别肆意杀戮，遭嘲笑、谩骂、鞭打也不怨恨。他悟到「杀戮只会带来仇恨、仇恨是杀戮的种子」（找他复仇的小海盗正是他当年埋下的恶果）。某日他被海盗割喉沉海——**魔海选中了他、把他没入原初之水**，让他成为史上第一位魔海神灵。
- **堕落**：神灵化途中他要承受魔海上所有生灵的负面情绪（杀戮 / 绝望 / 贪婪）——他失败了、堕落成异神，驾驭魔海试图淹没整个世界洗去一切（终于明白「人的贪婪无限、连天神也为之恐惧」）。他正是魔海灾难史的源头，后被 [厄运号第三任船长](厄运号第三任船长.md) 打败救世。

## 「恶念容器」机制

第三任船长灭不掉它的**驱壳**（已超越生物范畴），只能封存在永生岛。这驱壳成了魔海的**恶念接收体**——每当魔海有人惨死，死者最后的憎恨 / 绝望 / 愤怒（变种信仰之力）都转嫁进来。正常应「承载→满溢→杀戮（毁灭与恶念相关之物）→再承载」循环；但被守尸的 [泯光岛巫祀](泯光岛巫祀.md) 故意让恶念**郁结**（不放它出去杀戮），才积到燃起「罪孽之焰」的骇人程度。

## 战力（狂欢侦测）

一击秒杀镶三宝石的圣灵级 +14 大盾八阶坦系。世界青睐（生命 +98000·豁免所有控制 / 42% 伤害 / 火系伤害·**用火攻反给它回血**）+ 水中之火（每秒回 3.5% 生命·高优先）+ 罪孽之焰（攻击附 853 真实火伤 + 灼烧）+ 焚烧殆尽（手爪抓握按体力差造成海量真实火伤·无冷却回血）+ 地脉焰冲（拍地拔出地脉火·150 米五重伤害「拔罐拳法」）+ 三阶段形态。133 名契约者 + 精英海盗 + 诅咒人围攻致死（战死过半）。

## 覆灭（41卷67章）

[苏晓](../苏晓.md) 靠青影王铺垫 + 魔刃斩杀（斩生命值 25% 以下），斩掉了它由恶念形成的**意识**（≠杀死驱壳）——公告「恶念聚集体已被消灭」（全员奖励 5% 世界之源等·苏晓另得 32.72% 世界之源 + 不朽级宝箱·罪焰）。驱壳随即背后爬出枯尸、喷出原初之水后重新靠坐水池、恢复为无害容器。

## 关系

- [苏晓](../苏晓.md)（白夜）：斩掉它意识、夺全额击杀奖励的人。
- [厄运号第三任船长](厄运号第三任船长.md)：当年打败它、救世的世界之子。
- [泯光岛巫祀](../../人物/魔海/泯光岛巫祀.md)：守其驱壳、故意让恶念郁结放它出笼的乌鸦守尸人。
