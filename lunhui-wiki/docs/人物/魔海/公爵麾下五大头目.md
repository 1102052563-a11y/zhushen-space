# 公爵麾下五大头目

[公爵](公爵.md) 率近两千精英海盗登血蹄岛压场时，身后跟着的**五名海盗大头目**——每个都是魔海上名声赫赫、仅比四位大海盗略次一筹的人物（第41卷42章）：

- **海马·莫查兹**
- **舌猎手·左狄**
- **丧钟·冈吉**
- **白银蛇·修卢**：实为 [双塔·女帝](双塔·女帝.md) 的心腹，这次来是替公爵撑场面（女帝与公爵「结盟」的桥梁）。局面明朗后劝女帝撤走（「不能同时对上厄运号和公爵劫掠团」）。
- **剖心·列蒂西雅**：同为双塔·女帝的心腹。手背有「黑色双塔」纹身——[双塔·女帝](双塔·女帝.md) 正是**临时占据她的身体**现身谈判的（女帝惯用附体现身）。

## 背景 · 双塔女帝被算计

这五人里白银蛇、剖心两名双塔·女帝心腹的出场，暴露了「公爵与双塔·女帝结盟」的真相；但公爵实则**偷了双塔·女帝手中那段海王犄角、又把祸水引向她**——女帝借剖心的身体现身时才惊觉自己被算计，愤而撤走观望。

## 关系

- [公爵](公爵.md)：五人压场所随的大海盗。
- [双塔·女帝](双塔·女帝.md)：白银蛇·修卢 / 剖心·列蒂西雅两人效忠的大海盗（剖心被其附体）。
- [苏晓](../苏晓.md)（厄运号船长）：公爵谈判对象、他们压场所向的对手。
