# 章鱼哥（魔海之王·扎卡摩罗 · 海洋魔灾）

集齐三段 [海王犄角](../../力量体系/装备道具/index.md) 拼接后召来的**本世界最强海兽**——却是一只体长仅 3 米、活了 209 年的**章鱼**（自称「魔海之王·扎卡摩罗」，帝国君王与人称它「海洋魔灾」）。[巴哈](../../人物/剧毒魔鹰·巴哈.md) 一见傻眼（「小章鱼，你爸呢」），它是 [苏晓](../苏晓.md) 主线的关键 NPC（第41卷47–52章）。

## 身世 · 灵魂箭与最强海兽血脉

- **家里蹲 + 精神力极强**：本身战力不算顶（一觉睡半年、宅在海底宫殿），但精神力极为强横、能操控同族海兽（单刷其宫殿的冒险者都被它操控海兽弄死）。它继承了最强海兽的血脉——身上钉着三根**灵魂箭**（灵魂上的枷锁 + 路标·会随每代最强海兽死去转嫁到下一代）。
- **灵魂箭的来历**：先古某大海盗发现了藏「原初之水」的永生岛、为守秘密用灵魂箭奴役当代最强海兽、并砍下其一根犄角分三段（每段可解一根灵魂箭·给最强海兽一丝希望）。那大海盗饮原初之水称霸魔海许久、终被毒杀；灵魂箭遗传几代后已束缚不住章鱼哥（可离岛、但拔不出）。它本打算操控海兽帮厄运号船长夺犄角来换自由，没想到苏晓发展太快、把西海王直接打穿（章鱼哥在后头追得「怀疑鱼生」还没追上）。

## 交易 · 拔灵魂箭

- **被凯撒 / 巴哈忽悠**：[凯撒](../炎灵星/凯撒.md) 一顿「海洋魔灾」的彩虹屁把它捧飘（差点飘成水母），换来它献出 **鱼龙精魄（不朽级·升青钢影绝佳）+ 海灵精魄（圣灵级）+ 壳骨 / 珊瑚脂 / 冷凝的怨血 / 冰曜石** 等宝物。巴哈还回赠它一小瓶 [蠕动的触须](../../力量体系/装备道具/index.md)（用精神力抵抗以修炼·警告别碰别吃、否则引来茂生之狂乱）。
- **借犄角拔箭**：苏晓把海王犄角**借**给它（不能给·犄角与主线最终环有关）——它把犄角刺入头部、由 [阿姆](../阿姆.md) 帮忙硬拔三根灵魂箭（疼得吐墨汁、差点死）。得三根灵魂箭后，章鱼哥虚弱得没法操控海兽、干脆赖在厄运号主桅杆下酣睡（比回海里安全）。

## 关系

- [苏晓](../苏晓.md)（厄运号船长）：召来它、与它交易换灵魂箭 / 精魄的船长。
- [巴哈](../../人物/剧毒魔鹰·巴哈.md) / [凯撒](../炎灵星/凯撒.md)：把它捧飘、忽悠它献宝的两张嘴。
