# 海眷者（泯光岛第二守雾人）

[魔海](../../世界/任务世界/魔海-厄运与诅咒.md) 的 [泯光岛](../../世界/任务世界/魔海-厄运与诅咒.md) 上**第二位守雾人**——被封印在一间圆木小屋的木墙内、只有面部凸出（其余身体与墙面齐平、甚至内凹）的人形存在。他管束着泯光岛黑雾内的所有诅咒人（第41卷34章·解厄运号第二阶段封印后其屋外黑雾才散）。

## 作用 · 补充诅咒人

- **消耗魔海金币放诅咒人脱困**：[苏晓](../苏晓.md) 与他交涉后，可**每名诅咒人 50 枚魔海金币**、用金币内的魔海能量帮诅咒人挣脱黑雾束缚、为厄运号效力（脱困后诅咒人便与守雾人无关、改忠于苏晓）。苏晓花 17000 枚金币让约 340 名诅咒人脱困——只有足够强的诅咒人才有资格脱困，弱的仍困黑雾。他自身也极虚弱、渴求魔海的力量。
- **与「乌鸦」的恩怨**：他喃喃「不要相信乌鸦……是她害了我们」——「她」指同为守雾人的 [泯光岛巫祀](泯光岛巫祀.md)（守雾人·乌鸦）。苏晓触碰他时触发隐藏成就任务「消灭乌鸦」（击杀那老太婆完成·无惩罚·苏晓暂不考虑）。

## 🆕海眷族·伪 · 校正航海罗盘（41卷36章）

- **身世**：他曾随厄运号第三任船长冲入大漩涡、在大漩涡下得过一小瓶血液、饮下后自称「海眷者」——实为 **海眷族·伪**（血统源自 [风海大陆](../../世界/任务世界/魔海-厄运与诅咒.md)〔卷50同名世界·伏笔〕的海眷族·机缘巧合流落魔海）。他是魔海土著、却带着外世界血统。
- **校正 [航海罗盘](../../力量体系/装备道具/index.md)**：苏晓拿出死亡屋所得的不朽级「航海罗盘」（前置需求正是「海眷族」），海眷者以其精神之源为罗盘校正刻记（苏晓付 17000 金币）——罗盘从此可指向持有者渴望之物 / 标记目标锁其世界成钥匙。
- **「决断」的一环**：他与老太婆互指对方是「叛徒」、都劝苏晓杀掉对方（他反复提醒「小心乌鸦、一定要弄死她」），共同构成成就任务「决断」的两难。

## 关系

- [苏晓](../苏晓.md)（厄运号船长）：付魔海金币向他补充诅咒人、并让他校正航海罗盘的船长。
- [泯光岛巫祀](泯光岛巫祀.md)（守雾人·乌鸦）：他口中「害了我们」、要苏晓弄死的另一守雾人。
