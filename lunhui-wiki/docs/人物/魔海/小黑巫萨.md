# 小黑巫萨（黑巫萨之子）

[魔海](../../世界/任务世界/魔海-厄运与诅咒.md) 里 [黑巫萨](黑巫萨.md) 不到 20 岁的独子——面色苍白、气息如黑曼巴蛇般阴冷狠毒，弑父后成了黑巫萨海盗团的新船长（第41卷55–58章）。实则他从头到尾都是 [双塔·女帝](双塔·女帝.md) 的**傀儡棋子**。

## 傀儡的一生

- **被操控弑父**：他其实并非自私到弑父的地步——是女帝早在他身上埋下操控的「种子」（还给了他一把刀），让他趁黑巫萨传授黑巫术时一刀捅穿父亲头颅（女帝借此除掉最难缠的黑巫萨、吞其势力）。
- **永生岛的排除异己**：永生岛乱战中他头铁地揽下最难啃的**毒虫区**（其实是女帝借他排除异己）；当着众人反被 [双塔·女帝](双塔·女帝.md)（时为示弱的「妹妹」形象）一耳光差点抽下脑袋、又被女帝的人锤到意识模糊抬出来。他在女帝面前会**双膝跪地讨好**（人偶般的喜悦），女帝叫他「废物」。
- **被榨干处决**：待价值榨干，女帝在真女帝觉醒后**一刀斩下他的头颅**（「随便找个地方把这废物埋了」）。

## 关系

- [黑巫萨](黑巫萨.md)：被他捅死的亲生父亲（实被女帝操控）。
- [双塔·女帝](双塔·女帝.md)：操控他弑父 / 排除异己、又将他榨干处决的幕后主人。

## 出处
