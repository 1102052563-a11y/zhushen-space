# 卡尔

**卡尔** 是 [幻影旅团](../势力/index.md) 的成员、团内实力**排行第十一位**的契约者（四卷1–2章）。

## 简介
二十多岁、西装笔挺、步伐沉稳、面带从容笑意，常把玩一副**塔罗牌**。是一名**法师**——苏晓很少在契约者身上感到压力，却觉得卡尔「很不好惹」（自估正面对上卡尔约**六四开**·他六卡四）。他知晓苏晓的假名「白夜」与所在位置，登门**邀苏晓加入幻影旅团**，并说明入团须先去**竞技场打一场确定名次**。

被苏晓以塔罗牌留言婉拒（「独自一人已习惯，暂不加入」）——苏晓的顾虑是：自己是猎杀者、且卡尔仅排第 11 就这般强，旅团前列必更可怕，他想等自身实力进入**旅团前七**再考虑（四卷2章）。

## 关系
- [幻影旅团](../势力/index.md)——所属（第 11 位·法师）。
- [苏晓](苏晓.md)（白夜）——他出面招揽的对象（被婉拒）。
