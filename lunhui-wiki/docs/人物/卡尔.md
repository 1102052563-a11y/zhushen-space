# 卡尔

**卡尔** 是 [幻影旅团](../势力/index.md) 的成员、团内实力**排行第十一位**的契约者（四卷1–2章）。

## 简介
二十多岁、西装笔挺、步伐沉稳、面带从容笑意，常把玩一副**塔罗牌**。是一名**法师**——苏晓很少在契约者身上感到压力，却觉得卡尔「很不好惹」（自估正面对上卡尔约**六四开**·他六卡四）。他知晓苏晓的假名「白夜」与所在位置，登门**邀苏晓加入幻影旅团**，并说明入团须先去**竞技场打一场确定名次**。

被苏晓以塔罗牌留言婉拒（「独自一人已习惯，暂不加入」）——苏晓的顾虑是：自己是猎杀者、且卡尔仅排第 11 就这般强，旅团前列必更可怕，他想等自身实力进入**旅团前七**再考虑（四卷2章）。

## 现身斩·赤红之瞳（四卷24章）
卡尔也进入了 [斩·赤红之瞳](../世界/任务世界/斩赤红之瞳.md) 世界、潜伏在借夜袭部队名义同入的契约者中。当一群契约者旁听完夜袭会议、私下商议「保全夜袭战力以利最终决战、钥匙各凭本事」时，角落里把玩塔罗牌的卡尔（手背显出**黑色数字「11」纹身**）冷然道「比起帝都势力，我更想先杀掉各位」——众契约者一见旅团编号纹身即惊恐失色，被他以结界灭杀，三分钟尸骨无存（连血迹都没留）。可见幻影旅团成员的乖戾凶残（四卷24章）。

## 与苏晓死战（四卷44–46章）
他在帝都杀死了狩人部队的 Dr.时尚与兰（现场留塔罗牌），又在夜袭部队总部外的森林与苏晓正面开打。其底细揭明：

- **掌握七百多种法术技能**的法师；**双枪即法杖**（近金属质的树木·塔罗牌缩入弹匣即为「子弹」），**施法 = 扣扳机**（无前摇 / 吟唱·双枪每次可放两个技能·像「法术机关炮」，几秒连放十几术）。法术分**火 / 风 / 水**三系（风之弹·冲击炮、火之弹·大火球、水之弹·清波庇护、冰之弹等）；战前以三元素乱流作防御。
- 苏晓凭**绝魔体质免 40% 法术伤害**硬扛其六连射，并以**青钢影「[法力引爆](../力量体系/技能.md)」**（斩在法术上使其提前激发）破其法术洪流、一刀斩破其水护罩。
- **旅团规矩的诚意**：卡尔提出生死一战——苏晓胜则「主线钥匙无人能争」、且可拿走他的**编号「11」入团**（旅团强者生、弱者死，死了无人报仇）；他还透露**旅团里除第七位「左轮（牛仔）」外、其余成员他都没见过、只知称呼**。

## 死亡 · 出售编号（四卷46章）
苦战中卡尔用「绝对水域」（十米水球·双方都不能呼吸）、自身却也被淹而失利；苏晓暗以钢丝缠其小腿、破碎精灵六连射破盾断其左臂。卡尔踢飞左枪**自爆「火力全开」**（三十多个法术齐爆），苏晓以三笠的心意「庇护」硬扛、一刀**贯喉斩杀**之。苏晓不死的关键是**[布布特尼](苏晓的随从与召唤物.md) 在暗处偷偷为他回血 / 护主**（「因为我带了条狗」）。

卡尔（**10025 号契约者**）死后掉**猩红卡片** + **[幻影旅团的邀请编号](../力量体系/装备道具.md)**（用后直接成旅团 11 位·或售 5% 世界之源）。苏晓选择**出售换 5% 世界之源**、暂不入团。其死讯传开后，旅团多名怪物级成员（左轮牛仔 / 肌肉大汉 / 血红长发吸血鬼美女等）皆觉「把编号卖了的人有意思」（四卷46章）。
