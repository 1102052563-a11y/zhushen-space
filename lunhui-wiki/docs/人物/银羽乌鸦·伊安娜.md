# 银羽乌鸦·伊安娜

> ⚠ **本名伊安娜，小麦色皮肤、身形偏娇小的绝强极限暗杀者**——⚠ **她的力量体系注定必须靠戮杀至强才能更进一步，因此一直有个野心:成功刺杀一位至强**;⚠ **⚠ 问题是本世界的至强全都是各大势力的高层或顶层，那种没有势力的至强根本不存在**——⚠ **直到黑市里流传出一场围杀:被围杀者来自界外，在本世界毫无瓜葛，还是三大势力的死敌**（⚠ 她知道机会来了·⚠ 参与围杀者足有几百人、据说还有同为至强者参与——这反倒让她打起退堂鼓，可在内心纠结之下，**命运替她作出了抉择**）。

## 🗡 一次注定失败的暗杀

⚠ **她在刺出那一刀前恐惧到大脑空白、甚至想转身就逃，可刺客的本能告诉她必须继续，否则一点生机都没有**;⚠ **⚠ 没有想象中的反击——对方甚至没看她一眼，只是夺走了她刺出的「影舞」锐剑**（⚠ 苏晓握剑后罕见地开口夸了一个「好」字，并不理解一个被命运力量支配的绝强怎么可能拥有这等武器）;⚠ **⚠ 刺杀失败的瞬间她如梦初醒**:⚠ **自己为什么要去刺杀一位有几百名绝强围攻、却把这些绝强砍瓜切菜般弄死的至强者？**——⚠ **她被自己蠢笑了，随后释然:那至强宗师的气场实在过于惊人，靠近对方后她双腿发软、脚底踩着棉花**（⚠ 「这就是至强者吗，真是太可怕了」·⚠ 她的判定出现了巨大错误——她所暗杀的这位，**亲手杀死的至强者就有两位数**〔深渊生物与顶级炼金生物不计入内，这是共识〕）。

- ⚠ **⚠ 苏晓当天就坐在她身旁**:⚠ **「是啊，为什么做这种蠢事。」**——⚠ **她的唯一想法是「完了」，甚至预知到自己侧跃影化后被自己的「影舞」锐剑斩下脑袋〔那把剑在这敌人手中比在她手中锐利百倍不止〕**;⚠ **⚠ 苏晓却把剑柄递还给她**——⚠ **「别自我发挥，老实回答问题」**;⚠ **⚠ 怀表咔哒作响的十秒**:⚠ **每过10秒就会有一根「仁慈之刺」深深刺入她体内；而第一根刺入后，无论后续如何她都必须死**——⚠ **在时间即将到10秒时她双手合拍自己的脸，又被自己蠢到了:能活命比什么财富都重要**（⚠ 苏晓懒得杀死一个被命运半支配来暗杀自己的绝强，但**一定会彻底弄死一个被自己折磨过的绝强**——「让敌人成长回来报仇」的戏码在他这九成九不会有）。

## 🧩 命运的最后一环

⚠ **苏晓在她带路的东寒境冰殿里，找到了已经过世的 [老婆婆](命运暴君·老婆婆.md)**——⚠ **这次暗杀正是老婆婆命运编制中的一环:在命运中她必定无法伤及苏晓，且会触动另一个因果——前提是苏晓能洞悉这一切**（⚠ 命运系不会强行让某些事发生，那必定诱发相应代价·⚠ 若苏晓在遭遇暗杀的下一秒随手杀掉她，后续的一切就烟消云散）;⚠ **⚠ 冰殿里那尊束鱼骨辫、身着半甲的女强者塑像**——⚠ **她当初取走塑像手中的锐剑，苦寻大半个月也打不开的金属门扇，被苏晓把锐剑归位后轰隆隆打开**（⚠ 她当场傻在原地、抽了自己一巴掌并谎称「有蚊子」·⚠ 实际上锐剑根本不是钥匙:**放眼星界能不用暴力打开那扇门的只有苏晓一人**，他归位只是不想让她察觉到什么）。

## 🐦 「你叫银羽乌鸦」—— 钓枭的钩子

⚠ **苏晓仔细端详后发现，她与 [枭](枭.md) 的挚友兼宿敌「银羽乌鸦」有七成相似，且是同一个力量体系**——⚠ **在虚空网络上调出「银羽乌鸦」死后被报社公布的照片一对比，几乎一模一样**;⚠ **⚠ 「从今天开始，你叫银羽乌鸦。」「为什么？」「……」**（⚠ 她知道继续问，得到的肯定不是答案）;⚠ **⚠ 一瓶灵想秘药下肚、盯着照片看五分钟，她就接受了这张综合评分98分的容颜，连坐姿都温婉几分**——⚠ **嘴上说着毫不在意自己容貌是否美丽的她，「银羽乌鸦」的颜值的确比原本更能打些**。

- ⚠ **⚠ 苏晓的算计**:⚠ **「她是不是银羽乌鸦的转生并不重要，重要的是**枭认为她是**——这对枭才重要」**——⚠ **枭、银羽乌鸦、伊安娜是同一个力量体系:唯有杀死更强者才能进一步提升**;⚠ **⚠ 枭要杀死一位至强巅峰才能迈向至强巅峰，这不可能，所以只剩这个体系的另一种方法:「宿命的诅咒」**（⚠ 枭当年为救女儿而被亲情支配；若考虑到那女儿其实是银羽乌鸦亲生的，那么把她培养到至强再杀死，强烈的情绪爆发或许就能让枭迈出那从第二纪元起始终没能迈出的一步）;⚠ **⚠ 谈判过程**:⚠ **苏晓全程表现得像是「这事大概率不太行，但试试也没损失」——先说「杀了吧」，让阿姆提斧下楼**——⚠ **枭一句「等等！」就上了钩**（⚠ 阿姆那一斧从肩膀劈到几乎触碰心脏、是真的没留手；伊安娜也豁出去了，只因那看似灭法者、实则有魔鬼般蛊惑的男人之前问了她一句:**「想封临至强吗。」**·⚠ 苏晓摇头说「心脏被黑暗力量侵蚀了，很难救活，放弃吧」，枭立刻笃定「你能救活她」·⚠ 最后枭自己补上了最关键的一句:**「银羽乌鸦死前太平静，现在看来，她真的可能有转生能力。」**）。
