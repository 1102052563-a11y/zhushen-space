# 伊鸟

**伊鸟** 是 [东京喰种](../世界/任务世界/东京喰种.md) 世界的 SS 级 [喰种](../世界/生物万族/index.md)、14 区「Helter Skelter」酒吧老板，亦是一名**情报商**，属神秘组织 [小丑](../势力/index.md)（二卷14、18章）。

## 简介
橘红色长发、神态妩媚而内里警惕；以酒吧为小型情报中心、消息灵通（一见面便道破苏晓的战绩与已加入 CCG 之事）。喝人血（喰种除水与咖啡外无法进食人类食物）。据原著，曾向金木研提供神代利世的情报（二卷14、18章）。

## 与苏晓的交集
两度被苏晓登门：先以威胁问出共喰区位置，再以 [Rc 细胞浓缩液](../力量体系/消耗品材料/index.md) 为饵、换得 14 区喰种聚集地的地图。被苏晓盯上后心生畏惧，暗自决定搬家避祸（二卷14、18章）。

## 关系
- [苏晓](苏晓.md)（白夜）——被其两度逼问 / 交易情报，畏其如瘟神。
- 呗——同属小丑的喰种（四区面具店老板），与其相熟。
- 小丑（宗太 / 尼克 / 萝玛 / 呗）——所属组织的成员。
