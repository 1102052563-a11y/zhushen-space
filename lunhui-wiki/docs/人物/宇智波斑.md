# 宇智波斑

火影世界的传奇强者、宇智波一族最负盛名的人物——一头黑发披肩、生有一双**轮回眼**。他生前布下的「月之眼计划」是整个第四次忍界大战的底层棋局：[带土](宇智波带土.md) 这些年一直假冒「他」的身份行事，正是他死前的安排（带土只是替身、用以推进计划直到他被复活）。

## 被秽土转生 · 战场登场（十五卷84章）

[大蛇兜](大蛇兜.md) 秽土转生出的**最强死者，没有之一**。开战后他以「**花界**」（房屋大小的花朵在密林上方绽放·火焰灼烧花粉生致幻烟雾·树木结成密封囚笼）把现任五影困在内、用花粉慢慢摧毁其神经系统等死。[苏晓](苏晓.md) 循着冲天尘遁白光找来——斑见他「怎么看都不像好人」（战场上的恶徒一般是己方）便没直接动手；苏晓自报「**带土的合作者**」「这场战争是我挑起的」（斑不信、认定这是他生前布好的局）。

## 与苏晓的攀谈 · 长门的轮回眼（十五卷85章·进行中）

苏晓掌握的秘密多到能扯出他的全部计划——一上来就点破「**[长门](长门.md) 那双与身体严重排斥的轮回眼，其实就是斑你的眼**」（漩涡一族本不能觉醒轮回眼、长门却被自己的眼严重透支身体·很不合理）。苏晓曾用柱间细胞替命不久矣的长门续命，意外发现长门那透支的身体**毫不排斥柱间细胞、甚至对其侵蚀产生抗性**——这番话听得斑心惊（那是他的眼·若出问题是他无法接受的损失）。两人的攀谈正把斑往「最强之斑」/ 同盟的岔路上引。

## 被苏晓忽悠 · 谈拢同盟（十五卷85–87章）

苏晓拿捏住他「傲慢 + 不是守序阵营 + 无视魅力影响」的性子、又拿大蛇丸当背锅侠，**用真假掺半的「情报」动摇他**：① 抛出**大筒木一族与神树的由来**（见世界页）、暗示带土把无限月读投到封着辉夜的月亮上是引火烧身；② 点破解开辉夜封印的五条件（轮回眼 + 外道魔像 + 神树 + 九尾兽 + 海量查克拉）正是月之眼计划要件；③ **撼动他对宇智波石碑的信任**（「石碑在木叶地下、移植两只轮回眼即可阅读 / 篡改、不需宇智波血脉、长门也能」——其实苏晓没见过石碑、全靠查资料忽悠）；④ 泼脏水「黑绝很可疑」。苏晓的诀窍是**只给真假未知的线索、绝不替线索的真实性背书**——逼他自己去串联判断（人不信别人、总信自己的判断）。他终于动了与苏晓合作的念头（苏晓知道太多、危险、但身份可靠·非忍村间谍）。苏晓借此**触发隐藏任务「复生之斑」**（详见 [苏晓](苏晓.md)）。

## 挣脱秽土转生 · 完全体须佐能乎（十五卷88–89章）

[大野木](大野木.md) 以尘遁限界剥离斩掉他半个上身（秽土转生·瞬间恢复·左胸还现出柱间细胞的面庞〔木遁分身之源〕）、吹散花粉救醒被花界囚困的四影（五代雷影已**死于花界降临**·柱间所创）。他先用木遁分身戏耍纲手 / 大野木 / 水影代理 / 我爱罗，再以**完全体须佐能乎**（几十米高·双长刀）一击斩飞水影代理上半身（苏晓子弹补头）。中途**佐助 + 鼬两兄弟解决了大蛇兜、秽土转生被解除**——但他**自己懂秽土转生的印、主动重新结印「秽土转生·开」**，从此**不死之躯 + 无限查克拉 + 不受施术者控制、自由行动**，战斗继续。

## 与苏晓合谋 · 再战柱间（十五卷93–94章）

[大蛇丸](大蛇丸.md) 秽土转生出 [千手柱间](千手柱间.md) 后，斑最忌惮的正是这位宿敌（鸣人 / 佐助 / 波风水门都不放在他眼里）。苏晓抛来一个「奇思妙想」的计划——**让带土复活斑时把柱间一并复活、再由苏晓单挑柱间**，双方互相吸引火力 / 断后（斑一度疑苏晓是内奸·准备豪火灭却烧他，听完才将信将疑）。复生需约 1 小时成十尾人柱力。他以**完全体须佐能乎**再战秽土转生的柱间（双方各约生前 7 成实力·斑略强），靠苏晓 1 公里外蜘蛛女皇的火力支援占优、终把柱间引离主战场（顺便须佐踩死一个「手贱」结印的上忍）。

## 真正复活（十五卷96章）

整盘棋的真相揭晓：他让带土移植轮回眼、根本不是为了月之眼计划，而是**为了让带土用轮回眼复活自己**。黑绝附身带土夺眼后，**斑被真正复活**（不再是秽土转生·有了跳动的心脏与呼吸·终于「重新感觉到这个世界」），代价是那只轮回眼复活一人后即破碎（无法再造轮回眼·另一只复活了柱间）。复活的斑约有生前 7 成实力，却仍是鸣人 / 佐助 / 波风水门都不放在眼里的存在——他与苏晓按计划演双簧、把柱间留给苏晓，自己赶往主战场取回另一只眼。

## 关系
- **[带土](宇智波带土.md)**：被他当祭品的「替身」——带土多年假冒他的身份、最终被诱导用轮回眼复活了他。
- **[千手柱间](千手柱间.md)**：挚友兼终生宿敌（被大蛇丸转生→又被黑绝复活·斑视为唯一忌惮的对手）。
- **[大野木](大野木.md)**：战场上欣赏的对手（视死如归·斑从喊他「小鬼」改为直呼其名）。
- **[长门](长门.md)**：曾被植入斑的轮回眼（漩涡一族·被轮回眼透支身体）。
- **[苏晓](苏晓.md)**：战场上攀谈的「带土合作者」——苏晓握有他全部计划的秘密、正引他走向变数。
- **[大蛇兜](大蛇兜.md)**：把他秽土转生出来的人。
