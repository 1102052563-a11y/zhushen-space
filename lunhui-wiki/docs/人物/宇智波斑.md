# 宇智波斑

火影世界的传奇强者、宇智波一族最负盛名的人物——一头黑发披肩、生有一双**轮回眼**。他生前布下的「月之眼计划」是整个第四次忍界大战的底层棋局：[带土](宇智波带土.md) 这些年一直假冒「他」的身份行事，正是他死前的安排（带土只是替身、用以推进计划直到他被复活）。

## 被秽土转生 · 战场登场（十五卷84章）

[大蛇兜](大蛇兜.md) 秽土转生出的**最强死者，没有之一**。开战后他以「**花界**」（房屋大小的花朵在密林上方绽放·火焰灼烧花粉生致幻烟雾·树木结成密封囚笼）把现任五影困在内、用花粉慢慢摧毁其神经系统等死。[苏晓](苏晓.md) 循着冲天尘遁白光找来——斑见他「怎么看都不像好人」（战场上的恶徒一般是己方）便没直接动手；苏晓自报「**带土的合作者**」「这场战争是我挑起的」（斑不信、认定这是他生前布好的局）。

## 与苏晓的攀谈 · 长门的轮回眼（十五卷85章·进行中）

苏晓掌握的秘密多到能扯出他的全部计划——一上来就点破「**[长门](长门.md) 那双与身体严重排斥的轮回眼，其实就是斑你的眼**」（漩涡一族本不能觉醒轮回眼、长门却被自己的眼严重透支身体·很不合理）。苏晓曾用柱间细胞替命不久矣的长门续命，意外发现长门那透支的身体**毫不排斥柱间细胞、甚至对其侵蚀产生抗性**——这番话听得斑心惊（那是他的眼·若出问题是他无法接受的损失）。两人的攀谈正把斑往「最强之斑」/ 同盟的岔路上引。

## 关系
- **[带土](宇智波带土.md)**：生前安排的「替身」——带土多年假冒他的身份推进月之眼计划。
- **[长门](长门.md)**：曾被植入斑的轮回眼（漩涡一族·被轮回眼透支身体）。
- **[苏晓](苏晓.md)**：战场上攀谈的「带土合作者」——苏晓握有他全部计划的秘密、正引他走向变数。
- **[大蛇兜](大蛇兜.md)**：把他秽土转生出来的人。
