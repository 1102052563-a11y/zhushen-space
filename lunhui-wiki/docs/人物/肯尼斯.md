# 肯尼斯

全名 **肯尼斯·艾尔梅洛伊·阿其波卢德**——[Fate/Zero 世界](../世界/任务世界/fate-zero.md) 中 [迪卢木多](迪卢木多.md)（Lancer）的御主、[魔术协会](../势力/index.md) 的讲师、天才魔术师，在魔术师中威望极高。

- 因**召唤媒介丢失**，他本想召 Rider、却被迫召出了 Lancer [迪卢木多](迪卢木多.md)——偏偏迪卢木多眼下有颗「令女性无条件爱上」的魅惑泪痣，而他有个漂亮未婚妻，遂**满心猜忌、怕被戴绿帽**（自觉「满头黄发开始发绿」），对忠诚的迪卢木多极不友善。
- **十一卷16章**：迪卢木多被苏晓压制、连失双臂，他连用两枚令咒（传送 + 治伤·但断臂被 [万仞宗](../力量体系/装备道具.md) 封死无法恢复），最终**弃迪卢木多断后、独自撤退**（临走承认「你是名不错的骑士」）。迪卢木多一死、他便失去继续参战的资格。

## 关系
- **[迪卢木多](迪卢木多.md)**：他的从者 Lancer（被他猜忌、又被他抛弃）。
- **[魔术协会](../势力/index.md)**：讲师。
