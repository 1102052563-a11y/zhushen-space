# 鼠大人（虚空暴徒）

[二阶生存试炼](../世界/任务世界/T12K星球-生存试炼.md) 的**裁判**之一——自称「鼠大人」，也叫「**虚空暴徒**」（吞噬岛边缘共有十几名这种裁判）。

- **形象 / 能力**：身高约 1.2~1.3 米的**鼠人**（老鼠头颅 + 人类体型 + 鼠的四肢 / 尾巴）；一双**电子眼**、手臂等为**金属义体**（被自己激光枪后坐力震碎后，由数之不尽的**纳米机器人**瞬间重生）；腰挂**激光枪**（蓄能极快·一击把契约者爆成血雾连渣不剩）。苏晓判断它「**不是现在的自己能对抗的**」。
- **性情 / 立场**：凶残随性——**仅因一名契约者「目光不爽」就一枪秒杀**（杀鸡儆猴）。立的铁律 = **试炼期间禁止离开吞噬岛，违者一律处决**。它疑是乐园雇来的工作人员，却满腹牢骚（「该死的乐园居然把我拉来当苦力」）。
- **身世（伏笔）**：自称是虚空中某位「**大人**」的部下——当年随那位「大人」**在虚空中纵横、喜欢什么就抢什么（美女 / 宝物 / 美酒）**，后那位大人不知去向、留下他们这些小卒。**虚空生物 / 那位「大人」**是乐园背景的一条伏笔（参见乐园对契约者的警告「勿与虚空生物交流」）。

## 镇压黑色森林（八卷70–71章·试炼终局）
试炼最后一天黑雾散尽，**黑色森林的强者全数涌出**——67.5 万 [不死者](../世界/生物万族/index.md) + 几名完全苏醒的「暗王类」虚空流放犯，普通区瞬间沦陷、90%+ 契约者逃回红土区。**吞噬岛的真实危险远超试炼难度**——若放任，最终存活契约者不会超过十人，故**乐园授意鼠大人出面镇压**：它独自站上石桥拦住不死者大军（一名苍白光头老者上前质问「为何不许我等通过」，被鼠大人一句没说完的话吓得神色惊恐、转身就走——「鼠大人不可怕，可它背后的人太可怕」）。它放话「三小时后随便进攻红土区」、镇住场面。最终 **501 人登岛、仅 36 人存活**（存活率不到十分之一）。这也侧面坐实它「背后那位『大人』」之可怕。

## 虚空角斗场 · 轮回乐园的保险（十四卷50章·再登场）

[恶魔古堡游戏](../世界/任务世界/恶魔古堡游戏.md) 第五场「**虚空角斗场**」上，鼠大人再次出现——他手背刻着**临时轮回印记**，即是**轮回乐园派来的临时代表 / 保险**。原因：苏晓、皮胖等尚未晋三阶，被推上面向整个虚空的角斗场（300 万观众、灭法之影身份已被爱丽丝传单曝光）实属过早，乐园**不想过度干预虚空规则**，便派鼠大人坐镇——**死于斗技乐园不管，但斗技之外的危险（如被仇视灭法者的观众围攻）乐园绝不坐视**。苏晓见他手背的轮回印记，悬着的心才放下。登场时他**搂着表演赛的「啦啦塔星」萌妹子、灌烈酒、要小萌物捶腿**、公然挑衅主办方却无人敢管。

**逼退法师贤者瑟菲莉娅（十四卷54章）**：苏晓首战登场时，灭法世仇 **[瑟菲莉娅](瑟菲莉娅.md)**（守序阵营大人物）起身、凝起「灼热之痛」诅咒锁定他——鼠大人当即持古旧符文猎枪顶住、把两人传送到场外：一枪**穿透空间**（音爆扩散·空间崩碎）逼她以「禁」字符文勉强挡下、指尖渗血；再上膛威胁「这一发是**源爆弹**」，终于逼她收手回座。这正是他「斗技之外的危险乐园不坐视」的职责体现（他评瑟菲莉娅「虚伪的女人」·守序大人物只会暗杀、不敢明面出手）。

> **「背后那位大人」再被点明（十四卷49章）**：鼠大人背后的恐怖存在有几十个称号——**星球破坏者 / 战争厄难 / 生灵灾厄之源 / 虚空剑豪 / 虚空最强神经病 / 酒疯子**；它所到之处星球主动撤防（反抗即灭族）、直接间接死其手者**以亿计**，**据说消失在黑渊最深处**（生死不明·极护短：曾有种族围攻鼠大人、不久便从虚空中消失干净）。故借爱丽丝十个胆子也不敢招惹鼠大人。这与「[剑魔](剑魔.md)」「刀魔」等虚空剑道强者伏笔遥相呼应。
