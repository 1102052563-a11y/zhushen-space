# 挽歌（圣恶交融的二五仔奶妈 · J 先生）

[幽鬼大陆](../世界/任务世界/幽鬼大陆-五阶考核.md) 中 [苏晓](苏晓.md) 暗处的对手之一——白色神职长袍、肌肤吹弹可破、气质弱气的**奶妈系契约者**，外表如圣女、内在却是「**一团红黑交融、粘稠哭嚎的恶**」（她吞契约者心中的「恶」为力·圣与恶都是力量、她只得其恶·连苏晓都怀疑这还是不是人类）。天生体质弱、无战斗天赋、还胆小——全靠一个逆天天赋苟到四阶。

## 微笑天使（S 级天赋 · 二五仔之源）

`天赋·微笑天使（主动·S 级）`——每个世界内可**随意转换阵营 4 次**，每次转换获得对应的**伪装身份**（前一身份消失·对契约者无效）·每转一次任务奖励 −10%。这让她成为「天知道最终站哪边」的超级二五仔——宗旨就是**怂到最后、苟到最后**（认怂 + 累积·经历的世界最多）。

## 幽鬼大陆 · 反复横跳（二十四卷27–46章）

- **80 万劝退**：晋升任务与苏晓敌对、她第一反应是远离这疯子，曾**出 80 万乐园币让苏晓去刺杀琼斯议员**（隐藏分支任务·想以「不正面敌对」避开苏晓·被搅黄）。
- **杀光队友**：曾混在耳钉小队（耳钉 / 屠夫 / 曼陀罗）——三人后来**全被她吃干净**（只剩她）。
- **试探 + 出卖**：在废弃公寓现身试探重伤的苏晓（提议「恩怨两清」·实为试探苏晓虚弱程度·若答应即出手），顺手**出卖盟友 J 先生**的情报；被苏晓识破（不合作 + 命莱因哈特放电赶人）、随即被处刑机关 + 第七部队**通缉**（疑似墓园破坏者）。她笑嘻嘻地继续苟。

## J 先生（神秘契约者 · 真身炼狱 · 与罪恶合作）

戴**方片 J 金属面具**、玩扑克牌、涂口红的嘴、说话玩味感十足的神秘契约者——超级老阴哔、且足够「**稳**」（单挑苏晓敢周旋·遇苏晓 + 哈罗德组合则坚决退避）。与鬼环首领 [罪恶](罪恶.md) 临时合作（出力三件：用传说级道具窃取 [格斯](幽鬼大陆配角.md) 记忆助罪恶进地下室〔顺带把格斯标记成半妖魔化〕 / 帮罪恶潜入处刑机关 / 带走四间眼室去巴丹沙镇激活）。

- **扑克牌即死能力**：抽牌（梅花 9 / 黑桃 A 等）在敌人身上烙倒计时数字、归零即金色火焰爆炸（数字爆点随机移动·黑桃 A = 绝杀）——曾标记格斯。
- **真身「炼狱」 + 吞「不朽之王」**：他召出鬼环信奉的「不朽之王」（实为虚空一个二流能量生物·冒用恶魔族某老恶魔的封号）后**反手吞掉**——因他真身**炼狱**早把那位真·老恶魔吞了（暗红岩浆眼 / 火焰血盆大口）。夺得**时空之核**后嫌烫手（怕被苏晓任务追踪）、转手扔给挽歌。
- 他对苏晓坦言：「等待的主要原因是——**打不过**，我很久之前尝试过」（曾被苏晓按地上砍断两臂一腿）。苏晓疑其是已被自己杀过的红色小丑 / 希 / 死兆之一（存疑）。

## 关系
- **[苏晓](苏晓.md)（库库林·白夜）**：晋升任务的敌对方（出 80 万劝退未果·试探重伤的他·被其通缉）。
- **[罪恶](罪恶.md) / J 先生**：临时合作的鬼环 / 神秘契约者（挽歌出卖了 J 先生）。
- **耳钉 / 屠夫 / 曼陀罗**：被她吃光的原小队队友。
