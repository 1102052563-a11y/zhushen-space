# 泰莉儿

身材高挑、颜值非常能打，一身浅红色礼裙、踩着金属高跟鞋的女王般身影。深渊战役中，她是虚空之树公告的**三名最重要人员之一**:

1. 轮回乐园·001 号猎杀者 [团长](星空座/团长.md)——虚空区域总指挥官
2. **天启乐园·001 号战斗天使 泰莉儿——后勤总管**
3. 轮回乐园·002 号猎杀者 [苏晓](苏晓.md)（库库林·白夜）——最强增援小队队长

## 💰 后勤总管的权重

她的气场相当足。登场第一句就是意味深长的:

> 「我当是来见谁，原来是**轮回乐园的 001 号猎杀者大人**。」

（外界早有传闻，战斗天使最强的那位是团长的老相好；她此刻的语气「可比老相好的身份更有故事」，想来是以前两人的确相好过一段时间，后来因某件事分开、甚至已有段时间没再相见。）

看似她只有支出天启乐园资源的权限，可**本次己方总资源中天启乐园占比 68.5%**（轮回乐园约 10%）。她有**资源支出权**，但不是随意支配权。

⚠ 顺带记下两个阵营的定位差异:天启乐园之所以这么富，既因其初始公证体系，也因其契约者的确擅长资源的开发与运营（万族界的资源体系公证就是轮回乐园外部委托天启乐园做的——若由轮回乐园运营，万族界每月最多产出 30 个单位资源且消耗世界资源；由天启乐园运营，每月最起码 180 个单位，且对世界资源消耗很少）。但若论麾下契约者的综合素质与战力，**轮回乐园绝对是最强的**。

## 🤝 三方会面（54卷216章）

团长为苏晓争取增援补给额度的拉扯:

| 发言 | 内容 |
| --- | --- |
| 团长 | 「现在你是财神，财神爷，给个态度，能给白夜多高的补给额度。」 |
| 泰莉儿 | 「不会太高。」 |
| 团长 | 「必须高。」 |
| 泰莉儿（试探） | 「这么说，最后关头是白夜去深渊对付深渊之主？」 |
| 团长 | 「当然是他，难不成是我啊。」（说完自己都笑了——**他还真去过**） |
| 泰莉儿 | 「那这次谁开路。」 |
| 团长 | 「**白夜不需要别人开路。**」 |
| 泰莉儿 | 「这样吗，**最多 10%**，不可能更多，而且是**他晋升无上后**我才能完成申请。」 |
| 团长 | 「**30%**。」 |
| 泰莉儿 | 「深渊净化装置怎么维持？」 |
| 团长 | 「**如果赢不了，深渊净化装置维持多久都没意义。**」 |
| 团长（补刀） | 「你想不到，杀入深渊是什么感觉，无穷无尽的黑暗和深渊生物……」 |

——泰莉儿妥协。她走后，团长叹息:「我忽然感觉，我搞定这女人，不比我上次杀入深渊简单多少。」

⚠ 从认识团长到现在，他这一天叹的气比过去相加还多。当他半开玩笑地向苏晓讨建议时，苏晓说「我有个朋友，算是情感专家」——「哦？谁？」——「**[格林·吉莉安](格林·吉莉安.md)**。」（团长默不作声地看着苏晓，苏晓起身告辞。）

## 😅 「你别搞！」

苏晓把 [莫蕾](#)、月使徒、豪妹强制征召进最强增援小队后:

> 泰莉儿:「她们是你的辅助？她们也没辅助能力，哦～，没看出来，原来你这么严肃的人，喜欢莫蕾这样沙雕的，真是奇妙。」<br>苏晓:「**有她们三个，我能拿四份增援收益。**」<br>泰莉儿 ＆ 团长（同时）:「**你别搞！**」<br>（前者是提醒，后者则是很明确地告知别这么搞——团长很明确地知道，白夜是真能干出这事的。）

随后她试探性地问:「白夜，你现在多少万虚空之树信誉度？」——「130。」——「嗯～，少了点，但也还不错了。」——团长补刀:「**他不是少了点，他是根本没有万，他只有 130 点。**」

> 泰莉儿:「你是怎么成为最强增援小队队长的？」<br>苏晓:「**我实力尚可。**」<br>泰莉儿:「嗯，好吧，你说服我了。」

⚠ 后续苏晓的悬赏操作果然成真:天启乐园既要支付风海战场的悬赏金（800 万＋1080 万盎司），又要为苏晓「专属属性强化仓」的费用固化承担补正（估算约 **4800 万盎司**的资源补正），还被迫接受了他到天启乐园强化装备的休整申请。
