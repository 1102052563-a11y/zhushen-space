# 琵卡（唐吉诃德家族 · 黑桃干部）

[唐吉诃德家族](../势力/index.md) **黑桃部门的最高干部**、[多弗朗明哥](多弗朗明哥.md) 的护卫（实力在家族内排得进前三）——[苏晓](苏晓.md) 落点的「黑桃号」正归他管辖。一名 40 岁、三米多高、全身肌肉扎结的壮汉，**却生着一副尖细的童音**（体型与声音反差极大·他自知声音搞笑、对四皇子嗣作战时宁可沉默也不想给少主丢人）。

- **石石果实（超人系）**：可操控石头 / 把身体石化、配合武装色霸气与剑术（持两米多长大刀）。船上不利其发挥；陆地上是劲敌。

## 被苏晓「强行救命」（十八卷8–11章）

德雷斯罗萨海战中他与玲玲长子 [斯佩罗](夏洛特·佩罗斯佩罗.md) 厮杀、本占下风，[苏晓](苏晓.md) 在水下搅局把双方落脚的「小船岛」弄沉——斯佩罗造糖船撤、琵卡却无船可撤、被巨型冰雹砸碎甲板筏后呛水昏迷。苏晓**反手「救」了他**（其部下其实全是苏晓杀的、苏晓硬当救命恩人）：让阿姆造冰层落脚点、用「反重力排水法」把他抖醒。

- 琵卡醒来握刀警惕、并不领情（苏晓那 **30 点魅力毫无作用**·琵卡只觉苏晓「血气比烧杀抢掠的海贼还夸张」、随时想逃）。苏晓亮明来意——**投奔多弗朗明哥**（斯佩罗的首级是投名状 + 救命之恩·多弗朗明哥不敢拒收烫手的他）。
- **两个海上路痴**：苏晓「救」琵卡的真正动机之一是**自己不知道怎么去德雷斯罗萨**——结果琵卡也是海上路痴（新世界须暗紫 / 淡金以上的指针才能辨向），「确认过眼神，都不知道航路」。最终苏晓发信号弹引来一艘白胡子旗下的商船、由其送二人抵达德雷斯罗萨。经此初步交涉，苏晓**正式加入唐吉诃德家族**（完成晋升 / 主线任务首环）。

## 关系
- **[多弗朗明哥](多弗朗明哥.md)**：效忠的家族首领（黑桃干部·护卫·家族前三）。
- **[苏晓](苏晓.md)（白夜）**：「强行救命恩人」+ 引他入伙唐吉诃德家族的契约者（琵卡半信半疑、却无可奈何）。
- **[斯佩罗](夏洛特·佩罗斯佩罗.md)**：海战交手对手（因苏晓搅局双双落水·斯佩罗被杀）。
