# 老烟枪

轮回乐园契约者、[蚁王讨伐队](../世界/任务世界/全职猎人-蚁王篇.md) 五成员之一——年约 50 的干瘦男人，标志是叼着一根**至少 5 厘米粗、30 厘米长**的巨型大烟（尺寸只配牲口叼·屋里烟雾缭绕被人吐槽「不是契约者活不过 60 岁」）。

他是个**「老好人」、却心怀鬼胎**：平常谁都不得罪、爱出来打圆场调解，可一旦涉及自身利益就变得凶狠（尘一针见血戳穿他调解时「其实真实想法是杀掉房间里其他人」）。他得到 [尼特罗](艾萨克·尼特罗.md) 认可、是三阶中的高手；他与水咲 / 尘组队、并请尼特罗去寻其他契约者来对付比原著更强的蚁王。他清醒地指出「指望尼特罗引爆贫者的蔷薇把蚁王炸半死，基本不可能」。

## 起义军首领 · 双面人格（十六卷33–34章）

他原是个**服刑犯**、最大爱好是**集结狱友暴动越狱**（却被狱友出卖过很多次）——「生在古代绝对是起义军头领、太平盛世也要起义」。**精神严重不正常（妄想症）**：不犯病时是滴水不漏的「老好人」，一旦进入「起义军模式」就慷慨激昂、滔滔不绝（能把四人合作杀到五阶的「大好蓝图」都规划出来·吵得苏晓 / 水咲 / 尘头顶冒「怒气条」），见尼特罗推门又瞬间切回老好人。仇人极多——苏晓的旧识 [炎辰](炎辰.md) 就曾放话「要把老烟的舌头揪下来塞进菊花里」。他的**念能力是大范围感知**（约 4 公里·靠它定位三护卫的位置）。攻王宫时他理智地主持分工：三护卫里尤比最易（已被苏晓杀）、普夫能力麻烦但身体弱、猫女最难——遂定下「苏晓单挑猫女、他 + 水咲 + 尘围攻普夫、尼特罗拖蚁王」。

## 烟系能力 · 围攻蚁王（十六卷38–42章）

他的念能力全围绕那根巨烟展开——**烟阵**（八卦 + 法阵的「中西结合」·屏蔽感知 + 替队友承受攻击）、**烟使 / 烟分身**（吐烟凝成与他气息完全相同的分身·货真价实的「又一个老烟」）、**烟替身 / 虚假术**（关键时刻把队友与替身瞬间替换、还能用烟「奶」回队友伤势）。围攻蚁王时他靠这套替水咲挡命、用大范围感知（4 公里）调度战场。他借了个豪乳奶妈（中型冒险团出借·酬劳一片 3 公分蚁王心脏·够那团全员最低评价完成主线·他没签契约）。战中他与水咲都「有所保留」、并提出「弄死蚁王后要一半心脏」。

## 助攻杀蚁王 + 偷尸抢心脏（十六卷43–53章）

第一轮集火蚁王时他功不可没（烟阵掩护）；进「起义军模式」嘴炮「帅不过三秒」。蚁王选错方向冲他、他以烟雾包裹蚁王、在烟内用**压箱底本领削蚁王 24% 身体防御力**（双臂被扯断为代价、后以烟雾再造假臂）——正是苏晓那枪能爆头的关键。蚁王死后他**挖地洞用烟雾偷走蚁王尸体**想独吞心脏，被苏晓「寂灭公爵仅剩 1 子弹」威慑、只得把尸体抛给水咲；最终他与水咲分得 1/3 心脏（分尘 1 片续联手）。他拒绝沙暴献星火冒险团的交易——因他自己**曾有冒险团、却被团员「起义」推翻**，遂把那团改成了单人性质（「我可以起义别人，别人起义我就算了」）。
