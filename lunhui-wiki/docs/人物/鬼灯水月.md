# 鬼灯水月

[火影忍者世界](../世界/任务世界/火影忍者.md) [大蛇丸](大蛇丸.md) 的部下——白发、一口鲨鱼般尖牙，**鬼灯一族**、能将身体**化成水**。一副无精打采、刚睡醒的懒散模样。

- **伏笔兑现**：他正是雾忍狱友 [鬼灯残月](鬼灯残月.md) 临死前托苏晓带话的对象（鬼灯残月被处死前留下「鬼灯水月」的伏笔）——如今在大蛇丸麾下登场。
- **登场即逃（九卷50章）**：大蛇丸设局围困 [苏晓](苏晓.md) 时他与 [香磷](香磷.md) 被叫来，但局是 [宇智波鼬](宇智波鼬.md) 的钓鱼局；眼见不妙，他**化成一滩清水渗入地缝**、护着真大蛇丸一同逃脱。

- **入鹰小队 · 残月死讯（九卷69–70章）**：大蛇丸「死」后他归入 [宇智波佐助](宇智波佐助.md) 的「鹰小队」。[苏晓](苏晓.md) 当面告知他「**[鬼灯残月](鬼灯残月.md) 已死、临终留话让你别报仇**」——水月暴怒不信（亲眼证据是苏晓与残月同囚雾忍村、目睹其被押走处死）。
