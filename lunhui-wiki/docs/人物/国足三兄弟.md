# 国足三兄弟

轮回乐园里一支远近闻名的**逗逼契约者三人组**——人称「**锤子组合**」（能打 + 能抗 + 能奶、配置完整）。三人性格鲜明：**老大**爱在战斗前后「摆造型」（屡屡因此差点被砍）、**老二**口头禅是「大哥说的对啊」、**老三**是个憨直得有点傻的实在人。[苏晓](苏晓.md) 的老熟人（八卷拍卖会上占 49 号座；更早在东京喰种 / 海贼王世界交过手——老大曾被苏晓「打出翔」、对外只肯说「平手」）。

- **全职猎人世界·凑热闹（十卷18–19章）**：苏晓单挑斩杀 [独眼巨怪](../世界/生物万族/index.md) 后，他们和一对「兄贵组合」混在想偷袭苏晓、抢淡金宝箱的「作死小分队」里。老大本想上前「摆个造型」搭话，却被埋伏的旅团 [薇妮](薇妮.md) 一枪打中老二胸口——老大当机立断**抓着老三的头发 + 老二的腰带、用某种位移技带二人化作残影逃命**（跑得比兔子快·老三一路惨叫）。这场骚动恰好替苏晓吸引了火力、掩护他骑布布汪突围。

- **拍卖场攻防战 · 真本事（十卷52–58章）**：[红色小丑](红色小丑.md) 放 [尼拉潘](../世界/生物万族/index.md) 大军围攻拍卖场时，三兄弟主动跑来帮 [苏晓](苏晓.md) 守场，露出了远超「逗逼」印象的真本事——
    - **能量锤升级**：原本只能当一次性控制技的金色能量锤，被放大成**常规武器**，每击附带 **0.3~2 秒眩晕**（三人对一个目标「锤锤眩晕」、免控不强者必被活活锤死）。
    - **三主坦 + 互奶**：三人皆一流主坦、还能互相 / 给队友治疗（同时施治疗技、奶量胜过大部分奶妈，救活了重伤的结界师情侣）。
    - **守后门主力**：三兄弟挡在结界后门缺口、其余契约者只管输出，把后门防得滴水不漏；在契约者中威望颇高（分宝箱由他们主持）。老大有大团经验、提议「结界留缺口形成攻防战线」。后又被薇妮（旧怨·曾一枪打老二）一上膛吓得连连告饶。

- **突袭小队 · 与薇妮的恩怨（十卷58–65章）**：加入反袭杀 [红色小丑](红色小丑.md) 的六人小队。揭出他们与 [薇妮](薇妮.md) 的旧账——「**国足三贱客**」曾在「甲铁城世界」「黑子弹世界」**两度偷袭生擒薇妮、抢光她的战利品还围着圈踢羞辱**（放话「不服再追杀我们啊」），把薇妮气到吐血、追杀他们三个衍生世界（双方都没痛下杀手、故非死仇）。此战为杀小丑暂时言和。突袭中三兄弟贴身护着苏晓突围、又在酒店一楼拖住旅团成员 [芬克斯](../势力/index.md) 等。

- **恶魔古堡游戏 · 生命链接（十四卷1–5章）**：三兄弟也来参加 [爱丽丝恶魔古堡游戏](../世界/任务世界/恶魔古堡游戏.md)（强者之证明终关·9 人参与之一·与苏晓在一层重逢、习惯性装作互不相识）。一层「命运」游戏里三人**连开三枪命运主宰皆空枪**（老三还想换自带的白板左轮、被狼管家以「你的命运会很波折」劝住·开枪前先把装备转入团队空间「留遗产」·胆识过人），得「爱丽丝的祝福（友情）·**生命链接**」（永久·三人任一生命 <30% 时所受伤害的 70% 由另两人分担·直到三人都 <30% 或一人死才中断）——对本就能抗能奶的三圣骑士是**生存力的质变**（往后想集火秒掉其中一人难如登天）。

- **恶魔古堡「MVP」（十四卷28–29章）**：第三场「欺诈」里，**真国足老大恰是那名抽到「直接通过资格」的幸运者**——他被传送出局、由爱丽丝分身顶替（爱丽丝便伪装成「国足老大」、用音乐逼猫女换房号牌）。第三 / 七白昼他作为幸运者只能给「无身份」的图案线索，便绞尽脑汁画了精准的「**O＊**」（= 破碎眼镜 + 破碎蛋·唯爱丽丝伪装的「国足老大」同时关联两者）——被苏晓 + 老三破出、一举质疑成功通关。游戏后三兄弟摆造型自封「**国足在此参上，MVP 非我等莫属**」（老三的「破碎蛋」直觉也立了功）。

- **恶魔古堡「强者之森」 · 有奇遇（十四卷40章）**：第四场求生竞技里三兄弟在森林中**有奇遇**、不惧客人追杀（大小锤战术 + 生命链接太硬），还顺手把皮胖之外抢了老三食物——结算时**三人总积分 27431**、列第二（仅次苏晓 249430·胜过单人皮胖 24167），稳过、随众人登五层天梯。

- **虚空角斗场 · 命运相连（十四卷52–53章）**：第五场 [虚空角斗场](../世界/任务世界/恶魔古堡游戏.md) 首轮，三兄弟竟**只领到一张号牌、三人同场**——公证处认定他们「**命运相连**」（虚空视作一个个体·其一死则余者厄难重重活不久·这是高于种族天赋的罕见天赋）。这场坐实了三兄弟「独行侠克星」的恐怖控制：**小型能量锤命中即定身**（中武器 / 铠甲 2~3 秒、中身体 3~5 秒·无判定·还能屏蔽感知）、**大型能量锤命中高几率眩晕 0.3 秒**；三人轮替（两人锤、一人歇着回体力·老二在旁扎马步「替补」装气势），把维坦星战斗种族鳄鱼「**古厄**」按进沙坑锤了整整十分钟（老大还误来一记「铁锤砸蛋」），把这条不畏战死的硬汉锤到屈辱落泪。[苏晓](苏晓.md) 看完终于懂了 [薇妮](薇妮.md) 当年为何被他们锤到想宰人——但也判断他们「攻击力太弱、不足为惧，只忌惮被围攻晕住」。

- **虚空角斗场 · 「黑心奶妈」 + 弃权（十四卷64–65章）**：第三轮抽到虫族 [斑克](斑克.md) 当对手，三兄弟果断折断号牌**弃权**（第五轮已完成、再拼无谓）。弃权前还把被激光箭扎成「刺猬」的侦探小萝莉宰了一笔——**明码标价「奶一口 2000 乐园币、拔一根箭 3000、保证拔箭期间不死」**，八根箭算下来 3.4 万、打八五折 3 万（小萝莉没钱·只能用装备抵）；拔箭时她杀猪般哀嚎、还踹了老大一脸。苏晓看了直拍大腿「怎么没想到这是商机」（自己学样去坑石头人却推销失败、白损 2 万）。

- **联盟星 · 围殴不死暴君（四十三卷18章）**：三兄弟自五阶起被苏晓的变强速度刺激、开始「氪金」狂囤世界钥匙——别人过一个世界进度、他们至少过三个，极限压缩乐园停留时间、玩命刷任务世界（天生乐观、宁做逗逼不做咸鱼）。在 [联盟星](../世界/任务世界/联盟星-死亡圣杯.md) 友克市郊外，他们刚锤死一只八阶野生小 boss（老大照例上演「甚是感动、吾泣、拳弟、弟泣」的摆造型戏码），就撞上来抢击杀奖励的天启乐园不死坦系 [暴君·图尔](暴君·图尔.md)。暴君全开重力领域压他们，老二一个「双手比心」的骚姿势直接给暴君来了记**物理强制眩晕**（心脏似被一把攥住）——三兄弟极短时间摆成三角站位、抡起能量锤对着抱头蜷缩、动弹不得的暴君一顿「80/欧拉/木大」的鬼畜乱锤，把这位天启不死暴君锤到只想知道「自己遇到了三个什么东西」。国足三兄弟对付单个契约者近乎无解（除非对方能豁免物理强制眩晕）。

## 🆕 树生世界 · 爆衣登场 & 3V1 真男人大战（46卷1–3章）

三兄弟一同进入树生世界。飞船抵达中转站时，他们本想拉 [苏晓](苏晓.md) 合作，见苏晓不理会（老二判断:苏晓这次要么做遭全场痛恨的事、要么来了结私仇），便识趣不强求。下船时三人摆出各异健美姿势、「撕拉」一声爆衣成功——三个只穿健美三角裤（老大那条还是骚气的紫色）的猛男「黑暗势力·国足三兄弟登场」，惊呆半船人。他们的看家宝是从苏晓处买的钉锤【末陨】:核心能力本是把敌人拖进封闭圆柱空间「1V1 真男人大战」，到了三兄弟手里质变成**3V1 的欧拉欧拉**。飞船上他们又与 [咕噜](咕噜.md) 互喷（「手下败将亦敢汪汪狂吠」），却被咕噜一句「吾父」惊到目瞪口呆。
