# 阿尔冯斯·艾尔利克

[爱德华·艾尔利克](爱德华·艾尔利克.md) 的弟弟。童年人体炼成失败时**失去了全部身体**，灵魂被哥哥以一臂为代价**固定在一副全身铠甲上**（铠甲内是空的、内壁有一道血印固定其灵魂）——可见炼金术由灵魂发动、与肉体无关。

- **处境**：与哥哥、父亲、师傅同为 [瓶中的小人](瓶中的小人.md) 的「人柱」候选。
- **历程**：七卷11–13章被叛变的人造人 [格利德](格利德.md) 绑走、逼问「灵魂提取固定术」（铠甲内一度被塞入合成兽马黛儿限制行动）——苏晓借「解救国家炼金术师家属」为名闯入、击杀格利德（并顺手斩了铠甲内的马黛儿）。七卷29章随哥哥来 [布里克兹要塞](../世界/地点/index.md) 找小女孩学炼丹术。

## 出处
- 七卷2、11–13、29章。
