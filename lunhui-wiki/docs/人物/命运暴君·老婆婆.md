# 命运暴君·老婆婆

> ⚠ **她坐在深渊蔓延区一处树根窝棚的长椅上，身着灰黑色粗麻布衣、严重驼背、身体枯瘦、拄着扭曲长木杖，仿佛已经睡着**;⚠ **⚠ 身边两侧是一群拳头大、圆滚滚、肉嘟嘟的黑色小灵体，调皮东张西望却不敢离她太远**——⚠ **一群淘气又可爱的小家伙**;⚠ **⚠ 她视力不好、张嘴时上下加起来不到五颗牙**——⚠ **却是「事件悬赏·命运暴君:击杀老婆婆」的目标，总悬赏额度高达 600 万盎司时空之力**（⚠ 苏晓一见这数字就感到恶意迎面而来、果断屏蔽这悬赏「谁准备好、让谁去吧」）。

## 🕯 「小灭法，你来啦」（54卷110章）

⚠ **她慈蔼地唤苏晓「小灭法」**——⚠ **「老太婆在这等你好久啦，你的命运太难看清，来靠近些、让婆婆看看你的手」**;⚠ **⚠ 她用枯干但温暖的手按在苏晓手掌上一点点摸索、满意地点头**：⚠ **「这我就放心了，老太婆不会提议你去做什么，你只管做你想做的事，婆婆都支持你，小灭法」**（⚠ 她与树根窝棚、那群黑色小灵体，随后被黑暗吞没·⚠ 她离开的下一秒、失去她「误导」的黑暗野兽才发现苏晓、被「极刃·暗狱」+「极刃·血海」清空）;⚠ **⚠ 她「误导」了周边的黑暗野兽**——⚠ **让它们没发现苏晓**（⚠ 这是一位对灭法者纯然善意的深渊存在·⚠ 与「命运暴君」这凶名截然相反）。

## 🔗 关联

- ⚠ **⚠ 「命运暴君」的悬赏是留给别人的坑**——⚠ **苏晓不知她是何人，但 600 万盎司的分量足够他体会「机会是留给有准备的人」、果断避开**;⚠ **⚠ 她看手相的能力、身边的黑色小灵体、慈蔼的性格**——⚠ **让人联想到 [嘟嘟咕咕](#) 那类深渊小家伙**（⚠ 苏晓身上封着六件大爹级原罪物、命运「太难看清」·⚠ 她却只说「你只管做你想做的事」）;⚠ **⚠ 众神之界的深渊背景**:⚠ **本世界的「三大臣」背后是深渊族裔·[卢恩家族](#)，深渊通道多、深渊蔓延区遍布**——⚠ **她这样的深渊存在游走其间、并不稀奇**（⚠ 但她这份对灭法者的慈蔼、在深渊系中极为罕见）。
