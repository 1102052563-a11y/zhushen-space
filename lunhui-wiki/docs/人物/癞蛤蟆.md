# 癞蛤蟆（裁决者）

轮回乐园的一名**裁决者**——虽与 [阿尔托利亚（呆毛王）](阿尔托利亚.md) 没有名义上的师徒关系，却教导了她许久、早把她当**弟子**看待。它心细、护短，为呆毛王的伤情满眼担忧。

## 为呆毛王张罗治疗（42卷77章）

呆毛王为替它弄「黑枫树枝干」跑去渊龙底、被黑暗物质重度侵蚀后，是癞蛤蟆重金请来 [苏晓](苏晓.md) 拔除黑暗物质，治疗期间的器械、药剂材料也都由它提供（苏晓只负责动手）。它一眼看穿呆毛王「无论受多重伤、只要食物够美味就能爬起来吃两口」的吃货天赋，也在治疗的惨叫声中揪心不已。它与 [暴鼠](暴鼠.md) 是酒友。

## 关系

- [阿尔托利亚（呆毛王）](阿尔托利亚.md)：它当作弟子教导、为之张罗治疗的裁决者。
- [苏晓](苏晓.md)（白夜）：它重金请来替呆毛王拔除黑暗物质的「神医」。
- [暴鼠](暴鼠.md)：一同关照呆毛王的裁决者酒友。
