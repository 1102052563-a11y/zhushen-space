# 人物

## 契约者
- [苏晓](苏晓.md) — 主角，猎杀者（衍生世界化名「白夜」、现实化名「苏笑」）
- [希珞珞](希珞珞.md) — 职工者，苏晓在乐园结识的第一人，亦同入东京喰种世界
- 寒鱼 — 冰系法师，东京喰种喰种阵营契约者（7 人临时冒险团首领），联合火爆对抗苏晓（二卷1、13章）
- 火爆 — 火系法师，10 人临时冒险团首领，与寒鱼结盟（花 14000 乐园币得火属性法杖）（二卷13章）
- [违规者 12470 号](违规者12470号.md) — 苏晓在东京喰种世界的猎杀目标（喰种阵营·三角眼，以诱饵聚杀喰种引爆牟利，二卷41章登场）
- [女王](女王.md) / [叶子](叶子.md) / [黑白](黑白.md) — 东京喰种**敌方契约者**·[彼岸花冒险团](../势力/index.md)（盾 / 奶妈 / 狙击），伏击苏晓反被全歼（二卷19–31章殁）

### 东京喰种世界 · CCG
