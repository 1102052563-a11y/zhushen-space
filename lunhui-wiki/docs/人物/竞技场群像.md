# 五阶乐园竞技场群像（古圣伊 · 沸砂 · 刺猬妹 等）

[苏晓](苏晓.md) 在 [五阶乐园竞技场](../总览/轮回乐园机制.md)（跨乐园 PVP）冲榜到首位途中遇到的高名次竞技者。竞技场不会真死（败者传回休息室），故均在世。

## 古圣伊 / 古圣一（原首位 · 「海神」· 宇智波 · 轮回乐园猎杀者）

苏晓登顶前的**五阶乐园竞技场原首位**——马尾辫、休闲装、叼吸管的随性女人，自称「宇智波·古圣伊」（竞技场名「古圣一」是拼音打错·她也是**轮回乐园的猎杀者**·与苏晓互认同类）。号称「**海神**」、几乎不可战胜，靠一套专为竞技定制的无解能力：

- **蔚蓝之海（Lv.67）**：把整个竞技场封成密闭海水（敌人每秒受伤 + 体力消耗剧增·脱离才解除）；用此能力后海水中藏一具**分身**，须找出并击杀才能破除——但攻击分身会被海水反噬、攻击者随机部位**金属化**（次数越多面积越大·随机到大脑即败）。
- **金属界 / 铁锈化**：召半金属化海洋生物（鲨 / 蟹 / 章鱼）；让敌人身体逐步**金属化生锈**（无判定·关节锈死失去知觉）。竞技场外这是近乎不可逆的重伤（须稀有恢复权限解）。

苏晓是**第一个在竞技场伤到她的人**——靠刃道刀·万鬼逼出分身、放逐刺穿分身破蔚蓝之海，忍受双腿乃至全身 60% 金属化，缠斗 36 分钟险胜登顶（她宁挨刀也不弃权·猎杀者本色）。

## 沸砂（天启乐园 · 第11名 · 近战弓）

天启乐园寄予厚望的近战型竞技者——两米长折叠弓（可变长棍·镶八宝石·重金内甲）。被天启乐园契约者 / 职工者包场助威。与苏晓近战对拼，被刃道刀·时定身后一刀穿胸 + 一脚踢「死」（心态极好·复盘录像还津津有味）。

## 刺猬妹（死亡乐园 · 灵魂系控制）

束发瘦小女孩、精通物理 / 精神 / 灵魂三系控制的稀有辅助系（曾打到第 12 名·战力极强）。两次撞苏晓——灵魂系外的控制对苏晓全无效，连最稀有的**灵魂冲击在苏晓的高灵魂强度前也毫无反应**（连她自己都怀疑能力失灵），直接弃权并把苏晓列入拒战名单。后用「实力差距多绝望」劝退后来挑战的死亡乐园新秀枫叶。

## 其他高名次竞技者

- **阴郁男（死亡乐园惩戒骑士 · 原第2名）**：脸带斩痕的全能独狼、多次挑战古圣伊未果（最后一次骂了娘）。与苏晓缠斗后弃权交出第 2 名（神情似得解脱）。
- **枫叶（死亡乐园 · 新秀）**：白面具红枫叶、双刺剑、野望首位——被刺猬妹用灵魂冲击连退劝退（「前十有多强是绝望」）。
- **第10 / 第9名等**：酒红发少年（缠斗 19 分钟败）、黑雾人 / 黑衣女（半小时败）等，均被苏晓累计奖励一路打过。

## 关系

- **[苏晓](苏晓.md)（库库林·白夜 / 灭法之影）**：从第 14 名一路连胜登顶五阶竞技场首位、得起源石；古圣伊与他互认「同类猎杀者」。
