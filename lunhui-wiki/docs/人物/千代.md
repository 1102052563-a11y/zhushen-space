# 千代（千代婆婆）

[火影忍者世界](../世界/任务世界/火影忍者.md) 砂忍村**长老**——地位仅次于历届风影，是 [蝎](蝎.md) 的祖母。对五代风影 [我爱罗](我爱罗.md) 颇为满意。

- **与蝎的渊源**：蝎是她的孙子。若非蝎幼年父母为砂忍村战死、深受刺激而叛逃（转去钻研永恒生命与有意识傀儡），凭蝎的实力 + 父母功绩 + 千代相助，风影之位本极可能是蝎的（我爱罗反而没机会）。
- **砂忍村遇袭时按兵不动（九卷34–38章）**：砂忍村高端武力仅她与我爱罗，而她无空战能力、又**感应到蝎就潜伏在村子附近**，于是全程戒备、时刻准备应对蝎现身，始终没有出手——这也是砂忍村被苏晓三人打得狼狈的原因之一。
