# 死灵贤者（中立 · 圣殿兑换商店）

[希亚圣殿](../世界/地点/index.md) 「震天殿」偏殿石壁里的一道**人形石像**——介于活物与死物之间（被石墙同化 / 吞噬·凸出墙面约十公分）。[震天使·鸣](震天使·鸣.md) 的无敌护盾能量柱被布布汪毁掉后，它从石墙里挣出、茫然苏醒，对苏晓僵硬地笑了笑。轮回乐园判定它为**中立单位、禁止攻击**，并开启「**圣殿兑换商店**」（奖励设施）。

## 圣殿兑换商店 + 安全点（十七卷63章）

激活死灵贤者后，**震天殿的「圣殿能量中枢已摧毁」、此区域转为安全点**（乐园生成坐标）——这关乎「**[原生世界归乐园](../总览/轮回乐园机制.md)**」机制（先驱者的优待）。商店以 [远古死灵精粹](../力量体系/消耗品材料.md) 兑换：

- 金色品质套装宝箱 ×1（精粹 ×6）；淡金套装宝箱 ×2（×3）；暗紫套装宝箱 ×5（×1）。
- **圣洁庇护**（×8）：可隔绝**黑雾**的侵蚀——希亚圣殿**第三层充斥黑雾**（伏笔·闯三层的必备）。

## 圣殿建造者 · 灭之石情报（十七卷64章）

它其实是**希亚圣殿的建造者之一**、因遭同族背叛才沦为石墙里的石像（已不知活了多久）。它语速极慢（说的字越多越慢、几乎递增），但很和蔼、心情好时还想给苏晓讲「希亚王朝」的故事。苏晓花半小时套出关键情报：

- **灭之石镶在 [战甲之灵](战甲之灵.md) 的铠甲上**（希亚圣殿一层最强者·镇守通二层入口·长刀使·空 / 震天使·鸣只是「看门的」、把它俩加千名士兵都不是战甲之灵对手）。
- **希亚圣殿共三层**：一层 = 希亚王朝将军 / 士兵；二层 = 王族直属精英部队；**三层 = 「生灵禁区」**，囚着**希亚大帝·生灵屠夫**、**囚徒·刑**等恐怖存在（详见 [地点](../世界/地点/index.md)）——它告诫苏晓**千万别深入二、三层**，与希亚大帝交手「连存活都难」。

## 关系
- **[苏晓](苏晓.md)（铁之手）**：在它处用远古死灵精粹兑换装备 / 圣洁庇护。
- **[震天使·鸣](震天使·鸣.md)**：偏殿能量柱供其无敌护盾·柱毁后死灵贤者苏醒。
