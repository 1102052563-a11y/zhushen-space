## 组织 / 设定

- **火颅帮**：塞壬星人类幸存者避难所势力之一——盘踞 17 号避难所（百万人口·几万超凡者·来者不拒）。脸带刺青（燃烧颅骨）、金属外骨骼、磁力悬浮战车的悍勇帮众；首领 [莫格罗](塞壬星/莫格罗.md)。
- **血焚症**：塞壬星人靠超凡力量 + 机械外骨骼战斗（「猎手」）——用能力的副作用是脏器被侵蚀的「血焚症」（几乎无解·六成超凡者死于此）。苏晓伪装圣焰药师、以炼金治血焚症为筹码搅动避难所。

## 起源之岛土著（不死者与狄巴族）

起源之岛（超高危感染源·狄巴姆本体所在）两类居民——[不死者](../世界/生物万族/index.md)（圣物仪式失败品·失理智）与 [狄巴族](../世界/生物万族/index.md)（完美感染保理智·岛真正主人）。

- **奇亚（不死者 · 热情）**：有理智的不死者小伙——见人激动下跪喊「哑巴巴」、带苏晓去见酋长。
- **阿乌达（不死者部落酋长）**：老不死者——不食生肉保理智但日渐虚弱、被苏晓气势压到交代狄巴姆在岛中心。
- **黑沼 / 血鸦（狄巴族）**：**黑沼**（力 120 / 体 130 · 长武器大师 · 共生之力狂回血 · 苏晓砍数刀才杀 · 得 3.16% 世界之源）；**血鸦**（被苏晓用恶魔之力控制的女狄巴族 · 靠「乌嘎」吼声引同族当刷世界之源诱饵）。

## 其他（一句话龙套）

- **白裙妹 / 眼镜（开局临时矿队 · 殁）**：苏晓陨石降临时秒杀灭口——眼镜（资源探测专家·测出苏晓「57000+ 生物反应·六阶大 boss」喊逃）、白裙妹（控制系·想签契约买命·被放逐穿喉）。代表天启契约者「怕死、想交钱买命」的群像。
- **邋遢男（第二名战斗天使 · 殁）**：没睡醒模样的战斗天使——库隆打不进后独自进 C 形走廊「接手」、几分钟后战死。
- **诺拉（违规者厨娘）**：1000 多度近视的呆萌前矿工——挖矿手滑敲死两名队友被判「威胁度最低违规者」贬为厨娘；苏晓见过最弱最怂的违规者。
- **断牙 / 阴郁少年（猎手兄弟）**：断牙隐瞒苏晓能治血焚（差点害死自己·自断一臂受罚）；其弟阴郁少年重度血焚——被苏晓利用 + 放走当散播治疗药消息的棋子。
- **血眼（13 号避难所暴徒 · 投靠）**：重血焚症暴徒——偷袭原首领红斧老爹、带 13 号避难所 80% 猎手投靠 9 号避难所抢药。
- **三名战斗天使（自闭组）**：临时三人队来清理苏晓——眼睁睁看目标从孤身滚成几万猎手的势力 boss，只能趴草里「自闭」观望（「三万打一个、四舍五入是一个亿」）。
- **无编号契约者 / 暗处刺客**：**无编号契约者**（时空系·水晶枪·被苏晓放水钓出手→轮回乐园强制处决 30% + 掠 157.9 盎司时空之力）；**暗处刺客**（返园途中漆黑长枪·唯一能与青钢影对噬的黑能量·疑爱丽丝 / 瑟菲莉娅雇来·在逃隐患）。
