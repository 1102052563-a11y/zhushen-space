# 塞壬星群像（游侠小队 · 战斗天使维克 · 火颅帮）

[塞壬星·入侵天启](../世界/任务世界/塞壬星-入侵天启.md) 中 [苏晓](苏晓.md) 接触的天启乐园契约者、战斗天使与塞壬星土著。

## 游侠小队（天启乐园·临时小队 · 全灭）

苏晓伪装「召唤师」混入的 6 人临时小队（天启乐园契约者**普遍组临时小队**·配置齐全·这点与轮回乐园多独狼 / 冒险团截然不同）。被苏晓剥得烙印后 **10 秒团灭**（打团先秒坦）。

- **独眼队长（坦 · 殁）**：古铜肤、独眼、胡茬、三十出头的可靠中年坦——小队代表、人情世故通透（实力强就多分好处·不恭维新人）。持菱形金属盾、有嘲讽 / 力量反击。被阿姆终极能力冻结、巴哈刺后颈注 2300 毒 + 950 冰秒杀。
- **佐萨（近战 · 殁）**：十七八岁、耳钉、抱刀、外冷内骚的少年近战——绝技「时风燕流」（风元素裹刀延长到五米全力下劈）。对苏晓敌视、信「危机时只有近战可靠」。被苏晓一刀斩杀；剥烙印的两个候选之一。
- **多萝（辅助 / 结界 · 殁）**：娇小、中性打扮、白衬衫、爱玩掌机的辅助系——小队敢闯废城全靠她的结界。**最敏锐**——纯靠「被剥离者的直觉」隐隐起疑（试探苏晓「为何总看时间」·种下不信任·连累队长警惕全员）；剥烙印另一候选。临死骂「血气怪物」、预言「会有天使来裁决你」。
- **火力手（中近火力 · 殁）**：四十多岁、超大口径霰弹枪 + 满身炸弹的中年人·存在感极低。剥烙印成功瞬间**全凭生死直觉抢先开枪**戳破苏晓伪装（被晶体层挡下）·随即被刃道刀·血刃斩成四块。
- **古拉塞（狙击 / 侦查 · 殁）**：丛林作战服 + 油彩、背两米狙击枪的沉默狙击手·很敏锐。被苏晓 D·暗杀 12 发爆头。
- **小奶妈（治疗 · 殁）**：弱气、淡绿白长袍、持法杖的奶妈·中立无敌意——被苏晓一道青钢影刀芒真伤秒杀（「圣盾」没来得及放）。

## 战斗天使维克（天启乐园·殁）

追杀「高危违规者」苏晓的**天启乐园战斗天使**（相当于轮回乐园猎杀者）——褐皮衣、寸发、醉酒登场。说辞极熟练（劝违规者「去挖矿十个世界就能重新当契约者」·半真半假）。液态金属臂铠、圣血（金瞳爆发）、捏卷轴定身。实力其实平平（非乐园竞技场前百名·战斗天使「不交手感知不出实力」），被苏晓侧踢踢爆侧腰、放逐 + 龙影闪穿空间斩喉秒杀（得 ★★★★★ 猩红卡·入侵度 +5%·杀战斗天使会引来更多天使）。

## 莫尼卡（天启乐园·光杆辅助 · 幌子）

红裙、波浪长发、被队友嫌弃的光杆辅助系——苦求组队（「你加入我们就有两个人了」）被苏晓「你太弱了」噎到。被苏晓利用：带路进契约者营地、敲开舰长室、目睹苏晓一刀斩舰长后被逼着出去喊「舰长被袭击」当幌子（全员恶人既视感·一脸「我到底做了什么」）。

## 火颅帮（塞壬星土著势力）

塞壬星人类幸存者的避难所势力之一——盘踞 **17 号避难所**（百万人口·几万超凡者·来者不拒只要不偷资源）。脸带刺青（燃烧颅骨）、穿金属外骨骼、开磁力悬浮越野战车的悍勇帮众（秃头男豪迈）巡地盘时遇苏晓、见其强欲拉拢入伙（这世界人类太少、抱团才能活·战斗有成本）。废城（洛丹城）即其地盘。

## 哀嚎女王（洛丹城·不死者小 boss）

废城里的精英 [不死者](../世界/生物万族/index.md)——灰白肤、血红眼、满头垂地白发（可金属化护体）、有智慧、压迫感强。本想为部下报仇杀游侠小队，一对上苏晓三随从就只想逃——被阿姆（刃道刀·火时定身 + 抡砸）、巴哈（一次性注 2300 毒）、布布汪（跳板配合）按住围殴致死（「史上最惨 boss」）。

## 白裙妹 / 眼镜（开局临时矿队 · 殁）

苏晓陨石降临时秒杀灭口的临时矿队成员——**眼镜**（不戴眼镜的资源探测专家·测出苏晓「57000+ 生物反应·六阶大 boss」喊逃）、**白裙妹**（控制系·中枪不手软却被疼昏过去·想签契约买命·苏晓放逐穿喉）。代表天启契约者「怕死、想交钱买命」的群像。

## 库隆 / 邋遢男（契约者营地·攻 C 形走廊的天启契约者）

- **库隆（天启坦 · 指挥）**：脸带斩痕的坦系契约者——指挥契约者营地 329 名天启契约者三波猛攻 C 形走廊（十坦盾墙 + 生命之链 / 近战 + 海量控制 / 集束爆炸物）想夺回舰长室，全被苏晓凭地形 + 侧踢「一脚秒坦」挡下（48 人永远留在长廊）；被苏晓资料（六阶战力 / 独赢世界争夺战）打到「终于懂轮回乐园为何那么强」。
- **邋遢男（契约者营地·第二名战斗天使 · 殁）**：自然卷、没睡醒模样的战斗天使——库隆三波打不进后他独自进 C 形走廊「接手」，几分钟后靠坐转角处战死（苏晓斩的第二名战斗天使）。
- **诺拉（违规者厨娘）**：1000 多度近视的呆萌前矿工——挖矿手滑敲死两名队友被判为「威胁度最低违规者」、贬到契约者营地当厨娘。被苏晓拉去找藏宝库（结果只有冻肉·「这些很值钱！」）·是苏晓见过最弱最怂的违规者（天启的违规者·与他无关）。

## 9 号避难所：亚哈达 / 铁甲熊·纳加 / 断牙（苏晓的炼金舞台）

塞壬星人类靠超凡力量 + 机械外骨骼战斗（「猎手」），用能力的副作用是脏器被侵蚀的「**血焚症**」（几乎无解·六成超凡者死于此）——苏晓伪装圣焰药师、以炼金治血焚症为筹码搅动 9 号避难所。

- **亚哈达（9 号避难所原首领 · 殁）**：900 斤的「肉山」、贪婪残暴的中等老阴哔——拿到治疗药后骚操作（免费发药 + 主动散播消息招猎手）兵不血刃瓦解反叛势力，却想白嫖配方扩张、不付一枚肖克。苏晓血之兽 + 长刀斩杀（财产 10 万肖克币归苏晓）。
- **铁甲熊·纳加（9 号避难所新首领 · 同盟）**：长相粗犷却心细如发的猎手队长——为还断牙救命之恩（他半盲那只眼是替断牙挡的）巧妙保下必死的断牙、送苏晓「磁爆猎手」史诗设计图当见面礼。被苏晓扶上首领之位（治疗药换满世界找狄巴姆·野心渐长）。
- **断牙 / 阴郁少年（猎手兄弟）**：断牙 = 干瘦却狠（生气连自己都打）的猎手，为让弟弟先得治疗、隐瞒苏晓能治血焚（差点害死自己·自断一臂受罚）；其弟阴郁少年重度血焚、死气沉沉。被苏晓利用 + 放走逃火颅帮（散播治疗药消息的棋子）。
- **血眼（13 号避难所暴徒 · 投靠）**：莫西干发型、重血焚症的暴徒——偷袭原首领红斧老爹、带 13 号避难所 80% 猎手投靠 9 号避难所抢治疗药。
- **三名战斗天使（自闭组）**：临时三人一队来清理「高危违规者」苏晓的天启战斗天使（淡雅妹子 + 冷峻皮衣男 + 壮汉）——眼睁睁看目标从孤身一人滚成几万猎手的势力 boss，敢出手就被打成筛子，只能在草里趴几天「自闭」观望（「三万打一个、四舍五入是一个亿」·很天启）。

## 火焰颅·莫格罗（火颅帮首领 · 同盟）

统领 17 号避难所 28 年、没人战胜过的火颅帮首领——半脸金属面具 + 大量导管、骑 200 吨「不死蜥」、重度血焚 + 不死者性质「以毒攻毒」才活到现在的狠人。摧毁过 12 处避难所、却是**为子嗣 / 部下甘愿赴死的守信枭雄**（带全帮来换药、做好了死在苏晓刀下的准备）。苏晓认其诚意、与之结盟——用治疗药配方 + 9 号避难所换**莫格罗带路去狄巴姆本体（被污染的世界之核）**（声望友善·苏晓评其人格魅力吊打衍生世界主角）。**🆕进化暴食君主·反目（二十九卷43章起）**：登起源之岛后莫格罗暴露另有所图（早想来岛却没下定决心）——**吞噬不死虫进化成「暴食君主」（裂变体Ⅵ型终极形态）**、靠拉怪冲散苏晓小队、屠戮天启契约者，正入侵起源岛中心的世界之核所在（下次见面敌友难料）。

## 三人组战斗天使（克罗 / 皮衣男 / 天使妹子）

临时三人一队追杀苏晓的天启战斗天使（实力五阶顶尖梯队·前 30）——**克罗**（壮汉·摔投绝技「力霆·撼地」·被苏晓借力反摔「开大招自己摔自己」后斩杀）、**皮衣男**（能 65% 同步复刻斩龙闪 / 刃道刀·青鬼·但无苏晓体质硬撑会肌肉断裂）、**天使妹子**（淡雅圣光系·「天启之光」光柱·把苏晓视作大魔王·遇见就喊「跑！」）。被苏晓**选择性放水**钓出暗处刺客后斩克罗；皮衣男 + 天使妹子从此见苏晓掉头就跑（砍出阴影）。

## 无编号契约者（暗处时空系刺客）

藏在战斗天使背后的真正隐患——持**水晶枪**、用时空之力的「无编号契约者」（封冻几百米 + 空间偷袭）。苏晓装作只对付三战斗天使、放水把它钓出手→轮回乐园**强制处决其 30% + 掠夺 157.9 盎司时空之力**（分苏晓 25 颗灵魂结晶完整）后被驱逐。

## 起源之岛：奇亚 / 阿乌达 / 黑沼 / 血鸦（不死者与狄巴族）

起源之岛（超高危感染源·狄巴姆本体所在）的两类居民——**[不死者](../世界/生物万族/index.md)**（圣物仪式失败品·失理智·被放逐岛外围）与 **[狄巴族](../世界/生物万族/index.md)**（完美感染保理智·岛真正主人）。

- **奇亚（不死者 · 热情）**：褐发、有理智的不死者小伙——见到人类激动下跪喊「哑巴巴（呀爸爸）」、急求帮助（带苏晓去见酋长）。
- **阿乌达（不死者部落酋长）**：满头枯白发的老不死者——**不食生肉以保理智但日渐虚弱**（连完成同化都做不到·感染度远低于岛内怪物）。想忽悠苏晓帮部落做事换信任，被苏晓气势压到老实交代狄巴姆在岛中心；自承「我这种不死者在岛上连小喽啰都算不上」。
- **黑沼 / 血鸦（狄巴族）**：**黑沼** = 力 120 / 体 130、长武器大师、完美感染的狄巴族优秀猎手（刀刀入肉 + 共生之力狂回血·苏晓砍数刀才杀·得 3.16% 世界之源）；**血鸦** = 被苏晓用恶魔之力控制的女狄巴族——靠「乌嘎」吼声引同族来当苏晓刷世界之源的诱饵（恶魔之力损命·最多活两三天）。
