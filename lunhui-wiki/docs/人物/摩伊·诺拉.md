# 摩伊·诺拉（现今最强女巫 · 死亡之地守护者）

[暗鸦星](../世界/任务世界/暗鸦星-暗鸦纪元.md) **摩伊家族的长女**——现存于世**最强的女巫**（已逝者不计·五阶段元素系·据说是元素系中杀伤力最强的能力之一·圣议员 [雪伊](雪伊.md) 称「三个 [摩利莎](摩利莎.md) 一起上也不是她对手」）。她是**死亡之地（刀魔封印装置所在）的唯一守护者**——也是 [苏晓](苏晓.md) 主线任务里「唯一知道死亡之地位置的女巫」，找到她基本等于找到死亡之地。她定居南部**迷雾堡**（摩伊家族大本营·建死火山半山腰）。

## 分散战略 · 女巫之国（二十一卷48章）

五年前次女 [摩利莎](摩利莎.md) 离开南部去西港、两姐妹似生分歧。摩伊·诺拉继承家族权势后，没把南部女巫整合成军、反而**让她们分散在南部各处**——半年后圣愈教会再难踏足南部一步，「女巫之国」（无国王 / 无经济体系的松散国家）就此成形：分散的女巫如丛林野兽般阻外人于境外、又各自发展壮大、还顺势侵占东部土著的资源。这份「分散」的远见让她坐稳南部统治。

## 不死之谜 ·「杀诺拉必先杀厄运」

她的「不死」并非自身能力，而是三妹 [厄运](厄运.md)（五阶咒术）赋予的——只要厄运处于「生」的状态，诺拉就不会死；**破坏厄运身体使其沉寂半小时，诺拉才会失去不死性**（圣议员 [雪伊](雪伊.md) 多年宿敌经验所证·她曾差几分钟杀掉诺拉、为此失去右臂）。故苏晓的计划是：自己攻迷雾堡杀诺拉、[卡门](卡门.md) 的夕阳红敢死队同步去尘泥祭坛破厄运的身体。

## 迷雾堡决战 · 殒命（二十一卷61–65章）

苏晓瞬爆阿波罗气化迷雾山谷、屠尽她的部下后，她从半截迷雾堡里现身——多元素（土 / 火 / 雷 / 风）+ 摩伊家族历代化身权杖，气度是天生领袖（不像妹妹摩利莎拿部下当棋子、她待部下如可托命的家人）。苏晓先用 [灵魂行者](../力量体系/装备道具.md) 给她留不愈伤、拖住她。同步 [卡门](卡门.md) 的夕阳红敢死队在尘泥祭坛破了 [厄运](厄运.md) 的身体——她**失去不死性**。决战中 [无冕领主](无冕领主.md) 再现、想夺她的身体（阿波罗的太阳焰 + 部下生命力 + 摩伊血脉凑齐了仪式），但**这次她主导融合成功**（传承无冕领主的剑术、样貌不变、初步掌控力量）。可惜苏晓不与她比剑术——界断线把两人缠成一团贴身缠斗、用 [灵魂行者](../力量体系/装备道具.md) 乱刺 + [无尽黑暗](../力量体系/装备道具.md) 6 秒 ×3 把她意识折磨崩溃、拷问出死亡之地位置后斩杀（临死仍不甘「什么都没做就要败了」）。
