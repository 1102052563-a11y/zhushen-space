# 卡洛

[亚森曼部落](../世界/生物万族/index.md) 的一名**小部落首领**、也是 [巴尔](巴尔.md) 深入虫巢的同行人（八卷36章）。身材高大（约 3 米·30 岁上下）、**表情呆滞、口齿不清**——是个**[血藤战士](../世界/生物万族/index.md)**：左侧脸颊干枯、里面生着植物根须，被炸碎后浇上一种淡红色液体即能**像植物般再生**（每次再生身高都会缩一些·一天有再生「次数」上限·透支「会死、会痛」）。**他会说一点「契（约者）/ 冒险者的语言」**（话漏了「契」字，印证巴尔早知苏晓身份）。

- **与巴尔的真实关系**：表面是「巴尔最信赖的部下」，实则**两人更像平等**（巴尔甚至对他有照顾之意）——交易卡壳时，正是**卡洛一句「没问题，通过爆炸区后罗盘归你」拍板**让 [苏晓](苏晓.md) 与巴尔成交，可见其地位不低。
- **存在感**：极弱（「像一颗盆栽·看一眼就不会再注意」），却在过爆炸区时**充当头阵踩雷探路**（呆滞地直走、踩中即被炸碎再生·一天限五次）。

## 真相 · 吞噬者（八卷46章·重大反转）
「卡洛」其实**早已不是巴尔的儿子**——他是**契约者 12963 号**，**吞噬了巴尔之子（真·卡洛）的意识**、鸠占鹊巢（「你的崽子早就被我吞噬」）。真·卡洛是巴尔与最爱的女人所生、最满意的继承人，故巴尔虽看穿、仍痛苦地给「他」最后一次机会（「我儿子的意识还在吗」）——得到否定回答后，巴尔暴怒与之**殊死搏斗**（母巢通道·两人皆血藤战士、热浪滚滚·卡洛放绝招「**阳之怒**」吐出鸡蛋大的灼热种子）。

- **疑点**：卡洛多次提及「**乐园**」——而乐园**禁止向剧情人物（土著巴尔按规则属剧情人物）透露乐园相关事宜**；卡洛却口无遮拦，说明他**已非普通契约者**（身份 / 状态有异·待解）。
- 苏晓在一旁看戏「加油（声援也是支援）」、**不插手**（两人如随时自爆的炸弹·且隐藏任务「太阳部落」还需巴尔引导·巴尔不能死·他判断巴尔会赢）。

## 结局 · 被围杀（八卷48章）
苏晓最终**举枪助巴尔**——卡洛求他「帮我杀掉他、给你好处」，自以为同为契约者能看穿苏晓；苏晓却**一枪打碎卡洛手臂**，冷道「如果你闭上那张破嘴，枪口或许不是对准你」。理由很现实：**就算帮卡洛取胜也毫无收益**——亚森曼部落不会承认他、乐园也不会再给他契约者身份，而他「半契约者半剧情人物」的身份**口风太松、泄露太多秘密**（多次提「乐园」），是个隐患。

- 在苏晓与巴尔围攻下，卡洛**仅撑三十秒**即被巴尔以高温「灼烧」焚头跪地，只剩躯干与半个焦黑头颅。
- 卡洛临终自嘲「我到底做了什么蠢事……**被移植过一次的原生之种太不稳定**、让我头脑不清」，又装腔作势想说出秘密（「生存试炼？」）——**苏晓一脚踩碎其头**（厌其废话、爱卖弄秘密「有些事自己知道就好」）。死后地面**未掉猩红卡或宝箱**（卡洛已非正常契约者）。
