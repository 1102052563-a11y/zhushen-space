# 勘九郎

[火影忍者世界](../世界/任务世界/火影忍者.md) 砂忍——四代风影之子、[我爱罗](我爱罗.md) 与 [手鞠](手鞠.md) 的哥哥，**傀儡师**（十指延伸查克拉线操控傀偶、远程支援）。

- **砂忍村遇袭时（九卷34–38章）**：[苏晓](苏晓.md) 三人攻入砂忍村，他率十几名心腹赶来，却因苏晓在沙土中埋设的**炼金炸弹**（触发式·踩中即炸）无法靠近支援姐姐 [手鞠](手鞠.md)，只能解傀偶远程出手。战后他主持救援、咬牙看着风影 [我爱罗](我爱罗.md) 被掳走，那夜「血色月亮下持刀的身影」烙进他脑中。

## 出处
