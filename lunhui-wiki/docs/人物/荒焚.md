# 荒焚

**[血门冒险团](../势力/index.md) 的副团长（二号人物）**——一名**火焰系契约者**（红发男子），所过之处地面被烧成黑炭、留下两行焦黑脚印，靠近他的东西都会被焚烧。火影世界（九卷）中随团长 [斯坦](../势力/index.md) 等来争夺「[猎魔套装](../力量体系/装备道具.md)」。

- **衍生世界联络平台招人（九卷13章）**：他在 [衍生世界联络平台](../总览/轮回乐园机制.md) 发招募帖——「任意属性超 50、已掌握核心战斗技能，奶妈 / 法师 / 主坦优先；须人品正直，无恋尸 / 辱骂同伴 / 调戏女团员等怪癖、无叛团劣迹」。
- **追杀咕噜（九卷13–14章）**：木叶村外森林，他奉团长之命搜捕抢走猎魔戒的 [咕噜](咕噜.md)——一路焚烧、把藏在焦土里的咕噜烧得断臂重伤（咕噜骂他「该死的火人、非要从我头上走」），却仍没找到她（她水化龟缩躲过）。

- **Fate/Zero·暗中清理契约者（十一卷30章）**：揭晓「为何苏晓一路只见英灵 / 御主、不见契约者」——荒焚（**三阶**·斯坦最信任的两人之一·入园前就过命交情·平时沉默寡言）奉斯坦之命带 **蝶**（紫发性感少妇·魅力特长·能轻松魅惑控制二阶契约者为奴）、小胖子等人潜在冬木市一间地下酒吧，**正悄悄猎杀本世界的所有二阶契约者**（已剩 5 人躲起·催化时钟剩 5 天）——目的就是**不让别的契约者与苏晓〔那「打英灵打得不亦乐乎的小子」〕联手**。「实力受压制下走正规手段、不算触禁忌」。

- **被贪功部下连累、强制回归（十一卷38–46章）**：[蝶](蝶.md) 贪图苏晓人头、撇下他带 [小胖子](小胖子.md) 单独行动——结果蝶被苏晓斩杀、小胖子被夹击濒死。荒焚骑岩浆巨兽全速赶来（距战场约 5 分钟），途中已猜到是蝶的野心作祟（「**这就是你的选择？**」·他懂人心、并非看上去那样唯命是从）；可小胖子撑不住、主动回归触发**全体强制回归**，把还在赶路的荒焚也一并带走、终究**没能与苏晓正面交手**。伏笔：荒焚渴望的东西很特殊——**特殊到一旦暴露、斯坦马上就会杀他**；他奉命行事、却并不在乎任务成败。

## 圣战士 · 巧合进入希亚大陆（十七卷7章）

他**巧合地进入了** [苏晓](苏晓.md) 所在的原生世界 [希亚大陆](../世界/任务世界/希亚大陆-灭法遗迹.md)——**并非来寻仇**（团内仅他一人有进入资格）。此处揭其底牌：他是**火焰系「圣战士」**，操**蓝白色火焰**（温度恐怖到「不能用恐怖形容」·火焰环一扫成片死灵化灰）；是**三阶竞技场第 2 名**（仅次第 1 的 [狂狐](../势力/index.md)·远在苏晓的第 11 名之上）。他以「士兵」身份在 [神之墙](../世界/地点/index.md) 战场刷战功、想尽快升官去王都。**苏晓与荒焚彼此都不知对方在场**——一旦相遇必是你死我活的厮杀（刀锋 vs 火焰）。

## 三阶竞技场之战 · 「心焰」揭底 + 败于苏晓（十七卷96–99章）

返园后，苏晓一路打到竞技场第二名、挑战荒焚——这是二人**首次正面交手**（场地被苏晓改成「死亡沼泽」、荒焚本想要烈阳殿堂·半公开模式）。揭其底牌：

- **心焰（特殊火焰）**：荒焚**没有法力值**，体内是一种**生命能量**助燃；火焰性质类似宇智波鼬的「天照」（泥土岩石皆助燃·唯水 / 煤油可压灭·并非真正的火），能凭意念具现火焰铠甲 / 火焰盾 / 火柱（「焚烧」近身大招需 10 秒蓄力）。火焰灼烧敌人会附加**七种递增的减益**、令生命如流水般下滑——「很多强者含恨而亡」。可在某处持续蓄火、蓄满能把整片场地「焚炼成渣」。
- **免疫物理攻击**：有约 1 分钟冷却的**物理免疫**能力（斩龙闪刺穿胸膛只是烟雾弹、青钢影能量却能让他剧痛）——他用它挡下了苏晓的「龙影闪」偷袭（荒焚此前不知龙影闪、被空间瞬移惊到）。
- **真实属性**：真实**力 / 体 / 智均达 90**（三阶顶尖、媲美部分四阶），唯技能等级因灵魂结晶稀缺而不高。
- **败局**：苏晓 [木之灵](../力量体系/装备道具.md) + [动能战靴](../力量体系/装备道具.md) 超负荷侧踢击穿其火焰盾、踢碎其 60% 脏器（三指成肉泥）；荒焚以「焚烧」火柱在苏晓胸口烧出碗口大洞反击。双方重伤僵持、走不到对方身前，最终拼枪——苏晓掏出 **[寂灭公爵 +13](../力量体系/装备道具.md)**（黑狙击炮）碾压荒焚的金色左轮，**一击定胜负**。苏晓由此打上竞技场第二名。曾在 Fate/Zero 不可匹敌的对手，如今被他正面击败。战中荒焚一反阴郁、酣畅大笑「你过来啊！」——苏晓发觉战斗中的荒焚并不阴郁（否则觉醒不出心焰）。
