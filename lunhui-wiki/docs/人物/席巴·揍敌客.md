# 席巴·揍敌客

[揍敌客家族](桀诺·揍敌客.md) 当主、[桀诺](桀诺·揍敌客.md) 之子、奇犽 / [伊路米](伊路米·揍敌客.md) 之父。性格**严肃、古板、传统**——一名很纯粹的杀手。他是**用毒专家 + 生物变异高手**，强壮身体经多次改造、连金色装备 [卞氏匕首](../力量体系/装备道具.md) 的剧毒都能无视。

## 接待苏晓 · 下毒试探（十六卷7–8章）

[苏晓](苏晓.md) 来揍敌客家族探路时，他在苏晓的食物里**下了毒**试探（拼「苏晓会不会大意吃下」的几率·苏晓自然一口没碰）——他冷冷道「之前你是来路不明的客人，现在你是雇主」，遂命人换食物。他答应过奇犽「不再干扰其决定」，故在桀诺要派伊路米去带回身处东果托国边境的奇犽时**出面拦下父亲**（妻子基裘为此情绪暴走、被他一句「闭嘴」按停）。
