# 长刀使·空（首领级 · 陵墓守门死灵）

[希亚圣殿（不死者陵墓）](../世界/地点/index.md) 里负责**守门**的死灵——身高约 **4 米**、兼具死灵与人类特征、黑色甲胄、金属头盔只露一双**燃着绿色鬼火的眼睛**，持一把 5 米长的加大型长柄刀（**骑兵**）。平时假死躺在石台上，入侵者近至 20 米才苏醒。它说的是一种**已失传的古老语言**（非死灵语·苏晓听不懂、傻白甜公主 [艾丽莎](艾丽莎·赫伯特.md) 反而懂、能翻译「入侵者」「召兵」等）。

## 资料 + 与里德的反差（十七卷55章）

阶位**首领级生物**·死灵·远古血脉；生命 100%·法力 1250·**力 85 / 敏 80 / 体 80 / 智 5 / 魅 1**。

- **领主权威（被动 Lv.30）**：生命 +8000 / 法力 +1200 / 力 +13 / 敏 +13 / 体 +10。
- **使令（被动 Lv.48）**：生命低于 50% 时，召「**兵营**」内所有士兵作战。
- **刀气（被动 Lv.30）**：每次斩击带强力刀气。
- **死灵马（主动 Lv.30）** + **骑旋斩（主动 Lv.42）**：召死灵马、乘骑时挥刀范围 10 米 0~800 伤 + 流血。

它本是**骑兵**（骑上死灵马杀伤力不低），但与霸主级的 [里德](陵墓守卫者·里德.md) 形成**巨大反差**——单论对苏晓，它只是个「皮糙肉厚的小 BOSS」（对 90% 三阶契约者仍很强、技能少但等级高）。

## 被苏晓吊打 + 召死灵军（十七卷55–57章）

苏晓不给它骑马 / 召唤的机会，一路**把它按在墙上踹 + 砍**：龙影闪近身、怪力压制（力 89>85）、**速度完全碾压**（真实敏比它高 8 点·四属性同步发展的红利）、青钢影真伤侵体。它**皮糙肉厚、有领主权威**故几刀难倒（生命掉到 46% 才触发「使令」），怒吼召来**陵墓驻军 200 余死灵士兵**（盾步兵 + 长枪兵 + 长弓兵的精英方阵）支援。它还被苏晓**踢飞的骑士 [戈斯](戈斯.md) 缠住**（详见下）。

## 殒落（十七卷59章）

终被苏晓斩杀，死后绿色「幽灵之焰」仍熊熊燃烧（开陵墓内侧关卡的必要条件）。掉落 **[远古死灵精粹](../力量体系/消耗品材料/index.md) ×3**（希亚圣殿的硬通货）——可惜因 [戈斯](戈斯.md) 参战导致**收益大幅削减**（正常该掉 5~6 颗）；且杀它**既无世界之源、也无宝箱**。

## 关系
- **[苏晓](苏晓.md)（铁之手）**：闯陵墓的入侵者（把它按墙吊打）。
- **[里德](陵墓守卫者·里德.md)**：同陵墓的死灵强者（霸主级·与它首领级云泥之别）。
- **[戈斯](戈斯.md)**：被苏晓一脚踢来与它缠斗的骑士。
- **[艾丽莎·赫伯特](艾丽莎·赫伯特.md)**：能听懂它古老语言的公主。
