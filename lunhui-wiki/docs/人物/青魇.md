# 青魇（怒川冒险团团长 · 四阶第一老阴哔）

[怒川冒险团](../势力/index.md) 的团长——平凡无奇的面孔、惯常脸盖热毛巾靠在皮椅上运筹。**四阶内若给「老阴哔」排名、他排第二就没人敢排第一**。他原本不是大型冒险团：旧团长是个莽夫、被青魇搞死后他上位，怒川随即以极快速度崛起、隐隐威胁到神皇与血门（风格与那两团截然不同·走谋略 / 阴谋路线）。

## 搞垮太阳王国权力体系（十九卷26–27章）

第二场 [世界争夺战](../世界/任务世界/海蓝星-世界争夺战.md) 中，青魇全程不露面、带着一名**女性魅力特长契约者**深入太阳王国境内——**几天内就搞垮了太阳王国的权力体系**：让掌军权的**大神使**（太阳王国军方最高统帅·无战力却统帅能力不可替代）**于清晨饮弹自尽**，太阳王国军方瞬间动荡、主和派抬头、前线一夜溃败（2 个机械师被全歼、格雅联盟 6 小时推进 4 公里）。圣域乐园方为之懵逼——「轮回乐园那帮莽夫疯子、才进战争世界几天就搞死了我们阵营的军事最高统帅？」

- **真正图谋 = 团结轮回乐园**：青魇深知轮回乐园方**战斗力最强、却败在内部不团结**（疯子各行其是：[所罗门](所罗门.md) 杀进中型国、[鬼夫人](../势力/index.md) 阉己方契约者、[咕噜](咕噜.md) 绑架圣域契约者勒索、老烟拉起 3000+ 起义军〔起义对象竟含怒川自己〕、斩首的夜……）。他想把这群人**团结起来**对外——看似大公无私、实则另有自己的目的。

## 关系
- **[怒川冒险团](../势力/index.md)（鬼夫人 / 人民教师等）**：他搞死旧团长后上位执掌的四阶大型冒险团。
- **[苏晓](苏晓.md) / [咕噜](咕噜.md) / [所罗门](所罗门.md) / 老烟**：同战区的轮回乐园方契约者（他正设法团结的「疯子们」）。
