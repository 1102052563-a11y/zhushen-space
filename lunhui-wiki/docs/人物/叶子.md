# 叶子

**叶子** 是 [东京喰种](../世界/任务世界/东京喰种.md) 世界中 [女王](女王.md) 三人女子冒险团的成员、敌方契约者（二卷19章）。

## 简介
相貌清纯、长发披肩的少女，身上带有一股草木清香（非香水、而是大自然的味道）；性格慵懒爱睡、爱拌嘴。队中担任**奶妈（治疗）**，自称「奶量」超大，曾在巨人世界救过黑白，被黑白笑作「飞机场」（二卷19章）。

- **共生伴侣「咔咔」**：一只淡黄色小仓鼠，是叶子珍贵的**共生伴侣**，附带三种被动能力，含 **Lv.3 欺诈**（可欺骗目标视觉、令人下意识忽略它），叶子能远程操控它并**共享视觉**，是三人最重要的情报来源（二卷19–20章）。
- 咔咔潜入大元拍卖场监视苏晓，因苏晓感知敏锐（智力 12 + 刀术 + 冥想法）被识破并一脚踩死；叶子经共享视觉目睹、当场流鼻血脱力、悲痛痛哭，欲立刻杀苏晓被女王喝止（二卷20章）。**复活咔咔需巨大代价**，叶子坚持要在本世界结束后复活它（二卷25章）。

## 技能（辅助 / 治疗）
- **治疗光球**：远程射出绿色光球为队友持续回血（曾把 [女王](女王.md) 从 19% 奶到 34% 仍持续攀升），治疗能力极强（二卷27章）。
- **属性光环**：环绕队友的增益光环，可**切换所增益的属性**（如把女王的「体力光环」临时改成「力量光环」）（二卷26章）。
- **荣誉徽章**（增益技）：令女王绿光大盛、肌肉隆起、力量提升（二卷25章）。
- **神圣**（高级治疗）：召出一绿一蓝两只元素鸟绕队友盘旋，持续治疗（回复速度超每秒 3%）兼减痛，曾把女王从濒死奶到全盛（二卷29章）。
- **审判**（斩杀类法术·三人底牌）：头顶降下巨大能量十字架，**目标 HP <15% 即触发斩杀**；对苏晓施放却被其 [绝魔体质](../力量体系/技能.md) 免疫、转为仅扣 9% HP（二卷30章）。
- **自然之怒**（绿色一次性技能卷轴）：释放绿色冲击波**群体催眠**，被命中者沉睡并散发绿光；智力 >10 可免疫（苏晓免疫、真户晓中招睡着）。卷轴用后碎裂、十分珍贵（二卷25章）。
- 近战与生存能力极弱、胆小，作战时藏于暗处（二卷26–28章）。

## 结局
女王被斩首、黑白战死后，法力耗尽的叶子毫无自保之力；纵苦苦哀求「把所有东西都给你」，仍被苏晓一刀斩杀（其复活咔咔的心愿终成空）。苏晓忌惮她的「审判」斩杀技或为装可怜的诱杀，故未勒索即下杀手（二卷31章殁）。
