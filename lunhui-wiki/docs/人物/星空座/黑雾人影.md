# 黑雾人影（空座宴主导人·本体即黑雾·与马文有交情）

由黑雾聚成人形、声音苍老的存在——**本体就是黑雾**（并非拿黑雾掩饰容貌）。他是 [空座宴](../../世界/势力/星空座.md) 的**主导人 / 筹划者**（不是星空座正式成员）：选定召开场地、按湮灭之雾的位置开「星空之环」的门、主持每次交易。座位编号 **0**。

- 与无良导师 **马文·华尔兹** 有交情，无良程度可想而知（这次姗姗来迟）。开场按湮灭之雾墙、开雾墙巨门放成员进大殿；主持时会回顾上次各成员的需求（圣女座要星辰铭印 / 白牛要命源 / 团长要世界之核）、并宣读星空座铁规（**成员间不可互抢 / 互杀 / 互相出卖·有黑枫树的秘密也不得外传**）。

## 关系

- [苏晓](../苏晓.md)：他主持的这场空座宴让苏晓正式成为星空座成员。
- 马文·华尔兹：有交情的老相识。

## 出处

- 三十六卷 60–63 章：开湮灭之雾门→主持空座宴交易。
