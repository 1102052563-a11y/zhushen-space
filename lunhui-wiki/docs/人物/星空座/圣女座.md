# 圣女座（星空座成员·星空之环6·虚空颜值担当·看热闹不嫌事大）

白裙拖地、淡金长发、肤白弹指可破的女人——[星空座](../../世界/势力/星空座.md) 六成员之一（**星空之环 6·戴左耳**）。战力属虚空顶尖梯队（弱刀魔一筹），却是星空座「各种意义上的下限 + 唯一的颜值担当」。

- **看热闹不嫌事大**：思维跳脱、正经不过三秒——一路「顺手」把施法者聚自然元素的位置告诉 [苏晓](../苏晓.md)、坐等看好戏（还讹走苏晓的全息地图），是虚空里声名赫赫却专爱起哄的「吉祥物」。**某势力的大小姐**（众多老不死的掌上明珠），与已死的女灭法 **格林·吉莉安** 相熟（被那疯女人「早晚把你关进笼子」的目光吓到）。
- **每 20 年沉睡重练小号**：每隔 20 年沉睡一次、忘掉「大号的密码」重新练起、模样退回 7~10 岁、心智也变——沉睡期间极限实力提升一截，如此循环。**空座宴交易**：想要「星辰铭印」多年，却被刀魔无意捡到、还被吸干了能量（当场破防）。

## 🆕坑人空间卡牌 · 三次没抢到黑枫树（42卷86–88章）

这次空座宴的召开地点由她提供**空间卡牌**——却是**她自己做的坑货**（空间锁定点飘忽·把苏晓、白牛、不死老人、刀魔都传到各种鬼地方:阴森的钢铁鬼列车、沙滩、黑袍大头怪的仪式现场……苏晓称之「奇妙之旅」）。被众人追问来源，她死不承认、无中生友编出个「形色猥琐、总奸笑、不讲卫生的小老头」——巴哈一听「这不就是凯撒吗」（凯撒无辜背锅）。作为星空座「吉祥物」，她连着三次空座宴一渣黑枫树都没抢到苏晓的（团长 / 白牛总能拿出苏晓要的东西压过她）、急得抓苏晓衣服撒娘，这次终于从刀魔那换到 11 公斤黑枫树、满载而归（家里的老古董会很高兴）。她还委托苏晓续调配「圣火洗礼」。

## 关系

- [苏晓](../苏晓.md)：一路蹭他看戏、又想跟他换黑枫树的「热心」同行。
- [刀魔](刀魔.md) / 灭法者：自称「很仰慕你们灭法者」（其实是灭法时代的老资历·怕暴露年龄）。
