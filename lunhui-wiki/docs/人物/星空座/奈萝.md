# 奈萝（白牛带来的混血人族少女·空之环继承候选）

[白牛](../白牛.md) 带来空座宴见世面的 **17 岁混血人族少女**——白牛一名已死老友的女儿（那老友为白牛而死）。她是顶尖天才、自幼在同龄人中出类拔萃，加上白牛手下的恭维、白牛凶残妹妹的宠爱，养成了目中无人的骄傲性格（除白牛谁都不怕）。白牛希望她哪天能继承自己的 [星空之环](../../力量体系/装备道具/index.md)，但以她现在的性子放出去闯荡迟早死于骄傲，故带她来空座宴——**让她被自己人先「教育」一次**。

## 经历（38卷62–64章）

- **被下马威**：白牛的三个「教材」= 满身血气的 [苏晓](../苏晓.md)、杀念漆黑的刀魔、气息如深渊的 [不死老人](不死老人.md)。奈萝忍不住感知不死老人，瞳孔放大、如坠浑黑污泥、险被吞噬（[团长](团长.md) 打响指、[白牛](../白牛.md) 授「不要随便感知他人、这世界不会善待你」的第一课）；也偷感知过苏晓、却测不到他的血气。
- **被 [圣女座](圣女座.md) 认作妹妹**掐脸叫「姐姐」（圣女座残留的淡金温暖气息让她放松，才忽略了其他成员并非都好相处）。全程遵白牛叮嘱「多看多听、没人搭话一个字都不许说」。

## 关系

- [白牛](../白牛.md)：她的引路人 / 监护者（老友之女·想传她星空之环）。
- [不死老人](不死老人.md)：给她上「危险第一课」的恐怖存在。
- [圣女座](圣女座.md)：认她作妹妹的星空座吉祥物。
- [苏晓](../苏晓.md)：白牛用来吓她的「教材」之一（测不到他的血气）。

## 出处

- 三十八卷 62–64 章：随白牛首次参加空座宴、被下马威受教育。
