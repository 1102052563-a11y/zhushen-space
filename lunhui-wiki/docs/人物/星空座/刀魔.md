# 刀魔（前灭法者·星空之环1·虚空顶尖·十几万幽魂畏惧）

破烂长衣、凌乱黑发、蓝到渗人的眸子——**前灭法者**、[星空座](../../世界/势力/星空座.md) 成员（**星空之环 1·挂领口**）、虚空顶尖梯队之一。出场时十几万幽魂因恐惧他争先冲出空间裂缝、天降雷电劈他（「除了黑渊你走到哪都被雷劈」）。

- **理智时清时浊**：清醒时可交流、但持续不久；肆意杀戮时不受控（[团长](团长.md) 负责在他失控时把他清出空座宴）。**至少两次想杀 [苏晓](../苏晓.md)**（缘由不明·同为「灭法者体内的影」一脉）——但星空座规矩下不能明抢明杀。有虚空已知最高的黑枫树（27.54 米·在黑渊生灵花海中心·无人能近）。
- **能吸干高纯度能量**：只要是高纯度能量他就能吸收壮大自身——空座宴上「偶尔捡到」的**星辰铭印**给了圣女座，能量却已被他吸干（圣女座破防）。这次黑枫树产出**被偷了**（疑苏晓所为·苏晓那棵正是从老灭法「起源与终焉之地」的黑枫树扯皮栽出）。

## 关系

- [苏晓](../苏晓.md)：同为「灭法者的影」一脉·至少两次想杀他。
- [黑枫树老者](../黑枫树老者.md)（老灭法）：吞过刀魔能量的狠人。

## 出处

- 三十六卷 61–63 章：空座宴登场→给圣女座星辰铭印→黑枫树产出被偷。
