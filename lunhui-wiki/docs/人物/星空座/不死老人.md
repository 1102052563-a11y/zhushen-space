# 不死老人（星空座第六成员·被永生诅咒的吞噬者）

身穿破烂长袍、戴兜帽、气息宛如深渊（黑暗、深邃、来自精神的沉重）的苍老存在——[星空座](../../世界/势力/星空座.md) 的第六位、也是最后一位成员（星空之环 3 号），三十八卷空座宴首次到场，星空座至此除刀魔外五人齐聚。声音空洞如两重声交叠，气息之恐怖足以让白牛带来的天才少女 [奈萝](奈萝.md) 一感知就险些被吞噬（[团长](团长.md) 打响指才救回）。

## 永生的诅咒

不死老人几乎**不死**——来自未知的永生诅咒：他失去了睡眠 / 休眠等一切相关能力，明明疲劳至极却无法安眠，明明饥肠辘辘、吃下的一切却在腹中化为灰烬又吐出。千年前他甚至找上一名灭法者、拿出一大笔酬劳求对方杀掉他让他永远安眠——那灭法者用尽各种方法都失败。从此他踏遍虚空寻找永眠之法，直到加入空座宴、[团长](团长.md) 告诉他有「**原生世界**」的存在：在原生世界里他的诅咒被大幅压制、竟昏睡了几年才醒，从此不再求永眠、转而要找出诅咒的源头。他的气息越来越「鲜活」（白牛调侃当初还以为他是亡灵族），因为他**吞掉了经常找他麻烦的古神**。

## 经历（38卷62–64章）

- **空座宴登场**：坐 3 号座椅，卖白牛一个人情（若不出手救 [奈萝](奈萝.md)、她几个月都挣不脱其气息）；空座宴上需求 = 「不死诅咒」的情报。
- **交易 / 预定药剂**：对 [苏晓](../苏晓.md) 的黑枫树产出很感兴趣却下手慢（被 [团长](团长.md) 400 灵魂结晶截胡）；向苏晓预定一种炼金药剂（首次调配免费）。
- **争夺虚空之树果实**：与 [团长](团长.md)、[白牛](白牛.md) 一同去夺虚空之树果实（恶魔族 / 羽族 / 星族一群老不死都在争）。

## 🆕去陨灭星 · 又吞古神（42卷80、86–88章）

暗星返园期间，转型发新闻的「死斗游戏公司」爆料**不死老人几天前去了陨灭星**（古神老巢·冥神所在·标题「陨灭星惊现星空座成员」）——这与暗星那尊 [羽神·赫格拉](../暗星/羽神·赫格拉.md) 缩进封印躲避冥神追杀的背景遥相呼应（羽神在封印里也认出了「去过陨灭星、吃过很多古神」的不死老人）。随后的空座宴上他到场，气息比往常更「扭曲」——问及又吞了什么，他只淡淡答「**古神**」。他向苏晓拿出一堆**残缺神灵骨**（要 10 甚至十几块才能合成神灵之奇迹）+ 几种猎奇材料，委托调配一份「药剂」（生命力夸张到饮者可能被撑爆·大补之物·苏晓判成功率最高仅 20%）。

## 关系
