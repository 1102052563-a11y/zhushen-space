# 不死老人（星空座第六成员·被永生诅咒的吞噬者）

身穿破烂长袍、戴兜帽、气息宛如深渊（黑暗、深邃、来自精神的沉重）的苍老存在——[星空座](../../世界/势力/星空座.md) 的第六位、也是最后一位成员（星空之环 3 号），三十八卷空座宴首次到场，星空座至此除刀魔外五人齐聚。声音空洞如两重声交叠，气息之恐怖足以让白牛带来的天才少女 [奈萝](奈萝.md) 一感知就险些被吞噬（[团长](团长.md) 打响指才救回）。

## 永生的诅咒

不死老人几乎**不死**——来自未知的永生诅咒：他失去了睡眠 / 休眠等一切相关能力，明明疲劳至极却无法安眠，明明饥肠辘辘、吃下的一切却在腹中化为灰烬又吐出。千年前他甚至找上一名灭法者、拿出一大笔酬劳求对方杀掉他让他永远安眠——那灭法者用尽各种方法都失败。从此他踏遍虚空寻找永眠之法，直到加入空座宴、[团长](团长.md) 告诉他有「**原生世界**」的存在：在原生世界里他的诅咒被大幅压制、竟昏睡了几年才醒，从此不再求永眠、转而要找出诅咒的源头。他的气息越来越「鲜活」（白牛调侃当初还以为他是亡灵族），因为他**吞掉了经常找他麻烦的古神**。

## 经历（38卷62–64章）

- **空座宴登场**：坐 3 号座椅，卖白牛一个人情（若不出手救 [奈萝](奈萝.md)、她几个月都挣不脱其气息）；空座宴上需求 = 「不死诅咒」的情报。
- **交易 / 预定药剂**：对 [苏晓](../苏晓.md) 的黑枫树产出很感兴趣却下手慢（被 [团长](团长.md) 400 灵魂结晶截胡）；向苏晓预定一种炼金药剂（首次调配免费）。
- **争夺虚空之树果实**：与 [团长](团长.md)、[白牛](../白牛.md) 一同去夺虚空之树果实（恶魔族 / 羽族 / 星族一群老不死都在争）。

## 🆕去陨灭星 · 又吞古神（42卷80、86–88章）

暗星返园期间，转型发新闻的「死斗游戏公司」爆料**不死老人几天前去了陨灭星**（古神老巢·冥神所在·标题「陨灭星惊现星空座成员」）——这与暗星那尊 [羽神·赫格拉](../暗星/羽神·赫格拉.md) 缩进封印躲避冥神追杀的背景遥相呼应（羽神在封印里也认出了「去过陨灭星、吃过很多古神」的不死老人）。随后的空座宴上他到场，气息比往常更「扭曲」——问及又吞了什么，他只淡淡答「**古神**」。他向苏晓拿出一堆**残缺神灵骨**（要 10 甚至十几块才能合成神灵之奇迹）+ 几种猎奇材料，委托调配一份「药剂」（生命力夸张到饮者可能被撑爆·大补之物·苏晓判成功率最高仅 20%）。

## 关系

## 👑 真身揭晓 —— 上一任深渊之主「旧王」（54卷242章）

⚠ **全书埋得最深的一条伏笔，在苏晓斩杀当代深渊之主的那一刻收线。**

苏晓被深渊高座的因果拉扯、即将成为下一任深渊之主时，一只苍老遍布皱纹的手从身后按上他的肩头——缠绕他的黑暗触须顷刻收拢回高座内。老迈的身影坐上深渊王座，只说了四个字:**「终于，回家了。」**

### 四条早就摆在明面上的线索

| 线索 | 对应 |
| --- | --- |
| **没有记忆** | 那场决战让他失去了全部记忆与大部分因果 |
| **不死诅咒** | 深渊之主的不朽特性残留 |
| **寻找模糊的昔日归处** | 他一直在找的「安眠之处」，就是深渊高座 |
| ⚠ **吞噬古神没有任何反噬** | 深渊之主的位阶 |

### 那场决战的代价

上一次深渊战役中，他与 [团长](团长.md) 决战，最终**平局**:

- **团长**失去了无上位阶，以及除 [泰莉儿](../泰莉儿.md) 之外的一切挚友兄弟（连活下来的灭法好友 [刀魔](刀魔.md) 都刀魔化、已然不再认得他）；
- **旧王**失去了无上位阶，以及**全部记忆和大部分因果**。

⚠ **苏晓曾获得过的【黑暗蠕动】，就是旧王的部分脑髓液**——而其中包裹的那块被侵蚀到辨认不出原貌的金属碎片，**正是团长当年破碎的金属面具碎片**。

### 星空座的真正目的

星空座原本可能只是个松散组织，其成员全部身死后被团长接手。而在上次深渊战役后，团长把它**很好地使用了起来**:

| 席位 | 成员 | 身份 |
| --- | --- | --- |
| **1 号** | [刀魔](刀魔.md) | 上次深渊决战的**开路者** |
| **2 号** | [团长](团长.md) | 上次深渊决战的**参与者** |
| **3 号** | **不死老人** | ⚠ 上次深渊决战的**参与者**——旧王本人 |
| 4~7 号 | [白牛](../白牛.md) / [圣女座](圣女座.md) 等 | 组织成立许久后机缘巧合加入 |

> ⚠ **星空座的真正目的，并非成员间的空座宴交易，而是确定刀魔是否安好，以及旧王的情况。**

⚠ **完美的天然伪装**:一众寻找旧王的深渊系强者，完全没去考虑这位不死老人，而是跑到各大封印之地或险境里找蛛丝马迹。结果是——**团长根本没封印旧王**，其实也不用封印:当初那场战斗**旧王的因果都被打碎了近半**。

### 他坐上高座之后

随着他重登深渊高座，苏晓身上的深渊宿命拉扯荡然无存，破碎的深渊巨殿逐渐愈合、恢复完整。显然恢复了大部分记忆的他，用平和的语气让苏晓回去——**你应该震慑那些原罪，而不是被束缚在深渊。**

⚠ 巨殿门扉彻底关闭前，苏晓与他对视。苏晓看到的**不是野心，也不是最后赢家的阴谋得逞**，而只是一个**疲累到极点、只希望能安眠一会的老人**。

### 他所开创的那个时代

深渊总督·[莫亚姆](../深渊总督·莫亚姆.md) 追随他多年，其回忆中的旧王是这样的:

- ⚠ **昔日星界的深渊时代，就是他亲手开创的**——把元素之灵们打退、夺得星界；
- 那是个**奇特的深渊时代**:深渊侵袭了星界大半区域，**却没有让星界原本的生灵体系灭亡**。在那远古时代，深渊高座之上的他真正成就了**万王之王**的无上荣光，**那也是深渊时代持续最长的时期**；
- 可深渊的过度蔓延总会引发元素力量的共鸣。这比以往周期长几倍的深渊时代最终也迎来落幕，深渊种族退回深渊；
- 岁月流逝，昔日的万王之王开始老迈。就在力量稍有减弱前，一场意外引发的深渊战役，让一名**已有伤在身的年轻人**来到深渊高座前——**双方无需多言，这是宿命必然的死战**。

⚠ 也正因如此，莫亚姆当年才没有提出「攻占星界」的主张——**上一代深渊之主并非固守现有的老顽固**。
