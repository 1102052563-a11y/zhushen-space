# 约瑟·柯（叛国圣战士）

[希亚大陆](../世界/任务世界/希亚大陆-灭法遗迹.md) 上一名**叛出 [圣索丁王国](../势力/index.md) 的有称号圣战士**——秃头、头顶纹蜥蜴、**绿瞳**、穿轻便棕皮甲、戴镶金属片的黑拳套（满身肌肉 + 拳套都是迷惑用·近战其实不强）。自称「约瑟·柯」「为钱什么都做的鬣狗佣兵」，受 [首席法官埃尔维·安东尼](../世界/任务世界/希亚大陆-灭法遗迹.md) 重金雇来守宅。他曾立赫赫战功、却因未知原因叛国；眸子深处藏着落寞——是被一名「落魄骑士」点醒「没圣之力也能变强」的人。

## 圣之力 · 与苏晓激战（十七卷12–13章）

他登场即假意「公平决斗」、暗中以圣之力偷袭 [苏晓](苏晓.md)（不可见的高速小物 + 凭空砸地的巨拳印）——被苏晓凭 [直感](../力量体系/技能/index.md)（7 米预警·非「看见」圣之力）躲过，反让他惊骇「没才能的凡人竟能感知圣之力」。他随即把上空那团圣之力散成**几十万根不可见的牛毛毒针**（附剧毒）罩住自己；苏晓以青钢影噬灭其针、反击盾防身。

!!! info "约瑟·柯资料（使徒之眼比对·智力 0.92 倍·得 80% 资料）"

    - **技能2·圣之力·毒（被动 Lv.45）**：圣之力附**剧毒 + 麻痹**。

    - **技能4·信念光环（被动 Lv.51）**：50 米内友军意志 +10——**因信念动摇已失效**。

    - **技能6·战阵冲锋（主动 Lv.50）**：冲锋时 200 米内友军移速 +60% / 力 +5 / 体 +5 / 意志 +20 / 短时忽略痛觉。
    - **技能7 / 技能8·胜利与剧毒（终极）**：???（资料未全）。

    结论：他是**战场型强者**——身旁士兵越多、战力翻几倍、悍不畏死；但**单挑能力不强**（圣之力开发偏辅助 / 群体）。

## 殒落 · 还人情债（十七卷14章）

他的**致命弱点是近战**——一旦被苏晓近身就难以招架。苏晓一记**侧踢**（[动能战靴](../力量体系/装备道具/index.md) + [木之灵](../力量体系/装备道具/index.md) + [近战肉搏专精 Lv.40](../力量体系/技能/index.md) 三重加成·看似平凡却暗藏杀机）将他重创，再以界断线斩首。临死遗言「**埃尔，我已经尽力**」——揭示他不是纯佣兵，是来**还首席法官埃尔维的人情债**（十几年前的旧情）。

> 【你杀死了圣战士·**胜利之毒**·约瑟·柯。】**安高里战区前任副军团长**——世界之源 **1.6%**（原生世界世界之源极难获得）·噬灵者 +30 法力（→ 4106）·掉落**传说级宝箱**（原生世界对宝箱品质有加成·约瑟近战弱本该掉金色顶级）。
