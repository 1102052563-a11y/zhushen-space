# 黑枫树老者（灭法之影 · 马文的师傅 · 吞刀魔者）

一名衣衫破烂、满头松散白发、看似邋遢却有一双**漆黑到让人胆寒的眸子**的老者——他是 [马文·华尔兹](马文·华尔兹.md) 的**师傅**，也是（据十九卷所述）**唯一现存的灭法者**。最生猛之处在于：他把**刀魔虚影囚禁在自己体内、并将其吞噬、化为自身力量**（旁人提及他时直呼「非常生猛的老家伙」）。他栖身在 [黑渊](../世界/地点/index.md) 最深处那棵 [黑枫树](../力量体系/消耗品材料/index.md) 旁。

## 黑渊一脚 · 黑枫树之缘（十八卷83–88章）

[苏晓](苏晓.md) 在黑渊最深处见到这老者时被他「**踹一脚**」，慌乱间顺手从黑枫树上扯下一块树皮带回乐园——竟靠它在专属房间栽培出 [黑枫树](../力量体系/消耗品材料/index.md) 树苗（这棵母树是虚空仅存三棵黑枫树之一）。老者得知苏晓培出树苗时神情一变、很「肉疼」（黑枫树被剥皮处的抓痕让他抚之不忍），却笑言「那是你的时运 / 命运之物、与我无关」——又自陈他的处世逻辑「**是我的早晚找上我、不是我的就先弄死珍宝的所有者、手动中断其命运再把宝物抢来**」（把「抢」说得正大光明：「能有抢的机会、就说明那东西与你有缘」）。

## 刀术抉择 · 赠晋升资格（十九卷92章）

苏晓的 [刀术大师](../力量体系/技能/index.md) 逼近 Lv.50 上限时，老者以黑枫树为媒介把他召入幻境点拨抉择，给三条路：① **突破大师极限继续升**（下限高·初期强）；② **晋升刀术阶位**（上限高·后期强·大多数灭法者都这样选）；③ **极端之路**（短时间变强几十倍上百倍、但意识可能在过程中消亡·「一条不归路·很多人验证过」）。考虑到苏晓正培育黑枫树，他推荐第 ②（相对正确的选择）。苏晓选定后，他抬手放出一道黑光没入苏晓体内（赠予晋升资格），并留下灭法者的寄语——「唯一现存的灭法者、你无需光复灭法者、也没什么使命……**吾等元素守卫者、不会被那些施法者窃贼所灭绝**……灭亡并非终结、死亡亦为无声」；被苏晓嫌「别文绉绉的、后半句听不懂」后改口大白话「**简单讲就是狠狠教训那些施法者**、你这兔崽子、我能帮你的不多、也不需要你报答」。
