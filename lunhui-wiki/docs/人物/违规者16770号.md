# 违规者 16770 号（黑雾女）

**违规者 16770 号** 是 [进击的巨人](../世界/任务世界/进击的巨人.md) 世界 [彼岸花冒险团](../势力/index.md) 收留的一名契约者，也是苏晓本世界三名猎杀目标之一（三卷1、33章）。

## 简介
扎马尾的女契约者，平日混在彼岸花团内。被自己称作「**异端**」——身体可化作一团**黑色烟雾**，凡触及黑雾的血肉瞬间干瘪枯萎、并持续蔓延，同时疯狂吸食目标生命值（苏晓接触黑雾时感到强烈虚弱、HP 如流水般下滑）；力量属性极高（一拳带起足以爆头的拳风）。完全黑雾化后看不出人形，只余一双恶魔般的**血红竖瞳**，对负担极大。她深知自己「异端」之身只会给同伴带来厄难，却珍视与彼岸花姐妹相处的时光（三卷33章）。

## 围杀苏晓与自我牺牲（三卷33–34章）
彼岸花以血色追印围杀苏晓时，她现身阻挡，黑雾虽能伤到苏晓、却在其速度 + 刀术下屡屡落空。她认出苏晓的**猎杀者**身份，想揭露却被轮回乐园**屏蔽了声音**（嘴在开合却发不出声）——可见乐园在刻意隐蔽某些秘密；她戏称苏晓为「**轮回乐园的'亲儿子'**」（苏晓嗤之以鼻，认为猎杀者风险远高于普通契约者，到了衍生世界更被一视同仁）。

自知不敌，她念出咒语「**John MacGinnis**」、消耗生命发动大招「**暗黑·挽歌**」与一座**随机大型传送阵**：先一击轰飞苏晓，再以直径五米、缠绕厉鬼虚影的黑色法阵，将濒死的团长 [冷月](冷月.md) 与彼岸花众女**随机传送**离开战场（以此报答冷月当初接纳她的恩情），自身则消散身亡。苏晓由此完成对她的清理——**猎杀任务 2/3**（系统提示「已清理 16770 号违规者」）（三卷33–34章）。

- [血玫瑰](血玫瑰.md) / 彼岸花众女——同伴姐妹，临终前满是眷恋。
