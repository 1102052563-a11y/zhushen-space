# 狼管家（公平的裁决者）

[恶魔古堡游戏](../世界/任务世界/恶魔古堡游戏.md) 的**裁决者兼主持人**——黑燕尾服、狼头人身，气息深邃如深渊（一层登场时连 [苏晓](苏晓.md) 都不敢招惹）。他自始至终主持五场古堡游戏、对规则「绝对公平」，是 [爱丽丝](爱丽丝.md) 的造物与忠仆。

## 真身 · 高等炼金生物（十四卷40–41章）

苏晓凭炼金学（副职业）看破：狼管家**不是生物、而是爱丽丝亲手造的高等炼金生物**——

- **矛盾的构造**：体表是**特等魔导生物组织**（罕见高级货）、骨骼亦为高等材料，却配了一个**极低劣的金属核心**（高强度金属·魔能传导与生物能量储蓄功率都很低、靠加入大量高等炼金阵图弥补）——「**像给高科技铠甲装了部手机锂电池**」。这核心给苏晓一种奇怪的「**笼子**」既视感（伏笔：核心实为壁画里那只铁笼·见下）。
- **已有灵魂**：狼管家拥有**等级不低于人类的灵魂**，会厌倦、会恐惧、乐在主持游戏其中——已远超提线木偶。但**体内有上万个禁制类炼金阵图**，想脱离爱丽丝掌控根本不可能。
- **炼金生物的「保质期」**：再高等的炼金生物都有存在期限（如苏晓造的炼金渡鸦只有半月）、且**任何炼金师都无法延长**。苏晓判断狼管家**时日无多**、且对爱丽丝忠诚到畸形，恐在报废前与他「鱼死网破」。

## 银·厄泊的替代品（十四卷42章·壁画揭）

宝库壁画揭出：狼管家其实是爱丽丝**以挚爱「[银·厄泊](爱丽丝.md)」（狼头人·用生命换她自由）为蓝本造的「矛盾替代品」**——核心由那只囚禁过她的**铁笼**打造、样貌却仿救她的狼头人，集「囚禁与拯救」于一身。他最恐惧的不是消亡，而是**爱丽丝造一个新狼管家、从而忘记他**（他对爱丽丝的畸形忠诚已近乎爱慕、只求能站在她身后）。苏晓还瞎猜「你的核心被重复使用过、你可能已是第一百个狼管家」竟疑似猜中（古堡老房客对他很熟·他有记忆起游戏就在进行）。

## 苏大忽悠攻心（十四卷40–42章）

苏晓积分第一要去宝库领奖，狼管家故意拖延阻挠。苏晓遂当面拆穿其炼金生物身份、临场胡编「**炼金生物可异化为生物（分解→凝聚→异化·不可逆）**」给他「脱离炼金生物身份」的虚假希望（赌爱丽丝不了解轮回乐园的神秘），又用「新狼管家会取代你、女王会忘记你」攻心——**既打消其同归于尽的念头、又逼他乖乖带路下宝库**。一颗「异化」的种子就此埋下（何时发芽取决于狼管家还剩多少时间）。

## 关系
- **[爱丽丝](爱丽丝.md)**：制造者 / 主人（畸形忠诚·绝不背叛·以银·厄泊为蓝本造他）。
- **[苏晓](苏晓.md)**：第四场游戏胜者（拆穿其身份 + 攻心忽悠·勉强带他领宝库奖励）。
- **恶魔奴仆嘎噜**：同为爱丽丝直属「工作人员」（清理违规者）。
