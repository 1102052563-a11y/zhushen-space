# A先生

[全职猎人世界](../世界/任务世界/全职猎人.md) 的契约者——三十岁左右、戴金丝眼镜、举止沉稳老练的**近战**。在 [衍生世界联络平台](../总览/轮回乐园机制.md) 上发布「[独眼巨怪](../世界/生物万族/index.md)（领主级）」的情报、在友克鑫市「老内克餐馆」**招募临时团队**击杀它（[苏晓](苏晓.md) 以「狙击手」身份混入旁观）。

- **「胸襟」人设**：他对外只要宝箱 10%（20% 给主坦）、还主动推法师 [老狼](老狼.md) 当名义首领、自任副队长——一副大公无私的样子，赢得满堂好感。
- **真实阴谋（十卷17–18章揭）**：这一切是他与 [老狼](老狼.md)、少妇奶妈、弓箭手「大头」（四人原是一个小队）布的局——**与老狼假装对立**以镇住其他刺头，**目的是借独眼巨怪坑死散人契约者、独吞宝箱**：计划在巨怪濒死时让奶妈**停止给主坦 [石头](石头.md) 治疗造成「倒坦」**，放巨怪屠尽没防备的散人，再由他这个「勉强算坦」的近战收尾。但 [布布汪](布布汪.md) 的「冰雪女神光环」让石头扛住一棍逃脱、计划全盘崩坏——巨怪狂暴反杀，老狼、大头当场被秒，小团队几乎团灭，他自己被巨怪踹断一臂侥幸没死，只得作罢逃命（「我们没死就算赚大了」）。
