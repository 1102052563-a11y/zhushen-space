# 幽鬼大陆配角（无眼小队 · 卢卡斯 · 埃迪爵士 · 简·维拉）

[幽鬼大陆](../世界/任务世界/幽鬼大陆-五阶考核.md) 中的觉醒者 / 鬼环 / 帝国配角，集中收录。

## 无眼小队（苏晓的觉醒者秘密小队）

苏晓在处刑机关挑的三名可信、懂审时度势的觉醒者（事成各 1 万金勒）：

- **格斯**：满脸伤疤、悍勇的「觉醒者中的平头哥」（觉醒度 60%·即将报废）——幽鬼当街袭人时徒手扯断五眼幽鬼一臂、被开膛仍裹住肠子来报信，苏晓由此提拔他。生死看淡、不服就干。**半妖魔化（二十四卷49–50章）**：被 J 先生扑克牌爆炸标记重创、化为半妖魔（满身黑鳞 + 长尾·却**保留自身意识**·一边变怪物一边抢答智力测试）。苏晓给他植入超高浓度柱间细胞封存、关铅锌合金笼，他选择继续在处刑机关服役（与汉克一队·体内被植了「能被秒杀」的保险）。乐观依旧。
- **汉克**：大小眼、木腿、地中海的秃顶老头（觉醒度 75% 却保持清醒·异类）——有「**吃幽鬼**」的癖好（「凭什么只能幽鬼食人」·站在食物链顶端的拷问专家·挚爱被幽鬼生吞后变此模样）。曾被 [邦妮](伊瑟薇·邦妮.md) 卖的「祖传生发剂」坑出一圈黄毛只好剃光。**战死（二十四卷66章）**：苏晓返园后，他与莱因哈特仍为「战死的军团长」复仇（无人会为军团长复仇、唯他俩会·因苏晓未当军团长前二人就因吃幽鬼 / 狰狞样貌受尽白眼）——在祖尔城堵截入城的 J 先生时被其扑克牌即死能力斩杀（残尸躺街）。
- **莱因哈特（高领神父）**：没有嘴唇 + 缺半个鼻头的觉醒者——原是流浪汉、自愿吃幽鬼血肉觉醒。沉默寡言、是苏晓最信任的一个（沉稳·雷属性能力·餐刀飞射穿头）。**雷雨天重创 J 先生（二十四卷66章）**：汉克战死后，他冒雨为旧主复仇——舔食遗物「骨灰」+ **雷雨天引天雷**（方圆几十米蓝白电弧·等同十四眼幽鬼战力·比罪恶还强一截）把 J 先生（炼狱）打到几乎致死（J 先生千算万算没料到帝国会有人为「战死」军团长复仇·自闭）。「士，为知己者死」。

## 卢卡斯（鬼环刺客 · 殁）

处刑机关旧部、苏晓 / 利·克拉塞曾出钱帮其置业的觉醒者（觉醒度 93%·人为压制）——被 [鬼环](../势力/index.md) 操控、夜里来刺杀苏晓（黑烟构成的衣物 + 弯刀）。青钢影克其能量武器、一个照面被压制。苏晓用无尽黑暗拷问出「鬼环」二字时他开始妖魔化、求苏晓杀他「不想变成那种东西」——被斩首。掉首张秘典卡「烟雾」。

## 埃迪爵士（约书亚·埃迪 · 先驱者混血 · 殁）

伊甸庄园（欢乐园）主人、祖尔城红矿（致幻剂）毒枭——手眼通天、有议员撑腰（君主之下无人能制）。实为**人类 / 幽鬼混血**（血脉稀薄·战力仅一眼幽鬼·**气息与人类完全相同**）、是 [鬼环](../势力/index.md) 的**资金提供者 / 成立者之一（先驱者）**。给苏晓摆下马威（枪手 + 猎犬迎客），被苏晓识破窝藏幽鬼、**两枪击杀**（顺手宰的「超级大鱼」·君主 [哈罗德](哈罗德.md) 一直想杀却不能亲自动的政敌·得 9400 万金勒甩锅给君主）。其次子约书亚·艾伯特（稀薄混血·非鬼环）想用家财换命被苏晓拒、抓走。

## 简·维拉（斯德哥尔摩少女 · 殁）

被前任副军团长利·克拉塞强占、又对加害者生出斯德哥尔摩情感的平民少女（王裔的未婚妻·聪明）——是利·克拉塞「职务犯罪」案的唯一活口。苏晓接手身份后第一件事就是**炸死她灭口 + 切断所有线索**洗脱嫌疑（连同来审她的黑衣调查者一并炸死）。

## 月蚀（十二眼雌幽鬼 · 殁）

[鬼环](../势力/index.md) 的**眼室计划提出者**、十二眼强大幽鬼——苏晓到伊甸庄园寻找的目标（名册记载有误·她不止是提出者还是强幽鬼）。暗恋首领 [罪恶](罪恶.md)（混血·更喜人类女性·觉她「随心所欲」长相）。不朽城堡被苏晓阿波罗炸平时，她用青蓝粘液**牺牲自己保住眼室「核」、把罪恶从地洞送走逃命**（临死面甲裂开笑「抱歉啊」）·被太阳焰净化。

## 马库斯（前精神病人 · 能感知眼室）

帝国疯人院的精神病人——眼室转移到疯人院后，病人们竟被眼室「**治好**」（个别还展露特殊才能、击退来护眼室的幽鬼），院长与医护却疯了 / 想吃生肉（被病人善意控制）。马库斯绅士有礼、比正常人更清醒，**能模糊感知眼室的存在**（靠近会头疼）——成了苏晓找眼室的活雷达。

## 琼斯 / 利奥波特议员（殁）

祖尔帝国 12 议员中的两位领袖——琼斯（保守派·爱哭的胖议员·帝国财政危机时硬撑）与利奥波特（激进派·阴沉老人）。两派对立实为不团结向君主示好。与君主 [哈罗德](哈罗德.md) 形成「黄金三角」。被女仆幽鬼（细长舌）+ 「**不朽印纹**」操控刺杀——二人都在被操控的清醒瞬间**自爆右眼明志**（「有我和哈罗德在、你们休想」）·绝不背叛帝国而死。苏晓用「死亡回溯」临时技能查明死因（指向利奥波特也遇害）。

## 修·洛斯南勒（哈罗德祖先 · 题字者）

君主 [哈罗德](哈罗德.md) 的祖先——处刑机关地下室合金门上题字「**我们行走在黑暗中，为了那些渴望安宁的人们**」的「行走在黑暗中的觉醒者」。其陵墓在宁静墓园（被 [罪恶](罪恶.md) 刨开藏身）；**国足三兄弟的晋升任务第二环「黑暗与黎明」正是找他的遗骨**（苏晓提供线索换情报、引出国足盗墓闹剧）。

## 法迪尔（十六眼幽鬼 · 罪恶的恩人）

被囚于处刑机关地下室十几年的十六眼幽鬼——曾强大、如今骨瘦如柴仅剩一臂、生命力被黑暗监牢透支。是 [罪恶](罪恶.md)（本名小波伊尔尼）**童年时如父般的存在**（待他如子）。罪恶的眼室 / 不朽之王全套阴谋，真目的只是潜入地下室**救他**（伪装成苏晓·用空间卷轴带走）。被救后藏于东部森林木屋养伤，给罪恶一颗**三金瞳黑眼球**（脱眼窝后变这模样·与苏晓在眼室得的「未知眼球」同款·嘱罪恶找另一颗）。

## 凯斯·利科克（十五眼幽鬼 · 黑暗监狱囚徒）

处刑机关地下室「黑暗监狱」关押的十五眼幽鬼——被囚 36 年 11 个月（间歇精神失常 + 多重人格 + 妄想 + 自杀倾向 + 部分语言能力缺失·靠吃狱友与自己撑着）。罪恶来救的是法迪尔、不是它。苏晓 D·暗杀对准它时，它想**用一个秘密换一个痛快（死）**——对长囚者而言死是解脱。（这间黑暗监狱专关罪大恶极的幽鬼让其缓慢死去·风险极高·提出者「脑子不正常」。）

## 海鲨 · 克里斯（争时空之核的独狼 · 二十四卷59–62章）

挽歌把时空之核当诱饵召来的独狼中两人：**海鲨**（黄寸发 + 鲨齿·精神不正常·武力派·虎鲨召唤物「亲儿子」可地下潜行 / 把地面变海洋·30 万乐园币养的）+ **克里斯（赵铁柱）**（容貌绝美的「变身大佬」·感知系独狼〔比大熊猫还稀有〕·窒息〔排空敌体内氧〕/ 原·世界之壁〔曾挡八尾尾兽炮·被斩龙闪破〕/ 世界同步结界）。二人合作携时空之核出城——被苏晓循提示 + 布布汪 / 阿姆 / 贝妮三路跟踪揪出（阿姆假败诱敌 + 布布汪狼族传家宝坐标 + 苏晓「打地鼠」逼出地下潜行的海鲨）。**海鲨被苏晓斩杀**（尸体被罪恶的枪焰漩涡绞碎）；克里斯捡走时空之核观望、苏晓斩罪恶后向她索要，她**交核逃走**（临走没敢打苏晓、却气得锤布布汪一拳泄愤）。
