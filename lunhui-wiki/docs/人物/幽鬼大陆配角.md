# 幽鬼大陆配角（无眼小队 · 卢卡斯 · 埃迪爵士 · 简·维拉）

[幽鬼大陆](../世界/任务世界/幽鬼大陆-五阶考核.md) 中的觉醒者 / 鬼环 / 帝国配角，集中收录。

## 无眼小队（苏晓的觉醒者秘密小队）

苏晓在处刑机关挑的三名可信、懂审时度势的觉醒者（事成各 1 万金勒）：

- **格斯**：满脸伤疤、悍勇的「觉醒者中的平头哥」（觉醒度 60%·即将报废）——幽鬼当街袭人时徒手扯断五眼幽鬼一臂、被开膛仍裹住肠子来报信，苏晓由此提拔他。生死看淡、不服就干。
- **汉克**：大小眼、木腿、地中海的秃顶老头（觉醒度 75% 却保持清醒·异类）——有「**吃幽鬼**」的癖好（「凭什么只能幽鬼食人」·站在食物链顶端的拷问专家·挚爱被幽鬼生吞后变此模样）。曾被 [邦妮](伊瑟薇·邦妮.md) 卖的「祖传生发剂」坑出一圈黄毛只好剃光。
- **莱因哈特（高领神父）**：没有嘴唇 + 缺半个鼻头的觉醒者——原是流浪汉、自愿吃幽鬼血肉觉醒。沉默寡言、是苏晓最信任的一个（沉稳·雷属性能力·餐刀飞射穿头）。

## 卢卡斯（鬼环刺客 · 殁）

处刑机关旧部、苏晓 / 利·克拉塞曾出钱帮其置业的觉醒者（觉醒度 93%·人为压制）——被 [鬼环](../势力/index.md) 操控、夜里来刺杀苏晓（黑烟构成的衣物 + 弯刀）。青钢影克其能量武器、一个照面被压制。苏晓用无尽黑暗拷问出「鬼环」二字时他开始妖魔化、求苏晓杀他「不想变成那种东西」——被斩首。掉首张秘典卡「烟雾」。

## 埃迪爵士（约书亚·埃迪 · 先驱者混血 · 殁）

伊甸庄园（欢乐园）主人、祖尔城红矿（致幻剂）毒枭——手眼通天、有议员撑腰（君主之下无人能制）。实为**人类 / 幽鬼混血**（血脉稀薄·战力仅一眼幽鬼·**气息与人类完全相同**）、是 [鬼环](../势力/index.md) 的**资金提供者 / 成立者之一（先驱者）**。给苏晓摆下马威（枪手 + 猎犬迎客），被苏晓识破窝藏幽鬼、**两枪击杀**（顺手宰的「超级大鱼」·君主 [哈罗德](哈罗德.md) 一直想杀却不能亲自动的政敌·得 9400 万金勒甩锅给君主）。其次子约书亚·艾伯特（稀薄混血·非鬼环）想用家财换命被苏晓拒、抓走。

## 简·维拉（斯德哥尔摩少女 · 殁）

被前任副军团长利·克拉塞强占、又对加害者生出斯德哥尔摩情感的平民少女（王裔的未婚妻·聪明）——是利·克拉塞「职务犯罪」案的唯一活口。苏晓接手身份后第一件事就是**炸死她灭口 + 切断所有线索**洗脱嫌疑（连同来审她的黑衣调查者一并炸死）。
