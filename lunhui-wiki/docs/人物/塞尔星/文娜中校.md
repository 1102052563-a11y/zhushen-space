# 文娜中校

[塞尔星](../../世界/任务世界/塞尔星-不败要塞.md)·**眷族同盟**的中校——名将 [雷兹准将](雷兹准将.md) 的女儿，26 岁、有点近视、气质独特，用一把「枪剑合体」的骑士刺剑（拼刀时突然扣扳机打穿透弹）。她并非弱女子，最特殊的是一手**极精准的预知能力**。

## 恐怖的预知

钢铁要塞失守后，她率 5 万守军困守中心城区。[苏晓](../苏晓.md) 故意「露出破绽」引她袭杀——她假投降后瞬移到苏晓身侧、以「蜂蜇·穿骨」刺其脖颈，却在得手瞬间化为血雾般的碎肉:那是她**预知到的死亡画面**。她能预知「分支线」（看到无数种可能、选一种正确行动），准得让苏晓（见惯预言者）都首次见识——正因预知到「主动出击必死」，她才没动手、行动轨迹一变，苏晓反倒有种「记忆遗失感」。苏晓遂放逐钉她耳旁逼降。

## 命运金币的素材

苏晓看中她的预知，想从她身上「夺取部分预知性」——**把一枚金币移植进她心脏**，靠她的能力把金币淬炼成能探吉凶的「[命运金币](../../力量体系/装备道具.md)」（抛正面=顺、反面=险）。为此她躺在金属树台上，预知能力失控、不断预知自己的各种死法（被重锤砸死、斩首、万米坠地……分不清现实与预知），苦熬到午夜金币成型。苏晓给她喝「信仰之力药剂」坚定意志、事后放她归队——她嘴硬「不会背叛同盟」，却被苏晓对俘虏的果断与「友善」吓得发慌。她父亲 [雷兹准将](雷兹准将.md) 后来变成失忆的兽巨人、战死；赫·康狄威曾想拿筹码从苏晓那换回她。

## 关系

- [雷兹准将](雷兹准将.md)：她的父亲、眷族名将（后化兽巨人失忆战死）。
- [苏晓](../苏晓.md)（太阳领主）：引她袭杀、把她做成命运金币素材、又放她归队的猎杀者。
