# 神骸·加骨

[塞尔星](../../世界/任务世界/塞尔星-不败要塞.md) 世界争夺战里**守望乐园的守护者**（相当于轮回乐园猎杀者）——身高约 1.9 米、赤膊却有肋骨构成的贴身骨甲、身后拖一条 3 米多长的脊椎状**骨尾**（尖端剑刃、异常灵活）、下半张脸罩着骨面具。战力属八阶顶尖梯队，但比 [黄金伯爵](黄金伯爵.md) / [圣诗](圣诗.md) / [奥兰迪](奥兰迪.md) 等领袖弱一线。

## 骨系操控 · 扭曲的心理

他的能力全围绕骨骼:骨尾横扫切碎、地面生数米骨刺再爆成骨刀片、聚碎骨成十几米高的巨型骷髅（能张口喷黑色光柱）、大圆骨盾格挡、还有一手阴毒的「让敌人骨缝隙锈死、关节僵死」的地面波动。他心理扭曲——**越是美好、可爱的东西，亲手捏个稀巴烂就越有快意**，为此他团长 [奥兰迪](奥兰迪.md) 屡屡想用「哲♂学的力量」帮他矫正、却毫无效果。他追杀 [月使徒](../苍龙大陆/月使徒.md) 既因世界争夺战、更因「敌人长得这么可爱、弄死之前一定特别有趣」——一路把月使徒的召唤物切碎、斩落她的黑骑士（从五阶跟到八阶的老召唤物）。

## 被一脚踹碎

正当他准备取月使徒性命，[苏晓](../苏晓.md) 赶到——一根血枪破他骨盾，一脚直踹（近战宗师 Lv.60 + 誓约之徽·白龙 + 战靴 + 一堆被动的全部力量灌进这一脚）当场轰碎他的腹部骨甲、掀翻骨面具、把他踹得胸腔半空、飞出一道模糊虚线。他吐着脏器碎片惊问「这一脚是人能踹出来的?」，仍要挣扎（三属性发展 + 双宗师的技法型太难缠、近身即死）。最终苏晓在 [仙露露](../苍龙大陆/月使徒.md) 加成的极速下祭出「**刃道刀·血影**」，一道血线斩过他脖颈——世界褪成黑白，他的生命之火熄灭。苏晓得**八星猩红卡** + 5 点天启战功。

## 关系

- [奥兰迪](奥兰迪.md)：他的团长（守望方领袖）·屡次想用哲学矫正他扭曲心理未果。
- [月使徒](../苍龙大陆/月使徒.md)：他因「可爱想捏碎」而追杀的猎物——反被苏晓救下。
- [苏晓](../苏晓.md)（白夜）：一脚踹碎他骨甲、「刃道刀·血影」斩他首级的技法型强敌。
