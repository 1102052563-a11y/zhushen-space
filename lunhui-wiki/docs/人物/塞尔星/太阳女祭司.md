# 太阳女祭司

[塞尔星](../../世界/任务世界/塞尔星-不败要塞.md)·太阳要塞里一名从**雌性猪头人蜕变而来**的祭司——[苏晓](../苏晓.md) 权力金字塔的三巨头之一（与 [豪斯曼](豪斯曼.md)、厨师长并列）。她原是身材瘦弱、以野猪人审美「一堆光棍里也难找配偶」的劣等雌性猪头人，却不甘心、向豪斯曼申请进「圣巢」（[进化巢](../../世界/生物万族/index.md) 的敬称）——因她体内人类基因比战猪基因多几倍，蜕变后与人类相似度达 98%（只是耳朵偏尖、脸上有很细的金色纹路），身穿白色长袍、成了太阳女祭司。

## 太阳要塞的财神爷

她战斗力拉胯，但当祭司不需要战力。苏晓让她**主管伤员安置、地下矿脉开采、活性矿石储备**——是全阵营的「财神爷」（苏晓的专属会计）。苏晓重用她有讲究:她**善良**，与铁血的豪斯曼不亲不敌、与女汉子厨师长矛盾最大（厨师长曾当面骂她「一定是献了皮肉才搭上领主」）——正合苏晓「慈不掌兵、三巨头绝不能亲如一家（否则能联手哗变）」的分权之道。她在 [图弗](../../世界/生物万族/index.md) 的葬礼上主持默哀，正是她按上图弗的尸骨、从其眼窝里托出了那枚 [太阳之环](../../力量体系/装备道具/index.md)。她明确表态「信仰可以有、神权不发展」（单手按胸:「比起死在眷族领土，我更愿被埋葬在这」）——正是苏晓要的态度。

## 关系

- [苏晓](../苏晓.md)（领主大人）：重用她当财神爷、看中她善良且与厨师长不和的主公。
- [豪斯曼](豪斯曼.md)：金字塔并列的统帅（主战斗）·与她不亲不敌。
- 厨师长（摩提）：与她矛盾最大的女汉子后勤主管（骂过她「献皮肉」）——正是苏晓乐见的制衡。
- [图弗](../../世界/生物万族/index.md)：她在其葬礼上托出太阳之环的战死大战士。

## 🆕 名奥克塔薇 · 「死人的话少去听」（45卷68章）

首战后战场上，苏晓才唤出她的名字——**奥克塔薇**。她跪在尸堆间悲戚地说「我能听到死者在怨嚎、它们在怨恨领主大人」。苏晓半是敲打半是点化:「帮我转告它们，让它们尽快来找我，我最近正缺一批实验素材」；又冷冷提醒她「太阳侍女有两种用法」——不愿上战场的可用「骨什针」刺穿第五脊骨、装进竖笼当治疗柱（中枢神经瘫痪的代价），问她愿自己走上战场还是被抬上去。她当即表态愿自己走、「死者的声音」也不再听了。苏晓一句「死人的话少去听，听太多，或许你的死期也到了」，彻底定住了她的位置(信仰可有、神权不许)。（她瘦弱善良，正被最爱嚼舌根的野猪五兄弟盯上要「焚了她」，苏晓一个眼神吓得五兄弟赶紧堆笑。）
