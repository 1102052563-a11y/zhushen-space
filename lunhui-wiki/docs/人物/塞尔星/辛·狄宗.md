# 辛·狄宗

[塞尔星](../../世界/任务世界/塞尔星-不败要塞.md) 上猎手 / 杀手家族**「辛之一族」（「辛」之一族）**的族长——人称「**辛鬼**」。他单手拄拐、头发花白、脸上遍布陶瓷般会裂开的纹路（战斗时裂痕密集、露出「辛鬼」真容），十指指尖尽是黑曜石般的**黑色指甲**（辛之一族的血脉特征·非染）。**抛开势力规模，他是本世界公认最强的三人之一**。

## 辛之一族

辛之一族是个战力强悍的杀手家族——族人最鼎盛时也只有几十人，却大多是强者，历代族长好几位实力能排各时代前五。他们的家规极端:**只要雇主出钱，暗杀自己的族人子嗣都接**（但基本都是雇主先死、委托失效）。族人共有黑指甲特征。

## 与苏晓的博弈

[苏晓](../苏晓.md) 通过奴隶商人 [阿兹巴](阿兹巴.md) 联络辛之一族，「雇」他们暗杀一个人——正是狄宗的**嫡子 [辛·尤戈](../塞尔星/多萝西.md)**（40 多岁·三代 [吞噬者](../../力量体系/装备道具.md)「暗阳」的宿主·狄宗满意的子嗣）。辛之一族当即找上门。狄宗亲自现身黑巷、开口「暗杀嫡子酬金加七成」——苏晓却抛出关键情报:二代吞噬者「沸红」的宿主，是 [利·西尼威](利·西尼威.md) 的女儿 [多萝西](多萝西.md)（黑指甲、疑似辛之一族血脉·而杀多萝西母亲的正是辛之一族成员 [辛·阿丽丝](辛·阿丽丝.md)）。狄宗闻言现出「辛鬼」真容、气息与苏晓血气对撞、震裂整条街——最终却道「被你这小子算计了、我会保持观望」，收手离去、约苏晓日后去辛之一族喝茶。这场家族伦理悬疑剧就此埋下:苏晓故意把两代吞噬者分送多萝西与辛·尤戈，让辛之一族自己人内部（狄宗 / 尤戈 / 多萝西 / 阿丽丝）去博弈，他坐收吞噬者的战斗成长数据。

## 关系

- [辛·尤戈](../塞尔星/多萝西.md)：他满意的嫡子、三代吞噬者暗阳的宿主。
- [辛·阿丽丝](辛·阿丽丝.md)：他的第六个女儿、杀多萝西之母的凶手、利·西尼威的老情人。
- [多萝西](多萝西.md)：二代吞噬者沸红的宿主、黑指甲、疑似辛之一族血脉——被苏晓推去找辛之一族复仇。
- [苏晓](../苏晓.md)（白夜）：用「雇他杀嫡子」把辛之一族拖进博弈、气息对撞后相约观望的算计者。
