# 豪妹

天启乐园**封天公会**的白发女剑豪——本名不详（因坐拥一个矿石商会的净收益分成、富得流油，被戏称「豪妹」），塞尔星世界争夺战里天启方一员。天生浅白微灰的长发烫成大波浪，后腰斜挂一把比刺剑略宽、斩 / 刺 / 穿俱佳的**菱刺剑**，储存空间里最常备的是酒，是个没心没肺、爱喝酒感慨「人生啊～」的女剑豪。她在世界联络平台上以「豪妹」的 ID 和 [巴哈](../剧毒魔鹰·巴哈.md)（「莫雷的老父亲」）对喷过多次。

## 天赋「孤存之幸」· 团灭体质

她的富有源自一个惊悚的真相:天赋能力「**孤存之幸**」（觉醒至 SSS 级）——平时她的运势会被这天赋逐渐吸收储存，一旦她面临生命危险，储存的运势会让她的幸运属性以夸张速度飙升保命，且**会吸收身边人的运势**。代价是:她走到哪、队友就一茬茬暴毙到哪。从一阶到七阶她被迫当了 **29 次冒险团团长**（每次都是队友团灭、只有她靠倒霉侥幸活下来、继承团队资产），一次祸害一整个冒险团、越死越富。她一直遵纪守法、尽力完成任务、还很看重队友，却成了天启乐园人人退避的「团灭体质」。八阶她加入封天公会（团长泰默说会里有七个类似的倒霉蛋），却连公会都差点被她拖垮、被劝单独行动。她因此看透一切、烫了波浪卷、以酒度日。她有个测吉凶的土法子:抛空酒瓶砸头——砸到就是没转运、代表安全；一旦开始转运（如赌场十连胜），才是大凶之兆。

## 撞上苏晓 · 采血姬

[凯撒](../尼古拉斯·凯撒.md) 以矿工身份约她去荒废伐木场「帮忙清野兽占的矿洞」，实为 [苏晓](../苏晓.md) 设的局。她一路被 [布布汪](../布布汪.md) 跟踪埋雷炸到腿骨裂十几次（重力地雷 / 磁爆猎手 / 阿波罗轮番招呼·连沸红做的「你迟到了」复读机尸傀都上了），最终被苏晓的**宗师之威**（顿挫重刀、直踹、空间穿透）打服。她的奥义级底牌「剑舞·雷闪」（引界雷入体提升斩击速度）对灭法者苏晓无效——苏晓靠「天怒·奔雷落」硬抗强度更高的界雷、反倒盯上她「以心脏为提取器、让界雷与血液共频储存」的血脉秘法。她被逮回太阳要塞**抽了 4000 毫升血**研究（醒来虚弱到打不过二三阶·敌人管够补气血药膳）、还稀里糊涂签了 **483 份**低惩罚契约。她大心脏、干脆认栽「我付钱」（比当初怀疑人生的 [月使徒](../苍龙大陆/月使徒.md) 从容太多）——被苏晓选作**新提款姬 + 声望刷子**（配合 [凯撒](../尼古拉斯·凯撒.md) 的军需官身份刷阵营声望）。她与 [莫雷](../原始大陆/莫雷.md) 是老冤家（莫雷曾一脚爆她肾、抢她地宫 boss 击杀），此番莫雷又反手把她「有 2 万灵魂钱币 + 矿石商会分成」的底细全抖给了苏晓。

## 关系

- [凯撒](../尼古拉斯·凯撒.md)：以矿工身份把她钓到伐木场的中间人。
