# 风暴翼龙·巴洛缪

[塞尔星](../../世界/任务世界/塞尔星-不败要塞.md)·[野兽族](../../世界/生物万族/index.md) 四兽行之一的「**天空头领**」——一头体长约 7 米、全身黑蓝色大羽毛、似鸟似龙的**风暴翼龙**（鸟喙 + 龙颈 + 四羽翼·暗黄竖瞳）。它在野兽族地位仅次于兽王，曾被一种未知能量侵蚀、智慧受损却更凶狠好斗。真名「巴洛缪」。

## 湮灭吐息

它的招牌是「**湮灭吐息**」——口喷一道水桶粗的白色气柱，把命中物**分解到「原子状态」**（原理类似火影的「尘遁」、却更暴力·只追求威力）。[苏晓](../苏晓.md) 判断这吐息强到「命中要害有 95% 概率被秒」，对它产生浓厚兴趣（想结合尘遁开发「分解炸弹」）——故意挨它俯冲撞击 + 湮灭吐息也不还手，只为不惊跑这会飞的高速坐骑。

## 从天空之主到软座龙骑

兽潮夜袭时它主动扑向苏晓这个「敌方领袖」，反被血枪炸下、放逐碎片硬拖出天空、麻醉生擒。苏晓割掉它体内储存湮灭吐息能量的器官（它误以为要被「割了」当史上首头被割的龙、流下热泪），又用进化巢的太阳之力（[太阳鸟源血](../../力量体系/装备道具/index.md) 为引·它有羽毛喙契合度高，甚至掌握了**太阳焰**）驯化——每直面 [太阳之环](../../力量体系/装备道具/index.md) 就进入狂信状态、脑袋砸地，从愤怒到无奈到麻木。最终苏晓以「合作一月、事后给它那仅剩一份太阳鸟源血」为台阶，签约收它作**龙骑坐骑**（有羽毛的「软座」·比黑龙米狄斯的「刺座」舒服太多）。大决战里它载苏晓引界雷成雷枪、一枪穿透兽巨人胸膛。世界通关后它留在塞尔星。

## 关系

- [苏晓](../苏晓.md)（太阳领主）：生擒、驯化、签约收它当龙骑坐骑的主人（拿太阳鸟源血 + 太阳之环拿捏它）。
- 兽王 / 海怒头领美人蛇：它的野兽族君主与同僚（兽王最信任它·闻它被俘要被「割」而神色凝重）。
- [贝妮](../苏晓.md)：龙背上给它当兽语翻译的猫。
