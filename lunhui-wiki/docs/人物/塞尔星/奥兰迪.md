# 奥兰迪

[塞尔星](../../世界/任务世界/塞尔星-不败要塞.md) 世界争夺战里**守望乐园方的领袖级人物**——大名鼎鼎的狠角色。提及他，连天启方的 [黄金伯爵](黄金伯爵.md)、圣光方的 [圣诗](圣诗.md) 都会面色微变、露出一种难受的表情:他不只是强，还**诸多光环加身**（「哲♂学家」「魔男」等）。

## 坐收渔利的第三方

塞尔星争夺战中，天启方与圣光方因一场误会（[苏晓](../苏晓.md) 伪装天启、令 [猎潮](../联盟星/猎潮.md) 被误当天启召唤物伏击）打得不可开交——守望乐园方看到这一幕只会拍手叫好:天启、圣光两方斗得越狠，对守望方越有利。他作为守望方领袖，是这场三乐园混战里最可能坐收渔利的一方。

## 关系

- [黄金伯爵](黄金伯爵.md) / [圣诗](圣诗.md)：塞尔星争夺战中天启 / 圣光方的对手领袖（都对他有几分忌惮）。
- [苏晓](../苏晓.md)（隐藏的入侵者）：这场三方混战里，他和另两位领袖都还没察觉的真正变数。
