# 奥兰迪

[塞尔星](../../世界/任务世界/塞尔星-不败要塞.md) 世界争夺战里**守望乐园方的领袖级人物**——大名鼎鼎的狠角色。提及他，连天启方的 [黄金伯爵](黄金伯爵.md)、圣光方的 [圣诗](圣诗.md) 都会面色微变、露出一种难受的表情:他不只是强，还**诸多光环加身**（「哲♂学家」「魔男」等）。

## 坐收渔利的第三方

塞尔星争夺战中，天启方与圣光方因一场误会（[苏晓](../苏晓.md) 伪装天启、令 [猎潮](../联盟星/猎潮.md) 被误当天启召唤物伏击）打得不可开交——守望乐园方看到这一幕只会拍手叫好:天启、圣光两方斗得越狠，对守望方越有利。他作为守望方领袖，是这场三乐园混战里最可能坐收渔利的一方。

## 关系

- [黄金伯爵](黄金伯爵.md) / [圣诗](圣诗.md)：塞尔星争夺战中天启 / 圣光方的对手领袖（都对他有几分忌惮）。
- [苏晓](../苏晓.md)（隐藏的入侵者）：这场三方混战里，他和另两位领袖都还没察觉的真正变数。

## 🆕 不倒的魔男 · 硬挡重装坦克（45卷54–57章）

防御战里他与 [圣诗](圣诗.md) 分工:圣诗留住 [苏晓](../苏晓.md)、他去外围挡野猪战士大军。他的能力名副其实「不倒的魔男」——**不用盾的坦系，敌人攻击他越狠、他反轰出的扇形拳压范围越广威力越强**，还能把被轰碎的敌人血肉改性吸回体内回血回体力（故越战越猛）。他一拳能清空几十名野猪战士、硬正面锤翻 3 只 [重装坦克](../../世界/生物万族/index.md)（右臂皮肉裂开、腰也撑不住·第 4 只起挡不动了）。可野猪战士多到一望无际、重装坦克分割战场，他终究独木难支。他的冒险团里 [圣诗](圣诗.md) 是宽宏一面、副团长德鲁伊是严苛一面（两人竞争关系）。最终随圣诗的救赎传送逃走、带残兵在大平原奔逃。

## 关系（补充）

- **德鲁伊**：他冒险团的副团长（麋鹿角盔 / 兽皮·主纪律严苛）·与他竞争关系·带 600 殿后队。

## 🆕 大决战溃败 · 世界失守（45卷92–98章）

首战被血羽反制、带残兵逃亡后，他随黄金伯爵、圣诗一同押注克瓦勃环城的最后防线，与眷族 + 极光议会 + 人族联手死守。可太阳军团的野猪骑士 + 720 重装坦克 + 龙骑 + 灾难霸主·泽芜的组合势不可挡——他这个「不倒的魔男」也无力回天，眼睁睁看着眷族联防被 [苏晓](../苏晓.md) 一路平推，世界被轮回乐园夺走、守望乐园方随之出局。
