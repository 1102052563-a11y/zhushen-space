# 圣诗

[塞尔星](../../世界/任务世界/塞尔星-不败要塞.md) 世界争夺战里**圣光乐园方的领袖级人物**——她是圣光乐园的**处刑者**（相当于轮回乐园的猎杀者）。外表温柔得如邻家大姐姐，内心却坚韧强大到可怕，是 [仙姬](../联盟星/仙姬.md) 的死敌。

## 追杀仙姬的处刑者

她没打过乐园竞技场（八阶），但身为处刑者、已与违规者 [仙姬](../联盟星/仙姬.md) 死磕了很久——若非这次要参与塞尔星世界争夺战，她本会继续追查仙姬的踪迹。反过来，仙姬答应与 [灰绅士](../炎灵星/灰绅士.md) 合作的主要原因之一，正是想等树生世界后联合灰绅士、神父一同除掉圣诗（没人比仙姬更清楚这个「温柔大姐姐」内里有多难缠）。

## 塞尔星的领袖

她已与洛亚什城审判所的首席大法官达成合作，洛亚什成了圣光乐园方契约者盘踞的地盘。世界争夺战里，天启 / 圣光两方本已约定「找到世界之核前先别开战」——却因 [苏晓](../苏晓.md) 的烙印伪装成天启方、令 [猎潮](../联盟星/猎潮.md) 被误认成「天启召唤物」闯进圣光地盘遭伏击，一场大规模火拼被这误会点燃。作为圣光方领袖，她是这场乱战最头疼的操盘者之一，与天启方领袖 [黄金伯爵](黄金伯爵.md)、守望方领袖 [奥兰迪](奥兰迪.md) 三足鼎立。

## 关系

- [仙姬](../联盟星/仙姬.md)：她死磕追查的圣光乐园违规者、死敌（仙姬正谋划联合灰绅士除掉她）。
- [黄金伯爵](黄金伯爵.md) / [奥兰迪](奥兰迪.md)：塞尔星争夺战中天启 / 守望方的对手领袖。
- [苏晓](../苏晓.md)（入侵者·伪装天启）：无意间用一场误会把她卷进天启 VS 圣光乱战的搅局者。
