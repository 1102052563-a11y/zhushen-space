# 魂师

[塞尔星](../../世界/任务世界/塞尔星-不败要塞.md) 世界争夺战里圣光乐园方的**灵魂系强者**——身着黑袍绣金、戴兜帽，一双能「洞穿灵魂」的摄人眼睛。圣光 / 守望结盟后，他带一支偶遇拼凑的三十几人队伍（含刺过 [猎潮](../联盟星/猎潮.md) 的金属妹、肌肉男迪恩等），听信「太阳要塞只有一名天启契约者独守世界之核」，被他这灵魂系大佬「给了勇气」，一头闯进要塞去夺核。

## 撞上克星

他万万没想到，[苏晓](../苏晓.md) 恰恰最不怕两种人:法系（苏晓是灭法者）和**灵魂系**（苏晓灵魂强度 560 点 + 灵韧 Lv.30·大部分灵魂能力轰在他身上只是「局部刮痧」与「大面积刮痧」的区别）。他祭出灵魂系的招牌:「灵魂之手」拽敌人灵魂——却像幼儿园小朋友过肩摔成年人，非但没拽动苏晓的灵魂，反被苏晓在空间穿透状态下一抬手臂震得右臂软软垂下（灵魂反噬）；又用「灵魂震撼」把自己周边队友全轰飞、唯独轰不动淡定的苏晓（苏晓生命只掉 2.65%）。最终苏晓一脚直踹，把他腹部以下炸成血雾、上半身**镶进墙里抠都抠不出来**，发出将死般的吸气声。苏晓评价「这修灵魂系的，未免太不禁打」。

## 关系

- [苏晓](../苏晓.md)（白夜 / 灭法者）：他灵魂系的天然克星——拽不动灵魂反被震伤、一脚踹进墙。
- **金属妹 / 肌肉男迪恩**：他这支三十几人夺核队里的同伴（金属妹刺过猎潮·同批被苏晓团灭）。
