# 豪斯曼

[塞尔星](../../世界/任务世界/塞尔星-不败要塞.md)·末日要塞里第一个敢拿起武器反抗眷族的 [猪头人](../../世界/生物万族/index.md)——[苏晓](../苏晓.md) 夺要塞、养炮灰计划的**起始点与第一块基石**。他身穿脏到看不清本色的背心、耳朵被割下一块，身高约 2.6 米、体重 240 公斤以上，[豪斯曼](豪斯曼.md) 拎起前要塞头领 [利·西尼威](利·西尼威.md) 就像拎小孩。

## 抢名字的狠种

苏晓在矿井里射杀监工后，让一个憨厚猪头人「敲死看守」——那猪头人不敢，被苏晓判「没用了」；正当血枪成形，是豪斯曼用肩膀撞开挡路者、夺过铁棍，无视看守哀求把他活活砸死、一脚踩碎头颅。他为何这么猛?不是天选、不是资质非凡，纯粹因为**年轻**（28 岁·兽性最强的青壮年）、心中憋着太多「为什么」（为什么天生就是下等生物、为什么眷族能随意杀他们）。

他连**名字都是抢来的**——苏晓问他叫什么，他指着地上被砸死的看守尸体:「我杀了他，他的名字就属于我」（那看守原叫豪斯曼）。这是个「没什么就抢什么」的狠种。他本不太会说话（下矿十几年只敢挖矿时偷偷嘟哝、模仿眷族监工），却在苏晓面前一句句顺畅起来。

## 起义军头领

一个苹果（他这辈子吃过最好吃的东西·震惊到咬到自己拇指）+ 一句「做得好」，把他收为心腹。苏晓让他当**36 名猪头人起义军的头领**（要塞共 673 名猪头人·仅这些敢反抗）——凶狠、有胆、懂判局、不撒谎。他负责带头攻占要塞二三层、看押降兵、以「敲死这看守敢不敢」筛选新成员（敢的加入、不敢的淘汰）。攻要塞时冲锋在前，与铁头憨批 [钢牙](钢牙.md) 各拎一把大号剁肉斧守在总操控室门口。苏晓评他:懂观察局势、狠中带稳，是猪头人里最堪当头领的。

## 关系

- [苏晓](../苏晓.md)（库库林·白夜）：给他苹果、教他反抗、封他当起义军头领的「解放者」兼主公。
- [钢牙](钢牙.md)：同为敢反抗的猪头人、并肩冲锋的憨勇战友。
- [利·西尼威](利·西尼威.md)：他曾经的压迫者、如今被他拎来拎去的阶下囚。

## 🆕 蜕变野猪人头目 · 领袖气质（45卷23–33章）

深入异化兽领地的死战里，他成功从猪头人**返祖蜕变为野猪人**——头顶、后脑、脊背长出向后竖的浓密黑硬鬃毛，威武霸气，颇有头领气质，早不是憨憨的猪头人模样。作为**野猪人头目**（相当于野猪人里的小 boss·仍有成长空间），[苏晓](../苏晓.md) 查得他的属性:力 192 / 敏 170 / 体 201（皆受战争领主加持）、智 57、魅 1。技能有:**越战越勇**（生命 +14000·越掉血回血越快·最高每秒 +35）、**顽强不屈**（生命 +5200·受伤痛觉刺激野兽值）、**自由之兽**（野兽值满触发·全属性略升·持续 18 分 05 秒·状态内击杀有小概率永久涨六维）、终极技能**奋力一击**（扇形喷发钝 / 劈 / 震击）。此后他随 [莫雷](../原始大陆/莫雷.md) 带 300 野猪人去洗劫眷族要塞、守总指挥室门。
