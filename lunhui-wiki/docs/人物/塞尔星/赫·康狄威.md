# 赫·康狄威

[塞尔星](../../世界/任务世界/塞尔星-不败要塞.md)·**眷族同盟的元帅**，人称「**孤高之狼**」——是把人族军方打到胆寒的、眷族历史上唯一的元帅级军事家，[苏晓](../苏晓.md) 在本世界最忌惮的对手。他年逾 40、骨架宽大、黑红军装上只挂一枚白金同盟元帅徽，浑身不怒自威。他是**眷族权力最高四人**之一（另三位:眷族同盟长 [托因](../../世界/任务世界/塞尔星-不败要塞.md)、哨塔领袖 [斐迪南](../../世界/任务世界/塞尔星-不败要塞.md)、审判所首席大法官 [佛沃](../../世界/任务世界/塞尔星-不败要塞.md)）。

## 被克制的孤狼

他和眷族同盟长·托因是宿敌:托因是官僚派之首、掌经济财政大权，赫·康狄威是军方之首。因上次的失利，他屈居次位、被托因死死钳制多年——托因简直是他这辈子的「专属克星 / 宝具」（干别的不行、专治赫·康狄威手到擒来）。[苏晓](../苏晓.md) 为拖延开战，一夜暗杀眷族四巨头（本想弄乱眷族），却**误杀了托因**——等于亲手替赫·康狄威除掉了克星、放虎归山。托因一死，孤狼当场「支棱起来」、独揽军权，苏晓懊悔「误杀了队友」。

## 边壤条约 · 毒辣的和谈

翻身后他没蛮干，而是先停战、后求和，抛出一份**「边壤条约」**的毒计（同盟长 / 哨塔 / 大法官 / 十四议员全签字）:承认太阳阵营与野猪战士的存在、承认边壤区是苏晓的神圣领土、每月还送幼年猪头人助其繁衍——代价是苏晓不得干涉猪头人买卖、要把野猪战士 / 矮猪人与猪头人官方切割成「两个物种」。看似忍气吞声，实则毒辣:边壤区贫瘠、大矿脉半月挖空，签约后苏晓只能往东扩张、去和野兽族死磕——等于**替眷族白打工挡兽潮、还帮眷族清缴心腹大患野兽族**。他把苏晓当枪使的算盘打得精,却不知苏晓根本不受这「厕纸级」契约约束——签约反成了苏晓去克瓦勃环城搞崩敌方心态、找暗氤、重开角斗场的通行证。

## 关系

- 眷族同盟长·托因：他的宿敌 / 专属克星（掌财政钳制他多年）——被苏晓误杀、反让他翻身。
- [苏晓](../苏晓.md)（库库林·白夜 / 太阳领主）：他最忌惮的对手、又是无意间帮他除掉克星的人；他抛边壤条约想把苏晓当挡兽潮的枪使。
- [雷兹准将](雷兹准将.md)：他最信任 / 力挺回归的旧部名将。
- [利·西尼威](利·西尼威.md)：帮他解毒、收作狗腿子的「二五仔」——却又被苏晓派去毒杀他。

## 🆕 断绝希望的投降 · 全名格莱斯顿（45卷97–98章）

克瓦勃环城大决战里，苏晓的太阳军团在灾难霸主·泽芜、兽巨人的对撞后彻底冲垮眷族 + 极光议会 + 人族的联防。城破后，他做出一个理智到可怕的抉择——为避免眷族被苏晓屠戮到亡族灭种，他**亲手杀绝眷族一切崛起的希望**:割了继承人（就是**起草边壤条约的五子**·他从小倾注全部心血培养）的喉咙，杀尽其他能掌权的贵族亲系，连昔日好友首席大法官 [佛沃](../../世界/任务世界/塞尔星-不败要塞.md) 也死在他手中。他以「成为霸主很光鲜、屠灭一个族群很不光彩」向 [苏晓](../苏晓.md) 递上眷族最低微姿态的投降，用匕首从容挑开自己动脉——只求换眷族不被灭种、纵使沦为中小种族也算延续。他自报全名「**格莱斯顿·赫·康狄威**」，说会尽全力取苏晓人头慰祭英灵。苏晓接受投降、用 D·暗杀补一枪给他留了全尸。原来那份让苏晓忌惮的「边壤条约」鬼才，正是他这个五子。
