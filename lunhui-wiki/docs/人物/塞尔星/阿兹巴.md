# 阿兹巴

[塞尔星](../../世界/任务世界/塞尔星-不败要塞.md)·自由城**地下市场**的老大——一个戴小圆墨镜、穿西装、大腹便便、发尖鼻子的**侏儒**（疑有人类外的血统），[凯撒](../尼古拉斯·凯撒.md) 的生意伙伴。他专贩卖 [猪头人](../../世界/生物万族/index.md)、异化兽，以及被审判所判成囚奴的眷族 / 人族。人称「**奴隶商人·阿兹巴**」。

## 有规矩的黑商

他兼具生意人的奸猾与黑暗世界的狠辣，也是个「翻脸比翻书快」的家伙（前一秒摘墨镜冷脸、下一秒听到「买 4000 名以上」立马笑容满面搭肩）。他有个怪癖:每到新地方必去嫖、嫖到失联——按理奴隶商人不必如此，他的说法是「**我出售的是商品，进货什么样、出货就什么样，这是规矩，做生意要有规矩**」。地下市场是自由城**每晚财富流动量最大、也最黑暗的地方**:铁笼里关着囚奴眷族 / 人族、异化兽，猪头人戴脖镣铁链相连。

## 与苏晓的交易

经 [凯撒](../尼古拉斯·凯撒.md) 牵线，[苏晓](../苏晓.md) 在他这批量采购猪头人。他起初想推销 700 公斤活性矿石一名的**猪头人斗士**（自由意志、善战、卖给各环城角斗场），苏晓没兴趣;苏晓专挑**挖矿桀骜不驯的「残次品」刺头**（越不听话越有反抗意识），阿兹巴一度冷脸疑他要「繁育」（南边真有人这么搞、年年吃审判所罚单），被凯撒一句「你卖雄性给库库林、卖雌性给我，私下交易与你无关」+「买 4000 名以上」说动。最终苏晓以 4000 公斤活性矿石买下 **6359 名残次品刺头** + 500 名雌性猪头人（挑雌性时还找 10 名猪头人斗士来做「审美评委」——雄性猪头人偏爱壮硕大胖脸的雌性）。他还牵线把利·西尼威塞进审判所。

## 地下市场的血史

传奇斗士 [奥因克](../../世界/生物万族/index.md)（身价曾抬到 58600 公斤活性矿石）当年带猪头人斗士反抗、被审判所处死后，眷族一度全面镇压（猪头人戴镣挖矿、角斗场停业）——是阿兹巴等地下市场商盟集体断供审判所资金，逼得死了两名大法官、地下头领死过半、阿兹巴本人失去一条腿，才让局面恢复。

## 关系

- [凯撒](../尼古拉斯·凯撒.md)：多年生意伙伴、这次交易的中间人（俩人像多年好友）。
- [苏晓](../苏晓.md)（库库林 / 白夜）：在他这批量买猪头人刺头、发展炮灰军的大客户。
- [多萝西](多萝西.md) / [利·西尼威](利·西尼威.md)：他经手安插利·西尼威进审判所的相关方。
