# 黄金伯爵

[塞尔星](../../世界/任务世界/塞尔星-不败要塞.md) 世界争夺战里**天启乐园方的领袖级人物**——个人实力属八阶顶尖梯队，是**乐园竞技场（八阶）第七名**（之上六人:三名轮回乐园、两名死亡乐园、一名来自虚空·由此可见其实力）。在世界联络平台上以「**战争领袖**」的头衔发言、统筹天启方战局。

## 不看好月使徒

他始终不看好 [月使徒](../苍龙大陆/月使徒.md)——嫌她的召唤流太不稳定（发展起来固然强大、发育不起来挨捶也特别狠）。塞尔星争夺战里，[莫雷](../原始大陆/莫雷.md) 与月使徒虽在本方阵营内有点名气，却入不了他的法眼。乱战中莫雷、月使徒在联络平台跟他呛声（「渣渣，你不客气给我看看?」），他只淡淡回「私人恩怨，我不会轻易干涉，但影响战局走向别怪我不客气」——分得清轻重。

## 三方领袖之一

天启 / 圣光两方本约定「找到世界之核前先别开战」，却因 [苏晓](../苏晓.md) 伪装天启、令 [猎潮](../联盟星/猎潮.md) 被 [圣诗](圣诗.md) 方误当「天启召唤物」伏击，两方乱战被点燃——他作为天启方领袖，与圣诗、守望方领袖 [奥兰迪](奥兰迪.md) 三足鼎立，还没料到自己阵营里正藏着入侵者苏晓。

## 关系

- [月使徒](../苍龙大陆/月使徒.md) / [莫雷](../原始大陆/莫雷.md)：他不看好、却在本方阵营的召唤系与战斗天使。
- [圣诗](圣诗.md) / [奥兰迪](奥兰迪.md)：塞尔星争夺战中圣光 / 守望方的对手领袖。
- [苏晓](../苏晓.md)（伪装成天启方）：潜伏在他阵营里的轮回乐园入侵者。

## 🆕 一神带众坑到底 · 被围杀（45卷80–99章）

轮回乐园入场后，他与眷族阵营捆绑、成了苏晓最难缠的对手——不像希那样冲在最前，而是「看似没做什么、其实已力挽狂澜两波」:上次防御战他连战区都没来，明显是去找 [暗氤](../../世界/任务世界/塞尔星-不败要塞.md)（本世界补全世界之核的关键·他七成把握已得手），把即将崩盘的天启局势悄悄扳回。他自知无胜算却很有骨气、硬顶怠战惩罚、想帮天启保住半个世界（打平局则天启 / 轮回各占一半）。可 [苏晓](../苏晓.md) 靠半颗世界之核感应暗氤、率几十万野猪骑士围堵，让他连栖身之地都没有——最终 [黄金伯爵](黄金伯爵.md) 被野猪骑士围杀。他掉的**领袖专属猩红卡**、以及生前存款 **97560 灵魂钱币**（苏晓抽 20% = 19512），印证了天启乐园「有黄金伯爵这样的人才这般富有」。

## 关系（补充）

- [苏晓](../苏晓.md)（太阳领主）：把他一路逼到无处栖身、围杀的对手（掉领袖猩红卡）。
