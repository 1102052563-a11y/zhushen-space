# 辛·阿丽丝

[塞尔星](../../世界/任务世界/塞尔星-不败要塞.md) 杀手家族**辛之一族**的成员——族长 [辛·狄宗](辛·狄宗.md) 的**第六个女儿**，风韵犹存、指间夹黑色香烟、手上一道道疤痕透着危险气息。她是 [利·西尼威](利·西尼威.md) 的**老情人**，也是 [多萝西](多萝西.md) 的**杀母仇人**。

## 血债

当年 [利·西尼威](利·西尼威.md) 在极光议会当官时勾搭上她，她得知他有妻女后，一周后提着礼盒登门——盒里是利·西尼威**妻子的头颅**，还想抓走年幼的多萝西「转正」。这桩丑闻闹大，辛之一族丢不起人、本要连利·西尼威与多萝西一起处理掉，靠极光议会出面才作罢（苏晓看来极光议会只是把利·西尼威当弃子、想等风头过再秘密除掉卖辛之一族人情）。这血债，是 [多萝西](多萝西.md) 恨父、也最恨这个「小碧池」的根源。

## 藕断丝连的暗线

塞尔星世界争夺战里，她与老相好利·西尼威在洛亚什的酒店重续旧情（把他从床上一脚踹到地毯上、拿他寻开心）——正是靠她与辛之一族的暗杀本领，利·西尼威才在短短几天就**除掉了审判所那名贪婪的老吸血鬼、取而代之爬上中高层**。这也让 [苏晓](../苏晓.md) 警觉:处在绝境的利·西尼威联络她，说不准哪天会让这老情人来暗杀自己。

## 关系

- [辛·狄宗](辛·狄宗.md)：她的父亲、辛之一族族长「辛鬼」。
- [利·西尼威](利·西尼威.md)：她的老情人——借她与辛之一族的力量在审判所站稳脚跟。
- [多萝西](多萝西.md)：她害死其母、想抓走「转正」的少女——两人血仇未了。
- [苏晓](../苏晓.md)（白夜）：警惕她可能被利·西尼威派来暗杀自己的算计者。

## 🆕 真相 · 一场母爱的骗局（45卷91章）

托钵寺终章揭开惊天真相:化名 [辛·尤戈](辛·尤戈.md) 接受暗阳的其实是她本人，而**她才是 [多萝西](多萝西.md) 的生母、[利·西尼威](利·西尼威.md) 的原配大老婆**——多萝西认作母亲、被她当着幼年多萝西面虐杀的奥丽佩雅，其实是利·西尼威的女仆 / 后娶妻。当年辛之一族拒了这桩把家族架火上烤的婚事、把她逮回禁锢；几年后解封的她思女成魔、找到住址却见丈夫另娶、女儿唤别人母亲，精神失常下虐杀了奥丽佩雅。她深知**一体不容两吞噬者**、却故意化身「辛·尤戈」来夺 [多萝西](多萝西.md) 的沸红成「完美体」——这是她精心设计的赴死:让女儿以为「手刃了杀母仇敌」、有个轻松的未来，而非知道生母虐杀养母后浑浑噩噩一生。夺沸红后她当即自毁身亡，气绝在女儿怀中，留下沸红暗阳的融合体 + 灵魂匣。这真相的知情者只剩苏晓与 [辛·狄宗](辛·狄宗.md)。

## 关系（更正）

- [多萝西](多萝西.md)：其实是她的**亲生女儿**（非杀母仇人·她杀的是女仆养母奥丽佩雅）——她以赴死成全女儿的未来。
- [利·西尼威](利·西尼威.md)：她的**原配丈夫**（非老情人）。
