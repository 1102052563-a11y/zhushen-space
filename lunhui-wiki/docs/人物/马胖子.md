# 马胖子（现实·花圈寿衣店老板）

[苏晓](苏晓.md) 现实世界饰品店**隔壁花圈寿衣店的老板**——满身肥肉、自称「风水大湿」的江湖骗子，却是**有良知、讲义气**的那种。和情人厮混、被苏晓地下室的锤击声吓得一惊一乍，是个市井活宝。

- **讲义气的旧交**：早年苏晓复仇时被别的杀手追杀，马胖子无意间得知后**把自己老家的钥匙给了苏晓避难**、自己也消失了一阵——当时两人不过普通朋友，他却没出卖苏晓。自此苏晓对他信任有加，自己「猎杀者」的隐秘他也一向守口如瓶。
- **勇闯魔窟前奏（十二卷94章）**：苏晓为「突破身体极限」任务在饰品店后院埋了几十颗松发式**警戒地雷**（塑料·只警戒不杀人），马胖子端着十字弩 + 电棍来查动静、一脚踩中、吓得魂飞魄散（实则只是「延时爆竹」级别）。他头回见真枪（M9 / HK-416），兴奋地下地下室试枪、摆出「汉奸射击法」打得不亦乐乎。还把心爱的越野车借给苏晓去无人区。

## 关系
- **[苏晓](苏晓.md)**：信任的现实友人（曾给老家钥匙助其避难·守其「猎杀者」秘密·借车·请苏晓救弟）。
- **黑皮**：亲弟弟（灰色商人·倒卖弓弩 / 收来路不明货·十六卷被亡命徒绑架·苏晓救回）。

## 仗义挡「老墨」护友（十四卷85章）

苏晓返现实后，两名光头墨西哥杀手（[德席尔瓦家族](index.md)）正午直奔他的饰品店。马胖子以为是仇家来报复苏晓——这个对风水只一知半解、却「能把死人说活」的江湖骗子，**明知凶险仍背着手用「印堂发黑、有大凶、贫道愿为二位挡灾」的忽悠大法上前拦人**（背手暗中给苏晓发消息），险些把脾气火爆的老墨惹炸。关键时刻 [布布汪](布布汪.md) 赶到挡在他身前、用「狼王般的目光」把两个老墨盯出冷汗，马胖子才脱险。「能为朋友奋不顾身」正是他能与苏晓做朋友的最大原因。

## 求苏晓救弟黑皮（十六卷68–69章）

他**弟弟黑皮**（隔着苏晓饰品店的另一边开店·倒卖弓弩 / 收来路不明货的灰色商人）被一伙**亡命徒盗墓贼**绑去西郊（因收了不该收的东西、牵连对面古董店）。马胖子绑了对方留下的一名亡命徒「严刑逼供」却下不去手、又被威胁不许报警，走投无路只得**搬出「专业人士」苏晓**（他不敢亲自从后门找苏晓——上次去饰品店后院踩了苏晓埋的地雷）。苏晓端着面条赶来、界断线一勒就吓得亡命徒全招，转手喊清道夫头目 [魏冬](../人物/魏冬.md) 把黑皮安全带回、把亡命徒「处理掉」。苏晓帮他是还当年「给老家钥匙避难」的人情、也为帮这对普通兄弟以绝后患（亡命徒缠上普通人会要命）。
