# 魏冬

**魏冬** 是现实世界中国家机构「**清理不稳定因素规划局**」（俗称「**清道夫**」）的成员，主要负责与同类（契约者）**交涉或战斗**（四卷11章）。

## 简介
身材魁梧；本人也是契约者，被国家收编。他率上百名西装契约者在高铁站拦下化名「苏笑」的苏晓——并非来围杀，而是**奉命招安**：递上一份文件，只要签字（承诺不杀人、不制造社会动乱、不非法集资、不拉帮结伙），既往不咎、正常生活无人干涉，否则清道夫将「不死不休」。苏晓签字后，其现实身份**合法化**。

魏冬看出苏晓实力极强（「拔刀的气势吓了我一跳，两天前若交手可能要出动军方」），邀其加入被拒，转而提出**合作**：偶有强大外国契约者来犯时请苏晓出手、以**机密情报**为回报（苏晓视情况考虑）。事后判定苏晓「无害」（只在现实杀过一名仇人、潜入警局也只为取回己物）、撤除监视——却不知苏晓在衍生世界何等凶悍、也没察觉苏晓早已感知到监视。

## 关系
- 小叶——清道夫少女成员（负责「撑场面」·见苏晓拔刀汗毛倒竖）。
- [苏晓](苏晓.md)（苏笑）——招安并合法化其身份的对象，留有「杀外国契约者换情报」的合作余地。
