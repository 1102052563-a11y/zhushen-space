# 魏冬

**魏冬** 是现实世界中国家机构「**清理不稳定因素规划局**」（俗称「**清道夫**」）的成员，主要负责与同类（契约者）**交涉或战斗**（四卷11章）。

## 简介
身材魁梧；本人也是契约者，被国家收编。他率上百名西装契约者在高铁站拦下化名「苏笑」的苏晓——并非来围杀，而是**奉命招安**：递上一份文件，只要签字（承诺不杀人、不制造社会动乱、不非法集资、不拉帮结伙），既往不咎、正常生活无人干涉，否则清道夫将「不死不休」。苏晓签字后，其现实身份**合法化**。

魏冬看出苏晓实力极强（「拔刀的气势吓了我一跳，两天前若交手可能要出动军方」），邀其加入被拒，转而提出**合作**：偶有强大外国契约者来犯时请苏晓出手、以**机密情报**为回报（苏晓视情况考虑）。事后判定苏晓「无害」（只在现实杀过一名仇人、潜入警局也只为取回己物）、撤除监视——却不知苏晓在衍生世界何等凶悍、也没察觉苏晓早已感知到监视。

## 兑现人情 · 借刀杀外来者（八卷81–84章）
当初清道夫曾帮苏晓**调专机出国**（追查父母身世），这是一笔人情。如今他来「讨债」——辖区出现**三名「外来者」（国外偷渡入境的二阶契约者·[巴赫](巴赫.md) / [拉哈尔](拉哈尔.md) + 一名黑人感知型）**：他们组类佣兵团接脏活、因拉哈尔强奸酒店女服务员与清道夫翻脸开战，杀了魏冬许多人（含他**最信任的手下**）。城市中不能用大规模热武器、辖区又没强契约者，魏冬偏偏**不能向总部求援**（否则身份受冲击·正逢竞选），只得「**找个杀神教他们做人**」——请 [苏晓](苏晓.md) 出手（「上次帮你一次、这次你帮我们、很公平」）。

- 苏晓**一夜清剿三人**（详见各人物页）。出了岔子：目击者 [李慕蕊](李慕蕊.md) 竟是将军 [方老](方老.md) 的外孙女、不能灭口——魏冬出了清场不净的责任、急得满头冷汗，所幸方老正是苏晓旧识、私下了结。此后苏晓「**人情债最难还**」、不愿再与清道夫多有瓜葛；魏冬另给他一个邮箱 + 联络电话以备日后。
