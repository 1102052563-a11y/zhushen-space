# 枯屠（海贼声望榜首 · 五阶契约者）

[海贼王 · 德雷斯罗萨海战](../世界/任务世界/海贼王-德雷斯罗萨海战.md) 中**海贼声望排行榜首位**的五阶契约者——近 2 米、戴黑色金属面具，掌一种**漆黑龙卷风**能力（直径几十米·**卷入龙卷风的一切快速枯萎、化为粉尘**）。开局即在蛋糕岛掀龙卷风、把大妈的部下与建筑成片绞成粉尘，十分钟刷到惊人的海贼声望。

- **枯屠冒险团**：6 名五阶契约者组成的**五阶小型冒险团**（人数少）。手下有**演奏手**（礼帽 + 领结·腰挂骑士剑 + 细剑·像拉小提琴般用音爆 / 音波·一震碎 500 米建筑·斯文却精神不正常、自诩「儒雅」、怕女 BOSS「阉了他」）、一名持晶体重盾的大汉，以及一名冷气森森的女 BOSS。十八卷18章他们在大妈的蛋糕岛 / 冰岛分头行动——演奏手 + 女 BOSS + [浅梦](浅梦.md) 拖住大妈，枯屠主刷声望。
- **进入资格 = 战争世界奖励**：他们的「阶位比这片开放世界高」、进入资格是某个**战争世界奖励**的一部分。枯屠深知**大肆破坏大型开放世界会引来「怪物」**（四阶还能应付、**五阶的惹不起**·他按面具似有不好回忆），故避开有「区域惩罚机制」的德雷斯罗萨、下一站转去凯多地盘继续刷声望。

## 与苏晓的「声望之争」（十八卷18–23章）

枯屠开局声望 1506、迅速飙到 6255——但 [苏晓](苏晓.md) 看出其**声望是「跳跃性」提升、没有长期稳定的获取渠道**，而苏晓有（借整个唐吉诃德家族用他的炸弹杀敌 + 自己狙杀强者）。苏晓不慌反喜，立志在世界结束前把声望怼过枯屠。

## 真相推断 = 梧桐树冒险团团长（十八卷48章·苏晓推断）

德岛海战收尾时苏晓想通了「为何强如五阶的枯屠等人不顺手杀掉大妈、只跟自己比声望」——枯屠等 6 名五阶疑大多是**梧桐树冒险团**成员（枯屠疑为团长·[浅梦](浅梦.md) / 蛮石确为该团），他们**用特殊手段进入了「阶位高于自身」的海贼世界**，代价是受限：不能击杀影响巨大的剧情人物（否则新世界格局崩溃），故只能在海贼声望上与苏晓较劲。海战最终声望榜枯屠落到第三（13575）、被苏晓（15906）与浅梦（14800）反超。

## 声望终局 = 惜败白夜（十八卷78章）

德岛海战的最后阶段，枯屠与 [浅梦](浅梦.md) 联手完成一个高难支线、声望一度反超苏晓 5000 多点、自以为「稳了」——却被苏晓在世界结束前兑掉罗杰佩枪（10000 声望）的「绝命一击」翻盘。最终结算他以 **25978 点居第二**（白夜 30258 第一、浅梦 23075 第三）。被传送到星空内的众契约者中，戴金属面具的枯屠**扬下巴向苏晓打招呼**——双方是正常竞争，他有「输了不丢人、输了还叫嚣才丢人」的胸襟。

## 关系
- **演奏手 / 女 BOSS / 持盾大汉**：枯屠冒险团的五阶团友。
- **梧桐树冒险团**：苏晓推断枯屠为其团长（[浅梦](浅梦.md) / 蛮石同团·见 [势力](../势力/index.md)）。
- **[浅梦](浅梦.md)**：声望榜第二的五阶契约者（与枯屠冒险团一同在蛋糕岛拖大妈）。
- **[苏晓](苏晓.md)（白夜·三阶）**：海贼声望排行榜上的竞争对手（苏晓誓要反超他）。
