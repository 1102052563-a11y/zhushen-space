# 吉尔伽美什

[Fate/Zero 世界](../世界/任务世界/fate-zero.md) 第四次圣杯战争中 [远坂时臣](远坂时臣.md) 召唤的 **Archer（弓之骑士）英灵**——又称「**金闪闪**」，曾是半神、最古老的**英雄王**（王中之王）。金色铠甲、金色短发、赤红眼瞳；背后能召出一圈圈金色空间波纹（从中取出宝具）。

- **性格**：高傲蛮横、唯我独尊，习惯称别人「**杂种**」（「不自高自大做什么王」）。杀人遵循两原则：① 敌人灵魂丑陋（由他本人裁定）② 敌人对他抱杀意。是个不算贤明、却把国家治理得最安定的暴君。
- **能单独行动**的 Archer——令 [远坂时臣](远坂时臣.md) 头疼：他大多不听御主命令、有「请求」还得反过来商量（远坂「与其说召来从者、不如说召来个爹」）。他也想杀御主，却忌惮**令咒**而不敢。
- **十一卷7章**：奉远坂时臣之命演了出「**杀掉 Assassin**」的戏（实为与言峰父子结盟、迷惑众御主的把戏）。听闻间桐家被未知英灵炸毁、间桐脏砚死、疑是「英灵杀了自己的御主」，他大感有趣、想会会那名未知英灵（即苏晓）。
- **英灵齐聚 · 与苏晓针锋相对（十一卷51章）**：废弃工厂英灵齐聚时，他立于路灯之上俯视众英灵（视如蝼蚁），斥 [征服王](征服王·伊斯坎达尔.md) 是「背着我自称为王的鼠辈」。[间桐雁夜](间桐雁夜.md)（认出他是 [远坂时臣](远坂时臣.md) 的英灵）对他怒吼，他射出几把低阶宝具欲杀这「疯狗」——**却被 [苏晓](苏晓.md) 一道刀芒尽数斩断**。他骂苏晓「杂种」，反被苏晓顶回「**你身上有伪神的味道、也有人的特征，神 + 人混合、到底谁才是杂种**」。他身后金色空间波纹增至几十个、欲取真正宝具——苏晓则拔出斩龙闪，盘算着「凭对方筋力 B / 耐久 B / 敏捷 B 的属性，只要能近身、赶在他取出『天地乖离之剑』前就砍他个怀疑人生」。

- **三大宝具**：① **王之财宝**——背后金色空间波纹中藏一个时代的工艺神秘结晶、可射出几十把宝具当箭（圣杯无力将其全部实体化、故射出的宝具**非实体、无法被苏晓吞噬**）；② **天地乖离之剑（EX 级对界宝具）**——又称「切裂世界之剑」、威力恐怖，**但须先「解析」本世界法则才能动用**（被轮回乐园判为二阶世界后远不及全盛）；③ **天之锁**——他最信赖的宝具、「律神」之物，**目标神性越高越无法挣脱**（缠住了苏晓的反击盾外壳）。另疑有一件与天地乖离之剑同阶的 **EX 级「预知」宝具**（苏晓推测、未确认）。
- **vs 苏晓 · 断臂败退（十一卷52–53章）**：苏晓以 [剃](../力量体系/技能.md) 瞬移从所有英灵感知中消失、贴身斩向他首级——他凭那件「预知」宝具提前 0.2 秒预见、堪堪偏头躲过（只断几缕金发）；可当他去取天地乖离之剑时，**被苏晓追斩、连剑带持剑右臂一并斩飞**（剑尚未解析法则）。他以王之财宝 + 天之锁压制反击盾中的苏晓、收起傲慢改口称「你」（不再喊「杂种」、敬重实力、「你，伤了本王」），认真取出高阶宝具欲决战——却在此时**被御主 [远坂时臣](远坂时臣.md) 一道令咒强制召回**，暴怒大骂「**时臣、你胆量不小**」后化光消失。被斩断的右臂留着 [斩龙闪「万仞宗」](../力量体系/装备道具.md) 的伏笔（时臣的魔术未必接得回）。

- **改投言峰绮礼门下（十一卷59章）**：被时臣令咒召回、断臂未能恢复后，他**抛弃了 [远坂时臣](远坂时臣.md)、改作 [言峰绮礼](言峰绮礼.md) 的从者**（觉得弑父弑师的绮礼远比满脑子圣杯的时臣有趣）——还在绮礼背刺弑杀时臣时出手相助、并斩了赶来的 [间桐雁夜](间桐雁夜.md) 首级嫁祸。他给绮礼下了唯一命令：**找出 Berserker（苏晓）、「他必须死在我手中」**（要为断臂之耻再战一场），并提醒绮礼「以你的能力、被苏晓发现就是死期」。

## 死于「无源之水」的最终战（十一卷66–67章·殁）
[言峰绮礼](言峰绮礼.md) 一死、他的魔力供给被切断、成了「**无源之水**」——这正是苏晓**先杀绮礼削弱他、再引他来打的算计**（苏晓全盛单挑不是他对手、故避战削弱）。他傲慢赴约（被引到地下停车场），先用上百道空间波纹「洗地」式宝具压制；为速战，他罕见地取出 **天地乖离之剑（EX 级）**放出「**红色剑风**」（触之即「切」消失·连混凝土金属都抹去）。苏晓以 **300 法力青钢影 + 反击盾拼成全身能量铠甲**护体、用绝招「[连鞘斩·破空](../力量体系/技能.md)」冲入剑风——**一道蓝光直穿剑风、斩断他持剑的右臂**（苏晓自己也被剑风重创成独臂濒死）。他正得意，苏晓胸口暗藏的 [影刺](../力量体系/技能.md)（近实体的能量刺）从他体内连环破体而出、把他扎成「刺猬」——**被 [吞噬之核](../力量体系/技能.md) 吞噬**（本届最强英灵·因御主已死沦为残缺阵营·只给世界之源 +5.4%→88.9%·噬灵者 +30 法力 2320·掉**金色英灵匣**·青钢影借此升至 Lv.30）。

## 关系
