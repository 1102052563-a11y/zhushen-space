# 塔兹米

**塔兹米** 是 [斩·赤红之瞳](../世界/任务世界/斩赤红之瞳.md) 世界 [夜袭部队](../势力/index.md) 的新人成员（四卷24章）。

## 简介
来自帝国边境的乡下少年、心怀正义；本想加入帝国军，目睹帝都的黑暗后机缘巧合加入夜袭部队。会议上常率先发表「正义感爆棚」的言论。

!!! note "未必是本世界主角"
    塔兹米是原著《斩·赤红之瞳》的核心人物，但苏晓据原著情报判断**他或许并非本世界真正的「剧情主角」**、主角可能另有其人（四卷23章）——契合本世界「剧情主角 / 命运之力」可能另有归属的设定。

本批中他与王牌**布兰德**一队、被派去帝都港口豪华客船护官（四卷24、27章）。

## 关系
- [娜洁希坦](娜洁希坦.md)（首领）、[赤瞳](赤瞳.md) / [雷欧奈](雷欧奈.md) / [拉伯克](拉伯克.md) / 布兰德 等——夜袭同伴。
