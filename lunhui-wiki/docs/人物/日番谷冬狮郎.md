# 日番谷冬狮郎（十番队队长）

[死神世界](../世界/任务世界/死神-尸魂界.md) 护廷十三队**十番队队长**——身高仅约 1 米 3 的白发少年（死神年龄不能以模样判断）。性格不耐烦、嫌琐事多，却**善于读心**（「想判断一个人、最重要是知道他在想什么」）。是白道门门卫**兕丹坊**的好友（这也是苏晓选从白道门方向潜入的原因）。[苏晓](苏晓.md) 评其心机为 10（远不及卯之花烈的 80）。

## 被「钓出」 · 引苏晓入瀞灵廷（二十九卷4–6章）

兕丹坊被苏晓贿赂后找来好友冬狮郎接待这名「投诚的游魂」。冬狮郎一眼看出苏晓**「不是我见过最强的游魂、但却是灵压最不祥的」**，本想关门了事——却被苏晓抛出的那瓶**混了半固态灵子的炼金药剂**勾住（那东西经卯之花烈鉴定是「了不得的东西」）。他重返询问苏晓目的（试探「你生前杀了多少人」、由苏晓「忘记了 = 多到数不清」读懂其底细），终究为了炼金药剂对死神的战略价值（减少死神讨虚的死伤）把苏晓**引入瀞灵廷**、转交卯之花烈处理。他全程被苏晓艰涩难懂的炼金话术绕到脑壳隐隐作痛（「跨频聊天」）。

## 关系
- **[苏晓](苏晓.md)（白夜）**：被其「钓出」、引入瀞灵廷的第一个死神队长（始终觉得苏晓不对劲）。
- **兕丹坊**：白道门门卫好友（苏晓借这层关系接近瀞灵廷）。
- **松本乱菊**：十番队副队长 / 助手（金发丰满·凡事「略懂」）。

- 二十九卷4–6章（被兕丹坊找来·被苏晓的炼金药剂钓住·引苏晓入瀞灵廷）。
