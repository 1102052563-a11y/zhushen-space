# 四代雷影·艾

云隐村（雷之国）首领、**第四代雷影**——身高近两米的壮汉，白袍后绣「雷」字，光站着就有强大压迫感。脾气火爆（一拳拍碎办公桌）、却也识人。

## 弟被抓 · 召开五影大会（十五卷7–8章）

他的义弟 **[奇拉比](奇拉比.md)**（八尾人柱力·两人无血缘却情同手足、甚至胜过亲兄弟）被 [宇智波佐助](宇智波佐助.md) 一伙（晓制服）抓走，四代雷影震怒、下两天死命令找人。心腹 **达鲁伊**（黑皮白发·慵懒·被他视作「左手」、有意培养成下代雷影）点破：除木叶外各村尾兽都被晓夺走，可联合对付晓；四代雷影遂让秘书 **麻布依**（黑皮白发美女）联络各村、提议三天后在 [铁之国](../世界/任务世界/火影忍者-忍界公敌.md) 召开**五影大会**——三个目的：① 谋忍村联合 ② 追责 ③ 向木叶泼脏水（「各村尾兽都出事、为何独你木叶无恙」·云隐村曾用类似手法向木叶勒索白眼）。他是晓在五影大会上的**主要袭杀目标之一**（火影 / 雷影 / 水影任死一个即可搅乱局势）。

## 关系
- **[奇拉比](奇拉比.md)**：义弟（八尾人柱力·情逾亲兄弟·为其被抓震怒、发起五影大会）。
- **达鲁伊 / 麻布依**：心腹（左手 / 准下代雷影、秘书）。
- **[苏晓](苏晓.md) / 晓组织**：夺其尾兽、抓其弟的死敌（晓欲在五影大会袭杀他）。
