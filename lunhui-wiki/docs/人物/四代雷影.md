# 四代雷影·艾

云隐村（雷之国）首领、**第四代雷影**——身高近两米的壮汉，白袍后绣「雷」字，光站着就有强大压迫感。脾气火爆（一拳拍碎办公桌）、却也识人。

## 弟被抓 · 召开五影大会（十五卷7–8章）

他的义弟 **[奇拉比](奇拉比.md)**（八尾人柱力·两人无血缘却情同手足、甚至胜过亲兄弟）被 [宇智波佐助](宇智波佐助.md) 一伙（晓制服）抓走，四代雷影震怒、下两天死命令找人。心腹 **达鲁伊**（黑皮白发·慵懒·被他视作「左手」、有意培养成下代雷影）点破：除木叶外各村尾兽都被晓夺走，可联合对付晓；四代雷影遂让秘书 **麻布依**（黑皮白发美女）联络各村、提议三天后在 [铁之国](../世界/任务世界/火影忍者-忍界公敌.md) 召开**五影大会**——三个目的：① 谋忍村联合 ② 追责 ③ 向木叶泼脏水（「各村尾兽都出事、为何独你木叶无恙」·云隐村曾用类似手法向木叶勒索白眼）。他是晓在五影大会上的**主要袭杀目标之一**（火影 / 雷影 / 水影任死一个即可搅乱局势）。

## 五影大会暴走 · 雷遁之铠（十五卷12–15章）

会上 [团藏](志村团藏.md) 拿奇拉比的死讯刺激他，他怒砸会议桌；得知 [佐助](宇智波佐助.md) 就在高塔一层后，**不顾五影会议、不顾铁之国脸面径直下塔追击**（云隐村「给脸不要脸」的处事风格）。他开「**雷遁之铠**」（雷属性查克拉裹身·双肩生能量犄角·大背头根根直立）、身体**活化 60~70%**（超 90% 细胞即崩溃）——雷属性赋予他**碾压级的速度 + 侧拳的强大切割力**：一拳把半仙人化的 [重吾](重吾.md) 打成两截、速度力量全面压制刚觉醒须佐能乎的佐助（佐助靠天照黑焰才勉强周旋）。苏晓评他是**五影中最好对付的一个**（实力虽能排前三、却太易怒狂傲）。护卫 **希**（感知型·雷盾「感激波」）+ **达鲁伊**（被 [阿姆](阿姆.md) 缠住）随行。**他是苏晓本想伺机袭杀的影之一。**

## 被苏晓斩杀（十五卷16–18章）

他与苏晓的对决印证「刀客克贴身肉搏型」——他雷属性的速度虽比苏晓快，但身体扛不住斩龙闪（唯小臂金属臂环能挡、却挡不住苏晓刁钻刀法），一记「沉怒雷斧」踢飞苏晓的代价是被斩断左小臂。苏晓用 [狂猎之裁决](../力量体系/装备道具.md) 趁灵魂感电定身打入**五发爆炸针 + 同时引爆**（500 伤 + 易损状态·胸口五个露肋骨的肉坑），再削肉放血、斩断双臂、趁其失血眩晕一刀斩首（他那条蓄势同归于尽的雷电腿停在了半空）。**先除护卫达鲁伊（被阿姆冻结后割喉·雷影候选人·世界之源 7.6%）**。雷影败得心服（堂堂正正被斩首·技不如人·脸上没有不甘）。苏晓还让带土把他**完整头颅送给团藏**（木叶读其记忆可七成把握灭云隐村）。
