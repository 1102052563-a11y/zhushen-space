# 神裂火织（魔法侧圣人·七刀七闪）

[魔法禁书目录](../世界/任务世界/魔法禁书目录.md) **魔法侧**的重要战力——**「圣人」**（能使用部分「神子」力量的人类·多为身体素质极强的魔法师、**以肉体力量为主战手段、魔法为辅**）。年仅 18 岁，外表却是 20 岁以上的丰满女人：单辫、白 T 扎起露出小腹后腰、牛仔裤左裤腿几乎全扯掉露出长腿。她**靠刀术战斗**，主武器是长达一米五以上的归鞘长刀「**七刀七闪**」。

- **登场（十三卷12章）**：她在第七学区购物街布下「闲人回避符文」清场，与主角 [上条当麻](上条当麻.md) 对峙——她正是昨晚那名魔法师的**同伴**，强到上条当麻（Lv.0 + 幻想杀手）也对付不了。
- **苏晓的任务**：[苏晓](苏晓.md) 加入科学侧后、亚雷斯塔交办的第一件事就是**击退 / 击杀她**（即此战目标·此前他已用刀斩杀过同样以刀术见长的对手·一场「刀客对刀客」之战将启）。

## 关系
- **魔法侧**：重要战力「圣人」。
- **[上条当麻](上条当麻.md)**：对峙的主角（追杀其友茵蒂克丝）。
- **[苏晓](苏晓.md)**：奉亚雷斯塔之命来击退她的契约者（刀术对决在即）。
