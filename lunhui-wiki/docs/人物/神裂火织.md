# 神裂火织（魔法侧圣人·七刀七闪）

[魔法禁书目录](../世界/任务世界/魔法禁书目录.md) **魔法侧**的重要战力——**「圣人」**（能使用部分「神子」力量的人类·多为身体素质极强的魔法师、**以肉体力量为主战手段、魔法为辅**）。年仅 18 岁，外表却是 20 岁以上的丰满女人：单辫、白 T 扎起露出小腹后腰、牛仔裤左裤腿几乎全扯掉露出长腿。她**靠刀术战斗**，主武器是长达一米五以上的归鞘长刀「**七刀七闪**」。

- **真相：她是来「保护」茵蒂克丝（十三卷12–13章）**：[上条当麻](上条当麻.md) 误以为她追杀茵蒂克丝、其实她和上条一样想**保护**茵蒂克丝。茵蒂克丝是英国清教修女（14 岁·**完全记忆能力**·脑中存十万三千本魔道书 =「禁术目录」），清教编造「脑容量有限、每年须清记忆否则死」的谎言来控制她——神裂火织等亲人每年都要承受被茵蒂克丝遗忘之痛。茵蒂克丝逃跑、她与魔法师**史提尔·马格努斯**追来想救她（清记忆），途中因上条的幻想杀手破坏茵蒂克丝的防御衣「移动教会」、她才误伤了茵蒂克丝。她对上条「砍伤茵蒂克丝、又一无所知」之事失态怒吼。

## 能力 · 圣人体质（使徒之眼实测）

`力 77 / 敏 83 / 体 79 / 智 83 / 魅 68 ｜ 法力 2530 ｜ 圣人体质`——属性全面、敏 / 智 83 极棘手：

- **圣人体质·裂神者**：被动 = 生命 +4000 / 法力 +1700 / 各抗 +20 / 免 26% 魔法伤害 / **免 100% 神圣伤害**；主动（1.67 秒）= 移速 +100% / 神经反射 +30% / 肉体强度 +47。这超强血统是她的核心。
- **七闪 Lv.39**：刀鞘内藏**七根附魔钢丝**（锋利 / 初级不损 / 强度增幅·90~120 米·几乎不可见）做中远程攻击 / 布结界。
- **唯闪 Lv.28**（一击必杀拔刀斩）：须先开圣人体质、命中造 **600 + 武器 ×0.7 + 力 ×3 + 敏 ×4** 伤害·**50% 为神圣伤害（无视 75% 防御）**——开圣人体质后快到苏晓视线 / 感知都捕捉不到。

## vs 苏晓 · 两败俱伤的刀客死战（十三卷13–17章）

两个都「用长刀 + 金属丝、战斗方式惊人相似」的刀客死战：苏晓用使徒之眼摸清她底细后正面强攻——[气息外放](../力量体系/技能.md) 起初判定失败、却在他一句话戳中她心防、她骂「卑鄙」**心灵波动**的瞬间判定成功（苏晓由此发现「让敌人心灵波动可短时削其意志力」）；配 [界断线](../力量体系/装备道具.md)（被她挑开钩锁）、白蜈蚣炼金炸弹（炸破她衣服走光·布布汪还躲垃圾桶后单反拍照当狗仔队分散她注意）、[炼金渡鸦](../力量体系/技能.md) 助攻。她 83 敏的速度一度快到苏晓动态视力跟不上、斩中苏晓肩 / 胸；苏晓近身用 [屠戮](../力量体系/装备道具.md) 短刀捅她胸口三刀、把她手掌钉地。最后她摆拔刀斩姿势放 **唯闪**——苏晓靠 [神灵余晖](../力量体系/装备道具.md) 的「圣灵赐福·次级无敌护盾」硬挡（斩击化冲击、撞进商场重伤）；她受伤后强行连放唯闪，苏晓以 **300 法力的「[连鞘斩·破空](../力量体系/技能.md)」+ 反击盾拼成的全身能量铠甲**对轰——两斩相撞、商场成危楼，**苏晓被斩断一条（握斩龙闪的）手臂、重伤**。苏晓因感知到魔法师（史提尔）将至，由 [布布汪](布布汪.md) 叼着断臂 + 驮他**撤退**（双方都受伤不轻、未分生死）。
