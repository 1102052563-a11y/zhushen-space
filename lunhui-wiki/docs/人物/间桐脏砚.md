# 间桐脏砚

[Fate/Zero 世界](../世界/任务世界/fate-zero.md) **三大魔术家族之一「间桐家」的族长**——一具腐朽发臭的老者皮囊，**已活 500 余年**，渴望得到大圣杯以求**永生**。

## 长生之法 ·「刻印虫」

他**舍弃了原本的肉体、把自身转化为魔术性质的「刻印虫」**——虫群组成肉体、寄宿在人体内（如把虫植入「儿子」[间桐雁夜](间桐雁夜.md) 体内折磨他）。这种续命方式会让灵魂逐渐枯萎：早年几十年换一具身体，如今几个月就得换一次。其原理与火影 [大蛇丸](大蛇丸.md) 的换壳长生类似、但「低档很多」（大蛇丸是灵魂转移容器、他是本体化虫）。

## 死亡（十一卷2–3章）

[苏晓](苏晓.md)（伪英灵）走出空间裂缝时，他先想以「带英灵夺圣杯」交涉、又想甩锅给雁夜，被苏晓**一刀斩首**（魔术师硬拼英灵没好下场）。但那只是皮囊——其本体是上百只**刻印虫**、化虫群欲逃，被苏晓 [反击盾](../力量体系/技能/index.md) 结界 + 青钢影围杀（青钢影是半能量刻印虫的克星）。但「活 500 年的老虫子」没那么好死——苏晓**逐进度斩杀**（2/5 → 4/5 → 99%）：追进间桐家古堡虫池、踩碎藏在 [间桐樱](间桐樱.md) 体内吐出的本体血红虫、清掉雁夜 / 樱体内的虫，最后 **8 颗特级炼金炸弹炸毁整座古堡**（连同几万刻印虫）。**杀间桐脏砚 99%·世界之源 +5.2%→5.3%·圣堂教会声望 −20**——他的「执念」（年轻时理想是「废绝一切的恶」、后堕落为求永生）已不够强，残余执念约一小时内消散。

## 关系

- **[间桐樱](间桐樱.md)**：被他丢进虫池折磨一年、还藏了本体虫。
- **[苏晓](苏晓.md)**：刚入世便将其斩草除根。
