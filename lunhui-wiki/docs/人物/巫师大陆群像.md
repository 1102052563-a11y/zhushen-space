# 巫师大陆群像（凯撒 · 悲伤之女 · 摩恩家族 · 老巫师修斯）

[巫师大陆·赫鲁](../世界/任务世界/巫师大陆-赫鲁.md) 中 [苏晓](苏晓.md) 接触的旧友、人偶、巫师与违规者。

## 尼古拉斯·凯撒（地精 · 裁决者 · 老友）

苏晓首个战争世界结识的盟友——奸诈、狡猾、猥琐于一体的**地精**，本是剧情人物、后投靠轮回乐园成**裁决者**（裁决战争时「经济学方面的失误」= 受贿被降级、现兼军需官）。死神世界六番队声望商店那「热情老头」就是他伪装的。本世界任务里他比苏晓更早进入、靠「特殊方法」爬成巫师会接待（负责收购超凡材料 / 分配物资），有一身钻空子的权限（能把声望兑换改成银币兑换、绕开兑换风险）——贪婪吝啬（请客匀半杯可疑饮品都肉疼）却在关键处极靠谱，与苏晓二八分账做魔药生意（怕布布汪·一见就求饶）。

## 悲伤之女（旧神人偶 · 中立）

不朽巫师 **曙光**（见下）为**世界之子「尼亚」**所造的人偶——褐裙、淡金麻花辫、戴圣银头饰、皮肤柔软却无体温。**靠钥匙上发条**（一次活动七天）、有颗可拆的**水晶心**（装上心会学撒谎 / 想变活人·故只给半颗诱惑她）。知无不答、轻柔无波却藏着对初代主人尼亚的悲伤（尼亚「为希望殉难」后她才成「悲伤之女」）。**能驱散「黑暗」**（把侵蚀苏晓的黑暗吸入自己的圣银头饰）、且不伤任何活物——是赫鲁城中立的关键人物。**结局（三十卷55章）**：淬圣银时主动摘下圣银头饰投入熔炉、最后捧着一颗心脏靠在尼亚雕像上**自愿长眠陪伴尼亚**（满足地闭眼·证明人偶从始至终都有「心」、根本不需要水晶心）。

## 摩恩家族（巫师会魔药调配者）

巫师会负责调配 [魔药](../力量体系/装备道具.md) 的家族（衔尾蛇徽印·住巫师会九层）——魔药生意被苏晓的冰魂魔药抢光、派人除凯撒反被苏晓截杀。

- **摩恩·迪（秘法巫师 · 反伤 + 换位置 · 逃）**：满脸伤疤、精神错乱的秘法巫师（清醒时极可靠·疯起来认外婆当求婚对象）——能力近乎无解：**攻击他所受伤害会放大数倍反噬攻击者**、还能与队友**瞬间换位置**（被布布汪标注「自带反伤甲的精神病巫师」）。被苏晓看穿换位置能力后没能近身斩杀、靠换位逃走。
- **摩恩·德尔（灰袍巫师 · 枯萎焰 · 殁）**：黑鳞竖瞳、总在挥手赶不存在的蚊子——「枯萎焰」幽绿火无判定地燃敌灵魂 + 肉体（对苏晓只成了「给灵魂取暖」的增益）。逃跑时踩中布布汪埋的磁爆猎手、被苏晓一刀斩首。
- **摩恩·埃尔莎（女巫师 · 扯灵魂）**：冷艳女巫师——双手合十「扯出敌人灵魂」配合德尔枯萎焰（对灵魂强度 113 的苏晓扯不动）。是三人组配合的核心、被苏晓刃道刀·青鬼牵制、德尔死后落单待斩。
- **摩恩·阿尔弗思**：摩恩家族最高掌权者（黑袍·下令隐蔽除掉凯撒）。

## 老巫师·修斯·阿奇德（秘法巫师 · 友方引路人）

基尔城巫师会分部里满身灰褐根系、坐轮椅的老巫师——少数还清醒、看清世界真相的人（友方单位）。给苏晓 [真视之契约](../力量体系/装备道具.md)（解除世界限制见真貌）+ 暗门传送阵直送赫鲁。盼苏晓「杀光『它们』、让被魔药迷惑的可怜人看清真相」。

## 三不朽巫师：曙光 / 古斯 / 劳伦特（曾经伟大的可怜人）

巫师会三位不朽巫师（六阶顶尖）——皆是**天生巫师 + 曾为旧神**，为让世界恢复正常反抗「它们」、失败后被囚旧神教堂、死亡被推迟（活在痛苦中）。老巫师评其「曾经伟大的可怜人」。**结局（三十卷55章）**：为给苏晓淬出能斩狂兽化守暗人的高纯度圣银——**劳伦特 / 曙光主动入熔炉焚身**（劳伦特吞下古斯的两腿一臂同焚·曙光临终赞「原来最无畏的是你，古斯」），古斯独臂离去**继续培养新希望**（怕苏晓失败世界彻底绝望·最艰难的抉择）。

- **劳伦特（摩恩·劳伦特 · 精神领袖 · 被白山羊变成一坨）**：上个时代的真主角——19 岁聚起曙光 / 古斯、分享旧神古秘典、耗 20 年创立**巫师会**、凑齐魔药配方传给子嗣（摩恩家族始祖）。率 7.6 万巫师讨伐支配者大败、被触须刺穿变成「一坨」（靠意志压分裂·新生命诞生时能挣脱七日）。**最惨却最 MVP**：识破幕后黑手、关键时刻站出来让尼亚得以出战。
- **古斯（曾旧神 · 倒吊 · 擅战）**：燃烧自身淬出圣银的莽汉（90% 重度烧伤·倒吊缓痛）——嘴臭心善，多次隐晦劝苏晓「别去送死、离开这」。听闻苏晓杀了支配者后癫狂大笑；受苏晓所托盯住曙光 / 劳伦特。
- **曙光（制造者 · 偏执）**：制造**悲伤之女**（见上）、培养世界之子尼亚的不朽巫师——偏执地坚信「只有强大个体能对付古神」。靠谱不卖关子（语速慢·苏晓嫌慢）；为尼亚的暗无天日童年深深愧疚；给苏晓圣银稀释溶液。

## 世界之子·尼亚（殁）

曙光找到的天生巫师女童（带「它们」的神性）——12 岁起藏巫师会地下地牢、18 岁成年后比任何巫师都强，带一把混圣银的剑「残月光」+ 人偶悲伤之女去讨伐支配者，**斩下支配者半个头颅而殉难**（绝技「勿忘我」·用后自尽以免古神血被夺）。后世传颂「希望之光·尼亚」，但她本人只是个「想吃甜面包、比较强的普通人」——从只知杀戮的野兽被曙光救赎成人、为救赎者的愿望而战。其技法记忆「勿忘我」被苏晓剥得。

## 古神：支配者·索托斯 / 阿撒托斯（吮吸世界的「它们」）

「它们」= 从崩溃的巨大世界流亡来此、由「概念神」退化成的**古神**（重伤·吮吸此世界自疗）。

- **支配者·索托斯（古神 · 殁）**：人形 + 黑触手下身、持「扭曲恶意」剑的古神——**支配**能力可无限制穿梭多重空间（古神殿内提速 300%）、攻击附空间切割伤·力 140 / 敏 152 极致机动但防御垫底（极强与极弱结合体）。被尼亚斩去半头颅（永久 −13% 生命）。把苏晓扯进古神殿想单挑、反被四打一斩杀（死前畏惧「灭法者……抱歉」）。
- **阿撒托斯（幕后古神 · 意识被扯进贪婪之章）**：真正的幕后黑手——本是「概念神」、重伤后**意识与兽性本体分裂**（兽性本体太强不接纳原意识）。其**意识**借**修斯家族 / 摩恩家族**（同出一脉·修斯传承其意识 + 明面恭维索托斯·摩恩当棋子）暗中操控一切：劳伦特创会 / 讨伐 / 培养尼亚都是为造「能战古神的容器」好让它夺回本体。被苏晓不动声色拼出真相、设宴**扯进贪婪之章**喂生灵屠夫魂灵解决（意识）；**兽性·阿撒托斯本体仍在**（怕意识 + 兽性重合成七阶）。

## 摩恩·东尼 / 老巫师修斯·阿奇德（修斯-摩恩家族）

- **摩恩·东尼（摩恩家族对外脸面 · 装晕保命）**：少数不疯的灰巫师、聪明如鬼——目睹苏晓斩族长阿尔弗思 + 扯走阿撒托斯意识后，**果断「自己把自己撞晕」装死保命**（「见过太多聪明人惨死、想活蠢一点」）·苏晓留他处理后续（料定他是下任族长）。
- **老巫师·修斯·阿奇德（寄体 · 殁 · 见上「友方引路人」）**：阿撒托斯意识的上一届寄体（双腿被悲伤之女砸断·坐轮椅）——被苏晓用「能量性剧毒」当酒灌下毒杀前诈出全部真相（修斯 / 摩恩家族同源、阿撒托斯意识 / 兽性分裂、寄体机制）。

## 女巫师·可怜虫（巫师会十层守门 · 殁）

被挖去双眼、背黑木长枪、满口金牙的强大女巫师（守巫师会十层以上·摩恩·迪之流只是打杂）——自称「连懦夫都不如的可怜虫」、看清世界真相却无力。求苏晓「送走老巫师时给他瓶好酒」（指修斯·阿奇德·她疑似与之有旧）、求死般任苏晓出手。

## 违规者群像（黑哥 / 毒姬 等 · 殁）

开局围攻刚传送来的苏晓的七名违规者（**追一名战斗天使踪迹路过**）——**黑哥**（秃头莽夫·六阶中游·被苏晓徒手 + 长枪压制后斩首·掉钻石荣誉勋章）、**毒姬**（满脸疤的丰满女违规者·脸上疤是某猎杀者留的·被一刀斩开胸膛 + 踩头钉杀）等。本世界违规者 500+、不分乐园（杀非轮回乐园违规者无奖励、除非掉猩红卡）。

## 违规者联盟（艾森拉大本营 · 苏晓的对手）

全开放原生世界是违规者老巢——这伙违规者**与白山羊合作破坏轮回 / 圣光乐园的时空之力交易**、把支配者卖了当炮灰、还造谣「苏晓霸占赫鲁城」清场孤立他。盘踞在**奇迹之地·艾森拉**（古神留的阳光净土·违规者大本营）。

- **斯芬克（违规者联盟真大佬 · 圣域乐园出身 · 殁）**：白西装、蓄短须、成熟绅士的老阴哔——苏晓眼里「与他对等的超级老阴哔」（互评白山羊「真够蠢」）；为活可以**出卖一切**（牺牲 63 违规者挡阿波罗、把黑山羊·莎布也卖了把苏晓扯出艾森拉）。他与 J 先生是旧识。手下武器都靠圣灵级 [星陨熔炉](../力量体系/消耗品材料.md) 二次熔锻；杀手锏「**暗狱之门**」（以青铜门为媒介具现·剥离活物的「生」= 即死·被苏晓的灵魂之刃免疫）。最终被苏晓 A 计划掀桌打乱、冰封后斩杀（掉 ★★★★★★ 猩红卡 + 星陨熔炉·此事主谋）。
    - **康妮（包子脸熊孩子 · 刀术宗师 · 亡魂 · 殁）**：斯芬克唯一不舍得牺牲的心腹——「没有依托就无法活下去的亡魂」、双黑短刀（麻痹 + 介于史诗 / 圣灵级·星陨熔炉锻）、敏 132 凭空斩、纯黑眼瞳。瞬秒墨镜男；与苏晓单挑——力 / 敏更高却因**没在众生之地修行过**技法被碾压（同为刀术宗师实战差太多）、被苏晓断臂后斩杀。
    - **月夫人（衔尾蛇 · 慢性子 · 被灭口）**：右臂白纹 + 血纹（用苏晓的血勾勒）的心腹——能用磁吸把苏晓拖出艾森拉（拉 6 秒·强度 8500+）。危难关头怂了、被斯芬克命狙击手一枪爆头灭口（「怂了就可能背叛」）。
    - **死闷男（狙击手 · 殁）**：斯芬克称「搭档」、能半年不说一字的沉默狙击手——三米多狙击枪（圣灵级·星陨熔炉锻·一枪轰苏晓 34% 生命·要害一枪倒）。被阿姆缠住后枪变烧火棍、必死时果断自爆销毁武器（连名字代号都没有）。
- **墨镜男（临时首领 · 殁）**：率 105 违规者空间置换伏击苏晓的违规者——因「斯芬克来自圣域乐园不信任他」自任临时首领。伏击失败、违规者死伤近半；后被斯芬克的熊孩子瞬秒（「只有三个人能喝茶」）。
- **茉莉（战神奶 · 战斗奶妈 · 被收买成内线）**：红法袍紫眼影、战斗系 + 治疗系混合的女违规者（原是用花语作代号的「小白花」奶妈）——被苏晓 [血羽](../力量体系/装备道具.md) 天克（治疗反成真伤）逼得**只能偷偷奶苏晓**（开发「近战奶」法杖打人回血·秀到苏晓），还得演「女战神」掩饰。苏晓用「假心之毒」（其实是压缩晶体层·最难解的「心之毒」）收买她成内线（喂违规者联盟情报）。
- **魅幻（女刺客 · 雇佣）**：右臂布满上千道击杀刻纹的顶级刺客——能**远程锁定刺杀**（赤舞之牙 / 魔化短刃等史诗 / 传说级匕首·一击清空匕首耐久·需媒介锁定）。受雇刺杀苏晓两次（均被远古秘药 / 晶体层化解成轻伤）。
- **豪诺 / 马刀壮汉（违规者主力）**：**豪诺**（五短身材 + 络腮胡 + 房屋大巨锤·在场最强者之一）；**马刀壮汉**（锯齿斩马刀·力量强 / 速度慢的战斗狂·与苏晓擦身对斩后只想再打几场·这种人成长起来不得了）。

## 红色小丑·J先生（不死焰魔 · 轮回乐园援军）

苏晓 [幽鬼大陆](../世界/任务世界/幽鬼大陆-五阶考核.md) 时的肃清目标——曾组「绅士联盟」（全违规者）的红色小丑，被苏晓斩首时**与体内寄生的炎魔老哥强行融合**舍弃身体所有权而新生（失装备 / 编号 / 90% 能力·只剩身体素质 + 「炼狱颚」吞噬能力），后摆脱违规者身份成观察期契约者。本世界被轮回乐园**强制征召**来援助苏晓（能破青铜门·苏晓握「行为异常即强制处决」特权故不怕他耍花招）。善布局的老阴哔、招牌假笑（对苏晓不敢假笑·被斩过的剧痛记忆犹新）；能力倾向「焰 / 爆炸」+ 扯断手指化「血色小人·老鼠」当探子。A 计划里**整个吞走青铜门**瞬间得手、随即返园。

## 露露西（圣光乐园 · 冰系 · 求合作被拒）

途经艾森拉来赫鲁求与苏晓合作的圣光乐园冰系妹子——谦和柔弱、带五女小队（含顶级大奶妈）。打白山羊败、想用情报 / 治疗换合作，却被巴哈一通「情报全过时」怼到怀疑人生；苏晓因**时空之力交易的内奸未明**（可能是圣光乐园方）直接拒绝。
