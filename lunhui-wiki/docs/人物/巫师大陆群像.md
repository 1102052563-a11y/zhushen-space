## 旧友 / 援军（跨世界复用角色）

- **尼古拉斯·凯撒（地精 · 裁决者 · 老友）**：奸诈狡猾猥琐于一体的**地精裁决者**（兼军需官）——本世界比苏晓更早进入、爬成巫师会接待，与苏晓二八分账做魔药生意（怕布布汪·一见就求饶）；贪婪吝啬却关键处极靠谱。同一角色在多个世界返场（详见 [炎灵星·凯撒](炎灵星/凯撒.md)）。
- **[红色小丑·J 先生](红色小丑.md)（不死焰魔 · 轮回乐园援军）**：[幽鬼大陆](../世界/任务世界/幽鬼大陆-五阶考核.md) 时的旧敌——本世界被轮回乐园强制征召来援（能破青铜门·A 计划里整个吞走青铜门瞬得手·随即返园）；[斯芬克](巫师大陆/斯芬克.md) 的旧识。详见 [红色小丑专页](红色小丑.md)。
- **[夏](夏.md)（15053 号职工者）**：被困本世界某密室（白山羊 / 触手）、苏晓此行营救目标——**已救出**（当侦查 / 后勤·目睹苏晓单挑古神世界观崩塌·随通关返园）。

## 其他违规者 / 龙套

- **违规者群像（黑哥 / 毒姬 · 殁）**：开局围攻刚传送来的苏晓的七名违规者（追一名战斗天使踪迹路过）——黑哥（秃头莽夫·被徒手 + 长枪压制斩首）、毒姬（满脸疤女违规者·被一刀斩胸 + 踩头钉杀）。本世界违规者 500+。
- **违规者联盟成员**：**墨镜男**（临时首领·率 105 违规者空间置换伏击失败·后被康妮瞬秒）；**月夫人**（衔尾蛇·磁吸拖苏晓出艾森拉·怂了被斯芬克命狙击手爆头灭口）；**死闷男**（沉默狙击手·三米狙击枪·必死时自爆销毁武器）；**魅幻**（女刺客·远程锁定刺杀·受雇刺杀苏晓两次未成）；**豪诺 / 马刀壮汉**（违规者主力·巨锤 / 锯齿斩马刀）。
- **女巫师·可怜虫（巫师会十层守门 · 殁）**：被挖去双眼、满口金牙的强大女巫师——看清真相却无力、求死般任苏晓出手（求苏晓「送走老巫师时给他瓶好酒」）。
- **摩恩·东尼（摩恩家族对外脸面 · 装晕保命）**：少数不疯的灰巫师、聪明如鬼——目睹苏晓斩族长 + 扯走阿撒托斯意识后果断「自己把自己撞晕」装死保命（苏晓料定他是下任族长·留他处理后续）。
