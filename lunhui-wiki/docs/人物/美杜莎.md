# 美杜莎（Rider · 女妖）

[Fate/stay night 第五次圣杯战争](../世界/任务世界/fate-第五次圣杯战争.md) 的 **Rider（骑兵）**——[间桐樱](间桐樱.md) 的从者，慵懒邪魅的女性英灵。关键在于她是**「女妖」形态**而非女神形态：女神形态的美杜莎战斗力平平，被雅典娜诅咒成女妖后战力直线飙升（女神形态战力若为 5、女妖形态至少 60+）。

- **石化之瞳**：女妖形态拥有近乎能秒杀所有英灵的**石化之瞳**——单体威力不弱于 [金闪闪](吉尔伽美什.md) 的乖离剑，互相对视一眼神都要陨落（半神的金闪闪也扛不住）。正是这一点让旧识 [假面](假面.md) 看中、来抱间桐樱的大腿。
- **未与苏晓交手**：[斯坦](斯坦.md) 一死、护徒老魔术师 + 假面把间桐樱带离冬木市保命，她也随之退场——苏晓始终没寻获这对御主与从者。

## 关系
- **[间桐樱](间桐樱.md)**：御主。
- **[假面](假面.md)**：看中其石化之瞳、来抱大腿的旧识猫女。
- **[苏晓](苏晓.md)**：未交手（被带离冬木市）。

## 出处
- 二十二卷37–38章（第五次圣杯战争 Rider·女妖形态·石化之瞳·随间桐樱被假面带离·苏晓未寻获）。
