# 梦魇之神·尼加狄（黄昏之神 · 古神 · 泯光神教幕后 · 殁）

海上传说「黄昏的余光、夜晚的君王、绝望的起源」的真身——3.5 米外骨骼古神（智力 156 破壁障·为用夺来的中立女神武器「光之枪」不惜把右臂改造成全骨）。

- **泯光神教**（教义「抹除太阳、永夜降临」·盘根错节于各王国 / 海盗·想以此世界为起点打通虚空传教）的幕后古神，藏在无风孤岛、靠**梦境操控**（泯光领域剥感知 / 梦境化身神出鬼没 / 把人拖进梦境世界稳杀）操控一切。
- **结局（殁）**：被 [苏晓](../苏晓.md) 循血滴 + 鲜血印记找上——**刀术修心使苏晓内心强到拖不进梦境**（古神核心能力卡壳），反被「一颗真实石子破梦境化身」+ 心·魂·刃 + 血之兽斩杀（得史诗级神灵骨 / 圣灵级 [阿斯萨耶](../../力量体系/装备道具.md)·[神裁](../../力量体系/装备道具.md) 升级）。暴鼠曾透露它与月神同神源系。

## 关系

- [苏晓](../苏晓.md)：刀术修心克其梦境、斩杀它的猎手。

## 出处
