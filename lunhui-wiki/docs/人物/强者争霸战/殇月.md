# 殇月（羽族 · 装死求生 · 被苏晓放过）

黑羽衣、蓝眼影、不苟言笑的女性羽族（非最年轻一代·才有资格参战）——战斗素养高（羽族「艾文艾学院」教育·懂建掩体 / 防大威力远程），却带不动一盘散沙的杂牌队友。

- 被 [M·暴君](../../力量体系/装备道具/index.md) 清场时靠羽衣硬挡一枪、**装死求生**（兄长传授的战场绝技）——[苏晓](../苏晓.md) 识破却因**羽族 + 恶魔族的庇护**（单羽族身份必死）放她一马（只割一缕发当凭证）。死亡逼近时被「活着真美好」治愈了不苟言笑的性子。其兄长 = [泉羽](泉羽.md)（夺冠五大热门）。

## 关系

- [苏晓](../苏晓.md)：识破她装死、却放她一马的人。
- [泉羽](泉羽.md)：她的嫡亲兄长。
