# 蒙德（恶魔族 · 纯血战斗恶魔 · 越挨揍越强 · 第三名）

锯断双犄角 + 寸发的恶魔族——曾冲到羽族后方被老羽族拽着犄角 / 长发毒打四小时（黑历史·故锯角剪发）；**越挨揍越强、濒死最恐怖**（同辈防御最强·伤越重力越强）。

- [莉莉姆](../莉莉姆.md) 的「冤家」（互相偷袭 + 告状的循环）。第四天被虚空队 + [无名圣徒](无名圣徒.md) 四打一围攻（[休格](休格.md) 灵魂黑焰占六成伤），宁站着死不跪着生、把霸者勋章吞进肚也不交——[苏晓](../苏晓.md) 雪中送炭临时结盟（蒙德对付休格）。
- **结局（第三名）**：吞勋章卡肚被苏晓开刀取出、反杀了有血仇的 [泉羽](泉羽.md) 后提交勋章取**末位**得起源石（恶魔族老家伙们需次位黑枫树枝干续命·他乐得拿起源石）。与苏晓结下「雪中送炭」交情。

## 关系

- [苏晓](../苏晓.md)：雪中送炭、临时结盟的交情。
- [泉羽](泉羽.md)：被他反杀的血仇对象。
- [休格](休格.md)：围攻他的黑焰师。
- [莉莉姆](../莉莉姆.md)：互相偷袭告状的「冤家」。

## 出处
