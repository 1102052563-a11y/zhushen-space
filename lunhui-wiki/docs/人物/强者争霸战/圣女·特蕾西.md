# 圣女·特蕾西（亡者 · 灵魂割离的战争武器 · 解脱）

金发白裙、脊背刺入 60+ 根黑锁链（杀人无数被囚到极致）——翡廷神权帝国灭亡前把她改造成战争武器。

- **灵魂上半被劈成两半**（左侧保理智 / 右侧专战斗·可被咒语操控）、念对咒语降伏念反则暴走（海胆瞳 / 咧到耳根的尖牙大嘴）。看似柔弱胆小、侦测全是问号的危险类型；其实并不可怜（至亲被术士烧死后她意识清醒地屠了半个世界泄愤、还吃了唯一好友）。
- [苏晓](../苏晓.md) 用暗影石解开她的锁链、她**自愿投入黑雾化灰解脱**（留下白脂蜡 / 秘迹宝石 / 处决之刃）。

## 关系

- [苏晓](../苏晓.md)：解开她锁链、送她解脱的人。
- [痛苦女王·安娜](痛苦女王·安娜.md)：看守她的死亡屋主人。

- 三十二卷（[强者争霸战](../../世界/任务世界/强者争霸战-暮色之岛.md)）9–30 章：被降伏→自愿化灰解脱。
