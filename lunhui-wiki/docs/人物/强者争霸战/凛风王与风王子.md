# 凛风王 / 风王子（奥术永恒星主和派 · 父子）

- **凛风王**：正派威严——**千年前对抗灭法者的主力之一**、主和派；当众斥 [瑟菲莉娅](../瑟菲莉娅.md)「拿弟子 [狄琳](狄琳.md) 做人体实验、法之魂早被灭法者踩灭」。（看苏晓「能量阻断」把小胡子电成木棍时直拍额头「千年前我也这样」。）
- **风王子**：睡衣家里蹲——与 [苏晓](../苏晓.md) / [休格](休格.md) 有交情，曾托休格转告苏晓「别去恶魔古堡」，是苏晓的盟友 / 中立方。详见其 [专页](../风王子.md)。

## 关系

- [瑟菲莉娅](../瑟菲莉娅.md)：被凛风王当众斥责的主战派贤者。
- [苏晓](../苏晓.md) / [休格](休格.md)：风王子的交情方。

## 出处

- 三十二卷（[强者争霸战](../../世界/任务世界/强者争霸战-暮色之岛.md)）58–65 章：第四轮角斗场观众席·奥术永恒星主和派。
