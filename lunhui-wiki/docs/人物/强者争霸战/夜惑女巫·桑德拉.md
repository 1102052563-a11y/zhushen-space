# 夜惑女巫·尤·西斯·桑德拉（超脱·原生世界 · 女巫眼强盗 · 殁）

榜上只标「夜女巫」（真名涉灵魂·只至亲知）——来自夜惑女巫统治、排斥虚空与乐园的世界。

- **26 只「女巫眼」**的「女巫眼强盗」（女巫眼 = 存特殊能量的器官·常人仅 3 只·她全靠抢）；靠痛苦取乐、改造身体成夜战怪物（角质护臂 / 心脏掠夺 / 诅咒），盯上过 [蒙德](蒙德.md)（俩人不在一个频道）。保命绝技「**夜魔皮**」= 每用一次耗一只女巫眼的替死术。
- **结局（殁）**：第五轮被 [苏晓](../苏晓.md) 团队半包围围猎——[阿姆](../阿姆.md) 无解坦度死缠 + 巴哈 三连击放血 + 苏晓刀芒磨耗，夜魔皮被耗尽后斩首（×3 勋章归苏晓）。

## 关系

- [苏晓](../苏晓.md)：团队围猎、耗尽其夜魔皮斩杀她的人。

## 出处

- 三十二卷（[强者争霸战](../../世界/任务世界/强者争霸战-暮色之岛.md)）58–73 章：危险度第三→第五轮被围猎斩首。
