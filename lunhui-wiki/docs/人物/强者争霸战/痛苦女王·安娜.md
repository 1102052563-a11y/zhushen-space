# 痛苦女王·安娜（死亡屋主人 · 病娇 · 中立公正）

黑裙苍白、不时干咳（[蒙德](蒙德.md) 反复「你快病死了」她也只无奈微笑）的死亡屋主人——死亡屋是 [爱丽丝恶魔古堡](../../世界/任务世界/恶魔古堡游戏.md) 的高配版，内有诸多被囚的「亡者」。

- 绝对强大却**尽力维系公平**（危险的是死亡屋本身、非她）；并非自愿当主人（「死亡屋需有人看守、或是你或是我」）。
- 女灭法者 **格林·吉莉安** 送过她指环（留言「祝你晚点病死」·待她比待爱丽丝友善）。曾当 [苏晓](../苏晓.md) 面扯掉嚣张的守雾人大半躯干教做人。

## 关系

- [苏晓](../苏晓.md)：闯死亡屋、与她中立相处的契约者。
- [圣女·特蕾西](圣女·特蕾西.md) 等亡者：她看守的囚徒。
