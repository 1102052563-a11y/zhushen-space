# 尼塔（吟语族 · 伪名「瓦戈」· 诱饵 / 墙头草）

脸有疤的吟语族——**能拟态任何人的模样 / 体型 / 气息**（甚至衣着）。

- 开场伪装成虚空之树接引人「瓦戈」拉满仇恨、骗走奖励 [封之箱](../../力量体系/装备道具.md)；被 [苏晓](../苏晓.md) 识破击杀其伴生化身后归还封之箱、求合作当**诱饵**（把 30+ 参战者引到荒草地任 M·暴君清场）。
- **吟语族**：古老中小种族·因拟态天赋被虚空当「珍稀商品」禁买卖、千年前投靠奥术永恒星（年年纳贡）；处世宗旨「成功率没 20% 不赌命」。

## 关系

- [苏晓](../苏晓.md)：识破他、收他当诱饵的人。

## 出处

- 三十二卷（[强者争霸战](../../世界/任务世界/强者争霸战-暮色之岛.md)）1–30 章：伪装瓦戈骗封之箱→被收作诱饵。
