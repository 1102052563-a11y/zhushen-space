# 鼠大人 / 暴鼠（第七争霸区接引人 · 中立 · 嗜酒）

鼠头人身、扛能量霰弹枪的虚空强者——第七争霸区的接引人（虚空之树征召）。

- [布布汪](../布布汪.md) 旧识（虚空角斗场时受轮回乐园委托硬挡奥术永恒星法师贤者 [瑟菲莉娅](../瑟菲莉娅.md)、帮苏晓挡住她）。
- **嗜酒如命**——被布布汪用一瓶海贼世界「玛歌亚加」（凭酒品硬顶到史诗级的酒）换得隐晦情报（「天气太热 / 海」·暗示后续与海有关）。第四轮豁出引导人身份 + 星空座大人物撑腰，安排 [狄琳](狄琳.md) 第二场对上苏晓。

## 关系

- [布布汪](../布布汪.md)：拿酒换情报的旧识。
- [苏晓](../苏晓.md)：暗中给便利的接引人。

## 出处

- 三十二卷（[强者争霸战](../../世界/任务世界/强者争霸战-暮色之岛.md)）：接引 + 透露海线索 + 安排狄琳对局。
