# 双子星（死亡乐园 · 六阶最强 · 夺冠大热门 · 殁）

两名容貌全同、剃光头、头布满条形编码、眉心「￡」死亡印记的少年——**死亡乐园六阶最强者、六阶乐园竞技场第三位**、本届夺冠大热门（说话一人接一人、不分你我·老阴哔）。

- 开局与 [苏晓](../苏晓.md) **瓜分暮色之岛**（左 / 右各清场·不在第一阶段内耗），把世界之子 [阿雷克](阿雷克.md) 当「见面礼」让给苏晓。第四轮积分第一。
- **结局（殁）**：融合成「双生形态」（满头条码组倒五芒星·近战克制极强·招式双生·陨 / 混沌〔秒坦 80–95%〕/ 龙哮）——被苏晓硬抗两次「双生·混沌」+ 青钢影侵体后一刀贯首；临死激活心脏炸弹（黑紫丝线·触之即蒸发），苏晓靠猎魔时刻 + 龙影闪穿空间躲过。掉 [猩红宝箱](../../力量体系/装备道具.md)（可跳六阶乐园竞技场第 2 名）。

## 关系

- [苏晓](../苏晓.md)：从瓜分海岛到最终斩杀他的对手。
- [阿雷克](阿雷克.md)：被他当见面礼让给苏晓的世界之子。
