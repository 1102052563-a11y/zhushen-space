# 黑海女帝·阿加蒂（三大传说海盗 · 最富 · 血鸦 · 精明野心家）

竟是女人（嫌「黑海女帝」诗人吵故称黑海王·出名之战在黑夜的海上）——维京之海三大传说海盗里**最富**（走海运 / 贩海兽·守规矩〔不滥杀 / 不大量贩奴〕·疑暗控某王国）。

- **血鸦 / 红鸦症**：在人体种「种子」（宿主活不过三年、发作化血鸦）——[苏晓](../苏晓.md) 杀过她一只血鸦套情报。
- 精明 + 野心拉满：算到苏晓难惹便「这事没发生过」灭口目击者、白送古神右臂（图谋控泯光神教称霸维京之海）、谈判想坑苏晓换 [怒海之魂炼](../../力量体系/装备道具/index.md) 却失算。二副 **疯海牛·图尔特**（干瘦老头·曾与海兽角力）。

## 关系

- [苏晓](../苏晓.md)：算计又忌惮的谈判对手。
- [红胡子·伊格纳缇](红胡子·伊格纳缇.md) / [骨羊·凡俄西](骨羊·凡俄西.md)：另两大传说海盗。

## 出处

- 三十二卷（[强者争霸战](../../世界/任务世界/强者争霸战-暮色之岛.md)）31–57 章：血鸦套情报→龟岛三海盗会面→谈判失算。
