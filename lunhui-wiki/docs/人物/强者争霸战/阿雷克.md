# 阿雷克（陨灭星世界之子 · 被古神侵蚀 · 殁）

陨灭星（古神统治的超脱·原生世界）的世界之子——**被古神力量侵蚀**（无神源·黑触手 + 胸口蠕虫独眼·变 3 米怪物·古神所赐强铠 + 魂之咆哮 / 死亡领域）。

- 以「能与神灵直面交谈」为荣、早已彻底放弃反抗（那世界的人「双腿」早被打断·无劳伦特式的反抗者）。被 [双子星](双子星.md) 当见面礼让给 [苏晓](../苏晓.md)、被苏晓**心·魂·刃**破其最强一拳后斩杀（掠天惊澜任务 +13% → 96%·因不在陨灭星收益打折）。其不肯彻底「神灵化」是身为人最后的不屈。

## 关系

- [苏晓](../苏晓.md)：斩他的猎杀者。
- [双子星](双子星.md)：把他当见面礼让出的死亡乐园强者。

## 出处

- 三十二卷（[强者争霸战](../../世界/任务世界/强者争霸战-暮色之岛.md)）1–8 章：被当见面礼→苏晓心·魂·刃斩杀。
