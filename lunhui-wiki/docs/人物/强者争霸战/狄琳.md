# 狄琳（奥术永恒星施法者 · 瑟菲莉娅弟子 · 黎元素克灭法者 · 殁）

红法袍、满耳钉、中性短发的女施法者——**法师贤者 [瑟菲莉娅](../瑟菲莉娅.md) 的弟子**（异元素气息·与瑟菲莉娅同源·苏晓青钢影对她活跃）。

- 童年被「打着为你好之名虐待」的导师折磨成地狱（恨的是导师 + 灭法之影、非 [苏晓](../苏晓.md) 本人），却坦承「与你无冤无仇、只是立场不同」「杀出来的强大可敬」。
- **黎元素（针对灭法者）**：瑟菲莉娅把土 + 光元素剥离重铸的「黎」异元素（金属性 / 锋利 / 接触生命力起魔导反应放高温·且青钢影不主动噬灭未经魔力干涉的自然元素）——她整个童年的苦难就为得此能力克灭法者，却在环瘠岛被苏晓**两个照面就崩碎了黎元素武器**（青钢影一加力就崩）、信念心态一起血崩、果断撤。
- **结局（殁）**：第四轮被安排对上苏晓——黎元素盾 / 螺旋剑 / 黎羽翼一一被崩碎、连吸青钢影的特制耳钉也扛不住，**一分钟被钉死**。临死「这是你最后送我的礼物」（对瑟菲莉娅）。斩狄琳后苏晓与奥术永恒星**正式敌对**。

## 关系

- [瑟菲莉娅](../瑟菲莉娅.md)：把她当人体实验、逼她克灭法者的导师。
- [苏晓](../苏晓.md)：两招破其黎元素、一分钟钉死她的人。
