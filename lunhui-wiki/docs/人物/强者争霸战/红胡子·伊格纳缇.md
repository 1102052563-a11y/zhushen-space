# 红胡子·伊格纳缇（三大传说海盗 · 最强 · 瘸腿枭雄）

黑帽黑大衣、金属锥假腿、络腮胡（杀人血溅胡子得名）的瘸子——三大传说海盗里**总体战力最强**（主黑产·武器 / 野鸳），凶残却胸襟宽广、唯一敢怼 [骨羊](骨羊·凡俄西.md)。

- 因恐怖岛死伤近百部下 + 3 心腹被策反叛逃而暴怒；与 [苏晓](../苏晓.md) 谈判时试出他「不是来抢地盘的」便踩台阶、还以家乡朗姆酒「只与朋友共饮」、给了恐怖岛海图（[怒海之魂炼](../../力量体系/装备道具.md)·15 年仅一份）。

## 关系

- [苏晓](../苏晓.md)：从对峙到「朋友共饮」的枭雄。
- [阿加蒂](阿加蒂.md) / [骨羊·凡俄西](骨羊·凡俄西.md)：另两大传说海盗。
