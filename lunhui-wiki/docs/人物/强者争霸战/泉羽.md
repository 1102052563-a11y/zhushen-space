# 泉羽（羽族 · 殇月兄长 · 夺冠五大热门 · 殁）

透明丝线 + 黑羽的「羽界」束缚术——[殇月](殇月.md) 的嫡亲兄长、本届夺冠五大热门之一（自称「比殇月强 16.752 倍」·父亲顶压力把兄妹俩都送来历练·「羽族不需要温室的花朵」）。第二个抵达恐怖岛级别的强者。

- 与 [蒙德](蒙德.md) 有血仇（挚友塔努一个半月前被蒙德当面扯下头颅），强忍私怨与 [休格](休格.md) 结「虚空队」参与围攻蒙德。
- **结局（殁）**：血仇了结于此——重伤的铁憨憨蒙德反杀了泉羽。

## 关系

- [殇月](殇月.md)：他的嫡亲妹妹。
- [蒙德](蒙德.md)：有血仇、最终反杀他的恶魔族。
- [休格](休格.md)：结「虚空队」的搭档。

## 出处

- 三十二卷（[强者争霸战](../../世界/任务世界/强者争霸战-暮色之岛.md)）49–92 章：围攻蒙德→被蒙德反杀。
