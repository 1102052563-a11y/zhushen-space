# 骨羊·凡俄西（三大传说海盗 · 最疯 · 单挑最强）

彻头彻尾的疯子、喜怒无常滥杀成性、无心腹——**单挑无人能敌**（[红胡子](红胡子·伊格纳缇.md) / [阿加蒂](阿加蒂.md) 都不是对手）。

- 主营奴隶抓捕（维京之海八成奴隶经其手·仇家无数）；听闻三海盗会面也要掺一脚（行事无法预测）。
