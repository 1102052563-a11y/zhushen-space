# 休格（随缘施法者 · 奥术永恒星 · 修灵魂黑焰师 · 次位）

黑金法袍、慵懒、说话不自信的年轻施法者——**认同灭法之影的理念**（不维持元素平衡虚空早晚完）、跟 [苏晓](../苏晓.md)「没仇、连面都没见过」。

- 看似火系、实为**修行灵魂**——核心是能灼烧灵魂的「黑焰」（前期弱 / 后期越来越强·克纯肉体的 [蒙德](蒙德.md)）；人品奇好、与 [泉羽](泉羽.md) 结「虚空队」。多次替 [狄琳](狄琳.md) 挡场又随缘放走她、替 [风王子](../风王子.md) 转告苏晓「近期别去恶魔古堡」。
- **结局（次位·逃）**：被苏晓斩得死去活来、想魂仿先代灭法者反引来刀魔夺刀，见苏晓**吃灵魂石**当场认怂、交勋章取次位逃走。回头把那把**先代灭法者的刀卖给 [瑟菲莉娅](../瑟菲莉娅.md)**（坑她·这刀会招来苏晓和刀魔）。

## 关系

- [苏晓](../苏晓.md)：理念相同、多次两不相欠的对手。
- [泉羽](泉羽.md)：结「虚空队」的搭档。
- [瑟菲莉娅](../瑟菲莉娅.md)：被他坑卖先代灭法者刀的贤者。
