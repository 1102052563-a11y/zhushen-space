# 无名圣徒（天启乐园 · 科技流土豪矿工 · 第五名）

腰挂矿铲、十字枪可变高科技矿铲的「富到流油的矿工」——天启乐园主心骨、[苏晓](../苏晓.md) 的跨世界旧识（[强者争霸战](../../世界/任务世界/强者争霸战-暮色之岛.md) + 竞技场 + [陨灭之界](../../世界/任务世界/陨灭之界-五方混战.md) 多次相遇）。

- **装备 / 能力**：圣灵级动能机甲 +12 + 圣灵级十字枪 +14 镶三宝石 + 核心「**心灵意感**」（把机甲活化发挥 300% 威力）+ 海量一次性道具；圣灵级穿甲弹曾轰掉苏晓近半血 + 斩断其左小臂（讨厌技法类敌人·被苏晓夺武器后只得交勋章换回）。
- **结局（第五名）**：靠「双层契约」坑死圣光乐园折扇妹**阿芙妮**、邀苏晓合作被婉拒、以第五名淘汰；武器 [天陨之劫 +14](../../力量体系/装备道具.md) 被苏晓夺下公证。战后稳坐**六阶乐园竞技场首位**（苏晓拿双子星猩红宝箱来挑战被他直接弃权拉黑）。**陨灭之界**再遇：抛矿石「买命」拖异种、稳局手段漂亮。

## 关系

- [苏晓](../苏晓.md)：跨世界多次相遇、被夺武器又婉拒合作的旧识。
- [双子星](双子星.md)：其猩红宝箱成了苏晓挑战他竞技场首位的「挑战书」。
