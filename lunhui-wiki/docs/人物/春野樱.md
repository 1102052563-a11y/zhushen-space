# 春野樱

[火影忍者世界](../世界/任务世界/火影忍者.md) 木叶忍者——**医疗忍者**，粉发女忍（[纲手](纲手.md) 的弟子辈）。

- **登场（九卷38章）**：在木叶火影办公室在场，看到 [自来也](自来也.md) 胸前那道半米多长、深可及骨的伤口时有些骇然——作为医疗忍者她深知这种伤势之重，自来也能活下来全凭生命力顽强。该伤口正是 [苏晓](苏晓.md) 以「[寸断](../力量体系/技能.md)」所留。
