# 春野樱

[火影忍者世界](../世界/任务世界/火影忍者.md) 木叶忍者——**医疗忍者**，粉发女忍（[纲手](纲手.md) 的弟子辈）。

- **登场（九卷38章）**：在木叶火影办公室在场，看到 [自来也](自来也.md) 胸前那道半米多长、深可及骨的伤口时有些骇然——作为医疗忍者她深知这种伤势之重，自来也能活下来全凭生命力顽强。该伤口正是 [苏晓](苏晓.md) 以「[寸断](../力量体系/技能.md)」所留。

- **追击与留守对战蝎（九卷43–44章）**：作为卡卡西小队成员，以**挥拳姿势 + 黑色手套**第一个破墙冲入河之国山洞（怪力体术）。鸣人 / 卡卡西追迪达拉离去后，她与 [千代](千代.md) 留下对战独自断后赴死的 [蝎](蝎.md)；曾想拦住离场的 [苏晓](苏晓.md)、被千代制止（一个蝎已够难对付）。对应原著「千代 + 小樱 vs 蝎」之战。

## 出处
- 九卷38章（登场）、九卷43–44章（破墙·留守对战蝎）。
