# 黑血（轮回乐园·亚人不死自爆）

[维斯星球·多亚古大陆](../世界/任务世界/维斯星球-多亚古大陆.md) 世界争夺战中轮回乐园方的「疯子」代表——黑发红眼少年，赤膊上身绑满上百颗炸弹。死太多次，连自己叫什么都记不清（「近期死太多次了、我好像叫黑血？好像又不是」），常站在敌阵前「思考人生」（我是谁 / 我在哪）。

- **亚人血统 + 不死体质**：亚人血统生存力极强、**死后会重置身体、恢复一切伤势**（远胜人造人等血统），致命缺陷是攻击力偏弱——他便用「**随机神秘学炸弹生成 Lv.28**」（随机生成精神 / 魔法 / 炼金等几十种神秘学炸弹）来弥补。亚人不死 + 随机炸弹 = 旁人戏称的「**塔利班玩家**」。他唯一怕的是麻醉类药剂 / 技能（不让他重置）。
- **开战屠兽人（十二卷59–60章）**：[希女王](希女王.md) 驱万名兽人当炮灰，黑血独闯兽人军、反复搂着敌人自爆、炸死上千兽人（被砍掉头颅手臂也照样重生），连天启契约者都被他「对生死麻木」的态度看得心惊。
- **死 168 次拖希（十二卷67章）**：契约者大乱斗里他与一名独臂团长死守第二区，**一小时内死了 168 次**、硬生生拖住强到能秒人的 [希女王](希女王.md)——可亚人血统并非「完全不死」，他终于快撑不住了。
