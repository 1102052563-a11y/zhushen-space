# 飞坦

[全职猎人世界](../世界/任务世界/全职猎人.md) **[幻影旅团](../势力/index.md)（蜘蛛）成员·编号 2**——身高一米五左右、细长的死鱼眼、一身高领黑色风衣（领口挡住下半边脸）。**变化系念能力**，性格冷酷，**爱好拷问敌人**（手法丧心病狂、超出常人心理承受极限）。虽是变化系，却有**不弱于强化系的近身战斗力**，近战能力在旅团内能排进**前三**。

## 能力

- **伞中藏刀**：一把暗红色、伞面绘黑色骷髅的雨伞，矮小身材正好完全藏身伞后；伞内抽出一把轻薄细长的长刀（与 [苏晓](苏晓.md) 的武器款式相似）。
- **极致的速度**：速度快到视线只能捕捉残影；但**攻击力道与防御力都不强**——解除念防御后身体只比普通人强些（中弹会透体而出）。
- **超强分析力**：仅几分钟交手就分析出苏晓「蓝色能量须借斩击伤口侵入」「反击盾消耗大、持续短」两大底细并构思对策。
- **大招「rising sun」**：制造一颗**小型太阳**；**使用前受伤越重、小型太阳温度越高**——故他敢故意找黑帮枪林弹雨自伤来增强威力。须消耗念能量，若念能量被噬灭一空则大招作废、必死。

## 历程 · 袭击拍卖会（十卷27–30章）

奉团长 [库洛洛·鲁西鲁](../势力/index.md) 之命与 [富兰克林](富兰克林.md)、[小滴](小滴.md) 凿墙潜入友克鑫拍卖会地下室（三人负责吸引黑帮注意、窝金等四人真正去抢拍品）。

- **拷问并杀死「全知全能契约者」**：一名运气好、掌握剧情的契约者三天前找到库洛洛、自称对十老头了若指掌、协助旅团。飞坦早看其不爽，识破他「连念的基础都不懂、却装预言家全知全能」，借「拷问得情报」之名将其折磨致死（其吐露的轮回乐园情报被乐园屏蔽）。
- **vs 苏晓**：被布布汪发现行踪后，飞坦自愿断后、与 [苏晓](苏晓.md) 在地下室单挑（罕见地起了战意）。被苏晓的 [反击盾](../力量体系/技能/index.md) 克得头疼、两度被斩中（青钢影真伤 + 噬灭近 1/10 念能量）；苏晓则忌惮其大招、放弃了重伤他的机会。僵持中飞坦看出反击盾将失效，**改冲上层黑帮成员、故意挨枪自伤以蓄力放 rising sun**。
- **放大招「太阳」（十卷31章）**：飞坦换上念能量具现的「**罪无可赦之人**」黑红衣物（隔热 + 极强防御·防自己被焚），就在他蓄力时被苏晓一记 55 力的全力侧踢**踢穿天棚与外墙、踢飞出拍卖会**（出了半封闭环境威力大减）。半空具现出**直径十几米的小型太阳**（原著仅 5 米·受伤越重温度越高 / 持续越长）——670° 燃沥青、烤化钢铁、三百米内黑帮平民尽成白灰、地面玻璃化、持续 3 分钟。苏晓早通知布布汪撤离、自己钻进 −3 层大型冷库（开到最低温）避过，仅冻得不轻。
- **死亡（十卷32章）**：重伤的飞坦逃到东南角民宅、杀一家四口（夫 + 少妇 + 双胞胎小萝莉）休整，却被苏晓**以青钢影能量作坐标追踪**（Lv.26 青钢影 + 布布汪循味）找到。强弩之末的飞坦选择体面赴死（思考着「中午吃的什么」），被苏晓一刀斩首。**结算：[苏晓](苏晓.md) 得世界之源 +7.6%（→22.3%）+ 噬灵者 +14 法力（本世界已满 100·法力 1683）+ 暗紫宝箱 + 幻影旅团声望 +100 / 加入旅团资格**（苏晓不屑用·疑陷阱）；苏晓割下飞坦人头 + 带旅团编号的皮肤作威慑十老头之用。
