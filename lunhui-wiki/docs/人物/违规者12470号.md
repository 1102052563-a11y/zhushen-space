# 违规者 12470 号

**12470 号契约者** 是 [苏晓](苏晓.md) 在 [东京喰种](../世界/任务世界/东京喰种.md) 世界的**猎杀目标**——一名钻规则漏洞牟利的违规者（二卷2、41章）。

## 简介
黑色长袍、宽大兜帽，一双刻薄残酷、隐含疯狂的三角眼；属喰种阵营契约者，却有**特殊方法规避杀喰种应扣的阵营贡献值**（二卷41章）。

- **杀法**：从口中抠出一种黑色粘稠物质、沾在尸体（多以诱拐来的孩童分尸为饵）上，喰种吞食后他一声「爆」即可**无火光引爆**该喰种；如此在 4 区（无人类的喰种区）以血腥味聚来数十喰种，再一举炸杀、炸出白色宝箱（自称「**双人份**」、收益翻倍）（二卷41章）。
- 借此屡屡完成「成就任务」攫取双倍奖励，无视轮回乐园的多次违规警告——正是**猎杀者**当除的对象（二卷41章）。
- 截至当前章节与苏晓尚未交手（猎杀任务无惩罚、无时限）。

## 关系
- [苏晓](苏晓.md)（白夜·黑色死神）——奉乐园之命的猎杀者，尚未会面。
