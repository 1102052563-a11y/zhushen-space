# 违规者 12470 号

**12470 号契约者** 是 [苏晓](苏晓.md) 在 [东京喰种](../世界/任务世界/东京喰种.md) 世界的**猎杀目标**——一名钻规则漏洞牟利的违规者（二卷2、41章）。

## 简介
黑色长袍、宽大兜帽，一双刻薄残酷、隐含疯狂的三角眼；属喰种阵营契约者，却有**特殊方法规避杀喰种应扣的阵营贡献值**（二卷41章）。

- **杀法**：从口中抠出一种黑色粘稠物质、沾在尸体（多以诱拐来的孩童分尸为饵）上，喰种吞食后他一声「爆」即可**无火光引爆**该喰种；如此在 4 区（无人类的喰种区）以血腥味聚来数十喰种，再一举炸杀、炸出白色宝箱（自称「**双人份**」、收益翻倍）（二卷41章）。
- 借此屡屡完成「成就任务」攫取双倍奖励，无视轮回乐园的多次违规警告——正是**猎杀者**当除的对象（二卷41章）。
- 截至现身时与苏晓始终未正面交手（猎杀任务无惩罚、无时限）。

## 真容与结局（二卷90章）
其真实面目是一张干枯如丧尸的脸（人称「**丧尸脸**」），手臂上有**两枚编号相连的轮回烙印**（12470 / 12471）——他与孪生哥哥天赋相同（[「心映心」](../力量体系/天赋.md)），在「钢之炼金术师」世界被炼金术师**人体融合炼成**、失败后侥幸独活、哥哥的烙印留在他身上。

- **双重烙印**之利：可同接两份相同任务，击杀掉宝箱、开箱奖励皆翻倍（视作两人）。代价：身体近乎坏死，力 / 敏 / 体 / 魅均不能强化、只能强化智力，却又无法师天赋（感知不到元素），只得买**精神炸弹**技能牟利。
- **伏诛**：他在 20 区分部布满精神炸弹、等人多 + 等苏晓上门引爆牟利；却因苏晓完成主线③、喰种阵营契约者死光，导致**他的主线任务失败 → 被乐园强行处决**（双烙印令处决速度翻倍），灰飞烟灭。苏晓由此完成猎杀任务——印证「给乐园一个正当处决违规者的理由即可、不必拼命」。

## 关系

- 孪生哥哥（12471 号）——同遭炼成、已死，烙印留于其身。
