# 大臣

**大臣** 是 [斩·赤红之瞳](../世界/任务世界/斩赤红之瞳.md) 世界**帝国的真正掌权者**——操控小皇帝的奸臣（四卷22–23章）。

## 简介
身材肥胖、满头白须却因锦衣玉食不显老态；常端着一盘四成熟的肉边吃边说话（血水顺嘴角滑落）。相貌臃肿、智商却极高、老奸巨猾。他是先王的亲信，**毒害了小皇帝的双亲**，又自小皇帝年幼起将其**洗脑成傀儡**——经营十几年，帝国实权尽归其手。因**最强帝具「至高之力」须皇室血统才能启动**，他才留着小皇帝当「钥匙」、不敢杀。还以女色诱导小皇帝纵欲（致其瘦弱）。

帝都唯一令他忌惮的是 [艾斯德斯](艾斯德斯.md)（与之敌对必两败俱伤）。他借艾斯德斯归来献「**嫁祸夜袭部队**」之计——让艾斯德斯手下刺杀「违逆」官员、栽赃 [夜袭部队](../势力/index.md)，实则借刀**排除异己**（那些反他的忧国良臣），并以「军费 +3 成」打动不爱钱的艾斯德斯（四卷23章）。

## 结局（四卷67章）
苏晓夺走至高之力钥匙、断了他最后底牌后，他当众撕破脸骂小皇帝「废物」、欲弃之逃命。但帝都已被革命军近两百万军包围、又遭苏晓放出的 [牛头领主](../世界/生物万族/index.md) 横扫——黑化成牛头领主「核心」的小皇帝**亲自下令杀他报仇**，他被牛头领主像捏西红柿般**捏成一堆烂肉**，落得身死国灭。

## 关系
- 小皇帝——被他毒害双亲、洗脑操控、最终反被其令牛头领主捏死。
- [艾斯德斯](艾斯德斯.md)——帝国王牌，他唯一忌惮且不愿得罪之人；借其手排除异己。
- [苏晓](苏晓.md)（白夜）——曾欲以黄金挖角、被艾斯德斯挡回。
