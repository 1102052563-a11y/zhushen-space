# 老狼

[全职猎人世界](../世界/任务世界/全职猎人.md) 的契约者——干瘦老者、**精神系法师**，持一把暗紫色法杖（160 评分·暗紫里的垃圾·却仍令众契约者眼热）、主技能「**扭曲激光 Lv.27**」。

- **明面的「刺头」**：招募现场他假装跳出来挑拨 [A先生](A先生.md)、想孤立并取代 A 先生当团队代表——实为二人**串通演的双簧**（用一个「刺头」镇住其他刺头）。最终被 A 先生「推举」为名义首领（A 先生自任副队长）。
- **阴谋与反噬（十卷15–17章）**：他正是这场「坑散人独吞宝箱」阴谋的提议者之一。开战后他法术输出最猛、用轮回印记读巨怪血量指挥。但当奶妈按计划「倒坦」、[石头](石头.md) 却靠 [布布汪](布布汪.md) 光环逃脱后，**独眼巨怪狂暴**（回血 15%→40%）——它那只眼锁定伤它最重的指挥者老狼，一棍连砸两下，**把老狼的身体彻底砸碎、还舔食棍上碎肉**。机关算尽反误了卿卿性命。
