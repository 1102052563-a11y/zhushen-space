# 远坂凛

[Fate/Zero 世界](../世界/任务世界/fate-zero.md) 远坂家的小女儿——[远坂时臣](远坂时臣.md) 与妻子葵之女、[间桐樱](间桐樱.md)（被过继给间桐家）的**亲姐姐**。**扎着双马尾**的小女孩。

- **目睹凶案（十一卷56章）**：苏晓在街道上把 Assassin [百貌](哈桑·萨巴赫·百貌.md) 斩成「人棍」时，她正隔着自家二楼窗帘缝偷看——被这凶残一幕吓得泪水打转、双腿发颤；母亲**葵**（远坂时臣之妻·美少妇）悄声上楼从背后捂住她的嘴和眼、颤声叮嘱「**凛，千万不要出声**」，护着她躲过这场惊吓。

## 长大成人 · 苏晓的 Master（二十二卷·第五次圣杯战争）

童年那个躲窗帘后偷看的小女孩，在 [第五次圣杯战争](../世界/任务世界/fate-第五次圣杯战争.md)（[血门冒险团](../势力/index.md) 推进的剧情线）中已长成青春靓丽、外冷内傲娇的高中生魔术师——三大魔术家族「远坂家」独女、用**宝石魔术**（把稀有宝石当消耗品挥霍）。本是 [英灵卫宫](卫宫士郎.md)（Archer）的御主、想抱「同学」[卫宫士郎](卫宫士郎.md) 的大腿夺圣杯，却被 [苏晓](苏晓.md) 强行剥离卫宫士郎令咒、**改认她当 Master**（她秒懂「拒绝可能就会死」、识时务地同意）。

- **被苏晓「开挂」改造**：苏晓在她背上**刻画「第三星辉炼金阵图」**（详见 [装备道具](../力量体系/装备道具.md) / [机制](../总览/轮回乐园机制.md)·魔力存储 20→600+·当炮台·非刻印·安全极限 95% 否则化星辉粉尘）。
- **杀父之仇**：苏晓告知她父亲 [远坂时臣](远坂时臣.md) 是被「好学生」[言峰绮礼](言峰绮礼.md) 用赠予的对师宝具（宽刃 + 红宝石短剑·后又转赠给她本人）背后捅死——却不帮她报仇。
- **被苏晓支配的日常**：被「手动休学」、绑手脚扔沙发、家被血门重力磁场夷平、被半哄半骗去举报言峰绮礼 / 强闯爱因兹贝伦家；对 [布布汪](布布汪.md) 怕得要命（「地狱三头犬」）、受伤只敢找 [阿姆](阿姆.md) 包扎。圣杯战争结束后留在本世界。

## 关系
- **[远坂时臣](远坂时臣.md)**：父亲（被言峰绮礼背刺杀害）；母亲葵。
- **[间桐樱](间桐樱.md)**：被过继给间桐家的亲妹妹（第五次圣杯战争中是敌方 Rider 御主·未重逢）。
- **[苏晓](苏晓.md)（白夜）**：先隔窗目睹其斩杀百貌的凶相；第五次圣杯战争中成其 Master / 被支配对象。
- **[卫宫士郎](卫宫士郎.md)**：同学 / 原从者英灵卫宫的御主（令咒被苏晓换走）。
- **[言峰绮礼](言峰绮礼.md)**：杀父仇人。
