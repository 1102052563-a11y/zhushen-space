# 利威尔·阿克曼

**利威尔·阿克曼** 是 [进击的巨人](../世界/任务世界/进击的巨人.md) 世界 [调查兵团](../势力/index.md) 的**兵长**，号称「**人类最强士兵**」（三卷13、23章）。

## 简介
身材矮小（被苏晓拿身高反复嘲讽）、洁癖、言语毒舌，却是墙内战力天花板——曾与铠之巨人莱纳正面交战。与 [三笠·阿克曼](三笠.md) 同为**阿克曼**一族（天生身体素质远超常人）。罗塞之墙攻防战后期与 [韩吉·佐耶](韩吉·佐耶.md)、苏晓等同处调查兵团总部（三卷23、36章）。

对苏晓（其身份是「前调查兵团成员白夜」）**敌对而不仇视**：餐桌上逼他「别浪费食物」、互相嘴炮，苏晓却感觉不到太强敌意，推测前身与利威尔曾有交情、后反目，更像「竞争对手」。韩吉佐耶掀桌、[艾伦](艾伦.md) 受惊将变身时，利威尔与苏晓同时出手——苏晓锁喉、利威尔压身——合力制服艾伦，可见二人默契仍在（三卷36–37章）。

## 团长阵亡后（三卷46章）
追击叛逃的苏晓一行至玛利亚之墙外时，团长艾尔文被苏晓「凝视射击」**一枪爆头**身亡。利威尔自知战力虽强、却**无指挥才能**，当即指定 [韩吉·佐耶](韩吉·佐耶.md) **代理团长**、由她接过指挥继续追击。

## 关系
- [三笠·阿克曼](三笠.md)——同族（阿克曼），均有超常身体素质。
- 艾尔文——原团长（三卷46章被苏晓爆头身亡）；[韩吉·佐耶](韩吉·佐耶.md)——同僚（被他推为代理团长）。
- 艾尔文——[调查兵团](../势力/index.md) 团长（其上司）；[韩吉·佐耶](韩吉·佐耶.md)——同僚。
- [苏晓](苏晓.md)（白夜）——前身的旧识、敌对却不仇视的「竞争对手」；联手制服艾伦。
- [艾伦](艾伦.md)——被其监视看管的「进击的巨人」。
