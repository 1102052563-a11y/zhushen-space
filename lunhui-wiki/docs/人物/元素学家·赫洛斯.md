# 元素学家·赫洛斯（原魔能导师·被苏晓击碎信念后蜕变的施法者）

虚空 **奥术永恒星**（以魔法完美替代科技的大种族）的施法者——原为准八阶的「**魔能导师·赫洛斯**」，三十六卷空座宴外围苏晓「1 对 156 屠施法者」时与之战斗、惨败，那一战几乎击碎了他毕生坚信的强大（在灭法者面前毫无反击之力）。为此他消沉、消瘦、颓废了很久（这颓废气质反倒吸引了不少年轻女施法者、令他很头疼）。

## 蜕变 = 元素学家诞生（38卷56章）

在魔法大书库苦读古书后，赫洛斯读懂了自己驾驭多年的元素之力另有特性，决心做一件「哪怕下一秒死也要做」的事——**释放自己束缚多年的本源元素之力**（近乎自杀，皮肤生皱、生命之火将熄、魔路崩溃）。就在此时，那些木特性元素之力竟**全部倒涌回他体内**——他明白，这些元素不再是被囚禁，而是自愿归来；他随手结出一颗一直梦寐以求的木特性元素果实、甚至能让几十公里内植物疯长。**「你们疲劳了？那就回来休息吧」**——放下对「元素平衡」的执念、与元素达成真正的共存，**元素学家·赫洛斯就此诞生**。他的理想是探索未知，被苏晓「教育」后彻底认清自己不是战斗那块料。

## 🕯 真相 —— 「哪有什么突然醒悟的元素学家」（54卷169章）

> ⚠ **他不是一个人，而是一代代元素学家意识的载体。**

⚠ **苏晓在万族界的黑市里以12000枚灵魂钱币买下一颗未知七彩水晶**——⚠ **他并不知道那是什么，只是感受到其中有强烈的「起源石气息」，却又与起源石大不相同**（⚠ 后经珍品商鉴定完成、送回到他手中·⚠ 他在黑枫树下冥想时，从水晶内感受到了一种熟悉又陌生的「宿命感」——**那是灭法之影的宿命气息**·⚠ 由此他确定:买下这颗水晶并非巧合，是有一定的宿命干涉；且缔造这份宿命共鸣的那一方**并无恶意**，否则他能更早察觉）;⚠ **⚠ 水晶碎裂后，元素幻境展开**:⚠ **一望无际的白色花海、缓慢转动的风车、辛勤劳作的花农**——⚠ **一名灰色头发的幼童正在花田里追逐蜜腹独角仙**（⚠ 那并非贪玩:蜜腹独角仙是城里孩子们的最爱宠物，逮住一只就是笔额外收入——**「我那时就懂得为母亲分担家庭开支，但谁能想到，我今后会成为施法者，还被冠以魔能导师的称号」**）。

- ⚠ **⚠ 他的自陈**:⚠ **「白夜，其实，我早就死了」**——⚠ **被苏晓所杀、被施法者们折磨致死〔他们太想知道深渊和元素能不能互相转化〕、被星界异存在咒杀、被深渊滋生物吞吃……**;⚠ **⚠ 每一次死亡，他都能重新回忆起些什么**——⚠ **「哪有什么突然醒悟的元素学家，我不过是一代代元素学家意识的载体而已」**;⚠ **⚠ 「这是命运，也是痛苦到极点的诅咒」**:⚠ **他现在虽然没有力量，却比那些深渊族裔更难杀死——奥术永恒星的暗杀部成员用各种方法杀死他几万次，他依然会醒来、残破的身体逐渐恢复**——⚠ **可灵魂承受太多苦痛，会干瘪**（⚠ 他抬起右手，宽松衣袖滑落，露出干枯的右手与小臂——**宛如被风干，里面仿佛有蠕虫在缓慢钻动**）。
- ⚠ **⚠ 他的自我评价与旧事补完**:⚠ **「别看我们初次交锋时我只有八法核〔约八阶〕，我对新生施法者们的启迪授课水平，放眼奥术永恒星也是前十之位」**——⚠ **实力强大并不代表懂得教导；对新生一代的启迪用不上高深知识，都是三阶甚至两阶以下的法系知识，而启迪对一名新生施法者而言很重要**（⚠ 但这不代表他在奥术永恒星有很高地位，最多是比其他导师更有地位，类似教导主任·⚠ 当年那次「外出教学」总计出动156名施法者:准八阶1人、七阶102人、六阶41人、五阶12人——**七阶是来正式学习「怎样窃夺一个世界的自然元素」，六阶是他昔日的学生来提前熟悉，五阶则是几大派系里长辈关系很硬的、被硬塞来长长见识**·⚠ 那是个七阶世界，八阶战力根本进不来，对这些施法者堪称**极致的安全**——**结果他们遇到了灭法之影·白夜，全灭**·⚠ 他之所以活下来，是因为他有两具身体，返回奥术永恒星的主身体逃过一死）。

## ⏳ 30天倒计时 —— 他给苏晓的报酬与请求

⚠ **⚠ 「能和你吐露这些，真好啊」**——⚠ **他不能理解苏晓是怎么凭自身毅力一步步杀成唯一天选，但和苏晓说这些秘密**不会受到命运的反噬**;⚠ **⚠ 作为回报，他给了苏晓一个时间**:

> ⚠ **「你被原罪吞没的时间，只剩……30天。这些原罪物一直在等你封临至强之巅——你越强，它们能引动的原罪因果就越强大。如果不是收藏家的收藏癖一直得到满足，这个时间会更短。我不希望你死，但也只能帮你到这。」**

- ⚠ **⚠ 他索要的报酬**:⚠ **「如果，我说如果，你真有一天能战胜那位，你就是无可置疑的最强。到那时——杀了我，让我沉入黑暗吧，远离不断死而复生的痛苦。」**——⚠ **「当初，你们轮回乐园的那位〔[团长](#)〕，到了深渊之主那一步就被迫停下。希望你能更进一步。」**（⚠ 留下此言后，他的身体开始破碎，周边的元素环境如同被引燃的画作般快速燃尽、消散）;⚠ **⚠ 苏晓的判断**:⚠ **姑且不论他所说是真是假，尽快解决「原罪因果」的反噬的确很有必要**——⚠ **而陨灭星那五只叛逃的吞噬者，正是破除这因果的起始**（⚠ 「就像一部话剧，直接进入最后部分会很突兀，必然让原罪因果的破除变得异常凶险——所以这场演出的**第一幕，要由那五只先开场**」）。
