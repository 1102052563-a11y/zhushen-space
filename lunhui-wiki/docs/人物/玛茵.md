# 玛茵

**玛茵** 是 [斩·赤红之瞳](../世界/任务世界/斩赤红之瞳.md) 世界 [夜袭部队](../势力/index.md) 的远程火力（四卷28章）。

## 简介
扎粉发双马尾、一身粉色长裙的活力少女；是**狙击手 + 炮手**，年龄不大却是老练的暗部。

- **帝具「浪漫炮台·南瓜」**（枪形）：不打实体子弹，而将**精神能量转化为冲击**（苏晓觉得更像激光）；**使用者处境越危险、威力越强**；敌人近身后它就从「枪」变「炮」——能持续发射、端枪横扫的激光炮。

本批中她以南瓜的火力支援，助布兰德解决了三兽士；对苏晓的全程**远程牵制**令其难以全力（橙色激光逼退、掩护同伴撤退）。后与塔兹米一同跳河脱身（四卷28–30章）。

## 关系
- [娜洁希坦](娜洁希坦.md)（首领）、[赤瞳](赤瞳.md) / [雷欧奈](雷欧奈.md) / 布兰德 / [塔兹米](塔兹米.md) 等——夜袭同伴。
- [苏晓](苏晓.md)（白夜）——其远程牵制是苏晓客船战的最大变数。
