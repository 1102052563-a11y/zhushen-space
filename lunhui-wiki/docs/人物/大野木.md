# 大野木（三代土影 · 两天秤）

岩隐村（土之国）首领、**第三代土影**，人称「**两天秤**」——身高仅 1 米 3 的小老头，却**城府极深**（五影中唯一能与 [团藏](志村团藏.md) 比城府者·真在城府上交手约五五开）。

## 赴五影大会（十五卷9章）

[四代雷影](四代雷影.md) 提议召开五影大会时，他正犹豫——但料定其他四影都会到场，若独缺岩隐村难免被小觑，遂揉着额头决定「没办法了，黑土，准备出发」（黑土为其孙女）赴会。岩隐村对夺尾兽的晓组织态度同样不友好，他亦是这场暗流涌动的五影大会的一方。

## 关系
- **[团藏](志村团藏.md)**：城府旗鼓相当的对手（同赴五影大会·分代木叶 / 岩隐立场）。
- **[苏晓](苏晓.md) / 晓组织**：抓尾兽令各村敌视晓（岩隐对苏晓态度不友好）。

## 五影大会主谈 · 护弟子（十五卷22–24章）

会上他是己方与「宇智波斑」交涉的主谈者（团藏稳坐钓鱼台 / 风影太年轻 / 水影自顾不暇）——他年少时曾见真宇智波斑横扫岩隐村（人柱力毫无还手之力），故对其忌惮。带土摊牌月之眼计划 + 宣战后，他擒住的晓成员 [迪达拉](迪达拉.md)（其得意弟子·部下遗孤·若非太叛逆本是土影候选人）成了焦点——他**有意保迪达拉**、用「他自爆能再夷平木叶」挡回 [团藏](志村团藏.md) 的索人，又压三船「这是岩隐内政、武士少插手」。其「不因个人情绪影响忍村」的克制，正是 79 岁仍稳坐土影之位的原因。
