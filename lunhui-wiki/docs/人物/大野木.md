# 大野木（三代土影 · 两天秤）

岩隐村（土之国）首领、**第三代土影**，人称「**两天秤**」——身高仅 1 米 3 的小老头，却**城府极深**（五影中唯一能与 [团藏](志村团藏.md) 比城府者·真在城府上交手约五五开）。

## 赴五影大会（十五卷9章）

[四代雷影](四代雷影.md) 提议召开五影大会时，他正犹豫——但料定其他四影都会到场，若独缺岩隐村难免被小觑，遂揉着额头决定「没办法了，黑土，准备出发」（黑土为其孙女）赴会。岩隐村对夺尾兽的晓组织态度同样不友好，他亦是这场暗流涌动的五影大会的一方。

## 关系
- **[团藏](志村团藏.md)**：城府旗鼓相当的对手（同赴五影大会·分代木叶 / 岩隐立场）。
- **[苏晓](苏晓.md) / 晓组织**：抓尾兽令各村敌视晓（岩隐对苏晓态度不友好）。

## 五影大会主谈 · 护弟子（十五卷22–24章）

会上他是己方与「宇智波斑」交涉的主谈者（团藏稳坐钓鱼台 / 风影太年轻 / 水影自顾不暇）——他年少时曾见真宇智波斑横扫岩隐村（人柱力毫无还手之力），故对其忌惮。带土摊牌月之眼计划 + 宣战后，他擒住的晓成员 [迪达拉](迪达拉.md)（其得意弟子·部下遗孤·若非太叛逆本是土影候选人）成了焦点——他**有意保迪达拉**、用「他自爆能再夷平木叶」挡回 [团藏](志村团藏.md) 的索人，又压三船「这是岩隐内政、武士少插手」。其「不因个人情绪影响忍村」的克制，正是 79 岁仍稳坐土影之位的原因。

## 尘盾守监狱 + 追杀苏晓（十五卷55–59章）

苏晓攻竖岩村救 [迪达拉](迪达拉.md) 时，他**亲自坐镇地下监狱守株待兔**（算计了情报失误的 [白绝](绝.md)、报成是黄土守监狱·平民的冷静正因土影坐镇）。他是纯忍术型（年老 + 腰伤·体术不如中忍·但会飞），祭出岩隐最高秘术「**尘盾**」——预埋石刺秘术 + 「超加重岩之术」（罩一层万斤不可见铠甲）+ **原界 / 限界剥离之术**（分解一切·攻击力变态到反击盾都顶不住·限界版范围 / 持续更强）。这是他几十年来**首次遇到有人硬抗尘遁**（苏晓靠次级无敌护盾 + 多层能量盾扛下）。但他被 [布布汪](布布汪.md) 趁放尘盾时摸进牢房**救走迪达拉**（留「本汪到此一游」）打乱全盘——气得血压飙升（人被劫走还说得过去、怎么劫走的都不知道）。

随即他**率岩隐追击大军**穷追两天两夜：他 + **黄土**（其子·「岩隐之壁」·土遁专家·原著曾以大型土遁压制十尾）+ **黑土**（其孙女·后任四代土影·官三代）三直系亲属带队，上忍 10 / 特上 15 / 中忍 507、分追 / 围 / 补给三梯队的战争阵型（他经历过忍界大战·深知晓组织是战争专家、不以人数判其破坏力）。苦于双方速度差，终未追上（也不敢追进火之国、那等于对木叶宣战）。

## 其子黄土战死忍界大战（十五卷81–84章）

第四次忍界大战中，他的儿子**黄土（「岩隐之壁」·土遁专家）指挥岩隐忍者联军第二部队**（2 万人·含黑土 / 日向宁次 / 日向雏田 / 卡鲁伊）迎战白绝大军。黄土实力强却低调（土遁专家·失臂则大半忍术作废）——苦战近 7 小时、第二部队几乎全灭，他为掩护黑土等人撤退被大量白绝围攻，**[苏晓](苏晓.md) 趁机斩断他的手臂、再斩下头颅**（殁）。黑土、日向宁次、日向雏田三人逃脱。

## 关系
- **[迪达拉](迪达拉.md)**：得意弟子 / 部下遗孤（生擒后不刑讯、却被布布汪从他眼皮底下救走）。
- **黄土 / 黑土**：子 / 孙女（黄土 = 岩隐之壁土遁专家·忍界大战中战死于苏晓刀下·黑土 = 四代土影）。
- **[苏晓](苏晓.md)**：尘盾逼其硬抗、却被智取走人、又追杀两天未果的对手。
