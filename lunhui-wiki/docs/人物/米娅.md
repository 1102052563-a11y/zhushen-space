# 米娅

**米娅** 是 [海贼王](../世界/任务世界/海贼王.md) 世界、哥亚王国城外 [废品终点站](../世界/地点/index.md) 的少女，约 15、6 岁（第3章）。

## 简介
为在废品终点站求生而**隐藏性别**（当地女性是危险的身份）。亚麻色宽大衣服、脸颊白皙、身形瘦弱。误入苏晓藏身的破屋时被其一招制服；心脏遭重击仍未昏迷，被苏晓评价体质很强（第3章）。

## 经历
- **第3章**：避难误入破屋，被 [苏晓](苏晓.md) 制服、盘问。
- **第4–5章**：以压缩饼干为交换，带苏晓走出废品终点站、抵达哥亚王国侧门，随后离开（带路只为食物）（第5章）。

## 出处
- 首次登场：第3章。
