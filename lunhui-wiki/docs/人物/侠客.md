# 侠客

[全职猎人世界](../世界/任务世界/全职猎人.md) **[幻影旅团](../势力/index.md)（蜘蛛）成员**——金发，**操作系念能力者**，擅长分析情报。旅团辅助人员，但战斗力不弱（关键时操纵自身、瞬间爆发力不逊 [飞坦](飞坦.md)、[信长](信长.md)）。

## 能力 ·「自动操作」

- **天线操控**：在敌人身上插入念「天线」后，无论对方多强都能操控其行动；亦可操控自身或队友。
- **「最强组合」自动操作**：把天线插入 [窝金](窝金.md) 口腔、**耗尽自身几乎全部念能量**注入窝金，使其进入 **「自动操作」**——力 / 速 / 念 100% 凭本能爆发、攻击不经大脑、防御倍增（金色念气护体）。代价极大：施术时侠客本人不能移动（风险极高）、持续仅 9 分钟、事后他一个月不能用念，而窝金会卧床一周；**最致命的是——若窝金在自动操作中战死，侠客的念能量（即生命能量）会全部消散、当场死亡**。这正是 [窝金](窝金.md) + 侠客被称为旅团「最强组合」、却也是双刃剑的原因。

## 历程 · 死亡（十卷41–43章）

山谷之战中，为给 [窝金](窝金.md) 报 [信长](信长.md) 之仇、抗衡克制窝金的 [苏晓](苏晓.md)，侠客启动了「自动操作」。然而苏晓挖掉窝金双眼令其终止、继而斩杀窝金——**窝金一死，侠客便七孔流血、念能量尽散而暴毙**。

- **[苏晓](苏晓.md)**：间接死于他手（斩杀窝金连带）。
