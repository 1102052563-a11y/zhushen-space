# 间桐樱（远坂樱）

[Fate/Zero 世界](../世界/任务世界/fate-zero.md) 的紫发小女孩——本是 [远坂时臣](远坂时臣.md) 的女儿 **远坂樱**，被过继给间桐家、改姓间桐。本应天真无邪，却被 [间桐脏砚](间桐脏砚.md) 丢进虫池**折磨一年**（体内还被藏了间桐脏砚的本体血红虫），眸子黯淡无光、却有超乎常人的忍耐力（剧痛也不哭闹）。

- **被苏晓救出（十一卷3章）**：苏晓清虫池时发现赤身站在虫堆里的她，踩碎她吐出的间桐脏砚本体虫；她体质远超 [间桐雁夜](间桐雁夜.md)（虫只能侵其器官、不能侵血管组织），故得以存活。
- **黏着布布汪**：[布布汪](布布汪.md) 高达 53 的魅力属性对绝望的她形成「心灵慰藉 + 依靠」——她一步不离地跟着布布汪、被苏晓的凶气吓到就躲向它（一度连「拒绝」「笑」等情绪都重新出现，印证魅力属性的强大）。苏晓留她当「备用魔力供给者」（若雁夜的改造失败、就靠她继续留在圣杯世界）。
- **送回生母身边（十一卷58章）**：主线完成、生父 [远坂时臣](远坂时臣.md) 与「养父」[间桐雁夜](间桐雁夜.md) 皆死后，苏晓**把她送回了生母远坂葵处**（葵是时臣之妻、[间桐雁夜](间桐雁夜.md) 的青梅竹马、[远坂凛](远坂凛.md) 的母亲）——母女隔街相认、葵把她紧紧搂入怀。她临别对苏晓轻声道了句「谢谢」（小到没人听见·想起被他从虫池拎起那一幕）。苏晓送她、只因她够听话不哭闹（他并非滥杀之人）。

## 第五次圣杯战争 · 被血门培养的 Rider 御主（二十二卷·另一剧情线）

[血门冒险团](../势力/index.md) 推进的剧情线里，她幼年被血门从间桐家「老虫子」手中救出、送到时钟塔一位**护徒狂魔老魔术师**门下培养魔术回路（为做最纯净的人造圣杯容器）——长成 **Rider 职阶御主**（英灵 [美杜莎](美杜莎.md)·女妖形态·石化之瞳近乎能秒所有英灵）、还得了件圣遗物「石眸」。她性格依旧纯善、经历与原著大不相同（无黑化）。[斯坦](斯坦.md) 一死、老魔术师与她的契约解除，加上 [苏晓](苏晓.md) 在场是「噩梦难度」，老魔术师 + 旧识 [假面](假面.md) 遂把她**带离冬木市保命**——苏晓始终没寻获她、亲姐妹决战未发生。
