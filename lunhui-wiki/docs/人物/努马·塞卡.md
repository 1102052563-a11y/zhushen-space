# 努马·塞卡

**努马·塞卡** 是 [斩·赤红之瞳](../世界/任务世界/斩赤红之瞳.md) 世界**极北异族军的首领**、被称作「**北方异族王子**」（四卷14章）。

## 简介
手持长枪、**从无败绩**，是极北人民图腾般的人物。极北因帝国苛税揭竿起义，他率军以 **3 万大破帝国 20 万大军**（帝国仅余 7 万、13 万埋骨冻土），这是帝国数百年未受的惨败，逼得小皇帝震怒派 50 万大军 + 王牌 [艾斯德斯](艾斯德斯.md) 镇压。本人即极北最强战力（住处无侍卫）；不好女色、生活简朴，唯一弱点是**嗜酒**（极北苦寒、男子无不饮酒，他多在夜里十点后独酌）。心中似有一名「玛莎」要为之报仇（四卷14、19章）。

## 被刺杀（四卷20章）
帝国攻「咚咚塔要塞」前夜，[艾斯德斯](艾斯德斯.md) 派副官苏晓（化名白夜）潜入要塞刺杀他。苏晓杀守卫潜近其石屋、用试管灰白液体汽化的**毒烟**从门缝灌入（自戴防毒面具），待努马·塞卡中毒瘫软、连话都说不利落，**割喉斩首**。全程仅杀他一人 + 重伤一名守卫，六小时余返营。他作为「极北关键人物」死亡，给苏晓 **1.1% 世界之源**（噬灵者 +14 法力）；开战前首领阵亡 + 夜袭，令极北军遭灭顶之灾。掉落蓝色宝箱（开出 [冰域王子令牌](../力量体系/装备道具.md)）。

## 关系
- 极北异族军——其统率的起义军（首领阵亡后溃败）。
- [艾斯德斯](艾斯德斯.md)——奉命镇压极北、派人刺杀他的帝国王牌。
- [苏晓](苏晓.md)（白夜）——以毒烟 + 割喉刺杀他的帝国暗部。
