# 无伞兄（圣诞）

一名**契约者**——真名（代号）**圣诞**，但更出名的诨号是「**无伞兄**」。**主坦**（不使用盾牌），**生存能力极强、运气却极差**。

- **诨号由来**：八卷11–13章登岛时，他是**那个买不起降落伞、被裁判 [鼠大人](鼠大人.md) 从 4000 米高空直接投下的契约者**——[苏晓](苏晓.md) 当时以为他必死。谁知他靠超强生存力**接连大难不死**：高空坠落 + [钢羽黑鹰](../世界/生物万族/index.md) 抓走没死、被红皮土著俘虏当肉菜、逃跑「跳崖」也没死……苏晓再见他时大为诧异「这家伙居然没死？！」。他自嘲「**圣诞也只是个代号、往事不堪回首、叫我无伞兄就好**」。
- **能力 / 定位**：**主坦不用盾、抗揍抗摔到离谱**（空落 / 跳崖都死不了），就是**运气糟透**。也因此被 [阳之女](阳之女.md) 抓作强逼进黑色森林的两名契约者之一（脖颈攀着控制藤蔓）。

## 队伍中的「刺猬」（八卷60–61章）
与 [小迷糊](小迷糊.md) 一路从红土区闯出、结下交情。四人小队集结议推队长时，他看穿苏晓、[巴尔](巴尔.md) 这「一群狼」的算计，**毛遂自荐想保小迷糊**（不让这只「狼群中的小绵羊」当替罪羊），未果——苏晓与巴尔联手把小迷糊推上队长位。他暗下决心「要做一只**刺猬**」（让狼不好下口），并接手了被小迷糊开得险些漂移的 [装甲车](../世界/任务世界/T12K星球-生存试炼.md)。

## 主坦的尊严（八卷62–65章）
他的「主坦」不用盾——而是**一双厚重的金属拳套**（包到小臂、每只有人躯干粗·两拳挡身前即是盾、比盾灵活·拳套上常蒸汽升腾）。进黑色森林他**坚持走最前方探路、负重最少**（「只有正面硬钢死的主坦，没有逃跑的主坦」），徒手抓住会说人言的 [野猪](野猪.md) 头部硬接其冲撞（双肩崩出裂口仍一步不退）。

- **被「狼群」算计后**：四人小队里他早看穿苏晓、[巴尔](巴尔.md) 与自己目的不同（苏晓问「暗王」时他就懂了）。[苏晓](苏晓.md) 随野猪去寻暗王、与众人分道时，他替愣住的 [小迷糊](小迷糊.md) 点破「（我们）**从来都不是（一伙）**」，却不沮丧——「依靠别人能活一时、不能活一世」，决意与小迷糊、巴尔三人去闯阳之女的「骷髅洞穴」。

## 骷髅洞穴 · 重伤幸存（八卷71章）
他与 [小迷糊](小迷糊.md)、[巴尔](巴尔.md) 去阻止黑雾消散的「骷髅洞穴」之行失败。同行的「奶妈」小迷糊**原形毕露是刺客**——**毒死了巴尔**、还断了无伞兄双臂。可这位「超抗摔」的主坦**又一次没死**，靠在巴尔尸体旁咳血苦笑「**小迷糊，原来你才是藏得最深那个**」。试炼结束时他仍在存活之列（小迷糊「居然没杀无伞兄」，连苏晓都觉意外）。

## Fate/Zero·又是逃亡（十一卷30章）
他也被卷进了第四次圣杯战争所在的 Fate/Zero 世界——从进世界起就被 [血门冒险团](../势力/index.md)（[荒焚](荒焚.md) 一伙）追杀、一路逃亡，蹲在下水道里满身泥污地骂「这都是哪来的神经病、二话不说就追杀老子」，背后照例内置着一副简易降落伞（求生试炼留下的阴影），感慨「二阶好特么危险、还是一阶好」。是本世界那 5 名「躲起来的契约者」之一。

- **重伤获救 · 联手（十一卷38–46章）**：他被荒焚的火焰打成肠子焚毁五分之四的贯穿重伤、生还率不足一成；[苏晓](苏晓.md) 如约赶到、把他抬回别墅**开膛动手术**（取出内脏防 1 号药剂里的柱间细胞令其树木化、塞铁盒导管压制·又被他「我感觉还能抢救一下」反复整活）。痊愈后他**和苏晓联手对付那三人**——他不肯逃（「不应该逃吗、对方三阶」），却也认了「光脚的不怕穿鞋的」苏晓的硬刚。
    - **拖死三阶主坦小胖子**：扛着金属拳套（不用盾·两拳即盾）**正面纠缠 [小胖子](小胖子.md)**——从「狗皮膏药」升级到「搅屎棍」，一直挨揍、专抢小胖子的盾、用嘴炮「恶心」他，又适时打出「**二十倍重拳**」（拳套白色纹路·势大力沉·砸盾给苏晓制造斩击窗口）。靠这种「自损八千、纠缠到底」的打法，配合苏晓的青钢影真伤把小胖子逼到强制回归。
    - **亡妻怀表**：他脖颈戴一块**老式怀表**（一件品级特殊的装备·能力很强却极易破损·表盖内镶着**已故妻子**微笑的照片）——被小胖子打坏怀表令他**双目赤红、不顾反击狂捶**（虽只是「王八拳」）。
- **天赋「绝境逢生」（十一卷47章揭示）**：**稀有天赋·未觉醒**。幸运 −0.5（罕见的减幸运天赋、正应他霉运连连）；生命越低回血越快——低于 50% 每秒 +5、低于 40% +10、**低于 20% 每秒 +30**，且会一直回到 80% 以上才停。这正是他「空落 / 跳崖 / 断臂都死不了」的底牌；**觉醒后必有强效**。
- **再被苏晓救命 + 善后**：小胞子回归后他体内导管断裂、柱间细胞复发树木化、且突然眼前漆黑——苏晓一刀捅其小腹同时塞入 **1 号药剂**（速回 85% 生命 + 压制柱间细胞），又绑来一名中年医生给他二次手术。苏晓「不背后捅并肩战斗的队友」（捅是为救命）。醒来他回赠一枚**紫色品质戒指**致谢（苏晓已救他两次）。还见识了苏晓**居然会做饭**、味道不错，直呼三观尽毁。**他在现实有个 3 岁女儿「夕夕」**、是他拼命求生的执念。

- **「悲伤的故事」 · 睡了斯坦老婆（十四卷77章）**：苏晓返园在交易市场撞见戴半脸面具、瘦了一圈、鬼鬼祟祟摆摊的无伞兄——他被 [斯坦](斯坦.md) 一伙追杀得比 Fate 圣杯世界那次更狠，原因离谱：他在**现实世界**跟人一夜情，对方竟是**斯坦的老婆**、还缠上他要跟斯坦离婚。更微妙的是「相比斯坦本人，[荒焚](荒焚.md) 追杀得更凶」——因为**斯坦老婆是荒焚的嫂子**（嫂子红杏出墙、荒焚为何格外愤怒值得深思）。苏晓请他喝了两听啤酒、听完这桩孽缘便各自散去（苏晓直叹「人生啊」）。

## 人头被悬赏 100 万 · 求荒焚资料（十七卷94–104章）

那桩孽缘发酵成了乐园风波：**斯坦被绿**一事闹大（[斯坦](斯坦.md) 妻 [奈洛伊](莺澜.md) 实为利用无伞兄当「长期情人」棋子·斯坦因此杀了奈洛伊作恶的弟弟、即无伞兄的「大舅哥」），无伞兄成了各大冒险团博弈的靶子——**人头被悬赏 100 万乐园币**（神皇 / 血门 / 红枫各团借机角力·这已是面子之争·见 [势力 · 血门](../势力/index.md)）。十七卷104章他给 [苏晓](苏晓.md) 发邮件求助：「最近一直被血门的人追杀、斯坦他老婆还经常来找我，急需一份 [荒焚](荒焚.md) 的资料、在线等」——苏晓虽没有荒焚全部资料、但刚在竞技场与荒焚交过手（破其心焰），便把大致战斗方式回复给了他。

## 「送死队」· 暗星梦境世界深潜（42卷37–38章）

在 [暗星](../世界/任务世界/暗星-世界入侵.md)，他作为「**送死队**」的队长（名义上）再度与 [苏晓](苏晓.md) 重逢——此时苏晓已是**八阶**，把当年同为二阶的无伞兄惊到不行。「送死队」是他与 [奈洛伊](莺澜.md)、[小侦探](小侦探.md) 三名**六阶**契约者组成的解谜事件专业队，靠他的天赋当核心：

- **天赋觉醒「不死意志」**：他的「绝境逢生」天赋在**完成两次觉醒后蜕变成「不死意志」**——在精神世界内每次「死亡」都能精神吸附、把要散去的精神体吸回本体，真正的「不死」；还能与三名友方**共享**这一效果（须与他签「誓约之契」）。这正是送死队能一次次「送死铺路」解谜的底牌，代价是精神会逐渐疲劳、每次激活消耗生命力。
- **深潜解时停**：苏晓在梦境世界大教堂设坐标传送三人来「深潜」精神世界、解除「时间停滞」。三人一次次死亡铺路，驱散/困住了大贤者、[沙塔耶](暗星/次代神女·沙塔耶.md)、[老骑士](暗星/老骑士.md) 乃至**苏晓本人的无意识投影**（血气持刀人形·无限瞬移追猎，把他们秒到自闭）。解除时停后三人将被强制返园。
- **死了 825 次（42卷49章）**：到时空之力交易结束时，三人在精神世界里累计「死亡」**825 次**、彻底自闭——无伞兄目光呆滞骂「白夜你特么坑老子、脑壳都要被你敲爆了」，[奈洛伊](莺澜.md) 呆滞诡异地傻笑，[小侦探](小侦探.md) 更被砍成「阿巴阿巴」的二哈。更绝的是:苏晓那道血气投影**继承了部分吞噬之核能力**，把大贤者、老骑士、沙塔耶的投影**全部吞噬吸收**——如今精神世界里只剩苏晓的血气投影，谁进去 10 秒内必死。
- **死 1159 次终于通关（42卷58章）**：本以为要失败，三人却在死够 **1159 次**后摸清了血气投影的所有规律——它**只能空间移动、不会走路**，连续移动后感知范围会短暂骤缩（第 7 / 16 / 27 / 39 次…），趁这空档站着不动就会被忽略、才能往迷雾区挪几公分；另有 **42 种必死特性**（用魅力 / 控制 / 坦系 / 灵魂 / 技法 / 增益等 12 类能力必死、与它对视瞬死、朝向它必死…小侦探都哭了）。他们靠一路「坟头」节点标记警示，终于摸到迷雾区、打碎那口停在 6 点的**梦之钟**，**解除了梦境世界的时间停滞**。轮回乐园遂公证梦中世界（苏晓得 10% 世界之源 + 梦之钟碎片），三人被强制返园——只是那道血气投影始终没能驱散（需进去用一把刀斩它三刀），今后或许还有人要进这精神世界受苦。
