# 白羊（女职工者）

[暗星](../../世界/任务世界/暗星-世界入侵.md) 里一名戴矿工帽、腰挂矿铲、脸有雀斑、青春活力的女**职工者**（10***05 号·话痨、有点神经质）。正是她承接大任、在天启 / 圣光双方两百多名契约者的争夺中，**在巨大的「时空之力凝结体」上设下了隐匿坐标**——[苏晓](../苏晓.md) 得以凭此坐标世界入侵暗星（第42卷1–2章）。

## 作用 · 交接即走

- **谁说职工者只能做生意**：她证明了职工者也能承接高危潜入任务。她把**隐匿印记接收器**（半米长金属棍）交给苏晓——凭它可在天启 / 圣光交易时侦测到交易地点（隐匿坐标用时空之力构成·存在 20 天·20 天内不交易就失效）。
- **入乡随俗 + 忠告**：她已在暗星受十几次意志力判定（契约者都能通过）、伪装成矿工。她提醒苏晓：**别惹这世界的古神**（这里情况超复杂）、见到被腐化的人 / 动物马上宰了（这世界很多动物有智慧、能交流）。任务完成后她便利落离去（不与苏晓合作）。

## 关系

- [苏晓](../苏晓.md)（白夜）：她为其布设入侵坐标、交接接收器的职工者（轮回乐园护其不得被伤害 / 劫掠）。
