# 光之王（人族至强者）

[暗星](../../世界/任务世界/暗星-世界入侵.md) 人族的**领袖 / 至强者**——不是某国国王，而是立于人族数十国王之上、维持诸国平衡的「无神名的神灵」。他是封印古神的三条**灵魂锁链**之一（代表人族），也是 [苏晓](../苏晓.md) 主线第一环要见的人（第42卷11章）。

## 救世 · 封印古神的锁链

- **二次救母神 · 失双臂**：古神降临、母神与古神死战险陨后，光之王与野兽的大贤者、虫子的狼蛛女王二次宣战救回母神——他为此**失去双臂**。母神与他交合产下封印古神的「容器」，他再与大贤者、狼蛛女王以灵魂化作缠绕容器的「锁链」二次加固封印。
- **几百年的煎熬**：他坐镇白色小镇（苍白宫殿）的王座、被几百根半透明魂链常年拉扯——**全身如破碎陶瓷般裂成几百块、勉强构成人形、每道裂痕透出微光**，每分每秒承受常人难以想象的痛苦、忍受了几百年，也换来世界几百年的安定（相比古神破封，人族的战争不算什么）。
- **容器裂痕后的分歧**：容器出现裂痕、腐化再临后，他与另两王起分歧——**光之王主张重新培养容器、二次封印古神**（最稳妥）；狼蛛女王主张决战、大贤者主张中立。

## 沉眠中（42卷11章）

苏晓硬闯过白色小镇的「意志之路」抵达苍白宫殿，光之王缓缓抬头、露出遍布裂痕的双眼又闭合——因苏晓**来得过早**，他尚未进入苏醒状态（预计 6 小时后可苏醒 10~15 分钟）。护卫 [月灵](月灵.md) 守在其侧。

## 关系

- **母神** / **大贤者** / **狼蛛女王**：与他共同封印古神的世界级存在（三条灵魂锁链 + 容器之母）。
- [苏晓](../苏晓.md)（白夜）：主线第一环要见他获取情报的猎杀者（也图打开容器杀古神取神灵骨）。
- [月灵](月灵.md)：守护他几百年的王侍。
