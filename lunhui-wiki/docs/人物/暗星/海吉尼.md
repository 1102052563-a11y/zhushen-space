# 海吉尼（贪婪的水系法系保镖）

[暗星](../../世界/任务世界/暗星-世界入侵.md) 里 [天堂小队](天堂小队.md) 被 [苏晓](../苏晓.md) 抢过一次矿后、花大价钱雇来的**八阶水系法系**保镖——满头长发、身穿金蓝色法袍，实力毋庸置疑，却贪婪成性。

## 贪婪 · 捆绑式雇佣（42卷40章）

- **狮子大开口**：约好每天 300 枚灵魂钱币，见面就无理由加价到 400 枚；还要「入伙」分走矿产的五分之一（美其名曰「捆绑式雇佣」——既保护又提高产出）。本人是法系、不亲自挖矿，让「气分身」代劳。
- **被一棍砸残**：苏晓找上门，海吉尼构筑气分身、开出「击退他免费、杀了他 3000 灵魂钱币」的价码想从 [王子](天堂小队.md) 身上再敲一笔。结果他各种水特性法术（流冲·至压 / 水生域 / 溺毙·窒 / 气龙吼 / 海神怒）被苏晓靠青钢影噬灭 + 绝魔体质 + 晶体层正面硬抗（只掉 1.62% 生命）突到面前，一棍砸得他脑壳差点碎、双臂尽断、肋骨全折——他用一件不朽级保命道具才逃走（且因签了契约不敢乱来）。八阶法系的威力确实不俗，但撞上苏晓就是另一回事。

## 关系

- [苏晓](../苏晓.md)（白夜）：抢天堂小队矿石、把他一棍砸残的猎杀者。
- [天堂小队](天堂小队.md)：花高价雇他、却被他坐地起价 + 强行入伙的矿工雇主。
