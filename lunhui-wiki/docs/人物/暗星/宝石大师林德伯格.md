# 宝石大师·林德伯格

[暗星](../../世界/任务世界/暗星-世界入侵.md) 丹克市的宝石工匠——**晶脂提纯法的开创者**，丹克市「宝石街」正是为纪念他而改名（原名古街·第42卷6章）。

## 晶脂提纯法 · 无私公开

- **发现**：他发现把 [晶脂](../../世界/任务世界/暗星-世界入侵.md)（抗腐化硬通币）高温溶解、切入一块超凡宝石再冷却，能**大幅提纯晶脂、增强抗腐化性**——生命 / 自然 / 活力系宝石效果最佳（提纯到近乎透明）、火焰 / 地脉居中、黑暗 / 幽邃最差甚至反效果。他**毫不隐瞒、定期公开**所有心得，整个丹克市为之沸腾、他的小店差点被挤爆。
- **假死 · 伯格之心**：他的「过劳而逝」其实是**假死**——打造出至高之作 [伯格之心](../../力量体系/装备道具/index.md) 后，他为把它献给心中景仰的**光之王**，假死离开丹克市、踏上寻找光之王的旅途。

## 🆕真相 · 死在白色小镇（42卷59–60章）

苏晓在白色小镇一间锁着的商铺里（密令「**至高之作**」）找到了他的枯尸——原来他成功找到了 [光之王](光之王.md)，却在见到对方时僵在原地:光之王为封印古神、身躯早已破碎得不成样子。光之王醒来道「伯格，辛苦你了，我已经没有力量去对抗古神」后再度沉睡；[月灵](月灵.md) 当初见他跪地垂泪、不忍驱赶，便任他定居白色小镇，漫无目的地等待、直至死去。他脚前的木牌写着:「我所追随与景仰的王者，您为何变成那等模样……您已不需要我的至高之作——光之王子民·宝石工匠·林德伯格」。

- **三件套的秘密**：[伯格之心](../../力量体系/装备道具/index.md)（不朽级·项坠·适应之力）被苏晓取走后，其简介揭示——宝石大师早年还造过 **白银之心·护符**（16 护符中最强的一枚）与 **不屈之心·戒指**，三件同时佩戴会触发特殊效果。取走项坠后，枯尸与整间屋子化为飞灰。

## 关系

- [苏晓](../苏晓.md)（白夜）：取走他遗作伯格之心、解开他死于白色小镇之谜的契约者。
- [光之王](光之王.md)：他毕生景仰、想献上至高之作却已无力对抗古神的王者。
- [月灵](月灵.md)：当初不忍驱赶他、任他在白色小镇终老的光之王王侍。

## 出处

- 第42卷6章（丹克市宝石大师·开创晶脂 + 超凡宝石提纯法并无私公开·遗作「伯格之心」·宝石街因他改名）；59–60章（真相=假死后寻光之王献伯格之心·见光之王破碎无力对抗古神→定居白色小镇等死·枯尸在密令「至高之作」的商铺内·遗作还有白银之心护符 + 不屈之心戒指三件套）。
