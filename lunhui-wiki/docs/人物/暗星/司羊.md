# 司羊（诚信商会天启商人）

[暗星](../../世界/任务世界/暗星-世界入侵.md) 里天启乐园「诚信商会」的商人——长相极猎奇（巴哈说他「有古神之姿」），却给人发自骨子里的谦逊感。他进暗星既收矿石、又卖资源开采消耗品 / 恢复药剂（第42卷24–25章）。他是 [苏晓](../苏晓.md) 收购不朽级宝石的卖家。

## 卖裂空 · 交易被截胡

- **压箱底的裂空**：苏晓在世界联络平台收购锋锐特性的不朽级宝石，司羊以啄木鸟机械 + 血肉分身谨慎交易，最终掏出压箱底的 **[裂空](../../力量体系/装备道具.md)**（不朽级·撕破空间·苏晓 5800 灵魂钱币 + 159 光银矿购得·给恶魔铁匠镶斩龙闪的第一颗宝石）。交易中他从光银矿认出「面具男抢天堂小队」的事、被苏晓反手编了个「粉色短发女劫匪」的假线索（栽赃、制造第二个矿产掠夺者假象）。
- **被嵌合体盯上**：交易约在小酒馆——那酒保竟是**嵌合体违规者**（同化的土著）。司羊的血肉分身被 [嵌合体违规者一伙](嵌合体违规者.md)（盯了诚信商会几个世界·抢过他 3 次）截走裂空、分身与两名护卫契约者「封 / 殇」皆被杀（尸体被丢到苏晓房顶当警告）。苏晓遂决定猎杀这伙违规者（顺带夺回裂空）。

## 关系

- [苏晓](../苏晓.md)（猎手 / 白夜）：卖他裂空、被他栽赃假线索的天启商人。
- [嵌合体违规者一伙](嵌合体违规者.md)：盯上诚信商会、抢他 3 次、杀其分身的违规者。
