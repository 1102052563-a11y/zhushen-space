# 嵌合体违规者一伙（酒保嵌合体 + 三违规者）

[暗星](../../世界/任务世界/暗星-世界入侵.md) 里一伙以抢劫 [司羊](司羊.md) 的诚信商会为业的**违规者四人组**——领头的「酒保」是罕见的**嵌合体违规者**，另有两男一女三名违规者（含一名长腿御姐·第42卷25–26章）。他们成了 [苏晓](../苏晓.md) 在暗星的猎杀目标。

## 嵌合体 · 罕见的违规者

- **三意识共身**：领头的酒保是「嵌合体」——一个身体里嵌着**虚空生物 + 天启违规者 + 天启矿工**三个（都是男性）意识，还能**同化本世界土著、取而代之**（故伪装成酒馆酒保）。苏晓在低阶时见过两只嵌合体（宰过·晋七阶后基本绝迹于轮回乐园高阶世界·虚空生物不敢来·全跑全开放·原生世界），觉得这伙违规者「清新脱俗」（居然只是抢商人、不去毁灭世界）。
- **抢司羊 3 次**：他们盯了诚信商会几个世界、有战斗天使在就不来、否则每次劫走货品。这次用咒印（12 小时不可收储存空间）锁住裂空、四违规者围杀司羊的血肉分身 + 护卫「封 / 殇」——把三具尸体丢到苏晓房顶当**警告**（「契约者而已、接触不到我们的世界、警告下就行」）。

## 四名成员 · 佩亚大剧院团灭（42卷26章）

苏晓一路追踪到东区废弃的佩亚大剧院，把四人**团灭**（放逐满场穿梭 + 巴哈异空间脱离·让四人的能力根本没机会施展）：

- **由雉**（御姐 · 露背装）：魅力系·粉红魅惑俘虏型控制——被苏晓一刀从脖颈穿透到侧腰（靠两件不朽级装备没当场死）、最后青鬼刀芒斩杀。
- **大背头**（色而不迷）：寄生弹枪械（子弹分裂成线虫）——被刃道刀·时 + 流瞬间斩首。
- **懒散男**（布帽）：红刺（剧毒减益）+ 红光盾牌——被苏晓一脚直踹掀翻剧院房顶、削掉右手，终被斩首。
- **嵌合体酒保**：巨怪化（4 米+）——被巴哈扯出脊柱 + 脑内注毒 + 抹脖斩杀。

苏晓夺回 [裂空](../../力量体系/装备道具.md)（带咒印·11 小时 14 分内无所有权 / 不可入储存空间）。因他的天启临时烙印**完全拟态成功**（100% 伪装天启方），杀这 4 名违规者被判为天启乐园战功——获奖励（超凡植物种植卷轴 + 星陨熔炉）+ 被邀「战斗天使考核」（苏晓两次拒绝·怕露馅）。

## 关系

- [司羊](司羊.md)：被他们抢了 3 次、分身被杀的诚信商会商人。
- [苏晓](../苏晓.md)（猎杀者 / 白夜）：被丢尸警告、遂决定反猎他们（顺带夺回裂空）的猎杀者。
