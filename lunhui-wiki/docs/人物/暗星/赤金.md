# 赤金（天启乐园 · 科技装甲保镖）

[暗星](../../世界/任务世界/暗星-世界入侵.md) 里天启乐园的战斗系契约者——[天堂小队](天堂小队.md) 矿工雇的保镖（每天收 200 灵魂钱币还嫌无聊）。他是靠**科技动能装甲**硬顶到八阶的典型「养出来的」契约者，实战能力却很欠缺（第42卷12章）。

## 科技流 · 中看不中打

- **一身赛博武装**：动能铠甲（拼接头盔 + 游戏化标注视角）、干扰脉冲防感知锁定、雷电球（高压追踪）、追踪型能量导弹群、「压冲磁暴 / 感应磁场 / 立场分解粒子」三立场——火力华丽。但苏晓一眼看穿：「像赤金这样变强的契约者，遇到苏晓这种实战派，完全是噩梦」（他不去全开放·原生世界、更进不了世界争夺战）。
- **被苏晓一脚踹爆**：苏晓戴白金属面具、赤膊、持史诗级中短棍 [罪落天遗](../../力量体系/装备道具.md)（换掉刀的战斗风格）——硬抗其攻击、绕背一脚直踹踹爆他后腰装甲（身躯反 C 形、脊椎险断），再一棍抽落。他引爆能量核心自爆，被苏晓**徒手用简易吞噬之核网压制爆炸**；随后苏晓一棍棍砸（棍不能斩首、没击杀提示就不停）把他砸死。「坦系怎么可能有这种攻击强度……」——他至死怀疑人生。

## 意义

苏晓杀他后**同化其烙印**、获得天启乐园临时烙印（120012 号·交易 / 契约权限 + 世界联络平台），从而能混入天启乐园方契约者中。

## 关系

- [天堂小队](天堂小队.md)：雇他当保镖的矿工队。
- [苏晓](../苏晓.md)（白夜·白金属面具）：一脚踹爆他装甲、砸死他并同化其烙印的实战派。
