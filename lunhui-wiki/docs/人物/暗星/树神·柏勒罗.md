# 树神·柏勒罗（三分之二古神 · 封印「过滤网」）

[暗星](../../世界/任务世界/暗星-世界入侵.md) 封印体系里的一尊**「三分之二古神」**——身高 2.5 米、头顶一排眼睛、口部气孔、全身垂着毛发质感的黑色树根。它不是真古神，却在 [光之王](光之王.md) 苏醒时第四次「越狱」冲出、撞上 [苏晓](../苏晓.md) 小队（第42卷15章）。

## 身世 · 背锅的伪古神

- **被黑泥浸成伪古神**：它本是「中立·恶」阵营的神灵、原世界被足以淹没大陆的黑泥（黑暗物质）灌下、醒来已在暗星、被泡成三分之二古神——**永远无法拥有吮吸世界的能力**。它化参天巨树、放波动吸引智慧生灵来树下、被树根寄生陷入「花天酒地 / 无所不能」的美好幻觉、实则被树根吸干生命。
- **封印的「过滤网」**：真古神降临后三王造容器封印，先拿它试封（残缺容器 → 最强容器都在它身上试验成功）。封印分三层：容器（核心）+ 光之王 / 大贤者 / 狼蛛女王（外层魂链）+ **树神（夹在核心与外层间当「过滤网」·过滤容器漏出的微量古神能量）**。它过滤 + 吸收了百年、憋出一身庞大的古神能量（故初见气息惊人、实战却极弱）。

## 越狱专业户 · 被秒杀

它四次越狱都很惨：①被灵魂钟塔百余灵魂使者自爆炸残 ②被 [月灵](月灵.md) 带小镇魂灵打回 ③被 [恶魔铁匠](恶魔铁匠.md) 差点弄死（挺尸几十年）④这次趁光之王苏醒冲出、撞上苏晓的**专业猎神流**（抗侵蚀药剂 + 三心脏高震频脉冲器令其失神 + 界断线魂之丝 + 阿姆冰封 + 斩龙闪斩首）当场被制服，被月灵用刃枪挑起、化树根塞回光之王体内的封印外层。真古神却因它的「勇敢」破例传回精神波动、称赞了它。

## 🆕越狱后「做个好神」（42卷71章）

羽神破封时，它趁封印裂缝偷偷逃了出来，本想累积力量、报复那些小看它的人；却发现昔日盟友 [母神](母神.md) 的神宫已成废墟（判定母神被灵魂钟塔所害·萌生退意）。它来到大陆西南一片荒原生根蓄力——正在此时它**感应到羽神·赫格拉陨落了**。连那么强大的真古神都会陨落？它成为古神的野心当场动摇，回想起被母神击败、被大贤者逮住、被光之王 / 狼蛛女王封印、被月灵揍、被恶魔铁匠一锤差点灰飞烟灭、又撞上专业古神猎人的种种惨事，它树干一颤、树叶掉落——**忽然觉得还是安安分分做一棵树最安全，以后要「做个好神」，当恶神太危险了、还总被欺负**。它是本世界这场浩劫里少数活到最后、且「改邪归正」的存在。

## 🆕真相 · 召唤羽神的共谋（42卷73章·揭示）

苏晓通关后探知:当年正是**母神与树神合谋召来羽神**——母神想借外援除掉三王复辟，树神则想吞噬一具真古神的神躯、成为真正的古神，二者一拍即合布下召唤仪式。结果引狼入室、两个都后悔莫及（请来的不是打手、而是能俯视它们的「超级大爹」）。这场灾祸的源头，树神也有一份。

## 关系

- [苏晓](../苏晓.md)（古神猎人）：被其专业猎神流秒杀的伪古神。
- [光之王](光之王.md) / [月灵](月灵.md)：把它反复塞回封印外层的封印者 / 王侍。
- [母神](母神.md)：把它打半死、又与它合谋召唤羽神的旧盟友（它一度视其为标杆）。
- [羽神·赫格拉](羽神·赫格拉.md)：它与母神合谋召来的真古神（陨落后它吓得决定做个好神）。
