# 母神（暗星孕育之神）

[暗星](../../世界/任务世界/暗星-世界入侵.md) 的土著神灵——**孕育一切之神**，本体是一大团暗紫色、缓慢涌动的臃肿肉团（苍白宫殿画像题「孕育一切之母」）。她与古神死战大半身躯被腐化险陨，后由三王二次救回；她提出计划、与 [光之王](光之王.md) 交合产下封印古神的「容器」（第42卷23章）。

## 装虚弱看戏 · 想低成本分好处

她全程「虚弱」得插不了手古神之战——实则是**想以最低代价、坐等 [苏晓](../苏晓.md) 杀掉古神后分一杯羹**。她原以为来对付古神的会是「受世界眷恋之人」（随便安抚鼓励即可），见了苏晓才发现根本不是（估测苏晓的危险程度与古神相差不多、心里其实很想让苏晓与古神同归于尽）。

## 被苏晓翻脸威胁（42卷23章）

苏晓不做打工仔——他**人造腐化之力**（用残旧容器收集）当投名状拜访母神，母神验货后放喜悦波动、却在「灵魂石需要时间送」上想拖。苏晓当场翻脸：走到门口环顾、说「我在确定这是哪、然后找到这弄死你」——母神吓得立刻掏出全部灵魂石（**172 颗完整灵魂结晶 + 灵魂晶核 + [扭曲生命铭文](../../力量体系/装备道具.md)**〔不朽级〕）。苏晓借残旧容器（说日后或能盛古神之血）套出母神的真实态度，随即精神体撤回本体。苏晓给母神三选一：①一同对付古神 ②拿出让他满意的好处 ③永远别出现、别想看戏分羹。

## 🆕神性血肉 = 晶脂的核心原料（42卷58章·揭示）

诺厄主教向苏晓道破晶脂的秘密时，牵出了母神的另一重「贡献」:全暗星赖以抗腐化的硬通货 **晶脂** = 灵魂能量（打碎灵魂结晶抽取）+ **母神提供的神性血肉** + 上个文明遗留的生命力结晶体（让二者完美融合）。真正让晶脂能抵御腐化的，主要就是**母神的神性血肉**——她为此**自愿被斩断身体几十次、不断再生**供料。这也说明她在暗星的势力网里远比「装虚弱看戏」看上去更深地嵌在众生的生存命脉中。

## 🆕被异化 · 沦为羽神战力（42卷67章）

羽神破封、退守梦境世界后，一向龟缩看戏的母神竟出现在古神一方——她被异化成一个**全身血肉触须、每根触须末端带弯曲刀刃**的人形怪物，站在幽暗广场拦截苏晓一行（三名强敌中数她最危险，由异端处刑队 [处刑队长](异端处刑队.md) 迎战）。苏晓判断她本不该直接站羽神那边，多半是**被灵魂钟塔坑了、或被大贤者算计**，才落得这般被古神能量异化、沦为战力的下场——「装虚弱看戏」到头来自己也没能置身事外。

## 🆕降临之谜 · 一切恶的起始（42卷73章·揭示）

苏晓通关后探知的真相，把母神推到了「**恶的起始**」的位置:她本是本世界唯一的神灵、万众信奉，却被后起的光之王 / 大贤者 / 狼蛛女王窥觊神位、信仰急剧枯竭。她不甘就此沉寂陨落，又打不过三王（顶多和狼蛛女王平手），便想制造一场危机、逼三王去解决、再趁其油尽灯枯时出手复辟。她先把冒牌古神 [树神·柏勒罗](树神·柏勒罗.md) 打了个半死（嫌它太弱），又被树神忽悠「还有更强的古神、需召唤请来」，遂与树神合谋布仪式——恰逢真古神 [羽神·赫格拉](羽神·赫格拉.md) 正躲避冥神追杀、顺势应召入世。结果是**引狼入室**:她以为古神「只是战力强」，却被羽神一击击败关起来慢慢吞噬，四王联手也被吊打、只能造容器封印。世界简介早已点破「不要被表象所蒙骗，坚信之物，可能才是恶的起始」——被万民信奉的母神，正是这场灭世灾祸的源头。决战中她被异化成血肉触须怪、最终被处刑队长拼死吸收殆尽。

## 关系

- [光之王](光之王.md)：与她交合产容器的人族至强者（三条锁链之一·当年也窥觊她的神位）。
- [苏晓](../苏晓.md)（弑神之人）：她想坐收其杀古神之利、反被翻脸威胁交出好处的猎杀者。
- [树神·柏勒罗](树神·柏勒罗.md)：被她打半死、又与她合谋召唤羽神的冒牌古神。
- [羽神·赫格拉](羽神·赫格拉.md)：她亲手引入本世界、反把四王吊打封印的真古神。
- [大贤者](大贤者.md) / [灵魂钟塔](灵魂钟塔.md)：疑似坑她 / 算计她、致她被异化沦为羽神战力的古神阵营。
