# 伏恩医生（杜瓦·伏恩 · 开洞医生）

[暗星](../../世界/任务世界/暗星-世界入侵.md) 旧王都·塔洛「双月街·杜瓦诊所」的凶名医生，全名**杜瓦·伏恩**——两大「高超」疗法闻名:内部疾病**放血**、外部疾病**截肢**。他更出名的身份是「**开洞医生**」:替 **灵魂钟塔** 在受害者头部开洞（切一块脑组织），以助外放灵魂力量——被开洞者大多会加入灵魂钟塔（「看到有脑洞的，基本就是灵魂钟塔成员」）。他本人不算灵魂钟塔的人，只是收晶脂替对方做手术。

## 神经质 · 自我循环的怪物

原本自律，却在目睹「**古神之子降生事件**」（古神借一名女性降下婴孩、刚降生就变怪物在旧王都大肆破坏、致人口锐减十分之一）后对一切无所谓、变得神经质、邋遢。他偷藏了那次事件里的**一小段脐带**私下研究，某夜梦见古神黑影、只看一眼精神就几近崩溃。此后他**脸上会长出黑色触手**（用剃须刀刮下）、甚至开始**吃自己身上长的黑触手**（一种诡异的「自我循环」·吃后 5~10 天不再长、耳中会响起奇怪声音）——他已不只是「古神化」，而是变成了更匪夷所思的东西：**近乎不死**（被 [苏晓](../苏晓.md) 一刀斩首，断颈生触手把头拽回、瞬间愈合无痕），战力却只五阶。

## 被苏晓生擒布仪式（42卷54–55章）

苏晓摸清他是「古神之子降临」的知情人后，进诊所试刀（随手斩首验证其不死）、一拳打晕装袋拎走（诊所几十名灵魂钟塔暗哨被布布汪探 + 巴哈突袭 + 月灵收尾清掉）。苏晓给他两个选择:配合布设古神之子降临仪式，或成为古神之子的**耐受体**（男性也会出现「孕育」特性→「伏恩女医生」）。为了不变成「伏恩女医生」，他豁出去配合，还交代出自己藏在哨光废墟的那截脐带（仪式关键）。他一路废话连篇（被苏晓一拳打晕两次），最终帮苏晓在丹克市郊外古堡勉强布成残次品古神之子仪式。

## 关系

- [苏晓](../苏晓.md)（猎神者 / 白夜）：一刀验证其不死、生擒逼他布古神仪式的猎神者。
- [灵魂钟塔](灵魂钟塔.md)：长期雇他「开洞」的古神走狗势力（诊所有其几十名暗哨）。
- [摩利兰](摩利兰.md)：「古神之子二次降临事件」的关联者（诺厄主教靠伏恩这条线索牵出真相）。
