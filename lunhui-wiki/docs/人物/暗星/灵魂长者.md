# 灵魂长者（灵魂钟塔的领袖）

[暗星](../../世界/任务世界/暗星-世界入侵.md) [灵魂钟塔](灵魂钟塔.md) 的领袖——一名鹰钩鼻老者，**头部开有八个洞**（[苏晓](../苏晓.md) 见过开洞最多的人）。他是 [羽神·赫格拉](羽神·赫格拉.md) 眼中除 [大贤者](大贤者.md) 外、少数「值得保留的、强壮些的蝼蚁」之一（其余众生皆蝼蚁）。作为研究、扭曲灵魂的疯狂学者之首，他把整个灵魂钟塔带成了羽神的走狗。

## 梦境世界决战（42卷67章）

羽神破封后退守大贤者经营上百年的**梦境世界**，灵魂钟塔与大贤者势力都集结于此恭迎古神。苏晓率科多学派 27685 人杀入梦境世界乱战时，灵魂长者与大贤者、被异化的 [母神](母神.md) 一同挡在通往大教堂的幽暗广场——他开口便讥讽 [诺厄主教](诺厄主教.md)「科多·费加曼迪，臭虫永远是臭虫，就算活了几百年也只是老不死的臭虫」（却忘了身旁的大贤者也苟了几百年）。他点名要与诺厄主教一对一，苏晓遂安排诺厄 + [月灵](月灵.md) 联手除掉他（诺厄轻易不能死、否则科多成员会溃逃），自己继续前往大教堂寻羽神。

## 🆕被诺厄故意放走（42卷72章）

梦境世界决战他在 [月灵](月灵.md) + [诺厄主教](诺厄主教.md) 的围攻下「逃」了——实则是**诺厄故意放跑他**。「狡兔死走狗烹」:若灵魂钟塔当场被全灭，损失惨重的科多学派不出一个月就会被其他势力联合击垮；留着灵魂钟塔这只「过街老鼠」当死敌，科多学派才能借「扫平灵魂钟塔」的名头吸纳无阵营强者、稳住阵脚、长久崛起。他撤退时把大贤者的野兽族当炮灰断后（野兽族全灭）、收拢了灵魂钟塔八成战死后的残部。羽神已陨落，他这条古神走狗失了主心骨，却仍将作为科多学派需要的「反派」继续存在下去。

## 关系
