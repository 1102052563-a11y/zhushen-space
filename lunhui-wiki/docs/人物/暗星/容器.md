# 容器（封印羽神的王之子）

[暗星](../../世界/任务世界/暗星-世界入侵.md) 由 [光之王](光之王.md)、[大贤者](大贤者.md)、[狼蛛女王](狼蛛女王.md)、[母神](母神.md) 协力造出的**容器**——用来封印古神 [羽神·赫格拉](羽神·赫格拉.md) 的躯壳，[月灵](月灵.md) 之外白色小镇诞生的另一个「王之子」。它身高三米以上、高瘦、头生外骨骼（形如向后弯的王冠），持一把朴素却更锐利致命的刃枪 **低语**（恶魔铁匠所造·仿青钢影特性）。它**诞生之初就被夺走了思考能力**，封印羽神几百年后已被古神能量**彻底腐化**（面甲三道裂痕、亮黄瞳焰），陷入只有本能杀戮、无情感无痛觉的混乱。本世界 boss 级战力它稳居第二（仅次羽神）。

## 三王毕生本领的集合体

苏晓侦测其资料:力 217 / 敏 217 / 体 215（真实属性）、智 15（被侵蚀）、魅 1（原 108 被侵蚀）、古神灵能量 34000。它是光之王、大贤者、狼蛛女王毕生本领的集合:

- **王之技**（光之王传授）：长枪攻击 +755%、附高阶穿透、已衍生 16 种能力。
- **野兽之魂**（大贤者培养）：生命 +37000、能量侵蚀伤害 −92%、钝击伤 −50%。
- **狼蛛本能**（狼蛛女王教）：近乎预感未来 0.5 秒，成功「捕捉」敌方攻击后 2 秒内移动 / 反应速度 +60%。
- **神灵后裔**（母神遗传）：生命 +25400、身体速度 +0.7 倍、**断肢可再生**。
- **腐化·蔓延**（终极）：自刺胸膛喷出最浓腐化之力，350 米地面每秒 245 腐化伤、削治疗 95%。

## 被苏晓斩杀 · 临终的情感（42卷61–64章）

破封后它与羽神一同放出——因是无情感容器，连羽神都控制不了它，它只被腐化陷入混乱。经「魅力属性生死交锋」（苏晓魅 −3 < 容器魅 +1，容器选择冲向苏晓），它与苏晓在苍白宫殿 → 锻造铺 → 塌陷出的**地下宫殿**（它的诞生地·十几万容器面甲堆墙 + 三王石雕）一路缠斗。苏晓靠「刃道刀·时」抵消它的加速、界断线魂之丝缠困、血之兽钻进它头颅内爆炸，终将它重创。它临死时不是逃、而是费力爬向王座，抓起一块 [光之王](光之王.md) 的躯体残片按在自己眉心——**从这一刻起它不再是完美的容器，有了某种情感**。苏晓一刀贯穿其头颅斩杀（得 20.57% 世界之源 + [容器核心](../../力量体系/装备道具.md)），叹一句「并不完美的容器」。

## 关系

- [苏晓](../苏晓.md)（白夜）：斩杀它、让它临终得情感的猎神者。
- [光之王](光之王.md) / [大贤者](大贤者.md) / [狼蛛女王](狼蛛女王.md) / [母神](母神.md)：造它、把毕生本领灌入它的三王与母神。
- [月灵](月灵.md)：白色小镇另一个「王之子」（得光之王悉心教导与溺爱，与容器诞生即失一切的命运截然相反）。
- [羽神·赫格拉](羽神·赫格拉.md)：它封印了几百年的古神（破封后连羽神都控制不了这无情感的它）。
