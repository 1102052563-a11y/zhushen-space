# 恶魔铁匠（白色小镇锻造铺 · 恶魔族）

[暗星](../../世界/任务世界/暗星-世界入侵.md) 白色小镇锻造铺里的**恶魔族**铁匠——两米高、暗红粗糙皮肤、公羊角、眸中似有岩浆燃烧、肌肉如千锤百炼的精钢。他来历大到惊人、在暗星不知待了多少年，是 [苏晓](../苏晓.md) 意外撞见的**灭法者时代的旧盟友**（第42卷14章）。

## 来历 · 对灭法者的愧疚

- **至尊锋刃的顾问**：当年开发 [至尊锋刃](../../力量体系/技能/index.md) 能力的灭法者曾请他去提供意见——他一眼认出苏晓的 [斩龙闪](../../力量体系/装备道具/index.md) 带至尊锋刃、青钢影是「[马文·华尔兹](../../力量体系/技能/index.md) / 格林·吉莉安那一代（最完善、强大、也最不讲道理的一代）」的传承。恶魔族与灭法者世代结盟（灭法者末期就那几人生死与共、惊人团结）。
- **落寞的隐退**：他自称「和恶魔族 / 灭法者 / 奥术永恒星的恩怨早就了结、只是个想死死不成、活着没目标的铁匠」（大人物沃波尔的旧识）。看到苏晓会让他想起逝去的灭法者故人，遂请苏晓离开。苏晓察觉他对灭法者的**愧疚与歉意**——这次「天上掉馅饼」正是他对旧事的补偿。

## 成就任务「曾经的友谊」（42卷14章）

- 他嫌斩龙闪上的宝石太差，许诺：苏晓离世前若能弄来 **3 颗高于圣灵级品质、且适合斩龙闪的宝石**，他就**无偿**给斩龙闪镶嵌、且镶嵌效果在原基础上**额外 +30%**；完成后还会赠苏晓一件未知物品、今后偶遇愿协助。若失信（离世前未凑齐 3 宝石），他将视苏晓为失信、永不相助、并离开本世界去星海尽头。
- 他还把书屋里几十本锻造书（他带来存放的·虚空特殊文字）送给苏晓（苏晓转赠给 里德 提升锻造）。

## 关系

- [苏晓](../苏晓.md)（灭法之影 / 白夜）：他愧疚补偿的对象（末代灭法者传承者·成就任务「曾经的友谊」）。
- **沃波尔**：恶魔族的老不死大人物、他的旧识。
- **马文·华尔兹 / 格林·吉莉安**：苏晓青钢影所属那一代灭法者（他的旧故人）。
