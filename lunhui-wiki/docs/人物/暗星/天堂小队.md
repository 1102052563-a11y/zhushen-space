# 天堂小队（天启乐园矿工队）

[暗星](../../世界/任务世界/暗星-世界入侵.md) 里一支天启乐园的**矿工小队**——在迷失枯林附近地下 1240 米挖圣灵级「光银矿」（用天启消耗品「液流质」开拓地下空间·雇 [赤金](赤金.md) 当保镖）。他们是 [苏晓](../苏晓.md) 眼中的「野生资源」（第42卷12–13章）。

## 成员

- **王子**：金发帅气的年轻队长（代号是队友叫出来的）。雇赤金图便宜、见苏晓砸死赤金后立刻撤屏障投降（「大哥你想要什么只管说」）。
- **贝儿**：戴矿工帽的腼腆矿工妹（「这种妹子轮回乐园内已经绝种了」）——被苏晓平静含杀意的目光吓得偏头。
- **老鱼**：苦口婆心提醒「杀矿工有收益衰减惩罚」，被苏晓一眼瞪回。
- **乌面男**：队中一员。

## 被苏晓「养肥再宰」

苏晓砸死保镖赤金后**同化烙印**得天启乐园临时身份，遂改变计划：只抢走矿石（**光银矿 ×165**·圣灵级·价值约 7590 灵魂钱币）、给王子肩上悄悄种下**蓝色追踪印记**、放他们继续挖矿——等养肥了再来收割一次，「双重的收益，岂不是双倍的快乐」。程序上还是照例一人一闷棍走过场。

## 🆕再被收割 · 主动求签契约（42卷40章）

被养肥后，苏晓循着王子肩上的追踪印记又找上门「探访」。这次天堂小队学乖了、把地表几百米内的矿洞都封住转挖二号矿脉、还新雇了八阶水系法系保镖 [海吉尼](海吉尼.md)（贪婪·坐地起价 + 强行入伙）——结果海吉尼被苏晓一棍砸残、用道具逃走。矿石照旧被夺。

有趣的是，挨完闷棍的王子居然**主动提出合作**、各占 50%、还坚持要**签一份实体契约**（他发现这片矿脉隐秘、除苏晓没别人来抢，签约还能拴住跑路的海吉尼）。苏晓反问「你确定？」，王子异常肯定。他四人用放大镜仔细检查那张羊皮纸契约——却不知**这契约羊皮纸能揭开五层，第 1~4 层完全正常，第 5 层另有玄机**（苏晓在契约里埋了伏笔）。

## 关系

- [赤金](赤金.md)：他们雇的、被苏晓砸死的保镖。
- [海吉尼](海吉尼.md)：他们后来新雇的贪婪水系法系保镖（被苏晓一棍砸残）。
- [苏晓](../苏晓.md)（白夜）：抢他们矿石、种印记养肥再宰、又被王子主动拉着签下五层玄机契约的「野生资源」收割者。
