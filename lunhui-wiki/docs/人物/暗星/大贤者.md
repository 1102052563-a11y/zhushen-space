# 大贤者（野兽的智者）

[暗星](../../世界/任务世界/暗星-世界入侵.md) 野兽族崇敬的**大贤者**——手握木杖、身披粗麻布袍，栖身一片枯萎森林。他是封印古神三条灵魂锁链的**中层**（光之王最里 → 大贤者 → 狼蛛女王最外），也是 [苏晓](../苏晓.md) 破封要杀的第二个目标（第42卷21章）。

## 中立派 · 「别信他说的话」

- **主张苟住**：容器裂痕后三王分歧中，他主张**稳住局势、静观其变、保持中立**——二次培养容器成功率太低（上次交合光之王险死）、与古神决战更鲁莽（最辉煌时都没打过），尽量苟住给三族强者争取时间。他为二次救母神付出的代价是**不再能聆听自然**。
- **可疑的警告**：他感应到 [狼蛛女王](狼蛛女王.md) 逝去、只淡淡道「这几百年辛苦你了、蛛女王」便隐入枯林。狼蛛女王的临终遗言却是**「小心大贤者，别相信他说的话」**——他这中立老智者藏着什么，成了苏晓破封第二环的悬念。当年正是他找到 [树神·柏勒罗](树神·柏勒罗.md)、发现它也是「古神」、促成用它试封容器。

## 关系

- [苏晓](../苏晓.md)（古神猎人）：破封第二个要杀的目标。
- [光之王](光之王.md) / [狼蛛女王](狼蛛女王.md)：与他共同封印古神的另两条锁链（女王临终警告别信他）。
- [树神·柏勒罗](树神·柏勒罗.md)：当年被他找到、用于试封容器的伪古神。
