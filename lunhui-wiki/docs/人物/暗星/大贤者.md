# 大贤者（野兽的智者）

[暗星](../../世界/任务世界/暗星-世界入侵.md) 野兽族崇敬的**大贤者**——手握木杖、身披粗麻布袍，栖身一片枯萎森林。他是封印古神三条灵魂锁链的**中层**（光之王最里 → 大贤者 → 狼蛛女王最外），也是 [苏晓](../苏晓.md) 破封要杀的第二个目标（第42卷21章）。

## 中立派 · 「别信他说的话」

- **主张苟住**：容器裂痕后三王分歧中，他主张**稳住局势、静观其变、保持中立**——二次培养容器成功率太低（上次交合光之王险死）、与古神决战更鲁莽（最辉煌时都没打过），尽量苟住给三族强者争取时间。他为二次救母神付出的代价是**不再能聆听自然**。
- **可疑的警告**：他感应到 [狼蛛女王](狼蛛女王.md) 逝去、只淡淡道「这几百年辛苦你了、蛛女王」便隐入枯林。狼蛛女王的临终遗言却是**「小心大贤者，别相信他说的话」**——他这中立老智者藏着什么，成了苏晓破封第二环的悬念。当年正是他找到 [树神·柏勒罗](树神·柏勒罗.md)、发现它也是「古神」、促成用它试封容器。

## 🆕暗中干扰 · 忽悠苏晓被识破（42卷27–29章）

大贤者苟到极限（本体从不露面）——他暗中动手脚干扰虫塔传送，把苏晓「请」到野兽领地一处破损祭坛旁，占据一条黑蟒的身体现身，抛出一套「真相」忽悠苏晓：

- **「母神才是敌人」的说辞**：他讲母神原是俯瞰众生的至高神灵、被王者 / 贤者 / 女王比肩后终结了她的统治；母神遂**引来古神灾祸、想搞垮三王让他们与古神两败俱伤、再自己灭古神独占信仰**——三王封印古神是「受骗」、母神才是真敌人、「我与光之王才是你的盟友」；还说**撤下魂链也能开封印、不必杀三王**（那是母神的谎言）。他劝苏晓「别成为他人手中的刀子」、先杀母神取神源。
- **被苏晓当场识破**：苏晓看穿这套说辞是「放屁」——那祭坛不是母神布设、而是**科多学派召唤阴影古神的地方**（在野兽领地召唤·免得毁自己地盘）；狼蛛女王「大贤者的话一个字都别信」的遗言是对的。更可疑的是：光之王 / 狼蛛女王都生不如死、唯独大贤者**生存欲极强**（像想当「母神 / 光之王 / 古神都死后的最后赢家」·野兽族百年前的内部分裂疑是他挑起）。苏晓一刀斩落黑蟒头、踩碎——他决定去厄罗河以内「最后的绿地」找默洛斯（默洛斯带走的神女血对古神能量高度适应·与大贤者疑有牵连）查真相。

## 🆕魂链已被替代 · 死守梦境世界失败（42卷30–39章）

主线第三环揭示了大贤者「特别能苟」的真相：**它那条封印古神的魂链，早已被抽离、由 [次代神女·沙塔耶](次代神女·沙塔耶.md) 的魂链替代**缠在容器外——所以直接被古神能量侵蚀的已不是它，它当然主张维持现状、静观其变。它靠「牺牲自身封印古神」的传说在野兽族中积累了**难以想象的威望**，凭威望（而非强权）统治半个野兽族；一旦封印被打开、只剩它活着，威望就会崩塌，故它急需现在的威望去完成某件事。

- **梦境世界的老巢**：它把「最后的绿地·梦境世界」经营成囚困野兽族的牢笼——连坑带骗把大量野兽族弄进这永不衰老的循环空间、再作**灵魂改造**（半透明灵魂丝线可让断体重连、疑与「灵魂钟塔」势力勾结）。苏晓一到，它就叫醒 [老骑士](老骑士.md) 与沙塔耶守大教堂、命全体野兽族围杀闯入者。
- **被苏晓的「团灭套餐」捶到自闭**：苏晓用 阿波罗 + 吞噬者 + 哈迪斯的组合，在梦境世界内**炸死、烧死三万两千余名野兽族**（32600+，大贤者积累多年的力量近乎折半）。大贤者气得握杖咔咔作响、却始终**不肯出手**——它有一记**积蓄了 319 年的一击**，是专为古神所留（一击之后所有累积消耗一空，「要么苟到地老天荒、要么一击天崩地裂」）。最终它连老巢都不要了，从一道地洞撤走、带走剩余野兽族（续科多学派之后，它也被苏晓捶到「自闭」）。
- **一生算计默洛斯夫妇**：它当年害死 [默洛斯](默洛斯.md)、[玛西亚](玛西亚.md) 的孩子、又利用这对丧子的夫妻为其驱使（甚至逼他们绑上代神女取血）——二人识破后暗中倒戈，借假死把「骨戒」交给苏晓复仇。它撤离前疑派野兽族清剿这对夫妻（未果）。
- **精神层投影**：它的**无意识投影**存在于梦境世界的精神层面，被送死队用「吸雾鸟」驱散。

## 关系

- [光之王](光之王.md) / [狼蛛女王](狼蛛女王.md)：与他共同封印古神的另两条锁链（女王临终警告别信他）。
