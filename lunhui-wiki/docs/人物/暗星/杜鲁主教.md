# 杜鲁主教（科多学派的激进派）

[暗星](../../世界/任务世界/暗星-世界入侵.md) 科多学派的另一位主教，全名**杜鲁·格费**——与保守派的 [诺厄主教](诺厄主教.md) 分属不同派系：诺厄主张以丹克市为中心建庇护所防古神破封，杜鲁则要向外扩张、壮大科多学派势力范围（激进派）。两人相识几十年，都是老狐狸。

## 半本笔记 · 骨戒真身（42卷41章）

诺厄主教召回 [异端处刑队](异端处刑队.md)、又暗中与 [苏晓](../苏晓.md) 交易半本古神能量研究笔记的事走漏，杜鲁孤身找上门——他不是来兴师问罪，而是**冲着那本笔记来的**。他以「相识几十年、年纪大了会健忘、回总部或许就不记得今天的事」暗示诺厄交出笔记；诺厄会意，把笔记扔在桌上。

**关键伏笔**：杜鲁抬手拿笔记时袖口滑落，露出食指上戴着的一枚 **骨戒**——而苏晓后来验证:凡戴骨戒者，无论属于哪方势力，全都是**古神的真正信徒**（[见骨戒](../../力量体系/装备道具.md)）。这意味着身为科多学派主教的杜鲁，实为暗藏的古神真信徒、是苏晓的敌人。他拿走了那半本关于古神能量的笔记。

## 关系

- [诺厄主教](诺厄主教.md)：科多学派同僚、分属保守 / 激进两派的老对手（被他要走半本笔记）。
- [苏晓](../苏晓.md)（白夜）：他戴骨戒 = 古神真信徒 = 苏晓的潜在敌人（拿走了苏晓的半本古神笔记）。
