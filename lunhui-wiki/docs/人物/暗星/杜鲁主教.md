# 杜鲁主教（科多学派的激进派）

[暗星](../../世界/任务世界/暗星-世界入侵.md) 科多学派的另一位主教，全名**杜鲁·格费**——与保守派的 [诺厄主教](诺厄主教.md) 分属不同派系：诺厄主张以丹克市为中心建庇护所防古神破封，杜鲁则要向外扩张、壮大科多学派势力范围（激进派）。两人相识几十年，都是老狐狸。

## 半本笔记 · 骨戒真身（42卷41章）

诺厄主教召回 [异端处刑队](异端处刑队.md)、又暗中与 [苏晓](../苏晓.md) 交易半本古神能量研究笔记的事走漏，杜鲁孤身找上门——他不是来兴师问罪，而是**冲着那本笔记来的**。他以「相识几十年、年纪大了会健忘、回总部或许就不记得今天的事」暗示诺厄交出笔记；诺厄会意，把笔记扔在桌上。

**关键伏笔**：杜鲁抬手拿笔记时袖口滑落，露出食指上戴着的一枚 **骨戒**——而苏晓后来验证:凡戴骨戒者，无论属于哪方势力，全都是**古神的真正信徒**（[见骨戒](../../力量体系/装备道具/index.md)）。这意味着身为科多学派主教的杜鲁，实为暗藏的古神真信徒、是苏晓的敌人。他拿走了那半本关于古神能量的笔记。

## 🆕腐化骤停的惊惧（42卷56章）

苏晓布仪式**暂时驱除全世界腐化**、科多学派借势洗白那日，五名主教志得意满，唯独杜鲁主教心中如遭五雷轰顶——他想不通「至高的羽神为何会离开这个世界 / 出了什么事」，自从他戴上骨戒加入古神阵营就已没有回头路。直到几小时后腐化重新降临，他脸色大变、悬着的心却放了下来（暗喜「没选错·羽神的恐怖无人能敌」）。他不知道:腐化并非自行恢复，而是苏晓故意关闭古堡阵图、再借此向 [诺厄主教](诺厄主教.md) 敲诈资源的一环。

## 🆕收网反被反杀 · 装进铁铸女（42卷65章）

羽神破封后杜鲁第一时间感应、开始收网——他多年布局、把心腹安插进科多学派各要害部门，顺利肃清、一并铲除了三名主教，正得意地找 [诺厄主教](诺厄主教.md) 这老对手炫耀他多年的隐忍时，事态剧变:他的**所有心腹竟全部反叛**、将他拿下押到诺厄面前。原来诺厄（真身老不死科多·费加曼迪）早已把他的心腹一一策反。诺厄只回一句「杜鲁，你和小时一样，没任何长进」，把他装进科多学派专门对付叛徒的「**铁铸女**」（刺满铁钉的铁棺）、押往白色小镇——他败得彻彻底底。

## 关系

- [诺厄主教](诺厄主教.md)：科多学派同僚、分属保守 / 激进两派的老对手（要走半本笔记、又策反他的心腹把他装进铁铸女）。
- [羽神·赫格拉](羽神·赫格拉.md)：他暗中效忠的封印中古神（腐化骤停时他为羽神安危惊惧）。
- [苏晓](../苏晓.md)（白夜）：他戴骨戒 = 古神真信徒 = 苏晓的潜在敌人（拿走了苏晓的半本古神笔记）。
