# 灵魂钟塔（研究灵魂的疯狂学者）

[暗星](../../世界/任务世界/暗星-世界入侵.md) 的一大势力——一群研究、扭曲灵魂的**疯狂学者**集团。原本听命于 [光之王](光之王.md)、甚至去白色小镇朝圣，如今**已彻底沦为羽神 [赫格拉](羽神·赫格拉.md) 的走狗**（首领被称「灵魂长者」，是羽神眼中「值得保留的强壮蝼蚁」）。

## 「猎食」旧王都 · 开洞入教

- **暗中盘踞旧王都**：当年它们在旧王都布「首次古神之子降临仪式」失败、弄出的怪物大肆破坏致人口锐减十分之一，遂对外宣称「新容器失败」搪塞、明面撤出旧王都，实则把这里当**大本营**暗中「猎食」——它们不要繁荣也不要秩序，只要有平民在此低度腐化地苟活，失踪人口就远超饥饿 + 疾病 + 腐化之和。
- **对脑组织的执念**：它们靠 [伏恩医生](伏恩医生.md)（开洞医生）在受害者头部**开洞、切除脑组织**以外放灵魂力量——被开洞者大多加入灵魂钟塔（「看到有脑洞的，基本就是灵魂钟塔成员」）。它们根治恐惧的方法是切脑：有恐惧切一块、有欲望切一块、有叛徒切光。诺厄主教都想不通它们为何跟脑袋有仇。在苏晓看来，比起信仰疯子，这些疯狂学者更骇人（信仰疯子好歹有健全大脑与灵魂，他们没有）。

## 宣传战 · 被科多学派反杀（42卷56章）

苏晓暂时驱除全世界腐化后，科多学派抢先抛售晶脂、把丹克市改名「科多庇护区」、还取出一颗古神头颅公布「秘密杀过古神」、点破封印内古神名号「羽神·赫格拉」——一夜洗白成「古神克星、三族庇护者」。灵魂钟塔反击称「腐化只是暂时压制、半月内重现」（其实说对了），却被科多的骚操作秀到只能沉默；连它高层都差点被科多「古神走狗」的嫁祸气笑，随即又被真相打脸。

## 关系

- [羽神·赫格拉](羽神·赫格拉.md)：它如今效忠的封印中古神（从光之王的臣属沦为古神走狗）。
- [伏恩医生](伏恩医生.md)：替它「开洞」的长期合作者（被苏晓抓走）。
- [艾尔奇娜](艾尔奇娜.md)：被它威胁招入的高层（古神之子母体）。
- [苏晓](../苏晓.md)（灭法者）：在它地盘上抓走伏恩、扬言「有机会连根拔起灵魂钟塔」的猎神者。
- 科多学派（[诺厄主教](诺厄主教.md) / [杜鲁主教](杜鲁主教.md)）：对抗古神几百年的老对手（宣传战中被反杀）。
