# 狼蛛女王（虫子王国之王 · 蜜莉恩）

> 真名**蜜莉恩**（42卷22章）。斩她得 +17.57% 世界之源 + 蓝色宝箱（她太虚弱、掉落品阶低）。当年与古神一战后她天性变得凶残暴戾——**吃了自己四骑士中的两位、重伤一位致死、最后一位伤心消失**；回过神后她扯断并吃掉自己后半截身体（抑制近乎疯狂的捕食欲 / 恐怖扑杀速度）、用蛛丝困住前半身。故她的话仅供参考、大贤者那边需按实情判断。

[暗星](../../世界/任务世界/暗星-世界入侵.md) 地下虫子王国的领袖——一只大体型狼蛛，头顶端坐着下半身没入蛛头的女人（深紫长发、紫色竖瞳、神秘高傲）。她是封印古神三条灵魂锁链的**最外层**，也是 [苏晓](../苏晓.md) 破封第一个要杀的目标（第42卷21章）。

## 主张决战 · 封印最外层

- **激进派**：容器裂痕后三王分歧中，她主张**打开容器与古神决一死战**——而打开封印意味着她（最外层）要**第一个死去**，故她留在黑巢下方的寝殿，能杀过去才能取她性命。她与 [光之王](光之王.md) 曾是死对头（魂链缠绕顺序：光之王最里 → 大贤者 → 狼蛛女王最外）。
- **执念支撑**：「就算在封印最外层，也比我想象中痛苦千倍」——缠在最里面的光之王还撑着，她这最外层当然不放弃，支撑她到今天的只剩傲气与执念。

## 覆灭 · 破封第一环（42卷21章）

苏晓杀穿黑巢十层（一路虫战士、七彩人面蛾守门）拿到女王寝殿钥匙，踹门见她。她已虚弱到抬不起头，只留遗言——**「闯过黑巢你就有资格夺走我的一切；小心大贤者，别相信他说的话」**。苏晓一刀刺穿她胸膛：她黑瘪的肌肤恢复光泽、瞬间回复曾经的美貌（紫竖瞳睁开），随即被全身魂链扯到粉碎、漆黑古神能量涌出——**封印第一层破**（古神在封印内积蓄的能量开始爆发）。远方枯萎森林里的 [大贤者](大贤者.md) 感应到她逝去、道一句「这几百年辛苦你了、蛛女王」。

## 关系

- [苏晓](../苏晓.md)（古神猎人）：斩她、破封第一层的猎杀者。
- [光之王](光之王.md)：曾是死对头、封印顺序在她之内的另一条锁链。
- [大贤者](大贤者.md)：她临终警告苏晓「别信他」的第三王（下一个破封目标）。
