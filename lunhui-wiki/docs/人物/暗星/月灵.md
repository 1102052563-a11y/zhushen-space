# 月灵（光之王的王侍）

[暗星](../../世界/任务世界/暗星-世界入侵.md) 白色小镇苍白宫殿里、守护 [光之王](光之王.md) 的**王侍**——身着浅红精美贴身铠甲、身材曲线玲珑、眸子清澈灵动、持一把一米三长的水晶短枪（第42卷11章）。她自懂事起就是王护卫、在此沉眠守护了几百年。

## 憨直的守护者 · 被巴哈忽悠

- **战力犀利、脑子憨直**：[苏晓](../苏晓.md) 一个照面就看出她「脑袋不太聪明」——但若不是苏晓在无外力下硬闯过「意志之路」，她的水晶刃枪早已招呼上来（她实战绝对犀利）。她一句「不可惊扰王者，退后」寸步不让。
- **被手机忽悠**：[巴哈](../../人物/剧毒魔鹰·巴哈.md) 一顿胡扯（「我等受大贤者、狼蛛女王委托来与光之王会面」）、又抛给她一部**手机**当「证明」——月灵对着看不懂的手机陷入深深迷茫，犹豫再三还是退到一旁（但随时准备扑上来），[布布汪](../../人物/布布汪.md) 凑上前吃零食看戏。她的憨直让苏晓能顺利接近光之王。

## 🆕奉命随苏晓破封（42卷16–17章）

光之王给苏晓 **王之手令**、命月灵随他破封（月灵协助战斗且不削苏晓的击杀收益）。她伤感「已经不需要我守卫了吗」，仍以「使命高于守卫王」奉命跟随。

- **战力**（王护卫资料）：人类·真实**力 203 / 敏 220 / 体 201 / 智 92 / 魅 152 · 幸 52**（随环境增减）·**具后天加持的世界之子特性**。坚毅心智（免部分八阶精神控制·意志 +42）+ 光·王族枪术（Lv.56·长枪 +745%）+ 终极「**蜂灵刺**」（光耀化超高速穿刺 7~12 次·敏 ×2.1 无视防御伤·期间近乎不可捕捉且抗伤）。她还有母神赋予的「**最后的绽放**」（吸三王全部力量抹去意识 30 秒·被光之王永久封禁·因抹意识 = 敌我不分·实为废招）。
- **憨直善良**：她的「善良」很特别——被巴哈盘问「遇到有恶意的重伤者怎么办」，答「先捅死他、再劈脑袋」；答问题从不思索、心中还默默夸自己机智。

## 关系

- [光之王](光之王.md)：她守护几百年、又奉其王之手令随苏晓破封的王者。
- [苏晓](../苏晓.md)（古神猎人）：她奉命协助破封（杀狼蛛女王 / 大贤者 / 光之王）的对象。
- [苏晓](../苏晓.md)（白夜）：硬闯意志之路、被她放行接近光之王的猎杀者。
- [巴哈](../../人物/剧毒魔鹰·巴哈.md)：用胡扯 + 手机把她忽悠住的魔鹰。
