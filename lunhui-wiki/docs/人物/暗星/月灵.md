# 月灵（光之王的王侍）

[暗星](../../世界/任务世界/暗星-世界入侵.md) 白色小镇苍白宫殿里、守护 [光之王](光之王.md) 的**王侍**——身着浅红精美贴身铠甲、身材曲线玲珑、眸子清澈灵动、持一把一米三长的水晶短枪（第42卷11章）。她自懂事起就是王护卫、在此沉眠守护了几百年。

## 憨直的守护者 · 被巴哈忽悠

- **战力犀利、脑子憨直**：[苏晓](../苏晓.md) 一个照面就看出她「脑袋不太聪明」——但若不是苏晓在无外力下硬闯过「意志之路」，她的水晶刃枪早已招呼上来（她实战绝对犀利）。她一句「不可惊扰王者，退后」寸步不让。
- **被手机忽悠**：[巴哈](../../人物/剧毒魔鹰·巴哈.md) 一顿胡扯（「我等受大贤者、狼蛛女王委托来与光之王会面」）、又抛给她一部**手机**当「证明」——月灵对着看不懂的手机陷入深深迷茫，犹豫再三还是退到一旁（但随时准备扑上来），[布布汪](../../人物/布布汪.md) 凑上前吃零食看戏。她的憨直让苏晓能顺利接近光之王。

## 关系

- [光之王](光之王.md)：她守护几百年的王者。
- [苏晓](../苏晓.md)（白夜）：硬闯意志之路、被她放行接近光之王的猎杀者。
- [巴哈](../../人物/剧毒魔鹰·巴哈.md)：用胡扯 + 手机把她忽悠住的魔鹰。
