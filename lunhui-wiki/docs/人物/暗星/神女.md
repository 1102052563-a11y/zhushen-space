# 神女（科多学派）

[暗星](../../世界/任务世界/暗星-世界入侵.md) 科多学派供奉的**神女**——琥珀色眸子、金棕色长发、傲慢自负，靠古神能量能悬浮半空、真实魅力 135 点，最擅长用**演技**骗人（第42卷8–9章）。她是 [苏晓](../苏晓.md) 查光之王去向的目标。

## 「神女」的真相

- **人造的容器**：所谓神女，实则是「母亲分娩时被腐化、新生儿却奇迹存活」的**万分之一幸运儿**——历代幸运儿全是女性，苏晓判断这绝非巧合、是科多学派**人为制造**的。她本质是「另一种特殊容器」：身体能吸收 / 储存古神能量供科多学派研究，故谁动神女、科多学派立即炸毛。
- **两大能力**：①感应并吸收聚集的古神能量（吸收时有畅快感·此为「神女」尊称由来）；②**感应光之王的位置**、甚至与光之王超远距离沟通（因光之王是缠在容器外的人族灵魂锁链、与古神能量几乎相连）。

## 被绑 · 无尽黑暗项圈审问（42卷8–9章）

护卫首领**孤影**（见下）+ 十几名阴影侍者被 [阿姆](../../人物/阿姆.md) / [放逐](../../力量体系/装备道具/index.md) 团灭，她被掳走。她先装晕、又提议「找替死鬼顶包换晶脂」耍嘴皮子——苏晓给她戴上 [无尽黑暗项圈](../../力量体系/装备道具/index.md)（逆向魅力判定·她魅 135 vs 苏晓 −3 被碾压·谎言之诅咒每撒谎意志力 −10「掉节操」+ 黑之狱「1 秒 = 20 天」的绝对黑暗）——几轮下来她放弃尊贵身份哇哇大哭，招供**光之王在「白色小镇」**。

### 孤影

护卫神女的**阴影侍者首领**——感知 / 预警极强、能收拢自身阴影、双臂近半米长臂刃。他让神女先逃、自己死战阿姆（臂刃只能割进阿姆身体两公分·被寒冰冻住），最终被阿姆按头一斧斩首（**殁**·42卷8章）。

## 关系

- [苏晓](../苏晓.md)（白夜）：绑她、用无尽黑暗项圈审出光之王去向的人。
- **孤影** / 阴影侍者：护她、被阿姆 / 放逐团灭的护卫。
- [默洛斯](默洛斯.md)：设局把她引出来的科多学派前大神官。
