# 摩利兰（落魄贵族宝石商）

[暗星](../../世界/任务世界/暗星-世界入侵.md) 一名落魄贵族——面白无须、有黑眼圈，穿着尽力维持体面却洗到发白的呢子大衣，营养不良（上次吃饱还是五个月前的烤土豆）。他家族曾在丹克市经营超凡宝石生意（科多学派庇护·净利仅一成、两成供瘸腿帮、七成税给科多学派），却因祖父的一念之差败落。他是 [苏晓](../苏晓.md) 那颗斩龙闪宝石 [魂切](../../力量体系/装备道具/index.md) 的卖家（以「一餐尊重」当定金·很有骨气）。

## 仪式杯的厄运（42卷51–53章）

他委托苏晓查三件事:祖父遭遇 / 妻子死因 / 自己的霉运（幸运 −2·有 14 次喝水险些被呛死）。真相由苏晓 + [贝妮](../../人物/狩之影.md) 抽丝剥茧揭开:

- **祖父招灾**：祖父在旧王都买下一处古宅（那是 [灵魂钟塔](灵魂钟塔.md) 布过「首次古神之子降临仪式」的地方），把地下室里**半损坏的仪式杯**当古董带回丹克市（灵魂钟塔故意留下坑「有缘人」）→ 祖父发疯（非腐化·纯疯）、表哥洗脸淹死、祖母腐化后杀死摩利兰父母。
- **妻子清白**：妻子清扫时手指被仪式杯破碎处划破 + 她体质对古神能量适应性极高 → 孕育出**二代古神之子**（残次品·母腹中即死）。她割自己手臂喊「有东西在身体里钻」、失血「死」去，腹中破膛出怪物。
- **妻子未死**：苏晓道破——能孕育古神之子的母体耐受力极强、想让她死要斩首；她其实**没死、已是灵魂钟塔高层 [艾尔奇娜](艾尔奇娜.md)**（当年接受灵魂钟塔邀请、换取他们不动摩利兰·摩利兰因此是「二次降临事件」的唯一幸存者）。
- **霉运真相**：摩利兰的霉运源自家中那只残破的**金属仪式杯**（每次回家变倒霉、出行好转、多次想丢却「舍不得」）。苏晓花 5000 朗晶脂买下仪式杯（摩利兰一度鬼迷心窍死保、被苏晓一句「成交」惊醒）——霉运当场解除（幸运 −2 → 回升）。

一小时前还一无所有的他，转眼霉运尽解、得知妻子尚在人世、还揣着 5000 朗晶脂启动金，驼背都直了起来（「祖父说的对，苦难结束后，一切都会好起来」）——尽管这本质只是苏晓一桩划算的交易（换来 [魂切](../../力量体系/装备道具/index.md) 宝石 + [破损的仪式杯](../../力量体系/装备道具/index.md)）。

## 关系

- [苏晓](../苏晓.md)（白夜）：为他查真相 / 解霉运、又借机换走魂切宝石与仪式杯的委托对象。
- [艾尔奇娜](艾尔奇娜.md)：他以为已死、实为灵魂钟塔高层的妻子。
- [贝妮](../../人物/狩之影.md)：替他重葬妻子、去旧王都挖古宅取证的调查者。
- [诺厄主教](诺厄主教.md)：科多学派恩主（认不出他却仍热情相待）。
