# 蛇夫人（最后的绿地的占卜师）

[暗星](../../世界/任务世界/暗星-世界入侵.md) 大陆北部野兽领地、厄罗河以内「**最后的绿地**」里的一位大人物——上半身完全是人类、下半身为遍布蓝色鳞片的蛇身，所有野兽族都尊称她「**蛇夫人**」。她其实也是「野兽」的一员。她身穿蓝紫色修身衣物、胸前开叉大胆、常叼一根修长烟杆吐紫烟，慵懒而优雅。

## 占卜 · 入梦生意

- **两门生意**：她住在最后的绿地一间通体暗白的小木屋，屋内挂满蜥蜴眼球、带牙食鸟花、半颗还在跳动的心脏等古怪货物。对外做两门生意——**入梦 2000 朗晶脂、占卜 1000 朗晶脂**。占卜靠一颗水晶球（眼瞳化紫、繁星遍布），越占卜强者消耗越大，替 [苏晓](../苏晓.md) 占卜命运时只看到「之后几小时」就险些被「吸干」。
- **勉强算大贤者这边**：她欠了 [大贤者](大贤者.md) 的人情、虽已还清但想离开也难，故「勉强算大贤者这边的人」。立场上属于**中立·善阵营**——只要不伤害她，她帮谁都无所谓，在群魔乱舞的暗星，要么强大、要么像她这样中立才能活长。

## 被苏晓的杀意逼着站队（42卷31章）

苏晓选「占卜」是为试探她的态度——若她真站大贤者那边就斩了。她一占卜「我在想什么」，就算到「**如果不说出秘密、不站你这边，我马上会死**」（苏晓按刀的手都抬起了）；再算「大贤者何时死」，只看到「你在黑焰中战斗」。为了不被苏晓「戴上那个奇怪的项圈」（[无尽黑暗项圈](../../力量体系/装备道具.md)），她**免费**替苏晓入梦、还退了占卜费。之后苏晓每次靠她的水晶球在蛇夫人木屋与梦境世界间往返（[月灵](月灵.md) 奉命看守她，一旦水晶球出问题即刺穿她胸口）。她看着墙上投影里苏晓用阿波罗 + 吞噬者 + 哈迪斯团灭梦境世界的惨状，一度觉得「**苏晓比古神更可怕**」。

## 关系

- [苏晓](../苏晓.md)（白夜）：被其杀意逼着站队、替他反复入梦的占卜师。
- [月灵](月灵.md)：与她相熟、被她溺爱的小丫头（这次奉苏晓命看守她）。
- [大贤者](大贤者.md)：欠过人情、「勉强算它这边」却毫不情愿的中立者。
