# 羽神·赫格拉（暗星被封印的古神）

[暗星](../../世界/任务世界/暗星-世界入侵.md) 被封印在容器中的**古神**，大多数生灵尊称它「**羽神**」——是 [苏晓](../苏晓.md) 破封要猎杀的主线终极目标。它的战力足以碾压光之王 + 狼蛛女王 + 母神，蔓延全世界的**腐化**正是它逼迫众生匍匐于神座之下的手段。

## 身世 · 为何主动被封印

它并非被光之王等人「打」进封印——**是它故意被封印，为的是规避某种东西**：

- **来自陨灭星**：它源自古神老巢「**陨灭星**」（超脱·原生世界·最强古神是**冥神**）。它从陨灭星**带走了某样东西**，从此与陨灭星所有古神敌对——**冥神的狱犬与信徒正在虚空中搜寻它**。躲进暗星、缩在封印里，是为了不被那些同级古神感测到。它带走的那样东西「重要到连冥神那一级都会动容」，也可能正是众多顶尖强者齐聚暗星要夺的目标。
- **漠视仆从**：它把追随它的众生都视作「蝼蚁」，唯 [大贤者](大贤者.md) 与「灵魂长者」（[灵魂钟塔](灵魂钟塔.md) 之主）算「值得保留的、强壮些的蝼蚁」。它借人类女性孕育「**古神之子**」当外界化身、以此在封印中也能吮吸世界（[艾尔奇娜](艾尔奇娜.md) 之子即残次品）。

## 封印内被苏晓间接重创（42卷55–56章）

苏晓不破封就出手——用 [伏恩医生](伏恩医生.md) 帮忙、以炼金生物学培育的耐受体 + 脐带布成**残次品古神之子**（与羽神有直接关联），再让这残次品间接触碰**茂生之狂乱的树根**与**沾有不死老人源血的破布**。羽神在封印内接连认出这些虚空顶尖存在——**不死老人**（去过陨灭星、吃过很多古神）、**茂生之狂乱**、恶魔铁匠、乃至**灭法者**（苏晓）都在这世界，惊觉「这个世界完了」。茂生之狂乱的树根一闪而逝就**重创**了它（多年积累付之一炬）、更让它**再也无法强行冲破封印**。它甚至考虑强行破封逃离，却已经来不及。此役即苏晓「和古神打的招呼」——虽只重创、未濒死（它积蓄几百年、实力未必大损·仍不好对付），却已把主动权攥在手里。

## 🆕破封 · 决战云雾之顶（42卷61–70章·进行中）

苏晓斩断三王魂链破封后，它与 [容器](容器.md) 一同放出。苏晓先用炼金阵图模拟的**茂生之狂乱树根**唬它（谎称茂生与不死老人将至），逼它以黑翼**超大范围侦测**（消耗过半古神能量）——它发现被骗、恶魔铁匠已离开、灭法者也没想象中强，却也因此中了苏晓的计。它随即飞离苍白宫殿、单挑追来的 [阿姆](../阿姆.md)、[巴哈](../../人物/剧毒魔鹰·巴哈.md)、[月灵](月灵.md) 与 [异端处刑队](异端处刑队.md)——**黑色羽毛（近半米长如箭·刺穿空间连异空间都不安全）+ 领域禁锢（几百米内禁锢 3~10 秒·连异空间的巴哈都被拖出）** 把众人打到重伤（阿姆被扯掉左臂与部分躯干），但因先前重创、领域有缺陷才没团灭。之后它退守大贤者经营上百年的**梦境世界**。

- **恒星之眼**：苏晓的主线最终环并非「击杀古神」，而是「获得 **恒星之眼**」（奖励 [起源石·世界](../../力量体系/装备道具.md) 1/5）——它正藏在羽神手中，也正是羽神缩进封印规避冥神等古神的原因。
- **魂之殿堂 / 云雾之顶**：大教堂内它把苏晓灵魂拖入「魂之殿堂」（灵魂体死则意识身体灵魂全死），却因苏晓**灵魂强度 470 过高**而被迫将其驱出。苏晓遂捏碎诺厄给的石球，把战场换到**云雾之顶**（飞行遭雷劈·封住羽神的飞行）。
- **凐灭印记 · 神源永损**：它的独有能力「**凐灭印记**」——被它任何攻击伤到有 34.7% 概率叠一层，超 3 枚即精神斩杀（苏晓凭精神强度 + [伯格之心](../../力量体系/装备道具.md) 可抗 6 枚）；此能力本因封印重创需 159 天才恢复，它却**认可苏晓的强大、以神源永久受损为代价强行唤醒**。它战力、经验、觉悟俱全（甚至反手用精神力凝成双剑与苏晓拼近战·「连神灵技法都不懂的古神在陨灭星活不过半月」），是真有可能夺走苏晓性命的敌手。云雾之顶的死战苏晓靠 [伯格之心](../../力量体系/装备道具.md) 硬扛凐灭印记、龙影闪反复贯穿其要害，终于**将它斩杀**（42卷71章·苏晓得 29.94% 世界之源 + 弑神奖励神灵骨 / 精神印记 / 陨灭星知识「眼之仪式」）。它藏了几百年的**恒星之眼**从身躯剥离、被轮回乐园收走——它成功避开了冥神的追杀，却终究栽在一名灭法者手里。

## 🆕降临之谜 · 被母神引狼入室（42卷73章·揭示）

苏晓事后从光之王残念处探知它入世的真相:**是母神把它引来的**。母神本是本世界唯一神灵，被后起的光之王 / 大贤者 / 狼蛛女王窥觊神位、信仰枯竭，她不甘沉寂、想制造危机借三王之手除敌，遂与冒牌古神 [树神·柏勒罗](树神·柏勒罗.md) 合谋、以树神的古神之力布仪式召唤真古神——恰逢它正躲避冥神的狱犬追杀、便顺势应召入世暂避。母神以为古神「只是战力强」（看树神就知道），结果引狼入室、四王反被它吊打封印。它在被封印时反而借机想更好地规避冥神——直到撞上苏晓。

## 关系

- [苏晓](../苏晓.md)（灭法者 / 猎神者）：借残次品古神之子在封印内重创它、破封后在云雾之顶与它死战的主线宿敌。
- [容器](容器.md)：封印它几百年的躯壳（破封后连它都控制不了这无情感的容器）。
- [大贤者](大贤者.md) / [灵魂钟塔](灵魂钟塔.md)：它眼中「值得保留」的仆从（灵魂钟塔已成它的走狗）。
- 冥神 / 陨灭星古神：因它带走某物而与它敌对、正搜寻它的同级古神（背景伏笔）。
