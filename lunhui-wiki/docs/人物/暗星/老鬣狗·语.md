# 老鬣狗·语（北兀部族向导）

[暗星](../../世界/任务世界/暗星-世界入侵.md) 大陆北部野兽领地边境「**北兀部族**」（鬣狗族）的一员，名叫「**语**」——一只瞎了眼、老到毛发脱落的老鬣狗，是整个部族中**唯一能听懂并口吐人族语言的「野兽」**。它脖颈下方毛发内藏着伤疤模样的刻痕，每当口吐人言，刻痕便格外明显。

所谓野兽族，外貌与野兽无异却有智慧，最大特点是「只要没饿急就不袭击其他智慧生物」；但它们的文明早已断绝，「或许再过几代就会退化成真正的野兽」。

## 当向导（42卷30–31章）

[巴哈](../../人物/剧毒魔鹰·巴哈.md) 用一枚 **晶脂项坠·魂生**（特定古神能量抗性 +9、可降低野兽族雌性分娩时的腐化风险、提升幼崽存活率）作报酬，雇「语」当向导，带 [苏晓](../苏晓.md) 一行渡厄罗河、去「最后的绿地」。它坦承自己**尊敬大贤者、却不拥护它**——对大贤者选择更接近人形的「背叛者」野兽族心存不满（表面尊敬）；出发前只要求苏晓回答一个问题「你们是贤者大人的敌人吗」，得到「是」的答复后仍愿领路（「我活不了太久，也没什么顾及」）。

它领路穿过枯萎平原与废弃贸易站、渡过剧毒死水厄罗河上的五公里石桥，一路平安到达最后的绿地，还把苏晓一行引荐给老朋友 [蛇夫人](蛇夫人.md)，随后快步消失在黑暗中离去。它毕生的理想是重新与人族、虫族达成贸易，却屡屡换来敌视、险些因此丧命。
