# 尼飞比特（猫女）

[蚁王·梅路艾姆](蚁王·梅路艾姆.md) 的**直属三护卫之一**——白发猫耳、细长尾巴的少女，本该软萌、却**指尖尖利、下身两条「斯巴达」粗腿肌肉扎结**（她只在乎战斗力、不在乎美丑）。爱好是**研究其他生物的构造、追捕猎物**。三护卫中**最难对付**（智商高、身体强、直觉极敏锐、能力麻烦）；对蚁王绝对忠诚（任何能威胁到「王」的东西万分之一几率都不放过）。

## 念能力「黑子舞想」

她的特殊念能力——身后浮现一道**红色虚影、以丝线操控她如提线木偶**，故**被斩首 / 刺穿心脏也不会立刻死、仍能战斗**；给她时间甚至能把头颅与心脏缝合「原地复活」（但恢复慢、需全神贯注）。

## 殒落（十六卷35–37章）

她追杀洗劫了王宫宝库（抢走「迷藏宝箱」）的 [温妮莎](温妮莎.md) 小队到王都东侧地道，[苏晓](苏晓.md) 从上方落入堵截。地道地形（墙壁 / 天棚作跳板）本利于她、直觉也让她躲过多刀（尾巴抽伤苏晓）；她拿温妮莎小队当肉盾 / 人质（1 分 06 秒内把这小队全灭），险些掐住苏晓喉咙——却被苏晓**龙影闪瞬移到背后、斩龙闪贯穿心脏 + 斩首**。她靠黑子舞想「诈尸」再战，被苏晓界断线缠住 + 青钢影削散黑子舞想，约两分钟后念能力衰竭、彻底死亡。苏晓得世界之源 12.9%（→25.3%）+ 噬灵者 +30 法力 + 金色宝箱。

## 关系
- **[苏晓](苏晓.md)**：杀死她的猎杀者（龙影闪 + 青钢影破其「黑子舞想」复活）。
- **[蚁王·梅路艾姆](蚁王·梅路艾姆.md)**：她绝对忠诚的「王」。
- **[温妮莎](温妮莎.md)**：被她追杀、当肉盾全灭的冒险团。
