# 蝎（赤砂之蝎）

[火影忍者世界](../世界/任务世界/火影忍者.md)「**晓**」组织成员、砂忍村叛忍、**傀儡师**——人称「**赤砂之蝎**」。是 [苏晓](苏晓.md) 在晓的**临时搭档**之一（与 [迪达拉](迪达拉.md) 同队·因苏晓「情况特殊、有重要作用」被编入这一队）。

- **傀儡「绯流琥」**：登场时是一具**戴黑布面罩、秃头、四肢着地爬行**的高大傀儡——蝎的本体藏在傀儡「**绯流琥**」内部（爬行是因里头的蝎呈卧姿）。
- **性情**：**讨厌等人、也讨厌让别人等**（罕见迟到时一上来就道歉）；追求「**永恒的艺术**」（与搭档迪达拉「瞬间艺术」理念相左·常争执）。父母在第二次忍界大战时被**木叶白牙**所杀（详见 [河之国·绝莫桥](../世界/任务世界/火影忍者.md)）。
- **认可苏晓**：他用「让苏晓自己去解决跟踪者」试探——苏晓以剃瞬移、五分钟秒杀那名**音忍（大蛇丸的人·感知型·能屏蔽感知的弱血继·有被改造迹象）**、当「见面礼」扔回、还**精准分析出其来历**（实力、势力归属）。蝎由此认定「有了一个可靠的搭档」。

## 围攻自来也 · 三代风影傀儡（九卷24–26章）
苏晓一行截杀河之国大名时撞上来调查晓的木叶三忍 [自来也](自来也.md)，蝎主张「拖住敌人」、与 [迪达拉](迪达拉.md)、苏晓组「无耻的 C 阵型」围攻。他祭出压箱底的人形傀儡「**三代风影**」（砂忍村三代风影本尊做成的傀儡·操控**铁砂**·自来也年轻时险死于其铁砂之下、手臂至今残留铁砂）——以铁砂锤 / 矛 / 雨近身缠斗自来也的通灵兽蛤蟆。他看出苏晓近战强过晓内任何人。

- **绯流琥破碎 / 本体现身（九卷26–28章）**：自来也仙人模式「蛙震」震碎、又一脚踢碎傀儡绯流琥——蝎的**本体（一名红发俊朗少年模样·已非人类·不需睡眠进食）**因此露面（与 [迪达拉](迪达拉.md) 凑一块「像偶像组合」）。此战蝎断一臂。

## 返水之国 · 识破委托陷阱（九卷28章）
新任务「**挑起水之国内战**」（佣金是截杀河之国大名的几十倍）三人乘黏土鸟返水之国。蝎转述委托人情报「雾忍村分主和派 + 激进派」、提议削弱主和派壮大激进派——被苏晓当场点破**委托方有问题**（现在雾忍根本没有激进派之分·其首领人物早被苏晓亲手炸死·照美冥上位后绝不容激进派；委托方给错误情报、疑想让「晓与雾忍狗咬狗」从中得利）。蝎认同「晓可帮人排除异己、但绝不容被利用」，遂决定**先抓个雾忍上忍问清情况**（由蝎去抓·若情况不对就改去杀委托人）。

## 抓上忍验证陷阱（九卷30章）
水之国新任务中，他在雾忍村外**蹲守抓离村执行任务的雾忍**审问（不潜村）：先抓上忍**赤井**（有骨气、拷问无果被苏晓抹喉）+ 一名啥都不知的下忍，后一口气挑回「一个班」（一上忍三下忍），用比迪达拉血腥得多的手段拷出实情——**雾忍村已无所谓激进派**（水影吞并 + 三名激进派头目神秘失踪后土崩瓦解）。三人确认任务是陷阱、一致中止，由他幻灯身联络长门（小队中长门最信任蝎）。

## 砂忍村身世大揭露 · 心存死志（九卷31–33章）
新任务「**活捉一尾人柱力**」（五代风影 [我爱罗](我爱罗.md)），长门点名让熟悉砂忍村的蝎参与。其完整身世由此揭开：

- **砂忍村长老千代之孙**：祖母 [千代](千代.md) 地位仅次历届风影；蝎父母为砂忍村战死、功绩 + 自身实力本让他极可能争得风影之位。**幼年父母战死的刺激**令他叛逃、转而钻研**永恒的生命**与「将傀偶改造为有生命有意识的存在」。
- **不自知的间谍 [由良](由良.md)**：他多年前用**大蛇丸的咒术**在砂忍村安插了一枚棋子——连本人都不知道自己是间谍的砂忍上忍由良。任务前他解封由良 → 由良杀守卫、为三人打开砂忍村唯一入口「一线天」。
- **心存死志**：踏入这片沙漠后苏晓清楚感到蝎已存死志。他托苏晓在任务后**清理掉由良这枚棋子**（「我的事，我之后要做个了断」），迪达拉劝「这不是聪明的选择、和你的艺术不符」也不改。攻打砂忍村时他按苏晓分工**保存实力、负责对付追击的敌人**（祖母千代正因感应到他在附近而始终戒备未出手）。
