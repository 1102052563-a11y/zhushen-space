# 尤娜

**尤娜** 是 [东京喰种](../世界/任务世界/东京喰种.md) 世界 [CCG](../势力/index.md) **零队**的**准特等搜查官**，[24 区](../世界/地点/index.md) 勘察小队成员（二卷42章）。

## 简介
学生模样的美女，约十八九岁，黑色长发、白色水手服，惯提两只库因克箱。看似高冷冷淡，实则**神经质、喜怒无常**——苏晓评其为「[铃屋什造](铃屋什造.md) 的翻版、甚至更甚」；嗜血腥味，凑近闻苏晓身上的血气并说「我喜欢你身上的味道」（二卷42–43章）。

零队因苏晓临时加入而被总部抽调，现 24 区勘察队仅 [有马贵将](有马贵将.md)、苏晓、尤娜三人。她起初不信苏晓的白装斩龙闪能破喰种皮膜，被苏晓瞬间架刀于颈后改观（二卷43章）。

- 已探索 24 区约半年，惯用**库因克「M.红月」**（远程枪械）；24 区探索中负责居中支援、远程补枪（二卷45–46章）。
- 神经质本色尽显：放声源诱敌时，听到自己最爱的「**甩葱歌**」竟当场扭动起舞（二卷47章）。
- M.红月有狙击枪 / 镰刀双形态，但近战远逊苏晓、有马；另持铠甲库因克「**新**」（脱困时与苏晓轮替使用、被剥下时衣物尽碎）。有**幽闭恐惧症**（二卷49–51章）。

## 关系
- [有马贵将](有马贵将.md)——零队队长、上司。
- [苏晓](苏晓.md)（白夜）——24 区同队，初质疑其实力、后折服。
- [铃屋什造](铃屋什造.md)——性格如出一辙（苏晓如此评价，二人尚未会面）。
