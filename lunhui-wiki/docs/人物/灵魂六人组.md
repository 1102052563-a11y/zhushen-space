# 灵魂六人组

**灵魂守卫、荆棘女、幽魂猎影、灵魂法师、黑钢骑士，以及灵魂之主**——⚠ **算上永光世界这次，苏晓已是第三次遇到这六个家伙了。**

## 🎭 一份工打成三份

⚠ **最初时，苏晓认为这六个家伙是灵魂斗技场的常驻守关者**:⚠ **灵魂守卫、荆棘女是前两关的精英怪，幽魂猎影、灵魂法师、黑钢骑士是后续三关的小 boss，而灵魂之主是关底大 boss。**

- ⚠ **可后来苏晓发现，这六个家伙不仅是被虚空之树所公证的灵魂斗技场守关者**——⚠ **他们还接私活**:⚠ **如果给出的好处够高，是可以凭灵魂系的强大召唤能力，将他们临时召来的**（⚠ **之前与 [噩鬼·凯因](风海大陆/噩鬼·凯因.md) 交锋，对方就把灵魂六人组召出来，准备以此对付苏晓**）。
- ⚠ **现在看来，灵魂六人组不仅接私活，他们还是监视者高塔三层·第一关的 boss。**
- ⚠ **实际上，灵魂六人组不仅有这两种身份，他们还是幽魂之森的维护者。**
- ⚠ **至于灵魂六人组为何要同时做这些，这话苏晓不能问**——⚠ **否则容易把灵魂六人组气的都飙出眼泪。** ⚠ **倘若不是某人把灵魂斗技场洗劫一空、连固定在地上的灵魂石宝座都踹碎带走**，⚠ **在灵魂斗技场混日子＋单手吊打八阶挑战者的灵魂六人组，也不用充当这么多身份，并且偶尔接以前看不上的私活。**
- ⚠ **作为收获**:⚠ **被苏晓以灵魂能力吊打一番后，灵魂六人组再也没有以往的嚣张气焰**——⚠ **最起码在灵魂斗技场来了新的挑战者后，不再获胜后还格外嚣张。**

## 😖 出场词念到一半的痛苦面具

⚠ 灵魂迷雾中，六位现身，⚠ **出场姿势各有不同，再配合灵魂迷雾，那就更有派头**——⚠ **尤其是幽魂猎影，还是前两次见面时那么拽。**

> ⚠ 「**你就是挑战者吗，可别怪我们六对一卑鄙，这是公证的规则……**」

- ⚠ **背着双刀、双手抱肩的幽魂猎影，话说到一半突然卡壳**。⚠ **在他身后，目光向上 45° 看着侧面、根本不屑直视对手的灵魂之主，发现自己手下念出场词一半就卡壳了**——⚠ **灵魂之主保持现在的造型，但用手轻怼了下幽魂猎影的背**；⚠ **可这名平常除了喜欢装哔、没其他缺点的手下，此时却还是卡壳、依然不继续念词**，⚠ **这让灵魂之主心中愠怒。**
- ⚠ **灵魂之主带着几分怒意地转头看向前方，然后就表演了颜艺**:⚠ **在见到挑战者是苏晓后，灵魂之主脸上的几分怒意快速被惊愕取代**，⚠ **随后是强颜欢笑**——⚠ **但因为太勉强，强颜欢笑都无法保持，最后直接戴上了痛苦面具**；⚠ **一同戴上痛苦面具的，还有灵魂之主的几名部下。**

## 💥 「揍他！」→ 一颗黑球一个

⚠ **灵魂之主的算盘**:⚠ **根据他的分析，从上次在灵魂斗技场交手到现在其实也没过去太久**——⚠ **那时苏晓的灵魂强度为 590 点，如此算来，现在能达到 650 点已是极限**；⚠ **这样想来，以 650 点所能带来的临时技能槽与技能点，基本没可能让灵魂飞弹能力发挥出真正可怕的威力**（外加此地只能用灵魂能力）。⚠ **想到这些后，灵魂之主忽然感觉这次或许能赢**——⚠ **要是赢了，不仅有格外丰厚的守关酬劳，还能找回以前被吊打的牌面**；⚠ **想到这点，灵魂之主脸上的痛苦面具逐渐消失，昔日的嚣张回来了几分。** ⚠ **灵魂六人组彼此对视，他们在彼此的眼神中都看到两个字:『揍他！』**

- ⚠ **然而苏晓的灵魂强度是 859 点**——⚠ **因大幅度超出同阶所能达到的灵魂强度，他获得 460 点技能点、35 个灵魂技能槽**；⚠ 他掌握**灵魂飞弹（Lv.95 满级）＋ 中级增益（Lv.30 满级）＋ 灵能源泉（Lv.10 满级）**，⚠ **总计消耗 445 点。**
- ⚠ 「灭法，别说我们六对一卑鄙，**小的们，给我上……**」——⚠ **巨人般的黑钢骑士一马当先**（左手 5 米高塔盾、右手长柄战锤，⚠ **冲锋势头格外骇人**）；⚠ **可在下一秒，他就来了个急刹车，差点一屁墩坐地上。**
- ⚠ **苏晓上方出现一颗颗磨盘那么大的灵魂飞弹，单次释放就足足有几百颗**；⚠ **这还不算完**:⚠ **随着苏晓单手虚握，这几百颗全部压缩成只剩弹珠大小的黑球，还发出渗人的嗡鸣声。**
- ⚠ **一颗灵魂黑球掠出** → ⚠ **咚！！黑钢骑士的塔盾中心向内凹陷，他整个人向后飞射、轰入到墙壁中**；⚠ **随后几颗灵魂飞弹紧随而至，几声震耳的轰鸣后，大片破碎的铠甲飞出**——⚠ **周边的大殿墙壁崩塌，战斗地形扩大成一处废弃遗迹**；⚠ **依稀还能看到，黑钢骑士正一瘸一拐地向远处逃。**
- ⚠ **灵魂之主看到这一幕后，眼角狠狠颤抖了下**——⚠ **他转头看向苏晓，目光格外凝重:他灵魂之主，今天绝不会逃！**
- ⚠ **苏晓自然不知道灵魂之主的想法**——⚠ **他只是想试试这些临时掌握的灵魂能力**；⚠ **看了眼还剩 90% 的灵魂能量，反正有「灵能源泉」快速恢复，不妨多使用几次**。⚠ **很快，几千颗灵魂黑球出现，那压缩后的嗡鸣声显得格外恐怖**——⚠ **方才刚下定决心不逃的灵魂之主，给自己与周边的几名部下增益加速后，腾空而起，向远处飞去。**
- ⚠ **苏晓跃到一根高耸的石柱上，看了眼向远处逃遁的几人后，取出神秘之眼，像是拧动魔方般咔哒咔哒地调整此物**——⚠ **至于和灵魂六人组的战斗，等着就行**:⚠ **灵魂飞弹被提升到极限后，追踪力堪称变态**——⚠ **锁定目标后，或是命中，或是持续追着飞行一个多小时。**
- ⚠ **苏晓等待近半小时**，周边场景重新化为最初的大殿——⚠ **而鼻青脸肿的灵魂六人组，正躺在或趴在地上。** ⚠ 奖励:**灵魂糖果配方 ＋ 临时灵魂技能槽 +12 ＋ 一次临时灵魂技能重置权限**（⚠ **由此可知，原本这一关的挑战内容只需战胜灵魂之主即可，而非必须战胜所有人**）。

## 🍬 灵魂糖果与那个误会

⚠ 苏晓随后用「死亡拥抱」把**黑钢骑士那弯曲的塔盾**隔空挤压成**苹果大小的金属球**（⚠ **因其密度惊人，砸地的声响也格外大**）——⚠ **灵魂之主看到这一幕后，忽然感觉:作为老熟人，这灭法之前分明是手下留情了。**

- ⚠ **这误会出现后，灵魂之主眼中的嚣张与不忿收敛了不少**，他犹豫了下说道:「**灭法，后面的那两个，可比我们强多了。**」
- ⚠ **苏晓有些想知道，为何灵魂之主等人看上去都五大三粗，却研究出灵魂糖果这配方**——⚠ **听问此事后，荆棘女直言答道:其实不是他们爱吃灵魂糖果，而是有固定的客源，经常去找他们订购灵魂糖果。** ⚠ **得知此事后苏晓并没深问**，⚠ **他并不打算和灵魂竞技场抢这方面的客户。**
- ⚠ **碍于公证规则**，灵魂之主没透露后续两道关卡的守关者是谁——⚠ **只是隐晦地表示:那两人都很强**；⚠ **不过相比第三层的这些阻碍，大部分来监视者高塔的挑战者，都是落败于第七层的使徒游戏。**
