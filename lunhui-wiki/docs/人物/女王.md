# 女王

**女王** 是 [东京喰种](../世界/任务世界/东京喰种.md) 世界中一支**三人女子冒险团**的团长、敌方契约者（二卷19章）。

## 简介
高挑御姐，一身骑士铠甲、手持门板大小的金属盾（至少两三百斤重，她却使用轻松），近战防御型。性格谨慎冷静、平时好相处，严肃起来令队员畏惧（二卷19–20章）。

三人持「**豁免卷**」进入这低级世界、**没有主线任务**；女王判断苏晓「单独一人占一个阵营绝不正常」，杀掉他能破坏衍生世界平衡、产出大量**世界之源**，遂决意伏击苏晓（二卷19章）。据其透露，三人曾历过「巨人」世界（二卷19章）。

## 关系
- [叶子](叶子.md)——队员（奶妈），关系亲近，被其唤「女王大人」。
- [黑白](黑白.md)——队员（狙击手）。
- [苏晓](苏晓.md)（白夜）——伏击目标。
- 寒鱼——喰种阵营契约者，表面畏惧三人、暗怀鬼胎，因「某人面子」未被她们清除（二卷19章）。
