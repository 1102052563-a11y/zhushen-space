# 炼狱伯爵

全名 **阿尔弗雷德·兰斯洛特·威克利夫·伯伦**（姓名奇长，故众人皆称「炼狱伯爵」）——**第九魔使**，曾是**虚空黑渊**内的一位领主级大佬。形象为戴牛角王冠、**熔岩为身躯、岩浆为血液**的庞大火红虚影（其信徒称之为「神」）。脑子不太灵活、爱面子、嘴硬，曾因信徒反叛被盗走武器、又被另一名黑渊领主吊起来抽 + 当烤肉架（不愿被叫「信徒」「伯爵兄」）。

## 被红色小丑召唤（十卷45章）

[红色小丑](红色小丑.md)（绅士同盟老大·仅剩一臂）建邪教、聚百余信徒演假神迹，**凭一份「阶位高到让炼狱伯爵惊悚」的契约把它召唤来**（契约已极不稳定、随时会消失、无法强制签不平等条约）。红色小丑借它的力量布**传送阵**，从异界召来**上千头未知生物**当炮灰，去围攻拍卖场报复 [苏晓](苏晓.md)。

!!! note "与「灭法之影」的伏笔"

    炼狱伯爵警告红色小丑别招惹「**一群疯子的传承者**」——它口中那个曾「缔造人族在虚空中的辉煌、从不称王、不干涉人类内政、后被法师联合外族偷袭灭门」的恐怖存在，正是 [苏晓](苏晓.md) 所承的 **[灭法之影](../力量体系/技能.md)** 一脉（它们「看不上任何种族的法师」）。炼狱伯爵更回忆起曾被一个**淡蓝色双眼、手持长刀**者踩着头警告「我们这没人信神、只信自己」——与苏晓的青钢影特征吻合，是灭法之影克制神 / 法系的又一重伏笔。

## 只是「唬人」的虚影（十卷56–57章）
拍卖场之战中，红色小丑仅剩一臂、毫无战斗力，全靠炼狱伯爵的**虚影**唬住盟友 [库洛洛](库洛洛·鲁西鲁.md)——但这虚影**实则只能吓人、并无实战之力**。小丑起初选它正因它「没智谋、好利用」，相处后却发现它**蠢得离谱**（「若非血脉强大早死在黑渊」）、还几度**威胁要「吃了」它**（吞噬其灵魂），炼狱伯爵嘴硬反呛却也心生忌惮。
