# 阿尔托利亚（呆毛王 · 裁决者）

轮回乐园的一名**裁决者**——满头金发、绿宝石般的眸子、头顶一撮标志性的呆毛，故被戏称「**呆毛王**」。她原是 [Fate/Zero](../世界/任务世界/fate-zero.md) 第四次圣杯战争的英灵 **Saber**（骑士王阿尔托利亚·潘德拉贡），后成为轮回乐园的裁决者——已不再是当年的剧情人物，见识与实力都今非昔比。她**正义感极强、耿直到不知变通**，还是个「**吃货**」（无论伤多重，只要食物够美味就能从病榻上爬起来吃两口·吃得还挺香）。[癞蛤蟆](癞蛤蟆.md) 教导她已久、早把她当弟子看待，[暴鼠](暴鼠.md) 则是她的酒友前辈。

## 黑暗物质缠身 · 求治于苏晓（42卷74–77章）

她欠 [癞蛤蟆](癞蛤蟆.md) 太多恩情，为了替它弄到「黑枫树枝干」，竟真的听信 [暴鼠](暴鼠.md) 一句随口的玩笑、跑去**渊龙底接受龙之试炼**——结果被虚空特有的**黑暗物质**重度侵蚀（127 种异常般缠身、一度半昏迷、脸上生黑纹）。癞蛤蟆、暴鼠遂重金请 [苏晓](苏晓.md) 来拔除黑暗物质（报酬是一截女灭法者 **格林·吉莉安** 的佩刀刀尖·苏晓想拿去喂 [斩龙闪](../力量体系/装备道具/index.md)）。

- **地狱般的治疗**：拔黑暗物质要先用药剂把它从神经系统驱散（否则一昏迷就会连灵魂一起被拔走）。首次治疗她平均 3~5 秒昏迷一次、反复几百次，全身湿透；正式剥离时她惨叫到最后甚至求饶「求你杀了我」——却只拔除了约 1/10（更多是先修复被严重侵蚀的身体）。她咬牙硬撑，靠的正是「挨了现实毒打就能恢复」的顽强（连黑化 / 暴走都能被现实的痛打拉回来）。苏晓判她「还没侵蚀到大脑但快了」，往后还有更凶险的二次、三次治疗等着她。[莎](莎.md)（同为黑暗物质受害者）陪她进治疗室、以「病友惺惺相惜」打气。

## 关系

- [苏晓](苏晓.md)（白夜）：为报酬（灭法者佩刀刀尖）替她拔除黑暗物质的「神医」（治疗过程堪称地狱）。
- [癞蛤蟆](癞蛤蟆.md)：把她当弟子教导、为她张罗治疗的裁决者恩师。
- [暴鼠](暴鼠.md)：一句玩笑把她坑去渊龙底的酒友前辈（不厚道地看她笑话）。
- [莎](莎.md)：同为黑暗物质受害者、陪她进治疗室打气的「病友」。

## 🆕 三方交易的信用担保 · 二次治疗（44卷135–138章）

画之世界返园后再登场:苏晓要开启天启乐园产的**随机抽取权限**，[凯撒](尼古拉斯·凯撒.md) 自己信用度太低办不了，便拉上她——身为信用度高达 **227 点**的正牌裁决者，她是这单「三方交易」不可或缺的信用担保。她端庄地捧着热茶坐镇（癞蛤蟆 / 暴鼠 / 莎「关键时刻都太可靠」全溜了），全程被凯撒搞出的「虚空之树与轮回乐园公证冲突」看得一脸迷茫，还被 [巴哈](剧毒魔鹰·巴哈.md) 把名字连叫错四遍（阿多皮皮亚 / 阿皮托多亚 / 阿尔拖地亚）。

- **委婉拒绝入裁决伙 + 帮找铭文基座**：苏晓本想请她长期入伙补信用，她因正在接受黑暗物质治疗、以极委婉的措辞（难为她能想出那些词）婉拒，转而提出**帮苏晓寻找铭文基座、还愿分担部分购买费用**。
- **地狱般的二次治疗**：这次治疗还没开始，她就陷入巨大犹豫，最终「良心过意不去」还是答应——三小时里治疗室传出带哭腔的求救声（比上次坚强了，上次哭喊、这次老实很多）。苏晓一次就从她体内剥离出 **3 份暗之沉淀物 + 8 份黑暗物质**（力度没把控好、剥太多、心理阴影不小）。她如今资产贫穷（除一把剑与补给全卖了换灵魂钱币提升自身·当初正是为凑黑枫树枝干才去渊龙底被侵蚀）。
