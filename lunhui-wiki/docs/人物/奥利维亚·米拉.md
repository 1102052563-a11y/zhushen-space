# 奥利维亚·米拉·阿姆斯特朗（冰雪女王）

镇守 [布里克兹要塞](../世界/地点/index.md) 的**少将**·金发女将军，人称「**冰雪女王**」。骁勇善战、气质似 [艾斯德斯](艾斯德斯.md) 但无折磨人的嗜好；**只信自己亲眼所见**、讨厌到处乱窜的炼金术师。因能力太强被人造人忌惮、故被发配到西北苦寒之地（边塞已半自治、自征税）。弟弟是中央的阿姆斯特朗少校。

- **历程**：七卷28章苏晓潜入要塞**摊牌拉她谋反**——在地图上把历来战乱地点标点连线、勾出贤者之石炼成阵，又给她一小块贤者之石样本。她虽未当场表态、却暗中查实那些做不了假的战争记录，又从来访的主角艾尔利克兄弟口中印证了贤者之石 / 人造人的存在，**信了大半**——成为苏晓属意的**第二个盟友**。
- 是反 [瓶中的小人](瓶中的小人.md) 四伙势力中（布里古兹北壁军队）**人数最多、人脉最广**的一方。
