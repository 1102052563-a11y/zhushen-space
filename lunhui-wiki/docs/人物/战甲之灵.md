# 战甲之灵（领主级 · 灭之石持有者）

[希亚圣殿](../世界/地点/index.md) **一层真正的镇守者**、希亚王朝的**大将军**——身高约 2 米、穿一身甲片缝合的「战甲」（关节活动幅度大、技术远胜全身铠），**右手背的甲片上镶着「灭之石」**（蓝宝石·内有电光）。[长刀使·空](长刀使·空.md) / [震天使·鸣](震天使·鸣.md) 只是「看门的」——据 [死灵贤者](死灵贤者.md)，把它俩加上千名死灵士兵也不是战甲之灵的对手。它镇守着通往希亚圣殿二层的入口。

## 被阿波罗重创 → 尽职战死（十七卷65–67章）

硬拼苏晓拿不到灭之石，遂用 [隐之力](../力量体系/装备道具/index.md) 隐身、抛 [太阳神·阿波罗](../力量体系/消耗品材料/index.md)（炼金巅峰之作·密闭空间威力更强）把整片空地烧成火海——30 名普通长刀使瞬间化灰，战甲之灵被属下压在身下减伤、仍被烧到生命剩 23%。它本想把灭之石抛进二层入口（宁可死也不让灭法之影拿到），但苏晓追来，**勾起了它「战斗狂」的本性**——它生前最大乐趣就是与强敌一战，被囚此处镇守本就憋屈（它认为自己该战死沙场）。

激战中它一拳「震荡炮冲拳」被苏晓龙影闪躲过、心脏被斩龙闪贯穿；临死它**主动把镶灭之石的甲片抛给苏晓**（只为弄清「自己栽在何人手中」），随后被烈焰燃尽、**眼都不眨、尽职战死**。苏晓暗想：若非自己是灭法之影、若在它生前相遇，两人或许会有共同话题。

> 【你获得灭之石】【你已击杀战甲之灵】**远古死灵精粹 ×13**。
