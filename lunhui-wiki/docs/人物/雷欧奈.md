# 雷欧奈

**雷欧奈** 是 [斩·赤红之瞳](../世界/任务世界/斩赤红之瞳.md) 世界 [夜袭部队](../势力/index.md) 的主力成员（四卷24章）。

## 简介
金发、丰满有肉感的大骨架美女，性格大咧咧；出身平民窟、幼时以偷窃为生，14 岁拜「皇拳寺」老僧为师、16 岁得帝具、17 岁入革命军、18 岁成夜袭成员。

- **帝具「百兽王化·狮子王」**：使用后**人兽化**（头生兽耳、双手化兽爪），大幅提升身体素质与嗅觉感知；奥义「**狮子永不灭**」可大幅提恢复力、小部位断肢可再生（断臂断腿则不行）。
- **属性**（哈瓦呐之眼·狮子王状态）：力 30 / 敏 26 / 体 28 / 智 10 / 魅 12——**综合属性高于苏晓**，但智力低、纯靠肉搏。

## 被苏晓重创（四卷25–27章）
她奉命打探 [艾斯德斯](艾斯德斯.md) 动向，感知到其「头皮发麻的杀意」欲撤，却被艾斯德斯身边的苏晓盯上，逃亡上百公里、绕帝都两圈仍被追上（沼泽地里二人都狼狈）。苏晓判断「属性高 ≠ 实力强」——凭**刀术专精 Lv.11 + 锋利的斩龙闪**碾压她的肉搏：一刀斩断她半只兽爪、再一刀爆衣、一刀断臂（其兽爪硬膜、肉体在斩龙闪前如纸）。濒死之际被 [拉伯克](拉伯克.md) 的「交叉之尾」丝线拽走、缝合断臂续命（二人自称「大难不死就能续命」组合）。她力劝拉伯克别去寻仇（必死），并自行返回总部（四卷25–27章）。

## 关系
- [拉伯克](拉伯克.md)——搭档（「续命组合」·救她一命）；[娜洁希坦](娜洁希坦.md)（首领）、[赤瞳](赤瞳.md) 等同伴。
- [苏晓](苏晓.md)（白夜）——把她追杀至断臂濒死的帝国暗部。
