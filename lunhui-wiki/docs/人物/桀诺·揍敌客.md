# 桀诺·揍敌客

全职猎人世界**揍敌客暗杀家族**的强者——剧情主角奇犽·揍敌客的**祖父**、强化系念能力者。他是 [苏晓](苏晓.md) 上次入猎人世界（[绅士同盟](../世界/任务世界/全职猎人.md)）时结识的旧识：当年他与儿子席巴受十老头雇佣、与幻影旅团团长交手时「雇主出多少钱就出多少力」（没认真）。如今苏晓评估——他单独能胜桀诺 + 席巴两父子（拼命也至少能杀其一）。

## 「今日不杀」 · 接待苏晓（十六卷6–7章）

苏晓为主线「蚁王的心脏」来揍敌客家族探路：阿姆推开试炼之门（七扇对开门·中心 4 吨到第七扇 256 吨）、巨犬带路，总管家梧桐拦下要他等 2~3 天，桀诺挂着「**今日不杀**」的牌子现身解围——他一见便知「上次还有胜算、现在与苏晓交手必死」（苏晓变强之快令他心惊）。他半信半疑地邀苏晓入古堡（苏晓直言「有机会不介意宰了伊尔迷」、桀诺也直言「你杀他揍敌客家族会报复」——杀手家族的对话就这么直白）。苏晓真正想见的是**马哈·揍敌客**（桀诺的爷爷·只在出任务时露面）。

!!! note "揍敌客家族 · 纯粹的杀手家族"

    嫡系成员共十人、是纯粹的暗杀家族——**接到互相对立的任务时，自家人也可能彼此敌对**（任务第一·没有其他）；大本营枯枯戮山竟是旅游景点（坐收门票·不在乎风险）。家族里最深不可测的是**马哈·揍敌客**（1 米 4 的老头·实力与猎人公会会长尼特罗不分伯仲·比现在的苏晓强一筹）——苏晓想借与他交手来评估自己与蚁王的差距。
