# 罗娜

[彼岸花冒险团](../势力/index.md)（全女性冒险团）重整为精英小团后**新招募的主坦**（前排扛伤）——酒红色长发、身材瘦弱却散发茉莉花香，**实力甚至略强于团长 [冷月](冷月.md)**（以她的实力本可在大型冒险团受重用，却偏要加入只有五人的彼岸花）。

- **隐藏的野望**：她性取向特别、**心中藏着「得到冷月」的执念**。很久前在一个 Lv.2 世界她曾遍体鳞伤垂死街头，是路过的冷月一句「生存不易」让奶妈 [糖果](糖果.md) 救了她（冷月至今不知罗娜就是当年那名垂死的契约者）——这是她加入彼岸花、苦寻冷月与糖果的真正原因。
- **吞噬岛**：八卷11–22章随六人精英团参加 [二阶生存试炼](../世界/任务世界/T12K星球-生存试炼.md)。起初她不信「刀男」[苏晓](苏晓.md) 有多强、扬言要交手；亲眼见他在战场斩出**刀芒**（+7 以上强化武器的特殊光芒）、于万军中如入无人之境后，她**呆住改口、断了交手的念头**（自知必死）。

- **吞噬岛恶战（八卷27–29章）**：苏晓为清理 [腹黑女](腹黑女.md) 撞上彼岸花、开战。罗娜作主坦冲在最前——接住苏晓抛来的 [蜂](蜂.md) 尸（连炼金炸弹）被炸成重伤、又被界断线斩断一脚（她防御弱但**恢复力超强**·配糖果治疗很快接合）。她体力 47（比苏晓还高 3）、靠盾 + 战吼（嘲讽·被苏晓体力判定免疫）+ 防御技能硬撑，却仍被苏晓全速绕背、斩龙闪贯穿躯干 + **断头台**（勒颈 + 膝砸脊柱）压制至脑缺氧边缘——危急关头被副团长 [血玫瑰](血玫瑰.md) 一枪逼退苏晓、抓走救下。**她比谁都早察觉苏晓「一点都不脆」**（44 体力 + 灵影体质 + 岚凯防御·非冷月以为的薄血高攻）。八卷30章血玫瑰拼死掩护仍未能救她——重伤的罗娜被苏晓**割喉而亡**（至死怒骂「你不得好死」）；她暗恋冷月、却终究没能得到她。
