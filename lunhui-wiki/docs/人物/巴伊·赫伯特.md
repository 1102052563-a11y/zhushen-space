# 巴伊·赫伯特（圣索丁王国老国王）

[希亚大陆](../世界/任务世界/希亚大陆-灭法遗迹.md) 上 [圣索丁王国](../势力/index.md) 的**老国王**——现年 **65 岁**，须发皆白，纵有珍贵秘药滋养（皮肤无皱、面色红润），给人的整体感觉仍是「苍老」、自知**命不久矣**。坐天降陨石锻造的漆黑王座、戴千年传承的「铁王冠」（初代国王铁冠王·阿德莱德·赫伯特所传）。

## 登场 · 暗召铁之手（十七卷5章）

他**暗召** [苏晓](苏晓.md)（化名白夜·伪装「铁之手」）入王国议事厅。在臣子面前他是威严贤智的国王，独独对沉默的「铁之手」吐露心声——坦言「最后悔的事就是争到王位」（年轻时渴望权力、得到后才知其重）；又问「铁之手帮我除掉了多少异己」（答 **257 人**）。他凝视苏晓时眸子一度变得犀利如能看透人心，随即收起。

## 主线「权利之冠」的发布者（十七卷5–6章）

他自知将死、须尽快定继承人，遂托付苏晓两件事：① **监视赶回圣城争位的约 49 名子嗣**，谁起反叛之心就「处理掉」；② 待继承人胜出后，**像辅佐他一样辅佐新王**（由此激活苏晓主线 [权利之冠](../世界/任务世界/希亚大陆-灭法遗迹.md)）。他还令苏晓当晚**正大光明肃清异己**（不再「秘密消失」、而要把异己「杀破胆」、让全王都知道铁之手是「我巴伊·赫伯特最锋利的刀」），为新王上位铺路。唯独对**长子大王子哈帝**（47 岁）网开一面、特许「一次不杀的机会」。

> **冷酷的底色**：他 45 年前正是在同样血腥的权力争夺中获胜——**亲手除掉了所有至亲，包括自己的父王**。如今轮到他用同样的规则筛选自己的子嗣。

## 临终布局 · 王座之下的阴谋（十七卷39–40章）

他命不久矣（秘药从「5 天一次」恶化到「每天 30 次」、自言撑不过当晚），临终前**逐一试探手握大权的心腹**（怕新王压不住他们）：军团长赫士列特、执政官 [乔·瓦伦丁](../势力/index.md)（以「我撑不过今晚」试探·乔真情落泪 → 通过·老国王愧疚）相继通过；罗文则直接叛国。

**「王座之下」其实是他设的局**：他在「杀戮之夜」名单里特意加上大祭司（料定其临终会向铁之手吐露「王座之下」），意在**把铁之手 [苏晓](苏晓.md) 诱进王座之下除掉**（让他死在里面 / 成为被镇压的同类）——因为他虽信任铁之手、却不确定其会忠于新王，又不便用武力除之（会损王族声誉、且打不过）。但苏晓反将一军、与他摊牌交易（「带一名王子 / 公主进王座之下、我对王权没兴趣」），逼得他惊问「**你到底是谁**」。

**选定三王子**：这一切都是苏晓在**拖延时间**——三王子 [马蒂尔·赫伯特](马蒂尔·赫伯特.md) 已在上百禁卫军簇拥下入场（早与铁之手 / 乔 / 赫士列特串联）。老国王见来的是失势的三王子而非大热门大王子，**非但不怒、反而无比欣慰**：「马蒂尔，你成为了我最优秀的子嗣」——能在失势后暗中串联三名重臣上位，正是他要的帝王之才。

## 解脱 · 死于苏晓之手（十七卷41–42章）

他靠**秘药 + 注入「死灵君主之血」**（死灵君主由强大圣战士转化而来、非死灵转化）才硬撑到选出新王——代价是数日无法入眠、每时每刻忍受让人发疯的剧痛（健康时他该睿智到何种地步？）。待三王子上位、他终于得以解脱。他与苏晓做了一桩**只有两人知道的交易**：他提供进入「[王座之下](../世界/任务世界/希亚大陆-灭法遗迹.md)」的方法（写于羊皮纸），苏晓则**帮他解除痛苦（杀他）+ 承诺不叛出王国 / 不投死灵**。于是在三王子目瞪口呆下，**苏晓掐断了他的脖颈**——他**面带笑容而死**（既得解脱，又用性命试探出「苏晓不会像罗文那样叛国」、给新王留下关键信息）。
