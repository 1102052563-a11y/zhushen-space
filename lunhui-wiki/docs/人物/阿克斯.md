# 阿克斯（灭法者 · 希亚世界的奠基者）

千年前降临希亚世界的一位 **[灭法之影](../力量体系/技能/index.md)**（在世灭法者中实力**前十**），是整个希亚大陆现今格局的真正奠基者。[苏晓](苏晓.md) 在灭法之影遗迹的石门壁画（落款「**吾名阿克斯，已担负灭法 547 日轮〔1 日轮 ≈ 1.6 年〕**」）中，读到了被胜利者抹去的希亚世界真史。

## 降临希亚 → 拯救元素体系（十七卷81–82章·壁画揭示）

- **强到能刺穿世界**：希亚世界并非虚空中的星球，阿克斯被元素波动吸引而来——他太强（「若希亚世界是气球、他就是利刃」），降临即刺穿了这个世界，贸然离开世界构成就会崩溃，须先让世界「适应」他才能温和退场（适应花了 16 年）。形象的强弱比：本世界最强 BOSS [里德](陵墓守卫者·里德.md) 约「50 级」，**阿克斯则在 9000 级以上**（一个喷嚏喷死一大片里德），降临之初连移动都不敢（会引发空间崩溃天灾）。他放出 **20 具人偶机关**摸清世界现状。
- **元素体系将崩的祸根**：是**远古死灵族**所用的「**死灵法术**」（暗元素 + 水元素亲和的法术）——若放任，希亚世界这颗星球终将崩溃、生灵尽灭。阿克斯**断绝了死灵法术**，等同覆灭了 **希亚王朝**（远古死灵族的王朝）：**希亚大帝死、人族翻身成霸主**。他不在乎谁统治、只要元素平衡，遂把残存的远古死灵族传送到附近一颗荒星（死灵法术已断、它们死活已不重要）。

## 留下的格局（十七卷82章）

- **遣走巴特拉族**：希亚世界原有三族——死灵族、人类、[巴特拉族](../世界/生物万族/index.md)（原始种族·最先发现阿克斯并奉其为神·被远古死灵族奴役当苦力）。阿克斯被巴特拉族烦到、又不忍灭族，便把他们**传送到虚空一颗偏僻富饶的星球**（多数巴特拉族就此摆脱奴籍）。
- **扶植 [阿德莱德·赫伯特](阿德莱德·赫伯特.md)**：一名跟随他的人类，自称「阿克斯的使者」、聚拢穷苦人推翻五大王国暴政、建立 **圣索丁王国**——即俗称的「铁冠王 / 开国皇帝」。
- **改造灭法殿 + 留人偶机关镇守**：阿克斯离世前把巴特拉族为他修的「灭法殿」改造成今日的灭法之影遗迹（加建灭之殿 / 法之殿 / 影之殿三道石门），并留一具**人偶机关**镇守（有部分灭法能力 + 自身意识·似炼金生物）。后来流放的死灵族误开空间通道回到暗月沼泽（**这才是「现在的死灵族」**），来犯灭法殿被人偶机关教训得「杀怕」——灭法殿遂成死灵禁地。人偶机关后返虚空、得知灭法者时代已过，又回来仿制了殿内雕塑，再赴虚空与施法者拼命、最终消散。

## 关系
- **[马文·华尔兹](马文·华尔兹.md)**：同为灭法之影；马文的好友「阿克斯」即此人（马文本人没来过希亚世界、是听阿克斯说的）。
- **[阿德莱德·赫伯特](阿德莱德·赫伯特.md)（铁冠王）**：被他扶植的圣索丁王国开国之君。
- **远古死灵族 / 希亚王朝**：被他断绝死灵法术、覆灭王朝、流放荒星（见 [生物万族](../世界/生物万族/index.md)）。
- **[苏晓](苏晓.md)**：后世同脉的灭法之影传承者，借其壁画知晓希亚世界真史。
