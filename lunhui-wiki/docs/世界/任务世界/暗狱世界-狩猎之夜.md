# 暗狱世界·狩猎之夜（第38卷·七阶最危险四世界之一）

[苏晓](../../人物/苏晓.md) 用 [埃尔德的引荐信](../../力量体系/装备道具.md) 主动进入的**七阶最危险四个世界之一**——难度 Lv.42~70、**世界之源恒 0%**（改按「世界声望」综合评价结算）。这是个「神秘与平凡共存」、一切藏在黑夜里的**克苏鲁调性蒸汽 / 电气时代世界**；已有 **10 名猎杀者 / 14 战斗天使 / 12 先驱 / 11 处刑者**死在这里的主线上。

## 世界背景 · 暗狱之门（世界简介）

「**暗狱**」本是与本世界隔开的独立空间——混乱纪元暗狱之门开启，**暗魔**（七阶战力·破坏 / 杀戮欲强的智慧生物）与**幽魂**（稀少·寻常攻击无效·惧灵魂伤害·可控制任何人）从中涌出，隐入人类世界猎食灵魂。几大帝国团结反抗百年无果，只能「习惯」它们、学会夜间不外出。后有「**守望公会**」与「**守夜人**」（夺取暗魔 / 幽魂力量的人类超凡者·近战系夺暗魔力量 / 灵魂系夺幽魂力量）在夜里狩猎——却因常年杀戮渐渐失控、被人们唾弃驱逐。直到「**无阳之日**」：暗狱不再只是开门，而是**与本世界融合了一部分、且还在继续**——守夜人们又穿上旧风衣重新狩猎（如今台面上英俊貌美的「守夜人」只是公会脸面·真正夜猎的仍是无名之人）。

## 守望公会体系

- **守夜人 / 猎人**：夜里狩猎超凡怪物的无名之人（早期叫守夜人 / 守墓人·今统称猎人）；听命守望公会。莱茵王国国王远见——把平民与超凡完全分割（平民极难接触超凡·警官接手后转交公会派猎人）。
- **超凡事件四级 × 暗魔皮色**：三等（低危·**灰皮**暗魔·憨·新晋猎人）< 二等（危·**暗红**暗魔·资深猎人）< 一等（高危·**黑皮**暗魔·老猎人）< 特等（极危·**有称号 / 名字**的暗魔·屠杀过太多平民·须 10 老猎人 + 50 资深猎人）。
- **庇护屋**：守夜人补给点（每市一处·公会无偿供物资）——苏晓成了**「庇护屋（十六）」的管理人**，可接「委托」得声望、也可把积压超凡事件**发布给己方 / 天启 / 圣光 / 圣域契约者**（任务难度 ×10% 声望）。

## 苏晓的处境（埃尔德引荐信）

- **伪装 = 霸主级「神灵猎人·库库林」**（[埃尔德的引荐信](../../力量体系/装备道具.md) 为媒·探测只显霸主模板·旧识契约者也认不出真容）——[莫雷](../../人物/原始大陆群像.md) 带一队天启 / 圣光契约者来投（她本是来「躲某个人」·却把这里当剧情人物的庇护屋放了心），[圣鸳](../../人物/暗狱世界群像.md)（魅力系御姐）想用魅力交涉却撞上「与库库林交涉魅力特性全无效」的铁壁（侦测得他魅力是自己的 860 倍、显出力 181 / 敏 181 / 体 180 / 智 180 的霸主资料）。
- **主线任务「隐秘之物」**：找到暗狱与本世界融合的**原因**并中断（16 天·奖「豁免徽章」免除魅力 / 意志 / 幸运 <0 的减益〔−10 内〕·完成前无法离开）——苏晓判断这背后**大概率有人主导**（否则前面那些猎杀者不会连敌人是谁都不知道就惨死），压箱底的杀手锏是只能用一次的 [航海罗盘](../../力量体系/装备道具.md)。

## 进程（按章）

- **进入 · 接管庇护屋 · 主线**（三十八卷1–3章）：进索多市重工业区，从臃肿男手里接管「庇护屋（十六）」、伪装成霸主级神灵猎人·库库林；激活主线「隐秘之物」+ 世界声望系统（声望商店有 2 枚七星称号·最多兑 1 枚）。警探 **[道格拉斯](../../人物/暗狱世界群像.md)** 来电对接超凡事件。
- **委托 · 莫雷小队**（三十八卷4章）：[莫雷](../../人物/原始大陆群像.md) 带队来投（想躲人·被库库林的霸主资料吓到考虑跳窗）——苏晓把积压的一等超凡事件**发布**出去赚声望，莫雷小队接下「**红女妖**」（Lv.65·伪装美女杀男性的三阶段暗魔）。
- **暴食暗魔 · 世界声望商店**（三十八卷5–7章）：警探道格拉斯急电——贵族区巴特少校一家四口被**暴食暗魔**（异类·把人全吃光·案发现场蹲守偷袭猎人·胸口挂 16 银 1 黑 2 白猎人牌）连尸都吃没。苏晓循「暗狱裂缝」线索，一刀腰斩暴食暗魔（见「神灵猎人」牌就逃·得 62 世界声望）——**世界声望商店**开启（含七星称号「[追梦人](../../力量体系/称号.md)」·3000 声望·可把六星升七星）。
- **豪鬼 · 女猎人蒂法尼**（三十八卷7–9章）：去找记忆剥离的女猎人 **[蒂法尼](../../人物/暗狱世界群像.md)**（液体化·惧暗魔血·失双腿一臂）时，撞上**四阶段暗魔 [豪鬼](../../人物/暗狱世界群像.md)**（硬闯过守望公会总部·手刃 157 老猎人·比泰密莎强一筹·「老友」）抢先来杀蒂法尼——苏晓与之激战（毁灭·震界王 / 刃道刀·噬），豪鬼被两条**黑色触手能量手臂**拖回暗狱（有人操控·危险预知）。
- **猎人牌真相 · 圣堂冒险团**（三十八卷9–10章）：蒂法尼剥出暴食暗魔记忆——**主线真相**浮现：一个未知势力在收集**猎人牌（遗物·含微量「世界之力」·越强的猎人越多）**、想借此**加速暗狱与本世界的融合、把世界变成巨大暗狱**（世界之核 vs 暗狱谁先融合决定世界崩溃还是变暗狱）。苏晓请公会把遗物集中到庇护屋「守株待兔」。天启「**圣堂冒险团**」（165 人·守望公会阵营·被当精英炮灰）派团长 **[阿德利](../../人物/暗狱世界群像.md)**（「有情有义」的憨式交涉）来投——苏晓借机把 31 + 153 个「遗物收回 / 超凡事件」委托批量发出去攒声望。
- **豪鬼劝降 · 老猎人亚当 · 融合之根源**（三十八卷11–13章）：[豪鬼](../../人物/暗狱世界群像.md) 化身来庇护屋劝苏晓「合作·暗狱三分之一归你们」被拒。苏晓循隐藏任务斩狩猎 34 年的**老猎人·亚当**（放逐化晶体尖锥轰碎古堡 + 青影王晶体战镰·得黑暗处刑），发现亚当竟是被一种**黑色吞噬触手**侵蚀（非古神·纯「吞噬」气息·腐蚀痕迹同暗狱锈牌）——揪出**「融合之根源」**（暗狱融合本世界的本体·豪鬼用它吞猎人遗物夺世界之力）：**前面那些猎杀者的死因**正是「融合之根源」的偶发性侵蚀。苏晓采样卖轮回乐园换来「**暗生烙印**」（抵御一次侵蚀）、主线明确 = **摧毁融合之根源本体**。
- **道格拉斯真相 · 莫雷解约**（三十八卷13–15章）：暗狱深处——**融合之根源本体 = 遍布触手、见啥吞啥的黑色巨口**（由意识「魂」维系）；豪鬼唤醒了**警探 [道格拉斯](../../人物/暗狱世界群像.md)**——他竟是**「暗界」（暗狱吞世界后的新世界）的命名者与主导者**（封印自我融入人类社会的存在·唯一认可暗界的「那个人」）！道格拉斯当场发作、杀了警官维娜。庇护屋这边，[莫雷](../../人物/原始大陆群像.md) 拿一枚**心火遗物**（最高级猎人牌·1100 声望）跟苏晓换**解除炎灵星的永久契约**（灰绅士留下的契约恐惧症）、顺势晋升老猎人。
- **雾城 · 守望公会叛徒 · 厄梦**（三十八卷15–16章）：苏晓坐蒸汽火车去**雾城·伊辛格**收两件遗物——路上被**守望公会某高层指使的银发叛徒老猎人**暗杀（假意送牌 + 剧毒短刀·失败自刎）。到雾城后被监视，幽魂 **[厄梦](../../人物/暗狱世界群像.md)**（罕见的最强梯队幽魂）率 10 名三阶段暗魔来抢遗物、想拖住苏晓（布布汪去富商家才是真的取遗物·车站安排是掩人耳目）。

## 主线真相 · 猎人牌与暗狱融合（三十八卷9章）

**猎人牌 = 遗物 = 微量「世界之力」的载体**：这世界只有猎人会留下猎人牌（暗魔 / 幽魂没有遗物），越强的猎人对世界影响越大、牌里的世界之力越多。**未知势力**（豪鬼所属）正满世界搜集猎人牌，想提取其中的世界之力**加速「暗狱」与本世界的融合**——世界之核与暗狱都在融合本世界，若**世界之核先融合则世界崩溃**、若**暗狱先融合则本世界会化为一个巨大的暗狱**（暗魔 / 幽魂的新家园）。这就是那个势力的野心（要的是这世界的一切）、也是前面猎杀者们「危险预知般被提前除掉」的根源。苏晓的对策：把剩余遗物集中到庇护屋，等暗魔 / 幽魂主动送上门、来多少杀多少。

## 融合之根源 · 暗界 · 幕后（三十八卷13章）

**融合之根源**（暗狱深处那颗遍布触手、见啥吞啥、被意识「**魂**」维系束缚的黑色巨口）= 暗狱吞并本世界的本体：它无视暗狱与外世界的壁障、能直接把一小部分探进本世界。豪鬼用它吞掉猎人遗物、提取世界之力加速融合。摧毁它 = 主线完成。**幕后主脑 = 警探道格拉斯**——他是给「**暗界**」（暗狱吞世界后的新世界）命名、封印自我融入人类社会的存在，被豪鬼唤醒；守望公会宁愿世界毁灭也不许暗界出现，唯有「**那个人**」（藏在守望公会机密里、把苏晓的「白夜」化名告诉豪鬼的人）认可暗界。豪鬼「毁灭」、克不住苏晓「锋利」，才请出道格拉斯 + 守望公会内应联手拖住苏晓。

## 🚧 进度

本世界**进行中**（三十八卷16章）。苏晓 = 伪装神灵猎人、揪出「融合之根源 + 幕后道格拉斯」的主线核心（得暗生烙印护身）；正收集最强梯队猎人遗物（雾城两件）当诱饵 / 换声望，被守望公会叛徒 + 幽魂 [厄梦](../../人物/暗狱世界群像.md) 围堵。悬念 = 摧毁融合之根源本体（在暗狱内）/ 道格拉斯 + 豪鬼 / 守望公会里的「那个人」/ 追梦人七星称号。
