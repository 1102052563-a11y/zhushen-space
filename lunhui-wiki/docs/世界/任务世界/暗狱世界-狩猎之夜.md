# 暗狱世界·狩猎之夜（第38卷·七阶最危险四世界之一）

[苏晓](../../人物/苏晓.md) 用 [埃尔德的引荐信](../../力量体系/装备道具.md) 主动进入的**七阶最危险四个世界之一**——难度 Lv.42~70、**世界之源恒 0%**（改按「世界声望」综合评价结算）。这是个「神秘与平凡共存」、一切藏在黑夜里的**克苏鲁调性蒸汽 / 电气时代世界**；已有 **10 名猎杀者 / 14 战斗天使 / 12 先驱 / 11 处刑者**死在这里的主线上。

## 世界背景 · 暗狱之门（世界简介）

「**暗狱**」本是与本世界隔开的独立空间——混乱纪元暗狱之门开启，**暗魔**（七阶战力·破坏 / 杀戮欲强的智慧生物）与**幽魂**（稀少·寻常攻击无效·惧灵魂伤害·可控制任何人）从中涌出，隐入人类世界猎食灵魂。几大帝国团结反抗百年无果，只能「习惯」它们、学会夜间不外出。后有「**守望公会**」与「**守夜人**」（夺取暗魔 / 幽魂力量的人类超凡者·近战系夺暗魔力量 / 灵魂系夺幽魂力量）在夜里狩猎——却因常年杀戮渐渐失控、被人们唾弃驱逐。直到「**无阳之日**」：暗狱不再只是开门，而是**与本世界融合了一部分、且还在继续**——守夜人们又穿上旧风衣重新狩猎（如今台面上英俊貌美的「守夜人」只是公会脸面·真正夜猎的仍是无名之人）。

## 守望公会体系

- **守夜人 / 猎人**：夜里狩猎超凡怪物的无名之人（早期叫守夜人 / 守墓人·今统称猎人）；听命守望公会。莱茵王国国王远见——把平民与超凡完全分割（平民极难接触超凡·警官接手后转交公会派猎人）。
- **超凡事件四级 × 暗魔皮色**：三等（低危·**灰皮**暗魔·憨·新晋猎人）< 二等（危·**暗红**暗魔·资深猎人）< 一等（高危·**黑皮**暗魔·老猎人）< 特等（极危·**有称号 / 名字**的暗魔·屠杀过太多平民·须 10 老猎人 + 50 资深猎人）。
- **庇护屋**：守夜人补给点（每市一处·公会无偿供物资）——苏晓成了**「庇护屋（十六）」的管理人**，可接「委托」得声望、也可把积压超凡事件**发布给己方 / 天启 / 圣光 / 圣域契约者**（任务难度 ×10% 声望）。

- **伪装 = 霸主级「神灵猎人·库库林」**（[埃尔德的引荐信](../../力量体系/装备道具.md) 为媒·探测只显霸主模板·旧识契约者也认不出真容）——[莫雷](../../人物/原始大陆群像.md) 带一队天启 / 圣光契约者来投（她本是来「躲某个人」·却把这里当剧情人物的庇护屋放了心），[圣鸳](../../人物/暗狱世界群像.md)（魅力系御姐）想用魅力交涉却撞上「与库库林交涉魅力特性全无效」的铁壁（侦测得他魅力是自己的 860 倍、显出力 181 / 敏 181 / 体 180 / 智 180 的霸主资料）。
- **主线任务「隐秘之物」**：找到暗狱与本世界融合的**原因**并中断（16 天·奖「豁免徽章」免除魅力 / 意志 / 幸运 <0 的减益〔−10 内〕·完成前无法离开）——苏晓判断这背后**大概率有人主导**（否则前面那些猎杀者不会连敌人是谁都不知道就惨死），压箱底的杀手锏是只能用一次的 [航海罗盘](../../力量体系/装备道具.md)。

## 主线真相 · 猎人牌与暗狱融合（三十八卷9章）

**猎人牌 = 遗物 = 微量「世界之力」的载体**：这世界只有猎人会留下猎人牌（暗魔 / 幽魂没有遗物），越强的猎人对世界影响越大、牌里的世界之力越多。**未知势力**（豪鬼所属）正满世界搜集猎人牌，想提取其中的世界之力**加速「暗狱」与本世界的融合**——世界之核与暗狱都在融合本世界，若**世界之核先融合则世界崩溃**、若**暗狱先融合则本世界会化为一个巨大的暗狱**（暗魔 / 幽魂的新家园）。这就是那个势力的野心（要的是这世界的一切）、也是前面猎杀者们「危险预知般被提前除掉」的根源。苏晓的对策：把剩余遗物集中到庇护屋，等暗魔 / 幽魂主动送上门、来多少杀多少。

## 融合之根源 · 暗界 · 幕后（三十八卷13章）

**融合之根源**（暗狱深处那颗遍布触手、见啥吞啥、被意识「**魂**」维系束缚的黑色巨口）= 暗狱吞并本世界的本体：它无视暗狱与外世界的壁障、能直接把一小部分探进本世界。豪鬼用它吞掉猎人遗物、提取世界之力加速融合。摧毁它 = 主线完成。**幕后主脑 = 警探道格拉斯**——他是给「**暗界**」（暗狱吞世界后的新世界）命名、封印自我融入人类社会的存在，被豪鬼唤醒；守望公会宁愿世界毁灭也不许暗界出现，唯有「**那个人**」（藏在守望公会机密里、把苏晓的「白夜」化名告诉豪鬼的人）认可暗界。豪鬼「毁灭」、克不住苏晓「锋利」，才请出道格拉斯 + 守望公会内应联手拖住苏晓。
