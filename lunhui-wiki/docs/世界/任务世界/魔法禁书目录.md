# 魔法禁书目录·暗执事（第 13 衍生世界）

[苏晓](../../人物/苏晓.md) 的**第 13 个衍生世界**，也是他**晋升三阶的「试炼考核」世界**（因上个世界是战争世界、三阶试炼改在衍生世界内进行·世界初步定三阶初级·难度 Lv.20~23）。卷名「**魔法禁书目录·暗执事**」——融合《魔法禁书目录》与《黑执事》两部作品的世界。

## 进入与「保护机制」（十三卷1章）

- **入场判定**：`【衍生世界：魔法禁书目录开启·本世界无异常·空间坐标稳定·消耗时空之力 0.021 盎司·本世界最低综合评价需达 C− 级·衍生世界联络平台剩余发言 8/8】`。
- **被困玻璃柱（保护机制）**：苏晓与 [布布汪](../../人物/布布汪.md) 一睁眼便被泡在一座废弃实验室的**淡绿色溶液玻璃柱**里、被**空间之力禁锢**无法动弹（布布汪那根玻璃柱是直径五米的超大号「鱼缸」）。这是衍生世界的**保护机制**——新入场契约者有约 2~3 分钟的禁锢期，期间他不能动、别人也攻击不了他（双向禁锢），待空间之力解除即可行动。同被困的还有一名**寸发、黑皮衣、背着「欧式棺材」的男人**（也无法动弹·解异常技能无效）。
- **同场契约者**：附近几名三阶契约者（小型冒险团·以一名大汉为首）先醒，一名女契约者想趁机攻击被困的苏晓，被大汉喝止「那是乐园的保护机制、别招惹」。

## 世界观 · 魔法与科技共存（十三卷2章）

**三阶半开放世界**（力量体系在三阶~四阶之间·不会出现超四阶的人物·只三阶契约者可入·难度 Lv.15~?·世界之源 0%）。魔法与科技两大阵营并存、互不干扰（个别势力例外）：

- **魔法阵营**：以罗马正教、英国清教为代表，势力错综复杂。魔法师把魔力灌进血管与神经，靠身体动作 / 手势 / 符号 / 卡片摆出术式即可施法、强者还能借地形布大威力魔法。
- **科学阵营 · 学园都市**：230 万人口（八成是学生）的巨型学生都市、分 23 学区、与外界隔离、科技领先世界 20~30 年（实则更多），由「**统括理事会**」管理、对人口进出严格限制。表面以「记忆术 / 背诵术」开发超能力，实为静脉投药 + 电极通电 + 耳机催眠的人为开发（[苏晓](../../人物/苏晓.md) 评其为「大型合法试验场」）。因多重能力对大脑负担太大，**每名超能力者只能用一种能力、且不可变更**。
- **超能力者七级**：Lv.0 无能力者（占六成）→ Lv.1 低能力 → Lv.2 异能 → Lv.3 强能力 → Lv.4 大能力者（战术级·军队价值）→ **Lv.5 超能力者（战略级·可单挑一支军队）** → Lv.6 绝对能力者（「以非神之躯抵神之领域」·尚无人达到）。能力者身体多与普通人无异，靠电 / 空间 / 元素 / 力场 / 精神等能力作战。
- **两大首脑**：**亚雷斯塔·克劳利**（科学侧首领·学园都市理事长·几乎永久镇守此地·原著情报推断为**四阶顶级、不可招惹**·其「滞空回线」监控遍布全城）；魔法侧则以罗马正教 / 英国清教为首。

## 试炼 + 猎杀双任务（十三卷2章）

- **试炼任务「末灵法典」**（Lv.23·13 天）：得到「末灵法典」（未录入魔道书图书馆的特殊魔法书·遗失 300 年·现位于学园都市）；可找剧情人物 **土御门元春**（科学 / 魔法双重间谍·两边情报中转站·谁都不动他也都不信他）打听。奖励 = **晋升三阶猎杀者**；惩罚 = 全属性 -10（苏晓承受不起·幸运 -8 等于喝水都呛死）。
- **猎杀任务「异常违规者」**（离开本世界前）：清理「异常违规者」——比普通违规者更危险、**生命体征介于契约者与虚空生物之间**（疑与「虚空灭法者 / 古战场同类」一脉相关·待续）；进入苏晓 1 公里范围自动开启追猎。奖励 = 黄金荣誉勋章 ×2 + 权限 +Lv.3。
- 规则：**严禁在本世界提及「乐园」相关**（无效则强制处决）。

## 入场即遭「道具」突袭（十三卷3–6章）

苏晓一现身学园都市楼顶，便被统括理事会暗部组织 **[道具（Item）](../../势力/道具.md)** 盯上、来「清理外来入侵者」（即杀苏晓）。交手中他摸清超能力者的底细，并发现两条关键克制规律：

- **青钢影对超能力者近乎无效**：超能力者的智力「变异」、几乎不衍生法力（[麦野沈利](../../人物/麦野沈利.md) 法力 0/50）——青钢影靠噬灭敌人体内能量造真伤，敌人没能量可噬便几无杀伤。
- **气息外放正好克超能力者**：[气息外放](../../力量体系/技能/index.md) 被动让敌人体力消耗 +27%，而超能力者施展超能力恰恰靠「体力 + 大脑计算」——这成了苏晓对付他们的利器。

## 加入科学侧 · 阵营系统（十三卷7–12章）

苏晓杀掉道具队长 [麦野沈利](../../人物/麦野沈利.md)（世界之源 +5.6%）后，由空间系「领路人」[结标淡希](../../人物/结标淡希.md) 领去见 [亚雷斯塔·克劳利](../../人物/亚雷斯塔·克劳利.md)——伪装成「第三代猎魔家族」、亮青钢影、献「不问缘由替你做事」的筹码，**加入科学侧阵营**：

- **阵营商店 + 阵营贡献值**：加入后开放（商店位置需自找）；完成首领亚雷斯塔的委托可得贡献值，**见剧情人物土御门元春时贡献值至关重要**。
- **超能者排名**：Lv.5 共七人；第 1「一方通行」（矢量操作·可改斩击方向·防御近乎无解·破绽是命中瞬间反向）、第 3「电磁炮」御坂美琴、第 4 [麦野沈利](../../人物/麦野沈利.md)（已殁）。木原数多（猎犬部队领导·亚雷斯塔直属·协助开发一方通行能力·车载御坂美琴克隆体）替亚雷斯塔向苏晓派活。
- **首个委托 = 原著主线开端**：去**击退 / 击杀追杀「茵蒂克丝」的魔法师同伴 [神裂火织](../../人物/神裂火织.md)**（魔法侧圣人）——此时正是主角 [上条当麻](../../人物/上条当麻.md)（Lv.0·右手「幻想杀手」消除一切异能）剧情线的开端。

## 学园都市的「善 / 恶」两极 + 追猎神裂（十三卷17–22章）

苏晓与神裂火织死战断臂后由亚雷斯塔引荐求医，牵出学园都市的另一极：

- **善 / 恶双势力**：[亚雷斯塔·克劳利](../../人物/亚雷斯塔·克劳利.md)（恶·把这里当「大型试验场 / 疯狂计划基石」）并非一家独大——神级医生 [冥土追魂](../../人物/冥土追魂.md)（善·当年救活活埋的亚雷斯塔·视这里为「科学先驱」）是相对温和的另一极，两者亦敌亦友、互不干涉却共享技术。这也解开了「契约者进学园都市后怎么行动」之谜（有相对温和的冥土追魂一系可投靠）。苏晓选择站「恶」的亚雷斯塔（处事风格相合）。
- **第十学区（平民窟）**：地处学园都市边缘、实验动物焚化场 + 少年感化院所在·研究机构荒废后沦为平民窟、治安极差（无能力者集团 = Lv.0 势力·恐怖组织氛围·亚雷斯塔默许其为「废品堆积点」）。神裂火织、史提尔正藏身于此。
- **苏晓的小队「Dark」**：他追猎神裂时组起小队——苏晓 + [布布汪](../../人物/布布汪.md) + 空间系 [结标淡希](../../人物/结标淡希.md) + 从猎犬部队换来的狙击手 [叶戈尔](../../人物/叶戈尔.md)。靠水刑逼供魔法师 [塞丽娜](../../人物/塞丽娜.md)（艳花之蛇）问出神裂藏身的废弃仓库。

## 势力博弈 + 末灵法典真相 + 异常违规者（十三卷23–27章）

- **势力博弈**：苏晓攻仓库斩杀魔法师 [史提尔·马格努斯](../../人物/史提尔·马格努斯.md)（世界之源 →8.8%）后，仓库里跪着的「神裂」是替身——真相是英国清教主教**萝拉·斯图亚特**与 [亚雷斯塔](../../人物/亚雷斯塔·克劳利.md) 二次谈判：禁书目录 [茵蒂克丝](../../人物/茵蒂克丝.md) 暂归科学侧、清教带走重伤的 [神裂火织](../../人物/神裂火织.md)、史提尔成弃子。苏晓配合「神裂已死」的说法、换得**学园都市自由行动权**。
- **末灵法典真相**（试炼任务目标·[土御门元春](../../人物/土御门元春.md) 揭）：又称「邪灵魔法书」——记载一种**凌驾元素之上的「崩坏」之力**、被破坏的东西**不可逆、等于被清除「存在」**（400 年前一魔法师首用即自爆毁城·300 年前另一魔法师自爆把湖变「气态水」〔接触即分解〕、那湖至今是过午夜就吞掉填充物的天坑）。线索指向「**木原一族**」（学园都市大型集团·直属理事会·4000+ 人〔多为研究员·战力几百多是猎犬部队〕）——传闻有人发现了它、要拿它做事。
- **异常违规者 + 七欲之镰**：苏晓的猎杀任务目标「异常违规者」（生命体征介于契约者与虚空生物间）抢先出手——用「**七欲之镰**」从 1 公里外、仅凭照片 / 代号施法，在苏晓手臂留七符号、**放大愤怒 / 傲慢 / 暴食 / 怠惰 / 嫉妒 / 色欲 / 贪婪七情绪**（[心灵之刃](../../力量体系/技能/index.md) 不豁免·非强制控制·杀人于无形）。苏晓靠意志力没暴走、查清后用青钢影激射破解（造他 40% 生命伤但未触发最终效果 → 反射施术者生命上限 -10% / 智力 -3）。这违规者像当年的「红色小丑」、懂猎杀者、进世界就主动猎杀者、疑反杀过猎杀者。

## 闯木原研究所 · vs 最强一方通行（十三卷28–32章）

线索指向「木原一族」，苏晓硬闯 [木原幻生](../../人物/木原幻生.md)（疯狂科学家·私藏末灵法典）的研究所（在**第二学区**·重工业区 / 武器试验场 / 放射区·亚雷斯塔给了通行权限）。研究所如军事要塞（重机枪走廊 + 60 余架 DX-794M 战斗机甲），被苏晓一路斩穿。

- **绝对能力者进化计划（揭）**：让排名第 1 的 [一方通行](../../人物/一方通行.md) 在不同战场杀死排名第 3 的御坂美琴 128 次即可进化到 **Lv.6 绝对能力者**——因不可能有 128 个御坂美琴，便克隆 2 万份取代（实为根本不可能实现的骗局·连一方通行都清楚）。「体晶」「克隆 2 万御坂」皆木原幻生的提案。
- **vs 一方通行**：木原幻生把当看门狗的「实验动物」[一方通行](../../人物/一方通行.md)（学园都市最强·矢量操作·反射并加强一切攻击）放出拦路。苏晓发现 [气息外放](../../力量体系/技能/index.md) 无视其反射、且超能力者「靠体力 + 大脑运算」的软肋——遂打**消耗战**逼其连续高强度反射、意志动摇，再靠气息外放令其**超能力释放失败**、一拳砸碎其下颚（详见两人页）。这也印证「超能力也是一种技能、会被气息外放的 8% 失败概率波及」。
