# 原始大陆·狩猎之地（第37卷·七阶生存试炼·五乐园追猎与狂欢）

[苏晓](../../人物/苏晓.md) 通关 [源之世界](源之世界-源神乡.md) 后进入的**七阶生存试炼世界**——原始森林气候、高烈度超凡区域、多只**霸主生物**栖息地。因苏晓选了延时进入、世界匹配数量骤减，撞上了**由虚空之树公证的「乐园之试炼」**（五乐园混战·生存类世界）。

## 世界背景 · 生命之树（世界简介）

原始纪元，**生命之树**屹立大陆——土著捡食它落下的果实、生命形态质变，自称「**精灵**」；后发现神树在吸走万物生命力，古精灵们愤而**纵火烧断生命之树**、化为焦土上的灰烬（先驱者），而生命之树临终孕育出的一只**火焰巨龙**为它陪葬。上千年后大陆重焕生机、植被异常高大（水中都含微量治愈之力），成为**霸主生物 + 原始土著 + 超凡生物**三方主宰、彼此争夺资源之地。古精灵将延续的希望托付给了曾经的仇敌——残存至今的人类部落。

## 乐园之试炼 · 规则

- **五方参与**：轮回 / 天启 / 圣光 / 死亡 / 圣域五乐园、**324 名七阶契约者**（含战斗天使、先驱等）——虚空之树公证、**乐园烙印气息被彻底屏蔽**（无法侦测彼此乐园 / 阵营·全凭观察猜测·易误杀己方）。
- **两重身份**：**生存者 300 人**（存活 15 天 / 击杀 1 追猎者 / 完成 3 支线 即过关）· **追猎者 24→25 人**（须击杀 5 名他乐园生存者）——身份不公开、不可侦测、有绝对优先性。苏晓因战力过强**无抽签、强制为追猎者**。历代生存者存活率 11.5%、追猎者 10.7%。
- **物资投放**：每日随机时间投 5~7 箱——**白箱**消耗品（八阶恢复 / 道具 / 卷轴·不可带出）· **绿箱**稀有（灵魂结晶 / 圣灵装备 / 血统物品·可带出）· **蓝箱**=第 4 日投的**生命之龙精魄**（本世界唯一奖励·？？？品质·持有 48 小时永久获得·苏晓志在必得〔提青钢影〕）· 第 10 日一枚**紫箱**（唯一）。基础奖励 = 10 真实属性点 + 圣灵宝箱(30%)。
- **生存禁制**：食物 / 淡水 / 恢复品全封禁在储存空间——治疗系在此极吃香（苏晓想「逮一个」）。**击杀己方契约者无奖励且惩罚 ×3**（苏晓以往行为所致）；击杀他乐园 52.7% 出猩红卡。

## 古精灵 × 第二纪炼金文明（设定 · 三十七卷17章）

蜂女王 [阿芙拉](../../人物/原始大陆群像.md) 的蜂茧里藏着上百部古精灵书籍（如《第二纪炼金·剧毒与药剂——归纳者·精灵·斯特雷奇》）——揭示：远古精灵族曾与虚空的**第二纪炼金文明**炼金师们**互换知识**，古精灵烧毁生命之树的方法可能就来自炼金师，作为答谢古精灵给了炼金师精灵族独有的知识。阿芙拉的进化方向（长得极像人族）正是被这些古精灵知识影响。

## 灰绅士的人偶（重大线索 · 三十七卷23章）

[幻术师](../../人物/原始大陆群像.md) 那飘忽不定的「外挂」真相 = 他是违规者 **灰绅士的「工具人 / 人偶」**——灰绅士对他的远程操控被中断后改为「能力赋予」、让他自由行动，故外挂时灵时不灵、没了远程指挥就对做什么很迷茫；他捞的好处最终都归灰绅士。[苏晓](../../人物/苏晓.md) 由此看穿灰绅士的行事：**不亲自进任何世界、靠散在多个世界的「人偶」获取资源**（比绯世更难缠·人偶数量未知）。[牧枭](../../人物/原始大陆群像.md) 正是趁灰绅士失控幻术师时接近他、想循线追到灰绅士。

## 试炼区真相 · 纳吉·加的野心（三十七卷32–34章）

契约者们其实被困在水源 / 沼泽 / 虎蜂三区拼成的**圆形牢笼**里，唯一出口是沼泽边缘的「**死亡林地**」（灰鸦「死亡凝视」·须白琥珀通行）——外面是**虎蜂之主·泰蜜莎**与**迪亚部落**（十几万朝拜者·宏伟祭坛）的真正世界。[纳吉·加](../../人物/原始大陆群像.md) 抢先探明、靠古精灵祭坛垄断「白琥珀」入场资格：他要杀光追猎者 + 轮回生存者、独占后续物资，还有更大的图谋——古精灵记载「天赋成长到极限后有唯一方法**强行第三次觉醒天赋**」，他认定这场「神仙打架」正是那个机遇（水源区还藏着一只他要伺机瞬杀的旧八阶 boss）。
