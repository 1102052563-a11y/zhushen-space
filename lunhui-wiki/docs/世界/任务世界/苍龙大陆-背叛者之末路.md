# 苍龙大陆·背叛者之末路（第39卷·六族同盟vs混沌军团）

[苏晓](../../人物/苏晓.md) 用维京之海得的 **[永恒意志](../../力量体系/装备道具.md)**（圣光乐园叛徒宝石·通往一个藏着虚空之树果实的世界坐标）+ [门之书](../../力量体系/装备道具.md) 自选进入的**大型全开放原生世界**（Lv.41~70·5~7 阶契约者均可进·世界之源恒 0%·冷兵器 + 术式的战争世界）。他佩戴 [八星称号·掠天惊澜](../../力量体系/称号.md)、因魅力 0.1 + 血气被中立·恶排斥，被恒定分入**混沌·恶阵营「混沌军团」**、并**替换了被封印的第四位大领主「战争恶魔」身份**——由此触发本世界世界之子的强烈敌视。

## 世界背景 · 六族同盟 vs 混沌军团

苍龙大陆（原黑环大陆）千余年的战争史（世界简介逐条）：

- **光明纪**：**多因帝国**（人族·垄断「术式」）雄踞中原，与**极冰领主**（冰裔·极北）、**精灵女王**（精灵族·南部艾尔大森林）+ 边塞的野蛮人 / 矮人 / 龙人族（蜥族）组成 **六族同盟**，占大陆六分之五领土；边塞外的**灰兽人 / 古树族 / 蛇人 / 妖鬼**四族被困苦寒之地。
- **黑暗纪 01 年**：灰兽人用一片黑色铠甲残片，从**无尽炼狱**唤醒了**黑暗君主·[阿伽门](../../人物/苍龙大陆/黑暗君主·阿伽门.md)**、组建**混沌军团**——古树族金属化身体、蛇人熔铸武器、妖鬼注入超凡之力，在「烟囱山」地底繁衍军队。战争持续千年，混沌军团一度占据近半大陆。
- **黑暗纪 1263 年**：黑暗君主·阿伽门被一位**人类英雄**斩首、重囚无尽炼狱，混沌军团群龙无首；**1300 年**：「战争恶魔」被封印进「多因帝国·王都偏郡」的**环封树**，六族同盟才有反攻之势（这封印太突然、连六族同盟都打不开·似乎世界站在了六族同盟一方）。
- **苍龙纪 01 年**：混沌军团被驱逐境外、大陆重迎和平；**苍龙纪 17 年**（世界开始点）：和平已 16 年——**「到底是谁背叛，又是谁主导了封印？」** 正是本世界的核心悬念。

**当前态势**：六族同盟总兵力 600 万+，把战线推成 C 形、将混沌军团（总人口 760 万·能战 203 万·被断粮）半包围堵死在西部烟囱山，靠资源优势「拖到崩溃」。混沌军团再退就是熔岩海（生灵禁区）。

## 苏晓的身份 · 主线 · 真目标

- **伪装 = 混沌军团第四位大领主「战争恶魔·库库林·白夜」**（黑王护臂成伪装代表物）——被多因帝国囚困环封树十几年、对六族同盟六方均为血仇 / 死仇（声望全为负十几万）。他借灰兽人袭击环封树的时机、用 [魂之轻语](../../力量体系/装备道具.md) 操控女士兵长破封脱困。
- **主线「大领主 / 战争恶魔」**：第一环 = 回烟囱山黑环要塞重夺大领主位（3 天·5 属性点·已完成）；第二环 = 「你可攻占任何地方，前提是你能做到」（15 天·奖励按占领范围）。
- **真目标 = 找虚空之树果实的「背叛者」**：[永恒意志](../../力量体系/装备道具.md) 记载的背叛者是**多人**（可能混沌军团 / 多因高层 / 精灵），虚空之树果实就在其中某人手中——苏晓借放走大贵族 / [佩佩尼](../../人物/苍龙大陆/佩佩尼.md) 越狱回多因暗查。

## 战功排行榜（本轮核心玩法）

**527 名契约者**进入本世界（多为战功榜而来）——加入阵营、完成阵营任务 / 参与战役均得**战功**，脱离世界时按排名发奖：

- **首位（独有）= 猎龙者套装**（圣灵级套装 3/3·可熔锻为 1/1）——苏晓最想要（现阶段对他提升比虚空之树果实更大）。
- **2~5 名 = 起源石**（名次定品质·交易街 5500~8000 灵魂钱币）；**6~100 名 = 4 真实属性点**。本轮由**天启乐园**提供奖励。
- **强敌**：[灰绅士](../../人物/炎灵星/灰绅士.md)（轮回·「策略游戏」流·多因王都）· **[月使徒](../../人物/苍龙大陆/月使徒.md)**（天启·召唤系·可移动军团·攒 5 万召唤兽）· **[焰火师](../../人物/苍龙大陆/焰火师.md)**（圣域·把任何触碰之物变爆炸物）· [牧枭](../../人物/原始大陆/牧枭.md)（圣光）· [咕噜](../../人物/咕噜.md)（轮回·暗夜套「氪金玩家」·秒杀 [狂兽·加斯克](../../人物/原始大陆/加斯克.md)）。

## 混沌军团 · 军团结构

- **四大领主**（黑暗君主之下）：战争恶魔（苏晓伪装）+ 财政物资的 **[老蛇人](../../人物/苍龙大陆/老蛇人.md)** + 指挥战争部队的 **[阿卡迪亚](../../人物/苍龙大陆/阿卡迪亚.md)**（灰兽人 × 冰裔混血）+ 自称阿伽门之女的 **[希拉](../../人物/苍龙大陆/希拉.md)**（来自无尽炼狱·无实权）。之下为领主（各统 15~30 万）、掠夺者（各统 1~2 万）。
- **四族分工**：**灰兽人**（中坚·数量最多·能打能抗能当炮灰·管饱就卖命）· **蛇人**（苦力 / 锻造·原奴隶出身·受气）· **古树族**（8~10 米战争巨兽·天生盾牌·怕火 + 高度近视〔50 米外敌我不分〕·不吃粮靠吸地下养分）· **妖鬼**（精英·嚎叫震死敌人·吸战场灵魂能量·与古树族互看不爽）。**大领主能力·先古战魂**：召唤号角、征召 5 公里内四族向自己靠拢（冷却 1 天）。

## 进程（第39卷 · 按章）

- **进入 · 破环封树脱困**（1–4 章）：苏晓降临即被封印在多因王都偏郡的**环封树**内（数百层封印）——趁灰兽人袭击、用 [魂之轻语](../../力量体系/装备道具.md) 操控女士兵长 [易芬蒂·吉朵儿](../../人物/苍龙大陆群像.md) 反杀守卫、刃道刀·超·环断破封，斩士兵长甲巴、宫廷老法师，用 [先古面具](../../力量体系/装备道具.md)（浇宫廷老法师血）+ [异种胚胎](../../力量体系/装备道具.md) 拖延、空间阵图带走法师学徒 [佩佩尼](../../人物/苍龙大陆/佩佩尼.md) 脱身。
- **佩佩尼 · 幸运女神诱饵**（5 章）：主线「大领主」激活；发现 [佩佩尼](../../人物/苍龙大陆/佩佩尼.md) 幸运属性异常暴涨（5→12·受**幸运女神**祝福）——苏晓两度想顺手宰她都莫名遭雷劈，判定**幸运女神拿佩佩尼当保命符**，遂将佩佩尼当**采血诱饵**留着。
- **战功榜 · 大领主能力**（6–7 章）：527 契约者的战功榜开启（首位奖猎龙套）；苏晓进战区、大领主能力激活 30%（先古战魂号角征召四族）——征召来 625 灰兽人 + 1 古树人。
- **偶遇 · 首胜 · 塑造英雄**（8–10 章）：偶遇天启契约者 **[希波克](../../人物/苍龙大陆群像.md)**（想带 1 万重步兵去挖矿）——苏晓青鬼 / 环断 / 时**独挑上万帝国军**（37 分钟溃敌·得 7059 战功升第 3·巴哈重伤希波克）；把带头没逃的灰兽人 [乌古](../../人物/苍龙大陆/沃波尔.md) 改造 + 造神成「兽人英雄**沃波尔**」（解决「跟我冲 vs 给我冲」的士气难题）。

- **焱谷要塞攻破 · 斩精灵王子**（14–18 章）：向三大领主要来 30 万灰兽人 + 500 古树族 + 500 妖鬼（老蛇人只凑够 20~30 万人吃 4 天的粮），佩 [战争领主](../../力量体系/称号.md)（士气 60 + 全属性 +10 + 浴血奋战 + 燃魂之力）+ 领主之权威把士气顶满 100——布布汪把 [阿波罗](../../力量体系/装备道具.md) 埋进城墙内**五分钟炸破焱谷要塞**（精灵族最擅守要塞）、六小时近乎全歼 20 万精灵守军，**斩杀请降的精灵王子 [瓦伦](../../人物/苍龙大陆/精灵王子·瓦伦.md)**（震慑六族同盟·得 5.17% 世界之源 + 圣灵宝箱）。焱谷要塞战役 = **19 万战功登顶第一** + 独裁者 / 炼狱先锋军 / 精灵之敌三枚四星称号；缴 41620 吨谷物 + 大量肉干（混沌军团总储备才 6 万吨·贫富差距惊人）运回黑熔城、饥荒缓解。六族同盟六王震动「他回来了」。
