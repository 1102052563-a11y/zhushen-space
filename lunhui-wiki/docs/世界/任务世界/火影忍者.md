# 火影忍者（青钢之崛起）

[苏晓](../../人物/苏晓.md) **晋升二阶后的第一个衍生世界**（第九卷「火影·青钢之崛起」），以漫画《火影忍者》为蓝本。它是苏晓**靠「世界钥匙」道具主动指定**的目标世界——并非乐园原本安排。

> 📖 乐园本要送他进**大剑世界**（高武·剧情人物常爆种·危险），苏晓却用早先 5 万币买下的「[斩首大刀（残缺）](../../力量体系/装备道具.md)」（世界钥匙类道具·使用后可更改即将进入的衍生世界）**改去火影世界**——为的是一个**野心**（见下「为何选火影」）。

- **难度**：**LV.3～??**（**开放性世界**·区域不同难度不同·可遇极强敌·猎杀者单独落点）｜ 世界之源 0% 起。
- **最低评价要求**：消耗时空之力 0.022 盎司·本世界**最低综合评价须达 C− 级**。

- **查克拉**：**精神能量 + 身体能量完美融合**所产生的一种能量，施展忍术（乃至体术）的必要能量。

## ⚙️ 为何选火影（苏晓的野心）
- **青钢影克查克拉**：查克拉是种能量——青钢影**有超过 90% 几率可燃烧查克拉**。忍者一旦被燃烧查克拉，忍术、体术都受影响（与「灭法之影克法师」异曲同工）；而**忍者身体强度普遍不高**（连半神级也不变态），故苏晓近身偷袭胜算高。
- **较高智力 + 精神抗性**可豁免一般幻术（**月读之类的顶级幻术除外**）。
- **核心野心 · 吸尾兽喂吞噬之核**：他新得的 [吞噬之核](../../力量体系/技能.md) 可吸高等生物能量——**尾兽的查克拉**正是绝佳养料。**吸一只尾兽，青钢影或能从 Lv.10 直冲 Lv.20+（甚至更高·Lv.20 必质变）**；吸两只更甚。这是他指定火影世界的根本目的（虽难·但「要有梦想」）。
- **风险 = 时间点**：他最怕落在**战国时代**（上忍多如狗·影级遍地·遇初代火影柱间 / 宇智波斑必死）或**第四次忍界大战**（尾兽在带土 / 斑手里·抢不来）。

## 苏晓的开局（九卷1–3章）
- **位置 = 雾隐之村（血雾之里）**：世界钥匙进场落点看运气、身份要慢慢探索且「极其坑人」——苏晓**醒在雾忍村最深处的地牢**，身份是「**刺杀四代水影失败的刺客『白夜』**」（乐园不会乱安排身份·暗示他确有刺杀四代水影之能·靠青钢影克查克拉）。
- **时间点 = 血雾之里时期**（雾忍最乱）：叛忍组织「**晓**」控制雾忍村、**四代水影被幻术控制**；「血雾之里」指雾忍那段残酷时期——忍校毕业靠**学生互相厮杀**（存活不过半数、甚至仅剩一人）。
- **狱友**：隔壁牢房关着 **[鬼灯残月](../../人物/鬼灯残月.md)**（鬼灯一族·刺杀「照美冥」失败被擒·戴抑制水化的项圈）——话痨、饿到吃尸体，苏晓借他套情报（照美冥「主和派」·残月断言她「现在不是、以后也不可能是」五代水影·伏笔）。
- **实力定位（苏晓自评）**：中忍不是对手、上忍拼命也行、**影级拉开距离他必死**；但**若近身偷袭，凭刀术大师 Lv.21 + 暗紫 +10 [斩龙闪](../../力量体系/装备道具.md) + 燃查克拉的青钢影，连影级都可能死在他手里**；血继限界（白眼 / 写轮眼等瞳术·冰遁等）另当别论。

## 雾隐村政局（九卷3–4章·苏晓落点背景）
- **四代水影 枸橘矢仓已死**：他既是水影、又是**三尾人柱力**——身中 [苏晓](../../人物/苏晓.md)（白夜）**两刀重伤**、后「与外敌作战阵亡」（对外说法）。他一死，**三尾放归自然**（雾忍失去这枚「核武器」）。这正合苏晓「吸尾兽喂吞噬之核」的野心。
- **两派内乱**：雾忍村分裂为 **[照美冥](../../人物/照美冥.md) 的「主和派」**（停内乱 + 对外通商）与 **「主战派」**（停内战转而劫掠他国·占七成忍者·无顶尖首领·四代曾是其首）。两派都想停内战、却为水影之位互不相让。水之国又穷又封闭、地处海岛、常年内乱（战力反而极强、别国不愿来打）。**苏晓这种「不属于任何忍村的外人武力」**因此有了被主和派利用、清除异己的价值。
- **时间点**：约为**鸣人随自来也修炼三年归来、「晓」组织活跃**之时。

## 火影世界设定 · 忍村与大名（九卷10章）
- **大名府 ≈ 公司董事长、忍村 ≈ 旗下部门**：每个大名提供忍村的钱财与资源，**忍村不敢公然挑衅大名府**（否则没钱粮）。大名是**世袭制**（自身另有从不出动的武力），忍者死一批又一批、大名还是那几个。
- **忍村战争 ≠ 大名敌对**：忍村之间的仇杀、大名府**不卷入**（纯属忍者间的战斗）；规定**不可刺杀敌国大名**（雾忍与木叶打得天翻地覆、两国大名未必敌对）。这也是雾忍暗部**不敢破坏大名府的船**的原因。**第四次忍界大战后**这种格局才结束、各忍村开始发展经济。

## 火之国 · 抵岸找势力（九卷12–14章）
- **开宝箱 / 影刺**：航行中苏晓开两宝箱——蓝箱「闪光」出 5000 币 + **[查克拉修炼法（初级）](../../力量体系/消耗品材料.md)**（市价 20 万币），紫箱却开成「空箱」（仅 1000 币·比闪光还罕见）。又借机开发出自创招式 **[影刺](../../力量体系/技能.md)**（青钢影包裹刀芒生半米尖刺·险些把自己刺死）。
- **火之国地理**：火影世界最富饶之地（四季如春·农耕 / 渔猎皆易·少饿死人），却**被四大国包围在正中**（西沙之国 / 北土之国 / 东雷之国 / 南海即水之国），全靠木叶忍村够强才屹立不倒。
- **衍生世界联络平台**（详见 [机制](../../总览/轮回乐园机制.md)）：抵岸即解锁——苏晓由此得知**三大冒险团（[神皇](../../人物/神皇.md) / 血门 / 蛮天）在火影世界争夺「[猎魔套装](../../力量体系/装备道具.md)」、在木叶外森林大乱斗**，[咕噜](../../人物/咕噜.md) 抢到猎魔戒后藏匿（被血门 [荒焚](../../人物/荒焚.md) 焚烧追捕）。苏晓潜水不发言（神皇 / 咕噜认得他）。
- **找势力被拒**：主线「抉择」限期将到（不加入势力即处决）。苏晓排除音忍村（大蛇丸·解剖风险）、各弱小国，选去**铁之国**（武士用刀·擅能量包裹武器·中立国）——却被首领 [三船](../../人物/三船.md) **以「不收名声在外的刺客」婉拒**（苏晓已是火影顶尖恐怖分子）。

## 加入「晓」组织（九卷15–17章）
铁之国碰壁后，苏晓在雪原遇 **[宇智波鼬](../../人物/宇智波鼬.md) + [干柿鬼鲛](../../人物/干柿鬼鲛.md)**（[晓](../../势力/index.md) 组织·奉首领之命来招他）——苏晓**干脆答应加入**（晓比音忍村好、晓内几无契约者、又能躲雾忍暗部）。

- **正式成员 · 牵制人柱力**：首领 [长门](../../人物/长门.md) 给他正式成员身份（空闲戒指），用途是**近身牵制人柱力**（捕尾兽计划的关键空缺·原担此任的「枇杷十藏」已阵亡）——这恰合苏晓「吸尾兽查克拉喂青钢影」的野心（或能吸两只以上尾兽）。
- **晓 / 雨忍村**：晓是不属任何国家的佣兵 / 恐怖组织（此时期长门统治·以极端手段求和平·善恶不明），大本营在**雨之国雨忍村**（≠水之国·常年降雨实为长门的结界忍术「**雨虎自在之术**」·敌人被雨淋湿即被感知）。苏晓本能用青钢影噬灭雨中查克拉、惊动了鼬与鲛肌。
- **见首领**：到雨忍村，[长门](../../人物/长门.md)（身体枯槁·靠机械维生·操控六具尸体「佩恩」·天道=旧友弥彦尸体）派心腹 [小南](../../人物/小南.md) + 天道下楼迎接，叮嘱「白夜刀术堪比**木叶白牙**、来路不明、别靠太近」（伏笔·苏晓疑与木叶白牙有关）。苏晓需警惕晓内黑绝（幕后 BOSS）、宇智波带土（灭世）、宇智波鼬（间谍·心怀死志）。

## 晓阵营商店（九卷18–19章）
加入晓即解锁**阵营商店**（位于雨忍村后街铁楼·实为雨忍村后勤处·满忍者监视·凭 [空之戒](../../力量体系/装备道具.md) 才能通行·**贡献值**靠执行晓任务赚）。**属于长门的物品用贡献值换、属于其他成员的物品须「以物易物」**。镇店之宝是淡金色 **[空间秘纹](../../力量体系/装备道具.md)**（3000 贡献值·苏晓最想要）；另有 鬼鲛的「水牢鲨鱼游动术」（S 级）、蝎的高等傀儡、鼬的「火遁·豪龙火」、「水遁·水龙弹」（75 贡献值）等几十种（多为低级忍术）。**淡金色 = 暗紫之上的品级**（评分 260~310·已涉空间之力）。

- **群雄陨落**：① **[大蛇丸](../../人物/大蛇丸.md) 被弟子 [宇智波佐助](../../人物/宇智波佐助.md) 所杀**；② **[飞段](../../人物/飞段.md) + [角都](../../人物/角都.md) 被血门冒险团约百名契约者围杀**（晓主力十人已死蝎 / 飞段 / 角都三人）；③ **[自来也](../../人物/自来也.md) 潜入雨忍村调查、死于弟子 [长门](../../人物/长门.md)（佩恩）之手**，蛮天冒险团亦团灭于雨忍村。
- **借刀杀鼬**：苏晓为推进主线（鼬一死、长门便会袭木叶）+ 履行与带土的约定，设局**借佐助之手除鼬**：放归香磷传话「想报仇来神桥山」→ 佐助率香磷 / [水月](../../人物/鬼灯水月.md) / [重吾](../../人物/重吾.md) 寻来、被苏晓环断 + 刀尖压制（幻术对智力 50 的苏晓无效）→ 带佐助去宇智波遗址见 [宇智波鼬](../../人物/宇智波鼬.md)（鼬刻意放水送死），苏晓与鬼鲛 / [带土](../../人物/宇智波带土.md) / 黑白绝旁观。

## 鼬之死 · 1号药剂 · 血战木叶追兵（九卷62–69章）
- **鼬亡 · 大蛇丸「死」而不死**：[宇智波鼬](../../人物/宇智波鼬.md) 决战中天照 + 须佐能乎封印从佐助咒印逼出的 [大蛇丸](../../人物/大蛇丸.md)（大蛇丸分裂小白蛇逃）、最终放水力竭而亡，[宇智波佐助](../../人物/宇智波佐助.md) 惨胜 → 被 [带土](../../人物/宇智波带土.md) 带走收为新成员、告知真相。苏晓收走藏大蛇丸灵魂的小白蛇，以「对外称大蛇丸已死」换得去除柱间细胞木遁的方法 + 实验室。
- **1 号药剂**：苏晓试验千余次后，用柱间细胞 + 炼金药剂制成专属恢复药剂 **[1 号药剂](../../力量体系/消耗品材料.md)**（紫·他人用会树木化）。
- **血战木叶追兵**：[志村团藏](../../人物/志村团藏.md)（右臂柱间细胞感知到苏晓）率「根」来抢初代细胞 → [纲手](../../人物/纲手.md) 赶到喝止团藏、派暗部 + [夕日红](../../人物/夕日红.md) 小队 + 猪鹿蝶（[奈良鹿丸](../../人物/奈良鹿丸.md)/[山中井野](../../人物/山中井野.md)/[秋道丁次](../../人物/秋道丁次.md) 两代）层层追杀。苏晓靠淬毒陷阱 + 「傲歌」能量盾 + 影刺连斩 **[犬冢牙](../../人物/犬冢牙.md)（世界之源 +4.6%→36.2%）、[夕日红](../../人物/夕日红.md)（+3.7%→39.9%）**、斩断女主 [日向雏田](../../人物/日向雏田.md) 一臂（未杀成）、心转心被青钢影驱出令 [山中井野](../../人物/山中井野.md) 精神受创、斩瞎秋道丁座一眼，最后撒炼金炸弹群炸脱身（深感「缺蓝」）。苏晓定下战略：**等契约者返园、任务最后 15 天前再随长门攻木叶**（神皇已离开火影 / 蛮天已灭 / 仅余血门需避开），赴边境见带土。

## 手术救长门 · 战前（九卷69–75章）
- **带土招揽**：[带土](../../人物/宇智波带土.md)（阿飞）向佐助揭真相、逼其开万花筒后，转而招揽 [苏晓](../../人物/苏晓.md)、自称「**宇智波斑**」（点破晓真首领是已死的弥彦）、并透露**长门身受重伤**——这服务于他的「月之眼计划」。苏晓未答应（主线还得靠长门），却警觉去查。
- **长门垂死**：[长门](../../人物/长门.md) 杀师后与契约者交战、被一名四阶契约者留下**胸口腐烂伤 + 寄生甲虫**，命悬一线（六佩恩只能控三、间歇昏迷）。
- **救命换毁灭木叶**：苏晓以治疗换长门答应毁灭木叶 → 用 [1 号药剂](../../力量体系/消耗品材料.md) 续命、调 **[暗紫猛药](../../力量体系/消耗品材料.md)**（[香磷](../../人物/香磷.md) 血 + 柱间细胞）做「无证手术」：拔 87 根金属棍、徒手校正脊柱、取胸腔甲虫、重镶 4 根金属棍——**把长门救成能自由行走**（[小南](../../人物/小南.md) 全程当助手）。长门承诺康复后第一时间攻木叶、带苏晓同往。
- **终得空间秘纹**：借这份人情，苏晓用 1720 贡献（原价 3000）换得淡金 **[空间秘纹](../../力量体系/装备道具.md)**（封 50 米空间技·烙印部位待定）。另：[咕噜](../../人物/咕噜.md) 为躲追杀躲进木叶监狱（料定长门将攻木叶、再撑十天回归）。战前的火影世界，只待长门康复发动**毁灭木叶**之战。

## 毁灭木叶 · 通关（九卷76–84章·主线最终决战）
- **筹备 + 正面进攻**：[小南](../../人物/小南.md) 定计、为长门搭假树操控台；苏晓先清理木叶巡查忍者，六具佩恩以「畜生道高空抛入 + 通灵集合」潜入、瞬间把木叶打成炼狱。
- **苏晓的杀戮**：他不参与佩恩的破坏、径自去火影办公楼搜刮，沿途斩杀木叶忍者；接连斩死为子报仇的 [犬冢爪](../../人物/犬冢爪.md)（三头犬）、与开八门休门死战的 [卡卡西](../../人物/卡卡西.md)（卡卡西起爆符同归于尽失败、被斩首·掉淡金宝箱）。
- **九尾爆走**：[漩涡鸣人](../../人物/漩涡鸣人.md) 因卡卡西之死**爆走九尾**（四尾→六尾→八尾·九喇嘛夺身），死咬苏晓不放；苏晓凭刀术 + 牛头领主之怒勉强周旋、终乘怪鸟脱身（认定半身九尾已非己所能敌）。
- **神罗天征夷平木叶 → 主线完成**：[长门](../../人物/长门.md) 以缩寿大招**神罗天征把整座木叶夷为平地**（破坏 >70%·普通村民死 99%+）→ **主线「凋零之木」Lv.23 完成**（奖 灵魂结晶〔中〕×10 + 12 属性点）。
- **夺猎魔戒**：废墟中重伤的 [咕噜](../../人物/咕噜.md) 把 [猎魔戒](../../力量体系/装备道具.md) 抛向修罗战场想借九尾害苏晓，被苏晓**界断线极限一钩夺回**（结仇·咕噜「记住这张脸」）。
- **回归**：主线达成后苏晓返乐园（强行中断「佩恩袭村」大型剧情事件·该事件奖励作罢·九尾结局留给长门）。
