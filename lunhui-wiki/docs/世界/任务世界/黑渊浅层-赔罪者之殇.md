# 黑渊浅层 · 赔罪者之殇（苏晓的五阶晋升任务弧）

[苏晓](../../人物/苏晓.md) 真实属性满 99 后，为晋升五阶须进 [黑渊·浅层](../地点/index.md)（虚空险地）完成**「99 壁障突破考核」**。他选择**四种真实属性同时突破**（难度飙到 400%·噩梦级），晋升任务为「**灰烬之芽**」。落点恶魔族掌控的 **破碎大陆 · 梵落之城**（友军地盘·盟友），经玛门莫德要塞乘恶魔列车进入黑渊浅层。

!!! tip "黑渊 · 五层与生态"

    黑渊位于**虚空**之内、共**五层**：浅层 / 中层 / 下层 / 深层 / **最深处（生物禁区·刀魔坐镇的生灵花海·独有最高黑枫树）**。太强的虚空种族不能久留浅层（否则被拖入最深层有死无生）。黑渊资源极丰、故狂兽种族盘踞占地为王，是「彻头彻尾的无法地带」。16 处入口由各大种族掌控（恶魔族据其一）。苏晓凭灭法者传承的本能认知，把黑渊浅层霸主级的黑亚龙当「晚餐」（钻其胃囊内斩杀）。

## 晋升任务「灰烬之芽」

- **任务**：寻找灰烬之芽并带回轮回乐园·期限 20 天·奖**四主属性突破 99 壁障**·罚四属性永久 −5（惩罚轻 = 反衬 400% 难度下生存极难）。
- **灰烬之芽**：藏于赎罪者高地中心区域、[始祖](../../势力/index.md) 势力守护——是一团会幻化圣扎迦面孔、要人「臣服」的黑色灰烬（苏晓不理会、直接用金属管电流封存带走）。

## 三势力 · 加入阿萨卡（二十三卷9–13章）

赎罪者高地（比梵落之城更大的狂徒亡命徒之地）三大势力：**[黑泽](../../势力/index.md)**（拾荒人升级版·掌水源·人数最多·外围）/ **[阿萨卡](../../势力/index.md)**（杀手团体·中庭·无总部）/ **[始祖](../../势力/index.md)**（最神秘·中心区域·养灵魂藤蔓）。苏晓经杀手 [枭](../../人物/枭.md) 引荐**加入阿萨卡**（用恶魔指环把青钢影伪装成恶魔之力），凭杀手身份自由出入、接终端任务扬名引 [布布汪](../../人物/布布汪.md) 来会合。首单即斩 [盲圣刃 斯特·图尔](../../人物/斯特·图尔.md)（刀术宗师 Lv.17·敏 100·血泣流）——险胜、得 [生灵匣](../../力量体系/装备道具.md)。

## 灵魂藤蔓 · 阿波罗轰祭坛（二十三卷10–22章）

中心区域有株近千年的**[灵魂藤蔓](../生物万族/index.md)**（霸主级远古植物·成长期 89%·灵魂根系产灵魂结晶·成熟结**灵魂果实**〔吃了提升灵魂强度 + 几率觉醒 S~SSS 天赋〕）。恶魔族（[尤利塞斯](../../人物/黑渊配角.md)）与羽族（[残羽](../../人物/黑渊配角.md) / [羽鸦](../../人物/黑渊配角.md)）都来抢灵魂果实——苏晓则连日用 [阿波罗](../../力量体系/消耗品材料.md) 远程轰始祖首领 [圣扎迦](../../人物/圣扎迦.md) 的同化者大军（炸祖坟 179 座 / 同化者 27 万·圣扎迦「被炸习惯了」），逼出局面。期间救活赶来的重伤友军 [莎](../../人物/莎.md)（手术 + 解毒·恢复至五阶战力）。

## 决战圣扎迦 · 取灰烬之芽（二十三卷23–34章）

恶魔族 vs 羽族在中心区域混战（互耗各派一名势均力敌强者·老阴哔默契），苏晓 + 莎 + [莉莉斯](../../人物/莉莉斯.md) 趁乱杀入地下城。决战 BOSS 级主坦 [圣扎迦](../../人物/圣扎迦.md)（灵魂亡者·体 110·六生存技·植物君主）——莎抱摔拖坦、莉莉斯导恶魔之焰内燃、苏晓一刀穿三人 + 生命 ≤25% 时 **[魔刃](../../力量体系/装备道具.md) 即死斩杀**。取得灰烬之芽。莉莉斯锤趴羽鸦、灵魂果实归恶魔族——苏晓用 **20 片黑枫叶**向 [尤利塞斯](../../人物/黑渊配角.md) 换走灵魂果实（恶魔族视黑枫叶如绝种珍宝）、吃下后**觉醒第二天赋「[血之兽](../../力量体系/天赋.md)」SSS**。返园前 [恶魔老太婆](../../人物/黑渊配角.md) 还送来恶魔之罐答谢（顺手放过被她吞掉、与苏晓「同源」的 [呆毛王](../../人物/Saber.md) 一行）。
