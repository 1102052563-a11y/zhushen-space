# 全职猎人：蚁王篇（第十六卷再入）

[苏晓](../../人物/苏晓.md) 的**第 16 个「世界」**（第十六卷「猎人·传说之物，蚁猎人」）——以《全职猎人》**嵌合蚁·蚁王篇**为剧情背景，是他**第二次进入全职猎人世界**（首入见 [全职猎人·绅士同盟](全职猎人.md)〔第十卷·已通关 S−〕）。本世界**无异常**、消耗时空之力 0.02 盎司、**最低综合评价须达 C 级**、衍生世界联络平台 10 次发言。

## 为什么选这个世界（十六卷ch1）

返园后苏晓达成「8 次 S 级评价」获六世界**任选其一**：海贼王·德雷斯罗萨海战 / 一拳超人 / 黑之契约者·地狱门中心 / **全职猎人·蚁王篇** / 校园默示录（半崩损）/ 食戟之灵。他逐一排除——

- **一拳超人**：公认危险、奖励虽丰但全看运气（苏晓对自己运气「很有哔数」）；
- **黑之契约者·地狱门中心**：危险来自环境、走着走着就遇即死判定，最不受欢迎；
- **校园默示录（半崩损·死体病毒高浓度活化）**：生存类、产出多是生化 / 生物类奖励，与苏晓发展方向不符；
- **食戟之灵（98% 地区禁武力）**：要跟剧情人物搞好关系（满身血气的苏晓站主角面前只会被报警），不适合；
- **海贼王·德雷斯罗萨海战**：收获最丰，但苏晓**第二次入海贼时在司法岛杀了多名 CP9**、得罪了海军元帅战国 + CP 部门，以现在实力进去会被 CP 特务围杀，暂不去。

最终选 **全职猎人·蚁王篇**：他与揍敌客家族好感度不低、能去捞好处；蚁王本人虽非他对手，但**蚁王的三名护卫也不是他的对手**，可控。

## 任务

- **主线「蚁王的心脏」（Lv.30·十六卷5章）**：以任何方式获得蚁王·梅路艾姆的心脏（蚁王预计 10 天后诞生·期限 20 天·奖 3 真实属性点 + 灵魂结晶〔大〕×3·罚强制处决）。难度 Lv.30 极高——苏晓单挑不是蚁王对手，206 名同入契约者很可能也来抢心脏。
- **猎杀「[三合体](../../人物/三合体.md)」（已完成·S+）**：见下。

## 开局秒杀三合体 → S+（十六卷2–4章）

苏晓一进世界就追上猎杀目标 [三合体](../../人物/三合体.md)（史上最奇葩违规者·虚空咔伽族战斗种族厄塔斯 + 违规者 + 职工者小橘三意识共体）——靠斩龙闪金色 + **龙影闪瞬移** + 木之灵徒手力量 ×3 侧踢，把求死的战斗种族斩成碎肉。**虚空生物 + 违规者清理、职工者小橘存活被传回乐园**，5 分 36 秒完成猎杀、**评价 S+**（黄金荣誉勋章 ×5）。开局即清掉违规者、苏晓可在本世界放开手脚。

## 探路揍敌客家族（十六卷5–7章）

为评估自己与蚁王的差距、并捞好处，苏晓乘飞艇 + 号泣观光巴士到枯枯戮山（[揍敌客家族](../../人物/桀诺·揍敌客.md) 大本营·竟是旅游景点）——阿姆推开试炼之门、巨犬带路，旧识 [桀诺·揍敌客](../../人物/桀诺·揍敌客.md)（奇犽祖父）挂着「今日不杀」牌接待他。苏晓真正想见的是**马哈·揍敌客**（揍敌客最老一辈·与猎人会长尼特罗不分伯仲·比苏晓强一筹）——想借与他交手来评估「蚁王是否会碾压自己」。

- **隐藏任务「揍敌客家族的诅咒」（Lv.24·十六卷12章·苏晓主动触发）**：第一环 = 与 [马哈·揍敌客](../../人物/马哈·揍敌客.md) 深入枯枯戮山（期限 3 天·奖淡金色套装宝箱 ×1〔100% 随机淡金套装〕·罚马哈好感度 −80 敌对）。苏晓凭与马哈打平手的实力、直接追问其不死秘密而**主动找出**此任务（正常需层层取得家族信任才触发·这是战斗力强大的红利）。

## 与马哈打平手 → 确认可战蚁王（十六卷9–10章）

为评估自己与蚁王的差距，苏晓请揍敌客最老一辈 [马哈·揍敌客](../../人物/马哈·揍敌客.md)（与猎人会长尼特罗不分伯仲）「完成委托」陪练——两人毫不留手地酣战，**苏晓全靠 [龙影闪](../../力量体系/装备道具.md) 两次把大劣势瞬间扭转、打了个平手**。他由此确认：**自己完全有与蚁王正面战斗的能力**（龙影闪让战斗力质变·此前竞技场为省法力少用、没体现出来）；并初步算清——嵌合蚁此刻还弱（他能独自踏平蚁穴）、蚁王三护卫诞生后他 3V3 也能赢、会长尼特罗的「贫者的蔷薇」能炸蚁王半死、阿波罗更能灭之。

## 揍敌客家族的诅咒 + 永恒晶石交易（十六卷11–13章）

马哈道出**揍敌客家族的诅咒**：他自黑暗大陆起死回生后悠久不死，实为**剥夺至亲寿命**（详见 [马哈·揍敌客](../../人物/马哈·揍敌客.md) / [亚路嘉·揍敌客](../../人物/亚路嘉·揍敌客.md)）。他雇苏晓陪他深入**枯枯戮山**（揍敌客大本营所在的熄火活火山）切断与亚路嘉的「某种联系」——苏晓砍价**8 颗灵魂结晶（大·猎人世界称「永恒晶石」·此世界是灵魂结晶高产地）** 成交（他选猎人世界正为刷灵魂结晶·刀术大师升 Lv.50 需 100 颗大结晶）。

## 深入枯枯戮山（十六卷14–18章）

- **集结小队**：席巴 + 伊路米 28 小时把 [奇犽·揍敌客](../../人物/奇犽·揍敌客.md)（12 岁·剧情主角·能无视 60% 毒素 / 电击）从东果托国边境揪回；好友 [杰·富力士](../../人物/杰·富力士.md)（剧情主角·天生敏锐）跟来帮忙却被婉拒留古堡。小队 = 主力（[马哈](../../人物/马哈·揍敌客.md) / 苏晓 / [布布汪](../../人物/布布汪.md)）+ 帮手（[阿姆](../../人物/阿姆.md) / [桀诺](../../人物/桀诺·揍敌客.md)·山顶打通后离队）+ 特殊人员（[亚路嘉](../../人物/亚路嘉·揍敌客.md) / 奇犽·苏晓护奇犽、马哈护亚路嘉）。定金 4 颗灵魂结晶（大）。
- **枯枯戮山 = 异界火山**：揍敌客大本营所在的熄火活火山，揍敌客家族与 **V5 组织**有合约、在此「看守这座特殊火山」（故能光明正大当杀手家族）——马哈说「枯枯戮山根本不是这片大陆的东西」。桀诺以念龙「**龙头戏画**」冲散火山口熔灰、送众人下山；众人跳进岩浆湖（熔灰会抽走表层热量降到 200°·岩浆呈半固态）、打破晶体层、穿火红晶洞（连锁崩塌·阿姆「战车」把布布汪当导弹抛射），抵达 [亞巴耶](../../世界/生物万族/index.md) 盘踞的「异植物王国」溶洞。
- **过亞巴耶溶洞**：声感半植物 [亞巴耶](../../世界/生物万族/index.md)（麻痹毒 + 光合分解 + 超高速再生·弱点畏利器）——马哈上次被困此处一个月。苏晓以刀芒大片斩断触须开路、阿姆抛人过溶洞；更发现**亞巴耶只针对马哈有攻击性**（对苏晓不攻击·投喂半吨鲜肉就喂睡了任他通行）——「马哈到底在这里做过什么不能让人知道的事？」

## 枯枯戮山中心 · 马哈求死 → 隐藏任务 A+（十六卷19–22章）

- **闯异植物领地**：过 [哭嚎之花](../../世界/生物万族/index.md)（音波尖叫·引大群同类·马哈以重伤代价斩灭）、再苦斩三十余小时穿过 [巴伊拉石蔓](../../世界/生物万族/index.md)（包裹中心的超高速再生藤蔓球·靠消耗战拖垮）抵达中心。
- **马哈的真目的 = 求死**：中心拽出一株「**永生白花**」。马哈道出他来不是求生、而是**为揍敌客一族不再被诅咒剥夺寿命而求死**（不死与 [亚路嘉](../../人物/亚路嘉·揍敌客.md) 体内的拿尼加相关·[奇犽](../../人物/奇犽·揍敌客.md) 是唯一能在山内唤出拿尼加的人·带二人是保险）。他**吞白花、被根须缠茧、化成黑尘自尽**（与苏晓开好感度 93·留暗语「T3M77854」嘱苏晓向 [桀诺](../../人物/桀诺·揍敌客.md) 转告族长之事 + 领「小礼物」）。
- **隐藏任务「揍敌客家族的诅咒」完成（A+）**：四选一（永生之花果实 / 时空硬币 / 宗师之力 / 强制脱离豁免书）中苏晓**选了 [时空硬币](../../力量体系/装备道具.md)**（传说级·可进试炼 / 战争 / 原生世界）。他从东侧破壁离开枯枯戮山——桀诺凭暗语态度 360° 转变、恭敬相请。苏晓面临**杀奇犽（剧情主角宝箱）vs 马哈留的礼物**的抉择（杀奇犽则得不到礼物）。

## 赴东果陀国 · 寂灭公爵抢人头（十六卷23–27章）

- **马哈的「小礼物」**：苏晓凭暗语领取马哈留在桀诺处的炭盒——**5 颗灵魂结晶（大）+ 近战肉搏专精 Lv.25 卷轴 + 二星猎人执照**（可出入 90% 国家、免费用 95% 公用设施、进禁区·不绑身份）。这份收获不低于杀奇犽（且奇犽并非完全体）——苏晓由此**放弃杀奇犽**。
- **东果陀国**：版图边缘的**自治国**（穷到 V5 组织看不上、给自治权省得救济；贪乐王室禁一切现代化物品、平民男耕女织却幸福度不低）——正是嵌合蚁出没、主线展开之地。206 名契约者涌来，中小冒险团围杀军团长级嵌合蚁、危险独行侠则在等蚁王诞生。
- **蚁王已诞生**：苏晓追杀嵌合蚁（夺 DNA 的人兽结合体·偏好猎人类）审出——**蚁王·梅路艾姆一天前已诞生**（比原著提前 4 天）、老女王濒死、军团解散。
- **强势抢人头**：东果陀国王都南侧，星火冒险团（光头团长 + 1 米 5 塔盾主坦矮泰）正围杀三护卫之一 [孟徒徒·尤比](../../人物/尤比·孟徒徒.md)。苏晓凭女职工者情报赶到、1 公里外架 [寂灭公爵 +13](../../力量体系/装备道具.md)——**一枪土元素弹打穿五层结界、轰出脸盆大洞**吓退全团；尤比不退反进、被 [阿姆](../../人物/阿姆.md) 迎头撞飞，「抢人头」战正酣。

> **冒险团 vs 苏晓小队的战斗方式**：星火冒险团有严谨阵型（主坦 + 近战中心 → 中距离 → 辅助 → 法师 / 狙击 / 奶妈外围·主坦用锁链限 BOSS 突围）；苏晓小队则**几乎无战术**（阿姆吸火力 + 10 层霜冻后撤 → 苏晓单挑 BOSS → 布布汪观望奶 + 关键吸火力），一人一狗一牛变数太多、战术作用不大。

## 迟玫刺杀 + 组「蚁王讨伐队」（十六卷28–32章）

- **空间系女刺客迟玫**：苏晓抢人头后，[迟玫](../../人物/迟玫.md)（误以为他是狙击手）酝酿一刀力求秒杀（五重效果 + 淬毒·刺穿 500 反击盾）却失手——躲进异空间不敢动，被 [布布汪](../../人物/布布汪.md) 埋的 **14 颗高仿磁导雷**（炼金炸弹假货）连炸三雷「玩坏」、断肢逃走。
- **蚁王背景**：[蚁王·梅路艾姆](../../人物/蚁王·梅路艾姆.md) 已占领东果陀国王都、把居民当食物储备，但智商极高、谨慎扩张（血脉记忆：嵌合蚁是被从黑暗大陆赶出来的）。**它已比原著更强**（两天前吞了袭杀三护卫猫女的强化系契约者、得了更强身体）。V5 只派猎人公会来、但若失控、V5 军队会把东果陀国化为辐射地带——苏晓绝不能让嵌合蚁失控。
- **苏晓加入「蚁王讨伐队」**：苏晓凭马哈的二星猎人执照 + 枯枯戮山关键词取信 [艾萨克·尼特罗](../../人物/艾萨克·尼特罗.md)（猎人公会会长·与马哈同档）、加入**蚁王讨伐队**（原著本是尼特罗单挑·现凑成 **5 人围攻**·胜算更大）。其余三名队友是一屋「牛鬼蛇神」——[尘](../../人物/尘.md)（兄弟会全灭后独行的法爷·苏晓旧识·毒舌冷场王）、[老烟枪](../../人物/老烟枪.md)（心怀鬼胎的老好人·起义军首领）、[水咲](../../人物/水咲.md)（爱挑拨的心机婊近战）。**对付蚁王前各自为战、蚁王一死随时大乱斗**。

- **攻王宫开始**：猎人公会 + 冒险团 + 散人攻入东果陀王宫庭院、与各类嵌合蚁混战。[蚁王·梅路艾姆](../../人物/蚁王·梅路艾姆.md) 却在王宫内下棋不出（只把庭院的契约者当杂鱼·待嵌合蚁溃败或强者气息出现才动）。
- **三护卫分工**：讨伐队按 [老烟枪](../../人物/老烟枪.md) 的安排分头清掉两名直属护卫——[尤比](../../人物/尤比·孟徒徒.md)（已被苏晓杀）、[尼飞比特（猫女）](../../人物/尼飞比特.md)（最难·苏晓单挑）、[枭亚普夫](../../人物/枭亚普夫.md)（能分裂·尘 / 水咲 / 老烟三人围攻），尼特罗负责拖蚁王。苏晓**伪装成「狙击手 + 召唤」**（亮寂灭公爵）迷惑队友、好在终战时远程 + 近身两便。
- **杀猫女尼飞比特**：猫女正在王都东侧地道追杀洗劫了王宫宝库的 [温妮莎](../../人物/温妮莎.md) 小队，苏晓从天而降堵截。猫女靠地形 + 直觉 + 拿温妮莎小队当肉盾（1 分 06 秒全灭这队）周旋，险些掐死苏晓——被苏晓**龙影闪瞬移斩首贯心**；她以念能力「黑子舞想」（提线木偶·可复活）诈尸，又被界断线 + 青钢影削散。**斩杀猫女**（世界之源 +12.9% → 25.3%·噬灵者 +30 法力 → 3896·金色宝箱）。
- **迷藏宝箱**：温妮莎抢的「迷藏宝箱」（多环连锁任务最终奖·任务已失败但苏晓得了关键物品）落入苏晓手——可在萨利镇镇长克里多夫处重接任务。苏晓派 [布布汪](../../人物/布布汪.md)（75 魅力·好接任务）+ [阿姆](../../人物/阿姆.md) 去萨利镇完成（2 小时无果则放弃），不让布布汪去抢蚁王心脏（怕被集火秒）。同时尘 / 水咲 / 老烟在庭院围攻被蚁王吃了脏器、无法分裂的普夫。
