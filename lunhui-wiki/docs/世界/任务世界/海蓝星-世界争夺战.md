# 海蓝星 · 世界争夺战（第十九卷 · 四阶 1V2）

[苏晓](../../人物/苏晓.md) 晋升四阶后参与的**第二场 [世界争夺战](../../总览/轮回乐园机制.md)**（第十九卷「硝烟战场·轮回，天启，圣域」）——这次轮回乐园**同时对抗天启乐园 + 圣域乐园（1V2）**。战争世界**无法获得世界之源**、收益改走**乐园功勋**（杀敌方头目 / 将领 1~50、杀敌方契约者 3~5）。本世界亦是苏晓的**第 19 个衍生世界**。

## 三方乐园（十九卷1–3章）

- **轮回乐园（己方）**：崇尚力量 / 自由 / 无拘无束；惩罚机制最狠（主线失败 90% 直接强行处决），故契约者「十个里六个有问题」、是疯子 / 精神病 / 恶徒的高产地——单挑能锤爆天启乐园 99% 的人。组织形式 = 冒险团。
- **天启乐园（敌方·老对手）**：惩罚温和（连败五次才有几率处决），80% 是正常人、妹子多软萌；数量庞大、靠温和机制保住契约者基数（出过硬抗两颗阿波罗的「希女王」这类强者，却常「1 神带 999 坑」）。组织形式 = 各类组织。
- **圣域乐园（敌方·全新对手）**：崇尚**信仰**——金黑长袍、团结到爆表的「神棍」；并非信奉单一神明，「哪个教派强就信哪个」（典型墙头草信徒）；惩罚机制比天启强、比轮回弱。组织形式 = 教派。

## 海蓝星（战争世界设定 · 十九卷3–5章）

蒸汽与纹螺钢的二战级科技星球（陆地武器远超海空·97% 战场是陆战）。中型星球（陆 37% / 海 63%）、两大国 + 一中型国 + 上百小国：

- **格雅联盟（自由之国 · 己方阵营）**：多人种、民风彪悍铁血、唯才是举（乞丐有才也能飞黄腾达）；科技 / 军事碾压、长期对外掠夺资源、名声「拥兵自重血腥野蛮」。首都 **热都（自由城）**。
- **太阳王国（敌方阵营）**：王权 × 神权制、单人种（蓝眼）、重出身、法度严苛街道一尘不染；科技被格雅按着打、靠神权支撑。原信太阳神**阿瑟坦勒**（因不再赐神力被弃）、改信**晨曦女神**、首都改名 **晨光城**。神庭 = 其武力机构（修司 / 主教 / 神使六级·对应格雅尉 / 校 / 将）。**天启乐园**的人疑在太阳王国或那个中型国、**圣域乐园**疑在信神权的太阳王国。
- **关键资源**：**白火药**（普通火药几百倍动能·可低氧爆燃）、**纹螺钢**（硬度堪比钢材 200 倍·熔点 1600℃）——二者结合的枪械 / 坦克威力恐怖（一枪穿十几面墙·纹螺钢坦克群能锤哭一个四阶冒险团）。此世界**个人武力弱于军队**。

## 苏晓的身份 · 杀戮部队（十九卷3–8章）

- **身份**：格雅联盟·第二军团三师·**特殊情况应急与清理战略部队（绰号「杀戮部队」**·敌后暗杀 / 情报 / 爆破 / 镇压暴动·训练死亡率 73%）·**上等兵兼少尉**「**库库林·白夜**」。开局被关在**加克巴克监狱**（海蓝星最危险监狱·身份前主人因任务中射杀亚莱多国外交使被关押）。
- **脱身**：苏晓 12 小时内以正当手段出狱（触发战争分支任务）——监狱典狱长 [斯卡因](../../人物/斯卡因.md)（疤脸·娶了太阳王国公主·狠辣·即将升任杀戮部队团长）与外交部高层**莱宗**（兼情报组织「洛丝祇之眼」首领·洁癖毒舌）合演了一场「假口供」放他出狱、还给三天假。
- **战争任务**：**世界之核**（两环·Lv.38·30 天）——苏晓所在区域暂不必找核（核由侦查队找·[布布汪](../../人物/布布汪.md) 又被分进侦查队、混进冰原狼群·[阿姆](../../人物/阿姆.md) 混进奶牛农场）；他负责的是**战争后期的契约者大乱斗**。食物 / 淡水类物资被全程封禁。

## 追次级空间坐标 · 斩门罗（十九卷9–18章）

- **战争分支任务「清除空间坐标」**：圣域乐园契约者勾结太阳王国间谍 + 格雅叛徒、在热都布置「**次级空间坐标**」（一旦激活、圣域方就能探知世界之核是否在格雅境内·遥遥领先）。苏晓借莱宗手下情报组织「洛丝祇之眼」（毒美人 **莎琳**·枪械大师 Lv.40 + 制毒大师 Lv.36·曾狙杀 2700 米外目标）顺藤摸瓜——擒太阳王国神庭主教斯派克（狂信徒·被「心眼/心灵之刃」免疫精神控制·拷问出坐标在「狐狸占卜屋」）。
- **斩门罗**：占卜屋守着两名圣域乐园契约者。苏晓单挑王牌 [门罗](../../人物/门罗.md)（幸运女神信徒·「强运领域」言出法随、可把斩击转嫁周围）——苦战中识破其弱点（用运势改现实时无法同时转嫁伤害），更靠「被轮回乐园传送的锤击感比你的『晕眩』更狠」**硬扛空气锤不晕**、12 刀连斩击碎其领域 + 金光大招、斩杀门罗（另一名「冷笑男」被特务拖住、最后被苏晓捏断喉咙）。**完成分支任务**：50 乐园功勋 + 圣锡晶石（传说级稀有材料）+ 首次获 **[猩红宝箱](../../力量体系/装备道具/index.md)**（杀敌对乐园契约者掉落·三选一·远胜猩红卡）——代价是把热都商业区炸出直径百米的大坑，被斯卡因「奖励」关禁闭半年（玩笑）。

## 幸运女神 · 上战场 · 太阳王国溃败 · 圣利门要塞（十九卷19–29章）

- **幸运女神**：苏晓开 [猩红宝箱](../../力量体系/装备道具/index.md) 抽到「女神眷恋」卷轴、召出 [幸运女神](../../人物/幸运女神.md)（怕灭法之影·被乐园拽回讲和）、获 [女神眷恋 Lv.1](../../力量体系/技能/index.md)（幸运 2 → 3）。
- **热武器战场（澜川战区）**：海蓝星被判为**四阶高烈度战场**——白火药炮弹瞬间爆炸力强过阿波罗一筹、纹螺钢 + 白火药枪械几乎无坚不摧（仅纹螺钢 / 洛米斯晶体能防）、**士兵持枪即有四阶契约者的攻击力**，苏晓也只能硬扛 8~10 枪。他以杀戮部队**狙击手「白鬼」**身份上阵、配金色狙击枪 [Mr·24](../../力量体系/装备道具/index.md) 与观察手 [小助手](../../人物/小助手.md)（贝蒂·安妮特），步坦协同攻破敌阵地（占领局部阵地也给乐园功勋）。
- **战争个人主线任务「深入敌营」**：占领阵地后触发（Lv.36·15 天·深入太阳王国大本营**击杀神使杜波·依斯**·奖 **黄金技能点 ×2**〔可让吞噬之核等 MAX 能力连升两级质变〕·罚功勋 −500）——苏晓接下、带小助手潜入敌后。
- **太阳王国一夜溃败**：怒川冒险团团长 [青魇](../../人物/青魇.md)（四阶第一老阴哔）用一名女性魅力契约者**搞垮太阳王国权力体系**——掌军权的大神使清晨饮弹自尽、太阳王国前线一夜崩盘（2 机械师被歼·格雅 6 小时推进 4 公里），余部退守边境**圣利门要塞**（全金属·60 米高·驻军 10 万·H 形战壕容 20 万·易守难攻）。圣域乐园方为之懵逼。同战区的轮回乐园「疯子们」各显神通（[咕噜](../../人物/咕噜.md) 绑圣域契约者勒索、[所罗门](../../人物/所罗门.md) 杀进中型国、老烟拉 3000+ 起义军等）。
- **圣利门要塞攻坚战**：神使杜波·依斯进了要塞 → 苏晓必须参战。他在攻坚战里狙杀要塞瞭望口的敌方狙击 / 机枪手刷功勋，同时盘算「**炸掉圣利门要塞可得海量功勋**」——但外部摧毁几乎不可能（300 毫米炮轰不掉渣、阿波罗也难破纹螺钢），只能另谋夜潜之策。

## 夜潜炸要塞 · 斩晨曦女神 · 吞噬之核全激活（十九卷30–35章）

- **夜潜入要塞**：苏晓带 [小助手](../../人物/小助手.md) 夜潜圣利门要塞——清理外围哨兵（界断线勒杀）、靠那名烧伤狙击手一枪打防雷柱跳弹熄灯越过 50 米探照灯隔离带、换太阳王国军装混入。小助手用方言色诱一名副修司带路、在厨房将其捅死（「抱歉，我不是你的菜」）。要塞共十层（地下二层屯军火 / 易燃易爆）。
- **阿波罗炸要塞**：苏晓在二层楼梯下藏 [阿波罗](../../力量体系/消耗品材料/index.md) + 布 [弹射之牙](../../力量体系/装备道具/index.md)；四层暴露后用传说级 +13 斩龙闪刀芒近战屠几百士兵。士兵的探测装置触发弹射之牙、连带引爆阿波罗——**气化了地下一层至地上三层**（但纹螺钢框架只熔 30%、要塞上层下沉、地下二层弹药殉爆），格雅联盟趁势夜袭。
- **斩晨曦女神虚影**：苏晓上五层（军官起居·目标神使杜波·依斯）。四名军官割喉献血召出 **[晨曦女神](../../人物/晨曦女神.md)** 的四米高白色「生命能量虚影」（注视领域：力 / 敏 / 体 / 魅 −20、重力压移速 −50~60%、每秒损血·苏晓凭真实属性多半豁免）。苏晓识破它是「一大股生命能量」、硬冲到身前用青钢影按其小腹发动吞噬之核——**伴生的刀魔虚影主动现身、一口咬掉吞下虚影的头颅**、「神」当场殒灭。
- **吞噬之核的正确使用方法**：由此苏晓彻底想明白刀魔本质（= 吞噬之核伴生·无意识·青钢影压制·马文称其「足够强就是能力之一」），并**首次全激活吞噬之核**（蓝烟裹全身·视野化黑白·能直接「看见」周遭能量、甚至透视异空间的金色能量与五层深处一股守着什么的强大生命能量）——详见 [吞噬之核 · 刀魔真相](../../力量体系/技能/index.md)。

## 巴克五五战 · 要塞倒塌 · 深入太阳王国（十九卷36–41章）

- **吞虚影升青钢影**：苏晓吞下晨曦女神虚影后，青钢影从 Lv.40 连升到 **Lv.45**（刀魔休眠期间近乎可无限用吞噬之核·见 [青钢影](../../力量体系/技能/index.md)）。
- **巴克五五战**：神使 [杜波·依斯](../../人物/杜波·依斯.md) 已弃要塞乘飞机逃、留贴身猛将 [巴克](../../人物/巴克.md)（神罚体质 8000 血 + 斧之大师·力 98）殿后。苏晓与他在五层**硬碰硬五五开**（速度占优、却拿不下 8000 血 + 嚼木回血的他）；最终趁要塞下沉，巴克主动跃下、抓纹螺钢框架荡进地下通道脱身殿后。机敏的 [小助手](../../人物/小助手.md) 则乘在杜波的飞机下方追踪（吞信号发射器报信）。
- **要塞倒塌 + 奖励**：中空的圣利门要塞终于**轰然倒塌**——【摧毁程度 86%·起至关重要作用·**800 乐园功勋**·权限升 Lv.38·获传说级宝箱 [战争的馈赠](../../力量体系/装备道具/index.md)〔稀有奖励含觉醒之书 / 稀有血统 / 黑渊邀请函〕】。
- **深入太阳王国**：苏晓靠追踪小助手方位、绑车深入太阳王国境内（民风其实与各国平民无异、所谓「道德洁癖」只是高层 + 表面），到大城**安贝拉城**——小助手藏在城市下水道（活），苏晓继续追杀杜波·依斯。

## 安贝拉城晚宴刺杀杜波·依斯（十九卷42–48章）

- **布布汪会合 + 情报**：[布布汪](../../人物/布布汪.md) 当上冰原狼狼王后带四只冰原狼小弟与苏晓会合（阿姆留守冰原）——它潜进神庭分部、甚至和 [杜波·依斯](../../人物/杜波·依斯.md) 同桌吃早餐都没被察觉，交出「神庭分部草图 + 杜波作息」。
- **开宝箱**：袭杀前苏晓借幸运 +1 转运开两宝箱——金色战争宝箱得灵魂结晶（大）+ 2 万币 + [枪械专精卷轴](../../力量体系/技能/index.md)（自身枪械能力跨阶到 **专精 Lv.36**）；传说级 [战争的馈赠](../../力量体系/装备道具/index.md) 转盘抽中 [黑渊邀请函](../../力量体系/装备道具/index.md)。
- **晚宴一枪**：杜波·依斯（大神使候选人）赴大富商 **亨纳森** 的私人晚宴争金主，苏晓三方配合——[布布汪](../../人物/布布汪.md) 后厨炸弹制乱、[小助手](../../人物/小助手.md) 扮「大小姐」混场喷煤油引燃、苏晓在斜对面酒店架 [寂灭公爵](../../力量体系/装备道具/index.md)，**一枪 19.67mm 雷属性弹（从制乱到穿墙仅 2.53 秒）轰碎杜波·依斯上半身**——**完成战争个人主线任务**（巴克警告过杜波别去、护卫疤脸女扑救慢了一步）。
- **巴克再追**：[巴克](../../人物/巴克.md) 追不上苏晓、转追小助手（被布布汪 + 冰原狼拦下、小助手负伤遁走），又见东城求援信号弹（追苏晓那伙损失惨重）便转去追苏晓、途中斩杀了被苏晓重创的护卫疤脸女（给其痛快）。

## 斩巴克 · 世界之核现世 · 赶赴西斯巴宁（十九卷49–54章）

- **斩巴克**：完成战争主线（评价 **A+**·得 **黄金技能点 ×2**）后，苏晓在荒郊小河边与追来的 [巴克](../../人物/巴克.md) **终战**——生死一瞬赌赢预判、传说级 +13 斩龙闪一刀斩断其脖颈，破例埋之、命运救赎刻「巴」。
- **战局已定**：圣利门要塞倒塌后太阳王国边境驻军被全歼、格雅长驱直入、还在要塞遗址建临时要塞——太阳王国败局已定（[青魇](../../人物/青魇.md) 仍在其军权体系里搅得圣域乐园方崩溃）。
