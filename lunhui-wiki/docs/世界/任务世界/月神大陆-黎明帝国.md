# 月神大陆·黎明帝国（第28卷·二次觉醒噬灵者天赋·原生世界）

[苏晓](../../人物/苏晓.md) 进入的第 28 个世界——为 **二次觉醒 [噬灵者](../../力量体系/天赋.md) 天赋**而来的 **月之神**原生世界（蒸汽 + 神灵 + 超凡者共存的时代）。

!!! tip "二次觉醒天赋的代价"
    二次觉醒需 **[觉醒之书·二次](../../力量体系/天赋.md)**（**86% 契约者已放弃**·难度随阶位 / 等级提升）。噬灵者已达首次觉醒极值 → 二次觉醒难度再 +30%。轮回乐园给三个匹配世界三选一：**灵魂震颤**（原生·魂鬼 / 突变异种极危）/ **月之神**（原生·有神灵存在）/ **双子·基尔加丹**（虚空·苏晓魅力太低会被双子排斥、保护任务恐变绑架任务）。苏晓选**月之神**。

## 世界设定（月神大陆）

- **难度** Lv.37～Lv.52（原生世界）；**世界之源 0%**（获取难度为同阶衍生世界 16 倍以上）。
- **月之神 = 真实存在的唯一神灵**：信徒虽不见其真容、却能感受其存在，足够虔诚还能得「馈赠」成为**月使徒**（超凡者）——故信仰是帝国批量造忠诚超凡兵的手段，而非精神寄托。
- **三大帝国信同一月神、形象却截然不同**（互骂异教徒）：**黎明帝国**（蒸汽文明发源地·工业发达·苏晓所属·月神 = 女神 / 夜之神）/ **西莫罗帝国**（排斥蒸汽·道德洁癖·排外封闭·月神 = 无性公正之神）/ **洛丹帝国**（热情贫穷·幸福度第一·月神 = 男性自由之神）。三国已战 15 年、刚宣布停战。
- **蒸汽文明（科技树偏科）**：约一战前水平、无石油 / 汽车 / 飞机；基石是 **铬纤金属**（纤维凝合体·韧性极强）；照明用水银蒸汽灯；**蒸汽枪**笨重昂贵（填充 5~10 分钟·几乎不上战场）；士兵 / 月使徒身体素质极强，穿铬纤铠 + 重剑战力可媲美一个营的重机枪手。

!!! note "苏晓的身份：第九部队统帅·库库林·白夜"
    苏晓的临时身份 = **第九部队统帅**（特务 + 暗杀部队首领·战时打探情报 / 暗杀敌国要员·成员都是杀出精神病的隐性杀人狂）。停战后第九部队特权过大、太危险，被老圣王 + 月神女 + 鹰侯联手拔除（仅情报部门「乌鸦眼」被单独保留）。苏晓「曾间接 / 直接致死 17.53 万人·76 种重罪」，因战功太多由老圣王力保、暂关地下 175 米机密监狱。开局即「蹲号子」。

苏晓判断老圣王、月神女都是「超级老阴哔」（老狮子 / 哈罗德梯队），自己被放出来就是棋子，遂顺势利用——**公开自己在找灵魂语者**（反让两位老阴哔放松警惕）。

- **越狱 + 两方召见**（二十八卷1–9章）：苏晓在地下监狱破六道蒸汽门越狱（恰逢老圣王下令释放），救出旧副官 **[菲洛米娜](../../人物/月神大陆配角.md)**（蒸汽炸弹专家·小副官）；下午见 **[老圣王](../../人物/月神大陆配角.md)**（拿到「乌鸦眼」情报组织控制权）、晚见 **[月神女](../../人物/月神大陆配角.md)**（用机械蜜蜂窃听 + 反套路，化解其杀局、还坑死了她的内鬼线索来源）。查明灵魂语者被 **[鹰侯](../../人物/月神大陆配角.md)** 绑到首都——这是老圣王 / 月神女联手除鹰侯的导火索。
- **鹰侯的最后晚宴 + 夺灵魂语者**（二十八卷10–13章）：晚宴上小副官一声「我给各位表演个节目」、苏晓刃道刀·时秒斩鹰侯（得世界之源 9.7%），小副官白磷蒸汽炸弹屠尽满堂宾客；又斩鹰侯的超凡护卫 **魂灵剑·曼**（剥离能力·能拉扯灵魂 / 意识·被晶体层 + 侧踢秒）。攻鹰堡夺下 **[灵魂语者·尼迪亚](../../人物/月神大陆配角.md)**（途中和一群「信神信傻了」的智障月使徒头领打了场乌龙·对方装神秘不直说人在地牢）。
- **审问灵魂语者 + 引蛇出洞**（二十八卷12–15章）：苏晓凭推理逼出关键线索——灵魂语者需「**倾听**」才能找到老圣王 / 鹰侯渴求的「某地钥匙·**意志符石**」（月神女因「勿亵渎神明」独不渴求）。把灵魂语者押到第九部队废弃驻扎地当饵，引来其建立的民间组织 **[灵魂守卫](../../人物/月神大陆配角.md)**（兰 + 红发少女率百余超凡者来救）——苏晓 10 秒团灭、生擒兰 + 红发少女。混乱中越狱的**剥面人 [亚伦·鲁](../../人物/月神大陆配角.md)** 伪装成兰、骗灵魂语者临摹下她脸 / 手背的关键刺青后溜走。
- **反推定位 + 偷意志符石**（二十八卷16–22章）：苏晓盯着灵魂语者把她钓出反应、**反推确定意志符石早被她献给月神女**（换多年庇护）；遂让 [布布汪](../../人物/布布汪.md) 连偷月神女的**月之女神像 → 不朽水晶女神像**（国宝·史诗++·能换 37% 世界之源）把月神女心态偷崩，再「自导自演破案」（费伍德沃溶液 + 巨型蒸汽机械读取空间残影·栽赃给已死的鹰侯）逼月神女去查藏室、暴露意志符石位置；趁月神女庞特洛克晚宴时由布布汪潜 3 号储藏室偷走 **[意志符石](../../力量体系/装备道具.md)**（撞见也来偷符石的盗贼「猫女」·被月神女护卫长树狼 / 地龙塔门擒下）。觉醒第二环完成 → 第三环「**统帅之怒**」(Lv.48)：杀地龙·塔门 / 灵魂语者 / 守门人富兰克分别得地龙之血 / 月之发 / 地之城钥匙。
- **杀灵魂语者 + 富兰克 + 西莫罗局**（二十八卷22–24章）：苏晓回驻扎地一刀斩杀**灵魂语者**（月神修道院声望 −9600·死敌）；又夜访服装店斩**守门人 [富兰克](../../人物/月神大陆配角.md)**（密德尔顿家族世代守门人·灵魂语者竟是他亲姐·得地之城钥匙）。次日作帝国代表赴西莫罗使者「赔款」会谈——使者被洛丹刺客当场暗杀（索尼亚躺枪重伤），实为**黎明 + 西莫罗串通泼脏水给洛丹**（53.9 吨黄金是收买、两国进蜜月期联手孤立洛丹·苏晓险些以为是第九部队残党所为）。
