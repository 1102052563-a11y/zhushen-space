# 陨灭之界·五方混战（第33卷·世界争夺战·万兽大陆）

[苏晓](../../人物/苏晓.md) 觉醒 [青影王](../../力量体系/技能/index.md)（战力攀至七阶出头）后的第一个世界——一场**五方乐园**互相敌对的**世界争夺战**，战场在 **万兽大陆**，烈度「中高」。

!!! tip "为何「五方混战」而非一挑四"
    上次世界争夺战是轮回乐园一挑三（胜）。这次除争 **世界之核** 外，还多了一种资源 **时空之力凝合体**（万兽大陆旁一个超巨型世界崩溃、时空之力凝结成块散落各处·不止一块·谁抢到归谁）——正因这资源，**死亡 / 天启 / 圣光 / 圣域四方无法结盟**，五方彼此皆敌、背刺常有。虚空之树为绝对中立方。

## 世界背景 · 万兽大陆

- **异种统治**：兽人 / 人类 / 亚兽人现已弱势、文明断绝。**异种**是地心沉睡的智慧种族——待地表文明发展到一定程度便苏醒、清除 90%+ 人口再遏制其发展（既防统治被威胁、又留猎食对象）。多个纪元前这里是兽族领地（故名「万兽大陆」），兽族因超凡能力威胁异种而被灭绝。
- **异种内战**：一名亚兽人挑拨离间，令本非铁板一块的异种之间爆发**内战**——眼下大陆已是「人脑袋打成狗脑袋」：异种攻打人 / 兽 / 亚兽人领土，异种之间也互相厮杀。
- **五方布局**：五方乐园大本营围成圆形、将时空之力凝合体的陨落区包围（防本土势力来抢）。轮回乐园在最上、左天启 / 右圣光，天启下圣域、圣光下死亡。**圣光乐园**地形最惨（左右皆强邻）。

## 进程（按章）

- **乘飞船弹射登陆 · 1 对 53**（三十三卷1–5章）：苏晓因战力 = 六阶烙印 + 七阶战力（轮回乐园故不必派七阶契约者强入·虚空之树据此公证倾向轮回乐园），获**特殊任务**——独自乘大本营飞船、用弹射舱降落在天启乐园坐标附近，**1 对 53 名天启乐园契约者**：影燃状态下一刀破史诗级盾、三刀斩六阶主坦，先杀坦（软控制领域是他唯一软肋）再杀法系 / 狙击手、最优先杀对他做**魅力判定**的人（魅力仅 1 = 唯一弱点·[行者挽歌](../../力量体系/称号/index.md) 抵一次）——**杀 50 逃 3、得 183 功勋**。天启乐园特色「契约投降」对他无效（轮回乐园战败「喊爸爸都没用」）。
- **大本营会师 · 老阴哔与疯子们**（三十三卷6章）：苏晓回大本营（治愈光环每秒回 1% 生命）与轮回乐园方两百多名「疯子契约者」会合，核心阵容 = **[院长](../../人物/陨灭之界群像.md)**（超级老阴哔·布局）+ **[疯医生 / 蓝大夫](../../人物/陨灭之界群像.md)**（前违规者·**集齐六乐园烙印**·可潜入任何乐园·苏晓负责监督）+ **[寡妇西](../../人物/陨灭之界群像.md)**（布局强于院长）+ **[金刚王](../../人物/陨灭之界群像.md)**（战争巨兽·正面冲锋）+ 飞船班子 [癞蛤蟆舰长 / 1045 号 / 珞珞 / 夏](../../人物/陨灭之界群像.md)。圣域教团已找上门对峙未开打。战略 = 靠疯医生潜入 + 苏晓战力 + 院长 / 寡妇西布局，**先把四方逐一清出世界**（清完就不用抢时空之力凝合体了）。

- **临时联盟 + 自杀小队虐天启 + 甩锅天启**（三十三卷7–11章）：257 名契约者投票组「随时可退」的临时联盟（8 人不入·含 6 独狼）。**[自杀小队](../../人物/陨灭之界群像.md)**（6 个能力诡异敌我不分的独狼）去摧残天启乐园（小胖子「原地去世流」+ 黑魔恐惧传递 / 剪刀即死换命 / 幽灵妹 / 爆炸小男孩·暴击天启士气）。院长 vs 寡妇西分歧、全员投票定**先打死亡乐园**（院长当决策者·失败即被疯子轰杀）。圣光乐园先发现第一颗时空结晶（光柱暴露）→陨石坑四方混战，[布布汪](../../人物/布布汪.md) 融入环境叼走结晶、院长让 [疯医生](../../人物/陨灭之界群像.md) 转天启烙印举结晶**甩锅给天启乐园**（天启背锅四散被追杀·还蒙在鼓里）。
- **死亡+圣域夜袭圣光 + 圣光投轮回 + 向死亡开战**（三十三卷12–15章）：死亡乐园 + 圣域教团结盟夜袭圣光乐园（三号 [绯珊](../../人物/陨灭之界群像.md) 战死·圣光 431→109 残·教团谎称联合实则背刺）——满腔怒火的圣光领袖 [伊莫弥](../../人物/陨灭之界群像.md)（治疗系·能反治敌方致其麻痹）转而**与轮回乐园结盟报仇**（院长试探她是否识局）。教团又派侍者 [乌鸦女](../../人物/陨灭之界群像.md) 来联合轮回去背刺死亡——反被自杀小队「邪活种」玩死（[牧羊人](../../人物/陨灭之界群像.md) 弃子试探）。第三颗结晶陨落，苏晓（乘寡妇西的飞行器高空待命）配合潜伏的疯医生**正式向死亡乐园开战**。

- **清死亡乐园 + 异种宣战 + 血色战场鏖战**（三十三卷16–25章）：苏晓用 [阿波罗](../../力量体系/消耗品材料/index.md) 灭死亡乐园大部队（顺带团灭赤铁冒险团·猎团吞噬），攻其大本营（七阶援军见轮回疯子直接弃任务撤·203 疯子「没怂过」冲锋），破其「临时空间坐标稳定器」**逼死亡乐园付 7500 盎司时空之力强启飞船出局**（全员 +5 真实属性点）。随即**本土土著（异种 + 屈服兽人）大军袭来**——异种是想借时空之力凝合体举族迁离万兽大陆的战争种族（裂变种 / 寄生种高智 + 地行种为悍不畏死的主力·[异王](../../人物/陨灭之界群像.md) 统领）。轮回乐园方大本营成「**血色战场**」主战场：[布布汪](../../人物/布布汪.md) 开冰雪 + 苏醒光环、苏晓独挡南侧（影燃 + 超·环断神技）——鏖战三波 30+ 小时、战死 42、杀异种数十万；其他三方惨（教团 40 人将出局 / 圣光 <20 人 / 天启靠 [无名圣徒](../../人物/强者争霸战群像.md) 抛矿石「买命」拖住）。
- **断补给 + 探索队 + 贝妮找到结晶 → 反攻**（三十三卷22、26章）：揭飞船班子真相——[癞蛤蟆 / 1045 号 / 夏](../../人物/陨灭之界群像.md) 是**探索队**，去崩灭中的「拉美西斯星」夺**蠕行之灵**（杀异王的关键·7 天内）。异种第三波因地行种补给没跟上（饿了难控）而撤，院长抓机会派自杀小队**断异种补给**（小胖子放黑魔把异种粮草吃到剩 1/10·自杀小队死三个半）；贝妮则找到异种要地藏的 **9 颗时空之力凝结体**（任务只需 5 颗）——**反攻正式开始**。

- **反攻达普修要塞 · 夺 9 颗时空结晶**（三十三卷27–37章）：教团全灭（[牧羊人](../../人物/陨灭之界群像.md) 被寄生种围杀）/ 圣光 12 人被生擒 / 天启剩 142 人后，院长分四队反攻异种「达普修要塞」：强攻队（[金刚王](../../人物/陨灭之界群像.md) 兽化 12 米 + 重炮少女萝尼 + 重炮男 + 院长化万乌鸦死亡阵图）破城门吸火力；苏晓抛一颗 [阿波罗](../../力量体系/消耗品材料/index.md) 进要塞**炸死 11.5 万地行种 + 9535 裂变种 + 127 寄生种**（独得 1727 功勋）；[疯医生](../../人物/陨灭之界群像.md) 切寄生种脑前额叶把自己**异种化**潜入。苏晓潜入队（布布汪 / 巴哈 / 自闭兄弟 / 瓶装活地图幽灵妹）深入地下五层——**9 颗时空结晶（异种称「陨星石」）镶在一只雌雄同体的特殊异种身上**，巴哈 2 秒高速空间夺下、布布汪带 8 颗逃（苏晓拎 1 颗当诱饵断后·自闭兄牺牲争 5 秒）。苏晓断后用「[死寂降临](../../力量体系/装备道具/index.md)」秒杀裂变种、**一对三**（斩杀寄生霸主巨猿 [安基度](../../人物/陨灭之界群像.md)、逼退刀术宗师 [深红·塔莎](../../人物/陨灭之界群像.md)、斩毒爪），硬扛异王杀手锏「重甲部队」后猎魔时刻 + 龙影闪脱身——**断后反而血赚**（9 史诗宝箱 + 2 圣灵宝箱）。塔莎惨败欲自尽被异王制止。

- **第二阶段 · 沐光传教场 + 探索队夺蠕行之灵**（三十三卷38–47章）：9 颗结晶上交、第一阶段完成（超额·永久空间坐标解除大本营限制·8 名残废者撤回乐园、142 人留下争世界之核）。苏晓向大陆中心进发找人类势力，循混血 [多萝西娅](../../人物/陨灭之界群像.md) 的指引到隐秘的 **[沐光传教场](../../人物/陨灭之界群像.md)**（失败者苟延之地·真领袖是「死而复生」的亚兽人 [乌钟·泰尔德](../../人物/陨灭之界群像.md)）。揭 **王心**（透支潜力的异种宝物）作用，苏晓用 2 块王心 + 蓝矿造出一只「战力顶七阶」的一次性临时炼金生物。[探索队](../../人物/陨灭之界群像.md) 在西斯星**夺得 2 只蠕行之灵**（腐蚀异王居所「流雾」的关键·苏晓得保管权）——任务 2 失败将全员处决。
- **潜入沃辛格城 · 杀异王（三阶段死战）**（三十三卷48–52章·进行中）：泰尔德赠 [乌铁震钟](../../力量体系/装备道具/index.md)（豁免重甲部队震荡 4 分钟）+ 留守压制城内巨兽。苏晓 + 院长 + 寡妇西三队潜入异种大本营「沃辛格城」西斯古殿（[疯医生](../../人物/陨灭之界群像.md) 已混成指挥官副手接应）；穿越重甲部队巡逻空隙、巴哈缠住 [深红·塔莎](../../人物/陨灭之界群像.md)，苏晓用蠕行之灵腐蚀流雾见到 **[异王](../../人物/陨灭之界群像.md)**（力 150 / 敏 161 / 体 149·力量传承 37 代）。**异王三阶段**：一形态操控空气 / 岩石 / 重力（被苏晓以血破操控 + 放逐侵脑破）→ 吃自己大脑进二形态（传承技法 + 三骨尾 + 七阶速度）→ 苏晓刺穿其第一颗心脏后入**第三阶段**（最强）。苦战中苏晓被斩断左臂 + 多处贯穿、靠 [光之沙漏](../../力量体系/装备道具/index.md) 护身 + 布布汪垫脚两次救命。[寡妇西](../../人物/陨灭之界群像.md) 被最强指挥长曼索所杀。

- **斩异王 + 夺世界之核 + 通关**（三十三卷53–55章）：异王三骨尾抽断苏晓双臂、要捏头 + 刺心——苏晓**故意松开斩龙闪**，以血之兽逼退异王、用放逐操控脱手的斩龙闪从后心（被自造炼金生物咬破的甲片处）**刺穿异王两颗心脏**斩杀（世界之核就在「传承心脏」内）。临时空间坐标生成→**世界之核融合空间坐标·万兽大陆归轮回乐园·世界争夺战胜利**；淡红大雨肃清沃辛格城全部异种、入侵神灵化身被裁定灰飞。详见下方 🏁通关结算快照。
