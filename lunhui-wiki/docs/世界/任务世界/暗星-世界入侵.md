# 暗星·世界入侵（第42卷·苏晓晋八阶后第 2 个世界）

[苏晓](../../人物/苏晓.md) 以**世界入侵**方式进入的世界——不是常规传送，而是靠职工者 [白羊](../../人物/暗星/白羊.md) 在世界内设的临时坐标闯进来的（第42卷1章）。他的核心目标是**夺取一颗巨大的「时空之力凝结体」**（约 27650 盎司·某超大型崩溃世界的时空之力压合而成）。

## 世界归属 · 时空之力争夺（41卷86–42卷1章）

- **双重公证的混乱**：时空之力凝结体陨落暗星、归属虚空之树；而暗星世界本身归天启乐园（25.27%）+ 圣光乐园（24.74%）+ 虚空之树公证（49.99%）——归属权太乱，虚空之树公证到一半卡住，无法开「资源争夺战」、降级为常规物资争夺（圣光乐园胜出）。此后天启 / 圣光要在暗星内交易时空之力（双方各传契约者小队、用烙印携带时空之力 / 稀有物资交易）——**苏晓要干涉这场交易、抢夺资源**。凝结体尚在圣光乐园公证（5~7 天）、之后 5~12 天内交易。
- **世界入侵特性**：世界之源 +45% / 宝箱 +45% / 任务奖励 +70%、**杀戮权限完全解除**；任务惩罚按天启乐园标准（较轻）而非轮回乐园的强行处决。

## 世界史 · 古神封印

> 世界简介：「不要被表象所蒙骗，坚信之物，可能才是恶的起始。」

暗星曾是人族 / 野兽 / 虫子三族信仰**母神**（孕育之神）的和平世界、文明发展到蒸汽 / 电气时代。暗星纪元 1207 年**古神降临**——阳光失温、文明残败、资源枯竭；踏入其领地者意识狂化、灵魂扭曲、身体异变。母神与古神死战、大半身躯被腐化险陨。人族领袖**光之王**、野兽的**大贤者**、虫子的**狼蛛女王**二次宣战救回母神（光之王失双臂、大贤者不再能聆听自然、狼蛛女王变得暴戾疯狂）。母神提出计划：与光之王交合产下后代、由大贤者的知识养成封印古神的**「容器」**；光之王 / 大贤者 / 狼蛛女王再以灵魂为**「锁链」**缠绕容器二次加固。封印成功——但人们只赞颂母神与三王、没人记得那个「没有情感 / 自我 / 名字」的容器。

- **容器裂痕 · 腐化再临**（暗星纪元 1657）：几百年后容器出现裂痕、腐化再度降临（只要意志坚定、心无对古神的恐惧就不会被腐化·古神尚未脱困）。母神沉睡；三王起分歧——**光之王**主张重培养容器二次封印、**狼蛛女王**主张打开容器与古神决一死战、**大贤者**主张中立苟住。
- **腐化机制**：意识松懈即被古神能量侵入脑部腐化成狂化野兽（黑线蔓延·杀光目之所及）。**晶脂**（稀有水晶提取物·硬通币）戴身上可抗腐化、是与食物淡水并列的生存三要素；[宝石大师·林德伯格](../../人物/暗星/宝石大师林德伯格.md) 发现晶脂融合超凡宝石可大幅提纯抗腐（遗作「伯格之心」传戴上直面古神也不腐化）。

## 主线第一环「光之王」· 两大势力（42卷3章）

- **主线第一环**（Lv.73·4 天·奖真实属性点 ×2·罚真实体力 −3）：找到失踪多年的**光之王**、与其见面获取重要情报。去向只有两大势力知晓：
    - **灵魂钟塔**（光之王一脉）：开发超凡术士力量 / 研究新容器二次封印 / 扭曲增强灵魂对抗腐化（造了无数残次品容器）。
    - **[科多学派](../../人物/暗星/默洛斯.md)**（研究 / 崇拜 / 信仰古神）：理念「打不过就信仰、取信古神送它去祸害别的世界」，甚至「引另一位古神对付古神」——古神降临初期真引来「阴影古神」、旋即被原古神杀掉、只剩头颅做成标本供着。

## 苏晓的行动 · 绑架神女（42卷4–9章）

苏晓在旧王都·塔洛的科多学派分部收买了守分部 30 年的前大神官 [默洛斯](../../人物/暗星/默洛斯.md)（生发秘方 + 「既出言言必果」），随他到丹克市——用古神能量诱饵引出科多学派的 [神女](../../人物/暗星/神女.md)（能感应古神能量 + 光之王位置），[阿姆](../../人物/阿姆.md) 斩杀护卫首领 [孤影](../../人物/暗星/神女.md)、放逐团灭阴影侍者、掳走神女。苏晓用 [无尽黑暗项圈](../../力量体系/装备道具/index.md)（逆向魅力判定·神女魅 135 vs 苏晓 −3·谎言之诅咒 + 黑之狱）审出——**光之王在「白色小镇」**。

## 白色小镇 · 光之王（42卷10–11章）

- **赎回神女 · 钓科多学派**：苏晓向 [诺厄主教](../../人物/暗星/诺厄主教.md) 换回 10 万朗晶脂（买晶脂饰品抗腐化）、又「补偿」他一本只写一半的古神能量研究笔记当钓饵。据神女供词，苏晓经废弃都城**哨光**（八阶危险度·母神 vs 古神决战地）→ **迷失枯林**（S 形符印石开路·困古神、封印容器之地·压制飞行）→ 抵达 **白色小镇 / 苍白小镇**（危险度「高危 / 中立 / 安全」由行为而定）。
- **意志之路 · 据点**：小镇主街是「意志之路」（连续意志力判定）——苏晓无外力硬闯通过（精神力永久 +7、意志力永久 +5），白色小镇遂成他的**安全区据点**（数千光之王旧部虚影躬身退去·锻造铺 / 虫塔 / 书屋激活）。苍白宫殿里挂着母神画像（暗紫肉团「孕育一切之母」），[光之王](../../人物/暗星/光之王.md) 被几百根魂链拉扯成几百块沉眠、由王侍 [月灵](../../人物/暗星/月灵.md)（被巴哈用手机忽悠）守护——苏晓来早、光之王需 6 小时后才苏醒 10~15 分钟。

## 采割「野生资源」· 天启临时烙印（42卷12–13章）

苏晓趁光之王未醒，去收割迷失枯林附近的天启乐园矿工队 [天堂小队](../../人物/暗星/天堂小队.md)——戴白金属面具、持 [罪落天遗](../../力量体系/装备道具/index.md) 换刀风格，一脚踹爆保镖 [赤金](../../人物/暗星/赤金.md) 的科技装甲、砸死并**同化其烙印**得天启乐园临时烙印（120012 号·交易 / 契约权限 + 世界联络平台）。他抢走 165 块光银矿、给队长种蓝印记「养肥再宰」（双倍快乐）。他刻意避开熟人战斗天使**莫蕾**（单向伪装认不出、但战斗风格会露馅·曾与苏晓合作 + 目睹其战血神）。

## 恶魔铁匠 · 树神 · 光之王苏醒 · 破封顺序（42卷14–17章）

- **恶魔铁匠**：白色小镇锻造铺住着灭法者时代的旧盟友 [恶魔铁匠](../../人物/暗星/恶魔铁匠.md)（恶魔族·至尊锋刃顾问·对灭法者愧疚）——成就任务「曾经的友谊」：苏晓离世前凑齐 **3 颗圣灵级以上宝石**即可无偿镶斩龙闪、镶嵌效果 +30%。他还赠苏晓一批锻造书。
- **树神越狱 + 猎神**：光之王苏醒时，封印「过滤网」[树神·柏勒罗](../../人物/暗星/树神·柏勒罗.md)（黑泥浸成的三分之二伪古神·憋了一身古神能量却实战极弱）第四次冲出，被苏晓的**专业猎神流**（抗侵蚀药剂 + 三心脏高震频脉冲失神器 + 界断线魂之丝 + 阿姆冰封 + 斩首）秒杀、被月灵塞回封印。
- **光之王苏醒 · 破封三层**：[光之王](../../人物/暗星/光之王.md) 短暂苏醒——古神封印分三层（容器核心 + 光之王 / 大贤者 / 狼蛛女王的外层魂链 + 树神过滤网）。他不阻止苏晓杀古神、但叮嘱**层层解封的正确顺序：先杀最外层的狼蛛女王 → 大贤者 → 最后光之王**（让古神虚弱·否则突然破碎会放出古神几百年累积的全部力量）。他给苏晓**王之手令**、派善良憨直的王侍 [月灵](../../人物/暗星/月灵.md)（真实敏 220 / 世界之子特性）随行——**主线第一环「光之王」完成**（+真实属性点 ×2）。

- **黑巢十层考验**：黑巢是通向狼蛛女王的十层考验（一路腐化虫 / 螳螂战士 / 甲壳狼蛛 / 八阶「虫战士」〔漆黑人形·无主动能力〕·第十层七彩人面蛾精神魅惑〔刀术宗师豁免〕）。苏晓一路杀穿（月灵路痴走丢、还被环断 / 刃链误伤半小时）——通关得 **+15% 世界之源 + 3200 灵魂钱币 + 六星称号 [虫之荣耀 / 战斗大师](../../力量体系/称号/index.md) + 圣灵级套装宝箱 + 阵营声望「崇敬」+ 女王寝殿钥匙**。
- **斩狼蛛女王 · 破封第一环**：苏晓踹开寝殿门，虚弱的 [狼蛛女王](../../人物/暗星/狼蛛女王.md) 留遗言「**小心大贤者、别信他说的话**」后被斩杀——她魂链扯碎、漆黑古神能量涌出，**封印第一层破**。远方枯萎森林的 [大贤者](../../人物/暗星/大贤者.md)（破封第二个目标）感应到她逝去。

## 人造腐化 · 威胁母神 · 裂空宝石（42卷22–25章）

- **人造腐化收集器**：苏晓不愿当「杀古神让母神坐收好处」的打工仔——用斩狼蛛女王后涌出的古神能量把一只大甲蝎高度腐化成领主级虫战士 [虫领主·哈什](../../人物/暗星/虫领主·哈什.md)（移动的腐化收集器·取残旧容器后弃之暴走）。
- **翻脸威胁母神**：他以人造腐化之力当投名状拜访 [母神](../../人物/暗星/母神.md)（孕育之神·装虚弱想坐收渔利）——母神想拖，苏晓当场翻脸「找到这弄死你」，逼母神交出 **172 完整灵魂结晶 + 灵魂晶核 + [扭曲生命铭文](../../力量体系/装备道具/index.md)**，并给她「合作 / 拿好处 / 别看戏」三选一。
- **裂空宝石 · 嵌合体截胡**：贝妮的晶脂饰品生意做大（阿姆护卫劈三帮派·科多学派开藏库）。苏晓向天启商人 [司羊](../../人物/暗星/司羊.md) 买下不朽级宝石 **[裂空](../../力量体系/装备道具/index.md)**（撕破空间·给恶魔铁匠镶斩龙闪的第一颗）——交易被 [嵌合体违规者一伙](../../人物/暗星/嵌合体违规者.md)（专抢诚信商会）截胡、杀司羊分身丢尸警告，苏晓遂决定反猎他们夺回裂空。

## 主线第三环 · 梦境世界 · 团灭套餐（42卷30–39章 · Lv.75）

- **主线第三环「魂链替代」**：主线揭示 [大贤者](../../人物/暗星/大贤者.md) 那条魂链已被抽离、由 [次代神女·沙塔耶](../../人物/暗星/次代神女·沙塔耶.md) 的魂链替代——这正是大贤者能全盛苟住的原因。任务改为找到沙塔耶（在厄罗河以内）斩其魂链→再回白色小镇斩光之王魂链→彻底破封。8 天时限，大半耗在赶路上。
