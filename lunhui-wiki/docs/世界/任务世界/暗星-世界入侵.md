# 暗星·世界入侵（第42卷·苏晓晋八阶后第 2 个世界）

[苏晓](../../人物/苏晓.md) 以**世界入侵**方式进入的世界——不是常规传送，而是靠职工者 [白羊](../../人物/暗星/白羊.md) 在世界内设的临时坐标闯进来的（第42卷1章）。他的核心目标是**夺取一颗巨大的「时空之力凝结体」**（约 27650 盎司·某超大型崩溃世界的时空之力压合而成）。

## 世界归属 · 时空之力争夺（41卷86–42卷1章）

- **双重公证的混乱**：时空之力凝结体陨落暗星、归属虚空之树；而暗星世界本身归天启乐园（25.27%）+ 圣光乐园（24.74%）+ 虚空之树公证（49.99%）——归属权太乱，虚空之树公证到一半卡住，无法开「资源争夺战」、降级为常规物资争夺（圣光乐园胜出）。此后天启 / 圣光要在暗星内交易时空之力（双方各传契约者小队、用烙印携带时空之力 / 稀有物资交易）——**苏晓要干涉这场交易、抢夺资源**。凝结体尚在圣光乐园公证（5~7 天）、之后 5~12 天内交易。
- **世界入侵特性**：世界之源 +45% / 宝箱 +45% / 任务奖励 +70%、**杀戮权限完全解除**；任务惩罚按天启乐园标准（较轻）而非轮回乐园的强行处决。

## 世界史 · 古神封印

> 世界简介：「不要被表象所蒙骗，坚信之物，可能才是恶的起始。」

暗星曾是人族 / 野兽 / 虫子三族信仰**母神**（孕育之神）的和平世界、文明发展到蒸汽 / 电气时代。暗星纪元 1207 年**古神降临**——阳光失温、文明残败、资源枯竭；踏入其领地者意识狂化、灵魂扭曲、身体异变。母神与古神死战、大半身躯被腐化险陨。人族领袖**光之王**、野兽的**大贤者**、虫子的**狼蛛女王**二次宣战救回母神（光之王失双臂、大贤者不再能聆听自然、狼蛛女王变得暴戾疯狂）。母神提出计划：与光之王交合产下后代、由大贤者的知识养成封印古神的**「容器」**；光之王 / 大贤者 / 狼蛛女王再以灵魂为**「锁链」**缠绕容器二次加固。封印成功——但人们只赞颂母神与三王、没人记得那个「没有情感 / 自我 / 名字」的容器。

- **容器裂痕 · 腐化再临**（暗星纪元 1657）：几百年后容器出现裂痕、腐化再度降临（只要意志坚定、心无对古神的恐惧就不会被腐化·古神尚未脱困）。母神沉睡；三王起分歧——**光之王**主张重培养容器二次封印、**狼蛛女王**主张打开容器与古神决一死战、**大贤者**主张中立苟住。
- **腐化机制**：意识松懈即被古神能量侵入脑部腐化成狂化野兽（黑线蔓延·杀光目之所及）。**晶脂**（稀有水晶提取物·硬通币）戴身上可抗腐化、是与食物淡水并列的生存三要素；[宝石大师·林德伯格](../../人物/暗星/宝石大师林德伯格.md) 发现晶脂融合超凡宝石可大幅提纯抗腐（遗作「伯格之心」传戴上直面古神也不腐化）。

    - **[科多学派](../../人物/暗星/默洛斯.md)**（研究 / 崇拜 / 信仰古神）：理念「打不过就信仰、取信古神送它去祸害别的世界」，甚至「引另一位古神对付古神」——古神降临初期真引来「阴影古神」、旋即被原古神杀掉、只剩头颅做成标本供着。

苏晓在旧王都·塔洛的科多学派分部收买了守分部 30 年的前大神官 [默洛斯](../../人物/暗星/默洛斯.md)（生发秘方 + 「既出言言必果」），随他到丹克市——用古神能量诱饵引出科多学派的 [神女](../../人物/暗星/神女.md)（能感应古神能量 + 光之王位置），[阿姆](../../人物/阿姆.md) 斩杀护卫首领 [孤影](../../人物/暗星/神女.md)、放逐团灭阴影侍者、掳走神女。苏晓用 [无尽黑暗项圈](../../力量体系/装备道具.md)（逆向魅力判定·神女魅 135 vs 苏晓 −3·谎言之诅咒 + 黑之狱）审出——**光之王在「白色小镇」**。

- **赎回神女 · 钓科多学派**：苏晓向 [诺厄主教](../../人物/暗星/诺厄主教.md) 换回 10 万朗晶脂（买晶脂饰品抗腐化）、又「补偿」他一本只写一半的古神能量研究笔记当钓饵。据神女供词，苏晓经废弃都城**哨光**（八阶危险度·母神 vs 古神决战地）→ **迷失枯林**（S 形符印石开路·困古神、封印容器之地·压制飞行）→ 抵达 **白色小镇 / 苍白小镇**（危险度「高危 / 中立 / 安全」由行为而定）。
- **意志之路 · 据点**：小镇主街是「意志之路」（连续意志力判定）——苏晓无外力硬闯通过（精神力永久 +7、意志力永久 +5），白色小镇遂成他的**安全区据点**（数千光之王旧部虚影躬身退去·锻造铺 / 虫塔 / 书屋激活）。苍白宫殿里挂着母神画像（暗紫肉团「孕育一切之母」），[光之王](../../人物/暗星/光之王.md) 被几百根魂链拉扯成几百块沉眠、由王侍 [月灵](../../人物/暗星/月灵.md)（被巴哈用手机忽悠）守护——苏晓来早、光之王需 6 小时后才苏醒 10~15 分钟。
