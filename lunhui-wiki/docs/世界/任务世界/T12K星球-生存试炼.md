# 二阶生存试炼（T12K星球 · 吞噬岛）

[苏晓](../../人物/苏晓.md) 完成钢炼晋升任务、**获晋升资格**后必须通过的一场**生存试炼**——**成功即晋升为二阶契约者**。它**不是衍生世界**（无剧情、无原著背景），而是乐园开辟的**独立场景**；同批参与者共 **501 名**（均为完成晋升任务的契约者），故可预见会演变成契约者大乱斗（八卷10–11章）。

## 试炼规则（八卷11–13章）
- **储蓄空间被封印**：登岛前乐园封锁储蓄空间（如同现实世界）——**食物 / 淡水成稀缺物资**（这正是 Lv.16 难度的核心）；通关后恢复。契约者须提前把武器 / 物资取出随身带。
- **降落伞包 100 乐园币 / 个**：投放前强制购买（封储前最后一次消费·买不起者被直接投出 4000 米高空·几乎必死）。
- **裁判「[鼠大人](../../人物/鼠大人.md)」（虚空暴徒）**：鼠人外形的试炼裁判（吞噬岛边缘有十几名）——金属义体 + 纳米修复 + 激光枪，**仅因一名契约者「目光不爽」就一枪将其爆成血雾**（杀鸡儆猴）；铁律 = **试炼期间禁止离开吞噬岛**（违者处决）。它自称是虚空中某位「**大人**」失踪的部下，疑为乐园雇来的虚空生物。
- **感知误导**：吞噬岛的禁区会以「感知误导」伪装地貌——苏晓落地后一度没察觉自己身处**红土区**（见下）。

## 吞噬岛地形（靶子结构 · 八卷24章·老巴尔地图）
整座岛**像一个靶子**，从外到内分三圈：

| 圈层 | 地带 | 说明 |
| --- | --- | --- |
| 最外圈（黑） | **黑色森林**（占地最大） | **禁区中的禁区**（深处 >Lv.20·勿入） |
| 中圈（绿） | **普通区域** | **最安全、资源最富饶** |
| 内圈（红·靶心） | **红土区域**（正中心·>Lv.17） | 食物匮乏淡水稀缺·两族战争地 |

- **红土区四周一圈悬崖**（几十米宽、深不见底），把它与外圈隔开——**前后各有一条天然石桥通向普通区**（一条已半损毁·只剩一条）。**[巴尔](../../人物/巴尔.md) 的「罗盘」即指向这条石桥**：顺悬崖瞎找要十天，有罗盘则从部落到石桥最多一天。
- **试炼后五天难度陡升**：乐园第二天起逐步加大难度，前五天还能在亚森曼部落侥幸生存，后五天红土区会非常恐怖——故苏晓打算「最后五天前」再离开红土区、先趁机探索。

## 危险区域 / 警告
- **红土区域**：难度超过 **Lv.17**，乐园提示勿入。
- **黑色森林深处**：难度超过 **Lv.20**，乐园提示勿入。
- **⚠️ 虚空生物警告**：岛上可能出现**虚空生物**——「多为不稳定结构、具有未知危险性」，乐园**警告勿与之交流**（伏笔）。

## 血藤战士 / 亚森曼部落的秘密（八卷24–26章·待解）
- **亚森曼部落的暴兵之道**：为对抗虫族的暴兵海，红皮土著靠**惊人生育力**硬扛——女土著每胎产 2~5 个、16 岁生到 30~35 岁；一名战士之妻甚至产下**十五胞胎**；大本营 **40 万 + 人口**（幼儿居多·成年女几万·男丁稀少〔多去前线·断手脚才留部落〕·兄死则嫂由其他兄弟「继续用」）。
- **「血藤战士」**：部落另有一种**产出更强战士**的秘法——产物**失去大部分知觉、嗜血、冷酷、呆板、有太阳时战斗力极强**（苏晓夜里斩杀的那批「精英土著」即此）。**副作用 = 会失控自爆**：体内仿佛点起一颗「小太阳」、把抱住他的配偶瞬间焚成焦炭、自身炸成玻璃化大坑（八卷25章亲见）；长老 **[巴尔](../../人物/巴尔.md)** 疑也是血藤战士（气息时而一股时而两股·眼中藏着绝境的疯狂）。
- **后山的秘密**：部落依一座大山而建、后山正面有几十名呆板土著严守——苏晓直觉**血藤战士 / 图腾藤蔓 / 灵魂结晶 / 黑色种子 / 巴尔兄弟仇怨**的秘密都藏在后山，**一旦揭开恐得海量世界之源**（待后续）。

## 进度
- **八卷10章·试炼前准备**：苏晓在乐园拍卖会拍下双枪 [黑檀木·白象牙](../../力量体系/装备道具.md)（补中距离短板）；剩 1 万币后靠「制 [特级炼金秘药](../../力量体系/消耗品材料.md) 每瓶 6000 卖」滚雪球，三天攒下 5 瓶特级炼金秘药 + 一堆特级炼金炸弹。停留满 3 天 → **生存试炼开启**（501 人）。
- **八卷11–12章·噩耗 / 登机**：落在集合孤岛、501 人汇聚「和平区」。**裁判 [鼠大人](../../人物/鼠大人.md)** 驾超大型飞机来接、当众秒杀一名「目光不爽」的契约者立威，勒令全员登机、禁离岛。机上**浩海冒险团副团长**散资料拉结盟（500 人中 300+ 结盟分小团体）；**[彼岸花冒险团](../../势力/index.md) 6 女**（团长 [冷月](../../人物/冷月.md)、副团长 [血玫瑰](../../人物/血玫瑰.md)、智谋腹黑女、狙击手蜂、奶妈糖果、新主坦罗娜）在机上认出苏晓、**密谋上岛后为「三十二姐妹血仇」复仇**。
- **八卷13章·泥土（投放）**：储蓄空间封印（取物 2 分钟）；**降落伞包 100 币 / 个**（一名买不起的契约者被投出 4000 米高空·几无生还）。苏晓 + 布布汪跳伞——空中遇**五米钢羽黑鹰**（敏 40+·布布汪学鹰叫「交涉」竟打成一片·黑鹰转而抓走那位「无伞兄」）。500 米伞包自动开，落点随缘 = 一片**有五棵几十米高树的草原**（苏晓奇迹未挂树、布布汪倒栽葱落地）。
- **八卷13–14章·误入红土区**：布布汪啃树引出**直径两米的巨蜈蚣**（藏身泥土·血盘大口）——苏晓骑布布汪 + 炼金炸弹逃命。脱险后乐园提示「**已从感知误导中脱离**」，草原泥土显出血红色——他**落点正是禁区「红土区域」（难度 >Lv.17）**！困境随之而来：**无地图、食物在跳伞时丢失、淡水仅够苏晓撑一天**（布布汪靠脂肪能撑两天但需水）；背包仅余几颗炼金炸弹、5 瓶炼金秘药、一份当恢复品的酱汁烤肉。
- **八卷14–15章·红土区求生**：**红土区的水不可饮**（清澈河水满是寄生虫·喝一口必死）、**生物不可食**（怪鱼挂几十种寄生物）——唯拇指大的**蜜蜂**可生食（苏晓生吞补能量·布布汪嫌弃）。指南针在此失效（变「电风扇」），苏晓改用**木棍影子辨位法**（插棍标影顶 + 10 分钟后再标 → 连线得东西向）、设石块路标垂直前进，连设五次走到草原尽头的山地（仍在红土区）。
- **八卷15章·土著营地**：循碳灰 / 骨骸 + 一声惨叫 + 断剑血迹（布布汪嗅味追踪），苏晓发现一处 **[亚森曼人（红皮土著）](../生物万族/index.md)** 营地（43 人·全成年男性·疑被驱逐的小部落）——他们把抓到的契约者割头掏脏风干当干粮，还绑着活的「无伞兄」（与黑鹰冻结落地竟没死、又被土著俘虏当今晚肉菜）。苏晓不为救人、只为夺其**干净的淡水**而决定动手。
- **八卷16–17章·夜狙红皮土著**：入夜土著围篝火（**无夜视**）——苏晓凭 [蜘蛛女皇](../../力量体系/装备道具.md)（8 倍夜视 + 消音）+ [岚凯风衣](../../力量体系/装备道具.md)「夜岚」加成（夜间力 +4 / 敏 +5·**有效六维冲到力 49 / 敏 50 / 体 44 / 智 42**·力 49 无视后坐力）逐一狙杀（土著头硬扛一枪、改打脖颈才致命）；哈瓦呐之眼读出土著属性（体 48·野性 / 血藤蔓图腾·无痛觉·身体强度恐怖）。无伞兄被迁怒的土著一矛贯穿眼窝而死。子弹打光后转近身斩颈（斩龙闪锋利度 9 + 力 49 才勉强斩断·**每杀一名噬灵者 +15 → 法力 1093**）。**布布汪竟耳濡目染学会用蜘蛛女皇开枪**（一枪救主、却把苏晓吓得以为遭敌方狙击手·后坐力让它生命 −15%）。战至能战土著剩 14 人冲来、苏晓把空枪抛回布布汪。
- **八卷18–19章·斩杀土著酋长（高风险高回报）**：苏晓首次以**黑檀木·白象牙连射模式**（双枪各 30 发形成子弹链·冲锋枪射速）把冲前土著射成两截，再以炼金炸弹 + 闪光弹 + 界断线 + 斩颈血战剩余土著（中了红皮土著的**战术阵型**：石斧强攻 + 木矛包抄 + 吹箭首领放**箭毒蛙剧毒**致他半身麻木）；危急时**布布汪死咬吹箭土著的屁股**（高等生物免疫箭毒蛙毒·死不松口）助他翻盘——**斩杀亚森曼人酋长**（红土区关键人物·**世界之源 0% → 8%**·掉「酋长包裹」类宝箱）。搜刮营地大获：**[炭烤巴猡](../../力量体系/消耗品材料.md)**（蓝·1 小时回满生命·首领专享）、几竹筒山泉淡水、陶罐里 **5 颗灵魂结晶（中）** + **[未知生物粪便粉](../../力量体系/消耗品材料.md)**（白·黑森林某生物粪便·气味驱兽·夜里洒营地周围保平安·可用 5 次）。无伞兄逃出部落后竟「跳崖自杀」（地形不熟）。苏晓退守山洞（堵口 + 洒粪便粉）、吃炭烤巴猡补给后深睡。**第一天结束：剩 227 小时·存活 462 人**（生存难度梯度：普通区 2 / 红土区 10 / 黑色森林 ≥50）。
- **八卷20章·突如其来**：第二天中午，山洞外轰鸣——平原上**上万红皮土著大军**正与**[刺蝎（虚空虫族）](../生物万族/index.md)** 群对冲（刺蝎单体弱但靠海量数量「集体作战」加成）。乐园借此公告：**红土区两大主势力 = 亚森曼人（红皮土著）+ 刺蝎（虚空虫族）**（两族瓜分红土区 70%·亚森曼 31% / 虚空虫族 39%·**已战争三百余年**），红土区情报开放 20%、系自然生成、食物匮乏淡水稀缺。剩 215 小时·存活 456 人。
