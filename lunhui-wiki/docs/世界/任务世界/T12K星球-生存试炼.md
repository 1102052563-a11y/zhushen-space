# 二阶生存试炼（T12K星球 · 吞噬岛）

[苏晓](../../人物/苏晓.md) 完成钢炼晋升任务、**获晋升资格**后必须通过的一场**生存试炼**——**成功即晋升为二阶契约者**。它**不是衍生世界**（无剧情、无原著背景），而是乐园开辟的**独立场景**；同批参与者共 **501 名**（均为完成晋升任务的契约者），故可预见会演变成契约者大乱斗（八卷10–11章）。

## 试炼规则（八卷11–13章）
- **储蓄空间被封印**：登岛前乐园封锁储蓄空间（如同现实世界）——**食物 / 淡水成稀缺物资**（这正是 Lv.16 难度的核心）；通关后恢复。契约者须提前把武器 / 物资取出随身带。
- **降落伞包 100 乐园币 / 个**：投放前强制购买（封储前最后一次消费·买不起者被直接投出 4000 米高空·几乎必死）。
- **裁判「[鼠大人](../../人物/鼠大人.md)」（虚空暴徒）**：鼠人外形的试炼裁判（吞噬岛边缘有十几名）——金属义体 + 纳米修复 + 激光枪，**仅因一名契约者「目光不爽」就一枪将其爆成血雾**（杀鸡儆猴）；铁律 = **试炼期间禁止离开吞噬岛**（违者处决）。它自称是虚空中某位「**大人**」失踪的部下，疑为乐园雇来的虚空生物。
- **感知误导**：吞噬岛的禁区会以「感知误导」伪装地貌——苏晓落地后一度没察觉自己身处**红土区**（见下）。

## 吞噬岛地形（靶子结构 · 八卷24章·老巴尔地图）
整座岛**像一个靶子**，从外到内分三圈：

| 圈层 | 地带 | 说明 |
| --- | --- | --- |
| 最外圈（黑） | **黑色森林**（占地最大） | **禁区中的禁区**（深处 >Lv.20·勿入） |
| 中圈（绿） | **普通区域** | **最安全、资源最富饶** |
| 内圈（红·靶心） | **红土区域**（正中心·>Lv.17） | 食物匮乏淡水稀缺·两族战争地 |

- **红土区四周一圈悬崖**（几十米宽、深不见底），把它与外圈隔开——**前后各有一条天然石桥通向普通区**（一条已半损毁·只剩一条）。**[巴尔](../../人物/巴尔.md) 的「罗盘」即指向这条石桥**：顺悬崖瞎找要十天，有罗盘则从部落到石桥最多一天。
- **试炼后五天难度陡升**：乐园第二天起逐步加大难度，前五天还能在亚森曼部落侥幸生存，后五天红土区会非常恐怖——故苏晓打算「最后五天前」再离开红土区、先趁机探索。

## 危险区域 / 警告
- **红土区域**：难度超过 **Lv.17**，乐园提示勿入。
- **黑色森林深处**：难度超过 **Lv.20**，乐园提示勿入。
- **⚠️ 虚空生物警告**：岛上可能出现**虚空生物**——「多为不稳定结构、具有未知危险性」，乐园**警告勿与之交流**（伏笔）。

## 黑色森林 / 死亡之地（八卷52–53章·隐藏任务第二阶段）
苏晓完成隐藏任务第一阶段「太阳部落」后，**激活第二阶段「死亡之地」**（难度 **Lv.17**·进入黑色森林·**连锁任务·按完成度给奖·无期限无惩罚**）。

- **进入门槛**：黑色森林封着「**不死者**」与各种复苏的怪物（比红土区危险十倍不止），进入**必须携带大量「原生之种」当门票**，否则必死。原生之种须经 [阳之女](../../人物/阳之女.md) 改造（黑变淡红、生翠绿枝芽）：进林后随身带、**每两小时捏开一颗续命两小时、十颗用光前必须撤离**（苏晓讨价拿到 15 颗）。
- **苏晓的选择**：本可就地交差领少量奖励，但他**主动迎灾而上**（他的成就皆源于敢深入险地·此时「欧皇光环」仅剩三小时）。**队伍 = 苏晓 + [巴尔](../../人物/巴尔.md) + 阳之女联合的另两名契约者**（巴尔被阳之女种了一种黑森林专用的「一次性能力」、过桥前体温 >80 度即危险、故全程不出手）。

### 黑色森林实况（八卷62–63章）
- **黑雾之「牢笼」**：黑色森林外缘弥漫一层**黑雾**——抛入的树枝**瞬间被分解成烟尘**；黑雾如墙、把整片森林封成一个**牢笼**，**只能从唯一入口进出**（连入口也有黑雾、只是稀薄）。黑雾会**周期性消退**（短则三五天、长则半个月）；**试炼第十天（最后一天）黑雾彻底消失、里面的东西脱笼而出**（即「寒灾」）。如今黑雾已稀薄（几个月前还厚如墙）。进入靠 [巴尔](../../人物/巴尔.md) **吐一分钟白火**在入口烧出窟窿（只维持十几秒·吐完萎靡·入口随即愈合）。
- **黑雾之源·侵蚀**：入林即受「**黑雾之源**」侵蚀——**每秒 1 点真实伤害·满 10 分钟翻倍**（10 分 2/秒、20 分 4/秒、30 分 8/秒、**半小时后每 5 秒一次即死判定**）。**捏开一颗改造原生之种生出翠绿藤蔓攀身即可抵消侵蚀**（坐实「原生之种 = 黑色森林门票」·无它嗑药也撑不过半小时）。
- **内里反差**：黑雾之内的森林**并非死寂**——树木其实不黑（外面看黑是黑雾笼罩）、**鸟语花香、空气带淡淡甜味**，却处处剧毒（一只红色小虫的体液瞬间把匕首腐蚀成千疮百孔；林中野兽皆带毒不可食）。
- **地图真相 / 横穿**：那张兽皮地图**实为亚森曼祖先用鲜血与生命换来**（非巨人所绘·传说阳之女曾用它讨好「巨人族」）。**黑色森林面积不与吞噬岛挂钩**——队伍只需走半圆形横穿半个森林、抵达对面出口即可离岛、**不必返回**（老巴尔语·与阳之女所说不同）。

## 暗王 / 死亡之地最终阶段（八卷64–65章·隐藏任务第三阶段）
入林后乐园提示**隐藏任务第二阶段「死亡之地」完成、激活第三阶段（最终）「[暗王](../../人物/暗王.md)」**（Lv.**19**）：**在 [暗王](../../人物/暗王.md) 彻底苏醒前将其消灭**（暗王在森林深处·**若完全苏醒难度飙到 Lv.28**·连锁任务无惩罚）。

- **同化 / 不死者**：暗王把深入黑色森林的生灵**同化**成妖兽（[野猪](../../人物/野猪.md)、[托马](../../人物/托马.md) 等被同化的前辈契约者·失痛觉近乎不死·靠吃 [不死者](../生物万族/index.md) 存活）；林中**数以万计的不死者**远远包围小队（畏惧某种存在不敢近身、反吓退普通野兽）。亚森曼遗迹石碑残文「**第十三星·寒之·试验·失败……**」、老巴尔所知四词「**寒灾·不死者·暗王·同化**」皆指向此。
- **队伍分裂**：[野猪](../../人物/野猪.md) 主动找上苏晓、提议**带他找暗王换轮回印记**（≥5 枚·它想恢复契约者身份求死）——正合苏晓真目标（暗王=第三阶段任务）。于是**苏晓 + 布布汪跟野猪去寻暗王**；[巴尔](../../人物/巴尔.md)（自承「力量已适应、与苏晓拼命会同归于尽、低调因体内几万炸弹」）带 [无伞兄](../../人物/无伞兄.md) + [小迷糊](../../人物/小迷糊.md) 去 [阳之女](../../人物/阳之女.md) 的「骷髅洞穴」——两路「各求所需」分道（苏晓本就不在乎阳之女救不救吞噬岛）。生态畸形佐证危险：三百头「**犀牛群**」竟是食物链最底层、见苏晓普通气息就吓得四散逃。

## 石桥 / 堵桥（八卷53–57章）
红土区四周悬崖、仅剩一条天然**石桥**（几十米长、十余米宽、无护栏）通向普通区域，是离开红土区的唯一咽喉。

- **堵桥战术**：一队临时结盟的契约者**在石桥前堵了三天**——专杀从红土区脱出、饿到虚脱、战力大降的契约者（普通区资源不算紧缺、堵桥稳赚）。他们还在飞机上发现了一个「**秘密**」、为此险些内讧（秘密内容待揭）。先前逃过苏晓的 **[血门冒险团](../../势力/index.md)**（[铁大师](../../人物/铁大师.md) / [老黑](../../人物/老黑.md) 等）正死在此堵桥队手中（铁大师跪地求饶仍被爆头扔崖）。
- **苏晓反堵桥**：苏晓远远用蜘蛛女皇先狙杀狙击手 + 弓箭手 + 法师，再冲上桥近战——以「**环断**」（圆形刀芒·一天限三次·超用手臂数天不能持刀）一招放倒一片、青钢影专破冰法师魔力回路、徒手抓住金色长矛控制技反甩回敌方、**御力**硬接战锤（脚下石板裂而身不伤）、双枪法力填弹连射；堵桥队大部被歼，仅两名穿「滑翔衣」的疑似兄弟跳崖逃脱。他留下重伤的**阴郁头目 + 弓箭手 + 奶妈**（逼奶妈治疗保命），**审问堵桥的秘密**——答案是「**[轮回印记](../../总览/轮回乐园机制.md)**」：生存试炼中死者的轮回印记可剥离（约 50% 成功率）、一枚 = 1% 世界之源或 1000 币（详见 [机制](../../总览/轮回乐园机制.md)）。苏晓从尸体上剥得 7 枚、掐死色诱无效的奶妈，顺走那辆**装甲车**（介于召唤物与物品之间·无所有权但试炼内可用·半箱油 + 备用油桶·夜晚最佳庇护所）当代步工具。

## 鼠大人的空投箱（八卷58章）
裁判 [鼠大人](../../人物/鼠大人.md) **每天空投一个补给箱**（蓝 / 紫等品质·带降落伞 + 信号弹·里头有恢复品 / 食物 / 淡水、运气好还有灵魂结晶 / 技能卷轴 / 强化药剂）——一落地就引来大批契约者抢夺、自相残杀（昨日蓝箱「死了多少人」）。这是又一处「大逃杀」推手，也是鼠大人的恶趣味：他还会**把想逃离吞噬岛的契约者绑上铁块 + 信号弹当「假空投」扔下**、引契约者疯抢。苏晓**用装甲车顶接住一个紫色空投箱、再以界断线缠走**（名场面「让人窒息的操作」），独吞紫箱：**美味的烤火鸡（蓝）+ 灵魂结晶（中）×5 + 行军干粮 ×10 + 淡水 ×5 + [肾上腺素](../../力量体系/消耗品材料/index.md)（紫）+ [战术皮靴](../../力量体系/装备道具/index.md)（紫）**。鼠大人见状起了戏心「明天空投加重、解除降落伞、看你怎么接」。

## 血藤战士 / 亚森曼部落的秘密（八卷24–26章·待解）
- **亚森曼部落的暴兵之道**：为对抗虫族的暴兵海，红皮土著靠**惊人生育力**硬扛——女土著每胎产 2~5 个、16 岁生到 30~35 岁；一名战士之妻甚至产下**十五胞胎**；大本营 **40 万 + 人口**（幼儿居多·成年女几万·男丁稀少〔多去前线·断手脚才留部落〕·兄死则嫂由其他兄弟「继续用」）。
- **「血藤战士」**：部落另有一种**产出更强战士**的秘法——产物**失去大部分知觉、嗜血、冷酷、呆板、有太阳时战斗力极强**（苏晓夜里斩杀的那批「精英土著」即此）。**副作用 = 会失控自爆**：体内仿佛点起一颗「小太阳」、把抱住他的配偶瞬间焚成焦炭、自身炸成玻璃化大坑（八卷25章亲见）；长老 **[巴尔](../../人物/巴尔.md)** 疑也是血藤战士（气息时而一股时而两股·眼中藏着绝境的疯狂）。
- **后山的秘密**：部落依一座大山而建、后山正面有几十名呆板土著严守——苏晓直觉**血藤战士 / 图腾藤蔓 / 灵魂结晶 / 黑色种子 / 巴尔兄弟仇怨**的秘密都藏在后山，**一旦揭开恐得海量世界之源**（待后续）。

## 进度
- **八卷10章·试炼前准备**：苏晓在乐园拍卖会拍下双枪 [黑檀木·白象牙](../../力量体系/装备道具/index.md)（补中距离短板）；剩 1 万币后靠「制 [特级炼金秘药](../../力量体系/消耗品材料/index.md) 每瓶 6000 卖」滚雪球，三天攒下 5 瓶特级炼金秘药 + 一堆特级炼金炸弹。停留满 3 天 → **生存试炼开启**（501 人）。
- **八卷11–12章·噩耗 / 登机**：落在集合孤岛、501 人汇聚「和平区」。**裁判 [鼠大人](../../人物/鼠大人.md)** 驾超大型飞机来接、当众秒杀一名「目光不爽」的契约者立威，勒令全员登机、禁离岛。机上**浩海冒险团副团长**散资料拉结盟（500 人中 300+ 结盟分小团体）；**[彼岸花冒险团](../../势力/index.md) 6 女**（团长 [冷月](../../人物/冷月.md)、副团长 [血玫瑰](../../人物/血玫瑰.md)、智谋腹黑女、狙击手蜂、奶妈糖果、新主坦罗娜）在机上认出苏晓、**密谋上岛后为「三十二姐妹血仇」复仇**。
