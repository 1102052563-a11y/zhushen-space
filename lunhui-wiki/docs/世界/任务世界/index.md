# 任务世界

契约者被派往执行任务的各个独立世界（衍生位面），是无限流的核心。

> **按剧情顺序排列**（同一世界被多次进入时各列一条、不合并）。

- ① [海贼王 · 首入（哥亚王都）](海贼王.md) — 苏晓**首个**任务世界，难度 LV.6 · 噩梦（✅通关 A-）
- ② [东京喰种](东京喰种.md) — 苏晓第二个任务世界，难度 LV.3 · 困难（✅通关，评价 S-）
- ③ [进击的巨人](进击的巨人.md) — 苏晓第三个任务世界，难度 LV.7 · 困难（✅通关，评价 A+；苏晓屠尽巨人传承者致世界崩坏）
- ④ [斩·赤红之瞳](斩赤红之瞳.md) — 苏晓的**第四个任务世界**（第四卷·异族战场），难度 **LV.8 · 极困难**（✅通关，评价 A；核心力量体系=48 件帝具·苏晓放出牛头领主毁灭帝都）
- ⑤ [寄生兽 · 废土游侠](寄生兽.md) — 苏晓的**第五个衍生世界**（第五卷·**空间战场**），难度 **LV.10 · 极难**（✅通关，评价 S-）。取材《寄生兽》+ 原创「卡拉星人入侵 / 核战废土」末世；苏晓任**中立阶职·裁决者**，设满 3 处世界坐标（第三处藏进深海鲸腹）夺取此世界、并斩杀敌方裁决者图卡与卡拉星人王迪卡。通关后世界被重置为 Lv.3 新人温床、卡拉星人泯灭。
- ⑥ [海贼王 · 第六卷再入（七水之都）](海贼王-第六卷再入.md) — 苏晓的**第六个衍生世界**（第六卷·**二度进入海贼王**·开放性世界 LV.3~??·七水之都 / 司法岛），（✅通关，评价 **S**·世界之源 41.3%）。与首入分开记录。
- ⑦ [钢之炼金术师（晋升之路）](钢之炼金术师.md) — 苏晓的**第七个衍生世界**（第七卷·**一阶考核世界**·难度 LV.10·困难·**综合评价 B·已通关**）。取材《钢之炼金术师 FA》；无主线 / 猎杀任务、**只有「晋升任务·窃取神之力（伪）」**（三次失败即降职工者）；苏晓化名国家炼金术师「白夜·爆炎炼金术师」、谋削瓶中小人麾下的人造人，最终以魔晶瓶夺取神之力（伪）通关。
- ⑧ [二阶生存试炼（T12K星球·吞噬岛）](T12K星球-生存试炼.md) — 第八卷·**晋升二阶的生存试炼**（**非衍生世界**·独立场景·难度 LV.14~20 噩梦·**✅已通关·苏晓晋升二阶**）。同批 501 名契约者落一座环海孤岛废墟、**生存十天**（仅 36 人存活）；苏晓从红土区杀到黑色森林、斩杀首领级生物暗王，晋升二阶。
- ⑨ [火影忍者（青钢之崛起）](火影忍者.md) — 苏晓**晋升二阶后第一个衍生世界**（第九卷·**开放性世界 LV.3~??**），🚧进行中。苏晓用「斩首大刀」世界钥匙把乐园安排的大剑世界**改成火影世界**，野心是**吸尾兽查克拉暴涨青钢影**；落点为雾隐之村地牢、身份「刺杀四代水影失败的刺客」。主线「抉择」（5 天内加入一方势力）+ 猎杀「清理 10001 号违规者」。
