# 任务世界

契约者被派往执行任务的各个独立世界（衍生位面），是无限流的核心。

> **按剧情顺序排列**（同一世界被多次进入时各列一条、不合并）。

- ① [海贼王 · 首入（哥亚王都）](海贼王.md) — 苏晓**首个**任务世界，难度 LV.6 · 噩梦（✅通关 A-）
- ② [东京喰种](东京喰种.md) — 苏晓第二个任务世界，难度 LV.3 · 困难（✅通关，评价 S-）
- ③ [进击的巨人](进击的巨人.md) — 苏晓第三个任务世界，难度 LV.7 · 困难（✅通关，评价 A+；苏晓屠尽巨人传承者致世界崩坏）
- ④ [斩·赤红之瞳](斩赤红之瞳.md) — 苏晓的**第四个任务世界**（第四卷·异族战场），难度 **LV.8 · 极困难**（✅通关，评价 A；核心力量体系=48 件帝具·苏晓放出牛头领主毁灭帝都）
- ⑤ [寄生兽 · 废土游侠](寄生兽.md) — 苏晓的**第五个衍生世界**（第五卷·**空间战场**），难度 **LV.10 · 极难**（✅通关，评价 S-）。取材《寄生兽》+ 原创「卡拉星人入侵 / 核战废土」末世；苏晓任**中立阶职·裁决者**，设满 3 处世界坐标（第三处藏进深海鲸腹）夺取此世界、并斩杀敌方裁决者图卡与卡拉星人王迪卡。通关后世界被重置为 Lv.3 新人温床、卡拉星人泯灭。
- ⑥ [海贼王 · 第六卷再入（七水之都）](海贼王-第六卷再入.md) — 苏晓的**第六个衍生世界**（第六卷·**二度进入海贼王**·开放性世界 LV.3~??·七水之都 / 司法岛），（✅通关，评价 **S**·世界之源 41.3%）。与首入分开记录。
- ⑦ [钢之炼金术师（晋升之路）](钢之炼金术师.md) — 苏晓的**第七个衍生世界**（第七卷·**一阶考核世界**·难度 LV.10·困难·**综合评价 B·已通关**）。取材《钢之炼金术师 FA》；无主线 / 猎杀任务、**只有「晋升任务·窃取神之力（伪）」**（三次失败即降职工者）；苏晓化名国家炼金术师「白夜·爆炎炼金术师」、谋削瓶中小人麾下的人造人，最终以魔晶瓶夺取神之力（伪）通关。
- ⑧ [二阶生存试炼（T12K星球·吞噬岛）](T12K星球-生存试炼.md) — 第八卷·**晋升二阶的生存试炼**（**非衍生世界**·独立场景·难度 LV.14~20 噩梦·**✅已通关·苏晓晋升二阶**）。同批 501 名契约者落一座环海孤岛废墟、**生存十天**（仅 36 人存活）；苏晓从红土区杀到黑色森林、斩杀首领级生物暗王，晋升二阶。
- ⑨ [火影忍者（青钢之崛起）](火影忍者.md) — 苏晓**晋升二阶后第一个衍生世界**（第九卷·**开放性世界 LV.3~??**），**✅已通关·综合评价 S−·世界之源 39.9%**。苏晓用「斩首大刀」世界钥匙把乐园安排的大剑世界**改成火影世界**，野心是**吸尾兽查克拉暴涨青钢影**（吸一尾守鹤 + 四尾 → 青钢影 Lv.10 冲 Lv.26）；落点雾隐之村地牢、身份「刺杀四代水影失败的刺客」。**主线**「抉择」（加入晓）→「凋零之木」（协助长门毁灭木叶·已达成）+ **猎杀**「清理 10001 号违规者」（已完成）；通关升 Lv.10→Lv.14。
- ⑩ [全职猎人（绅士同盟）](全职猎人.md) — 苏晓的**第 10 个衍生世界**（第十卷·**半开放世界 LV.5~??**），**✅ 已通关（综合评价 S−·世界之源 54.6%·Lv.14→16）**。以《全职猎人》为蓝本（念能力·六块大陆·猎人执照）；因异次元乱流传送偏差、落点从友克鑫市错到 1353 公里外的**齐鲁市**（赌城）。**主线**「抵达友克鑫市 + 参与拍卖会」+ **猎杀**「清理绅士同盟」。全程：清绅士同盟（斩枯尸 / 犀 / 艳妖 / **红色小丑**）→ 拍卖会卷入 **幻影旅团**（一人连斩飞坦 / 信长 / 窝金 / 侠客 / 团长库洛洛、打到旅团仅剩 2 人）+ 联手红色小丑的尼拉潘攻城战。
- ⑪ [Fate/Zero（圣杯战争）](fate-zero.md) — 苏晓的**第 11 个衍生世界**（第十一卷「圣杯战争·天赋觉醒之路」），**✅ 已通关·综合评价 S+·世界之源 88.9%·Lv.16→20**。从「魔法禁书目录 / Fate/Zero / 妖精的尾巴」三选一中选定（特意挑不大的世界来**激活天赋觉醒任务**）。被乐园安排**特殊身份 = 伪英灵**（替代 Berserker 职阶·名义 Master = [间桐雁夜](../../人物/间桐雁夜.md)）。全程：斩四英灵（兰斯洛特 / 迪卢木多 / 百貌 / 英雄王）+ 斩领主修斯 + 完成主线天之杯 + 天赋觉醒任务（[噬灵者](../../力量体系/天赋.md) 觉醒 S 级）+ 隐藏任务「英灵的荣耀」（清光 6 御主令咒）。
- ⑫ [维斯星球·多亚古大陆（世界争夺战）](维斯星球-多亚古大陆.md) — 苏晓的**第 12 个衍生世界**（第十二卷「世界争夺战·汹涌的人潮」·**战争世界 Lv.10~29**），🚧进行中。**轮回乐园 vs [天启乐园](../../人物/希女王.md) 争夺一整个世界的「世界争夺战」**（[虚空之树](../../总览/轮回乐园机制.md) 公证·乐园功勋系统·本世界无法获世界之源）。苏晓 = 龙腾帝国铁血军团**三等兵**、战区黑土平原；战争任务「**世界之核**」（先夺回黑土要塞）；十余冒险团 + 散人 + 天启乐园千余「教团」契约者同场。
