# 永光世界·厄难囚笼

⚠ **世界简介只有一句话**:「**欢迎来到，厄难囚笼。**」

⚠ 这是 [苏晓](../../人物/苏晓.md) **一路防备、却终究要来**的地方——⚠ **他甚至亲手往里送过两位「客人」**（[食暗者](../../人物/风海大陆/黑眼存在.md) 与 [贝芙丽](../../人物/幽暗大陆/贝芙丽.md)）。⚠ 世界难度:**Lv.???**；⚠ 世界之源 **0%**；⚠ 初始世界声望 **0 点**；⚠ **因灭法之影身份，苏晓在本世界处于初始无阵营状态。**

## 🌞 一个世界，怎么被元素之力玩坏的

⚠ **永光世界为何如此可怕？** 不仅是因为此地的灭世级存在与深渊滋生物们，⚠ **还有这世界的构成:这个世界的基础力量中，只有元素之力，没有哪怕一丁点的深渊能量。**

- ⚠ **元素稀薄 → 深渊侵袭**：当深渊通道开启、海量深渊能量涌入后，⚠ **那个世界会被绝对的黑暗所笼罩——那是光与火都无法驱散的黑暗，那是深渊。**
- ⚠ **元素过强 → 植物王国**：⚠ **各类植物以充足的元素力量全方面超越其他生灵**，当植物们崛起后**简直噩梦，一切都将是它们的养分**。
- ⚠ **元素充沛到极点、半点深渊都没有 → 永昼**：⚠ **足以将所有生灵灼烤成灰烬的极昼之光会一直映照而下，阴凉的夜晚成为奢望，永恒的白昼炙烤大地，让这个世界的各个区域产生不同特性的畸变。** ⚠ **没错，永光世界只有白昼**——⚠ **而所谓的庇护所，就是能提供阴凉与休息的地方。**
- ⚠ **昼光的杀法**:苏晓把一块骸骨丢到庇护所外——⚠ **刚落在沙水上这骸骨就下沉，而且在下沉途中因被昼光照到，这骸骨开始半透明，之后化为透明的水液融入沙海**。⚠ **这就是此地昼光的特性:蒸发生物的血肉，水化骨骼。**
- ⚠ **元素浓度**:「如果将元素力量比喻成氧气，那么**外界的氧含量就是 25% 左右**，⚠ **至于永光世界，这里的氧含量是 5000%**」——⚠ **这不是夸张的比喻，说这里的元素力量含量是 5000%，那都是有些含蓄了。**
- ⚠ **施法者为什么不来这里吸元素**:⚠ **这就好像呼吸——吸入有 21% 氧含量的空气会让人延续生命，可如果吸入 5000% 的超浓缩纯氧，只会中毒**。⚠ **施法者们吞噬的是与深渊能量共存、接触后的自然元素，而非永光世界这种没有深渊能量滋润的可怕元素力量。**
- ⚠ **本土生物有多离谱**:⚠ **这些生物的特性非常极端——它们可能有种绝强者、乃至至强者的强大战力，与之相对，一旦它们离开永光世界，会在短时间内死亡。**
- ⚠ **空间也变了**:⚠ **永光世界内的空间结构，与苏晓去过的任何一个世界都不同**——⚠ **他掌握的基础空间学在此完全不管用**（修传送阵当场炸了）。⚠ **可有趣的是，他自己搞出的「灭法传送阵」竟然可以用**:⚠ **这倒不是它有多高级，而是「灭法传送阵」类似于空间轨道炮，是从起始点轰到目标点**——⚠ **至于什么空间规则、空间构成，这和它不搭边:原理为「只要能轰穿，那就能传送」，就是这么的简单粗暴。**

## 📜 世界编年史（世界简介全文）

- ⚠ **超脱纪元**：**即使是超脱，也终将消亡**——⚠ **昔日的荣光已然不在，被新起之秀「[风海大陆](风海大陆-初始印记.md)」所超越。**
- ⚠ **超脱纪元**：**世界崩灭的速度，比想象中的要慢**——⚠ **这巨大世界，崩灭到只剩不足十分之一，但它的广袤，依旧不是寻常原生世界所能比拟。**
- ⚠ **超脱纪元**：⚠ **灭法者与深渊监守者们，干涉了这崩灭中的世界，将一颗富有活力的世界之核融入到此世界**——⚠ **崩灭停止，可元素之力与深渊之力的流逝已不可逆，其中深渊之力的流逝最为严重**；⚠ **这必定导致，过于壮大的元素力量会将本就不多的深渊之力蒸发殆尽。**
- ⚠ **超脱纪元**：**灭法者与深渊监守者们给这世界带来新的生机与责任**——⚠ **虽说它已无法恢复曾经的超脱荣光，但此地将成为厄难的牢笼。**
- ⚠ **血誓**：**灭法者 / 深渊监守者 / 永光监视者。**
- ⚠ **永光纪元**：**此地，生灵止步，为厄难之牢笼。**
- ⚠ **极昼纪元·01 年**：**灭法的时代已然终结，深渊监守者的传承近乎断绝**，⚠ **永光监视者们也只能在灰暗圣所（最后的圣所）内苟延残喘**。⚠ **极昼之光的炙烤下，近乎九成生灵绝迹**，⚠ **其余生灵即使幸存，也畸变成可怕的生物**。
- ⚠ **185 年**：**无光神殿日渐崛起，庇护城的雏形逐渐展现。**
- ⚠ **279 年**：**庇护城彻底建成**——在 [永暗之主](../../人物/永光世界/永暗之主.md) 的庇护下成为**永光世界唯一的生灵大型聚集地**，⚠ **无光神殿在永暗之主的授意下成为庇护城的管理者。**
- ⚠ **350 年**：**大部分封印被破除**，那些被灭法者、深渊监守者囚困的可怕存在们陆续破封而出。
- ⚠ **352 年**：**庇护城被毁灭八成以上，直至永暗之主降临。**
- ⚠ **405 年**：**庇护城与几位最可怕的存在签订誓约，彼此互不招惹**——至此庇护城终于迎来稳定。
- ⚠ **419 年**：⚠ **一名被放逐至此的魂族，成功破解了永光监视者们遗留的秘法，制作出了「庇护石」**——⚠ **庇护城内的人们不再只求安全，而是逐渐走出庇护城的遮蔽探索**；⚠ **然而，探索并未带来收获，而是让庇护城的居民们真正认识到永光世界的绝望与可怕。**
- ⚠ **850 年**：**庇护城已少有人敢外出探索**，⚠ **并且一种可怕的诅咒不知何时出现，让庇护城的居民罕有能活过 60 岁之人**——⚠ **即使所有死者都是自然衰老而死、无病症表现**，这也让居民们很是不安，**作为管理者的无光神殿开始严密调查此事。**
- ⚠ **925 年**：**已少有居民可活过 50 岁。**
- ⚠ **1029 年**：⚠ **已罕有人能存活过 40 岁**——⚠ **这片永光世界唯一的安稳之地虽依旧有几分繁荣，但在这繁荣之下，似乎隐藏着什么秘密。**

## 🗺 四大区域

⚠ **本世界四大区域:湿盐区、荒芜沙漠、蚀雨地、庇护城**——⚠ **三个都有格外危险的环境**。⚠ **昼光是本世界最基础的危险环境之一。**

- **荒芜沙漠**（苏晓的落点）：⚠ **外围区域只需有庇护石**；⚠ **如果是内区或中心区，庇护石也不能保证一定安全，还需要有足够多的生命力硬抗**——⚠ **也就是说，生命值上限越高，越有深入荒芜沙漠的资格**。⚠ [索恩斯](../../人物/永光世界/索恩斯.md) 的警告:「**千万别去**，边缘区还好，⚠ **等你深入到一定程度，你会被拖入到炼狱噩梦**。」
- ⚠ **它其实是「沙之海」**:苏晓单手捧起一把黄沙，⚠ **除了热感外，他竟感到这黄沙的湿润感**——⚠ **当黄沙从指缝间淌没后，他手上竟沾上些水液**。⚠ **这很不妙**:如果是沙漠，用了庇护石徒步前行即可；⚠ **可如果是沙之海，那就需要以船渡海了**——⚠ **而且在这平静的沙漠之下，天知道会有什么可怕的生物**。⚠ **表面只有一层薄沙，到了十几公分以下深度水沙就稀释了很多，超过一米深后完全稀薄到和水液一样**；⚠ **随着温度持续升高，沙之海的真正样貌展现:除了表面的一层薄沙外，下面完全化为水液。**
- **湿盐区**：⚠ **在这里水很宝贵，比灵魂石更受欢迎**——⚠ **但别被这诱惑到，能别去尽量别去**；⚠ **非去不可的话，多搞些纯盐，那是好东西。**
- **蚀雨地**：⚠ **最后一座「监视者高塔」就在这里的深处**（老树族:「那些高塔被毁了不少，**唯一有可能剩下的只可能是蚀雨地的深处了**」）；⚠ 暗之女给的**灭法宝库地图**也指向这片区域。
- **庇护城**：⚠ **永光世界唯一的安稳之地**——详见下文。

## 🏙 庇护城 · 一座由灭世级存在「庇护」的城

⚠ 一棵**黑石巨树**耸立在天地间——⚠ **高大到离谱、顶部都耸立到云层间**；⚠ **它的所有枝干都垂下，就像一个巨伞般的形状，所有黑石枝条垂到地面、乃至深深没入泥土中**。⚠ **它所笼罩的范围最起码有一座中型城市大小、乃至还要大一些**；⚠ 那些垂下的枝杈很严密，**形成了树墙般的屏障，只剩一处几十米高的入口**。

- ⚠ **三层结构**:**下城**（平民居住区·**占整个庇护城面积的七成以上**）· **中城**（**无光神殿把控，经批准的平民可进入**）· **顶城**（**代表平民对永暗之主的朝拜之处**——⚠ **虽说永暗之主并不在那，却也无人敢踏足其中**）。
- ⚠ **入口守卫并未上前盘查**，那态度分明是**只要是智慧生命就可以随意出入**——⚠ **想来也是，这庇护城根本不是什么安全区域**。⚠ **一走进庇护城，清凉感迎面而来，没有昼光的照耀，光线也暗淡下来**——⚠ **一种既安全又危险的感觉出现**。
- ⚠ **苏晓一眼看穿的真相**:⚠ **庇护城的顶部有灭世级存在的气息**——⚠ **除灭法者或深渊监视者外，其他人基本没可能察觉到不对**（那气息会被误认成神灵的气息）。⚠ **这解释了他之前的疑惑:永光世界的这些可怕怪物，怎么会忍住不来此地猎食生灵？** ⚠ **因为庇护这里的，就是本世界最强的灭世级存在之一·永暗之主。**
- ⚠ **土著的日常**:⚠ **这里的土著民根本不知道是什么在庇护这里，误认为永暗之主是神灵存在**——⚠ **街道上的行人神情自然，街边的小贩烹煮着小吃，更暗的小巷内，几名少年扒手正满脸得意地清点刚入手的财物。** ⚠ **越往下城区深处走，光线越发暗淡，最后只剩上方映下的微光**:⚠ **周边的建筑变得拥挤，有些平民干脆只能生活在屋棚中，身上盖着勉强遮体的破布，蜷缩在黑暗中，骨瘦如柴。**
- ⚠ **两个致命陷阱**:①⚠ **庇护城的庇护石都被无光神殿动了手脚**（[罪亚斯](../../人物/画中世界/罪亚斯.md) 进屋前就在小队频道留言「**不要用庇护城的庇护石**」，索恩斯也独立警告过）；②⚠ **庇护城有「空间震膜」**——⚠ **[尤莎](../../人物/永光世界/尤莎.md) 的传送阵是唯一不会被感应到的传送方式**。⚠ 「想进入这里简单，**可想要离开，那就困难重重**。」

⚠ **除了庇护城所在的中心地外，永光世界可以分为几大区域**——⚠ **之前只知四片，第五片最广袤、也最有欺骗性。**

### 🌿 狂茂之地

⚠ **不仅没有昼光映下，还是一片植被繁茂的景象**——⚠ **甚至可以随处找甘甜的果子吃，喝干净的河水。**

- ⚠ **昼光去哪了**:⚠ **这里的植物都能吸收极端自然元素**——⚠ **并且在吸收后会进行与光合反应相近的元素反应，从而大量放出植物生命力**;⚠ **这些植物生命力滋润了大地与水源**——⚠ **当水汽漂到上空构成云时，这些富含植物生命力的云层在高空生成了一种透明薄膜**;⚠ **昼光映过这薄膜后，化为很接近阳光的自然光。**
- ⚠ **万界中，植被中虽很少出现强者**——⚠ **可有时它们才是隐藏起来的强大者**;⚠ **就比如狂茂之地的植被，竟凭借自身特性契合环境。**
- ⚠ **黑铁城的居民为何不来这**:⚠ **答案是，所有族群类的灭世级存在全部在狂茂之地**——⚠ **这些族群互有地盘**;⚠ **在外界是灾祸的 [银皇后](../../人物/银皇后.md)，到了这里后也只是占了处还算大的地盘而已**——⚠ **寄星蟹、腐虫、眷族都位于这片区域内。**
- ⚠ **一条铁律**:⚠ **在狂茂之地，无论多凶狠的族群也不会大量吞噬植物**——⚠ **那会成为本区域内所有灭世级族群的敌人**;⚠ **其中 [蛀世](../../人物/永光世界/蛀世.md) 例外**——⚠ **这家伙已经进化到连极端自然元素都能吞噬。**
- ⚠ **在狂茂之地，每一块地盘都是无价之宝**——⚠ **这也是苏晓不格杀银皇后的原因之一**:⚠ **一旦她死，她控制的所有下位个体会失控并在几天内陆续死亡，虫族建筑更会以更快的速度凋零**;⚠ **附近的其他灭世级族群势必注意到，必定来争夺。**

### 🌑 湿盐区的黑夜

⚠ **与其他地方永远是白昼不同，湿盐区有黑夜**——⚠ **可这黑夜与寻常意义的夜晚不同，是浓郁的暗能量汇聚在了上空**;⚠ **当黑夜降临于湿盐区，这里才会显露其真正样貌。**

- ⚠ **一道万米级体型的飞行生物，夹带着能量袭来的恐怖呼啸声在上空掠过**——⚠ **沿途散落大片黑色液体**;⚠ **这些液体刚落地就开始升腾**——⚠ **仔细观察会头皮发麻地发现，那根本不是黑色液体，而是一条条黑色水蛭般的蠕虫，正肆意扭动着搜寻周边的活物。**
- ⚠ **甜美的歌声传来，隐约可见圣洁的白色微光出现在远处，让人心生安全感、想尽快跑过去**——⚠ **可如果真的靠近会发现，那是一团寄生了黑夜女妖的畸变物**;⚠ **正通过生长在它身上的女妖头颅，发出能蛊惑神灵的歌声。**
- ⚠ **一只由大量黑色触须构成的人形存在，胸膛处的独眼盯着苏晓**——⚠ **感觉苏晓不好惹后，轰的一声掠过空间震爆，消失在远处。**
- ⚠ **夜间的湿盐区，是黑暗地狱，是生灵止步之地**——⚠ **（⚠ 布布汪两条狗爪抱着苏晓腿，咽了下口水，它当时害怕极了）**
- ⚠ **票价也涨**:⚠ **因出发点湿盐区处于夜间，去 6 号车站的车费要 120 枚黑铁币**——⚠ **比白天贵了一半。**
- ⚠ **獗渴症**:⚠ **并非是病症**——⚠ **而是体内的水分被暗盐缓慢吸收，最终在渴死前，理智值会滑落到负数，从而引起的狂乱或疯狂**;⚠ **隧道内壁的泥土漆黑，但表面渗出一层细密的白色颗粒**——⚠ **用食指触碰会发现这是一种盐**;⚠ **皮肤触碰到后水分快速被吸收，如果不尽快清洗，整根手指都会快速干枯，最后变成枯树枝般。**
- ⚠ **湿盐区深处有个小镇**，⚠ **还有不少居民**——⚠ **但一般绝强级的灭世级生物，都不敢冒然进入这小镇**;⚠ **可以想象，那座满是黑色钢铁尖顶建筑的小镇到底有多可怕**——⚠ **⚠ 最坏的消息是，君主宝藏的入口，就位于那可怕的小镇内。**
- ⚠ **狂笑镇**:⚠ **白天时一切正常，可在暗夜到来后，冒险者们发现这小镇内传出各种狂笑声**——⚠ **去里面探查的冒险者，没一个能出来。**

## 🌗 让永光世界恢复正常（51卷57章）

⚠ **放眼虚空万界，只有正封印着猩红权杖的苏晓能做到**:

1. ⚠ **抽空猩红之力**:⚠ **用猩红权杖（原罪之书第五页）把本世界所有猩红之力吸空**——⚠ **⚠ 暗血城主两父子当年想开深渊通道让本世界恢复平衡，方向没错，但本世界涌入深渊的区域都被猩红之力占据（这是猩红君主冲破封印的手段），一开通道涌出的是猩红之力而非深渊能量**;⚠ **苏晓先抽空猩红，就不用担心这点。**
2. ⚠ **斩出深渊通道**:⚠ **连用三次「极刃·世界」斩破空间→破坏「自然元素隔层」→纯粹深渊力量顺着新生的深渊通道狂涌入世界**（⚠ 半个世界深渊、半个世界极端元素）——⚠ **二者交界处融合:过极的元素被深渊冷却，纯黑暗的深渊被元素稀释出清凉。**
3. ⚠ **缝合平衡**:⚠ **放灵影线缝合深渊通道＋海量世界之力封平**——⚠ **深渊能量被推移至「自然元素隔层」之外，慢慢渗透，让本世界极端自然元素逐渐恢复正常。**

- ⚠ **灭法之影的真意**:⚠ **元素守卫者的意义，并非完全遏制深渊，而是让元素与深渊平衡**——⚠ **过浓的自然元素或过度的深渊，都会破坏平衡**;⚠ **⚠ 先代灭法们非不想让永光恢复，而是时机未到**——⚠ **永光原是广袤的超脱·原生世界，因灾祸降临而崩溃，先代灭法造的极端元素环境让崩溃停止、并滋养世界重新凝聚焕发生机，正是多年铺垫，永光才能在今天重新承载元素与深渊平衡。**
