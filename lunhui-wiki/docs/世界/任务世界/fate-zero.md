# Fate/Zero（圣杯战争）

[苏晓](../../人物/苏晓.md) 的**第 11 个衍生世界**（第十一卷「圣杯战争·天赋觉醒之路」），以《Fate/Zero》为蓝本——**第四次圣杯战争**。从「魔法禁书目录 / Fate/Zero / 妖精的尾巴」三选一中选定（消耗时空之力 0.021 盎司·**最低综合评价须达 C−**）。**苏晓特意选这个不太大的世界来激活 [天赋觉醒任务](../../力量体系/天赋.md)**。

## 苏晓的特殊身份 = 「伪英灵」（十一卷1–2章）

因苏晓实力与本世界契约者相差过大、且这不是开放性世界，乐园给他安排了**特殊身份 = 伪英灵**：

- **替代 Berserker（狂战士）职阶**：开局他被丢进一处结界，须**击败真正的 Berserker [兰斯洛特](../../人物/兰斯洛特.md)**（间桐雁夜召来的圆桌骑士）才能正式入世——他将兰斯洛特斩成四段（其能量被圣杯吸收），由此顶替其职阶（杀兰斯洛特无任何奖励、只是入场考验，杀不掉则永困结界）。
- **伪英灵 ≠ 英灵**：他保有自己的肉体、不需魔力供给即可存在、灵魂与圣杯毫无关系（生死不影响圣杯）；但**一旦其 Master [间桐雁夜](../../人物/间桐雁夜.md) 死亡，他在本世界的停留时间将骤减至最多 2 天**（强制回归）。**只有当本世界没有与他实力相近的契约者时，他才会被安排成伪英灵**（否则正常进入）。
- 其余六骑英灵照常出现；间桐雁夜手上的三枚**令咒**本是用来控制已死的兰斯洛特、对苏晓这个伪英灵无效。

## 世界观 · 圣杯战争

- **圣杯**：200 年前三大魔术家族为抵达「**根源**」（魔术师的终极梦想）联手召唤的「万能机」（能实现任何愿望）；但圣杯只一个、三家矛盾激化，遂以「小规模厮杀决定归属」的形式开战。**每 60 年一次、规则逐次完善，如今是第四次圣杯战争。**
- **规则**：大圣杯从地脉吸魔力 → 选 7 名魔术师为 **Master**（授令咒）→ 各召唤 1 名 **Servant（英灵 / 从者）** → 七组厮杀 → **七英灵最终都须死**（圣杯需其灵魂盛满）→ 胜者打开通往根源之路。
- **与轮回乐园相比**，大圣杯的能力「不值一提」。

## 世界信息（十一卷5章·正式进入提示）
- **世界难度 Lv.20**·世界之源 5.2% 起·**主线任务「天之杯」**（Lv.18·以任何方式触碰大圣杯·15 自然日·奖 5000 币 + 6 属性点·罚强制处决）·**猎杀任务：无**（本世界无违规者）。地点 = **冬木市**。
- **魔术师**：日常以「研究」为主；把魔术当工具用（如暗杀）者被蔑称「魔术使」。须守「**神秘的隐匿**」——当众用魔术会被魔术协会肃清。魔术强弱看体内**魔术回路**数量（生来固定、不可外增，故魔术师攒几代血统让后代回路更多）。
- **英灵 / 从者**：死后留为传说、成信仰对象的「英雄」才可能成英灵；人召出的英灵称「从者」。英灵由能量构成、能**灵体化**（唯独苏晓这个伪英灵不能）。
- **两大势力**：**[魔术协会](../../势力/index.md)**（管魔术师）+ **[圣堂教会](../../势力/index.md)**（管圣杯战争·有「疯狗」级战力）——冬木市最强两股势力。

## 苏晓的英灵属性（十一卷5章）
英灵名「白夜」·职阶 **Berserker**·属性中立·混沌·邪恶·出典轮回乐园；筋力 A / 耐久 A+ / 敏捷 A+ / 魔力 B / 幸运 E−~C+ / 宝具 A。职阶技能「**狂化 C**」**已被刀术大师 Lv.30「心灵之刃」免疫**（故他是个不发狂的狂战士）。固有技能：**灭法 A++**、心眼 C、刀术大师 A+、枪械 B+、绝魔体质 B+、灵影体质 B++、**[吞噬之核 EX](../../力量体系/技能.md)**——他由此动了「**吸食英灵能量**」的念头（但与圣杯抢能量、反噬未知）。

## 🩸 天赋觉醒任务（十一卷5章激活）
返园集齐 [天赋之书（完整）](../../力量体系/消耗品材料.md) 后，进本世界即激活（提示为**血红色**·严禁外泄、违者强制处决）：**共三环、难度持续攀升、最终环上限 Lv.26**；给三次机会（连续三个衍生世界各失败一次则天赋永久无法觉醒）。**第一环「抵达柳洞寺」**（Lv.3·4 自然日·罚扣一次觉醒机会）——柳洞寺是冬木市地下大型灵脉之上的灵地、被圣堂教会用结界藏了起来。苏晓决定先专注天赋觉醒任务、主线天之杯靠「失败豁免权」垫底。
