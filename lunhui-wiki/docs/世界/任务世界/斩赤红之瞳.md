# 斩·赤红之瞳（任务世界）

苏晓的**第四个任务世界**，以漫画《斩·赤红之瞳》为蓝本（四卷13章）。开篇副标题「**异族战场**」。在轮回乐园中**很有名气**——异常危险、收益却极高，大量任务与高等级装备令契约者垂涎（四卷12章）。

## 世界信息
- **难度**：LV.8 · 极困难（大量契约者同入）。
- **开局**：消耗乐园 0.018 盎司**时空之力**，本世界契约者最低综合评价须达 **D+ 级**；未掌握语言耗 100 乐园币自动掌握（四卷13章）。
- **世界梗概**：一个存在千年的**帝国**日渐腐朽，高税与暴政逼得平民揭竿，民间反叛军四起；其中最强、最得民心者为**革命军**。帝国与革命军的战争、双方**暗部组织**的残酷交锋，是本世界主线（四卷12章）。
- **世界简介**（原文）：四十八件帝具、帝国变迁、狩人部队、夜袭部队、人体实验、已腐朽的千年帝国——「刀与剑的碰撞，帝具使的对决，善与恶的抉择，杀戮与和平的循环」。

## 帝具
本世界的核心力量体系，详见 [力量体系 · 帝具](../../力量体系/装备道具.md)。约一千年前帝国举国之力、以超级危险种素材 + 稀有金属**奥利哈钢** + 失传秘术，打造出 **48 件无法复制的帝具**（造毕即兔死狗烹处死工匠）。苏晓视之为「**削弱版的恶魔果实**」，部分帝具甚至更强（如艾斯德斯的冰帝具可**冻结空间**）。**主线目标「帝具·至高之力」即最强帝具**。

## 任务
- **主线任务：帝具·至高之力**（Lv.8）——获得最强帝具「**至高之力**」的**启动钥匙**（现持有者为帝国**小皇帝**）；30 个自然日；奖 **4 属性点 + 12000 乐园币**；罚 **全属性 -3**。
    - 钥匙**只有一把**、契约者却众多，竞争极激烈。苏晓判断：循正轨须打败小皇帝才「有机会」拿钥匙，而革命军绝不会放任他取走——遂决定**潜入帝国高层、接近小皇帝、直接夺钥匙**（加入帝国的契约者少、夺钥匙反而更易）。
- **猎杀任务**：清理 **113？？ 号违规者**（编号带「？？」的奇葩残缺·乐园称编号无误·苏晓疑有猫腻）；1 公里内自动开启追猎。

## 势力概览（详见 [势力](../../势力/index.md)）
- **帝国** — 千年腐朽帝国，幼主**小皇帝**在位、被奸臣把持；暗部统一交女将军 [艾斯德斯](../../人物/艾斯德斯.md) 管理。
- **革命军** — 最强反叛军，其暗部为 **「Night Raid」夜袭部队**（首领娜洁希坦），专司暗杀帝国头目、筹款、刺探情报。
- **极北异族军** — 帝国北疆因苛税起义的异族，首领**北方异族王子努马·塞卡**（长枪无败绩）曾以 3 万破帝国 20 万大军；帝国遂派 50 万大军 + 王牌艾斯德斯镇压（四卷14章）。

## 苏晓在本世界（进行中）
- **入世即落极北**冰原（零下 40°·寒风刺骨·热量流失会昏睡），与十几名契约者一同被传送（老火法师等「同类」·互不打扰）（四卷13章）。
- 凭指南针向南奔帝国方向，抵达**极北边塞**（黑石城墙·帝国北疆咽喉·百年未修）；自称「云游旅者」求加入帝国军，被疑为极北军探子围捕，拔斩龙闪迎战——**女将军 [艾斯德斯](../../人物/艾斯德斯.md) 现身亲自试招**（细剑 + 冰刃），苏晓挡住其一剑（四卷14章·未完）。
