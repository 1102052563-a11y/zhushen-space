# 斩·赤红之瞳（任务世界）

苏晓的**第四个任务世界**，以漫画《斩·赤红之瞳》为蓝本（四卷13章）。开篇副标题「**异族战场**」。在轮回乐园中**很有名气**——异常危险、收益却极高，大量任务与高等级装备令契约者垂涎（四卷12章）。

## 世界信息
- **难度**：LV.8 · 极困难（大量契约者同入）。
- **开局**：消耗乐园 0.018 盎司**时空之力**，本世界契约者最低综合评价须达 **D+ 级**；未掌握语言耗 100 乐园币自动掌握（四卷13章）。
- **世界梗概**：一个存在千年的**帝国**日渐腐朽，高税与暴政逼得平民揭竿，民间反叛军四起；其中最强、最得民心者为**革命军**。帝国与革命军的战争、双方**暗部组织**的残酷交锋，是本世界主线（四卷12章）。
- **世界简介**（原文）：四十八件帝具、帝国变迁、狩人部队、夜袭部队、人体实验、已腐朽的千年帝国——「刀与剑的碰撞，帝具使的对决，善与恶的抉择，杀戮与和平的循环」。

## 帝具
本世界的核心力量体系，详见 [力量体系 · 帝具](../../力量体系/装备道具.md)。约一千年前帝国举国之力、以超级危险种素材 + 稀有金属**奥利哈钢** + 失传秘术，打造出 **48 件无法复制的帝具**（造毕即兔死狗烹处死工匠）。苏晓视之为「**削弱版的恶魔果实**」，部分帝具甚至更强（如艾斯德斯的冰帝具可**冻结空间**）。**主线目标「帝具·至高之力」即最强帝具**。

## 任务
- **主线任务：帝具·至高之力**（Lv.8）——获得最强帝具「**至高之力**」的**启动钥匙**（现持有者为帝国**小皇帝**）；30 个自然日；奖 **4 属性点 + 12000 乐园币**；罚 **全属性 -3**。
    - 钥匙**只有一把**、契约者却众多，竞争极激烈。苏晓判断：循正轨须打败小皇帝才「有机会」拿钥匙，而革命军绝不会放任他取走——遂决定**潜入帝国高层、接近小皇帝、直接夺钥匙**（加入帝国的契约者少、夺钥匙反而更易）。
- **猎杀任务**：清理 **113？？ 号违规者**（编号带「？？」的奇葩残缺·乐园称编号无误·苏晓疑有猫腻）；1 公里内自动开启追猎。

## 势力概览（详见 [势力](../../势力/index.md)）
- **帝国** — 千年腐朽帝国，幼主**小皇帝**在位、被奸臣把持；暗部统一交女将军 [艾斯德斯](../../人物/艾斯德斯.md) 管理。
- **革命军** — 最强反叛军，其暗部为 **「Night Raid」夜袭部队**（首领娜洁希坦），专司暗杀帝国头目、筹款、刺探情报。
- **极北异族军** — 帝国北疆因苛税起义的异族，首领**北方异族王子努马·塞卡**（长枪无败绩）曾以 3 万破帝国 20 万大军；帝国遂派 50 万大军 + 王牌艾斯德斯镇压（四卷14章）。

## 苏晓在本世界（进行中）
- **入世即落极北**冰原（零下 40°·寒风刺骨·热量流失会昏睡），与十几名契约者一同被传送（老火法师等「同类」·互不打扰）（四卷13章）。
- 凭指南针向南奔帝国方向，抵达**极北边塞**，自称「云游旅者」求加入帝国军被疑探子，与 **[艾斯德斯](../../人物/艾斯德斯.md)** 交手；借「极北军 + 雪原狼来袭」化解、随其退守要塞（四卷14–15章）。
- **杀超级危险种芬里尔投名状**（四卷16–18章）：随艾斯德斯去除掉与极北军勾结的超级危险种 [芬里尔](../生物万族/index.md)（巨狼·速度残影）；苏晓潜入偷袭、最后**飞掷斩龙闪贯喉**钉杀芬里尔、救下被抓伤的艾斯德斯（+2.6% 源·噬灵者 +15 法力）。**首个加入帝国阵营**（+1% 源）、被艾斯德斯任命为**直属副官**（顶替老副官杰罗姆）。
- **奇袭斩杀努马·塞卡**（四卷19–20章）：开战前夜，艾斯德斯派他潜入「咚咚塔要塞」刺杀极北军首领 [努马·塞卡](../../人物/努马·塞卡.md)——苏晓以毒烟 + 割喉，六小时余取其首级而归（+1.1% 源·噬灵者 +14 法力）；极北军遭夜袭灭顶。艾斯德斯定**战后带苏晓回帝都**。
- **班师帝都 · 见小皇帝与大臣**（四卷21–23章）：随艾斯德斯回帝都、入皇宫面见傀儡 [小皇帝](../../人物/小皇帝.md) 与真正掌权的 [大臣](../../人物/大臣.md)；殿内苏晓**感知并斩杀偷袭皇帝的契约者刺客**、初识三兽士同僚（利瓦 / 达伊达斯 / 妮乌）。大臣献「**嫁祸夜袭部队**」之计（借刀排除异己）。
- **当诱饵引夜袭 · 重创雷欧奈**（四卷24–27章）：[夜袭部队](../../人物/娜洁希坦.md) 登场（首领娜洁希坦 + 赤瞳 / 雷欧奈 / 拉伯克 / 塔兹米 / 布兰德等）。苏晓协艾斯德斯当诱饵，追杀来探敌情的 [雷欧奈](../../人物/雷欧奈.md)（狮子王·属性高于他）上百公里、断其臂砍成血人，被 [拉伯克](../../人物/拉伯克.md)「交叉之尾」拽走续命。**幻影旅团 [卡尔](../../人物/卡尔.md)（编号11）亦在本世界、灭杀了一批旁听契约者**。
- **港口血战 · 杀布兰德 / 赤瞳逃生**（四卷28–32章）：苏晓拦截来收帝具的夜袭部队（已反杀三兽士），以震荡弹开局、**极限青钢影斩杀王牌 [布兰德](../../人物/布兰德.md)**（+3.6% 源），三连击放倒 [赤瞳](../../人物/赤瞳.md)、贯心一刺却因其**心脏右位**未杀掉（赤瞳=小气运之子主角）。玛茵 / 塔兹米跳河、赤瞳逃逸——苏晓追击中开发出「**青钢影坐标感知**」，追至花街地道、夺得秘境里的紫色「[帝具改造蓝本](../../力量体系/装备道具.md)」（合成最强帝具图纸·**首见紫色品质**）。
- **讨伐苏晓会议**（四卷33章）：夜袭部队损失惨重（布兰德死、赤瞳/雷欧奈重伤），连夜召开「讨伐苏晓作战会议」（天然呆 [希尔](../../人物/希尔.md) 因蝴蝶效应未死、亦在席）；娜洁希坦称苏晓为「第二个艾斯德斯」、令四人以下遇之即撤、并外出求助。
- **入狩人部队 / 洗劫革命军 / 斩娜洁希坦**（四卷34–40章）：三兽士死后，[艾斯德斯](../../人物/艾斯德斯.md) 重组暗部「**狩人部队**」（苏晓 + [黑瞳](../../人物/黑瞳.md)〔赤瞳之妹〕/波鲁斯/威尔/兰/Dr.时尚），苏晓兼**第三暗部长官**、自悟「心之冥想」。为「搅浑水」逼帝国与革命军提前决战，苏晓追踪接头人摸到**革命军总部**古堡，心理战逼供军官特利、绑走臣具少女**洗劫帝具库**（凑齐「帝具改造蓝本」材料），并**腰斩夜袭首领 [娜洁希坦](../../人物/娜洁希坦.md)**（+3.9% 源·她刚得人形生物帝具须佐之男）。

- **合成布布特尼 / 逼出决战**（四卷41–42章）：逃出革命军总部后（追兵 [切尔茜](../../人物/切尔茜.md) 被沿途陷阱整惨放弃），苏晓用「帝具改造蓝本」**合成出生物类帝具召唤物「[撕空恶犬·布布特尼](../../人物/苏晓的随从与召唤物.md)」**（狼狗坐骑 + 奶妈·他的首个随从）。大闹革命军直接逼得**革命军首领震怒、提前进攻帝都**。
- **剿杀夜袭 / 撞上卡尔**（四卷43–46章）：狩人部队的 Dr.时尚、兰被 [卡尔](../../人物/卡尔.md) 所杀；苏晓借布布的嗅觉循塔罗牌气味**嗅出夜袭部队总部**。艾斯德斯率狩人部队全员（5 人 + 几万军压境）剿杀夜袭（代理首领赤瞳），苏晓负责拖住赤瞳——却在森林中先撞上 [卡尔](../../人物/卡尔.md)（700+ 法术的双枪法师），二人死战（进行中）。
