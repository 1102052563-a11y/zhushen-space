# 棘虫星·虫族战场（第27卷·五阶世界争夺战·1打3）

[苏晓](../../人物/苏晓.md) 五阶的第一场 **世界争夺战**——轮回乐园被 **死亡 / 圣光 / 天启三乐园 1 打 3** 围攻的「血色战场」。战场是中型星球 **棘虫星**（虫族 + 原始图腾部落的领地·资源富饶却危机四伏）。

!!! tip "为何是 1 打 3"
    世界争夺战是各乐园**靠战争抢世界、掠夺资源**来供养契约者变强的机制（属性强化仓的养护费等大头由乐园出·这些资源全靠抢）。天启乐园输轮回乐园太多次成「死敌」（但因是「挖矿的」资源多、还有资格再战）。本次轮回乐园被三方围攻——契约者得在战场里**加入土著势力**补人数差。

## 世界设定（棘虫星）

- **难度** Lv.43～Lv.53（战争世界）；**本世界无法获得世界之源**；战争任务 = **世界之核**（两环·找到并夺取）；开**功勋系统**（杀敌方契约者得 3~5 功勋·胜利按功勋发奖）。**严禁向土著提及轮回乐园**（违者处决）。
- **唯一迫降点**：四方乐园的飞船被强制迫降到同一片大草原（开局即千人混战）。
- **四方乐园人数**：轮回乐园 336 / 死亡乐园 342 / 圣光乐园 409 / 天启乐园 820（合计 1500+·5 打 1）。临时获**通用语 Lv.50**（四方认证·战后遗忘）。
- **两大土著势力**：
    - **虫族**：分**智慧虫族**（有文明 / 社会 / 法律·因打不过战争虫族 + 图腾部落而躲海岛）与**战争虫族**（以母虫为核心建虫巢·三大型巢 + 无数中小巢·极危险）。虫族五阶段：**母体 → 虫后 → 女王 → 母皇 → 主宰**（主宰对所有母皇有支配权·史上仅一位·那时图腾部落差点灭）。
    - **五大图腾部落**：**太阳**（火·农耕 / 熔锻金属·最和平）/ **亜蛙**（毒·保守居雨林）/ **高卓**（巫咒·原始神棍·得「神谕」就疯）/ **涅摩尤**（祭献·信几十个邪神）/ **哈萨罗**（**掠食**·最凶残·吞食生物获其部分能力·体壮战力最强）。**真相**：原始部落才是**入侵者**（家园资源枯竭、祭献 1/3 人口迁入棘虫星）。
- **祭神（水牛哥）**：[霸主级虚空生物](../生物万族/index.md)——分叉巨角的浅黄水牛、与自然融为一体却强到非几百名五阶契约者可敌（从虚空逃来避羽族猎人追杀·角是力量之源）。被契约者用小石子砸到怒、一脚震干湖 / 踩成肉泥，**驱散了开局的千人混战**。

## 角色在本世界（苏晓·养虫族大军）

苏晓判断硬拼必输（1 打 3），不与大部队（[共生盟·暮光](../../人物/虫族战场配角.md) 领的轮回乐园方）混入土著势力，而是**自己创造一支虫族大军**——他有 [战争领主](../../力量体系/称号.md) 称号（指挥战争按数量加群体增益·万人时全属性 +2）+ [恶魔之罐](../生物万族/index.md)（控生物）+ [枯萎之种](../../力量体系/装备道具.md)（伪主角种子）三件利器。

- **开局混战 + 死寂降临**（二十七卷1–4章）：苏晓先看完世界简介才参战、斩死亡乐园机械眼 / 圣光乐园寒焰；被三乐园十几人围攻时用刃道刀·时 / 血刃车轮反杀，最狠的是用黑王套装终极 **[死寂降临](../../力量体系/装备道具.md)** 把 300 米内**百余名契约者一次性同化秒杀**。混战因 **祭神** 暴走被迫中止（各方四散）。
- **抓母虫立虫巢**（二十七卷7–10章）：满山找虫族母体、捡到被原女王赶出、快饿死的一阶**幼生母体「[棘拉](../../人物/虫族战场配角.md)」**——喂饱 + 注 [恶魔之罐](../生物万族/index.md) 恶魔之力（提忠诚）+ 喂 [枯萎之种](../../力量体系/装备道具.md)（命运之力降进化风险）+ 连用 15 张临时召唤契约才契约成功。在远离部落的森林立**大本营虫巢**（母体产「螳甲」工兵建巢·吞血肉转「生物能」）。
- **爆兵发展**（二十七卷11–14章）：布布汪 / 阿姆 / 巴哈打猎喂虫巢、螳甲搬运、虫巢长到 10 米（一阶上限）；爆出**毒刺**（战斗猎手·生物毒 + 利刺喷射）猎兽攒生物能。攒满 2000 生物能让棘拉 + 虫巢**晋二阶**（解锁狩猎齿 / 孢子坦克 / 监视巢穴·能量上限 1 万·可吞别的虫巢获基因记忆）。苏晓由此定策：**直接打劫别的虫巢夺生物能 + 抢基因片段**（比打猎快·四阶虫巢避开·先打二阶虫巢）。

!!! note "各方加入的土著势力（二十七卷11章）"
    [暮光](../../人物/虫族战场配角.md)（轮回乐园方）→ **哈萨罗部落**（掠食·喝血盟誓）；[亚克洛](../../人物/虫族战场配角.md)（死亡乐园）→ **太阳部落**（火）；圣光乐园 → **亜蛙部落**（毒）；天启乐园 → 80% 在挖矿。苏晓独走「自建虫族大军」路线。

- **打劫虫巢 + 偷母皇生命石 + 晋三阶**（二十七卷15–19章）：苏晓以「噬灭 700 米外狙杀母体 + 布布汪报点」秒杀二阶母体、巴哈引走失去母体指挥的虫族（虫族失母体战力暴跌 90%），整巢搬回吞噬——比打猎快几百倍。布布汪潜入 **四阶母皇**巨型虫巢偷出 11 颗高纯度 [生命石](../生物万族/index.md)（≈11 万生物能·启动资金）。棘拉吃生命石**晋三阶（女王级）**，因体内恶魔之力觉醒**恶魔兽**（近人形·四阶战力的三阶战斗虫族·进化室出精英个体·战斗本能互通会成长）——七分钟爆 150 只，迅速爆到数千。
- **世界之核被夺 → 决战转场**（二十七卷22章）：**圣光乐园**仅 3 天就找到**世界之核**、送到棘虫星中心的**古遗迹·巴巴卡伯**（放置后 11 天形成「世界坐标」·守住即胜）。苏晓拒征召、凭 [老猎人](../../力量体系/称号.md) 称号接下**特殊任务「力挽狂澜」**（Lv.59·本方代表·其余轮回乐园契约者无收益返园·成功功勋 +300%·败则处决）——独自扛三乐园联盟。
- **诱敌之计 · 雾光虫巢**（二十七卷23–29章）：苏晓料定三乐园会找他虫巢、遂演一出戏——打下一处三阶 **雾光虫巢**、用意识传导装置 + 3 颗灵魂结晶强控其女王爆 **3538 只普特斯**当诱饵，引三乐园联盟 + 太阳部落 **2500 烈阳战士**（克虫族）来「摧毁」。圣光暗杀手 **[千花](../../人物/虫族战场配角.md)** 暗杀掉雾光女王（被苏晓 [血之兽](../../力量体系/天赋.md) 秒杀）——三乐园误以为已灭苏晓虫族、安心了。真大本营的恶魔兽继续疯狂爆兵（已 4288 只·42 精英）。
- **贪婪之章**（二十七卷30章）：苏晓破解死寂城老圣修用两瓶酒换的「神灵木」真身 = **[贪婪之章](../../力量体系/装备道具.md)**（黑王套装吸走其死寂之力才显形·10 位魂灵具像挑战·练刀宝物）——挑战排第 10 的老圣修魂灵（纯刀术六阶战力），12 秒败。

!!! note "三乐园联盟智囊三巨头"
    死亡乐园 [亚克洛](../../人物/虫族战场配角.md)（惩戒骑士·机械高温）+ 天启乐园 **[沐雨师](../../人物/虫族战场配角.md)**（水 / 冰系·如沐春风的智囊）+ 圣光乐园 **[阿芙萝](../../人物/虫族战场配角.md)**（超级大奶妈·自带圣光环）。死亡第一梯队 / 天启第二（出钱出矿）/ 圣光第三（辅助）·分工明确。
