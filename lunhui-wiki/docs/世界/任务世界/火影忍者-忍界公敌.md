# 火影忍者：第十五卷再入（忍界公敌）

[苏晓](../../人物/苏晓.md) 晋升三阶后**被强制经历一次「大型开放性世界」**——从龙珠 / 死神 / 火影 / 猎人四者中抽中**火影忍者**，且系**首次以常规手段进入**（第九卷那次是用「斩首大刀」世界钥匙把大剑世界改成的火影·见 [火影忍者（青钢之崛起）](火影忍者.md)）。乐园**同步了他上次的世界进度**：衍生世界身份、声望机制、剧情人物（友好 / 敌对）关系全部保留——这把他直接推到了「**忍界公敌**」的险地。

## 世界设定 · 任务（十五卷2章）

- **位置**：雨之国（雨忍村附近）｜**世界难度 Lv.3～??**（开放性世界·区域不同难度不同）｜世界之源 0%｜同入契约者 **157 名**（进度相近）。
- **主线任务「忍界大战的序幕」**（Lv.28·60 天）：以任何方式**挑起忍界大战**（须五大忍村同时参与、逼其结盟）。**奖 2 点真实属性点**（≈ 旧 10 点普通属性点）·罚强制处决。难度 Lv.28 远超苏晓的 Lv.20——乐园对他实力判定极高。
- **猎杀任务「清理 TQ-9670 号外来者」**：用任何手段清理之（实力较强·相距 < 1 公里自动开启追猎）。**奖黄金荣誉勋章 ×5**·罚强制参与「死斗囚笼」一次。**前缀「TQ」+ 罕见的「有惩罚」**令苏晓怀疑这是**其他乐园的契约者（间谍 / 外来者）**，而非内部违规者——疑为开放性世界独有。

## 苏晓的处境 · 忍界公敌

- **三村死敌 + 两村敌对**：上次他作为晓主干、与 [长门](../../人物/长门.md) 一同攻打木叶，亲手杀了 [夕日红](../../人物/夕日红.md) / [犬冢牙](../../人物/犬冢牙.md) / [卡卡西](../../人物/卡卡西.md)；又与 [迪达拉](../../人物/迪达拉.md) 绑走一尾、[我爱罗](../../人物/我爱罗.md) 被抽尾兽而死（七成概率被 [千代](../../人物/千代.md) 复活）；起始点雾忍村也被他杀了 [青](../../人物/青.md) 而结仇。**五大忍村三死敌（木叶 / 砂忍 / 雾忍）+ 两敌对（岩 / 云）**。
- **但实力今非昔比**：四属性已转**真实属性**、刀术大师 Lv.39、[青钢影 Lv.39](../../力量体系/技能.md)——上次他近身才能袭杀影级（拉开距离必死），现在**有了与普通影级正面战斗的能力**（胜负看临场）。但千手柱间 / 宇智波斑 / 六道级 / 八门凯 / 辉夜仍非他所能敌。**火影力量三层**：① ≈ 二阶（前期卡卡西 / 夕日红）② ≈ 三阶高级 = 影级 ③ 跳跃到五~六阶（苏晓推测）。
- **优势 = 晓元老 + 带土合作**：他是捕了多只尾兽的晓元老级人物，且与 [带土](../../人物/宇智波带土.md)（伪「宇智波斑」）有密切合作——晓内带土真正信任、能共同执行计划的仅苏晓。这次话语权绝对不低。

## 剧情背景（十五卷2–3章）

- **长门已死**：上次进攻木叶后，**[长门](../../人物/长门.md) 被木叶下忍 [漩涡鸣人](../../人物/漩涡鸣人.md) 感化**（鸣人继承了自来也的意志）、放弃毁灭木叶而死。[小南](../../人物/小南.md) 因受苏晓「救长门」之恩而照顾昏迷一月的他，却在察觉苏晓是「带土的手下」后翻脸逃走（长门「背叛」晓、她不敢回雨忍村）。
- **鸣人「英雄」的真相**：他「感化」长门的背后，是**木叶趁机吞下雨之国的经济**——雨之国外贸被切断又重开、被课 40% 税、酒水只能贱卖给木叶（[志村团藏](../../人物/志村团藏.md) 主导压榨）。鸣人这下忍之所以封「英雄」，是因他间接帮木叶夺取一国经济、圆了两代火影攻雨之国的夙愿。**雨之国实为代晓组织吸火力**。
- **找晓据点**：火之国境内的晓据点已被晓自行炸毁；「空之戒」的「幻灯神之术」因长门已死而失效（联络中断）。苏晓打算「钓鱼」闹事引晓主动联络，途中以直感揪出 **[白绝](../../人物/绝.md)** 的细胞分身（孢子之术·坐标监控）——离找到晓不远了。

## 与晓会合 · 三方「月之眼计划」（十五卷5–7章）

苏晓被 [白绝](../../人物/绝.md) 引到火之国地下新基地、重归晓组织——现任首领是 **[带土](../../人物/宇智波带土.md)**（仍伪装「宇智波斑」·苏晓早知此秘）。带土邀苏晓当首领（论捕尾兽 / 攻木叶的资历他不输任何人），苏晓婉拒、选择拥护（当首领 = 拉仇恨当活靶子）。现役晓仅 6 名正式成员：带土 / 黑白绝 / [迪达拉](../../人物/迪达拉.md) / [鬼鲛](../../人物/干柿鬼鲛.md) / [佐助](../../人物/宇智波佐助.md) / 苏晓（迪达拉最信任苏晓·曾同生共死）。

## 任务分配 + 五影大会（十五卷6–9章）

- **五影大会（导火索）**：[奇拉比](../../人物/奇拉比.md) 是 [四代雷影·艾](../../人物/四代雷影.md) 的义弟，被抓后四代雷影震怒、提议三天后在中立国**铁之国·雪风山**召开**五影大会**（明为忍村联合对付晓、暗为**向木叶追责**〔尾兽都丢、独木叶无恙〕+ 泼脏水）。白绝的孢子情报传回后，**带土改变计划**：全员（除抓八尾的鬼鲛）扑向五影大会、伺机**袭杀火影 / 雷影 / 水影**（死一两个影即可搅乱局势、利于抢剩余尾兽、且暂不想让五大忍村真联合）。**苏晓主动请缨参与**——既要清算与 [照美冥](../../人物/照美冥.md) 的旧怨，又好奇「**没有漩涡鸣人参与的忍界大战会是什么结果 / 改变它能换多少世界之源**」。
- **五影齐聚**：[团藏](../../人物/志村团藏.md)（代理火影·「败犬的绝地反击」·趁机为转正铺路）、[照美冥](../../人物/照美冥.md)（五代水影·新官）、[我爱罗](../../人物/我爱罗.md)（五代风影·需立威）、[四代雷影·艾](../../人物/四代雷影.md)（追责发起者）、[大野木](../../人物/大野木.md)（三代土影·城府堪比团藏）皆赴会。苏晓带队提前埋伏雪风山外围（阿姆拉雪爬犁·布布汪入寒区智商解锁），佐助一心要趁机杀团藏（已知 [鼬](../../人物/宇智波鼬.md) 真相）。

- **带土决定宣战**：开会前带土找苏晓商定——五影大会无论结果如何都会达成「灭晓」共识（晓崛起本就靠四大忍村廉价雇佣代战、云隐村是间接推手），故他要**趁机向五大忍村宣战**（先发制人·宣战≠开战·晓在暗 / 五村在明、开战时机由晓定）。苏晓附加了条件（换好处）。
- **会场博弈 + 团藏的别天神**：五影齐聚雪风山高塔六层（铁之国大将 [三船](../../人物/三船.md) 主持·曾与半藏交手存活成名）。[大野木](../../人物/大野木.md) 连番嘲讽 [我爱罗](../../人物/我爱罗.md) 下马威；**[团藏](../../人物/志村团藏.md) 暗用「别天神」**（宇智波止水的写轮眼瞳术·无察觉侵入大脑改人意志·用一次需歇十几年）操控了 [三船](../../人物/三船.md)——再抖出「晓首领是宇智波斑」（以火影之名担保）搅乱四影，引三船「中立」提议**五大忍村联盟、联军首领交火影**（团藏借此谋六代火影之位·原著本由 [青](../../人物/青.md) 戳穿、但青早死在苏晓手中）。
- **晓搅局**：[白绝](../../人物/绝.md) 分身突现五影包围圈嚣张挑衅，苏晓的**炼金渡鸦落其头顶、骤然自爆炸碎五影会议桌**（苏晓临时把阿姆的召唤位切给渡鸦以操控）；[佐助](../../人物/宇智波佐助.md) 在高塔一层大开杀戒、引 [四代雷影·艾](../../人物/四代雷影.md) 不顾会议下塔追击。

## 斩杀四代雷影 + 送头颅离间（十五卷16–19章）

苏晓收下带土的**原始柱间细胞资料（1/5·金色·369 评分）**后正式出手（他懂炼金药剂学 Lv.20·研究柱间细胞能大幅提升自制药剂）：

- **先除护卫**：[阿姆](../../人物/阿姆.md)（[布布汪](../../人物/布布汪.md) 操控·凛冬血脉 + 冰雪女神光环极抗揍）的终极技「凛冬之怒」把雷影护卫**达鲁伊**冻成冰雕，苏晓剃身贴近一刀斩首（心灵之刃免疫了希的幻术）——**斩雷影候选人达鲁伊·世界之源 7.6%·得淡金宝箱**；另以界断线 + 替身术周旋间逼退护卫希。
- **斩杀 [四代雷影·艾](../../人物/四代雷影.md)**：雷影一记「沉怒雷斧」踢飞苏晓、险断其小腿，自己却被斩断左小臂；苏晓用 [狂猎之裁决](../../力量体系/装备道具.md)（猎魔套手枪）趁灵魂感电定身**五发爆炸针全中 + 同时引爆**（500 伤 + 易损状态〔防御 −35% / 治疗 −20% / 移速 −10%〕·胸口炸出五个露肋骨的肉坑），再削肉放血、斩断双臂、一刀斩首。**斩杀四代雷影·世界之源 +10.4% → 累计 18%·法力升至 3586·首得「传说级宝箱」**（暗金色·80% 出金 / 2% 出传说级装备·传说级与灵魂绑定·苏晓首次见识此品级 + 装备品级完整阶梯）。雷影败得心服（堂堂正正被斩首·技不如人）。
- **送头颅离间**：[我爱罗](../../人物/我爱罗.md)（被阿姆拖住没及时救雷影）赶到、被带土用「佐助已去找其他影」逼退；苏晓让带土把**雷影完整头颅送给 [团藏](../../人物/志村团藏.md)**——这是利诱（木叶的幻术忍者 + 科技器械可读取雷影记忆 → 七成把握灭云隐村、入驻雷之国得两大国之力），团藏几乎无法拒绝。此举正合苏晓主线「挑起忍界大战」（借晓身份做让五村愕然的事、逼其尽快联合开战）——**苏晓的计划比带土更疯狂**：带土要一步步铺月之眼，苏晓只要双方开打、不管胜负。

## 月之眼计划摊牌 · 第四次忍界大战宣战（十五卷21–24章）

雷影死后带土现身会议室、向四影**摊牌「月之眼计划」全貌**：宇智波一族地下石碑（六道仙人留·按写轮眼→万花筒→轮回眼解读渐多）记载——**九尾兽是「十尾」被六道仙人分散的查克拉、十尾的躯壳即月亮**；带土要**复活十尾、成其人柱力**、把瞳术映在月亮上发动**无限月读**，以大幻术把全人类合为一体、造一个无纷争（且有野原琳）的世界。他抓尾兽、要把五大忍村忍者聚起来，正是为凑发动无限月读的海量查克拉。四影一致反对（「活在幻术里与死有何异」），拒交剩余两只尾兽——带土遂宣布「**第四次忍界大战，就此宣战！**」。

- **苏晓的阿波罗威慑**：苏晓中途带 [阿姆](../../人物/阿姆.md) 进场、捏 [太阳神阿波罗](../../力量体系/消耗品材料.md) 威慑（诈称爆炸范围五千米·实则一千米），逼四影乖乖坐下听带土说完——比带土口头的「宇智波斑」虚名威慑力大得多。
- **苏晓的宣战收益**：轮回乐园判定他参与「第四次忍者大战的宣战」、阵营 = 宣战方、**贡献度 18.2%**、此为 **S 级事件**；三选一奖励（A 成就「[掠天惊澜](../../力量体系/成就.md)」+15% / B 称号「战争领主」/ C 真实属性点 ×1），他选 **A**（掠天惊澜从 25% → **40%**）。这呼应「**世界之子**」机制——杀 [路飞](../../人物/路飞.md) / 一护这类世界之子会遭世界反噬（28000 米金属陨石砸死·迄今无契约者杀世界之子能活·剧情人物杀则无反噬、但苏晓拿不到掠天惊澜进度）。
- **雷影头颅物归原主**：带土临时改主意、没把雷影头颅送团藏（佐助的须佐能乎更值得培养、让佐助与团藏实战即可）。苏晓也乐得不送（从他这进攻过木叶的人手里送出、团藏为名誉必不收）。两人分头行动：**苏晓盯水影 [照美冥](../../人物/照美冥.md)（旧怨待清）、带土盯团藏**、事成火之国据点会合。

## 袭杀照美冥 · 斩鬼灯残月（十五卷27–32章）

五影大会散场（四忍村结成联盟、[大野木](../../人物/大野木.md) 被推为联军首领·[团藏](../../人物/志村团藏.md) 顺水推舟支持——阴谋）。苏晓尾随水影 [照美冥](../../人物/照美冥.md) 去清旧怨，却撞上她的**「意料之外的护卫」[鬼灯残月](../../人物/鬼灯残月.md)**——苏晓初入火影时的雾忍村狱友（雾隐主战派·四代水影旧部·曾刺杀照美冥失败〔被 [长十郎](../../人物/长十郎.md) 所阻〕、照美冥诈称处死实则留他当**暗中利刃**），其实力**比照美冥本人更强**：他是**二代水影鬼灯幻月的传人**，会无限爆破的「**蒸危暴威**」（孩童分身外油内水·运动积热膨胀→ 800 度蒸汽大爆炸 200 米·结合雾隐之术才达此高温·不能用利器斩〔斩即爆〕），辅以水化 / 气泡 / 水炮空练弹。

苏晓没去破解蒸危暴威（人家钻研十年），而是**布局杀本体**：埋蜘蛛炼金炸弹、把界断线藏进泥水、远程压制青钢影能量蓄势——趁鬼灯残月引爆蒸危暴威水化时，界断线勒腿 + 压制的青钢影骤然爆发（328 真伤）令他剧痛失防，被自己的蒸汽爆炸反噬烧成重伤、再一刀断喉。**斩杀鬼灯残月·世界之源 +9.7% → 累计 27.7%·法力 3616·得金色宝箱**。苦战两影级、苏晓肺脏 / 小腿受创，靠 [永恒水晶瓶](../../力量体系/装备道具.md) 续命。[阿姆](../../人物/阿姆.md) 缠住 [长十郎](../../人物/长十郎.md)（忍刀七人众唯一健在·双刀「鲆鲽」可合成战锤）；**狡猾的照美冥见冲天蒸汽柱、料知鬼灯残月已败，当即逃跑**（待续）。

## 密室斩杀照美冥 · 旧怨了结（十五卷33–35章）

照美冥逃命途中被一支三人小冒险团（**络腮胡 / 毒雉 / 叛逆少年**·用淡金传送卷轴）「虎口夺食」式救下、还用毒解了 [白绝](../../人物/绝.md) 缠她的孢子之术——但她识破三人「无巧合」的图谋（想结交剧情人物牟利），又算准苏晓必至，便**用假情报支走护卫 [长十郎](../../人物/长十郎.md)（谎称雾隐暗杀部队被截于东侧）、独守山洞密室赴死**。她以山洞地形 + 熔遁「火焰海」+ 沸遁「巧雾」（火水结合的强腐蚀酸雾·封闭环境威力倍增·三浓度）+ 水遁「大瀑布之术」注水成强酸液，把苏晓逼到极限（苏晓靠水下护目镜 + 反击盾 + 剃硬冲·算准 10~12 秒内必须分胜负）。最终苏晓**抛斩龙闪（界断线锚定）贯穿照美冥心脏钉上岩壁 + 屠戮钉手 + 断喉**——**斩杀照美冥·世界之源 +6.2%（→ 累计 33.9%·因白绝全程孢子辅助、收益削减 40%）·法力 3664·暗紫宝箱**。照美冥至死不屈（「永远不会与你这种恶徒合作」），苏晓只为「你当初差点宰我、所以我回来找你」的旧怨（与任务无关·清完即炸塌山洞离开）。这了结了他入火影世界的雾隐村起始恩怨（[青](../../人物/青.md) / [鬼灯残月](../../人物/鬼灯残月.md) / 照美冥皆殁）。

## 佐助弑团藏 · 柱间细胞资料（十五卷36–37章）

## 团藏之死 + 收编香磷 + 佐助当炮灰（十五卷38–43章）

- **团藏自尽**：垂死的 [团藏](../../人物/志村团藏.md) 挟持 [香磷](../../人物/香磷.md) 当人质（「我是唯一能改变忍者世界的变革者、必须活下去」），却被 [佐助](../../人物/宇智波佐助.md) 千鸟锐枪一并贯穿（连人质带敌人）。逃到桥中段被带土堵住，团藏遂启动早布于胸口的「**里四象封印术**」（同归于尽·把周围一切封入施术者尸体·施术者必死）——带土虚化下沉、佐助靠通灵鹰才逃出百米黑球。**团藏死**（临终「就算到最后、我也没当上火影」·并自毁止水写轮眼 + 大脑成浆糊、死无全尸不留情报）；带土回收其尸体写轮眼。苏晓评团藏「狗急跳墙说得这么清新脱俗」、又叹他一手好牌被带土搅乱。
- **收编香磷**：[佐助](../../人物/宇智波佐助.md) 弃下被自己刺穿心脏的香磷，苏晓却把她救活、收为「**可移动的超级恢复药剂**」——香磷有**漩涡一族稀薄仙人体**（生命能量多·治疗效果强过五代火影纲手·咬一口即愈、≈ 一号药剂 ×20~30），又是感知型。苏晓用白色小蜘蛛炼金炸弹堵住她心脏伤口（每天最多咬 5 次·超 10 次会死），并**激活她的个人好感度**（详见「好感度机制」）——香磷极度惧怕苏晓（曾多次被他挟持 / 咬伤），好感度下限被压到 2、上限因苏晓低魅力降至 80，等于「**刀架脖子**」保她不背叛（布布汪魅力高、其对香磷好感度反达 72、上限 130）。
- **佐助当炮灰**：追兵是**剧情主角 [漩涡鸣人](../../人物/漩涡鸣人.md)（九尾·仙人模式）+ 几十名契约者**——苏晓本想抓鸣人，但与鸣人已是死敌（抓我爱罗 / 斩卡卡西 / 攻木叶 / 断雏田臂 / 斩犬冢牙）、且鸣人身边契约者太多，遂作罢。带土与苏晓改**给佐助注射提纯柱间细胞**（恢复瞳力 + 查克拉·活性 2 小时·无树木化风险·苏晓凭炼金 Lv.20 调出）——先戴香磷的高度近视镜废其天照、再打服强注，让佐助**开两小时第三形态须佐能乎**当炮灰挡木叶追击部队、吸引契约者火力（柱间细胞副作用：此战后佐助万花筒严重透支、恐失明）。佐助第三形态须佐能乎（持盾持剑）与仙人模式鸣人激战（鸣人查克拉太暴躁、幻术无效），顺手斩杀围观的刺客契约者。

## 好感度机制（十五卷42–43章）

剧情人物对契约者有「**个人好感度**」——**核心作用不是攻略、而是判断该剧情人物是否会背叛**。规则：好感度跌破下限即「冻结」、需重新升到某阈值才再激活；**契约者魅力越低、好感度上限越低**（苏晓魅力低、上限被永久 −20）；但**让对方极度惧怕，会大幅压低「下限」**（香磷惧苏晓、下限 +8 → 只要好感 ≥2 就不背叛）——即「用刀架脖子」代替魅力。对比：地精 [尼古拉斯·凯撒](../../人物/尼古拉斯·凯撒.md) 那种是「没下限」（好感骤升骤降）。
