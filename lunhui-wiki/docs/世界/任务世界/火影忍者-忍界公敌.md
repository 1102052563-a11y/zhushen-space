# 火影忍者：第十五卷再入（忍界公敌）

[苏晓](../../人物/苏晓.md) 晋升三阶后**被强制经历一次「大型开放性世界」**——从龙珠 / 死神 / 火影 / 猎人四者中抽中**火影忍者**，且系**首次以常规手段进入**（第九卷那次是用「斩首大刀」世界钥匙把大剑世界改成的火影·见 [火影忍者（青钢之崛起）](火影忍者.md)）。乐园**同步了他上次的世界进度**：衍生世界身份、声望机制、剧情人物（友好 / 敌对）关系全部保留——这把他直接推到了「**忍界公敌**」的险地。

## 世界设定 · 任务（十五卷2章）

- **位置**：雨之国（雨忍村附近）｜**世界难度 Lv.3～??**（开放性世界·区域不同难度不同）｜世界之源 0%｜同入契约者 **157 名**（进度相近）。
- **主线任务「忍界大战的序幕」**（Lv.28·60 天）：以任何方式**挑起忍界大战**（须五大忍村同时参与、逼其结盟）。**奖 2 点真实属性点**（≈ 旧 10 点普通属性点）·罚强制处决。难度 Lv.28 远超苏晓的 Lv.20——乐园对他实力判定极高。
- **猎杀任务「清理 TQ-9670 号外来者」**：用任何手段清理之（实力较强·相距 < 1 公里自动开启追猎）。**奖黄金荣誉勋章 ×5**·罚强制参与「死斗囚笼」一次。**前缀「TQ」+ 罕见的「有惩罚」**令苏晓怀疑这是**其他乐园的契约者（间谍 / 外来者）**，而非内部违规者——疑为开放性世界独有。

## 苏晓的处境 · 忍界公敌

- **三村死敌 + 两村敌对**：上次他作为晓主干、与 [长门](../../人物/长门.md) 一同攻打木叶，亲手杀了 [夕日红](../../人物/夕日红.md) / [犬冢牙](../../人物/犬冢牙.md) / [卡卡西](../../人物/卡卡西.md)；又与 [迪达拉](../../人物/迪达拉.md) 绑走一尾、[我爱罗](../../人物/我爱罗.md) 被抽尾兽而死（七成概率被 [千代](../../人物/千代.md) 复活）；起始点雾忍村也被他杀了 [青](../../人物/青.md) 而结仇。**五大忍村三死敌（木叶 / 砂忍 / 雾忍）+ 两敌对（岩 / 云）**。
- **但实力今非昔比**：四属性已转**真实属性**、刀术大师 Lv.39、[青钢影 Lv.39](../../力量体系/技能.md)——上次他近身才能袭杀影级（拉开距离必死），现在**有了与普通影级正面战斗的能力**（胜负看临场）。但千手柱间 / 宇智波斑 / 六道级 / 八门凯 / 辉夜仍非他所能敌。**火影力量三层**：① ≈ 二阶（前期卡卡西 / 夕日红）② ≈ 三阶高级 = 影级 ③ 跳跃到五~六阶（苏晓推测）。
- **优势 = 晓元老 + 带土合作**：他是捕了多只尾兽的晓元老级人物，且与 [带土](../../人物/宇智波带土.md)（伪「宇智波斑」）有密切合作——晓内带土真正信任、能共同执行计划的仅苏晓。这次话语权绝对不低。

## 剧情背景（十五卷2–3章）

- **长门已死**：上次进攻木叶后，**[长门](../../人物/长门.md) 被木叶下忍 [漩涡鸣人](../../人物/漩涡鸣人.md) 感化**（鸣人继承了自来也的意志）、放弃毁灭木叶而死。[小南](../../人物/小南.md) 因受苏晓「救长门」之恩而照顾昏迷一月的他，却在察觉苏晓是「带土的手下」后翻脸逃走（长门「背叛」晓、她不敢回雨忍村）。
- **鸣人「英雄」的真相**：他「感化」长门的背后，是**木叶趁机吞下雨之国的经济**——雨之国外贸被切断又重开、被课 40% 税、酒水只能贱卖给木叶（[志村团藏](../../人物/志村团藏.md) 主导压榨）。鸣人这下忍之所以封「英雄」，是因他间接帮木叶夺取一国经济、圆了两代火影攻雨之国的夙愿。**雨之国实为代晓组织吸火力**。
- **找晓据点**：火之国境内的晓据点已被晓自行炸毁；「空之戒」的「幻灯神之术」因长门已死而失效（联络中断）。苏晓打算「钓鱼」闹事引晓主动联络，途中以直感揪出 **[白绝](../../人物/绝.md)** 的细胞分身（孢子之术·坐标监控）——离找到晓不远了。

## 与晓会合 · 三方「月之眼计划」（十五卷5–7章）

苏晓被 [白绝](../../人物/绝.md) 引到火之国地下新基地、重归晓组织——现任首领是 **[带土](../../人物/宇智波带土.md)**（仍伪装「宇智波斑」·苏晓早知此秘）。带土邀苏晓当首领（论捕尾兽 / 攻木叶的资历他不输任何人），苏晓婉拒、选择拥护（当首领 = 拉仇恨当活靶子）。现役晓仅 6 名正式成员：带土 / 黑白绝 / [迪达拉](../../人物/迪达拉.md) / [鬼鲛](../../人物/干柿鬼鲛.md) / [佐助](../../人物/宇智波佐助.md) / 苏晓（迪达拉最信任苏晓·曾同生共死）。

!!! abstract "三人共谋的「月之眼计划」（十五卷5章）"

    带土 / 宇智波斑 / 黑绝三方的计划**初 / 中 / 后期相同（轮回眼 → 十尾 → 月之眼无限月读），唯终极目的不同**：
    - **[带土](../../人物/宇智波带土.md)**：用无限月读造一个有「野原琳」的幻术世界（情种）。
    - **宇智波斑**：同样月之眼、造无战争世界——实则**利用带土当祭品复活自己**。
    - **黑绝（[绝](../../人物/绝.md)）**：终极目的是**复活母亲「大筒木辉夜」**——利用带土与斑两人。

    **当前卡在轮回眼**：[长门](../../人物/长门.md) 已死、尸体与轮回眼被 [小南](../../人物/小南.md) 藏起（下落只她知·她已失踪）。带土虽找回自己那只眼，仍须移植轮回眼（要件 = 宇智波血继 + 千手生命力·长门当年是靠漩涡一族生命力硬撑才枯成那样）。**苏晓的主线「挑起忍界大战」与此挂钩**——带土弄不出十尾、大战就打不起来。

## 任务分配 + 五影大会（十五卷6–9章）

- **任务分配**：苏晓用青钢影一验，戳穿 [佐助](../../人物/宇智波佐助.md) 抓来的「八尾人柱力 [奇拉比](../../人物/奇拉比.md)」是替身（带土拧断脖颈→现出章鱼触须·佐助首次捕尾兽被骗·「我亲手抓过的尾兽比你见过的更多」）。带土遂派：鬼鲛抓八尾、**苏晓抓九尾**（他熟木叶结界·须等九尾离木叶）、绝 / 迪达拉找小南。
- **五影大会（导火索）**：[奇拉比](../../人物/奇拉比.md) 是 [四代雷影·艾](../../人物/四代雷影.md) 的义弟，被抓后四代雷影震怒、提议三天后在中立国**铁之国·雪风山**召开**五影大会**（明为忍村联合对付晓、暗为**向木叶追责**〔尾兽都丢、独木叶无恙〕+ 泼脏水）。白绝的孢子情报传回后，**带土改变计划**：全员（除抓八尾的鬼鲛）扑向五影大会、伺机**袭杀火影 / 雷影 / 水影**（死一两个影即可搅乱局势、利于抢剩余尾兽、且暂不想让五大忍村真联合）。**苏晓主动请缨参与**——既要清算与 [照美冥](../../人物/照美冥.md) 的旧怨，又好奇「**没有漩涡鸣人参与的忍界大战会是什么结果 / 改变它能换多少世界之源**」。
- **五影齐聚**：[团藏](../../人物/志村团藏.md)（代理火影·「败犬的绝地反击」·趁机为转正铺路）、[照美冥](../../人物/照美冥.md)（五代水影·新官）、[我爱罗](../../人物/我爱罗.md)（五代风影·需立威）、[四代雷影·艾](../../人物/四代雷影.md)（追责发起者）、[大野木](../../人物/大野木.md)（三代土影·城府堪比团藏）皆赴会。苏晓带队提前埋伏雪风山外围（阿姆拉雪爬犁·布布汪入寒区智商解锁），佐助一心要趁机杀团藏（已知 [鼬](../../人物/宇智波鼬.md) 真相）。
