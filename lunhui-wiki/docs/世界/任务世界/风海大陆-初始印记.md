# 风海大陆·初始印记

轮回乐园第 **50** 个衍生世界，苏晓**晋升九阶后的第二个世界进度**——一个广袤程度远超以往原生世界千百倍的 **超脱·原生世界**。此地**没有法系文明**、自然元素充沛，是被虚空之树与多方乐园阵营公证的争夺场。世界任务只有一环:**找到并夺得「初始印记」**。

## 🎯 目标 · 曙光乐园的初始印记

⚠ **初始印记（曙光乐园·初始印记）**并非乐园本源，而是曙光乐园存放时空之力的**「储藏装置」的唯一钥匙**。当年曙光乐园惨败后，这枚印记被紧急投放到风海大陆、下落不明，直到近期一名海族在深海发现了它——那「储藏装置」至今仍被虚空之树公证，里面**至少存着 200 万盎司以上的时空之力**。⚠ 坏消息是:投放时打得太激烈，初始印记**碎裂成很多块**（海族发现的是最大一块），且因有人试图激活，**散落全大陆的碎片全被半激活**——任何契约者靠近一定范围就会收到虚空之树提示。⚠ 更棘手的是，印记碎片**无法收进储存空间**（拿到就「不是猎人就是猎物」），且世界进度结束时，**持有碎片最多者独得全部**（连未被发现的碎片都会定向回收）。所以重点不是抢到多少，而是**弄死持有得比自己多的人**、成为最多的那一个。

## 🌏 世界史 · 异兽 → 兽族与海族

⚠ 风海大陆已是第八纪元。上古的**异兽**（自称神灵）本因世界之树而不死、争斗无意义;当太阳升起、世界古树枯倒、有了昼夜冷暖生死后，异兽会死了，便**奴役起源稍晚的两族当「斗兽」**替它们争地盘划资源——陆上的称 **古尔瓦族（兽族）**、海岛的称 **迦壬族（海族）**，两族信奉异兽为神灵、为它们惨烈厮杀。⚠ 深渊侵袭时异兽驱两族当炮灰、十不存一，却也在无意间让两族**繁衍能力暴增**;黑暗退去后，兽族与海族**反过来给沉睡苏醒的异兽套上世界树枯木锁枷、当战争巨兽**，成了这一纪元的主角。如今两族各千亿 / 数百亿人口、祖辈死仇、停战即内战，打了几万年。

⚠ 苏晓死死捂着菌毯、坚称那只是「**防滑菌毯**」（凯恩举证有战熊军团战士倒在菌毯上被治疗、跑得更快、体力恢复快三成，苏晓答「还有此事？我之后试验下，不过我猜测，这应该是错觉」；凯恩讨要几包，他答「当然可以，**一个月后**我让人送到永环城」，把老狐狸气到无语）。⚠ 这是精算过的:太早拿出去，一百四十多位领主大半会怀疑并研究菌毯由来，一旦查到虫族苏晓就成众矢之的；**死死攥住、让领主们争破头去讨、最后闹到兽王、在兽王命令下才「不太情愿」地拿出**——那九成以上的领主都敢用。领主们误以为凛冬军团的凶悍全是菌毯之功（实为战争领主称号），渴求更盛;五十多位领主与凯恩干脆一起赖在暮冬城庄园「和苏晓耗上」，直到兽王召见全体领主才散。

- ⚠ **[兽王](../../人物/风海大陆/兽王.md)** 在大议厅睁着眼睛睡觉，只说了一句「白夜，我听说，你最近开发出了种菌毯」就定了调（他从不直接下令）。散会前的闲聊却石破天惊:「**你那菌毯，很像虫族的产物**」——但他选择**替苏晓保密**，条件是「**别在这里孵化战斗虫族，哪怕是一只**」，并递给苏晓一张救命纸条:「今日勿离永环城，海族、施法，多名绝强伏杀」。
- ⚠ 苏晓的还礼:菌毯**交由大统帅·凯恩分配**给各领主。看似利益被兽王与凯恩占尽，实则——由兽王在幕后、凯恩在明面分配，所有领主都不在意菌毯来历、还会争破头抢分配额度，苏晓的进化点收益**超级加倍后再翻一倍**；凯恩「不会让你白提供、这是对兽族的功绩」的许诺又构成虚空之树公证行为，苏晓每提供一份菌毯就能得声望。众领主则把这情分记在兽王头上，七大家族由此不敢再与兽王明争暗斗。

⚠ 苏晓、[神父](../../人物/神父.md)、[巫毒术士·巴泽](../../人物/风海大陆/巫毒术士·巴泽.md)、[凯撒](../../人物/尼古拉斯·凯撒.md) 组成「弑神队」——各怀鬼胎却利益一致。苏晓要 [灵魂死神](../../人物/风海大陆/灵魂死神.md) 的灵魂源质，神父要三神祗的神灵源血（他能吸收任何特性的源血），巴泽要扩张部落地盘 + 渎食神灵血肉，凯撒要财富。⚠ 灵魂死神极谨慎、一个部落倾力祭献都钓不出，却栽在「仪式布设 + 黑暗蠕动 + **地精跳**」的组合上（凯撒人罐合一 + 欺诈者头裹跳萨满舞，中途还误钓来半神、半兽神、混种神灵生物、吞过神血的异生物各一只，全被神父的无形巨口吞了个半饱）。斩神后:苏晓抽走 **2000 多盎司液态灵魂源质**、神父以黄金圣杯抽干神血畅饮、巴泽嘴角裂到耳根吞噬神躯、凯撒扛大布袋卷走全部身家——⚠ 欲神与蛇神经空间镜像目睹「夺魂、饮血、噬血肉、窃财富」这一幕，又看到那四道「**新的野生宝箱 / 新的神血来源 / 新的神灵血肉 / 新的移动钱袋**」的目光，匆忙关闭镜像:**蛇神当夜就逃出了风海大陆**，欲神则被苏晓调来的上千名人马族重骑兵射爆在自己的神域里。黑暗神教大本营（部落南侧黑暗之地的幽暗大教堂）虽被突袭平推、却远谈不上伤筋动骨——高层都藏在暗处，这才是它最难缠之处。
