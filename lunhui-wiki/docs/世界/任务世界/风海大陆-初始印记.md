# 风海大陆·初始印记

轮回乐园第 **50** 个衍生世界，苏晓**晋升九阶后的第二个世界进度**——一个广袤程度远超以往原生世界千百倍的 **超脱·原生世界**。此地**没有法系文明**、自然元素充沛，是被虚空之树与多方乐园阵营公证的争夺场。世界任务只有一环:**找到并夺得「初始印记」**。

## 🎯 目标 · 曙光乐园的初始印记

⚠ **初始印记（曙光乐园·初始印记）**并非乐园本源，而是曙光乐园存放时空之力的**「储藏装置」的唯一钥匙**。当年曙光乐园惨败后，这枚印记被紧急投放到风海大陆、下落不明，直到近期一名海族在深海发现了它——那「储藏装置」至今仍被虚空之树公证，里面**至少存着 200 万盎司以上的时空之力**。⚠ 坏消息是:投放时打得太激烈，初始印记**碎裂成很多块**（海族发现的是最大一块），且因有人试图激活，**散落全大陆的碎片全被半激活**——任何契约者靠近一定范围就会收到虚空之树提示。⚠ 更棘手的是，印记碎片**无法收进储存空间**（拿到就「不是猎人就是猎物」），且世界进度结束时，**持有碎片最多者独得全部**（连未被发现的碎片都会定向回收）。所以重点不是抢到多少，而是**弄死持有得比自己多的人**、成为最多的那一个。

## 🌏 世界史 · 异兽 → 兽族与海族

⚠ 风海大陆已是第八纪元。上古的**异兽**（自称神灵）本因世界之树而不死、争斗无意义;当太阳升起、世界古树枯倒、有了昼夜冷暖生死后，异兽会死了，便**奴役起源稍晚的两族当「斗兽」**替它们争地盘划资源——陆上的称 **古尔瓦族（兽族）**、海岛的称 **迦壬族（海族）**，两族信奉异兽为神灵、为它们惨烈厮杀。⚠ 深渊侵袭时异兽驱两族当炮灰、十不存一，却也在无意间让两族**繁衍能力暴增**;黑暗退去后，兽族与海族**反过来给沉睡苏醒的异兽套上世界树枯木锁枷、当战争巨兽**，成了这一纪元的主角。如今两族各千亿 / 数百亿人口、祖辈死仇、停战即内战，打了几万年。

## ⚔️ 三方势力 · 当前战局

- **兽族（陆地）**：君主**兽王**（钢羽家族·已连出三位兽王·现任垂垂老矣）下设**两位大统帅**（掌 100 多位**兽族领主**·领主有封地、可册封 5 名首领、每首领掌 60~70 万军团）。七座高墙城由**战熊 / 胡狼 / 蛇巫 / 占星 / 风暴 / 铁纹**等七大家族把持——他们正暗中联手要把钢羽家族拉下王位。
- **海族（海洋/岛屿）**：君主**海王**（这一代最狠·要灭了兽族）。海族擅**蛊惑灵魂 / 伪装**（深海女妖），长老会为伪装大统帅·凯恩筹备几十年。⚠ 海王囚困了三大传说铁匠之一的**矮人王**，用他打造的**增长自然元素的元素器物**换来**奥术永恒星（施法者）**的助力——如今是「奥术永恒星 + 海族」在捶兽族（法系炮台把战争族群打懵）。兽族反找**陨灭星（古神）**结盟。**初始印记大概率在海王手中**。
- **部落**：掌图腾 / 剧毒 / 巫术，古老神秘却太松散，相比两霸不够看。

## 🎭 苏晓的身份 · 兽族领主·怒泽·戈温（50卷1–4章）

⚠ 苏晓抽到的初始伪装身份是**兽族北部领主·怒泽·戈温**——一位脾气刚硬的领主，与仇敌**大统帅·凯恩**积怨多年、连续两次「一剑砍下凯恩的脑袋」，却都因凯恩身边的**因果系高手**（戈温的传家宝**骨齿项坠是厄运物**、注定他杀不死凯恩），砍中的都是海族伪装的深海女妖，反被关进监牢等兽王裁定。⚠ 与超脱·原生世界的特殊之处:乐园**不再公证改写原住民认知**（原住民仍叫他「戈温」，需自行演技伪装）。⚠ 这身份表面身陷囹圄、被七大家族下毒陷害凯恩，实则价值全在**领主级·声望等阶**（乐园契约者近乎不可能达到）。苏晓当机立断:把骨齿项坠冲进马桶、亮出灭法身份，与**假死布局**的大统帅·凯恩谈成合作——**苏晓牵制风海大陆所有施法者，换领主之位 + 一块（偏远也行）封地**（好藏几百米高的虫巢）。凯恩让戈温「把领主之位世袭给侄子」（即苏晓），阵营任务**领主威仪**要求声望 >100（苏晓现 −38950），苏晓遂搬出养了很久的太阳领主称号 ★★★★★★★★（塞尔星信仰之力·太阳所养）应对。

## 🗡 猎杀名单·血契 · 转为悬赏功能（50卷1章）

⚠ 六叛徒全斩后，**猎杀名单·血契没消失、反而彻底激活**成了新能力:可**悬赏与自己相关的事物 / 敌人**——进新世界时，名单会概率具现出与苏晓有关的悬赏（付抵押金、完成得 5 倍悬赏、失败抵押金没收）。风海大陆开出四项:①**狼冢**（100 盎司·找本世界银·月狼狼冢·升银月之刃）、②**狩猎死神**（300 盎司·斩本世界灵魂死神）、③**斩杀绝强**（1000 盎司·斩奥术永恒星一名绝强者·苏晓押满 1000·赌法系绝强者可逃可操作）、④**平息混乱**（10 盎司·让两族停战·500 倍回报·后知难度极高、当打水漂）。

## 🐺 领主队 → boss 队（50卷5–12章）

⚠ 苏晓走「精英路线」组队:靠**太阳领主 ★8 + 布布汪光环**把一群精英级单位怼成首领级，自己再以霸主级实力压阵，凑成「霸主领着一群首领」的 **boss 队**，用来奇袭海族要地、狩猎绝强、消化负声望。招募之路堪称群魔乱舞:
- ⚠ **铁堡城贫民窟招募**（凯恩交易添头的名单）:先后收下 [厄格因](../../人物/风海大陆/厄格因.md)（反骨二当家·统领潜质S）、[寂兽·迦·哈维](../../人物/风海大陆/寂兽·迦·哈维.md)（忠诚95的靠谱首领级）、[蝰蛇](../../人物/风海大陆/蝰蛇.md)（光速墙头草）、[恶齿](../../人物/风海大陆/恶齿.md)（生吞海兽垂体中毒的亡命徒）——三人忠诚度相加不过 5 点，全靠太阳领主 +70 压着。皮鲁（斑狐族前财政官）被派去潜伏黑暗戈壁地下城。
- ⚠ **黑暗戈壁地下城**（兽海心照不宣赎回俘虏的地下市场）:以数万金币赎聘 300 名精锐 + 50 头目 + 3 大头目补员（含哈维），全程「要么走、要么来打一仗每人 1000 金币」，用**七战渊龙长子**的车轮战一批批淘汰弱者、把领主队淬炼成人均大头目级、共 450 人（含 150 名越跑越快的人马族重骑兵）。
- ⚠ **凯撒（尼古拉斯·凯撒）** 抢先当上**铁堡城军需官**，用凯恩的批文白提几百套战甲武器、又用「屎黄黏液」破开海族空间合金箱，把劫来的 2000 吨能量水晶变现 200 多万金币（五五分账）——他下一步要去当**海族的军需官**、去敌方阵营商店进货。

## 🏔 暮冬城封地（凛冬封地）· 主线任务·凛冬（50卷15–24章）

⚠ 苏晓领到的封地是北境凛寒之地**暮冬城封地**（又称凛冬封地）:辖十座高墙大城、几千万人口、六个兽族军团，却因老领主去世、财政崩坏、军团无薪无粮而不出战，陷入恶性循环，初始封地评价仅 **D 级**。名义领主是 10 岁的 [小领主·古尔薇](../../人物/风海大陆/小领主·古尔薇.md)，被管家勾结两城主架空。苏晓的经营手法是「氪金游戏」——把白蹄港劫来的粮食、晶脂砸下去拉居民幸福度，封地评价 D→D+→C+ 一路飙升。
- **收服城主**:苏晓请八名城主赴宴、发「缺领主印徽」的粮食批文当诱饵，八城主当晚就取了架空古尔薇的**管家**首级来投诚（管家背后有部落暗中扶持，一碰就碎）。他重用敢站出来的 [城主·芬里斯](../../人物/风海大陆/城主·芬里斯.md)（情报）、[斑狐族·皮鲁](../../人物/风海大陆/皮鲁.md)（财政）、[厄格因](../../人物/风海大陆/厄格因.md)（军团），三方鼎立、狼狈为奸却互不合流。
- ⚠ **主线任务·凛冬**要求封地评价达 C 级/B 级。因苏晓把后续几环（揪出隐患=杀管家/退巫毒术士、军团出战、征战白蹄港）都提前完成，虚空之树**预测式合并**了任务、把奖励上限从 100% 提到 180%、变更为「离世前把封地评价提到 B 级」的最终环。
- ⚠ **军团出战 + 菌毯**:苏晓任 [厄格因](../../人物/风海大陆/厄格因.md)/哈维/蝰蛇/恶齿为军团首领、佩战争领主称号靠**统领潜质**给六军团 95% 加成出战主战场（每日稳定 1800 声望），并让军团带 **菌毯（Lv.30）**——菌毯吸海族灵魂能量、经母巢转「进化点」喂 [棘拉](../../人物/棘拉.md)。苏晓的算计是让**其他领主争着用菌毯**（不择手段争来的宝物才没人防），最大化进化点。

## 🌿 部落阵营 · 黑暗神教 · 虚空深渊危机

- **部落阵营**:风海大陆三大势力之三，地盘在兽族下方，被剧毒沼泽、湿地、红树沼泽、最危险的**暗雨带**（生灵禁区）层层隔开，故兽海几万年混战却无暇理它。**本世界 99% 的异兽栖息在部落地盘**（产永恒命源），部落掌图腾/剧毒/巫术，最强战力是**赫鲁伊**（巫毒术士）与**德鲁伊**（巫医）。信奉灵魂死神的 [巫毒术士·巴泽](../../人物/风海大陆/巫毒术士·巴泽.md) 即赫鲁伊。
- **黑暗神教**:本世界三大教派（海神教/异兽教/黑暗神教）里信仰深渊的那个，是各封地的毒瘤——在暮冬城活跃，拉低超凡治安与居民幸福度、令苏晓每日少得 1500 声望。苏晓不愿亲自动它（会不死不休），改栽赃它「劫了巴泽长孙的灵魂药材」、把巴泽的杀意引过去。
- ⚠ **虚空深渊危机**（[秘纹师](../../人物/风海大陆/秘纹师.md) 泄露）:虚空出现一处**一次比一次大的特殊深渊通道**（核桃→3 米），根源是奥术永恒星吞了太多自然元素。这才是奥术永恒星肯来风海大陆帮海族锤兽族、换矮人王元素器物的**真正原因**（提升虚空自然元素总量才是解法）。苏晓判断那深渊通道七成在瑟菲莉娅去的**绿茵大湿地**。
