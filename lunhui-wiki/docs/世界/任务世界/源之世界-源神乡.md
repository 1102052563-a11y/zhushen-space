# 源之世界·源神乡（第36卷·七阶高危原生世界·技法强大·资源极丰厚）

## 世界背景 · 源之世界

- **两大国度 · 陨神之战**：**圣守国**（陨神战守卫人族最前线·衰败严重）+ **西塔国 / 塔林王国**（大陆西侧霸主·艾德里王族·9500 万人口）结盟——曾以首位屠神者天巴为榜样，向古老神灵总攻、几近屠灭之（陨神之战·两国半残、古老神灵「归乡」）；周边上百小国贫苦、受两国压制。

苏晓剥出**迄今体积最大的「完整」灵魂记忆**（仅观看不吞噬·+8 意志力）——初王·艾德里·德温的真相：他本是战场上杀了 16 名战士被俘的「**笼中野兽**」（骨瘦如柴、被烙满铠甲当仆从贩卖）；一生只有两人对他有善意——赠他名字「德温」并放走他的**老铁匠**、以及善待他的神灵后裔少女**艾德里·希耶儿**（偷跑出神乡·被其兄「人族都该死」所累）。希耶儿死于入侵者手，笼中野兽背起她的尸体、胸口嵌上【源】、屠尽 12575 名入侵者，从此成为**守源人**（取她姓「艾德里」+ 铁匠名「德温」）。他在熔山**等了 447 年**继承人、感到自己将「崩朽」——直到等来有资格取走【源】的苏晓，「他可以休息了」。
