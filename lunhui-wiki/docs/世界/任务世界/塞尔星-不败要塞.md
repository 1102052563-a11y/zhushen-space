# 塞尔星 · 不败要塞

[苏晓](../../人物/苏晓.md) 画之世界通关后进入的**第 5 个八阶世界**（第45卷）——一个因「**机械污染**」而改写命运的后启示录世界。它属于**天启乐园**，因半颗世界之核被机械污染融入世界本体、只剩半颗被天启公证，归属权不完整，遂再度爆发**世界争夺战**（这次是天启乐园 VS 圣光乐园 VS 守望乐园·因轮回 / 死亡 / 圣域三乐园上次争此世界败过、无法参战）。苏晓以**入侵者**身份潜入（未佩戴 [掠天惊澜](../../力量体系/称号.md) 称号·以免与世界之子敌对），世界难度 **Lv.53～78**。

## 世界起源 · 黑雨与机械污染

塞尔星本是**低神秘、重科技**的世界（航海→蒸汽→电气→科技一路发展）。冷兵器时代起，天空就飘着一块块被当地科学家归为自然现象、命名为「**暗氤**」的铁黑色液体——真身是**第二纪·炼金文明的巅峰造物之一**（一颗半机械 / 半生物结构的星辰化成的液态·苏晓触到那金色触须时竟有「仿佛是缔造者」的错觉，疑与灭法之影 / 炼金文明渊源相关）。当塞尔星人研究出核弹、进行首次试爆，蘑菇云引爆了暗氤——它气化融入云层，降下一场**黑雨**，带来「机械污染」:

- **四成以上生灵**无法适应污染而死；活下来的**超九成生灵金属化**（骨骼金属化 / 半身金属+血肉·活性金属而非工业金属·内部仍流血）。
- 「灾后纪元」开启，几百年间三大种族与势力格局稳固下来。

## 三大种族

- **人族**（正常人类）：凭灵魂力量具现替身雏形、再以「**生物活性金属**」填补完善，打造替身 / 生物装甲作战。已从历史主角退居二线。
- **眷族**（半金属化人类·自称「受上天眷恋进化的半机械种族」）：部分躯体「活性金属化」，身体素质超强 + 精神异变带来的**金属系超凡能力**（如把铅弹展开成螺旋金属刃条把人绞成肉馅）。整体最强，三大不败要塞皆归其掌握。
- **异化兽 / 畸变兽**：机械污染后的野兽（对污染适应力更弱、死亡率高，但异变更彻底），战斗力强、不乏**霸主级**。数量达阈值就会集结冲击各大要塞（削减觅食者、也抑制人族 / 眷族繁衍）。
- 另有**其他种族**：全是眷族后天杂交培育的苦力（如 [猪头人](../生物万族/index.md)），被眷族视作「下等生物」乃至可宰卖的私有财产。

## 移动要塞 · 活性矿石

由于异兽、猎手、拾荒者横行，定居与游牧皆不可能，人们培育出**移动要塞**——一种半机械 / 半生物的**超巨型活体生命**，外表是金属装甲层、下方探出几十条粗壮机械触角移动，内部有可居住空间，是可移动的家园。它**有简单智慧、满足条件可进化成长**（吃活性矿石供能）。

- **要塞等级**：T0～T5 六阶（数字越小体型越大、人口越多），之上还有三座眷族独有的「**不败要塞 / 不动要塞**」（常年展开如要塞城·对应眷族最强三大势力）。
- **[活性矿石](../../力量体系/装备道具.md)**：塞尔星独有资源，维持要塞运转 / 供要塞进化，是全世界的**硬通货**。猪头人下矿采「粗矿」，经洗炼提纯成活性能量。

## 苏晓的开局 · 夺要塞养炮灰

苏晓延续「坐牢开局」的老传统——因**魅力 −10**，他接管了一个「宰过几百名眷族」的恶名身份，被判送到**末日要塞（T5·619号要塞）**当矿奴。他没急着越狱，而是观察后精准夺权:

- **策反猪头人**：给第一个敢反抗的 [豪斯曼](../../人物/塞尔星/豪斯曼.md) 一个苹果、一句「敲死他」，就此点燃 36 名猪头人**起义军**（要塞共 673 名猪头人·仅十分之一敢反抗）。他的算盘是——用抗揍的猪头人组「人海战术」上战场当炮灰，在「血肉磨盘」里靠 [战争领主](../../力量体系/称号.md) 士气加成磨出精英战士。
- **夺要塞核心**：逼要塞头领 [利·西尼威](../../人物/塞尔星/利·西尼威.md) 交出**要塞核心**（一颗跳动的心脏·控制要塞的钥匙），苏晓成为**末日要塞新任指挥官**，反过来留西尼威当「眷族代言人」经营挖矿。
- **召唤高级炮灰**：从 [【源】石](../../力量体系/装备道具.md) 里召回 [溺之首领·猎潮](../../人物/联盟星/猎潮.md)——用「不战斗就没价值」的无敌之盾抗拒他，却被苏晓一句「天巴老斑鸠」+「带你回源·神乡」拿捏，乖乖入队当高级炮灰。
- **发展蓝图**：提升伙食、改建要塞、去哨塔要塞城买雌性猪头人（让雄性为雌性十倍·争配偶权），给猪头人「看见太阳」再让他们怕失去——从而自觉拎起武器追求自由。

## 敌对阵营（三乐园参战者）

- **天启乐园**：主力含 [莫雷](../../人物/原始大陆/莫雷.md) + [月使徒](../../人物/苍龙大陆/月使徒.md)（画之世界后自信恢复·月使徒开启苟命发育流欲当团战幻神）·百余名契约者聚废弃商场·因无轮回 / 死亡 / 圣域三方疯子参战而窃喜、忘了「世界入侵」这回事。
- **圣光乐园**：违规者 [仙姬](../../人物/联盟星/仙姬.md) 等；[灰绅士](../../人物/炎灵星/灰绅士.md) 与仙姬正躺在虚空·奥术永恒星海港的沙滩椅上**钓白夜**（灰绅士拦住想去现实世界杀白夜的仙姬:「在现实你引以为傲的主动能力用不出分毫」）。
- **守望乐园**：第三方参战乐园。
