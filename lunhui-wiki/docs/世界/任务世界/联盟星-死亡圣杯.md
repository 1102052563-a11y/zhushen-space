# 联盟星 · 死亡圣杯（第43卷）

[苏晓](../../人物/苏晓.md) 的第 3 个八阶世界（继魔灵星、暗星之后）——**全开放·原生世界**，世界之源 0%、世界简介「**我们隐于夜晚，敬畏神秘**」。这是一个**蒸汽 + 电力科技时代**、超凡之力已隐入黑夜的世界，核心主题是「**危险物（异诡之物）的收容与灭杀**」。苏晓因这次世界选取方式为「恒定」（现实世界所得「生命残灰」变成被动钥匙物品）而被拖进此地；又因佩戴 [八星称号·掠天惊澜](../../力量体系/称号/index.md) 进入，落得「坐牢开局」。

## 世界史 · 从「厄运物」到「危险物」

- **异诡之物的战乱（王国纪元 117 年）**：国王与领主纷争，异诡之物被滥用于战争。最富有的「**阿陀斯家族**」资助各方、贩卖异诡之物牟取暴利，导致人口断崖式下跌（最少时仅剩不足 200 万平民、男性占比不到五分之一）。
- **阿陀斯家族的转变与衰败（295→373 年）**：阿陀斯家族促成三王、五公、十二领主签下「不可再侵占」停战条约，从此不再贩卖、转而疯狂收集异诡之物；却因得到一件命名为「**厄运物-01**」（无法销毁 / 摆脱 / 控制）的异诡之物而家道中落、仅剩几人。
- **联盟成立 · 神圣骑士团（联盟纪元 419 年）**：各王国 / 公国 / 盟和国联合成立**联盟**——废奴隶制 / 世袭制 / 领主制 / 私掠许可 / 国王法，改行议会制（近乎平民平等·当初成立联盟者正是最强的超凡者们）。阿陀斯家族仅剩的嫡系血脉耗尽家财成立「**神圣骑士团**」，专门寻找、消灭、囚困厄运物，厄运物-01 是首个封印的。
- **超凡隐入黑夜 · 一分为二（1272→1672 年）**：科技井喷，神圣骑士团淡出视线、把「厄运物」更名「**危险物**」并分级（厄运物-01 = 危险物 **S-001**）。骑士团因理念分歧分裂:一派「敬畏神秘、只收容 / 消灭」→ 第三研究所 → **收容机构**；一派「存在即合理、可利用危险物」→ 永夜教会 → **日蚀组织**。此后蒸汽机、电力、钢铁巨舰陆续问世。

## 危险物 · 分级与「必死性」

危险物（异诡之物）分四级——处理它们「95% 以上」靠情报人员发现、文职排查、外勤试探、超凡者成员灭杀 / 收容，人手不足则层层加派（外勤 → 猛犬小队 → 副军团长亲自出马）:

- **C 级**：能致多人死亡、无成长性（2~3 名机关成员处理）。
- **B 级**：有一定成长性、初期即致死、放任会造成区域恐慌。
- **A 级**：完善的成长性 + 隐匿性、可致一市死亡——开始带「**必死性**」，须用人命试探摸清规律才能灭杀 / 收容。
- **S 级**：快速成长 / 初始即处于最强点，**必带必死性**，只靠蛮力或高智商都会「白给」，须实力 + 极强临场应变力、绝不触发必死性。必死性大致分**肉体 / 精神 / 灵魂**三类——苏晓四属性发展抗肉体、伯格之心 + 高灵魂强度 500 抗灵魂即死，唯独精神类必死性最需防范。
- **深渊残片**：消灭危险物的特有奖励「扭变的深渊能量凝结体·残片」——危险物疑似源自某次「深渊之孔」在本世界打开、放出并「扭变」的**深渊能量**（深渊即黑枫树种子的起源·灭法者 / 奥术永恒星等都因探深渊亏到吐血）。若属实，最高梯队的危险物会比想象更危险。

## 冬泉镇 · 灾厄铃铛（S 级危险物 · 42卷8–11章 · 已消灭）

苏晓的首个收容目标——边境度假小镇**冬泉镇**里盘踞四年、令上任副军团长威德曼铩羽而归的 **灾厄铃铛**（S 级·灵魂特性）:

- 一个**头扣竹篮、腰挂铃铛的女鬼**形象——以**温泉水为媒介**杀人（受害者体表结霜、内脏被灼伤）、直面它会触发**灵魂即死**、还能束缚奴役幽魂为其做事（整个冬泉镇居民、老板娘千婆婆都是它的傀儡幽魂）。老巢在「红池温泉」（千婆婆世代经营）。它把红池旅馆变成隔绝外界的幻境领地——供台上挂着上百颗铃铛，每往木碗里投一颗铃铛就得进它领地打败它的一部分、才能碎一颗，全碎才能逼出本体（这是它自定的「打猎」规则）。
- **苏晓的破法**：他有意激怒它、把矛头引到自己身上（护住布布汪 / 巴哈）——凭 **灵魂强度 500 + 刀术宗师「灵魂之刃」双重豁免**了它的灵魂即死、反噬令它 3 天失去该能力缩回老巢。随后靠阿姆冰封温泉水断媒介、龙影闪穿透避开细水线，用**普通阿波罗包铃铛投碗**（火力足够、鬼祟皆渣）连炸九颗炸穿它藏身地，逼出本体（一具有血有肉有灵魂的躯体·灾厄铃铛的「完美载体」铃铛女）一刀斩杀、并斩了扑来的千婆婆幽灵。**得 9.72% 世界之源 + 灾厄宝箱 + 深渊残片 ×1**。

## 美人鱼争夺战 · 苏晓 vs 金斯利的对弈（43卷12–21章·进行中）

- **冬泉镇的圈套**：苏晓识破被囚一事是 [日蚀组织](../../人物/联盟星/日蚀组织.md) 领袖 [金斯利](../../人物/联盟星/金斯利.md) 与联盟议会合谋的圈套——目的是拖住机关、好在海上截获危险物 **[美人鱼（S-006）](../生物万族/index.md)**。冬泉镇灾厄铃铛孕育的小男孩身上有靠近美人鱼的关键（子嗣之血）。日蚀「环 8」[华兹沃](../../人物/联盟星/华兹沃.md) 率近 200 名日蚀成员来夺男孩，被苏晓的**刃之领域**（首次实战·100 米高频斩击）杀得溃逃；金斯利的后手潜伏者 [哥雅](../../人物/联盟星/主角队.md)（清纯少女）也开始接近苏晓。
- **美人鱼（S-006）的真相**：苏晓查得美人鱼会**哭泣**（引来其他危险物·含死亡圣杯 S-002！）与**歌唱**（俘获一切生灵的爱意、变成护她的奴仆）——金斯利要它，是为了用它引来并利用多种 S 级危险物。而苏晓要它，是为了**用它的哭声引来死亡圣杯**。
- **两颗棋子的对弈**：苏晓与金斯利像坐棋盘两侧对弈——都不先拍碎对方棋子，而是推动各自的「伪世界之子」互相靠近美人鱼:苏晓一方是吞噬者寄体 [艾奇](../../人物/联盟星/艾奇.md)（还偷偷喂了 [命运之种](../../力量体系/装备道具/index.md) 强化、拿到 S 级危险物残骸打造的手套 [裂杀](../../力量体系/装备道具/index.md)），金斯利一方是寻母的 [白发少年](../../人物/联盟星/主角队.md)（另备替补道尔·穆）。两个伪世界之子叠加、加上众多成员暗中铺路 + 子嗣之血，主角队捕获美人鱼的概率就变得可观。联盟议会则是「一神带一坑」里的顶级猪队友（炸棘花报社、潜入窃血等一连串「让人智熄的操作」，反倒帮了苏晓）。
- **主角队五人出海**：苏晓与金斯利各自暗中选人，凑成主角队——[艾奇](../../人物/联盟星/艾奇.md)、白发少年、[奈奈尼 / 道尔·穆 / 曼黎](../../人物/联盟星/主角队.md)。他们查清棘花报社被炸案、乘渔船出海找美人鱼。苏晓则借休琳从联盟军方借来钢铁战舰（葛韦少校护航）、伪造特许出海批文，率机关全部可抽调力量随后出海——**主角队捕到美人鱼后，才是苏晓与金斯利真正夺鱼的时刻**（美人鱼只会落入苏晓 / 金斯利 / 联盟议会三方之一）。

## 苏晓的终极目标 · 第三天赋觉醒链（43卷18章·揭示）

苏晓图谋美人鱼与死亡圣杯，是为了一条跨越多个世界的**天赋觉醒计划**——最终目的是获得**灭法者的独有天赋能力**:

1. 捕获美人鱼 → 用其哭声引来死亡圣杯（S-002）；
2. 饮下死亡圣杯的水液 → **临时觉醒第三种天赋**（持续约 15 天·默认二次觉醒状态）；
3. 用 [古老意志](../../力量体系/装备道具/index.md)（圣灵级·古精灵杰作·激活极限突破天赋任务）→ 将临时第三天赋**突破为永久天赋**（九成把握）；
4. 进入 **树生世界**（每人毕生仅能进一次·苏晓近期将接征召）→ 找到灭法者遗留的 [天赋唤醒装置](../../力量体系/装备道具/index.md)；
5. **牺牲掉第三天赋** → 换取**灭法者的独有天赋能力**。


## 得手美人鱼 · 决战金斯利 · 主线第二环（43卷22–31章）

- **追到未知大陆**：主角队渔船出海遇暴风雨（[猎潮](../../人物/联盟星/猎潮.md) 乌鸦嘴「言出法随」·机械大鸟遭雷劈），金斯利安排海兽载他们加速。他们潜到美人鱼原居所（海底巨大贝壳·十几万被投海的尸骨），靠奈奈尼「回溯」查出美人鱼被赤膊草裙的原始人掳往一片**未知大陆**（阿兹特克风·葛韦少校靠螺环仪逆转判出是未知大陆边角）。

- **给仙姬扣「全民公敌」**：世界之源排行榜首位是圣光乐园违规者 [仙姬](../../人物/联盟星/仙姬.md)（与灰绅士 / 神父同为要在树生世界搞事的头目·八阶最顶尖梯队）。苏晓单挑九成会败，便使阴招——让机关全面通缉仙姬，使她与南部 / 东部联盟 / 收容机构 / 日蚀四大势力全敌对、强制佩戴**全民公敌**减益称号（交涉 −30 好感、无法与 98.7% 商人交易），大幅拖慢她的世界之源获取。
