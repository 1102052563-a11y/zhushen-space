# 进击的巨人（任务世界）

苏晓的**第三个任务世界**，以漫画《进击的巨人》为蓝本（三卷1章）。开篇副标题「**帝国之毁灭**」。

## 世界信息
- **难度**：LV.7 · 困难（大量契约者同入）。
- **开局**：消耗乐园 0.015 盎司**时空之力**，本世界契约者最低综合评分须达 **D 级**；未掌握语言耗 100 乐园币自动掌握（三卷1章）。
- **世界梗概**：人类被普通巨人威胁、退守**三面「神之壁」**百年。845 年超大型巨人踢破最外的**玛利亚之墙**，人类死伤惨重、退守**罗塞之墙**；850 年罗塞之墙又被破坏（剧情切入点）（三卷1章）。
- 剧情主角 **[艾伦](../../人物/index.md)**：生母被巨人吞食，为复仇加入训练兵团、立志进调查兵团驱逐巨人。

## 任务
- **主线任务：罗塞之墙攻防战**（Lv.7）——抵挡罗塞之墙被攻破后涌入的巨人、并封锁破损洞口；10 个自然日、奖 **3 属性点 + 8000 乐园币、无惩罚**。这是比连锁任务更难的「**攻防任务**」（详见 [机制](../../总览/轮回乐园机制.md)）。
- **猎杀任务**：清理 **18433 号、16770 号、14030 号**三名违规者契约者（1 公里内自动开启追猎、无惩罚）。乐园一反常态地给猎杀者派了**三个**目标，似对清理违规者颇为急迫（三卷1章）。
- **提示**：主线为攻防任务，请与除违规者外的契约者**积极合作**。
- **回归资格任务（攻防战后发布）**：主线「**世界之源**」（Lv.5）——获得 **15% 以上世界之源**；可用一切手段获取；期限 **16 个自然日**；奖励 **回归资格**、惩罚**强行处决**。苏晓注意到本世界回归任务出现得比东京喰种早得多，推测之前的攻防战正是乐园让契约者「适应」此类大型合作任务的铺垫（三卷35章）。
- **职工者随团**：彼岸花等冒险团进入衍生世界会招收付费的**职工者**（详见 [机制](../../总览/轮回乐园机制.md)）随队，提供保护以换取乐园币；彼岸花被苏晓杀退后，失去保护的职工者一度向苏晓求护、被其拒绝（三卷34章）。

## 分区与势力（三卷2章）
罗塞之墙被踢出三道破口，攻防战按破口分三战区：**一区（东）** 由 [彼岸花冒险团](../../势力/index.md)（全女性·兵长**利威尔**支援）守；**二区（西）** 由 [兄弟会](../../势力/index.md)（研究狂韩吉·佐耶 + 调查兵团）守；**三区（南）** 仅有艾伦三新兵，乐园遂补入强者（[尘](../../人物/尘.md)、[恩徒](../../人物/恩徒.md)、苏晓）。
- **巨人要害 = 后颈一片**（伤它处会蒸汽恢复）；分普通巨人（呆缓）与**奇行种**（快·难防）。详见 [生物·万族](../生物万族/index.md)。
- **攻防贡献值**：杀巨人累积，胜利时前 5 名额外奖励（第 1 灵魂结晶×5 / 第 2 蓝装 / 第 3 绿装 / 第 4–5 乐园币）。试炼阶段榜：① [亚当](../../人物/index.md)56 ② 冷月50 ③ [尘](../../人物/尘.md)48。
- 原著民：剧情主角 **艾伦**、超人气 **三笠·阿克曼**、智囊 **阿尔敏**（靠**立体机动装置**斩巨人）；契约者被乐园安排成「戴罪立功的前调查兵团囚犯」身份。

- **世界崩坏**（四卷1章）：因九巨人中**超半数传承者被同时杀死**，巨人之力混杂、**始祖巨人（初始）试图复生却找不到合适载体→巨人之力崩溃→巨人世界规则崩坏→时间重置→世界崩坏**（陨石坠落、岛屿碎裂、时空之力笼罩全世界）。苏晓因已完成回归任务，赶在时空之力侵袭前**回归乐园**。这是「气运之子杀之无益」的极端破解：让始祖之力**找不到载体复生**，等于把整个世界规则玩崩。

## 进度（已通关 · A+）
苏晓以假身份「TK·317」**反向操控**马莱间谍三人组（[莱纳](../../人物/莱纳·布朗.md) / [贝特霍尔德](../../人物/贝特霍尔德·胡佛.md) / [阿尼](../../人物/阿尼.md)），趁城墙骚乱**斩残伪死劫走 [艾伦](../../人物/艾伦.md)**、超大型砸墙重创调查兵团，一行叛逃出墙；途中又俘虏颚之巨人 [尤弥尔](../../人物/尤弥尔.md) 与王族血脉 [赫里斯塔](../../人物/赫里斯塔.md)。他借原著情报回避巨人之森、横穿**帕拉蒂岛**奔艾伦故乡，在**地下室**出售实验物品得 **14% 世界之源**（回归任务 15% 即将达成）+ 蓝药剂、揭开艾伦身世。出墙时遭调查兵团追上，苏晓**一枪爆头狙杀团长艾尔文**、令敌群龙无首（韩吉代理团长），在追击战中打散调查兵团。得知主线时限不足以抵达马莱后，他**翻脸屠尽六名巨人传承者**（仅阿尼携赫里斯塔逃脱）——杀超半数传承者导致**始祖之力复生失败、巨人世界规则崩坏、时间重置**，苏晓回归乐园、**通关 A+**。

> **本世界教训**：任务时限短、可接任务极少、地图广阔到无快速赶路手段——苏晓判断这是乐园让他「**熟悉中小型战场任务**」的过渡世界、并不适合深度开发；他靠「聚拢巨人传承者 + 取地下室之利」硬是把坑爹世界玩出花样。深层真相已全明：艾伦兼**进击 + 始祖巨人之力**且为气运之子；唯王族血脉（如赫里斯塔）能彻底发挥始祖之力；初代王与始祖之力签**不战契约**；马莱以「无垢巨人」惩罚艾尔迪亚罪人、运至本岛（墙外巨人之源）。
