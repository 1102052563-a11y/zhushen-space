# 巫师大陆·猎人与猎物（第30卷·全开放原生世界·营救夏）

[苏晓](../../人物/苏晓.md) 晋六阶后的第一个世界——为**营救被困此处的职工者 [夏](../../人物/夏.md)、带回 10850 盎司时空之力**而来的 **全开放原生世界**（难度 Lv.49～56·世界之核归虚空之树）。

!!! tip "全开放原生世界 = 违规者老巢"
    任何乐园的契约者都可能被随机抽到进入（一场混战可能凑齐 5+ 乐园的人）、均可激活任务 / 宝箱 / 拿世界之源；但**世界之核归虚空之树**、且**违规者多达 500+**（不分乐园）——比战争世界更热闹凶险。本世界**禁止伤害己方契约者**，主线全部完成后须回初始降临点才能离开。

## 世界设定（巫师大陆）

- **科技绝迹、神秘学独大**：环境恶劣到**只能发展神秘学**（科技发展不起来·故仍是冷兵器时代）；无王国、**巫师会掌管一切**。
- **巫师 = 潜质 + 魔药**：有潜质者饮「[魔药](../../力量体系/装备道具/index.md)」即成巫师（过程极危险·要先变怪物再恢复·仅巫师会掌握技术）——**灰巫师**（五阶战力）→ **秘法巫师**（六阶）→ **不朽巫师**（六阶顶尖·仅三位·巫师会最高掌权者）。力量来得太快太简单（背后有代价）。
- **🩸真视之契约 · 世界的真面目**：苏晓初入时所见是「**虚假的繁荣**」（阳光 / 幸福平民）——签 [真视之契约](../../力量体系/装备道具/index.md)（老巫师 [修斯·阿奇德](../../人物/巫师大陆群像.md) 给）解除世界限制后才见真实：破败腐朽、阴云蔽日、平民面黄肌瘦食不果腹。难度也从 Lv.10~52 还原为 **Lv.49~56**。
- **黑暗 / 旧神 / 它们**：本世界被「**它们**」（站在最高处的存在·如巨婴般**吮吸世界**）侵蚀——**「黑暗」**会侵蚀一切（**在本世界伤害活物即被黑暗侵蚀**·巫师们的疯狂正源于魔药副作用 + 黑暗·侵蚀超 53% 永久削体力 / 生命）。**[巫师之城·赫鲁](../生物万族/index.md)** = 旧神影响的中立危险区。

## 任务结构（引导式四环）

- **主线·加入巫师会（第一环·Lv.49）**：抵达基尔城加入巫师会。✅已完成（老巫师修斯送真视之契约 + 传送阵直送赫鲁）。
- **主线·支配者（第二环·Lv.55）**：找到并击杀**支配者**（与「恶之源·白山羊」同阶的存在·「它们」之一）。🚧进行中。
- 后两环未知（最终目的 = 夺回 10850 盎司时空之力 = 找到夏）。
- **猎杀任务·自由猎手**：违规者太多、追猎权限全开·可自由清理（按危险度奖钻石荣誉勋章）。
