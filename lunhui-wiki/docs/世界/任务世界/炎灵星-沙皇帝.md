# 炎灵星·沙皇帝（第35卷·灭法者之狩猎·未开发原生世界·猎杀违规者）

[苏晓](../../人物/苏晓.md) **晋七阶后的第一个世界**——一个不属任何乐园 / 虚空之树的「三不管」未开发原生世界，违规者能在此为所欲为。苏晓的任务 = **将闯入此地的违规者杀绝**。

!!! tip "为何是「灭法者之狩猎」"
    此世界已战死 5 名猎杀者 + 16 名战斗天使 + 7 名先驱 + 6 名游侠 / 惩戒骑士，违规者被围杀后临时撤走、只留 1 人观察。**4 名违规者**（都是七阶顶尖·恐含老阴哔）将陆续返回——苏晓是唯一进入的猎杀者（另有天启乐园战斗天使 [莫蕾](../../人物/炎灵星群像.md) 等）。轮回乐园判定他硬拼胜率低，故给了 **5 瓶荣誉复苏药剂 + 勇士族徽 + 称号燃炼权限**。此世界**无法获世界之源**（评分只按任务完成数 / 度）、但支线 / 隐藏任务极多（给低品阶称号）。

## 世界背景 · 炎灵星

- **沙漠世界 + 圣魂**：43% 严重沙漠化、超凡随处可见。**圣魂**修行 = 捕超凡生物的灵魂在体内构建「圣魂」、以之沟通 / 共鸣自然元素而变强（稳扎稳打·战力 3~7 阶）；将圣魂淬炼至极限者 = **半神**（仅 6 位）。士兵通过考核得的是次级「**灰魂**」（只增体魄·不能觉醒超凡·50 岁后衰退）。
- **沙焰国 vs 秘鲁国**：**沙焰国**（沙漠之国·渴望绿洲·奴隶制横行·首都沙都·14 岁 [沙皇帝](../../人物/炎灵星群像.md) + 美艳王后 + 大祭法 [乌诺](../../人物/炎灵星群像.md) + 司政 [卡罗丝](../../人物/炎灵星群像.md)）vs **秘鲁国**（矮人的绿洲联盟国·渴望黄沙下矿脉）——两国各有一中立神灵庇佑、互相牵制、连年交战（现伪停战）。神灵只与两国之王平起平坐、半神能把神灵拉下神坛。

## 进程（按章）

- **混商队 + 会合凯撒 + 违规者挑衅**（三十五卷1–7章）：苏晓（勇颅族直系后裔身份·勇士族徽）混沙漠商队到埃斯沙巴，与领路人 **[凯撒](../../人物/炎灵星群像.md)**（前沙焰财政官·被卡罗丝排挤）会合。途中遭**违规者远程操控本土居民下毒袭击**（留「斯芬克不会白死」的字误导·实为拖延苏晓探索）；[莫蕾](../../人物/炎灵星群像.md) 也在秘鲁国被镶钻金牙违规者逼到「变青蛙」逃命。苏晓定下计划：**从沙焰国弄到军权 → 用 [战争领主](../../力量体系/称号.md) 称号加成 → 前线打胜仗招兵**，用「称号燃炼」把战争领主升到四 / 五星，再反杀违规者。
- **进沙都见沙皇帝**（三十五卷7章）：卖巨岩龙 4250 金币起家；进沙都罗赛宫，[半神赤沙](../../人物/炎灵星群像.md)（第五半神·王影）引路见 **14 岁的沙皇帝**——沙皇帝竟与苏晓伪装的身份「6 年前有私交、还被他揍过一拳」。
- **揍乌诺立威 + 讹到大统帅之位**（三十五卷8–11章）：苏晓当场跑去把大祭法 [乌诺](../../人物/炎灵星群像.md) 揍一顿立威，在早宴上被激将「七天内攻占塞戈纳城，破不了就处决我」——换来**大统帅之位**（30 万编制·实得 2 万败军）。乌诺派副统帅 [阿瓦雷](../../人物/炎灵星群像.md) 想害他、被苏晓「都是自己人别客气」反手毒杀；布布汪去乌诺金库偷 8 万金币「友情赞助」，苏晓给士兵**发安家费 + 治伤 + 每人 1 金币**收买军心，把败军变成愿卖命的兵。
- **称号燃炼 + 首战破断角部队**（三十五卷11–15章）：把 [战争领主](../../力量体系/称号.md) 称号从 ★★★ 燃炼到 **★★★★**（浴血荣耀 / 善战者 / 伸张正义 / 团队屠戮者作燃料）。抵边境首战碾破秘鲁精锐**断角部队**、生擒指挥官 [赖顿·红须](../../人物/炎灵星群像.md)（秘鲁王表弟），换回 40 颗灵魂结晶。赖顿·红须爆出 **弑王史**（见下「弑王史真相」·当今沙皇帝是「假王」）。
- **荣沙要塞·截溃军发家**（三十五卷16–20章）：秘鲁蓄谋已久攻破荣沙要塞、围困半神大统帅 [尼波索](../../人物/炎灵星群像.md)。苏晓不救要塞、专**截留溃军**滚雪球（26000 → 满编触发战争领主最大效果），率 5 万兵全灭秘鲁**暴风锤军团** + 杀退第三军团（浴血奋战 + 布布汪光环），解尼波索之围、收 10 万金币谢礼。战斗天使 [莫蕾](../../人物/炎灵星群像.md) 传纸条警告：**七阶最强违规者 [绯世](../../人物/炎灵星群像.md) 将至**。
- **造黑龙战争武器·备攻塞戈纳**（三十五卷21–22章）：士兵累到 72590 名；苏晓用恶魔之力 + 古神阿撒托斯心脏组织 + 黑渊黑龙血 + 生之泉，靠**洛氏阵图**改造野生巨岩龙成战争生物 **[黑龙·米狄斯](../../力量体系/装备道具.md)**（喷黑焰·52600 古龙之力）。备战塞戈纳城（5 万驻军·战争兵器林立）。
- **黑龙屠城·一小时破塞戈纳**（三十五卷23–24章）：苏晓骑黑龙飞临「秘鲁之门」塞戈纳城，**黑焰扫平城墙战争兵器 + 烧融城门**，7 天之约压哨达成——**一小时攻破号称 67 年不落的塞戈纳城**（此前沙焰 17 万兵攻城铩羽）。第三次生擒 [赖顿·红须](../../人物/炎灵星群像.md)。莫蕾被同乐园「[微笑男](../../人物/炎灵星群像.md)」救走，见黑龙屠城彻底认怂。
- **生擒神女·吞茂生之神化身**（三十五卷25–28章）：查出秘鲁重要人物**神女**藏在平民中（赖顿·红须为拖时间不惜屠杀平民护她）→ 女军官 [温·艾尤儿](../../人物/炎灵星群像.md) 生擒之。苏晓故意激怒神女召来中立神灵 **[茂生之神](../../人物/炎灵星群像.md)** 的化身，**吞噬之核全开吞掉神灵化身**（青钢影升至 Lv.59）——神女被神抛弃、崩溃投诚。查明「[微笑男](../../人物/炎灵星群像.md) = 灰绅士」、违规者 [锯毒](../../人物/炎灵星群像.md) 跑去沙都竟想操控沙皇帝、被赤沙 + 乌诺王卫军教训、遭全国通缉。
- **门之战役·破秘鲁之门**（三十五卷29–33章）：沙都「隐秘支持者」增援 14 万精锐 → 苏晓指挥 **20 万大军**。与秘鲁总指挥 [斯普林·铁羊](../../人物/炎灵星群像.md) 的 30 万矮人（含 5 万雄羊骑兵王牌）决战，靠黑龙烧毁战争兵器 + 战争领主浴血奋战获胜——**秘鲁百年最惨大败**（矮人 19 万战死）、**秘鲁之门彻底洞开**。赖顿·红须第三次被擒（替兄斯普林·铁羊而死）。获 10572 **战功**、秘鲁声望 −197500「**永久死敌**」；触发「反攻」成就 + **称号兑换商店**（★~★★★★·轮回乐园的隐性支援）。
- **红姬暗杀失败·佩尼尔之环**（三十五卷33–34章）：秘鲁王请异族刺客 **[红姬](../../人物/炎灵星群像.md)**（金属丝线）夜袭，被苏晓反手斩杀（焚克遭其杀害）。开 30 余枚战争宝箱得 [蛮牛号角](../../力量体系/装备道具.md) + **圣灵级 [佩尼尔之环](../../力量体系/装备道具.md)**（需佩尼尔嫡系后裔·「将假王拖下王座」的伏笔）。

## 弑王史真相

由 [赖顿·红须](../../人物/炎灵星群像.md) 之口揭出（对应世界简介里「历史大片空白」）：原本的沙皇帝是 **[佩尼尔家族](../../人物/炎灵星群像.md)** 一脉，早被杀绝——**当今沙皇帝是「假王」**，其父与大祭法 [乌诺](../../人物/炎灵星群像.md) 是弑王叛徒（乌诺一介大祭法却能掌全国军权即源于此）；苏晓伪装的**勇颅族当年冲在弑王最前，后却被灭族**。半神大统帅 [尼波索](../../人物/炎灵星群像.md) 亲口承认「佩尼尔家族最后一位沙皇帝是我亲手处决」。秘鲁矮人敬重佩尼尔旧王、鄙夷假王。
