# 暗影世界 · 背叛者猎杀

## 千年血战 · 三霸并立的历史

⚠ 这世界的超凡力量强到「征兵门槛需觉醒超凡资质」——冷兵器时代的十几个王国 / 大领主互殴成**四王国**，四王国联合成**联盟**；北境凛冬之地的部族以血为盟组成**北境帝国**；居中制约的是稍弱的**圣兰王国**。联盟与北境帝国既争领土、又争资源、更有信仰冲突，**打打停停血战千年**（每逢两强杀红眼，就默契地一起揍圣兰王国一顿再接着打）。

⚠ 转折点是**魂鬼一族**从西部大沼泽的世界通道入侵（它们透支了原世界、迁来后自毁通道）——联盟与北境帝国趁「联手揍鬼族」的窗口紧锣密鼓谈成**和平条例**（鬼族太不抗揍、谈慢了前线就把它灭了）。鬼族被打到人口锐减九成，从此转向歌唱、艺术、冷兵器锻造。1367 年的铁律:只承认**四神教**（晨曦 / 太阳 / 黄金 / 黑暗），其余按邪教剿灭，且四神教**不得干政**。

## 联盟内部势力

- **议会院**:联盟权力中心，四位议员长把控，位于首都圣都。
- **猎手部队**:直属议会院的**隐秘组织**（以安保公司为掩护），负责各市的高危超凡案件——现任领袖 **泰莎**（老领袖培养的继承人）。
- **四神教**:晨曦 / 太阳 / 黄金 / 黑暗。⚠ 对苏晓的天然好感度:**太阳神教 +45 / 黑暗神教 −20（深渊倾向）/ 晨曦神教 −40**（因其阵营倾向 + 魅力属性）。
- **[黄昏疯人院](#)**:收容、矫正、感化穷凶极恶罪犯之地（联盟无死刑，故以此惩治罪大恶极者）。前身是与猎手部队同期建立、**对抗入侵古神的「猎人机构」**——后因四神教与陨灭星达成共识、古神不再入侵，「猎人机构」长期无本职工作，遂改建为后勤 / 医疗机构，几代发展成今天的黄昏疯人院。
