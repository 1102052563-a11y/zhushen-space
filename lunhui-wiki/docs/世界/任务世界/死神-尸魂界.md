# 死神 · 尸魂界（无害的科研人员）

    原著 txt 此卷标题误印为「**第二十九卷**」——实为「**第二十卷**」之误（全书卷号 1→54 严格连续、唯此处断裂；且「第二十九卷」另有正主「入侵天启·不死者狂潮」，属重号）。本 wiki 按正确顺序统一记为**第二十卷**（19 卷硝烟战场 → 20 卷死神 → 21 卷暗鸦纪元）。

## 死神世界 · 力量体系

- **死神**：灵子浓度极高的生命体（有肉体·身体由灵子构成·本质同人类只是更强）。与**虚**战斗以维系现世与尸魂界平衡——但其真正维系的是**尸魂界的统治权**。力量体系 = **灵压**（灵压越强、可调动灵子越多、战力越强）。
- **斩魂刀**：死神的武器，形态 / 能力多以死神自身灵魂为原型构建；死神知晓斩魂刀之名后逐渐开发其能力（**始解 → 卍解**）。
- **虚**：现世死者魂魄被内心欲望腐蚀幻化而成；与死神「只是不同的进化方式」。**破面** = 进化的虚。
- **灭却师**：死神的对立方（死神净化虚、灭却师消灭虚）。其理论「全灭虚也不危及现世」（尸魂界游民其实不转生·灭却理论无害）却因威胁死神统治叙事而被死神**大举进攻、险些灭族**。
- **苏晓视角**：死神体内**可吞噬的生物能量不多**（不到尾兽 1%）、但**纯度极高**（队长 / 副队长级最纯净·普通死神斑驳）——[吞噬之核](../../力量体系/技能/index.md) 的新猎物。

## 尸魂界 · 瀞灵廷 · 护廷十三队

尸魂界是个**巨大的封建社会**：中心 **瀞灵廷**（死神居所）+ 周边广阔 **流魂街**（魂魄游民·基础劳动力·不会转生）。瀞灵廷外围由**杀气石石墙**隔绝灵子（游民不敢近）。统治机构 = **护廷十三队**（一番队总队长统辖其余十二队·之上还有灵王宫 / 零番队 / 王室 / 中央四十六室）：

- 一番队（总队长·[山本元柳斎重国](../../人物/山本元柳斎重国.md)）/ 二番队（刑军·碎蜂）/ 三番队（支援）/ 四番队（医疗·[卯之花烈](../../人物/卯之花烈.md)）/ 五番队（救援·[蓝染惣右介](../../人物/蓝染惣右介.md)）/ 六番队（贵族专属·掌财·[朽木白哉](../../人物/朽木白哉.md)）/ 十番队（[日番谷冬狮郎](../../人物/日番谷冬狮郎.md) + 副队长松本乱菊）/ 十一番队（战斗·死伤最高·队长更木剑八）/ 十二番队（技术开发·涅茧利）。

## 钓鱼 · 潜入瀞灵廷（二十卷3–9章）

苏晓判断自己在死神世界**战力不算强**（朽木白哉 / 京乐春水级队长打不过、总队长山本更是被秒）——遂不硬闯、改走**「无害科研人员」**路线伪装潜入：

1. 贿赂白道门门卫**兕丹坊**（一枚 [世界政府纪念币](../../力量体系/装备道具/index.md)·绿色贵重品·混入灵子伪装成尸魂界产物）→ 钓出十番队队长 [日番谷冬狮郎](../../人物/日番谷冬狮郎.md)；
2. 抛出一瓶**混了半固态灵子的炼金药剂**→ 钓出四番队队长 [卯之花烈](../../人物/卯之花烈.md)（医疗队·初代剑八·心机极深）；
3. 把 [远古秘药](../../力量体系/消耗品材料/index.md) 改良成死神专用的 **[魂灵药](../../力量体系/消耗品材料/index.md)**（含灵子·恢复 + 永久增 0.2% 灵压·队长级是上限），靠它打动山本元柳斎——**加入四番队**（特殊编制·只受山本 / 卯之花烈管辖·实验室设在一番队驻地隔壁·激活瀞灵廷声望）。

苏晓的算盘：把魂灵药「先大量、后限量、最后少量」吊住死神们胃口换声望与话语权，伺机**夺斩魂刀**（斩魂刀 + 生物能量 + 宝箱 + 世界之源·杀死神收益远高于杀虚）。他全程伪装懒散、谦和、无害（与卯之花烈「面带笑容随时可能厮杀」的对视把松本乱菊吓出冷汗），暗里在一番队驻地研究**微型阿波罗炼金炸弹**（卯之花烈看破不说破）。

## 蓝染识破 · 镜花水月失效（二十卷10–11章）

「老好人」[蓝染惣右介](../../人物/蓝染惣右介.md)（同是尸魂界「起义军」·同类识同类）最先确定苏晓有问题——借带路之机用斩魂刀 **镜花水月**（完全催眠系）控制苏晓。但**苏晓的 [刀术宗师](../../力量体系/技能/index.md) 衍生效果 = 完全免疫精神 / 心灵系控制**，镜花水月对他根本无效。苏晓压制衍生效果、装作被催眠 10 分钟骗过蓝染——蓝染已察觉他的伪装、双方进入互相试探（蓝染下次再想催眠苏晓时会很尴尬）。苏晓盘算：加入蓝染方收益最高（杀队长后有蓝染背锅、自成一股势力·护廷十三队奈何不得他）。

## vs 死兆 · 死亡之钟结界（二十卷12–15章）

猎杀目标 [死兆](../../人物/死兆.md)（违规者 10073·话痨逗逼·身体素质流）主动找上门、用「世界阻断」造**死亡之钟结界**（20 分钟倒计时·世界炭化）单挑苏晓。苏晓解开压制血气的金属环镣应战——死兆**力量 / 速度强于苏晓、技巧却远不及刀术宗师**（三回合挨三下狠的）；其武器 **沙耶诺德**（来自虚空·可变任意形态·吸血·**一眼认出苏晓是灭法之影「魂影 / 断魂影」**·劝死兆快逃）。死兆瞳孔血芒、以自身鲜血催力（双刺剑吸血形态·中一刀被吸 20% 血）、组合技伤了苏晓一刀——但结界倒计时结束、**苏晓判胜**，从死兆储存空间抽走金色特殊装备 **[无尽黑暗](../../力量体系/装备道具/index.md)**（死兆气哭）。死兆逃走、扬言「下次绝不这么狼狈、这次当打招呼、我感觉要升级了」。苏晓对会说话的虚空武器沙耶诺德更感兴趣。

## 老阴哔结盟 · 入蓝染阵营（二十卷16–18章）

死兆袭击后苏晓与 [蓝染](../../人物/蓝染惣右介.md) 再碰面、镜花水月已对苏晓失效，气氛尴尬到极致——苏晓索性**直接挑明**「我的目的是斩魂刀、你的目的是推翻尸魂界统治、并不冲突、不如合作」（与老阴哔交涉的最好办法就是把事挑明、能合作就合作、不能就鱼死网破）。他抛阿波罗 + 展示**纯度 500 的灵子球**（远超蓝染收的纯度 3）+ 自曝魂灵药弊端（最多饮 15 瓶到顶·有副作用），又**虚构一种「制约 X 契约」念能力**（拿斩龙闪演示·险些说漏「念能力」），逼蓝染签下**轮回乐园公证的双向契约**（违约者弹指化灰）——蓝染权衡「百年计划付之一炬 vs 合作」后**签约**。苏晓主线第一环完成、**阵营变更为「蓝染惣右介阵营」**（蓝染独立前仍可用瀞灵廷声望）。三番队队长 [市丸银](../../人物/市丸银.md) 是同党。

## 崩玉计划 · 蓝染假死（二十卷23–26章）

三人裏廷间密会，蓝染和盘托出**崩玉计划**：崩玉是「简易版许愿机」（按使用者渴望提升实力）、由浦原喜助与蓝染各造出一半成品，浦原把崩玉封印在流魂街女婴**露琪亚**体内——蓝染遂一手安排露琪亚入现世遇黑崎一护 / 被捕、**杀光中央四十六室 + 镜花水月催眠**、判露琪亚死刑，只为夺其体内崩玉融合成完整崩玉。苏晓接主线第二环**「崩玉之战」**（协助蓝染夺崩玉·难度 Lv.38·奖 3 真实属性点 + 传说级宝箱）。计划高潮 = **蓝染假死**（镜花水月把一具秃头死神尸体伪装成蓝染钉上东大圣壁·[市丸银](../../人物/市丸银.md) 扮凶手·苏晓扮好友调查·留信嫁祸冬狮郎 / 害 [雏森桃](../../人物/雏森桃.md)）——苏晓演技爆裂、又顺势拦下提刀冲杀市丸银的雏森桃（保她以利布局）。蓝染假死 = 黑崎一护已闯瀞灵廷、苏晓潜伏到头、**该收获了**。

## 开猎夺刀 · 兕丹坊 / 狛村左阵（二十卷26–30章）

借「蓝染遇害」的混乱与合作者身份，苏晓正式开猎（蓝染替他掩饰痕迹·除几个关键人物外随便杀）：

- **斩兕丹坊**（白道门守卫）：一道刀芒枭首——得**世界之源 0.8%**（入世界以来首次）+ [噬灵者](../../力量体系/天赋.md) 触发永久 +9 法力（上限 6085）+ 暗紫宝箱。再几刀劈开杀气石石墙离开瀞灵廷。
- **诱杀狛村左阵**（七番队队长·狼头）：[布布汪](../../人物/布布汪.md) 扮大老鼠衔「蓝染同款破眼镜」把谨慎的他单独引出、召唤死神 [影](../../人物/影-左宗卫.md)（其旧识左宗卫）诱敌、苏晓现身亮明身份。见识「死神刀术素养」之差（狛村格挡如挥棒），被压制后逼出卍解黑绳天谴明王——苏晓掏 [寂灭公爵](../../力量体系/装备道具/index.md) 三枪打崩（天谴明王与本体生命链接）、吞其斩魄刀再枭首，**得世界之源 12.9%（累计 13.7%）+ 法力上限 6125 + 传说级宝箱**。

## 偷刀 · 审问 · 灵子脉冲推平瀞灵廷（二十卷31–38章）

杀狛村后苏晓玩得更花——让 [影](../../人物/影-左宗卫.md) 伪装「左宗卫复活」背锅、自己靠微型播音器 + 全息投影仪伪造不在场证明；看穿监视他的 [南雀](../../人物/南雀.md)（[碎蜂](../../人物/碎蜂.md) 的暗杀部队卧底）却不杀、反喂追踪器把她变成最好的不在场证人。改走更稳的**偷刀**路线——[布布汪](../../人物/布布汪.md) 潜入十二番队偷走 [涅茧利](../../人物/涅茧利.md) 的斩魄刀（测出死神与斩魄刀感知距离 2.3 公里）、又偷浮竹十四郎的刀。涅茧利等遂力主**秘密审问**苏晓（试探其反应）——但苏晓早联合**朽木家 / 四枫院家 / 纲弥代家**三大贵族（每天暗造 200 颗灵子球·明面只交 100）、瀞灵廷动他不得。

识破苏晓的 [卯之花烈](../../人物/卯之花烈.md) 索性**狗急跳墙**、拉朽木白哉 / 冬狮郎 / 碎蜂四队长无视瀞灵廷规则联手伏杀（割 [雏森桃](../../人物/雏森桃.md) 喉当诱饵）。苏晓提前激发埋了多日的杀招——**灵子脉冲**（7659 颗灵子球 + 以自己为中枢的大型炼金阵图·对人类几乎 0 伤害、对死神 / 建筑极强）：直接**推平一番队驻地**（波及三 / 五番队）。山本元柳斎硬扛流刃若火火焰冲入、却因不敢在高纯度灵子区卍解（会夷平小半个尸魂界）只能逼退苏晓——苏晓用机械妹造的反击盾逃脱。卯之花烈其实是瀞灵廷功臣（再拖 3 天苏晓就能引爆「灵子大爆炸」）。

**战果**：世界之源暴涨到 **86.5%**、收宝箱无数（蓝 42 / 紫 26 / 暗紫 11 / 淡金 6 / 金 4…）；代价是瀞灵廷声望 **−275800（死仇·被通缉）**——猎杀者基础权限激活、本次声望惩罚被豁免（不会沦为其他契约者目标、可续执行猎杀）。

## 吞刀涨锋刃 ·「刃之觉醒」（二十卷39章）

苏晓发现至尊锋刃能吞噬死神「未公证的斩魄刀」涨锋刃值（详见 [斩龙闪](../../力量体系/装备道具/index.md)）——布布汪 / 贝妮捡来几百把斩魄刀，斩龙闪连吞近 300 把、**锋刃值冲到 100% 却不晋级**、品质改显「**刃之觉醒中**」（评分 530++）。苏晓感到 [马文·华尔兹](../../人物/马文·华尔兹.md) 跨维度呼唤、须返园后请教如何渡过「刃之觉醒」。

## 决战违规者死兆（二十卷41–44章·进行中）
