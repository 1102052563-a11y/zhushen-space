# 陨灭星

> ⚠ **「一切的开始与终结」——世界简介只有这一句**;⚠ **⚠ 世界难度 Lv.89～Lv.105 · 世界之源 0%〔无法获取〕**——⚠ **超脱之界不产世界之源，取而代之的是「世界声望」〔可在虚空之树公证的世界商店消费，并按总数计入返回乐园后的世界结算〕**;⚠ **⚠ 轮回乐园的入界警告:陨灭星为至强梯阶世界中最危险梯队的世界，需慎重考虑是否进入**;⚠ **⚠ 这次的世界简介信息量极少——并非此界没有历史事件，而是此界生灵的痛苦多到无以计数**。

## 🌑 环境 —— 活着本身就已费尽全力

⚠ **阴沉昏黄的天空·燥热的气温·土黄色的大地寸草不生·溪流淌着有剧毒的浑水·空气弥散沉闷气味且带有毒性**——⚠ **陨灭星很少有上午与中午，更多是下午日落时分与清晨的暗淡光芒**;⚠ **⚠ 荒凉大地上最常见的不是石头，而是隔三差五就有的骸骨堆**——⚠ **被堆砌成「丰」字形，骨上还粘连着少量干枯暗红血肉，邪异又威严**;⚠ **⚠ 想活下去必须信仰古神**:⚠ **成年后抽出自己的一根肋骨献给某位古神、刻上神之尊名、用粗糙线绳或动物筋线挂在脖颈上**——⚠ **祈祷时肋骨上的古神语神名会透出白色微光**（⚠ 否则有毒的空气、剧毒的水、以及随处可见能通过多种手段传染的疯狂/失控/理智吞没，都足以夺走性命）。

- ⚠ **⚠ 土著的「众生平等」**:⚠ **人族、蜥族〔蜥蜴般的人形〕、异族之间完全不存在种族矛盾——只要是生灵，就会在可能的情况下尽量互相伸出援手**——⚠ **因为在这里活着与繁衍已费尽全力，真的没有其他力气去进行种族纷争或利益争夺**;⚠ **⚠ 古神们对所有智慧生灵保持的「众生平等」标准是**:⚠ **只要不把这些「牲畜」搞灭绝，那就都没问题**（⚠ 哪怕土著即将灭亡也没关系:四原罪中的**色孽·茵娜**与**狂食·魁克**都能让人口在几年内疯长——这也是本世界智慧生灵至今还没灭亡的主要原因）。
