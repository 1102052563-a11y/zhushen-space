# Fate/stay night · 第五次圣杯战争（苏晓第 22 个世界）

[苏晓](../../人物/苏晓.md) 的**第 22 个衍生世界**——买「珍藏的纪念品」世界钥匙、与 [血门冒险团](../../势力/index.md) 同步进入 Fate/stay night 第五次圣杯战争（撞同世界几率 60%+），主动追上多年死敌、清算旧账。落点冬木市，入场被判定为 **Saber（剑之骑士）**、虚假契约者编号 **13013 号**（战斗力极高·优先度 27 压过契合度），借平凡少年 [卫宫士郎](../../人物/卫宫士郎.md) 当降临坐标。

!!! tip "圣杯战争 · 世界观（二十二卷1–4章）"

    - **圣杯战争**：传说圣杯能实现拥有者愿望，7 位魔术师各召一名英灵争夺圣杯。冬木市为舞台、圣堂教会派代行者监督、魔术协会（时钟塔）主导。
    - **魔术师**：力量看体内**魔术回路**数量（天生固定·靠世代血统积累）。三大魔术家族 = 远坂 / 间桐 / 爱因兹贝伦。
    - **英灵 / 从者**：成信仰对象的「英雄」死后留为传说、可被召为从者，分七职阶（Saber / Archer / Lancer / Rider / Caster / Berserker / Assassin）。**苏晓非能量体、却被判定归为「英灵」**——故主线任务格外简洁（仅「存活」）。
    - **令咒**：御主控制从者的三道标记（由圣痕演化）。苏晓凭老神父 [言峰璃正](../../人物/言峰绮礼.md) 留下的特殊十字架，可**强行剥离 / 赋予令咒**（把 [卫宫士郎](../../人物/卫宫士郎.md) 换成 [远坂凛](../../人物/远坂凛.md) 当 Master）。
    - **「圣杯」真相**：第三次圣杯战争后圣杯已质变、不再是许愿机——被侵浊的小圣杯盛满即开「天之孔」喷污泥（可焚毁数个冬木市）。卫宫切嗣当年察觉此点、令呆毛王破坏圣杯。

## 任务结构

- **主线任务·英灵之战**（Lv.37）：存活至圣杯战争结束·期限 11 天·奖 2 真实属性点（因苏晓「英灵」身份 + 与大型冒险团同入·平衡机制给了极简任务）。
- **猎杀任务·清理 10777 号违规者**：见 [10777 号违规者](../../人物/违规者-10777号.md)（天启乐园裁决者融合体·奖黄金荣誉勋章 ×7）。
- **苏晓的真实目的**：与 [血门冒险团](../../势力/index.md) 清算旧账（最优先）> 清违规者 > 主线（顺势）。

## 三击肃清队 → 钟楼狙击战（二十二卷5–15章）

血门派「肃清队」清场（保人造圣杯计划），苏晓 4 打 39（[阿姆](../../人物/阿姆.md) 冰墙分割战场 + [布布汪](../../人物/布布汪.md) / [贝妮](../../人物/狩之影.md) 骚扰 + 自己刀刀破盾）全歼，斩主坦 [铁鼠](../../人物/铁鼠.md)。随后玩「布布汪潜伏报坐标 + 自己 3.5 公里外寂灭公爵狙击」战术昼夜蚕食血门（[斯坦](../../人物/斯坦.md) 亲挨一枪靠恶魔之印甲片硬扛、部下死伤逾 20）。双方在联络平台互相问候恶心。

## 换 Master · 强化远坂凛（二十二卷11–21章）

苏晓嫌 [卫宫士郎](../../人物/卫宫士郎.md)「正义的伙伴」理念不合、把他换回 [英灵卫宫](../../人物/卫宫士郎.md)（Archer·未来的他自己）的御主，自己改认 [远坂凛](../../人物/远坂凛.md) 为 Master，还在她背上**刻画「第三星辉炼金阵图」**（魔力存储从 20 升到 600+·当炮台）。途中揭穿她杀父仇人 = [言峰绮礼](../../人物/言峰绮礼.md)（用对师宝具背刺远坂时臣）。

## 决战 · 斩斯坦 · 团灭血门（二十二卷22–31章）

苏晓推断血门团队主线任务 = 找 [言峰绮礼](../../人物/言峰绮礼.md)（监守自盗的监督方），遂借本土势力赛跑：胁迫 [伊莉雅](../../人物/伊莉雅丝菲尔.md)（控制小圣杯 + 喂生物炸弹）交出电报机、让远坂凛向时钟塔举报言峰绮礼「监守自盗」，圣堂教会派咸鱼司机 [神父道恩](../../人物/圣杯世界配角.md) 来追踪。血门用杀手 [死刑](../../人物/圣杯世界配角.md) 放血强化狙杀苏晓（被龙影闪躲过）。最终苏晓靠 [布布汪](../../人物/布布汪.md) + [狼族传家宝](../../力量体系/装备道具.md) 接力赶在斯坦前一步刺杀言峰绮礼——**血门团队主线任务失败**、76 名团员被强行处决 / 退团。残局二打一：苏晓开猎魔时刻、用「[绝影闪](../../力量体系/技能.md) ×2 + [魔刃](../../力量体系/装备道具.md) 即死斩杀」**三刀斩杀团长 [斯坦](../../人物/斯坦.md)**（破其暗金属领域 + 恶魔之印甲片）；[奈洛伊](../../人物/莺澜.md) 断臂逃走。苏晓获临时称号 [团队屠戮者](../../力量体系/称号.md)。

## 通关 · 圣杯未召（二十二卷32–39章）

清算完血门，苏晓没找到隐于幕后的最后一名御主 [间桐樱](../../人物/间桐樱.md)（Rider·美杜莎御主·被时钟塔护徒老魔术师 + 旧识 [假面](../../人物/假面.md) 带离冬木市保命）。[金闪闪](../../人物/吉尔伽美什.md) 被近 20 名独狼围阴致死、[卫宫士郎](../../人物/卫宫士郎.md) 被苏晓顺手清掉（开出史诗级宝箱）。圣杯战争未彻底结束、天之孔被召出，苏晓进「星空之遗」等结算。
