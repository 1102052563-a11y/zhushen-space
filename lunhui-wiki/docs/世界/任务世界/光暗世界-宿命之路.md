# 光·暗世界（第34卷·光与暗·宿命之路·七阶·超凡蒸汽时代）

[苏晓](../../人物/苏晓.md) 用来**突破 150 点属性壁障**（晋七阶前置）的世界。因他选择四属性同破（难度 190%）+ 自身战力，壁障突破难度被硬顶到**七阶**——故进入这个全开放原生世界完成考核。

!!! tip "为何是七阶世界 + 主线即壁障考核"
    单属性突破壁障难度 100%、每多一种主属性 +20%（超两种每种再 +10%）→ 苏晓四属性同破 = **190%**，加战力被判进七阶世界。**主线任务最后一环就是壁障突破考核**；本世界会遇七阶契约者（含一名 17 天前晋七阶、来执行特殊任务的己方契约者·苏晓不可主动伤害）。苏晓缴 50 枚灵魂钱币换高初始身份 = **炼金学大师·库库林·白夜**。

## 世界背景 · 超凡蒸汽时代

- **历史**：奥卡兹大帝统一大陆、废 17 种文字 / 立议会制（光·暗纪元 392 年去世）→ 四嫡子裂土建**四王国**（雷曼等·彼此世仇）→ **秘术师**时代（秘纹来自上个文明的地下墓穴·一度沦为战争武器·后王权束缚使秘纹排斥灵魂）→ 四王国 + 秘术师成立 **议会院**（权力最中心·制约王权）→ **炼金学**（来源未知·疑来自虚空·门槛极高）+ **超凡蒸汽技术**（现今主流·蒸汽火车 / 蒸汽车）。
- **三大势力 + 鬣狼**：**四王国**（各有国王）/ **议会院**（8 议员·集体高于国王·主政贸易中枢 [安戈城](../../人物/光暗世界群像.md)）/ **秘术师**（秘纹系世袭 / 烙刻系刻身·需「自由灵魂」）；**鬣狼族**（人 / 兽可切换的夜行族·能与人婚配生纯血·专猎四王国 + 议会院高层·始终无法根除）。
- **星光铁**：本世界特有稀缺资源。

## 进程（按章）

- **蒸汽火车劫持 + 抵安戈城**（三十四卷1–6章）：苏晓（炼金大师身份·乘蒸汽火车赴议会院）遭**鬣狼族劫持**——「月光大人」派的鬣狼 [卡律](../../人物/光暗世界群像.md) + 通缉犯 [野兽绅士·图亚克](../../人物/光暗世界群像.md)（被奇多尔曼公司逼退）；护送的十几名新晋秘术师弱得离谱。苏晓识破这是**有人借鬣狼之手控制他、好害死老议员**的局，用 [魂之轻语](../../力量体系/装备道具/index.md) 秒鬣狼、伪造「营救者」痕迹藏战力。同车 [南希·埃尔娜](../../人物/光暗世界群像.md)（迷糊却能打的雷曼三等兵）暗中护送。
- **议会院 + 老议员病危**（三十四卷6–9章）：[阿撒兹](../../人物/光暗世界群像.md)（老议员心腹）引苏晓见**代理议员 [普里](../../人物/光暗世界群像.md)**（不正经爱装死·圣境老议员之子）——请苏晓救其病危的父亲 [圣境老议员](../../人物/光暗世界群像.md)（179 岁·猎杀鬣狼最多·三实权议员之一）。苏晓查出老议员**非病、是被「微型超凡生物」侵体吸生命力所害**（与水 100% 兼容·极难清除）——锁定幕后是议会院内与鬣狼勾结之人。暗处还有**等奥卡兹大帝死了千年、要夺回大陆的黑触手黑袍人**。

- **斩内奸 + 格鲁灭四王国**（三十四卷10–13章）：苏晓造「伪生心脏过滤器」治老议员，用排除法揪出内奸——老议员的女仆 [瑞丝](../../人物/光暗世界群像.md)（真身「骨蜘蛛·卡罗」炼金师·潜伏 24 年双重人格·几年前主意识苏醒、用微型超凡生物在老议员心脏植可远程操控的核）当场斩杀，掐断幕后关键一环。幕后 = **[埃尔维斯·格鲁](../../人物/光暗世界群像.md)**（即黑袍人）——被苏晓逼得狗急跳墙，用石板裂空间放「黑色光球」**三日内灭掉东泽 / 月溪 / 风暴三王国**（平民 / 秘术师变「干枯怪物」·秘术师秘纹被剥离）。四王国仅 [雷曼王](../../人物/光暗世界群像.md) 活、[安戈城](../../人物/光暗世界群像.md) 成唯一未陷之城（99 万士兵 + 平民涌入）。
- **存亡之战 + 探厄运镇**（三十四卷13–20章）：主线从第一环（237% 超额）直跳**第四环「存亡之战」**（保安戈城 10 天不毁·10 天后光暗对撞开间隙）。议会院（[霜蛇](../../人物/光暗世界群像.md) / [蔷薇](../../人物/光暗世界群像.md) 议员 + 雷曼王）开奥卡兹大帝建的**地城**（容百万·安戈城曾是光暗帝国首都）、定「主动迎击」战略、蒸汽被打回冷兵器时代。苏晓查明真相：一切源于**厄运镇**里一位远超七阶、却比月神还安分的**古神**——格鲁借其力灭国。苏晓以老议员头发为坐标**焚灵魂灰烬开「间隙」潜入厄运镇**（捡到史诗级 [遗忘之环](../../力量体系/装备道具/index.md)·遇守门领主 **[不朽之人](../../人物/光暗世界群像.md)**·靠 [神裁](../../力量体系/装备道具/index.md) 抑古神侵蚀撤回），得知镇中部有唯一性的 **[灰烬熔炉](../../力量体系/装备道具/index.md)**（重锻装备·需 260 克星光铁·带出可免壁障失败惩罚）。
- **格鲁谈判 + 木眷族攻城**（三十四卷20–22章）：格鲁派干枯怪物点名找苏晓谈判、要交出漏网秘术师（阳谋挑拨），被苏晓放逐刺穿拒绝；格鲁与**鬣狼族**合作（安戈城灭则大陆归鬣狼）、**送 120 万老幼妇孺给安戈城**耗其资源。**[木眷族](../../世界/生物万族/index.md)**（古神之力侵蚀重生的「干枯怪物」·可吞同类成长、成长后有理智会说人话）大军压境——苏晓奔赴最前线守城。

- **守城 + 秒杀格鲁肉体 + 青影王真身**（三十四卷23–27章）：苏晓上最前线杀木眷族只是**演戏麻痹格鲁**——真正的杀招是让布布汪找踪迹、贝妮追踪，锁定格鲁藏身月溪王宫后，靠贝妮的 [庇护宝石](../../力量体系/装备道具/index.md) 开空间裂缝、全身影燃 + 猎魔时刻**穿三层一刀秒了格鲁肉体**（格鲁只死 1/3·灵魂被困厄运镇·木眷族失「被污染的世界之种」加持而溃败）。揭：格鲁灭国靠的是奥卡兹大帝当年封在安戈城的「被污染的世界之种」。返园苏晓**彻底掌握 [青影王](../../力量体系/技能/index.md)**（影燃只是体验卡·真身 = 疾影 + 灭法之刃）。
- **进厄运镇 + 大主教真相**（三十四卷28–38章）：主线第四环完成得**光暗护符 ×5**（免古神侵蚀）→ 开永久间隙进厄运镇（第五环 = 探索 20%）。斩四人小队（天启 [白牛](../../人物/光暗世界群像.md) / 队长·死亡乐园 [杀人蜂](../../人物/光暗世界群像.md) 携召唤物俄伊司深入）、用**灭法之刃 0.8 秒斩杀守门 [不朽之人](../../人物/光暗世界群像.md)** 得救赎教堂钥匙 + 猩红卡 ★7。进救赎教堂（唯一安全区）唤醒 [圣女](../../人物/光暗世界群像.md)——**从圣女 + 守炉人 [齐斯尔曼](../../人物/光暗世界群像.md) 处揭开厄运镇全部真相**：真正幕后是教堂最深处的 **[大主教](../../人物/光暗世界群像.md)**（千年布局·奥卡兹大帝 / 格鲁只是棋子·想成为新古神），而苏晓无意间**顶替了原定「救赎者」[阿撒兹](../../人物/光暗世界群像.md)（老议员心腹·古神之力主角）的剧本**（杀格鲁得石板→杀不朽之人得钥匙→取灰烬熔炉→战怨愤孤儿→唤大主教）。

- **联手屠 boss + 通关 + 突破 150 壁障**（三十四卷39–52章）：苏晓靠「甩锅大主教」把守炉人 [齐斯尔曼](../../人物/光暗世界群像.md) + [怨愤孤儿](../../人物/光暗世界群像.md) 忽悠成帮手（先斩了隐患杀人蜂），**三对一斩杀大主教**（齐斯 + 怨愤扯出其灵魂 → 苏晓刃道刀斩喉 → 众从者扯下头颅）；再**背刺斩齐斯尔曼**（猎魔时刻 4 次龙影闪躲其「黑暗埋葬」）、用 [冥神脐带](../../力量体系/装备道具/index.md) 的「污秽之呓语」弄死全盛复活的怨愤孤儿（炼金销毁防引冥神）。得**灰烬熔炉 + 纯净世界之种**（完成主线）；找最弱的**幽蛇姐妹**拿神殿秘纹、进光暗神殿焚古神驱壳（内困格鲁灵魂·被大主教骗当诱饵·苏晓 D·暗杀给其体面）——**完成 150 属性壁障考核**。铁匠用灰烬熔炉把黑王套装重锻成一件护臂。**S− 通关**，返园用 43 属性点把四属性拉满 150、**晋七阶前置全达成**。详见下方 🏁通关结算快照。
