[苏晓](../../人物/苏晓.md) 的**九阶晋升考核世界**——他自行选定的起源级封顶难度世界，也是他与「死寂」了断因果的地方。此地即 **[黑王座大陆·死寂城](黑王座大陆-死寂城.md)（第26卷）** 的再入:苏晓一直随身的「**黑王护臂 / 黑·王之轮回**」（圣灵级套装·唯一）正源自卷26 那座死寂城，用得越久、与死寂的因果越深，如今光用「死寂降临」就会招来死之民对视——晋升九阶前必须回来做个了断，否则今后只会付出更惨痛的代价。⚠ 死寂城是 [苏晓](../../人物/苏晓.md) 散布「死寂城产黑枫树种」假消息、卖【枯声大剑】碎片当门票，故意引来众多八阶强者一同分担风险的目标地。

## 高墙城 · 灾难史

千年前古神庇护的时代终结，**死寂**蔓延到每个角落、死之民步步紧逼。灾难中一种名为「**圣痕**」的力量被发现，治愈教会以此立教、建起百米高、刻满圣痕的**高墙城**，生者才得以苟延残喘。此后蒸汽神教、瓦迪家族相继兴起，高墙议会成立——⚠ 关键转折是「**永生之神**」（古老神灵）的庇护驱走了城内死寂，土地重新肥沃、淡水甘甜、果实丰盈，高墙城人口从灾难时的十几万恢复到如今**超千万**。三天后将迎来一年最盛大的「**神祭日**」（祭奠永生之神）。⚠ 在高墙城，可以不信任何势力，但**绝不能对永生之神有半点不敬**（曾有人自称神使，被四方势力联手当晚送上火刑架）。

## 四大势力

- **治愈教会**:主掌神系（圣痕之力·治愈灾祸·信奉古老神灵、重度敌对古神）——四大势力中最低调也最狠。下辖三机构:**工坊**（对外武力 + 武器开发）、**学术派**（肉体 / 心理治疗 + 圣痕研究 + 育才）、**治疗院**（「根治」= 处决冥顽罪徒·苏晓所在）。顶层为大主教·卢索、圣祭祀·安托莎（常年不出大教堂·心机深沉）。
- **蒸汽神教**:「血肉苦弱，机械飞升」——城内一切科技造物（蒸汽列车 / 电力 / 黑科技械体）的来源，最高综合战力。领袖 [机械公爵](../../人物/幽暗大陆/机械公爵.md)。旗下「怒锤机构」一直想取代治疗院处理超凡案件。
- **瓦迪家族**:农耕 / 畜牧技术的掌握者——没它城内五分之一的人吃不上饭。综合战力最末但贡献巨大。
- **高墙议会**:高墙城官方机构（税收 / 财政 / 民生）·掌宪兵队与治安队。
