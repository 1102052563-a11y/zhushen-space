# 画中世界（第44卷）

[苏晓](../../人物/苏晓.md) 的第 4 个八阶世界——一个**极特殊**的世界。进入前需通过全属性判定，场上有奥术永恒星、魔鬼族、恶魔族、陨灭星、天启乐园、羽族、圣光乐园等多方阵营。世界简介只有一句反复的告诫:「**保持理智，这很重要。**」苏晓因灭虫任务奖励攒够 2 块 [画卷残片](../../力量体系/装备道具.md) 而获入场资格，实为**代表轮回乐园出战的一场小型世界争夺战**。

## 世界结构 · 主画与里画

画中世界是「**真实存在、却濒临消亡的世界**」——它的中枢是一幅【世界画】，画每被撕下一块，世界就消失一部分、被黑紫色的黑暗混沌半流体填充。如今**主画世界**只剩一栋两层古堡（外面全是奔流的混沌流体）。古堡会客厅墙上挂着**四幅画 = 四个「里画世界」**（每幅是一种风格迥异的世界，[画卷残片](../../力量体系/装备道具.md) 藏在里画世界中）:

- **噩梦画（噩梦之地）**：哥特黑暗风·由 [噩梦之王](../生物万族/index.md) 主宰（详见下）。
- **沙漠画**：一片沙漠 + 太阳，可进入。
- **另两幅**：被细密锁链缠死、暂不可进。

## 画卷争夺战 · 规则

- **入场与阵营**：各阵营须先向虚空之树出资、出资最多的前七阵营获首批入场资格。**首批七阵营**:轮回乐园（[苏晓](../../人物/苏晓.md)）、奥术永恒星（[炎启·索耶格](../../人物/画中世界/炎启·索耶格.md) + [洛希](../../人物/画中世界/洛希.md)）、魔鬼族（[沃波·伍德](../../人物/画中世界/沃波·伍德.md)）、恶魔族（[莉莉姆](../../人物/莉莉姆.md)）、陨灭星（[罪亚斯](../../人物/画中世界/罪亚斯.md)）、天启乐园（[莫雷](../../人物/原始大陆/莫雷.md) + [月使徒](../../人物/苍龙大陆/月使徒.md)）、羽族（[天羽](../../人物/画中世界/天羽.md)）。某阵营参战者全灭 / 脱离即淘汰，圣光乐园 / 星族 / 死亡乐园等在星空中转站候补。
- **胜负与排行**：向 [大小姐](../../人物/画中世界/大小姐.md)（世界中枢的白发少女）上交画卷残片数量最多者胜；可**击杀他人篡夺其已上交的残片数**。胜者独得，败者「食屁」。前五名奖励由各阵营出资经虚空之树折算——**首位:[起源石·世界（1/5）](../../力量体系/装备道具.md)**（天启乐园出·苏晓已有一块）、第二黑枫树枝干、第三传承结晶（海洋圣者）、第四神之诅咒卷轴、第五灵魂晶核 ×20。
- **观战 · 直播**：虚空台西环**莫乌斗技场**十余万观众通过洛希保管的**【洞察眼】**看实时影像（观战席有奥术永恒星凛风王 / [风王子](../../人物/风王子.md)、羽族殇羽等）。

## 理智值（本世界独有属性）

进入即激活「理智值」——苏晓初始 100，加意志力 +35、灵魂强度 +107、刀术宗师豁免精神控制 +88 = **330**（多数契约者到此精神控制抗性为负、他因刀术宗师部分保留）。被特殊单位攻击、受惊、恐惧、被精神控制、探索特殊区域都会掉理智值；**低于 1 点即被疯狂侵蚀、永远留在此地（死）**。理智值随时间缓恢复或靠本世界物品恢复。

## 噩梦之地 · 生存游戏

第一个里画世界。[噩梦之王](../生物万族/index.md)（无聊的地盘主宰）设「**宰杀场**」生存游戏:进入者以**化身**（可死三次·带至多两个从者）探索，分**猎命人（追猎方）**与**生存者（逃生方）**。

- **苏晓被强制转职「猎命人」**（本无收益·经轮回乐园 + 虚空之树双重修正为:12 小时内不让 8 名生存者逃出宰杀场即胜、得 4 块画卷残片）:穿猎命人套装（力/体 +25、生命 +8500、防御 +39）碾压生存者，但转弯速度 −50%（锁链惯性）；追杀太久会被猎斧 / 面具反馈的「快意」削理智（低于 50% 即被同化）。持猎斧、布锯齿捕兽夹。
- **生存者**（莫雷 / 月使徒 / 莉莉姆 / 天羽 / 洛希 / 炎启·索耶格 / 沃波·伍德等）:被赋予「装死」高阶隐匿（不在视线内躺平即隐身、但不能动）。宰杀场有 5 处锁盘、校正 4 处开门（苏晓可打乱、可感知首次校正）。
- **战况**：苏晓斩了 [炎启·索耶格](../../人物/画中世界/炎启·索耶格.md)、逮住并吊起 [莫雷](../../人物/原始大陆/莫雷.md) / [月使徒](../../人物/苍龙大陆/月使徒.md)（「膀胱姐妹花」喝多生命泉水尿崩被擒）/ [莉莉姆](../../人物/莉莉姆.md)；魔鬼族 [沃波·伍德](../../人物/画中世界/沃波·伍德.md)（骗术师·骗走上任猎命人的套装、当「背叛者」）与陨灭星 [罪亚斯](../../人物/画中世界/罪亚斯.md) 主动找苏晓**结成三方临时联盟**（先清场 → 灭噩梦之王 → 最后三选一分胜负），[天羽](../../人物/画中世界/天羽.md) 跟踪被逮、遭罪亚斯挖眼吃。

## 堵门赢生存游戏 · 深渊之罐 · 厄梦镇决战（44卷13–24章）

- **堵初生广场 · 赢生存游戏**：三方联盟把生存者逐一擒获吊起（[洛希](../../人物/画中世界/洛希.md) 被追杀三小时又中捕兽夹「千层饼」局、全员被丢进初生广场），12 小时到 [苏晓](../../人物/苏晓.md) 得胜、获 4 块画卷残片——[噩梦之王](../生物万族/index.md) 却「玩不起」，按自定义「对等」把 4 块残片换成 4 颗灵魂结晶碎片、还关掉第二轮游戏。苏晓、[沃波·伍德](../../人物/画中世界/沃波·伍德.md)、[罪亚斯](../../人物/画中世界/罪亚斯.md) 遂凭身躯汇聚的血印**诈死回本体**、再冲回初生广场硬闯厄梦镇。

- **厄梦镇决战（进行中）**：厄梦镇是噩梦之王的领域之源（借它放大「恐怖」领域、放无限**黑犬**）。苏晓一颗 [烈阳之怒·阿波罗](../../力量体系/装备道具.md) 夷平厄梦镇、废掉其领域。噩梦之王（持长柄钉锤、披 [冥铠](../../力量体系/装备道具.md)）与前来夺画卷残片的 [大骑士](../../人物/画中世界/大骑士.md) 死战，伍德、罪亚斯加入围攻，苏晓则架起 [J·恶魔](../../力量体系/装备道具.md) 狙击炮远程助攻——决战胶着（未完）。

## 无尽沙漠的血魂 · 茂生之狂乱降临（44卷36–43章）

- **无尽沙漠的防御机制**：沙之世界的入口是一片走不出去的**无尽沙漠**（其实是沙之世界与主画世界间的独立缓冲空间、也是噩梦之王缝合噩梦世界的灵感来源）。这里游荡着无形的**魂**——魂一遇生灵就跟上几分钟、脱离后化为那人的**心灵野兽**（仿制品、不代表本人已兽化）、杀掉入侵者后再退化成更强的魂。这是土著对「灭世的外来者」的绝杀:不战胜自己的心灵野兽，就走不出无尽沙漠。
- **血魂·暗魔之影**：苏晓的魂本该化成他的心灵野兽，却因当时他正与洛希放血气而**吸血气蜕变成「血魂」**。血魂一现身就吞了伍德的「镰刀死神」、罪亚斯的「触手男」两个心灵野兽（三合体），又吞掉莫雷 / 月使徒 / 莉莉姆三人投影的合体（六合体）——**补齐了苏晓的负魅力弱点**（月使徒 / 莉莉姆带魅力抗性），集齐苏晓的空间穿透 / 技法（无冷却版）、罪亚斯的不灭、伍德的适应、莫雷的精密控制、月使徒的召唤于一身，还在战斗中**觉醒智慧、心灵映射肉体、越打越强**。
- **围杀血魂**：硬拼必亏（赢了也没资源夺画卷残片），苏晓布**炼金阵图陷阱**（凝合性结晶空间阻断 + 烈阳汇聚成八米烈阳柱），众人押上不朽 / 圣灵级装备当一次性道具，由 [莫雷](../../人物/原始大陆/莫雷.md) / [月使徒](../../人物/苍龙大陆/月使徒.md) 当诱饵引血魂上钩。收尾靠 [莉莉姆](../../人物/莉莉姆.md) 的「沉眠」定身 3 秒 + 伍德幽绿焰 + 罪亚斯老化触手 + 苏晓埋下的 [茂生之狂乱](../../人物/茂生之狂乱.md) 树根，最终苏晓魔刃斩落其首。苏晓得 5.42% 世界之源、3227 灵魂钱币、不朽级宝箱·暗魔之影、**七星称号 [血意](../../力量体系/称号.md)**（伤害贡献:白夜 49.62% / 罪亚斯 21.59% / 伍德 20.77% / 莫雷 7.49% / 莉莉姆 0.53% / 月使徒 0% 躺赢）。
- **茂生之狂乱 vs 深渊之罐**：血魂一死，茂生之狂乱**本体**（循深渊之罐之气而来的千米级树根团）降临——它与深渊之罐是宿敌、此前按兵不动全因与苏晓有交情（苏晓许它一页树生之页）。二者各占半边天空开战、直接打崩了无尽沙漠这处缓冲空间:紫黑色半流体倾泻灌满。[伍德](../../人物/画中世界/沃波·伍德.md) 因与深渊之罐的血契困死原地、烧契约赌苏晓等人逃出；[莫雷](../../人物/原始大陆/莫雷.md) 的「黄金时刻」怀表险险激活、带重伤的苏晓一起脱离。
- **太阳教会（极恶阵营）**：苏晓重伤坠入沙之世界真正的地界，被中立·轻度恶阵营的 [老骑士（大骑士）](../../人物/画中世界/大骑士.md) 所救（没趁昏迷补刀、苏晓回赠压制古神系能力的药剂）。随后他误入 **太阳教会** 上百信仰疯子（戴铁桶头盔、死后自爆）的集会——戴上 [教会骑士头桶](../../力量体系/装备道具.md) 即被认作「自己人」、**加入太阳教会（极恶阵营）**（因恶阵营出身、声望获取速度 +45%）。（此前奥术永恒星淘汰腾出的名额被守望乐园顶上，那名参战者却晚来 12 小时、撞上无尽沙漠崩塌当场去世。）
