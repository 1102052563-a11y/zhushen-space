# 维斯星球·多亚古大陆（世界争夺战）

苏晓的**第 12 个衍生世界**、也是他经历的**二阶世界争夺战**战场——一颗未知星系的星球「**维斯星球**」上的「**多亚古大陆**」（被划为世界争夺区域）。剑与魔法、种族与信仰、帝国与部落为争夺肥沃疆土永无止境地厮杀。**本世界无法获得世界之源**（争夺战世界的特殊规则·收益改走「乐园功勋系统」）。

## 世界争夺战 = 轮回乐园 vs 天启乐园（十二卷1章）

- **对手 = [天启乐园](../../人物/希女王.md)**：与轮回乐园**疑同出一源**的另一个「乐园」。两个乐园争夺的「资源」惊人——**一整个世界**。世界越多、治下契约者变强的机会越多、契约者越强又能夺下更多世界，形成良性循环、乐园愈发浩瀚。这也部分揭开了「轮回乐园培养契约者变强」的目的之谜。
- **[虚空之树](../../总览/轮回乐园机制.md) 公证**：争夺战由「虚空之树」作**绝对中立**的公证方（初始法则第 1548762 条）——**战争期间双方乐园均不可对治下穿梭者作任何资助**（公平内战式争夺）。
- **乐园功勋系统开启**：杀敌方将领得 1~3 功勋、杀敌方契约者得 3~5 功勋；若战争胜利，按功勋值发奖。
- **传送两站**：星空中转站（无形屏障隔开双方·与苏晓同世界的约千名二阶契约者 + 天启乐园千余人）→ 维斯星球（30 分钟）。苏晓另获**通用语 Lv.40**（临时·战后遗忘·双方认证赋予）。

## 多亚古大陆 · 两大阵营

| 阵营 | 构成 | 政体 / 首都 |
|---|---|---|
| **龙腾帝国**（己方） | 已存 3700 年·人类为首 + 木 / 月精灵 + 巨人族依附 | 帝国制 · 哥特比亚（圣龙城） |
| **部落联盟**（敌方） | 更古老·兽人（牛头 / 狗头…炮灰）+ 矮人（工匠 / 中坚）+ 地精（炼金 / 战争武器）+ 巨魔（步兵杀手·压轴·人丁稀少） | 议会制 · 克伊山（烟囱山） |

帝国与部落已打了 **70 年（第 36 次大型战争）**，本次因精灵与矮人的矛盾激化而起（多由部落挑起）；根因是大陆资源养不活七个种族。三大战场：**希尔大峡谷**（坚守）、**奥德勒要塞**（坚守）、**黑土平原**（15 天前被攻破·苏晓战区）。

## 苏晓在本世界

- **身份**：龙腾帝国 · **铁血军团 · 三等兵**（战区黑土平原）。无猎杀任务（本世界无违规者）。**禁止向本世界生物提及轮回乐园**（违则公证后处决）。
- **战争任务「世界之核」**（Lv.29·共两环）：先**夺回黑土要塞**稳定前线、再激活战争支线；世界之核本身由 50 人的**敏捷类契约者斥候小队**寻找（[布布汪](../../人物/布布汪.md) 竟被编入该队、在圣龙城）。限 30 天·失败强制处决。
- 苏晓**不惧人海战术**（[噬灵者](../../力量体系/天赋.md) 杀敌回血回蓝·只要不被强敌杀 / 体力不耗尽就能撑到最后），反而对这超大型战场生出兴奋感。

## 黑土要塞 · 开战（十二卷3–5章）
- **黑土要塞**：全长 176 公里、黑曜石堆砌、加持 6782 个魔法烙印（坚过金属）、最高 60 米、屹立 3600 年——半月前被部落踩着十二万尸体攻破。
- **铁血军团 马克侯爵（军团长·世袭废物·贪墨要塞翻修费）** 限副军团长 [卡洛斯](../../人物/卡洛斯.md)（「边塞血狮」）一周夺回要塞——旋即王命传到：马克**降为伯爵**（贪墨东窗事发）、卡洛斯**转正军团长** + 增援 30 万。
- **冷兵器大战开打**：部落在要塞前列阵 + 从内部炸毁要塞（地精炸弹）；帝国以「铁血军团战鼓曲」群体增益、长矛兵 + 塔盾兵阵列冲锋，炮灰兽人如串糖葫芦般被矛戟穿透——十几万人血战拉开序幕。

## 黑土平原首战（十二卷6–10章）
- **战场众生**：部落方有狗头 / 狮头等兽人炮灰（力敏 30~40·无军阵技能·木甲精钢武器）、地精的**绿磷火燃烧瓶**（50 米玻璃化人成灰）与老式大型炮弹、矮人的**山岳骑兵**（骑血眼野猪·短柄战锤·冲锋猛于重骑兵）、压轴**巨魔**（5 米·钢甲·王牌·代表「山姆·斯普林霍尔」力 70 / 体 80〔本世界极数〕·战争血统 +8000 生命 + 次级魔抗 −60% 魔法 + 巨魔咆哮·相当于二阶小~大 BOSS）。帝国方靠精灵族**箭雨清场 → 军阵冲锋 → 箭雨再清场**的循环稳占优势。
- **[苏晓](../../人物/苏晓.md) 战场封神**：[噬灵者](../../力量体系/天赋.md) 杀敌回血回蓝、不到一小时吞满本世界 200 法力上限（恢复 +50%），他如开无双——力 73 徒手捏碎狮头人喉 / 砸断野猪鼻、反击盾化「能量刀刃 + 界断线」结成扇形刀网 / 旋转切割锯片、环断清场，杀敌数排进战场前几（黑土平原契约者 428 → 396 存活），军团长 [卡洛斯](../../人物/卡洛斯.md) 注意到他、欲提为小队长。
- **vs 天启乐园「[琉璃（银发少女）](../../人物/银发少女.md)」**（[希女王](../../人物/希女王.md) 之妹）：她趁乱以「狂想者」幻象 + 枪 + 匕首偷袭苏晓，反被识破吊打（肘撞「很软很软」），变小逃命时苏晓一狙**打掉她下半身**（她捏爆金色道具「海皇」也没能反杀·靠队友送去找希女王救）。

## 世界之核争夺 · 升小队长（十二卷11–14章）
- **首战落幕**：黑土平原一天一夜血战结束（部落 20 余万死约 8 万、余者溃逃黑土要塞·全歼不能·双方死伤逾 10 万·后方忙处理尸体防瘟疫）。帝国靠 5000 重弩**督战队**（逃兵格杀）维持军纪。
- **「世界之核」机制**：核心**无法放进任何储存空间、只能随身携带**；谁持有满 **20 个自然日**就完全激活。[布布汪](../../人物/布布汪.md) 在圣龙城地下与天启乐园侦查猫 [贝斯妮](../../人物/贝斯妮.md) 上演抢核「争霸战」，乐园公告一度在「己方得核 / 敌方夺回」间反复横跳——布布汪屡屡偷回、只肯把核交给苏晓。
- **圣龙城**：帝国首都（3700 年·四季如春·经济枢纽）——表面繁荣、底下是大贵族横行 + 王族腐朽（老国王 83 岁掌权 60 年、两年前废长立幼〔最平庸的幼子·疑非亲生〕、靠五大公爵 + 七亲王勉强支撑），如一棵根须溃烂的将倾古树。
- **苏晓升小队长**：战后他煮 [夏](../../人物/苏晓.md) 配的「秘制海鲜汤」（恢复体力 1 点 / 秒）引来军团长 [卡洛斯](../../人物/卡洛斯.md)——卡洛斯赏识其能力、**破格提拔他为小队长**（管 ≤200 散兵）。苏晓获**龙腾帝国官职**（共 36 阶·三等兵→…→小队长）+ **战功系统**：战功 **1436**（杀千余普通士兵·1 兵 ≈ 1 战功·杀敌方军官可选战功 / 乐园功勋），可在**军需官尼古拉斯·凯撒**（1/3 人类 + 1/3 地精 + 1/3 矮人混血·卡洛斯亲认·不可恐吓威胁否则遭血色军团驱逐）处**战功兑换**（乐园功勋 100 战功〔降 90%〕/ 灵魂结晶〔中〕260 /〔小〕90 / 金色「圣橡树枝干」等）。军需官帐前他还撞见曾被忽悠买牛头臂铠的少年姐弟（金发御姐冒险团团长·世界内不能敌对·尴尬收场）。

- **「炼金论交」军需官**：苏晓借炼金术拉拢 [尼古拉斯·凯撒](../../人物/尼古拉斯·凯撒.md)（好感度顶到 95）——他能调控战功商店物价、给苏晓把圣橡树枝干开内部价 9000 战功（详见其页 + [好感度系统](../../总览/轮回乐园机制.md)）。

- **卡洛斯夜攻**：30 余万增援到位、[卡洛斯](../../人物/卡洛斯.md) **亲率冲锋**强攻黑土要塞（战术 = 尸体 + 石块堆到要塞等高·备副官罗纳替身防战死）；阿姆斯特望见卡洛斯亲自带兵、大感不妙——惨烈攻坚战一触即发。

## 黑土要塞攻坚战（十二卷19–23章）
- **护城重轮弩**（矮人发明·战争兵器）：金属铸造的大型机弩（要塞上千架·上弦后 1 分钟内必发否则变形）——发射螺旋状椭圆「弩箭铰刀」（飞行中展开三米刀片、化高速旋转的绞肉铰刀·一轮齐射绞十几人成肉泥·还能绞碎精灵箭雨）。攻坚战初期几乎是对帝国军的单方面屠杀。
- **堆坡攻城**：帝国以重盾兵 + 背石块的运输兵**用尸体 + 石块堆到要塞等高**强攻；半小时死 6 万、累计逾 8 万，配合绿磷火 / 地精炮弹石坡融成岩浆坡、军心溃散。
- **卡洛斯的岩浆蒸汽计**：实则全在 [卡洛斯](../../人物/卡洛斯.md) 算中——他花 9 万金镑买暗藏「**热冷换置炼金阵图**」的石块混进石坡，由暗夜精灵公主 [安娜](../../人物/安娜.md) 激活、副官罗纳掌石板控制：岩浆涌至要塞齐高 → 冷热交替迸**过热蒸汽** → 借西风**「清蒸」死驻守要塞的三万部落军**（详见 [卡洛斯页](../../人物/卡洛斯.md)）。地精发明家识破「中计了」、部落首领仓皇逃窜。
- **高地战**：蒸汽散后岩浆坡浇水冷却成直通要塞顶的斜坡，黑土要塞两侧各有斜坡（一侧岩浆冷却·一侧部落预建）→ 变「高地战」（双方各 20 余万军冲要塞顶对撞·把对方打下去者得要塞）；部落这次出**精英部队**（矮人重盾兵叠 2 米盾墙破帝国盾矛阵 + 地精绑绿磷火燃烧瓶 + 护城重轮弩绞箭雨），战况空前惨烈、双方踩着尸堆血战。

## 苏晓的攻坚战（十二卷19–23章）
- **狙杀攒战功**：苏晓在帝国军中段以 [蜘蛛女皇](../../力量体系/装备道具.md) 狙杀矮人机弩手（**每杀 1 个得 60 战功**·用机械街仿制的 25 币劣质子弹·攻坚战中段共狙杀 50+），又杀「护城重轮弩指挥旗兵」（**200 战功**·与杀敌方头目相当）。
- **首获乐园功勋**：天启乐园狙击手「黑枪」（紫色狙击枪）嘲他「火力不怎样、纸老虎」、偷袭其大腿——苏晓以特殊穿甲弹反击、打碎黑曜石掩体、轰杀其队友，**击杀敌方契约者得 3 乐园功勋**（首次·乐园借此鼓励杀敌方契约者）。
- 蒸汽散后苏晓随大军踩着「血河」冲上要塞、投入高地白刃战（垂涎要塞上特殊兵种的战功·圣橡树枝干在望）。

## 高地战 · 王牌兵种 ·「坐过站」（十二卷24–28章）
- **冒险团团灭 vs 独行侠**：高地战惨烈——部落出**精英部队**（远强于首战杂牌军），轮回乐园契约者死伤惨重。中型冒险团**共生盟**（团长葛司令·96 人）为夺圣橡树枝干冲上要塞，主坦倒下后五分钟被冲散、二十分钟**团灭**（葛司令死于牛头人头目）。揭示规律：**冒险团不适合大型战场**（人多目标大、配合在人海中失效），**独行侠反而更适合**（无短板、同时面对的敌人少·苏晓乐在其中）。
- **矮人「大地骑兵团」**（王牌·1200 人·代表巴特力 52）：骑「尖獠战猪」、披矮人锻造 + 地精炼金的「大地战铠」，技能 **T 形冲锋阵**（千人「力量联动」成一整体土黄光环·平推一切）+ **生死与共**（与战猪共享生命·一方濒死则均分、致命伤则同死）。苏晓硬抗其千人冲锋、虽斩碎数名却被联动之力震飞（生命跌 17%）。
- **巨人族**（依附帝国·10 米·20cm 金属鞋底·**大地骑兵团的克星**·脾气憨厚「人类是朋友」·力 80 = 本世界极值）：一蹦几百米跨战场踩死大地骑兵、夺回要塞。苏晓搭巨人兄肩膀被「坐过站」横跨整个战场、甩到部落军后方。
- **暗夜精灵族**（帝国特殊兵种·斥候 / 暗杀·七成女性·吸光隐身·不能穿衣·携炼金腐蚀炸弹专破攻城器械·公主 [安娜](../../人物/安娜.md) 即此族）：部落为防其破坏，把「绿磷火燃烧瓶投掷器」（紫·攻城器械·战利品类·须投掷器具精通 Lv.10）等藏偏僻处。落单的苏晓想用投掷器攒战功却不会操作，引来一名赤果暗夜精灵妹子指点。

## 杀回血路 · 兰爵登场 · 埋伏（十二卷29–32章）
- **苏晓杀穿部落军**：用缴获的「绿磷火燃烧瓶投掷器」发射 30 瓶燃烧瓶攒大量战功（投掷器因被他卸了底座、反倒扣砸了自己一脸·没死）——随即从部落军后方**杀出一条血路**回黑土要塞（600+ 米·挨 13 利器 + 3 钝器 + 牙印·失血脸白）；途中以「[破空](../../力量体系/技能.md) 对众」一招把几十名拦路的大地骑兵斩到仅剩 4 人，终登要塞顶。期间一名赤果暗夜精灵妹子（话痨）远观、惊叹其强。
- **巨人族习俗**：寿命 600~700 年、50 岁才入青年期；**只有决斗胜者才有生育权**（女巨人眼光高·相亲即决斗·败者终生打光棍），优胜劣汰保族群强大（帝国王族屡欲废此俗未果）。
- **天启乐园强者 [兰爵](../../人物/兰爵.md) 登场**：被征调入希团队的机械侧高科技强者，收 [琉璃](../../人物/银发少女.md) 一颗红水晶、赶赴黑土战区**寻杀苏晓**。[希女王](../../人物/希女王.md) 以恐怖智商看穿兰爵「专猎能力异常的契约者」、隐觉触到「不可触碰的禁忌」，却顺势布双赢局、另派琉璃去**埋伏**。
- **苏晓战功 9243**（此战杀敌 3000+ + 狙杀矮人机弩手 + 旧存）——已够兑圣橡树枝干（9000）、但战时离场算逃兵会清空战功，故继续厮杀。他胸斩牛头人后，遭遇兰爵 / 琉璃布下的**机械蜜蜂**（敌方契约者 + 炸弹）埋伏。

## 猎杀者大战 · vs 兰爵（十二卷33–37章·未分胜负）
- **首遇另一个乐园的猎杀者**：埋伏中现身的 [兰爵](../../人物/兰爵.md) 自报「**肃清使**」= 天启乐园版的猎杀者（与苏晓「同一类人」·两个同类对话竟不被乐园屏蔽），奉强制任务来分生死（这世界只能活一个），刻意等苏晓体力消耗两小时才出手。
- **PVP 死战**：兰爵以机械召唤师之能（机械蜜蜂自爆 / 萤火虫四层荧光护盾 / 电浆球追踪麻痹 / 最强武器「**幽灵蜂**」金属箭穿盾刺杀）压制；苏晓靠「[看到](../../力量体系/技能.md)」预判 + 反击盾 + [界断线](../../力量体系/装备道具.md) 缠住幽灵蜂踢飞 100 米外令其失效、再 [剃](../../力量体系/技能.md) 冲入「自杀」爆炸群斩断兰爵左臂 + 肩膀。
- **结局**：苏晓不顾穿胸的幽灵蜂（偏移没穿要害·随之休眠 72h）直冲、**把 [兰爵](../../人物/兰爵.md) 斩成六段斩首**——兰爵却以保命大招「**机械之心**」（机械心脏 + 全身机械化撑 2 小时）拖断躯狼狈逃。苏晓算赢（只胸口一洞），但无力追杀。

## 卡洛斯之伤 · 追击 · 战争残酷（十二卷38–41章）
- **卡洛斯重伤将死**：高地战中他单挑牛头人首领阿姆斯特（断牛角逃）+ 冰霜巨魔「巨魔贤者」（斩杀）「一死两伤」、腹部被牛角顶洞；又险中地精发明家设的毒药剂。他**自评活不过三个月**、已属意副官罗纳接任军团长；恋人 [安娜](../../人物/安娜.md)（暗夜精灵公主·「终身绑定·只有丧偶」）为救他疯狂、盯上军需官凯撒限期想办法。
- **追击战 · 攻部落小镇**：部落军溃逃（地精先逃→巨魔→兽人→矮人），帝国转入追击；苏晓随军（铁血指挥官**罗素**·卡洛斯的拥护者）攻入一座部落小镇。**战争的残酷**毕现：帝国军烧俘虏、抢女性兽人、以细匕首捅女兽人子宫绝育、半人马族几近被屠灭（卡洛斯只抓士兵当苦力、不卖女兽人为奴）。苏晓不滥杀平民、向三名女性狮头人讨热水洗血后离去。
- **战功商店暂闭 · 兑换争夺**：战时商店关闭、须帝国驻扎黑土要塞（清石坡 + 打通内侧通道·1~2 天）才重开。返要塞后苏晓去找 [尼古拉斯·凯撒](../../人物/尼古拉斯·凯撒.md) 兑「圣橡树枝干」——4 层入口被冒险团安插的士兵拦阻，走廊已聚着**四个冒险团**十几名契约者（含金发御姐 + 少年「铁柱」姐弟）在争这件金色奖励。
