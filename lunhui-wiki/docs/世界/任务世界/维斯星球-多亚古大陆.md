# 维斯星球·多亚古大陆（世界争夺战）

苏晓的**第 12 个衍生世界**、也是他经历的**二阶世界争夺战**战场——一颗未知星系的星球「**维斯星球**」上的「**多亚古大陆**」（被划为世界争夺区域）。剑与魔法、种族与信仰、帝国与部落为争夺肥沃疆土永无止境地厮杀。**本世界无法获得世界之源**（争夺战世界的特殊规则·收益改走「乐园功勋系统」）。

## 世界争夺战 = 轮回乐园 vs 天启乐园（十二卷1章）

- **对手 = [天启乐园](../../人物/希女王.md)**：与轮回乐园**疑同出一源**的另一个「乐园」。两个乐园争夺的「资源」惊人——**一整个世界**。世界越多、治下契约者变强的机会越多、契约者越强又能夺下更多世界，形成良性循环、乐园愈发浩瀚。这也部分揭开了「轮回乐园培养契约者变强」的目的之谜。
- **[虚空之树](../../总览/轮回乐园机制.md) 公证**：争夺战由「虚空之树」作**绝对中立**的公证方（初始法则第 1548762 条）——**战争期间双方乐园均不可对治下穿梭者作任何资助**（公平内战式争夺）。
- **乐园功勋系统开启**：杀敌方将领得 1~3 功勋、杀敌方契约者得 3~5 功勋；若战争胜利，按功勋值发奖。
- **传送两站**：星空中转站（无形屏障隔开双方·与苏晓同世界的约千名二阶契约者 + 天启乐园千余人）→ 维斯星球（30 分钟）。苏晓另获**通用语 Lv.40**（临时·战后遗忘·双方认证赋予）。

## 多亚古大陆 · 两大阵营

| 阵营 | 构成 | 政体 / 首都 |
|---|---|---|
| **龙腾帝国**（己方） | 已存 3700 年·人类为首 + 木 / 月精灵 + 巨人族依附 | 帝国制 · 哥特比亚（圣龙城） |
| **部落联盟**（敌方） | 更古老·兽人（牛头 / 狗头…炮灰）+ 矮人（工匠 / 中坚）+ 地精（炼金 / 战争武器）+ 巨魔（步兵杀手·压轴·人丁稀少） | 议会制 · 克伊山（烟囱山） |

帝国与部落已打了 **70 年（第 36 次大型战争）**，本次因精灵与矮人的矛盾激化而起（多由部落挑起）；根因是大陆资源养不活七个种族。三大战场：**希尔大峡谷**（坚守）、**奥德勒要塞**（坚守）、**黑土平原**（15 天前被攻破·苏晓战区）。

## 苏晓在本世界

- **身份**：龙腾帝国 · **铁血军团 · 三等兵**（战区黑土平原）。无猎杀任务（本世界无违规者）。**禁止向本世界生物提及轮回乐园**（违则公证后处决）。
- **战争任务「世界之核」**（Lv.29·共两环）：先**夺回黑土要塞**稳定前线、再激活战争支线；世界之核本身由 50 人的**敏捷类契约者斥候小队**寻找（[布布汪](../../人物/布布汪.md) 竟被编入该队、在圣龙城）。限 30 天·失败强制处决。
- 苏晓**不惧人海战术**（[噬灵者](../../力量体系/天赋.md) 杀敌回血回蓝·只要不被强敌杀 / 体力不耗尽就能撑到最后），反而对这超大型战场生出兴奋感。

## 黑土要塞 · 开战（十二卷3–5章）
- **黑土要塞**：全长 176 公里、黑曜石堆砌、加持 6782 个魔法烙印（坚过金属）、最高 60 米、屹立 3600 年——半月前被部落踩着十二万尸体攻破。
- **铁血军团 马克侯爵（军团长·世袭废物·贪墨要塞翻修费）** 限副军团长 [卡洛斯](../../人物/卡洛斯.md)（「边塞血狮」）一周夺回要塞——旋即王命传到：马克**降为伯爵**（贪墨东窗事发）、卡洛斯**转正军团长** + 增援 30 万。
- **冷兵器大战开打**：部落在要塞前列阵 + 从内部炸毁要塞（地精炸弹）；帝国以「铁血军团战鼓曲」群体增益、长矛兵 + 塔盾兵阵列冲锋，炮灰兽人如串糖葫芦般被矛戟穿透——十几万人血战拉开序幕。

## 黑土平原首战（十二卷6–10章）
- **战场众生**：部落方有狗头 / 狮头等兽人炮灰（力敏 30~40·无军阵技能·木甲精钢武器）、地精的**绿磷火燃烧瓶**（50 米玻璃化人成灰）与老式大型炮弹、矮人的**山岳骑兵**（骑血眼野猪·短柄战锤·冲锋猛于重骑兵）、压轴**巨魔**（5 米·钢甲·王牌·代表「山姆·斯普林霍尔」力 70 / 体 80〔本世界极数〕·战争血统 +8000 生命 + 次级魔抗 −60% 魔法 + 巨魔咆哮·相当于二阶小~大 BOSS）。帝国方靠精灵族**箭雨清场 → 军阵冲锋 → 箭雨再清场**的循环稳占优势。
- **[苏晓](../../人物/苏晓.md) 战场封神**：[噬灵者](../../力量体系/天赋.md) 杀敌回血回蓝、不到一小时吞满本世界 200 法力上限（恢复 +50%），他如开无双——力 73 徒手捏碎狮头人喉 / 砸断野猪鼻、反击盾化「能量刀刃 + 界断线」结成扇形刀网 / 旋转切割锯片、环断清场，杀敌数排进战场前几（黑土平原契约者 428 → 396 存活），军团长 [卡洛斯](../../人物/卡洛斯.md) 注意到他、欲提为小队长。
- **vs 天启乐园「[琉璃（银发少女）](../../人物/银发少女.md)」**（[希女王](../../人物/希女王.md) 之妹）：她趁乱以「狂想者」幻象 + 枪 + 匕首偷袭苏晓，反被识破吊打（肘撞「很软很软」），变小逃命时苏晓一狙**打掉她下半身**（她捏爆金色道具「海皇」也没能反杀·靠队友送去找希女王救）。

## 世界之核争夺 · 升小队长（十二卷11–14章）
- **首战落幕**：黑土平原一天一夜血战结束（部落 20 余万死约 8 万、余者溃逃黑土要塞·全歼不能·双方死伤逾 10 万·后方忙处理尸体防瘟疫）。帝国靠 5000 重弩**督战队**（逃兵格杀）维持军纪。
- **「世界之核」机制**：核心**无法放进任何储存空间、只能随身携带**；谁持有满 **20 个自然日**就完全激活。[布布汪](../../人物/布布汪.md) 在圣龙城地下与天启乐园侦查猫 [贝斯妮](../../人物/贝斯妮.md) 上演抢核「争霸战」，乐园公告一度在「己方得核 / 敌方夺回」间反复横跳——布布汪屡屡偷回、只肯把核交给苏晓。
- **圣龙城**：帝国首都（3700 年·四季如春·经济枢纽）——表面繁荣、底下是大贵族横行 + 王族腐朽（老国王 83 岁掌权 60 年、两年前废长立幼〔最平庸的幼子·疑非亲生〕、靠五大公爵 + 七亲王勉强支撑），如一棵根须溃烂的将倾古树。
- **苏晓升小队长**：战后他煮 [夏](../../人物/苏晓.md) 配的「秘制海鲜汤」（恢复体力 1 点 / 秒）引来军团长 [卡洛斯](../../人物/卡洛斯.md)——卡洛斯赏识其能力、**破格提拔他为小队长**（管 ≤200 散兵）。苏晓获**龙腾帝国官职**（共 36 阶·三等兵→…→小队长）+ **战功系统**：战功 **1436**（杀千余普通士兵·1 兵 ≈ 1 战功·杀敌方军官可选战功 / 乐园功勋），可在**军需官尼古拉斯·凯撒**（1/3 人类 + 1/3 地精 + 1/3 矮人混血·卡洛斯亲认·不可恐吓威胁否则遭血色军团驱逐）处**战功兑换**（乐园功勋 100 战功〔降 90%〕/ 灵魂结晶〔中〕260 /〔小〕90 / 金色「圣橡树枝干」等）。军需官帐前他还撞见曾被忽悠买牛头臂铠的少年姐弟（金发御姐冒险团团长·世界内不能敌对·尴尬收场）。

## 战功商店 · 攻坚战前夕（十二卷14–18章）
- **战功兑换商店**（军需官 [尼古拉斯·凯撒](../../人物/尼古拉斯·凯撒.md) 处）：金色物品有「**圣橡树枝干**」（10000+ 战功·永久最大生命 +15% / 最大法力 +15%〔限 3000 以下〕/ 每分钟回血 +20·木精灵先祖之树·苏晓盯上、决定**存战功**待法力升到 2700 再用以最大化〔+405 法力〕）、达瑟汀（金剑·死亡诅咒）、邓肯的呼唤（金法杖·永恒之棺）等；上方乐园功勋（限 200·战争世界打 1 折 100 战功）/ 灵魂结晶则限量热抢。
- **「炼金论交」军需官**：苏晓借炼金术拉拢 [尼古拉斯·凯撒](../../人物/尼古拉斯·凯撒.md)（好感度顶到 95）——他能调控战功商店物价、给苏晓把圣橡树枝干开内部价 9000 战功（详见其页 + [好感度系统](../../总览/轮回乐园机制.md)）。
- **部落黑土要塞高层会议**：要塞顶端石屋内，四位无上下级的首领议事——牛头人 **阿姆斯特**（兽人首领·脾气最好）、山地矮人王、地精发明家（墙头草·十指戴满本领戒指）、冰霜巨魔（巨魔中少见的施法者）。部落虽兵力三倍于帝国、却因**议会制 + 各族矛盾**（选不出大首领·须四族混血而几乎不可能通婚）始终龟缩大陆边缘；为「谁当前锋」争吵不休，最终兽人首领阿姆斯特主动请兽人打头阵（兽人前一战已死伤惨重）。
- **卡洛斯夜攻**：30 余万增援到位、[卡洛斯](../../人物/卡洛斯.md) **亲率冲锋**强攻黑土要塞（战术 = 尸体 + 石块堆到要塞等高·备副官罗纳替身防战死）；阿姆斯特望见卡洛斯亲自带兵、大感不妙——惨烈攻坚战一触即发。
