# 希亚大陆·灭法遗迹（第十七卷·原生世界）

[苏晓](../../人物/苏晓.md) 的**第 17 个「世界」**、也是他进入的**第一个「[原生世界](../../总览/轮回乐园机制.md)」**——用传说级道具 [时空硬币](../../力量体系/装备道具.md) 开启（嫌乐园派的 EVA 半崩溃世界不适合自己·见下）。希亚大陆是一颗**超大型星球**、正处于**战争时期**、**不在虚空之内**；最关键的一条系统提示让苏晓心跳加速——**「希亚大陆内存在灭法之影所残留遗迹」**。对身负唯一职业 [灭法之影](../../力量体系/技能.md) 的他，这是一个意义重大的世界。

## 为什么来这（十六卷70章）

返园待进新世界时，乐园派给苏晓的是 **EVA 世界（半崩溃）**——科学侧 / 机甲流的福地，但对用刀的苏晓收益很差，且半崩溃世界不稳定、世界之子已死、还要在此做冒险团晋升考核，太头疼。苏晓遂掏出 [时空硬币](../../力量体系/装备道具.md)（传说级·可进试炼 / 战争 / 原生世界）——试炼 / 战争世界无主线剧情他不选，**选了「原生世界」**。因阶位过低无法进入「与虚空相连的原生世界」，乐园匹配到了**希亚大陆**。意外的是：时空硬币放出时空之力后**没有消失、而是重新聚拢**（非一次性·还能再用几次）。

## 原生世界机制 · 次级星空中转站（十七卷1章）

详见 [机制 · 原生世界](../../总览/轮回乐园机制.md)。要点：

- **原生世界 = 付费的「野生衍生世界」**：未经开发、从无契约者涉足、不能用常规渠道进入。乐园在此**关闭一切无偿机制**——语言、身份伪装、物品伪装都要花**世界功勋**（甚至语言功能要花乐园币）。世界为**同阶完全开放**（三阶 Lv.20~30 均可入）、主线任务**根据你在世界内的身份 / 地位 / 财力综合发布**。
- **次级星空中转站**：进入原生世界前的「预备战场」——一块直径约 5 米的石台、四周漆黑。苏晓在此被烙印 **「生命锁定」印记**（金色羽翼·作用未明）。**进世界前须先靠击败同批契约者获取「世界功勋」**（每杀 1 人得 1~5 点·按等级）；受致命伤即判战败、**世界功勋清零**。
- **本批入场名单**：76 名契约者用「空间战场荣誉证明（三阶）」、32 名用「原生世界邀请函（三阶）」进入。**苏晓用时空硬币（传说级）**——价值远高于前两种入场券，乐园按平衡机制直接**补偿他 16500 点世界功勋**：别人在起跑线、他「直接出生在终点站」，根本不是同台竞技。
- **联络平台**：随之从「衍生世界联络平台」变更为「**原生世界联络平台**」。苏晓进世界时**没挨惯常的「闷锤」**（异常·疑与时空硬币有关）。

## 破晓冒险团晋升考核（十六卷70章接受）

进新世界前，苏晓在冒险团公会接受了 [破晓冒险团](../../势力/index.md) 的**晋升考核**：现 B 级（单人模式的 B 级很「咸鱼」），共 **3 次晋升机会**（失败无惩罚但耗 1 次·3 次全败则再不能晋升）、每成功一次乐园按表现提升等级（最高 SSS 级·第一次约能到 A 级、之后越来越难但跨度越大）。考核任务将在进入世界后**自适应发布**。

## 兑换身份「铁之手」（十七卷3章）

凭 16500 世界功勋，苏晓在身份列表里挑了 **「铁之手」**（16000 功勋·进圣城王都）。身份列表跨度极大——财务官 6 万 / 王后 5 万 / 执政官 3.56 万 / 王国军团长 2.6 万 / 王国议员 1.5 万 / 首席法官 1.3 万 / 富商 1 万 …… 直到书记官 120、屠夫 6、农夫 2、**乞讨者 1（落点：随机粪坑）**。法官 / 议员要日常文官工作、苏晓干不来（三天就得进监狱），唯**铁之手**最合他意：

!!! info "铁之手 = 王族御用杀手（官阶 ★★★★★）"

    老国王最信任的几人之一、**专替老国王让某人「神秘消失、不留痕迹」**。性格设定**沉默寡言、冷酷无情**，受众官员忌惮却无人敢得罪；伪装要诀：禁称老国王为「国王」（重要时刻直呼姓氏「赫伯特」即可·老国王特允）、看任何官员都要满含杀意、得老国王首肯甚至可暗杀官员 / 王子 / 公主。对 9 点魅力、走「生杀大权」路线的苏晓再合适不过（魅力特长者拿到这职位「会哭晕在厕所」）。

苏晓被传送进铁之手的卧室——身旁是两名约 10 岁的**暖床混血小侍女**（姐姐 + 妹妹**丝露露**·见下「死灵族」·因暖床睡着、怕重蹈前任侍女「剥皮喂蚂蚁」的下场而吓尿）。苏晓的气息比前任铁之手更骇人。

## 王位继承制度 + 主线「权利之冠」（十七卷5–6章）

圣索丁王国选新王是**双梯队制**（防昏君）：

- **第一梯队 = 老国王的约 49 名子嗣**（王子 / 公主·15 岁后外放各地管领地〔似公爵·有封地财权、唯无大军〕）。老国王命不久矣时召他们回圣城**争位**——可暗杀 / 下毒 / 夺权 / 断财，只要不当面撕破脸；**最终能留在圣城者即继承人**。应召即入局、拒召即弃权。
- **第二梯队 = 圣城培养的数十名继承人**（非血脉·胜出后赐姓「赫伯特」·全国选拔·近乎选举制）——第一梯队全败时启用。

**主线任务「权利之冠」**（Lv.30·十七卷5章激活）：**选一名王子 / 公主、助其成功继承王位**。15 自然日选择期 + 30 日期限·**无惩罚**（原生世界「付费世界」故宽松）。奖励极厚——**4 真实属性点 + 任意一项「已掌握技能」等级 +1**（苏晓极珍视·上次同类奖励把至尊锋刃 MAX +1、斩龙闪才得以速晋金色）。老国王另令苏晓当晚**正大光明肃清异己**（带 20 名王族直属护卫·把异己「杀破胆」为新王铺路），唯独对**大王子哈帝**（47 岁）网开一面、给「一次不杀的机会」。

## 圣战士 · 圣之力（希亚大陆力量体系 · 十七卷12章）

希亚大陆的特殊力量体系——**圣战士**是「**人类经特殊改造**」而成（大幅提升体力 / 智力、并把智力衍生的法力**转化为生命值**）。其力量称「**圣之力**」：性质类似**可离体操控的精神力**，能随意变形 / 调整攻击角度（如凝成不可见的巨拳、或散成几十万根牛毛细针）。**只有圣战士、及少量死灵族能看见 / 对抗圣之力**——没有圣之力的人类被视作「没才能的凡人」、连圣之力都看不到。**「死灵君主」= 强大圣战士转化而成**（非死灵转化·呼应约瑟·柯的「死灵君主」侵蚀技能）；老国王 [巴伊·赫伯特](../../人物/巴伊·赫伯特.md) 临终正是**注入「死灵君主之血」**才硬撑到选出新王（代价是无法入眠、备受剧痛）。

!!! note "约瑟·柯——叛国圣战士（十七卷12–13章）"

    [约瑟·柯](../../人物/约瑟·柯.md)（秃头蜥蜴纹身·绿瞳·首席法官雇来的「鬣狗佣兵」）是**叛出圣索丁王国**的有称号圣战士。他的圣之力**附剧毒 / 麻痹**、可散成几十万根不可见毒针；是**战场型**强者（身旁士兵越多战力翻几倍·单挑反而不强）。苏晓虽看不到圣之力、却凭 **[直感](../../力量体系/技能.md)**（7 米预警）躲过其偷袭、以青钢影噬灭圣之力、反击盾防毒针。详见其页（含完整 8 技能 / 六维资料 + 「死灵君主」侵蚀伏笔）。

## 灭法殿堂遗迹 · 三殿三石（十七卷37章 · 苏晓的核心目标）

## 找「王之血」艾丽莎公主（十七卷43、45–46章）

老国王羊皮纸三句：「**日月同在、王座即可开启**」（王座机关）/「**王之血·艾丽莎·赫伯特**」（带这公主开机关）/「**放逐之人陵墓、永不得安息**」（含义未明·或关键或烟雾弹）。苏晓遂去找公主 [艾丽莎·赫伯特](../../人物/艾丽莎·赫伯特.md)（20 岁·无权无势·门前却种剧毒蓝玫瑰、还有强圣战士**骑士戈斯**贴身守护——老国王送的「开路炮灰」大礼）。苏晓拿金狮信物 + 新王书信逼戈斯放行、一脚踢飞硬拦的他。

## 深入王座之下 → 希亚圣殿（十七卷47–49章）

苏晓**暴力开王座**（不用机关·真实力 89 推得王座倾斜、斩龙闪斩 + 一脚踹飞），露出一道 3 米宽的金属竖井。一行人（苏晓 + 布布汪 + 阿姆拎着 [艾丽莎](../../人物/艾丽莎·赫伯特.md) + 骑士 [戈斯](../../人物/戈斯.md)）跳下——竖井其实是**年久失修的升降梯**（金属托盘卡在半空·因苏晓非常规进入未启动机关），阿姆千斤坠砸断托盘、众人在加速下坠中靠短棍撑壁减速（戈斯两度自由落体先砸到底）。竖井底有需**王族血脉**开启的金属门，苏晓哄骗傻白甜公主艾丽莎开了门。

**进入隐藏区域「希亚圣殿 / 不死者陵墓」**（详见 [地点](../../世界/地点/index.md)）：高度危险·宝箱 / 世界之源奖励 **+20%**；进外围**每小时耗 600 乐园币**（不足则强制传送出入口）。苏晓有 199000 币（够停 14 天），但任务只剩不到 4 天——须速找「灭之石」、否则可能因停留时间不足被强制传出。

## 抽签整戈斯 + 霸主级里德（十七卷50–52章）

陵墓阴森潮湿、前方有敲击声 + 热浪。苏晓用「**抽签**」逼戈斯探路——四张纸条其实**全是红色**（苏晓借储存空间把自己 / 布布汪 / 阿姆的调包成白色），戈斯必抽到红、沦为开路炮灰，还被苏晓按着艾丽莎脖颈逼着去探。戈斯探到一间火红的锻造室，里头是——

!!! info "陵墓守卫者·里德（霸主级 · 中立）"

    [里德](../../人物/陵墓守卫者·里德.md)：身高 1 米 5、镶满铁甲、双眼是锈望远镜探头、头被铆钉 / 订书钉钉成三截（换头即换语言）的**锻造者**。阶位是**「霸主级生物」**（高于敦克·尼克的领主级）·死灵 + 人类各半·智 98·法力 9860·「霸主权威」给生命 +10 万——但他是**中立、友善对待任何种族**。苏晓自估「最多挨他两锤就会死」；霸主级与领主级是云泥之别（领主级敦克·尼克在他面前「不超两锤就成碎肉」）。

## 忽悠霸主级里德进乐园（十七卷53–54章）

斩不动霸主级的 [里德](../../人物/陵墓守卫者·里德.md)、又怕他守门碍事，苏晓改用「等价交换」忽悠——拿 [时空硬币](../../力量体系/装备道具.md) 大谈虚空、承诺「帮你进虚空」（里德被困陵墓几百年早想出去）。里德很穷、只肯给情报换：**影之石失踪多年、灭之石在陵墓深处、法之石他知道下落**。苏晓布**空间阵图**（虚空角斗场结识的女魅魔莉莉姆所授·灭法之影友军）、趁里德试探时把他**吸进裂缝送进了轮回乐园**（乐园招纳这种锻造者）。

> 【为轮回乐园获得一名锻造者】苏晓得 **3 真实属性点 + 4 万币 + 权限 +Lv.2 + [里德之勋](../../力量体系/装备道具.md)**（可在乐园找里德每 15 天无偿锻造一次）。入口隐患就此清除。

## 深入陵墓 → 长刀使·空 + 戈斯逆袭（十七卷55–58章）

艾丽莎一路开门、深入陵墓（更宽的油灯石廊·途中两扇刻着「剑 / 盾」「长弓 / 利爪」图案的门·苏晓不开）。石廊尽头空地的石台上躺着守门的 [长刀使·空](../../人物/长刀使·空.md)——**首领级死灵·远古血脉**（4 米高·绿瞳·5 米长柄刀骑兵），与里德的霸主级**反差极大**：被苏晓**按在墙上踹 + 砍**（速度碾压·真实敏高它 8 点），生命掉到 46% 才触发「使令」、召来**陵墓驻军 200 余死灵士兵**（盾步兵 + 长枪兵 + 长弓兵精英方阵）。傻白甜艾丽莎此时**意外有用**——她听得懂长刀使·空的失传古语、替苏晓翻译。

死灵军杀来时，骑士 [戈斯](../../人物/戈斯.md) 突然退守、**想借乱害死苏晓等人**——苏晓一狗一牛一人齐冲、龙影闪绕后一记侧踢**踢爆戈斯下阴**（长刀使·空都看得脸抽搐），把惨嚎的他踢去与长刀使·空缠斗。苏晓一行随即退守来时石廊当防线（长刀使·空已重伤、翻不起风浪）。

## 杀长刀使·空 → 战震天使·鸣 → 死灵贤者（十七卷59–63章）

苏晓斩杀 [长刀使·空](../../人物/长刀使·空.md)（[戈斯](../../人物/戈斯.md) 不敌战死·只剩骨灰·艾丽莎不知真相），得 **[远古死灵精粹](../../力量体系/消耗品材料.md) ×3**（希亚圣殿硬通货·戈斯参战削减了收益）。傻白甜 [艾丽莎](../../人物/艾丽莎·赫伯特.md) 靠在父王书房偷看过的书、认得古死灵语，帮苏晓破解金属拱门的开门密码（幽灵之焰按「太阳起→月亮终→再回太阳」的顺序点燃孔洞）。

拱门后是「**震天殿**」：500 死灵军 + 守卫精英 [震天使·鸣](../../人物/震天使·鸣.md)（5 米·**战鼓增益**令士兵实力翻 2~3 倍·**近乎无敌的灰色护盾**·寂灭公爵 +13 轰头都毫发无伤·虚无鼓锤把布布汪锤到惨嚎、阿姆锤出满头包）。苏晓开局一枪 + 抛阿姆缠住它、自己半放弃防御**清空 500 死灵军**；发现它仍需呼吸，便用毒雾耗破护盾、将它斩杀（精粹 ×5）——其无敌护盾原是连着偏殿一处**能量柱装置**（布布汪潜进啃坏）。

能量柱毁后，偏殿石壁里的 [死灵贤者](../../人物/死灵贤者.md)（中立石像）苏醒、开启「**圣殿兑换商店**」（以远古死灵精粹兑套装宝箱 / **圣洁庇护**〔抗黑雾·圣殿三层充斥黑雾〕）；**震天殿能量中枢摧毁、此区域转为安全点**——关乎「[原生世界归乐园](../../总览/轮回乐园机制.md)」机制（先驱者优待）。深入震天殿后每小时耗币从 600 飙到 **3000**。

!!! note "希亚圣殿生物 = 被囚的「战场之物」"

    苏晓发现这里的守卫（长刀使·空 / 震天使·鸣）都是为战场而生、放到战场会强好几倍，更像**被囚禁在此、并拒一切生物进入**：艾丽莎开的金属门全是**单向、只能从外开、且需王族血脉**——耐人寻味。远古死灵精粹则可换 3% 世界之源 / 8 万币 / 套装宝箱、或直接吞用（永久 +异常抗 3 / 生命 80 / 法力 50·限 3 颗）。

## 战甲之灵 · 阿波罗 → 灭之石（十七卷64–67章）

[死灵贤者](../../人物/死灵贤者.md)（圣殿建造者之一·遭同族背叛）告知：**灭之石镶在 [战甲之灵](../../人物/战甲之灵.md) 的铠甲上**——它是希亚圣殿一层最强者（领主级大将军·实力不弱于敦克·尼克·长刀使·空 / 震天使·鸣只是「看门的」），镇守通二层入口、身边还有 30 名普通长刀使。**希亚圣殿三层**：一层将军 / 士兵、二层王族精英、**三层「生灵禁区」**囚着希亚大帝·生灵屠夫等恐怖存在（连存活都难·别深入）。

硬拼拿不到灭之石，苏晓用 [隐之力](../../力量体系/装备道具.md) 隐身、抛 [太阳神·阿波罗](../../力量体系/消耗品材料.md)（密闭空间威力更强）——把整片空地烧成火海、30 长刀使化灰、战甲之灵被烧到生命 23%。战甲之灵本想把灭之石抛进二层险地（宁死不让灭法之影得），但苏晓追来勾起它「战斗狂」本性、决意尽职一战；激战中被苏晓龙影闪躲过「震荡炮冲拳」、刺穿心脏，临死它**主动把灭之石抛给苏晓**（只为知道栽在谁手）、烈焰焚身眼不眨地战死。

> 【你获得**灭之石**】——至此**灭 / 法 / 影三石齐全**（法之石苏晓「多次见过」、已知下落）；【击杀战甲之灵】**远古死灵精粹 ×13**（本世界共得约 16 颗）。

## 阿姆融冰蓝战锤 → 冰装甲 + 离开圣殿（十七卷67–69章）

火焰区里，[阿姆](../../人物/阿姆.md) 被一把**冰蓝战锤**（里德锻造、用于镇压某存在的冰武器）吸引而拿起（冰能量相似度 >95%）、核心阵图严重过载失控。苏晓剖背为它构建「能量渠道」（冰锤当心脏、冰能量循环），既解过载又**变相延寿**——阿姆**存在时间 256 → 3756 天**、进化出「**冰装甲**」（伤害 40% 由冰能量替代·抗揍系数飙到 ~20）+ 多了把武器（详见其页）。

撤离时火焰区涌出一团**黑雾人形**（超高速·诡异·苏晓「想不出办法对付」·疑即冰锤镇压的存在）追来，被关闭的金属门挡回。苏晓判定**自己以现在的实力（乃至四阶）都杀不穿希亚圣殿**（二、三层的东西只能用「诡异」形容），遂带队**原路离开希亚圣殿**——三石到手、又得金色套装宝箱 + 16 颗远古死灵精粹。下一步：去灭法之影遗迹 + 完成冒险团晋升、再离开本世界。

## 得法之石〔三石齐全〕→ 赴前线（十七卷70章）

回到王宫，苏晓与三王子 [马蒂尔](../../人物/马蒂尔·赫伯特.md) 做最后一桩交易：要走铁王冠上那颗红宝石（**红宝石内藏着「法之石」**）、答谢是「不再出现在王都」。三王子乐得他赴前线（铁之手在前线被架空·百利无害），给了宝石——苏晓捏碎红宝石得**法之石**。**至此灭 / 法 / 影三石齐全**（可花 5 万币合成「断魂影之石〔残缺〕」、苏晓不合成·进遗迹还要用）。他警告三王子别再开王座之下（希亚圣殿已盘踞恐怖之物），随即赴前线。

**路线**：王都 → 亚讷山脉 → 图兰尔城 → 坤济北森林 → 廷萨港沙漠 → 塞梅湿地 → **神之墙**。墙外即死灵领土、**灭法之影遗迹就在死灵帝国首都暗月沼泽附近**（死灵称之「**诺托埃克 = 生灵止步之地**」·死灵进不去）。

## 激活冒险团晋升考核（十七卷71–72章）

苏晓特意选「**塞梅湿地**」这段最安逸的神之墙翻越（死灵只攻运输物资难的区段·这里物资丰富、几百年没被攻打）。一上墙外，终于**激活了破晓冒险团的晋升考核**（王都激活不了、唯前线能激活）——并被烙下「**死灵定位印记**」。详见 [机制 · 冒险团晋升考核](../../总览/轮回乐园机制.md)：

- **停留时间清空、可随时返园**；考核评分按「激活后还能撑多久」——四阶段难度递增：**初始（小股死灵追）→ 生存（死灵军队围剿）→ 炼狱（+死灵强者）→ 噩梦（+原生世界排斥·终极）**。
- 死灵定位印记让**死灵族持续追杀**苏晓。他的破局思路：躲进死灵进不去的灭法之影遗迹，就能边躲追杀边刷考核时长。

## 死灵「启迪」之死 → 全族追杀（十七卷73章）

追杀的根源揭晓：苏晓激活考核时，**轮回乐园弄死了死灵族的 [启迪](../../人物/启迪.md)**（百年一出、维系死灵族延续的「引路萤火」）——启迪临终遗命「找到铁之手·白夜、杀了他，挖出我的心指引你们」。死灵王·森迈尔捧其心脏（印记与苏晓手背的相似度 >90%）下令**停战、举族追杀苏晓、以其血祭奠启迪**。苏晓测试后发现印记**不拉普通死灵平民的仇恨**（农夫只恐惧、甚至向他竖拇指求生），追杀主力是死灵的军队与强者。

## 分头 + 死灵大军围剿（十七卷74–76章）

初始阶段极短。苏晓让 [布布汪](../../人物/布布汪.md) + [阿姆](../../人物/阿姆.md) 拿地图绕路去暗月沼泽找灭法遗迹（俩非人类、在死灵帝国行动方便），自己**直线突围吸引火力**。死灵统帅老练——**清空沿途村庄**（苏晓连过 4 座空村）施压 + 封锁情报；斥候村庄埋伏发信号弹（死了家眷优待）、魁梧死灵**奥乌·南萨**（人类语流利·死灵王许其取苏晓头颅赏「相澜之位」）都被苏晓秒杀。随后**上万死灵军**（精锻铠甲 + 2 米塔盾盾墙 + 招牌远程「黑色暮雨」）把他团团围住——死灵已停攻神之墙、抽大军在领土巡查（可聚百万）。苏晓靠噬灵者 + 嗜血礼赞 + 木之灵在混战中回血、使徒之眼第三能力「警戒」（半公里雷达）辨向，准备拔刀突围。
