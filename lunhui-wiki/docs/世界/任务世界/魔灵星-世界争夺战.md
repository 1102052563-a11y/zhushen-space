# 魔灵星·世界争夺战（第40卷·七阶世界争夺战 · 苏晓的八阶晋升试炼）

[苏晓](../../人物/苏晓.md) 苍龙大陆通关、达 Lv.70 七阶满级后接下的**八阶晋升试炼**世界——但公证方是**虚空之树**，故形式为一场**七阶世界争夺战**（本世界之核由己方七阶契约者争夺、苏晓则被派往完成前置任务）。因苏晓**自身战力远超七阶**，虚空之树判定他**延时几天进入、且落点与 99% 以上契约者相距极远**（进入次序按战力排序、越弱越早进）。晋升试炼**最多两次、两次皆败则永失晋升八阶资格**。

## 世界背景 · 魂能 / 黑塔 / 四大势力

魔灵星号称**「契约者墓场」**（八阶契约者去了都不安全）——一颗蒸汽 + 电力时代的原生世界，核心设定是**「魂能」**（世界简介逐条）：

- **黑烟 → 魂能**：远古人们困居高墙内、惧怕墙外的黑色铁砂状「黑烟」；**黑塔组织**最先举火把带 43 人出墙探索（仅 2 人生还），历经数百年、无数次牺牲，才发现「黑烟」真身是**生命与超凡之源**——吸入体内经稳定即可强身延寿，若**与部分灵魂融合则诞生一件仅本人可见的器物（剑 / 种子 / 特殊生命体）并觉醒超凡能力**，此即「**奇迹者**」，黑烟遂改称「**魂能**」。后奇迹者把魂能与灵魂结合的器物二次构建成「**圣印**」（收入体内不断想象完善即可突破上限），创圣印者成为**魔法协会**首任领袖。
- **黑潮**：原始魂能在某点喷发、笼罩十几公里的现象——是超凡力量 + 寿命 + 花不完的黑晶 + 地位的来源，故黑潮过后聚来的强者与亡命徒才最可怕（烟之都 30 区就因黑潮荒废）。**黑潮雨**沾之变黑、危险。
- **原始魂能 vs 人工魂能（黑晶）**：原始魂能可吸入体内觉醒超凡；**人工魂能「黑晶」**是魔法协会人工合成原始魂能的失败降级品、不可吸入、只能驱动机械 / 照明 / 取暖（相当于本世界的「石油」、利润是生活物资的几十上百倍）。西大陆的独有货币是「**古朗**」（魔法协会发行·1 标准单位黑晶 = 175 古朗）。

**四大势力**（东大陆）：**魔法协会**（本世界最强·掌握圣印构建法·守序阵营形象·实则在西大陆秘密造人工魂能）、**治愈神教**（信魂能危险却只能以魂能克魂能·真正为西大陆流民着想·驻烟之都 1~3 区救黑潮）、**蒸汽联盟**（科技至上·掌控海运）、**命运教会**（常做「人类迷惑行为」如建 117 米金属塔迎神迹反成避雷针·四大势力中最弱却从不缺魂能）。三方为原始魂能争斗不休、命运教会从不争夺。

## 两片大陆 · 黑塔

- **西大陆（苏晓落点）**：环境恶劣、混乱无序、只有流民没有平民（警惕凶狠贫穷、连苦力都随身带杀鱼刀）。被魔法协会完全封锁（连空间都封）、成了造人工魂能的实验区。核心城市**烟之都**（32 个区·相当于一座中小城市 ×3）：1~3 区治愈神教（最安定）、4~9 区魔法协会、10~15 区炼金联盟工业区、16 区野鸳、17~29 区混乱收尸区、30~32 区边缘荒废危险区。
- **东大陆（世界争夺战区域）**：阳光明媚、风景如画，四大势力坐落于此，己方七阶契约者大部队在此等世界之核出现。
- **黑塔**：最先举火把探索墙外的元老组织——如今成员是「黑暗中的观望者」，时善时恶、令人捉摸不透又无比忌惮（苏晓在本世界的所属势力即黑塔·化名「白夜」/「斩首的夜」）。

## 战争任务 · 晋升任务

- **战争任务「世界之核」（第一环 / 共三环 · Lv.70 · 3 日）**：在烟之都找到 6 处**节点**、至少在 4 处布设**临时坐标**（金属柱·每 6 小时随机一处发生空间共振反应约 40 分钟·布设需 10 分钟稳固·被打断即收拢）。成功布 4 处后轮回乐园可剥离世界之核（初始归属己方）。失败无惩罚但强制回归、收益清空、晋升失败。
- **晋升任务**：完成战争任务夺得世界之核后开启；夺核后本世界世界之源获取量 +200%。
- **解除的限制**（因需争节点坐标权·至战争结束）：任务惩罚机制解除 50%、世界联络平台发言限制解除、**杀戮限制完全解除**——击杀敌方乐园契约者额外得**猩红之勋**（详见 [装备道具](../../力量体系/装备道具.md)）。

第二、三节点连续出现在 **5 区**——因 5 区次日中午将爆发**落柱潮**（[休西罗](../../人物/魔灵星/休西罗.md) 与其他势力都能预示），故各方齐聚。争夺关键在苏晓从 [圣域神棍](../../人物/魔灵星/圣域神棍.md) 口中套出的**「节点的秘密」**：

- **黑潮落柱潮**（18章）：5 区落柱潮爆发、众人疯抢原始魂能（命运教会却在楼顶炫「黑晶发电」探照灯），尤妮丝趁乱夺第五节点 → **节点态势 轮回 3 / 天启 2 / 空 1**。苏晓让布布汪用治愈神教的 [吸魂能油灯](../../力量体系/装备道具.md) 苟着偷吸满 5000 单位魂能、转手抛给面罩男·巴斯开交易。

## 太阳圣殿防御战（第40卷25–30章）

世界坐标布设点=冈卢王朝古都的**太阳圣殿**——冈卢王朝当年历练「太阳勇士」之地、被虚空之树加持得连 [烈阳之怒·阿波罗](../../力量体系/消耗品材料.md) 都炸不动（墙不可破、门可破但会自动恢复）。**七层结构**：① 一层防御区（石柱林立·守卫 [复仇男爵 / 木乃伊](../../人物/魔灵星/复仇男爵.md) 二人组 + 液态火焰机关）② 二层决斗区（**50 个无门房间 = 50 重试炼**·各驻 1 名己方契约者·敌人经过即被强制拉入 1v1 决斗、死一个才能出）③ 三层杀戮区（太阳光束·[稻草人](魔灵星-世界争夺战.md) 女骗敌进来）④ 四层虚影弓手区（102 道人形虚影射太阳光束·会弹射误伤）⑤ 五层石墙混战区（决战区）⑥ 六层紧急退守区（石雕·「前无退路，闯入者死」·藏惊喜）⑦ **七层世界之核**（黄金王座镶世界之核）。

- **苏晓夺核者优势 + 被束缚七层**（25–27章）：作为最先夺世界之核者，苏晓得**烈阳庇护印记**畅行各层不触发机关；把世界之核镶上七层黄金王座（**世界坐标 52 小时后开始布设、需 2 个自然日**），但开战后 5 小时内被光壁困在七层不能离开。
- **己方防御分工**（27、29章）：[升天兄](../../人物/魔灵星/升天兄.md) 组织 156 名己方契约者按七层地形布防（二层抽签选 50 单挑手、三层稻草人、五层决战、六层留惊喜）；「黑魔」[小胖子](../../人物/魔灵星/小胖子·黑魔.md) 主动去外围拖住联军、吞掉其 49 人（连己方都忌惮、绝不与之联手）。
- **联军受阻一层**（28–30章）：天启死亡联军 592 人（被黑魔吞剩 543）由 [怒犬](../../人物/魔灵星/怒犬.md) / [海伊左](../../人物/魔灵星/海伊左.md) / [莫雷](../../人物/原始大陆/莫雷.md) 率领攻圣殿——却被一层 [复仇男爵 / 木乃伊](../../人物/魔灵星/复仇男爵.md) 的再生 + 幻觉 + 液态火焰耍得团团转（150 进、[兔糖](../../人物/魔灵星/兔糖.md) 泡泡捞出 126）。海伊左识破二层 50 试炼机关、联军遂以怒犬为首率 50 强者强攻。

## 苏晓的八阶晋升考核伏笔 · 冈卢五物

贝妮在库巴鲁克遗迹夺得的**衔尾蛇石板**是苏晓**八阶晋升考核任务**的关键物之一——冈卢王朝覆灭真相竟非「修太阳之梯」、而系于**五件物品：太阳圆盘 / 权杖 / 棘鞭 / 衔尾蛇 / 火焰**（上有一张眉心流黑血的面具·正是 [神父](../../人物/魔灵星/神父.md) 献祭圣域方的大殿石雕图案）。另一件「**心之焰**」在烟之都、为黑塔组织所有——苏晓的初始势力是黑塔正因于此（打赢世界争夺战后需接触黑塔取心之焰）。

## 出场契约者（己方 · 东大陆）

苍龙大陆见过的老面孔多在东大陆蒸汽联盟风笛市的大剧院选「临时领袖」（各投自己票·场面尴尬）：**升天兄**（顶绿光圈的正常人·被恶搞投 3 票）、**魔女**（女巫帽·刚晋七阶的旅团成员）、**钉脸小男孩**（金属钉小脸·舔钉舌）、**娇小少女**（丢了心爱耳坠装备、咬牙要找轮回某人报仇）、召集众人的**正装男人**等。
