# 暗鸦星 · 暗鸦纪元（女巫与灭法者）

[苏晓](../../人物/苏晓.md) 的**第 21 个「世界」**、**自费进入的原生世界**（第二十一卷「暗鸦纪元·女巫与灭法者」）。死神世界返园后，苏晓用 [女巫指环](../../力量体系/装备道具/index.md) + [时空硬币](../../力量体系/装备道具/index.md) 同时使用、以超额时空之力换「**高级 VIP 待遇**」进入——可选主线、宝箱收益 +50%、物品公证、可世界结算，但**无世界之源、改获「开拓之源」**（详见 [机制](../../总览/轮回乐园机制.md)）。

- **落点**：边陲·马德高人类聚集地（一座雨夜里的奴隶城）。难度 **Lv.30～Lv.42**。掌握本世界语言需 5000 币（通用语 1 + 常用 7 + 王国语 18 + 部落语 47·语言之杂印证世界之乱）。
- **主线任务「刃之觉醒」（自费 VIP·四选一）**：四个主线（女巫猎人 Lv.35 / 探索者 Lv.31 / 暗鸦 Lv.37 / **刃之觉醒 Lv.40·极度危险**）里，苏晓必选**刃之觉醒**——找到「刀魔能量封印装置」（女巫们称之「**死亡之地**」）完成刃之觉醒·奖 **断魂影之石·残缺 ×1**（凑齐 5 颗可合成完整断魂影之石）·停留期限延到 30 天·罚强行处决。线索仅一名女巫得知。
- **目的**：完成 [马文·华尔兹](../../人物/马文·华尔兹.md) 教的「把刀魔能量注入 [斩龙闪](../../力量体系/装备道具/index.md)」（详见斩龙闪「刃之觉醒」）。

## 世界观 · 女巫与暗鸦纪元

- **起源石 + 初代女巫暗鸦**：草原上一块被挤出地表的**紫色晶体「起源石」**——靠近的动物会癫狂自噬胀死、强行承载的将领七孔流血而亡（「我不足以承载它」）。直到一名**被不幸逼疯**的女人跪在起源石上、体内分裂出的「另一个自我」吸收了起源石之力——她觉醒成**初代女巫 [暗鸦](../../人物/暗鸦.md)**（百万意识乌鸦·把黑夜白昼翻转·**67 天征服一颗星球**·用王的头盖骨饮酒、自封女王）。她只活 67 天便如烟花爆开、留下一块石板，**暗鸦纪元**就此开始。
- **女巫的转化 / 化身**：女巫由普通女性转化——先**折磨自己（溺水 / 鞭打 / 暴晒 / 弑亲）逼出精神分裂的副人格**，再用副人格**吸收起源石之力、杀死本人格、形成「化身」**（有化身才是女巫·只有女巫能看见化身·化身模样多与本人不同·具现后近乎实体·甚至与女巫不同意识）。元素系女巫用化身直接粗暴地驱动元素（同级战力是法师数倍·法师见之多半转头就逃·「没人想和疯子拼命」）。
- **三大类**：**元素系**（战力最强·极端性格）、**咒术系**（阴损·收集头发 / 血液为媒介杀人）、**预言系**（佛系和善·最强能力是语言）。**黑女巫** = 元素 + 咒术（平民惧之·视平民为活体实验材料）；**白女巫** = 预言（受欢迎·惧黑女巫·黑白不敌对）。**克星 = 金刚石**（打入体内大幅削弱女巫能力·几万人未必敌一女巫·但有此弱点）。「暗鸦」既是初代之名、也是女巫的最高荣誉称号（仅最强女巫可得）。

## 大陆格局

- **中 / 西部**：[圣愈教会](../../势力/index.md) 管制、法度相对完善（苏晓落点的西部常年雨季湿冷）。
- **东部**：土著势力居多、几无法律、**女巫最多的「无法之地」**。
- **北境**：冰原冻土、游猎民风彪悍、女巫少（太冷·女巫会被冻死）。**南部**：女巫地盘、人类禁区。
- **科技**：暗鸦纪元 1433 年·蒸汽机 / 火车 / 电（仅权贵用）·武器只钻研大威力霰弹枪「喷子」（金刚石子弹·因敌人是女巫·无坦克火炮）。
- **奴隶制**：恶魔牙 / 里可 / 巴巴厄三大奴隶庄园承包大陆 70% 奴隶生意（恶魔牙最彪悍·连女巫都敢卖）。许多平民走投无路自卖为奴换一顿饱饭。

## 加入圣愈教会 · 成巫猎人（二十一卷17–18章）

苏晓凭杀女巫所得的**化身具现晶体**（元素系化身被打碎所留·要在女巫活着时打碎才有）当投名状，被圣议员 [卡门](../../人物/卡门.md)（同类相吸·一眼看穿苏晓是「怪物」）直接收入 [圣愈教会](../../势力/index.md)、成「**女巫猎人**」（白宝石指环）。巫猎人三级：**白级（兔子）< 黑级（夜犬）< 红级（血猎人·猎杀 ≥20 女巫）**——红级才能查阅图书馆机密。圣愈教会设施：图书馆（二层）、阵营商店（四层·需黑级）、武器锻造工坊（制式武器免费）。苏晓的目的：升红级查「摩」字机密 + 攒声望兑阵营商店 + 收开拓之源 / 宝箱。〔孪生姐妹 [雪莉 / 雪伊](../../人物/卡门.md) 前台 / 图书管理员暗中替卡门观察他·苏晓一接就把全部委托〔含五阶女巫摩利莎那张〕拿走、把他们惊到。〕

- **尼克多多城·黑鱼虫**（三名三阶元素女巫 + 幕后 [米拉](../../人物/米拉.md)）：女巫用改造黑鱼虫（产卵即死·寄生卵吸生命力）荼毒万人大城。苏晓识破内奸（心腹侍从坎等被收买）、在城堡里一刀秒三阶元素女巫（贵妇 / 皮衣 / 金发）、逼问出幕后是四阶 [米拉](../../人物/米拉.md)。

## 衔尾蛇禁忌 ·「摩」字另有含义（二十一卷28–29章）

## 东部 · 牙骨部落灭族（二十一卷29–30章·插叙）

暗鸦大陆东部的彪悍土著「**牙骨部落**」（上千战士·35 岁即献身给后代分食「与我同在」·曾吃掉圣愈教会两万讨伐军 / 安葬巫猎人后与教皇签互不侵犯约）——七名女巫（含四阶）去抢东西、反被牙骨部落不死不休追杀几乎团灭。最后那名逃出的四阶女巫，被那名经黑渊进入暗鸦星的**虚空黝黑人形**（四米高·惨白瞳·一气爆秒杀·触须读取记忆·学人类语言飞快·自称同「巴朵厄族」成长方式相似的「精神凝结体」）一击枭首、顺手屠灭整个牙骨部落、循「最近的人类文明」而去。

## 西港（西湾） · 巫女王摩利莎（二十一卷28–31章）

[美洛狄](../../人物/美洛狄.md) 的老家**西港 / 西湾**——三面环山（峭壁满布数万元素 / 咒术陷阱 + 琥珀状粘稠气体）、一面环海（12 座女巫瞭望塔 + 海下能量网）的天然要塞，是女巫在圣愈教会边境的钉子（圣愈教会鞭长莫及·心腹大患是南部「女巫之国」而非这里）。统治者 **[摩利莎](../../人物/摩利莎.md)**（五阶元素「巫女王」）**圈养人类**（买奴隶世代腐蚀其精神、奉摩利莎为活神·圣愈教会领地内 80% 流窜女巫出自西港）。苏晓接了悬赏 7000 金币（死活不论）的摩利莎那张通缉令、正用 [布布汪](../../人物/布布汪.md) 的 [狼族传家宝](../../力量体系/装备道具/index.md)（涂鸦定点单向传送）筹划潜入。

## 西港攻坚战 · 覆灭西港（二十一卷32–44章）

苏晓 D·暗杀清掉一座女巫瞭望塔潜入西港，进城就放 **冥界之焰·哈迪斯**（绿焰·烧城逼出女巫·当神流血便无人惧）——摩利莎却下令所有女巫退守城堡（护着「日阳转化水晶」这西港的根本）。苏晓组**攻坚小队**强攻：[阿姆](../../人物/阿姆.md) 撞门当主坦、自己用 [精英死灵召唤凭证](../../力量体系/装备道具/index.md) 召出狩猎者 + 2 生命祭祀、[布布汪](../../人物/布布汪.md) 偷袭奶妈、震爆弹 + 环断 + [绝影闪](../../力量体系/技能/index.md) 连斩——一层秒杀大批一二阶女巫、三刀连斩三名四阶女巫（咒术系的各种诅咒被刀术宗师 + 咒术抗性大半豁免）。西港六名四阶女巫死五名；二号人物 [卡伊](../../人物/卡伊.md)（金属系·有理想的女巫）被摩利莎见死不救寒心、咬断左臂割席而去。

**摩利莎 / 无冕领主决战**：摩利莎激活日阳转化水晶成「光之女巫」（五阶·耀环 / 炽阳·见死不救部下换力量），却迷之自信跟苏晓近战被打懵；水晶里封的 [无冕领主](../../人物/无冕领主.md)（剑术宗师）夺其身体迎战，苏晓被迫开**猎魔时刻**险胜（无冕领主因灵魂排斥自败）。苏晓斩门触发禁制、**破坏日阳转化水晶 = 覆灭西港**——得**开拓之源 +20%（累计 26.3%）+ 圣愈声望 12700 + 灵魂结晶大 ×10 + 传说级宝箱·猎**。[美洛狄](../../人物/美洛狄.md) 压在心头的旧主就此伏诛。

## 升红猎人 ·「旅行者」称号（二十一卷45–46章）

## 图书馆机密 · 死亡之地与摩伊家族（二十一卷47–50章）

苏晓在阵营商店购入传说级长皮衣 [狂猎之夜](../../力量体系/装备道具/index.md)（暗鸦为爱人所缝·法力转防御·替下旧的岚凯风衣 / 舍弃初级空间禁锢烙印），再用红猎人权限查图书馆机密笔记《死亡之地与摩伊家族》，揭开真相：

- **死亡之地在南部、摩伊家族世代守护**（守护的其实是死亡之地内的「某种东西」= 摩伊家族强大之源）。**摩伊家族**历代家主皆五阶女巫、直系皆女性、化身极优。这一代嫡系三姐妹：长女 **[摩伊·诺拉](../../人物/摩伊·诺拉.md)**（死亡之地唯一守护者·现今最强女巫·五阶元素）、次女 **摩伊·利莎**（= 苏晓已斩的 [摩利莎](../../人物/摩利莎.md)）、三妹 **[厄运](../../人物/厄运.md)**（天生怪胎·父母不认故名·五阶咒术·衔尾蛇诅咒源头）。
- **杀摩伊·诺拉必先杀厄运**：诺拉的「不死」其实是厄运的咒术能力所赋——破坏厄运身体使其沉寂半小时、诺拉便失去不死性（圣议员 [雪伊](../../人物/雪伊.md) 多年宿敌经验所证）。

## 夕阳红敢死队 · 深入南部（二十一卷50–54章）

## 抢摩伊家族传家宝（二十一卷55–58章）

入白森林（迷雾堡所在·满布女巫合成兽）后，苏晓逼问出 [美洛狄](../../人物/美洛狄.md) 的秘密——她当年偷的摩伊家族传家宝（短剑状·**完整时御寒极强**·进死亡之地不冻死的关键）虽弄丢、却**打碎一小块带在身上**（吊坠·近传家宝则发热）。循吊坠发热苏晓找到迷雾堡原址、从被女巫追杀的「伪主角」红发少年手里**抢走那枚黑蓝晶石（摩伊家族传家宝）**——这正是苏晓带美洛狄来南部的真正目的。

## 覆灭迷雾堡 · 斩摩伊·诺拉（二十一卷59–65章）

苏晓用 [布布汪](../../人物/布布汪.md) 的 [狼族传家宝](../../力量体系/装备道具/index.md) 把阿姆 + **瞬爆阿波罗**（升级版·范围 2 公里）传送到迷雾山谷上空——一颗阿波罗**整谷气化**、近百女巫连惨叫都来不及。仅剩半截迷雾堡里 [摩伊·诺拉](../../人物/摩伊·诺拉.md)（不死·多元素 + 权杖）现身，苏晓先用 [灵魂行者](../../力量体系/装备道具/index.md) 给她留下不愈伤、稳住她等夕阳红那边。[卡门](../../人物/卡门.md) 的敢死队同步在尘泥祭坛**化身成恶鬼破了 [厄运](../../人物/厄运.md) 的身体**（卡门等全员牺牲·骨骼崩裂而亡）——诺拉就此失去不死性。决战中 [无冕领主](../../人物/无冕领主.md) 再现、想夺诺拉之身（太阳焰 + 元素 + 部下生命力 + 摩伊血脉凑齐了他的转化仪式），但这次诺拉主导融合成功（传承剑术·样貌不变）；苏晓界断线缠斗 + 灵魂行者 + [无尽黑暗](../../力量体系/装备道具/index.md) 6 秒 ×3 拷问出死亡之地后斩杀她。期间虚空杀手 [葬戈](../../人物/葬戈.md)（巴京斯族·风王子派来）杀到，被布布汪 + 阿姆骗进 [负离子电浆超导装置](../../力量体系/装备道具/index.md) 困住。

## 死亡之地 · 刃之觉醒完成（二十一卷67–72章）

苏晓循情报登上北境**永冻海**的极寒冰山（零下 200 多度·御寒服一件件冻碎·靠摩伊家族传家宝吊坠 + 柱间细胞硬撑·阿姆无视低温拖他上山）——山顶冰洞里正是**死亡之地 = 刀魔能量封印装置**（摩伊家族世代守护、却打不开〔需青钢影激活〕）。苏晓注入 1000 克 [意识传导晶体](../../力量体系/消耗品材料/index.md) 驱动装置，把体内**刀魔能量剥离、注入斩龙闪**完成「**刃之觉醒**」——斩龙闪晋**史诗级**、觉醒「**魔刃**」（敌生命 ≤25% 即死·无视抗性/防御）、龙影闪变成苏晓的**自身能力**（不必握刀也能用）。从此他可随意使用吞噬之核（刀魔隐患拔除）。葬戈一路被陷阱坑十几次、追到极寒冰山被布布汪「智商碾压」遛住，苏晓返园后他困在北境（传送装置已坏·回不去虚空）。
