# 幽鬼大陆 · 五阶考核（苏晓第 24 个世界 · 3 星奖励世界）

[苏晓](../../人物/苏晓.md) 达成五阶晋升前置 9/10（超额）后匹配到的 **3 星五阶考核世界**（奖励世界·难度 +100%·**评分达 S+ 可得「100 点属性的灭法之影能力」**专属奖励）。超额补偿五选一中，苏晓选了**最硬核的「处刑机关副军团长」身份**（职务犯罪嫌疑中·有死亡风险）——搏 S+。同入此世界的有 58 名契约者，含 [国足三兄弟](../../势力/index.md)（黑暗势力）+ [咕噜](../../人物/咕噜.md)。

!!! tip "幽鬼大陆 · 世界观（二十四卷1章）"

    **祖尔帝国**——17 先王和谈建国（非战争统一·因有更大的敌人）·议会制 + 选举出的「君主」·首都祖尔城·蒸汽机 / 火车问世。人类与三种掠食性智慧生物长期共存：

    - **幽鬼**：智慧生物·极强掠食 / 共生性·嗜食人类（按**眼睛数量**判战力：1~3 眼普通 / 5~10 眼精英 / 10 眼以上极危·15 眼以上苏晓开猎魔时刻都打不过·**10 眼幽鬼 = 四阶契约者**）。能伪装成人类潜伏。食人增寿（常食人活 150 年+·不食仅 30~50 年·故永不停食人）。
    - **妖魔**：半能量 / 半实体·高智商 + 天赋·嗜食血肉与灵魂·**每个都是 BOSS 级**（≥10 眼幽鬼战力·无法从外观判断强弱）。
    - **诡兽**：次等智慧生物·攻击性强·可驯服·无法人工培育。
    - **觉醒者**：人类吃幽鬼新鲜血肉有概率觉醒（死亡率 70%+）·杀幽鬼变强·**觉醒度（被侵蚀度）越高越危险**（>65% 情绪不稳·>70% 蜕变期·完全觉醒即化为妖魔·意识死亡）。帝国主战力。「昔日屠龙少年终成恶龙」。

## 苏晓的身份与开局（二十四卷2–5章）

苏晓替换的是 **处刑机关副军团长「利·克拉塞」**（伪装机制连名字也换成库库林·白夜）——[处刑机关](../../势力/index.md) 的实际掌权者、觉醒者们的老大。开局即「职务犯罪嫌疑中」：前任副军团长利·克拉塞（觉醒度 86%·好色不下流）酒后对一名王裔的未婚妻 [简·维拉](../../人物/幽鬼大陆配角.md) 用强、又杀了跟踪而来的王裔灭口、却舍不得杀那患斯德哥尔摩症的少女留了活口。苏晓接手第一件事就是**炸死简·维拉灭口 + 切断所有线索**洗脱嫌疑，又与名义军团长 [南茜夫人](../../人物/南茜夫人.md)（毒蛇）、君主 [哈罗德](../../人物/哈罗德.md) 周旋。**晋升任务「鬼环」**（分环·第一环 = 清除祖尔城内由幽鬼组成的势力「[鬼环](../../势力/index.md)」≥60% 成员·惩罚强制处决）。

## 秘典系统（奖励世界专属·二十四卷7章）

进此世界的所有契约者都能召出**秘典**（金属封皮巨书）——杀幽鬼 / 妖魔掉「秘典卡」（宝箱类道具），可镶进秘典换**秘典功勋**、再用功勋兑换秘典首页十大宝箱（先到先得·各仅 1 枚）：**鬼之财宝**（450 功勋·保底 5 颗灵魂结晶〔完整〕）/ 贪婪（随机史诗装备）/ 植物精粹（不死酒·永久增益）/ 精湛（大师级能力卷轴）等。

## 鬼环 · 第一环推进（二十四卷5–24章）

- **斩鬼环刺客**：鬼环先派觉醒者 [卢卡斯](../../人物/幽鬼大陆配角.md)（觉醒度 93%）刺杀苏晓 / 南茜 / 君主哈罗德三巨头（三人都遇刺·唯苏晓全身而退）——苏晓青钢影克其能量武器、拷问出「鬼环」后斩之、掉首张秘典卡。
- **天才侦探入伙**：苏晓强征 12 岁天才侦探 [伊瑟薇·邦妮](../../人物/伊瑟薇·邦妮.md)（智商 170+·女版柯南）查鬼环踪迹、查「**十六眼室 / 不朽之王**」（鬼环真正目的）。
- **组「无眼」小队**：以觉醒者 [格斯 / 汉克 / 莱因哈特](../../人物/幽鬼大陆配角.md) 组秘密小队。靠地下室钥匙（三把合一·一把藏在副军团长办公桌的「幽鬼爪根雕」里）牵出君主线。
- **覆灭据点**：① 野猪庄园血战——苏晓寂灭枪 / [噬灭](../../力量体系/装备道具.md) 远程狙杀 33 只幽鬼、用**血之兽**把鬼环首领 [罪恶](../../人物/罪恶.md)（枪术宗师 Lv.27·能量枪「死刑·暮」·混血 BOSS）扑成重伤逃走、阻止「十六眼室」当晚成形（第一环 +21%·得名册）；② 占卜屋斩情报中转的 [乔安娜](../../人物/罪恶.md)；③ 伊甸庄园两枪击杀**先驱者** [埃迪爵士](../../人物/幽鬼大陆配角.md)（人类 / 幽鬼混血·鬼环资金提供者·气息与人类无异·第一环 +10%·得 9400 万金勒甩锅给君主）。

## 关键悬念

- **十六眼室 / 不朽之王**：鬼环的真正目的（眼室 = 镶满各类生物眼球的祭坛·极难破坏·与「不朽之王」相关·待解）。
- **混血「先驱者」**：幽鬼与人类的混血气息与人类完全相同（埃迪爵士即一例）——鬼环高层多为此类。
