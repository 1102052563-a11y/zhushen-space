# 强者争霸战（第32卷·虚空赛事·三大阵营·觉醒青影王之途）

[苏晓](../../人物/苏晓.md) 晋六阶后的重头戏——由 **虚空之树公证**的跨阵营大赛。对苏晓意义非凡：**夺前三即可让轮回乐园模拟「灭法殿堂」、觉醒奥义级大招 [青影王](../../力量体系/技能.md)**（灭法之影的终极能力·缺灭法殿堂无法觉醒）。

!!! tip "三大阵营 + 噩梦级第七争霸区"
    参与方分三大阵营：**乐园**（轮回 / 天启 / 圣域 / 死亡 / 圣光 / 守望…）、**虚空种族**（羽族 / 恶魔族 / 奥术永恒星 / 魔鬼族 / 星族 / 梦灵族…）、**超脱·原生世界**（陨灭星 / 风海大陆 / 夜惑女巫公会…）。苏晓被分进 **第七争霸区·暮色之岛**——「噩梦级」（轮回 + 死亡 + 羽族 + 陨灭星等最狠的三分之二都在此·56 阵营·巴京斯族 / 亡祭族当场弃权）。

## 赛制（五阶段 · 公证方虚空之树）

- **第一阶段 · 淘汰制（逃杀）**：3 天内在暮色之岛厮杀，**杀 1 人 = 1 积分**；积分前 5 晋级，其余「**3 天内至少得 1 积分**」者可被淘汰拿安慰奖、0 积分直接处决。无任何规则、强者生弱者死。
- **第二、三阶段**：过程较长 / 「很有趣」（接引人未细说）。
- **第四阶段**：30 万观众的角斗场（只有强者争霸战参与方可观战）。
- **第五阶段（最终）**：仅第四轮前十有资格——巅峰决战，胜者所得「足以改变命运、连各族老不死都眼红」。
