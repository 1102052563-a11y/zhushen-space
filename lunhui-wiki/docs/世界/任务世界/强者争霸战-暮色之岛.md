# 强者争霸战（第32卷·虚空赛事·三大阵营·觉醒青影王之途）

[苏晓](../../人物/苏晓.md) 晋六阶后的重头戏——由 **虚空之树公证**的跨阵营大赛。对苏晓意义非凡：**夺前三即可让轮回乐园模拟「灭法殿堂」、觉醒奥义级大招 [青影王](../../力量体系/技能.md)**（灭法之影的终极能力·缺灭法殿堂无法觉醒）。

!!! tip "三大阵营 + 噩梦级第七争霸区"
    参与方分三大阵营：**乐园**（轮回 / 天启 / 圣域 / 死亡 / 圣光 / 守望…）、**虚空种族**（羽族 / 恶魔族 / 奥术永恒星 / 魔鬼族 / 星族 / 梦灵族…）、**超脱·原生世界**（陨灭星 / 风海大陆 / 夜惑女巫公会…）。苏晓被分进 **第七争霸区·暮色之岛**——「噩梦级」（轮回 + 死亡 + 羽族 + 陨灭星等最狠的三分之二都在此·56 阵营·巴京斯族 / 亡祭族当场弃权）。

## 赛制（五阶段 · 公证方虚空之树）

- **第一阶段 · 淘汰制（逃杀）**：3 天内在暮色之岛厮杀，**杀 1 人 = 1 积分**；积分前 5 晋级，其余「**3 天内至少得 1 积分**」者可被淘汰拿安慰奖、0 积分直接处决。无任何规则、强者生弱者死。
- **第二、三阶段**：过程较长 / 「很有趣」（接引人未细说）。
- **第四阶段**：30 万观众的角斗场（只有强者争霸战参与方可观战）。
- **第五阶段（最终）**：仅第四轮前十有资格——巅峰决战，胜者所得「足以改变命运、连各族老不死都眼红」。

## 关键人物 / 接引

- **鼠大人（暴鼠 · 接引人）**：鼠头人身、扛能量霰弹枪的虚空强者——[布布汪](../../人物/布布汪.md) 旧识（虚空角斗场时帮苏晓挡过法师贤者瑟菲莉娅）；嗜酒，被布布汪用一瓶海贼世界「玛歌亚加」换得隐晦情报（「天气太热 / 海」）。
- **[争霸战群像](../../人物/强者争霸战群像.md)**：死亡乐园双子 / 羽族殇月 / 吟语族尼塔 / 陨灭星阿雷克 / 恶魔族蒙德 / 随缘施法者 / 夜惑女巫 / 魔鬼商人等。

## 苏晓在第七争霸区（暮色之岛 · 第一阶段）

- **开局秒鸟人 + 瓜分海岛**（三十二卷1–4章）：吟语族 **[尼塔](../../人物/强者争霸战群像.md)**（伪名「瓦戈」·能变任何人模样 / 气息·拉满仇恨）骗走奖励 [封之箱](../../力量体系/装备道具.md)；苏晓发现参战者实力参差（虚空中小种族多是来捞保底），与死亡乐园 **[双子](../../人物/强者争霸战群像.md)**（六阶竞技场第三·夺冠热门）**瓜分海岛**（左 / 右各清场）；斩陨灭星世界之子 **[阿雷克](../../人物/强者争霸战群像.md)**（古神力量侵蚀·靠心·魂·刃破其大招）—— **掠天惊澜任务推进到 96%**（每杀一名剧情主角 +完成度）。
- **M·暴君清场右半岛**（三十二卷7–8章）：苏晓登百米冰柱、用 [天行](../../力量体系/装备道具.md)（临时枪械宗师 Lv.10）+ 圣灵级狙击炮 **[M·暴君](../../力量体系/装备道具.md)**——尼塔把 30+ 参战者引到荒草地，苏晓一枪秒木灵族触发**高能状态**（法力造能量风痕子弹自动填弹）、「一人一枪 = 一个团火力」清场；羽族 **[殇月](../../人物/强者争霸战群像.md)** 装死、因羽族 + 恶魔族庇护被苏晓放过。

- **第一阶段晋升 + 休息区**（三十二卷9–10章）：暮色之岛清场结束（轮回 47 / 死亡 40 积分·苏晓队杀 85 人）、3 人全晋级。休息区是多阵营公证的绝对安全点（管家机器人由 [巴纳小人族](../../人物/布布汪.md) 特约·甚至插播广告）；遇法师贤者 **[瑟菲莉娅](../../人物/瑟菲莉娅.md) 的弟子 [狄琳](../../人物/强者争霸战群像.md)**（异元素 / 受导师虐待长大·奉命杀苏晓）；开 [封之箱](../../力量体系/装备道具.md)（50 灵魂结晶 + 机械图 + 树之生命配方〔体力 +3〕+ 高震钢矿脉）。
- **第二阶段 · 死亡屋（痛苦女王安娜）**（三十二卷10–23章）：第二轮抽到 **死亡屋**（[爱丽丝恶魔古堡](恶魔古堡游戏.md) 的高配版·主人 **[痛苦女王·安娜](../../人物/强者争霸战群像.md)** 绝对强大却公正·危险的是死亡屋本身）——6 参战者（苏晓 / 狄琳 / 殇月 / 尼塔 / 蒙德 / 圣域先驱老者〔秒死〕）。任务 = **找到「遗忘骨牌」**通关。机制：**[诅咒金币](../../力量体系/装备道具.md)**（抛正面增益 / 反面引「斧哥（暴食兽）」追杀·攒 [暗影石](../../力量体系/装备道具.md) + 幸运）、猎龙蛛引路（考洞察力·生死印记倒画当过路费）、亡者迷宫（蒙德越挨揍越凶）。苏晓**硬遛斧哥 20 小时**抗过追杀刷得 3 暗影石 + 幸运 +2；进黑雾木屋区斩亡者 **[大祭祀·奴埃](../../人物/强者争霸战群像.md)**（开关门战略）、靠 [白脂蜡](../../力量体系/装备道具.md) 驱黑雾深入，遇 **[无声者](../../人物/强者争霸战群像.md)**（缝嘴老太婆·卖白脂蜡指路）、**[圣女·特蕾西](../../人物/强者争霸战群像.md)**（灵魂割离的战争武器·念反咒语降伏）。

- **死亡屋通关 + 真相**（三十二卷23–29章）：与圣女交易得超大 [白脂蜡](../../力量体系/装备道具.md) + [秘迹宝石](../../力量体系/装备道具.md)（圣灵）+ [处决之刃](../../力量体系/装备道具.md)（圣女自愿投黑雾解脱）；黑雾里斩尼塔（放走殇月·获羽族 + 恶魔族两老家伙送的圣灵级 [樱风酒](../../力量体系/装备道具.md)）；引硬核信徒 **[摩匹](../../人物/强者争霸战群像.md)** 给 **[守雾人](../../人物/强者争霸战群像.md)** 换 **遗忘骨牌通关**。揭死亡屋真相：处决之刃是**制约死亡屋主人**的（非杀 [痛苦女王·安娜](../../人物/强者争霸战群像.md)·苏晓拿它换 **[航海罗盘](../../力量体系/装备道具.md)**〔指针指心中所想·一次性〕）；死亡屋有意识、在等能彻底掌控它的人（安娜将成为那个人）。最终之所选了件神秘奖励（玻璃瓶灰黑细沙·留待第三轮）。
- **第三轮 · 维京之海（开篇）**（三十二卷29–30章）：返园统计第二轮（己方 [牧花](../../人物/强者争霸战群像.md) 抢世界之核被判出局 / [巨杰士](../../人物/强者争霸战群像.md) 被 17 人围攻战死）；**第三轮 = 所有参战者同区域**「维京之海」（苏晓推测死亡屋那件神秘奖励将大放异彩）。
- **维京之海 · 厄运号 + 黑海王 + 找古神**（三十二卷31–39章）：第三轮 = 15 天内抵世界中心「**洛什岛（恐怖岛）**」的海盗世界（无世界之源·有宝箱）。死亡屋那件「玻璃瓶灰黑细沙」化成有生命的劫掠船 **[厄运号](../../力量体系/装备道具.md)**（吞超凡海兽成长·毁灭重炮轰船）——苏晓靠它机动碾压、撞沉幽灵船海盗团、钓章鱼喂船。在海盗岛**环瘠岛**重遇 [狄琳](../../人物/强者争霸战群像.md)（其「黎元素」是 [瑟菲莉娅](../../人物/瑟菲莉娅.md) 专门针对灭法者培养·却两招就被青钢影崩碎、心态崩）+ [休格](../../人物/强者争霸战群像.md)（随缘放走狄琳·替 [风王子](../../人物/风王子.md) 转告「近期别去恶魔古堡」）；从传说海盗 **[黑海王·阿加蒂](../../人物/强者争霸战群像.md)**（血鸦 / 红鸦症）+ 斩狂信徒 [费尔德](../../人物/强者争霸战群像.md) 套出古神线索：**黄昏之神**（「绝望的起源」）在 **弗因丁王国·海妖港**。
