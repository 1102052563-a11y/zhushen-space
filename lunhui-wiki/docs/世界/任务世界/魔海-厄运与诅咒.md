# 魔海·厄运与诅咒（第41卷·苏晓晋八阶后的首个世界）

[苏晓](../../人物/苏晓.md) 正式晋升八阶后进入的**首个原生世界**——他不是随机被传送来的，而是被一艘名为**厄运号**的船「**呼唤**」而来（强者争霸战期间他曾当过厄运号的船长、并因此被**魔海诅咒**）。进入本世界的同时，他还**同步进行 200 点真实属性壁障的突破**（力 / 敏 / 体 / 智四主属性同破·累积难度 **235%**）。本世界 Lv.53~77、全开放、世界之源 0%、**食物与淡水资源被封禁于储存空间内**（逼契约者融入海盗世界）。

## 世界背景 · 厄运号 / 大漩涡（世界简介）

魔海正逐渐淹没所有陆地，世界史围绕**「大漩涡」**的传说展开：

- **破浪号 → 幸运号 → 厄运号**：航海历 279 年，五桅船「**破浪号**」直奔魔海尽头的大漩涡求陆地、满船骸骨而归（船长成骸骨仍屹立船首、紧握抵达大漩涡的海图）；几十年后一名好运却行骗的年轻船长把它改名「**幸运号**」再航、被大海盗阉割扒皮惨死；那大海盗夺船冲入大漩涡——**一年后海面退下、露出三片大陆**。当那位无名大海盗被遗忘、永沉海底、再也压不住海底之物，「幸运号」便化作**厄运号**冲上海面。
- **厄运号 = 死灵水手的执念**：船上是最初那批「渴望陆地的航海者」死灵——如今陆地虽现，它们却被魔海束缚、无法踏足陆地半步，遂化作屠戮海上一切的噩梦（连海中巨兽都被它奴役）。最终魔海把厄运号吞入海底——但**它已有了新的主人（苏晓）**。

- **规格**：三桅杆船 →（解一阶封印后）四桅杆帆船；生命值 47500、防御 162、耐久 217/260、航速 +140%。
- **船员**：诅咒人（介于生死之间、被厄运号奴役、唯船长命令不可违）——初始 37 名，解一阶封印后**提升至 350 名**且战力大增（三重封印全解约上千名封顶）。

- **诅咒同源**：苏晓被魔海诅咒（左手指尖起蔓延的黑色纹线·青钢影只能遏制不能根除），根源正是「厄运号船长」身份——**解除厄运号的诅咒，他所承受的诅咒便不攻自破**（但硬毁厄运号可能反把全部诅咒转移到他身上）。他打算凑够金币解封厄运号 → 攻沉船岛 → 一并了结诅咒。

## 泯光岛（厄运号的补给地 · 41卷12–20章）

厄运号船首挂上骷髅提灯、驶入黑雾漩涡即可瞬移抵达的**独立异空间**——上空悬着半破的血月、九成区域被黑雾笼罩，是厄运号「诅咒人」平时沉眠之所（船长需要时才召少量到船底仓）。**敌人被拖进泯光岛必死**（诅咒人太多），但厄运号进岛途中极脆弱、遭一击就有 95% 概率被毁。岛上有码头 + 几间木屋（多被魔海封印·随厄运号解封而开）：

- **杂货铺**：老板正是苏晓的老友、地精裁决者 [凯撒](../../人物/炎灵星/凯撒.md)（用道具「协助式进入」魔海）——卖厄运号专用**特制炮弹**（毁灭重炮的弹药·1400 金币/箱）与船饰品：**厄运血帆**（降敌士气 + 提速·32000）、**怒鲨撞角**（16% 完全割裂敌船·25000）、**无畏船舵**（船体防御阶位 +1·42000）、**诅咒之心**（船员 +50、全属性 +3 / 生命 +3500·10 万）等 27 件。苏晓靠凯撒开的「赊账」权限几乎清空库存、欠下 −29.5 万金币。
- **黑鸦屋**：[泯光岛巫祀](../../人物/魔海/泯光岛巫祀.md)（厄运号第三任船长的爱人·老太婆）居所——赠 [黑鸦戒](../../力量体系/装备道具/index.md) 并为其镶诅咒石。
- **诅咒船工**：十几名生前是船工的诅咒人常驻码头，负责给厄运号安装船饰品 / 修理船体（比「吞噬敌船」更稳，吞噬用多了会掉耐久上限）。

## 黑咒乌鸦 / 黑鸦戒 / 海魂石 · 登岛方案（41卷14–20章）

- **[黑鸦戒](../../力量体系/装备道具/index.md) + 黑咒乌鸦**：戒指融入苏晓左臂的诅咒黑纹、可从泯光岛召唤「黑咒乌鸦」（初始 1.5 万·镶诅咒石递增·镶 2 颗达 8.5 万）。乌鸦=**士兵类单位**，两大能力：①战场上空盘旋、遏制敌方生命恢复②每次攻击累积「诅咒值」满 100% 即死判定。数量达标即触发苏晓的 [战争领主](../../力量体系/称号/index.md) 称号五档加成（士气 +70 / 全真实属性 +20 / 生命上限 +45% 防御 +30 / 全能力等级 +10 / 血魂之力）——加持给诅咒人 + 乌鸦，战力暴涨。

## 西海王海战 · 双塔女帝「三支箭」真相（41卷29–32章）
