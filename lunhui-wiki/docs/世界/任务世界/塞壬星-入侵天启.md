# 塞壬星·入侵天启（第29卷·晋六阶考核·乐园入侵·天启乐园原生世界）

[苏晓](../../人物/苏晓.md) 的**晋升六阶考核**——首次「**乐园入侵**」：被轮回乐园伪装后空降到**天启乐园的原生世界塞壬星**，去掠夺天启乐园的资源（世界争夺战刚结束、天启占了便宜变更富，轮回乐园上门「砍一块肉」）。

!!! tip "乐园入侵机制（晋六阶考核）"
    晋六阶考核「乐园入侵」的几条核心规则：

    - **伪装进入**：被传送到大气层外、用陨石包裹砸落（97% 判定为自然现象、不被世界排斥）；轮回乐园伪装烙印 / 宝箱、临时隐没苏晓血气（全力战斗会暂失效）、储存空间临时缩到 2.9×0.6×0.5 米。
    - **战斗天使追杀**：苏晓被伪装成**高危违规者**，会被天启乐园的 **[战斗天使](../../人物/塞壬星群像.md)**（相当于轮回乐园猎杀者）追杀，可规避或击杀。
    - **入侵度系统**：签天启契约 / 组队 / 杀战斗天使等提升入侵度（达 10% 开天启世界频道·20% 额外奖励）；杀敌方契约者几率掉 **猩红卡**（★~★★★★★·按敌人实力定）。
    - **紧急脱离**：仅天启乐园识破苏晓是入侵者并试图强制处决时被动触发（触发即晋升失败）。

## 世界设定（塞壬星）

- **难度** Lv.23～Lv.52（原生世界·99% 区域 Lv.23~48·少量六阶危险区）；**世界之源 = 掠夺性质**（获取极难·结算奖励 +50%）。
- **灾难日（157 年前）**：原是科技发达的宜居星球，某天**太阳不落（「阳夜」）**——七天里地表温度飙到 157°，289 亿人口只剩 2.7 亿，空气中诞生新能量、出现**不死者**，人类文明崩塌。第八天太阳落下、气候恢复，幸存植物 / 野兽疯长进化出智慧，新秩序建立。
- **今日塞壬星**：文明大半遗失、机械科技发达；**人类**靠超凡力量 + 机械科技（械铠 / 磁力悬浮 / 蒸汽枪械）战斗，抱团建避难所势力（如 **[火颅帮](../../人物/塞壬星群像.md)**·17 号避难所·百万人口）；**[不死者](../生物万族/index.md)**（异化人类·数量庞大·可繁衍·有精英个体如哀嚎女王）盘踞废城；变种兽 / 异种兽登顶食物链。
