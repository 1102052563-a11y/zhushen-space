# 深渊世界·神灵挽歌（第31卷·半开放原生世界·圣裔 / 灭法者 / 猎杀希）

[苏晓](../../人物/苏晓.md) 晋六阶后的第二个世界——**三乐园共有的半开放原生世界**（轮回 / 天启 / 死亡乐园·难度 Lv.47～59·世界之源掠夺性 = 三乐园抢蛋糕）。本世界**双重主线 + 邻近强者争霸战**（奖励额外 +10%）。

!!! tip "半开放原生世界 = 三乐园抢蛋糕"
    几个乐园共有一个原生世界：**天启乐园**擅开发资源（蛋糕本体）、**轮回 / 死亡乐园**契约者强（来抢）。谁先破坏对方的「**时空枢纽装置**」（类似临时世界之核），谁就能独享世界。本世界 [苏晓](../../人物/苏晓.md) 刚来就撞见天启乐园矿工挖矿（「可持续压榨」分成而非杀鸡取卵）。

## 双重主线

- **主线 1 · 破坏时空枢纽装置**：摧毁**死亡乐园**藏在本世界的「时空枢纽装置」（极隐秘·引导式任务·自行找线索）。
- **主线 2 · 猎杀希**：击杀无编号契约者 / 超脱者 **[希](../../人物/深渊世界群像.md)**——实为揪出**她背后那个神秘存在**（屡次隐匿希踪迹、却屡被轮回乐园夺走时空之力·处决希远不如夺时空之力重要）。
- **混入帝国的引导环**：苏晓选「高风险」打法跳过前三环、直奔**第四环 = 杀圣裔王 [亚历山](../../人物/深渊世界群像.md) 取其心脏（完整度 ≥50%）**（难度 Lv.56~63·七阶级硬莽难度·苏晓选搭老公爵的线智取）。

## 世界设定（深渊世界 · 神灵挽歌）

- **噬神时代 → 不死诅咒**：本是低阶神灵守护的「神之国」（圣战封印深渊怪物）；后深渊蛊惑凡人**斩神夺力**（噬神者）、神灵被屠戮殆尽。最后一名死亡之力神灵 **狄尼**临终向虚空存在祈求**诅咒所有噬神者永不得死**——这就是**圣裔（不死人）**的由来。
- **圣裔（不死人 / 圣裔）**：生命能量会缓慢从体内飘散（生与死不兼容）、流尽则变成「只知跪地向神灵忏悔的白痴」；须**摄取同族生命力 / 鲜血**维持意识（杯中「猩红之酒 / 绯红恩赐」= 同族生命力酿的酒）。**被斩碎仍会真死**（非真不死）；诅咒可遗传，故圣裔严控后代数量、数量过多就暗中自相残杀。
- **塞拉帝国 vs 北部联盟**：圣裔靠武力建**塞拉帝国**（农耕 / 畜牧发达·燧发枪冷兵器混合·杀人可花金镑赎罪）、与苦寒善战的**北部联盟**长年开战。权力中心 **圣廷**（圣裔王 + 五公爵决策）。
- **深渊（祸根）**：当年被低阶神灵冰封的怪物巢穴——一切的源头（圣裔们也忽略了它仍在）。
