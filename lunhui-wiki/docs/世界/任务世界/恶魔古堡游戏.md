# 恶魔古堡游戏（爱丽丝·强者之证明）

[苏晓](../../人物/苏晓.md) 为 80 点壁障的最后一关「**强者之证明**」而进入的特殊**封禁世界**（第十四卷「恶魔古堡游戏·强者之证明」）。这不是普通衍生世界，而是 [爱丽丝](../../人物/爱丽丝.md) 用 [爱丽丝的邀请函](../../力量体系/装备道具.md) 召集契约者来玩的「**恶魔游戏**」——成功通关即由轮回乐园授予「**强者之证明**」。

## 世界设定（十四卷1章）

- **封禁世界**：难度 Lv.1~?·**世界之源 0%**（无法获取）·**绝对封禁、任何方法都无法离开**·**无主线任务**、有 **147 个支线任务**（需激活）·**无违规者**。入场时苏晓被空间之力绑在铁柱上（约束期）。
- **爱丽丝**：恶魔古堡的**女王、一切的主宰**——**不可对抗剧情人物**，在古堡内力量近乎神灵。她无数岁月来反复接待游戏参与者、孜孜不倦地欣赏参与者的喜怒哀乐，并把自制宝物散落古堡各处当「答谢」。**关键规则**：不可讨好她、只有「公平的游戏」才能让她露出笑颜——**一定要让爱丽丝微笑；若她愤怒或伤心，呵呵呵呵……**（后果不堪）。
- **狼管家**：「**公平的裁决者**」（爱丽丝造物）·中立·不可对抗。
- **古堡恶魔奴仆**：古堡的原主人、被爱丽丝夺走三分之一大脑而失语·成了她的奴隶苦力·中立·不可对抗。
- **参与者**：预定 7 人，因发现 3 名参与者是同一冒险团成员、补到 **9 人**实际参与。游戏一旦开始无法中断。

## 9 名参与者（十四卷1章）

能进恶魔古堡的都不是咸鱼（须经战争世界存活 + 80 点属性 + 突破身体极限 + 20 万币 + 灵魂结晶〔中〕×15 兑邀请函的层层筛选）：

- **[苏晓](../../人物/苏晓.md) + [布布汪](../../人物/布布汪.md)**（布布汪无座位·不参与游戏）。
- **[国足三兄弟](../../人物/国足三兄弟.md)**：苏晓旧识·三圣骑士·能打能抗能奶（与布布汪交情好）。
- **猫女**：金发大胸御姐·猫耳猫爪（本体猫·火影世界曾以猫形态被布布汪叼到苏晓面前·隐匿极强、苏晓当时没认出是契约者）·从不主动出头（苏晓评其危险度第 3）。
- **皮胖**：200 斤胖子、知名「游戏编制者」·肉食不离手——实为近战高手（脂肪如水流涌动·疑 [武装色霸气](../../人物/index.md) 未硬化）、不怕死的疯子（苏晓评危险度**第 1**）。
- **金丝眼镜男「沃什」**：文质彬彬、第一时间宣称「辅助系」——实则进攻性最强、明哲保身（拉拢众人去排挤阴郁男·疑魅力特长 / 控制 / 召唤 / 精神系·苏晓评危险度第 2）。
- **侦探小萝莉**：鸭舌帽·自称辅助系（苏晓评第 5·被沃什拉去当帮手）。
- **阴郁男**：一脸「谁都欠他几个亿」·疑近期遭背叛、本能不信任 / 排斥他人（被众人默认当「探雷的倒霉蛋」·苏晓评第 4）。

## 一层「命运」游戏（十四卷3–5章）

由 **狼管家**（黑燕尾服狼头人·公平裁决者·气息深邃如深渊·苏晓都不敢招惹）主持：金色左轮「**命运主宰**」（**对每人只有一发子弹**·评分极高的金色装备）——参与者拿起它对自己头部扣扳机、「**想不死就选择你们的命运**」（墙上几百件带血装备是历届失败者「送给爱丽丝的纪念品」）。完成者由狼管家贴符文卡给「**爱丽丝的祝福**」奖励：

- [苏晓](../../人物/苏晓.md) 那枪**真响了**（次级无敌护盾挡子弹 + 狼管家手刀消冲击），因「选择命运由自己主宰」得**幸运永久 +1**（最难提升的属性）。
- [国足三兄弟](../../人物/国足三兄弟.md) 连开三枪空枪、得「祝福（友情）·**生命链接**」（三人任一生命 <30% 时承受伤害 70% 由另两人分担·永久·对能奶圣骑士是质变）。
- 其余人也都空枪——**8 人空、唯独苏晓那枪响**（运气差到让众人深思）。狼管家透露「**前三场游戏固定规则、之后自行探索**」。

## 一层探索 · 48 囚徒 + 介绍信（十四卷5–9章）

- **第一轮淘汰**：阴郁男最后那枪**真响了、被命运主宰爆头**（防御技能全无用·头颅整个爆开）——第一轮淘汰一人。**恶魔奴仆「嘎噜」**（三米灰黑人形·秃头·头上犄角已被锯断·古堡原主人·被爱丽丝夺三分之一大脑而失语、只会念「嘎噜」的苦力）扛走尸体。
- **一层 = 牢房区**：通过第一轮可在一层**自由活动三天**。一层是上千平米空地（几十石柱·西侧通上层的石梯已封锁）+ 南北几十间**牢房**——关着 **48 名各世界的囚徒**（种族 / 能力各异·不是谁都有资格被关这）。**囚徒靠摄入特定东西逐步剥离封印、恢复自由**（安莉亚要糖果、某蝙蝠要人血、某水牛要圣物〔装备〕、[莎](../../人物/莎.md) 要纯净灵魂）；解封前不能擅离房间、否则被狼管家清理。狼管家 / 嘎噜不是囚徒、是爱丽丝直属的「工作人员」。
- **囚徒分三类**：① 固执（不愿听爱丽丝摆布·无法交涉）② 认命（只求早日解封·如吃货小女孩 **安莉亚**）③ 纠结（如 [莎](../../人物/莎.md)）。能否交涉**既看魅力属性、也看「相性」**——苏晓魅力只 8、与「凶险」囚徒相性反而合（怂囚徒不理他、凶囚徒肯回应）。
- **安莉亚的介绍信**：小女孩囚徒安莉亚（浅绿眸·吃货仓鼠·吃糖解封·每天限三颗）——苏晓喂光布布汪的糖也只换来一句「你是好人」，遂用屠戮威胁逼她吐出 **56 封「介绍信」**（白色·评分 2·**提升古堡剧情人物好感、让无法交涉者变可交涉**）。靠这叠介绍信，他硬把固执的 [莎](../../人物/莎.md) 烦到肯交涉。

## 古堡五层结构（十四卷18章·莎 / 狼管家透露）

对游戏者而言古堡**共五层**，每层**独立封锁、无退路**（除非爱丽丝 / 狼管家指引、否则无法在古堡内穿行·游戏没有回头路）。**前三场固定规则**（① 命运 ② 美食天国 ③ 欺诈）、四五层自由度高——**通过五层即通关、返回乐园得「强者之证明」**。

## 第二场「美食天国」（二层·十四卷15–18章）

二层是「美食天国」，[爱丽丝](../../人物/爱丽丝.md) 的幻象现身亲自主持。游戏 = **吃光餐盘**（牛排 / 甜点 / 红酒）：

- **毒龙肉**：牛排是「**毒厄之龙**里脊肉」——带「颤栗猛毒」（真实魔法伤害·完整版闻一下都毒死人·每秒按肉量扣生命）。[苏晓](../../人物/苏晓.md) 靠 [绝魔体质](../../力量体系/技能.md) 免 40% 魔法伤害（每秒 6%）、一口肉一口 1 号药剂；皮胖吃大块每秒 35% 差点死。
- **血腥玛瑙**：红酒喝下 20 分钟**无法治疗 + 易伤（受伤 ×2）**——比毒肉更恶毒（爱丽丝亲手斟、笑吟吟）。
- **吊死机制**：可问爱丽丝「**能否被吊死**」——她**拒绝 = 直接通过本场 + 奖励**，**同意 = 真被绳索吊死**（国足老三申请被拒、顺利通过；侦探小萝莉申请被同意、靠「替死娃娃」才保命）。其余询问（如「能否休息」）一概无效。
- **结果**：8 人全过（猫女赌消化能力、胃里白丝网包毒肉、7% 濒死过关；金丝眼镜男用三天帮狼管家做事换的「权限」豁免）。苏晓**通过**得**毒抗永久 +15% + 每分钟生命恢复 +4**；他切毒龙肉喂 [布布汪](../../人物/布布汪.md)（吞超高等生物血肉、属性大涨）。

## 第三场「欺诈」（三层·十四卷18–19章·开始）

三层是复古公寓（客厅 + 8 间编号房 1~8·号越前设施越好·抽房卡分配）。第三场游戏 = **「欺诈」**——真与假、谎言与欺骗，核心问题只有一个：**谁才是爱丽丝**。

- **规则（十四卷20章）**：抽房卡时已有一名契约者随机抽到「**直接通过资格**」，**爱丽丝伪装成那名契约者**（临时获其全部记忆·分身**五属性 80 点、全技能 Lv.40**）——找出爱丽丝即通过本轮。**黑夜 2 小时**（只能待自己房间·爱丽丝可自由进房杀人）/ **白昼 1 小时**（自由行动）·合 3 小时 = 1 游戏日·**7 个游戏日内找不出则失败、狼管家肃清三层所有人**（含那名幸运者）。最初的幸运者会在第 3 / 7 白昼提供「无身份 / 无样貌」的图案线索。
- **质疑权**：每人**每白昼 1 次**（须所有存活者在场 + 白昼）·质疑成功则游戏结束、失败则继续。爱丽丝会设法减少众人的质疑次数。
- **天花板血书**（前游戏者遗言）：「三层不要相信任何人（哪怕兄弟 / 恋人）·**实力才是硬道理·爱丽丝的分身并非无法对抗**」；另有「爱丽丝是左撇子 / 讨厌麝香 / 没有善恶观 / 房间不隔音 / 使魔不可信、仆从或更高的从者可信 / 交换房卡前别进别人房间」。
- **第一白昼**：苏晓**思路清奇**（别人想规避分身、他想弄死分身），与皮胖以武力威压定「排除法」。让国足老三**自我质疑**、公告「失败、非爱丽丝」——既验证「自我质疑有效」、又一举排除形影不离的国足三兄弟（爱丽丝藏不进他们里头）。黑夜降临，1 号房侦探小萝莉的门锁自动开、现出个卖蜡烛的黑袍老婆婆。

- **第一黑夜 · 苏晓硬怼分身（十四卷23–24章）**：[爱丽丝](../../人物/爱丽丝.md) 分身（伪装者·本体 3% 实力·五属性全 80 + 四专精 Lv.40·全能战力）第一夜来袭 [苏晓](../../人物/苏晓.md)——苏晓靠全开的 [直感](../../力量体系/技能.md) 灵活躲避、斩她六刀报废一件金色铠甲，分身没能杀掉他、反陷劣势。**她袭苏晓是私怨**（曾被灭法之影吊死在虚空台·见 [爱丽丝](../../人物/爱丽丝.md)）。
- **博弈 · 惊天逆转（十四卷25–27章）**：游戏结束的唯一方法是**白昼指认出爱丽丝**（白昼杀人无用·杀错还失质疑权·黑夜杀掉则她白昼无法露面）。规则倾向爱丽丝（7 日不指认出她即她胜）。第一夜袭苏晓让爱丽丝一度劣势——可第二夜她改去「享受游戏」：把推理高手侦探小萝莉**扒光当一小时抱枕**（没杀·她事后迷之脸红喊「爱丽丝小姐姐」被苏晓抽醒）、**杀了搅局的猫女**（殁）、还让皮胖 / 眼镜 / 国足老大老二「违规」第三白昼禁出房——**一夜把契约者方逼入大劣势**（能自由行动仅苏晓 / 侦探小萝莉 / 国足老三、有质疑权仅苏晓一人）。第三白昼放出幸运者线索：一个看不懂的「**O＊**」。

- **通过第三场 · 真相（十四卷28–29章）**：[苏晓](../../人物/苏晓.md) 与国足老三破出「O＊」= 破碎眼镜 + 破碎蛋——爱丽丝伪装的是**国足老大**（用音乐骚扰逼猫女换房号牌、真国足老大进 6 号房即被替换）；她第二夜杀猫女（已是 5 号）后取回原属自己的 5 号号牌，故苏晓**质疑 5 号成功**、通过第三场（得**潘多拉魔盒**·金色一次性·可能出装备 / 血统、也可能出诅咒 / 死灵）。9 进 7、存活率名列前茅。

## 第四场「强者之森」（四层·十四卷31–32章·开始）

四层是「**强者之森**」（人造太阳的森林·面积极大）·**求生竞技**——7 名游戏者在森林里**生存 5 天**、但**只 6 个通关名额**（积分最低者淘汰 = 死）。补给仅木屋里的有限食物 / 水 / 外伤药。三种积分途径：① 杀森林里来自各星球的生物（1~2 分）② 杀「**客人**」（爱丽丝刚抓的囚徒·杀掉游戏者即可重获自由、故会主动来袭·2~3 分）③ 杀爱丽丝分身（1000 分）；也可**自相残杀**（死一人就够 6 名额）。**积分最高者可在爱丽丝的藏宝库（1582043 件藏品）任选一件宝物**。狼管家主持、恶魔奴仆嘎噜负责清理违规者（爱丽丝分身也会参与游戏）。
