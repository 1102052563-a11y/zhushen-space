# 东京喰种（任务世界）

苏晓的**第二个任务世界**，以漫画《东京喰种》为蓝本（二卷1章）。与海贼王的单人试炼不同，本次**大批契约者同时进入**（二卷1章）。

## 世界信息
- **难度**：LV.3 · 困难（因普通契约者实力被拉低而下调）（二卷2章）。
- **简介**：现代东京潜伏着食人怪物 [喰种](../生物万族/index.md)——外表与常人无异，体内 Rc 细胞活跃则赤眼，须吞食人类维持 Rc 细胞活性；Rc 细胞赋予其远超常人的身体素质（二卷2章）。
- **语言**：未掌握则消耗 100 乐园币自动掌握（二卷2章）。
- **开局**：时间线被**重置**（剧情回到最初），消耗乐园 0.03 盎司**时空之力**——故本世界契约者**最低综合评分需达 C 级**（二卷1章）。

## 任务
- **主线（连锁任务）**：加入 **CCG（喰种搜查官）** 或 **喰种一方（青铜树 / 小丑）**，选边后激活后续任务。完成 **3 次**主线后可选回归乐园（猎杀者须先完成猎杀任务）；完成越多综合评价越高。时限 48 小时，失败扣 13000 乐园币（不足扣装备、再不足等死）。苏晓选 **CCG**（二卷2章）。
- **猎杀任务**：清理违规者 **12470 号契约者**；进入其 1 公里范围内追猎功能自动开启。无惩罚、无时限（苏晓疑有隐情）（二卷2章）。

## 关键人物
- 喰种：[神代利世](../../人物/神代利世.md)（S~SS 级）、[壁虎](../../人物/壁虎.md)（虐待狂）。
- 契约者：寒鱼（7 人临时冒险团首领）、[希珞珞](../../人物/希珞珞.md)（职工者，同入此世界）等（二卷1–3章）。
- **命运之子**：本世界为金木研（前期尚弱）；因苏晓这等「怪物级契约者」入场，乐园为平衡剧情，开局即放出两名 S 级喰种（二卷1–2章）。

## 苏晓在本世界（进行中）
化名「白夜」。开局即在利世吃人的大楼内，一脚踢飞神代利世、两刀（**首次开** [青钢影](../../力量体系/技能.md)）将壁虎斩至半死，碾压级展现实力，准备借杀喰种加入 CCG（二卷2–3章）。

## 进度（截至二卷·3章）
苏晓压制壁虎、锁定神代利世；尚未正式加入 CCG、未触及 12470 号猎杀目标。
