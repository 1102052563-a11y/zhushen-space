# 全职猎人（绅士同盟）

[苏晓](../../人物/苏晓.md) 的**第 10 个衍生世界**（第十卷「猎人·绅士同盟」），以漫画《全职猎人》（Hunter×Hunter）为蓝本。半开放世界·**难度 LV.5～??**·世界之源 0% 起·消耗时空之力 0.019 盎司·**本世界最低综合评价须达 D+ 级**。

## 世界信息
- **简介**：浩大而危险的世界——财富 / 美食 / 珍宝 / 秘境，敢探索就有机会；追逐各自所求的人有个共同称呼：**猎人**。
- **猎人执照制度**：猎人协会每年办严格考试、合格者得执照（可免费用 95% 国家公共设施等特权）；分美食猎人 / 珍宝猎人 / 珍兽猎人等。
- **念能力**：深层操纵生命能量的能力——四大行「**缠 / 绝 / 练 / 发**」，以「发」为主；发分六系：**强化系 / 放出系 / 变化系 / 操作系 / 具现系 / 特质系**（不同系战斗方式迥异）。
- **六块大陆 = 猎人世界的「现阶段」**：苏晓所在的「**僈路比安大陆**」是六块大陆之一·六块大陆都有人类文明、交通便利、**热武器极先进（核弹应有尽有）**；但六块大陆只是**黑暗大陆的一个小边角**——黑暗大陆太恐怖（那里混不下去的种族到六块大陆都能称王称霸），以现今武力根本无法向外探索。
- **蚁王**：原著最强剧情生物之一·强在身体素质（中一枚核弹即濒死·防御没强到绝望·但速度太快）。**此时期蚁王尚未诞生**——故苏晓在这一时间点虽有死亡风险、但对大多数对手有一战之力。

## 苏晓的开局（十卷1–2章）
- **传送误差落错城市**：因卷入「异次元乱流」（正常宇宙现象·修正需大量时空之力、会把最低评价拉到 A−，故乐园不修正）——预定落点**友克鑫市**、实际落在 **1353 公里外的齐鲁市**（一座赌城）。苏晓在高空坠落，凭乐园「保证安全抵达」的条例（15794 条）被托住、最终砸在一辆甲壳虫车引擎盖上、踩着引擎盖当滑板着陆。
- **146 名契约者扎堆齐鲁市**：传送偏差令本批进入猎人世界的契约者都集中在齐鲁市——苏晓视之为「清理违规者的天赐良机」。

## 主线与猎杀任务
- **主线·抵达友克鑫市**（Lv.2·6 自然日·奖 1000 币·罚扣 1000 币 + 强制回归无收益）：友克鑫市是繁华糜烂、由多个黑帮掌控的城市（拍卖会将在此开），苏晓须在拍卖会开始前赶到。
- **猎杀·清理绅士同盟**（期限 = 离世前·奖 **黄金荣誉勋章 ×1 + 衍生世界多选机会一次**〔下次进世界前可在三个里选一〕）：**[绅士同盟](../../势力/index.md) = 5 名违规者抱团组成的组织**——疑似察觉到「猎杀者」的存在才抱团（5 人散在世界各处则极难清·齐鲁市一锅端是良机）。

## 势力 / 货币（齐鲁市背景）
- **V5（五大政府）**：猎人世界并非无政府——**五个大型政府处于联合状态（类似联合国·统称 V5）**，掌握世界最先进科技、是真正恐怖的存在（后来嵌合蚁入侵 V5 都没出面、只让猎人公会处理）。**猎人公会**只是 V5 承认、给予特权的民间武力组织（凌驾大部分国家机构·会长曾与蚁王同归于尽）。**被 V5 盯上则五阶以下契约者必死无疑**——故契约者在猎人世界不敢太张扬。
- **十老头 / 阴兽**：齐鲁市（无政府的赌城）由 **[十老头](../../人物/比克.md) 势力**掌控——这十人管着六块大陆的所有黑帮、手下成千上万打手、**每年的友克鑫拍卖会就是他们组建的**。在普通人世界算强、在念能力势力中只算中下游；最强部下是十名念能力者「**阴兽**」（一头老 = 一头阴兽·相互制约·苏晓单挑几只无压力）。**富有但实力不足**（故被幻影旅团盯上）——代表人物是 [比克](../../人物/比克.md)。
- **友克鑫拍卖会**：每年 9 月 1–10 日在友克鑫市开、**世界最大的拍卖会**——奇珍异宝总价超乎想象；8 月起数百万人（统计超 500 万）涌向友克鑫，全城变成大集市。
- **戒尼**：猎人世界货币（约 16 戒尼 = 1 RMB）。乐园币换戒尼有额度（100 币换 100 万戒尼·超额则费用剧增——因为足够戒尼能买小型核弹）。苏晓进场换了 100 万戒尼、又在「黄金亨特」赌场凭感知赢了约 2000 万戒尼作开销（赌场偶尔出千、他刻意赢输交替不暴露）。

## 进展（十卷2–7章）
- **首杀新人 5 号**：落地即触发追猎（1 公里内 2 名违规者）——[枯尸](../../人物/枯尸.md)（尸臭·感染体）+ 刚入伙的「新人 5 号」（火药桶·高温·绅士同盟入伙即弃原名用代号·他是「第三个 5 号」）。苏晓用升级后的暗紫蜘蛛女皇一枪**把新人 5 号躯干打碎爆开**（实测狙击打躯干比打头更优）。
- **斩枯尸（猎杀 2/5）**：枯尸是召唤 / 寄生型——液化重组 + 三百米光环把 176 名平民变成 **[感染体](../../世界/生物万族/index.md)**（变异死体·T 病毒式）+ 寄生换身到精英感染体。苏晓环断秒八成感染体、界断线送青钢影克之，影刺斩首 + 踩爆寄生肉球 → **清理 10207 号违规者**。
- **红色小丑约战**：绅士同盟老大 [红色小丑](../../人物/红色小丑.md)（空间系·神经质）以**投影**现身、识破苏晓是「追猎者」，提议一场「游戏」——他带部下追杀苏晓、苏晓反杀他则得知「违规者的秘密」、苏晓败则吐露「追猎者的秘密」，约「友克鑫市见」（本体不敢进空间裂缝、防被苏晓空间禁锢困死）。
- **赶赴友克鑫**：齐鲁市感染体两小时内被清完（枯尸走「增强个体」路线、二代感染体不会传播、杀光即遏制）。苏晓戴透明面具防监控（彼岸花教训），8 月 26 日动身去友克鑫市（距拍卖会 5 天·乘飞艇〔猎人公会管·凌驾国家机构〕等·交通超负荷）。
- **炮台伏击 · 四打一反杀（十卷8–11章）**：[红色小丑](../../人物/红色小丑.md) 掌握行踪、**炸火车设伏**（苏晓凭心眼预警跳车救一对兄妹乘客）+ 花 6/12 万乐园币请旅团 [薇妮](../../人物/薇妮.md) 当「炮台」（30mm 狙击炮 + 分裂导弹）远程压制，加部下 [犀](../../人物/犀.md)（续力连撞）、[艳妖](../../人物/艳妖.md)（隐身偷袭）四打一。苏晓界断线缠犀角 + 次级无敌护盾 + 连斩**反杀犀与艳妖**；红色小丑祭献二人灵魂、激活**恶魔血统**（改名「地狱小丑」·地狱焰 + 双短刀近战 + 瞬移），苏晓注射 1 号药剂回血 + 岚凯风衣空间禁锢封瞬移 + 牛头领主之怒（力 58→73）**斩其两臂一腿**——但小丑用「**所罗门·马奇·低等深渊恶魔**」替死鬼诈死逃脱（**猎杀 4/5**·仅剩残废的小丑）。攻守逆转、轮到苏晓追杀他。
- **抵友克鑫市 · 主线②**：一天后抵友克鑫市（距拍卖会 3 天），主线①完成。**主线②「自由人」**（Lv.16·9 月 1 日激活·**以任何身份参与友克鑫拍卖会、9.1–9.10 不可离开**·奖 3000 币 + 2 属性点·罚全属性 −3）——参与资格 = 加入某方势力（[十老头](../../势力/index.md) / 猎人公会 / [幻影旅团](../../势力/index.md)）或持足够戒尼，苏晓倾向加入势力。

## 组队杀独眼巨怪 · 阴谋反噬（十卷12–18章）
等候参与拍卖会期间，联络平台爆出友克鑫市外 15 公里有 [独眼巨怪](../../世界/生物万族/index.md)（领主级·暗黑大陆召来）的消息。[A先生](../../人物/A先生.md) 在「老内克餐馆」招募 15 人临时团队（苏晓以「狙击手」身份混入旁观、只露枪械精通 Lv.30 + 蜘蛛女皇）——苏晓借此摸清**二阶契约者的真实水平**（多数只有蓝 / 紫装·暗紫已是强者·淡金仅大团有；他因每次深挖世界、综合评分高、装备远超同辈）。

- **A先生的阴谋**：A 先生 + [老狼](../../人物/老狼.md) + 少妇奶妈 + 弓箭手「大头」原是一个小队——演双簧、推老狼当名义首领，真图是**借巨怪坑死散人独吞宝箱**（计划在巨怪濒死时让奶妈停奶「倒坦」害死主坦 [石头](../../人物/石头.md)、放巨怪屠散人）。
- **布布汪搅局 + 反噬**：[布布汪](../../人物/布布汪.md) 的「冰雪女神光环」（先免费后**每分钟 10 币收费**）让石头扛住断奶那一棍逃脱、阴谋崩盘 → **独眼巨怪狂暴**（回血 15%→40%）反杀，老狼、大头被秒，小团队几乎团灭、A 先生断臂逃命。
- **苏晓破防 + 单挑**：苏晓 900 米外狙击——首枪被巨怪侧头骨骼挡下（镶脸），改打无骨部位后**蜘蛛女皇暗紫连射轰出血洞、把巨怪从 91% 打到 34% 破防**（150 币/发·布布汪光环分成补贴）。巨怪狂暴后冲上山坡来杀这个「伤它最重的小矮子」，苏晓收枪拔斩龙闪迎战（后方还有几十契约者等坐收渔翁）。

- **单挑斩杀巨怪（十卷18–19章）**：苏晓退出临时团队（成员死亡过半可无惩罚退团），界断线借力 + 300 能量盾化解、斩断脚踝韧带 / 手臂、环断切喉、口腔刺穿脑髓**单挑斩杀独眼巨怪**——**世界之源 0%→13.5%**、初杀首领级奖 5000 币 + 灵魂结晶〔小〕×3、法力增至 1658、掉落**领主宝箱（淡金）**，还拖走巨怪尸体（暗黑大陆研究价值）。这一幕震慑住盆地周围 37 名等坐收渔翁的契约者（没人敢当出头鸟）。
- **拎首级突围**：苏晓骑布布汪带巨怪头颅突围。恰逢「老熟人」[国足三兄弟](../../人物/国足三兄弟.md)（逗逼锤子组合）凑来摆造型搭话、被旅团 [薇妮](../../人物/薇妮.md)（也在此世界·嫌苏晓枪「口径太小」没架枪）一枪打中老二、国足仓皇逃命——这场骚动替苏晓拉走火力。但 37 名契约者随即「团结」起来浩浩荡荡追杀苏晓（淡金宝箱·有缘者得之）。

## 借十老头入拍卖会（十卷20–24章）
- **甩开追兵 + 处理战利品**：布布汪「跑得快如狗」甩开 37 人；契约者们转而哄抢独眼巨怪尸体（暗黑大陆之物·能卖十几亿戒尼·因 V5 管控不能乱抢平民钱财）。苏晓拆头颅留下**会感知声音的眼球**（暗黑大陆怪物之眼·收藏家垂涎·能换贵宾入场资格），其余喂了 [布布汪](../../人物/布布汪.md)（吞高等生物血肉·力 +1 / 敏 +2）。
- **斩水蛭 · 擒病犬**：阴兽 [水蛭](../../人物/水蛭.md) + [病犬](../../人物/病犬.md) 爬墙偷袭（图眼球 / 报独眼之利）——苏晓隔墙一刀刺穿水蛭喉咙（化紫宝箱 +1.2% 世界之源→14.7%）、又把病犬擒住（植入炼金炸弹要挟）。
- **与十老头「合作」**：苏晓接到 [比克](../../人物/比克.md)（十老头代表）来电、押病犬赴友克酒店谈判——他拒绝当部下、以刀架病犬逼十老头让步，谈成**「合作护场到拍卖会结束」**、报酬 **每天 5 颗「永恒晶石」（即灵魂结晶〔中〕）**、恩怨一笔勾销（先得 5 颗）。比克把女儿 [黑玛丽](../../人物/黑玛丽.md) 安插当他助手。**伏笔**：比克收到密报「**幻影旅团（蜘蛛）盯上了这次拍卖会**」——这才是他高价雇苏晓护场的真正动机。
