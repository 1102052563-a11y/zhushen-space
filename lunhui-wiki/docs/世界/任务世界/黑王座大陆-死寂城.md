# 黑王座大陆·死寂城（第26卷·原生世界·清违规者 + 黑王套装）

[苏晓](../../人物/苏晓.md) 连续五个世界没遇违规者，这次被乐园直接丢进的**原生世界**——同时也是直接产出 **[黑王之魂套装](../../力量体系/装备道具.md)** 的地方（该套装无法从衍生世界产出·原本就是这个世界的东西·集齐 3 件有几率晋升套装阶位）。一个高危违规者被强制拖入同一世界。

!!! tip "苏晓在本世界的三大目标"
    ① 清理 **违规者 9744 号**（猎杀任务·异常·杀过 6 名猎杀者 / 1 名裁决者 / 1 名战斗天使）；② 完成主线「王权之争」；③ 找到 **死寂城** 集齐 [黑王之魂套装](../../力量体系/装备道具.md)。三者全与「王权」死死绑在一起——尤其死寂城，没夺下王权连找它的资格都没有。

## 世界设定

- **黑王座大陆 / 安多哈帝国**：难度 **Lv.27～Lv.50（原生世界·近六阶）**；世界之源获取难度是同阶衍生世界的 **15 倍以上**；另有「**黑暗之源**」（需进特定区域才能获得）。科技断层（有风力发电却仍用冷兵器·电力 90% 只用于照明）。
- **黑之王·阿德格什**：活 369 岁、一统大陆所有势力（连海族都要定期献最美人鱼朝拜）、把大陆更名黑王座大陆——[黑·王之陨祭](../../力量体系/装备道具.md) 正是他的护臂。**暗纪元 1675 年离奇死于某地**（独自离城、半月后死讯传来·令领主们至今怀疑他没死、不敢造反）。其威仪让帝国百年内无惧外敌、唯有内乱。苏晓进入时间 = 暗纪元 1691 年（黑之王已死约 20 年）。
- **永恒之力**（本世界超凡之力）：教堂顶金属圈吸取的**无属性能量**「永恒」、既是光也是暗。需天生强精神力才能接纳；初期会**放大情绪**——抗不过即变成「死寂城的居民」，抗过则按性格转化出能力（光明 = 善良 / 倔驴·暗 = 上千种分类·90% 的人转向暗但不必是坏人）。
- **死寂城**：黑之王永远安眠之地、「一切希望泯灭之地」——**只有成为黑之王才有资格进入**，否则会被那里的「居民」同化 / 感染。黑王之魂套装 / 黑暗之源很可能都在此。黑之王临死在此留下东西镇压居民、如今快维持不住（[肯·拉罕](../../人物/黑王座大陆配角.md) 的执念所在）。

### 主线任务 / 猎杀任务

- **主线·王权之争（第一环·Lv.46）**：加入王族任意势力并获话语权（**已完成**·苏晓进帝国议会·奖灵魂结晶〔完整〕×2）。
- **主线·黑王座（第二环·Lv.48）**：找到黑王座、取下镶在其上的**世界之核**（奖 [黑王之魂](../../力量体系/装备道具.md)·**若违规者先得世界之核则任务立即失败**）。
- **猎杀·清理违规者 9744 号**（奖黄金荣誉勋章 ×25·堪比 4~5 次猎杀收益）：异常·极危险·善伪装·**能强行替代世界内关键人物**——苏晓推断它原是天启乐园的契约者（故能杀战斗天使）、夺世界之核后跳槽轮回乐园仍不安分、反杀来追的猎杀者才触发此局。主线与猎杀息息相关（执行主线即可循到违规者踪迹）。

## 角色在本世界（苏晓·五阶 + 王族身份）

- **入场即王族**（二十六卷1–4章）：乐园因违规者的特殊性给苏晓拟定 **王族身份**（以 [黑·王之陨祭](../../力量体系/装备道具.md) 为凭·暂未公开）——开局被当「外乡人」绑在火刑架上、被 [肯·拉罕](../../人物/黑王座大陆配角.md)（老神棍·黑之王右御）接走。苏晓定下策略：不自己出面夺权、而是**立一个可控的傀儡**——挑中软萌的 **15 岁 [晨曦公主·佩黛菈](../../人物/黑王座大陆配角.md)**（民间有呼声、对各方威胁小·被刺杀过三次）。
- **夺军权 + 收军方**（二十六卷5–9章）：闯元老院、用炼金炸弹试探逼出元老首 [布卢默](../../人物/黑王座大陆配角.md)，又拿其「囚禁月王后」的丑闻 + 海族动乱施压，**夺下 54 万军权**的转让批文。再向统帅 [伍弗](../../人物/黑王座大陆配角.md) 全额拨军费（砸断一指立威 + 撒钱收心）、让其率 1200 精锐入驻庄园当护卫。
- **夜战群刺客**（二十六卷10–12章）：布卢默 + 神秘势力派 14 名刺客夜袭（血肉宴·曼 / 操血壮汉 / 异族佣兵等·混杂得反常），苏晓 + 布布汪 / 阿姆 / 贝妮全歼、把庄园变成军营防线。
- **帝国议会 · 除银羽公爵**（二十六卷13–15章）：被推上台面进**帝国议会**（四夺权者：[布卢默 / 王女沙耶托 / 银羽公爵 / 小公爵](../../人物/黑王座大陆配角.md)）。苏晓当庭揭银羽公爵「与亲姐玛姬通奸」的丑闻搅局，再用 **[恶魔之罐](../生物万族/index.md)**（布布汪三次喂仆从恶魔之力）短控其心腹仆从、递淬毒匕首**当场刺杀银羽公爵**（嫁祸 + 物理除掉一个夺权对手）——王权斗争「从树根锯断」的打法。

- **空间陷阱 + 斩巴哈**（二十六卷20–21章）：回程被埋空间陷阱伏击（说明阵营有内奸）——西境侏儒 **[巴哈 + 小肉山](../../人物/黑王座大陆配角.md)**（人造生物·血肉礼堂·爆体九具肉山的绞肉风暴）险些得手，苏晓斩其制造者 [布嘉恩·费迪](../../人物/黑王座大陆配角.md) 后获人造生物核心 [血肉礼堂·巴哈](../../力量体系/装备道具.md)。

- **揪出违规者狄鸦**（二十六卷25–31章）：循「木詹棘果壳」（违规者能力的副产物·失依托即蒸发）线索，确认违规者 = 天启乐园来的 **[狄鸦](../../人物/黑王座大陆配角.md)**（意识箭寄存灵魂·替代过莱娜 → 佩黛菈）。她当庭以「佩黛菈」身份捅苏晓一刀露馅（佩黛菈从不敢发问·苏晓由此识破），粗暴拔意识**害死佩黛菈**（灵魂被扯碎）。苏晓早让布布汪 + 8000 士兵定位其本体，调来 **1.5 万攻城部队**（兽骑兵 / 重盾 / 长戟 / 攻城炮）把狄鸦车轮围殴 3 小时耗死（「有种单挑」=1.5 万单挑她一人）——**清理违规者完成·评价极其优秀**·得 28 枚黄金荣誉勋章 + 特殊称号 **[老猎人](../../力量体系/称号.md)**（鲜血印记·可跨世界追踪猎物）。
