# 海贼王·顶上战场（第25卷·万能世界钥匙·五阶首战）

[苏晓](../../人物/苏晓.md) 晋升五阶后用 **[万能世界钥匙](../../力量体系/装备道具.md)**（[咕噜](../../人物/咕噜.md) 卖的「生灵花」·指定世界进入）随机+指定进入的衍生世界——**海贼王·顶上战争前十天**。这是他第三次踏入海贼世界（前两次见 [海贼王](海贼王.md) / [海贼王·德雷斯罗萨海战](海贼王-德雷斯罗萨海战.md)），却是**晋升五阶后的第一场实战**。

!!! tip "为何选顶上战场"
    顶上战场是海贼世界**最高奖励事件**，违规者无权进入。备选还有「龙珠（大型开放世界）」「虫族战场（虚空·生物学六阶以上的高等文明虫族）」，苏晓权衡后仍选海贼（奖励更优）。冒险团团队技能「精英团队（被动 Lv.30）」额外追加一个可选衍生世界。

## 世界设定（进入信息）

- **世界难度**：Lv.3～？？；**世界之源** 0% 起；时间线 = **顶上战争十天前**（草帽团尚在香波地群岛、超新星云集）。
- **主线任务**：加入势力（任务时限 3 天·奖 20000 乐园币）——只是初始任务，真正的好处要等顶上战场开打。
- **猎杀任务**：无（顶上战场为最高奖励事件·违规者无权限进入此时间点）。
- **本世界综合评价门槛**（按已消耗时空之力 0.19 盎司）：五阶需 **B+** / 四~三阶 C / 二~一阶 D+。
- **苏晓进度**：世界政府死敌、BIG MOM 海贼团死敌、**悬赏 14 亿贝利（死活不论·因杀天龙人）**；身份「科研人员 / 高危武器开发人员」；曾杀天龙人故继续受 **CP0** 追杀（杀天龙人事件被乐园强制提前三天、不再波及其他契约者的进度）。

## 角色在本世界（苏晓·五阶首战）

### 入场即试探大将（二十五卷1–5章）

- **新造青钢影武器「[放逐](../../力量体系/装备道具.md)」**：进世界前苏晓用青钢影能量塑出第二件本命飞行武器（继 [斩龙闪](../../力量体系/装备道具.md) 后）——状态一「无柄刺剑」追求极致穿透（速度 / 穿透胜斩龙闪一筹）、状态二「数十碎刃」可钻入敌体顺血管绞向心脏（逼意志不坚者自尽·对付大体型敌人）。平时化作袖口淡蓝纹路携带。
- 一阶段被乐园传送到香波地外 10 海里高空（保护低阶契约者）——苏晓队伍靠降落伞 + [阿姆](../../人物/阿姆.md) 冰面落海、逼降一艘金鲨海贼团的船登岛。
- **试探战海军大将黄猿**：苏晓在 12 号区（黄猿正对峙 [雷利](../../人物/顶上战场配角.md) + 重创的草帽团）「路过」试招——用放逐穿透黄猿头颅（黄猿主动开洞躲）/ 青钢影裹斩龙闪斩断天丛云剑 / 状态二碎刃钉进黄猿元素化小腿出血 / 800 强度能量盾硬扛八尺琼勾玉半分钟。**结论**：单挑大将不虚、抓破绽可杀大将、同时对两大将能撤但硬拼必死、肯下血本用一次性物品则对大将有 70%+ 胜率。详见 [黄猿·波鲁萨利诺](../../人物/黄猿·波鲁萨利诺.md)。

### 三线布局（二十五卷6–15章）

- **情报碾压白胡子海贼团**：苏晓偷 [多弗朗明哥](../../人物/多弗朗明哥.md)（已被世界政府征召）的渠道、联络白胡子海贼团 [马尔科](../../人物/顶上战场配角.md)，免费抛「艾斯每五小时被毒打 + 世界政府已知艾斯是罗杰之子」的情报立信、约见白胡子。
- **大修白胡子**（二十五卷9–12章）：登白胡子的伪装船（莫比迪克号仍留新世界迷惑四皇），用炼金学 + 等离子锯 + 金属疗法 + 远古秘药给重病濒死的 [白胡子](../../人物/顶上战场配角.md) 连做三场手术、**恢复其 96.7% 状态**（不完全麻醉以免重演莎的惨痛体验）。乐园奖「白胡子击杀单位苏晓得 25% 奖励」+ 全服公告「白胡子恢复 90%+」→ **海军阵营契约者任务难度暴涨**（联络平台炸锅）。报酬 = 莫比迪克号到伟大航路后任选宝库十件物品 + 额外赠史诗红酒 **[玛歌亚加](../../力量体系/装备道具.md)**。
- **清算 CP0 + 顺手绑天龙人**（二十五卷13–15章）：回达坦奥岛拷杀欢乐街女王 / CP0 间谍 **[斯图茜](../../人物/顶上战场配角.md)**（得世界之源 7.5%·金色套装宝箱·天赋噬灵者激活 +26 法力），逼问出岛上藏着天龙人——狙落绑走绿发天龙人 **[唐吉诃德某人](../../人物/顶上战场配角.md)**（多弗家族），用其当饵引出上次追杀他的 CP0 小队头目 **[埃迪·克](../../人物/顶上战场配角.md)**，同时革命军 **[克尔拉 / 雷兹·马克斯](../../人物/顶上战场配角.md)** 也找上门。

!!! note "暗处的契约者"
    本世界已现身的其他契约者：[幻影旅团](../../人物/咕噜.md) 的「[灰发男人](../../人物/顶上战场配角.md)」（劝苏晓入团 + 转达旅团老大想借苏晓和恶魔族谈事）、变态契约者 **[阿普斯](../../人物/顶上战场配角.md)**（生殖之神·种「种子」）、传情报的兔耳少女、五阶情侣 **[黑月 / 花影](../../人物/顶上战场配角.md)**（海军阵营）、中型冒险团 **[哮天团](../../人物/顶上战场配角.md)**（女神经艾露莎）等。

### 斩埃迪·克 + 加入革命军（二十五卷16–20章）

- **斩 CP0 头目 [埃迪·克](../../人物/埃迪·克.md)**：在锁好天龙人的麻醉剂蓄水池上设死局、用传说级道具 [精神污染](../../力量体系/装备道具.md) 乱其思维，放逐 + 斩龙闪洞穿心脏斩首（世界之源 +5.3% 累至 16%·噬灵者 +30 法力·史诗宝箱）。布布汪 / 阿姆全程封船兜底。
- **加入 [革命军](../../势力/index.md) 阵营**（最大自由度·只为有个参战阵营·不指望武力支援）：给革命军 [克尔拉](../../人物/顶上战场配角.md) 丢淘汰的炼金炸弹图纸 + 样品立信，再以「替革命军调配药剂」为名**白嫖革命军稀有材料刷炼金「植物调和篇」熟练度**（[圣痕药剂](../../力量体系/消耗品材料.md) → [黎明之焰](../../力量体系/消耗品材料.md)）。住香波地 72 区革命军地下分部、闭门炼丹到开战。革命军首领 [蒙奇·D·龙](../../人物/顶上战场配角.md) 视苏晓为「天上掉的馅饼」。

### 取宝库报酬 + 装备杀戮本能（二十五卷21–22章）

莫比迪克号进伟大航路后，苏晓上船在白胡子宝库**任选十件**：[杀戮本能](../../力量体系/装备道具.md)（史诗腰带·加速直感成长·判定成功得本能激活）/ [月之风华](../../力量体系/装备道具.md)（首颗史诗宝石）/ [依恋的海螺](../../力量体系/装备道具.md)（传说道具）/ [米奈希尔深海矿石](../../力量体系/消耗品材料.md)×2 + 3 超人系 + 2 动物系恶魔果实。返航途中佩戴杀戮本能、**意志判定成功**、直感冲到 Lv.42。

### 顶上战争爆发 · 苏晓参战（二十五卷23–32章·进行中）

- **四主线任务四选一**（加入革命军的好处·完成一种即免其余三种「强行处决」惩罚）：① 营救艾斯（奖 [断魂影之石·残缺](../../力量体系/装备道具.md)）② 瓦解黑胡子海贼团（奖黄金属性点）③ 杀一名中将以上军官 ④ 击杀白胡子（奖成长类装备宝箱）。
- **开战**：白胡子海贼团从海下浮入内港、[白胡子](../../人物/顶上战场配角.md) 震震果实开场扫飞大将 [青雉](../../人物/顶上战场配角.md)。**战争功勋系统 / 战争商店**开启（见下）。苏晓潜入海军本部侦查（布布汪混进七武海会议 + 关押艾斯的地牢 + 给多弗朗明哥塞东西），战国对苏晓「仇恨度爆表」下「进战场格杀勿论」令。
- **苏晓参战**：先斩偷袭的少将（新兵克比旁观）、马尔科帮拖黄猿、反杀围攻他的五阶情侣 **黑月 / 花影**（首次吃瘪受伤·二人放黑色光柱想同归于尽）、与大将 **青雉** 缠斗（阿姆克制青雉·苏晓用众生之地启发的新刀招刃道刀·极 / 刃风暴擦伤其小臂）。

!!! tip "战争功勋系统 / 战争商店（顶上战场专属机制）"
    剧情事件「顶上战场」期间，**击杀剧情人物不直接得世界之源**（事件结束统一结算）、且**无法获得史诗级以下宝箱**（转化为「战争功勋」）。功勋可在**战争商店**（对所有契约者开放）换购：灵魂结晶〔完整〕150 功勋（仅此档对低阶开放·低价让弱者也能获益）/ 海之秘宝 1700 / 钢铁之试炼 1200（无风险把装备强化到 +13）/ 黑王之魂 2000（传说套装件）等。杀一名少将仅 56 功勋。**收益有阶位上限**：五阶烙印 >Lv.45 收益降 80%、六阶降到不足 10%——这是苏晓刚晋五阶就来打顶上战场的原因（趁烙印等级低吃满奖励）。
