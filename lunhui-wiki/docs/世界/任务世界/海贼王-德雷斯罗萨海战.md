# 海贼王 · 德雷斯罗萨海战（第十八卷·四阶之始）

[苏晓](../../人物/苏晓.md) 的**第 18 个衍生世界**、也是他的**四阶晋升考核世界**（第十八卷「海贼·四阶之始」）——他第三次进入海贼王世界（前两次见 [首入·哥亚王都](海贼王.md) / [第六卷再入·七水之都](海贼王-第六卷再入.md)）。这是苏晓**第二次**抽到「德雷斯罗萨海战」（上次因太危险被迫放弃）——他据此判断：**想进入海贼世界其他剧情线、必先经历德雷斯罗萨海战**。

## 选定本世界（十七卷104章）

苏晓 Lv.30 触发四阶考核（已通过 [强者之证明](恶魔古堡游戏.md) 故免去三阶战争世界·见 [机制](../../总览/轮回乐园机制.md)），靠破晓冒险团「精英团队」技能抽到**两个大型开放世界二选一**：

- **型月世界**：力量体系复杂（直死魔眼涉因果）·非法系的苏晓未选。
- **海贼王·德雷斯罗萨海战**（选定）：作为三阶最强者、苏晓认为此行有机会大捞一笔。

## 时代背景与战局（十八卷1章）

- **时间点**：**艾斯刚被捕**、距「**顶上战争**」开战仅剩约一个月。德雷斯罗萨海战与顶上战场息息相关。
- **牵涉势力极多**：四皇中的三位（**夏洛特·玲玲**、**凯多**、**红发**）、海军、革命军、**多弗朗明哥**势力都可能卷入。
- **契约者投放**（公共衍生世界联络平台·消耗时空之力 0.17 盎司·最低评价须 B 级）：
    - **五阶契约者 6 名**——位于「玲玲 VS 凯多 VS 红发」战场。
    - **四阶契约者 278 名**——位于德雷斯罗萨附近海域、「多弗朗明哥势力 + 部分凯多势力 VS 玲玲势力」战场。

## 契约者投放与区域（十八卷1章）

近 700 名契约者涌入这场海战（消耗时空之力 0.17 盎司·最低评价须 B 级），按阶位分区：

- **五阶 6 名** → 蛋糕岛附近（玲玲 / 凯多 / 红发地盘夹缝·直接参与四皇较量）。
- **四阶 278 名** → 德雷斯罗萨海域（多弗朗明哥 + 部分凯多势力 VS 玲玲势力）。
- **三阶 397 名** → 庞克哈萨德岛（海贼 VS 海军）。
- **苏晓**：因实力远超三阶、又超出部分四阶，乐园为保护低阶契约者把他**直接传送到德雷斯罗萨海战中心区域**（第二批·落在唐吉诃德家族「黑桃号」上、开局正逢暴风雨海战）。

## 时代格局 · 四皇（十八卷2章）

时间点 = **顶上战争一月前·白胡子尚在世、艾斯刚被捕**（白胡子因艾斯被捕传闻忧心、其余三皇借机蠢动）。新世界四皇：

- **白胡子·爱德华·纽盖特**：四皇最强·不看重财富地位、视船员部下为最重的珍宝。
- **百兽·凯多**：生存能力变态、「爱好自杀」。
- **红发·香克斯**：大海般的胸怀·多数势力都给他几分薄面。
- **夏洛特·玲玲**：非常能生、子嗣中强者众多、本人实力不俗、痴迷甜食。玲玲与凯多年轻时同属 **Rox 海贼团**（罗杰时代前的海上霸主）。

## 海战的起因（十八卷2–3章）

表面是 **BIG·MOM 海贼团 VS 唐吉诃德家族 + 百兽海贼团**抢夺富饶的德雷斯罗萨岛——实则争夺**人造动物系恶魔果实技术**（战略价值）：

- 多弗朗明哥靠资助某人得到了**人造恶魔果实技术**、近期趋于量产；核心是科学家 **M·凯撒·库朗**（提供原料）+ 居于德雷斯罗萨地下的 **咚塔塔小人族**（培植·费用极高、产量低）。
- 凯多极看重人造果实、遂派**三灾之一「旱灾·杰克」**助阵多弗朗明哥；玲玲想夺技术、派长子 **夏洛特·卡塔库栗**（BIG·MOM 团除玲玲外最强·觉醒超人系果实 + 三色霸气·见闻色强到**预知未来**）+ 弟弟 **夏洛特·克力架**（8.6 亿赏金）来攻。此役 = 多弗朗明哥 + 杰克 VS 卡塔库栗（暴风雨中三人皆果实能力者、藏身船舱）。
- 这片海域具体战局：黑桃号最高干部 **琵卡** VS 玲玲长子 **夏洛特·佩罗斯佩罗**（舔舔果实·近乎无限造糖、堪比自然系）。

## 苏晓选定多弗朗明哥势力 + 海贼声望（十八卷3–5章）

可加入三方（堂吉诃德家族 / 百兽 / BIG·MOM），苏晓选**最弱势的多弗朗明哥势力**——因多弗朗明哥**原是天龙人**、其地盘 **CP 部门不敢轻易涉足**，正好让司法岛后与世界政府彻底翻脸、被悬赏 **2 亿贝利（只活捉）**的苏晓避开 CP 纠缠（详见 [多弗朗明哥](../../人物/多弗朗明哥.md)）。

- **晋升 / 主线任务合一**（共四环·首环 Lv.20·12 小时·1 万币·失败强行处决）：**加入唐吉诃德家族 / 百兽 / BIG·MOM 任一势力**。击杀某方海贼即与其敌对、可临时加入对立方（获 5 点海贼声望可正式加入）。
- **🆕海贼声望（新机制）**：杀敌 / 完成大事件 / 隐藏任务可得；**海贼声望排行榜**结束前结算、**前五名获丰厚奖励**（有五阶契约者参与、第五名的奖励都极诱人）。开局排行：① [枯屠](../../人物/index.md)（五阶·1506 点·实力恐怖）② [浅梦](../../人物/浅梦.md)（五阶·287）③ 森（五阶·76）④ 萨尔夫（四阶·51）⑤ 白夜（苏晓·三阶·27）。杀普通海贼约 2~3 点声望。

## D·暗杀首秀（十八卷4–6章）

苏晓用新得的传说级手枪 **[D·暗杀](../../力量体系/装备道具.md)** 首秀——隔几十米透过 20 公分瞭望口**连爆 12 名 BIG·MOM 精英海贼的头**（无声领域让满船炮手浑然不觉同伴已死），**一人压制四艘大船的火力**、声望飙到 106 点排第三。黑桃号主桅被炮断、注定要沉，他索性用 [寂灭公爵 +13](../../力量体系/装备道具.md) 灌土属性弹**轰穿沉掉敌船「特兰斯号」**（代价是后坐力震断黑桃号龙骨、自己破船而出入海）。

## 拖海溺杀斯佩罗 + 强行救琵卡（十八卷7–11章）

水下苏晓与一名佛系四阶女契约者「潜水妹」约定**互相破坏对方阵营的船、彼此不冲突**，又用炼金炸弹炸沉大妈方船只。海战主将 [斯佩罗](../../人物/夏洛特·佩罗斯佩罗.md)（玲玲长子·舔舔果实）用糖浆把残船粘成「小船岛」压制黑桃干部 [琵卡](../../人物/琵卡.md)——苏晓三人组**7 秒配合**把斯佩罗拖进海里（恶魔果实能力者入海即废）溺杀：

- **独立击杀海战区首领斯佩罗** → 1000 海贼声望 + **世界之源 14.8%（首领加成 5%）** + 噬灵者 +30 法力（达 5456）+ **第三级别传说级宝箱**（见 [机制 · 传说级宝箱四级别](../../总览/轮回乐园机制.md)）。海战判唐吉诃德家族胜·苏晓公开击杀消息刷海贼威望。
- 随后新世界特有的**巨型冰雹**（直径 1~2 米）砸碎琵卡的甲板筏、把他呛得昏迷，苏晓「**强行救命**」（琵卡部下其实全是苏晓杀的）、当上他的「救命恩人」——但两人都是海上路痴，苏晓发信号弹引来一艘白胡子旗下商船、由其送二人抵达德雷斯罗萨。

## 加入唐吉诃德家族 + 主线四环（十八卷11章）

经与琵卡初步交涉，苏晓**正式加入唐吉诃德家族**——晋升 / 主线任务（共四环）进度：

- **第一环（Lv.20·已完成）**：加入势力 → 1 万币。
- **第二环（Lv.15·进行中）**：抵达德雷斯罗萨（5 自然日·奖 1 真实属性点·罚强行处决）。苏晓警觉「**任务越来越简单不是好事**」——后两环难度恐怕会很夸张（他另有一次主线失败豁免权）。
- 苏晓此行的私心：海贼世界的**名刀 / 体技 / 恶魔果实**，尤其想学**武装色霸气**（攻防兼备）与**见闻色霸气**（可融入 [直感能力](../../力量体系/技能.md)·大幅增强感知）。

## 与多弗朗明哥的「合作」+ 炸弹刷声望计划（十八卷12–16章）

苏晓在 [王之高地](../../世界/地点/index.md) 见 [多弗朗明哥](../../人物/多弗朗明哥.md)（线线果实觉醒·原天龙人），双方挑明利用、谈成合作：苏晓为唐吉诃德家族造**阿波罗 + 炼金炸弹**（多弗朗明哥视他为「瘟神」·他造过害惨海军的特级炼金炸弹），**爆炸物所有权归苏晓**、家族有使用 / 征调 / 出售权。这正是苏晓选最弱势力的算计——**让整个唐吉诃德家族用他的炸弹杀敌、自己刷海贼声望**（如愿的话连榜首枯屠都怼不过他）。多弗朗明哥以七武海身份替他挡 CP 作回报；苏晓还点破「凯撒·库朗造的人造果实只是半成品垃圾、真正开发者是 Dr.贝加庞克」（削寿 / 暴走为其缺陷）。

- **战报 · 猪队友**：海战中唐吉诃德家族三最高干部除琵卡外尽墨（托雷波尔失踪、迪亚曼蒂重伤），普通干部死的死俘的俘（拉奥·G 战死等）；契约者加入后大妈势力七成把握获胜。多弗朗明哥遂**放弃海战、固守德雷斯罗萨**（巨岩区天然屏障·打登陆战），还要防「盟友」凯多坐等吞并他。新世界大事频发（艾斯被捕、白胡子团四番队长萨奇被同伴杀后叛逃、海侠甚平〔七武海〕被捕、白胡子反常沉默）——危险指数较路飞初入时飙升数十倍。

## 大妈登陆战 · 炸弹陷阱（十八卷17章）

玲玲因长子斯佩罗之死震怒、要一举攻破德岛（攻下后任部下消遣三天）。大妈船首炮轰海岸半小时后登陆——多弗朗明哥早在七处登陆点的沙滩**埋大量炮弹 + 炼金炸弹远程引爆**，瞬间把沙滩化成火海炼狱（唯卡塔库栗凭见闻色预知避开）。苏晓在别墅四层望远镜旁坐收**海贼声望**（炸弹炸死敌方海贼即给他声望·计划见效）。

## 登陆战 · 狙克力架 + 围伤打援（十八卷18–23章）

声望榜首 [枯屠](../../人物/枯屠.md)（五阶·龙卷风枯萎）则在大妈老巢 [万国](../../世界/地点/index.md) 掀风刷声望（手下演奏手 + [浅梦](../../人物/浅梦.md) 拖住大妈）——他声望飙到 6255，但苏晓看出那是「跳跃性」提升、**没有长期稳定渠道**，而自己有（炸弹 + 狙强者），不慌反喜。

德岛战场上，BIG·MOM 派来甜点四将星 [克力架](../../人物/夏洛特·克力架.md)（饼干果实·造「饼干士兵」军团冲 7 号登陆点）+ [卡塔库栗](../../人物/夏洛特·卡塔库栗.md)（四将星之首·年糕果实·见闻色预知）。苏晓的连环操作：

- **狙沉敌船**：寂灭公爵隔 4 公里轰沉 14 艘大妈船（50 声望/船·巨石后的船幸免）。
- **狙爆克力架后脑**：靠情报 + [布布汪](../../人物/布布汪.md) 潜行标位，隔 4 公里 + 一块巨石一枪轰碎藏在巨石后遥控造兵的克力架真身后脑——被赶来的卡塔库栗预知 + 年糕墙救下。
- **围伤打援**：苏晓锁定 [多弗朗明哥](../../人物/多弗朗明哥.md) 引其注意、布布汪巨石写「拖住卡塔库栗」→ 多弗拖卡塔、苏晓继续狙克力架。卡塔护弟分心、沙滩半边年糕化半边丝线化缠斗。
- **海贼声望升至第三（2860 点）**：炸弹刷 + 狙强者并进。这引来数十名四阶契约者觊觎（杀苏晓得其声望 1/2）——「送死小分队」一眼看穿他是全面独行侠而怂退，但四阶刺客团「**[暗影团](../../势力/index.md)（黑耗子）**」7 人用影分身突进围杀（详见 [势力 · 暗影团](../../势力/index.md)）。

## 暗影团围杀 + 三人围攻卡塔库栗（十八卷24–29章）

四阶刺客团 [暗影团](../../势力/index.md) 潜入别墅区围杀苏晓——被苏晓用龙影闪 + 直感 + 简易脆响装置反杀 2 人（「影一」「影三」），其余 5 名隐身系逃散（苏晓由此悟「对隐身小队该静候不该主动出击」）。随后苏晓亲赴 7 号登陆点，与 [多弗朗明哥](../../人物/多弗朗明哥.md) **三人围攻 [卡塔库栗](../../人物/夏洛特·卡塔库栗.md)**（四将星之首·年糕果实·见闻色预知）：

- **海楼石 + 青钢影双克**：苏晓掏出 [海楼石金属棍](../../力量体系/装备道具.md)（果实者最大克星）当刀使，又确认**青钢影能噬灭果实者的「元素化」**（详见 [技能](../../力量体系/技能.md)）——卡塔库栗因此视苏晓比多弗危险几个档次、宁挨武装色也不碰海楼石。
- **破预知**：卡塔能预知未来、却**无法预知「必定发生、躲不开」的事**；苏晓 + 多弗靠前后合围 + 让 [布布汪](../../人物/布布汪.md)（热感应定位重伤的克力架）/ [阿姆](../../人物/阿姆.md) 牵制分心，逼出破绽。战况胶着——苏晓还「手滑」一棍砸飞多弗朗明哥的鼻梁（多弗心里苦）。

## 海贼世界力量体系（十八卷·补充）
- **三色霸气**：霸王色 / 武装色 / 见闻色（对身体的高等开发）。**武装色**= 能捕捉并攻击果实者本体（卡塔的年糕、苏晓的多弗都靠它互伤）；**见闻色**= 感知，卡塔库栗强到**预知未来**（苏晓想把见闻色融入 [直感能力](../../力量体系/技能.md)）。
- **恶魔果实**：价值通常 自然系 > 超人系 > 动物系（非绝对·幻兽种动物系可超自然系·超人系上限最高、顶级强者多超人系如白胡子）。能力者**惧海水 / 海楼石**——海水使其无力（果实仍在）；**海楼石是最大克星**（脱力 + 临时封印果实 + 元素化身体变回血肉）。

## 角色在本世界
