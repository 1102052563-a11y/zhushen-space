# 海贼王（任务世界）

[苏晓](../../人物/苏晓.md) 进入轮回乐园后的**首个任务世界**，以漫画《海贼王》为蓝本；第22章完成主线、返回乐园（第2、22章）。

## 世界信息
- **难度**：LV.6 · 噩梦（结算时记为「炼狱」）——**实为哥亚王都「区域」难度**，并非整个世界的真实难度（该世界存在 LV.79 的超强生物 [蒙奇·D·卡普](../../人物/路飞.md)）（第2、16章）。
- **简介**：大海、自由、海贼、海军、世界政府（第2章）。
- **本地货币**：[贝利](../../总览/货币与商店.md)（第7章）。
- **时间点**：原著焚烧废品终点站前夕（约为路飞幼年、出海十年前）；半月后天龙人将来哥亚王国巡查（第6、8、17章）。

## 规则与提示（第2章）
- 不得提及「乐园」（违者强制处决）；勿离开哥亚王国区域（否则难度大增）；初次进入自动掌握本世界语言（三日）。

## 任务
- **主线**：刺杀哥亚王国国王 **菲利普·赫伯特**（LV.3，72 小时）。**已完成**——苏晓破窗腰斩斩首之（第22章）。奖励：猎杀者身份正式解锁、开启猎杀任务（下个世界发布）。
- **支线 · 左大臣的藏品**（LV.2）：杀 [科尔波山的主人](../生物万族/科尔波山的主人.md) 取牙。**已完成**（第15、17章）。
- **触发任务 · 传奇的扼杀**（LV.79）：接触位面之子（幼年 [路飞](../../人物/路飞.md)）触发，苏晓直接放弃（第16章）。

## 苏晓在本世界（化名白夜）
- 入侍卫队 → 杀首领欧卡 → 完成左大臣支线升任**侍卫首领** → 借焚烧废品终点站引克利克海贼团攻王宫、调开国王守卫 → 腰斩国王完成主线（第7–22章）。
- 过程中牺牲助手 [汉克](../../人物/汉克.md)、以「买命钱」收买死刑犯诱导 [克利克](../../人物/克利克.md)（第18–20章）。

## 结算快照（离开本世界 · 第22章）
- 难度 Lv.6（炼狱）｜ 世界之源 **8.9%** ｜ 完成任务 **2**（主线 + 左大臣支线）。
- **综合评价 A-**（评分 E- ~ S+，按世界之源总和与完成任务数量计算）。
- 因世界超出猎杀者等级 5 级（噩梦难度）**奖励翻倍**：得 **4 属性点 + 10000 乐园币**；评价 A- 使等级 **LV.1 → LV.2**（试炼世界单次升级上限 +1）。
- 携出物：[斩龙闪（白·稀有）](../../力量体系/装备道具.md)、[世界树指环（白）](../../力量体系/装备道具.md)（国王宝箱所得，含大量世界之力）、[灵魂结晶（小）](../../力量体系/消耗品材料.md)、[虎爪](../../力量体系/消耗品材料.md) 等。
