# 烈阳星·奇利亚德

> ⚠ **世界简介:**赞美太阳****——⚠ **虚空·第一纪元三个超脱之界之一「太阳阵营」的原生世界**（⚠ 与 [古龙国度·埃伯亚思](女巫界-巫师界.md) 一冷一热·打了整整两个纪元）;⚠ **⚠ 如今已不再是超脱之界，是和 [永光世界](永光世界-厄难囚笼.md) 类似的特殊世界**;⚠ **⚠ 苏晓来此是为了找【太阳圣徽】→ 配【古龙遗匙】开禁足塔 → 补足冲击至强级的底蕴。**

## ☀ 一座只许进不许出的囚笼

⚠ **太阳与古龙混战、两败俱伤后，太阳阵营获胜**——⚠ **趁火打劫的强大神灵们（众神殿）直奔烈阳星而来，想夺走太阳与古龙的所有资源**;⚠ **⚠ 可太阳神族早有远见:把双方海量资源都存藏在了古龙国度·埃伯亚思**（⚠ 详见女巫界页·⚠ **这就是白龙女继承两阵营遗产却被囚禁足塔的由来**）;⚠ **⚠ 迎接众神殿的，是天空中那颗太阳之力已畸变的烈阳所带来的无尽炙烤**——⚠ **⚠ 此地是「只许进不许出」的牢笼**（⚠ 除非得到虚空之树或乐园阵营的公证传送权限，进入后根本不可能）;⚠ **⚠ 迄今为止，烈阳星依然是太阳神族让那些趁火打劫之人承受无尽折磨的囚笼世界。**

- ⚠ **⚠ 第一纪元九成以上的强大神灵，其实都在本世界内**——⚠ **本应因时间而陨落的它们，已沦为**扭曲又不死的存在**;⚠ **⚠ 天空中的太阳与月亮都被无光区蔓延出的深渊之力侵蚀**（⚠ 太阳畸变·⚠ 曾是万界奇观的银月畸变成了**血月**——⚠ **银月狼一族全族成为深渊监视者、多数被深渊侵蚀，或许就因这轮血月**）。

## 🗺 南北大陆与无光带

⚠ **一圈黑色环带「无光带」将烈阳星从中隔成南北两大陆**——⚠ ****南大陆**:黄昏城、灵魂学院、诸神教、神灵放逐之地（智慧生灵势力强于怪物）·**北大陆**:黑暗神教的「地城」、古老高塔、死城、暗月噩梦（普遍处于塔防处境）。**

- ⚠ **⚠ 无光区**:⚠ **总计几百个小型深渊通道**——⚠ **两侧各有一面漆黑岩石天壁近乎密封（十之八九是太阳神族的杰作），各留一处破口做泄压口**（⚠ **若彻底封闭，累积的深渊高压会让天壁龟裂崩碎**·⚠ 这些深渊通道亘古有之、强行缝合只会更糟）;⚠ **⚠ 无光区内无法传送、走回头路会让灵魂余味引来怪物**;⚠ **⚠ 天壁孔道飘散的深渊能量都被天空中的太阳吸收**（⚠ **这是太阳神族与烈阳的因果:那些原本该由太阳神族承受的深渊侵蚀，被烈阳全部吸收，故太阳神族被称「完美族裔」**）。

## ⚔ 深渊战役·烈阳星战区（54卷229–230章）

⚠ 苏晓时隔多卷重返此界，身份已从「进来找太阳圣徽的至强者」变成**无上之境的最强增援**。

| 项 | 内容 |

| 深渊侵蚀度 | 21.77% → 85.66%（苏晓抵达时）→ 80.2% → ⚠ **被压下并摧毁深渊之门** |

### 为什么这里格外惨烈

⚠ **第一军团是深渊大军的最强军团**——完全由**扭曲茂树人、深渊巨兽、暗黑巨人、幽鳞族、暗黑者**组成，看不到恶普族、黑骑士、蛇人族这些普通单位；加之**三分之一的「上古深渊存在」**都会在很大程度上听从莫亚姆调遣。

在这里，别说绝强、至强，**巅峰之境敢很嚣张地持续开领域，那也得死**（唯一例外是圣歌·萨曼娅——她的领域没有任何攻击性、甚至无法被察觉）。

### 苏晓的到来

传送过程本身就已异样:目光所见的一切被黑暗吞没，宛如乘坐一辆穿梭在黑暗液质中的管道列车、周边传来嘶嘶的摩擦声——⚠ **昔日虚空之树或乐园阵营的传送，从未如此过。**

落地即遭遇一把遮天蔽日的漆黑战锤。苏晓抬手，闪着血芒的指尖触碰上落下的巨锤——持锤的几万米高暗黑巨人被武器上的巨力掀翻，猩红裂痕从超巨锤蔓延到它的右臂、脖颈、胸膛，直到多瞳眼球猩红破碎，**当场崩碎为余烬**。

### 【刃之域】首秀

苏晓抬手朝向远处一名暗黑巨人大头目。那只自成为黑暗先锋后就再不知恐惧为何物的巨人，突然回忆起了幼时——那只吞吃了它双亲与弟弟妹妹、并从它藏身的枯木上走过的**渊底噬咬**，利爪踩在漆黑朽木上发出咔哒的声响，利爪上至亲的鲜血顺着缝隙滴落在它脸上。

下一刻，它与周边很大一片区域的深渊巨兽、扭曲茂树人**全身爆出黑血、一把把黑暗血刃半破体而出，当场骤死**。

⚠ 这就是【刃之域】。血刃脱离时摄取敌人剩余鲜血、转化为**泯灭血枪**飞回苏晓身后；随他前行，身后血枪累积到**上万把**并向四方射杀。以他为中心，深渊军团快速倒下——**冲锋的速度没有血枪屠戮的快**。

原本在浴血奋战的契约者们逐渐停下战斗，因为**他们面前的敌人都因体内突然爆发血刃而当场身死**。从上空俯瞰，涌来的深渊军团在数之不清的血枪与上空元素龙焰之下，**宛如被一只大手扼住，向它们自己的大本营赤红堡垒推进**。

### 元素虫族的复制

苏晓以召唤卷轴（⚠ **用轮回乐园产地的召唤术式而非灭法传送阵**）把 [棘拉](../../人物/棘拉.md) 召来。⚠ 风海大陆积攒的「元素晶核」带不过来——在「初始条例」下，乐园相关造物必须有公证才能进行界级互通，也就是需要向虚空之树申请公证那些元素晶核，**这是笔不小的费用**。

于是从一颗工蝎胎胚卵重新开始:菌毯最先蔓延 → 半小时后母巢包裹住深渊净化装置、高达几十米 → 首批「生命守护者」产出。⚠ 苏晓没让棘拉留下，而是让她给母巢设定自行指令（吸收与储存这么简单的指令，以当前的母巢等级当然可以无限循环）。

⚠ 深渊阵营那边显然已知道谁才是大敌:**有不少上古深渊存在潜入烈阳大本营内企图灭杀棘拉**，怎奈棘拉根本没在母巢内，只能尝试破坏母巢——⚠ **母巢被破坏完全不是问题，只要位于母巢地下部分的「元素晶核」不出问题，那都是小问题。**

灭世之翼冲出穹顶时，**首只刚飞出三秒就被黑暗者的百米粗光柱轰成漫天血雨**；但第二批几十只、第三批上百只源源不断——接下来就是元素龙焰洗地、菌毯快速蔓延、储存数字飞快滚动的老流程。

### 收官

赤红堡垒宛如按下开关，骤然不再涌出深渊军团，原有部队**首次撤退**，战场陷入诡异的死静。随后灭世之翼吞元素晶核、对赤红堡垒喷吐元素龙焰，两小时后深渊侵蚀度以肉眼可见的速度下滑，深渊之门展现后在周边几百股元素龙焰的喷吐下咔嚓崩碎。

⚠ 但星界阵营一方保持了清醒（圣域乐园巅峰**天帝**语）:

> **我们不是彻底赢了，是敌人迫于我方增援的强大，放弃了这主战场。我倒是希望深渊阵营死守这，这样在后续的虚空决战，我们就多了分优势。**

**苏晓的个人结算**:128532 点功勋值＋烙印 +5 → **Lv.110（烙印极限等级）**。
