# 灾祸军团

> ⚠ **一个「通过不断侵略、毁灭，才能继续存在」的势力**——⚠ **领袖是 [腐败领主](../../人物/虚空/腐败领主.md)**;⚠ **⚠ 52卷41章经 [阿格耶](../../人物/女巫界/阿格耶.md) 的心脏所化的黑暗空间门，入侵 [女巫界](../任务世界/女巫界-巫师界.md)·天空城。**

## 🦠 侵蚀三部曲

⚠ **入侵 → 侵蚀 → 同化**:⚠ **这就是灾祸军团把一个世界从生机勃勃化为一片暗冷破败的过程。**

| --- | --- |

| **腐化者** | ⚠ **核心战力**·数量多到难以统计（⚠ 普通腐化者已能把巫师军团小队长一爪拍飞） |
| **精英腐化者** | ⚠ 体长 12 米以上·⚠ **只有腐败领主能统御**（⚠ 侵袭多个世界、以亿万生灵才转化出） |
| **三从卫** | ⚠ **死咒男爵**（操控普通腐化者·伪装成巫师军团最高统领吉德乌斯·被苏晓斩杀）· **邪灵王**（四臂·乘腐化飞龙）· **[厄运夫人](../../人物/女巫界/厄运夫人.md)**（最弱·只掌控沉沦者） |
| **异魔首领** | ⚠ **腐败领主最信任的心腹**（⚠ 比三从卫更得信任·驻守永冬城原址的大本营） |

## ☠ 真正的杀招：深度腐化（52卷43章）

⚠ **巫师军团的「碾压式胜利」其实是陷阱**——⚠ **他们轰碎千米高的黑暗空间门、斩下「死咒男爵」头颅、山呼海啸地欢呼**;⚠ **⚠ 可九成以上被消灭的腐化者，都是故意留下的炮灰**——⚠ **真正的手段是「深度腐化」:肉眼不可见、甚至能屏蔽感知的腐化真菌飘在空气中，地下水早已被污染，天空落下紫黑色雨滴**;⚠ **⚠ 战后收拾残局时，士兵、队长、乃至最高统领·吉德乌斯本人陆续腐化——巫师军团在天空城战役阵亡超过七成**（⚠ 吉德乌斯化为死咒男爵）。

## 👑 腐败领主的由来

⚠ **详见 [腐败领主](../../人物/虚空/腐败领主.md) 页**——⚠ **他曾是第二纪元初一个九阶顶尖帝国的大统领**;⚠ **⚠ 他拒绝了「掩护王室与大贵族逃往联盟星」的命令、舍弃帝国战旗，率部退守人口最多的贸易之城·伯圣敦，镇守十五年、直到手下战士只剩一成**——⚠ **最后他站在高耸如山的巨兽尸堆上，失血九成以上，一言不发地坐在巨兽头颅顶，被聚拢而来的黑暗慢慢吞没**;⚠ **⚠ 大统领死了，腐败领主现世——他心中如九幽之寒的怨愤，让那个世界的初始深渊能量开始异变（更具侵蚀性、蔓延性、同化性、适应性），昔日部下逐渐站起来，成为第一批腐化者。**

## 📉 90 级 VS 110 级（52卷46章）

⚠ **灾祸军团一直自认在军团流体系中是顶尖梯队**——⚠ **可这次撞上的是「成功单挑永光世界内所有灭世级族群」的恶魔虫族**;⚠ **⚠ 数据化恒定:巫师军团 60 级 · 灾祸军团 97 级 · 恶魔虫族 110 级**（⚠ **100 级之后每 1 级都是一个梯阶**——⚠ 蛀世约 109 级，只差 1 级仍败在恶魔虫族之手）;⚠ **⚠ 邪灵王眼看着 100 万只恶魔焰龙从虚空之门涌出，再也顾不上三从卫的逼格，狠狠咽了口唾液、掉头就逃**——⚠ **龙焰洗地＋几万只泰坦巨兽的电浆炮雨，让永冬城原址的千万精英腐化者军团彻底覆灭。**

- ⚠ **⚠ 它们为何敢来**:⚠ **① 女巫界的军团流不堪一击 ② 有黑暗神教作同盟牵制 ③ 另一个隐秘同盟「陨灭星」给了腐败领主底气**;⚠ **⚠ 而神父三人明知苏晓有恶魔虫族，却绝不会告诉腐败领主**（⚠ 坦诚告知的话，腐败领主一定不会来）。
