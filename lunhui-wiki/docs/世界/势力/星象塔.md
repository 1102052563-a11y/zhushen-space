# 星象塔

> ⚠ **[女巫界](../任务世界/女巫界-巫师界.md) 最隐秘的势力**——⚠ **前身是「议会」**（⚠ 为抵抗深渊侵袭而成立的联盟·在成功抵抗深渊、本世界蜕变为超脱后，「议会」这过于强大与混乱的势力便没必要存在）;⚠ **⚠ 星象塔完美继承了议会的理念:不效忠于任何人，只为这个世界的正常运转而存在**——⚠ **成员不论善恶，只要不违反星象塔的规则即可。**

## 🗝 它守护着什么？

⚠ **月女巫·瑟希莉丝知道星象塔守护之物是什么，却不能说**——⚠ **她以任何方式透露相关之事，都会遭到本世界·世界意识的放逐**（⚠ 哪怕世界不能直接驱逐这位月女巫，也会不断加大排斥力度，直到她被驱离本世界）;⚠ **⚠ 苏晓的推断:黑暗双子这等 boss 只有「看门」的资格——星象塔所守护之物的重要性可想而知。**

- ⚠ **[星象圆盘](../../力量体系/装备道具/index.md) 是星象塔的至宝**——⚠ **吸收五份黑暗之血后可开启「星象塔」**（⚠ **苏晓现持有此圆盘，等于截胡了星象塔的至宝**）。
