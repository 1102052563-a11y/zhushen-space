## 东京喰种世界
### 喰种（种族）
- 外表与常人无异的**食人怪物**；体内 **Rc 细胞**活跃时**赤眼**，须吞食人类维持 Rc 细胞活性（二卷2章）。
- **Rc 细胞**：一种神奇能量，既强化人体、又能形成**赫子**；该世界普通人体内亦含微量 Rc 细胞（被 [青钢影](../../力量体系/技能.md) 视为可噬灭的「其它能量」之一）（二卷2–3章）。
- **赫子**：喰种由 Rc 细胞形成的武器，分 **羽赫 / 鳞赫 / 甲赫 / 尾赫** 四种，各有优劣、无固定形态、随喰种想象成型；赫包位置因种而异（二卷2–3章）。
- **库因克**：CCG 用喰种赫子制造的武器（二卷3章）。
- 已收录喰种：[神代利世](../../人物/神代利世.md)（S~SS 级·尾赫）、[壁虎](../../人物/壁虎.md)（鳞赫）。
