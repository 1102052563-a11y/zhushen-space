# 生物·万族

记录各任务世界中的生物，以及万族界相关种族。

## 生物分级
- **精英生物**：强于普通生物的强力个体（如科尔波山的主人）（第15章）。
- **领主级生物**：远强于精英·需冒险团围攻级别的强者（如希亚的死刑官 [敦克·尼克](../../人物/敦克·尼克.md)·十七卷）。
- **霸主级生物**：**高于领主级**的存在（十七卷51章·希亚圣殿守卫者 [里德](../../人物/陵墓守卫者·里德.md)）——「霸主权威」一项就给生命 +10 万；与领主级是**云泥之别**（领主级在霸主级前「不超两锤就成碎肉」）。

## 海贼王世界
- [科尔波山的主人](科尔波山的主人.md) — 科尔波山的巨虎，精英生物（第13–15章）
- **哈巴克（违规者 9740 号 · 外来物种）**（六卷57–58章）— 混进司法岛的**虚空外来生物**（轮回乐园判定「9/10 外来物种」·靠近猎杀者即开启 200 米淡金色空间结界）。形如全身覆**墨绿色剧毒液**的人形怪物（绿色竖瞳·分叉长舌·毒液滴地即腐蚀），**本体无实体**（斩击穿过如斩毒液、只有 [青钢影](../../力量体系/技能.md) 能量能伤）。
    - **毒性**：周身喷「未知毒素」（每秒 2 点伤害、恢复效果 −50%·直接侵入人体）；招式「蛇之毒」（铺天盖地喷毒液）、「毒之痛」（绿芒贯体·3 秒 120 真实伤 + 剧痛）。**弱点 = 盐**（盐水可化开毒液）、且**胸口正中心**藏着死穴（会主动闪避对该处的攻击）。
    - **真相**：胸膛正中嵌着一颗活的**人类头颅**——即 9740 号契约者本人（17–8 岁黑发少年），被哈巴克吞噬当「**进化载体**」、灵魂被啃食殆尽。少年被苏晓劈开胸膛后短暂清醒、含泪供出弱点（盐 / 头颅），求苏晓杀他解脱。苏晓用「白色小青蛙」（特级炼金炸弹）炸烂少年头颅、青钢影爆体，最终由轮回乐园（外来生物·无轮回烙印）放透明锁链**强行处决·寸寸搅碎**。掉落**灵魂结晶（中）×3** + 白银荣誉勋章（猎杀者表现优异全额奖励）。

## 东京喰种世界
### 喰种（种族）
- 外表与常人无异的**食人怪物**；体内 **Rc 细胞**活跃时**赤眼**，须吞食人类维持 Rc 细胞活性（二卷2章）。
- **Rc 细胞**：一种神奇能量，既强化人体、又能形成**赫子**；该世界普通人体内亦含微量 Rc 细胞（被 [青钢影](../../力量体系/技能.md) 视为可噬灭的「其它能量」之一）。CCG 凭 **Rc 细胞检测门**辨别喰种（其 Rc 含量是常人十几至上百倍；极强者可规避）（二卷2–3、8章）。
- **赫子**：喰种由 Rc 细胞形成的武器，分 **羽赫 / 鳞赫 / 甲赫 / 尾赫** 四种，各有优劣、无固定形态、随喰种想象成型；由**赫包**生成（赫包位置、数量因种而异，如高槻泉有 10 个）（二卷2–3、6章）。
    - **甲赫**：赫包多在肩部，展开如半身铠甲；耐揍、防御高，但速度慢（如拍卖场 S 级胖子喰种）（二卷22章）。
    - **尾赫**：各项素质均衡、下肢力量出色，但缺乏爆发力；善用者多为体术高手（如神代利世、拍卖场月山家族喰种）（二卷23章）。
- **等级**：D（幼儿）< C（能用赫子但弱）< B（普通·藏匿人类）< A（势力精英）< S（少数天才）；**S 之后只能靠互食**提升 Rc 浓度——喰种实力靠「吃」同类、并非锻炼。**独眼**为罕见喰种（如独眼之王·枭）；S 级以上密集的区域称**共喰区**（二卷5–6、13章）。
- **赫者**：吞食**大量其他喰种**后才能成就的强大形态（体型巨兽化、防御与攻击俱强）；因体内 Rc 细胞极活跃，精神趋于不稳（二卷6–7章）。
- **库因克**：CCG 用喰种赫包制造的武器（二卷3、6章）。
- 已收录喰种：[高槻泉（枭）](../../人物/高槻泉.md)（SSS·独眼·赫者）、[神代利世](../../人物/神代利世.md)（S~SS·尾赫）、[壁虎](../../人物/壁虎.md)（S·鳞赫）、多多良 / 野吕 / 雾岛绚都（羽赫）。

- **安定区喰种**：20 区安定区咖啡厅一群特殊喰种（店长 [芳村功善](../../人物/芳村功善.md)、雾岛董香、西尾锦等），**不主动捕食人类、只食自杀者的尸体**，谋求与人类和谐共处（二卷40章）。
- **肉墙**：24 区地下由**赫包**堆制的墙体（技术近似库因克制造），暗红坚硬而有韧性、会锁紧夹住刺入的刀、拔出后一分钟内自愈；盘踞通道、大幅提升探索难度（二卷43–44章）。

### 24 区特殊生物（地底）
24 区不止有喰种，还有人所罕知的怪物（二卷44–48章）：

- **光花**：攀附 24 区洞顶的灰色枝干植物，结幽白光花朵为地洞照明；其分泌的汁液**剧毒**，曾毒死 50 余名搜查官（二卷44章）。
- **血眼（动物喰种 · 精英生物）**：CCG 史上从未发现的**动物喰种**——一条体长数十米、三米粗、青黑鳞片的巨蟒，头颅中央生一只喰种赤眼；强到连 [有马贵将](../../人物/有马贵将.md) 都须避走（二卷44、74章）。
    - 苏晓【基础探测】所得（二卷74章）：HP 100%、法力 110、**力量 30（本世界极值）**、敏 12、体 ???、智 11、魅 2。技能：精英生物（力+10/敏+5/体+10）、热感应（追踪 +60%）、古生物（防+6/异抗+30%）、**疯狂撕咬**（咬中后若 HP<80% 触发体力判定、失败**即死**）等。
    - 有智慧、好以追猎为乐；先咀嚼再吞食。增援队遭其追杀，连特等 黑磐岩 都被一口吞没、瞬间丧命，27 人最终仅 3 人逃脱（二卷74章）。
- **原始人喰种（迪亚克曼部落 / 「原始种」）**：草裙 + 骨齿项链、脸涂臃肿鱼形图案；**不使赫子**（或无赫子），却力量强、肉体防御高、生命力顽强（断头仍咬人），部分持黑色石矛。成群（数百）行动、有首领（持树根、插鸟羽）。杀一名所得贡献是地面喰种的十余倍（46 点 + 赫包「一型」40 点），但触发**部落声望**机制（见 [机制](../../总览/轮回乐园机制.md)）（二卷45–46章）。
    - **真相（V 所知）**：CCG 总议长 [和修常吉](../../人物/和修常吉.md) 称原始种是「**失败品**」、虽无理智却**变相守护着「源」**——没有「源」喰种将绝种，故只可少杀、须驱回地底最深，不能大量诛灭（二卷70章）。
- **源**：一枚拳头大小、通体赤红、表面带裂痕的晶体，置于竖井最深处石台祭坛的石柱上、裂痕渗出淡红雾气飘向地表；是喰种存续的根本（**初代 Rc 细胞之起源**），由初代喰种 [马尔斯](../../人物/马尔斯.md) 与原始种守护。毁掉它会重创 V（有马 / 高槻泉的目标），而苏晓的隐藏任务是「**得到**」它——最终苏晓斩杀马尔斯、自其体内取出源、完成任务带走（二卷82、94–97章）。
    - 通往源的层层关卡：钢铁通道→竖井→刻「王冠 / CCE / 怪鱼 / 蛇」的金属门→原始种石林→**灼热岩浆湖**（湖中潜伏靠听觉感知的「**章鱼怪**」·暗红球体 + 触手·极恐怖）→石台祭坛。古代帝国背景词：博肯侯爵、圣鱼、圣战士、实验品三型 / 四型、CCE（疑与 CCG 同源）。
- **喰种的「王」**：地下喰种社会的统领。**上一代王 = [有马贵将](../../人物/有马贵将.md)**（有王之实力却非纯喰种）；这一代由 [高槻泉](../../人物/高槻泉.md) 一系（芳村）担当，但她是喰种却无「王」的实力。两人皆「不完整」，遂合作培养下一代「**完整的喰种之王**」——选中金木研（已被苏晓杀死，计划失败）（二卷68、81章）。
- **半人类 / 半喰种**：喰种与人类结合的两类后代（二卷81章）——
    - **半人类**（常见）：身体素质极强、却**寿命很短**（如有马贵将，二十几岁发白、估计仅余三四年寿、伴机能衰竭与青光眼）。
    - **半喰种**（独眼）：**极稀有**，移植喰种器官而成，目前已知仅 [高槻泉](../../人物/高槻泉.md)（金木研蜕变后亦属此类）。
- **无眼喰种**：皮肤焦黑粗糙如炭、尾赫常放、**双眼退化**（覆灰白硬壳）；**靠听力捕食**、极难对付，常上千只聚于空地（盘踞地底要道·「**禁声区**」）。苏晓三人以手机播放声源引开、潜行通过（二卷47–48章）。

> 24 区深处另有衣着正常、会说人话的**组织化喰种**（聚居「**中心城**」、有「首领」在地面与 CCG 抗争），与原始人 / 无眼喰种不同——详见 [地点 · 24 区](../地点/index.md)（二卷46章）。

## 进击的巨人世界
- **巨人**：巨大的人形生物（常见 5~15 米级）；**普通巨人无智慧、无目的**、模样呆傻、缓步前行、会下意识吞食人类，**夜间无法活动**，数量庞大。**唯一要害是后颈一片约 1 米宽 ×10 厘米的区域**——伤其它部位都会蒸汽缭绕快速恢复（三卷1–2章）。
- **超大型巨人**：约 50 米高的特殊巨人，惨白头骨与血红肌肉外露、周身蒸汽；曾踢破玛利亚之墙、又一脚在罗塞之墙踢出三道破口（三卷1–2章）。
- **奇行种**：行为不可捉摸的巨人——速度快、会四肢爬行 / 横爬 / 旋转跳跃、以鬼魅姿势近身，比呆板的普通巨人难防；但许多奇行种有探头吞咬的习惯、低头时露出后颈破绽（三卷5章）。
- **九巨人**：九种拥有智慧、可由人**传承**的特殊巨人（三卷11–14章）：
    - **进击的巨人**：现任传承者剧情主角 [艾伦](../../人物/艾伦.md)（约 15 米级·金色闪电中变身）；综合素质优秀、适应性强、为追求自由不断奔跑——无女巨人之灵敏、铠之巨人之防御、超大型之力量，胜在「适应」。首次变身无理智、会攻击其它巨人（潜意识保留战斗本能）。
    - **始祖巨人**：九巨人之最，可**指挥无脑巨人**（最大 3700 头·无条件顺从）；与进击的巨人之力同存于艾伦体内（暂未激活）（三卷22章）。
    - **女巨人**：约 14 米·灵敏（速度快艾伦数倍·人类杀手），有**硬化 / 晶体化**能力；传承者 [阿尼·利昂纳德](../../人物/阿尼.md)（马莱帝国间谍）。
    - **铠之巨人**：全身骨甲、防御极强（兵长利威尔都难破）；传承者 **莱纳·布朗**（间谍·融入人群过深致精神分裂）。
    - **超大型巨人**：约 50 米·力量惊人（踢破城墙）；传承者 **贝特霍尔德·胡佛**（间谍）。
    - **颚之巨人**：约 7 米·模样凶恶、四肢着地如野兽奔跑、不擅长距离、最强是一口咬合力惊人的**尖牙**；另有一手**吐牙**绝技——含住一颗尖牙全力吐出、动能不亚于炮弹。原属间谍组同伴贝利库（途中被无脑巨人吞食而亡），后该力转由 [尤弥尔](../../人物/尤弥尔.md)（曾化无垢巨人六十年、吞食贝利库而得力恢复人形）持有（三卷40–41、49章）。
    - **兽之巨人（野兽巨人）**：传承者 [吉克·耶格尔](../../人物/吉克·耶格尔.md)（马莱战士·艾伦同父异母之兄），截至本批留在墙外、未正面登场（三卷38、44章）。
    - **气运之子（不灭）**：本世界真正的「气运」不在某人身上，而**寄于巨人之力**——艾伦若死，其双巨人之力会随机转嫁给另一名「**艾尔迪亚人**」、成为下一任气运之子。故艾伦这类「主角」杀之无益（死了立刻有新主角接棒）（三卷22章）。
- 巨人**掉落宝箱几率很低**（普通巨人尤低、奇行种略高）；杀死后身体数分钟内气化、不留尸（骨头气化稍慢）（三卷5、8章）。

!!! warning "巨人的真相（三卷18章·艾伦记忆揭示）"

    - **人可变巨人**：注射**巨人药剂**会使人变成**无脑巨人**（智慧丧失、本能吞人）。
    - **无脑巨人吞食「智慧巨人」可恢复人类**、并**传承该智慧巨人之力**——这正是普通无脑巨人不需进食却本能渴望吞人的原因（潜意识想吞到智慧巨人变回人）；但智慧巨人仅九个，几率渺茫。
    - 传承巨人之力者**寿命仅余十三年**。
    - **无垢巨人的来源（三卷41章补）**：马莱帝国以注射巨人药剂、把犯罪（或被定罪）的**艾尔迪亚人变成无垢巨人**作为惩罚——他们虽存自我意识却不能控制身体、如幽魂般游荡，生不如死；马莱将这些无垢巨人统一运到墙壁所在的**帕拉蒂岛**，这正是墙外巨人杀之不尽的真正原因。无垢巨人若机缘巧合吞下「智慧巨人」便可恢复人形（如尤弥尔游荡六十年后吞贝利库得颚之力）。
    - 艾伦十岁时被父亲注射药剂变无脑巨人、**吞食了身为智慧巨人的父亲**，从而恢复人形并同时传承**进击的巨人 + 另一种「最强巨人」之力**；父亲留下钥匙、嘱他回玛利亚之墙地下室寻真相（三卷18章）。

!!! abstract "巨人 / 九巨人的起源（三卷27章·世界史）"

    - 帝国统治大陆 1700 年，后被马莱用离间计挑起九大家族内战、夺得七种巨人之力；仅余最强一支（近始祖、能控艾尔迪亚平民的「次级巨人」）。**始祖巨人传至 145 代**，该代王带艾尔迪亚人逃往小岛、用始祖之力驱巨人**硬化成三圈墙**（玛利亚 / 罗塞 / 希娜），自立为**初代王**、并夺「篡改记忆」之力抹去墙内人对大陆的记忆，与马莱签互不侵犯，换来墙内**百年和平**。
    - 墙外无脑巨人是马莱破解巨人之秘后的试验品。马莱觊觎小岛资源，派四名巨人间谍潜入：[阿尼](../../人物/阿尼.md)（女巨人·13 岁）、莱纳（铠之巨人·18 岁）、贝特霍尔德（超大型·17 岁）、贝利库（颚之巨人·途中被无脑巨人吞食而亡）——贝特踢外墙、莱纳撞内门，剧情由此开始。

## 斩·赤红之瞳世界

## 二阶生存试炼世界（T12K星球 · 吞噬岛 · 第八卷）
详见 [二阶生存试炼（T12K星球·吞噬岛）](../任务世界/T12K星球-生存试炼.md)。岛上生物极凶，**红土区（>Lv.17）的水不可饮、生物不可食**（满身寄生虫）。

- **亚森曼人（红皮土著）**（八卷15–19章·红土区主势力之一）：人类模样的直立生物·**黑红皮肤**、脸涂图腾鸦纹、鼻穿兽牙、**身材高大（2~2.5 米）**·凶蛮残酷（看非同类皆如食物）；懂生火、群居有图腾（藤蔓类）、平齿能咬断骨头吸髓、每天用木炭清洁牙齿（牙=生命）。把抓到的契约者割头掏脏风干当干粮。属性 **力 32 / 敏 29 / 体 48 / 智 39 / 魅 −2 ｜ 法力 390**；技能①野性（被动·生命 +100·力 +10·体 +10）②血藤蔓图腾（主动·**燃烧生命爆发战力**）③④未知。**身体强度 / 生存力恐怖**（中胸口、肋骨外露仍能战；脖颈才是弱点）、**无明显痛觉**、**无夜视**（夜晚靠篝火）。
    - **战术阵型**：常年与野兽战斗、彼此无需言语——一名正面强攻（石斧土著·黑矿石斧一斧碎山石）+ 数名木矛包抄（木矛打磨后**比金属还锋利**·一拥而上把敌人架离地）+ 年长**吹箭土著**远程伺机（**箭毒蛙神经剧毒**·每秒 10 真伤 + 神经麻痹·苏晓体力判定过、异常抗性未过而半身麻木·青钢影驱毒需时间）。
    - **酋长「巴里」**：那名吹箭土著首领即**巴里**（长老 [巴尔](../../人物/巴尔.md) 的**弟弟兼竞争对手**）。八卷18–19章苏晓双枪连射 + 炼金炸弹 + 闪光弹 + 界断线 + 布布汪死咬其屁股（高等生物免疫箭毒蛙毒），险胜全歼这 43 人营地、**斩杀巴里**（红土区**关键人物**·**世界之源 +8%**·首次在试炼中得世界之源·掉「**酋长包裹**」类宝箱）；陶罐里更藏着 **5 颗灵魂结晶（中）** + 黑森林某生物的「未知生物粪便粉」。这 43 人营地全是成年男性（疑被大部落驱逐、女性被抢的小部落·**昨晚那批是「血藤战士」精英**·身体强韧如铁、无视青钢影剧痛）。
    - **血藤战士（亚森曼部落秘法产物）**（八卷24–26章）：部落为对抗虫族暴兵、除靠惊人生育力（女土著每胎 2~5 个、一战士妻产 15 胞胎、大本营 40 万 +）外，另有一种**产出更强战士**的秘法——产物**失去大部分知觉、嗜血冷酷呆板、有太阳时战斗力极强**；**副作用 = 会失控自爆**（体内点起「小太阳」·焚妻成炭·自身炸成玻璃化大坑）。其名「血藤」、与部落图腾「张牙舞爪的藤蔓」+「黑色种子」+ 灵魂结晶疑有关联（秘密藏于部落后山·待解·苏晓直觉揭开恐得海量世界之源）。
    - **原住民 vs 入侵者**：亚森曼部落是这颗 T12K 星球红土区的**原住民**，与**外来入侵**的**刺蝎（虚空虫族）**（见下）已**战争三百余年**（红土区 70% 被两族瓜分·亚森曼占 31% / 虚空虫族占 39%）。**战场上的普通土著比那批精英弱得多**（会惨叫、有恐惧、更接近人类·几百也能被苏晓轻松解决）。
    - **大本营部落**（八卷23章）：远离战场（怕殃及族群）、石屋林立、几百名女性土著 + 大量幼童（**一夫一妻制·每胎生 2~3 个**·孩子分三个年龄段）。长老 **[巴尔](../../人物/巴尔.md)** 会说通用语、待苏晓以食物淡水（苏晓获**阵营好感「友好 21/500」**）——可见**「在生存基础上探索世界」会带来巨大好处**（试炼的隐藏机制）。
    - **真相 · 看守者（八卷49–51章）**：亚森曼人**原是虚空种族**——奉「虚空中一位王者」之命**驻守吞噬岛、看守某种东西**，已守了**几千年**；困守孤岛、无法离开，文明科技逐代退化到近乎原始人。几百年来**一直是亚森曼主动攻打虫族**（非虫族入侵），只为夺取母巢产的**再生卵**（血藤战士唯一生路）。他们**血藤战士并非为对抗虫族而生，而是为应对「寒灾」**；定居四面悬崖、仅一条石桥的红土区，正是把这里当防黑色森林之变的天然堡垒。
    - **太阳部落古圣地 / 阳之女（八卷51–52章·后山）**：血藤仪式在部落后山一处山洞「**太阳部落古圣地**」进行（契约者踏入即得**世界之源 +3%**）。仪式真相 = **把生根的 [原生之种](../../力量体系/消耗品材料.md) + 灵魂结晶植入活体、血藤接驳血管供血**——这股「力量之源」实为红土区地下一株巨藤「**[阳之女](../../人物/阳之女.md)（血藤之母）**」。亚森曼先祖与她立**共生契约**：移植原生之种即获植物 + 火焰之力（成血藤战士），代价是阳之女**抽离所有血藤战士的生命力**（故血藤战士短命、归宿即「回归太阳」自爆）。仪式「成功」出老巴尔型，「成功一半」出巴里型（呆滞嗜血无痛觉）。
- **黑色森林 / 寒灾（八卷48–52章·伏笔）**：吞噬岛普通区域外围紧挨着**黑色森林**——里面封着「**不死者**」与各种怪物。每隔一段会有「**寒灾**」降临，黑色森林的不死者「出笼」、怪物复苏（**没有寒灾就不会有血藤战士**）。[阳之女](../../人物/阳之女.md) 警告寒灾**五天后**就要开始、初始阶段就足以致契约者于死地。进入黑色森林必须持有一定数量的 **[原生之种](../../力量体系/消耗品材料.md)（黑色种子）当「门票」**，否则必死。苏晓的隐藏任务第二阶段正指向此处。
- **虚空虫族（社会结构 · 八卷37章）**：外来入侵红土区的虫海种族，靠**母巢 + 女王**繁衍暴兵。其聚居地是**五座绿色椭圆、会轻微颤动、表面有血管的「虫巢」**（一大四小）：最大那座 = 女王住所「**母巢**」（刺蝎 / 更高等虫族护驾·**再生卵**就在女王身边）；另四座 = **卵化池**（暴兵）/ **生物侦测组织**（警戒）/ **防空塔**（对空）/ **进化所**（产出高级虫族）。**虫族女王能魅惑人**（巴尔反复叮嘱「别被她魅惑」）。
    - **刺蝎**（基础兵种）：3 米长、黑色外骨骼的**巨蝎**·尾部与口腔各一大钳——**完全为战斗而生**（**两个心脏、无大脑**·苏晓一眼联想到「虫族」）。属性 **力 16 / 敏 14 / 体 8 / 智 5 / 魅 −1 ｜ 法力 50**（单体弱）；技能①外骨骼（防 +2）②酸性血液（腐蚀）③**虚空虫族**（被动·极恶劣环境生存·消化 80%+ 有机物）④**集体作战**（数量 >300 力 / 敏 +2·>1000 +3·**>10000 +5**）——靠**海量数量**碾压。
    - **爆炸蚁**（八卷37–38章·「生物地雷」）：藏身地表、只露头部的兵种——踩中其头部即触发：**新生爆炸蚁踩上就爆**（感知敏锐），藏匿超一周的则不那么敏锐（**不移开脚就不爆**·最多活一月）；威力极大（炸死人没问题·三米大坑）。母巢外围的「**爆炸区**」即布满爆炸蚁、是侵巢必经之路。
    - **精英刺蝎**（八卷40章·女王护卫）：体型大几圈、甲壳**紫红色**的刺蝎异种——**有大脑、有智慧**（懂偷袭 / 战术·会侧步绕侧面），负责保护女王。属性 **力 42 / 敏 45 / 体 29 / 智 11 / 魅 1 ｜ 法力 290**；技能①外骨骼 Lv.7（生命 +260·防 +9）②酸性血液 Lv.5③**独立个体**（有大脑）④**尾刺 Lv.12**（巨大伤口 + 断骨 / 流血不止）⑤**战斗本能**（全身皆为武器·苏晓最忌惮此技）。苏晓判断「**五只以内能对付、超五只很麻烦**」（曾被 10 精英刺蝎 + 5 飞螳追得拔腿就跑）。
    - **飞螳**（八卷37、40章）：身体细长、生一对肉翅、能低空飞的虫族·**喷射强酸**（杀伤极强）——综合属性不强，但在母巢通道 / 空旷地很麻烦（苏晓一行就是被飞螳冲散的）。
    - **分解菌**（虫族设施）：母巢外覆盖的一层绿色苔藓——**死在其上的生物会被迅速分解、养分输回母巢**。
- **3 号太空飞船**（八卷40–41章·母巢深处）：母巢最里层竟**包裹着一艘抛锚的虚空飞船**（钢铁科幻结构·多年未启动·电子器械损坏冒电火花）——正是 [虫族女王「亚」](../../人物/虫族女王亚.md) 逃亡迫降时坠毁的那艘。其「**核心能源**」可回收，苏晓拆下出售换得 **世界之源 12%**。印证**虚空虫族本就来自虚空**（虚空辽阔·种族繁多·有的科技先进、有的肉体强大）。
- **踏蹄**（八卷37章·坐骑）：似马的**肉食性**动物·耐力极强，驯服后是红土区的好脚力（巴尔 / 卡洛的坐骑）。
- **巨型蜈蚣**（八卷13章·红土区）：直径约两米、体长未知、藏身泥土、血盘大口满排尖牙——能**误导感知**、破土活吞猎物（苏晓险被吞·靠骑布布汪 + 炼金炸弹脱身）。
- **钢羽黑鹰**（八卷13章·空中）：体长至少 5 米、羽毛黑如钢铁、**敏捷 40+**·力不低；会发出禽鸣「交流」（布布汪学其叫声竟与之打成一片）。
- **红土区中心的猛兽群**（八卷31–32章）：红土区**正中心**（靶心）盘踞着各种凶兽——苏晓登岛见过的**巨型蜈蚣**也在其中。代表是**蛮角兽**（八卷32章·哈瓦呐之眼读：**力 42 / 体 45**·像四米高七米长的河马、实为食肉兽·把任一种放回地球都是食物链上层）；另有 3 米猎豹、雄狮（掉绿宝箱）、犄角羚羊等。这些猛兽被**[原生之种](../../力量体系/消耗品材料.md)（黑色种子）的致命吸引力**驱使，连兔 / 龟等温和食草兽都疯抢——[腹黑女](../../人物/index.md) 八卷31章正是用三颗原生之种引来**几万头野兽围困山谷**做死局。
- **巴哈姆（翼龙 · 霸主级生物）**（八卷33–34章·红土区中心顶级）：橘黄色、翼骨突出、有黑色花纹的**大型翼龙**，能轻松单爪拎起以吨计的蛮角兽飞行——**红土区中心的绝对霸主**（哈瓦呐之眼仅探得 11% 资料·力 / 敏 / 体 / 智 / 法力**全是「??」**·魅 51）。技能①**红土霸主**（被动·生命 +10000·力 +25·敏 +25·体 +30）+ 技能 2~9 全未知；苏晓自评「**连百分之一的胜算都没有**」（问号越多 = 差距越悬殊）。在陡峭高山顶的山洞筑巢（一家四口·两只成年 + 两只不会飞的幼崽），**用「太阳恩赐果实」的汁液喂幼崽**使其更强壮；幼年期却很脆（被苏晓一脚一个踹成熊猫眼·但苏晓不敢杀·在海岛杀飞行霸主的幼崽是「最愚蠢的选择」）。苏晓藏在被巴哈姆抓走的蛮角兽尸体内、又趁尸体被抛下山崖时挂峭壁逃生。
- **双头犬 / 蛮角兽等中心猛兽**：红土区中心还有**双头犬**（中心区强者·咬蛮角兽尸时被藏身其中的苏晓界断线刺喉拖入斩杀）、**蛮角兽**（见上）等——但**99% 的中心野兽都去抢 [原生之种](../../力量体系/消耗品材料.md) 了**，苏晓返程时畅通无阻、连昆虫都没见。
- **红土区的水 / 鱼**：清澈河水实则满是寄生虫（喝一口必死）；河中怪鱼（半米·头大身小·身上挂几十种寄生物）也不可食。唯拇指大的**蜜蜂**可生食（味美·苏晓在地球见过近似种）；中心地带另有一条**相对安全的河**，苏晓用「户外净水管」过滤后才敢喝。
- **狮群（普通区）**（八卷59章）：普通区野兽虽不如红土区凶险，却胜在**种群庞大**——苏晓撞见一个**超两百头的狮群**（追杀小迷糊 / 无伞兄），被他十秒斩出大片残骸、再以「杀了几千生物才有的气息」（小迷糊误认作「霸王色霸气」）一震、令巨型雄狮率群狼狈退去。
- **巨人（黑色森林边缘 · 守门者）**（八卷60章）：黑色森林外缘一头**四五米高的类人生物**——外形近人但**鼻子很大、秃头、留长黑须**，身穿各种兽皮缝的衣服、棕黄色眼睛，说一口瓮声瓮气的通用语，**蛮横却「信誉很好」**。它向路过者讨要**一颗未经 [阳之女](../../人物/阳之女.md) 处理的原生之种**（直接扔进嘴里吃下·满足地打饱嗝）当「过路费」，**回赠一张黑色森林兽皮地图**（[阳之女](../../人物/阳之女.md) 早安排 [无伞兄](../../人物/无伞兄.md) 带这颗「麻烦的」生种就是为换它）。走路时一直侧头戒备着什么。此即传说中 [阳之女](../../人物/阳之女.md) 用亚森曼地图讨好的「**巨人族**」。

### 黑色森林生物（八卷62–65章·牢笼内）
黑雾笼罩的牢笼内**生态畸形**——林中野兽**皆带毒不可食**（能在黑雾之源侵蚀下活命、体内多有毒素）。

- **红色毒虫**（八卷62章）：树上落下的红色小虫——**体液剧毒**，仅划破其外皮、滴落的青烟就把一把匕首**瞬间腐蚀成千疮百孔**。苏晓凭超强视力（非感知）才发现它落在无伞兄肩上、以匕首挑飞。
- **犀牛群**（八卷63章·食物链最底层）：三百余头、体型比地球犀牛更庞大的「犀牛」——本该懒散凶悍，这里却**胆小如兔鹿**，见苏晓以普通人气息走近就**四散奔逃**。这种「畸形生态」（最强者反成最底层猎物）正说明黑色森林之危。
- **不死者**（八卷63–71章）：黑色森林内**数以万计**（[暗王](../../人物/暗王.md)「不死者头领」技能标明**共 67.5 万**）的存在——身体干枯的人类或野兽。**斩首 / 碾碎都无效**（打碎只是使其失去战斗力·剩一根手指仍能行动），**唯有火焚才能彻底消灭**。单体极弱，但靠**人海战术**——几千尚可杀出，上万则会被拖垮。入林时它们远远包围小队（几百瞬增至几万）、却畏惧某种存在不敢近身、反吓退普通野兽。**试炼最后一天黑雾散尽，67.5 万不死者倾巢涌出**、瞬间淹没普通区（90%+ 契约者逃回红土区），是吞噬岛真实危险远超试炼难度的体现（详见 [鼠大人](../../人物/鼠大人.md) 镇压）。

- **扭曲（次级霸主生物）**：人类 + 幽鬼 + 妖魔 + 诡兽**拼接**而成的怪物（三米五·遍体外骨骼凸起·力 115 / 体 118·**肉体防御 120+**·豁免 30% 伤害·无视濒死）。两形态（半人类 / 诡兽）·撕扯 / 高声波震荡眩晕 / 扑杀 / 二度扭曲（濒死时 2% 几率进化为真霸主）。智商仅 15、靠蛮力（无技巧）。被 [罪恶](../../人物/罪恶.md) 高价买来当「宠物」守眼室——制造者被它生吞、前两任主人非死即残。被 [苏晓](../../人物/苏晓.md) 引到旷地用噬灭风筝 + 哈迪斯 + 魔刃斩杀（独立击杀得 100 灵魂钱币 + 扭曲精魄）。
