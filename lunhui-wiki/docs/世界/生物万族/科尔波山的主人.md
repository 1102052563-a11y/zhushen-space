# 科尔波山的主人

**科尔波山的主人** 是 [海贼王](../任务世界/海贼王.md) 世界、[科尔波山](../地点/index.md) 的一只巨虎，**精英生物**（第13、15章）。

## 形态
外貌似普通老虎而体型庞大：高四米多、体长七八米，比地球大象还大、约一间房屋大小、重达数吨；深黄色毛发光亮。智商极高（约十岁儿童以上），雄霸科尔波山多年（第13章）。据苏晓原著记忆，曾险些杀死幼年路飞（第11章）。

## 被击杀
为完成左大臣的支线任务，[苏晓](../../人物/苏晓.md) 用 [木天芷](../../力量体系/消耗品材料.md)（类猫薄荷的干果）诱其入陷阱、引爆埋设的炸药与铁钉重创之，再以 [斩龙闪](../../力量体系/装备道具.md) 缠斗——斩爪、断后腿筋、放血，最终刺穿脑髓 + 燧发枪爆其独眼而杀（第13–15章）。

- 苏晓代价：右臂被咬成重伤（骨骼无损）（第15章）。
- 击杀收获：世界之源 +3.2%、噬灵者 +15 法力；掉落**绿色宝箱**（苏晓首次见到）+ 虎齿（任务道具）（第15章）。
