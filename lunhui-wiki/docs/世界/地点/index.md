# 地点

记录乐园设施，以及各任务世界的重要地理。

## 海贼王世界
- **香波地群岛（失败与再出发之岛 · 十八卷58–60章）**：位于红土大陆下方、伟大航路前半段的尽头——由 **79 颗「亚尔其蔓红树」**盘结的树根组成（远看翠绿一片）。海贼想进新世界**必经此地**，故称「失败与再出发之岛」。世界政府曾在此设关卡镇守 267 天、付出 2 中将 + 5 少将 + 8 万海军 + 5 万牺牲（海贼联合冲关、年轻的战国驻此累成狗）后放弃——如今形成**独特生态平衡**：海军 / 海贼 / 世界政府 / 天龙人 / CP / 革命军都会出现、却不轻易出手。岛上 **79 个区（小岛相连）**分四大块：① **1~29 区**（人口贩卖 / 拍卖会 / 酒吧 / 销赃 / 不法地带 / 医院·鱼龙混杂）② **30~39 区**（泡泡公园 / 美食街 / 商业区·相对安全）③ **40~69 区**（观光 / 船坞 / 海军驻扎）④ **70~79 区**（旅馆 / 欢乐街 / 情人小道·灰色产业）。地下红树根分泌天然树脂、覆盖 60% 地面、浮出**七彩肥皂泡**（可建房 / 代步 / 给船镀膜）。**1 号区的唐吉诃德家族拍卖会**是岛上最大、无人敢竞争（即 [苏晓](../../人物/苏晓.md) 袭杀天龙人的预定地）。
- **红土大陆（十八卷54–55章）**：一道横贯世界、赤红色的巨大「天壁」。海军凭借它修建出人工岛、设海军分部（麻雀虽小五脏俱全）。[苏晓](../../人物/苏晓.md) 被海军假降押送、途经此处人工岛分部（前往司法岛中转）；港口列着鹤中将的百名女子精英部队 + 上千男海军。
- **万国 / 托特兰（新世界 · 大妈地盘 · 十八卷18章）**：四皇**夏洛特·玲玲（大妈）**的老巢——官方名「托特兰」、海贼惯称「**万国**」（指代一整片海域群岛、非单岛）。住着几乎所有种族（人类 / 鱼人 / 毛皮族 / 长手族…）、**唯独没有巨人族**（玲玲与巨人族有旧怨）。岛屿建筑 95% 可食用（核心岛 = **蛋糕岛**、另有由坚硬冰淇淋构成的「冰岛」）。十八卷18章 [枯屠](../../人物/枯屠.md) 冒险团在此掀龙卷风 / 音爆刷海贼声望、拖住大妈。
- **德雷斯罗萨（王国 · 新世界 · 十八卷12章）**：[多弗朗明哥](../../人物/多弗朗明哥.md) 统治的新世界知名大型岛屿（被巨大岩石围拢、易守难攻）。**繁荣在地表、黑暗在地下**——普通人眼中是景色优美、美女美食之地；海贼眼中是赌场 / 补给站 / 庇护所；地下世界眼中是**军火贸易站**（有钱能采购任何军火·真正做生意的船进岛下地下港口）。国王做黑色生意、故岛上经济富足。特殊区域：**斗牛竞技场 / 恋人大道 / SMILE 工厂〔人造恶魔果实厂〕/ 花田〔大片向日葵〕/ 王之高地**。
    - **王之高地**：德岛最高点、原王宫改建的唐吉诃德家族别墅区（干部居住·俯视全岛 + 连通地下港口·家族老巢·靠升降梯出入·非家族干部禁入）。苏晓在此与多弗朗明哥谈判、并把炼金炸弹作坊设在别墅四层（[琵卡](../../人物/琵卡.md) 把墙地加厚至 3 米 + 钢板）。岛上近半居民被 [砂糖](../../人物/砂糖.md) 的童趣果实变成「玩具」、多在地下港口当苦力。
- **哥亚王国**：东海国家，统治附近城镇村庄；周边有风车镇、科尔波山；首都即「王都」（第2、4、6章）。
    - **王都**：分平民住宅区等区域；街道熙攘，但平民挣扎在温饱边缘（第6章）。
    - **欢乐街**：王都的花街，粉红灯光、楼阁林立；为左大臣经营、侍卫首领欧卡常客，苏晓在此暗杀欧卡（第9章）。
    - **嘟噜杂货铺**：王都一间生意萧条的杂货铺，常被守卫队压榨；苏晓在此购得木天芷（第12章）。
    - **王宫**：国王所在，戒备森严；主线刺杀的最终舞台，苏晓于第20章末潜入（第2、20章）。
- **废品终点站**：哥亚王国城外大型垃圾场、流民聚居的「无法地带」；**第20章被克利克海贼团焚烧**，数千流民遇难（第3–6、20章）。
- **科尔波山**：哥亚王国周边的山林，原著主角路飞长大处；风景秀丽但猛兽出没、水源稀少（海岛缺水）；精英生物 [科尔波山的主人](../生物万族/科尔波山的主人.md) 的领地（第3、11–16章）。
- **风车镇**：哥亚王国周边地名（第3章）。
- 其它（提及）：王都监狱（苏晓在此救出死刑犯做交易）、平民区旧仓库（密会克利克）（第19、20章）。

### 七水之都（第六卷再入）
- **七水之都（Water Seven）**——伟大航路上的「水之都」。本是一座陆地岛，岛体下沉后居民不迁、在旧建筑上层层加盖、高出海面，千百年累积成**铁塔状**的海上城市（顶端涌海水如喷泉、沿数条**水道**流回大海·「名副其实的海景房·窗口就能钓鱼」）。海军与海贼都来此买船 / 修船、彼此克制；只驻十几名海军（地位特殊·古代兵器**冥王**即在此建造·世界政府对其态度暧昧）。海贼须从**后街**低调进城（遮羞布）。**交通工具「布鲁」**：温和的两栖「牛鱼」、背驮带座椅小船载人穿行水道（租约 1000 贝利）（六卷4–5章）。
- **一号船坞**：[卡雷拉公司](../../势力/index.md) 旗下船厂之一，堆满木材、火炮；苏晓受聘负责其安保。**市政办公楼**（人工陆地区·权贵聚居·社长冰山办公处）。
- **船墓场**：废弃造船厂、堆放旧船之地；人烟稀少，苏晓选作与左轮男决斗的战场（六卷6章）。
- **水之诸神（年度大海啸）**：每年一次席卷七水之都的巨大海啸——居民躲进后街避难所或乘海上列车避开；海啸却**冲洗全城**（卫生大好、少疫病）、**冲来鱼群**（渔业富饶），故居民虽住在下沉的岛上也不肯迁走。海啸前全城办**庆典**（化妆舞会·华丽长袍 + 肃穆木质面具·祭水神祈福）——苏晓、布鲁诺正借庆典服面具掩护抓捕罗宾（六卷22章）。
- **废弃工厂 / 废弃码头**：七水之都僻静处，苏晓练剃、押罗宾审讯、与璃 / 莉莉丝交战的场所（六卷17、23–25章）。
- **海上列车（海列车）**：海面上铺设铁轨、靠煤炭驱动的列车，速度是船的十几倍、可克服一切恶劣气候；连接七水之都与司法岛。苏晓一行押着罗宾、弗兰奇乘此列车赴司法岛（贵宾在第三节·弗兰奇被关第七节）——苏晓正是在暴雨夜爬车顶潜入第七节、盗走弗兰奇体内的宝物（六卷33–37章）。
- **司法岛**：海贼王世界主线的最终舞台、世界政府的法外审判 / 处决之地（鱼人汤姆即在此被处决）；藏六式、恶魔果实等好东西、亦极危险。**三座岛由吊桥相连**：前岛（正门）→ 本岛（审判所·走流程·凡押来者皆已定罪）→ 司法岛（司法之塔·关押要犯）；司法岛后经**踌躇之桥 → 正义之门 → 大漩涡**（漩涡把海军本部、司法岛、推进城监狱三大要害连通）。苏晓预判**草帽海贼团与大量契约者都将汇聚于此乱战**（六卷6、30、38章）。
- **海军秘密基地（荒岛）**：司法岛附近、乘小船半小时可达的隐秘海岛（无码头·林间遍布十几座瞭望塔 + 隐藏大炮 + 数百埋伏海军·已肃清大型动物·驻守 4 万精英海军 + 黄猿 + 改造中的暴君熊）。世界政府的天才博士**贝加庞克**在此设耗资 50 亿贝利的大型实验所（正改造王下七武海熊·PX 计划）。苏晓被海军「护送」来此**量产炼金炸弹**；地下斜下方另有**紧急避难所**（苏晓在此试制太阳神阿波罗·失误炸出 300 米巨坑）（六卷40–45章）。
- **玛丽乔亚（提及）**：世界贵族**天龙人**的住址 / 圣地；世界政府供苏晓造太阳神阿波罗用的灵魂结晶即出自此地。苏晓动了「洗劫玛丽乔亚必暴富」的念头，但前提是先灭世界政府、难度太高（六卷44章）。

## 东京喰种世界
- **东京**：喰种潜伏的现代都市，分区管理；开局大楼（**20 区**）为神代利世吃人之处（二卷1–3章）。
    - **20 区**：设 CCG 分部（门口有 Rc 细胞检测门），苏晓初入 CCG 处（二卷8章）。
    - **14 区**：苏晓任职区（紧挨 20 区、不太平）；下辖**西街区**——14 区的**共喰区**，S 级喰种极多（二卷9–13章）。
    - **11 区**：青铜树老巢，近乎人类禁区（喰种困住平民当「口粮」）；后青铜树**攻陷 11 区 CCG 分部**、使其彻底沦为喰种领地，金木研被抓至此折磨（二卷13、34、38章）。
    - **24 区**：**人类禁区**——并非特定区域，而是喰种在东京地下挖出、遍布全城、错综复杂尚未探尽的通道网；地面入口在东京边缘郊野（直径十几米的洞口直通地下、风声如鬼哭）；「A 级遍地走、S 级多如狗、SS 级才能勉强抖一抖」，共喰为常态（无人类可食），深处喰种或数以万计。地下多**肉墙**（赫包所制、类库因克技术）大幅抬高探索难度。准特等搜查官都无资格独自进入，唯 [有马贵将](../../人物/有马贵将.md) 的零队敢深入；苏晓随其展开第 108 次探索（二卷36–37、42–43章）。
        - **24 区临时分部**：地面一栋三层别墅，仅有马、苏晓、[尤娜](../../人物/尤娜.md) 三名搜查官 + 后勤 / 信息调查人员；其**后勤部**是「**隐藏区域商店**」，可凭 CCG 贡献值 + 职位兑换稀缺物品（人体基因强化液 / 粉碎精灵 / C 型注射器等，他区无此功能）（二卷42–43章）。
        - **金属门关卡**：CCG 早年所建、阻 24 区喰种外出；钥匙仅两把，分别在 [有马贵将](../../人物/有马贵将.md) 与 CCG 局长**修吉时**手中。门内即 [肉墙](../生物万族/index.md) 世界（二卷44章）。
        - **「活的迷宫」**：地下肉墙非固定，岔路时为二 / 三 / 五条、约七八天一大改（平时细微变化），故无法彻底探索；整体呈**漏斗形**（深处为大头、地面为小头），凭**指南针**锁定一个方向便可走出（苏晓据 17 条死路推断出漏斗结构）（二卷44、47章）。顶部有剧毒 [光花](../生物万族/index.md)、地底盘踞 [血眼 / 原始人喰种 / 无眼喰种](../生物万族/index.md)。
        - **中心城（地下城市）**：24 区深处掏空土层形成的**东京最大喰种聚居地**——金属建筑林立、设几十个入口孔洞，喰种如人类居于地面般在此生活；有「**主控室**」操控 24 区肉墙的变化（故地下通道总令人迷路）。暴走后被原始种 / 无眼喰种占领、遍地老幼喰种尸体（二卷46、75–76章）。
            - **井（迪亚克曼部族出口）**：中心城中心一口直径十几米、深不见底的竖井，原始种正是自井下深处爬出；井盖为特殊合金（每扇约八九百斤）。苏晓与有马以诱饵把原始种引到井口、引爆炸药尽数炸杀、并炸毁竖井封死出口（乐园判为「炸毁迪亚克曼部族出口」、世界之源 +5%）（二卷77–78章）。
        - **古密室 / 旧实验室**：肉墙缝隙后的一间古老房间，满布灰尘、无电子设备（天平、烧杯、木桌），疑为远古遗留的实验室；苏晓与尤娜曾藏身于此疗伤（二卷51章）。
    - **1–4 区**：喰种密度极大的共喰区，普通人无法生存（二卷13章）。
    - **库克利亚（喰种收容所）**：地处 23 区的喰种监狱，关押实力强、但非 V 成员的危险喰种（V 觉得杀了可惜）；CCG 在此对喰种做残酷实验（壁虎的扭曲人格便是在此被一名搜查官长期折磨而成）。高槻泉被「生擒」后即押往此处（二卷84章）。
    - **安定区咖啡厅**（20 区）：喰种聚集地，店长 [芳村功善](../../人物/芳村功善.md) 是年老的 SSS 级喰种（战技强于枭）；店长与店员皆喰种，却主张与人类和谐共处、不主动捕食（只食自杀者尸体），20 区双方互不主动出手（二卷9–10、40章）。
    - **Helter Skelter 酒吧**：14 区西街区一间复古无窗酒吧，SS 级喰种 [伊鸟](../任务世界/东京喰种.md) 开设，是喰种的小型情报中心；苏晓两度造访问取共喰区与 14 区喰种聚集地图（二卷14、17–18章）。
    - **大元拍卖场（大元体育场）**：14 区东侧，白天为人类正规拍卖场、每周日晚 11 点变为喰种大型拍卖会（与会者皆戴面具）；设正门 / 后门 / 紧急逃生通道三出口。伊鸟地图所标聚集地之一，苏晓调 CCG 近百人封锁三出口、切电屠场，于此肃清上百喰种（二卷18–23章）。

## 进击的巨人世界
- **三面「神之壁」**（由外而内）：**玛利亚之墙**（最外·845 年被超大型巨人踢破）、**罗塞之墙**（第二重·850 年被踢出三道破口、攻防战战场）、**希娜之墙**（最内·贵族区·范围小·容不下逃难平民）。墙内人类靠三墙抵御墙外游离巨人、安稳百年（三卷1–2章）。
- 罗塞之墙攻防战分 **一区（东）/ 二区（西）/ 三区（南）** 三个破口战区，分别由彼岸花、兄弟会、及苏晓所在的杂队防守（三卷2章）。
- **帕拉蒂岛** — 三面墙所在的岛屿，面积广阔、资源丰富（马莱觊觎之地）；岛上散布马莱流放来的无垢巨人。苏晓一行劫艾伦后须横穿此岛逃往墙外（三卷43章）。
- **玛利亚之墙（已失守区）** — 最外墙、845 年失守后大片沦为巨人横行的废墟，范围广阔、需长途跋涉穿越；内有：
    - **巨人之森** — 一大片巨树（树径数米）林，可在树顶躲避无垢巨人；但**原著中莱纳等人正因在此休整而被调查兵团追上、夺回艾伦**，故苏晓特意回避（三卷40章）。
    - **艾伦故乡城镇** — 5 年前超大型巨人踢破之处，已成荒废空城。
- **耶格尔家地下室**（玛利亚之墙·艾伦旧居）：艾伦之父 [格里沙·耶格尔](../../人物/格里沙·耶格尔.md) 留钥匙嘱其来此寻巨人真相。苏晓据原著线索（母亲被飞石压死的房屋）找到、**一脚踹开门**（钥匙「并没什么卵用」）；木质书房内藏可换**世界之源**的实验物品、暗格蓝色药剂、及揭示艾伦身世的三本日记（三卷43–45章）。

## 斩·赤红之瞳世界
- **极北 / 极北要塞** — 帝国北疆咽喉（黑石城墙·百年未修·零下数十度·两侧终年雪山）；异族军起义之地、艾斯德斯率 50 万镇压处。内有「咚咚塔要塞」（极北军据点·努马·塞卡被刺杀处）（四卷13–20章）。
- **帝都** — 帝国首都、大型阶梯城（越中心越高）；中心是**皇宫**（城中城·圆形尖顶·最高百米·小皇帝与大臣居此）。下层是**平民窟 / 花街**（致幻药控女人、人体实验的黑暗之地·地下有废弃地道与帝国秘境实验室——藏紫色「帝具改造蓝本」·设卡尼亚特毒气机关）（四卷22、31–32章）。

## 希亚大陆（原生世界 · 第十七卷）

（详见 [希亚大陆·灭法遗迹](../任务世界/希亚大陆-灭法遗迹.md)·十七卷4–7章）

- **圣城（王都）**：[圣索丁王国](../../势力/index.md) 的国都。王宫是带巨大庭院的**城堡**（国王与众王后居住 + 重臣办公一体·占地不大、不尚奢华——常年战争故王族不贪享乐）；庭院围一圈铁栏（饰「犬身狮头」兽）、上千金甲直属护卫把守；入门处摆郁金香、议事厅设陨石锻造的漆黑王座。[铁之手](../任务世界/希亚大陆-灭法遗迹.md)（苏晓）的官邸是带荒芜大庭院的庄园、距王宫不到半公里——另有一座横向逾一公里、摆满历代国王与殉国英雄雕塑的大庄园（拉拢人心）。
- **神之墙（卢利马斯雪原 等）**：圣索丁王国与死灵帝国交界处的巨墙——**并非全是冰墙**：大部分由岩石堆砌，地理特殊处则用坚冰 / 不可燃木材、甚至有一段是**动物骨骼**堆成。死灵只猛攻**人类运输物资困难的几段**（卢利马斯雪原〔荒焚处〕/ 珂珂巴亚戈壁等），物资丰富的**塞梅湿地段**几百年没被攻打（苏晓特意从这段翻墙去死灵领土）。卢利马斯雪原段为**百米高、十几米宽的冰墙**（外侧近乎垂直、平滑如镜·极难攀登）。人类靠冰墙上的上千架**重型机弩**（金属弩箭 / 高价「分裂箭」）死守、抵御 [死灵族](../生物万族/index.md) 的潮水猛攻——是人类抵挡死灵、不被赶尽杀绝的生命线。[荒焚](../../人物/荒焚.md) 巧合进入本世界后即在此战场以士兵身份刷战功。
- **灭法之影遗迹 / 灭法殿（诺托埃克 = 生灵止步之地 · 十七卷79–82章）**：暗月沼泽附近森林中的一座几十米高、半坍塌的古老殿堂——[苏晓](../../人物/苏晓.md) 此行的终极目标。原是巴特拉族为灭法者 [阿克斯](../../人物/阿克斯.md) 修的神殿，阿克斯改造后留人偶机关镇守。标志是冲天的蓝色光柱；外围几百米笼罩**雷属性青钢影能量**（电弧伤人、**死灵族千年来连靠近都做不到**，须以自身青钢影护体方能通过，进遗迹需「青钢影 + 灭/法/影三石之一」）。殿内空旷、立着先代灭法之影们的雕塑（[马文·华尔兹](../../人物/马文·华尔兹.md) / [格林·吉莉安](../../人物/格林·吉莉安.md) 等·多已崩碎）；深处石墙嵌**三道石门 = 灭之殿 / 法之殿 / 影之殿**（阿克斯加建·须以灭/法/影三石分别为钥），石门上方壁画记载着希亚世界真史（苏晓查阅得 5% 世界之源）。死灵称此地「诺托埃克」。
- **暗月沼泽（死灵帝国首都 · 十七卷72章）**：[死灵帝国](../../势力/index.md) 的国都——大沼泽上方盘结着大片粗壮的「**暗月藤**」（无月夜散绿荧光·故名），死灵把城建在藤上、面积比人类王都更大（藤层分解垃圾、产清新空气、无臭）。中心一棵参天「**庇护之树（巴巴卡纳之树）**」遮去三分之一城区，其主干树屋住着死灵族的精神图腾「[启迪](../../人物/启迪.md)」（连死灵王森迈尔都没资格住）。**灭法之影遗迹就在暗月沼泽附近**（死灵称「**诺托埃克 = 生灵止步之地**」·死灵进不去）。
- **王座之下 · 希亚圣殿（不死者陵墓）**（十七卷47–49章·**隐藏区域**）：王宫议事厅的漆黑王座是入口（暴力掀飞 / 或用机关均可）——下方是一道几十米深的金属竖井（实为年久失修的升降梯），底部金属门需**赫伯特王族血脉**开启，再往里就是「**希亚圣殿 / 不死者陵墓**」。这是本原生世界的**隐藏区域**（非特殊任务难寻）：暗红石廊、阴森潮湿；**宝箱 / 世界之源等奖励 +20%**，但进外围**每小时消耗 600 乐园币**（不足则被强制传送至入口）。深处有**陵墓守卫者 [里德](../../人物/陵墓守卫者·里德.md)**（霸主级中立锻造者）把守；苏晓此行为找灭法遗迹的钥匙「**灭之石**」。**圣殿守卫多是「为战场而生、却被囚于此」的死灵**（金属门全是单向、只能从外开、需王族血脉）——苏晓一路过 [长刀使·空](../../人物/长刀使·空.md)（首领级·守门）、**震天殿**的 [震天使·鸣](../../人物/震天使·鸣.md)（精英·战鼓增益 + 能量柱供能的无敌护盾）等。杀守卫掉 [远古死灵精粹](../../力量体系/消耗品材料.md)（圣殿硬通货）；摧毁区域能量中枢可转安全点、并唤醒 [死灵贤者](../../人物/死灵贤者.md) 的「圣殿兑换商店」。**越深入耗币越高**（震天殿已 3000 币 / 时）；**第三层充斥黑雾**（需「圣洁庇护」抵御）。一层最终镇守者是 [战甲之灵](../../人物/战甲之灵.md)（领主级大将军·灭之石持有者）。**圣殿共三层**：一层 = 希亚王朝将军 / 士兵；二层 = 王族直属精英部队；**三层 = 「生灵禁区」**——囚着 **希亚大帝·生灵屠夫**、**囚徒·刑** 等恐怖存在（连四阶契约者也未必杀得穿·与希亚大帝交手「连存活都难」）。火焰区还涌出过一团**诡异的黑雾人形**（疑即冰蓝战锤所镇压之物·超高速·苏晓自言无法对付）。

## 现实世界
- 苏晓所在的二线城市：医院停尸间（其「假死」尸体存放处）、东城分局等（第29章）。
- **DL 市**：三面环海的沿海城市，苏晓（化名苏笑）逃亡后的落脚地，于城市边缘老旧小区租房隐居（月租 800，同楼疑住契约者）；另有一条蹊跷的「十万元/月、天海路 37 号」租房信息，实为契约者陷阱（第30–31章）。
- **金巴布韦（非洲）**（六卷82章）— 苏晓现实复仇线的出国目的地。**货币机制半崩溃**（面值动辄上亿·40 万万亿金巴布韦元仅折合约 1 万 RMB·当地极度贫困、连「吃土饼」都真实存在）；外币在此流通不畅。复仇目标就在其边境附近。
- **洗钻河（金巴布韦边境）**（六卷82–85章）— 一片无名的产钻区（70% 钻石被国人收购·故 RMB + 美元广泛流通、汉化严重），实为**「无法之地」**：军阀横行、几股武装割据、贩卖 AK / 重机枪 / 炸弹如卖菜。大军阀 [撒姆尔](../../人物/index.md) 的城堡即在此。**苏晓在现实世界·衍生世界装备 / 加成全被封印**、只能凭裸装六维 + 被动技能作战（详见 [轮回乐园机制](../../总览/轮回乐园机制.md)）。
- **「小城」（三国边境无人区）**（十二卷95–98章·[苏晓](../../人物/苏晓.md) 的「勇闯魔窟」任务地）— 三国交界缝隙处的**无法之地**，地图与新闻都不存在、有「保质期」（大人物需要时它才存在）。满地黄沙、唯一的「资源」是野生动物（街边烧烤野味、铁笼关动物），偷猎组织 / 毒枭 / 劫匪盘踞。入城前要过劫匪堵车的「**破财桥**」（只勒索财物、给受害者留下够回文明区的食物 / 水 / 油）和四国混编士兵的关卡（交过路费 + 带枪费）。城现由毒枭 [巴谷](../../人物/巴谷.md) 统治、其**废品厂**是大规模制毒窝点（守卫注射药物、体力超 5 点）。苏晓为「**突破身体极限**」（五属性压制到 5 点·见 [属性](../../力量体系/属性.md)）须深入废品厂地下室停留 30 分钟——这是他当杀手时脖颈中过一枪、险些丧命的旧地。
