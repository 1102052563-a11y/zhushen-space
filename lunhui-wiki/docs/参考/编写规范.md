# 条目编写规范

本页定义所有条目的统一格式，确保逐章扩充时风格一致、可交叉引用、不重复、不剧透。

## 总原则
1. **忠于原著**：只写正文明确写到的内容；推断须标注「（推测）」。
2. **渐进不剧透**：只记录已读章节的信息；未读到的不提前写。
3. **增量更新**：同一对象再次出场，**在其已有页面追加**，不新建重复页。
4. **必标出处**：关键信息注明相关章节，如 `（第123章）`。
5. **命名**：以原著正式名作页面名；别名写进 front-matter `别名`。

## front-matter 模板

### 人物
```yaml
title: <姓名>
分类: 人物
身份: 契约者 / NPC / 反派
阶位:
天赋:
所属:
别名: []
首次登场: 第X章
状态: 存活 / 阵亡 / 未知
```
正文建议分节：**简介 / 外貌 / 性格 / 能力（天赋·技能·装备）/ 经历（按章节）/ 关系 / 出处**。

### 任务世界
```yaml
title: <世界名>
分类: 任务世界
难度:
人数: 单人 / 组队
```
正文：**背景 / 任务目标（主线·支线·隐藏）/ 关键NPC与势力 / 奖励与惩罚 / 出场契约者 / 出处**。

### 势力
```yaml
title: <势力名>
分类: 势力
类型: 冒险团 / 组织 / 国家 / 宗门
所属世界: 轮回乐园 / <任务世界>
```

### 技能·功法 / 装备道具 / 物品
```yaml
title: <名称>
分类: 技能 / 装备 / 消耗品 / 材料
品级:
来源: 乐园商店 / 任务掉落 / 自创
```

### 体系页（阶位 / 天赋等框架）
用表格列层级，细节随章节补；断言框架须加 `::: tip 来源` 说明。

## 交叉引用
用相对链接互链，例如 `[苏晓](/人物/苏晓)`、`[阶位](/力量体系/阶位)`。
