# 章节梗概

按章记录剧情梗概，作为其它条目的引用锚点（仅覆盖已读章节）。

## 第一卷 海贼，猎杀者的试炼

- **第1章 复仇者**：苏晓在现实中蹲守刺杀仇人，割喉成功，却被对方持枪保安击中（小腿、胸口），临死前抛刀反杀保安；濒死之际听到「猎杀者，‘轮回乐园’为你开启」，在黑暗中醒来，面前出现倒计时与乐园的契约邀请。
- **第2章 猎杀者**：乐园说明「猎杀者」是天赋特殊、需额外执行猎杀任务的契约者。苏晓以血签订羊皮纸契约，伤势被修复、强制觉醒天赋、开启半数据化，随后被传送进 **海贼王** 世界，获得烙印与主线任务「刺杀哥亚王国国王」。
- **第3章 独行者**：苏晓查看个人信息（六维、等级、噬灵者天赋）；从破屋窥见废品终点站两伙流民（托比、罗吉尔）火拼；制服误入避难的少女米娅。
- **第4章 武器与货币**：火拼后苏晓拾得「破旧的燧发枪（白）」，乐园讲解装备品级与乐园币机制；他以压缩饼干利诱米娅为其带路。
- **第5章 枪火**：米娅领苏晓走出废品终点站到哥亚王国侧门。苏晓尾随垃圾马车，发现车夫实为黑市商人并将其枪杀（**首杀**：噬灵者 +5 法力、开启储蓄空间、获宝箱·白），换上其衣物、混入哥亚王国首都城。
- **第6章 这是，真实的世界**：苏晓驾垃圾车蒙混过城门，混入哥亚王都；从侍卫闲谈确认时间点（路飞出海前），并捕捉到「冒充黑市商人侄子入侍卫队」的机会，感慨任务世界亦是有血有肉的真实世界。
- **第7章 引荐信**：苏晓变卖马匹得贝利，潜入民宅清洗、化名「白夜」，开启首个宝箱（50 万贝利 / 城门守卫队长罗文的引荐信 / 100 乐园币）；持引荐信到侍卫队总部，结识年长侍卫汉克。
- **第8章 斩龙之刃**：苏晓向侍卫首领布兰登·欧卡付 40 万贝利入队；在仓库选武器时夺得评分 10 的稀有白装「斩龙闪」（斩龙剑豪龙马旧物），并得知后天侍卫队将焚烧废品终点站。
- **第9章 刺穿他们**：苏晓灌酒套出汉克的情报，夜入欢乐街，潜伏至欧卡幽会的楼阁，一刀同时贯穿欧卡与同房女子。
- **第10章 渴望强大的心**：苏晓补刀斩杀欧卡（首次见到血条 / 假死，世界之源升至 5.7%、法力值 76）；开其宝箱得「亡妻的项坠」（力量 +1，终于找到变强之法），并花 100 乐园币开启装备隐藏功能，力量由 6 升至 7。
- **第11章 总有一天**：欧卡之死惊动左大臣、全城搜捕；汉克带「白夜」面见左大臣——原来苏晓所杀黑市商人卡洛斯是左大臣的供货人。左大臣以侍卫首领之位为饵，发支线任务「杀科尔波山的主人、取其牙齿」（LV.2，10 小时）。
- **第12章 布置与估测**：苏晓在「嘟噜杂货铺」购得猫薄荷类干果木天芷，又取侍卫队仓库炸药，潜入科尔波山选址布设炸药陷阱；并系统总结了半数据化的规则（以现实为准、非游戏化）。
- **第13章 猫科类动物的弱点**：木天芷诱使巨虎入陷阱、满地打滚，苏晓引爆地下炸药与铁钉重创之，但巨虎未死。
- **第14章 战斗之美**：苏晓持斩龙闪与巨虎缠斗，斩爪、断后腿筋、不断放血；一名草帽小孩在灌木丛目睹这场恶战。
- **第15章 精英生物**：经反复缠斗，苏晓刺穿巨虎脑髓、燧发枪爆其独眼，终杀「精英生物」科尔波山的主人（世界之源 8.9%、法力 91），右臂被咬重伤；巨虎掉落苏晓首见的**绿色宝箱**，他取下虎齿，随后察觉灌木中疑似有人。
- **第16章 脱笼之刃**：灌木中的小孩竟是幼年路飞——轮回乐园血红警告「猎杀者接触**位面之子**」，并检测到超强生物卡普（LV.79）靠近、触发任务「传奇的扼杀」，苏晓果断放弃离开。河边洗血、烤鱼，发觉经巨虎一战刀术大进，悟得现实是「牢笼」、决意在乐园不断变强。
- **第17章 初步成功**：苏晓回王都交虎齿；左大臣意图赖账，旋被乐园「强制纠正」操控兑现，运作苏晓上任侍卫首领（左大臣事后形同废人）。苏晓定下刺杀国王之策：借焚烧废品终点站引强敌冲击王宫、调开国王守卫，趁机动手。
- **第18章 海贼**：苏晓获得在三伙海贼（布尔萨姆 / 克利克 / 骷骨）中择一执行焚烧的决策权，选中克利克海贼团，命汉克联系。
- **第19章 前夜**：苏晓于旧仓库密会克利克，以世界政府盖章的贵族封爵凭证为饵谈妥、要其自备炸药；夜里开启绿色宝箱（1000 乐园币、灵魂结晶（小）、虎爪）。
- **第20章 欺骗与买命钱**：克利克海贼团焚烧废品终点站（数千流民遇难）。苏晓按计划不开城门、将海贼困于火海（伪装随行的汉克被克利克烧死），再以「买命钱」收买死刑犯投炸药、诱克利克炸开城门攻入王都、杀向王宫；死刑犯事后被克利克枪杀，苏晓守约将钱交其妻女。苏晓随即奔赴王宫，准备最后一击。

## 第一卷 收官 · 回归乐园

- **第21章 人形坦克**：克利克海贼团攻破王宫区、与侍卫激战。惊慌的国王派出贴身护卫——戴奴隶项圈的哑巴「人形坦克」豪森（力量>20）迎敌，正中苏晓下怀（调开守卫）；苏晓攀上王宫高层搜寻仓皇逃窜的国王。
- **第22章 回归与奖励**：苏晓破窗而入，一刀腰斩、再补一刀斩首国王菲利普·赫伯特，完成主线、灭口王妃。开国王宝箱得「世界树指环」。30 秒后脱离世界——途中遭「虚空剑豪」打断，被乐园修复续传，回到猎杀者专属房间。结算：世界之源 8.9%、完成 2 任务、综合评价 **A-**，因噩梦难度奖励翻倍得 4 属性点 + 10000 乐园币、升 **LV.2**。
- **第23章 不作死与咸鱼有什么区别**：乐园索要世界树指环（含「世界之力」），苏晓拒售、与乐园僵持。乐园透露与契约者是**共生**关系、需世界之力进化，并印证猎杀者「处理违规却未犯条例之契约者」的本质。苏晓赌赢——乐园发「抉择」任务，他交出指环、换得「与天赋极默合的传承」。
- **第24章 意料之外的人**：传承来自「诞生之圣所」六尊石雕（灭法之影，施法者天敌）。苏晓承接后力 7→12、敏 7→10、智 6→12，获**唯一职业「灭法之影」**与技能树（青钢影 / 刀术专精 / 绝魔体质）。随后推门，初见轮回乐园全貌（建筑、街道、奇装异服的人群、摆摊交易）。
- **第25章 专治各种炮台**：职工者小萝莉希珞珞收 50 乐园币，在冷饮店「猫与狗」为苏晓讲解轮回乐园——职工者、契约者「乐园 3 天 / 现实 1 周」周期、除竞技场外不可互攻的安全规则、隐藏相貌、设施与惊人的死亡率，并提醒勿招惹强如炮台的「法爷」；而苏晓正是法师天敌。
- **第26章 成长性**：苏晓逛乐园中心区（可见属性强化大厅 / 装备强化大厅 / 试炼场 / 竞技场，余被迷雾），进属性强化大厅用 4 属性点（1 力 + 3 敏，裸装 力12/敏13/智12），拒用麻醉。其亲和属性因传承固化为力 / 敏 / 智，且三者差距不得超 3 点。
- **第27章 不传之秘**：苏晓在试炼场召唤海贼世界剑术宗师「耕四郎·镜像」（每小时 1000 乐园币）。耕四郎不教招式，只点拨他「感知兵刃、使之成为肢体延伸」，令其三年苦练的刀术开始升华。
- **第28章 黄沙与死亡？**：以猎杀者权限再召耕四郎切磋，又开实战模式（机械人偶围攻、战死不真死）适应暴涨的身体；刀术专精由 LV.1 升至 LV.3（实战 + 消耗灵魂结晶）。其间确认灵魂结晶用于升级 / 解锁技能、极稀有。
- **第29章 想想你死后～**：乐园停留 3 天期满，苏晓被传送回现实世界（属性 / 被动技能有效、主动技能与装备锁定、严禁泄露乐园）。醒于医院停尸间（假死的「尸体」），为夺回父母所留家传佩刀，绑架一名年轻警察逼问下落。
- **第30章 无法无天**：苏晓挟警察潜入其局长家中、取回佩刀；约 6 小时后通缉令发出，他化妆成学生、辗转四天逃至三面环海的沿海城市 DL 市，并留意到一条蹊跷的「十万元/月」租房信息。
- **第31章 恐怖的精神系**：苏晓化名「苏笑」在 DL 市边缘小区租房安顿；识破那条「十万/月」租房广告——竟是某契约者花 10000 乐园币、借「心理系」法师手段设的诱骗陷阱（仅契约者可见、智力 >20 可免疫），遂不予理会。点明现实中确有契约者违规牟利，而猎杀者正为此而设。
- **第32章 新的衍生世界**：苏晓被乐园召回，在交易广场向职工者「药罐」购入恢复品（盐饭团 / 黄金炒饭 / XT-12 药剂）；乐园提示其将进入第二个衍生世界——东京喰种。

## 第二卷 东京喰种 · Rc 细胞的咆哮

- **二卷·第1章 我有个大胆的想法**：大批契约者同入东京喰种（时间线重置、耗时空之力、评分门槛升 C 级）。苏晓了解「冒险团」（临时 / 永久）。开局被传送进神代利世吃人的大楼，与 7 人临时冒险团首领寒鱼等十余契约者一同被光膜限制；目睹 S~SS 级喰种神代利世吃人，却直觉自己能吊打她。
- **二卷·第2章 獠牙初现**：世界信息揭晓——难度 LV.3、主线为连锁任务（择 CCG 或喰种）、另有清理 12470 号契约者的猎杀任务。喰种壁虎来抓利世；光膜一消，苏晓一脚踢飞利世、亮出斩龙闪，扬言杀两喰种以加入 CCG。寒鱼悟出：开局两 S 级喰种正因契约者中有苏晓这等怪物、乐园为平衡剧情所致。
- **二卷·第3章 青钢影**：苏晓首开青钢影（燃敌能量 + 真实伤害；对喰种则噬灭 Rc 细胞），一刀斩断壁虎鳞赫令其惨叫，两刀将其斩至半死，随即冷冷锁定欲逃的神代利世。
- **二卷·第4章 绝望的利世**：苏晓连斩四刀斩首壁虎（世界之源 5.2%、法力 163）；转向利世，利世以媚惑偷袭反被一刀贯穿心脏。乐园提示「杀重要剧情人物前需取出赫包以延续剧情」，苏晓正欲取赫包，青铜树四人赶到救人。
- **二卷·第5章 叛逆少年**：青铜树首领独眼之王·枭（化名高槻泉、身缠绷带）携多多良、野吕、叛逆少年雾岛绚都现身。多多良命雾岛绚都杀苏晓，苏晓一脚踢断其迎面骨、险些斩喉，被野吕的赫子救下；旁观契约者纷纷逃散。
- **二卷·第6章 敌手**：苏晓直呼「芳村艾特」点破枭的真身，枭遣多多良等带雾岛绚都先撤、野吕追逃跑（跳楼）的利世，自己化**赫者**形态留下死战。
- **二卷·第7章 美如花，狠如狼**：苏晓激战赫者枭，斩断其一只手爪、斩碎背部赫子（青钢影燃 Rc 致痛使其落入下风），自身小腹被抓伤；枭终跳楼遁走。CCG 一等搜查官亚门钢太朗到场，苏晓以壁虎尸体为「见面礼」求入 CCG；亚门上司、痛恨喰种的真户吴绪赶到，验明苏晓为人类后愿做其担保人；医疗车小护士为苏晓缝合伤口。
- **二卷·第8章 未知的计划**：苏晓随亚门入 20 区 CCG 分部（过 Rc 细胞检测门），视频见特等搜查官丸手斋——对方愿在核实战力后破例接纳。苏晓借乐园**伪造身份**「喰种搜查官（已战死）家属」取得合法身份。另一边，遁走的枭忧心这突现的「怪物」会打乱其筹备 13 年的计划。
- **二卷·第9章 平衡机制**：苏晓开壁虎绿宝箱（900 乐园币、赫包 S、绿色戒指**恶灵咆哮**——20 秒无视痛觉的保命底牌）。正式加入 CCG（三等搜查官，主线①达成、②为「升上等搜查官」）。**平衡机制**开启：因苏晓入 CCG，其余契约者被强制划入喰种阵营，CCG 仅他一人。苏晓被分派紧挨 20 区的 14 区。
- **二卷·第10章 小受受**：苏晓到 14 区分部见负责人特等搜查官篠原幸纪，切磋考核中被评 SSS+ 战力；并结识篠原性格怪异的下属、白发少年铃屋什造（开口便问「我可以砍你吗」）。
- **二卷·第11章 开放的女高中生**：篠原给出公园连续失踪案线索，破例准苏晓单独行动；苏晓在公园蹲守，期间遇一名搭讪的开放女高中生。
- **二卷·第12章 喰种×人类**：那名女高中生竟是喰种「千夏」（援交诱杀好色男性、嫉妒并吞食女学生）。苏晓现身、几刀将其斩杀（S- 喰种、世界之源 5.7%、法力 170、得 CCG 贡献值）。
- **二卷·第13章 吉祥物**：苏晓破格升二等搜查官；篠原把美女二等搜查官真户晓拨作其部下（被苏晓称「吉祥物」、用作诱饵），二人前往共喰区**西街区**。另一边，11 区喰种阵营的契约者寒鱼与火系法师火爆结盟，欲合力对抗「怪物」苏晓，并得知主线第三环是阵营对抗。
- **二卷·第14章 恐怖的直觉**：抵西街区（14 区中野区·共喰盛行），苏晓忽凭「**直觉**」一眼认出路上喰种、将其扼杀——他推测这是属性提升的副产物（连带右腕旧伤渐愈）。随后造访 SS 级喰种伊鸟开设的 **Helter Skelter 酒吧**（小型情报中心），不杀其人、只问情报，套出西街区共喰区即在此处、每晚 9 点后于中心地带最盛，疑由某势力组织作「兵源选拔」。
- **二卷·第15章 死神**：入夜，苏晓把真户晓单独留作**诱饵**引出十余喰种，自己自楼顶杀入；开**青钢影**（每分钟耗 2 法力）斩断成片赫子、如「死神」般几个照面连斩诸喰种，真户晓苦缠一名 A 级喰种时，他已清掉其余、坐于尸堆叼烟。
