## 细纲（第 1 章）

### 第 1 章：初来
- 核心事件：前世"伤仲永"神童林源猝死魂穿青松武馆少馆主，睁眼即在父母灵堂；接收记忆（父母被周家所杀、这具身体天赋平平）、绑定宗门修仙系统；以前世神童留存的百科博识 + 成年谋略算计强压悲痛、当众挡回周家吊唁施压、由护法林忠扶保接掌武馆
- 字数目标：4000 字（范围 4000–4400）
- 目标情绪：悲怆压抑 → 冷静识局 → 隐忍立威（血债 + 掌权）
- 章节定位：高压（穿越 + 丧亲 + 仇家逼门 + 接掌掌权）
- 章首钩子：代入钩 + 悬念钩 —— 神童猝死，睁眼却在一片缟素灵堂、面前两口棺材
- 爽点：无传统爽点（开局是丧），靠 强代入（临危受命）+ "少馆主忽然沉稳、滴水不漏挡回仇家"的扮猪吃虎预埋 撑追读

#### 内容概括（五段式）
- 起因：林源猝死→魂穿→睁眼灵堂，两口棺材、白幡、哭声；接记忆知父母被周家所杀、身体天赋平平
- 发展：护法林忠扶住少馆主；宗门修仙系统觉醒；长老林广阴阳试探（欺少馆主年幼）
- 转折：周家以吊唁为名上门施压，图谋趁丧吞并青松武馆
- 高潮：林源反常平静，以谋略 + 口才 + 信息差滴水不漏挡回周家挑衅、话里留钩，逼退来人；林忠当众扶其接掌，门人归心、长老被压
- 结尾：系统气运涨；林源在父母棺前冷立誓——这个武馆我接了，复仇既除患、也是立威掌权第一步棋；望镇岳山（崛起方向）

#### 情节安排（多线）
- 主线（掌权）：年少接掌、长老不服 → 林源首度立威、坐上馆主位
- 主线（复仇/立威）：周家逼门 → 林源挡回，结仇、立"复仇即立威"之志
- 主线（崛起）：系统觉醒（1 级），指向"带武馆招贤纳士、踏入修仙"的方向
- 感情线/关系线：对"父母"无真情（冷）；对林忠的忠诚起用（用而信之）
- 逻辑线：魂穿→识局（武馆没了顶梁、长老不服、仇家逼门、身体天赋平平）→稳场挡敌→接掌立威→立志

#### 人物关系和出场顺序
- 出场顺序：林源（灵堂醒来）→ 林忠（扶少馆主）→ 长老林广（阴阳试探）→ 周家来人（吊唁施压）→ 满堂门人杂役（观望）
- 人物关系变化：门人/长老对林源 由轻视观望 → 忌惮归附；林源 ↔ 周家 结仇明面化
- 视角/信息差：读者知林源是穿越者 + 有系统 + 一脑子百科博识 + 对父母无情、复仇是工具；门人/周家/长老只当他是天赋平平的少馆主——扮猪吃虎与掌权布局的底座

#### 情节细化
- 情节点序列（密/疏 + 预算）：
  - P1 前世"伤仲永"神童猝死回闪：幼年过目不忘、通读百科名动一时，越长大天赋越流失、泯然众人，天赋像被无形之物一点点锁住——意难平中猝死【铺垫·密200】
  - P2 睁眼灵堂：缟素、两口薄棺、白幡拂面、四面哭声，认知错位【氛围/代入·慢镜360】
  - P3 记忆涌入：青松武馆——林氏先祖开创的凡俗武学小馆，父母老馆主夫妇被邻镇周家设计害死，原身跪灵三日哭晕；这具身体天赋平平【信息/背景·密320】
  - P4 老护法林忠扶住摇晃的少馆主，红着眼唤"少主"——忠心先天高手登场【人物·密240】
  - P5 宗门修仙系统觉醒：面板、系统 1 级、气运值低、双层机制（发展度解锁等级 + 气运兑换）核心规则、鉴人之能【设定·密280】
  - P6 林源冷静识局（神童博识 + 成年谋略）：这爹娘于他是陌生人、悲不属于他；但武馆是他的根基与棋局。眼下年少接掌、长老未必服、仇家环伺、身体天赋平平——先稳局、坐实位子（顺带一丝旧日灵光乍现又被"锁"回的端倪）【人物/大局观·密300】
  - P7 长老林广皮里阳秋，话里试探暗贬"少馆主年幼、恐难当大任"【内部冲突·密260】
  - P8 周家来人以吊唁为名上门，阴阳怪气、图穷——趁丧逼交产业/逼少馆主让步【外部冲突·慢镜380】
  - P9 满堂悲愤无力：林忠攥拳、门人敢怒不敢言、长老观望【蓄势/反应·密220】
  - P10 林源站出来，不哭不闹、反常平静：先谢周家"惦记"，再以谋略口才把每句挤兑滴水不漏挡回、话里留钩，让周家拿不准青松武馆底牌【爽点/扮猪吃虎·慢镜460】
  - P11 周家来人诧异拿不准、悻悻撂话而去；长老林广也怔住【反应层·密240】
  - P12 林忠当众扶林源接掌馆主之位；门人归心，长老林广被压得只能顺势附和【接掌/立威·慢镜360】
  - P13 系统提示（接掌聚人心→气运涨）；林源在父母棺前冷立誓：这武馆我接了，周家、焚天宗欠的早晚连本带利讨回——复仇既除患、也是立威掌权的第一步棋【点题/内心·慢镜340】
  - P14 望窗外镇岳山（山那头是修仙世界，是武馆挣活路的唯一方向）【章尾钩·密160】
- 预算合计：约 4120 字（目标 4000，范围 4000–4400）
- 代价兑现/收益兑现：付出——背上血债、扛起一个烂摊子、与周家/长老的暗斗开局；收益——坐上馆主位、初步立威、系统正循环启动、崛起方向确立

#### 结尾设定和钩子
- 结尾设定：临危接掌、初步稳场，但周家不会善罢、长老未真服、自己修为为零且天赋平平——三重压力压顶
- 章尾钩子：血债钩 + 愿景钩 —— 冷立誓复仇立威 + 望镇岳山定崛起之路（期待度：强）；承接第 2 章 林忠辅佐摸家底、系统起步、长老与周家再施压
