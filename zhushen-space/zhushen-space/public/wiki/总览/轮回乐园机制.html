    轮回乐园机制¶

    随阅读更新；以原文为准。乐园作为场所的样貌见 乐园。

    「轮回乐园」是一处神秘存在，与人类签订契约后，派其穿梭于各个「衍生位面 / 衍生世界」执行任务。乐园自称「无所不能」，但对猎杀者不能复生已故之人（第1、2章）。第17章苏晓首次听到乐园之声——机械音、毫无感情、冰冷残酷。

    职工者：首个衍生世界主线任务失败、却侥幸存活的契约者——乐园延长其进衍生世界的周期（每两个月才进一次），多从事制造（恢复品 / 食物 / 装备）、商贸（如黑帆商会黑商）、情报（如竞技公会）、后勤等，与契约者共构乐园生态。翻身机会：若在那两月一次的衍生世界里表现优异、拿到 B 级以上评分，可恢复契约者身份。职工者可花币买冒险团随队保护进衍生世界、享「任务失败豁免卷」（活到最后即通关）。希珞珞、夏、竞技公会 魅姬 等皆职工者（三卷34、四卷69、二卷91章）。

    副职业：与战斗主职业（如灭法之影）并行、单独计算、互不影响的一类能力，从「知识类技能卷轴」习得（分一~九阶）。苏晓的第一个副职业是 炼金学（五阶·从夏洛克遗物得·能造炼金炸弹 / 药剂 / 生物·有被科研阵营招揽 / 绑架之险）（四卷69–70章）。

    契约与条例¶

    • 以一张羊皮纸签约，按下带血的指印即成立；签约后乐园会修复伤势、强制觉醒天赋（第2章）。
    • 乐园条例：「一切都将等价交换」（第2章）。
    • 印象：冰冷残酷但公正，从未言而无信，且遵循某种规则行事（第17、23章）。

    共生与世界之力¶

    • 轮回乐园与契约者是「共生」关系：契约者进入衍生位面探索、获取世界之源；乐园借此提取「世界之力」以进化并维持自身运作——故按世界之源多少给予奖励（第23章）。
    • 含大量世界之力之物（如 世界树指环），乐园会主动开价收购而非强取，正体现其受规则约束（第22–23章）。

    契约者 与 猎杀者¶

    • 契约者：普通人签约后成为契约者，为乐园执行任务、领取奖励（第2章）。
    • 轮回印记（八卷57章·相当于契约者的「身份证」）：每名契约者必有的印记——普通契约者是「一圈圈圆环、中心一道五芒星」，苏晓 因是猎杀者而不同，是一把「黑色利剑」图案。
      • 可剥离 / 可变现：通常不可剥离，但在生存试炼这类淘汰场中，死者身上的轮回印记会进入「可剥离状态」（剥离成功率约 50%），剥下后化成一张卡片——一枚 = 1% 世界之源 或 1000 乐园币（二者择一·结束前出售即可）。这正是 二阶生存试炼 中契约者「堵桥」「抢空投」乃至自相残杀的根本动机（乐园借此淘汰弱者、却也使试炼随时可能演变成「大逃杀」）。苏晓在石桥反堵桥后从十余具尸体上剥得 7 枚、赶路两天后增至 10 枚。
    • 猎杀者：天赋特殊、更适合战斗的契约者，须有「杀人的胆量」（第2、9章）。
      • 本质：乐园正常情况下不能直接处决 / 处罚未触犯条例的契约者——故设猎杀者，由其在衍生世界处理那些「违规但未触犯乐园条例」的契约者（如钻规则漏洞牟利者，通常强而狡猾）（第23章）。
      • 无法完成专属试炼 → 被剥离身份、降为普通契约者；任务中死亡 → 彻底死亡（第2章）。主线试炼成功后，猎杀者身份才正式解锁、开始发布猎杀任务（第22章）。
      • ⚠️ 阶位考核（晋升任务）（七卷1章）：猎杀者每个「阶位」有等级上限（如一阶上限 = Lv.10）；到顶后须通过「一阶考核」才能晋升。考核世界无主线 / 无猎杀任务、只有一个「晋升任务」（难度远高于普通主线·如「窃取神之力（伪）」Lv.14·失败罚全属性 −10），完成即得晋升资格回归。晋升任务有 3 次机会——三次全部失败 → 猎杀者降级为「职工者」（重罚）。苏晓的一阶考核世界 = 第七个衍生世界 钢之炼金术师。
        • 晋升阶位 ≠「升一层」（七卷19章·苏晓推断）：晋升二阶并不是楼层上移；乐园的交易市场等是公共空间（各阶契约者共用），晋升后更可能是解锁一些新设施、且二阶起所经历的衍生世界与一阶不同（难度 / 类型升级）。阶位差是碾压——二阶对一阶近乎全面压制（如二阶感知系契约者「桃子」装备加成智 35+·本以为一阶绝不可能反察觉她，却被苏晓的强感知锁定，可见苏晓远超一阶常理）。
        • ✅ 苏晓正式晋升二阶（八卷70–72章·通关二阶生存试炼后）：返园结算时乐园判定「已完成二阶试炼·晋升二阶」，验证了上述推断——仍与一阶契约者共用同一属性强化大厅 / 交易广场（不是升一层）。晋升收益：① 属性枷锁解除·可突破 50 点（突破身体极限·从此真正「一飞冲天」）；② 今后衍生世界最低难度 = Lv.10（能进全职猎人、圣杯战争等高难世界·风险更高但成长更快）；③ 竞技场提升——改与二阶契约者竞技、收回原「竞技天王印记」，因曾持该印记，二阶竞技场排名定在5 万名；④ 解锁「衍生世界联络平台」（进入新衍生世界后生效）。
        • 二阶试炼 = 「生存试炼」（八卷·吞噬岛）：与一阶考核（单个「晋升任务」）不同，二阶试炼是纯生存模式——同批 501 名契约者空降一座难度 Lv.14~20 的孤岛、生存十天，成功（活到最后）即晋升。生存模式无综合评价；猎杀者另有隐藏任务可叠加奖励。淘汰极残酷——501 人登岛、仅 36 人存活（存活率不足十分之一·真实危险远超标称难度·靠裁判 鼠大人 镇压才没全灭）。
        • 属性「破 50」的特殊奖励（八卷72章）：单项属性达 50 点（须先晋升二阶解锁、且不能同时把多项冲到 50·身体无法承载、须分次强化·强力量属性时痛得险死）会触发一次专属奖励四选一。若该属性是职业亲和属性（如灭法之影的力 / 敏 / 智），奖励尤其丰厚——苏晓发现窍门：带「影」字的选项一般都不弱、且很少出现（灭法之影专属）。苏晓三主属性破 50 的选择：力「傲君」（豁免一定阈值以下的力量系控制技能）、敏「超速反射」（+7% 神经反射）、智「血歌影」（意志力 +12）。
        • 意志力（八卷73章·不可见属性）：六维之外的隐藏属性——对疼痛的忍耐力、对魅惑类技能的抗性等（乐园未给全资料）。可经属性破 50 奖励等途径提升。
    • 恶魔体质 / 第 IV 恶魔改造所（八卷73–74章）：契约者可凭「燃烧的恶魔之心」（门票）进入「第 IV 恶魔改造所」、把人类改造为恶魔体质（一种很完善的力量体系·恶魔血统很强）。门槛极高：改造生存率仅 60%、须懂恶魔语与改造所的恶魔交易、付高昂改造费（约 60 万币·且只收 灵魂结晶（中）），把九成以上契约者挡在门外。 - 一阶（晋升二阶前）属性上限 = 49 点（七卷25章）：未晋升二阶时，单项属性最高只能堆到 49；而属性「破 50」会发生质变——「50 力与 49 力是两个次元」（如钢炼黑猩猩力 50 即本世界极值、近乎无解碾压）。苏晓走四属性（力/敏/体/智）均衡发展的罕见路线（现最高才 38），前期不如单属性流爆发，但配合刀术大师 Lv.18 等高阶技能、晋升后有望「一飞冲天」（疯奶 = 力/体双主属、左轮男 = 力/敏双主属·均非纯单属性）。
      • 猎杀任务的完成≠硬拼：只要给乐园一个正当处决违规者的理由（如让其主线失败），乐园便会亲自强行处决——苏晓未与 12470 号正面交手，仅因其主线失败即完成猎杀（二卷90章）。
      • ⚠️ 猎杀者身份是「禁忌」（八卷22章）：猎杀者的真实身份受乐园禁忌保护——他人即便推理 / 说出这层身份，话也会被乐园当场屏蔽、并在其手背浮现蓝色惩罚印记（腹黑女 凭智商 156 推断出苏晓「每次进衍生世界有特殊任务 / 实力增长异常 / 专克契约者」，话没说完即触禁忌、吓得急忙噤声收手）。这既保护了猎杀者（一旦身份曝光会被众契约者围杀），也威慑了知情者（泄露者自身受罚）。
      • 身份被「独立推断」出 → 限时清理知情者（八卷26章）：即便不是猎杀者自己泄露、而是被他人凭推理猜中身份，乐园也会判定身份暴露——在那名知情契约者身上打「追猎（蓝）」印记（猎杀者可感应其位置），并勒令猎杀者12 小时内清理掉知情者，否则剥离猎杀者身份。苏晓全程没与任何契约者交流，却因 腹黑女 的推断「躺枪」、被迫连夜去清理她。暴露的真正代价不在被围攻（苏晓已成长、来犯多是违规者或想取而代之者），而在被所有契约者排挤（没人愿与猎杀者组队 / 合作）、严重拖慢变强速度。
        • 追猎印记的破绽（八卷27章）：印记长在四肢则可强行剥离——腹黑女 竟自断带印记的手甩掉追踪、孤身遁入红土区中心；苏晓只得支付 5% 世界之源催生第二枚印记（赌它这次长在头 / 躯干让她无法再断）。可见这套机制并非绝对，知情者够狠仍有一线生机。
      • 青铜荣誉勋章：初次完成猎杀任务的全额奖励之一；持之可在「荣誉商店」（回乐园后开于猎杀者专属房间）兑换物品（二卷91章）。
      • 白银荣誉勋章（稀有品·高于青铜）：成功猎杀违规者本体且「表现优异」的全额奖励——苏晓六卷58章在司法岛亲手处决外来违规者 哈巴克（9740 号） 即得此勋章。同样在荣誉商店兑换、锁定不可交易、禁外传。一枚白银勋章可四选一（六卷76章）：① 灵魂结晶（中）×1；② 荣誉复苏药剂Ⅱ型（2 秒内回满 HP + MP·「原地复活」）；③ 主线任务失败豁免权（免一次扣属性 / 扣币惩罚·不可免强行处决）；④ 生命体征检测与修复（生命之泉修复历战暗伤隐疾 / 校正错位骨骼 / 身体调到最佳·但力 / 敏 / 体任一超 50 点后即无法兑换）。苏晓选了④。

    烙印¶

    契约者手臂上的利剑状纹身，以意识激活，是与乐园交互的核心界面（第2章）。功能：个人信息 / 任务 / 储蓄空间 / 技能 / 装备（阶位不足时部分未解锁）（第2章）。

    • 双重轮回烙印（异数）：极罕见地一人身负两枚编号相连的烙印——违规者 12470 号（与孪生哥哥被「炼成」、独活而承哥哥之印）即如此。可同接两份相同任务、掉宝箱与开箱奖励皆翻倍（视作两人）；但相应地，主线失败被处决时处决速度也翻倍（二卷90章）。

    任务、世界之源与结算¶

    • 任务分主线 / 支线（支线可在世界途中触发），标注难度（LV.）与期限（第2、11章）。任务奖励由乐园保障：发布方违约会被乐园强制纠正、操控其兑现（左大臣事后成废人）（第17章）。
    • 「世界之源」可累计：探索、交流、改变剧情、击杀世界内人物（越强所得越多；任务目标本身不计）皆可（第2、8、22章）。
      • 出售实验物品换世界之源（三卷44章）：某些剧情研究产物（如「进击巨人之骨髓」「始祖巨人之骨髓」「巨人生命图纸」等）不作普通物品卖、而直接折算成世界之源 %——苏晓在艾伦家地下室一次就售得 14% 世界之源（接近回归任务所需 15%）。这类换源物品在已读章节首见于进击的巨人世界。
    • 回乐园结算（第22章）：
      • 综合评价按「世界之源总和 + 完成任务数量」计算，分 E- ~ S+。
      • 世界等级超出猎杀者等级 5 级（噩梦难度）→ 奖励翻倍。
      • 奖励（属性点 / 乐园币等）自动存入烙印；评价决定升级幅度。例：苏晓海贼王结算 A-、+4 属性点 + 1 万币、升 LV.2（试炼世界单次 +1）；东京喰种结算 S-（世界之源 39.3% + 5 任务）、+15 属性点 + 1.9 万币、等级 +3 升 LV.5（多人世界升级幅度按评价、可 >1）。
      • 荣誉商店（猎杀者专属房间·凭荣誉勋章兑换·锁定不可交易·禁外传）现有：灵魂结晶（小）、荣誉复苏药剂（回满 HP）、细胞级修复（愈伤 + 清隐疾 + 脏器复苏）（二卷98章）。
    • 连锁任务：部分世界主线为连锁式（如东京喰种），完成数次方可选择回归，完成越多评价越高（二卷2章）。
    • 阵营任务：加入某阵营后可接的任务，不计入通关评价，奖励为阵营贡献值等。如苏晓接「掩护作战」（Lv.4·助特别对策班对抗青铜树·奖 10000 CCG 贡献值·无惩罚）（二卷53章）。借 24 区后勤部的兑换功能，CCG 贡献值已与乐园币近乎等价（二卷53章）。
    • 攻防任务：比连锁任务更难的一类主线——契约者多为防御方、共同抵挡来犯的大批敌人，防御成功即过、防线一破基本全员等死。须与（除违规者外的）契约者积极合作。如进击的巨人主线「罗塞之墙攻防战」（Lv.7）（三卷1章）。
    • 成就任务：达成特定条件自动触发的隐藏任务，额外给奖、有时比主线更丰厚。如苏晓亲手驱逐 14 区 34% 喰种、致全区 95% 喰种死亡 / 逃离，触发成就「清道夫」（亲驱区域 30%+ 喰种且致全区 90%+ 死亡或逃离），奖 1 属性点 + 7% 世界之源（二卷36章）。
    • 主线第三阶段＝优胜劣汰：连锁主线常以「最终决战」收尾（如东京喰种主线③：5 天内消灭敌方阵营半数以上契约者，奖回归资格、失败强行处决）——乐园借此淘汰没潜力的契约者（二卷86章）。
    • 任务失败豁免卷：职工者专有福利（4000 乐园币/张）；主线任务失败也不被强行处决，仅损失通关奖励。职工者只需活到最后即可（希珞珞 即靠此保命）（二卷91章）。
    • 主线时限策略：主线达成后可不立即提交——一旦提交，剩余停留时间即清零；而衍生世界是变强资源的唯一来源，故延后提交可多换自由行动时间。苏晓升上等达成主线②后即按兵不动，且因 CCG 阵营仅他一人，他不交、喰种阵营契约者的对抗类主线③也无从开启（二卷37章）。

    衍生世界¶

    • 可能是真实世界：苏晓推测衍生世界极可能是真实的（因左大臣会违约，纯 NPC 则不会）；猜想其由人们的幻想衍生、最终成为真实世界（第17章·推测）。
    • 时间线重置：契约者进入时，衍生世界的时间线会被重置 / 刷新（剧情回到最初），消耗乐园「时空之力」；消耗带来代价（如东京喰种耗 0.03 盎司 → 该世界契约者最低评分门槛升至 C 级）（二卷1章）。
    • 剧情关键人物：击杀重要剧情人物前，乐园可能要求取出其关键部位（如喰种的赫包）以保剧情延续，否则世界恐生变故（二卷4章）。
    • 位面之子：衍生世界存在关键人物「位面之子」（如幼年路飞）。乐园会血红警告令猎杀者远离；接触会触发超高难任务（如「传奇的扼杀」LV.79）（第16章）。
    • 剧情主角与命运之力：衍生世界设若干「剧情主角」、承载命运之力。剧情主角分两类——真正的「世界之子」（如路飞，气运护身、几乎不可杀），与名义主角（如金木研：蜕变前不可杀、蜕变后死活无妨）。剧情主角死亡后命运之力会重新分配给新主角（金木研死后转给雾岛董香与神代利世各 1/2，全服公告）。杀剧情主角的宝箱品级高于其本身实力 1–3 级（气运越强、死亡掉落越丰，故强主角反而死亡率高）（二卷64章）。
    • 剧情人物助战影响掉落：若衍生世界的剧情人物出手帮忙战斗，会大幅压低宝箱掉落率（苏晓在 24 区由有马 / 尤娜同行时几乎开不出宝箱）（二卷64章）。
    • 击杀剧情关键人物 → 世界之源：斩杀「剧情关键人物」可获大量世界之源（如进击的巨人世界：杀团长艾尔文 +2.1%、铠之巨人莱纳 +4.2%、超大型 +4.1%、艾伦 +9.6%、颚之巨人尤弥尔 +4.9%）；越是核心、所得越多（三卷46、四卷1章）。
    • 世界规则崩坏：若一次性破坏衍生世界的核心承载结构，世界规则会崩溃、时间重置、整个世界毁灭，乐园会血红警告猎杀者立即撤离（时空之力侵袭下幸存率 0%）。如苏晓同时杀死超半数「巨人之力传承者」，致始祖巨人之力复生时找不到载体、巨人世界规则崩坏毁灭——这也是绕过「气运之子 / 剧情主角不灭」（死后命运之力会转给新主角）的极端解法：让承载之力无新载体可接棒（四卷1章）。
    • 秘境（藏宝室）：衍生世界中一些隐秘地点藏有固定价值的好东西（无运气之说、不像开宝箱看手气）。已遇到的秘境：海贼世界藏斩龙闪的仓库（小）、东京喰种 24 区竖井（中）、进击的巨人艾伦家地下室（小）、斩·赤红之瞳平民窟地道尽头的帝国实验室（藏紫色「帝具改造蓝本」·设「卡尼亚特毒气」机关·每秒 5 真伤）。「秘境」一词为契约者所创；有契约者把这些地点汇总成「秘境大全」出售（约 15000 乐园币）（四卷32章）。

    开放性世界（猎场）¶

    部分较大的衍生世界为开放性世界（六卷起的海贼王即是）（六卷1–2章）：

    • 难度跨度大：标 LV.X~??（不同区域难度不同），可能遇到实力极强、当前根本打不过的敌人，须格外谨慎。
    • 猎场：契约者在此掉落猩红卡片的几率 >60%——是「强者的猎场、弱者的地狱」；有实力者趋之若鹜、无实力者避之不及。
    • 传送更狠：进入时连那记招牌「后脑闷棍」都升级成「蓄力重锤」（眼冒金星）；且会把猎杀者单独传送、与契约者落点错开（防开局即遭强敌 / 同类围杀）。
    • 指定世界·烧焦图纸：一种稀少的消耗品图纸（市价可达 2 万币），使用后即把猎杀者下一个衍生世界指定为图纸对应的世界与落点（进入流程、任务一切照常）。苏晓花 7000 币购得一张烧焦图纸，把第六个世界指定为海贼王·七水之都（五卷31章）。
      • 更改世界·世界钥匙类道具（八卷80 / 九卷1章）：另一类「世界钥匙」消耗品，效果是在乐园即将开启的衍生世界上、把它『更改』成自己想去的世界（与烧焦图纸「指定」近似、但落点身份随机·「极其坑人」）。苏晓 5 万币购入 斩首大刀（残缺）（紫·150）——乐园本要送他进大剑世界，他用它改去 火影忍者世界（消耗时空之力 0.022 盎司·最低评价随之定为 C−）。用世界钥匙进场=落点 / 身份全看运气（苏晓直接醒在雾忍地牢、被安排成「刺杀四代水影失败的刺客」）。
    • 时间线对齐：猎杀者因参与过战争 / 空间世界，会与普通契约者的进入时间产生偏差；乐园会提前送其进入下一个衍生世界以对齐时间线（故苏晓的现实复仇计划被打断、改日再续）（五卷31章）。

    空间战场（特殊衍生世界）¶

    轮回乐园与虚空中的敌对势力开战时，会向契约者发布的特殊战场（首见于轮回纪元 4576249 年·乐园与「虚空中某部落」宣战，因「最初条例第三十一条」乐园不得强行处决虚空部落，遂派契约者出战）（五卷1章）：

    • 紧急征调·无拒绝权：未完成猎杀任务的猎杀者可被强行征调入战场（苏晓因斩赤「113？？号」未现身被征调），且内容列为高等机密、不得外泄。
    • 无主线、改战场任务：空间战场内没有主线任务，改为战场任务；猎杀者与契约者的战场任务不同；本方契约者之间、契约者与猎杀者之间均不得互相攻击 / 挑衅。
    • 中立阶职 · 裁决者（Ruler）：因战场内「不存在违规者」，猎杀任务变更为特殊任务「镇压暴乱」——猎杀者以中立身份参战、负责战场内裁决与维和、清扫双方的违规者；进入战场后阶职激活为裁决者（Ruler），烙印为黑色镰刀 + 天平（裁决与公平），是「战场内最高权限」（因战场上动用「强行处决」太耗资源，遂由猎杀者代行）。阶职在战场结束后消失。
      • 裁决者特权：黑白面具 + 黑风衣造型；鸟瞰视角（战场专属·浮屏看飞船各方向战况）；可设结界困住违规/逃跑的契约者单挑裁决（他人不得插手）；伪装（免费·把自己与生物坐骑在契约者视角中改头换面，避免日后被认出）；在战争频道（裁决者+各团长的通讯平台·团长名后显杀敌数）以黑色文字发言。
      • 不得直接干涉战场（最初条例第 1578 条）：裁决者不能亲自参战，违者警告、五次警告后剥夺身份并惩罚；但「间接干涉」（指挥溃散的散人、在战争频道发布情报 / 征集令等）只算擦边、不亲自杀敌即可——苏晓借此为本方提供关键帮助。表现优异还会获乐园奖励（变相鼓励夺世界）。
      • 裁决者可互杀：敌对势力也会派己方裁决者参战；双方裁决者交手，乐园不阻止、不视为「参与战争」——苏晓据此斩杀卡拉星人裁决者图卡而不算违规。
      • 裁决结界：裁决者对「攻击自己的敌方单位」也可发动结界裁决（本世界最高等能力·除非裁决者战死、被困者无法离开，连卡拉星人王迪卡的瞬移都出不去）。苏晓故意激怒迪卡使其先动手，便取得反击 + 结界裁决之权，把瞬移强敌锁进结界单挑（五卷22章）。
      • 督战实质：苏晓看穿「中立」是幌子——乐园让裁决者兼任契约者一方督战官（防逃兵、防敌方阴招），以确保战争胜利、夺取该衍生世界。所谓「公平」只是说辞。
      • 世界三方：空间战场的衍生世界往往有三方——契约者（代表乐园）、虚空敌对种族（唯一知道乐园存在、负隅顽抗）、本土势力（不知乐园存在，可争取为联军）。寄生兽世界即：契约者 / 卡拉星人 / 人类联盟（五卷8章）。
    • 职工者强制摆摊（战时物价管制）：战争将至时，乐园强制所有职工者摆摊、且不得哄抬物价甚至须降价（消耗品降约 30%、装备更低·摊位标题清一色「求别买」）；违者交易权限被剥离 30 天；摆摊额度按其储蓄物品价值裁定、摊位免费。这是给参战契约者的隐性优待（也侧面反映战场之凶险）（五卷2章）。
    • 世界坐标（夺取衍生世界）：未被乐园占有的衍生世界无法重置时间线。猎杀者须用「世界坐标生成装置」在指定位置设满 3 处坐标（每处需契约者保护 12 小时），设满后乐园即重置该世界、回归剧情时间线，今后可直接把契约者传送进来（不必再乘飞船）。这揭示了一个推论：衍生世界并非乐园制造，而是乐园「抢来」的——世界本就独立存在、会正常推演，「重置时间线」才是乐园的能力（五卷3–4章）。
      • 坐标须设在乐园指定区域内（不能随便选址·缘由未明），猎杀者只能在抵达点附近半公里内择地——故苏晓总挑「只需守一个洞口 / 隘口」的易守地形、且不给守军留退路（防御失败即死）。设坐标的猎杀者孤身深入废土，面临的危险是普通契约者的十几倍（五卷11–12章）。
    • 裁决者阻止越境：契约者擅自离开战场范围（「越境」）会触发警告、要求本方裁决者阻止；本方裁决者若不阻止，乐园会改派敌方裁决者来清理越境者——但若敌方裁决者已被杀，此惩罚便落空（苏晓正是借此放魅力特长者出界去联络盟友·「裁决者身份意外好用」）（五卷10章）。

    • 夺取后 → 新人温床：衍生世界被夺取后首次（温和）重置时间线、回到剧情早期、难度大幅下降（寄生兽 Lv.10→Lv.3）；据初始条例 13572 条乐园可处决已战败的虚空种族（卡拉星人就此泯灭）；此后该世界沦为培养新人契约者的温床，坐标会被挪到剧情主角附近（五卷24章）。
    • 难度：以战场难度评级（苏晓所遇为 D 级战场·极难；落地后的寄生兽世界本身难度 Lv.10）。最大停留时间按难度定（D 级为 8 天）。

    世界争夺战（乐园 vs 乐园 · 十二卷1章）¶

    区别于「空间战场」（乐园 vs 虚空敌对种族）——世界争夺战是两个「乐园」之间争夺一整个世界的内部资源争夺（详见 维斯星球·多亚古大陆）：

    • 对手 = 天启乐园：与轮回乐园疑同出一源的另一个乐园（其契约者发色 / 瞳色特殊·组织度高·成「教团」）。两个乐园争夺的「资源」是一整个世界——世界越多、契约者变强机会越多、契约者越强越能夺世界，良性循环、乐园愈浩瀚。这部分揭开了「乐园为何培养契约者变强」之谜（养兵千日、用兵一时）。
    • 虚空之树公证（绝对中立）：争夺战由「虚空之树」依「初始法则第 1548762 条」作绝对中立公证——战争期间双方乐园均不可对治下穿梭者作任何资助（公平争夺、无神魔互撕既视感）。苏晓由此推断：轮回乐园虽于他已近乎无所不能、却并非没有对手，其上还有「虚空」「虚空之树」等更高存在（灭法之影或源自虚空·他现连进虚空的资格都没有·重大伏笔）。
    • 乐园功勋系统开启：争夺战世界无法获得世界之源，收益改走功勋——杀敌方将领 1~3 功勋、杀敌方契约者 3~5 功勋，胜利后按功勋值发奖。
    • 流程：传送经「星空中转站」（无形屏障隔开双方·同世界的同阶契约者集结）→ 目标世界。另发通用语 Lv.40（临时·战后遗忘）。禁止向争夺世界的土著提及乐园（违则公证后处决）。
    • 敌乐园也有「猎杀者」：天启乐园的对应身份叫「肃清使」（同样清理触禁忌的契约者·发展方向同为「人对人 PVP」）。世界争夺战中敌乐园的猎杀者会奉强制任务来与己方猎杀者分生死（一世界只能活一个）。两个「同类」相遇有奇特之处——猎杀者 / 肃清使之间直接对话不被乐园屏蔽（普通契约者听某些话会被屏蔽）。苏晓在维斯星球首遇天启肃清使 兰爵。
    • 战功系统（争夺世界本地·可与乐园功勋互换）：获本地官职后激活——杀普通士兵 ≈ 1 战功、攻破工事 / 杀敌方军官另计（杀军官可在战功 / 乐园功勋间二选一）。战功可在军需官处兑换（乐园功勋 100 战功〔战争世界打 1 折〕/ 灵魂结晶〔中〕260 /〔小〕90 / 金色装备万级战功等）；战功可在同阵营契约者间转让（利于冒险团集中资源·也易生矛盾）。乐园功勋限量（疑战后按比例兑属性点等·故被抢购）。

    剧情人物好感度（魅力特长机制 · 十二卷16章首见）¶

    苏晓激活军需官 尼古拉斯·凯撒 时首次接触——这是魅力特长契约者司空见惯的系统（魅力属性无独立战斗力、却能借好感度从剧情人物处捞取大量好处）：

    • 好感度分级：1~10 初次见面、10~60 朋友、60~80 挚友、80~100 特定情况可招其为奴仆 / 从者（异性剧情人物好感够高甚至可发展私情·同性则止于朋友 / 奴仆）。
    • 可大起大落：好感度随言行实时增减——凯撒为压价买卖、好感度一度 −60 / −30 / +90 地剧烈横跳（见利忘义型 NPC 的写照）。好感度高也未必触发招募（视 NPC 性格而定）。

    返回现实世界¶

    契约者在乐园停留期满后被传送回现实，约一星期后接召唤返回乐园（见 契约者周期）（第25章）。返回现实时（第29章）：

    • 裸装六维与被动技能在现实有效——苏晓现实力量暴涨（踹飞停尸柜、徒手制服、车速 160 仍游刃有余、食量数倍）；但主动技能无法在现实使用。六卷82–85章在金巴布韦实战印证：现实里衍生世界的装备 / 加成全被封印，他只能凭裸装六维 + 被动技能（刀术大师斩飞子弹 / 斩断坦克炮管、绝魔体质、心眼预警、灵影体质、恶魔之印）作战；即便如此仍是顶尖战力，唯独还扛不住现代热武器（最多硬抗重狙三枪、被打中头 / 心脏即死）。
      • 「堆被动」碾压「堆装备 / 主动」（八卷82–83章·补充）：现实禁主动技能 + 封装备，使靠增属性装备 + 主动技能吃饭的契约者沦为「一阶较强」的肉身（如外来者 巴赫 乐园力 56→现实仅 45）；而苏晓四属性均衡 + 一身被动 + 刀术大师「刀芒」（非主动技能、现实可用），对这类人是降维打击。现实感知力受压制——除专职感知系外，因空气中「元素 + 一种未知能量」太稀薄，感知只及身周几米（几米外模糊）。
    • 乐园内所得的装备 / 物品在现实锁定、无法取出，回乐园后恢复。现实无任何恢复品（受伤只能就医或自愈）——故六卷79章进化为真生命「从者」的 布布特尼 随苏晓回到现实后，其「每秒回 1 生命」光环弥足珍贵。
    • 不可以任何方式向现实泄露乐园之事，否则警告、无效则处决。
    • 现实中的违规契约者：乐园不支持干涉现实，但有人铤而走险牟利（如花乐园币买「心理系」广告设陷阱诱骗同类）——这类「违规但未犯条例」者，正是猎杀者的清理对象（第31章）。
    • 国家已收编契约者：现实国家早已察觉契约者的存在，成立机构「清理不稳定因素规划局」（俗称「清道夫」·由收编的契约者组成）。对无害者柔性招安——签约（不杀人 / 不乱社会 / 不非法集资 / 不拉帮结伙）即既往不咎、身份合法化，违者「不死不休」；契约者能在现实安逸生活，正因有这层管控（也可能是「杀出来的」平衡）。苏晓签约后身份合法（四卷11章）。
    • 临时储蓄功能：花 100 乐园币可把现实物品暂存入储蓄空间 7 天（有重量 / 长度 / 非高爆放射等限制·刀类擦边、枪械不行）——苏晓借此带家传长刀「唐红」过高铁安检（四卷11章）。
    • 放松有益感悟：长期绷紧神经有害；苏晓在现实游玩放松后，刀术专精竟自行感悟提升一级（Lv.10→Lv.11）（四卷12章）。
    • 时间转换功能（九卷92章·二阶解锁）：契约者晋升二阶 + 通过晋升后的第一个衍生世界后激活——可把现实世界停留时间按最大 1/2 折算成乐园停留时间。乐园时间宝贵（能打竞技场 / 卖炼金炸弹 / 逛市场变强），故无现实牵挂者多选择折算多留乐园（苏晓借此把乐园停留从 0 增至 84 小时）。阶位越高、乐园停留越久——长此以往现实世界的契约者越来越少，对现实的干扰也随之减小。

    荣誉商店（猎杀者专属·补充）¶

    猎杀者专属房间角落的隐藏商店——仅能用荣誉勋章兑换（青铜 / 黄金等）、所兑物品锁定不可交易、禁止向任何人提及（否则强制处决）、限量、不退不换。九卷火影通关后苏晓持黄金荣誉勋章可兑（三选一·均 1 枚）：

    • 主线任务失败豁免权：豁免一次主线任务失败的惩罚（扣属性 / 乐园币 / 强行处决）——苏晓所选（保命底牌·乐园对失败者绝不留情：失败 = 没培养价值 = 死）。
    • 空间破冲装置：可带 1 件蓝色以下恢复品进入现实（一次性）。
    • 紧急医疗箱：荣誉复苏药剂Ⅱ型（2 秒回满 HP/法力）+ 肾上腺素 + 急救绷带 ×10（虚空游侠部队常备）。

    冒险团¶

    契约者的组队方式（二卷1章）：

    • 临时冒险团：进入衍生世界前临时组建（交易广场招募，须同往一个世界）；世界内不可互相攻击，但付 10000 乐园币违约金可强退。
    • 永久冒险团：到乐园「冒险大厅」申报、完成任务后获得，团长可邀人入团；等级相近的成员会进入同一世界，有特殊加成、提高生存率。需等级达 Lv.10 方可建立。
    • 苏晓作为猎杀者不便入团（怕猎杀目标是队友），但对永久冒险团的加成心动，考虑日后建「一人冒险团」。
    • 冒险团专属团队技能：永久冒险团可拥有专属技能。如彼岸花的「血色追印」——其成员被某契约者所杀后、一旦该契约者靠近团队一定距离即被激活（每团仅一次、持续半小时），半小时内全团可追踪该猎杀目标的位置。但猎杀者有「虚假编号」：脱离视线后乐园会为其锁定虚假位置，使追踪失效（三卷28章）。
    • 剧情临时冒险团：与衍生世界的剧情人物组队时形成、可随时退出且无惩罚。苏晓接「源」任务后与剧情人物有马贵将、高槻泉组成此团——乐园借此变相帮苏晓提防背叛（队友若起背叛之心会自动退团、给他预警）（二卷82章）。

    世界难度¶

    进入时显示的难度，是契约者所在区域的难度，并非整个世界的真正难度——海贼世界显示 LV.6 仅指哥亚王都，而该世界存在 LV.79 的 蒙奇·D·卡普（第16章）。一般不会被送往高出自身 5 级以上的区域（猎杀者例外）（第3章）。

    半数据化¶

    辅助战斗能力（第2章）。要点：以现实为准、非游戏化；血条显示生命体征、受致命伤直接死亡（哪怕余 100%）；脊柱被毁可「假死」；血量<10% 不狂化反而虚弱；要害重伤仍会死（第2、10、12、14章）。法力值恢复：每小时回 10 点（苏晓 613 上限·耗尽后约三天回满）（四卷77章）。

    物品与宝箱¶

    物品分规则物品（可带出）、几率拾取物品（不可带出）、触发类道具等；宝箱品质白 < 绿 < 蓝、所开物与掉落者相关。详见 装备·道具（第4、7、8、19章）。

    装备强化与品质提升（八卷77–78章·补充）¶

    • 品质提升靠「吞同类武器」：成长 / 可升品的武器要升一个品质档，须祭献 / 吞掉若干把同类、同档（或更低档）的武器（如斩龙闪升暗紫需吞 2 把紫色刀类）。升品质大概率会掉强化等级——蓝→紫时费用相近、还可能保留；紫→暗紫则强化费陡增、强化等级必降（咨询乐园需付 500 币咨询费）。
    • 暗紫色 = 一阶天花板：暗紫色装备是少部分一阶契约者能获得的最强装备，更高品质非一阶契约者所能得。
    • 装备强化大厅：交易市场外人最多的地方——风险极高、收益也极大（暗紫装备强化失败会破碎·门口常有人扇自己耳光发誓再不强化）。强化费随品质 / 等级飙升（暗紫武器每次约 23000 币）；幻魔石 可保底（免一次失败惩罚·成功不消耗）。成功率随等级骤降——+11 仅约 6%、+12 仅约 1%（苏晓「一向倒霉」却把斩龙闪强到 +10·见好就收）。职工者在此衍生出卖强化技巧 / 代强化等生意。

    竞技场（一阶 / 二阶分场）¶

    • 模式：竞技模式（个人战 1V1 / 团队战 10V10·两种排行分别计算）+ 观战模式（按场次 / 名次档收费·5 币 / 时起·看前 100 名 30 / 前 10 名 100 / 前 3 名 300）。

    衍生世界联络平台（二阶契约者功能 · 九卷13章）¶

    晋升二阶解锁的功能——进入新衍生世界后生效（一阶没有）。是一个虚拟聊天框，让身处不同衍生世界的二阶契约者跨世界交流（免费·每人有发言次数上限·苏晓 5 次·团长 / 更高阶契约者发言次数有加成）。

    • 用途：乐园借此促进契约者抱团组队、减少死伤（过一阶考核的都非弱者）；也能即时叫卖 / 交易衍生世界内所得之物（但交易风险高、需双方实力对等）。
    • 副作用：身份会随发言暴露——钢炼世界时，正因有此平台，所有二阶契约者都知道苏晓在地下清场。苏晓因 神皇、咕噜 等认得他，进火影世界后选择潜水不发言。他正是靠这平台得知三大冒险团（神皇 / 血门 / 蛮天）在火影世界争夺「猎魔套装」、咕噜抢到猎魔戒等情报。

    询问限制¶

    开局阶段，乐园对契约者的提问只回答有限次数，耗尽后不再回答（第2章）。
