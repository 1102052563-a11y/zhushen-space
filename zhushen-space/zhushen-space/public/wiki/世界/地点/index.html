    • 哥亚王国：东海国家，统治附近城镇村庄；周边有风车镇、科尔波山；首都即「王都」（第2、4、6章）。
      • 王都：分平民住宅区等区域；街道熙攘，但平民挣扎在温饱边缘（第6章）。
      • 欢乐街：王都的花街，粉红灯光、楼阁林立；为左大臣经营、侍卫首领欧卡常客，苏晓在此暗杀欧卡（第9章）。
      • 嘟噜杂货铺：王都一间生意萧条的杂货铺，常被守卫队压榨；苏晓在此购得木天芷（第12章）。
      • 王宫：国王所在，戒备森严；主线刺杀的最终舞台，苏晓于第20章末潜入（第2、20章）。
    • 废品终点站：哥亚王国城外大型垃圾场、流民聚居的「无法地带」；第20章被克利克海贼团焚烧，数千流民遇难（第3–6、20章）。
    • 科尔波山：哥亚王国周边的山林，原著主角路飞长大处；风景秀丽但猛兽出没、水源稀少（海岛缺水）；精英生物 科尔波山的主人 的领地（第3、11–16章）。
    • 风车镇：哥亚王国周边地名（第3章）。
    • 其它（提及）：王都监狱（苏晓在此救出死刑犯做交易）、平民区旧仓库（密会克利克）（第19、20章）。

    七水之都（第六卷再入）¶

    • 七水之都（Water Seven）——伟大航路上的「水之都」。本是一座陆地岛，岛体下沉后居民不迁、在旧建筑上层层加盖、高出海面，千百年累积成铁塔状的海上城市（顶端涌海水如喷泉、沿数条水道流回大海·「名副其实的海景房·窗口就能钓鱼」）。海军与海贼都来此买船 / 修船、彼此克制；只驻十几名海军（地位特殊·古代兵器冥王即在此建造·世界政府对其态度暧昧）。海贼须从后街低调进城（遮羞布）。交通工具「布鲁」：温和的两栖「牛鱼」、背驮带座椅小船载人穿行水道（租约 1000 贝利）（六卷4–5章）。
    • 一号船坞：卡雷拉公司 旗下船厂之一，堆满木材、火炮；苏晓受聘负责其安保。市政办公楼（人工陆地区·权贵聚居·社长冰山办公处）。
    • 船墓场：废弃造船厂、堆放旧船之地；人烟稀少，苏晓选作与左轮男决斗的战场（六卷6章）。
    • 水之诸神（年度大海啸）：每年一次席卷七水之都的巨大海啸——居民躲进后街避难所或乘海上列车避开；海啸却冲洗全城（卫生大好、少疫病）、冲来鱼群（渔业富饶），故居民虽住在下沉的岛上也不肯迁走。海啸前全城办庆典（化妆舞会·华丽长袍 + 肃穆木质面具·祭水神祈福）——苏晓、布鲁诺正借庆典服面具掩护抓捕罗宾（六卷22章）。
    • 废弃工厂 / 废弃码头：七水之都僻静处，苏晓练剃、押罗宾审讯、与璃 / 莉莉丝交战的场所（六卷17、23–25章）。
    • 海上列车（海列车）：海面上铺设铁轨、靠煤炭驱动的列车，速度是船的十几倍、可克服一切恶劣气候；连接七水之都与司法岛。苏晓一行押着罗宾、弗兰奇乘此列车赴司法岛（贵宾在第三节·弗兰奇被关第七节）——苏晓正是在暴雨夜爬车顶潜入第七节、盗走弗兰奇体内的宝物（六卷33–37章）。
    • 司法岛：海贼王世界主线的最终舞台、世界政府的法外审判 / 处决之地（鱼人汤姆即在此被处决）；藏六式、恶魔果实等好东西、亦极危险。三座岛由吊桥相连：前岛（正门）→ 本岛（审判所·走流程·凡押来者皆已定罪）→ 司法岛（司法之塔·关押要犯）；司法岛后经踌躇之桥 → 正义之门 → 大漩涡（漩涡把海军本部、司法岛、推进城监狱三大要害连通）。苏晓预判草帽海贼团与大量契约者都将汇聚于此乱战（六卷6、30、38章）。
    • 海军秘密基地（荒岛）：司法岛附近、乘小船半小时可达的隐秘海岛（无码头·林间遍布十几座瞭望塔 + 隐藏大炮 + 数百埋伏海军·已肃清大型动物·驻守 4 万精英海军 + 黄猿 + 改造中的暴君熊）。世界政府的天才博士贝加庞克在此设耗资 50 亿贝利的大型实验所（正改造王下七武海熊·PX 计划）。苏晓被海军「护送」来此量产炼金炸弹；地下斜下方另有紧急避难所（苏晓在此试制太阳神阿波罗·失误炸出 300 米巨坑）（六卷40–45章）。
    • 玛丽乔亚（提及）：世界贵族天龙人的住址 / 圣地；世界政府供苏晓造太阳神阿波罗用的灵魂结晶即出自此地。苏晓动了「洗劫玛丽乔亚必暴富」的念头，但前提是先灭世界政府、难度太高（六卷44章）。

    东京喰种世界¶

    • 东京：喰种潜伏的现代都市，分区管理；开局大楼（20 区）为神代利世吃人之处（二卷1–3章）。
      • 20 区：设 CCG 分部（门口有 Rc 细胞检测门），苏晓初入 CCG 处（二卷8章）。
      • 14 区：苏晓任职区（紧挨 20 区、不太平）；下辖西街区——14 区的共喰区，S 级喰种极多（二卷9–13章）。
      • 11 区：青铜树老巢，近乎人类禁区（喰种困住平民当「口粮」）；后青铜树攻陷 11 区 CCG 分部、使其彻底沦为喰种领地，金木研被抓至此折磨（二卷13、34、38章）。
      • 24 区：人类禁区——并非特定区域，而是喰种在东京地下挖出、遍布全城、错综复杂尚未探尽的通道网；地面入口在东京边缘郊野（直径十几米的洞口直通地下、风声如鬼哭）；「A 级遍地走、S 级多如狗、SS 级才能勉强抖一抖」，共喰为常态（无人类可食），深处喰种或数以万计。地下多肉墙（赫包所制、类库因克技术）大幅抬高探索难度。准特等搜查官都无资格独自进入，唯 有马贵将 的零队敢深入；苏晓随其展开第 108 次探索（二卷36–37、42–43章）。
        • 24 区临时分部：地面一栋三层别墅，仅有马、苏晓、尤娜 三名搜查官 + 后勤 / 信息调查人员；其后勤部是「隐藏区域商店」，可凭 CCG 贡献值 + 职位兑换稀缺物品（人体基因强化液 / 粉碎精灵 / C 型注射器等，他区无此功能）（二卷42–43章）。
        • 金属门关卡：CCG 早年所建、阻 24 区喰种外出；钥匙仅两把，分别在 有马贵将 与 CCG 局长修吉时手中。门内即 肉墙 世界（二卷44章）。
        • 「活的迷宫」：地下肉墙非固定，岔路时为二 / 三 / 五条、约七八天一大改（平时细微变化），故无法彻底探索；整体呈漏斗形（深处为大头、地面为小头），凭指南针锁定一个方向便可走出（苏晓据 17 条死路推断出漏斗结构）（二卷44、47章）。顶部有剧毒 光花、地底盘踞 血眼 / 原始人喰种 / 无眼喰种。
        • 中心城（地下城市）：24 区深处掏空土层形成的东京最大喰种聚居地——金属建筑林立、设几十个入口孔洞，喰种如人类居于地面般在此生活；有「主控室」操控 24 区肉墙的变化（故地下通道总令人迷路）。暴走后被原始种 / 无眼喰种占领、遍地老幼喰种尸体（二卷46、75–76章）。
          • 井（迪亚克曼部族出口）：中心城中心一口直径十几米、深不见底的竖井，原始种正是自井下深处爬出；井盖为特殊合金（每扇约八九百斤）。苏晓与有马以诱饵把原始种引到井口、引爆炸药尽数炸杀、并炸毁竖井封死出口（乐园判为「炸毁迪亚克曼部族出口」、世界之源 +5%）（二卷77–78章）。
        • 古密室 / 旧实验室：肉墙缝隙后的一间古老房间，满布灰尘、无电子设备（天平、烧杯、木桌），疑为远古遗留的实验室；苏晓与尤娜曾藏身于此疗伤（二卷51章）。
      • 1–4 区：喰种密度极大的共喰区，普通人无法生存（二卷13章）。
      • 库克利亚（喰种收容所）：地处 23 区的喰种监狱，关押实力强、但非 V 成员的危险喰种（V 觉得杀了可惜）；CCG 在此对喰种做残酷实验（壁虎的扭曲人格便是在此被一名搜查官长期折磨而成）。高槻泉被「生擒」后即押往此处（二卷84章）。
      • 安定区咖啡厅（20 区）：喰种聚集地，店长 芳村功善 是年老的 SSS 级喰种（战技强于枭）；店长与店员皆喰种，却主张与人类和谐共处、不主动捕食（只食自杀者尸体），20 区双方互不主动出手（二卷9–10、40章）。
      • Helter Skelter 酒吧：14 区西街区一间复古无窗酒吧，SS 级喰种 伊鸟 开设，是喰种的小型情报中心；苏晓两度造访问取共喰区与 14 区喰种聚集地图（二卷14、17–18章）。
      • 大元拍卖场（大元体育场）：14 区东侧，白天为人类正规拍卖场、每周日晚 11 点变为喰种大型拍卖会（与会者皆戴面具）；设正门 / 后门 / 紧急逃生通道三出口。伊鸟地图所标聚集地之一，苏晓调 CCG 近百人封锁三出口、切电屠场，于此肃清上百喰种（二卷18–23章）。

    进击的巨人世界¶

    • 三面「神之壁」（由外而内）：玛利亚之墙（最外·845 年被超大型巨人踢破）、罗塞之墙（第二重·850 年被踢出三道破口、攻防战战场）、希娜之墙（最内·贵族区·范围小·容不下逃难平民）。墙内人类靠三墙抵御墙外游离巨人、安稳百年（三卷1–2章）。
    • 罗塞之墙攻防战分 一区（东）/ 二区（西）/ 三区（南） 三个破口战区，分别由彼岸花、兄弟会、及苏晓所在的杂队防守（三卷2章）。
    • 帕拉蒂岛 — 三面墙所在的岛屿，面积广阔、资源丰富（马莱觊觎之地）；岛上散布马莱流放来的无垢巨人。苏晓一行劫艾伦后须横穿此岛逃往墙外（三卷43章）。
    • 玛利亚之墙（已失守区） — 最外墙、845 年失守后大片沦为巨人横行的废墟，范围广阔、需长途跋涉穿越；内有：
      • 巨人之森 — 一大片巨树（树径数米）林，可在树顶躲避无垢巨人；但原著中莱纳等人正因在此休整而被调查兵团追上、夺回艾伦，故苏晓特意回避（三卷40章）。
      • 艾伦故乡城镇 — 5 年前超大型巨人踢破之处，已成荒废空城。
    • 耶格尔家地下室（玛利亚之墙·艾伦旧居）：艾伦之父 格里沙·耶格尔 留钥匙嘱其来此寻巨人真相。苏晓据原著线索（母亲被飞石压死的房屋）找到、一脚踹开门（钥匙「并没什么卵用」）；木质书房内藏可换世界之源的实验物品、暗格蓝色药剂、及揭示艾伦身世的三本日记（三卷43–45章）。

    斩·赤红之瞳世界¶

    • 极北 / 极北要塞 — 帝国北疆咽喉（黑石城墙·百年未修·零下数十度·两侧终年雪山）；异族军起义之地、艾斯德斯率 50 万镇压处。内有「咚咚塔要塞」（极北军据点·努马·塞卡被刺杀处）（四卷13–20章）。
    • 帝都 — 帝国首都、大型阶梯城（越中心越高）；中心是皇宫（城中城·圆形尖顶·最高百米·小皇帝与大臣居此）。下层是平民窟 / 花街（致幻药控女人、人体实验的黑暗之地·地下有废弃地道与帝国秘境实验室——藏紫色「帝具改造蓝本」·设卡尼亚特毒气机关）（四卷22、31–32章）。
    • 革命军总部 — 藏于深山老林的庞大古堡（约 40 米高·常驻 2000+ 军 + 几百官员·门外百余巡逻）；帝具库在地下室、由臣具「莫森尼亚·永固之门」封锁。苏晓潜入洗劫帝具库、杀首领娜洁希坦（四卷37–40章）。
    • 比鲁马克炼金试验场（皇宫地下 · 衍生世界隐藏地点 / 秘境）— 先代皇帝所建的帝国炼金试验场，深埋皇宫下、入口是祀堂祭坛旁可踩塌的十米竖井（井底有 1~13 号门洞）。唯皇室成员（免疫炼金毒素）与炼金师能自由出入；遍布无色无味的炼金毒素（外层卡尼森每秒 -10 / 中心斯德罗洛 +力敏体各 -10、每秒 -7·均真实伤害），猎杀者须花乐园币开「空气过滤装置」（100→150→300 币/小时·越深越贵）。内部分外围 / 中心地带 / 核心——核心充斥「V 型炼金病毒」；有迷惑方向感的草原、危险的炼金生物（蓝色小鸟 / 聚合虫 / 地穴人等）。向东穿过中心地带有通帝都郊外的紧急出口。苏晓被围皇宫时踩塌竖井、带小皇帝跳入此地逃生（艾斯德斯、威尔追入）（四卷58–63章）。

    寄生兽世界（空间战场 · 第五卷）¶

    详见 寄生兽 · 废土游侠。

    • 亚洲大陆（人类领域）— 末世人类联盟的聚居地，科技发达、高楼林立；外层设防护网过滤空气、阻卡拉星人进入。本批战场即位于其板块辐射区（缓冲带）内。地理上仍约略保留旧格局：下方原为中国（周边小国开战初即灭）、偏上为俄罗斯，二者是人类联盟中最大的两股势力；苏晓的战场在俄、中交界附近。
    • 冰封海洋（亚洲大陆最边缘）— 第三处世界坐标的指定区域，在俄罗斯原近海一带。卡拉星人改造大气致地球剧变、常年大雪，连含盐的海洋都结成厚冰层（疑蔓延至北美洲）。冰原无遮挡、又临大陆边缘，卡拉星人可出全力——契约者根本守不住，故苏晓索性凿穿两百多米冰层、把第三坐标设到冰层下的海水中，再炸塌出口、困于深海与之共存亡（五卷16–18章）。
    • 欧洲大陆（卡拉星人领域）— 卡拉星人最初降临并占领之地，危险重重。
    • 辐射废土 / 战场 — 经常被核弹轰炸、方圆百里寸草不生的焦土；空气含卡拉星人释放的甲醛酸味。本批战场为纵 10 公里 × 横 0.5 公里的长方形区域，空间穿梭飞船停于一端，契约者守四面。
    • 空间穿梭飞船 — 直径几百米、稀有金属铁黑外壳、满载未知武器的太空飞船；运契约者跨虫洞（673 马赫）抵战场，内为安全区（不得互攻）、内置第一枚世界坐标。

    钢之炼金术师世界（第七卷）¶

    详见 钢之炼金术师（晋升之路）。

    • 亚美斯多利斯国（亚美国）— 实为 BOSS「瓶中的小人」一手发展壮大的国家、军方高层皆其鹰犬；国土地图呈圆形，把历来战乱地点连线即是一个笼罩全国的超大型炼金阵（以冤魂为能量·终极目的是把全国人炼成贤者之石）。军力布局畸形：强军全在边境防外敌、中央军是弱鸡，导致边关将领各自为政、内部空虚（独靠瓶中小人这个「炼金中转站」维持强盛）。
    • 中央市— 亚美国首都、人造人的地盘（大总统拉斯坐镇）；苏晓化名爆炎炼金术师的住所与活动中心。
    • 里森布尔镇（伊修瓦尔附近）— 炼金阵外的小镇；七卷6章爆发暴乱（伊修瓦尔人挑起「圣战」·军队失踪），苏晓奉命「镇压」却布特级炼金炸弹炸毁整镇（已无平民·借此换 2~3 天自由 + 不在场证明）。
    • 里欧尔城（亚美国东侧·大沙漠旁）— 炼金阵节点城市，注定要发生暴乱且无人阻止；已被邪教（一名持贤者之石的炼金术师冒充「神之使徒」·人造人在背后支持）控制。苏晓在此教堂连斩人造人 拉丝特 与 格拉特尼（七卷6–9章）。
    • 伊修瓦尔（提及）— 被亚美国压迫的少数民族「伊修瓦尔人」（白发红眼·原不许出境）的故地；曾遭「伊修瓦尔歼灭战」（苏晓的伪装身份「白夜」即该战参与者）。
    • 第三研究所（七卷21–24章）— 中央市的军区研究机构、两大职能 = 研究合成兽 + 合成贤者之石（材料是死囚与战俘·库存超 8~10 块即送总统府）；外紧内松、地下层鬼片般破败（有灵魂固定铠甲守卫）。苏晓与其他契约者来抢贤者之石时惊动了所内压箱底的怪物 黑猩猩、整所塌陷。
    • 布里克兹（西北大雪山）（七卷25–26章）— 亚美国西北边塞·气温零下 20°↓；冰雪女王 奥利维亚·米拉 的半自治王国（军费自征、中央插不了手）。因它是国土炼成阵的一个角落、注定会爆发战事，苏晓要去此地联合冰雪女王结盟（通往西北的线路已被人造人戒严）。
    • 卡诺小镇（七卷35–37章）— 地处西北边塞与中央市的中心位置、居民不多却重兵把守，因握有一座大型火车站（战略要地·没火车就无法把几万军队 + 坦克大炮运往中央市）。北方军起兵后首先攻取此地：镇长因不许在镇前开战被中央军处决吊尸示众；五小时巷战夺站后，这里成为联军直插中央市的「跳板」与补给枢纽。

    轮回乐园（乐园）¶

    见 乐园：中心区直营设施（属性强化大厅 / 试炼场 / 竞技场等）、专属房间、职工者商铺街道。空间战争期间：天空透红光、职工者被强制摆摊降价（详见 机制 · 空间战场）。

    • 交易市场：契约者自由摆摊买卖装备 / 材料 / 宝箱 / 猩红卡之处（有「运气与实力」等专卖宝箱 / 猩红卡的店铺·赌运气如赌石）；可花 10 币屏蔽纠缠的低价收货职工者商人。
    • 机械街区（九卷90章）：乐园西南角由机械系（科学侧）契约者自发聚成的分区——科幻风（机甲 / 半机械人 / 喝机油的改造人），门口立牌「神系契约者有多远滚多远……不信神！」。苏晓 来此找 机械妹（机械改造 Lv.17+）用物品改造装置升级装备。乐园内类似的「分区」由同类契约者自发组成、非正式组织。
    • 洛克街（十二卷88章·乐园「黑市」）：一条街道、两旁尽是稀奇古怪的店铺——旧事屋（情报组织）、焚烧小店（可雇佣强大契约者杀人）、耳朵（情报中转站）等。卖的多是「来路不正」的东西（某大型团成员的标志性装备、争夺中「神秘失踪」之物）：便宜、却可能引来杀身之祸。整条街归 幻影旅团 所有（商铺使用 / 租赁权皆在旅团成员手里），故成了疯子与神经病云集之处、职工者鲜少踏足。苏晓 与旅团有旧怨（左轮男 / 咕噜）、平日不常来；十二卷在此卖月之精髓给法爷 所罗门、并与咕噜重逢。

    现实世界¶

    • 苏晓所在的二线城市：医院停尸间（其「假死」尸体存放处）、东城分局等（第29章）。
    • DL 市：三面环海的沿海城市，苏晓（化名苏笑）逃亡后的落脚地，于城市边缘老旧小区租房隐居（月租 800，同楼疑住契约者）；另有一条蹊跷的「十万元/月、天海路 37 号」租房信息，实为契约者陷阱（第30–31章）。
    • 金巴布韦（非洲）（六卷82章）— 苏晓现实复仇线的出国目的地。货币机制半崩溃（面值动辄上亿·40 万万亿金巴布韦元仅折合约 1 万 RMB·当地极度贫困、连「吃土饼」都真实存在）；外币在此流通不畅。复仇目标就在其边境附近。
    • 洗钻河（金巴布韦边境）（六卷82–85章）— 一片无名的产钻区（70% 钻石被国人收购·故 RMB + 美元广泛流通、汉化严重），实为「无法之地」：军阀横行、几股武装割据、贩卖 AK / 重机枪 / 炸弹如卖菜。大军阀 撒姆尔 的城堡即在此。苏晓在现实世界·衍生世界装备 / 加成全被封印、只能凭裸装六维 + 被动技能作战（详见 轮回乐园机制）。
    • 「小城」（三国边境无人区）（十二卷95–98章·苏晓 的「勇闯魔窟」任务地）— 三国交界缝隙处的无法之地，地图与新闻都不存在、有「保质期」（大人物需要时它才存在）。满地黄沙、唯一的「资源」是野生动物（街边烧烤野味、铁笼关动物），偷猎组织 / 毒枭 / 劫匪盘踞。入城前要过劫匪堵车的「破财桥」（只勒索财物、给受害者留下够回文明区的食物 / 水 / 油）和四国混编士兵的关卡（交过路费 + 带枪费）。城现由毒枭 巴谷 统治、其废品厂是大规模制毒窝点（守卫注射药物、体力超 5 点）。苏晓为「突破身体极限」（五属性压制到 5 点·见 属性）须深入废品厂地下室停留 30 分钟——这是他当杀手时脖颈中过一枪、险些丧命的旧地。
