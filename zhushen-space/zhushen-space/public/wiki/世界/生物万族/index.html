    • 独眼巨怪（领主级·暗黑大陆生物）（十卷13–18章）：被人「随机召唤卷轴」从暗黑大陆召来的怪物——5 米高、深紫皮肤、短红犄角、嘴咧到脖颈、一只占脸三分之一且密布十几个瞳孔的独眼，嗜食人肉（拿树干当棍）。其在暗黑大陆只是中下层生物（温饱都难、强敌环伺），到六块大陆却如入天堂（弱小猎物=人类数不尽），但出于本能不敢远离召唤点。属性力 70（含领主 +13）/ 敏 34 / 体 59；技能「领主生物」（生命 +3000·力 +13 / 敏 +15 / 体 +10）+ 不屈 / 暗黑生物 / 狂暴（濒死回血·15%→40%·秒杀级反扑）等。防御力恐怖（弓箭 / 法术 / 自动步枪如挠痒·只苏晓暗紫蜘蛛女皇能破防出血洞），但智商是硬伤。用以衡量阶位：领主级之上是霸主级（如吞噬岛巴哈姆·苏晓在其面前曾首尝「无力」·随手一击即秒）。被苏晓单挑斩杀（十卷18–19章）：先前十枪有三枪打中巨怪关节、令其近战大减破绽变大——苏晓界断线借力 + 能量盾化解、斩断脚踝韧带 / 手臂、环断切喉、最后从口腔刺穿脑髓（巨怪求饶亦无用）。初杀首领级生物奖 5000 币 + 灵魂结晶〔小〕×3 + 世界之源 13.5% + 掉落领主宝箱（淡金）；苏晓还拖走其尸体（暗黑大陆之物·研究价值高）。
    • 尼拉潘（暗黑大陆·红色小丑炮灰）（十卷52–56章）：上文「异界召唤生物」的真名——红色小丑 凭 炼狱伯爵 的传送阵从暗黑大陆召来围攻拍卖场的类人形大军。约 2 米高、全身漆黑、额头并排四眼、双手长及膝带利爪；属性力 35 / 敏 36 / 体 32 / 智 28 / 魅 1·法力 280。技能：群居（数量超 10 / 100 / 1000 时力敏体各 +1 / +2 / +5·封顶 +5）、开阔视角（四眼 230° 视野）、爪刃（爪极坚韧锋利）、分裂繁殖（死亡 5 分钟内分裂——一具尸体可裂出3 只幼年尼拉潘·幼体需催生 10 小时成年〔代价：寿命仅 2~5 天、不能再分裂〕）。个体战力弱、却生存力极强、悍不畏死（对「高位尼拉潘」绝对服从、可为命令赴死、幼体被杀会激起暴怒）；极度依赖水（无水则分裂受遏制·苏晓正是靠炼金炸弹提前炸 / 烧尸断其分裂）。是剧情生物（死掉绿 / 蓝宝箱·非召唤物）。三米高的首领与红色小丑约定「攻下拍卖场 = 把尼拉潘全族传送来繁衍」——实则被小丑欺骗（小丑无此能力）；族人惨死后首领愤怒撤兵、约一刻日（1 小时 13 分）后发起最后一波。

    Fate/Zero 世界（第十一卷）¶

    详见 Fate/Zero（圣杯战争）。

    • 怨生物 · 「生命同化度」转化体（十一卷20–24章）：盘踞柳洞寺的诡异生物——由普通人 / 普通生物经一种法阵转化而来（法阵用白色粉末勾画、引柳洞寺地下的大量魔力·转化成功率仅约 5%·失败者被送「永生池」当养料）。共同特征：头上有红色「戒疤」（其实是头皮反复被骨刺从内部刺破留下的疤）、技能邪恶信仰（生命 +200·意志 +5）、骨骼外放（局部外放骨骼·防御 +7~8·双臂可包裹外骨骼成利爪）、生命同化（吞噬生物血肉增「生命同化度」+ 回血·甚至能吞自身血肉回血·同化度满 100% 则进化）；情绪剧烈波动时会变得极度弑杀。分级：怨僧（苦行僧转化·力 40 / 敏 42 / 体 55·同化度 30%）、怨侍（平民转化·力 39 / 敏 43 / 体 53）、怨侍头目（三米高·要害覆外骨骼·主人在永生池沉睡时代管柳洞寺）、骨刺老鼠（同化度 3%）等。「普通级」怨生物拿到别的二阶世界已是小 BOSS 级——苏晓 凭斩龙闪锋利 + 刀术 Lv.30 速度多能一刀两断、但忌惮数量（上百只则吃力）。杀怨生物得世界之源（柳洞寺为隐藏地域·奖励翻倍）+ 圣堂教会功勋（可在言峰教会老神父处兑奖）。动物也能被转化：法堂看守「平头哥（利牙）」是一只蜜獾转化的怨生物（猎豹大小·黑色外骨骼裹全身仅露眼 + 头顶一块白·咬合力能咬穿 400 点反击盾·脾气暴烈六亲不认·咬住斩龙闪团成球软磨硬泡·被苏晓收刀后一脚踢飞十几公里·后被 布布汪「大忽悠 + 零食」劝和、得知修斯已死便「恩怨一笔勾销」返回冬木市找旧仇〔一只母狮〕去了）。顶端 = 领主级 怨之孤儿·修斯（永生池之主·生命同化度 97%·只差一步「进化成神」·拥灵魂感染光环等成堆 Lv.20+ 技能·被苏晓圣水爆头斩杀）。苏晓最后用 太阳神阿波罗 把整座柳洞寺连同上百怨生物一并气化夷平。

    恶魔古堡游戏世界（第十四卷 · 强者之森）¶

    详见 恶魔古堡游戏。第四场「强者之森」（人造太阳的森林）里关着各星球生物与「客人」（爱丽丝抓的囚徒）——森林里十之八九的生物体内有毒、不可食（爱丽丝特意安排）。

    • 寄星蟹（虚空生物 · 强者之森食物链顶端）（十四卷35–39章）：通体洁白、巴掌大的螃蟹，能屏蔽感知（近 7 米才被苏晓感知到）；以百万计群居、汇聚成一股股潮水般涌动（远超单体爬行速度），还能群体伪装成山岳 / 星球诱猎物靠近。使徒之眼测得数据：生命 100%·法力 180·力 36·敏 69·体 16·智 18·魅 3；五技能——①啃食星球（被动 Lv.59·数量超 7000 万即寻小型星球寄生、啃光一切有机 / 无机物后伪装成那颗星球）②无氧环境生存（被动 Lv.30·甲壳叶形肺存固态氧·无氧可活 30~50 年）③疯狂繁殖（被动 Lv.63·雌雄同体·繁育力已被爱丽丝封印）④酸性体液（被动 Lv.12·唾液血液强腐蚀·能在两秒内啃穿 400 点能量盾）⑤群体伪装（被动 Lv.41）。是虚空里的「蛀虫」——每毁一星即破坏大范围元素平衡，故灭法之影（如「卡恩·塞力斯」）见之必诛、寄星蟹血脉深处恐惧灭法之影 / 青钢影。苏晓只在它们领地打了桶水、那身精纯 青钢影 便惹得百万寄星蟹不死不休追杀，他靠 气息外放 反复屠戮、五次骚扰把这群虚空生物逼到「搬家」，由此攒下 24 万游戏积分（淡水从此取之不竭）。爱丽丝弄进古堡的只是个被封印繁育力的「小族群」（否则一周内古堡四层会被啃成蟹巢）。
