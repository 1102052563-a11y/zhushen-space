    黑色森林 / 死亡之地（八卷52–53章·隐藏任务第二阶段）¶

    • 进入门槛：黑色森林封着「不死者」与各种复苏的怪物（比红土区危险十倍不止），进入必须携带大量「原生之种」当门票，否则必死。原生之种须经 阳之女 改造（黑变淡红、生翠绿枝芽）：进林后随身带、每两小时捏开一颗续命两小时、十颗用光前必须撤离（苏晓讨价拿到 15 颗）。

    黑色森林实况（八卷62–63章）¶

    • 黑雾之「牢笼」：黑色森林外缘弥漫一层黑雾——抛入的树枝瞬间被分解成烟尘；黑雾如墙、把整片森林封成一个牢笼，只能从唯一入口进出（连入口也有黑雾、只是稀薄）。黑雾会周期性消退（短则三五天、长则半个月）；试炼第十天（最后一天）黑雾彻底消失、里面的东西脱笼而出（即「寒灾」）。如今黑雾已稀薄（几个月前还厚如墙）。进入靠 巴尔 吐一分钟白火在入口烧出窟窿（只维持十几秒·吐完萎靡·入口随即愈合）。
    • 黑雾之源·侵蚀：入林即受「黑雾之源」侵蚀——每秒 1 点真实伤害·满 10 分钟翻倍（10 分 2/秒、20 分 4/秒、30 分 8/秒、半小时后每 5 秒一次即死判定）。捏开一颗改造原生之种生出翠绿藤蔓攀身即可抵消侵蚀（坐实「原生之种 = 黑色森林门票」·无它嗑药也撑不过半小时）。
    • 内里反差：黑雾之内的森林并非死寂——树木其实不黑（外面看黑是黑雾笼罩）、鸟语花香、空气带淡淡甜味，却处处剧毒（一只红色小虫的体液瞬间把匕首腐蚀成千疮百孔；林中野兽皆带毒不可食）。
    • 地图真相 / 横穿：那张兽皮地图实为亚森曼祖先用鲜血与生命换来（非巨人所绘·传说阳之女曾用它讨好「巨人族」）。黑色森林面积不与吞噬岛挂钩——队伍只需走半圆形横穿半个森林、抵达对面出口即可离岛、不必返回（老巴尔语·与阳之女所说不同）。

    暗王 / 死亡之地最终阶段（八卷64–65章·隐藏任务第三阶段）¶

    石桥 / 堵桥（八卷53–57章）¶

    红土区四周悬崖、仅剩一条天然石桥（几十米长、十余米宽、无护栏）通向普通区域，是离开红土区的唯一咽喉。

    • 堵桥战术：一队临时结盟的契约者在石桥前堵了三天——专杀从红土区脱出、饿到虚脱、战力大降的契约者（普通区资源不算紧缺、堵桥稳赚）。他们还在飞机上发现了一个「秘密」、为此险些内讧（秘密内容待揭）。先前逃过苏晓的 血门冒险团（铁大师 / 老黑 等）正死在此堵桥队手中（铁大师跪地求饶仍被爆头扔崖）。
    • 苏晓反堵桥：苏晓远远用蜘蛛女皇先狙杀狙击手 + 弓箭手 + 法师，再冲上桥近战——以「环断」（圆形刀芒·一天限三次·超用手臂数天不能持刀）一招放倒一片、青钢影专破冰法师魔力回路、徒手抓住金色长矛控制技反甩回敌方、御力硬接战锤（脚下石板裂而身不伤）、双枪法力填弹连射；堵桥队大部被歼，仅两名穿「滑翔衣」的疑似兄弟跳崖逃脱。他留下重伤的阴郁头目 + 弓箭手 + 奶妈（逼奶妈治疗保命），审问堵桥的秘密——答案是「轮回印记」：生存试炼中死者的轮回印记可剥离（约 50% 成功率）、一枚 = 1% 世界之源或 1000 币（详见 机制）。苏晓从尸体上剥得 7 枚、掐死色诱无效的奶妈，顺走那辆装甲车（介于召唤物与物品之间·无所有权但试炼内可用·半箱油 + 备用油桶·夜晚最佳庇护所）当代步工具。

    鼠大人的空投箱（八卷58章）¶

    裁判 鼠大人 每天空投一个补给箱（蓝 / 紫等品质·带降落伞 + 信号弹·里头有恢复品 / 食物 / 淡水、运气好还有灵魂结晶 / 技能卷轴 / 强化药剂）——一落地就引来大批契约者抢夺、自相残杀（昨日蓝箱「死了多少人」）。这是又一处「大逃杀」推手，也是鼠大人的恶趣味：他还会把想逃离吞噬岛的契约者绑上铁块 + 信号弹当「假空投」扔下、引契约者疯抢。苏晓用装甲车顶接住一个紫色空投箱、再以界断线缠走（名场面「让人窒息的操作」），独吞紫箱：美味的烤火鸡（蓝）+ 灵魂结晶（中）×5 + 行军干粮 ×10 + 淡水 ×5 + 肾上腺素（紫）+ 战术皮靴（紫）。鼠大人见状起了戏心「明天空投加重、解除降落伞、看你怎么接」。

    血藤战士 / 亚森曼部落的秘密（八卷24–26章·待解）¶

    • 亚森曼部落的暴兵之道：为对抗虫族的暴兵海，红皮土著靠惊人生育力硬扛——女土著每胎产 2~5 个、16 岁生到 30~35 岁；一名战士之妻甚至产下十五胞胎；大本营 40 万 + 人口（幼儿居多·成年女几万·男丁稀少〔多去前线·断手脚才留部落〕·兄死则嫂由其他兄弟「继续用」）。

    • 八卷48章·门票（围杀卡洛 / 巴尔吸再生卵 / 虫妹入乐园）：日记把线索全串通——血藤战士皆出自后山圣地仪式（成功型 = 老巴尔、半成功型 = 巴里那种呆滞嗜血），「卡洛」实为吞噬了巴尔之子意识的契约者。苏晓举枪助巴尔围杀 卡洛（口风太松、泄密太多、留着是隐患）——卡洛仅撑三十秒被巴尔焚头，苏晓嫌其废话一脚踩碎其头（死后无猩红卡 / 宝箱·已非正常契约者）。巴尔随即捏碎再生卵化藤茧、吸收化解自爆危机；趁其脆弱，苏晓把特级炼金炸弹化粉尘植入巴尔体内反将一军，逼巴尔履约带他看仪式、并点破「原生之种 = 进黑色森林的『门票』」。母巢下层，虫族女王亚（虫妹）接受乐园招揽成为契约者「虫族唤醒者」、约定六天后随 艾达 一同离岛（届时母巢消失）。剩 154 小时·存活 342 人。
    • 八卷49–50章·奇怪的组合 / 惊悚（亚森曼真相 / 血门冒险团）：返部落途中巴尔倾吐部落根底——亚森曼人原是虚空种族、奉虚空王者之命驻守吞噬岛的「看守者」（已守几千年·文明退化）；几百年来一直是亚森曼主动攻打虫族夺再生卵；血藤战士非为对抗虫族、而是为应对「寒灾」。途中撞见 血门冒险团 的装甲车（老黑 等六人护送锻造大师 铁大师 晋升）——铁大师一眼识出斩龙闪是成长武器欲抢，探得苏晓「三维问号」后果断逃离（苏晓懒得追·留「希望之后能见面」）。回到部落，巴尔为一名断臂少年主持血藤仪式（麻醉、开膛、植生根原生之种 + 灵魂结晶〔中〕）。
    • 八卷51–52章·阳之女 / 谈判（隐藏任务第二阶段）：仪式意外唤出沉睡的 阳之女（血藤之母）——红土区地下巨藤本体、亚森曼世代供奉的「太阳化身」。苏晓踏入后山「太阳部落古圣地」即得世界之源 +3%。阳之女认出苏晓是她「发自骨子里讨厌」的灭法之影，却因有求于他而谈判：警告「黑色森林的不死者要出笼、寒灾五天后降临」（初始阶段就能致他死），求与他合作、保他安全进黑色森林。乐园提示「隐藏任务第一阶段『太阳部落』完成、激活第二阶段」。
    • 八卷52–53章·死亡之地 / 堵桥：隐藏任务第二阶段「死亡之地」（Lv.17·进黑色森林）激活——苏晓选择主动迎灾，与 巴尔 + 阳之女联合的另两名契约者组队，揣 阳之女 改造的 15 颗原生之种（每两小时一颗续命）出发。出洞后巴尔露真面目（「弑神不错、先度过寒灾再说」·被种了黑森林专用一次性能力、体温 >80 度即危险、过桥前不出手）。求生第五天赶到红土区唯一出口「石桥」，撞见一队契约者堵桥三天（血门冒险团 铁大师 / 老黑 等正死在其手）。
