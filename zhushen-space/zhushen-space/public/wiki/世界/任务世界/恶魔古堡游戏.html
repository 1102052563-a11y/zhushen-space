    恶魔古堡游戏（爱丽丝·强者之证明）¶

    苏晓 为 80 点壁障的最后一关「强者之证明」而进入的特殊封禁世界（第十四卷「恶魔古堡游戏·强者之证明」）。这不是普通衍生世界，而是 爱丽丝 用 爱丽丝的邀请函 召集契约者来玩的「恶魔游戏」——成功通关即由轮回乐园授予「强者之证明」。

    世界设定（十四卷1章）¶

    • 封禁世界：难度 Lv.1~?·世界之源 0%（无法获取）·绝对封禁、任何方法都无法离开·无主线任务、有 147 个支线任务（需激活）·无违规者。入场时苏晓被空间之力绑在铁柱上（约束期）。
    • 爱丽丝：恶魔古堡的女王、一切的主宰——不可对抗剧情人物，在古堡内力量近乎神灵。她无数岁月来反复接待游戏参与者、孜孜不倦地欣赏参与者的喜怒哀乐，并把自制宝物散落古堡各处当「答谢」。关键规则：不可讨好她、只有「公平的游戏」才能让她露出笑颜——一定要让爱丽丝微笑；若她愤怒或伤心，呵呵呵呵……（后果不堪）。
    • 狼管家：「公平的裁决者」（爱丽丝造物）·中立·不可对抗。
    • 古堡恶魔奴仆：古堡的原主人、被爱丽丝夺走三分之一大脑而失语·成了她的奴隶苦力·中立·不可对抗。

    第五场「虚空角斗场」（五层·十四卷45–50章·开始）¶

    第五场（最后一场）有两种：① 王者棋局（不在五层进行）；② 虚空角斗场（古堡五层别名「黄沙与鲜血」）——苏晓要打的是后者。这是 爱丽丝 的杀招阳谋：

    • 规则：虚空各族购票即可观战 / 参赛、本届实力上限≈女王分身（无下限·鱼龙混杂）、300 万观众、260 名参赛者、淘汰制（260→130→65→33→17）、前 20 名即通关第五场（可选返园或续战·前三名得虚空宝物）；几乎无规则（爆炸物 / 机甲都能用），死亡率 40~50%。主办方共三家（爱丽丝 / 恶魔古堡仅其一·故她无法在赛中作弊·只能靠正规手段）。
    • 爱丽丝的阳谋：她在第四场苏晓「钓鱼」时偷拍照片，印成传单「消逝千余年的灭法者再现」发遍角斗场（第 15487 届）——把苏晓的 灭法之影 身份公之于众，引虚空中灭法者的世仇（法师 / 老不死 / 被毁星球者）来围杀他。但灭法之影盟友也不少（虫族 / 恶魔族），新生代对灭法概念淡薄，真正危险的是与灭法同代的「老不死」（如来观战的法师贤者瑟菲莉娅）。
    • 轮回乐园的保险 · 鼠大人：苏晓、皮胖等尚未晋三阶、接触虚空过早，乐园派鼠大人（持临时轮回印记·二阶生存试炼的老熟人）坐镇——死于斗技乐园不管，但斗技之外的危险（如被观众围攻）乐园不坐视。鼠大人背后还有个「星球破坏者 / 虚空剑豪 / 酒疯子」级的恐怖存在（死其手者以亿计·据说消失在黑渊最深处·护短），故无人敢动他。
    • 领号牌：苏晓抽到 106 号；号牌三方势力图案（六菱形 / 王冠〔爱丽丝〕/ 圆形货币），唯独没有他最想见的「恶魔山羊头」（恶魔族盟友的标记）。解说是魔鬼族明星「哈克」（守序邪恶·奸商·爱抢镜·开场不吊胃口直接开赛反受热捧）。

    第一轮战斗（260→130·十四卷51–57章）¶

    • 苏晓的盟友 / 敌人：恶魔族魅魔外交官 莉莉姆 主动来传讯（警告他踏足虚空太早会死、远离「奥法永恒星」）；十几万恶魔族观众为「世代盟友」助威。而灭法世仇法师贤者 瑟菲莉娅 当场要用「灼热之痛」诅咒杀他、被 鼠大人 一枪（穿空间弹 + 源爆弹威胁）逼退。
    • 苏晓首轮观察的 6 名强敌：风 + 水融合的元素法师、家园被「刀魔」毁的疤痕壮汉、机械铠长臂男、恶魔混血（带空间属性·非友军）、独眼黑鳞甲生物、虫族 斑克（天生双臂刀·一直压制实力）。
    • 皮胖 · 武装色霸气坐实：第一场秒杀机械族奥勒——他脂肪能吸收 / 储存所受攻击的力量再反弹（「冲击」一拳把奥勒体内 90% 血液逼出当场毙命）、拳头可武装色硬化、燃脂瞬移；苏晓评其战力七人中仅次于自己。
    • 国足三兄弟 · 命运相连：三人命运相连（虚空视作一个个体·一死则余者厄难重重活不久·高于种族天赋），故只领一张号牌、三人同场。打法是「独行侠克星」——小型能量锤命中即定身（中武器 / 铠甲 2~3 秒、中身体 3~5 秒·无判定·能屏蔽感知）、大型能量锤命中高几率眩晕 0.3 秒；三人轮替（两人锤、一人歇着回体力）把维坦星战斗种族鳄鱼「古厄」按沙坑里锤了十分钟（还误来一记「铁锤砸蛋」），古厄战败屈辱落泪。
    • 苏晓 vs 51 号小巨人 · 「优雅的屠宰」：对手是黑渊最浅层流放区生存一年、为战而活的 5 米铠甲小巨人（持 3 米宽刃巨剑·力略强 / 速稍弱 / 战斗经验丰富·暗红气息）。苏晓凭新集齐的猎魔套（近战 +40% / 生命 +20%）+ 直感碾压——界断线缠脖 + 虚晃踢飞头颅（脑震荡）+ 青钢影斩破甲破防、灵魂感电麻痹 + 斩双腿肌腱 + 斩首；全程毫发无损，被观众称「优雅的屠宰」（赔率他 1 赔 0.2）。此战实测：猎魔套 + 直感令他闪避 / 攻击大涨、生命比主坦还高，可大胆涉险近战；套装「猎魔的战利品」攒到 1.4%。侦探小萝莉（可可豆）也在参赛、看完直呼祈祷别匹配到苏晓。

    间隙 · 迷火森林速成炼金（十四卷57–64章）¶

    第一轮后趁空档，恶魔族 莉莉姆 用「恶魔虫洞」把苏晓传到偏僻星球的迷火森林、见虚空顶尖炼金师 树贤者（恶魔族对苏晓的「试探」·苏晓不索宝物只求一个速成机会）。苏晓在那狂刷炼金渡鸦（10 分钟 / 只 → 32 秒 / 只·熟练度 1876/10 足够升 炼金生物制造学 Lv.20）、造近 900 只 + 一枚「阵图坐标接收器」。期间灭法世仇 瑟菲莉娅 追踪空间波动赶到、被树贤者的领地禁制挡退（她曾两次伏杀刀魔、被剥皮扔黑渊·「疯女人」）。

    第二轮（76→22·十四卷61–63章）¶

    第一轮 260 人实际只剩 76 人晋级（死伤 / 重伤无法续战）。苏晓第二轮对手 17 号·羽卢（圣翼族·双翼 + 羽毛裹生命提纯能量〔似念·提升羽毛穿透 / 锋利、侵蚀减速流血压制再生〕·飞行极快·欲拉距离打「千根落羽」消耗战）——苏晓直接用蜘蛛女皇狙击 + 近 900 只炼金渡鸦群 + 阵图坐标接收器把羽卢逼成「活靶子」（最高飞 500 米受能量罩限制），1 分 20 秒三连射爆头击杀。第二轮后仅剩 22 人。

    第五轮游戏完成 · 晋级前 20（十四卷64章）¶

    第三轮一开打就有选手同归于尽、场上人数破 20——轮回乐园提示「已晋升虚空角斗场前 20 名、恶魔古堡第五轮游戏完成」：苏晓只要返回古堡一层、由爱丽丝授予「游戏胜者徽章」即正式通关恶魔古堡游戏、随时可返园（强制回归剩 49 小时 59 分）。他可就此弃权返园，也可继续冲前三名（前三名是来自虚空的宝物·见下）。侦探小萝莉、国足三兄弟（这轮对手是斑克）、多名带伤者纷纷折断号牌弃权（第五轮一完成、再拼无谓），场上很快只剩约 10 人。
