    维斯星球·多亚古大陆（世界争夺战）¶

    苏晓的第 12 个衍生世界、也是他经历的二阶世界争夺战战场——一颗未知星系的星球「维斯星球」上的「多亚古大陆」（被划为世界争夺区域）。剑与魔法、种族与信仰、帝国与部落为争夺肥沃疆土永无止境地厮杀。本世界无法获得世界之源（争夺战世界的特殊规则·收益改走「乐园功勋系统」）。

    世界争夺战 = 轮回乐园 vs 天启乐园（十二卷1章）¶

    • 对手 = 天启乐园：与轮回乐园疑同出一源的另一个「乐园」。两个乐园争夺的「资源」惊人——一整个世界。世界越多、治下契约者变强的机会越多、契约者越强又能夺下更多世界，形成良性循环、乐园愈发浩瀚。这也部分揭开了「轮回乐园培养契约者变强」的目的之谜。
    • 虚空之树 公证：争夺战由「虚空之树」作绝对中立的公证方（初始法则第 1548762 条）——战争期间双方乐园均不可对治下穿梭者作任何资助（公平内战式争夺）。
    • 乐园功勋系统开启：杀敌方将领得 1~3 功勋、杀敌方契约者得 3~5 功勋；若战争胜利，按功勋值发奖。
    • 传送两站：星空中转站（无形屏障隔开双方·与苏晓同世界的约千名二阶契约者 + 天启乐园千余人）→ 维斯星球（30 分钟）。苏晓另获通用语 Lv.40（临时·战后遗忘·双方认证赋予）。

    • 敌我标识：开战后双方头顶各有光点（轮回乐园红 / 天启绿·隐身也不灭、死亡才消失）；杀敌越多光点越深，苏晓头顶已是暗红色。
    • 炎辰 vs 希：炎辰 魔枪克希、半小时压制并杀希一次，却惊见希原地复活（恢复 80%+），且杀她一次的代价是被其「死亡水晶化」溅中、晶体化垂死——遂下令退守三石桥。
    • 第一波战报：一轮混战后轮回乐园 658 存活 / 137 阵亡、天启 783 存活 / 202 阵亡——天启虽占人数 + 治疗优势却死伤更重。退守石桥后轮回乐园方压力骤减（只需守三桥 + 防飞行），天启驻守二区迷宫当大本营。炎辰估算「天启撑不住十天、每天打退一波则五天后我方人数反超、可考虑反扑」。

    千米水晶球全灭 + 大菠萝计划（十二卷74–77章）¶

    • 希的「超大型水晶球」杀招：希女王 从一开始就借古遗迹地形布局（兽人炮灰 / 迷宫死战 / 重伤炎辰皆为铺垫·只有炎辰垮了轮回乐园方才失主心骨）。她死 3 次、抽干自身 + 几十名法系契约者的法力，催出直径千米的实心水晶球把三石桥 + 黄昏宫殿连同数百契约者同化——一举灭 403 人（晶能 = 法力转化的无活性能量·稳定不暴走）。轮回乐园方仅剩 255 人 vs 天启 783 人（三倍差距）；希复活只剩 5 命、威望大减。
    • 剩下全是疯子：被灭的多是「精神还算正常、时刻警戒」的契约者，活下来的几乎全是疯子（大敌当前还在睡觉那种），反成最难缠的对手。一批「正常人」（独行侠 灰熊、幸运女神 艾达、虫妹、幻想乡团长娜莎等）不愿与疯子并肩、离开黄昏宫殿打游击。
    • 大菠萝计划：苏晓 与 炎辰（电疗化险后）退守黄昏宫殿、共谋用太阳神「阿波罗」全灭杀入宫殿之敌（苏晓的计划「更丧心病狂」被采用）。布局：放冒牌小丑割喉假死、黑乌鸦潜入天启队伍；空城诱敌逐层深入（一层空城 / 二层墙上写字 + 埋炼金炸弹炸死十几人 / 三层真空 / 四层是阿波罗的杀场），黄昏宫殿压制感知至 4 米。希逐层分兵深入，正中圈套。

    大菠萝引爆 · 希之死 · 大局已定（十二卷78–81章）¶

    • 太阳火密室全歼：黑血 带 16 人自杀小队攻一层、冒牌小丑 撕金色传送阵把 炎辰 一伙瞬移到四层，与 苏晓 把 希女王 和 300 天启困进四层密室。苏晓引爆预埋的第二颗太阳神「阿波罗」（密闭环境 500% 威力·6000 真实灼烧 + 2000 后续），气化希 + 307 天启契约者，四层化作 2000~3000 度的「生灵禁区」（岩浆 + 无氧 + 毒气 + 辐射）。
    • 逼死希：苏晓与炎辰要让希在火焰地狱里死满九次——她每次复活活不过 2 秒，终极「水晶源」冲五层也被苏晓的青钢影反击盾墙挡住、一脚踢飞，最终宁可引火自焚也不死在苏晓刀下，残躯被未知空间波动卷走。苏晓举报至 虚空之树、公证为「未知空间之力入侵」（非天启偏袒），裁定己方「大型空间定点坐标」形成时长砍半（再约四天即可夺下世界）。希殁（复活只有四次、已用尽）。
    • 大局已定：希一死，天启 300 余残兵斗志全无、向宫殿外溃逃（被几十疯子 + 岩浆追击、撞见一层呼呼大睡的黑血）。轮回乐园这边凭百余「疯子」+ 苏晓 / 炎辰镇守，天启已几无可能攻入——世界争夺战基本大局已定。苏晓阿波罗共进账 1084 乐园功勋。
    • 铭门重组：炎辰 借分赃 1089 功勋拉拢人心、收编百余疯子 + 冒牌小丑 + 黑血，把铭门冒险团改成「能者居之」的松散组织（吸取希「首领一死就散」的教训）。

    通关 · 坐标形成 + 最强 BOSS 25 秒灰飞烟灭（十二卷82章）¶

    • 铭门内斗 + 分赃：新铭门以「能者居之」选首领——炎辰 与 冒牌小丑、黑血 三人真刀真枪混战，半小时后炎辰胜出（小丑吐血、黑血被砍八段还拎汽油烧·铭门日常）。苏晓 与炎辰把阿波罗功勋谈成五五分（苏晓杀希 + 供阿波罗占大头·交易给炎辰 544 功勋·签临时契约四天互不干涉）。

    • 战争结果：胜利：【大型空间定点坐标已完成·归属轮回乐园·世界之核融合·本世界并入轮回乐园（用途：试炼 / 考核 / 突破阶位）·己方存活 247 人·全任务完成】。因已设坐标，契约者无需经星空中转站、直接传送回轮回乐园。
