# 轮回乐园·无限流 —— 新手游玩指南（保姆级）

> 这是一款"你打字、AI 当游戏主持人(GM)给你讲故事"的文字冒险游戏，背景是「轮回乐园 / 主神空间·无限流」。
> 你扮演一名"契约者"，被拉进各种世界（动漫、电影、原创世界…）做任务、变强、收集装备、结交 NPC。
> **没有标准答案，你想干嘛就打字说**，AI 会顺着演下去。

本指南把你当成完全没用过的新手，一步一步教。看不懂的地方按顺序照做就行。

## 1. 这游戏到底怎么玩

一句话：**你在输入框打字描述你想做的事，AI 生成一段剧情正文回你，如此往复。**

- 你说："我走进酒馆，找老板打听情报。" → AI 描写酒馆、老板、对话、可能的事件。
- 你说："我拔刀冲上去砍那个哥布林。" → AI 描写战斗过程和结果。
- 你说的越具体，AI 演得越准。也可以只写一句话让 AI 自由发挥。

游戏会**自动帮你记账**：你的属性、HP、背包物品、学的技能、认识的 NPC、势力关系、任务进度……这些都有专门的面板可以查看（右边那排按钮），而且大多能**自动跟着剧情更新**（这就是后面说的"演化系统"）。

> 💡 你不需要手动填那些数值，AI 看正文自己写。你只管打字推进剧情。

## 2. ⭐ 第一步：配置 API（不配，啥都动不了）

这游戏**自己不带 AI**，它需要连接一个"AI 大脑"（大语言模型）才能生成剧情。这个大脑要你自己提供——就是所谓的 **API**。

### 2.1 你需要准备 3 样东西

| 名称 | 是什么 | 举例 |
|---|---|---|
| **API 地址**（baseUrl） | AI 服务的网址 | `https://api.openai.com/v1`、或某个"中转站"给你的地址 |
| **API Key**（密钥） | 一串密码，证明是你在用 | `sk-xxxxxxxxxxxx` |
| **模型名**（model） | 用哪个 AI 模型 | `gpt-4o`、`claude-3-5-sonnet`、`deepseek-chat` 等 |

这三样从哪来？从你购买/注册的 AI 服务商那里拿。只要对方是 **"OpenAI 兼容"接口**（绝大多数都是），就能用。官方 OpenAI、各种第三方"中转站"、本地部署的模型都行。

> 🔒 **安全提示**：Key 只存在你自己的浏览器里，不会上传到别处。但别把 Key 截图发给别人。

### 2.2 怎么填（最简单的走法）

1. 在**封面页**点 **「系统设置」**（或游戏里右下角 **⚙ 设置**）。
2. 找到 **综合设置 → 「API 接口库」**。
3. 点**新增一条接口**，把上面 3 样填进去：
   - 地址（baseUrl）
   - Key
   - 模型名（有的接口可以点「拉取模型」自动列出可选模型，挑一个）
   - 给它起个名字，比如"我的主力模型"，并**启用**它。
4. 保存。

这样你就有了一个"接口"。

### 2.3 让所有功能都用这一个接口（新手强烈推荐）

游戏里很多功能（正文、物品、NPC、生图…）都能各自选 API。新手别折腾，**让它们全部共用你刚填的那个就行**：

- 大部分功能的 API 设置里都有「**共用 API / 单独配置**」开关，保持**共用**即可。
- 进阶玩法（可跳过）：每个功能旁边有个「**接口路由**」选择器，可以多选几个接口**轮流用 + 失败自动切换**。你接口多、怕某个挂掉时才需要。留空就用"共用"。

> ✅ **判断配好了没**：随便开一局，打一句话发送，如果几秒后出现一段剧情正文，就成了。如果报错（红字 / 401 / 超时），看第 12 节排查。

## 3. 开始一局：创建角色

封面点 **「开始游戏」** → 进入角色创建，一共几步：

1. **游戏难度**：决定你能分配的属性点。简单(50点) / 普通(40) / 困难(30) / 绝望(20) / 无用之人(10)。点越多开局越强。
2. **选择乐园**：你的"阵营/出身"，如轮回乐园 / 圣光 / 死亡 / 天启 / 守望，或自定义。
3. **基本信息**：姓名、年龄、性格、入园前职业（你穿越前是干嘛的）。
4. **六维属性**：力/敏/体/智/魅/幸，每项≤10，总和不超过难度给的点数。右边会实时显示剩余点数。
   - 体质(体)决定血量上限，智力(智)决定 EP(蓝条)上限，其它影响战斗和判定。
5. **天赋**：给自己一个开局天赋（名字 + 效果，自己编）。

确认后，游戏自动生成一段开场白发给 AI，你就进入故事了（通常先在主神空间/安全区适应一会儿）。

> 💾 **存模板**：创建页顶部有「存为模板 / 导入模板」，可以把这套开局设定存起来下次复用。和存档无关。

## 4. 主界面长啥样

进游戏后，屏幕大致分这几块：

- **正中间**：剧情正文（AI 生成的故事）。重要的结算（伤害、获得物品、时间推进等）会用**琥珀色小格子**单独框出来。
- **最底部**：你的**输入框**，打字 + 回车/发送，推进剧情。
- **左侧**：**主角栏**——你的姓名、阶位、等级、身份、六维、HP/EP 血蓝条、当前状态(Buff/Debuff)、外观、所属乐园等。**很多字段点一下就能手动编辑**。
- **顶部状态栏**：双时间（**轮回历**＝主神空间日历 + **当前世界时间**）、天气、当前世界名。
- **右侧**：一排功能按钮（下一节详解）。
- **正文两侧浮窗**：左上"主角装备"、右上"在场人物(NPC)"、右下"物品栏"——快速一览，点开有详情。

> ℹ️ **HP/EP 怎么算的**：HP 上限 = 体质×20，EP 上限 = 智力×15，跟着你的属性自动变。受伤/回血由剧情驱动；正文里写"当前HP：145/160"这种，游戏会自动同步到血条。

## 5. 怎么进入一个"任务世界"

开局你在主神空间（安全区）。要去做任务，需要"选择世界"：

1. 在正文下方找到 **「🌍 选择世界」** 按钮，点开。
2. 选/掷你的**目标阶位**和随机 ID（不手动掷会自动凑齐）。
3. AI 生成 **10 个候选世界卡**，每张写明：世界简介 / 巅峰战力 / 契约者分布 / 切入点 / 主线任务 / 支线 / 警告 / 奖励预览。
4. 用左右箭头翻看，挑一个点 **「进入此世界」**。
5. 游戏把该世界设为当前世界，AI 开始演这个世界的剧情。

任务做完/想回家时，按剧情回到主神空间即可（顶部世界名会变回轮回乐园）。

> 🌍 **领地、冒险团跟着你跨世界保留**（它们属于你个人）；**势力是绑定世界的**——换了世界，旧世界的势力会自动移出"当前世界"列表。

## 6. 右边那一排按钮都是干嘛的

点开都是一个个面板，**纯查看/管理用，不影响打字推进**。常用的：

| 按钮 | 作用 |
|---|---|
| ⚔ **装备** | 你的装备槽（武器/防具/饰品/特殊/技能槽）。点空槽从背包装上，点已装备的可卸下/编辑/锁定/删除。 |
| 🎒 **储存空间**（背包） | 所有物品 + 4 种货币（乐园币/灵魂钱币/技能点/黄金技能点）+ 进阶点数。可搜索/排序/筛选。 |
| ✨ **技能** | 主角的技能 + 天赋（评级 D→SSS）。点卡片看冷却/消耗/效果。 |
| 🛠 **副职业** | 非战斗手艺（机械师/药剂师…），含熟练度和配方。 |
| 🎖 **称号** | 你的称号库，最多佩戴 1 个。 |
| 🏆 **成就** | 解锁的成就（隐藏成就带🔒）。 |
| 📇 **NPC** | 你遇到的所有角色档案（身份/属性/关系/好感/装备/经历…），死掉的不显示。 |
| 🏛 **势力** | 世界里的组织/帮派/政府等，分"当前世界/非当前世界/已覆灭"。 |
| 🏯 **领地** | 你在主神空间的私人基地（等级/建筑/成员/仓库），跨世界保留、绝对安全。 |
| 🛡 **冒险团** | 你自己的冒险团（阶位 E→SSS、经验、活跃度、成员、考核）。 |
| 🔍 **回合洞察** | 每回合自动抓快照，对比"这回合 vs 上回合"有啥变化（好感、状态、势力动态…）。 |
| 📋 **任务** | 当前任务、世界大事。 |
| 🧠 **记忆** | 长期事实/小总结/大总结（剧情梗概）。 |
| 📡 **频道** | 公共频道（见第 8 节）。 |
| 💾 **存档** | 存档/读档/新游戏（见第 11 节）。 |
| ⚙ **设置** | 所有配置入口。 |

> 这些面板里的数据**大多是 AI 看剧情自动维护的**，你也可以手动编辑纠正。

## 7. "演化系统"是什么？要不要开？

这是本游戏的**核心特色**，但也是**最费 token（最烧 API 额度）**的部分，所以**大多默认关着**，你按需开。

**它是什么**：每次 AI 写完正文后，游戏可以**额外**再调用 AI，专门去"整理"某一类数据。比如：

| 演化 | 它帮你自动维护什么 | 默认 |
|---|---|---|
| **物品管理** | 获得/消耗/装备物品、货币、定价 | 关 |
| **主角演化** | 你的属性/技能/天赋/称号/身份/状态 | 关 |
| **NPC 演化** | NPC 的档案、属性、关系、登场/离场 | 关 |
| **势力演化** | 世界里各势力的动态 | 关 |
| **领地演化** | 你的私人基地建设 | 关 |
| **冒险团演化** | 你的冒险团成长/考核 | 关 |
| **杂项演化** | 时间、天气、任务、世界大事、剧情总结 | 关 |
| **生平压缩** | 把零碎记忆压缩成档案 | 关 |

**在哪开**：**设置 → 变量管理 → 对应的演化**（如 🧬 主角演化、🧑‍🤝‍🧑 NPC 演化…）。每个里面有：

- **启用开关**。
- **频率**：每几回合跑一次（设大一点省 token，比如每 2~3 回合）。
- **预设**：控制 AI 怎么整理的规则。**已经内置好了**，一般不用动；想换可以导入/导出。
- **API 设置**：默认共用你的主 API，也能单独配（比如用更便宜的模型跑演化）。

> 🧭 **新手建议**：先**只玩正文**，熟悉了再逐个打开你最在意的（比如想要物品系统就开"物品管理"，想要丰满的 NPC 就开"NPC 演化"）。全开最沉浸，但最费钱、也最慢。
>
> ⚠️ **NPC 演化**有"策略 A / 策略 B"：A 省 token（一次搞定），B 更细致（每个 NPC 单独跑，默认）。端点不稳老超时就调低并发或切策略 A。

## 8. 公共频道（买卖东西 + 发言吹水）

右侧 **📡 频道** 打开。这是"契约者公共广场"，单机模拟一群虚拟玩家在发帖。需要先在**设置 → 变量管理 → 📡 公共频道**里**启用**，并配好它的 API（默认共用正文 API）。

里面有几个频道（综合/交易/战斗/世界/情报/组队/系统），点 **🔄** 刷新会生成一批帖子。你能做：

- **买 NPC 的东西**：交易频道里别人挂的出售帖，点「购买」直接成交（自动扣钱给物）。
- **自己挂单**：点 **🛒 求购 / 🏷 出售**，发帖后会有"契约者"陆续来报价，你点「买下/卖出」成交。**成交后帖子自动删除**。
- **🏪 系统商店**：一个稳定的官方商店。
  - **买**：一次生成 20 件商品（消耗品/制式装备/技能书…，价偏高），勾选批量买。
  - **卖**：把背包里能卖的勾上 → 「询价」AI 给报价 → 「确认出售」。
- **发言吹水**：在非系统频道底部输入框打字发言 → 你的发言**立刻**贴出来 → 然后会有数量不等的"契约者"**陆续回复你**（嘲讽/认同/吐槽/玩梗…）。
  - 想**回复某个人**：点那条消息上的 **「↩ 回复」**，那个人会**先**回你，其他人再插嘴。频道有上下文记忆，对话能接得上。

## 9. 生图（给角色和装备画图）

**设置 → 🖼 生图设置**。支持 NAI / OpenAI 图片 / Gemini / ComfyUI / 自定义 这些出图服务（**和聊天 API 是分开的，要单独填出图服务的地址/Key**）。

三条独立的"出图线"，各自选服务、各有开关：

- **肖像**：给 NPC / 主角画立绘。开「自动生成肖像」后，会自动给没图的在场角色补图。
  - **外观变了会自动重绘**（「外观变化时刷新肖像」默认开）——你的外观描述一变，立绘跟着更新。
- **装备图**：给有外观描述、没图的装备补设定图。
- **正文配图**：在剧情正文里插图。

每个角色/装备的图也能**手动点 ✨ 生成 / ⬆ 上传 / ✕ 移除**。

> 💡 出图很慢也费量，**不强求**。想要沉浸感再开。NAI 这类对提示词敏感，游戏会自动把角色外观翻译成英文 tags（存"生图提示词/列19"）保证同一角色多次出图一致。

## 10. 叙事记忆（让 AI 记性好一点）

**设置主页 → 🧠 叙事记忆**。默认关。

问题背景：聊久了，早期剧情 AI 就"忘"了（上下文装不下）。这个功能帮 AI **回忆相关旧剧情**：

- **关键词召回**：根据你这句话，把相关的旧事实/总结找出来塞给 AI。
- **结构化档案召回**（默认随总开关生效）：把**主角 + 相关 NPC + 当前世界势力**的档案，自动塞进正文 AI，让它"看得到"你的属性、技能、NPC 关系等（否则正文 AI 其实读不到这些面板数据）。
- 可选的 **LLM 两步法**：用 AI 改写检索词 + 抽取长期事实，更准但更费。

> 🧭 想让 AI 更"记得住设定、对得上人物"，就把叙事记忆打开。它依赖"杂项演化/总结"积累的素材，所以最好配合**杂项演化**一起开。

## 11. 存档

右侧 **💾 存档**（封面也有「读取存档」）：

- **手动存档**：随时存一个槽，可改名/导出/导入/删除。
- **⏱ 自动存档**：每回合结束约 20 秒后自动覆盖存一份（只保留一个）。
- **🆕 新游戏**：清空进度（角色/NPC/物品/对话…），但**保留你的配置**（API/预设/世界书等），回到封面重开。
- **读档**会刷新页面来恢复，正常现象。

> 存档存在浏览器本地（IndexedDB）。**换浏览器/清缓存会丢**，重要存档记得「导出」备份成文件。

## 12. 卡住了 / 出问题了怎么办

**打字没反应 / 报错**
- 多半是 **API 没配好**。回第 2 节检查：地址结尾对不对（一般要带 `/v1`）、Key 有没有过期/欠费、模型名拼对没。
- 报 **401/403**：Key 错或没权限。报 **超时/CORS/524**：服务端太慢或不稳，换个模型/接口，或把演化频率调低、并发调小。

**HP/EP 一直是 100/50 不变**
- 老存档的兜底：正常推进一回合会自动按你的六维重算（体质×20 / 智力×15）。或在正文里让剧情明确写出当前 HP。

**用了消耗品却没消失 / 说"未找到物品"**
- 现在支持模糊匹配（"止血喷雾"能对上"次级止血喷雾"）。如果还不行，去背包确认物品全名，或在剧情里说清楚用的是哪件。

**势力换了世界还显示在"当前世界"**
- 一般会自动移出。没移的话，等势力演化下一次更新补上它的"所处世界"信息即可。

**界面元素显示不全 / 字挤在一起**
- 试试调整浏览器窗口大小或缩放（Ctrl + 加减号）。

**想彻底重来**
- 用「🆕 新游戏」（保留配置）。要连配置一起清，就清掉浏览器里 `drpg-` 开头的本地存储（开发者工具 → Application → Local Storage / IndexedDB）。

## 一句话总结给最懒的你

1. **设置里填好 API**（地址 + Key + 模型），其它功能全选"共用"。
2. **开始游戏**，捏个角色。
3. **打字**推进剧情，想去任务就点 **🌍 选择世界**。
4. 想要更丰富的系统（物品/NPC/势力/生图…），去 **设置 → 变量管理** 把对应"演化"**逐个打开**（费量，按需）。
5. 记得 **💾 存档** 并**导出备份**。

玩得开心。剩下的，交给你的想象力和 AI。
