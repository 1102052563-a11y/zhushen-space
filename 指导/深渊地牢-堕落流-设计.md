## 〇、四个已锁定决策（本设计的前提）

| 维度 | 选定方案 | 设计含义 |
|---|---|---|
| run 结构 | **大型多层地牢** | 不是 Slay-the-Spire 分叉小关；是逐层下沉、程序生成房间网络的真地牢 |
| 死亡惩罚 | **软撤退（死了也保留部分）** | 任意回溯阵主动撤退=全保留；死亡=按比例保留+meta 不丢 |
| 入口限制 | **只在主神空间开** | 右导航「深渊」仅主神空间可见可用，绑 `reconcileHomeWorld` 生命周期 |
| 腐蚀定位 | **可主动堆的「堕落流 build」** | 腐蚀是核心资源轴，不是纯负担；有堕落天赋树+失控/净化对冲+堕落形态 |

忠于原著：深渊=贯穿全书的暗黑核心势力，原罪物「**腐蚀持有者**」——腐蚀机制即由此而来（见 `指导/轮回乐园_势力与种族.md` 第七节）。

## 一、地牢结构：多层下沉 + 五险地 biome

### 1.1 整体形态
- 深渊是一条**不断下沉的多层地牢**：第 1 层 → 第 N 层，越深越凶、奖励越肥、腐蚀压力越大。
- **主线模式（固定封顶）**：五险地各约 3 层、共 **~15 层**，最深处「界之底」有终点 Boss = **通关**。
- **无尽模式**：通关界之底后**永久解锁**「无尽深渊」——无限下沉刷深度/分数（延后到 M4 做）。
- 每层 = **程序生成的房间网络**（房间节点 + 连接边 + 迷雾探索），由**种子**确定性生成（`abyssEngine`）。
- 每跨越若干层切换一个 **险地 biome**，biome 决定敌人/事件/掉落主题与腐蚀强度。

### 1.2 五险地 = 五个深度梯队（白嫖 canonical 设定）
设定文件 `指导/轮回乐园_势力与种族.md:118` 已列：

| 深度区 | 险地 | 主题 | 腐蚀强度 |
|---|---|---|---|
| 1 | **黑渊** | 入门污染区 | 低 |
| 2 | **古战场** | 亡灵 / 残兵败将 | 中低 |
| 3 | **渊龙底** | 精英兽潮 / 古龙残骸 | 中 |
| 4 | **死亡屋** | 陷阱 / 诅咒密集 | 高 |
| 5 | **界之底** | 界外存在 / 原罪物巢穴（终局区） | 极高 |

> 每个深度区跨多层；区末有「**区主 Boss**」（守关界外存在），击败后开放下一区入口（meta 永久解锁，下次可直达）。

### 1.3 房间类型
| 图标 | 房间 | 作用 |
|---|---|---|
| ⚔ | 战斗房 | 复用 `CombatPanel` 结算 |
| 💀 | 精英房 | 强敌，掉落更好 |
| 👑 | Boss房 | 层主 / 区主 |
| ❓ | 事件抉择房 | 走 `diceEngine` 判定（陷阱/机关/奇遇） |
| 🩸 | **堕落祭坛** | 主动献祭堆腐蚀换强度（**堕落流核心交互**） |
| 🛒 | 商店房 | 魂币消费 |
| 💎 | 宝藏 / 原罪物封印房 | 深层稀有，原罪物来源 |
| 🔥 | 休整房 | 回血 / **净化**（降腐蚀） |
| 🌀 | 回溯阵 | **撤退锚点**（软撤退入口） |
| ☠ | 污染走廊 | 通过即涨腐蚀（可绕路） |

## 二、软撤退 & 死亡保留（降硬核度）

### 2.1 主动撤退（全保留）
- 任意「🌀 回溯阵」房间可**主动撤退回主神空间**：本局**全部战利品保留**、腐蚀清零、run 结束。
- 这是 push-your-luck 的「见好就收」出口——和原著「契约者量力而退」一致。

### 2.2 死亡 = 深渊放逐（部分保留，非清零）
队伍全灭时：
1. **meta 永久（结晶 / 卡牌库 / 堕落星图 / 觉醒充能 / 解锁）+ 已得称号**：**全保留**，永不丢；
2. **本局未带出的战利品**（装备 / 原罪物 / 魂币 / 材料）：**按比例保留**（如 50%，`ABYSS_TUNING` 可调），其余丢失；
3. **meta 货币「堕落结晶」**：按本局**到达深度**结算发放（哪怕死了也有，仿强化「账号级垫子保底」）；
4. run 状态清空，可重开。

> 即「软 roguelike」：永久进度从不倒退，单局亏损有限——适合休闲长线。

## 三、腐蚀 = 堕落流核心（本设计灵魂）

腐蚀不是惩罚条，是**主玩法资源轴**。玩家可**主动堆腐蚀换强度**，并围绕它构筑 build。

### 3.1 腐蚀值 & 堕落等级
- **腐蚀值（corruption）**：本局内累积，来源——戴原罪物、踩污染走廊、堕落祭坛献祭、用黑暗系技能。
- 腐蚀值跨阈值 → 提升**堕落等级（0→5）**，每阶：
  - **+** 解锁/强化「堕落天赋」、整体增益（攻防%、暴击、吸血、原罪物威力）；
  - **−** 提高「失控概率」、负面随机事件权重。

### 3.2 堕落星图（**meta 永久层**，run 内构筑见 §八）
- 仿 `skillTreeStore`（径向星图）做一棵**独立的「堕落星图」**，但**定位为跨周目 meta 永久层**（用**堕落结晶**解锁，非 run 内主轴）。
- 作用：① 加权/解锁 run 内「战斗后加成卡」池；② 提供起手流派倾向；③ 解锁高级加成、起手卡组位、起始腐蚀缓和。
- run 内的随机构筑主要靠「**战斗后加成卡**」（§八）——**星图给方向，加成卡填内容，互不重复**。
- 节点风格：黑暗增益（嗜血/腐蚀爆发/亡灵召唤/痛苦反伤…）+「失控缓和」容错节点。

### 3.3 失控 / 净化对冲（build 张力）
- **失控**：高堕落等级下，每回合/每房间有概率触发反噬（随从叛变、属性反转、强制暴走、提前 Boss）。
- **净化**：🔥 休整房 / 道具 / 「净化流」天赋可降腐蚀——**稳健流**牺牲爆发换安全。
- 由此分化两类 build：
  - **堕落爆发流**：拉满腐蚀，吃高增益+高风险，追求速通深层；
  - **净化稳健流**：压低腐蚀，靠装备/技能慢推，容错高。

### 3.4 堕落形态（终极爆发，复用战斗重皮）
- 满堕落（等级 5）可发动「**堕落形态 / 魔化**」：一次 run 内的限时变身——大幅增益 + 战斗系统已有的 `蓄力大招/领域/锁血` 重皮（见战斗系统 Layer2）。
- 代价：形态结束后腐蚀飙升 / 强制失控判定 / 或推进一次「堕落结局」。
- 满堕落通关「界之底」→ 解锁**堕落专属原罪物 / 称号**（接现有称号系统）。

## 四、原罪物（双刃顶级奖励）

### 4.1 获取
- 仅 **💎 封印房 / 区主 Boss / 界之底** 掉落，稀有。
- 走现成 `createItem` helper 落进物品系统，加标记字段 `sin: true`。

### 4.2 双面设计（忠于「腐蚀持有者」）
- **正**：强力主动技 / 被动 / 六维加成（经 `effect` 入有效六维，战斗骰子已切有效六维）。
- **负**：持续诅咒——每回合涨腐蚀、随机负面、需「喂养」（献祭/击杀维持）。
- 内置首批用 canonical 物：**黑暗面具、死灵之书**（设定文件已点名）。

### 4.3 带出主线（原罪物=战利品，走白名单 §13.3）
- 原罪物是**战利品/装备**——撤退 100% / 死亡 50% 带出（§2.2），进主线背包，可正常装备生效。
- 带出后**诅咒下行转化为装备负面**（主线无腐蚀系统，诅咒以装备代价体现），保留"力量与代价并存"的原罪气质、兼顾平衡。
- 升级走**觉醒**（§10.2），非带出闸口。

### 4.4 联动
- 可与宝石镶嵌系统联动：镶「深渊宝石」强化原罪物（复用 `gemEngine`）。
- 原罪物图鉴（收集进度）→ meta 成就 / 称号。

### 4.5 随机生成（前端定数值 + API 配文）——副本"开箱"惊喜感
铁则：**AI 不碰数值**。两段式生成：
1. **前端确定性骨架**（`abyssEngine`）：品质(深度+腐蚀) → 类型/槽位 → 六维预算分配 → 从**效果池**抽 1 主动 + N 被动 → 从**诅咒池**抽诅咒 → 输出 `SinTemplate`（纯数值 + effect id）。
2. **API 配文**（`resolveApiChain('abyss-sin')`）：把骨架交 AI，生成 名称/背景/技能名+描述/诅咒文案/外观——**只润色文案，禁止改数值/改效果**（提示词 `ABYSS_SIN_GEN_RULE`，见 §13.2）。超时/失败回退纯前端模板名。
3. **组装**：前端把 AI 文案 + 自己的数值合成原罪物，`createItem` 入库（`sin:true`）。
- **原罪武器**：type 锁武器、效果池偏 攻击/暴击/领域 的专池（同机制）。
- 收益：数值可控可平衡（铁则），但每件名字/背景/技能都不同 → 开箱惊喜。

## 五、怪物设定

### 5.1 设计原则
- 全部 **深渊滋生 / 深渊生物 / 亡灵 / 界外存在** 主题（忠于原著第七节）。
- **六维机械生成**：怪物 tier = biome 基础阶位 + 深度偏移；前端用 `npcAttrGen` 机械反推六维（复用 bio-strength，治 AI 幻觉），**不让 AI 编数值**。
- **行为标签 → 战斗机制**：每个怪带标签，直接映射战斗系统 Layer2（DoT / 控制 / 护盾 / 召唤 / AoE / 锁血 / 领域），结算原样复用 `CombatPanel`。
- **腐蚀联动**：部分怪带「污染攻击」→ 命中给玩家加腐蚀；击杀深渊生物可掉少量**堕落结晶**（meta 货币）。

### 5.2 各险地怪物池（杂兵 / 精英）
| 险地 | 杂兵 | 精英 | 战斗标签 |
|---|---|---|---|
| 黑渊 | 渊蛆 / 腐蚀史莱姆 / 黑雾爬虫 / 污染残魂 | 污泥巨怪 / 堕落契约者 | 污染攻击(加腐蚀)、分裂、低阶 DoT |
| 古战场 | 枯骨战士 / 亡灵弓手 / 怨灵 | 无头骑士 / 亡灵将官 | 召唤(亡灵)、群攻、怨念 DoT |
| 渊龙底 | 渊龙幼体 / 龙骸傀儡 / 深渊兽群 | 极寒龙裔 / 狂暴渊兽 | 极寒控制(减速/冻结)、高防、蓄力 |
| 死亡屋 | 诅咒人偶 / 血肉缝合体 / 影魔 | 瘟疫使徒(黑暗神教) / 死亡屋管理者 | 诅咒 debuff、陷阱、护盾、阶段转换 |
| 界之底 | 界外触手 / 深渊祭司 / 原罪具现 | 卢恩家族战狂 / 先王眷属 | 领域、群体控制、锁血、召唤 |

> **阶位缩放**：黑渊≈低阶起，每深一层加偏移；界之底逼近「至强者」级（对应原著等级体系：一阶→至强者→绝强→无上→天选→至高）。`ABYSS_TUNING` 控基础阶位与每层偏移。

## 六、Boss 设定

两级 Boss：**层主**（深度区内的小高潮）+ **区主**（守关界外存在，击败解锁下一区直达）。

### 6.1 层主（floor boss）
- 每个深度区中段/末层出现一只，是 biome 精英的强化版。
- 一套核心机制（取自战斗 Layer2 中 1–2 个），中等难度，保底掉精良+装备。

### 6.2 五区主（zone boss，set-piece）
| 区 | 区主 | 原型 | 核心机制（映射战斗系统） | 特殊回报 |
|---|---|---|---|---|
| 黑渊 | **黑雾母体 / 腐渊之核** | 污染源头 | 腐蚀领域(每回合全场涨腐蚀)+召唤渊蛆 | 解锁古战场直达 |
| 古战场 | **枯骨统帅 / 无尽亡灵将军** | 亡灵族 | 召唤亡灵大军+自身锁血(军队未清不掉血)+怨念 DoT | 解锁渊龙底直达 |
| 渊龙底 | **极寒渊龙先祖** | 古龙残骸 | 高 HP/高防+极寒冻结控制+蓄力龙息 AoE | 解锁死亡屋直达 |
| 死亡屋 | **黑暗神教主教 / 死亡屋主** | 黑暗神教 | 诅咒流+血祭+多阶段变身(阶段转换) | 解锁界之底直达 + 高原罪物掉率 |
| 界之底 | **深渊先王（眷属）** | 界外存在 | 终局多阶段：领域+召唤+锁血+蓄力全开 | **通关** + 顶级原罪物保底 + 大量结晶 + 称号 |

> **卢恩家族战狂**（代表「毁灭」、至强巅峰、持始源族徽）作界之底的**精英前置 / 护法**出现，不单独当区主——契合原著「全族战斗狂」设定。

### 6.3 堕落镜像（隐藏强化 Boss，堕落流回报闭环）
- 进区主房时若**腐蚀值 / 堕落等级够高**，触发区主的「**堕落强化态 / 心魔镜像**」：更强，但掉率/品质大幅上调、可掉**堕落专属原罪物**。
- 把「主动堆腐蚀」的风险直接兑成更肥的 Boss 回报——堕落流的核心诱惑。

## 七、奖励设定

### 7.1 品质分级
普通 → 精良 → 史诗 → 传说 → **原罪**（5 级，颜色区分；品质 roll 由 **深度 + 堕落等级** 抬升）。

### 7.2 掉落来源矩阵
| 来源 | 主产出 | 稀有产出 |
|---|---|---|
| ⚔ 战斗房 | 魂币（+少量堕落结晶）+ **战斗后加成卡（§八）** | 低概率装备 / 材料 |
| 💀 精英房 | 装备 + 材料 + 宝石 | 较高品质、强化石 |
| 👑 层主 | 保底精良+ + 强化 / 打孔石 | 史诗装备、宝石 |
| 💎 宝藏房 | 装备 / 宝石 / 材料（按深度 roll） | 高品质堆叠 |
| 💎 封印房 | **原罪物**（或高保底装备） | — |
| 👑 区主 | 保底史诗+ + 大量堕落结晶 + 解锁直达 | 原罪物（堕落镜像必出 / 普通态概率） |
| 🏁 界之底终局 | **顶级原罪物保底** + 通关奖励 + 称号 | 堕落专属原罪物 |

### 7.3 货币 & 材料产出
- **魂币**：局内即时货币，战斗/事件/宝箱产出，🛒 商店、🩸 祭坛消费。
- **堕落结晶**（meta）：按 run **到达深度 + 击败区主数 + 极限模式倍率** 结算（死/退都给，仿强化垫子保底）。
- **强化材料**（强化石 / 保护石 / 打孔石 / 宝石）：直接喂**强化 & 镶嵌系统**——深渊成为这两个子系统的新产出口（跨系统经济联动）。

### 7.4 腐蚀 × 奖励 闭环（堕落流的回报）
- 腐蚀值 / 堕落等级越高 → **掉率↑、品质 roll 上限↑、堕落镜像 Boss 触发**。
- 即「越堕落、越肥、越险」——把 §三 的风险轴直接接到奖励轴，形成 push-your-luck 闭环。

### 7.5 进阶点（克制处理）
- 副本内**默认不发进阶点**（避免与主线击杀结算重复、破坏平衡）。
- 仅**击败区主 / 界之底**按现有 `advancePoints` 击杀结算给**少量**进阶点（可在 `ABYSS_TUNING` 关）。

## 八、Build 系统（战斗后加成 + 流派构筑）

run 的"乐趣核心"。让每局都不一样、有 build 成型感。

### 8.1 核心循环：战斗胜利 → 三选一加成
- 每场战斗（⚔战斗房 / 💀精英房 / 👑层主）胜利后弹「**深渊馈赠**」**3 选 1**；精英/Boss 升级为 **4 选 1 / 更高品质**。
- 加成卡 **API 生成**（§8.5，**一次 API 出 3 张**），按 当前险地/层深/堕落等级/已成型流派 **随阶段变化**；超时回退种子池。
- 玩家逐场累积 → run 内成型一套 build；死亡/撤退随 run 清空（但选过的卡进**卡牌库**永久收藏，§8.6）。

### 8.2 加成流派（synergy 是灵魂）
六大流派，**同流派攒够数量解锁「质变核心」（capstone）**——这是 build 成型的爽点，不是纯叠数值：

| 流派 | 风格 | 质变核心（举例） |
|---|---|---|
| 🩸 腐蚀/堕落 | 腐蚀转伤害、越堕落越强、献祭 | 满腐蚀=伤害翻倍但每回合自损 |
| ⚔ 武道 | 攻击/暴击/连击/破甲 | 暴击必触发追击 |
| 🛡 守护 | 护盾/减伤/反伤/回复 | 护盾溢出转化为攻击 |
| 💀 亡灵召唤 | 召唤物数量/强化 | 召唤上限+2，亡灵分担伤害 |
| 🔥 领域/法术 | AoE/领域/元素 | 进战自动展开腐蚀领域 |
| 🎲 赌徒 | 高风险高回报/随机爆发 | 伤害 ±50% 随机，暴击翻三倍 |

- 加成卡分品质（普通/精良/史诗），越深、越堕落越易出高品质（接 §7.4 闭环）。

### 8.3 三层 build 结构（消除与堕落星图的重叠）
| 层 | 是什么 | 范围 | 随机性 |
|---|---|---|---|
| 装备 / 原罪物 | 数值 + 独立词条/技能 | 掉落，撤退/死亡带出主线（§2.2） | 中（掉落随机 + §4.5 API 配文） |
| **战斗后加成卡** | **run 内随机流派构筑（本系统）** | 单局，撤退/死亡清空 | 高（3 选 1） |
| 堕落星图（§3.2） | **跨周目永久树**（meta）：加权加成池 / 起手流派 / 解锁高级加成 | 永久 | 低（玩家主动规划） |

> 三层互补：星图给**方向**，加成卡填**随机内容**，装备/原罪物给**数值底**。分工清晰、互不重复。

### 8.4 与腐蚀流的咬合
- 部分加成「**需腐蚀 ≥X 才生效 / 随腐蚀放大**」→ 主动堆腐蚀的玩家解锁更强加成与质变。
- **失控**（§3.3）可临时"污染"某条加成（反转/失效）——堕落爆发流的风险面。

### 8.5 加成卡 API 生成（随阶段变化，铁则仍 AI 不碰数值）
借鉴 StS「卡靠组合产生 synergy」+ Hades「boon 随进度演化」。两段式（同原罪物 §4.5 思路）：
- **前端给上下文 + 效果词表**：把 `{险地, 层深, 堕落等级, 当前卡组流派分布, 候选数=3}` + **固定「效果原语词表」**（如 `atk%` / `crit` / `addDoT` / `summonN` / `shieldOnHit` / `corruptionToDmg` / `lifesteal` / `aoe` / `domain`…）+ **强度档**（low/mid/high，**不是具体数字**）交给 AI。
- **API 生成**（`resolveApiChain('abyss-boon')`，**一次出 3 张**）：AI 负责 **挑 1–2 个效果原语 + 强度档 + 流派 + 取名 + 写描述**，主题契合当前险地（黑渊污浊→界之底界外），并参考已有卡组**给 synergy 或 反向多样**的选项。
- **前端定数值 + 落地**：前端按 `强度档 × 层深 × 品质` **确定性映射成真实数值**，组装成 `BoonCard`，效果由引擎实现。**AI 只产组合+档位+文案，绝不出真实数字**。
- 防稀释（StS 教训）：AI 必须从词表里选、不得自创效果；前端校验非法 effect → 丢弃重抽/回退种子池。

### 8.6 卡牌库 · 初始卡组 · 渐进出现（meta 收藏，重玩价值）
借鉴 StS 卡池 + Hades「记忆」。**通关后**把本局**选过的加成卡注册进「卡牌库」**（meta 永久，按 流派/品质/来源险地 **分类**，去重按 effect 签名）：
1. **作初始加成**：下次开局可从卡牌库**选 N 张作起手卡组**（N 由堕落星图/通关数解锁，仿 Hades 起手；防 OP→起手只能选低/中品质，史诗需更高解锁）。
2. **渐进出现**：已收藏的卡**按层深权重注入本局三选一池**——越深、越契合当前流派的收藏卡越容易刷出（"随本轮进度出现"）。
3. **去重/精炼**：同签名卡升「熟练度」（收藏×N 解锁该卡的强化变体）；卡牌库可看图鉴/收集度 → 称号。
- 注意：起手卡组 & 收藏注入都受**完全隔离**约束（§13.3）——只在副本沙盒内生效，不碰外部属性。

## 十、Meta 永久成长（跨周目）

- **险地直达**：通关区主后永久解锁，下次可从该险地起跳（省重复刷浅层）。
- **无尽模式解锁**：通关界之底后永久开放「无尽深渊」（见 §1.1）。
- **起始增益**：永久解锁的开局 buff（起始腐蚀缓和、初始道具、队伍上限）。
- **堕落星图永久层**（§3.2）+ **觉醒充能**（§10.2）+ **卡牌库 / 原罪物图鉴**。
- **最深记录 / 通关险地 / 极限模式通关** → 称号系统（极限单人通关给专属称号）。

### 10.1 开局修饰：极限模式（单人）
- 开一局时可勾选「**极限模式**」：**强制主角单人**（禁召唤/随从/NPC），全程更高难度。
- 回报：掉落/堕落结晶倍率上调（`ABYSS_TUNING`）、专属称号、独立最深记录榜。
- 与「可带队伍」并存——默认带队，极限是给硬核玩家的自选高难档。

### 10.2 觉醒系统（通关积累 → 升级原罪物 / 属性装备）
借鉴 Hades「meta 货币跨局升级」+ reforge「重铸词缀」：
- **积累**：每**通关一次**（到达界之底击杀终局 Boss）`clearsCount += 1`；**每满 3 次** → `awakenCharges += 1`（`ABYSS_TUNING` 可调 3→N）。
- **觉醒**：花 1 枚觉醒充能，选一件**你已带出主线的原罪物 / 属性装备**（§4.3）→ `resolveApiChain('abyss-awaken')` 升级：
  - **基础六维大幅跃升**（前端确定性大倍率，AI 不碰数值）；
  - **新增 1 条词缀 + 可能 1 个新效果**（AI 从效果词表选 + 配文，`ABYSS_AWAKEN_RULE`，§13.2）；
  - 记录觉醒阶数（`item.awakenLv += 1`），有上限防无限膨胀。
- 觉醒作用于**已带出的装备/原罪物**——属白名单产出（§13.3），是长线养成目标。

## 十二、前端确定性 vs AI 分工（铁则：AI 不碰数值）

- **前端确定性（数值/规则）**：地牢/房间生成（种子）、怪物六维、掉落表、腐蚀/堕落数值、失控判定、战斗与判定结算、死亡/撤退结算；**所有真实数值**——加成卡(强度档→数字)、原罪物骨架、觉醒倍率，一律前端定。
- **AI（生成/配文，不碰数值）**：
  - `resolveApiChain('abyss-boon')`：加成卡 **挑效果原语+强度档+流派+取名+描述**（随阶段变化，§8.5）。
  - `resolveApiChain('abyss-sin')`：随机原罪物 名称/背景/技能文案/外观（§4.5）。
  - `resolveApiChain('abyss-awaken')`：觉醒新词缀+效果+文案（§10.2）。
  - `resolveApiChain('abyss')`：房间/事件氛围、怪物命名、堕落剧情、Boss 旁白。
- **效果词表约束**：加成卡/觉醒的效果必须从前端固定「效果原语词表」里选，AI 不得自创；前端校验非法即丢弃重抽（防稀释/防越权）。
- 进阶可做「**深渊裁判剧情局**」（仿赌坊魂赌局）：AI 当裁判出剧情，前端兜底数值。

### 13.1 数据结构草案（示意）
```ts
// run 态（单局，撤退/死亡后清空；腐蚀每局从 0 起，§14.3）
interface AbyssRun {
  seed: string;
  mode: 'normal'|'endless'; // 主线封顶 / 无尽（无尽 M4）
  hardcore: boolean;        // 极限模式=强制主角单人（§10.1）
  zone: number;             // 1-5 险地区
  floor: number;            // 当前层
  map: AbyssFloor;          // 当前层房间图
  posRoomId: string;        // 所在房间
  corruption: number;       // 腐蚀值（每局独立）
  fallLevel: number;        // 堕落等级 0-5
  boons: BoonCard[];        // 本局已选加成卡 = run 内 build（§八）
  party: AbyssUnitState[];  // 主角(+召唤/随从/NPC，hardcore 时仅主角)；接战斗
  loot: AbyssLoot[];        // 本局未带出战利品（死亡保留 50%，§14.2）
}

interface AbyssFloor {
  rooms: AbyssRoom[];               // 节点
  edges: [string, string][];        // 连接
}
interface AbyssRoom {
  id: string;
  type: 'battle'|'elite'|'boss'|'event'|'altar'|'shop'|'treasure'|'rest'|'beacon'|'corridor';
  cleared: boolean; revealed: boolean;
  payload?: unknown;                // 敌人/事件/商品配置
}

// 怪物模板（内置数据 abyssDefaultData.json，六维交 npcAttrGen 机械生成）
interface AbyssMonster {
  id: string; name: string;
  biome: number;                    // 所属险地 1-5
  rank: 'mob'|'elite'|'floorBoss'|'zoneBoss';
  baseTier: number;                 // 基础阶位（+深度偏移）
  tags: string[];                   // 战斗机制标签：'dot'|'summon'|'aoe'|'shield'|'control'|'hpLock'|'domain'|'pollute'…
  lootTable: string;                // 引用掉落表 id
}

// 加成卡（API 生成组合+文案，前端定数值，§8.5）
interface BoonCard {
  id: string;                       // = effect 签名（去重/进卡牌库用）
  name: string; desc: string;       // AI 生成
  school: 'corruption'|'martial'|'guard'|'undead'|'domain'|'gambler';
  quality: 'common'|'fine'|'epic';
  prims: { id: BoonPrim; tier: 'low'|'mid'|'high' }[]; // AI 从效果词表选；前端按 tier×层深→真实数值
  value: number[];                  // 前端确定性映射出的真实数值（与 prims 对应）
  needCorruption?: number;          // 需腐蚀≥X 才生效/放大（§8.4）
  capstone?: boolean;               // 是否流派质变核心
}
type BoonPrim = 'atk%'|'crit'|'addDoT'|'summonN'|'shieldOnHit'|'corruptionToDmg'|'lifesteal'|'aoe'|'domain'|'…';

// 原罪物数值骨架（前端 roll，再交 AI 配文 §4.5 / §13.2）
interface SinTemplate {
  quality: 'common'|'fine'|'epic'|'legendary'|'sin';
  type: string;                     // 槽位/类型，如 '武器/单手剑'；原罪武器锁武器
  biome: number;
  stats: Record<string, number>;    // 六维加成（前端定）
  active: { id: string; tag: string };       // 主动技能（效果由 id 实现，tag 给 AI 语义）
  passive: { id: string; tag: string }[];    // 被动
  curse: { id: string; tag: string };        // 诅咒（腐蚀持有者）
}

// 卡牌库条目（meta 收藏，§8.6）
interface CardLibEntry {
  card: BoonCard;                   // 卡定义（按 effect 签名去重）
  school: BoonCard['school'];       // 分类
  count: number;                    // 收藏次数→熟练度
  fromBiome: number;                // 首次获得险地
}

// meta 永久态（跨周目，从不倒退；唯一跨出沙盒边界的存档，§13.3）
interface AbyssMeta {
  crystals: number;                 // 堕落结晶
  unlockedZones: number;            // 已解锁直达险地
  starmapNodes: string[];           // 永久点亮的堕落星图节点（§3.2，meta）
  cardLibrary: CardLibEntry[];      // 卡牌库（§8.6，初始卡组来源 + 渐进注入池）
  startDeck: string[];              // 已选作起手的卡 id（开局注入 run.boons）
  // 原罪物/装备直接入主线背包；觉醒阶数记在 item.awakenLv，不在 meta 列
  sinCodex: Record<string, boolean>;// 原罪物图鉴
  deepestFloor: number;
  clearsCount: number;              // 通关累计（§10.2）
  awakenCharges: number;            // 每 3 通关 +1，可花于觉醒
  endlessUnlocked: boolean;         // 通关界之底后开放无尽模式
  hardcoreClears: number;           // 极限模式通关次数（专属称号）
}
```

### 13.2 AI 指令 / 提示词常量
- `promptRules.ts`：`ABYSS_BOON_GEN_RULE`（加成卡，下附）/ **`ABYSS_SIN_GEN_RULE`**（随机原罪物，下附）/ `ABYSS_AWAKEN_RULE`（觉醒，下附）/ `ABYSS_ROOM_FLAVOR_RULE` / `ABYSS_MONSTER_FLAVOR_RULE` / `ABYSS_FALL_NARRATIVE_RULE` / `ABYSS_BOSS_INTRO_RULE`。
- 接口键：`abyss-boon`（一次出 3 张卡）/ `abyss-sin`（原罪物配文）/ `abyss-awaken`（觉醒）/ `abyss`（剧情配文）。
- 三者**统一范式**：前端把【上下文 + 效果词表 + 强度档】拼在常量后发出 → AI 回 JSON（只选效果原语 + 档位 + 文案）→ 前端校验词表合法性 + 按档位确定性算真实数值 → 落地（加成卡入 run/卡牌库；原罪物/觉醒走 `createItem`/装备升级）。非法 effect 即丢弃重抽/回退种子池。

#### `ABYSS_SIN_GEN_RULE`（随机原罪物 / 原罪武器 配文）
```text
你是「轮回乐园·深渊」原罪物的命名与文案生成器。系统会给你一件原罪物的【机械骨架】（已由前端确定：品质 / 类型 / 六维加成 / 主动技能 / 被动 / 诅咒 / 所属险地）。你只负责为它生成暗黑而契合的【文案外壳】。

【世界观锚定】
- 原罪物是深渊的造物，本质会「腐蚀持有者」——力量越强，反噬越烈。
- 风格：阴郁、古老、危险、带宿命感，参考黑暗面具、死灵之书等原罪级物品的气质。
- 用轮回乐园术语（阶位 / 契约者 / 虚空 / 界外存在 / 深渊滋生），禁用修仙、境界、灵根、修为等词。

【硬性铁则】
1. 绝对不得修改、新增、删除或编造任何数值。六维、技能效果、冷却、诅咒强度全部以骨架为准；文案中不得出现与骨架矛盾的数字或效果。
2. 你只产出"名称 + 文案 + 外观"。每段机制描述必须严格对应骨架里的 active / passive / curse 的 tag 语义，只做文学化润色，不得改变其作用，也不要写具体数字。
3. 严格输出下面的 JSON，不要任何额外解释，不要 markdown 代码块包裹。
4. name ≤ 6 字、有原罪/深渊气质；curseDesc 必须点明"代价"，与 curse.tag 一致。
5. type 标明武器时，文案需体现"武器"形态（原罪武器）。

【输入骨架示例】
{ "quality":"传说","type":"武器/单手剑","biome":"界之底",
  "stats":{"力":120,"敏":40},
  "active":{"id":"abyssal_rend","tag":"范围斩击+流血"},
  "passive":[{"id":"corruption_power","tag":"腐蚀越高攻越高"}],
  "curse":{"id":"hp_drain","tag":"每回合自损并增加腐蚀"} }

【输出 JSON 格式（仅输出此对象）】
{
  "name": "原罪物名称",
  "title": "一句话定性，如：界之底的噬主之刃",
  "lore": "2-4 句背景，点明来历与腐蚀本性",
  "activeName": "主动技能名",
  "activeDesc": "主动技能的文学化描述（对应 active.tag，不写数字）",
  "passiveDesc": ["逐条对应 passive[] 的描述"],
  "curseDesc": "诅咒描述，必须说明持有代价（对应 curse.tag）",
  "appearance": "外观描述（供生图，一句话）"
}
```

}, {...}, {...}]

#### `ABYSS_AWAKEN_RULE`（觉醒：给原罪物/装备加新词缀+效果）

你是「轮回乐园·深渊」的觉醒仪式文案与效果生成器。契约者用觉醒充能升级一件原罪物/装备。基础六维的大幅提升由前端确定，你只负责【新增 1 条词缀（+可能 1 个新效果）】与升级文案。

### 13.3 完全隔离（沙盒铁则）🔒
副本是**纯沙盒**，与主线存档**强隔离**——你点名的"卡片加成不改外部属性"就是这条：
- **进副本**：`abyssEngine.initRun()` 把主角六维/装备/技能 **快照拷贝**进 `AbyssRun.party`（`AbyssUnitState`）。run 内一切**临时加成**——加成卡、腐蚀、堕落等级与形态——**只改这份沙盒拷贝**，战斗也跑在沙盒上（`CombatPanel` 本就吃独立战斗态）。
- **绝不触碰**：`playerStore` 的真实六维/HP/EP/装备/技能、其它 `drpg-*` 进度，run 内**只读不写**。
- **白名单（可带出）**：撤退/死亡/通关的**结算函数**才写回——**装备 / 原罪物 / 奖励（货币·材料）**进主线背包、**称号**进称号系统、**结晶 / 卡牌库 / 星图 / 觉醒充能 / 解锁**进 `AbyssMeta`。除此之外没有写回路径。
- **隔离（带不出）**：加成卡 build、腐蚀/堕落等级、堕落形态、堕落星图的 run 内增益——这些纯临时战力**只在沙盒生效，结算即蒸发**。装备穿上当然影响主线六维（那是装备本职），但**加成类绝不外泄**。
- **存档**：只持久化 `AbyssMeta`（跨周目）+ 进行中的 `AbyssRun`（断点续）；二者都在 `drpg-abyss` 内，纳入 saveManager。

| # | 决策 | 结论 |
|---|---|---|
| 1 | **地牢规模** | **固定通关封顶为主线**（五险地各约 3 层、共 ~15 层，有「通关界之底」终点）；**通关后解锁「无尽深渊」刷分模式**（双模式，无尽延后到 M4） |

| 3 | **腐蚀清零时机** | **每局独立，撤退即清零**——不做跨局堕落 meta 层，单局闭环 |
| 4 | **堕落星图定位** | 独立一棵星图，但**改为 meta 永久层**（堕落结晶解锁，加权加成池/起手流派）；run 内构筑改由「战斗后加成卡」担当（§八） |

| 7 | **Build 系统** | **每场战斗胜利后 3 选 1 加成卡**（精英/Boss 升级 4 选 1）；六大流派 + 同流派质变核心（§八） |
| 8 | **随机原罪物** | **前端 roll 数值骨架 + API 配文**（`ABYSS_SIN_GEN_RULE`，铁则禁改数值）；原罪武器=锁武器的专池（§4.5） |
| 9 | **三层 build 分工** | 装备/原罪物（数值底）+ 加成卡（run 内随机构筑）+ 堕落星图（meta 方向），互不重复（§8.3） |
| 10 | **加成卡=API 生成** | （翻案 #7 旧"不调 API"）一次 API 出 3 张，**随阶段变化**；AI 只挑「效果原语词表+强度档+流派+文案」，**前端定真实数值**，非法 effect 丢弃重抽（`ABYSS_BOON_GEN_RULE`，§8.5） |

| 12 | **觉醒系统** | 每**通关 1 次**积 1 值，**每 3 次**觉醒 1 次：API 给（已带出的）原罪物/装备**加新词缀+效果 + 前端大幅提升基础数值**（`ABYSS_AWAKEN_RULE`，§10.2） |
| 13 | **完全隔离🔒** | 隔离的是**临时加成**（加成卡/腐蚀/堕落形态）——只在沙盒生效、结算即蒸发；**白名单可带出**=装备(含原罪物/觉醒后)/奖励/称号/meta 解锁（§13.3） |
