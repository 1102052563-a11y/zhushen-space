# 新手游玩攻略 · 装备强化 / 竞技场 / 战斗系统

> 写给**完全没玩过、第一次接触**的新人。从零讲起，照着做就能上手。
> 三套系统互相关联：**强化**让装备变强 →**战斗**靠装备和属性分胜负 →**竞技场**就是用战斗去打榜拿奖励。建议按本文顺序读。

---

## 开始之前：先认几个名词

这游戏是「轮回乐园 / 无限流」题材（你是被拉进主神空间的契约者，要去各个任务世界闯关）。下面这几个词在三套系统里反复出现，先混个眼熟：

| 名词 | 一句话解释 |
|---|---|
| **轮回乐园 / 主神空间 / 专属房间** | 你的「老家」「休息站」，也叫**枢纽**。打完一个任务世界会回到这里。**强化和竞技场只能在这里用**，进了任务世界就用不了。 |
| **任务世界** | 出任务去的副本世界。在里面没法强化、没法进竞技场（但可以正常战斗）。 |
| **阶位** | 你的等级段位，从低到高：**D → C → B → A → S → SS → SSS**（也常说一阶、二阶……）。越高越强。 |
| **乐园币 🪙 / 灵魂钱币 💎** | 两种钱。**乐园币**是日常货币（强化全程花它）；**灵魂钱币**是稀有货币。两者可以互换（见下）。 |
| **装备品质** | 装备稀有度，由低到高：白 → 绿 → 蓝 → 紫 → 淡金 → 金 → 暗金 → 永恒 → 起源 → 创世。品质越高越贵、越强。 |
| **六维属性** | 力 / 敏 / 体 / 智 / 魅 / 幸 六个数值。战斗里的攻击、防御、血量都由它们换算出来。 |
| **HP / EP** | HP=生命值（归零就倒下）；EP=能量/法力（放技能要花）。 |
| **技能 / 天赋** | 技能=主动放的招（攻击、治疗、控制等）；天赋=被动加成。战斗和竞技场都会用到。 |
| **在场 NPC** | 当前剧情里「出现在你面前」的角色。发起战斗时只能选在场的人当对手或队友。 |

> 💱 **两种钱怎么换？**
> 打开右侧导航的**「背包 / 储存空间」**，在货币栏最下面有个**兑换条**：
> **1 灵魂钱币 = 150,000 乐园币**，可双向兑换。
> 强化烧的是乐园币，缺钱时可以拿灵魂钱币换一批出来。

# 一、装备强化系统 ⚒

## 1.1 一句话理解

花**乐园币**给装备「打 +号」（+1、+2……最高 **+16**），等级越高装备越强，**但越往后越容易失败，失败可能掉级、清零、甚至把装备直接打碎消失**。本质是一个**搏运气的养成小游戏**（玩过 DNF 强化、点炮的话就是那个味）。

## 1.2 在哪打开

- 右侧导航栏点**「⚒ 强化」**。
- ⚠️ **只有在轮回乐园 / 专属房间里**才能用。在任务世界里这个入口是**灰的**，点开也会提示「仅乐园内可用」。
- 打开后是一个叫**「装备强化所」**的大面板，顶部会显示「**乐园 · 营业中**」（绿字）才说明能用。

## 1.3 界面长什么样

面板分上下两块：

**上半部分 —— 强化师（看板娘 / 老板）立绘**
- 中间一张人物大图，就是帮你强化的「老板」。
- 左右两个箭头 **‹ ›** 可以**切换不同老板**（每个老板有不同性格和加成参数）。
- **点立绘**，老板会冒出一句话（吐槽、鼓励或怂恿你继续氪），纯氛围、不影响数值。

**下半部分左 —— 正在强化的装备**
- 大图 + 右上角的 **+N**（当前强化等级，等级越高角标越华丽）。
- 强化时这里会放特效：成功发光、暴击紫光、失败变暗、归零、碎裂。
- 下方有红字警告，提醒你当前处于「归零区」还是「分解区」。

**下半部分右 —— 操作区（最常用，重点看这里）**
- **选择装备**：列出你背包/身上所有能强化的装备，点一件选中。
- **强化等级**：显示 `+N → +N+1`。
- **成功率**：这次强化成功的概率。
- **强化保护石**（勾选框）：防止失败掉装备，**只有 +7 及以上才能勾**。
- **强化符**（勾选框）：直接**+12% 成功率**，任何等级都能用。
- **本次花费 / 乐园币余额**：这次要花多少、你还剩多少。
- **本轮记录**：这一轮强化的成功/降级/归零/爆装次数。
- 最下方两个大按钮：
  - **「⚒ 强化 · XXX 🪙」**：点一次，强化一次。
  - **「✓ 结束强化」**：强化出成果后才出现（见 1.6）。

顶部右上角还有个**「垫子计数 N / 10」**，这是保底机制，后面 1.5 讲。

## 1.4 怎么玩（操作步骤）

1. 回到**乐园**，点右侧**「⚒ 强化」**。
2. 在右侧「选择装备」里**点一件你要练的装备**（中间会显示出来）。
3. 看一眼**成功率**和**本次花费**，决定要不要上**强化符**（+12%）。
4. 到了 **+7 以上**，强烈建议勾上**保护石**（防止辛苦攒的等级清零/装备爆掉）。
5. 点**「⚒ 强化」**。出结果：成功就 +1，失败按下面规则处理。
6. 重复点强化，直到满意或不敢再赌。
7. 升上去了就点**「✓ 结束强化」**收尾（见 1.6），让装备真正刷新属性。

## 1.5 核心规则（一定要懂）

### 成功率 & 失败后果一览表

| 当前 | 强化到 | 成功率 | **失败会怎样** |
|:--:|:--:|:--:|---|
| +0 | +1 | **100%** | 必成功，不会失败 |
| +1 | +2 | **100%** | 必成功 |
| +2 | +3 | **100%** | 必成功 |
| +3 | +4 | 95% | 掉 1 级（回 +2） |
| +4 | +5 | 90% | 掉 1 级 |
| +5 | +6 | 80% | 掉 1 级 |
| +6 | +7 | 75% | 掉 1 级 |
| +7 | +8 | 62% | ⚠️ **归零**（直接回 +0）|
| +8 | +9 | 54% | ⚠️ **归零** |
| +9 | +10 | 41% | ⚠️ **归零** |
| +10 | +11 | 34% | 💥 **装备分解（永久消失）** |
| +11 | +12 | 28% | 💥 **分解消失** |
| +12 | +13 | 21% | 💥 **分解消失** |
| +13 | +14 | 17% | 💥 **分解消失** |
| +14 | +15 | 14% | 💥 **分解消失** |
| +15 | +16 | 10% | 💥 **分解消失** |

**记住三个分界线：**
- **+0 ~ +2**：白送，必成功，放心点。
- **+3 ~ +6（降级区）**：失败只掉 1 级，损失小，可以放心冲。
- **+7 ~ +9（归零区）**：失败**直接回 +0**，前功尽弃。**这里开始上保护石。**
- **+10 ~ +16（分解区）**：失败**装备直接消失**。**没保护石别碰，除非你在故意爆装刷保底（见下）。**

### 两个辅助道具（现买现用，不占背包）

- **强化符**：勾一下就 **+12% 成功率**，按等级收费，越高越贵。冲高分险时建议常备。
- **保护石**：**只有 +7 起能勾**。勾上后，这一次强化**就算失败也不会归零/不会爆装**，只是停在原地（白花钱但东西还在）。冲 +10 以上几乎必带。

### 保底机制：「垫子计数」

- 顶部那个 **N / 10** 就是保底。
- **只有装备真的被「分解爆掉」时（+10 以上失败），计数才 +1**。掉级、归零都**不算**。
- **攒满 10**，下一次强化就是**「★ 保底 · 必成」**（100% 成功），用掉后清零。
- 高玩玩法：拿**便宜的垃圾装**故意冲到分解区，让它爆掉攒计数，攒满 10 之后再去强化你真正想要的装备，保证那一发必成。

### 关于花费

强化费用 = 看**等级**（越高越贵）×**品质**（越稀有越贵）×**评分**。所以练一件创世级装备会比白板装贵得多。缺乐园币就用灵魂钱币换。

## 1.6 「✓ 结束强化」是干嘛的（别忘了点！）

- 你强化升级后，操作区底部会冒出**「✓ 结束强化」**按钮。
- 点它，会让 AI 给这件装备**真正刷新属性**：每升 4 级追加一条词缀/特效，重算攻防，外观描述也越来越华丽。
- **不点也没关系**：当你**关掉面板**或**切换到另一件装备**时，会自动触发同样的收尾。
- 简单说：强化只是把 +号 打上去，**「结束强化」才把强化的好处兑现到装备面板上**。

## 1.7 新手建议 / 避坑

- 🟢 **先无脑冲到 +6**：这段几乎没风险，失败最多掉一级。
- 🛡️ **+7 起永远带保护石**：归零一次心态就崩了。
- 🎯 **冲 +10 以上前，先用垃圾装把保底攒到 10**，再用保底那一发去推你的主力装。
- 💰 **别一上来就练顶级装**：费用是按品质涨的，新手先拿便宜装练手、熟悉机制。
- 🧪 **强化是纯前端算的，不花 AI 额度**（不耗 token），只有「老板说话」和「结束强化刷装备」会调 AI。可以放心多点。

# 二、竞技场 🏟

## 2.1 一句话理解

一个**打排行榜**的玩法：系统按你的阶位生成一份高手榜单，你**挑战榜上的人**，打赢就**抢占他的名次**，名次越靠前奖励越好。挑战的过程**就是一场战斗**（具体怎么打看第三章）。

## 2.2 在哪打开

- 右侧导航点**「🏟 竞技场」**。
- ⚠️ 同样**只能在乐园里用**，任务世界中是锁的（会提示「需在乐园内才能使用」）。
- 如果显示「竞技场已停用」，是设置里被关了：去**设置 → 变量管理 → 🏟竞技场**重新打开。

## 2.3 什么阶位能玩

- **一阶（D）就能玩**普通的「**本阶竞技场**」，只和同阶位的人排名。新手开局就能打。
- **五阶（S）起**才解锁更高级的「**强者争霸战**」。
- **七阶（SSS）起**会变成「**树之竞技场**」，前列有特殊称号 badge。
- 看不懂没关系：**能点开、卡片不是灰锁的，就是你能打的**。带 🔒 的卡片是还没到阶位、暂时打不了的。

## 2.4 界面长什么样

- **顶部**：标题 + 右上角**「🏆 击败记录」**（看你打赢过谁）。
- **上方一排卡片**：不同的竞技场分支，点一张选中（灰锁的进不去）。
- **工具条**：
  - **我的名次 #** —— 你目前排第几。
  - **最佳 #** —— 你打到过的最高名次。
  - **🔥连胜 N** —— 连胜会加成奖励。
  - **🔄 刷新** —— 重新生成榜单。
  - **名次输入框 + 跳转** —— 想打第几名，直接输数字跳过去（比如输 50，就给你看第 50 名附近的人）。
- **下方榜单**：一行一个对手，显示 `#名次 名字 · 阶位 · 职业 · 强度`。**你自己那行会高亮**并标「（我）」。每个对手右边有红色**「挑战」**按钮。

## 2.5 怎么玩（操作步骤）

1. 回到**乐园**，点**「🏟 竞技场」**。
2. 选一个**竞技场卡片**（一阶就选「本阶竞技场」）。
3. 榜单会自动加载。**找一个你觉得打得过的对手**（一般挑比你名次高一点点的）。
4. 点他那行的**「挑战」**。
5. 弹出确认框，告诉你「打赢后名次取代为 #X，前 100 名有奖励」。点**「挑战」**确认。
6. **进入战斗**（自动建好对手的装备/技能/天赋）—— 接下来就是第三章的战斗流程。
7. **打赢** → 你的名次直接变成对手的名次，发奖励；**打输** → 名次不变，可以再来。

## 2.6 核心规则

- **取代名次**：赢了就**直接顶替**被你打败那个人的排名（比如打赢 #30，你就变成 #30）。所以**一路往上挑战名次更高的人**就能爬榜。
- **挑战 = 一场真战斗**：对手会被自动配齐**至少 6 件装备、至少 4 个技能 / 天赋**，是个完整对手，不是软柿子。打不打得过取决于你的属性和装备（所以**先强化、先练技能**很重要）。
- **奖励（只有前 100 名有）**：
  - 进前 100 名：发**物品 + 货币**，连胜还有加成。
  - **第 1 名**：额外给一个**唯一称号**。
  - **100 名开外**：只给**进阶点数**，没有物质奖励。
- **击败记录**：打赢过的人会进「🏆 击败记录」，能回看对手简介、当时拿的奖励。
- **榜单有记忆**：生成过的榜单会缓存下来，下次进来还是那批人（前 50 名资格榜尤其固定）。

## 2.7 新手建议 / 避坑

- 🎯 **挑战比自己高一点点的人**，别一上来挑 #1，循序渐进往上爬。
- ⚔️ **打不过先回去变强**：竞技场赢面 = 战斗赢面，去**强化装备、升技能/天赋、提属性**再来。
- 🔥 **能连胜就连着打**：连胜有奖励加成。
- 💡 **没把握就先看榜不挑战**：单纯刷新看榜、跳转看高名次是免费的，挑战才会进战斗。

# 三、战斗系统 ⚔️

## 3.1 一句话理解

一套**回合制战斗**（像很多 RPG 那样轮流出手）。你和队友一边、敌人一边，轮流行动，靠**普攻、技能、防御**把对面血量打空就赢。**胜负是前端按属性 + 骰子确定性计算的**（公平、可复现），**没网 / 没配 AI 也能打**。竞技场挑战、剧情里想打架，都走这套。

## 3.2 怎么发起战斗

1. **先让角色登场**：战斗只能选「在场 NPC」，所以你得先在**正文剧情里**让对手 / 队友出现在场上。
2. 右侧导航点**「⚔️ 战斗」**，打开「**发起战斗**」面板。
3. 面板列出所有**在场 NPC**，给每个人点**「敌方」**或**「我方」**（不点就是不参战）。**主角自动算我方**，不用选。
4. 至少要选 **1 个敌方**，底部**「开始战斗」**才能点。
5. 点「开始战斗」→ 进入战斗面板。

> 竞技场的「挑战」会**自动**帮你建好对手并开战，不用走上面这步。

## 3.3 战斗界面长什么样

- **顶部**：第几回合 + 地点；右上角 AI 在算时会显示「思考中…」。
- **中间双方阵容**：左边**我方**（青色），右边**敌方**（红色）。每张**角色卡**显示：
  - 名字、阶位 / 强度；
  - **HP 条**（绿，生命）、**EP 条**（蓝，能量）；
  - **护盾**（有的话，金色，先于血量被打掉）；
  - **状态胶囊**（中毒、眩晕、增益等小图标）。
  - **轮到谁出手，谁的卡会有金色高亮框**。
- **领域/阵法条**：如果有人放了领域，这里显示当前领域和剩余回合。
- **战斗日志**：中间滚动的文字，叙述每回合发生了什么。
- **底部行动区**：轮到你时，在这里选这一回合怎么打。

## 3.4 轮到你时怎么操作

轮到你（或你操控的队友）时，底部会出现行动按钮：

| 按钮 | 作用 |
|---|---|
| **普攻** | 普通攻击。**要先点一个敌方角色卡选目标**，再点「出手」。 |
| **技能** | 弹出下拉选一个技能。技能可能是攻击、治疗、控制、增益等。 |
| **防御** | 这一回合**承受伤害减半**（🛡 标记），适合扛一下。 |
| **逃跑** | 尝试脱离战斗。 |
| **出手** | 选好后点它，确认这一回合的行动。 |

**关于「技能」下拉里的标记：**
- `🔋蓄力` —— 蓄力大招，见下。
- `🌀领域` —— 领域/阵法技，见下。
- `⏳冷却N` —— 还在冷却，这 N 回合不能用（灰的）。
- **治疗技** → 点**我方**角色当目标；**攻击技** → 点**敌方**。
- **群体技 / 自身增益 / 领域** → **不用选目标**，自动生效。

**特殊状态下的按钮：**
- 被**眩晕/控制**时：只能点「跳过」，这回合不能动。
- 正在**蓄力**时：出现「继续蓄力 / 🔥 释放大招 / 中断」。

## 3.5 进阶机制（看懂会打得更聪明）

这些都靠技能**名字/描述里的关键词**自动触发，你正常放技能就行：

- **持续效果**：技能带「中毒 / 灼烧 / 流血」→ 每回合掉血（DoT）；「再生」→ 每回合回血（HoT）。
- **控制**：带「眩晕 / 定身」→ 让对方一回合不能动。
- **增益 / 减益**：「强化 / 战意」→ 自己攻击变高；「破甲」→ 对方防御变低；「虚弱」→ 对方攻击变低。
- **护盾**：「护盾 / 结界」→ 给自己一层盾，先挡伤害。
- **群体（AoE）**：带「群体 / 全体 / 范围」→ 一下打到敌方所有人。
- **🔋 蓄力大招**：技能带「蓄力 / 充能 / 吟唱」等词。用了之后会**连续几回合灌注能量、暂时不出伤害**，蓄满再**一次性放出超高伤害**（蓄越久倍率越高）。注意：蓄力时**被控制会溃散、法力不够会中断**。
- **🌀 领域 / 阵法**：技能带「领域 / 结界 / 法阵 / 大阵」等词。放出后**笼罩全场，之后每回合持续生效**（治疗 / 护盾 / 增益 / 压制 / 持续伤害，看技能性质），持续 3~4 回合，**主人倒下领域就消失**。
- **不死 / 锁血**：技能带「不死 / 金身不灭 / 无敌 / 绝对防御」等词 → 短时间内**扣血保底留 1 滴血**或**完全不掉血**，保命用。

## 3.6 伤害大概怎么算（不用记公式，懂个意思）

- 伤害 ≈ **你的攻击力 − 对方防御的一部分**，再乘上**这一击命中的「骰子成功倍率」**。
- 出手时会掷骰子定成败：**大成功**伤害翻倍、**普通成功**正常、**失败**没伤害、**大失败**还会反伤自己。
- **物理攻击**比的是物攻 vs 物防，**法术攻击**比的是法攻 vs 法防。
- **防御姿态**承伤减半；**护盾**会先于血量被消耗。
- 结论：**攻防差距 + 技能运用 + 一点运气**决定输赢。所以**装备强化、技能天赋、六维属性**练得越好，赢面越大。

## 3.7 战斗结束后

- 一方全倒下 → 顶部显示**「🎉 战斗胜利」**或**「💀 战斗失败」**。
- 胜利会发**进阶点数 + 世界之源**等奖励。
- 点**「关闭」**结束面板。
- 战斗的**总结会自动填进你的输入框**，你**确认 / 编辑后手动发送**，剧情就会接着这场战斗的结果往下写。

## 3.8 新手建议 / 避坑

- 🧍 **先让人登场再发起战斗**：面板说「当前没有在场的 NPC」就是还没人出场，回正文让角色出现。
- 🎯 **普攻/单体技记得选目标**：没选目标「出手」是灰的。
- 🛡️ **扛不住就「防御」**：承伤减半，给治疗 / 蓄力争取时间。
- 💧 **留意 EP**：技能要花 EP，EP 空了就只能普攻，别一上来全放大的。
- 🔋 **蓄力期很脆**：蓄力时不出伤、还可能被打断，最好有队友帮你扛住再放。
- 🌐 **离线也能打**：没配 AI 也能正常战斗（系统会用确定性结算兜底），放心试。
- 🧪 想纯练手，可以去**设置 → 变量管理 → 战斗设置**里的**「🧪 测试战斗」**直接开一场。

## 三套系统怎么串起来玩（总结）

```
        强化装备 ⚒              练技能/天赋/属性
            │                        │
            └──────────┬─────────────┘
                       ▼
                  战斗变强 ⚔️
                       │
                       ▼
              去竞技场打榜 🏟  → 赢 → 抢名次 + 拿奖励
                       │
                       ▼
              奖励/资源 → 回头继续强化、变更强（循环）
```

- **想变强** → 回乐园**强化装备**、练技能天赋、堆属性。
- **检验强不强** → 去**竞技场挑战**，挑战就是打**战斗**。
- **赢了拿奖励** → 换资源继续强化，名次越爬越高。

> 📌 再强调两条最容易忘的：
> 1. **强化和竞技场只能在乐园（老家）里用**，任务世界中是锁的。
> 2. **强化到 +7 以上一定带保护石**；**冲 +10 以上先用垃圾装把保底攒满 10**。

祝你玩得开心，强化十连不爆装 🎉
