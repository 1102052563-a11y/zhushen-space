# 尸祸一六四四

## 剧情

**【作品来源】**
《尸祸一六四四》（又名《大明、丧尸与中二病》）作者**稚嫩小菠萝**，起点中文网历史·架空历史连载，作品ID **1048675924**。标签：明朝、争霸、权谋、皇帝流；公开统计约**56万字**量级、连载中（章节推进至约198章级，随更新浮动）。气质：明末丧尸灾变 + 南明假太子案史实骨架 + 穿越者中二史观喜剧/黑色幽默；**非纯正史复刻**，尸变与蒸汽机妄想并列为双重异质。**连载中，不编造终局。**

**【世界定位】**
崇祯十五年起，流星落开封，死人食生人；顺军怖而决马家口灌城，祸水横流。两年后时序进入崇祯十七年兵荒：14岁初二学生**王之明**穿越，发现自己竟是逃离山海关后的皇太子**朱慈烺**（化名王之明），由鸿胪寺少卿**高梦箕**家仆**穆虎**陪同南下寻路。并行穿越者**方枝儿**（现代咨询顾问/明史大V气质）以侍女身份贴身观察。一句话：**大明末世 + 尸祸 + 假太子南下 + 中二太子要「从《永乐大典》找蒸汽机挫败文官集团」的荒诞史诗。**

**【世界观 · 力量体系】**
本世界战力分层必须同时写清「凡人军政」与「尸变」两条轴，禁止用修仙境界偷换。

1. **凡人军政轴**  
   - 流民/纤夫/漕卒/乡勇：持木器铁器，遇尸群与乱兵皆高死亡率。  
   - 生员/保正/牙人/驿站系统：社会秩序残片，能调动的是人情与文书，不是法术。  
   - 家丁部曲、绿营/边军残部、南明诸镇（如章题出现的**黄得功**等）：建制武力，远高于个人。  
   - 藩王/太子名器、高官印信：政治权柄，能征兵筹饷也能招来追杀。  
   死亡规则硬：无复活；冻死、咬伤、决堤、仇杀均可永久退场。

2. **尸变轴（公开简介与开篇可核）**  
   - 崇祯十五年开封流星后，出现「死人食生人」；顺军因怖而决堤，尸祸与水祸叠加。  
   - 开篇邳州码头：面色青黑、流涎、屡倒屡起的「流丐」撕咬生员——尸变个体速度与执念高于常人饿殍，但尚非无敌。  
   - 传染/咬伤规则以正文展开为准；档案写法：咬伤=高危不可轻视，禁止写成无症状的「轻微挂彩」。  
   - 尸群规模可从单体→巷战→城破级，但低阶切入不得直接「清图开封尸潮」。

3. **信息与科技妄想轴**  
   - 朱慈烺的「P社/全战史观」与《永乐大典》蒸汽机执念：是角色动机，不是现成工业外挂。  
   - 方枝儿的明史预知：优势在路线与案件（假太子案），不是开箱现代军火。  
   - 真正的「战力放大」来自：名器（太子身份）、漕路与江南市场、满饷逻辑、以及能否在尸祸下重建秩序。

乐园阶位映射（宁低勿高）：流民/侍女/驿路行人≈**一阶**；家丁精锐/乡勇头目/能单对单处理低阶尸个体≈**二阶**；部曲将领/一城守备/小规模清尸与护送≈**三阶**；南明一镇级（黄得功等公开点名级）/区域军政≈**四阶**；清军主力、顺军大股、开封级尸潮与决堤灾难＝**超阶阴影**（情报优先/条件性胜利，严禁战力归零）。

**【地理 · 舞台】**
- **开封（前史）**：流星、尸变源头意象、决马家口灌城——世界创伤原点，低阶不宜直接开局于此屠城战场。  
- **邳州赵村驿港口（开篇）**：京杭运河节点；米麦竹木布帛磁器市集与冻死尸骨、卖儿卖女并存；夜航沙船上下层舱室分明。  
- **运河夜航船**：官舱与密舱；穆虎、王公子、方枝儿同船；生员与「流丐尸变」冲突的第一现场。  
- **南下杭州线**：高梦箕「表侄」名义；假太子案史实走廊；牙行、驿站、关卡。  
- **南京/南明政治阴影**：名器与文官集团、定策与真假太子博弈的下游舞台。  
- **北方**：清军入关与满饷叙事的压力源；太子「北望」与中二宣言的对照。  
写场景先定：是否在运河秩序残片内、最近尸变目击点距离、船上舱等与名帖是否匹配。

**【世界剧情线】**
① **崇祯十五年·开封流星与尸食**  
流星落开封，死人食生人；顺军怖而决堤，水祸扩散尸祸。此为世界底层灾变，不是可跳过的背景板。

② **崇祯十七年·太子南下（开篇已核）**  
朱慈烺以**王之明**化名，在**梅金英**（梅大伴）与**穆虎**护持下走运河；他坚信「满清入关是大明文官集团阴谋」，要满饷、要江南市场、要《永乐大典》蒸汽机线索。开篇码头童子、饼子、冻尸与市集同框，奠定「热闹与末世叠影」。

③ **方枝儿线并行**  
现代人方枝儿穿越为侍女，识别「穆虎 + 王公子 + 高梦箕」=南明假太子案链条，计划先保命随行至杭州，再图福建/郑家等统战价值——她与太子的史观冲突是长期喜剧与权谋双响。

④ **第一口尸咬（开篇名场面）**  
生员喂食反被青黑流丐撕咬，屡倒屡起；保正乡勇介入。朱慈烺尚未完全理解尸祸语法，方枝儿则以现代危机感压指甲——两人信息差即戏剧核。

⑤ **名器与私刻宝印**  
太子要角弓、长刀、邸报塘报与太子宝印；匠人不敢私刻。名器线=政治炸弹，比杀几个流丐更危险。

⑥ **南下博弈（连载推进层）**  
运河—杭州—南明诸镇；章题级冲突含大辩、非友军的黄得功、大水冲了明王庙等（以连载章节为锚，不编未核终局）。尸祸、水祸、藩镇、清军、文官集团多线绞杀。

⑦ **连载边界**  
截至公开：连载中，约198章级「大水冲了明王庙」等；**不宣布谁最终称帝、尸祸是否根除、蒸汽机是否落地**。契约者任务停在已公开可验证节点。

**【主要人物】**
- **朱慈烺 / 王之明（穿越太子）**｜性格：中二、狂热、史观魔怔、对「文官集团」极端敌视，同时又有饼子施童的市井热络；装备：暗龙纹黄素绫裤等名器痕迹、角弓长刀意向；弧光：假名南下→名器与尸祸双重压力下的成长/崩坏（连载中）；立场：自居挽天之人，对梅金英依赖、对穆虎既用且防。  
- **方枝儿（侍女·并行穿越者）**｜性格：理性算计、学术绿茶姿态、先保命后统战；能力：明史预知与危机评估，非武力；弧光：假太子案旁观者→可能的关键变量；立场：可随行可甩团，对太子既当机会也当危险。  
- **穆虎（高梦箕家人/管事）**｜性格：实务护送、官场下人逻辑；立场：完成主家差遣，真假太子问题是政治雷。  
- **梅金英（梅大伴）**｜性格：贴身照料、谨慎；职能：斗篷、办货、拦太子胡闹。  
- **高梦箕（鸿胪寺少卿·幕后名义）**｜政治链条上的「表侄」掩护来源。  
- **开篇生员 / 保正 / 乡勇 / 牙人**｜尸变与社会秩序的第一层NPC。  
- **黄得功等南明武将（章题级）**｜四阶阴影，非友即敌需验证。  
- **顺军 / 清军 / 文官集团（集合意象）**｜超阶压力源，低阶只闻其名。

**【贵重物品】**

- 《永乐大典》线索（蒸汽机执念的文本圣杯）。  
- 邸报、塘报、角弓、长刀。  
- 夜航船票与舱位（活下去的物流权）。

- 开封流星/决堤的「灾变记忆」本身（解释世界为何如此）。

**【隐藏剧情 · 伏笔】**
- 开封流星与尸变的真正机制（连载未完全摊牌则保持开放）。  
- 假太子案史实与「真太子穿越」叠加后的政治爆炸半径。  
- 方枝儿与朱慈烺史观对撞：谁才是更大的危险。  
- 黄得功等「非我友军」式藩镇信任危机。  
- 蒸汽机线是喜剧噱头还是工业奇点——以正文为准，禁止提前封神。  
- **不编终局**。

**【大事记时间线】**
崇祯十五年：开封流星，尸食，决堤。  
崇祯十七年：朱慈烺化名南下；邳州码头开篇；方枝儿并行穿越。  
运河夜航：第一尸咬目击；采办兵器邸报；宝印受阻。  
南下杭州/南明政治：假太子案走廊；诸镇博弈。  
连载中段：大辩、黄得功线、大水与明王庙等章题冲突。  
**当前**：连载未完，尸祸与国运双线仍在烧。

**【叙事基调 · 雷区】**
基调：末世残酷 + 中二喜剧 + 权谋黑色幽默；口吻可市井可奏疏，忌纯小白无脑。  
忌①把尸变写成无传染的打怪副本；忌②一阶斩黄得功/清军主力；忌③忽略假太子案政治雷；忌④把方枝儿写成无脑后宫；忌⑤编造完结称帝与尸祸永灭；忌⑥用灵气境界解释火力。  
最早切入锚点：**邳州码头夜航船前后、第一口尸咬刚发生、太子宝印尚未办成时。**

**【可观察细则·运河与尸】**
写战斗先写舱门宽度、火把数量、被咬者是否还能说话；写权谋先写名帖与舱等是否匹配。满饷逻辑在本作是口号也是后勤学：没有粮台与市场，中二宣言只是河风。

**【可观察细则·双穿越信息差】**
朱慈烺看见「流丐好汉」，方枝儿看见「异常感染者」；同一场面两种语法，契约者站哪边解释世界，会改变 NPC 信任与任务结算。

**【尸祸一六四四·开局生存解剖学】**
在《尸祸一六四四》里，稚嫩小菠萝的第一夜不是升级教程，而是顺军怖而决马家口灌城的温差、气味与可数资源：还剩几顿吃食、几件铁器、几张能用的名帖或船票。契约者若开场就谈「称王/证道」，会与本作节奏脱节。正确写法：先让稚嫩小菠萝（或切入身份）完成一次可观察的小胜——弄到干粮、避开一次撕咬/鞭打/搜查、换到一个能睡的角落——并留下仇或恩的记账对象（优先点名稚嫩小菠萝或1048675924）。

**【尸祸一六四四·名场面可复述因果】**
公开可核的关键因果应能被复述为「谁在何地因何失去什么」：开局立足导致第一次质变，再逼出中期站队。写卡时禁止用「然后就变强了」跳步。每一次质变旁边必须站着具体人：稚嫩小菠萝、1048675924、56万字，以及具体物：蒸汽、大典。

**【尸祸一六四四·地理使用手册】**
顺军怖而决马家口灌城不是背景板，是规则：谁控制进出口，谁就能收过路钱与情报税。发现自己竟是逃离山海关决定后勤半径；由鸿胪寺决定政治风险。场景描写至少回答：此刻离补给多远、离顶点阴影多远、离稚嫩小菠萝的短期目标多远。

**【尸祸一六四四·力量展示边界】**
个人武勇再高，也扛不住建制与灾变的双重碾压。展示战力时优先写：编制是否完整、粮台是否在、信息是否对称。稚嫩小菠萝的「强」应体现为组织与决断，而不是突然学会越阶大招。触及非纯正史复刻时，胜利条件改为带回情报/人质/文书。

**【尸祸一六四四·对话与口吻】**
《尸祸一六四四》对白应带本作特有的制度词与市井词，忌修仙腔、系统提示音、跨书黑话。稚嫩小菠萝说话要符合公开人设；1048675924的反应要能被旁观者听懂利害。一句好对白=暴露立场+推动交易/冲突。

**【尸祸一六四四·任务三件套】**
可挂任务必须同时具备：可见目标（护送蒸汽 / 守住发现自己竟是逃离山海关 / 说服1048675924）、可数代价（伤、钱、名、时间）、可观察结算（谁还活着、账本如何变）。禁止「击杀X只怪」式空壳。

**【尸祸一六四四·失败与残局】**
失败不是读档，而是残局：丢蒸汽、被56万字记名、在顺军怖而决马家口灌城失去落脚点。残局支线往往比正线更贴本作气质——善后、赔偿、谣言、第二次更贵的入场券。

**【尸祸一六四四·时间表驱动】**
用日程推进：考期、军运、潮汛、市集、疫与灾的窗口。开局立足错过就走残局，不假倒流（除非原作明确允许）。写「等三天」必须写出这三天顺军怖而决马家口灌城会变什么。

**【尸祸一六四四·人性锚与犹豫帧】**
当1048675924或亲友锚被威胁，稚嫩小菠萝应出现至少一帧犹豫：不是圣母，而是账本与情感的冲突。没有犹豫帧，本作会滑成纯工具人打怪。

**【尸祸一六四四·顶点阴影写法】**

**【尸祸一六四四·物资与硬通货】**
本作硬通货优先是：蒸汽、大典、名帖、宝印、角弓、长刀、邸报、塘报、龙纹、船票。它们比「经验值」更能决定下一章能不能开。交易场面要写清成色、数量、抵押与毁约后果。

**【尸祸一六四四·势力交易筹码】**
每个势力出场先写要什么、怕什么、与稚嫩小菠萝的可交易筹码。稚嫩小菠萝侧与56万字的筹码应不同：一个要编制，一个要面子，一个要粮。冲突落到具体地点顺军怖而决马家口灌城/发现自己竟是逃离山海关。

**【尸祸一六四四·情报真伪】**
邸报、塘报、谣言、口供在《尸祸一六四四》里可以杀人。契约者获得的情报必须标注来源可信度；用假情报换取的胜利会在下一阶段被追债。

**【尸祸一六四四·身体与伤病】**
伤病是剧情：冻伤、咬伤、鞭伤、弹片、疫病。医疗资源稀缺时，救谁不救谁就是站队。禁止无代价满血复活（除非原作体系明确允许且写清代价）。

**【尸祸一六四四·身份与文书】**
籍、帖、印、票、合同：纸面权柄常比刀锋决定进场资格。稚嫩小菠萝的身份变化应反映在文书与称呼上，而不是只改称号特效。

**【尸祸一六四四·群像站位】**
同场至少三人有不同目标：稚嫩小菠萝、1048675924、56万字。路人不是经验包，是镜子与谣言源。

**【尸祸一六四四·环境杀】**
水、火、寒、疫、挤踏、断粮可以杀死比刀更多的人。写群戏时优先环境杀，再写个人武勇，更贴乱世/灾变气质。

**【尸祸一六四四·喜剧与残酷的叠影】**（若本作有喜剧核则启用，否则作对照）
《尸祸一六四四》若存在中二宣言/反差萌，必须与尸骨、鞭痕、空饷同框，避免把残酷滤成轻飘段子。笑点过后要有账单。

## 阶位切入点

> 阶位↔：流民/侍女≈一；家丁/乡勇头≈二；部曲将/一城守≈三；南明一镇≈四；清军主力/开封级尸潮＝超阶。超阶＝情报优先／条件性胜利；严禁顶点战力归零。

**一阶（尸祸一六四四 · 驿路与夜航）**
切入身份/时点：纤夫、舱役、牙行跑腿、新收侍女；邳州码头至夜航船，第一尸咬前后。
初始事件：青黑流丐扑咬生员，保正要人作证；梅金英在找走散的「王公子」；方枝儿压着指甲不让自己失态。
开场白建议：「河风里有饼香也有尸臭。有人喊王家小哥，有人喊捉拿疯丐。你手里的号衣还没暖热，船桅已经在喊客。」
关键NPC立场：**朱慈烺（王之明）**——兴奋地旁观并随时开麦史观；**方枝儿**——冷静到发冷；**穆虎**——要秩序；**梅金英**——要小官人上船。
主线钩子/支线：活着登船/下船；支线＝救生员或抢邸报、替牙行送舱单。
危险度/规避：中——单体尸与冻饿；避大股乱兵。
任务方向/奖励：船票、干粮、一件铁器、一句「下次还能上这船」的信用。

**二阶（尸祸一六四四 · 家丁与清舱）**
切入身份/时点：穆虎侧家丁、保正临时雇手、船上司事；运河多港停靠段。
初始事件：舱内疑似二次撕咬，要隔离还是抛江；太子要私刻宝印，匠人连夜逃走。
开场白建议：「隔舱板后面有东西在撞。穆管事只问你一句：你是护印的，还是护命的？」
关键NPC立场：**穆虎**要差事完成；**朱慈烺**要印与弓刀；**方枝儿**要情报；地方保正要息事。
主线钩子/支线：清舱/护送采办；支线＝截下假邸报。
危险度/规避：中高——小股尸与仇家刀。
任务方向/奖励：家丁身份、刀弓、半真半假的塘报抄本。

**三阶（尸祸一六四四 · 部曲与名器）**
切入身份/时点：小股部曲头目、南下官船护卫、地方守备参谋；杭州方向政治阴影加厚时。
初始事件：有人指认「假太子」；另一人要借名器募兵；城外出现成群青黑身影。
开场白建议：「名帖能开城门，也能引来斩立决。城垛上的火把数得清，城外的影子数不清。」
关键NPC立场：**朱慈烺**要正名；**方枝儿**要抽身或加码；地方军头要价。
主线钩子/支线：护名器过关；支线＝交换一份南明诸镇风向。
危险度/规避：高——政治杀与尸群并至。
任务方向/奖励：部曲编制、城防一夜指挥权、可核的盟友名册。

**四阶（尸祸一六四四 · 一镇棋盘）**
切入身份/时点：镇标参谋、粮台官、使者；触及黄得功等章题级武将周边。
初始事件：「非我友军」的火力误会；大水与祭庙政治；尸祸被某方当武器或当借口。
开场白建议：「鼓角里有人喊勤王，有人喊屠伪。大水不认太子，只认地势。」
关键NPC立场：诸镇武将——利益与名义；太子侧——要满饷与承认；尸潮——无谈判对象。
主线钩子/支线：促成一次有粮台的合作或全身而退；支线＝夺回被水冲散的名册。
危险度/规避：极高——一镇级与灾难级阴影；严禁硬刚清军主力/开封级尸潮本体。
任务方向/奖励：一镇临时印信、粮台份额、战略级情报（仍禁止发「天下兵马大元帅」空衔当无代价通关）。
