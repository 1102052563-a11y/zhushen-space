# 装甲骑女伊莉丝-堕落魔将
<!--meta lib=主库 tiers=四-->

## 剧情

**【作品来源】**
《装甲骑女伊莉丝》（装甲騎女イリス / Soukou Kijo Iris / Armored Knight Iris）为 Lilith Soft 旗下 Black Lilith 于 2006 年 3 月发售的 PC 成人向 ADV（DLsite 产品号 RJ022698），企划·监督笹@，剧本そのだまさき，原画 ZOL；主角声优春日アン。后由 PIXY 改编为 OVA 四卷（2007–2009），另有 Special 番外。系列属「星际联邦·美少女装甲骑 + 敌后沦陷」军事科幻。**本档案专写「堕落魔将」分支：伊莉丝·雷贝尔被俘后被军阀情报部截胡，经神经项圈与识别码改造，授衔为敌方魔将级战力符号，反过来攻打联邦——强调军事叛旗、指挥权转移、宣传战与士气污染，不写色情细节。** 与同系列其他分支边界：骑女沦陷＝商品化监禁；魔铠融合＝人机嵌合；双堕＝姐妹同没；妹妹援救＝梅莉主视角强攻；姐妹侍奉＝双人联锁人质；**本线核心是她穿上敌方将旗、反过来打联邦**。

**【世界定位】**
远未来，人类与多种族组成的宇宙联邦保安军 SSF 以「装甲骑」人型机动兵器维持治安。第 101 特殊装甲中队王牌中尉伊莉丝·雷贝尔在行星レフト（Left）超空降下战中被俘。敌方军阀不满足于奴役或实验，而要制造「联邦英雄变魔将」的政治兵器：让她以魔将身份率异种与叛军攻打故土。契约者面对的是**旗帜、影像与身份的战争**——输出可毁街区，真正的胜负在识别码、转播权与授衔密钥。

**【世界观 · 力量体系】**
本世界战力以**装甲骑（装甲騎）**为核心。装甲骑是联邦最新锐人型机动兵器，可超空降下、音障机动、据点攻坚；驾驶员与机体通过神经接口与战术链路协同。联邦保安军 SSF 第 101 特殊装甲中队因成员几乎全为女性，被宣传部称为「装甲骑女」。行星レフト歧视女性、人口贩卖猖獗，反对联邦的城市与军阀、兽人佣兵（オーク，战场拾荒者）并存。黑市、奴隶拍卖与改造工房构成敌后灰色生态。

战力阶梯（本条目仅开放**四阶**切入，其余作背景标尺）：

1. **非武装平民 / 缴械俘虏 / 难民营民兵**：徒手与冷兵器级，≈一阶以下～一阶。无装甲骑时对正规军几乎无威胁。
2. **常规步兵 / 奴隶警备 / 黑市打手**：热武器与轻装甲，可端小据点、清街，≈二阶～三阶边缘。对装甲骑只能骚扰。
3. **正役装甲骑驾驶员（伊莉丝、梅莉等王牌）**：驾驶 ADT 系列机体，可掀建筑群、突破音障、单机攻坚卫星城级据点，≈**四阶**（破坏力对齐图鉴：摧毁据点 / 掀平建筑群，抗常规热武器与小型导弹，速度近瞬移）。
4. **中队协同 / 舰炮支援 / 重型异种战兽**：四阶中上～顶；多机编队可覆盖城区，舰炮可改写局部战场。
5. **堕落魔将完成态**（敌方将旗 + 神经项圈强化 + 宣传光环 + 魔将外骨骼）：仍锚定**四阶顶**，可造成城市街区级毁伤并瓦解友军士气；其真正威胁是「士气污染」——看见旧英雄旗帜的联邦士兵射击精度与服从度下降。**宁低勿高，不升五阶。**

特殊系统：
- **识别码**：联邦旧码以 SSF-101 序列开头；敌方魔将码以黑旗序列开头。窗口期伊莉丝会交替广播两者，形成可被友军定位的「错误噪声」。
- **神经项圈 / 抗命瘫痪模块**：违令则脉冲瘫痪，授衔后升级为「意志校准」——播放联邦遗弃剪辑（真假参半）同步脉冲。
- **中队暗语**：旧版暗语表（如「月影—三连—归队」）可短暂压过抗命脉冲，是人格锚与非战斗破局手段。
- **宣传伤害**：本线独有——影像比炮弹先抵达前线；转播权等于半个军团。

死亡与复活：本世界无复活机制。驾驶员阵亡即永久；机体可修复，黑匣子可回收。魔将态「不可逆区」是剧情阈值（可用技术推迟，不可空口取消），指洗脑接口进入高稳定态后人格窗口极度缩短，**不是**战力归零。

乐园阶位映射：平民/俘虏≈一阶；常规军/奴隶警备≈二～三阶；装甲骑女主战（伊莉丝、梅莉、正役 ADT）≈四阶；堕落魔将完成态 + 黑旗要塞协同≈四阶顶。顶点＝黑旗要塞指挥链 + 魔将伊莉丝 + 转播网，**条件性胜利**打密钥、识别码、转播节点与暗语锚，严禁战力归零式削弱，严禁四阶开六阶外挂。

**【地理 · 舞台】**
- **宇宙联邦 / SSF 辖区**：多种族星际政府；宣传部把第 101 中队做成招募海报与儿童读物生产线。第 101 特殊装甲中队基地含机库、暗语室、食堂与摄影棚——英雄影像与真实伤亡在同一走廊并存。
- **行星レフト（Left）**：极端歧视女性、人口贩卖网络盘根错节；兼具治安维护与军阀游击。超空降下战区是伊莉丝被俘触发点。
- **ミリオン（Million / 通称ダークタウン / Dark Town）**：敌对城市，法律允许邪恶、嘲笑正义；奴隶拍卖、改造工房、黑市零件与情报中转。本分支中它是「从商品到将器」的中转站，而非终点。
- **黑旗要塞**：军阀情报部与宣传部共建的主舞台。魔将授衔仪式、指挥舰对接、影像友好型走廊（处处隐藏摄像机）均在此。要塞内部派系裂缝：野战派、改造工房、奴隶商、异种部队互不服气。
- **联邦边境卫星城**：魔将第一次公开作战的牺牲品舞台；市长与平民疏散窗是可介入节点。
- **轨道转播站 / 转播联盟**：中立但卖信号的第三方；敌我双方抢夺「影像真实」的节点。出价、加密、延迟秒数均可成为战术资源。
- **埃尔芬族联络点**：梅莉所属长寿高智种族；是否承认「魔将伊莉丝」为外交议题，可决定技术支援与观礼立场。
- **月影残骸展示台**：ADT-74 ゲツエイ残骸被做成授衔背景或被魔将外骨骼吞噬部件，是象征与技术双重道具。

**【世界剧情线】**

① **英雄符号的制造（战前）**  
伊莉丝·雷ベル（Iris Rebel，地球人）与搭档梅莉·纳セリ（Mei Li Naseri / メイリィ・ナセリ，エルフィン人）同为 SSF 第 101 特殊装甲中队中尉。伊莉丝驾驶新锐装甲骑 **ADT-74 ゲツエイ（Getsuei / 月影，亦有 Griffin 译）**，梅莉驾驶 **ADT-73 コンゴウ（Kongou / 金刚，亦有 Chimera 译）**。二人屡建战功，联邦宣传部把伊莉丝做成招募海报与儿童读物中的「美丽战少女」；中队内部清楚海报是滤镜，但镜头前不许显出疲惫。伊莉丝性格明朗、对后辈温柔、有点孩子气，正义感极强，对犯罪者冷酷——被敌人畏惧，也被宣传部当成最易消费的符号。梅莉外表幼态，实际年龄超百（长寿种族），高傲争强、科技与战术素养极高，常与伊莉丝拌嘴却是最可靠的精神锚。主题句：战争需要英雄，宣传部需要滤镜。

② **レフト超空被俘**  
反对联邦的城市与军阀在レフト发动攻击，第 101 中队超空降下。激战中机体重创，通讯失联。战场拾荒的兽人佣兵（オーク，被称为「战场鬣狗」）捕获伊莉丝；月影重创，军籍徽章被剥。她仍以中尉口令要求依战俘公约——敌方报以嘲笑。梅莉同时或先后失踪（OVA/ADV 版本在失踪时点上略有差异），成为后续所有路线的牵引线。本分支在此分叉：普通奴隶商路线会把她送往ダークタウン拍卖与娼馆；**堕落魔将线则在拍卖前被军阀情报部截胡**。

③ **从商品到将器（情报部截胡）**  
奴隶商**ボズク（Bozuk）**本拟把伊莉丝作为高价商品送入百万都市拍卖。军阀情报部发现她的战术价值高于拍卖价：若让「联邦英雄」现身敌阵，联邦士气可断一截，招募海报会变成反宣传。改造工房植入**指挥用神经项圈**与敌方识别码；爆弹项圈升级为「抗命即瘫痪」。伊莉丝被要求朗读敌方效忠文，她故意读错一个星图坐标——这是窗口期第一次人格痕迹。博兹克后悔没拿拍卖全款，却仍作为「货源中介」保留二次出售权与情报贩子角色。此阶段名场面：徽章落地、项圈咔哒声、效忠文读错坐标、宣传官第一次说「我们要的不是奴隶，是旗帜」。

④ **授衔仪式（堕落魔将成立）**  
黑旗要塞举行公开授衔：伊莉丝被冠以敌方魔将称号，身后是月影残骸与联邦俘虏阵列（俘虏被换上联邦旧制服充当「降兵」背景板）。转播信号打入联邦频段。授衔彩排时，每一处转角的隐藏摄像机已在剪辑「完美叛将」。联邦侧第 101 中队有人砸碎屏幕，有人开始怀疑「她是不是自愿」；食堂出现「仍称中尉」与「改称目标体」两派。宣传官比军官更危险——他不靠武力，靠「下一条全球头条」。T 轴关键节点：T-72h 神经项圈终检；T-48h 授衔彩排；T-24h 转播链路加密、轨道联盟出价；T-12h 梅莉侧可能干扰、鹰派暗杀小组进入射程；T-0 授衔直播。

⑤ **第一次叛旗作战（边境卫星城）**  
魔将伊莉丝被投入攻打联邦边境卫星城。她可能在短窗口故意打偏主炮；或因洗脑接口强行执行清扫；或用旧中队暗语向轨道扔「假投降真坐标」。无论哪种，联邦记录都会写「装甲骑女叛变」。平民疏散窗、市长求援、中队暗语噪声同时出现。四阶战场表现：魔将外骨骼一击可掀翻轨道电梯基座周边建筑群；主炮擦城而过造成街区级火灾带；近战可在音障机动中拆解常规装甲骑。联邦常规军需中队级协同才能围猎，且**士气检定先于装甲检定**。T+6h 进入交战；T+30h 第二次作战目标（补给枢纽）公布；T+72h 若仍未夺旗，洗脑接口进入「不可逆区」。

⑥ **三次公开作战时间表与联邦手册撕裂**  
军阀在授衔后强制她出席三次公开作战：一打边境示众，二打补给枢纽断联邦后勤，三打母舰符号终结「装甲骑女」神话。每次作战前有「意志校准」：播放联邦遗弃她的剪辑影像（真假参半），同步项圈脉冲。契约者若能替换剪辑母带，校准失败率上升，伊莉丝短窗口失控概率增加。联邦侧应对手册分三级：**舆论否认（假人偶）**、**战术击毁（鹰派）**、**特种夺还（鸽派 + 梅莉）**。三种手册互相拆台，前线指令一日三变。第 101 中队若收到击毁令，会有人「弹药卡壳」式抗命；若收到夺还令，又有人担心救回的是「带病毒的魔将」。这种撕裂本身就是敌方第二战果。

⑦ **梅莉与指挥链分裂**  
梅莉·纳セリ若在外线：要求夺回「中尉」而非击杀「魔将」。她拒绝族人把伊莉丝写成「已死人类符号」；埃尔芬族因她的立场摇摆——承认魔将人类化可能损害族誉，拒绝则失去技术介入窗口。第 101 中队队长在批准或拒绝「击杀魔将」令时成为关键枢纽：他若坚持在战报里写「中尉」，等于政治自杀；他若改写「目标体」，等于杀死自己人。联邦鹰派议员要格杀令；鸽派情报官要活捉研究逆转。リルル（Riruru）——ダークタウン中与梅莉外貌相似的エルフィン侍从——在本分支可作为「假梅莉」干扰或「真身份线索」支线，但**堕落魔将主线不以娼馆为终点**，她更常作为情报与情感误导装置出现。

⑧ **黑旗要塞内部裂缝**  
要塞并非铁板：军阀野战派嫌改造工房抢功；奴隶商博兹克后悔没拿拍卖全款；异种部队不满听命于「前人类女」。伊莉丝若学会利用派系矛盾，可比正面硬刚更早拿回能动性。军阀后勤官开始担心：魔将太成功会让士兵崇拜她超过崇拜军阀本人——这是第三势力终局的引线。被俘联邦士兵群是魔将阅兵时的人肉背景板，也是暴动火种。城市废墟里的孩子指认「那个飞的是联邦姐姐」，父母捂住他们的嘴——战争的语义被一层层改写。

⑨ **终局三分支（必须写清世界变成什么样）**  
- **旗碎人存**：契约者 / 梅莉切断授衔密钥，伊莉丝失去魔将权限但联邦永不信任她。她活着，名字在两套新闻稿里分别是「被救回的受害者」与「未清洗干净的叛将」。第 101 中队可能收容她，宣传部可能要求她继续拍「悔过片」。世界状态：黑旗要塞受挫但未灭，识别码战争转入地下。  
- **旗在人灭**：她撞毁己方指挥舰，以魔将身份完成最后一次联邦式突击，死后仍被两边当符号消费。联邦立碑，军阀立反面教材，真相沉入黑匣子。世界状态：士气短暂回升后陷入更深的符号虚无。  
- **旗换主人**：她夺取黑旗要塞，建立既非联邦也非军阀的第三势力「无旗装甲骑」，专打奴隶船与宣传塔。梅莉可能加入或反对；联邦与军阀短暂联手围剿「无旗」。世界状态：战争从二元变成三元，转播联盟成最大赢家。

⑩ **主题与可介入因果**  
堕落魔将问的是：**当英雄被改写成敌人，战争还需要真相吗？** 奖励应偏向识别码、转播权、授衔密钥、中队人情、第三势力徽章，而非更高输出。契约者任何行动都会被记入「是否动摇卡里斯马」的账——联邦与军阀两套新闻稿会立刻互相矛盾，第 101 中队食堂里沉默的那一桌比任何战报都诚实。

⑪ **授衔前后的联邦内部 72 小时**  
授衔直播打入联邦频段的同一小时，宣传部紧急会议三选一：否认（「那是假人偶」）、定性（「叛将」）、沉默（「调查中」）。鹰派议员当场起草格杀授权；鸽派情报官要求冻结所有对外通稿 12 小时。第 101 中队食堂的电视被拔掉电源，士兵改用走私芯片循环敌台——有人哭，有人骂，有人申请调离。队长下令：战报暂写「目标体·身份待核」，私下对核心机师说「她还是中尉，直到黑匣子证明相反」。梅莉若在外线，会在同一 72 小时内尝试三次干扰：一次对转播延迟、一次对项圈校准频段、一次对埃尔芬联络频道申请「技术观察席」。三次里至少一次会被宣传官剪进「联邦内鬼」叙事。契约者若在此窗介入，最小有效行动是**改一帧通稿用词**（「叛将」改「失踪中尉」或反向），其政治当量可能高于击落一架常规装甲骑。

⑫ **边境卫星城作战的微观因果**  
第一次叛旗不是抽象的「打一城」，而是可拆解的事件链。T+6h 前：市长打开三级避难广播；轨道电梯基座周边进入强制疏散；常规军布置防空但士气检定已开始掉点——有人认出海报上的脸。交火中：魔将主炮若擦城而过，火灾带按街区编号蔓延；若正中指挥塔，联邦会立刻把通稿从「营救」改成「清除」。伊莉丝窗口期的故意偏射需要暗语确认与转播延迟配合——延迟太短，宣传官会当场重剪；延迟太长，项圈脉冲会打断她的手指。战后 6 小时：平民尸体统计与「仍称姐姐的孩子」口述同时进入两边新闻；中队暗语噪声若被友军接收，会出现「弹药卡壳」式抗命；若未被接收，击毁令执行率上升。契约者可介入点具体到：替换避难广播文案、伪造电梯维修工单挡开平民、在火灾带边缘投放黑匣子碎片、向市长出售 90 秒疏散窗。

⑬ **第二次与第三次作战的战略含义**  
第二次目标是补给枢纽：打的是联邦前线三个卫星城 14 日份弹药与净水器材。枢纽若毁，レフト方向的 SSF 驻军会被迫收缩，宣传部会把收缩写成「战略重组」。伊莉丝若在此战故意留一条输送带，会被军阀记为「抗命嫌疑」，校准强度上升；若彻底摧毁，联邦鸽派会失去「她还在反抗」的证据。第三次目标是第 101 母舰符号：不是单纯炸船，而是在镜头前让「装甲骑女」神话当场死亡。母舰若空城，伊莉丝可能撞指挥舰自毁（旗在人灭）；若满员，她可能被逼完成清扫后彻底进入不可逆区。军阀时间表写死三次作战，是因为宣传节奏需要「示众—断粮—弑神」三幕剧；打断任一幕的转播，比击落机体更伤黑旗卡里斯马。

⑭ **人物关系网（可被契约者撬动的边）**  
伊莉丝—梅莉：暗语「月影」三连是人格锚；任何救法若要求梅莉先弃族誉，梅莉会拒绝。伊莉丝—队长：上下级信任被宣传污染，队长能给的最大庇护是战报用词。伊莉丝—博兹克：恨意明确但可交易情报；博兹克永不谈理想。伊莉丝—宣传官：最危险的非战斗关系——宣传官要的是「可控的表情」。梅莉—埃尔芬使节：族援以「不承认魔将人类化」或「有限技术观察」为条件。鹰派—鸽派：互拆手册，前线一日三令。野战派—改造工房：抢功裂缝，可被伊莉丝或契约者扩大。被俘士兵—阅兵阵列：从背景板到暴动，只需一声可识别的旧中队呼号。契约者插入时必须先选「夺旗 / 碎旗 / 换旗」语义，否则所有 NPC 都会把你当第四种宣传素材。

**【势力图谱】**
- **宇宙联邦 / SSF / 第 101 特殊装甲中队**：宗旨＝治安与符号生产；地盘＝联邦辖区与レフト前线；代表＝伊莉丝（战前）、梅莉、队长、鹰派议员、鸽派情报官；战力＝装甲骑四阶主战；与军阀敌对，内部被宣传撕裂。宣传部与作战部互相拆台是常态：前者要「可播出的胜利」，后者要「可执行的命令」。第 101 因全女中队形象被过度消费，士气对「海报脸叛变」的敏感度高于其他部队。
- **敌对军阀 + 黑旗要塞指挥部**：宗旨＝要魔将旗帜与士气污染；地盘＝黑旗要塞、边境交战区；代表＝宣传官、改造术者、野战派指挥官；战力＝魔将完成态 + 异种部队 + 舰炮；内部派系裂缝可利用。野战派认为改造工房「用摄像机打仗」；改造工房认为野战派「不懂现代战争是影像战争」。异种部队编制复杂，对「前人类女魔将」的服从靠恐惧与战利品维持，不靠忠诚。
- **ミリオン黑市 / 奴隶商网络**：宗旨＝利润；地盘＝ダークタウン与中转站；代表＝ボズク；战力＝二～三阶打手 + 情报；向军阀供货亦可倒卖。ダークタウン的法则是「邪恶被允许、正义被嘲笑」——这是原作地理气质，本分支把它当作中转生态而非终点舞台。
- **エルフィン族**：宗旨＝族誉与技术伦理；地盘＝联络点与观礼席；代表＝梅莉、族内使节；战力＝科技与外交；是否承认魔将人类化为议题。长寿种族的时间观使「一次授衔直播」在族内可能被当成短期丑闻或长期外交案，取决于使节怎么写回报告。
- **轨道转播联盟**：宗旨＝卖信号；地盘＝轨道转播站；代表＝临时雇员与出价官；战力＝无直接武力，掌握影像真实；中立可被买。延迟秒数、加密层级、优先级令牌都是明码标价的战术资源。
- **第三势力「无旗装甲骑」（终局可选）**：宗旨＝专打奴隶船与宣传塔；地盘＝夺下的黑旗要塞残部；代表＝伊莉丝（换旗后）、可能的梅莉；战力＝四阶顶但孤立，遭联邦与军阀短暂联手围剿。其存在本身会改写「二元战争」语义，转播联盟将获得最长的竞价窗口。

**【贵重物品】**
- **魔将授衔密钥**：敌方将权核心；切断则魔将权限崩解。来历＝黑旗要塞改造工房与指挥舰双锁。持有者可临时关闭抗命脉冲，但无法单独删除识别码——密钥与晶片成对才完整。
- **敌方识别码晶片 / 联邦旧识别码（SSF-101 序列）**：交替广播可形成定位噪声；伪造噪声＝非战斗破局。原作语境里识别与军籍徽章被剥是身份剥夺的起点；本分支把「再赋予敌方码」写成政治处决的反面——她被赋予了一个不能拒绝的新名字。
- **神经项圈抗命模块 / 意志校准固件**：版本号可调包；替换剪辑母带可提高校准失败率。固件日志会记录每一次脉冲与朗读错误，是法庭与宣传双重证据。
- **联邦英雄影像母带**：可被伪造的招募海报源头；保护或抹除都改变舆论战。母带若公开「未滤镜花絮」（疲惫、骂街、真实伤亡），英雄工业会先于战场崩溃。
- **月影黑匣子**：证明伊莉丝未主动叛变的物证；授衔台上的残骸部件亦可回收。黑匣子时间戳若覆盖「被俘前最后 30 秒」，可反驳「自愿降敌」叙事。
- **中队暗语表（旧版）**：含「月影—三连—归队」等；可短暂压过抗命脉冲。新版暗语在被俘后已轮换，故旧表对联邦现役价值有限，对伊莉丝窗口期价值极高。
- **轨道转播优先级令牌**：决定全球先看见「叛将」还是「求救」。令牌可租不可永久买断——联盟靠重复收费活命。
- **联邦弃子黑档**：证明联邦曾计划弃子的机密；公开则两边同时受伤，是终局级舆论核弹。黑档与授衔密钥加密协议同源的伏笔在此落地。
- **逆授衔公式（梅莉持有半页～全页）**：技术 + 象征双重需求；完整页是「旗碎人存」关键。半页只能制造 90 秒权限抖动，全页才能稳定剥离将权。
- **第三势力无旗徽章（终局互斥奖励）**：换旗路线象征物，与联邦人情、军阀密钥互斥。徽章本身无输出加成，只有政治承认度。

**【隐藏剧情 · 伏笔】**
- 授衔密钥的加密协议与联邦早期「黑计划」同源——联邦曾设计过把己方王牌「伪叛」植入敌阵的弃案，后因伦理否决；军阀抄了半套。契约者若同时持有黑档与密钥，可证明「两边用同一套锁」。
- 宣传官可能是联邦叛逃摄影师，熟悉第 101 海报拍摄角度，故剪辑极具杀伤力。他知道伊莉丝哪一侧脸在镜头里更「英雄」，也知道哪一帧眨眼会被观众读成「动摇」。
- 伊莉丝孩子气外表下的冷酷执法记忆，是洗脑最难覆盖的层；暗语触发的是这层，不是「温柔」。对奴隶商与人口贩子的旧案卷宗，比任何爱国演讲更能唤醒她。
- 第三次叛旗作战预定目标是第 101 母舰——若契约者泄露时间表，中队可空城，代价是卫星城可能被放弃。空城成功则旗在人灭或旗换主人概率上升；失败则不可逆区提前。
- 联锁 / 双驾驶员共死弃案（与姐妹侍奉线互文）在联邦档案中存在，本线仅作技术溯源，不展开双俘。改造术者可能读过该弃案的接口章节。
- 军阀后勤官对「士兵崇拜魔将超过军阀」的恐惧，是第三势力终局的另一条引线——可策反的不是理想，是嫉妒。
- リルル外貌接近梅莉：可能是整容 / 基因仿制 / 真梅莉失忆分叉；本分支建议作干扰项，主线不依赖揭晓。若契约者把资源押在「救リルル＝救梅莉」上，会错过密钥窗口。
- 月影残骸展示台的某块装甲内侧刻有中队维修工的涂鸦——非官方暗语变体，仅老地勤认得，可作第二套噪声源。

- 第三次作战 / 母舰符号战 → 旗碎人存 / 旗在人灭 / 旗换主人 三终局之一。
- 终局后：两套（或三套）新闻稿定调；转播联盟清点利润；无旗或残中队进入长期游击或审查法庭；伊莉丝名字的官方写法成为新战场。

**【叙事基调 · 雷区】**
画风：军事科幻 + 宣传战残酷 + 身份政治；镜头感强（授衔、转播、食堂沉默桌）；四阶战场可毁建筑群，不可炸行星。口吻冷硬，制度语言优先于煽情。写战斗先写谁在看镜头，再写炮弹落点。

忌：
1. **忌色情描写**——用授衔、叛旗、识别码、士气崩溃、项圈脉冲替代一切成人向细节。原作成人向背景只作作品来源标注，正文不展开。
2. **忌把伊莉丝写成自愿卖国**——她是被改造与符号绑架的军人，窗口期反抗必须可写。
3. **忌四阶开六阶外挂**——破坏力封顶街区 / 据点级；士气污染是机制不是神通。
4. **忌与系列其他分支串台**——本线不是拍卖终点、不是人机嵌合主轴、不是双人联锁侍奉、不是梅莉单视角强攻。
5. **忌代称人名**——必须伊莉丝·雷ベル、梅莉·纳セリ、ボズク 等真名；宣传官 / 队长等原作未具名者用职衔，禁止「女主角模板」「魔将群像」。

最早可切入锚点：**授衔仪式前 12 小时**，或**第一次叛旗作战倒计时**。契约者开局须立刻面临「击毁 / 保全 / 抬价」三指令冲突，并听到暗语「别信旗」。

## 阶位切入点

> 阶位↔原作：四阶≈正役装甲骑王牌与堕落魔将完成态交战（ADT 系列 / 魔将外骨骼，破坏力＝摧毁据点·掀平建筑群·音障机动）。世界顶点＝黑旗要塞指挥链 + 魔将伊莉丝·雷ベル + 转播网，**条件性胜利**打授衔密钥、识别码、转播节点与暗语锚；情报优先，严禁战力归零，严禁越阶外挂。本世界仅覆盖四阶。

**四阶（堕落魔将·叛旗夺还线）**

切入身份/时点：契约者以**轨道转播联盟临时技师**、**联邦宣传部叛逃剪辑员**、**第 101 中队失踪认定官**或**梅莉·纳セリ的临时联调员**切入，锚定**授衔完成、第一次叛旗作战前 12 小时**（T-12h 至 T+6h 为黄金窗；可延至 T+30h 枢纽战前）。必须写清你为何能接近核心区：转播权限、失踪认定公文、厨房后勤工单、交换观察请柬四选一。四阶身份自洽：你有能力在音障边缘存活、读懂装甲骑链路噪声、不被常规步兵挡在门外——但你仍不是一人推平黑旗要塞的「六阶外挂」。

初始事件：你的控制台被同时塞进三份信号——联邦鹰派要你直播「击毁叛将」，军阀宣传官要你直播「魔将凯旋」，鸽派情报官要你保留「可逆转帧」。与此同时，伊莉丝·雷ベル的私频用中队旧暗语跳出三个字：「别信旗。」月影黑匣子片段被篡改后塞进你的收件箱；梅莉·纳セリ私频塞来逆授衔公式半页。改造工房工单显示抗命固件版本将在 T-6h 锁定。你必须在倒计时内选一条路：插入 3 秒中队旧影像 / 裁掉伊莉丝眨眼的三帧 / 把公式转给改造工房后门 / 把疏散窗时间表卖给市长——任一选择立刻改写转播脚本，并触发项圈脉冲、暗杀小组换频或联盟加价。

开场白建议：「重力场把黑旗撑开，像一口倒悬的井。伊莉丝站在井底中央，盔甲反光处是联邦旧徽章的刮痕。你耳麦里同时响起三种笑：鹰派的、军阀的、以及她自己在校准脉冲间隙漏出的那一声——像还记得月影启动音。倒计时数字跳过你的心跳。屏幕右下角，轨道联盟的出价还在跳；左下角，边境卫星城的避难灯一盏盏亮起。」

关键NPC立场：**伊莉丝·雷ベル（魔将）**——窗口期求夺旗或求死，要暗语不要同情，会交替广播 SSF-101 与黑旗识别码制造噪声；对廉价「我来救你」会冷笑，对能改转播脚本的人会给第二句暗语。**梅莉·纳セリ**——要逆授衔完整页与双人同生，拒绝族人单救，对「击毁更干净」论敌对；可提供技术后门但要求契约者不得向鹰派泄公式。**军阀宣传官**——要完美叛将镜头，契约者在镜头前露馅会被剪成新素材；愿出高价买「她眨眼的三帧」。**ボズク**——要二次出售权与密钥情报差价，先谈价格；可卖要塞后勤图，也可把你卖给改造术者。**第 101 中队队长**——要战报里保留「中尉」称呼，抗命与服从之间摇摆；能给你失踪认定公文，也可能在击毁令下被迫对你开火。**联邦鹰派议员**——要格杀直播，视鸽派与梅莉为内鬼。**联邦鸽派情报官**——要活捉与黑档时机；结盟后可能「不便迎回」式灭口。**边境卫星城市长**——只要疏散窗与偏射，不在乎旗帜。**改造术者**——要固件协议不被调包；发现后门会提前锁死不可逆区。**リルル**（若出场）——用外貌混淆「救梅莉」动机，可买可防。

主线钩子/支线：主线节点——T-0 插入旧影像打断授衔语义；T+6h 边境卫星城偏转主炮换平民疏散；T+30h 前夺授衔密钥 / 调包抗命固件；可选终局推动旗碎人存 / 旗在人灭 / 旗换主人。支线（本阶独有）：①伪造鹰派击毁令诱敌深入；②煽动阅兵阵列俘虏暴动；③向轨道联盟购买双向加密频道；④保护或抹除英雄海报母带；⑤公开联邦弃子黑档（舆论核弹，两边受伤）；⑥回收月影黑匣子证明未主动叛变；⑦利用野战派 vs 改造工房裂缝策反；⑧向异种部队散布「人类魔将抢功」谣言加速内讧；⑨用地勤涂鸦变体暗语开第二噪声源；⑩阻止鸽派对「不便迎回」的伊莉丝实施灭口。

危险度/规避：极高——四阶顶输出 + 士气污染 + 转播网；必须规避无暗语锚时与魔将伊莉丝·雷ベル正面对波；规避在转播镜头前暴露乐园外挂（会被制成新宣传素材）；规避第三次作战前泄露母舰空城计划给错误一方；规避把全部资源押在リルル身份赌局上。顶点＝黑旗要塞 + 魔将协同，**条件性胜利**打密钥与转播，不硬拼街区级火力对轰；严禁「她被削弱所以战力为零」。

任务方向/奖励：决定旗帜归属与伊莉丝名字的写法（中尉 / 叛将 / 无旗）；保住选择权本身即主目标。奖励取向＝授衔密钥、识别码晶片、转播优先级令牌、中队暗语完整表、月影黑匣子、逆授衔公式全页、第 101 人情、弃子黑档副本或第三势力无旗徽章（互斥，不越级发行星级神器）。失败即代价：转播被切、项圈进入不可逆区加速、卫星城疏散窗关闭、梅莉公式作废、你自己被两套新闻稿点名为「第四种叛徒」、下一次校准影像里出现你的脸。
