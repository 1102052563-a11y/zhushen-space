# 机巧少女不会受伤
<!--meta lib=主库 tiers=一、二、三、五、六-->

## 剧情

**【作品来源】**
《机巧少女不会受伤》（日文：機巧少女は傷つかない；英译 Unbreakable Machine-Doll）为海冬零児撰写、LLO绘制插画的日本轻小说，Media Factory（MF文库J）2009年11月至2017年7月刊行全16卷（末卷分上下）。漫画由高城计作画连载于《月刊Comic Alive》；电视动画2013年10–12月由Lerche制作播出全12话（覆盖小说1–3卷），监督吉本欣司，系列构成柿原优子。尖端出版（繁中）、天闻角川（简中）有授权。类型为蒸汽奇幻·校园战斗·人偶使与自动人偶搭档；文风热血、阴谋与人偶伦理交织。**本档案只写学院、夜会、势力与战斗因果，不写色情细节。**

**【世界定位】**
架空20世纪初的大英帝国：机巧魔术（Machinart）把魔术回路嵌入自动人偶，使之成为各国军备与学院竞赛核心。主角赤羽雷真携禁忌人偶夜夜留学华尔普吉斯皇家机巧学院，以「倒數第二」身份挤入四年一度的夜会，目标是魔王（Wiseman）资格与禁术权，并追查灭族的兄长赤羽天全（面具人马格努斯）。契约者切入的是「人偶使—人偶—国家—结社」四角棋盘，而非单纯校园恋爱。

**【世界观 · 力量体系】**
机巧魔术：近代发明，以魔术回路取代冗长咏唱与法阵；自动人偶（Automaton）内藏回路，由人偶使供魔或强制支配（Force）、高等念动操纵关节。
- **夏娃的心脏**：赋予人偶生命与智慧的基础回路，唯一突破「魔活性不协调原理」（一体内不可并存两种魔术）的例外；心脏毁则人偶不可复原。
- **其他回路**：金刚力（分子级超硬度物质化、肌力千倍）、魔剑（灭元素/存在反转）、完全统制振动、音压操纵、热风操纵、魔法噬食、雷霆神器、距离操作等。
- **禁忌人偶（Bandoll）**：材料含活体，魔力亲和更高，伦理规范禁止量产，夜会未禁参赛。
- **神性机巧（Machine-Doll）**：可自给魔力、接近完全自律；「机巧士兵系列」把人体改造成禁忌人偶并搭载完全统制振动，口号「神性机巧不会受伤」。
- **承蒙誓約之子 / 白神子**：极稀有高魔力亲和者；D-works以孤儿心脏机巧手术量产伪品，代价白化（珍珠发、红瞳）。
- **强制支配 / 念动**：对自主人偶的控制层级；念动是无自主人偶的高等操纵。
- **夜会规则**：成绩前百获白手套（动画为黑）资格；夜间对战，负者交手套出局；存活到最后者成魔王，可阅禁书、研禁术，无视部分国际魔术宪章。顶尖十三人称「Rounds」。

战力阶梯（破坏力对齐乐园图鉴，本条目覆盖一、二、三、五、六，无四阶切入）：
1. **无回路平民 / 未供魔人偶躯壳**：≈一阶以下。
2. **入学级人偶使 + 量产/初级人偶**：碎石掀车、近战压过凡人小队≈**一阶**。
3. **手套持有者中坚（约20–100名）**：稳定超凡、可毁装甲载具/小建筑≈**二阶**。
4. **手套上位 / 十三人边缘**：一域高手、端据点≈**三阶**。
5. **十三人中坚—上位（军团级战队、魔剑龙、剑帝等）**：掀平建筑群至街区级、一人军队≈**五阶**（宁低勿高，不单开四阶条目）。
6. **魔王候选巅峰 / 蔷薇师团干部 / 神性机巧完成体 / 红翼阵完整态**：地区级灾难、领域与规则皮毛≈**六阶**。
7. **世界叙事顶点**（结社顶级蔷薇、完全神性机巧灭世风险）：六阶顶到七阶门槛，**本条目按六阶封顶处理**。

乐园阶位映射：学院杂鱼≈一阶；手套中坚≈二阶；手套上位/十三人下≈三阶；十三人主力与军舰级战队≈五阶；魔王·蔷薇干部·神性机巧≈六阶。顶点条件性胜利：打心脏/回路协调/供魔链，严禁「战力归零式削弱」。

**【地理 · 舞台】**
- **华尔普吉斯皇家机巧学院**：英国最高机巧学府，夜会主舞台；宿舍、医务室、实验室、钟楼、地下道、愚者的聖堂等场景密集。
- **机巧都市利物浦**：港口都市，乘列车半日可达伦敦；雷真入境节点。
- **伦敦 / 王室—军部—魔术协会三角**：专利、宪章、处刑权所在。
- **日本赤羽本家遗迹 / 军部密探线**：灭门案与「红翼之血」源头。
- **贝琉家旧领地与城堡**：夏露精灵术与魔剑传承地。
- **D-works 孤儿院—实验设施**：加姆系列、天使系列、人造白神子生产线。
- **德意志—罗伦斯基金会线**：十字架骑士的情报与奖学金网络。
- **结社蔷薇师团地下网络**：无固定首都，以「蔷薇」干部议事支配暗线政治。

**【世界剧情线】**
① **前史：赤羽灭门与雪月花**  
极东人偶使名门赤羽一族以集团战与「红翼阵」闻名，血脉「红翼之血」出优秀人偶使。故事两年前，族中鬼神级天才赤羽天全屠尽本家（雷真幸存），目的牵涉「造神／神性机巧」。当代名匠花柳斋硝子以自身肉体为素材完成禁忌三姊妹〈雪月花〉：伊吕里（雪）、夜夜（月）、小紫（花）。雷真在军部与硝子安排下获得夜夜，经死斗式对决后结成主从——他坚持把人偶当人。

② **第1卷 Facing "Cannibal Candy"：入学与食魔者**  
雷真以全年级倒數第二（登录代码「倒數第二」）入学，夜夜以金刚力碾压挑衅者。他向排名第一的马格努斯递交妹妹抚子骨灰罐而非手套战。风纪委员菲利克斯·金斯佛特以「食魔者」连环毁人偶、失踪学生案换夜会名额委托雷真。真凶是菲利克斯与禁忌人偶爱莉莎（〈魔法噬食〉，伪装莉瑟特），企图嫁祸夏洛特·贝琉。雷真与夜夜摧毁爱莉莎，夏露赠防御吊坠；雷真获第一百位手套资格。因果：他进入夜会视野，同时被「十三人」与军部密探身份双重标记。

③ **第2卷 Facing "Sword Angel"：D-works 与剑帝姐弟**  
芙蕾与洛基为D-works制造的人造承蒙誓約之子（珍珠发红瞳，心脏机巧化）。芙蕾的加姆系列（〈音压操纵〉犬型禁忌人偶）若败则被废弃。布朗森主导非法实验与天使系列（路西法）武装。雷真夜会首战对上洛基（代码「自我飞翔的焰剑」），保护夜夜中重伤。孤儿院线：犹宓等加姆原型被弃。最终雷真—洛基联手摧毁路西法，金柏莉率十字军逮捕布朗森。因果：洛基立志成魔王修复心脏；芙蕾对雷真产生依赖；D-works黑幕公开一角。

④ **第3卷 Facing "Elf Speeder"：钟楼、安莉与机巧士兵**  
夏露之妹安莉艾特多次自杀未遂；塞德里克（实为爱丽丝·拉赛福／学院长之女伪装）胁迫夏露暗杀学院长拉塞福。爱丽丝的管家辛是机巧士兵MK4，〈完全统制振动〉近自律，无供魔时仍可压倒十三人级。雷真绑架式保护安莉，引出塞德里克真身爱丽丝·伯恩斯坦（代码「加速妖精」）。钟楼被西格蒙特魔剑轰毁。终局：辛被夜夜终装＋芙蕾定位＋魔剑协同击退；爱丽丝身份暴露；贝琉家被没收财产部分解冻用于重建；安莉转任克鲁尔实验室助手。动画主线止于此。

⑤ **第4卷起 Facing "Rosen Kavalier" 等：十字架骑士**  
德意志名门罗森堡领衔「十字架骑士」，奖学金来自罗伦斯基金会，垄断夜会约74–86名以收集技术。成员多搭载机巧士兵MK5（运动/静止特化）。罗森堡曾将索菲亚等人体改造；洛基与其再战使用传说级菲尼克斯（〈永劫之火〉）。此阶段学院成多国谍战场：日本军部（雷真、日轮）、结社、王室、德意志实验派交叉。

⑥ **中盘 Facing "Crimson Red"～"Star Gazer"：雪月花、未婚妻与结社**  
土门日轮（伊邪那岐流公主、雷真未婚妻）率式神入局，代码「统领魔军的黑曜姬」。伊吕里、小紫与夜夜三姊妹同台，红翼阵雏形成关键。金蔷薇阿斯特丽德之孙女奥尔嘉·萨拉汀（「金色的奥尔嘉」）与人偶托尔（〈魔剑〉）登场；白蔷薇之子奥古斯特·威隆（「倒數第一」）与奥尔嘉记忆被结社手术反复抹消以维持势力平衡。爱丽丝策动「雷真订婚」闹剧搅局。因果：结社不再是背景板，而是改写学生记忆与政治联姻的黑手。

⑦ **后盘 Facing "Target Gold"～"Elder Empress"：十三人混战**  
阿斯拉·厄恩（「三千世界天子」）以「魔王权限共享」聚拢集团，后与王妃葛洛丽雅（银蔷薇）联手制造学生对立；洛基以新人偶吉卜利勒（四回路切换念动）揭露阿斯拉「苏美人」身份并剥夺其资格。索涅奇卡·斯尼特金娜（「冻土的炎帝」）以耶梦加得（〈因果性置换〉）击败日轮、芙蕾等。海赛尔·海姆达尔（布朗森亲女）持「敕命诏书」黑刀（言语支配）复仇，被洛基姐弟放过后与芙蕾和解。桃乐西·麦高芬（「死者之王」、黑蔷薇孙女）以死灵术回收假死的芙蕾。

⑧ **终盘 Facing "Machine doll"：神性机巧与新平衡**  
马格努斯战队（火垂、玉虫、镰切、蜻蛉、蜘蛛姬、蜜蜂）对雷真形成「一人军队」压迫。第12卷雷真破坏阿里阿德涅之线，释放完整红翼阵。天全承认身份；与雷真联手欲以赤羽秘术复活抚子时，夜夜献出生命完成复活。终局：雷真于神性机巧内唤醒夜夜，阻止灭世级危机；夜夜以能使用金刚力的「人类」形态回归。魔术协会认定雷真「杀死马格努斯」而取消其魔王资格；洛基以第三名递补成「苍穹之魔王／雷帝」。雷真加入蔷薇师团，与黑太子艾德蒙、拉塞福等达成新暗约平衡。主题落点：**人偶是否为人、造神是否值得灭族、魔王权是否等于正义。**

⑨ **支线可介入轴**  
- 夜会每晚对战排表与弃权、换人、军团战。  
- 专利局与D-works式贿买。  
- 禁忌人偶黑市与夏娃的心脏回收。  
- 军部密探任务与学院长派系对冲。  
- 雪月花三姊妹人格与供魔生命力逆流伦理。

**【主要人物】**
- **赤羽雷真**（主角/日本留学生·军部密探）｜性格：把人偶当人、好管闲事、自嘲极低、吐槽役；格斗与射击顶尖｜装备·能力：夜夜；赤羽秘传红翼阵（弱化→完整）；空手道柔道暗杀剑术｜人物弧光：灭门幸存者→倒數第二手套者→红翼阵完成→拒魔王而入结社｜立场：仇视马格努斯/天全；保护夜夜优先；可被契约者当临时同盟但极度厌恶「把人偶当耗材」。
- **夜夜**（女主/禁忌人偶〈月〉）｜性格：自称妻子、善妒、对雷真极端忠诚｜装备·能力：〈金刚力〉；指令天嶮/吹鸣/光焰/森闲与「绝冲」；可吸收雷真生命力自愈｜弧光：不服主→死斗结心→损坏修复→为抚子献命→神性机巧内觉醒为人｜立场：敌视接近雷真的女性；对夏露先敌后友。

**【势力图谱】**
- **华尔普吉斯皇家机巧学院 / 学院长派**：维持夜会与学术秩序；与王室、协会博弈。
- **手套持有者与十三人**：合法暴力竞赛层；内部再分阿斯拉集团、结社安插、中立强者。
- **日本军部 + 赤羽/土门/贺茂**：输出密探与式神流派；要技术与天全下落。
- **D-works（Divine Works）**：专利、贿买、孤儿院实验；布朗森被捕后残余技术外流。
- **十字架骑士 / 罗伦斯基金会**：德意志技术收集与人体改造。
- **蔷薇师团**：最古老魔术结社，蔷薇干部无单一领袖；反政府恐怖与暗中支配并轨。

- **英国王室线（含黑太子艾德蒙、银蔷薇葛洛丽雅）**：学院政治的顶层变量。

**【世界剧情线·分卷细述】**
动画与小说前三卷构成学院入门三连。食魔者以「吞噬回路」把伦理推到台前：爱莉莎每换一次回路就抛弃旧能力，本质是把其他学生的人偶生命当一次性弹药。雷真拒绝用舆论处死夏露，坚持现场物证与尸体柜，这一选择奠定他与「制度捷径」的对立。D-works卷把战争工业搬进孤儿院：加姆系列为「新手也能军团战」而设计，失败品直接焚毁；天使系列路西法是把儿童实验数据武器化的完成态。雷真与洛基从互殴到联手，不是和解，而是共同承认「实验体也有名字」。钟楼卷把家庭伦理和政治暗杀绑在一起：安莉的自杀企图、夏露的暗杀胁迫、爱丽丝的变装双线，全部指向学院长位置的权力真空。辛以机巧士兵身份演示「无供魔仍可战」的恐怖前景，直接预告后盘神性机巧危机。

第四卷以降学院变为多国代理人战场。十字架骑士是罗伦斯基金会资助的技术收割机：用排名区段垄断制造可重复观测的实战数据。罗森堡把索菲亚等人改造成MK5，再以「希望以人类身份死去」的愿望把洛基拖入伦理泥潭——洛基成魔王的动机因此不只是修复心脏，也是拒绝再当「温柔的处刑人」。雪月花三姊妹同台后，赤羽秘传从个人体术升级为可与上位强者对拼的阵法体系；阿里阿德涅之线说明完整红翼阵曾被外部封印，军部与结社早知雷真潜力。

结社线的残酷在于「爱情手术」：奥尔嘉与威隆的记忆被反复切除，只为金白蔷薇席位不因私情失衡。爱丽丝介入手术让记忆残留，引发订婚闹剧，雷真被卷进结社联姻棋盘。阿斯拉以「魔王权限共享」招募同志，表面理想主义，实为苏美人血脉与王妃银蔷薇的政治同盟；洛基两次击败阿斯拉，完成从实验体剑帝到可裁定他人资格的权威跃迁。索涅奇卡作为俄罗斯公主参赛，耶梦加得的因果置换把「曾经搭载过的武装」变成即时军械库，代表十三人顶格的规则系战力。海赛尔的敕命诏书把语言本身变成武器，洛基以自毁听力对抗，是全书对「命令与自由」主题的暴力寓言。

终盘神性机巧危机把标题「不会受伤」反转为诅咒：追求绝对不坏的士兵，等于取消痛觉、伦理与死亡的边界。天全造神与雷真唤醒夜夜形成镜像——前者以灭族换神，后者以真名换人。协会取消雷真魔王资格而让洛基递补，揭示「魔王」是政治认证而非单纯最强者。雷真入蔷薇师团，不是投敌，而是选择在暗处继续制衡黑太子与学院长派系，世界进入「新平衡」而非童话式和平。

任务方向/奖励：活过第一周并建立「不把人偶当垃圾」的声誉；奖励＝基础回路知识、维修执照、雷真侧弱联系。

**二阶（手套资格战外围·食魔者阴影）**
切入身份/时点：契约者以夜会预备观察员、风纪辅佐或日本军部联络员切入，锚定食魔者事件发酵期、菲利克斯发布委托前后。
初始事件：你被塞进一具「失踪学生」的储物柜调查，柜中不是书，而是被吸干的夏娃的心脏空壳；柜门内侧刻着莉瑟特·诺登的名字。
开场白建议：「医务室的灯一夜没灭。走廊尽头有人把『白色幻舞』的手套放在窗台，像祭品。菲利克斯对你笑得很好看——好看到像陷阱。」
关键NPC立场：**菲利克斯·金斯佛特**——利用你当第二把刀；**爱莉莎（伪莉瑟特）**——毒舌试探；**夏洛特·贝琉**——被舆论指为食魔者；**雷真**——要证据不要谣言；**金柏莉**——要可控的真相。
主线钩子/支线：证明夏露清白或坐实嫁祸链；守住一枚未噬尽的心脏样本。支线＝宿舍搜查、伪造不在场证明、与夜夜夜巡。
危险度/规避：中——爱莉莎可吞回路；规避单独与菲利克斯进地下实验层。
任务方向/奖励：手套观察权、防御吊坠仿制图纸、食魔者案卷副本。

**三阶（D-works孤儿院·加姆废弃夜）**
切入身份/时点：契约者以十字军临时翻译、孤儿护工或洛基的「讨厌的证人」切入，锚定犹宓牺牲、加姆系列面临废弃的夜。
初始事件：下水道警铃大作，芙蕾的拉比失控吸血，布朗森的人在焚化炉前清点「失败品编号」。你必须在三分钟内选择：救犬型人偶、拍下实验证据、或把名单卖给结社。
开场白建议：「孤儿院的钟声被音压回路压成嗡鸣。芙蕾在哭，洛基在骂，雷真的血还没干。焚化炉门缝里，有一只金属狗爪还在抓。」
关键NPC立场：**芙蕾**——求你别让加姆被烧；**洛基**——不许你可怜他姐姐；**布朗森**——要毁灭证据；**雷真/夜夜**——会打穿封锁；**犹宓（若仍存）**——用命换情报。
主线钩子/支线：救出至少一具加姆核心；录下天使系列启动日志。支线＝护送芙蕾撤离、与十字架骑士前哨遭遇。
危险度/规避：高——MK级与路西法级火力；规避正面对抗完成态天使。
任务方向/奖励：音压回路残页、D-works专利黑账、洛基「三种人」名单上的暂缓处决标记。

**五阶（十三人混战·一人军队阴影）**
切入身份/时点：契约者以夜会特准观战魔术师、结社见习蔷薇或同盟国武官切入，锚定马格努斯战队出动、阿斯拉集团扩张期。
初始事件：观战席被空间转移（镰切）切成两半，火垂的压力波把石栏压成粉末；有人把你的手套（或观战牌）扔进场——按规则你已「参赛」。
开场白建议：「第一位不需要热身。六具少女型人偶同时抬头时，蒸汽管道像军舰汽笛。有人喊『伟大之人』，有人已经在算你能撑几秒。」
关键NPC立场：**马格努斯/天全**——观察雷真多于杀你；**火垂**——敌视雷真阵营；**阿斯拉·厄恩**——拉你入「共享魔王权」；**奥尔嘉**——要你别掺和结社记忆案；**雷真**——若你伤夜夜则敌。
主线钩子/支线：活着退出「误入场」；偷到战队联动用蜘蛛丝逻辑的一角。支线＝阻止阿斯拉招募、保护日轮式神阵眼。
危险度/规避：极高——五阶军团火力；规避与六具战队正面车轮战。
任务方向/奖励：军团战法笔记、空间转移坐标残片、十三人内部站队情报。

**六阶（神性机巧临界·红翼阵夜）**
切入身份/时点：契约者以魔术协会审计官、结社新蔷薇或「阻止灭世」临时契约者切入，锚定阿里阿德涅之线将断、神性机巧启动前后。
初始事件：监测仪显示魔活性不协调原理在学院上空局部失效；夜夜生命读数暴跌又暴涨。协会要你按下「清除雷真」授权，结社要你保雷真入社，洛基要你别妨碍他拿魔王。
开场白建议：「钟楼还没修好，天却先裂开一道回路纹。有人在喊抚子的名字，有人在喊魔王。你手中的清除钥烫得像夏娃的心脏——按下去，世界会安静，也可能再也没有『夜夜』这个回答。」
关键NPC立场：**赤羽雷真**——要唤醒而非清除；**夜夜**——在神性机巧内等待真名；**洛基**——接受递补魔王；**天全**——以命换抚子；**协会代表**——要稳定；**蔷薇干部**——要新平衡棋子。
主线钩子/支线：选择清除/唤醒/改写认证；保护红翼阵施术不被第三方劫持。支线＝转移安莉与平民学生、销毁多余机巧士兵胚。
危险度/规避：贴近顶点——灭世级风险；条件性胜利：断错误供魔链、保护真名应答窗口，严禁宣称「顶点已削弱」。
任务方向/奖励：红翼阵残识、神性机巧伦理白皮书、结社见习蔷薇徽或协会特赦令（互斥）。
