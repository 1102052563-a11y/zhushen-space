# 装甲骑女伊莉丝-姐妹侍奉

## 剧情

**【作品来源】**
《装甲骑女伊莉丝》（装甲騎女イリス / Soukou Kijo Iris / Armored Warrior Iris / Armored Knight Iris）为 Black Lilith（Lilith Soft 旗下品牌）于 2006 年 3 月 24 日发售下载版、3 月 31 日发售包装版的成人向 ADV（DLsite 等渠道常见编号 RJ022698；VNDB v2306），类型常被标为「娼馆监禁 ADV / 女机师沦陷」。企划·监督笹@；剧本原案·剧本そのだまさき等；原画 ZOL；装甲骑机械设定·图形 やしガニ / チームやしガニ。主角声优：春日アン（伊莉丝）、鸟瀬くるみ（梅莉）。MangaGamer 于 2014 年推出英译下载版（Armored Warrior Iris）。PIXY / Anime Antenna Iinkai 于 2007–2009 年推出同名成人 OVA 四卷：vol.01《超空の降下作戦》、vol.02《娼館の女戦士》、vol.03《戦士から悦虐娼婦へ》、vol.04《被虐戦士の生還》。系列核心命题是「美少女装甲骑驾驶员在敌后被俘、商品化、制度性磨损」与搭档羁绊能否撑过敌后。**本档案专写「姐妹侍奉」分支：伊莉丝·雷贝尔与梅莉·纳セリ双俘后被编为联锁「侍奉单位」——互为人质、外交观礼展示、交换谈判与条款博弈。档案只写组织、战争、俘获、联锁协议、观礼政治与交换条款，不写色情细节。** 与同系列边界：单人沉没见「骑女沦陷」；梅莉夺还见「妹妹援救」；双人尽没绝望见「双堕深渊」；机体强制融合见「魔铠融合」；单人叛旗见「堕落魔将」。**本线独特性＝互锁人质＋礼仪展示＋「只换一个」外交。**

**【世界定位】**
本世界锚定**第 101 双王牌同时或先后被俘后的联锁侍奉编制**：敌方不以「单独拍卖一人」为终点，而以成对爆弹项圈、情绪同步芯片与礼仪装甲骑双人程序，把她们绑成一对可观礼、可议价、可巡回展示的「侍奉单位」。逃一炸双，展示时互为担保，谈判桌常提出「只换一个」。契约者要解的是**条款、钥匙与观礼转播权**，不是单一反派。主题句：**锁住你的不是项圈，是你还在乎的人。**

**【世界观 · 力量体系】**
远未来，人类离开地球向星际扩张，与多种族组成松散的**宇宙联邦政府**（Space Federation）。联邦对行星自治的控制力有限：法律覆盖的都市与反联邦都市可同处一颗行星，边境治安靠军警与特种部队维持。联邦保安军 **SSF**（资料中亦写作 Space / Soviet Security Force 等转写，以「联邦保安军」统一指称）负责治安维持、反奴隶贸易与区域冲突介入。核心兵器是人型机动兵器**装甲骑**（Armored Knight / Armored Warrior，ADT 系列等），由驾驶员神经链接操作；驾驶员在机内可完成据点攻坚与街区级机动打击，**离机后几乎等同受训特种兵，无法与完整装甲骑正面对抗**。

装甲骑战斗的实际表现决定乐园映射：正役特种机可机动摧毁装甲目标、突入都市与轨道设施、突破音障级机动并掀平建筑群——对应图鉴四阶「摧毁据点 / 掀平建筑群」。中队协同、敌方主力舰炮、重型异种战力与军阀私兵装甲骑可把毁伤抬到四阶中上～四阶顶。本世界**没有**稳定的行星爆破级战力作为日常棋子；叙事顶点（ミリオン军阀核心、联锁侍奉完成态、观礼武装护卫）仍落在四阶顶～五阶门槛之间。**本条目按四阶世界处理**，禁止以六阶以上外挂入场秒杀。

**战力阶梯（破坏力对齐《阶位战力图鉴》，本条目仅开放四阶切入）：**
1. **非武装平民 / 被俘缴械士兵**：徒手与冷兵器级，≈一阶及以下；战场上面对装甲骑无有效反抗。
2. **常规步兵 / 轻型载具 / 百万都市奴隶警备 / 观礼门禁部队**：可清据点杂兵，≈二阶～三阶边缘。
3. **SSF 特种装甲骑驾驶员（伊莉丝·雷贝尔、梅莉·纳セリ等）驾驶正役机**：据点攻坚、建筑群级毁伤，典型≈**四阶**。
4. **装甲骑中队协同 / 敌方主力舰炮 / 重型异种战力 / 军阀私兵装甲骑 / 宴会轨道站护卫机群**：街区级毁伤，≈四阶中上～四阶顶。
5. **本世界叙事顶点·联锁侍奉完成态**（双机共享供能与爆弹协议、礼仪装甲骑双人程序、观礼武装与转播护卫）：协同输出≈四阶顶，战术自由度≈零；仍按四阶世界结算。

**特殊系统与死亡/俘获规则：**
- **神经链接驾驶**：装甲骑战力绑定驾驶员适配与机体完整度；机体大破后驾驶员可能被弹射或拖出，进入「离机即降阶」状态。
- **爆弹项圈（恶魔项圈）**：敌后制度核心。逃离或强拆失败即死，把特种兵最擅长的「硬突围」改写成自杀；营救方若不知频率，强突等于杀死人质。
- **联锁侍奉协议（本分支核心）**：双爆弹共享状态机＋情绪同步芯片＋礼仪装甲骑双人程序。规则要点：A 逃则 B 炸；B 近解锁则 A 被转移或示警；分离距离超限则双处决；观礼违仪则扣对方维修/食水配额。任何单线英雄主义都会被制度反噬。
- **奴隶拍卖与成套产权晶片**：ミリオン把战俘变成可交易动产；成套俘获的回报指数级高于单俘，因毁灭的是「装甲骑女」品牌。
- **外交观礼制度**：中立种族与埃尔芬使节被邀观看「侍奉礼仪」——权力羞辱被国际化。使节鼓掌＝承认族人可被礼仪化；退席可能连累族内人质。
- **单换谈判条款**：联邦/埃尔芬常提出「只换一个」；二人互相拒绝成为「被留下的那个」。交换文书、蜡印与议会预算共同决定窗口期。
- **肉体改造订单**：原作以「客人订单」改写身体规格。档案只记制度功能——驾驶适配下降、识别码失效、无法再合法归队——不写过程。
- **宣传符号战**：装甲骑女既是精锐也是广告；联锁展示可被剪成「联邦英雄互为人质」的舆论武器。
- **双重人格/记忆封锁（梅莉—リルル线）**：原作中梅莉以「リルル」人格作为创伤防御机制出现；本分支中可作为联锁后门的钥匙或观礼场的身份污染源，不把双堕绝望写满主轴。
- **情绪同步芯片**：把二人的应激、恐惧与战斗冲动采样对齐；暗语「金刚—月影—同频」可短暂降低误爆概率，亦可能被敌方读取为「仍可利用的羁绊」。

乐园阶位映射：平民≈一阶；常规军≈二～三阶；装甲骑女主战力≈四阶；联锁侍奉完成态≈四阶顶控制向（宁低勿高）。超阶巅峰不适用；若遇完全支配/联锁完成体，按四阶 Boss＋条款/钥匙条件性胜利处理，严禁战力归零解释。

**【地理 · 舞台】**
- **行星レフト（Left）**：极端女性蔑视风习残留，人身买卖网络盘根错节。伊莉丝与梅莉日常任务是强袭并扑灭奴隶贸易节点。猎奴成功时她们是秩序的锤；猎奴失败时同一套物流链会把她们当货物。レフト不是单一国家，而是联邦辖区与反联邦都市交错的殖民地行星——同一片大气下并存两套法律与两套价格表。
- **联邦辖区都市群**：受联邦法律保护，与レフト上反联邦都市对立。宣传部在这里循环播放装甲骑女的胜利影像，与敌后现实形成时间差。联锁展示外泄后，这里的广告牌会紧急撤换，或被敌方伪讯反向覆盖。
- **区域冲突前线 / ホリック降下作战空域**：联邦对反联邦都市联军的空降与突入战区。对空网、诱饵航迹与兽人清道夫共同把「特种作战」改写成「俘获经济」。本分支中，这里还埋着诱导第二只王牌入瓮的伪信标或双机同时击落点。
- **敌对势力都市ミリオン（Million）·通称ダークタウン**：黑市、军阀、奴隶拍卖与娼馆经济共生的暗黑都市。权力以竞价、编号与项圈频率运行，而非军法。双俘后的**双囚层**在此：两人可互相可见，却不能无代价触碰。
- **SSF 第 101 特殊装甲中队基地**：通称「装甲骑女」部队驻地，女队长编制，最新锐装甲骑列装。食堂空座、机库空位与宣传片同框；双俘消息后分成「沉默桌」与「吵闹桌」——前者仍摆两人的杯子，后者争论该救谁。
- **奴隶拍卖会场 / 成套打包区**：商品化仪式的中心舞台。本分支强调「成套藏品」专场与博兹ク打包合同。
- **礼仪厅 / 宴会轨道站（观礼主舞台）**：本分支独有核心地理。中立使节、埃尔芬观礼席、联邦观察员与转播机位在此共处。灯光与重力场为镜头服务；桌下可摸枪，桌上仍奏乐。
- **联邦「只换一个」谈判桌 / 交换窗口舱**：预算、舆论与种族政治在此写成条款。空白交换纸比枪更烫。
- **金刚/月影拆件库与敌方研究所**：解析 ADT 黑匣子、联锁芯片、项圈频率与驾驶适配；双黑匣子若同时流入，敌方装甲骑技术可能跃进一代。拆件本身是谈判筹码——「不签就拆她的机」。
- **タゾック邸等レフト黑产宅邸**：战前猎奴名场面发生地；战后残党可在观礼或拍卖以报复性竞价/观礼赞助现身。
- **埃尔芬聚居舱 / 族使驻地**：族内就是否「只救本族」分裂；聚居舱的微观政治决定契约者募集帮手的难度。

本分支额外强调：**宴会轨道站礼仪厅**、**双囚层互见结构**、**交换窗口舱**、**成对解码钥保管点**、**转播延迟控制室**。

**【世界剧情线】**

① **双王牌神话：月影与金刚的宣传姐妹**  
人类扩张至星际后，宇宙联邦政府以多种族松散结盟维持秩序。联邦保安军 SSF 第 101 特殊装甲中队因女队长与几乎全女队员、列装最新锐装甲骑，被通称为「装甲骑女」。中尉**伊莉丝·雷贝尔**（地球人）驾驶 **ADT-74 月影（ゲツエイ；英译亦作 Griffin）**，搭档**梅莉·纳セリ**（埃尔芬族，外表幼小而实际年龄超百岁）驾驶 **ADT-73 金刚（コンゴウ；英译亦作 Chimera）**。

联邦宣传把两人打造成不可分割的「姐妹」符号：人类热情与埃尔芬理性、月影与金刚、护短与吐槽、正义冲动与传感器推演。真实关系是互补战友与深度羁绊，并非血缘姐妹；但广告与模型包装需要「姐妹」二字，殖民地学校的「女英雄日」也靠这套符号运转。レフト猎奴影像被剪成英雄广告——伊莉丝踹开黑产宅邸大门、梅莉在通讯里骂她「笨蛋却偏要把机体挡在我前面」。神话越亮，成套俘获越值钱。对ミリオン军阀与黑产联盟而言，俘获一名王牌是利润；俘获两名王牌并编成**联锁侍奉单位**，是政治武器与外交展示资产。

两人在惑星レフト反复强袭人身买卖网络：扑灭奴隶贸易既是任务，也是「对恶人可以冷酷到底」的正义出口。日常任务节奏可复原为「情报切入—突入—清点—移交—宣传剪辑」五段。伊莉丝把「对恶人可以冷酷」当作职业伦理；梅莉以埃尔芬式长期视角记录对空网、走私合金与项圈黑市原型。这些参数在ホリック之后全部变成联锁协议与交换条款的变量。

② **区域战争前夜：符号被磨成刀，饵俘预案启动**  
レフト上反联邦都市结盟，对联邦侧都市发动区域战争。战争经济与黑产共生：俘虏、残骸零件、识别码与宣传素材都可出售。第 101 中队被派往前线时，伊莉丝与梅莉已不只是战术单位，更是联邦「女性精锐可镇压黑暗」的活广告。敌方从一开始就把「猎杀双王牌」写成可回收资产项目：以一人诱捕另一人，或同时击落，再由博兹ク体系打包定价。

战争前夜的情报战同样关键：反联邦侧释放虚假航迹，诱导联邦把装甲骑女投入对空网最密的空域；联邦宣传部则要求「必须有影像可剪」，逼中队在战术上多做一次高风险过顶。伊莉丝的护短伦理与梅莉的传感器谨慎在此出现裂痕——前者要先救人，后者要先确认诱饵。裂痕本身不是背叛，却在ホリック日成为可被敌方利用的缝。

③ **ホリック降下作战：秩序崩溃与双俘窗口**  
ホリック降下作战中，先行部队暴露，空降队遭对空炮火。对空网强度超预期；梅莉机偏航或陷入不利射界。伊莉丝为掩护梅莉机，切断自动回避系统改全手动，把自己机体滑入射线上——这是她「孩子气却绝对护短」的驾驶伦理，也是战术上把自身暴露给敌方火力与俘获链的关键失误点。激战中两机先后重创。

本分支关键分叉有二：
- **同时击落**：两机残骸落入同一清道夫分账区，オーク战场清道夫与奴隶商当场打包；
- **饵俘**：先俘伊莉丝，保留半真半假信标（月影残核辐射、私人短码、「她还在等你」通讯残片），诱导梅莉违令前来，完成成套俘获。

无论哪一种，结果相同：两名中尉进入ミリオン物流链。军籍徽章被剥，机体拖入拆件库，**成对爆弹项圈**安装。猎奴者成为同一物流链上的成套货物——结构性反讽完成。

④ **打包合同：从战俘到侍奉单位**  
博兹ク（ボズク）一类奴隶商人/娼馆利益链人物不只做单人拍卖，而推动**联锁侍奉打包**：把「联邦双王牌」写成可观礼、可议价、可巡回的长期资产。军阀情报部与改造术者介入——他们发现战术价值与政治羞辱价值高于普通拍卖：若让两人在中立使节面前以「侍奉礼仪」并肩而立，联邦士气与同盟信用可断一截。

打包合同签字后，改造工房完成三件套：
1. **双爆弹联锁**（逃一炸双、距离锁、抗命脉冲）；
2. **情绪同步芯片**（应激对齐、误爆抑制与羁绊遥测）；
3. **礼仪装甲骑双人程序**（观礼姿态、应答脚本、协同供能接口）。

逃跑＝杀死同伴；违仪＝扣对方维修时间与食水；单方加速＝脉冲警告。战术自由度被制度压到接近零，但协同输出仍可达四阶顶——她们打得动四阶目标，却不能朝「回家」的方向飞。

⑤ **联锁协议的残酷：决策瘫痪**  
联锁残酷在决策瘫痪：任何突围先过「同伴是否同意死」。伊莉丝的一人突击被克；梅莉的最优解变成牺牲函数。每日侍奉演练扣对方维修时间，二人积「容忍点」——羁绊亦腐蚀。伊莉丝愿替死，用军人谈判语言要条款；梅莉愿用技术换对方活，却成为埃尔芬政治负担。敌方允许她们进行一次「只能活一个」的虚假谈判——真正目的往往是**录下谁求谁死**，剪成「联邦英雄互卖」纪录片，或在观礼前夜播放以摧毁士气。

暗语「金刚—月影—同频」可短暂对齐情绪同步芯片、降低误爆；但暗语本身若被采样，会成为敌方证明「她们仍有利用价值」的证据。契约者切入时，最可玩的是**打破虚假二选一**——同步解锁、改条款为双换、假同殉真逃生、宴会反咬——而不是站队谁更该死。

⑥ **观礼展示：羞辱国际化**  
中立种族与埃尔芬使节被邀观看「侍奉礼仪」。宴会轨道站的灯光为镜头而生；请柬、蜡印、座席图与转播延迟秒数构成权力语言。使节鼓掌＝承认族人可被礼仪化；退席可能连累族内人质，也可能成为契约者策反的第一火种。联邦观察员被要求「保持外交礼貌」——等于被要求在镜头前默认联锁合法。

观礼不是单纯羞辱，而是**外交资产证券化**：军阀借此招股战争经济；タゾック残党可作报复性赞助；伪讯剪辑主管握母带与声纹库，是隐形 Boss。伊莉丝与梅莉若被拍到互相搀扶，会被剪成「联邦已跪」；若被拍到互相冷漠，会被剪成「英雄互卖」。无论哪一种，都是叙事武器。舆论力学清晰：展示成则联邦征兵降、黒产升；反转则ミリオン可能遭定点清除。常赢在转播权与条款，不在巷战火力。

⑦ **单换谈判：空白交换纸**  
联邦与埃尔芬常提出「只换一个」。动机各异：联邦预算与舆论只能消化一名「污染资产」归队；埃尔芬族使可能主张优先撤本族成员梅莉——这一提案会摧毁伊莉丝对盟友的信任，也会在殖民地被解读为「种族优先高于搭档道义」。二人同时拒绝成为「被留下的那个」：伊莉丝拒当被留下者，梅莉拒族人只救自己。

谈判桌上常同时出现三份草案：只换伊莉丝、只换梅莉、以及一张空白纸。空白纸是契约者的；撕毁单换草案会立刻招来狱卒长与谈判官的火力——四阶战斗可在宴会桌下开始，桌上还在奏乐。交换窗口通常只开一次（约观礼后 12 小时内）；未解联锁则进入「永久展示巡回」。

⑧ **钥匙、后门与第三只手**  
解码钥分主从：主钥在改造术者，从钥在双囚层狱卒长，传说第三钥在联邦「双驾驶员共死」弃案档案中——联锁协议本身可能抄自联邦旧弃案。梅莉可能在联锁固件里留有仅对伊莉丝生效的后门：双人同意则联锁失效——但「同意」本身会被情绪同步芯片记录，需契约者充当**第三只手**在观礼转播延迟窗口同时转双钥。

集齐两钥才能同生；只有一钥则被迫进入一人活路线的道德酷刑。假同殉真逃生需要传感器回波伪造与距离锁欺骗；宴会反咬需要藏匿武器＋转播延迟秒数。政治后果最大的是反咬：宴会死者名单可能包括中立使节，引发更大战争。

⑨ **联邦与中队平行线：沉默桌与吵闹桌**  
第 101 食堂在双俘消息后分裂：沉默桌仍摆两人的杯子；吵闹桌争论该救谁。队长拒签「弃一人」令，却可能被议会预算绑架。宣传官要「英雄不死」；情报官可能压住观礼实况以免士气崩溃；鹰派要切割「污染资产」；鸽派要活捉研究逆转。埃尔芬聚居舱有人主张弃人类搭档，有人记得梅莉自己选的姐妹。这些微观政治决定契约者募集帮手的难度。

联邦侧最危险的不是冷漠，而是**切割算法**：若承认双俘，征兵与同盟信用受损；若坚持阵亡，交换窗口就没有合法名分。「救回」与「归队」是两个不同任务；同生回归后还有「侍奉污染」审查庭——问的不是战斗数据，是「谁先求饶」。

⑩ **リルル冲击与身份污染（本分支弱化但保留）**  
容貌酷似梅莉、性格迥异的**リルル**可能出现在双囚层或观礼后勤链：若把リルル当假货，可能错过唯一内应；若把リルル当真梅莉，可能被伪自愿结构利用。当「第三张同脸」进入转播，联邦识别系统置信崩溃；埃尔芬可能宣布「梅莉已死、影像皆伪」以止损——救身体与救法律身份分裂。本分支把リルル定位为**联锁后门钥匙 / 观礼场身份污染源**，不把双堕绝望写满主轴。

⑪ **终局四路（选路即选战后地图）**  
- **同生解联锁**：双人活；审查风暴；军阀丢面子；契约者需提交联锁非法证据与谈判官回扣，审查可转为控诉军阀的国际案。  
- **同殉拒展示**：符号殉道；两边宣传战升级；中队神话以「共死」定格，但活人政策失败。  
- **一人活背债**：幸存者线＋仇恨线；幸存者申请永不编入双人任务——创伤，也是可玩后续钩子。  
- **宴会反咬屠场**：政治后果最大；中立使节死伤可能引发更大战争；转播母带成为新战场。

契约者应把**条款与钥匙**当 Boss，而非单一打手。四阶稳玩法是经济/外交刺客：伪造交换文书蜡印、夺转播延迟、策反使节退席、集齐双钥、触发后门——而不是在礼仪厅正面对轰。

⑫ **与系列其他分支的边界**  
本分支强调**互锁＋外交人质＋观礼展示**；禁止写成：单人娼馆日常为主（骑女沦陷）；梅莉单线夺还爽文（妹妹援救）；双人尽没无条款博弈的绝望复读（双堕深渊）；驾驶员与机体/异物强制合体（魔铠融合）；伊莉丝穿敌方将旗叛旗作战（堕落魔将）。姐妹侍奉的独特性在于：**逃一炸双的联锁**、**观礼把羞辱国际化**、**「只换一个」的道德酷刑**、**同生需要第三只手**。

## 阶位切入点

> 阶位↔原作：四阶≈装甲骑王牌驾驶员与主战机体交战级（掀平建筑群/据点攻坚）；联锁侍奉完成态≈四阶顶控制向。世界顶点为联锁协议＋观礼武装＋转播护卫，按四阶顶处理。低配切入禁止开五阶以上外挂；**情报、条款与钥匙优先于蛮力**。世界无超阶巅峰；若遇联锁完成体，按四阶 Boss＋条件性胜利（解联锁/改条款/夺转播），严禁「被封印/削弱」式战力归零。乐园阶位映射与剧情条目一致：平民≈一阶；常规军≈二～三阶；装甲骑女主战力≈四阶；联锁侍奉完成态≈四阶顶。

**四阶（姐妹侍奉 · 联锁夺还与观礼条款线）**

切入身份/时点：契约者以 **中立使节随员**、**联邦交换观察员**、**梅莉族内临时翻译**、或**宴会轨道站后勤技师**身份切入。锚定**观礼前 6 小时至交换窗口开启**（T-6h～T+12h）——联锁手术与礼仪程序已完成，成对解码钥尚未集齐，单换草案已起草。使节随员可碰请柬与座席图；交换观察员可碰空白交换纸与蜡印；族内翻译可进埃尔芬观礼席与聚居舱；后勤技师可获转播延迟控制台或维修工单三十秒级接触。四种身份都会在观礼前被四方同时试探：联邦要你「可交代的一人」；博兹ク要你「守场子」；改造术者要你「别碰固件」；第 101 要你「双人同归」。

初始事件：你被同时塞来**两份交换草案**——一份只换伊莉丝，一份只换梅莉——外加一张空白纸。宴会轨道站灯光已为镜头调好；项圈灯同色闪烁。伊莉丝与梅莉并肩而立，用中队旧暗语（灯光间隔/唇语）同时对你说「拒绝」。第三份空白纸烫在你手里：签任何一份单换，另一人项圈进入「弃子脉冲」预备；当众撕毁，则狱卒长与谈判官的火力会在桌下升起，而桌上乐队仍在奏下一曲。倒计时内你必须决定：双换条款、假同殉、夺钥、策反退席、或沉默——沉默超过一个祝酒轮次，联邦谈判官会默认你接受「只换一个」，埃尔芬使节则可能单独签字只救梅莉。

开场白建议：「宴会轨道站的灯光为镜头而生。伊莉丝与梅莉并肩而立，项圈灯同色闪烁。有人举杯祝贺『姐妹侍奉』，有人在桌下摸枪。你的通讯器里叠着两份单换草案和一张空白纸——空白纸比枪更烫。你数过项圈闪烁的间隔，发现她们在用灯光说话：联邦的灯语教材里没有这一章。」

关键NPC立场：  
- **伊莉丝·雷贝尔**：拒当被留下者；仍以军人逻辑谈判；对「买下/单换」感到羞辱，但可能利用观察员作为突围道具；最怕你是联邦来毁证或逼她求梅莉死的人。  
- **梅莉·纳セリ**：拒族人只救自己；要技术解法与双人同生；会用传感器验证你是否说谎；后门只在双人同意时对你开启。  
- **联锁协议设计师 / 改造术者**：要协议完美与零故障观礼；你碰固件即标杀。  
- **联邦谈判官**：要可交代的一人；空白纸的敌人；可能有回扣与「不便迎回」预案。  
- **埃尔芬观礼使**：要族誉；可能单救梅莉；怕鼓掌等于承认礼仪化；可退席。  
- **博兹ク**：要违约金与场子信誉；守规矩加价可暂时当你客户。  
- **双囚层狱卒长**：握从钥；赌债/回扣可勒索。  
- **第 101 中队队长 / 宣传官**：要双人同归或至少不要丑闻；可能事后把你当弃子。  
- **リルル**（若在场）：假解锁顺序或真内应；不可仅凭外貌信任。  
- **中立使节**：可录下联锁非法证据；也可在反咬中成为政治炸弹。

主线钩子/支线：  
主线＝集齐成对解码钥、改条款为双换、观礼夜解联锁或假同殉撤离、决定终局（同生 / 同殉 / 一人活 / 宴会反咬）。  
支线①：使节录下联锁非法证据，把审查转为国际控诉。  
支线②：伪造「只换梅莉」文件激怒窗口期，逼出第三钥或谈判官回扣。  
支线③：说服埃尔芬公开退席，打断观礼合法性。  
支线④：狱卒长赌债把柄，换从钥三十秒。  
支线⑤：修一机（月影或金刚残件）作逃脱载具。  
支线⑥：触发梅莉后门——双人同意则联锁失效（需你当第三只手）。  
支线⑦：阻止联邦「不便迎回」暗杀。  
支线⑧：假同殉真逃生（伪造传感器回波骗过距离锁）。  
支线⑨：夺转播延迟令牌，控制反咬或羞辱片是否外泄。  
支线⑩：同生后护送通过「侍奉污染」审查庭。

危险度/规避：危险度**极高**。四阶顶协同输出＋联锁双爆＋观礼武装护卫并存。必须规避：①触发双爆（逃一炸双/盲拆/距离锁）；②在签字前默认单换导致弃子脉冲；③镜头前暴露乐园外挂被制成新素材；④把リルル当梅莉本体公开点名触发身份污染升级；⑤宴会反咬误杀中立使节引发全面战争。世界顶点级联锁完成体与改造术者**情报·条款优先**，不硬刚。

任务方向/奖励：目标自然落在「撕开双换口 / 解联锁 / 夺转播与非法证据」。奖励取向贴合四阶：成对解码钥或频率片段、情绪同步芯片、埃尔芬血誓卷、转播令牌、审查豁免、打包合同与回扣证据、伪军籍芯片工艺、维修工单所揭内部地图。禁止发放可越阶毁城的战略兵器。互斥奖励：审查豁免与战争升级线索不同时满额发放；一人活终局的仇恨线与同生终局的双人羁绊线互斥。

开局节奏建议（本阶独有事件链）：第 1 回合确认三份草案真伪并选定公开身份；第 2 回合必须回应双人暗语灯光——你的沉默会被读成「接受单换」；第 3 回合接触主钥或从钥之一，制造不对等谈判；观礼祝酒轮次内决定撕毁/改写/假签。成功撤离的最低证据包是：双人存活信号（或合法同殉记录）、联锁固件版本号、至少一枚解码钥齿痕或后门日志、转播母带未遭羞辱剪辑证明——四者缺一，联邦仍可能按「任务失败/目标已污染」切割你。撕毁单换草案后三分钟内必须撤离主桌或转入已预谋的桌下战场。

补充场景钩（写入开局即可落地）：维修工单号、项圈固件版本、转播延迟秒数、请柬名单、暗语灯语间隔、蜡印齿痕、食水配额扣减记录——任取其二。
