# 王国之心
<!--meta lib=主库 tiers=一、二、三、四、五、六、七、八-->

## 剧情

**【作品来源】**
史克威尔艾尼克斯与华特迪士尼公司合作、野村哲也监督的动作角色扮演系列《王国之心》（キングダム ハーツ / Kingdom Hearts）。2002 年 PS2 首作起，经《记忆之链》《II》《358/2 天》《梦中降生》《编码》《3D 梦降深处》《χ/Union》《III》《记忆旋律》等，构成「暗之探求者」长篇。文风：迪士尼童话外壳 + 心之形而上学 + 友情羁绊。卡片锚点：钥匙刃使者索拉穿梭各世界，封印钥匙孔，对抗无心者与 XIII 机关。

**【世界定位】**
多元世界（迪士尼与原创界）漂浮于光暗之间。主角**索拉**自命运岛出发，与**里克**、**卡伊莉**分离后获钥匙刃，与**唐纳德**、**高飞**同行，寻找朋友并阻止**安森·席德**与**马列菲森特**夺取「王国之心」。系列后扩展至**泰拉/菲恩图斯/亚克雅**前史、**洛克萨斯**与 XIII 机关、**赛亚诺特**夺心计划，直至《III》收束暗之探求者篇。

**【世界观 · 力量体系】**
「心」是存在核心：失去心→无心者（Heartless）；身体与魂残留可成有心者（Nobody）。**王国之心**是心之集合的终极光。**钥匙刃**选人，可封印世界钥匙孔、战斗与开门。移动靠古米船、黑暗回廊等。战斗为动作 RPG：连段、魔法、召唤、反应指令（KH2）、形势变化等。

战力阶梯：
1. 普通无心者杂兵、世界平民 ≈ 一阶。
2. 大型无心者、迪士尼反派爪牙 ≈ 二阶。
3. 世界级中 Boss、新晋钥匙刃使 ≈ 三阶。
4. 熟练钥匙刃使（索拉中期）、机关末座 ≈ 四阶。
5. XIII 机关干部、里克（反英雄期）≈ 五阶。
6. 机关上位、安森残影、大师级候补 ≈ 六阶。
7. 钥匙刃大师、赛亚诺特完整计划体、χ时期强者 ≈ 七阶。
8. 王国之心本源争夺、大师赛亚诺特终局、光暗平衡级 ≈ 八阶（本世界最高覆盖；再上不硬凑九阶）。

乐园阶位映射：杂兵无心者≈一阶；中型威胁≈二阶；世界守卫战≈三阶；钥匙刃中坚≈四阶；机关干部≈五阶；上位干部/暗之强者≈六阶；大师级≈七阶；王国之心终局争夺≈八阶。宁低勿高；顶点条件性胜利。

**【地理 · 舞台】**
命运岛、迪士尼城、穿越式迪士尼世界（惊奇、奥林匹斯、深海、万圣节等）、无心者世界、忘却之城、黄昏镇、虚无世界、光之花园、沉睡世界、日之终焉等。卡片常用：命运岛分别 + 穿越封印钥匙孔。

**【世界剧情线】**
① **KH1：心之门**  
命运岛被暗吞噬，索拉与里克、卡伊莉分离。索拉获钥匙刃，与唐纳德、高飞寻王。马列菲森特集七位光之公主开暗；安森·席德利用无心者。终盘索拉为救卡伊莉暂时失去心，成无心者；里克与王关闭门。索拉恢复后与里克、王于光暗夹缝重逢——衔记忆之链。

② **COM / 358/2：记忆与机关**  
忘却之城：索拉记忆被改写，**娜米妮**、**组织成员**介入；里克侧线对暗。同步时间线：**洛克萨斯**作为索拉的 Nobody 在 XIII 机关度过 358 天，与**阿克塞尔**、**希恩**建立羁绊，终为「存在」离开机关，被收束回索拉。

③ **KH2：有心者**  
索拉沉睡后于黄昏镇醒来，对抗无心者与有心者。XIII 机关要拼凑王国之心；洛克萨斯抗争后与索拉融合。终盘虚无世界，击败机关成员与赛亚诺特相关存在，索拉与里克归途。

④ **BBS：十年前**  
大师艾拉库斯门下泰拉、菲恩图斯、亚克雅寻失踪的大师赛亚诺特、灭「无知」。泰拉被暗侵蚀成「泰拉-赛亚诺特」；菲恩图斯心入索拉；亚克雅沉暗之世界。设定索拉为何能用钥匙刃、里克的暗、卡伊莉的公主之光。

⑤ **Coded / DDD / KH3：收束**  
数据索拉解吉米尼日志谜题；梦降深处索拉与里克考大师试炼，遇青年赛亚诺特与时间机关。KH3：为救被心溶解者集七光再封，迪士尼世界再访，终战光暗，暗之探求者篇收束；Re Mind 与记忆旋律延伸凯莉等后日谈。

⑥ **日常与可改**  
古米船航线、世界钥匙孔封印顺序、迪士尼角色委托、机关观测者、记忆瓶。可改局部世界顺序与谁先救；不可无代价改：心与无心者规则、钥匙刃选拔、赛亚诺特长计、索拉-洛克萨斯关系核。

**【主要人物】**
- **索拉**｜性格：阳光、重友情、少想哲学却能以心突破。｜装备·能力：钥匙刃、魔法、召唤、驱驶形态。｜弧光：岛民少年→钥匙刃英雄→多次失去与夺回。｜立场：光侧核心。
- **里克**｜性格：好强、易堕暗亦能自拔。｜装备·能力：钥匙刃、暗之路。｜弧光：岛→暗→守护者。｜立场：索拉半身。
- **卡伊莉**｜性格：温柔坚韧。｜装备·能力：光之公主、后亦钥刃。｜弧光：被掳→心之锚。｜立场：羁绊核。
- **唐纳德 / 高飞**｜性格：吵闹忠心 / 迟钝可靠。｜能力：魔法与盾。｜弧光：王的使者全程同行。
- **洛克萨斯**｜性格：冷淡求真。｜能力：双钥匙刃。｜弧光：机关→融合索拉。
- **阿克塞尔 / 雷亚**｜性格：毒舌重情。｜能力：火枪轮。｜弧光：机关→友。
- **希恩**｜性格：记忆造物的悲哀。｜能力：钥刃。｜弧光：存在证明。
- **泰拉 / 菲恩图斯 / 亚克雅**｜性格：倔强/纯真/认真。｜能力：钥刃大师候补。｜弧光：BBS 悲剧三角。
- **赛亚诺特 / 安森·席德**｜性格：探求心之闇。｜能力：暗、钥刃、组织。｜弧光：系列黑手。
- **马列菲森特**｜性格：野心女王。｜能力：暗绿火。｜弧光：KH1 主反派之一。
- **米奇国王**｜性格：王者担当。｜能力：钥刃。｜弧光：光侧领袖。
- **莉雅 / 伊安佐 / 艾文等**｜黄昏镇与科学侧配角，各有立场。

**【势力图谱】**
光之守护（迪士尼城、钥刃使）、无心者军团、XIII 机关/真 XIII、马列菲森特同盟、大师与预知者（χ 时代）、世界当地王权（迪士尼各作）。

**【贵重物品】**
钥匙刃诸型、王者与英雄之证、光之公主力量、心之瓶/记忆瓶、古米船零件、黑外套、机关徽章、χ 之钥等。

**【隐藏剧情 · 伏笔】**
菲恩图斯心在索拉；卡伊莉心曾藏索拉内；希恩与娜米妮记忆链；赛亚诺特多体与时间旅行；钥刃战争与χ 前史；「连接」主题贯穿。

**【大事记时间线】**
钥刃战争（古）→ 黑暗之路/赛亚诺特少年 → BBS → KH1 同步 0.2 → COM 与 358/2 → KH2 → Coded → DDD → KH3 → 记忆旋律 →（IV 待定）。

**【叙事基调 · 雷区】**
友情与心之诗；忌把索拉写成无脑；忌低阶灭赛亚诺特本体；忌无视迪士尼世界「不干涉日常」潜规则；忌用修仙词替换心/钥刃/无心者。最早切入：命运岛异变当夜。

**【扩写档案·本世界独有】**

**【世界剧情线·KH1 细部】**
命运岛的木剑决斗与「小筏计划」建立三人羁绊。风暴夜无心者入侵，索拉落入穿越点，在迪士尼城接受王的委托：寻王、寻友、封印钥匙孔。各世界流程固定为「当地故事被无心者/马列菲森特扰乱→索拉介入→Boss→钥匙孔」。光之公主被集中时，安森的真实目的逐渐压过女巫同盟。终盘门之世界，索拉刺自己救卡伊莉成无心者的场面是系列情感顶点之一；里克与王关暗之门的决定把「胜利」写成分离。

**【世界剧情线·COM 与 358/2 细部】**
忘却之城以卡牌规则具现记忆：房间=世界卡片，打出=行动。组织成员轮流试探索拉；娜米妮被迫改写，使「友谊」本身成可编辑对象。里克侧线用暗对抗暗，完成「不靠光也能守护」。洛克萨斯在虚无世界起床，被交给阿克塞尔「照看」，每日任务杀无心者收集心——他不懂心却在堆心。希恩的出现让「复制的感情是否真实」成为主题；希恩消散、阿克塞尔背叛组织、洛克萨斯与幻影索拉对决后被收束，是 KH2 开场的钥匙。

**【世界剧情线·KH2 细部】**
黄昏镇的虚拟暑假被无心者打破，迪兹引导索拉苏醒。有心者与无心者双线敌人让战斗语言更新。组织成员各自世界设伏：某些成员可笑可恨，赛亚诺特计划则始终冷酷。洛克萨斯序章让玩家先当「另一人」再当索拉，融合后能力与回忆回流。终盘虚无世界连战是干部点名册；胜利后沙滩重逢把 COM 的分离暂时缝合。

**【世界剧情线·BBS 细部】**
外岛的训练、大师考核、赛亚诺特失踪报告。泰拉被暗诱，菲恩图斯被保护欲驱动，亚克雅被责任压弯。关键因果：菲恩图斯之心入索拉，使索拉能挥钥刃；泰拉之躯成赛亚诺特容器；亚克雅沉暗之世界等待（0.2 线）。马列菲森特与安森的早期接触把 KH1 反派接到前史。

**【世界剧情线·DDD 与 KH3 细部】**
沉睡世界的「梦的逻辑」允许时间机关插手；青年赛亚诺特与「真实组织」布局七心与十三暗。KH3 重访迪士尼世界不仅是怀旧，更是集「守护的光」。终战规模从个人剑斗抬到光暗军势；卡伊莉被消引发索拉逆行规则的代价，在 Re Mind 中被加厚。记忆旋律以凯莉视角整理系列记忆，是元叙事收束。

**【主要人物·补档】**
安森·席德的科学家冷感与钥刃大师野心；马列菲森特的同盟破裂；莉雅（娜米妮完成体）的记忆管理；伊安佐、艾文、艾拉库斯、耶那·席德等科学家/大师层；迪士尼侧米奇、米妮、皮特、迪士尼反派群——各世界具名按该世界原作。

**【地理·补档】**
无心者世界的崩坏美学；忘却之城无限走廊；黄昏镇钟塔与火车；虚无世界的白荆棘；光之花园与日之终焉的时间夹层；χ 时代的日之终焉前史都市。

**【贵重·隐藏补档】**
驱驶钥匙、胜利配方、证明系列、黑外套制作、心之容器理论、钥刃继承仪式、预知者与大师盟约。忌把王国之心写成可随便打碎的石头——它是心之集合概念战。

**【契约者须知】**
低阶只做世界内委托与封印辅助；中阶可接触组织外围；高阶才碰赛亚诺特计划边角。迪士尼世界有「少破坏日常」的隐形规则：乱杀当地角色会触发世界排斥与钥匙孔不稳。


**【阶段纪要·岛到门】**命运岛异变夜，卡伊莉护符与索拉木剑是两件「心之信物」。迪士尼城简报后，古米船教程把世界旅行变成可玩地图。各世界钥匙孔封印要写清当地危机：被夺心的居民、被马列菲森特利用的反派、被无心者污染的地标。光之公主集结段，安森的实验室台词揭示无心者实验本质。门之世界终局，索拉变无心者与里克关门必须保留代价感。

**【阶段纪要·记忆与机关】**忘却之城的卡牌规则要写进任务：没有正确卡就无法开门。娜米妮的房间是安全屋也是罪证库。洛克萨斯的 358 天用「每日任务板」呈现，阿克塞尔的冰棍约定是情感锚，希恩的存在危机推动洛克萨斯质问组织。KH2 黄昏镇虚拟世界被撕开后，有心者登场改写敌人图鉴。组织干部战要具名：赛克斯、萨辛、德克西、勒克希斯、泽姆纳斯等，各带世界主题技。

**【阶段纪要·前史与收束】**BBS 三徒考试失败的因果链条必须连到索拉能用钥刃。DDD 梦逻辑里青年赛亚诺特的时间借心是 KH3 的棋。KH3 终战要写连接主题：朋友的心是门，不是耗材。卡伊莉被消后的逆行是规则级冒险，回报是关系而非数值。

**【阶段纪要·记忆与机关】**忘却之城的卡牌规则要写进任务：没有正确卡就无法开门。娜米妮的房间是安全屋也是罪证库。洛克萨斯的 358 天用「每日任务板」呈现，阿克塞尔的冰棍约定是情感锚，希恩的存在危机推动洛克萨斯质问组织。KH2 黄昏镇虚拟世界被撕开后，有心者登场改写敌人图鉴。组织干部战要具名：赛克斯、萨辛、德克西、勒克希斯、泽姆纳斯等，各带世界主题技。

**【阶段纪要·前史与收束】**BBS 三徒考试失败的因果链条必须连到索拉能用钥刃。DDD 梦逻辑里青年赛亚诺特的时间借心是 KH3 的棋。KH3 终战要写连接主题：朋友的心是门，不是耗材。卡伊莉被消后的逆行是规则级冒险，回报是关系而非数值。

**【人物与势力补】**光侧：索拉、里克、卡伊莉、米奇、唐纳德、高飞、亚克雅、尤拉。暗与灰：赛亚诺特诸体、安森、组织、马列菲森特、皮特。记忆：娜米妮、莉雅、希恩。科学家：耶那、艾文、伊安佐、艾拉库斯。

**【契约者锚点】**一阶岛民；二阶古米船；三阶护公主；四阶忘却；五阶黑外套；六阶外岛；七阶梦试炼；八阶终战守门。禁止低阶击杀赛亚诺特本体。
## 阶位切入点

> 阶位↔强度：杂兵无心者≈一阶；中型≈二阶；世界守卫≈三阶；钥刃中坚≈四阶；机关干部≈五阶；上位/暗之强者≈六阶；大师级≈七阶；王国之心终局≈八阶。顶点（大师赛亚诺特/王国之心本源争夺）条件性胜利，低阶规避。

**一阶（命运岛·异变前夜）**
切入身份/时点：岛民少年/旅人，锚定岛被暗吞噬前。
初始事件：海滩出现无心者阴影，卡伊莉的护符发光，你要选择叫醒索拉还是先护孩童撤离。
开场白建议：「浪声里多了一层耳语。命运岛的木桥尽头，黑影像墨滴进水。里克握着木剑说『我先去看』——你知道有些门一开就回不来。」
关键NPC立场：**索拉**——拉你一起；**里克**——好强探路；**卡伊莉**——担心；**岛民长**——不信暗。
主线钩子/支线：岛陷落逃生；支线＝找小船、安抚孩子。
危险度/规避：中；规避暗之回廊深处。
任务方向/奖励：启程、初级护符、同行资格。

**二阶（初访迪士尼世界）**
切入身份/时点：古米船见习，锚定索拉初封印钥匙孔期。
初始事件：某世界钥匙孔泄漏无心者，当地公主/英雄要你证明「不是来夺心」。
开场白建议：「古米船跳出超空间，城堡的烟囱还在冒童话式的烟。无心者的触手却从井盖爬出——唐纳德大喊：钥匙孔在那边！」
关键NPC立场：**唐纳德**——命令多；**高飞**——信任；**当地迪士尼英雄**——警惕外人；**马列菲森特爪牙**——破坏。
主线钩子/支线：封印钥匙孔；支线＝找回被夺心的市民。
危险度/规避：中高；世界崩坏即团灭。
任务方向/奖励：世界通行、魔法石、信任。

**三阶（光之公主危机）**
切入身份/时点：护卫佣兵，锚定马列菲森特集公主期。
初始事件：一名光之公主被掳，你与索拉分头——你负责假情报引开无心者大军。
开场白建议：「七种光被点名。马列菲森特的绿火在地图上烧出路线，你手中的名单少了一个名字——少的是你刚才还说过话的人。」
关键NPC立场：**马列菲森特**——主谋；**索拉**——冲锋；**贝儿等公主**——各有意志；**安森**——暗中利用。
主线钩子/支线：夺回公主；支线＝破坏集心装置。
危险度/规避：高；巨型无心者。
任务方向/奖励：光之碎片、钥刃强化素材。

**四阶（忘却之城·记忆）**
切入身份/时点：城内「卡片房间」迷途者，锚定 COM 线。
初始事件：你的记忆被抽成卡，门口站着组织黑衣人——用真名换通路或用假名冒险。
开场白建议：「走廊没有尽头，只有门。每扇门后是你以为经历过的世界，娜米妮的声音在纸背后发抖：『对不起，我必须改写。』」
关键NPC立场：**娜米妮**——被迫改写；**组织成员**——回收；**索拉**——记忆混乱；**里克**——暗侧平行。
主线钩子/支线：夺回记忆；支线＝保护娜米妮、识破伪卡。
危险度/规避：高；记忆抹杀。
任务方向/奖励：记忆卡、组织情报。

**五阶（XIII 机关·虚无）**
切入身份/时点：伪装黑外套的潜入者，锚定 KH2 机关期。
初始事件：机关会议点名「叛徒」，你的座位空着——阿克塞尔用火挡住追兵问你站谁。
开场白建议：「虚无世界的天空从不天亮。十三席黑影围桌，XIII 的声音像冰：『没有心的人，不需要名字。』你的假外套袖口滴着别人的血。」
关键NPC立场：**洛克萨斯**——求存在；**阿克塞尔**——重情；**希恩**——摇晃；**赛亚诺特残影**——收网；**索拉**——外部变量。
主线钩子/支线：阻止拼心；支线＝救希恩/莉雅线。
危险度/规避：极高；干部级。
任务方向/奖励：黑外套真货、机关密钥。

**六阶（梦中降生·三徒）**
切入身份/时点：艾拉库斯外门助手，锚定 BBS。
初始事件：泰拉的暗纹扩散，菲恩图斯要跟去，亚克雅要你保密——大师令你选一人同行监视。
开场白建议：「外岛的风比命运岛更冷。艾拉库斯的胡须里藏着没说完的警告，赛亚诺特的名字一出口，三名徒弟的眼神就碎成三种未来。」
关键NPC立场：**泰拉**——畏暗；**菲恩图斯**——纯；**亚克雅**——责；**艾拉库斯**——苦；**赛亚诺特**——棋手。
主线钩子/支线：寻师灭无知；支线＝阻止泰拉被夺身。
危险度/规避：极高；大师陷阱。
任务方向/奖励：前史知情、钥刃传承碎片。

**七阶（大师试炼·梦降）**
切入身份/时点：沉睡世界见习，锚定 DDD。
初始事件：青年赛亚诺特邀你「改时间」，里克沉睡中的心被钉——你要在梦的逻辑里拆钉。
开场白建议：「梦的重力反着来。索拉笑着挥手，下一秒脸换成别人。青年的银发在钟塔上：『心，可以借，也可以偷。』」
关键NPC立场：**索拉/里克**——考生；**青年赛亚诺特**——诱；**机关时间成员**——辅；**尤拉**——光侧。
主线钩子/支线：通过试炼不堕；支线＝救梦中迪士尼灵魂。
危险度/规避：极高；时间悖论。
任务方向/奖励：大师资格边缘、时间锚。

**八阶（KH3·光暗终局）**
切入身份/时点：七光再集的协力者，锚定 KH3 终战（非扮演赛亚诺特）。
初始事件：真正的组织拼出王国之心雏形，卡伊莉被消——索拉要逆行规则，需要你守住「回来的路」。
开场白建议：「天空裂成钥匙孔的形状。有人喊卡伊莉的名字，声音被风撕碎。米奇王把钥刃插入光中：『守住连接——这是唯一的门。』」
关键NPC立场：**索拉**——逆行；**里克**——守护；**米奇**——指挥；**赛亚诺特**——八阶顶点，条件性胜利；**马列菲森特**——渔利；**唐纳德高飞**——死守。
主线钩子/支线：收束暗之探求者；支线＝救被溶解同伴、护迪士尼世界不崩。
危险度/规避：贴近顶点；存在·情报优先/条件性胜利。
任务方向/奖励：终局连接权、后日谈席位、钥刃残响。
