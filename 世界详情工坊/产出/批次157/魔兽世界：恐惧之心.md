# 魔兽世界：恐惧之心

## 剧情

**【作品来源】**
《魔兽世界：恐惧之心》（Heart of Fear）是暴雪娱乐 MMORPG《魔兽世界》资料片《熊猫人之谜》（Mists of Pandaria，2012）的首发团队副本之一，属 T14 团本序列，与「魔古山宝库」「永春台」同期。舞台锁定在潘达利亚西南部**恐惧废土**（Dread Wastes）深处的螳螂妖皇权核心。媒介为大型多人在线角色扮演游戏；相关剧情分布于恐惧废土任务线、卡拉克西声望、团本地下城日志与后续永春台收束。文风：虫族帝国史诗、煞魔心理腐蚀、声波与琥珀科技交织的哥特式恐怖。本档案服务「轮回乐园」世界卡生成与入世正文——信息密度优先，忠于公开设定；查不到写不详，严禁编造原作没有的终局与宝物来历。

**【世界定位】**
本世界不是整部《魔兽世界》宇宙的缩影，而是以**恐惧废土—恐惧之心—惧之煞**为轴的可切入任务切片。契约者通常以「外来冒险者 / 卡拉克西唤醒者」身份卷入：螳螂妖大女皇**夏柯希尔**被**惧之煞**附身，提前发动虫群，几乎压垮影踪派长城；古老议会**卡拉克西**不得不唤醒琥珀中的历代英杰，并与外来者结盟，潜入皇宫处决女皇。终局：女皇死，惧之煞破顶逃往**永春台**；螳螂妖文明暂时由卡拉克西托管，等待新女皇成熟。契约者是变数，不是天选免死金牌——虫群不讲情面，琥珀不讲人权。

**【世界观 · 力量体系】**
艾泽拉斯六大宇宙力（圣光、暗影、奥术、自然、邪能、死亡）在潘达利亚以本土形态显化。本切片核心力量源有四：

1. **螳螂妖虫群生理与种姓战力**：个体以角质甲、镰刃肢、飞行/滑翔与声波共鸣为武器。虫群幼体（swarmborn）周期孵化、冲墙、归巢筛选；幸存者进入种姓。种姓并非单纯职业，而是社会职能与作战风格的固化——Kor'thik 重战、Set'thik 风刃与声波、Zar'thik 琥珀祭司与战地医疗、Sra'thik 琥珀运用、Vor'thik 等分支在不同凯帕树下形成地域军。
2. **琥珀 / 凯帕石（kyparite）科技**：凯帕巨树分泌的琥珀是螳螂妖的「大地之血」——建材、刃材、能源、防腐剂与生命维持介质。**琥珀塑形者**可把液态琥珀铸成活体造物、护甲与拘束茧；英杰（Paragon）在巅峰期被封入琥珀，供后世按需唤醒。琥珀枯竭等于文明贫血。
3. **声波权柄**：螳螂妖以声波传递女皇意志、鼓舞与杀伤。**皇家宰相佐尔洛克**的「女皇之声」可令战士无条件赴死；宫殿回廊本身就是共鸣器。
4. **煞能（Sha）**：上古之神亚煞极被斩杀后，负面情感具象为煞。**惧之煞**是恐惧的具现，能附身心智、腐化琥珀与凯帕树、扭曲女皇的统治歌。被煞浸透的个体会偏执、多疑、拥抱恐惧为力量。

死亡规则：普通螳螂妖与凡人冒险者身死即死；英杰可再度琥珀化保存，但需完整尸体与仪式；女皇死亡后魂可入暗影界（后续资料提及黑锋收割其精华），但**惧之煞本身不是可被「杀死即永久消失」的凡物**——它可弃容器逃遁。世界顶点在本切片内是「惧之煞 + 被完全灌注的大女皇」复合体，低阶严禁硬刚本体。

境界 / 战力阶梯（按破坏力表现，宁低勿高）：
1. **废土边缘民兵 / 新兵冒险者**：冷兵器与初级法术，能对付散兵与幼虫。乐园观感约一阶。
2. **成熟种姓战士 / 影踪派长城守军**：可单挑小队，声波刃与琥珀甲初成。乐园观感约二阶。
3. **虫群军官 / 唤醒初期的英杰侧翼 / 影踪派精英**：端据点、裂甲、乱军取首。乐园观感约三阶。
4. **团本中坚首领级（佐尔洛克、塔亚克、加拉隆）**：掀平建筑群、领域式声波 / 风刃 / 巨虫冲撞，一城仅数人可挡。乐园观感约四阶（质变分水岭）。
5. **风领主梅尔加拉克、琥珀塑形者昂舒克**：城市级战场指挥与造物军团，领域成型。乐园观感约五阶。
6. **大女皇夏柯希尔（煞附前夕 / 未全开）**：地区灾难级，声波加煞能覆盖皇宫与外围虫群。乐园观感约六阶。
7. **大女皇夏柯希尔（终极腐化阶段）/ 惧之煞显化侧翼**：战略级恐惧场、幻象与灾星式打击，可动摇一域秩序。乐园观感约七阶。
8. **惧之煞本体（完整形态，主要在永春台收束）**：行星表面级负面情感风暴，本切片内以「破顶逃遁」为顶点侧影；完整对决在永春台。乐园观感约八阶（本世界八阶封顶，勿抬到九阶或超阶）。

乐园阶位映射：废土杂兵与新兵约一阶；种姓战士 / 长城守军约二阶；虫群军官与初期英杰协同约三阶；恐惧之心前厅三王（佐尔洛克、塔亚克、加拉隆）约四阶；梅尔加拉克与昂舒克约五阶；女皇未全开约六阶；女皇终极腐化约七阶；惧之煞本体 / 完整逃遁后的永春台形态约八阶。宁低勿高；顶点忠于原作；惧之煞以条件性驱逐与追击论，严禁「被封印削弱成可随便砍」式解释。

**【地理 · 舞台】**
- **蛇脊长城（Serpent's Spine）**：把恐惧废土与潘达利亚其余地区隔开的巨墙。影踪派与联盟 / 部落驻军在此抵挡虫群；日落之门一带是常见切入走廊。
- **恐惧废土**：琥珀巨树枯黄、空气带酸腐甜腥的虫族家园。地表满是凯帕树残骸、虫巢、琥珀池与煞雾。主要聚点：
  - **卡拉克西维斯（Klaxxi'vess）**：卡拉克西议会据点，唤醒者（Wakener）与英杰集结处，声望与日常任务枢纽。
  - **古尔山台地、星湖、盐水泥沼、日落酒坊、索吉的赌局**等：任务线分支点，分别对应不同英杰唤醒与资源线。
  - **凯帕·扎尔、凯帕·沃尔**等具名凯帕树：文明血脉；枯死等于琥珀断供与种姓军心溃散。
- **恐惧之心**：废土西部巨型腐朽树心 / 皇宫复合体，分两翼——
  1. **恐惧进军（The Dread Approach）**：议事厅（佐尔洛克）→ 训练厅（塔亚克）→ 恐惧露台（加拉隆）。
  2. **夏柯希尔的疯狂（The Madness of Shek'zeer）**：集结露台（梅尔加拉克）→ 琥珀研究圣所（昂舒克）→ 女皇王座厅。
- **关联舞台**：**永春台**（Terrace of Endless Spring）是惧之煞逃遁后的下一棋盘，本档案只作因果出口，不展开永春台全本。

AI 写场景时先定契约者在「长城—废土地表—卡拉克西维斯—皇宫外廊—两翼深处—王座」哪一层，再选冲突强度：外层是虫潮与补给；中层是英杰政治与琥珀资源；内层是声波与煞能的规则杀。

**【世界剧情线】**

① **前史：虫群周期与女皇制**
螳螂妖文明以「约百年一次的大虫群」自我淬炼：幼体冲撞蛇脊长城，弱者死、强者归。大女皇以声波意志统御，卡拉克西议会监督传承——女皇衰老时由选定继任者取代，继任者往往需完成残酷试炼（含吞噬前任遗骸等传统）。历代最强战士在巅峰被封入琥珀，成为可唤醒的**英杰**。凯帕树是国本；琥珀枯竭即亡国预兆。螳螂妖敬畏上古之神亚煞极的遗产，对「恐惧」并不陌生，但从未以「让恐惧统治自身」为正统。

② **惧之煞入局与夏柯希尔失心**
联盟与部落在潘达利亚开战、煞被大规模惊醒后，**惧之煞**挣脱旧日束缚（与砮皂寺等地的恐惧叙事相连），选定螳螂妖大女皇**夏柯希尔**为容器。她正当壮年，预定继任者（后称大女皇夏柯扎拉一系）尚幼。惧之煞灌入偏执与多疑：女皇开始视卡拉克西为叛徒、视长城外一切为必须吞噬的威胁，并下令**拥抱煞能**——用恐惧强化虫群、腐化琥珀与凯帕树。虫群周期被提前约十年发动，影踪派防线几近崩溃。凯帕树开始枯死，琥珀河流干涸；部分种姓（如拥抱煞的 dreadsworn 倾向群体）主动饮毒。

③ **卡拉克西的「令人遗憾的决定」**
卡拉克西人数稀少，无常备大军，但握有英杰与传统合法性。他们判定：女皇已不适合统治——「即使在睡梦中也能听见她的恐惧」。按律本应等待继任者成熟，可等不起。议会决定：必须处死被腐化的女皇。为此他们需要外来者充当**唤醒者**——打开琥珀、喂养英杰、在战场上充当人手。若干卡拉克西瓦（Klaxxi'va 祭司级）在女皇整肃中死亡（Ik、Kek、Krik、Set、Zan 等名见于设定），加深了议会的杀意。

④ **恐惧废土任务线：唤醒者之路（可玩主轴）**
外来者经蛇脊 / 影踪派线索进入废土。典型事件链：
- 于台地发现琥珀中的**风掠者基尔鲁克（Kil'ruk the Wind-Reaver）**，以树液蝇碎块等权宜之计暂缓其苏醒饥渴，再护送至卡拉克西维斯见议会。
- 并行唤醒：**无伤者马里克（Malik the Unscathed）** 在女皇爪牙区被困；**至尊者柯尔文（Korven the Prime）** 在星湖琥珀矿被释放——他是琥珀封存术的提出者 / 第一位英杰，带领调查**凯帕·扎尔**枯死之根；**操纵者卡兹提克（Kaz'tik the Manipulator）** 在泥沼驯服巨虫昆虫；其后随声望再唤 **寻血者斯吉尔、解剖者里卡尔、毒心者夏里尔、虫群看守者希塞克、明晰者伊约库克、蝗虫卡罗兹** 等。
- 日常与支线：采集活琥珀、守卫声纳塔、清剿 Ik'thik 塔卫、协助熊猫人酒坊试酿琥珀酒、在索吉的赌局等灰色据点换情报。
- 高潮任务：**女皇的诡计 → 夏柯希尔之怒 → 帝国之影**。在击退女皇外围大军后，卡拉克西与唤醒者获得突入恐惧之心的政治与军事许可。

⑤ **恐惧之心·恐惧进军（团本前翼）**
- **皇家宰相佐尔洛克（Imperial Vizier Zor'lok）**：女皇之声。无论女皇是否疯癫，他绝对忠诚，用声波赞美「恐惧的统治」，其言可令螳螂妖赴死。他曾在「女皇的诡计」线中击杀英杰**无伤者马里克**（马里克因此缺席后续奥格瑞玛之围九英杰战）。名场面：议事厅多平台声波移转，沉默其「女皇之声」是突入的第一道门槛。
- **刀锋领主塔亚克（Blade Lord Ta'yak）**：风与刃的导师型杀手，耐心、聪慧，专破敌人破绽。战前以四名教官分别演示力量、钢铁、速度与风；本人则将四法合一。训练厅风墙与刀网是第二道门槛。
- **加拉隆（Garalon）**：最强昆虫巨兽，由风领主侧的法师唤来的活体攻城引擎。战术要点是摧毁其足以降甲再击体。露台上的巨影碾压，宣告前翼结束、内殿开启。

⑥ **恐惧之心·夏柯希尔的疯狂（团本后翼）**
- **风领主梅尔加拉克（Wind Lord Mel'jarak）**：空中军团统帅，傲慢坚信螳螂妖不可败，却又护兵如命。加拉隆死后，他集结 Kor'thik 战士、Zar'thik 愈疗者、Sra'thik 琥珀师等精锐守内殿。战中与虫群共存亡——虫群减员会令他更疯狂。
- **琥珀塑形者昂舒克（Amber-Shaper Un'sok）**：年轻天才炼金 / 塑形师，曾与英杰**毒心者夏里尔**有学术渊源。为取悦女皇而贴近宫廷，反被煞能浸透，对「失败」的恐惧驱动他制造越来越扭曲的活琥珀、琥珀巨怪，甚至把入侵者暂时改造成变异构装。圣所是王座前最后的实验室。
- **大女皇夏柯希尔**：终局。惧之煞全程低语赋能。阶段概要——个人战：不和谐力场、女皇之眼（魅惑役使）、恐惧尖啸与哭号；能量耗尽则退入虫茧，召 **Set'thik 风刃** 与 **Kor'thik 掠夺者** 等皇家卫队回血；约三成生命时煞完全灌注，施放灾星、吞噬恐惧、死亡幻象等。临终：「我……失败了。」惧之煞斥其无用，破殿顶逃遁，扬言后来者将「尖叫着死去」。

⑦ **终局与余波**
女皇身死，卡拉克西暂时接管政权，等待新女皇成熟。大量凯帕树已毁，文明陷入迷茫但摆脱全面煞化。惧之煞逃往**永春台**，成为 T14 终局对决对象（本世界因果出口）。部分英杰在后续《决战奥格瑞玛》中因亚煞极之心而站队加尔鲁什，化为「卡拉克西英杰」议会战——那是本切片之后的蝴蝶，本档案仅作伏笔提示。夏柯希尔之魂后入暗影界，被黑锋收割精华（跨资料片补充，点到为止）。

⑧ **微观政治与可介入日常**
除团本高光外，正文高频事件应落在：长城夜班虫潮、琥珀采集配额、唤醒仪式所需的活琥珀与树液、卡拉克西内部对「是否该用煞反制煞」的争执、影踪派与虫族停火谈判的破裂边缘、种姓军对女皇死令的服从度测试、酒坊与赌局的中立补给点、昆虫失控冲阵、声波塔被破坏导致的指挥链瘫痪。AI 优先写这些，而不是一上来王座斩首。

⑨ **蝴蝶效应边界**
契约者可改：多救一名英杰侧翼、提前枯死或保住某棵凯帕树的局部、改写某场虫潮的伤亡数字、在昂舒克实验室截获一批活琥珀造物。不可无代价改写：女皇已被惧之煞深度附身的事实、惧之煞在女皇死后逃往永春台的核心因果、卡拉克西「必须处决女皇」的政治结论。改了局部，原作会以更猛的虫潮、更早的英杰折损或煞雾扩散回流。

**【主要人物】**

- **大女皇夏柯希尔（Grand Empress Shek'zeer）**｜性格：本为螳螂妖帝国意志的化身，被惧之煞浸透后偏执、多疑、以恐惧为统治美学；偶有残存的女皇尊严在临终一现。｜装备·能力：女皇声波权柄、煞能灌注、虫茧再生、皇家卫队召唤、魅惑之眼与恐惧尖啸；全开时接近地区—战略级。｜人物弧光：正当壮年的统治者 → 惧之煞容器 → 被卡拉克西与外来者处决 → 魂入暗影界。｜立场关系：本切片最大敌对顶点；对契约者只有消灭与役使，无谈判窗口。

- **惧之煞（Sha of Fear）**｜性格：恐惧本身，冷酷、嘲弄、视容器为消耗品。｜装备·能力：情感具现、附身、煞雾污染琥珀与心智、破空间逃遁；完整形态破坏力对本世界映射至八阶。｜人物弧光：挣脱旧囚 → 附身夏柯希尔 → 女皇死后逃往永春台。｜立场关系：真正幕后；低中阶只闻其声，高阶才见其形；条件性驱逐，严禁硬刚「已完整降临」的本体。

- **皇家宰相佐尔洛克（Imperial Vizier Zor'lok）**｜性格：绝对忠诚的传声筒与狂热信徒，女皇疯癫与否不影响服从。｜装备·能力：声波赞美诗、平台移转、令虫群无条件赴死的「女皇之声」。｜人物弧光：女皇喉舌 → 恐惧进军第一王 → 战死。｜立场关系：敌；曾亲手杀死无伤者马里克，与卡拉克西血仇。

**【势力图谱】**
- **夏柯希尔皇权 / 恐惧之心宫廷**：宗旨是以恐惧统一虫群、吞噬长城外。地盘是恐惧之心与废土大部种姓军。代表是夏柯希尔、佐尔洛克、塔亚克、梅尔加拉克、昂舒克、加拉隆。战力为四到七阶梯队。敌对卡拉克西、影踪派、联盟 / 部落远征军。
- **卡拉克西议会**：宗旨是保全螳螂妖文化与琥珀命脉，必要时弑君。地盘是卡拉克西维斯与隐秘琥珀窖。代表是诸英杰与残存卡拉克西瓦。战力为三到六阶个体尖兵，缺大军。友为有限条件下的外来唤醒者。
- **种姓军群（Kor'thik / Set'thik / Zar'thik / Sra'thik / Vor'thik 等）**：皇权肌肉；部分被煞污染后战力上升、理智下降。
- **影踪派**：长城防守方，目标是挡住虫群而非灭亡螳螂妖文明。
- **联盟与部落远征军**：视恐惧之心为 T14 战略目标，内部仍有阵营摩擦，但在虫潮前可临时协同。
- **惧之煞及其煞雾**：无领土的感染源，真正跨势力公敌。

敌友逻辑：利益与生存优先；卡拉克西盟约以「杀女皇、保琥珀」为硬条款；暴露可利用价值会被双方同时盯上。

**【贵重物品】**
- **活琥珀 / 凯帕石原液**：文明燃料与英杰苏醒口粮；谁控制矿脉谁控制谈判桌。低阶可少量窃运，高阶可截断军需。
- **琥珀封存茧与英杰琥珀**：内含历代英杰；破坏等于毁掉战略储备，开启等于获得强力盟友或强敌。
- **女皇声波权杖 / 共鸣水晶（团本掉落向神器感装备的叙事核）**：放大声波指挥；落入非女皇之手会引发种姓军短暂混乱。
- **昂舒克的塑形模具与变异血清**：可临时改造肉体，代价是煞污染与心智损伤。
- **卡拉克西声望徽记与琥珀蝎坐骑权限**：制度性身份票，决定能否在卡拉克西维斯自由行动。
- **影踪派长城令箭**：通过蛇脊关卡的通行证。
- **团本套装与英雄模式武器（恐惧之心 T14）**：游戏内战力成长的物质化，叙事上视为从女皇金库夺来的种姓级武装。
- **惧之煞残渣 / 恐惧结晶**：高危材料，可强化武器亦可诱发散煞；低阶碰等于自寻死路。

每件写清：夏柯希尔派想回收、卡拉克西想管制、冒险者想兑换——暴露天价库存等于全民公敌。

**【隐藏剧情 · 伏笔】**
- 惧之煞选中女皇，不仅是方便，更是因为螳螂妖本就亲近亚煞极遗产——「恐惧」在其文化中可被美化成淬炼。
- 凯帕树枯死不只是战术破坏，而是煞在挖文明墙脚；保住树等于保住战后重建。
- 马里克之死解释了为何九英杰战缺席一人，并强化卡拉克西对佐尔洛克的仇恨。
- 昂舒克与夏里尔的学术对照：同一套琥珀 / 毒物知识，一边疯一边冷，暗示「技术中立、意志决定方向」。
- 女皇死后英杰为何会在奥格瑞玛之围站队加尔鲁什：亚煞极之心的诱惑是本切片的远期毒丸。
- 惧之煞破顶离去，是永春台副本的直接钩子——本世界通关不等于恐惧结束。

**【大事记时间线】**
远古：亚煞极败亡，煞遗留；螳螂妖立国，凯帕树与女皇制成型，柯尔文提出琥珀封英杰。  
周期史：百年虫群冲撞蛇脊，影踪派死守。  
潘达利亚开启：联盟部落登陆，煞被大规模惊醒。  
惧之煞附身夏柯希尔；虫群提前；凯帕树枯死加速。  
卡拉克西决定弑君；外来者成为唤醒者；基尔鲁克等英杰陆续苏醒。  
外围大战「夏柯希尔之怒」；恐惧之心门户打开。  
恐惧进军：佐尔洛克 → 塔亚克 → 加拉隆。  
内殿：梅尔加拉克 → 昂舒克 → 夏柯希尔；惧之煞逃往永春台。  
卡拉克西临管；新女皇谱系等待成熟；恐惧主题转场永春台。

**【叙事基调 · 雷区】**
基调：虫族帝国的冷艳与窒息感，声波回廊、琥珀冷光、煞雾粘稠；美与恐怖同体。忌把螳螂妖写成无脑虫潮——他们有文化、有议会、有匠人与导师。忌把夏柯希尔写成「单纯被附身的无辜受害者」而抹去她仍以女皇身份下死令的责任，也忌写成无动机的疯婆。忌卡拉克西无条件圣母——他们杀女皇是种族自救，不是普世道德。忌低阶单挑女皇或惧之煞。忌把琥珀科技写成东方玄幻丹药炼化体系。忌用跨世界模板套话填充。最早可切入锚点：蛇脊长城夜班虫潮前夜，或基尔鲁克琥珀被发现之时。

**【世界剧情线·废土细目】**
恐惧废土地表是契约者最常落地的「活地图」。凯帕树冠投下琥珀色阴影，地面裂缝渗出半凝固的树液，幼虫与树液蝇抢食同一摊光斑。卡拉克西维斯的石台与旗幡在雾里像一排审判席：外来者被称作唤醒者，这个头衔既是权限也是枷锁——你唤醒谁，谁就有权支使你去取下一批活琥珀或砍倒一队 Ik'thik 塔卫。星湖底的古代琥珀块是柯尔文的棺，也是课堂：他苏醒后第一课不是杀敌，而是指出凯帕·扎尔的汁液已断，「河流」成了历史名词。盐水泥沼里卡兹提克与昆虫的精神丝线在空气中几乎可见；搞砸驯虫会让己方阵线被友军碾死。日落酒坊的熊猫人试图用琥珀酿酒，是废土里罕见的暖色幽默，也提醒玩家：文明冲突之外仍有生活。影踪派从长城上射来的信号箭有时是求援，有时是警告你别把虫群引回墙内。这一层的名场面应具体到「哪一棵树、哪一次苏醒、哪一车琥珀被劫」。

**【世界剧情线·皇宫细目】**
恐惧之心内部是声波建筑学的展厅。佐尔洛克的议事厅把女皇颂词变成可致死的驻波；站错平台等于被自己的耳膜背叛。塔亚克的训练厅像一座转动的刃之法场，教官的死不是过场，而是告诉你刀锋领主如何把「教学」做成杀戮。加拉隆的露台没有外交，只有足与甲的工程学弱点。进入后翼后，梅尔加拉克的集结露台是种姓军的阅兵场，你每杀掉一队亲卫都在给他的傲慢加油。昂舒克的圣所漂浮着未完成的躯壳，空气甜得发苦——那是活琥珀挥发分；被他点名「塑形」的队员会短暂成为敌方构件，这是团队信任的酷刑。王座厅的虫茧与女皇之眼把「恐惧」机制化：你害怕失控，就真的把刀挥向同伴。惧之煞的台词贯穿全程，提醒所有人：女皇已是麦克风，真正的歌手在更深的地方。

**【世界剧情线·政治细目】**
卡拉克西不是圣骑士团。他们保存英杰是为了在文明危机时取出「古代核武器」，唤醒者不过是钥匙。女皇派指控议会叛国，议会指控女皇叛文明，双方都有部分真相：女皇确实在加速灭亡，议会也确实在挑战尚且「合法」的声波主权。种姓军中有人只服从声波，有人开始私下囤琥珀准备战后；Vor'thik 一类拥抱煞的分支视恐惧为进化。联盟与部落的冒险队可能在同一走廊抢掉落与先杀权，但虫群刷新不在乎你的阵营旗帜。契约者若两边拿钱，会在「夏柯希尔之怒」阶段被同时列入清除名单。通关后卡拉克西的临管充满真空：没有女皇的声波，虫群幼体更难约束，新的秩序要用血重新校准。

**【世界剧情线·因果细目】**
本世界的「通关」在叙事上等于「砍断惧之煞在螳螂妖的权杖」，不等于消灭恐惧。惧之煞破顶那一刻，天空裂开的不是胜利烟火，而是下一张地图的邀请函——永春台的纯净之泉正是它要去玷污的对象。英杰的存活名单会影响后续资料片的战力表：马里克已死是钉死的；其余英杰若在本世界被契约者尽量保全，也改变不了亚煞极之心对部分人的远期诱惑，但可改变废土重建速度。凯帕树的存续决定战后经济：树死，则琥珀武器与封存术都成绝唱，螳螂妖只能沦为散兵文明。影踪派是否信任「杀了女皇的外来者」，决定长城是否对你敞开回程。写结局时必须交代：谁死、谁接管、煞去了哪、树还在不在。

**【世界剧情线·种姓与琥珀经济细目】**
Kor'thik 提供重装冲击，Set'thik 把风与声波编成刀阵，Zar'thik 在战场上用琥珀祷词续命，Sra'thik 负责把液态琥珀变成可投掷的杀伤件。每一种姓都绑着一棵或一片凯帕林地：树死，种姓就失血。女皇下令「拥抱煞」之后，部分种姓把污染琥珀当兴奋剂，短期战力暴涨，长期甲壳开裂、心智幻听。卡拉克西的日常任务本质上是战时配给制——你交活琥珀，他们给你武器附魔与英杰随身增益；你欠债，就去塔卫区送死。索吉的赌局与日落酒坊是配给制外的灰市：情报、走私刃、未登记的昆虫卵在此换手。契约者若只打团本Boss而忽视经济层，会在正文里显得「打完副本却饿死在废土」。

**【世界剧情线·影踪派与长城细目】**
蛇脊长城不是风景，是文明分界线。影踪派在此耗了数百年，知道螳螂妖不是天灾而是邻居——可怕的邻居。联盟与部落的旗帜挂上城垛后，补给线变复杂：有人要把火药运上去，有人要把伤员运下来，有人要把「唤醒者」这种政治烫手货送进废土。夜班虫潮的节奏可被女皇声波加速；白天则有走私琥珀的车队试图走墙根。契约者在长城的选择会回传：放任破口，影踪派记仇；死守并分享情报，换取回程与医疗。弓弩手李一类具名引路者不是工具人，他们是「外面的世界是否还把废土当绝地」的温度计。

**【世界剧情线·首领战因果链细目】**
六个首领不是六个无关房间。佐尔洛克死后，女皇失去最稳的喉舌，种姓军的服从出现缝隙，卡拉克西的离间才能生效。塔亚克死后，训练体系断裂，新兵质量下降，梅尔加拉克不得不把精锐提前压上。加拉隆倒下等于攻城引擎报废，皇宫外围由「碾压」改成「巷战」。梅尔加拉克若因亲卫被针对而早狂暴，会打乱昂舒克的实验节奏，导致实验室门提前洞开或提前锁死。昂舒克死后，女皇失去最后的造物壁垒，只能亲自下场并更快进入虫茧与终极灌注。女皇死，惧之煞立刻评估容器报废并转移——这是铁因果，不是可有可无的过场。

**【世界剧情线·可玩任务板细目】**
适合正文反复抽取的任务板：① 为柯尔文采集星湖活琥珀，途中遭遇雾刃鳄鱼人劫矿；② 在泥沼为卡兹提克找回失控昆虫，避免它冲进卡拉克西维斯；③ 护送一车未污染琥珀去酒坊，被 Vor'thik 煞化巡逻队盯上；④ 潜入女皇爪牙区给马里克断锁，限时内佐尔洛克声波扫场；⑤ 摧毁一座被煞污染的声纳塔，恢复议会对局部虫群的监听；⑥ 在「夏柯希尔之怒」外围战中守住议会旗帜，失败则突袭恐惧之心的许可延后；⑦ 从昂舒克运输队截获变异血清，交给夏里尔解析；⑧ 女皇死后镇压无声波约束的幼虫潮，保护一棵尚未枯死的幼年凯帕树。每条任务都有明确地点、敌人与政治后果，禁止空转。

**【补足四百五十字】** 在恐惧之心之外，长城上的守夜人通过望远镜看废土金尘浓度，像看天气。浓度降则敲铜锣报「可开城门贸易两时辰」。这种民生节奏比团本更长久。卡拉克西陶罐巷的孩子用破风镜玩角色扮演，一人扮宰相唱歌，一人扮影踪喝止；大人不笑，只把稳定香往他们口袋里塞。联盟记录官试图把女皇厅战斗写成「文明对野蛮的胜利」，卡拉克西改正为「对腐化主权的手术」。用词之争持续到庆功宴结束。部落萨满在半封谷立小小图腾，不是占地盘，是安抚「被封过的风」。熊猫人厨师研究如何把抗信息素草药煮进面里，让士兵爱喝。这些细节不改变女皇已倒的事实，却决定废土能不能住人。契约者若参与重建，会发现八阶之后的敌人是会议、教材、净水与失眠，不是新的血条。

## 阶位切入点

> 阶位↔原作强度：废土杂兵与新兵冒险者约一阶；种姓战士 / 长城守军约二阶；虫群军官与初期英杰协同约三阶；佐尔洛克 / 塔亚克 / 加拉隆约四阶；梅尔加拉克 / 昂舒克约五阶；夏柯希尔未全开约六阶；夏柯希尔终极腐化约七阶；惧之煞本体（永春台完整侧影）约八阶。世界顶点为惧之煞与被其完全灌注的大女皇复合体，低阶切入一律「规避顶点·情报优先」。本世界仅覆盖：一阶、二阶、三阶、四阶、五阶、六阶、七阶、八阶。

**一阶（蛇脊夜班 · 虫潮边缘）**
切入身份/时点：契约者以联盟 / 部落新兵、影踪派后勤辅助或流浪佣兵身份，锚定「蛇脊长城夜班、日落之门附近发现废土裂口」前后。
初始事件：夜班号角乱响，一小队 Kor'thik 散兵与幼虫翻墙缺口；弓弩手李一类驻军点名你去堵缺口，同时有人要你护送伤员下墙——站队堵口还是保命后撤，立刻见血。
开场白建议：「长城砖缝里渗出的不是雨水，是带着甜腥的树液味。下方黑暗里，甲壳摩擦声像有人用刀刮锅。你握着刚配发的武器，听见影踪派老兵低声说：别朝雾里看太久——雾里先看见你的，是恐惧，然后才是虫子。」
关键NPC立场：**弓弩手李（影踪派）**——要你守纪律；**风掠者基尔鲁克**（尚未苏醒，仅作远处琥珀轮廓传闻）——未知；**Kor'thik 散兵头目**——猎食；同班新兵——抢功或逃跑。
主线钩子/支线：钩子是守住缺口并听到「女皇提前发动虫群」的流言；支线是运送箭矢、焚烧虫尸防污染、追查是谁打开了墙缝栅。
危险度/规避：中——死于虫潮与跌落；须规避女皇直属军官与任何煞雾核心。
任务方向/奖励：存活、长城令箭残页、劣质琥珀碎片、影踪派初阶信任。

**二阶（废土初入 · 树液与琥珀）**
切入身份/时点：以被许可进入恐惧废土的斥候 / 拾荒工匠身份，锚定「发现基尔鲁克琥珀、执行苏醒补给」前后。
初始事件：你在台地敲开琥珀裂缝，**风掠者基尔鲁克**苏醒后饥渴难耐，命令你采集树液蝇碎块；同时女皇巡逻队逼近——是喂英杰还是弃茧而逃。
开场白建议：「琥珀里的刀光先于眼睛醒来。基尔鲁克的第一句话不是感谢，是索取：给我食物，唤醒者，否则你只是一块会走路的饵。远处种姓军的声波短啸扫过沙丘，像有人用指甲划过你的脊骨。」
关键NPC立场：**风掠者基尔鲁克**——工具性结盟；**琥珀匠齐克**——要材料；**Ik'thik 巡逻官**——清除外来者；影踪派联络兵——要你回报虫群密度。
主线钩子/支线：钩子是护送基尔鲁克至卡拉克西维斯；支线是清树液蝇、夺回被劫的琥珀车、为苏醒仪式找干净水源。
危险度/规避：中——死于巡逻队与英杰试探性杀招；勿深入恐惧之心外廊。
任务方向/奖励：卡拉克西初阶标记、活琥珀样品、英杰侧的一次性增益。

**三阶（卡拉克西维斯 · 议会棋子）**
切入身份/时点：以正式唤醒者候选身份，锚定「卡拉克西议会接见、多线唤醒开启」前后。
初始事件：议会要你在「星湖释放柯尔文」与「泥沼协助卡兹提克」两条线中先选一条；另一条线的延误会导致一名英杰苏醒不稳——资源不够两全。
开场白建议：「卡拉克西维斯的石台上，没有掌声，只有甲节敲击的表决声。他们叫你唤醒者，眼睛却在称你的骨头：能开几具琥珀，能死几次还爬回来。你的选择会写进他们的账册——账册比女皇的颂词更长。」
关键NPC立场：**至尊者柯尔文**——考你对琥珀传统的尊重；**操纵者卡兹提克**——考你是否敢靠近昆虫；**毒心者夏里尔**（若已解锁）——冷眼看你是否乱碰毒物；女皇派奸细——要离间。
主线钩子/支线：钩子是完成至少一名核心英杰的稳定苏醒；支线是保卫声纳塔、调查凯帕·扎尔枯死表象、截获宫廷奸细的声波符文。
危险度/规避：高——死于政治背刺与昆虫失控；规避佐尔洛克本体级声波场。
任务方向/奖励：英杰战技残篇、中等琥珀武器、议会任务权限。

**四阶（恐惧进军 · 前厅破门）**
切入身份/时点：以卡拉克西认可的突袭队骨干身份，锚定「夏柯希尔之怒后、恐惧之心前翼开放」前后。
初始事件：突袭队在议事厅外被**皇家宰相佐尔洛克**的声波定住半队；你必须决定是冲平台打断颂词，还是撤退保存柯尔文提供的琥珀护符。
开场白建议：「门后传来的不是军号，是赞美。佐尔洛克把女皇的名字唱成刀子，你的牙齿跟着发麻。同伴有人跪下去，不是臣服，是耳膜先于意志投降。你踏进恐惧进军，知道从此每一步都在女皇的嗓子里。」
关键NPC立场：**皇家宰相佐尔洛克**——杀；**刀锋领主塔亚克**——在更深处等待；**无伤者马里克**（若时点在其死前）——愿当锋矢；**风掠者基尔鲁克**——要战果。
主线钩子/支线：钩子是击破佐尔洛克或制造其指挥链真空；支线是在训练厅偷师塔亚克刀法残招、给加拉隆足部预埋爆裂琥珀、救出被声波震伤的队员。
危险度/规避：高——死于声波与刃墙；须规避女皇本尊与惧之煞直视。
任务方向/奖励：前厅首领级武装、声波抑制饰品、卡拉克西战功。

**五阶（内殿门槛 · 风领主与塑形者）**
切入身份/时点：以团队指挥 / 独行高手身份，锚定「加拉隆已倒、梅尔加拉克集结精锐」前后。
初始事件：**风领主梅尔加拉克**派出 Zar'thik 愈疗编队回拉前线，同时**琥珀塑形者昂舒克**的活造物从侧廊封路——你要拆哪条腿，决定突袭速度。
开场白建议：「露台上的风全是翅膀煽起来的。梅尔加拉克不屑看你的脸，只数他的亲卫还剩几列。另一侧走廊滴着金色的脓——那是昂舒克的杰作在呼吸。内殿的门还没开，战争已经学会两条喉咙说话。」
关键NPC立场：**风领主梅尔加拉克**——敌，护兵；**琥珀塑形者昂舒克**——敌，实验；**虫群看守者希塞克**——要你优先断虫群指挥；**解剖者里卡尔**——要活体样本。
主线钩子/支线：钩子是削弱内殿双重门槛之一；支线是偷运昂舒克模具、策反一名未煞化的 Sra'thik 琥珀师、保护希塞克的虫群标记不被改写。
危险度/规避：高——死于精锐协同与变异构装；规避女皇虫茧阶段的皇家卫队海。
任务方向/奖励：种姓精锐装备、塑形抑制药剂、内殿地图残片。

**六阶（王座前夜 · 女皇未全开）**
切入身份/时点：以斩首队核心身份，锚定「昂舒克已倒、夏柯希尔王座厅开启但煞未完全灌注」前后。
初始事件：女皇以女皇之眼魅惑你的一名队友；你必须在「击晕队友」与「硬吃一波恐惧尖啸」之间选，同时虫茧开始凝聚。
开场白建议：「王座比想象中更静。夏柯希尔的冠像一丛倒生的刺，眼睛却像两口深井——深处有东西在看你。她还没有把全部恐惧铺开，只是试探：你们是来杀女皇的，还是来给惧之煞送新面具的？」
关键NPC立场：**大女皇夏柯希尔**——敌，试探性杀戮；**惧之煞**（声音）——催促女皇；**明晰者伊约库克**——要你保持阵型不被魅惑；**寻血者斯吉尔**——要女皇血作祭。
主线钩子/支线：钩子是打掉女皇第一阶段并阻止虫茧满能量；支线是破坏共鸣水晶、疏散被魅惑的炮灰、记录惧之煞语供卡拉克西存档。
危险度/规避：极高——死于魅惑与尖啸；严禁在此阶段宣称「已可斩煞」。
任务方向/奖励：女皇金库级武器半成品、恐惧抗性斗篷、议会紧急动员令。

**七阶（终极腐化 · 斩首女皇）**
切入身份/时点：以世界级侧翼强者 / 团队之矛身份，锚定「夏柯希尔约三成生命、惧之煞完全灌注」前后。
初始事件：虫茧碎裂，灾星与死亡幻象同时落下；卡拉克西远程传讯要求你「必须留下女皇尸体完整以证弑君合法性」——与现场爆发输出冲突。
开场白建议：「煞从她的甲缝里长出来，像第二层皮肤。夏柯希尔的声音叠上第二个人的笑声：容器在哭，主人在笑。你知道这一刀下去，杀的是女皇，也是潘达利亚某段恐惧史的书名。别眨眼——幻象里死去的人，都会用你熟悉的嗓音喊你的名字。」
关键NPC立场：**大女皇夏柯希尔（终极腐化）**——死敌；**惧之煞**——即将弃壳；**风掠者基尔鲁克**——要临门一刀；**至尊者柯尔文**——要程序正义与琥珀后续。
主线钩子/支线：钩子是击杀夏柯希尔并见证惧之煞破顶；支线是保全女皇尸体关键部位、抢救被幻象逼疯的队员、封存现场煞残渣。
危险度/规避：极高 / 贴近顶点——女皇七阶全开；惧之煞本体仍以逃遁为主，条件性胜利是杀容器、逼煞走，严禁战力归零式硬刚。
任务方向/奖励：女皇级掉落、卡拉克西最高战功、废土临时治理发言权。

**八阶（破顶之后 · 恐惧未尽）**
切入身份/时点：以可与战略级存在擦边的强者身份，锚定「女皇已死、惧之煞破恐惧之心殿顶逃往永春台」当时或之后数日。
初始事件：殿顶大洞灌入暴雨与煞雾，种姓军群龙无首；你收到双线急报——卡拉克西要你镇压失控幼虫，影踪派要你立刻转进永春台方向拦截惧之煞残影。
开场白建议：「天空破了。惧之煞走的时候没有留下遗体，只留下一句预言式的咒骂，和满地不知该向谁跪的螳螂妖。女皇的尸体还热，帝国已经冷了。有人把琥珀剑插进地里当新图腾，有人已经在往永春台的风里跑。你站在废墟中央，第一次看清：恐惧之心从来不是终点，只是恐惧换房间。」
关键NPC立场：**惧之煞**（逃遁中）——邀战于永春台；**卡拉克西议会**——要秩序；**影踪派**——要追击；**蝗虫卡罗兹**——要扩张权力真空中的虫群管理权；残余 **Set'thik 风刃**——为女皇复仇的散兵。
主线钩子/支线：钩子是选择「镇废土」或「追永春台」并承担另一线恶化；支线是护送新女皇谱系相关卵茧、重建一棵凯帕树的人工灌溉、清除昂舒克实验室残余。
危险度/规避：贴近顶点——惧之煞完整形态映射八阶，存在与情报优先、条件性胜利，严禁战力归零解释与单人硬刚；本切片以「逼其离开 / 转场永春台」为合法终局。
任务方向/奖励：战略级情报（永春台坐标与煞迁徙轨迹）、文明级琥珀储备支配权、跨副本因果钩子。
