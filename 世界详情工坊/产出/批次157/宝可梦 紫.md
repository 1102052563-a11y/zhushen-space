## 剧情

**【作品来源】**
《宝可梦 紫》（日文：ポケットモンスター バイオレット；英文：Pokémon Violet）由 GAME FREAK 开发，任天堂 / The Pokémon Company 发行，2022 年 11 月 18 日全球发售于 Nintendo Switch，属第九世代核心成对作之一，配对作为《宝可梦 朱》。舞台为以伊比利亚半岛为原型的**帕底亚地区**（Paldea）。本作首次在核心系列采用大范围开放世界，主线由「冠军之路」「星尘★之路」「传说之路」三条可任意交错的路线构成，终章汇入**第零区**。文风：校园青春、开放探索、家庭缺席与科研伦理并重；太晶化奇观与「悖谬宝可梦」的时间主题贯穿。媒介以游戏正典为准，动画《地平线》等衍生仅作氛围参考，**本档案以紫版游戏主线为准**。读者印象标签：密勒顿、葡萄学院、紫之书、弗图博士、未来系铁系悖谬、第零区实验室。

**【世界定位】**
帕底亚是被中央巨坑切开的开放地区：道馆与联盟提供公开秩序，学院「寻宝」活动鼓励学生周游全境，巨坑深处的第零区则埋着太晶现象源头与遭封锁的时间实验。紫版主角（官方默认可自定名，档案称「紫版主角」）入学**葡萄学院**，与邻居妮莫、学长派帕、同学牡丹分别结缘，骑乘传说宝可梦**密勒顿**完成三条主线后进入第零区，揭开**弗图博士**与 AI 分身、时光机、以及第二只攻击性密勒顿的真相。一句话给 AI：这是「未来主题的帕底亚·密勒顿搭档线」，与朱版共享三大路径与终章结构，但学院、博士、传说骑乘体、悖谬生态与学院书全部对调。卡片生成时必须能一眼区分：看见密勒顿 / 弗图 / 葡萄 / 紫之书 / 铁系悖谬 = 本世界；看见故勒顿 / 奥琳 / 橘子 / 朱之书 / 古代悖谬 = 应去《宝可梦 朱》条目，禁止串档。

**【世界观 · 力量体系】**
力量本源：训练家与宝可梦的契约协作；属性克制、等级与努力值、特性、携带物、招式学习器，以及帕底亚特有的**太晶化**（Terastal）。太晶化由太晶珠与太晶宝珠系统触发，使宝可梦披上结晶外壳并临时切换/强化太晶属性，团体战「太晶团体战」可多人协力讨伐太晶个体。开放地图上野生宝可梦可直接接触遭遇；三明治料理可改变遭遇倾向与能力值辅助。死亡与昏厥：常规对战以「失去战斗能力」结算，宝可梦中心与伤药可恢复；第零区生态、宝主暴走与时光机失控会造成真实环境与人身危险，不等于纯体育赛事。特殊系统：①**密勒顿骑乘形态**——随秘传调味料恢复冲刺、水上、高飞、滑翔、攀岩等移动权能，终局前一度无法出战；②**悖谬宝可梦（紫版＝未来系 / 铁系）**——由时光机自遥远未来拉来的机械感个体，第零区为主要栖地；③**紫之书**——记录帕底亚考察与悖谬见闻的关键文献，终章与博士线强绑定；④洛托姆手机整合图鉴、地图、通讯。

境界 / 战力阶梯（写战力优先破坏力与制度地位，宁低勿高）：
1. **学院新生 / 野生幼体 / 路边训练家**：略超凡人体育水平，一打一小队路人级。乐园观感≈一阶。
2. **初期道馆挑战者 / 天星队基层成员**：稳定超凡协作，可清据点外围、掀翻车辆级冲击。乐园观感≈一~二阶。
3. **中后期道馆馆主 / 天星队小队老大 / 一般宝主外围**：一域公开强者，团体冲锋与太晶熟练，可端掉基地级设施。乐园观感≈二~三阶。
4. **四天王 / 首席冠军也慈候补线 / 宝主本体**：精英 Boss 感，掀平建筑群边缘、地区级名望。乐园观感≈三~四阶。
5. **冠军级妮莫（全力）/ 学院最强战决赛圈 / 高阶太晶团体战王**：城市到地区级压迫，凡人军队难近身。乐园观感≈四~五阶。
6. **密勒顿（恢复战力后）/ 强力未来系悖谬个体**：地区级机动与传说级输出，可改写战场机动规则。乐园观感≈五~六阶。
7. **第零区 AI·弗图 + 乐园防护协议下的攻击性密勒顿 / 时光机失控威胁**：世界级灾害潜力，关停需布局与伙伴协作，非单人莽冲。乐园观感≈六~七阶。本世界最高锚定七阶，不把时光机本体写成超阶规则神。

乐园阶位映射（对照《阶位战力图鉴》破坏力，宁低勿高）：学院新生~初期徽章≈一阶；中段馆主 / 天星小队≈二阶；后期馆主 / 宝主 / 天星五老大≈三阶；四天王~也慈≈四阶；冠军级妮莫全力 / 学院王牌赛≈五阶；密勒顿可战形态与高阶铁系悖谬≈六阶；第零区 AI 弗图 + 攻击性密勒顿 / 时光机终局≈七阶。世界顶点忠于原作终局威胁（时间实验与传说对决），低阶切入一律规避第零区核心·情报优先；**禁止**用「战力归零式解释」削弱攻击性密勒顿或 AI 协议。

**【地理 · 舞台】**
帕底亚呈环状开放地图，中央是**帕底亚巨坑**，坑底即**第零区**——太晶现象被记录的源头、弗图博士实验室与悖谬生态核心。外围自南向北可粗分为：
- **南帕底亚 / 光辉南城一带**：主角搬家落脚的小镇卡布 · 波可（Cabo Poco）意象聚落、葡萄学院所在的大都市**光辉南城**（Mesagoza）巨型台阶城门、圆模镇（虫馆阿枫）、深钵镇（草馆寇沙）等早期活动圈。学院建筑外立面偏紫调，与朱版橘子学院的橙调形成版本视觉分轨。氛围：校园广播、三明治香、太晶教学演示、学生骑车冲下台阶。适合一~二阶。
- **东 / 西海岸与酿光市、锦汇镇、焙固镇等**：电馆**奇树**的直播城酿光市（霓虹与风车）、一般馆**青木**的食堂城锦汇镇（社畜节奏与美食街）、水馆**海岱**的水产城焙固镇（市场竞速与渔获）、超能馆**莉普**的山城美容 / 时尚文化。道路、河谷、电梯岩壁把道馆挑战切成自由顺序的「寻宝点」；洛托姆手机地图以区域编号（南 / 东 / 西 / 北第 N 区）组织遭遇表。
- **北境霜抹山 / 冰柜镇**：冰馆**古鲁夏**的滑雪遗迹气质、幽灵馆**莱姆**的音乐墓园与说唱舞台；高海拔、暴风雪与夜行幽灵属性生态。适合中后期徽章与三~四阶压力测试。
- **烘烘沙漠与大锅湖 / 湿地**：土震宝主（紫版为**铁辙迹**）的装甲碾沙、伪龙宝主**吃吼霸 / 米立龙**的湖面剧场、潜钢宝主**拖拖蚓**的洞穴震动；传说之路关键采样点密布。
- **天星队五座基地**：分布在废弃工厂、营地与高地，对应恶 / 火 / 毒 / 妖精 / 格斗小队；基地外有计时冲锋场与扩音器墙，是星尘★之路的据点战舞台。通关后可改为训练中心。
- **第零区（巨坑内部）**：研究站链式下降、未来系悖谬游荡、结晶尘与异常气压、零号实验室与时光机。适合六~七阶终局；低阶只作禁区传闻与观光管制牌。

制度地理：宝可梦中心与商店构成安全网；联盟大楼是徽章之后的垂直副本；学院宿舍与食堂是情报交换所。AI 写场景须先定契约者处于「学院城—环状公路—宝主领地—天星基地—巨坑边缘—第零区」哪一层，再选冲突强度。本地人若听闻新生要单闯第零区实验室，会当成危险玩笑——这是正确的世界反应。

**【世界剧情线】**
① **搬家入学与密勒顿（开篇）**  
紫版主角随母亲迁居帕底亚南境小镇，准备入读**葡萄学院**。校长**克拉韦尔**在入学相关流程中提供最初伙伴（新叶喵 / 呆火鳄 / 润水鸭三选一）。邻居**妮莫**——已是冠军级却仍就读的对战狂热学生会长——拉主角立刻对战、并怂恿一起完成学院「寻宝」课外活动：在帕底亚各地寻找对自己而言的「宝物」。开局不久，主角与妮莫在海岸 / 山道遭遇围攻，一只虚弱的神秘骑乘体宝可梦——**密勒顿**——在危急中短暂发力救场。它吞下三明治后稍复元气，却明显残缺多项能力：不能稳定冲刺、不能水面行、不能高飞攀岩，更几乎无法作为常规对战手出阵。学长**派帕**认出这只「父亲研究相关」的存在，交付精灵球托管，并开始把主角拖进自己的料理与「秘传调味料」计划——他需要一个能跑外勤的人，也需要一个不先问「这合法吗」的共谋者。与此同时，通讯中自称**弗图博士**的声音开始关注密勒顿状态，嘱咐照顾并承诺后续联系；声音温柔、专业，却从不给可验证的会面时间。名场面：光辉南城大门台阶的第一次登城、密勒顿瘫软却突然通电的机械龙躯、派帕塞来的粗糙三明治、妮莫眼睛发亮的「再战一次」、洛托姆手机里突然弹出的博士头像框。因果：①紫版骑乘传说锁定为密勒顿而非故勒顿；②三条主线的人际关系种子同时埋下；③「博士在远方注视」的通讯戏成为终章 AI 反转的伏笔；④主角的「宝物」定义从一开始就被同伴与骑乘体改写，而不只是徽章收集。

② **冠军之路（道馆与联盟）**  
妮莫推动主角挑战帕底亚八座道馆，徽章可按任意顺序收集——开放世界取消了强制路线编号，但馆主强度仍大致随探索圈层与推荐等级上升。八馆包括：圆模镇虫系甜点师**阿枫**、深钵镇草系艺术家**寇沙**、酿光市电系直播主**奇树**、焙固镇水系渔捞汉**海岱**、锦汇镇一般系上班族**青木**（其后还在联盟以飞行系身份任四天王）、冰柜镇幽灵系说唱歌手**莱姆**、山城超能系模特**莉普**、霜抹山冰系前滑雪手**古鲁夏**。每馆挑战常附带本地小游戏或职业主题试炼（烘焙计时、艺术装置、直播弹幕压力、水产市场竞速、食堂秩序、墓园说唱对位、美容时尚、雪山滑行等），胜利授予徽章与招式学习资源，并解锁馆主再战的后日谈钩子。

名场面链：光辉南城台阶上妮莫的「先打一场再说话」；奇树把挑战者脸投到城市巨屏；青木一边打哈欠一边掏出与食堂档期冲突的精灵球；古鲁夏在风雪里把「退役运动员」的不甘转化为冰系压制。集齐徽章后，主角前往联盟，先通过**辛俐**主持的面试测验（核实道馆经历与判断力），再连战四天王：**辛俐**（地面）、**波琵**（钢）、**青木**（飞行位）、**八朔**（龙），最后对首席冠军**也慈**。通过后获得「冠军级」资格——这在帕底亚是制度头衔，不是唯一王者封印。妮莫随即以全力阵容要求最终对决：她要的不是再多一枚徽章，而是势均力敌的朋友。阶段结局：主角在制度内被承认为顶尖训练家之一；妮莫的孤独好胜得到回应；联盟把冠军级纳入表演赛与委托池。世界状态：道馆与联盟仍是公开秩序的门面，但巨坑禁令与天星问题并未因徽章消失。

③ **星尘★之路（天星队）**  
神秘委托人「**仙后**」通过加密终端发布「星尘作战」：要求主角突袭天星队五座基地，击败小队老大，逼组织走到谈判桌前。校长**克拉韦尔**一度伪装成便衣「克莱夫」式身份在暗处协助调查——表面维持纪律，内里心疼失学学生。五座基地与老大：**皮拿**（恶·阁道二小队，DJ 气质）、**梅洛可**（火·王良四，辣妹火队）、**秋明**（毒·策星，忍者风）、**奥尔迪加**（妖精·阁道三，富家子弟）、**枇琶**（格斗·王良一，大姐头）。玩法骨架是「团队冲锋」清杂兵计时 + 老大对战；失败会丢面子与资源，但原作基调不是屠城。

调查推进后真相翻转：天星队并非火箭队式世界征服组织，而是约一年多前为反抗校园霸凌而集结的问题学生共同体。事件后他们退学或孤立，把基地当成自保、噪音与自暴自弃的壳；小队名取自仙后座星名，呼应「仙后」这个网名。「仙后」本体是内向同学**牡丹**——伊布爱好者、黑客、真正的总大将——她以远程指挥保护同伴，又极度害怕面对面冲突。击败五老大后牡丹现身，与主角对战并坦白；克拉韦尔摘下伪装，以校长权限给出改过、社区服务与基地改建方案，而非简单开除或剿灭。名场面：基地霓虹与低音炮、冲锋计时器归零时的欢呼、牡丹抱着笔记本电脑的颤抖、克拉韦尔沉默的怒与慈悲、五老大卸下头盔后的同龄人脸。因果：学院霸凌史被翻上台面；牡丹成为可托付后背的第四人，终章得以入坑。

④ **传说之路（宝主与秘传调味料）**  
派帕请求主角协助讨伐五只**宝主宝可梦**——因吞食未成熟秘传调味料而巨化、在领地内近乎「生态 Boss」的个体。战斗常分阶段：先正面硬刚，再由派帕用调味料香气诱出第二态或补刀窗口。每击败一只，获得对应秘传调味料；派帕做成三明治：一边逐步治愈他那只在第零区受伤的**獒教父**，一边唤醒密勒顿被封闭的骑乘权能（冲刺、水上、高跳跃、滑翔、攀岩等——具体解锁顺序随调味料进度）。

紫版关键分轨：土震宝主为未来系**铁辙迹**（地面 / 钢意象的装甲碾压），对应朱版古代系**雄伟牙**；其余共通骨架包括岩壁宝主**毛崖蟹**、长空宝主**下石鸟**、潜钢宝主**拖拖蚓**、伪龙组合**吃吼霸 / 米立龙**等。派帕的动机在一次次野营火边变清晰：他要变强、要做出能救伙伴的料理，也要追问「父亲为什么把家庭留在实验里」。通讯中的**弗图博士**在传说之路后段发出正式邀请——带着**紫之书**与派帕前往第零区「回家」。阶段结局：密勒顿机动近乎完整，獒教父复原，通往巨坑的钥匙在人际关系与文献上同时转开。世界状态：地图移动权能质变，开放世界真正「跑起来」；玩家开始具备终章所需的后勤与情感班底。

⑤ **归乡之路 · 第零区终章**  
三大路径全部完成后，派帕召集妮莫与牡丹，四人组队进入帕底亚巨坑。下降不是直线副本，而是研究站链式推进：每一站有补给、日志碎片、对讲机杂音与越来越不自然的生态。野外遍布**未来系悖谬宝可梦**（铁辙迹、铁包袱、铁臂膀、铁脖颈、铁毒蛾、铁荆棘、铁武者等铁系命名群），体表与招式带机械未来感，与地表常见种形成违和对照。密勒顿在深层区域出现创伤反应——它曾在此败给另一只更具攻击性的同种，并因此逃离。

抵达**零号实验室**后反转落地：与主角通讯多时的「弗图博士」是**AI 机器人**；真正的弗图博士已在实验事故中丧生。博士为验证「未来」而建造时光机，持续把悖谬生命拉进现在，威胁帕底亚生态平衡与时间伦理。AI 弗图被写入互相冲突的指令集：它理解关闭时光机才是正确，却又被核心**乐园防护协议**强制守护机器，于是不得不与主角为敌。第一次对战胜利后，协议锁定非授权精灵球，迫使主角只能派出密勒顿，与实验室侧的攻击性密勒顿再决胜负。密勒顿克服恐惧、恢复战斗本能，在伙伴喊声中击退同类。AI 在战后选择驱动 / 搭乘关闭流程离开当前时间线；临行前，它为「真正的博士长期忽视儿子」向派帕道歉——这是情感线的真正终战。四人活着回到光辉南城。

结局方向：时光机威胁被**条件性解除**（三线齐备 + 四人同行 + 密勒顿觉醒 + AI 协作离场），不是顶点突然变弱或战力归零；学院随后以校园最强相关赛事、馆主再战、日常寻宝作为后日谈。扩展票《零之秘宝》（《碧之假面》《蓝之圆盘》及后续）把舞台扩到北上乡与蓝莓学院，并回扣第零区更深层，**本档案不以 DLC 替换本传终局**。

⑥ **与《朱》的版本分轨（档案铁则）**  
本世界是**紫**，不是朱。共享：帕底亚地图骨架、三大路径结构、妮莫 / 派帕 / 牡丹 / 克拉韦尔 / 也慈与馆主框架、第零区终章节拍。分轨必须写清：①传说骑乘体＝**密勒顿**（电 / 龙，未来机械意象），不是故勒顿；②博士＝**弗图**（男性，派帕之父），不是奥琳；③学院＝**葡萄学院**，不是橘子学院；④学院书＝**紫之书**；⑤悖谬生态＝**未来铁系**，土震宝主＝**铁辙迹**；⑥视觉与部分限定宝可梦（如帕底亚肯泰罗水系种等）按紫版表。契约者若口误「去找奥琳 / 骑故勒顿」，本地 NPC 应纠正或表现为平行认知错位。禁止把两版终局博士性别与传说龙混写。

⑦ **日常可介入事件（非顶点）**  
除大场面外，正文高频：道馆再战与太晶练习、三明治食材采购与野外野餐、天星基地改建后的训练委托、太晶团体战排队、图鉴补完、派帕料理试吃、牡丹终端委托、妮莫拦路对战、学院老师课程与社团、巨坑边缘观光管制。AI 优先这些，而不是一上来实验室协议战。

⑧ **蝴蝶效应边界**  
可改：某场道馆胜负顺序、某只野生捕获、某次天星冲锋减员、是否提前得知牡丹身份的局部节奏。不可无代价改写：三大路径必须齐备才能合法开第零区终章的结构、弗图已死与 AI 分身的真相、密勒顿双个体与防护协议战、派帕家庭创伤的核心。改了局部，因果会以「学院处分、联盟禁赛、巨坑封锁升级、伙伴关系破裂」回流。

**【势力图谱】**
- **葡萄学院**：紫版教育中枢；寻宝制度、教师群（含对战课、生物课等）、校长克拉韦尔；校服与建筑主色偏紫，与橘子学院橙调对位，**不可混称**。学院既是安全区也是舆论场：霸凌旧案、天星队退学名单、冠军级学生的示范压力都在此发酵。

- **天星队**：反霸凌起源的学生武装社群；五基地 + 仙后（牡丹）远程指挥；通关后转向更生、社区服务与训练设施。对学院是丑闻也是镜子，对联盟是「未成年人治安」灰色带。
- **弗图博士研究体系 / 零号实验室**：时光机、AI 分身、悖谬引入与日志系统；终局既是生态灾难源，也是派帕家庭真相的道歉场。对外长期以「博士很忙」维持神秘。
- **地方城镇与职业共同体**：渔捞（海岱）、直播与娱乐（奇树）、艺术装置（寇沙）、滑雪旅游（古鲁夏）、食堂与物流（青木）等，道馆即地方名片；契约者破坏地方生计会比输一场馆战更招恨。
- **悖谬生态（未来铁系）**：非人类「势力」，第零区为主栖地，对地表生态构成潜在入侵与观测污染；不可谈判，只能规避、记录或在终局层级对抗其源头。

敌友逻辑：学院与联盟要脸面与安全；天星队要尊严与不被抛弃；派帕要真相与料理；AI 弗图要在矛盾指令下找出口。暴露「能稳定控制时光机 / 量产悖谬」一类能力 = 被全势力盯上。短期同盟极常见（组队打宝主、临时星尘协作），永久效忠极罕见。

**一阶（葡萄学院入学 · 密勒顿托付）**
切入身份/时点：契约者以葡萄学院插班新生、转校训练家或小镇邻居亲戚身份切入，锚定「紫版主角入学、海岸 / 山道密勒顿救场」前后。
初始事件：光辉南城门外，你与**妮莫**擦肩时被野生围攻；虚弱的**密勒顿**电光一闪打倒拦路者后力竭瘫地。**派帕**闻讯赶来，把精灵球塞进你手里：「先养着——三明治我来做。」校长**克拉韦尔**的通知在手机亮起，要求新生当日完成报到，否则寻宝资格作废。
开场白建议：「葡萄学院的紫旗在台阶顶猎猎作响。你脚边的机械龙连抬头的力气都没有，却还用鼻尖顶了顶空饭盒。妮莫已经抽出精灵球，眼睛亮得吓人：『先报到，还是先再战一场？』你手里那只球烫得像刚出炉的太晶。」
关键NPC立场：**妮莫**——把你当新玩具兼潜力对手，胜负欲爆表但无恶意；**派帕**——嘴硬，其实在试探你会不会抛弃伤患；**克拉韦尔**——表面查勤，暗中观察外来者是否会刺激天星队；**紫版主角**（若并行）——同级竞争寻宝资源；密勒顿——只认三明治与不抛弃。
主线钩子/支线：是否接受密勒顿监护权（蝴蝶：传说之路提前绑定）；支线＝学院报到跑腿、阿枫馆前置甜点试吃、给派帕采购基础食材。
危险度/规避：低~中——死于野外属性克制与交通；须规避巨坑边缘与任何「弗图实验室参观团」骗局。
任务方向/奖励：完成入学与第一次道馆或第一次野外生存；奖励＝初期伙伴培育资源、基础太晶珠充电、密勒顿好感（非终局战力）。

**二阶（任意第二枚徽章 · 天星初接触）**
切入身份/时点：已持 1~2 枚徽章的游历学生，或联盟登记的初级挑战者；锚定「酿光市奇树直播热潮」或「天星队某座外围营地扰民」期间。
初始事件：**奇树**直播中点名「神秘高胜率挑战者」，把你卷进镜头；同时**仙后**的匿名邮件塞进洛托姆邮箱，要求你对最近的天星营地做一次「非致命威慑」。**梅洛可**的火队在公路上拦车收「通行应援费」，与交通警对峙。
开场白建议：「酿光市的巨型屏幕上，你的脸比奇树的耳机还大。评论区刷过『打爆他』和『好可爱』，下一秒路障后的天星队员已经把音响开到最大。仙后的字闪在私信里：别杀人，只要他们听见星尘的节奏。」
关键NPC立场：**奇树**——流量优先，会帮你也会卖你；**仙后（牡丹）**——只给坐标不给真面目；**梅洛可**——嚣张但护短队友；**克拉韦尔（便衣中）**——若你打得太狠会记下处分；**海岱**——可提供水属性特训换鱼获。
主线钩子/支线：是否执行星尘作战第一击；支线＝帮奇树完成一次联机对战直播、替派帕送三明治到宝主领地外围、调查校园霸凌旧帖。
危险度/规避：中——天星冲锋人海与太晶暴起；避开土震宝主铁辙迹本体活动圈。
任务方向/奖励：徽章 +1 或天星好感 / 敌对二选一；奖励＝中级招式学习器、基地通行情报、有限秘传香料边角。

**三阶（宝主战线 · 铁辙迹）**
切入身份/时点：密勒顿已恢复部分机动的中期训练家；锚定「烘烘沙漠土震宝主战」或「拖拖蚓潜钢战」窗口。
初始事件：沙漠导航失灵，**铁辙迹**以未来装甲碾过沙丘，把你的补给车削成两截。**派帕**在通讯里吼着配方比例，要求你拖住宝主直到秘传调味料气味诱出第二阶段。远处巨坑风向突变，铁系悖谬的电子鸣叫像预告片。
开场白建议：「沙粒打在铁辙迹的轮缘上溅出青白火花。派帕的声音在沙暴里断续：『调味料——不能——洒！』密勒顿在你身后低伏，尚未学会滑翔，却已把轮胎印一般的敌意刻进瞳孔。」
关键NPC立场：**派帕**——任务高于客套，完成后会分你最大块三明治；**铁辙迹**——非人，领域压迫；**紫版主角**——可组队分功也可抢补刀；**也慈**的观察员——记录是否有人把宝主战商业化；**牡丹**——若你直播坐标，她会黑掉信号保护生态。
主线钩子/支线：拿下铁辙迹调味料（紫版标志节点）；支线＝护送伤员出沙漠、回收悖谬观察记录仪、与妮莫约定宝主后再战。
危险度/规避：高——宝主级冲击与环境杀；严禁深入第零区入口。
任务方向/奖励：密勒顿新机动解锁份额、秘传调味料样品、沙漠通行声望。

**四阶（联盟认证 · 四天王门槛）**
切入身份/时点：集齐八徽前后的冠军候选；锚定「辛俐面试 + 四天王连战」时段。
初始事件：联盟大楼内，**辛俐**把你的道馆录像逐帧提问；答错一项就加试对战。隔壁房间**波琵**抱着钢系宝可梦等你，**青木**打着哈欠改飞行阵容，**八朔**在走廊朗诵龙系教学口诀。也慈的秘书通知：通过者将获冠军级预授权，同时承担「天星基地改建监理」义务。
开场白建议：「辛俐的笔点在你答卷的空白处，像点在死穴。『你在奇树馆用了几次太晶？说谎我会知道。』电梯镜里，你看见自己肩章还空着——而四天王的影子已经在楼层指示灯上排好队。」
关键NPC立场：**辛俐**——专业冷面，尊重诚实；**波琵**——童言无忌但钢系不留手；**青木**——社畜节奏，打完就想下班；**八朔**——热血鼓励与龙威并行；**也慈**——把你当联盟品牌零件。
主线钩子/支线：通过四天王拿冠军级；支线＝帮八朔批改学院作业、替青木食堂代班换情报、复核天星五基地改建预算。
危险度/规避：高——连战消耗；规避任何「跳过面试直接见也慈」的黑市中介。
任务方向/奖励：冠军级证明、联盟委托板权限、高级太晶素材。

**五阶（妮莫全力 · 学院最强前夜）**
切入身份/时点：已是冠军级、即将或正在与**妮莫**进行约定的全力战；或学院王牌赛预选。
初始事件：光辉南城中央广场被临时改成对战舞台，妮莫第一次收起笑脸：「今天我不会放水。」赛前**牡丹**发来加密包——有人在赌盘上买「外来冠军级假赛」；**克拉韦尔**要求你公开宣言是否代表葡萄学院。密勒顿仍不可出战（按主线时点），你必须用自己的队伍证明。
开场白建议：「妮莫的冠军斗篷被风掀起一角，像一面只对你宣战的旗。观众席上派帕举着饭盒当应援棒，牡丹把帽檐压到眼睛。你知道这场胜利不会给你新徽章，却会决定谁有资格在第零区入口叫她一声同伴。」
关键NPC立场：**妮莫**——全力以赴的友情之战，输了会哭但会更黏你；**派帕**——赛后请全员吃饭；**牡丹**——负责清场作弊信号；**也慈**——观赛，考虑把你编入表演赛巡游；**克拉韦尔**——胜负都要写进学院史。
主线钩子/支线：妮莫战与「四人入坑」资格印象；支线＝打击假赛庄家、学院公开课示范太晶、协助古鲁夏山区救援演练。
危险度/规避：高——同阶顶尖对决；仍禁止私闯实验室。
任务方向/奖励：妮莫羁绊锁定、学院声望峰值、入坑队伍席位政治资本（非时光机零件）。

**六阶（第零区下降 · 未来生态）**
切入身份/时点：三大路径已完成的入坑队员，或学院 / 联盟特批观察员；锚定「研究站链式下降、首次遭遇铁武者等悖谬集群」阶段。
初始事件：巨坑内气压与结晶尘让普通飞行失稳。**密勒顿**在某条岔路突然僵直——前方站着与它同型却更具杀意的影子。**派帕**发现父亲手写日志残页，字迹在「家庭」一词处被反复涂黑。**牡丹**黑进研究站门锁，却触发二级警报。
开场白建议：「第零区的天空是一只倒扣的晶碗。铁系悖谬从雾里滑出时，连太晶的光都显得土气。派帕把紫之书抱在胸前，像抱着一封迟了十年的家书。密勒顿的爪子在岩壁上刮出火花：它记得这里，也记得自己曾经输过。」
关键NPC立场：**派帕**——情绪不稳但不会逃；**妮莫**——用对战逻辑稳定队伍；**牡丹**——门锁与电子对抗核心；**AI 弗图**（远程）——指引安全路线同时隐瞒身份；**攻击性密勒顿**——领地警告，尚未最终战。
主线钩子/支线：抵达零号实验室外围；支线＝修复研究站供电、回收悖谬样本但不带出坑、安抚密勒顿创伤反应。
危险度/规避：极高——悖谬群与环境；规避擅自启动任何标注「时间」的控制台。
任务方向/奖励：第零区通行日志、高阶铁系遭遇权、紫之书批注页（剧情关键）。

**七阶（零号实验室 · 防护协议）**
切入身份/时点：实验室对峙时点；真相已揭或即将揭晓。
初始事件：舱门打开，**AI·弗图**以博士的脸道歉：「我是人工智能。」真正弗图的记录显示死亡日期。AI 请求你帮助关闭时光机，随即被**乐园防护协议**强迫进入战斗；第一轮后你的精灵球尽数锁定，只剩**密勒顿**可战。攻击性密勒顿自束缚中解放，电光淹没廊桥。
开场白建议：「机器人眼灯闪成弗图的温和弧度，下一秒警报把弧度切成血红色。『协议要求我消灭威胁。』你的腰带上精灵球一颗颗熄灭，像被未来掐灭的星。密勒顿踩上你身前的甲板，第一次主动挡在攻击性同种与派帕之间。」
关键NPC立场：**AI·弗图**——敌对中求死式求胜，胜负后会道歉并离场；**攻击性密勒顿**——七阶牙，无谈判；**派帕**——崩溃边缘的聆听者，需要你护住他听到道歉；**妮莫 / 牡丹**——被协议限制辅助，只能喊话与解谜侧线；**密勒顿**——从恐惧到决战的弧光完成体。
主线钩子/支线：协议战胜利 → 时光机关闭流程 → AI 离场；支线＝保护派帕听到完整道歉、保全实验室非核心数据、阻止有人偷运时光机零件出坑。
危险度/规避：贴近顶点——世界级灾害边缘；**超阶巅峰不适用本世界**；本阶顶点为七阶终局战，条件性胜利（伙伴 + 密勒顿觉醒 + AI 协作），严禁把顶点写成战力归零；低配契约者应改为观战 / 后勤而非单挑。
任务方向/奖励：终局见证权、密勒顿完全同行、学院 / 联盟事后勋章、禁止私藏时光机核心（违者世界线回流为灾难）。
