## 剧情

**【作品来源】**
《X战警》（X-Men，大陆通称《X战警1》）是二十世纪福斯出品、布莱恩·辛格执导、大卫·海特编剧的超级英雄电影，劳伦·舒勒·唐纳与雷夫·温特等监制。2000年7月北美上映，片长约104分钟，为《X战警电影系列》首部正典，灵感源自斯坦·李与杰克·科比漫画团体设定，人物命运与主线因果以本片银幕正典为准（不倒灌《X2》《第一战》等后续档正文）。文风：社会寓言式群像、偏见与注册法案政治、黑黄制服战术小队；核心矛盾是查尔斯·泽维尔的共存路线 vs 艾瑞克·蓝歇尔（万磁王）的强制进化。卡片锚点：**自由女神像之战**——万磁王将转化机器架上自由女神像，以小淘气为电池强行启动辐射场，企图把埃利斯岛峰会的各国首脑变为变种人；金刚狼、镭射眼、暴风女、琴·葛雷登顶对抗兄弟会，摧毁机器、阻止辐射扩散。

**【世界定位】**
约2000年代近未来：变种人（Mutant）已公开存在于人类社会，美国参议院推动《变种人注册法案》；西彻斯特的泽维尔学校暗中培养年轻变种人，吉诺沙岛上万磁王的兄弟会准备以机器改写人类基因。一句话：以罗根（金刚狼）与小淘气入校为视点，人类与变种人第一次在银幕上正面碰撞「共存还是征服」。契约者是变数：可改火车站疏散、Cerebro破坏时机、女神像侧翼战损，不可无代价抹杀凯利参议员溶解之死、小淘气被掳、机器启动与罗根输血式自愈的核心因果。

**【世界观 · 力量体系】**
力量本源为 **X基因觉醒** 带来的个体异能，无统一修炼阶梯；战力按**实际破坏力/情报层级**对照乐园阶位（宁低勿高）：
1. **凡人与基层**：密西西比小镇居民、加拿大酒吧拳赛观众、火车站警察、峰会安保普通员——枪械/徒手。≈一阶。
2. **觉醒初期变种人 / 精英凡人**：小淘气（接触吸血未控）、冰人巴比·德雷克（校园级制冰）、反变种民兵与国会助理——碎冰裂门、近身制服常人。≈二阶。
3. **成体系变种中坚 / 战术小队**：镭射眼史考特·萨默斯（视镜控光）、暴风女奥罗罗·门罗（天气）、剑齿虎维克多·克里德（野兽格斗）、蟾蜍莫蒂默·托因比（弹跳毒液）——端据点、拆楼层局部。≈三阶。
4. **质变精英·一城压力**：金刚狼罗根（亚德曼合金骨爪+自愈）、魔形女瑞文·达克霍姆（变形渗透）、琴·葛雷（心灵感应+念力，本片未满开）——掀建筑群、高速肉搏、身份渗透。≈四阶。
5. **城市至地区灾难级**：万磁王艾瑞克·蓝歇尔（磁场操控金属与机甲）、转化机器辐射覆盖（对凡人强制突变）、Cerebro 脑波增幅器全开搜索——可瘫痪城市金属结构、威胁峰会一岛。≈五阶。
6. **地区至准世界级战略**：X战警小队联合作战峰值、X战机远程投送、注册法案与国会政治链、塑料监狱级封锁——国家级协同与制度武器。≈六阶。
7. **世界级战略节点**：自由女神像上机器全功率辐射场、若成功则「全球首脑突变」的政治末日；Cerebro 被破坏后的昏迷连锁。≈七阶。
8. **行星/规则阴影·本片顶点**：万磁王完整磁场权柄+转化机器体系（对凡人种群的强制改写）；塑料监狱中的艾瑞克仍是本时间线最大可对话威胁。≈八阶；对低阶为**超阶情报事件**时写「规避顶点·情报优先」。

乐园阶位映射：凡人/安保≈一阶；觉醒初期变种/精英凡人≈二阶；镭射眼/暴风女/剑齿虎/蟾蜍中坚≈三阶；金刚狼/魔形女/琴·葛雷≈四阶；万磁王与转化机器覆盖≈五阶；X战警联合作战与政治战略≈六阶；女神像辐射场全开与峰会级后果≈七阶；万磁王完整权柄+机器体系≈八阶。**宁低勿高**。世界顶点（本片）=万磁王+转化机器；低阶切入一律**规避顶点·情报优先**；严禁「被塑料监狱关着所以战力为零」式解释——他是被规则关押，不是被削成凡人。

死亡规则：凡人与多数角色身死即死；凯利参议员遭强制突变后身体排斥溶解为液体（本片真死）；小淘气触碰可抽干生命与能力至昏迷/濒死；金刚狼自愈极强但仍可被重创昏迷。特殊系统：X基因、亚德曼合金、万磁王头盔（隔绝心灵感应）、Cerebro、转化机器、注册法案制度武器、塑料/非金属监牢。

**【世界剧情线】**
① **前史·奥斯威辛的门**  
1944年纳粹占领波兰，十四岁犹太男孩艾瑞克·蓝歇尔在奥斯威辛与父母被强行拆散，情绪爆发扭曲金属栅门，遭卫兵击昏。这段记忆锁死他一生的政治结论：人类对「异类」的灭绝不会停，唯有力量能改规则。半个多世纪后，他以万磁王身份出现在注册法案听证会的阴影里，与老友查尔斯·泽维尔分道扬镳——查尔斯要共存教育，艾瑞克要先下手为强。

② **开篇·注册法案与小淘气出走**  
美国参议员罗伯特·凯利在国会推动《变种人注册法案》，要求变种人公开身份与能力，演讲充满恐惧动员。同期，密西西比十七岁少女玛莉·丹肯托不知自己能通过接触吸收生命与能力，亲吻男友大卫后使其陷入三周昏迷，她改名「小淘气」北上逃亡。世界状态：凡人媒体把变种人写成定时炸弹；变种人社区分裂为「藏好」与「打回去」。名场面应具体到：凯利手中「名单」式修辞、小淘气手套不敢摘、罗根酒吧里爪子弹出与伤口愈合。

③ **立名·金刚狼入局与公路伏击**  
小淘气在加拿大酒吧遇见罗根（金刚狼）：地下拳赛、金属爪、自愈因子。两人上路后遭兄弟会成员剑齿虎维克多·克里德袭击——万磁王误判目标以为要的是罗根。查尔斯派出镭射眼史考特·萨默斯与暴风女奥罗罗·门罗及时救下二人，带回西彻斯特X宅邸。罗根醒来发现这里是变种人学校：危险室式训练痕迹、制服、Cerebro 的传说。查尔斯请罗根暂留，调查为何兄弟会盯上他；小淘气入学，对冰人巴比·德雷克产生好感。因果：罗根被绑进「共存派 vs 征服派」棋盘，小淘气成为后续电池人选的伏笔尚未揭开。

④ **吉诺沙实验·凯利突变**  
魔形女（罗刹女）瑞文·达克霍姆与蟾蜍莫蒂默·托因比绑架凯利参议员，送至吉诺沙岛。万磁王以自身磁场为能源启动**转化机器**，对凯利强制激发变种细胞——实验「成功」：凯利获得身体伸缩变形能力，却趁机逃出。查尔斯通过后续读心得知：驱动机器几乎抽干万磁王生命，故艾瑞克需要更合适的「电池」。凯利逃到学校求助，身体开始排斥突变，最终在众人面前溶解成液体死亡——这是本片对「强行把人变成变种人」的残酷判决，也锁死X战警必须阻止机器对峰会首脑使用。

⑤ **校园渗透·小淘气被掳**  
小淘气夜探罗根房间，罗根噩梦中误伤她，她被迫吸收其自愈才活下，罗根昏迷。魔形女潜入校园，变形为巴比·德雷克，谎称X教授因事故生怒，诱使小淘气离校。查尔斯用 Cerebro 锁定火车站；罗根抢先上车劝回。万磁王、蟾蜍、剑齿虎突袭站台：罗根全身合金被磁场轻松制伏，小淘气被掳。查尔斯远程占据剑齿虎意识本可阻止，但万磁王以警察为人质迫使他松手——共存派第一次在人质伦理上败给征服派的冷酷算术。魔形女同时破坏 Cerebro，查尔斯再次使用时遭反噬陷入昏迷。

⑥ **自由女神像之战（卡片锚点·质变副本）**  
琴·葛雷冒死修复并使用 Cerebro，定位兄弟会在**自由岛自由女神像**：机器对准埃利斯岛峰会，万磁王要把各国首脑变成变种人，逼人类「接受」新秩序。镭射眼、暴风女、罗根、琴组成行动X战警登岛。像内逐层作战：击败魔形女（罗根利爪重创其伪装）、蟾蜍（暴风女雷电与高空）、剑齿虎（镭射眼光束轰落）。顶层：万磁王将能力转移给小淘气，强迫她启动机器，辐射场开始成形。罗根上前牵制艾瑞克；镭射眼抓住空隙击落万磁王；罗根摧毁机器核心。小淘气濒死，罗根将手贴上她脸，主动输出自愈因子救活她，自己重伤昏迷。辐射未扩散至峰会——锚点胜利条件达成：机器毁、首脑未突变、小淘气存活。

⑦ **终局·塑料监狱与寻根**  
查尔斯与罗根先后苏醒。新闻画面：魔形女假扮凯利参议员，宣布搁置注册法案相关事务——政治线并未因战场胜利而纯洁，变形者仍在国会话语里行走。查尔斯给罗根一条过去线索：加拿大森林中废弃军事基地（碱湖方向），罗根告别小淘气踏上寻根路。查尔斯前往全塑料构造牢房探望艾瑞克，两人下棋；万磁王警告终将逃出并完成使命，查尔斯答「我会一直在这里阻止你」。结局锁死：机器计划失败、万磁王被关、魔形女在逃并操控政治表象、X战警从学校私兵变成公开救场的事实、罗根离校寻根、《X2》阴影未展开。

⑧ **微观政治与日常可介入**  
除大场面外，本世界高频事件：注册法案听证会旁听与抗议、变种人诊所与黑市抑制剂、校园宵禁与能力失控处分、火车站安检与媒体直播、峰会通行证与非金属安检门、Cerebro 使用许可、吉诺沙逃亡者收容、亚德曼相关军工谣言。AI优先写这些可接触层，而非一上来对轰万磁王。

⑨ **蝴蝶效应边界**  
契约者可改：公路伏击伤亡、Cerebro 破坏是否被提前发现、火车站人质谈判多争取的秒数、女神像侧翼是否多救一名安保、凯利溶解现场的样本是否被谁取走。不可无代价改写：小淘气被选为电池的逻辑、机器启动与摧毁的主轴、罗根输愈救淘气、万磁王战败入塑料监狱、魔形女假凯利搅局。改局部后因果以「更高辐射泄漏、更多首脑伤亡、学校被曝光清剿、万磁王提前越狱」回流。

> 阶位↔本片强度：凡人/安保≈一阶；觉醒初期变种/精英凡人≈二阶；镭射眼/暴风女/剑齿虎/蟾蜍中坚≈三阶；金刚狼/魔形女/琴·葛雷≈四阶；万磁王与转化机器覆盖≈五阶；X战警联合作战与政治战略≈六阶；女神像辐射场全开与峰会级后果≈七阶；万磁王完整权柄+机器体系≈八阶。世界顶点（万磁王+转化机器）为跨阶/情报级存在，低阶切入一律「规避顶点·情报优先」。本世界仅覆盖一～八阶。

**一阶（默里迪恩·出走夜）**
切入身份/时点：契约者以密西西比小镇邻里、加油站夜班、长途车乘客或小淘气同班同学身份，锚定玛莉·丹肯托吻晕男友、决定北上前后。
初始事件：急诊室封锁，有人说「她碰了一下他就不动了」；你被托付送一个裹紧手套的女孩上北上车——车窗外已有记者在喊「变种人」。
开场白建议：「手套拉链拉到顶，她还是不敢碰扶手。广播里参议员在谈注册，售票员在谈证件——你手里的车票像一张逃票，也像一张罪证。」
关键NPC立场：**玛莉·丹肯托 / 小淘气**——怕被碰也怕被丢下；**大卫**（昏迷）——只是代价的标本；本地警察——要隔离「病人」；反变种邻居——要报警领赏。
主线钩子/支线：护送小淘气离开默里迪恩；支线＝藏起她的真实姓名、毁掉吻伤目击者录像、把抑制剂谣言卖给黑市或交给未来的学校眼线。
危险度/规避：中——死于私刑、车祸与恐慌开枪；规避主动皮肤接触未知变种。
任务方向/奖励：存活+北上线索；奖励＝手套式隔绝贴、假身份证、急诊封口费。

**二阶（劳克林酒吧·公路伏击前夜）**
切入身份/时点：加拿大地下拳赛赌徒、修车行学徒或边境货车副驾，锚定罗根拳赛之夜至剑齿虎伏击前。
初始事件：你押注的「爪子男」赢了又愈合；一个戴连帽衫的女孩要搭车，而树林里有比狼更重的呼吸——有人出钱买「金属骨架的活体」。
开场白建议：「拳馆地板上的血到天亮就干了，他的伤口却先干。外面引擎熄火声不像警察——你藏在后斗里的撬棍，第一次觉得太轻。」
关键NPC立场：**罗根 / 金刚狼**——不信人但护淘气；**玛莉·丹肯托 / 小淘气**——要一张通往「正常人」的票；**维克多·克里德 / 剑齿虎**——要活捉目标；酒吧老板——要消灾。
主线钩子/支线：公路伏击存活与是否把罗根坐标卖给不明买家；支线＝调包军牌信息、给救护车清路、录下爪子却不上传。
危险度/规避：中高——剑齿虎级野兽；规避正面对轰合金爪与野兽力。
任务方向/奖励：逃亡资金、旧皮卡钥匙、半真的「武器X」传闻纸条。

**三阶（西彻斯特校园·入学与宵禁）**
切入身份/时点：新入学变种学生、校园维修工或查尔斯授权的编外观察员，锚定小淘气入学、罗根暂留、魔形女尚未得手时。
初始事件：宵禁铃响，你的室友手心结冰；地下层门禁闪红——有人用「巴比·德雷克」的脸进出 Cerebro 走廊，而真的巴比在宿舍睡觉。
开场白建议：「草坪干净得像谎言。教室教控制，地下室教战争——你口袋里的访客牌写着临时，走廊尽头的电梯却写着永夜。」
关键NPC立场：**查尔斯·泽维尔 / X教授**——要秩序与保密；**史考特·萨默斯 / 镭射眼**——要纪律；**琴·葛雷**——要医疗与读心边界；**巴比·德雷克 / 冰人**——要证明自己不是内鬼；**瑞文·达克霍姆 / 魔形女**——要渗透。
主线钩子/支线：识破伪巴比或保护真学生；支线＝宵禁外送药、危险训练事故善后、拒绝向国会线人卖学生名单。
危险度/规避：中高——渗透与能力事故；规避擅自启动 Cerebro。
任务方向/奖励：校园通行证、基础能力抑制贴、一节「控制课」合格记录。

**四阶（火车站台·掳人夜）**
切入身份/时点：铁路警强化编组、学校外勤或能与金刚狼短暂同场的契约者，锚定小淘气离校、罗根上车、万磁王突袭站台。
初始事件：列车广播要全员下车；金属扶手忽然自己弯成牢笼，合金爪子的男人被按在地上——有人要你在「救警察人质」和「抢回戴手套的女孩」之间选先砍哪条链。
开场白建议：「站台的风里有臭氧和枪油。教授的声音在你脑子里闪了一下又断了——金属在唱歌，人质在哭，女孩的手套掉在铁轨旁。」
关键NPC立场：**罗根 / 金刚狼**——要淘气；**艾瑞克·蓝歇尔 / 万磁王**——要电池；**查尔斯·泽维尔**——要两全；**维克多·克里德 / 剑齿虎**——要撕开阻拦者；警察指挥官——要你放下「怪物」。
主线钩子/支线：火车站人质伦理与小淘气归属；支线＝切断站台供电减磁场精度、转移媒体镜头、给史考特发坐标。
危险度/规避：高——万磁王与剑齿虎同场；禁止金属战场硬刚磁王。
任务方向/奖励：非金属攀爬套件、站台监控残片、学校紧急召回令（带监视）。

**五阶（吉诺沙残骸·凯利溶解后）**
切入身份/时点：变种诊所医生、学校应急小组或国会泄密调查编外员，锚定凯利逃到学校、溶解死亡、团队确认机器杀伤性前后。
初始事件：参议员在医疗床上伸长手臂又融化；化验报告写着「排斥」——有人出双倍买「突变稳定剂假说」，兄弟会余党要烧掉现场。
开场白建议：「液体还在滴，国旗别针浮在上面。有人说这是天谴，有人说这是科学——你握住取样管时，管壁在震，像有第二个心跳。」
关键NPC立场：**琴·葛雷**——要样本与真相；**查尔斯·泽维尔**——要定性机器为灭绝武器；**罗伯特·凯利**（临终）——要你听完恐惧与悔；兄弟会回收者——要灭口。
主线钩子/支线：凯利之死定性与样本归属；支线＝伪造「自然死亡」新闻、保护溶解目击学生、追踪吉诺沙逃出路线。
危险度/规避：高——政治灭口与能力残留污染；规避宣称可量产稳定突变。
任务方向/奖励：医疗级隔绝箱、半份机器频谱数据、国会旁听证（高危）。

**六阶（Cerebro 反噬·定位自由岛）**
切入身份/时点：地下层技术员、琴·葛雷助手或能在教授昏迷期协调外勤的战术契约者，锚定 Cerebro 被破坏、查尔斯昏迷、琴强行定位兄弟会。
初始事件：头盔下的人鼻血流进键盘；屏幕跳出自由岛轮廓——你必须在十分钟内决定：让琴继续赌命，还是把坐标卖给政府换「合法打击」。
开场白建议：「地下比夜更黑。机器哼了一声，像整座学校的脑在抽筋——有人喊教授倒下了，有人喊峰会明天开始，你的手按在启动键上。」
关键NPC立场：**琴·葛雷**——要坐标；**史考特·萨默斯 / 镭射眼**——要立即出击；**奥罗罗·门罗 / 暴风女**——要天气窗口；**瑞文·达克霍姆 / 魔形女**——要你们找错岛；政府热线——要接管学校。
主线钩子/支线：Cerebro 修复与自由岛坐标确认；支线＝清除后门程序、伪造错误定位骗开兄弟会、保护昏迷的查尔斯不被「接管」。
危险度/规避：极高——感应反噬与政治接管；规避宣称可稳定驾驭满功率 Cerebro。
任务方向/奖励：X战机副席一次性授权、破坏频谱日志、与小队的实名协作记录。

**七阶（自由女神像顶层·毁机窗口）**
切入身份/时点：能登顶参战的高阶契约者（X战机突击编入、非金属特战、或与金刚狼同步的侧翼），锚定机器启动、小淘气供能、辐射场成形前后。
初始事件：火炬层金属全部「活」了；女孩尖叫着成为电池，磁场按住合金爪子——你获得一个窗口：砍电缆、挡辐射边缘、或替罗根多撑三秒让镭射眼光束穿过。
开场白建议：「女神冠上的风是热的。机器的光把峰会的灯映成靶心——有人在喊摧毁，有人在喊救人，你的靴底粘着要融化世界的尘。」
关键NPC立场：**罗根 / 金刚狼**——要毁机并救淘气；**艾瑞克·蓝歇尔 / 万磁王**——要辐射完成；**史考特·萨默斯 / 镭射眼**——要射击窗口；**奥罗罗·门罗 / 暴风女**——要控场；**玛莉·丹肯托 / 小淘气**——要结束痛苦。
主线钩子/支线：毁机与救电池的秒差；支线＝击退剑齿虎残势、标记魔形女逃逸路线、减少峰会侧平民目击创伤。
危险度/规避：极高——五～七阶交火与辐射；条件性胜利，严禁改写「机器必须被毁、淘气必须被救」的双核而无代价。
任务方向/奖励：机器残核数据、亚德曼微屑（微量）、X战警承认的战场编号。

**八阶（塑料监狱·假凯利与碱湖路）**
切入身份/时点：能接触战后政治收尾、监狱探视外围或罗根寻根交接的巅峰契约者，锚定魔形女假凯利播出、万磁王下棋放话、罗根赴碱湖线索前后。
初始事件：新闻里的凯利笑容不对；塑料墙后的人说「还会见面」——有人要你在交接前复制磁场频率，有人要你把碱湖坐标卖给军工，有人要你当众拆穿假参议员。
开场白建议：「城市庆祝英雄，屏幕里的死人还在立法。棋盘上只剩残局，金属在很远的地方共振——你知道战争没结束，只是换了一张人脸。」
关键NPC立场：**查尔斯·泽维尔 / X教授**——要守住共存叙事；**艾瑞克·蓝歇尔 / 万磁王**——要越狱与再战，**超阶巅峰侧：存在·情报优先/条件性胜利**，本片内严禁击杀式硬刚「关着就等于死」；**瑞文·达克霍姆 / 魔形女**——要政治线；**罗根 / 金刚狼**——要过去。
主线钩子/支线：假凯利拆穿与否、碱湖线索真伪、监狱安防漏洞；支线＝销毁你的女神像目击档案、保护小淘气术后隔离、向查尔斯提交「变形者已入国会」证据链。
危险度/规避：贴近顶点·情报优先；不可无回流永久截留转化机器，不可在本时间线内「彻底消灭」万磁王理念与越狱可能。
任务方向/奖励：终局情报锚、塑料监狱外围权限、与罗根或学校的一次实名债务（双刃）。
