# 日本战国：功名十字路

## 剧情

**【作品来源】**
作者**吉良上总介**，起点中文网签约长篇历史小说，分类「历史·外国历史」。标签：穿越、争霸、日本、考据、杀伐果断。连载中（检索时约 65 万字、更新至第 197 章量级），书号 1049112947。同作者另有战国向作品线（如《日本战国：真田家的野望》等检索可见）。文风：考据向战国权谋 + 穿越者搞钱经营 + 杀伐果断；名词多做中文可读化处理；作者声明史料多歧说、择一说法服务剧情，不编造「定论终局」。

**【世界定位】**
永禄前后的日本战国尾张—美浓舞台。现代人**王瑾**穿越为岩仓织田家家老之子**山内一丰**（通称伊右卫门），在浮野之战—岩仓城陷落的绝境中求生，以木曾川河运、稻木庄年贡与战场军功「太想进步」，逐步从牢人/下级武士爬到织田家侍大将乃至家老台面，与**宁宁**、**织田信长**、**木下秀吉（藤吉郎）**等人的命运交织。档案只写至已公开连载层，**不编终局**。

**【世界观 · 力量体系】**
本世界是**冷兵器—铁炮过渡期的写实战国**，无灵力、无系统面板、无超凡法术。力量=人身武艺 + 铁炮火器 + 兵力编制 + 粮饷知行 + 同盟与情报。死亡即死，无复活；「进步」靠首级、知行、官途与人脉，不是修炼飞升。

阶层与战力表现（自下而上）：
1. **足轻 / 暴民 / 落武者狩**：竹枪镰刀、村社互助杀败兵；能围杀落单武士，抗打=普通人。
2. **地侍 / 编制外武士**：有一点经济与武备，可出仕入家臣体系。
3. **下级武士 / 牢人**：长枪打刀、初阵讨首；单挑小队农民或一两个同级。
4. **侍大将 / 一番队头目**：能统数百至千余军势、乱军取首、指挥铁炮与长枪阵；「鬼山内」一类绰号出现在此层。
5. **城主 / 国人众 / 家老**：支配一城一郡石高、签发安堵状、左右主家决策；个人武艺未必顶尖，但账本与军役决定生死。
6. **战国大名（今川义元、织田信长、斋藤龙兴、松平元康等）**：举国军势、联姻与包围网；单挑仍是凡人，胜负看军制与奇袭。
7. **幕府将军 / 朝廷名义（足利义辉等）**：权威大于野战破坏力；上洛、官位、尾州守护等是政治核武器。

铁炮：已出现在战场（铅弹贴身飞过、落武者狩闻硝烟而逃），能瞬间改写小规模对峙，但装填慢、依赖编制，不能当「魔法」。

乐园阶位映射（宁低勿高，对照凡俗段破坏力）：
- 足轻·地侍·牢人下级武士 ≈ **一阶**（徒手/冷兵器格斗、踹门级）
- 侍大将·一城小名主·稳定铁炮队头 ≈ **二阶**（单挑小队、裂石断木级战阵表现）
- 织田家老·一郡城主·战国大名野战主力 ≈ **三阶**（踏平山寨/端据点、乱军取首级）
- 天下布武后期、举国决战、本能寺级政变阴影 = **顶点情报层**（本世界最高覆盖三阶；更高只作时代背景，契约者勿硬刚）
- 顶点规则：存在·情报优先 / 条件性胜利；严禁「被封印所以战力为零」。

**【地理 · 舞台】**
- **尾张国（爱知）**：主舞台。浓尾平原富庶，五条川绕**岩仓城**，**清州城**为织田信长整合后的政治中心，**小牧山**是平野制高点与后日大战伏笔，**犬山城**扼木曾川南岸（「白帝城」别称见于后史）。
- **岩仓城 / 浮野**：开局战场。岩仓织田（上四郡守护代）对阵胜幡—清州织田；浮野在岩仓北，木瓜纹对木瓜纹的「此织田非彼织田」。
- **木曾川沿岸**：经济命脉。**松仓城**（前野家）、**刈安贺**、**津岛**、**鹈沼**、**伊木山**港口构想；川并众、河运通行费、净土真宗坊市仓库是山内起家的钱袋。
- **稻木庄 / 丹羽郡 / 前野村**：年贡、惣村自治、与犬山织田信清势力重叠的「一块地多个领主」困境。
- **美浓方向**：斋藤义龙/龙兴、稻叶山城、森部—墨俣筑城线；浓尾全面战争的前线。
- **三河方向**：松平元康（德川家康前身）、香积寺、织德同盟上线后的同盟—摩擦带。
- **京都—畿内阴影**：上洛、尾州守护诉求、松永久秀码头、将军足利义辉庭院——连载后期政治台面，三阶切入才碰。

**【世界剧情线】**

**阶段一 · 浮野败战与岩仓笼城（约永禄元—二年）**
现代人**王瑾**在起点给作者吉良上总介打赏盟主后睁眼，已是**山内一丰**（通称伊右卫门），身处尾张**浮野**战场。父**山内盛丰**（但马守）属**岩仓织田氏**家老，主公**织田信贤**端坐本阵；对面敌军亦是木瓜纹——**织田信长**。同场年轻武者**堀尾吉晴**（幼名仁王丸，通称茂助）初阵讨取一番首，友军欢呼。盛丰令一丰冲阵跟紧，声称信长「不过两千人」、岩仓「三千必胜」。一丰瞬间意识到「此织田非彼织田」：己方正与后来统一尾张的怪物作战。铅弹贴身飞过，长枪阵拉锯，骑马武者高呼家名冲入敌阵。浮野一战岩仓侧大败，组织度崩溃；外围城砦被信长与**织田信清**（犬山）迅速扫平，只剩岩仓城一颗钉子。

笼城三月，时间进入永禄二年量级。城中粮食见底，本家一日只分三个饭团；弟**吉助**把饭团塞给兄长，一丰又转给母亲与妹妹。法螺三声示警，敌军提前于预想十日之内兵临城下。盛丰与军奉行**前野宗康**等主战派密议：一旦不支，护送家督信贤出城；盛丰本人选择城中死节——既是武士义务，也是用命换家人出逃资格的交易。「想让你们活下来是真，想为岩仓尽忠也是真。此，人之常情也。」一丰劝父无效，信念钉死为「必须活下去」。

岩仓侧政治前史在笼城对话中补齐：尾张守护斯波—守护代清州织田与岩仓织田—胜幡织田奉行的多层结构；**织田信秀**联姻整合与对外强硬；信长继位后集权、灭弟**织田信胜**、收拾名义主公，再趁岩仓内乱（信安被信贤放逐）下手。信贤曾引美浓**斋藤义龙**为援，援军不至；信贤在御殿空酒壶前诅咒「斋藤义龙不得好死」。城外，信长从京都返回本阵，宣布「斋藤义龙不会来了」——上洛政治动作与围城并行，家臣震惊。

**阶段二 · 诀别与出奔**
夜，橹台值守**五藤净基**（三郎左卫门，叶栗郡黑田乡地侍出身，与盛丰渡船定交后出仕）见盛丰父子提酒而来。盛丰以挚友身份——非家督军令——求五藤随一丰出城护家；五藤初拒「不弃主」，终因幼子与「山内未来」受酒立誓，向一丰行礼称主公。半个时辰后开大手门：同行含山内家眷、堀尾泰晴父子、前野胜长等；斗笠男子身形似信贤，堀尾父子「不该看的别看」。前野胜长路线掌控权在握，山内家已无居城（黑田城早年被盗贼攻破后地位下滑），只能依附。

出城后队伍偏东，堀尾认出**小牧山**——尾张中西部几乎唯一的制高点。一丰按刀质问「非松仓方向」；前野胜长以「避追兵绕道」搪塞，信贤摘斗笠劝「一起去松仓」。并行线：犬山**织田信清**因与信长约定平分岩仓旧领却被独吞而怒，正室犬山殿（信秀之女、信长姐）调解无效；前野家与信长谈不妥后转向信清，「只要确保前野领地，信贤便不落信长之手」。信清下令接应，「领地吾得不到，大家就都别要了」。一丰一行被夹在清州、犬山、前野三方棋盘缝里。

**阶段三 · 落武者狩与流浪尾张**
败兵逃难遭遇**落武者狩**：惣村农民在政权更迭后搜杀落武者夺甲胄刀剑，大名难干预（丰臣「总无事令」与刀狩之前的常态）。**祢兵卫**率十余人持竹枪镰刀，专撕妇孺。五藤净基、堀尾父子与一丰苦战；十岁吉助挥胁差无效。暴民扯住山内夫人衣角，五藤夫人护幼子**吉藏**被镰刀连砍，背上血淋漓后喷血瘫倒。铁炮声突兀响起，一人倒地，祢兵卫以为遇埋伏而撤——救星是熟人（章中「三郎左卫门，是我」式接应）。五藤夫人身亡成为一丰心中血债。

一行沦为**牢人**，寄食亲族与**前野时之**（盛丰弟，过继前野，通称勘八郎系）门下。时之因养父**前野时氏**晚年侧室有孕而失势，知行被转封丹羽郡「收不上贡」之地，又欠债约四十贯；山内租种的自耕地也要被收回。两个妹妹在柳树下拾马粪作肥，母亲**法秀尼**在佛堂念佛。章题「事已至此，先种地吧」「怎一个穷字了得」定下基调：没有安堵状就没有武士身份，只有活下去的力气。

**阶段四 · 稻木庄表演与灭口立威**
一丰自请代时之赴**稻木庄**收年贡。丹羽郡同时落在织田信长赐领与犬山信清势力重叠区，「一块地多个领主」；前野父子选择性无视此地以免得罪信清。一丰先至**前野村**拜隐居的前野宗康（法号南窓庵自观，教族中孩童汉学，考《长恨歌》）。一丰以「杀落武者狩仇人」求背书；宗康一眼看穿年贡意图，拒沾血：「杀人容易，灭口难……绝不能跟吾扯上关系。」一丰遂自行联合**蜂须贺正胜**、堀尾吉晴等动手。

祢兵卫家中堂前供佛像。一丰踹倒对方，令五藤长子**吉兵卫**亲手报仇；胁差连捅后，一丰反手砍忠兵卫，并令堀尾「念法华经后动手」——「放下屠刀，立地成佛」的黑色幽默下灭口满门。草垛下地窖搜出永乐通宝数贯、长枪弓箭、足轻胴丸、酒与粮俵二十余。杀伐与搞钱同步完成。上任稻木庄后，一丰以「表演」分配利益压惣村，凑齐叔父催逼贯高；山内家从咸鱼变为「能办事」的标签。前野时氏正室（山内姑奶奶）已死、岩仓覆灭使时之继承价值归零——一丰看清国人众「多方下注」的生存论。

**阶段五 · 木曾川财源与藤吉郎**
有了第一笔现金与恶名，一丰与蜂须贺正胜把战场转到**木曾川**。津岛行、通行费、净土真宗本龙寺坊市仓库；游说美浓侧**大泽正次**（鹈沼城，原土岐系、上面无人罩）：小川众船只停靠、借用坊市仓库，通行费分一成，并宣称本龙寺将归净土真宗伞下。「不谈信仰只谈钱」。大泽更提出在**伊木山**筑简易架子与港口，与鹈沼南北卡犬山河段——三方起请文互换。蜂须贺欲请近江穴太众，一丰否决：买不起真城，搭架子能停船即可，工匠找松原内匠级本地人。

途中与尚未发迹的**木下藤吉郎（秀吉）**相遇：章题「初见，秀吉凭什么」「藤吉郎？」「木下秀吉是也」。一丰以带资进组、身份自封、「大航海时代」式搞钱思路，把山内从单纯报仇者推成河运利益网节点。**宁宁**再遇线启动（女主），与「清州爱情故事」等章题并行。信仰被工具化：净土真宗门槛低、「阿弥陀佛」即可，书中借此讽刺寺庙与武士一样割韭菜；一丰用其网络扩商，也埋下门徒众与一揆的火药。

**阶段六 · 桶狭间前夜与巅峰赛**
外部高压升级：**今川义元**已控大高城、鸣海城；若非三河内乱与瘟疫波及骏河，攻势更早。信长整合尾张的时间窗关闭中。尾张内：小川众覆灭危机、枪之又左、家臣团+1、利家与丰等人物登场；「时机已到，今日起兵」把内部清算与外战绑在一起。

永禄三年，**桶狭间**开打。章题串起一丰侧战术性格：「想想山内一丰会怎么做」「飞龙骑脸怎么输」「梭哈是一种智慧」「上来就打头阵」「烂命一条就是干」「置之死地而后生」「《孙子兵法》诚不欺我」「雷公助我」「好猛的山内一丰」。义元弯腰欲入轿时前方嘈杂——「义元之死」节点。战后「这份荣光，我不会独享」「带投大哥山内一丰」：军功结算讲究分润与站队。清州夜，信长翻身而起；一丰正式「见过主公」，获侍大将格评价与「全场MVP」式叙事抬升。画风从牢人种地骤变为织田军势中的「鬼山内」苗头。历史桶狭间以少胜多、敦盛「人间五十年」意象被章题直接点名，服务「奇袭改命」而非超自然。

**阶段七 · 羽翼渐丰：婚礼、买马、夜战斋藤**
荣光引来亲戚与家臣扩编：「发达了，亲戚也变多了」「山内家真是人才济济」。木曾川「你的王回来啦」、小川沉银、恩怨难两清；再添大将、净土真宗「众生皆苦」式整肃、午夜惊魂与放火章——河运利益必须用火与刀维护。信长认可加深；婚期将至。

**宁宁**婚礼线：宴宾客、花烛夜（二合一章）、婚前检查式幽默、「宁宁你真美」。历史千代（见性院）卖妆奁买马助夫的典故，在书中以买马、点兵、信长当众赞扬等桥段变奏（章题「山内一丰买马，尾张战端再起」）。尾张战端再起：领内遭袭、夜战斋藤、山内家「百万一心」；信长怒骂无能饭桶，一丰「挥泪斩马谡」肃军，「有债必偿」。悄悄进村、刮地三尺恶名远扬——进步伴随恶名。**前田庆次**等名号擦边（「杀人者！前田庆次是也」）；三河魂、松平压力、「鬼山内一出，压力给到了三河」。仗打到哪里生意做到哪里：军功与商路绑定。

**阶段八 · 美浓攻略与织田家老**
永禄四年坐标明确后，攻略美浓开场。雨季木曾川涨水，大型商船重现；**森部大捷、墨俣筑城**成为后勤与奇袭双名场面。浓尾全面战争，六武众出击；**斋藤龙兴**「是个人物」，信长亦有「赛季首败」。一丰「来自山内一丰的降维打击」：经营账本与战场指挥同开。偷渡矢作川直取冈崎、三方原体验服、与松平宿命对决——三河方向惨败「松平之殇」后，反而「顺利迈入织田家重臣行列」；自立一念与能进能退的法器哲学并写。

香积寺线：樱井松平、深沟松平联合净土真宗本证寺门徒奇袭饭盛山香积寺，不敌；松平元康事后称赞真弓山城等——为织德同盟铺垫。章题「敌在香积寺，织德同盟上线」「他才是众望所归」「白月光山内一丰」。**织田家老，一步登天**：搬家做城里人，群贤贺但马守；清州家老日常、信长心里苦、清廉如水但马守人设与「搞钱，疯狂搞钱」并行。秀吉「给山内一丰跪下了」式镜像进步。永禄六年：新财源一本万利、小牧山城计划启动、尘埃落定准备上洛。

**阶段九 · 上洛阴影（连载前沿，不编终局）**

**【剧情补全·阶段细目（据公开章题与试读，不编终局）】**

开局身份锚定：现代读者王瑾穿越为山内一丰，通称伊右卫门，父山内盛丰（但马守）为岩仓织田家老。浮野战场上，堀尾吉晴初阵一番首；对面军势同为木瓜纹，实为织田信长。岩仓主公织田信贤误判兵力优势，浮野惨败后组织度崩溃，外围城砦被信长与犬山织田信清扫平，岩仓城陷入长期笼城。城中饭团配给、法螺示警、三月围困，把武士体面压成「活下去」三字。

笼城政治账：尾张多层主从——斯波守护、清州与岩仓守护代、胜幡织田奉行——被信长集权策略撕开。信秀联姻路线与信长强硬整合路线冲突；信胜、名义主公等障碍清除后，岩仓内乱成为天赐切口。信贤指望斋藤义龙驰援落空；信长高调上洛后回营断言义龙不会来，围城与政治同步推进。

出奔交易：盛丰与前野宗康等主战派约定护家督撤离；盛丰本人留城尽义，以死换家眷生路。五藤净基（三郎左卫门）由挚友之托而非军令，改随一丰。大手门夜开，队伍含山内家眷、堀尾父子、前野胜长；斗笠人影疑为信贤。路线被带向小牧山方向，一丰察觉非松仓。犬山信清因分赃不满，接洽前野，欲把信贤变成筹码——出奔从逃生变成三方棋盘。

落武者狩与牢人期：惣村暴民在政权更迭后搜捕败兵夺甲，祢兵卫团伙专攻妇孺侧翼。五藤夫人护子重伤不治；铁炮硝烟吓退暴民。山内家寄食前野时之门下：时之因养父前野时氏侧室有孕失势，知行转封难收，欠债数十贯，连租地都要收回。妹妹拾马粪、母亲法秀尼念佛，章题「先种地吧」「怎一个穷字了得」定下调性——没有安堵状就没有武士。

稻木庄翻身：一丰自请代收丹羽郡稻木庄年贡。此地同时落在信长赐领与犬山势力重叠区，前野选择性无视。一丰拜前野宗康（自观公），以报仇为名求背书，被老狐狸拒绝沾血。一丰联合蜂须贺正胜、堀尾吉晴等自行处理祢兵卫一系，搜出永乐通宝、武具与粮俵，完成叔父催逼贯高。表演压惣村：利益分配比刀锋更持久。山内标签从咸鱼变为能办事。

木曾川立棍：津岛行、通行费、净土真宗坊市仓库；游说鹈沼大泽正次，伊木山架子港口卡犬山河段，起请文互换。蜂须贺欲请穴太众，一丰否决——买不起真城，能停船即可。与木下藤吉郎（秀吉）相遇：「初见，秀吉凭什么」「木下秀吉是也」。宁宁再遇线启动。信仰被工具化：门槛越低的宗门，商路与动员力越强；一丰不谈信仰只谈钱。

桶狭间改命：今川义元压境，大高鸣海在握。永禄三年会战，章题串起一丰战术性格——头阵、置之死地、雷公助我、好猛的山内一丰。义元之死节点后，荣光分润与带投站队并重。清州见主公，侍大将格抬升，鬼山内绰号发芽。敦盛「人间五十年」是文化意象，不是超凡法术。

婚约与肃军：亲戚涌来、家臣扩编；小川沉银与河运冲突维护利益网。宁宁婚礼花烛夜与买马线并行（历史贤内助买马典故的小说变奏）。夜战斋藤、处置失职者、有债必偿；信长骂无能也认军功。前田庆次等名号擦边；三河压力上升。仗打到哪里，生意就做到哪里。

美浓与家老：森部大捷、墨俣筑城，浓尾全面战争；斋藤龙兴难缠，信长亦有败绩。一丰经营与战场双开，迈入重臣。香积寺冲突铺垫织德同盟；白月光山内一丰声望溢出，织田家老一步登天，迁居清州。搞钱与清廉人设同框，秀吉镜像竞争。小牧山城计划与上洛准备启动。

连载前沿：剑豪流落、铁炮名人、见松永久秀拜码头。幕府足利义辉庭院对尾州守护诉求难色；章题「尾张有孤忠」「有山内一丰这样的家臣，织田信长真是走了大运」。停在上洛前政治台面。后史土佐二十万石等仅百科对照，禁止写成已发生结局。

岩仓城（平城、五条川）、浮野古战场、清州城（信长政治中心）、小牧山（制高点与后日大战伏笔）、犬山城（木曾川要冲）、松仓城与前野村、稻木庄与丹羽郡惣村、津岛凑、鹈沼与伊木山港口构想、美浓稻叶山—墨俣线、三河香积寺—冈崎方向、京都幕府庭院（后期阴影）。写场景先定：谁的安堵状覆盖此地、年贡交给几家、最近的铁炮编制在谁手里。

**【力量体系补充·军役与身份】**
石高与贯高决定能拉多少足轻；安堵状决定你是武士还是牢人；一番首与军忠状决定晋升；铁炮改变小战场但不取消粮道。代官与惣村谈判是文斗，落武者狩与野战是武斗，河运通行费是钱斗。三者同时失败才会真正出局。乐园定阶只看可观察破坏力与编制，不看官称自我吹嘘。

足轻与暴民：竹枪镰刀、村社联动，围杀落单武士足够致命。地侍：编制外武士，可被招募入家臣团。下级武士与牢人：有刀枪无领地，一丰开局即此层。侍大将：统军取首、铁炮与长枪协同，桶狭间后一丰进入此层。城主家老：签发安堵、左右主家，清州城里人阶段。战国大名：举国军势与同盟，信长义元龙兴元康。幕府将军：名义权威，上洛期才成主矛盾。

**【人物补述·关系网】**
山内家：盛丰死节模板；法秀尼与弟妹是软肋；五藤一门是最后家臣。堀尾家：与山内互保出奔。前野家：救命与卖友同一套多方下注。织田两系统：岩仓旧主与清州新主，一丰必须完成站队跃迁。秀吉与宁宁：进步者镜像与内助锚点。信清：同盟裂痕的活教材。义元、龙兴、元康：外压三方向。义辉与松永：上洛期名义与码头。

山内一丰对契约者：能办事给粮给刀，背后捅刀则清算到底。五藤净基：护主优先，可结盟护送。堀尾吉晴：同袍情重，战场可靠。前野宗康：不沾血收渔利。蜂须贺正胜：利益绑定则生死与共。织田信长：只认结果。木下秀吉：同类相惜亦相防。宁宁：家计与人心的稳定器。

**【势力补充·交易逻辑】**
国人众生存论：安堵状可收多份，直到被迫选边。寺社：仓库、坊市、门徒动员。商人：津岛与河运，谁控制泊位谁收钱。幕府：官位与守护名义，野战打不赢文书战。契约者介入时，优先改账本与盟约，再谈斩将夺旗。

岩仓织田余烬：忠义与出奔并存，覆灭后人才外流。清州织田：整合与清洗并行，能臣超车。犬山织田：分赃政治，卡河要挟。前野川并众：河运即政权。小川众与蜂须贺系：通行费武装。净土真宗：信仰外壳下的钱与人。今川：桶狭间前外部高压。斋藤美浓：不救岩仓后成攻略对象。松平三河：织德同盟变量。足利幕府：尾州守护文书战。

## 阶位切入点

> 阶位↔原作阶层：一阶≈足轻/地侍/牢人下级武士；二阶≈侍大将/河运武装头目/一城小名主；三阶≈织田家老/一郡城主/战国大名野战主力。世界顶点（织田信长整合尾张后的政治—军事机器、今川大军压境瞬间、幕府官位博弈）为跨阶存在，低阶切入一律「规避顶点·情报优先 / 条件性胜利」。

**一阶（牢人·稻木庄 · 活下去与第一笔年贡）**

切入身份/时点：契约者以**岩仓败亡后的牢人扈从 / 稻木庄代官手下属吏 / 落武者狩村社边缘人**切入，锚定**山内一丰**出奔后至上任稻木庄、灭祢兵卫前后。

初始事件：小牧山附近道路上，祢兵卫率村社暴民围住拖家带口的武士队；铁炮硝烟炸响，五藤夫人倒在血泊——你必须在「趁乱抢甲」「救人换人情」「向犬山或前野告密」中选一条活路，并在三日内见到要收 40 贯的**前野时之**账房。

开场白建议：「浓尾平原的风往领口里灌。岩仓城方向的烟还没散尽，你脚边的竹枪上沾着别人的血——落武者狩的笑声从田埂后传来。有人喊山内伊右卫门的名字，像喊一块还能卖钱的甲胄。」

关键NPC立场：**山内一丰**——刚失去父亲与部分家眷安全感，对能办事的外人给粮给刀，对背后捅刀者灭口；**五藤净基**——护主优先，可结盟护送；**堀尾吉晴**——佛心外表、出手极快，重同袍；**前野时之**——欠债焦虑，年贡到手才认人；**祢兵卫**——暴民头目，见财起意，必须清除或利用后弃子；**前野宗康**——老狐狸，不沾血却收渔利。

主线钩子/支线：协助或阻挠稻木庄收贡；落武者狩复仇链；向松仓/犬山/清州三边卖情报。支线：拾马粪种地换夜宿、五条川渡船费纠纷、替法秀尼（一丰母）护弟妹。

危险度/规避：中高——暴民群殴与地侍仇杀可致命；规避织田信长本队、犬山城正规军围捕、净土真宗一揆规模冲突。

任务方向/奖励：活过第一个月、换到安堵式容身或一笔贯高；奖励取向=胁差/足轻胴丸、少量永乐通宝、河运水手名分、山内家最外围奉公人身份。

**二阶（侍大将·木曾川与桶狭间 · 军功与财源双开）**

切入身份/时点：契约者以**小川众船头 / 蜂须贺系武者 / 织田一番队足轻组头 / 津岛商人武装扈从**切入，锚定一丰做大木曾川至**桶狭间**前后（章题「桶狭间巅峰赛」「义元之死」「山内一丰，见过主公」）。

初始事件：津岛凑涨水，大型商船归港；你押运的通行费账册被犬山方向武士扣押，**蜂须贺正胜**要你今夜在伊木山架子港口点火为号——同时清州急使传令：今川大军已动，所有能拿枪的人三日后到指定军列。钱与命同一晚到期。

开场白建议：「木曾川的浊浪拍着船舷，账本上的贯高还没晒干。上游有人喊封锁犬山，下游有人喊义元的马印已经过境。你把铁炮的火绳咬在嘴里，第一次意识到：在尾张，『进步』两个字闻起来像硝烟。」

关键NPC立场：**山内一丰**——要人要船要首级，赏罚分明；**蜂须贺正胜**——合伙人，利益绑定则生死与共；**木下秀吉**——爱用能跑腿搞钱的人，也防备同类型竞争者；**宁宁**——透过家计与婚约影响一丰判断；**织田信长**——只认结果，桶狭间后按军功重新洗牌；**今川义元**——敌方顶点，接触=送死；**堀尾吉晴**——战场可靠队友。

主线钩子/支线：桶狭间参战位置选择（先锋/侧翼/清州留守）；木曾川封锁犬山的港口之战；「带资进组」织田。支线：再遇宁宁的礼物链、小川沉银打捞权、铁炮名人手艺偷师。

危险度/规避：高——野战与铁炮；必须规避义元本阵硬刚、信长打扫战场时的清算、美浓斋藤突然介入。

任务方向/奖励：拿到侍大将级认可或一笔稳定知行；奖励取向=安堵状碎片、铁炮编制名额、名马一匹、山内家正式家臣席。

**三阶（家老·美浓与上洛台面 · 清州城里人）**

切入身份/时点：契约者以**织田家中坚家臣 / 美浓攻略方面奉行副手 / 清州城吏 / 三河交涉使**切入，锚定**森部—墨俣、织德同盟、家老登天、小牧山计划与上洛准备**（连载前沿）。

初始事件：清州城夜宴刚散，贺「但马守」的酒气未消；急报称香积寺方向松平支族与门徒众交火，同时京都细川藤孝一线传来幕府对「尾州守护」诉求的难色。**织田信长**要你在天亮前给出：是先填美浓缺口，还是先派人去松永码头「拜码头」。

开场白建议：「清州的灯火把木瓜纹照得发亮。有人笑称山内一丰是主公的白月光，有人在廊下盘算上洛后的官位。你展开舆图：一边是墨俣的木桩，一边是将军庭院里的沉默——战国的十字路，从来不是写在路牌上，而是写在谁先签字的安堵状上。」

关键NPC立场：**织田信长**——多疑且依赖能臣，家老级失误=饭桶骂名或更糟；**山内一丰**——已是城里人，搞钱与孤忠人设并重，可结盟亦可防其抢功；**松平元康**——同盟者，钓鱼式试探；**斋藤龙兴**——美浓顽敌；**松永久秀**——畿内码头，笑面虎；**足利义辉**——名义顶点，不可武力解决；**织田市**——政治联姻筹码，勿轻辱。

主线钩子/支线：墨俣筑城后勤；织德同盟条款；上洛队列与尾州守护文书。支线：剑豪流落尾张的收留与监视、小牧山筑城民伏、山内「伊贺越」式仇杀是否批准。

危险度/规避：极高——贴近信长与幕府双重顶点；超阶阴影按「存在·情报优先/条件性胜利」处理，严禁写信长被随意斩杀或「封印归零」。规避单独深入美浓腹地、在将军面前妄议废立。

任务方向/奖励：保住方面职责或换取一城代官；奖励取向=家老级知行、铁炮队编制、上洛随行资格、与山内/秀吉的书面起请文（人脉）。

**一阶补充钩子（牢人账本）**
可观察细节：饭团个数、贯高数字、安堵状有无、铁炮硝烟方向、谁先喊家名。事件链建议：护送法秀尼弟妹过五条川渡口→稻木庄惣村谈判破裂→祢兵卫余党夜袭租地→向蜂须贺借三条枪换三成货。奖励勿发城主印，只发一夜粮与一张手书推荐。

**二阶补充钩子（河运与桶狭间）**
可观察细节：船头旗印、通行费分成比例、玉药库存、军忠状署名。事件链建议：伊木山点火为号被犬山巡逻发现→津岛商人要你在信长与信清间选保人→桶狭间前夜一丰要你带二十人断粮道。奖励可至铁炮编制名额与半份知行许诺，勿发尾州守护。

**三阶补充钩子（清州与上洛）**
可观察细节：清州夜宴座次、起请文朱印、幕府回书语气、墨俣木桩进度。事件链建议：香积寺急报与尾州守护文书同一夜到达→松永久秀码头见面礼是情报不是刀→信长问你「山内但马守的账，你核过没有」。奖励取向：方面副职、上洛随行、与山内或秀吉的书面盟约。顶点信长与义辉：情报优先，条件性胜利，禁止归零式削战力。
