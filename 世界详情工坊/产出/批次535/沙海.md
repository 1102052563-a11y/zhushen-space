# 沙海
<!--meta lib=主库 tiers=一、二、三、四-->

## 剧情

**【作品来源】**
《沙海》（全称常写作《盗墓笔记·少年篇·沙海》）为南派三叔著，接续《盗墓笔记》正传后的“少年篇”，以**吴邪**十年布局灭**汪家**为主轴，视角落在高中生**黎簇**被卷入局中。单行本分卷常见为《荒沙诡影》《沙蟒蛇巢》《鬼城幻影》《怒晴湘西》等。类型：盗墓冒险、沙漠秘境、组织博弈、成长悬疑。文风较正传更“布局—棋子—代价”，少年线与幕后吴邪线双线交织。**搜笔趣阁站内检索（keyword=沙海）未收录本书页**（2026-07 机查无书目），正文以出版卷目、公开剧情解读与百科式人物关系为准；改编网剧《沙海》（2018）仅作对照，**本档案以小说少年篇为准**。

**【世界定位】**
青铜门之约前后的当代中国地下世界：吴邪已从“天真小三爷”长成布局者，以摄影师“**关根**”等身份操盘，把**黎簇**、**杨好**、**苏万**等少年推进巴丹吉林与**古潼京**，用九门禁地与汪家千年局对撞。一句话：**少年被扔进沙海，大人在沙海上方下棋。**

**【世界观 · 力量体系】**
本世界无修仙境界，战力来自**倒斗经验、组织训练、人体异能边缘、秘境机制杀伤**。

1. **市井与学生层**：普通人、高考落榜生、小混混——徒手与冷兵器级，沙尘与热武器均致命。
2. **盘口与训练层**：汪家“训练营”、倒斗班底、枪械与野外生存；能单挑小队、扛住沙漠行军与机关外围。
3. **顶尖倒斗者**：黑瞎子级夜战与近身、解雨臣级谋略与缩骨易容、吴邪级信息战与蛇信感应——区域强者，仍惧秘境规则。
4. **秘境与异物**：古潼京白沙、沙蟒、黑毛蛇、激素信息、无法成像的岩石与“影子规则”；威胁按**机制**而非纯数值。
5. **组织力**：汪家（汪藏海后裔）、吴邪布局网、九门残余、边境物流与假身份。

**乐园阶位映射（宁低勿高，对照阶位战力图鉴破坏力）**：
- 普通人/学生/未训练棋子≈**一阶**（徒手、冷兵器、坠楼可致命）
- 有经验倒斗者/汪家基层训练员/能活过沙漠外围≈**二阶**（碎砖裂石、小队级）
- 黑瞎子、解雨臣、成熟期吴邪、汪家高层执行者≈**三阶**（裂石断金、端据点、乱军取首）
- 古潼京核心异象、顶级蛇灾、张起灵若出场、区域级“局”崩盘≈**四阶**（掀平建筑群/据点级，非城毁星爆）
**顶点≈四阶**；“汪家千年局/终极相关”按**情报优先·条件性胜利**处理，禁止写成顶点无威胁。

**【地理 · 舞台】**
- **城市侧**：黎簇所在中学与出租屋、苏万家、快递与“寄尸”线路、吴邪外围据点。
- **巴丹吉林沙漠**：摄影采风假面下的实战课堂；白沙带、岩石无法成像区、干尸与刀痕。
- **古潼京**：九门禁地级目标，吴邪引汪家视线的核心饵；白沙漠中的鬼城结构。
- **汪家控制区**：训练营、渗透用身份链、 snatch 与监视。
- **湘西/怒晴相关线**（后卷）：与怒晴湘西、边境势力交接，把少年从“沙”拖进更复杂的人局。
- **前史地理回响**：西王母蛇沼、蛇激素信息、与正传西沙/蛇线的伏笔互文——沙海不重讲正传，但**读蛇**能力是同一套规则。

**【世界剧情线】**

① **前史与动机（接盗墓笔记）**  
青铜门前，张起灵入门守约；吴邪在十年窗口内摸清**汪家**对九门与“终极”的觊觎。他意识到硬刚会死人，于是把战场选在**信息、棋子与禁地**。古潼京被定为能同时勾动汪家与九门旧账的“饵”。吴邪以旅行摄影作家等身份试探沙漠与人脉，结识曾深入巴丹吉林的女作家**蓝庭**线相关线索：友人**叨叨**登上古潼京岩石后出现**无法在照片中成像**的诡异，离开后自杀——照片与故事成为吴邪确认“局可用”的证据之一。他先后筛选约十七名“陌生人”作棋子，失败则以自伤计数（手上刀痕）记录代价，最终盯上**黎簇**。

② **黎簇入局（荒沙诡影）**  
黎簇：普通高三、成绩差、家暴父亲、高考失利。同学/相关人物**沈琼**递来奇怪盒子，打开即不可回头。归途昏暗巷中被打晕，**后背刻上复杂诡异图案**。吴邪以关根身份接近，解释图案与“计划”的一部分，把黎簇拖进巴丹吉林“采风”。初始事件链：绑架感、失踪、黄沙、白骨、刀痕——少年第一次意识到自己不是主角而是**工具**。同期，吴邪向**苏万**等寄出含干尸、探险设备、武器与怪蛇的包裹，用恐惧与好奇心把外围少年网织紧。

③ **沙漠课堂与第一次古潼京（沙蟒蛇巢）**  
在**海子**等向导/协助下，黎簇一行触及白色沙漠与古潼京外围。遭遇沙蟒、黑毛蛇、大白蛇级威胁与生存极限训练。黎簇逐渐觉醒与吴邪同型的**蛇激素信息读取**——能“读懂”蛇带来的信息，从而接触西王母/蛇沼系秘密的边角。吴邪不亲自当英雄，而是用任务、断粮、假死信息与局部真相控制节奏。杨好、苏万等成为黎簇情感锚与拖油瓶式战力，梁湾等角色在少年线提供情绪与立场拉扯。阶段结局：黎簇暂时离开沙漠回家，但快递、疤痕与记忆已把他焊死在局上。

④ **再探与汪家渗透（鬼城幻影）**  
黎簇据快递与沙漠线索再度寻找古潼京。鬼城幻影级机关与“成像规则”迫使团队用摄影、标记与人祭式试错前进。吴邪布局进入关键阶段：**让黎簇取得汪家信任并潜入**。汪家训练营用纪律、暴力与“千年不可抗拒之力”的叙事洗脑棋子。黎簇在训练中拼出吴邪真实身份与汪家对九门的野心，同时发现自己已被两边当枪使。名场面逻辑：少年第一次有能力反问“谁在下棋”，却仍缺终局情报。

⑤ **湘西延伸与局的收束向（怒晴湘西等）**  
战场从沙海延到湘西/边境人脉，怒晴系线索把“灭汪”从沙漠奇观拉回人情与旧派规矩。解雨臣、黑瞎子等九门侧强援以交易或暗助形式出现，不抢少年主视角但提供三阶天花板。阶段目标：把汪家力量导入古潼京陷阱并尽量清肃；代价是棋子身心损伤与关系撕裂。

⑥ **结局方向**  
小说线收束为：黎簇完成关键渗透与触发后**失踪/消失**（叙述上留给后续《重启》等世界的钩子），吴邪的灭汪布局取得阶段性成果但**远非无代价完胜**；古潼京与蛇信息的秘密只揭开一层。铁三角（吴邪—张起灵—王胖子）在沙海多以影子或短促现身存在，重点是**吴邪独立成棋手**。改编剧集数与人名有增删，写卡时以小说人物真名为准。

⑦ **因果链摘要（供 AI 时点判断）**  
刀痕十七次失败 → 选中黎簇 → 盒子与背纹 → 关根入沙 → 读蛇能力 → 古潼京饵 → 汪家训练营 → 湘西延伸 → 黎簇消失与布局半成。任意时点问“世界在哪”：看黎簇是否已入沙、是否已进汪家、吴邪是否已亮出布局者身份。

**【主要人物】**

- **黎簇**（少年主角/棋子）｜性格：叛逆、敏感、嘴硬心软，被家暴与落榜压出的尖锐｜装备·能力：背纹相关异能逐步苏醒；后获读蛇激素信息的能力；野外生存与枪械由局中训练强行灌入｜人物弧光：普通高中生→沙漠幸存者→汪家渗透者→局终失踪的半成品棋手｜立场关系：恨又依赖吴邪；与杨好、苏万是少年同盟；对汪家由恐惧转利用。

- **吴邪 / 关根**（布局者）｜性格：冷静、残忍化的温柔、算无声｜装备·能力：建筑学与倒斗经验、读蛇、情报网、多身份｜人物弧光：正传天真→沙海邪帝式棋手｜立场关系：对黎簇是导师亦是加害者；与解雨臣、黑瞎子交易；与汪家死敌。

- **杨好**（少年同伴）｜性格：义气、 impetuous｜能力：少年级动手与跑腿｜弧光：被卷进局的损友｜立场：黎簇铁杆。

- **苏万**（少年同伴）｜性格：相对理性、家境线常被利用｜能力：后勤与信息节点（收快递即入局）｜弧光：从旁观到共谋｜立场：可被吴邪远程“投喂”恐惧。

- **梁湾**（少年线关键女性角色）｜性格：坚韧、有自己的目标｜能力：不详以原作为准的野外/人际｜弧光：情感与立场拉扯｜立场：可合作可冲突。

- **沈琼**（入局触发）｜性格：不详/功能性｜能力：递盒者｜弧光：打开黎簇命运的钥匙｜立场：间接把黎簇推给局。

- **解雨臣（小花）**｜性格：算无声、同辈顶级｜能力：缩骨易容、解家资源｜弧光：九门侧暗助吴邪灭汪｜立场：盟友，非无条件。

- **黑瞎子**｜性格：戏谑狠厉｜能力：近瓶身手、夜视｜弧光：可雇佣的暴力天花板｜立场：交易型强援。

- **王胖子**｜性格：油滑重义｜能力：摸金经验｜弧光：沙海中多为影子支援｜立场：吴邪旧部。

- **张起灵**｜性格：寡言｜能力：发丘指、顶身手｜弧光：青铜门内/约定相关，沙海少直接主场｜立场：吴邪守约对象，顶点级存在。

- **汪家人（具名以卷内为准，如汪家训练官/执行者）**｜性格：纪律、冷酷、信仰“千年局”｜能力：组织训练、渗透、暴力｜弧光：被饵引入古潼京｜立场：主敌。

- **海子**｜性格：务实向导型｜能力：沙漠生存｜弧光：带队见白沙与古潼京｜立场：中立偏合作。

- **蓝庭 / 叨叨（前史线索人物）**｜性格：采风者/受害者｜能力：无｜弧光：成像诅咒与自杀触发吴邪确认饵｜立场：故事引子。

- **黎簇之父**｜性格：酗酒暴力｜能力：无｜弧光：把黎簇推出家门式命运｜立场：日常加害源。

**【势力图谱】**
- **吴邪布局网**：关根身份、快递线、棋子筛选、古潼京饵；宗旨=灭汪保约与清肃九门隐患。
- **汪家**：汪藏海后裔组织，觊觎九门权势与终极秘密；训练营与监视网。
- **九门残余**：解/霍/吴等家资源，立场不一，可交易。
- **市井少年圈**：黎簇—杨好—苏万，被两方收割的战场基层。
- **沙漠与秘境本身**：不站队，用规则杀人。

**【贵重物品】**
- **沈琼之盒**：打开即入局的触发物。
- **背纹/图案**：黎簇被刻印的能力锁与标记。
- **蛇与激素样本**：读信息的介质，连西王母蛇沼系秘密。
- **古潼京相关影像/无法成像的岩石记录**：证明禁地规则的证据。
- **吴邪刀痕与十七次失败档案**：布局者的道德账单。
- **汪家训练营信物/证件**：渗透用身份。
- **探险设备与寄尸快递**：吴邪远程操盘工具。

**【隐藏剧情 · 伏笔】**
- 黎簇与吴邪同型读蛇能力从何而来、是否被定向培养。
- 古潼京与西王母/蛇沼、正传终极的同构关系。
- 十七名失败者下落与汪家是否反向利用过棋子。
- 黎簇消失后与《重启·极海听雷》等后续的衔接（本世界点到为止）。
- 吴邪是否把“成为棋手”的创伤遗传给下一代棋子——主题伏笔。

**【大事记时间线】**
青铜门之约生效 → 吴邪确认汪家威胁 → 蓝庭/叨叨采风案 → 十七次选棋失败 → 黎簇开盒与背纹 → 关根入沙 → 白沙古潼京初探 → 读蛇觉醒 → 再探与汪家渗透 → 湘西延伸 → 局阶段性收束、黎簇消失。

**【叙事基调 · 雷区】**
基调：黄沙冷硬、少年脏话与黑色幽默、大人布局的窒息感。忌：把黎簇写成无敌穿越者；忌把吴邪洗白成无害导师；忌古潼京无机制乱开怪；忌张起灵全程外挂抢戏；忌用网剧人设覆盖小说真名关系。最早锚点：黎簇开盒之夜，或关根第一次把少年带进巴丹吉林。

**【分卷事件链·细述】**
荒沙诡影卷确立“棋子被选中”的伦理暴力：吴邪不解释全盘，只给刚好够活的信息。沙蟒蛇巢卷把威胁动物化、机制化，蛇不是经验包而是语言系统。鬼城幻影卷把摄影/成像变成生存规则，呼应叨叨之死。怒晴湘西卷把沙漠奇观回收成人间派系与旧账，防止故事停在“打怪升级”。全程重复主题：**你以为你在探险，其实你在被人读档。**

**【少年三人组动力学】**
黎簇负责撞墙与质疑；杨好负责义气式莽；苏万负责把“家”和“正常世界”拖进局。三人同时在场时，吴邪的快递与任务才有情感杀伤力——拆散三人组等于拆沙海的读者锚。

**【吴邪布局方法论】**
信息分层、身份隔离、失败可抛弃、成功可弃子。对契约者而言，学会识别“关根式任务书”比学会开枪更重要：目标写得很美，报酬写得很虚，撤退路线从不写全。

**【汪家叙事话术】**
用“千年”“不可抗拒”“归属”包装人身控制。黎簇能反杀的关键不是武力，而是识破话术后仍选择利用敌营——这是三阶切入才站得住的心理战。

**【古潼京机制要点】**
白沙导航失效、成像异常、蛇群激素场、鬼城结构重复与幻影。写战斗优先写**迷路、失温、中毒、队友失踪**，再写开枪。

**【与正传接口】**
沙海不重演鲁王宫到青铜门，但所有“蛇、西王母、汪藏海、九门”名词都指向正传档案。低阶契约者只给名词闪回，高阶才允许拼图。

**【结局后世界状态】**
汪家受创但未从设定上蒸发；黎簇缺席造成少年线空洞；吴邪更接近“邪帝”公众形象；沙漠仍在，饵仍可能被后来者复用——适合乐园反复开局。

**【荒沙诡影·卷内因果补全】**
卷首把“正常生活”写到令人窒息：父亲的酒、母亲的缺席、分数条与同学的嘲笑，都是为了让读者接受——黎簇为什么会打开盒子。沈琼递盒不是浪漫邂逅，是命运的冷交接。背纹被刻上的那个巷子，是城市版的“下斗口”：没有洛阳铲，只有蒙眼布与消毒水气味。吴邪第一次出现时不谈终极，不谈青铜门，只谈“你背上的东西会要命，我可以教你怎么少死一点”。这种半真半假的救援，是沙海全篇的说话方式。寄给苏万的包裹把恐怖从沙漠寄回客厅：干尸手指、蛇蜕、说明书一样的威胁——吴邪在用“污染日常”的方式，把少年的社会关系改造成情报节点。蓝庭讲述叨叨之死时，重点不是鬼故事吓人，而是**照片里的人不在了**：成像规则第一次以都市传说形态进入正文，为后文鬼城幻影做预告。十七道刀痕若被契约者看见，应读作“失败档案”而非酷刑美学：每一个失败者都可能仍活在汪家或沙漠边缘，成为二阶支线的失踪人口。

**【沙蟒蛇巢·生存课细目】**
入沙后的第一课是水、方向与闭嘴。海子类向导角色负责把“摄影队”翻译成“还能活的队形”。沙蟒的威胁在于埋沙突袭与缠绞，黑毛蛇/大白蛇则把恐惧视觉化。黎簇的读蛇不是突然开挂，而是在多次接触激素与幻听后，把“恶心的味道”识别成“句子”。吴邪在此卷最常做的事是**不救**：让少年自己爬出沙坑，才配继续当棋子。杨好的莽撞制造几次差点团灭的喜剧与悲剧边缘；苏万的犹豫则提供“要不要报警回城”的凡人选项——报警在这个世界几乎等于把线索送给汪家。梁湾线若展开，多与药品、立场和“谁更像正常人”有关，用来对比黎簇的异化速度。白沙带的导航失效要写成物理体验：指南针乱跳、脚印被风抹平、同一块岩石出现两次。取回第一份“无法成像”样本时，相机报错或底片空洞，比任何怪物吼叫更符合南派气质。

**【鬼城幻影·结构与人心】**
鬼城不是开放世界地图，是重复走廊与错误出口。幻影机制让队员看见已死之人或未发生的救援，吴邪偶尔利用幻影发布假指令，测试谁会违抗。汪家训练营段落把“组织”写到皮肤上：队列、口号、告密、夜间拉练、对逃跑者的公开处理。黎簇在营中的成长是双刃的——他获得枪感、纪律与谎言技术，也失去对“家”的退路。吴邪与汪家在此卷进入信息对射：你以为你在渗透，对方可能在养刀。解雨臣的远程协助常以资源、假身份、一次恰到好处的医疗出现，从不长期同行，保持九门侧的距离感。黑瞎子若下场，应是短促、昂贵、结束得干净的暴力，用来标定三阶上限。本卷结束时，读者应能回答：黎簇还信不信关根？若不信，他为什么还走？

**【怒晴湘西·人局压过沙局】**
战场东移后，沙子变成潮湿的山与旧派规矩。怒晴相关线把“灭汪”从奇观任务拉回人情债：谁欠九门、谁卖过盘口、谁在边境运过不该运的东西。少年组在此卷更容易内讧，因为威胁从“蛇”变成“人言”。吴邪的布局进入收网：需要汪家主力相信古潼京值得倾巢，又需要己方有人在核心按下落点。黎簇的失踪不是随便写的开放结局，而是布局完成时的必要空白——棋子知道太多，活着离开会成为新的泄漏点，死了又浪费培养；于是“消失”成为叙事上的第三选项。契约者在四阶若想改写失踪，必须付出同等重量的替代代价（另一人顶上、或局失败），否则违背世界因果。

**【读蛇体系·可观察细则】**
读蛇者能从蛇的行为、蜕皮、腺体气味中获得方位、危险等级与“谁在撒谎”的模糊提示，不是全知雷达。过度使用导致幻听、脱水与把人类语言听成蛇鸣。吴邪与黎簇同能不同德：前者用信息建网，后者用信息求活。无读蛇者进入蛇场，只能依赖火、光、噪声与向导经验。

**【古潼京作为饵的工程学】**
饵要同时满足：汪家垂涎、九门忌惮、公众不可见、可被摄影/传说放大。吴邪用采风、论文、都市怪谈、失败棋子的尸体传闻多层投放信号。真正的引爆不靠一句咒语，靠**人在错误时间走进正确结构**。因此四阶任务的核心是调度表与信任链，不是打龙。

**【汪藏海遗产在沙海中的形状】**
正传中的汪藏海是历史布局者，沙海中的汪家是执行公司：训练、渗透、档案、清理。他们信仰的“不可抗拒之力”既可能是终极相关，也可能是组织自我神话。揭开真面目时，应保留一半迷雾，让重启线仍有空间。

**【铁三角的影子用法】**
张起灵少出场即是设定：守门与约定仍在生效。王胖子提供后勤笑话与突然靠谱。吴邪独自成篇，是为了证明“小三爷”已死，活着的是愿意在自己手上刻刀痕的人。写沙海时把正传名场面当闪回即可，勿重开鲁王宫。

**【契约者道德量表】**
救黎簇出局=可能破坏灭汪；帮吴邪弃子=可能赢得阶段胜利但失去人性锚；帮汪家=短期资源、长期被清理。沙海没有完美选项，只有账单。

**【环境杀伤清单】**
失温、中暑、沙盲、脱水、蛇毒、幻影致走失、训练营私刑、城市尾随绑架、快递爆炸式惊吓。热武器有，但沙漠里子弹比怪更珍贵。

**【语言与称呼】**
吴邪对外关根、对内小三爷余波、对棋子不报全名。黎簇骂人脏但不无因。汪家称谓纪律化。写对话忌网文无敌宣言，宜短句、命令、反问。

**【时间线锚点速查表】**
T0 开盒；T1 背纹；T2 首次入沙；T3 读蛇初成；T4 离沙回家仍被包裹追击；T5 再探古潼京；T6 入汪家营；T7 湘西交接；T8 饵爆；T9 黎簇消失。选世界卡时点=选 T。

**【与乐园术语对齐的一句】**
本世界最高有效破坏力停在**据点/区域秘境**，故封顶四阶；组织与信息战可让低阶角色杀死高阶莽夫——沙海奖励谨慎，不奖励喊战力。

**【卷末主题句（非灌水，供基调）】**
沙海不是少年热血冒险，是**成年人把战争外包给孩子**的记录；黄沙底下埋的不是财宝，是选择权。

**【人物关系网·可查询】**
黎簇←监护式操控→吴邪；黎簇↔杨好（损友同盟）；黎簇↔苏万（恐惧共同体）；黎簇↔梁湾（立场拉扯）；吴邪↔解雨臣（同辈交易）；吴邪↔黑瞎子（雇佣暴力）；吴邪↔王胖子（旧日信任）；吴邪→张起灵（守约方向的单箭头牵挂）；汪家⊃训练官⊃黎簇（假归属）；海子⊥队伍（有偿向导）；蓝庭/叨叨→吴邪（线索单向输入）。写局时先查边，再决定谁能出现在同一场戏。

**【吴邪十七年失败的叙事功能】**
每一次失败都在回答：什么样的人不配进古潼京。太听话的会被汪家直接收编；太聪明的会提前拆局；太善良的会在第一具干尸前崩溃。黎簇的“刚好”在于：有恨、有脑、有被社会抛弃的弹性，且身上出现可被利用的生理标记。契约者若扮演“第十八人”，默认你也曾被评估过——可要求吴邪出示评估标准，作为二阶情报任务。

**【城市篇的空间】**
网吧、天桥下、老旧居民楼、校门口的 autom 贩卖机、深夜的打印店——这些地方承载快递与密谈。汪家在城市的存在感是“尾随的电动车”和“突然修好的监控”，不是黑衣人 trace  trace  trace。吴邪的城市据点则像临时摄影工作室：灯箱、样片、没人的暗房。

**【沙漠篇的队形】**
前点、侧卫、断后、水源官、通讯官。黎簇常被放在既危险又“能学到东西”的位置。杨好抢前点，苏万被按在水源。向导海子有一票否决权：沙暴来时谁说了都不算。夜营忌讲鬼故事——不是迷信，是幻影机制会吃故事。

**【训练营一日流程（供三阶场景）】**
黎明集合、点名、体能、枪械拆装、沙漠伪装、政治课式洗脑、两人一组互监、夜间突发拉练。私藏通讯设备=清洗。黎簇的优势是已在沙海见过真蛇，对“神话课”免疫更快。

**【湘西篇的气味与声】**
湿木、酒、枪油、方言吵架、远处的鼓。与沙漠的干、亮、静形成感官对比。怒晴线人物说话绕弯，不像汪家口号，也不像吴邪的任务书，更像旧社会讨价还价。

**【关键道具再列（防AI用代称）】**
沈琼之盒；背纹刀具痕迹；关根的相机与镜头盖；寄尸快递单；无法成像的岩石照片；蛇蜕样本瓶；汪家工牌；训练营名册；古潼京结构草图（可能有假）；吴邪手上的新旧刀疤；海子的沙地车钥匙；解雨臣提供的假证件；黑瞎子的墨镜（符号性出场）。

**【战斗写法示例（机制优先）】**
错误：黎簇一拳打死沙蟒。正确：沙蟒从侧翼埋沙绞住杨好，苏万开枪惊散沙层，黎簇用火与噪声逼蛇改道，吴邪在高处计时——三分钟后沙暴到，谁还纠缠谁死。错误：与张起灵对砍。正确：影子一闪，场上最强敌手失去武器，吴邪说今天的课到此为止。

**【情报等级】**
D：都市传说（叨叨成像）。C：沙漠生存与蛇习性。B：汪家训练营位置与作息。A：古潼京饵的时间表。S：黎簇消失条件与吴邪真实损耗。低阶任务勿直接发S。

**【死亡与失踪规则】**
普通人死了就是死了；棋子失踪可能被两边伪造；高手“死”要验尸，因为易容与替身存在。青铜门相关死亡另论，沙海正文少碰。

**【金钱与报酬】**
吴邪付现金、装备与“你家人暂时安全”；汪家付归属与前途幻觉；九门付人情。乐园结算时把“暂时安全”标为高风险契约条款。

**【可重复开局的裂缝】**
十七名失败者之一仍活着；某张无法成像照片在拍卖；苏万家再次收到包裹；古潼京外围出现新的摄影团；黎簇失踪后的衣物被寄回——任意一条都可开一阶到三阶副本。

**【与《盗墓笔记》《老九门》分界】**
老九门=前史派系；盗墓笔记=铁三角成名与青铜门；沙海=吴邪单人战争与少年棋子。同库多世界时勿串时间线人格：沙海吴邪更冷，正传早期吴邪更懵。

**【叙事禁用清单（本世界专用）】**
禁把黎簇写成无脑复仇工具；禁把沙漠写成无限经验本；禁让张起灵长期打工；禁汪家全员脸谱白痴；禁古潼京无代价通关；禁用“主角光环”解释读蛇。

**【长场景：第一次看见白沙】**
风停的三秒里，地平线变成一张过曝的纸。相机自动关机。黎簇说“像有人把世界的颜色抽走了”。海子让所有人用布裹脸，不是因为冷，是因为沙子会往牙缝里钻进“细得像记忆的东西”。吴邪数人数，数到你时停顿半秒——那是评估，不是关心。远处有黑色线状物移动，苏万想喊蛇，杨好按住他的嘴：喊了，它就知道你怕。这一段可直接用作二阶开场白的加长版。

**【长场景：训练营夜话】**
黎簇问你：如果关根让你死，你死不死？你还没回答，哨声响了，全员趴下。探照灯扫过沙地，像在找谁的良心。汪家教官说：问这种问题的人，明天加练。黎簇却在暗里笑了一下，那笑很像吴邪——沙海最恐怖的传承不是能力，是表情。

**【长场景：饵爆前夜】**
所有通讯频道同时出现沙沙声，像千百条蛇在电缆里爬。解雨臣的消息延迟三小时到达：计划有变，但仍执行。黑瞎子的价码翻倍。吴邪把相机电池取出来，说从现在起，不留下影像，只留下结果。黎簇背对着众人，问最后一句：我是不是从一开始就不能回家？没有人回答。风替他们回答了。

**【主题回收】**
盒子、背纹、刀痕、成像、蛇语、训练营、古潼京、消失——八个词串起全书。契约者记住这八个词，就能在任何时点定位自己站在哪一环。

**【字数向补强的原作信息密度说明】**
以上补述均对应少年篇公开情节结构与人物功能，不引入原作不存在的第三势力神器；查无的地名细节写“不详”，已知的人名保持真名。

**【分卷对照·出版常见四卷结构】**
公开分卷多作：《沙海1·荒沙诡影》《沙海2·沙蟒蛇巢》（或古镇谜局向中间卷）、《沙海3·鬼河死海》《沙海4》及相关湘西延伸。网络连载与实体卷序偶有出入，部分卷四时间线回补前史。**荒沙诡影**：高考落榜、家暴与幽闭、沈琼之盒、七指/背纹、关根邀约、海子入白沙、古潼京初见、黑毛人面蛇与干尸、黑眼镜救援、寄尸包裹。**古镇/城市场景卷**：回城后快递与尸块、浙南古镇或诡异农宅、梁湾线、蛇群与快递公司冲突、父亲与古潼京工程旧账浮出。**鬼河死海·沙蟒蛇巢向**：再探地下管道壁画黄金洞、沙蟒巢、人蛇与读信息能力坐实、汪家寻蛇矿、少年组与胖子/黑眼镜阶段性汇合。**收束向**：沙海计划以古潼京混淆汪家视线、黎簇因读蛇能力成关键棋子/刀，原著部分伏笔留白，连向《重启之极海听雷》。影视2018版人物保留大框架但细节不可覆盖小说真名关系。

**【七指图与背纹机制】**
背上被刻的诡异图腾不是单纯恐吓，而是与古潼京结构、七指图腾骸骨迷宫或地图线索绑定的生理标记。黎簇因此被两边盯上：吴邪要可用的读图/读蛇载体，汪家要可被训练的容器。写一阶开局时，背纹发炎、渗血、被拍照存档，都是合法钩子；禁止写成无代价的满级地图外挂。

**【古潼京的双重身份】**
表层是沙漠死亡禁地、会移动的海子与白沙；里层叠着八十年代按古图推进的**机密工程遗址**——管道、设计图、黄金洞与现代施工痕迹并存。因此“倒斗”在本卷同时是考古、谍战与生物灾害。无法成像的岩石、相机冢、消失的人，都服务于同一规则：这里不欢迎被记录，只欢迎被消耗。

**【黎簇家庭与入局心理】**
酗酒暴力的父亲、出走的母亲、分数与同学嘲笑，构成“为什么能被带走”的社会条件。幽闭恐惧被关小黑屋触发后，沙漠与地宫的狭窄反而成为二次创伤与成长场。吴邪拆开缝合线式的强硬邀约，是布局者语言：不给你拒绝的体面。契约者若想“拯救少年回家”，必须回答——家是否仍比汪家安全。

**【新铁三角功能定位】**
黎簇＝质疑与异能载体；杨好＝义气与莽撞的行动力；苏万＝正常世界与资源接口（家庭、药品、胆小却在场）。三人同时存活时，读者与AI才有情感锚；任意一人长期缺席，沙海会滑向纯谍战，失去少年篇辨识度。

**【梁湾与身体标记线】**
梁湾线常与纹身、身世、药品或“谁更像正常人”对照。她不是花瓶支线，而是用来测量黎簇异化速度的尺子：当少年开始用吴邪的语气说话，梁湾的在场能逼出“你还要不要做人”的选择。

**【黑眼镜与解雨臣的出场经济学】**

## 阶位切入点

> 阶位↔原作强度：一阶≈普通学生/市民；二阶≈受训倒斗者/汪家基层；三阶≈黑瞎子·解雨臣·成熟吴邪级；四阶≈古潼京核心异象/区域级局崩。世界顶点（古潼京机制+顶尖倒斗者）≈四阶；低阶切入一律**规避顶点·情报优先/条件性胜利**。

**一阶（高考落榜·开盒前后 · 被选中的普通人）**

切入身份/时点：契约者以复读生/网约车司机/快递员等凡人身份出现在黎簇所在城市，时点锚定**沈琼之盒即将被打开前后**。

初始事件：夜巷传来闷响，一名后背渗血的少年（**黎簇**）被拖上厢式车；你捡到掉落的金属盒角或未寄出的“苏万收”快递单，下一秒有人请你“帮忙送一趟沙漠摄影器材”。

开场白建议：「高考分数条被风吹进下水道的那天晚上，你本只想买瓶冰水。巷子深处有人用刀在少年背上刻图案，血味混着沙土——电话响了，对面自称关根，说你已经看见不该看的，现在有两个选择：报警，或者来巴丹吉林挣一笔“采风费”。」

关键NPC立场：**黎簇**（戒备、把你当又一个骗子）；**杨好**（冲动护友）；**苏万**（害怕却好奇快递）；**吴邪/关根**（打量你是否配做第十八枚棋子）；**沈琼**（若在场则闪躲）。

主线钩子/支线：主线=是否阻止/协助开盒与背纹事件；支线=追查十七刀传闻、拦截寄往苏万的干尸包裹、在网吧还原叨叨成像诅咒的都市传说。

危险度/规避：中（对人）；规避汪家早期监视与直接对抗黑瞎子。报酬：生存经验、吴邪外围情报碎片、沙漠入场券。

任务方向/奖励：活着摸清“关根要少年干什么”；奖励以人脉、基础装备、一张真假难辨的古潼京照片为主，不发读蛇神功。

**二阶（入沙·沙蟒外围 · 能活过第一周的人）**

切入身份/时点：沙漠车队向导助手/私人安保/假考古队成员，时点=**黎簇首次随关根进入巴丹吉林**。

初始事件：白沙边缘发现无法对焦的岩石与一具半埋干尸；沙蟒游迹切断归路，海子要求有人断后。

开场白建议：「卫星电话只剩杂音。关根把镜头盖扣上，说今天不拍照，只教你们怎么在沙子里听蛇。你后腰的水只剩三分之一，而黎簇盯着沙丘后面说：有东西在跟我们数人数。」

关键NPC立场：**海子**（务实、可雇）；**黎簇**（开始疑心却依赖团队）；**吴邪**（用任务筛选）；**杨好/苏万**（恐慌中的忠诚）；汪家外围眼线（冷眼记录谁活下来）。

主线钩子/支线：主线=护送少年组抵达白沙观测点并取回成像异常样本；支线=救下中暑队员、破解寄尸快递源头、与梁湾争夺有限药品。

危险度/规避：高；规避古潼京核心区与主动挑衅沙蟒巢。奖励：沙漠生存术、初级蛇类行为笔记、二阶级枪械/绳索套装。

任务方向/奖励：证明你不是炮灰；奖励以生存装备与“可被吴邪暂时信任”的标记为主。

**三阶（汪家训练营·渗透 · 棋手与刀）**

切入身份/时点：假投名状进入汪家训练营的教官候选人/九门侧派遣的暗桩，时点=**黎簇取得汪家信任前后**。

初始事件：训练营夜跑，一名试图逃跑的学员被公开惩戒；你的档案被要求“证明对古潼京饵的忠诚”。

开场白建议：「扩音器里循环播放千年与归属。黎簇的眼神已经不像高中生，更像一把还没开刃的刀。解雨臣的短信只有六个字：别信营里水。而你的工牌背面，刻着汪字。」

关键NPC立场：**黎簇**（可结盟可背刺）；**汪家训练官**（考核与清洗）；**解雨臣**（远程交易）；**黑瞎子**（高价暴力）；**吴邪**（要结果不要你命）。

主线钩子/支线：主线=配合或破坏“送黎簇入核”的节奏，决定灭汪饵是否提前引爆；支线=盗取训练营名册、调包蛇样本、在湘西线接头怒晴相关人物。

危险度/规避：极高；规避与黑瞎子无准备的正面死斗、规避在训练营公开喊吴邪真名。奖励：组织情报、易容/假身份术、三阶任务级装备。

任务方向/奖励：拿到汪家调度图或古潼京行动时间表；人脉向解/黑/吴三方之一倾斜。

**四阶（古潼京核心·局崩 · 区域灾厄）**

切入身份/时点：吴邪核心执行组/九门联合行动顾问，时点=**古潼京诱饵正式吞没汪家主力**前后。

初始事件：白沙风暴中成像设备集体失效，蛇激素场让全员幻听；有人必须进入鬼城结构最深层点燃“饵”的最后一环。

开场白建议：「关根摘下墨镜，第一次用吴邪的声音说话：从这里开始，棋子和棋手都可能死。古潼京的影子在沙下站起来，汪家的旗帜在风里裂开——你手中的起爆方案，签名栏是空的。」

关键NPC立场：**吴邪**（终局操盘，可牺牲任何人）；**黎簇**（觉醒后的反噬与失踪前夜）；**解雨臣/黑瞎子**（有限强援）；**张起灵**（若影子现身则情报优先，勿硬刚）；汪家高层（疯狗式反扑）。

主线钩子/支线：主线=完成或改写灭汪引爆条件，决定黎簇是否必须消失；支线=保存成像异常样本带出沙海、救出苏万杨好、截获汪家带出的终极碎片情报。

危险度/规避：贴近顶点；**顶点按四阶区域灾厄处理，存在·情报优先/条件性胜利**。奖励：布局者级情报、古潼京机制笔记、四阶许可的区域任务结算（装备/人脉/蛇系能力线索，不发行星级神器）。

任务方向/奖励：让局“成”或“可控地败”；奖励以真相、残局话语权与后续《重启》向钩子为主。
