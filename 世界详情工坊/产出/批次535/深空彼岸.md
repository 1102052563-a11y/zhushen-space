## 剧情

**【作品来源】**
《深空彼岸》为**辰东**（陈东）创作的异术超能/科幻玄幻长篇，2021年5月1日至2023年6月1日连载于起点中文网，约534万字，已完结；后有番外《道外与后院》。实体书由安徽文艺出版社等出版，并有B站国创动画改编。类型：星际文明+东方旧术+超凡周期性兴衰。文风为辰东式前期悬念探险、中期密集突破、后期格局拉至宇宙/纪元；与《遮天》三部曲无直接主线关联，独立成篇。搜笔趣阁书页：`https://www.sobqg.com/book/s2-shenkongbian.html`（slug 带 **s2-** 前缀）。

**【世界定位】**
人类在世纪大战后分裂为衰落的**旧土**（地球）与高科技繁荣的**新星**；科技“新术”压过先秦“旧术”。主角**王煊**被阻无法移民新星，加入秘路探险，凡人期开启**内景地**，在超凡落幕时代成为唯一可持续修行的**逆行者**，以**全领域六破**走上归真之路，揭开深空尽头与灾主真相。一句话：**旧土青年练旧术，逆行穿过光海与大幕，问道彼岸。**

**【世界观 · 力量体系】**
力量来源分两轨：
1. **旧术**：先秦方士根法、体术（金身术、道家散手、大金刚拳、菩萨拳等）、精神内视、采气内养；核心金手指是**内景地**（精神/秘境空间，可遇古强者残影、时间差加速、接引伤者）。
2. **新术**：基因优化、科技+超自然物质沟通，财阀与新星主流。
3. **秘路体系**：寻路/燃灯、命土、采药、天药、逝地、荒芜之地；对应超凡源头与落幕后的逆行路径。
4. **超凡规则**：超凡会周期性**落幕/退潮**；列仙曾抽走能量留大幕与旧约；神话腐朽、真实物质与真实晶石争夺贯穿中期。
5. **破限与归真**：各大境界常规约9段/重天，可**破限**至第10段起，理论最高**六次破限**（六破≈一次归真）。王煊在六大境界均完成六破，称**全领域六破**。
6. **更高层**：真圣道场、超凡中心宇宙、地狱、起源海、归真之地、灾主、道主、唯一道/道之外。

**凡人阶段（旧术基础）**：
- **采气**：采集外界能量入体。
- **内养**：滋养血肉、改体质。
- **准宗师 / 宗师**（初中后圆满）：一域高手，有宗师意识。
- **大宗师**：近超凡门槛，可吐能量光束级表现。

**超凡六大境界**（命名多取自《庄子》意象）：
1. **人世间**：超凡入门。王煊路标细分含迷雾、燃灯、命土、采药、定路、化劫、外感、拓荒、空幻、破限、边荒等；可开内景地。
2. **逍遥游**：壮大元神、神游太虚；中段可称地仙级神游。
3. **养生主**：形神合一温养；肉身可手撕战舰级，精神可星空生存；以“第几生”计。
4. **羽化仙**：渡成仙劫，正式真仙；极难保留**原始仙体**（完整肉身羽化）；准绝世在九重天附近。
5. **幕天境**：绝世强者；十段起元神涅槃为**超绝世**（星域共主级）；方雨竹曾为十一段超绝世标尺。
6. **御道境**：可御天下万道。1–9重多为**异人/奇人**（御道化积累）；破限后公认真圣；更高有大能、**真王**、**灾主**、道主等。

**乐园阶位映射（对照阶位战力图鉴破坏力，宁低勿高）**：
- 普通人/采气/内养/准宗师≈**一阶**（冷兵、徒手、坠楼可致命）
- 宗师≈**二阶**（碎砖裂石、小队级）
- 大宗师≈**三阶**（裂石断金、端据点）
- 人世间（超凡入门~破限前）≈**四阶**（据点/建筑群、超凡异能成型）
- 逍遥游≈**五阶**（城市/区域级、神游与领域雏形）
- 养生主≈**六阶**（地区级、手撕战舰、御空长程）
- 羽化仙≈**七阶**（行星/世界级、成仙劫与真仙手段）
- 幕天境绝世~超绝世≈**八阶**（星域/行星表面级、规则权柄初显）
- 御道境异人~真圣≈**九阶**（恒星系/文明级、言出法随雏形）
- **真王 / 灾主 / 道主 / 道之外·王煊终局**＝**超阶**（跨世界规则级；低阶切入一律**规避顶点·情报优先/条件性胜利**，禁止用“任务限制战力/暂时沉睡=无威胁”解释顶点）

**【地理 · 舞台】**
- **旧土（地球）**：世纪大战后边缘废土；大学旧术实验班、都市、荒漠矿脉、葱岭争锋地、养生殿据点、灰血组织巢穴。
- **新星**：虫洞移民后的高科技主星；财阀（钟家、孙家等）盘踞；密地探索出发港；新旧术舆论场。
- **密地 / 深空密地**：类福地秘境的生命星球或奇异空间，黑狐、地仙残存、图腾与异宝并存。
- **命土 / 荒芜之地**：超凡立足与采药关键层；天药主根、药雾；落幕后可通超凡源头。
- **逝地**：真实物质与交易节点，地仙泉等机缘所在。
- **大幕 / 混沌洞 / 神话残域**：分隔现世与超凡、锁旧约；逍遥舟曾驻守混沌洞守护沉睡者。
- **超凡光海**：超凡能量退潮时的迁徙通道；通往**超凡中心宇宙**与真圣道场（无忧宫、冲霄殿、出世山等）。
- **归真之地**：大道符文、天灾奇景、古关；灾主主场；宇宙本源**鼎**相关。
- **星空战场**：虫洞航线、列仙飞船残骸、真实晶石矿带、星空大雾裂缝。

**【世界剧情线】**

① **旧土起步：被阻的新星船票与秘路入局**  
王煊出身旧土，入大学**旧术实验班**，痴迷先秦根法与体术，成绩出众。本可前往新星，却因前女友家族等阻挠滞留旧土，错失主流上升通道。他加入**秘路探险组织**，与**青木（赵青空）**、**陈永杰（老陈）**等共事。灰血组织暗杀线触发：王煊在长时间超感状态中**凡人开启内景地**——此为全书核心金手指。内景地中修成**金身术**层层推进，接触古强者残影与秘法。突袭灰血、获取道家散手等秘技，借**养生炉**等至宝反杀已踏超凡的敌人。建立**养生殿**，与财阀孙家等在旧土资源与声望上硬刚。同期结识大学好友**秦诚**、**吴茵**等，埋下“凡人期情谊在落幕后成血泪”的伏笔。阶段结局：王煊从“被淘汰的旧术生”变为旧土秘路焦点人物。

② **新星与密地：赵清菡、天药与内景交汇**  
王煊赴新星，卷入财阀与家族联合的**密地**探索。**赵清菡**（新星血统、大学同学，后称赵女神）在密地遇险，被王煊救下并祛除病根——感情线与“神”之真灵印记伏笔自此绑定。密地中有黑狐、旧狐地仙、图腾与异宝；王煊用内景地时间差狂练**大金刚拳、菩萨拳**等佛门绝学。穿**命土**入荒芜之地，取得**三株天药**（三药并存），养生炉养药养身养神的体系成型。钟家**钟晴、钟诚**等财阀二代以资源与秘法交换情报，蛇鹤八散手等流入王煊之手。葱岭等地发生新旧术宗师级大战：老陈重伤，王煊接引入内景地与炉中恢复——“互坑搭档”关系坐实。阶段结局：王煊站稳超凡门槛，命土—天药—内景地三角闭环。

③ **列仙回归与至宝风暴**  
神话回光返照，**列仙**影响力回潮，妖魔猎杀、真实晶石争夺、跨星系奇异空间齐发。王煊主导或深度参与猎杀行动，诱取天药与遗物。上古强者残影/真身相关线展开：**方雨竹**（先秦顶尖女方士，幕天境十一段超绝世标尺，羽化金竹船与幕天镯）、**姜清瑶**（剑仙，残骨入内景、旧土三年之约、后入冲霄殿）、**燕清妍**（红衣女妖仙/妖主，半人半妖，猎杀顶尖羽化方士，王煊视其为干姐姐）等通过内景、遗骨、白虎大妖内景等地与王煊结下因果。商毅（剑疯子）等人世剑+羽化幡线制造教祖级威胁；逍遥舟速度神话、混沌洞驻守等至宝规则被写清。父母**王泽盛、姜芸**与兄长**王御圣**的真圣/奇人身份逐步揭开——王煊并非无根的孤胆英雄，而是有深渊背景的“家传逆行”。阶段结局：现世超凡被列仙棋盘化，王煊成为少有能同时摸到旧术根法与列仙遗产的棋手。

④ **超凡落幕：逆行者诞生**  
宇宙规则校正，超凡能量被抽走，**超凡落幕**。依赖外在超凡的强者大面积退化；好友凡人化、死亡与离散。赵清菡为渡劫式蜕变被安排入**混沌洞/逍遥舟**系庇护沉睡。方雨竹、姜清瑶、燕清妍等顶尖者随**超凡光海**迁往超凡中心宇宙，分别加入无忧宫、冲霄殿、出世山等真圣道场。王煊发现**命土之后的超凡源头**，成为母宇宙中**唯一能持续修行**的人——“逆行者”名号坐实。他将**古今**等友人送往超凡大宇宙，自己另寻路径；羽化时保留**原始仙体**（完整肉身），与碎身后只留元神的常规列仙形成对比。阶段主题：文明与神话都会退潮，个体是否还要逆行。

⑤ **超凡中心宇宙与全领域六破**  
格局从旧土—新星升至超凡中心宇宙、真圣道场、地狱、起源海。王煊在**人世间、逍遥游、养生主、羽化仙、幕天境、御道境**六大境界均完成**六次破限**，持有六件**元神圣物**（草藤、沙漏、混沌物质、银色纸张、阵图、氤氲雾霞），并以沙漏等赠方雨竹、阵图等予姜清瑶。真圣大劫、红尘劫、人祸与言出法随级战例密集出现。父母走**枯寂之路**尝试全境六破；兄长王御圣背景复杂。云舒赫（上古第一人）、麻（旧圣第一人相关）、齐天等历史级名字进入叙事。王煊自称调侃式“**王教祖**”，实则开创破限归真可复制讨论的道路。阶段结局：他从逆行者成长为可俯视星海的御道级存在，并继续向上。

⑥ **归真之地、灾主与正篇终局**  
王煊成就**真王**级，再触**灾主**门槛，进入传说中的**归真之地**——大道符文、天灾奇景、古关密布。与一众灾主（如“狱”等公开讨论中的存在）大战，**斩杀灾主**，自身重伤，进入宇宙本源至宝**鼎**中（可复活/重生级）。深空尽头关于超凡大循环、升维逃亡、旧约与真实的谜题在此收束一层。正篇收束偏仓促的读者评价存在，但因果上完成：旧土青年走到了灾主战场与本源鼎前。

⑦ **番外《道外与后院》与结局方向**  
王煊在**唯一道/道**边缘区域活动，发现“第15人”等设定延伸；**自斩境界**后复活故人（如秦诚、吴茵等），重构超凡界。与**赵清菡**团聚，育有三子**王晔、王昕、王晖**。方雨竹、姜清瑶、燕清妍等红尘因果在番外有暗示性后续（公开资料未全部明说，写卡时保留迷雾，禁止编造具体婚配细节）。主题落点：彼岸不是逃离，是把后院与故人从神话退潮里捞回来。

⑧ **因果链摘要（供 AI 时点判断）**  
旧术实验班 → 阻新星 → 秘路/灰血 → 开内景地 → 养生炉反杀 → 新星密地救赵清菡 → 命土三天药 → 葱岭与老陈 → 列仙回归与三女仙因果 → 超凡落幕与光海迁徙 → 逆行唯一 → 全领域六破与元神圣物 → 真王/灾主 → 归真之地入鼎 → 番外自斩复活与团聚。任意时点问“世界在哪”：看超凡是否已落幕、主角是否已渡光海、是否已入归真。

- **云舒赫**（上古第一人相关）｜性格：历史级压迫感｜装备·能力：上古顶点级｜人物弧光：复苏与御道化相关｜立场关系：时代天花板之一。

**【势力图谱】**
- **秘路探险组织**：旧土侧寻命土/天药/逝地的实务集团；代表陈永杰、青木、王煊。
- **灰血组织**：暗杀与超凡黑产，开篇仇敌。

- **真圣道场**（无忧宫、冲霄殿、出世山等）：超凡中心宇宙的收容与修炼体系。
- **灾主群体**：归真之地顶级存在，灭世级，正篇终局对手。
- **王煊后院/教祖影响力**：后期以个人道路吸附盟友与故人，非传统宗门。

**【贵重物品】**
- **养生炉**：母宇宙至宝；养药养器养神养身；反杀、救姜清瑶、争渡光海时失控等关键戏。
- **内景地（特殊开启）**：非器物但等同核心至宝空间。
- **三株天药**：命土荒芜之地所得，早期质变资源。
- **人世剑 / 羽化幡**：商毅持有；劈大道、划大幕级。

- **幕天镯**：方雨竹炼出，关联王煊父母线。
- **六件元神圣物**：草藤、沙漏、混沌物质、银色纸张、阵图、氤氲雾霞——破限与权柄。
- **因果钓竿、归墟灯**：旧圣遗物系，王煊持有。
- **羽化石 / 羽化金竹船**：接通方雨竹内景与肉身保留的媒介。
- **真实晶石 / 真实物质**：落幕前后争夺的硬通货。
- **宇宙本源鼎**：归真之地终局级，可令重伤者入鼎重生。

**【大事记时间线】**
世纪大战→旧土/新星分裂→旧术实验班→王煊被阻滞留→秘路入职→灰血暗杀·开内景→养生炉反杀·建养生殿→新星密地救赵清菡→命土三天药→葱岭大战救老陈→列仙回归·三女仙因果→至宝风暴（人世剑/逍遥舟等）→超凡落幕·光海迁徙→逆行唯一·原始仙体羽化→全领域六破·元神圣物→真圣道场博弈→真王/灾主→归真之地大战入鼎→番外道外自斩·复活故人·团聚三子。

**【叙事基调 · 雷区】**
基调：星际冷漠感+旧术烟火气+互坑喜感+后期神话压迫。忌：把王煊写成无代价无敌（落幕死人、自斩、入鼎皆是账单）；忌忽略内景地时间差与破限规则乱开战力；忌用“新术科技碾压”抹掉旧术主题；忌把赵清菡、方雨竹、姜清瑶、燕清妍写成无名字红颜群像；忌把灾主/真王写成“沉睡/任务限制=无战力”。最早可切入锚点：旧术实验班毕业被阻之夜，或灰血暗杀开启内景地前后。

**【旧土社会肌理】**
旧土不是纯废土打怪图：有大学编制、实验班政策、秘路编制工资、财阀外包勘探、灰血地下经济。王煊的选择成本是“失去新星户口式前途”。写低阶任务应先写签证、船票、组织徽章与实验室门禁，再写拳头。

**【新术与旧术的舆论战】**
新术掌握媒体与基因诊所，把旧术描述为封建残留；旧术掌握命土矿权与方士遗产。葱岭大战既是武力也是舆论——宗师死亡会改变投资人与移民局态度。契约者若只打架不碰舆论，会漏掉半条主线。

**【内景地可观察规则】**
进入需根法/媒介/重伤临界等条件；内部时间流速可差；可遇残影不可默认全知；接引他人有反噬风险；落幕后多数人的内景枯死，王煊例外。战斗描写优先“精神过载、记忆反灌、药香与血雾”，少写无脑光波对轰。

**【破限的社会含义】**
破限不是加属性点：每次破限引发天劫式现象、吸引真圣注视、改变命格可见度。六破者在超凡中心是传说，在母宇宙落幕后是孤本。写九阶以上必须写“被注视的代价”。

**【超凡落幕执行细则】**
能量退潮表现为：术法失灵、天药枯、战舰上的超凡镀层剥落、强者肉身回归凡人病痛。赵清菡沉睡、故人 degenerate、光海启航应同一时间窗发生，避免时间线打架。

**【光海渡口场景】**
渡口是人群、哭声、道场使者与最后一艘“船”（光海形态）的混响。王煊送人而不自送，是人物弧光高点：他不是逃难领袖，是留下点火的人。

**【归真之地法则感】**
此地不以星球国家划分，而以天灾奇景与古关权限划分。灾主战斗应写概念与奇景对撞，禁止写成普通星球擂台。入鼎=承认本源规则高于个人怒火。

**【家族线用法】**
父母真圣不是开局外挂：他们在枯寂与道场政治中自顾不暇。兄长王御圣可作高阶情报源与压力源。低阶切入不应让父母下场清图。

**【女仙三人因果边界】**
方雨竹、姜清瑶、燕清妍各有道场与时代账单；与王煊是因果非后宫模板。写互动用旧约、三年之约、干亲、赠宝、联手杀敌，禁止无来由倾城。

**【财阀钟家窗口】**
钟晴可提供资源与星空情报，但钟家永远先算账。蛇鹤八散手等交换要写成合同与人情债，不是无脑送宝。

**【商毅威胁标尺】**
人世剑在手的教祖级疯子，用来标定八阶上限的“人祸”。他的死/败应牵动羽化幡内女奇人、云舒赫复苏等连锁，不是单挑打卡。

**【终局道德账单】**
斩灾主≠幸福结局；入鼎、自斩、复活故人都是用自身境界与因果付账。契约者若想改结局，必须提供等价替代（另一人入鼎、放弃复活名单等）。

**【动画与小说边界】**
国创动画多覆盖旧土—新星—密地前中期；人物造型可参考，**剧情与人名以小说为准**。禁止用动画原创桥段覆盖正传因果。

**【语言与称呼】**
王煊自称老王/王教祖带自嘲；老陈叫法随意；列仙称呼讲究道号与时代。对话宜短句、互坑、突然认真，忌中二长篇独白。

**【可重复开局的世界状态】**
落幕前后、光海启航前夜、归真前夜均为高密度开局窗。世界不会因王煊通关而变成无冲突乌托邦——灾主残韵、道场政治、旧土复兴与后院重构仍提供任务面。

**【旧土实验班一日流程】**
晨训体术（金身架势、散手对桩）、根法冥想课、超感仪器检测、午后文献（先秦残简数字化）、夜间可选秘路兼职。资格审查不只看成绩，还看背景审查与“新星适应评估”——王煊被卡在最后一项与人际关系项，是剧情发动机而非随机不幸。契约者若同班，应被同一套表格羞辱过一次，才算进世界。

**【灰血组织运作方式】**
灰血不是街头混混：有超凡边缘杀手、转运矿洞、对秘路探矿队的定点清除单。暗杀王煊的逻辑是“旧术苗子+可能接触内景=不安定因素”。清剿灰血不是升级打怪，是证物链、内鬼与炉火三线并行；养生炉第一次大规模反杀，确立“至宝可越阶”的早期爽点，也确立“越阶必付精神/肉身账单”。

**【养生殿的政治学】**
养生殿对外是疗养与药草机构，对内是旧术资源集散与舆论据点。孙家等财阀打压它，因为它把“封建旧术”重新产品化。王煊用殿养药、养伤、养名声；老陈用殿养鱼（情报）。低阶任务大量发生在殿外排队、药方伪造、竞品纵火，而非星球大战。

**【密地生态与黑狐】**
密地有独立生态链：黑狐群、旧狐地仙残存、图腾柱、异常磁场干扰导航。救赵清菡之战要写能见度、通讯延迟与分队割裂，而不是无脑冲塔。赵清菡身上的异常读数（神之印记相关）会吸引密地原生意志与人类监军同时盯梢——救人等于同时抢夺叙事权。

**【命土三药的经济后果】**
三株天药并存改写旧土—新星药价与矿权谈判。药雾可致幻、可洗髓、可引来列仙残念。王煊取药不是单人副本通关，是引发后续猎杀与交易链的按钮。写四阶任务时，天药每一次露面都应带来至少一股新敌人或新买家。

**【人世间路标的实操意义】**
迷雾=精神内视启蒙；燃灯=寻路与精神淬炼；命土=超凡立足；采药=自体内外药性；定路=路径选择（丹、武、神等）；其后化劫、外感、拓荒、空幻逼近破限。AI 写战斗可用路标名作家招后缀，但不可把路标当独立第九套境界。

**【列仙回归的社会冲击】**
列仙回光返照时：博物馆文物活化、旧约条文被引用为国际法、真实晶石期货暴涨、妖魔猎杀变成有奖金的国家项目。钟诚向往列仙、钟晴学习身世，是财阀家庭被神话刺穿的样板。王煊在此阶段的优势是“能进内景见活人残影”，不是单纯战力表第一。

**【三女仙登场媒介差异】**
方雨竹：羽化石/金竹船/元神投影，志在列仙与幕天之路。姜清瑶：残骨、小道观羽化痕迹、剑意误伤后的和解，三年之约是硬时限。燕清妍：白虎大妖内景、红衣声传、猎杀羽化方士的战绩，干亲称呼锁定伦理位置。三人不可互换脸谱。

**【超凡落幕的个体样本】**
秦诚、吴茵等凡人期好友的退化与死亡，是落幕的人脸；赵清菡沉睡蜕变是落幕的特例通道；多数宗师发现术法像断电。王煊逆行不是“光环免疫落幕”，而是找到命土后源头、用更高成本维持修行——写他必须写疲惫、孤独与送人登船。

**【原始仙体的稀缺】**
常规羽化多碎身后元神登仙；王煊保留完整肉身的原始仙体，是后期肉身手段与精神手段双开的资格证。渡劫场面应写筋骨与元神同时承雷，而非只写元神飘逸。七阶切入若碰羽化，优先写“碎或不碎”的选择恐惧。

**【元神圣物权柄分工（公开归纳）】**
草藤偏生机/缠缚道韵；沙漏偏时空；混沌物质偏本源浑沌；银色纸张偏记录/裁写；阵图偏空间阵法；氤氲雾霞偏隐匿/气化。赠宝=嫁接因果：沙漏予方雨竹、阵图予姜清瑶等，是政治联姻式的道韵绑定，不是单纯送礼。

**【真圣道场三宫速写】**
无忧宫（方雨竹线）：收容与造化气相关考验。冲霄殿（姜清瑶线）：剑与炉的归属争议高发。出世山（燕清妍线）：妖族/天血叙事与杀伐传统。三宫不是善恶阵营，是资源与道路竞争；王煊常被三方同时要债。

**【归真之地开战逻辑】**
灾主不是“更大号真圣”，而是归真规则的既得受益者/天灾具现。王煊斩灾主后重伤入鼎，说明胜利条件是**条件性**的：赢了战斗，输了完好身躯，把续命权交给本源鼎。九阶任务应围绕古关权限、名单伦理、鼎的观察权，而不是 DPS 竞赛。

**【番外后院的叙事功能】**
《道外与后院》把爽文终点折返成人情：自斩境界以复活故人、重构超凡、发现第15人。它允许契约者在“通关后世界”开局——冲突来自道外规则、复活名单排序、教祖声望与家庭烟火。禁止把番外写成无代价全员永生。

**【与乐园术语对齐的一句】**
本世界破坏力从凡人冷兵拉到灾主/本源级，故切入覆盖一至九阶，**峰值超阶**；低阶奖励停在根法/药/人脉，高阶才碰圣物残韵与道场推荐，严禁越级发“六破外挂”。

**【关键战役/名场面索引（供时点跳转）】**
灰血反杀·炉光初现；葱岭宗师战·老陈重伤接引；密地救人·印记初显；命土取三药；列仙猎杀与至宝互夺；光海渡口送别；六破领域天劫；归真古关灾主战；入鼎；番外自斩复活。选世界卡=选上列任一帧的前后二十四小时。

**【敌对力量分层再表】**
街头灰血＜财阀安保＜新术宗师团＜列仙走狗/商毅系＜真圣道场政治＜灾主。跨层硬刚必须有至宝、内景、破限迷雾或情报陷阱，禁止“越两层无伤”。

**【真实物质与真实晶石】**
落幕前后的硬通货：可稳超凡、可交易、可引发战争。逝地是重要交易节点。写经济线时，晶石比乐园币更像本地硬通货；契约者若只认乐园币会OOC。

**【旧约与大幕】**
旧约约束列仙如何影响人间；大幕是物理/规则屏障。幕天境“身在大幕中登顶”的意象，决定八阶战斗可涉及幕壁反光、旧约反噬。乱开“一剑斩幕”而不付代价=违背世界规则。

**【父母枯寂之路】**
王泽盛、姜芸尝试全境六破的枯寂路线，是家庭线的高阶副本：成功则改写“六破唯一性”舆论，失败则王煊多一份血债。九阶可闻其名，低阶只许听说“有真圣父母”的传闻级压力。

**【王御圣的使用边界】**
兄长是真圣棋手，可提供道场通行证级帮助，也可把弟弟拖进更高层政治。写他出场必须带目的，禁止当随身保镖。

**【云舒赫与历史层】**
上古第一人级名字用于标定“比超绝世更老的账”。商毅围猎史、羽化幡内女奇人、云舒赫复苏等应连锁出现，避免孤立名场面。

**【冷媚、黎琳等次要线处理】**
公开资料中尚有冷媚、黎琳等名姓出现于人物表边缘：写卡时若无完整弧光，可作一次性情报源或场景NPC，**禁止用代称凑主要人物栏**；主要人物栏已列真名者优先。

**【齐天、麻等旧圣坐标】**
“麻”等旧圣第一人相关称谓、齐天等名，用于起源海/旧圣时代闪回。低阶只给传说句，高阶才允许残影对话。

**【战舰与手撕的标尺】**
养生主“手撕战舰”是六阶破坏力锚，对照图鉴地区级；羽化爆星感、幕天星域感、御道俯视星海，逐级升，不可让大宗师撕舰。

**【精神天眼与超感】**
王煊早期超感/精神天眼是开内景前的铺垫：长时间保持超感导致凡人开内景。写一、二阶可给“短暂超感过载”任务，不可直接送完整内景地。

**【蛇鹤八散手】**
钟家线输入的体术资源之一，体现财阀也能供养旧术碎片。与道家散手、佛门拳掌并列时，要写流派矛盾（刚猛/巧化/鹤蛇意象），禁止合成万能拳。

**【地仙泉与逝地交易】**
逝地不只打怪，还有类集市的残酷交易：寿命、记忆、真实物质、地仙泉。五、六阶支线适合“带一样东西进去换另一样出来”，出来时少一样感情或记忆。

**【落幕后的母宇宙】**
神话抽走后，旧土/新星进入科技主场回潮，但残留药田、失控图腾、假列仙诈骗仍在。逆行者火种使母宇宙仍可开低中阶任务，不必整图迁去中心宇宙。

**【中心宇宙的生存税】**
真圣道场收税方式是考核、站队、贡献造化气与战场出勤。八、九阶日常是会议与劫难，不是每天斩灾主。

**【鼎的伦理】**
本源鼎能续命/重生，于是产生“谁有资格入鼎”的政治。王煊入鼎是重伤后的无奈与资格并存；契约者若抢鼎，必引灾主级仇恨与规则反噬。

**【第15人】**
番外线索：唯一道边缘的编号感存在。具体身份公开资料不完全透明处写「不详」，保留探索钩子，禁止编造完整身世。

**【三子与后院烟火】**
王晔、王昕、王晖是终局烟火锚，证明彼岸不是永恒战场。涉及孩童线只写保护与日常，不写战力剥削。

**【互坑喜剧的正确剂量】**
老陈与王煊的互坑是减压阀，出现在生死战后的包扎、分赃、甩锅。高压归真段应减少耍贫，避免基调崩坏。

**【时间线锚点速查表】**
T0 实验班被阻；T1 灰血·开内景；T2 养生殿；T3 密地救人；T4 命土三药；T5 葱岭；T6 列仙回归；T7 落幕·光海；T8 六破；T9 归真·入鼎；T10 番外后院。选 T=选开局。

## 阶位切入点

> 阶位↔境界：一阶≈采气~内养~准宗师；二阶≈宗师；三阶≈大宗师；四阶≈人世间；五阶≈逍遥游；六阶≈养生主；七阶≈羽化仙；八阶≈幕天境（绝世~超绝世）；九阶≈御道境（异人~真圣）。世界顶点（**灾主**、**真王**、终局**王煊**、道主/道之外）为**超阶**：低阶切入一律**规避顶点·情报优先/条件性胜利**，禁止用“任务限制战力/暂时不在场=无威胁”矮化顶点。

**一阶（采气·内养·准宗师 · 旧土实验班与被阻船票）**

切入身份/时点：契约者以旧土大学旧术实验班学员、秘路外围临时工或移民中介跑腿身份切入；锚定**王煊毕业被阻无法去新星**前后数周。

初始事件：实验班公示“新星联合培养”名单，你的名字与**王煊**一样被卡在资格审查外；夜路遇灰血外围试探性绑架，必须选择报警、求助秘路，或跟**青木**的野路子勘探车离开市区。

开场白建议：「旧土的夜风带着沙和消毒水味。电子屏上，新星的虫洞航班又取消了一班。你的培养资格和那个叫王煊的同学一起被标成红色——有人说是财阀，有人说是前女友家族。巷口有人叫你的学号，手里不是录取通知书，是消过毒的刀。」

关键NPC立场：**王煊**（同班竞争/潜在同伴，执拗护短）；**青木（赵青空）**（秘路领路，收临时工）；**陈永杰**（高层，冷眼评估炮灰）；灰血外围打手（要活口或要恐吓效果）；大学导师（怕事、可卖情报）。

主线钩子/支线：主线=资格卡被谁做掉、灰血为何盯旧术班；支线=帮实验班偷运根法残页、护送一名准宗师级伤员出城、查清前女友家族与孙家关联的流言。

危险度/规避：低~中。规避灰血正式超凡杀手、财阀武装安保核心；超阶存在完全不在视野。

任务方向/奖励：旧术入门功法残卷、实验班编制、秘路外围徽章、小额真实物质粉末（微量）、人脉（青木线）。

**二阶（宗师 · 秘路编制与都市据点）**

切入身份/时点：秘路正式探勘员/养生殿外围护卫；锚定王煊**开启内景地初期、突袭灰血、建养生殿**阶段。

初始事件：你被派往一座废弃矿区“采普通药草”，实际坐标贴着灰血转运点；行动中目睹**王煊**以非常规状态击溃超凡边缘敌人，组织要求你交“内景地目击者报告”——写实则泄密，写假则被老陈看穿。

开场白建议：「矿灯照见岩壁上的血手印，不是野兽。耳机里老陈的声音懒洋洋：‘采药就采药，别问旁边那炉子为什么在响。’炉盖缝里漏出的雾气，让你的宗师意识第一次感到——自己可能只是报告书上的一行字。」

关键NPC立场：**陈永杰**（考核你是否嘴严）；**王煊**（需要可信外围）；**秦诚**（凡人视角的良心）；孙家代理人（收购/收购）；钟家跑腿（试探交易）。

主线钩子/支线：主线=灰血巢穴清剿的后勤与证物链；支线=养生殿选址争夺、宗师比武争资源配额、截获一份“命土矿图”伪件并鉴定。

危险度/规避：中。规避大宗师对决中心、养生炉失控区；禁止挑战任何列仙遗物自主激活现场。

任务方向/奖励：宗师圆满丹药（凡级）、道家散手残页复印件、养生殿临时通行证、小额天药残渣。

**三阶（大宗师 · 葱岭与新旧术舆论战）**

切入身份/时点：大宗师客卿、媒体特约“超凡观察员”或钟家安保顾问；锚定**葱岭大战、老陈重伤、新旧术争锋**。

初始事件：葱岭直播信号突然中断，你收到两份互相矛盾的委托——新术联盟要你证明“旧术恐怖主义”，秘路要你带出重伤的**陈永杰**；战场边缘出现可接引内景地的异常雾气。

开场白建议：「雪线上的炮火被‘能量光束’撕开一道白。有人喊新术胜利，有人喊旧术反杀。你的通讯器同时震动：一边是财阀报价，一边是王煊的短讯——只有坐标，没有谢谢。你知道，坐标尽头是炉，和可能把你一起炼进去的风险。」

关键NPC立场：**王煊**（要活口与真相）；**陈永杰**（重伤仍算计）；**钟晴**（资源换立场）；新术三大宗师残部（仇杀）；边境驻军指挥官（怕担责）。

主线钩子/支线：主线=护送老陈脱离与舆论叙事权；支线=回收战场真实晶石碎、阻止媒体深度伪造、谈判钟家临时停火。

危险度/规避：中高。规避人世间级对轰中心；严禁靠近自主激活的人世剑级至宝。

任务方向/奖励：大宗师秘药、半份内景接引法门（残）、钟家人情、战场特许打捞权。

**四阶（人世间 · 密地与命土采药）**

切入身份/时点：密地联合探险队成员/命土矿权技术员；锚定**新星密地救赵清菡、命土三天药**前后。

初始事件：密地导航失灵，黑狐潮逼近营地；**赵清菡**所在分队求救信号与“神之印记”异常读数同时出现——救人或抢药雾，队内投票撕裂。

开场白建议：「新星的广告牌还在卖基因年轻，密地里的雾却像上古的药。你的手持终端显示三株天药的理论坐标，也显示赵清菡生命体征在暴跌。王煊的声音从频道里压过来：‘先救人。药跑不了——人会。’你不确定他是在下令，还是在求你。」

关键NPC立场：**王煊**（救人为先）；**赵清菡**（关键被救者/印记载体）；密地地仙残存（交易或猎食）；**青木**（舰修保命）；财阀监军（要药不要人）。

主线钩子/支线：主线=密地救援与三天药归属；支线=鉴定伪天药、关闭一座失控图腾、与逝地商人交换地仙泉线索。

危险度/规避：高。规避命土深层未知生物与药雾反噬；超阶不直接下场但遗物可能误触。

任务方向/奖励：人世间段位稳固资源、天药残枝、密地图腾碎片、赵清菡线好感/印记观察权（非控制）。

**五阶（逍遥游 · 列仙回光与精神神游）**

切入身份/时点：地仙级神游者、列仙遗物鉴定人；锚定**列仙回归初期、妖魔猎杀启动**。

初始事件：你神游太虚时撞见**方雨竹**元神投影清扫一队绝世以下猎手；她给你一个选择——立刻退出并忘掉坐标，或帮她送一件不能沾血的信物给旧土某人（王煊）。

开场白建议：「元神离开肉身的第三息，你看见红与白两道光在星尘里交手。女方士看都不看你，只丢下一句：记住炉的温度，别记住我的脸。你低头，掌心多了一枚温热的竹片，竹片上写着旧土的门牌号。」

关键NPC立场：**方雨竹**（考校嘴与胆）；**燕清妍**（可能截胡信物）；**姜清瑶**（剑意警戒外来神游）；**王煊**（收信人）；商毅外围信徒（抢幡抢剑）。

主线钩子/支线：主线=列仙回光下的信物投递与猎杀名单；支线=鉴定逍遥舟残速数据、阻止神游者集体走火入魔、交易真实物质配额。

危险度/规避：高。规避超绝世正面对决；**超阶巅峰：存在·情报优先**——真圣投影仅可观察。

任务方向/奖励：逍遥游稳境资源、神游护符、列仙情报残卷、一次内景地“访客时限”。

**六阶（养生主 · 落幕前夜与养生炉风暴）**

切入身份/时点：养生主温养者、至宝护卫编制；锚定**超凡落幕前夜、养生炉救姜清瑶/争渡准备**。

初始事件：炉中仙雾外泄导致整座基地“第几生”紊乱；**姜清瑶**被放入炉中续命，冲霄殿使者与王煊同时到达——你必须决定炉的控制密钥交给谁。

开场白建议：「炉壁上的经文像活的。剑仙的呼吸时有时无，真圣使者的笑容很客气，王煊的眼睛很红。有人告诉你：交密钥，保你下半生；有人告诉你：交密钥，保她还能醒。你的手按在符文上，第一次明白养生主为什么不宜多战——因为一战就动因果。」

关键NPC立场：**王煊**（要人要炉）；**姜清瑶**（无声的账单）；冲霄殿使者（收编）；**陈永杰**（建议你别当英雄）；基地凡人技师（怕被灭口）。

主线钩子/支线：主线=炉控制权与落幕撤离名单；支线=稳固“第N生”不被打回、转移一批将枯死的天药主根、伪造一份已登船名单掩护逆行者。

危险度/规避：极高。规避真圣正式出手；炉失控=团灭级。

任务方向/奖励：养生主破限辅助（一次性）、炉中仙雾瓶、撤离船票（假/真）、剑仙因果线线索。

**七阶（羽化仙 · 光海启航与原始仙体）**

切入身份/时点：羽化仙/准绝世乘客或光海引航员；锚定**超凡光海迁徙、王煊保留原始仙体羽化**前后。

初始事件：光海渡口配额不足，**古今**等人的席位被挤；王煊要你在“抢席”和“留下点燃母宇宙火种”之间做证人——你的羽化劫将在渡口当众降下。

开场白建议：「光海不是海，是把神话抽走时留下的伤口。有人哭着登船，有人笑着留下。王煊把一张名单拍在你胸口：这些人必须走。你的仙骨已经在响——成仙劫不认配额，只认你是否敢在众目睽睽下碎或不碎。」

关键NPC立场：**王煊**（送人的逆行者）；**方雨竹/姜清瑶/燕清妍**（已在对岸或渡中，远程施压）；**赵清菡**（沉睡中，需确认护持）；**钟晴**（家族席位谈判）；渡口执法列仙（冷规则）。

主线钩子/支线：主线=光海配额与原始仙体见证；支线=护送沉睡舱至混沌洞、截获假船票诈骗团、收集落幕时代最后一株可用天药。

危险度/规避：极高。渡口规则杀；**超阶：情报优先**——道场真圣不可硬刚。

任务方向/奖励：羽化稳劫资源、原始仙体观察笔记（珍贵）、光海对岸联络符、一次真圣道场推荐信（有条件）。

**八阶（幕天境 · 超绝世与中心宇宙道场）**

切入身份/时点：绝世/超绝世客卿、无忧宫或出世山外门执事；锚定王煊**全领域六破中段、元神圣物散落因果**。

初始事件：沙漏（时空权柄）与阵图在两大道场间引发礼仪级冲突；**方雨竹**与**姜清瑶**各执一词，要你以“中立超绝世”身份做七日公证——公证失败则两宫开战，成功则你被两边记恨。

开场白建议：「幕天之上没有玻璃窗，只有规则的反光。你看见沙漏里的花开了一瓣，阵图上的线亮了一寸。有人说这是王煊的彩礼，有人说这是战争借口。七日之期，你的元神涅槃次数还剩几次？」

关键NPC立场：**方雨竹**（要公证公平）；**姜清瑶**（要公证不偏）；**燕清妍**（看戏或搅局）；**王煊**（远程甩锅式信任）；道场老真圣（考校你是否可收编）。

主线钩子/支线：主线=元神圣物公证与道场冷战；支线=猎杀假冒超绝世骗子、进入一次起源海边缘取样、阻止商毅余孽启动人世剑残影。

危险度/规避：贴近顶点。超绝世互斩可毁星域表面；**超阶灾主/真王：存在·情报优先/条件性胜利**。

任务方向/奖励：幕天境破限辅助、圣物道韵残片、道场客卿位、王煊线“六破领域”见闻（不可复制）。

**九阶（御道境 · 真圣、归真与灾主前夜）**

切入身份/时点：异人/真圣级行走者、归真之地边缘守关人；锚定**真王之路、归真之地开战前夜**。

初始事件：古关开放半扇，灾主气息外泄导致一座小世界法则崩溃；**王煊**以真王姿态要你送一份“可入鼎者名单”的反提案——你是送自己人去填，还是说服他改用本源鼎规则之外的第三条路（条件性胜利）。

开场白建议：「归真之地的风不是风，是未写完的符文。你看见王煊的影子比他的人先到，影子里有炉、有竹、有剑、有红衣，也有旧土的沙。他问你：谁可以死，谁必须活。你第一次意识到，御道不是驾驭万道，是给万道写讣告。」

关键NPC立场：**王煊**（真王/将成灾主，要结果）；父母**王泽盛、姜芸**（枯寂与家国）；**王御圣**（兄长压力）；灾主使者（招降或猎杀）；**赵清菡**（团聚前的最后变数）。

主线钩子/支线：主线=归真古关权限与入鼎伦理；支线=拦截假灾主投影、保全一座将崩小世界人口、收集第15人相关流言（番外向）。

危险度/规避：贴近顶点/极高。**超阶巅峰（灾主群、终局王煊、本源鼎）：存在·情报优先/条件性胜利**；禁止用“暂时沉睡/规则压制”把顶点写成废战力。硬刚=任务世界级团灭。

任务方向/奖励：真圣劫辅助资源、归真符文拓片、本源鼎观察权（一次性）、番外向“自斩/复活”伦理选项解锁（叙事层，非数值无敌）。
