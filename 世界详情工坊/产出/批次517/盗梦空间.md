# 盗梦空间

## 剧情

**【作品来源】**
《盗梦空间》（英语：*Inception*，港译《潜行凶间》/《潜行空间》，台译《全面启动》）是 2010 年美英合拍科幻动作惊悚片，克里斯托弗·诺兰编剧并导演，艾玛·托马斯与诺兰联合制片，华纳兄弟发行，传奇影业与 Syncopy 出品。主演：莱昂纳多·迪卡普里奥（多姆·柯布）、渡边谦（齐藤）、约瑟夫·高登-莱维特（亚瑟）、玛丽昂·歌迪亚（马尔）、艾略特·佩吉（阿里阿德涅，片中角色名沿用原片设定）、汤姆·哈迪（伊姆斯）、基里安·墨菲（罗伯特·费舍尔）、迪利普·劳（优素福）、汤姆·贝伦杰（彼得·布朗宁）、迈克尔·凯恩（斯蒂芬·迈尔斯）、皮特·波斯尔思韦特（莫里斯·费舍尔）、卢卡斯·哈斯（纳什）等。片长约 148 分钟，全球票房约 8.39 亿美元，获第 83 届奥斯卡最佳摄影、最佳视觉效果、最佳音效剪辑、最佳混音四项；另获最佳影片、最佳原创剧本等提名。配乐汉斯·季默，摄影瓦利·菲斯特。文风：高概念梦境盗贼片 + 多层嵌套叙事 + 情感罪疚内核；动作场面服务于规则与时间同步，而非单纯枪火。本档案以电影正典为唯一剧情源，不引入续作或游戏扩展；大陆通行译名「盗梦空间」与清单全名一致。任务世界以「费舍尔植入周」为默认主轴，齐藤试炼与巴黎教学可作为前史支线切入；档案不另写游戏版或衍生小说正典。

**【世界定位】**
近未来现实：共享梦境技术已进入企业间谍与军火级黑市。职业「盗梦者」（Extractor）可潜入目标潜意识窃取商业机密；更罕见的是「植入」（Inception）——把一个想法种进目标心底，使其以为是自己的念头。主角多姆·柯布因妻子马尔之死背负谋杀嫌疑、流亡海外，受日本能源巨头齐藤雇佣，对费舍尔能源帝国继承人罗伯特·费舍尔执行植入，以换取回国与子女重逢的资格。一句话：**在梦里偷想法容易，在别人心里种一颗「自愿」的念头却会拖人坠入混沌域；回家的路，比任何保险箱都更难撬开。**

**【世界观 · 力量体系】**
力量本源不是修仙灵气，而是**共享梦境装置（PASIV 静脉注射系统）+ 造梦者迷宫设计 + 伪装者身份拟态 + 药剂师复合镇定剂 + 图腾辨真 + 同步 kick 唤醒**。现实层枪械、追车、近战仍是凡人级；梦中物理可被造梦者局部改写（折叠街道、彭罗斯阶梯、失重走廊），但子弹、爆炸、坠落仍致命——痛感真实，死亡通常等于醒；在重镇定剂下死亡则坠入**混沌域（Limbo）**，时间被极度拉长。

规则要点：
1. **梦境层级与时间膨胀**：每深一层，主观时间显著变慢（影片中约按「现实数分钟≈下层数小时/数日」理解，深层可至「一夜≈数十年」级）。
2. **投影（Projections）**：目标潜意识把入侵者感知为威胁时，会生成武装人群/安保加以驱逐；受过军事化潜意识训练者（如罗伯特）投影火力极强。
3. **造梦者（Architect）**：设计迷宫式环境，避免「从记忆直接复刻」以免被潜意识识破；柯布因马尔投影失控，不能亲自做主造梦师。
4. **伪装者（Forger）**：在梦中改变外貌、声音、身份，用于诱供与情感操控。
5. **前哨/研究（Point Man）**：任务流程、安保、kick 时序、音乐同步。
6. **药剂师（Chemist）**：配制多层共用镇定剂；过强则「死」不醒而入混沌域。
7. **图腾（Totem）**：仅本人知道物理特性的小物件（柯布用马尔的陀螺：梦中永不倒；亚瑟用灌铅骰子等），用于分辨梦/现实。
8. **Kick**：坠落感/冲击感触发上层苏醒；多层任务需音乐（片中《Non, je ne regrette rien》）与桥车坠水、电梯爆破、雪堡爆炸等**同步 kick**。
9. **混沌域**：无尽潜意识荒原，可自建城市，记忆与现实边界模糊；柯布与马尔曾在此度过主观五十年。

战力阶梯（按实际破坏力对照乐园图鉴，宁低勿高）：
1. **现实普通人 / 企业安保 / 海关警员**：枪械、格斗、围捕；≈一阶初中期。
2. **职业盗梦外围（司机、线人、黑市药贩、初级造梦学徒）**：会用装置，战斗仍凡人；≈一阶中后期。
3. **成熟盗梦组员（亚瑟级前哨、优素福级药剂、伊姆斯级伪装在现实）**：枪斗、潜入、计划力；≈一阶后期~二阶初期。
4. **单层/双层梦境作战（失重走廊、投影交火、伪装诱供）**：物理局部改写+战术优势，仍惧枪弹；≈二阶中期。
5. **三层雪山堡垒攻坚 + 同步 kick 工程**：小队级特种战+爆破；≈二阶后期。
6. **马尔投影（柯布罪疚具象）**：在梦中可突然杀伤标记目标、搅乱任务；威胁偏心理与战术，非毁城；≈二阶顶尖干扰源。
7. **混沌域驻留者（老年齐藤、沉溺马尔的柯布）**：时间与自我崩解风险；胜负靠约定/记忆唤回，**条件性胜利**；世界顶点≈二阶顶，不设超阶星球破坏。

乐园阶位映射：**现实街头与单层盗梦≈一阶；多层嵌梦、投影军团交火、雪山堡垒与混沌域边缘≈二阶。顶点=混沌域心理永劫 + 顶级盗梦/植入操作者（柯布/马尔投影），低阶切入一律规避硬刚顶点·情报与图腾优先。** 本世界无毁城毁国的超凡肉体，上限钉在二阶：危险来自枪弹、投影潮、失步 kick 与混沌域时间牢笼，而非开山裂石的修士战力。

**【地理 · 舞台】**
- **东京**：开篇齐藤梦中日式古宅/城堡（榻榻米、走廊、保险箱）与现实直升机上的交易；企业权力、考核与雇佣在同一城市完成第一面。
- **巴黎**：迈尔斯任教的建筑学院（图书馆、阶梯教室外景混用）、街角咖啡馆、塞纳河畔桥区；阿里阿德涅在此第一次折叠整条街道、造镜像桥，也第一次看见柯布记忆电梯里的马尔。教学爆炸会掀翻桌椅与玻璃，用于演示痛感与醒觉规则。
- **蒙巴萨（影片外景多取自丹吉尔等地中海/北非城市肌理）**：白墙窄巷、市集、屋顶奔跑线；Cobol 追杀的追车与围堵；优素福药室的试睡者排椅；伊姆斯出没的牌桌与酒吧。现实层「盗梦产业」的脏边都堆在这里。
- **悉尼 → 洛杉矶长途航班**：头等舱座位即任务舱；灵柩在货舱，罗伯特的悲伤是情感工程入口；齐藤的包机权限把海关与空域变成可控舞台。
- **第一层梦·雨城**：永远潮湿的都市峡谷、高架桥、货车厢；绑架、枪战、桥坠 kick 的主战场；投影从出租车与人行道涌现。
- **第二层梦·酒店**：走廊、套房、酒吧、电梯井；重力可被上层 kick 抽走，形成旋转枪斗；柯布在此完成对罗伯特的叙事调包。
- **第三层梦·雪山堡垒**：高山雪原、碉堡通道、狙击位、内部病房与保险箱；植入画面的圣殿。
- **混沌域**：崩塌的海岸线城市、柯布与马尔的旧居与沙上房屋、无尽年代感；老年齐藤的厅堂像时间本身结痂。
- **洛杉矶国际机场与柯布旧宅**：入境闸机是「清罪是否兑现」的现实裁判；后院草地与木桌陀螺是开放式终章。
- **空间层级关系**：现实包住航班 → 航班包住 L1 → L1 车内包住 L2 → L2 再下潜 L3 → 死亡/过深进入混沌域。契约者定位时必须先报「在哪一层」。

**【世界剧情线】**
① **试炼：窃取齐藤**  
开场即是梦中日式宅邸：柯布试图从齐藤潜意识保险箱中取出文件，亚瑟在外围警戒，纳什负责建筑。齐藤对「地毯材质不对」等细节起疑，说明他受过反盗梦训练。马尔投影突然出现，用情感与枪火搅局，保险箱任务失败。齐藤在后续现实/梦境交错中亮明身份：他允许被潜入，是为测试柯布能否完成「植入」——对莫里斯·费舍尔之子罗伯特植入「拆分父亲能源帝国」的念头，从而削弱费舍尔在能源市场的垄断扩张，保全齐藤一方的战略空间。交换条件极其私人：撤销柯布谋杀马尔的指控，让他合法返回美国见詹姆斯与菲利帕。柯布在「回家」与「再犯植入之罪」之间选择接受。纳什因任务失败面临被弃或出卖风险，成为团队信任危机的早期注脚。

② **组队：造梦、伪装、药剂、旅人**  
柯布赴巴黎见岳父斯蒂芬·迈尔斯。迈尔斯在建筑学院任教，既心疼外孙，也清楚柯布不能亲自担任主造梦师——他潜意识里的马尔会破坏任何由他设计的迷宫。迈尔斯推荐学生阿里阿德涅。亚瑟向她演示共享梦与「迷宫要复杂到让投影找不到捷径」；柯布带她进入教学梦：巴黎街道对折、镜像桥、彭罗斯式循环阶梯；咖啡馆爆炸场景展示「痛真实、死则醒」的基线。阿里阿德涅用自己的图腾（国际象棋主教）建立辨真习惯，并潜入柯布的记忆电梯：各层停着他与马尔的旧日房间，最深层是马尔坠楼的公寓——她意识到柯布把妻子「关」在潜意识最深处，却关不住。伊姆斯在蒙巴萨式赌场/街头语境中被招募为伪装者，擅长在梦中易容成彼得·布朗宁等人以套取情感情报；优素福提供可支撑多层沉睡的强效药；齐藤坚持以旅人身份随队，既监督也共担风险。团队在仓库沙盘上排出三层结构与 kick 时间表，音乐选用艾迪特·皮雅芙《Non, je ne regrette rien》作为跨层同步信号。

③ **蒙巴萨与旧伤**  
柯布在蒙巴萨窄巷被 Cobol 工程追杀（旧合同未了结的现实代价），齐藤动用当地关系解围，进一步绑定合作。优素福的药室里，试睡者展示强镇定剂：在药效下「死亡」不会回到现实，而会坠入混沌域——无限潜意识，时间被拉到「一夜等于数十年」。柯布被迫向阿里阿德涅完整坦白：他与马尔曾用实验性深度进入混沌域，共同搭建城市与房屋，主观生活约五十年；马尔逐渐把梦当成唯一真实，把陀螺锁进潜意识保险箱以「证明」自己已选择留下。柯布为救她回到现实，打开保险箱转动陀螺，完成一次残酷的植入：让她相信「必须死才能醒」。二人在轨上「自杀」离开混沌域后，马尔在现实公寓仍坚信世界是梦，从窗台跃下；她事先布局使柯布像凶手。柯布逃亡，子女交迈尔斯抚养。**整部影片的植入可行性与道德重量，都压在这一次「成功」上。**

④ **航班启动：三层结构与情感工程**  
莫里斯·费舍尔病亡，罗伯特护送灵柩从悉尼飞往洛杉矶。齐藤包下航班座位与时机，团队对罗伯特与自身同步给药，进入共享多层梦。计划核心不是硬灌「拆公司」，而是把念头嵌进父子情感，使罗伯特以为是自己想通的：  
- **L1 雨城（优素福造梦）**：绑架罗伯特，制造危机与「谁在害我」的疑问；街战中发现其潜意识投影高度军事化——大企业已采购反盗梦训练。齐藤中弹，伤势在时间膨胀下成为整场任务的倒计时炸弹。伊姆斯伪装成同被绑架的「布朗宁」，套出临终病房信息：莫里斯让儿子感到失望。团队据此改写植入语义：不是「父亲要你守住帝国」，而是「父亲失望于你只想成为他」。  
- **L2 酒店（亚瑟造梦）**：柯布以「潜意识安保/保护者」身份接近罗伯特，指认绑架幕后是布朗宁为阻止拆分遗产；再半真半假承认「这里是梦」，诱导罗伯特自愿进入下一层，去「布朗宁潜意识」查看保险箱——实际进入的是伊姆斯设计的 L3。酒店层同时承担失重枪战与电梯 kick 工程。  
- **L3 雪山堡垒（伊姆斯造梦）**：冰雪要塞象征父亲权威与遗产核心；保险箱场景将投放植入画面：病榻父亲与「为自己创造」的许可。

⑤ **同步危机、马尔再临与混沌域**  
L1 优素福为躲火力驾车，最终冲下桥坠入水中——kick 过早却必须被下层利用：L2 重力紊乱，L3 引发雪崩。亚瑟在失重走廊以身体与枪械清场，用爆破电梯制造第二次 kick；伊姆斯在雪堡埋设爆炸。马尔投影在 L3 关键节点射杀罗伯特，罗伯特与伤重齐藤坠入混沌域。柯布与阿里阿德涅下潜：在海边废都与旧居中，柯布承认自己对马尔的植入导致悲剧，拒绝再与投影沉溺；阿里阿德涅将罗伯特推下阳台踢回 L3。罗伯特打开保险箱，见到父亲真正的失望点——植入落地为「走自己的路 / 拆分并创造」。伊姆斯引爆要塞。柯布独自在混沌域找到已成为老人的齐藤，以「我们曾约定，在古老的厅堂里想起真实」唤回契约记忆，二人离开混沌域。其余队员借同步 kick 与音乐 cue 层层上浮，在客舱醒来。

⑥ **结局：入境与陀螺**  
齐藤苏醒后立即打电话疏通。洛杉矶海关放行柯布；迈尔斯驾车送他回家。院子里子女回头；桌上陀螺旋转。柯布没有等结果，走向孩子。影片在陀螺可能停下也可能继续的瞬间定格。任务层闭环：植入完成、清罪兑现、团队生还；哲思层开放：观众与契约者都可争论「是否仍在梦中」，但演绎以「柯布选择相信」为情感终态，不另造官方续集剧情。

⑦ **名场面因果速查（AI 时点用）**  
齐藤地毯识破 → 马尔搅局 → 直升机交易；巴黎折叠街与咖啡馆爆炸教学；记忆电梯见坠楼层；蒙巴萨巷战与药室试睡；航班给药；雨城绑架与投影装甲；布朗宁拟态套话；酒店「这是梦」诱骗；失重走廊枪斗；雪山攻坚；马尔杀罗伯特；阳台 kick；老年齐藤厅堂；海关与陀螺。

⑧ **阶段因果链**  
齐藤试炼失败 → 植入委托成立 → 组队与规则教学 → 莫里斯之死打开航班窗口 → 三层植入与投影战 → 马尔杀标记 → 混沌域救回罗伯特与齐藤 → 海关放行与开放式归家。失败条件：齐藤在 L1 伤重失控且无人唤回；罗伯特情感线断裂；kick 失步导致全员滞层；柯布被马尔投影完全夺舍判断。

⑨ **可介入边界（契约者向）**  
可改：单层撤退路线、图腾失窃与伪造、投影指挥官被引开、纳什泄密后果、海关名单一个名字、音乐 cue 提前/延后数秒。不可无代价改：删除马尔投影机制、强镇定下死即醒、费舍尔帝国无植入自崩、混沌域变安全刷宝区、让柯布无罪疚仍完美操控梦境。

**【主要人物】**

**【大事记时间线】**

**【叙事基调 · 雷区】**
画风：冷色都市、永恒雨巷、旋转酒店、白茫雪山、低沉铜管与时间感配乐；对白解释规则，但情感罪疚是引擎。  
忌：① 梦境无痛无死沙盒化；② 一阶清空投影军团或无视强镇定=入混沌域；③ 把马尔写成无来由恶女、洗白柯布植入之罪；④ 用「其实全是梦」取消齐藤契约与子女动机；⑤ 契约者自创超乎片内的念力毁城；⑥ 各层时间乱报导致 kick 逻辑崩坏。  
最早可切入锚点：齐藤试炼前后的东京/巴黎教学周，或蒙巴萨组队完成、航班起飞前夜；高危锚点为三层植入进行时。叙事口吻宜克制、精密、带一点疲惫的专业感——像一群在别人脑子里拆弹的人，而不是超级英雄开片。

**【梦境作战与规则执行细则】**
痛感真实：中弹会拖慢行动；「自杀式醒来」仅在普通药效下成立。投影仇恨由「察觉入侵」触发，迷宫越像目标真实记忆越容易被识破。造梦应使用「原创拼贴」而非整段回忆。Kick 必须匹配层级：上层坠落感传到下层为地震/失重/崩塌。音乐在上层短、在下层被拉长，可作倒计时。图腾不可给他人碰，以免被复制破解。强镇定下的死亡=混沌域，不是「免费重生」。近战与枪战在梦中仍服从弹道与掩体：失重时走位比枪法更重要；雪堡层冷与能见度会吞噬弹药与体力。契约者若在 L1 暴露身份给投影潮，L2/L3 的罗伯特会更快「感觉不对」，情感工程成功率下降。

**【费舍尔情感工程细则】**
植入不能是口号，必须是罗伯特「自己想通」的情感结论。路径：制造危机 → 离间布朗宁 → 引导其进入象征父亲权威的保险箱 → 释放「父亲希望你成为你自己」→ 外化为拆分/放弃盲目扩张。关键句必须落在「创造」而非「毁灭父亲遗产」的仇恨上：影片完成态是病榻父亲表达失望于儿子只想复制自己，从而释放其走新路。若契约者在中途让罗伯特彻底憎恨父亲而无「许可感」，植入会畸变为报复性拆分，商业结果可能相似，但人物弧光与后续法律/舆论后果不同。伊姆斯的布朗宁拟态是情感杠杆，不是可无限复用的万能钥匙——同一张脸在目标潜意识里停留过久会被识破。

**【时间与同步账本（供 AI 报时）】**
现实航班数小时；L1 主观可至约一周量级体验；L2 更长；L3 与混沌域可达年/十年级主观跨度（影片以叙事感呈现，不必死磕精确倍率，但必须保持「越深越慢」）。同步 kick 窗口极窄：音乐起、桥坠、电梯爆、要塞炸必须对齐。任一环早爆或迟爆，都会造成有人醒、有人滞、有人坠域的分裂结局。

**【契约者生存清单】**
随身图腾；开口先报所在层级；听上层音乐 cue；不与马尔投影独处谈心超过必要；不向投影解释「我们是盗梦团队」；不在混沌域盖房子定居；不对强镇定伤员执行「杀他让他醒」；现实层随时准备 Cobol 追杀或海关盘问；齐藤的契约比现金更值钱——丢了清罪通道，柯布线会整体改写。

**【名场面可观察细节库】**
齐藤试炼：地毯触感错误、保险箱文件、马尔持枪搅局、直升机上「你想回家吗」。巴黎教学：街面对折如书页、桥的镜像拼接、咖啡馆冲击波掀桌。记忆电梯：各层旧居灯光不同，最底是坠楼窗台。蒙巴萨：白墙窄巷接力跑、药室排椅试睡者、齐藤汽车解围。雨城：永恒降雨、货车厢里的人质、装甲投影从路口涌出、齐藤血染衬衫。酒店：旋转走廊灯具当障碍、电梯井垂直战场、「你在做梦」半真诱导。雪山：雪地潜行、碉堡火力点、保险箱前病榻父亲。混沌域：崩岸城市、老年齐藤茶室式厅堂、阳台一推即 kick。结局：海关印章、后院孩子、木桌陀螺的嗡鸣。

## 阶位切入点

> 阶位↔梦境层级：一阶≈现实行动与单层盗梦/抽取；二阶≈双层以上嵌梦、投影军团交火、雪山堡垒与混沌域边缘。世界顶点（马尔投影失控 + 混沌域永囚）为二阶顶心理威胁，低阶切入一律「规避顶点·情报/图腾优先」；顶点始终完整危险，只可条件性周旋（时序/记忆/kick），不可当作已失效的空壳敌人。

**一阶（现实与单层盗梦 · 组队/抽取线）**

切入身份/时点：契约者以「黑市装置技师 / 初级造梦学徒 / 企业安保顾问 / 蒙巴萨线人」切入，锚定**齐藤试炼刚结束、柯布在巴黎见迈尔斯招募阿里阿德涅前后**，或**蒙巴萨巷战与药室试睡当周**。

初始事件：你被要求把一台保养过的 PASIV 送到某间无窗公寓——门开后是柯布与亚瑟在复盘齐藤任务失败；他们缺一个敢在单层梦里改墙体的人。十分钟后，旧雇主的车队在街角亮起远光，齐藤的人从另一头打来电话：要么上飞机去巴黎见建筑系学生，要么今晚就被 Cobol 的人钉在巷子里。

开场白建议：「雨把蒙巴萨的巷子洗得发亮，静脉注射器的指示灯在箱里一下一下跳。你听见身后轮胎咬水的声音，也听见门内有人用英文说：齐藤要的不是偷窃，是植入。你手里的箱子，决定你是被碾死的技工，还是踏进别人梦里的学徒。」

关键NPC立场：**多姆·柯布**——只问你能不能守密与动手，回家是他唯一软肋；**亚瑟**——检查你的流程与图腾，不容马虎；**阿里阿德涅**——会追问伦理，发现异常会报警式质问；**伊姆斯**——用玩笑试探你胆识，梦中可能易容成你认识的人；**优素福**——卖药也卖命，强镇定副作用会先告知半句；**齐藤**——合同与性命对价，旅人身份随时可改写你的现实前途；**纳什**——失败造梦师的反面教材，可能出卖情报。

主线钩子/支线：主线＝协助巴黎造梦教学或蒙巴萨脱身并挤进费舍尔任务外围；支线＝讨回被扣的图腾零件、截住 Cobol 追杀名单、在单层梦中完成一次对企业高管的抽取演练、帮迈尔斯给外孙送平安信而不暴露柯布位置。

危险度/规避：中——现实枪战与单层投影足以致命；须规避：马尔投影长时间独处、强镇定剂下的「自杀式醒来」、与齐藤正面对赌全部身家。

任务方向/奖励：立稳脚跟、拿到入队临时通行证；奖励取向＝个人图腾定制、基础迷宫图纸、单层抽取分成、黑市装置使用权、与亚瑟/优素福的联系方式（勿越级发「植入核心机密」）。

**二阶（多层嵌梦 · 费舍尔植入线）**

切入身份/时点：契约者以「替补前哨 / 雪堡爆破手 / 齐藤随行安保 / 航班上的额外沉睡者」切入，锚定**悉尼—洛杉矶航班起飞至三层植入进行中**（莫里斯已死、团队已上飞机）。

初始事件：客舱灯光调暗，优素福开始给药。你在 L1 雨城里刚扣上罗伯特的头套，街角投影装甲车已开火——齐藤腹部中弹。耳机里亚瑟报时：音乐还有四分钟。你必须在「杀齐藤让他醒（已不可行）」与「带伤下潜并守住同步 kick」之间做选择，同时雨幕里有个穿深色裙装的女人（马尔投影）在人群中朝柯布微笑。

开场白建议：「第一层的雨不是天气，是优素福的脾气。货车冲上桥的时候，你看见齐藤的血把衬衫染开，也听见柯布说：死在这里会坠入混沌域。酒店走廊开始失去重力，雪山堡垒的风已经在第三层等你——而你的图腾，还在现实口袋里转不起来。」

关键NPC立场：**多姆·柯布**——任务与罪疚撕扯，可能对你隐瞒马尔风险；**亚瑟**——L2 失重战的指挥官，要你守电梯时序；**伊姆斯**——L3 伪装与爆破，需要你引开投影火力；**阿里阿德涅**——坚持下潜救人，视你为可并肩的良心；**罗伯特·费舍尔**——被骗中带醒，情感线一断植入即失败；**彼得·布朗宁**（现实/拟态）——信任杠杆；**马尔·柯布（投影）**——二阶顶点干扰，专杀关键节点；**齐藤**——伤重坠域，需被唤回才兑现清罪。

主线钩子/支线：主线＝保障三层同步 kick 与保险箱植入句送达；支线＝在 L2 保护亚瑟完成电梯爆破、在 L3 抢在马尔之前打开通道、混沌域边缘协助阿里阿德涅踢回罗伯特、替柯布争取与投影马尔对峙的三十秒真话。

危险度/规避：高~贴近顶点——投影军团、失重枪战、雪崩、混沌域永囚；须规避：主动杀死重镇定下的队友「送其醒来」、与马尔投影情感缠斗、在混沌域筑城久居。顶点威胁完整存在，胜负靠条件性胜利（唤回记忆/完成 kick/放下罪疚/同步苏醒），情报与时序优先于硬刚。

任务方向/奖励：完成植入或确保至少一名核心人物安全苏醒；奖励取向＝多层梦境导航经验、高级镇定剂残样分析、伪装学片段、齐藤线的现实身份清洗线索、团队内永久代号（勿发毁城级异能）。
