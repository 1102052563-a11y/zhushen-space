# 问题儿童来自异世界（轻小说）
<!--meta lib=主库 tiers=一、二、三、四、五、六、七-->

## 剧情

**【作品来源】**
《问题儿童都来自异世界？》（問題児たちが異世界から来るそうですよ？）龙之湖太郎著，插画天之有（第一部）/Momoco（第二部），角川 Sneaker 文库 2011 年起刊行。第一部 11 卷完，另有短篇；第二部《问题儿童的最终考验》系列后续。2013 年电视动画（Diomedéa，全 10 话+OVA）主改编前几卷。曾获店员最爱轻小说相关名次。文风：箱庭神话狂欢、恩赐游戏、共同体重整、吐槽与热血并存。本条目锚定**轻小说本篇（第一部主线）**。

**【世界定位】**
现实世界三名「问题儿童」——逆廻十六夜、春日部耀雅、久远飞鸟——被箱庭贵族黑兔的邀请函召入箱庭世界，加入没落共同体 **No Name**，为夺回被魔王夺走的名字与旗帜，挑战恩赐游戏与魔王，重建阿卡迪亚荣光。

**【世界观 · 力量体系】**
箱庭：神佛修罗为娱乐创造的舞台，面积近恒星表面，有「世界尽头」。外门位数=层级：位数越小越近中心、战力越高；五位数中层，四位数以上上层。境界门连接外门，收费由地域支配者定。
恩赐（Gift）：超常能力/ equip。恩赐游戏：主办定规则与奖品，契约文件（Geass Roll）载明条件；原则须可通关。主办者权限可强制开赛。审判权限（月兔）可审议决议。
灵格/神格/系统树/纯种/混血/最强种（神灵·星灵·龙种纯血）。魔王=滥用主办者权限者。太阳/月之主权：星辰权柄，可召星兽、影响大天幕。
人类最终考验：封闭世界、绝对恶阿吉·达卡哈、衰微之风、永动机 Last Embryo 等灭绝级试炼显化。
映射（宁低勿高）：外门外围挑战者≈一~二阶；六位数共同体主力≈三阶；五位数/阶层战≈四~五阶；四位数上层与准魔王≈六阶；魔王与最强种片段≈七阶。顶点=人类最终考验级与上层神群，条件性胜利。乐园阶位映射：一~三≈下外门与重建期；四~五≈中层争霸；六~七≈魔王战与主权级。

**【地理 · 舞台】**
箱庭东区下外门 2105380（No Name 根据地）；北区煌焰之都（Salamandra）；南区 Underwood 与龙角鹫狮子；千之眼商业网；各外门街区、收获祭会场、魔王游戏场域。

**【世界剧情线】**
①召唤：十六夜（全权力能者/疑似无尽恩赐）、耀雅（星杯巫女）、飞鸟（孙悟空灵格相关）入箱庭，黑兔接待，入 No Name 废园。②初期游戏：与周边共同体周旋，夺地域支配者等，重建领地与人气。③火龙诞生祭与 Salamandra 线：北区火龙族政治、继承与魔王黑死斑等危机，No Name 协力。④南区收获祭：巨龙/巨人与殿下势力，阶层支配者考验。⑤绝对恶阿吉·达卡哈：二百年前重创阿卡迪亚与月兔的人类最终考验再临；多共同体联战；旗与名夺回关键。⑥联盟化：与 Will o'wisp、Perseus、六伤等结盟升格。⑦上层伏笔：万圣节女王、白夜叉、七天大圣、Ouroboros 魔王联盟、太阳主权分配、Last Embryo 等指向第二部。第一部以 No Name 夺回关键象征（旗帜/名）与打倒绝对恶段落为情感高潮，后续挑战未完。

**【主要人物】**
- 逆廻十六夜｜问题儿童｜慵懒好战｜全权/不明恩赐｜从无聊到守护共同体。
- 春日部耀雅｜巫女｜认真天然｜星杯｜治愈与裁定向。
- 久远飞鸟｜中国来客｜大胃王好胜｜齐天灵格｜武力担当。
- 黑兔｜箱庭贵族月兔｜女仆装毒舌｜审判相关｜召唤者与向导。
- 白夜叉｜千之眼干部/前魔王｜佛门｜主权持有｜秩序侧巨头。
- 金丝雀（历史）｜No Name 创建者相关｜人类史打倒魔王传说。
- 珊朵拉等 Salamandra；杰克等 Will o'wisp；帕修斯末裔；殿下/蕾蒂西亚相关；佩丝特等敌对游戏主办；阿吉·达卡哈。

**【势力图谱】**
No Name；千之眼；Salamandra；龙角鹫狮子/六伤；Will o'wisp；Perseus；Queen Halloween；七天大圣残响；Ouroboros；魔王诸类。

**【贵重物品】**
恩赐卡（拉普拉斯纸片）；契约文件；星之主权/太阳主权/月之主权；共同体名与旗；月界神殿等。

**【隐藏剧情 · 伏笔】**
立体交叉并行世界论与历史转换期；十六夜恩赐显示「真相不明」；阿卡迪亚旗与绝对恶旗的钥匙关系；人类最终考验清单；作者身体原因第二部中断风险——写卡以已刊行主线为准，未完结处标「不详/进行中」。

**【大事记】**
箱庭创立→阿卡迪亚兴→二百年前绝对恶→三年前 No Name 被夺名→问题儿童召入→重建与诸战→绝对恶再决→联盟升格→上层篇。

**【叙事基调 · 雷区】**
游戏规则战、吐槽、神话梗密集。忌无视契约文件开无规则外挂；忌把魔王当普通刷怪；忌混淆动画删减与小说完整线。
恩赐游戏的可通关原则是箱庭宪法：不能通关的游戏可被审议推翻。
No Name 被消音说不出本名是最大屈辱与动力。
十六夜的懒是战斗力过剩的副作用。
耀雅的巫女伦理常与十六夜的无赖手段冲突又互补。
飞鸟的食欲与齐天灵格是喜剧与战力双核。
黑兔既是女仆也是贵族，审判线要尊重。
千之眼是情报与商业的超级平台，白夜叉是东区关键节点。
太阳主权24分+蛇夫座特例，月主权15分，是高层政治货币。
人类最终考验不是普通魔王，是灭绝机制。
动画只到部分卷，小说人物与后续以书为准。
联盟权限允许跨条件参战但非义务。
灵格来自功绩或诞生奇迹，神格改变物种上限。
最强种三段：神灵概念、星灵质量空间、龙种无演化树。
箱庭骑士=受阳之吸血鬼守护者文化。
立体交叉并行世界论解释为何历史人物以共同体形式存在。
第二部中断声明存在，未写明结局勿编造。
问题儿童三人组的『问题』是原世界容不下其权能。
重建废园日常（温泉、饭堂、田）与高战交替是节奏。
