# 自新世界

## 剧情

**【作品来源】**
《自新世界》（日文：新世界より；英文：From the New World）是日本作家**贵志祐介**的长篇科幻小说（讲谈社，2008 单行本；后文库三卷），获第29回日本SF大赏。2012-2013 由 A-1 Pictures 改编电视动画（石滨真史导，全25话）；及川彻漫画连载于别册少年Magazine。书名呼应德沃夏克《新世界交响曲》第二乐章改编的归途广播曲。本条目以**小说正典为主、动画重要差异标注**，锚定反乌托邦超能力社会、化鼠起义与恶鬼伦理。文风：田园表象下的恐怖、成长编年、制度残酷与微光改革。

**【世界定位】**
约公元2011年后人类出现念动力（咒力/Canto），文明崩溃，人口锐减。千年之后，日本残存约五六万咒力者，分居九町；主角生活于**神栖66町**。社会以基因改造植入**攻击抑制**与**愧死机构**，使咒力者无法对「人类」杀戮；无咒力者被改造为**化鼠**供役使。完人学校培养儿童，教育委员会以**不净猫**处分「不合格」孩子并篡改记忆。渡边早季从12岁野营接触伪蓑白得知黑历史，经友人业魔/逃亡/恶鬼之乱，至成年与朝比奈觉改革伦理。一句话：**在会读心也会杀人的田园里，用记忆与选择撬动制度。**

**【世界观 · 力量体系】**
力量本源：咒力（念动，理论上近乎无上限，强者传说可裂地球——叙事夸张需宁低勿高折算）、真言催眠开关、精神感应与记忆篡改、基因愧死与攻击抑制、化鼠生物社会、不净猫杀戮兽、先史生化武器（精神破坏者/炭疽等）。死亡观：愧死可致心脏停跳；业魔无意识泄漏咒力畸变化周围；恶鬼无抑制屠杀。特殊系统：八丁标结界绳、伦理/教育/安保三权、化鼠部族朝贡。

战力阶梯（破坏力表现，宁低勿高）：
1. **未觉醒儿童、普通町民日常**。≈**一阶**。
2. **和贵园~完人学校学生基础咒力**。≈**二阶**。
3. **成熟町民、保健所 levell 公务员**。≈**三阶**。
4. **专职鸟兽官、死神小组级**。≈**四阶**。
5. **日野光风、一般强咒力职人**。≈**五阶**。
6. **镝木肆星级「町最强」**（仍惧恶鬼与干涉）。≈**六阶**。
7. **恶鬼（小真理亚类）、业魔全盛、朝比奈富子级长生权威**；真言制度本身。≈**七阶顶点（条件）**。  
注：小说有「裂地球」传说，乐园映射**宁低勿高**，以实际战役表现封顶七阶；不设八九阶爆星。

乐园阶位映射：儿童≈一阶；学生≈二阶；成人咒力≈三阶；专业处分层≈四阶；强职人≈五阶；肆星≈六阶；恶鬼/富子/制度顶点≈七阶。**顶点＝七阶：存在·情报优先／条件性胜利**；恶鬼非「削弱归零」，须用愧死对象错认、生化、不净猫等条件策略。

**【地理 · 舞台】**
- **神栖66町**：利根川下游七乡（茅轮、松风/朽木、白砂、水车、见晴、黄金、栎林），约三千人，无货币互助。
- **八丁标**：注连绳结界，童禁外。
- **和贵园 / 完人学校 / 清净寺**：教育与真言授受。
- **妙法农场**：基因改造粮畜。
- **化鼠领地**：食虫虻、大黄蜂、土蜘蛛、木蠹蛾等部族。
- **东京废墟**：先史文明与生化武器库。
- **其它八町**（夕张新生、白石71等）：孤立联络。

**【世界剧情线】**

**① 黑历史（伪蓑白揭示）**  
咒力出现后热兵器无效，世界人口仅余约2%，东北亚分裂为奴隶王朝、狩猎逃亡团、移动掠夺团、科技继承团。科技团以愧死基因稳定社会，并把无咒力人类改造成化鼠以免触发愧死。

**② 12岁·野营与伪蓑白**  
早季、觉、瞬、真理亚、守（及一度同班的丽子）野营捕获自进化图书馆终端**伪蓑白**，得黑历史。僧人离尘毁终端、封印咒力押回；气球狗炸死离尘。早季觉被化鼠俘，食虫虻奏上役**斯奎拉**求援；觉恢复咒力后大战土蜘蛛；**奇狼丸**率大黄蜂灭敌并暗助归町。

**③ 处分与记忆**  
丽子、片山学等因咒力弱或违规「消失」，实为不净猫处决+记忆抹除。教育委员会以「直至成年无人权」逻辑运转。

**④ 14岁·业魔瞬**  
瞬咒力失控成**业魔**（桥本—阿培巴姆症候群），畸变松风之乡，自沉地下死前向早季告白。记忆被替换为稻叶良。富子看中早季心性稳定，有意培养继任。

**⑤ 守与真理亚逃亡**  
守恐被处分，与真理亚越八丁标；斯奎拉（野狐丸）助演「雪崩身死」。二人实则产子后被化鼠杀害，遗骸后送回；孩子被化鼠抚养。

**⑥ 26岁·化鼠战争与恶鬼**  
斯奎拉改革食虫虻（废女王专权、发展工业与火器），养**小真理亚**（认知为化鼠的咒力童）为武器，歼大黄蜂。死神小组覆灭，仅乾生还。瑞穗与敬返町放不净猫、毁库后死。早季、觉、奇狼丸、乾赴东京取「精神破坏者」类武器；乾死于异形；终局奇狼丸扮人类诱小真理亚杀「化鼠」触发愧死，与之同尽。斯奎拉遭无间地狱，早季私下咒力安乐死并焚尸。

**⑦ 终局改革**  
觉分析化鼠染色体23对，确认其人类起源。早季入伦理委员会，后任议长，推广恶鬼业魔知识，推动町际联络，与觉结婚期待下一代。手记封存时间胶囊与伪蓑白，千年后再公开。

**⑧ 主题**  
和平以处分为税；记忆是统治工具；化鼠镜映人类；爱可救人亦可生恶鬼；改革只能从制度内一点点凿。

**【主要人物】**
- **渡边早季**｜性格：坚韧、共情、渐具政治意志｜能力：稳定强咒力、领导质｜弧光：少女→保健所→伦理核心→议长｜立场：主角/手记作者。
- **朝比奈觉**｜性格：好动→成熟｜能力：反射面等战术咒力｜弧光：青梅→伴侣→科学真相揭示者｜立场：核心同伴。
- **青沼瞬**｜性格：温柔天才｜能力：顶尖咒力→业魔｜弧光：领袖→自沉｜立场：初恋与精神幻影。
- **秋月真理亚**｜性格：好强热情｜能力：物质操作优秀｜弧光：恋人→逃亡→惨死｜立场：悲剧核。
- **伊东守**｜性格：内向服从｜能力：弱咒力｜弧光：恐惧逃亡→死｜立场：弱者镜。
- **朝比奈富子**｜性格：深不可测长生者｜能力：端粒修复、记忆政治｜弧光：伦理议长→托孤早季｜立场：制度化身。
- **斯奎拉 / 野狐丸**｜性格：马基雅维利｜能力：政治与军事改革｜弧光：奏上役→叛乱→无间→被安乐死｜立场：最复杂反派。
- **奇狼丸**｜性格：武士忠诚与压抑恨｜能力：兵法｜弧光：救孩→灭族→牺牲触发愧死｜立场：悲壮盟友。
- **镝木肆星**｜性格：自信最强｜能力：全方位咒力｜弧光：听证→被小真理亚咒力干涉扭曲死｜立场：力量上限标尺。
- **日野光风**｜性格：傲慢艺术｜能力：强咒力｜弧光：夏祭屠化鼠→被偷袭击杀｜立场：傲慢代价。
- **渡边瑞穗 / 杉浦敬**｜性格：司书与町长｜能力：情报与行政｜弧光：毁库殉职｜立场：父母。
- **乾**｜性格：罪疚鸟兽官｜能力：实战｜弧光：独活→东京死｜立场：赎罪。
- **小真理亚（救世主）**｜性格：化鼠认同的咒力童｜能力：无对人类愧死｜弧光：屠町→中计愧死｜立场：条件顶点。
- **天野丽子 / 少年K**｜性格：弱者与历史恶鬼｜能力：弱咒力 / 无抑制屠杀｜弧光：处分；K被毒杀｜立场：制度起源。
- **鸟饲宏美**｜性格：谨慎教育长｜能力：处分权｜弧光：夏祭死｜立场：教育暴力。

**【势力图谱】**
- **伦理委员会**：最高伦理与记忆。
- **教育委员会**：儿童处分。
- **安全保障会议**：军事强者。
- **化鼠诸部族**：朝贡与叛乱。
- **清净寺**：真言宗教装置。
敌友：对恶鬼可临时团结；对化鼠「非人」共识是战争燃料；契约者若公开人类改造真相会触发清洗。

**【贵重物品】**
- **伪蓑白（SE终端）**：黑历史库。
- **真言**：咒力开关。
- **不净猫**：处分兽。
- **精神破坏者/炭疽类先史武库**：东京目标。
- **八丁标**：结界社会装置。
- **早季手记时间胶囊**：元叙事。
禁止编造「解除全人类愧死神器」除非条件剧情。

**【隐藏剧情 · 伏笔】**
- 化鼠=人类+裸鼹鼠基因。
- 人权自成年才赋予=合法杀童。
- 瞬幻影是早季心象非灵魂。
- 少年K事件塑造富子与处分制度。
- 归途广播曲与《美丽新世界》互文气质。
- 动画性别等与小说小有差异，以小说为准并注。

**【大事记时间线】**
2011咒力出现 → 黑暗时代 → 愧死与化鼠制度 → 少年K → 早季入学 → 野营伪蓑白 → 丽子等处分 → 瞬业魔死 → 守真理亚逃 → 26岁化鼠战 → 恶鬼袭町 → 东京 → 奇狼丸牺牲 → 斯奎拉之刑 → 早季改革与手记。

**【制度暴力细录】**
纸牌与推球比赛是筛选；记忆抹除是麻醉；不净猫是礼貌的死刑；真言是催眠主权。契约者低阶最该体验的是「同学突然从未存在」。

**【化鼠政治细录】**
斯奎拉议会、水泥与电灯、火枪、伪恶鬼战术；奇狼丸曾赴东京寻武库。叛乱不是兽潮，是被压迫「非人」的启蒙战争——早季的安乐死是慈悲也是封口。

**【与相邻条目边界】**
本世界独立；不与其它超能力IP混设定。前传《新世界零年》可点到。

**【契约者禁忌】**
教儿童无抑制攻击；向化鼠走私咒力者精液育种；公开播放伪蓑白无保护；复制无间地狱虐囚。失败：不净猫、记忆抹、全町敌对。

**【叙事基调 · 雷区】**
基调：金绿田园+血与地下通道；平静声线讲残酷事。忌：化鼠纯喜剧吉祥物；忌早季无成长的圣母；忌「群像学生」；忌把咒力写成修仙境界名。最早切入：和贵园入学；或野营夜；或26岁保健所到任。

**【咒力战斗物理】**
咒力互撞产生干涉波纹可扭曲肉体（肆星之死）；气球狗自爆；业魔环境畸变。乐园翻译：强不强看能不能在愧死框架内完成任务。

**【九町与人口】**
总人口五六万，通讯差，恶鬼后早季推联络协议会——改革是基础设施。

## 阶位切入点

> 阶位↔：一阶≈儿童入学；二阶≈完人学生；三阶≈成年町民；四阶≈专业处分/鸟兽；五阶≈强职人；六阶≈肆星级；七阶≈恶鬼/富子/制度战。顶点条件胜。

**一阶（和贵园 · 觉醒前后）**
切入身份/时点：与早季同龄转入生，咒力刚不稳。
初始事件：纸牌课上你的牌飞向真理亚的塔，鸟饲的目光已经在量不净猫的食量。
开场白建议：「教室里的纸牌倒了，倒的是你的未来。窗外归途旋律还没响，老师已经不说你的名字。」
关键NPC立场：**早季**想帮你；**教育委员**要处分；**丽子**是前车之鉴。
主线钩子/支线：稳住咒力；支线：藏日记防记忆抹。
危险度/规避：中（制度杀）。规避八丁标外。
任务方向/奖励：真言辅导、护身暗示。

**二阶（完人学校 · 野营年）**
切入身份/时点：野营编入五人组外围。
初始事件：伪蓑白开口，离尘的锡杖已经抬起，你要选择听完黑历史或捂耳。
开场白建议：「蓑白的触角亮着图书馆的光。它说：你们的和平用谁的血买——僧人的影子盖住了字幕。」
关键NPC立场：**伪蓑白**要被毁；**离尘**要审判；**斯奎拉**要利用。
主线钩子/支线：逃出生天；支线：记真言备份。
危险度/规避：高。
任务方向/奖励：黑历史残页、化鼠语词。

**三阶（成年 · 町内职）**
切入身份/时点：保健所或农场职员，14~26岁空窗期。
初始事件：同事的名字从名册蒸发，你被要求对空座位微笑。
开场白建议：「名册上有一行空白，空白还会微笑着跟你打招呼。觉说：别念出那名字。」
关键NPC立场：**富子**要观察；**瑞穗**要书；**觉**要真相。
主线钩子/支线：查处分档案；支线：保护逃亡线索。
危险度/规避：中高。
任务方向/奖励：权限徽章、记忆锚定物。

**四阶（鸟兽/死神外围）**
切入身份/时点：准死神或乾的助手。
初始事件：食虫虻前线，小队被伪恶鬼一扫，你要决定救乾还是救情报。
开场白建议：「咒力的风把化鼠的旗撕成牙。你第一次看见『救世主』的背影——那背影比恶鬼童谣更矮。」
关键NPC立场：**乾**要赎罪；**斯奎拉**要帝国；**奇狼丸**要女王。
主线钩子/支线：撤离；支线：不净猫投放点。
危险度/规避：高。
任务方向/奖励：四阶实战装、化鼠地图。

**五阶（恶鬼袭町）**
切入身份/时点：防卫职人。
初始事件：夏祭火枪响，光风倒下，小真理亚的咒力撕开街道。
开场白建议：「祭典鼓停了。鼓停的地方，人的形状在弯折——肆星的双虹膜里第一次有恐惧。」
关键NPC立场：**小真理亚**顶点童；**富子**要托孤；**早季**要决策。
主线钩子/支线：掩护撤离；支线：释放不净猫。
危险度/规避：极高。
任务方向/奖励：五阶、父母遗言任务。

**六阶（东京废墟）**
切入身份/时点：四人行替代位或支援。
初始事件：异形沙蚕袭来，生化库门密码是先史伦理题。
开场白建议：「东京的地下河反着没有星星的光。乾说：别回头——回头的人都想活在旧世界。」
关键NPC立场：**奇狼丸**要交易；**早季**要武器；**斯奎拉**要追。
主线钩子/支线：取得对策；支线：确认化鼠染色体。
危险度/规避：极高。
任务方向/奖励：六阶、武库样本。

**七阶（终局伦理 · 无间与改革）**
切入身份/时点：伦理委员会旁听或早季助手。
初始事件：斯奎拉喊「我们是人类」，无间地狱启动，早季要你保密安乐死。
开场白建议：「锁链上的神经在唱歌。斯奎拉的眼睛仍是奏上役的眼睛——早季问你：慈悲算不算叛伦理。」
关键NPC立场：**早季**要改；**觉**要科学；**委员会**要示众；**斯奎拉**要死得像人。
主线钩子/支线：安乐死与手记；支线：推联络协议会、孕期政策。
危险度/规避：贴近顶点（政治+愧死伦理）。条件性胜利：制度改革而非杀光。
任务方向/奖励：七阶条件认可、手记副本权限、非越级灭世武。
