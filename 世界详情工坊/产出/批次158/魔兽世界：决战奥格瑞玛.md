# 魔兽世界：决战奥格瑞玛
<!--meta lib=主库 tiers=一、二、三、四、五、六、七、八-->

## 剧情

**【作品来源】**
《魔兽世界：决战奥格瑞玛》（Siege of Orgrimmar，缩写 SoO）是暴雪娱乐 MMORPG《魔兽世界》资料片《熊猫人之谜》**补丁 5.4.0（2013-09-10）** 终局团队副本与同名战役。地点覆盖**锦绣谷·永春台残迹 / 奥格瑞玛 / 怒焰裂谷下的幽暗堡（Underhold）**。本条目锚定**加尔鲁什·地狱咆哮「真部落」末日、傲之煞与亚煞极之心、联盟与暗矛起义联军攻城**正典；辅以《战争罪行》审判起笔。不写修仙词，不把德拉诺铁堀整本倒灌。文风：内战史诗、荣誉崩坏、古神低语、阵营政治。

**【世界定位】**
大酋长加尔鲁什把部落从「被压迫者同盟」扭成**真部落**——以兽人至上、科卡尔与黑索科技、螳螂妖帕拉贡与**亚煞极之心**为支柱。他在锦绣谷掘出古神之心、击溃傲之煞后仍把谷地变成煞魔伤疤；沃金的暗矛起义与瓦里安的联盟舰队夹击奥格瑞玛。英雄从永春台清煞，在刀锋湾登陆击破龙喉与迦拉卡斯，破城门、斩纳兹戈林、下幽暗堡，最终在内殿击败握着**夏拉托（Xal'atoh）**与亚煞极之心的加尔鲁什。他被生俘交付潘达利亚审判，沃金继任大酋长，联盟—部落暂时停火。一句话：**在奥格瑞玛的钢铁与煞气里，用联军纪律结束一场内战。**

**【世界观 · 力量体系】**
力量本源：战士荣誉与萨满元素（被黑暗萨满扭曲）、古神亚煞极/煞魔、螳螂妖琥珀与帕拉贡、地精黑索工程、凡人职业与阵营军事工业。死亡观：城内拷打与绞杀战犯常见；古神增强的加尔鲁什可对整队造成战略级威胁，但非行星爆破演出。特殊系统：四翼结构（锦绣谷→外围登陆→城内→幽暗堡）、弹性10-30人、传说披风线与勇气装、阵营专属过场。

战力阶梯（破坏力表现，宁低勿高）：
1. **杜隆塔尔/贫瘠之地外围杂兵、城内平民械斗级**。≈**一阶**。
2. **暗矛起义正规、联盟登陆水兵、科卡尔巡逻**。≈**二阶**。
3. **城门前精英、黑暗萨满学徒、黑索技师**。≈**三阶**。
4. **永春台前段 / 迦拉卡斯 / 钢铁战蝎门槛**。≈**四～五阶**。
5. **科卡尔黑暗萨满、纳兹戈林、曼科里克**。≈**五～六阶**。
6. **索克、黑索、帕拉贡议会**。≈**六阶**。
7. **加尔鲁什（亚煞极强化）**。≈**七～八阶顶点（条件战）**。
8. **亚煞极之心本身 / 傲之煞余波**。≈**八阶级存在威胁（非可永久抹杀）**。

乐园阶位映射：外围民兵≈一阶；起义/登陆正规≈二阶；城门精英≈三阶；永春台与登陆诸王≈四～五阶；城内将军层≈五～六阶；幽暗堡中层≈六阶；加尔鲁什≈七～八阶顶点。宁低勿高。**顶点＝八阶条件：存在·情报优先／条件性胜利**；加尔鲁什被俘非「战力归零」，亚煞极亦非永杀。

**【地理 · 舞台】**
- **锦绣谷 / 永春台**：亚煞极穹顶、堕化守护者、诺鲁什、傲之煞。
- **刀锋湾 / 德拉诺什尔登陆点**：龙喉与原型龙迦拉卡斯。
- **奥格瑞玛城门与主城**：钢铁战蝎、黑暗萨满、格罗玛什要塞、怒焰裂谷入口。
- **幽暗堡**：曼科里克、潘达利亚战利品库、索克、黑索工厂、帕拉贡、内殿。
- **关联前史**：塞拉摩灰烬、神钟、匕首暗影刺杀沃金未遂、贫瘠之地战场。

**【世界剧情线】**

**① 前史·大酋长的滑坡**  
萨尔将大酋长之位予加尔鲁什。北裂与大灾变中他积累战功与偏执：塞拉摩法力炸弹、压迫暗矛、神钟邪恶魔力、刺杀沃金未遂。沃金串联暗矛起义；联盟亦要清算。贫瘠之地战场上联盟与起义军协作，刀锋湾留给舰队。

**② 亚煞极之心**  
科卡尔与地精发掘队在锦绣谷挖出亚煞极穹顶。加尔鲁什携心至谷，击败塔兰·朱，用圣泉能量唤醒古神之心。谷地化为「永春台」伤疤：伊墨苏斯（圣泉堕化本质）、堕落守护者、诺鲁什试炼、傲之煞。吉安娜与洛瑟玛在傲之煞后见到**血吼**遗落，确认加尔鲁什所为，临时联手。

**③ 围城准备**  
瓦里安舰队突破封锁；泰兰德哨兵运投刃车；沃金与贝恩控制杜隆塔尔与贫瘠之地大部；血精灵与被遗忘者跨海；萨尔与萨鲁法尔潜入城内策反。加尔鲁什调龙喉扎伊拉守港、纳兹戈林守门、黑索重武与科卡尔、并召克拉希帕拉贡。

**④ 登陆与破门**  
联军登陆德拉诺什尔，击破龙喉，打倒扎伊拉与原型龙**迦拉卡斯**。城门前钢铁战蝎阻路，英雄摧毁之；泰兰德投刃车拆门。城内解救季·火掌、加摩尔等俘虏，击杀黑暗萨满（哈罗姆与卡德里斯），见艾特里格负伤指路：加尔鲁什在怒焰下幽暗堡。

**⑤ 纳兹戈林之死**  
将军纳兹戈林以战士全技巧死战旧袍泽与宿敌，死前认可以后的部落将更好——本档最「荣誉」的敌对名场面之一。

**⑥ 幽暗堡**  
曼科里克（右膀，亚煞极腐蚀）、潘达利亚战利品（魔古/螳螂妖/雷电宝藏的军事仓储）、嗜血的索克、攻城匠师黑索（工厂与自动械）、克拉希帕拉贡九人轮战。每一步都显示真部落把「战利品与古神」产业化。

**⑦ 内殿·加尔鲁什**  
萨尔先以元素对质，因黑暗萨满污地而败。英雄以团队机制对抗夏拉托与亚煞极之心增幅的加尔鲁什，最终击倒但生俘。瓦里安阻止萨尔当场处决；塔兰·朱坚持潘达利亚审判。沃金被推举为大酋长；瓦里安以「荣誉」为条件给和平一次机会；吉安娜主张趁机摧毁部落被拒。谷中绍浩与天神生出希望之树。加尔鲁什日后越狱导向德拉诺——本档终局以**被俘+沃金继任+停火**为准。

**⑧ 主题**  
荣誉可被权力蛀空；古神给「捷径力量」；内战里仍有纳兹戈林式死战尊严；胜利者选择审判而非屠城定义文明。

- **瓦里安·乌瑞恩**｜性格：战士国王、务实和平｜能力：联盟最高军事权威｜弧光：攻城→阻止处决→承认沃金｜立场：联盟顶点。

- **伊墨苏斯 / 傲之煞 / 诺鲁什**｜性格：环境与试炼｜能力：煞与净化逻辑｜弧光：永春台｜立场：前段威胁。
- **季·火掌 / 加摩尔 / 艾特里格**｜性格：被囚的火与荣誉｜能力：武僧/战士/顾问｜弧光：解放与指路｜立场：友。

**【隐藏剧情 · 伏笔】**
- 纳兹戈林之死是荣誉主题反题。
- 吉安娜灭族冲动为日后阵营冲突埋伏笔。
- 加尔鲁什越狱→德拉诺铁堀。
- 傲之煞与「骄傲」精准咬合加尔鲁什性格。

**【大事记时间线】**
塞拉摩与神钟 → 刺杀沃金 → 起义与贫瘠之地 → 掘心与永春台堕化 → 傲之煞 → 舰队登陆 → 迦拉卡斯 → 钢铁战蝎破门 → 黑暗萨满 → 纳兹戈林 → 幽暗堡诸王 → 加尔鲁什败俘 → 沃金大酋长 → 停火 → （后）审判与越狱。

**【逐段战役细录】**
永春台伊墨苏斯是「圣泉被煞化」的视觉课；堕落守护者在金莲故地打架是道德恶心；诺鲁什先把你当污染源再认你；傲之煞点名骄傲。登陆战要处理原型龙空中阶段与塔楼。钢铁战蝎是黑索美学。黑暗萨满让元素尖叫。纳兹戈林战是怒火与军旗。幽暗堡越深越工业+古神。加尔鲁什阶段转换是「真部落演讲」与「古神低语」交替。

**【城内社会与战俘】**
塞拉摩平民与暗矛猎头被囚拷打；术士被绞杀示众。契约者解救路线是低~中阶高价值任务。加摩尔的牢房是荣誉笑话与泪点并存。

**【与相邻条目边界】**
雷电王座神钟线可口头互文；不写德拉诺全本；不写第四次大战二次围城。审判见《战争罪行》点到为止。

**【契约者禁忌】**
向真部落出售元素污秽样本；暗杀沃金或瓦里安「以加速和平」；盗走亚煞极之心碎片私用；在内殿抢处决权。失败：双阵营通缉或煞化。

**【叙事基调 · 雷区】**
基调：赤红钢铁、煞白裂痕、海风与机油。忌：加尔鲁什脸谱小丑；忌纳兹戈林无尊严；忌「群像起义」；忌修仙词。最早切入：贫瘠之地起义周；或刀锋湾登陆；或纳兹戈林门前。

**【阵营过场差异】**
联盟侧强调瓦里安秩序；部落侧强调沃金与萨尔的痛苦。AI 应按契约者阵营选择开场声线，禁止混用到OOC。

**【幽暗堡工业气味】**
机油、琥珀甜腥、兽栏血腥、古神低语的耳压、熔炉硫磺。可观察细节优先于抽象「邪恶」。

## 阶位切入点

> 阶位↔：一阶≈外围民兵；二阶≈起义/登陆正规；三阶≈城门精英；四阶≈永春台/登陆王；五阶≈城内将军；六阶≈幽暗堡中层；七阶≈加尔鲁什战备；八阶≈加尔鲁什/古神条件战。顶点情报优先。

**一阶（贫瘠之地 · 起义民兵）**
切入身份/时点：暗矛新兵或联盟斥候学徒，贫瘠之地战场期。
初始事件：剃刀岭外，科卡尔巡逻要你交出「非兽人」同伴，沃金的信使要你送弹药。
开场白建议：「红土粘在靴底像干血。远处奥格瑞玛的城墙还只是一条锯齿剪影——近处科卡尔的斧已经抬起来。」
关键NPC立场：**沃金**要人；**科卡尔步兵**要服从；**贝恩**要伤亡账。
主线钩子/支线：送弹与救人质；支线：破坏投石车、收集塞拉摩俘虏名单。
危险度/规避：低~中。规避加尔鲁什本人。
任务方向/奖励：起义徽记、基础武器、地图。

**二阶（舰队与登陆准备）**
切入身份/时点：舰员或起义士官，刀锋湾开战前夜。
初始事件：龙喉炮弹打穿邻船，你必须在三分钟内救出落水的血精灵游侠。
开场白建议：「海水是铁味的。信号旗在烟里变成黑布，有人喊扎伊拉的龙来了——你的绳结比口号重要。」
关键NPC立场：**瓦里安**要登陆窗口；**扎伊拉**要沉船；**洛瑟玛**要舰队阵型。
主线钩子/支线：登陆桥头堡；支线：拆防空、救地精工兵。
危险度/规避：中。规避迦拉卡斯软狂。
任务方向/奖励：登陆装、抗火药水。

**三阶（城门前 · 钢铁战蝎）**
切入身份/时点：破门突击队。
初始事件：钢铁战蝎锁定你的投刃车坐标，你要引走炮火好让泰兰德完成拆门。
开场白建议：「蝎螯落下时地面像鼓面。泰兰德的投刃车还差一次齐射——你的位置如果错了，齐射会先打中自己人。」
关键NPC立场：**泰兰德**要窗口；**黑索技师**要数据；**沃金**要死守。
主线钩子/支线：破门；支线：解救外围战俘。
危险度/规避：中高。
任务方向/奖励：三阶级城战装。

**四阶（永春台/登陆王 · 双线英雄）**
切入身份/时点：可被派往永春台或迦拉卡斯的团队成员。
初始事件：傲之煞点名「以你为傲的瞬间」，幻象里塞拉摩在烧，你必须驱散自己。
开场白建议：「白雾里有人用你的声音说：你比加尔鲁什更适合握心。你知道这是煞——你仍想听完。」
关键NPC立场：**傲之煞**要腐蚀；**吉安娜**要证据；**迦拉卡斯**要吞噬。
主线钩子/支线：清永春台或港口；支线：血吼线索、龙喉降书。
危险度/规避：高。
任务方向/奖励：四~五阶装、煞抗。

**五阶（城内 · 纳兹戈林）**
切入身份/时点：入城后将军战。
初始事件：纳兹戈林点名旧战友「来杀我」，你的阵营光环决定他台词的温度。
开场白建议：「格罗玛什要塞的旗在风里裂开。纳兹戈林把刀横在肩上，像在等一个配得上死的理由。」
关键NPC立场：**纳兹戈林**可敬之敌；**黑暗萨满**要污地；**艾特里格**要指路。
主线钩子/支线：纳兹戈林；支线：解放季·火掌与加摩尔、砍断绞架。
危险度/规避：高。
任务方向/奖励：五~六阶装、荣誉叙事物。

**六阶（幽暗堡中层 · 工业与帕拉贡）**
切入身份/时点：下怒焰后的核心队。
初始事件：黑索传送带把炸弹送到你脚下，帕拉贡之一基尔鲁克在暗处计数你的心跳。
开场白建议：「机油从天花板滴进煞光里。黑索的笑声从扩音器里出来，像拍卖——你的命是今晚的报价。」
关键NPC立场：**黑索**要利润；**曼科里克**要忠；**帕拉贡**要古神逻辑。
主线钩子/支线：打通至内殿；支线：炸工厂核心、偷战利品清单。
危险度/规避：极高。规避加尔鲁什。
任务方向/奖励：六阶装、工程蓝图残页。

**七阶（内殿战备 · 对质前夜）**
切入身份/时点：加尔鲁什战前编入。
初始事件：萨尔被击飞，元素沉默，你被点名顶上第一轮嘲讽轴。
开场白建议：「萨尔的膝盖砸在金属地板上。加尔鲁什说：看，元素也不认你。你听见自己的牙齿在响——那是鼓点。」
关键NPC立场：**加尔鲁什**要碾压；**萨尔**要赎罪；**瓦里安**要活口。
主线钩子/支线：压低加尔鲁什血线；支线：保护塔兰·朱观战位。
危险度/规避：贴近顶点。
任务方向/奖励：七阶装、审判押送权相关。

**八阶（顶点 · 俘获与审判）**
切入身份/时点：击倒后的政治现场。
初始事件：萨尔要锤死加尔鲁什，瓦里安横剑，吉安娜要屠城，你必须在三秒内表态并承受后果。
开场白建议：「铁链比欢呼先响。沃金的影子第一次盖过萨尔——瓦里安问：谁来当大酋长？空气比亚煞极还重。」
关键NPC立场：**瓦里安**要秩序；**吉安娜**要灭绝；**塔兰·朱**要审判；**沃金**要承担。
主线钩子/支线：护送战俘、停火谈判；支线：阻止盗心者。
危险度/规避：极高/政治杀。超阶：存在·情报优先/条件性胜利。
任务方向/奖励：八阶条件声望、停火文书副本、非越级神器。
