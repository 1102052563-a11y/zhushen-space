# 魔兽世界：悬槌堡

## 剧情

**【作品来源】**
《魔兽世界：悬槌堡》（Highmaul）是暴雪娱乐 MMORPG《魔兽世界》资料片《德拉诺之王》（Warlords of Draenor）**首个团队副本**（2014-12-02 开放普通/英雄，随后史诗与LFR分期）。地点在**平行宇宙德拉诺·纳格兰**西部，**高里亚帝国**首都。本条目锚定**高里亚食人魔文明巅峰、恺撒马尔高克、铁堀盟友卡加斯刃拳、虚空扭曲的暗夜魔仆**正典；不写阿古斯/暗影国度倒灌。文风：罗马式竞技场暴力、奥术贵族、食人魔「并不愚蠢」的帝国美学。

**【世界定位】**
在德拉诺之王时间线，食人魔高里亚帝国仍统治 generational 荣光，悬槌堡是其首都：集市、贵族、竞技场欢呼与**恺撒马尔高克**的要塞阴影同在。铁堀战争期间，卡加斯·刃拳等在此结盟表演；英雄突入，从竞技场杀到奥术圣所再到皇帝高塔，阻止马尔高克用远古奥术让食人魔重夺德拉诺霸权，并面对虚空 entwined 的古加尔投影威胁。一句话：**在食人魔帝国的巅峰之城里，打断一场奥术复辟。**

**【世界观 · 力量体系】**
力量本源：高里亚奥术符文与空无法术、食人魔双头/独眼生理与力役、独眼魔与真菌巨人自然暴力、虚空/古加尔低语、铁堀军事、德拉诺原住民萨满传统（对照）。死亡观：竞技场公开处刑是国本；皇帝级奥术可改写局部空间。特殊系统：非线性部分路径（城墙城/奥术/高塔）、LFR三翼、废法者钥石线、装等640起。

战力阶梯：
1. **集市流民、竞技场苦力、外围狼獾人**。≈**一阶**。
2. **高里亚卫兵、符文师学徒、铁堀步兵**。≈**二阶**。
3. **竞技场精英、屠宰场监工**。≈**三阶**。
4. **卡加斯·刃拳、屠夫**。≈**四～五阶**。
5. **泰克图斯、布兰肯斯波**。≈**五阶**。
6. **独眼魔双子、克尔拉夫**。≈**五～六阶**。
7. **马尔高克（及古加尔阶段）**。≈**七～八阶顶点（条件战）**。

乐园阶位映射：流民苦力≈一阶；卫兵≈二阶；精英≈三阶；刃拳/屠夫≈四～五阶；山与菌≈五阶；圣所≈五～六阶；皇帝≈七～八阶顶点。宁低勿高。**顶点＝条件战：存在·情报优先**；古加尔投影非主宇宙全盛。

**【地理 · 舞台】**
- **纳格兰（德拉诺）西部**：悬槌入口，近莫克戈尔哨站。
- **城墙之城**：角斗场、屠宰场下层、菌毯海岸。
- **奥术圣所**：活体山泰克图斯、废法室、独眼魔厅。
- **皇帝高塔**：马尔高克王座、虚空扭曲仪。
- **社会空间**：集市、贵族区、贫民看台——帝国「文明」与血腥一体。

**【世界剧情线】**

**① 高里亚帝国背景**  
食人魔曾是德拉诺霸主，高里亚文明以奥术与奴隶制著称，打破「食人魔只有蛮力」的偏见。悬槌堡是权力之座：竞技场凝聚民意，要塞凝聚恐惧。德莱尼到来后帝国受挫，但仍具复辟野心。

**② 铁堀与角斗**  
古尔丹与铁堀崛起期，卡加斯·刃拳等铁堀军阀与悬槌发生利益捆绑。角斗场上，刃拳作为碎手氏族战力表演，铁堀投弹手与血喉兽共同制造观众高潮——政治即娱乐。

**③ 攻入城墙之城**  
- **卡加斯·刃拳**：碎手之王，刃拳武器与观众机制，醉酒投弹食人魔。  
- **屠夫**：下层肉山， cleave 与血怒，夜扭曲尸体。  
- **布兰肯斯波**：深行者真菌巨人，孢子与铁堀技术人员混线。  
主题：帝国用暴力秀维持秩序。

**④ 奥术圣所**  
- **泰克图斯，活体山**：分裂为碎片与微粒，夜扭曲土法师。  
- **独眼魔双子波尔与费莫斯**：一盾一锤的协同暴君。  
- **克尔拉夫，法术破坏者**：废法屏障，专克施法者，高里亚对奥术的垄断暴力。  
主题：谁控制法术，谁控制帝国。

**⑤ 皇帝之陨**  
**恺撒马尔高克**在高塔解锁远古秘密，召唤奥术畸变，并在关键阶段与**古加尔**虚空投影纠缠——夜扭曲信徒涌入。英雄击杀马尔高克，打断高里亚奥术复辟。铁堀仍在德拉诺其它战线；本档终局以**皇帝陨落、悬槌权力真空**为准。

**⑥ 主题**  
野蛮与文明的假二分；竞技场是政治；奥术贵族比兽人更老谋；虚空是复辟的利息。

**【主要人物】**
- **恺撒马尔高克**｜性格：奥术帝王、监视一切｜能力：高里亚至高法术、畸变召唤｜弧光：复辟→与虚空纠缠→陨落｜立场：顶点敌。
- **卡加斯·刃拳**｜性格：残酷表演者｜能力：刃拳、角斗规则｜弧光：门神→败亡｜立场：铁堀—悬槌纽带。
- **屠夫**｜性格：无言肉山｜能力：物理碾压｜弧光：下层恐惧｜立场：敌。
- **泰克图斯**｜性格：山岳意志｜能力：分裂山体｜弧光：圣所守门｜立场：环境级。
- **布兰肯斯波**｜性格：深海/菌毯异化｜能力：孢子战争｜弧光：海岸线｜立场：敌。
- **波尔 / 费莫斯**｜性格：双子羁绊暴力｜能力：盾锤协同｜弧光：废法前卫｜立场：敌。
- **克尔拉夫**｜性格：猎法者狂热｜能力：废法｜弧光：圣所尾｜立场：敌。
- **古加尔（投影）**｜性格：虚空双头｜能力：暗夜扭曲｜弧光：皇帝战阶段｜立场：条件威胁。
- **高里亚议会成员（达格拉特等）**｜性格：官僚好战｜能力：符文与卫队｜弧光：高塔前｜立场：敌。
- **加尔格（竞技场主）** 等｜性格：娱乐官｜能力：调度野兽｜弧光：开场｜立场：可贿赂或杀。
- **玩家远征军指挥官（霜狼/卡拉波侧口头）**｜性格：务实｜能力：调度｜弧光：雇主｜立场：友。

**【势力图谱】**
- **高里亚帝国**：皇帝—贵族—竞技场—奴隶。
- **铁堀**：军事同盟与技术渗透。
- **夜扭曲虚空信徒**：古加尔线。
- **德拉诺联盟/部落远征军**：攻城方。
敌友：反皇帝可合作；战后抢奥术库会开杀。

**【贵重物品】**
- **刃拳（The Bladefist）**：卡加斯标志武器。
- **废法者相关石与阿布罗加托尔石**（传说戒指线）。
- **高里亚符文武器与皇帝法器**。
- **活体山之心类饰品叙事**。
- **孢子瓶、心能式奥术残渣**。
禁止编造「一杖统一德拉诺」。

**【隐藏剧情 · 伏笔】**
- 悬槌展示食人魔历史高度，对照主宇宙衰落。
- 克尔拉夫是「反法者」国家暴力。
- 古加尔线连接上古虚空。
- 非线性路线鼓励不同政治切入（先屠宰场或先海岸）。
- 卡加斯死后碎手与铁堀其它线仍续。

**【大事记时间线】**
高里亚霸权世代 → 德莱尼冲击 → 铁堀崛起结盟 → 远征军攻纳格兰 → 角斗场刃拳 → 屠夫与菌毯 → 圣所山与双子与废法 → 皇帝高塔 → 马尔高克陨落 → 权力真空。

**【逐王名场面】**
刃拳把观众变成机制；屠夫让近战学会「散」。泰克图斯越打越碎越危险。布兰肯斯波要团队管理蘑菇阵营。双子要分摊与合。克尔拉夫让法系体验「被帝国立法剥夺」。马尔高克把王座厅变成奥术几何题，古加尔是利息到期。

**【城市生态】**
集市叫卖、贵族香料、奴隶烙印、竞技场欢呼、下水道菌臭。低阶可做角斗士契约工。

**【与相邻条目边界】**
不写黑石铸造厂全本；不写地狱火堡垒；卡加斯故事在悬槌收一段。主宇宙奥格瑞玛食人魔流亡可口头对比。

**【契约者禁忌】**
向铁堀出售废法技术；释放全部奴隶无安置导致屠城；与古加尔交易寿命；窃皇帝冠加冕。失败：帝国余党与远征军双杀。

**【叙事基调 · 雷区】**
基调：赤砂、紫奥术、 bilious 绿孢子。忌：食人魔纯傻；忌马尔高克战力归零；忌无名「议会群像」。最早切入：集市苦力；或角斗场入场式；或废法室门前。

**【高里亚法律物象】**
符文烙印、角斗契约、废法项圈、皇帝敕令石。乐园翻译：法律是伤害类型。

**【气味与声景】**
血沙、奥术臭氧、真菌甜腐、铁堀机油、观众喉噪、虚空耳鸣。

## 阶位切入点

> 阶位↔：一阶≈苦力流民；二阶≈卫兵；三阶≈精英；四阶≈刃拳/屠夫；五阶≈山/菌/圣所；六阶≈克尔拉夫；七阶≈皇帝战备；八阶≈马尔高克/古加尔条件战。

**一阶（集市苦力）**
切入身份/时点：奴隶账房助手或远征军间谍学徒。
初始事件：角斗士契约被改成你的名字，加尔格要你今晚上场或找出伪契者。
开场白建议：「香料盖不住血。契约泥板上的名字还在渗墨——那是你的，或你必须在日落前找到的替死鬼。」
关键NPC立场：**加尔格**要节目；**卫兵**要贿赂；**奴隶**要活。
主线钩子/支线：毁契或赢一场；支线：送情报出城。
危险度/规避：低~中。
任务方向/奖励：伪装服、硬币、地图。

**二阶（高里亚卫兵）**
切入身份/时点：雇佣符文卫或渗透的远征军。
初始事件：夜扭曲仪式在下水道被你撞见，你要上报贵族还是卖给虚空。
开场白建议：「符文灯把你的影子切成三段。第三段影子先动了——那不是你的。」
关键NPC立场：**符文师**要保密；**夜扭曲**要发展；**远征军接头人**要证据。
主线钩子/支线：选线；支线：缴获虚空残渣。
危险度/规避：中。
任务方向/奖励：二阶符文甲。

**三阶（精英清场）**
切入身份/时点：突击队前锋，刃拳战前。
初始事件：血喉兽出笼，你的小队被点名为「开胃菜」。
开场白建议：「兽栏门的锁比法律松。血喉的呼吸喷在脸上时，观众已经开始下注。」
关键NPC立场：**刃拳**要表演；**铁堀投弹手**要炸席。
主线钩子/支线：清至刃拳；支线：救俘虏角斗士。
危险度/规避：中高。
任务方向/奖励：三阶装。

**四阶（刃拳与屠夫）**
切入身份/时点：主队成员。
初始事件：刃拳点名你进入「火焰走道」，观众扔瓶。
开场白建议：「刃拳的铁手在火光里像第二张嘴。他喊你的种族名当笑点——你要让笑停在他的喉咙里。」
关键NPC立场：**卡加斯**敌；**屠夫**无言敌。
主线钩子/支线：双王；支线：破坏铁堀弹药。
危险度/规避：高。
任务方向/奖励：四~五阶装、刃拳碎片叙事。

**五阶（圣所山与菌）**
切入身份/时点：选择路径的队伍。
初始事件：泰克图斯分裂，你被分配去踩微粒；或孢子阵营选错。
开场白建议：「山在走路。你的靴子上粘着还在跳的石头——治疗说别舔。」
关键NPC立场：**泰克图斯**环境敌；**布兰肯斯波**菌毯敌；**土法师**要献祭。
主线钩子/支线：打通圣所；支线：采集绝缘孢子。
危险度/规避：高。
任务方向/奖励：五阶装。

**六阶（废法者克尔拉夫）**
切入身份/时点：法系核心或废法反制专家。
初始事件：你的法术条被沉默成「帝国许可制」，近战必须掩护导条窗口。
开场白建议：「法术在舌尖变成沙子。克尔拉夫说：会施法的奴隶不是奴隶——是叛徒。」
关键NPC立场：**克尔拉夫**猎法；**双子**要分摊。
主线钩子/支线：击杀克尔拉夫；支线：夺废法项圈样本。
危险度/规避：极高。
任务方向/奖励：六阶装、废法研究。

**七阶（皇帝战备）**
切入身份/时点：高塔突击。
初始事件：奥术畸变复制你的技能，议会卫队切断退路。
开场白建议：「王座的影子比白天长。马尔高克的眼睛在三个方向同时睁开——他说他一直看着集市上的你。」
关键NPC立场：**马尔高克**顶点；**议会**要殉国。
主线钩子/支线：压阶段；支线： indestructible 难民撤离。
危险度/规避：贴近顶点。
任务方向/奖励：七阶装。

**八阶（皇帝与古加尔）**
切入身份/时点：尾王条件战。
初始事件：古加尔投影要你献出一只眼睛换「真知」，马尔高克要你跪。
开场白建议：「虚空在耳蜗里鼓掌。古加尔的两张脸问同一个问题：你为谁复辟？」
关键NPC立场：**马尔高克**；**古加尔**；**远征军**要你别交易。
主线钩子/支线：击杀/驱离；支线：封库。
危险度/规避：极高。超阶：存在·情报优先/条件性胜利。
任务方向/奖励：八阶条件物、高里亚法器（需上交）。
