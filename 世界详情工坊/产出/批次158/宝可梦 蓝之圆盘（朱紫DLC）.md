## 剧情

**【作品来源】**
《宝可梦 蓝之圆盘》（日文：藍の円盤 / 后篇：蓝之圆盘；英文：The Indigo Disk）是 Game Freak / 任天堂 / The Pokémon Company 为《宝可梦 朱／紫》推出的付费扩展包**「零之秘宝」后篇**，2023-12-14 全球同步配信；前篇为《碧之假面》，后有尾声《点心大骚动》。本条目锚定**蓝莓学园留学线 + BB 联盟四天王与冠军 + 蓝之圆盘开启零之下层 + 太乐巴戈斯**正典，辅以星晶太晶、悖论宝可梦与晶光池教授邂逅；不写修仙境界，不把关都/城都整史倒灌。文风：学院对战青春、友情裂痕与和解、太晶与地底遗迹神秘感。

**【世界定位】**
完成帕底亚主线与北上乡《碧之假面》后，主角受蓝莓学园园长**西拉罗**邀请（经**极星**推荐）赴合众地区附近的海上学园留学。学园主体在海下**生态巨蛋**（四生物群系），以**蓝莓联盟（BB League）**培养对战精英。表弟**乌栗**自北上乡后性格扭曲，执着变强并夺得冠军；主角挑战四天王与乌栗后，随**白露**持**蓝之圆盘**深入帕底亚**零之大空洞·零之下层**，唤醒传说**太乐巴戈斯**，平息其星晶失控，并在事后晶光池与过去的**奥琳博士/弗图博士**短暂交错。一句话：**在海底学院与地心太晶之间，用对战缝合裂开的友情。**

**【世界观 · 力量体系】**
力量本源：宝可梦属性相克与个体培养、太晶化（含第19属性**星晶**）、蓝莓点数与蓝乐活、同步器共视角、传说/悖论宝可梦 ecologic。死亡观：对战以昏厥为默认；传说失控可造成区域级太晶吸收与设施破坏，但不走「人类法术毁国」路线。特殊系统：精英试炼→四天王→冠军；道具打印机；联盟社特邀教员；蓝之圆盘作为零号电梯密钥。

战力阶梯（破坏力表现，宁低勿高）：
1. **学院新生 / 生态巨蛋野生低阶**：常规野战与社团对战。≈**一阶**。
2. **蓝乐活精英训练家 / 精英试炼守卫**：有体系配招。≈**二阶**。
3. **BB 联盟四天王**：紫竽、乌栗对手线中的**蕾荷、青木、乌栗前冠军线的**四天王**（蕾荷/青木/乌栗相关试炼后的**四天王：紫竽、青木、乌梅、杜若**等）——以游戏内四天王**蕾荷（Lacey）、青木（Crispin）、乌梅（Amarys）、杜若（Drayton）**为准。≈**三～四阶**。
4. **冠军乌栗（BB）**：执念强化后的冠军级。≈**四～五阶**。
5. **零之下层星晶太晶宝可梦 / 悖论种**：地底生态威胁。≈**五阶**。
6. **太乐巴戈斯（星晶形态失控）**：太晶现象本源级。≈**六阶顶点（条件战）**。

乐园阶位映射：新生与野战≈一阶；精英试炼≈二阶；四天王≈三～四阶；冠军乌栗≈四～五阶；零之下层≈五阶；太乐巴戈斯失控≈六阶顶点。宁低勿高。**顶点（太乐巴戈斯星晶失控）＝六阶顶：存在·情报优先／条件性胜利**，严禁「捕获即战力归零」；和解与再捕获是条件窗口。

**【地理 · 舞台】**
- **蓝莓学园**：合众近海，海上建筑+海底生态巨蛋（サバンナ/海岸/洞窟/ freeze 等四区生物群）。
- **联盟社 / 宿舍 / 食堂 / 园长室**：学院日常与蓝莓点数经济。
- **帕底亚·零之地区 / 零之实验室**：蓝之圆盘改写电梯，下至**零之下层**水晶洞。
- **北上乡·晶光池**（后日谈）：太乐巴戈斯引发时空薄雾，博士短暂现身。
- **关联**：朱紫主线学院、天星队、四天王会与冠军之路为前置条件。

**【世界剧情线】**

**① 前置条件**  
需完成《朱／紫》主线，并推进《碧之假面》主线后方可深入蓝之圆盘主线（可早进学园但剧情锁）。主角在北上乡与**极星、乌栗、白露**等结下因果：乌栗因欧尼尔普与对战挫败扭曲。

**② 留学蓝莓学园**  
园长西拉罗迎接交换生。生态巨蛋由白露设计的太晶核心维持太晶化。极星告知乌栗性格大变、对主角敌意化。**杜若**（四天王、前冠军）怂恿主角打 BB 联盟；四天王与冠军表决允许交换生直通四天王挑战。

**③ 精英试炼与四天王**  
四生物群各有试炼与四天王：  
- **蕾荷**：妖精系，审判感与秩序。  
- **青木**：火系，料理与热血。  
- **乌梅**：钢系，冷静精确。  
- **杜若**：龙系，懒散外皮下的实力与政治手腕。  
每胜一场，乌栗的执念与风评对照越刺眼。主题：学院把「变强」制度化，却无法制度化「为何变强」。

**④ 冠军战·乌栗崩溃**  
主角击败冠军乌栗。乌栗近乎精神崩溃——所有为赢主角而做的努力瞬间归零。极星焦虑，杜若冷眼，白露则在科研许可上取得进展。

**⑤ 蓝之圆盘与零之下层**  
白露获帕底亚联盟许可进入零之地区，目标证实希斯所记传说**太乐巴戈斯**。乌栗为再战胜主角同行欲收服传说；极星同行；杜若懒得去帕底亚。启程前，帕底亚冠军**也慈**与四天王**辛俐**交付关键道具**蓝之圆盘**——原属零之地区研究员的遗物。

在零号电梯，圆盘将行程改写至时间机器室更深层的**零之下层**。洞内出现**星晶**太晶属性（全属性复合）。最深处休眠结晶即太乐巴戈斯；苏醒后接近主角，乌栗拒绝「又一个传说被主角夺走」，用大师球强行收服。

**⑥ 失控与和解**  
白露催促对战以观察太乐巴戈斯。乌栗即使用出仍败。白露误判让乌栗对其太晶化，力量溢出、挣碎大师球。主角与极星联手仍被吸走太晶能量；极星力劝下乌栗终于协力，压制后主角正式收服。乌栗和解，一行返回蓝莓学园。白露出版远征记录《白露的书》赠主角。

**⑦ 晶光池后日谈**  
持书前往北上乡晶光池，太乐巴戈斯引发薄雾，过去时间点的**奥琳博士（朱）/弗图博士（紫）**现身；故勒顿/密勒顿喜见旧主。博士用《朱之书/紫之书》与《白露的书》交换后消失。主线意义上「零之秘宝」收束；其后可接《点心大骚动》桃歹郎线（本档点到）。

**⑧ 主题**  
变强若只为压过某人会空洞；传说不是奖杯；太晶是双刃；学院制度与个人执念对撞；和解需要一起承担失控。

敌友：对战对手可敬；强行大师球传说会引发伦理危机；契约者若盗卖星晶样本会被学园与联盟双通缉。

- 星晶暗示太晶本源与太乐巴戈斯一体。
- 蓝之圆盘证明零之设施有多层权限。
- 晶光池博士线连接主线时间机器伦理。
- 桃歹郎尾声与北上乡神话互文（点到）。

**【大事记时间线】**
朱紫主线毕业 → 碧之假面北上乡 → 受邀蓝莓学园 → 巨蛋与乌栗敌意 → 四天王表决 → 精英试炼与四天王战 → 击败冠军乌栗 → 也慈交付圆盘 → 零之下层 → 乌栗大师球 → 星晶失控 → 协力收服 → 返学园出书 → 晶光池博士 → （可选）点心大骚动。

**【学院日常与经济】**
蓝乐活任务换蓝莓点数；赞助打印机、投球方式、社团装饰；特邀教员；点心大叔点心引传说重逢。低阶契约者可完全沉浸学院生活而不碰零之。

**【对战美学与名场面】**
乌栗冠军战的压迫感在于「他为你而扭曲」；星晶战的视觉是全属性粒子风暴；大师球碎裂是制度与执念双重失败；极星喊乌栗出手是友情的最后保险丝。

**【与相邻条目边界】**
不重写朱紫三路线全文；不展开碧之假面全剧情，只保留乌栗因果；不写其他地区联盟史。关都神兽等可以点心大叔线遇见，不作主线。

**【契约者禁忌】**
盗走蓝之圆盘；在巨蛋破坏太晶核心；对乌栗使用洗脑式「必胜暗示」；把太乐巴戈斯卖给黑市；在晶光池强行绑架博士。失败反馈：学园除名、联盟通缉、星晶暴动。

**【叙事基调 · 雷区】**
基调：明亮学院色+地底水晶冷光；青春与克苏鲁式地底神秘并存。忌：乌栗永久黑化无和解；忌太乐巴戈斯工具化无形态名；忌四天王「群像」无名；忌修仙词。最早切入：刚下船换校服；或首场精英试炼；或圆盘交付时。

**【生态巨蛋四区细录】**
每区对应试炼逻辑：植被区适合耐久与变化招，海岸区测抗水与节奏，洞窟区测黑暗与钢岩，低温区测速度与冰龙。野生初伙伴回流让图鉴党与对战党目标分裂——契约者可做向导赚钱。

**【乌栗心理因果】**
北上乡：被忽视、被主角抢先、对传说的渴望。蓝莓：用冠军头衔证明。零之：用大师球作弊式兑现。终局：承认「一起打」比「独占」强。正文禁止把和解写成一句话原谅。

**【太晶与星晶规则翻译】**
太晶改属性与外观；星晶是全类型复合，克制关系特殊。乐园侧：星晶环境是规则场，不是可偷的永久 Buff。太乐巴戈斯可吸收他人太晶能量，使「太晶珠无限」策略失效。

## 阶位切入点

> 阶位↔原作：一阶≈学院新生/巨蛋野战；二阶≈精英试炼；三阶≈四天王门槛；四阶≈四天王~冠军；五阶≈零之下层；六阶≈太乐巴戈斯顶点条件战。顶点（太乐巴戈斯星晶失控）情报优先。

**一阶（交换生入学 · 校服与巨蛋）**
切入身份/时点：刚到蓝莓学园的交换生，乌栗敌意尚未全面爆发。
初始事件：西拉罗致辞后，杜若在走廊拦住你，用「要不要打联盟」的懒音把你推进社团目光。
开场白建议：「海底穹顶外是蓝色的压强。你拉了拉新校服的领口，极星在通讯器里嚷乌栗又对你放狠话——杜若已经笑着把报名板拍在你胸口。」
关键NPC立场：**西拉罗**要交流成果；**极星**护弟又护你；**杜若**要看热闹；**乌栗**要你输。
主线钩子/支线：熟悉巨蛋与蓝乐活；支线：打印第一件道具、同步器教程、食堂对战。
危险度/规避：低。规避零之、大师球冲突。
任务方向/奖励：蓝莓点数、初级球与伤药、地图。

**二阶（精英试炼 · 四区入门）**
切入身份/时点：通过表决后的挑战者，首场试炼前。
初始事件：蕾荷的试炼要求你在限时内完成「秩序课题」，失败则当日不可挑战四天王。
开场白建议：「试炼告示写着：迟到视同弃权。生态巨蛋的风把试卷吹起一角——你看见蕾荷的粉发在检录处一动不动。」
关键NPC立场：**蕾荷**讲规则；**青木**讲胃和火；**乌梅**讲数据。
主线钩子/支线：通过至少一项试炼；支线：收集BP、观察乌栗训练。
危险度/规避：中。规避连战消耗过度。
任务方向/奖励：试炼徽章、配招石、BP。

**三阶（四天王战 · 学院顶点门槛）**
切入身份/时点：四天王挑战周。
初始事件：杜若把龙队压到你最后一匹，观众席上乌栗不鼓掌只记招式。
开场白建议：「巨蛋的灯把场地照成白昼。杜若打了个哈欠，下一秒龙息擦过你的袖口——他懒的是表情，不是胜负。」
关键NPC立场：**四天王**各要你证明资格；**乌栗**要看你破绽；**极星**转播似的紧张。
主线钩子/支线：击败四天王；支线：联盟社装修、特邀教员。
危险度/规避：中高。规避未恢复就打冠军。
任务方向/奖励：三~四阶级队伍资源、龙抗用品。

**四阶（冠军乌栗 · 执念对决）**
切入身份/时点：冠军战预约日。
初始事件：乌栗先声夺人逼掉你的太晶，观众席极星喊暂停被裁判拒绝。
开场白建议：「乌栗的眼睛里没有北上乡的怯。他报出队伍时像在报一串仇恨的名字——你的名字在最后，也在全部。」
关键NPC立场：**乌栗**要碾压；**白露**在场边记数据；**西拉罗**要公平。
主线钩子/支线：夺冠；支线：战后是否递手帕；安抚极星。
危险度/规避：高。规避立刻接受零之同行无补给。
任务方向/奖励：冠军认证、四~五阶资源、圆盘线触发资格。

**五阶（零之下层 · 星晶地底）**
切入身份/时点：持蓝之圆盘的远征队员。
初始事件：电梯越过时间机器层，星晶光刺痛眼睛，野生太晶体围上来。
开场白建议：「圆盘嵌入槽位时发出图书馆还书似的轻响。下一秒，显示层数的数字变成无法朗读的符号——极星骂了一句，乌栗却笑了。」
关键NPC立场：**白露**要发现；**乌栗**要传说；**也慈**远程要安全。
主线钩子/支线：抵达最深处；支线：星晶样本登记、绘制下层地图。
危险度/规避：高。规避乌栗强行大师球前的无人监督。
任务方向/奖励：星晶道具、五阶级捕获资源。

**六阶（太乐巴戈斯 · 顶点条件战）**
切入身份/时点：结晶苏醒现场至失控镇压。
初始事件：大师球碎裂，太乐巴戈斯吸收你的太晶，极星被击退，你必须让乌栗选择「协力」或「继续独占」。
开场白建议：「龟壳上的晶棱像小型地平线。太乐巴戈斯看你，不像看敌人，像看失而复得的光源——乌栗的手还在抖，球已经不在了。」
关键NPC立场：**太乐巴戈斯**条件顶点；**乌栗**可敌可盟；**白露**要停手记录；**极星**要弟弟活。
主线钩子/支线：镇压并收服；支线：返校出书、晶光池后日谈。
危险度/规避：极高/贴近顶点。超阶式处理：存在·情报优先/条件性胜利；禁止无机制硬刚。
任务方向/奖励：太乐巴戈斯相关叙事、六阶条件认可、书与时空薄雾线索。
