# 宝可梦 白金

## 剧情

**【作品来源】**
《宝可梦 白金》（日文：ポケットモンスター プラチナ，英文：Pokémon Platinum Version）由 GAME FREAK 开发，任天堂与宝可梦公司发行。2008年9月13日日本发售于任天堂DS，北美2009年3月22日。本作是《宝可梦 钻石／珍珠》的强化扩展版（第三版本），封面传说为**骑拉帝纳**（起源形态）。文风在钻石珍珠的少年冒险上加重「反物质／毁坏的世界」神话压迫与国际警察暗线。本档案锚定**白金版主线**：时空二龙同召、骑拉帝纳拖拽赤日、毁坏的世界决战与通关后冥王线。

**【世界定位】**
舞台仍为**神奥地区**，但气候意象偏冬（双叶至雪峰可见积雪）、建筑与银河队设施全面改版。契约者主线：双叶镇少年从201号道路获最初伙伴，环神奥收集徽章，挫败**赤日**以红色锁链同召**帝牙卢卡**与**帕路奇亚**的计划，追随**骑拉帝纳**进入**毁坏的世界**，最终挑战联盟与冠军**竹兰**；通关后协助国际警察**帅哥**逮捕干部**冥王**。一句话：在反物质之龙的彼岸，用羁绊把「无感情新世界」从根上拆掉。

**【世界观 · 力量体系】**
力量本源与钻石珍珠一致：战力寄托于**宝可梦**对战（属性、物理/特殊按招式、特性、等级、个体）。白金追加关键维度——**毁坏的世界**（Distortion World）：时间与空间规则扭曲，重力方向不定，平台悬浮，是骑拉帝纳的领域；在此骑拉帝纳以**起源形态**（特性飘浮）现身，出界则为**别种形态**。死亡规则仍以昏厥可疗愈为主，但神话级失控可撕裂现实与反物质边界。特殊系统：精灵球、神奥图鉴（扩至210种）、宝可表、HM、超级华丽大赛、地下通道、对战开拓区（五设施）、对战记录器、全球终端、Vs.Recorder、帅哥调查线。

战力阶梯（宁低勿高）：
1. 未训练野生／见习：一阶。
2. 1～2徽章与普通训练家：二阶。
3. 3～5徽章中坚道馆（白金中梅丽莎提前为第三馆）：三阶。
4. 6～8徽章与银河干部／冥王技术线：四阶。
5. 四天王：五阶。
6. 冠军竹兰与对战开拓区顶点人类互研：六阶。
7. 湖之三神、帝牙卢卡、帕路奇亚、**骑拉帝纳（起源）**与毁坏世界权柄：七阶顶点。

乐园阶位映射：见习≈一阶；初级道馆≈二阶；中坚道馆≈三阶；高级道馆与银河干部≈四阶；四天王≈五阶；竹兰≈六阶；三龙与三神≈七阶。低阶切入规避毁坏的世界与骑拉帝纳本体·情报优先；顶点写条件性胜利，严禁写成「战力归零的弱化靶」。

**【地理 · 舞台】**
神奥骨架同钻石珍珠，但白金改观显著：双叶周边积雪；祝庆全球终端扩建；黑金炭坑输送带动画；花苑拱门；百代森林阴影加重；百代／帷幕银河建筑内外重设；自行车道加棚与灯；家缘互动广场扩建；失落之塔三层有雾；帷幕百货增地下层；野原湖畔酒店步道更新；天冠山神柱后连通**毁坏的世界**；通关后对战公园升级为**对战开拓区**（对战塔／工厂／城堡／轮盘／大厅）。道馆顺序变化：**梅丽莎**提前为第三馆（帷幕阿李与野原吉宪之前），遗迹徽章成为第三枚。钢铁岛怪力HM改由莱希交付；除雾HM改在随意遗迹。AI写场景先定：冬日乡镇 / 改版道馆谜题 / 银河实验室 / 神柱双龙 / 毁坏的世界平台战 / 开拓区，再定强度。

**【世界剧情线】**
① 开场与最初伙伴：主角在双叶镇收看山梨博士归神奥的新闻；**阿驯**闯门拉人去找博士要宝可梦。201号道路草丛前被山梨喝止「勿无防护入草」；助手**小光／小刚**到场后，山梨允许二人从草苗龟／小火焰猴／波加曼中选择。阿驯选克制属性并立即约战。战后二人赴心齐湖，意外遇见黑衣的**赤日**——他冷静观察湖面后离开，留下「感情无用」的阴影。回镇获母亲**茉莉**跑步鞋，真砂镇正式领取图鉴与伙伴确认，冒险开始。

② 祝庆～黑金：祝庆训练家学校、三小丑换**宝可表**；在此首次遇见国际警察**帅哥**（Looker），获**对战记录器**，他以夸张口音调查银河队。黑金炭坑找回**瓢太**，获煤炭徽章；碎岩打通花苑方向。

③ 花苑～百代：花苑银河队冲突，**火星**初败；百代森林可与**须行**同行。百代市遇**竹兰**获居合斩；草馆**菜种**后，突入改版**银河百代大楼**，对决**木星**，并获竹兰赠波克比蛋（替代钻石珍珠的幸福蛋线）。自行车道南下。

④ 家缘提前幽灵馆：白金将**梅丽莎**挑战前移为第三馆——家缘道馆以探照灯解谜推进，获遗迹徽章。此后才是帷幕格斗馆**阿李**、野原水馆**吉宪**等中后期馆。家缘可从**水梧桐**处提前获得伊布；帷幕市民可赠多边兽。

⑤ 帷幕～三湖：帷幕总部改版更深，帅哥伪装杂兵潜入。仓库钥匙链与飞翔等权限推进。南线野原、大湿原、秘传药驱鸭、神和壁画与冲浪、水脉钢馆**东瓜**、钢铁岛**莱希**与路卡利欧蛋等骨架保留但等级与对话强化。银河队对三湖实施爆破／强袭，以湖之传说力量结晶锻造**红色锁链**。水脉图书馆情报会合后，主角立志湖战**土星**、心齐湖再战火星；阿驯在锐智湖线苦战。

⑥ 雪峰～总部～神柱：雪峰**小菘**获冰柱徽章；攀岩开路。帷幕总部多层实验中可见干部**冥王**（Charon）的技术痕迹与「利用传说」的论文口吻。顶层战**赤日**，得大师球；赤日前往天冠山神柱。神柱上赤日以红色锁链**同时**召唤**帝牙卢卡**与**帕路奇亚**，时空碰撞令世界濒崩。湖之三神抑制时，反物质之影**骑拉帝纳**撕开裂口，将赤日拖入**毁坏的世界**。

⑦ 毁坏的世界：竹兰引导主角进入彼岸：平台倒置、瀑布倒流、重力换向。途中与赤日最终对峙——他仍坚持无感情完美世界，却在骑拉帝纳的领域中无法完成控制。主角与**起源形态骑拉帝纳**对战（可捕获或打倒）；世界边界稳固后返回现实。赤日消失／信念崩解；神奥天空恢复。

⑧ 滨海～联盟：停电解除后说服**电次**归馆获灯塔徽章；阿蜜赠瀑布。立志之路与阿驯终战，四天王**阿柳／菊野／大叶／悟松**，冠军**竹兰**（队伍强化）。殿堂登录。

⑨ 通关后：全国图鉴、宝可雷达、对战特区与**对战开拓区**五设施；神柱后可凭金刚宝珠／白玉宝珠（天冠山雪原洞窟取得）分别挑战帝牙卢卡与帕路奇亚；归途洞窟可再遇别种形态骑拉帝纳。生存区线：帅哥追查银河残党，最终在严酷山相关事件中与主角合力阻止**冥王**携席多蓝恩等研究逃离，逮捕冥王，银河队技术线收束。战斗区可触发与电次、大叶的双打等角色战。

⑩ 结局落点：白金把钻石珍珠的「单龙危机」升级为「双龙碰撞＋反物质介入」；主题仍是感情与羁绊对抗虚无，但增加「国际执法」与「科学家失控」两层。主角成冠军；骑拉帝纳危机解除；冥王落网；赤日理想破产。

**【叙事基调 · 雷区】**
冬日清冷＋神话诡谲；毁坏的世界要写出空间不安，不要写成普通洞窟。忌低阶硬刚起源骑拉帝纳；忌忽略道馆顺序与HM点变更；忌把帅哥写成纯搞笑无贡献；忌把冥王线提前到神柱前当主线终局；忌与珍珠档复读同一段开场而不提赤日早遇与双龙。最早切入：201号道路选伙伴，或祝庆遇帅哥。

**【世界运行细则 · 白金差异】**
道馆谜题全面改版（黑金除外）：百代花时钟、帷幕沙袋、家缘探照灯等。中后期训练家与劲敌等级整体上调，联盟部分等级回调。道馆队伍属性更纯。传说再战规则扩展：打倒静止传说后再次殿堂可刷新；漫游传说需重触触发条件。神柱不再直接放单龙，而需彼岸事件后持宝珠分别挑战。对战开拓区替代单纯对战塔，成为六阶人类互研主场。积雪与阴影树等视觉强化「白金之冬」。神奥日常仍靠训练家礼仪、中心医疗、地下挖宝、大赛宝芬运转；银河队出现后城镇对话从观光抱怨转为低声恐惧；双龙碰撞期会出现「天空折皱」「罗盘乱转」类异常描写。

主线钩子/支线：主线=图鉴启程；支线=冬日道路捕捉、向帅哥提交赤日素描。
危险度/规避：低——规避银河干部与任何传说。
任务方向/奖励：稳定御三家；初级球与伤药。

**二阶（黑金～花苑 · 煤炭与初银河）**
切入身份/时点：一徽章前后，花苑事件期。
初始事件：炭坑输送带卡住伤员，瓢太要你清小拳石；出坑后花苑花田被银河队踩出军靴印，火星在发电厂方向集结。
开场白建议：「煤灰沾在徽章盒上。花苑的风本来该有花香，现在只有扩音器里的『征用』二字。」
关键NPC立场：**瓢太**认可实干；**火星**视你为障碍；**帅哥**要求别打草惊蛇；镇民求你救人质宝可梦。
主线钩子/支线：主线=煤炭徽章→挫败火星；支线=发电厂外围侦察。
危险度/规避：中——规避百代大厦顶层。
任务方向/奖励：草系轴与驱虫药。

**三阶（百代～家缘 · 木星与提前幽灵馆）**
切入身份/时点：双徽章以上，锚定木星战后至梅丽莎第三馆。
初始事件：百代大楼警报时，冥王的实验终端闪过「锁链材料」字样；家缘道馆探照灯熄灭，梅丽莎的幽灵在暗中排队。
开场白建议：「大楼的G徽比钻石珍珠时期更刺眼。竹兰把波克比蛋放进你手里：『去看看家缘的灯。』」
关键NPC立场：**菜种**忧心生态；**木星**冷酷；**梅丽莎**以表演与惊吓考验；**竹兰**给蛋与谜语。
主线钩子/支线：主线=森林徽章→大厦→遗迹徽章；支线=伊布领取、互动广场。
危险度/规避：中高——规避红色锁链完成态。
任务方向/奖励：幽灵对策与中级配置。

**四阶（三湖～总部 · 锁链战争）**
切入身份/时点：五～七徽章，三湖爆破至总部。
初始事件：立志湖仪器抽出结晶，土星下令启爆；帅哥从通风管摔出，要你掩护他偷数据芯片。
开场白建议：「湖床裂开时，锁链的红光像血管。土星说『感情是噪声』，你的伙伴却在球里怒震。」
关键NPC立场：**土星**任务优先；**火星／木星**分湖施压；**阿驯**锐智湖硬刚；**帅哥**要证据链。
主线钩子/支线：主线=三湖救援→矿山／冰柱徽章→总部赤日；支线=钢铁岛、钥匙链。
危险度/规避：高——规避神柱双龙同在场。
任务方向/奖励：干部级情报与大师球叙事。

**五阶（联盟门槛 · 四天王）**
切入身份/时点：八馆后立志之路。
初始事件：假「开拓区招募员」骗取新人的对战录像设备；阿柳的虫群暴动与骗案同时发生。
开场白建议：「立志之路的雪比双叶更硬。八枚徽章冰凉，门后虫鸣如雨。」
关键NPC立场：**阿柳／菊野／大叶／悟松**各以属性压迫；骗案主使会咬出冥王余党线索。
主线钩子/支线：主线=四天王连战；支线=假招募案。
危险度/规避：高——规避未准备的冠军与彼岸残留裂缝。
任务方向/奖励：联盟战术升华。

**六阶（竹兰与开拓区 · 人类顶点）**
切入身份/时点：冠军战前后或通关后开拓区。
初始事件：竹兰问你「在毁坏的世界有没有放手」；若你曾弃同伴，她的烈咬陆鲨会更针对性。开拓区接待员同时递来五设施挑战书。
开场白建议：「冠军殿的灯落下，竹兰的眼睛像看穿彼岸。她说：『英雄会回来。』」
关键NPC立场：**竹兰**考官；**山梨**记录；**阿驯**再约；开拓区首领级训练家要连胜。
主线钩子/支线：主线=殿堂；支线=开拓区BP、电次大叶双打。
危险度/规避：极高——仍须规避失控的骑拉帝纳再临。
任务方向/奖励：冠军名分与开拓区资格。

**七阶（神柱与毁坏的世界 · 顶点）**
切入身份/时点：神柱双龙后、进入彼岸前后；或通关后再访起源骑拉帝纳。
初始事件：双龙碰撞撕开天空，骑拉帝纳的影把赤日与锁链一并吞入；竹兰抓住你的手腕：『跳。』毁坏世界中赤日仍在演讲，重力突然倒置。
开场白建议：「瀑布往上流。赤日的声音在倒置的殿堂里回响，骑拉帝纳的六影落在你脚边。现实像一层纸，而你站在纸的背面。」
关键NPC立场：**赤日**不接受劝说；**竹兰**要你活着带回答案；**骑拉帝纳**驱逐入侵者；**由克希／艾姆利多／亚克诺姆**远程抑制崩溃；**冥王**可能在现实侧趁乱偷数据。
主线钩子/支线：主线=彼岸稳定、对战/捕获起源骑拉帝纳、阻止控制仪式；支线=保护竹兰撤退点、回收锁链残骸、为通关后逮捕冥王留证据。
危险度/规避：贴近顶点——**超阶巅峰：存在·情报优先/条件性胜利**。低配走干扰与撤离，不写单挑创世权柄无代价。
任务方向/奖励：世界级因果、神话认可、白金宝珠相关叙事；不发日常刷龙奖励。
