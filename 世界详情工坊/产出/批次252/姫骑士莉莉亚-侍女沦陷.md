# 姫骑士莉莉亚-侍女沦陷
<!--meta lib=主库 tiers=三-->

## 剧情

**【作品来源】**
《姫骑士莉莉亚～魔触的王城堕落～》（姫騎士リリア 〜魔触の王城に堕つ〜）为 Lilith 旗下 BLACK Lilith 于 2004年8月28日（下载）/9月17日（包装）发售的成人向模拟/冒险游戏；2005年推出追加剧本的**完全版**；2006–2010年由 PIXY 推出全六卷OVA，累计销量号称约15万。气质：中世纪边境国政变、魔族契约对价、姫骑士意志与「魂之能量」征收。**本档案只写政治、战争、契约机制与人物因果，不写色情细节。** 本条目标题「侍女沦陷」锚定：莉莉亚身边忠实部下与宫廷侍从体系（含**米娅**等）在政变后被卷入调教棋盘的支线——主君未倒，侧翼先被拆。

**【世界定位】**
边境国**鲁芬（ルーフェン）王国**：表面上是可被大势力忽视的小国，实际上因「白之姫骑士」莉莉亚的武勋而在周边战局中有份量。故事从義兄**迪尔克·艾贝尔瓦因**借助魔族之力完全掌握王国写起——他要得到义妹莉莉亚与后母蕾拉，也必须向魔族支付「堕落时溢出的魂之能量」作为对价。一句话：这是「政变成功后的债务战争」，契约者切入的是对价倒计时下的王城，不是轻松王宫恋爱。

**【世界观 · 力量体系】**
力量本源：中世纪奇幻边境国，圣骑士系武技/圣光与魔族契约/触手魔触并存。力量靠血统、修炼、神器与魔族馈赠；「魂之能量（エナジー）」是魔族征收的硬通货——女性意志崩解时溢出，用于支付魔王对价。死亡：凡人与骑士战死即终；魔族将官级可长期存续；支付失败时签约者本人可被魔王索命。

境界/战力阶梯：
1. ** lev民/侍女/宫廷杂役**：无超凡。乐园≈一阶。
2. **士兵/白骑士见习/娼馆打手**：可清匪窝。乐园≈二阶。
3. **白之姫骑士·成型白骑士·娼馆主私兵队长**：一城精锐，可左右小国战场。乐园≈三阶。
4. **魔族将官（サンド级）·触手仪式主持·堕落强化体**：毁城一角、仪式级威胁。乐园≈四阶。
5. **魔王本体·边境国总动员级**：跨王国灾难。乐园≈五阶/触及超阶边缘。

乐园阶位映射： lev民≈一阶；士兵≈二阶；姫骑士全盛/白骑士主力≈三阶；魔族将官≈四阶；魔王≈五阶及超阶边缘。宁低勿高；本条目覆盖**三阶**为主。

特殊系统：
- **魂之能量征收**：魔族将官定期上门；数量与「目标意志崩解程度」挂钩。
- **日数限制**：对价有支付期限；逾期可索签约者性命。
- **调教/仪式菜单锁**：体力与精神阈值未到则部分行动不可执行——乐园中写成行动槽与检定，不写情色过程。
- **触手·魔物·鬼畜器具**：作为魔族侧「非人战力与仪式工具」存在，战场功能＝拘束、持续伤害、意志检定劣势。

**【地理 · 舞台】**
- **鲁芬王城**：主舞台。王座厅、公主居室、地牢、仪式厅、城墙与训练场构成「合法—非法」双层空间。政变后城门由迪尔克—キリコ人马控制。
- **黑之乐园馆（黒の楽園館）**：キリコ经营的娼馆，迪尔克早年出入之地；政变后升格为参谋总部与情报 hub，传闻有数千私兵。
- **克洛维斯战役旧战场（前史）**：莉莉亚成名地；可作为回忆与声望来源，不在正文展开长期战役。
- **邻国边境**：OVA终盘远征与离反战场；魔王军与鲁芬残存战力在此碰撞。
- **魔族侧领域（提及）**：魔王本座不轻易现身人间；通过将官サンド回收对价。
写场景优先：白甲反光、城石冷汗、契约羊皮纸、触手仪式阵纹、娼馆甜腥与铁血混杂的走廊。

**【世界剧情线】**
① **前史·白之姫骑士与废物皇子**  
莉莉亚在克洛维斯战役中立下武勋，白衣与容姿被讴为「白之姫骑士」。迪尔克早丧生母，沉溺享乐，被诸侯阴称废物皇子；对义妹怀有激烈劣等感。国王（游戏中病倒/动画中被刺）权威动摇，是政变窗口。

② **政变·魔族借力**  
迪尔克在娼馆主**キリコ**参谋下，向魔族借力完全掌握王国。没有魔族，他无法以个人武力压过莉莉亚。借力成功的瞬间，故事从「夺权」变成「还债」。

③ **对价条款生效（卡片锚点）**  
魔族要求：以女性堕落时溢出的魂之能量支付。支付对象首先指向莉莉亚（及并行的蕾拉线）。将官**サンド**定期出现回收。迪尔克若付不清，自身性命不保——残暴与恐慌并存。

④ **莉莉亚线：抵抗与阈值**  
莉莉亚作为一国之姫与屈强女战士，初期能承受显著精神与肉体负荷。系统/剧情要求「计划性」推进：体力上限、精神变化、可执行项目互相锁定。阶段名场面：她仍能怒斥迪尔克，却开始在仪式后出现判断迟滞；白骑士试图联络，城门却已换旗。

⑤ **蕾拉线：母后并行失陷**  
后母**蕾拉·艾贝尔瓦因**（「边境的至宝」）在莉莉亚不知情时被キリコ调教。母女信息差是迪尔克的武器：莉莉亚以为自己在独力支撑王国尊严，实际上家庭结构已被掏空。

⑥ **キリコ的双面**  
キリコ是魔族，亦是迪尔克的政治教师：教他邻国情势、王子心术，并提供贵族豪商人脉。路线不同：她可能继续掌控、被迪尔克反噬凌辱、或在魔王怒火下沦为苗床式惩罚（动画）——参谋不是永恒盟友。

⑦ **多结局分轨**  
- 支付失败：迪尔克被索命风险兑现；王城混乱。  
- 支付成功但莉莉亚崩坏：鲁芬名义存续，精神支柱成魔族资产。  
- 支付成功且迪尔克失控：动画式全国动员式凌辱后，对价表面上完成，王国却荒废，迪尔克精神崩溃。  
- OVA后段：莉莉亚失圣光能量后被魔王收为养女远征，途中宣言离反，蕾拉为人质，与米娅及白骑士残部对魔王军开战——**陷落之后仍可能再起**。

⑧ **微观政治贯穿**  
除大仪式外，正文高频：粮仓钥匙易主、城门换岗、白骑士名单删改、娼馆私兵进城「维持治安」、邻国使节被拦下、圣物库被贴魔族封条。AI优先这些小事件。

**【世界剧情线·侍女沦陷专轴】**
① **宫廷侧翼**：政变不只夺剑，还夺后勤、情报与贴身侍从。侍女/忠仆体系是莉莉亚的眼睛与手脚；先砍侧翼，公主会被迫独自决策。
② **米娅线**：米娅作为忠实部下兼好友，在调教期间仍试图关照莉莉亚——这种关照在敌方棋盘上会被利用为「软肋展示」。
③ **信息封锁**：侍女沦陷后，城外白骑士收不到真实军情；城内谣言由キリコ的人脉网书写。
④ **营救窗口**：契约者低中阶最易介入的是「救出一名侍从、带出一道密信、伪造一枚通行令」，而不是直接挑战魔王。
⑤ **连锁**：一名侍女失陷→其他侍从恐惧服从→莉莉亚的「宫廷意志」被架空。主线仍是公主与对价，支线是人走茶凉的速度。
⑥ **结局功能**：侍女线决定真结局时莉莉亚身边还剩几把可信的刀。

⑨ **蝴蝶效应边界**  
契约者可改局部：藏起一道支付记录、救出米娅、让一名白骑士带信出城、破坏一次仪式阵眼、挑拨ディルク与キリコ。不可无代价：魔王本体秒杀、让莉莉亚无痛保持「从未受创的完人」、抹掉对价机制。改了局部，魔族会加码下一期对价或换人质。

**【势力图谱】**

- **迪尔克—キリコ政权**：政变后实际统治；魔族债务绑定。
- **白骑士团 / 莉莉亚直属**：军事忠仆；主仆双堕的关键变量。
- **黑之乐园馆网**：情报、人脉、私兵。
- **魔王军 / サンド回收体系**：对价与终局压力源。
敌友：利益与恐惧优先；盟约在对价面前极脆；暴露「可支付的替代魂能来源」会立刻改变人质名单。

**【贵重物品】**
- **白之姫骑士甲与圣剑/骑枪（编制武装）**：莉莉亚身份与战力的外在符号；被缴＝政治宣告。

- **黑之乐园馆账本**：贵族黑料，可换同盟。

**【隐藏剧情 · 伏笔】**
- 迪尔克的恐慌：他不是无敌暴君，是怕死的债务人。
- 蕾拉线信息差：莉莉亚长期不知道母亲处境。
- キリコ可被反噬：参谋线不是单行道。
- OVA离反：陷落不是绝对终局，可写条件性再起。
- 完全版追加剧本与动画路线差异：入库以游戏骨架+OVA著名分歧互文，不把全媒体捏成无矛盾一条线。
- 魔王真实目的：不详处写不详；可见动机是魂能与强者收编。

**【大事记时间线】**
前史 → 克洛维斯战役，莉莉亚成名「白之姫骑士」。  
政变前夜 → 国王病倒/被刺；迪尔克与キリコ结盟魔族。  
政变日 → 王城易主；莉莉亚武力被魔族因素压制。  
对价期 → サンド回收；莉莉亚/蕾拉双线推进；白骑士分割。  
支付截止 → 多结局分轨：付清/未付清；莉莉亚状态分岐。  
OVA后段 → 莉莉亚魔王养女远征；宣言离反；与残部开战。  
终局状态 → 鲁芬或荒或存；魔族债务阴影不自动消失。

**【叙事基调 · 雷区】**
基调：阴暗宫廷奇幻、债务与意志、姫骑士尊严被制度性拆除；可写再起的铁与血。  
忌：把莉莉亚写成无脑从始至终崩坏的扁平角色；把迪尔克写成无敌魔王本体；低阶硬刚魔王；用「突然失忆变弱」解释白之姫骑士战力；展开原作色情描写。  
最早可切入锚点：政变当夜城门易帜，或对价第一次回收日前夕，或莉莉亚仍能公开巡城的最后一天。

**【王城债务战争细则·侍女沦陷】**
鲁芬是边境小国，却因「白之姫骑士」神话在周边战局有份量。迪尔克借魔族夺权的本质是：用无法偿还的高利贷购买「今晚的王位」。サンド每次出现都像银行催收——他不参与宫斗台词，只看账册数字。

行动槽模型（乐园化原作系统）：每日三条——宫廷公开（巡城/议事/接见使节）、地牢私刑（意志检定）、仪式厅（高风险高产出魂能）。体力阈值不够则仪式不可开；精神未跌破则部分项目锁死。契约者破坏任意一条槽，等于改写支付曲线。

莉莉亚的战斗力在政变夜被「魔族因素+里应外合」压制，不是面板归零。她仍是三阶天花板级个人战力；真正杀伤她的是时间、信息差与人质。蕾拉线并行推进时，莉莉亚的「独力支撑尊严」会变成讽刺——家庭结构已被掏空。

危险度/规避：高。サンド级将官不可硬刚；魔王＝**超阶巅峰：存在·情报优先/条件性胜利**。规避在无撤离路线时公开打出「我要杀魔王」。

任务方向/奖励：存活、带走一名关键NPC、破坏仪式阵眼或取得对价账册抄本；奖励＝三阶骑士装备、王室密道地图、白骑士余党联络权、反魔族圣水/阵具残件（不越级发「弑神武器」）。

**三阶补充·侍女沦陷深度钩子**

切入身份扩展：邻国雇佣骑士、未暴露的白骑士余党、黑之乐园馆灰色买办、魔族账册审计员、城内面包师（低权限高移动）。

初始事件扩展：你在下水道找到莉莉亚掉落的半枚白甲甲片，内侧刻着克洛维斯战役的纪念纹。同一夜，サンド的影子出现在议事厅窗外。迪尔克把支付期限表钉在墙上，像钉自己的墓志铭。

开场白建议（备用）：「甲片还留着战场的刮痕，锁链却是新的。你把甲片塞回掌心，听见姫骑士在塔楼骂废物皇子的声音——骂声比昨天短。账册上的数字比骂声长。你知道该砍的是锁链，还是数字。」

关键NPC扩展立场：**莉莉亚**——若你出示克洛维斯纹会多给你十秒信任；**迪尔克**——恐慌时比残暴时更危险；**キリコ**——开价「你的灵魂或你的刀」；**サンド**——只认账；**米娅**——把你当可能的信使；**四白骑士残部**——要口令不要热血演说。

主线钩子扩展：推迟一期对价；救出米娅；带走两名白骑士；挑拨ディルク与キリコ；给邻国递真实报告换边境接应。

危险度扩展：高。将官不可硬刚；魔王＝超阶边缘情报优先。规避在无撤离时公开弑君/弑魔王宣言。

任务方向扩展：阵眼破坏工具、对价账册抄本、密道地图、反魔族圣水残瓶、一次「被莉莉亚点名同行」的资格。


**切入点密度补述·姫骑士莉莉亚-侍女沦陷·1**
开场必须有物件钩子（账本/铃舌/甲片/地图/药瓶）+人名+抉择。关键NPC全部真名加粗，各附一句图什么/怕什么。主线钩子写原作节点名；支线写本阶独有，禁止复制「假货护送名额」套话。危险度写清规避对象；涉及顶点写「超阶巅峰：存在·情报优先/条件性胜利」。任务奖励贴合本阶：通行、情报、残件、人脉，不越级。
