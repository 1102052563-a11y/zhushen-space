# 姫骑士莉莉亚-妹妹堕落
<!--meta lib=主库 tiers=三-->
## 剧情

**【作品来源】**
《姫骑士莉莉亚～魔触的王城堕落～》（姫騎士リリア 〜魔触の王城に堕つ〜）为 Lilith 旗下 **BLACK Lilith** 于 2004年8月28日（下载）/9月17日（包装）发售的成人向模拟/冒险游戏；2005年推出追加剧本的**完全版**（DLsite 可见 RJ018305 等）；2006–2010年由 **PIXY** 推出全六卷 OVA，累计销量号称约15万。气质：中世纪边境国政变、魔族契约对价、姫骑士意志与「魂之能量」征收。**本档案只写政治、战争、契约机制与人物因果，不写色情细节。** 本条目标题「妹妹堕落」锚定：义兄义妹结构

**【世界定位】**
边境国**鲁芬（ルーフェン）王国**：表面上可被大势力忽视的小国，实际上因「白之姫骑士」**莉莉亚·艾贝尔瓦因**的武勋而在周边战局中有份量。故事从義兄**迪尔克·艾贝尔瓦因**借助魔族之力完全掌握王国写起——他要得到义妹莉莉亚与后母蕾拉，也必须向魔族支付「堕落时溢出的魂之能量」作为对价。一句话：这是「政变成功后的债务战争」，契约者切入的是对价倒计时下的王城，不是轻松王宫恋爱。

**【世界观 · 力量体系】**
力量本源：中世纪奇幻边境国，圣骑士系武技/圣光与魔族契约/触手魔触并存。力量靠血统、修炼、神器与魔族馈赠；「魂之能量（エナジー）」是魔族征收的硬通货——女性意志崩解时溢出，用于支付魔王对价。死亡：凡人与骑士战死即终；魔族将官级可长期存续；支付失败时签约者本人可被魔王索命。

境界/战力阶梯：
1. ** lev民/侍女/宫廷杂役**：无超凡。乐园≈一阶。
2. **士兵/白骑士见习/娼馆打手**：可清匪窝。乐园≈二阶。
3. **白之姫骑士·成型白骑士·娼馆主私兵队长**：一城精锐，可左右小国战场。乐园≈三阶。
4. **魔族将官（サンド级）·触手仪式主持·堕落强化体**：毁城一角、仪式级威胁。乐园≈四阶。
5. **魔王本体·边境国总动员级**：跨王国灾难。乐园≈五阶/触及超阶边缘。

乐园阶位映射：平民≈一阶；士兵≈二阶；姫骑士全盛/白骑士主力≈三阶；魔族将官≈四阶；魔王≈五阶及超阶边缘。宁低勿高；本条目覆盖**三阶**为主。

特殊系统：
- **魂之能量征收**：魔族将官定期上门；数量与「目标意志崩解程度」挂钩。
- **日数限制**：对价有支付期限；逾期可索签约者性命。
- **行动菜单锁**：体力与精神阈值未到则部分行动不可执行——乐园中写成行动槽与检定，不写情色过程。
- **触手·魔物·拘束器具**：作为魔族侧「非人战力与仪式工具」存在，战场功能＝拘束、持续伤害、意志检定劣势。

**【地理 · 舞台】**
- **鲁芬王城**：主舞台。王座厅、公主居室、地牢、仪式厅、城墙与训练场构成「合法—非法」双层空间。政变后城门由迪尔克—キリコ人马控制。
- **黑之乐园馆（黒の楽園館）**：キリコ经营的娼馆，迪尔克早年出入之地；政变后升格为参谋总部与情报中枢，传闻有数千私兵。
- **克洛维斯战役旧战场（前史）**：莉莉亚成名地；可作为回忆与声望来源，不在正文展开长期战役。
- **邻国边境**：OVA 终盘远征与离反战场；魔王军与鲁芬残存战力在此碰撞。
- **魔族侧领域（提及）**：魔王本座不轻易现身人间；通过将官サンド回收对价。
写场景优先：白甲反光、城石冷汗、契约羊皮纸、触手仪式阵纹、娼馆甜腥与铁血混杂的走廊、支付期限表钉在墙上的锈钉。

**【世界剧情线】**
① **前史·白之姫骑士与废物皇子**  
莉莉亚在克洛维斯战役中立下武勋，白衣与容姿被讴为「白之姫骑士」。迪尔克早丧生母，沉溺享乐，被诸侯阴称废物皇子；对义妹怀有激烈劣等感。国王（游戏中病倒/动画中被刺）权威动摇，是政变窗口。

② **政变·魔族借力**  
迪尔克在娼馆主**キリコ**参谋下，向魔族借力完全掌握王国。没有魔族，他无法以个人武力压过莉莉亚。借力成功的瞬间，故事从「夺权」变成「还债」。

③ **对价条款生效（卡片锚点）**  
魔族要求：以女性堕落时溢出的魂之能量支付。支付对象首先指向莉莉亚（及并行的蕾拉线）。将官**サンド**定期出现回收。迪尔克若付不清，自身性命不保——残暴与恐慌并存。

④ **莉莉亚线：抵抗与阈值**  
莉莉亚作为一国之姫与屈强女战士，初期能承受显著精神与肉体负荷。系统/剧情要求「计划性」推进：体力上限、精神变化、可执行项目互相锁定。阶段名场面：她仍能怒斥迪尔克，却开始在仪式后出现判断迟滞；白骑士试图联络，城门却已换旗。

⑤ **蕾拉线：母后并行失陷**  
后母**蕾拉·艾贝尔瓦因**（「边境的至宝」）在莉莉亚不知情时被キリコ调教。母女消息隔绝是迪尔克的武器：莉莉亚以为自己在独力支撑王国尊严，实际上家庭结构已被掏空。

⑥ **キリコ的双面**  
キリコ是魔族，亦是迪尔克的政治教师：教他邻国情势、王子心术，并提供贵族豪商人脉。路线不同：她可能继续掌控、被迪尔克反噬、或在魔王怒火下受惩罚（动画）——参谋不是永恒盟友。

⑦ **多结局分轨**  
- 支付失败：迪尔克被索命风险兑现；王城混乱。  
- 支付成功但莉莉亚崩坏：鲁芬名义存续，精神支柱成魔族资产。  
- 支付成功且迪尔克失控：动画式全国动员后，对价表面上完成，王国却荒废，迪尔克精神崩溃。  
- OVA后段：莉莉亚失圣光能量后被魔王收为养女远征，途中宣言离反，蕾拉为人质，与米娅及白骑士残部对魔王军开战——**陷落之后仍可能再起**。

⑧ **微观政治贯穿**  
除大仪式外，正文高频：粮仓钥匙易主、城门换岗、白骑士名单删改、娼馆私兵进城「维持治安」、邻国使节被拦下、圣物库被贴魔族封条。AI优先这些小事件。

**【世界剧情线·妹妹堕落专轴】**
① **义兄义妹结构**：迪尔克早丧生母，被诸侯阴称「废物皇子」；莉莉亚是后妻蕾拉所出的义妹，武勋与声望全面压过義兄。劣等感是政变引擎。  
② **「妹妹」作为武器**：迪尔克要的不只是王位，是确认「被看不起的人终于能支配看不起他的人」。每次称呼都在完成所有权确认。  
③ **蕾拉线并行**：后母在莉莉亚不知情时被キリコ调教——母女双线并行，使「妹妹」在家庭结构上孤立。  
④ **国王之死版本**：游戏中国王病倒；动画中迪尔克刺死国王。弑君版正统性更脆，恐慌更高。  
⑤ **支付与恐惧**：迪尔克并非无敌魔王，他怕支付失败被杀——妹妹线里残暴常混杂恐慌。  
⑥ **终局讽刺**：即使对价付清，鲁芬也可能崩溃；「得到妹妹也失去王国」是核心讽刺。  
⑦ **反击方式**：公开旧嘲讽书信、引入诸侯证人、保护法理身份文件、让称呼在公共场合变成丑闻。

⑨ **蝴蝶效应边界**  
契约者可改局部：藏起一道支付记录、救出米娅、让一名白骑士带信出城、破坏一次仪式阵眼、挑拨迪尔克与キリコ。不可无代价：魔王本体秒杀、让莉莉亚无痛保持「从未受创的完人」、抹掉对价机制。改了局部，魔族会加码下一期对价或换人质。

**【主要人物】**
- **莉莉亚·艾贝尔瓦因（リリア・エーベルヴァイン）**｜身份：鲁芬王女，「白之姫骑士」，本作核心女主角。｜性格：高傲、坚韧、蔑视迪尔克的无能与卑劣；对王国与部下有责任感；被逼到阈值后仍可能选择再战而非彻底跪服（视结局）。｜装备·能力：圣骑士级武技与白甲；战役级个人武力，边境国天花板之一。｜人物弧光：神话姫骑士→政变囚徒→魂能被征收的对象→多结局下的崩坏者或离反再起者。｜立场关系：迪尔克义妹兼仇敌；蕾拉之女；米娅与白骑士的主君；魔王侧可能的养女/敌将。

- **迪尔克·艾贝尔瓦因（ディルク・エーベルヴァイン）**｜身份：鲁芬王子，游戏视角主人公，政变主导者。｜性格：享乐、自卑、记恨、恐慌与残暴混杂；被诸侯看不起后把支配欲投射到莉莉亚身上。｜装备·能力：个人武斗远弱于莉莉亚，靠魔族与キリコ人脉。｜人物弧光：废物皇子→借魔族夺权→对价债务人→支付成功也可能精神崩溃。｜立场关系：莉莉亚義兄；キリコ合作者/仇怨对象；サンド的催债对象。

- **蕾拉·艾贝尔瓦因（レイラ・エーベルヴァイン）**｜身份：国王后妻，莉莉亚生母，「边境的至宝」。｜性格：端丽、作为王太后的尊严；被袭后成为迪尔克政变的导火线之一。｜装备·能力：无前线武勋，价值在政治象征与魔族眼中的「高纯度魂能来源」。｜人物弧光：后妃→被袭→并行调教线→动画中成魔王侧人质。｜立场关系：莉莉亚至亲软肋；迪尔克的欲望对象之一。

- **キリコ**｜身份：黑之乐园馆女主人，迪尔克参谋，魔族。｜性格：老练、野心、教唆与利用并重；被背叛后可转为仇恨。｜装备·能力：魔族手段、情报网、传闻中的私兵；政治操作。｜人物弧光：娼馆主→政变参谋→或反噬或被魔王惩罚。｜立场关系：迪尔克的刀与教师；莉莉亚的间接迫害者。

- **サンド**｜身份：魔王使者/将官级回收人。｜性格：冷静执行契约。｜装备·能力：将官级魔族规格，人间谈判代表。｜人物弧光：定期出现的「债务闹钟」。｜立场关系：对迪尔克是催债者；对莉莉亚是系统级威胁的脸。

- **米娅（ミーア）**｜身份：莉莉亚忠实部下兼好友。｜性格：忠诚、共情、在高压下仍试图保护主君尊严。｜装备·能力：骑士/侍从级，弱于莉莉亚但强于普通兵。｜人物弧光：侧翼支柱→可能沦陷→OVA中随离反再战。｜立场关系：主仆与友情双重绑定。

- **弗兰克 / 沙姆洛克 / 卢滕 / 梅策尔**｜身份：莉莉亚麾下四白骑士（フランク / シャムロック / ルッテン / メッツァー）。｜性格：忠勇；卢滕线有重伤仍被救回的羁绊样本。｜装备·能力：白骑士编制，三阶协作战力。｜人物弧光：政变中被分割→或随魔王军编制→离反时再集结。｜立场关系：主仆结构的核心棋子。

- **鲁芬国王**｜身份：边境国君。｜性格：权威象征；游戏中病倒，动画中被迪尔克刺杀。｜弧光：权威真空制造政变窗口。｜立场：死后其名仍可被双方用于正统性宣传。

- **魔王**｜身份：对价体系顶点。｜能力：可收莉莉亚为养女、以蕾拉为人质、发动远征军。｜立场：超阶边缘；低阶只做情报与条件性周旋。

- **黑之乐园馆打手/私兵**｜身份：キリコ武力。｜性格：佣兵逻辑。｜能力：二～三阶群战。｜立场：城内治安的灰色替代品。

- **邻国使节/诸侯代表**｜身份：观望鲁芬内乱者。｜性格：趁火打劫或划清界限。｜立场：可买可卖情报。

- **魔族将官—魔王体系**：对价与催收。
- **邻国与诸侯**：观望、买情报、必要时出兵。

- **支付期限表**：钉在墙上的倒计时。
- **克洛维斯纪念甲片**：可换信任的物件钩子。

**【隐藏剧情 · 伏笔】**
- 母女消息隔绝是武器，不是疏忽。
- 国王病倒/被刺两种时间线改变迪尔克正统性脆度。
- キリコ可被反噬：参谋不是永恒盟友。
- OVA 离反证明：陷落之后仍可能再起。
- ルッテン重伤被救：主仆羁绊的肉体代价样本。

**【大事记时间线】**
克洛维斯武勋 → 莉莉亚成名。  
国王权威动摇 → 政变窗口。  
魔族借力成功 → 对价条款生效。  
第一次サンド回收 → 债务闹钟响起。  
妹妹堕落节点 → 见专轴。  
多结局分轨 → 支付失败/成功/失控/离反再起。  
余波 → 邻国态度、魔王阴影、荒城或再战。

**【叙事基调 · 雷区】**
基调：政变债务、意志阈值、主仆伦理、边境国残酷。战斗写白甲、锁链、账册与阵纹。  
忌：把调教过程写成色情表演；把迪尔克写成无敌魔王（他怕支付失败）；无代价净化莉莉亚；低阶硬刚サンド/魔王；抹掉对价机制。  
最早可切入：政变已成功、第一次对价回收前后，莉莉亚仍具部分行动能力之时。

**【对价战争作战要点】**
本世界胜负关键不是单挑迪尔克，是**支付日与账册**。迪尔克武斗远弱于莉莉亚，却因魔族借力与人质结构而不可仅用剑解决。契约者应像审计官与渗透者一样行动：改数字、抢时间、拆阵眼、挑拨盟友。

日数压力下的王城气氛：粮仓钥匙易主、城门换岗、使节被拦、圣物库封条。把这些写成每日新闻，AI 就有的写。莉莉亚的巡城若还被允许，那是迪尔克的自信或キリコ的诱饵；若不允许，说明意志条已让他害怕公开场合。

真结局/再起结局不要写成无伤白凯旋。白甲可以回来，裂痕应留下。魂能被征收过的身体在战斗中可能出现短暂眩晕——作为代价，而不是战力归零。魔王侧若仍存在，债务阴影不自动消失。

**【妹妹堕落·可观察现场】**
写正文优先：人名、物件、门是否还开着、信号是否还亮、账册是否还在、锁链是否还扣着。契约者每做一个选择，就改写局部因果，并准备回流的追杀、催收或丑闻。

物品奖励宁低勿高：密道地图、账册抄本、圣水残瓶、一次被莉莉亚点名同行的资格。禁止发「弑神武器/永久精神免疫」。

结局检查：谁活着，谁欠债，谁的编制还在，下一期催收何时来。

**【妹妹堕落·王城日程样例】**
黎明：城门换岗，キリコ私兵混入「维持治安」。  
午前：サンド的影子出现在议事厅窗外，迪尔克核对支付期限表。  
午后：仪式厅清场，菜单锁检查体力与精神阈值。
