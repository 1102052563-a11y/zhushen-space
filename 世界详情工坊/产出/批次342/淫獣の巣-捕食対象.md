# 淫獣の巣-捕食対象

## 剧情

**【作品来源】**
《淫獣の巣-捕食対象》为轮回乐园休闲库收录的**巢穴驿站·「捕食」隐喻与可退出契约**情景档案。清单标题点出「淫獣／巣／捕食対象」母题，**无与标题逐字对应的单一出版长篇**；气质整合 DLsite 同人圈常见的「淫獣の巣」「魔物娘の館」「异种巢穴」公开作品惯例（如相关关键词检索页与《淫獣の巣の中で》等短篇标题氛围）。媒介印象：同人 CG／音声／短篇 RPG 跨媒介；搜笔趣阁检索本条目标题无对应长篇小说书页。本档案把「捕食」**改写**为：巢穴以契约换取体温与陪伴的古老隐喻——可协商、可中止、可离开——核心是**谁被当成「对象」编号，谁被当成名字**，不写猎杀细目，不写强制吞食美化。写作时请把恐惧放在「制度默认」而不是「血盆大口特写」：真正吓人的是名簿栏位与藏起的绳，而不是无意义的残虐展览。人物冲突应能在围炉旁用一碗汤的时间讲清立场，也能在竖井口用一场绳检验证信任。

**【世界定位】**
奇幻边陲雾林深处的「潮痕巢」：对外是魔物巢穴传说，对内是一处供旅人与「兽形眷属」共住的地下驿站。契约者被雾气引入时，名簿上默认栏位写着「捕食対象候补」——你的第一份工作就是把这三个字划掉，改成「访客」。一句话：在潮气与兽瞳之间，用**可撕毁的名簿与出口绳**证明没有人必须成为食物。适合想写边界、命名、共同体改革与慢热羁绊的契约者入世体验；也适合想把「怪物传说」拆成邻里政治的玩家。

**【世界观 · 舞台设定】**
时代感为剑与魔法世界的边陲日常：镇上人把雾林当禁地，猎人和药草师偶尔失踪的故事被夸大成「被巢吃掉」。巢内真相更接近**另类共同体**：兽形住民需要契约确认的亲密与魔力分享（写成可协商的体温交换与共眠守护），人类旅人需要过夜与疗伤；冲突来自旧规矩把人类一律登记为「対象」。

「淫獣」在本档案中的公开形象与内部自称分裂：对外是恐吓用的传说外壳，对内是自嘲式族名——他们知道人类故事里自己是嘴与爪，于是有人选择把传说当铠甲，有人选择把传说当耻辱。你的入巢正好撞上「要不要继续靠恐吓活下去」的意识形态分裂，而不是一场猎杀游戏的开局动画。

软规则：
1. **名簿可改**：任何「対象／猎物」字样必须可被本人或见证人划改；不可用血契永久锁死。  
2. **出口绳常亮**：巢穴主通道挂着夜光苔绳通向地表；绳断或被藏＝危险信号。  
3. **停词有效**：默认停词「潮退」；一喊即停一切亲密与登记仪式。  
4. **禁锁死侧室**：休息室门不得从外焊死；第二钥匙在医寮。  
5. **禁幼化**：全员成年眷属与旅人。
6. **交易优先于恐吓**：与巢外人类的接触以药草、盐、布、情报互换为默认路径；任何「抓一个示众」提案都应被公约否决。

世界的温度来自：石壁渗水的滴答、炖兽肉汤的油脂香、有人把你的名字用炭笔写在名簿正栏而不是脚注、出口绳在指间发亮的触感。也来自一些很小的秩序：汤勺有专属挂钉、湿尾巴不许甩进药柜、人类访客的鞋子要在门槛石上磕两下——这些规矩不浪漫，却证明这里是「住」的地方，不是临时屠场布景。

**【地理 · 生活舞台】**
- **名簿厅**：入巢第一间。石台、炭笔、旧皮名簿、墙上挂着已划改成功的访客名条（芙恩的那张还在最边上）。灯光来自嵌壁萤石，偏冷，适合让人看清自己被写成什么。名场面：划掉「捕食対象」、公约签字、烧纸前的默哀式沉默。  
- **潮痕主巢厅**：壁炉与兽皮垫；晚间围炉是情感主舞台。吊锅常年挂着，汤的味道会泄露当天的情绪——盐多是凯拉值夜，药草多是芙恩回来了。辩论、道歉、起哄式的温柔都在这里发生。  
- **侧室走廊**：莉露、奥萝、凯拉、访客间成排；门上挂着可翻转的「可访／勿扰」牌。牌是弥莎做的，背面写着「敲三下，等三息」。走廊潮气重，说话声发闷，适合秘密也适合误会。  
- **医寮**：弥莎的领地。药柜按「外用／内服／停词相关」分格；第二钥匙在第二抽屉。伤员躺椅旁有一盆不需要太阳的苔，她说「看着它就知道绳还亮不亮」——其实是自己吓自己的玩笑。  
- **出口竖井**：苔绳、铁钉、地表雾洞；每日点名确认绳在。竖井壁上有旧爪痕与新刻的箭头，箭头是莉露画的「给人看的路」。雨天竖井会滴成帘，走的人必须牵绳。  
- **雾林外围猎人小屋（巢外）**：人类侧补给点；木墙钉着兽爪恐吓牌（多半是假的）。哈根在此秤盐。可写「回小镇买盐、听谣言、澄清失踪故事」的日常，防止世界缩成纯密闭巢。  
- **地下温泉脉**：共浴可选，但须先停词演练；池边石上刻着「先说潮退」。强调边界与放松，而非猎奇展览。  
- **储藏侧洞**：干肉、药草、破损的旧锁链（奥萝故意留着当反面教材：「我们用过这种东西，所以更不能再用」）。  

地理的情感功能可以记成一句话：**名簿厅决定你是谁，主巢厅决定你们怎么吵，竖井决定你还能不能走，医寮决定疼的时候有没有人问停。** 若正文需要快速建立空间记忆，可让角色用「去 induct 名簿／去盛汤／去数绳」代替抽象的「去谈恋爱」，情感会更贴地。

**【故事主线 · 情感线】**

**共通线·名簿上的三个字**  
你在雾中失温，被兽耳少女莉露拖入巢。意识清醒时，见巢主奥萝把你的名字写在「捕食対象」栏——她的动作并不残忍，甚至带着一种「按旧法办事」的疲惫。医寮弥莎把停词卡塞进你手里：先学会喊停，再谈住。你在第一碗汤见底前就明白：这里的恐怖不是立刻被吃，而是**系统默认你可被吃**。

你发现巢内一半住民反对旧栏——他们经历过被人类猎人当标本的日子，不愿再以「对象」回敬人类。另一半（以凯拉为代表）认为不写「対象」就无法震慑盗猎者。围炉辩论的声音会震动石壁上的水珠。七日议题（可作日程骨架）：  
- 第 1 日：改自己的栏；学停词；见出口绳。  
- 第 2 日：旁听旧法派与改册派第一次吵架；汤被打翻。  
- 第 3 日：出口绳夜检，发现有人把绳尾塞进石缝「藏好」。  
- 第 4 日：巢外猎人小屋接触；盐与偏见同时出现。  
- 第 5 日：公约草案一读；凯拉离席。  
- 第 6 日：弥莎停词全员建档；奥萝公开布条故事。  
- 第 7 日：表决；烧纸或回潮。

共通线的情感主轴不是「你征服巢穴」，而是**你是否坚持每个人都该有名字**。你若用「我也可以成为顶级対象／反向捕食者」逻辑回击旧法，叙事应判 OOC 并滑向 BE：那只是把刀换了个方向。

**莉露线（救你入巢的向导）**  
兽耳、话多、怕你怕黑。她把你拖进来时用的是「不能让人死在我的雾里」而不是「捡到食物」。个人线要处理她的愧疚：旧法下她救过的人有的逃走后带回猎人，有的留下来却一直叫她「怪物」。你与她走熟出口绳每一段——竖井、侧岔、会湿鞋的浅潭——并把「我不是把你拖来吃」说清楚。名场面：她在地表雾洞口把护身符塞给你，又立刻收回，说「你要是怕我就扔了」。HE：她成为向导公会与巢的双籍联络人，你的名字绣进护身符内侧。BE：她被旧派强迫执行「対象」仪式；你若见死不救或冷眼旁观，线冻结为互相回避。

**奥萝线（巢主）**  
沉稳、角上绑旧布条（故去同僚的遗物——只作纪念，不写强弱）。她想改规矩又怕巢散。个人线是政治与私情的双线：白天你帮她起草《潮痕居留公约》，把「捕食」定义为「自愿的守护性共处」并加退出章；夜里她只在出口绳旁抽烟叶（无害草），承认自己害怕「一温柔就会死」。HE：公约通过，她卸下「必须凶」的表演，布条仍在但不再当权杖。BE：她为稳固权力恢复対象栏，你被驱逐或关系破裂；她的台词会很平静，这比咆哮更冷。

**凯拉线（旧派狩卫）**  
刺头、反对改册。真正恐惧是「一旦示弱，猎人会灌巢」。她巡夜时步子很重，故意让石壁听到，好让自己也相信「我很强」。个人线要你陪她去巢外与猎人小屋谈判：用贸易（药草换盐）代替恐吓。过程中她可能对你发火、把你推到树干上又立刻松手——停词在此同样有效。HE：她在公约上签字但保留巡夜权，学会把警戒对准盗猎与设阱者，而不是对准迷路访客；签字时手抖是名场面。BE：她藏起出口绳，线必须中止亲密推进并标危险，直到绳被找回并公开道歉。

**弥莎线（医寮）**  
冷静、管停词与伤药。她的桌子永远比主巢厅更亮一点，因为她坚持「看清伤口才能包」。个人线：为每位住民建「停词偏好」档案，发现自己的停词从未被登记——大家都默认医官不会需要停。HE：她成为公约监察，第一次有人在给她递工具前问「可以说吗」。BE：监察职被撤，她带走全部药箱离开，巢里的伤只能用旧法硬扛。

**芙恩线（人类药草师·长期访客）**  
曾被登记为対象后划改成功的前辈。提供「过来人」视角：她知道改册后的羞耻、愤怒与奇妙的轻松。可攻略为知己或恋人，戏多在雾林外围采药与晒药。HE：她教你如何在恐惧中仍要求名字，并留下半本空白名簿给你继续写公约附件。BE：她因旧伤复发离开，只在门槛石上用炭写「别让他们把栏加回来」。

**微观日常事件池**  
汤盐不够、苔绳发暗要换、兽尾扫翻墨水、人类访客鞋子湿滑、凯拉与莉露抢巡夜班、弥莎没收过期药、芙恩的药筛坏了、奥萝在名簿厅对着空白页坐一整夜。每件事都可推进信任：关键是你有没有把任何人的恐惧当笑话。

**结局**
- **公约 HE**：名簿废除対象栏；出口绳双人保管；围炉用真名点汤。  
- **贸易 HE**：与猎人小屋互市，巢不再靠恐吓生存；凯拉学会在交易桌而不是树后出现。  
- **离开 HE**：你划改自己名字后体面离巢，绳仍亮；莉露送到地表不再拉袖口。  
- **BE**：绳被藏、名簿用血填；叙事冷处理，不给「黑暗正确」光环。  
- **True**：対象栏被撕下做成壁炉引火纸；全员用真名点名；你可留任见证人；盐袋与药草袋并挂在门槛两侧。

**【可攻略角色 / 主要人物】**

- **莉露（向导眷属）**｜外貌：浅棕兽耳兽尾，齐肩发微卷，皮甲补丁多，左耳有缺一小口｜性格：热心、心虚、怕被恨，紧张时尾巴先说话｜角色类型：元气向导／愧疚救赎｜萌点/魅力：数出口绳节数会念出声；给人汤前先吹凉；说谎时耳尖发红｜个人线剧情：证明拖你来不是为了吃；处理「救人反被猎」的创伤；弧光是敢在地表用真名自我介绍｜与主角关系：救命恩人 → 向导搭档 → 恋人可选项

- **奥萝（巢主）**｜外貌：黑角微弯，长银发常束，眼金，披兽皮披风，角根布条旧成灰褐｜性格：沉、倦、要强，笑起来像在道歉｜角色类型：引退边缘的领导者／改革犹豫者｜萌点/魅力：改公约时字迹极正；会把最大的肉留给伤员；布条松了自己不敢让人重绑｜个人线剧情：从恐吓统治到公约统治；公开友人被示众的过去；弧光是允许自己被需要而不是被害怕｜与主角关系：登记官 → 共笔者 → 可成深度羁绊（非必须恋爱）

- **凯拉（狩卫）**｜外貌：赤短发，牙尖外露一点，臂环铜绿，巡夜披硬皮｜性格：暴、直、怕示弱，被认真倾听会怒｜角色类型：傲娇反对派／恐惧武装者｜萌点/魅力：签字手抖；把盐袋抱得很紧；骂完人会留下烤薯｜个人线剧情：贸易代恐吓；学会区分盗猎与迷路者；弧光是把刀插在门边而不是指向访客｜与主角关系：对立 → 谈判搭档 → 战友式信任（可恋爱）

- **弥莎（医寮）**｜外貌：白袍改兽皮围裙，细框眼镜，指尖常有药渍，翼或耳饰极简｜性格：冷、准、底线硬，对夸赞过敏｜角色类型：监察医／隐形守门人｜萌点/魅力：停词卡手写体一丝不苟；会把苦药混进甜浆；被问「你的停词呢」时第一次卡壳｜个人线剧情：被登记自己的停词；建立可审计伤情与同意档案；弧光是承认医官也需要被保护｜与主角关系：同事 → 静默羁绊 → 可选恋

- **芙恩（药草师·人类）**｜外貌：人耳，风雨斗篷，指有草渍，锁骨旧疤一条｜性格：淡、幽默、有旧伤，不美化自己的幸存｜角色类型：前辈旅人／镜像角色｜萌点/魅力：把盐袋当礼物；笑称「我曾经也是脚注」｜个人线剧情：教你要名字；采药路上讲改册那天的天气；弧光是允许自己不只以「幸存者」身份存在｜与主角关系：镜像 → 知己 → 可选恋

- **哈根（猎人小屋联络人·配角）**｜外貌：中年人类，帽檐压得很低｜性格：多疑、要脸、可被盐与诚意打动｜不攻略，但是贸易线关键 NPC；代表巢外世界的偏见与可转化性

- **主角视点（契约者）**｜姓名外貌自定；默认雾林失温旅人／公约见证人（可兼采购）｜成长体现为：从求生到坚持名单上是名字不是对象；最强能力不是「反向捕食」，是**改栏与护绳**

**【人际关系网 / 社团势力】**
- **改册派**：莉露、弥莎、芙恩——要名字、要绳、要贸易。  
- **旧法派**：凯拉及部分狩卫——要震慑、要边界、要「别再被猎」。  
- **中轴**：奥萝，掌握名簿与围炉表决。  
- **巢外**：猎人小屋（哈根）；小镇盐商与药铺（远景）。  
- **契约者位置**：程序性第三方与情感变量；拉任何一派都要付出信任成本，不能无代价全收。

**【情感事件 · 名场面】**
1. **炭笔改栏**：你按住奥萝的手，自己划掉「捕食対象」，写上姓名。  
2. **停词首验**：你说「潮退」，围炉真的静了两秒——有人第一次看见旧法可被语言打断。  
3. **出口绳夜检**：莉露教你数节；发现绳尾被塞进石缝。  
4. **汤碗事件**：辩论打翻的汤；弥莎不骂人，只说「先擦，再吵」。  
5. **公约一读**：草稿被凯拉撕开，奥萝又用浆糊粘回——裂痕留着当记录。  
6. **猎人小屋盐**：哈根不看角，只看盐袋分量；凯拉第一次把刀留在门外。  
7. **凯拉签字**：手抖到炭笔断尖，换笔再签。  
8. **弥莎的卡**：她自己的停词被写进档案，全场起哄式的温柔。  
9. **温泉前演练**：先停词，后解衣；有人中途喊停也被递毛巾。  
10. **烧纸 True**：旧栏纸进壁炉，烟气里有人笑有人哭；地表阳光漏进竖井。

**【隐藏剧情 · 真结局 · 伏笔】**
- 奥萝角上布条来自曾被人类当対象示众的友人；她改规矩是还债，不是时髦。  
- 凯拉幼时（成年后回忆）见过灌巢——所以她的极端是创伤，不是天生嗜暴。
