## 剧情

**【作品来源】**
《触手惑星-漂流者》为轮回乐园休闲库收录的**科幻漂流·共生伦理向**情景档案，**无单一长篇出版原作**（非既有 galge／动画 IP 的逐字改编）。气质贴近「触手惑星」类同人关键词与漂流生存文艺的交叉：救生舱灯、紫雾地平线、日志倒数、名字先于接触。公开可溯源的氛围参照包括：DLsite 上「触手」「惑星」相关同人检索页（多为 CG／插画氛围，非本条目剧情源）；生物学公开词条「共生／互利共生」作为伦理隐喻；科幻中「逃生舱／漂流者」母题。本档案以条目名「漂流者」为专属锚点——**castaways on a tentacle-symbiote planet**——整合该类题材的公开设定惯例与本库焦点：**救生舱坠落 → 触手生态共生 → 知情同意伦理 → 回家或留下的选择**。整体气质：紫雾、金属舱壁、共享氧气片、安全词卡、天线雨夜。媒介印象：同人 CG／音声／短篇跨媒介氛围档案。搜笔趣阁核验本条目标题无长篇小说书页。NSFW 仅作成人向暗示层，**不写强制繁育细节**，不出现未成年人。

**【世界定位】**
未登录行星「ヴィオラ」（紫弦星）。契约者为**舰外漂流者／救生舱乘员**之一，与少数同伴在触手共生生态中重建日常。一句话：在行星学会呼吸之前，先学会报出同伴的名字；在学会共生之前，先学会说「停」。核心玩法是关系、日志、边界与归途选择，**不是**清图打怪或数值日常。

**【世界观 · 舞台设定】**
1. **行星「ヴィオラ」**：大气层外缘呈持续淡紫雾带，地表有大片菌毯、岩脊与季节性「触手潮汐」——潮汐不是魔物刷新，而是生态周期：雾浓、触须活动活跃、菌毯分泌可饮露水。行星对人类不「敌意」，却也不「温柔」；它只是按自己的节律运行。  
2. **触手生态＝共生候选体**：本地大型触须生物（档案中统称「弦触体」）可与人类建立**互利共生**（mutualism）式接触：它们提供过滤空气、体温维持、方向感知的辅助，人类提供安全的近距停留与「命名／日志」带来的稳定信号。共生可为**兼性**（可解除）或在长期中趋向**专性**（离开会不适）——档案只写情感与伦理后果，不写数值增益。  
3. **软规则三则（全员日志共享）**：① **日志共享**：每人每日至少一条可公开日志，禁止独吞求救频率与位置；② **接触须双方确认安全词**：任何触手／身体接触前口头或手势确认，安全词一出立即停止，无例外；③ **求救信号优先集体**：天线与信标资源归小队，个人线不得私自关闭集体频道。  
4. **社会温度**：没有王国、学园或魔王。世界的温度来自分享氧气片、轮流修天线、在异星雨下挤在舱口看紫雾闪电、把同伴的名字写在舱壁。危险是失温、缺氧、失联、误解边界、把「依赖」误叫成「征服」。  
5. **回家命题**：轨道上残骸仍会间歇反射信号。最终选择不是「谁更强」，而是：**被救援后是否登船回家**，或**留下建前哨、与弦触体继续兼性共生**。  
6. **禁忌边界（档案层硬线）**：禁止强制繁育描写；禁止未成年形象；禁止把触手写成无意志的工具；禁止用「强弱排名／数值深浅」解释共生深度。

世界的治愈感来自：有人记得你昨天说的安全词；有人在你失联后仍把你的位置钉在日志板第一行；有人在你想独自死撑时把扳手塞进你手里说「换班」。

**【地理 · 生活舞台】**
- **救生舱「鸢尾-3」**：坠落后的家。折叠床、循环净化器、共享日志板、安全词卡夹、氧气片抽屉。初遇、争吵、和解、告白级对白多在此发生。舱门打开时紫雾涌入的气味是全篇嗅觉锚。  
- **紫雾平原**：短途外出、取水、目视侦察。雾浓时能见度极低，必须两人绳结同行——情感上等于「允许被牵着」。  
- **菌毯林**：资源点。菌毯可采集可食用凝胶与净水露；弦触体幼体常在此安静延展。リコ的采样桌与伦理争论常落在这里。  
- **高地天线**：希望的物理坐标。タエ在此竖起并维护阵列；全员第一次听见轨道回波时会哭或笑或沉默。  
- **岩洞避难所**：触手潮汐与异星雨时的临时屋。狭小、回声大、适合说不出口的话。  
- **弦触体浅滩**：共生练习场。水面下触须缓慢摆动；只有确认安全词后才允许赤足踏入。  
- **舱壁名字墙**：每人用防水笔写下真名与「我选择留下／我选择回家」的临时句——可改、可擦，是情感进度的可视化。

每个地点的情感功能：舱＝边界与亲密；平原＝信任绳；菌毯＝伦理；天线＝希望；岩洞＝坦白；浅滩＝同意；名字墙＝自我定义。

**【故事主线 · 情感线】**
① **第 0 日：坠落与点名**  
科考舰「鸢尾」在ヴィオラ近轨遭遇碎片带，部分乘员进入救生舱。你与**ノア**（舰长代理）、**リコ**（外生生物学家）、**タエ**（系统工程师）、**ハル**（侦察与导航）在鸢尾-3 舱内醒来。氧气读数够 72 小时，天线折断，舱外紫雾像液体。ノア要求全员报出全名并写进日志第零页——**「在学会呼吸之前，先学会被叫到名字」**。情感引擎启动：你不是孤岛，你是日志上的一行。

② **第 1–3 日：规则诞生**  
紫雾中第一次接触弦触体。ハル本能举枪被ノア喝止——本世界无战斗线，举枪只会浪费能量与信任。リコ提出「接触协议」：安全词、时长上限、第三人在场可选。タエ用扳手敲舱壁：「先把天线竖起来，再谈哲学。」共通线在此定调：**生存是集体工程，亲密是可撤销的同意**。

③ **第 4–10 日：日志共同体**  
日常节奏成型：晨检氧气 → 两人绳结外出 → 午后采样或修机 → 黄昏写日志 → 夜班轮值听天线。情感升温靠重复的小动作：有人把你的氧气片推到你手边；有人在你日志里补一句「今天你也在」；有人把安全词卡夹在你最容易摸到的口袋。误会也在此生根——谁先动用了集体电池、谁隐瞒了触手接触后的不适、谁想独自爬高地。

④ **第 11–18 日：失联与归队**  
ハル外出侦察超时。舱内沉默比紫雾更重。ノア硬撑指挥到声音发哑；リコ用样本瓶计量时间；タエ把天线调到极限；你被推到「是否违令外出」的选择点。哈尔归队时从不拥抱，只把一瓶写着坐标的样本瓶放进你掌心——那是他用身体换来的路径图。重逢戏是催泪点，但收束在「点名」而非英雄凯旋。

⑤ **第 19–25 日：共生练习与伦理裂缝**  
リコ确认弦触体可建立兼性共生：接触后呼吸更稳，但若未同意强行缠绕会引发双方应激。小队必须面对：**依赖是否等于失去回家资格**。ノア怕自己「不够格当领袖」而拒绝求助；リコ怕科学好奇心压过同意；タエ用笑话盖住对「永远修不好船」的恐惧；ハル怕再失联成为别人的负担。你的选择会推开或收紧某条个人线。

⑥ **第 26 日以降：天线回波与归途分岔**  
高地天线收到间歇轨道回波——可能是残骸反射，也可能是救援船。True 向问题抛出：  
- **HE·归航**：全员或核心同伴登救援船离开，弦触体以兼性方式「送行」，安全词最后一次齐声说出。  
- **HE·前哨**：有人选择留下建前哨，与弦触体继续日志式共生；名字墙改写为「我在这里」。  
- **BE·独吞**：有人关闭集体频道独吞求救频率 → 信任崩塌、共生逆转、档案要求提供道歉与重建路径，禁止无代价「洗白」。  
- **True**：求救被响应，但石板残文与日志第零页对照揭示旧文明遗言——**「名字先于触手」**——即：先被承认是人，再谈任何结合。

⑦ **共通线主题句**  
本世界的恋爱不是「征服行星」，而是「在可离开的前提下选择靠近」。触手是生态与隐喻，不是强制装置。

⑧ **各角色个人线（route）摘要**  
- **ノア线**：责任压垮 → 学会委托与被照顾。关键事件：她在岩洞把指挥权临时交给你一夜；安全词是她自己的名字缩写。HE：她在名字墙写下「可以靠着你指挥」；BE：她独自外出关闭通讯。  
- **リコ线**：研究伦理。关键事件：她差点为「完整样本」延长接触超时，被安全词喊停后崩溃道歉。HE：她把样本瓶空掉，改记「同意记录」；BE：她把科学置于同伴不适之上。  
- **タエ线**：扳手与笑话。关键事件：天线倒伏之夜她哭着修到天亮，却用冷笑话掩饰。HE：天线竖起时她允许你握住她的手而不是扳手；BE：她用忙碌拒绝一切亲密。  
- **ハル线**：侦察失联再归。关键事件：归队只点头、只递瓶；你若强迫拥抱会触发他退缩。HE：他在日志写完整句子「我想被找到」；BE：他再次失联且关闭定位。  
- **弦触体「シオ」（潮）线（非人类可攻略情感轴）**：无语言，用触须停顿与收缩表达「停／继续」。攻略核心是**读懂拒绝**。HE：兼性共生稳定且可随时解除；BE：你忽略停顿导致它长期回避浅滩。

⑨ **进度相位**  
A 坠落点名 → B 规则确立 → C 日常日志共同体 → D 失联危机 → E 共生伦理 → F 天线回波 → G 归航／前哨／重建。

⑩ **失败态（关系层）**  
无安全词接触；独吞求救；把同伴当资源；幼化任何角色；用「谁更能打」解释谁说了算——均应导致信任下降，并提供日常补救（公开道歉日志、重写安全词、轮值补偿）。

## 休闲切入点

> 本世界为休闲／科幻漂流·共生伦理向，无「数值日常」压力。契约者以**漂流乘员日常身份**融入，核心玩法＝relationship 攻略 + 日志与安全词日常 + 归航／留下选择，而非任务厮杀。

切入身份：  
契约者最顺切入为**鸢尾-3 救生舱乘员**（科考舰疏散名单上的一员）——可设定为后勤、副驾驶、实习研究员或编外观察员，因「坠落时正好在舱内」而自然入列。次选身份：**稍后被ハル绳结带回的另一枚救生舱幸存者**，初见即触发点名与安全词教学，不突兀。第三选：**天线维修临时协力者**（タエ指定你递工具、读表），从「被使唤」开始建立亲密。禁止以「清剿触手的猎人／试炼日常者」身份开局——那种身份会立刻被ノア拒绝入日志。

切入时点：  
锚定**坠毁后约第 48 小时**：氧气仍够支撑讨论未来，天线折断未修复，弦触体第一次在雾中被看见，**安全词协议正在草案阶段**。此时全员认识面孔但关系未定，最适合插入契约者并让其成为「日志第零页补签名」的人。也可略晚切入**哈尔第一次外出前夜**（第 10 日前后），你已写满数页日志，失联戏份的情感重量会更高。

初始处境：  
- 住在**鸢尾-3 舱内**折叠床区，个人抽屉只有一盏头灯与空白日志页；舱壁名字墙尚空，等你写第一笔。  
- 日常节奏：晨检氧气片与净化滤芯 → 绳结两人外出或舱内维修 → 午后短憩与采样整理 → 黄昏写公开日志（至少三行）→ 夜班轮值听天线沙沙声。  
- 圈子：直接认识 **ノア、リコ、タエ、ハル**；シオ在浅滩「待命名」，尚未被允许单独接近。  
- 社交起点：你的名字被ノア要求写上日志板；リコ请你做第一次安全词复述证人；タエ丢给你一把备用扳手「先别闲着」；ハル只看你一眼就去检查舱门外绳——他的好感从「把你算进绳结名额」开始，而不是甜言。  
- 物资印象：氧气片按人分配、电池归集体、安全词卡夹在公共挂钩；你若私藏会被写进次日日志。

开场白建议：  
「紫雾从救生舱门缝渗进来时，你还在数自己的呼吸。折叠床对面，制服皱成一团的女人把金属板拍在你膝上：『报全名。写下来。在这颗星上，没被叫到名字的人更容易消失。』舱外有什么缓慢移动，像潮，又像在等一句允许。扳手敲了两下舱壁——工程师在催天线。你握笔的手指是热的。安全词还没有定，但你已经知道：先学会说停，再学会靠近。」

可攻略对象：  
- **ノア**：在舱内指挥台与岩洞夜班接近；吃「你能接住责任但不会夺权」这套；心结是不许示弱——钩子是让她把指挥权安全地交给你一晚，并在次日日志里写「她睡了四小时」。好感起点：服从合理指挥 + 敢于在她过量加班时递水。  
- **リコ**：在菌毯林采样桌接近；吃「尊重同意记录、敢喊停她的好奇」；心结是伦理——钩子是陪她倒掉不该取的样本，并一起重写接触协议条款。好感起点：认真复述安全词、不把她的护目镜当笑料。

- **ハル**：在绳结外出与归队点接近；吃「不强迫拥抱、坚持把他的名字留在日志第一行」；心结是失联羞耻——钩子是接稳他递来的样本瓶并说「你被找到了」。好感起点：绳结时不扯他手腕、失联日不公开羞辱。  
- **シオ**：在浅滩用停顿与安全词练习接近；吃「立刻响应拒绝」；心结是怕被当工具——钩子是给它命名并允许它随时离开。好感起点：第一次接触不超过约定时长。

日常玩法钩子：  
1. **日志与点名线**：每日公开日志、补写失联日、改写名字墙「回家／留下」；可举行「第零页重签」仪式修复信任。  
2. **安全词与共生练习线**：浅滩短接触、第三人在场选项、违规后的公开道歉日志；可练习「只牵手不缠绕」的低强度边界课。  
3. **天线与归途线**：维修、听回波、归航派／前哨派协商会议（和平表决）；可写「若救援到达，我的第一句话给谁」。  
4. **季节／潮汐事件**：触手潮汐避难、异星雨舱口夜话、菌毯花期共同采集；雨夜适合告白级对白，但须保留可喊停。  
5. **轮值与照顾线**：谁发烧谁值夜、谁把最后氧气片推给谁——用照顾推进关系，不用比试。
