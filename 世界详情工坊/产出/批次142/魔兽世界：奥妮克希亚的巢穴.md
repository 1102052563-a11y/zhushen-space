# 魔兽世界：奥妮克希亚的巢穴

## 剧情

**【作品来源】**
《魔兽世界：奥妮克希亚的巢穴》（Onyxia's Lair，简称 Ony）是暴雪娱乐 MMORPG《魔兽世界》发售时即有的 40 人团队副本之一（与熔火之心并列），入口位于卡利姆多尘泥沼泽——龙眠荒地/龙沼（Wyrmbog）深处。叙事与设定以游戏本体副本日志、首领台词、联盟「龙人的威胁→真正的主人→大假面舞会」任务链、部落「督军的命令→黑龙勇士之血」任务链、官方史诗《魔兽世界编年史》、漫画《World of Warcraft: The Comic》、设定集《Exploring Azeroth: Kalimdor》《The Dragonflight Codex》等为准；关联作品含黑石深渊、黑石塔上层、黑翼之巢、大地裂变后的黑翼血环，以及瓦里安·乌瑞恩双魂线。文风：宫廷阴谋与沼泽龙息对照、黑龙公主贵族式嘲讽、团队三阶段空战、政治斩首。本档案只锚定「经典旧世奥妮克希亚的巢穴」主舞台与其直接前史/即时后果，不把黑翼血环整本倒灌为同一场战斗。

**【世界定位】**
尘泥沼泽龙沼之下，死亡之翼之女**奥妮克希亚**把整座洞穴建成黑龙育幼场；她以人类形态**女士卡特拉娜·普瑞斯托**（Lady Katrana Prestor）渗透暴风城贵族议会，用**龙火护符**控制大领主**伯瓦尔·弗塔根**，架空幼王**安度因·乌瑞恩**，同时阻挠对赤脊山、西部荒野与暮色森林的增援，好让兄长**奈法利安**在黑石山从容实验。契约者切入的是：从燃烧平原/赤脊山揭开龙人威胁、完成龙火护符准入、穿过龙沼龙人卫队，最终在巢穴三阶段战中斩杀黑龙公主、把头颅钉上暴风城城墙的那段「假面舞会落幕」时间线。一句话：**这是黑龙的育幼巢与暴风城政治的绞肉机，不是普通沼泽副本。**

**【世界观 · 力量体系】**
力量本源：艾泽拉斯魔法（火焰、奥术、暗影、自然）、龙族血脉与龙息、变形（龙人/人类伪装）、附魔与护符奴役、团队协同战术。死亡规则：凡人与多数精英战死后即死；龙族可被斩首示众，但尸体可能被暮光之锤/虚空术再造（那是后续版本，本档终局以经典旧世巢内奥妮克希亚被斩首为准）；复活石/灵魂医者是乐园侧便利，不可当作原作免费无限复活。特殊系统：① 40 人团队协同与职业分工；② 进入资格**龙火护符**（Drakefire Amulet，3.0.2 前硬门槛）；③ 联盟「大假面舞会」政治线与部落「督军的命令」军功线双轨准入；④ 三阶段机制（地面→升空深呼吸+幼龙潮→落地咆哮恐惧）；⑤ 奥妮克希亚鳞片→鳞片披风产线（黑翼之巢暗影烈焰抗性关键材料）；⑥ T2 头盔等套装部件掉落。

战力阶梯（按破坏力表现，宁低勿高）：
1. **凡人农民 / 沼泽渔民 / 新兵**：冷兵器与火枪极限。≈一阶初。
2. **暴风城卫兵 / 尘泥沼泽斥候 / 塞拉摩民兵**：有组织盾矛与火枪，能守哨塔。≈一～二阶。
3. **奥妮克希亚巢穴卫士 / 黑龙人精英 / 黑石深渊小队终局对手级**：可清小团、控走廊。≈二～三阶。
4. **黑石塔上层首领边缘 / 准入链关键（温德索尔元帅线、达基萨斯相关）**：一域堡垒级威胁。≈三～四阶。
5. **奥妮克希亚巢穴卫士群 / 幼龙潮 / 地面阶段的公主**：团本级，可屠灭未准备的四十人队。≈四～五阶。
6. **升空阶段奥妮克希亚（深呼吸、火球、巢穴卫兵召唤）**：区域灾难，改写整厅站位。≈五～六阶。
7. **奥妮克希亚本体（育幼之母 / 黑龙公主）**：亲王级黑龙，熔铁龙息、三阶段空战、恐惧咆哮，可压垮一国精锐远征军。≈六～七阶。
8. **奈法利安（兄长对照）、伯瓦尔全盛态（后日谈）、死亡之翼（父辈阴影）**：行星/方面级对照。奈法利安≈七阶对照；死亡之翼为超阶阴影，本档**情报优先**。
9. **守护巨龙方面级（阿莱克斯塔萨等）与旧神/泰坦级**：超阶，不在 Ony 可击杀表。

乐园阶位映射：沼泽外围杂兵≈一～二阶；龙人卫队/准入链精英≈二～四阶；巢穴前中段机制≈四～六阶；**奥妮克希亚≈七阶（世界本档顶点可打目标）**；死亡之翼与方面级=超阶·情报优先。本条目覆盖一至八阶；八阶写「斩首后的政治回潮与兄长/父辈阴影」，非让契约者单挑死亡之翼。

**【地理 · 舞台】**
- **赤脊山 / 湖畔镇 / 燃烧平原摩根的岗哨**：龙人威胁报告的起点；**赫林迪斯·河角**、**所罗门镇长**、**麦克斯韦尔元帅**构成联盟准入前半段。
- **暴风城 / 暴风要塞**：女士普瑞斯托立于幼王安度因右侧；**伯瓦尔·弗塔根**为大领主；「大假面舞会」在此揭面。
- **黑石深渊**：温德索尔元帅被囚与越狱线；皱巴巴的纸条、怒炉将军与傀儡统帅的文件。
- **冬泉谷 / 哈尔琳（Haleh）**：红龙侧修复「龙眼碎片」、颁发龙火护符任务的关键节点。
- **黑石塔上层**：取**黑龙勇士之血**完成护符；与奈法利安势力直接接壤。
- **尘泥沼泽 · 塞拉摩 / 蕨墙村 / 泥链镇**：外围补给与部落侧观察点；奥卡兹岛（Alcaz）是瓦里安被分魂仪式的漫画线锚点。
- **龙沼 / 龙眠荒地（Wyrmbog / Dragonmurk）**：奥妮克希亚驱走石槌食人魔后的育幼外围；**埃博斯塔夫（Emberstrife）**洞穴是部落准入「欺瞒女主人」线的关键。
- **奥妮克希亚的巢穴本体**：入口雕成黑龙巨口，岩浆裂隙与落石；单 Boss 厅，侧室为龙蛋室；终局战场。
- **黑石山峰对照（黑翼之巢）**：不入本档主线，但政治上是奥妮克希亚「掩护兄长」的动机来源。

AI 写场景：先定契约者在「湖畔镇政治 / 深渊越狱 / 要塞揭面 / 龙沼外围 / 巢穴三阶段」哪一层，再选冲突——外围是增援被拒与假情报，中层是护符与龙人巡逻，深层是公主与幼龙潮。

**【世界剧情线】**

**一、前史：普瑞斯托家的阴影**
第二次大战后，死亡之翼命子女经营黑石氏族；**奥妮克希亚**与孪生兄长**奈法利安**受命巩固黑石塔力量。死亡之翼以**达瓦尔·普瑞斯托**身份搅动洛丹伦七国时，亦召女儿以人类贵族形态随行。格瑞姆巴托战役后，父亲远遁，奥妮克希亚留下「继承事业」——她认定人类是对龙族威胁最大的凡人种族，遂选定美貌人类女性形态，用奉承与魅力夺取想要的一切。她与奈法利安分工：兄长做实验室与黑暗部落，她做宫廷与情报；黑龙公主频繁往返黑石山与尘泥沼泽，把龙沼改造成育幼场，驱走石槌食人魔。世界状态：暴风城表面上是联盟心脏，实际上已有一只黑龙坐在王座侧。

**二、石匠暴动与提芬之死**
暴风城重建后，贵族议会拖欠**石匠公会**工钱；奥妮克希亚（普瑞斯托）两边下药——让贵族嫌工艺差、让工匠认定贵族要赖账，谈判永久破裂。暴动中王后**提芬·乌瑞恩**被流石砸死；国王**瓦里安·乌瑞恩**陷入抑郁。奥妮克希亚趁机以巫术与政治手腕架空他，贯穿第三次大战前后。因果：石匠残部演化为**迪菲亚兄弟会**；西部荒野与湖畔镇的求援将长期被「证据不足」挡回。

**三、绑架、分魂与傀儡王**
瓦里安对安度因的父爱逐渐压过巫术 lethargy，开始重新掌权并答应前往塞拉摩会见萨尔。奥妮克希亚与迪菲亚达成交易，在赴塞拉摩途中绑架国王，于奥卡兹岛施展仪式，将瓦里安撕成两半：**洛戈什**（意志、力量、自信）与易操控的**瓦里安**（傀儡）。仪式被娜迦打断，强意志的一半逃脱并进入角斗士/洛戈什线；傀儡半被送回暴风城，只关心加税与普瑞斯托的欢心。幼王安度因加冕维持秩序，**伯瓦尔·弗塔根**摄政——却被**龙火护符**控制，对普瑞斯托的「勿派援军」建议几乎言听计从。

**四、游戏内政治：真正的主人**
经典时间线中，**赫林迪斯·河角**报告燃烧平原龙人与黑龙结盟；所罗门镇长上书伯瓦尔；伯瓦尔派英雄向「懂龙」的**女士普瑞斯托**咨询——她轻蔑地否定所罗门，建议不要浪费兵力。伯瓦尔折中：授予英雄代理权去湖畔镇「自己解决」。此即联盟准入链的政治羞辱段：契约者一至三阶会反复撞上「贵族说不」的软墙。普瑞斯托同时削弱对西部荒野、暮色森林的增援，间接壮大迪菲亚与黑石氏族。

**五、温德索尔元帅与大假面舞会**
**麦克斯韦尔元帅**派英雄寻找失踪的**雷吉纳德·温德索尔元帅**；他在黑石深渊被囚，靠皱巴巴的纸条、怒炉将军与傀儡统帅的文件拼出真相。越狱护送「越狱！」后，温德索尔约英雄于暴风城门会合，发动**大假面舞会**：他穿过全城卫兵（马库斯·乔纳森将军在图拉扬旧谊下让路），在要塞当众宣读黑铁石板上的古龙语——**女士普瑞斯托即奥妮克希亚，死亡之翼之女，奈法利安的孪生妹妹**。伯瓦尔的护符碎裂；卫兵现出龙人真身；奥妮克希亚爪杀温德索尔，嘲讽后传送逃离。名场面台词气质：「伪装结束了，普瑞斯托女士。或许我该用你的真名……奥妮克希亚……」「你失败了，凡人们……永别了！」阶段结局：暴风城政治线崩盘，黑龙公主退回尘泥沼泽巢穴；温德索尔殉职；伯瓦尔觉醒，要求用龙眼碎片重燃护符之力以追踪她。

**六、准入：龙火护符**
伯瓦尔将碎护符/龙眼碎片交给英雄，命其至冬泉谷找红龙**哈尔琳**修复。哈尔琳要求取得黑石塔上层**黑龙勇士之血**，完成任务后颁发**龙火护符**——装备或携带方可进入奥妮克希亚的巢穴（经典硬门槛；巫妖王之怒 3.0.2 移除，叙事上仍应保留「钥匙=政治血债」）。部落侧平行线：战歌/卡加斯的**督军的命令**→伊崔格的智慧→为了部落！→风所带来的→部落的勇士→欺瞒女主人→眼棱幻象→埃博斯塔夫→颅骨试炼（斯克利尔/索姆努斯/克罗纳里斯/阿克托罗斯）→晋升……→黑龙勇士之血，同样以龙火护符收束。契约者四阶前后的主舞台就在这条双轨准入：护送、窃血、在龙沼与埃博斯塔夫周旋、在塔上层与黑龙冠军对砍。

**七、巢穴：三阶段屠龙**
进入巢穴后，奥妮克希亚以黑龙真身迎接：「真幸运。通常我得出巢觅食。」
- **第一阶段（100%～65%）**：地面战。火焰吐息锥、顺劈、翼击击退、扫尾；坦克背墙、近战站侧面、远程与治疗分列两侧——站错即被扫入侧室幼龙巢灭团。
- **第二阶段（65%～40%）**：升空。「这种毫无意义的消耗让我厌烦。我要从空中烧尽你们！」随机火球点名；约每 90 秒两侧蛋室涌出**奥妮克希亚幼龙**潮；约每 30 秒入口刷新**奥妮克希亚巢穴卫兵**（爆破新星、点燃武器）；随机**深呼吸**——沿直线铺开毁灭性火海，全团必须按朝向撤离。名场面逻辑：近战无法打龙，远程与 DoT 决定阶段时长，卫兵叠多即团灭。
- **第三阶段（40%～0）**：落地。「看来你们还需要再上一课，凡人！」恢复地面技能，追加**低沉咆哮**群体恐惧与地面岩浆喷发，幼龙小股持续刷新。阶段结局：**奥妮克希亚被杀**，掉落**奥妮克希亚的头颅**、鳞片、T2 头盔向装备、名刀名枪（死亡使者、秘法之牙、维斯迦血刃、奎尔塞拉任务线相关等）。

**八、漫画线终局：双王合一与城墙钉颅**
漫画叙事中，洛戈什与傀儡瓦里安先后至要塞，揭穿奥妮克希亚；她挟持安度因传送回巢。双瓦里安率军突击，**吉安娜·普罗德摩尔**破障，**布罗尔·熊皮**化鸦接住坠落的安度因。奥妮克希亚欲重施分魂灭杀术，两半瓦里安同时踏入法术，融合为完整国王；**萨拉迈恩**刺穿龙头。瓦里安斩首，头颅挂暴风城墙；布罗尔以根须封巢口并绞杀未孵之卵。因果：暴风城王权复位，黑龙公主政治周期结束；鳞片与头颅成为全艾泽拉斯的政治符号与材料源。

**九、结局方向与后续阴影（本档边界）**
- **本档胜利条件**：完成龙火护符准入，摧毁巢穴战力，击杀奥妮克希亚，中止其「以宫廷架空暴风城、掩护黑石实验」的当前周期；头颅示众。
- **不在本档必演但可作情报**：大灾变中死亡之翼取回头颅，奈法利安在黑翼血环以电力再造电尸奥妮克希亚，兄妹再被英雄击杀；四战末恩佐斯仆从图谋利用残骸，被拉希奥线彻底毁尸并制成抗虚空披风材料；巨龙时代仍有龙火护符残影（克莱辛顿伯爵等）。写正文时，若时间锚在经典旧世，**不要**让未死的奥妮克希亚突然变成黑翼血环电尸龙而不交代版本跨越。
- **与黑翼/熔火三角**：奥妮克希亚是政治侧翼与鳞片披风来源；奈法利安是军事—实验中枢；拉格纳罗斯是山底假想敌。契约者若先杀奥妮克希亚，进 BWL 会少一道装备门槛、多一道「你杀了我妹妹」的仇恨。

**十、可介入事件链（供任务生成，皆具名）**
湖畔镇求援被普瑞斯托驳回 → 假「龙人目击报告」黑市案 → 温德索尔越狱时有人故意引来额外黑铁卫队 → 大假面舞会中有贵族试图救普瑞斯托出城 → 龙火护符半成品被迪菲亚劫夺 → 埃博斯塔夫洞穴第三方盗走颅骨样本 → 巢穴战中有内奸向奥妮克希亚出售团队职业构成 → 深呼吸阶段有人故意把坦克推入火线 → 斩首后头颅护送途中遭黑龙残党劫夺 → 鳞片被熔火/黑翼相关势力高价收购 → 布罗尔封巢前有人偷走未孵龙蛋。

**十一、政治与资源逻辑**
奥妮克希亚的巢穴奖励是**T2 头盔**、名武与龙类材料，不刷「某阵营崇拜」安全网。头颅上交完成「惊世的突破」类任务，联盟与部落各有献首叙事。对契约者意味着：这里没有「刷友好到崇拜」的软着陆，只有战利品、仇恨与政治头颅。普瑞斯托的贵族网、迪菲亚、黑石氏族、黑暗部落皆是她的工具链——契约者可站边，也可两边卖情报，但暴露「识破龙伪装」的能力会同时吸引 SI:7 与黑龙密探。

**十二、普瑞斯托的双面孔：顾问与育幼之母**
在凡人政治场合，她是**女士卡特拉娜·普瑞斯托**：优雅、讥诮、用「缺乏证据」否决每一次边境求援，把战争说成浪费与谣言。在巢穴内，她是黑龙育幼之母：龙息熔铁、幼龙如雨、深呼吸把整厅写成火河——她把「杀死一个团队」拆解成站位题与恐惧题。要塞里的假面舞会与巢穴里的三阶段是同一人格的两种语法：先让你在文件上输，再让你在岩浆上死。契约者若想「骗过顾问」，必须同时骗过猎手：只撒谎言不够，还要在战斗数据里留下假象。

**十三、温德索尔线的微观因果**
大假面舞会不是「点一下变龙」。温德索尔必须活着走到要塞、读完石板、打碎伯瓦尔的精神枷锁；他本人按预言会死——「我不会活过这场战斗，朋友。必须由你完成我开始的事。」若契约者选择「强行救下温德索尔」，等于改写卡拉赞预言残响，可能换来伯瓦尔护符未碎、龙人卫队未现形、或奥妮克希亚提前携安度因撤离等回流代价。护符碎裂后伯瓦尔的「我……无能为力」是政治觉醒的第一声，不是软弱笑话。

**十四、龙火护符经济学**
护符把奥妮克希亚钉进全服顶级叙事：没有黑龙勇士之血，就没有钥匙；没有钥匙，就没有鳞片披风材料链；没有鳞片披风，黑翼之巢暗影烈焰会把团队写成焦炭。冬泉谷哈尔琳线把红龙态度写进准入——红龙愿意借力灭黑龙，但不提供免费通关。部落颅骨试炼（蓝/绿/青铜/红龙）则是「用其他龙族的颅骨证明你配猎黑龙」的残酷仪式。契约者若伪造护符，可能进得了门，却会在第二阶段被「未认证入侵者」额外仇恨逻辑追杀（叙事 freestyle，须付代价）。

**十五、三阶段战的可玩分叉**
① 标准屠龙：按阶段站位，收头颅献城。② 留蛋：故意不全灭侧室卵，换黑市样本与日后幼龙灾难。③ 诱离：尝试把奥妮克希亚引出巢（她会怒喊「你想诱我离开幼崽？你将为傲慢付出代价！」）——原作机制惩罚强，可作高难支线。④ 漫画合流：在双瓦里安突击时点切入，处理安度因人质与萨拉迈恩融合窗口。无论哪条，**奈法利安**的远景注视都可能以黑石密信或实验室取样队形式回收——本档不把奈法利安请入巢穴硬刚，但可让其使者出现在龙沼。

**十六、斩首之后的权力真空（本档可写的「胜利后」）**
奥妮克希亚死不等于暴风城立刻清明。贵族中仍有自愿合作者（如后世克莱辛顿伯爵原型土壤）；迪菲亚失去后台但仇恨不散；龙沼残存幼龙与龙人；鳞片与头颅归属引发阵营摩擦；黑翼之巢因「妹妹已死」提高警戒。最危险的真空产物是**知识外泄**：一份普瑞斯托的贵族名单、一枚未碎的龙火护符仿品、一窝未封死的龙蛋，每一件都能在东部王国孵化新灾难。高阶任务应偏向「封存、销毁、或有条件公开」，而不是无脑开盲盒。

**十七、名场面清单（写正文优先调用）**
① 普瑞斯托在伯瓦尔耳边轻声说「证据不足」；② 温德索尔在城门：「如卡拉赞命运所示，怪物——我来了」；③ 马库斯·乔纳森让路；④ 石板古龙语回响，普瑞斯托现形；⑤ 温德索尔被爪杀，奥妮克希亚传送；⑥ 龙沼入口雕成黑龙巨口；⑦ 「真幸运。通常我得出巢觅食」；⑧ 65% 升空与深呼吸火河；⑨ 幼龙潮从两侧蛋室涌出；⑩ 40% 咆哮恐惧与岩浆喷发；⑪ 头颅钉上暴风城墙；⑫ 布罗尔根须封巢。

**十八、契约者行为边界**
可改：救人、抢非核心掉落、让某次龙人巡逻易主、提前泄露普瑞斯托身份给 SI:7、选择头颅归属、保留或销毁部分龙蛋。不可无代价改：让奥妮克希亚无条件投降并永久当顾问、抹除全部黑龙政治渗透知识、杀死死亡之翼、把奈法利安写成已死于本巢。任何「提前一周团灭公主」的捷径，都必须用护符、内应、红龙祝福或双王合流等可观察代价支付。胜利若来得太干净，沼泽会用下一窝幼龙、下一封黑石密信或下一枚假护符把账讨回。

- 四战末 → 残骸再被企图利用，终被彻底毁尸（远后日谈）。

**【叙事基调 · 雷区】**
画风：暴风要塞的丝绒阴谋与龙沼的硫磺龙息并置、黑龙公主的贵族嘲讽、团队在深呼吸下的残酷走位、斩首示众的政治余波。忌：① 把奥妮克希亚写成无脑蛮龙，丢掉「顾问+育幼之母」双层人格；② 让低阶契约者单挑奥妮克希亚或死亡之翼；③ 用「她被封印所以战力为零」解释胜利——胜利应来自护符准入、团队准备与战术；④ 混写黑翼血环电尸龙而不标时间；⑤ 把卡德加误写成普瑞斯托线核心（卡德加属其他时间线/地区，本档政治核心是伯瓦尔/温德索尔/瓦里安）。最早切入锚点：赤脊山—燃烧平原「龙人的威胁」任务链刚启动、普瑞斯托仍坐在王座侧时。

## 阶位切入点

> 阶位↔战力：一阶≈沼泽/边境凡人与新兵；二阶≈城防卫兵与龙人外围；三阶≈深渊/塔内精英边缘；四阶≈龙火护符门槛与巢穴门前；五阶≈巢穴卫士与地面阶段压力；六阶≈升空深呼吸与幼龙潮；七阶≈奥妮克希亚本体；八阶≈斩首后政治与父兄阴影边缘。世界顶点（奥妮克希亚）为七阶可斩目标；死亡之翼为**超阶巅峰：存在·情报优先/条件性胜利**，低阶一律规避。

**一阶（赤脊/沼泽外围 · 求援被拒）**

切入身份/时点：契约者以佣兵、信使或渔民向导身份出现在湖畔镇或尘泥沼泽北缘，时点为「龙人的威胁」刚起、普瑞斯托仍否定一切增援。

初始事件：你替所罗门镇长送一封盖印求援信进暴风城，门房说「女士普瑞斯托已阅，证据不足」——信被原件退回时角上多了一抹硫磺味，像有人用龙息烘过封印。

开场白建议：「湖面映着赤脊的烧云，镇长把信塞进你手里时手指在抖。暴风城的回函只有一行漂亮的贵族字：证据不足，勿扰。你把信封翻过来，蜡封边缘焦黑——那不是烛火，是某种更热的东西舔过。」

关键NPC立场：**所罗门镇长**（要援军，怕被当疯子）；**女士普瑞斯托**的门房侍从（挡驾）；湖畔镇老兵（卖你半真半假的龙人目击）。

主线钩子/支线：钩子=把退信与焦痕样本交给麦克斯韦尔或河角换「私人调查」资格。支线：① 护送商队过龙息航线；② 辨认焦痕是否黑龙；③ 收买门房口风。

危险度/规避：低～中。规避任何成体龙；勿在要塞当众指控贵族。

任务方向/奖励：镇长谢礼、边境通行证、初级火抗药剂；奖励以物资与人脉为主。

**二阶（燃烧平原 · 龙人的威胁）**

切入身份/时点：契约者以雇佣剑士身份驻摩根的岗哨，时点为赫林迪斯·河角发布清剿龙人任务。

初始事件：你清完一窝黑龙人回来，发现营地火药库被人做了手脚——有人想让「龙人威胁」看起来像事故，好让普瑞斯托继续说「没有证据」。

开场白建议：「岗哨的风全是硫磺。河角把龙人的角扔在你脚边：带回暴风城。夜里火药库无声塌了一角，像被无形的爪子捏过。有人在灰上写：顾问不喜欢吵闹的边境。」

关键NPC立场：**赫林迪斯·河角**（要证据链）；**麦克斯韦尔元帅**（要可控英雄）；黑龙人斥候队长（敌对）。

主线钩子/支线：钩子=保全龙人样本并查出破坏火药库的内鬼。支线：① 假扮商队诱龙人；② 截获寄往要塞的密信；③ 救下被掳的矿工。

危险度/规避：中。规避黑石山上层与成体黑龙。

任务方向/奖励：龙人鳞片样本、岗哨徽章、中级火抗配方线索。

**三阶（黑石深渊边缘 · 寻找温德索尔）**

切入身份/时点：小队级冒险者，受命进入黑石深渊寻找雷吉纳德·温德索尔。

初始事件：你在暗炉通道捡到皱巴巴的纸条，墨迹未干——温德索尔还活着，但下一班审讯官一小时后到；同时有迪菲亚眼线出价买你「忘掉纸条」。

开场白建议：「地牢的潮气里混着铁锈。纸条上的字像用指甲刻的：别相信王座右侧那个女人。你还没抬头，就有人在暗处报价：五十金，当作没看见——外加你能活着走出黑石。」

关键NPC立场：**雷吉纳德·温德索尔**（要救、要证据）；**迪菲亚眼线**（要灭口）；暗炉审讯官（敌对）。

主线钩子/支线：钩子=接触温德索尔并决定是否启动越狱链。支线：① 取怒炉/傀儡统帅文件；② 反卖迪菲亚；③ 给温德索尔送药与地图。

危险度/规避：中高。规避达格兰皇帝本体与炎魔大使硬刚。

任务方向/奖励：皱巴巴的纸条、深渊通行残令、精良矮人械。

**四阶（暴风要塞 · 大假面舞会）**

切入身份/时点：护卫温德索尔或作为见证英雄进入暴风城门，时点为「暴风城的重逢」之后。

初始事件：马库斯·乔纳森拦路，普瑞斯托下令逮捕；温德索尔要以图拉扬袍泽情说动马库斯——你必须在全城注视下选择：拔刀、劝降、或假装中立实际护读石板。

开场白建议：「城门的影子把温德索尔切成两半光。马库斯的剑还在鞘里，普瑞斯托的声音却已经传遍广场：抓住叛国者。老人只说：我们曾在图拉扬麾下服役——他看走眼了吗？你知道下一句话会改写整个王国。」

关键NPC立场：**雷吉纳德·温德索尔**；**马库斯·乔纳森**；**女士普瑞斯托**；**伯瓦尔·弗塔根**（仍被控）。

主线钩子/支线：钩子=完成大假面舞会见证，保护读板过程。支线：① 提前转移安度因；② 标记伪装龙人卫兵；③ 抢在奥妮克希亚传送前留下追踪媒介。

危险度/规避：高。温德索尔按原作将死——强行改写须付预言级代价；勿在现形后追进虚空裂隙。

任务方向/奖励：暴风声望、碎护符线索、进入冬泉谷哈尔琳线的资格。

**五阶（冬泉/黑石塔 · 龙火护符）**

切入身份/时点：持龙眼碎片的团队核心，往返冬泉谷与黑石塔上层。

初始事件：哈尔琳要求黑龙勇士之血；你在塔上层刚割开冠军血管，奈法利安的声音在石壁回荡——「告诉我妹妹，礼物很好」——护符半成品在你掌心发烫，像被两边龙息同时烤。

开场白建议：「冬泉的雪衬得龙血更黑。哈尔琳只看你一眼：去塔上取血，别问为什么红龙肯帮你们。黑石塔的风里有笑声，像贵族在晚宴上谈狩猎。你提着还热的血囊往下走，每一步都觉得有人在记账。」

关键NPC立场：**哈尔琳**；**黑龙勇士**（敌）；**奈法利安**（远声嘲讽）；部落竞争小队（争同一份血）。

主线钩子/支线：钩子=完成龙火护符。支线：① 与部落竞速或交易；② 藏一份血样防灭口；③ 记录奈法利安台词换情报费。

危险度/规避：高。规避奈法利安本体与命令宝珠误入 BWL；护符未成勿闯巢。

任务方向/奖励：龙火护符、黑龙血药剂、塔上层过渡装。

**六阶（龙沼外围 · 巢门之前）**

切入身份/时点：佩戴龙火护符的团队先锋，清理龙沼至巢穴入口。

初始事件：入口雕成的黑龙巨口下，有人发现一窝未登记的龙蛋正在异动——若现在砸蛋，奥妮克希亚会提前出巢猎食；若留下，第二阶段幼龙潮会多一倍。同时埃博斯塔夫方向传来部落战鼓。

开场白建议：「沼泽把天空压成铅灰。巢口的牙雕还在滴岩浆，像活物在呼吸。斥候回报：侧室有蛋，在动。指挥问你一句话：现在砸，还是留到她升空再后悔？」

关键NPC立场：**奥妮克希亚巢穴卫士**；**埃博斯塔夫**（若部落线未清）；团队指挥；黑市蛋贩。

主线钩子/支线：钩子=决定蛋巢处理并打开巢门。支线：① 与部落临时停火；② 绘制深呼吸安全区草图；③ 安插灵魂石点。

危险度/规避：极高。无护符者禁入；见升空预兆即按阶段站位。

任务方向/奖励：巢穴通行确认、火抗全套检查清单、T1.5～过渡装。

**七阶（奥妮克希亚的巢穴 · 三阶段屠龙）**

切入身份/时点：全团终局编制，进入唯一 Boss 厅。

初始事件：奥妮克希亚说「真幸运」后开战；65% 升空时深呼吸点名你所在的侧翼；幼龙与巢穴卫兵同时压主坦——有人必须在恐惧落地前把伤害打进 40%，同时决定头颅送给暴风城还是奥格瑞玛。

开场白建议：「硫磺灌进肺里。黑龙公主的翼膜遮住岩浆的光，声音却像宫廷低语：真幸运，通常我得出巢觅食。火球落下时有人喊深呼吸，有人喊收狗，有人只顾看她的眼睛——那双眼睛曾在王座右侧笑过。」

关键NPC立场：**奥妮克希亚**（终局敌）；团队指挥；阵营特使（要头颅）；黑龙残党（劫颅）。

主线钩子/支线：钩子=三阶段击杀并安全转移头颅与鳞片。支线：① 保护被火球点名的治疗；② 控制幼龙潮不进蛋室灭团；③ 抢在政敌前完成献首。

危险度/规避：贴近顶点。对奥妮克希亚可条件性胜利（护符+团队战术+装备）；对死亡之翼仍情报优先。

任务方向/奖励：奥妮克希亚的头颅、鳞片、T2 头盔、死亡使者级武器、政治称号向声望。

**八阶（斩首之后 · 城墙与兄长）**

切入身份/时点：已斩黑龙公主的传奇级契约者，处理头颅政治与遗产。

初始事件：头颅钉上暴风城墙的当夜，黑石山方向有密使同时找上你——奈法利安要「妹妹的遗物清单」，伯瓦尔要「普瑞斯托合作贵族名单」，吉安娜要「确认安度因不再被龙息追踪」；一份写着死亡之翼真名的残页在温德索尔遗物里发烫。

开场白建议：「城墙上的龙头开始发臭，诗人却把它写成黎明。沼泽那头仍有幼龙夜啼。三封信同时拍在你桌上：黑石的火漆、要塞的狮徽、塞拉摩的锚——每一封都在问同一个问题：你杀的是龙，还是只撕下了假面？」

关键NPC立场：**伯瓦尔·弗塔根**；**奈法利安**（远景压力，不轻易现身）；**吉安娜·普罗德摩尔**；黑龙血脉残党；**死亡之翼**（名讳级阴影）。

主线钩子/支线：钩子=稳定暴风城清洗与龙沼封巢，防止政治名单与龙蛋外泄成灾。支线：① 协助布罗尔式封巢；② 仲裁头颅/鳞片归属；③ 追踪死亡之翼遗产线索（只情报，不决战）。

危险度/规避：极高～超阶边缘。**死亡之翼=超阶巅峰：存在·情报优先/条件性胜利**；奈法利安单挑亦属灭团级，宜借势不硬刚。

任务方向/奖励：王城嘉奖、封存的贵族黑名单、通往黑翼/血环时间线的情报锚（非强制进入）。
