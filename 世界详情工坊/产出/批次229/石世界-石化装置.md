# 石世界-石化装置

## 剧情

**【作品来源】**
《Dr.STONE 石纪元》（日文：Dr.STONE，中文亦称新石纪）由稻垣理一郎原作、Boichi作画，于集英社《周刊少年Jump》2017年14号至2022年14号连载完结，单行本全27卷、本篇232话加番外；动画由TMS娱乐制作，多季覆盖至登月与终局线。类型为后启示录科幻冒险少年漫画：以「科学重建文明」为引擎，穿插理念对撞、资源战与智斗。本条目口径聚焦**「石化装置」**：全球石化现象的机制、复活液后勤、美杜莎（Medusa）装置争夺、WHY人（Why-man）指令源与月球真相相关冲突，而非百科式全史逐话复述。文风：实验过程爽感、口播科学步骤、信念对决、禁止伪科学乱编反应。

**【世界定位】**
约2019年6月，一道神秘绿色石化光波及全球，现代人类几乎全体化为石像，文明崩解。约3719年后（约公元5738年），科学少年石神千空以「每秒数数」保持意识并苏醒，联合大木大树等人制作复活液，立志以科学复活全人类、重建文明。力量主义青年狮子王司则主张只复活「纯净青年」、摧毁成人石像，双方爆发石之战争。战后科学王国造出珀尔修斯号远航，在宝岛首次夺取可人为操控的石化装置「美杜莎」，并追查反复以无线电发出「WHY?」与伪千空声指令的WHY人至月球。一句话：**谁掌握石化与解石化，谁就掌握石之世界的生杀、疗愈与秩序。**

**【世界观 · 力量体系】**
本世界几乎没有传统「灵力境界」。战力与「破坏/控制力」来自四条并行轴：

1. **科学产出轴**：从棍棒、陶器、火药、抗生素、无线电、蒸汽/内燃载具、战车、舰船，到人工钻石电池、超合金、登月火箭与计算机。科学不直接等于个人肉体强度，但决定后勤、情报、远程火力与战略选项。千空本人肉体脆弱，却是文明级战力的大脑。
2. **个人武勇轴**：狮子王司（人类最强级体术）、冰月（尾张贯流枪术/管枪）、琥珀（村级最强战士、超视力）、宝岛的莫兹与雾雨（投掷与近战顶尖）、史丹利·斯奈德（特种作战与狙击）等。可在无装置介入时以冷兵器/体能主导局部战场。
3. **石化装置轴（本口径核心）**：美杜莎等石化装置可按**语音指令**设定射程与倒计时，发射绿色石化波，将生物（以DNA/生命体为目标）瞬间石化为雕像。装置可投掷、可远程话筒/耳机/合成语音触发；光波穿透障碍、速度可被科学侧测算（千空曾据此卡复活液自救）。电池为特制钻石能源，使用会耗尽变黑。石化可「保存生命时钟」、复活时部分治愈伤势甚至近乎「重置肉体损伤」；真正死亡＝石像被粉碎且无法修复/或复活前被毁到无法复原。装置是**战略级武器**：一次正确设定可清场一支舰队或半座岛，错误设定则同归于尽。
4. **信息与权限轴**：谁持有装置、谁能发出合法英语指令、谁掌握无线电频段与声纹伪装、谁控制复活液原料（硝酸源、酒精/玉米、铂金催化量产）。WHY人侧可从月球对地球装置下达超远距指令（曾尝试约1280万米行星级范围），属规则级威胁。

死亡与复活规则：石化≠立刻死亡；解石化用硝酸系复活液（nital，蝙蝠粪硝酸+酒精等路径，铂金催化可量产）。装置石化—复活循环可疗愈重伤（故科学王国后来用美杜莎治疗濒死的司）。粉碎石像、丢入深海且无法打捞修复、或在错误时机被二次破坏，才是永久剔除。

层次（对照实际破坏力，宁低勿高）：
1. 石化后苏醒初期、棍棒与基础化学、普通石神村村民：≈一阶（凡俗新人）。
2. 石神村精锐战士、司帝国早期青年战士、火药与冷兵器混战：≈二阶（正式战力·超凡起点观感主要来自体能精英）。
3. 石之战争期科学王国（无线电、硝化甘油、战车策略）与司帝国精锐、冰月级枪术：≈三阶（一域中坚，可端据点、改写局部战争）。
4. **持有/争夺石化装置、宝岛独裁与跨海战略武器冲突**：**四阶**（本条目覆盖）——装置一次启动可摧毁据点级有生力量、掀平建筑群级战场；持有者≈精英Boss级战略存在。
5. 全球再石化指令、美洲军工集团（杰诺/史丹利）、登月与WHY人机械生命集群：≈五阶及以上至规则级背景；WHY人/行星级石化权＝超阶约束，情报与谈判/条件性胜利优先，不可当作「随便砍死的小怪」。

乐园阶位映射：石器时代武斗与村级战≈一~二阶；科学战争中期（石之战争）≈三阶；美杜莎争夺、跨海权与装置启动权≈**四阶**；全球石化波/军工文明/登月对峙≈五~六阶及更高背景；WHY人装置集群与行星级指令＝超阶（存在·情报优先/条件性胜利）。宁低勿高；顶点忠实为WHY人侧装置权柄，不许压成「普通敌人」。

补充战力注记：司级体术在无装置场可压倒几乎所有凡人战士，但面对已启动的石化波仍必须躲、隔、抢时差——肉体再强也抗不过「变成石头」。史丹利的热武器与特种战术在美洲线可清扫科学王国临时阵地，仍低于「一次指令覆盖全岛」的装置战略价值。因此四阶的质变分水岭不是某人的拳头，而是**美杜莎控制权**。

**【地理 · 舞台】**
- **石化后的日本列岛**：城市被藤蔓与动物夺回，公路断裂，石像林立。大树与千空苏醒周边的山林洞穴（蝙蝠粪硝酸源）是复活液原点。旧东京方向残存的现代废墟提供金属、电池残骸与「文明曾存在」的视觉冲击。
- **狮子王司帝国领地**：以力量统治青年战士，摧毁成人石像，占有关键硝酸洞窟等战略资源。帝国更像移动的猎人营地+洞窟要塞，而非官僚国家。
- **石神村**：位于远离旧都市的村落，居民为宇航员后裔（石神百夜等六人求生线），以「百物语」口述传承科学碎片；有村祭比武、巫女琉璃、战士团与工匠。村外有桥梁、温泉、矿点与「科学设施」逐步被千空改建。
- **奇迹之洞 / 硝酸相关洞窟**：早期量产复活液命脉，石之战争焦点；失去它等于失去扩军速度。
- **珀尔修斯号航线**：科学王国远洋科学船，连接日本—石油点—宝岛—美洲；船上有实验室、通信、仓储与临时监牢功能，是移动国都。
- **宝岛（Treasure Island / 石化王国）**：宇航员分支后裔聚落，Soyuz舱藏铂金「宝藏」；岛上有「主人」权威、选秀婚配制度、战士雾雨/莫兹、独裁者荆棘（Ibara）操控唯一可用美杜莎。关键微场景：码头与断崖（Wavebreak 类地形）、后宫殿廊、石像刑场、无线电塔、海底碎像区。四阶主舞台。
- **北美 / 玉米都市构想与杰诺势力区**：前NASA科学家杰诺与特种兵史丹利的军工科学城，玉米＝酒精原料＝复活液放大；与宝岛「装置战」不同，这里是枪炮与科学体制对撞。
- **南美（石化源方向）**：原初石化光与装置坠落相关地理线索，美杜莎「金字塔」等遗迹暗示装置曾被集中投放。
- **月球**：WHY人信号源与终局对峙舞台；四阶只以「天线指向的白点」出现，不落地开战。
本口径四阶重点锁在：**宝岛海域—岛上宫殿/断崖—珀尔修斯号甲板—无线电塔—装置光束轨迹**。契约者活动半径建议不超过「一日船程+岛内徒步」，以免剧情提前跳美洲。

**【世界剧情线】**
① 石化与计数苏醒（序章）
2019年前后，绿色光波在全球蔓延，人类在恐惧与不解中石化。石神千空在石化中保持意识，以「一秒一秒」数完约3719年，于5738年春苏醒。大木大树几乎同时挣脱（对杠的执念支撑），两人重逢。千空用科学解释：需要硝酸与酒精配制复活液。他们从蝙蝠粪提取硝酸、用土法酒类完成第一批解石化，复活小川杠。面对狮子与野兽威胁，他们复活「最强高中生」狮子王司。司以一人之力开路，却很快暴露理念：新世界应清除腐朽成人，只留青年。千空坚持复活全人类。冲突升级——司折断千空颈椎，千空以假死与残留石化治愈能力逃过死局；大树与杠表面屈服，暗中收集石像情报。

② 石神村与科学王国奠基
千空孤身进入石神村势力圈，以「妖术使」铬为切入点展示科学（玻璃、药品、电力雏形）。村巫女琉璃病危，千空用抗生素路线救命，换取信任与村长之路。百物语揭示：千空养父石神百夜等宇航员未在轨道上被石化，返回后建立村落并把科学以故事形式留给后代。科学王国成型：发电、通讯、工艺分工；浅雾幻（心灵魔术师）被司侧派来却倒戈；七海龙水被复活执掌航海与欲望驱动的资源扩张。五智将/三贤人框架逐渐清晰——思考、策谋、决断、发想、工艺与武力协作。

③ 石之战争（Stone Wars）
科学王国对司帝国：无线电攻心、硝化甘油与战车、人质与洞窟争夺。司帝国青年战士剽悍，擅长伏击与体能冲锋，但缺乏远程情报网。千空侧先以「声音」打碎士气——让帝国士兵听到科学、听到家人、听到另一种未来；再以化学火力与载具撕开口子。战争中段出现多条支线：间谍幻的双面牌、工匠连夜赶工、战士团护送运输线、硝酸洞窟的反复易手。战争后期冰月背叛重伤司；千空与司短暂联手击败冰月，证明「科学与力量可以暂时站在同一侧」。千空治好司的妹妹未来，换取理念松动：司开始承认科学能救具体的人，而不只是抽象的「纯化世界」。司因伤势过重被科学冷冻保存，等待未来医疗/石化疗愈。帝国人口并入科学王国，文明人口基数第一次真正扩张。此阶段确立主题：科学不是否定力量，而是把力量纳入可复制的体系；也留下伏笔——司的致命伤最终要靠石化装置的「石化—复活疗愈」闭环解决。

④ 出航准备与「WHY?」信号
石之战争后，科学王国把目标从「活下去」升级为「找出石化原因并复活七十亿人」。资源清单上出现：石油（燃料）、铂金（复活液催化量产）、粮食与玉米（酒精放大）、金属与电力。珀尔修斯号在龙水主导与全员协作下建成，试航、补给、编制船员。无线电监听中反复出现无意义却令人不安的「WHY?」——后来称为WHY人的存在第一次以声波而非光束现身。百物语与宇航员航迹把航线指向远海中的宝岛：那里既有 Soyuz 舱级「宝藏」线索，也有另一支人类后裔。

⑤ 宝岛登陆与美杜莎初现（口径核心·上）
接近岛屿时，多数船员遭绿色光束石化——岛方持有可主动发射的石化装置。船不是被炮击，而是被「规则」点名：谁暴露在波及半径内，谁就变成货物。千空、浅雾幻、琥珀、索尤兹（Soyuz，岛民血脉相关）等岸上侦察组幸免或脱困，结识岛民少女阿玛莉莉丝（Amaryllis）：她的同伴因反抗「主人」制度被石化，她计划借「选妃/新娘」制度潜入夺取装置。科学王国立刻把目标拆成三块：救人（打捞/修复/复活同伴）、夺宝（铂金）、夺权柄（美杜莎）。

⑥ 荆棘政权与装置战（口径核心·中）
岛上真相：真正的主人（索尤兹之父一脉）早被石化；大臣**荆棘（Ibara）**篡权，以美杜莎实行恐怖统治，粉碎反抗者石像，禁止离岛与创新。战士**雾雨（Kirisame）**专精投掷装置，可在空中卡定时距引爆石化波；**莫兹（Moz）**为最强近战之一，野心勃勃，把外来者当成夺权棋子。科学王国侧以伪装（琥珀/金狼等潜入后宫线）、无线电耳饰、移动实验、无人机、潜水与爆破、打捞海中碎石像再修复复活等手段推进。幻负责剧本与背叛节奏；铬负责土法材料与现场发明；龙水在多次石化中仍执着于船与指挥；琥珀用武力打开缺口；千空计算一切「看起来像奇迹」的窗口。

名场面链：假装置试探叛徒→真假美杜莎信息战→大岚（Oarashi）被逼吞下真美杜莎跑向岛心以执行全岛计时石化→雕像阵列成为测波速的标尺→千空在被石化瞬间泼复活液自救→无人机/遥控抢夺→电话/耳机链路发出英语指令反石化荆棘。这一整条链的核心不是「打赢最强战士」，而是**在倒计时内完成科学闭环**。

⑦ 夺装置、WHY人露脸与指月（口径核心·下）
夺取装置后，科学王国复活岛民与船员，揭露荆棘罪行，索尤兹成为合法主人并结盟。铂金到手，复活液产能质变——从此「复活谁」第一次变成真正的产能问题而非纯理论。无线电联系本岛（琉璃等）时，**WHY人**以合成的千空声音插入频道，尝试对美杜莎下达行星级范围石化指令（约千万米量级）——证明存在可远程劫持装置的敌对智能。现场紧急处置：切断可被听令的链路、控制装置、确认电池。抛物面天线测向：信号来自**月球**。另有古代石化者松风（Matsukaze）等线索表明：数百年前已有多枚装置自天而降，岛民历史本身就是「装置坠落文明」的缩影。四阶任务卡应停在「装置易手、指令权、复活液与铂金、岛民起义」——登月是后续背景威胁，不可四阶直接打赢月球。

⑧ 装置后的世界：美洲、再石化、登月与终局方向
持美杜莎返回后，科学王国用残余充能石化—复活疗愈司，司正式成为战力盟友：力量主义与科学主义第一次在制度上合流。航向美洲：为玉米（酒精）与火箭材料，遭遇杰诺博士（Dr. Xeno）与史丹利部队的先进军工科学文明。杰诺是千空童年科学偶像级存在，冲突因此更痛——不是文盲对科学家，而是两种「重建秩序」的蓝图对撞。史丹利的特种作战多次把科学王国逼到绝境。南美方向逼近原初石化地理点。高压追击下，千空侧利用装置与WHY人周期指令触发**第二次全球石化**；约七年后西瓜（Suika）先醒，独立重制复活液救回核心人员，成为「文明火种」具象。科学侧与杰诺和解协作，全球分工采集、重建通讯与航天。登月小队直面WHY人真相：其为石化装置集群的机械生命/集合意识，曾以石化「赠予永恒」方式筛选智慧宿主；与人类谈判破裂后主体离去，残留个体与人类共同走向更疯狂的科学目标——时光机构想，试图回溯拯救七十亿人。终局基调是文明复兴与「科学永不止步」，而非单纯屠龙。

⑨ 微观可介入（四阶常用）
甲板突发石化碎片、声线真伪辨别、复活液只够一人的伦理选择、选妃间谍线、海中打捞雕像、装置电池变黑的倒计时、伪指令广播、岛民人质、莫兹临时同盟价码、雾雨投掷轨迹计算干扰、铂金舱开启权限争执、是否公开「石化可疗愈」给岛民政治派系。

⑩ 蝴蝶边界
可改：某次抢装置的伤亡、某雕像是否及时打捞、荆棘提前暴露与否、契约者是否暴露超常科技、莫兹倒戈时点。不可无代价改：装置语音指令基本规则、WHY人月球源、石化可疗愈与粉碎即死的底层逻辑、铂金作为量产关键瓶颈的资源逻辑。打碎全部装置＝同时毁掉战略疗愈与威慑工具，科学王国会反对；正确方向是夺取控制权并建立使用伦理（双人口令、电池账本、禁用平民区全功率）。

### 石化装置作战规则（口径）
装置要素：持有者、英语语音指令（距离+时间）、投掷/放置点、电池残量、是否被远程话筒劫持。战场胜负常取决于「谁先完成合法启动」而非谁刀更快。科学王国优势：欺骗、无线电、无人机、测距与复活液。荆棘优势：先占装置、人质、雾雨投掷。契约者超常武力会刺激荆棘提前全岛启动——应转化为「抢时间」压力而非无脑清场。

### 复活液后勤
蝙蝠粪、硝酸、酒/玉米酒精、铂金催化构成产能边界。四阶仍会为「只够救几个人」打架。契约者若带来现代硝酸工业知识可加速，但须付代价：暴露身份、被抢配方、被司哲学质问「谁有资格决定复活名单」。

### 四阶名场面锚点
宝岛选秀间谍、美杜莎空中起爆、龙水多次石化梗、海中碎像打捞、千空波速自救、伪千空声全球指令恐惧、天线指月。

思考（千空）—策谋（幻）—决断（龙水）—发想（铬）—工艺（杠/卡西）—武力（琥珀/司/冰月等）。四阶海战与岛战需要全角色协作，契约者可填补战斗、后勤、间谍、翻译（英语指令）、电子对抗空位。

WHY人与幻、千空都会利用声线。四阶可设计：录音诱饵、口型识别、噪声覆盖、只有特定句式才能启动的装置协议。禁止把装置写成随意心念触发的魔法石。

## 阶位切入点

> 阶位↔体系：四阶≈石化装置（美杜莎）争夺与跨海战略武器级冲突（摧毁据点/建筑群级有生力量的装置战）。世界顶点（WHY人/行星级石化权）为跨阶约束，切入一律「规避登月硬刚·情报与条件性胜利优先」。乐园阶位映射与剧情段一致：村级武斗≈一~二阶；石之战争≈三阶；美杜莎权≈**四阶**；全球再石化/登月≈五阶+背景。

**四阶（石化装置 · 宝岛夺还与启动权）**

切入身份/时点：契约者以「珀尔修斯号临时船员 / 科学王国特遣观察员 / 宝岛反抗居民（阿玛莉莉丝线） / 司帝国残留精锐浪人」之一切入，锚定科学王国船队已抵宝岛海域、荆棘以美杜莎实行独裁、装置即将易手或刚出现第一次甲板石化前后。不要从「千空刚苏醒」开局——那是低阶资源战，装不下四阶装置权柄。

初始事件：甲板上警报与石粉齐飞——一名瞭望员在绿色闪光后变成雕像，有人下意识一脚把碎片踢散。通讯里同时出现两道「千空」的声音：一道令全员集中抢救，一道令立刻把美杜莎扔进海里。浅雾幻在你耳边低压：「只有一个是人。选错，整支舰队变园艺摆设。」你手里刚好有一瓶复活液，体积只够救一个完整成人；雾雨的投掷抛物线正在夕阳里闪。三秒内必须决定：救谁、信谁、是否暴露你听得懂英语指令。

开场白建议：「海风里有硝烟和石粉。美杜莎的金属反光比刀更冷。谁能在三秒内设定距离，谁就能把一整支舰队变成雕像——而你，站在按钮与药水都够得着的地方，第一次听清有人用你听得懂的语言问：Why?」

关键NPC立场：**石神千空**——把你当变量与工具人，完成测距/英语指令/材料任务可换信任，空谈神秘学会被直接排班去搬硝酸；**浅雾幻**——要你入戏欺骗荆棘或雾雨，撒谎是生存技能，也会把你卖作诱饵；**七海龙水**——要船、要胜利、要利益，可交易指挥权与赌约；**琥珀**——优先救人与斩首暴政，厌恶屠民；**荆棘**——要维持独裁与后宫秩序，外来者＝石化或奴役；**雾雨**——职业投掷杀意，真相前难策反；**莫兹**——开价美女/权力/装置份额；**西瓜**——情报与希望符号，保护她＝站队科学；**阿玛莉莉丝**——起义向导；**冰月**（若随行）——杀手逻辑，效率优先；**铬**——拉你做危险实验助手。

主线钩子/支线：夺取或夺回美杜莎；阻止错误全球/全岛指令；在波速窗口内救回被石化同伴；保护铂金与复活液产能；协助索尤兹揭露真主人石像。支线＝选妃间谍与化妆潜入、海中碎像打捞修复、声线伪装战、装置电池抢修、与莫兹的有限同盟、岛民人质交换、伪千空声源定位。蝴蝶点：是否让荆棘说出真主人已被石化、是否在全岛波前把龙水/关键工匠优先复活、是否公开石化疗愈秘密。

危险度/规避：极高——瞬发石化与粉碎即死；须规避WHY人通讯源正面对线、无防护全岛启动、四阶直接登月决战。世界顶点WHY人＝**超阶巅峰：存在·情报优先/条件性胜利**；保持其完整威胁，只能靠切断指令链、耗电、藏匿装置、谈判窗口等条件手段处理，不得写成「随便打倒的残血怪」。

任务方向/奖励：装置短时使用权或监护权、复活液配方改良线索、铂金份额、科学王国声望、宝岛盟友与航线安全、英语指令协议片段。禁止永久无限制全球石化权、禁止稳定量产「无限电池美杜莎」、禁止开局赠送时光机。

补充执行要点：四阶战斗应以抢装置、电子/声码对抗、人质博弈、无人机与投掷线干扰为主；展示超常个人武力会促使荆棘提前启动全岛倒计时。打碎所有装置会被科学王国视为战略犯罪。正确通关姿态＝夺取控制权+建立「谁有权按按钮」的伦理与技术保险（口令、双人确认、电池监管）。若契约者站荆棘一侧，任务改为「维持独裁N日」并承担科学王国反攻与WHY人劫持双重压力，奖励偏向黑暗权柄但会锁死后期同盟。
