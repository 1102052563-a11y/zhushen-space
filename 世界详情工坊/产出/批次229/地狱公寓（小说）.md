# 地狱公寓（小说）
<!--meta lib=主库 tiers=一、二-->

## 剧情

**【作品来源】**
《地狱公寓》是网络作家黑色火种（真名董协）创作的长篇恐怖悬疑小说，2010年6月6日起在起点中文网连载，2012年8月16日完结，共571章、约213.8万字，分三十卷，终卷为《魔王》。后有中国戏剧出版社、九州出版社等实体书版本。同作者关联作品含《恶魔公寓》《地狱电影院》《死咒岛》《444号医院》等，其中《恶魔公寓》承接部分公寓与地狱入口设定。文风：规则怪谈、单元剧式血字任务、高密度惊吓、人性猜忌与牺牲。它是早期中文网络「规则流/无限恐怖」代表作之一，强调真鬼真咒、文字陷阱与生路解析。

**【世界定位】**
现代都市（主舞台K市及周边）中存在一栋只有被选中者才能看见并进入的「地狱公寓」。公寓外表豪华正常，对住户却是无法长期逃离的牢笼：房间墙壁会不定期浮现血字，强制住户前往凶地执行生存任务；违者被影子或厉鬼杀死。主角李隐是网络小说家，因目睹车祸等因缘入住后，在一次次血字中从普通人成长为谋略型幸存者，与同伴争夺地狱契约碎片、最终以巨大代价封印魔王。一句话：**被公寓选中就踏上不归路；血字是剧本，影子是刽子手，契约是唯一的刀。**

**【世界观 · 力量体系】**
本世界几乎不存在「修炼升阶」体系。住户不能靠打坐升级，也不能靠买枪成为超凡者——公寓的杀伤主通道是**规则与诅咒**，其次才是凶地厉鬼的直接猎杀。力量与威胁层级如下：

1. **被选中的住户（凡人）**：普通人，无超凡肉体。生存力取决于观察力、规则解析、团队协作与心理韧性。典型表现：徒手搏斗级，热武器对厉鬼多数无效，对影子诅咒无效。≈乐园一阶（Lv.1–10）。
2. **老住户 / 情报网 / 小团体**：经历多次血字、掌握生路笔记、隐秘道具（人偶、镜子、头盔、手机摄像等）与隐藏规则（如用自身血字记录救人、血画倒十字取消他人正在执行的血字）。仍是肉体凡胎，但信息战与资源战远超新人。≈乐园一阶后期～二阶初期。
3. **特殊能力住户**：如蒲深雨的「预知画」、完成一定次数后的「瞬间回归」类权限、仓库卡相关权限等。能力高度场景绑定，不是全能异能，且常附带诅咒反噬（如预知画关联蒲靡灵、每年特定日期风险）。≈二阶。
4. **普通怨灵 / 厉鬼（血字场景）**：附着于凶地、可伪装人形、隐形、附体、文字误导。对普通人即死威胁；有明确生路时可规避。≈二阶威胁（对一阶住人是极高致死环境）。
5. **仓库鬼 / 特殊恶灵 / 心魔投影**：公寓「仓库」相关的伪住户级威胁。住户可持卡进入仓库取攻防道具，但某抽屉一旦打开会放出仓库鬼：杀死住户并**冒充其身份**，不受普通公寓规则完全约束，可污染后续血字、制造无解局，甚至持有契约碎片。嬴子夜之死与其直接相关。≈三阶以上背景威胁，一、二阶只作传闻、假身份恐慌与余波。
6. **地狱契约碎片与完整契约**：五十年一现的关键权柄碎片，集齐可合成完整地狱契约，是封印魔王与逃离的钥匙。持有者成为众矢之的。≈权柄级道具，非个人战力。
7. **魔王 / 公寓意志 / 五十年轮回源**：世界顶点。魔王级血字五十年一发，可波及非住户；封印仅能暂时终结本轮轮回，公寓本身不消失。**超阶巅峰：存在·情报优先 / 条件性胜利**，严禁写成「被封印后战力归零」。

**核心规则细则（写正文必守）**：
- **选中机制**：只有被神秘力量选中的人才能看见并进入公寓；成为住户后极难真正离开。
- **48小时法则**：非血字任务期间，连续离开公寓超过48小时，会被自己的影子杀死。
- **血字强制**：房间墙壁不定期渗出鲜血文字，规定时限、地点、动作；违时、违令、失败即死。
- **文字陷阱**：血字常把「东西/乘客/姿态/正常」等词做成多义；真生路往往与流言相反。
- **双轨逃离**：①成功完成十次血字；②集齐地狱契约碎片并在魔王级节点封印魔王。后者绑定五十年周期。
- **隐藏救人规则**：住户可用自身血字执行记录（割指血画倒十字等）取消另一住户正在执行的血字甚至将其传送回公寓，代价巨大，且被救者该次不计入完成次数。
- **魔王级例外**：部分魔王级/模拟场景中，非住户到场未必死；正式魔王夜完成者须在限时内离开否则仍被影子处决。

死亡规则：
- 非任务期连续离开公寓超过 **48小时** → 被自己的**影子**杀死。
- 血字任务失败、违时、违令、踩中文字陷阱 → 厉鬼/影子处决。
- 住户间可互相残杀（为碎片、为顶锅、为取消血字代价）。
- 仓库鬼可杀后假扮，导致「队友已死却仍在说话」。
- 复活极度罕见：续作线索中有灵魂火烛等设定，本世界主线内多数死亡即终局；楚弥天式「主动承担多重诅咒」属于牺牲，不是复活外挂。

乐园阶位映射（对照破坏力：本世界主体是凡人+规则杀）：
- 新住户 / 普通人≈**一阶**；
- 老住户·生路解析者·小团体头目 / 普通厉鬼场景≈**二阶**；
- 仓库恶灵·高阶怨灵≈三阶+（背景）；
- 魔王 / 公寓本体 / 完整地狱契约权柄＝**超阶巅峰**（情报优先、条件性胜利）。
宁低勿高；一、二阶切入禁止硬刚魔王与公寓意志。

**【地理 · 舞台】**
- **地狱公寓（K市）**：核心牢笼。外表高档住宅楼，仅被选中者可见可入。内部有普通居住层、管理员/隐藏空间、**隐藏第30层**（封印魔王关键地点，需钥匙或热武器破锁）。住户房间会单独浮现血字。公寓周边在魔王夜可化为无人区战场。
- **K市及市郊**：李隐等人的日常生活与集合点；废弃公寓（宁风小区鬼镜任务）、暮松小区、真田路–北遥路建筑群、国家历史博物馆、封通大厦等均在此都市圈。
- **幽水村（X市市郊）**：开篇血字地。表面乡土凶案传说，实为水缸女鬼猎场。
- **华岩山**：捉迷藏血字地，山界内外决定「鬼」能否被抹杀。
- **银月岛（S市海域度假开发地）**：隐形鬼海域，手机/摄像可显形。
- **直永镇**：梦境式妖异。
- **东明镇—月华村 145 路末班车**：午夜巴士，碎片与人偶盒。
- **江枫制衣厂**：魔性嫁衣与第三碎片。
- **月影馆 / 日冕馆**：送信血字，性别礼服陷阱。
- **地狱古堡**：魔王级模拟场景，入则无生还（华连城死于此）。
- **六号林区 / 宛天河桥**：恶魔的传承、第四碎片。
- **月光号幽灵船、星辰别墅、大暮黑岭叶神村、朴夏山、叶风山剧组、里昂市（法国）** 等：中后期单元凶地，体现血字可跨国、跨媒介（游戏、片场、博物馆）。
- **巴黎**：终局后柯银夜/柯银羽定居地；「绿」等待许允转世（柯允）之处。
- **仓库（公寓内部特殊空间）**：持卡可入，表面提供攻防道具，实为仓库鬼温床；某抽屉是禁忌。
- **公寓隐藏第30层**：终局封印点，钥匙或热武器破锁；一、二阶只闻其名。

一、二阶主舞台几乎不出「公寓 + 早期血字凶地」；高阶地理仅作情报。地理写法铁则：先定契约者在公寓内还是某血字凶地，再决定冲突是「影子/邻里」还是「厉鬼/生路」；不要一上来跨国片场或30层。

**【世界剧情线】**

① **入住与第一滴血：幽水村（2010.6）**  
网络小说家李隐因机缘（目睹车祸等公开资料所述因缘）被公寓选中，成为住户。公寓规则初次揭晓：血字、48小时、影子。第一卷血字要求 6月7日至7月7日在幽水村住满一个月。同伴含秦守天、罗恒炎、叶可欣等。村中流言称李冰自杀成厉鬼、靠近阿秀最安全——实为**阿秀家水缸女鬼**杀人，李冰另有隐情。名场面：水桶底惨白女面、同伴逐个失踪、有人死在公寓门前。生路：远离阿秀（或让阿秀认清真凶）。李隐生还；罗恒炎、叶可欣、秦守天等死亡。阶段因果：主角学会「表面安全往往是死路」，开始以写手式拆解文字；读者/契约者第一次理解——**流言是公寓的第二套血字**。

② **瞳、捉迷藏、应誓：规则课（2010.9–11）**  
「瞳」：2010.9.15–20 别墅住满五日；鬼可化作「完全正常的东西/人」；夏渊先遣已死被替换，镜中见真身即死。生路：不看真容（闭眼）或不主动搜「伪装」。嬴子夜生还；音桐幸子、唐文山等死。李隐与嬴子夜理性—直觉互补搭档成型。  
「捉迷藏」：2010.11.1 华岩山顶，选「鬼」、不离山，零点后回；生路涉及引鬼出山界抹杀，或开局有人主动当鬼（资料存疑）。唐兰炫、杨临等生还。  
「应誓」：2010.11.3 全日不发誓，违者一周禁入公寓——公寓开始用「生活化禁令」折磨住户，把社交语言变成地雷。  
阶段结局：住户意识到血字会玩文字游戏；临时同盟极不稳；「正常」二字成为全书最高危形容词。

③ **银月岛与直永镇：工具与梦境（2010.11–2011.1）**  
银月岛：2010.11.12–14 须在S市海域度假开发地待至14日12:30；鬼隐形，手机/摄像可拍到并规避；与华连城、伊莣过往绑定。李隐、华连城、伊莣、嬴子夜生还；欧阳靖死于公寓门，段一哲、苏朗等死。  
直永镇：2011.1.1–8；妖异皆梦，知梦则破。柯银羽、梁冰等生还。  
阶段因果：科技道具与认知战进入工具箱；华连城—伊莣线埋下后续古堡悲剧；柯银羽开始成为可复用的生还名册核心。

④ **午夜巴士·鬼镜：地狱契约碎片登场（2011.1）**  
145路末班车（东明镇→月华村）：须在1.15零点前上车；血字强调「乘客中有厉鬼」携**第一碎片**——文字把注意力从司机旁引开。生路：司机旁盒子装执行者人偶，让鬼杀人偶。柯银夜（自撑伞老妇侧得碎片）、夏小美生还；林灵、吴小川、梁冰等死。  
废弃公寓鬼镜（宁风小区）：1.16 0:00–6:30，**第二碎片**；两镜相对困鬼于无限镜像。李隐（七楼衣柜取碎片）、嬴子夜、柯银羽生还，杨临死。  
阶段结局：碎片把「共同求生」升级为「零和争夺」；柯氏兄妹与李隐—子夜线成为两条核心人脉；此后每一次结盟都先谈碎片归属。

⑤ **不要回头、魔性嫁衣、断头魔：代价升级（2011.2–4）**  
不要回头：2.4 20:00–0:00 真田路–北遥路建筑群；回头则眼见幻觉；地上头盔可换回眼。卞星辰生还。  
魔性嫁衣：3.2 0:00–12:00 江枫制衣厂，**第三碎片**在红嫁衣内；衣架可近距封印鬼。嬴子夜、蒲敏等生还。  
断头魔：4.1–15 六人须持K市1–3月六起断头案死者人头之一回公寓，否则影子自杀；断头魔可拼接五官，防假目击者。柯银夜、柯银羽、华连城、皇甫壑等生还。  
阶段因果：任务开始强制道德绑架（人头、嫁衣），住户间信任崩坏加速；「完成血字」与「做人」第一次系统性冲突。

⑥ **信与地狱古堡：魔王级预告（2011.4）**  
「信」：4.17–18 单数房去月影馆、双数去日冕馆，为地下室鬼送信（窗递、三小时内回，可拆/伪/扣）；信内容逼疯；须识破性别礼服与曾未幸男、任里昂女等名陷阱。李隐、嬴子夜、上官眠、慕容蜃、封煜显等生还。  
「地狱古堡」（魔王级模拟，4.19 12:30–18:00）：非公寓正式发布的无解局，入则无生还；**华连城死亡**。非住户亦可能卷入但不一定死——规则开始溢出公寓，五十年后再发同类。  
阶段结局：众人首次直面「无解」与魔王级概念；华连城之死重创团体士气，银月岛生还组碎裂。

⑦ **恶魔的传承至接近：碎片战争（2011.5）**  
六号林区/宛天河桥（5.1前后）：**第四碎片**埋桥端，盒子内人偶+匕首刺穿消鬼。卞星辰、神谷小夜子、皇甫壑生还。  
幽灵船月光号（5.4）：笔仙咒死颜凌霄以抵其咒；柯银夜生还。  
魔棺星辰别墅（5.7）：魔棺现后须塞活人眼入星辰眶；嬴子夜、卞星辰、蒲深雨生还。  
七个人（大暮黑岭叶神村）：毁铁链破「七」咒；神谷小夜子、裴青衣生还。  
水墨画：取对应房号画随身，至画中地按女子位杀之；上官眠、安雪丽、邱希凡生还。  
索魂咒（5.23）：跟严琅/千汐月百米内；生路是垂直距≤100米（鬼只杀横向）。李隐、蒲深雨、封煜显生还。  
接近（5.24 2:00–5:00）：找能条沙绘（身有碎片）；鬼附后脑勺发遮，不被见则不死。柯银羽、神谷小夜子、罗密梓、林天泽生还。  
阶段因果：蒲深雨「预知画」与外公蒲靡灵线索浮出；神谷小夜子、上官眠成为后期关键武力/冷静变量；碎片持有者名单公开化。

⑧ **后面是谁至化妆舞会：仓库鬼阴影（2011.6–7）**  
暮松小区「后面是谁」（6.8）：鬼在两房间特殊空间，破中间墙阻其出；皇甫壑死，生还记录极惨。  
颜玉疗养院游戏「不归村」（8.20前后规则极繁）：选项生/死、存档多数决、伤即实伤；防游戏角色入周边或夺「冥钞」。神谷小夜子、罗兰、风烈海、徐饕、公孙郯、嬴子夜等生还记录见诸解析。  
朴夏山「恐怖小说」：仓库恶灵改楚弥天原著触发死路；**卞星辰死**。  
封通大厦化妆舞会：须完全化妆难辨含性别、分头走、禁言身份；真鬼混入；卡片持时最短者为鬼。  
阶段结局：仓库卡/仓库鬼线全面爆发——鬼可杀住户并假扮，不受普通规则完全约束；公寓不再保证「有生路」；住户从解谜转向亡命与献祭逻辑。嬴子夜中后期死于仓库鬼相关事件，其形象/心魔在结局卷仍撕扯李隐。

⑨ **异国凶劫、灵异片场、古尸、蜡像、你是谁：失控与总清算（2011.7–10）**  
里昂市「异国凶劫」：七杀手追杀（实则全鬼），禁遮容；非住户易容可躲。李隐、上官眠、洪相佑生还。  
香港叶风山「灵异片场」：超公寓控但规则仍效；删减女鬼仍按原剧本杀；坚持至女主死则龙套活。柯银夜、上官眠、徐饕、洛亦水生还。  
国家历史博物馆「古尸苏醒之夜」：两古尸互寻，遇则厉鬼全灭；白焰火种引路。柯银羽、神谷小夜子生还。  
死诞之馆「蜡像」、摩尔斯电码「你是谁」：仓库恶灵致失控无解，拉无辜。2011年末血字进入总清算。  
阶段因果：李隐等人从「完成十次即可逃」转向「必须杀死伪人、集齐地狱契约、迎接魔王」。

⑩ **仓库鬼之死与契约合成**

一阶池：首夜血字解读会、假生路传单、48小时卡点归巢、影子延迟、邻居借盐试探、幽水村水缸异象、别墅镜面禁忌、华岩山边界石、应誓日禁语社交、银月岛手机没电危机。  
二阶池：人偶盒争夺、双镜材料黑市、碎片持有者点名、血画倒十字救人交易、仓库卡诱惑、古堡劝退会、月影馆/日冕馆分房、嫁衣碎片绑架、断头案目击者作伪、预知画一页天价、上官眠逼宫前夜。  
事件写法：先写谁要什么，再写规则字面，再写流言反转，再写伤亡与信任变化——禁止写成无规则的砍杀爽文。

## 阶位切入点

> 阶位↔原作强度：一阶≈新住户/普通人（肉体凡胎，刚接触血字与48小时规则）；二阶≈老住户·生路解析者·小团体头目 / 普通厉鬼血字场景。世界顶点（魔王 / 公寓意志 / 完整地狱契约权柄）为**超阶巅峰**，低阶切入一律「规避顶点·情报优先 / 条件性胜利」，严禁「封印后战力归零」式削弱。本世界仅覆盖：一阶、二阶。

**一阶（新住户 · 幽水村前后 / 第一滴血）**

切入身份/时点：契约者以「刚被公寓选中的新住户」切入，锚定李隐入住前后至**幽水村血字**执行期（2010年6–7月），或等价的「房间首次浮现血字」之夜。

初始事件：你拖着行李推开只有「被选中者」看得见的公寓门，房门在身后自己锁上。深夜，墙壁渗出一行尚未干涸的血字——要求你在期限内前往X市郊幽水村住满一个月。走廊里，秦守天一类同期住户敲门约伴；有人塞给你「靠近阿秀就安全」的假生路。窗外，你的影子比身体慢了半拍。

开场白建议：「租金低得不像话，中介却从不回电话。你听见楼上拖动的脚步，低头时影子慢了半拍——血字还在渗。有人说去幽水村、跟着阿秀就没事；可合同里没有逃生条款，只有一句：违约者，由公寓全权处理。」

关键NPC立场：**李隐**——同期新住户，多疑但肯分享观察，可组队也可在假情报上犹豫；**秦守天 / 罗恒炎 / 叶可欣**——早期同伴，热情或慌乱，极易成为死亡教材；**阿秀**——村中关键人物，靠近即踩死路；**嬴子夜**——若时点稍后登场，理性分析型可结盟对象；**公寓意志（非人）**——只通过血字与影子发言，不可对话。

主线钩子/支线：活过第一次血字并记录真生路（远离阿秀/水缸女鬼）；支线＝识破「李冰自杀厉鬼」假说、交易48小时内归宿闹钟、拒绝用同伴顶影子。

危险度/规避：中高——死于规则与假生路，而非武力碾压；须规避主动寻找魔王、破坏公寓承重式硬刚、把乐园道具借给未知住户「试影子」。

任务方向/奖励：活过首月、带出可用的生路笔记片段、一把能在银月岛类任务显形的摄像工具线索；奖励取向＝信息、保命小道具、短期同盟；禁止发完整地狱契约。

**二阶（老住户 / 碎片博弈 · 午夜巴士至信）**

切入身份/时点：契约者以「已完成至少一次血字的老住户」或「小团体情报掮客」切入，锚定**2011年1月午夜巴士·鬼镜**至**4月信/地狱古堡**前后，地狱契约碎片已入场、住户从求生转为夺权柄。

初始事件：145路末班车的人偶盒传说传遍楼层；有人在废弃公寓用双镜困住鬼后，怀揣第二碎片失踪。柯银夜与柯银羽在门厅对你点头——那是出价，不是友谊。墙上新血字要求你在月影馆/日冕馆之间选边送信，而信的内容据说能逼疯读信人。你口袋里的生路笔记在发烫：下一行写着「古堡无生还」。

开场白建议：「你已经会在第47小时赶回家，也会在借盐之前先问对方上次血字死了谁。可当第一块地狱契约碎片出现时，走廊里的笑脸全变了味——有人要你上午夜巴士，有人要你把镜子对着镜子，有人只要你在古堡门口站成诱饵。」

关键NPC立场：**李隐**——已成谋略核心，愿交换信息但防被当枪；**嬴子夜**——逻辑拆招优先，讨厌莽撞外挂；**柯银夜 / 柯银羽**——高信任二人组，对外一致，碎片上极硬；**华连城 / 伊莣**——银月岛生还者，古堡是红线；**上官眠**——武力聚碎片倾向，合作需先谈分配；**卞星辰 / 蒲深雨**——预知与代价，情报极贵；**魔王/公寓（传闻）**——只以无解局与五十年轮回压迫，**超阶：情报优先**。

主线钩子/支线：参与或破坏某次碎片夺取（巴士人偶/鬼镜双镜/嫁衣第三碎片）；翻译「信」的性别礼服陷阱；组织抵制地狱古堡无解招募。支线＝鉴定假生路笔记、用血字记录高价救人（承受代价）、调查仓库恶灵污染的早期征兆。

危险度/规避：高——死于碎片火并、文字陷阱与无解局；须规避地狱古堡强闯、魔王夜30层、长期对抗公寓意志。**超阶巅峰：存在·情报优先/条件性胜利**。

任务方向/奖励：碎片线索（非完整契约）、双镜/人偶类可验证道具、老住户级生路合集、进入中层情报网的「信用」；禁止直接给「删除公寓/秒杀魔王」权柄。任何炸楼方案应触发规则反噬并失败。
