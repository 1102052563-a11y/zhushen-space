## 剧情

**【作品来源】**
《远星号》（Starsector，早期名 Starfarer）由 Fractal Softworks 开发，主创 Alexander Mosolov。2011 年 Alpha，2013 年 4 月进入 Early Access，Java/LWJGL 跨平台，主要经官网长期更新。类型：俯视角太空舰队战术 + 开放世界贸易 / 探索 / 殖民 / 经济与派系战争模拟。文风：后崩塌硬科幻、派系冷战、废土星海、玩家自写史诗。标签：Domain 崩塌、Persean Sector、Hegemony、Tri-Tachyon、Luddic、Remnants、Omega、AI 战争、Gate、殖民地危机、全面战争。本档案属主库，**仅覆盖六阶**——切片对准 **Remnants 高危巢穴、Omega 终局遭遇、派系全面战争与扇区级殖民霸权冲突**；一至五阶日常贸易 / 护航 / 系内会战不在本条展开（可与批次 207 低阶档案并用，但本条不写低阶切入）。忠于原作公开设定与可玩沙盒结构，不编造 Domain 全盛舰队路边摊化，不编造单一电影式强制终局。

**【世界定位】**
人类 **Domain of Man（人类领域）** 曾以 **Gate（星门）** 虫洞网络维系银河殖民文明。约 **206 个周期（cycle）前**，全部门同时失效，史称 **The Collapse（崩塌 / 大崩溃）**。舞台是银河一隅、原属边境的 **Persean Sector（英仙座 / 珀尔修扇区）**：核心世界（Core Worlds）五大人权势力战略僵持；边缘是海盗王朝、路德狂徒、Domain 遗迹与被官方史 **[REDACTED]** 掉的 **Remnants（AI 残党 / 遗迹机群）**；更深一层是伪派系级他者 **Omega** 与 **Coronal Hypershunt（日冕超能采集遗构）**。六阶契约者不是码头跑商，而是能改写战局的舰队所有者——多主力混编、殖民地工业链、高危信标系统与全面战争侧翼的参与者。一句话：**星门熄灭后，舰队就是国家；到六阶，国家开始互相吞并，而算法与奇点正在边缘等你。**

**【世界观 · 力量体系】**
力量本源纯科技与组织：舰船体级与武器（实弹 ballistics / 能量 energy / 导弹 missiles）、护盾与装甲、**Flux（通量，类废热 / 能量负荷）** 管理、舰员素质、**军官（Officer）** 技能与人格、船体改装（Hullmods）与改装点（OP）、派系科技梯度、**AI 核心（Gamma / Beta / Alpha）**、后勤（燃料 fuel、补给 supplies、船员 crew、重机械 heavy machinery）。无魔法超人、无念力飞升；“战力”= 舰队构成 × 指挥 × 后勤可持续性 × 工业与声望账本。

战斗规则：俯视角实装战术。护盾吸收能量 / 导弹更优，装甲抗实弹；过载（overload）等于短暂失能窗口——六阶会战里被导弹饱和撕开过载窗是常规死因，不是玄幻读条。舰毁可救部分乘员；玩家旗舰全灭通常以逃生舱与残存舰队“重开”，但殖民地、声望、蓝图与军官损失真实且不可刷。成长：舰长技能树、故事点（Story Points）用于破例行动与关键改造；蓝图（Blueprint）与纳米锻造（Nanoforge）决定工业上限；**Automated Ships** 与 AI 核心军官是六阶质变器。

战力阶梯（破坏力表现，宁低勿高；本条只展开六阶可玩层，低阶仅作背景气压）：
1. 武装小艇 / 走私单舰 / 海盗猎犬编队：劫商、毁小型站。≈一阶（背景）。
2. 护卫舰为主的正规巡逻 / 佣兵狼群：压过地方治安。≈二阶（背景）。
3. 驱逐 + 巡洋为主的作战舰队：恒星系内会战、袭击殖民地轨道设施。≈三阶（背景）。
4. 主力战列 / 战巡 + 完整后勤的派系打击群 / 成熟殖民舰队：摧毁轨道基建、迫使殖民危机。≈四~五阶（背景气压，本条不切入）。
5. **六阶本条核心**：高危 **Remnant Ordo** 级机群与完整 **Battlestation / Nexus**、**Omega** 级遭遇（Tesseract 等）、**派系全面战争** 级军势、玩家扇区级多殖民地霸权与饱和轰炸 / 反轰炸能力——覆盖一整片星域战役、地区级灾难、可改写核心世界均势。
6. Domain 全盛门网军势、完整第十四战团编制、未解的奇点级 Omega 本质：超阶对照，不作为可刷常态。

乐园阶位映射：小艇海盗≈一阶；护卫编队≈二阶；巡洋会战≈三阶；主力打击群 / 单殖民地霸权≈四~五阶；**Remnants 高危巢穴 · Omega 遭遇 · 派系全面战争 · 多殖民地扇区霸权 ≈ 六阶**；Domain 全盛 / 完整奇点传说 = 超阶。宁低勿高。世界顶点 = 高端 Remnants 节点、Omega 级遭遇与派系全面战争级军势（六阶可接触，条件性胜利）；严禁把 Remnants 当新手怪，严禁“一炮打服 Domain 幽灵”，严禁把 Omega 削弱成杂兵。

**【地理 · 舞台】**
- **核心世界（Core Worlds）**：扇区已殖民恒星系密集区，六阶全面战争的主舞台。关键系气质：Aztlan（Hegemony 首都行星 **Chicomoztoc**）、Hybrasil（Tri-Tachyon 工业重心如 **Culann**）、Thule（Persean League 重镇 **Kazeron**）、Askonia（Sindrian Diktat 燃料霸权，旧都 **Opis** 已毁，现都 **Sindria**）、Canaan（Luddic 圣域 **Gilead**）、Magec（海盗女王 **Kanta’s Den**）、Galatia（学院与事故后跃迁点不稳）、Arcadia（**Agreus**，独立打捞商 **Callisto Ibrahim** 据点）等。
- **超空间（Hyperspace）**：系间旅行层；风暴、滑流、危险与捷径并存。无 Gate 后，超空间是文明血管也是坟场。六阶战役常因燃料与风暴被锁死在错误一侧。
- **边缘 / 外围高危带**：空行星、Domain 残骸、Explorarium 探针遗产、**Warning Beacon（警告信标）** 标记的 Remnant 系统（1 / 2 / 3 次脉冲 = 低 / 中 / 高危）、路德路径基地、海盗站。**三脉冲高危系统**是六阶主战场：完整 Battlestation、Ordo 级编队、追击逻辑。
- **Coronal Hypershunt**：恒星能量采集遗构；常驻 **两艘 Omega Tesseract** 级守护。不是刷经验副本，是扇区科技天花板遭遇点。
- **特殊地点**：**Alpha Site**（Project Ziggurat 相关隐秘站点，可回收独特相位主力 **Ziggurat** / TTS *Xenorphica* 级气质）、各类 Domain 遗迹与 Cryosleeper 残骸、玩家多殖民地联邦的工业星链。
- **市场与战争经济**：开放市场约 30% 关税；黑市无税但抬嫌疑；军事基地 / 高指挥部子市场需高好感才卖军用舰武。六阶玩家同时是买家、军火商与宣战对象。

**【世界剧情线】**（聚焦六阶：AI 战争遗产 · 残党 · 奇点 · 全面战争）

① **Domain 黄金时代与技术铁律（前史气压）**  
Domain 以 Gate 网络统一人类殖民空间，核心据推测在猎户臂一带，并向人马臂、英仙臂（含 Persean Sector）扩张。千余周期繁荣留下几乎全部现存科技母本：舰船分类（第 4 修订军舰图式仍在用）、纳米锻造与蓝图 DRM、改造生物与器官、恒星镜 / 遮光罩与孢子船等改造行星手段、轨道熔炉灯、土壤纳米体等。学说上先后有 **Decisive Battle（决战学派·低科厚甲实弹）**、晚期 **Cruiser School（巡洋学派·中线机动）**，高科能量 / 相位舰在正规军中相对稀少。Domain 严控 **AI 核心与自动化军舰**（核心区禁自动战舰），却在 Explorarium 勘测与边陲治安上搞双重标准——为日后崩塌后的 AI 灾难埋雷。殖民地被设计成相互依赖：切断门网即可饿死叛星。六阶正文应让人感到：**你拿到的每一份蓝图与每一个 Alpha 核心，都是在偷一个已死帝国的骨头。**

② **崩塌（c.-206）与正统幽灵**  
全部门同时失效，原因官方未知。阴谋论指向 Tri-Tachyon 的 AI、路德派、或与囚犯 **Ludd** 同时被门网转移等传说。Persean Sector 与其他人类中心彻底断联；超空间不稳；先进科技大量沦为不可复制的 lostech。饥荒、盗匪、权力真空同时爆炸。正在轮换驻防的 **第十四战团（XIV Battlegroup）** 等部队被卡在转移途中——他们后来成为 Hegemony 的骨头。六阶叙事里，崩塌不是背景板，而是**所有全面战争的原始债务**：谁也回不去 Domain，谁都在抢 Domain 的尸骨。

③ **第十四战团抵达与霸权建国（约 c.+49 起）**  
XIV 经残酷超空间长征抵达扇区，击溃军阀 **Leonis** 等，以对 Domain「网络失效 / 戒严协议」的创造性解读颁布 Eventide Diktat，建立 **Hegemony（霸权）**：军政府凌驾民选殖民宪章，自认 Domain 正统继承者，继续并加码执行 AI 禁令。首都 **Chicomoztoc**（Aztlan），舰名前缀 HSS，学说继承低科决战线（Onslaught 等重甲炮舰、XIV 特装变体）。Hegemony 是核心世界最大军力与殖民地集群之一，几乎没有真盟友。六阶全面战争中，Hegemony 是最可能同时对你发起 **AI 巡检** 与 **法统讨伐** 的一方。

④ **派系成型：联盟、教派、巨企与燃料国**  
- **Persean League（珀尔修联盟）**：约 c.+83 Mayasura / Maairath 事件后，反 Hegemony 军管的自治同盟成形。政治由 **Kazeron** 大氏族把持，现任 **Prime Demarchon Reynard Hannan**（gens Hannan）。中线舰、导弹与“自治执念”。六阶可成为玩家入盟或被封锁的谈判桌主人。  
- **Tri-Tachyon Corporation（三合星 / 三合会）**：崩塌前就专精尖端 AI、超驱、力场与高能武备的巨企残余；崩塌后吞并其他公司碎片。高科技能量强盾、相位舰、质量优先。CEO **Artemisia Sun** 是第二 AI 战争与战后权斗的核心人物。舰前缀 TTS。**Remnants 的造物主阴影**。  
- **Luddic Church（路德教会 / 银河救赎教会）**：崩塌后约 c.+8 起统合各派，尊 **Ludd** 的禁欲、反技术与农耕救赎；视技术僭越为文明创伤。精神领袖 **Arch-Curate Sophronia Jaspis**（盲女教宗级），武装臂 **Knights of Ludd** 由 **Excubitor Orbis Gideon Oak** 等统领。低科舰，仇视 Tri-Tachyon 最深。  
- **Luddic Path（路德路径）**：视主流教会软弱的极端分裂；**Livewell Cotton** 为最大头目级人物。Safety Overrides 等“疯狂改装”低科舰，恐袭高科技世界，对几乎全员敌对。六阶殖民星若“科技洁净度”不够，Path 会来饱和轰炸。  
- **Sindrian Diktat（辛德里亚独裁）**：Askonia 危机（约 c.+181）中 **Planetkiller** 摧毁 Opis；前 Hegemony 上将 **Philip（Phillip）Andrada**——“辛德里亚之狮”——抗命建国，军政府 + 个人崇拜，扇区顶级燃料产地。后期内容中 Andrada 或陷入昏迷，权力在 **Oxana Hyder、Horacio Caden、Dolos Macario** 等之间撕咬（“The Usurpers”等任务）。六阶战争里，**没有燃料就没有全面战争**。  
- **Independents（独立者）**：中立站、商队、佣兵、勘探者，非铁板一块。  
- **Pirates（海盗）**：松散匪帮与军阀王朝；现以 **Jorien Kanta（海盗女王）** 最著名——前 Domain / Hegemony 后勤军官，Onesimos Loke 入侵时叛变，后再清洗 Loke 余党，盘踞 Magec 的 Kanta’s Den。六阶可买 **Piracy Respite**，也可被她的网络永久标记。

⑤ **第一次 AI 战争（约 c.+89–95）与清剿谎言（至约 c.+107）——六阶史诗的源头**  
Hegemony 以 Domain AI 法强制检查 Tri-Tachyon；League 站 TT，Church 站 Hegemony。TT 战败接受巡检，基础设施重创。战后 Hegemony 与 Knights of Ludd 联合扫荡边缘 AI 舰队，直至约 c.+107 宣布“胜利”、合作破裂。官方史称清剿完成——**这是政治叙事，不是事实**。残存机群退入外围，警告信标按 1 / 2 / 3 次脉冲标低 / 中 / 高危。  
**Remnants 战力谱（六阶必背）**：  
- 舰船：护卫 **Lumen / Glimmer**；驱逐 **Fulgent / Scintilla**（航母向）；巡洋 **Brilliant / Apex**；主力 **Nova（战巡）/ Radiant（战列）**；舰载机 Flash / Lux / Spark 等。前缀气质 TTDS（Tri-Tachyon Drone Ship / Defense System 社区解读）。  
- 舰队层级：Fragment（碎队，小）→ Sub-Ordo → **Ordo（十至十余艘起，可含多主力与大量核心军官）**；摧毁舰队会激怒后续刷怪规模（对完整站点可连升数档）。  
- 站点：中危系统约 50% 损坏 Battlestation；**高危系统必有完整运转站点**，摧毁掉落 **Alpha 核心**，也会永久改变该系威胁生态。  
- 军官：Gamma / Beta / Alpha 核心，精英技能，**不受玩家 Automated Ships 的部署点硬限那套人类规则**。  
- 殖民危机：玩家殖民地若与 **Remnant Nexus** 同系，唯一永久解法是摧毁 Nexus；否则多支 Ordo 会来 **饱和轰炸**——这是六阶“地区级灾难”的标准剧本。  
Remnants 对几乎所有人类派系敌对；与 TT 关系暧昧（造物主残留）；对 **Omega** 呈现“朝圣 / 搜寻 / 准神性”态度——可出现 Tesseract 率领 Remnant 舰队的终局遭遇。

⑥ **Loke 入侵、Andrada 成名与 Askonia 撕裂（约 c.+160s–181）**  
军阀 **Onesimos Loke** 入侵；军官 **Phillip Andrada** 在 Maxios 等战役成名。后勤官 **Jorien Kanta** 叛变后自立为盗。约 c.+181 Askonia 内战与 Hegemony 干涉中 Opis 被 Planetkiller 抹除，Andrada 建 Diktat，Hegemony 舰队与威望两伤——直接给第二次 AI 战争递刀。六阶正文写全面战争时，应让人记住：**Planetkiller 的阴影仍在政治语言里**——它是“可以抹星”的历史证据，不是玩家开局玩具。

⑦ **第二次 AI 战争（约 c.+189–194）与当代僵持**  
**Artemisia Sun** 借巡检争端与 Hegemony 虚弱开战；TT 用通讯后门与破交早期得手。约 c.+193“新卫”倒孙，冒险强攻 Chicomoztoc 失败（含 TTS *Invincible* Paragon 等惨烈掩护、Planetkiller 阴影）。**Baikal Daud**——出身 Chicomoztoc 蜂巢贫民、曾击沉 TT Astral 的巡逻指挥官——推动紧急征用民船守城，成为英雄，后升 **High Hegemon（最高霸主）**。**Treaty of Crom Cruach / Crom Protocol** 条件性结束战争：Hegemony“赢了”，双方皆残。Sun 后重掌更大个人权力。至玩家开局约 **c.206**，核心世界是残酷多极平衡：谁也吞不掉谁；热战以月计，冷战以代计。  
**六阶切入的“当代”**：僵持随时可碎。玩家多殖民地、高危清巢、持有 Ziggurat / Janus / 大量 Alpha 核心、公开站队或自立——任一项都足以把冷战拧成全面战争。

⑧ **Omega：伪派系、日冕守门人与不可谈判他者**  
Omega（舰名 / 前缀气质 **01011001**，二进制 “Y”）于版本约 0.95 起作为终局挑战出现。非常规外交派系：无委托、无市场、无人类领袖。舰型：**Tesseract**（巡洋级核心威胁）、**Facet**（驱逐）、**Dextral / Sinistral Shard**（护卫）、**Aspect** 舰载机群。战斗特征：顶级通量 / 机动 / 防护；被击毁后 **碎裂再生** 为更小活体单位（Tesseract→Facet+Shards+Aspects 等），并 **自适应负载** 针对你的护盾 / 装甲 / 点防 / 过载窗口。武器谱系含 Rift 系列、Reality Disruptor、Cryoblaster、Volatile Particle Driver 等近“黑箱”科技，部分效应与其他异常存在（如 Shrouded 相关讨论）有机制互文。  
**遭遇锚点**：每座 **Coronal Hypershunt** 常驻两艘 Tesseract；另有与大型 Remnant 舰队绑定的一次性军事联系人赏金遭遇。击败是 Omega 武器的主要来源之一。  
**与 Remnants 关系**：不明但紧密——Remnants 搜寻 / 崇敬 Omega；存在 Tesseract 带队 Remnant 的案例；社区与设定暗示可能是 Alpha 集体追求的更高形态 / 奇点（对话碎片如 “join as omega”）。**Omega Core** 存在于数据但不可常规获取，军官级能力远超普通核心。  
六阶正文铁律：**Omega 是事件，不是刷怪表**。接触即“贴近顶点”的叙事事件——可风筝、可拆解、可缴获残骸武器，不可人格化谈判，不可写成会被演讲感化的 AI。

⑨ **可选文明级钩子：Galatia Academy 与“通向诸门”**  
学院前院长 **Tomas Kallichore** 脉冲实验触发 **Galatia Incident**（跃迁点失稳、Derinkuyu 矿站沦为海盗等）。后任院长 **Anahita Baird** 冷酷务实，以津贴与勒索推动 **重开 Gate 网络** 研究。任务链骨架（版本约 0.95–0.98）：学院接洽 → 可重复委托（**Alviss Sebestyen**）→ **Kallichore Archive**（勒索 TT 高管 **Rayan Arroyo**、黑客 **Gargoyle**、**Elissa Zal** 暗中提供设备黑 Aztlan 中继）→ **Finding Coureuse**（找回学者 **Scylla Coureuse**）→ **Project Ziggurat**（追查 TT 秘项；经 **Callisto Ibrahim** 与残骸 *ISS Hamatsu* 至 **Alpha Site**，遭遇可回收的独特相位主力 **Ziggurat**，无人类船员，Mote Attractor 等异质系统；附近可有 Omega 武器缓存）→ **At The Gates**（经 Kazeron 贵族 **Horus Yaribay** 取得密钥见 **Baikal Daud** 获准实验；扫描多座 Gate；**Janus Device** 原型由 Coureuse 理论 + Zal 工程；失败测试可毁掉选定无人系跃迁点约一周期；为完成装置须从 **Kanta** 处救出 Zal——可强攻或交易，常牵涉 Livewell Cotton 绑走的 Loke 克隆体；成功后 Janus 整合入舰队实现门网旅行，Zal 与 Coureuse 在公开前私奔，Baird 从狂喜坠入受控绝望）。  
六阶意义：此线是沙盒里最接近“文明主线”的链——走完获得门网机动与扇区情报网威胁评级跃迁；不走完，全面战争与 Remnants 清巢仍自洽。**Ziggurat 污名**：持有即被全图识别（即便关应答机）——你变成行走的战略核武器与外交事故。

⑩ **殖民、危机矩阵与全面战争机制（六阶日常即史诗）**  
玩家殖民即宣告进入政治桌：  
- Hegemony：AI 核心 / 非法工业 → 巡检、最后通牒、法统讨伐。  
- League：封锁、入盟施压、Kazeron 利益交换。  
- Diktat：燃料竞争 → 市场战乃至饱和轰炸。  
- Church / Path：路德多数人口夺权或反科技洁净屠星。  
- Pirates：劫掠与保护费。  
- **同系 Remnant Nexus：必清，否则多支 Ordo 饱和轰炸。**  
每一种危机都有谈、打、站队、灭源四种解法；解法留下永久声望疤。六阶“全面战争”不是单一 BOSS 战，而是 **多战线并行**：你在边缘拆 Nexus 的同时，核心世界有人在烧你的粮星；你在 Hypershunt 外风筝 Tesseract 的同时，Diktat 把燃料价抬到让你的舰队动不了。

⑪ **探索终局层：遗迹、威胁与他者**  
外围有 Domain derelict、Explorarium 自复制遗产、Cryosleeper、Guardian 级守军。深渊超空间侧有被压制记录的 **Threat（威胁）** 类自复制战争机械传说（Megadeath 事件、蓝图 DRM 与 AI 禁令的可能远因）。Omega 与 Hypershunt 是六阶天花板遭遇。正文应保持：**门网为何全灭仍是最大未解之谜**；Remnants 的存在被官方史淡化；玩家可以成为扇区史作者，却买不断 Domain 全盛。

⑫ **微观政治贯穿（六阶仍要有账本）**  
码头讨价、关税与罚款、船员欠薪哗变、走私被扫、军官人格冲突、黑市中间人、悬赏合同、护航背叛、轨道炮威胁、燃料短缺困在风暴里、殖民地管理员贪腐、派系宣传战、AI 巡检登舰、Alpha 核心藏匿点、Kanta 贡礼谈判、军官在过载窗口抗命或死战。AI 写六阶时优先“**战役 + 账本**”双线，而非只写主炮齐射。

⑬ **蝴蝶效应边界**  
可改：局部 / 全面战争胜负、某殖民地归属、某军阀存亡、学院线是否走完、是否公开使用 Ziggurat / Janus / 大量 AI 核心、Kanta 保护费交易、派系好感版图、是否摧毁某系 Remnant Nexus、是否尝试 Hypershunt。  
不可无代价改：Domain 全盛舰队路边摊化、Remnants / Omega 降智成杂兵、把硬科幻改成魔法宇宙、抹去燃料 / 补给 / 通量规则、让 Omega 坐下来签条约。改了局部，因果以封锁、赏金、巡检、饱和轰炸、多线宣战回流。

**【叙事基调 · 雷区】**
冷硬、账本与炮火并重；后启示录但不末日摆烂；六阶是“地区级灾难与扇区史作者”气质，不是超级英雄单人肉身锤战列。  
忌太空魔法万能；忌无视燃料补给通量；忌 Remnants 杂兵化；忌 Omega 人格化劝降；忌 Domain 全盛随便刷；忌把所有派系写成脸谱善恶——Hegemony 的秩序有血，TT 的进步有罪，Church 的救赎有火，Kanta 的匪帮有规矩。  
忌契约者用“乐园外挂”口吻说话——本地人只认信用点、军阶与炮口；超常能力应被解读为 Domain lostech、TT 实验或 Path 宣传素材。  
最早可切入（本条仅六阶）：已具备巡洋~主力混编与至少一处中型以上殖民地 / 等价战争承包资质，锚定核心世界战争一触即发、外缘三脉冲信标与 Hypershunt 信号并存的 c.206+ 当代。

**【六阶专篇 · 舰队战与全面战争写法】**
描写通量过载窗口、护盾朝向、导弹饱和、舰载机群、撤退与打捞、多舰队协同与行星轨道目标。六阶战役应有：**旗舰指挥 + 侧翼诱敌 + 后勤油线 + 殖民地防空** 四层同时响。契约者“个人武力”转化为指挥、旗舰操作、故事点破例与工业决策，而非在舱外用拳头锤舰。Remnants 无士气崩溃——打赢靠拆站点、拆核心军官、控通量循环，不靠恐吓。Omega 靠风筝、拆解再生链、不在风暴与过载窗硬刚。

**【六阶专篇 · 经济、声望与燃料政治】**
关税 30% 的公开市场、黑市无税但损声望；声望是第二货币：跌破阈值则军用子市场关门、佣兵合同枯竭。Diktat 卡喉全扇区；超空间风暴里没油等于死刑。殖民地燃料产业会招来 Andrada 体系的饱和报复。AI 巡检：Hegemony 登舰搜核心；藏 Alpha 是暴利也是宣战。六阶任务可围绕「保护新殖民星度过饥荒 + 海盗潮 + Path + 同系 Nexus」叠加设计。

**【六阶专篇 · 派系作战教条速查】**
Hegemony：集火决战、装甲墙、实弹雨、军官质量、巡检与法统。League：中线角色分工、导弹与航母辅助、机动局部优势、封锁。TT：能量、强盾、相位伏击、精兵、合同战。Church：低科信仰军、民心与骑士。Path：覆写安全协议的疯狗突击。Diktat：中线 + 特改、燃料后方。Pirates：狼群、技术改装残舰、数量骚扰。Remnants：无士气机海 + 精英核心军官 + 站点生产线。Omega：拆解增殖与负载适应，遭遇即顶点事件。

**【对契约者的世界态度】**
你不是天选，是又一个带枪的变量。六阶时派系先评估你的吨位、殖民地数量、是否持有 dirty 科技，再决定合同、招安或饱和轰炸。土著不知“契约者”概念。低阶在码头被敲诈是正确世界反应；六阶被两派同时招安、同时列入清除列表也是正确反应。

## 阶位切入点

> 阶位↔舰船 / 舰队尺度：一阶≈小型民用船与码头生计；二阶≈护卫舰、走私与近域探索；三阶≈驱逐舰至巡洋舰会战；四阶≈主力舰队与单殖民地危机；五阶≈单殖民霸权与派系级对抗；**六阶≈Remnants 高危巢穴、Omega 遭遇、派系全面战争与多殖民地扇区霸权**。无魔法超人。世界顶点（高端 Remnants 节点 / Omega / 全面战争级军势）= **存在·情报优先 / 条件性胜利**，严禁将其简化为“战力归零”。

**一阶（核心世界码头 · 小艇与第一桶信用点）**

切入身份/时点：契约者以Jangala、Sindria或其他核心世界市场的临时货运员、刚签下短期合同的补给船水手，或独立小艇的副官身份进入，锚定c.206年一支只够维持燃料和工资的小型舰队刚脱离教程合约的阶段。可用资产限于穿梭机、补给舰或破旧护卫艇，目标是活着离开港口而非挑战巡洋舰。

初始事件：市场管理员突然宣布粮食短缺，公开市场价格暴涨；一名独立商人愿以低价把医疗物资交给你转运，却要求关闭应答机避开Hegemony巡逻。与此同时，船员名单上有两人因欠薪要离舰，港外还有海盗小艇在扫描刚出港的货船。你得决定走缴税但安全的航线、冒险走黑市合同，还是先把货卖掉支付工资。

开场白建议：「Jangala的起降灯在舰桥玻璃上拉成长线，货舱里只有几箱药品和半罐燃料。港口管制员报出三成关税时，副官把船员的欠薪单压在导航图上；外侧航道则亮起一枚陌生的传感器回波。这个扇区不问你有没有理想，它先问你能不能让反应堆烧到下一个补给点。」

关键NPC立场：**Hegemony港口巡检官**要求应答机、税单与货物来源对得上，违规者会被扣船；**独立商人**愿意为紧缺货提供合同，却把风险转嫁给承运者；**海盗掠夺者**只计算目标吨位和护航强度，弱船最先被试探；**Luddic Church牧师**可提供低价补给与人情，但反感高科违禁品；**Tri-Tachyon市场代表**出售昂贵零件，愿意以情报换取客户忠诚；**船员主管**优先工资、口粮与返航承诺，不能用空头战利品安抚。

主线钩子/支线：主线是完成一次合法或非法的短程货运，保住船员与燃料储备。支线可在酒吧接取海盗悬赏、替港口找回失联穿梭机、从残骸中打捞补给，或通过关闭应答机躲开一支不想见的巡逻队。选择黑市会提升利润与怀疑度，选择公开市场则保住关系但削薄现金流。

危险度/规避：低至中。规避与巡洋舰、军用巡逻主力和深空风暴纠缠；一阶舰队一旦通量过载只能撤退，不能把护盾当无限资源。Remnant、Omega和殖民地危机均是传闻级远端风险，优先保存导航数据与燃料。

任务方向/奖励：奖励取向为信用点、燃料、补给、低级舰载武器、核心世界联系人及一艘可修复的小艇；不得获得AI核心、主力舰、殖民地地契或高危探索坐标。

**二阶（外缘航线 · 护卫舰与废墟打捞）**

切入身份/时点：契约者以拥有一至两艘护卫舰的独立舰长、派系商队护航员，或外缘测绘小队的武装领航身份进入，锚定开始离开核心世界、接受赏金和勘测合同、却尚无巡洋舰火力与殖民地后方的阶段。行动半径受燃料、补给和船员经验直接限制。

初始事件：一座Domain时代探测站在偏远星系发出微弱求救信号。抵达后，打捞班发现站内有可用蓝图和一批机械零件，但传感器同时标出一支改装海盗护卫舰队正沿超空间航迹靠近；你的补给只够在“拖走重货”“彻底勘测”“立即返航”三者中完成一个。合同发布人还隐瞒了该星系有不稳定跳跃点的事实。

开场白建议：「探测站的气闸半开着，几百年前的Domain标志被冰霜盖住。打捞钩刚扣住一箱机械零件，雷达就报出三枚护卫舰信号从超空间边缘拐来。导航官说返航还来得及，工程师却指着蓝图笑得像找到了新世界。你看了一眼补给表，知道每多拿一件东西，都是拿回程的氧气和炮弹去赌。」

关键NPC立场：**独立打捞队长**愿按贡献分账，但会在舰队受损时要求先保自己的船；**Hegemony赏金官**只认可登记目标和战果记录，不替走私事故善后；**Persean League商队长**需要护航与自治航线，愿以货物抵佣；**Luddic Path伏击者**把高科货舱视为可焚毁的罪证，无法靠普通交易安抚；**Tri-Tachyon勘测代理**想要蓝图和遗物，常隐去风险信息；**海盗舰长**会在你露出补给短缺后试探性追击，击败他不等于消灭整个海盗网络。

主线钩子/支线：主线是从探测站带走一项有价值的打捞物，并在护卫舰对护卫舰的交战中维持可撤退的通量与阵型。支线可救出失联勘测员、将蓝图交给派系换声望、伪装货舱躲过海盗扫描，或放弃重货以修复受损引擎。不要把单次打捞写成无成本获得完整Domain工业链。

危险度/规避：中。规避在补给见底时继续勘测、让护卫舰对抗巡洋舰火力、携带违禁AI核心进入Hegemony港口，以及在超空间风暴中强行追击。战斗要体现护盾、装甲、通量和撤退窗口，而非角色在舱外解决敌舰。

任务方向/奖励：可获得勘测数据、舰船零件、基础蓝图、赏金、护卫舰维护权限和外缘联系人；奖励上限为改善舰队续航与装备，不提供主力舰、殖民地行政权或Remnant Nexus坐标。

**四阶（殖民星危机 · 主力舰队与派系施压）**

切入身份/时点：契约者以拥有巡洋舰或主力舰核心、正建立第一处殖民地的独立舰队司令，或受Hegemony、Persean League、Tri-Tachyon委派的战区承包人身份进入，锚定一颗新殖民星开始产出、同时被海盗、Luddic Path和相邻派系注意到的阶段。舰队能处理局部会战，却还无法支配多个星系或直面高危Remnant巢穴。

初始事件：殖民星的燃料厂刚开工，海盗袭击警报、Luddic Path渗透报告和Hegemony检查舰的到访通知在同一日抵达。检查舰要求登舰核验AI核心，海盗则准备趁港口防御尚未建成劫走燃料；殖民地总督提出向Persean League买防务合同，却会立刻恶化与Hegemony的关系。你必须分配主力舰队、行星防御预算和外交口径，任一环处理失误都可能把发展期变成封锁期。

开场白建议：「你的殖民星终于在黑暗边缘亮起第一串工业灯，燃料厂的火焰刚升过地平线，轨道上就出现了三种不同颜色的识别码。海盗要货，Path要火，Hegemony要看你的货舱。总督问舰队该去护港还是去迎检；你看着尚未建成的防御平台，明白一座殖民地从来不是资产表上的数字，而是所有人都想伸手掐住的喉咙。」

关键NPC立场：**Hegemony检查舰指挥官**要求AI巡检与法统服从，隐瞒核心会升级为政治冲突；**Persean League特使**愿以自治和防务合同拉拢新殖民地，代价是站队；**Tri-Tachyon联络官**看中你的工业产能和技术采购权，会把安全与商业绑在一起；**Luddic Path特工**以高科设施为破坏目标，常利用民用航道渗透；**海盗军阀**会以燃料和补给线为目标，除掉一支舰队仍可能留下报复；**殖民地总督**优先人口、粮食与稳定，不会接受只为舰队战绩牺牲平民的方案。

主线钩子/支线：主线是让殖民地撑过首次多重危机，可选择护航燃料船、截击海盗、接受或规避巡检、与派系谈判防务条款。支线可建立轨道站防御、清理Path藏点、以市场交易补足粮食，或将一件敏感技术交给可信联系人保管。选择公开拥有AI核心会换来短期技术优势和长期派系压力。

危险度/规避：高。规避同时向多个核心派系宣战、在无维修与燃料储备时打持久战、把Path威胁当成普通海盗，以及让巡洋舰单独冲进战列舰火力网。Remnant Nexus、Omega与全面战争属于更高阶阴影，应靠侦察、外交和撤离预案规避。

任务方向/奖励：奖励取向为殖民地稳定度、轨道站与防御建设资格、巡洋舰级改装、派系合同、可靠军官与贸易航线；不提供多殖民地霸权、Omega武器、Janus门网或可压制全扇区的舰队。

**六阶（高危残党 · 奇点守门 · 扇区全面战争）**

切入身份/时点：契约者以「拥有巡洋~战列核心混编、至少一处中型以上殖民地或等价战争承包资质的独立舰队司令 / 某派系特遣提督 / 新殖民联邦总督兼旗舰指挥官」切入，锚定约 c.206+ 当代——核心世界多极僵持一触即碎、外缘三脉冲警告信标频闪、Hypershunt 与学院门网密信并存的时期。时点具体可落在：某次殖民地危机叠加周、某条 Remnant 高危系被首次标红、或 Baird 发出 Gate 实验相关密信前后。

初始事件：你的旗舰刚完成超空间补给，全息板上同时炸开四条红讯——① 同系传感器确认 **Remnant Nexus** 完整运转，三支 Ordo 级编队已转向你的殖民星轨道；② 两个人类派系（例如 Hegemony 巡检使与 League 封锁舰队）在同一小时内向你发来最后通牒 / 入盟强制条款；③ 军事联系人丢来一份带 **Tesseract** 特征的赏金坐标，旁边叠着 Hypershunt 的日冕噪声；④ 货舱保险柜里一枚 **Alpha 核心** 的热签名刚好被 Hegemony 远程扫描仪扫到边角。三分钟内必须决定：先拆巢、先谈、先藏核心，还是先把燃料转到能跑的船上。

开场白建议：「超空间的蓝壁在观舰窗上碎成噪声。殖民星的防空炮在另一频道里咳嗽，警告信标的第三下脉冲像有人用金属指节敲你的颅骨。副官把四份红讯叠成一张牌——Nexus、封锁、Tesseract、巡检。在这片被大崩溃咬断的扇区里，没有中立的燃料，也没有免费的和平；你只有通量读数，和还愿不愿意按下去的主炮保险。」

关键NPC立场：**Baikal Daud** 体系下的战区 / 巡检代理人——视你为可征召炮灰或潜在 AI 叛军，法统与扫描仪是他的武器；**Artemisia Sun** 的合同官 / 高管链（可下探至 **Rayan Arroyo** 级）——只要核心、数据与你愿不愿意把 Ziggurat 阴影变成交易；**Reynard Hannan** 的谈判代表——开价自治同盟或封锁绞索；**Jorien Kanta**——贡礼可换 Piracy Respite，人质与克隆体是另一套价码；**Anahita Baird**——用津贴与勒索推你去摸门网，胜利后也可能弃子；**Livewell Cotton** / Path 小队——你的高科殖民地在他们眼里是该烧的僭越；**Remnant 站点逻辑（非人）**——把你算进清除列表，无劝降；**Omega 遭遇（非人）**——不可谈判，情报优先。

主线钩子/支线：  
- 主线节点（蝴蝶效应）：摧毁或永久压制同系 **Remnant Nexus**；在双线人类全面战争中选择站队、中立渔利或自立第八极；推进或拒绝 **At The Gates / Janus**；决定是否公开 **Ziggurat** 与 AI 真相；对 **Coronal Hypershunt** 选择回避、试探或代价性夺取。  
- 支线（本阶独有，勿与低阶复制）：向 Kanta 交付完好纳米炉 / 旗舰战利品换保护费；在 Askonia 燃料航线上反制 Diktat 市场战；护送或截获学院线相关人员（Zal / Coureuse 相关）；黑市出清一批 Omega 残骸武器并引发 TT / Hegemony 双线追杀；用故事点强改一次 AI 巡检结果并留下永久声望疤。

危险度/规避：贴近顶点。**六阶巅峰：存在·情报优先 / 条件性胜利**——风筝、电磁与通量循环、拆旗舰 / 拆站点 / 拆再生链，不在风暴与过载窗口硬刚 Omega；不在多线宣战同时硬闯三脉冲高危；禁止“一炮打服 Domain 幽灵”；禁止把 Remnants 写成会讲和的人类军阀。必须规避：无后勤状态下的深缘追击、Planetkiller 级禁武当玩具、同时向两大核心派系公开宣战且不留油线。

任务方向/奖励：自然目标 = 保住殖民星 / 清巢 / 打赢或调停一场全面战争侧翼 / 夺取 Hypershunt 外围科技 / 完成或放弃门网钥匙。奖励取向贴合六阶：Alpha 核心与 Remnant 船体、Omega 残骸武器、战列 / 战巡与 S-mod 关键改装、扇区话语权与历史书上的舰长之名、门网机动（若完成学院线）、私人殖民联邦或某派系实际共治权——**不许越级发放 Domain 全盛军团或可谈判的 Omega 仆从**。
