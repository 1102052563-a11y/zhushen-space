# 噬神者3
<!--meta lib=主库 tiers=一、二、三、四、五、六、八-->

## 剧情

**【作品来源】**
《噬神者3》（God Eater 3 / ゴッドイーター3）为万代南梦宫娱乐发行的动作角色扮演游戏，系列第三作主线。开发由 Marvelous First Studio 与 Bandai Namco Studios 合作，音乐椎名豪，角色设计小林くるみ，开场动画 ufotable。日本 PS4 于 2018-12-13 发售，全球 PS4/Steam 2019-02-08，Switch 2019-07。承接《噬神者2 愤怒爆裂》之后约十余年时间线，与手机作《Resonant Ops》有年代衔接但不强制前置。文风：灰烬末世、被当猎犬的自适应噬神者、港口政治与羁绊突围。

**【世界定位】**
2080 年代起「灰域（Ashlands）」扩张，芬里尔（Fenrir）总部与多数支部化为灰烬，人类残存于地下「港口（Port / 港）」与可在灰中航行的「灰行者（Ash Crawler）」商队。玩家扮演港口 Pennywort 的自适应噬神者（AGE），与雨果小队逃离奴役契约，护送神秘少女菲姆，卷入格雷普尼尔与绯红女王的路线之争，最终以共鸣改写灰域腐蚀性。一句话：灰海里的猎犬觉醒，用神机与羁绊咬开锁链。

**【世界观 · 力量体系】**
力量本源：神谕细胞（Oracle Cells）构成的荒神（Aragami）吞噬一切有机物与无机物；人类以偏压因子（Bias Factor）改造身体，驱动「神机（God Arc）」——唯一能稳定吞噬/破坏荒神的兵器。死亡规则：凡人与普通噬神者身死即止；被灰域侵蚀或被灰诞种（Ashborn）「吞噬污染」的 AGE 会大幅易伤且倒地难救；神机失控可反噬持有者。AGE 比旧型噬神者更耐灰、更近荒神，双臂戴手环（armlet），被港口当耗材使役。

战斗机制（写正文时当可观察规则）：
1. **吞噬与爆发（Burst）**：近战吞噬荒神部位积攒爆发，解锁 Burst Arts / 插件增益。
2. **加速扳机（Acceleration Trigger）**：达成条件（格挡/吞噬次数等）叠临时强化，Engage 时可共享。
3. **Engage 共鸣**：与队友近距作战积攒共鸣槽，满后共享增益；主角具备特殊共鸣能力，终局可放大至灰域尺度。
4. **神机形态**：近战（短剑/长刃/锤斧/刺枪等）+ 枪械切换；本作新增咬刃（Biting Edge 双刀/长戟）、重月（Heavy Moon 弯刀/巨斧）、射线枪（Raygun 持续光束）。
5. **灰诞种 Burst**：Ashborn 暴怒后发动吞噬，进入强化态并可再生，可污染 AGE。

战力阶梯（按破坏力表现，宁低勿高）：
1. **港口平民 / 后勤 / 无神机者**：冷兵器与枪械护卫，对荒神几乎无效。乐园观感≈一阶。
2. **新锐 AGE / 小型荒神（如奥加尾等）**：小队清剿、裂甲碎车。乐园观感≈二阶。
3. **正式 AGE 小队 / 中型荒神**：据点级交火、灰中护送战。乐园观感≈三阶。
4. **精锐 AGE / 大型荒神 / 初见灰诞种**：建筑群级破坏、港口级危机。乐园观感≈四阶。
5. **王牌小队 + Engage / 灰诞种爆发态**：一击可毁街区、威胁整座港口。乐园观感≈五阶。
6. **区域战役级（回收芬里尔旧址、山地禁路、多港口联军）**：覆盖大片灰域航线与要塞。乐园观感≈六阶。
7. **（本世界无七阶覆盖）**
8. **终局尺度：奥丁兵器、灰岚/灰岚风暴、菲姆全域共鸣改写灰质腐蚀**：行星表面灾难级现象与对冲。乐园观感≈八阶。

乐园阶位映射：港口凡人≈一阶；新锐 AGE/小型荒神≈二阶；正式 AGE 小队/中型荒神≈三阶；精锐/大型荒神/初见灰诞≈四阶；王牌 Engage/灰诞爆发≈五阶；区域战役与旧总部回收≈六阶；奥丁·灰岚·全域共鸣≈八阶。顶点=八阶现象级（非无限复活）；低阶切入一律规避顶点·情报优先；本世界峰值停在八阶现象，写条件性胜利（切断供能/共鸣中和/撤离窗口），严禁「奥丁被临时关机所以变弱」。

**【地理 · 舞台】**
舞台锚定旧芬里尔总部周边欧洲灰域：地表化为持续扩张的灰质荒原，基础设施尽毁，唯一动脉是灰行者商队。地下/半地下「港口」是人类据点——Pennywort（利润至上、AGE 损耗极高的猎杀港）、Chrysanthemum（希尔达经营的中型运输港，母舰灰行者武装薄弱但航速高）、Baran（工商业与 AGE 研发强港）、Gleipnir 控制的枢纽港群、Dusty Miller（收容/中立色彩，EIN 坐镇）等。灰中任务场包括废城、平原、沉没电网、神殿遗迹、废弃航母、地铁、伤痕壁垒等可切换猎场。政治地理：格雷普尼尔以《灰域航行法》仲裁港口冲突；绯红女王走「适应灰域」路线与之对立。AI 写场景先定：港口内政 / 灰中护送 / 猎杀任务 / 灰风暴来袭 / 总部废墟战役，再定冲突强度。

空间层级可记为：灰面（致死腐蚀+荒神）→ 港口闸门与气密层 → 居住环廊/食堂/神机库 → 指挥室与契约档案馆 → 灰行者泊位。垂直高差被本作强调：废楼、沉船甲板、峡谷航路都能成为 Dive 冲锋与重月抡斧的立体战场。旧芬里尔总部废墟是六阶以上战役锚点，内含奥丁相关设施与象征「旧秩序尸骸」的纪念空间。禁路山地是逃亡线高潮地形，狭窄、落石、噬 AGE 型威胁并存。Dusty Miller 相对「外缘庇护港」，信息流通靠 EIN 的旧人脉而非格雷普尼尔官方网。

**【世界剧情线】**
① **Pennywort 的锁链（开局）**  
自适应噬神者主角与机械师基思、近战手齐克、队长雨果同属港口 Pennywort 契约 AGE。港口把他们当「猎犬」：高危远征、报酬微薄、物资短缺时先砍 AGE 配额。小队日常是清小型/中型荒神、回收零件、修神机、听管理层训话。世界状态：灰域已吞芬里尔体系十余年，港口靠 AGE 与灰行者续命。因果：雨果盘算攒够「买自由」的资金，成立自己的港口。

② **灰风暴与菊港救援**  
一场灰风暴吞没 Pennywort。管理层弃 AGE 先逃，小队濒死时被 Chrysanthemum 港主希尔达·恩里克斯的灰行者救起。船上还有格雷普尼尔护卫型 AGE 克莱尔·维克托利尔斯，以及须按航行法「日后送还」的法律阴影。希尔达允诺载他们赶路，条件是沿途清航道、分成运费。雨果提出：用分成与战绩买断契约。名场面：灰墙压境、港口警报、被扔下的 AGE 在灰尘里听见救援引擎。

③ **灰中航路与露露加入**  
途中执行清路、护货、猎杀。从 Baran 港流落的侦察 AGE 露露·巴兰被收容，交出加速扳机原型，基思完成实用化。小队战力质变：Accel Trigger + Engage 让他们能咬大型目标。因果：技术差开始拉开「被当耗材的 AGE」与「能谈条件的战力」之间的缝。

④ **货舱裂开——菲姆**  
灰诞种袭击撕开货舱，所谓格雷普尼尔贵重货竟是神秘少女。众人命名菲姆：能治愈创伤、净化灰诞污染。荒神攻击频率异常升高，怀疑被菲姆吸引。借助菲姆与主角共鸣，小队史无前例击杀一头灰诞种。抵达菊港后与希尔达改签合同，雨果小队从「乘客」变成「可谈判佣兵」。

⑤ **格雷普尼尔总部与犬饲阴谋**  
抵达 Gleipnir 总部，犬饲博士收走菲姆。灰诞再袭，小队夺回菲姆并击破怪物，说服总督亚伯拉罕·加多林：菲姆是人形荒神级样本，却只有他们能杀被她吸引的灰诞，故应留在小队。交换条件：带头执行「夺回芬里尔旧总部 + 启动超兵器奥丁」作战。调查内鬼时，希尔达一行揭穿犬饲要把菲姆卖给绯红女王；绯红女王领袖是加多林之子维尔纳，掌握以 AGE 性命为代价触发灰风暴的技术。犬饲被捕，回收作战启动。

⑥ **巴兰胁迫与总部回收**  
Baran 护航队长胁迫露露，企图借绯红女王之手夺菲姆；事败被捕，Baran 为保面子退出联合。维尔纳当面招揽小队加入绯红女王，却放行他们完成总部回收。格雷普尼尔拿回旧址与奥丁，小队返回菊港休整。阶段结局：表面上人类夺回「灭灰钥匙」，暗线是奥丁燃料是什么。

⑦ **奥丁的真相与 AGE 起义**  
约一月后，加多林宣布奥丁完工，却揭示启动需吞噬大量 AGE 作燃料；格雷普尼尔将强制收缴各港 AGE。全港 AGE 暴动，许多人投奔维尔纳；Baran 公开站绯红女王。希尔达拒交小队，全港乘灰行者突围。禁路山地遭遇「噬 AGE 荒神」，救出齐克失散之兄尼尔。抵达 Dusty Miller，港主 EIN（旧识希尔达）庇护并联系维尔纳。EIN 指出：菲姆独特偏压可替代全部 AGE 进入奥丁。菲姆得知后独自前往献身。

⑧ **背叛、灰岚与共鸣终局**  
Baran 背叛绯红女王，格雷普尼尔清剿残部。维尔纳一派绝境召唤末日级灰岚风暴，奥丁仓促出击。菲姆勉强挡住第一波未死，但赶工奥丁单元暴走，小队击破失控奥丁。重聚后，菲姆发动横贯灰域的大规模 Engage 共鸣，中和灰质腐蚀性，灰岚止息。冲突落幕，奥丁停机，小队归港。尾声：EIN 真身实为前作索玛·席克萨尔化名，展望未来后启程远东。结局方向：灰域腐蚀被改写而非简单「清空所有荒神」；AGE 人身权政治仍未一劳永逸，但雨果小队赢得自由与家园锚点。

⑨ **可介入日常与战后余波**  
港口食堂、神机工坊、契约文书、航行法听证会、黑市偏压剂、失踪 AGE 名册、灰行者航线图——正文高频应是这些。战后：谁接管奥丁残骸、绯红女王残党、Pennywort 旧管理层清算、菲姆的身份公开与否，皆可作支线。

补充因果链（供 AI 判时点）：  
契约奴役 → 弃民风暴 → 菊港收容 → 技术升级（Accel） → 菲姆出现吸引灰诞 → 政治利用（格雷普尼尔/绯红女王） → 奥丁「灭灰」承诺 → 燃料是 AGE → 起义与逃亡 → 菲姆替代牺牲 → 灰岚对冲 → 共鸣改写规则 → 自由港雏形。  
任一环节被契约者改写，都会回流到「谁掌握菲姆/奥丁/航行法」三张牌。

⑩ **猎场与任务节奏（可观察细节）**  
灰中任务不是抽象「刷怪」，而是：简报（目标荒神种别、材料优先级、灰浓度）→ 出港闸门气密检测 → 灰行者投放或徒步切入 → 锁定核心部位 → 吞噬积攒 Burst → 若遇灰诞则防污染与倒地规则生效 → 撤离窗口以分钟计。评级影响配额：完整度高换零件与权限，低则扣餐与维修优先权。AI 写战斗优先给可观察细节：手环指示灯、神机齿轨过热、灰附着在镜头上的沙沙声、Engage 时队友呼吸同步、菲姆净化时伤口结霜般的光。

⑪ **Pennywort 内部生态**  
港口把 AGE 编成可交易资产：契约年限、战损保险、材料分成比例写在薄薄一张单上。管理层会议室有灰浓度全息图，红色是「可牺牲航线」。食堂分两区：人类职员热食与 AGE 高热量糊状配给。工坊夜里灯不灭，基思一类技师用报废神机拆齿轨。雨果在账本背面算「买断自由还差多少」，齐克把怒气砸在训练桩上。这种日常比任何演讲更能解释后来为何全港 AGE 会起义。

⑫ **航行法与商队政治**  
《灰域航行法》规定：救援落难 AGE 须登记并在期限内送还原属港，否则视为挖角或人口贩运。希尔达救雨果小队时就踩在这条灰线上——她给船票，也给法律债务。商队母舰以母港名命名，武装常被减重换航速，遇灰诞只能靠随舰 AGE。港口之间用物资、情报与 AGE 租借互相勒索；格雷普尼尔以「调解人」身份抽成并扩权。契约者若改写送还条款，会立刻触发多港通缉或黑市竞价。

⑬ **菲姆伦理核**  
菲姆不是单纯吉祥物：她是人形荒神样本，能治伤、净污染，也像灯塔吸引灰诞。犬饲要的是数据与交易；加多林要的是武器燃料替代方案出现前的稳定；维尔纳要的是适应灰域的叙事道具；雨果小队要的是「家人」。正文应反复让同一能力被不同势力命名为「奇迹 / 样本 / 电池 / 女儿」。终局她自愿入奥丁，是伦理线的顶点而非突然降智。

⑭ **奥丁与双黑结构**  
奥丁被宣传为「收回地表的钥匙」。真相是启动需要吞噬大量 AGE 的生命/偏压，格雷普尼尔选择强制征用。绯红女王反对征用叙事，却掌握以 AGE 性命触发灰风暴的技术，绝境时放出灰岚。两边都把 AGE 当耗材，差别只在口号。条件性胜利写法：切断奥丁供能、保护共鸣窗口、疏散港口，而不是「主角一刀砍爆行星」。

⑮ **系列位置与互文**  
时间线上，GE3 位于红雨/螺旋树事件多年之后、灰域出现约七年后的 2087 年前后。旧世代角色以 EIN（索玛）压轴点题，不负责代打全程。Resonant Ops 可作背景广播与遗物，不强制展开。远东支部在尾声被 EIN 重新指向，暗示灰域问题未覆盖全球同一解法。

⑯ **武器与战斗读感（正文可用）**  
咬刃：双刀高速连斩，切长戟形态换范围但烧耐力且难防。重月：宽弧重击，斧形态越挥越狠、对重甲有效但机动差。射线枪：持续光束，长按越强，要抓硬直窗口。Accel Trigger 条件各异（格挡 N 次、吞噬 N 次等），成功则全队吃增益。Engage 近距叠槽，满则共享包括 Accel 在内的强化。灰诞进入 Burst 后招式刷新、回血、污染 AGE——被污染者倒地代价极高。这些规则应写成战场气压，而不是系统菜单朗读。

⑰ **名场面清单（锚定用）**  
Pennywort 弃民夜的灰墙与救援引擎声；货舱裂开看见菲姆的静场；首次击杀灰诞时共鸣光与污染反冲；总部走廊夺回菲姆；巴兰胁迫露露的通讯勒索；奥丁吃人真相广播；菊港全员登舰断后；山地救出尼尔；菲姆独自走向奥丁；灰岚抹灯；暴走奥丁击破；全域共鸣后灰质不再腐蚀；EIN 摘下化名走向远东。契约者介入任一场，都会改「谁活着听见下一句广播」。

**【主要人物】**

**【舞台调度备忘 · 仅本世界】**
写港口夜班：先给气密门气压差的耳鸣，再给手环充电座的蜂鸣，最后才给人物对白。写灰中行军：能见度用「十步外队友只剩剪影」而不是形容词堆砌。写谈判：合同页角的灰渍、印章颜色、送还期限倒计时比口号重要。写菲姆：她学人类用错敬语、把荒神叫「好疼的光」，比旁白解释身世更有力。写终局：灰岚不是背景板，是一口口吞掉航标灯的进行时；共鸣成功后第一反应应是「灰还在，但皮肤不再烧」，而不是烟花式大团圆旁白。

**【AGE 社会位置】**
自适应噬神者被官方语言称为「必要战力」，私下叫法是猎犬、电池、会走路的神机接口。他们比旧型噬神者更耐灰，也因此被派进旧世代不敢久留的航线。报酬结构故意让人无法攒够买断金：战损维修从个人分成里扣，伤病停工不发配给，立功则被加更危险的班。雨果的「建港自由」不是中二梦想，是唯一能改合同主语的路径。起义不是突然道德觉醒，是账算到死仍还不清的总爆发。

**【克莱尔与体制内裂痕】**
克莱尔代表格雷普尼尔「合法暴力」的年轻一代：相信航行法能减少港口互咬，相信护航是保护人类。与雨果小队同行后，她看见弃民、卖样本、奥丁吃人——条例无法覆盖的恶。她的倒戈不是瞬间黑化，而是一次次在耳麦里把「按规定」改成「先救人」。契约者若在五阶前羞辱她的信仰，她会更硬；若在五阶后出示奥丁燃料证据，她可能成为体制内最大杠杆。

**【露露与巴兰】**
露露被母港当弃子又当刀：Accel 原型在她身上，政治价值在巴兰账本上。胁迫事件证明强港可以把亲属/同僚当人质遥控外派 AGE。她加入雨果小队后的忠诚，是「第一次有人把她当同伴而不是资产」。巴兰后续公开投绯红女王再背叛，说明工商港的旗帜只是报价单。

**【EIN 的功能】**
EIN 不是来抢主角光环的外挂，而是：提供庇护港口、点破菲姆可替代 AGE 入奥丁的关键情报、用旧世代视角标定「这不是第一次有人把人当燃料」、终局后把目光拉回远东。他与希尔达的旧识让菊港逃亡线有落脚点。禁止写成「索玛一人砍爆灰岚」。

**【战后未闭合问题】**
灰质腐蚀被中和后，荒神是否整体衰退不详；港口是否立法承认 AGE 公民权不详；奥丁残骸由谁监管会引爆新冲突；绯红女王残党与格雷普尼尔强硬派都可能重启「可控灰风暴」研究；Pennywort 旧管理层的审判是私刑还是法庭；菲姆公开身份会否引来全球样本猎人。这些空白是支线富矿，禁止用「世界和平」一笔勾销。

**【物资与黑市】**
偏压剂、神机齿轨、防灰涂层、Accel 芯片、伪造航行许可、菲姆相关谣言录音带，构成灰域硬通货。黑市摊位常开在气密门维修夹层，老板同时给格雷普尼尔线人与绯红女王跑腿。假药会让 AGE 在灰中突然失爆发，便宜货有命价。契约者若靠倒卖暴富，会同时被多港通缉与荒神「材料任务」点名——因为战力上升意味着你被写进更危险的简报。

**【雨果小队作战分工】**
雨果控节奏与谈判窗口；齐克开路与嘲讽型拉仇恨；基思报过热、改插件、算撤离秒数；主角吃共鸣位与关键爆发；克莱尔补支援与法规掩护；露露侦察与 Accel 条件引导；菲姆在安全圈净污染、战时也可能被逼到前线。缺一人，Engage 链条就会在灰诞 Burst 时断开。正文写团战先点名谁在喊什么口令，再写刀光。

**【可对照的威胁表】**
小型荒神：清杂、教学、港口外围。中型：据点与护航常态。大型：单次战役 Boss 级，需 Engage。灰诞种：污染+Burst 再生，四阶起主威胁。区域联军与督战队：六阶人祸。奥丁暴走单元与灰岚：八阶现象，只写窗口与条件胜。AI 选敌先问「本场要证明什么规则」，再选种别，避免无脑堆怪。契约者改局部救人，常回流为更狠的征用令或灰风暴技术竞赛——本世界惩罚「无代价英雄」。自由要用合同与血同时付清。灰还在，人先站直。

## 阶位切入点

> 阶位↔战力：一阶≈港口凡人/后勤；二阶≈新锐 AGE/小型荒神；三阶≈正式 AGE 小队/中型荒神；四阶≈精锐/大型荒神/初见灰诞；五阶≈王牌 Engage/灰诞爆发；六阶≈区域战役/旧总部回收；八阶≈奥丁·灰岚·全域共鸣。世界顶点为八阶现象级，低阶切入一律「规避顶点·情报优先」。本世界无七阶覆盖。

**一阶（港口凡人 · 灰下生存线）**
切入身份/时点：以 Pennywort 或菊港后勤/见习船员/文书身份切入，锚定雨果小队仍受契约束缚、灰风暴尚未吞港的日常。
初始事件：仓库盘点时发现偏压剂与弹药被管理层挪去黑市；警报却报「外围小型荒神靠近通风井」——你必须在告发、沉默分赃、跟 AGE 小队跑腿之间抉择。
开场白建议：「地下港的灯管永远缺一支。广播用最平静的声音播报灰浓度，像在报天气。你抱着清单走过神机库时，几个戴双环的 AGE 正被骂着「猎犬」，而你手里的笔，比枪更先决定谁今晚能吃上热食。」
关键NPC立场：**雨果·佩尼沃特**要你别添乱但缺人手；**基思·佩尼沃特**要零件账对得上；**港口管理员**要封锁丑闻；**希尔达·恩里克斯**（若在菊港）只关心船期。
主线钩子/支线：通风井清残党（观察 AGE 作战）、伪造的航行许可、失踪装卸工线索；支线＝食堂配额、手环充电排队、黑市假偏压剂。
危险度/规避：低~中——危险来自疏漏与人祸；须规避擅自出灰面、触怒管理员、靠近未镇压的中型以上荒神。
任务方向/奖励：站稳后勤信誉、换取船票或见习资格；奖励取向＝防灰面具、基础药剂、情报便签、AGE 小队好感。

**二阶（新锐 AGE · 猎犬契约线）**
切入身份/时点：以 Pennywort 新晋 AGE 切入，锚定主角小队仍接高危低酬任务时期。
初始事件：第一次正式出击清奥加尾级目标，通讯里管理层催「材料完整度」，队友却缺氧；你要在硬冲抢材料与保命撤出之间选。
开场白建议：「双环扣上的时候没有仪式，只有登记号。灰一漫过护目镜，世界就只剩神机的震动和耳麦里的倒计时。他们说这是工作；你很快明白，工作的另一种写法是消耗品。」
关键NPC立场：**齐克·佩尼沃特**喊你跟上别掉队；**雨果·佩尼沃特**盯契约剩余年限；**基思·佩尼沃特**远程报武器过热；**克莱尔·维克托利尔斯**（后期遭遇）会按条例记录你的违规。
主线钩子/支线：材料回收评级、同伴受伤是否上报（上报扣配额）、小型护航；支线＝神机校准、手环故障、私下修理黑活。
危险度/规避：中——中型荒神与恶劣灰况可灭队；规避单独追击、违抗归港命令过深。
任务方向/奖励：存活率与评级；奖励取向＝初阶神机部件、Accel 入门、小队位次。

**三阶（正式小队 · 灰中护送线）**
切入身份/时点：以正式 AGE 或随舰护卫切入，锚定菊港灰行者清航道、雨果谈分成阶段。
初始事件：护航中段货舱警报，不是货动是荒神贴舷；克莱尔要求优先保「格雷普尼尔货物」，雨果要保船员——命令冲突炸在你耳麦里。
开场白建议：「灰行者的甲板在震动，像一头在灰海里用肚皮犁路的兽。你的任务简报写着『确保航路』，没写的是：航路两边，人和港口的脸面都比你的命值钱。」
关键NPC立场：**希尔达·恩里克斯**要船期；**克莱尔·维克托利尔斯**要合规；**雨果·佩尼沃特**要分成与自由金；**露露·巴兰**（刚加入前后）冷眼看你是否拖后腿。
主线钩子/支线：清航路猎杀、Accel Trigger 试装任务、航行法送还期限临近的谈判；支线＝船员恐慌安抚、雷达假信号、巴兰遗弃侦察兵搜救。
危险度/规避：中高——大型荒神与灰暴前兆；规避离舰过远、泄露货舱机密。
任务方向/奖励：护航成功分成、合同改签筹码；奖励取向＝中级神机技能、防灰涂层、港口临时户籍。

**四阶（精锐 · 灰诞初见/菲姆线）**
切入身份/时点：以精锐 AGE 切入，锚定货舱裂开、菲姆现身前后。
初始事件：灰诞种撕裂装甲，你看见「货物」是个女孩；污染值飙升，犬饲系通讯要求「优先回收样本」，小队要救人——你的第一枪/第一刀决定立场标签。
开场白建议：「警报不是红色，是一种更干的灰白。你听见神机齿轨咬合的声音盖过尖叫，又在某一秒突然静掉——因为所有人都看见了同一个不该出现在货舱里的孩子。」
关键NPC立场：**菲姆**本能靠近能共鸣者；**犬饲冬彦**（远程）要样本完整；**雨果·佩尼沃特**要守护；**亚伯拉罕·加多林**稍后会把你们当刀用。
主线钩子/支线：首次击杀灰诞协同、菲姆命名与保密、样本争夺；支线＝净化污染伤员、伪造货舱日志、Baran 间谍试探。
危险度/规避：高——灰诞爆发与污染；规避单独护送菲姆、接不明研究舱。
任务方向/奖励：灰诞素材、Accel 进阶、菲姆信赖；奖励取向＝抗污染插件、精锐权限。

**五阶（王牌 · 总部政治与灰诞连战）**
切入身份/时点：以可杀灰诞的王牌小队成员切入，锚定 Gleipnir 总部交涉与内鬼揭露期。
初始事件：菲姆被收走的当夜灰诞攻门，你得在「服从总督回收令」与「跟雨果抢人」之间当场站队，摄像头全开。
开场白建议：「总部的走廊亮得像手术室，谈话却比灰里更冷。他们称菲姆为样本，称你们为必要之恶。当第二声爆炸震落吊灯时，你知道谈判已经结束，狩猎重新开始。」
关键NPC立场：**亚伯拉罕·加多林**要成果；**克莱尔·维克托利尔斯**信仰动摇；**希尔达·恩里克斯**要交易条件；**维尔纳·加多林**在暗处递橄榄枝；**露露·巴兰**被母港阴影牵扯。
主线钩子/支线：夺回菲姆、揭犬饲、巴兰胁迫露露事件、启动总部回收前夜；支线＝AGE 暴动传单、航行法听证会捣乱、绯红女王接头。
危险度/规避：高——政治背刺+灰诞；规避无授权刺杀总督、对维尔纳无条件投诚。
任务方向/奖励：总部通行证、作战指挥权份额；奖励取向＝顶级插件、情报网、港口庇护函。

**六阶（区域战役 · 旧芬里尔回收/举港逃亡）**
切入身份/时点：以联合作战干部或菊港防卫核心切入，锚定夺回芬里尔旧址或奥丁真相曝光后的举港突围。
初始事件：加多林宣布奥丁需 AGE 作燃料的广播响彻港口，外面封锁线合拢；希尔达下令全员登舰，你负责断后——第一波来的不是荒神，是格雷普尼尔督战队。
开场白建议：「广播把『人类的未来』念了三遍，第三遍时神机库的门锁自动落下。你听见有人砸门，有人哭，有人笑着骂。灰行者的引擎点火，像一句终于说出口的『不』。」
关键NPC立场：**希尔达·恩里克斯**拒交人；**克莱尔·维克托利尔斯**可能倒戈协助；**齐克·佩尼沃特**要去找尼尔；**EIN**在 Dusty Miller 伸手；**戈·巴兰**等投机者改旗帜。
主线钩子/支线：旧总部夺控、山地禁路、救出尼尔·佩尼沃特、联系 EIN；支线＝安置普通居民、抢奥丁设计残页、拦截征用船。
危险度/规避：极高——联军火力与大型威胁；规避硬闯奥丁核心、单独承诺菲姆安全而不设援。
任务方向/奖励：区域航线控制、起义同盟；奖励取向＝战役级素材、灰行者权限、多港通缉或荣誉（取决于站队）。

**八阶（顶点段 · 奥丁/灰岚/共鸣）**
切入身份/时点：以终局同行者切入，锚定菲姆入奥丁、灰岚风暴与全域共鸣夜。
初始事件：天穹被灰岚撕开，奥丁强行升空；菲姆生命读数暴跌，失控单元开始无差别攻击——你的任务不是「打赢行星」，而是在窗口内击破暴走单元并保护共鸣释放。
开场白建议：「没有人再讨论契约年限。灰岚把远方的港口灯一盏盏抹掉，像有人在地图上用橡皮拼命擦。菲姆的声音在频道里断续出现，你听见她说『没关系』，而你第一次希望条例里有一条能禁止英雄牺牲。」
关键NPC立场：**菲姆**执意替代 AGE；**雨果·佩尼沃特**小队全员死守；**维尔纳·加多林**余党与灰岚绑定；**亚伯拉罕·加多林**要结果不论代价；**EIN**提供关键情报但不代你当救世主。
主线钩子/支线：阻止灰岚第一波、击破暴走奥丁、守护菲姆共鸣、战后奥丁残局与 AGE 权利谈判；支线＝搜救被征 AGE、绯红女王残党去向、EIN 东行送别。
危险度/规避：贴近顶点——灰岚/奥丁为八阶现象；**存在·情报优先/条件性胜利**，严禁「封印削弱归零」；低准备者应转六阶撤离线。
任务方向/奖励：终局发言权、自由港雏形、共鸣相关奇物（不越级发「灭世按钮」）；奖励取向＝传奇神机涂装/权限、政治免罪、菲姆同行或远东线索。
