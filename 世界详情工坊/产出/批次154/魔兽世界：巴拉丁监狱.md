# 魔兽世界：巴拉丁监狱
<!--meta lib=主库 tiers=一、二、三、四、五、六、七、八-->

## 剧情

**【作品来源】**
《魔兽世界：巴拉丁监狱》（Baradin Hold）是暴雪娱乐 MMORPG《魔兽世界：大地的裂变》（Cataclysm）中位于托尔巴拉德（Tol Barad）岛中心的 10/25 人团队副本，补丁 4.0.3a 加入，4.2 增补奥库萨（Occu'thar），4.3 增补阿丽萨巴尔（Alizabal）。叙事以游戏副本日志、Adventure Guide、首领说明文案、托尔巴拉德 / 托尔巴拉德半岛日常与战场任务、官方史诗与区域设定、术士职业大厅相关线（军团再临「An Eye for a Scepter」语境）、以及《Exploring Azeroth: The Eastern Kingdoms》中马迪亚斯·肖（Mathias Shaw）的战后报告为准。关联舞台含托尔巴拉德半岛补给港、三座控制要塞、监狱侧翼（诅咒深渊 / D 区 / 地洞）、巴拉丁堡外围炮火与地下团队翼。文风：海岛监狱恐怖、二战废墟幽灵、联盟与部落控制权争夺、恶魔收容失效的黑色幽默。本档案锚定「裂变期托尔巴拉德控制权 + 巴拉丁监狱三首领清剿」主舞台及其直接前史 / 即时后果，不把暮光堡垒、黑翼陷落、火焰之地整本倒灌为同一场。

**【世界定位】**
托尔巴拉德是东部王国外海、吉尔尼斯以西的荒岛要塞，由阿拉索（Arathor）初建、历经第二次大战夷平，再被库尔提拉斯法师改建为「把最危险的东西关在石头与符文里」的神秘监狱。大地裂变撕开封印完整性后，**巴拉丁典狱官（Baradin's Wardens）**与**地狱咆哮近卫军（Hellscream's Reach）**为控制三座外围要塞而战——胜者控制巴拉丁堡，并获得进入地下团队本的权限（制度上类似诺森德阿尔卡冯的宝库）。副本内关押不宜流放、不宜上链的存在：东翼深渊领主**阿尔加洛斯**、西翼空灵恐魔**奥库萨**、北翼希瓦拉**阿丽萨巴尔**。契约者切入的是：从港口日常与三据点拉锯，换取打开监狱门的政治钥匙，再分期斩首三翼恶魔，并在军团再临后面对「清理失败—失守—暗影议会夺眼—战后无典狱长」的长尾真空。一句话：**这是先抢控制权才能进门的恶魔最高安保监狱，不是排队进本的普通团本。**

**【世界观 · 力量体系】**
力量本源：艾泽拉斯常规职业体系（武技 / 奥术 / 神圣 / 暗影 / 邪能）、装备与团队战术、阵营声望与战场控制权、肯瑞托收容阵与库尔提拉斯式「向内防御」的符文工程、燃烧军团恶魔层级（歼灭者魔 / 空灵恐魔 / 希瓦拉）、暗影议会仪式学。死亡规则：凡人与多数精英战死后即死；团队内可战斗复活 / 灵魂石等职业手段，但不可把复活石当原作免费无限；监狱越狱与军团入侵属于世界级地区灾难（军团再临阶段托尔巴拉德一度失守）。特殊系统：① 10/25 人团队协同；② **控制三据点 → 控制巴拉丁堡 → 开启团队入口**的权限锁；③ 1 小时级战场节奏与通令（Tol Barad Commendation）经济；④ 控制方日常（中心任务板 + 轮换侧翼：诅咒深渊 / D 区 / 地洞）；⑤ 非控制方仍可在半岛补给区活动；⑥ 走廊生态随补丁替换（收容单元 → 奥库萨之眼 → 憎恶门徒）；⑦ 同阵营准入、与声望军需绑定。

战力阶梯（按破坏力表现，宁低勿高）：
1. **港口新兵 / 半岛杂兵 / 逃囚残党 / 幽灵游荡者**：冷兵器与火枪极限。≈一阶初～中。
2. **据点争夺中的精锐步兵 / 攻城器械操作手 / 日常侧翼精英**：可清小队、守旗。≈一～二阶。
3. **典狱官 / 近卫军军官、英雄五人本级清剿队、监狱入口守卫**：一域据点级。≈二～三阶。
4. **团队走廊小怪包（收容单元残渣 / 眼魔群 / 憎恶门徒）与监狱构造体**：团本前厅。≈三～四阶。
5. **阿尔加洛斯（T11 / 凶恶角斗士档深渊领主）**：可屠灭未分工的十 / 二十五人队。≈四～五阶。
6. **奥库萨（T12 / 残酷角斗士档空灵恐魔 + 眼球机制）**：西翼灾难体。≈五～六阶。
7. **阿丽萨巴尔（T13 / 灾变角斗士档希瓦拉·仇恨之主）**：北翼终局。≈六～七阶。
8. **军团再临期全面失控 + 奥库萨隆等继任恶魔 + 暗影议会夺眼行动**：地区级监狱崩盘。≈七～八阶。
9. **燃烧军团方面军本尊、古尔丹级仪式、萨格拉斯阴影**：超阶，不在本档可击杀表。

乐园阶位映射：外围日常≈一～二阶；三据点战场≈二～三阶；军官与入口≈三～四阶；团队走廊≈四阶；**阿尔加洛斯≈五阶**；**奥库萨≈六阶**；**阿丽萨巴尔≈七阶（本档裂变期可打顶点）**；军团期失守与夺眼≈八阶地区灾难。宁低勿高。本条目覆盖一至八阶；八阶写「失守—夺眼—战后无典狱长」的真空与条件性善后，非让契约者在巴拉丁单挑军团方面军本体。

**【地理 · 舞台】**
- **托尔巴拉德岛（大海上，吉尔尼斯以西）**：焦黑要塞、锈蚀港口、雾与炮火；契约者全阶共用岛景。
- **托尔巴拉德半岛（补给与日常 hub）**：不论控制权均可活动的外围舞台——军需、声望起步、资源点（草药 / 矿石）、海盗锈港（Rustberg）余波。
- **三座控制要塞**：
  - **铁甲驻地（Ironclad Garrison）**——北；
  - **熔渣车间（Slagworks）**——东南；
  - **典狱官岗哨（Warden's Vigil）**——西南。
  攻方须同时控制三处并维持（含攻城摧毁尖塔等延时机制），结算后控制巴拉丁堡。
- **巴拉丁堡 / 巴拉丁监狱本体**（约 [47.7, 53.0] 中心）：向内防御的塔楼与地下；团队入口在堡垒西墙外侧约 [46, 48] 附近（集合石旁）。
- **东翼**：阿尔加洛斯监牢与深渊领主战场。
- **西翼**：奥库萨吞噬后的「以体型自锁」巨室；4.2 后走廊多奥库萨之眼。
- **北翼**：阿丽萨巴尔与憎恶门徒舞台；4.3 终局翼。
- **侧翼任务区**：诅咒深渊（Cursed Depths）、D 区（D-Block）、地洞（The Hole）——控制方轮换清剿与档案回收。
- **对照：紫罗兰监狱 / 达拉然**：肯瑞托曾试图关押阿尔加洛斯与阿丽萨巴尔，失败后转送托尔巴拉德——政治上是「肯瑞托装不下，所以扔进海岛」的前史。

AI 写场景：先定契约者在「半岛补给 / 三据点炮火 / 堡门权限 / 东翼 / 西翼 / 北翼 / 军团失守」哪一层，再选冲突——外围是阵营与幽灵，中层是控制权与封印巡检，深层是三恶魔与后期夺眼。

**【世界剧情线】**

**一、前史：阿拉索要塞与第二次大战的焦土**
托尔巴拉德要塞由人类首国**阿拉索**修建，历史逾一千二百年；第一次大战前属**激流堡（Stromgarde）**势力圈。第二次大战中，它是洛丹伦联盟打击部落占据的丹莫德等地的前哨与补给跳板。兽人视其为眼中钉，发动凶猛反攻：激流堡守军措手不及，岛民大多被屠杀，要塞被夷为焦黑废墟，成为部落「胜利的焦炭符号」。死者的幽灵据说仍在雾里呼喊复仇——这不是装饰性恐怖，而是日后日常任务里「战场重演的亡魂」的源头。少数原激流堡血脉幸存者（含流亡者）仍与土地绑定。

**二、改建：库尔提拉斯法师与「向内的墙」**
战后废墟闲置，直至**库尔提拉斯**法师宣称岛屿、用战争伤疤的石料重建高墙，并把防御逻辑巧妙**翻转向内**：不是防外敌进城，而是防内囚出城。一座神秘最高安保监狱——**巴拉丁监狱**——就此成型。激流堡与库尔提拉斯混居人口组成**巴拉丁典狱官**，其传统可追溯到**雷金纳德·巴拉丁二世公爵（Duke Reginald Baradin II）**一类奠基叙事：职责是确保致命囚犯「永不再踏上艾泽拉斯自由的土地」。**肯瑞托**亦将过于危险的恶魔与实体送入此处。锈港（Rustberg）后因激流堡国势衰颓被弃，沦为海盗据点；海上无尽的海军摩擦环绕此岛。

**三、为何关在这里：三名不宜「别处」的囚徒**
Adventure Guide 与首领文案给出三枚钉子：
- **阿尔加洛斯（Argaloth）**：体型与战力巨大的深渊领主（歼灭者魔 / pit lord）。肯瑞托无法将其安置于达拉然强大的**紫罗兰监狱**，只得关入托尔巴拉德地下。
- **奥库萨（Occu'thar）**：贪婪的空灵恐魔。它挣脱牢笼，进入西翼，吞噬那里所有囚犯；暴食使它臃肿到无法逃出环境——西翼本身变成它的新囚笼。黑色幽默：越狱成功了一半，失败在「吃太多」。
- **阿丽萨巴尔（Alizabal）·仇恨之主**：希瓦拉。腐化法师将其召唤到艾泽拉斯后，肯瑞托关入紫罗兰监狱；即便奥术结界限制其力量，她恶意的**影响本身**仍令守卫在突发暴怒中自相残杀。裂变前被转送巴拉丁，只让已然崩坏的监狱雪上加霜。

肯瑞托一度管理这座「装满怪物与恶魔」的监狱，直至被内部冲击与越狱浪潮冲垮。

**四、裂变：完整性破坏与控制权战争**
死亡之翼破界、大地裂变后，监狱结构与符文完整性受损。加尔鲁什·地狱咆哮派遣精锐**地狱咆哮近卫军**「从人类手中夺走此岛」；巴拉丁典狱官死守。岛上同时滋生：守卫互杀或释放囚犯、疯狂、升起的亡灵、重演旧战的幽灵。联盟与部落的周期性战场围绕三要塞展开——**同时控制铁甲驻地、熔渣车间、典狱官岗哨**者控制巴拉丁堡，获得日常中心任务板与**进入地下团队本的权限**。堡外炮火连天，堡内却长期「安静得可怕」：关着的东西太致命，既不能简单链在地面，也不能流放。

**五、团队清剿三阶段（补丁史即剧情史）**
副本在设计上预留扩展首领位，避免「地图突然变长」的违和。
- **4.0.3a · 东翼阿尔加洛斯**：初始唯一团队首领。掉落 T11 与凶恶角斗士（Season 9）相关档。战斗强调流星斩、邪能风暴、吞噬黑暗等深渊领主式范围与团队分散。叙事意义＝「肯瑞托装不下的东西，由控制权胜者来处决」。
- **4.2 · 西翼奥库萨**：破封印登场。原恶魔收容单元被其摧毁，走廊改为**奥库萨之眼**巡逻包。掉落升至 T12 / 残酷角斗士（Season 10）。叙事意义＝「越狱失败的暴食者仍是团灭级威胁」。
- **4.3 · 北翼阿丽萨巴尔**：终局希瓦拉。走廊换成**憎恶门徒（Disciples of Hate）**。掉落 T13 / 灾变角斗士（Season 11）。叙事意义＝「仇恨本身腐蚀制度——她不必出手，守卫先自相残杀」。

冒险者按控制权轮流进入斩首。同阵营 10/25 人；权限与巴拉丁典狱官 / 地狱咆哮近卫军声望经济绑定。

**六、清理、反扑与军团再临失守**
奥库萨被杀后，清理队试图重建，却遭**奥库萨隆（Occul'tharon）**等新恶魔反客为主。第三次燃烧军团入侵期间，托尔巴拉德失守于狱中与入侵者，驻军撤离。阵营间为监狱的大规模火并在此阶段基本停歇——更大的敌人已压过岛影。

**七、暗影议会与达拉然之眼**
术士相关线语境中，**暗影议会**术士（古尔丹体系下）潜入，杀死奥库萨隆，取走其不知如何在扭曲虚空中获得的**达拉然之眼（Eye of Dalaran）**用于仪式。行动者中甚至混有「黑穗议会（Council of the Black Harvest）」身份的复杂立场。此物如何落入虚空恐魔之手，成为跨阵营情报谜题：监狱不仅关恶魔，还曾关过——或被恶魔吞过——肯瑞托级神器线索。

**八、第四次大战后：肖的报告与无典狱长真空**
《Exploring Azeroth: The Eastern Kingdoms》语境下，**马迪亚斯·肖**的军情七处报告（与巴拉丁典狱官监视结论一致）：狱中仍有恶魔；**内部无典狱长驻守**；偶有零星恶魔出逃（从不成建制），被击杀在目击处；**余数未知**。世界状态长期停在「半失控最高安保监狱」而非「已净化旅游点」。

**九、主题与可改 / 不可改**
主题：向内的墙与向外的炮火、控制权即钥匙、收容制度的失败美学、暴食与仇恨对秩序的嘲讽、战后「清点不了」的诚实。可改：契约者阵营站队、三据点争夺路径、先清哪一侧翼日常、是否追查达拉然之眼。不可无代价：无控制权硬进团队本、把三首领写成被封印废柴、让监狱一夜净化为景区、白送完整巴拉丁控制权与全部恶魔名单。

**【主要人物】**
- **阿尔加洛斯（Argaloth）**（深渊领主 · 东翼）｜性格：歼灭者魔式暴虐与傲慢，视锁链为挑衅｜装备·能力：流星斩、邪能火风暴、吞噬黑暗；T11 档团本破坏力｜人物弧光：紫罗兰监狱装不下 → 托尔巴拉德地下收容 → 裂变后被控制权胜者处决｜立场关系：本档五阶主威胁；低阶一律情报优先。
- **奥库萨（Occu'thar）**（空灵恐魔 · 西翼）｜性格：贪婪、暴食、非人理性｜装备·能力：体型碾压、眼球召唤与分割战场；T12 档｜人物弧光：破牢 → 吞尽西翼囚犯 → 胖到逃不出 → 被冒险者斩杀｜立场关系：六阶；「越狱失败」黑色传说本体。
- **阿丽萨巴尔（Alizabal）**（希瓦拉 · 仇恨之主 · 北翼）｜性格：恶意、挑衅、以仇恨为武器｜装备·能力：刃舞与暗影、影响他人暴怒互杀；T13 档｜人物弧光：被腐化法师召唤 → 紫罗兰监狱令守卫互杀 → 转送巴拉丁加剧崩坏 → 4.3 终局被斩｜立场关系：七阶裂变期顶点。
- **奥库萨隆（Occul'tharon）**（继任空灵系威胁）｜性格：压迫式入侵者｜装备·能力：军团期监狱反扑核心之一；曾持达拉然之眼｜人物弧光：清理期入侵 → 被暗影议会杀死夺宝｜立场关系：八阶线关键节点。

**【叙事基调 · 雷区】**
画风：锈雾海岛、焦黑要塞、炮火与突然安静的地下、邪能绿与仇恨紫、眼魔的湿冷凝视。口吻阴冷务实，可带黑色幽默，忌土味沙雕。忌把巴拉丁写成普通五人本节奏；忌忽略「先打控制权再进本」；忌把三首领写成被封印废柴或战力为零；忌把托尔巴拉德当主城安全区；忌与暮光堡垒串台（古加尔 / 升腾者 ≠ 阿尔加洛斯 / 奥库萨 / 阿丽萨巴尔）。最早切入：托尔巴拉德半岛港口日常，或三据点争夺的第一声号角。

**【托尔巴拉德控制权机制详述】**
托尔巴拉德不是传统「排队进本」地图：联盟与部落必须周期性争夺三座主要要塞。任一阵营在结算窗口同时控制**铁甲驻地、熔渣车间、典狱官岗哨**，即控制巴拉丁堡，并获得进入巴拉丁监狱团队副本的权限与中心日常板。战场节奏接近资源点拉锯：旗标占领（类风暴之眼式存在与击杀推进）、攻城器械摧毁尖塔以延长或巩固、1 小时级循环。胜者获通令与声望向军需。契约者若忽略战场、直接走向副本门口，会在**权限**上被硬挡——这是本世界与普通副本世界的最大制度差异。AI 写「进本」前必须交代本周 / 本阶段谁控制岛屿。

**【三要塞与半岛双层舞台】**
半岛 hub 对全阵营开放：补给、基础日常、资源采集、锈港摩擦。中心岛控制权决定：堡内任务板、侧翼轮换（诅咒深渊 / D 区 / 地洞）、团队入口。写日常应区分「半岛总能做」与「控制权才能做」。三要塞各有地形性格：铁甲偏北防、熔渣工业热与器械、典狱官岗哨偏西南监视与纪律——可作风味，勿编造不存在的独立团本首领。

**【三首领机制与补丁史细述】**
**阿尔加洛斯**：东翼。深渊领主站场，流星与邪能风暴逼迫分散，吞噬黑暗点名处理。掉落 T11 + 凶恶角斗士档。文案钉：紫罗兰监狱装不下。
**奥库萨**：西翼。暴食后体型锁死；战斗含眼球与爆炸性威胁，走廊预热眼魔包。掉落 T12 + 残酷角斗士档。文案钉：吞尽西翼囚犯，胖到逃不出。
**阿丽萨巴尔**：北翼。希瓦拉刃与咒、仇恨主题；憎恶门徒前哨。掉落 T13 + 灾变角斗士档。文案钉：恶意影响令守卫互杀，转送后加剧巴拉丁崩坏。
副本结构设计预留扩展首领位：同一地下，分期开门，走廊生态随「里面的东西」改写——收容单元 → 眼魔 → 门徒。

**【恶魔分类学与鉴别】**
阿尔加洛斯＝深渊领主 / 歼灭者魔；奥库萨＝空灵恐魔（虚空犬类巨大化）；阿丽萨巴尔＝希瓦拉；奥库萨隆＝后续虚空系威胁。门徒与眼球是可重复小怪文化符号。契约者鉴别种类的能力可成中阶任务技能奖励：认错种类=战术全错（对希瓦拉用对空灵的打法会团灭）。

**【军团再临与战后真空细述】**
冒险者击杀奥库萨后的清理并未完成重建：奥库萨隆等反客为主。第三次燃烧军团入侵使托尔巴拉德失守，驻军撤离。暗影议会潜入杀死奥库萨隆、取达拉然之眼——阵营夺岛战在更大战争下熄火。第四次大战后，肖与典狱官一致：仍有恶魔、无内部典狱长、零星出逃即杀、总量未知。世界状态＝半失控，不是旅游点。八阶线写「条件性胜利」：夺回一条走廊、截住一件仪式物、带回一份可信名单——而非一回合净化全岛。

**【阵营日常与可切入微观事件】**
护送补给穿过炮火；摧毁敌对攻城器械；审讯逃囚；修复封印符文；争夺军需折扣声望；夜间巡逻遭遇越狱个体；从漂流船打捞肯瑞托旧档案；侧翼清亡灵与邪能兽；锈港交易情报。写契约者日常优先这些可重复事件，而不是一上来阿丽萨巴尔。权限链：半岛立足 → 日常声望 → 战场贡献 → 控制权 → 团队钥匙 → 首领战 →（后期）失守与夺眼。

**【与暮光堡垒 / 阿尔卡冯的对照（防串台）】**
巴拉丁＝海岛监狱 + 控制权团队本 + 恶魔三翼；暮光堡垒＝暮光高地邪教总部 + 上古之神崇拜线性团队本（半萨 / 升腾者议会 / 古加尔等）。阿尔卡冯宝库＝诺森德冬拥湖式控制权短本，制度相似但地理、首领、掉落完全不同。契约者若把古加尔台词或阿尔卡冯龙相带到托尔巴拉德，即 OOC。

**【监狱建筑学与翼区动态】**
向内之墙：外墙厚、内通道收束、符文嵌石。东 / 西 / 北三翼分期开放；西墙外侧入口与中心堡垒控制权分离，强化「政治权限」与「物理空间」双锁。走廊生态随补丁替换，适合写契约者跨版本 / 二周目：同一走廊不同时期是不同战场。AI 写移动必须交代「在哪一翼、本周走廊是眼魔还是门徒、控制权是否仍在」。

**【声望商人与周常节奏】**
控制权阵营提供日常与周常，声望兑换装备与消耗。团队首领击杀与战场胜利共同构成托尔巴拉德玩家经济。写任务方向时应把「本周是否控制岛屿」作为世界状态开关：无控制权时，团队入口关闭，故事转向外围游击、半岛情报战与敌后破坏。

**【感官辞典（巴拉丁专用）】**
气味：铁锈、硫磺、潮雾、邪能甜腥、火药。声音：远处攻城器、近处滴水与锁链、眼魔湿响、希瓦拉金属环佩、号角比咒语先到。视觉：雾里抢颜色的三色旗帜、地下突然的安静、收容符文裂开的齿痕、西翼门矮因为「东西太大」。触觉：靴底粘冷霉、手握通令徽章发烫（刚打完战场）。一～二阶对话轮换：港口军需「先活过外围」；伤兵「堡里安静才可怕」；敌对斥候「控制权不写在纸上」。

**【战役阶段锁定（供 AI 控地图权限）】**
阶段零：半岛登岛，听炮火方向。阶段一：日常与声望，侧翼初探。阶段二：三据点拉锯，首次控制权结算。阶段三：胜方整备，封印巡检，入口开启。阶段四：团队走廊（按版本生态）。阶段五：东翼阿尔加洛斯。阶段六：西翼奥库萨。阶段七：北翼阿丽萨巴尔。阶段八：清理与奥库萨隆反扑。阶段九：军团失守与撤离。阶段十：夺眼与肖的报告式真空。AI 可按阶段锁「有无控制权 / 在哪一翼 / 三首领谁存活 / 是否已失守」。

**【契约者阶位使用说明】**
一阶：半岛跑腿、杂兵、幽灵边缘。二阶：据点精锐与侧翼精英。三阶：军官级护送与封印巡检。四阶：团队走廊。五阶：阿尔加洛斯。六阶：奥库萨。七阶：阿丽萨巴尔。八阶：失守、夺眼、战后清点。越阶硬刚=团灭叙事。低阶对三首领本体一律情报优先。

**【失败 / 成功态与团队政治】**
失败：控制权丢失导致入口关闭；无人处理点名导致灭团；眼魔分割治疗链；仇恨影响导致队伍内讧（叙事可借用阿丽萨巴尔主题，勿无代价洗脑主角）；失守时硬守全灭。成功：三据点在手、三首领分期斩首、截获夺眼线索、带回可信余数报告。联盟部落在入口仍可互黑，进战必须同步指挥。掉落须绑首领击杀叙事，禁止任务 NPC 白送 T13。

**【人名强制真名清单】**
阿尔加洛斯、奥库萨、阿丽萨巴尔、奥库萨隆、马迪亚斯·肖、雷金纳德·巴拉丁二世公爵、加尔鲁什·地狱咆哮、大法师加卢斯、斯瓦诺斯、普罗布利姆；势力名：巴拉丁典狱官、地狱咆哮近卫军、肯瑞托、暗影议会、军情七处、紫罗兰监狱。禁用「牙人 / 群像 / 相关存在」代称。

**【收束 · 巴拉丁监狱】**
向内的墙、三色旗帜、三翼恶魔、控制权钥匙、吃到逃不出去的恐魔、令守卫互杀的仇恨、战后清点不了的诚实——写尽这些，巴拉丁不是「打完三个 boss 的短本」，而是一座永远半开的最高安保伤口。契约者带着通令与一件邪能腥味的掉落离开时，应记住：海上的雾不会告诉你余数，只有炮火和报告会。

**【半岛经济与资源循环】**
草药（艾萨拉雾菇类、燃烬草、鞭尾等区域表）、矿石（源质、燃铁矿脉语境）、通令兑换、声望军需折扣、战场击杀与占领贡献。写「赚钱」优先这些，勿把主城拍卖行逻辑硬塞进雾岛。控制权易手=价格与货架政治性跳变。

**【侧翼三区戏剧功能】**
诅咒深渊＝亡灵与诅咒叙事；D 区＝编号化监狱暴力与档案；地洞＝地下结构失稳与逃亡通道。轮换任务板让同一契约者周常不重复同一翼，却重复同一恐惧：名单上的名字可能已在西翼的胃里。

**【阿尔加洛斯战斗与文案锚点】**
「肯瑞托的紫罗兰监狱关不住他。」开战即温度上升；流星斩分割站位；邪能风暴覆盖；吞噬黑暗点名。坦克链与治疗吞吐决定东翼是否周常化。击杀后搜刮短暂：东翼安静，西翼仍在呼吸。

**【奥库萨战斗与文案锚点】**
「它破牢、吞尽西翼、胖到逃不出。」门比东翼更矮是建筑被「内容」改写的证据。眼球先于本体；本体是会呼吸的监牢。击杀意义＝确认「关得住」的失败案终于以死亡结案，而非以释放结案。

**【阿丽萨巴尔战斗与文案锚点】**
「恶意影响令守卫互杀。」北翼终局；憎恶门徒让出舞台；刃与咒与仇恨主题。写她忌扁平魅魔，应保留「制度腐蚀者」：她让纪律先死，再动手。T13 档掉落是裂变末期装备阶梯的海岛出口之一。

**【夺眼线执行要点】**
目标物：达拉然之眼。持有过：奥库萨隆。夺取者：暗影议会（含黑穗立场混杂）。舞台：失守后的巴拉丁。契约者八阶可写：抢先一步、半路截击、只夺回碎片、或只留下肖式报告。禁止无代价从古尔丹手中抢全胜。

**【可观察细节扩写库（巴拉丁专用）】**
雾里三旗抢色；集合石旁西墙入口的冷风；收容符文上的齿痕；西翼门楣被撑裂的石缝；眼魔液体滴在靴钢上的声；通令徽章边缘的血垢；战后报告末行「余数未知」的墨水未干。AI 优先用这些，禁止跨世界「护送假货」套话。

**【开场六小时与终局六小时】**
开场：半岛旅店铁锈味，军官抛来外围清理；远处巴拉丁堡炮口沉默，像在听谁获胜。终局：三翼曾安静又再嘈杂，撤离号角比战斗号刺耳；有人抱着档案跑，有人抱着眼睛的传说跑；肖的报告在未来某页写下「无典狱长」。监狱教会艾泽拉斯：有些墙是为了不让里面出来，有些战争是为了谁有资格打开门。

**【战场口令样本】**
「铁甲需要增援——熔渣攻城车左转——岗哨旗要掉了——三据点亮了守十五——入口队伍集合石集合。」失败态：一据点被偷导致控制权整体丢失。成功态：三据点稳、军需开门、团队钥生效。

**【元素与邪能伦理（本档）】**
本档主威胁是恶魔与制度，不是元素领主。勿把奥拉基尔 / 拉格纳罗斯战力表硬套阿尔加洛斯。映射宁低勿高：阿尔加洛斯五、奥库萨六、阿丽萨巴尔七，失守八。条件性胜利可接受逼退、夺物、政治控制；不可把顶点写成无威胁空壳。

**【与紫罗兰监狱边界】**
阿尔加洛斯与阿丽萨巴尔的前史在达拉然；本条终局以托尔巴拉德收容与处决为主，紫罗兰监狱五人本线点到为止，避免整本倒灌。

**【最后事实钉】**
入口托尔巴拉德中心地下；西墙外约 [46,48]；10/25 人；三首领阿尔加洛斯 / 奥库萨 / 阿丽萨巴尔；控制权三要塞铁甲 / 熔渣 / 典狱官岗哨；联盟巴拉丁典狱官 vs 部落地狱咆哮近卫军；后期奥库萨隆与达拉然之眼；战后无典狱长。人名须真，掉落须绑首领，低阶勿硬刚希瓦拉。

**【补足 · 二战幽灵作为日常层】**
雾中重演的冲锋、无实体的号角、对「兽人」或「人类」无差别的怨恨——侧翼与半岛任务常用。契约者一阶应学会：有些敌人不是恶魔，是历史。

**【补足 · 向内防御工程】**
墙垛箭孔朝内、吊桥逻辑反装、符文锁从外加固——库尔提拉斯改建的可观察细节。写潜行进堡：难度不只在守卫，在「建筑与你作对」。

**【补足 · 团队编制与指挥伦理】**
10/25 人要求坦克稳定阿尔加洛斯与阿丽萨巴尔近战位，奥库萨眼球分配明确；治疗盯吞噬黑暗与范围邪能；打断与驱散按版本小怪表。联盟部落混编时战前互黑、战后分赃常态，战斗中指挥权临时统一。契约者若持跨阵营指挥外挂，应写成政治代价。

**【补足 · 战后清剿诚实原则】**
领主 / 恐魔 / 希瓦拉倒下 ≠ 名单清空。余数未知是设定，不是偷懒。远征军转入监视与零星击杀。本条收束句：雾还在，门还在，余数未知。

**【补足 · 收束句】**
谁控制三座要塞，谁书写巴拉丁的开门权；谁走完三翼，谁把「装不下」的东西从石头里量出。锈仍腥，雾仍低，任务仍在半岛与地下之间循环。档案完。

**【地理坐标与实例规则备忘】**
巴拉丁堡约 [47.7, 53.0]；团队路径西墙外约 [46, 48]；同阵营实例；控制权丢失可在叙事上关闭入口直至下一结算。写「飞进地下」无原作依据——准入是政治与门，不是飞行坐骑副本。

**【军需与掉落经济终表】**
战场通令；声望军需；阿尔加洛斯 T11+S9；奥库萨 T12+S10；阿丽萨巴尔 T13+S11；侧翼材料与档案。每件对应：半岛、据点、东翼、西翼、北翼、夺眼。写尽控制权、三翼、失守与余数未知，巴拉丁监狱常青档案完成。

**【肯瑞托责任链】**
紫罗兰监狱两次溢出（阿尔加洛斯体量、阿丽萨巴尔影响）→ 转送托尔巴拉德 → 裂变后完整性崩溃 → 控制权战争把「处决权」外包给阵营胜者。契约者若持肯瑞托身份，可获档案权限，但须背负「谁把东西扔进海里」的舆论。

**【加尔鲁什动机与近卫军纪律】**
近卫军不是散兵：夺岛令来自加尔鲁什扩张逻辑，岛上表现是攻城、守旗、军需纪律。写部落侧忌「无脑蛮子」；写联盟侧忌「永恒正义」——两边都在抢开门的钥匙，门后是同一类要命的东西。

**【阿丽萨巴尔与「队伍内讧」叙事边界】**
可用环境与 NPC 互疑表现仇恨影响；不可无代价剥夺契约者神智或永久洗成仇敌。机制上的挑衅 / 凝视应落在可驱散、可打断、可走位层。

**【奥库萨隆与「继任」逻辑】**
虚空恐魔生态不因一只死亡而断绝；奥库萨隆是清理失败的象征，不是简单换皮重复。夺眼使其从「又一只恐魔」升级为「神器节点」。

**【马迪亚斯·肖报告文体】**
战后线可模仿密探备忘：短句、不确定量词、建议监视而非豪言净化。这是八阶收束口吻，不是一阶港口任务口吻。

**【Rustberg 锈港】**
弃城—海盗—情报黑市。低阶可买到假控制权徽章（陷阱）与真侧翼地图抄本。中阶可清海盗换半岛安全。勿写成主城。

**【成就与周常叙事化】**
公会击杀、各首领成就可化为「军需额外补给」与「阵营宣传单」——仍须绑真实击杀，禁止广告式白给。

**【与冬拥湖 / 阿尔卡冯制度对照一句】**
同为「战场控制 → 短团队本」，巴拉丁多一层监狱恐怖与恶魔名册，少一层龙相看守；勿混掉落表与首领名。

**【邪能与雾的视觉对立】**
岛上外景灰锈雾；地下邪能绿与仇恨紫。转场时写「雾被门缝切断」比写长段风景更有效。

**【最终可执行检查表（给 AI）】**
本周控制权？在半岛还是堡内？走廊生态版本？三首领谁存活？是否已进入军团失守后？任务是日常 / 战场 / 团队 / 夺眼哪一种？人名是否真名？掉落是否绑击杀？低阶是否在硬刚希瓦拉？——任一项「否」则重写该段。

## 阶位切入点

> 阶位↔巴拉丁监狱：一阶≈半岛外围杂兵与幽灵边缘；二阶≈三据点精锐；三阶≈军官与收容巡检；四阶≈团队走廊；五阶≈阿尔加洛斯；六阶≈奥库萨；七阶≈阿丽萨巴尔；八阶≈军团期失守与夺眼真空。世界顶点（裂变期阿丽萨巴尔 / 后期地区失控）为跨阶存在，低阶切入一律「规避顶点·情报优先」。

**一阶（半岛 · 登岛日常线）**
切入身份/时点：刚登岛的志愿兵 / 佣兵 / 运输船水手，锚定裂变后首次托尔巴拉德半岛日常。
初始事件：港口军官抛来一叠外围清理——幽灵、逃囚残党、失控构造体；远处巴拉丁堡炮口沉默，像在听谁获胜。
开场白建议：「海风全是铁锈味。军官说：先活过外围，再谈堡垒里的东西。你抬头时，看见三座据点的旗帜在雾里抢颜色。」
关键NPC立场：**巴拉丁典狱官军需官**（或部落对等军需，按站队）——按声望给任务；**地狱咆哮近卫军斥候**——敌对；**本地伤兵**——传播狱中传闻；**锈港走私者**——出价买你的通行证。
主线钩子/支线：日常清理、声望起步；支线＝捞取漂流物资、审讯逃囚、绘制幽灵出没点。
危险度/规避：低～中——规避单独深入堡内地道与任何「深渊领主」本体。
任务方向/奖励：站稳半岛；日常币、基础装、通令初见。

**二阶（据点战 · 三要塞线）**
切入身份/时点：战场参战者，锚定争夺铁甲驻地 / 熔渣车间 / 典狱官岗哨。
初始事件：号角响起，契约者被编入冲锋波，必须在计时内拿下并守住据点，否则整队无法获得巴拉丁控制权。
开场白建议：「旗帜换手的速度比咒语还快。你刚把旗插上，敌对攻城车就从斜坡碾过来——控制权，从来不是写在纸上的。」
关键NPC立场：**据点军官**——要战果；**敌对突击队**——要人头；**战场军医**——中立偏阵营；**攻城器械技师**——要护车。
主线钩子/支线：三据点循环、控制权结算；支线＝破坏器械、截获军情、护旗十五分钟窗口。
危险度/规避：中——规避敌对人数碾压的空窗；勿在结算前冲团队门。
任务方向/奖励：控制权贡献；战场荣誉、通令、堡垒外围权限。

**三阶（军官 · 权限钥线）**
切入身份/时点：受信任的军官级打手，锚定控制权易手后的堡垒整备夜。
初始事件：胜方要求护送法师小队检查监狱收容符文裂隙；走廊深处传来非人呼吸，检查队有人失踪。
开场白建议：「控制权到手的夜晚最危险——因为你们终于有资格打开那扇门。门后第一口空气，像是铁与硫磺煮过的。」
关键NPC立场：**胜方指挥官**——要稳定；**肯瑞托遗留文献员**——要档案；**失踪法师的同僚**——要救援；**大法师加卢斯**类残留异常——要回避或净化。
主线钩子/支线：收容巡检、入口开启仪式；支线＝修复防御塔、清点囚犯名册、侧翼（诅咒深渊 / D 区 / 地洞）轮换。
危险度/规避：中高——规避擅自释放任何「看起来虚弱」的囚室。
任务方向/奖励：团队入口资格；中层装、钥匙碎片、声望跃升。

**四阶（团队走廊 · 门徒与眼魔线）**
切入身份/时点：团队先锋，锚定副本走廊生态（收容单元 → 眼魔 → 憎恶门徒随版本）。
初始事件：第一包走廊怪拉空，旧收容单元已被摧毁，取而代之的是会凝视的眼球或挥舞仇恨的门徒。
开场白建议：「走廊的灯灭了三盏。地上的收容符文裂开，像有人从里面咬穿。你的队伍还没见到首领，空气已经先敌对了。」
关键NPC立场：**团队指挥**——要节奏；**憎恶门徒**——要血；**奥库萨之眼**——要视野；**记录官**——要生态版本记录。
主线钩子/支线：走廊清场、版本生态辨识；支线＝搜集狱卒日志、拆除残余构造、确认本周开哪一翼。
危险度/规避：高——规避无打断无驱散硬推；勿把走廊当主城过道。
任务方向/奖励：抵达首领门；团队初步分配权、消耗品、走廊材料。

**五阶（阿尔加洛斯 · 东翼深渊线）**
切入身份/时点：正式团队成员，锚定东翼深渊领主战。
初始事件：阿尔加洛斯从锁链与符文中起身，邪能流星划开场地，团队必须在击杀与分散之间找到节奏。
开场白建议：「东翼的温度先升起来。深渊领主的笑声不像嘲讽，更像在点名——点那些敢把它关进石头里的名字。有人低声说：紫罗兰监狱关不住它。」
关键NPC立场：**阿尔加洛斯**——要毁灭；**团队坦克链**——要仇恨；**治疗链**——要活口；**阵营观察员**——要宣传首杀。
主线钩子/支线：首杀 / 周常击杀、T11 与凶恶角斗士档掉落；支线＝研究其收容原因、比对紫罗兰监狱档案。
危险度/规避：高——规避集火失误灭团；**低阶禁止单挑**。
任务方向/奖励：东翼首脑；T11 档装备、通令加成叙事。

**六阶（奥库萨 · 西翼吞噬线）**
切入身份/时点：经验团队，锚定 4.2 后西翼空灵恐魔。
初始事件：奥库萨的体型塞满房间，眼球在走廊预热；它曾吃光西翼囚犯，现在轮到你们证明「关得住」或「杀得掉」。
开场白建议：「西翼的门比东翼更矮，因为里面的东西把自己吃得太大。眼球先看着你，然后本体才挪动——像一座会呼吸的监牢。」
关键NPC立场：**奥库萨**——要吞噬；**眼球群**——要分割团队；**记录官**——要确认它不会再破牢；**名册员**——要被吞噬者名单。
主线钩子/支线：奥库萨击杀、T12 / 残酷角斗士档；支线＝清点被吞噬者、重建西翼临时收容阵。
危险度/规避：极高——规避视线与眼球机制失败；勿在击杀前打开侧门「疏散」叙事导致二次越狱。
任务方向/奖励：西翼安定；T12 档装备、黑色幽默战报。

**七阶（阿丽萨巴尔 · 北翼希瓦拉线）**
切入身份/时点：满编团队，锚定 4.3 北翼终局。
初始事件：阿丽萨巴尔以希瓦拉之姿接受挑战，憎恶门徒让出舞台；刃与咒把北翼变成屠宰场。
开场白建议：「北翼的香火味是假的。希瓦拉低头看你们时，像在看一盘已经摆好的祭品——而你们偏要自己跳上桌。有人想起：她曾令紫罗兰监狱的守卫自相残杀。」
关键NPC立场：**阿丽萨巴尔**——要征服；**憎恶门徒**——要助战；**阵营观察员**——要宣传胜利；**军需官**——要 T13 入库。
主线钩子/支线：北翼击杀、T13 / 灾变角斗士档；支线＝阻断其与狱外的精神连线、收缴门徒徽记。
危险度/规避：极高——规避嘲讽与特殊阶段失误；**超阶巅峰：存在·情报优先/条件性胜利**——可接受惨胜与重伤撤离，严禁把顶点写成无威胁空壳。
任务方向/奖励：三首领闭环；T13 档装备、监狱稳定声望、裂变末期海岛出口装。

**八阶（军团失控 · 失守与夺眼线）**
切入身份/时点：军团再临期特遣 / 黑穗相关术士协力者 / 军情七处编外，锚定托尔巴拉德失守、奥库萨隆与暗影议会行动。
初始事件：防线崩溃，恶魔从裂隙涌出；契约者在撤退命令与「达拉然之眼线索」之间抉择，目睹暗影议会比你们更早摸到宝物。
开场白建议：「撤离的号角比战斗号更刺耳。典狱官说守不住了——可军情七处的密信写着：有东西比越狱更重要，它叫达拉然之眼。」
关键NPC立场：**奥库萨隆**——要撕开监狱；**暗影议会术士**——要眼睛；**马迪亚斯·肖**——要战后真相；**残存典狱官**——要活口名单。
主线钩子/支线：失守战役、夺眼任务线、战后清点；支线＝营救被困法师、销毁被污染档案、确认「无典狱长」真空。
危险度/规避：贴近顶点——八阶地区灾难：**存在·情报优先/条件性胜利**（截获碎片、带回报告、守住一条撤离廊）；恶魔仍具完整威胁，严禁写成无害空壳，严禁一回合净化全岛。
任务方向/奖励：阻止宝物落入最坏之手或记录损失；军团期情报、仪式残件、阵营密令、肖式备案权。
