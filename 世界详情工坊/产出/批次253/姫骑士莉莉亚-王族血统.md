# 姫骑士莉莉亚-王族血统
<!--meta lib=主库 tiers=三-->
## 剧情

**【作品来源】**
《姫骑士リリア 〜魔触の王城に堕つ〜》（中译常见「姫骑士莉莉亚」）为 Lilith 旗下 BLACK Lilith 2004年8月28日下载／9月17日包装发售的成人向多结局 AVG；2005年完全版追加剧本；2006–2010年 PIXY 改编 OVA 全六卷（累计约15万本），2011年 DVD-BOX。原画 ZOL；OVA 监督浅野ヒロ、系列构成向原作。气质：中世纪边境王国政变、魔族契约、白衣姫骑士的抵抗与沦陷。**以游戏主线＋OVA 公开梗概为准**。成人向条目只写剧情、战力、组织与任务介入；不写色情过程描写。

**【世界定位】**
边境国**ルーフェン王国**。义兄**ディルク・エーベルヴァイン**借魔族之力夺权，向魔族支付「堕落时剥离的魂能」作对价；期限未满则自身性命不保。义妹**リリア・エーベルヴァイン**（白之姫骑士）与義母**レイラ**成为调教与夺能对象；娼馆「黒の楽園館」女主人**キリコ**（魔族）作参谋。卡片锚点随副标题切换：王族血统／完全堕落。一句话：ルーフェン王城里，白之姫骑士与魔族契约的死线赛跑。
本条目卡片锚点：**王族血统——エーベルヴァイン之血被计价为魔族对价**。

**【世界观 · 力量体系】**
力量本源：骑士武技与圣性魂能、魔族契约与魔触、王权私兵与魔将代收。死亡：凡人战死即灭；魔族将官级难以常规斩杀；魂能被抽空导致人格崩坏而非「免费复活」。特殊系统：对价期限日数、魂能计量、多结局（付清与否×リリア状态）。

力量阶梯（宁低勿高）：
1. **平民／侍女／杂兵**：刀剑棍棒，≈一阶。
2. **王国常备兵／白骑士下级**：破门级战，≈二阶。
3. **リリア・エーベルヴァイン（白之姫骑士）／ミーア与四白骑士精锐**：可破城门、单骑冲阵、圣性魂能抗魔触；本世界任务主战力，≈**三阶**。
4. **キリコ（参谋魔族）／私兵数千级**：城内权谋＋魔族体术，≈四阶边缘（本档仅三阶覆盖时作压迫者，不发四阶切入）。
5. **魔族従者サンド（将官级代收）**：跨阶威压，≈五阶边缘。
6. **魔王本体**：国家级征服与养女化，**超阶巅峰**：存在·情报优先／条件性胜利，严禁「被封镇所以战力归零」。

**乐园阶位映射**：平民杂兵≈一阶；常备兵／下级白骑士≈二阶；リリア与精锐白骑士／クローヴィス战役级女战士≈**三阶**；キリコ与私兵网≈四阶边缘；サンド将官≈五阶边缘；魔王≈超阶巅峰。本任务世界仅覆盖**三阶**；低阶切入规避魔王本体。

**【地理 · 舞台】**
- **ルーフェン王城**：政变舞台；地牢、谒见厅、寝殿、城墙。
- **黒の楽園館**：キリコ势力据点；情报与私兵入口。
- **クローヴィス战役旧战场**：リリア成名地（背景）。
- **邻国边境**：OVA 终盘リリア率军侵攻／反旗之地。
- **魔族领域投影**：代收日与魔王现身时短暂覆盖王城。
写场景：白衣铠甲污痕、城钟期限、私兵火把、魔族绷带目光。

**【世界剧情线】**
① **クローヴィス战役余晖**  
リリア以「白之姫骑士」之名载入王国史；エーベルヴァイン王族血统同时意味着圣性魂能浓度与政治象征。ディルク因母亡后的放纵被诸侯称「ダメ皇子」，对义妹战功怀劣等感。

② **国王倒地与政变**  
国王病倒（游戏）或被ディルク刺杀（OVA）。キリコ献策：借魔族之力一夜掌握ルーフェン。リリア武勇再高，亦难敌跨阶魔援＋内应。

③ **王族血统成为对价燃料（卡片锚点）**  
魔族要的不是金币，而是「女性堕落时剥离的魂能」。王族血统＝高纯度エナジー。ディルク必须在期限内从リリア（及レイラ）身上榨出对价，否则自身抵命。血统从荣耀变为计量条。

④ **白骑士与侍女网**  
ミーア与四白骑士试图护主；全国动员令把臣民变成施压工具。契约者可在「护送王族信物」「截获代收清单」「保护ミーア联络线」介入。

⑤ **期限日逼近**  
サンド现身代收。付清与否决定ディルク存亡；リリア魂能残留决定她是否仍握剑。

⑥ **终局分叉**  
付清但王国崩坏；或未付清ディルク死；OVA 向：リリア失能→魔王养女→侵攻中反旗。王族血统主题在终局回收为「血是燃料也是反叛火种」。

**【主要人物】**
- **リリア・エーベルヴァイン（白之姫骑士）**｜性格：高傲刚直、蔑视ディルク、护国优先。｜装备·能力：白衣铠、骑士剑、圣性魂能、クローヴィス战役级武技≈三阶。｜人物弧光：姫骑士→被政变围困→魂能流失边缘→（OVA）魔王养女／反旗。｜立场：对契约者先审后用；软肋是レイラ与白骑士部下。
- **ディルク・エーベルヴァイン**｜性格：享乐、劣等感、惧期限。｜能力：王权与魔族契约，本人战力平庸。｜弧光：政变→调教计划→付清或崩溃。｜立场：可用其死线交易，不可信其承诺。
- **レイラ・エーベルヴァイン**｜性格：端庄「辺境の至宝」。｜能力：无战场位，政治象征。｜弧光：被袭→视线外调教→（OVA）魔王情妇。｜立场：リリア最大软肋。
- **キリコ**｜性格：冷艳参谋、魔族。｜能力：私兵、人脉、魔族体术。｜弧光：扶植ディルク→关系恶化→可反噬。｜立场：情报价最高，背刺率高。
- **サンド**｜性格：将官代收者。｜能力：跨阶威压。｜弧光：期限执行人。｜立场：情报优先，勿硬刚。
- **ミーア**｜性格：忠义。｜能力：白骑士精锐。｜弧光：伴随リリア至反旗。｜立场：可结盟护主。
- **フランク／シャムロック／ルッテン／メッツァー**｜四白骑士；战场护卫与终盘反旗战力。
- **ルーフェン国王**｜病倒或被刺；政变导火索。
- **魔王**｜超阶巅峰；养女化与征服；条件性胜利。

**【势力图谱】**
- **ルーフェン王室残党／白骑士团**：リリア、ミーア、フランク、シャムロック、ルッテン、メッツァー；宗旨护国与救出；战力三阶核心。
- **ディルク政权**：政变军＋全国动员；对价驱动，战力二～三阶人海。
- **黒の楽園館／キリコ派**：魔族参谋与私兵；可背刺ディルク。
- **魔族代收网**：サンド与魔王；终局规则制定者。
敌友：ディルク惧期限；キリコ可交易可反噬；白骑士只信リリア。

**【贵重物品】**
- **白衣铠与姫骑士剑**：リリア身份象征；破则士气崩。
- **圣性魂能／エナジー**：魔族对价硬通货；抽空＝人格崩坏。
- **魔族契约书与期限砂漏**：任务计时器。
- **炸弹式胁迫道具（OVA 动员令）**：全国凌辱动员的政治工具（只写组织后果）。
- **魔王养女印（OVA）**：终局身份改写；低阶不发本体。
每件写清：谁想要、抢到后果、当前三阶能否用。

**【隐藏剧情 · 伏笔】**
- ディルク对リリア的劣等感是政变心理根，不是单纯好色。
- キリコ既是参谋也是可凌辱／可反杀的变量；部分 END 她杀ディルク。
- OVA 终盘リリア失能后成魔王养女，又在侵攻途中反旗——「完全堕落」≠永久无意志。
- レイラ线常在リリア视线外推进，母女互知会触发意志崩溃。
- 游戏多结局依赖「计划性调教＋付清对价」；任务世界用「情报／人质／期限」映射，不写过程。

**【大事记时间线】**
クローヴィス战役リリア成名 → 国王病倒／被刺 → ディルク借魔族夺权 → 对价期限启动 → リリア／レイラ调教线并行 → 期限日结算 → 王国崩溃或魔王养女／反旗（OVA）。

**【叙事基调 · 雷区】**
中世纪黑暗奇幻＋政变绝望感；白之姫骑士的气骨是基调。忌把リリア写成无脑无敌；忌低阶硬刚魔王；忌战力归零削弱魔王；忌把ディルク写成单纯喜剧丑角而抹掉契约死线。最早切入：政变当夜或对价首次代收前夜。

**【王族血统·档案1】**
エーベルヴァイン家谱在王城图书馆有两套：公开谱系强调クローヴィス战功，密档记录圣性魂能检测值。契约者若偷出密档，可证明魔族定价算法，换白骑士信任或卖给邻国间谍——后者会引来キリコ灭口。

**【王族血统·档案2】**
ディルク童年丧母后被诸侯排挤，リリア越是完美越是刺。政变不是突然疯狂，而是劣等感＋キリコ灌输的「王子心構え」扭曲合体。写ディルク勿脸谱化成纯丑角。

**【王族血统·档案3】**
聖性魂能在战场上可短暂驱散低级魔触，但对サンド将官无效。三阶任务允许「用魂能脉冲打开地牢锁」类解谜，禁止「一刀斩魔王」。

**【王族血统·档案4】**
レイラ作为「辺境の至宝」本身是外交筹码；其被调教线常在侧殿进行，リリア若在期限前得知，意志检定暴跌——这是锚点「血统」的另一面：血亲即弱点。

**【王族血统·档案5】**
四白骑士各有防区：フランク守门、シャムロック巡墙、ルッテン后勤（OVA 重伤线）、メッツァー突击。契约者加入须选防区，不能同时出现在四条线。

**【王族血统·档案6】**
黒の楽園館账本记录各国贵族欠债，是キリコ私兵的财政。烧毁账本＝断私兵粮；交出账本＝换ディルク暂时庇护。

**【王族血统·档案7】**
对价期限在城钟与砂漏双重显示；每次代收サンド只问数字不问手段。任务成功条件可写成「压低单次抽取量」而非阻止代收本身。

**【王族血统·档案8】**
邻国密使在政变第三日抵达边境，想趁乱瓜分ルーフェン。リリア若仍具王族象征，可签发停战密诏——需要未被抽干的魂能签名。

**【王族血统·档案9】**
魔族契约原文有歧义：「付清」是总量还是日均。律师型契约者可钻空，但サンド用武力解释条文。

**【王族血统·档案10】**
完全版追加剧本扩展了部分侍女与侧殿事件；任务世界可作支线 NPC 池，不改变魔王／リリア终局骨架。

**【王族血统·档案11】**
OVA 主题歌「Stand up,Princess knight」定调：即使魂能见底，公主仍应站起。叙事上允许惨胜，不允许无代价无敌。

**【王族血统·档案12】**
王族纹章戒指可打开秘密军械库；ディルク已更换部分锁芯，需ミーア旧记忆或锻造师重配。

**【王族血统·档案13】**
「血统纯度」谣言被政变宣传机器利用：声称リリア非亲生以瓦解民心。澄清谣言是三阶非战斗任务。

**【王族血统·档案14】**
サンド的绷带下目光是将官级威压，凡人直视会腿软。契约者接触时必须隔着门缝或传声筒——规则杀示范。

**【王族血统·档案15】**
リリア对ディルク的蔑视是性格底色，不会因「他其实很可怜」而突然和解；任何感化线需长线代价。

**【姫骑士莉莉亚-王族血统·要证1】**
王城东翼纹章厅的玻璃柜里锁着クローヴィス战役勋章：铜质边缘刻着リリア的名字缩写。ディルク政变后派人用黑布罩住柜子，却不敢销毁——销毁会被诸侯解读为承认自己非战功继承人。契约者若当众揭布，等同于逼贵族站队。

**【姫骑士莉莉亚-王族血统·要证2】**
圣性魂能检测在旧王国用水晶天平，读数以「灯」为单位。リリア成年礼记录是十七灯，レイラ是九灯，ディルク本人未测或测出极低——档案被撕。补测需要宫廷医官与魔族代行人同时在场，后者会把读数记入对价账。

**【姫骑士莉莉亚-王族血统·要证3】**
ミーア保存着一枚备用王族蜡印，藏在靴跟。蜡印可签发「停战密诏」但只能用一次，印泥配方只有老书记官知道。书记官在政变夜躲进下水道，密码是クローヴィス旧军歌第二段。

**【姫骑士莉莉亚-王族血统·要证4】**
吊桥绞盘的齿轮被换成魔族合金，普通人转不动。白骑士ルッテン重伤前留下的血手印还在把手上，可作为「仍有抵抗」的物证拍照。

**【姫骑士莉莉亚-王族血统·要证5】**
邻国密使的临时候馆设在城外磨坊，暗号是三下两短的敲门。他们收购的是「血统证明原件」而非リリア本人——有证明就能扶植傀儡。

**【姫骑士莉莉亚-王族血统·要证6】**
サンド代收时只接受「灯」为单位的魂能结晶，不收金银。结晶在代收后会灰化，无法回炼。

**【姫骑士莉莉亚-王族血统·要证7】**
黒の楽園館二楼有各国欠条墙，キリコ用别针标「可召私兵」。拔掉某国别针等于该国私兵三日不动——昂贵的外交键。

**【姫骑士莉莉亚-王族血统·要证8】**
リリア的白衣铠内衬绣着亡父留言：「剑可折，名不可卖。」内衬被撕开会触发她最强的一次意志反弹。

**【姫骑士莉莉亚-王族血统·要证9】**
政变宣传单把リリア写成「假王女」，印刷厂在南市；烧毁铅字可止谣，但铅字本身可变卖换伤药。

**【姫骑士莉莉亚-王族血统·要证10】**
城钟与沙漏双计时：钟慢三分钟，沙漏准。懂行的人看沙漏，平民看钟——信息差可救人。

**【姫骑士莉莉亚-王族血统·要证11】**
军械库B区藏有对魔用的银矛，对低级魔触有效，对サンド无效。钥匙在纹章戒指与ミーア记忆双开。

**【姫骑士莉莉亚-王族血统·要证12】**
ディルク的寝殿装了隔音符，实为防止他听见城外哭声——心理弱点，可被声音类幻术利用。

**【姫骑士莉莉亚-王族血统·要证13】**
完全版追加的侍女线可作炮灰情报源：她们知道侧殿门禁时间表。

**【姫骑士莉莉亚-王族血统·要证14】**
王族血统主题收束：血是灯，灯是税，税是命——谁定价谁就是临时的王。

**【姫骑士莉莉亚-王族血统·要证15】**
三阶任务禁止「唤醒全国忠心」；最多唤醒一个防区的白骑士。

**【姫骑士莉莉亚-王族血统·要证16】**
クローヴィス旧战场导游图仍在卖，游客不知政变；可把游客线与政变线叠成掩护。

**【姫骑士莉莉亚-王族血统·要证17】**
魂能估值表的纸是魔族制，普通火烧不毁，需圣性残渣点燃。

**【姫骑士莉莉亚-王族血统·要证18】**
ミーア靴跟蜡印若被发现，ディルク会立刻搜捕白骑士——使用即暴露。

**【姫骑士莉莉亚-王族血统·要证19】**
诸侯联名信需要三枚真纹章，目前只剩リリア与一台空头印章。

**【姫骑士莉莉亚-王族血统·要证20】**
