# 怪物猎人XX（GU）

## 剧情

**【作品来源】**
卡普空（Capcom）开发发行的动作角色扮演狩猎游戏。日版名《モンスターハンターダブルクロス》（Monster Hunter XX），欧美名 *Monster Hunter Generations Ultimate*（MHGU）。本作是《怪物猎人X》（MHX / Generations）的完整扩展版（含G级），2017年3月18日任天堂3DS日版/繁中区发售，2017年8月25日任天堂Switch日版，2018年8月全球Switch版（MHGU）。制作人小嶋慎太郎，系列总监系辻本良三一脉。气质：历代猎场巡礼庆典、流派·狩技深度构筑、从村落日常到极位凶星与金色女王的残酷狩猎史诗。文风偏热血硬核、生态写实与「人与自然博弈」并存；禁止写成修仙升级爽文。平台跨3DS与Switch，存档互通在世界内可解作跨港猎人档案。

**【世界定位】**
以贝尔纳村（Bherna）与龙识船·飞空艇调查团（Wycademy / Soaratorium）为枢纽的「历代猎场大巡游」世界。契约者扮演一名受龙识船征召的猎人，在下位→上位→G级中逐步面对狂龙种（Deviant）、旗舰天彗龙（Valstrax，中文亦称天廻龙/银翼凶星）与最终王阁螳螂（Ahtal-Ka）及其机甲「阿特拉尔·涅塞特（Ahtal-Neset / 女王的玉座）」。主线不是灭世预言，而是：调查、讨伐、把失控的生态与「极个体」压回人类可存续的边界。

**【世界观 · 力量体系】**
本世界力量不来自灵气飞升，而来自：**武器锻造体系 + 防具技能珠 + 狩猎流派（Hunting Styles）+ 狩技（Hunter Arts）+ 随从猫（Palico / 猫猫猎人Prowler）+ 生态知识与团队协作**。死亡规则极硬：猎人倒地三次任务失败；被大型怪一击、连环冲撞、龙属性异常（龙属性异常状态 Dragonblight）可直接出局；古龙级与极位狂龙种没有「讲道理复活」。特殊系统：
- **十四种武器**：大剑、太刀、片手剑、双剑、大锤、狩猎笛、长枪、铳枪、斩击斧、充能斧、操虫棍、轻弩、重弩、弓。
- **六大狩猎流派**（XX新增勇气·炼金）：公会流（Guild，均衡，可装2狩技）、强袭流（Striker，强化狩技槽，可装3）、空中流（Aerial，借弹跳骑乘）、武士道流（Adept，极限回避/格挡反击）、**勇气流（Valor，拔刀蓄力与勇气槽爆发，仅1狩技）**、**炼金流（Alchemy，炼成支援道具，可装3狩技但冷却更长）**。
- **狩技**：攻击积蓄量槽后释放的大招（如大剑真蓄、太刀樱花气刃等），靠任务推进解锁。
- **任务位阶**：村任务 / 集会所下位 / 上位 / **G级**；另有**特殊许可任务（Special Permit）**专打狂龙种。
- **狂龙种（Deviant）**：外形近似原种，却拥有全新招式与称号的「极个体」，需特殊许可与票券；G级独占狂龙种更狠。
- **古龙种 / 古龙级**：不属常规生态位、足以改写地貌的灾难级存在。

战力阶梯（按破坏力表现，宁低勿高）：
1. **见习猎人 / 村1～3星**：能处理小怪群、青熊兽（Arzuros）、大凶豺龙（Great Maccao）等。≈乐园一阶。
2. **正式猎人 / 村4～6星·集会所下位**：稳定讨伐雄火龙（Rathalos）、雌火龙（Rathian）、轰龙（Tigrex）等同阶飞龙。≈一～二阶。
3. **熟练猎人 / 村高星·下位尾声**：可应对斗技场、多目标、部分特殊个体；团队协作成型。≈二～三阶。
4. **上位猎人**：面对上位强化种、部分亚种与初期狂龙种；一次失误可团灭。≈三～四阶。
5. **G级猎人**：G级飞龙、爆锤龙（Glavenus）系、海龙（Lagiacrus）回归线、G级狂龙种门槛。≈四～五阶。
6. **极位讨伐者**：天彗龙（Valstrax）、鏖魔角龙（Bloodbath Diablos）、金火龙/银火龙、荒厄龙（Alatreon）等古龙级；以及最终王**阁螳螂**与机甲。≈五～六阶。
7. **本世界不设「毁国飞升」以上**：顶点=阁螳螂（最终Boss）与天彗龙/鏖魔角龙等极位强敌；再往上无原作依据，禁止灌入七阶以上。

**乐园阶位映射**：村下位见习～正式≈一阶；下位熟练～上位入门≈二阶；上位中坚≈三阶；上位尾声～G级入门≈四阶；G级主力与G级狂龙种≈五阶；天彗龙 / 鏖魔角龙 / 阁螳螂等极位强敌≈六阶（地区灾难级，条件性胜利·情报优先）。宁低勿高；本世界覆盖仅到六阶。对照破坏力：一阶仅能处理村寨级兽害；三阶可端掉据点级飞龙巢；五阶一战可让街区级猎场暂时清空；六阶对应「军队级覆灭传说 / 飞空艇航线改写 / 废都级决战」。

**【地理 · 舞台】**
空间结构以「聚落枢纽 + 历代猎场 + XX新图」展开，AI写场景先定契约者落在哪一层：
- **贝尔纳村（Bherna）**：单人主线起点。设计灵感偏瑞士/提洛尔牧区——风与草原的牧羊村，畜养绒毛羊（Moofah），有玩家之家、铁匠与防具屋、市场、猫饭馆、随从猫牧场、流派师父道场、准备区与集会所入口。村任务看板娘（Bherna Gal）发村任务；村长管民心。低阶冲突=牧场被夺、采集配额、新人被同期坑。
- **龙识船 / 集会所（Wycademy Hunters Hub）与飞空艇调查团（Soaratorium）**：多人与高阶任务枢纽。龙人族首席研究员更新怪物图鉴；特殊许可、G级简报、天彗龙航线事故报告多从此出。飞空艇是村落间移动动脉，也被天彗龙「擦尾」风险绑定。
- **回归村落**：科科特村（Kokoto，森丘文化）、波克村（Pokke，雪山温泉）、结云村（Yukumo，溪流与和风）——承担村任务支线与「历代巡礼」氛围，证明本世界是系列庆典而非孤岛地图。
- **猎场分层**：入门区原生林（Jurassic Frontier，大凶豺龙与斩龙舞台）；森丘（Verdant Hills）、沼地（Marshlands）、雪山（Arctic Ridge）、火山、孤岛（Deserted Island）、溪流（Misty Peaks）；四世代遗迹平原（Ancestral Steppe）、原生林深处、冰海（Frozen Seaway）、地底火山（Volcanic Hollow）、圣地与塔；沙原（Dunes）与旧沙漠系；斗技场/水上斗技场等限时舞台。**XX追加关键图**：密林（Jungle）、沙漠（Desert）、**遗群岭（Ruined Pinnacle）**——高空遗迹与天彗龙强关联、**孤城废都（Forlorn Citadel）**——阁螳螂最终战唯一主舞台，金丝封墙、龙杭与弩炮可交互。
- 势力地盘：人类安全圈靠公会任务与猎人密度维持；圈外是完整食物链。低阶猎人若扬言单杀阁螳螂，本地人会当疯子——正确反应。

**【世界剧情线】**
① **开篇·贝尔纳村与龙识船征召**  
猎人抵达贝尔纳村，协助周边生态调查与村务讨伐。最初任务是采集、讨伐小怪、压制大凶豺龙、青熊兽、奇猿狐（Kecha Wacha）等「入门大型」。名场面：第一次在原生林被跳踢兽群围攻、第一次扛起大剑或太刀完成回村；猫饭与随从猫编队确立日常节奏。因果：猎人从「外人」变成村与龙识船可派遣的战力。此阶段世界状态=一阶叙事，强者如天——集会所墙上的古龙画像只是装饰。

② **历代巡礼·四村联防**  
任务线将猎人送往科科特、波克、结云等回归村，重走森丘、雪山、溪流等经典猎场。此阶段确立「本作为历代集大成」的格局：旧世代怪物与新世代怪物同图，生态冲突被写成调查课题而非单纯刷怪。名场面：在森丘再遇雄火龙、在雪山与轰龙对峙、在溪流听结云风铃与泡狐龙（Mizutsune）的水雾；波克温泉边听老猎人讲「雪山那头有比轰龙更吵的东西」。阶段结局：村任务高星开放，集会所下位全面铺开，猎人开始在多村之间建立名声。

③ **四天王与X线旗舰（继承自MHX）**  
主线压力抬升到X世代旗舰群：电龙（Astalos，带电翼膜与俯冲）、斩龙（Glavenus，刀尾蓄热）、巨兽（Gammoth，冰雪重压）、泡狐龙（Mizutsune，泡沫与滑动），以及千刃龙（Seregios）等锋刃系压力，并有黑蚀龙（Gore Magala）系阴影作背景。猎人学会骑乘、地形利用、公会/强袭/空中/武士道四流派切换。名场面：首次被斩龙刀尾刮出火花、首次在电龙蓄电时赌极限回避、首次在巨兽踏雪时用地形卡位。因果：猎人名声传到龙识船上层，开始接触「特殊许可」传闻。

④ **上位·狂龙种开闸**  
上位开放后，生态中出现**狂龙种**：红盔青熊兽、大雪主白兔兽、矛碎盾蟹、岩穿鬼蛙、紫毒姬雌火龙、黑炎王雄火龙、金雷公雷狼龙、荒钩爪轰龙、烬灭刃斩龙、独眼黑狼鸟等。它们不是换色亚种，而是「被猎人与环境反复逼出的极个体」，招式全套重做，必须走特殊许可。名场面：票券不足时在集会所被前辈嘲讽；第一次票券本被红盔一掌拍进土里；黑炎王在火山把陷阱连同猎人一起烤焦。阶段结局：猎人成为「可持特殊许可」的上位精锐，开始有人叫你「那谁——打过红盔的」。

⑤ **G级门槛·XX真正开幕**  
进入G级后，旧威胁全面强化，**XX独占内容**上线：勇气流/炼金流、G级狂龙种——铠裂镰蟹、胧隐夜鸟、天眼泡狐龙、青电主电龙、银岭巨兽，以及旗舰级狂龙种**鏖魔角龙**。沙漠与密林等旧图以高规格回归。名场面：第一次在沙漠听见鏖魔传说——「整支军队讨伐后只剩沙子」；第一次用勇气流收刀架势硬吃一记G级冲锋后反打。因果：龙识船将「银翼凶星」与「金色女王」的调查优先级拉到最高；公会开始筛选「能进终盘」的名单。

⑥ **天彗龙（Valstrax / 天廻龙）线**  
银白古龙自超高空巡航，胸腔进气、双翼喷吐龙属性推进焰，被目击时往往只剩一道红彗星尾迹，故称银翼凶星。它不主动屠城，却会因航线碰撞摧毁飞空艇；被激怒后用翼枪突刺、推进连打，并释放「环球一击」——冲天绕后以超音速彗星姿态俯冲，落地后有短暂硬直。主舞台偏向遗群岭及多地形巡回（冰海、沼地、密林等亦可能目击）。名场面：首次遭遇可不击破、可退回村庄——剧情允许「活着看见它」就算推进；真正讨伐时龙属性异常状态让全队武器属性归零，消散果实成为比回复药更紧俏的物资。阶段结局：猎人证明能在极速古龙下生存并回收素材，龙识船图鉴写入「天彗龙」。

⑦ **终局·阁螳螂与孤城废都**  
金色雌性甲虫种**阁螳螂**以金丝操纵废铁与遗迹残骸，把城镇与要塞踩碎后吞并进自己的「塘城/女王玉座」。最终战于**孤城废都**：本体用金丝背负钢梁投掷、释放降防信息素；危急时唤出巨型人偶机甲**阿特拉尔·涅塞特**——丝线木偶式踏地震击、吐丝束缚；机甲崩解后它会背负龙杭甚至三发连射，末段再以巨大车轮与金丝陷阱收场。公会空投弩炮/炮弹；猎人可攀上机甲关节破坏丝茧；龙杭可射使机甲失衡，但女王会用金丝封死设施。名场面：机甲倒塌、城墙丝线崩解、阳光重新照进废都。结局：阁螳螂被讨伐，女王玉座崩塌，聚落威胁解除；世界恢复「狩猎可续」的平衡，而非神话式飞升。猎人成为G级传说级讨伐者，但生态永不止息——下一头狂龙种仍在沙漠或密林某处。

⑧ **微观日常贯穿**  
除大讨伐外，正文高频事件：猫饭组合试验、素材搬运与仓库爆仓、武具强化等级（用同类素材灌经验升级）、随从猫技能编排、斗技场限时、集会所组队黑吃黑（抢斩尾/抢骑乘）、特殊许可票券肝度、飞空艇航线被天彗龙擦过导致物资延误、假护石骗局、牧场猫走失。AI写日常优先这些，而非一上来古龙大战。

⑨ **蝴蝶效应边界**  
契约者可改局部：提前警告某村迁徙、抢走非核心素材、让某次特殊许可由别人完成。不可无代价改写：阁螳螂终局战的生态逻辑、天彗龙的极速本质、狂龙种「特殊许可制度」。强行越级硬刚顶点=任务失败或真实死亡；顶点始终保持完整威胁，禁止用任何「战力归零」借口。

⑩ **战斗与生态细则（AI写战必读）**  
本世界战斗不是「放技能清屏」，而是部位破坏、斩尾、骑乘、陷阱、炸弹、属性相克与异常状态的组合拳。龙属性异常状态会抹掉武器属性，对天彗龙战几乎是必带消散果实/属耐的理由。狂龙种与古龙的吼叫常需高级耳塞；勇气流收刀架势可吸收伤害并转爆发，但空窗期被抓=秒杀。炼金流用时间换队伍续航，适合长期G级战。随从猫可作支援也可切换为猫猫猎人出战，但输出低于人类猎人。乱入机制真实存在：你打A时B可能进场，把任务从「可完成」变成「该撤退」。生态上，草食种（黄速龙、精灵鹿、绒毛羊等）是诱饵与素材源；鸟龙种与兽种是入门课；飞龙种与海龙种构成中坚；古龙种不进常规食物链。写正文时优先描写：足迹、擦伤树皮、地面震动、飞空艇警报——而不是突然Boss血条。

⑪ **狂龙种全谱系（X继承+XX追加）**  
X世代已确立的狂龙种包括但不限于：红盔青熊兽、大雪主白兔兽、矛碎盾蟹、岩穿鬼蛙、紫毒姬雌火龙、黑炎王雄火龙、金雷公雷狼龙、荒钩爪轰龙、烬灭刃斩龙、独眼黑狼鸟等——它们各自拥有称号与全新招式，特殊许可分级提升强度。XX在G级追加：铠裂镰蟹（Rustrazor Ceanataur，锈刃连斩）、胧隐夜鸟（Nightcloak Malfestio，雾隐干扰）、天眼泡狐龙（Soulseer Mizutsune，水泡与「天眼」压迫）、青电主电龙（Boltreaver Astalos，极端带电机动）、银岭巨兽（Elderfrost Gammoth，冰原重压），以及被视作狂龙种之首级的**鏖魔角龙**。解锁鏖魔通常要求完成一系列G级狂龙前置与最终Boss相关条件（公会制度锁），防止新人直接送死。每只狂龙种都是一条可独立成卷的「许可线」：刷票→适应招式→部位破坏→最终许可。

⑫ **旗舰与终局战再述**  
双旗舰结构：**天彗龙**负责「速度与龙属性」的垂直恐惧，**鏖魔角龙**负责「陆战冲锋与军队覆灭传说」的水平恐惧；**阁螳螂**则把恐惧升级为「文明残骸被怪物征用」。天彗龙胸腔进气口被集火可引发内部龙能量爆喷而倒地——这是情报战的奖励，不是削弱设定。阁螳螂战分阶段：本体金丝钢梁→唤出涅塞特机甲→机甲崩解后龙杭背负→再起机甲与车轮末段；两套BGM（女王盘踞废城 / 蠕动的废城）随形态切换。公会空投弩炮弹、龙杭可射机甲使之失衡；猎人可在机甲踏地卡住时攀爬破坏丝茧。这些机制必须写成「可执行战术」，禁止写成主角光环秒杀。

⑬ **与MHX的继承关系（防串台）**  
本档案以**XX/GU**为准：含G级、勇气流、炼金流、天彗龙、阁螳螂、鏖魔角龙与XX新图。若剧情停在「仅四天王与X狂龙种、无G级」，那是MHX而非本世界。契约者从X存档继承经验在设定上可解释为「同一猎人的G级续章」，但世界状态必须以XX终局为终点。

**【势力图谱】**
- **猎人公会**：签发任务等级、报酬与讨伐伦理（禁止无意义屠杀草食种、约束特殊许可发放）。战力从见习到G级严格分层；越级接单会被劝退或默许你去死。代表接口=各村看板与集会所柜台。

- **贝尔纳村及回归三村（科科特/波克/结云）**：民生据点与村任务池。要的是安全、声誉与牧场/温泉/和风产业不被毁。村长与看板娘是本地权威；外来猎人有用则捧，惹祸则逐。
- **集会所猎人共同体**：松散利益联盟。组队狩猎、黑吃黑抢素材、分享狂龙种情报均在此发生。没有统一领袖，只有实绩与装备说话。

- **无「魔王军团」式反派势力**：对立面是失控生态、狂龙种与古龙级。敌友逻辑=生存与报酬，不是善恶阵营。契约者若组建「屠龙邪教」会被公会与调查团双向打压。

**【隐藏剧情 · 伏笔】**
- 天彗龙并非「为毁灭人类而生」，而是高空生态位古龙；与飞空艇冲突是航线悲剧，也是龙识船坚持调查的理由。它多数时间远离人烟，被惹怒才展现「环球俯冲」级杀伤。
- 狂龙种制度暗示：猎人活动本身在「塑造」更强的个体——狩猎越兴盛，极个体传闻越多。特殊许可既是保护猎人，也是公会给自己留的刹车。
- 阁螳螂的塘城是「人类工业残骸的镜像」：它用人类武器与废铁反杀文明边界；末段车轮被广泛解读为「人类最初发明被女王盗用」。金丝金茧中沉睡的「各地夺来的财宝」是支线寻宝钩子。
- 勇气流/炼金流标志猎人技术在G级再进化，与X四流派合成完整六流派；这意味着「同一把太刀」在不同流派下是完全不同的战斗哲学。
- 与前作X共享世界底盘；XX是同舞台G级终章，不是平行灭世。黑蚀龙等旧阴影可作背景压力，但本世界终盘主角是天彗龙—鏖魔—阁螳螂三角。
- 双旗舰「无双双璧」叙事：天彗龙与鏖魔角龙分别占据空与陆的极致，阁螳螂则占据「智械/丝线战争」的第三极。
- 结局后：废都见光、航线暂安，沙漠仍有鏖魔传说、天上仍可能划过红彗星——循环式生态终局，非一劳永逸。契约者若以为「通关=世界永远和平」会遭到下一次乱入打脸。

## 阶位切入点

> 阶位↔原作强度：一阶≈村下位见习～正式猎人；二阶≈下位熟练～上位入门；三阶≈上位中坚；四阶≈上位尾声～G级入门；五阶≈G级主力/G级狂龙种；六阶≈天彗龙·鏖魔角龙·阁螳螂等极位强敌。世界顶点（阁螳螂及极位古龙/狂龙）为跨阶灾难级，低阶切入一律「规避顶点·情报优先」。宁低勿高。本世界仅覆盖：一阶、二阶、三阶、四阶、五阶、六阶。

**一阶（村下位见习 · 贝尔纳起步）**
切入身份/时点：契约者以「刚被龙识船登记的新人猎人」身份切入，锚定抵达贝尔纳村、村1～3星任务前后。
初始事件：看板娘派你去原生林驱赶骚扰牧场的大凶豺龙；出发前流派师父逼你选定初始流派与武器，铁匠只肯给你村级武器胚。
开场白建议：「风穿过贝尔纳的草场，绒毛羊叫得人心发毛。看板娘把一张薄薄的任务书拍在你胸口——『先活着回来，素材的事回来再说。』远处原生林里传来跳踢与嘶鸣，你握紧还没开刃的武器。」
关键NPC立场：**贝尔纳看板娘**——发活、验货、看你是否靠谱；**流派师父**——只教不护，打不好就骂；**猫娘卡塔琳娜**——塞给你第一只随从猫；**猎人主角（若并行）**——同期竞争采集点与斩尾；**铁匠**——素材不够免谈强化。
主线钩子/支线：完成村低星建立信誉；支线＝猫饭食材采购、牧场猫雇佣、森丘/沼地采集跑图。
危险度/规避：中——死于大凶豺龙连踢、青熊兽抱摔、迷路耗尽回复药；须规避上位怪、任何狂龙种与古龙。
任务方向/奖励：活着交任务、攒第一套整装；奖励取向＝村级武器防具、入门狩技、随从猫基础技能。

**二阶（下位熟练 · 四村巡礼）**
切入身份/时点：以「可跨村派遣的正式猎人」切入，锚定科科特/波克/结云巡礼、集会所下位中段。
初始事件：飞空艇把你送到科科特，村任务要求讨伐出没森丘的雄火龙；同期猎人组队时有人提议「你断尾我主攻」——分配不公一触即发。

关键NPC立场：**贝尔纳村长**——远程要你顾及村誉；**首席研究员**——开始要生态观察笔记；**同期猎人**——可结盟可抢功；**猫饭馆女主人**——用高级猫饭换你珍奇食材；**看板娘**——提高星级门槛。
主线钩子/支线：雄火龙/轰龙等下位旗舰级讨伐；支线＝回归村声望、斗技场限时、随从猫武器打造。
危险度/规避：中高——死于火龙空袭、轰龙直线冲锋；须规避上位强化个体与特殊许可怪。
任务方向/奖励：下位全装成型、解锁更多狩技；奖励取向＝下位稀有素材、技能珠、武器升级经验。

**三阶（上位中坚 · 狂龙种门槛）**
切入身份/时点：以「上位登记猎人」切入，锚定上位开放、首次特殊许可说明会前后。
初始事件：集会所公布红盔青熊兽特殊许可；你只有一张体验票，前辈警告「红盔不是大号青熊」——是否现在就申票，成为站队与否的抉择。
开场白建议：「上位的任务板上多了血红的边框。有人把『特殊许可』四个字念得像诅咒。你摸了摸腰间仅有的票券，听见酒馆里有人笑：『拿去喂红盔正好。』」
关键NPC立场：**特殊许可管理员**——按规矩放人；**流派师父**——逼你为狂龙种改配装与流派；**首席研究员**——要极个体行为记录；**同期猎人**——组队或骗票；**猎人主角**——可能已先你一步讨伐某只狂龙种。
主线钩子/支线：红盔/大雪主等初期狂龙种；支线＝上位多目标任务、护石赌博式强化、集会所声望。
危险度/规避：高——死于狂龙种新招与乱入；须规避G级怪、鏖魔、天彗龙、阁螳螂。
任务方向/奖励：拿下第一套狂龙装部件；奖励取向＝狂龙票、上位稀有、进阶狩技。

**四阶（上位尾声～G级入门 · 勇气与炼金）**
切入身份/时点：以「冲G级的精锐猎人」切入，锚定G级开放说明、勇气流/炼金流教学任务前后。
初始事件：流派师父把勇气流拔刀架势拍进你身体记忆；同时G级首单要求你在沙漠或密林证明生存能力——物资车被未知个体惊走，必须选择追车还是保人。
开场白建议：「G级的门槛不是门，是沙漠夜里的角影。师父说：『勇气流不是更凶，是更敢在收刀的瞬间把命赌对。』你的狩技槽空得吓人，可任务已经开始倒计时。」
关键NPC立场：**流派师父**——验收勇气/炼金实战；**首席研究员**——下发G级调查简报；**飞空艇乘务猫**——透露「天上有道红线」的传闻；**铁匠**——开口要G级素材才肯开新树；**特殊许可管理员**——预告G级狂龙种锁。
主线钩子/支线：G级入门飞龙、海龙回归线；支线＝勇气槽/炼金罐熟练、G级防具基底。
危险度/规避：高——死于G级数值与新机制；须规避天彗龙本体战、阁螳螂、鏖魔最终许可。
任务方向/奖励：G级立足、六流派自由切换；奖励取向＝G级武器胚、技能搭完善、高级猫配置。

**五阶（G级主力 · 狂龙种全开）**
切入身份/时点：以「G级主力讨伐者」切入，锚定G级狂龙种（银岭巨兽/青电主电龙/天眼泡狐龙/胧隐夜鸟/铠裂镰蟹等）开放中段。
初始事件：沙漠出现「整支讨伐队失踪」的报告，管理员把鏖魔角龙的许可条件摊开——你必须先清一串G级狂龙种前置，才能碰那张血红的最终许可。
开场白建议：「票券在桌上排成一列，每一张背后都是一具狂龙种的尸体。管理员指着最末那张：『想碰鏖魔，先证明你不是沙子。』窗外沙暴像有什么在远方转头。」
关键NPC立场：**特殊许可管理员**——冷酷卡关；**首席研究员**——要鏖魔与其他G级狂龙的对照样本；**同期猎人**——组满四人还是被坑成三人；**猎人主角**——可能与你争夺「首通」名誉；**铁匠**——用G级狂龙装图纸钓你反复出猎。
主线钩子/支线：G级狂龙种连锁讨伐；支线＝极限解禁装备、斗技场G级、素材市价操纵。
危险度/规避：极高——死于鏖战蒸汽爆炸、青电主连闪、天眼水激光等；须规避阁螳螂最终战，天彗龙仅可在充分情报下挑战。
任务方向/奖励：G级狂龙全装、解锁鏖魔前置；奖励取向＝顶配护石技能、终盘武器强化。

**六阶（极位强敌 · 天彗龙与阁螳螂）**
切入身份/时点：以「龙识船特准极位猎人」切入，锚定天彗龙正式讨伐令发布后，或孤城废都最终作战前夜。
初始事件：飞空艇在遗群岭上空被银翼擦尾，首席研究员把你推上甲板——要么立刻组织驱离/讨伐天彗龙，要么目送航线永久改道；若你已完成鏖魔与诸狂龙，最终令将指向孤城废都的金色女王。
开场白建议：「天上那道红线突然变粗。有人喊天彗龙，有人喊银翼凶星，有人只是在发抖。研究员的声音被风撕碎：『带够消散果实——龙属性异常会吃掉你的全部属性。』更远的废都方向，金丝在月光下像在缝合一座死城。」
关键NPC立场：**首席研究员**——要存活数据胜过要你的命；**飞空艇乘务猫**——恐惧但给你最后补给；**特殊许可管理员**——确认你已具备终局资格；**猎人主角**——终局队可能的队长或对手；**阁螳螂/天彗龙**——不可对话的顶点。
主线钩子/支线：天彗龙讨伐、鏖魔角龙最终许可、孤城废都阁螳螂决战；支线＝龙杭操作、机甲攀爬破坏、战中空投弹药护送。
危险度/规避：贴近顶点——**超阶边缘的灾难级：存在·情报优先/条件性胜利**；顶点保持完整战力，只许条件性胜利。单人无准备硬刚=死亡。
任务方向/奖励：终局素材与传说地位；奖励取向＝天彗龙/阁螳螂最终武具、世界内声望顶格；不发放超出本世界逻辑的创世权柄。
