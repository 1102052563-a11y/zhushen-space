## 剧情

**【作品来源】**
《机动战士高达ZZ》（Mobile Suit Gundam ZZ，日文：機動戦士ガンダムΖΖ）是日本 Sunrise 制作、富野由悠季总监督的电视动画，1986年3月1日至1987年1月31日播出，全47话，为《机动战士Z高达》的直接续作，同属宇宙世纪（Universal Century）正史主线。故事紧接格利布斯战役之后，时间锚定 U.C.0088 至 U.C.0089 年1月，核心冲突为第一次新吉翁战争（亦称哈曼之乱）。文风前半偏轻快的殖民地底层少年群像，中后段急转严肃战争与新人类悲剧；与《逆袭的夏亚》《机动战士高达UC》等互文，但本档案只写 ZZ 本篇正典因果。

**【世界定位】**
宇宙世纪地球圈：联邦军力空虚，奥古残存舰队阿伽玛号漂泊，阿克西斯残党在哈曼·卡恩摄政下改称新吉翁并发动侵攻。主角是 Side 1 香格里拉殖民地的废品回收少年杰特·亚西塔，从偷机体卖钱被卷进战争，到驾驶 MSZ-010 ZZ 高达成长为足以与哈曼正面对决的新人类。乐园侧本条目仅开放四阶切入：精英机动战士驾驶员、新人类侧翼任务、殖民地据点战与舰队战外围，不开「殖民地坠落本体」与「阿克西斯终战顶点」的无条件碾压位。

**【世界观 · 力量体系】**
力量本源不是灵气修仙，而是工业文明战争体系叠加「新人类（Newtype）」精神感应。凡人靠训练、机体性能与后勤；新人类能预知敌意、共振友机、驱动精神感应兵器（浮游炮/漏斗、精神感应框架、生物传感器）。强化人（Cyber-Newtype）以药物与手术强行制造感应能力，代价是人格崩解与被当消耗品。死亡规则极硬：机体被击破即人死；克隆体（璞露系列）可再量产，但「那一个」不可复活。米诺夫斯基粒子使雷达失效，近距目视与新人类感应成为决胜关键。

战力阶梯（按破坏力表现对齐乐园图鉴，宁低勿高）：
1. **殖民地平民、废品回收少年、联邦新兵**：徒手/轻火器，踹门级。≈一阶。
2. **量产机驾驶员（吉姆 III、卡萨 C 等）**：单挑小队、掀翻装甲车、毁仓库。≈二阶。
3. **精锐 MS 驾驶员、奥古老兵、新吉翁尉官机（德莱森、扎古 III 等）**：端掉据点、裂开港湾设施。≈三阶。
4. **精英新人类+高阶机体（Z 高达、高达 Mk-II、百式、强化 ZZ、卡碧尼、精神感应高达 Mk-II 外围表现）**：摧毁建筑群/舰桥模块、舰队战中改变局部战局；一城级威胁的「人+机」组合。≈四阶（本条目覆盖）。
5. **顶级精神感应兵器全开、殖民地坠落级战略打击**：街区至城市覆灭，非个人肉身可扛。≈五阶事件尺度（本世界不开放五阶切入，仅作背景灾厄）。
6. **舰队核武/殖民地投掷/阿克西斯级天体动能**：地区至行星表面灾难。≈六阶及以上战略层，契约者不可当「个人战力」对撞。

乐园阶位映射：平民与拾荒≈一阶；量产机≈二阶；精锐 MS≈三阶；精英新人类高阶机体≈四阶；战略兵器与殖民地坠落为五阶起事件尺度；阿克西斯动能撞击威胁为六阶级灾难。宁低勿高。世界顶点（哈曼·卡恩驾驶卡碧尼的巅峰决战态、昆曼沙全开、殖民地坠落）对四阶为**超阶巅峰：条件性胜利**——靠情报、联合作战、诱导内战、生物传感器共振窗口与舰队火力协同取胜，严禁写成「顶点战力归零式崩坏」。

**【地理 · 舞台】**
- **Side 1 香格里拉殖民地**：开篇舞台。杰特·亚西塔、邱尼·亚西塔与比查、蒙德、艾露、伊诺等拾荒少年的家。阿伽玛号入港补给修理，卡缪·维丹精神受创后在此疗养。底层管道、废品场、黑市零件摊、港区弹射轨道是四阶前夜的日常冲突场；港湾被卡萨突袭时，平民疏散与 MS 起降抢同一条廊道。
- **阿伽玛号（Argama）**：奥古残存旗舰，舰长布莱德·诺亚。中前期母舰与移动基地，舰内同时住着伤员、少年新兵与昂贵高达；后在地球侧移交，船员转乘纳赫尔·阿伽玛。
- **纳赫尔·阿伽玛（Nahel Argama）**：后期奥古/联邦侧新旗舰，高达小队以强行征用姿态保住继续作战的权利，象征「军队编制」与「战争意志」的分裂。
- **阿克西斯小行星与 Core 3 殖民地**：新吉翁权力中枢。哈曼·卡恩摄政所在；密涅瓦·劳·扎比名义王座；格雷米·托托叛乱爆发地；终战主战场。内部有克隆培育舱、浮游炮库与礼仪用的「幼主走廊」。
- **月面城市格拉纳达、月球背面航路**：补给与伏击走廊，安纳海姆利益与吉翁系旧网交织。
- **地球·非洲战线 / 达卡尔**：新吉翁地球侵攻与联邦议会争夺；第二次达卡尔战中新吉翁一度压制联邦政治中枢；沙漠据点与水源比光束枪更致命。
- **爱尔兰·都柏林**：U.C.0088.10.31 殖民地坠落目标城市，平民屠城级灾难现场；疏散半径与坠落预测椭圆是任务地图本身。
- **Side 3（旧吉翁故乡圈）**：战后被联邦割让给新吉翁的政治筹码，终战与投降仪式相关。
- **木星船团航路**：终局后杰特与露·露卡的离场方向，象征离开地球圈权力游戏；登船券比军功更稀缺。

地图层级：殖民地底层巷战 → 单舰 MS 出击 → 地球大气层内 MS 战 → 殖民地/阿克西斯总攻。AI 写场景须先定契约者在哪一层：底层是零件债与黑帮；中层是奥古征召与新吉翁扫荡；高层是舰队决战与摄政政治。四阶默认活动在「中层到高层外围」，王座与投掷按钮不在个人任务栏。

**【世界剧情线】**
① **前史·格利布斯余烬（U.C.0088 初）**  
格利布斯战役以提坦斯覆灭收场，奥古亦付出近乎自毁的代价：干部折损、舰艇残破、卡缪·维丹因过度使用新人类能力与精神感应战场而精神崩溃，花·园丽留守护理。布莱德·诺亚拒绝让阿伽玛号在联邦官僚体系里被拆分报销，率残舰转入 Side 1 香格里拉殖民地补给修理，名义上是「休整」，实质上是在联邦与新吉翁夹缝中找下一战的口气。阿克西斯方面，摄政哈曼·卡恩以幼主密涅瓦·劳·扎比为旗帜，整编一年战争与格利布斯后的吉翁系残党、部分提坦斯散兵与殖民地技术官僚，将势力公开改称新吉翁，目标从「流亡自保」升级为「支配地球圈」。马修马·赛罗等狂热军官被派往追击阿伽玛，把奥古最后火种当作必须掐灭的象征。因果锁：地球圈无主控强权，权力真空由哈曼以「秩序」名义填满；联邦议会则准备用割地与沉默换取自己的座位。

② **香格里拉·拾荒少年入局**  
杰特·亚西塔为给妹妹邱尼·亚西塔攒学费，与朋友比查·欧雷格、蒙德·阿加克、艾露·维亚诺、伊诺·阿巴夫以回收废品、倒卖零件为生。殖民地底层的「合法」是联邦配额与黑市差价，少年们的道德底线是「别害邱尼」。提坦斯残党亚赞·盖布尔看中他们的胆大与本地地形熟，怂恿偷走停泊中的 Z 高达变卖——在亚赞眼里这是报复奥古，在少年眼里这是一夜翻身。少年们多次潜入阿伽玛，与布莱德、花·园丽、技术班冲突：闹剧式追逐、禁闭、再潜入，构成前几话的喜剧节奏。新吉翁卡萨编队来袭时，喜剧被炮火撕开，少年们被迫联手；杰特在极限中展现强烈新人类资质，布莱德以舰长权限将其编入候补驾驶员，同时把比查等人一并绑上舰队责任。露·露卡以志愿者身份加入，作战风格直率狠厉，迅速成为高达小队不可缺的一角。阶段名场面：偷高达的巷战、第一次真正出击、Z 高达/高达 Mk-II/百式由新人接力。阶段结局：香格里拉不再是避风港，少年团随阿伽玛离港，正式卷入第一次新吉翁战争。

③ **ZZ 诞生、航路追击与阿克西斯潜入**  
安纳海姆电子与奥古体系交付可变三分离机体 MSZ-010 ZZ 高达：性能夸张、维护吃人、分离合体战术要求驾驶员与僚机高度信任。杰特从「吵闹的新兵」被推上旗舰驾驶员席，布莱德用军纪压他的嘴，用出击机会喂他的成长。新吉翁持续追击阿伽玛；途中插入月月殖民地等支线——封闭社群、奇异信仰与战争逻辑碰撞，提示地球圈并非只有联邦/吉翁二元。邱尼·亚西塔被战火卷走并辗转至阿克西斯相关势力控制，杰特的「赚钱」动机瞬间变成「救妹」刚需。他潜入阿克西斯，第一次与哈曼·卡恩正面对话：哈曼以摄政者的冷傲打量这名底层少年，察觉其新人类强度甚至让她联想到可被收编的「下一代」，试图劝诱；杰特拒绝，因其感应到哈曼秩序背后的怨与压。同潜入中结识被当作兵器培养的少女艾露比·璞露：璞露对杰特产生强烈依恋，称呼与行动都像把「哥哥」当成唯一锚，后倒向奥古侧随舰。阶段因果：个人救妹与反战觉悟开始重叠；哈曼与杰特的「招募—拒绝—再遇」成为终局镜像；璞露线埋下克隆伦理炸弹。

④ **地球降下·达卡尔与非洲战线**  
新吉翁发动地球侵攻，占领与威慑并举：地面据点、宣传战与 MS 重力下作战同步推进。阿伽玛与卡拉巴系残存力量在非洲等地联合作战，遭遇沙漠补给地狱、沙尘中的米诺夫斯基干扰、以及吉翁残党「蓝队」等局部势力的游击与交易。第二次达卡尔战中，新吉翁一度控制联邦议会象征空间，镜头对照议会厅的西装与城外燃烧的街道，把「合法政权」的软弱与交易嘴脸钉死。邱尼线在地球侧反复牵动杰特：失踪、误报死亡、再重逢的消息在舰内广播与士兵闲话中传播，布莱德既要利用杰特的战斗欲，又要防止他为妹妹擅自离队。阶段名场面：重力下 MS 关节负荷与热管理、断链补给的强制迫降、平民疏散与政治演说的割裂、杰特与哈曼在地球侧的再次交锋（劝诱再失败）。阶段结局：新吉翁在政治上得分，奥古在道义上愤怒却缺席「能阻止屠城的权力」。

⑤ **都柏林殖民地坠落（U.C.0088.10.31）**  
新吉翁实施殖民地坠落，目标爱尔兰都柏林，以屠城级平民伤亡恐吓联邦并证明「不听话就抹城」。联邦部分高官默许牺牲平民换取政治缓冲与停火谈判筹码——这是 ZZ 最冷的一刀：敌人投掷，己方签字。奥古/卡拉巴尽力疏散；早乙女隼人（小林隼人）在行动中牺牲，把一代高达延续下来的「护平民」火种烧在爱尔兰的天空。艾露比·璞露为保护杰特，对抗格雷米侧派出的克隆体璞露兹所驾精神感应高达 Mk-II，最终牺牲；杰特在死者怨念与新人类共鸣中精神重创，却也因此彻底觉悟：战争不是香格里拉式的零件生意，哈曼的「秩序」建立在他人尸体上。事后联邦将 Side 3 等利益割让给新吉翁以换撤军，奥古怒火与无力感并存。此卷是卡片锚点相关的良心崩坏点，也是四阶任务最重的「救人/疏散/取证/录音议员」舞台——你改得了疏散半径，改不了投掷按钮在更高权力手里。

⑥ **纳赫尔·阿伽玛与格雷米叛乱**  
地球阶段后，阿伽玛号相关编制调整，船员最终以纳赫尔·阿伽玛为新旗舰继续宇宙作战；高达小队甚至以「劫持/强行征用」的姿态保住继续打哈曼的资格——布莱德的军纪与少年们的任性在此再次对撞，却共同指向「不能把战争交给只会割地的联邦」。年轻军官格雷米·托托（作品暗示其与扎比血脉/克隆计划相关）拥兵自重，以「真正复兴扎比」为名发动叛乱，控制阿克西斯部分区域与 Core 3 相关设施，动用璞露克隆部队、量产卡碧尼系与超重精神感应机昆曼沙（Queen Mansa）。新吉翁分裂为哈曼派（马修马·赛罗、卡拉·苏等）与格雷米派，三方混战（含奥古）使新吉翁战力自毁。露·露卡在混战中以果断狙击击毙格雷米，斩断叛军指挥核。璞露兹在杰特与「原璞露」残响影响下动摇，从杀戮程序短暂回到「人」，终至悲剧收场。阶段意义：顶点哈曼并非铁板一块；内战是条件性胜利的正典路径——不是哈曼变弱，是她的帝国先被自己人撕开。

⑦ **阿克西斯终战与哈曼之死（至 U.C.0089.1.17）**  
联邦/奥古向阿克西斯与 Core 3 发动总攻，战场同时存在哈曼派残存、格雷米派余烬与奥古矛头。杰特驾驶强化/全装甲系 ZZ，生物传感器在极限共鸣下爆发，光束军刀级输出切开战舰结构的名场面属于此战语境：那是新人类意志与机体设计的共振，不是无代价外挂。哈曼驾驶 AMX-004 卡碧尼与杰特一对一决战：浮游炮网与 ZZ 的火力、感应与诱导缠斗，双方机体互创。哈曼拒绝杰特伸来的手与「一起离开/别再这样」的可能，选择以机体撞击阿克西斯外壁自尽；临终承认遇见杰特这样的孩子本身即是某种救赎——她看到的或许是自己若未被权力与孤独吞没时的镜像。密涅瓦「真身」早被调离、阿克西斯上者为替身等政治谎言浮出，哈曼苦心经营的「扎比正统」在死后露出空壳。新吉翁投降，第一次新吉翁战争于 U.C.0089.1.17 前后实质结束。

⑧ **终局之后与蝴蝶边界**  
奥古与卡拉巴实质解散或并入联邦再建部队（为日后隆德·贝尔埋线）。杰特与邱尼重逢，与露·露卡一同加入木星船团离开地球圈——这是对「留下来就会被权力驯化」的主动拒绝。卡缪在花·园丽照料下缓慢恢复，曾以新人类方式间接声援决战，证明格利布斯的创伤并未使感应完全熄灭。地球圈迎来短暂「和平」，却为 U.C.0093 夏亚的逆袭与后续袖章党冲突留白。契约者可改局部：多救一船都柏林平民、保下某驾驶员、提前暴露格雷米的克隆舱、偷走议员默许屠城的录音。不可无代价抹掉「哈曼之死/新吉翁投降/杰特离地球圈」的核心收束；改写局部会以联邦更脏的交易、克隆体追杀、新军阀补位或另一场投掷回流。

⑨ **日常可介入层（贯穿）**

最早切入锚点：阿伽玛刚入香格里拉、少年们第一次密谋偷 Z 高达的当夜；或都柏林坠落前 72 小时的疏散动员。

## 阶位切入点

阶位↔战力：一阶≈殖民地平民/拾荒；二阶≈量产 MS 驾驶员；三阶≈精锐 MS；四阶≈精英新人类+高阶机体（ZZ/Z/卡碧尼外围至决战窗口）。世界顶点为哈曼·卡恩（卡碧尼巅峰）与战略层殖民地坠落/阿克西斯动能威胁——**超阶巅峰：条件性胜利，情报与协同优先**，严禁「战力归零式削弱」写法。本世界仅开放**四阶**。

**四阶（精英新人类/高阶 MS 驾驶员 · 第一次新吉翁战争侧翼至决战窗口）**

切入身份/时点：契约者以奥古志愿驾驶员、安纳海姆随舰技师、香格里拉黑市「特级零件掮客」被征召者，或新吉翁逃兵中的精锐驾驶员身份切入；时点锚定 U.C.0088 阿伽玛停靠香格里拉之后、至都柏林坠落前后或格雷米叛乱前夜（三选一，开局声明）。禁止开局即站在哈曼王座前。

初始事件：阿伽玛号在香格里拉港区遭遇马修马·赛罗部卡萨编队突袭，港务封锁，少年杰特·亚西塔一伙正抬着偷来的驱动部件往废品巷跑。契约者被布莱德·诺亚临时点名：「会开就上，不会开就去修弹射器。」同时邱尼·亚西塔在疏散人流中失踪的消息通过甲板广播传出——你必须在「服从军令出击」与「帮杰特找妹」之间做第一次可见选择，并在 30 分钟内面对一架德莱森级压迫。

开场白建议：「米诺夫斯基粒子把雷达涂成雪花，香格里拉的人造天空却仍亮得刺眼。阿伽玛的甲板在震动，Z 高达的眼睛一开一合，像某种昂贵的动物刚醒。你听见布莱德的吼声、杰特的骂街，以及更远处卡萨推进器撕裂港湾的尖啸。零件债、军令状和妹妹的名字缠在同一根通讯链上——这不是演习，是 U.C.0088 年第一次有人把你的命标成弹药。」

关键NPC立场：
- **杰特·亚西塔**：先防后用；你若帮他护邱尼或肯一起挨布莱德骂，好感暴涨；抢他机体=死敌。
- **布莱德·诺亚**：把你当人力资源；守纪律可信任，玩失踪就关禁闭。
- **露·露卡**：务实同盟；战场上救过她一次就能换情报。
- **哈曼·卡恩**：远程观察新人类苗子；招募失败即清除；正面硬刚=条件性胜利课题，非开局内容。
- **艾露比·璞露**：若在阿克西斯线相遇，会迅速把你划进「杰特的人/敌人」二元。
- **格雷米·托托**：视你为可收编的强化人素材或内战炮灰；其数据舱是高价值目标。
- **比查·欧雷格 / 艾露·维亚诺**：香格里拉人脉入口，可提供黑市与舰内八卦，也会在分赃时扯皮。

主线钩子/支线：
- 主线：护送阿伽玛脱离香格里拉封锁 → 参与地球侧补给与达卡尔情报战 → 都柏林坠落前 72 小时平民疏散与「阻止/延缓投掷链」 → 格雷米叛乱中选择站队或两边卖情报 → 阿克西斯总攻外围撕开缺口，为杰特 vs 哈曼创造窗口（条件性胜利）。
- 支线A：追查亚赞·盖布尔残留的提坦斯军火窖，换取 ZZ 备用驱动。
- 支线B：偷录联邦议员默许都柏林牺牲的通讯，作为战后政治武器。
- 支线C：保护或转移一名璞露克隆体样本，道德与战力双重风险。

危险度/规避：危险度高，贴近顶点。必须规避：单独拦截殖民地坠落本体；无舰队支援挑战哈曼·卡恩卡碧尼全开；在米诺夫斯基高浓度区关闭通信硬闯阿克西斯核心。**超阶巅峰：哈曼·卡恩与战略投掷——存在·情报优先/条件性胜利**（借格雷米内战、生物传感器窗口、露·露卡斩首格雷米后的权力真空）。昆曼沙与精神感应高达 Mk-II 视为战役级灾厄，优先解体驾驶舱控制链而非比拼输出。

任务方向/奖励：自然目标=保全阿伽玛战斗力、救出邱尼或等价人质、取得叛乱情报、在终战外围击破哈曼直属浮游炮编队之一。奖励取向：四阶匹配的机体权限（Z/Mk-II/次级可变机时限驾照）、安纳海姆维修协议、新人类感应训练舱数据、联邦/吉翁双系军功票、黑市硬通货与「木星船团登船券」线索。禁止发放可无代价引爆殖民地的战略按钮作为「个人宝物」。
