## 剧情

**【作品来源】**
《爱上火车 Last Run!!》（日文：まいてつ Last Run!!）由 Lose 开发，2020年10月30日发售的视觉小说，是2016年《爱上火车》的扩充再发售版：收录原版全部路线，并新增总集后的「仲国」篇与更多铁路人偶戏份。原画 cura、剧本进行豹；气质为铁道乡愁 + 小镇复兴 + 人偶温情恋爱，带成人内容（PC18禁），亦有全年龄 Origin 等移植。系列另有动画《铁路浪漫谭》旁支，但本档案以 Last Run 游戏主线为准。

**【世界定位】**
架空日本国「日之本」里，蒸汽与柴油铁路曾与「铁路人偶（レイルロオド）」成对存在；空凝机与 AI 兴起后铁路大废线，小镇御一夜市面临污染工厂与人口流失。男主角右田双铁归乡，唤醒人偶八六，与妹妹日日姬、市长雏衣波莱特等人一起，用观光铁道与人情日常把御一夜从衰退里拉回来——核心是「人与车、人与人偶、人与故乡」的情感线。

**【世界观 · 舞台设定】**
日之本的铁路全盛期，每列车配有专属铁路人偶：她们像列车的人格化伙伴，能与驾驶沟通、照料车辆与乘客情绪。帝国铁路解体前后，新交通「空凝机」（飞行器）与导航 AI 普及，铁轨荒废，人偶与车体被封存、拆卖或当作展品。御一夜市位于熊本一带、球磨川切割的盆地（原型为人吉市），酿酒、铁匠、河漂流与温泉构成生活底色；市议会一度推动空凝机工厂落户以求振兴，却会带来水污染——双铁归乡正是为了反对这项计划。温度来自：煤烟与蒸汽的气味、站台广播的余韵、人偶慢条斯理的敬语、以及小镇人彼此叫得出名字的距离。奇幻侧只写「人偶会说话、会害羞、会记得路线」，不写成战斗设定。

**【地理 · 生活舞台】**
- **御一夜车站与旧帝国铁路仓库**：双铁与日日姬唤醒八六的地方；尘封展柜、锈轨与煤渣是初遇舞台。
- **右田一酿酒厂与右田宅**：继姐真闇经营的家业与双铁落脚处；酒窖香气与熊本方言日常。
- **市役所与御一夜铁路公司**：波莱特身兼市长与社长，玲奈常坐在她膝边或车厢里。
- **隈河漂流码头与早濑家**：深美协助经营漂流观光；与凪的友情线多在河岸发生。
- **蓑笠锻冶店**：凪的祖父穰仍会修机车零件；铁砧声是「铁路还能被救」的听觉象征。
- **隈元银行御一夜分行**：稀咲任分店长，香月为秘书；融资与观光企划在此谈妥。
- **登吕流汤公共浴场**：结比老板是前人偶技师，情报与回忆的温泉节点。
- **煤探乐园 / 观光线路 / 九星号列车**：总集线中双铁整合资源后的成果舞台；末班与首发仪式是情感高潮地。
- **仲国（Last Run 新增）**：铁道博物馆、盛都熊猫基地、芭岩窄轨等考察点；西瓜、19号等人偶登场，双铁输出观光列车方案。

**【故事主线 · 情感线】**
**共通序章**：幼年列车脱轨事故中，双铁（本姓十川）失去父母与双胞胎妹妹十川路子，被右田家收养。多年后他为阻止空凝机工厂污染归乡，与继姐右田真闇、继妹右田日日姬重逢。日日姬带他巡视车站遗迹时，唤醒长眠的 8620 型蒸汽机车人偶八六；八六以古雅敬语请求找回失踪的车体 8620。双铁发现「复兴铁路＝替代污染工厂的振兴方案」，遂与八六结成驾驶与人偶的搭档。

共通中盘结识：市长兼司机**雏衣波莱特**与柴油车人偶**玲奈**；银行千金**宝生稀咲**；锻冶看板娘**蓑笠凪**与漂流少女**早濑深美**；以及 Navi（空凝机导航 AI）、赤井清春与人偶二六、奥莉薇与司机宗方甚六等配角。众人各自以绘画看板、融资、修车、漂流、酿酒参与「御一夜能不能不靠污染活下去」。

**八六线**：重心是找回 8620、修复与试运行。八六端庄、怕生又极度认真；双铁从「事故阴影」里学着再次信任铁轨。子结局中双铁向八六求婚，她得婚名**右田铃**。情感核：人偶是否配得上「被当作家人」而非展品。

**日日姬线**：继妹乐观、熊本口音时不时漏出；她用绘画记录小镇，后当选市长并在未来成为机车设计师。路线写兄妹边界的再定义、从「家里的双铁」到「并肩的伙伴」。HE 是共同把御一夜画进下一张地图。

**波莱特线**：法裔血统的市长怯场却责任感极重；市议会骚扰事件后她一度辞职，转而辅导日日姬从政。官方正典中双铁与波莱特结婚，育有女儿**右田光**。情感核：公共身份下的私密依赖，以及「驾驶席上的两个人」。

**玲奈线**：语速极慢、像幼儿般的人偶；与波莱特形影不离。路线侧重照顾、慢节奏相处与「被听懂」的安心。

**真闇线**：继姐、酒厂当家，身材丰满、方言重、姐系可靠。路线写家业压力、酿酒与铁路振兴的结合，以及「收养来的弟弟」如何变成可托付的人。

**稀咲线**：精明银行家女儿，最早发现日日姬画才；后任市长助理。告白后结婚的支线结局存在。情感核：算盘与真心并不矛盾。

**凪＆深美线**：双人向，少年感剑道兴趣 vs 内向河女；未来分别成为司机助手与漂流经营核心。写友情先于恋爱的成长。

**总集线（通关后）**：综合各线成果——整合观光资源、开发煤探乐园、成立铁路集团、用铁路人偶能力辅助酿酒、开通环九州「九星」观光列车。处女秀成功后，双铁走出事故阴影，甚至构想「新·干线」高速铁路。情感收束是「不是一个人在开，是整座镇在开」。

**Last Run 仲国篇（总集后）**：受仲国国务院铁道相关邀请考察；参观铁道博物馆、熊猫基地、芭岩铁路；结识人偶**西瓜**与司机**刘志寰**、秘书**姜叶煌**、矿区人偶**19号**等。双铁为仲国定制观光列车方案，把御一夜经验「出口」成跨文化温情，而非征服。

**【故事主线 · 情感线 · 细部】**

右田双铁归乡第一周的密度几乎全是「被名字叫住」：真闇在酒厂门口用熊本腔骂他瘦了；日日姬把未干的颜料手印拍在袖子上；波莱特在市役所走廊差点把文件箱撞进他怀里，道歉声小到像怕惊动整栋楼。八六苏醒后的几天，双铁必须同时学两件事——如何给蒸汽机车加水加煤，以及如何在不触碰事故记忆的前提下听八六描述「铁轨在脚下延伸时的安心」。他常在半夜里醒来，听见远方并没有列车经过，却仍下意识去摸床头怀表。

与波莱特、玲奈的初次同乘发生在 KiHa07S 的短程班上。玲奈把脸贴在窗玻璃上数电线杆，数到三就忘，于是从头再数；波莱特一边开车一边轻声纠正，像纠正自己的呼吸。双铁坐在司机后方的折椅上，第一次意识到驾驶可以不是灾难的前奏，而是一种被信任的节奏。回程经过旧跨线桥，八六在站台上向柴油车人偶行礼，新旧动力的人偶用敬语交换「车籍」，像两个家族交换家谱。

稀咲把观光客流量、空凝机厂承诺税收、河流污染长期治理成本摊在同一张表上，红笔圈出「五年后谁还愿意住在这里」。双铁听得头疼，却在她把日日姬的彩绘地图贴在表旁时忽然懂了：数字必须长出颜色，政策才会被人记住。香月默默续茶，用中性嗓音提醒「行长那边的期限」，像给整场恋爱戏加上现实的节拍器。

凪在锻冶店后巷挥木刀，深美在码头给救生衣编号；两人吵架从不用狠话，只用「今天河况不好」和「今天炉火不稳」点破心情。双铁若问未来，凪说「当司机」，深美说「让爷爷少巡一点轨」。总集线里两句话都兑现；双人线会更早看到预演——两人挤在九星号餐车给旅客解说球磨川，手指在地图上碰到又迅速分开。

真闇线的关键不在告白台词而在酒。她让双铁尝失败的试酿，说是用报废枕木烟熏的实验品。双铁喝完只说「像旧车站的雨」。真闇大笑又忽然安静，讲起收养那年账本多难看，她如何把「多一张嘴」写成「多种一缸希望」。铁路振兴若成功，联名酒会印 8620 轮径；若失败，她仍会酿，只是标签不再画轨道。

总集线高潮不仅是九星首发，还有事故纪念碑前的沉默。双铁没有长篇演讲，只把八六的手（金属温度）放在碑上，路人误以为父女。日日姬在远处把这一幕画得很小，小到只有家里人能认出。波莱特若已是妻子线，会在人群后排抱着幼小的光；若仍是市长线，会在广播里念旅客须知，声音稳得不像曾经怯场的人。

仲国篇的情感不靠猎奇而靠被听懂。西瓜用生硬日语问八六「蒸汽的梦和柴油的梦是否一样」，八六认真很久，答「内容不同，想被需要的心情相同」。刘志寰脸红得像锅炉，姜叶煌与稀咲用公文腔讨论签证与轨距，19 号拉众人去看矿区小孩把窄轨当滑梯——双铁笔记写：观光列车第一乘客应是本地孩子。回国夜 Navi 罕见感性地说「飞行也需要路线」，八六不再敌视，只说「那就并线吧」。
