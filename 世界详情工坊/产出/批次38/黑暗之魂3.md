# 黑暗之魂3
<!--meta lib=主库 tiers=一、二、三、四、五、六、七-->

## 剧情

**【作品来源】**
FromSoftware 制作、万代南梦宫发行的动作角色扮演游戏《黑暗之魂3》（DARK SOULS III / ダークソウルIII），总监宫崎英高，2016 年 3 月日本发售、4 月全球发售（PlayStation 4 / Xbox One / PC）。后续 DLC「亚莉安诺尔的余烬」（Ashes of Ariandel）与「环印城」（The Ringed City），合集称 The Fire Fades Edition。本作是魂系列第三作，气质偏「火之时代终章」：与《黑暗之魂》《黑暗之魂2》共享「初始之火—传火循环—不死诅咒」宇宙，但不把前作主线硬塞成本作线性剧情；互文多藏在道具说明、环境废墟与 NPC 只言片语中。文风是末世灰烬、薪王拒召、碎片叙事、高难度战斗与诗意残酷。

**【世界定位】**
洛斯里克王国火将熄灭的黄昏。王子洛斯里克拒传火，旧日薪王自传火祭祀场王座出走；灰烬墓地中「灰烬人」（Unkindled / 无火的余灰，亦称灰烬）苏醒，奉命猎回薪王余烬，坐满五座空王座，再入初始火炉面对「薪王的灵魂」。主线是传火体系的最后痉挛，与「传火 / 熄火 / 篡火」的终局选择题。契约者切入即成为灰烬巡礼的一环：篝火传送、余烬燃起、入侵与协助并存。

**【世界观 · 力量体系】**
远古差异未生时，世界无形态。初始之火燃起后，热与冷、生与死、光与暗分立，火之时代开启。古龙近乎不朽，葛温等持王魂者率众猎龙，建立神族秩序。火衰则传火：历代强者入初始火炉，以己身为薪延续光明，亦延续不死诅咒与游魂化。每一次成功传火，残烬与战意汇入火炉门卫「薪王的灵魂」——它不是单一君主，而是历代传火者之集合影，第一阶段轮转诸战技，第二阶段回响葛温式阳光枪与压迫。

到洛斯里克时代，火已极弱。钟声响起，本应再作薪柴的旧薪王被掘出，却大多拒绝归位。未能承受火焰而烧成灰的失败者称「灰烬人」：连当薪的资格都没有，却被最后希望唤醒，去猎回他人的余烬。灵魂仍是经验与货币；**余烬（Ember）** 燃起可增生命上限并打开召唤与被入侵窗口；**集中力（FP）** 驱动战技与法术，灰白瓶回 FP、橙瓶回血。武器强化、奇迹 / 咒术 / 魔法三系、装备负重与翻滚无敌帧构成战力骨架。死亡后自篝火再起，掉落的灵魂可在尸体处夺回——叙事上「死」是进度税，不是永久退场（契约者若无不死特权，则按乐园规则另计）。

属性大致包括活力、精神、耐力、力气、敏捷、智力、信仰、运气等，决定武器战技与法术阈值。战技（Weapon Art）是本作战斗签名：每把武器自带特殊动作，耗 FP。法术三系大致分工——奇迹偏信仰、雷电与恢复；咒术偏火焰、混沌与黑火；魔法偏结晶、灵魂箭与控场。契约（太阳战士、蓝监察、法兰看门犬、罗萨莉亚指头、剑之墓碑、黑暗手等）把协助与入侵写成世界规则，而非现代公会任务板：白记号协助、红灵入侵、蓝监察来援，皆是咒与契约的可视化。

战力阶梯（按破坏力与区域压迫，宁低勿高）：
1. 灰烬墓地 / 高墙杂兵、空洞骑士 ≈ 一阶（徒手可杀、冷兵器格斗）。
2. 洛斯里克骑士、道外巡游战士、早期区域精英 ≈ 一～二阶（碎砖裂石、小队威胁）。
3. 咒术师、深渊教徒、法兰狼、中期区域主（咒树、水晶贤者等）≈ 二～三阶（据点级、成型异能）。
4. 深渊监视者、深渊主教、冷谷舞者、王城门卫级 ≈ 三～四阶（质变分水岭、建筑群级压迫）。
5. 巨人尤姆、噬神者奥尔德里奇、伊鲁席尔主与古龙侧 ≈ 四～五阶（街区～城级、领域雏形）。
6. 双王子洛斯里克与罗利耶尔、无名王、黑龙米迪尔残影 ≈ 五～六阶（地区灾难、高速与规则皮毛）。
7. 薪王的灵魂、奴隶骑士盖尔（环印城终局）、初始之火规则本身 ≈ 六～七阶及超阶边缘（战略级考验、传火史集合体）。

乐园阶位映射：灰烬起步 / 高墙杂兵 ≈ 一阶；洛斯里克骑士与中区精英 ≈ 二阶；区域 Boss 与法兰外围 ≈ 三阶；深渊监视者·薪王模式 ≈ 四阶；尤姆 / 奥尔德里奇 / 伊鲁席尔主 ≈ 五阶；双王子 / 无名王 ≈ 六阶；薪王之魂 / 盖尔 / 火之将熄 ≈ 七阶及超阶边缘。宁低勿高；顶点条件性胜利，严禁把顶点写成「战力归零」。

**【地理 · 舞台】**
世界以传火祭祀场（火祭场）为辐辏，篝火传送较早开放。灰烬墓地外接高墙与祭祀场；洛斯里克高墙俯瞰王国，飞龙残骸挂在城垛。高墙下是不死聚落：木桥、绞架、树人与咒术师构成底层绝望景观。道外湿地分叉——一侧通向法兰要塞（狼血不死队与深渊监视者），一侧通向卡萨斯地下墓所与深渊大书库，再连卡利姆大教堂（深渊教团）。伊鲁席尔冷谷月下如梦，庞蒂夫苏利梵统治其间，外置监狱与奴隶并存；由此可登亚诺尔隆德幻影与古龙顶风雪。罪业都市埋在熔渣与罪业之火的余热里，巨人尤姆的王座厅空旷得像坟。洛斯里克城垂直上升，冷谷舞者守门厅，双王子居高塔。终局灰烬荒原只余螺旋与剑冢，通向初始火炉。

DLC 舞台：绘画世界亚莉安诺尔以血与腐雪维系停滞的慈悲；地之尽头渣滓堆是诸国废墟的挤压层；环印城是远古对人类 / 猪米族的封印之城，黑暗灵魂的神话层在此落地。地图本身在说：火的世界已经烧完了，只剩余烬与渣滓。火祭场五王座是进度 UI 也是政治地图：空位对应深渊监视者、奥尔德里奇、尤姆、洛斯里克一脉，以及守约留下的卢德里特。

**【世界剧情线】**
① 前史·火与薪的循环
初始之火燃起差异，葛温等王魂灭古龙，开火之时代。火衰则传火：历代强者入火炉为薪，延续光明，亦延续不死诅咒。每一次传火，残烬汇入薪王的灵魂。循环重复至洛斯里克：王室为「造出能传火的王子」不择手段，幼王子洛斯里克体弱遭咒，兄长罗利耶尔与之灵魂绑定——罗利耶尔曾是勇武骑士，为弟承受诅咒后跛足失语，仍以剑开路。火极弱时钟声响起，掘出旧薪王再作燃料：深渊监视者（法兰狼血军团之集合）、巨人尤姆、噬神者奥尔德里奇、库兰的卢德里特（路德雷斯），以及本应传火的洛斯里克一脉。旁白与道具说明反复强调同一结构：火将熄，薪王离席，灰烬将起。

② 开篇·灰烬苏醒
灰烬墓地中，灰烬人从棺中爬起。旁白点明：薪王离开王座，灰烬将起；无名、被咒、连薪都不配的不死者，却被指望猎回余烬。传火祭祀场见火防女（火女）：升级、叙事、王座空位一目了然。五座王座中，唯有库兰的卢德里特守约在席，自嘲「最不配」却留下，并精通灵魂转置；其余皆空。铁匠安德烈在此提供强化与楔石体系，是几乎所有灰烬的后勤起手。高墙外洛斯里克骑士与飞龙余威构成第一课——死了再来，灵魂掉在尸体处。主题：你不是被选中的英雄，你是最后的应急方案。霍克伍德亦在火祭场现身：法兰逃兵，厌战自嘲，嘴里仍有狼血与监视者的余响。

③ 高墙、不死聚落与道外
洛斯里克高墙是王国脸面：骑士盾墙、弩手、龙息扫过城垛，教会「余烬燃起等于邀请入侵」。墙内空洞者重复无意义动作，展示诅咒日常。出高墙后，不死聚落以绞架与「还想活下去的绝望建筑学」承接。此处可遇伦道尔的约尔（约拉）：他以「引出真正的力量」在灰烬人身上刻下黑暗火印，埋下篡火线种子；约尔死后，约莉娅现身，称灰烬人为王，导游魂之王路线。聚落与道外亦可遇格雷拉特（小偷巡礼者，后期可成商人或悲剧）、西格瓦德（洋葱骑士落井，憨直重诺）、安里与艾丽戈（双人巡礼，后分复仇与婚礼陷阱）、西里斯（蓝监察 / 秩序侧协助）、欧贝克（温海姆魔法师，求卷轴畏囚禁）、利奥纳德（罗萨莉亚指头，优雅残忍）——日常是送刀、送魂、送情报，而非一上来讨论火炉哲学。道外十字路正式分流：法兰、卡萨斯、卡利姆。

④ 猎薪·第一薪王：法兰·深渊监视者
法兰要塞建立在对深渊的恐惧与模仿之上。狼血不死队继承狼骑士亚尔特留斯遗志，猎杀深渊所在；军团灵魂借狼血相连，一人即众人。他们曾传火为薪王，复活后却陷入互斩：不是友军误伤，而是深渊侵蚀与「以同伴为柴」的残酷逻辑同时发作。灰烬人闯入试炼场，面对持火焰大剑的监视者高速连斩；增援尸体会加入混战，血量过半后进入薪王模式，战场点燃，身法更诡。击败后得薪王余烬，送回传火祭祀场王座。阶段因果：灰烬人第一次「杀死曾为薪王的存在」，理解传火不是荣耀而是消耗。霍克伍德曾属此军团，逃至火祭场后仍会谈狼血与爪——法兰荣光在他嘴里是「我再也不想回去」；战后他可能转向敌对夺爪。法兰看门犬契约则把「反入侵」写成军团外围的规则暴力。

⑤ 侧翼·卡萨斯、深渊大书库与卡利姆
卡萨斯地下墓所与深渊大书库提供「知识型恐怖」：吞食、咒术、对神性的学术肢解。沃尔尼尔等区域主展示贵族腐烂；书库结晶与学者残骸呼应「求知即疯狂」的传统。欧贝克可在此线成为魔法教师——他要卷轴与知识，忌讳被当作「只配教人」的囚徒。卡利姆大教堂则是深渊教团的圣域：他们把「深层」与吞噬圣化，活祭队列通向奥尔德里奇的神话。深渊主教们以信仰为皮，以人渣为骨。伊莉娜与艾瑞昂走奇迹师徒伦理线：信仰、囚禁与「是否继续传授」构成支线道德题。灰烬人若走此翼，战力与情报同步上升，但主线仍要求回到薪王名单——书库不能替代王座。

⑥ 猎薪·第二薪王：噬神者奥尔德里奇
奥尔德里奇本为神职，因嗜食人类膨胀成泥，被强行立为薪王（因力量而非美德）。复活后他逃离王座，受伊鲁席尔政治与深渊教团牵引，吞食暗月之王葛温德林，于亚诺尔隆德幻影中以弓箭与生命猎食战技迎敌。泥浆中的神骸、暗月残响、教团与庞蒂夫的政治阴影叠成名场面。击败后余烬归位。此线常与安里线交织：安里为复仇或「婚礼」走向庞蒂夫与黑暗月墓——政治与亲密关系的残酷在此同时结算。利奥纳德则走罗萨莉亚指头与「舌头」的黑暗侧，把重生与背叛写成另一条人性解剖。教团的「深层」不是单纯水潭，而是吞噬与沉沦的神学包装。

⑦ 伊鲁席尔冷谷与庞蒂夫苏利梵
苏利梵以法环、外置监狱与冷谷骑士控制伊鲁席尔：美与奴役并存。他是中期政治枢纽——推动奥尔德里奇、安置舞者、连接古龙顶与亚诺尔隆德废墟。击败苏利梵不是薪王任务，却几乎是通往噬神与古龙的必经。西里斯走蓝监察 / 暗月相关协助线，与利奥纳德、教团阴影形成对照：有人守契约秩序，有人把舌头当货币。冷谷月下广场、外宾礼仪与突然落下的红灵，构成魂式社交生态。自伊鲁席尔可登古龙顶：风雪、大弓、龙裔与**无名王**。无名王战分两阶段，雷枪与龙息把「被放逐的葛温之子 / 猎龙战神」传说打成实体。此战非强制薪王，却是六阶级试炼与系列彩蛋高地——强敌不等于坐王座的钥匙。

⑧ 猎薪·第三薪王：巨人尤姆
罪业都市曾遭罪业之火吞噬。罪业之火非初始之火，是另一种「燃尽一切」的隐喻。巨人尤姆为救民而传火，归来却见城市已毁，孤独坐于王座，手持巨砍刀，丢弃曾用的大盾。好友西格瓦德线可在此收束：持风暴统治者赴约，用「友谊」完成对巨人的慈悲击杀。击败尤姆得余烬。名场面是王座厅的空旷、巨人体型与「曾想守护」的悲剧——传火没有救城，只留下更冷的王座。契约者在此可做的不是抢终极权柄，而是决定是否把风暴统治者交到正确的人手上、是否在巨人倒下后染指罪业遗宝、是否把尤姆的故事传回火祭场。

⑨ 第四座·库兰的卢德里特（路德雷斯）
小个子薪王卢德里特未出走，坐于火祭场王座。他承认自己「最不配」，却守约留下；精通灵魂转置，可将 Boss 魂炼成武器与咒物。他是五王座中唯一的合作薪王，也是终局前的伦理对照：并非所有薪王都叛逃。转置窑把「吃掉敌人的故事」变成可装备的记忆——魂系列一贯的残酷诗学。他亦暗示过黑暗时代的幻象，与无名墓地 / 火防女之眼线形成互文。

⑩ 拒薪者·双王子洛斯里克与罗利耶尔
洛斯里克城高耸入云。王室为传火制造容器，幼王子洛斯里克体弱遭咒，兄长罗利耶尔与之共命运。洛斯里克拒绝传火，躲在高塔——灰烬人必须攻入王城，击败冷谷舞者、三尖枪骑士与诸多王室守军，最终与双王子战：罗利耶尔挥剑，洛斯里克施咒，二人灵魂绑定；一人倒下另一人狂化。击败后得关键余烬。主题：被制造的救世主有权说不；灰烬人则用刀回答「世界不允许你说不」。此战是叙事高潮，也是坐满王座前的最后一枚钉子。王城每上一层都是对「使命」的加码：旗帜更旧，守卫更疯，谈判桌从不存在。

⑪ 质变·坐满王座与初始火炉
五座王座坐满，火防女引导灰烬人于螺旋剑前受余烬承认，开启通往初始火炉之路。灰烬荒原上，薪王的灵魂现身：第一阶段轮转剑盾枪咒诸战技（历代传火者之影），第二阶段阳光枪与葛温式压迫——音乐与姿态直接回响初代薪王。此战是系列终章门槛：你打的不是单一 Boss，是「传火史本身」。薪王的灵魂不是「可以被削弱的老王」，而是规则级门卫：你必须证明有资格动火。

⑫ 终局三岔（及变体）
- **传火**：战胜薪王之魂后，灰烬人将己身投入火中，延续火之时代——火仍将再熄，循环未断，世界以更苍白的余烬态续命。
- **火之终结**：若取得火防女之眼（无名墓地 / 无主墓地线）并完成仪式，可召唤火防女熄火，迎来黑暗时代；旁白暗示日后微焰仍将如余烬般跳动。变体：在熄火瞬间攻击火防女夺火，叙事上为个人夺焰。
- **火之篡夺**（伦道尔线）：完成约尔 / 约莉娅与安里相关步骤、累积黑暗火印且不令火防女治愈，可在终局篡夺火焰，成为游魂之王式终局，把火夺给人类 / 游魂一侧。
三种结局都昂贵。传火＝续命循环；熄火＝把选择交给暗与火防女；篡火＝人类夺权。写结局后须先声明选了哪条。低阶契约者只应听闻火炉传说，不应被写进「一刀斩杀薪魂」的桥段。

⑬ DLC·亚莉安诺尔的余烬
绘画世界中，修女芙莉德与父亲亚莉安诺尔以血作颜料维系腐烂的画。奴隶骑士盖尔请灰烬人「向画中之火亮出火焰」——实为烧掉腐画以便重绘。芙莉德三阶段战（镰、黑火、与父亲共斗）是中高阶试炼。主题：腐烂的慈悲与「重画世界」的代价。少女画师等待颜料，为环印城线埋线。腐画与主线「循环是否该断」同构：停滞的温柔也是一种暴力。

⑭ DLC·环印城与盖尔
地之尽头，渣滓堆挤压诸国废墟；环印城是远古对人类 / 猪米族的封印之城，猪米王与黑暗灵魂碎片的叙事收束系列伏笔。唤醒菲莉安诺尔公主后时间层错位，世界进一步沉入灰烬荒原。奴隶骑士盖尔为绘画少女收集黑暗灵魂，在渣滓堆尽头与灰烬人决战——盖尔自枪盾到狂化撕咬与十字弩弹幕，是系列最强单体压迫之一。击败后，黑暗灵魂血渍成为新画颜料；绘画少女启程绘制「一个温柔的世界」。此线将「火与暗」从政治选择拉回神话材质本身。盖尔不是简单反派，是「忠诚执行到世界尽头」的镜映。

⑮ 微观可介入层与世界状态
火祭场商人、铁匠安德烈、安里线、西格瓦德线、霍克伍德线、罗萨莉亚指头、太阳战士契约、法兰看门犬与蓝监察等——AI 优先写：余烬是否燃起、哪座王座已坐、谁在雾门前放召唤记号，而非一上来火炉决战。每放上一份余烬，火祭场光影与 NPC 台词推进一层。契约者改变局部（救下西格瓦德、杀或放生某商人）可以；改写「必须集齐薪王才能动火炉」的结构则需付出乐园级代价。终局后：传火则循环延续；熄火则暗沉降；篡火则游魂称王；环印城线可与主线结局并存于不同年表页。

⑯ 薪王出走的各自理由（正典动机）
深渊监视者并非「想通了循环无意义」才离席，而是狼血与深渊侵蚀把军团拖进互斩；尤姆的出走更接近悔恨——传火没能保住罪业之民，王座只剩空旷；奥尔德里奇的出走是野心与食欲，他宁可噬神也不愿再当燃料；洛斯里克一脉的出走是拒命，被制造出来的传火容器选择与兄长共死也不肯下楼；唯卢德里特留下，证明守约与叛逃可以并存于同一套火祭场礼仪中。灰烬人猎回的不是抽象「进度点」，而是这五段拒绝被烧掉的人生。

⑰ 战斗与社交的日常语法
燃余烬后走廊可能突然多出一个姿态嘲讽的红灵；苦战中白灵落地鞠躬；蓝监察可能在你被入侵时闪现制裁。这些不是现代任务板，而是契约把「协助 / 猎杀」写进世界物理。火祭场则是唯一中立补给点：安德烈认矿石，卢德里特认灵魂，火防女认等级与仪式。格雷拉特可能从聚落一路偷到伊鲁席尔后失踪或归来；西格瓦德可能卡在井、厨房与电梯之间，直到罪业厅赴约。正文密度应落在这些可观察细节上，而不是空喊「火之时代崩坏」。

**【叙事基调 · 雷区】**
画风：灰白末日、哥特废墟、悲壮而非爽文打脸；台词少而重。忌把灰烬人写成天选龙傲天（他是不合格品）；忌把传火写成单纯正义、把熄火写成单纯邪恶（皆有代价）；忌低阶正面硬刚薪王之魂 / 盖尔；忌把魂世界套成修仙升级文；忌把法兰监视者写成单人武侠——他们是军团与互斩。最早可切入锚点：灰烬墓地苏醒至高墙。写战斗先写地形与雾门；写薪王先写「他为何曾传火、为何拒召」；写灰烬人先写「失败与再起」而非「天命」。契约者不是灰烬人的替代容器，而是多一个可能抢余烬、卖情报、或在法兰雾门里当白灵的变数。本地人听不懂「阶位」，只懂「你燃没燃余烬、你是白是红、你有没有碰我的魂」。主题句不是「英雄拯救世界」，而是「灰烬寻火」——王座空着，火防女仍履行职责，循环靠烧掉强者续命，最后烧掉的可能是「资格」本身。

## 阶位切入点

> 阶位↔原作战力：一阶≈灰烬墓地/高墙杂兵；二阶≈洛斯里克骑士与中区精英；三阶≈区域主与法兰外围；四阶≈深渊监视者·薪王模式；五阶≈尤姆/奥尔德里奇/伊鲁席尔主；六阶≈双王子/无名王；七阶≈薪王之魂/盖尔/火之将熄。世界顶点（薪王的灵魂、盖尔、初始之火规则）为跨阶存在，低阶切入一律「规避顶点·情报优先」；超阶巅峰：存在·情报优先/条件性胜利，严禁把顶点写成「战力归零的空壳」。

**一阶（灰烬墓地·高墙求生 · 井底灰烬）**
切入身份/时点：契约者以初醒灰烬人、高墙逃兵或火祭场杂役切入，锚定灰烬墓地开棺至洛斯里克高墙初段。
初始事件：你在棺中推开盖板，灰扑满面；火祭场方向传来火防女低语「薪王已离席」。高墙巡逻骑士一矛刺穿你身旁的空洞者——他要你立刻选择：跟死者抢灵魂，或滚进篝火阴影。
开场白建议：「棺木裂开时，你咳出的不是尘土，是灰。远处高墙上有龙骨挂着，像旧旗。有人在雾门另一侧放下白记号，又有人已把红记号按在你脚边——余烬还没燃，命却已经开始喊价。」
关键NPC立场：**火防女**（升级与指引，温和但只服务传火结构）；**安德烈**（铁匠，只认楔石与魂）；**霍克伍德**（逃兵，嘲笑监视者荣光，可套法兰情报）；**灰烬人**（若并行，则是潜在抢魂对手）。
主线钩子/支线：主线——抵达火祭场见空王座；支线——高墙取余烬、与霍克伍德对话、送第一份材料给安德烈。
危险度/规避：危险度中；规避洛斯里克飞龙正面、规避任何薪王名。
任务方向/奖励：存活、学会篝火节奏；奖励：基础武器强化、少量余烬、火祭场落脚权。

**二阶（不死聚落·道外立名 · 游魂边缘）**
切入身份/时点：以道外巡礼者或聚落猎手切入，锚定不死聚落通关前后至道外十字路。
初始事件：绞刑架下，咒术师与穴居者夹击一支运尸队；你被当成「还能动的货」。道外湿地雾起，法兰方向传来狼嚎——有人出钱买「活着走到要塞门」的带路人。伦道尔的**约拉**在阴影里低声问你，要不要「引出真正的力量」。
开场白建议：「聚落的木桥在晃，桥下不是水，是不肯散的怨。你刚学会不在敞开处喝药瓶，道外的雾就把火祭场的光吞掉——法兰在前，卡萨斯在侧，走错一步就是别人的灵魂拾取点。」
关键NPC立场：**霍克伍德**（给法兰半真半假的忠告）；**欧贝克**（咒术 / 魔法交易前的求知者姿态）；**西格瓦德**（井中或电梯处的憨声，可结盟）；**约拉**（黑暗火印诱惑）；**灰烬人**（可能已燃余烬）。
主线钩子/支线：主线——打开通往法兰 / 卡萨斯的路；支线——救洋葱骑士、学基础咒术 / 魔法、处理聚落咒物、决定是否接受约拉火印。
危险度/规避：危险度中高；规避深渊监视者本体硬刚，仅侦察要塞外围。
任务方向/奖励：地图打开、中阶防具；奖励：狼血相关线索、契约徽章入门。

**三阶（法兰外围·教团阴影 · 准质变）**
切入身份/时点：以法兰外围斥候、看门犬契约登记者或卡利姆外门切入，锚定法兰要塞门至监视者雾门前，或卡利姆大教堂前段。
初始事件：要塞门内两名狼血战士忽然互砍，第三人把刀指向你：「外来灰，也配沾狼血？」与此同时，卡利姆方向有活祭队伍索要「自愿者」——你必须在「被军团当柴」与「被教团当食」之间找第三条路。
开场白建议：「石墙缝里长着树根，像从尸体里钻出来的。雾门缝隙漏出火光与金属撞击——那不是演武，是薪王候选人在互相点燃。你握紧刀，第一次听清火祭场交待的真正含义：要的不是劝降，是余烬。」
关键NPC立场：**深渊监视者残员**（敌对或契约敌）；**霍克伍德**（要你证明配拿狼血）；**教团主教外围**（劝你献祭）；**西里斯**（秩序侧协助可能出现）；**灰烬人**（可能已在雾门内反复死亡）。
主线钩子/支线：主线——侦察监视者战并决定是否助战 / 抢尾刀；支线——法兰看门犬入侵任务、卡利姆潜入、卖要塞地图。
危险度/规避：危险度高；规避薪王模式二阶段硬抗，优先白灵协助或环境杀。
任务方向/奖励：准四阶资格；奖励：狼血材料、教团情报、中阶特武线索。

**四阶（深渊监视者 · 薪王模式锚点）**
切入身份/时点：以正式猎薪者、法兰试炼挑战者或灰烬人同僚切入，锚定深渊监视者 Boss 战全程。
初始事件：雾门开启，持火焰大剑的监视者迎面冲锋；第二、第三名自黑暗中加入。血量过半，全场燃起——你的任务不是「旁观传说」，而是在薪王模式下活到余烬掉落，并决定余烬送回火祭场还是另藏。
开场白建议：「第一刀过来时，你还以为是一个人。第二刀、第三刀从侧翼咬住翻滚的终点，你才明白：狼血共享的不是荣耀，是杀意。火焰爬上剑脊的刹那，法兰要塞不再是地图，是炉膛——而你是尚未成型的柴。」
关键NPC立场：**深渊监视者**（必须击败的薪王集合体）；**火防女**（催促余烬归位）；**霍克伍德**（战后态度剧变，可转敌对夺爪）；**路德雷斯**（赞你「办了大事」并暗示转置）；其他灰烬人（抢魂 / 抢余烬）。
主线钩子/支线：主线——击败监视者、余烬上座；支线——霍克伍德对决、转置狼骑士武器、法兰清残党。
危险度/规避：危险度极高；规避同时惹伊鲁席尔庞蒂夫；此阶仍远离火炉。
任务方向/奖励：第一薪王余烬政治权；奖励：监视者魂转置武、法兰通行、四阶公认战力。

**五阶（尤姆·奥尔德里奇·冷谷 · 领主棋盘）**
切入身份/时点：以猎第二 / 第三薪王的正式灰烬、伊鲁席尔渗透者或罪业遗民切入，锚定奥尔德里奇战、尤姆战或苏利梵厅前后。
初始事件：冷谷月下，苏利梵的法环在广场展开；或罪业王座前，巨人睁眼只问「又来取薪？」——你带着火祭场的空位名单，发现每一份余烬背后都是一座已死的城与一套自圆其说的正义。
开场白建议：「伊鲁席尔的雪是安静的，泥里的奥尔德里奇却在消化一位神。另一边，罪业的热风把洋葱骑士的铠甲烤得发亮——他说有约要赴。你开始明白：薪王不是 Boss 列表，是这个世界拒绝熄灭时咬碎的牙。」
关键NPC立场：**奥尔德里奇**（噬神灾厄）；**巨人尤姆**（可敬敌）；**庞蒂夫苏利梵**（支配者，可先杀可后杀）；**西格瓦德**（赴约）；**安里**（复仇线）；**利奥纳德**（指头阴影）；**路德雷斯**（转置升级）。
主线钩子/支线：主线——再取一至两份薪王余烬；支线——西格瓦德收束、安里婚礼 / 复仇、苏利梵是否击杀、西里斯与利奥纳德的对撞。
危险度/规避：危险度极高；规避双王子与火炉；古龙顶可选但慎入。
任务方向/奖励：多王座进度；奖励：生命猎食 / 巨武级转置、冷谷权柄碎片、五阶领域感装备。

**六阶（洛斯里克城·双王子 · 拒薪之塔）**
切入身份/时点：以王城突入者、王室叛徒或资深猎薪者切入，锚定冷谷舞者后至双王子战。
初始事件：舞者在门厅旋转，三尖枪骑士封锁楼梯；塔顶传来咳嗽与咒言——洛斯里克仍拒绝下楼。火祭场只差最后一座王座，整个王国的军队却堵在你与「拒薪」之间。
开场白建议：「王城比高墙更像牢。每上一层，旗帜更旧，守卫更疯。你听见两个声音在争论：一个说『不要传火』，一个说『那就死在一起』。雾门后没有谈判桌，只有必须坐满的空位。」
关键NPC立场：**洛斯里克与罗利耶尔**（拒薪双王子）；**冷谷舞者**（门厅处刑者）；**火防女**（五王座将满的仪式语气）；**灰烬人**（争终局资格）；王室残党（死战）。
主线钩子/支线：主线——击败双王子、第五余烬上座；支线——王城支线 NPC 命运、是否提前杀舞者开门、无名王可选试炼。
危险度/规避：危险度贴近顶点；规避薪王之魂，仅筹备火炉。
任务方向/奖励：开启火炉资格；奖励：王子魂转置、王城遗宝、六阶公认。

**七阶（初始火炉·环印城 · 火之终局）**
切入身份/时点：以真薪王候选、熄火 / 篡火仪式执行者或环印城巡礼者切入，锚定五王座坐满后火炉战，或 DLC 盖尔战前后。
初始事件：螺旋剑承认你，灰烬荒原上薪王的灵魂自火中成型，第一招便是你在法兰见过的剑法——又在下一秒变成阳光枪。另一条时间线上，渣滓堆尽头盖尔转过身，说还差最后一点黑暗灵魂。
开场白建议：「这里没有王国，只有火还记得的名字。薪王的灵魂抬剑时，你看见的不是一个敌人，是所有肯把自己烧进去的人叠成的影。你若传火，循环继续；你若熄火，暗会来；你若篡火——先问问自己，配不配称王。远处环印城的钟声像从地心敲上来。」
关键NPC立场：**薪王的灵魂**（顶点考验，条件性胜利）；**火防女**（熄火线关键）；**约莉娅**（篡火线）；**盖尔**（DLC 顶点）；**绘画少女**（终局象征）；**路德雷斯**（可已燃尽）。
主线钩子/支线：主线——火炉三结局之一；支线——环印城黑暗灵魂、绘画世界重绘、系列伏笔收束。
危险度/规避：危险度顶点；超阶巅峰：存在·情报优先/条件性胜利；禁止把薪魂 / 盖尔解释成可随便碾压的空壳；低配契约者只做仪式边角与情报。
任务方向/奖励：终局权柄与系列级道具（黑暗灵魂颜料、传火 / 熄火 / 篡火叙事结算）；奖励取向：魂转置顶点武、契约终极徽记、火种或黑暗火种——不得无代价复制「初始之火本体」。
