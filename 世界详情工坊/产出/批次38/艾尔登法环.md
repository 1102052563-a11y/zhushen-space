# 艾尔登法环

## 剧情

**【作品来源】**
《艾尔登法环》（日：エルデンリング / 英：Elden Ring）是 FromSoftware 开发、万代南梦宫娱乐全球发行的魂系开放世界动作角色扮演游戏。总监宫崎英高，神话总设定由奇幻小说家乔治·R·R·马丁撰写，宫崎英高与团队再将神话落成可探索的交界地与破碎战争后史。2022年2月25日发售；2024年6月21日推出大型DLC《黄金树幽影》（Shadow of the Erdtree）。文风为黑暗奇幻：史诗凋零、神性腐朽、探索无指引、死亡可循环却代价沉重。本作无固定“小说主角姓名”，视点角色为玩家自定的**褪色者**；叙事靠开场旁白、物品描述、NPC对话与地形环境拼合。

**【世界定位】**
交界地是黄金树与艾尔登法环主宰的中世纪黑暗奇幻大陆。永恆女王玛莉卡打碎法环后，碎片“大卢恩”落入诸半神之手，破碎战争爆发，无上意志抛弃交界地。玩家作为失去赐福又被重新召回的褪色者，目标是收集大卢恩、修补法环、成为艾尔登之王——或走向星之时代、癫火之王等另类终局。契约者切入时，世界处于“法环已碎、黄金树仍立、半神割据、圆桌厅堂重开”的后破碎时代。

**【世界观 · 力量体系】**
力量本源不是灵气修仙，而是**赐福（Grace）**、**卢恩（Runes）**、**黄金律法**与诸外神/神祇权柄。黄金树降下赐福之光，引导英雄与王者；艾尔登法环是无上意志借“艾尔登之兽”植入交界地的秩序本源。玛莉卡打碎法环后，律法裂缝、死亡被扭曲、半神因大卢恩而疯。

核心资源与成长：
- **卢恩**：击杀生物与强敌获得的辉光碎片，可在赐福处由梅琳娜（或替代者）转化为属性与等级，也可用于强化武器、购买、交易。
- **赐福点**：世界中的金色光点，作存档、回血、传送与升级锚；点亮赐福即被引导线牵引。
- **大卢恩**：法环碎片，持有者多为半神“碎片君王”。至少两枚大卢恩是修补法环、进入王城深层政治的门槛。
- **战灰 / 战技 / 祷告 / 魔法**：战斗权能分别来自武器记忆、信仰（黄金树/双指/外神）、智力（卡利亚魔法、辉石、星光）等。
- **灵马托雷特**：梅琳娜赐予的灵体坐骑，开放世界机动与骑乘战的核心。
- **骨灰召唤**：圆桌与各地骨灰可召唤灵体助战，非无限复活的队友，是战术资源。
- **死亡规则**：褪色者可在赐福处反复“归来”，但非无敌——高阶半神、命定之死、癫火、外神侵蚀会改写“可重来”的意义；真正的命定之死被封印后，死亡一度无法彻底终结神性血脉，黑刀之夜即利用偷出的命定之死刺杀黄金葛德文。

战力阶梯（按交界地实际破坏力与地位，非修仙境界）：
1. **流民 / 士兵 / 卫兵 / 流浪骑士**：凡人冷兵器与粗浅武技，毁木门、守据点。≈一阶。
2. **骑士团精锐 / 英雄级战士 / 低阶法师**：碎石裂甲、小队歼灭、初级魔法/祷告。≈二阶。
3. **名城将领 / 中阶魔法骑士 / 地下墓穴Boss级**：端据点、裂开车辆级防御、成型战技与领域雏形。≈三阶。
4. **英雄级强敌 / 部分传说骨灰本尊投影 / 初阶半神代行者**：摧毁建筑群、一城数人级精英。≈四阶。
5. **碎片君王级半神（中位）**：一击夷平街区、领域/灾厄显化（如恶兆、猩红腐败局部）。≈五阶。
6. **顶级半神 / 镇国级灾难**（如满状态碎星将军拉塔恩、女武神玛莲妮亚巅峰）：覆盖地区、移山陨星级表现。≈六阶。
7. **王城守护者 / 火焰大锅与神皮级威胁 / 接近神座者**：毁国毁域级战略威胁。≈七阶。
8. **王夫拉达冈、艾尔登之兽、神人/外神代行**：行星规则级、秩序本源战。≈八阶~超阶边缘。
9. **无上意志、外神本体、完全命定之死与癫火终局级**：已非单人武力可量度，属超阶。

乐园阶位映射：流民士兵≈一阶；骑士精锐/低阶法师≈二阶；名城将领/英雄Boss≈三阶；精英英雄/初阶半神代行≈四阶；中位碎片君王≈五阶；顶级半神（拉塔恩、玛莲妮亚等）≈六阶；王城与神座门槛（蒙葛特后、火焰大锅、灰城）≈七阶；拉达冈/艾尔登之兽/完全神性=超阶巅峰。宁低勿高；顶点写条件性胜利与情报优先，严禁“被封印所以战力为零”。

**【地理 · 舞台】**
交界地以巨大**黄金树**为视觉与信仰中心，根须与光辉覆盖大陆。主要区域与剧情意义：
- **宁姆格福**：褪色者离开漂流墓地后的起点原野，风暴山丘、关卡前哨、湖泊与遗迹交错；有「接肢」葛瑞克盘踞的史东薇尔城。
- **湖之利耶尼亚**：卡利亚王室与魔法学院雷亚卢卡利亚的舞台；满月女王蕾娜菈、辉石魔法与卡利亚骑士传统在此。
- **盖利德**：猩红腐败蔓延的红土荒原；红狮子城、恸哭沙丘、兽人与腐败生物；**碎星拉塔恩庆典**在此举行。
- **亚坛高原与王城罗德尔**：黄金树脚下的王都；蒙葛特以“赐福王”之名守城；后期火烧后成灰城。
- **啜泣半岛 / 风暴山丘周边**：前期支线与城堡密度区。
- **火山官邸（格密尔火山）**：亵渎君王拉卡德与蛇神信仰、火山官邸刺客任务线。
- **星界 / 诺克隆 / 永恒之城**：地底与星光文明遗迹，与诺克斯人、夜与星相关。
- **巨人山顶 / 火焰大锅**：古巨人火焰与烧树仪式之地；梅琳娜或癫火可作火种。
- **法姆·亚兹拉**：远古龙王与神皮贵族的空中遗迹，解放命定之死的关键舞台。
- **海舍尔禁域 / 日荫城 / 圣树艾布雷菲尔**等：米凯拉、玛莲妮亚、蒙格相关深层线。
- **幽影之地**（DLC）：被玛莉卡帷幕隔绝的黄金树阴影之陆，梅瑟莫、神兽、角人与神人米凯拉的“慈爱”计划舞台。
- **安格斯大矿道 / 希芙拉河 / 安舍尔河**：纵贯地底的水道与矿脉，连接永恒之城、鲜血王朝边缘与王城地下根须；是罗葛尔、蒙格线与诺克斯遗迹的空间骨架。
- **日荫城与圣树艾布雷菲尔**：米凯拉营造的“无垢”圣域与女武神沉睡之处，腐败与纯白建筑并置，象征慈爱与灾厄同源。
- **化圣雪原与米凯拉圣树外围**：通往巨人山顶与禁域的高寒走廊，古巨人尸骸与火焰僧团遗迹提示烧树并非首次动用“火”。

势力地盘：黄金树正统（王城、双指、圆桌）、卡利亚魔法王族、红狮子军（拉塔恩残部）、火山官邸、鲜血王朝（蒙格）、圣树与女武神、野兽神官与古龙信仰、诺克斯/星月派、癫火教团、黑刀刺客残党等犬牙交错。交通逻辑上，宁姆格福是所有路线的辐射源；利耶尼亚以湖面与学院关卡控魔法资源；盖利德因腐败成为“英雄坟场”却仍吸引碎星荣耀；亚坛高原以升降机与正门双重锁死王城。低阶契约者若扬言单挑拉塔恩或玛莲妮亚，本地人会当疯子——这是正确反应。AI写场景须先定区域气候与赐福密度：宁姆格福尚有商队与教会残灯，盖利德则连鸟都染猩红，王城在烧树前金碧辉煌、烧树后只剩灰与未熄的余烬祷告。

**【世界剧情线】**
① **神话前史 · 黄金律法建立**  
更古时代有巨龙、古王、外神与诸神战争。无上意志遣星兽（艾尔登之兽）降临，玛莉卡成为神人/神王容器，建立黄金树与黄金律法。黄金树不只是信仰图腾，更是律法的根系网络：赐福之光引导英雄，死者也曾被纳入树的循环。王夫谱系复杂：霍斯劳一系与卡利亚联姻带来拉塔恩、菈妮、拉卡德等半神；拉达冈作为黄金律法的战士象征，最终与玛莉卡呈现“同一神性容器两面”的结构。黄金葛德文作为“完美”的黄金血脉象征，地位极高，是黑刀之夜的首要目标。玛莉卡从法环中剥离并封印**命定之死**，使神与受赐福者一度远离真正的终焉——这一决定既造就黄金时代的不死荣光，也为后来的死诞灾难埋下伏笔。古巨人战争、龙飨仪式、野兽神官传统等前史，以遗迹与骨灰的形式残留在巨人山顶与法姆·亚兹拉。

② **黑刀之夜 · 法环破碎**  
黑刀刺客盗用命定之死，刺杀黄金葛德文，使其灵魂先死、身体成“死诞”根源——此后死诞者无法真正安息，成为交界地的疫病式存在。同夜，神人菈妮抛弃肉身、将自身灵魂移入人偶，以逃脱双指对神人命运的操控；她的阴谋与自由意志成为暗月结局的伦理核心。玛莉卡随后以神锤打碎艾尔登法环；拉达冈试图以黄金律法修复，失败，法环碎片化为大卢恩，落入诸半神之手。半神们未成为王者，反而因大卢恩互相征伐——**破碎战争**：盖利德化为拉塔恩与玛莲妮亚的修罗场，猩红艾奥尼亚污染整片大地；王城内外恶兆、贵族与叛军反复易手；火山与地底各自开出亵渎与鲜血的旁支王朝。无上意志弃置交界地，赐福引导一度熄灭，褪色者被流放；后来引导重新闪烁，召回褪色者争夺王位。开场旁白点名同期被召者包括荷莱·露、光耀金面具、死眠少女菲雅（Fia）、食粪者与百智爵士基甸·奥夫尼尔等，暗示“称王”从一开始就是多方竞速。

③ **开篇 · 候王礼拜堂与漂流墓地**  
褪色者在候王礼拜堂苏醒，见指头女巫已死。建言：“即使引导已经破碎，也请您当上艾尔登之王。”经漂流墓地教程区进入宁姆格福：此地仍有教会残灯、士兵巡逻与接肢流言。与**梅琳娜**相遇是开篇最重要契约——她请褪色者带她至黄金树脚下的王城罗德尔，作为交换给予灵马托雷特，并教会以卢恩强化自身。梅琳娜不谈自己的来历，只谈“引导”与“约定”；她能否同行取决于玩家是否接受癫火等禁忌。点亮足够赐福后，梅琳娜开启通往**圆桌厅堂**之路。此阶段名场面：第一次在赐福处“把死亡换成进度”、第一次骑马越过风暴山丘、第一次听见史东薇尔城的角声。

④ **圆桌厅堂 · 双指与大卢恩门槛**  
圆桌厅堂是破碎后褪色者的政治沙龙与情报交易所，靠半神情报与互相试探维持运转。**百智爵士基甸·奥夫尼尔**掌控最全碎片君王档案，自称追求成为真正全知的王，实则畏惧神性回归；**女战士涅斐丽·卢**追求值得追随的王，厌恶卑劣接肢政治；**D**猎杀死诞，与**Fia**的死眠拥抱形成意识形态对撞；**罗葛尔**执着地底与王城根须；**食粪者**带来诅咒与“绝望”修复的阴影；厅堂深处是无上意志的使者**双指**与**解指老妪**。百智告知：击败第一位碎片君王并取得大卢恩后，可谒见双指。解指老妪说明：至少两枚大卢恩才能修补法环。此阶段世界状态=半神割据，王城封印仍在，褪色者尚是“被引导的猎犬”。契约者在此最容易被派去送死式侦察，也最容易用情报换资源。

⑤ **宁姆格福 · 接肢葛瑞克与初代碎片**  
史东薇尔城由**接肢葛瑞克**占据。葛瑞克本是流亡的黄金贵族后裔，以接肢拼凑力量、绑架与残虐闻名，城内遍布被接走肢体的牺牲者与恶兆牢笼。他是许多褪色者的第一座碎片君王城：攻城需处理大门、城堡内部贵族与恶兆狮等层层关卡。击败葛瑞克获大卢恩，圆桌气氛变化——有人祝贺，有人开始把你当竞争对手——双指路线正式开启。支线交织：啜泣半岛城堡、商人卡雷的铃与“会说话的尸体”式黑色幽默、边缘地流放者、兽人村落、墓地与英雄墓。**布莱泽**与**菈妮**线的早期钩子可在宁姆格福—利耶尼亚交界被触发：半狼骑士的忠诚与神人冷淡的差遣，为暗月结局埋第一颗钉。

⑥ **利耶尼亚 · 满月与学院**  
湖区通往卡利亚书房、学院城镇与雷亚卢卡利亚。**满月女王蕾娜菈**沉溺于重生卵与学院政治：她曾是卡利亚与黄金联姻的关键，后因律法与月亮的裂痕陷入精神创伤，把“重生”当作唯一救赎。学院门禁、辉石龙、卡利亚骑士与月亮仪式构成中期探索主体；玩家可在此学习辉石魔法、参与学院内部派系，并与蕾娜菈战斗或利用其重生权能（游戏机制上的属性重分配，在叙事中体现为卡利亚秘仪）。**菈妮**的暗月之路在此全面展开：人偶身、隐藏的神人身份、对双指的憎恶、需要群星复转才能推进的命运。取得第二枚大卢恩后，王城封印解除——无论第二枚来自蕾娜菈、拉塔恩、拉卡德或其他碎片君王，解指老妪都会承认你已具备“修补者”资格。

⑦ **卡片锚点 · 碎星拉塔恩庆典**  
盖利德红狮子城外，**恸哭沙丘**上举行碎星战争的“庆典”。半神**拉塔恩**将军曾是交界地最强武者之一：自幼学习重力魔法以驾驭巨马，后在破碎战争中与女武神玛莲妮亚大战。玛莲妮亚的猩红艾奥尼亚令盖利德永久腐败；拉塔恩为抑止腐败与自身“星之饥饿/疯厄”，被红发女巫**塞露维斯**等以咒术封印心智，沦为只余战斗本能的巨神。即便如此，他仍以巨弓与重力魔法**封印群星**，使命运星轨（含菈妮的命运）停滞——这是暗月线必须打破的物理锁。红狮子军残部不愿让将军无声无息烂死在沙里，于是以“庆典”为名召集褪色者与英灵，集体挑战拉塔恩。开战时他自天空化作陨石坠落；沙丘上可召唤的英灵包括布勒亚、亚历山大、布莱泽、帕奇等（视任务进度）。战斗过程充满重力崩陷、箭雨与近身巨剑，是开放世界中最典型的“多人英灵协作Boss战”。击败拉塔恩后：群星恢复运转；菈妮线与诺克斯/星命运线解锁；大卢恩入手；塞露维斯的悲剧得到某种了结；红狮子军的荣耀与虚空同时抵达顶点。契约者介入时，红狮子城内外的召集令、塞露维斯的药剂、是否在庆典前劫掠军需、是否保护亚历山大抵达战场，都是明确抉择点。

⑧ **火山官邸 · 亵渎与蛇**  
格密尔火山的**拉卡德**吞食“亵渎之蛇”，以自己的身体作为神之容器的另一种解法——不是修复法环，而是吃掉诸神与碎片。火山官邸由**塔妮丝**主理，招募刺客猎杀其他碎片君王；**莉雅**、伯纳尔等构成任务网，充满背叛与试探。拉卡德线提供“以亵渎成王”的歧路，与黄金律法正统、星月、癫火并列为道德与终局分叉。火山内部有古蛇、异端祷告与被吞食者的残渣；击败拉卡德可获大卢恩，也会关闭或改写多条NPC的存活条件。

⑨ **王城罗德尔 · 赐福王蒙葛特**  
梅琳娜在褪色者入城后感谢履约并暂别——她的“到罗德尔”之约完成，下一约是火。王城罗德尔是黄金树脚下的权力纪念碑：贵族区、下水道恶兆、神授塔方向的视野、以及最高处的王座路。恶兆之子**蒙葛特**以“赐福王”之名守护黄金树道路：他憎恨自己的恶兆血脉与被排斥的童年，却比任何贵族更忠诚于黄金律法，甚至愿意以假名与幻影在破碎战争中作战。击败蒙葛特后，黄金树以荆棘与拒绝回应一切王者：连守护者亦不得入内。这一“树拒诸王”是剧情质变——武力收集大卢恩不够，必须引入**火**与**命定之死**才能打破神性门锁。梅琳娜再约：前往**巨人山顶火焰大锅**取火，烧尽挡住道路的黄金树之棘；火种由她承担（默认线）。

⑩ **烧树 · 命定之死 · 灰城**  
通往火焰大锅的道路经过化圣雪原、古巨人战场与火焰僧团。大锅处，梅琳娜自焚为火种（若玩家已染癫火，则她决裂并誓言将来以命定之死相送）。黄金树燃烧，王城化为灰烬之城：街道被灰覆盖，未逃出的居民与士兵成为终局氛围的一部分。远古遗迹**法姆·亚兹拉**中，击败黑剑马尔基特相关守密与神皮贵族等，**解放命定之死**，使燃烧彻底、通往神性核心之路打开。此阶段世界状态=律法崩毁可视化，天空与王城同色，终局倒计时。烧树不可逆，是所有“王”结局的共同门槛（癫火则是以自身为火种的变体）。

⑪ **艾尔登之王战 · 拉达冈与艾尔登之兽**  
进入黄金树内部，先与**王夫拉达冈**决战——他是试图修复法环的黄金律法执行者，圣枪与律法战技体现“秩序的暴力”。战斗中揭示玛莉卡与拉达冈的合一神性：打碎与修复是同一存在的撕裂。拉达冈倒下后，**艾尔登之兽**显化，作为无上意志植入的活体法环/秩序本体，技能含引力、圣光与法环具现。击败后，褪色者站在可修复法环的位置，终局选择真正打开。

⑫ **多重结局（原作可选终局）**  
- **艾尔登之王（衰颓时代）**：直接修复法环，在破碎后的衰颓中称王，秩序延续但疮痍仍在。  
- **律法时代**：经金面具与矫治者科琳线，以更“完美/矫治”的律法修复，试图修正无上意志的缺陷。  
- **潜藏者当道的时代**：经Fia与死王子（葛德文）线，让死诞纳入秩序，死亡与生命的边界被改写。  
- **绝望时代**：经食粪者诅咒线，以诅咒修复，世界落入怨念的循环。  
- **星之时代**：完成菈妮暗月线（含碎片任务、诺克斯、神皮、弃指等），成为她的暗夜之王/伴侣，带交界地脱离无上意志的指头秩序，走向星与月的漫长旅途。  
- **癫火之王**：接受三指癫火，烧尽一切，包括黄金树与自身理性的秩序，化作癫火之主；梅琳娜若在，将以命定之死狩猎此王。  
结局谁活谁死取决于线路：梅琳娜可能已焚、可能决裂；菈妮离去或同行；半神大多已作为Boss倒下；交界地秩序被重写或彻底烧毁。

⑬ **黄金树幽影（DLC主线提要）**  
被放逐的幽影之地中，**米凯拉**抛弃自身部分神性（包括腐朽与爱的片段），试图以“慈爱”建立新神秩序；其行军与茧化吸引无数被蛊惑的追随者。**梅瑟莫**奉玛莉卡之令对角人等发动圣战，后又遭遗弃，持“刺”与火，成为幽影之地的暴烈守门者。角人、神兽、幽灵花、修道院与影之城构成新地图政治。褪色者追寻米凯拉，最终直面其神性计划与“神”之位的代价：慈爱可以是救赎，也可以是剥夺自由的强制。DLC不替代本体终局选择，但加深玛莉卡原罪、黄金树阴影、梅瑟莫与米凯拉的真相，为高阶契约者提供第二大陆舞台。

⑭ **破碎战争关键战役补述（可作中高阶切入背景）**  
除盖利德决战外，破碎战争还包括：王城攻防中蒙葛特以恶兆与黄金幻影周旋；火山与高原之间的亵渎渗透；各神授塔与大卢恩塔的驻守与失守；英雄墓地中失败王者的骨灰化。这些战役的残骸成为游戏内地下城与精英战的来源，也解释为何交界地“遍地王军却无王”。

⑮ **蝴蝶效应边界**  
契约者可改：某场精英战归属、某NPC存亡（如提前介入红狮子召集、保护或出卖塞露维斯相关利益、是否在火山签下红信）、非核心遗物归属、某条支线的触发顺序。不可无代价改写：黑刀之夜与法环已碎的既定前史、拉塔恩封印群星的物理事实（除非重演碎星战）、玛莲妮亚与拉塔恩之战的历史结果、无上意志与艾尔登之兽的规则级存在、玛莉卡打碎法环的动机级事件。改了局部，因果以追杀、代行者、外神注视、圆桌排挤或半神点名回流。

## 阶位切入点

> 阶位↔交界地战力：一阶≈流民士兵与初级流浪者；二阶≈骑士精锐/低阶魔法祷告使用者；三阶≈名城将领与英雄级Boss门槛；四阶≈精英英雄/初阶半神代行；五阶≈中位碎片君王；六阶≈顶级半神（拉塔恩/玛莲妮亚级）；七阶≈王城后·烧树前后的世界级舞台。世界顶点（拉达冈、艾尔登之兽、外神权柄）为超阶巅峰：存在·情报优先/条件性胜利；低阶一律规避。

**一阶（流民·宁姆格福求生线）**

切入身份/时点：契约者以刚被引导召回的**褪色者新兵**或随商队混入的流浪佣兵身份切入，锚定离开漂流墓地、尚未攻打史东薇尔城、仅点亮宁姆格福数处赐福之时。

初始事件：风暴山丘下，一队史东薇尔士兵与流浪者争夺商队的卢恩与干粮；**商人卡雷**的铃铛声在雾里响起，有人高喊“接肢王又在城里接新人的手臂”——你必须在 complement 逃进教会废墟点亮赐福、或跟着士兵送死式冲城之间做选择。

开场白建议：「宁姆格福的风里有腐烂与金光两种味道。你手腕上的赐福细如残烛，远处史东薇尔城墙像一只张开的接肢。有人在喊卡雷，有人在喊葛瑞克——而你口袋里只有刚够买一瓶红露滴的卢恩。」

关键NPC立场：**商人卡雷**——卖铃与基础货，怕强者但给新人活路；**接肢葛瑞克**——城中暴君，视底层褪色者为接肢材料；**梅琳娜**——尚未深交时只远远观察谁配得上托雷特；**帕奇**——可能在洞窟设陷阱劫财；**圆桌传讯者（间接）**——尚未接纳你。

主线钩子/支线：主线钩子＝是否参与对史东薇尔的试探进攻（蝴蝶：提前削弱守军或打草惊蛇）；支线＝教会寻赐福、洞窟清巢、保护卡雷一次交易、猎杀恶兆新生体。

危险度/规避：中——规避葛瑞克本体、飞翔鹰群精英与任何半神名号；城内接肢贵族可致死。

任务方向/奖励：活过第一周、点亮区域赐福、入手基础武器战灰与少量卢恩；奖励取向＝初阶武器、红露滴杯相关、地图残片、托雷特任务前置情报。

**二阶（骑士·史东薇尔与湖岸立名线）**

切入身份/时点：以成名流浪骑士或学院外围佣兵身份切入，锚定史东薇尔攻防白热化、利耶尼亚湖岸已可望见魔法辉光之时。

初始事件：攻城临时同盟在城门前因“谁先拿葛瑞克大卢恩”内讧；**涅斐丽·卢**或另一名圆桌相关战士要求你表态——助攻城、卖情报给城内、或趁乱窃取城中贵重战灰。

开场白建议：「城门被砸出裂缝时，你看见接肢在墙头晃。同盟里每个人眼里都是大卢恩的反光。涅斐丽说真正的王不该靠接别人的手——可现在，不伸手的人会被踩进护城河。」

关键NPC立场：**接肢葛瑞克**——垂死反扑；**涅斐丽·卢**——试探你是否配称王；**百智爵士基甸·奥夫尼尔**——远距收情报，给任务换情报；**蕾娜菈（遥闻）**——湖区重生传说开始流传；**罗葛尔**——可能买你的城内路线图。

主线钩子/支线：主线＝葛瑞克讨伐战与第一枚大卢恩归属；支线＝城中地下墓穴、湖岸猎杀辉石生物、为圆桌带回半神情报残卷。

危险度/规避：高——规避葛瑞克完全体硬刚若装备不足；规避任何“去盖利德找拉塔恩”的怂恿。

任务方向/奖励：取得或参与分配第一枚大卢恩相关战利；奖励取向＝中阶武器、盾反战灰、初级魔法书、圆桌准入。

**三阶（英雄·双指门槛与利耶尼亚线）**

切入身份/时点：以已持一枚大卢恩的英雄级褪色者身份切入，锚定谒见双指前后、学院门禁与卡利亚政治发酵时。

初始事件：圆桌深处**解指老妪**宣布第二枚大卢恩的必要性；同时**满月女王蕾娜菈**的重生仪式被学院派系利用，有人出高价买“能进雷亚卢卡利亚的假身份”——你被卷进学院门禁战与圆桌派系站队。

开场白建议：「双指无声，老妪的嗓音却像律法本身。她说一枚大卢恩只是门票，两枚才是钥匙。厅堂外有人议论蕾娜菈的卵，有人议论王城的封印——你意识到自己已被写进半神的棋谱。」

关键NPC立场：**解指老妪**——转述双指，冷淡；**蕾娜菈**——沉溺重生，可被挑战亦可被利用；**菈妮**——若已接任务，以人偶之姿冷淡下达星月差遣；**基甸·奥夫尼尔**——用知识换你的服从；**D与Fia**——在厅堂形成死诞观念冲突的背景压强。

主线钩子/支线：主线＝第二枚大卢恩路线选择（学院/火山/盖利德等）；支线＝卡利亚书房潜入、死诞猎杀或庇护、火山官邸招募信。

危险度/规避：高——规避拉卡德本体、拉塔恩沙丘、玛莲妮亚；学院内辉石龙与满月战需准备。

任务方向/奖励：稳定英雄级配置、打开王城封印前置；奖励取向＝学院魔法、卡利亚剑阵、重生系统使用权（游戏内蕾娜菈重生逻辑的叙事对应）、中高阶防具。

**四阶（精英·火山与地底歧路线）**

切入身份/时点：以可接火山官邸契约或地底探索的精英猎手身份切入，锚定第二大卢恩入手前后、王城将开未开。

初始事件：**塔妮丝**以拉卡德之名发出刺杀其他碎片君王的红信；同夜**蒙格**的鲜血教团在地底留下祭祀残骸。你必须在“签官邸契约”与“拒绝并可能被灭口”之间抉择，且情报显示盖利德的庆典召集已散开。

开场白建议：「火山的硫磺味混着契约墨水。塔妮丝微笑着问你：敢不敢杀半神。另一封没有署名的信只画了三滴血——蒙格的记号。你忽然懂了基甸为何说“知识是唯一不会接肢的力量”。」

关键NPC立场：**塔妮丝**——官邸管家，任务与灭口并重；**拉卡德**——尚未见面时已是亵渎象征；**蒙格**——地底王朝的遥威胁；**莉雅**——官邸任务链中的关键棋子；**罗葛尔**——地底向导型盟友。

主线钩子/支线：主线＝是否加入火山猎杀网（影响拉卡德线与多NPC命运）；支线＝永恒之城探索、诺克斯遗产、为碎星庆典做情报准备。

危险度/规避：高~极高——规避拉卡德真身、蒙格本体；官邸内叛变任务可能反噬。

任务方向/奖励：拿下第二条大卢恩路径或等价权柄；奖励取向＝亵渎系武器、鲜血抗性装备、地底地图、高阶战灰。

**五阶（碎片君王门槛·碎星拉塔恩庆典线）**

切入身份/时点：以受红狮子军承认的参战褪色者/佣兵队长身份切入，锚定**碎星拉塔恩庆典**正式开幕前夜至恸哭沙丘集结时——卡片锚点本阶主打。

初始事件：红狮子城头吹响号角，**贾伦**宣读庆典规则：召唤英灵、共享荣耀、狩猎封印群星的将军。**塞露维斯**以咒术维持拉塔恩残存的一丝清明；沙丘上已能看见重力扭曲。有人想偷走赛前祭品卢恩，有人想在陨石落下前逃走——你被点名担任左翼召唤协调或先锋。

开场白建议：「盖利德的天是红的，沙丘却在发抖。号角一响，半神将如陨石坠下。塞露维斯的声音沙哑：请让他作为将军死去。你握着召唤铃，不知道下一秒落地的是荣耀，还是被重力撕开的尸体。」

关键NPC立场：**拉塔恩**——疯狂的碎星将军，战斗中无交涉；**塞露维斯**——乞求体面之死；**战士壶亚历山大**——兴奋参战可同盟；**布莱泽**——可被召唤的可靠狼骑；**帕奇**——可能混进后勤赚钱；**玛莲妮亚（历史投影）**——本地人不断复述她与拉塔恩的大战，形成心理压强。

主线钩子/支线：主线钩子＝碎星庆典本战（群星是否复转、菈妮命运线是否解锁）；支线＝红狮子城后勤保卫、腐败沼泽取样、战前英灵联络（亚历山大/布莱泽等）。

危险度/规避：极高——拉塔恩≈六阶顶半神，五阶需组队/召唤/战术条件性胜利；严禁宣称单挑无伤。规避更北的圣树女武神。

任务方向/奖励：参与击坠拉塔恩、分享大卢恩与重力战灰级战利；奖励取向＝拉塔恩相关武器记忆、红狮子军功名、星轨复转后的暗月线钥匙、高阶卢恩。

**六阶（顶级半神·圣树与女武神线）**

切入身份/时点：以粉碎星后声名大噪的半神级猎手身份切入，锚定盖利德战役余波、圣树艾布雷菲尔与玛莲妮亚沉睡传说被证实可挑战之时。

初始事件：圣树根下，**女武神玛莲妮亚**的花海张开；有人带来米凯拉的蛊惑遗物，声称“只要击败女武神就能靠近神人”。猩红艾奥尼亚的孢子在空气中跳舞——你的盟军在进门前已有人开始咳血。

开场白建议：「花瓣落下时没有香，只有金属与腐朽。玛莲妮亚单膝点地，义肢刀反光如月。她说她从未被击败——而你带来的所有大卢恩，在她眼中只是下一场舞的节拍。」

关键NPC立场：**玛莲妮亚**——好战且悲壮，几乎不结盟；**米凯拉**——缺席的神性中心，全程被提及；**蒙格**——若未处理则仍绑着米凯拉线；**基甸·奥夫尼尔**——愿出天价买女武神战报；**梅琳娜**——提醒王城与烧树才是正途，勿沉溺半神收藏。

主线钩子/支线：主线＝玛莲妮亚战与米凯拉相关物归属；支线＝圣树内部净化/堕落选择、蒙格王朝突入、为王城决战做最后大卢恩凑齐。

危险度/规避：极高——玛莲妮亚为顶级半神；规避拉达冈/艾尔登之兽；腐败叠加可团灭。

任务方向/奖励：女武神级战利与圣树资源；奖励取向＝女武神武器、艾奥尼亚抗性、米凯拉针相关、接近王级的配置。

**七阶（王城·烧树·神座侧翼线）**

切入身份/时点：以持复数大卢恩、可踏入罗德尔的准王者身份切入，锚定击败**蒙葛特**前后至火焰大锅、法姆·亚兹拉解放命定之死之间。

初始事件：黄金树拒绝蒙葛特的忠诚也拒绝你的大卢恩；**梅琳娜**在赐福旁重提火焰大锅之约。灰将落下之前，**基甸·奥夫尼尔**可能拦路索取“神之情报”，**Fia**或**食粪者**残党在灰城阴影里兜售另一种修复法。你要决定：随梅琳娜取火、改走癫火、或先去法姆·亚兹拉夺命定之死。

开场白建议：「罗德尔的金顶开始剥落。蒙葛特的咒剑还插在台阶上，树却把所有王关在门外。梅琳娜说：火种的事交给她。远方山顶有一口锅在等一个愿意燃烧的人——你知道，下一步不是称王，是先学会如何烧掉自己的路。」

关键NPC立场：**蒙葛特**——守城终局之敌或已倒下的警示；**梅琳娜**——烧树契约核心；**基甸·奥夫尼尔**——后期敌对可能；**黑剑相关守密者**——法姆·亚兹拉；**拉达冈 / 艾尔登之兽**——超阶顶点，仅终局条件战；**三指（若染癫火）**——彻底烧尽的诱惑。

主线钩子/支线：主线＝烧树→解放命定之死→神战→结局分支（王/星/癫火/死诞/诅咒/律法）；支线＝灰城搜救NPC、圆桌崩解后的立场清算、幽影之地入口级线索（若开启DLC时间线）。

危险度/规避：贴近顶点——拉达冈与艾尔登之兽为**超阶巅峰：存在·情报优先/条件性胜利**，严禁写其被随意削弱归零；七阶可参与烧树与侧翼，正面神战需尽一切条件。

任务方向/奖励：终局资格与时代选择权；奖励取向＝王级武器、命定之死相关权柄碎片、结局路线信物（暗月、癫火、死诞、完美律法等），禁止低阶提前发放。
