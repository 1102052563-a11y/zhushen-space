## 剧情

**【作品来源】**
《尘白禁区》（Snowbreak: Containment Zone）为悠星网络开发、西山居发行的第三人称射击角色扮演手游，2023年7月20日公测。舞台为近未来「海姆达尔部队」对抗「神御物」与「泰坦」灾害的封锁战。玩家扮演副官，指挥女性战术干员「绝弦」执行清剿与调查。文风：科幻军事、少女战术小队、都市封锁与神话科技并置。**以游戏主线公开资料为准**；查不到写不详。标签：海姆达尔、绝弦、神御物、泰坦、副官。

**【世界定位】**
近未来地球因「神御物」异常导致区域污染与泰坦级威胁，多地被划为禁区。海姆达尔部队以副官为核心，率领绝弦干员执行突入、回收与歼灭。一句话：封锁线内的科幻枪战，与副官并肩的绝弦们。

**【世界观 · 力量体系】**
力量本源：现代/近未来热武器、绝弦个体强化与技能、神御物衍生能力、载具与后勤体系。死亡：凡人与未强化士兵战死即灭；绝弦有后勤医疗与归队规则（以游戏呈现），世界顶点不可无限复活。特殊系统：副官指挥、小队编成、禁区突入评级。

战力阶梯（宁低勿高）：
1. **普通市民 / 后勤文职**：无战力。乐园观感≈一阶。
2. **常规士兵 / 禁区外围巡逻 / 低级污染体**：枪械格斗，车辆级破坏。乐园观感≈二阶。
3. **成熟绝弦干员 / 海姆达尔战术小队**：据点清剿、装甲突破。乐园观感≈三阶。
4. **精英绝弦 / 强化型污染体 / 小队Boss**：建筑群级交火、高机动突击。乐园观感≈四阶。
5. **泰坦级威胁 / 神御物核心灾变 / 城区封锁崩溃级**：城区级灾难。乐园观感≈五阶（本世界顶点段）。

乐园阶位映射：市民后勤≈一阶；士兵与低级污染≈二阶；常规绝弦≈三阶；精英绝弦与强化体≈四阶；泰坦/神御物核心≈五阶。宁低勿高；低阶切入规避顶点·情报优先；严禁用「泰坦临时关机」等战力归零式解释顶点。

**【地理 · 舞台】**
海姆达尔基地与训练场；城市封锁线与检查站；污染禁区废墟街道；神御物回收现场；地下设施与实验室；前线临时指挥所；民用避难区；弹药库与医疗舱；黑市抗污染剂交易点。AI写场景先定：基地日常 / 封锁线 / 禁区突入 / 回收战 / 泰坦警报，再定冲突强度。

**【世界剧情线】**
① **入队与副官定位**  
玩家以副官身份进入海姆达尔体系，认识核心绝弦，完成基础训练与首次禁区外围任务。世界状态：污染与封锁已成常态，强者如天是「编制与火力」。因果：指挥信誉从零开始。

② **禁区突入日常化**  
小队反复突入污染区清剿、护送科研、回收样本。名场面应具体到：哪条街失守、哪件神御物样本被争、哪名干员险些掉队。阶段结局：副官被更高层注视。

③ **神御物真相加深**  
任务从单纯清剿转向「神御物是什么、为何与神话意象绑定」。军方、科研、灰色回收商抢样本。伦理与评级冲突并行。

④ **泰坦级危机**  
泰坦级威胁逼近城区，常规火力不足，绝弦必须协同特殊战术。胜利昂贵：补给、伤亡、封锁线后撤。

⑤ **组织政治与信任**  
海姆达尔内部预算、权限、实验伦理冲突。副官须在「救人」与「完成评级任务」间抉择。

⑥ **主线推进方向（进行中）**  
作为运营游戏，主线随版本扩展；已公开核心是「副官+绝弦对抗污染/泰坦/神御物体系」。未实装终局细节写不详，禁止编造原作没有的终局人名与宝物来历。

⑦ **微观可介入**  
枪械保养、弹药配额、干员宿舍日常、避难区物资、假情报、黑市抗污染剂、训练考核——正文高频应是这些。

⑧ **蝴蝶效应边界**  
可改局部：提前疏散一条街、截一批黑市样本、救回一名掉队干员。不可无代价改写：神御物危险逻辑、泰坦级破坏力、副官「指挥负责」底色。

**【世界剧情线·作战节奏档案】**
禁区任务通常分：简报→突入→目标点交火→异常事件（污染潮/泰坦信号）→撤离评级。绝弦技能与属性克制决定清场效率；副官不亲自当无脑冲锋肉盾（除非叙事需要前线同步）。污染值与封锁等级是世界气压计：等级升高，市民撤离半径扩大，黑市涨价，科研抢人。

**【世界剧情线·样本争夺政治】**
神御物样本同时是科研圣杯与战场诅咒。海姆达尔要合规回收；科研要完整数据；黑市要溢价；媒体要英雄叙事。契约者若只站火力，会错过情报战；若只站科研，会被当成搬运工。评级系统把「救人」量化成分数——分数不够，下一场支援延迟。

开场白建议：「消毒水味盖不住恐慌汗味。广播重复『请保持秩序』，扩音器却滋滋作响。一名绝弦从你身边跑过，枪机上还挂着训练场的尘。」

关键NPC立场：**副官**快速评估你是否碍事；**后勤医官**要你别添乱；**幸存者代表**抓住你的袖口；**灰色回收商**塞来名片。

主线钩子/支线：主线=样本夹带事件；支线=安抚避难区、追查假药。

危险度/规避：低~中。规避禁区深部与泰坦信号源。

任务方向/奖励：通行证、基础防护、人情；勿发精英武装。

**二阶（外围清剿 · 污染体）**

切入身份/时点：以协勤士兵或见习协力者切入，锚定首次外围清剿。

初始事件：巡逻队失联，你与绝弦小队被派去接应，污染雾浓度上升。

开场白建议：「雾是绿色的，通讯是断的。路面上的弹壳还烫。你的面罩计数器开始跳红——有人在雾里笑，那声音不像人。」

关键NPC立场：**绝弦突击手**要你跟上；**上级军官**要评级；**友军队长**抢功；**污染体**无谈判。

主线钩子/支线：主线=接应与清剿；支线=救伤员、夺回被抢弹药。

危险度/规避：中。规避强化体与泰坦。

任务方向/奖励：制式枪械改装、抗污染剂配额。

**三阶（禁区突入 · 绝弦级）**

切入身份/时点：以成熟绝弦级或副官特派协力者切入，锚定正式禁区突入战。

初始事件：目标建筑内神御物样本被多方盯上，突入口被炸毁，必须改道。

开场白建议：「简报板上的红圈比你的心跳还密。绝弦调整枪灯：『副官说改道——你要么当包袱，要么当第二把枪。』废楼里有人先开了一枪，弹道带着金粉。」

关键NPC立场：**副官**要结果；**科研负责人**要样本完整；**回收商**要你卖情报；**精英污染体**堵门。

主线钩子/支线：主线=样本回收；支线=防止友军误击、保护市民通道。

危险度/规避：高。规避泰坦本体。

任务方向/奖励：绝弦级改装件、样本分析残页。

**四阶（精英战 · 强化威胁）**

切入身份/时点：以精英小队成员切入，锚定强化型污染体与多方争夺战。

初始事件：两支部队为同一核心碎片交火，泰坦远距信号出现。

开场白建议：「夜视仪里全是曳光。有人喊停火，有人喊抢。远处地平线像被什么巨大的东西踩塌了一截——那是信号，不是山。」

关键NPC立场：**海姆达尔上级**要优先权；**绝弦精英**要你选边；**媒体**架起镜头；**强化体**猎杀指挥链。

主线钩子/支线：主线=碎片争夺与撤离；支线=假情报战、伤员转运。

危险度/规避：极高。泰坦=规避顶点·情报优先。

任务方向/奖励：精英权限、高阶药剂、观测数据。

**五阶（泰坦级 · 城区灾难）**

切入身份/时点：以能影响城区疏散的决策协力者切入，锚定泰坦逼近城区。

初始事件：封锁线崩溃，你被点名为「临时前线协调」——神御物核心同时失控。

开场白建议：「防空警报把城市撕成两半。副官的终端只剩一行字：条件性胜利。绝弦们的呼吸声在频道里同步，像倒计时。」

关键NPC立场：**副官**要你守住撤离窗口；**科研**要冒险回收；**泰坦级威胁**碾压一切常规火力；**幸存者代表**要你保证孩子先走。

主线钩子/支线：主线=疏散+条件性压制；支线=核心临时封印、舆论稳定。

危险度/规避：贴近顶点。**超阶巅峰：存在·情报优先/条件性胜利**，严禁「泰坦临时关机」等战力归零式解释。

任务方向/奖励：终局见证权、高阶战术权限、核心封印技术碎片；不发一键灭泰坦外挂。
