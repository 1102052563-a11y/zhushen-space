# 绝区零（ZZZ）

## 剧情

**【作品来源】**
《绝区零》（Zenless Zone Zero）为米哈游开发、2024年7月4日全球公测的动作角色扮演游戏，带有roguelike探索与都市幻想风格。故事舞台为后末日城市「新艾利都」，玩家扮演绳匠「法厄同」（哲/铃兄妹）。制作人李振宇，音乐HOYO-MiX。主线分季更新：第一季（序章至尾音「将眼泪与过往一同埋葬」）与第二季（「云霞同归处」起）持续推进旧都陷落与称颂会阴谋。文风：街头赛博、阵营群像、空洞灾难与人情日常并置。**以游戏主线公开剧情为准**。标签：法厄同、空洞、以骸、代理人、新艾利都、零号空洞。

**【世界定位】**
灾难后幸存的最后大都市「新艾利都」。空洞（超自然维度灾害）吞噬旧文明，以骸自空洞涌出。哲与铃表面经营六分街录像带店「Random Play」，暗中以代号「法厄同」做非法空洞向导（绳匠），用HDD系统远程操控邦布「伊埃斯」为代理人导航。真正目标是调查旧艾利都废墟上的「零号空洞」与老师卡洛丝·阿尔娜的清白。一句话：录像带店里的绳匠，与城市边缘的空洞战争。

**【世界观 · 力量体系】**
力量本源：以太（空洞内异常能量）、代理人战斗技术、邦布与HDD远程支援、希人/仿生人/赛博格体质差异。死亡：凡人与低抗蚀者在空洞中侵蚀致死或心智崩坏；代理人靠抗蚀与战术存活，无无限复活；邦布可修复但非无敌。特殊系统：绳匠导航、连携技、电视模式迷宫探索、阵营委托。

战力阶梯（按破坏力/对街区威胁，宁低勿高）：
1. **普通市民 / 店员 / 非战斗后勤**：徒手或工具，无抗蚀。乐园观感≈一阶。
2. **治安局基层警员 / 外环混混 / 赤牙帮打手 / 初级以骸**：枪械与近战，可毁车辆级。乐园观感≈二阶。
3. **成熟代理人小队 / 狡兔屋主力 / 白祇工程武装 / 中型以骸**：裂石断甲、连携技清场据点。乐园观感≈三阶。
4. **对空六课精锐 / 虚狩级（星见雅） / 大型以骸与牲鬼残影 / 称颂会武装仪式**：一击掀翻建筑局部、空洞内高强度作战。乐园观感≈四阶。
5. **牲鬼完全威胁 / 旧都级灾变核心 / 布林格牲鬼化 / 零号空洞深层异常**：街区至城区级灾难、系统性污染。乐园观感≈五阶（本世界常用顶点段）。

乐园阶位映射：市民与后勤≈一阶；基层武装与初级以骸≈二阶；常规代理人小队≈三阶；虚狩/对空精锐/大型以骸≈四阶；牲鬼化灾难与零号深层≈五阶。宁低勿高；世界顶点=零号空洞真相与牲鬼级灾变，低阶切入一律规避顶点·情报优先；严禁用「空洞临时关闭」等战力归零式解释顶点。

**【地理 · 舞台】**
- **六分街 / Random Play**：日常据点，录像带店即绳匠基地。
- **新艾利都各区**：雅努斯区、外环、澄辉坪、云巋山周边、卫非地莱姆尼安空洞附近。
- **空洞内部**：死路空洞、芭莱大厦被吞没区、燃晶火湖、莱姆尼安空洞、废弃地铁与矿区。
- **旧艾利都废墟 / 零号空洞**：十年前灾难核心，主线终局锚点。
- **势力据点**：狡兔屋事务所、白祇重工工地、维多利亚居家服务宅邸、对空六课本部、治安局刑侦特勤组、卡吕冬之子外环营地、云巋山门派。

AI写场景先定：店面日常 / 街区委托 / 空洞深潜 / 阵营联合作战 / 零号阴影，再定冲突强度。

**【世界剧情线】**
① **序章·生意×诡异×道义**  
哲铃受妮可·德玛拉委托，追回被赤牙帮盗走的保险箱。安比·德玛拉与比利·奇德坠空洞，保险箱被神秘以骸劫走；HDD被骇客挟持，资料清空。读保险箱数据后，AI「Fairy」强制与法厄同缔约并占领HDD。世界状态：绳匠身份曝光风险骤升。

② **第一章·猫的失物**  
猫宫又奈（猫又）因赤牙帮恩怨接近狡兔屋，任务中被善良打动并加入。远景实业在死路空洞爆破，狡兔屋被困；法厄同停「无人」炸药列车、揭穿伪装警察，救出市民，与猫又合力擒获执行长查尔斯·珀尔曼，远景实业因丑闻瓦解。因果：法厄同从「接私活」卷入企业黑幕。

③ **第二章·在空洞中心呼唤**  
白祇重工委托搜索失控智能工程机械。信号来自已故创办人霍尔斯·贝洛伯格；珂蕾妲·贝洛伯格发现父亲未卷款潜逃，原型机满是弹孔。击败封印纪念碑中的异界生物「牲鬼」残影。与治安局副司令贾斯汀·布林格、朱鸢、青衣联手粉碎证物窃取阴谋。因果：牲鬼与旧案浮出。

④ **第三章·危楼夜寻踪**  
妮可委托查骇客芮恩失踪，线索引至被空洞吞没的芭莱大厦。飞空艇遇劫，维多利亚居家服务（莱卡恩等）协助救出芮恩，珀尔曼趁机逃离。

⑤ **第四章·火狱骑行**  
外环「火狱骑行」机车大赛。法厄同助卡吕冬之子击破卢修斯诡计；凯撒·金英勇清燃晶火湖以太污染（代价沉重）。事后法厄同弔唁老师卡洛丝·阿尔娜，立志还她清白。

⑥ **第五章·星流霆击与尾音**  
珀尔曼供出助手莎拉与布林格为幕后黑手。多阵营联手救出被剑灵附体的星见雅，在空洞击败牲鬼化布林格。法厄同与星见雅结盟追查旧城沦陷真相。莱卡恩请其替市长夺回拍卖中的牲鬼核心，升级HDD与义眼抗蚀。斡旋反舌鸟雨果·维拉德、拉文洛克、薇薇安·班希等，制止蒂娜以太实验、对抗称颂会，取回牲鬼核心；法厄同觉醒以太操纵倾向，受市长引荐赴云巋山投奔仪玄。尾音「将眼泪与过往一同埋葬」收束第一季情感线与旧伤。

⑦ **第二季·云巋山与称颂会**  
市长告知云巋山在莱姆尼安空洞回收到卡洛丝照片。仪玄收兄妹为徒，飞船遭莎拉击落，抵澄辉坪。矿工因洛尔医生（称颂会劣质抗蚀药）被以太侵蚀；法厄同深入空洞击杀（表面）梅若拉可司教并回收神秘装置——实则司教未死，与莎拉密谋。辉晶美克逼爱丽丝·泰姆菲尔德参与调查；与浮波柚叶、狛野真斗等揭人体实验，破坏秽息妖鬼核心，爱丽丝公开让渡缓蚀药专利，解除家族威胁。后续章节（迟吟的浪涌、不要温和地走入那良夜、可曾记得梦、将临未抵的深渊、微光引灯时、间章旧梦安可曲、尾音新艾利都日落时）持续推进称颂会、旧都真相与都市政治，**进行中主线以已公开章节为准，未公开细节写不详**。

⑧ **微观日常可介入**  
录像带出租、邦布维修、六分街委托板、阵营食堂与宿舍、抗蚀药黑市、外环机车改装、治安局笔录、云巋山功课——正文高频应是这些，而非一上来零号终战。

⑨ **蝴蝶效应边界**  
可改局部：提前救出某市民、截断劣质药一批、帮珂蕾妲藏住数据晶片、外环赛道多一辆掩护车。不可无代价改写：旧都陷落历史、牲鬼核心危险逻辑、法厄同「查清卡洛丝」底色、Fairy契约存在。改局部会以更狠追杀或空洞扩张回流。

**【世界剧情线·分章状态机】**
1. 公测序章：保险箱与Fairy。2. 赤牙/远景：猫又入队、珀尔曼落网。3. 白祇：霍尔斯真相、牲鬼初见。4. 芭莱大厦：芮恩与维多利亚。5. 火狱骑行：凯撒与卡洛丝追悼。6. 星流霆击：布林格倒台、牲鬼核心。7. 云巋山：仪玄、称颂会药祸、爱丽丝线。8. 后续版本：深渊与日落——高阶锚点。

**【叙事基调 · 雷区】**
街头轻快外壳+空洞残酷内核；阵营群像与委托单元优先。忌把哲铃写成无脑冲锋肉盾（他们是导航者）；忌无视抗蚀与侵蚀规则；忌低阶硬刚牲鬼完全体/零号深层；忌把所有希人当吉祥物而忽略战争史（月城柳线）。最早切入锚点：六分街接第一单狡兔屋委托前后。

**【世界剧情线·代理人作战与绳匠规则】**

主线钩子/支线：主线=保险箱委托外围跑腿；支线=帮猫又打听赤牙帮动向、鉴定一瓶来路不明抗蚀药、护送邦布零件过治安局临检。

危险度/规避：低~中。规避空洞深区与称颂会仪式场。

任务方向/奖励：绳匠联络方式、六分街人情、初级抗蚀喷剂；勿发虚狩级武装。

**二阶（赤牙与远景 · 企业黑幕边缘）**

切入身份/时点：以能轻微战斗的协力代理人、治安局协勤或外环信使切入，锚定猫又入队至珀尔曼被擒前后。

初始事件：死路空洞传来爆破倒计时，市民被困废弃地铁；你被要求在「停炸药列车」与「先撤平民」之间选优先级——伪装警察已就位。

开场白建议：「隧道里的灰尘尝起来像铁锈。广播说列车无人，对讲机里却有人在数弹匣。猫又的耳朵转向黑暗：『……有人在哭。也有人在装哭。』」

关键NPC立场：**猫宫又奈**半信半疑；**朱鸢**要求合法程序；**查尔斯·珀尔曼**试图收买；**青衣**平静记录你的异常生理数据。

主线钩子/支线：主线=协助揭穿远景实业；支线=截获爆破遥控器、护送被困市民、防止HDD再次被骇。

危险度/规避：中。规避牲鬼封印点与布林格核心圈。

任务方向/奖励：治安局临时通行证、A级委托分成、抗蚀升级模块。

**三阶（白祇与维多利亚 · 牲鬼阴影）**

切入身份/时点：以成熟代理人或绳匠认证协力者切入，锚定霍尔斯真相与芭莱大厦夜。

初始事件：三台失控工程机械收到已故创办人信号；纪念碑下牲鬼残影苏醒。同时芮恩失踪线把你拽上遇劫飞空艇。

开场白建议：「工地探照灯把雨丝切成白线。珂蕾妲把数据晶片攥到指节发白：『这不是卷款——这是弹孔。』电梯井深处，有什么东西用人类的频率在哭。」

关键NPC立场：**珂蕾妲·贝洛伯格**要真相不要同情；**格莉丝·霍华德**把机械当孩子；**冯·莱卡恩**以礼待你但计价；**贾斯汀·布林格**微笑着要证物。

主线钩子/支线：主线=白祇真相与芮恩救援择一深挖；支线=保护晶片、飞空艇反劫、阻止珀尔曼二次逃脱。

危险度/规避：高。规避布林格完全摊牌后的牲鬼化与零号。

任务方向/奖励：重工武装试用、维多利亚家政合同、牲鬼资料残页。

**四阶（火狱与对空 · 旧都线索）**

切入身份/时点：以对空协力、外环大赛车手保镖或虚狩见习切入，锚定火狱骑行至星见雅剑灵危机。

初始事件：燃晶火湖污染失控，卢修斯设局；星见雅被剑灵侵蚀，多阵营临时同盟点名你入队。

开场白建议：「外环的风里全是轮胎皮与以太焦味。凯撒把头盔抛给你：『赛道不认身份，只认谁敢压过火线。』远处对空六课的旗刃切开夜空——狐狸耳朵的虚狩正摇摇欲坠。」

关键NPC立场：**凯撒·金**以命清污；**星见雅**警惕仍合作；**月城柳**分配战术；**莎拉**在暗处记录你的以太反应。

主线钩子/支线：主线=火湖清污或剑灵镇压；支线=卡洛丝名誉调查、市长拍卖会潜入、蒂娜实验阻止。

危险度/规避：极高。牲鬼核心与零号=规避顶点·情报优先。

任务方向/奖励：对空临时编制、机车特种改装、抗蚀义眼线索。

**五阶（牲鬼核心与称颂会 · 灾变级）**

切入身份/时点：以能影响城区疏散的决策协力者、以太亲和者或云巋山客卿切入，锚定牲鬼核心争夺至莱姆尼安药祸与司教线。

初始事件：核心拍卖场变战场；澄辉坪矿工成批侵蚀，梅若拉可司教的秽息诱捕网张开——你被市长与仪玄同时点名。

开场白建议：「新艾利都的天空像被撕开一条黑缝。Fairy在耳机里平静到残忍：『抗蚀归零倒数，三分钟。』仪玄的袖风掠过废矿：『收徒可以，送死不行——选。』」

关键NPC立场：**仪玄**考你心性；**爱丽丝·泰姆菲尔德**要专利正义；**梅若拉可司教**（及未死阴影）视你为祭品或变数；**哲/铃**请你「别变成下一个被埋葬的眼泪」。

主线钩子/支线：主线=核心保全/药祸揭露/称颂会据点摧毁（条件性）；支线=假死司教追踪、辉晶美克舆论战、零号外围测绘。

危险度/规避：贴近顶点。**超阶巅峰：存在·情报优先/条件性胜利**，严禁「空洞临时关闭」等战力归零式解释。

任务方向/奖励：云巋山秘传残章、牲鬼核心封印技术碎片、都市级通行权；不发「一键净化零号」外挂。
