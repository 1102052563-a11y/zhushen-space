# 王贼
<!--meta lib=休闲 tiers=休闲-->

## 剧情

**【作品来源】**
《王贼》（日文原题：王賊）是日本美少女游戏品牌 **ソフトハウスキャラ（SoftHouse Chara）** 于 **2007年6月29日** 发售的 **18禁幻想战争模拟＋恋爱／后宫** 作品。对应机种 Windows 98/Me/2000/XP/Vista；类型标签为 **ファンタジー戦争 SLG**，以大地图战役与女角色攻略并行为骨架。剧本 **内藤骑之介**，原画 **佐々木珠流・明音・Ｇ・むにょ**，主题歌「King for the day」由 **ことり** 演唱（作词作曲あきづきかおる）。据品牌相关访谈，本作一度是软屋角色销量最高的作品之一。气质：**乱世军略喜剧＋女癖军师的多角恋爱＋「救国／复仇／血缘」三重情感债**；有 H 场景，本档案只保留关系成立与情感钩子，不展开露骨描写。官方小说版（ハーヴェスト出版／影山二階堂，2007-10）为同世界观衍生，主档案以 **PC 游戏本篇** 为准。

**【世界定位】**
大陆正被 **ヴィスト王国** 的铁骑席卷。弱小 **エルト王国** 在女王与少将的撑持下濒临亡国边缘，把最后的希望压在一名女癖极差却军略通天的雇佣兵军师身上。核心不是「谁的军队更强」，而是：**一个被祖国追杀的前王子，如何在敌国军帐、女仆副官、敌国王女与诸国公主的环绕中，重新学会「被需要」与「选择站在谁身边」**。战争是舞台布景；真正可玩的是军帐夜谈、人质外交、俘虏后的和解，以及「救国英雄」与「王贼」两种身份在同一人身上的撕扯。

**【世界观 · 舞台设定】**
时代感为 **架空中世纪大陆**，诸国林立，和风领国与西式王国并存。表面上：征兵、行军、联合作战、城塞攻防、诸国使节往来。情感侧只保留其「日常／关系」一面——

- **エルト王国**：小国，连失要人后国土半壁沦陷；军帐里的酒、文书、伤兵与将领的私语，比城墙更常成为戏台。
- **ヴィスト王国**：扩张中的霸权国，王都礼仪与前线军团长的私心并存；「血缘」「王位」「父王的刀」是人物心结，而非数值榜。
- **オルトレア联合**：反ヴィスト诸国松散同盟，盟主 **奈宫皇国** 带和风侍从文化；加盟国之间的猜忌、求援与人质式联姻，是情感线的外交外皮。
- **神乐家领国（已灭）**：和风领国，曾以少数精锐「王虎」拖住大军三年；灭亡后的遗孤与旧部，把「亡国之恨」写成私人忠诚与伪装日常。
- **种族与奇谭（只写生活侧）**：「森の住人」、兽人族、龙族血脉、飞龙部队等，在档案里表现为 **口音、饮食、禁忌称呼、军团气质与恋爱喜剧的道具**（例如钓竿会让某兽人性格变样；某飞龙使因旧怨专盯军师），不写破坏力分级。

季节与氛围：军旗、篝火、泥路、城门税、伤愈后的第一次复职训话、庆功宴上的不合时宜玩笑。世界的「温度」来自——**阿尔耶特从「被骂无能」到「救国英雄」的成长掌声**、**八重用女仆微笑掩盖亡国公主的刀**、**莉迪亚在「兄长」与「敌军军师」之间的认知崩坏**、**内政大臣穆斯托用幼女外表压住整个王国的账本**。治愈感不是没有战争，而是 **在战争间隙仍有人替你留一碗热汤、一间可卸甲的帐**。

**【地理 · 生活舞台】**
- **埃尔特王都／军司令部**：阿尔耶特的办公桌、穆斯托的文书山、女王埃尔内亚的「装饰性」谒见厅；初遇军师、要求报酬（含肉体条款的喜剧谈判）、第一次成功演说与溃兵回流，多发生于此。
- **军帐・行军营地**：篝火边的策略夜谈、八重的「女仆问候」安抚狂乱精兵、俘虏交换与酒后失言；恋爱升温常在「胜仗后的疲惫」里。
- **旧神乐家领・边境废墟**：八重／七姬的故乡阴影；「王虎」旧日传说在本地人嘴里是恐怖童话，对军师与女仆却是共同记忆。
- **维斯特北方战线・莉迪亚军团驻地**：王女兼军团长的指挥所；第一次战场对峙、警告「别上前线」、身份识破的萌芽。
- **旧赫尔米纳／加恩德等被吞并领地**：地名在对话里反复出现，是军师生母出身与复仇动机的地理锚点。
- **奈宫皇国宫廷**：和风礼仪、皇子女争权；一叶以身体为筹码的「请君入瓮」式外交发生地。
- **诺伊尔王国**：魔法国气质的宫廷与战场；胆怯公主露蒂莫内的逃兵与「惩罚后反而依恋」的喜剧舞台。
- **霍克兰德战线・要塞都市**：联合军东侧飞地；援军派遣与单独媾和的外交戏，可写成「站在谁一边」的关系测验。
- **地方领主キューベル领**：爱国领主为保民而叛、被镇压后又被说服归队——「敌我」标签在人情里软化的样本地。
- **王都／古都（维斯特侧）**：终局父子对峙与真相揭晓的礼仪空间；情感上是「回家却不能回家」。

**【故事主线 · 情感线】**

**共通线 · 被诱捕的军师与濒死的小国**
大陆被维斯特王国鲸吞。雇佣兵 **ジン＝アーバレスト** 曾以独立部队「王虎」在神乐家领国翻弄大军，因而被诸国争抢。埃尔特王国用「先抓再雇」的粗暴方式把他绑到军帐——他反而因此起了兴趣。报酬谈判里，他除了金钱，还厚颜要求 **ムスト** 与 **アルイエット** 的身体（软屋角色传统色喜剧开场）。时年二十的女总大将阿尔耶特本是前将军之女、与战无关的大小姐，父亲战死后硬接军权，国土已失一半，演说时三万兵中近乎全员溃逃，落得「无能」恶名。军师入伙后，战线止跌回升；她本人也从「只会发抖的继承人」长成能站在旗下的将领。共通线推进：**整军—夺回国土—联合军协作—对维斯特诸军团的各个击破—身份谜底—终局决战**。分支由「与谁加深羁绊／俘虏后如何处置／外交选择」决定情感落点；DEAD 端存在，档案只标「失败即关系断裂」，不细写死法。

**阿尔耶特线 · 从「无能女将」到「救国英雄」的并肩**
主轴：**被全国嘲笑的继承者，如何在同一人面前先丢脸、再被托付、最后被承认**。阿尔耶特外表是标准大和抚子式将门之女，内里经验为零；军师用计谋与毒舌逼她学会站在士兵前面。关系推进：报酬条款的羞耻谈判 → 第一次小胜后的共同庆功 → 她发现自己开始在意「他是不是只把我当棋子」 → 真相层：她名义上是前将军阿尔巴尔之女，实为先王拉尔夫伍德托付给战友抚养的孩子，与王族无直接血缘，却在终局应女王埃尔内亚之愿成为埃尔特女王。攻略吃：**认真听她的不安、不在公开场合拆穿她的未熟、在胜仗后把功劳推回她身上、把「身体报酬」逐渐改写成「自愿的亲密」**。HE：她以救国英雄之姿与军师并肩，或即位为女王后仍选择他；情感落点是 **「无能之名被洗掉的那天，她先看的是你」**。BE／DEAD：公开资料细部不详处写「战败或关系破裂导致国灭／离散」，不编造。

**八重／七姬线 · 女仆微笑下的亡国公主**
主轴：**「请杀了我」被改写成「等你怕死之后再杀」的终身契约**。副官 **奈津山八重** 女仆装、永远微笑、武术与谍报一流，却不会家务尤其不会做饭（吃了会损害健康的程度）。她负责把埃尔特新兵练成「战场看起来像乐园」的精兵，并用「女仆问候语」平息被训练逼疯的士兵——军师为此既安心又不安。真相：本名 **神楽七姫**，灭亡神乐家领国的公主，幼时被当作暗杀对象，辗转藏于花山家、奈津山家；因领内背叛而亲手促成亡国导火线后，曾求当时的部下军师杀她。军师拒绝，条件是「等你重新害怕死亡之后」。于是她穿上女仆装、改用语癖、把刀藏进微笑。个人线写：伪装日常的崩裂、亡国记忆回流、对「主人」从忠诚到恋心的越界。攻略吃：**接住她的武力与脆弱、别把女仆服务当理所当然、允许她偶尔放下笑容、正视神乐家之名**。HE：她以七姬或八重之名留在军师身边；落点是 **「死的借口用完了，只好活下去」**。二周目起可作为部队使用（系统侧），情感上仍是「副官永不离帐」。

**莉迪亚线 · 「兄长」与「敌军军师」的双重曝光**
主轴：**从战场敌手到军门降将，再确认「ジン＝ラフィルト」**。**リディア＝アライゼル＝ヴィストランド** 是维斯特王女、北方军团长，军事才能出众。她与军师并无血缘——父王卡迪尔曾爱其母，为保「血脉叙事」让她与当时的王子拉菲尔特订婚。她自幼亲昵叫拉菲尔特「お兄様」。联合军大败后仅一面之缘，她便直觉「ジン就是拉菲尔特」；王都确认父王口供后笃信。旧赫尔米纳防线再败后，她降入埃尔特军。军师对她始终有情，曾警告她别上前线。个人线写：敌我旗号下的惦记、降将身份的屈辱与安心、兄妹婚约叙事被「失散重逢」改写为恋人叙事。攻略吃：**在战场上仍留活路、降后不折辱其尊严、承认「拉菲尔特」这个名字、让她从军团长身份里暂时做回「莉迪亚」**。HE：她以恋人／伴侣身份留在他身边，或在新秩序里继续持军但心有所属；落点是 **「哥哥死了，活下来的人可以改口叫名字」**。注意：她与军师名义上曾是订婚兄妹，原作中世纪王族语境下不罕见；档案写情感，不猎奇化。

**穆斯托线 · 幼女外表的内政怪物与「雇佣合同」**
主轴：**把军师「谈下来」的人，其实一直在用账本保护整个国家**。**ムスト** 外表幼小，因龙族血脉实际活了极久；没有她，埃尔特内政早已崩盘。国土只失一半而未全灭，很大程度靠她。雇佣谈判里她看似被军师占了便宜，实则把「色狼军师」稳稳锁进国运。个人线侧重：**公文桌对面的互刺、长生者对短命人的克制温情、把身体条款从交易改成选择**。攻略吃：尊重她的内政权威、别只当萝莉笑料、在军费与人命之间站到她能接受的一侧。HE：她继续掌内政，关系从合同变为共犯式伴侣；落点是 **「账本最后一页写的是你的名字」**。

**一叶线 · 皇女的身体外交与「被需要」的政治恋**
主轴：**奈宫皇国第三皇女用自己当筹码，却在过程中长出真心**。皇帝病重、兄姊争权，**一葉** 在家内被忌惮，被派来以身体拉拢军师加盟奈宫。个人线写：和风宫廷礼仪下的羞耻任务、对「自己是否只是道具」的自我厌恶、军师是否把她当人看。攻略吃：拒绝把她纯当外交礼物、给她在皇国斗争中的退路或尊严。HE：她脱离纯道具身份，与军师建立对等情感；落点是 **「筹码自己从棋盘上走了下来」**。

**露蒂莫内线 · 逃兵公主的惩罚与依恋**
主轴：**胆小到抛下部下逃跑的公主，在「惩罚」后反而黏上军师**。诺伊尔王国姬 **ルティモネ** 怯懦，战场逃逸激怒军师；之后她却因那次强制亲近而依赖上他。背负可瞬间移动的翼状魔导具，但需清晰想象目的地且一日只能充能一次——喜剧与「逃向你」的道具。攻略吃：严厉中留温度、练她站在旗下、别永久羞辱其王室身份。HE：她学会一点点勇气，依恋转为恋心；落点是 **「逃跑的终点变成你的帐帘」**。公开资料对独立长线细节较少处，以「依赖→自立萌芽」为弧，不编造未证实终局。

**其他可推进的情感支线（非完整独立长线亦写清关系）**
- **ネイ**：兽人，莉迪亚忠臣兼挚友，乌尔加族；钓鱼时性格大变。降后或同行时，是「守护王女」与「接纳新主」的友情／主从三角缓冲。
- **ケーニスクフェア**：飞龙军团长，森之住民；曾被军师凌辱而怀恨，知他在埃尔特便屡次袭击。关系弧是 **恨意→再败→强制或和解后的扭曲亲近**；忌把恨写成无脑对决比拼，写成「旧债如何被关系改写」。
- **エルネア**：埃尔特女王，先王妃，军事政治皆无才却民望极高、政敌极少；23 岁却被军师误认年长而大受打击。她是把阿尔耶特推上王位的关键推手，情感上是「无能却温柔的姐姐型君主」。
- **トール**：军师发掘的青年才俊，故乡有青梅竹马史黛拉；镜像「若你不曾失去一切」的年轻人。
- **ルカ**：内通贵族之女，却在解散宣言后仍留在军中，以自身换家族免罪——「罪债与身体」的平行样本，可作短线救赎。

**终局情感核 · 王贼之名**
军师本名 **ラフィルト＝ウィルマー＝ヴィスト**，维斯特第一王子、卡迪尔唯一亲生子。曾被父王突袭暗杀，靠部下与友人舍命逃出，此后以雇佣兵身份复仇。终局父子对决后揭晓父王动机（年轻时战场负伤导致无法再生育、对莉迪亚之母的偏爱等——以原作揭示为准）。情感落点不是「谁统一大陆」，而是：**被祖国叫作贼的人，是否还愿意为某一国、某几个人停下脚步**。各女主 HE 多为「战后仍在身边／共同治理或共同流浪」；DEAD 端为战败离散。无「推翻全部恋爱」的隐藏真结局；真结局感来自 **身份坦白后仍被选择**。

**【可攻略角色 / 主要人物】**

- **ジン＝アーバレスト／ラフィルト＝ウィルマー＝ヴィスト**（主人公／雇佣兵军师／29 岁）｜外貌：成年男性武者气质，军旅装束，不修边幅时更显痞气｜性格：军略天才、无类女好き、拘束系趣味、对真正在意的人会突然认真；嘴上要报酬，账上却常把小国往活路上推｜角色类型：色痞军师／失国王子／复仇者｜萌点：谈判要价、对莉迪亚的例外温柔、被八重「女仆问候」无语、对阿尔耶特又损又捧｜个人线：全线视点；真名揭露是所有关系的试金石｜与契约者：默认同位，或并行的客将／参谋，忌抢「拉菲尔特身份」与「王虎传说」核心。

- **アルイエット＝ウルザー**（可攻略／埃尔特军总大将／20 岁）｜外貌：将门之女仪态，军装与礼装切换；初登场更显未经战阵的柔｜性格：责任感强却经验为零，易被舆论击垮；成长后沉静可靠｜角色类型：成长型女将／大和抚子｜萌点：溃兵演说惨案、第一次胜利的泪、被要求「报酬」时的脸红、即位后的重量｜个人线：见上阿尔耶特线｜CV：羽高なる｜与主角：雇主→战友→恋人／君臣一体。
