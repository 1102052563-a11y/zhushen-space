## 剧情

**【作品来源】**
《结界师》（日文：結界師，英译 Kekkaishi）是日本漫画家**田边伊卫郎**（Yellow Tanabe / 田辺イエロウ，中文亦常称田边刚）创作的少年漫画，2003–2011 年连载于小学馆《周刊少年 Sunday》，全 35 卷、345 话，累计发行逾 1700 万部，获第 52 回小学馆漫画赏少年部门。2006–2008 年由 Sunrise 改编电视动画，全 52 话，主线覆盖至**黑芒楼终焉**前后；完整终局、裏会内战与乌森真相以漫画正典为准。类型：校园夜巡 + 灵异战斗 + 家族对立与成长；气质偏热血守护、略带哀伤与责任重压，战斗以空间结界的技巧、意志与情感为核，而非单纯数值碾压。本档案以漫画原作为准，动画作对照。卡片锚点：黑芒楼崩塌——异空间黑芒楼因力量失衡而崩塌，墨村良守在崩溃的空间中制造绝界，对抗试图吞噬一切的土地神相关余波与妖灾核心。

**【世界定位】**
现代日本都市的「影子灵异层」：表面上是私立乌森学园与周边住宅街的日常，夜幕一落则有妖怪（あやかし）被神祐地乌森之力吸引、强化并入侵。墨村家与雪村家两支结界师后裔世代夜巡，以「方围—定礎—结—灭/解」的空间术式捕捉并消灭妖怪。主角**墨村良守**从抗拒职责的中学生，成长为能以**真界**改写封印的守护者；搭档**雪村时音**以精湛技巧与他并肩。一句话：这是「守护神祐地乌森、揭开宙心丸真相、最终以牺牲与创造完成永久封印」的结界师世界常青档案。

**【世界观 · 力量体系】**
力量本源：异能者体内的灵力/妖力与对「空间」的操纵。结界师是异能者中专精空间的一脉——把空间切成可塑的障壁，用来捕捉、隔绝、压缩、消灭。妖怪（ayakashi）为灵异存在，可进化、再生；在**神祐地**停留越久越强。**混合妖**是天生带妖力的人类，肉体与感知常超常，亦更易失控。**土地神**是神祐地的核心高位存在，失地则神祐地力量崩坏或狂乱。死亡规则：凡人与低阶妖一击可死；强妖被「灭」后须开**天穴**把残骸送入异界，否则可能再生；高阶异能者可留精神体、夺容器续命（裏会总帅线），但终局仍有同归于尽与彻底封印，**不存在无限复活**。特殊系统：①结界基础三步——**方围**（指定目标）→**定礎**（划线定范围）→**结**（展开障壁困敌），再接**灭**（歼灭内容物）或**解**（解除）；②**念丝**绑缚灵体、从异空间拉拽；③**绝界**——间时守领悟的高阶破坏性结界，紫黑混沌域，拒他物、触即消，偏歼灭；④**真界**——近神级创造/排除结界，白域可护友排敌、构筑与回归空间，终局用以安置宙心丸；⑤**方印**——正统继承人印记（墨村多在掌、雪村时音在左胸），与乌森之力/宙心丸共鸣相关；⑥**无想**修行（无想盒/无想房）锤炼精神，破心障；⑦**波同**读波长穿透，奥义**空身**与力量同化、在异界自由行动；⑧**神祐地**（しんゆうち）——内含异界与唯一土地神的特殊土地，失神则失力。

战力阶梯（按破坏力表现，宁低勿高；名称取原作习惯，非修仙境界）：
1. **凡人 / 无异能**：普通学生、教师、路人。战力：徒手或冷兵器级；对妖几乎无力。乐园观感≈一阶初期。
2. **弱妖 / 见习结界师**：初成结界、小范围方围；弱妖可伤人、毁门窗。战力：超凡门槛下沿，毁木门、掀翻壮汉级。≈一阶中后期～二阶初期。
3. **合格夜巡结界师 / 中级妖**：稳定方围—灭循环，可塑矛、墙、平台；中级妖可裂石断金、威胁校舍一角。战力：正式超凡。≈二阶。
4. **强力结界师爆发 / 黑芒楼中坚**：大范围多层结界、高速矛突、短时绝界雏形；干部级妖可端据点、横扫小队。战力：一域中坚～精英边缘。≈三阶（部分顶峰逼近四阶观感，本世界无四阶切入）。
5. **夜行精锐 / 城市级妖灾 / 绝界实用**：一击夷平街区级妖力或结界对轰；火黒级剑妖、黑芒楼全面侵攻属此档。≈五阶（城市级·领主感战场）。
6. **神祐地战 / 十二人会席 / 土地神侧翼**：覆盖一城至地区的灵压与领域；绝界熟练、土地神权柄边缘。≈六阶（地区级灾难）。
7. **裏会顶点 / 真界终局 / 间时守残存指导**：可左右全国影子秩序、以精神体跨容器续命、以真界构筑可永久安置禁忌存在的「新域」。战力：人形战略武器·世界级灵战。≈七阶。

乐园阶位映射：凡人～弱妖/见习≈一阶；合格夜巡结界师/中级妖≈二阶；强力爆发与黑芒楼中坚≈三阶；夜行队长实用战力、黑芒楼崩塌级战场、城市妖灾≈五阶；神祐地/土地神侧翼与十二人会主力≈六阶；逢海日永·逢海月久·真界完成态·间时守历史峰值相关≈七阶。**本世界无八、九阶行星星空级表现**；顶点落在七阶后期～巅峰，忠于原作「全国影子秩序 + 神祐地永封」尺度。世界顶点（终局真界与土地神领域级存在）对低阶＝**存在·情报优先 / 条件性胜利**，严禁「战力归零式削弱」叙事；条件性胜利来自布局、共鸣、牺牲与正确场地，而非把顶点写成废柴。

**【地理 · 舞台】**
地图由「校园夜巡圈」向外扩到「全国影子组织」：
- **私立乌森学园 / 乌森之地（からすもり）**：开篇与全程主舞台。地表是普通中高一体校园，地脉下沉睡着禁忌——**宙心丸**的封印与泄漏的神祐之力。夜间妖怪被吸来并急剧强化；良守与时音每晚在此巡逻。氛围：月光、校舍阴影、结界蓝白光、突然炸开的妖形。适合一～三阶。
- **墨村宅 / 雪村宅**：一墙之隔的两家祖宅，世代不和却同守一地。宅内有家传妖犬（斑尾、白尾等）、修行空间与长辈督导。适合日常与一阶切入。
- **黑芒楼（こくぼうろう）异空间**：以「黑芒」土地为核的妖之城郭，楼阁漆黑、内部迷宫化。黑芒楼组织据此发动对乌森的侵攻；决战中城郭因力量失衡崩塌，良守在崩溃空间中展开绝界。**卡片锚点副本**。适合三～五阶。
- **夜行据点 / 裏会设施**：异能者地下组织的训练场、收容所、会议室。正守率领的**夜行**从此出动支援乌森。适合二～六阶侧线。
- **各地神祐地**：日本各地特殊灵地，内有土地神与异界结构；后期成为狩猎与迁封的棋盘。失土地神则失力量。适合五～七阶。
- **无想房 / 波同房等修行异域**：正统继承人精神试炼场，管人缟野、三毛野等；幻障与心魔并行。适合三～六阶成长节点。
- **终局新域 / 土地神领域**：为安置宙心丸而争夺、改写的关键领域；墨子以身相封之处。适合六～七阶。

势力地盘：两家结界师守点 → 夜行机动支援 → 裏会十二人会定国策级影子规则 → 黑芒楼等妖组织夺神祐 → 土地神与禁忌子嗣构成终局棋盘。本地人（普通学生）不知夜巡真相；契约者若当街大喊「我要屠总帅」，只会被当神经病——正确反应。

**【世界剧情线】**
① **开篇·乌森夜巡与童年伤痕（井底规则）**  
数百年前，间时守为保护被妖怪盯上的乌森一族与相关之人，立下结界师一脉；其后墨村、雪村两家自其助手一系分出，以竞争激励正统继承人，世代守护乌森。传说被简化成「土地吸引妖怪，须灭之」，真相被层层遮掩。  
现代：中学生**墨村良守**是墨村家第 22 代正统继承人（右手方印），却更爱做点心城堡、白天在课堂打瞌睡——因为每晚要夜巡。邻居高中生**雪村时音**技巧远胜于他，常在他失误时补刀收场。童年关键：良守曾同情一只靠近乌森的妖，结果妖被土地之力急剧强化，时音为救良守重伤，留下疤痕。良守自此立誓：**变强，不让时音与任何人再因乌森受伤；最终目标是永久封印乌森，解放两家。**  
此阶段世界状态：弱～中级妖夜夜入侵，两家长辈（墨村繁守、雪村时子等）督导严厉，家族不和对日常拌嘴级。名场面：校舍屋顶的结界对刺、方围失败被时音骂、点心与职责的拉扯、第一次独自完成「灭」。因果：良守保护欲与自责成为全剧引擎；时音「年长可靠」面具下藏着同等重压。契约者此时多以转校生、夜行见习、两家旁支等身份卷入——抢风头不如先学会「灭后开天穴」。

② **立名·夜行、志志尾限与裏会露面**  
裏会（异能者影子组织）开始显形。良守兄长**墨村正守**以夜行首领身份介入乌森防务，带来组织化支援，也带来「把弟弟当棋子/期待」的复杂亲情。混合妖少年**志志尾限**转入学园，表面不良、实为夜行成员，与良守由冲突到互相认可。影宫闪、秋津秀、花岛亚十罗等夜行同伴陆续登场，战斗从「两人夜巡」变成「小队协同」。  
裏会高层——**十二人会**——开始注视乌森这枚棋子：有人要利用，有人要收编结界师，有人已与妖侧暗通款曲。良守第一次感到：守住校园不够，影子国家在头上。名场面：限的变身与速度、正守绝界的压迫感、十二人会「客」席的冰冷会议。阶段结局：良守在同辈中立住「力量粗暴但意志极硬」的名；时音确认他值得托背。

③ **质变副本·黑芒楼侵攻与崩塌（卡片锚点卷）**  
**黑芒楼**——以土地神「姫」为象征、由原人类**白沼**（化名白）实际统率的高等妖组织——盯上乌森之力，企图强化自身、延续存在。干部多能穿「人皮」隐匿气息，名带颜色：牙银、火黒、紫远、蓝绯、碧闇、江朱等。他们宣战、渗透、人柱、魔方阵，最终把战火烧到乌森学园，校舍炎上，夜巡升级为攻防战。  
关键链：①黑芒楼试探与干部击破；②全面侵攻，夜行死守；③**火黒**（原黑田源一郎）以剑与高机动屠戮，**志志尾限**在激战中重伤牺牲，给良守留下无法愈合的「没能保护同伴」之痛；④良守被认定为有用容器/战力，一度被掳入黑芒楼异空间；⑤时音与夜行救援，迷宫与人柱机关层出；⑥姫对良守产生兴趣甚至力量牵连；白沼过往（为妻实验、半人半妖的中途半端）揭开；⑦**城郭因力量失衡崩塌**，空间碎裂；良守在崩溃的异空间中制造**绝界**，硬刚试图吞噬一切的灾厄级存在与土地神相关余波；⑧姫寿元将尽，将黑芒变回金芒原野，与白一同消散，黑芒楼瓦解。  
名场面必须可写：入口门槛（人皮与隐匿）、核心争夺（乌森之力/良守本人）、原作级压迫者（火黒、白、姫侧）、出逃与崩塌时的追杀与绝界绽放。阶段结局：良守战力与心志质变，绝界从无意识雏形走向可主动运用；时音与他的羁绊凝固；裏会与正守对乌森的介入不可逆。动画主线多止于此前后；漫画继续向下挖真相。

④ **格局战·神祐地、裏会阴谋与无想**  
黑芒楼消亡后，真正的棋盘是**裏会内部**。总帅**逢海日永**与以**梦路久臣**为表的**逢海月久**（创会双核之一）互相仇视、精神体夺容器，把十二人会与全国神祐地变成猎场。零号、叁号（后为人形的**冰浦苍士**）、架袈琉、魅稚琉等「数字」与实验体被投放；扇一族、各席干部各怀鬼胎。良守为掌控力量、接近封印乌森的方法，进入**无想**等精神修行，与土地神、间时守残存意志产生接触。  
正守在十二人会中的位置上升，夜行成为内战中的关键武装；良守必须在「信兄长」与「防裏会」之间走钢丝。名场面：神祐地失守的连锁、精神附身揭伪、无想房内心障具象、冰浦苍士的身份震荡。阶段主题：个人守护升级为影子内战的侧翼。

⑤ **真相·宙心丸与间时守**  
乌森传说被推翻：真正的「核」不是单纯土地，而是**宙心丸**——间时守与乌森家继承人（月影一系）之子。间时守以禁术赋子力量，反噬使宙心丸无意识吸取周围生命，酿成悲剧；间时守以真界系大结界将子封印于原属土地神（乌龙大人等相关）的神祐地，泄漏之力却仍吸引妖怪，两家结界师的永夜勤务由此而来。方印与继承人制度，亦是间时守为将来「能共鸣者」埋下的机制。  
良守作为历代最强共鸣者之一，逐渐能承载、对话、甚至短暂共有宙心丸的力量。他明白：单纯「灭妖」只是治标；要解放乌森，必须给宙心丸一个不会再杀人的归所。名场面：封印底层的童形存在、躲猫猫般的交流、间时守斗笠与法杖的幻影指导、时音得知真相后的沉默与坚持。

⑥ **终局·迁封、墨子之牺牲与新秩序**  
终局行动：良守、时音，在母亲**墨村守美子**与**间时守**协助下，寻找可安置宙心丸的新域——涉及驱逐占据土地神领域的裏会创始相关存在、改写领域归属。最终，守美子以自身完成领域封印（留在封内），换取宙心丸的永久安放与乌森泄漏的终止。裏会在双核同归于尽与改革力量（含正守一线）作用下转向改善；墨村与雪村的守护职责结束，数百年对立失去燃料。良守感到「世界终于正常」，日后志向与建筑/创造相关（与点心城堡一脉相承的「造物」），并与时音的关系落到可托终身的安静处。  
谁活谁死（写手勿乱改核心）：守美子牺牲；逢海日永与月久在仇怨中共灭；限已在黑芒楼弧牺牲；良守、时音、正守、利守等核心存活向；宙心丸得安放而非简单抹杀。胜利昂贵：**布局 + 共鸣 + 牺牲**，不是把顶点削成零。

⑦ **微观政治贯穿（日常可介入）**  
除大场面外，正文高频：夜巡排班与两家抢功、学园祭与考试和妖袭叠在一起、夜行悬赏与临时雇佣、人皮假货与情报贩、神祐地配额与裏会征税式征调、混合妖失控收容、结界材料与术式口诀的保守传承。AI 优先这些小事件，而不是一上来总帅对轰。

⑧ **蝴蝶效应边界**  
契约者可改局部：救一名夜巡见习、提前击破某干部、抢走非核心妖具、让某场神祐地争夺易主。不可无代价改写：宙心丸必须被正确安置的逻辑、守美子牺牲的结构位（可换等价代价，不可无代价免伤）、火黒与限之战的悲剧重量、真界完成所需的共鸣与修行。改了局部，因果以「更强的妖被吸引来、裏会提前收网、同伴代死」回流。

**【主要人物】**

**【隐藏剧情 · 伏笔】**
- 乌森「吸引妖怪」的教科书解释是遮羞布；真核是宙心丸的生命吸收与泄漏。
- 两家竞争制度是间时守为选出能完成终局者而设的残酷沙盒。
- 裏会创会双核的精神体仇杀，比任何外敌更毁神祐地网络。
- 良守的「做点心/建造」爱好与真界「创造世界」在主题上同构——伏笔在日常。
- 动画止于黑芒楼，不代表世界结束；契约者若只按动画记忆行动，会被漫画后期真相打脸。
- 结局后：乌森不再夜夜灾厄，两家卸责，裏会改革，世界进入「可过普通夜里」的新日常。

**【叙事基调 · 雷区】**
画风：夜色校园的紧张感 + 少年漫画的羁绊热血 + 偶尔的哀伤牺牲；战斗要写结界的几何感、口令感与空间切割，不要写成无脑光波对轰。  
忌：把良守写成无脑滥好人（他护短且会怒）；把时音写成只会花瓶助攻；把黑芒楼当唯一终局（后还有裏会与宙心丸）；把宙心丸当经验怪击杀；把顶点写成「被封印所以战力为零」——封印是代价与结构，不是削属性。  
最早切入锚点：良守仍常失误的夜巡初期，或黑芒楼宣战前夜。


**【世界剧情线·续：夜巡微观与黑芒楼战役肌理】**
乌森学园的夜巡不是「刷怪任务列表」，而是一整套可被打断的生活节奏。傍晚，墨村良守在厨房捏点心城堡的糖霜尖顶，雪村时音在邻宅擦拭结界用的短杖与手套；夜半，两人在走廊交叉，用极短的口令交换方位——谁守旧馆，谁守体育馆，谁去天台看风。弱妖常从下水道、储物室、被遗忘的社团室渗入；它们被乌森之力浸过之后，爪牙会突然长出一截，速度会突然快一拍。合格的结界师必须在「方围」喊出的瞬间完成对目标的锁定，否则定礎线会像湿粉笔一样断。良守早期最大的毛病是力量过大、边缘毛糙：结界壁能挡住妖，也会误伤门框与窗玻璃，第二天被凡人同学当成「奇怪的破坏王」。时音则相反：矛结界又细又准，但面对同时多目标时，必须把良守的巨壁当盾。两家的不和在此具象为抢功——繁守与时子会在事后分别盘问「是谁先结的」「是谁漏的」，把一次普通除妖写成家族脸面战。

黑芒楼侵攻前，渗透以「人皮」为刀。人皮让高等妖走在白日校园里像转校生、像工务员、像失踪已久的校友。夜行最先察觉异常的是气味与波长，而不是外形。志志尾限以混合妖的直觉撕开过不止一张皮，却也因此更被同学当怪物。花岛亚十罗的暴脾气、影宫闪的感知、秋津秀的不稳定变身，构成夜行小队的日常噪音与战力互补。正守下达的优先级永远是：先保凡人，再保结界师，最后才是情报完整——这让急性子的良守多次怒骂「情报比人命重要吗」，而正守只回以更冷的任务单。

全面侵攻夜，校舍火光与结界光叠在一起。牙银一类干部冲阵，紫远、蓝绯、碧闇、江朱在侧翼制造混乱与人柱；火黒则专挑最强的刀口。限与火黒的对决是黑芒楼弧的情感核：限以命换时间，良守赶到时只剩「没能保护」的空白。被掳入黑芒楼后，迷宫层会改写方向感，天穴与廊/陋交错，普通人会在三分钟内失去上下。白沼的虫网在墙里爬，把伤者变成监听器。姫的出现不是传统魔王演讲，而是土地神式的疏离打量——她看良守，像看一件可能托付的容器，又像看一只误入神域的幼兽。城崩时，空间碎块会割伤灵体；绝界在此不是炫技，而是「拒绝被世界揉碎」的生理级反抗。良守的紫黑域展开的瞬间，火黒的刃第一次真正被「空间的拒绝」挡住，而不是被力量硬撞。

**【世界剧情线·续：裏会政治与神祐地经济学】**
裏会不是单一反派会社，而是影子国家。十二人会的每一「客」都有部门、预算与私兵逻辑。梦路久臣的会议室永远恒温、永远礼貌，签字比刀快；逢海日永的容器可以是孩童、可以是干部，记忆被改写的人会微笑着把神祐地钥匙交出去。神祐地「失神」在凡人侧只表现为地方怪谈升温、事故率上升；在异能侧则是战略资源蒸发——失土地神的地会狂乱或枯死，吸引来的妖会无序扩散。正守把夜行绑在改革侧，等于把弟弟的乌森也绑上内战；良守若在错误时间暴露与宙心丸的共鸣，会被两派同时当成「可移动的神祐核心」。

无想修行把战斗从「外面的妖」改成「里面的心」。无想盒要求无心，无想房制造幻障：良守会看见时音再一次受伤、看见限再一次倒下、看见自己把学园灭成白地。过关不是变强数值，而是承认恐惧后仍选择守护。间时守的残影不给外挂，只给条件：你若只想「消灭灾厄」，你永远只能灭到下一只妖；你若想结束永夜，你必须给灾厄一个不会再杀人的形状。

**【世界剧情线·续：迁封执行细则与战后日常】**
迁封不是传送门闪白。新域要先清场，再展开真界外壁，再把宙心丸的吸收涟漪压进可居住的结构，最后由守美子以身完成「封口」。外围必须有人挡住裏会残党与失控妖潮——这是契约者七阶最可能站的位置。战后，乌森学园的夜重新安静，良守的黑眼圈会慢慢淡，点心城堡可以做给还活着的人吃。时音仍会在傍晚看一眼校舍方向，像确认一种习惯的死亡。斑尾与白尾失去「夜夜出征」的工作，变成真正的家犬式吐槽役。裏会改革后的证件能让异能者合法行动，也能变成新的官僚牢笼——世界变好，不等于变软。

**【主要人物·补全立场与可互动钩子】**
- **雪村时雄 / 雪村静江等雪村侧长辈线**（以原作在场者为准）：维护雪村家名，对墨村式「力量优先」嗤之以鼻；契约者若只靠蛮力解决夜巡，会在时子/时雄一系面前扣分。  
- **钢夜等与斑尾旧识相关的妖**：可作一～二阶支线——旧债、旧伤、是否灭尽的道德题，呼应良守童年「同情妖」的禁区。  
- **扇一族相关者（扇七郎等）**：裏会侧翼政治筹码，六阶神祐地案中可交易假情报，也可背刺。  
- **神田百合奈等学园凡人侧**：不知情的日常锚；保护他们是两家与夜行的共同底线，也是检验契约者是否「把乌森当副本」的标尺。

**【地理·舞台补密】**
- **旧校舍与天台**：一阶高频遭遇点；结界余波常震碎灯管。  
- **体育馆地下室 / 水道**：人皮妖潜伏点；二阶线索多。  
- **黑芒楼核心殿与金芒原**：五阶终局舞台；崩塌后残土仍带微弱土地安抚性。  
- **地方鸟居神祐地**：六阶案件现场；空神的鸟居是最明确的「出事」路标。  
- **真界白域**：七阶唯一正确的「家」的形状；不是宝库，是监牢与摇篮的合一。

**【贵重物品·使用边界再钉】**
方印不可移植；假方印会引来夜行审查。绝界强行模仿易反噬经脉式灵伤。真界材料不存在「掉落」，只存在「观礼级理解」。人皮持有超过一夜，斑尾级感知者会报警。神祐地碎片若带离本土，可能引发土地神追杀或裏会征用。宙心丸相关容器一经暴露，黑芒楼残党（若有）与裏会双派会同时上门——这是全世界最高危掉落。

**【隐藏剧情·契约者特记】**
若契约者在一阶就宣称「我知道宙心丸」，会被当成泄露禁咒的疯子或裏会间谍。正确的信息节奏是：先学会夜巡，再看见人皮，再卷入黑芒楼，再被政治撕开，最后才配站在真界外壁。动画记忆者若在黑芒楼后收工，会错过母亲之死与职责解放——那是漫画正典的重量，不可用「已打完最终Boss」糊弄。
## 阶位切入点

> 阶位↔原作强度：凡人/弱妖/见习≈一阶；合格夜巡结界师/中级妖≈二阶；强力爆发与黑芒楼中坚≈三阶；夜行实用天花板、黑芒楼崩塌级、城市妖灾≈五阶；神祐地/土地神侧翼与十二人会主力≈六阶；逢海日永·月久·真界终局·间时守峰值相关≈七阶。世界顶点（真界完成态/土地神领域级）为跨入七阶巅峰的存在，低阶切入一律「规避顶点·情报优先」；条件性胜利靠布局与牺牲结构，严禁战力归零解释。本世界仅覆盖：一、二、三、五、六、七阶（**无四阶**）。

**一阶（见习夜巡 · 乌森学园井底）**
切入身份/时点：契约者以「私立乌森学园新转校生」或「两家旁支送来的见习结界师」身份切入，锚定良守仍频繁被时音收拾残局、黑芒楼尚未宣战的早期夜巡期。
初始事件：入夜后第一声警报——一只被乌森之力强化的低级妖撞破旧校舍窗；**墨村良守**方围变形差点漏妖，**雪村时音**的矛结界已经刺出。契约者被要求立刻选边：帮忙定礎、撤离凡人同学、或贪功抢「灭」——抢错会同时惹两家。
开场白建议：「门铃在午夜不成样子地响了一下，其实是结界的余波震的。走廊尽头，那个总揉眼睛的男生喊着方围，邻家的学姐已经把结界矛送进妖喉——有人把扫帚塞进你手里：今晚起，你也是乌森的人。」
关键NPC立场：**墨村良守**——防你抢功，但真帮忙会记情；**雪村时音**——冷静评估你是否拖后腿，失误一次记一周；**斑尾**——用鼻子验你身上有没有妖气；**墨村繁守**——盘问师承；**雪村时子**——冷笑看墨村是否又丢人。
主线钩子/支线：童年伤疤相关的「同情妖」禁令是否重演；支线＝旧校舍盲区巡查、给斑尾买粮换情报、两家赌气的重复除妖单。
危险度/规避：中——死于漏网中级妖与定礎失误；须规避正守级介入、任何黑芒楼干部、神祐地深层。
任务方向/奖励：活过第一个月夜巡、学会灭后开天穴；奖励取向＝基础结界口诀校正、低阶感应护符、夜行临时通行证（见习）。

**二阶（合格夜巡 · 夜行阴影下的学园）**
切入身份/时点：以「夜行外围协力者」或「已能独立完成灭的年轻结界师」切入，锚定**志志尾限**转学前后、正守开始定期过问乌森防务之时。
初始事件：学园后山出现会穿人皮的妖，普通结界刮不破皮；**志志尾限**以混合妖之姿先一步撕开伪装，却被夜行纪律勒令「少暴露」。契约者掌握一段人皮来源线索，必须在上交给**墨村正守**、私下给良守、或卖给来路不明的收买者之间抉择。
开场白建议：「人皮叠在体育仓库的垫子下，还带着体温。限把烟掐了，说：你选夜行，还是选那两个死正经的结界师？——远处正守的绝界余压像一场无声的雷。」
关键NPC立场：**志志尾限**——认可实力不认可说教；**墨村正守**——收编或监控二选一；**花岛亚十罗**等夜行——任务向；**影宫闪**——感知你的情绪破绽；**雪村时音**——警告你别把学园当裏会试验场。
主线钩子/支线：人皮供应链是否通向黑芒楼前期渗透；支线＝护送夜行伤员撤离、假扮学生引妖、与限的速度对练换情报。
危险度/规避：中高——人皮刺客与裏会观察眼；须规避十二人会本尊、火黒级。
任务方向/奖励：建立夜行或两家其中一条稳定人脉；奖励取向＝中级结界塑形指导、混合妖对策笔记、可移动小型结界锚。

**三阶（黑芒楼前夜 · 干部级威胁）**
切入身份/时点：以「夜行正式队员」或「两家认定的战力外援」切入，锚定黑芒楼干部开始点名乌森、火黒之名尚未染血限之死以前的高压期。
初始事件：**牙银**或同级干部级妖袭击夜巡路线，目标试探良守的方印共鸣；时音受伤风险上升，良守第一次把「绝界」的杀意露在眼底。契约者手中有一份黑芒楼入口坐标半成品——交出来会引爆提前战，藏起来可能让限少走一步死路，也可能让更多学生死。
开场白建议：「校门口的樱树一夜焦黑。牙银留下的爪痕里嵌着人皮纤维，正守只说了三个字：准备打。你看着良守捏紧的右手掌心——方印发烫，像有谁在地底醒来。」
关键NPC立场：**墨村良守**——接受并肩但拒绝你碰时音的伤；**雪村时音**——把你编进防守阵型；**志志尾限**——邀你一起当刀；**白沼**的虫侦——已在暗处编号；**墨村正守**——下令死守优先级：凡人 > 结界师 > 情报。
主线钩子/支线：是否加速或延缓黑芒楼总攻；支线＝摧毁一处人柱预备点、保护神田百合奈等知情边缘学生、偷录白的虫频段。
危险度/规避：高——干部级与虫控；须规避姫本尊、火黒全力、城内深层迷宫。
任务方向/奖励：在总攻前削弱一层黑芒楼牙；奖励取向＝抗虫符、绝界观摩残页（不可稳定复制）、夜行战功等级。

**五阶（黑芒楼崩塌 · 城市级妖灾与绝界）**
切入身份/时点：以「夜行主力/结界师主力外援」切入，锚定**黑芒楼异空间崩塌**当夜：城郭力量失衡，良守在碎裂空间中展开绝界，火黒之战与姫—白终焉重叠。
初始事件：你被卷进崩塌的黑芒楼层，重力与方位失效；**火黒**的刃光切断你刚立的结界边角，**墨村良守**在更深处点燃紫黑绝界。选择：为限报仇式突进、护住时音救援队通道、或抢夺姫留下的土地权柄碎片——后者会让白沼与正守同时追杀你。
开场白建议：「楼在塌。不是比喻，是整座异空间像被揉皱的纸。有人喊灭，有人喊解，更多人喊不出声——良守的绝界在碎石间张开，像拒绝被世界吞掉的一口逆呼吸。」
关键NPC立场：**墨村良守**——此时只认「能不能挡住下一刀」；**雪村时音**——救援优先于击杀；**火黒**——视强者为剑冢；**白沼**——要完整回收姫的终焉；**墨村正守**——外围开绝界走廊接人；**姫**——寿元将尽，态度超然可托付或无视。
主线钩子/支线：崩塌夜的生存与通道；支线＝找回限的遗物交给良守、阻止虫群逃入现实学园、护送伤员过天穴边缘。
危险度/规避：极高——城市级对轰与空间崩坏；须规避把姫当小怪斩杀的写法和日永级存在提前下场。
任务方向/奖励：活着走出崩塌、保住学园侧翼；奖励取向＝绝界抗性经验、黑芒金原残土（纪念/微弱土地安抚）、夜行英雄级记名（也是裏会备案）。

**六阶（神祐地棋盘 · 十二人会内战侧翼）**
切入身份/时点：以「神祐地监察官」「十二人会某席附属武」或「独立高阶结界师」切入，锚定黑芒楼后、日永与月久代理战争白热化、良守进行无想/接触间时守的阶段。
初始事件：一座地方神祐地的土地神失踪，裏会两派均指控对方「偷神」；**冰浦苍士**奉命调查却记忆缺块，**梦路久臣**的表脸要你在报告上签字。真相指向精神附身链——签字即站队，拒签即被两边当灭口材料。
开场白建议：「神祐地的鸟居还在，神却空了。苍士站在鸟居下说：我有三天的记忆是白的。你身后的通讯器里，梦路与正守的指令同时到达，内容相反，语气同样礼貌。」
关键NPC立场：**墨村正守**——要实证反内奸；**冰浦苍士**——求完整自我；**梦路久臣/逢海月久**——要签字与容器；**逢海日永**的眼线——要你把火烧向月久；**间时守**残影——只在你靠近乌森真相时给一句谜语；**雪村时音**——警告良守别被裏会借刀。
主线钩子/支线：神祐地失窃案与附身链；支线＝保护无想房入口不被政治化、截获零号级实验体转移、与扇一族交易假情报。
危险度/规避：极高——地区级灵灾与精神污染；须规避日永/月久本体对碰、强夺宙心丸。
任务方向/奖励：保全至少一处神祐地或换取良守修行时间；奖励取向＝波同入门、土地神临时借权（有期限）、十二人会黑材料（烫手）。

**七阶（真界终局 · 迁封与顶点侧翼）**
切入身份/时点：以「可参与终局迁封的七阶级异能者/结界大师」切入，锚定**墨村守美子**准备以身封域、良守展开真界安置**宙心丸**、裏会双核走向共灭的终章窗口。
初始事件：新域入口开启，原占据者（裏会创始相关精神体势力）反扑；**宙心丸**的童形在真界边缘伸出手，生命吸收的涟漪让整支部队短暂窒息。**墨村守美子**要求契约者做一件具体的事：守住真界外壁三息、或替她把间时守的残存意志送入核、或在外围挡住正守都未必能挡的一击——并明确告诉你：她不打算出来。
开场白建议：「白域铺开的时候，夜第一次像被洗过。宙心丸问你：这里能不能躲猫猫？守美子站在你身前，背影已经像一道要落下来的印——她说：别劝。去挡住不该进来的东西。」
关键NPC立场：**墨村良守**——完成真界的执行者，谢绝争功；**雪村时音**——守外壁与凡人侧；**墨村守美子**——托付死局；**间时守**——核对封印条件；**宙心丸**——要「家」不要「灭」；**逢海日永 / 逢海月久**——各自想吞终局权柄或同归于尽前多拉垫背；**墨村正守**——外围改革军，防裏会残党抢封。
主线钩子/支线：迁封成功与否的蝴蝶；支线＝转移学园残存夜巡职责、保全利守等非战斗家族、阻止双核战斗波及新域。
危险度/规避：贴近顶点——真界/土地神领域/双核精神体＝七阶巅峰战：存在·情报优先 / 条件性胜利，**严禁硬刚本体与战力归零解释**；胜利只能来自协助正确封印结构与代价分配。
任务方向/奖励：新域落成、乌森泄漏终止；奖励取向＝真界观礼级理解（不可完整复制）、卸责后的两家人情、裏会改革后的合法异能身份。勿发「宙心丸本体」或「无限绝界」。
