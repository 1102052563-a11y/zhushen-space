## 剧情

**【作品来源】**
《仙剑奇侠传三》（Chinese Paladin 3 / 仙剑奇侠传Ⅲ）由上海软星开发，台湾大宇资讯与大陆寰宇之星发行，2003年7月31日台湾首发、8月4日大陆发售；总监姚壮宪、张毅君，制作人张孝全。系列第三代正作，主题「轮回」，时间线约在初代《仙剑奇侠传》故事五十年前、二代五十八年前。本作确立五灵（水火雷风土）与六界（神魔鬼人妖仙）框架，以两把剑（魔剑、镇妖剑）与景天前世今生纠葛为核心。2009年有电视剧改编（胡歌等主演），人物关系与部分桥段有改写；**本档案以2003年游戏正典为准**，外传《仙剑奇侠传三外传·问情篇》接完美结局后约八年。文风：仙侠奇遇、前世今生、牺牲与情义、门派江湖与神魔并重。卡片锚点：神界/新仙界天帝决——景天持剑挑战重楼，魔剑与镇妖剑级威能对撞，神界建筑崩塌，触及六界顶级战力门槛。

**【世界定位】**
人界江湖与蜀山锁妖塔为舞台、神魔鬼妖仙六界并行的仙侠任务世界。主角**景天**（永安当伙计，前世神将飞蓬、姜国太子龙阳转世）因重楼取出锁妖塔中魔剑而卷入唐门内斗、五灵珠寻珠、邪剑仙为祸与新仙界决战。一句话：当铺伙计背起轮回之剑，在情与劫之间补塔、铸剑、挑战魔尊。契约者是变数，不是免死金牌；低阶改不了飞蓬—重楼因果，高阶仍须付牺牲代价。

**【世界观 · 力量体系】**
力量本源：天地灵气、五灵属性（火克雷、雷克风、风克土、土克水、水克火）、仙术、剑诀、妖力、鬼力、魔气、女娲血脉与神将神力。蜀山仙剑派以仙术降妖，将强妖囚于锁妖塔；唐门擅制毒暗器；霹雳堂擅火器机括；蓬莱派偏清净修仙。战斗观感（游戏半回合制映射到叙事）：精（生命）、气（技能）、神（仙术）；五灵法阵站位可放大或克制属性；后期魔剑可御剑飞行。

力量与战力表现（写战力优先破坏力，宁低勿高）：
1. **凡人 / 江湖三流**：徒手与冷兵器，踹门格斗，一刀可致命。≈一阶。
2. **武林高手 / 唐门弟子 / 霹雳堂火器手**：毒蒺藜、暗器、火器、小队战；可掀翻壮汉、威胁凡人据点。≈一~二阶。
3. **入门修仙 / 蜀山外围弟子 / 低阶妖魅**：成形仙术、裂石断金级；能清一处妖巢、护一镇。≈二阶。
4. **蜀山精英弟子 / 弃徒徐长卿中期 / 景天队伍中期 / 中阶妖将**：远程仙术与剑招成体系，可端山寨级据点、在乱军中取首。≈三阶。
5. **蜀山长老级 / 邪剑仙前中期 / 紫萱灵力显化 / 塔内高层妖**：摧毁建筑群、掀平据点；一城仅数人。≈四阶（质变分水岭）。
6. **邪剑仙全盛 / 女娲后裔全力 / 锁妖塔天妖皇级危机**：街区至城市级威胁，领域/阵法雏形，凡俗军队难挡。≈五阶。
7. **蜀山镇派级合力 / 补塔级灵力 / 六界通道侧翼**：地区级灾难，移山填海初现，锁妖塔崩塌余波可波及人界一方。≈六阶。
8. **神将飞蓬 / 魔尊重楼（本体级交手）**：世界级人形战略武器；神魔之井侧、新仙界决战余波可令神界建筑崩塌。≈七阶。
9. **天帝体系 / 六界规则本源向存在**：跨阶；本库切入不覆盖其本体硬刚。≈超阶向。

死亡与复活：凡人死则死；鬼修（龙葵）以魂魄宿剑可长存但受剑体与炉火约束；女娲后裔、神将转世有轮回与记忆封印；重楼级存在可在特定结局散尽魔力救人、复活牺牲者，皆为高代价，非无限读档。特殊系统：魔剑宿灵、五灵珠封印、锁妖塔阵眼、铸剑跳炉、好感度与援护（叙事上=羁绊强度）、六界书等秘典（以游戏出现为准）。

乐园阶位映射（对照《阶位战力图鉴》，宁低勿高）：凡人~江湖三流≈一阶；武林高手/唐门霹雳堂骨干/低阶修仙≈二阶；蜀山精英弟子/中期队伍/中阶妖≈三阶；蜀山长老级/邪剑仙前中/紫萱显化≈四阶；邪剑仙全盛/女娲全力/塔内天妖皇级≈五阶；补塔级/镇派合力/地区灾难≈六阶；飞蓬·重楼本体交手≈七阶；天帝/规则本源≈超阶。世界顶点（重楼、飞蓬神将期、天帝侧）忠于原作：**超阶巅峰写条件性胜利，严禁「战力归零式削弱」**；低阶切入一律规避顶点·情报优先。本库仅覆盖一~七阶。

**【地理 · 舞台】**
- **渝州**：开篇人间城；**永安当**（景天当伙计，擅鉴宝）与周边竹林、民居是一阶主舞台。日常=当票、赝品、邻里、唐门产业阴影。
- **唐家堡（唐门）**：渝州东郊，制毒世家总部；掌门**唐坤**、大小姐**唐雪见**、内斗与霹雳堂勾结线。适合一~二阶。
- **霹雳堂**：与唐门争锋的火器门派，掌门**罗如烈**；炼毒人、夺五毒珠等阴谋。适合二阶。
- **蜀山派 / 锁妖塔**：仙剑降妖总坛；塔囚天下强妖，塔基与五灵封印相关。重楼取魔剑导致塔顶坍塌、封印松动。中高阶核心副本。
- **蓬莱岛 / 蓬莱派**：清净修仙侧翼，寻珠与支线舞台。
- **雷州、蛮州、各地坊市与秘境**：五灵珠散落点；妖、人、门派混杂。适合二~四阶。
- **神魔之井**：连接神魔两界的通道，飞蓬曾镇守。
- **新仙界 / 神界战场**：飞蓬与重楼约定决战之地；终局天帝决舞台，建筑可被两人交手震塌。适合六~七阶。
- **姜国故地（前史）**：龙阳、龙葵铸魔剑悲剧舞台，以回忆/剑灵叙事出现。
- **璧山 / 九龙坡 / 宾化 / 大渡口**：渝州周边人间路网，假货、护送、门派眼线密集。
- **长江船线 / 船海**：线性迷宫式水路，连接蓬莱；宝箱与遭遇战后期难回补。
- **九顶山 / 德阳 / 安宁村 / 古藤林**：入蜀山前的凡人—修仙过渡带；安宁村是补给与还神丹式药材点。
- **蜀山故道**：机关、石柱、大石考验，仙门后山资格考。
- **神界残片 / 神树相关空间**：经神魔之井触发，飞蓬记忆与夕瑶线的高阶舞台。
- **草海 / 灵山仙人洞 / 古城镇**：蛮州周边；支线、古董、女娲侧翼。
- **熔岩地狱 / 海底城 / 冰风谷 / 鬼界·黄泉路**：属性主题迷宫，服务灵珠、材料与心魔试炼。
- **剑冢**：铸剑总账台，结局分叉的物理地点。
- **新安当铺**（终局完美线）：景天回归烟火的落点，与开局永安当对照。
写场景先定契约者在哪一层：渝州当铺烟火 → 唐门毒雾 → 船海潮腥 → 蜀山云海与塔内腥风 → 蛮州巫祭 → 新仙界神光与崩石。本地人若听闻凡人要硬刚魔尊，当疯子——正确反应。

**【世界剧情线】**
① **前史·飞蓬与重楼**  
神将**飞蓬**镇守神魔之井，与魔界至尊**重楼**多次交手，势均力敌、惺惺相惜。重楼邀约新仙界一决胜负；飞蓬明知违律仍应战。天庭追兵将至，飞蓬分心，佩剑被击落人间，本人被贬下凡，开启轮回。飞蓬第一世转生为姜国太子**龙阳**；为救国铸**魔剑**，城破身死，妹**龙葵**跳炉以处女之血成剑，魂魄封于剑中千年，性格裂为温柔怯弱的蓝葵与强势护妹的红葵。魔剑后成蜀山锁妖塔封印物之一。飞蓬最终转世为渝州**景天**。主题：轮回、约战、情义与违律代价。世界状态：神魔如天，人界不知。

② **渝州结缘·永安当**  
景天在永安当做伙计，鉴宝吹牛、梦想发财或成蜀山剑仙。**唐雪见**为找海棠红紫砂壶盖等因由闯入其生活，毒蒺藜「伤」他却结缘；宠物五毒兽**花楹**相随。日常冲突是赝品、当票、街坊与唐门规矩。此阶段战力=凡人江湖；正确玩法是鉴宝、跑腿、躲暗器、别碰塔。

③ **唐家堡内斗与霹雳堂**  
雪见卷入唐门权力与**霹雳堂**阴谋：毒人、五毒珠、掌门之争。景天被拖入家族政治；雪见一度被揭为唐坤收养的弃婴（实则与女神**夕瑶**、天庭神树果实化身相关，后文揭示）。**徐长卿**（蜀山弃徒）现身相助，众人结识。阶段结局：雪见与景天离开唐家堡尺度，踏上更大江湖。可介入：护送唐门信物、识破假毒、截霹雳堂火器车、在比武夺帅外围站队。

④ **魔剑一文钱与龙葵现身**  
重楼因寂寞与「帮飞蓬转世变强」的任性，破坏锁妖塔取出魔剑，以「一文钱」当给景天。塔封印松动，妖魔外逃；蜀山五长老修仙排除的邪念趁机成形，是为**邪剑仙**。景天以血唤醒剑中**龙葵**，蓝/红性格交替登场，认景天为「王兄」。魔剑带来力量也带来晕厥与心性扰动，后须蜀山心法等约束。因果：取剑=锁妖塔危机的直接扳机；邪剑仙出逃=中后期主敌。

⑤ **寻五灵珠·补塔之旅**  
为重新封印锁妖塔，须集齐**五灵珠**（水、火、雷、风、土）。景天、雪见、龙葵与徐长卿、**紫萱**（女娲族后裔，与长卿有三世情缘）结伴，足迹遍及雷州、蛮州、蓬莱等地。旅途中：门派恩怨、妖魔袭扰、前世记忆碎片、重楼时隐时现的「助拳」与调侃。紫萱温柔内敛却背负女娲使命；长卿重情重义，一心想立功归山。中期后魔剑可御剑飞行，队伍战力质变。名场面：寻珠、塔外周旋、好感互动与援护、邪剑仙阴影逼近。

⑥ **锁妖塔与邪剑仙（中阶核心副本）**  
集珠后入塔：九层杀机、天妖皇级存在、阵眼与封印逻辑。邪剑仙狡诈强大，利用剑与塔的力量意图彻底破坏秩序、祸乱武林。塔内是规则杀与人心杀并行：同伴可牺牲，宝物可反噬，逃出生天比「称霸塔内」更现实。卡片可玩性：入口门槛（蜀山许可/五灵珠）、核心争夺（灵珠、剑、阵眼）、原作级压迫者（邪剑仙、高层妖）、出逃时的追杀与补塔倒计时。

⑦ **铸剑抉择（结局分叉锁）**  
关键节点：两剑受损，须在剑冢/炉前选择重铸**魔剑**或**镇妖剑**；常需女主角跳炉牺牲成剑救景天。  
- 选镇妖剑 → 往往对应龙葵跳炉牺牲线；  
- 选魔剑 → 往往对应雪见以神树果实之力修复并牺牲线。  
好感度（雪见、龙葵、紫萱、花楹）与是否战胜新仙界重楼，共同决定五结局。写正文：铸剑不是数值菜单，是「谁愿为谁死」的公开处刑；契约者若插手，必付对等因果。

⑧ **新仙界天帝决（卡片锚点·终局）**  
消灭邪剑仙后，景天赴**新仙界**履飞蓬与重楼之约。重楼变身增强；两人交手震塌神界建筑，触及六界顶级门槛。  
- **完美结局**（常作外传衔接）：打赢重楼 → 重楼复活牺牲者 → 景天与雪见、双龙葵于新安当铺过烟火日子。  
- **雪见结局**：镇妖剑线 + 新仙界败给重楼 → 重楼放雪见回人间，景天于渝州竹林寻回。  
- **龙葵结局**：魔剑线 + 新仙界败 → 红葵离开，蓝葵与景天。  
- **紫萱结局**：紫萱好感最高 + 镇妖剑 → 紫萱欲以身补塔，重楼散尽魔力救之成凡人，后探望。  
- **花楹结局**：花楹好感最高 + 魔剑等条件 → 与花楹浪迹；龙葵红蓝分离等变化以正典分支为准。  
终局铁则：重楼不是「被削成杂兵的BOSS」，是可救人可毁约可成全的魔尊；战胜=条件性、高准备、高羁绊，禁止低阶硬刚本体。

⑨ **微观政治贯穿**  
当票与赝品、唐门配额与毒方、霹雳堂火器走私、蜀山任务与弃徒脸面、五灵珠黑市情报、锁妖塔外围收尸与悬赏。AI优先写这些小事件，而不是一上来新仙界决战。

⑩ **蝴蝶效应边界**  
可改：救一路人、截一批毒、助唐门支线、提前警告某妖袭、换一条寻珠路线。不可无代价改：飞蓬—重楼约战逻辑、魔剑取自锁妖塔的因果、邪剑仙由五老邪念成形、铸剑必须有人付代价、完美结局外的核心死亡与补塔需求。改局部，原作以追杀、取代、更惨回流。

⑪ **分卷细目·渝州至唐家堡（低阶主舞台）**  
开局严格落在人间烟火：景天在永安当写票子、鉴古董、吹牛「这物件值三座山」，夜里被唐雪见为海棠红紫砂壶盖等琐事踹门、毒蒺藜招呼。两人从互坑到并肩，背景是唐门产业网——当铺、毒坊、暗桩与保护费式规矩。璧山、九龙坡、宾化、大渡口构成渝州周边跑图：护送、假货、路匪、门派眼线。长江船线把舞台从内河推到海路，为蓬莱铺垫。此卷战力全是凡人与江湖暗器；蜀山、重楼只存在于景天的白日梦与民间传说。可介入事件：当票纠纷、赝品鉴定对赌、替雪见找壶盖撞见内宅密谈、护送押运避霹雳堂劫、在唐家堡演武场外围下注或递暗器。世界状态判断：若景天仍叫「伙计」且魔剑未入当，即一阶开局。

⑫ **分卷细目·蓬莱与蜀山故道（入仙门槛）**  
船长江/船海迷宫后至**蓬莱**：码头、蓬莱派、迷宫蛊阵（叙事上常需紫萱类灵力破阵）、毒地与尸块炼器材料。相关节点含「前往蓬莱」「蓬莱有变/灭门式危机」——清净修仙地被血洗或重创，点明江湖与仙门都不安全。之后经九顶山、德阳、霹雳堂外围、安宁村、古藤林，入**蜀山故道**：平台机关、石柱伸缩、大石需特定人（龙葵）带队等，是「凡人闯入仙门后山」的资格考。初至蜀山见五长老，正式领取寻五灵珠、助修锁妖塔封印的任务；紫萱与长卿关系、弃徒脸面在此摊牌。回唐家堡比武夺帅、弃婴身世爆发，把家族线与仙门线焊死。可介入：故道开路、安宁村买还神丹式补给、为长卿传话、在比武场阻止毒人。世界状态：队伍成形，二~三阶。

⑬ **分卷细目·雷州怪谭与神魔之井（中阶转折）**

- **锁妖塔阵眼与封印**：地理级权柄；破坏=妖灾；补塔可要女娲级灵力或五珠+剑。
- **六界书等秘典**（以游戏出现为准）：世界观与通道情报硬通货。
- **尸块 + 矿物炼器**：战斗掉落与地图收集，强化装备的人间逻辑，不是系统外挂。
- **当票、唐门毒方、霹雳堂火器图纸、还神丹类药材**：低阶真硬通货。
- **古董真品（玉双龙首璜等）**：景天本职线；可换资源与好感事件。
每件写清当前阶能否用：魔剑初得反噬；五灵珠未齐不可当永久外挂；重楼级权柄低阶只闻其名。

**【隐藏剧情 · 伏笔】**
- 景天=飞蓬=龙阳三层身份，记忆分段解封（神魔之井、神树、剑冢等节点），禁止一觉全知。
- 雪见=夕瑶神树果实化身，弃婴真相与牺牲线同源。
- 龙葵红蓝分裂=千年剑中人格防御；结局可分离成全。
- 紫萱—长卿三世情缘，女娲使命与私情冲突；记忆封印是保护也是债。
- 邪剑仙=五长老邪念，点题「正也生魔」——蜀山制度本身产魔。
- 重楼取剑表面任性，实为逼飞蓬转世变强以履约；一切人间灾难是约战副产品。
- 回魂仙梦按好感改地图，说明「情」在本世界有物理权重。
- 完美结局接《问情篇》：新安当铺日常后约八年再生变；本档案主线止于仙三本传终局。
- 跨作：与仙一李逍遥线相隔约五十年，勿把一代人物无说明塞进三开局；女娲、蜀山、锁妖塔为系列共享底盘。

**【大事记时间线】**

**【叙事基调 · 雷区】**
基调：仙侠奇遇+轮回虐恋+门派烟火；笑中有泪，牺牲必须疼；鉴定古董的嘴贫与跳炉的血要同框。  
忌：把景天写成无脑无敌或纯舔狗；把重楼削成杂兵；把龙葵红蓝写成随机精神分裂笑话。  
忌：低阶锁妖塔顶层、新仙界、重楼本体满地走。  
忌：用「战力归零式削弱」解释重楼/飞蓬。  
忌：编造原作没有的神器名与必胜公式；电视剧（胡歌版）设定勿无标注覆盖游戏正典。  
忌：五结局改写成全员无条件团圆。  
最早切入锚点：渝州永安当景天仍是伙计、魔剑尚未入当之时；或唐家堡比武前后；中阶可从寻珠中段/雷州；高阶从锁妖塔前夜或铸剑前夜。

## 阶位切入点

> 阶位↔原作强度：凡人~江湖三流≈一阶；武林高手/唐门霹雳堂骨干/低阶修仙≈二阶；蜀山精英弟子/中期队伍/中阶妖≈三阶；蜀山长老级/邪剑仙前中/紫萱显化≈四阶；邪剑仙全盛/女娲全力/塔内天妖皇级≈五阶；补塔级/镇派合力/地区灾难≈六阶；飞蓬·重楼本体交手≈七阶；天帝/规则本源≈超阶。宁低勿高。世界顶点（**重楼**、神将期**飞蓬**）为跨阶存在，低阶切入一律「规避顶点·情报优先」。本世界仅覆盖：一阶、二阶、三阶、四阶、五阶、六阶、七阶。

**一阶（渝州永安当 · 当票与暗器）**
切入身份/时点：契约者以渝州跑腿伙计、小贩、唐门外围杂役或路过刀客身份切入，锚定「景天仍在永安当做伙计、雪见因紫砂壶盖等事纠缠前后」。
初始事件：永安当夜半有人砸门——有人要赎一块来历不明的玉佩，**唐雪见**的毒蒺藜先落在柜台，景天护当票喊价；你必须在「帮景天顶假货」「把玉佩卖给唐门眼线」「报警官却被门派压下」之间抉择。
开场白建议：「渝州的潮气混着当铺陈旧木香。景天把放大镜抵在鼻尖，刚夸完『这壶盖值三座山』，窗外飞来一簇毒蒺藜——唐雪见踩着门槛骂他骗子。你手里的铜钱忽然比刀还烫：今晚这票子，是当，是抢，还是逃？」
关键NPC立场：**景天**——把你当潜在客户或抢生意的；**唐雪见**——刁蛮试探，毒先礼后兵；**花楹**——只认雪见，乱摸挨咬；**唐坤**眼线——收保护费式盘问。
主线钩子/支线：钩子=永安当玉佩/紫砂纠纷如何滚成唐门事；支线①护送当铺押运避霹雳堂劫匪；②辨一炉假丹换唐门人情；③夜里替雪见找壶盖却撞见内宅密谈。
危险度/规避：中——死于毒、暗器与街头火并；须规避蜀山长老、重楼现身、锁妖塔内。
任务方向/奖励：活过唐门一周、拿到合法当票利润或一件无毒的护身暗器；奖励=银两、低阶解毒散、唐门外围人情，不发魔剑。

**二阶（唐家堡—霹雳堂 · 毒与火器）**
切入身份/时点：以唐门外姓弟子、霹雳堂逃兵、游方大夫或受雇保镖切入，锚定「雪见卷入掌门之争、霹雳堂炼毒人/夺五毒珠阴谋前后」。
初始事件：唐家堡演武场比武夺帅前夜，**罗如烈**暗桩在井里下慢性毒；你发现水源异常，必须选择通知**唐坤**、私下告诉**唐雪见**，或把情报卖给蜀山弃徒**徐长卿**换路引。
开场白建议：「演武场的沙里埋着蒺藜，火药味从后山飘来。唐雪见攥着比武名帖手心全是汗，景天在角落里数他那一文钱当来的『破剑』——你知道井水甜得不对。今晚谁先喝水，谁就先变成别人的棋子。」
关键NPC立场：**唐雪见**——要盟友但防夺家产者；**景天**——想帮又怕事；**徐长卿**——可合作清邪，但查你来历；**罗如烈**——要封口或收买。
主线钩子/支线：钩子=比武结果是否改变雪见被揭弃婴身世的节奏；支线①截火药车；②护花楹不被抓去炼珠；③假死逃出堂口黑牢。
危险度/规避：中高——火器阵与群毒；规避邪剑仙本体、塔顶、重楼。
任务方向/奖励：保住雪见或换得离开渝州的路引；奖励=唐门毒方残卷、火器零件、解毒珠，不发五灵珠本体。

**三阶（寻珠中段 · 江湖成队）**
切入身份/时点：以散修、蜀山记名弟子、商队护卫或地方妖籍归顺者切入，锚定「五人初步结伴寻五灵珠、魔剑已醒、龙葵可现身前后」。
初始事件：某州坊市拍卖「伪土灵珠」，**邪剑仙**眼线搅局；真珠线索在妖巢。**龙葵**（蓝）求你帮「王兄」鉴宝，**红葵**半夜威胁你别靠近魔剑。
开场白建议：「坊市灯笼把魔剑映成一截冷铁。龙葵的声音从剑里溢出，软得像在道歉；转眼又变成红衣少女按住你的脉门：『碰他一下，我让你后悔一千年。』远处蜀山令旗与霹雳堂残部同时出现——五灵珠的名字，开始用命标价。」
关键NPC立场：**景天**——收可靠帮手，防夺剑；**龙葵**——护兄偏执；**紫萱**——温和观察，测你心性；**徐长卿**——按蜀山规矩分功；**重楼**（远观）——只对飞蓬转世感兴趣。
主线钩子/支线：钩子=伪珠案是否暴露邪剑仙布局；支线①护紫萱采药；②与长卿清一处低层妖巢；③替雪见送家书却被唐门旧部追杀。
危险度/规避：高——中阶妖与门派精英；规避锁妖塔深层、重楼交手。
任务方向/奖励：确认一枚真灵珠方位或换蜀山临时令；奖励=五灵仙术残卷、中阶兵器、队伍短期同盟。

**四阶（锁妖塔外围 · 封印松动）**
切入身份/时点：以蜀山外门执事、地方镇妖官、女娲一族联络人或重伤归山的剑客切入，锚定「五灵珠将齐未齐、锁妖塔封印松动、邪剑仙活跃前后」。
初始事件：塔周夜风有腥；你被派守阵旗，**邪剑仙**残影袭营，**紫萱**以灵力稳住一角阵眼，要求你在「撤民」与「死守旗」中选——撤则失功，守则可能被夺魂。
开场白建议：「锁妖塔的影子比云还重。紫萱的手指按在阵纹上发白，徐长卿的剑出了鞘却不敢乱挥——塔里有东西在笑。你的令牌烫得像烙铁：这一夜守不住，渝州以北都要重新学『什么叫妖灾』。」
关键NPC立场：**徐长卿**——令如山，可共死；**紫萱**——护生优先；**景天**——想冲塔又怕连累人；**邪剑仙**——要旗、要剑、要乱。
主线钩子/支线：钩子=阵旗是否被夺决定入塔时间表；支线①疏散山民；②夺回被污的灵珠容器；③识破蜀山内部被邪念沾染的执事。
危险度/规避：高——四阶压迫、规则杀；严禁硬刚重楼，塔内深层限入。
任务方向/奖励：保住外围一阵、获入塔资格或女娲侧护符；奖励=镇妖符、中高阶药材、蜀山记功。

**五阶（锁妖塔内 · 邪剑仙）**
切入身份/时点：以获准随队入塔的镇妖精英、女娲护卫或被魔剑「选中」的剑客切入，锚定「入塔讨伐邪剑仙、天妖皇级威胁前后」。
初始事件：九层杀机中，**邪剑仙**抛出选择——交出魔剑信息换一条生路，或看**唐雪见**/**龙葵**被剑气锁定。你只有三息决定：欺敌、替挡、或破一阵眼改道。
开场白建议：「塔砖缝里渗出旧日妖血。邪剑仙的声音像许多长老叠在一起：『正也生魔，你还不懂吗？』景天把魔剑横在身前，龙葵的红衣猎猎作响。你的五灵法阵站位空了一角——那一角，不是给英雄的，是给肯死的人的。」
关键NPC立场：**邪剑仙**——诱降与屠杀并行；**景天**——护人优先于杀敌；**龙葵/雪见**——可互相争着挡剑；**紫萱**——算补塔代价；**徐长卿**——蜀山脸面与人命撕扯。
主线钩子/支线：钩子=邪剑仙是否提前夺走关键剑意；支线①救被困剑客残部；②稳一层阵眼；③把情报送出塔外给五长老（可能反被邪念利用）。
危险度/规避：极高——五阶主敌与塔内规则；规避新仙界、重楼本体对轰。
任务方向/奖励：协助灭邪剑仙一阶段或夺回剑匣；奖励=高阶剑丸、灵珠共鸣残片、蜀山重诺（可兑现一次庇护）。

**六阶（铸剑前夜 · 补塔代价）**
切入身份/时点：以蜀山长老代行、女娲族使者、魔界中立使节（非重楼本体）或见证铸剑的炼器师切入，锚定「两剑受损、剑炉前抉择镇妖剑/魔剑、紫萱欲补塔前后」。
初始事件：剑炉烈焰中，**雪见**与**龙葵**同时上前；**紫萱**已写好以身填阵的诀。你掌握一枚可暂替的灵物——用则自己灵力大损，不用则必有一人跳炉。
开场白建议：「炉火把所有脸映成同一颜色。重楼的气息在云上发笑，像在等一场好看的戏。紫萱把水灵珠按在心口，徐长卿的声音破了音。你手里那点『可代替牺牲』的东西突然重如六界——你要改命，就得先交出自己的命。」
关键NPC立场：**紫萱**——使命压过私情；**徐长卿**——求你拦她；**景天**——谁跳炉他都要疯；**重楼**——可救人可袖手，看你是否敢赌；**花楹**——只懂「雪见不能死」。
主线钩子/支线：钩子=铸剑选择与补塔人选（蝴蝶极强）；支线①稳剑冢外阵；②送重楼一句话换「晚一刻动手」；③保住花楹不被炉火波及。
危险度/规避：极高——六阶地区灾难级灵压；禁止宣称秒杀重楼。
任务方向/奖励：换得一人暂活或炉火稳定一刻；奖励=女娲灵物残、铸剑见证身份、高阶疗伤圣药（仍不发重楼权柄）。

**七阶（新仙界 · 天帝决侧翼）**
切入身份/时点：以神界巡卫残部、魔界观战使、女娲大祭酒或被景天羁绊牵引的「飞蓬旧识碎片」身份切入，锚定「景天赴新仙界履约挑战重楼前后」。
初始事件：新仙界玉阶开裂，**重楼**变身，**景天**持剑而上；你的任务不是替他赢，而是在崩塌中护住一条回人界的缝——或把「复活所需的代价清单」在交手前递给重楼（可能被一掌掀开）。
开场白建议：「神界的风没有人间潮气，只有剑意与魔息对撞出的白光。重楼笑着叫飞蓬的名字，景天却用渝州口音回了一句当铺行话。建筑在碎。你站在侧翼，清楚自己不是这场约的主角——但缝要是合上，渝州的竹林就永远等不到人。」
关键NPC立场：**重楼**——只认对手强度与诺，契约者=尘埃除非你碰到约本身；**景天**——要你护人界出口而非抢功；**飞蓬**记忆残响——督促履约；**紫萱/雪见/龙葵**（依是否存活）——要你带话或带骨灰都不一定。
主线钩子/支线：钩子=是否打赢重楼决定完美线与复活；支线①护崩塌中的弱神吏；②记录重楼变身破绽给景天一次；③守住人界缝至胜负分晓。
危险度/规避：贴近顶点——**重楼/飞蓬神将级=超阶巅峰：存在·情报优先/条件性胜利**，严禁硬刚本体与战力归零解释；天帝体系不写可击败。
任务方向/奖励：保住回界缝或促成「可复活」条件之一；奖励=神魔交手残留的剑意领悟、六界通道临时通行证、完美线相关的人间安顿权（当铺/竹林级），绝不发魔尊重力复制品。
