## 剧情

**【作品来源】**
《全职法师》作者**乱**，2015 年起于起点中文网/创世中文网连载，约 2024 年完结；公开统计约 3370 章量级、七百余万至近八百万字。搜笔趣阁收录书页标注作者乱、**已完结**、约 **796.26 万字**，最新章记为第 3233 章「人间,惹不起了」（平台分章与起点统计可略有出入）。文风：都市魔法外壳下的热血升级、猎魔与校园并线、国际势力倾轧、海妖与图腾长线；读者常议「作者乱」指后期节奏与支线膨胀，本档案**以小说正典设定为准**，动画/漫画仅作名场面交叉，不与改编私设混写。

**【世界定位】**
现代都市魔法觉醒世界。表面是「魔法高考 + 都市猎魔」，底层是国境外的妖魔帝国、黑教廷献祭链、海洋位面压迫，以及圣城 / 帕特农等国际权柄。主角**莫凡**（穿越者）佩戴「小泥鳅」吊坠，天生每晋升一大阶可觉醒两系，从博城学渣成长为多系禁咒、终局**邪圣王 / 法神**级存在。一句话：以星子连轨为功法、以魔石为石油、以城市警戒色为生死线的残酷魔法地球——契约者是变数，不是免死金牌。

**【世界观 · 力量体系】**
力量本源：**星尘与精神世界**。法师觉醒后在精神海凝结星子，按系别连成结构，再以魔石、冥修、精魄、元素种强化。普通人多在十六岁前后于魔法高中觉醒；绝大多数一生主修一系，双系 / 多系属天赋级稀有。

**法师纵向等级（主轴）**  
学徒 → **初阶** → **中阶** → **高阶** → **超阶** → **禁咒**（及极少数法神 / 创造级、黑暗位面帝王权柄）。每大阶内自然 1～3 级；可用妖魔精魄等将星子强化至 4 级以上（威力近翻倍，影响伪禁咒融合上限）。半禁咒 / 伪禁咒（多超阶融合 + 强化）可逼近全禁咒破坏力。

**精神世界结构（横向成长）**  
- **星子**（学徒）：脑海七颗星子，与星尘沟通，尚难正式施法。  
- **星轨**（初阶）：七星连成一条星轨。  
- **星图**（中阶）：七轨成图，脚底六芒星显化。  
- **星座**（高阶）：七图成座，星座环身。  
- **星宫**（超阶）：七座成宫，星宫映天，毁灭气息可震慑全城。  
- **星宿 / 星神殿**（禁咒）：七宫汇流，禁忌之力可撕空间，受国际禁咒公约与政治监管。

**魔法系别大类**  
- 元素：雷（极稀有）、火、冰、风、土、水、光。  
- 白魔法：治愈、心灵、植物、祝福。  
- 黑魔法：暗影、诅咒、亡灵、毒、**恶魔**（莫凡线关键）。  
- 次元：召唤、空间、音、混沌。  

**元素种与领域**  
灵种（多对应中阶质变）→ 魂种（高阶，部分带**领域**）→ 天种（超阶及以上「种王」，可附天种禁界）。每档再分伪 / 亚 / 小 / 大。领域内己方魔法暴涨、敌方被压制，是高阶对决分水岭。穆宁雪天生冰种可成长至魂种 / 天种；莫凡则靠玫炎、暴君荒雷、流星绯火、劫炎、星之尘埃等一长串种与领域叠乘。

**魔具与外物**  
斩、翼、臂、履、铠、盾、凝神等；魔石是能源硬通货；精魄强化星子；亡魂器皿等属管制边缘。契约兽（小炎姬、阿帕丝、狼族军团等）是第二战力曲线。合体附身、超然力、伪禁咒融合属高阶玩法。

**破坏力粗标（宁低勿高，攻击系估算）**  
- 初阶：火器 / 手雷到小建筑级，清奴仆群。  
- 中阶：商场 / 街区局部，可抗战将。  
- 高阶：单栋到街区，可屠普通统领（顶尖高阶另论）。  
- 超阶：干扰 / 瘫痪一城运作，威胁君主。  
- 半禁咒：大都市岛级。  
- 全禁咒：常被比作核弹量级灾难，区域生命绝迹向，需监管。  

**妖魔等级**（与法师不完全对等，同级妖魔通常更耐打）：奴仆 → 战将 → 统领 → 君主（含至尊君主）→ 帝王（及世界屋脊级表述）。成长阶段：婴儿—幼年—成长—成熟—巅峰。群居：巢 → 族群 → 部落 → 帝国。城市以警戒色标妖魔密度（白 / 橙 / 红 / 紫 / 黑等）。经验法则：一只奴仆可虐杀多名普通初阶；统领完虐普通高阶；君主需多名超阶或顶尖超阶；帝王对标禁咒联手 / 图腾 / 神魂 / 恶魔等特殊手段。

**死亡与复活**  
低阶法师与普通人一击可死；高阶靠魔具、空间手段、契约兽与组织后援，不是无限复活。帕特农神庙系存在复活 / 神魂权柄，但有代价、有政治、有失败案例。恶魔系觉醒伴随失控与人性风险。世界顶点（多系禁咒、邪圣王、帝王级妖魔、黑暗王）不可被「战力归零式削弱」。

**乐园阶位映射**（对照《阶位战力图鉴》，宁低勿高；本库仅一～七阶）：  
- **初阶** ≈ **一～二阶**（学徒 / 初阶前期偏一阶；初阶圆满偏二阶）  
- **中阶** ≈ **二～三阶**  
- **高阶** ≈ **三～四阶**（四阶为高阶圆满～超阶门槛质变）  
- **超阶** ≈ **五～六阶**（五阶超阶稳固；六阶顶尖超阶 / 半禁咒 / 准禁咒）  
- **禁咒** ≈ **七阶**  
- 多系禁咒巅峰、法神、邪圣王、帝王级海妖、黑暗王级 = **超阶巅峰：存在·情报优先 / 条件性胜利**，严禁写成战力归零。同境多系 / 天种 / 领域 / 图腾可越阶，映射仍按破坏力落位。低阶任务勿发禁咒触媒、小泥鳅原理、帝王级本体战；一阶只给学籍、魔石月耗与奴仆级清巢即可用。

**【地理 · 舞台】**
由小到大：  
- **博城**：开篇主场。天澜魔法高中、穆氏地方网、黑教廷诱导的血色警戒 / 博城之灾起点；莫凡与**穆宁雪**少年纠葛地。一～二阶日常。  
- **魔都（上海一线）/ 明珠学府**：顶尖学府、火院、吸血鬼与都市猎魔；**青天猎所**等猎者组织辐射。二～四阶。  
- **杭州 · 西湖下**：图腾**玄蛇**栖息地，审判会**唐月**守护线。三～五阶。  
- **古都（西安一线）**：卡片锚点。亡灵潮、死街、煞渊、古老王（嬴政相关亡灵帝王线）、外城沦陷。三～五阶核心副本。  
- **国府与亚洲魔法协会网络**：**邵郑**等议长层、国馆、世界学府大赛选拔、海岸战略。四～七阶政治。  
- **灼原 / 火焰圣地线**：寻**小炎姬**（炎姬）的火种舞台。三～四阶。  
- **国际历练链**：日本、埃及（胡夫金字塔 / 冥界之门）、欧洲交流、秘鲁 / 中美等；**世界学府大赛**与威尼斯相关赛点。四～六阶。  
- **帕特农神庙（希腊权柄）/ 雅典一线**：神女、骑士殿、神印礼赞、伊之纱与叶心夏线。五～七阶。  
- **圣城**：西方秩序审判与天使级战力。六～七阶。  
- **海洋前线**：飞鸟市海妖、厦门 / 上海等沿海海洋之灾、海妖帝国压迫。五～七阶战争副本。  
- **图腾节点**：澜阳、天山、旭岛、蛾凰 / 神鹿 / 青龙等；对抗海洋的战略资产。五～七阶。  
- **北原 / 西部沙漠**：冷爵、蛾凰等线。四～六阶。  
- **极南、昆仑、黑暗位面**：后期禁咒任务与黑暗王线。六～七阶。  
- **凡雪山 / 凡雪城**：莫凡城主线与保卫战，有地可守的责任锚。五～七阶。  

写场景口吻：魔石灯与霓虹并存、校徽与猎者徽章、警戒色警报、死街腐气、海风腥咸、神庙白石与黑教廷红袍。底层是学费、魔石配额、家族联姻与猎杀悬赏；中层是学府大赛、国府征召、城防战；高层是禁咒政治、图腾复苏与位面级灾变。AI 落笔先定「高中—猎所—古都—国际—海洋—位面」层级，再定冲突强度。

**【世界剧情线】**
① **魔法高中 · 井底觉醒**  
莫凡穿越后发现世界已是魔法文明。天澜 / 博城魔法高中觉醒：**雷系 + 火系**天生双系；与**穆宁雪**少年约定与纠葛并行；校内资源、家族背景碾压学渣。名场面：星子连轨、雷火初显、与**宇昂**等地圣泉资格争夺。世界状态：初阶是学生，中阶是老师级，高阶如神。因果：**小泥鳅**坠作为冥修与异常容器埋线；莫凡「有仇必报、重情重义、风流但不弃底线」底色成型。契约者在此阶段正确目标是连轨、打工赚魔石、活过校内决斗，不是挑战黑教廷执事。

② **博城之乱与明珠入学**  
黑教廷布局导致博城灾难（血色警戒相关），城市秩序崩坏，何雨等普通人与法师双线惨死，**张小侯**因此投身军方。莫凡考入**明珠学府**。家庭线（父亲底层挣扎、非血缘妹妹**叶心夏**）与阶层落差推他必须变强。阶段结局：离开纯高中井底，进入学府—猎者双轨；与黑教廷的梁子钉死。

③ **青天猎所与猎者生涯**  
加入青天猎所，与**灵灵（冷灵灵）**等搭档接悬赏、清妖、与黑教廷外围交锋。中阶成长、精魄与元素种争夺日常化。名场面：历练任务、猎队背刺与合作、魔石与名望同步增值。**华展鸿（华军首）**等军方高层日后从猎者 / 国府通道与莫凡结缘。主题：悬赏金是血味的，名号值钱也会招死。

④ **图腾玄蛇 · 伙伴线萌芽**  
杭州线应**唐月**（审判会，唐忠侄女）请求，保护蜕皮期脆弱的**图腾玄蛇**（西湖下，体量如摩天楼宇，至尊君主级向）。莫凡获得图腾相关机缘与「大家伙」式羁绊。主题：中国图腾不是装饰，是未来海洋之灾的战略储备；敬，不奴役。

⑤ **火院、小炎姬、魔都暗潮**  
转火院、赴灼原寻找天地火焰圣灵**小炎姬（炎姬）**，后为第一契约兽，可合体附身；魔都吸血鬼事件、火焰魔女线并行。召唤系与契约兽养成成为莫凡战力第二曲线。**牧奴娇**（明珠植物系、牧氏千金）等学府线交织，提供资源与情感支线，但主情仍锚定穆宁雪 / 叶心夏。

⑥ **古都浩劫（卡片锚点卷）**  
为救高中好友**张小侯**，莫凡放弃国府大赛提名之争赶赴古都。黑教廷（**撒朗**等）借古老王与**煞渊**引八方亡灵围城：外城数百万亡灵、**七百人过死街**、国府与协会联防仍至绝境。**苏小洛**等古都线人物与张小侯情义线并行。莫凡跳入煞渊，**恶魔系**完全觉醒，破坏毁灭古都的关键布局。名场面必须含：入城门槛与警戒、死街活人统计变成减法、煞渊入口、亡灵帝王级压迫、出逃 / 清场时的追杀与论功。此卷是中阶向高阶质变副本：多方云集、规则杀与人心杀并行。阶段结局：古都浩劫英雄名号；恶魔系成为双刃剑；黑教廷仇恨升级；**华展鸿**等因此更重视莫凡，国府通道打开。

⑦ **国府、国际历练与世界学府大赛**  
飞鸟市海妖、日本历练、中国国馆、多国历练链。国府队由**艾江图**（空间 / 诅咒等，军方艾氏，艾图图之兄）任队长；队内有莫凡、**穆宁雪**、**赵满延**、牧奴娇等。**世界学府大赛**中国队夺冠（夺宝赛「凡雪盗团」等名场面），获帕特农相关**神印礼赞**资源，国际声望暴涨。主题：个人恩怨被国家与协会利益吞没，名号开始招死也招招募。

⑧ **帕特农之乱与叶心夏线**  
营救叶心夏，卷入帕特农神庙神女 / 神魂政治；灭黑暗死神等强敌。叶心夏治愈 / 心灵 / 祝福系成长，真实身世线逐步揭开：父**文泰**（帕特农前任大贤者 → 黑暗位面黑暗王布局）、母**叶嫦 / 撒朗**（黑教廷红衣）、**伊之纱**（文泰之妹、神女野心家）的嫉妒与陷害。小炎姬等曾为救人心夏付出燃烧级代价，情感锚点加固。

⑨ **西部沙漠、蛾凰、北原冷爵**  
沙漠阴谋、月蛾凰图腾线、北原危机与黑教廷红衣**冷爵**等。图腾搜寻从「奇闻」变为「国家战略」。**邵郑**委托寻找沉睡图腾，为两万公里海岸线备战。

⑩ **埃及冥界之门、欧洲交流与图腾长线**  
埃及胡夫金字塔、冥界大门、法老势力与美杜莎相关线；**阿帕丝**（美杜莎一脉，心灵 / 石化）作为契约兽 / 伙伴的地位上升。欧洲交流、元素之都等推进图腾神鹿等线索；海洋之灾预警升级。

⑪ **海洋之灾全面爆发与沿海保卫**  
旭岛、厦门海洋救援、上海海洋之灾、夏威夷救援等：海妖帝国压迫人类海岸线。图腾圣兽（含青龙等）复苏、**凡雪山**保卫战、亚洲协会协调。莫凡城主 / 凡雪城线提供「有地可守」的责任，而非无限浪迹。**赵满延**以光 / 水 / 土防御三系 + 图腾**霸下**木鱼等成为「龟壳」级队友。

⑫ **圣城审判、南美战乱与红魔线**  
圣城政治审判、南美联邦战乱、**红魔一秋**布局飞升 / 八魂等。国际魔法秩序的伪善与实力说话暴露无遗。一秋相关线为莫凡恶魔系晋升**邪圣王**埋因果（善恶八魂收集、义魂等）。

⑬ **极南征召、神女礼赞与黑教廷终局**  
极南之地征召；帕特农神女线收束。叶心夏当选神女并一度握有黑教廷教皇位相关权柄，在礼赞节点设计屠灭黑教廷高层（含撒朗派系），以血腥手段铲除长期反人类邪教——「温和外表、果决屠刀」完成弧光。黑教廷作为贯穿全书的人为灾源被结构性抹除，残余与仇恨仍可能以个人线回流。

⑭ **禁咒 · 暗位面与昆仑**  
暗位面探索、昆仑修行，与**穆白**（后堕暗 / 黑暗魂圣协作）等联手揭秘黑暗位面；黑暗王**文泰**身份线落定。莫凡多系禁咒成型，恶魔系至帝王权柄，成就**邪圣王**，逼近 / 达到法神（创造法神、半全职 / 全职多系）境界。对圣城大天使米迦勒等终局强敌：击败但不必屠尽，可废除魔法使其凡人化——胜利昂贵且带政治后果。

⑮ **终局与日常**  
莫凡与穆宁雪、叶心夏结为夫妻；小炎姬、阿帕丝等契约兽与狼族军团仍是战力与羁绊。人间「惹不起了」式收束：法神 / 邪圣王级威慑下仍有未填尽的海妖 / 极南余波，但主线从「活命升级」转为「守护与余波清理」。低阶禁止硬刚法神与帝王本体。

⑯ **微观政治贯穿（日常可介入）**  
魔石借贷、家族联姻、学府积分、猎所分成、假精魄、警戒色撤离、城防徭役、协会执照、国际签证式通行令、悬赏榜买卖情报。AI 优先这些小事件，而不是一上来禁咒互轰。

⑰ **蝴蝶效应边界**  
契约者可改局部：救一名猎者、抢一件非核心魔具、让某场清巢战易主、提前一小时拉响警戒。不可无代价改写：古都浩劫核心因果、恶魔系觉醒节点、叶心夏铲除黑教廷的权柄逻辑、海洋之灾的战略压迫、邪圣王 / 法神级顶点战力。改了局部，原作因果以追杀、取代、更惨的回流方式补偿。

**【世界剧情线·分卷作战图】**
**卷一：高中与博城**——觉醒雷火、地圣泉、博城灾难、入明珠。可介入：校内决斗、家族羞辱、灾难撤离。危险：规避高阶黑教廷执事。

**卷四：国府与大赛**——国馆、艾江图带队、国际历练、世界学府大赛夺冠。可介入：选拔赛、情报买卖、针对中国队的暗手。  
**卷五：帕特农与身世**——救心夏、神庙政治、撒朗 / 伊之纱 / 文泰血脉阴影。可介入：神女派系站队、复活资源争夺。  
**卷六：图腾与海洋**——多图腾复苏、沿海沦陷与反推、凡雪山。可介入：城防、护送图腾祭祀、海港爆破。  
**卷七：禁咒与终局**——暗位面、昆仑、邪圣王、法神、黑教廷覆灭余波。可介入：情报、护航、清理残党；本体战仅七阶条件性参与。

- **魔具（斩翼臂履铠盾凝神等）**：战力外骨骼；有品级与认主。

- **帕特农神印 / 礼赞相关**：大赛与神庙线权柄物。  
- **图腾信物 / 祭祀器 / 霸下木鱼**：唤醒或安抚图腾兽；赵满延线关键。

- **城主令 / 凡雪城权柄**：领地与征兵。

每件写清：谁想要、抢到的后果、当前阶能否用（低阶只用魔石、低阶魔具与猎者徽章，高阶才碰神印与禁咒触媒）。

**【隐藏剧情 · 伏笔】**
- 小泥鳅真实来历与位面级功能，远超「好用的吊坠」。  
- 天生双系与恶魔系是否被更高存在「选中」；邪圣王八魂是布局不是偶然。  
- 古都浩劫是黑教廷全球献祭链的一环，不是孤立副本。  
- 叶心夏神魂 + 撒朗血脉 + 文泰黑暗王线，是「拯救者出身深渊」的结构反讽。  
- 图腾复苏速度与海洋之灾的赛跑：慢一步即沿海文明塌方。  
- 圣城与帕特农的「秩序」对东方禁咒新贵的忌惮。  
- 穆白堕暗 / 魂圣选择，是把人间留给莫凡的对照。  
- 终局后的世界：黑教廷灭、海妖仍在、法神威慑下的冷和平。  
- 跨媒介（动画多季等）细节以小说正典优先，冲突时从小说。  
低阶只给线索，不直接抛邪圣王本体战力。

速查：锚点 = 古都浩劫；质变 = 恶魔系；格局 = 学府大赛与图腾；收束 = 邪圣王与人间惹不起。

**【地理补述】**
博城街道与高中钟楼；明珠学府院楼与训练场；古都城墙—死街—煞渊垂直恐惧；西湖底玄蛇巢；神庙白阶与骑士殿；圣城法理大厅；凡雪山风雪；上海 / 厦门海堤在警戒色下的灯火；暗位面非欧几何空间。写战斗先写地形与补给线：魔石运不上来，禁咒也放不出第二发。

**【势力补述】**
学府要成绩与赞助；协会要秩序表象；国府要国土完整；军方要编制与军功；猎所要分成；家族要联姻垄断；教廷要献祭与恐慌；神庙要神女正统；圣城要解释权；图腾要祭祀与时间；海妖要陆地。莫凡策略：用国家任务换资源，用兄弟线换人性，用恶魔与多系换谈判桌资格，从不真正属于单一势力。

**【主要人物·对话立场速查】**
对莫凡：可并肩猎魔，勿打小泥鳅与心夏 / 宁雪的坏主意；他记仇也记恩。对穆宁雪：冰面下是执念，激将不如交底。对叶心夏：别把温柔当可欺，神女权柄下无人幸免。对艾江图：服从任务逻辑，勿碰艾图图。对赵满延：钱可谈，背刺兄弟不可。对撒朗系：先断献祭链再谈脸。对国府 / 军方：任务换资源，合同看细则。对神庙：站队即信仰审查。对海妖：谈判窗口极短。对图腾：敬，不奴役。对红魔一秋：别以为自己在下棋。

**【世界剧情线·古都七日节奏】**
D1 外围亡灵潮与难民；D2 外城失守；D3 死街与补给线；D4 古老王压力上升；D5 煞渊真相；D6 跳渊与恶魔；D7 清场论功与追杀残留。契约者最优解多在 D3～D6 之间完成「救人 + 断节点 + 活命」。

**【世界剧情线·海洋战线经济】**
每一米海堤 = 魔石燃烧率；图腾未醒时人类用命填；图腾醒后是「用神话换时间」。禁止把海妖写成无脑冲车：它们有帝国、有君主、有战术。

**【世界剧情线·魔法高中生存细则】**
学费、魔石月耗、训练场预约、家族赞助名额构成「合法的弱肉强食」。未觉醒者在社会底层；只觉醒废系者被嘲。莫凡的雷系稀有度本身就是资本，火系补足爆发。小泥鳅让他冥修效率离谱，必须用「熬夜学渣」人设掩盖。契约者若同校：可组队刷奴仆级、可卖假情报、可在博城灾难夜选逃或守——守住一条街不等于改写教廷剧本。

**【世界剧情线·猎者经济】**
猎所接单 = 预支棺材本。战将级妖魔的精魄能换一套中阶魔具首付；分赃不均当场反目是常态。青天猎所有规矩也有灰色：私藏元素种、调包残魄、把新人当诱饵。莫凡在此学会「名号值钱也会招死」。赵满延式财迷队友提供补给线喜剧，也提供关键时刻的土系硬墙。灵灵卖的不是地图，是活命概率。

**【世界剧情线·恶魔系双刃剑】**
恶魔系不是酷炫皮肤，是人性与力量的对赌。古都后莫凡获得破局手段，也获得协会侧目、教廷追杀、盟友不安。正文写恶魔：先写呼吸与瞳色失控，再写敌阵崩溃。邪圣王是「至邪至圣」的完成态，不是无代价开挂。契约者妄图复制恶魔：缺小泥鳅与血债因果则狂化或被猎杀。

**【世界剧情线·图腾与国运】**
玄蛇、蛾凰、神鹿、霸下、青龙等不是宠物图鉴，是中国对抗海洋帝国的「沉睡战略导弹」。唤醒要祭祀、时间窗、超阶护法与政治妥协。图腾之间有兄弟阋墙，人类不能当饲养员颐指气使。莫凡与玄蛇的情分是「救过命的伙伴」，不是主奴。

**【世界剧情线·帕特农与复活政治】**
神庙贩卖的不仅是治疗，是「谁配被复活」的解释权。神女之位 = 全球话语权。叶心夏的神魂使她成为棋子也成为棋手；撒朗血脉使她被教廷与神庙双重盯防。伊之纱的野心是神庙内鬼的标本。莫凡多次为她撕破国际礼仪。契约者介入神庙：先选派系，再谈任务——骑墙者死得最快。

**【世界剧情线·海洋之灾作战条令】**
海堤法师阵列分层：初中阶清杂兵，高阶锁君主技能，禁咒留给破城级目标。魔石车队被截 = 整段防线哑火。平民疏散路线是支线主菜：救下十万与救下一个具名 NPC，叙事权重不同但道德账都要记。凡雪城提供「有家要守」的锚，逼主角从浪子变城主。

**【世界剧情线·黑教廷覆灭后的真空】**
心夏屠高层不等于思想灭绝。残余灰衣、私设熔池、冒名红衣、海外分舵仍可制造局部灾难。圣城与协会会争「谁有权接收教廷遗产」。莫凡成家后仍会被这些余波叫走——法神不是退休。

**【世界剧情线·因果长链】**
莫凡每多觉醒一系，就多一张国家与国际的牌，也多一重被解剖的风险。雷火让他活过高中；暗影召唤让他活过猎所；空间恶魔让他活过古都；土混沌与后续诸系让他走进禁咒桌；八魂与邪圣王让他坐上黑暗权柄的椅子却仍守人间。小泥鳅始终是暗线心脏。穆宁雪是冰与执念的镜；叶心夏是治愈与屠刀的镜；小炎姬是火与牺牲；阿帕丝是异族与人性；张小侯是兄弟；赵满延是盾；艾江图是编制与纪律；唐月是启蒙与图腾。黑教廷用人祸逼出英雄，也被英雄之妻从内部剖开。海洋用种族战争逼出图腾，图腾用神话时间换人类文明续命。终局法神废除敌手法而非只求屠戮，是「秩序可以重写，仇恨不必无限循环」的作者立场——但海风仍腥，余波仍在。

时点速查：高中博城 = 一～二阶；猎所玄蛇炎姬 = 二～三阶；古都浩劫 = 三～四阶；大赛帕特农 = 四～五阶；海洋图腾 = 五～六阶；暗位面邪圣王法神 = 六～七阶。

**【叙事基调 · 雷区】**
热血但不洗白阶层；猎魔有代价；恶魔系有失控风险。忌把莫凡写成纯白圣母或无脑后宫机器；忌低阶秒杀禁咒；忌公开解剖小泥鳅原理；忌古都 / 海洋当背景板无尸臭与魔石账；忌黑教廷灭后立刻「世界和平」忽略海妖与圣城；忌把「艾英峰」一类无出处假名塞进正典（国府队长正典为**艾江图**）。最早可玩锚点：魔法高中 / 猎所，或古都浩劫期。写战斗先写星轨完成度与魔石余量；写政治先写协会执照与国府征召。全文真名优先：莫凡、穆宁雪、叶心夏、小炎姬、张小侯、赵满延、艾江图、唐月、穆白、灵灵、牧奴娇、撒朗、文泰、邵郑、华展鸿、阿帕丝、伊之纱、红魔一秋。

## 阶位切入点

> 阶位↔境界：一阶≈学徒 / 初阶前期；二阶≈初阶圆满～中阶初；三阶≈中阶～高阶初；四阶≈高阶圆满～超阶门槛；五阶≈超阶稳固；六阶≈顶尖超阶 / 半禁咒 / 准禁咒；七阶≈禁咒。世界顶点（多系禁咒巅峰、邪圣王、法神、帝王级海妖、黑暗王）为**超阶巅峰：存在·情报优先 / 条件性胜利**，低阶切入一律规避顶点，严禁用「战力归零」式解释去硬刚。

**一阶（学徒 / 初阶前期 · 魔法高中线）**  
切入身份/时点：契约者以「博城或同级城市魔法高中新生 / 插班生 / 校工旁听」切入，锚定**莫凡**刚觉醒雷火、校内资源战开始时。  
初始事件：觉醒仪式上你的星尘颜色引发旁议；有人出高价买你「双系传闻」的情报；穆氏相关学生与你争训练场配额。远处钟楼第三次敲响时，邻桌痞气男生掌心里雷与火同时亮了。  
开场白建议：「钟楼响第三遍时，教室里所有人都闭上眼等星尘落下来。你听见邻桌那个痞里痞气的男生低声骂了句脏话——他掌心里，雷与火同时亮了。老师的粉笔停在半空，像被钉住。」  
关键NPC立场：**莫凡**——可并肩亦可抢资源，防心极重；**穆宁雪**——清傲，不轻易假以辞色；**宇昂**——踩人；觉醒导师——要成绩；黑教廷外围灰衣——灾难前的影子。  
主线钩子/支线：连成第一条星轨；地圣泉资格周边；支线＝魔石打工、校内决斗、家族联姻窥探。  
危险度/规避：中——真危险来自家族与灰衣，不是普通同学；须规避中阶以上教员真怒、黑教廷执事。  
任务方向/奖励：站稳学籍；低阶魔具；小额魔石；初阶功法笔记；一句「那个双系也看了你一眼」（比魔石烫）。

**二阶（初阶圆满～中阶 · 猎所 / 明珠外围线）**  
切入身份/时点：以「见习猎者 / 明珠外围任务生 / 青天猎所临时工」切入，锚定莫凡入猎所、接清巢单时期。  
初始事件：一张战将级悬赏被三队同时接；你的队伍在分赃精魄时内讧；警戒色从橙跳红。柜员只问两句：几人队，接不接过夜单。  
开场白建议：「猎所大厅的魔石灯从来不亮到角落。你把徽章拍在柜台上时，柜员只问两句：几人队，接不接过夜单。门外风里有腥，那是城外奴仆级妖魔的味道——也是你下个月的学费。」  
关键NPC立场：**灵灵**——卖路线与活命概率；**莫凡**——中阶上升期竞争者 / 临时队友；青天猎所领队——分成冷酷；黑衣教士探子——诱骗献祭；**牧奴娇**学府线——可远观勿亵玩。  
主线钩子/支线：清巢；护商；支线＝假精魄鉴定、魔具租赁高利贷、吸血鬼巢外围探路。  
危险度/规避：高——规避高阶妖魔统领、红衣级；勿私藏元素种被猎所记账。  
任务方向/奖励：中阶门槛资源；元素灵种线索；猎者执照升级；一次「灵灵标了价的活路」。

**三阶（中阶～高阶初 · 图腾 / 火院 / 大赛前夜）**  
切入身份/时点：以「独立猎师 / 学府精英队 / 审判会外协」切入，锚定玄蛇线、小炎姬线或国府观察期。  
初始事件：西湖异常毒雾；有人出天价买「图腾蜕皮时刻表」；灼原方向出现火焰圣灵传闻。审判会的人按住你的肩：今晚谁敢靠近褪皮的「大家伙」，国府与妖魔都会要他的命。  
开场白建议：「你以为统领级已经是噩梦，直到毒雾里那条如楼宇的黑影慢慢翻身。审判会的人按住你的肩：今晚谁敢靠近褪皮的『大家伙』，国府与妖魔都会要他的命——包括你。」  
关键NPC立场：**唐月**——守护优先；**图腾玄蛇**——可感不可辱；**莫凡**——已掺和进图腾因果；火院导师——抢炎姬线索；**张小侯**军方线——征召前的影子。  
主线钩子/支线：护蛇；寻炎；支线＝元素种拍卖、吸血鬼巢清剿、假图腾祭祀骗局。  
危险度/规避：高——规避超阶、君主级巅峰；勿盗玄蛇蜕皮。  
任务方向/奖励：魂种；契约兽火种级线索；国府观察员推荐；唐月一句「你还算守规矩」。

**四阶（高阶～超阶门槛 · 古都浩劫线）**  
切入身份/时点：以「国府临时征召法师 / 高阶猎者 / 死街敢死队」切入，锚定**古都亡灵围城**、煞渊将开。  
初始事件：外城失守电报；死街需要敢死队送药；有人悄声问你是否见过「红衣女人」的毒雾印记。城墙上的魔石炮打到发红，有人喊张小侯的名字，另一个更疯的声音说：去煞渊。  
开场白建议：「古都的夜没有星星，只有亡灵眼里的绿。城墙上的魔石炮打到发红，协会的人还在统计还能守几小时。你听见有人喊张小侯的名字，另一个更疯的声音说：去煞渊。」  
关键NPC立场：**莫凡**——为救友不要命；**撒朗**系——要完成献祭；国府指挥官——弃子与死守之间；**古老王**压力——场上气压本身；**苏小洛** / 死街难民——道德压测；**华展鸿**线眼线——事后论功。  
主线钩子/支线：守墙；过死街；破坏祭阵；支线＝运送地系法师、抢救精魄库、护送伤员出城。  
危险度/规避：极高——规避古老王本体硬刚、恶魔失控无准备接触；勿在煞渊口「捡漏」。  
任务方向/奖励：高阶魔具；浩劫功勋；恶魔系相关情报（危险）；元素魂种；一次「华军首记得你的名字」。

**五阶（超阶 · 国际 / 帕特农 / 学府大赛后）**  
切入身份/时点：以「超阶法师 / 国馆顾问 / 国际佣兵」切入，锚定大赛后或帕特农之乱、**艾江图**队国际声望期。  
初始事件：神庙发来「协作」实为审查的请柬；你的名字出现在某张针对中国队的暗杀名单旁边。白石台阶亮得像审判，有人微笑着问：复活一位故人，你愿意用什么系的未来交换？  
开场白建议：「超阶之后，国境线变成了谈判桌。帕特农的白石台阶反射阳光，亮得像审判。有人微笑着问你：复活一位故人，你愿意用什么系的未来交换？」  
关键NPC立场：**叶心夏**——神魂与神女政治核心；**莫凡**——国际名人；**艾江图**——编制与纪律；**穆宁雪**——凡雪招牌；神庙骑士——信仰刀；**伊之纱**阴影——血脉与仇恨；圣城观察员——记录威胁等级。  
主线钩子/支线：救心夏协作；神女派系；支线＝禁咒材料黑市、国际执照、针对国府队的暗手。  
危险度/规避：极高——规避禁咒互轰中心、黑暗死神级；勿骑墙神女派系。  
任务方向/奖励：天种线索；神庙临时权柄；超阶魔具；神印外围见闻。

**六阶（顶尖超阶 / 半禁咒 · 海洋之灾 / 图腾圣兽线）**  
切入身份/时点：以「准禁咒国家资产 / 沿海城防超阶 / 图腾祭祀护法」切入，锚定上海 / 沿海海洋之灾、图腾青龙等复苏窗口、凡雪山保卫。  
初始事件：海堤阵列缺口；君主级海妖同时三处登陆；图腾祭祀需要活人超阶级护法。国府密令只有一行：拖到图腾抬头——拖不过，就用你的星宫填海。  
开场白建议：「半禁咒的星流在天上流，像第二轮月亮。可海平面上那座移动的黑影帝国不看月亮，只看陆地。国府的密令只有一行：拖到图腾抬头——拖不过，就用你的星宫填海。」  
关键NPC立场：海妖君主 / 帝王代行——种族战争；**莫凡（城主 / 多系）**——前线搅局者；**赵满延**——龟壳与霸下；图腾圣兽——友但不听人令；**邵郑** / 亚洲协会——要协调不要背锅；**华展鸿**——军方编制。  
主线钩子/支线：守港；唤醒 / 护送图腾；支线＝凡雪山补给线、难民海桥、魔石车队护航。  
危险度/规避：灾难级——规避帝王本体、多禁咒对轰余波；勿奴役图腾。  
任务方向/奖励：禁咒触媒线索；图腾友谊信物；国家级领地 / 编制；半禁咒稳境资源。

**七阶（禁咒 · 暗位面 · 终局线）**  
切入身份/时点：以「禁咒法师 / 国家柱石 / 神庙并列强者」切入，锚定暗位面 / 昆仑 / 黑教廷覆灭后余波与邪圣王 / 法神级对决前夜。  
初始事件：黑暗位面裂口观测；圣城发来审判或结盟双头文件；有人用命换你一封给「莫凡」的密信——关于文泰与心夏、关于八魂与邪圣王。  
开场白建议：「到了这一层，魔石账变成了文明账。你站在位面裂口边缘，星宿在身后完整如殿，却仍觉得冷。有人告诉你：邪圣王已经可以废掉天使的魔法——那么你，是去见证，是去分一杯羹，还是去阻止下一次献祭换皮复活？」  
关键NPC立场：**莫凡（邪圣王 / 法神向）**——终局变量；**叶心夏（神女 / 屠教廷后）**——新秩序裁缝；**穆宁雪**——冰系禁咒级并肩；**穆白**——暗位面协作 / 对照；**文泰 / 黑暗残余**——幕后；**红魔一秋**——八魂棋手；米迦勒级秩序强敌——条件性胜利对象；帝王级海妖残余——仍在。  
主线钩子/支线：暗位面探险；护航法神决战外围；清理教廷残党；支线＝昆仑机缘、国际新条约、八魂线索截断。  
危险度/规避：顶点级——严禁无布局硬刚法神 / 邪圣王 / 黑暗王 / 帝王本体；胜利靠情报、神庙权柄、图腾、位面规则与结盟。  
任务方向/奖励：法神级见闻；位面材料；传说魔具；改变一条海岸线归属的权柄；条件性胜利份额（非战力外挂）。
