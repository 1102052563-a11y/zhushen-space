# 合金装备2
<!--meta lib=主库 tiers=一、二、三、四、五、六、七-->

## 剧情

**【作品来源】**
《合金装备2：自由之子》（Metal Gear Solid 2: Sons of Liberty，2001）由小岛秀夫导演、小島プロダクション开发、科乐美发行，初登场于 PlayStation 2，后有 Substance 扩展版及 HD / Master Collection 复刻。本作是《合金装备》（Metal Gear Solid，1998）正统续作，时间上接「Shadow Moses 事件」之后，前承《合金装备：和平行者》等前传线，后启《合金装备3：食蛇者》《合金装备4：爱国者之枪》中爱国者（The Patriots）与雷电命运。类型为潜行动作 + 政治惊悚 + 元叙事；文风冷硬军事、黑色幽默与哲学独白并重，核心母题为「模因 / 信息管制 / 被编程的英雄」。本档案以游戏正典（Tanker Chapter + Plant Chapter）为准，小说化设定与衍生仅作补充。卡片锚点：Big Shell 核心，雷电以高频周波刀对抗索利达斯·斯内克指挥的量产型 Metal Gear RAY 机甲洪流。

**【世界定位】**
2007–2009 年前后的近未来平行地球。表面是冷战后特种作战、石油污染治理与反恐舞台；底里是「爱国者」以 AI 与信息过滤重塑民主与战争经济的棋盘。视点主角分两段：**固体蛇（Solid Snake）** 在 2007 油轮事件中潜入美军运输 RAY 的假油轮；**雷电（Raiden / Jack）** 在 2009 Big Shell 事件中以 FOXHOUND 新人身份执行解救人质与排除「自由之子」恐怖组织任务。一句话：这是「被剧本写就的英雄如何夺回自我」的潜行世界，契约者是局外变数，不是爱国者剧本里的天选。

**【世界观 · 力量体系】**
力量本源不靠灵气法宝，而靠**训练、基因工程、义体改造、纳米机器、战术装备、情报权限与战略兵器**。个体战力高度依赖后勤、弹药、Codec 支援与「是否被系统注视」。死亡规则：常人与士兵一刀一枪可死；纳米机器可加速止血与镇痛，但无法无限复活；Vamp 一类「再生型」仍可被持续重创与环境杀死；战略级存在（Arsenal Gear / GW）靠系统冗余与舰体装甲，不是神体。特殊系统：① **Codec / 远程支援**——上校、萝丝、Otacon 等以无线电注入情报，后期可被 AI 冒充；② **潜行判定**——视野、声响、警戒阶段（Alert / Evasion / Caution）决定战局；③ **纳米机器**——战斗员体内植入，可被远程监控、注入指令甚至「扳机管制」；④ **Metal Gear 系战略兵器**——可搭载核武 / 反 Metal Gear 武装的步行或潜航平台；⑤ **爱国者信息权**——控制媒体叙事、选举、预算与「何为真相」。

战力阶梯（以破坏力与战场角色为准，名称自拟便于映射，非原作修炼境界）：
1. **凡俗人员**：平民、清洁工、媒体、低级船员。徒手与冷兵；热武器一发致命。≈一阶。
2. **正规军 / 海军队列**：美军陆战队、油轮护卫、Big Shell 安保。班排火力、固定机枪、装甲车边缘。≈一~二阶。
3. **特种部队与雇佣兵**：海豹突击队、Gurlukovich 雇佣兵、Philanthropy 行动员。小队歼灭、夜间渗透、反器材步枪。≈二阶。
4. **精英特种 / 具名行动者**：固体蛇（油轮期）、雷电（Big Shell 中期）、奥尔加·古尔科维奇（佣兵指挥态）。单兵清点、非致命压制、与 Boss 级周旋。≈三阶。
5. **强化战力 / 小队 Boss**：胖子（Fatman）爆破专家、Vamp 再生近战、福尔琼（Fortune）电磁轨道炮与「幸运力场」、Deepthroat / Mr.X（奥尔加·义体忍者态）。可清空走廊、击落直升机级威胁、硬抗常规火力。≈四阶。
6. **传说级士兵 / 克隆体巅峰**：固体蛇（巅峰传说）、索利达斯·斯内克（Solidus Snake / George Sears）、后期持高频刀的雷电、义体忍者峰值。单兵对机甲、舰桥决斗、可在都市战中决定局部胜负。≈五阶。
7. **战术兵器与精锐编制**：Metal Gear RAY（有人 / 无人量产型）、Arsenal Tengu 高频刀部队、Harrier 舰载机、Stinger 防空。毁舰、毁桥、局部街区级火力。≈六阶。
8. **战略平台 / 信息奇点**：Arsenal Gear 潜航要塞（搭载 GW AI、核武选项、量产 RAY 集群）、爱国者（The Patriots）全局信息过滤与 S3 计划。国家/文明尺度的战争与认知管制。≈七阶（本世界覆盖上限）。

乐园阶位映射：凡俗~新兵≈一阶；正规军精锐~雇佣兵≈二阶；具名特种/FOXHOUND 新人≈三阶；Dead Cell 成员与义体忍者≈四阶；固体蛇 / 索利达斯 / 高频刀雷电≈五阶；RAY / Tengu / 舰载机群≈六阶；Arsenal Gear + GW + 爱国者体系≈七阶。宁低勿高。世界顶点是**爱国者信息权 + Arsenal Gear 战略平台**，非无敌神仙；对个体的「超阶巅峰」写条件性胜利（病毒瘫痪、身份揭露、切断后勤、利用 Liquid 人格冲突），严禁「战力归零式削弱」。

**【地理 · 舞台】**
- **纽约港 / 哈德逊河口（2007）**：油轮事件舞台。假油轮 USS Discovery 运载美军陆战队 Metal Gear RAY 原型；乔治·华盛顿大桥一带为固体蛇入场点。氛围：夜雨、铁锈、货舱回声、陆战队与俄雇佣兵交火。
- **Big Shell（2009）**：曼哈顿外海约 30 公里的巨型海上「海洋污染净化设施」。表面由 Shell 1 / Shell 2 两个六边形结构体与连接桥组成，各有中央核心与六根支柱（发电、居住、直升机库/仓储、油处理、生化实验、排放处理等）；实则是 **Arsenal Gear** 建造与伪装外壳。总统詹姆斯·约翰逊视察日被「自由之子」占领。
- **Arsenal Gear 内部**：以人体消化道命名的分区（胃、升结肠等），部分区域刻意复刻 Shadow Moses 走廊与关卡逻辑，服务 S3 计划；内驻 Arsenal Tengu、量产 RAY、监控与核武相关设施。
- **曼哈顿下城 / 联邦厅（Federal Hall）**：Arsenal Gear 冲向纽约后的终局决斗地，乔治·华盛顿雕像旁；雷电与索利达斯决战。
- **幕后空间（非地理）**：爱国者的服务器与「贤人委员会」叙事、Codec 虚拟上校房间、萝丝的监视日志——比任何甲板更致命。

AI 写场景须先定契约者在「油轮甲板 / Shell 支柱 / 核心 / Arsenal 内部 / 曼哈顿终局」哪一层，再选冲突强度：底层是警戒与人质；中层是 Dead Cell 与雇佣兵；高层是 GW 与爱国者剧本。

**【世界剧情线】**
① **前史：Shadow Moses 余波与 Philanthropy（~2005–2007）**  
Shadow Moses 事件后，固体蛇与哈尔·「Otacon」·艾默里奇组建反 Metal Gear 非政府组织 Philanthropy，专事揭露与摧毁新型 Metal Gear。液体蛇之死、灰狐之死、雷克斯（REX）残骸与「爱国者」阴影仍在。雷电（杰克）此时尚未登场主舞台，正以虚拟现实训练与 FOXHOUND 编制被塑造成「第二个固体蛇」。世界状态：媒体仍把固体蛇当英雄或怪物摇摆；军工复合体加速 RAY 等反 Metal Gear 兵器。

② **油轮事件（Tanker Incident，2007-08-08）**  
固体蛇自乔治·华盛顿大桥绳降潜入假油轮，任务是拍摄并曝光美军陆战队的 Metal Gear RAY。同期，谢尔盖·古尔科维奇率领的俄雇佣兵登船夺 RAY，奥尔加·古尔科维奇随队。蛇与奥尔加交手（可非致命），拍下 RAY 后目睹**左轮手枪奥西洛特（Revolver Ocelot）** 背叛：奥西洛特杀死谢尔盖与陆战队司令斯科特·多尔夫，夺走 RAY。见到固体蛇时，奥西洛特移植自液体蛇的右臂引发人格闪现，「液体」嘲讽蛇的加速老化。奥西洛特驾 RAY 击沉油轮，制造巨大原油污染；媒体与爱国者叙事将罪魁指向固体蛇与 Philanthropy。蛇假死脱身。因果：① RAY 技术落入奥西洛特/爱国者相关手中；② 污染成为建造 Big Shell 的公开借口；③ 福尔琼等死者家属的复仇种子埋下；④ 蛇的公众形象崩塌，为 2009 化名「易洛魁·普利斯金」埋伏笔。

③ **Big Shell 占领与雷电入场（2009-04-29 起）**  
Big Shell 建成于油轮沉没点上方。美国总统詹姆斯·约翰逊视察时，自称「自由之子（Sons of Liberty）」的武装集团占领设施，扣押总统与大量人质，索要巨额赎金并威胁环境灾难。成员核心为叛变的海军反恐小队 **Dead Cell**（福尔琼、Vamp、胖子）与残余古尔科维奇雇佣兵，首领对外化名「固体蛇」，实为**索利达斯·斯内克**（前总统乔治·西尔斯，Big Boss 克隆体第三号）。FOXHOUND 新人**雷电**由「上校」与女友分析员**萝丝玛丽（Rosemary）** 远程支援，自水下进入 Shell 执行潜入。海豹十队协同登陆后几乎被 Vamp 屠杀殆尽。雷电遭遇化名易洛魁·普利斯金的固体蛇、爆破顾问彼得·斯蒂尔曼、以及神秘义体忍者 Deepthroat / Mr.X（后揭为奥尔加）。

④ **炸弹、Harrier 与「假蛇」**  
胖子在设施内布设真假炸弹，雷电与斯蒂尔曼联手拆除；胖子以自身为诱饵让斯蒂尔曼死于隐藏炸弹，随后雷电击败胖子。福尔琼以「子弹与炮弹皆偏转」的幸运姿态与轨道炮压制雷电；Vamp 以倒吊、刀术与吸血传闻制造恐惧。索利达斯驾驶 Harrier 出击，被蛇与雷电以刺针导弹击落，身份逐渐暴露为「不是固体蛇的蛇」。Otacon 远程接入，确认蛇仍存活并协助疏散。世界状态：人质线、炸弹线、Boss 线并行；Big Shell 结构开始进水与崩坏。

⑤ **总统之死与 Arsenal Gear 真相**  
雷电抵达总统所在，詹姆斯·约翰逊吐露核心政治惊悚：**美国民主是爱国者操控的表象**；Big Shell 是 Arsenal Gear 的外壳；Arsenal Gear 是搭载 AI「GW」的巨型潜航要塞，用于监控、过滤与管制全球数字信息，并可接入美军战术网与核武相关权限。索利达斯要夺取 Arsenal 以对抗爱国者、取得「贤人委员会」名单。奥西洛特当场枪杀总统。雷电被推上「必须继续前进」的轨道——他已不是单纯救人，而是在爱国者写好的剧本里跑完全程。

⑥ **艾玛·艾默里奇与病毒上传**  
Otacon 的继妹、AI 工程师**艾玛·艾默里奇（Emma Emmerich）** 被困 Shell 2。雷电穿越水下通道与 Vamp 再战，救出艾玛。艾玛向 GW 上传病毒试图瘫痪 Arsenal；撤离时被 Vamp 刺伤，重伤身亡。病毒仅部分生效——GW 受损却未彻底死亡，为后续「上校 AI 崩溃」与信息错乱埋线。蛇与 Otacon 协助疏散人质；奥尔加以忍者身份继续暗助。

⑦ **被捕、S3 计划与身份崩解**  
为让蛇与奥尔加潜入启动中的 Arsenal Gear，雷电被「背叛式」捕获，遭受拷问与纳米机器层面的羞辱。奥尔加（爱国者双重间谍，以女儿性命被胁迫）释放雷电并交付高频周波刀。Arsenal 启动，Big Shell 解体沉没。雷电在 Arsenal 内部发现：① 上校是受损 GW 生成的拟态；② 萝丝的部分通讯亦被系统操控，她本人曾是爱国者安插的监视者（同时怀有雷电的孩子）；③ 索利达斯曾在利比里亚内战中将儿童兵杰克训练成「开膛手杰克」；④ 整场 Big Shell 任务是 **S3 计划**——表面「Solid Snake Simulation」（复刻 Shadow Moses 条件以量产等同固体蛇的士兵），深层「Selection for Societal Sanity」（筛选与管制社会认知，过滤「有害模因」）。奥西洛特作为爱国者棋子再度翻转；液体人格借右臂夺回身体，驾 RAY 离去，并将 Arsenal 推向曼哈顿。

⑧ **量产 RAY 洪流与联邦厅决战（卡片锚点）**  
索利达斯指挥 **25 台量产型 Metal Gear RAY** 与 Tengu 部队清场。雷电持高频周波刀在 Arsenal 甲板上与机甲洪流死战——这是卡片锚点名场面：单兵高频刃对抗机甲潮，金属与火花、Codec 崩溃、自我意识在枪炮里重生。Arsenal Gear 撞击曼哈顿下城，停于联邦厅附近。爱国者以「上校 / 萝丝」拟态命令雷电杀死索利达斯，否则萝丝与奥尔加之女将遭毒手。雷电在固体蛇的劝诫下选择**自己的意志**，与索利达斯刀剑对决并重创之。蛇在索利达斯遗体附近留下追踪，与 Otacon 继续追查爱国者并营救奥尔加之女；雷电与萝丝重逢，开始以「杰克」而非脚本角色活下去。尾声暗示「贤人委员会」十二人早已死亡约百年——爱国者真正的控制核并非传说中的老人会，而是更冷酷的系统。

⑨ **微观可介入事件（日常与支线密度）**  
除大场面外，正文高频事件：① 走廊警戒与非致命制服；② 人质舱救人与谈判；③ 炸弹倒计时与假炸弹甄别；④ Codec 情报真伪辨别；⑤ 偷拍 / 窃取 RAY 或 Arsenal 资料；⑥ 雇佣兵与海豹残部的临时同盟；⑦ 媒体直播如何扭曲现场。AI 应优先这些「潜行与信息战」小事件，而不是一上来对轰 Arsenal。

⑩ **蝴蝶效应边界**  
契约者可改局部：多救几名人质、提前拆除某组炸弹、改变胖子或 Vamp 的交战结果、协助艾玛多活一段时间、让某段 Codec 录音外流。不可无代价改写：爱国者存在与 S3 计划的核心逻辑、Arsenal Gear / GW 的战略定位、固体蛇与雷电的人格底色、索利达斯「为成为独立国家之父而战」的执念。强行提前击杀索利达斯或摧毁 GW 完整核心，会引发爱国者更狠的清洗、媒体定性「生态恐袭」或 Liquid 线提前失控。

**【主要人物】**
- **雷电 / 杰克（Raiden / Jack）**｜身份：FOXHOUND 新人，Plant 章主角；利比里亚内战儿童兵「开膛手杰克」。｜性格：表面冷静礼貌、渴望被认可，实则被 VR 训练与纳米机器塑形，崩溃后偏执而锋利；对「自己是否真实」极度敏感。｜装备·能力：SOCOM 手枪、M4、冷却剂喷雾、弹道盾；后期**高频周波刀**可斩切量产 RAY 关节与装甲薄弱处；身体经强化与 VR 战术灌输，近战与潜行均衡。｜人物弧光：脚本里的「第二固体蛇」→发现自己是实验品 → 在曼哈顿夺回「杰克」之名。｜立场：对固体蛇从偶像到战友；对索利达斯是弑父式决裂；对萝丝爱恨交织；对契约者先当任务变量审查，真名与童年往事是雷区。

- **固体蛇（Solid Snake）**｜身份：传奇士兵，油轮章主角；Philanthropy 成员；Plant 章化名易洛魁·普利斯金（Iroquois Pliskin）。｜性格：厌战却不停战斗，毒舌、烟与口粮、外冷内热；拒绝被神话。｜装备·能力：潜行大师，刺针导弹、镇定枪、C4；体能因加速老化下滑但仍处传说级。｜人物弧光：被诬为恐袭者 → 暗中引导雷电 → 继续反爱国者。｜立场：对雷电是严苛导师；对 Otacon 是共生搭档；对奥西洛特宿敌；对契约者「有用则并肩，暴露则警告」。

- **索利达斯·斯内克 / 乔治·西尔斯（Solidus Snake / George Sears）**｜身份：自由之子首领；前美国总统；Big Boss 完美克隆取向；雷电的「养父 / 教官」。｜性格：卡里斯玛暴君，渴望留下「建国者」之名，蔑视被爱国者操纵的民主。｜装备·能力：双高频刀、外骨骼强化、Harrier 驾驶、指挥量产 RAY 与雇佣兵。｜人物弧光：总统位被爱国者罢黜 → 夺 Arsenal 反叛 → 联邦厅败死。｜立场：视雷电为作品与叛徒；视爱国者为必毁之神；契约者若妨碍其「建国」则杀无赦。

- **左轮手枪奥西洛特（Revolver Ocelot）**｜身份：多重间谍；爱国者棋子；右臂移植自液体蛇。｜性格：戏谑、背叛成性、享受局中局；液体人格会突然夺舍。｜装备·能力：左轮枪法、拷问、政治操盘；可驾驶 / 调度 RAY。｜人物弧光：油轮夺 RAY → Big Shell 再叛 → 液体夺体离去。｜立场：无人可完全信任；与契约者交易必留后手。

**【势力图谱】**
- **爱国者（The Patriots / La-li-lu-le-lo）**：幕后信息帝国。控制选举、军工、媒体叙事与 S3 计划；GW 是其数字时代执行核。代表：系统本身、奥西洛特（棋子）、虚拟上校。战力层级：七阶信息/战略。对一切「不可控模因」清洗。
- **Philanthropy**：蛇 + Otacon 的反 Metal Gear NGO。地盘：流动安全屋、偷来的资料与舆论战。战力：五阶个体 + 骇入。与爱国者敌对，与官方若即若离。
- **自由之子（Sons of Liberty）**：索利达斯旗帜下的恐怖/革命联合体。成员：Dead Cell + 古尔科维奇残部。目标：夺 Arsenal、摆脱爱国者、建立「外天堂」式独立叙事。战力：四~六阶混合。
- **Dead Cell**：原海军反恐小队，因丑闻与私刑叛变。代表：福尔琼、Vamp、胖子。战力：四阶 Boss 集群。
- **Gurlukovich 雇佣兵**：俄系职业佣兵。代表：谢尔盖、奥尔加。战力：二~三阶部队，首领更高。
- **美军 / 海豹 / 陆战队**：国家暴力合法面。油轮陆战队、海豹十队、总统安保。战力：一~三阶；在 Big Shell 被降维屠杀以衬托 Dead Cell。
- **FOXHOUND（重建编制）**：雷电所属的官方特种符号，实则被爱国者当实验容器。
- **Arsenal Gear 守备（Tengu + 量产 RAY）**：系统直属武装。战力：五~六阶集群。

敌友逻辑：情报 > 子弹；今日同盟明日枪口；暴露「你知道 S3」比暴露枪法更招杀身。

**【贵重物品】**
- **Metal Gear RAY（有人原型 / 无人量产）**：反 Metal Gear 两栖机甲，导弹、水电喷射、厚重装甲；量产型由 AI / 遥控集群作战。油轮章被奥西洛特夺走原型；Plant 章 25 台量产围攻。谁控制 RAY 谁拥有局部六阶火力。
- **Arsenal Gear**：潜航信息战争要塞，内置 GW、核选项、RAY 机库。Big Shell 是其蛹。低阶不可「抢走开回家」，只能破坏子系统或逃离。
- **GW（AI）**：爱国者信息过滤中枢之一。艾玛病毒可部分瘫痪；完全控制=七阶权柄。
- **高频周波刀（High-Frequency Blade）**：刃身高频振动，强化切割；雷电于 Arsenal 自奥尔加处获得，联邦厅再战索利达斯。可斩机甲薄弱结构，是五阶个体对抗六阶兵器的关键杠杆。
- **福尔琼的轨道炮与力场装置**：城市级直射威胁 + 偏转弹道；装置与爱国者监控相关，非无敌神迹。
- **刺针导弹（Stinger）**：便携防空，击落 Harrier 的关键道具。
- **纳米机器与 Codec 套件**：战斗员标配；可被远程读写——既是增益也是项圈。
- **艾玛的 GW 病毒**：一次性战略软件武器；上传成败改写终局难度。
- **RAY 设计数据 / 油轮照片**：Philanthropy 的舆论炮弹；2007 任务目标物。
- **贤人委员会名单**：索利达斯的执念目标；真相是更大的虚无与系统谎言。

每件写清归属与后果：抢 RAY 数据=爱国者与军工双重追杀；持有高频刀=被当成 Arsenal 逃兵或忍者余党；碰 GW 核心=信息战升级。

**【隐藏剧情 · 伏笔】**
- **S3 双关**：Solid Snake Simulation 与 Selection for Societal Sanity 同构——英雄叙事本身是社会控制工具。
- **模因与「被编程的自我」**：雷电的记忆、恋爱、任务台词都可能被写入；自由意志是战斗后的战利品而非前提。
- **液体之臂**：奥西洛特右臂是液体人格的后门，贯穿至后续作品。
- **奥尔加之女**：被爱国者扣押，成为奥尔加叛变与牺牲的真正引擎；蛇承诺营救。
- **萝丝怀孕**：个人线与监视线交缠，终局「要不要相信她」是雷电成人礼。
- **贤人委员会已死**：爱国者不是十二老人密室，而是自我演化的控制系统——为 MGS4 爱国者 AI 线铺路。
- **Shadow Moses 复刻**：Arsenal 内部关卡、Boss 配置、人质结构刻意镜像一代，服务实验对照。
- **跨作互文**：Big Boss 克隆计划（Les Enfants Terribles）、灰狐忍者谱系、REX→RAY→Arsenal 兵器进化，点到为止，勿整本倒灌 MGS3/4。

**【叙事基调 · 雷区】**
基调：冷雨金属、Codec 杂音、黑色幽默、突然插入的哲学长谈；潜行比无脑枪战更「像这个世界」。忌把雷电写成无脑热血勇者或纯粹弱鸡——他是被伤害的利刃。忌固体蛇全知全能现身挂。忌低阶就打爆 Arsenal 本体。忌把爱国者写成会逛街的超级反派脸谱，它是系统。忌用修仙词替换纳米与模因。忌「被封印/战力归零」处理索利达斯或 GW。最早切入锚点：2007 油轮事件前夜，或 2009 总统视察抵达 Big Shell 之前两小时。

## 阶位切入点

阶位↔原作战力：一阶≈凡俗/新兵，二阶≈正规精锐与雇佣兵，三阶≈具名特种（雷电初期/蛇的协助位），四阶≈Dead Cell 与义体忍者，五阶≈固体蛇/索利达斯/高频刀雷电，六阶≈RAY 与 Tengu 集群，七阶≈Arsenal Gear·GW·爱国者体系。世界顶点（爱国者 + Arsenal/GW）为信息与战略层存在，低阶切入一律「规避顶点·情报优先」；对索利达斯等个体顶点写**超阶巅峰：条件性胜利**，禁止写成对方战力凭空消失。

**一阶（凡俗船员·清洁工·媒体跟拍 · 井底求生）**

切入身份/时点：以 Big Shell 外包清洁工、油轮厨房帮工或随总统船队的媒体助理切入；时点锚定 2009 总统视察抵达前 2 小时，或 2007 油轮离港补给阶段。

初始事件：警报误鸣，你被塞进错误的密封走廊；对讲机里有人用俄语与英语对骂，一具陆战队头盔滚到脚边——你必须在 90 秒内选择躲进储物柜、跟着人群撤离，或捡起无弹匣的手枪。

开场白建议：「潮水拍打双壳外壁的声音像有人在敲你的肋骨。工牌上的名字是假的，对讲机里的『一切正常』也是假的。头顶管道滴下的水带着柴油味——你还不知道，真正的漏油在更深处，是一整个国家的秘密。」

关键NPC立场：**彼得·斯蒂尔曼**会把你当需要保护的平民；**海豹残兵**可能征用你的门禁卡；**古尔科维奇巡逻兵**视你为麻烦；**萝丝玛丽**（若你误连公共频道）只给你标准安抚话术。

主线钩子/支线：钩子——活着见到「总统还在 Shell 1」。支线：① 为被锁住的参观团打开消防通道；② 偷拍一张 RAY 涂装的模糊照片卖给地下记者；③ 给斯蒂尔曼递工具箱换一条安全路线。

危险度/规避：危险度低~中。规避 **Vamp** 巡逻廊、任何标注「Strut F」深区、以及对空广播里的「固体蛇」号召点。

任务方向/奖励：撤离、开门、传话。奖励：门禁级、急救喷雾、低级情报芯片、乐园币结算向「存活见证」。

**二阶（海豹边缘·雇佣兵新兵·Philanthropy 跑腿）**

切入身份/时点：海豹十队外围通讯兵、古尔科维奇新招募、或 Otacon 在岸上雇佣的数据跑腿；时点为海豹登陆 Big Shell 同时段，或油轮事件中雇佣兵登船后 30 分钟。

初始事件：你的小队在连接桥上遭伏击，队友喉咙被切开却不见凶手——对讲里只留下湿漉漉的脚步声。你要决定是呼叫「普利斯金」、撤退报点，还是追血脚印。

开场白建议：「消音器打在金属壁上的声音比电影里干。有人叫你别看尸体的眼睛，你还是看了——那双眼睛提醒你：这里的恐怖分子不是为了上新闻，是为了改写新闻本身。」

关键NPC立场：**易洛魁·普利斯金（固体蛇）**会评估你是否拖后腿；**奥尔加·古尔科维奇**（佣兵态）可能收编你；**胖子**把你当爆破观众；**Otacon** 可远程给你一张结构图换你拍摄的角度。

主线钩子/支线：钩子——协助拆除 Shell 1 一组真炸弹。支线：① 护送两名人质到直升机甲板；② 窃取雇佣兵的 RAY 运输日志；③ 给 Philanthropy 上传 30 秒无压缩视频。

危险度/规避：危险度中。规避与 **福尔琼** 正面对射、油库区无保护爆破、以及被奥西洛特点名问话。

任务方向/奖励：小队战术、拆弹辅助、情报盗拍。奖励：镇定枪、半自动步枪、结构 U 盘、与 Philanthropy 的一次呼叫权。

**三阶（FOXHOUND 协作者·蛇的临时搭档）**

切入身份/时点：与雷电同期投放的「第二潜入者」或蛇指定的岸上接应潜入 Big Shell；时点锚定雷电刚会晤普利斯金之后。

初始事件：Codec 突然串线，一个自称上校的声音给你与雷电完全矛盾的指令——你必须在 10 分钟内判断哪条是真任务：救总统，还是优先摧毁某根支柱的服务器。

开场白建议：「你的血型、步长、甚至怕黑的习惯，档案里都有。雷电在另一条通风管里朝同一扇门前进。你们像两枚被同一只手投出的骰子——差别只是，你还不知道庄家是谁。」

关键NPC立场：**雷电**视你为竞争者或对照组；**固体蛇**只认结果；**艾玛·艾默里奇**若已救出可给你 GW 接口口令半段；**詹姆斯·约翰逊**只对能带他走的人开口。

主线钩子/支线：钩子——抢在奥西洛特之前听到总统关于爱国者的证词。支线：① 保护艾玛通过水下通道；② 校准刺针导弹锁定 Harrier；③ 复制一份 Codec 异常日志。

危险度/规避：危险度中高。规避与 **索利达斯** 单挑、在 Vamp 满状态时近身、以及把爱国者名字说进公开频道。

任务方向/奖励：潜行协作、Boss 战辅助、证词保全。奖励：高规格消音武器、冷却剂、半套 Arsenal 结构图。

**四阶（Dead Cell 对抗位·忍者周旋）**

切入身份/时点：被两边同时通缉的独立承包人，或奥尔加暗中雇佣的「第二忍者」；时点为胖子死后、福尔琼轨道炮封锁主桥阶段。

初始事件：你在 Shell 2 走廊与 **Vamp** 照面，他邀请你「先杀一个海豹再谈生意」；拒绝则刀战，接受则道德与后续奥尔加线崩坏。

开场白建议：「血在天花板上逆着重力爬。Vamp 笑着问你怕不怕死——你怕的不是死，是死了还会被他们做成宣传片里的背景板。」

关键NPC立场：**Vamp** 以猎手目光看你；**福尔琼** 在你不提「固体蛇」时可停火谈条件；**奥尔加（Mr.X）** 可借你高频刃技巧一招；**胖子**（若尚存）只跟你比爆破美学。

主线钩子/支线：钩子——破坏福尔琼轨道炮的外部电容。支线：① 从 Vamp 手中抢下艾玛；② 伪装 Dead Cell 通讯骗开一扇 Arsenal 预备闸门；③ 录下力场装置的电子噪声供 Otacon 分析。

危险度/规避：危险度高。规避同时激怒福尔琼与 Vamp；**超阶巅峰**尚远，但力场+轨道炮可瞬间收掉四阶以下整队。

任务方向/奖励：Boss 机制战、破力场、救人。奖励：实验性护甲片、轨道炮残骸样本、忍者光学迷彩耗材（限次）。

**五阶（传说士兵同台·高频刀时代）**

切入身份/时点：与固体蛇齐名的退役传说、或雷电获刀后的并肩剑士；时点锚定 Arsenal 启动、Big Shell 沉没前后。

初始事件：奥尔加把第二把校准过的高频刃塞进你手里，条件只有一个——若她死了，带话给固体蛇去救她女儿。甲板外已是 RAY 的光学镀层反光。

开场白建议：「刀鸣比汽笛更响。雷电的眼睛里第一次有了『我』而不是『任务进度』。索利达斯在广播里称你们为孩子——你们回敬他的只有斩开导弹尾焰的轨迹。」

关键NPC立场：**雷电**认你为同类；**索利达斯·斯内克**视你为值得亲自斩首的障碍；**固体蛇**给你烟和一句「别死」；**奥西洛特**尝试收买你叛变。

主线钩子/支线：钩子——与雷电共同削减量产 RAY 数量，为联邦厅决战创造窗口。支线：① 保护奥尔加撤离通道；② 阻止液体人格完整夺体前的一次右臂失控；③ 向公众暗网投送 S3 片段（真假难辨，爱国者会反制）。

危险度/规避：危险度极高。正面对抗 **索利达斯** 须条件战（环境、双人夹击、先破外骨骼）；禁止假设爱国者「不会管你」。

任务方向/奖励：机甲斩击、双蛇博弈、身份选择。奖励：完整高频刀运用数据、RAY 弱点图谱、与 Philanthropy 长期呼叫协议。

**六阶（RAY 集群·Tengu 战场指挥官）**

切入身份/时点：Arsenal 内部叛变的 Tengu 小队指挥官、或劫持单架 RAY 的驾驶员；时点为 25 台量产 RAY 出击阶段。

初始事件：你的 RAY 神经同步中混入 GW 残缺指令，系统要你炮击曼哈顿民用码头——你只有 60 秒选择断链、假射击或真成为屠城者。

开场白建议：「座舱里 HUD 把人群标成灰色色块。有个声音用上校的嗓音说『为了社会健全』。你扳下保险时想：健全的社会需不需要你这种人。」

关键NPC立场：**量产 RAY 集群**无感情地执行清场；**Arsenal Tengu** 视断链者为叛徒；**雷电**可能把你当临时火力；**GW 残响**不断尝试重写你。

主线钩子/支线：钩子——带至少一架 RAY 转向拦截飞向联邦厅的弹道/掩护雷电。支线：① 物理拔除三台同伴机的遥控天线；② 向 Otacon 发送 GW 内核坐标；③ 打开 Arsenal 腹部逃生闸放人。

危险度/规避：危险度极高。规避爱国者完整接管信号、核保险相关面板、以及索利达斯的直接指挥频道。

任务方向/奖励：机甲战、系统断链、城市级火力克制。奖励：RAY 驾驶残章、反制纳米干扰模块、战略级情报（仍非爱国者根权限）。

**七阶（信息奇点边缘·爱国者棋盘）**

切入身份/时点：能接触 GW 物理接口的最高级工程师、爱国者末端执行官叛逃者、或手握艾玛病毒完整拷贝的人；时点为 Arsenal 撞击曼哈顿前后 1 小时。

初始事件：你手中的终端同时弹出两道命令：一道以萝丝的声音求你「杀掉索利达斯以救孩子」，一道以蛇的编码求你「别让任何 AI 替人做选择」。你的下一键将改写直播全球的叙事。

开场白建议：「曼哈顿的灯在 Arsenal 阴影里像被吞掉的星星。GW 用所有你爱过的人的声音说话。你终于明白：这世界的顶点不是最强的士兵，是决定『什么算真』的那只手——而你正把手指按在它的脉搏上。」

关键NPC立场：**爱国者（系统）**要收编或抹杀你；**固体蛇 / Otacon** 要你把选择权还给人类；**雷电**需要你别替他按下杀意；**液体·奥西洛特** 会趁乱偷走任何可移动的战略资产。

主线钩子/支线：钩子——在病毒与人工选择之间完成一次「有限瘫痪 GW」而非毁灭纽约。支线：① 伪造贤人委员会数据干扰索利达斯；② 救出奥尔加之女的坐标链启动；③ 向未来的自己留下不被过滤的日志（模因种子）。

危险度/规避：危险度贴近顶点。**超阶巅峰：爱国者 + 完整 Arsenal/GW 体系——情报优先 / 条件性胜利**（病毒、断链、舆论反噬、多重背叛利用），严禁硬刚「系统本体战力归零」。避免核选项面板与无备份死机全城。

任务方向/奖励：信息战终局、叙事权、有限系统权限。奖励：GW 残核样本、反过滤通讯协议、世界结算向「认知解放」高评价；不发「一键删掉爱国者」的外挂。
