# 泰拉瑞亚：Otherworld（企划/衍生）
<!--meta lib=主库 tiers=八-->

## 剧情

**【作品来源】**
《泰拉瑞亚：Otherworld》（Terraria: Otherworld）是 Re-Logic 宣布的《泰拉瑞亚》衍生作/精神续作向企划，定位为开放世界沙盒 RPG + 策略塔防混合，探索「如果泰拉瑞亚走向另一条路会怎样」。2015 年前后公开预告与 GDC 演示，Engine Software 初期开发，2017 年 4 月改由 Pipeworks Studios 接手，2018 年 4 月 **正式取消**。未发售完整游戏；设定、预告片、概念图、预览截图与未发布原声构成主要公开资料。原声后来部分进入《泰拉瑞亚》1.4 作为异界曲目（需派对女孩与醉酒种子等条件）。本档案写**企划世界**本身：以公开设定与预览为准，禁止编造已取消的完整商业剧情结局为「已发生正史」。文风：末世净化战争、据点塔防、腐化全面化。

**【世界定位】**
泰拉宇宙**平行次元**：腐化几乎吞噬一切，旅人与「一群衣衫褴褛的幸存者」试图用**力量之塔（Towers of Power / Purifying Tower）**与**战争塔（Wartower）**夺回世界。不是主作「打 Boss 升级」的同一条进度轴，而是**净化与防守**的战略史诗。一句话：在另一个泰拉，腐化先赢了。卡片锚点：**力量之塔防御战**与**世界核心（The Core）**。

**【世界观 · 力量体系】**
力量本源：方块改造、武器技能等级、塔防建筑、净化能量。腐化拥有**每一块方块的腐化形态**（如腐化土 Ebondirt、腐化雪 Ebonsnow），蔓延是国土法则。玩家侧靠力量之塔净化区域，靠战争塔防守净化节点。武器与技能可升级、带升级槽；合成系统有标签与收藏配方。摆荡抓钩等机动强化探索。

死亡与世界：公开资料未定永久死亡细则；以预览为准，失败=据点失守、净化圈被压回。顶点威胁是**全面腐化生态 + 新 Boss 群（至少 Crimother、Octoeye 等已公开名，另有未知）**，规模对标主作天界/月球级战役，乐园映射为**八阶**战略战场（地区/世界改造级），宁低勿高。

阶梯（破坏力表现）：
1. 幸存者营地与铜铁起步 ≈ 主作一阶质感，但环境敌对度更高。
2. 建立第一座力量之塔与战争塔 ≈ 战略二~三阶。
3. 多塔联防、净化走廊 ≈ 四~五阶战役。
4. 新 Boss（Crimother、Octoeye 等）与腐化潮 ≈ 六阶。
5. 核心生态、天空/水晶/核心群系总攻与世界级净化决战 ≈ **八阶**。

乐园阶位映射：本世界**仅覆盖八阶切入**（清单 max 八）；低阶生存细节写入剧情供正文，不单列一~七阶切入点。顶点=腐化意志/世界级 Boss 集群，条件性胜利=守住塔网并完成净化链，严禁写成「塔一开腐化归零」。

**【地理 · 舞台】**
预览可见：森林、腐化、猩红、沙漠、极地（Arctic）、太空、海洋等回归群系；新增**天空（Sky）、水晶（Crystal）、核心（The Core）**。核心似熔岩/行星内核美学。力量之塔周围形成**防护泡**，泡内可建要塞；泡外腐化敌潮冲击。地下亦可建塔防守。超大世界选项暗示战略纵深。

**【世界剧情线】**
① **前提：另一个结局**  
Re-Logic 描述：平行维度中腐化已主导。玩家不是从「友好向导教程夜」开始，而是从**末世残存**开始。向导、护士、机械师、巫医等熟悉 NPC 可能回归，但带有更明确的角色故事与任务线（企划卖点：NPC 剧情与成长）。

② **力量之塔战争**  
核心玩法循环：选址建力量之塔以净化土地 → 建战争塔与工事抵御腐化反击 → 扩展净化圈 → 获取新资源与 Boss 战。腐化不再只是「可挖沟隔离的污染带」，而是**全方块可腐化的活体法则**。防守失败会吞回已净化区，形成拉锯。

③ **新威胁与新生态**  
公开敌人概念：Triffid、Scavenger、Super Shark、Crystal Archer、Spawnpod、Crimson Eater、Corruption Poisonbag、Floating Sword Guy 等。Boss 公开名含 **Crimother、Octoeye**，另有未公开。群系扩展到水晶与核心，暗示终局深入行星结构而非仅打月亮。

④ **幸存者政治**  
「衣衫褴褛的幸存者团」暗示据点内有派系、任务板、资源配给。蜗牛 NPC 等新角色出现在预览。任务系统与武器技能升级把 RPG 权重抬高。塔防与沙盒建造必须同时服务「别被腐化推回出生点」。

⑤ **取消与遗产**  
2018 年 4 月 Re-Logic 宣布：游戏状态距愿景过远且进度严重滞后，终止开发。2020 年后社区请愿开源，Redigit 澄清非请愿驱动。1.4 将 Otherworld 原声以异界曲目形式部分活用于主作——**音乐成为平行世界的回声**。乐园侧可将 Otherworld 作为「可切入的未完成平行泰拉」：规则完整度以公开企划为准，空白处写「不详」而非脑补完整三幕剧。

⑥ **与主作《泰拉瑞亚》关系**  
共享方块、部分武器（迷你鲨等重绘）、腐化主题，但目标从「杀月球领主」转为「塔网净化世界」。主作困难模式双蔓延在此被推到极端：腐化先手胜利。契约者不可把主作进度装备直接等价搬入，除非叙事标明跨维携带代价。

⑦ **可玩冲突链（据预览重构）**  
建立第一泡 → 守夜尸潮/腐化专属敌 → 扩展到沙漠与极地 → 地下塔防 → 海洋净化镜头（预览有净海） → 水晶与核心远征 → 对决 Crimother/Octoeye 级 → 未完成的终局（公开资料止于企划中断）。

**【主要人物】**
**旅人/契约者**（玩家角色）｜末世净化者｜塔与武器双修｜从幸存者到世界工程师。

**向导（Guide）**｜若回归：配方与世界观讲解，语气可能更沉重。

**护士（Nurse）**｜据点医疗｜塔战后勤。

**机械师（Mechanic）**｜工事与线路｜战争塔维护核心。

**巫医（Witch Doctor）**｜丛林/召唤侧知识｜净化仪式顾问向。

**蜗牛 NPC（Snail）**｜预览新镇民｜任务与幽默点｜立场：据点吉祥物式存在。

**幸存者团首领（公开资料未定真名）**｜分配任务与资源｜可结盟可冲突｜不详处保持不详。

**Crimother / Octoeye**（公开 Boss 名）｜腐化/猩红系世界级灾厄｜塔网压力源。

**腐化意志（概念）**｜非具名对话体，环境本身｜全方块腐化规则的人格化威胁。

**【势力图谱】**
幸存者据点联盟；力量之塔网络（基础设施即政权）；腐化生态军团；可能回归的主作式入侵事件变体；企业/旧文明遗迹（核心与水晶暗示）。权力来自「你守住多少净化泡」而非徽章。

**【贵重物品】**
力量之塔与战争塔蓝图/升级件；硅剑（Silicon Sword）等新武；锌砧（Zinc Anvil）；武器升级槽材料；防护泡发生装置；净化核心；Crimother/Octoeye 掉落（企划向）；Otherworld 原声作为跨媒介遗物（主作可听）。

**【隐藏剧情 · 伏笔】**
「what might have been」=官方主题句；取消本身是元叙事：平行世界停在草稿；主作异界音乐=次元裂缝泄漏；开源未成=世界源码仍锁。写终局时用**条件性未完成**：契约者可推动局部净化胜利，但「全图净化」在公开资料中无验证结局。

**【大事记时间线】**
企划宣布 → GDC/预告展示塔防与群系 → Engine 开发 → Pipeworks 接手评估 → 2018 取消 → 原声流入泰拉 1.4 → 社区请愿与未开源。

**【叙事基调 · 雷区】**
基调：末世战略悲壮、据点温情、腐化压迫感；像素仍在但更暗。忌：写成已发售完整三幕并捏造官方结局；与主作进度混档；把八阶塔战写成休闲装修；用空泛跨作品套话凑字；忽略取消事实。

**【企划世界密度补块1】**
Otherworld 的公开演示强调：腐化不是背景色，是可吞噬每一种方块的法则。力量之塔创造可居住的泡，战争塔把沙盒建成前线。武器升级与技能等级让「同一把铜剑」在守塔千次后长成异物。核心群系把终局从天空拉向地心，与主作「打月亮」形成镜像。取消令世界停在草稿，乐园切入应尊重「未完成」：可推进局部战役，不可声称官方已有完美结局动画。预览中的净海、地下塔防、超大世界，都是八阶战场的地理零件。幸存者团的任务板把 RPG 日常塞进末世。与主作共享的护士与向导若出现，应带着「我们记得另一个世界」的互文，而非无缝衔接。Crimother 与 Octoeye 之名是少数钉死的 Boss 真名，其余未知 Boss 写不详。原声泄漏进 1.4 是最温柔的彩蛋：你在主作醉酒种子里听见平行世界的挽歌。

> 阶位↔：八阶≈力量之塔网络战争 / 核心与世界级腐化 Boss 战带。世界顶点为腐化全面化与 Crimother/Octoeye 级集群，条件性胜利=守塔+净化链，严禁「一键净化世界」。

**八阶（塔网决战 · 世界净化窗口）**

切入身份/时点：契约者以「异界工兵统帅 / 幸存者军事顾问 / 跨维泰拉旅人」切入，锚定**多座力量之塔已联通、核心群系或 Crimother/Octoeye 级威胁逼近**的战役夜。

初始事件：前线战争塔被腐化潮冲开一道口，防护泡闪烁；蜗牛 NPC 尖叫着报告东侧净化圈回退三成；机械师要你在黎明前决定——放弃外围塔收缩核心，或押上全部锌锭与升级槽材料打反击。

开场白建议：「泡外的世界是错色的。力量之塔的嗡鸣像倒计时，战争塔的炮口还在过热。有人把主作里的迷你鲨塞进你手里——枪还是那把枪，但这里的腐化会吃枪也会吃土地。」

关键NPC立场：**向导**——给配方却警告「这里没有月球领主这条捷径」；**机械师**——要你优先修塔不修房；**护士**——据点床位已满，伤员=减员；**巫医**——主张净化仪式与塔同步；**蜗牛 NPC**——任务板与士气；**幸存者首领（不详真名则保持）**——要政治承诺：净化后土地归谁；**Crimother/Octoeye**——环境级压力，非可闲聊对象。

主线钩子/支线：主线＝守住塔网并推进净化至核心/击败公开 Boss；支线① 回收被吞据点的幸存者；支线② 升级武器技能槽以抗腐化精英；支线③ 在水晶群系建立前进基地。蝴蝶效应：弃塔收缩可保核心但永久丢失区域剧情 NPC。

危险度/规避：贴近顶点。必须规避：无塔裸奔核心、让全部防护泡同时掉线。顶点＝**八阶巅峰：存在·情报优先/条件性胜利**（塔网完整度、净化百分比、Boss 阶段），腐化不会因剧情需要而突然失去威胁。
