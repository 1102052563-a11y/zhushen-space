# 假面骑士真

## 剧情

**【作品来源】**
《真・假面骑士 序章》（日文：真・仮面ライダー 序章(プロローグ)，英题 Shin Kamen Rider: Prologue／True Masked Rider: Prologue）为东映／东映视频 1992 年 2 月 20 日发售的 V-Cinema 特摄，纪念假面骑士诞生 20 周年。原作石之森章太郎，监督辻理，脚本宫下隼一・小野寺丈，音乐宇崎龙童等。定位成人向生物恐怖＋改造人悲剧，与 1 号～RX 正史平行。主角**风祭真（かざまつり しん）**被「财团」日本支部 **ISS（Institute of Super Science）** 暗改造成蝗虫基因改造兵士 Level 3，觉醒后称**假面骑士真／MASKED RIDER SHIN**。关键人物：护士兼监视员**明日香爱（あすか あい）**、父**风祭大门（かざまつり だいもん）**、ISS 所长**冰室严（ひむろ いわお）**、研究者**鬼塚义一（おにづか ぎいち）**、改造兵士 Level 2 **豪岛（ごうじま）**、CIA 工作员**赛拉·深町（セーラ・ふかまち）**、友人**结城卓也（ゆうき たくや）**。 主题歌《Forever》（渡边典子）。后在《假面骑士帝骑》剧场版等作品中作为历代骑士一员客串。本任务世界覆盖乐园阶位**一、二**，锚定影片主线：ISS 人体实验、鬼塚暴走杀人、真觉醒、爱之死与财团清算，不扩展到未制作的「本篇续作」。

**【世界定位】**
1990 年代初日本都市：表面上是医学研究与赛车梦想，暗面是跨国「财团」把人类改造成可商品化的生物兵器。一句话：变身不是喊口令，是愤怒把基因点燃——你可能救了人，也可能被全世界当怪物猎杀。

**【世界观 · 力量体系】**
基本规则：ISS 以治疗癌症／艾滋病为名推进基因改造，真实产品是**改造兵士（Cyborg Soldier）**等级制。蝗虫基因带来再生、跳跃、高周波爪、呼应反应（テレパシー／同族意识共享）。死亡：凡人一死即终；改造兵士可再生，但高温火焰（约 700–800℃ 持续）可致命——鬼塚即被火箭弹烧死。真的「第三之眼」使他变身后仍保自我，区别于失控的鬼塚。

等级与战力（按片中破坏力，宁低勿高）：
1. **普通人／研究员／特工**：枪械与谋略。≈一阶下限。
2. **改造兵士 Level 2（豪岛）**：机械臂斩击、非完整再生；可屠普通部队。≈**一阶～二阶**。
3. **改造兵士 Level 3 失控体（鬼塚）**：完整蝗虫化、呼应反应、无理性屠杀。≈**二阶**。
4. **改造兵士 Level 3 稳定体（假面骑士真）**：再生约 5000 倍、拳 2–2.8 t、踢 3–4.2 t、跳百米级、高周波爪与脊状刃；可毁直升机／建筑局部。≈**二阶顶**。
5. **财团武装／CIA 特遣**：组织火力，非个人超人，但可围剿。≈一～二阶组织压。

乐园阶位映射：普通人～Level2≈**一阶**；Level3 失控／稳定（真）≈**二阶**；财团全球网与「商品化人类」体制为叙事顶点压力，**非超阶个人战力**，但低阶切入须规避被机构定点清除。本世界仅覆盖一、二阶。

**【地理 · 舞台】**
- **ISS 生化学研究所（日本）**：实验、监禁、爆炸终局地。
- **都内街道与连续杀人现场**：鬼塚夜袭舞台。
- **体育俱乐部／城南大学关联空间**：真与爱、结城的日常锚。
- **高速公路／护送路线**：CIA 袭卡车、鬼塚之死。
- **纽约财团本部（提及）**：更高决策层，本阶只闻其名。
- **直升机与坠落点**：大门之死、ISS 崩坏。

**【世界剧情线】**
① **表面日常与噩梦**  
风祭真，25 岁，城南大学体育学部出身，500cc 路赛日本预选初出场初优胜，IQ250，为助父大门的免疫工程研究自愿成为 ISS 受试者。他夜夜梦见自己化为蝗虫怪物杀人，白天与护士明日香爱相恋，友人结城卓也鼓励其感情。世界状态：一阶叙事，制度比拳头可怕。

② **财团与鬼塚**  
大门不知研究已被财团军事化。同僚鬼塚义一暗中把蝗虫基因植入真，并自我实验。鬼塚失控化为 Level3，夜袭都市；其杀戮记忆经呼应反应灌入真，使真误以为自己是凶手。

③ **CIA 介入**  
赛拉·深町率队调查财团，袭击护送鬼塚的卡车。鬼塚被火箭弹焚死。真在场受呼应冲击，首次完整异形化——但第三之眼令他保持理性，并与突然出现的 Level2 豪岛交战。赛拉目睹后暂不处决真。

④ **爱的背叛与真心**  
明日香爱实为财团监视员，却真正爱上真并怀孕。胎儿（后称突变婴儿／风祭新）已能对父发送遥测式感应。财团要回收「商品样本」。

⑤ **营救与惨剧**  
大门知真相后被囚，与爱试图逃亡。真救出二人，却再被财团直升机捕获。乱战中冰室严为灭口扫射，爱为真挡枪身亡。真完全觉醒为假面骑士真，撕裂冰室，再战豪岛，最终将 Level2 头部连脊椎拔起——血腥、原始、无骑士踢套路。

⑥ **终局**  
赛拉接总部「处死真」令率新部队逼近。大门在直升机／ISS 爆炸中死亡。真抱爱的遗体（腹中胎儿仍有反应）消失在夜色。CIA 档案写入代号 MASKED RIDER。财团日本支部溃，全球网仍在。无「正义胜利烟花」，只有幸存者与未完成的序章。

⑦ **可介入微观**  
受试者档案失窃、体育俱乐部尾随、ISS 门禁、CIA 与财团火并现场的录音带、冰室的商品目录、大门的良心实验笔记。

⑧ **蝴蝶边界**  
可改：早发现鬼塚、救下个别路人、截获一份 Level 名单。不可无代价改：抹掉爱之死的悲剧核、让财团全球一夜消失、把真写成无痛热血儿。

**【主要人物】**
- **风祭真／假面骑士真**｜性格：正直、痛苦、被愤怒与爱驱动。｜装备·能力：Level3 稳定蝗虫化；再生；高周波爪；第三之眼保理性。｜弧光：受试者→自疑凶手→觉醒复仇→抱尸消失。｜立场：对契约者先防后用；勿碰爱的纪念。
- **明日香爱**｜性格：温柔、矛盾、最终选择爱。｜能力：护士技能、财团监视权限。｜弧光：监视员→恋人→挡枪亡。｜立场：关键情感锚。
- **风祭大门**｜性格：学者良心与被骗后的悔。｜能力：免疫工程权威。｜弧光：被利用→逃亡→死于爆炸。｜立场：可合作的真相源。
- **鬼塚义一**｜性格：狂信「新文明之神」。｜能力：Level3 失控体、呼应反应。｜弧光：研究者→杀人鬼→被焚。｜立场：二阶威胁。
- **冰室严**｜性格：温和皮囊下的冷酷管理者。｜能力：ISS 权限、枪械。｜弧光：欺骗大门→灭口→被真撕杀。｜立场：一阶机构恶。
- **豪岛**｜性格：忠诚兵器。｜能力：Level2 斩击。｜弧光：追杀真→被拔头。｜立场：武力敌。
- **赛拉·深町**｜性格：任务忠诚与一丝不忍。｜能力：CIA 指挥、非合法行动。｜弧光：猎杀→暂赦→再奉命处决。｜立场：可谈不可托付。
- **结城卓也**｜性格：开朗前辈。｜能力：常人。｜弧光：日常支撑。｜立场：情报与人情。
- **突变婴儿（风祭新）**｜性格：未成长。｜能力：胎内遥测、能量波。｜弧光：爱死后仍指引真。｜立场：未来火种（本阶勿神化）。
- **财团干部（纽约线）**｜性格：商品逻辑。｜能力：资金与政治。｜弧光：幕后。｜立场：情报优先。

**【势力图谱】**
- **财团**：军工复合跨国体，要统一人从生到死的经济。
- **ISS**：日本生化学支部，表面医疗、实际造兵。
- **CIA 赛拉队**：防生物兵器扩散，手段不干净。
- **真的私人圈**（爱、大门、结城）：被碾碎的人性侧。

**【贵重物品】**

- **鬼塚手术记录**：呼应反应关键。
- **第三之眼控制器官**（生理）：真保持理性的关键。
- **CIA 代号文件 MASKED RIDER**：历史命名瞬间。

**【隐藏剧情 · 伏笔】**
- 片名「序章」暗示未完成续作；石之森称其为「0 号」根。
- 初期名「假面骑士盖亚」设计更传统，后改为生物恐怖蝗虫人。
- 真无专属摩托车、不喊「变身」、战斗原始——刻意反骑士神话。
- S.I.C. HERO SAGA 等外传写数十年后失忆真与子「新」——本任务世界可不启用。
- 财团未灭：击败 ISS≠击败体制。

**【大事记时间线】**
真自愿受试 → 鬼塚植入基因 → 都市连续杀人 → 真自疑 → CIA 焚鬼塚 → 真初觉醒战豪岛 → 爱怀孕与逃亡 → 冰室射杀爱 → 真屠冰室与豪岛 → 大门死、ISS 爆 → 真消失，档案写入 MASKED RIDER。

**【叙事基调 · 雷区】**
成人向生物恐怖、悲剧、体制绝望。忌：儿童向口癖变身秀；无代价复活爱；把真写成快乐英雄；忽视血腥拔头等尺度。最早切入：真仍只做噩梦、尚未知鬼塚真相时；或 CIA 袭卡车前后。

**【ISS 走廊】**白壁、消毒水、微笑海报写着「为明日医疗」。海报背后是 Level 培养观察窗。契约者若穿白袍，比穿铠甲安全。门禁分访客／研究员／所长三级，冰室的卡能开你不该开的门。

**【呼应反应机制】**鬼塚杀戮时，真同步「看见」。切断同步需鬼塚死亡或药物抑制——药物在冰室保险柜。真若在同步中变身，可能误伤目击者；赛拉因此更难判断他是人是怪物。

**【赛车手身体】**真的基础体能解释其为何被选为 Level3 容器：高耐受、高恢复潜质。结城的训练日志是侧面证据，可被财团销毁。

**【CIA 与财团的平行恶】**一方要灭口，一方要商品，双方都不在乎爱是否微笑。契约者夹在中间，最优是「让两边互相消耗」而非选边站到底。

**【战斗美学】**无必杀姿势，只有求生与复仇的撕咬。写动作用「爪、牙、再生、拔扯」，慎用「骑士踢定胜」。豪岛被拔头是尺度锚，不可美化成帅。

**【一阶任务适配】**窃档、跟踪、救人出研究所、录音带，比正面打 Level3 合理。奖励是信任与情报，不是变身器。

**【二阶任务适配】**协助真对豪岛、干扰直升机、在爆炸前抢出大门笔记。爱之死是悲剧核，可延缓不可无代价取消。

**【财团商品逻辑】**目录把受试者按再生率、服从度、呼应强度标价。真因第三之眼「服从度不可控」而被标为风险商品——这是他活到觉醒的缝。

**【爱的双面工牌】**一面护士，一面监视。她关掉监视摄像头的那晚，是人格压过任务的名场面。胎儿反应记录是后半程导航。

**【赛拉的扳机纪律】**她能下令开火，也能下令「先观察」。对契约者而言，让她看见「真救人」比看见「真杀人」更能换临时豁免。

**【大门的悔】**他签字时以为在救人，发现真相后所有公式都成了罪证。笔记里有「停止人体实验」草稿，未寄出。

**【都市夜景】**连续杀人案让巷弄空、霓虹仍亮。真骑 500cc 穿过空街，像逃离自己的影子。写氛围用雨、血、消毒水，不用王道 BGM。

**【体育俱乐部夜】**真练完车下来，爱递水，结城起哄。普通灯光下看不出 Level 标签。契约者若在此切入，冲突是「尾随与窃听」，不是开打。冰室的人坐在角落看报纸，其实在看真的恢复曲线。

**【公路火光】**CIA 火箭弹不是正义特效，是生物兵器管控的脏手。鬼塚的尖叫经呼应灌进真的颅骨，变身从痛苦开始。赛拉的瞄准镜先锁怪物，后锁「仍用人眼看人的怪物」。

**【爆炸与拥抱】**大门之死与爱之死叠在同一夜，真的复仇没有观众掌声。他抱遗体离开时，都市继续亮广告。MASKED RIDER 是档案员的命名，不是他自己的宣言。

**【一阶奖励池细化】**门禁卡复制、抑制剂半支、被救路人的证词、大门未寄出的停实验草稿复印件。

**【二阶奖励池细化】**商品目录全页、第三之眼观察草图、赛拉临时「观察豁免」口述、爆炸前抢救的硬盘。

**【序章观测1】**时点切片：实验室门禁灯色、鬼塚绷带缝与手术记录页码、赛拉耳麦口令班次、爱的工牌编号与关掉的摄像头编号、真的手套握痕与车钥匙、直升机机尾号、冰室保险柜药瓶标签、商品目录价位栏。观测1绑定不同介入入口（ISS／俱乐部／公路／爆炸夜），供物证链引用，禁止空话填充。

**一阶（都市阴谋 · 受试者与档案）**

切入身份/时点：契约者以 ISS 临时实验助理／体育俱乐部会员／私家侦探切入，锚定真仍只做噩梦、连续杀人案见报、鬼塚尚未被 CIA 焚死之前。

开场白建议：「白墙上的微笑海报写着为明日医疗。你手里的档案却把明日标了价。窗外都市灯火正常，有人已经在用别人的基因练习神。」

主线钩子/支线：主线＝确认 Level 计划并让大门或真看见证据。支线＝跟踪夜袭路线不硬刚、复制档案、保护一名潜在受害者、在俱乐部给真不在场证明。
