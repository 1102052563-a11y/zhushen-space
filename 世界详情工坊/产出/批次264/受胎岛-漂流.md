# 受胎岛-漂流
<!--meta lib=主库 tiers=二-->

## 剧情

**【作品来源】**
《受胎島「どうしてアンタみたいなブサ男に種付けされなきゃいけないのよ?!」》为ルネ于 2012年7月27日发售的 18禁 ADV；原画ブッチャーU，后有 OVA。舞台：课外船难后的病毒孤岛。本档分锚「**漂流**」。成人向只写生存规则、病毒机制、权力反转与任务介入，不写色情过程。

**【世界定位】**
神宮司学園师生因暴风雨漂流至废墟孤岛；岛上病毒仅感染女性，强制排卵与高热，精液被设定为「特效药」从而把唯一男性**田中翔太**推上资源垄断位。本档锚点：**漂流**。一句话：雨与废墟之间，治疗权就是主权。

**【世界观 · 力量体系】**
力量本源：病毒时限、体力、工具、药品知识、人数与暴力、岛地形。死亡：病毒末期、坠崖、互斗可致死；无超自然复活。特殊系统：
- **女性感染时钟**（与生理周期类比的致死窗）；
- **特效药垄断**造成的权力反转；
- **一条静**的医学话语权（OVA 中更接近病毒相关黑手）。

力量阶梯：
1. **重病／濒死感染者**：行动力崩，≈一阶下。
2. **健康男性幸存者／轻症女性／本档主战**：近战与工具，≈**二阶**边缘到二阶（岛内无枪械体系，以生存暴力计）。
3. **持械小团体／医疗知识垄断者**：≈二阶上。
4. **病毒源头与完整疫苗控制者**：对本档超阶规则杀。

**乐园阶位映射**：濒死≈一阶下；岛内生存冲突≈**二阶**；疫苗垄断≈更高。本任务世界仅覆盖**二阶**。

**【地理 · 舞台】**
- **废墟聚落**：教室残壳、诊所、仓库。
- **海岸与船骸**：信号与撤离幻想。
- **密林与水源**：日耗资源点。
- **地下／隔离病房**：病毒观察。
- **电台残塔**：求救。
写场景：盐腥、铁锈、雨、发热喘息、发电机噪声。

**【世界剧情线】**
① **船难**  
课外船遇风暴；定员问题与排斥导致翻船；众人漂至孤岛。

② **病毒发现**  
女医**一条静**指出感染规律与「特效药」结论；女性阵营对**田中翔太**的态度从蔑视转为依赖与恨。

③ **本档锚点：漂流**  
本档只写**船难到登岛的 0–48 小时**：定员排斥、翻船、盐腥昏迷、废墟第一夜。病毒尚未被一条静完整命名，权力结构仍是校园欺凌延长线——如月麗仍在骂田中翔太「キモ男」。卡片锚点＝**漂流生存＋排斥因果＋第一夜发热前夜**。可玩：救落水者、藏淡水、阻止私刑、保护电台零件不被当柴烧。

④ **权力反转与分轨**  
翔太从被霸凌者变为垄断者；如月麗等原霸凌结构崩解。OVA 分轨中存在死亡、监禁、疫苗争夺与一条静黑手化。

⑤ **蝴蝶边界**  
可改：分发治疗节奏、保护弱势、摧毁病毒样本、发出求救。不可无代价改：病毒已在岛上循环的生态事实。

**【隐藏剧情 · 伏笔】**
- 病毒来源与一条静关系（OVA 黑手化）。
- 墓与死亡人数分轨。
- 「繁殖／受胎」在规则上被包装成治疗，实为权力。
- 本档「漂流」只结算该切片。

**【大事记时间线】**
课外出航 → 风暴翻船 → 登岛 → 病毒确认 → 本档「漂流」 → 权力反转／死亡／求救分轨。

**【叙事基调 · 雷区】**
生存惊悚、权力反转、伦理崩坏。忌美化强制；忌二阶无视病毒时钟；忌「被削弱所以病毒无效」。最早切入：登岛第一夜发热者出现时。
