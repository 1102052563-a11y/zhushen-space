# 甜蜜家园

## 剧情

**【作品来源】**
《甜蜜家园》（韩语스위트홈，英文Sweet Home）为Netflix韩国末日恐怖剧，改编自金坎比（Kim Carnby）原作、黄永灿（Hwang Young-chan）作画的Naver网络漫画。导演李应福等，主演宋江、李阵郁、李施昤等。第一季2020年12月18日上线（10集），第二季2023年、第三季2024年完结。本档案以**剧集第一季绿家公寓围城主线**为卡片锚点核心，二、三季作世界扩展层。文风：欲望具象化怪物、封闭公寓群像、人性与「成为怪物」的伦理。卡片锚点：**蛋白质怪物——公寓幸存者对抗巨大肌肉型蛋白质怪物，物理攻击难以奏效，需利用火焰与地形将其推下楼**。

**【世界定位】**
近未来/当代韩国都市突发「欲望怪物化」：人若被强烈欲望支配，会异变成对应形态的怪物（暴食、肌肉、眼球、蜘蛛、吸血等）。主舞台是破旧公寓「绿家（Green Home）」：幸存者封锁大门，在楼层间争夺水电、食物与信任。一句话：怪物从心里长出来，公寓是最后的甜，也是最先的地狱。

**【世界观 · 力量体系】**
无传统灵力。规则核心=**欲望→怪物化**；部分个体可成「半人半怪」保留理智（如车贤秀）。战力=**冷兵器与即用武器 + 地形与火 + 组织纪律 + 特殊体质再生**。鼻出血与幻听是变异前兆。枪械稀缺；铁匠韩斗植可打造盾与矛。军方后期对公寓「清场」级别介入。死亡大体即死；附身/拟态（郑义明等）属高阶规则威胁。

威胁/战力层级（宁低勿高）：

1. **未武装居民 / 恐慌者**  
   凡人。乐园对照≈一阶。

2. **持棒/刀的幸存者小队**  
   可清单体弱怪。乐园对照≈一～二阶。

3. **普通怪物（暴食、眼球、吸血等）**  
   个体强于人，可破门。乐园对照≈二～三阶。

4. **蛋白质怪物（卡片锚点）**  
   巨大肌肉型，物理打击抗性极高，需火与地形击落。乐园对照≈三～四阶。

5. **半人半怪（车贤秀级）与组织化匪帮**  
   再生、怪臂、战术协同。乐园对照≈四～五阶。

6. **军方清场火力 + 高位拟态/研究黑幕（郑义明、南相元线）**  
   系统杀与规则杀。乐园对照≈五～六阶。

乐园阶位映射：恐慌居民≈一阶；持械守门≈二阶；普通怪物巷战≈三阶；蛋白质怪物攻防≈四阶；半怪觉醒与匪帮≈五阶；军方炮击/研究体与拟态≈六阶。世界顶点=**欲望怪物化规则+国家实验与清场机器**，条件性胜利=守住绿家一夜/护送撤离/揭露实验，严禁一阶「净化全国欲望」。

**【地理 · 舞台】**
- **绿家公寓（Green Home）**：垂直迷宫；地面层、地下室、楼层、天台。
- **车贤秀的房间**：抑郁与监控自我的起点。
- **韩斗植工坊**：武器与防御工程核心。
- **地下停车场**：找车与伏击高发。
- **城市街道与便利店**：外出补给的尸海。
- **军事隔离与实验设施**（中后期扩展）：南相元研究线。
- **体育场/避难新城**（二季向）：出公寓后的新牢笼。

**【世界剧情线】**

○ **前史：车贤秀的空**  
车祸夺去车贤秀全家，他退学、抑郁、搬进绿家，靠外卖与隔绝度日。都市裂缝已在别处渗出怪物新闻，尚未成为「全国事实」。

○ **第一夜：邻人吃猫**  
女邻居偷外卖后生吃宠物并异变成暴食怪，袭击尹智秀后坠楼。地面层出现吸血怪，李恩赫与前消防徐以景将其逼出并封锁入口。居民惊恐发现整座城市已沦陷。

○ **集结与前兆**  
贤秀鼻出血、幻听，获再生能力。孤儿金秀英、金永秀被救；任明淑短暂怪物化又退回。电力中断；以景在地下室遇蜘蛛怪。全国转播总统异变，希望破碎。车珍玉为寻女打开卷帘，军人李秀雄冲出皆死。

○ **欲望规则公开**  
军方广播否定「咬伤传染」，揭示**强烈欲望导致异变**。贤秀被部分居民视为威胁；金锡贤亦出现症状被关。片商相旭追杀儿童性侵者崔允宰（契约杀手线）。蛋白质怪物威胁儿童，明淑再异变掩护。

○ **蛋白质怪物与外出（卡片锚点）**  
巨大肌肉型蛋白质怪物反复冲击防线，刀棍难伤。幸存者利用火焰、车辆、楼层落差将其逼退或击落——物理硬刚无效，**火与地形**是解。以景驾车冲撞等名场面强化此逻辑。

○ **手术、匪帮与郑义明**  
智秀阑尾炎，恩赫操刀；外出取药遇强化蛋白质怪。以景带回军方追踪器与南相元研究线索。匪帮入侵杀人；郑义明以拟态/怪物本质挑拨，诱导贤秀质疑人类是否容纳半怪。贤秀完全怪物化击退义明，斗植以人性将其拉回并牺牲。

○ **坍塌与撤离**  
军方炮击绿家。幸存者经地下通道撤离；恩赫似亡又留鼻血伏笔。贤秀「投降」后发现身旁的相旭实为义明拟态——第一季收束于更广阴谋。

○ **二、三季扩展边界**  
出公寓后：特种部队「乌鸦」、疫苗研究、徐以景的怪物之子、新人类/茧生威胁等。任务默认可停在绿家闭环，或延伸至「出城即新地图」。

○ **蝴蝶效应边界**  
可改：某次守门成败、谁先异变、蛋白质怪是否更早击落。不可无代价改：欲望→怪物规则、蛋白质怪的物理抗性、军方清场逻辑、半怪被社会排斥的主题。

**【主要人物】**
- **车贤秀**｜性格：抑郁、孤僻、内在善良。｜装备·能力：半怪再生、怪翼臂。｜人物弧光：求死者→绿家盾→被国家与义明争夺的样本。｜立场关系：智秀、斗植的牵绊；义明的宿敌。
- **片商相旭**｜性格：疤面、冷硬、护弱时有信。｜装备·能力：格斗顶尖、杀手履历。｜人物弧光：寻仇者→居民盟友→被义明附身/冒充。｜立场关系：追杀允宰；后期身份可疑。
- **徐以景**｜性格：冷静、行动派（剧集原创消防/特战向）。｜装备·能力：枪械与近战、领导力。｜人物弧光：寻未婚夫→军方交易→撤离核心。｜立场关系：南相元线；孕线（剧情进展后）。
- **李恩赫**｜性格：理性、看似冷血的领袖。｜装备·能力：医疗知识、决策。｜人物弧光：地面层指挥→为撤离牺牲姿态。｜立场关系：恩瑜义兄。
- **尹智秀**｜性格：创伤吉他手、倔。｜装备·能力：球棒。｜人物弧光：幸存者战士。｜立场关系：在宪、贤秀。
- **李恩瑜**｜性格：高傲、受伤的芭蕾梦想。｜装备·能力：有限。｜人物弧光：被保护者→撤离成员。｜立场关系：恩赫义妹。
- **韩斗植**｜性格：残疾铁匠、邻人温情。｜装备·能力：锻造武器与工事。｜人物弧光：后勤核心→拉回贤秀时牺牲。｜立场关系：贤秀的人性锚。
- **郑在宪**｜性格：教师、信仰与酒。｜装备·能力：真剑。｜人物弧光：守护者→断臂后牺牲。｜立场关系：智秀暗线。
- **郑义明**｜性格：研究者外壳的怪物意识形态。｜装备·能力：拟态、枪、怪物本质。｜人物弧光：煽动半怪独立→季终伏笔。｜立场关系：人类秩序的颠覆者。
- **蛋白质怪物**｜性格：无（欲望具象）。｜装备·能力：巨力、高抗物理。｜人物弧光：卡片锚点环境BOSS。｜立场关系：逼出战术创新。
- **其他居民**｜金锡贤/安善英、安吉燮/朴俞梨、任明淑、车珍玉、柳在焕、孤儿兄妹等：群像零件，须真名出现。

**【贵重物品】**

- **真剑（在宪）**。
- **球棒（智秀）**。

- **军方追踪器与南相元文件**。

**【隐藏剧情 · 伏笔】**
- 异变非病毒咬传，是欲望伦理灾难。
- 政府早知并实验（南相元失踪）。
- 半怪是否被人类接纳=主题题眼。
- 相旭的「归来」可能是义明。
- 恩赫鼻血=未完的怪物化。

**【大事记时间线】**
- 车祸前史 → 贤秀搬入。
- Day1夜 → 暴食怪、封锁大门。
- Day2–N → 集结、停电、总统异变。
- 中期 → 蛋白质怪攻坚、手术、外出。
- 后期 → 匪帮、义明、炮击、通道撤离。

**【叙事基调 · 雷区】**
基调：窒息、腥、绝望中的微光。  
忌：把怪物当数值图鉴刷；忌美化欲望无代价；忌忽略火/地形解法硬刚蛋白质怪；忌低阶净化全国。最早锚点：贤秀关门夜，或蛋白质怪撞门瞬间。

**【世界剧情线·细部展开一：欲望即形状】**
暴食者成巨口，迷恋肌肉者成蛋白质怪，恋婴者成胎儿姿态——身体是欲望的说明书。AI写新怪须先写「他缺什么」。前兆：鼻血、耳语、体温。压抑可延迟，爆发更烈。

**【世界剧情线·细部展开二：绿家政治】**
恩赫用逻辑分配危险任务，常派贤秀。居民用投票决定谁该被赶出。善英长期受家暴，异变后的处决是解放也是罪。配额表比怪物更先引发拳头。

**【世界剧情线·细部展开三：蛋白质怪战术学】**
弱点击：关节？不——**火、坠落、重型撞击**。楼道狭窄可限制其转身；天台与破窗是陷阱。硬刚=团灭。卡片锚点要求写清：谁点火、谁引怪、谁推。

**【世界剧情线·细部展开四：半怪伦理】**
贤秀救人后仍被枪指。斗植的死是「人拉住怪」。义明提供相反答案：人类不配被守。契约者站队会改变撤离名单。

**【人物群像补强】**
在宪的酒与剑、俞梨的哮喘与护理、吉燮的病与幽默、在焕的卫生焦虑、孤儿的沉默——每次死亡留一句生前特征。

**【公寓垂直地理】**
上层=资源与伏击；底层=门与尸压；地下室=电与蜘蛛；天台=刑讯与对峙。写场景先报楼层与电力状态。

**【微观可介入事件库】**
①帮斗植打磨矛头。②蛋白质怪来袭时负责点火。③陪恩赫做阑尾手术护灯。④跟踪以景与军方交易。⑤识破匪帮内应。⑥在贤秀失控时喊他的人类名。⑦偷出南相元档案页。⑧炮击前打开地下通道锁。

**【主题与AI禁区】**
主题是欲望、接纳与「家」的崩解。禁止无脑刷怪升级；禁止把儿童创伤写成噱头。

**【开场分镜】**
外卖袋、没洗的碗、新闻里模糊的「暴动」。猫叫被咀嚼声取代。卷帘落下的巨响是绿家的生日。

**【补完块·武器工业】**
垃圾桶盖→盾；管钳→锤；窗帘绳→套索。斗植的砂轮声是希望的BGM。

**【补完块·儿童线】**
秀英永秀不是道具，是居民是否仍人类的检验。谁先把食物让给孩子，谁就在道德账本上记账。

**【补完块·以景的过去】**
消防与特战训练解释其枪感；未婚夫失踪把个人线焊上国家实验。

**【补完块·二季接口】**
出通道后的光刺眼，新制服与新谎言等待。可玩「体育场纪律」与「乌鸦部队」但不强制。

**【补完块·声音与味】**
怪物关节脆响、血铁锈、断水后的干臭、莫洛托夫的汽油甜。

**【甜蜜家园·场景零件库】**
电梯废井、灭火器、儿童画、监控死角、天台水箱。禁止都市奇幻串味。

**【同盟税率】**
默认脆弱；恩赫的 Rational 决定常被恨。背叛披「他会变成怪物」外衣。

**【时间状态机】**
AI回答：电力？门？蛋白质怪是否已击杀？贤秀是否暴露半怪？军方距离。

**【补完块·匪帮逻辑】**
他们要的是女人、食物与「可变现的半怪」。不是意识形态，是掠夺。义明混入是降维打击。

**【补完块·总统异变】**
电视里的国家形象碎裂，比窗外怪吼更毁希望。此后「等救援」变成笑话。

**【补完块·终季语气锁定】**
炮火中的通道很挤，有人推有人拉。贤秀走向灯光时，家在身后塌成尘。契约者最高奖励是「一张撤离腕带」与「半瓶还能用的汽油」，以及「仍能叫出彼此真名」。

**【补完块·更多居民切片】**
锡贤的便利店钥匙曾是权力，异变后成忌器；善英洗碗的手后来握刀；明淑推婴儿车的幻觉与真实怪物形态重叠——写她须痛而不猎奇。在焕抱怨没有护肤品的台词，是末日里可笑又真实的人味。

**【补完块·停车场战役】**
熄火的车、闪烁的应急灯、蛋白质怪的脚步让后视镜抖。找车=找棺材或找生路。恩赫的计时与相旭的咒骂并置。

**【补完块·信仰与剑】**
在宪的剑是传统文化残片，挥砍时他念的不是经，是人名。断臂后他选择把时间卖给同伴。

**【补完块·研究话术】**
文件会写「诅咒」「受试者」「欲望阈值」。官僚语言清洗血腥，与釜山行的企业来电同构。


**【欲望形变百科（绿家已见证）】**
暴食怪：巨口、不停咀嚼、以宠物与人为食。蛋白质怪：肌肉山、抗物理、畏火与坠落。眼球怪：视域诡异、走廊猎手。吸血怪：大厅初夜。蜘蛛怪：地下室电网守护者的噩梦。胎儿姿态怪：明淑线，须严肃。拟态/液态高位：义明。半怪：贤秀的翼臂与再生。AI新增怪物必须先写欲望词条，再写外形，禁止无来由的「很强」。

**【绿家垂直战争史】**
B1：电、蜘蛛、以景的枪。1F：门、投票、配额。中层：清怪、手术、匪帮破门。上层：相旭寻仇、天台刑讯。天台：义明哲学课。每一层有自己的王与税。恩赫的命令从1F向上传，常在上层变味。

**【蛋白质怪物战役完整复盘】**
第一次撞击：拒马断，有人哭。第二次：在宪剑折一截。第三次：以景引入火与车。儿童转移路线必须先于点火。贤秀若以肉身挡，会触发半怪暴露公投。卡片要求写出：火点、推点、落点、谁喊撤。物理硬刚=团灭，写都不要写成功。

**【半怪接纳公投】**
居民用眼神投票。斗植用命投票。义明用枪投票。军方用炮投票。贤秀的鼻子是否流血，是下一轮公投的议程。契约者的发言会改变名单：谁上通道，谁留下「自愿」。

**一阶（绿家封锁夜）**

切入身份/时点：契约者以「新搬入租客 / 外卖骑手被关在内 / 贤秀网友」在暴食怪事件夜进入绿家。

切入身份/时点：以「斗植学徒 / 巡逻队员」切入物资紧张期。

初始事件：水只剩半箱；有人隐瞒零食。你要参与制定或推翻一份不公平的配额。

开场白建议：「怪物在门外排队，人在门内用勺子战争。铁匠的砂轮比口号管用。」

关键NPC立场：**韩斗植**——给工具；**恩赫**——冷逻辑；**锡贤**——自私样本；**在宪**——道德声。

主线钩子/支线：钩子=完成一层拒马；支线=揭发藏粮；支线=夜巡活捉前兆者。

危险度/规避：中高；内斗。

任务方向/奖励：工事权；奖励=自制矛、斗植信任。

**三阶（楼层清怪）**

切入身份/时点：以「突击队员」切入眼球怪/蜘蛛怪等清楼任务。

初始事件：走廊灯闪，墙上有爬行黏液。你的小队少一人——他鼻血流了一路。

开场白建议：「欲望有脚声。你追的不是猎物，是某人昨天还在笑的邻居。」

关键NPC立场：**贤秀 / 智秀 / 在宪**——可能队友；**普通怪物**——环境敌。

主线钩子/支线：钩子=夺回一层安全；支线=救下未完全异变者；支线=破坏招怪声源。

危险度/规避：高。

任务方向/奖励：战斗经验；奖励=药品、真剑保养油、半怪情报。

**四阶（蛋白质怪物 · 锚点）**

切入身份/时点：以「引火手 / 消防车接应 / 儿童疏散员」切入蛋白质怪物攻坚。

初始事件：肌肉山撞裂防栅；刀折了。有人喊火，有人喊推，有人喊跑。你握着汽油瓶，手抖。

开场白建议：「它的皮肤像轮胎，你的刀像牙签。要想让山移动，就让山燃烧，或让山坠落。」

关键NPC立场：**蛋白质怪物**——BOSS；**以景**——冲击战术；**贤秀**——肉盾/诱饵；**儿童与老弱**——必须先撤。

主线钩子/支线：钩子=卡片锚点击退/击落；支线=保护点火点；支线=修复被撞开的门；支线=记录抗性数据给「未来的研究者」。

危险度/规避：极高；禁止无火硬刚。

任务方向/奖励：中阶峰值；奖励=剩余汽油、战术声望、恐惧症。

**五阶（半怪与匪帮）**

切入身份/时点：以「贤秀监护人 / 匪帮卧底 / 手术灯守护」切入智秀手术与匪帮入侵、义明现身期。

初始事件：匪帮枪口对准居民；屋顶上义明对贤秀说「他们不会要你」。你要选：帮人类杀半怪，或帮半怪逃。

开场白建议：「枪指着怪物，也指着还长着人脸的怪物。你分得清欲望和人吗？匪帮不分。」

关键NPC立场：**郑义明**——意识形态；**车贤秀**——失控边缘；**相旭 / 以景**——武力；**匪帮头目**——掠夺。

主线钩子/支线：钩子=击退匪帮并稳住贤秀；支线=保护手术中的智秀；支线=识破拟态；支线=转移孤儿。

危险度/规避：极高。

任务方向/奖励：伦理立场固化；奖励=枪械零星、义明样本碎片、贤秀羁绊。

**六阶（炮击与通道）**

切入身份/时点：以「通道向导 / 军方翻译叛逃者 / 贤秀投降同行」切入绿家被炮击、地下撤离。

初始事件：楼体倾斜；灰尘里有人拉你的手。外面扩音器叫「半怪交出」。相旭的笑看起来不太对。

开场白建议：「家在倒塌，国家在喊口令。通道里只有一种光，通向另一种围墙。你还要不要叫它甜蜜？」

关键NPC立场：**撤离核心名单**（以景、智秀、恩瑜、孤儿等）；**军方**——清场；**义明（拟态）**——终局阴影；**斗植/在宪等亡者**——记忆压力。

主线钩子/支线：钩子=带出最大存活人数；支线=交出或藏匿贤秀；支线=销毁/公开实验文件；支线=验证相旭身份。

危险度/规避：极高·贴近顶点；条件性胜利=撤离与真相，不求灭规则。

任务方向/奖励：出公寓门票；奖励=腕带、文件、永远的咳嗽与尘肺式记忆。
