# 釜山行

## 剧情

**【作品来源】**
《釜山行》（韩语부산행，英文Train to Busan）为2016年韩国动作恐怖片，导演延尚昊（Yeon Sang-ho），编剧朴柱锡，主演孔刘、郑裕美、马东锡、金秀安、崔宇植、安昭熙、金义城等。戛纳午夜场首映，韩国票房破千万观影。同宇宙尚有动画前传《首尔站》（2016）与独立续作《半岛》（2020）；本档案以**电影《釜山行》正典**为准，不自动并入续作时间线。文风：封闭列车惊悚、丧尸群像、阶级讽刺与父女救赎。卡片锚点：**大田站军队丧尸——主角团在大田站遭遇已被感染的军队，玻璃门后尸群涌出，赤手空拳与棒球棍对抗成群军人丧尸**。

**【世界定位】**
当代韩国（约2010年代中期）突发高传染性尸变。故事主舞台是从首尔站开往釜山的KTX-101列车及沿线车站：天安、大田、东大邱直至釜山外围封锁线。病毒使感染者在极短时间内变为高速、趋光、群聚的「活尸」。一句话：列车是移动的棺材，车站是假安全区，人性比病毒更先撕开车门。

**【世界观 · 力量体系】**
无超凡灵力。战力=**体能与搏击 + 即用武器（棒球棍、消防斧、门闩）+ 空间控制（封门、隔车厢）+ 情报与阶级话语权**。感染路径为咬伤/血液接触；潜伏极短，症状：出血、痉挛、瞳孔与攻击性骤变。丧尸特性：高速、可攀爬堆叠、对声音与光敏感；黑暗中行动迟缓（可被利用）。死亡即死，无复活。军方与企业救援承诺常比尸群更不可靠。

威胁/战力层级（宁低勿高）：

1. **未武装乘客 / 恐慌踩踏者**  
   近乎凡人，死于推挤与误判。乐园对照≈一阶下沿～一阶。

2. **持钝器的幸存者小队**（石宇、尚华、勇国等）  
   可清单节车厢、封门据守。乐园对照≈一阶～二阶初。

3. **密集尸潮 / 车站失控区**  
   数量碾压，个人武力无效。乐园对照≈二～三阶环境威胁。

4. **大田站军人丧尸潮（卡片锚点）**  
   制式训练残留的冲击力+玻璃门后的尸海，是列车中段最强压迫。乐园对照≈三～四阶场面。

5. **企业高管与制度性弃人（延锡等）**  
   个人武力弱，但可煽动乘客锁门、把「可能被感染者」隔离致死。乐园对照≈四阶社会杀。

6. **国家级封锁与釜山检疫区**  
   路障、火力、误杀与救援并存；顶点是**失控的国家应急机器**，非单只BOSS丧尸。乐园对照≈五～六阶（系统侧）。

乐园阶位映射：恐慌乘客≈一阶；持械小队清车厢≈二阶；尸潮与站台混战≈三阶；大田军人尸潮与东大邱火灾事故≈四阶；企业弃人与制度性隔离≈五阶；釜山外围封锁/国家机器与父女终局≈六阶。世界顶点=**全国级尸变+应急国家暴力与企业责任链**，低阶切入情报优先，条件性胜利=护人抵达安全区/揭露企业责任，严禁写成「一刀砍停全国疫情」。

**【地理 · 舞台】**
- **首尔 / 基金公司办公室**：石宇的日常与缺席的父爱。
- **首尔站 KTX 月台**：被抓伤的感染者上车，疫情上车时刻。
- **KTX-101 各车厢**：门、过道、厕所、餐车——移动迷宫。
- **天安站**：短暂停靠，外部已乱。
- **大田站（卡片锚点）**：本应有军队接应，实为军人丧尸与玻璃门尸海。
- **东大邱站**：轨道堵塞、火烧机车、换车与背叛高发。
- **釜山方向隧道与外围封锁线**：终局步行与军方误射风险。
- **感染源头暗示**：生化相关企业事故（石宇所在资本链间接责任）。

**【世界剧情线】**

○ **前史：鹿与城市裂缝**  
开场被卡车撞倒的鹿重新站起，暗示尸变已在郊外蔓延。首尔表面正常，新闻与短信逐步报出「示威」「事故」「封路」。

○ **父女上车**  
基金经理石宇因工作错过秀安的夏威夷民谣《Aloha ʻOe》演出，用电脑游戏敷衍生日。秀安坚持去釜山见母亲罗英。二人登上KTX，同车有：怀孕的成景与工人丈夫尚华、高中棒球队（含勇国与啦啦队长珍熙）、高管延锡、年迈姐妹仁吉与钟吉、一名精神创伤的流浪汉等。一名满身伤痕的女子在发车前挤上车，随即在车厢内尸变，咬伤乘务员，疫情爆炸式扩散。

○ **车厢争夺与封门**  
幸存者逃向后部车厢，学会「关死门、别开门」。石宇起初只护女儿，在尚华的斥责与秀安的目光下逐渐从「利己中产」转向「为他人挡门」。列车经天安，外部已现尸变。

○ **大田站崩盘（卡片锚点核心）**  
广播称大田有军队与隔离。下车后发现士兵与石宇公司派来的接应队均已感染。玻璃门后尸群冲出，棒球棍与赤手对抗军人丧尸。幸存者被迫退回列车，队伍被拆散进不同车厢。

○ **穿越尸海重聚**  
石宇、尚华、勇国武装穿越被尸占的车厢，与秀安、成景、仁吉、流浪汉重聚。其他乘客恐惧「新来者可能已感染」，在延锡与乘务员基哲煽动下拒绝开门。尚华在强行开门时被咬，托付石宇照顾成景、为未出生女儿取名「尹秀允」，自己断后牺牲；仁吉亦失散/被弃。

○ **自私的车厢与钟吉开门**  
延锡一派要求「新乘客」隔离到前庭。钟吉厌恶众人的冷血，主动打开通往尸区的门，让尸群吞噬自私者；延锡与基哲躲进厕所逃过一劫。石宇接到电话，得知自己公司与疫情源头有间接关联，罪感压顶。

○ **东大邱换车与终局**  
轨道堵塞，司机寻找新车。延锡推基哲挡尸逃生，又把勇国与珍熙当诱饵；珍熙被感染后勇国绝望赴死。石宇、秀安、成景、流浪汉被倾覆车厢压住；流浪汉牺牲开路。三人登上新车头，遭遇已感染的延锡；石宇将其抛下却被咬。石宇把秀安与成景锁进驾驶室，教成景开车，回忆秀安出生后跳车自尽。秀安与成景在釜山外围隧道前被军方瞄准，秀安唱《Aloha ʻOe》证明自己是人类，二人获救。

○ **蝴蝶效应边界**  
可改：某次封门是否成功、谁在大田先死、延锡煽动是否被揭穿。不可无代价改：尸变高速规则、大田「军队已死」的残酷反转、尚华式牺牲的道德重量、终局国家封锁的误杀风险与企业责任阴影。

**【主要人物】**
- **石宇（Seok-woo）**｜性格：犬儒、工作狂、离婚后缺席的父亲。｜装备·能力：中等搏击，后期以决断与挡门为核心。｜人物弧光：利己中产→护女与护孕的牺牲者。｜立场关系：秀安的父；尚华的战友；延锡的对立面。
- **秀安（Su-an）**｜性格：早熟、正直、会唱歌。｜装备·能力：无武力，道德罗盘。｜人物弧光：被保护的孩子→以歌声救下自己与成景。｜立场关系：石宇的救赎锚。
- **尚华（Sang-hwa）**｜性格：粗犷、重义、护妻。｜装备·能力：顶尖肉搏，棒球棍级清场。｜人物弧光：列车良心→大田后牺牲。｜立场关系：成景丈夫；石宇的「做人」教师。
- **成景（Seong-kyeong）**｜性格：坚韧、怀孕中的镇定。｜装备·能力：辅助与终局驾驶。｜人物弧光：被保护者→带秀安抵达封锁线。｜立场关系：尚华遗志承载者。
- **延锡（Yon-suk）**｜性格：高管惯性、恐惧驱动的冷酷。｜装备·能力：话语煽动、推人挡灾。｜人物弧光：制度性反派→感染后仍乞命。｜立场关系：乘客中的「阶级病毒」。
- **勇国 / 珍熙**｜性格：少年恋爱与义气。｜装备·能力：棒球体能。｜人物弧光：支援小队→东大邱被利用致死。｜立场关系：可结盟的青年火力。
- **仁吉 / 钟吉**｜性格：姐妹情与对自私的厌恶。｜装备·能力：有限。｜人物弧光：仁吉被弃；钟吉开门惩罚车厢。｜立场关系：道德审判的开关。
- **流浪汉**｜性格：创伤沉默、关键时舍命。｜装备·能力：耐力。｜人物弧光：被社会忽视者→开路牺牲。｜立场关系：终局三人组的隐形第四人。
- **基哲（乘务员）等车组**｜性格：服从到崩溃。｜装备·能力：门控与广播。｜人物弧光：系统螺丝钉。｜立场关系：可被延锡裹挟。
- **KTX司机**｜性格：职业责任。｜装备·能力：列车控制。｜人物弧光：换车时殉职。｜立场关系：移动舞台的「神」。

**【势力图谱】**
- **KTX-101 乘客共同体（脆弱）**：封门派 vs 开门派。
- **延锡—恐惧乘客集团**：以「检疫」为名的排他。
- **石宇—尚华—勇国行动组**：穿越尸海的突击轴。
- **韩国军方 / 釜山检疫区**：封锁与误杀并存的终点裁判。
- **石宇所属资本/生化关联企业**：疫情责任阴影（电话揭示）。

**【贵重物品】**
- **棒球棍与即用钝器**：车厢近战标配。
- **车厢门与门闩**：比枪更重要的「城墙」。
- **手机新闻与短信**：外部世界崩坏的唯一窗。
- **石宇公司来电**：罪证与动机双刃剑。
- **新车头/驾驶室钥匙**：东大邱后的生存门票。
- **《Aloha ʻOe》歌声**：终局证明「人类」的符号武器。
- **孕妇身份与未出生的「尹秀允」**：尚华托付的道德契约。

**【隐藏剧情 · 伏笔】**
- 企业利益与疫情源头的间接链条（石宇职业罪感）。
- 大田「军队安全」是最毒的希望营销。
- 钟吉开门：被压迫者对冷漠中产的报复，不是单纯疯癫。
- 黑暗减缓尸动：可被战术利用，也可被恐慌浪费。
- 续作《半岛》属同宇宙四年后，本世界默认电影单集终局：秀安与成景获救，石宇已死。

**【大事记时间线】**
- Day0 晨 → 鹿尸变；石宇接秀安。
- Day0 上午 → 首尔站上车；首名感染者发作。
- Day0 中午前后 → 天安、车厢封门战。
- Day0 下午 → 大田站军人尸潮；尚华牺牲。
- Day0 傍晚 → 钟吉开门；东大邱事故与换车。
- Day0 夜 → 石宇牺牲；秀安成景过封锁。

**【叙事基调 · 雷区】**
基调：窒息、血腥、讽刺、父爱迟到却沉重。  
忌：写成无脑爽文砍尸；忌美化延锡式利己；忌删企业责任暗示；忌让一阶契约者「一人之力清全国」；忌用套话代替真名。最早锚点：首尔站上车前，或大田站玻璃门将破瞬间。

**【世界剧情线·细部展开一：列车社会学】**
KTX车厢本是韩国中产通勤与探亲的正常空间：商务舱逻辑、学生团体、工人家庭、老人姐妹同处一列。尸变把「座位等级」瞬间改写成「谁离门更近、谁能说服别人锁门」。延锡的西装与名牌包在平时是地位，在列车上变成煽动权杖——他不需要最强拳头，只需要最会说「为了大家安全」。石宇的基金经理身份则构成反讽：他习惯用数字处理风险，却无法用数字处理女儿的生日与尸变的伦理。

**【世界剧情线·细部展开二：大田站的希望工程】**
大田站在广播里是「军队与隔离」，在视觉上是空荡月台与玻璃门后的阴影。乘客下车时的松一口气，是全片最残酷的情绪设计。军人丧尸保留部分装备与冲击姿态，使「国家机器已死」具象化。棒球棍敲在头盔与肩甲上的手感，应写成「规则仍在、人已不在」。石宇公司接应队的感染，把私人特权救援也一并埋葬。

**【世界剧情线·细部展开三：尚华的伦理课】**
尚华强迫石宇「做人」：不是鸡汤，是生存技术——没有同盟的车厢会先从内部烂掉。他牺牲时要求石宇记住未出生女儿的名字，把抽象「人性」钉成具体人名。之后石宇的每一次挡门、每一次把秀安推到身后，都是在偿还这堂课。延锡则提供对照实验：同一套恐惧，一种人用来护孕，一种人用来锁门。

**【世界剧情线·细部展开四：东大邱与父职完成】**
东大邱的火与倾覆车厢制造物理绝境，流浪汉的牺牲补上「无名者」的伦理位置。石宇对延锡的最后一战是清算，也是感染的代价。他把驾驶技能与女儿托付给成景，完成从「缺席父亲」到「以死在场」的弧光。封锁线士兵的瞄准与秀安的歌声，把主题从「打丧尸」抬到「如何被承认为人」。

**【人物群像补强】**
除核心外应记住：基哲（乘务）、司机、棒球队教练与队员、被延锡利用的乘客群。电影不做「全员名单式吃鸡」，但每次死亡要有面孔与一句生前特征。秀安的母亲罗英在釜山等待，是线外希望，不宜写成突然空降救援。

**【列车地理与门禁机制】**
每节车厢=可争夺的城池。门缝、连接处、厕所锁、紧急锤是关键道具。尸群会从破碎窗与解开的门涌入。夜晚隧道外的光会吸引尸潮。AI写场景先报：现在第几站之间、门是否锁死、谁站在门边。

**【微观可介入事件库】**
①帮尚华封住一扇将破的门三十秒。②在大田玻璃门前拉住试图「相信军队」的乘客。③偷听延锡煽动并当众拆穿（高社交风险）。④把水和饼干让给流浪汉换情报。⑤护送秀安穿过黑车厢时选择是否救陌生人。⑥东大邱帮司机找到备用车头钥匙。⑦阻止延锡把勇国当诱饵。⑧在封锁线前教秀安如何举手与唱歌。

**【主题与AI禁区】**
主题是灾难中的父职、阶级与「谁配活」。禁止把丧尸写成数值怪；禁止无视感染速度；禁止低阶推翻全国疫情。允许的最大胜利：多活一站、多救一人、把企业责任录音带出车。

**【开场首尔分镜】**
石宇的早晨是电梯、会议、错过的电话。秀安的房间有练习《Aloha ʻOe》的痕迹。车站大屏滚动延误与「局部事故」。伤痕女子爬上车时，无人伸手——冷漠是病毒的预演。

**【武器与非武器】**
消防栓、行李杆、高跟鞋、婴儿车轮、棒球手套与棍。枪极少，因为这是客运列车不是军火库。真正的武器是「决定为谁开门」。

**【大田后队伍拆分状态机】**
AI须能回答：石宇组是否已与秀安重聚、延锡车厢是否锁门、尚华是否仍存活、下一站是哪里。契约者改写局部后，用「更狠的锁门」或「更早的尸潮」回流，不假装原作没发生。

**【感染规则写作要点】**
被咬后倒计时以分钟计。亲人感染时的犹豫=死亡。成景的怀孕使「推搡」变成双重谋杀风险。石宇被咬后的清醒时间窗口用于托付，不可写成无限扛毒。

**【釜山行·场景零件库】**
气味：血、消毒水、尿、烧焦机油。声音：KTX轮轨、尸吼、广播女声、棒球棍击骨。制度：检票、站名广播、企业来电。价格：一张通往釜山的票在末日变得无意义，一条命的价格变成「你让不让门」。

**【同盟与背叛税率】**

关键NPC立场：**石宇**——只护秀安，视你为变数；**秀安**——会求你帮忙；**尚华**——观察你是否「像人」；**延锡**——已开始盘算隔离谁。

主线钩子/支线：钩子=活过首波咬杀并学会封门；支线=把秀安推到安全车厢；支线=录下首名感染者症状；支线=阻止乘客踩踏老人。

危险度/规避：中——主威胁是恐慌与首尸；规避盲目开门、与尸正面缠斗。

任务方向/奖励：建立生存节奏；奖励=钝器、车厢地图半张、临时同盟。

**二阶（封门战争 · 车厢政治）**

切入身份/时点：以「尚华临时帮手 / 棒球队随行」切入列车中段封门与资源分配期。

初始事件：门缝外有人哭着说自己没被咬；门内有人说「不能赌」。你的一票可能决定一车人性命。

开场白建议：「门是唯一的国境。门这边是呼吸，门那边是牙齿。有人用公文包砸门，有人用身体顶门——你站在哪一边？」

关键NPC立场：**尚华 / 成景**——护孕优先；**勇国 / 珍熙**——年轻人的冲动与恋爱；**延锡**——煽动锁门；**石宇**——逐渐被改变。

主线钩子/支线：钩子=守住或攻下一扇门；支线=分配水和食物；支线=安抚秀安；支线=收集外部新闻短信。

危险度/规避：中高；规避被认定为「可能感染者」。

任务方向/奖励：话语权与同盟位；奖励=更好的钝器、门闩技巧、尚华的认可。

**三阶（站台与尸潮 · 天安至大田前）**

切入身份/时点：以「下车侦察兵 / 失散重聚者」切入站台短停与尸潮初现。

初始事件：月台灯光下有人穿制服却用四肢奔跑。你的队伍要决定：冲进站房找药，还是立刻退回列车。

开场白建议：「站名广播还在礼貌地报站，月台上的影子却已不礼貌。你听见军靴，却看不见还是人的士兵。」

关键NPC立场：**石宇小队**——目标釜山；**车组**——执行时刻表惯性；**外部感染者**——环境杀。

主线钩子/支线：钩子=安全返回列车；支线=夺取灭火器/斧；支线=救出一名站台幸存者（可能已咬）；支线=破坏吸引尸群的声源。

危险度/规避：高；尸潮数量碾压。

任务方向/奖励：战场感知；奖励=站台平面图、对讲残机、药品。

**四阶（大田站 · 军人丧尸锚点）**

切入身份/时点：以「石宇—尚华—勇国突击编外 / 大田本地撤离失败的公务员」切入大田站玻璃门危机。

初始事件：广播说军队在此；玻璃后先出现的是手印，然后是集体撞击。棒球棍不够用，你要选：断后、护秀安、或去启动某扇卷帘。

开场白建议：「他们本该开枪保护你们。现在他们用牙。玻璃裂成蛛网时，你看见头盔下的白眼——国家在这一秒对你投降。」

关键NPC立场：**尚华**——肉盾与伦理；**石宇**——护女；**勇国**——少年火力；**军人丧尸潮**——环境BOSS；**成景 / 秀安**——必须转移的核心。

主线钩子/支线：钩子=卡片锚点大田突围；支线=为尚华争取十秒关门；支线=夺回被冲散的秀安；支线=记录「接应队已感染」证据。

危险度/规避：极高；不可恋战清场幻想。

任务方向/奖励：中阶质变体验；奖励=军用匕首/盾残片、队伍核心信任、创伤。

**五阶（弃人政治 · 东大邱换车）**

切入身份/时点：以「被锁门外的『可疑感染者』 / 延锡车厢卧底 / 换车技术员」切入钟吉开门前后与东大邱事故。

初始事件：延锡说你该去前庭隔离；钟吉的手按在门把上。另一条线：轨道堵塞，火光中必须找到下一列车头。

开场白建议：「他们用『安全』两个字把你关出去。火在站台烧，人在车厢里投票决定谁不是人。你要撬开的不是门，是一套中产的良心会计学。」

关键NPC立场：**延锡 / 基哲**——排他；**钟吉**——报复式开门；**石宇 / 成景 / 秀安**——重组核心；**勇国 / 珍熙**——将被利用；**司机**——换车关键。

主线钩子/支线：钩子=在弃人逻辑下活到换车；支线=阻止或加速钟吉开门（道德分支）；支线=保护珍熙不被当诱饵；支线=帮司机接通备用轨道。

危险度/规避：极高；社交杀+尸杀+火灾。

任务方向/奖励：破局入场券；奖励=车头权限、延锡罪证、尚华遗言的传递权。

**六阶（封锁线 · 国家与挽歌）**

切入身份/时点：以「驾驶室共犯 / 釜山外围感染区向导 / 叛逃卫生官员」切入石宇牺牲前后、秀安成景走向封锁线。

初始事件：石宇把血手按在玻璃上道别；探照灯扫来，枪口抬起。你要让士兵听见歌声而不是开枪，或用自己的身份换两张通行证。

开场白建议：「列车停了，国家的轨道才刚开始。女儿唱歌，孕妇扶着方向。你站在铁丝网前，想起车上所有被锁在门外的脸——现在，轮到国家决定锁不锁你。」

关键NPC立场：**秀安 / 成景**——终点依赖；**釜山军方**——误杀与救援双面；**石宇（亡）**——道德压力；可能的企业灭声人员——要录音带。

主线钩子/支线：钩子=护送二人过检；支线=公开企业来电；支线=假扮感染测试；支线=寻找罗英（高不确定性）。

危险度/规避：极高·贴近顶点——国家封锁机器；条件性胜利=获救与留下罪证，不求清国。

任务方向/奖励：活着离开故事；奖励=临时身份、罪证拷贝、永远的列车记忆。
