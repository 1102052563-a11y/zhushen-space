# 剑豪生死斗
<!--meta lib=主库 tiers=一、二、三、四-->

## 剧情

**【作品来源】**
《剑豪生死斗》（日文シグルイ，英文Shigurui / Shigurui: Death Frenzy）为山口贵由作画的青年向时代剧漫画，原案取自南条范夫小说《骏河城御前比武》首章；2003–2010年连载于秋田书店《Champion Red》等，全15卷。2007年由MADHOUSE改编12集电视动画《シグルイ》，监督浜崎博嗣。本档案以**漫画主线+动画已改编范围（虎眼流恩仇与御前比武前史）**为准。文风：血腥残酷、权力与肉体美学、武士道被撕碎的写实。卡片锚点：**虎眼流道场复仇——藤木源之助与伊良子清玄的恩怨，独臂复仇者使用「无明逆流」斩杀道场高手，血腥与美学的剑术死斗**。

**【世界定位】**
江户时代初期，宽永前后（约1629年）德川忠长领内。故事以骏河城「真剑比武」为现在时框架，主体回溯虎眼流道场内：岩本虎眼的暴政、藤木源之助的忠与忍、伊良子清玄的野心与残废、岩本三重的工具化命运。一句话：剑不是武士的魂，是领主取乐与道场吞人的牙。

**【世界观 · 力量体系】**
无超凡灵力。战力=**流派剑术 + 身体改造/残缺后的补偿技术 + 意志与忍从 + 领主权力杀**。真剑对决、斩指、剜目、断肢皆可发生。虎眼流以残酷修行与「流星」等秘剑立威。死亡即死；残废者可凭技法重返顶点，但代价写在肉上。

威胁/战力层级（宁低勿高）：

1. **普通藩士 / 道场末席**  
   凡人剑术。乐园对照≈一阶。

2. **虎眼流正式门人 / 中坚剑客**  
   可斩杀浪人小队。乐园对照≈一～二阶。

3. **牛股权左卫门级巨汉剑士 / 道场干部**  
   怪力木剑与实战压迫。乐园对照≈二阶。

4. **藤木源之助（独臂）/ 伊良子清玄（盲）巅峰技**  
   以残躯使出「无明逆流」等，可屠道场高手。乐园对照≈三阶。

5. **岩本虎眼（清醒期）与领主德川忠长的制度暴力**  
   个人剑技恐怖+可发动真剑御前比武的生杀权。乐园对照≈三～四阶（体系侧顶点）。

乐园阶位映射：末席门人≈一阶；正式剑士与复仇初步≈二阶；独臂藤木/盲眼伊良子死斗≈三阶；虎眼+忠长真剑比武体制≈四阶。世界顶点=**领主以人命为赌的权力与虎眼流吃人结构**，条件性胜利=活过一场比武/完成个人复仇/救出一人，严禁低阶斩杀「制度本身」。

**【地理 · 舞台】**
- **虎眼流道场**：修行、凌辱、继承权斗争的核心牢笼。
- **岩本家屋敷**：三重与女中郁的日常地狱。
- **街道、桥、刑场与旅舍**：浪人伊良子的来路。
- **骏河城御前比武场**：现在时舞台，真剑、血、观礼的藩士。
- **德川忠长的权力空间**：比武的发起者与规则制定者。

**【世界剧情线】**

○ **现在时：真剑比武**  
1629年，忠长不顾家臣反对，举办以真刀代替木刀的御前比武。第一场：独臂的藤木源之助 vs 盲眼的伊良子清玄。观众与家臣战栗——故事由此回溯二人如何被同一道场研磨成「死狂」。

○ **虎眼的王国**  
岩本虎眼以残酷与秘剑统治道场，视女儿三重为繁衍工具，宠妾郁如牲口。门人以忍耐为忠，以告密为生存。牛股等干部维持暴力秩序。

○ **伊良子入门**  
出身卑微、野心勃勃的伊良子清玄进入虎眼流，剑才惊人，却缺乏藤木对「流」的绝对忠诚。他与藤木竞争继承与三重的婚约；虎眼以双重标准玩弄二人。

○ **凌辱、残废与出奔**  
道场内发生性暴力、斩指、刺目等事件（须严肃不猎奇）。伊良子与虎眼决裂，遭残酷惩罚，双眼与足受伤，被逐或逃亡。藤木在忠诚与良知间撕裂，最终亦在冲突中失去一臂，身负「无明逆流」等禁技或极致体术补偿。

○ **复仇与道场清算（卡片锚点）**  
独臂藤木回归/突入，以残躯剑术斩杀道场高手，血洗虎眼流的象征秩序；伊良子则以盲剑走自己的复仇与求道之路。两人的恨重叠又相反：一个为「忠被吃尽」，一个为「野心被碾碎」。

○ **回到比武场**  
回溯合龙：两人在忠长面前以真剑相见。胜负是领主的余兴，也是二人全部人生的压缩。动画止于改编范围；漫画更延展残酷因果。任务世界以「比武+道场恩仇」为可玩闭环。

○ **蝴蝶效应边界**  
可改：某次稽古胜负、谁先失去肢体、三重的命运岔路。不可无代价改：真剑比武的权力本质、虎眼流吃人结构、残躯剑术的代价逻辑。

**【主要人物】**
- **藤木源之助**｜性格：沉默、忠诚到近乎自我消灭。｜装备·能力：虎眼流正统，独臂后以背肌与极致技术补偿，卡片技法「无明逆流」。｜人物弧光：明星弟子→残废复仇者→比武场上的刀。｜立场关系：虎眼的刀；伊良子的宿敌；三重的情愫对象。
- **伊良子清玄**｜性格：野心、算计、求上升。｜装备·能力：天才剑质，盲后仍具致死技术。｜人物弧光：入门挑战者→被罚致残→比武另一极。｜立场关系：藤木镜像；虎眼的弃子。
- **岩本虎眼**｜性格：暴君、痴呆与清明交替，残酷。｜装备·能力：流星等秘剑、道场生杀权。｜人物弧光：绝对师傅→被复仇阴影笼罩的旧神。｜立场关系：体系顶点之一。
- **岩本三重**｜性格：被压抑的意志与恐惧。｜装备·能力：无战力，政治联姻棋子。｜人物弧光：工具女儿→两人争夺的象征。｜立场关系：藤木/伊良子婚姻线核心。
- **女中郁**｜性格：长期受虐后的复杂生存。｜装备·能力：有限。｜人物弧光：虎眼暴行的见证与受害者。｜立场关系：道场丑闻的活档案。
- **牛股权左卫门**｜性格：粗豪干部。｜装备·能力：巨木剑、怪力。｜人物弧光：秩序打手→可被斩于复仇夜。｜立场关系：藤木前辈。
- **德川忠长**｜性格：任性、以血为赌。｜装备·能力：领主权力。｜人物弧光：比武发起者。｜立场关系：世界规则杀。
- **其他门人与藩士**｜性格：各异的忍从与窥伺。｜装备·能力：流派剑。｜人物弧光：血的背景板与告密网。

**【势力图谱】**
- **虎眼流**：暴力宗门。
- **藤木路线**：忠与复仇。
- **伊良子路线**：个人野心与残后求道。
- **德川忠长政权**：真剑比武的舞台主。
- **观礼藩士与家臣**：道德反对但无力阻止。

**【贵重物品】**
- **真刀（御前比武用）**：取代木刀的死亡契约。
- **虎眼流秘剑「流星」等**：恐惧品牌。
- **无明逆流（藤木）**：独臂杀阵的技术核。
- **婚约与继承文书**：三重与道场未来的纸刃。
- **伤残的身体本身**：独臂、盲眼——既是弱点也是标志。

**【隐藏剧情 · 伏笔】**
- 忠长比武是政治娱乐，不是武士荣誉。
- 虎眼的痴呆清醒周期影响谁活谁死。
- 藤木的忠与伊良子的欲是同一结构的两面产品。
- 性暴力与肢体刑罚是权力语言，写作须克制不展览。

**【大事记时间线】**
- 入门前 → 伊良子浪人期。
- 道场岁月 → 竞争、凌辱、惩罚。
- 残废节点 → 盲与断臂定型。
- 复仇夜 → 道场血洗（锚点）。
- 1629 → 骏河城真剑第一战。

**【叙事基调 · 雷区】**
基调：冷、脏、美、恨。  
忌：热血热血的无脑剑戟；忌美化凌虐；忌把残废写成酷炫无代价；忌低阶秒杀忠长政权。最早锚点：比武场击鼓，或道场夜雨。

**【世界剧情线·细部展开一：忠长的赌局】**
家臣劝谏「真剑会死人」，忠长要的正是死人。比武场铺沙、设座、置刀，仪式越完整，荒诞越强。藤木与伊良子被介绍残缺时，观众的惊叹是消费。契约者若以藩士身份劝停，会碰壁于领主任性。

**【世界剧情线·细部展开二：道场日常恐怖】**
稽古不是青春，是筛选。虎眼可因情绪斩人。门人学习的不是剑，是「如何在暴君身边多活一天」。告密与连坐使同盟极难。三重的婚约是奖品也是锁链。

**【世界剧情线·细部展开三：双雄语法】**
藤木少言，伊良子多算。一个把自我献给「流」，一个想让「流」献给自我。虎眼同时摧毁两者：把忠变成愚，把欲变成刑。独臂与盲是结果，不是标签玩梗。

**【世界剧情线·细部展开四：复仇美学】**
无明逆流的描写应重「身体如何代偿」与「斩击的政治含义」：砍的是干部，震的是虎眼的神性。血是墨，道场是纸。伊良子的盲剑则是另一种听觉与间合的地狱特训成果。

**【人物群像补强】**
牛股的大剑、郁的沉默、门人的耳语、比武场医生与介错人，都是零件。三重不是花瓶，是结构受害者，给她能动性时勿写成无代价逆袭。

**【剑术与身体细则】**
间合、踏足、木剑反震、断臂后的重心、盲术后的听风。战斗后写感染与发热。禁止无限连斩不喘气。

**【微观可介入事件库】**
①替三重传一封不该存在的信。②在稽古中故意输掉以保某人手指。③教独臂藤木一种杠杆借力（有限）。④为伊良子引路逃离追捕。⑤在比武场做介错预备。⑥偷出虎眼秘剑图谱半页。⑦当众记录忠长「以死为乐」的发言。⑧阻止一场对女中的私刑（极高风险）。

**【主题与AI禁区】**
主题是权力如何生产「死狂」。禁止把故事写成爽快升级；禁止轻佻处理性暴力；禁止忽略残躯代价。

**【开场比武分镜】**
鼓响，砂扬，刀出鞘的金属味。藤木断袖露疤，伊良子白眼无神。观礼席有人呕吐，忠长微笑。

**【道场夜】**
雨打瓦，鞭与木剑声。郁数鱼的声音。三重捂耳。藤木在廊下站一夜，像一柄还未出的鞘。

**【补完块·名场面语法】**
写斩击先写呼吸与脚掌起泡。写权力先写谁跪着说话。写爱情先写不能碰的指尖距离。

**【补完块·御前比武规则】**
真刀、无护具、败者可能死，胜者成为领主的谈资。医疗简陋。赌注是面子与领地娱乐。

**【补完块·虎眼流内部经济】**
束脩、献金、联姻、把弟子当打手出租。剑名是广告，血是成本。

**【补完块·时间跳切】**
现在时比武与过去道场交叉。AI须标明「比武当日」或「三年前稽古」避免混线。

**【补完块·声音设计】**
盲剑依赖：布袜摩擦、观众屏息、刀风。独臂剑依赖：肩背爆筋的呼吸。

**【补完块·终局开口】**
无论谁胜，忠长都赢。契约者最大胜利可以是「让二人 simultaneous 拒剑」——但会触发领主怒杀，条件性极难。

**【补完块·更多道场切片】**
新人入门要看前辈被打掉牙仍道谢；夜巡弟子互相踩脚争宠；虎眼清明日忽然讲剑理，门人如听神谕——第二天他又可因一碟菜杀人。这种无常性是恐惧引擎。

**【补完块·伊良子的阶级】**
他知道自己不是「干净出身」，所以把剑当梯子。梯子抽走后，恨比死更长。藤木则出身「可被承认为弟子」的轨道，却用忠诚把自己锯成独臂。

**【补完块·三重的房间】**
针线、嫁妆清单、窗外稽古的喝声。她的反抗可能只是藏起一把小刀或拒绝梳头——小，但真实。

**【补完块·比武场外围】**
赌徒、货摊、藩士家属、抬尸体的足轻。血会渗进经济。

**【剑豪生死斗·场景零件库】**
铁锈、榻榻米血臭、香、粪、雨。禁止写成清爽武士剧。人物开口带身份：门人、女中、领主、浪人。

**【同盟税率】**
道场内盟约默认被虎眼撕毁。比武场上无人是队友。

**【身体档案】**
写清谁缺哪只眼/哪只手、伤疤新旧、能否奔跑。避免前后矛盾。

**【权力回流】**
契约者若当众羞辱虎眼，次日可能是连坐斩。制度反噬优先于私人决斗。


**【虎眼流吃人结构长卷】**
虎眼流对外是名门，对内是肉食机器。束脩银两养活暴君的酒与妾，弟子的血养活「流星」的传说。继承权不是德，是谁更能忍受虎眼的痰与鞭。藤木把忍受当成忠，伊良子把忍受当成投资；虎眼把两者都当成玩具。牛股的巨木剑是秩序的广告牌：你连举都举不起，就别谈真理。

道场的一日：鸡鸣稽古、午间告密、夜雨私刑。三重的针线声与木剑声对位。郁在厨房处理鱼，鱼眼像门人的眼。契约者若以末席身份活过七日，学到的第一课不是剑，是「何时该装死」。

**【无明逆流与身体代偿】**
独臂不是酷，是重心永久偏移。藤木用腰背与残肩重新发明砍击，步法更短、更狠、更接近自杀式突进。「无明逆流」之名暗示盲与逆——对伊良子是嘲，对藤木是自况。斩干部时，血溅在昔日跪拜的地板上，忠诚以物理方式破产。

盲剑的伊良子把世界缩成声音与风。布袜、呼吸、刀鞘轻碰，都是坐标。他的恨更干净：不忠于流，只忠于自己的上升欲被碾碎的事实。两人在比武场相遇，是两种破产的股东大会。

**【德川忠长的娱乐政治】**
真剑比武的谏言被当成扫兴。沙地、鼓、观礼席、介错人，仪式越满，人命越轻。忠长要看的是「残缺者是否仍像人一样死」。家臣的反对是背景合唱，改变不了鼓点。七阶不在此世界；四阶已是领主生杀的天花板。契约者劝停比武，应碰壁，再找「救一人」的缝。

**【三重与郁的女性战场】**
三重被父亲视为失败的性别，婚约是补救。她可能的能动：藏刀、传信、拒绝在众人前被检视身体。郁的生存策略是顺从与记忆——她记得谁被虎眼叫去夜里没回来。写她们禁止色情化暴力，只写权力。

**【复仇夜分镜加厚】**
雨、犬吠、门栓断裂。独臂踏入，干部以为人多。第一斩断腕，第二斩断号令，第三斩断「虎眼不败」的集体幻觉。有人逃去报官，官在领主比武逻辑下可能变成「新的观众」。伊良子或许在另一条巷同步杀人，两条血河在比武场汇合。

**【门人生态志】**
告密换药；联姻换安稳；把师弟当挡箭牌换晋级。藤木年轻时也参与过沉默，这是他后来独臂的道德利息。AI写群戏用原社会关系解释谁先跪谁先砍。

**【比武场外围经济】**
赌徒押独臂、押盲者；货摊卖假药；抬尸足轻按具数收钱。血渗进铜钱。记录官的笔比刀更危险——写下的字会进藩的档案。

**【剑理与间合课】**
间合三寸是生，一寸是死。木剑反震伤骨，真剑无声。断臂者攻击线路可预测却更快。盲者诱敌出声。禁止无限连斩不写呼吸。

危险度/规避：中高；告密网。

任务方向/奖励：情报位；奖励=伤药、派系徽章、三重的信任碎片。

**三阶（残躯 · 无明逆流之夜）**

切入身份/时点：以「复仇同行者 / 道场叛逃向导」切入藤木独臂复仇或伊良子盲剑杀阵（卡片锚点）。

初始事件：夜袭。独臂男人踏入旧庭，干部的木剑与真刀一起折断。你要帮他断后，或趁乱救三重。

开场白建议：「一只手足够写下血书。他的袖管空着，刀却满。道场的灯一盏盏灭，像虎眼的眼皮。」

关键NPC立场：**藤木源之助**——复仇主轴；**伊良子**——平行杀意；**牛股等干部**——被斩对象；**虎眼**——可能现身或躲在闇里。

主线钩子/支线：钩子=见证/参与道场清算；支线=保护非战斗人员；支线=夺取流星秘传残页；支线=为比武留下挑战状。

危险度/规避：高；真剑无赦。

任务方向/奖励：中阶峰值；奖励=刀、仇名、被领主耳目盯上。

**四阶（骏河城 · 真剑御前）**

切入身份/时点：以「比武介错候补 / 忠长侧近记录官 / 藤木或伊良子的最后助手」切入1629真剑第一战。

初始事件：鼓响。独臂与盲者相对。忠长问你：要不要加注「败者斩首」。砂地上已有人呕吐。

开场白建议：「领主要看血，不是看理。两个人用尽一生走到沙地中央，而你的笔要写下谁先倒下——或者你自己的刀也出鞘。」

关键NPC立场：**藤木 / 伊良子**——死斗双方；**德川忠长**——规则之神；**家臣**——无力的反对；**观众**——恐惧与兴奋。

主线钩子/支线：钩子=影响比武条件或救下败者一口气；支线=当众朗读反比武谏文；支线=护送三重离开观礼席；支线=刺探忠长下一场名单。

危险度/规避：极高·贴近顶点——领主生杀；条件性胜利=活着离开城、保留两人之一的命、留下罪证。

任务方向/奖励：终局入场；奖励=比武记录、领主忌器情报、残躯剑术的目击传承（非无敌）。


**一阶补充钩子**
夜间搬尸换药；替三重藏针；在牛股面前故意摔跤装蠢；偷听虎眼清明期的一句剑理。

**二阶补充钩子**
伪造告密信保护同门；在婚约宣布时制造混乱；带郁见地下医者；记录门人被斩名单。

**三阶补充钩子**
为无明逆流之夜点燃引路火把；挡住报官的门人；护送三重出侧门；与伊良子短暂停火换情报。

**四阶补充钩子**
介错时「刀下留人」；篡改记录官笔录；在鼓响前调换木刀/真刀（极难且有后果）；护送败者出城。
