# 特种部队

## 剧情

**【作品来源】**
《特种部队》（此处对应世界库条目「特种部队」，卡片与剧情锚点为**纳米虫巴黎吞噬**）取自2009年好莱坞军事科幻动作片《特种部队：眼镜蛇的崛起》（G.I. Joe: The Rise of Cobra）关键高潮段落：M.A.R.S.纳米机器人武器在巴黎起爆，绿色雾气吞噬埃菲尔铁塔与街区。导演斯蒂芬·索莫斯，主演查宁·塔图姆、马龙·韦安斯、休·杰克曼未参演本品、丹尼斯·奎德、西耶娜·米勒、李秉宪、约瑟夫·高登-莱维特等。本档案**只覆盖七阶单一切入**（清单tiers仅七），聚焦纳米虫城市级灾难与关闭窗口，不展开一至六阶成长线（同宇宙完整线见条目「特种部队：眼镜蛇的崛起」）。文风：大片军事科幻、纳米恐慌、倒计时拆弹。

**【世界定位】**
近未来国际军工与黑科技并行的世界。詹姆斯·麦卡伦（毁灭者）的M.A.R.S.出售可吞噬无机物的纳米虫弹头；G.I. Joe跨国特种部队试图阻止其被用于「新世界秩序」。卡片时刻：巴黎——弹头命中铁塔，绿色纳米雾蔓延，若无法关闭将吞噬全城。一句话：七阶开局即终局，城市在雾里溶解，你只有关闭窗口。

**【世界观 · 力量体系】**
高科幻军事：加速服、脉冲武器、纳米控制、忍者级近战（蛇眼/暴影）。本档战力阅读以**城市级纳米虫灾**为尺：个人武力再强，也须服务「关闭弹头/控制中枢」。无神话复活。

威胁/战力层级（本世界仅开放七阶叙事）：

1. **巴黎纳米虫雾（卡片锚点）**  
   绿色雾气吞噬金属与街区建筑，扩散速度以街区/分钟计。乐园对照≈七阶环境灾难。

2. **M.A.R.S./眼镜蛇指挥与纳米控制权**  
   关闭密钥、北极基地链路、医生/麦卡伦的控制。乐园对照≈七阶敌对体系。

3. **G.I. Joe精锐介入**  
   杜克、里普考德、蛇眼等具备拦截与近战关闭能力，但在雾中仍受时间压迫。乐园对照≈七阶友军标尺（非低阶可扮演成长）。

乐园阶位映射：本世界**仅七阶**——巴黎纳米虫吞噬与关闭作战。世界顶点=**失控纳米虫城市湮灭+麦卡伦/医生的军工神权**，条件性胜利=关闭弹头/切断扩散，严禁写成「雾自己散了」或低阶无关切入。

**【地理 · 舞台】**
- **巴黎街区与埃菲尔铁塔**：雾的第一现场。
- **塞纳河、地铁、警用封锁线**：疏散与死亡带。
- **M.A.R.S.远程链路 / 北极基地方向**（关闭指令可能来源）。
- **G.I. Joe机动单位与空中支援走廊**。
- **已被腐蚀的金属废墟**：移动的地图减法。

**【世界剧情线】**

○ **前史压缩（七阶必读）**  
麦卡伦研发纳米虫武器，出售北约；杜克与里普考德护送中遭男爵夫人等劫弹头。G.I. Joe介入。弹头被武器化后，一枚在巴黎起爆演示/攻击——铁塔溶解，全球直播恐慌。

○ **卡片现在时：雾**  
绿色纳米雾从爆点呈花瓣状扩散，吞噬街灯、车辆、建筑骨架。未关闭则巴黎归零，并可能被复制到北京、莫斯科、华盛顿等预设目标（电影中另有多弹头线；本档以巴黎关闭为胜负手）。

○ **关闭窗口**  
需要物理接近未完全释放的控制节点/使用特制杀死开关（kill switch）/切断麦卡伦遥控。杜克曾在叙事中触及关闭；契约者七阶任务即站在同一倒计时上。暴影、男爵夫人等敌对忍者/特工在雾缘阻击。

○ **政治后果**  
即使关闭，铁塔已毁，国际秩序震动，眼镜蛇/医生线仍在北极与白宫冒名等更大棋盘（点到为止，本档不强制打完所有续线）。

○ **蝴蝶效应边界**  
可改：关闭者是谁、牺牲哪条街换时间。不可无代价改：纳米虫吞噬无机物的规则、七阶难度、城市级后果。

- **康拉德·豪瑟 / 杜克**｜性格：领袖责任感。｜装备·能力：战术与加速装具。｜人物弧光：护送者→巴黎关闭关键。｜立场关系：乔队核心。
- **华莱士·威姆斯 / 里普考德**｜性格：飞行员幽默外壳。｜装备·能力：飞行与战场机动。｜人物弧光：巴黎空中/地面支援。｜立场关系：杜克搭档。
- **蛇眼**｜性格：沉默忍者。｜装备·能力：近战顶点。｜人物弧光：雾缘斩敌开路。｜立场关系：暴影宿敌。
- **暴影**｜性格：冷酷。｜装备·能力：忍术与刺杀。｜人物弧光：阻止关闭。｜立场关系：眼镜蛇侧。
- **男爵夫人（安娜）**｜性格：被纳米控制的复杂。｜装备·能力：枪与间谍术。｜人物弧光：敌我之间。｜立场关系：杜克前史。
- **詹姆斯·麦卡伦 / 毁灭者**｜性格：军工野心。｜装备·能力：纳米主控。｜人物弧光：灾难源。｜立场关系：M.A.R.S.。
- **雷克斯 / 医生 / 眼镜蛇指挥官**｜性格：面具下的新秩序。｜装备·能力：纳米与意识形态。｜人物弧光：更大反派。｜立场关系：幕后。
- **霍克将军等乔指挥**｜性格：全球授权。｜装备·能力：指挥链。｜人物弧光：授权七阶行动。

**【隐藏剧情 · 伏笔】**
- 巴黎是展示也是威胁样本。
- 纳米可被用于控制人体（男爵夫人线）。
- 关闭巴黎≠结束眼镜蛇。

**【大事记时间线】**
- 弹头被劫与武器化 → 巴黎起爆 → 关闭窗口（本档主战场）→ 国际余波。

**【叙事基调 · 雷区】**
基调：倒计时、金属尖叫、大片速度感。  
忌：写成低阶巷战升级；忌纳米雾无害；忌忽略七阶唯一性硬凑低阶。最早锚点：铁塔被染绿的第一秒。

**【世界剧情线·细部展开一：雾的物理】**
纳米虫优先金属与玻璃，混凝土较慢；人若被「机械义肢/枪」拖入可能截肢式溶解。绿色是视觉警报。风会改变扩散瓣。

**【世界剧情线·细部展开二：关闭工程】**
需要：接近、身份验证、双重确认、可能有人断后挡暴影。杜克式英雄主义与团队工程并行。

**【世界剧情线·细部展开三：巴黎社会学】**
游客奔跑、警察设无效封锁、直播主播哭腔。乔队在镜头前是拯救者也是机密泄露风险。

**【世界剧情线·细部展开四：战后】**
铁塔没了，明信片经济崩了一角；军工听证会开始；医生在暗处微笑。

**【微观可介入事件库】**
①在雾缘救出一车游客换情报。②为蛇眼挡住暴影十秒。③黑客干扰麦卡伦遥控半拍。④用电磁脉冲尝试局部冻结雾（有限）。⑤保护杀死开关运送。⑥直播反向舆论争取北约火力支援。⑦进入已溶解的地铁找次级节点。⑧决定是否公开M.A.R.S.客户名单。

**【主题与AI禁区】**
主题是军工失控与英雄窗口。禁止把七阶当观光；禁止雾自动失效。

**【开场分镜】**
埃菲尔铁塔的铆钉尖叫着变成粉。绿色像反的春天。有人喊“Fermez！”——关的不是门，是世界。

**【补完块·装备在雾中】**
枪被吃、刀被吃、车轮被吃。近战窗口极短。非金属武器临时更值钱。

**【补完块·加速服】**
提供速度换时间，不提供无敌。过热与维护是七阶代价。

**【补完块·忍者战】**
蛇眼与暴影在溶解的钢梁上跳，每一步都可能少一块落脚点。

**【特种部队·场景零件库】**
警报法文广播、纳米沙沙声、直升机螺旋桨被腐蚀的尖叫、塞纳河上的浮尸金属。

**【同盟税率】**
现场盟军默认听霍克与杜克。叛变可能来自被纳米控制者。

**【时间状态机】**
扩散半径？杀死开关距离？暴影是否已交手？全球其它弹头状态？

**【补完块·媒体】**
直播会把契约者的面罩拍进去——七阶还有舆论战。

**【补完块·北约与客户】**
买弹头的人正在后悔；有人已在黑市加价买「关闭技术」。

**【补完块·医生的电话】**
雾中可能响起合成音：「好看吗？」——意识形态恐怖。

**【补完块·终局语气】**
雾退时巴黎缺了一截天际线。胜利像战损报告。契约者奖励是「城市还在地图上」与「一份被列为最高机密的关闭日志」。

**【补完块·与「眼镜蛇的崛起」条目分工】**
完整一至六阶成长、埃及坑基地、北极终局等见「特种部队：眼镜蛇的崛起」。本档是七阶灾难切片，禁止注水写低阶日常。

**【补完块·更多巴黎街景】**
咖啡座溶解成泥，警察盾牌变蜂窝，恋人在桥上被冲散——写灾难要具体到店名式细节（可用化名）。乔队通讯代号与真实姓名对照表要稳定。

**【补完块·关闭后的身体】**
吸入纳米尘的咳嗽、金属牙酸、加速服勒出的瘀青。七阶胜利后仍要医疗。

**【补完块·麦卡伦的哲学】**
他相信恐惧统一世界。关闭雾是打他的演示，不是打他的思想——思想在医生那里续命。


**【七阶唯一性说明（写作铁律）】**
本世界清单仅七阶。禁止写一至六阶日常训练、禁止写「从新兵打起」。开场即巴黎纳米虫灾，契约者默认已具备与乔队并肩的七阶资格（乐园投放/紧急编入）。同宇宙低阶成长请转「特种部队：眼镜蛇的崛起」。

**【纳米虫扩散模型】**
爆点：铁塔与邻近金属结构。扩散：瓣状，风速修正。优先：钢、铝、枪械、车辆。较慢：石材、人体（但义肢/武器会拖人入雾）。视觉：绿。听觉：金属砂化尖啸。关闭前每分钟损失可量化街区。

**【杀死开关工程】**
开关不在雾心最深处，而在「仍可接近的半腐蚀带」或护送箱。需要双人确认、可能生物识别、可能麦卡伦遥控对抗。杜克式冲刺与蛇眼斩敌开路是标准解；契约者可替换为黑客半拍或电磁局部冻结（有限、有冷却、有失败爆炸风险）。

**【战场单位】**
乔：杜克、里普考德、蛇眼、疤片/重火力等（按电影编制取用真名）。敌：暴影、男爵夫人、M.A.R.S.遥控、雾本身。民：警察、游客、记者。媒体直播是第四战场。

**【加速服与装备腐蚀】**
加速换时间，不换无敌。枪在雾中寿命以秒计。临时非金属武器（陶瓷、木、复合材料）更值钱。通讯频道会被腐蚀天线切断，需预埋中继。

**【巴黎街景具体化】**
咖啡座溶泥、警察盾蜂窝化、桥上冲散的恋人、卢浮宫方向的无效封锁、塞纳河上漂浮的金属渣。写法要具体，避免「城市很乱」空句。

**【关闭后政治】**
天际线缺口、听证会、黑市关闭技术、医生电话里的「好看吗」。胜利=城市还在地图上，不是烟花。

**【对话声口】**
杜克命令句；里普考德缓解紧张的玩笑；霍克频道权威；暴影少言；麦卡伦产品发布腔；法警法语口令与英语夹杂。

**【失败与胜利条件】**
失败：扩散超阈值、开关被毁、主队团灭。胜利：扩散停止、核心人口带撤离、日志保全。支线成功不替代主线关闭。

**【与媒体】**
面罩被拍到=身份风险。可利用直播争取北约火力，也可被敌利用制造「乔队实验武器」谣言。

**【忍者战在溶解钢梁】**
落脚点每秒减少。蛇眼与暴影的对决是关闭窗口的倒计时器，不是单纯打戏。契约者断后十秒可改结局。

**【七阶心理】**
开局即终局，没有「先刷经验」。恐惧合法，慌乱会吃掉加速窗口。医疗在胜利后仍然必要：纳米尘咳嗽、瘀青、临时失聪。

**【可介入事件加厚】**
护送开关跨过已溶的路口；用广告屏做法文疏散引导；电磁脉冲赌局部冻结；反向追踪麦卡伦；决定是否公开客户名单；进入地铁次级节点；与记者交换镜头遮罩换直升机。

**【终章语气再锁】**
雾退，塔不在。有人欢呼，有人拍空荡的天空。契约者的奖励清单里没有神器，只有密级日志与「还呼吸的巴黎人」。
