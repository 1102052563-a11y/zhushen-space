# 忍者神龟
<!--meta lib=主库 tiers=一、二、三-->

## 剧情

**【作品来源】**
《忍者神龟》（Teenage Mutant Ninja Turtles，缩写 TMNT；亦称 Teenage Mutant Hero Turtles 于部分欧陆译名）由凯文·伊斯特曼（Kevin Eastman）与彼得·莱尔德（Peter Laird）于 1983 年草创、1984 年 5 月由幻影工作室（Mirage Studios）自费发行黑白漫画起家，原为对《夜魔侠》《新变种人》《浪人》《蚁熊》等超级英雄与独立漫画潮流的戏仿。角色与设定迅速扩张为跨媒介特许经营：1987–1996 动画、1990–1993 真人电影三部曲、2003–2009 动画、2012–2017 CGI 动画、2007 CGI 电影《TMNT》、2014–2016 贝氏真人重启、2018《风云再起》、2023《变种大乱斗》及 IDW 漫画等。2009 年莱尔德将特许权售予维亚康姆旗下尼克儿童频道（今属派拉蒙）。本档案以**纽约下水道忍者·家族羁绊·脚族宿仇·变异泥沼**为共同基底，综合 Mirage 早期正典、1990 电影与主流动画/影视名场面，供契约者切入「都市暗战·青少年英雄」任务世界；各连续统差异在隐藏剧情中标注。文风：热血、兄弟斗嘴、忍道与披萨并置；可偏黑暗（Mirage/2003）或轻喜剧（1987），忌写成纯儿童无害或纯写实黑帮。

**【世界定位】**
当代（或近当代）纽约市曼哈顿：下水道与屋顶之间藏着四只被放射性/变异泥沼（Mutagen / Ooze）改变的青少年拟人龟，以及他们的师父——变异鼠斯普林特（Splinter）。他们以忍术隐于人群，对抗脚族（Foot Clan）与其领袖许瑞德（Shredder，本名小禄崎 Oroku Saki）、街头黑帮、其他变异体乃至外星势力（克朗 Krang / 乌特罗姆 Utrom / 克朗族 Kraang 等版本）。一句话：半壳英雄在暗处守护城市，家族即道场，宿敌即镜中影。

**【世界观 · 力量体系】**
本世界**无统一修仙境界**，战力来自：① 凡人武术与体能；② 忍术（潜行、投掷、连锁技、气感训练）；③ 变异泥沼带来的肉体强化与物种化；④ 科技义肢/机器人/外星装置（因版本而异）。死亡规则：多数版本身死即死；少数线有克隆、时间术、次元穿越与「赤红铠甲」式复活，契约者勿默认无限复活。特殊系统：变异泥沼（接触者依「最近接触物/环境」变形成拟人生物）、脚族组织、频道六/记者曝光、下水道据点网络。

战力表现（按破坏力·宁低勿高）：
1. **凡人·街头炮灰**：市民、普通警察、小偷、脚族新丁、帮派打手——徒手/球棒/手枪，踹门格斗。≈一阶（初期至中期）。
2. **受训义警与老兵**：凯西·琼斯（Casey Jones）级街头义警、资深脚族忍者、艾波·奥尼尔（April O'Neil）经速成忍训后的体术线、紫龙帮（Purple Dragons）头目级——能打小队、屋顶追逐、近距闪第一枪。≈一阶后期～二阶初期。
3. **成型忍者与中坚变异体**：四龟常规作战状态、斯普林特日常教导战、卡拉（Karai）等脚族精英、比巴普与洛克斯特迪（Bebop & Rocksteady）等中型变异打手、巴克斯特·斯托克曼（Baxter Stockman）的鼠型机器人集群外围——裂石断金边缘、掀翻车辆、端掉据点。≈二阶。
4. **都市顶点·一域中坚**：四龟全力联手、巅峰斯普林特、着甲许瑞德（含部分版本「超级许瑞德」的**肉体近战上限**）、1990 电影屋顶决战级、2003 线前期脚族与变异鳄头（Leatherhead）肉搏峰值——踏平黑帮据点、屋顶团战、扛爆炸与刀剑雨；**不以夷平曼哈顿为常态**。≈三阶。
5. **情报级/越阶威胁（本世界封顶外）**：科技恶魔（Technodrome）全功率、克朗/克朗族殖民装置、部分跨次元「龟元」崩坏设定——对一～三阶为**存在·情报优先**，禁止低阶硬刚本体；本档案世界最高可玩战力封顶**三阶**（与清单 maxTier 一致）。

乐园阶位映射：市民/帮派杂兵/脚族新丁≈一阶；受训义警/精英脚忍/中坚变异体≈二阶；四龟巅峰联手/斯普林特/许瑞德（近战巅峰）≈三阶。世界顶点＝许瑞德（及版本中等价宿敌）与脚族体系 + 泥沼科技阴影；低阶切入一律**规避顶点·情报优先**。严禁「许瑞德被垃圾车压过所以战力为零」式削弱解释——其威胁以组织、铠甲与武道为准，是否当场死亡视连续统。

**【地理 · 舞台】**
- **曼哈顿下水道据点（Lair）**：四龟与斯普林特的家；训练场、厨房（披萨）、电视与电脑、武器架；多次被脚族端掉后重建。
- **纽约屋顶与消防通道**：忍战主舞台；1990 电影拉斐尔被围殴、脚族追逐艾波等名场面发生地。
- **时报广场 / 中城街道 / 布鲁克林桥**：公开犯罪、义警出没、市民误认「怪物」的冲突界面。
- **艾波的公寓 / 古董店 / 农场（上州）**：人类盟友据点；1990 电影撤退休整地；2012 线北安普顿避难所。
- **脚族据点 / 仓库 / 废弃地铁**：许瑞德与龙田（Tatsu）训练逃学少年为盗；赃物堆与刑讯间。
- **TCRI 大厦 / 科技宇宙研究所**：泥沼来源与外星科技入口（1987/2003/2012/2023 各有变体：TCRI、Techno Cosmic Research Institute 等）。
- **科技恶魔（Technodrome）与 X 次元（Dimension X）**：1987 线克朗移动要塞；可埋地底、沉北极或滞留异次元——本档案作**情报与偶发副本**，非日常街战舞台。
- **日本·滨户/小禄血仇旧地**：滨户良（Hamato Yoshi）与小禄崎的恩怨原点；部分版本有回乡篇。
- **上州农场 / 史泰登岛弃船 / 未来 2105（2003 Fast Forward）**：休整、变异团伙据点、时间跳跃副本——按切入时点选用。

写场景先定契约者在「街面 / 屋顶 / 下水道 / 脚族仓 / TCRI」哪一层，再定是否引入外星/时间线。唐人街夜市、废弃地铁隧道、布鲁克林仓库区与上州林间公路是脚族运赃与四龟伏击的次级舞台；时报广场的巨型屏幕则常在终局被艾波或反派用来争夺「谁是怪物」的叙事权。

**【世界剧情线】**
① **前史·日本血仇与泥沼坠落**  
日本忍术宗师滨户良与同门小禄崎因情（唐慎 Tang Shen）与地位交恶。小禄崎杀害滨户良（及部分版本中的唐慎），夺婴或结下不死之仇；脚族被其腐化为犯罪忍团。滨户良之宠物鼠（或滨户良本人，视版本）流落纽约下水道。与此同时，一罐放射性/外星变异泥沼在街头事故中落入下水道——与《夜魔侠》起源的戏仿同构：罐体曾险些砸中男孩。四只小龟与鼠接触泥沼，长成拟人少年与拟人鼠。鼠取名斯普林特，以观摩旧主忍术的记忆（或自身即滨户良转生/变异）将四龟收为义子，以文艺复兴艺术家之名命名：**李奥纳多（Leonardo）**、**多纳泰罗（Donatello）**、**拉斐尔（Raphael）**、**米开朗琪罗（Michelangelo）**，授忍术与「不轻易现世」的戒律。因果锁死：泥沼造英雄，血仇造宿敌，纽约成棋盘。

② **现身·艾波与脚族少年军（1990 电影/Mirage 早期合流线）**  
纽约犯罪潮升温。电视记者**艾波·奥尼尔**调查连环盗窃，遭劫时被四龟所救；拉斐尔遗落一把钗，成为人类世界与半壳英雄的第一枚信物。斯普林特告诫继续修炼。拉斐尔夜巡时与戴冰球面具、背高尔夫球袋的义警**凯西·琼斯**冲突后结谊。艾波上司查尔斯·彭宁顿之子丹尼被脚族吸收：许瑞德以「归属感」诱骗问题少年行窃，警察局长施压媒体，艾波的报道触怒脚族。许瑞德派忍者暗杀艾波；拉斐尔救下昏迷的她带回下水道——一名脚族忍者尾随，据点坐标泄露。斯普林特向艾波讲述泥沼起源。四龟送艾波回家后返回，发现据点被毁、斯普林特被掳。丹尼被捕后向许瑞德告密，指向艾波公寓。拉斐尔与李奥纳多因领导权争吵，拉斐尔屋顶遭脚族围殴重伤；凯西与三龟杀入救援，大楼起火，一行逃往艾波上州破败农场。丹尼在脚族窝点听斯普林特讲滨户良往事，醒悟逃离。农场休整中，李奥纳多感应师父，四龟灵视受教，决定返城。

③ **屋顶决战·垃圾车与家族重圆**  
四龟在旧据点找到丹尼；凯西尾随至脚族窝点，与丹尼救出斯普林特并说服被操控的少年：他们被许瑞德利用。巷战将脚族逼上街面，许瑞德持矛登屋顶与四龟决战。许瑞德宣称斯普林特已死，压制李奥纳多。斯普林特现身，指认其为小禄崎；小禄崎认出眼前之鼠即滨户良旧宠。冲锋中斯普林特以米开朗琪罗的双节棍绞矛，将许瑞德荡出屋顶，落入下方垃圾车；凯西启动压缩机——许瑞德此连续统中被压毁（续作可复活/克隆，本阶段作「阶段性击破」）。警方收缴赃物；查尔斯与丹尼和解；艾波复职；凯西与艾波接吻；四龟与斯普林特庆祝。名场面：屋顶兄弟分裂与复合、斯普林特「最后一课」、垃圾车终局。

④ **泥沼之秘·第二部电影线（概要）**  
许瑞德残党与科学家线揭开泥沼更多来历；出现更大型变异体（如德川/德古拉式强化或 Tokka & Rahzar 等，视剪辑版）。四龟继续在喜剧与动作间平衡，基调较第一部轻，但仍围绕「泥沼是诅咒也是起源」。契约者可介入实验室失窃、动物园变异、媒体狂欢。

⑤ **1987 动画长线·克朗与科技恶魔**  
此连续统中斯普林特常为滨户良本尊变异；艾波为频道六记者；许瑞德与来自 X 次元的无躯大脑**克朗**结盟，据点为可变形移动的**科技恶魔**。打手**比巴普**（疣猪变异）与**洛克斯特迪**（犀牛变异）成固定喜剧反派。主线循环：克朗欲以地球为跳板或改造行星 → 四龟破坏能源/传送门 → 科技恶魔坠入地心/北极/异次元 → 再启动。中后期「红天季」基调转暗，频道六大楼被毁，新反派德雷格勋爵（Lord Dregg）以宣传战抹黑四龟。终局方向：主要宿敌被放逐 X 次元，斯普林特称弟子已可与己平齐。适合契约者打「周常反派计划 + 披萨日常」节奏。

⑥ **2003 动画·乌特罗姆真相与许瑞德三重身**  
基调贴近 Mirage：严肃、动作、家族羁绊。开场鼠型机器人（Mousers）毁据点；四龟救下原斯托克曼助手艾波；结识凯西；对上紫龙帮头目**汉恩（Hun）**与脚族。关键揭示：许瑞德本体可为流亡地球的邪恶乌特罗姆**切雷尔（Ch'rell）**，穿人类外皮号令脚族；卡拉为其养女/弟子，立场在忠义与良知间摇摆。主线阶段：与乌特罗姆接触 → 卷入三角龙人（Triceraton）与联邦战争 → 时空学徒瑞奈特（Renet） → 忍者法庭与「天狗许瑞德」神话线 → 《加速前进》跳至 2105 科迪·琼斯时代 → 《重返下水道》网络许瑞德与艾波·凯西婚礼。跨界终章《永远的忍者龟》：2003 龟遇 1987 龟，多宇宙「龟元」设定登场。名场面：李奥纳多被脚族重创后退隐、卡拉继任、切雷尔被流放。

⑦ **2012 CGI 线·克朗族入侵与滨户美和**  
滨户良携四龟遇克朗族（Kraang），泥沼使鼠与龟变异；李奥纳多为首，多纳泰罗暗恋少女艾波（具克朗相关异能血脉）。第一季：登顶、救艾波、对抗许瑞德与卡拉（实为滨户美和 Hamato Miwa，被许瑞德偷养并灌输仇恨）；击沉科技恶魔式母舰。第二季：残余泥沼引发城市变异潮；虎爪（Tiger Claw）加盟脚族；卡拉知身世后叛出，却被许瑞德逼入泥沼成蛇型。第三季：北安普顿休养 → 夺回纽约 → 三角龙人「黑暗之心」黑条机器毁星危机 → 未来机器人弗吉托德（Fugitoid）救走主角团。第四季：宇宙搜集零件改写末日 → 超级许瑞德（Super Shredder）杀斯普林特 → 李奥纳多斩杀超级许瑞德。第五季：亡灵与魔神卡瓦萨斯、多世界交叉（1987 龟再登场）。终章方向：斩断脚族神话与克朗殖民，兄弟在废土或日常中重聚。

⑧ **2007 电影《TMNT》与 2014/2016 真人重启（概要）**  
2007：四龟疏离后因古代石像将军与商人马克西米尔·温特斯的阴谋再集结；强调「兄弟分裂与复合」。2014《变种世代》：艾波记者线 + 贝氏动作美学；许瑞德与脚族；泥沼与斯普林特起源重述。2016《破影而出》：克朗、比巴普与洛克斯特迪、监狱越狱式闹剧。作支线舞台即可，勿与 1990 正典时间线强行同一。

⑨ **2023《变种大乱斗》·青少年成长线**  
科学家巴克斯特·斯托克曼制泥沼欲建变异家族，遭 TCRI 高管辛西娅·乌特罗姆清剿而死；泥沼入下水道造就四龟与斯普林特。十五年后四龟渴望被人类接纳，结识高中生记者艾波，追查「超级蝇（Superfly）」带领的变异黑帮。超级蝇欲用 TCRI 科技让地球动物全员变异统治人类；四龟先被 captivate 后反抗，终与市民、其他变异体合作以逆突变剂解除超级蝇的海洋巨兽化。结局：四龟入学、被城市欢呼；辛西娅囚住恢复原形的蝇，意图招揽**许瑞德**——为续作钩子。主题：青春期、归属、怪物与英雄的身份政治。

⑩ **IDW 漫画与《最后的浪人》等（方向）**  
IDW 自 2011 重启：转世、克隆、克朗、脚族、更长政治线；《最后的浪人》展示「只剩一人的未来浪人」悲剧可能。作隐藏结局分支与高阶情报，不强制本档案主时间线。

⑪ **结局方向（任务世界默认闭环）**  
默认可玩终局：许瑞德（或等价宿敌）被阶段性击破或放逐，脚族瓦解或由卡拉等改革，四龟仍隐于下水道但与艾波、凯西等人维系「地表桥梁」，纽约犯罪潮暂时平复；泥沼仍可能再泄，外星/科技线可留尾巴。若走黑暗 Mirage 支线：代价是同伴重伤、据点反复被端、胜利从不永久。若走 1987 喜剧线：敌人永不根绝但永远「下周再来」。契约者蝴蝶效应可改：谁先发现泥沼、丹尼是否告密、斯普林特是否被掳、卡拉是否知身世、科技恶魔是否上浮——不可无代价抹掉「四兄弟与师父的家族结构」与「许瑞德作为镜像宿敌」的叙事功能。

⑫ **日常节奏与可介入微观事件**  
除屋顶决战外，本世界高频可写事件：夜巡拦截抢劫、脚族在唐人街收保护费、紫龙帮飙车与涂鸦领地战、艾波赶截稿与频道六主编施压、凯西在仓库私刑过度被四龟劝住、多纳泰罗黑市淘零件被盯梢、米开朗琪罗为披萨店打白工换情报、李奥纳多屋顶冥想时撞见契约者、斯普林特用茶与木杖训话、下水道涨水被迫临时搬家、警察误将变异体当「生化恐袭」、媒体把英雄写成怪物。AI 优先落这些可接触层，再偶尔拉高到许瑞德或泥沼危机。

⑬ **家族矛盾作为战斗外主线**  
李奥纳多与拉斐尔的指挥权之争是跨版本母题：前者要计划与师父认可，后者要速度与情绪出口；多纳泰罗夹在技术解决方案与「来不及发明」之间；米开朗琪罗用玩笑推迟崩溃。斯普林特被掳或重伤时，队伍会短暂解体——1990 农场线、2003 李奥纳多放逐线、2012 北安普顿休养线、2007 兄弟分居线皆同构。契约者介入「劝架」比「替他们砍许瑞德」更能改因果：让拉斐尔先道歉或让李奥纳多先听前线，会改变下一场屋顶战的站位与伤亡。

⑭ **泥沼伦理与城市政治**  
泥沼不是单纯升级药：它制造英雄也制造超级蝇、比巴普、鳄头与整座城市的恐慌。TCRI / 神盾式机构要回收样本，黑帮要武器化，克朗要行星改造，四龟要藏起源。公开身份（2023 入学式结局）与永久隐秘（经典下水道）是两条政治分叉：公开换来资源与监控，隐秘换来自由与孤独。契约者若把泥沼卖给黑市，回流通常是「变异潮 + 脚族竞价 + 艾波报道点名你」。

⑮ **跨版本名场面速查（供 AI 锚定镜头）**  
必保可复现镜头：四龟破门救艾波；拉斐尔与凯西巷战；斯普林特被囚仍教化丹尼；农场休整与灵视；屋顶刃甲对禅杖；垃圾车终局；科技恶魔钻出地面；卡拉拔刀却手抖；李奥纳多重伤后退隐再归队；多纳泰罗倒计时关门；米开朗琪罗喊出时代口号却在下一秒认真挡刀；市民从恐惧转为递水与鼓掌（2023）。写战斗时先给蒸汽、霓虹、鸽群与警笛，再给兵器交击，避免「无空间的武打字幕」。

**【叙事基调 · 雷区】**
画风应是「下水道霓虹 + 忍道热血 + 兄弟斗嘴」：可黑暗但不宜无意义虐杀；可搞笑但不宜消解许瑞德威胁。忌：① 把四龟写成无差别的同一人格；② 删除披萨/口癖导致 OOC 却又堆无根的「中二忍道演讲」；③ 低阶契约者秒杀许瑞德或摧毁科技恶魔；④ 忽略「隐藏身份」导致全纽约无代价公开化。最早可切入锚点＝四龟刚开始上街、尚未与艾波建立信任的犯罪潮初期；或 1990 电影艾波调查脚族之夜。

## 阶位切入点

> 阶位↔战力：一阶≈市民/帮派杂兵/脚族新丁与初入街头的义警；二阶≈成型忍者、凯西级义警、中坚变异体与精英脚忍；三阶≈四龟巅峰联手、斯普林特、许瑞德近战顶点。世界顶点（许瑞德体系 / 科技恶魔全功率 / 克朗殖民装置）对低阶为跨阶威胁，一律「规避顶点·情报优先」，严禁「被垃圾车压过所以为零」。

**一阶（街头犯罪潮 · 下水道见习线）**

切入身份/时点：契约者以「纽约夜班送货员 / 实习记者跟班 / 刚被脚族物色的问题少年 / 下水道维修工」切入，锚定**艾波开始调查盗窃潮、四龟刚频繁上街、斯普林特尚未被掳**的早期窗口。

初始事件：你在中城小巷目击一群少年搬运平板电视——领头的说「脚会给你家」。警察喇叭将至时，屋顶闪过蓝与红的头带；一名脚族新丁把你当成「看见太多的人」抽刀，而你口袋里恰有艾波名片或一罐疑似化工泄漏的发光残液样本。

开场白建议：「曼哈顿的蒸汽从井盖往上冒，像城市自己在喘气。你刚把车锁好，就听见刀柄撞盾牌的轻响——不是电影，是有人在消防通道上用命练功夫。一个戴橙色头带的影子丢下一句“披萨以后再说”，又没进夜色；而你脚边，滚着一枚不属于任何人类的、湿冷的龟壳扣。」

关键NPC立场：**艾波·奥尼尔**——需要线人与现场，愿用曝光换真相，怕连累普通人；**拉斐尔**——夜巡时最可能撞见你，先疑后认；**丹尼·彭宁顿**——脚族少年军一员，怕爹又渴望归属，可策反可告密；**龙田**——训练逃学孩子的冷酷教官，视你为耗材；**米开朗琪罗**——若你递披萨或冷笑话，好感来得最快；**斯普林特**（远距）——尚未允你入据点，但会托梦式警告「勿追刃甲之人」。

主线钩子/支线：主线＝脚族少年盗窃网与艾波报道的蝴蝶效应（是否提前暴露四龟）；支线＝追回拉斐尔遗落的钗、下水道泥沼残液取样、帮凯西拦住一次过火私刑、在频道六垃圾桶里翻到彭宁顿家的矛盾录音。

危险度/规避：中——热武器与脚族新丁可致命；须规避许瑞德本尊、完整脚族精英队、任何发光泥沼直触。

任务方向/奖励：活过第一个「屋顶夜」、建立与艾波或单龟的信任；奖励取向＝入门忍具、脚族联络暗号、少量稳定化泥沼样本（剧情用）、披萨店长期赊账权（正经资源：情报点）。

**二阶（据点沦陷前后 · 义警同盟线）**

切入身份/时点：以「私人安保 / 地下拳手 / 凯西式独行义警 / 多纳泰罗黑市零件供货商」切入，锚定**艾波已被救入下水道、脚族尾随得手、斯普林特被掳前后**。

初始事件：你收到匿名龟讯器坐标，赶到时据点已遭洗劫，只剩打翻的半盒披萨与斯普林特的毛。屋顶上李奥纳多与拉斐尔拔刀相向争「下一步听谁的」；远处脚族火把连成线，指向艾波公寓——你必须在「劝架」「先救人」「先夺回师父」里选一个会得罪人的答案。

开场白建议：「下水道里的灯串还在闪，可训练木人已经断成两截。蓝头带说计划，红头带说拳头，紫头带在砸坏掉的主机，橙头带试图用笑话填缝——然后所有人同时看向你，像在看第五只不该存在的龟。井盖外，纽约的警笛与忍者铃铛叠在一起。」

关键NPC立场：**李奥纳多**——要纪律与救人序位，厌恶你煽动内讧；**拉斐尔**——要立刻硬闯脚族仓，可把你当临时出气筒或刀友；**凯西·琼斯**——正在跟踪脚族卡车，愿组队但讨厌「官方腔」；**卡拉**（若该连续统已登场）——来清场或传许瑞德令，对「有用的外人」保持专业残忍；**巴克斯特·斯托克曼**——鼠型机器人二次清扫现场，要活体数据；**艾波·奥尼尔**——公寓成战场，需要你帮她销毁未播磁带或反杀镜头。

主线钩子/支线：主线＝营救斯普林特 / 阻止丹尼彻底叛向许瑞德 / 农场撤退护航；支线＝黑市卖掉的脚族刃片溯源、为多纳泰罗抢芯片修通讯、在上州农场挡住第一波追兵、调查泥沼是否与 TCRI 货车同源。

危险度/规避：高——精英脚忍与中型变异打手出没；须规避许瑞德着甲决战、科技恶魔主炮、克朗本体交易现场。

任务方向/奖励：成为「地表腿脚」或临时第五人；奖励取向＝强化忍具、斯普林特亲传步法残篇、脚族仓库钥匙层、稳定交通工具（改装厢型车零件）、与凯西的义警同盟标记。

**三阶（屋顶终局 · 家族与刃甲线）**

切入身份/时点：以「成名地下武者 / 脚族叛逃精英 / 政府灰色清道夫 / 四龟认可的盟友」切入，锚定**农场归来、四龟决意总攻、许瑞德与斯普林特即将屋顶对决**（或 2012 超级许瑞德之战、2003 放逐切雷尔之战等等价终局窗口）。

初始事件：脚族窝点内，被洗脑的少年举刀对准斯普林特的笼；许瑞德已在屋顶整甲。你手中握有三选一的关键物：压缩机的遥控、卡拉的身世证据、或一管足以把许瑞德「再变异」的不稳定泥沼——用任何一个都会改写终局的味道，也会让四龟中至少一人永远记住你的选择。

开场白建议：「风把时报广场的广告光削成刃。你看见刃甲在屋顶边缘张开，像一只金属甲虫；四道色带从消防通道跃起，排成你在传闻里听过无数次的阵型。斯普林特的声音很轻，却压过直升机螺旋桨：“今晚不是为了报仇——是为了证明，我们仍是一家人。”」

关键NPC立场：**许瑞德 / 小禄崎**——视你为变数，可招揽为「有用的人类刃」或优先斩杀；**斯普林特**——若你护住少年兵，会给你「滨户之名下的一诺」；**卡拉**——身世或忠义临界点，可反戈可成最终壁垒；**李奥纳多**——要活口与纪律，反对泥沼滥改人体；**拉斐尔**——支持把许瑞德「送进压缩机」，厌恶伪善招安；**克朗 / 切雷尔**（若线启用）——在维生液罐中出价「科技换许瑞德的命」；**艾波与凯西**——街面疏散与媒体叙事权在他们手里，能把你写成英雄或怪物。

主线钩子/支线：主线＝屋顶决战介入（保斯普林特 / 保李奥纳多 / 改写许瑞德下场）；支线＝说服脚族少年军临阵倒戈、夺取黑兜鍪、切断克朗传送门三十秒窗口、在垃圾车与「活捉送审」之间做政治选择。

危险度/规避：极高·贴近顶点——许瑞德近战即本世界天花板；若科技恶魔升空或克朗机甲落地，转为**情报优先 / 条件性胜利**（毁能源核、拆传送环、引开主炮），严禁宣称「我单挑拆了母舰」。

任务方向/奖励：参与书写「半壳英雄是否公开于世」的结局；奖励取向＝许瑞德铠甲碎片（剧情级防具）、斯普林特认可的道场挂名、泥沼与逆泥沼剂配对样本、脚族残卷与日本血仇情报、四龟家族式羁绊（人脉，非血缘）。
