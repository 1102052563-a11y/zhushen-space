# 敢死队

## 剧情

**【作品来源】**  
《敢死队》（英语：*The Expendables*，港译《轰天猛将》，台译《浴血任务》）是2010年美国动作片，席维斯·史特龙（Sylvester Stallone）执导并与大卫·卡拉汉（David Callaham）共同编剧；卡拉汉故事稿源自其早期雇佣兵题材脚本 *Barrow*。制片 Nu Image / Millennium Films，狮门发行。2010年8月13日北美上映，片长约103分钟，预算约8000–8200万美元，全球票房约2.745亿美元。主演史特龙、杰森·斯坦森（Jason Statham）、李连杰（Jet Li）、道夫·龙格尔（Dolph Lundgren）、泰瑞·克鲁斯（Terry Crews）、兰迪·库图尔（Randy Couture）、艾瑞克·罗伯茨（Eric Roberts）、史蒂夫·奥斯汀（Steve Austin）、米基·洛克（Mickey Rourke）等；布鲁斯·威利斯（Bruce Willis）、阿诺·施瓦辛格（Arnold Schwarzenegger）客串。续集《敢死队2》（2012，西蒙·韦斯特执导）、《敢死队3》（2014）、《敢死队4 / Expend4bles》（2023）。文风：向1980–90年代动作片致敬的雇佣兵群像、高密度枪战爆破、队内兄弟义气与「可牺牲者」黑色幽默。本档案以**第一部电影正典为常青主轴**，续集作系列互文与舞台延伸；搜笔趣阁未收录该片小说化全本（非网文），剧情以中英文维基、角色列表与公开影评可核资料为准。

**【世界定位】**  
近现代雇佣兵地下经济：新奥尔良据点、索马里亚丁湾海盗线、墨西哥湾虚构岛国维连那（Vilena）的毒枭—独裁复合体。主角**巴尼·罗斯（Barney Ross）**率领敢死队（The Expendables）——一群被官方当耗材、却仍能完成官方不敢亲自做的脏活的精英佣兵。一句话：**没有异能、没有复活外挂的热武器 + 近战 + 爆破写实战**；契约者切入的是佣兵合同、CIA 外包与岛国政变，顶点是「敢死队核心成员联手 + 叛变CIA代理人」，不是灭城魔神。

**【世界观 · 力量体系】**  
本世界**无修真、无异能、无系统面板**。战力 = 特种/佣兵训练 + 枪械与重火器熟练度 + 近战（刀/摔角/武术）+ 爆破与载具 + 情报与地形。死亡即死亡；重伤靠医疗与意志，不能靠「突破境界」回满。

**1. 凡人 / 平民层**  
维连那市民、新奥尔良酒吧客、货船水手、无作战经验青年：徒手或偶发手枪，一枪可致命。≈一阶初期。

**2. 地方武装 / 普通士兵 / 海盗**  
索马里海盗、维连那普通驻军、码头打手：有AK与编制，但缺乏对抗老牌佣兵小队的协同与重火力。≈一阶中期。

**3. 精锐卫队 / 雇佣打手 / 老练执法**  
嘉萨将军精锐、蒙罗私人卫队外围、有实战经验的码头安保与地下拳手：能组织封锁与刑讯，仍难在开阔火力下锁死敢死队。≈一阶后期～二阶初期。

**4. 职业佣兵 / 特种退伍 / 敢死队正式成员**  
前SAS、前陆战队、重火器手、爆破手、武术近战手：可在人数劣势下以小队战术、重武器与近战组合清场据点。≈二阶中期。

**5. 顶点：巴尼·罗斯小队作战态 + 詹姆斯·蒙罗体系**  
敢死队全员协同（飞机突入、宫殿爆破、机枪压阵、刀斗）可端掉独裁宫殿级目标；蒙罗一侧有私人直升机、刑讯专家、训练有素的卫队与毒资网络。破坏力 = 单人至小队清场 + 车辆/宫殿连环爆炸 + 重机枪与燃烧，**达不到**夷平城池、抗导弹。  
**政治 / 编制顶点**：「教堂先生」所代表的CIA外包体系、壕沟鼠（Trench）竞争佣兵团——个人武力未必最强，但可调动合同、情报与合法化掩盖。对契约者 = **情报与合同碾压**，不是魔法。

**关键规则（比「境界」更重要）**  
- **可牺牲合同**：敢死队之名即「可被消耗」；任务失败或雇主反水是常态，信任只建立在队内。  
- **CIA 不亲自动手**：官方怕暗杀本国前特工引发丑闻，故雇佣兵——合同常隐瞒真实目标。  
- **毒资即政权**：维连那表面是嘉萨独裁，实质是前CIA特工蒙罗的可卡因利润机器。  
- **队内纪律 vs 个人崩溃**：吸毒/精神不稳成员（如冈纳）会被开除；被开除者可能投敌。  
- **兄弟义气触发任务重启**：巴尼可因「扔下的人」内疚而单人返岛，队员会硬跟。  
- **续集火力升级（互文）**：第二部起出现武器级钚、桑格斯佣兵团、孤狼佣兵；本世界卡常青主轴仍停在维连那终局，续集作远征可选。

**乐园阶位映射（对照《阶位战力图鉴》，宁低勿高）**  
平民 / 普通士兵 / 海盗 / 码头帮工≈**一阶**；职业佣兵、特种退伍、敢死队正式成员、蒙罗卫队精锐、嘉萨精锐指挥官≈**二阶**；世界顶点 = **巴尼·罗斯小队联合作战态 + 蒙罗/嘉萨体系 + 教堂先生所代表的CIA合同权柄**。低阶切入一律**规避与敢死队/蒙罗正面对冲、规避宫殿爆破核心区**，情报与合同优先。严禁用「任务限制战力/剧情削弱」解释顶点——顶点始终是完整战力的佣兵与叛变特工体系，低阶只能规避或条件性周旋。

**【地理 · 舞台】**  
- **新奥尔良（New Orleans）**：敢死队大本营。图尔的刺青店兼据点与「任务后庆功场」；酒吧、法式区街道、高架桥与工业区构成日常与私斗舞台。李·圣诞与蕾西的感情线在此撕裂又修补。  
- **亚丁湾 / 索马里海盗船**：开场人质营救。货船甲板、货舱与海盗小艇——冈纳失控与阴阳冲突的起点，也是「活口政策」与「私刑」的分界。  
- **教堂（Church）会面点**：巴尼与壕沟鼠同见「教堂先生」的封闭空间；合同、竞价与「你去竞选总统」式嘲讽发生地。  
- **维连那（Vilena）**：虚构拉丁美洲/墨西哥湾岛国。港口、贫民区、宫殿、军营与直升机坪。嘉萨表面执政，蒙罗在幕后以毒资控场；珊卓是将军之女兼反独裁联络人。侦察夜的街道枪火与宫殿夜攻是全片主战场。  
- **废弃仓库 / 公路追逐线**：冈纳投敌后伏击巴尼与阴阳；阴阳与冈纳二次决斗、巴尼开枪「劝降」的节点。  
- **嘉萨宫殿复合体**：炸药埋设点、庭院火力网、刑讯室、停机坪。终局爆破与人质营救在此合流。  
- **续集延伸舞台（互文）**：尼泊尔营救壕沟鼠与中国富商（第二部开场）；阿尔巴尼亚坠机与废弃苏军矿井钚库；保加利亚村庄与机场决战；索马里军火商、罗马尼亚交易、阿兹梅尼斯坦（第三部）；第四部核武与「豹猫」内鬼线。本世界卡以维连那事件为核心。

**【世界剧情线】**  

**〇 · 前史（佣兵经济与敢死队成型）**  
巴尼·罗斯前为空军飞行员与海军陆战队员，退役后组建以「敢死队」为名的精英佣兵小队，据点在新奥尔良。核心成员长期固定：刀战专家**李·圣诞**（前SAS）、武术手**阴阳**、重火器手**海尔·凯撒**、爆破/摔角手**托尔·罗德**、化学工程背景却精神不稳的**冈纳·詹森**。老队友**图尔**已退居刺青与招募联络，不再出任务。竞争者**壕沟鼠·毛瑟**经营另一支佣兵团；CIA侧**教堂先生**习惯用外包解决「不能写进档案」的目标。维连那方面，前CIA特工**詹姆斯·蒙罗**以可卡因利润绑架岛国政治，扶植**嘉萨将军**作傀儡，并以**丹尼尔·潘恩**、**劳伦斯「英国人」·斯帕克斯**等打手训练卫队、执行刑讯。

**① · 亚丁湾人质（开场）**  
敢死队空降/登船营救被索马里海盗扣押的人质。协议赎金三百万，海盗坐地起价五百万；冈纳主张直接开火并造成海盗伤亡，甲板瞬间从谈判场变成屠宰场。人质救出后，冈纳欲对俘虏施以绞刑式私刑——把人吊起来当杀鸡儆猴——阴阳以近战制止，两人在货舱级别空间里斗殴，巴尼强行分开并**当场开除冈纳**：纪律优先于火力，活口政策优先于个人发泄。此阶段定调三条铁律：①队内有道德底线，但底线靠队长拳头与开除权维持；②被开除者带着怨恨离开，可能成为下一单的敌方眼线；③「敢死」不等于「失控」，失控者先被队内处理。契约者若在此阶段以水手/谈判译员切入，会先看见赎金讨价还价，再看见自己人互殴——比海盗更可怕的是队内裂缝。

**② · 教堂合同与维连那侦察**  
巴尼与竞争者壕沟鼠同见教堂先生。壕沟鼠以「忙别的事」（巴尼嘲其忙着选总统——对施瓦辛格州长身份的元梗）让出合同，巴尼接下：推翻维连那独裁者嘉萨，酬金五百万（半数预付）。合同措辞只写「倒嘉萨」，不写蒙罗——这是CIA外包的典型话术：目标封面化。巴尼带李·圣诞赴岛与联络人**珊卓**接头，街头与宫殿外围侦察中暴露，发现真正控场者是前CIA**蒙罗**，嘉萨只是毒资政权的傀儡；珊卓竟是嘉萨之女，却站在反独裁一边，为岛民提供内部路线与军情。卫队围攻，二人突围离岛——**珊卓拒绝撤离**，理由是岛不能只剩将军与毒枭。巴尼推断：教堂是CIA，真实目标是叛变的蒙罗；官方不敢亲手杀「自己人」引发丑闻，才雇佣兵当刀。内疚成为全片情感引擎：扔下珊卓 = 扔下图尔故事里那个没伸手的女人；合同可以撕，良心账撕不掉。

**③ · 队内裂缝与私人线**  
李·圣诞发现女友**蕾西**另结新欢——佣兵长期失踪的生活方式先杀死关系，再被家暴二次伤害。圣诞见其遭新男友施暴，闯入痛殴施暴者及其同伙，并当众暴露「我靠刀吃饭」的真实职业：私人线炸开，等于把队的存在方式亮给民用世界。与此同时，被开除的冈纳投靠蒙罗，出售敢死队动向与战术习惯；蒙罗对珊卓水刑逼供，要她吐出佣兵路线；嘉萨对蒙罗的颐指气使日益不满，主仆关系进入破裂倒计时。巴尼在图尔刺青店倾诉「扔下女人」的恶心感；图尔坦白波黑时期曾眼睁睁让一名女子自杀而未救——独白没有配乐英雄感，只有退役刺青师的手在抖。这段把「佣兵的良心账」说死：你不是没能力救，是你选择了不当英雄。巴尼决定**独自返岛救珊卓**；阴阳坚持同行——队的最小作战单元在道德重启时仍是两人。

**④ · 仓库决斗与情报回收**  
公路上冈纳率蒙罗方面人马伏击巴尼与阴阳；废弃仓库内阴阳与冈纳二次近战，冈纳以体型与疯劲占上风，欲将阴阳钉死在管道上时，**巴尼开枪击中冈纳**。濒死的冈纳回心转意，交出嘉萨宫殿布局、卫队换班与薄弱点——情报价值高于当场处决叛徒。巴尼登机准备单人赴死，却发现凯撒、罗德、圣诞等**全队已在机上**：没人让队长一个人去当「可牺牲者」。此阶段名场面因果链：开除→投敌→再开枪→情报→原谅与归队，完成冈纳弧光的第一次闭合，也完成敢死队「纪律残酷但允许救赎」的队规展示。契约者若站在蒙罗侧，此战是眼线报废；若站在敢死队侧，此战是夜攻可行性的钥匙。

**⑤ · 宫殿夜攻与蒙罗之死**  
敢死队潜入宫殿复合体，由托尔·罗德主导埋设炸药网，目标不是屠城，是炸掉指挥与撤离节点。嘉萨误判蒙罗雇人来杀自己，令士兵涂脸备战，并与蒙罗摊牌、下令驱逐美国人、收回尊严。蒙罗枪杀嘉萨，嫁祸敢死队，军队因「将军被美国人杀」而全面开火——嫁祸战术把敢死队从「倒独裁」瞬间变成「全民公敌」。英人「英国人」斯帕克斯被圣诞与阴阳联手近战击杀；潘恩与巴尼肉搏后逃走，终被托尔·罗德拖入火中烧死；蒙罗挟珊卓逃向直升机，凯撒等以重火力摧毁直升机，巴尼与圣诞追上——巴尼用背后藏枪出其不意击中，圣诞补刀，蒙罗死。宫殿在连环爆破中崩塌，嘉萨军指挥链断裂，维连那恐怖统治实质结束。终局主题：**傀儡死于主子，主子死于自己雇来的刀；钱留给岛，人不留下统治。**

**⑥ · 收场与系列钩子**  
巴尼把任务酬金账户留给珊卓，用于重建维连那——佣兵史上少见的「把报价单改成重建基金」。队员返回新奥尔良，在图尔店庆功；**生还的冈纳归队**并与阴阳握手言和，队内裂缝表面缝合。圣诞与图尔飞刀较量，圣诞在店外扔出红心并作嘲讽诗，把战场紧张泄成兄弟玩笑。定格：重机车驶入街头，像下一单随时会来。  
（系列后续钩子：第二部教堂以「蒙罗之死的补偿」迫使敢死队取回阿尔巴尼亚坠机电脑，比尔·「小子」·蒂蒙斯被让·维兰杀害，钚矿复仇；第三部共创者康拉德·斯通班克斯回归军火商线；第四部核武与内鬼「豹猫」。常青档案默认停在第一部庆功定格，续集仅在玩家主动远征时展开。）

**⑦ · 续集因果链（系列互文 · 详述）**  
- **第二部（2012）**：尼泊尔开场救出被俘的壕沟鼠与中国富商，阴阳护送「钱袋子」回中国暂离队；教堂派技术员**玛姬·陈**同行，任务是取回苏军遗留**五吨武器级钚**的坐标电脑。阿尔巴尼亚坠机点遭**让·维兰**与桑格斯团伏击，维兰夺电脑并当众杀害新人狙击手**比尔·「小子」·蒂蒙斯**——队史首次「新人被公开处决」级创伤。敢死队经保加利亚废弃基地、村庄奴隶矿井，得孤狼**布克**、教堂与壕沟鼠支援，在机场决战：圣诞杀副手**赫克托**，巴尼手刃维兰。比尔遗书与巨款被送至女友索菲亚门前。  
- **第三部（2014）**：原队医**死神医生**（Doc）从囚车被救出归队；共创者**康拉德·斯通班克斯**以军火商身份重创**海尔·凯撒**。巴尼解散老队，经**波拿巴**招募新血**约翰·斯迈利、卢娜、索恩、马尔斯**，并接纳话痨射手**加尔戈**；CIA联络换为**马克斯·德拉默**，要求活捉送海牙。波兰/罗马尼亚交易线抓捕失败后，新队被俘至阿兹梅尼斯坦，老队重聚，巴尼枪决斯通班克斯，建筑崩塌中撤离。  
- **第四部（2023）**：核武威胁与CIA内鬼**马什/代号豹猫**；巴尼旧任务被出卖的阴影回收，小队再组。系列主题从「倒一个岛国毒枭」升级为「队史背叛与大国脏弹」，但第一部仍是「可牺牲者找回良心」的原点。  

**【主要人物】**

**【叙事基调 · 雷区】**  
基调：老派动作片的爆炸美学 + 佣兵黑色幽默 + 「可牺牲者」的兄弟情；暴力血腥但情感核是「不扔下自己人」。忌把巴尼写成无脑爱国英雄（他收钱、谈价、也会逃）；忌把敢死队写成正规军（否认机制与脏活是铁律）；忌消解冈纳吸毒/精神问题或无代价归队；忌低阶任务让契约者正面硬刚宫殿爆破核心与全员敢死队；忌用异能解释飞刀与枪斗。最早可切入锚点 = **亚丁湾任务前后新奥尔良据点**，或**巴尼已接合同、尚未登岛侦察**的窗口。

**【世界剧情线 · 补充细节（电影正典关键因果）】**  
维连那事件时间密度：从合同到宫殿夜大约横跨数日。契约者判断世界状态可按下表：  
1. **平静窗口**：冈纳仍在队、亚丁湾未打——据点训练与酒吧日常。  
2. **合同窗口**：教堂会面后——情报贩与军火商开始盯维连那航班。  
3. **侦察窗口**：巴尼与圣诞在岛——珊卓线暴露风险最高。  
4. **内疚窗口**：小队已离岛、珊卓留下——巴尼最易被说服返航。  
5. **夜攻窗口**：炸药已埋、嘉萨涂脸备战——非战斗员应远离宫殿半径。  
6. **收场窗口**：蒙罗死、赠款完成——岛国权力真空，敢死队已撤。  

**敢死队作战伦理（写给 AI 的行为锚）**：他们杀人效率极高，但队内有「活口/私刑」争论；巴尼会为珊卓这种「具体的人」重启已放弃的合同；对雇主可以骂、可以加价，对队友极少真弃。若任务把敢死队写成无差别屠城享乐团，即 OOC。  

**教堂为何必须雇人**：杀本国前特工的政治成本高于五百万佣金；敢死队的价值是「可否认」。契约者若想取代敢死队接单，须能提供同等可否认性与火力，否则只是送死。  

**与乐园任务的接口提示**：  
- 一阶任务多在「新奥尔良外围 / 海盗船底层 / 维连那市民与联络网」层，奖励是情报与人脉。  
- 二阶任务可介入「侦察夜 / 仓库线 / 宫殿外围爆破组」，奖励可含高级枪械与佣兵声望，仍禁止灭国级道具。  
- 任何阶位若要「击败巴尼小队或蒙罗」，须设计为合同欺诈、人质交换、情报出卖、地形隔离等**条件性胜利**，不得写成属性碾压秒杀。

**【地理 · 舞台补充】**  
新奥尔良据点不是「军事基地」，而是刺青店、酒吧与工业区混搭的灰色地带——警察睁一只眼，黑市另一只眼。维连那在片中呈现为热带岛国：港口集装箱、军用卡车、宫殿庭院与贫民屋顶连成一张枪战地图。拍摄实景多在巴西里约与路易斯安那，但叙事上应保持「新奥尔良—维连那」设定，避免把拍摄国直接写成剧情国界。宫殿爆破后的权力真空、珊卓手握重建资金时的民兵重组、以及CIA对「任务完成」的冷淡结案，都是可写的余波层。

**【主要人物补充弧光】**  
巴尼与圣诞是「队长—刀锋」对位：一个谈价与内疚，一个执行与感情爆炸。冈纳是纪律的反面教材也是救赎样板。蒙罗与嘉萨是「寄生者—宿主」：宿主想恢复尊严的瞬间即被杀。珊卓不是花瓶人质，而是敢死队唯一的本地道德坐标。图尔戏份少却决定巴尼是否成为「只收钱的屠夫」。把这些人写成标签（「好人佣兵」「坏人将军」）会失去第一部「脏手与良心」的厚度。

**【贵重物品与情报资产补充】**  
除枪刀外，第一部真正值钱的「战利品」是：蒙罗毒资流水残页、嘉萨军电台频段、宫殿地下通道手绘、教堂合同编号、珊卓反独裁联络网名单、以及「蒙罗水刑珊卓」的录音/目击。契约者若以记者/岛民医生/军火掮客身份切入，这些比重机枪更符合一阶奖励逻辑。

## 阶位切入点

> 阶位↔战力：一阶≈平民/海盗/普通士兵/码头帮工/新奥尔良外围；二阶≈职业佣兵/特种退伍/敢死队正式成员/蒙罗卫队精锐。世界顶点（巴尼·罗斯小队联合作战态 + 蒙罗/嘉萨体系 + 教堂先生CIA合同权柄）为跨阶存在，低阶切入一律「规避顶点·情报优先」。严禁用任务限制或剧情削力解释顶点；对顶点采取存在·情报优先/条件性胜利。

**一阶（新奥尔良据点与维连那街头 · 跑腿/目击者/底层兵线）**

切入身份/时点：契约者以「新奥尔良酒吧夜班/图尔店学徒跑腿/货船水手/维连那港口搬运工/反独裁小组外围传话人」切入，锚定**敢死队刚结束亚丁湾任务、冈纳被开除前后**，或**巴尼已接教堂合同、尚未登岛**的数日窗口。

初始事件：你在图尔刺青店后门听见重机车熄火——巴尼与圣诞低声争论「五百万值不值得去一个叫维连那的岛」。下一班船期，有人出高价要你把一只密封信封送到法式区某酒吧包厢，收件人只说「教堂的人」；同日码头流传「索马里那单有人被队长一脚踢出队」。你必须在「送件不问」「拆封复印卖情报」「把消息透给冈纳那帮失意佣兵」之间选边，而选边会在48小时内被敢死队或蒙罗眼线记住。

开场白建议：「密西西比河的潮气把霓虹糊成一片脏黄。刺青针的嗡鸣里，有人在谈五百万和一个你没听过的岛。信封烫手，像刚从枪匣里抽出来。你若拆开，你是情报贩；你若原样送到，你至少还能在明天的码头活着搬箱子——前提是，没人告诉冈纳你看见他被赶出队。」

关键NPC立场：**巴尼·罗斯**——只对「守口如瓶的跑腿」给活路，厌恶八卦队内纪律；**李·圣诞**——心情差时刀比话快，但家暴案前后可能需要证人闭嘴；**图尔**——不参战，却能决定你是否被介绍进外围活；**冈纳·詹森**（刚被开除）——缺钱、恨队，危险的买方；**珊卓**（远距）——若你在维连那，她需要能带路却不向蒙罗告密的人。

主线钩子/支线：保住或泄露教堂合同相关地点、在维连那港口为侦察组提供无记录车辆/假证件、目击蒙罗卫队提人却选择报警或沉默；支线＝帮蕾西匿名报警家暴（勿直接卷入圣诞报复）、倒卖海盗船遗留弹药箱、替图尔送药给退役老兵（事件与人名须本阶独有，勿与二阶雷同）。

危险度/规避：中——主要死于码头火并、卫队勒索、拆封后的灭口；须规避敢死队训练场流弹、冈纳酒后私刑、维连那宵禁 Sweep。巴尼小队与蒙罗本尊 = 情报优先，勿持枪去「证明自己配进队」。

任务方向/奖励：活过合同周、拿到可卖的港口地图/卫队换班表、与图尔或珊卓外围网建立信任；奖励取向＝防刺衣、合法/半合法枪证线索、新奥尔良黑市联络、维连那平民避难屋钥匙（一阶勿发重机枪或宫殿炸药原件）。

**二阶（侦察夜与宫殿外围 · 佣兵/卫队/中间人线）**

切入身份/时点：契约者以「敢死队临时外援/壕沟鼠团借调火力手/蒙罗卫队新晋教练/教堂线下观察员」切入，锚定**巴尼与圣诞登岛侦察前后**，或**全队返岛埋弹、嘉萨涂脸备战、蒙罗准备杀嘉萨**的宫殿夜时间窗。

初始事件：你随第二批补给接近维连那海岸，对讲机里同时出现两个声音：巴尼要「只救人、少爆炸到平民区」；蒙罗的人要你「把将军女儿带去刑讯室」。下一秒，宫殿探照灯扫过甲板——侦察已经暴露。你必须在**执行敢死队撤离掩护**与**执行蒙罗清场令**之间做一次无法两全的选择，而圣诞的刀与潘恩的拳头，可能已经在你侧翼十米内。

开场白建议：「岛风里有柴油和夜来香。你的靴底刚沾上维连那的土，耳机里就响起两套命令：一套要你活着把人带出宫，一套要你把宫门焊死。探照灯白得像审讯灯——你忽然明白，敢死队这个名字不是酷，是账单上的损耗率。」

关键NPC立场：**巴尼·罗斯**——对「不开枪扫平民的佣兵」可谈分成，对「卖队」者当场处决；**李·圣诞**——侦察搭档，需要你提供巴尼顾不上的侧翼；**詹姆斯·蒙罗**——视一切中间人为耗材，怕的是教堂档案曝光；**丹尼尔·潘恩**——喜欢用肉搏测试新人，失败即死；**珊卓·嘉萨**——可作人质也可作向导，取决于你是否把她当筹码；**冈纳·詹森**（若在投敌段）——情报源与背刺源一体。

主线钩子/支线：是否向巴尼报告「真实目标是蒙罗」、在水刑前抢出珊卓或至少留下录音、宫殿埋弹组的引信时间是否提前/推迟以保平民巷；支线＝截获冈纳卖队证据、与壕沟鼠频道做双份保险合同、阻止英国人处决战俘（或相反：确保其不活口泄密）。

危险度/规避：高～极高——宫殿交叉火力、直升机扫射、连环爆破坍塌、队内误伤；须规避与巴尼全队或潘恩/蒙罗正面单挑、规避引信核心区。世界顶点敢死队联合作战态 = 二阶巅峰人体+火力极限，须以**存在·情报优先/条件性胜利**对待：改合同、换人质、炸撤离通道，严禁一人硬刚传说小队。

任务方向/奖励：促成珊卓存活与蒙罗体系断裂（可与原作并行：多重撤离）、保全巴尼赠款转移通道、拿到蒙罗毒资残账与教堂合同编号；奖励取向＝佣兵级枪械与刀具、假身份与离岛船期、敢死队/壕沟鼠任一侧声望、特种爆破入门（二阶可给高级装备与训练，勿给灭城/核级道具）。
