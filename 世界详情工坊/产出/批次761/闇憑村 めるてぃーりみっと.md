# 闇憑村 めるてぃーりみっと

## 剧情

**【作品来源】**
《闇憑村》（やみつきむら）是漫画家**あるぷ**单行本《めるてぃーりみっと》（ジーオーティー刊，约 2020 年前後）所收的和风伝奇成人短篇系列；结构常以「第一夜／第二夜／第三夜」分话推进。2022 年由**ショーテン**改编为成人 OVA《闇憑村／めるてぃーりみっと The Animation》上下卷（约 2022-03 上卷、2022-04 下卷，各约 17 分钟量级）。气质：明治后封闭山村的淫魔传说、夜這い、巫女「お祓い」与民俗学者调查喜剧。本档案写关系因果、秘密与可退出伦理，过程从略；角色均为**成人村民／成人调查者**。

**【世界定位】**
一句话：这是「男人不要单独进村」的谣传背后，民俗学者**門部幸太郎**被村娘**静**接住、却在夜這い与憑印中卷入**淫魔**因果的和风情感世界。契约者进入后玩的是：调查权限、谁先承认夜訪、巫女祓除是净化还是邀请、淫魔身份揭晓后的谈判——不是清图杀怪。核心符号是**村口「男人勿独行」的闲话**、**静端来的补食**、**胸前浮现的淫印**、**祠堂小灯**。

**【世界观 · 舞台设定】**
时代气质为明治维新后若干十年的封闭山村（原作和风，不必死钉公元）。无现代化冲突强度叙事。世界规则是：**民俗调查伦理**、**夜這い作为地方性亲密脚本**、**憑きもの／淫魔以精气与印记绑定关系**。超自然只写情感侧：印记出现＝秘密被共享；白天冷淡、夜里主动＝社会脸与欲望缝；「お祓い」名义可被用来逼近身体，也可被契约者改写成真正的边界协商。

软规则五条：①每个女角有「公开村民脸」与「夜的缝」；②任何亲密升级必须可退出（「待って」「今夜は帰って」）；③第三人视线是压力计——其他村娘、巫女、门外脚步；④默认相位＝调查进村→静收留→夜這い与印记→祠堂对峙；⑤失败态是被精气掏空失忆、关系沦为单方面捕食、或调查者把村民当样本羞辱，不是数值战死亡。

NSFW 为 R18 和风成人向；档案只写关系结构与事后对视，过程从略。入世铁则：角色均为成人；禁止未成年性描写。

**【地理 · 生活舞台】**
- **村口土路与告示式闲话空间**：外来者第一道墙。情感功能＝「你被当作危险还是猎物」。
- **静的家／客房被褥**：收留与夜訪的主容器。补食、灯火、榻榻米气味。
- **村道与井边**：白天公开脸。桜歌等可在此冷淡对待调查者。
- **神社／祠堂／小社**：终局揭晓场。灯、铃、淫魔真身。
- **巫女居所／祓所**：蓮華的主场；「お祓い」名义下的权力与亲密交界。
- **山缘林道**：调查者准备药草／抗魔手段的缓冲带。
- **村民集体视线（虚写）**：无人旅馆、不卖饭——社会性孤立装置。
- **后山溪边**：洗濯与偶遇；功能＝白天的「几乎像普通村」的假象。
- **村境石标**：进村／出村的心理关；印记亮起的人会觉得石标变近。

每个地点的危险时间：客房＝子夜；祠堂＝第三夜；井边＝晨间对视测验；村口＝午后闲话峰值。

**【故事主线 · 情感线】**

**相位 A · 谣传与进村**  
外界流传：「那村，男人不要单独去。」民俗学者**門部幸太郎**为求证进村。村人冷淡，无宿、不答、把外来者当不祥。世界的第一口气味是山土与炉灰，第一道声音是拒绝的沉默。他在笔记首页写下三个字：「封闭。」

**相位 B · 静的收留**  
村娘**静**在门前伸出援手：安排一宿、端补食、用温柔盖住全村的敌意。情感引擎不是战斗，是**被需要的假象与真实善意混杂**——门部感激，也开始记录异常：为什么只有她愿意说话？为什么补食的碗比普通人多一双？

**相位 C · 第一夜：桜歌的夜這い**  
白天冷淡的**桜歌**夜里来访。亲密发生后，門部看见她胸前的印记，自己胸口也浮出对应纹样。调查从「民俗笔记」变成「身体也被写进村的故事」。因果：印记＝可观察的绑定证据，不是经验条。次晨井边，桜歌再次冷脸——公开脸与夜的缝裂开。

**相位 D · 第二夜：蓮華与祓除**  
門部求**蓮華**巫女祓除。祓除话术与身体接近重叠；蓮華亦有印记。门部意识到「净化者」可能同在结构内。他开始准备民间药方／抗魔手段——档案写为**可退出与反制的知识准备**，不写数值战。药草在林道采集时，他听见远处铃与呼吸同步。

**相位 E · 第三夜：静＝淫魔**  
在祠堂，**静**露出淫魔侧：小角、翼、尾；承认操控村娘为精气。她召唤／牵引桜歌与蓮華同场。表面是捕食，情感内核是：**长期以「帮忙」为名的支配，是否还允许被拒绝**。門部若只抵抗羞辱，会走向 BE；若在对峙中保留尊重与谈判，可把「捕食」改写为「可约定的交换／共栖」。

**相位 F · 收束分支（入世可改写）**  
- **HE 向**：淫魔身份公开后建立可退出契约；静不再单方面操控同伴；門部完成调查却选择守密或有限度发表。  
- **共栖向**：契约者成为第二知情者，帮村与外界隔离带定规矩。  
- **BE**：精气被掏空失忆；或把门部的笔记公开导致村庄被猎巫式摧毁。  
- **开放接口**：下一位外来调查者、印记复发、邻近村落的新谣传。

**共通情感因果**  
谁害怕：静怕被当怪物；桜歌怕白天的脸碎掉；蓮華怕神职身份破产；門部怕自己变成只取精的样本。  
谁靠近：静用收留；桜歌用夜；蓮華用祓；門部用笔记与药。  
哪一句改变权力：「男人不要单独来」把外来者钉成猎物；「我帮你一宿」打开私人入口；「印记在你胸口」把调查变成共犯。

**【可攻略角色 / 主要人物】**

1. **門部 幸太郎（もんべ こうたろう｜民俗学者·可玩家化视点）**  
   外貌：成年男性学者气质，行装带笔记与药草。  
   性格：求知、礼貌、会害怕但会准备。  
   角色类型：外来调查者／选择轴。  
   萌点：被拒绝仍鞠躬；发现印记后先写笔记再恐慌。  
   个人线：调查→被卷→对峙→守密或共栖。  
   与契约者：默认可代入；勿写成无敌除灵师。

2. **静（しず｜村娘／淫魔·核心可攻）**  
   外貌：温柔村娘公开脸；真身含角翼尾（可爱向设计，OVA 强调粉长发与小翼）。  
   性格：日间助人、夜间主导；欲望深但可被「停」打断才是 HE 条件。  
   角色类型：伪善／真饿的双面核心。  
   萌点：补食与笑容；揭晓时的坦白。  
   个人线：收留→布局→暴露→谈判。攻略关键不是打败她，是**让她承认可退出**。

3. **桜歌（おうか｜村娘·夜這い）**  
   外貌：强势、白天冷。  
   性格：口硬、夜主动；印记持有者。  
   角色类型：反差夜訪。  
   个人线：从执行夜訪到质疑静的支配。  
   与主角：好感从「被冷待仍礼貌」起步。

4. **蓮華（れんげ｜巫女）**  
   外貌：巫女装束。  
   性格：仪式感强；祓除与欲望纠缠。  
   角色类型：神圣话术下的共犯／可救赎。  
   个人线：从「我在净化」到「我也在印记里」。

5. **村民群（虚写）**  
   功能：拒绝宿、拒绝答、形成孤立。

6. **学会／外界通信（可选）**  
   功能：BE 的猎巫出口；HE 的「不发那封信」。

- **True 向**：淫魔身份与共栖规矩并存；村不再靠猎杀外来者维生；印记可在同意下「休眠」。  
- **假 HE**：只有身体满足，静仍操控同伴记忆。  
- **物证**：印记、补食碗、巫女铃、民俗笔记缺页、祠堂小灯、未寄出的学会信。  
- **系列接口**：めるてぃーりみっと 合集其他短篇可作平行一周目，勿无铺垫并村。

**【关系计量】**
- **信任（静）**：是否在你说停时收翼。  
- **调查进度**：笔记是否从「淫魔＝敌」改成「淫魔＝可谈判主体」。  
- **失败态**：失忆、被赶出村、或把印记照片寄去学会。  
- **停止词**：「待って」「今夜は帰って」被遵守＝HE 条件。

**【三夜结构细目】**
第一夜建立「收留—夜訪—印记」因果；第二夜把神圣祓除拖进共犯结构；第三夜揭晓静的淫魔身份并迫使契约者在「猎巫」与「共栖」之间选择。写正文时每夜只推进一个关系变量：夜1＝印记可见；夜2＝神职裂缝；夜3＝真名与谈判。

**【民俗调查伦理】**
門部的笔记不是经验条，是道德账本。他写下村民的冷淡、静的善意、桜歌的夜、蓮華的铃——若最终选择公开论文导致猎巫，则走 BE。HE 要求他学会「有些真相只写给自己」。

**【淫印的情感读法】**
印记亮起＝秘密被共享；印记发烫＝欲望上升；印记在白天被衣领遮住＝社会脸优先。禁止写成升级纹章。

**【共栖立法草案（可入正文）】**
①外来者须两人以上或持村许可；②夜訪必须可退出；③精气交换须双方清醒同意；④巫女不得以祓除名义强迫；⑤违反者由静亲自出面协商而非猎杀。

**【扮演调度备忘】**
写正文优先「气味—物件—一句短话」：补食热气、印记、铃。静白天温柔完整，夜里才短促；桜歌不要白天突然软；蓮華保持仪式腔。时间压力用「第三夜前」「学会回信期限」「村祭」。场景收束停在未完成动作：手停在印记上、铃举到一半、信封未封口。

**【调查日程的七天写法（入世用）】**
Day1 进村被拒、静收留；Day2 白天碰壁、夜里桜歌；Day3 印记焦虑、求蓮華；Day4 药草准备、村民闲话升温；Day5 祠堂试探；Day6 第三夜对峙；Day7 出村或留下。每天只改一个关系变量，禁止一日内从陌生人跳到共栖终局。写正文时，Day1 的关键词是「碗」，Day2 是「门响」，Day3 是「铃」，Day4 是「药草苦」，Day5 是「灯」，Day6 是「角」，Day7 是「石标」。用物件锚定比用旁白宣布「他爱上了她」更稳。

**【静的公开脸与饥饿隐喻】**
静的温柔不是单纯伪装，是捕食策略与真孤独的混合物。她需要精气，也需要「有人愿意被她照顾」的叙事。HE 不是饿死她，是让她承认「饿可以协商」。写她时避免扁平恶女：她会为门部多盛一勺汤，也会在揭晓后发抖。她的公开脸是「村里唯一肯说话的人」；她的缝是「角露出来时仍想被叫静而不是淫魔」。契约者若只消费她的身体却在茶话间式闲话里把她当怪物展览，好感冻结为切割。

**【桜歌的昼／夜切换脚本】**
白天：短句、不看眼睛、把柴放重。夜里：先敲门三下、先问「可以吗」再靠近——若契约者删掉这句问，路线倾向 BE。印记出现后，她会用「这是诅咒你的错」推责，再用「别告诉静」求同盟。她的萌点不是单纯强势，是**强势者第一次害怕白天的自己**。攻略她要在井边递水、在夜里给退出句，并在次晨不揭穿她的冷脸。

**【蓮華的铃与神职破产恐惧】**
铃是职业身份。祓除成功＝社会信用；祓除失败＝被村民怀疑。她用神圣话术接近门部，既是任务也是欲望。攻略关键是把「神职」与「人」分开谈：你可以尊敬铃，也可以拒绝身体。她若主动说「我也有印」，不是堕落完成，是信任升级。契约者若把祓所变成纯情色场景而无仪式尊重，她会用铃终止并封锁后续线。

**【門部笔记的可观察条目模板】**
①拒绝次数；②谁第一个说话；③碗的数量；④印记形状草图；⑤铃响的时刻；⑥自己的心率（主观）；⑦是否准备寄学会。这些条目应出现在正文，而不是旁白宣布「他在调查」。笔记的最后一页若从「淫魔＝敌」改成「淫魔＝可谈判主体」，即 True 向条件达成一半。

**【共栖后的日常切片】**
若选 HE／共栖：清晨井边三人保持公开脸；夜里印记可「约谈」；村口石标旁多一块「外来者须两人同行」的木牌。木牌是关系物证，比告白更重要。静会学着在白天也偶尔不完美——比如补食咸了还笑。桜歌会在夜里少一点表演性的强硬。蓮華会把铃放在门边而不是一直握在手里。

**【失败态的可执行描写】**
失忆 BE：门部在村外醒来，包里只剩空碗。猎巫 BE：学会来人、静被网、桜歌与蓮華被押。工具化 BE：印记还在，但三人不再对视，只说「今晚轮到谁」。写失败必须给出可观察证据，禁止只写「坏结局」三字。

**【气味与声音进度条】**
炉灰→补食味噌→榻榻米潮气→药草苦→铃→角蹭木→契约草稿墨。声音：拒绝沉默→碗筷→门响→铃→呼吸→笔停。正文若连续三场缺少气味与物件，会滑向空喊伝奇。

**【与 OVA 上下卷对齐】**
上卷偏进村、夜這い、印记初现；下卷偏揭晓与同场。档案默认完整三夜，可按玩家进度只开上卷相位。声优与作画差分不改变因果。写「同场」时优先关系功能：谁先看谁、谁先说停、谁在事后帮谁整理衣领。

**【学会与猎巫的外部压力】**
学会不是反派组织，是「把异常写成论文」的正常学术机器。门部的道德选择在于：公开＝可能救其他外来者，也可能毁掉这座村的所有白天脸。契约者若扮演记录员，开局就带着交差压力，HE 更难，也更有戏。

**【夜這い作为地方脚本的伦理改写】**
原作夜這い带伝奇强制感；入世档案要求任何夜訪保留可退出句。契约者可把夜這い改写成「村里年轻人的约会习俗」并要求静公开规则——这是共栖立法的前奏。删除可退出句的写法一律按失败伦理处理，即使场面「成功」。

**【角色对话指纹（防 OOC）】**
静：句子完整、尾音软、很少命令式，揭晓后才会出现短促的「别走」。桜歌：白天单字或短句，夜里句子变长但仍别扭。蓮華：敬语与祝词残留，紧张时铃先响。門部：提问多、陈述少，害怕时仍先写笔记。

**【祠堂对峙的可写节拍】**
1 灯亮；2 角露；3 静承认饿；4 桜歌与蓮華到场；5 门部出示笔记或药；6 有人说停；7 谈判或崩溃。七拍缺一不可。若跳过「停」直接到身体，判失败伦理。节拍之间插入可观察细节：灯油味、翼扫落灰尘、铃落地、印记同步发烫。

**【外来者第二次来村】**
HE 后若再有外来者，静必须遵守木牌规则；契约者可协助「接待」而非猎杀。这是 True 向的日常化考试：她能不能在不饿死自己的前提下不伤害下一个门部。考试失败的可观察证据是：新外来者失忆、木牌被拆、印记在白天强制发亮。考试成功的证据是：新外来者被安全送出村口，静只在夜里低声说「我有点饿」，并接受拒绝。门部若在场，应把这一幕写进笔记最后一页，标题只写两个字：「守住。」这四个字是 True 的签名。

## 休闲切入点

> 本世界为休闲／伝奇恋爱向（成人）。契约者以**日常身份**融入，核心玩法＝调查＋关系谈判＋可退出契约，而非任务厮杀。

切入身份：最顺是 **門部幸太郎本人（可改名）**。次选「同行助手」「邻村药师」「被静捡到的迷途旅人」「学会派来的记录员」。忌一上来自称最强除灵师。

切入时点：**刚进村、被拒宿、静开口帮忙前 10 分钟**——蝴蝶最大。也可切入印记出现次晨。

初始处境：无旅馆、包里有笔记与干粮、对谣传半信半疑。第一日目标：找到愿意说话的人、记下三处拒绝、决定是否接受静的一宿。

开场白建议：「土路上最后一个农夫把脸扭开。你的笔记本还翻在『男勿独行』那一页。静从井边抬起脸，说『先吃饭吧，调查的事明天也来得及』。碗沿的热气模糊了她的表情。你是先道谢，还是先问：这座村里，夜里敲门的都是谁？」

可攻略对象：  
- **静**：从接受收留与尊重她的节奏切入；钩子是「帮忙」背后的饿。吃诚实谈判，不吃羞辱。  
- **桜歌**：从白天被冷待仍礼貌、夜里给退出句切入。  
- **蓮華**：从认真对待祓除仪式但不消费神职切入。  

日常玩法钩子：  
1. **民俗笔记线**——每夜只记一件可观察异常。  
2. **印记观测线**——印记何时亮、谁先遮胸口。  
3. **祓除改写线**——把お祓い改成双方同意的边界仪式。  
4. **共栖立法线**——与静约定外来者规则，停止猎杀式夜訪。  
5. **清晨井边线**——白天如何称呼昨夜的人。  
6. **学会信线**——写或不写、寄或不寄，决定 BE／HE。

氛围／雷区：保持和风湿气与秘密甜腻；**忌**数值除灵；忌无同意；忌未成年。NSFW 由玩家开关，情感优先。

切入补充·调查包清单：  
笔记本、铅笔、干粮、退烧药、民间护符材料、备用草鞋。缺「护符材料」则第三夜谈判更难。

切入补充·称呼政治：  
白天称「静さん」「桜歌さん」「蓮華さん」；夜里若被允许可名字。淫魔真名是否公开是 True 条件之一。

切入补充·失败修复：  
若失忆 BE，可从村口闲话「又有个学者走丢了」重开，记忆以梦残片回收。

切入补充·开场第二版：  
「祠堂的铃在风里自己响了一下。静站在你身后，声音仍温柔：『调查得怎么样了？』你的胸口印记发烫。你是把笔记合上说『今晚想听你说实话』，还是假装什么都没发现继续吃饭？」

切入补充·第一夜选项树：  
A 拒绝夜訪并锁门（安全，延后线，静会观察你是否「有用」）。B 接受但要求「天亮前离开」（测试边界）。C 接受并问印记（加速调查）。D 羞辱（BE 倾向）。每个选项后必须写次晨井边的对视差。

切入补充·第二夜祓除脚本：  
蓮華要求宽衣／近身时，契约者必须有可退出句。若无退出句却继续，路线标记为失败伦理，即使字面「成功祓除」。可执行退出句示例：「铃先放下」「我保持坐姿到你说完祝词」「今天只到问诊」。

切入补充·第三夜谈判筹码：  
筹码不是武力，是：①笔记是否烧毁；②是否告诉学会；③是否允许静继续「帮助」外来者；④桜歌与蓮華是否恢复自主夜訪权。谈判成功的可观察结果＝村口出现木牌、印记可休眠、三人清晨仍能互相叫名字。

切入补充·身份变体·药师：  
邻村药师可提前持有护符材料，换取静的信任速度加快，但会触发村民「你是来抓妖的吧」敌意。药师线适合走共栖立法，不适合装除灵英雄。

切入补充·身份变体·记录员：  
学会派来的记录员开局持有「必须交差」压力，BE 风险更高；HE 条件是学会认可「不公开」的伦理报告。记录员的第一冲突往往是：静问「你会不会把我写进书里」。

切入补充·一周目标清单（可贴世界卡）：  
①三处拒绝记录；②一次补食；③一次印记目击；④一次停止词被遵守；⑤一份共栖草稿或一封未寄学会信。五项齐＝可进第三夜 True 向。缺任何一项不得强行揭晓淫魔真名。

切入补充·开场第三版：  
「后山溪边有人洗衣，看见你就沉了布。静却把干毛巾递过来：『外来者也会脏。』毛巾上绣着小角一样的纹样——也许是装饰。你收不收？」

切入补充·关系计量扩展：  
信任静 0–10（停翼次数）；桜歌防备 0–10（白天对视秒数）；蓮華神职裂痕 0–10（是否主动提「我也有印」）；学会压力 0–10（信件进度）。任一度极端（0 或 10 持续三日）触发事件，不得无声跳结局。

切入补充·失败速览与修复：  
失忆＝村口重开；猎巫＝需另周目潜入救人或接受村毁 BE；工具化＝先恢复对视与名字，再谈身体。修复优先级永远是「名字与停止词」，不是「再来一次夜」。
