# 黑兽帝国终焉-世代繁衍
<!--meta lib=主库 tiers=六-->

## 剧情

**【作品来源】**
《黑兽 ～气高的圣女们被白浊染指～》（黒獣 〜気高き聖女は白濁に染まる〜，Kuroinu）是 Liquid（Nexton）于 2010 年 4 月 23 日发售的十八禁凌辱向 AVG，后有 ERE 重制、2018《黒獣・改》、续作《黒獣II 〜淫欲に染まる背徳の都、再び〜》（2018）、《黒獣2・改》（2021）与外传。原作核心：佣兵王**沃尔特（ヴォルト）**率黑犬佣兵团反噬七盾同盟，建立强制女性性奉仕的「奉仕国家」。续作将舞台推至建国百数年后的**尼格拉姆提亚帝国（ニグラムスティア）**，建国王沃尔特已成为传说。本档案锚定「帝国终焉·世代繁衍」——不是单场破城，而是**奉仕秩序如何靠生育、调教、法令与教科书把一代征服写成下一代常识**的地区级制度战。六阶对应地区/大城级灾难与帝国制度机器。

**【世界定位】**
セレヌス大陆西方エオス平原。七盾同盟以七砦锁北境，盟主高等精灵女王**塞蕾丝汀·卢克卢斯**；暗精灵女王**奥莉加·迪斯科迪亚**率魔军南侵数百年。沃尔特受雇奇袭黑之城、俘虏奥莉加后吞并魔军，宣建奉仕国家。本世界时点落在**七盾已多座陷落、奉仕法令开始在占领区落地、第一批「名门展品」被押送帝都雏形、第二代混血与奴隶生育政策启动**的「繁衍期」：军事胜利已发生，真正可怕的是**把胜利变成世世代代的日常**。契约者切入的是制度与血脉的战场，不是只有攻城。

**【世界观 · 力量体系】**
力量来源：武技与骑士道、精灵/暗精灵魔力、巫女异能、魔兽与亚人蛮力、魔导拘束具、帝国法令与恐惧政治。没有现代热武器；战争靠军团、城塞、个人武勇、背叛与「展示性征服」。

等级表现：
1. ** lev民 / 民兵**：农具短矛。≈一阶。  
2. **正规士兵 / 佣兵新人**：守墙垛。≈二阶。  
3. **精锐骑士 / 魔军士官**：裂盾断甲，破小寨。≈三阶。  
4. **名将级团长 / 黑犬骨干 / 帝国百夫长**：一城胜负手，毁建筑群。≈四阶。  
5. **传说姬骑士（阿丽西亚位阶）/ 砦主将 / 魔军大干部**：左右要塞，重创军团。≈五阶。  
6. **佣兵王沃尔特 / 塞蕾丝汀地区指挥 / 盛期或半解放奥莉加 / 帝国制度机器运转下的地区会战**：覆盖大城与整片战区。≈六阶（本世界主战场）。  
7. **女神拉伦提亚完全神格 / 跨世纪帝国级灾厄**：更高，情报顶点。

乐园阶位映射：民兵≈一阶；士兵≈二阶；精锐≈三阶；名将≈四阶；姬骑士团长≈五阶；沃尔特·塞蕾丝汀·盛期奥莉加·地区制度战峰值≈六阶；完全神格=情报顶点（规避·条件性胜利）。宁低勿高；只写六阶。

死亡与「繁衍」规则：凡人战死即止；高阶精灵寿长，可被魔力拘束与长期凌辱造成「灵魂磨损」；奉仕国家优先用性、公开羞辱与**强制生育登记**摧毁意志与血统纯粹性。奥莉加对自身施受胎封印却仍遭持续凌辱——魔导政治样本。世界顶点不可用削弱解释。

**【地理 · 舞台】**
- **一至七之砦**：阿丽西亚、普莉姆、辉夜、玛雅、露露、克劳迪娅等分守；繁衍期多座已陷或危，沦陷砦被改造成「登记所+展示场+生育营」。  
- **白之城**：塞蕾丝汀坐镇，预知指挥；难民营与圣堂接收逃出的孕妇与孤儿。  
- **黑之城**：原魔军本营，新魔军/奉仕国家权力核；拘束奥莉加处。  
- **帝都雏形 / 战利品都市**：公开奉仕法令的橱窗；名门展品押送终点；混血户籍科所在。  
- **エオス平原**：军团拉锯；粮道=繁衍资源运输线。  
- **奴隶市场与「授田」区**：军功与人体资源的交易所。  
- **百年后遥指（II）**：尼格拉姆提亚帝都、建国王神殿——繁衍期成功与否决定教科书内容。

**【世界剧情线】**
① **百年对峙与七盾**  
暗精灵魔军与人类·精灵同盟军大战数百年。塞蕾丝汀被称女神拉伦提亚转生，以预知调度全军。七砦锁北境：一砦姬骑士**阿丽西亚·阿尔克图尔斯**、二砦花月姬君**普莉姆·菲奥里雷**、三砦巫女长**辉夜**、四砦女佣兵**玛雅**（前黑犬）、五砦半身人长**露露**、六七砦骑士团长**克劳迪娅·莱凡坦**（塞蕾丝汀护卫）。

② **雇佣黑犬与反噬**  
魔军魔力显衰，塞蕾丝汀重金雇沃尔特。黑犬突入黑之城，俘虏奥莉加并以弱化拘束具制衡。沃尔特随即吞并魔军，向同盟宣战，宣布奉仕国家：女性须对不特定男性性奉仕。新魔军（黑犬+魔军）以掠夺、公开凌辱、政治收买并行。

③ **破城连锁与第一批展品**  
一砦大臣**贝亚兹利**卖城换阿丽西亚；二砦**摩尔塔德拉兄弟**（猪人商人）卖普莉姆；四砦贵族**米歇尔**逆恨玛雅；克劳迪娅线有养父**格拉夫**与子嗣压力；大司教企图给塞蕾丝汀扣「骗国魔女」卖国。军事上砦陷，政治上「高贵失败者」被制成展品——帝国合法性来自展示。

④ **世代繁衍期启动（本篇核心）**  
占领区颁布：奉仕法令、生育登记、混血户籍、军功授田与「战利品分级」。目标不是一夜屠尽，而是：  
- 让名门女体成为公开符号，摧毁地区反抗意志；  
- 用生育把征服写入下一代血脉与姓氏；  
- 用教科书把「圣女如何学会奉仕」写成历史；  
- 用授田把士兵绑在扩张与看守生育营上。  
奥莉加受胎封印使其遭受持续凌辱却难孕——魔导与肉体政策的残酷对照；人类/精灵俘虏则被纳入登记。第一代混血与「营中出生者」开始出现时，繁衍期进入不可逆。

⑤ **白之城与预知的空白**  
塞蕾丝汀预知在「门从内开/人心背叛」处空白。难民营涌入从生育营逃出的人，民称「女神为何不应」。教会投机派与真正救济派分裂。克劳迪娅要保护「还是人的塞蕾丝汀」；辉夜的异神信仰可能拒绝拉伦提亚独占叙事。

⑥ **暗精灵变量**  
奥莉加高贵好战，被俘后恨人类也恨沃尔特；亲卫**克洛伊**（半暗精灵）忠主憎人。繁衍期若帝国试图用暗精灵血脉做「强化混血」实验，克洛伊会先抢主。拘束具松动是六阶级战略事件。

⑦ **与II的因果桥**  
百数年后尼格拉姆提亚帝国仍在，建国王沃尔特传说化；淫魔女帝**拉德米拉**、黑犬骑士团长**迪雷克·隆德**等登场。世代繁衍篇的成功=II的舞台前提；若在此打断登记、烧毁户籍、救出关键母体与孤儿、摧毁教科书草稿，则帝国「起源神话」开裂，II式长期压迫的难度上升。

⑧ **可介入终局方向**  
- **焚册**：烧生育登记与混血户籍，断制度记忆。  
- **破营**：武装撤离生育营，护送至白之城或山地。  
- **斩展示**：劫名门展品押送队（阿丽西亚/普莉姆等），毁帝国橱窗。  
- **反间**：用贝亚兹利型密约反噬卖城网。  
- **松拘束**：风险极高地接触奥莉加线，换暗精灵内乱。  
- **失败**：登记完成、第二代出生、教科书定稿→通向II。  
- **伪胜**：救了人但册在副本；烧了册但展品已公开，民心已碎。

⑨ **蝴蝶边界**  
可改：某砦陷落时间、某人是否成为展品、登记是否完成、第一代混血是否被帝国收养。不可无代价改：沃尔特以暴力与性秩序建国的主逻辑、奉仕国家作为制度的存在、顶点神格的规则。

**【势力图谱】**
- **七盾同盟残军**：守境与救济，塞蕾丝汀、克劳迪娅等。  
- **奉仕国家/新魔军**：沃尔特、希克斯、基恩、黑犬+魔军。  
- **砦内叛徒网**：贝亚兹利型，决定城门与登记所位置。

- **教会投机派 vs 救济派**：神权话语战。  
- **奴隶商与授田兵**：经济与恐惧的毛细血管。

- **奉仕法令诏书 / 铜板**：意识形态根。  
- **生育登记册与混血户籍**：繁衍期核心物证。

- **弱化奥莉加的魔导拘束具**：制衡魔军钥匙。  
- **圣鸢团长剑 / 各砦旗**：象征资本。

- **贝亚兹利密约等卖城书**：反间用。

**【隐藏剧情 · 伏笔】**
- 预知对人心背叛空白→登记官与叛徒同构。  
- 奥莉加受胎封印 vs 人类生育营：两种残酷。  
- 第一代混血的身份政治→II帝国社会结构。  
- 建国王传说靠繁衍期展品与册。  
- 女神分魂（II赛拉菲娜等）在难民营孤儿梦中可伏笔。

**【大事记时间线】**
百年对峙→雇黑犬→黑之城陷→奉仕宣言→砦陷连锁→**生育登记启动（本篇）**→展品押送→第二代出生→帝都成型→百年后II。

**【叙事基调 · 雷区】**
残酷战争+制度恐怖+贵族伦理崩坏。忌把奉仕写成无痛的情色乌托邦；忌美化叛徒；忌低阶斩杀沃尔特；忌忽略生育与户籍的政治性（本篇核心不是单挑）。锚点＝第一份生育登记在沦陷砦盖章之夜，或名门展品押送队出发时。

**【制度细节：奉仕国家如何「繁衍」】**
1. **展示**：公开凌辱名门，摧毁「可反抗」的想象。2. **立法**：奉仕为法，拒者为叛。3. **登记**：受孕、生产、混血姓氏归帝国。4. **授田**：看守营区的兵获得土地，叛变成本上升。5. **教育**：下一代读「圣女如何学会奉仕」。6. **经济**：奴隶价随破城波动，人体是资产。契约者摧毁任一节点都改变扩张速度；全毁才能叫打断世代。

**【一砦到七砦在繁衍期的状态模板】**
一砦：展品橱窗原型，登记所设在原贵族厅。二砦：商人叛徒引入「以身抵债」契约。三砦：巫女信仰与帝国神权冲突，可能屠杀或改宗。四砦：前黑犬玛雅的旧识网络可策反。五砦：半身人工艺被征用造拘束与囚车。六七砦：克劳迪娅若在，护卫线最后完整；格拉夫型压力使子嗣政治渗入。

**【战区资源四项】**
粮、水、药、士气；另加本篇独有第五项：**册**（登记完整度）。册升则帝国下一代稳；册毁则只剩暴力无制度。

**【名场面清单（繁衍期）】**
登记官在阿丽西亚纹章下盖章；普莉姆求援信被鹰杀；奥莉加拘束室与人类生育营并列剪辑；塞蕾丝汀拥抱难民婴儿时预知空白；沃尔特在黑之城颁布授田诏；教科书草稿上第一句「圣女曰」；克洛伊劫囚车只为女王。

**【可玩任务链】**
1. 烧一砦登记所。2. 劫展品押送。3. 护送孕妇至白之城。4. 窃拘束具图纸。5. 反间贝亚兹利余党。6. 毁教科书工坊。7. 策反玛雅旧部。8. 向塞蕾丝汀证明「预知被政治污染」。9. 破坏授田碑。10. 记录证人名册送地下反抗（接后续篇）。

**【经济与恐惧】**

**六阶（奉仕登记落地夜 · 世代繁衍阻断线）**

切入身份/时点：契约者以「白之城密使」「地下救济军顾问」或「黑犬叛逃军需官（假身份）」切入，锚定至少一座名砦（优先一砦）已陷、第一份生育登记册在占领区盖章前后的七十二小时；展品押送队与授田诏试点并行。

初始事件：你在沦陷的一砦贵族厅看见登记官把阿丽西亚·阿尔克图尔斯家的纹章描进「奉仕户籍」栏。隔壁仓库锁着待产与待押的女俘；城门希克斯清点囚车。塞蕾丝汀·卢克卢斯的密信要求：「救下一代，烧那本册——剑可以再铸，名册不能留下。」贝亚兹利余党在酒窖销毁密约副本。你必须选择：当夜烧主册并暴露行踪，或先劫押送队救出关键母体再追副本；也可假投登记处调包名册。

开场白建议：「纹章还金着，墨已经黑了。登记官的笔尖在『母名』一栏停住，像在给帝国写未来。城外平原上囚车的铃响成一条河。你摸到袖中的火折子与白之城蜡封——这一夜，若名册活过黎明，下一代会把锁链叫做法律；若你让火先说话，沃尔特会亲自来看是谁敢烧他的未来。」

关键NPC立场：**沃尔特**——要制度与橱窗，可开「你献册我留她们体面」的假条件；**塞蕾丝汀·卢克卢斯**——要难民与下一代，远程军令可能过时；**阿丽西亚·阿尔克图尔斯**——若仍在，要真相与剑，厌怯懦；**普莉姆·菲奥里雷**——求援与被保护，宣传战焦点；**克劳迪娅·莱凡坦**——护送优先，护人先于护神；**希克斯 / 基恩**——执行登记与清场；**贝亚兹利余党**——灭口持密约者；**奥莉加·迪斯科迪亚 / 克洛伊**——仇视人类但可恨沃尔特；**玛雅**——旧识变量，可策反；**辉夜 / 露露**——信仰与器械支援；**大司教**——投机卖神女风险。

主线钩子/支线：主线＝烧毁或夺出生育登记与混血户籍主册+至少一份副本，并破坏至少一个展示橱窗（劫名门展品押送或毁诏书铜板/授田碑）；支线＝护送孕妇与孤儿至白之城、反间卖城网、窃拘束具情报、毁教科书草稿、策反玛雅旧部、向二砦送预警、与克洛伊短交易。蝴蝶点：是否公开处决登记官——士气爆但可能提前引来沃尔特亲临六阶压场。

危险度/规避：极高。规避沃尔特正面单挑；规避魔军主力军团中央；女神完全神格仅情报。展示场与登记所有重兵，优先渗透与夜袭。

任务方向/奖励：烧册、破营、劫囚车、反间、护民、毁教科书。奖励＝密约副本、授田名册残页（反用）、同盟军功、破营器械图纸、地区级武具与「空白户籍页」象征物。禁止直接击杀战区外神格本体；禁止无准备广场宣战灭国。

战术补充：
- 夜袭登记所优先于强攻城墙。  
- 假投黑犬送假名册，调包真册。  
- 截断粮道使营区自乱再救人。  
- 露露系工艺砸囚车轴。  
- 宣传战：被救婴儿对抗「圣女奉仕」画报。  
- 雨季泥泞劫囚；旱季尘伪装拆碑。  
阶段：T-72侦册与副本位置→T-24部署内应→T-0盖章夜三刀齐砍→T+24清副本与追押送→T+72白之城交接与伪胜检查（册是否仍在）。

第二钩子：你在酒窖捡到半页未烧尽的贝亚兹利密约，背面竟是「登记优先权」转让——卖城者已预定繁衍红利。交阿丽西亚·阿尔克图尔斯看会点燃刃之觉醒；交塞蕾丝汀·卢克卢斯可稳白之城；自己留着可勒索余党开城门。三选一改变地区颜色。

第三钩子：基恩的报表抄本从黑之城流出，写明「一砦孕数、逃数、册页」——那是帝国心跳。你可交给玛雅煽动旧部哗变，或交给露露计算砸哪一辆囚车最伤报表，或当众贴到胜利街神坛下逼沃尔特提前亲征（高风险高收益）。

阶段检查表：T-72是否定位主册+副本；T-24是否买通/清除至少一名本地合作者；T-0是否三刀齐砍（册/橱窗/授田）；T+24是否清口述户籍；T+72白之城交接是否附伪胜四项否定答复。任一项「否」则任务评价不得完美，但可带伤过关。
