# 黑暗圣经深渊篇-城市沦陷

## 剧情

**【作品来源】**
《黑暗圣经》（Bible Black，日文バイブルブラック，又译黑魔术学园）是 ActiveSoft 于 2000 年 7 月 14 日发售的十八禁视觉小说，原画·脚本圣少女（Sei Shoujo）。后由 Milky Studio 改编为多部成人 OVA：本篇《Bible Black: La Noche de Walpurgis》（六集，2001–2003）、前传《Bible Black: Origins / 外传》（二集，2002）、续作《新·Bible Black / New Testament: La Lanza de Longinus》（六集，2004–2007）及 Only 特别版。本档案锚定「深渊篇·城市沦陷」——以本篇终章「黒の降臨」后的溢出态与新约中「都市连续情杀·特搜介入·银行劫案仪式」为核，将黑魔术从学园地下室扩到整座城市的灾厄窗口做成五阶任务世界。文风：黑魔术恐怖、校园伦理崩坏、都市灵异与公权力失效。人名地名忠于公开资料。

**【世界定位】**
表面仍是日本普通高中与周边街区，地下与历史却是黑魔术结社 Rose Cross 的祭坛。十二年前瓦普吉斯之夜，处女献祭失败却打开地狱反应，北见丽华与恶魔缔约续命；十二年后水无濑多喜在地下室发现《黑暗圣经》，黑魔术回流。本世界处于「城市沦陷」：学园门已开，咒文沿欲望网络扩散到便利店、公寓、警局与十字路口；失踪海报一夜翻倍，性交中发生的残酷杀人案在市区连环出现；特搜侧（新约伊万里线）开始介入，却发现「案件现场」与「学园保健室」是同一规则的两张皮。契约者要决定：把门压回学园、把城市当隔离区，还是眼看整座城变成常设巢穴。

**【世界观 · 力量体系】**
力量本源是**黑魔术 / 卡巴拉式恶魔学与契约**，媒介为《黑暗圣经》及衍生咒文、祭坛、处女/血/性行为作为能量回路。魔法在此不是闪光特效，而是**改写欲望、肉体与灵魂归属**的规则攻击：恋爱咒、强制发情、肉体改造、附身、夺舍、开门。城市沦陷期的特点是：**低阶咒可自动施放**，像空气里漂着未完成的句子。

等级表现（破坏力与规则权能并重）：
1. **普通人 / 普通师生 / 路人店员**：无抗性，易被暗示与群体幻觉左右。≈一阶。
2. **初习术者 / 结社新人**：低阶咒（魅惑、小占卜、恶作剧级强制），伤及个体。≈二阶。
3. **正式结社成员 / 持残页者 / 被长期中咒的「放大器」**：中阶仪式，影响班级至楼层、一条街的集体失控。≈三阶。
4. **高阶术者 / 前结社干部（高城宽子级知识）/ 持完整书页的水无濑**：可改写多人关系与肉体，威胁整栋学园与相邻街区。≈四阶。
5. **与恶魔缔约的容器（北见丽华）/ 降临中的恶魔分身 / 朗基努斯之枪级圣遗物介入时的战场 / 城市级精神灾厄**：一击可造成街区级精神与物理崩坏，打开稳定地狱门缝，让警察与特搜火力失效。≈五阶（本世界主战场）。
6. **撒旦/契约本尊完全体、原典级魔导书觉醒态**：战略级。≈六阶+，本档仅情报顶点。

乐园阶位映射：普通人≈一阶；初习≈二阶；结社正式成员≈三阶；高阶术者/完整书持有者≈四阶；缔约容器·降临分身·枪与城市级灾厄≈五阶；撒旦本尊与完全开门=情报顶点（规避本体·条件性胜利）。宁低勿高；本条目只写五阶。

特殊规则：处女性作为献祭与夺舍关键燃料；非处女献祭会导致仪式「扭曲成功」——城市沦陷正是扭曲成功的扩散形态；性行为可成为施法姿势；拉丁-法语-希伯来混合咒文需翻译才能稳定施展；烧书不等于断根，若灵魂契约仍在，灰烬也会笑。死亡：肉体死可被夺舍绕过；灵魂被恶魔收走则角色退场。世界顶点不可用「被封印所以变弱」解释，只能打断仪式链、替换祭品逻辑、封门、隔离街区与撤离。

**【地理 · 舞台】**
- **学园主楼与地下室贮藏室**：全书原点。废弃房间藏《黑暗圣经》与旧祭坛；城市沦陷期，地下室红光透过排水管渗到校外下水道。
- **保健室**：北见丽华的猎场，处检记录与「下一具壳」名单的起点；沦陷后成为城市级「欲望诊所」的模板。
- **美术教室 / 美术部**：高城宽子与水无濑、伊万里的日常锚点；宽子在此回忆十二年前。
- **学生会室、游泳池、更衣室**：白木里香、游泳部等支线欲望场，咒文最早的社会实验场；沦陷夜这些子系统同时短路。
- **结社集会室（Rose Cross / 新魔术部）**：佐伯香织重建的圈子，塔罗与魔导书流通点。
- **校外仓库 / 十二年前行刑地**：前传中北见被轮暴与献祭的场所，与学园地下室双门共鸣；城市沦陷时仓库区出现与学园同步的失踪。
- **市区十字路口与钟塔**：午夜钟塔倒走；灯火在幻觉中连成五芒星。
- **便利店、公寓走廊、地下停车场**：普通人最早「自动中咒」的场所——店员反复擦同一块玻璃，租客梦见自己翻书。
- **警局与特搜（Tokken）办公室（新约时间线可叠合）**：伊万里胡桃作为灵能搜查官调查性交中残酷杀人与幽灵；银行劫案中的即兴仪式可使北见再占伊万里，朗基努斯之枪从金库进入城市棋盘。
- **银行金库 / 枪保管节点**：新约核心地理，城市沦陷任务可将「学园门」与「枪之门」叠成双锚。
- **地狱之门投影**：不是固定坐标，是规则浓度最高处——地下室、仓库、银行劫案现场、连环案现场可轮流成为门缝。

**【世界剧情线】**
① **十二年前·起源（Origins）**  
高城宽子学生时代与好友森田理惠、新藤早纪沉迷神秘学，在古董店获得《黑暗圣经》。学生会长小园奈美与恋人持田润子掌控学生会，拒绝她们成立社团。结社以咒文作弄持田（公开失控）、再应男生平谷请求对奈美施恋爱咒，又用解放咒让奈美清醒并仇恨。奈美反过来加入并夺权，社团更名 Rose Cross，计划召唤恶魔。试召以奈美杀害的小狗为祭，失败反噬宽子。瓦普吉斯之夜，奈美以转校生北见丽华为「处女祭品」；事前北见遭奈美指使的暴徒侵犯，其中一人破坏处女条件。仪式反应却不完全失败：门动了，奈美屠杀社员，重伤的北见反杀奈美，并在濒死时接受撒旦交易——以灵魂换数年续命。宽子逃过屠杀，余生背负罪与知。城市沦陷的「双门结构」在此埋下：学园地下室与校外仓库共享同一扭曲成功。

② **十二年后·发现书与学园内扩散（本篇前中段）**  
水无濑多喜，普通高中生、美术部副部长，在地下室发现《黑暗圣经》。堂姐水无濑由起子协助翻译拉丁-法语内容。多喜试咒：对同学伊藤美香等施术，对白木里香施恋爱咒使其痴迷，又帮村井亚由美撮合——黑魔术以「满足欲望」为糖衣。保健教师北见丽华接近他，诱导其依赖书；同班佐伯香织痴迷神秘学，重建结社并企图夺书。北见需要处女容器以在瓦普吉斯之夜夺舍续命，目标曾包括白木与伊万里胡桃；白木因与多喜发生关系失去资格。北见附身/操控多喜，随其参加佐伯集会并制服佐伯，夺下新结社主导权；同时折辱已识破她的高城宽子。伊万里发现多喜与白木后缺席学校，精神防线松动，成为可掳目标。

③ **本篇终局·黒の降臨（城市沦陷的点火）**  
低阶恶魔离开多喜身体后，他认清自己爱的是青梅竹马伊万里胡桃，决心救人。高城宽子提供她认为能阻止北见的咒文。瓦普吉斯之夜：北见割腕与伊万里交合以完成夺舍，多喜闯入施咒打断。表面上北见失血而死。事后多喜与伊万里确认心意；佐伯企图盗书；伊万里进入祭坛室以火咒毁书——随即露出北见仪式「其实成功」的征象：伊万里成为新容器，并转向侵犯佐伯。官方动画终局即是**恶魔侧的胜利以最亲密之人的脸完成**。城市沦陷篇将此夜之后的数日拉长：门未关，书未真正死，胡桃/北见复合人格以学园为巢，咒文沿人际关系与下水道、电话线、电视雪花溢出校门。

④ **溢出第一周·街区症状**  
失踪海报一夜翻倍；十字路口出现集体性幻觉（行人突然当众失控，重演持田润子式公开羞辱）；教堂十字架倒悬却无人敢扶；收音机自动播出拉丁咒片段；警察无线电出现十二年前旧案编号。便利店店员循环同一句「欢迎光临」却看不见客人；公寓楼道有人反复按已搬走住户的门铃。这些不是背景板，而是五阶「城市级」的可观察读数。

⑤ **特搜与连环案（新约逻辑接入）**  
时间可叠合或作为「若学园门未关十年后必然发生」的加速版：白木与佐伯成为教师，高城一度出家，伊万里成为特搜侧灵能搜查官，调查性交中发生的残酷杀人与幽灵（朱迪·克劳利相关）。佐伯再结社；银行劫案中的即兴仪式使北见再度占据伊万里；朗基努斯之枪被夺，处女「绯衣之女」秋成为容器争夺焦点。城市沦陷任务世界允许把「本篇降临后溢出」与「新约都市案」叠在同一灾厄窗口：门开到最大的数小时至数日，整座城同时是学园与战场。

⑥ **城市沦陷态·世界正在发生什么**  
- 学园：夜间课程取消却有人强制到校；影子在无光处移动；保健室床位出现非人类的湿润痕迹；塔罗牌自己洗牌指向死神与恶魔连张。  
- 仓库区：十二年前死者残影（森田、新藤、奈美残响）在巷口拦人「借火」。  
- 市区：钟塔午夜倒走；灯火幻觉成五芒星；连环案现场与地下室血纹同构。  
- 公权力：普通警察当「集体癔症」处理；特搜能看见门缝却火力有限；引入媒体会把学园变成新闻地狱并加速欲望网络。  
- 结社：佐伯派与北见派与（若叠新约）朱迪派在城市层抢容器、抢书页、抢枪。

⑦ **可介入终局方向**  
- **封城式封门**：以宽子一脉封咒+正确处理书（封存优于莽撞焚烧）+切断仓库共鸣+隔离潜在处女容器，把门从「开」压回「缝」，城市症状在黎明衰减。  
- **夺人优先**：在复合人格稳定前隔离伊万里/北见，代价可能是多喜灵魂受损或某街区被当作诱饵烧掉。  
- **枪与银行线**：若叠新约，阻止枪与绯衣女嵌合，避免「学园门+枪门」双开。  
- **交易**：与中间阶恶魔谈条件——必写清代价，禁止免费午餐。  
- **失败态**：学园成为常设巢穴，城市周期性瓦普吉斯化，特搜队全灭，乐园再入时直接更高压力。  
- **伪胜态**：门关了但北见已换壳离开；屏保式隐患（新约末屏保闪过北见威胁）留到下一任务；城市表面恢复，失踪者名单却永不结案。

⑧ **日常可介入层（优先于冲顶点）**  
翻译残页、抢在佐伯前控制魔导书流通、保护仍是处女的潜在容器、在保健室安装非电子（防咒）监视、护送宽子不被再掳、切断血祭物料链、向特搜递交「非癔症」证据、在便利店与公寓做低阶咒消毒、护送由起子保持翻译链、拆仓库旧阵防双门。优先这些，而不是喊着冲撒旦。

⑨ **蝴蝶效应边界**  
可改：某街区是否沦陷、某人是否失踪、伊万里是否在当夜被完全固定为容器、枪是否进入棋盘、佐伯是否直播仪式。不可无代价改：扭曲开门的历史成因、北见缔约续命的根、瓦普吉斯作为时间锚的规则、顶点（撒旦收债）的存在逻辑。改了局部，原作因果会以「追杀、换壳、更惨的城市案」回流。

**【游泳部与街区嫉妒链】**
野野草美由纪、小林忍、美树本构成的嫉妒三角，在学园内是咒的小型社会实验；城市沦陷时同构事件出现在公司夜班、公寓邻里、便利店排班表。契约者清魔不清人祸，门关了城市仍是地狱。伊藤美香对咒过敏可作早期传感器；村井亚由美代表「自愿卖灵魂换小事」的普通人逻辑，其请托网络是城市欲望外包的毛细血管。

**【地理作战层补强】**
地下室外间堆废弃桌椅，内间才是祭坛；墙内夹层可藏人，也是残影钻出的缝。主楼楼梯在沦陷夜会出现「多走一层」的拓扑错误——你以为到一楼，推门却是天台。保健室抽屉未登记处检记录是猎名单。美术教室石膏像在红光下假哭，宽子用它当情绪锚。游泳池夜间空无一人，水面却映仓库屋顶。校门外便利店店员反复擦同一块玻璃。下水道连接学园排水与仓库渗水点，是双门的物理隐喻。钟塔维修口可俯瞰五芒星幻觉灯火，也可成为狙击佐伯观礼的位置——但开枪会乱血纹。

**【与封印仪式篇接口】**
城市沦陷解决「门已渗到哪、读数如何降」；封印仪式解决「四锚如何逆向关闸」。两篇人物与四锚一致：本篇偏街区症状、特搜封锁、媒体风险与伪胜的城市版检查；封印篇偏发音、铅盒、收债几何与仓库同步信号。契约者可先压街区读数再进祭坛，或先断仓库再封学园，顺序错误会双开。

## 阶位切入点

> 阶位↔战力：五阶≈恶魔缔约容器（北见）·降临中的恶魔分身·朗基努斯之枪介入战场·城市级精神/物理灾厄。世界顶点（撒旦本尊/完全开门）为情报顶点：规避·条件性胜利，严禁削弱式解释。

**五阶（城市沦陷窗口 · 封城封门与夺人线）**

切入身份/时点：契约者以「特搜编外顾问」「转校插班的神秘学研究者」或「被高城宽子秘密叫来的外援术者」身份切入，锚定本篇瓦普吉斯之夜之后至黎明后的城市溢出高峰——伊万里已显容器征象或特搜线连环案爆发、街区读数爆表的十二至四十八小时内。

初始事件：你在钟塔下看见指针倒走，便利店电视雪花里闪过北见的笑。高城宽子把烧焦边的封咒纸按进你掌心：「学园的门已经渗到下水道了。水无濑那孩子还想救她——你若只想看热闹，现在离开；若留下，就准备用灵魂当抵押，并且先决定救的是『人』还是『这座城』。」与此同时佐伯香织在公寓楼组织观礼式集会，特搜无线电里旧案编号自动跳出，仓库方向有人喊「奈美学姐」。

开场白建议：「钟塔在午夜倒走。保健室的床是空的，垫单上却有新鲜的血与不属于人类的温度；校门外的失踪海报一夜贴满整条街。高城老师的声音发哑：『北见要的不只是一具壳，她要整座城给她续命。你跟我去仓库切共鸣，还是跟多喜去地下室夺人——选错，黎明时你会看见十字路口的人一起开始跳舞。』」

关键NPC立场：**水无濑多喜**——只要能救伊万里可结盟，但会抢着持书，可能把城市当代价；**伊万里胡桃**——容器/特搜双态，时间窗口极短；**北见丽华**——视你为额外祭品或有趣变量，城市是她的零件箱；**高城宽子**——给封咒与双门真相，要求你不重蹈玩火；**佐伯香织**——抢书与观礼优先于救人；**水无濑由起子**——电话那头可提供最后一句拉丁译文；**弓削**——可给封锁线，但怕媒体；**朱迪·克劳利**（若叠新约）——与北见争夺门控与枪；**秋**——绯衣女风险，保护对象。

主线钩子/支线：主线＝把城市级门开压回「缝」以下并尽量夺还容器人格；支线＝切断仓库共鸣、控制魔导书城市流通、保护处女容器、阻止佐伯直播仪式、（可选）阻断枪-容器嵌合、向特搜递交非癔症证据。蝴蝶点：是否告诉多喜「救下的可能已不是原来的她」——告知可能让他崩溃误咒并加速街区失控，不告知可能让他亲手把容器带出封印圈污染下一区。

危险度/规避：极高。避开撒旦本尊显化核心；不在门内对视超过三秒；不用自己的血随意补阵；不在媒体直播下施高阶咒。顶点：情报优先/条件性胜利。

任务方向/奖励：封城封门、夺人、毁书或封书、双门切断、提取反缔约知识、特搜临时权限。奖励＝残页级咒文（有代价标注）、封咒完整式、对「欲望规则攻击」的抗性片段、铅盒封存技术、街区封锁经验。禁止无代价全能破契与「一拳打爆撒旦」。

补充战术：
- **强闯祭坛**：与多喜分两侧进，一人断血纹一人护施咒；若北见以伊万里为人盾，优先破肉体锚而非砍容器。  
- **仓库共鸣切断**：带宽子笔记，用盐与反转咒拆旧阵；遭遇十二年前死者残影。  
- **城市消毒**：对便利店/公寓低阶自动咒做拉丁收语清扫，降低群体失控率。  
- **夺书小组**：对佐伯可交易「活着离开」，换取她不在十字路口开观礼。  
- **翻译链**：电话保持由起子在线，线路会被拉丁回声干扰，需复述校验。  
- **收债时刻**：门开过大时黑影点名缔约者；让北见/朱迪成为焦点，契约者退到封咒圈外。  
- **黎明窗口**：日出衰减一档，仍可能留下容器人格——黎明是换班不是结束。

关键抉择：1. 救伊万里优先 vs 封城优先。2. 告诉多喜真相 vs 隐瞒。3. 杀北见肉体 vs 困住待收债。4. 引入特搜/媒体 vs 纯地下解决。5. 是否把枪线拉入本窗口。

战场阶段：T-48h 街区读数上升；T-24h 宽子接触与仓库预警；T-6h 佐伯集会与特搜介入；T-0 门峰与收债影；T+6h 黎明衰减与伪胜检查。
