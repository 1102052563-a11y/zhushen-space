# 乙女理論とその後の周辺 -Epilogue-

## 剧情

**【作品来源】**（~200 字）
《乙女理論とその後の周辺 -Belle Epoque-》（おとめりろんとそのごのしゅうへん ベル・エポック；本作清单条目亦作 Epilogue）是 Navel 于 2016 年 5 月 27 日发售的 PC 美少女视觉小说 / 后日谈粉丝盘（FD）。系列前作为《月に寄りそう乙女の作法》（つり乙）与《乙女理論とその周辺 -Ecole de Paris-》（乙りろ）；官方明确要求先通关前作再入。角色原案西又葵 / 铃平ひろ，剧本延续系列时装学园 + 大蔵家门政治 + 女装女仆喜剧线。气质：巴黎时装日常、家族和解、甜宠后日谈与一条完整改写的「真·エッテ线」；含 H，尺度成人向、量偏少且多带幽默。曾获 Moe Game Award 2016 粉丝盘奖。整体不是从零开的恋爱，而是「巴黎之后」的余韵与补完。

**【世界定位】**（~150 字）
一句话：这是 Navel「乙女」系列巴黎篇的后日谈世界——舞台仍在菲利亚女学院巴黎本校与大蔵 / プランケット周边，核心玩法是**时装创作与秀场压力 + 异母兄妹 / 幼驯染 / 令嬢三角的情感收束 + 大蔵家督余波**。契约者以日常身份（转校生、助手、住家女仆、实习裁缝等）融入，不写压力强度，只写关系、秘密与成衣针脚。

**【世界观 · 舞台设定】**（~800 字）
时间落在《乙女理論とその周辺》各女主路线结局之后，或エッテ线改写分岔点附近：巴黎仍是时装之都，空气里是塞纳河湿气、咖啡与裁缝室的浆糊味。**菲利亚女学院（フィリア女学院）巴黎本校**由传奇设计师 **ジャン＝ピエール・スタンレー** 创办，课程重立体裁剪、面料学、品牌企划与秀场走步，学生多为欧洲旧贵族与新兴财阀子女；日本分校（前作舞台）在此被当作「分校生的回声」，偶尔以通信、客串、日本 Philia Collection 特邀的形式回流。

社会规则不是武力，而是**身份与目光**：女仆可以进小姐卧室却不能上家督宴席正座；模特可以站在 T 台中央却在家族投票里没有一票；女装身份一旦被公开，会牵动学园名誉、媒体与大蔵家「失踪重要人物」的搜寻令。**大蔵家**是跨日欧美的巨大财阀，老当家年迈，**家督争い**长期在 **大蔵衣遠** 一系与 **大蔵駿我 / 大蔵アンソニー** 一系之间拉锯；失踪的 **大蔵遊星** 本人是可以撬动天平的棋子，却选择以女仆 **小倉朝日** 的假名活在妹妹身边。

超自然设定为零。世界的「温度」来自：深夜 atelier 的台灯、缝纫机节奏、秀前最后一针、塞纳河边并肩走时谁先伸手、以及大蔵宅宴会上礼貌笑容背后的沉默。失败态是关系破裂、品牌信誉崩盘、家族公开撕破脸、或女装身份被恶意曝光——不是死亡竞赛。H 场景存在但服务于情感确认（恋人间的身体诚实、兄妹禁忌的羞耻与依赖），正文不展开露骨描写。

系列前置因果（入世时可作背景注）：遊星本是大蔵家不受待见的庶子，在曼彻斯特等地被隔离培养，因容貌酷似母亲、无喉结等特征极易女装；前作中以女仆身份侍奉樱小路宅，性别暴露后被逐，衣遠收留，后被异母妹 **大蔵りそな** 重新要回并带往巴黎本校，再次以「小倉朝日」身份入学圈。

**【地理 · 生活舞台】**（~500 字）
- **菲リア女学院巴黎本校 · 特别编入班教室**：りそな、メリル、エッテ、ディートリンデ、リリアーヌ等同窗日常；公开脸与小动作（低头缝样衣、传纸条、午休吃点心）的主场。
- **本校 atelier / 裁缝工房**：布料卷、人台、蒸汽熨斗；メリル的天才与りそな的哥特萝莉灵感在此碰撞；夜半加班是告白与崩溃高发点。
- **プランケット家别墅（エッテ宅）**：りそな与朝日留学初期的落脚处；大厅水晶灯与女仆通路并存，主仆 / 主从身份在此被不断重写。
- **Maison de Papillon 系 atelier / 工作室（蝴蝶馆意象的私人工房）**：品牌草创与样衣试穿；后日谈中りそな个人品牌与メリル专业化工程的核心舞台。
- **大蔵家巴黎侧宅 / 宴会厅**：家督投票、亲族会面、衣遠与駿我暗流；安东尼的轻快与沉重在此同时可见。
- **巴黎 Collection 会场 · T 台后台**：大奖、走秀、媒体闪光；りそな Grand Prix 与エッテ模特成长的名场面坐标。
- **塞纳河堤 / 巴黎公园步道**：牵手散步、对比前作公园 CG 的成长镜头；告白与和解的「慢场景」。
- **日本 Philia Collection / 日本组来访会场**：后日谈里りそな作为特邀的压力源；前作角色（如樱小路ルナ等）短暂客串的接口。
- **修道院回忆空间（メリル身世闪回）**：不常驻，却是メリル「灰姑娘入大蔵」线的情感原点。
- **咖啡馆与车站月台**：转校日、送别、临时约谈的过渡区；第三人视线最容易刺破女装秘密。

**【故事主线 · 情感线】**（~1800 字 · 大头）
本作结构几乎无中途选项，标题菜单直接选剧本。四条可玩块：① **ブリュエット・ニコレット・プランケット アナザーストーリー**（エッテアナザー，主菜）；② **大蔵りそな アフター**；③ **メリル・リンチ アフター**；④ 通关アナザー后解锁的 **エッテアフター**。推荐顺序：先アナザー，再三条 AFTER。

**共通前提（前作巴黎篇收束）**  
りそな从日本宅女成长为巴黎编入生，带着女装女仆朝日（遊星）进入菲利亚本校；结识缝纫天才 **メリル・リンチ** 与旧贵族令嬢 **ブリュエット（通称エッテ）**，以及军装风的 **ディートリンデ・ツヴァイゲルト**、贵族感的 **リリアーヌ・セリア・ラグランジェ**、パターン师 **華花** 等。主线压力始终双轨：① 时装课业与秀场；② 大蔵家督与「失踪的遊星」之谜。各路线是 what-if 分岔，无单一正史；Epilogue 则在各 HE 之后或エッテ改写线上补完。

**エッテアナザー（本作最大补完 · 完整改写线）**  
分岔点在前作エッテ线中段附近：エッテ更早、更主动地察觉 **小倉朝日 = 大蔵遊星** 的性别秘密，不是被草草揭穿，而是在共同备赛、模特试穿与家庭危机中一步步确认，并把秘密当作「要守护的人」而非丑闻。衣遠在此线对遊星 / りそな转为高压对抗姿态；エッテ站到二人一侧，动用プランケット家与同学网络缓冲。

家族线被大幅加厚：**大蔵アンソニー** 获得接近正角的戏份——他轻快、爱吃（豚カツ ネタ贯穿）、对堂兄 **駿我** 怀有复杂的憧憬与恐惧，努力在家督局里做「不让血溅到身边人」的缓冲；与エッテ对 **メリル** 的偏执守护形成镜像。高潮包括：秀场 / 步行（ウォーキング）场面中エッテ作为ヒロイン・学生・模特三重身份的自觉；以及对家族投票现场的介入——不是武力打断，而是用立场、情报与人情把天平拨回「允许遊星以自己的方式活」。结局 HE：エッテ与遊星关系确立，模特事业起步，家族至少达成阶段性停火。H 场景自然，エッテ侧带轻微 S / 主导色彩。评者普遍认为此线质量追平甚至超过前作りそな / メリル本线，是「真正的エッテ线」。

**大蔵りそな アフター**  
承接りそな本线：巴黎 Collection **Grand Prix** 之后，りそな被推为**下任大蔵家当主候选人 / 事实上的下一代门面**。家庭表面走向团结，暗处仍有衣遠—駿我轴线的余震。りそ一边要维持个人品牌（哥特萝莉 / 高辨识度设计语言），一边陷入创作低谷，又被日本 Philia Collection 以「特邀」名义压上更高期待。遊星（朝日）在身边从「被照顾的兄」渐渐变成「被她牵着手走的搭档」——后日谈名 CG 与前作公园牵手形成对照：从前是她依赖他走出房间，现在是她主动拉他走向会场。日本组客串不损面子，ルナ戏份被控制在不抢戏的范围。结局 HE：品牌与家门压力下她仍选择「做衣服的自己」与「爱兄的自己」并存，关系稳定甜蜜，H 与日常里保留调皮妹属性。

**メリル・リンチ アフター**  
承接メリル本线：她以大蔵血脉身份被纳入家族（灰姑娘式接纳）。身世核心——父母死于与アンソニー / 駿我父辈相关的旧怨，**衣遠**将幼年的她安置修道院并长期以监护人自居——在后日谈中转化为「如何在豪门里保持无垢针线」的课题。亲族与友人（含エッテ、遊星）合力把她「只会凭感觉做出神级成衣」的野路子，推成可发布、可定价、可秀场化的专业流程。恋爱面补前作「情感未熟」：她仍慢半拍，却会在关键时刻用缝好的衣服代替告白。结局 HE：她在家族中找到位置，才能不被异化成商品，与遊星的关系轻柔而坚定。篇幅较短，气质最治愈。

**エッテアフター**  
紧接アナザー：エッテ模特 / 杂志露出成功，事业上升。情感上出现与 **りそな** 的三角余波——りそな一度设计不顺，原因与「エッテ—遊星」关系有关，遊星心里清楚却必须在妹妹与恋人间走钢丝。冲突烈度中等，重在关系诚实而非狗血拆散。收束 HE：三人关系在时装共同体框架下找到可呼吸的距离，エッテ以杂志与秀场实绩证明自己不只是「有钱的令嬢」。

**系列接口与后日谈总调**  
全线强调：巴黎不是战斗地图，是**成衣与眼神的战场**；大蔵家督是人情剧不是宫斗厮杀；女装身份是喜剧与紧张的双重引擎。通关后另有愚人节式短剧彩蛋。与《つり乙2》等后续作时间线松散相连，本档案以 Belle Epoque / Epilogue 四剧本为边界。

**【可攻略角色 / 主要人物】**（~2200 字 · 核心 · ≥6 人）

**大蔵遊星 / 小倉朝日（主角 · 女装女仆 · 设计师志望）**｜外貌：中性美人，继承爱尔兰裔母亲的柔和轮廓，无显眼喉结，女装时黑长直 / 齐肩姬发皆可成立，女仆装与便装切换自如；男装时仍偏纤细。｜性格：温柔、勤奋、多语言（英/法/德/俄/中等）、家务全能、内心对家族既惧且义。｜角色类型：女装主人公 / 奉献型兄长 / 吐槽役与被吐槽役一体。｜萌点/魅力：被识破前后的落差、为妹妹与恋人不惜继续女装的觉悟、裁缝与料理双手都很稳。｜个人线剧情：Epilogue 中在 AFTER 多任「支持位」，在エッテアナザー则深度卷入家督与恋情；必须在「大蔵遊星」真名与「朝日」假面之间反复选择站哪一边。｜与主角关系：契约者可投射为遊星本尊，或以转校助手身份与朝日并列——后者需尊重其女装秘密的知情权规则。

**大蔵りそな / 大蔵里想奈（异母妹 · 下任门面设计师）**｜外貌：哥特萝莉向私服、长发、精致人形感；学园制服下仍难掩大小姐气场。｜性格：外冷内粘、重度兄控、曾是引きこもり，巴黎后变得敢在公开场合牵兄之手；创作时任性又认真。｜角色类型：病娇边缘的妹系 / 傲娇大小姐 / 设计师。｜萌点：对兄「所有权」宣言与真正为他梦想留学的反差；Grand Prix 后的责任崩溃与重建。｜个人线剧情：AFTER 中作为准当主兼品牌主理人，在日本特邀秀与家族团结压力下确认「衣服与兄」二者皆不可弃；HE。｜与主角关系：血缘 + 恋爱双重绑定（系列核心禁忌甜）；攻略关键是承认她的主体性，而非只把她当需要被救出的宅女。

**メリル・リンチ（缝纫天才 · エッテ付き人 · 大蔵血脉）**｜外貌：娇小、幼气脸、常素朴发型与实用装；手指有长期针茧。｜性格：无垢、慢热、欲望极低、对衣服诚实到近乎神圣；不擅长豪门辞令。｜角色类型：天然 / 天才工匠 / 灰姑娘。｜萌点：用成衣代替语言；被家族包围仍保持修道院式清澈。｜个人线剧情：AFTER 写专业化与入族适应；身世（父母遇害、衣遠监护、修道院）是心结也是她拒绝仇恨的理由；HE。｜与主角关系：亦师亦友的共同制衣者 → 恋人；吃「认真看她针脚」这一套。

**ブリュエット・ニコレット・プランケット（通称エッテ · 旧贵族令嬢 · 模特/女優志望）**｜外貌：橙长发、蓝瞳垂眼、约 160cm、曲线明显；制服与礼服都带华美饰品，偶有迷你礼帽 / 丝带。｜性格：奔放、亲切、不拿身份压人、略对金钱钝感、对喜欢的人很大方；表面轻快，认定目标后极粘着。｜角色类型：元气令嬢 / 轻百合倾向（对メリル）/ 主动攻。｜萌点：为メリル入学的主从故事；识破朝日后仍选择守护；T 台与投票场的双栖决断。｜个人线剧情：アナザー为完整恋爱 + 家督介入线；AFTER 为事业上升与りそな三角收束；双 HE。｜与主角关系：从「妹妹的同学」到「知道一切仍伸手的恋人」；吃真诚与勇气，不吃敷衍身份游戏。

**ディートリンデ・ツヴァイゲルト（通称リンデ · 同班 · 军装风）**｜外貌：金发或浅色系、姿态笔直、制服常被改出纪律感。｜性格：严格、护短、口吻像小军官；对主从秩序敏感。｜角色类型：凛然御姐系同学 / 吐槽。｜萌点：与付き人ヴァレリア的主从日常。｜个人线剧情：Epilogue 戏份有限，多作学园侧压力计与喜剧节拍；可作好感配角线展开「纪律 vs 巴黎自由」。｜与主角关系：初期怀疑朝日身份 → 尊重其实力后的战友式信任。

**リリアーヌ・セリア・ラグランジェ（通称リリア · 同班贵族）**｜外貌：优雅长发、贵气笑容、着装一丝不苟。｜性格：礼貌、观察力强、身世线在本 FD 未充分展开但带同情底色。｜角色类型：大和抚子欧版 / 观察者。｜萌点：与パターン师華花的主从搭档感。｜个人线剧情：Epilogue 中多为社交场 NPC；契约者可挖「贵族联姻压力 vs 学园友情」。｜与主角关系：社交入场券与情报源。

**大蔵衣遠（异母兄 / 非血缘兄长位 · 天才设计师 · 家族棋手）**｜外貌：冷艳成人男性向，设计师气场。｜性格：极度理性、掌控欲强、对遊星既保护又工具化；路线不同可成盟友或对手。｜角色类型：腹黑兄 / 最终压力源。｜萌点：把メリル送进修道院的「扭曲温柔」。｜个人线剧情：エッテアナザー中为对立轴；りそな AFTER 中为必须共治的家人。｜与主角关系：血缘名义复杂；不可用武力解决，只能用立场与谈判。

**大蔵アンソニー / 大蔵駿我（家督对立侧 · 表亲系）**｜外貌：安东尼偏轻快贵公子；駿我更沉、更像继承人。｜性格：安东尼善良带怯意、努力讨好又恐惧；駿我冷硬务实。｜角色类型：情敌位家族 / 可同情反派。｜萌点：安东尼豚カツ梗与对駿我的忠。｜个人线剧情：エッテアナザー大幅升华安东尼；投票与宴会是关键。｜与主角关系：可敌可谈；忌简单脸谱恶役。

**ジャン＝ピエール・スタンレー（学院创始人 / 导师）**｜身份：巴黎时装教育顶点象征。｜性格：艺术家脾气与教育者耐心并存。｜功能：课业与秀场权威认证，不介入恋爱本身。

**華花 / ヴァレリア・テレンチェヴナ・ブッテルスカヤ 等付き人组**｜華花：リリアーヌ侧パターン师，手艺稳。ヴァレリア：リンデ付き人，主从喜剧担当。均为学园生活密度的保证。

**前作客串：桜小路ルナ 等日本组**｜在りそな AFTER 的日本 Collection 接口短暂登场，提醒系列从「月に寄りそう」走来的长线友谊，不可写成抢主线的第二女主。
