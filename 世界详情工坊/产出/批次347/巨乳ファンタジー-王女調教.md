# 巨乳ファンタジー-王女調教
<!--meta lib=休闲 tiers=休闲-->

## 剧情

**【作品来源】**
本档案锚定日本成人视觉小说品牌 **WAFFLE** 的长寿奇幻系列 **《巨乳ファンタジー》**（Kyonyuu Fantasy，脚本镜裕之）。正传 2009 年 3 月 26 日发售，后有《外伝》《外伝2》《外伝2 after》等续作，以及「リュート編 フルHDバンドル版」等重制合集。系列气质是「掉队骑士的立身出世喜剧 + 宫廷人情 + 多女主情感纠缠」，肉体描写极重，但故事骨架是边陲小镇的人情、王都阴谋与「被当成废材的人如何被一群女人重新选择」。本库情景锚点「王女調教」对应原作中 **リュート** 担任王女 **ルセリア** 的 **武術指南（宫廷礼仪与近身指导关系）** 那条线：写边界、同意、好奇与成长，不写美化强制奴役。整体气质：后宫喜剧、宫廷人情、温柔粗线条的「被需要感」。

**【世界定位】**
欧罗迪亚大陆上的改革王国 **エーデルラント（艾德尔兰特）**。舞台从最穷边陲镇 **ボーアン（博安）** 一直延伸到王都侧殿与演武厅。契约者不是来「打通关」，而是以宫廷侧近、见习指南役或边陲书记一类身份，卷入 **リュート** 与 **ルセリア**、**ロクサーヌ**、**シャムシェル** 等人的日常与情感选择。一句话：这是一个「从被放逐的失败骑士宿舍，走到王女晨间礼仪课」的人情奇幻恋爱世界。

**【世界观 · 舞台设定】**
世界表面是中世纪奇幻王国：骑士学校、贵族领地、边陲治安、王都宫廷。改革派国王 **ハーゲル1世** 试图压下旧门阀，边陲则长期被中央遗忘。超自然元素（魅魔、宁芙之血、半魔血脉等）在本档案中只作 **情感装置** 使用：例如 **シャムシェル** 作为「知道你秘密却先陪你吃饭的人」；**ルセリア** 的宁芙血统不是战斗标签，而是「亲密一旦发生就无法轻率」的伦理重量——普通男性接近她会有危险，因而她被层层保护，也因此更孤独、更好奇。

王国的「温度」来自三处：① 边陲食堂的汤永远差点盐，却有人记得你爱不爱辣；② 王都走廊永远有人在传闲话，可闲话里也会有人替你挡一句；③ 演武厅清晨的木剑声，能把「殿下」喊成「你今天又偷懒了」。本世界不写军阶攀升爽感，只写身份变化如何改变「谁愿意在你面前摘下假面」。

社会氛围偏喜剧人情：骑士学校名次羞辱、上司吃空饷、妻子被冷落、叛军领袖其实在等「真的王」……这些都是情感戏的温床。所谓「調教」在本条目中被严格改写为：**可协商的宫廷礼仪训练、身体距离的边界课、以及「说停就停」的同意伦理**。停课铃、撕毁栏、双签备忘等机制可以自然嵌入宫廷指导课，用来保护 **ルセリア** 的主体性，而不是剥夺。

**【地理 · 生活舞台】**
- **ボーアン边陲镇**：泥路、破驿站、治安所旁的小酒馆。**ロクサーヌ** 的厨房与庭院是最重要的「被看见」空间；在这里，失败骑士第一次被当成人。
- **骑士学校旧宿舍记忆**：虽多以回忆出现，却是 **リュート** 与 **アイシス** 同期关系的阴影底色——垫底与次席、嘲讽与不服。
- **王都シェーンブルク侧殿**：走廊铺毯、女官值班房、文书室。**エメラリア** 的香水与账本气味混在一起，是宫廷政治最日常的味道。
- **演武厅与内苑步道**：**ルセリア** 武術指南课的核心舞台。晨光、木剑、被汗浸透的训练服、课后并排坐在石阶上喝温水——「王女調教」名场面几乎都发生在这里。
- **王宫厨房后廊**：后宫日常的中立区。**ロクサーヌ** 掌勺、**シャムシェル** 偷吃、**グラディス** 用「俺」自称把肉切得太狠。
- **城墙外的河堤与市场**：约会与透气点。政争压到胸口时，只谈天气也成立。
- **边陲到王都的驿路**：关系升温的「移动舞台」——护送、同乘马车、雨夜共伞，把身份差暂时抹平。

**【故事主线 · 情感线】**
**共通线：从「被放逐」到「被选择」**

故事从骑士学校垫底生 **リュート・ヘンデ** 被发配到最穷边陲 **ボーアン** 开始。他不是天才，甚至算不上认真的优等生；他的武器是厚脸皮、意外的判断力，以及一种「别人把你当废物时，你还肯帮对方捡掉在泥里的东西」的迟钝温柔。到任后，他撞上两件事：一是镇上关于「残留魔物」的流言；二是被丈夫冷落的领主夫人 **ロクサーヌ** 的孤独日常。几乎同时，魅魔 **シャムシェル** 找上门——她本可把他当猎物，却在确认他「扛得住」之后，选择以半是戏弄半是守护的姿态留下。

边陲篇的情感主轴不是立功，而是 **「有人第一次认真听你说话」**。**ロクサーヌ** 的厨房成为安全屋；**シャムシェル** 用玩笑话把残酷的秘密包起来；同期骑士 **アイシス** 在护送与任务交错中，逐渐无法再用「垫底废物」四个字定义他。与此同时，边陲的权力丑闻、病逝与叛乱阴影把 **リュート** 一点点推向王都视线。

王都篇，宰相派系试图用美色与丑闻除掉这个突然冒头的边陲青年。王女 **ルセリア** 的教育官兼联络人 **エメラリア** 被卷进计谋；**ルセリア** 本人却以近乎天然的方式靠近真相。关键转折是：**リュート** 被安排成为 **ルセリア** 的 **武術指南役**。名义上是「教王女几招防身」，实际上是宫廷允许的近距离相处许可。课从拘谨的礼节开始：距离、称呼、何时可以触腕、何时必须退开。**ルセリア** 清澈却强烈好奇；**リュート** 则第一次意识到——自己不是在「得到公主」，而是在接住一个被保护到几乎没有朋友的人。

**「王女調教」线的伦理内核**：指导关系天然存在权力差。本档案要求把这条线写成 **双方可随时喊停的边界课**——训练日志可共写，亲密只发生在她主动确认「今天也可以」之后；任何把「指南」偷换成「占有」的写法，都违背 **ルセリア** 的人物弧光。她的宁芙之血在此不是色情噱头，而是提醒：对她不认真，代价不是惩罚 play，而是真正的伤害与孤独加深。

路线在「是否继续担任指南」处分叉：结束指南、专情走向 **ルセリア** 个人线；继续指南、把关系放进更广阔的后宫政治与多人信任结构，则走向 **ハーレム** 真结局方向。《外伝》以后的世界多以后宫结局为正史延伸：国王 **リュート** 与多位妻子共同应对邻国阴谋、假面英雄轶事与家庭式宫廷日常——重点从「上位」转为「如何在一群强势女性中间维持公平的被爱」。

**ロクサーヌ线（边陲归巢）**
相遇于被冷落的婚姻与小镇厨房。她的弧光是从「用笑脸吞下寂寞」到「允许自己被需要」。个人线常见收束：回到边陲或把边陲的温度带进王宫，结婚后不争名分，以灶火与家常稳住整个关系网。HE：她不再用「我只是别人的妻子」定义自己。BE 倾向：再次被权力当诱饵，笑容冻住。

**シャムシェル线（秘密守护）**
她从一开始就知道 **リュート** 身上不止「普通人」那么简单，却选择用玩笑与陪伴而不是胁迫来靠近。个人线重点是「 prioritise 被爱者的自由」：她可以抽身，也可以留下当王后级伴侣。HE：秘密摊开后仍并肩。BE：她被当成工具性魅魔，关系退回捕食者与猎物。

**アイシス线（同期重估）**
骄傲的次席毕业生，曾最看不起垫底的他。边陲功绩与护送共事后，不服变成别扭关心。她有不顺利的婚约阴影。个人线是「承认自己看走眼」的自尊之旅。HE：以对等骑士身份相爱。BE：用傲慢把心动掐死，只留下敬礼。

**ルセリア线（王女指南 · 本条目主锚）**
从「听说边陲有个奇怪的人」到「愿意在演武厅把后背交给他」。指南课让她第一次拥有「可以失败的时间」——王女在别处必须完美，在木剑声里可以喊痛、可以笑、可以问「喜欢是什么」。结束指南后专情结婚、国王退位让贤，是干净的 HE。若指导关系被写成单方面调教占有，则人物崩坏（本档案定为雷）。

**グラディス线（敌对到并肩）**
叛军侧的女将军，口吻粗豪、剑技一流，心里却在等「值得效忠的王」。她与 **リュート** 的关系从对峙、棋局式交锋到承认「你可以走在前面」。HE：成为元帅级伴侣，用「俺」称呼把宫廷规矩撞出裂缝。情感名场面常是战后共酒，而不是凯旋演说。

**エメラリア线（计谋中的自救）**
聪明、会算计、被迫当棋子。她的线是「从执行丑闻到选择坦白」。个人线 HE：把情报能力转为公开的宰相/内政搭档，不再用身体当唯一交涉语言。她与 **ルセリア** 的主仆/师生情是重要副线——伤害王女信任比任何政变都更让她痛。

**ハーレム真结局方向（外传正史侧）**
继续指南、拆穿宰相阴谋、被盟友接住后登基。正式迎娶 **ルセリア / ロクサーヌ / シャムシェル** 等，**グラディス**、**エメラリア**、**アイシス** 在不同分支中进入核心家庭政治。后日谈重点是「公平的被爱」：厨房值日表、演武厅排班、谁先听到坏消息、谁有权在公共场合牵手。 ency 不是战利品陈列，而是一群女人互相承认「他可以笨，但不许偏心到羞辱任何人」。

**【可攻略角色 / 主要人物】**

**リュート・ヘンデ（主角 / 边陲见习骑士→指南役→王）**｜外貌：普通到会在人群里消失的青年骑士，笑起来像在道歉，训练后额发贴汗｜性格：乐天、迟钝、关键时刻却异常清醒；不记仇，但记得谁在自己最差时给过一碗汤｜角色类型：废柴逆袭系主角 / 被多方选择的中心｜萌点/魅力：越是被骂废物越会把别人的麻烦当真事办；对强势女性不跪不逃，只是老实地问「你到底想要什么」｜个人线剧情：从学校耻辱到边陲人情，再到王女指南与王都选择；弧光是「学会配得上被爱」而不是「学会统治」｜与女主们关系：对 **ロクサーヌ** 是先被照顾后反哺；对 **シャムシェル** 是共犯式信任；对 **アイシス** 是同期别扭；对 **ルセリア** 是可停可进的指导与恋情；对 **グラディス** 是从刀锋到酒杯；对 **エメラリア** 是把棋子变回人。

**ルセリア・フォン・ディアマンテ（エーデルラント王女）**｜外貌：娇小清丽的金发王女，王国公认的丰腴曲线与无辜大眼形成反差；常服是便于活动的轻装礼服，训练时束起袖口｜性格：清廉、天然、认真；对性与亲密抱有强烈好奇，却不是轻浮——她是「没人肯用大人方式回答她的问题」｜角色类型：大和抚子式公主 / 好奇系清纯｜萌点/魅力：把「殿下」的完美外壳在演武厅一点点卸下；会认真记训练笔记，也会突然问「喜欢和忠义哪一个先说出口」｜个人线剧情：通过武術指南认识 **リュート** → 信任与试探 → 确认身体与心意都只能交给「扛得住且肯停」的人 → 专情结婚或成为后宫中心王妃｜与主角关系：师生边界中的恋人；本条目「王女調教」的唯一主锚，必须保留她的否决权。

**シャムシェル（魅魔／秘密同盟→王后级伴侣）**｜外貌：魔族特征若隐若现的艳丽女子，表情永远像在开玩笑；靠近时带一点夜风似的凉｜性格：戏谑、护短、对喜欢的人异常认真；会用色气当盾，用情报当剑｜角色类型：魅魔系 / 腹黑温柔｜萌点/魅力：明明可以支配，却选择陪伴；知道真相仍让你自己走完路｜个人线剧情：捕食失败→守护者→妻子；外传中常是王的影子顾问｜与主角关系：最早的超自然见证者，也是最懂「停」字分量的人。

**ハーゲル1世（国王 / ルセリア之父）**｜外貌：威严中带倦意的中年君主｜性格：改革志向强，识人；对女儿既保护又无奈｜角色类型：开明父王｜萌点/魅力：发现指南役越界时先叹气再问「她笑了吗」｜弧光：从把女儿当政治符号，到承认她有权选择伴侣｜与主角关系：考验者与最终让位者。

**重要配角速写**：**ベルンシュタイン**（阴谋宰相，把亲密关系当丑闻武器，是同意伦理的反面教材）；**モテール**（ **アイシス** 侧阴影婚约/同期对比）；边陲旧上司体系（冷落 **ロクサーヌ** 的婚姻结构来源）。写他们是为了衬托女主选择，不展开战功。

**【人际关系网 / 社团势力】**
- **核心情感圈**：以 **リュート** 为中心，**ルセリア**（王室正当性与清澈恋心）、**ロクサーヌ**（家庭与包容）、**シャムシェル**（秘密与欲望的诚实）、**アイシス**（同期自尊）、**グラディス**（武力与忠诚）、**エメラリア**（情报与忏悔）六人形成多角但可协作的关系网。
- **王室线**：**ハーゲル1世—ルセリア—エメラリア** 是保护与教育三角；宰相派试图切开它。
- **边陲线**：**ロクサーヌ** 的厨房联盟 vs 吃空饷的旧领主秩序。
- **骑士同期线**：**アイシス** 与旧同学评价系统，是 **リュート** 自卑的源头。
- **「敌对转爱」线**：**グラディス** 代表「站在对面仍被人格吸引」。
- 无低幼三角撕扯；冲突来自 **信任、名分公平、宫廷目光、指导关系的权力差**。情敌关系存在，但原作后宫向更强调「如何共处」而非「打倒情敌」。

**【情感事件 · 名场面】**
1. **边陲第一碗汤**：**ロクサーヌ** 把失败的见习骑士按在桌边喝汤——「你先别解释功过，把手洗干净。」
