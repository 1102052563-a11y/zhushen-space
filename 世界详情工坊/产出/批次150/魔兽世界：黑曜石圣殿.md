# 魔兽世界：黑曜石圣殿

## 剧情

**【作品来源】**
《魔兽世界：黑曜石圣殿》（The Obsidian Sanctum，TOS/OS）是暴雪 MMORPG《巫妖王之怒》中的 10/25 人团队副本，随资料片推出（Patch 3.0.2 起），地点在**龙骨荒野龙眠神殿（Wyrmrest Temple）**下、泰坦之路裂隙进入的**龙眠联军层面殿堂·黑龙军团翼**。终局首领**萨塔里奥（Sartharion）**「黑曜石守护者」，及暮光幼龙副官**沙德隆、塔尼布隆、维斯匹隆**。本条目锚定**WotLK 黑曜石圣殿正典**（含 1/2/3 龙硬模式），不写红玉圣殿哈立东全本倒灌，不写大地的裂变死亡之翼落地战。文风：黑龙阴谋、暮光实验、熔岩与暮光双压、精锐小队史诗。

**【世界定位】**
龙眠神殿下的**黑曜石圣殿**：黑龙军团在龙眠层面殿中的巢穴与实验室。死亡之翼（耐萨里奥）的余党在此孵化**暮光龙**——以其他龙族精华实验出的扭曲新品种。红龙女王**阿莱克丝塔萨**在魔枢战争后令子女清查龙眠内奸，其伴侣**克里奥斯塔兹（科拉斯特拉扎相关调度）**向达拉然六人议会求援，英雄进入圣殿摧毁暮光卵与萨塔里奥。一句话：**在龙眠地基下，选择留几条暮光龙活着，再与黑曜石守护者共舞熔岩。** 卡片锚点：龙眠裂隙入口→三暮光龙台→萨塔里奥熔岩殿→（余波）红玉圣殿报复。

**【世界观 · 力量体系】**
力量本源：黑龙火焰与大地、暮光龙虚空—暗影混合、火元素、龙类魔法、凡人职业。死亡观：灭团可重置；叙事上暮光龙与萨塔里奥是地区级龙灾，非行星毁灭本体（死亡之翼另档）。特殊：**龙的数量是难度开关**——先杀 0/1/2/3 条副官再打萨塔里奥，存活副官越多越难、掉落越好（暮光领域、双龙、暮光区成就链）；熔岩裂隙、火墙、暮光传送门召唤信徒。

战力阶梯：
1. **龙眠外围/龙骨荒野杂兵**。≈一阶。
2. **龙眠联军辅助、英雄五人本**。≈二阶。
3. **圣殿内黑龙兽人/龙兽精英**。≈三阶。
4. **单条暮光幼龙（沙德隆/塔尼布隆/维斯匹隆）**。≈四～五阶。
5. **萨塔里奥（0 龙）**。≈五～六阶。
6. **萨塔里奥+1/2 龙**。≈六～七阶。
7. **萨塔里奥+3 龙（暮光区）**。≈七～八阶顶点。

乐园阶位映射：外围≈一；联军正规≈二；圣殿精英≈三；单龙≈四～五；萨塔里奥 0 龙≈五～六；+龙硬模式≈六～八。顶点 3 龙萨塔里奥＝八阶条件战。

**【地理 · 舞台】**
- **龙骨荒野·龙眠神殿**：红蓝绿铜黑五翼政治中心；入口在神殿下冰裂/泰坦路径。
- **黑曜石圣殿内部**：环形熔岩平台；三名暮光龙各据一侧；中央萨塔里奥。
- **关联**：黑龙圣地、暮光龙卵、红玉圣殿（事后报复，另档点到）；达拉然议会求援线。

**【世界剧情线】**

**① 前史·黑龙与暮光**  
耐萨里奥堕落为死亡之翼后，黑龙军团持续实验。暮光龙由其他龙族精华扭曲而成，曾被认为在格瑞姆巴托等事件后受挫，但卵在黑曜石圣殿被重新发现。

**② 清查内奸**  
魔枢战争后阿莱克丝塔萨要求清查龙眠。红龙发现圣殿内暮光卵，资源不足下通过科拉斯特拉兹等向达拉然求援，冒险者进入。

**③ 三龙选择**  
- **塔尼布隆**：卵与幼龙波次。  
- **沙德隆**：暮光传送门与信徒、庇护。  
- **维斯匹隆**：暮光领域与减益。  
可先击杀任意组合，或全部保留挑战萨塔里奥。

**④ 萨塔里奥**  
黑龙守护者，熔岩打击、火墙、裂隙；存活副官会在战斗中加入，显著提高难度与奖励。名场面：三龙齐啸、熔岩地板、暮光传送门。

**⑤ 余波**  
暮光卵被毁，黑龙军团后以**红玉圣殿**报复（哈立东线另档）。本档终局以**萨塔里奥与暮光卵威胁解除**为准。

**⑥ 主题**  
龙族内战的手术刀；难度自选即叙事自选；暮光=被窃的龙族未来。

**【主要人物】**
- **萨塔里奥**｜黑曜石守护者｜暴虐忠诚于黑龙事业｜熔岩与龙息｜圣殿终局。
- **沙德隆 / 塔尼布隆 / 维斯匹隆**｜暮光幼龙副官｜实验体与看守｜传送门/卵/领域。
- **阿莱克丝塔萨**｜生命缚誓者｜清查令来源。
- **科拉斯特拉兹等红龙**｜求援达拉然。
- **死亡之翼**｜幕后，不在本档可战。
- **龙眠联军各色龙族代表**｜政治背景。
- **黑龙兽人、龙兽、火焰元素**｜小怪层。

**【势力图谱】**
黑龙军团；暮光龙实验体系；龙眠联军（红主导清剿）；达拉然六人议会外援；天灾为同时代背景噪音非主敌。

**【贵重物品】**
萨塔里奥掉落史诗；多龙模式坐骑/稀有（如黑龙幼龙相关版本掉落叙事）；暮光龙卵（销毁对象）；龙类精华样本（肯瑞托研究违禁风险）。

**【隐藏剧情 · 伏笔】**
暮光龙技术通向 downstream 大地的裂变；红玉圣殿报复；龙眠五色政治裂痕；3 龙成就是公会招牌。

**【大事记时间线】**
死亡之翼堕落→暮光实验→魔枢战争→龙眠清查→发现卵→英雄入殿→可选屠龙数→萨塔里奥败→卵毁→黑龙报复红玉圣殿。

**【三龙与硬模式细录】**
塔尼布隆迫使团队处理卵与地面幼龙，像在踩一颗定时孵化器。沙德隆的传送门吐出信徒，庇护技能惩罚「集火不集火」的犹豫。维斯匹隆把暮光领域盖在人头上，减益叠到治疗崩溃。零龙萨塔里奥是熔岩走位课；一龙是「加减法」；二龙是编制考试；三龙是公会史诗——火墙、裂隙、三龙技能同时说话时，语音频道会变成另一种乐器。

**【龙眠政治】**
红龙要清剿，黑龙要实验，蓝龙余波尚在，青铜中立，绿龙沉睡 co 相关。契约者在龙眠广场公开炫耀暮光卵会被至少两色龙同时攻击。

**【与相邻条目】**
不写永恒之眼玛里苟斯全战、不写冰冠、不写死亡之翼落地；红玉圣殿仅作余波。纳克萨玛斯同时代可口头提起。

**【叙事基调 · 雷区】**
灼热、暮光、龙吼。忌萨塔里奥战力归零；忌忽略三龙开关；忌把死亡之翼拉进可击杀列表；忌套话。最早切入：龙眠裂隙外；或一龙台前。


**【三龙技能与叙事对应】**
塔尼布隆：卵与幼龙=未来被量产的恐惧。沙德隆：传送门与信徒=暮光教会式渗透。维斯匹隆：领域减益=认知与治疗被污染。萨塔里奥：熔岩火墙=黑龙「守护扭曲未来」的暴力。零到三龙是四档故事，不是同一场战斗的皮肤。零龙是熔岩走位课；一龙是加减法；二龙是编制考试；三龙是公会史诗。

**【龙眠政治显微镜】**
红龙清剿令；黑龙实验；蓝龙余波；青铜中立；绿龙事务另线。广场上炫耀卵=多色敌对。达拉然六人议会提供凡人打手合法性，也索要样本。红龙要少破坏建筑，因为龙眠是政治圣殿不是废墟。

**【难度自选即道德自选】**
为成就留龙可能增加伤亡；为进度屠龙可能失去顶级掉落与荣誉。团内成就党与进度党冲突是五～八阶核心社会事件。投票灭龙减负会作废当晚成就。

**【入口与外围】**
裂隙测量晶片、静音巡逻数步、火焰元素桥——低阶故事在此。热气同时有硫磺与墨水甜味，是黑龙与暮光的混合签名。

**【余波红玉】**
黑龙在卵毁后报复红玉圣殿；本档只给预警情报，不写哈立东全战。成功后的寂静里能听见上层龙眠的风。

**【死亡之翼边界】**
可作远距意志与笑法意象，不可作本档可击杀顶点。顶点是 3 龙萨塔里奥。向死亡之翼祈祷求胜应得到更残酷难度。

**【火与暮光的感官】**
硫磺、墨甜、熔岩光、错色鳞、火墙热浪、传送门紫黑。卵裂时有人说可爱有人说集火。

**【掉落与坐骑叙事】**
多龙模式更高奖励与稀有坐骑是公会史诗货币；写成物象与名号。肯瑞托密封卵壳若流出会引发五色听证会。

**【失败反馈】**
失败则卵被传送；龙眠上方政治压力上升；黑龙加速报复红玉。成功则圣殿威胁暂时解除，英雄名号进入龙眠记录。

**【契约者身份】**
雇工、学徒、静音渗透者、10 人精锐、硬模式指挥——按阶。

**【与纳克萨玛斯同时代】**
可口头提起天灾，但圣殿主敌是龙而非亡灵。永恒之眼、奥杜尔另档。

**【主题句】**
在龙眠地基下手术；留下几条龙，就是选择多残酷的未来。

**【萨塔里奥人物】**
暴虐、忠诚黑龙事业、视暮光实验为未来之翼；台词围绕守护与蔑视凡人。站起来时熔岩让路。

**【红龙与肯瑞托】**
科拉斯特拉兹等调度求援；要结果不要借口。肯瑞托要数据；样本外流=外交事件。

**【黑龙兽人与元素】**
巡逻改变路线考验静音；火焰元素核心里有小小龙影，像圣殿用火做广告。通桥时红龙观察员要求少破坏。

**【成就链叙事】**
击败黑龙军团、熔岩打击躲避、人少击杀、暮光协助/双龙/暮光区——每一档对应一种团队人格。

**【乐园阶位映射·复核句】**

萨塔里奥的熔岩殿不是孤立盒子：龙眠神殿上层五色政治、泰坦路径裂隙、黑龙圣地与暮光卵室连成一条因果链。契约者在上层广场公开炫耀暮光样本，会同时触发红龙清剿令与黑龙灭口。三龙存活开战时，火墙路线必须用人身标点，因为熔岩会吞掉地面标记。沙德隆传送门内的信徒若不被优先处理，会把治疗节奏打成碎拍；塔尼布隆的卵若漏爆，幼龙会涌向主坦身后；维斯匹隆的领域叠满时，再强的装等也救不回减益崩溃的团队。0龙通关是资格，3龙通关是史诗；两者掉落与名号不同，公会会用此区分「能进」与「能挂名」。红玉圣殿的报复在本档只作为预警：卵毁之夜，上层红龙侍从会送来密封信，信上只有一个词——「准备」。死亡之翼的笑法可在萨塔里奥临死语音里一闪，但禁止把死亡之翼拉进可击杀列表。肯瑞托要密封卵壳，龙眠要销毁，拍卖行要出价，三方拉扯是七～八阶支线。低阶只在裂隙外测温与护晶片，中阶静音清精英，高阶投票留龙或屠龙——投票本身就是事件。

**【黑曜石补足·人物与失败】**
萨塔里奥视暮光实验为「未来的翼」，台词应傲慢而非无脑。沙德隆、塔尼布隆、维斯匹隆是副官也是难度旋钮。阿莱克丝塔萨的清查令是合法性来源，科拉斯特拉兹等红龙是雇主姿态。失败则卵被传送、黑龙加速报复；成功则圣殿威胁暂时解除，英雄名号写入龙眠记录。同时代可提纳克萨玛斯，主敌仍是龙族实验而非天灾工厂。

## 阶位切入点

> 阶位↔：一≈龙眠外围；二≈联军辅助；三≈圣殿精英；四≈单暮光龙；五≈萨塔里奥 0 龙；六≈+1 龙；七≈+2 龙；八≈+3 龙。顶点 3 龙条件战。

**一阶（龙眠外 · 裂隙）**
切入身份/时点：龙眠雇工、地精爆破助手、肯瑞托学徒。
初始事件：裂隙渗出暮光雾，你的测量晶片录到「卵的心跳」——红龙要原件，黑龙奸细要销毁。
开场白建议：「龙眠的风从五色旗帜之间挤过来。裂隙里的热气有硫磺，也有像墨水一样的甜。」
关键NPC立场：**红龙侍从**——要证据；**肯瑞托学徒**——要数据；黑龙奸细——要你的命。
主线钩子/支线：护晶片；支线＝修补裂隙护栏。
危险度/规避：低；规避萨塔里奥殿。
任务方向/奖励：龙眠通行证、初级火焰抗性。

**二阶（入口走廊）**
切入身份/时点：小队队长。
初始事件：黑龙兽人巡逻改变路线，你要在无惊动中央龙的前提下清掉两波精英。
开场白建议：「熔岩光把斧影拉得很长。有人教你数巡逻步数：错一步，三只龙都会回头。」
关键NPC立场：**黑龙兽人队长**——敌；龙眠向导——要静音渗透。
主线钩子/支线：静音清场；支线＝标记卵室方向。
危险度/规避：中。
任务方向/奖励：精良装、圣殿草图。

**三阶（精英环）**
切入身份/时点：团队副手。
初始事件：火焰元素从裂隙涌出，挡住通往三龙台的桥，你要决定用水元素契约还是硬突。
开场白建议：「桥在抖。元素的核心里看见小小的龙影——像圣殿在用火做广告。」
关键NPC立场：火焰元素——敌；红龙观察员——要少破坏建筑。
主线钩子/支线：通桥；支线＝救被困地精工程师。
危险度/规避：中高。
任务方向/奖励：火抗、桥通行。

**四阶（单暮光龙）**
切入身份/时点：10 人精锐。
初始事件：团长指定先杀塔尼布隆「练手」，卵爆时你负责拦幼龙不进主坦。
开场白建议：「暮光龙的鳞是错的颜色，像把夜空钉在肉上。卵裂开时，有人喊可爱，有人喊集火。」
关键NPC立场：**塔尼布隆/沙德隆/维斯匹隆**之一——敌；团长——要进度。
主线钩子/支线：单龙击杀；支线＝收集卵壳给肯瑞托（密封）。
危险度/规避：高；规避 3 龙开战。
任务方向/奖励：龙血样本、史诗散件。

**五阶（萨塔里奥 0 龙）**
切入身份/时点：核心团。
初始事件：三龙已清，萨塔里奥激怒；火墙路线需要你当「 ent 标点」。
开场白建议：「黑龙站起来时，熔岩都让路。他说守护的不是宝藏，是未来的翼。」
关键NPC立场：**萨塔里奥**——敌；治疗领导——要火墙图。
主线钩子/支线：0 龙击杀；成就避熔岩打击。
危险度/规避：高。
任务方向/奖励：萨塔里奥基础掉落、龙眠声望。

**六阶（+1 龙）**
切入身份/时点：硬模式尝试团。
初始事件：保留沙德隆，开战后门与信徒打乱火墙路线，你要重排打断表。
开场白建议：「一条龙的影子已经够黑。第二份技能栏打开时，语音里只剩倒计时。」
关键NPC立场：**萨塔里奥+沙德隆**——敌；副团——要分伤。
主线钩子/支线：暮光协助成就；支线＝门内信徒优先级。
危险度/规避：极高。
任务方向/奖励：提升掉落、硬模式徽章。

**七阶（+2 龙）**
切入身份/时点：公会主力。
初始事件：双龙存活，有人提议灭一名减负——灭了则今晚成就作废，你投票。
开场白建议：「两道暮光与一道黑炎把场地切成三份语法。走错颜色的人会先死。」
关键NPC立场：**萨塔里奥+两龙**——敌；成就党 vs 进度党。
主线钩子/支线：双龙击破；政治安抚。
危险度/规避：极高。
任务方向/奖励：高阶史诗、公会记录。

**八阶（+3 龙 · 暮光区）**
切入身份/时点：顶尖团队指挥。
初始事件：三龙全留开战；龙眠上方红龙传来讯息：若失败，卵将在一小时内被传送走——你没有第二周。
开场白建议：「三道龙息与火墙合唱。萨塔里奥的眼睛里有死亡之翼的笑法。你举起手，不是因为勇敢，是因为倒计时归零。」
关键NPC立场：**萨塔里奥+三暮光龙**——顶点；**阿莱克丝塔萨的意志（远距）**——要结果；死亡之翼——不可战。
主线钩子/支线：暮光区成就；毁卵确认；支线＝阻止黑龙传送阵。
危险度/规避：贴近顶点；**条件性胜利/情报优先**。
任务方向/奖励：顶尖掉落与坐骑叙事、龙眠英雄名号、红玉圣殿预警情报。
