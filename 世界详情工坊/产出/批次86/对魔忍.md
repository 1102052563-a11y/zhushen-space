# 对魔忍

## 剧情

**【作品来源】**
《对魔忍》（対魔忍 / Taimanin）为日本成人向厂商 **LILITH（黑莉莉丝 / Black Lilith）** 旗下跨媒介系列，企划·原案 **笹山逸刀斎（Sasayama Ittousai）**，代表原画 **カガミ（Kagami Hirotaka）**、**葵渚（Aoi Nagisa）** 等。2005-10-21 以《対魔忍アサギ》开山，其后扩展为朝香三部曲与外传、雪风双部、紫、红、ZERO、妈妈是对魔忍等视觉小说主轴，并衍生 OVA、轻小说、卡牌/RPG 网游（决战竞技场 TABA、RPGX）、全年龄动作作 *Action Taimanin*、小队制 *Taimanin Squad* 等。世界观核心：**近未来日本—人界与魔界并存，政府暗部以「对魔忍」对抗人魔外道与跨国企业**。文风：任务/组织/异能与地下社会博弈为主，残酷间谍战气质；**本档案以公开剧情脉络与设定为纲，信息密度走任务·组织·忍法·势力，避免过度成人描写**。卡片锚点：胧的背叛/魔界入侵——对魔忍部队遭魔族伏击，鬼崎大刀（鬼崎切华等）展非人战力，井河朝香重伤下以疾风战技突围。

**【世界定位】**
近未来架空日本。表面上是现代都市与国家机器，暗面是魔界渗透、诺玛德等跨国复合体、人魔结托犯罪网，以及挂在政府暗部名下的忍者特种部队「对魔忍」。系列群像以 **井河朝香（アサギ）** 为早期中心视点，后扩展至 **水城雪风、八津紫、心願寺红、风魔小太郎** 等多线。一句话：这是「五车村忍校 + 第三课任务板 + 东京暗街与魔界口岸」的残酷任务世界；契约者是变数，不是天选免死金牌。

**【世界观 · 力量体系】**
力量本源三叠：**① 忍法（Ninja Art）**——血脉觉醒后以「气」驱动的个人异能，元素系（风·火·水·土·木等）与特异系（幻术、傀儡、雷电、空间、吸收等）并存，同名忍法因血脉纯度与修行深度差距极大；**② 对魔忍装与兵器**——贴合身体的战斗服强化机动、防护与气路传导，配合忍刀、苦无、枪械与现代电子战；**③ 魔族血脉/魔导/改造技术**——魔界血脉、吸血鬼始祖级权能、诺玛德与魔科式人体改造、克隆、外骨骼、恶魔化（Demonization）等，可把凡人抬到超凡，也可把对魔忍拖进「改造俘虏」命运。死亡规则：低阶任务员一击即死；高阶对魔忍可凭装具与气路硬抗热武器与低阶魔；魔王/始祖级近乎不灭或可再生，但非无限——真名、灵魂、核心器官与契约仍可成为弱点。特殊系统：五车村选拔与编入、第三课（Section Three）任务编号、诺玛德「混沌竞技场」地下战、UFS/中国联盟等国际暗战、魔界九大贵族与地狱骑士序列。

战力阶梯（按**实际破坏力/任务位阶**，宁低勿高）：
1. **凡人·警察·普通佣兵**：手枪、刀棍、街头火并。≈一阶初中期。
2. **见习对魔忍 / 未完全觉醒血脉者**：能用基础体术与初级忍法，可端小据点、杀低阶魔眷。≈一阶后期~二阶初期。
3. **正式对魔忍（五车毕业·可独立任务）**：一骑当千小队战力，裂装甲、穿乱军、对低阶魔将。≈二阶~三阶初期。
4. **精英对魔忍 / 小队长·教官级**：可独当一面清剿街区级魔巢、与名号魔将互砍。≈三阶~四阶初期。
5. **传说级对魔忍（井河朝香巅峰向、八津紫、秋山凛子等）**：一城仅数人；可掀平建筑群、在魔界口岸硬刚中高阶魔；朝香「杀阵华」分身杀阵属此档标尺。≈四阶~五阶。
6. **魔将·地狱骑士·改造兵器体 / 诺玛德高管战力**：城市级威胁，可让一城治安瘫痪；热武器与常规部队近乎无效。≈五阶~六阶。
7. **魔王候补 / 九大贵族本家 / 诺玛德顶点侧翼**：地区级灾难，可改写局部战场规则。≈六阶~七阶。
8. **吸血鬼始祖爱德温·布莱克、顶级魔王本体**：跨城市~准战略级，再生与生命吸取近乎规则。世界顶点向。≈七阶巅峰~**超阶边缘**（本世界库覆盖至七阶，顶点写情报优先/条件性胜利）。

乐园阶位映射：凡人~见习≈一阶；正式对魔忍≈二阶；精英/小队长≈三阶；传说对魔忍≈四阶；魔将/地狱骑士≈五阶；贵族本家/魔王候补≈六阶；始祖·顶级魔王≈七阶（顶点本体贴近超阶，低阶严禁硬刚）。宁低勿高；同名角色因线体（VN 主轴 / RPGX / Action）状态不同可上下浮动一档观感。

**【地理 · 舞台】**
- **五车村（Gosha Village）**：对魔忍总根据地，隐于日本山间的忍者聚落+忍校+兵营一体化。编成、训练、医疗、档案与家族政治在此完成。村外有结界与伪装成普通山村的外围；村内分区：宿舍与食堂、道场与射击场、禁书库、禁闭与体检楼、长老会馆。适合一~三阶日常与受训切入。氛围：晨雾、号令、压抑的安静。
- **东京及近郊暗街**：表面繁华，地下是人魔交易、走私口岸、诺玛德子公司与「乐园/Under Eden」一类俱乐部。任务最密集的舞台。雨夜巷战、屋顶机动、地铁弃线是常见场景语言。
- **混沌竞技场（Chaos Arena）**：诺玛德系地下格斗与「展示·改造·处刑」场所，朝香线核心副本点。入口门槛=被俘或潜入；内部有观众席、改造间、囚室与直播机房。
- **东京王国（Tokyo Kingdom）等特区**：朝香 2 线舞台，企业城邦与魔界资本深度绑定；法律是公司规章，警察是保安升级版。
- **魔界（Dark World / Demon Realm）**：与人界以隙缝/传送相连；贵族领地、地狱骑士驻地、血族城邦。空气与重力规则可异于人界。高阶任务与终局阴影。
- **UFS（美联邦侧）/ 中国联盟（Chinese Union）设施**：国际暗战与魔导科技试验场；龙门等组织在东亚伸手；实验室比战场更危险。
- **风魔残党据点 / 甲贺系叛徒巢穴**：胧忍军与叛忍活动区；地形熟、陷阱多。
- **鬼崎相关魔界战场 / 边境临时前线**：卡片锚点「魔界入侵·伏击」常落于此；泥、铁丝网、失效的卫星链路。

势力地盘：五车/第三课管「合法任务」；诺玛德管「资本与地下娱乐」；魔界贵族管「血统与领地」；国际势力管「技术与代理人」。层层碾压；本地人若听闻见习要单挑布莱克，会当疯子——正确反应。

**【世界剧情线】**
① **前史·人魔不干涉破裂**  
太古以来人界与魔界有「互不干涉」默契。近未来东京被魑魅魍魉侵蚀，人魔结托的犯罪组织与企业暗跃。日本政府暗部以忍者血统者编成「对魔忍」，在阳光下不存在的编制里执行猎魔与肃清。井河、水城、八津、风魔、甲河等忍族在此框架下或效忠、或分裂、或叛逃。世界状态：表面法治国家，暗面战时体制。

② **《对魔忍朝香》——胧的死与复活·竞技场**  
约 2072 年前后（系列后期作品回溯推算）。**井河朝香**已是对魔忍中一骑当千的顶尖，以忍法「杀阵华」分身杀阵闻名。她曾击杀叛徒 **胧（Oboro）**——原甲贺系、后率「胧忍军」投靠诺玛德、猎杀同袍者。朝香以为仇敌已除，退居日常，准备与幼驯染未婚夫 **泽木恭介（Sawaki Kousuke）** 成婚，并照看自学忍术的妹妹 **井河樱（Igawa Sakura）**。  
转折：胧借魔族之躯复活，任混沌竞技场侧关键人物，设计绑架恭介并设局。朝香与樱被拖入地下世界，被迫在竞技场求生。真结局路线上，朝香选择**彻底击杀胧**、承担任务与亲情的双重代价；坏结局则是改造·堕落·姐妹双堕等 IF（乐园叙事默认以「任务线可介入的真结局向」为主轴，IF 作隐藏分支提及）。名场面：新月夜数百胧忍军包围→杀阵华同时斩杀；竞技场中气路被封仍拼死护妹；对胧「多数无意义」的冷宣告。因果：朝香与诺玛德、与吸血鬼始祖 **爱德温·布莱克** 的债未了；樱从见习被迫成长；「对魔忍也会被俘、被改造」成为系列母题。

③ **外传·混沌竞技场 / 梦魇 / 夏日**  
外传补全竞技场生态、朝香被俘期的支线任务与日常裂缝。作用：让「地下规则」具体化——门票、赌盘、改造医师、魔族观众、叛徒中介。契约者低中阶最易介入的是**潜入、救出、毁证据、截获改造方案**，而非正面掀翻诺玛德。

④ **《对魔忍朝香 2》——东京王国**  
舞台扩到企业特区「东京王国」。诺玛德与政商勾结更深，朝香再入局，面对更复杂的**情报战+都市攻防**。主题从「个人复仇」转为「国家暗部能否在资本魔界同盟前保持牙齿」。阶段结局：朝香线暂时压住局部危机，但布莱克与诺玛德体系仍在。

⑤ **《对魔忍紫》——傀儡与忠诚**  
**八津紫（Yatsu Murasaki）** 线：强力对魔忍陷入傀儡化/精神操控危机（「くノ一傀儡」主题）。组织内部出现「同伴是否已被改写」的不信任。名场面：紫的怪力与忠诚被武器化；小队不得不在「救同伴」与「灭威胁」间抉择。因果：改造与精神控制技术从诺玛德扩散到更多黑医与魔导学者（如桐生系相关线在系列中反复出现）。

⑥ **《对魔忍雪风》1·2——卧底与母女债**  
**水城雪风（Mizuki Yukikaze）** 与搭档 **秋山凛子（Akiyama Rinko）** 执行长期卧底，目标牵涉魔界娱乐设施与人魔交易网；雪风之母 **水城不知火（Mizuki Shiranui）** 的失踪/堕落传说是情感核。任务结构：潜入→身份维持→情报外送→撤离失败风险。雪风 2 将战线扩大，母女线与组织线缠死。阶段主题：**对魔忍最强的不是斩击，是还能不能相信任务简报上的「友军」。**

⑦ **《对魔忍红》——半魔血与家族**  
**心願寺红（Shinganji Kurenai）** 线：半魔血脉、家族诅咒与猎魔使命冲突。舞台更贴魔界边境与血族政治。适合写「血统污名 vs 任务价值」。

⑧ **《对魔忍朝香 3》——总清算前夜**  
朝香线集大成。魔界入侵、叛忍、国际势力、诺玛德与布莱克的布局在同一时间窗挤压。卡片锚点「胧的背叛/魔界入侵：部队遭伏击，鬼崎侧非人战力，朝香重伤疾风突围」可落在此阶段前后的大规模冲突中：前线指挥链断裂、友军里有内鬼、魔族精英（含鬼崎切华等半魔/鬼血角色在系列中的战力展示）撕开防线，朝香以风系高速战技带残部突围，用「活着回去报信」换下一次反攻权。阶段结局：人界暂时守住，但付出小队减员、据点暴露、朝香身心双重透支；为 ZERO 与网游时间线的「多线并行」埋钩。

⑨ **《对魔忍朝香 ZERO》——再年轻的朝香**  
时间线/IF 向：更早时期的朝香如何成为「最强」。用于解释杀阵华完成度、井河家教育、早期对胧与诺玛德的第一次碰撞。契约者可在此线体验「见习→传说」成长，但改变过大仍会被主轴因果回流。

⑩ **网游轴（TABA / RPGX / Action Taimanin）——群像任务板**  
- **TABA（决战竞技场）**：卡牌战+短篇任务，强化「竞技场规则宇宙」。  
- **RPGX**：**风魔小太郎（Fuuma Kotaro）** 等视点扩编五车日常、任务委托、魔界贵族与国际事件；大量具名对魔忍（甲河明日香、上原铃、神村舞华等）可组队。  
- **Action Taimanin**：全年龄向动作，保留组织与异能，淡化成人内容，适合作为「可公开执行的任务表现」参考。  
网游轴结局开放：没有单一「打败布莱克就天下太平」，而是**任务永续、势力拉锯**。乐园侧写终局时：以 VN 主轴「击破胧、重创诺玛德局部、布莱克仍为顶点阴影」+ 网游「持续治安战」并记。

⑪ **微观任务贯穿（正文高频）**  
除大场面外，可写：第三课编号任务领取与毁尸灭迹、五车村内家族政治与恋爱禁令、东京暗街护送证人、假货魔药稽查、魔界隙缝封闭、叛忍悬赏、改造医师黑诊所突袭、UFS 特工临时同盟、中国联盟龙门线争夺、竞技场赌盘做空、小队粮弹与气路药剂配额。AI 优先这些，而不是一上来顶点对轰。

⑫ **蝴蝶效应边界**  
可改：救一名炮灰任务员、截获一份改造清单、让某次伏击少死几人、提前揭露某个内鬼。不可无代价改：布莱克作为顶点的战力逻辑、朝香「最强对魔忍」的叙事功能、系列对「被俘风险」的残酷底色。改了局部，因果以追杀、顶替、更惨的改造实验回流。

⑬ **任务流程与组织伦理（可玩细则）**  
标准任务链：第三课下发编号 → 五车编成小队 → 简报（目标/交战规则/灭口条款）→ 潜入或强袭 → 回收证物与黑匣子 → 归队体检（查魔气污染与精神暗示）→ 报告被改写后归档。伦理冲突固定三题：① 平民与任务孰先；② 同伴被俘是救还是毁证据；③ 国际同盟要你的样本时是否交。正文高频可写「简报与现场不一致」：地图过时、友军已叛、目标其实是诱饵。配额现实：气路药剂、特种苦无、隙缝钉、卫星静默窗口都有限；露富（私藏魔界坐标）会触发审计。契约者若跳过体检直接回村，可能被当成污染源隔离——这是组织正确反应，不是针对玩家。

⑭ **忍法实战与克制关系**  
元素系擅长战场塑造：风=机动斩首，雷=瘫痪电子设备与神经，火=清场与恐惧，水/冰=控场，土=防御与封锁，木=束缚与侦察。特异系改变规则：幻术破情报、傀儡破信任、吸收破续战、空间破包围。克制不靠「属性表」，靠情报：不知对手忍法名就等于送人头。魔族侧克制对魔忍的常用手段：气路干扰结界、血咒标记、改造寄生、人质道德绑架、白天政务夜间屠杀的时间差。热武器在二阶以上逐渐贬值，但狙击、炸药与电磁脉冲对未展开气防的目标仍有效——所以精英仍练枪。

⑮ **卡片锚点战役拆解（伏击·突围）**  
阶段：① 前线集结，多支对魔忍小队与临时佣兵混编；② 隙缝异常扩大，侦察失联；③ 内鬼切断补给与上行通讯；④ 鬼血/魔将级单位从侧翼插入，常规队形崩溃；⑤ 朝香带伤以疾风战技切开退路，优先保全「能回去说话的人」而非死守地标；⑥ 归队后政治审查：谁下令、谁泄密、谁该背锅。可介入点：战前安检揪内鬼、夜袭中抢回通讯中继、突围时殿后换朝香脱身、战后藏起一份未被改写的黑匣子。名场面气质：雨、泥、断裂的耳麦、以及「最强也会吐血」的标尺。

**【主要人物】**

**【势力图谱】**
- **对魔忍 / 五车村**：宗旨=以忍制魔、守人界暗线和平；地盘=五车+全国任务网；代表=朝香、樱、雪风、紫、红、小太郎等；战力=见习到传说完整金字塔；与第三课是「刀具与持刀人」。村内有家族长老会、教官团、医疗班与档案库；恋爱与外泄被视为战力风险。日常=训练、任务、忌讳话题（不知火、竞技场幸存者）。
- **第三课（Section Three）**：政府暗部；发令、预算、灭口与政治甩锅；代表=山本信繁等；对魔忍战死可写成「交通事故」。与五车互相需要也互相提防：课要战果，村要人命与尊严。
- **诺玛德（Nomad）**：多国籍复合企业；表面合法商业，暗面魔界资本、改造、竞技场、人口与器官级交易；代表=布莱克、胧、诸多高管与科学家。子公司可伪装成制药、安保、娱乐、物流；打掉一家会再长一家。
- **胧忍军**：叛忍佣兵团；猎杀对魔忍；后并入诺玛德暴力工具。懂五车战术，擅长伪旗与「自己人最懂怎么杀自己人」。
- **魔界贵族 / 九大贵族 / 地狱骑士**：血统统治；与人界有贸易也有入侵；英格丽等骑士为执行端。宴会与战争是同一张脸的两面。
- **UFS / DSO 等**：美系超常战力与科技；可同盟可夺成果。常以「联合演习」名义收割样本。
- **中国联盟 / 龙门等**：东亚代理人与超能者；与五车有情报交换与冲突。口岸与南海—东海隙缝是摩擦带。
- **风魔残党 / 甲贺叛徒**：忍族内战余烬；小太郎线常牵出「名门沉没后的遗产与仇恨」。
- **地下中介层**（黑医、赌盘庄、魔药贩、人口牙行）：无大义，有报价；契约者最稳的情报源也是最稳的出卖者。
敌友逻辑：任务同盟脆弱；暴露核心情报=被灭口；钱与血统常常压过正义；「临时同盟」默认附带背叛倒计时。

**【贵重物品】**
- **对魔忍装（各型）**：气路传导+防护+身份识别；被俘后常被脱卸/破解作为羞辱与分析样本。低阶只能领制式，高阶可定制传导回路。
- **忍刀·专武（朝香爱刀等）**：与杀阵华配合的斩具；丢刀=战力与象征双损。名刀有登记，黑市出现=有人叛或有人死。
- **杀阵华完成体的气路秘传**：井河系核心；外人难学，强学易走火。契约者最多拿到残页与训练记录，拿不到「成为朝香」。
- **改造药剂 / 神经重写方案 / 服从实验档案**：诺玛德黑医核心资产；夺取=救人，流落=灾难。写任务时点到机制与争夺，不展开描写过程。
- **混沌竞技场通行/赌盘权限**：地下世界硬通货；可换情报与入口，也可被追踪。
- **魔界隙缝坐标与封印钉**：战略物资；丢一份坐标等于丢一座城；钉的部署需要小队掩护。
- **吸血鬼始祖血样 / 布莱克相关研究**：终局级诱饵；持有即被全世界追杀；使用条件极苛刻且易反噬。
- **第三课任务黑匣子 / 灭口名单**：政治核弹；比魔刀更危险。
- **雷系/风系等专属触媒与气路药剂**：续战资源；配额制，多拿会被审计。
每件写清：谁想要、抢到的后果、当前阶能否使用。低阶禁用终局级道具。

**【隐藏剧情 · 伏笔】**
- 胧之死不是结束，魔躯与备份人格使「叛徒」可反复登场；每一次「再斩胧」都可能只是剪除一个迭代版本。
- 布莱克对朝香的兴趣不止仇恨，更像长期观察与实验：最强对魔忍的气路、意志阈值与崩溃点，对他是数据。
- 不知火失踪连接诺玛德与魔界娱乐产业链；雪风线的真相往往比「简单复仇」更脏——涉及组织是否知情、是否默许。
- 紫线揭示：对魔忍最大威胁之一是「自己人被改写成敌人」；体检与精神鉴定因此成为权力工具。
- 井河家与其他名门的联姻/禁恋规则，表面是风纪，实则是血脉与战力管理；恭介线是「制度不允许的幸福」样本。
- 网游轴暗示多宇宙/多时间线（ZERO、Dark Future 等），主库档案以 **VN 主轴+可共用设定** 为准，互文点到为止，勿把全年龄动作作的轻松日常覆盖残酷主轴。
- 卡片锚点伏击：内鬼+魔界精锐+情报滞后的合成灾难；突围成功只换来下一场政治审查与减员名单。
- 监狱战舰等外宇宙作品与本系列有角色串场可能，但本档案不强制并入主线因果。
- 结局方向：没有「魔界永久关闭」的廉价和平；有的是任务永续、顶点仍在、幸存者继续拿刀。真结局向=局部胜利+长期阴影；IF 堕落向=组织崩溃的警示分支，不作默认世界状态。

**【叙事基调 · 雷区】**
基调：残酷间谍动作 + 组织伦理；色调偏夜、雨、霓虹与血。任务简报冷静，现场不冷静。对话可冷幽默，不可轻浮消解死亡。战斗描写优先：谁先动手、气路是否展开、旁观者什么反应、撤离路线是否还在。  
忌：把五车写成热血忍者校园无代价；把朝香写成无脑无敌；低阶满地魔王；**把布莱克写成「被封印所以战力为零」**——应写**超阶边缘：存在·情报优先/条件性胜利**。  
忌：过度展开原作中的性暴力与改造细节；用「堕落」当唯一剧情发动机——应用任务失败、情报泄露、同伴被俘、组织弃子、预算与灭口来推动。  
忌：编造原作没有的神器名与终局死亡；跨世界套话；把 Action 全年龄轻松日常覆盖 VN 主轴残酷底色。  
忌：用修仙境界名（凝气筑基）硬套本世界——本世界用「见习/正式/精英/传说/魔将/始祖」。  
最早切入锚点：五车村见习编入；或朝香斩胧后、退隐前夜；或卡片锚点伏击前的前线集结。信息密度优先，空话删掉换成可执行细节。

## 阶位切入点

> 阶位↔战力：一阶≈凡人~见习对魔忍；二阶≈正式对魔忍；三阶≈精英/小队长；四阶≈传说对魔忍；五阶≈魔将/地狱骑士；六阶≈贵族本家/魔王候补；七阶≈始祖·顶级魔王（布莱克本体贴近超阶）。世界顶点（爱德温·布莱克）为跨阶存在，低阶切入一律「规避顶点·情报优先」。

**一阶（见习·五车编入线）**
切入身份/时点：契约者以「五车村新入见习 / 第三课编外协力者」切入，锚定朝香已成传说、雪风世代尚在训练或刚接简单任务的时期。
初始事件：夜间紧急集合——东京某区出现低阶魔眷伤人；你被编入三人小队，装备老旧对魔忍装与通信耳麦，要求「确认隙缝、救出平民、禁止恋战」。现场却发现尸体上有胧忍军旧纹样，任务从驱魔变成「可能有叛忍」。
开场白建议：「五车的夜里没有虫鸣，只有训练场的脚步。广播叫到你的编号时，教官只丢给你一套还带着别人汗味的忍装：『今晚东京有缝。活着回来，再谈你能不能留下。』」
关键NPC立场：**井河樱**——若同期见习，会把你当竞争对手或临时队友；**山本信繁**——远程听汇报，失败就改写成事故；**井河朝香**——高岭之花，只在你搞出大事时投下视线；低阶魔眷与街头中间人——可打可问。
主线钩子/支线：第一次隙缝封闭；追查胧忍军纹样来源；支线＝五车内务（偷懒被罚、家族子弟欺压）、暗街买药被坑、护送一名目击证人。
危险度/规避：中——热武器与低阶魔可致命；须规避正式叛忍小队与任何诺玛德武装。
任务方向/奖励：活着完成报告、拿到正式见习章；奖励取向＝基础忍装维护权、入门忍法卷、第三课低密权限。

**二阶（正式对魔忍·东京暗街线）**
切入身份/时点：以「可独立带队的正式对魔忍」切入，锚定雪风·凛子接都市任务、朝香线尚未或刚结束竞技场浩劫前后。
初始事件：第三课派你潜入一家挂名贸易公司，实为诺玛德洗钱点；撤离时发现地下室关着「待运送的实验体编号」，其中一人竟能报出五车内部呼号——你的选择是毁设施、救人，还是先保情报。
开场白建议：「霓虹下面是另一张地图。你的任务板写着『例行审计』，可门锁后的冷气和血腥味告诉你：审计两个字，从一开始就是谎言。」
关键NPC立场：**水城雪风**——可能同场任务，毒舌但专业；**秋山凛子**——更愿与靠谱刀合作；**胧**的眼线——试探你是否值得设局；黑医中间人——卖药也卖你。
主线钩子/支线：摧毁洗钱点；截获实验体名单；支线＝假货魔药稽查、叛忍悬赏、与 UFS 特工临时换情报。
危险度/规避：高——须规避地狱骑士与布莱克直属；中招改造药剂=任务失败级灾难。
任务方向/奖励：立住「可带队」名声；奖励取向＝改良忍装、中级忍法深化资源、暗街线人网。

**三阶（精英·小队长·边境隙缝线）**
切入身份/时点：以「五车小队长 / 前线协调官」切入，锚定魔界隙缝频繁、卡片锚点级冲突酝酿期。
初始事件：你率队接管一段边境临时防线；友军里混入鬼血/半魔佣兵。夜袭中通讯被干扰，侧翼崩溃——有人高喊「鬼崎」一类的非人刀影撕开阵地。你必须决定：死守、诱敌、或把幸存情报送回五车。
开场白建议：「对讲机里先是尖叫，后是电流。你看见友军的灯一盏盏灭掉，像有人在地图上用刀划掉编号。风里有魔界的味道——甜的，像腐蚀。」
关键NPC立场：**鬼崎切华**——前线压迫者或可谈判的刀；**甲河明日香**——可能的增援刃；**英格丽**——若现身则本阶不可硬刚；**风魔小太郎**——若网游时点，可提供非常规编制。
主线钩子/支线：边境夜袭与突围；揪出内鬼；支线＝安置难民、抢修隙缝钉、与中国联盟龙门线抢坐标。
危险度/规避：高~极高——须规避贵族本家与布莱克；内鬼比魔更致命。
任务方向/奖励：保住防线或换来战略预警；奖励取向＝小队专武、隙缝封印组件、跨组织临时同盟。

**四阶（传说级·朝香同阶任务线）**
切入身份/时点：以「传说候补 / 可与朝香同框行动的顶尖对魔忍」切入，锚定朝香1真结局后或朝香3总力战前。
初始事件：第三课给你一份黑任务：确认胧是否再次复活。线索指向混沌竞技场深层。你在观众席或管道中目睹一场「展示处刑」，发现目标不是击杀，而是直播给魔界买家——你的存在本身若被拍到，五车将遭受外交级报复。
开场白建议：「竞技场的鼓声隔着三层混凝土还能震到牙。你不是来赢比赛的；你是来确认一个死人有没有重新学会呼吸。若她笑了，东京今晚会更长。」
关键NPC立场：**井河朝香**——同阶刃，可能抢你的击杀权也可能救你；**胧**——若复活则本阶终极仇敌；**八津紫**——可作突入怪力点；**泽木恭介**——若被再度利用，是人质变量。
主线钩子/支线：确认/再斩胧；摧毁一层竞技场核心；支线＝救出樱或同袍、盗取改造档案、给朝香断后。
危险度/规避：极高——布莱克可能「观赏」；须以情报与爆破取胜，不与始祖正面持久战。
任务方向/奖励：传说级认定；奖励取向＝杀阵华级参考资料（非完整传承）、顶格忍装、第三课豁免权碎片。

**五阶（魔将·地狱骑士交锋线）**
切入身份/时点：以「可斩魔将的国家级刀具」切入，锚定雪风2扩大战或朝香3魔界入侵高峰。
初始事件：地狱骑士级单位（如英格丽或其同阶）出现在东京湾口岸，名义「护商」实则开路。常规部队失能；你被授权动用战略忍具与魔界对策弹。开战三分钟后你发现骑士的真实目标是**护送一份布莱克血样**。
开场白建议：「警报不是鸣笛，是整座港区的灯同时变成血色。你的权限被临时抬到『可毁码头』——代价是事后有人要你的命，或要你的沉默。」
关键NPC立场：**英格丽**——骑士矜持，可谈条件但难叛主；**水城不知火**相关线索人物——情感与情报双钩；**诺玛德高管**——要血样过关；**山本信繁**——要你把事做脏并背锅。
主线钩子/支线：夺血样或毁血样；逼退骑士；支线＝港区平民疏散、反间诺玛德船队、与 UFS 抢样本。
危险度/规避：极高——骑士持久战必败；须条件战（地形、符咒、联手）。
任务方向/奖励：国家级战功与仇敌；奖励取向＝魔界对策武装、领域级忍法素材、国际情报交换权。

**六阶（贵族本家·魔王候补线）**
切入身份/时点：以「可踏入魔界贵族领地谈判/斩首的超规格执行者」切入，锚定红线或 RPGX 贵族事件。
初始事件：九大贵族某家邀请「人界刀具」出席宴会，条件是交出叛徒名单。宴会上你的酒被下了显影剂——所有对魔忍气路轨迹对贵族一目了然。你要在暴露前完成：救人、窃印、或掀桌。
开场白建议：「魔界的月光比人界更假。宴会厅的笑都很标准，像排练过一千年。有人碰杯时说：欢迎国家的狗——请证明你还能咬主人以外的人。」
关键NPC立场：**心願寺红**——半魔向导或立场摇摆；**魔界贵族家主**——要脸与盘；**胧忍军残党**——在后厨接脏活；**布莱克**——可能只派影分身旁观。
主线钩子/支线：贵族宴会窃印；半魔血统谈判；支线＝救出血统实验体、结盟弱支贵族反强支。
危险度/规避：贴近顶点——须规避始祖本体；贵族本家=六阶墙。
任务方向/奖励：魔界通行证与诅咒级契约；奖励取向＝贵族血印（双刃）、魔导兵器、跨界传送锚。

**七阶（始祖阴影·条件性胜利线）**
切入身份/时点：以「人类侧几乎顶点的执行者 / 多组织临时统帅」切入，锚定系列长线终局窗：诺玛德核心暴露、布莱克亲自下场或即将下场。
初始事件：布莱克现身（或高度具现的分身/血影），提出交易：交出朝香或某份「始祖研究完整档」，换东京一年不受大规模入侵。第三课要你答应；五车要你拒绝；你自己的小队只想活。真正可执行的胜利条件是：**毁其实验室集群 + 断其在人界的三处锚 + 逼其撤退**，而不是击杀声明。
开场白建议：「他站在废墟上，像站在客厅。你的刀第一次觉得短。耳麦里有人喊『开火』，有人喊『谈判』，有人已经把你的死讯草稿写好。你知道：今晚若有胜利，一定很贵，而且不会被写进新闻。」
关键NPC立场：**爱德温·布莱克**——**超阶边缘顶点：存在·情报优先/条件性胜利**；**井河朝香**——人类侧标尺，可并肩不可当炮灰；**英格丽**——主君之盾；**风魔小太郎**——非常规策略位；**山本信繁**——政治胜利优先于人命。
主线钩子/支线：三锚点破坏战；实验室集群爆破；朝香存活与否的蝴蝶；支线＝转移平民、伪造「交易成功」骗过媒体与贵族。
危险度/规避：顶点级——严禁硬刚本体至死；胜利=布局+代价+撤退条件。
任务方向/奖励：世界线暂时稳住；奖励取向＝始祖级情报残卷、跨组织统帅权、下一次战争的预备队名册（不是无敌神器）。
