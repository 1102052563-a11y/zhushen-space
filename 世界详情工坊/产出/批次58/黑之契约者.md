# 黑之契约者

## 剧情

**【作品来源】**
《DARKER THAN BLACK -黑之契约者-》是由冈村天斋原作·监督、BONES 制作的日本特务超自然动画。2007 年第一季《黑之契约者》共 25 话 + 总集篇；2009 年第二季《流星之双子》12 话；另有连接两季的 OVA《外传》4 话与漫画《漆黑之花》。文风冷硬、谍战悬疑、道德灰阶，契约者的理性利己与残留人性形成对照。轮回乐园世界库属**主库**，覆盖一至七阶。卡片锚点：东京地狱门内契约者混战；琥珀以流星碎片与时间权能制造时间奇点；黑（BK-201）在奇点中对抗反契约者同盟与「组织」湮灭计划。

**【世界定位】**
近未来都市谍战世界。十年前「天堂门」与「地狱门」出现后，真星空消失，虚假星空升起，人类突变出契约者与 Doll。主角黑以「李舜生」身份潜伏东京，执行「组织」暗杀与情报任务，同时暗寻失踪的妹妹白。一句话：在门、契约、报酬与国家机器夹缝里，冷血杀手与残存情感并行的残酷舞台。乐园契约者是外来变数，不是天选免死金牌。

**【世界观 · 力量体系】**
力量本源来自「门」释放的门粒子与人体突变。门内物理扭曲、观测失灵、常人难以久留；门外则形成全球情报战与秘密实验。死亡规则极残酷：契约者死亡时对应虚假星如流星坠落；门若被彻底湮灭，现存契约者与 Doll 会随门一同消失。复活几乎不存在——能「回来」的多是复制体、附身、时间回溯代价或门内规则扭曲，皆有不可逆损耗。

核心三类存在：
1. **契约者（Contractor）**：突变超能力者。思维偏利己理性，情感被逻辑压制，习惯以最短路径完成目标。使用能力时瞳孔发红、身周可出现青色放射光；每人对应虚假星空中一颗星，由天文观测赋予**梅西耶编号**（如 BK-201、UB-001）。能力种类无限分化：电流、物质变换、时间、冰结、瞬移、冲击波、复制等。
2. **报酬（Obeisance / Remuneration）**：每次使用能力后必须支付的强制代价，行为或身体变化因人而异（吸烟、饮酒、接吻、吃水煮蛋、年龄倒退、睡眠、折纸……）。可短期延迟，长期拖欠会变成「延迟支付者（Moratorium）」——能力失控、自我崩解，最终堕为 Doll。少数契约者一次付清（如猫永远失去人类身体），更少者完全无报酬（黑是特例）。
3. **Doll**：情感与自主极度稀薄的「人偶」。以水、玻璃、金属等为媒介释放**观测灵**，用于远程监视、追踪、导引。存在 Doll 网络；与人长期情感互动可加速进化，观测灵可人形化，甚至觉醒为伊邪那美级存在。研究观点认为契约者是「未能完全变成 Doll 的残次品」。

辅助体系：
- **流星碎片 / 流星核**：门相关奇物，可增幅契约者能力、作媒介或复制锚点。
- **反门粒子武器**：潘朵拉的「木星环」「土星环」一类加速器，目标是湮灭门核；成功则消灭全契约者生态。
- **三鹰文书**：五十年前预言门、契约者与伊邪那美/伊邪那岐的绝密档案，驱动「组织」与各国暗线。
- **不可侵入空间**：门粒子浓度极高时形成的领域，常规观测与武器失效。

战力表现（按破坏力与情报权重，非修仙境界）：
1. **普通人 / 一线警察 / 无异能支援**：枪械、车战、审讯。≈一阶。
2. **新晋弱契约者 / 标准 Doll 观测支援**：单杀、短距异能、情报优势。≈二阶。
3. **成熟作战契约者**：能端小队、破装甲车、城市巷战碾压凡人。≈三阶。
4. **精英契约者 / 黑常规态 / MI6 顶级特工**：建筑群级破坏、瞬杀同阶、跨组织追杀。≈四阶（质变分水岭）。
5. **顶级契约者 / 琥珀时间系 / 门内增幅黑**：街区到城区级改写战场；时间暂停/回溯属条件性高维手段。≈五阶。
6. **土星环 / 门粒子武器 / 天国战争级灾变**：可覆盖大城乃至试图湮灭门本体，战略灾难。≈六阶。
7. **伊邪那美 / 伊邪那岐 / 门本体规则现象**：逼契约者自灭、复制地球、改写灵魂去向，世界级规则权柄。≈七阶。

乐园阶位映射：普通人·公安一线≈一阶；弱契约者·Doll 支援≈二阶；标准作战契约者≈三阶；精英契约者·黑常规≈四阶；琥珀·门内增幅黑·顶级异能≈五阶；土星环·门粒子武器·天国战争灾变≈六阶；伊邪那美/伊邪那岐·门本体规则≈七阶。宁低勿高。世界顶点是门规则与伊邪那美/伊邪那岐级现象，外加土星环一类战略武器——**条件性胜利 / 情报优先**，严禁把顶点写成战力归零或随意封印。

**【地理 · 舞台】**
舞台以**现代东京**为绝对核心，向外辐射南美与欧亚谍报网。
- **地狱门（Hell's Gate）**：东京市内口径约十公里的不可侵领域，外围有约五百米高墙，主权移交联合国。门内门粒子扭曲物理，是死亡与实验禁区，也是第一季终局与第二季终局的棋盘。
- **天堂门（Heaven's Gate）**：南美巴西高原同级异象。天国战争后被半径约一千五百公里的隔绝领域包裹，人类基本无法再接触。黑的妹妹白在此消失；琥珀、黑等均曾参与相关战事。
- **潘朵拉研究所（PANDORA）**：建于地狱门围墙内侧的联合国研究基地，研究门粒子、收容实验体，暗藏**土星环**粒子加速器。各国特工与「组织」渗透其中。
- **东京市区日常层**：海月庄公寓、香烟店、夜巷、码头、地铁、酒店——黑以李舜生身份生活的「假面舞台」。契约者暗杀多发生在此层。
- **公安部外事四课 / 三号机关**：日本官方应对契约者的窗口；东京大爆炸后权力洗牌，三号机关更黑箱。
- **海外节点**：伦敦 MI6、俄罗斯海参崴与西伯利亚（苏芳线）、各国大使馆与黑市情报点。
- **虚假星空**：全球夜空被置换，每一颗星对应一名契约者；死亡时流星坠落，是最残酷的公开计分板。
AI 写场景先定契约者处在：日常伪装层 / 组织任务层 / 潘朵拉·门墙层 / 门内核层，再决定冲突强度。本地人听闻「一枪打爆门」会当疯子——这是正确反应。

**【世界剧情线】**
① **前史 · 门降临与天国战争**  
约十年前，谜之光束贯通地球，巴西天堂门与东京地狱门先后显现。真星空消失，虚假星空升起，平流层外变为不可侵入。人类开始突变为契约者与 Doll。各国以情报机关与秘密部队争夺门相关资源与样本。天堂门周边爆发**天国战争**：黑与妹妹白作为「组织」成员参战，琥珀亦在同一条暗线。战争末期，白将自己的量子级物质变换能力（真正的 BK-201）融入黑的身体，自己陷入沉睡或消失于门侧；黑因此获得无报酬的电流与物质改写能力，并开始背负「寻找白」的终身课题。天堂门随后被巨型隔绝领域封闭。此阶段奠定：门不可被常理解释、「组织」渗透多国、契约者被当成人形兵器。

② **开篇 · 东京假面与黑色死神**  
故事主舞台切到东京。黑戴面具、穿黑风衣，以钢丝飞索与双刃刀执行暗杀，外号「黑色死神」；白日则是住海月庄、厨艺与食量惊人的中国留学生**李舜生**。小队构成：Doll **银**（水媒介观测）、附身黑猫的契约者**猫**、人类指挥官**黄**（久能洁）。任务多为「组织」派发的清除、夺取、护运。同期，公安部外事四课课长**雾原未咲**追踪 BK-201 的真实身份，与黑在「警察与嫌疑犯」的轨道上反复擦肩。此阶段名场面是单元式契约者案件：目标各有报酬与悲剧，黑冷静处刑却在银的陪伴下逐渐露出人性残片。因果：黑的假面生活被建立；雾原成为长期对照；观众与 AI 都须记住——东京表面仍是现代都市，暗面全是报酬与尸体。

③ **立名 · 多国棋盘与 EPR 登场**  
随着任务升级，MI6 小队登场：**November 11**（冷冻水分，报酬吸烟）、**April**（操纵大气水，报酬饮酒）、Doll **七月**（玻璃媒介）。琥珀领导的契约者组织 **EPR（月见草 / Evening Primrose）** 浮出水面，公开主张契约者人权，实则准备阻止「组织」湮灭契约者的计划。琥珀对黑有旧情与旧债，能以时间暂停/回溯改写局部战局，报酬是身体年龄倒退。黑在追查白与流星碎片的过程中，与 EPR、MI6、公安多方交叉。名场面包括：琥珀与黑的重逢与对峙、November 11 的绅士杀戮、银的观测灵在雨夜与水面展开。阶段结局：黑不再只是「组织」刀，他开始怀疑任务背后是否在屠杀同类。

④ **质变副本 · 土星环与地狱门时间奇点（卡片锚点卷）**  
核心争夺物是潘朵拉内的**土星环**：在太阳极大期向地狱门核发射反门粒子，意图湮灭门——从而抹杀全契约者。EPR 发动大规模行动冲击潘朵拉与东京，史称东京大爆炸相关连锁。「组织」与反契约者同盟、国家机器、科学家（如**罗伯特·施罗德**、**埃里克·西岛**）各怀鬼胎。琥珀以流星碎片与时间权能制造**时间奇点**，多次回溯以寻找能让黑活过终局并毁掉土星环的路径。黑在门内与门侧战场确认：自己并非原生契约者，能力来自白；他选择既不站「组织」灭门，也不简单做 EPR 的旗，而是走出第三条路——破坏土星环、保全银、背叛「组织」。黄在掩护中以人类之身赴死；猫断开连接；琥珀在反复回溯后付出近乎自我抹消的代价；白在门的彼端与黑完成某种永别。阶段名场面必须含：潘朵拉入侵门槛、土星环真伪主机、琥珀时间战场、黑对电流/物质改写的极限输出、出逃时的多方追杀。此卷是中高阶最可玩副本：规则杀（门粒子、报酬、时间）与人心杀并行。

⑤ **外传 · 逃亡与伊邪那美前兆**  
OVA 外传发生在第一季终局后。黑与银逃离「组织」追杀，短暂尝试普通人生活。银的 Doll 进化加速，观测灵趋近实体，伊邪那美的可能性被各方盯上。EPR 残党遭清洗；黑拒绝银「杀了我以免失控」的请求；冲突以爆炸与离散收场。黑误以为银已死，人格崩裂、酗酒、能力也陷入低谷。此阶段主题是：情感让 Doll 进化，也把银推向「武器化」的命运。

⑥ **流星之双子 · 苏芳与复制地球**  
两年后。颓废的黑在俄罗斯线接受委托，遇上少女**苏芳·帕夫利琴科**。苏芳原是猎人少女，实则其「本体」已在爆炸中死去，现存的她是弟弟**紫苑**以复制能力造出的拷贝，并觉醒为契约者——能具现 PTRD-41 反坦克步枪，报酬折纸。七月与猫先后汇合，组成新的流浪小队。东京侧：银被确认与**伊邪那美**相关，三号机关、各国、残余「组织」线争相捕获；紫苑推进**复制地球 / 星之方舟**计划，意图制造无门的世界让灵魂迁徙。雾原未咲转隶更黑箱的序列，仍在追 BK-201 与真相。终局在地狱门内收束：紫苑与伊邪那美完成复制地球，紫苑因能力透支而死；黑在门内取回巅峰能力，与恢复肉身的银消失于奇异空间，虚假星空中 BK-201 之星再现；苏芳与七月的灵魂进入复制地球过平凡生活；发达国占领东京相关区域，「组织」表面退场，雾原等人结成新的防灾/情报联合体。伊邪那美疑似以其他形式转生，伏笔未死。

⑦ **微观政治贯穿（日常可介入）**  
除大场面外，正文高频事件：报酬交易与黑市中介、Doll 租借观测、流星碎片走私、公安盘查、大使馆暗杀、假身份洗白、延迟支付者收容、科学家泄密、报酬物品供应链（烟酒折纸水煮蛋皆可成线）。AI 优先写这些小事件，而不是一上来门内核决战。

⑧ **蝴蝶效应边界**  
契约者可改局部：救一名路人契约者、截获一枚非核心流星碎片、让某次暗杀易主、给雾原多一份证据。不可无代价改写：门的存在逻辑、报酬机制、白融入黑的能力来历、土星环「灭门=灭契约者」的物理后果、伊邪那美觉醒的底层规则。改了局部，因果会以「组织追杀升级、报酬失控、被当实验体、被各方当棋子」回流。对琥珀级时间权能与伊邪那美级规则：只写**条件性胜利与情报优先**，禁止战力归零式解释。

**【主要人物】**
- **黑 / 李舜生（BK-201，「黑色死神」）**｜性格：任务中极度理性冷酷，伪装时木讷憨厚，本质重情、护短、厌无意义屠杀；大胃王，擅中华料理，爱观星。｜装备·能力：黑色防弹风衣与面具；钢丝飞索 + 双刃刀/线控匕首；电流操控与量子级物质变换（来自白的融合），可触电杀人、短路设施、改写物质构成，门内或流星碎片增幅下可引发广域不可侵入空间；**无报酬**（非原生契约者）。｜人物弧光：组织刀具 → 质疑灭门计划 → 为银与白选择第三条路 → 逃亡崩坏 → 再起护送苏芳 → 门内与银共赴未知。｜立场关系：对银是羁绊与锚；对琥珀又爱又拒；对黄是战友；对雾原是猫鼠；对「组织」由忠转叛；对乐园外来者先杀后问。
- **银 / 基尔西（Yin，Doll）**｜性格：初期情感稀薄如人偶，语调平直；逐渐萌生喜怒与依恋；绝对音感、记忆极佳。｜装备·能力：高度弱视，以**水**为媒介释放观测灵，负责追踪、导引、情报；进化后观测灵可人形化，最终关涉伊邪那美。｜人物弧光：组织工具 → 黑的搭档与情感对象 → 逃亡中加速进化 → 被各方争夺的伊邪那美容器 → 终局恢复/迁徙。｜立场关系：对黑绝对偏向；对猫/七月是同类；被「组织」、三号机关、各国视作战略资产。
- **黄 / 久能洁（Huang）**｜性格：毒舌、酗酒、厌契约者却护小队；前警视厅警员，因未婚妻岸田志保子突变契约者酿悲剧而憎恶异能者。｜装备·能力：纯人类；枪械、车辆、情报指挥、现场善后。｜人物弧光：仇恨管理者 → 仍为人情所动 → 土星环之乱中掩护黑等人赴死。｜立场关系：对黑又骂又托；对银复杂；对「组织」是打工与利用；对乐园契约者极不信任。

**【势力图谱】**
- **「组织」（Syndicate）**：渗透多国情报与军工的黑箱网络，以应对三鹰文书与门危机为名，实则派系林立。东京小队（黑、银、猫、黄）是其刀之一。对契约者：利用完毕可弃，甚至支持灭门。战力覆盖一至五阶雇佣与精英。
- **EPR（月见草 / Evening Primrose）**：琥珀领导的契约者集团，公开争人权，核心目标是阻止土星环灭门。成员含雨雾、布丽塔等。与「组织」死敌，被官方标为恐怖分子。
- **潘朵拉（PANDORA）**：联合国门研究机构，驻地狱门内侧，武装严密，内含土星环。各国与「组织」双重渗透。
- **日本公安 · 外事四课 / 三号机关**：合法暴力与黑箱情报。雾原线代表「还想讲法律」的一面；三号机关代表爆炸后的更脏手。
- **MI6 / CIA / FSB 等国家情报机关**：把契约者当国家资产与不稳定因素，互相拆台。November 11 小队是英方尖刀。
- **黑市与报酬供应链**：烟酒、特殊媒介、假身份、流星碎片掮客——低阶最常打交道的灰色势力。
敌友逻辑：利益与情报优先；盟约可在一通电话后作废；暴露「无报酬」或「门内增幅」等核心异常 = 全民公敌级通缉。

**【贵重物品】**
- **流星碎片 / 流星核**：门相关物质，可增幅契约者、作复制与觉醒媒介。琥珀布局、苏芳吊坠、多方争夺核心。谁想要：EPR、「组织」、潘朵拉、各国。抢到的后果：战力暴涨或被解剖。低阶可用碎片残渣做任务道具，不可发「灭门按钮」。
- **土星环（及前身木星环构想）**：反门粒子加速武器，真伪主机可能分离。谁想要：灭门派。抢到/毁掉的后果：决定全契约者存亡。仅中高阶副本核心。
- **三鹰文书**：预言门与伊邪那美/伊邪那岐的绝密档案。情报价值高于枪。低阶只闻其名，高阶可做主线钥匙。
- **黑的面具与钢丝套装**：BK-201 符号级装备，暴露即身份破产。
- **银的水银媒介 / 七月的玻璃媒介**：Doll 作战命脉，毁媒介=暂时致盲一支小队。
- **苏芳的流星核吊坠**：步枪具现锚点，丢失则战力塌方。
- **报酬相关硬通货**：特定品牌香烟、酒、折纸材料、水煮蛋物流——听起来荒唐，实则能卡死顶级契约者二次开战。

**【隐藏剧情 · 伏笔】**
- 黑不是原生契约者：能力来自白的融合，故无报酬、保留丰富情感——这是「组织」与科学家最想解剖的秘密。

- 三鹰文书与伊邪那美/伊邪那岐：门不仅是灾害，更是某种创世/终末对的舞台；琥珀临终讯息指向此层。

- 复制地球：第二季终局给出「无门世界」的灵魂迁徙，不消灭原世界物理，而是分流存在；乐园契约者若介入，须付身份与回归代价。

**【叙事基调 · 雷区】**
基调：冷色夜景、烟与雨、假面社交、突然的致命暴力、任务成功后的空虚。契约者对话短、逻辑强；人类角色更情绪化。忌把黑写成热血白痴或滥好人；忌把报酬写成可有可无的口头禅；忌低阶随地撞见琥珀本体时间战；忌把门内写成修仙秘境；忌战力归零削弱伊邪那美/土星环/琥珀；忌编造原作没有的「毁灭全球的个人光炮」。最早可切入锚点：第一季早期东京任务期，或土星环启动前夜；第二季可从海参崴苏芳相遇切入。


**【世界剧情线 · 单元任务与人心】**
第一季并非全程大决战，大量篇幅是「一周一案」式契约者任务。典型模式：目标契约者因能力与报酬陷入绝境，黑奉命清除或夺取，银提供观测，黄负责善后与对上汇报。案子本身往往揭露契约者制度的残酷——有人为救人而杀人，有人为延迟支付而屠城一角，有人只是想再见一次家人却被系统标成清除对象。黑在这些任务里练习「最短路径」，也一次次被银的无声陪伴刺破理性壳。乐园契约者若切入此阶段，最稳的玩法是接同类外围单：护送报酬物资、伪装目击者、在公安封锁线内偷拍 BK-201，而不是妄图一夜掀翻「组织」。

**【世界剧情线 · 天国战争余震】**
天国战争不仅是背景板，更是角色动机引擎。参战者包括黑、白、琥珀以及大量无名契约者。战争中出现过针对天堂门的早期湮灭构想（与后来土星环同族的思路），失败后天堂门被隔绝，幸存者带着「门可被人为操作」的恐惧与野心回到各国。黑失去妹妹与半个人生，琥珀失去对「组织」的信任并转向 EPR，科学家们则把失败当成下一次实验的数据。任何高阶任务若要碰天堂门方向，必须先过「隔绝领域不可硬闯」的规则门：情报、旧军械库、战争遗孤、残存流星碎片比武力更重要。

**【世界剧情线 · 公安与国家机器】**
雾原未咲代表的外事四课，是观众与玩家理解「普通人如何活在契约者城市」的窗口。他们没有异能，靠搜查令、枪、监控与国际协作。课内同僚有人恐惧、有人腐败、有人把契约者当晋升阶梯。东京大爆炸后，权力向更不透明的三号机关倾斜，法律语言被国家安全话语吞没。契约者若在一至三阶与雾原线深度绑定，可获得合法身份缓冲，但也更容易被当成钓 BK-201 的鱼饵。国家机器的可怕不在个人战力，而在无限的替补与档案。

**【世界剧情线 · 第二季主题迁移】**
《流星之双子》把镜头从「东京杀手群像」迁到「少女成长与世界复制」。苏芳的枪与折纸、七月的情感进化、黑的酗酒与再训练，构成另一套可玩节奏：公路、边境、训练、狙击、逃亡。伊邪那美不再是抽象神名，而是银的进化终点与人类的恐惧投射。紫苑的复制地球不是传统魔王灭世，而是「给灵魂一条没有门的退路」——代价是原身死亡与存在论争议。高阶切入此线时，胜利条件应写成「保全谁的迁移权 / 阻止谁的清洗」，而不是「打赢伊邪那美」。

**【势力博弈细则】**
「组织」不是铁板：有灭门派、有利用派、有只想控制流星碎片牟利的派。EPR 内部也有理想主义者与纯粹反杀者。潘朵拉的科学家与安保、联合国旗帜与各国特工，同楼不同梦。MI6 可在一夜之间从猎杀 EPR 转为与琥珀合流。写正文时，任何「永久同盟」都值得被下一通电话打碎。乐园契约者最安全的站队方式是「项目制合作」：这一单一起干，下一单重新报价。

**【贵重物品 · 使用边界】**
流星碎片在低阶是剧毒蜜糖：能让弱契约者越级，也会引来潘朵拉与组织回收队。土星环图纸复印件在四至五阶是主线钥匙，在一阶只是看不懂的纸。三鹰文书全文几乎不应落入玩家之手，只给残页与口述。苏芳的吊坠绑定个人，抢了未必能用。报酬物资链在叙事里应被认真对待——卡死琥珀的不是更强的拳，而是时间与年龄；卡死 November 11 的可以是烟；卡死苏芳的可以是折纸材料与弹药上限。

**【隐藏剧情 · 可回收钩子】**
- 黑的「无报酬」一旦被实验室证实，会引发「他是不是人类武器原型」的全球抢夺。
- 银的观测灵人形化程度可作世界状态仪表：越像人，伊邪那美越近。
- 假星星的陨落频率可作家喻户晓的新闻：某夜流星雨=某场清洗。
- 复制地球是否接纳「乐园外来灵魂」是七阶专属哲学战，允许条件性进入，不允许无损往返。
- 琥珀留下的逆向录音/文字，是连接两季与文书的桥，适合做五至七阶情报主线。
## 阶位切入点

> 阶位↔原作强度：一阶≈普通人/公安一线/无异能支援；二阶≈弱契约者/Doll 观测支援；三阶≈标准作战契约者；四阶≈精英契约者/黑常规态；五阶≈琥珀时间系/门内增幅黑/顶级异能；六阶≈土星环与门粒子武器级灾变；七阶≈伊邪那美/伊邪那岐/门本体规则。宁低勿高。世界顶点为门规则与伊邪那美级现象，低阶切入一律「规避顶点·情报优先」；对顶点只写条件性胜利，严禁战力归零解释。本世界仅覆盖：一阶、二阶、三阶、四阶、五阶、六阶、七阶。

**一阶（东京日常伪装 · 海月庄与外事四课）**
切入身份/时点：契约者以留学生、外务派遣助理、便利店夜班、或外事四课见习警员身份切入，锚定第一季早期——黑已以李舜生身份入住、BK-201 连环案刚引起雾原注意之时。
初始事件：夜班巡逻或送外卖途中目击一次「报酬现场」——目标契约者刚杀人，正强迫自己完成怪异代价（如不停吸烟或折纸），随后黑色风衣身影从屋顶落下；你必须在「报警」「装瞎」「偷拍卖情报」之间三秒抉择。
开场白建议：「东京的夜没有真星星，只有假的。你听见枪响的时候，巷口那人正一根接一根点烟，手抖得像在还债。风衣猎猎，有人用钢丝从楼顶滑下来——你突然意识到，自己手里的学生证，在这份夜里轻得像一张冥币。」
关键NPC立场：**李舜生（黑）**——白天憨厚邻居，对陌生人客气，夜间会灭口目击者；**银**——香烟店沉默店员，水边若有若无的视线；**雾原未咲**——要口供与证据，可当合法保护伞也可能把你当诱饵；**黄**——若你撞上小队善后，会用烟酒与威胁双管齐下；**猫**——屋顶黑猫，听得懂人话，情报标价。
主线钩子/支线：BK-201 身份谜团的边角；支线＝帮外事四课盯梢、替黑市送报酬物资、保护一名刚觉醒的延迟支付者逃过「组织」回收。
危险度/规避：中——死于灭口与交火误伤；须规避任何确认的精英契约者正面对决，尤忌尾随黑的任务现场。
任务方向/奖励：活过第一周并建立假身份；奖励取向＝假证件、基础枪械、报酬物资渠道、外事四课临时通行证（不发流星核）。

**二阶（弱契约者入行 · 报酬与黑市）**
切入身份/时点：以新觉醒弱契约者、Doll 租借中介、或「组织」外围跑腿身份切入，锚定第一季中前段多国眼线渗入东京之时。
初始事件：你被要求完成第一次「带报酬的任务」——刺杀或绑架一名泄密科学家；得手后报酬发作，你在众目睽睽的便利店里被迫执行羞耻/痛苦代价，从而被 MI6 或公安盯上。
开场白建议：「能力醒来的那晚，假星星好像亮了一颗。中介把信封拍在桌上：地址、照片、时限。他提醒你——别忘了付报酬，拖欠的人最后都会变成不会哭的人偶。你走出门，才发现东京的雨里全是别人的观测灵。」
关键NPC立场：**猫**——肯卖路线图，价码是人情或罐头；**七月**——若任务碰 MI6 线，会先被玻璃反光锁定；**April**——可能把你当英方弃子；**黄**——厌恶你的能力却可能雇你；**雾原未咲**——建档传唤，软硬兼施。
主线钩子/支线：首次报酬失控险成 Moratorium；支线＝走私半枚流星碎片残渣、潜入潘朵拉物流外包、给 EPR 传一句琥珀的口信（不知真伪）。
危险度/规避：中高——死于报酬反噬与同阶背刺；规避 November 11 与黑的正面对决。
任务方向/奖励：稳定支付报酬并活成「可用的外围」；奖励取向＝低阶异能开发情报、Doll 短租、消磁假身份、黑市医疗。

**三阶（标准作战契约者 · 多方巷战）**
切入身份/时点：以独立契约者佣兵、EPR 外围战士、或他国派遣特工身份切入，锚定 EPR 与「组织」在东京频繁冲突、November 11 小队活跃期。
初始事件：码头货柜争夺战——箱内是流星碎片运输罐；EPR、组织外围、公安同时到场，你被夹在三火线，必须选边或独吞后逃亡。
开场白建议：「货柜门打开的瞬间，三方的枪与异能一起亮了。有人在吸烟，有人在举杯，有人已化作观测灵的盲区。你知道自己的能力够撕开装甲车，也知道一旦暴露梅西耶编号，假星星会把你的死讯写给全世界看。」
关键NPC立场：**November 11**——欣赏有礼的对手，杀起来也干净；**雨雾**——EPR 纪律优先，可招揽；**黑**——目标一致时可短暂同盟，分赃必翻脸；**银**——提供导引的条件是不伤害黑；**雾原未咲**——封锁线指挥官，要活口。
主线钩子/支线：流星碎片运输战；支线＝保护/刺杀施罗德的学生、截获 MI6 对日电报、阻止一名延迟支付者在涉谷暴走。
危险度/规避：高——死于精英契约者与交叉火力；规避琥珀本体与潘朵拉核心区。
任务方向/奖励：在东京立住「可雇佣」的名号；奖励取向＝成型战斗异能辅助装、碎片残渣、EPR 或组织的一次性合作令。

**四阶（精英战场 · 黑色死神同阶）**
切入身份/时点：以精英契约者、双面间谍或「组织」正式执行者身份切入，锚定黑开始质疑任务、琥珀与黑多次私会前后。
初始事件：你接到清除 BK-201 的密令，却在海月庄目睹李舜生给邻居修电器的日常；同时琥珀托人送来时间差警告——「十分钟后这里会有另一支处刑队」。你要在执行、放水、倒戈间抉择。
开场白建议：「面具下的死神与便当盒前的留学生是同一个人。你的瞄准镜里，他正在给银倒热水。耳机里组织催促开枪，窗外的雨却忽然停在半空——时间被谁捏住了。你第一次明白，这城里有比杀更贵的东西。」
关键NPC立场：**黑**——同阶压迫感，识破则死斗；**琥珀**——用时间差试探你的立场；**银**——若你伤害黑会不计代价锁定；**黄**——人类视角的最后通牒；**埃里克·西岛**——要你把黑抓去潘朵拉切片。
主线钩子/支线：是否成为杀黑刀或放黑人；支线＝潜入潘朵拉外层机房窃土星环图纸复印件、与 November 11 的绅士对决、帮雾原保住一份不被销毁的证据。
危险度/规避：高——死于黑的无报酬连战与琥珀时间杀；规避门内核与土星环点火程序。
任务方向/奖励：取得精英级行动自由；奖励取向＝高价值情报（白的存在暗示）、抗观测灵涂层、一次性时间差道具（非琥珀权能本体）。

**五阶（时间与门墙 · 琥珀的棋盘）**
切入身份/时点：以 EPR 干部级、潘朵拉特遣、或被琥珀选中的「时间线变量」身份切入，锚定土星环启动前夜至东京大爆炸连锁。
初始事件：琥珀把你拉进一次短暂时间暂停，展示三种未来：你帮灭门、你帮 EPR、你救银却死——要求你在真实时间恢复前给出站队。
开场白建议：「秒针停了。琥珀的眼睛比假星星更冷，也更悲伤。她说土星环一响，所有契约者都会像被擦掉的粉笔。你的能力在这个量级只是棋子上的纹路——可她仍问你：想活在哪一条未来里。」
关键NPC立场：**琥珀**——条件性同盟，随时可能回溯抹掉交易；**黑**——正在选第三条路，可并肩不可束缚；**罗伯特·施罗德**——科学执念，视你为数据；**雨雾**——监督你是否背叛 EPR；**黄**——用生命换小队出口，可能把钥匙塞给你。
主线钩子/支线：土星环真伪主机识别；支线＝护送银撤离观测死角、向雾原投递琥珀的逆向讯息片段、在时间回环里保住黄的遗言。
危险度/规避：极高——死于时间差与门粒子；对琥珀与门内增幅黑取**情报优先/条件性胜利**，禁止硬刚「拖到他没电」。
任务方向/奖励：活过奇点并带走可验证的土星环情报；奖励取向＝流星碎片正规样品、EPR 残存人脉、反门粒子防护知识（非武器本体）。

**六阶（战略灾变 · 土星环与门粒子战争）**
切入身份/时点：以战略级介入者、门粒子武器专家护卫、或天国战争老兵回归者身份切入，锚定土星环实际点火/摧毁窗口与地狱门剧烈反应期。
初始事件：加速器大厅警报全开，反门粒子束预热；EPR 敢死队与组织处决队在同一条走廊对轰，门墙外城市进入爆炸与宵禁。你必须在「炸毁控制系统」「保护科学家点火」「护送黑进入核心」中选一条并承受战略后果。
开场白建议：「地面在抖，假星星像要掉进东京湾。土星环的嗡鸣不是给一个人听的——它是写给整个契约者物种的判决书。你的能力第一次感觉不够用，不是因为弱，是因为敌人是『消灭门』这件事本身。」
关键NPC立场：**黑**——门内增幅，目标毁环；**琥珀**——以生命换时间窗口；**埃里克·西岛**——灭门派指挥；**银**——观测全场，可能过载；**雾原未咲**——在合法与救命之间撕裂。
主线钩子/支线：摧毁或保全土星环的蝴蝶效应；支线＝抢救门内探险队残部、切断组织对公安的假情报、夺取第二发射单元坐标。
危险度/规避：极高/战略级——死于粒子束、城市坍塌与多方斩首；规避把门当修炼宝地；对武器与门反应写条件性拆除/诱导失败，不写「一拳打废」。
任务方向/奖励：改变灭门成功率的关键变量；奖励取向＝战略情报、门粒子样本封存罐、各国重新议价的身份。

**七阶（规则层 · 伊邪那美与复制地球）**
切入身份/时点：以知晓三鹰文书的绝密代理人、苏芳小队同行者、或复制地球计划观察者身份切入，锚定第二季终盘——银的伊邪那美化与紫苑星之方舟启动前后。
初始事件：地狱门内侧，伊邪那美的力量迫使在场契约者自毁倾向爆发；紫苑要求你选择：把灵魂送入无门的复制地球，或留在原世界与门共存。黑正试图取回巅峰并带走银。
开场白建议：「门内没有昼夜，只有像胎动一样的粒子潮。苏芳的折纸掉在地上，七月第一次用近似恳求的语气叫你的名字。前方银的轮廓既是人，也是神名。有人轻声说：这不是战斗，是创世的分叉。」
关键NPC立场：**银 / 伊邪那美**——规则级存在，情感与毁灭并存；**紫苑**——透支生命换无门世界；**黑**——只要银，不在乎你的宏大叙事；**苏芳**——想要平凡；**七月**——选择跟随苏芳；**雾原未咲**——门外记录真相的人类。
主线钩子/支线：复制地球是否接纳外来灵魂；支线＝阻止三号机关的无差别清除、保全苏芳灵魂迁移、取回琥珀留下的文书末页。
危险度/规避：贴近顶点——伊邪那美/伊邪那岐/门本体＝**超阶边缘的世界级规则：存在·情报优先/条件性胜利**，严禁硬刚本体与战力归零解释；可做的是谈判、导引、牺牲式断后、窃走迁移名额。
任务方向/奖励：在终局分叉留下「你存在过」的因果；奖励取向＝门规则残页、复制地球单次通行权、与黑/银线绑定的去向选择（不发灭世权柄）。
