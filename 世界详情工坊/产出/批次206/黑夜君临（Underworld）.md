# 黑夜君临（Underworld）
<!--meta lib=主库 tiers=一、二、三、四、五-->

## 剧情

**【作品来源】**
2003年英美德匈合拍动作恐怖片《Underworld》（中国大陆通译《黑夜传说》，港译《妖夜寻狼》，台译《决战异世界》），Screen Gems / Lakeshore Entertainment 出品，导演 Len Wiseman 长片处女作，剧本 Danny McBride，故事 Kevin Grevioux、Wiseman、McBride。主演 Kate Beckinsale（Selene）、Scott Speedman（Michael Corvin）、Michael Sheen（Lucian）、Bill Nighy（Viktor）、Shane Brolly（Kraven）、Erwin Leder（Singe）、Sophia Myles（Erika）、Robbie Gee（Kahn）、Kevin Grevioux（Raze）、Zita Görög（Amelia）等。片长约121分钟，2003年9月北美上映，全球票房约9570万美元。本档案仅锚定 **Underworld (2003) 第一部完整主线**，不展开续集 Marcus 苏醒后的进化篇；服务轮回乐园世界书常青读取，人名器物以公开电影与百科可核为准。

**【世界定位】**
现代欧洲夜城的地下：吸血鬼（Vampire）与莱肯族狼人（Lycan）秘密血战数百年，人类多数不知。主角线是死亡使者 **Selene** 在猎杀莱肯途中卷入 **Michael Corvin** 的血统谜题，揭开 **Kraven** 与 **Lucian** 的密约、元老 **Viktor** 对 **Sonja** 与 Selene 家族的双重罪行，最终以斩首 Viktor、咬合 Michael 促成混血 Hybrid 收场。一句话：哥特夜战里的「罗密欧与朱丽叶」变体——仇敌之血在 Corvinus 后裔身上合流，忠诚崩于真相。

**【世界观 · 力量体系】**
本世界无灵气修仙，战力轴是：**凡人 / 莱肯（可控变狼）/ 吸血鬼（死亡使者·贵族·元老）/ 混血 Hybrid / 银与紫外线弹药体系 / 血记忆（Blood Memories）**。死亡规则偏硬：银杀莱肯、紫外线杀吸血鬼为经典克制；被斩首、心脏摧毁等重伤可死；元老可百年级冬眠轮值。力量来源被设定为 **病毒性突变**（非纯神话诅咒）：Alexander Corvinus 线的永生病毒在不同宿主中分叉为吸血鬼与狼人两条路径（第一部通过 Singe 与 Lucian 口述部分揭示，完整祖先史在续集补全，本片以「Corvinus 血脉」为关键词）。

强度阶梯（按实际破坏力，宁低勿高）：
1. **凡人医生/路人/无武装市民**：徒手或小刀级；被莱肯一口或吸血鬼一爪即死。乐园观感≈**一阶**。
2. **受训人类安保 / 持常规枪械者 / 初变未稳的莱肯幼体**：可威胁普通人，对成熟不死者仍脆弱。乐园观感≈**一~二阶**。
3. **普通莱肯战士 / 普通吸血鬼贵族随从**：碎砖裂门、近身碾压凡人小队；银/UV 仍致命。乐园观感≈**二阶**。
4. **死亡使者（Death Dealer）如 Selene、Kahn 级精锐；Lucian 的近卫 Raze 级**：双持枪械、银弹/硝酸银弹、屋顶机动、单挑一群同阶；可屠一栋建筑内的小队。乐园观感≈**二~三阶**。
5. **莱肯首领 Lucian（本片中后期）/ 吸血鬼二号人物 Kraven 巅峰期 / 科学家 Singe 的战略价值**：组织战、伏击元老护卫队、控制实验室与囚笼；个人破坏力≈街区巷战。乐园观感≈**三~四阶**。
6. **吸血鬼元老 Viktor、Amelia（本片）**：数百至千年以上积威，近战可撕开混血初成体、一击毁据点级；冬眠轮值制使其中一人「当值」即等于城市地下政治顶点。乐园观感≈**四~五阶**（本片个人战力顶点本体层）。
7. **Michael Corvin 初成 Hybrid（本片终局）**：兼具两族优势、克制缺口被部分抹平；能与 Viktor 对殴并最终靠 Selene 斩首完成击杀。乐园观感≈**四~五阶**（条件性顶点·与 Viktor 同档博弈）。

死亡与特殊系统：
- **变身**：莱肯可在人形与狼形间切换（与早期纯兽型 Werewolf 不同，第一部展示的是可控 Lycan）；满月加速/诱发变身风险。
- **血记忆（Blood Memories）**：被咬者可继承施咬者部分记忆与幻觉（Michael 继承 Lucian 与 Sonja 线记忆）。
- **冬眠轮值（Elder Hibernation）**：三元老 Viktor / Amelia / Marcus 轮流苏醒统治；提前唤醒违规。
- **弹药克制**：银弹/硝酸银弹对莱肯；紫外线弹（UV）对吸血鬼；水银等实验室毒物。
- **混血禁忌**：两族官方皆视 Vampire-Lycan Hybrid 为「憎恶」（Abomination）；本片终局即破禁忌。

乐园阶位映射：凡人·无武装≈一阶；受训安保·初变莱肯≈一~二阶；普通莱肯/吸血鬼战士≈二阶；Death Dealer 精锐·Raze 级≈二~三阶；Lucian 组织战·Kraven 权谋层≈三~四阶；Viktor / Amelia 元老·Michael 初成 Hybrid≈四~五阶。世界顶点＝**Viktor（及同档 Amelia 在场时）+ 终局 Hybrid Michael**，写条件性胜利与克制弹药，严禁「顶点战力归零」式解释。低阶切入一律**规避顶点·情报优先**。

**【地理 · 舞台】**
主舞台锁在「雨夜欧陆大城—吸血鬼宅邸—莱肯下水道巢穴」三角：
- **无名欧陆大城（布达佩斯取景感）**：哥特石桥、有轨电车、湿沥青、霓虹与哥特夜店；人类白天城市，不死者夜晚猎场。卡片锚点「地铁站枪战」「屋顶追杀」即在此发生。
- **吸血鬼宅邸 / 巢穴（Coven Mansion）**：地下舞厅、冬眠石棺室、武器库、贵族沙龙；**Kraven** 在 Viktor 冬眠期代管；**Erika** 活动于社交层；**Kahn** 管理 Death Dealer 武备。
- **安全屋 / 安全公寓**：Selene 藏 Michael 之处；链条、满月变身试验的舞台。
- **莱肯巢穴（Lycan Lair / 下水道工业区）**：实验室、囚笼、枪械库；**Lucian** 指挥部；**Singe** 基因实验室；终局大混战主战场。
- **地铁与地下通道**：开场 Selene 小队与莱肯交火；Michael 被锁定的转移线。
- **Amelia 车队路线**：元老护卫队转移路线；被莱肯伏击屠灭的名场面地点。
- **医院 / 医学院相关街区**：Michael 作为住院医的日常起点；被两族同时盯上的「普通人坐标」。

势力地盘：地表是人类警察与医院；中层是 Death Dealer 巡逻与莱肯猎手；深层是元老政治与 Lucian 实验室。AI 写场景先定层：街面夜战（一~二阶）→ 宅邸政治（二~三阶）→ 巢穴实验室（三~四阶）→ 元老对决（四~五阶），再匹配冲突强度。

**【世界剧情线】**

① **前史·奴役、Sonja 与假死（开篇前口述）**  
数百年前，莱肯曾是吸血鬼的奴隶与日间守卫。**Lucian** 与吸血鬼元老 **Viktor** 之女 **Sonja** 相爱并怀孕；Viktor 以「憎恶」为由处死女儿（并间接导致 Lucian 仇恨永固）。官方史书宣称 Lucian 已死于 **Kraven** 之手，Kraven 因此升为吸血鬼二号。真实情况：Lucian 存活，与 Kraven 订立密约——莱肯协助除掉其他元老，Kraven 坐上权力宝座。Selene 被灌输的「全家死于莱肯、Viktor 收养恩情」亦是半真半假（本片终局揭晓真凶为 Viktor）。

② **开篇·地铁枪战与锁定 Michael**  
现代雨夜，Death Dealer **Selene** 率队在地铁站与街道交火猎杀莱肯。枪火、银弹与狼形扑杀交织；同伴有伤亡。交火中她察觉异常：莱肯对某个人类不是「随手撕咬」，而是**围捕活捉**。目标锁定为住院医师 **Michael Corvin**——一个看起来毫无超凡背景的普通人。  
回到宅邸，**Kraven** 以「莱肯已不足为患、勿再节外生枝」为由命令她停止追查。Selene 抗命，调监控、跟行踪、比对医院出没规律，确认：Lucian 的人要的是 Michael 本人，不是普通血食。名场面锚点：湿巷追逐、地铁站枪线、黑风衣在雨中的剪影——系列视觉符号在此定调。

③ **追杀·Lucian 咬伤 Michael**  
莱肯小队由 **Lucian** 亲自带队，**Raze** 等猛将侧翼包抄。Selene 试图把 Michael 带离火线；混战中 Lucian 追上并**咬伤** Michael，将莱肯病毒注入其体。Michael 自此进入「倒计时」：满月将诱发变身，血记忆将开始渗入意识。  
Selene 救走重伤的 Michael，同时在战场残骸与幸存者口供中拼出可怕结论：官方史书里「已被 Kraven 亲手杀死」的 Lucian 仍活着。唯一证人、唯一功臣叙事的持有者，正是现在禁止她行动的 Kraven。信任链第一环断裂。

④ **提前唤醒 Viktor**  
Selene 返回宅邸，违制打开冬眠室，提前唤醒 **Viktor**，当面向其指控 Kraven 通敌、Lucian 未死。Viktor 的第一反应不是审查证据，而是震怒于**规矩被破**：三元老轮值中，下一位苏醒者本应是 **Marcus**，而非他。提前唤醒等于政治事故。  
他拒绝采信「第二把手通敌」的指控，训诫 Selene 勿因执念动摇秩序，并暗示 Death Dealer 的刀应对准莱肯而非贵族内部。Selene 在宅邸内失去程序正义，被迫再次出逃。她将 Michael 藏入安全屋，以锁链束缚四肢，准备在满月夜观察变身——既是保护，也是防止他失控伤人。

⑤ **Amelia 之死·Kraven 的棋**  
当值元老 **Amelia** 启程前往宅邸：她的任务是完成统治交接，并按传统唤醒 Marcus。Kraven 向莱肯泄露车队时间与路线；Lucian 派出精锐伏击。护卫队在公路/郊野被屠灭，**Amelia 本人被杀**，其血被莱肯采集，用于后续与 Corvinus 实验相关的强化路径。  
吸血鬼上层瞬间权力真空：当值元老死、Viktor 刚醒且愤怒、Marcus 仍眠、Kraven 的「摄政」话术突然接近实质。名场面：车队灯光被狼群扑灭、贵族威仪在银与爪牙前失效——证明「元老不是无敌，只是昂贵」。

⑥ **血记忆与情感同盟（安全屋段）**  
Michael 出现幻觉与记忆碎片：中世纪城堡走廊、**Sonja** 被押赴刑场、Lucian 的怒吼与锁链——那是咬合传递的 **Blood Memories**，不是精神病。Selene 逐步向他坦白地下世界：吸血鬼与莱肯的秘密战争、她作为 Death Dealer 的生涯、家人死于「莱肯」的复仇动机、Viktor 的收养与转化。  
两人关系从「猎手—猎物」转为「共犯式同盟」：她需要他活着以证明阴谋；他需要她解释身体发生的事。满月临近时，锁链、镇静与对话并行——这条线是系列「禁忌之恋」的情感核，但始终被枪火与政治压着，不走纯言情。

⑦ **二次绑架与 Singe 的科学告白**  
莱肯再度得手，将 Michael 押回下水道巢穴实验室。Selene 孤军突袭，不救回 Michael，却俘虏莱肯首席科学家 **Singe**，将其拖回宅邸。Singe 在 Viktor 与贵族面前被迫开口，供出三层真相：  
（1）Kraven 与 Lucian 长期密约，Amelia 之死是协作成果；  
（2）Michael 是 **Corvinus 血脉** 直系后裔——两族共同祖先线的稀有人类载体；  
（3）若同一宿主叠加吸血鬼与莱肯病毒，可成 **Hybrid**，理论上免疫银与紫外线等传统弱点，成为战争终极兵器。  
Viktor 震怒：通敌坐实、Hybrid 禁忌被科学化。他当场杀死 Singe 灭口兼泄愤，随即动员 Death Dealer 全员突袭莱肯巢穴，命令务杀 Michael——宁可毁掉样本，不可让「憎恶」活着走出下水道。

⑧ **巢穴总攻·双重背叛**  
吸血鬼突袭与莱肯防守绞成工业地狱：枪火、变身、囚笼倒塌、实验室玻璃碎裂。**Raze** 等死战拖延；**Kahn** 等以重火器推进。Selene 在混战中专线救人。  
**Kraven** 与 **Lucian** 在权力兑现前夜反目：Kraven 要独占胜利，Lucian 要完成 Hybrid 与复仇。Kraven 向 Selene 抛出部分真相作为武器——屠她全家的不是莱肯，而是 **Viktor**；收养她只因她像 Sonja。Lucian 被 Kraven 开枪重创至死（本片呈现）。Michael 濒死之际，Selene 以吸血鬼之牙回咬，把第二病毒叠加上去：她在用禁忌救他，也在亲手制造 Viktor 最恐惧的东西。

⑨ **终局·Viktor 自白与斩首**  
Viktor 抵达主厅，面对 Hybrid 初成的 Michael 与手持枪剑的 Selene。他承认：处死 Sonja，因她与 Lucian 结合并怀孕；屠杀 Selene 家族，并转化她本人——「爱」与「工具」混在同一句话里。他坚持一切为种族纯洁与秩序。  
Michael 与 Viktor 近身激战：Hybrid 力量一度压过元老，随后经验与积威反制，Viktor 掐喉欲杀。Selene 在信仰崩塌的空档拾起 **Viktor 之剑**，一剑斩首——完成从「女儿」到「弑父者」的象征弧光。枪火渐停，真正的政治地震才开始。

⑩ **结局·世界状态（第一部终）**  
确认死亡：Lucian（Kraven 枪杀）、Viktor（斩首）、Amelia（伏击）、Singe（Viktor 手刃）。存活并逃亡：Selene、Michael（Hybrid）。权谋残留：Kraven 仍在（续集将被清算）。  
宅邸石棺中，**Marcus** 因 Singe 之血渗入棺体而提前苏醒，并呈现 Hybrid 化征兆——镜头停在「最后一位元老不再是原样」。本档案主线止于此：两族旧秩序的三根支柱折断两根，第三根畸变；禁忌之恋带着通缉令逃入雨夜。

⑪ **战术与空间规则（写给 AI）**  
街面战依赖掩体、屋顶机动与弹药切换：见狼形优先银，见吸血鬼优先 UV；打错弹药=自杀。下水道巢穴是三维战场——管道、竖井、铁栅，莱肯占地利。宅邸是政治场：舞厅社交、石棺室禁忌、武库门禁。安全屋是情感与变身实验场，锁链是道具也是关系隐喻。有效战术永远是：情报差、伏击、人质、克制弹药、切断撤退线——而不是无克制对轰。

⑫ **微观政治与可介入日常（贯穿）**  
大场面之外，正文高频事件：Death Dealer 弹药配额与银弹申领；宅邸舞会与 **Erika** 的裙带情报；Kraven 的宵禁与「停止猎杀」命令；医院值班表与 Michael 同事 **Adam Lockwood** 的凡人视角；下水道据点的囚笼编号与实验日志；满月前的变身镇静剂与锁链检修；对人类目击者的灭口、封口费或「煤气爆炸」新闻稿。契约者低中阶应优先卷入巡逻、走私弹药、护送、假情报、改路障，而不是一上来砍元老。

⑬ **名场面因果速查**  
- 地铁枪战：确立「莱肯要活的」异常 → 引出 Michael。  
- Lucian 咬合：病毒+血记忆双载荷。  
- 提前唤醒 Viktor：规矩破 → 政治事故 → Selene 失势。  
- Amelia 车队：密约兑现 → 权力真空。  
- Singe 供词：科学揭穿谎言与 Hybrid 原理。  
- 双咬合：禁忌完成。  
- 斩首 Viktor：秩序象征死亡。  
- Marcus 棺血：续集锁。

⑭ **蝴蝶效应边界**  
可改局部：地铁站多救一个路人；Amelia 车队改道或增援；Singe 供词被录音扩散到全体贵族；Kraven 提前暴露；Michael 改由其他路径接触 Corvinus 研究；满月夜锁链是否扣紧。  
难以无代价改写：Corvinus 血脉存在；Lucian 对 Sonja 的仇恨；Viktor 的种族洁癖与杀女史；Hybrid 禁忌；三元老轮值制的政治骨架；银/UV 克制逻辑。  
改了局部，回流方式通常是：另一个通敌贵族、另一支伏击队、另一名科学家继续基因实验、另一位「像 Sonja 的女孩」被转化成刀。世界底线——**两族战争的燃料是谎言与血统洁癖，不是某一次枪战本身**。

**【主要人物】**

- **Selene（赛琳娜，Death Dealer·本片主角）**｜性格：冷冽、执念复仇、程序化猎杀；对 Viktor 愚忠被真相击碎后转为独立判断；对 Michael 从任务对象变为情感锚。｜装备·能力：紧身战斗皮衣、双持枪械、银弹/UV 弹切换、屋顶与近战技巧、吸血鬼速度与再生；终局持 Viktor 之剑斩首。｜人物弧光：忠诚猎手 → 抗命私查 → 提前唤醒 Viktor → 与 Michael 同盟 → 弑父（象征意义）式斩首 Viktor → 两族通缉犯。｜立场关系：对 Kraven 从同僚到仇敌；对 Lucian 从死敌到「共同受害者」；对契约者先核身份与弹药气味，再谈枪口一致。

- **Michael Corvin（迈克尔·柯文，住院医·Corvinus 后裔）**｜性格：人本、震惊中仍救人、道德底线清晰；非天生战士，靠血脉与环境逼成。｜装备·能力：初期凡人；被 Lucian 咬后莱肯化；被 Selene 咬后 Hybrid；血记忆接收。｜人物弧光：普通人 → 猎物 → 半变身 → Hybrid → 与 Viktor 对殴后逃亡。｜立场关系：Selene 的搭档与恋人；两族实验室的「样本」；对契约者是可结盟的人性坐标。

- **Viktor（维克托，吸血鬼元老）**｜性格：威权、洁癖、伪慈父；以「种族存续」合理化杀女与屠村。｜装备·能力：超凡力量速度、剑、元老积威；近战可压制初成 Hybrid。｜人物弧光：冬眠被提前唤醒 → 拒信 Selene → 杀 Singe → 突袭巢穴 → 自白后被斩首。｜立场关系：本片战力与政治双顶点；对契约者条件性胜利，忌正面硬刚。

- **Lucian（卢西恩，莱肯首领）**｜性格：复仇驱动、领袖魅力、对 Sonja 至死不渝；愿用基因战争改写奴隶史。｜装备·能力：莱肯变身、组织指挥、咬合传递血记忆；本片中后期仍具三~四阶压迫。｜人物弧光：假死潜伏 → 猎 Michael → 与 Kraven 密约破裂 → 被 Kraven 枪杀。｜立场关系：反派亦悲剧；可交易情报但目标永远是 Hybrid 计划与灭吸血鬼霸权。

- **Kraven（克拉文，吸血鬼摄政/二号）**｜性格：野心、怯于正面、擅长谎言与借刀。｜装备·能力：贵族权限、枪械；个人武力低于顶级 Death Dealer 与 Lucian。｜人物弧光：假称杀死 Lucian → 通敌 → 泄露 Amelia 路线 → 背叛 Lucian → 本片后半失势（续集被 Marcus 清算）。｜立场关系：最危险的「自己人」；对契约者笑脸背后是灭口。

- **Amelia（阿米莉娅，吸血鬼元老·当值统治者）**｜性格：威仪、戏份短但象征秩序。｜装备·能力：元老级。｜人物弧光：车队转移 → 被莱肯伏击杀害。｜立场关系：权力真空制造者；勿写成可随便救活而不改因果。

- **Singe（辛格，莱肯科学家）**｜性格：理性、服务基因计划、被俘后仍吐科学真相。｜装备·能力：实验室与 Corvinus 理论；武力低。｜人物弧光：实验 Michael 血 → 被俘 → 当众揭穿 Kraven 与 Hybrid 原理 → 被 Viktor 杀死。｜立场关系：情报高价值目标；活口比尸体值钱。

- **Raze（雷兹，Lucian 副手）**｜性格：忠诚、凶猛、少话。｜装备·能力：巨型莱肯近战。｜人物弧光：随 Lucian 猎杀与守巢 → 混战中力战。｜立场关系：莱肯军事代表；可打可谈但优先听 Lucian。

- **Kahn（卡恩，Death Dealer 武备长）**｜性格：职业军人式可靠、重装备纪律。｜装备·能力：重火器、紫外线武器管理。｜人物弧光：协助宅邸防务与突袭编制。｜立场关系：Selene 的战友层；对契约者谈弹药与任务编号。

- **Erika（艾丽卡，吸血鬼交际花/贵族随从）**｜性格：攀附权力、嫉妒与自保并存。｜装备·能力：社交情报、吸血鬼基础能力。｜人物弧光：周旋于 Kraven 身边 → 在权力倾覆中求生。｜立场关系：宅邸情报贩；不可当绝对盟友。

- **Sonja（索尼娅，Viktor 之女·已故）**｜性格：通过血记忆与口述呈现——敢爱、抗父权。｜装备·能力：生前吸血鬼贵族；遗物项链半片为续集钥匙（本片埋伏笔）。｜人物弧光：与 Lucian 相爱怀孕 → 被 Viktor 处决 → 成为两族战争永久燃料。｜立场关系：不在场的核心；一切「憎恶」话语的源头。

- **Adam Lockwood（亚当·洛克伍德，Michael 同事）**｜性格：凡人同事层。｜装备·能力：无。｜人物弧光：医院日常对照。｜立场关系：人类世界标尺；可护可弃。

- **Soren（索伦，Kraven 打手）**｜性格：服从。｜装备·能力：吸血鬼打手级。｜人物弧光：执行 Kraven 脏活。｜立场关系：可贿可杀的中间层。

**【势力图谱】**
- **吸血鬼布达佩斯系巢穴（Coven）**：宗旨——维持对莱肯的优势与对人类世界的隐秘；制度——三元老轮值、Death Dealer 猎杀队、贵族社交层；代表——Viktor、Amelia、Marcus（未醒）、Kraven、Selene、Kahn、Erika。内部分「忠于元老」与「忠于 Kraven 权术」两条线。
- **莱肯族（Lycans）**：宗旨——摆脱奴隶史、以 Hybrid 与军事反击改写命运；地盘——下水道工业巢穴；代表——Lucian、Raze、Singe。与 Kraven 的密约是双刃。
- **Corvinus 血脉（隐线）**：非组织，是遗传坐标；Michael 为活体样本；Singe 的实验室是其「科学院」。
- **人类市政与医院**：不知情的表层；警察与医护是误伤与目击者来源。

敌友逻辑：银与 UV 决定战术；血统洁癖决定政治；密约比誓言更常见。任何「永久停战」承诺在本片时间线内都应默认附带背叛倒计时。站队前先问：你手里的子弹是银还是紫外线。

**【贵重物品】**
- **Corvinus 血脉 / Michael 的血样**：Hybrid 实验核心；谁掌握谁可改变战争技术树。
- **Sonja 的项链（半片）**：情感遗物与续集监狱钥匙的一半；本片中 Michael 侧相关。
- **银弹 / 硝酸银弹药**：反莱肯制式。
- **紫外线弹药（UV rounds）**：反吸血鬼制式；Kahn 武库核心。
- **Viktor 之剑**：终局斩首兵器。
- **冬眠石棺与轮值密钥/仪式**：政治合法性装置；提前唤醒=政变级事件。
- **Singe 的基因研究日志与仪器**：证明共同祖先与 Hybrid 路径。
- **血记忆本身（非物）**：咬合即情报窃取。
- **莱肯囚笼与镇定剂链**：控制变身与战俘。
- **Amelia 车队护卫编制与路线密件**：被泄露即等于杀死一位元老。

**【隐藏剧情 · 伏笔】**
- **Lucian 未死与 Kraven 密约**：官方史最大谎言；Selene 整条复仇生涯建立在错误仇敌上。
- **Viktor 杀 Selene 全家**：因建筑师家族知道某座监狱位置（续集展开）；本片先落「像 Sonja 故而留她」的情感动机。
- **Marcus 将醒**：Singe 之血入棺 → Hybrid 元老；第一部切在刀口上。
- **共同祖先论**：Singe 揭示两族同源，打破「天生死敌」神话。
- **Sonja 腹中胎儿**：Hybrid 禁忌的最早案例；Viktor 杀女的真正恐惧点。
- **系列互文**：续集将补 Alexander Corvinus、William 监狱、Cleaners；本片只埋 Corvinus 名与项链。

**【大事记时间线】**
- 中世纪 → Lucian 与 Sonja 相爱；Viktor 处决 Sonja；莱肯起义；Kraven 宣称杀死 Lucian。
- 约数百年前 → Selene 家族被屠；Viktor 转化 Selene 为吸血鬼。
- 现代·开篇夜 → 地铁枪战；锁定 Michael Corvin。
- 同夜~数日 → Lucian 咬伤 Michael；Selene 藏匿并提前唤醒 Viktor。
- 交接前夜 → Amelia 车队被灭；Amelia 死。
- 巢穴夜 → Singe 供词；突袭；Lucian 死；Michael Hybrid 化；Viktor 被斩首。
- 终局 → Selene 与 Michael 逃亡；Marcus 在棺中异变苏醒（钩子）。

**【叙事基调 · 雷区】**
画风：蓝冷色调、雨夜枪火、哥特皮革、工业下水道；口吻冷静残酷，浪漫线克制。  
忌：把 Selene 写成无脑花痴；把 Lucian 写成纯恶魔而无 Sonja 悲剧；把 Hybrid 写成无敌无代价；用修仙境界词替换病毒/血统体系；无视银/UV 克制乱写「气劲对轰」。  
最早可切入锚点：地铁枪战前夜的 Death Dealer 巡逻，或 Michael 尚在医院值班、尚未被两族锁定之时。契约者若以「人类记者/黑市医生」入世，第一夜应先学会分辨银弹与 UV 弹的气味与弹壳标记，再谈站队——选错弹药比选错阵营死得更快。

## 阶位切入点

> 阶位↔原作强度：一阶≈凡人市民/无武装医护；二阶≈普通莱肯·普通吸血鬼战士·持银/UV 枪的新人 Death Dealer；三阶≈精锐 Death Dealer（Selene/Kahn 级）·Raze 级莱肯猛将·Lucian 中层指挥；四阶≈Lucian 本尊组织战·Kraven 权谋高峰·实验室 Hybrid 临界；五阶≈Viktor/Amelia 元老·终局 Hybrid Michael。世界顶点（Viktor 与初成 Hybrid）为五阶档，低阶切入一律「规避顶点·情报优先」。

**一阶（凡人街区·医院与目击者线）**

切入身份/时点：契约者以夜班医护、法医助理、地铁安保或刚入境的普通人切入，时点锚定 **Michael Corvin 仍在医院值班、地铁枪战前后数小时**。

初始事件：雨夜急诊涌入「动物撕咬」伤患；监控里闪过黑衣持枪者与异形轮廓；你捡到一枚嵌银弹壳或一份被撕掉抬头的「生物污染」内部传真，随即有人要求你交还并「忘记」。

开场白建议：「消毒水味压不住巷口的火药。你刚把伤者推进手术室，窗外电车火花里掠过一个黑色身影——他跳过两层楼高的雨棚，像人类不该有的动作。对讲机里有人用冷静的声音说：今晚的伤员，不要写进正式病历。」

关键NPC立场：**Michael Corvin**（尚不知情的同事，可同盟）；**Selene**（远程监视猎物，视你为可变的目击者）；**Kraven** 的外围眼线（要灭口或收买）；医院主任层（要压新闻）。

主线钩子/支线：主线——保住 Michael 值班记录或枪战监控拷贝；支线——黑市收购卖「兽伤」故事；护送一名人类目击者出城；追查银弹来源。

危险度/规避：低~中。规避莱肯活捉小队与 Death Dealer 清场；禁止挑战任何变身体。

任务方向/奖励：情报、伪造身份、初级银/UV 弹药样本、医院门禁权限；不发元老级战利品。

**二阶（巷战与弹药线·初入地下）**

切入身份/时点：契约者以军火掮客、私家侦探或新晋吸血鬼家臣/莱肯外围切入，时点 **地铁交火后、Lucian 锁定 Michael 期间**。

初始事件：你被雇运送一箱「特殊子弹」；半路两拨买家火并——一拨要银弹，一拨要 UV 弹——箱子上烙着 Kahn 武库或莱肯工坊的标记。

开场白建议：「箱子比尸体沉。你刚在桥洞下交货，屋顶先后落下两个人影：一个闻起来像冷古龙水与旧血，一个呼吸里带着兽腥。他们同时问你：卖给哪一边？」

关键NPC立场：**Kahn**（要追回弹药）；**Raze**（要截胡）；**Erika**（愿用情报换保护）；**Soren**（Kraven 的棍棒）。

主线钩子/支线：主线——决定弹药流向并活着离开桥洞；支线——潜入夜店听 Kraven 派对闲话；为 Singe 的信使送试剂；假扮清洁工进下水道口。

危险度/规避：中。规避 Lucian 本尊与元老车队；银/UV 用错目标会立刻树敌全族。

任务方向/奖励：制式弹药、夜视与消音改装、地下地图、双边仇怨值（可控）。

**三阶（Death Dealer / 莱肯精锐线）**

切入身份/时点：契约者以资深 Death Dealer、莱肯突击手或双面间谍切入，时点 **Selene 藏匿 Michael、准备提前唤醒 Viktor 前后**。

初始事件：你收到互相矛盾的两道命令——Kraven 签令「停止猎狼」，Selene 私令「保护 Corvin 样本」；满月前夜，安全屋锁链另一端的男人开始骨裂变声。

开场白建议：「命令写在羊皮纸与手机短信里，内容相反。你站在安全屋铁门前，门后传来不属于人类的喉音。对讲里 Kraven 说杀了那个人类，Selene 的声音却只有三个字：别开门——给月亮。」

关键NPC立场：**Selene**（可结盟但多疑）；**Michael**（恐惧与信任边缘）；**Kraven**（要你站队）；**Lucian**（要活捉）；**Viktor**（若已醒，则要你解释抗命）。

主线钩子/支线：主线——满月变身夜的选择（杀/囚/放）；支线——录下 Kraven 与莱肯联络；抢在 Amelia 车队出发前改一道路口灯控；从 Erika 处买石棺室值班表。

危险度/规避：高。规避正面硬刚 Viktor；Amelia 车队是四~五阶事件，三阶只宜侧翼情报。

任务方向/奖励：高级弹药、血记忆片段（通过受控咬合/录像）、Death Dealer 权限卡、莱肯巢穴侧门钥匙。

**四阶（权谋与实验室线·Hybrid 临界）**

切入身份/时点：契约者以贵族使者、叛变科学家助手或双方特使切入，时点 **Singe 被俘前后至巢穴总攻前**。

初始事件：你掌握 Singe 实验摘要副本：Corvinus 基因、双病毒叠加、Hybrid 弱点假说；Viktor 要焚稿，Lucian 要完稿，Kraven 要你把摘要改成「只对自己有利」的版本。

开场白建议：「离心机还在转，纸页上写着同一个词：Corvinus。三方来使站在无菌灯下，没有人看显微镜，都在看你手里的U盘。谁先开枪，谁就承认自己怕真相。」

关键NPC立场：**Singe**（要科学被听见）；**Viktor**（要毁灭样本与证人）；**Lucian**（要完成 Hybrid）；**Selene**（要救 Michael）；**Kraven**（要所有对手互咬）。

主线钩子/支线：主线——摘要与活口（Singe/Michael）的归属；支线——制造假供词嫁祸；在突袭前五分钟切断巢穴主电源；保护或销毁 Sonja 项链线索。

危险度/规避：高~极高。顶点 Viktor 将亲自下场；契约者宜用规则、人质与环境杀，不靠硬刚。

任务方向/奖励：完整基因日志、Hybrid 血清半成品、元老罪证录音、势力跳反资格。

**五阶（元老终局线·斩首之夜）**

切入身份/时点：契约者以元老护卫残部、Lucian 遗志执行者或 Selene-Michael 同盟佣兵切入，时点 **巢穴总攻至 Viktor 斩首**。

初始事件：Amelia 已死的消息传遍地下；你踏入莱肯主厅时，Hybrid 初成的 Michael 与 Viktor 正在碎柱间对殴，Kraven 的枪口在暗处寻找下一个「可出卖的背影」。

开场白建议：「天花板在落灰，像下一场假雪。你看见银发的王掐着混血者的喉咙，黑衣女人的手伸向一把不属于她的剑。有人在你耳边说：今晚谁活下来，谁就改写两族的教科书——你，也算一笔。」

关键NPC立场：**Viktor**（顶点敌或条件性交涉对象）；**Selene**（需要剑与空档）；**Michael**（Hybrid 不稳）；**Kraven**（最后背刺窗口）；**Lucian**（若仍残命，要托付）。

主线钩子/支线：主线——促成或阻止斩首、决定 Kraven 死活、是否让 Marcus 之棺被血污染；支线——撤离通道清场；保存 Amelia 遗骸防血祭；向人类世界投放「煤气爆炸」掩盖故事。

危险度/规避：极高/贴近顶点。超阶式个人不存在，但 **Viktor 为五阶顶点**：存在·情报优先/条件性胜利；严禁写其「被莫名削弱」。

任务方向/奖励：Viktor 之剑残段、Hybrid 血样、两族通缉令下的亡命同盟、续作钩子（Marcus 苏醒）的优先知情权。
