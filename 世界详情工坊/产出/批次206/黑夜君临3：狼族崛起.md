# 黑夜君临3：狼族崛起
<!--meta lib=主库 tiers=一、二、三、四、五-->

## 剧情

**【作品来源】**
《黑夜君临3：狼族崛起》（英文原名 *Underworld: Rise of the Lycans*，2009）为美国动作恐怖电影，由 Patrick Tatopoulos 执导，Len Wiseman / Danny McBride 等参与故事与剧本，Screen Gems / Lakeshore 出品。本片是《黑夜君临》系列时间线上的**前传**：舞台定在中世纪东欧吸血鬼要塞，讲述第一只可化人形态的狼族 **Lucian** 与吸血鬼长老 **Viktor** 之女 **Sonja** 的禁忌之恋，以及狼族奴隶起义如何点燃吸血鬼—狼族千年血战。气质：暗黑、压迫、银项圈与日光处决；政治阴谋重于枪战。轮回乐园归**主库**，覆盖阶位：一、二、三、四、五。查不到写不详，严禁编造原作没有的终局与宝物来历。

**【世界定位】**
中世纪吸血鬼领主制下的东欧：Viktor 以银质枷锁奴役可化形的狼族 Lycan，野外则游荡着无法变回人形的第一代野狼（William 血脉余孽）。主线视点是奴隶铁匠 **Lucian**——他自幼被 Viktor 收养为“日间守卫与夜班劳力”的样板，却与 **Sonja** 秘密相爱，最终因妻子与未出世骨肉被日光处死而率众攻破要塞。一句话：银项圈帝国的奴隶起义史诗，契约者是变数不是天选免死金牌。

**【世界观 · 力量体系】**
力量本源：Corvinus 不朽病毒在吸血与狼化两条线上的分化。吸血鬼靠血液维持超凡体能、愈合与长生；狼族（Lycan）可在人形态与狼形态间切换，狼形态力量与速度通常压过同龄吸血鬼，但对银极度过敏——银能抑制变身、灼伤血肉，重创可致死。第一代野狼（William 直系感染）多数**无法变回人形**，近乎野兽；Lucian 是第一只天生可化人形态的“真·Lycan”，可在成年后召唤并短暂统御部分野狼。

死亡规则：断颈、重创头/心、放血、日光（吸血鬼）、银（狼族）、狼毒/夜影等毒素均可致死；本片无无限复活。血记忆：吸血鬼可通过咬食血液读取对方记忆（Viktor 对 Sonja 使用）。

战力阶梯（写表现优先）：
1. **人类农奴/贵族扈从**：冷兵器凡人。乐园≈一阶边缘。
2. **新晋狼族奴隶 / 新血吸血鬼侍从**：略超凡人，银项圈下难完全变身。乐园≈一阶。
3. **成年 Lycan 劳工 / 普通 Death Dealer**：可完整变身或熟练用银刃弩弓；能撕开甲胄、翻越城墙。乐园≈二阶。
4. **精英 Death Dealer 队长 / 狼族猛将（如 Raze）**：小分队级清场，可单挑多名同阶。乐园≈三阶。
5. **Sonja 级贵族战力 / Lucian 解放后统帅态**：可指挥野狼潮、与长老短暂缠斗。乐园≈四阶。
6. **吸血鬼长老 Viktor（醒着）**：中世纪最强吸血鬼之一，经验、阴谋与武力并存；可在要塞内碾压多数个体。乐园≈五阶。
7. **野狼潮 / 要塞级攻防**：非单体境界，是本片“战力上限场景”——群狼灌城可屠贵族与议会。乐园观感≈四~五阶场面。
8. **William 本尊 / 更早的 Markus**：本片多作背景威胁与传说，低阶禁止硬刚本体。乐园≈更高档/超阶边缘（情报优先）。

乐园阶位映射：人类与戴项圈奴隶≈一阶；成年 Lycan / 普通 Death Dealer≈二阶；精英战士（Raze 等）≈三阶；Lucian·Sonja 高潮战力≈四阶；Viktor 醒着≈五阶；William 野狼本源与 Markus 系长老=更高档·低阶规避。宁低勿高；顶点忠于原作。

补充规则细节（写战斗与任务时用）：银对狼是“毒+抑制”，对吸血鬼无同等特效；日光对吸血鬼是战略级环境杀，对狼无效。血记忆不是全知，需咬破吸血读取，过程可被反抗或中断。野狼统御不是无限群控，Lucian 也需代价与专注，战后狼群仍可能反噬人类村庄。Death Dealer 的优势在编制、银武与城堡地形，不在“单兵永远强过狼形态”。人类在本片是资源与舞台，不是可忽视的背景板——没有贡银，项圈工业与庇护合约都会停摆。

**【地理 · 舞台】**
核心舞台是 **Viktor 的吸血鬼要塞**（东欧山地/河谷地带，中世纪石堡，哥特式拱廊与铁栅并存）：日间由 Lycan 奴隶在银项圈压制下守卫与劳作，夜间吸血鬼贵族活动。要塞不是单一大厅，而是分层权力机器——**城门与吊桥**控制人类贡使进出；**下层奴隶区**含铁匠铺、锁链大厅、项圈锻造间、鞭刑柱与潮湿地牢；**中层军械廊**供 Death Dealer 换装、银弩库与马厩；**上层贵族翼**含议会厅、Viktor 议事室、Sonja 居室与宴会廊；**日光处决庭**是可开合天窗的露天石院，法律与恐怖在此合一。要塞外围是被 William 血脉野狼侵扰的**人类贵族领地与农庄**：领主以白银进贡换夜间庇护，贡道即商路也是情报路。森林与河谷是野狼的“流动国境”，夜间嚎声决定人类是否敢出门。Lucian 解放后的活动圈扩大到**人类庄园银库与奴隶屋**——他在那里用“自由+不朽”招募。终局战场锁定要塞内庭、城墙垛口与光影交错的水池/通道：野狼灌城、议会屠戮、Viktor 与 Lucian 决战。AI 写场景先定契约者在：奴隶区 / 军械廊 / 贵族走廊 / 贡银商路 / 野狼森林 / 庄园火场 哪一层，再决定冲突强度（配额鞭刑 vs 议会阴谋 vs 破城屠杀）。

**【世界剧情线】**
① **前史·William 之祸与奴隶制度**  
远早于本片主线：不朽者 Alexander Corvinus 的儿子 **William** 被狼咬后成为第一代无法化人的狂狼，在匈牙利乡野以极强感染力传播狼化，连死者都可能复起为兽；其孪生兄弟 **Markus** 被蝙蝠咬后成为吸血鬼始祖。为对抗野狼潮，Markus 招揽濒死军阀 **Viktor**，以不朽换其军事天才与军队——这支军队演化为专职猎狼的 **Death Dealers**。另有女长老 **Amelia** 等进入长老平衡局（本片多作背景）。战争持续数十年后，Viktor 抓获一只产下**可化人幼崽**的母狼——幼崽即 **Lucian**。Viktor 杀死其母，在 Lucian 身上烙下奴隶印记，设想用可化形的 Lycan 作为**日间守卫与夜间苦力**：白天他们能在阳光下活动以补吸血鬼的致命短板，夜间则做工、运货、守库。控制手段是**银质项圈、锁链与鞭刑**——银抑制变身、灼伤血肉。与此并行，人类贵族以白银进贡换取庇护；白银既是货币，也是刑具原料。制度逻辑冷酷而自洽：野狼可怕 → 人类求吸血鬼 → 吸血鬼用银奴役可化形狼 → 可化形狼成为制度的齿轮。

② **开篇·银项圈下的样板奴隶**  
成年 Lucian 已是要塞内最得用的奴隶铁匠与“模范狼族”：人形态时聪慧、克制、会说话会谈判，忠诚表象完整，实则把愤怒锻进每一条锁链。Viktor 公开把他当制度广告——“看，狼也可以文明”。与此同时，Viktor 之女 **Sonja** 任性、擅战、对议会礼仪不耐，常违抗父命夜出巡猎或私会。二人自幼在同一屋檐下成长：一个是烙印奴隶，一个是长老之女；成年后堕入禁忌恋，并秘密成婚。这触犯三重禁忌——跨种族、跨主奴、跨“契约”（Covenant）所维护的血统纯洁妄想。日常张力来自：Sonja 必须在议会露面，Lucian 必须在点名时低头；一次眼神过久就可能被 Death Dealer 记下。名场面底色：锁链摩擦、银项圈灼肤、铁匠炉火映在 Sonja 披风内侧的纹章上；他们的爱不是浪漫点缀，而是炸弹的引信。

③ **立名·夜袭、鞭刑与第一次解放**  
Sonja 某夜在外围行动中陷入野狼或敌对威胁，Lucian 为救她挣脱抑制变身的枷锁，以狼形态杀出重围。Viktor 承认女儿得救，却不能容忍奴隶“违禁变身与脱逃”——秩序比恩情重要。Lucian 被当众鞭笞、投入地牢，银器与锁链双重示众，旨在吓阻其余奴隶。Sonja 不走父女求情的老路，她以**出让自己在吸血鬼议会的席位**为政治硬通货，说服史官兼阴谋家 **Andreas Tanis** 安排越狱通道：Tanis 要的是上升，Sonja 要的是爱人，交易即时成交。Lucian 逃出时拒绝独活，顺势打开锁链库与项圈库，**解放大批 Lycan 奴隶**；有人跟着跑，有人吓得跪回原位——自由第一次变成可选的、也是可死的。Sonja 暂留要塞以掩人耳目，约三日后与 Lucian 会合。此阶段因果：Lucian 完成从“模范奴隶”到“逃亡领袖”的身份跃迁；Tanis 拿到席位承诺，开始在 Viktor 与叛军两边记账；Death Dealer 进入高度戒备，奴隶区宵禁加码。

④ **质变·血记忆、怀孕与日光处决（卡片锚点卷）**  
Viktor 直觉女儿涉案，在对质中咬破 Sonja 脖颈，以**血记忆**强行读取真相：恋情、越狱协助、身体里的新生命。对 Viktor 而言，这不是家事，而是对吸血鬼神话与权力合法性的污损——他甚至曾暗示 Sonja 未来可成“长老级”自己人，此刻全部作废。Sonja 被囚于高塔/居室变监牢。外围：Lucian 招揽人类奴隶与部分野狼，袭击进贡贵族庄园，承诺“跟我走，以狼之身换不朽与自由”；庄园起火、银库被破、贵族向议会哭诉。议会与贵族联合施压：必须抓回 Lucian，否则贡银体系与庇护合约崩盘。Viktor 的回应阴冷而精准——他不必满山搜狼，他手里有 Lucian 最想要的 **Sonja**。Lucian 果然潜入营救，短暂重逢后被 Viktor 截住。Sonja 为护 Lucian，当众坦白**已怀上 Lucian 的孩子**。Viktor 视混血胎儿为“污秽与末日”，当场制服女儿与女婿。随后是法律仪式化的杀戮：议会全票死刑，父亲 Viktor 亲自主持，在 Lucian 被镣铐固定、被迫直视的情况下打开天窗，**日光处决**烧死 Sonja 与腹中胎儿。Sonja 未求饶的沉默与 Viktor 的“纯洁”说辞同样刺耳。Lucian 夺下/保有 Sonja 的吊坠（Viktor 曾赠女的信物），仇恨从此有了可触摸的形状。名场面清单（写文必备）：议会投票的手势、天窗齿轮咬合、阳光如刀、银鞭与 Death Dealer 阵列、吊坠反光、Lucian 喉咙里未成形的狼嚎。

⑤ **格局战·野狼灌城与要塞易主**  
处决后的 Lucian 完全狼化冲击封锁线，仍一度被 Death Dealer 压制于城内；关键转折是他发出对**野外第一代野狼**的呼唤——第二代可化形领袖与第一代狂兽之间出现可统御的共鸣。野狼潮如黑水灌入要塞：城门、廊桥、议会厅相继失守；议会成员、扈从与部分贵族在混战中被撕碎，吸血鬼数百年的“绝对安全屋”神话当夜破产。Viktor 试图有序撤离以保长老本尊，Lucian 追击至光影交错处，利用阳光射入的几何死角限制其游走，近身剑斗中将剑刺入 Viktor 口中并推入水中。副手 **Raze** 环顾尸山，说“结束了”；Lucian 否定：这只是**种族战争的开端**——奴隶破城不等于历史正义自动降临，逃亡的贵族、存活的长老、人类的恐惧都会把今晚写成下一场屠杀的理由。

⑥ **终局余波·Viktor 假死入眠与对第一部的钩子**  
黑船离岸。重伤未死的 Viktor 被 **Tanis** 封入长老休眠舱——Tanis 再次证明自己是“谁都要欠他人情”的活体档案库。要塞易手、狼族获得战略胜利，但领导结构、补给与人类关系全是白纸。时间跳至后世（系列第一部开场钩子）：吸血鬼 **Kraven** 对 **Selene** 透露——杀害 Selene 家族的是 Viktor 而非狼族；Viktor 因 Selene 酷似 Sonja 而在灭门时留她一命，并把她训练成 Death Dealer。Selene 斥为谎言。本片在叙事上闭合于：起义成功、爱人已死、暴君未死、谎言将传百年。契约者若活到“战后周”，面对的是权力真空、复仇清单与银库再分配，而不是庆功宴。

⑦ **微观政治贯穿（日常可介入）**  
贡银车队编成与劫道、项圈锻造的日配额、奴隶鞭笞示众与连坐、人类贵族求庇护的谈判桌、议会席位私下买卖、Tanis 式情报双卖、Death Dealer 夜巡与口令、野狼袭村后的收尸与嫁祸、铁匠铺夹带私货、地牢换班表。AI 优先这些可玩的小事件，让玩家在“井底”也能感到制度的牙齿，而不是一上来长老决战。

⑧ **蝴蝶效应边界**  
契约者可改局部：救某个将被鞭死的奴隶、截一车贡银改写当季配额、提前警告 Sonja 勿在血记忆前归城、刺杀某 Death Dealer 队长、在灌城夜打开侧门放走平民。不可无代价改写：Sonja 之死作为战争火种的核心地位、Lucian 从奴隶到领袖的弧光、Viktor 存活入眠以衔接后世、William 野狼威胁作为生态背景的存在。改了局部，原作因果常以“更早屠城、更狠清洗、Lucian 更极端、Tanis 换边更快”回流。

**【世界剧情线·阶段细读：奴隶日常】**  
点名以银铃声开始。项圈工坊的配额写在石板上：今日完成多少环、废品如何连坐。Lucian 作为铁匠既是生产力也是“榜样”，他必须在鞭子与同情之间走钢丝——多给新奴一个提示可能被记为煽动，少给一口热水又会被同类记恨。Sonja 若下到奴隶区，全场呼吸都会变浅：贵族的脚步声等于审查。可介入事件：调换废银环、篡改点名册、把野狼来袭的假消息传给某队 Death Dealer 以调离看守、为即将被鞭死的少年换班。此阶段世界状态=一阶叙事，强者如石像，契约者先学“如何在锁链下搬家”。

**【世界剧情线·阶段细读：越狱夜的几何】**  
越狱不是开门就跑。通道分三层：地牢锁、中层军械岗、外门吊桥。Tanis 提供的是“某一扇门在某一刻无人”的时间缝，不是无敌通行证。Lucian 选择解放同伴，意味着噪音、犹豫者、告密者同时出现——自由军从第一秒起就不是精锐军团。Sonja 留城是诱饵也是盾：她必须继续在贵族宴上举杯，假装无事。可介入事件：伪造 Viktor 手令、在马厩放走备用坐骑、用炉渣烟雾遮挡廊桥视线、在项圈库门口与 Sabas 巡逻队错峰。此阶段结束时，Lucian 的名字从“奴隶”变成“通缉令上的字”。

**【世界剧情线·阶段细读：庄园火与议会怕】**  
人类庄园被袭的意义超过抢银：它证明吸血鬼的庇护合约可被撕毁，贡银贵族会把恐慌翻译成对 Viktor 的政治压力。议会要的是可展示的“狼首”，不是长老的耐心布局。Viktor 偏要用 Sonja 作饵，展示他对 Lucian 心理的精确阅读——暴君懂爱，所以更会利用爱。可介入事件：在议会走廊散播“Sonja 已死/已逃”的假讯以打乱时间表、护送一车被解放的人类奴隶过河、潜入高塔给 Sonja 递刀具或假钥匙、记录天窗机构的齿轮位置供日后破坏。此阶段世界状态=三到四阶压力，信息比刀锋更值钱。

**【世界剧情线·阶段细读：日光之后】**  
阳光落下后，语言失效。Lucian 的统御野狼是悲愤的延伸，不是突然觉醒的“外挂说明书”。破城后的权力真空里，有人要屠尽贵族，有人要谈判要银库，有人要带着家人逃进山。Raze 适合当刀，不适合当外交官；Lucian 必须同时是复仇者与立法者，而他刚失去立法的理由（妻子与孩子）。船上的 Viktor 入眠把“胜利”改写成“延期宣判”。可介入事件：争夺休眠舱坐标、伪造 Viktor 死讯稳定军心、阻止野狼屠村、保存一份真实的日光处决记录以防历史再被篡改。此阶段适合五阶切入：战争元年的建国与追杀并行。

**【主要人物】**
- **Lucian（第一只可化人 Lycan / 起义领袖）**｜性格：隐忍、聪慧、重情，被逼到极点后冷酷决绝；对人说话像铁匠量尺寸，对敌人则不再留余料。｜装备·能力：人/狼双形态完整切换；铁匠与械具知识；银项圈下的生存与伪装智慧；解放后可呼唤并短暂统御第一代野狼；持有 Sonja 吊坠作为记忆与动员符号。｜人物弧光：烙印幼崽→模范奴隶→禁忌恋人→解放者→战争之父。｜立场关系：爱 Sonja 至死；仇 Viktor 入骨；信任 Raze；对人类奴隶可盟可弃；对契约者先审“你能带来几条命的自由”，软肋是亡妻与未出世孩子。
- **Sonja（Viktor 之女 / 议会席位持有者）**｜性格：骄傲、勇武、不服管教，爱憎分明，敢用政治资本换一条命。｜装备·能力：吸血鬼贵族战力，骑射与近战娴熟，可与 Death Dealer 同级搏杀；拥有议会席位（后出让）。｜人物弧光：任性贵族→秘密妻子与母亲→为爱牺牲席位与生命。｜立场关系：对 Viktor 是反抗的女儿而非无脑叛国；与 Lucian 秘密成婚怀孕；被日光处决。对契约者：助她护 Lucian 可短暂信任，出卖即死敌。
- **Viktor（吸血鬼长老 / 要塞主宰）**｜性格：残酷、伪父爱、权力狂；把“血统纯洁”与“秩序”当成杀人许可证；对 Lucian 有扭曲的养育者傲慢。｜装备·能力：长老级吸血鬼巅峰，剑术、体能与战场经验碾压绝大多数；血记忆读取；贡银—议会—Death Dealer 三线操控。｜人物弧光：猎狼军阀→奴隶制设计师→处决亲女的暴君→重伤入眠的未来威胁。｜立场关系：仇混血；榨人类；与 Markus/Amelia 长老局有历史博弈（侧写）。对契约者：有用则用，染指 Sonja/Lucian 即灭口。**五阶顶点感·条件性胜利，严禁战力归零。**
- **Andreas Tanis（史官 / 阴谋掮客）**｜性格：圆滑、投机、记仇亦记账，笑容比剑危险。｜装备·能力：中高阶吸血鬼；真正武器是档案、伪证、通道与双头合同。｜人物弧光：用 Sonja 席位换越狱协助→要塞崩盘护送 Viktor 入眠→后世因“真实记录”被 Viktor 流放（跨片）。｜立场关系：谁出价高帮谁；对契约者可买可卖，合同永远有第二页。
- **Raze（Razahir Khemse，Lucian 副手）**｜性格：沉稳、忠诚、战场直觉强；少废话，多结果。｜装备·能力：精英 Lycan，人形态即魁梧战士，狼形态可单挑强敌；组织突击与断后。｜人物弧光：奴隶→起义副帅→灌城夜的执行核心。｜立场关系：忠 Lucian；对吸血鬼无怜悯；考核外人只用战绩。
- **Coloman（议会贵族）**｜性格：要脸、要程序、要贡银稳定。｜装备·能力：贵族吸血鬼，个人武力未必顶尖，话语权在议会。｜人物弧光：代表“制度要 Lucian 人头”的集体意志，破城时成为被清算对象之一。｜立场关系：逼 Viktor 行动；仇视奴隶解放。
- **Orsova（议会/贵族侧）**｜性格：冷、护特权。｜装备·能力：贵族层战力与人脉。｜人物弧光：旧秩序的齿轮，在狼潮中破碎。｜立场关系：与 Coloman 同类，利益同盟。
- **Sabas（Death Dealer）**｜性格：服从、残忍、职业化。｜装备·能力：银刃、弩、骑枪；镇压与处刑熟练。｜人物弧光：制度之牙，越狱前后的主要军事阻力。｜立场关系：对奴隶零同情；只听长老与队长链。
- **Kosta（Death Dealer 体系成员）**｜性格：执行命令的冷硬。｜装备·能力：制式银武与骑战。｜人物弧光：夜巡与封锁线的具名阻力。｜立场关系：可被设计调离岗位，不可被“感化”。
- **人类贡银贵族（片中 Nobleman 线）**｜性格：怕野狼更怕失去庄园与继承人。｜装备·能力：凡人+扈从+银库。｜人物弧光：以银换庇护→庄园被 Lucian 军袭击→向议会哭诉。｜立场关系：可交易银与情报，不可共生死。
- **野外第一代野狼（William 血脉）**｜性格：非人、饥渴、无谈判。｜装备·能力：无法化人，感染强，群体可破城。｜人物弧光：背景生态威胁→Lucian 终局的生物攻城锤。｜立场关系：可被 Lucian 短暂统御；对契约者默认敌对。
- **Selene（片尾钩子）**｜性格：冷冽、被骗的忠诚（后世）。｜装备·能力：本片不展开；后世顶尖 Death Dealer。｜人物弧光：因酷似 Sonja 被 Viktor 留命。｜立场关系：本片时点未入主线；高阶可作未来因果情报。
- **Kraven（片尾声线）**｜性格：怯懦投机（跨片）。｜装备·能力：本片仅声线。｜人物弧光：向 Selene 泄密。｜立场关系：情报钩子。

**【势力图谱】**
- **Viktor 要塞吸血鬼政权**：长老独裁+议会背书+Death Dealer 暴力；地盘=要塞、贡道与庇护合约网；代表=Viktor、Sonja（叛出）、Tanis、Sabas 等；战力在本地是天，破城后权威破产。
- **Lycan 奴隶群体 → 起义军**：从锁链劳工到自由军；代表=Lucian、Raze；资源=解放人口、劫掠银库、野狼同盟；纪律参差，胜利后易内裂。
- **人类贵族进贡联盟**：以白银买命；两头受气；是银、粮、消息的来源，也是双方勒索对象。
- **野外野狼群**：无政治，只有饥渴与感染；被 Lucian 短时统御即改变战局。
- **长老休眠/后世吸血鬼国家（伏笔）**：本片末 Viktor 入眠，连接第一部苏醒政治与谎言史。  
敌友逻辑：血统与主奴压迫优先于盟约；暴露混血/协助越狱=死刑；露银露兵必招夺；Tanis 式中间人永远给自己留船舱。

**【贵重物品】**
- **银质项圈与锁链**：奴役核心刑具；抑制变身；项圈库=奴隶区政权钥匙。抢到可解放也可二次奴役。
- **贡银（白银硬通货）**：进贡物与军工原料；经济线与战争线的交叉点。
- **Sonja 的吊坠**：Viktor 赠女信物；死后 Lucian 保存；跨片成 *Evolution* 关键钥匙。本片内=仇恨与动员圣物。真品极难夺，低阶只碰仿品/拓印/传闻。
- **议会席位（政治筹码）**：Sonja 出让给 Tanis；可换通道、伪证与沉默。
- **长老休眠舱**：收容 Viktor；后世权力轮替容器；坐标=顶级机密。
- **Death Dealer 银刃/银弩矢**：对狼特攻；流入黑市会改变局部战力平衡。
- **烙印与奴隶名册**：身份控制工具；烧毁名册=行政意义的解放。  
谁想要什么：Viktor 要秩序与纯洁神话；Lucian 要自由与复仇；Tanis 要永远有用；人类要活命与银。一阶就能用银器害人；休眠舱与真吊坠禁止越级发放。

**【隐藏剧情 · 伏笔】**
- Sonja 之死是千年血战的正式点火，不是孤立家事；每场后世仇杀都可回溯到这束日光。
- Lucian 统御野狼，暗示第一代/第二代狼的血缘层级与“可沟通的兽性”。
- Viktor 未死+入眠=第一部苏醒前提；他对 Selene 的“温柔”是对 Sonja 的扭曲补偿。
- Tanis 的真实史书日后被 Viktor 诬为谎言而流放——记录者与篡史者的长线。
- 混血胎儿被杀，把“吸血鬼×狼族混血”钉成禁忌，对照后世 Michael/Eve。
- 片尾咬合第一部：Kraven 的真话 vs Selene 的拒绝=被篡改历史的受害者循环。
- 人类贡银体系若崩溃，吸血鬼也会失去日间奴隶经济与银来源，内战必然外溢。

**【叙事基调 · 雷区】**

## 阶位切入点

> 阶位↔原作强度：人类/戴项圈奴隶≈一阶；成年 Lycan/普通 Death Dealer≈二阶；精英战士（Raze 级）≈三阶；Lucian·Sonja 高潮≈四阶；Viktor 醒着≈五阶。William 本尊与更高长老=规避·情报优先。本世界仅覆盖：一阶、二阶、三阶、四阶、五阶。

**一阶（银项圈奴隶区 · 贡银与鞭影）**
切入身份/时点：契约者以人类贡银车队护卫、新捕奴隶、或低阶杂役切入，锚定 **Lucian 仍为模范奴隶、Sonja 尚未入狱** 前后。
初始事件：夜班点名时，一名 Lycan 因项圈松动被 Death Dealer **Sabas** 当众鞭笞；车队账簿缺银，贵族归罪于你；你必须选择：告发同袍换口粮、帮 Lucian 藏一件半成品银环、或把消息卖给 Tanis 的眼线。
开场白建议：「要塞底层的空气里是铁锈、湿石与烧焦毛皮的味道。银项圈贴着你的锁骨发烫，头顶石缝漏下一点月光——那是奴隶唯一被允许看见的“外面”。远处铁匠铺里，Lucian 的锤声一下一下敲着，像在数谁今晚会被拖去示众。」
关键NPC立场：**Lucian**——审慎观察，不轻易托命；**Sonja**——若你涉她夜巡路线会警惕；**Viktor**——根本看不见你，除非你坏了配额；**Andreas Tanis**——愿出低价买嘴；**Sabas**——找茬的鞭子。
主线钩子/支线：挂钩“项圈库钥匙流转”；支线＝护贡银车过野狼区、偷换损坏项圈、替 Sonja 递一封无署名短笺。
危险度/规避：中——死于鞭刑、银灼、告密链；须规避 Viktor 本体与完整野狼群。
任务方向/奖励：活过点名周并拿到一件可用银器/通行腰牌；奖励取向＝银粉、路线图、奴隶区人情。

**二阶（夜巡与越狱前夜 · Death Dealer 走廊）**
切入身份/时点：以资浅 Death Dealer、自由民佣兵或已挣脱项圈的新 Lycan 切入，锚定 **Lucian 救下 Sonja 后入狱、Tanis 安排越狱** 前后。
初始事件：你被编入夜巡，奉命搜查铁匠铺违禁物；同时 Tanis 的人塞给你一枚议会印章蜡，要你“恰好看反方向”。走廊另一头 Sonja 正在为席位交易压价。
开场白建议：「弩矢上的银光比烛火更冷。你的队长说：今晚谁放走狼，谁就去填护城河。可你刚转过拱门，就听见锁链一根根落地的声音——有人在解放整排奴隶，而那人的名字，全要塞都认识：Lucian。」
关键NPC立场：**Lucian**——要人手与向导，会给自由不会给信任；**Sonja**——以席位换行动，讨厌被同情；**Andreas Tanis**——双头合同，谁赢站谁；**Raze**——只认战场表现；**Viktor**——开始怀疑女儿。
主线钩子/支线：介入“越狱当夜开门顺序”；支线＝伪造点名册、转移 Sonja 的坐骑、截获议会密信。
危险度/规避：中高——死于同袍清洗与狼化误伤；规避 Viktor 亲自审讯。
任务方向/奖励：在越狱夜活着选边；奖励＝银刃、假身份、Tanis 一条情报。

**三阶（袭庄园扩军 · 议会逼宫）**
切入身份/时点：以起义军小队头目、人类庄园武官或吸血鬼贵族扈从切入，锚定 **Lucian 袭击进贡庄园、议会要求 Viktor 抓人** 前后。
初始事件：某人类领主的银库被破，奴隶喊着 Lucian 的名字冲进夜色；议会使节要求你方三日内交出“狼首”，否则取消庇护。与此同时有线报：Sonja 已被软禁。
开场白建议：「燃烧的庄园把半边天映成血色。狼在笑，贵族在喊价，议会的信使把“和平”两个字咬得跟银币一样硬。你知道 Lucian 会回来——不是为银，是为塔里那个被父亲锁住的女人。」
关键NPC立场：**Lucian**——招兵买马，允许暂时同盟；**Raze**——考核你是否拖后腿；**Viktor**——用 Sonja 作饵的布局者；**Coloman** 类议员——要脸面与银；庄园领主——愿出钱买一条活路。
主线钩子/支线：是否参与营救 Sonja 的潜入；支线＝护送被解放奴隶过河、伪造 Viktor 军令、抢夺一批银弩。
危险度/规避：高——死于 Death Dealer 精锐反扑；规避日光处决庭的戒严圈。
任务方向/奖励：拿到营救路线或议会投票倾向；奖励＝狼牙徽章、银弩、人质交换权。

**四阶（日光处决与灌城 · 战争点火）**
切入身份/时点：以 Lucian 侧先锋、叛变 Death Dealer 或 Tanis 特使切入，锚定 **Sonja 受审至野狼灌城** 当夜。
初始事件：你被允许进入旁听席或地牢外围——议会全票死刑，天窗机构开始咬合；Lucian 在锁链另一端，Sonja 的吊坠在光里晃。处决结束后，城外第一声狼嚎响起。
开场白建议：「阳光落下来的时候，连火把都显得懦弱。Sonja 没有求饶，Viktor 的脸比石墙更冷。你听见 Lucian 的骨头在变形，也听见历史被人用日光签了名——从这一秒起，和平只是笑话。」
关键NPC立场：**Lucian**——只剩战争；**Viktor**——要撤离与清算；**Raze**——执行破城；**Andreas Tanis**——已在准备船舱后路；**Sonja**——此阶段已逝，仅以遗物与记忆存在。
主线钩子/支线：是否打开某道侧门放野狼/放贵族逃命；支线＝夺回 Sonja 吊坠（极难）、护送一名议员换情报、破坏天窗机构“迟到一分钟”。
危险度/规避：极高——死于狼潮误杀或 Viktor 剑下；**Viktor** 为五阶级压迫，条件性周旋，严禁硬刚当儿戏。
任务方向/奖励：在灌城夜活着离开要塞并带走一件可证历史的信物；奖励＝真假难辨的吊坠拓印、船舱航线、起义军坐标。

**五阶（船舱入眠与战争元年 · 长老余波）**
切入身份/时点：以双方残存指挥官、Tanis 随船密使或跨域吸血鬼使节切入，锚定 **Viktor 被封入休眠舱、Lucian 宣告战争开始** 之后数日到数周。
初始事件：湖上黑船离岸，舱内长老气息沉如死；岸上 Lucian 拒绝“胜利庆典”，下令清点银库与狼群。有人悬赏“杀死 Lucian 者封爵”，也有人悬赏“唤醒 Viktor 者分享新秩序”。
开场白建议：「要塞还在冒烟，可真正的火在人们眼睛里。Raze 说结束了，Lucian 说这才开始。你站在码头，手心是一枚还带着体温的银环——它可以是枷锁，也可以是戒指。选择哪一种，会决定你在接下来的千年里被谁猎杀。」
关键NPC立场：**Lucian**——多疑的战争领袖；**Raze**——军事执行；**Andreas Tanis**——掌握 Viktor 入眠秘密；**Viktor**——休眠中，仍是政治核武；后世线的 **Kraven/Selene** 仅作情报传说。
主线钩子/支线：守护或泄露休眠舱坐标；支线＝建立第一批自由狼族聚落、追杀漏网议员、伪造“Viktor 已死”公告。
危险度/规避：高——死于清洗与暗杀；休眠 **Viktor** 为超规格隐患：存在·情报优先，禁止“偷舱秒杀长老”的儿戏写法。
任务方向/奖励：确立战后立场并获得一块地盘/一条秘密航线；奖励＝兵权、银库份额、真实史书残页（Tanis 风格）。
