# 机神咆吼 Demonbane

## 剧情

**【作品来源】**
日本 Nitro+（ニトロプラス）视觉小说系列，主笔钢屋ジン（Jin Haganeya），原画 Niθ。2003-04-25 发售 18 禁 PC 作《斩魔大圣デモンベイン》（Zanma Taisei / Deus Machina Demonbane）；2004-07-01 角川书店推出全年龄 PS2 移植《机神咆吼デモンベイン》（Kishin Houkou Demonbane，本作世界书标题所取）；2006-05-26 续作《机神飞翔デモンベイン》（Kishin Hishou）。衍生含 View Works 制作 TV 动画 12 话（2006，WOWOW）、OVA、角川 Sneaker 文库小说 6 卷（本篇 3+外传 3）、漫画与《超级机器人大战 UX》客串。文风：荒唐无稽超级机器人 × 克苏鲁神话 × 热血魔导英雄谭；台词与机体名大量戏仿特摄/机战，同时把「旧日支配者」「外神」「魔导书精灵」当真。**剧情主轴以 PC/PS2 本篇（斩魔大圣/机神咆吼）为准，飞翔作与小说作补充因果与后续。** 搜笔趣阁未收录该 IP（非中文网文），资料以维基、ANN 动画百科与公开剧情综述为准。

**【世界定位】**
科学与魔导技术双轨繁荣的克苏鲁式北美都市——**阿卡姆市（Arkham City）**。贫穷侦探**大十字九郎**受**霸道财阀**当主**霸道瑠璃**委托寻找魔导书，与魔导书「阿尔·阿吉弗」本源精灵**阿尔·阿ジフ（Al-Azif）**契约，驾驶仿制鬼械神**戴蒙贝恩（Demonbane）**，对抗秘密结社**黑衣结社 / ブラックロッジ（Black Lodge）**及其大导师**マスター・テリオン（Master Therion / 雷德利亚德·克劳利系称号）**与七名**逆十字（アンチクロス / Anticross）**干部。主线=魔导书页片回收→因斯茅斯/达贡→C 计划召唤克苏鲁→幻影心脏母舰→穿过约格-索托斯之门，与 Liber Legis 时空对决，并揭开幕后**奈亚拉托提普（Nya / Nyarlathotep）**的棋局。顶点线可触及「旧神」化与外神级存在。

**【世界观 · 力量体系】**
本世界力量来源分四层，可并存、可叠乘：
1. **科学·炼金·机械**：纯物理巨型兵器（**破坏机器人 / Destroyer Robot** 系列）、银键守护神机关、动力炉「狮子的心脏（Cor Leonis）」等。科学侧可仿造鬼械神外壳，但缺少真正魔导书召唤的「神性现界」。
2. **魔导书与契约**：魔导书（Grimoire）可具现为精灵（如阿尔、艾塞尔德蕾达、叶月）。术者与魔导书订立契约后获得魔术；阿尔可将九郎变身为**魔导士形态（Magius Style）**——长发、贴身装束、魔力增幅，并在驾驶舱内以「ちびアル」形态辅助。
3. **鬼械神（Deus Machina）**：由魔导书召唤或人类仿制的巨型「神机」。真·鬼械神以魔导书为核，性能随术者魔力与书页完整度暴涨；仿制机（戴蒙贝恩）基础参数极低，却有「不耗魔力即可现界」的神性特例，靠脚部时空扭曲盾与禁忌必杀翻盘。
4. **神话位阶（克苏鲁体系）**：深潜者/眷属 → 旧日支配者（Great Old Ones：克苏鲁、达贡、克图格亚、伊塔库亚等）→ 外神（Outer Gods：约格-索托斯、奈亚拉托提普、阿撒托斯等）→ 与之对抗的**旧神（Elder Gods）**。召唤、封印、门扉与「星辰归位」仪式可改写城市乃至宇宙因果。

**成长与死亡规则**：凡人死即死；魔术师可被魔导书复活/置换页片；鬼械神可被打坏再修；**Master Therion 因父系约格-索托斯血统可「重生」**；奈亚拉托提普被击败后仍可换方案归来；真结局线九郎与阿尔可升格为**旧神**，跳出单次时间线。世界可被「修正因果」重置（飞翔作背景：黑衣结社阴谋被击破后重生的阿卡姆）。

**力量表现（由低到高，便于映射）**：
- **普通人 / 治安警察 / 修女教会线**：手枪、拳脚、情报；破坏力≈街区斗殴。
- **入门魔术师 / Magius Style 初期 / 破坏机器人量产型**：可崩塌建筑、拦截载具；破坏力≈街区～城区局部。
- **逆十字干部（中低）+ 专属鬼械神 / 戴蒙贝恩常规武装**：拆毁街区、对抗巨型机；破坏力≈城区。
- **戴蒙贝恩·亚特兰蒂斯踢 / 巴尔扎伊偃月刀 / 克图格亚&伊塔库亚神威弹**：可击杀鬼械神级；破坏力≈城市级。
- **雷姆利亚冲击（Lemuria Impact）**：右掌希拉尼普拉系统直连「狮子的心脏」，从另一宇宙抽取无限热量近距打入——对鬼械神级「必灭」；破坏力≈城市～山岳/战役级。
- **未完全体达贡 / 被束缚的克苏鲁 / 幻影心脏母舰 C 计划**：伴随疯狂波与舰队级威胁；破坏力≈城市覆灭～海域战役。
- **Liber Legis 全力 / 闪耀的梯形宝石（Shining Trapezohedron）互轰 / 约格-索托斯之门内时空战**：跨越时间与平行宇宙；破坏力≈星球～宇宙局部规则。
- **奈亚拉托提普本体棋局 / 阿撒托斯（被封于梯形宝石内）/ 旧神化九郎&阿尔**：可玩弄或重写现实；**超阶**。

**乐园阶位映射**（对照《阶位战力图鉴》破坏力，宁低勿高）：普通人～治安/修女线≈**一阶**；入门魔导·Magius 初·量产破坏机器人≈**一～二阶**；逆十字中低干部+专属机 / 戴蒙贝恩常规战≈**二～三阶**；戴蒙贝恩必杀级（亚特兰蒂斯/雷姆利亚）·Liber Legis 常规≈**三～四阶**；C 计划·克苏鲁束缚战·海域决战≈**四～五阶**；约格-索托斯门内时空对决·闪耀梯形宝石≈**五～六阶**；世界顶点（**奈亚拉托提普 / 旧神化大十字九郎&阿尔·阿ジフ / 阿撒托斯封印体**）= **超阶**：存在·情报优先 / 条件性胜利，严禁「被封印所以战力为零」。

**【地理 · 舞台】**
- **阿卡姆市（Arkham City）**：主舞台。科学与魔导并立的大都会；**霸道财阀**总部与地下戴蒙贝恩机库在此；**黑衣结社**暗中盘踞；**米斯卡托尼克大学（Miskatonic University）**隐秘学科培养魔术师与考古学者；**大十字侦探事务所**位于贫民侧；**莱卡修女**的教会与孤儿院是九郎的「食堂」与情感锚点。
- **因斯茅斯（Innsmouth）与近海孤岛**：深潜者聚居与达贡复活仪式场；度假线突变为神话战。
- **敦威治（Dunwich）等地名**：洛夫克拉夫特乡作背景提及，作支线/氛围锚。
- **勒莱耶（R'lyeh）坐标海域（约 47°9′S 126°43′W）**：C 计划后幻影心脏母舰转移点；多国联合舰队与戴蒙贝恩突击的终盘战场之一。
- **亚利桑那沙漠**：时间线扭曲后戴蒙贝恩坠落点；与**霸道钢造**身份谜题相关。
- **约格-索托斯之门 / 外宇宙**：终盘 Liber Legis 与戴蒙贝恩跨时空战场；可通向「所有可能的戴蒙贝恩」与旧神领域。
- **修正后的阿卡姆（机神飞翔）**：因果修正后的「新世界」；血之怪事件、二斗流九朔与 Another Blood 登场舞台。

**【世界剧情线】**

**① 开局：委托与坠落的魔导书（阿卡姆市·侦探线）**
地球上空，真·鬼械神**Aeon（アイオーン）**因缺少「主人」被不明机体击坠。阿卡姆市，三流侦探**大十字九郎**常年吃不饱，依赖修女**莱卡**施粥。**霸道财阀**少当主**霸道瑠璃**与管家**温菲尔德（Winfield）**登门：以高额报酬委托九郎寻找「真正的魔导书」——因其曾就读米斯卡托尼克、懂隐秘学却因某次事件退学、忌惮魔术。九郎为生存接下。寻书过程中他进入神秘书店，店主**Nya（妮娅）**拒绝出售现有强力书，却预言「有更强的一本注定属于你」。随后，少女形态的**阿尔·阿ジフ**从天而降砸到九郎身上——她是**死灵之书（Necronomicon）**原典「Kitab al-Azif」的精灵，正被黑衣结社追杀。

**② 契约与戴蒙贝恩启动（黑衣结社初战）**
黑衣结社科学家**Doctor West（ドクター・ウェスト）**率破坏机器人追击。九郎被迫与阿尔契约，获得 Magius Style 与魔术；在蒙面英雄**Metatron（メタトロン）**等援助下逃入地下，发现霸道财阀仿制鬼械神**Demonbane（戴蒙贝恩）**。阿尔强行启动，九郎在瑠璃抗议中首次驾驶，击退破坏机器人。瑠璃最初愤怒「外姓人抢走祖父遗产」，但战力现实迫使她正式雇佣九郎为驾驶员；阿尔要求立刻开课教魔术。大导师**Master Therion**亲自试探：九郎被压倒性击败，却因戴蒙贝恩迫使对方亮出魔导书**Pnakotic Manuscripts（纳科特写本）**精灵**Etheldreda（艾塞尔德蕾达）**。Therion 声称九郎是唯一能解闷的对手，饶他一命后消失。

**③ 页片回收与逆十字压迫（阿卡姆巷战·成长）**
阿尔在与 Therion 交手中散失大量书页，力量不全。九郎一边接受魔术特训，一边在市内回收页片——页片会自行活化威胁市民，须物理制服后由阿尔重组。黑衣结社方面，**Doctor West**与自造人造人**Elsa（エルザ）**多次抢夺页片，甚至复制出仿戴蒙贝恩**Demonpain（デモンペイン）**；逆十字干部**Titus（ティトゥス）**与**Tiberius（ティベリウス）**等袭击霸道总部。九郎在战斗与「看不下去」的正义感中从「被卷进来」转为「必须打倒黑衣结社」。平行线中，Metatron 持续对结社独战；**Sandalphon**等设定在动画中戏份压缩，游戏本篇更完整。

**④ 因斯茅斯假日与达贡（深海神话战）**
瑠璃安排九郎、阿尔、支援班与莱卡赴因斯茅斯「海滩休假」。离岛上感知到邪恶：深潜者袭击导致队伍失散，众人各自登岛，发现因斯茅斯居民本为深潜者，正举行复活其神**Dagon（达贡）**的仪式；逆十字**Vespasianus（ウェスパシアヌス）**与魔导书**R'lyeh Text（勒莱耶文本）**在场监督。仪式失败却使达贡以**未完全·兽化形态**苏醒。戴蒙贝恩借旧日支配者**Cthugha（克图格亚）**的原始火力击退达贡，九郎几乎力竭。此后他取得阿尔关于**Ithaqua（伊塔库亚）**的页片，并从黑衣结社所称的「暴君」**Nero / Ennea（ネロ / エンネア）**处获得双枪导管——**Cthugha 与 Ithaqua 手枪**，作为神威弹的人体侧武器。

**⑤ C 计划：幻影心脏母舰与克苏鲁（城市崩坏）**
黑衣结社真正大业是 **C Project（C 计划）**：需要死灵之书与「暴君」级存在作触媒，但 Therion 宣称可用纳科特写本与自己替代，干部们仍不信任并密谋叛乱。星辰归位时，秘密本营**幻影心脏母舰（Illusionary Heart Mother）**升空；以勒莱耶文本召唤并束缚**Cthulhu（克苏鲁）**，配合破坏机器人军团与疯狂波蹂躏阿卡姆。高潮中逆十字背叛 Therion，看似将其与 Etheldreda 一并摧毁；**Doctor West**因厌恶干部视部下为弃子、滥用己之科技，倒戈协助霸道侧。戴蒙贝恩击退机器人潮，母舰却带着被紧急措施控制的克苏鲁传送撤离。

**⑥ 勒莱耶海域总攻与约格-索托斯之门（终盘上）**
霸道财阀凭国际关系定位母舰于勒莱耶坐标，动员多国联合舰队海战牵制；戴蒙贝恩突入要塞，逐一击破残余逆十字。**Master Therion 重生**——其父为外神**Yog-Sothoth（约格-索托斯）**，C 计划真意是以克苏鲁为祭品召唤「门与钥匙」，打开外神入侵物质宇宙的门户。Therion 邀九郎「追进门内」；九郎告别瑠璃、莱卡等人后应战。戴蒙贝恩与 **Liber Legis（リベル・レギス）** 在时空中缠斗，终以双方最强武装**闪耀的梯形宝石（Shining Trapezohedron）**对撞。Liber Legis 败北。

**⑦ 幕后：Nya = 奈亚拉托提普（终盘下·本篇收束）**
阿尔在决战中解锁被封的全知，指认一切的编排者：**Nya = 外神 Nyarlathotep（奈亚拉托提普）**。九郎与阿尔击败其当前方案，阻止现实崩坏。奈亚承认「这次」失败，却声明随时可换一局再来；戴蒙贝恩滞留时空裂隙。各路线结局分化：**阿尔真结局**中九郎与阿尔成为**最古亦最新的善神——旧神（Elder Gods）**；**瑠璃线**等有九郎重伤、瑠璃临时契约阿尔、温菲尔德对决 Titus 等分支；瑠璃线更揭示「坠落亚利桑那的戴蒙贝恩驾驶员=未来的九郎→被当作**霸道钢造**」的时间悖论（真钢造死于坠落冲击）。动画版结局折中：取阿尔好结局元素，由旧神化的九郎&阿尔把漂流的阿尔送回阿卡姆。

**⑧ 机神飞翔：修正世界与血之继承（后续）**
因果修正后的新阿卡姆，九郎以「魔导侦探」身份受雇霸道，调查连环血之怪事件。红裙少女**Another Blood（アナザー・ブラッド）**与异名「二斗流 / 二斗枪」的少年**大十字九朔（クザク）**出现；血色戴蒙贝恩与「本不该存在」的逆十字残影回归。九朔实为**旧神化九郎与阿尔之子**，被送往别世界霸道财阀抚养，驾驶**Demonbane Twosword**；Another Blood 是奈亚抽离九朔之「恶」并赋形的**死灵之书血液语言版**，可召**Demonbane Blood**。邪神猎人**Laban Shrewsbury（ラバン・シュリュズベリイ）**与魔导书精灵**Hazuki（葉月 / 塞拉耶诺断章写本）**、前代契约者**Azrad（アズラッド）**与真·鬼械神 Aeon 加入战线。终局仍指向奈亚的新棋与「所有可能的戴蒙贝恩」级手段（含隐藏结局 Remulia Impact 000 式 Ain Soph Aur）。

**【主要人物】**

> 阶位↔境界：一阶≈普通人·治安·修女教会与穷侦探日常；二阶≈入门魔导 / Magius Style 初期 / 量产破坏机器人；三阶≈逆十字中低干部+专属鬼械神 / 戴蒙贝恩常规武装战；四阶≈雷姆利亚冲击级·Liber Legis 常规·城市级魔导决战；五阶≈C 计划·克苏鲁束缚战·勒莱耶海域战役；六阶≈约格-索托斯门内时空战·闪耀梯形宝石对决。世界顶点（**奈亚拉托提普 / 旧神化大十字九郎&阿尔·阿ジフ / 阿撒托斯封印体**）为超阶存在，低阶切入一律「规避顶点·情报优先 / 条件性胜利」。

**一阶（穷侦探与寻书委托 · 阿卡姆巷陌线）**

切入身份/时点：契约者以「新来阿卡姆的私家侦探助手 / 教会临时工 / 米斯卡托尼克旁听生」切入，锚定**大十字九郎**尚未与阿尔契约、刚被**霸道瑠璃**委托寻书前后。

初始事件：你在教会施粥处遇见总欠饭钱的侦探九郎；当日下午霸道本邸的黑色轿车停在侦探事务所楼下——瑠璃与**温菲尔德**给出「寻找真正魔导书」的委托与预付金。你可选择：跟着九郎跑旧书店街、帮莱卡照看孤儿、或偷偷跟踪黑衣结社的低级眼线。

开场白建议：「阿卡姆的雨总是带着铁锈味。你夹在『三流侦探』大十字九郎和修女莱卡的施粥队伍之间，听见他咕哝『再没委托就要去啃鞋底』——下一秒，霸道财阀的车灯就刺破了巷子。有人出大价钱买一本会吃人的书。」

关键NPC立场：**大十字九郎**——缺钱但护短，可结盟带路，怕你卷入魔术；**莱卡**——提供食宿与情报，劝你别碰黑市魔导书；**霸道瑠璃**——高傲雇主，视外人工具，完成委托才给正眼；**温菲尔德**——礼貌测你动机，护主优先；**Nya**——书店里意味深长的旁观，话里有钩；**Doctor West** 的底层爪牙——抢书、开火、可被吓退。

主线钩子/支线：寻书委托的蝴蝶效应（是否目击阿尔坠落）；支线=教会孤儿乔治/科林/艾莉森的失踪传闻、旧书店假货鉴定、治安警察 Ness/Stone 的「破坏机器人扰民」报案。

危险度/规避：中低——街头枪战与低级魔术；须规避逆十字干部、真·鬼械神、任何「旧日支配者」名号的仪式现场。

任务方向/奖励：站稳脚跟、摸清阿卡姆双轨（科学/魔导）；奖励取向=入门护符、假面情报网、与九郎/莱卡的信任、低阶魔导书残页（非原典）。

**二阶（契约之夜与戴蒙贝恩初啼 · 巷战启动线）**

切入身份/时点：以「目击契约现场的局外人 / 霸道财阀临时雇员 / 黑衣结社叛逃杂兵」切入，锚定阿尔坠落、九郎契约、戴蒙贝恩首次出击前后。

开场白建议：「天空裂开时，先掉下来的是一个骂骂咧咧的少女，然后是钻头与火箭弹。大十字九郎把你推进垃圾箱盖后面，自己却冲进了弹幕——他说『不管会后悔什么，至少今晚别死人』。地下，有什么巨大的东西睁开了眼。」

危险度/规避：中——破坏机器人与 Magius 级交火可夷平街区；须规避 Master Therion 亲自现身、雷姆利亚冲击解冻现场。

任务方向/奖励：成为「戴蒙贝恩周边」的人脉；奖励取向=初级魔术教程残卷、西式护具、霸道临时通行证、页片容器。

**三阶（页片回收与逆十字压迫 · 都市魔导线）**

切入身份/时点：以「魔导书猎人 / 霸道财阀战斗编制顾问 / 独立魔术师」切入，锚定九郎回收阿尔页片、Titus 与 Tiberius 袭击霸道本邸阶段。

初始事件：市中心博物馆夜间，一枚阿尔的页片具现为吃人的纸化生物；同时逆十字干部对霸道本邸发动斩首。你被通讯呼叫——是去博物馆压制页片，还是去本邸拖延 Titus 的拳锋。

开场白建议：「纸会呼吸。你在展柜玻璃上看到自己的倒影裂成无数张嘴，而耳机里温菲尔德的声音冷静得可怕：『Titus 已破一层。戴蒙贝恩尚未归港。』阿卡姆的夜，开始用两种方式杀人。」

关键NPC立场：**Titus**——好战，认强者，对温菲尔德有对位执念，可能接受「单挑」；**Tiberius**——残忍嗜杀，无荣誉可言；**艾塞尔德蕾达**——极少正面，出现即代表 Therion 注视；**チアキ**——技术口，可给你维修/侦测任务；**大十字九郎**——已决心开战，愿收「靠谱的帮手」但不许你送死。

主线钩子/支线：页片回收进度（直接影响阿尔战力）；本邸攻防是否失守；支线=Elsa 的自我认知危机、假魔导书黑市、Metatron 身份线索。

危险度/规避：高——干部级魔术与鬼械神余波；须规避 Liber Legis 召唤、完整态阿尔敌对（若你偷页片）。

任务方向/奖励：成为阿卡姆魔导战的「第三势力」或霸道编外；奖励取向=中阶咒具、页片密封匣、干部级情报、Magius 入门触媒。

**四阶（因斯茅斯与雷姆利亚级 · 神话踏入线）**

切入身份/时点：以「随行保镖 / 海洋学者掩护的魔术师 / 联合技术观察员」切入，锚定因斯茅斯休假突变、达贡未完全体、九郎掌握克图格亚火力前后。

初始事件：游艇靠近离岛时深潜者翻船；你与队伍失散，在湿漉漉的祭坛上听到「父神将醒」。Vespasianus 的勒莱耶文本仪式已进行到第三段——阻止祭品、抢走文本残页、或给戴蒙贝恩争取 90 秒蓄力窗口。

开场白建议：「海水是温的，像血被稀释过。因斯茅斯的渔民微笑着把你往岛上让，牙齿却尖得不对。无线电里只剩下霸道瑠璃咬牙的一句：『解冻纳阿卡尔……九郎，别死。』」

关键NPC立场：**Vespasianus**——仪式优先，视你为可消耗祭品；**深潜者祭司**——可被 negotiable 的恐惧与信仰动摇；**霸道瑠璃**——远程授权必杀，重视「活着回来的人」；**阿尔**——需要火力页片配合，对畏惧神话者嗤之以鼻却保护同伴；**Nero/Ennea**——若时点稍后，可能以「暴君」身份给出双枪线索，立场阴晴不定。

主线钩子/支线：达贡战的介入方式（硬刚 / 破仪式 / 引开眷属）；支线=救出莱卡与孤儿同行者、夺取勒莱耶文本抄本、记录克图格亚显灵的代价。

危险度/规避：高～极高——未完全体旧日支配者余波可屠岛；须规避完整召唤成功、Master Therion 下场、对雷姆利亚冲击的「友军伤害」。

任务方向/奖励：踏入神话级战场的生存者资质；奖励取向=深海护符、克图格亚/伊塔库亚相关触媒残片、海域地图、干部级魔导书抄本。

**五阶（C 计划与勒莱耶总攻 · 城市覆灭线）**

切入身份/时点：以「多国联合舰队联络官 / 霸道战时参谋 / 倒戈结社技术人员」切入，锚定幻影心脏母舰升空、克苏鲁被束缚、Therion「被杀」与 West 倒戈、定位勒莱耶坐标发起总攻前后。

初始事件：阿卡姆上空出现第二轮疯狂波，街道上的人开始朝同一方向跪拜。你的终端收到两条互相矛盾的命令：黑衣结社要求你「维护母舰内侧回路」，霸道侧要求你「把 West 安全带出」。选择将决定你在总攻中的阵营编号。

开场白建议：「星辰『正确』的那一夜，城市像被无形的手捏碎。破坏机器人的队列从地平线铺到市政厅，而幻影心脏母舰在云层里跳了一支没有观众的舞。有人在广播里笑：『C 计划，启动。』你听得出那不是 Master Therion 的声音——干部们提前翻盘了。」

关键NPC立场：**Doctor West**——倒戈后仍中二，但技术可靠，要你「承认他是天才」；**Elsa**——跟随 West，火力支援换九郎关注；**逆十字残党**——控制克苏鲁的紧急措施，杀叛逃者；**霸道瑠璃**——统筹舰队与戴蒙贝恩突入窗口；**Master Therion（重生前）**——若你目击「处刑」现场，会获得「他未死」的关键情报。

主线钩子/支线：母舰传送前能否植入定位；突入勒莱耶时的破防任务；支线=疏散教会区难民、抢修戴蒙贝恩损伤、阻止克苏鲁完全脱缚。

危险度/规避：极高——城市级疯狂与旧日支配者；须规避约格-索托斯之门内侧（六阶域）、奈亚本体现身。

任务方向/奖励：参与「人类侧总攻」的资格；奖励取向=战舰级魔导炮授权残片、母舰内部地图、干部鬼械神残骸样本、West 系列特殊武装。

**六阶（约格-索托斯之门与奈亚棋局 · 时空终局线）**

切入身份/时点：以「自愿追入门内的魔导士 / 旧神侧观察者碎片 / 平行世界的戴蒙贝恩关联者」切入，锚定 Therion 重生揭开 C 计划真意、戴蒙贝恩与 Liber Legis 时空对决、阿尔全知指认奈亚前后。

初始事件：门在母舰核心打开，像一把钥匙插进宇宙的锁孔。Therion 回头对九郎（以及你）说：「无聊的棋盘该翻面了。」你要么握住戴蒙贝恩外挂的时空锚定索一起跃入，要么留在物质界启动「门关闭协议」——两者都是终局必需。

开场白建议：「没有上下前后。你看见阿卡姆在燃烧，也看见它从未存在；Liber Legis 的翼在所有世纪同时展开，戴蒙贝恩的右掌亮起不该属于一个宇宙的热。阿尔的声音第一次发颤：『那书店的女人……是奈亚拉托提普。』」

关键NPC立场：**Master Therion**——要劲敌，不要奴隶，可能与你「限时同盟」只为打得尽兴；**Etheldreda**——忠诚到可替主人挡梯形宝石余波；**阿尔·阿ジフ**——解锁全知后成为战术核，需人保护其「翻页」间隙；**大十字九郎**——已告别尘世牵挂，只求砍断门；**Nya/奈亚拉托提普**——承认失败也要埋下一局，**超阶巅峰：存在·情报优先/条件性胜利**，严禁硬撼「真身全权」或宣称其被永久消灭；飞翔时点另有**大十字九朔 / Another Blood** 可作为奈亚新棋的切入接口。

主线钩子/支线：梯形宝石对撞的条件胜（封印而非打碎）；阻止外神涌入的门闸；支线=取回滞留时空的戴蒙贝恩、保全瑠璃线时间悖论的「钢造」闭环、飞翔线阻止血之死灵之书吞没九朔。

危险度/规避：贴近顶点——门内=六阶主场；奈亚/旧神/阿撒托斯封印体=超阶，只允许布局、情报、代价交换与条件性胜利。

任务方向/奖励： closure 级任务（关上门/保住一条世界线）；奖励取向=旧神印记碎片、梯形宝石「箱」的认知残片、平行戴蒙贝恩共鸣许可、与九郎·阿尔的终局羁绊（非无脑战力复制）。
