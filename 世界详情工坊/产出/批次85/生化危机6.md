# 生化危机6
<!--meta lib=主库 tiers=一、二、三、四、五、六-->

## 剧情

**【作品来源】**
《生化危机6》（日版《バイオハザード6》，英文 *Resident Evil 6*）由卡普空开发并发行，制作人小林裕幸，总监佐佐木英一郎，编剧菅正太郎，引擎 MT Framework。2012年10月2日北美/欧洲 PS3、Xbox 360 首发，日本10月4日，2013年3月 Windows，2016年3月 PS4/Xbox One 全集 DLC 重发，2019年10月 Switch。本作是系列动作化顶点之一：四条交织战役（里昂/克里斯/杰克/艾达），双人合作、边跑边射、技能点强化，主敌为 **C病毒（C-Virus / Chrysalid Virus）** 造成的丧尸与智能突变体 **吉瓦弗（J'avo）**。文风：全球生化恐袭动作大片、政治阴谋、战友牺牲与身份谎言并存。**以 2012 游戏主线正典为准**。关联：前作《生化危机5》威斯卡之死与 BSAA 体系；里昂浣熊市创伤与 DSO 身份；雪莉·柏金 G 病毒残留；后续系列中 C 病毒与「家族」余波仍被引用。

**【世界定位】**
2012 年末至 2013 年夏，东欧虚构国 **伊多尼亚共和国（Edonia）**、美国 **高橡树市（Tall Oaks）** 与中国虚构港城 **兰祥（Lanshiang）** 先后爆发 C 病毒恐袭。四组视点并行：**里昂·S·肯尼迪**与 **海伦娜·哈珀**追查总统被杀真相；**克里斯·雷德菲尔**与 **皮尔斯·尼凡斯**重建 BSAA 小队并追杀「假艾达」；**杰克·穆勒**与 **雪莉·柏金**为抗体血液逃亡并遭 **捕缚者（Ustanak）** 追杀；**艾达·王**拆穿 **德瑞克·C·西蒙斯**与 **卡拉·拉达梅斯** 的双线阴谋。一句话：全球 C 病毒链式恐袭——四线交织，截导弹、抢抗体、杀伪神。卡片锚点：**捕缚者（Ustanak）** 以机械臂与巨力在废墟/雪山/矿道多次追杀杰克与雪莉，力量足以掀翻坦克与装甲车。

**【世界观 · 力量体系】**
本世界战力不来自灵气，而来自 **枪械技战术 + 组织权限 + C 病毒分级生物兵器**。C 病毒可由气溶胶、注射、导弹弹头等途径传播：普通暴露多产生丧尸化；经特殊处理的注射可产生保留智能、会用枪械与载具、伤口定向突变的 **吉瓦弗**；极端个体可入茧（chrysalis）后破茧成完全脱离人形的高阶 B.O.W.。热武器、爆炸物、载具与技能点改装是人类侧主战手段。死亡规则：宿主被彻底破坏即死；部分高阶突变体可反复变形（如西蒙斯）；皮尔斯自行注射 C 病毒可获短时强化，但不可逆。抗体关键：**杰克·穆勒**（阿尔伯特·威斯卡之子）血液可研发抗 C 疫苗；**雪莉·柏金**体内 G 病毒残留赋予超常愈合。

力量/威胁层级（对照《阶位战力图鉴》宁低勿高）：

1. **未感染平民 / 外围目击者 / 恐慌疏散人群**  
   徒手或临时武器，无组织。乐园对照≈一阶下沿。

2. **普通 C 病毒丧尸 / 初期吉瓦弗步兵**  
   扑咬、围攻；吉瓦弗会开枪、投弹、协同包抄。单兵≈训练暴徒；群压可吞没街区巡逻。乐园对照≈一阶～二阶初。

3. **武装吉瓦弗小队 / 载具单位 / 局部突变肢（盾臂、刃肢、枪架突变）**  
   巷战墙推、屋顶火力点、摩托车与吉普追击。乐园对照≈二阶。

4. **中型 B.O.W. 与仪式/教堂级威胁**  
   **黛博拉·哈珀**茧化后的触手高机动体、**莱波蒂萨（Lepotitsa）**类释放感染雾的巨体、高橡树大教堂地下墓穴实验体：可摧毁石室、走廊级杀伤。乐园对照≈三阶。

5. **捕缚者（Ustanak）与设施级追猎体**  
   机械臂可抓取载具、掀翻坦克级装甲、在雪山/废墟/矿道持续追踪目标；伊多尼亚与兰祥多处 Boss 级遭遇。乐园对照≈四阶。卡片锚点落此档。

6. **西蒙斯多形态突变 / 航母与导弹阵列 / 精锐特务对决**  
   **德瑞克·C·西蒙斯**被注入强化 C 病毒后可反复改变外貌与攻击形态，覆盖码头—城区战场；新保护伞航母上的 C 病毒巡航导弹可整城投放。乐园对照≈五阶。

7. **顶点：哈奥斯（Haos）与城市毁灭级布局**  
   水下设施巨体 **哈奥斯** 配合兰祥全城 C 病毒爆发与导弹网，形成地区级灾难。本世界 **顶点≈六阶**（镇国档下沿：覆盖港城与跨洋恐袭网络，凡俗军队难在爆发窗口内正面清剿）。**无七阶及以上**。

乐园阶位映射：未感染/外围≈一阶；丧尸群与武装吉瓦弗≈二阶；黛博拉、Lepotitsa、教堂实验体≈三阶；Ustanak 与同级追猎 B.O.W.≈四阶；西蒙斯多形态、航母导弹、精锐对决≈五阶；哈奥斯+兰祥/高橡树毁灭级投放≈六阶。世界顶点=**哈奥斯 / 新保护伞城市毁灭布局（及西蒙斯终形态作为政治—生物复合顶点）**，低阶切入一律「规避顶点·情报优先」，条件性胜利（截导弹、护送抗体血液、引爆水下设施、击破西蒙斯当前形态后撤离）优先于硬刚本体。

**【地理 · 舞台】**
- **伊多尼亚共和国（Edonia）**：东欧战乱虚构国；雇佣兵战场、雪原、矿坑、废弃城镇。2012-12-24 首波 C 病毒与吉瓦弗叛军战场；Ustanak 首次大规模追猎杰克与雪莉。
- **高橡树市（Tall Oaks）**：美国中部虚构城市；**长青藤大学（Ivy University）** 为总统演讲会场；市区丧尸潮；**高橡树大教堂**地下墓穴关押黛博拉并藏「家族」实验痕迹。市在里昂与海伦娜逃出后被 **核弹摧毁** 以灭证。
- **兰祥（Lanshiang）**：中国南方虚构港城（多城区、市场、天桥、码头、地下铁）；新保护伞与「家族」终局主舞台；C 病毒导弹空爆后全城感染。
- **新保护伞航空母舰 / 海上平台**：卡拉据点；巡航导弹装填 C 病毒弹头；克里斯与皮尔斯截击点。
- **海上石油平台与水下设施**：关押杰克、雪莉；饲养哈奥斯；皮尔斯牺牲引爆点。
- **外围情报网**：DSO 通过 **英格丽德·汉尼根** 远程支援里昂；BSAA 北美分部调度克里斯小队；「家族（The Family）」以国家安全顾问西蒙斯为公开面，暗中维持「可控秩序」。

写场景时强调：圣诞夜雪原枪火、大学礼堂血雾、教堂钟声与茧壳、兰祥霓虹下的吉瓦弗摩托、航母甲板风雨、水下蓝光中的哈奥斯。

**【世界剧情线】**

○ **前史：浣熊市之后到 C 病毒登场**  
1998 年浣熊市事件后，里昂进入政府特工体系，后属 **安全行动处（DSO）**；克里斯加入并领导 **BSAA** 一线；雪莉·柏金因体内 G 胚胎残留被美国长期监控，后以特工身份换取有限自由。2009 年前后非洲线中阿尔伯特·威斯卡败死，但其基因与技术遗产未绝——私生子 **杰克·穆勒** 在伊多尼亚以雇佣兵维生，血液中潜藏对多种病毒的抗性。国家安全顾问 **德瑞克·C·西蒙斯** 领导秘密组织「家族」，坚信「以可控恐怖维持秩序」；其研究者 **卡拉·拉达梅斯** 被强迫接受基因手术，外貌重塑为 **艾达·王** 的复制体，后建立 **新保护伞（Neo-Umbrella）**，表面听命西蒙斯，暗中计划以 C 病毒报复并毁灭世界。

○ **2012-12-24 · 伊多尼亚：抗体与屠杀（一～四阶舞台）**  
伊多尼亚爆发生物恐袭。雇佣兵杰克在战火中突围，被 DSO 特工雪莉接触：政府需要他的血液研制抗 C 疫苗，出价可达五千万美元量级。二人合作逃亡时遭 **捕缚者 Ustanak** 反复追杀——机械臂可掀翻装甲、在废墟中拆墙抓人，是本世界「被盯上就无法靠藏」的标尺。同步，BSAA 北美 A 队由克里斯与副队长皮尔斯率领清剿吉瓦弗；自称「艾达·王」的新保护伞特工（实为卡拉）发动突袭，以注射装置令芬·麦考利等队员感染突变，克里斯目睹战友变为怪物，仅他与皮尔斯生还。克里斯陷入创伤性失忆与酗酒消沉。雪莉与杰克的 BSAA 撤离被破坏，坠落后在雪山被「艾达」俘获，关押约六个月。

○ **2013-06-29 · 高橡树：总统之死与核灭证（二～三阶舞台）**  
美国总统 **亚当·本福德** 决定公开 1998 年浣熊市真相及联邦与保护伞的交易，认为透明有助遏制生化恐袭。演讲会场长青藤大学遭 C 病毒袭击；总统感染丧尸化，DSO 高级特工里昂被迫开枪击杀旧友总统。特勤局探员海伦娜·哈珀与里昂突围，她承认「这一切与我有关」。二人进入高橡树大教堂：海伦娜之妹 **黛博拉** 被西蒙斯扣为人质并注射 C 病毒，茧化后突变，最终死于交战/海伦娜之手。海伦娜吐露真相——西蒙斯以黛博拉胁迫她协助制造总统遇刺窗口，西蒙斯并与新保护伞有勾连。二人逃出教堂后，高橡树市被核弹摧毁灭证。里昂与海伦娜伪造死亡记录，前往兰祥追捕西蒙斯。真·艾达亦在事件中现身接触里昂，确认西蒙斯的双面身份。

○ **兰祥上半：归队、对质与身份交错（三～五阶）**  
皮尔斯在酒吧寻回消沉的克里斯，说服其归队 BSAA，编入新小队进入已被围困的兰祥。克里斯记忆恢复后执着追杀「艾达」（卡拉），行动鲁莽导致新队员伤亡。里昂与海伦娜、杰克与雪莉、克里斯与皮尔斯的路线在兰祥市区交错：市场巷战、天桥追击、地下铁与码头火力点。雪莉曾将杰克的医疗数据暗中交给里昂备份。四方逐渐拼出同一图景：西蒙斯为阻止总统公开浣熊市黑幕而发动恐袭；卡拉以新保护伞推进更大规模的 C 病毒投放。

○ **航母与导弹：截断城市毁灭（五阶）**  
克里斯与皮尔斯追踪卡拉至新保护伞航空母舰，摧毁装填 C 病毒的巡航导弹。此节点是「条件性胜利」范本——目标不是击杀所有突变体，而是阻止弹头升空。卡拉与西蒙斯互相算计：西蒙斯要可控秩序，卡拉要毁灭式报复。

○ **西蒙斯终战与二次俘获（五阶）**  
里昂、海伦娜与杰克、雪莉对质西蒙斯。西蒙斯承认刺杀总统以「维护国家安全」，并在吉瓦弗/强化 C 病毒作用下反复突变，外貌与攻击形态多变，仍保留强烈意志。里昂与海伦娜击破其多个形态；杰克与雪莉再度被俘，押往海上平台。兰祥上空 C 病毒引爆，全城爆发。里昂将杰克的真实身份（威斯卡之子）告知克里斯，委托 BSAA 执行营救。

○ **石油平台 · 哈奥斯 · 皮尔斯之死（六阶顶点）**  
克里斯与皮尔斯潜入石油平台下水下设施，救出杰克与雪莉，遭遇巨体 **哈奥斯**。皮尔斯重伤后自行注射 C 病毒获得突变臂膀与火力，与克里斯合力重创哈奥斯，但知其不可逆，遂将克里斯推入逃生舱，自己留下引爆/摧毁设施，战死。杰克与雪莉在升井/火箭推进式升降中与 Ustanak 最终对决并击杀之，浮出水面。抗体血液线在「人还活着」的意义上暂时保住。

○ **艾达线收束：卡拉真相与实验室焚毁**  
假艾达真身为 **卡拉·拉达梅斯**：因爱慕西蒙斯反被其利用，接受手术成为艾达复制体后心生恨意。她在濒死时自我注射强化 C 病毒，膨胀为快速增殖的 B.O.W.，最终被真·艾达杀死。艾达协助里昂与海伦娜彻底终结西蒙斯残存形态，并摧毁制造卡拉的实验室，随后接下新任务离开。里昂与海伦娜洗清嫌疑归队；克里斯克服罪疚继续指挥 BSAA 新小队；雪莉续任 DSO；杰克在 BSAA 掩护身份下于发展中地区继续对抗生化恐袭。

○ **支线与可切入模式（非改正史）**  
- **佣兵模式 / Agent Hunt**：战力沙盘与「扮演敌人入侵他人战局」，不改主线因果。  
- **Ada 战役 Agent 搭档**：非正史搭档位，可作契约者高阶扮演接口。  
- **技能点经济**：击杀掉落 SP，兑换弹药特化、减伤、道具掉率等，对应「战区成长」而非神器。

○ **结局状态（写给 AI 的世界快照）**  
亚当·本福德死；高橡树被核灭；黛博拉、芬·麦考利、皮尔斯、卡拉、西蒙斯、Ustanak、哈奥斯等主要敌对/牺牲节点按原作收束；兰祥遭重创但导弹主波被截；抗 C 疫苗路线因杰克生还而仍可能；新保护伞核心覆灭，「家族」公开面崩溃但残余理念可能潜伏。低阶切片可停在伊多尼亚圣诞夜或高橡树演讲前；高阶切片面对航母导弹与水下哈奥斯。

○ **四战役叙事功能（AI 调度用）**  
里昂线=哥特教堂与政治真相（生存恐怖残留）；克里斯线=战场恐怖与队长罪疚；杰克线=被追杀的恐惧与血缘诅咒；艾达线=间谍解密与双艾达镜像。写正文时应让契约者先锚定「当前跟的是哪条线」，再允许线与线在兰祥交汇，避免无故同屏全员。

○ **C 病毒传播与变异逻辑**  
气溶胶/导弹→大规模丧尸化；定向注射→吉瓦弗智能体；创伤部位决定突变器官（臂盾、刃、枪架、翅等约十余种）；茧化→完全体 B.O.W.。抑制与治疗在本篇几乎不靠「药丸通关」，而靠 **撤离、截源、抗体研发**。雪莉的愈合与杰克的血是战略资源，不是随便可复制的外挂。

○ **Ustanak 生态与卡片锚点细部**  
捕缚者被设计为「活体猎犬+工兵」：机械臂抓取、背负装甲、在雪崩/矿塌后仍追踪。杰克与雪莉在废墟多次靠环境爆炸、狭窄管道与载具甩脱，而非正面对砍。契约者介入 Ustanak 段落时，正确玩法是 **拆追踪链、毁补给茧、护送目标进掩体**，不是六阶以下单挑击杀。

○ **高橡树核打击的政治含义**  
灭证核爆证明「家族」可调动国家级武力抹除城市级证据。里昂伪造死亡不仅是逃生，更是脱离被西蒙斯渗透的指挥链。海伦娜的罪疚是可角色扮演的情感轴：她不是单纯反派胁从，而是被妹妹生命勒索的特勤。

○ **兰祥巷战与中西混搭符号**  
霓虹、湿巷、天桥摩托、市场喇叭、码头集装箱与吉瓦弗的战术小队形成「现代都市战争+生化」混合恐怖。与 RE4 乡间邪教不同，RE6 强调 **全球同步多热点**。

○ **航母截导弹的任务设计哲学**  
成功标准=弹头失效，而非清图。皮尔斯狙击与克里斯突击分工明确。契约者若在此阶段介入，最有价值的是 **弹道数据、甲板油阀、通讯干扰**，不是与卡拉单挑。

○ **皮尔斯牺牲的主题**  
「天才狙击手用病毒换三十秒」——强化即死刑。严禁写成无代价超级赛亚人；他的 C 化是悲剧性条件胜利，留给克里斯的是活着指挥下一次任务的义务。

○ **西蒙斯动机分层**  
表层：国家安全顾问、总统旧友。中层：「家族」秩序神学——宁可屠杀也不可公开浣熊市罪责。深层：对艾达形象的病态执着，导致制造卡拉并反噬。把西蒙斯写成单纯「要毁灭世界」会削弱政治惊悚。

○ **卡拉动机分层**  
被爱、被改造、被抛弃→以新保护伞之名报复全人类。她与真艾达的镜像战是身份恐怖：敌人长着你在乎之人的脸。里昂因此与克里斯在「是否当场击毙假艾达」上发生路线冲突——证人价值 vs 战友仇杀。

○ **与系列宇宙的接口**

**二阶（吉瓦弗巷战·智能感染墙）**

切入身份/时点：以「BSAA 编外支援 / 被困雇佣兵 / 弹药走私跑腿」身份切入伊多尼亚市区全面接敌或高橡树大学外围刚爆发时。

初始事件：吉瓦弗用步枪组成交叉火网，一名突变臂盾的个体把汽车门撕下来当掩体。有人把你认成「第二个特工」。你必须在被推上感染注射点前，选择帮克里斯小队吸引火力，或带着大学疏散名单冲向地铁。

开场白建议：「他们不是乱咬的死尸。他们会喊掩护、会换弹、会把受伤的胳膊当场长成刀。烟幕弹拉响时，你听见的是战术靴，不是拖步。」

关键NPC立场：**克里斯·雷德菲尔**（允许你殿后但禁止乱指挥他的队员）；**皮尔斯·尼凡斯**（可给你狙击窗口坐标）；**里昂·S·肯尼迪**（高橡树线：评估你是否拖累总统撤离）；武装吉瓦弗小队（见人即清）。

主线钩子/支线：钩子=破坏吉瓦弗的电台中继；支线=护送未感染医护到野战帐篷；支线=偷出一箱未开启的 C 注射器；支线=为杰克标出无 Ustanak 痕迹的小巷。

危险度/规避：危险度中；规避多突变体同巷；群战用爆炸物拆火力点。

任务方向/奖励：生存与破局；奖励=冲锋枪扩容、燃烧瓶、街区手绘火力图。

**三阶（教堂与大学·茧与总统）**

切入身份/时点：以「特勤外围安检员 / 大学校工 / 海伦娜的旧同事假身份」切入高橡树长青藤大学演讲前后或大教堂地下墓穴段。

初始事件：总统车队静音驶入；礼堂空调口飘出甜腥。海伦娜眼圈发红地塞给你一把备用钥匙：「地下第三层，别让我妹妹变成他们的数据。」茧壳在墓穴灯光下起伏——黛博拉的倒计时已经开始。

开场白建议：「国歌还没放完，第一排已经有人咳出黑血。你看见特工举枪的手在抖，不是怕死，是怕开枪的对象戴着你投票选出来的脸。」

关键NPC立场：**海伦娜·哈珀**（把你当赎罪计划的一环）；**里昂·S·肯尼迪**（临时同盟，任务优先于情绪）；**黛博拉·哈珀**（茧化后无谈判）；**亚当·本福德**（感染前可对话，感染后必须放弃）；教堂「家族」看守（灭口优先）。

主线钩子/支线：钩子=抢出黛博拉被注射前的监控卡；支线=修改礼堂通风争取两分钟疏散；支线=给汉尼根上传西蒙斯出现在指挥链的照片；支线=在核打击预警前打开地下民防通道。

危险度/规避：危险度高；黛博拉与礼堂丧尸潮为三阶标尺；核打击倒计时开始后禁止恋战。

任务方向/奖励：真相链条与撤离；奖励=抑制用防护面罩、墓穴地图、假死身份空白表格（高风险违禁）。

**四阶（捕缚者·废墟追杀）**

切入身份/时点：以「被俘后逃脱的共同囚犯 / 雪莉的一次接头人 / 矿区爆破技师」切入杰克与雪莉遭 Ustanak 追杀的废墟、雪山或矿道段（卡片锚点主场）。

初始事件：机械臂从墙后插入，把装甲车像罐头一样捏扁。雪莉把样品箱塞进你怀里：「人比血重要，但血也不能碎。」杰克骂着让你跑，自己却回头开火——Ustanak 的脚步让积雪成圈脱落。

开场白建议：「坦克的炮塔歪在沟里，像被巨人玩坏的玩具。那只手不是肉长的，是液压与骨头的合约。你每呼吸一次，追踪标记就亮一格。」

关键NPC立场：**捕缚者（Ustanak）**（锁定优先目标：杰克/雪莉/持有血液数据者）；**杰克·穆勒**（可认你作炮灰或临时盾）；**雪莉·柏金**（愿用情报换你护运）；**卡拉·拉达梅斯（伪艾达）**（远距收网，要活口）；BSAA 残存通讯员（只能给你一发烟雾坐标）。

主线钩子/支线：钩子=破坏 Ustanak 的阶段装甲/拖住其进入载具；支线=用矿坑炸药制造假死信号；支线=复制一份血液数据到里昂线备份；支线=救出另一名被点名的「抗体候选」平民（可能是诱饵）。

危险度/规避：危险度极高；Ustanak 为四阶标尺与卡片锚点，**禁止四阶以下宣称击杀**；优先风筝、环境杀、护送脱离。

任务方向/奖励：保住抗体线；奖励=重型弹药、追踪干扰片、雪莉的半枚徽章（兰祥接头信物）。

**五阶（兰祥—航母·西蒙斯与导弹）**

切入身份/时点：以「港务维修工 / 家族底层文书卧底 / 艾达的一次性外围接头 / BSAA 增援潜水员」切入兰祥四线交汇后、航母截导弹或西蒙斯首次对质前后。

初始事件：广播用两种语言重复「空气安全」；天上却落下红色雨。克里斯要冲向「艾达」，里昂按住他的枪：「她可能是证人。」西蒙斯在屏幕里微笑，皮肤已经开始不属于人类。你手上的平板显示三枚巡航导弹仍在倒计时。

开场白建议：「霓虹在毒雨里溶成血色。顾问的声音从每台公共电视里出来，说这是为了和平。你的任务简报只有一行字：别让弹头亲吻城市。」

关键NPC立场：**德瑞克·C·西蒙斯**（视你为可抹除变量）；**卡拉·拉达梅斯**（可交易也可灭口）；**里昂·S·肯尼迪**（要你守数据链）；**克里斯·雷德菲尔**（要你别挡他对卡拉的枪）；**艾达·王**（可借你钩索路线，换实验室坐标）；**海伦娜·哈珀**（要西蒙斯活着认账或死透）。

主线钩子/支线：钩子=协助甲板使导弹失效；支线=录音西蒙斯承认总统遇刺；支线=阻止克里斯当场击毙卡拉（或相反，帮他完成复仇——会改变证人线）；支线=给杰克打开码头地下铁逃生闸。

危险度/规避：危险度极高；西蒙斯多形态与航母战为五阶标尺；规避与西蒙斯终形态长时间正面对砍。

任务方向/奖励：截断毁灭投放、坐实主谋；奖励=军用狙击改件、导弹黑匣子拷贝、兰祥封锁区临时通行。

**六阶（哈奥斯与终局·条件性胜利）**

切入身份/时点：以「平台维修主管 / 政府收尾先遣 / 艾达撤离第二降落点联络人」切入石油平台水下设施、哈奥斯现身、皮尔斯即将注射 C 病毒或 Ustanak 终战同步窗口。

初始事件：水下蓝光里巨体翻身，逃生舱指示灯只剩两个绿。皮尔斯把注射器按进自己脖子，对克里斯说「队长，走」。杰克在井筒另一侧与机械臂搏命。你必须在三十秒内选择：帮皮尔斯多争取引爆时间、帮杰克切断 Ustanak 动力、或帮艾达确认卡拉实验室焚毁——原作走向是皮尔斯牺牲、抗体线生还、主谋与假艾达覆灭，你只能争取「更少伤亡」或「多一份可公开的证据」。

开场白建议：「整座海像被从底下煮沸。狙击手的右臂已经不是人类的颜色，他却还在校准最后一枪。你知道历史会写下病毒与英雄，但未必写下你按下的那个键——它让舱门早关一秒，或晚关一秒。」

关键NPC立场：**哈奥斯**（顶点巨体，条件性胜利优先）；**皮尔斯·尼凡斯**（已接受死刑式强化）；**克里斯·雷德菲尔**（必须被推上逃生链）；**杰克·穆勒**/**雪莉·柏金**（Ustanak 终战，需要火力或干扰）；**艾达·王**（焚实验室/助终结西蒙斯残存）；**德瑞克·C·西蒙斯**（若残存形态仍活动，政治顶点余波）。

主线钩子/支线：钩子=水下引爆与逃生舱同步；支线=保存皮尔斯临终通讯给 BSAA 抚恤链；支线=确认 Ustanak 彻底停机；支线=把西蒙斯罪证送到无法被「家族」抹除的公域节点。

危险度/规避：危险度贴近顶点；**本世界顶点为六阶**；对哈奥斯/西蒙斯终形态采「存在·情报优先/条件性胜利」，严禁「剧情削弱=战力归零」式硬刚；优先截断设施、保人、保证据。

任务方向/奖励：终局存活与秩序残响；奖励=残存高级武器、未污染的血液数据副本、BSAA/DSO 任务评价（影响你在后续世界是否被当成英雄或通缉犯）。
