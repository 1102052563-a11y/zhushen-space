# 轩辕剑外传：穹之扉
<!--meta lib=主库 tiers=一、二、三、四、五、六-->

## 剧情

**【作品来源】**
《轩辕剑外传：穹之扉》是大宇 DOMO 2015年3月26日发售的系列外传（陆之外传），原定1月因品质延期。背景上古「绝地天通」：伏羲女娲协同颛顼说服华胥之主关天门，凡人不可再随意觐天学神力。即时交错战斗、明雷、天赋先手；阿奇收妖与操作并行。主题曲《青空行》《比翼》《星念》。Steam/PS后发。文风神话史诗+人间守土。

**【世界定位】**
天地尚可相连的时代末端。男主**司空宇**（棍·风）外冷内热；女主**沐月**（魂玉·水）非凡间人，为寻归路与双生姐**季阳**相关；**凤煜**实为商王武丁（子昭）；**子巧**映射妇好；宠物**阿奇**背炼妖壶可收妖，精怪态似琥珀。配角**青梅**义妹魂献；**青榆**建木化形持伏羲琴复仇。一句话：关天门前夜，守土者与坠天者同行。

**【世界观 · 力量体系】**
力量本源：武技奇术、魂玉、建木化形、伏羲琴、炼妖壶、神力（天门未合前）、绝地天通后神力稀薄化。死亡：凡人死；魂可被献予妖灵；仪式级代价。特殊：明雷天赋（降血/掉宝等）、交错CD即时战、战术模式。

阶梯：
1. **部落武士** ≈ 一阶。
2. **有熊精锐/小妖** ≈ 二阶。
3. **商室剑士/方国将** ≈ 三阶。
4. **建木化形/琴控魂** ≈ 四阶。
5. **绝地天通仪式参与者** ≈ 五阶。
6. **伏羲女娲颛顼华胥主** ≈ 六阶。

乐园阶位映射：部落≈一阶；精锐≈二阶；商室≈三阶；建木≈四阶；仪式≈五阶；三皇级≈六阶。顶点情报优先。

**【地理 · 舞台】**
有熊村、中原商道、商王权场、建木相关神木域、天门/华胥边缘、壶中。卡片：灭村、寻归路、妇好钺礼法战、琴夺魂、季阳代仪、门合。

**【世界剧情线】**
① **有熊与青梅**  
司空宇对陌生人冷、对家人热。青榆以伏羲琴诱，青梅因灭村之痛献魂。灭村事件是仇恨引擎。

② **沐月坠世**  
沐月冰冷少言，众人助其寻天路。阿奇收妖开壶玩法。种族特性使其难近。

③ **凤煜与子巧**  
凤煜风趣俊美藏剑术智谋，真身武丁；子巧娇小持巨钺，真身妇好。人间商室线与神话线并行：平乱、礼法、军功。

④ **建木复仇**  
青榆因绝地天通恨轩辕氏相关，独行复仇。琴与魂、树与年轮。司空宇责在守土与救妹间撕裂。

⑤ **绝地天通**  
季阳如阳光，与共工情侣；沐月临时退缩，姐代完成关天门。人间进入「不可随意觐天」新纪元，神力与秩序重写。

⑥ **蝴蝶边界**  
可改支线妖、商道护送；难改绝地天通完成、妇好武丁史影、青梅献魂动机核。

**【主要人物】**
**司空宇**｜棍风｜外冷内热守土。**沐月**｜魂玉水｜坠天寻路。**凤煜（武丁）**｜剑火｜商王前史。**子巧（妇好）**｜钺土｜女将政。**阿奇**｜收妖壶。**青梅**｜义妹献魂。**青榆**｜建木·伏羲琴。**季阳**｜姐·代仪。**共工**｜季阳情侣。

**【势力图谱】**
有熊遗民、商室、建木复仇者、华胥/天庭侧、方国。敌友：关不关天门是总分歧。

**【贵重物品】**
炼妖壶（阿奇背）；伏羲琴；魂玉；钺；天门残钥（不可开）；建木年轮。

**【隐藏剧情 · 伏笔】**
阿奇与琥珀形态；武丁妇好史映；绝地天通系列宇宙地位；神魔之塔合作DLC接口。

**【大事记时间线】**
天地相通乱 → 有熊灭村阴影 → 沐月坠 → 商室同行 → 建木复仇 → 季阳代仪关门 → 守土新纪元。

**【叙事基调 · 雷区】**
神话沉重+人间烟火。忌：沐月无口癖废话；妇好纯萌化；低阶弑伏羲。最早切入：**有熊村琴声初起**或**沐月坠落后第一夜**。

**【世界剧情线·绝地天通的政治】**
关天门不是纯善：断有心人乱政，也断凡人学神自保。写仪式双方都有理，胜利昂贵。

**【世界剧情线·守土】**
司空宇主题是「卫吾土」：天门合后更要守脚下。高阶奖励是秩序与免死，不是天帝术。

**【名场面】**
琴诱青梅；沐月雨中鸿雁；子巧斧落宴；建木恨语；季阳代仪回头；门合人间。

**【可介入】**
灭村预警疏散、魂救、商道护钺、夺琴不断弦、仪式外围清场。

**【地理感官】**
村火焦味；魂玉冷光；钺沉重；琴弦树香；天门风直上。

**【时点三问】**
青梅魂是否已献？季阳是否已代仪？天门是否已合？

**【世界剧情线·武丁妇好史映】**
游戏明示凤煜=武丁、子巧=妇好。写商室线用军政与祭祀，不用穿越笑料消解。

**【世界剧情线·收妖并行】**
阿奇收妖独立于战斗指令，适合「战斗中仍完成收集」的任务设计。

**【世界剧情线·建木之恨】**
绝地天通断建木通天，青榆复仇指向轩辕氏符号。契约者若炫耀黄帝血统会加速敌对。

**【世界剧情线·双生太阳与月亮】**
季阳暖、沐月冷；退缩与代行是核心悲剧。关天门成功日即姐妹永恒亏欠日。

**【系统革命意义】**
系列从回合到即时交错，档案写世界时用「出手节奏/冷却」描述战力，而非纯回合气势。

**【世界剧情线·绝地天通题解】**
天地通→有心人乱→伏羲女娲+颛顼说华胥主关门→凡人不可随意觐天。善：断借神作乱。恶：断学神自保。仪式成功日即能力稀薄日，战后世界战力整体下修——高阶奖励勿发天帝术。

**【世界剧情线·司空宇守土】**
外冷内热、棍、风。主题诗「卫吾土」。灭村与青梅是软肋。对陌生人难近，是性格不是反派。

**【世界剧情线·沐月与季阳】**
月冷日暖。沐月退缩、姐代仪，是双生悲剧核心。共工情侣线增加代仪的私人代价。归路与留世是终局选择题，档案两边都可条件性。

**【世界剧情线·武丁妇好映射】**
凤煜=子昭武丁，子巧=妇好。商室线用军政祭祀写，宴会斧落是礼法战。禁止纯萌化女将。

**【世界剧情线·建木青榆】**
持伏羲琴、化形、恨轩辕氏相关。诱青梅献魂是仇恨具体化。夺琴不断弦是四阶好结局方向之一。

**【世界剧情线·阿奇与壶】**
小野猪→精怪态似琥珀；背炼妖壶；收妖与战斗指令并行。收集玩法服务探索，不打断即时战节奏。

**【战斗规则转译】**
明雷、天赋先手（降血/掉宝）、交错CD、战术模式——写成「先手抢气机、冷却轮转、可暂停整队指令」。

**【名场面链】**
琴声村→青梅危→沐月坠→同行寻路→商宴斧→建木恨→季阳代仪→门合守土。

1. 低阶弑伏羲女娲。2. 沐月话痨。3. 妇好萌宠化。4. 忽略青梅献魂。5. 门合后仍人人飞天。

任务方向/奖励：奖励：长棍技法、魂玉碎片线索、村寨信物。

**二阶（寻归路 · 立名）**

切入身份/时点：以商道向导身份，切入沐月坠落凡间、众人助其寻回天路。

初始事件：有人要抓「天女血」卖给有心人，沐月冰山外壳裂开一线。

开场白建议：「「她说自己不是人，却会在雨里停步看鸿雁。阿奇拱你小腿：收妖。」」

关键NPC立场：**沐月**少言；**凤煜**风趣藏锋；**子巧**斧钺开路；**阿奇**收妖。

主线钩子/支线：主线：寻归乡之路；支线＝阿奇形态与琥珀相似之谜、炼妖壶背负。

危险度/规避：危险度：中高；可战妖灵。

任务方向/奖励：奖励：魂玉用法、收妖协同、商道文书。

**三阶（商室棋盘 · 质变）**

切入身份/时点：以子巧身边副将身份，切入凤煜真实身份（武丁）与妇好历史映射逐渐显影。

初始事件：诸侯宴上有人质疑「女子持钺」，你要在礼法与军功间发言。

开场白建议：「「斧落案，酒器齐静。子巧笑：个头小，不代表砍不倒。」」

关键NPC立场：**凤煜（子昭/武丁）**智谋；**子巧（妇好）**军政；商贵族要礼；方国要利。

主线钩子/支线：主线：人间权力与天路并行；支线＝平叛、护祭祀、拒建木复仇代理。

危险度/规避：危险度：高；三阶可镇方国战。

任务方向/奖励：奖励：钺技法、商室军功、奇术。

**四阶（建木复仇 · 格局）**

切入身份/时点：以伏羲琴事件调查者身份，切入青榆建木化形、绝地天通仇恨。

初始事件：青梅魂灵被献，你要选救魂还是夺琴。

开场白建议：「「琴音里有树的年轮。青榆说：关天门的人，欠我整片天。」」

关键NPC立场：**青榆**复仇；**青梅**被诱；**司空宇**责重；轩辕氏相关被恨。

主线钩子/支线：主线：建木与绝地天通因果；支线＝琴弦禁咒、灭村真相、沐月双生。

危险度/规避：危险度：极高；建木化形≈四～五阶。

任务方向/奖励：奖励：伏羲琴残韵（高风险）、救魂法、建木年轮样本。

**五阶（绝地天通仪式 · 神战代理）**

切入身份/时点：以华胥边缘见证者身份，切入季阳代沐月完成仪式、与共工情侣线。

初始事件：沐月临时退缩，姐姐季阳顶上——你递的是「勇气」还是「退路」。

开场白建议：「「天门将合。阳光般的女子回头：告诉她，我替她痛一次。」」

关键NPC立场：**季阳**温热；**沐月**退缩愧疚；**共工**情侣；伏羲女娲侧代行监仪。

主线钩子/支线：主线：绝地天通完成与人间代价；支线＝颛顼说服链、有心人夺门、阿奇壶变。

危险度/规避：危险度：极高；仪式级≈五阶。

任务方向/奖励：奖励：天门残钥（不可开）、仪式见证、神魔之塔互文DLC接口（可选）。

**六阶（天庭门槛 · 顶点）**

切入身份/时点：以天门关闭后的人间秩序维护者身份，切入「凡人不可再随意觐天」新纪元。

初始事件：有人要炸开残余缝隙，你只能条件性协作封印，禁止写成弑神成功。

开场白建议：「「门合了，人间第一次真正只剩人间。司空宇握棍：那就守好脚下土。」」

关键NPC立场：**伏羲/女娲/颛顼/华胥主**顶点情报优先；**司空宇**守土；**沐月**归路两难。

主线钩子/支线：主线：新秩序落地；支线＝清残余妖、护有熊遗民、记青梅魂安。

危险度/规避：危险度：贴近顶点；**情报优先/条件性胜利**。

任务方向/奖励：奖励：守土盟约、壶权限、系列级天道认知。
