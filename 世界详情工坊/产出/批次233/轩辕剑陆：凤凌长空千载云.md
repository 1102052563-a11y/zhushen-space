# 轩辕剑陆：凤凌长空千载云
<!--meta lib=主库 tiers=一、二、三、四、五、六-->

## 剧情

**【作品来源】**
《轩辕剑陆：凤凌长空千载云》是大宇 DOMO 2013年8月9日两岸同步发售的正传第六作，Unity3D、水墨回归，商周背景。蔡明宏领衔；大陆畅游代理（后合约到期由大宇承接激活）。DLC《天外之章：浊山铸篇》2013年11月免费。主题：诸夏文化存续、牧野后东方暗战、凤灵与黑龙。片头《凤凌长空》、片尾《千年醉》。

**【世界定位】**
约公元前1046年牧野之战后。周武王克商，纣王鹿台自焚，但东方商遗民与西岐矛盾未消。主角**凤天凌**为商朝最大方国**奄国**大夫**凤千平**之子，奉命查牧野黑龙屠商军而商守护神**凤灵**迟迟不至的原因。同伴：金发**瑚月**（自称六十年姻缘）、周武王女**姬亭**（太姬，欲见姜契）、身毒后裔**迦兰多**、古蜀战甲师**蓉霜**。符鬼**琥珀**仅特殊者可见。一句话：商裔少年以鬼眼与剑，问诸夏与神灵为何缺席。

**【世界观 · 力量体系】**
力量本源：青铜武技、奇术（雷岩水火瘟邪六属，抗性非生克）、符鬼、脉轮（身毒七轮开合）、战甲术、奥义连击合击、壶中世界（炼妖冶金嫁术聚合鬼神塔隐藏）、剑气（白王）。死亡：凡人战死；DLC浊山铸「死而甲响」非常态。特殊：鬼眼见符鬼；千兵系统改体质（黑火门）。

阶梯：
1. **方国武士/工匠** ≈ 一阶。
2. **符鬼协同者/战甲学徒** ≈ 二阶。
3. **方国名将/子囂级前** ≈ 三阶。
4. **脉轮深开/战甲宗师边缘** ≈ 四阶。
5. **白王剑气/姜尚谋略战力投影** ≈ 五阶。
6. **凤灵·黑龙级神异** ≈ 六阶。

乐园阶位映射：武士≈一阶；符鬼显≈二阶；名将≈三阶；宗师/脉轮≈四阶；白王≈五阶；凤灵黑龙≈六阶。顶点情报优先。

**【地理 · 舞台】**
奄国、朝歌故墟、孟津牧野记忆场、东方方国、周镐京与周公线、身毒来路、古蜀战甲隐居山、渭滨姜尚余波、壶中世界。卡片：寻凤灵、四国合流、东战、浊山铸异变、白王问道。

**【世界剧情线】**
① **牧野余烬与使命**  
牧野：周袭朝歌，黑龙屠商军，凤灵缺席。凤天凌承父「保诸夏文化」教导，轻视周人，查神失踪。琥珀随行。

② **同伴合流**  
瑚月金髮执着姻缘；姬亭私自东行见姜子牙孙姜契，周公忧；迦兰多寻失踪亲人、脉轮引各方注目；蓉霜奉战甲宗师命入中原。文化冲突：商礼、周德、身毒术、蜀甲。

③ **东方暗战**  
商周紧张转局部战争。子囂大剑士、姬克等将领登场。姜子牙（姜尚，东夷出身辅周）定纲与商遗民骂名并存。主题从复仇转向「何为诸夏」。

④ **战甲与浊山铸**  
蓉霜兄浊山铸葬后异变（DLC），黑火门千兵、天书系统。战甲存续与玄火任务。

⑤ **白王与终局方向**  
白王白衣白发驾剑气，纹路通神器，问王道。凤灵与黑龙真相回收（档案写：协助召回/封印裂隙，不写未证实的私设终战细节）。文化存续高于灭周口号。

⑥ **蝴蝶边界**  
可改支线交易、隐藏迷宫、某场方国战；难改牧野史框架、姬亭太姬身份、凤灵黑龙神级逻辑。

**【主要人物】**
**凤天凌**｜火·青铜剑·鬼眼｜仇周保商→问王道。**瑚月**｜水·篪·云身｜姻缘执。**姬亭**｜岩·长弓·贯鹄｜王姬东行。**迦兰多**｜雷·脉轮·解封｜寻亲。**蓉霜**｜瘟·战甲·碎岩｜宗门命。**琥珀**｜符鬼。**姜子牙**｜谋主。**浊山铸**｜兄·DLC可控。**白王**｜剑气神器纹。**子囂**｜朝歌大剑士。**姬旦**｜周公。**姬克**｜周将。

**【势力图谱】**
商遗方国（奄等）、周室与周公、身毒来使、古蜀战甲宗、黑火门、神灵凤灵/黑龙。敌友：文化与生存优先。

**【贵重物品】**
琥珀契约；脉轮法；战甲；壶中世界；兵工谱；白王剑纹拓；凤羽；青铜礼器。

**【隐藏剧情 · 伏笔】**
瑚月上古族血；白王与轩辕纹；浊山铸死而不息；姜尚东夷身份政治；阿奇/琥珀跨作形态联想（穹之扉）。

**【大事记时间线】**
牧野克商 → 黑龙凤灵异象 → 凤天凌出发 → 同伴集 → 东战 → 浊山铸篇 → 白王/凤灵线收束。

**【叙事基调 · 雷区】**
水墨史诗、文化命题。忌：无脑仇周屠杀；忽略姬亭政治身份；低阶斩凤灵。最早切入：**大夫府接查神之命**或**姬亭出宫东行夜**。

**【世界剧情线·诸夏不是口号】**
父教「保存诸夏」在游戏中常落地为礼器、乐、文字与民，而非单纯帝号。周礼与商祀冲突戏应写具体器物与祭祀权。

**【世界剧情线·四国少年】**
每人一条文明线：商、神秘金发族、周、身毒、蜀。合击与奥义是系统也是主题：异质力量必须协作才有「长空」。

**【世界剧情线·姜尚的叛徒叙事】**
东夷人辅周，商遗民骂叛徒，周人尊尚父。契约者站队决定东方任务板。

**【名场面】**
鬼眼见琥珀；金发姻缘宣言；王姬上弦；脉轮演示引觊觎；浊山铸甲自握；白王问王道；黑龙压城。

**【可介入】**
护礼器南迁、假周令辨伪、战甲密令递送、壶中炼妖、隐藏支线迷宫、交易智斗奸商。

**【地理感官】**
青铜冷绿；镐京夯土；蜀山湿雾；身毒来客的异香；牧野骨白想象。

**【时点三问】**
四国是否已齐？浊山铸是否已葬？白王是否已问道？

**【世界剧情线·符鬼社会】**
一般人看不见琥珀，主角像自言自语——可写「疯」与「通神」的舆论战。鬼眼是天赋也是政治负担。

**【世界剧情线·六属抗性】**
雷岩水火瘟邪不相生克只看抗，战斗设计强调准备与配装，而非五行嘴炮。

**【世界剧情线·壶界七功能】**
炼妖、冶金、嫁术、聚合、鬼神塔、隐藏支线、隐藏迷宫——支线池巨大，低阶以迷宫与交易为主。

**【DLC浊山铸】**
葬后异变、可控、千兵改体质不可逆。黑火门独有。写四阶必给「开棺道德」。

**【文化冲突对白】**
商祀用牲与周德轻巫的辩论；身毒脉轮被当作「邪术」排挤；蜀甲被当成「妖甲」。

**【世界剧情线·牧野黑龙与凤灵缺席】**
史：周袭朝歌、纣鹿台死。戏：黑龙屠商军、凤灵不至。查因是主线引擎。黑龙可能是神异战争的外溢，凤灵缺席可能是人心/祭祀/阴谋多层——档案要求写「调查链」而非唯一官解私设。六阶任务是裂隙封印与文化守护，不是斩神秀。

**【世界剧情线·奄国与诸夏】**
奄为商大方国，大夫府保礼器即保文明。假周令收器是一阶经典冲突。凤天凌轻周是环境塑造，弧光应走向「问王道」而非永远种族歧视。

**【世界剧情线·四国同伴主题句】**
瑚月：姻缘与上古族谜。姬亭：王姬礼与私自东行的政治风险。迦兰多：脉轮他者化与寻亲。蓉霜：战甲存续与兄死异变。琥珀：可见性即特权。

**【世界剧情线·姜尚身份政治】**

开场白建议：「「金发女子说与你有六十年姻缘。王姬的弓已上弦：先报上名号。」」

关键NPC立场：**瑚月**执着姻缘；**姬亭**王姬傲骨；**迦兰多**寻亲；**蓉霜**战甲使命。

主线钩子/支线：主线：东行查因；支线＝身毒脉轮展示引各方觊觎、古蜀战甲密令、姜子牙渭滨余波。

危险度/规避：危险度：中高；可战地方精锐。

任务方向/奖励：奖励：脉轮入门、篪曲残谱、战甲碎片。

**三阶（商周暗战 · 质变）**

切入身份/时点：以东方方国联军参谋身份，切入商遗民与周东方经营冲突。

初始事件：子囂级剑士出场，战场从「查神」变成「谁配主中原」。

开场白建议：「「朝歌故墟夜火。大剑士子囂一剑断旗：诸夏不是口号，是血。」」

关键NPC立场：**子囂**商剑；**姬克**低调战功；**姬旦（周公）**制礼忧东；**姜尚**谋主。

主线钩子/支线：主线：东西方国战；支线＝护姬亭联姻线、拦黑火门、救浊山铸。

危险度/规避：危险度：高；三阶可镇城战。

任务方向/奖励：奖励：方国盟约、奇术抗性、兵工谱条目。

**四阶（战甲与黑火 · 格局）**

切入身份/时点：以战甲宗门客卿身份，切入蓉霜师门任务与DLC浊山铸线接口。

初始事件：浊山铸「死后异变」，你要选埋或开棺——开则黑火门千兵系统介入。

开场白建议：「「土未干，甲自响。蓉霜跪哭时，哥哥的拳套自己握紧。」」

关键NPC立场：**蓉霜**不舍；**浊山铸**可控异变；黑火门要体质改写；宗师要玄火。

主线钩子/支线：主线：古蜀战甲存续；支线＝壶中世界炼妖、嫁术、隐藏迷宫。

危险度/规避：危险度：极高；四阶≈战甲宗师边缘。

任务方向/奖励：奖励：战甲术、千兵残法、天书功能。

**五阶（白王与轩辕纹 · 神器层）**

切入身份/时点：以白王行迹追踪者身份，切入白发白衣剑气与上古神器纹路。

初始事件：白王要「王道」而非灭族，与凤天凌「复商」理想冲突。

开场白建议：「「剑气成纹，如鼎铭。白王问：你保的是民，还是共主虚名？」」

关键NPC立场：**白王**游云式强者；**姜子牙**定纲；**凤灵代行**测心；周成王侧使者观望。

主线钩子/支线：主线：神器与王道辩；支线＝瑚月身世（上古族）、迦兰多亲人下落。

危险度/规避：危险度：极高；白王剑气≈五阶。

任务方向/奖励：奖励：剑气纹拓片、王道策论、神器线索。

**六阶（凤灵与黑龙 · 顶点）**

切入身份/时点：以牧野天象见证者身份，切入黑龙屠商军、凤灵迟迟不至的真相回收。

初始事件：你只能协助「召回/安抚」之一环，禁止写成一掌斩龙。

开场白建议：「「天上黑鳞压城。有人喊凤灵，回声却是商民自己的哭。」」

关键NPC立场：**凤灵**守护神；黑龙神异；**凤天凌**鬼眼见因果；诸夏文化存续成终题。

主线钩子/支线：主线：神失踪之因与文化存续；支线＝封黑龙裂隙、护东方礼乐、DLC浊山铸收束。

危险度/规避：危险度：贴近顶点；**情报优先/条件性胜利**。

任务方向/奖励：奖励：凤羽信物、诸夏礼器守护权、系列级人脉。

**四阶补场** 浊山铸篇千兵改体质有不可逆。
