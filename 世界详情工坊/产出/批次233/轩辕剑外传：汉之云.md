# 轩辕剑外传：汉之云
<!--meta lib=主库 tiers=一、二、三、四、五、六-->

## 剧情

**【作品来源】**
《轩辕剑外传：汉之云》是大宇资讯 DOMO 小组开发的轩辕剑系列作品，正繁体版2007年12月19日发售，简体版2009年1月10日；首次全程过场动画声优配音。官方与《轩辕剑伍：一剑凌云山海情》《轩辕剑外传：云之遥》并称「轩辕剑伍系列三部曲」：时间上云之遥在前、汉之云居中、轩五殿后。背景三国（游戏称魏汉吴），主题为北伐、特战、剑气与「大汉」命运。2017年有电视剧改编。文风悲壮史诗，对诸葛亮形象偏「应变将略非其所长」的争议解读，以游戏为准。

**【世界定位】**
公元231年起，蜀汉丞相诸葛亮第四、五次北伐期间。汉军神秘特战部队**飞羽**（取关羽、张飞之名；天干十杰代号出自《史记·历书》）奇袭魏后方，与魏帝曹叡私建的**铜雀尊者**对撞。主角**焉逢**本名**皇甫朝云**，轩辕剑转世之一，孪生弟**皇甫暮云**失散后成白衣尊者。女主**横艾（笙儿）**为玉笙化身、现任壶中仙。一句话：飞羽刺与护之间，汉之云三义（朝云、暮云、横艾）共担倾覆危机。

**【世界观 · 力量体系】**
力量本源：兵法武技、奇术道法、机关术、剑气（轩辕剑转世）、仙器乐器化身、续命之法、炼妖壶/壶境。死亡：凡人战死即止；续命之法可令死者复生约十年（需女娲石、蜀山琼芝、雪山玉参、巫山瑶花），宿疾并继承；剑气可重铸肢体（耶亚希双手）。特殊：天干十杰排名、四大天王使职（持国/增长/多闻/广目）、回忆之殿幻术、镜界/炼妖。

阶梯（破坏力宁低勿高）：
1. **民夫/士卒/探子** ≈ 一阶。
2. **飞羽外围/魏军精锐** ≈ 一～二阶。
3. **天干十杰中坚（彊梧、游兆、尚章等）** ≈ 二～三阶。
4. **铜雀尊者成员/觉醒前剑气** ≈ 三～四阶。
5. **持国使赵云（续命）、壶中仙横艾、商横法力尽** ≈ 四～五阶。
6. **焉逢/暮云剑气合、赤衣全元神、山海界门槛** ≈ 五～六阶。

乐园阶位映射：士卒探子≈一阶；飞羽精锐≈二阶；十杰中坚≈三阶；尊者/剑气觉醒≈四阶；持国使·壶中仙≈五阶；转世合剑/山海界≈六阶。顶点=六阶巅峰：存在·情报优先/条件性胜利。

**【地理 · 舞台】**
- **益州/成都**：飞羽指挥、观星台、牢狱、朝议质疑北伐处。
- **北伐线**：祁山、街亭余波、木门道、五丈原、子午谷、葫芦谷、北原山、流马渊（木牛流马：独轮车与索道流笼）。
- **吴境建业**：孙尚香府、耶亚希历练起点、孙峻冲突。
- **巫山**：四仙子归处，横艾携妹终点。
- **魏境洛阳/五丈原魏营**：铜雀尊者、曹璿影子皇帝土木。
- **山海界/海岸**：终局焉逢离去与耶亚希灵体等待。
卡片可玩：奇袭魏后、寻兵器与瑶花、刺孔明观星台、巫山决战、海岸灵体。

**【世界剧情线】**
① **飞羽与第四次北伐**  
飞羽配合北伐奇袭魏后，遇铜雀尊者。最终打死张郃（游戏：以孙为饵迫其出，端蒙下令射杀；司马懿上表改写为中伏膝箭），北伐仍失败。名场面：特战对特战、名将之死的政治甩责。

② **兵器、瑶花与耶亚希**  
羽之部奉命寻蜀汉名将兵器与巫山瑶花。吴境接纳孙尚香义女**耶亚希·芭娜**（夷洲人）历练。耶亚希一见钟情焉逢；弟巴鲁瓦被孙峻所杀，她评理被砍双手，焉逢以剑气重铸。后绝命崖她救焉逢，焉逢以身相许报恩（编剧称先责任后培养感情，横艾线因临时加角与结尾未及调和而有争议）。

③ **飞之部、端蒙仇与刺孔明**  
端蒙（马蕴，马良女、马谡养女）因街亭斩谡恨孔明与张郃。飞之部端蒙一度行刺，羽之部救出。领导人焉逢渐知己为轩辕剑转世。第五次北伐，子午谷围司马懿、打死紫衣尊者（曹叡），汉军再陷缺粮。

④ **赵云续命与刺杀令**  
死而复生的持国使赵云见北伐劳民，私劝孔明息兵被拒，派焉逢等刺孔明。白衣尊者护主。飞羽内部迷茫：彊梧（严鹏）大义脱队后自尽；游兆（赵昂，赵云孙）转飞之部，北原刺司马懿失败牺牲；昭阳爱端蒙，临死表白；商横（费庭）法力尽亡；祝犁（黄采儿）葫芦谷負傷。

⑤ **观星台兄弟与合剑**  
焉逢再刺，与白衣决斗方知是失散弟暮云（徐庶养子徐暮云，误杀养父后遵遗志护孔明）。暮云化剑气被吸收，意识强烈影响焉逢。孔明走完人生（游戏：遭剑气重伤，随白衣逝世而逝）。焉逢决意赴山海界完弟愿，身体交暮云意识主导；后与赤衣尊者（磬儿，横艾妹，恋曹叡）决战，横艾携妹回巫山。

⑥ **山海界与灵体终局**  
山海界中暮云为证黄帝对天女之爱而自杀，外形复为轩辕剑，暮云魂与天女同封，朝云魂沉睡剑内。海岸**耶亚希**苦等，人间已数百载，肉身尘土，灵体靠剑气维持。青龙圣者（轩五夏柔）抱剑告知焉逢亡故；灵体相拥消散。赵云等十年未见焉逢归，续命期满再逝。增长使魏延后被屠三族，广目使费禕泄名单并抹飞羽史。

⑦ **微观可介入**  
运粮假账、瑶花护送、夜袭救人、狱中救端蒙、回忆之殿破幻、续命四物采集、防疫宣导（司徒蔷）、马弃马家玉、青冥自废双目赎罪。

⑧ **蝴蝶边界**  
可改局部护送胜负、非核心配角生死、某次夜袭路线；难无代价改：张郃死的政治后果、兄弟相残认知、孔明死期框架、山海界剑形态回收、耶亚希灵体终局。改动以追杀升级、史笔抹杀、续命崩溃回流。

**【主要人物】**
**焉逢（皇甫朝云）**｜性格：反思北伐、重诺、剑气难驭后觉醒｜能力：轩辕剑转世剑气（分予彊梧、耶亚希后更弱，巫山仙子助醒）、方天画戟｜弧光：飞羽第一→刺孔明→杀弟合剑→山海界→灵体消散｜立场：师赵云；爱耶亚希（责任起）；旧情横艾；敌铜雀后和解于巫山。

**横艾（笙儿）**｜性格：深情执着、实力深藏｜能力：壶中仙、玉笙化身、法术无底｜弧光：恋青年孔明被弃→助刘备檀溪至南阳后离开→爱朝云→入飞羽第九（只因喜「横艾」名）→救妹回巫山｜立场：巫山三仙子；曾制游兆；赵云曾欲任广目。

**皇甫暮云（白衣尊者/徐暮云）**｜性格：护孔明、悔杀养父｜能力：铜雀最强剑、纯熟剑气（磬儿打通气脉）｜弧光：失散→徐庶养子→误杀徐庶→白衣→死于兄剑｜立场：敌飞羽任务，血缘兄弟。

**耶亚希·芭娜**｜性格：单纯勇敢、萝莉气｜能力：武艺（孙尚香义女）、剑气铸手｜弧光：夷洲虏→义女→铸手→未婚妻→灵体等百年｜立场：恋焉逢。

**端蒙（马蕴）**｜性格：冷艳别扭、复仇优先｜能力：弯刀暗器、飞之部领导｜弧光：南蛮艰辛→恨孔明→刺败寻死激剑气｜立场：昭阳暗恋对象。

**彊梧（严鹏，字子君）**｜性格：直率嫉恶｜能力：巨弓强弩｜弧光：深交焉逢→拒刺孔明→对战后自尽｜立场：义高于令。

**游兆（赵昂）**｜性格：求名、常山枪｜能力：枪法｜弧光：不满袒护横艾→飞之部领导→刺司马懿死｜立场：赵云孙。

**徒维**｜性格：沉默｜能力：疗愈｜弧光：回忆殿唯一清醒｜真相：横艾以青年孔明为模的草人。

**尚章**｜性格：没心机｜能力：飞之部转羽｜弧光：马谡幺子、认端蒙姐→战后伴父坟与司徒蔷终老。

**赵云（持国使）**｜性格：爱民｜能力：名将、续命十年｜弧光：创飞羽→死→复活止北伐→再逝｜立场：朝云师。

**诸葛亮**｜性格：执念北伐、治国长｜能力：丞相权、非用兵神｜弧光：屡败屡战→观星台终｜立场：游戏批判与人格正面并存。

**曹叡（紫衣尊者）**｜性格：英主与土木两极（游戏用双生曹璿解释）｜能力：调度、铜雀领袖｜弧光：亲征破吴→葫芦谷救懿中剑气亡→托懿｜立场：敌飞羽。

**赤衣尊者（磬儿）**｜性格：为爱入魏｜能力：乐器化身、打通尊者气脉｜弧光：恋曹叡→狂怒决战→被姐带回。

**司徒蔷、青冥、马弃**｜蔷：女装书生剑客、防疫；青冥：异瞳通灵、废目赎罪；马弃：马超侄、马家玉。

**【大事记时间线】**
赤壁后皇甫家散 → 赵云收朝云 → 飞羽立 → 四次北伐张郃死 → 寻兵器瑶花耶亚希入 → 五次北伐紫衣死 → 赵云续命令刺 → 观星台兄弟合剑孔明终 → 巫山赤衣 → 山海界剑封 → 海岸灵体消散 → 飞羽史灭。

**【叙事基调 · 雷区】**
悲壮史诗、特战伦理、反神化孔明但不抹人格。忌：把焉逢写成无脑反贼；忽略耶亚希灵体重量；低阶硬刚剑气合；把魏蜀吴写成演义脸谱（以游戏魏汉吴为准）。最早切入：**第四次北伐奇袭前夜**或**羽之部接耶亚希时**。

**【世界剧情线·飞羽编制与天干政治】**
飞羽对汉军本身保密，代号甲乙丙丁…癸对应焉逢至尚章。排名是战力也是话语权：横艾第九是自愿，徒维高于师姐是草人设定的讽刺。领导层四大天王使：赵云创部，魏延、邓芝继管，费禕传令却最终灭史。契约者低阶任务应是「送令、守口、运药」，不是「取代焉逢」。

**【世界剧情线·铜雀与飞羽的镜像】**
两边都是特战、都有颜色/代号美学、都服务「政权存续」却走向互屠。曹叡比司马懿更早看穿孔明意在凉州；紫衣亲征是英主面，影子帝土木是恶评面。写战斗先写情报与补给，再写剑气特效。

**【世界剧情线·续命之法的伦理税】**
四圣物续命约十年，记忆渐回、宿疾仍在。赵云复活后看见的是民疲，不是胜利。他下令刺孔明，是爱民逻辑走到极端。契约者若滥用续命复活核心死者，应触发「宿疾+十年债+史笔反噬」。

**【世界剧情线·巫山四仙子赌约】**
无肉体、互不干涉、不靠肌肤之亲找挚爱：琴儿耗修为助周瑜东风后隐居；笛儿与曹植洛水后元神尽；笙儿孔明失败再朝云；磬儿曹叡。横艾救妹是仙约之上的血缘（乐器姐妹）。

**【世界剧情线·名场面清单】**
木门道张郃死；剑气铸手；绝命崖许婚；端蒙寻死激剑；观星台兄弟相认；赤衣全元神；海岸灵体；青龙圣者抱剑。

**【宝物使用边界】**
剑气分予会弱化；壶炼失败出异物；续命四物不齐无效；名将兵器是任务象征物非无限成长外挂。

**【时点三问（汉之云）】**
耶亚希是否已铸手？白衣是否已死于兄剑？山海界是否已启程？

**【世界剧情线·耶亚希铸手与责任婚】**
孙峻砍手不仅是残害，是吴廷权力对夷洲俘虏的蔑视。剑气铸手建立恩情债，焉逢许婚是古人「以身相许」逻辑，云之遥五丈原篇侧面印证非现代恋爱三选一。写感情线忌用后宫积分，宜用战场托付与海岸百年。

**【世界剧情线·诸葛亮观星台】**
游戏孔明不善奇谋却鞠躬尽瘁，自我认识后接受结局。批判来自端蒙、赵云、徐庶、横艾之口，终局仍正面人格。契约者四阶任务应是「传话/拦剑/护灯」，不是「取代丞相」。

**【世界剧情线·飞羽灭史】**
费禕泄名单、抹资料，使飞羽在历史上消失——解释玩家「正史无飞羽」的元叙事。高阶可夺史笔孤本，奖励是叙事权。

**【战斗与补给写法】**

## 阶位切入点

> 阶位↔境界：一阶≈士卒/探子，二阶≈飞羽精锐，三阶≈天干十杰中坚，四阶≈尊者/剑气觉醒，五阶≈持国使级·壶中仙，六阶≈轩辕剑转世合体/山海界门槛。顶点焉逢·暮云剑气与山海界天女线为六阶巅峰：存在·情报优先/条件性胜利，严禁战力归零。

**一阶（益州民伍与飞羽外围 · 井底）**

切入身份/时点：以益州运粮民夫、驿站听差或飞羽外围线人身份，切入公元231年诸葛亮第四次北伐前后、飞羽奇袭魏后方的前夜。

初始事件：军粮账本被改写，你被指控「通贼」；羽之部路过时，你必须在被斩与被征用之间立刻表态。

开场白建议：「「栈道上的火把一夜没灭。监军把你的名字从「民」改成「嫌疑」，刀已经出鞘。远处有人低喝代号：焉逢。」」

关键NPC立场：**焉逢（皇甫朝云）**先观察你是否泄密；**端蒙（马蕴）**冷淡务实；**游兆（赵昂）**要名望；本地监军要替罪羊。

主线钩子/支线：主线：第四次北伐奇袭链的后勤一环；支线＝救被征壮丁、截铜雀细作密信、护瑶花线索不外泄。

危险度/规避：危险度：中；规避铜雀尊者本体与剑气爆发。

任务方向/奖励：奖励：飞羽外围信物、益州路引、基础兵法/暗器。

**二阶（羽之部同行 · 立名）**

切入身份/时点：以临时编入羽之部的弓弩手/术士学徒身份，切入寻蜀汉名将兵器与巫山瑶花任务段。

初始事件：吴境接应耶亚希时，孙峻系势力盯上「夷洲义女」；你被要求在护航与出卖之间选。

开场白建议：「「建业夜雨里，耶亚希握紧刀柄。孙尚香的信半湿，信尾只有一句：让她历练。」」

关键NPC立场：**耶亚希**信任你若你护她；**横艾（笙儿）**测你动机；**彊梧（严鹏）**重义；**尚章**天真可被利用。

主线钩子/支线：主线：兵器与瑶花任务；支线＝耶亚希寻弟、游兆对横艾的不信任、飞之部夜袭魏营救人。

危险度/规避：危险度：中高；可战普通魏军与低阶法阵，规避白衣尊者。

任务方向/奖励：奖励：飞羽编号残片、疗伤奇术、吴蜀通行文书。

**三阶（张郃死与北伐折戟 · 质变）**

切入身份/时点：以战场斥候或飞羽辅战身份，切入飞羽诱杀张郃、第四次北伐仍失败的节点。

初始事件：端蒙以张郃之孙为饵，你要决定是否参与「人质道德」；战后司马懿上表改写战报，你手中有真伪两份。

开场白建议：「「木门道箭雨停时，名将倒地。端蒙的脸上没有喜，只有复仇未完。」」

关键NPC立场：**端蒙**要张郃血债；**司马懿**要责任外推；**诸葛亮**要继续北伐；**曹叡（紫衣）**侧细作盯你。

主线钩子/支线：主线：北伐失败后的民怨与飞羽裂痕；支线＝保护真战报、救伤兵、拒续命之法被滥用。

危险度/规避：危险度：高；三阶可镇一战，勿硬刚紫衣尊者。

任务方向/奖励：奖励：战利兵器胚、军功可兑药草、情报网。

**四阶（刺孔明与剑气觉醒 · 格局）**

切入身份/时点：以观星台外围护卫或赵云续命后的传令身份，切入持国使令刺诸葛亮、白衣尊者现身护主。

初始事件：你发现「刺杀令」与「广目使泄名单」并行：站队决定飞羽存亡。

开场白建议：「「观星台的灯一盏盏灭。白衣剑气割开夜雾，有人喊：焉逢，别下这手。」」

关键NPC立场：**赵云（持国使·续命）**要止北伐；**诸葛亮**要鞠躬尽瘁；**白衣尊者（暮云）**要护孔明；**费禕（广目）**要灭飞羽痕迹。

主线钩子/支线：主线：刺与护的双线；支线＝救端蒙出狱、彊梧大义自尽前后的人心、徒维草人真相边缘。

危险度/规避：危险度：极高；四阶可与尊者周旋，剑气本体条件性。

任务方向/奖励：奖励：剑气余韵（有副作用）、续命四圣物线索、飞羽最高权限碎片。

**五阶（兄弟合剑 · 壶中仙层）**

切入身份/时点：以巫山边缘道士或壶中仙客卿身份，切入焉逢认出暮云、合剑、横艾携赤衣回巫山。

初始事件：赤衣尊者狂怒全元神，你要在「帮羽之部」与「帮横艾救妹」间做有代价选择。

开场白建议：「「玉笙与玉磬同震，巫山云开一线。横艾说：她是我妹妹，先活，再谈魏汉。」」

关键NPC立场：**横艾**要救磬儿；**赤衣尊者**要为曹叡复仇；**焉逢**被暮云意识牵引；**徒维**守规则。

主线钩子/支线：主线：巫山收束与山海界启程；支线＝耶亚希铸手恩情、司徒蔷防疫、马弃马家玉。

危险度/规避：危险度：极高；壶中仙与剑气转世≈五～六阶边缘。

任务方向/奖励：奖励：乐器仙缘碎片、壶境权限、巫山通行。

**六阶（山海界与灵体终局 · 顶点窗口）**

切入身份/时点：以青龙圣者线旁观者或海岸守灵人身份，切入焉逢入山海界、耶亚希百年等待、灵体相拥消散。

初始事件：你被要求见证「轩辕剑形态回收」与「天女封印」之一环，禁止改写核心终局。

开场白建议：「「海面起雾。自称青龙圣者的女子抱剑而来，说：人间已过数百年。」」

关键NPC立场：**青龙圣者（夏柔）**传噩耗与剑；**耶亚希灵体**以剑气维持；**焉逢灵体**感动消散；天女线情报优先。

主线钩子/支线：主线：终局见证与飞羽史抹消；支线＝护海岸灵位、拦盗墓者掘剑、记录「汉之云」三义（朝云/暮云/横艾）。

危险度/规避：危险度：贴近顶点；**六阶巅峰存在·情报优先/条件性胜利**。

任务方向/奖励：奖励：系列级见证权、剑气残韵（不可爆星）、飞羽史孤本。

**一阶补场** 栈道绳桥断一根，你要用民夫绳结法抢修，否则羽之部迟到。
**二阶补场** 耶亚希双手被砍前后，铸手需要剑气与医者双线，司徒蔷可入。
**三阶补场** 张郃之死战报两份，送成都或送洛阳决定史笔。
**四阶补场** 端蒙故意寻死激剑气——你可拦可助，后果不同。
**五阶补场** 徒维草人破回忆殿，只有冷静者能醒。
**六阶补场** 海岸灵体消散时禁止强留，否则剑气反噬。
