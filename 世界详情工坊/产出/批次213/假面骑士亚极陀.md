# 假面骑士亚极陀
<!--meta lib=主库 tiers=一、二、三、五、六-->

## 剧情

**【作品来源】**
《假面骑士亚极陀》（仮面ライダーアギト，Kamen Rider Agito）为东映制作、朝日电视台2001年1月28日—2002年1月27日播出的特摄剧，平成假面骑士第2作，全51话，纪念假面骑士30周年。编剧以井上敏树为主，主演贺集利树、要润、友井雄亮、秋山莉奈。设定为《空我》两年后同一宇宙的间接续作：未确认生命体事件平息后，新威胁「Unknown/Lord」出现。文风神秘、哲学、三角骑士对立与「人类进化」主题。**以TV正典为准**。标签：津上翔一、冰川诚、苇原凉、G3、Lord、超能力种子。

**【世界定位】**
空我事件约两年后的日本。警方以「G3系统」（仿空我开发的强化服）应对新型超常杀人；失忆青年津上翔一在Unknown面前觉醒为亚极陀；苇原凉因父亲自杀之谜成为未完成的亚极陀「Gills」。一句话：神与人之间的进化审判，三骑士共斗与对立。

**【世界观 · 力量体系】**
力量分为三条路径：① **Agito种子**——人类潜在的进化之力，觉醒者可变身亚极陀，过强则暴走；② **G系列机械铠**——警视厅对空我数据逆向的装备，需后勤与冷却；③ **Lord（Unknown）**——自视为人类守护者的超常存在，按「资格」处决特定人类，受「光之种子/神」体系约束。死亡：凡人一击即死；G3机体会报废；Agito可重创再生但记忆与人格有代价；Lord可被骑士必杀击破。无无限复活。

战力阶梯（宁低勿高）：
1. **普通人 / 普通警员**：≈一阶。
2. **G3初期 / 轻伤Agito / 下位Lord边缘**：能抗小型异能、毁车级。≈二阶。
3. **Agito基本形态 / Gills / G3-X稳定运用**：裂石、高速格斗、对抗常规Lord。≈三阶。
4. **Agito Burning / 强化Gills / 上位Lord（El系）**：街区级火焰与杀伤，精神压迫。≈五阶。
5. **Agito Shining / 接近神之侧的存在**：高维打击、光剑级决战。≈六阶。
6. **Overseer / 神之侧顶点（星之支配者意象）**：文明级「资格审判」威胁。本世界顶点≈六阶顶，低阶规避。

乐园阶位映射：市民与普通警≈一阶；G3初期/觉醒边缘≈二阶；Agito基础·Gills·G3-X≈三阶；Burning与上位Lord≈五阶；Shining与终局神侧≈六阶。宁低勿高；顶点写条件性胜利，严禁战力归零式削弱。

**【地理 · 舞台】**
现代日本都市与海岸、湖畔、警视厅G3机库、美杉家与风谷家日常空间、ラーメン屋台等市井据点、「晓号」相关海域与船难记忆地、Lord出没的高楼与荒野。电影线另涉军方GA设施与Project G4。AI分层：市井日常→警视厅作战室→Lord猎场→神之侧启示空间。

**【世界剧情线】**
① **两年后的新杀人**  
空我事件结束后，警方以「不可能的死法」立案。强化服G3交付优秀警员冰川诚。失忆的津上翔一寄住风谷家，在Unknown出现时变身亚极陀。世界状态：英雄已成传说，新怪物不按旧规则。

② **第三战士Gills**  
苇原凉调查父亲自杀，身体异变成为Gills——不完全的Agito，战斗越强越被反噬。三骑士初次冲突：G3要捕捉未知，Agito要救人，Gills要真相。名场面：湖上激突、红炎之剑、两人G3。

③ **资格与记忆**  
翔一碎片记忆指向「晓号」海难与超能力者；Lord以「守护人类」为名杀掉「不合格」或「会带来混乱」的种子持有者。精神科、数字谜题、可疑女人等单元推进世界观。冰川在制度与正义间摇摆；小泽澄子等技术组支撑G3升级为G3-X。

④ **暴走与复活**  
Burning形态力量过大导致失控；凉一度濒死又复活。木野薫等「第四人」线揭示更深层的Agito复制与黑暗战士。警方、民间、神之侧三方情报战白热化。

⑤ **光与暗、共同战线**  
Lord攻势升级，三骑士由对立转为协同。风谷真鱼等日常角色成为「要保护的普通」象征。北条透等内部政治制造摩擦。

⑥ **星之支配者与终局**  
「神」与El（水/风/地）级存在下场，灭绝脚步逼近。翔一以Shining完成对「人类是否有资格进化」的回答。结局：Lord危机平息；众人回到各自人生（冰川一度卸任G3等设定随资料）；进化主题开放收束。低阶禁止硬刚神侧本体。

⑦ **微观可介入**  
尸检报告造假、G3冷却液短缺、超能力者名单泄露、屋台情报、美杉家晚餐——优先这些小事件。

⑧ **蝴蝶效应**  
可改：救下某集预定牺牲的种子持有者、提前修好G3-X、调解三骑士内斗。不可无代价改：Agito失控逻辑、Lord「资格」审判规则、终局神侧战力。

**【世界剧情线·分卷细目】**

**卷一：觉醒（约1—12话）**  
G3出阵、Agito觉醒、Gills登场、三角对立成型。可介入：G3运输护卫、湖畔目击、风谷家寄宿纠纷。

**卷二：资格之谜（中盘）**  
记忆碎片、数字谜、Burning暴走、G3-X、内部政争。可介入：捕获作战、技术泄密、凉的医疗线。

**卷三：共同战线（后盘）**  
黑暗战士、El级、三骑士联手。可介入：联合演习、真鱼保护任务。

**卷四：终局**  
星之支配者、灭绝前夜、Shining决战。电影《Project G4》可作支线副本：军方GA、少女预言、蚁Lord、G4系统。

**【世界剧情线·三骑士结构】**
核心是Agito（进化种子）/ G3（国家机器）/ Gills（不完全进化）三角互相审视。翔一失忆寄住风谷家，Unknown出现即变身；冰川接G3任务本为捕捉不明战士；凉为父自杀真相承受生物铠反噬。中期冲突多于合作，后期El级威胁下才共同战线。主题句：人类获得力量后能否仍守人性。

**G系列与电影G4**
G3仿空我数据，依赖车辆后勤、冷却与小泽澄子维护；G3-X提升稳定与火力，仍有时限故障。电影Project G4：军方GA窃取技术，用预言少女纱绫香强化系统，水城驾驶G4与冰川对决，蚁Lord攻袭设施。可作五阶支线：救少女、夺G4核心、阻止军方量产。

**Lord与资格**
Lord自视为人类守护者，处决不合格或会带来混乱的种子持有者。水/风/地之El为上位执行者。神之侧定义资格，终局逼人类回答进化是否被允许。与空我古朗基游戏不同：敌人有道德辞令。数字谜、名单、晓号海难记忆是中盘解谜主轴。

**Burning、Gills与第四人**
Burning战力质变，人格身体双重风险；凉死而复活强化Gills。木野薰等第四人线暗示Agito可被复制诱导。北条透用逮捕与舆论挤压翔一。特别篇国枝东助控Burning。终局翔一以Shining回答，危机平息，众人回归日常。

**【世界剧情线·记忆与晓号】**
翔一失忆不是开挂，是创伤与神侧实验的后遗。晓号船难、超能力者聚集、名单行刑，构成中盘推理骨架。数字谜贴在被害人门上，解开可延命一晚——契约者最适合的三阶任务是「解谜救人」而非「越级斩El」。风谷真鱼作为普通人锚，反复提醒力量存在的理由是保护灯火不是成为神。

**【世界剧剧情线·三骑士关系学】**
冰川要程序与公共安全，会先指向捕获；翔一要救人，会先挡在受害者前；凉要真相与复仇，会先挥向可能相关的影子。早期误战是结构必然。G3-X出现后冰川战力接近生物骑士但仍受后勤束缚；Gills越强越痛；Agito越强越怕失控。协同的前提是共同敌人El级与神侧，不是突然变好友。

**【世界剧情线·制度反派】**
北条透窃听、逮捕、抢功、推动更危险武装，是「人类内部比Lord更先伤人」的代表。小泽澄子夹在技术理想与滥用之间。本部解散危机、新上司、捕获作战等单元展示官僚对非常事态的笨拙。契约者可用制度漏洞，也可能被制度当弃子。

**【世界剧情线·电影与特别篇定位】**

**二阶（G3与觉醒 · 第三阴影）**

初始事件：G3奉命捕获「不明战士」；红与金的影子在屋顶交错，你必须选择帮哪一边打开撤退通路。

任务方向/奖励：G3模块试用、冷却剂、Agito目击报告权限。
