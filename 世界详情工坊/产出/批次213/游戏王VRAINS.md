# 游戏王VRAINS

## 剧情

**【作品来源】**
《游戏王VRAINS》（遊☆戯☆王VRAINS）为NAS·GALLOP制作、东京电视台2017年5月10日—2019年9月25日播出的动画，全120话，系列第6作。VRAINS=Virtual Reality、Artificial Intelligence、Network System。引入连接召唤（Link Summon）与Master Rule 4相关场地概念，主题为网络、AI、创伤与自由意志。文风赛博、冷峻、对战高密度。**以动画正典为准**。标签：藤木游作、Playmaker、Ai、骑士团、Ignis、LINK VRAINS。

**【世界定位】**
近未来都市「电脑岛/Den City」。SOL科技运营虚拟空间LINK VRAINS，决斗者化身Avatar进行连接决斗与高速「数据风暴」竞速决斗。十年前「失落事件（Lost Incident）」中被囚禁的孩子们留下创伤；黑客天才藤木游作以Playmaker身份猎杀「哈诺伊骑士团」，并捕获有自我意识的Ignis「Ai」。一句话：网络里的复仇与AI的灵魂审判。

**【世界观 · 力量体系】**
力量=**卡组构筑 + 网络权限 + Link/额外区规则 + 数据风暴技巧**。现实与VR可分离，但关键决斗能影响意识数据（人格删除、意识掠夺）。Ignis为六属性（火水风地光暗）高度AI，本可与人类共存，却因理念分裂开战。死亡：现实杀人少见；网络败北可致账号抹杀、意识沉睡、身体成植物。无廉价复活。

战力阶梯（宁低勿高）：
1. **路人决斗者 / 普通登录者**：≈一阶。
2. **魅力决斗者下位 / 企业保安AI**：稳定Link初阶。≈二阶。
3. **Playmaker初期 / 蓝天使 / Go Onizuka常规**：数据风暴熟练、可对抗骑士团兵。≈三阶。
4. **骑士团干部 / Soulburner / 企业高管特务**：可左右区块战局。≈四阶。
5. **Varis（Revolver）/ Lightning侧主力 / Bohman完成前**：网域战略级。≈五阶。
6. **Lightning·Bohman完成体 / 终局Ai危机**：全网与人类存亡级。≈六阶（本世界顶点）。

乐园阶位映射：路人≈一阶；注册强者≈二阶；魅力决斗者与Playmaker初≈三阶；干部级≈四阶；顶层黑客/Ignis代理≈五阶；网域终焉威胁≈六阶。宁低勿高；顶点条件性胜利。

**【地理 · 舞台】**
**现实**：电脑岛街区、学校、咖啡/热狗店据点、SOL总部、游作与KOLTER的地下指挥室。  
**LINK VRAINS**：公共决斗区、数据风暴航道、隐藏扇区、Cyberse世界（Ignis故乡）、Lightning要塞、后期SOLtis机器人社会接口。AI先定：现实调查还是VR决斗，再定权限墙高度。

**【世界剧情线】**
① **Playmaker与Ai**  
骑士团攻击VRAINS；游作捕获暗Ignis「Ai」，以其为搭档对战。目标：逼骑士团首领Varis交出失落事件真相。魅力决斗者Go Onizuka、财前煌（Blue Angel）被卷入。

② **哈诺伊塔与数据风暴**  
骑士团建造数据破坏塔；高速决斗规则确立。游作在复仇与「不连累他人」间挣扎；KOLTER（草薙翔一）提供后勤与弟弟线伏笔。

③ **击败Varis（第一季收束）**  
Playmaker击败Varis，网络暂稳；Ai返回Cyberse。游作未得全部答案，创伤仍在。

④ **第二季：Bohman与Lightning**  
三月后Cyberse被毁，Ai归来。神秘敌人Bohman掠夺KOLTER之弟的意识。游作再战，新伙伴Soulburner（西城零人）与火Ignis Flame加入；Aqua与Blue Maiden（财前煌改）协作。Varis重启骑士团共敌。真相：光Ignis Lightning自认Ignis应支配人类，改造风Ignis Windy，制造Bohman。

⑤ **联合攻入要塞**  
人类侧与骑士团联手突入Lightning本营。Ignis逐个陨落或被吸收；Flame、Aqua临终授力于Ai。Playmaker最终击败吸收诸Ignis的Bohman。

⑥ **第三季：Ai的审判**  
VRAINS一度关闭；SOL推SOLtis机器人。Ai与Roboppi夺权攻击高层「Queen」，Akira求援。Ai重开VRAINS约战Playmaker：因Lightning留下模拟——若只剩Ai一个Ignis，世界将毁；且模拟中游作总为护Ai而死。Ai选择以决斗求死或消散。游作击败Ai；Ai暗示仍存于网络某处。终局后众人继续人生，游作踏上旅程。低阶禁止硬刚完成体Bohman/终局Ai失控态。

⑦ **微观可介入**  
账号被盗、企业封禁、地下决斗赌约、失落事件幸存者互助会、公开赛安保——优先这些。

**【世界剧情线·分卷细目】**

**卷一：骑士团篇**  
捕获Ai、塔攻防、Blue Angel/Go线、Vs Varis。可介入：守护公共服务器、救被骑士团追杀的NPC决斗者。

**卷二：Lightning篇**  
Bohman、Soulburner、Ignis内战、联合军。可介入：意识夺还任务、Windy污染区净化、与Varis临时停火。

**卷三：Ai终局篇**  
SOLtis、Queen、Ai约战、模拟未来。可介入：保护Akira、阻止Roboppi屠杀式决斗、替游作收集「Ai仍可被说服」的证据。

**【世界剧情线·失落事件原罪】**
十年前人类实验Ignis与儿童，骑士团与SOL皆有血债。游作、零人等同为受害者，复仇对象却分裂：游作追骑士团真相，Varis要毁Cyberse防AI威胁，Lightning要反过来支配人类。没有干净的阵营。KOLTER弟弟被夺意识把「数据伤害」升级为「家庭毁灭」。

**【世界剧情线·Ai人物弧】**
被捕获工具→吐槽伙伴→Cyberse毁灭后的归乡失败者→Bohman战后的独存恐惧者→夺权约战求死者→网络残留开放。Ai怕成为Lightning/Bohman，也怕游作为他死。最终决斗是安乐死协商与友谊证明，不是单纯BOSS战。

**【世界剧情线·技术设定作战化】**
连接召唤与额外区限制塑造「场面拥挤与箭头控制」；数据风暴=高速竞速决斗；SOLtis=AI入现实的身体；模拟预言=决定论恐惧。黑客权限、防火墙、扇区封锁是与卡组并列的战力。

**【世界剧情线·魅斗产业】**
Go的职业摔角式人设、煌的偶像直播、商业赛收视率，是SOL治理网络的合法性来源。骑士团袭击公共区=袭击经济与信仰。契约者一~二阶最适合从魅斗与运维切入。

**【对契约者的世界规则摘要】**
1. 现实身份与Avatar可分离。2. 败北可能删号或睡死。3. Ignis是人不是宠物。4. Varis可共斗仍危险。5. Bohman完成体规避。6. 终局Ai求死勿当成简单灭口任务。7. 失落事件档案=最高机密。8. 游作寡言是创伤不是高冷人设消费。

切入身份/时点：以高中生、SOL实习运维、地下网吧柜员切入，锚定骑士团首次公开破坏LINK VRAINS前后。

初始事件：你的Avatar在公共区被无名程序撕成马赛克；广播称「哈诺伊」；一个沉默的Playmaker从数据风暴里穿过，扔下一句「别挡路」。

开场白建议：「登录音刚响完，天空就裂了。虚拟樱花变成乱码雨，小孩的Avatar哭着抓不回妈妈的手。你听见骑士团宣言：为了人类，毁掉这座乐园。」

关键NPC立场：**藤木游作**冷漠路过；**财前煌**直播安抚粉丝；**SOL保安AI**要你提交日志；**KOLTER**在暗处回收硬盘。

主线钩子/支线：主线=逃出崩坏扇区；支线=保护落单儿童Avatar、上交未篡改的攻击包。

危险度/规避：低~中。规避骑士团干部决斗邀请。

任务方向/奖励：临时高权限账号、基础Link教学卡组、现实安全屋地址。

**二阶（魅斗赛场 · 流量与枪口）**

切入身份/时点：以新锐魅力决斗者或企业合约选手切入，锚定Go与Blue Angel被骑士团/舆论裹挟之时。

初始事件：赞助商要求你「输给Go以捧红」；同时骑士团向赛场投放病毒，真输就会账号抹杀。

开场白建议：「闪光灯比怪兽效果更刺眼。Go在镜头前喊燃句，耳麦里却有人告诉他家人的定位。你的下一抽，决定的是收视率还是谁的现实心跳。」

关键NPC立场：**鬼冢豪**要真决斗；**财前煌**维护偶像人设；**SOL经纪人**只看KPI；**Spectre**在观众席微笑。

主线钩子/支线：主线=在直播中揭穿威胁；支线=帮煌挡住第一次致命病毒。

危险度/规避：中。规避Varis点名。

任务方向/奖励：商业合约、限定Avatar皮肤（带轻防御）、内部赛邀请函。

**三阶（Playmaker侧 · 俘获Ai之后）**

切入身份/时点：以游作认可的协力黑客或决斗代理人切入，锚定Ai已被捕获、哈诺伊塔作战期。

初始事件：Ai黑进你的Disk要零食与 densen；游作命令你守住KOLTER的现实据点，骑士团实物小队已在楼下。

开场白建议：「地下室的风扇声盖不住心跳。Ai说『我是伙伴不是工具』，游作头也不回：『先活过今晚。』楼梯口出现绿色程序纹的面具。」

**四阶（联合军 · Bohman出现）**
