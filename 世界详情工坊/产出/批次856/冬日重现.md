# 冬日重现

## 剧情

**【作品来源】**
《冬日重现》作者**雪梨炖茶**，起点中文网连载轻小说（书号约1044572097），标签偏恋爱日常、悬疑、校园、时空。文风是「小岛日常壳 + 时间回溯刃」：表面写高中同学重聚与青涩拉扯，内里是连环凶杀、神庙祭祀与死亡触发的时间折返。公开页更新已至四百九十余章量级，**连载中，档案不编造未公布终局**。  
**身份核验（强制）**：本书与田中靖规漫画/动画《夏日重现（Summer Time Rendering）》**不是同一IP**；与东野圭吾作品及影视改编亦无直接关系。乐园档案一律按雪梨炖茶笔下**张述桐**线书写，禁止串影平人名（慎平、潮、小舟等）。

**【世界定位】**
冬日小岛。八年前连环凶杀把两名女同学的命运钉在错位时间点上；主角**张述桐**重返小岛后获得「死亡触发回溯」的私人能力，试图救下**顾秋绵**与**路青怜**，却反复撞上规则反噬与岛上人情网。一句话：用高中生的体温去撬时间的冻土。

**【世界观 · 力量体系】**
本世界无修仙境界，战力轴是「凡人肉体 + 时间异常 + 岛上祭祀/凶案体系」。

- **普通高中生/岛民**：徒手与冷兵器级，坠崖、溺水、刀伤皆可致命。破坏力≈一阶。
- **回溯初成**：死亡瞬间将意识/状态拽回过去某一锚点，窗口有限，记忆与肉体负担累加；能改小范围选择。≈二阶。
- **回溯有成**：多次折返后预判凶手动线、布置伪证与撤离，把「再来一次」变成战术资源。≈三阶。
- **连环案层**：直面八年前连环凶杀执行逻辑与祭祀结构，个人武力仍凡人，信息与布局可压住一域。≈四阶。
- **改写命运层**：试图同时保住顾秋绵、路青怜与无辜者，触及时间规则反噬（人格侵蚀、时间债、重复死亡）。≈五阶。
- **时间规则本体**：不可作为战力碾压对象；低阶仅情报优先/条件性胜利。＝超阶阴影。

破坏力校准：全程贴近凡俗到城市悬疑，**严禁**毁城爆星写法。  
乐园阶位映射（宁低勿高）：普通高中生/岛民≈一阶；回溯初成≈二阶；回溯有成·布局救人≈三阶；介入连环案核心≈四阶；改写八年命运·多线同保≈五阶；时间规则本体＝超阶。

**【地理 · 舞台】**
- **冬日小岛**：四面环水，冬潮重、游客少；渔业与旅店支撑日常，同学重聚与凶案回声在此闭环。
- **深山神庙**：顾秋绵线核心空间，香火与禁忌并存，「岛记得血」的具象锚点。
- **大湖/水下倒影**：公开简介点出的第二视觉标记，可作案发现场或回溯触发的地理信号。
- **校园与双路**：一路通镇中心，一路通深山；顾秋绵放学常走山路，构成早期失踪叙事地理理由。
- **城里线**：张述桐曾离岛，再因同学聚会式重返——提供局外人对照。

**【世界剧情线】**

**阶段一 · 重返与第一次死亡**  
二十四岁的张述桐回到冬日小岛，名义是久违同学聚会，实际被路青怜等人的消息与未愈旧案拖住。他确认：八年前连环凶杀并未真正落幕；顾秋绵与路青怜的死亡时点在不同回溯里会错位。第一次真正「死亡」后，他意识到能回到过去——窗口短、代价不明，岛上每个人都像在演他看不穿的戏。

**阶段二 · 双线救人与同学网**  
他在不同时间锚点分别接近顾秋绵（神庙、山路、禁忌）与路青怜（社交试探、「你怎么知道」）。救A可能导致B更早死；逼急凶手会让普通人变祭品。公开角色榜以顾秋绵、路青怜、张述桐为核心，日常恋爱与悬疑并行，禁止后宫打卡式感情线。

**阶段三 · 连环案真相层**  
八年前连环凶杀与神庙祭祀体系挂钩。张述桐在多次回溯中拼出「谁在何时何地动手、谁沉默、谁借刀」，开始主动假死、调包、引凶，用凡人手段做时间战术。胜利标准是「多带出一个活人离开当晚」，不是打赢超自然本体。

**阶段四 · 时间债与人格磨损**  
回溯堆高后记忆错乱，亲友觉得他「不像本人」。公开连载仍在推进寻找顾秋绵/路青怜相关线索与重返善后，**档案停在已公开冲突层**：不宣布最终存活名单，不发明终局BOSS真名。

**阶段细目（可观察因果）**  
- 聚会夜：迟到、失联、岸边谈话——任何未接来电都可能是下一具尸体的预告。  
- 神庙夜路：山风、香灰、水下倒影同现时，回溯窗口易被「岛」感知。  
- 城市来电：留言与岛上时间不同步，是发现时间错位的廉价线索。  
- 假死演练：死前布置证据链，让下一次自己少走弯路。  
- 同学群里的玩笑话反复在不同回溯里改口，说明有人在故意统一口径。  
- 旅店登记本上的笔迹前后不一，指向有人在改「谁来过」。  
- 山路护栏缺口与八年前新闻照片重叠，是地理记忆的硬锚。  
- 张述桐开始给每次死亡编号，编号本身成为人格是否还完整的指标。

**【主要人物】**
- **张述桐（主角·回溯者）**｜性格：表面木头、内里高压，嘴笨但记仇也记恩；对「再来一次」既依赖又恐惧。｜装备·能力：死亡触发时间回溯（窗口/代价随次数恶化）；凡人肉体，靠布局与情报。｜人物弧光：逃避旧案的归乡者 → 主动扛时间债的拯救者。｜立场关系：视顾秋绵、路青怜为必须救下的人；对旧案相关者戒备。
- **顾秋绵（核心·神庙线）**｜性格：安静执拗，习惯把危险揽到自己身上；带巫女/祭祀疏离感。｜装备·能力：凡人；掌握神庙路径、禁忌与岛上传说。｜人物弧光：从「注定死在山路」的符号 → 被反复改写的活人。｜立场关系：张述桐最想改写的死亡之一；与深山神庙绑定。
- **路青怜（核心·社交线）**｜性格：更外放、会试探，电话与留言是她的战场。｜装备·能力：凡人；情报与人际关系强于武力。｜人物弧光：重聚牵线人 → 连环案棋盘上的活靶。｜立场关系：与张述桐有未说完的旧话；死亡时点常与顾秋绵错位。
- **岛上同学（公开具名以外写不详）**｜性格：八卦、冷漠或假装失忆。｜装备·能力：凡人。｜人物弧光：局外人 → 证人/牺牲品。｜立场关系：提供假线索与真目击。
- **八年前连环凶杀相关者（执行/conceal，未核真名则不详）**｜性格：把「岛的秩序」置于人命之上。｜装备·能力：凡人凶器 + 地理熟悉 + 可能的祭祀。｜人物弧光：旧案阴影 → 现时威胁。｜立场关系：张述桐的狩猎对象。
- **神庙相关长辈/祭司（不详）**｜性格：守禁忌。｜装备·能力：仪式知识。｜人物弧光：解释「岛为什么记得血」。｜立场关系：可合作可敌对。
- **城里旧识（不详）**｜性格：劝他别回去。｜装备·能力：凡人。｜人物弧光：对照线。｜立场关系：情感锚。
- **警察/岛上治安（不详）**｜性格：程序正义但缺人手。｜装备·能力：公权力。｜人物弧光：官方迟钝。｜立场关系：可提供尸检时间线。
- **旅店/渔民目击者（不详）**｜性格：怕事。｜装备·能力：无。｜人物弧光：一次性证人。｜立场关系：卖情报换封口。
- **时间规则回声（非人格化则不单列）**｜性格：无。｜装备·能力：超阶阴影。｜人物弧光：只露边。｜立场关系：情报优先。

**【势力图谱】**
- **同学重聚圈**：松散社交网，信息真假混杂，一~二阶主舞台。  
- **深山神庙体系**：祭祀与禁忌，掌握山路与「不该说的故事」。  
- **旧案沉默联盟**：受益于八年前结案叙事者，阻止重开调查。  
- **岛外公安/媒体（弱）**：偶尔施压，难长期驻岛。

**【贵重物品】**
- **未接来电/留言录音**：路青怜线关键物证，时间戳对死亡窗口。  
- **神庙香袋/祭器**：顾秋绵线信物，可能标记「可被岛感知」之人。  
- **旧案卷宗复印件**：名字涂黑处即伏笔。  
- **张述桐回溯手记**：防人格磨损后失忆的「第N次死亡备忘」。

**【隐藏剧情 · 伏笔】**
- 双女主死亡时点错位，暗示非同一凶手时间表或时间被折叠。  
- 水下倒影与神庙对称，或存在第二倒影空间。  
- 能力来源简介未写死——禁止系统面板化。  
- 连载中：禁止发明终局真凶全名与全员存活BE。

**【大事记时间线】**
- 八年前 → 连环凶杀，结案叙事成型。  
- 张述桐离岛 → 城市线。  
- 重返冬日小岛（同学聚会） → 开端。  
- 第一次死亡与回溯 → 能力觉醒。  
- 多次救人失败/部分成功 → 中期。  
- 逼近连环案核心与时间债 → 当前公开层。  
- 终局 → 连载中，不详。

**【叙事基调 · 雷区】**
基调：冷潮、雨雾、没说完的话；恋爱克制，悬疑要落到时间与地点。  
忌：写成《夏日重现》影平线；忌无代价无限读档；忌「群像」代真名；忌把张述桐写成超能力英雄。  
最早切入：张述桐刚下船/刚到旅馆的黄昏。

## 阶位切入点

> 阶位↔原作：一阶≈普通高中生/岛民；二阶≈回溯初成；三阶≈回溯有成·布局救人；四阶≈介入连环案核心；五阶≈改写八年命运。顶点（时间规则）＝超阶：存在·情报优先/条件性胜利，严禁战力归零。

**一阶（普通岛民 · 同学聚会夜）**

切入身份/时点：转学生/回岛远亲，卡在张述桐回岛后第一次聚会前夜。

初始事件：旅馆走廊未接来电震动；有人说路青怜在岸边等「早该回来的人」。你被拉去买酒，半路见神庙方向有灯。

开场白建议：「冬雨把码头灯牌泡白。你拎着罐装啤酒回旅馆，手机震了一下——本地号码只丢五个字：别让他上山。身后楼梯有人喊张述桐，你还没见过他长什么样。」

关键NPC立场：**张述桐**（警惕但缺帮手）；**路青怜**（试探你是否「也知道」）；**顾秋绵**（若出场则回避）；旅店老板（不详，劝你别多事）。

主线钩子/支线：跟着避开第一次无意义冲突；支线：替路青怜找回码头钥匙；记录当晚进出旅馆时间。

危险度/规避：中。规避独走山路、神庙直播。

任务方向/奖励：活过聚会夜；人情、手绘地图、涂黑旧新闻复印件。

**二阶（回溯初成 · 第一次死亡后）**

切入身份/时点：知情者，时点在他首次回溯苏醒前后。

初始事件：同一巷口见他走过两次，第二次袖口多未干血。

开场白建议：「潮水声倒带三秒。张述桐抬头辨认你是第几版。他问：这次，你还记不记得岸边那通电话？」

关键NPC立场：**张述桐**（临时同盟）；**路青怜**（开始疑时间）；**顾秋绵**（神庙钥匙）；旧案沉默者（不详，盯梢）。

主线钩子/支线：验证回溯规则；支线：保护普通同学不被灭口。

危险度/规避：中高。规避在窗口内把「未来死法」告诉错误的人。

任务方向/奖励：可信度；共同手记一页、神庙外围口头许可。

**三阶（布局救人 · 双线并行）**

切入身份/时点：私家调查员/实习记者，同时盯山路与城区。

初始事件：两个死亡预告落在同一晚不同地点。

开场白建议：「左手山路手电，右手城区电台。张述桐只给你一句：先保活的，再解释时间。」

关键NPC立场：**张述桐**（指挥）；**顾秋绵**（可劝离山路）；**路青怜**（可转移旅馆）；警察（不详，半信半疑）。

主线钩子/支线：完成一次「双线都活过凌晨」；伪造不在场证明。

危险度/规避：高。规避对刀；优先调虎离山。

任务方向/奖励：救人次数；旧案卷宗关键页、祭器照片。

**四阶（连环案核心 · 逼近八年前）**

切入身份/时点：愿重开旧案的协作者，锁定执行逻辑之后。

初始事件：神庙夜，水下倒影与八年前现场重叠。

开场白建议：「香灰落进湖里没有涟漪。张述桐说：他们不是在杀人，是在让岛重新数一遍死人。」

关键NPC立场：**张述桐**（偏执边缘）；**顾秋绵**/**路青怜**（或成诱饵）；祭司相关（不详）；沉默联盟（不详）。

主线钩子/支线：切断一次祭祀节点；救出「补祭」岛民。

危险度/规避：极高。只拆人祸，不与时间本体硬刚。

任务方向/奖励：物证链；可带出岛的录音与手记副本。

**五阶（改写命运 · 时间债）**

切入身份/时点：有限代价分担者，他人格磨损最重时。

初始事件：他认不出某次回溯里的你，却仍记得必须救下的两个名字。

开场白建议：「他问你叫什么。你报出名字的下一秒，潮水声又断一截。你意识到：这一次，死掉的可能是你。」

关键NPC立场：**张述桐**（需被拉回人性锚）；**顾秋绵**；**路青怜**；时间规则回声（只露边）。

主线钩子/支线：在不摧毁心智前提下多线保全；留下给下一次自己的信。

危险度/规避：贴近顶点。超阶：情报优先/条件性胜利；严禁「规则被封印故战力为零」。

任务方向/奖励：命运局部改写；短窗口稳定技术（仍有代价）、双线存活确认（以公开层为准）。
