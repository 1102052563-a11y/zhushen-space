# 天命之上

## 剧情

**【作品来源】**
《天命之上》作者**王小烟**，网文连载作品（公开检索与清单笔记指向全职高手式职业竞技+英雄叙事；字数体量公开讨论约三百万字级，**以实际连载平台书页为准**）。主角**叶修**——**注意：本世界档案按「王小烟《天命之上》」条目写作；若读者熟悉蝴蝶蓝《全职高手》叶修，二者人设有竞技神之共鸣，但乐园世界名以清单「天命之上」为准，不直接等同版权合并。** 文风：职业选手、荣耀式网游竞技、团队与个人荣耀。**若原作连载/完结状态有变，以平台书页为准；档案不编造未核结局细节。**

**【世界定位】**
职业电竞与网游「荣耀」式世界里，**叶修**从神坛跌落再重攀，要的不只是冠军，而是把「天命」从俱乐部资本与舆论手里夺回。舞台：职业联盟、俱乐部、网游区服、全明星与总决赛。一句话：在键盘上改写天命。

**【世界观 · 力量体系】**（竞技战力，非物理毁城）
- **路人玩家/网瘾少年**：操作普通。≈一阶（叙事战力）。  
- **职业新秀/一线队边缘**：有体系训练。≈二阶。  
- **职业主力/全明星级**：联盟顶尖操作与意识。≈三阶。  
- **冠军核心/战术大师**：改变系列赛走势。≈四阶。  
- **一代传奇/联盟规则影响者**：跨赛季统治力。≈五阶。  
- **「天命」叙事顶点·荣耀之巅**：历史级地位。≈六阶。  

物理破坏力全程凡人；「战力」指对局统治力与影响力。  
乐园阶位映射（宁低勿高）：路人≈一；职业边缘≈二；主力/全明星≈三；冠军核心≈四；一代传奇≈五；荣耀之巅≈六。无超阶物理神，禁止写成修仙。

**【地理 · 舞台】**
- **网游区服/第十区类新服**：重返的起点。  
- **网吧兴欣式据点**：草根队组建地。  
- **职业俱乐部大楼**：嘉世类老东家与对手主场。  
- **联盟赛场/全明星周末**：公开表演与暗战。  
- **总决赛舞台**：天命加冕处。

**【世界剧情线】**

**阶段一 · 跌落**  
叶修离开或被离开一线俱乐部核心位置，带着未完成的荣耀与一身争议，回到网游层。他从账号卡、材料、队友三个维度重新积累。

**阶段二 · 组队与挑战赛**  
草根队成型，挑战赛杀回职业联盟。训练、材料副本、对手侦察构成日常；「团队第一」压过个人秀。

**阶段三 · 联盟内战**  
升上联盟后面对老东家、豪门与舆论。系列赛是战争，记者会是第二战场。叶修用经验压年轻人，也用年轻人补自己的体能短板。

**阶段四 · 冠军与天命**  
向总冠军推进。结局方向公开常识为「重夺荣耀、完成未竟」，**细节以原作公开层为准，不编造未核的退役演说全文**。

**阶段细目**  
- 网吧夜：一叶之秋式传说在新人耳中只是「过气」。  
- 材料本：他用意识带菜鸟抢紫装。  
- 挑战赛签表：强敌被他用战术拆成作业。  
- 老东家对决：私怨公开化。  
- 全明星：表演赛里下战书。  
- 记者会：一句话顶掉十篇黑稿。  
- 总决赛局点：团队倒在他一声指挥里又站起来。  
- 散场：冠军杯比账号卡重。

**【主要人物】**
- **叶修（主角·荣耀教科书）**｜性格：嘴欠、烟味、责任感深藏；对荣耀较真到偏执。｜装备·能力：顶尖操作与意识、多职业理解、指挥官。｜人物弧光：被赶下神坛 → 再登巅峰。｜立场关系：兴欣式队友的主心骨；老东家的刺。
- **苏沐橙（核心队友/羁绊）**｜性格：明媚、坚定，枪炮师印象。｜装备·能力：职业级远程火输出。｜人物弧光：从旧日同伴到决赛同侧。｜立场关系：最稳信任。
- **唐柔（新一代锐刺）**｜性格：好胜、直。｜装备·能力：战斗法师向操作。｜人物弧光：菜鸟 → 可托付的主攻。｜立场关系：叶修调教对象。
- **黄少天（对手/友人向）**｜性格：话痨、剑客。｜装备·能力：机会主义者。｜人物弧光：联盟标杆对手。｜立场关系：场上死敌场下损友。
- **喻文州（对手·战术）**｜性格：温和毒计。｜装备·能力：指挥型术士。｜人物弧光：智斗线。｜立场关系：棋逢对手。
- **周泽楷（对手·枪王）**｜性格：少言。｜装备·能力：神级枪系。｜人物弧光：个人能力天花板对照。｜立场关系：对决象征。
- **王杰希（对手·魔术师）**｜性格：冷静。｜装备·能力：魔道学者。｜人物弧光：战术多样性对照。｜立场关系：尊重的敌人。
- **张新杰（对手·精确）**｜性格：规律。｜装备·能力：牧师/战术纪律。｜人物弧光：体系流代表。｜立场关系：系列赛噩梦。
- **陈果（老板/家长）**｜性格：热血、护短。｜装备·能力：运营与资金。｜人物弧光：网吧老板 → 俱乐部老板。｜立场关系：后勤与情感后盾。
- **嘉世相关高层（陶轩等，以原作公开为准）**｜性格：资本逻辑。｜装备·能力：俱乐部权力。｜人物弧光：把叶修推下神坛的推手。｜立场关系：结构性对立。

**【势力图谱】**
- **草根队/兴欣系**：挑战者。  
- **嘉世系老东家**：资源与舆论。  
- **蓝雨/微草/轮回/霸图等豪门**：联盟山峰。  
- **联盟官方与媒体**：规则与叙事权。

**【贵重物品】**
- **账号卡（一叶之秋/君莫笑等公开经典）**：战力与身份。  
- **银武/自制装备**：材料战争结晶。  
- **挑战赛门票/联盟名额**：命运门票。  
- **冠军杯**：天命具象。

**【隐藏剧情 · 伏笔】**
- 「团队第一」与个人传奇的张力。  
- 资本与热爱的长期战争不会因一座冠军结束。  
- 新一代是否复制或背叛叶修的路。

**【大事记时间线】**
- 三连冠时代 → 前史。  
- 离开一线 → 开端。  
- 网游重起与组队 → 前期。  
- 挑战赛回归 → 中期。  
- 联盟夺冠之路 → 后期。  
- 荣耀之巅 → 公开结局方向。

**【叙事基调 · 雷区】**
基调：热血、嘴炮、职业尊重；比赛描写要具体到职业克制与战术，不写「气势赢了」。  
忌：把叶修写成无敌外挂；忌抹掉苏沐橙/唐柔等真名；忌物理超能力。  
最早切入：叶修刚进网吧、还没组齐人的那个夜班。
