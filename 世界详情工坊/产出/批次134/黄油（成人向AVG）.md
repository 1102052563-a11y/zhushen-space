## 剧情

**【作品来源】**
「黄油」是华语玩家对**含成人情节的日式美少女游戏 / 成人向 AVG（Adventure Game）/ 视觉小说（Visual Novel）**的口语总称，对应日语 eroge（エロゲー）、H-game、R18 游戏，英文常称 erotic visual novel / adult bishōjo game。它不是单一 IP，而是 1980 年代起在日本 PC 平台成型、经 EOCS（电脑软件伦理机构）等自律审查、以**选项分歧—多路线—多结局—CG/H 场景**为核心语法的类型总览世界。代表谱系包括：光荣早期《Night Life》《団地妻の誘惑》（1982）、PSK《ロリータ野球拳》、Elf《同級生》（1992）与《ドラゴンナイト》、AliceSoft《ランス》系列、Leaf《雫》《痕》《To Heart》、Key《Kanon》《AIR》《CLANNAD》、Tactics《ONE～輝く季節へ～》、âge《君が望む永遠》《マブラヴ》、TYPE-MOON 早期《Fate/stay night》（初版含 18 禁）、以及拔作向、泣きゲー、鬱ゲー、同人 NScripter/RPG Maker 作品等。平台以 Windows PC 实体盘与 DLsite/DMM 数字版为主；家用机多删减为全年龄。文风跨度极大：从硬核色情拔作到催泪神作、致郁实验、科幻战争，同一「黄油」标签下可完全不同。本档案把该类型写成可切入的**元世界舞台**：契约者进入的是「某部具体作品的剧本宇宙」或其产业侧影，须锚定真名作品与角色，禁止空喊「黄油世界」。

**【世界定位】**
这是一个由**立绘、文本框、BGM、选项、好感度、Flag、CG 回收与 Bad End** 构成的叙事物理宇宙。玩家/主角多为「看不见脸的少年」或限定视角角色；女主角按路线拆分；H 场景是剧情标点或卖点本体（视作品类型）。一句话：你在选项里谈恋爱、破案、救世界或只是收集 CG——但每条路线都可能锁死另一条；契约者是外来变数，不是自动 True End 外挂。

**【世界观 · 力量体系】**
本类型的「力量」不是灵气飞剑，而是**叙事与社会资源**：
1. **文本推进权**：谁控制对话框与场景切换，谁定义「发生了什么」。
2. **好感度 / 参数**：约会地点、礼物、对话选项抬升数值；阈值不足则路线锁死。
3. **Flag / 条件分支**：隐藏条件（某日必须去保健室、必须在第 3 章选「沉默」）决定能否进入个人线。
4. **CG / 场景权**：达成 H 或关键事件=获得「证据/记忆碎片」；全 CG 收集常是通关指标。
5. **多周目知识**：二周目带着记忆改变选项，是类型内合法「外挂」；契约者的跨线记忆若暴露，会被剧本当 bug 或被角色当疯子。
6. **审查与马赛克规则**：日本本土版性器打码；海外无码版与全年龄移植版是平行规则层——同一角色在不同版本可能「从未发生过那一夜」。
7. **品牌/引擎权**：Key 的 NScripter 系泣き节奏、Leaf 的视觉小说标签、AliceSoft 的 RPG+H、Illusion 的 3D 沙盒，各自是不同「功法门派」。

死亡与复活：普通 AVG 里「死」常=Bad End 黑屏重读；部分鬱ゲー/猎奇作（如《沙耶の唄》）死亡是永久与致残式结局。契约者若把「读档」当现实复活，会与角色认知冲突——写戏时优先用「失去路线资格 / 被拉黑 / 社会性死亡」而非无脑满血。

战力阶梯（映射社会/叙事破坏力，宁低勿高；本条目**仅二阶**）：
1. **路人学生 / 无参数男主开局**：被同学欺负、被选项嘲讽、打工被炒。≈一阶边缘。
2. **能稳定推进一条女主线、在社团/班级有话语权、能在冲突中打赢校园级对手（口才/体力/信息）**：≈**二阶**（本世界主舞台）。
3. **S 级人气路线核心、能撬动整部作品 True End 的关键选择者、或品牌级「神作」主角在终局的叙事权**：≈三～四阶观感，但本条目不开放高阶切入——若剧情需要，只作远景压力。
4. **跨作品产业层（发行、审查、盗版、粉丝审判）**：≈超阶边缘，情报优先，不可硬刚。

乐园阶位映射：开局路人≈一阶观感；**稳定推进个人线 / 校园—都市日常冲突中的主角与主要女主 ≈ 二阶**；True End 级叙事权与战争向 eroge 终局（如《マブラヴ オルタネイティヴ》）可临时抬到更高，但本世界收录最高**二阶**，高阶仅作背景。宁低勿高：校园吵架、情敌对峙、家族施压、黑帮边缘威胁——破坏力止于「毁一条关系 / 毁一个班级名声 / 毁一桩婚约」，不要写成屠城。

**【地理 · 舞台】**
类型高频舞台（写具体作品时换成真名地点）：
- **日本私立高中 / 学园**：教室、天台、保健室、社团、放学坡道——《同級生》《To Heart》《Kanon》系标配。
- **地方小镇与车站前商店街**：《AIR》海边、《CLANNAD》坂道与面包店感。
- **都市公寓与电车通勤**：社会人 AVG、人妻线常见。
- **异世界 / 魔王城 / 剑与魔法大陆**：《ランス》系 RPG+H；选项与战斗并存。
- **科幻基地 / 战场**：《マブラヴ》系列；黄油外壳下的战争 AVG。
- **封闭洋馆 / 医院 / 实验设施**：猎奇·鬱ゲー（《沙耶の唄》等）。
- **产业侧影（可作二阶支线）**：秋叶原电器街与同人展、DLsite 数字货架、EOCS 审查会议室、同人志即卖会摊位——契约者可扮演翻译、校对、声优经纪人、地下汉化组，而非只当男主。

舞台口诀：学园=萌与青春；小镇=催泪；异世界=RPG 数值；战场=泣き+战争；洋馆=致郁；秋叶原=产业元叙事。

**【世界剧情线】**

**〇、类型语法：AVG 如何「发生故事」**
标准流程：开场 OP 主题曲 + 立绘剪辑 → 序章建立主角身份与女主群像 → 公共线（common route）提升全员好感 → 分歧点进入个人线 → 高潮事件（告白 / 秘密揭晓 / 灾难）→ H 场景（18 禁版）或情感高潮（全年龄版）→ Good / True / Bad / Normal End → 二周目补全。契约者介入时，AI 必须先判定「当前处于公共线还是个人线、哪条 Flag 已立」。

**一、史前与 1980 年代：从野球拳到团地妻**
1982 年前后，PSK《ロリータ野球拳》、光荣《Night Life》与《団地妻の誘惑》等确立「用电脑玩色情互动」的商业路径；Hudson 等平台亦有野球拳类。早期作品剧情极薄，媒体谴责与「花 8800 日元买简单游戏」的疲劳并存。ASCII《Chaos Angels》等 RPG+H 成功，催生 Elf《ドラゴンナイト》、AliceSoft《ランス》——**战斗与色诱并存**的支脉从此分叉。此阶段「力量」=硬件与发行渠道，不是角色战力。

**二、1990 年代 PC-98 黄金与恋爱模拟发明**
PC-98 / FM Towns 成为 eroge 主战场。1992 年 Elf《同級生》（Dōkyūsei）要求先赢得女主好感再进入成人内容，**恋爱模拟（dating sim）**成型：时间表、地点、约会、情敌，成为后续二十年模板。主角「渡边」类透明男与大量女主（二阶可玩冲突：抢告白日、撞见情敌、期末考与约会冲突）。Leaf 提出 visual novel 标签：《雫》（1996，黑暗）《痕》《To Heart》（1997）——《To Heart》以高中纯爱与神曲级 BGM 把 eroge 推入御宅主流，主题曲进入卡拉 OK。Tactics《ONE～輝く季節へ～》（1998）与 Key 从 Tactics 核心班底成立后的《Kanon》（1999）把「只有少量 H、主打催泪」做成现象级：Satoshi Todome《A History of Eroge》称《Kanon》为年轻御宅的「洗礼」。

**三、2000 年代：泣きゲー、神作与拔作两极**
Key《AIR》《CLANNAD》巩固「催泪游戏（泣きゲー）」；âge《君が望む永遠》以三角与时间线撕裂著名；《マブラヴ》到《マブラヴ オルタネイティヴ》把学园恋爱突然切到异形战争，是类型内「文风欺诈」经典。另一极是**拔作（抜きゲー）**：剧情服务 H，路线短、CG 密。Nitroplus《沙耶の唄》等把猎奇与爱捆绑。TYPE-MOON《月姬》同人起家后《Fate/stay night》初版含 18 禁，后以全年龄与动画扩大——证明「黄油出身」可升级为跨媒体宇宙。海外 MangaGamer、Sekai Project 等开始英译；Steam 多上架全年龄版。

**四、产业机制：审查、品牌、同人与汉化**
EOCS 等自律机构推动「R18 游戏」公式名称；家用机与任天堂/索尼平台长期排斥色情，导致「PC 18 禁 → 主机全年龄移植」双版本。同人圈用 NScripter、吉里吉里、RPG Maker 低门槛量产；顶级品牌垄断货架，小会社要么提质量要么做小众癖好。华语圈「黄油」与「Galgame」混用：严格说 Gal/美少女游戏未必 18 禁，黄油强调成人向。盗版与少子化使 2000 年代后半市场收缩——写产业线可切入「汉化组伦理 / 正版数字版 / 同人即卖会」。

**五、可玩的「元主线」：契约者如何在类型世界里行动**
推荐把「黄油世界」写成**作品宇宙叠加产业宇宙**：
1. **作品内线**：选定一部真名作品（如《To Heart》藤崎诗织线、《Kanon》月宫亚由线、《同級生》公共线某周），契约者以转校生/转学生/转职社员/外聘脚本医生身份入场，目标=改 Bad End 或抢 True End 条件而不毁灭角色弧光。
2. **跨线线**：多周目知识外泄，女主 A 质问你为何知道女主 B 的秘密——这是二阶核心冲突。
3. **产业线**：你为某品牌做翻译校对，发现原稿 Flag 写崩；或在同人展被狂热粉丝围堵「你毁了我的本命」。
4. **伦理线**：拔作与神作同台时，角色会质问「我们是故事还是商品」——可写但不说教。

**六、代表作品剧情锚点（须用真名，供 AI 切换副本）**

*《同級生》（Elf，1992）*  
私立高中夏季：主角在有限日期内通过行动与对话攻略多名女同学与教师等；核心可玩性=时间管理与情报。名场面：文化祭、海边、毕业将至的告白窗口。Bad End=错过季节与人。二阶冲突：情敌截胡、风纪委员盯梢、家庭施压。

*《To Heart》（Leaf，1997）*  
主角藤原拓海（或作品既定男主名以公开资料为准）与女主群的学园日常；**藤崎诗织**等经典青梅/同学线确立「纯爱 eroge」标杆。H 占比低，情感与音乐权重高。二阶冲突：社团活动与考试、朋友变恋人的尴尬、机器人/特殊设定角色线（视版本）。

*《Kanon》（Key，1999）*  
雪之町，**相泽祐一**回到七年前的城市；**月宫亚由**的「加奈陀」口癖与寻觅、**名雪**的巫女与狐、**舞**的剑与伤、**真琴**的活力、**美坂栞**的身体秘密等个人线。公共线后分歧；True 向需要理解「奇迹与谎言」。二阶冲突：失忆与欺凌、家族病、是否相信不可能之事。全年龄与 18 禁版差异：H 可删而主线仍立。

*《AIR》（Key）*  
海边小镇，**国崎往人**与**神尾观铃**、**雾岛佳乃**、**远野美凪**；「翼人之诗」与轮回诅咒。泣き核心：陪伴与失去。二阶冲突：小镇闲言、身体衰弱、是否接受注定的离别。

*《CLANNAD》（Key）*  
**冈崎朋也**与**古河渚**等；学园篇到 After Story 的人生跨度。二阶可玩：光坂高校废部重建、家庭暴力阴影、生子与失去——但高阶人生苦难在本条目只作远景，切入点锚定学园篇二阶冲突。

*《ランス》系列（AliceSoft）*  
**兰斯**以「好色战士」横行异世界，战斗+攻略女角色；政治、战争与 H 并进。二阶切入选早期作或局部城邦任务：救村、决斗、结盟，避免直接写灭国终局。

*《マブラヴ オルタネイティヴ》（âge）*  
学园恋爱表象下的 BETA 异形战争；**白银武**等被扔进平行残酷世界。二阶切入选「尚未揭示战争」的学园周或刚穿越后的适应周：军纪、训练、与**鉴纯夏**等的信任——终焉级战争作背景压力。

*《沙耶の唄》（Nitroplus）*  
**匂坂郁纪**与**沙耶**的畸爱与世界观崩坏。二阶危险在于精神污染与社会隔离，不是枪林弹雨；Bad End 与「幸福结局」的伦理翻转是核心。

*《Fate/stay night》（TYPE-MOON，初版 18 禁）*  
圣杯战争：**卫宫士郎**、**Saber**、**Rin**、**Sakura** 线。二阶切入选战争前夜学园日常或初次召唤后「还不会死人」的窗口；Servant 战力远超二阶，须情报规避。

**七、微观可介入事件（二阶专用，勿复制到其他世界）**
① 文化祭摊位抢位置，风纪委员**点名你的本子 ent 边**。② 二周目你脱口说出只有个人线才知道的绰号，女主报警或冷淡。③ 汉化组要你在「直译 H / 和谐 / 外挂补丁」中选，原作者代理人出现。④ 同人展本命角色 Coser 被跟踪，你选报警或私了。⑤ 情敌用拍照威胁公开你们的秘密约会。⑥ Save 点损坏（叙事化=失忆发作），必须靠角色提醒 Flag。

**八、蝴蝶效应边界**
可改：单条路线结局倾向、某次约会成败、是否解锁隐藏 CG、某个配角是否转学。不可无代价改：整部作品的主题核（如《AIR》的轮回悲、《Kanon》的奇迹结构）、把拔作强行改成无 H 的「净化」而不改版本层、让所有女主同时 True End 且无嫉妒（除非原作明确后宫机制如部分兰斯）。乱改回流：Bad End 增多、角色 OOC、产业线被起诉、粉丝舆论绞杀。

**九、与「全年龄 Gal」「乙女」「BL」边界**
黄油≠所有美少女游戏；乙女/BL 有独立市场。本档案默认**男性向成人 AVG/视觉小说**；若切入乙女/BL 须另锚真名作品，不可混贴。

**【主要人物】**（类型代表：真名来自公开代表作，写作时按当前副本启用）

- **相泽祐一**（《Kanon》男主）｜性格：毒舌懒散，外冷内软。｜装备·能力：多周目记忆（玩家侧）、雪镇生活常识。｜人物弧光：返乡→卷入奇迹→理解 inter 与谎言。｜立场关系：对各女主路线互斥；对契约者「知道太多」会警惕。

- **月宫亚由**（《Kanon》）｜性格：天然、加奈陀口癖、执着寻找。｜装备·能力：奔跑、意外的身体能力。｜人物弧光：寻物→揭示→奇迹。｜立场关系：依赖信任对象；被嘲讽「怪」会受伤。

- **水濑名雪**（《Kanon》）｜性格：温柔巫女气质。｜装备·能力：神社事务、与「狐」相关线。｜人物弧光：守护与秘密。｜立场关系：可成为稳定锚。

- **美坂栞**（《Kanon》）｜性格：努力、身体弱。｜装备·能力：学生会相关日常。｜人物弧光：隐瞒病弱→托付。｜立场关系：需要被认真对待而非同情消费。

- **国崎往人**（《AIR》男主）｜性格：漂泊、表演艺人底色。｜装备·能力：翼人传说知识碎片。｜人物弧光：遇见观铃→面对轮回。｜立场关系：对「再来一次」的渴望可被契约者利用或治愈。

- **神尾观铃**（《AIR》）｜性格：开朗外表下的孤独。｜装备·能力：想象力、小镇孩子的行动力。｜人物弧光：夏天→离别。｜立场关系：核心泣点；不可当工具人。

- **冈崎朋也**（《CLANNAD》男主）｜性格：不良外表、怕认真。｜装备·能力：球棒记忆、废部重建号召力。｜人物弧光：学园→家庭。｜立场关系：对强行说教反感。

- **古河渚**（《CLANNAD》）｜性格：软弱但坚持梦想。｜装备·能力：团子家族、话剧。｜人物弧光：建社→人生。｜立场关系：需要同伴而非救世主姿态。

- **藤原拓海**（《To Heart》男主，公开资料口径）｜性格：普通、被推着成长。｜装备·能力：学园人际关系。｜人物弧光：日常→恋爱。｜立场关系：多女主友好竞争。

**【隐藏剧情 · 伏笔】**
「黄油」标签下藏着市场与创作自由的百年拉锯；神作与拔作共享技术栈却互相鄙视；多周目是否算「对角色不忠」；汉化与盗版是否杀死正版；全年龄删除 H 后主题是否仍成立（《Kanon》证明可以）；战争 AVG 是否「骗恋爱粉」；猎奇作是否越界。契约者最大伏笔：你是玩家还是角色——一旦角色意识到「选项」，四壁会崩。

- 1996–1997 → Leaf 视觉小说与《To Heart》  
- 1998–1999 → 《ONE》与 Key《Kanon》泣き浪潮  
- 2000s → AIR/CLANNAD、âge 战争向、TYPE-MOON 跨媒体

- 2010s– → 数字发行、Steam 全年龄、同人与商业并行  
- 现在时 → 契约者可切入任一经典剧本的「公共线第 1–2 周」或产业展会周  

**【叙事基调 · 雷区】**
基调：按副本切换——纯爱甜、泣き催泪、拔作直白、猎奇压抑、战争残酷；禁止用同一套「修仙套话」覆盖。忌：无同意、未成年性描写（角色年龄与分级必须按原作与当地法；写档案时用「高中生设定的恋爱 AVG」并避开违法细节，H 只写到「发生路线事件」层面的剧情功能）；忌把所有女主写成无性格 CG；忌读档玩梗毁掉角色痛苦；忌把 Key 神作写成只有卖肉。
**十六、版本差分与「平行自我」**
同一角色在 18 禁 PC 版、全年龄主机版、动画改编版中可能拥有三套记忆。契约者若混用版本知识，会被当成说谎者。写二阶案件时可做：角色否认某段亲密，你必须用「全年龄也成立的共同经历」（一起补习、一起躲雨、一起守夜）重建信任。动画删减的政治或黑暗情节（部分战争 AVG）只能作传闻，不可当已发生事实强加给未看过该线的角色。

**十七、结局类型辞典（供任务成功条件）**
- Good End：关系成立、主题正向收束。  
- True End：隐藏条件满足，常为作者意图终点。  
- Normal End：半吊子收场，可作「未完成任务」。  
- Bad End：死亡、离别、精神崩溃、被遗忘。  
- 后宫/多线同时：仅部分作品机制允许；强行套用=OOC。  
任务简报应写清要哪一种 End，避免 AI 默认推倒全员。

最早切入：公共线开学周 / 返乡第一场雪 / 夏日祭前一周 / 同人展前夜。


**十、公共线运作细则（AI 写日常必读）**
公共线不是注水，而是「全员好感的资源分配战场」。每天通常只有有限行动点：放学后去天台见 A，就不能去车站接 B；周末约会占用整天。礼物有贵贱与雷区——送错礼物直接扣好感甚至锁线。对话选项常有「表面正确但减好感」的陷阱：对泣き向角色说「想开点」可能比沉默更糟。契约者若拥有多周目知识，正确用法是「提前出现在关键地点」，而不是「当众背出台词」。写二阶冲突优先：时间不够、钱不够、面子不够、情报被情敌截获——而不是突然出现魔王。

**十一、个人线高潮的因果链模板**
好的个人线高潮必须有「秘密 → 选择 → 代价 → 关系质变」。例：《Kanon》亚由线的寻找对象与「奇迹是否被允许」；《AIR》观铃线的夏天长度与接受离别；《CLANNAD》渚线的建社成功与家庭阴影并行；《同級生》则是夏季结束前的告白窗口与毕业分流。H 场景在 18 禁版中常作为关系确认的标点：写档案时用「路线事件发生、双方关系进入不可逆」表述功能，不写操作性细节。Bad End 要具体：被拉黑、转学、住院、角色死亡（部分作）、或「你赢了身体输了心」。True End 往往要求二周目知识或隐藏 Flag，这是类型对「认真玩」的奖赏。

**十二、拔作、神作、鬱ゲー的任务设计差**
拔作副本：目标可写成「在时限内收集指定 CG 并确保至少一名角色不陷入强制黑暗结局」；冲突来自时间表与癖好路线互斥。神作副本：目标是情感正义与主题核，奖励是关系与音乐记忆，不是装备爆率。鬱ゲー副本：成功条件可能是「理解并选择某种残酷真相」，失败是精神崩溃式 BE；危险度写精神污染。战争 AVG 副本：二阶只允许训练周/适应周，任务是活过通报与体罚、与队友建立最低信任。RPG+H（兰斯系）：二阶用「清一只哥布林、救一个村庄、在王都接委托」级任务，禁止直接开大陆终局。

**十三、产业侧可玩长线（补足类型总览厚度）**
契约者可扮演：① 汉化组校对，发现某条 Flag 在日文原稿是 Bug，修还是保留「原汁原味」；② 声优经纪人，协调 H 场景配音伦理与档期；③ 同人作者，被正方发警告函；④ 实体店店员，未成年试图购入 R18 盘时的执法边缘；⑤ 海外发行对接，Steam 页面被举报下架，需在 48 小时内提交分级材料。这些线把「黄油」从卧室屏幕拉到秋叶原与会议室，提供非恋爱的二阶冲突：合同、钱、名誉、法律。

**十四、名场面速查（跨作，供随机事件池）**
- 天台风太大吹走情书，两人同时去捡手碰手。  
- 保健室只剩一张床，选项决定谁躺下谁守门。  
- 夏日祭走散，烟火升起时你选找谁。  
- 考试前夜家教，家长突然回家。  
- 文化祭女装/女仆装强派，好感与社死并存。  
- 车站末班车，说不说「喜欢」决定整条线。  
- 二周目你先女主一步出现在事故点，被问「你怎么知道」。  
- 存档损坏（发烧失忆），角色用你们的共同回忆考你。  
- 同人展本命立牌被砸，粉圈内战。  
- 全年龄移植删除关键夜后，角色坚持「我们没做过」，你必须用非性证据证明羁绊。

**十五、契约者外挂与世界反制**
允许的外挂：模糊的既视感、对 BGM 情绪的敏感、对「这是 BE 前兆」的不安。高危外挂：完整攻略背板、直接读心、强制修改好感数值。世界反制手段：角色 OOC 报警、路线强制锁死、产业线账号封禁、猎奇作精神污染、战争作提前征召。写「胜利」时用：指定角色 Good End 达成、关键配角存活、自己的身份没被拆穿为「玩家」——而不是「推倒全员」。

## 阶位切入点

> 阶位↔：二阶≈能稳定推进一条个人线、在学园/小镇/局部任务中拥有话语权与冲突胜负的主角及主要女主。更高叙事权（True End 神、Servant 战争、BETA 战争）为远景；产业审查与舆论为**超阶边缘**，情报优先。本世界仅开放**二阶**。

**一阶（公共线开端 · 看懂选项前先活下去）**

切入身份/时点：契约者以刚转入学校的学生、商店街短期打工者或刚加入同人社团的新人身份切入，锚定《Kanon》相泽祐一返乡首日、《CLANNAD》冈崎朋也仍未参与社团活动，或《同級生》暑假前公共线的最初数日。

初始事件：放学铃响后，你在车站出口捡到一封没有署名的情书，信中只写了约会地点与时间；**相泽祐一**或**冈崎朋也**正被误会成寄信者，而**月宫亚由**或**古河渚**恰好看见你把信收进包里。交给老师会让秘密公开，私下找人则会错过当天唯一能建立关系的行动时段。

开场白建议：「放学后的站台比教室安静。风把一封没写名字的信吹到你鞋边，墨迹还没干。远处有人叫住你，近处的列车已经进站；你忽然明白，所谓选项不是发光的按钮，而是下一分钟你会不会被谁记住。」

关键NPC立场：**相泽祐一**嘴上刻薄却会帮被误会的人，讨厌外人拿女主的秘密取乐；**月宫亚由**愿意信任主动伸手的人，却不接受被当成攻略目标；**冈崎朋也**对学校规矩抱有抵触，仍会在真正需要时出面；**古河渚**想重建话剧部，害怕好意只是一时兴起；**藤原拓也**（《To Heart》主人公）处于平凡日常的中心，外来者若扰乱同学关系会先被他观察。

主线钩子/支线：主线是交还情书并避免在公共线过早锁死关系；支线可替渚跑一趟社团申请、陪亚由寻找遗失物、在《同級生》的放学日程中替同学顶一次值日换得情报。

危险度/规避：低，真正风险是误会、被贴上跟踪或说谎标签、错过不可重来的日程；规避暴露多周目知识、替任何角色强行决定恋爱关系，也规避涉及未成年角色的成人内容。

任务方向/奖励：目标是获得公共线的可靠立足点；奖励为一张可核对的日程表、社团临时通行、能开启后续对话的共同回忆，以及一名角色对你「不会乱说秘密」的信任。

**二阶（公共线—个人线门槛 · 选项即刀）**

切入身份/时点：契约者以「转校生 / 返乡亲戚 / 转职到镇上的新人 / 品牌社实习脚本 / 同人展临时翻译」之一入场；锚定任选一部真名作品的**公共线第 1–3 周**（推荐默认：《Kanon》返乡首周、《To Heart》学期初、《同級生》夏季开始、《CLANNAD》废部传闻期）。禁止开局即终局战争或圣杯决战。

初始事件：你在天台/车站/教室门口撞见本命女主线角色（如**月宫亚由**追着什么跑过，或**藤崎诗织**点名你仪容），对话框弹出三选项——其中一项会立下错误 Flag（例如当众拆穿秘密）。同时你口袋里的「二周目笔记」（契约者外挂）被风吹掉半页，被**风纪委员或情敌**捡到：上面写着「第 12 日不要选安慰」。对方微笑问你：这是诅咒还是剧本？

开场白建议：「BGM 淡入的时候，雪（或蝉）刚好落在校服领口。你听见自己的心跳被音效放大，像 ADV 里那一下『心跳 SE』。亚由——或不，是诗织，或是渚——看着你，等你开口。三个选项在视网膜上燃烧：①随便应付 ②认真倾听 ③把多周目的真相说出去。你知道第三项会毁线，但你的舌头比手指快——而风纪委员已经在走廊尽头举起了你的半页笔记。」

关键NPC立场：  
- **相泽祐一 / 冈崎朋也 / 藤原拓海**（视副本男主）：把你当抢路线的情敌或莫名其妙的全知者。  
- **月宫亚由 / 藤崎诗织 / 古河渚 / 神尾观铃**（视本命线）：好感可涨可崩；讨厌被「剧透式关心」。  
- **兰斯**（若异世界副本）：视你为障碍或抱团对象，随时可能 H 指向错误目标。  
- **白银武**（若学园周 Muv-Luv）：还不知道战争，对「你太悲观」不耐烦。  
- **匂坂郁纪 / 沙耶**（若猎奇副本）：极度危险，二阶建议只做外围调查。  
- **卫宫士郎 / Saber**（若 Fate 前夜）：Saber 战力超阶，只可委托跑腿与情报。  

主线钩子/支线：  
- 主线——在不毁灭角色弧光的前提下，把当前女主线从 Bad End 走廊推到 Good/True 门槛（例如正确处理《Kanon》某日的「是否相信」；《同級生》某日的约会优先级）。  
- 支线 A——帮社团抢文化祭位置，换取情报 Flag。  
- 支线 B——追回被盗的原画 U 盘（产业线），发现里面是未公开 Bad End 脚本。  
- 支线 C——情敌用照片勒索，你选报警、交换或反将一军。  
- 支线 D——全年龄版与 18 禁版记忆错位：角色否认「那一夜」，你必须用非 H 证据重建信任。

危险度/规避：**中**（学园纯爱）到**中高**（猎奇/战争前兆）。规避：对未成年角色写性描写细节；强制后宫破坏单线主题；对 Saber/战术机/BETA 硬刚；公开剧透全员真名秘密。超阶：审查机构与网络舆论绞杀——**存在·情报优先**，条件性胜利=撤稿道歉/改档过审，不是打爆。

任务方向/奖励：目标=指定路线 Good/True 门槛 + 至少保护一名配角免于社会性死亡；奖励=「稳定好感信物」、二周目笔记补全页、社团钥匙、汉化/翻译合同、限定 CG（剧情功能：证据）、与本命女主的信任 Flag。不许发 Servant 契约完全体、兰斯后期军势、或「全女主同时爱你无代价」系统。


**二阶补充·学园纯爱默认包（《Kanon》/《To Heart》口径）**
