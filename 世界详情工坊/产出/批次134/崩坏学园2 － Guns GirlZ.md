## 剧情

**【作品来源】**
《崩坏学园2》（日文：崩壊学園2；英文：Houkai Gakuen 2 / Guns GirlZ）是米哈游开发运营的 2D 横版射击手游，崩坏系列早期正统续作（前作《崩坏学园》/「崩坏1」）。中国大陆 2014 年 1 月 26 日公测，后登陆日/韩/台等；制作人历蔡浩宇、曹霞、蒋大卫。引擎 Unity；平台 iOS / Android。海外曾名「Guns Girl - School DayZ」。文风：美少女枪战 + 崩坏末日 + 校园日常；主线分归零纪元、追溯篇、新生篇等大周期，后期扩至军团纪元与「崩坏学园」校政。关联：《崩坏3》沿用部分角色名但世界观重构；与《崩坏：星穹铁道》同属米哈游崩坏宇宙外围互文。本档案服务 AI 常青读取，忠于公开剧情与角色设定，查不到写不详。

**【世界定位】**
「崩坏」周期性降临，摧毁主宰物种；被选中者可利用规则扭曲获得超常力量。主线以**琪亚娜·卡斯兰娜**等女武神/圣痕使为轴，从千羽学园崩坏、寻父与对抗律者，到圣痕计划周期循环、终焉威胁，再到军团纪元后以「崩坏学园」庇护圣痕使的新秩序。一句话：少女们用枪与崩坏能在学园废墟与文明废土上活下去，并试图改写崩坏周期——契约者是周期缝隙里的变数，不是免死金牌。

**【世界观 · 力量体系】**
力量本源：**崩坏**——周期性灾难现象，可表现为战争、疫病、气候灾变、彗星撞击乃至巨神兵；伴随世界规则扭曲。学会利用扭曲者可获超常识能力。**圣痕 / 圣痕使**：圣痕计划与圣痕终端相关，使人类在崩坏下获得构型/能力（金属、冰、火、风暴、血肉等）。**女武神 / 瓦尔基里**：天命等组织训练的对崩坏战士，分 B/A/S 级。**律者**：崩坏降下的权能个体（如空之律者西琳）。**Nexus / 装甲 / 枪械模组**：横版射击的装备化战力。**量子化 / 人格抹除 / 脑机接口**：逆熵实验副作用（布洛妮娅、希儿线）。死亡：常人战死即死；部分存在以量子态、克隆、圣痕残响例外；周期重置是文明级机制，非个人满血复活。

战力阶梯（破坏力，宁低勿高；本条目覆盖一～七阶）：
1. **普通人 / 无武装学生**：徒手格斗、冷兵器；街区枪战即极限。≈**一阶**。
2. **民兵 / 低适应者 / 普通崩坏兽**：小队清剿、碎砖裂石。≈**一～二阶**。
3. **B 级女武神 / 学园战术生 / 中阶崩坏兽**：据点战、裂石断甲、端山寨级据点。≈**二～三阶**。
4. **A 级女武神 / 精英圣痕使 / 装甲操作手**：建筑群级破坏、硬抗常规热武器。≈**四阶**。
5. **S 级女武神 / 伪律者 / 皇帝级崩坏兽**：街区至城市灾害、领域雏形。≈**五阶**。
6. **完全觉醒律者残响 / 人造律者小队 / 英灵殿顶尖 / 军团长级圣痕使**：地区至国家级、领域成型。≈**六阶**。
7. **完全展开律者战役层 / 终焉前夜联合作战层 / 行星级崩坏兽逼近时的「文明存续」压力层**：一怒可毁国级叙事压力。≈**七阶**。
8. **崩坏意志 / 周期机制本体 / 圣痕终端完全战争层**：**超阶**——存在·情报优先，本条目不开放八阶切入。

乐园阶位映射：普通人～低适应者≈一阶；B 级女武神/中阶崩坏兽≈二～三阶；A 级≈四阶；S 级/伪律者≈五阶；完全觉醒律者/军团顶尖≈六阶；律者完全展开·终焉前夜联合作战≈**七阶**；崩坏意志/周期本体=**超阶**。宁低勿高：单人 S 级女武神仍做不到「一拳打爆地球」，需要终端、联军与代价；七阶写「文明级压力与毁国叙事」，禁止写成爆星。

**【装备与战斗表现（横版规则的叙事化）】**
角色通过武器模组、圣痕技能、角色切换形成输出循环；叙事上对应「弹药种类、崩坏能过载、换人喘息、Boss 破防窗口」。女武神评级决定任务权限：B 级清外围，A 级攻坚，S 级对律者/高阶崩坏兽。Nexus 类系统用生命换火力，适合写「打完这枪可能站不起来」。量子化单位难以被常规子弹消灭，需要共鸣或指定救赎条件（希儿线）。机械与崩坏兽混编时，优先切断指挥核心（可可利亚遥控、军团通讯、律者领域源）而非逐个清潮。

**【地理 · 舞台】**
- **千羽学园 / 长空市**：早期崩坏爆发与校园横版主舞台；教室、天台、地下铁、废商业街是清怪与疏散双重场景。天鹅酒店等据点在军团纪元仍以「大师女武神」驻守叙事出现。
- **圣芙蕾雅学园**：女武神培训与第五师团前线（姬子体系）；宿舍日常与出任务的落差是文风关键。
- **天命总部与欧陆网络**：主教奥托、阿波卡利斯/沙尼亚特/卡斯兰娜三大家族政治、人造律者实验室、神权小队驻地。
- **逆熵北美 / 实验室 / 可可利亚孤儿院**：科学伦理黑暗面；布洛妮娅、希儿、杏·玛尔出身地。
- **神州 / 太虚 / 东方战线**：符华、司徒箐泓、巫静妃等；传统武术与人合联神州理事政治。
- **军团纪元废土**：第一军团残部活动带（赫曼尼亚周边）、南方新长空市（斯泰基列车/铁之手/侠胆会）、东方极乐都与太阳系资产、死区游击带。
- **崩坏学园**：蓬莱寺九霄主权领土式校园；战术科靶场、建设科工坊、网球社、宿舍与「庇护边界」——边界外即无三年保护。
- **四大传送都市**：关税来源，学园财政命脉；切断关税=学园停摆。
- **归零纪元末世城邦 / 超电社**：原初人类合约联盟与圣痕计划起点，闪回与终端深层可进。
- **量子态徘徊域 / 圣痕空间**：希儿等消失后的非欧几里得「等待室」。

地理口诀：校园=萌与死并列；总部=政治；孤儿院=伦理；废土=军阀；学园=庇护契约；终端=高阶棋盘。

**【世界剧情线】**

**〇、崩坏是什么：周期、扭曲与被选中者**
世界每隔数百年迎来一次「崩坏」：形式不可预测——战争、传染病、气候灾难、彗星、巨神兵皆可能。结果总是主宰物种大量毁灭。典籍称「神之意志」，研究者则追踪周期规律。崩坏扭曲世界规则：一面毁灭，一面让适应者获得超常识力量。游戏玩家/契约者视角的「被选中者」即深谙此道之人。横版关卡里的枪火与模组，是这套规则的操作界面；主线则要求记住：**清图不等于周期结束**。

**一、归零纪元（原初时代 / 第零周目）**
崩坏把人类生存范围压缩到极限，国与家消亡。残存者建立**人类合约联盟**，聚于最后城邦。原初末期文明高度发展，地球崩坏威胁下降，却迎来外太空**行星级崩坏兽**。超电社以 **MEI 博士 / 原初雷电芽衣** 等为核心推进科技；**圣痕计划**被启用：以圣痕与终端保存火种，换取文明在毁灭后仍能重启。原初**琪亚娜**以天启教会圣女与超电社女武神身份作战；**无量塔姬子**、**无色辉火**同为女武神；**奥托**任天启主教；**凯文·卡斯兰娜**等战力顶点登场；**菲米莉丝**作为伊瑟琳系顶级 AI 具备情感模块。**迦娜**受崩坏意志引导降临：毁灭原初文明，却也与琪亚娜约定维护圣痕终端——高阶战役的「既是灾厄也是守门人」。归零纪元以圣痕计划启动与漫长抗崩坏战争作结，火种埋入时间。

**二、追溯篇（约 +42000 年 / 第一周目）**
圣痕计划支持下文明再快速发展。普通人不知崩坏；**天命**与**逆熵**在幕后争夺解释权与实验体。**琪亚娜·卡斯兰娜**（第 69 世）：十二岁随父至极东调查疑似崩坏前兆的「地铁沙林恐怖事件」；十四岁父授家学奥义后只留一句「我去探寻崩坏的本源」消失。琪亚娜寻父路上卷入千羽学园/长空市崩坏：横版关卡中的教室、天台、地下铁成为战场。结识**雷电芽衣**——十五岁家变后性格封闭的前千金；**布洛妮娅**——俄罗斯重工业养女，次世代人形兵器试验体，人格被抹以兼容脑机；**德丽莎**——天命极东支部指挥官，外表幼年、指挥成熟，与八重樱邂逅获奇妙能力；**姬子**——圣芙蕾雅/第五师团少佐，绝对前线指挥。逆熵侧**可可利亚**以孤儿院收容希儿、杏·玛尔等，实验制造「可用的孩子」。该纪元在**终焉律者**破坏下再度毁灭——玩家第一次被明确告知：个人英雄主义挡不住周期终焉。

**三、新生篇（再 +42000 年 / 第二周目）**
圣痕计划干涉使世界「微妙变化」。琪亚娜以 **B 级女武神**再开篇，父母线在部分口径中呈现为**塞西莉亚·沙尼亚特**与**齐格飞·卡斯兰娜**的圣女—骑士结构。人类逐渐知道崩坏真相。

**三·a、第二次大崩坏与西琳**
**西琳**觉醒为空之律者，因目睹世界丑陋暴走，史称第二次大崩坏。塞西莉亚、瓦尔特、天命女武神联军苦战；西琳后陷入沉睡。此战奠定「律者=文明级灾难」的高阶标尺，也埋下西琳与琪亚娜的镜像关系（后续学园时代西琳以辅导员身份回归的荒诞与沉重并存）。

**三·b、奥托与人造律者**
**奥托**统治天命约五百年，推动人造律者计划：复活**卡莲·卡斯兰娜**为无记忆的女武神并任「神权」队长；**芙洛拉、赛露蒂、爱莎·洛佩兹**等神权成员；**黛尔塔**可吞噬控制崩坏能；**宁蒂**为史上最年幼 S 级、对律者特攻；**无色辉火**以 Nexus 耗命换输出。政治上沙尼亚特圣女位、卡斯兰娜骑士位、阿波卡利斯主教位三角撕扯。契约者若介入，任务是「截一份实验名单」而非「当场斩首奥托」。

**三·c、逆熵执行者与希儿线**
**瓦尔特**为逆熵司令官（第一律者系）；执行者含军火商可可利亚、企业家雷电龙马、博士伊瑟琳、旅人。布洛妮娅小队（希儿、杏）执行残酷任务；希儿在失败实验中量子化消失，后因对姐姐的思念被布洛妮娅救赎苏醒——横版射击外壳下的量子亲情核。**耀夜姬**等合成人「只能活十五天」的遗弃伦理提供支线深度。

**三·d、新生篇终局方向**
公开叙事：人类逐渐理解崩坏真相，与其进行最后决战，将世界从崩坏意志威胁中解放。七阶正文建议锚定决战前夜：终端过载、律者残响、行星崩坏兽阴影同时在线，胜利必须昂贵。

**四、人合联、英灵殿与过渡**
为取代天命，各国建**人类合约联盟**（后约 2019 解体为人类解放联盟与世界合约联盟等口径）。梅琳·卡文迪许、丽塔、托托莉、法贝菈、司徒箐泓等构成过渡期群像。天命灭亡后改组**英灵殿**，以情报买卖与雇佣兵维生——「根除崩坏」理想堕落为生意。

**五、军团纪元：统一与内战**
约军团纪元 10 年（公元 2027 口径）建**世界建设兵团**，第一军团为中央。苏尔特创建、克里斯玛·桑德兰发动分裂内战（约纪元 38）。**南方军团**军力最强却思想保守，纪元 40 击败第一军团后再内战，裂为战争仲裁委员会（三头同盟）与大陆联合战线（南陆联合）。**东方军团**经济最强、远离地球坐镇太阳系资产，内战渔利。第一军团残部：不屈师团、死区师团、启蒙师团、灭绝师团等在赫曼尼亚等地抵抗。新长空市出现斯泰基列车物流、铁之手建筑、侠胆会娱乐等灰色政权。**拉维妮娅·阿维尼翁**被广泛认为世界最强圣痕使之一；戈尔迪乌斯白金蝶家族、巫树家族、杜鹃佣兵团等提供骑士—军阀混搭政治。七阶任务可写「停火监督被双方当枪使」。

**六、崩坏学园：庇护所政治**
**蓬莱寺九霄**于军团纪元 40 年（公元 2057 口径）建**崩坏学园**：征四大传送都市关税，圣痕使入学获三年庇护。战术科院长德丽莎、建设科院长伊瑟琳、辅导员西琳、地脉首领布洛妮娅等「前救世主」变教师。新生代主轴：**卡琳·梅莉德**（七不管地区快递少女，无限圣痕）、**阿库莉亚·赫萝普特**（被畏为魔女，黑暗之心契约）、**拉妮艾特**（南方名门出走，双链谱系）、**蒂亚拉·星灯**（网络投机，天下一人）、**煌嵐**（风暴构型黑帮大小姐）、**红理惠**（血肉圣痕医学）、**洛茵·克里曼**（化名实习净化委员会）等。主题从「打终焉」转为「庇护所里如何做人、如何不被外部战争征召为炮灰」。快递穿越封锁、网球社路人、建设科全科零分，皆可写。

**七、组织暗战与资源逻辑**
天命要神权解释；逆熵要科学新秩序；人合联要国家垄断暴力；兵团要统一与继业；学园要关税与庇护配额；英灵殿要情报溢价。写冲突优先：实验体、圣痕配额、传送关税、停火线、入学名单——而非无脑清图。

**八、微观可介入**
入学审核、实弹考核、Nexus 透支换甜食、姬子酒后指挥、德丽莎外表与权威反差、杏的残酷、希儿呼唤、量子档案、快递单上的假名、预言偏差的九霄、东方偶像公演下的暗杀。

**九、蝴蝶效应边界**
可改：单次清剿、学员生死、局部战役、实验体提前救出、关税谈判细节。不可无代价改：崩坏周期、终焉/行星级顶点、琪亚娜寻父与律者情感核、圣痕计划历史地位、学园「三年庇护」制度骨架。乱改回流：更快终焉、庇护取消、奥托清洗、兵团屠城学园。

**十、与崩坏3 / 星穹铁道的边界**
同名角色（琪亚娜、芽衣、布洛妮娅、德丽莎、姬子、符华、西琳、瓦尔特等）**经历与结局不可直接等同崩坏3**；瓦尔特离地等互文仅作彩蛋。禁止「用崩坏3终焉律者琪亚娜设定覆盖学园2西琳线」。

**十一、追溯篇关键战役读法（供 AI 写名场面）**
千羽学园崩坏夜：警报、封锁门、同学异化/崩坏兽化、琪亚娜第一次必须开枪救人还是开枪杀「还像人的东西」。地铁沙林调查的后遗症以「父辈阴影」形式出现在对话，而非必须复刻全案。圣芙蕾雅出任务：姬子远程指挥、德丽莎下达「可撤退」却没人真撤的讽刺。逆熵突袭或联手的临时同盟极脆：共同敌人消失后立刻回到实验体争夺。终焉逼近时天空异象、终端过载读数、普通人祈祷与女武神弹药告罄同时发生——七阶要写「系统崩溃感」。

**十二、新生篇政治与律者读法**
奥托的「为了卡莲」使一切实验获得自我合法化；契约者若想道德说服他，会失败；若想交易，必须提供「更接近卡莲理想世界」的筹码。西琳沉睡不是永久删除，而是定时炸弹；学园时代让她当辅导员，是九霄式黑色幽默：让灾厄教孩子如何活。塞西莉亚之死/重伤类名场面（以公开版本为准）应保持重量，不可用「其实没死」轻易消解，除非版本明确复活线。瓦尔特作为逆熵理性象征，与天命信仰战是「科学能否替代神权」。

**十三、军团内战微观政治**
南方骑士家族用纹章与决斗规则包装现代火炮；东方用娱乐产业与制药资本软控制；第一军团残部用「正统」话术招募圣痕使。崩坏学园的三年庇护成为稀缺商品：军阀送人质入学、赏金猎人送货夹带情报、净化委员会实习生双重效忠。卡琳的快递路线=情报网；阿库莉亚的「魔女」标签=公众恐慌武器；拉妮艾特的出走=家族继承权危机。高阶介入兵团时，真正危险往往是「停火协议上的签字笔」而非前线一枪。

**十四、学园日常为何必须写**
没有宿舍拌嘴、考核分数、网球社、建设科零分，高阶终焉会失去「要保护的日常」。九霄预言偏差制造喜剧，也制造误判危机。德丽莎的外表让敌人轻敌，让学生敢顶嘴——直到她下令封锁校门。布洛妮娅开孤儿院的战后身份，是「银狼」放下枪后的另一种守护。写学园：钥匙串、名牌、食堂窗口、庇护到期日倒计时，比写「S 级光柱」更像这个世界。

**十五、各阶任务样例因果链**
例 A（低阶）：千羽学园封锁门内侧，同学异化，你必须在「开枪 / 拖走 / 上报」中选，选错会害死走廊里的芽衣或琪亚娜。例 B（中阶）：可可利亚孤儿院名单被掉包，你护送的「实验体」其实是诱饵，杏会先对你开枪。例 C（高阶）：西琳领域边缘出现，学园要在两小时内完成低年级疏散；契约者可选择强化领域裂缝（伤己）或放弃外围街区（伤民）。例 D（七阶）：迦娜逼近终端，旧契约要求「琪亚娜系血脉对话」；或南方军征用学园作炮兵阵地，九霄要关税主权，德丽莎要学生命——必须在「开战/交人/交钱」外找到第四条路。每条路都要回流：救了人可能缺弹药，守了主权可能缺盟友。

初始事件：走廊警报尖叫，封锁门落下；同班同学眼白翻黑、皮肤裂出崩坏兽纹路。你手里只有消防斧或一把没上膛的练习枪——必须在「拖伤员进厕所反锁 / 跟琪亚娜枪口方向跑 / 冲去天台求救」里选一个，且每选都会听见有人叫你名字却来不及回头。

开场白建议：「广播还在播课间操，走廊已经开始喷血。你看见琪亚娜把弹匣拍进枪膛，芽衣用一把断掉的栏杆挡在身前。有人喊你的学号，声音在第三声时变成了非人的嘶鸣——你第一次明白，这个世界的『崩坏』不是网游词，是你同学脸上裂开的那条缝。」

关键NPC立场：**琪亚娜·卡斯兰娜**会把你当需要护送的累赘但不会弃你；**雷电芽衣**要秩序，讨厌你乱跑；**无量塔姬子**频道里只给坐标不给安慰；**德丽莎·阿波卡利斯**童声念撤退点，末尾加「来不及的不要回头」。

主线钩子/支线：主线——活着撤到指定集合点并护送至少一名未异化学生。支线——找回琪亚娜掉落的家学笔记残页；帮芽衣确认父亲企业旧案档案是否被崩坏淹没；给姬子传一条「B 区已失守」的手写字条。

危险度/规避：**中高**。规避：单独硬刚异化潮；对「还像人的东西」无脑开枪或无脑救人导致全灭。超阶存在本阶不可见。

任务方向/奖励：目标=生存+情报；奖励=入门枪械模组、临时女武神观察员胸章、圣芙蕾雅一日参观券、关于「地铁沙林」的传闻碎片。不许发 S 级装备。

**二阶（低适应者·外围清剿 · 第一次开枪）**

切入身份/时点：契约者以民兵、天命外围辅助、或逆熵临时雇佣入场；锚定崩坏夜后**长空市外围清剿**或圣芙蕾雅外围据点巩固期。

初始事件：你被编入三人小队清商场负一层；通讯里姬子说「十分钟内出图」。中途发现「崩坏兽」其实是被实验改造的逆熵遗弃体——杏·玛尔的小队在另一侧同时清场，她先对你开枪再问口令。

开场白建议：「负一层的喷淋系统还在空转，水里漂着学生证。布洛妮娅的电子音从频道里平平地问：『你是友军吗？回答超时将被视为目标。』你看着准星里那张还像孩子的脸，第一次学会在三秒内报出正确代号。」

关键NPC立场：**布洛妮娅·扎伊切克**要数据真实与任务完成；**杏·玛尔**残酷优先；**希儿·芙乐艾**若在场会成为你「不能开枪」的软肋；**可可利亚**远程视你为耗材。

主线钩子/支线：主线——完成外围清剿并带回一份逆熵实验编号。支线——救出孤儿院走失的孩子；截获军火商货单；与布洛妮娅交换「希儿相关」情报换活命。

危险度/规避：**中**。规避：对可可利亚当面道德审判；抢杏的击杀数。

任务方向/奖励：目标=清剿+编号；奖励=B 级模组零件、崩坏能抑制剂、孤儿院通行证复印件。

**三阶（B 级女武神·据点战 · 权限门）**

切入身份/时点：契约者以 B 级女武神、圣芙蕾雅战术生、或天命极东编外入场；锚定琪亚娜 **B 级女武神再起**后的据点攻坚/护送任务。

初始事件：德丽莎下令「可撤退」但没人真撤；你的小队要端掉一座被崩坏兽占的地铁站。站内发现塞西莉亚时代留下的圣女纹章——奥托的人要回收，逆熵的人要炸毁，你必须在「交纹章 / 藏纹章 / 伪造」中选。

开场白建议：「德丽莎站在指挥车顶，看起来像逃课的小学生，直到她念出你的编制号。『B 级权限，只能进 A 区外围。』她眨眨眼，『如果你非要进去，记得先写遗书——写给我，我会寄给还在意你的人。』」

关键NPC立场：**德丽莎·阿波卡利斯**要指挥链；**琪亚娜**要同伴；**塞西莉亚·沙尼亚特**（若以回忆/影像出现）要圣女责任；**奥托**的代理人要纹章。

主线钩子/支线：主线——攻下地铁据点并决定纹章归属。支线——帮姬子补酒与弹药；为符华带路东方崩坏迹象；拒绝或接受神权小队的「升编面试」。

危险度/规避：**中高**。规避：假传军令；把圣女纹章当街叫卖。

任务方向/奖励：目标=据点+政治选择；奖励=B 级正式编制、家学残页、真实之瞳观测记录（残）。

**四阶（A 级·建筑群战 · 政治刀）**

切入身份/时点：契约者以 A 级女武神、逆熵执行顾问、或人合联观察员入场；锚定**人造律者计划曝光边缘**或长空市建筑群级崩坏灾害。

初始事件：神权小队「卡莲」无记忆地执行清洗；你被要求「协助回收黛尔塔样本」。样本会吞噬崩坏能——带它走等于带炸弹，留下等于让奥托完善计划。

开场白建议：「卡莲看着你，眼神干净得像没活过五百年。她说『请配合』，身后芙洛拉与赛露蒂已经举枪。频道里奥托的声音温柔：『孩子，代价可以另算。』你知道温柔后面是实验室的灯。」

关键NPC立场：**卡莲·卡斯兰娜**执行命令；**奥托·阿波卡利斯**报价附灵魂利息；**宁蒂·阿波卡利斯**对律者特攻，可能把你当练习靶；**无色辉火**可能用命换一次狙击窗口。

主线钩子/支线：主线——截获或护送人造律者样本并决定去向。支线——帮无色辉火买到最后一块蛋糕换一次掩护；向瓦尔特递交「科学侧」副本；阻止克莱尔/芙洛拉姐妹线的伦理崩坏扩大。

危险度/规避：**高**。规避：当面刺杀奥托；把样本当普通货物快递。

任务方向/奖励：目标=样本政治；奖励=A 级模组、Nexus 使用许可（限时）、英灵殿情报单半价券。

**五阶（S 级·城市灾害 · 领域雏形）**

切入身份/时点：契约者以 S 级协力者、雪狼相关编外、或英灵殿高价佣兵入场；锚定**第二次大崩坏西琳线边缘**或皇帝级崩坏兽攻城。

初始事件：天空裂开，西琳领域边缘让导航失灵；塞西莉亚与瓦尔特分别给你两条互相矛盾的集结点。选错一边=另一边的友军全灭。

开场白建议：「西琳的笑声从云层里漏下来，像小孩发现玩具会哭。塞西莉亚的频道在滴血，瓦尔特只说坐标。你抬头看见城市交通网像被橡皮擦掉一半——有人告诉你这叫『领域雏形』，你只觉得胃在下坠。」

关键NPC立场：**西琳**视你为丑陋世界的一部分或意外保护者；**塞西莉亚·沙尼亚特**要平民；**瓦尔特**要切断权能源；**时雨绮罗**忠于塞西莉亚。

主线钩子/支线：主线——延缓西琳完全展开，为疏散争取小时级窗口。支线——护送齐格飞残部；回收空之律者相关观测；阻止玛丽亚/安洁利亚圣女候选政治趁乱夺权。

危险度/规避：**极高**。超阶：崩坏意志——**存在·情报优先**。规避：单独硬刚完全展开律者；「感化奥托一集成功」。

任务方向/奖励：目标=延缓+疏散；奖励=S 级模组材料、领域抗性药剂、圣女政治一次性豁免文书。

**六阶（律者残响·军团长 · 地区灾难）**

切入身份/时点：契约者以英灵殿顶级佣兵、军团停火监督、或崩坏学园特聘实战教官入场；锚定**军团内战停火线**或人造律者小队全编制出动。

初始事件：南方军与第一军团残部在学园外十公里对轰；拉维妮娅的使者与克里斯玛的使者同时要你「作证」。同时终端读数开始发黄——你必须在「停火签字 / 护送九霄谈判 / 切断炮兵标定」中三选二，且每个选择都会得罪一方。

开场白建议：「停火协议的墨还没干，炮兵标定激光已经爬上崩坏学园的塔楼。拉维妮娅的名字在通讯录里像一把剑，克里斯玛的名字像另一把。九霄在频道里说：『预言告诉我你会选第三条路——可惜预言又歪了。』」

关键NPC立场：**拉维妮娅·阿维尼翁**要体面胜利；**克里斯玛·桑德兰**要正统；**蓬莱寺九霄**要关税主权；**德丽莎**要学生命；**布洛妮娅**要战后孤儿院不被征兵。

主线钩子/支线：主线——促成可维持 72 小时的停火或成功转移炮兵标定。支线——帮卡琳送一份夹带假情报的快递；帮阿库莉亚洗掉「魔女屠城」谣言；截获东方军团对学园的收购收购。

危险度/规避：**极高**。规避：把学园学生当人质筹码；公开出售三年庇护名额。

任务方向/奖励：目标=停火/主权；奖励=军团级圣痕材料、学园永久访客权、地脉一次免费护送。

**七阶（终焉前夜·文明压力 · 条件性胜利）**

切入身份/时点：契约者以「天命特遣 / 逆熵执行顾问 / 学园特聘 / 兵团停火总监督」入场；锚定**终焉降临前夜**、**西琳完全展开**、**迦娜级行星威胁逼近终端**或**学园被征为炮兵阵地**叠加危机。

初始事件：终端读数转红，天空异色；姬子或德丽莎下达「非 S 级禁止靠近核心区」。你脚边柏油裂开，崩坏兽潮与律者领域同时出现——必须在 **疏散平民 / 护送琪亚娜突入 / 切断逆熵实验源 / 向迦娜出示旧契约信物** 中二选一加一个代价。学园线叠加：九霄同时收到「关税被劫」与「终焉读数上升」。

开场白建议：「天空像被揉皱的锡箔。琪亚娜的枪口还冒着热气，芽衣的刀光在废校走廊里一闪。『别看天上太久，』姬子的声音从频道砸下来，『看久了会以为自己也能当律者。』广播里德丽莎用童声念撤退坐标，听起来像课间操——直到坐标末尾多了一句：来不及的，不要回头。你把终端按在胸口——它比心脏吵。」

关键NPC立场：**琪亚娜**揽险；**芽衣**要计划；**布洛妮娅**要希儿与真实；**德丽莎**要指挥链；**姬子**要学生；**西琳**领域扩张；**奥托**灵魂利息；**迦娜**只认旧约；**九霄**玩笑后极狠；**卡琳/阿库莉亚**用行动换信任。

主线钩子/支线：主线——延缓终焉/律者完全展开，为圣痕终端或学园撤离争取小时级窗口；或阻止兵团把学园改成炮兵阵地同时保持关税链不断。支线——救出孤儿院实验体、夺回圣痕计划伪本、护送希儿稳定量子态、帮无色辉火在透支前完成关键狙击。

危险度/规避：**贴近顶点**。超阶：崩坏意志·周期本体、行星级崩坏兽迦娜完全体——**存在·情报优先/条件性胜利**，严禁「削弱终焉/迦娜当周本」、严禁「奥托五百年执念一集感化」。规避：单独硬刚完全展开律者而不带撤离方案；把学园学生当经验包。

任务方向/奖励：目标=保全核心角色存活与周期火种/学园庇护主权；成功条件优先写「终端读数从红降到橙并维持六小时 / 领域半径缩小到可疏散 / 军阀撤炮 / 量子同伴可对话」而非「击杀血条归零」。奖励=S 级模组材料、圣痕构型碎片、学园突破三年限制的特例合同、女武神指挥临时权限、英灵殿情报一次免费单。不许发「崩坏意志杀灭按钮」或「永久律者核心」。
