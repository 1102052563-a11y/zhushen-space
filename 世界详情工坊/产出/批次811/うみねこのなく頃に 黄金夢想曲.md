## 剧情

**【作品来源】**
《うみねこのなく頃に 黄金夢想曲》（黄金夢想曲／Umineko: Golden Fantasia）是 07th Expansion 于 2010-12-31（C79）发售的海猫衍生 **2D 标签对战格斗**；2011-12-31《黄金夢想曲†CROSS》追加碟；Xbox 360《黄金夢想曲X》；MangaGamer 英译《Umineko: Golden Fantasia》2017-12-08（含 rollback 联机）。原案竜骑士07。气质：角色对抗舞台剧 × 平行 IF 短篇情感。休闲档案**不写招式数值**，只写角色配对故事、台词魅力与「如果六轩岛换一种演法」。

**【世界定位】**
这是六轩岛被抽成「舞台」后的黄金剧场：同一批角色以双人搭档登场，在另一条连续性里完成各自的小故事与结局。契约者可作观众、主持人、或某对搭档的临时队友，核心玩法＝角色互动、搭档默契与 IF 恋爱／和解，而非生存恐怖。

**【世界观 · 舞台设定】**
时间仍锚定 1986 六轩岛惨剧氛围，但梦想曲明确标注 **alternate continuity**：每对搭档有独立故事与结局，不必服从本传 EP 因果。元世界（Metaworld）被视觉化为可切换的华丽背景——休闲侧解释为「情绪高潮时舞台布景变了」，用来强化宣言、羁绊与对手戏，而不是比谁更强。

温度来自：战人与贝阿朵的吵闹搭档感；纱音的防守与嘉音的突进像两种自我；缘寿带着樱太郎与炼狱七姐妹的「孩子气魔法」；绘梨花与德拉诺尔的法理口角；拉姆达与贝伦的观测者互怼。格斗外壳下是**角色关系的嘉年华**。

**【地理 · 生活舞台】**
- **六轩岛大屋回廊／玄关肖像前**：开幕与对峙的经典布景。  
- **玫瑰庭园**：纱音、让治系故事的柔光场。  
- **礼拜堂与黄金蝶飞舞的幻景**：魔女侧故事的「魔法被看见」瞬间。  
- **炼狱茶会式舞台**：元世界切换时的金色大厅——用于告白、反转与和解。  
- **联机／对战大厅（玩家层）**：CROSS 与英译强化的社交场；档案写为「黄金剧场的观众席」。

**【故事主线 · 情感线】**
**框架**  
故事模式以「标签搭档」为单位推进：每位角色有独特能力名与性格表演，但情感线写的是——谁和谁被绑在同一队、他们如何用对手戏完成未在本传说完的话。

**初始十人情感素描（本传关系的 IF 压缩）**  
- 战人＋贝阿朵：从互骂到互信的经典对；IF 里可以更早并肩。  
- 缘寿＋路西法（炼狱）：妹妹与家具桩的奇妙组合，写「被抛弃者互相取暖」。  
- 纱音＋嘉音：同一灵魂的两种姿态同台——梦想曲最直接的「自我对话」。  
- 瓦尔基莉亚＋罗诺威：导师与执事的优雅日常战。  
- 绘羽贝阿朵＋夏妃410：暴力华丽与远程冷感的对照喜剧。

**X／CROSS 扩展的情感加笔**  
- 朱志香、让治、楼座加入，补全「人类侧亲属」的舞台戏份。  
- 绘梨花、德拉诺尔：侦探与异端审问的法理调情（轻松向）。  
- 黑战人：本传外 TIPS 的暗色 IF，写「如果战人走向残酷」的自我恐惧。  
- 威拉德：长刀与倾听，EP7 气质的短篇。  
- 拉姆达戴露塔、贝伦卡斯特尔：观测魔女的零食战与猫 pun，提供轻松后日谈。

**梦想曲的情感总弧**  
本传的血与泪被暂时按下暂停键 → 角色在规则化舞台上「玩」自己的关系 → 每对搭档走到专属小结局（和解／告白／并肩／互相承认）→ 玩家带走的是**角色魅力图鉴**，而非新的惨剧真相。若与本传连读，梦想曲是「散之后的粉丝向嘉年华」；单独入世则是黄金岛主题乐园。

**【故事主线 · 情感线 · 补述】**
黄金夢想曲的「剧情」分散在数十个搭档故事里，像一本可重演的舞台剧本集。它不回答「六轩岛谁是凶手」的最终卷，而回答另一个粉丝向问题：**如果把这些人从惨剧时间里暂时请出来，让他们只负责完成一段对手戏，他们会说什么？**

战人与贝阿朵的搭档故事往往从互骂开场，在 Touch 换位的瞬间变成「把后背交给你」。纱音与嘉音同台时，观众看到的是自我整合的寓言：防守与进攻、承受与突进、想活与想被爱。缘寿带着炼狱七姐妹与樱太郎出场时，舞台忽然变成儿童剧——残酷本传里的「被遗弃」在这里被改写成「我带了很多朋友」。瓦尔基莉亚与罗诺威的组合像成熟组的双人舞：照顾、吐槽、护短，不需要大喊爱字。绘羽贝阿朵与西埃斯塔410则是狂气与冷静的喜剧对撞，把本传压力释放成「可以输、可以闹、可以再来一局」。

CROSS 与追加角色把剧场座位加长：朱志香终于能用吉他 riff 表达心意；让治用耐心换纱音的安心；楼座获得「若能重来一次对真里亚」的 IF 练习场；绘梨花与德拉诺尔把法条吵成一种奇怪的约会；黑战人逼本传战人直视「我也可能变成只想赢的人」；威拉德继续当倾听者；拉姆达与贝伦把观测变成零食与猫的综艺。每个小结局都短，但情绪点明确：握手、并肩、认输却被拉起来、或一起看海猫。

对休闲契约者而言，梦想曲是海猫宇宙的**主题乐园日**：可以玩对抗的形式感，但计分板写的是「默契」与「是否尊重角色心结」，不是谁更强。

**【可攻略角色 · 深度补述】**
在梦想曲里攻略角色，本质是「选搭档写短篇」。贝阿朵线把她从反派皮肤还原成想被喝彩的主角；战人线让他学会赢了之后还要留下来；纱音线多给她「请站在我前面也可以」的反向保护；嘉音线让他完成一次对朱志香说出口的练习；缘寿线减少「寻仇」，增加「被需要」；绘梨花线在她宣布胜利时问「你开心吗」；德拉诺尔线用条文保护弱者而不是惩罚情感；黑战人线是劝回与整合；拉姆达／贝伦线是轻松后日谈，适合不想碰惨剧的玩家。

**【舞台剧式篇章展开】**
梦想曲没有传统「一章推主线到终局」的长弧，而是以「选角→搭档故事→小结局→解锁更多对手戏」循环。对档案而言，应把每个循环写成可入世的短篇剧本：

循环A「初演」：玩家从战人、贝阿朵、纱音、嘉音、缘寿等初始座中选两人。开场总在六轩岛布景，台词却故意轻：战人抱怨「又是黄金」，贝阿朵大笑「黄金不够才要用梦想补」。三场表演后进入专属对话，主题固定为「你为什么站在我身边」。若选情侣投射档，故事会走向告白练习；若选对立档，故事会走向停火握手。

循环B「换位」：Touch 系统在叙事上等于「把聚光灯让给对方」。契约者可扮演教练，训练角色在危机时说「换我来」而不是「我一个人上」。纱音学会进攻一句台词，嘉音学会防守一个拥抱，这种交换比胜负更重要。

循环C「元世界」：背景切换成金色大厅时，允许角色说平时说不出口的话。贝阿朵可承认「我害怕被当怪物」；绘梨花可承认「我害怕不被需要」；黑战人可承认「我是你不敢看的那部分」。元世界结束后，台词收回日常皮囊，但好感残留。

循环D「CROSS 联欢」：全员到齐的周目像同人祭。拉姆达发糖果，贝伦发猫 pun，威拉德发安静，楼座发「对不起」的练习。契约者的任务是给每个人安排一个「被认真对待的三分钟」。

**【与本传的情感对接】**
玩过散篇的人，会在梦想曲里看到「伤口的玩具化治疗」：不是否认惨剧，而是允许角色在安全规则里重演关系。没玩过本传的人，也能把这里当角色魅力展览，再反向进入六轩岛的雨与台风。两种路径都合法；档案要求入世时标明「本世界是IF剧场，不覆盖本传因果」。

**【人物群像补强】**
炼狱七姐妹在缘寿线里不是召唤物，是「被丢掉的孩子互相认亲」。路西法的傲慢是怕再被弃。西埃斯塔系列的编号感，对应家具伦理：先有用，后有名。瓦尔基莉亚像把贝阿朵的「以前」具象成可拥抱的长辈。罗诺威则是把「服务」从贬义翻成专业与尊严。绘羽贝阿朵的狂气是生还者压力的二次创作释放阀——让她在舞台上大闹，是为了不在现实里把缘寿掐死。黑战人是道德惊悚镜：当你只想赢推理时，你会长成他。

**【名场面脚本化】**
1. 选人界面长按角色，听一句本传名台词的轻量版。  
2. 纱音与让治搭档的「未来」一词被贝阿朵偷听后学舌。  
3. 朱志香赢了却去拉嘉音的手，说「才不是因为弱」。  
4. 缘寿把樱太郎塞给败者：「给你，别哭。」

## 休闲切入点

> 本世界为休闲／角色互动向（黄金夢想曲），外壳是对战舞台，内核是**搭档故事与 IF 和解**。契约者以观众或临时队友身份入场。

切入身份：黄金剧场司仪／新人练习生／某角色的「Touch 搭档」／缘寿带来的人类朋友／茶会记录员。最自然是被贝阿朵点名：「来，人类，站到战人边上。」

切入时点：任意故事模式开幕；或 CROSS 全员到齐的「大联欢」前夜。

初始处境：后台化妆间式的走廊，墙上挂着全员立绘海报；你与**右代宫战人**、**贝阿朵莉切**共用休息室；第一场表演前要决定站谁身边。

开场白建议：「灯暗下来，海猫的叫声被鼓点盖过。金色蝴蝶扑到你鼻尖时，战人已经在喊『别发呆！』贝阿朵则笑得像要吞掉整个舞台：『欢迎来到我的黄金夢想——今日只许华丽，不许无聊。』你深吸一口气，握住某人伸来的手。」

可攻略对象：
- **贝阿朵莉切**：双人故事通关后的茶会；吃被当成对等明星；心结＝怕被只当反派。  
- **右代宫战人**：练习赛与互怼；吃义气；心结＝赢了之后的空虚。  
- **纱音／嘉音**：防守与突进的互补教学；心结＝自我统合与恋爱资格。  
- **右代宫缘寿**：带她走「不孤独」的搭档路线；心结＝被遗弃感。  
- **古户绘梨花／德拉诺尔**：法理辩论约会；心结＝想被承认真相。  
- **拉姆达戴露塔／贝伦卡斯特尔**：零食与猫的轻松线；心结＝观测者的寂寞。

日常玩法钩子：
1. **搭档编成**：每周换搭档，写短篇 IF（告白／和解／并肩）。  
2. **Touch 默契训练**：非数值，而是「何时把舞台让给对方」。  
3. **元世界演唱会**：全员合唱式后日谈。  
4. **黑战人劝回**：用本传记忆感化 IF 暗色自我。

氛围/雷区：嘉年华、华丽、轻；可甜可燃，但保持角色本传性格。忌强弱排位化、忌无视心结的无脑后宫点名、忌把楼座／绘羽线写成单纯虐角。  
补充：第一周只打表演赛交朋友；第二周专精一对情侣档；第三周办「无胜负」的和解舞台——让输家也能被拥抱。

切入时点补充：CROSS 全员解禁的联欢周；或某对情侣档第一次被系统强制组队的那天。

节奏建议（四周模板）：
第一周只建立面孔与称呼，禁止推进惨剧／国战主线；每天记录「谁对你笑了」。
第二周进入共同事务（祭典、军粮、药铺、舞台排练），用劳动换信任。
第三周面对压力事件时，强制执行「先找同伴说话」流程，禁止独行英雄。
第四周处理结局选择：公开或守护、摘面或继续演、送行或同行——每种选择写清情感代价。
关系推进检查表：是否用真名呼唤对方；是否记得对方怕什么；是否在对方崩溃时在场；是否拒绝用强弱排序定义对方价值。
禁忌总表：禁止把创伤当萌点消费品；禁止外来者优越感羞辱本地信仰与习俗；禁止用厮杀任务替代谈心；禁止角色OOC成无脑后宫工具人；禁止在休闲档写入对决排位式升级。

开场任务三选一：为战人贝阿朵写停火喜剧；为纱音嘉音写自我整合小品；为缘寿写不孤独庆功宴。
联机向入世：把对手玩家写成「来访剧团」，用友谊赛换角色对话，不比强弱比谁更懂角色。
