# 欲望回帰-城市游乐场
<!--meta lib=主库 tiers=一-->

## 剧情

**【作品来源】**
《欲望回帰》为日系成人向都市调教／精神侵染系 ADV 气质作品群在「轮回乐园」世界库中的分锚条目；本档卡片锚点为「**城市游乐场**」。公开可溯源的单一同名商业原作页不足时，以副标题场景逻辑与同类型公开设定（都市、便利店、校园、游乐场、数字下载）为舞台骨架，**禁止编造未核验的作者真名与终局独占宝物**。气质：现代日本都市、欲望被「回帰」系统放大、普通人被卷入规则杀。18禁条目只写组织、规则、任务介入与人物弧光，不写色情过程。

**【世界定位】**
现代都市。存在名为「欲望回帰」的黑市／地下程序／都市传说式装置：它不创造新欲望，只把被压抑、被道德压回潜意识的欲望「回帰」到行动层。本档锚点：**夜场手环与游乐广播同步触发大规模欲望回帰**。一句话：在「看起来一切正常」的城市表层下，谁先触发回帰，谁就改写关系与支配权。

**【世界观 · 力量体系】**
力量本源不是灵气，而是**信息、药物、暗示、社会把柄、空间权限**四件套。死亡：凡人一死即止；「回帰」不提供复活，只提供行为改写与记忆模糊。特殊系统：
- **欲望回帰度（0–100%）**：超过阈值后公开人格与夜间人格分裂，可被观察为「口令触发／场所触发／时间触发」。
- **触发器**：灯光色温、手机推送、店内BGM、学号广播、游乐场广播等日常物件。
- **把柄经济**：把柄＞武力；一阶世界里一把刀不如一段录音。

力量阶梯（宁低勿高，破坏力对齐街区级以下）：
1. **普通人／店员／学生**：徒手、工具、情绪，≈**一阶**（本档主覆盖）。
2. **不良／黑市跑腿／保安队长**：械斗、围殴、撬锁，≈一阶上沿。
3. **地下诊所医生／洗脑技术员／系统运维**：药物与暗示组合，≈二阶边缘（本档一阶覆盖时作压迫者，不发二阶切入）。
4. **都市传说级「回帰宿主」／幕后金主**：可调动整条街的监控与舆论，**超阶巅峰（对本档一阶）**：存在·情报优先／条件性胜利，严禁「被封印所以战力归零」。

**乐园阶位映射**：普通人与店员／学生主战力≈**一阶**；械斗级≈一阶上沿；地下技术员≈二阶边缘；幕后金主与完整回帰宿主≈超阶（对本档）。本任务世界仅覆盖**一阶**；低阶切入规避宿主本体与金主直属武装。

**【地理 · 舞台】**

- **城市游乐场入口闸机**：手环权限。
- **旋转木马与鬼屋**：广播与黑暗触发。
- **员工通道与机房**：母带与手环服务器。
- **烟花广场**：公开事故高发。
- **闭园后的停车场**：回收完成品。


写场景优先：霓虹反光、监控死角、便利店冷柜嗡鸣、教室粉笔灰、游乐场广播失真、手机下载进度条。

**【世界剧情线】**

① **限时夜场**  
园方推出「欲望夜场」折扣——实为大规模触发测试。入园手环在特定游乐器响起时振动。

② **城市游乐场（卡片锚点）**  
回归在人群与烟花下发生：有人在鬼屋出口对陌生人服从，有人把奖品全部送给同一测试员。可观察、可拍照、可报警，但报警语言对不上「系统」。

③ **员工分裂**  
老员工想停机房；新经理要数据。

④ **烟花最高点**  
烟花高潮＝总触发。毁掉此时机房电源可救全场一成。

⑤ **闭园回收**  
未出园的高回归度访客被「专车」接走。

**【势力图谱】**
- **表层日常圈**：学校、便利店连锁、游乐场运营公司、下载平台客服——维持「什么都没发生」。
- **欲望回帰地下网**：技术员、跑腿、把柄中介；以「回归度」收费。
- **受害互助残党**：被回归后仍保留记忆碎片的人；可信度低但情报真。
- **幕后金主线**：不详真名时只写职能——出资、洗钱、回收「完成品」。
敌友：把柄优先；盟约可破；公开喊「欲望回帰是真的」会被当疯子——这是正确社会反应。

**【贵重物品】**
- **欲望回帰客户端／安装包**：真名级道具；安装即绑定设备与生物特征；下落随锚点场景。
- **触发器物件**：色温遥控、店内BGM母带、学号广播授权卡、游乐场广播密钥、下载序列号。
- **把柄U盘／云端相册**：硬通货；抢到＝谈判权。
- **阻断剂安瓿（地下诊所）**：短暂压低回归度，副作用失忆与手抖。
- **金主黑卡／权限腕带**：通行地下会所；一阶可用但会被追踪。
- **手环服务器备份盘**。
每件写清：谁想要、抢到后果、一阶能否用。

**【隐藏剧情 · 伏笔】**
- 「欲望回帰」不是魔法口号，是可计费、可版本迭代的地下产品；每次副标题场景是一次「现场测试」。
- 回归度100%后仍可能残留「旧自我锚」（一枚旧车票、一句暗号、一个未完成的约定）——条件性胜利抓的是锚，不是杀宿主。
- 幕后金主可能同时资助「受害者互助」与「完成品回收」，两边收割。
- 烟花点火键与机房电源联锁。

**【大事记时间线】**
前史：都市传说论坛出现「欲望会回来」帖 → 黑市出现安装包 → 首批测试者失踪或「性格突变」 → 本档锚点场景启动 → 回归度攀升与公开事故 → 阻断剂黑市化 → 金主回收或测试收束 → 城市表层恢复平静（地下继续）。

**【叙事基调 · 雷区】**
都市黑暗、规则杀、把柄与日常的落差。忌写成超能力大战；忌低阶硬刚金主武装；忌把回归写成永久无自我洗脑（应保留旧锚与可逆窗口）；忌假造未核验的作者／角色全名——查无写「不详」。最早切入：安装包出现后第一夜，或锚点场景第一次触发器响起前。

**【游乐场·失败态】**烟花后全场鼓掌节奏完全一致——一致即完成。

**【游乐场·消防借口】**「电路过热」可合法要求机房开门；需温度计造假或真过热。

**【游乐场·气球小贩】**阻断剂藏在氦气瓶夹层；买错是普通气球。

**【游乐场·保安无线电】**频道3被经理监听；真联络用频道7旧频率。

**【游乐场·契约者皮】**临时夜场员、消防协力、手环厂商调试。

**【游乐场·闭园专车】**车牌每夜不同，但司机同一人——记脸不记牌。

**【游乐场·旧锚】**入园合影小票编号可作旧锚索引。
