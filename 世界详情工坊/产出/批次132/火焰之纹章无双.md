# 火焰之纹章无双
<!--meta lib=主库 tiers=一、二、三、六-->

## 剧情

**【作品来源】**
《火焰之纹章无双》（Fire Emblem Warriors / ファイアーエムブレム無双）由ω-Force与Team Ninja共同开发，光荣特库摩日本发行、任天堂海外发行；2017年9月28日日本、10月20日全球登陆Nintendo Switch与New Nintendo 3DS。系列首次与《真·三国无双》式割草动作深度融合的跨界作品，智能系统（Intelligent Systems）担任角色与设定监督。故事原创，舞台为艾多利斯（Aytolis）与格里斯通（Gristonne）邻国；可操作角色大量来自《新·纹章之谜》《觉醒》《if》及部分《外传》《烈火之剑》等。文风：热血中世纪奇幻、多世界英雄集结、无双战场与纹章武器三角并存。DLC追加苍炎/暗黑龙等角色。**本条目以本体故事模式主线为准**。

**【世界定位】**
中世纪奇幻大陆上，艾多利斯双生王储**罗安（Rowan）**与**莉安娜（Lianna）**在邻国格里斯通王子**达利欧斯（Darios）**来访比武之日，遭遇自城堡传送门涌出的魔物袭击。母后**叶莲娜（Yelena）**被掳前将受神龙祝福的**炎之盾（Shield of Flames，本作火焰纹章）**托付双子。格里斯通国王**奥斯卡（Oskar）**正复活混沌龙**维勒札克（Velezark）**；双子须集齐其他世界英雄之力凝聚的**辉石（Gleamstones）**为炎之盾充能，并联合被卷入的历代纹章英雄，最终击败混沌龙。一句话：这是「双生王储借多世界英雄之火守护本国」的无双集结剧。

**【世界观 · 力量体系】**
力量本源：①**炎之盾/火焰纹章**——神龙祝福的圣器，可抗黑暗，需辉石充能至完全态。②**辉石**——由其他世界英雄之力结晶而成，是跨世界协作的具现。③**武器三角**——剑克斧、斧克枪、枪克剑；无双动作中仍作克制轴。④**兵种与转职**——纹章式职业（飞马、重装、魔法等）影响机动与特技；大师证转职。⑤**觉醒模式（Awakening Mode）**——击杀积攒后强化输出并暂时无视三角。⑥**羁绊（Bond）**——角色间支援提升，解锁合击与特殊行动。⑦**混沌龙与魔物**——奥斯卡召唤的黑暗军势，可附身操纵人类。

死亡规则：故事模式按关卡配置；经典模式阵亡需金币与道具复活，休闲模式自动复苏——乐园可选用经典硬核规则。跨世界英雄在危机解除后归还本世界，非永久驻留。

战力阶梯（宁低勿高）：
1. **民兵·城卫·低阶魔物**≈一阶。
2. **精锐士兵·初级英雄学徒**≈二阶。
3. **成名纹章英雄入门态·要塞军官**≈三阶。
4. （清单无四、五阶）中坚英雄与地区Boss观感并入三~六。
5. **完全充能炎之盾侧·高阶英雄连携·混沌龙复活战**≈六阶。
顶点：混沌龙维勒札克；附身态达利欧斯为高威胁人型关。

乐园阶位映射：民兵~低阶魔物≈一阶；精锐/学徒≈二阶；成名英雄入门~要塞战≈三阶；炎之盾完全态决战/混沌龙≈六阶。**世界顶点＝维勒札克**。宁低勿高；条件性胜利（夺回盾、断献祭、集英雄连携）。

**【地理 · 舞台】**
- **艾多利斯王城与城堡传送门**：开篇被袭地，门通向其他世界与魔物源。
- **艾多利斯全境**：双子集结义军、夺取据点、护送与防守的主舞台。
- **格里斯通要塞与王城**：奥斯卡据点，终盘救人质与献祭场。
- **跨世界战场切片**：各英雄原作世界被“拉入”后的战场皮肤（觉醒、if、谜之等气质关卡）。
- **历史模式地图**：复刻历代名场面的挑战舞台，非主线时间线。
势力：艾多利斯王室与义军、格里斯通王军与混沌龙势力、跨世界英雄联军、商人安娜等中立服务者。

**【世界剧情线】**
① **比武之日与王城沦陷**  
罗安与莉安娜与好友达利欧斯比武练剑时，城堡内部门户开启，魔物洪水般涌入。叶莲娜被俘前将炎之盾交给双子，嘱其以神龙祝福抗黑暗。达利欧斯的父王奥斯卡被揭为复活混沌龙的主谋。世界状态：王城火光、母后失踪、双生子被迫出奔集兵。

② **集结他界英雄与辉石**  
双子得知：要阻止维勒札克完全复活，须以其他世界英雄之力凝聚辉石，为炎之盾充能。他们穿越被卷入的各战场，与**马尔斯系/觉醒/if**等英雄接触——如**库洛武、露琪娜、鲁弗莱、神威、镰刀妹卡米拉、利昂**等（具体可操作名单以游戏与DLC为准），以共同敌人说服其并肩。关卡结构：夺点、击破将、限时救援、毒雾地形等无双目标与纹章战术混合。

③ **达利欧斯被附身与盾被夺**  
格里斯通要塞一战中，维勒札克成功**附身达利欧斯**；炎之盾在即将完成充能之际被夺走。双子追至格里斯通，发现叶莲娜将被献祭。名场面：挚友反目、盾光易主、母后祭坛。

④ **奥斯卡被献、混沌龙现世**  
双子救出叶莲娜，达利欧斯却将**奥斯卡本人**献作完成仪式的祭品，混沌龙维勒札克完全复活。摆脱附身的达利欧斯归还炎之盾后坠入空间裂隙，生死成谜（官方有意留白）。

⑤ **终战与双王加冕**  
双子联合全体跨世界英雄，以炎之盾之力击败维勒札克。危机结束，他界英雄归还本世界；叶莲娜为罗安与莉安娜加冕为艾多利斯**双王**。结局方向：本国光复、邻国混沌威胁暂除、达利欧斯下落开放。主题：兄弟/姐弟共同成长、友情被黑暗利用、团结胜过单骑。

⑥ **羁绊与历史模式（系统叙事）**  
Bond事件补全英雄之间的原作式对话与互文梗；历史模式让玩家重温历代名场面，是“粉丝服务档案馆”。写任务可借用历史模式地图作副本，但主线因果以故事模式为准。

⑦ **DLC角色线边界**  
追加角色（如利昂、塞莉卡、琳、纳巴尔等）丰富可操作池与Bond，不改本体终局定局。契约者招募他们作并肩，勿宣称其改变加冕结局。

**【势力图谱】**
- **艾多利斯王室与义军**：守国、夺盾、集英雄。
- **格里斯通王军/混沌龙派**：献祭与魔物。
- **跨世界英雄联军**：临时同盟，危机后解散。
- **商旅与中立村镇**：补给与Bond舞台。

**【贵重物品】**
- **炎之盾（火焰纹章）**：核心圣器，充能-被夺-夺回-终战。
- **辉石**：英雄之力结晶，充能材料。
- **大师证与徽章**：转职与能力强化。
- **各英雄固有武器意象**（法尔西昂等）：尊重原作，勿乱改归属。
- **金币与复活道具**：经典模式规则道具。

**【隐藏剧情 · 伏笔】**
- 达利欧斯生死留白，官方曾考虑免费DLC回归，最终聚焦历代角色。
- 传送门机制解释“为何历代英雄同场”：设定服务集结，非严谨多元宇宙编年。
- 奥斯卡献己子国与自身：黑暗仪式吃掉施术者。
- 双王结局强调“共同统治”而非单一继承人。

**【大事记时间线】**
比武日魔物袭城 → 叶莲娜托盾被俘 → 双子集英雄与辉石 → 达利欧斯被附身夺盾 → 救母 → 奥斯卡被献、龙复活 → 达利欧斯还盾失踪 → 联军斩维勒札克 → 英雄归乡、双王加冕。

**【叙事基调 · 雷区】**
画风：无双华丽连段+纹章角色立绘感。忌写成纯割草无剧情；忌让罗安/莉安娜一人独占加冕；忌达利欧斯无代价洗白；忌低阶硬刚完全体混沌龙。OOC：库洛武变搞笑役；卡米拉全程无战斗只卖萌。最早切入：比武日城堡警报响起时。


**【加详·剧情密度】**
开篇比武不只是教程：罗安好胜、莉安娜克制、达利欧斯劝和，三人关系在第一场过场就立住。魔物出现后，叶莲娜的托盾是合法性转移——此后任何持盾者若无王室承认，都是政治问题。奥斯卡的计划分两步：开门引他界混乱，再以献祭完成龙。辉石收集章节让玩家在“无双清场”与“纹章式目标”之间切换：限时救人对应支援死亡焦虑，夺点对应战线。

达利欧斯被附身是本作情感核：敌人不是脸谱邻国王子，而是被黑暗写成容器的挚友。夺盾失败后的追击战应写愧疚与怒。奥斯卡被献证明混沌不讲忠诚。达利欧斯还盾坠隙，官方留白，写任务可搜裂隙边缘，不可擅自复活为正史。

终战强调“多英雄齐射式协作”：无双连段是表现，炎之盾导能是胜负手。加冕双王拒绝“唯一真命天子”，契合双主角设计。历史模式与Bond是档案馆，主线不要被粉丝服务带偏因果。

地理补：艾多利斯平原适合骑兵冲锋镜头；森林毒雾关限制飞行；格里斯通要塞多层城门适合无双破门。跨世界切片用原作BGM气质区分，避免地图同质。

人物补：**弗雷德里克、利兹、赛莉卡、琳、马尔斯、希达、奥丁、卡缪**等可按解锁出现，字段从简但必须真名。商人安娜提供刷新与情报。

物品补：各角色专武强化线、徽章技能、金币复活（经典）。炎之盾完全态是六阶叙事门槛。

雷区补：勿让维勒札克战力归零；勿抹杀达利欧斯悲剧；勿把无双写成无剧情刷子。契约者可作艾多利斯骑士、他界随从、格里斯通倒戈兵。

状态机：袭城中/集结中/盾被夺/龙复活/龙死加冕。AI先判定再写。

再补战役：一阶守地窖与灭火；二阶夺村与护粮；三阶要塞延缓夺盾；六阶护盾导能。每阶NPC立场必须换人换动机，禁止复制粘贴同一句假货护送。

再补名场面：托盾、附身夺盾、献奥斯卡、还盾坠隙、双王加冕。感官：铸铁味的门、辉石冷光、龙风无雷。

收束句：艾多利斯的火不是单人英雄叙事，是双子与多世界英雄一起把盾重新举起来的火。契约者可以成为举盾的手之一，但不能把别人的手砍断还自称正义。


**【长档A·政治与门】**
艾多利斯王室以炎之盾为合法性象征，格里斯通则以“开门者”自居。奥斯卡相信混沌龙能让小国跃升为大陆主宰，却不懂龙吃祭品不认国王。叶莲娜早知门之风险，故把盾的使用条件定为“神龙祝福+双子共持”的叙事习惯（实际游玩中盾为关键道具）。罗安的冲动与莉安娜的慎重是国家两种未来：战与治。达利欧斯夹在友情与父命之间，是最容易被黑暗选中的缝。

**【长档B·战役百科】**
典型关卡目标：歼灭、夺据点、防守、护送、限时击破、毒雾生存、双线作战。玩家同时操纵最多四名指挥官，其余AI。武器三角在无双中表现为破防与压制，不是SRPG格子，但克制逻辑仍在。转职与徽章让后期build分化：有人清兵，有人专杀将。Arena与Free Mode提供练级，Story Mode锁角色是为了戏剧节奏。

**【长档C·英雄名录补】**
马尔斯（正统剑）、希达（飞马）、赛莉卡（魔）、琳（剑速）、利昂（魔）、库洛武、露琪娜、鲁弗莱、利兹、弗雷德里克、神威、卡米拉、利昂（if）、零、铃火、奥丁、埃尔菲、利昂等以解锁为准。DLC：阿库娅、雾、零、纳巴尔、米奈娃、琳德、欧文、撒拉、奥利薇等。每人一句：战场定位+与双子关系（导师/朋友/损友/战力外挂）。

**【长档D·情感节拍】**
1托盾2集结3第一次大胜4达利欧斯异常5夺盾失败6救母7献父8还盾9斩龙10加冕。节拍之间插入Bond与村镇日常，防止一路冲杀。达利欧斯失踪后，双子加冕典礼应留空席或默哀式目光，而不是纯欢庆。

**【长档E·契约者长规】**
禁止：杀死双子夺位；永久囚禁跨世界英雄；把炎之盾卖到乐园黑市而不引发追讨。允许：改变支线村镇存亡；提前发现奥斯卡仪式；在坠隙边缘找到达利欧斯遗物（非正史复活）。阶位奖励：一阶民兵装，二阶英雄学徒武，三阶要塞战利，六阶终战装与加冕纪念。

**【长档F·地理志】**
王城、平原、森林毒雾、山道、河桥、格里斯通多层要塞、终焉祭坛、空间裂隙边缘。每地给一种主色：蓝白王城、绿毒林、灰要塞、黑红祭坛。

**【长档G·终局哲学】**

## 阶位切入点

> 阶位↔：一阶≈民兵/低阶魔物；二阶≈精锐/学徒英雄；三阶≈成名英雄入门与要塞战；六阶≈炎之盾完全态与混沌龙决战。顶点维勒札克：情报优先/条件性胜利。

**一阶（王城警报·民兵夜）**

切入身份/时点：艾多利斯城卫或侍从，比武日魔物涌出时。

初始事件：你负责护送平民到地窖，同时有人喊女王被拖向门——你够不到女王，但能保住十二个孩子。

开场白建议：「剑还没比完，墙就先倒了。门里爬出来的东西没有眼睛，却知道往人多的地方走。侍从把一面小圆盾塞给你：今天你不是看客。」

关键NPC立场：**罗安**/**莉安娜**（突围中）｜**达利欧斯**（尚友军）｜城卫队长｜低阶魔物。

主线钩子/支线：护送平民；支线灭火、找回炎之盾箱匣线索。

危险度/规避：中。规避传送门核心。

任务方向/奖励：民兵声望、劣质兵器、双子动向情报。

**二阶（集结前期·夺村）**

切入身份/时点：义军斥候或他界英雄随从。

初始事件：你要在时限内夺回被魔物占领的村镇粮仓，否则辉石搜寻队断粮。

开场白建议：「村里的井漂着旗。旗不是艾多利斯的颜色。远处有人喊英雄的名字——那声音来自别的故事书。」

关键NPC立场：**罗安**/**莉安娜**｜**安娜**｜跨界英雄先遣（如露琪娜级印象）｜格里斯通军官。

主线钩子/支线：夺粮仓；支线Bond式送信、清毒雾小径。

危险度/规避：中高。

任务方向/奖励：二阶装备、辉石碎片线索。

**三阶（要塞与附身前夜）**

切入身份/时点：联军中坚，格里斯通要塞攻防时。

初始事件：达利欧斯眼神在某一秒变了；你必须选择是挡住他夺盾的路径，还是先救被围的叶莲娜方向人质。

开场白建议：「挚友的剑路你熟悉，可熟悉里多了一寸恨。炎之盾在身后发热，像在喊你的名字，也像在喊他的名字。」

关键NPC立场：**达利欧斯**（将附身）｜**叶莲娜**｜**奥斯卡**｜双子｜跨界英雄。

主线钩子/支线：延缓夺盾/救人质；支线破坏召唤阵支点。

危险度/规避：高。完全体龙未出。

任务方向/奖励：三阶武器、英雄Bond提升、要塞地图。

**六阶（混沌龙决战）**

切入身份/时点：炎之盾归还后的终战联军成员。

初始事件：维勒札克展开龙躯，战场被黑暗潮汐分割；你的任务是保护持盾的双子完成导能，并斩断侧翼魔物再生点。

开场白建议：「天裂开的时候没有雷，只有一种像笑的风。盾在双子掌心重新亮起，亮得让人想起母后把盾塞进他们手里的那一秒。有人喊：别看龙的眼睛，看盾。」

关键NPC立场：**维勒札克**｜**罗安**/**莉安娜**｜全体跨界英雄｜**达利欧斯**（缺席/幻影）｜**叶莲娜**（后方）。

主线钩子/支线：条件性胜利——护盾导能、破核心、清再生点；支线护送平民撤离、记录达利欧斯失踪坐标。

危险度/规避：贴近顶点。**超阶巅峰：存在·情报优先/条件性胜利**。

任务方向/奖励：六阶向装备、双王加冕见证、跨世界友谊信物。


**【长档·切入扩写】**
一阶加长：开场可写比武观众席推挤；魔物从地下管道先涌出。你救下的孩子里有人捡到辉石粉末（无充能）。关键NPC：**罗安**、**莉安娜**、**达利欧斯**、**叶莲娜**、城卫老兵。主线：护民至地窖。支线：灭火三处、找回王旗。危险度中。奖励：民兵装、伤药、托盾目击证词。

二阶加长：身份义军或他界随从。事件：限时点三烽火+护送粮车。关键NPC：**库洛武**、**露琪娜**、**鲁弗莱**、**安娜**、**罗安**。主线：粮道。支线：Bond、清毒雾。危险度中高。奖励：二阶武、Bond、辉石线索。

三阶加长：要塞。事件：达利欧斯双声。抉择吊桥/祭坛。关键NPC：**达利欧斯**、**奥斯卡**、**叶莲娜**、**神威**、**卡米拉**。主线：延缓夺盾或救人。支线：破召唤支点。危险度高。奖励：三阶装、地图、附身情报。

六阶加长：龙战。事件：三十秒导能。关键NPC：**维勒札克**、双子、**马尔斯**、**叶莲娜**、联军诸将。主线：护盾破核。支线：平民撤离、坠隙坐标。危险度贴近顶点。条件性胜利。奖励：六阶向、加冕见证、友情遗物。

每阶开场白必须不同画面：雨门/烽火/吊桥/龙风。禁止复制假货护送套话。
