# 合金装备5：幻痛
<!--meta lib=主库 tiers=一、二、三、四、五、六、七-->

## 剧情

**【作品来源】**
《合金装备5：幻痛》（Metal Gear Solid V: The Phantom Pain，2015）由小岛秀夫主导、小岛制作组以 Fox Engine 开发，Konami 发行，系《原爆点》（Ground Zeroes）正续，时间轴落在 1984 年前后、距 MSF 母基地覆灭九年。开放世界潜行行动作，主舞台为苏军占领下的阿富汗与安哥拉—扎伊尔边境内战区，外加塞舌尔近海可扩建的 Mother Base。主题：幻痛（身体与身份的缺失）、复仇、语言与文化作为武器、传说与替身、核扩散与生物兵器。卡片锚点：萨赫勒猿人（Sahelanthropus / ST-84）战——骷髅脸启动可直立行走的核武装 Metal Gear，第三子（Tretij Rebenok / 螳螂）的精神控制使其暴走，Venom Snake 需借坦克、支援火力与弱点打击将其瘫痪。标签：Venom Snake、Big Boss、Quiet、Kazuhira Miller、Skull Face、Diamond Dogs、XOF、声带寄生虫、Sahelanthropus、Outer Heaven 伏笔。

**【世界定位】**
1980 年代冷战阴影下的雇佣兵与情报暗战世界。正规国军、苏联占领军、非洲内战武装、Cipher 的影子与其叛离黑部队 XOF 并存；海上 Mother Base 是无国籍的「第三空间」。原作主线：假死九年后「苏醒」的 Big Boss 之影——Punished "Venom" Snake——在 Revolver Ocelot 与 Kazuhira Miller 协助下重建 Diamond Dogs，追查毁灭 MSF 的真凶，卷入 Skull Face 的语言灭绝与核 Metal Gear 计划，并在终局面对「自己究竟是谁」的幻痛。一句话：在阿富汗与非洲的开放战场上，用 Fulton 与子弹堆出新母基地，再撞上会走路的核威胁与身份谎言。

**【世界观 · 力量体系】**
本世界无灵气功法，战力来自训练、装备、情报、载具、生物强化与战略兵器。层级从下到上：
1. **凡俗兵员与民兵**：步枪、据点哨兵、补给线押运；单兵难撼装甲。
2. **精锐特种与雇佣兵**：Diamond Dogs 招募对象；潜行、爆破、审讯、空中回收（Fulton）。
3. **传说级独狼操作员**：Venom Snake 级——断肢假肢、雪茄、潜行渗透、单人撬据点；依赖 buddy（Quiet / D-Horse / D-Dog / D-Walker）与母基地研发线。
4. **寄生虫强化体（Skulls / Parasite Unit）**：XOF 精英；高速、隐身/相位、抗常规火力，战场压迫感远超正规军。
5. **生物兵器与语言武器**：声带寄生虫（Vocal Cord Parasites）——可按语言株系致死（尤其英语株），亦是 Quiet 能力的双刃源；Code Talker 是关键研究者。
6. **Metal Gear 级战略兵器**：Sahelanthropus（ST-84）可直立行走、核投送威胁；需特殊驾驶/精神媒介（第三子），非单纯 AI。
7. **情报与传说权柄**：Cipher / Zero 的全球信息控制、Big Boss「传说」本身、Outer Heaven 构想——不靠一拳毁城，而靠改写谁能开战、谁被遗忘。

死亡规则：低阶一枪一刀即死；中阶靠医疗平台与假肢续命；寄生虫感染可整营崩溃；Mother Base 检疫失败会逼指挥官亲手处决己方；战略级冲突可把据点从地图抹掉。无人无限复活。核与语言灭绝是文明级威胁，不是个人血条。

乐园阶位映射（按破坏力表现，宁低勿高）：普通士兵/村民/新兵≈一阶；老兵、据点士官、初级雇佣≈一~二阶；精锐 PMC / 苏军特种 / Diamond Dogs 骨干≈二阶；Venom Snake 常态渗透战、Quiet 狙击与近战爆发≈三阶；Skulls 小队、Man on Fire 级异变、据点级装甲集群指挥≈四阶；Sahelanthropus 全开、寄生虫区域爆发、Mother Base 级资产战≈五阶；跨战区核讹诈 / 语言灭绝部署、XOF 全局行动≈六阶；Cipher 阴影下的世界级信息战与 Outer Heaven 传说成型≈七阶边缘。本库仅覆盖一~七阶；真正「传说 Big Boss / 全球 Cipher」的终局棋手写条件性胜利与情报优先，严禁把顶点写成无代价可碾压。

**【地理 · 舞台】**
- **塞浦路斯英军医院（Dhekelia SBA Memorial Hospital）**：开篇苏醒与逃亡；XOF 清场、Quiet 首刺、Ishmael 带路、Man on Fire 与第三子登场。
- **Mother Base（塞舌尔近海海上平台）**：Diamond Dogs 总部；战斗/研发/医疗/情报/动物保护等平台可扩建；Fulton 回收的人员物资在此落脚；检疫与「我们是钻石犬」的残酷仪式亦在此。
- **阿富汗（苏军占领区，卡布尔省一带开放地图）**：主前期舞台。据点、村落、补给哨、Serak 发电厂、OKB Zero、Da Smasei Laman、中央基地营地、拉马尔哈特宫等。苏联—圣战者冲突为背景噪音，真正的猎人是 XOF 与寄生部队。
- **安哥拉—扎伊尔边境（中非内战区）**：中后期开放。Kiziba 营地、矿场、种植园、Nova Braga 机场、油区、Lufwa 谷地等；童兵、雇佣、资源掠夺与 Huey / Code Talker 线交织。
- **OKB Zero / 萨赫勒猿人相关设施**：Metal Gear 研发与展示的心脏；卡片锚点 Sahelanthropus 战在此叙事链上高潮。
- **情报与磁带空间**：卡带录音、Ocelot 简报、Kaz 政治咆哮构成「第二地图」——真相往往不在据点箱子里，而在磁带与沉默里。

写场景时先定契约者落点：医院走廊的消毒水与枪声、阿富汗尘土与苏军喇叭、非洲湿热与童兵目光、母基地铁板与海风、Sahelanthropus 直立时的液压尖啸。本地人若听闻「一人打爆 Metal Gear」会当疯子——正确反应；真能做到的，身后必有母基地与传说成本。

**【世界剧情线】**

① **1975 原爆点余烬 · 九年幻痛**  
MSF 母基地被 Cipher 黑部队 XOF 突袭摧毁。撤离直升机上，一名战斗医护兵（日后的 Venom Snake）替真正的 Big Boss（Naked Snake）挡下爆炸；两人重伤陷入长期昏迷。医护兵失左臂，体内嵌弹片。真 Big Boss 与「幻影」的命运在此分叉——但玩家在开局尚不知晓。

② **塞浦路斯医院 · 苏醒与逃亡**  
1984，自称 Big Boss 的男人在医院醒来。XOF 刺客 Quiet 行刺失败，被火与坠落「杀死」又被寄生虫改写。裹满绷带的 Ishmael（真 Big Boss）注射强心剂助其活动萎缩肌肉，带他逃出医院：XOF 士兵、精神异童第三子、燃火的 Man on Fire（与 Volgin 残响相连）轮番追杀。车祸与混乱后 Ishmael 消失，Revolver Ocelot 骑马接应，送往新母基地。男人接受代号 **Venom Snake**。

③ **救出 Kaz · Diamond Dogs 立旗**  
Kazuhira "Kaz" Miller 在阿富汗被俘、遭酷刑，右臂与左腿被截。Venom 将其救出。Kaz 以幻痛与仇恨驱动：重建武装、向 Cipher / XOF 复仇。Mother Base 以 Fulton 气球回收士兵、车辆、资源甚至动物扩张；研发线产出义肢、武器、制服与支援。Ocelot 提供战术电台与训练。组织名：**Diamond Dogs**——失去的肢体与战友化为「钻石」般的纪念与杀意。

④ **阿富汗渗透战 · 招募与真相碎片**  
开放世界主循环：渗透据点、窃听、劫持、空投、昼夜与沙尘。关键人物线并行——俘获并收编 Quiet（Kaz 强烈反对）；接触 Huey Emmerich（金属齿轮与寄生研究的污点科学家，Otacon 之父）；追查 Skull Face 与声带寄生虫。磁带揭示 Cipher 内部分裂：Skull Face 已架空 Zero 的路线，改走「用语言与核威胁重写世界」的复仇。

⑤ **非洲战线 · Code Talker 与童兵**  
舞台扩至安哥拉—扎伊尔边境。Code Talker（纳瓦霍老人、寄生虫权威）被 XOF 胁迫研究；Eli（日后 Liquid Snake 相关的童兵领袖）与第三子形成危险同盟。童兵营、资源矿、机场与油区提供另一套残酷规则：这里的「新兵」可能是孩子，这里的「合同」可能是卖命。

⑥ **质变副本 · Sahelanthropus 战（卡片锚点）**  
Skull Face 推动 ST-84 Sahelanthropus——可直立行走的核 Metal Gear，企图以核扩散威慑与远程控制并存的方式改写大国规则，并以语言寄生虫打击「英语文化霸权」。第三子的念动力是机体关键媒介。战场上，第三子被附近最强恨意（Eli 一侧）牵引，Sahelanthropus 反噬 Skull Face，Man on Fire 亦被碾灭。Venom Snake 借坦克、火箭、支援火力与机体弱点（腹部、头部、罐体等）瘫痪 Sahelanthropus。名场面：直立液压、核阴影、精神浮空童子、尘土中的单兵与母基地空援。此卷是中高阶核心可玩锚点：入口是情报与战区控制，争夺物是机体/寄生样本/Skull Face 本人，压迫者是 Metal Gear + 寄生部队，出逃时仍有政治与检疫后患。

⑦ **Skull Face 之死与复仇仪式**  
重伤的 Skull Face 被 Diamond Dogs 控制。Kaz 与 Venom 以「以其人之道」的肢体毁灭回应当年 MSF 与 Kaz 的残害；Huey 参与最终处置。寄生虫样本部分回收，一瓶后落入 Eli / 第三子之手。仇恨闭环并未带来平静——它只证明幻痛会传染。

⑧ **母基地疫情 · 钻石与处决**  
寄生虫疫情在 Mother Base 爆发（与 Huey 实验、Quiet 携带的英语株等纠缠）。Venom 被迫隔离并处决感染者；骨灰/遗骸被制成钻石——「We are Diamond Dogs」。组织认同用最残酷的方式加固。Huey 被控多重背叛（含对 MSF 沦陷的牵连、对妻子 Strangelove 的罪行、对幼子 Hal 相关实验的阻挠与杀害叙事）后遭放逐，而非简单枪决——放逐本身是另一种羞辱与隐患。

⑨ **Quiet 的沉默与离去**  
Quiet 因寄生虫以皮肤呼吸与光合代谢，衣物过厚会窒息——暴露装既是设定也是战场借口。她感染英语株，为防传染坚持沉默、不写英文。忠诚转向 Diamond Dogs 后仍被 Kaz 视为定时炸弹。二次疫情压力下她离开；Venom 在阿富汗找到她。Venom 被毒蛇咬伤时，她打破沉默用英语呼叫救援救下他，随即走入沙漠自我放逐，切断传播链。主题落地：语言即武器，沉默即爱与罪。

⑩ **身份终局 · 幻影与真身**  
真相：Venom Snake 不是 Big Boss。他是 1975 年挡爆的医护兵；昏迷期间由 Zero 一系（并涉 EVA 等）以手术与催眠改造成完美替身——面容、声音、习惯与植入记忆。真 Big Boss（Ishmael）更早苏醒，借医院逃亡测试替身，自己转入暗处构筑 Outer Heaven，而让「公开的 Big Boss」=Venom 吸引火力并壮大 Diamond Dogs。Ocelot 知情且配合；Kaz 得知后愤怒到扭曲，发誓要扶植 Big Boss 之子 David（Solid Snake）反噬。Venom 接受自己是「phantom」，继续前行——这条线通向日后 Outer Heaven 起义中替身的死与真 Big Boss 在赞比亚兰等地的再登场（系列后续，本世界点到为止）。

⑪ **微观政治贯穿**  
据点配额与征兵、黑市金刚石与资源箱、假情报磁带、童兵交易、Fulton 误回收、检疫封锁、平台派系、雇佣合同违约、苏军与游击队夹缝收保护费。AI 优先写这些「小刀」，而不是一上来 Sahelanthropus 对轰。

⑫ **蝴蝶效应边界**  
契约者可改局部：救下某哨兵、截获非核心磁带、让某次 Side Op 易主、提前预警一次检疫。不可无代价改写：Venom 替身身份的结构性真相、Skull Face 语言—核双线动机、Quiet 沉默与离去的因果、Sahelanthropus 作为时代威胁的存在逻辑、Diamond Dogs「用死者做钻石」的组织原罪。改了局部，因果以 XOF 清场、疫情提前、Kaz 更疯、Eli 更早夺机的方式回流。

**【世界剧情线·分卷】**

**卷一：医院与幻影苏醒（一~二阶体验）**  
消毒水、绷带、萎缩的腿、义肢尚未装上。Quiet 的第一枪与 Ishmael 的注射是开场仪式。此卷教学：你不是神，你是刚从九年死亡里爬出来的残骸；逃出生天靠的是另一个「你」与 Ocelot 的接应。可介入：医院侧翼清场、护送伤员、截获 XOF 通讯。危险：Man on Fire / 第三子 = 规避或借地形。

**卷二：救 Kaz 与母基地冷启动**  
阿富汗初次部署。救 Miller、立 Diamond Dogs、第一次 Fulton 一整支小队。Kaz 的咆哮与 Ocelot 的冷静构成指挥双螺旋。可介入：劫囚车、抢医疗物资、为母基地送第一批兵。宝物：初级义肢方案、Fulton 气球、据点钥匙卡。

**卷三：阿富汗猎场 · Quiet 收编**  
狙击决斗后俘获 Quiet；母基地伦理撕裂（Kaz 要杀，Snake 要留）。Buddy 系统改变渗透语法：她在远处开枪，你在近处掐喉。可介入：护送 Quiet 上平台、反间谍搜查、与 Kaz 谈判「她的存在权」。危险：Skulls 初现 = 三~四阶压迫。

**卷四：Huey、磁带与 Skull Face 轮廓**  
科学家、卡带、OKB 线索拼出反派：不是单纯「坏人要统治世界」，而是被语言与历史灼伤的人要用寄生虫和核改写文化霸权。可介入：窃取研发资料、保护/绑架研究员、伪造成交武器订单。

**卷五：非洲 · Code Talker 与童兵**  
湿热、矿尘、童兵眼。Code Talker 的知识是解药也是毒药。Eli 与第三子让「恨」有了可被念动力放大的天线。可介入：撤侨、炸桥断补给、潜入童兵营救人（道德代价极高）。

**卷六：Sahelanthropus（锚点卷）**  
Metal Gear 直立。第三子浮空。Skull Face 的演讲与失控同时发生。Venom 的胜利定义：瘫痪机体、夺样本、活着回母基地——不是在沙漠称王。出战后政治：谁拥有残骸、谁拥有磁带、谁拥有叙事。

**卷七：疫情、钻石、放逐与沉默**  
对内屠杀式防疫；Huey 放逐；Quiet 走入沙漠。组织从「复仇俱乐部」变成「知道自己脏手的军队」。

**卷八：真相磁带 · 幻影接受**  
医院夜的第二层含义：Ishmael 是谁、你是谁、Outer Heaven 为谁而建。Kaz 的恨转向更复杂的坐标。契约者若在此阶段泄密，可能比 XOF 的子弹更快摧毁 Diamond Dogs。

**【主要人物】**
- **Venom Snake（Punished "Venom" Snake / Ahab）**｜性格：寡言、行动派、对部下有温情却能下死手；接受残缺与仇恨。｜装备·能力：义肢、潜行工具、多样化枪械、Buddy 协同、母基地后勤；传说级独狼。｜弧光：苏醒的「Big Boss」→复仇领袖→发现自己是幻影仍继续走。｜关系：Kaz 战友/共谋；Ocelot 导师式盟友；Quiet 无声羁绊；Skull Face 仇敌；真 Big Boss 的影子与盾。
- **Big Boss（Naked Snake / Ishmael）**｜性格：冷、棋手、把人变成理念的燃料。｜能力：系列巅峰独狼与领袖；本篇多在暗处。｜弧光：用替身吸引世界火力，暗建 Outer Heaven。｜关系：Venom 是他最成功的幻痛；Ocelot 执行；Kaz 终将反目。
- **Kazuhira "Kaz" Miller**｜性格：煽动、痛苦、政治动物；幻痛人格化。｜能力：指挥、招募、情报、意识形态。｜弧光：残躯复仇→与 Snake 共治→得知替身真相后恨意改道。｜关系：Diamond Dogs 双头之一；厌恶 Quiet；仇视 Cipher/XOF/Huey。
- **Revolver Ocelot**｜性格：戏谑、忠诚于更大棋盘、训练狂。｜能力：枪术、审讯、战术指导、骑马接应。｜弧光：医院救场→母基地操盘→替身计划知情者。｜关系：联接真假 Big Boss 的铰链。
- **Quiet**｜性格：沉默、锐利、以行动表达忠诚与罪。｜能力：寄生强化——超远狙击、爆发机动、隐匿；英语株载体。｜弧光：XOF 刺客→俘虏 Buddy→为爱与防疫自我放逐。｜关系：Venom 的镜面；Kaz 的眼中钉；Skull Face 的弃子与叛旗。
- **Skull Face**｜性格：雄辩、创伤驱动、多语言；仇恨英语文化霸权。｜能力：XOF 指挥、寄生部队、Sahelanthropus 计划、政治恐怖。｜弧光：Cipher 黑手套→自立复仇神→死于自己放出的恨与机甲。｜关系：Zero 的背叛者；Venom/Kaz 的血仇。
- **Huey Emmerich**｜性格：懦弱、自辩、天才与罪人一体。｜能力：Metal Gear / 寄生相关工程。｜弧光：被救→被用→被审→被放逐。｜关系：Strangelove 与 Hal 的阴影；母基地科技原罪。
- **Code Talker**｜性格：沉稳、文化守护者、被胁迫的智者。｜能力：寄生虫学权威。｜弧光：XOF 囚徒→Diamond Dogs 关键顾问。｜关系：Quiet 生理秘密的解释者。
- **Eli（童兵领袖）**｜性格：桀骜、恨意纯粹、领袖欲。｜能力：童兵指挥；与第三子共鸣。｜弧光：被俘与对峙→借第三子夺 Sahelanthropus 逃离。｜关系：未来 Liquid 线伏笔；本世界核威胁余党。
- **第三子（Tretij Rebenok / 螳螂）**｜性格：非成人逻辑；放大附近最强情感。｜能力：念动、驾驶/干扰 Sahelanthropus。｜弧光：Skull Face 工具→Eli 天线→锚点战变量。
- **Man on Fire**｜性格：纯粹执念与火焰。｜能力：超常追猎、燃烧不死表象。｜弧光：医院追杀→与 Sahelanthropus 事件中被终结。｜关系：Volgin 残响级威胁。
- **Paz / Chico 等 MSF 幽灵（磁带与回忆）**｜性格：回忆杀。｜能力：无直接战力。｜弧光：幻痛来源。｜关系：驱动 Kaz 与 Venom 的情绪燃料。
- **Diamond Dogs 兵员群像**｜性格：雇佣、忠诚可买也可殉。｜能力：二阶向集体战力。｜弧光：Fulton 来→平台住→或成钻石。｜关系：契约者最常打交道的「人海」。
- **苏军军官 / 非洲军阀 / 黑市贩**｜性格：占领者与生意人。｜能力：一~二阶。｜弧光：舞台背景与任务板。｜关系：可贿可杀可交易。
- **XOF 部队与 Skulls**｜性格：清场机器 / 非人精英。｜能力：二~四阶。｜弧光：贯穿压力。｜关系：遇则战争。
- **Strangelove（磁带与回忆）**｜性格：偏执的理想与母性扭曲并存。｜能力：AI/Metal Gear 相关。｜弧光：Huey 罪证链。｜关系：Hal 的母亲阴影。
- **Major Zero（Cipher 远端）**｜性格：秩序狂。｜能力：信息帝国。｜弧光：本篇多在磁带。｜关系：被 Skull Face 架空的旧主。

**【势力图谱】**
- **Diamond Dogs**：海上母基地 PMC；复仇与雇佣双驱动；Fulton 经济；检疫专制。
- **XOF**：Cipher 黑部队，Skull Face 私兵化；突袭 MSF 的元凶；寄生强化。
- **Cipher / Zero 一系**：信息控制与「和平」工程；本篇多在磁带与阴影。
- **苏联占领军（阿富汗）**：正规军事存在；据点网格；非最终反派但日常敌人。
- **非洲内战各方与童兵武装**：资源与人口的绞肉机。
- **OKB / Metal Gear 研发链**：Huey—Skull Face—Sahelanthropus。
- **黑市与民间村庄**：情报、钻石、动物、叛徒士兵的来源。
敌友逻辑：合同与仇恨优先；暴露「你知道 Snake 是替身」或「你持有英语株样本」= 多势力同时猎杀。

**【贵重物品】**
- **Venom 的义肢与战斗装**：身份与战力的外壳；可升级，毁了会大幅削弱渗透效率。
- **Fulton 回收系统**：母基地经济引擎；抢谁、抽谁，决定你站哪边。
- **声带寄生虫样本（含英语株）**：文明级生物兵器；谁持有谁被全世界追杀；Quiet 的身体即活体样本。
- **Sahelanthropus（ST-84）机体与残骸**：核 Metal Gear；直立行走的时代恐吓；控制权=地区级讹诈。
- **第三子 / 精神媒介相关「钥匙」**：无此难稳控 Sahelanthropus。
- **关键磁带与 Cipher 档案**：比枪值钱的真相；可颠覆 Kaz、Ocelot、Venom 关系。
- **Mother Base 平台模块与研发蓝图**：组织战力的真身。
- **粗钻石与资源箱**：硬通货；亦是死者纪念的黑色隐喻。
- **Buddy 契约（Quiet / 犬 / 马 / 步行器）**：战术外挂；Quiet 线有伦理与疫情风险。
- **医疗与检疫协议**：疫情章的生杀权柄。
- **Outer Heaven 构想碎片**：高阶才完整；低阶只闻传说。
- **消音枪械与非致命抓捕套件**：Diamond Dogs「要人不要尸」的生产力工具。
- **防毒与检疫面罩**：疫情章硬通货；没有它你进不了封锁平台。
每件写清：谁想要（XOF/Kaz/Eli/黑市）、抢到的后果（猎杀/疫情/核危机）、当前阶能否用（寄生与 Metal Gear 低阶=自杀）。

**【隐藏剧情 · 伏笔】**
- Venom 是医护兵替身：医院夜、Ishmael 消失、基因检测与 Eli「不是你的克隆」等线索早埋。
- Quiet 的皮肤呼吸与英语株：暴露装的合理性与沉默的防疫伦理。
- Skull Face 反英语霸权：多语言台词不是炫技，是动机。
- Huey 的罪与 Hal（Otacon）的未来：系列因果链。
- Eli + 第三子 + 被盗 Sahelanthropus：未完成的「Kingdom of the Flies」阴影——童兵与核的后续威胁。
- Outer Heaven：真 Big Boss 的暗线；Venom 的公开起义是诱饵与祭品。
- 「幻痛」双关：断肢、死友、假身份、被偷走的九年。
- 跨作互文（Snake Eater / Peace Walker / 后世 Solid Snake）点到为止，勿整本倒灌。
- 卡带里 Paz 的声音可能是创伤回响，也可能是 Huey 技术作伪——让玩家与角色一起不确定。

**【大事记时间线】**
1975 → MSF 覆灭，医护兵与 Big Boss 重伤昏迷。  
1984 初 → 医院苏醒、逃亡、Ocelot 接应。  
同年 → 救 Kaz，Diamond Dogs 立；阿富汗渗透扩张。  
中段 → Quiet 收编；Huey / 磁带；Skull Face 与寄生计划浮出。  
中后 → 非洲展开；Code Talker；Eli 线。  
高潮 → Sahelanthropus 战，Skull Face 败亡。  
后潮 → 母基地疫情与钻石化；Huey 放逐；Quiet 离去。  
终局认知 → 替身真相；Outer Heaven 棋盘启动。

**【叙事基调 · 雷区】**
基调：冷、脏、政治、残肢美学、磁带 synch、开放世界的重复中长出的空虚与恨。可黑深残，勿写成无敌超级英雄打卡游。  
忌把 Venom 写成全知真 Big Boss（除非终局已揭且角色自己接受幻影定位）。  
忌低阶满地 Metal Gear 与 Skulls。  
忌把 Quiet 写成无代价恋爱脑或无视寄生设定。  
忌美化童兵线。  
忌编造原作没有的神器名与终局死亡。  
忌空话循环与模板复读。  
忌把顶点写成「无代价可碾」——条件性胜利与情报优先。  
最早切入：医院逃亡前夜，或阿富汗第一次部署，或 Sahelanthropus 启动前的情报战。

**【开放世界运行逻辑 · 给 AI 的场景语法】**
阿富汗与非洲不是「关卡列表」，而是可持续渗透的战区网格：哨站—据点—中央营地—特殊设施层层嵌套。昼夜、沙尘暴、雨季改变能见度与敌军警觉；直升机支援受天气与防空威胁约束。敌人会强化：同一据点多次闹事后换装备、加巡逻、上头盔。Fulton 不只是收集玩法，是人口政治——你抽走的士官可能成为平台科长，也可能把仇人的口音带进食堂。Side Ops 与主线交织时，优先尊重「合同截止」与「疫情封锁」这类组织内部时钟，而不是让契约者无限刷怪。动物保护平台、金刚石、蓝图、卡带四条收集线各自服务母基地某一侧面：战力、资金、研发、真相。写战斗先写潜入失败后的逃路与空援呼叫，再写击杀数。

**【名场面清单（可直接调用）】**
- 医院走廊：Quiet 的瞄准镜反光；Ishmael 把香烟按进你手里之前先把你从床上拽起来。  
- 燃烧的「火人」追车：轮胎、火焰、第三子漂浮的剪影。  
- Ocelot 的马与第一次看见海上母基地：锈与旗，不是圣殿。  
- 救出 Kaz：残肢、脏话、复仇演说盖过医疗器械声。  
- 与 Quiet 的狙击对决：雾、距离、谁先暴露谁先死。  
- Code Talker 的小屋：语言、虫、古老与现代生物兵器叠在同一张桌上。  
- Sahelanthropus 直立：液压、扬尘、童子、核阴影同时压下来。  
- 母基地检疫通道：面具、名单、开枪的手是指挥官自己的。  
- Quiet 沙漠离去：英语破戒的那一句话比任何枪响更响。  
- 真相磁带：你听完后仍要给平台下达下一道出击令。  
- Huey 放逐：没有痛快的处决，只有海风与耻辱。  
- Eli 夺机逃离：童兵、念动力、核阴影一起离开你的控制。

**【世界剧情线·因果长链】**
原爆点之后，世界需要一个「活着的 Big Boss」来填仇恨与雇佣市场的洞。Cipher 造了替身；真身去暗处画 Outer Heaven。替身在阿富汗与非洲用 Fulton 把人变成组织，用组织去撞 XOF。Skull Face 不满足当黑手套，要把语言变成瘟疫、把 Metal Gear 变成直立的核图腾。第三子让恨意有物理出口，于是 Sahelanthropus 先咬主人再被 Venom 打瘫。胜利后母基地发现：敌人的虫子已在自己喉咙里。Quiet 用沉默保护平台，用一句话救 Snake，再用消失保护世界。最后磁带告诉你：你报的仇、建的家、爱的人，有一半是写给另一个人的剧本——而你仍要走下去，因为幻影若停下，真身的棋就暴露。

时点速查：医院/新兵据点=一阶；精锐渗透与 PMC=二阶；Venom/Quiet 独斗=三阶；Skulls/异变=四阶；Sahelanthropus/疫情组织战=五阶；跨战区核与语言部署=六阶；Cipher/Outer Heaven 传说层=七阶。

**【势力·战场分工】**
苏军生产占领网格；军阀生产童兵与矿；XOF 生产清场与寄生；Diamond Dogs 生产复仇与合同；黑市生产钻石与叛徒；磁带生产真相；Sahelanthropus 生产时代恐慌。契约者最好的活法：当可交易的变数，而不是想取代 Big Boss 传说本身。

**【对话立场速查】**
对 Venom：少废话，给目标与退路；别当面拆穿替身（终局前）。对 Kaz：附和他的恨或准备被骂；提 Quiet 即开战。对 Ocelot：他把训练当游戏，把棋盘当家。对 Quiet：不逼她说话；英语是武器也是瘟疫。对 Skull Face：他要观众与语言，不是单纯要你的命。对 Huey：他的每一句自辩都可能是罪证。对 Code Talker：尊重知识与文化，别把虫当玩具。对 Eli：别用「孩子」当轻敌借口。

**【地理补述 · 据点体感】**
阿富汗：干冷、扬尘、苏制电台杂音、山羊与地雷并存；夜色是潜行者的盟友，也是 Skulls 的披风。非洲：湿热、锈蚀铁皮、童声与枪栓同频；机场与油区是物流咽喉。母基地：浪声、焊接火花、食堂笑声突然被检疫警报掐断。医院：白、红、焦糊味。写任何一场戏先选气味与声音，再选枪型。

**【母基地日常可玩事件池】**
平台扩建谈判、逃兵再捕获、动物平台走失、研发事故、士兵恋爱纠纷升级为决斗、磁带泄露恐慌、Fulton 把敌军将军抽到甲板上的外交事故、检疫演习变真、Kaz 演讲导致士气暴涨或暴跌。这些事件服务「组织是活的」：契约者可以是士兵、医官、情报员，而不必永远当 Venom。

## 阶位切入点

阶位↔力量对照：一阶凡俗兵民与据点日常；二阶精锐雇佣/特种渗透；三阶传说独狼与 Quiet 级 Buddy 战；四阶 Skulls·异变猎人与据点集群；五阶 Sahelanthropus·寄生区域爆发与母基地存亡；六阶跨战区核讹诈与语言兵器部署；七阶 Cipher 阴影与 Outer Heaven 传说层。宁低勿高；低阶严禁硬刚 Metal Gear 本体与全局寄生释放。

**一阶｜新兵、村民、据点杂役、黑市跑腿**  
落点：阿富汗外围村落、补给哨外围、非洲种植园边缘、母基地最低层甲板杂役区。你是占领与内战的灰尘：送弹药箱、修篱笆、被双方收「过路费」。冲突：苏军巡逻索贿、童兵抢粮、黑市假钻石、Fulton 误把你当战俘抽上天（喜剧与惨剧一线）。见闻：远处直升机与扩音器里的俄语；有人低声说「Big Boss 没死」。禁止：挑战 XOF 小队、靠近 OKB Zero 核心、宣称能单灭 Metal Gear。收益：本地货币、粗情报、一张被吐槽的假磁带。角色：村长安抚、逃兵、黑市小贩。任务板：送信、埋雷又挖雷、指路过的装甲车方向。死亡极便宜——一梭子结束。叙事目标：让玩家闻见尘土与恐惧，不给英雄幻觉。

**二阶｜老兵、PMC 骨干、据点士官、初级 Diamond Dogs**  
落点：阿富汗中小型据点渗透、非洲营地外围袭击、母基地战斗平台见习。你能单挑一小队、会用消音与夜色，仍怕装甲与迫击炮。事件：执行 Side Op 式抓人、抢蓝图、炸雷达；与 Diamond Dogs 签短期合同；被 Kaz 式广播洗脑「为死者复仇」。遭遇：苏军精锐、叛变雇佣、XOF 普通清场组。可与 Venom 的行动擦肩——他是传说，你是报表上的编号。收益：正规枪械、Fulton 初级权限、据点钥匙卡。禁忌：正面硬刚 Skulls；窃取寄生样本。角色接口：Ocelot 训练课的炮灰学员、Miller 怒吼的执行者。胜利定义：活着完成合同并拿到尾款。

**三阶｜独狼渗透者、Quiet 协同级战场、区域知名雇佣**  
落点：纵深据点、夜袭电厂、与 Buddy 联动的猎杀。战力：可清一个中型基地、与 Quiet 完成狙击—突入配合、用载具与空援改变局部战局。主线接口：参与救 Code Talker 外围、护送重要磁带、在 Sahelanthropus 启动前的情报战里当尖刀。人物：短暂受 Venom 指挥；被 Kaz 怀疑「你是不是 Cipher 的眼」；被 Quiet 无声救过一命——你欠的是沉默不是情话。危险：Skulls 开始针对你；Man on Fire 级事件仅远观。收益：高级义肢配件、母基地宿舍权限、半真半假的 Cipher 传闻。禁忌：公开「Snake 是假的」（此时你多半也不真信）；释放任何寄生样本。叙事：你开始理解幻痛——你失去的战友名字被做成钻石玩笑。

**四阶｜抗 Skulls 的指挥官、异变猎手、平台安保核心**  
落点：Skulls 出没的雾夜战场、母基地检疫封锁线、OKB 外围强袭。战力：带领小队周旋寄生强化体；摧毁据点集群；在非洲切断童兵武装的补给脊梁。主线接口：Sahelanthropus 战前夜的清场与诱饵；阻止 Eli 一方提前夺样；审讯 Huey 相关技术员。人物：与 Venom 并列行动但仍差传说半级；Ocelot 开始用你做「可抛弃的第二方案」。危险：寄生感染风险上升；一次失误可让整平台封锁。收益：反寄生防护、重型兵器调用、对 Skull Face 动机的完整磁带。禁忌：以为自己能肉身扛 Sahelanthropus 主炮。胜利：为锚点战创造「可打」的窗口，而不是私吞机体。

**五阶｜Metal Gear 响应官、疫情总指挥、战区级资产持有者**  
落点：Sahelanthropus 主战场边缘与核心协同、Mother Base 疫情章、残骸回收舰队。战力：调度坦克与空援参与瘫痪 ST-84；执行感染者隔离与残酷清算；在非洲—阿富汗双线挪动资源。卡片锚点完全可玩：你不是唯一主角，但是能改变 Venom 弹药与退路的那个人。人物：直面 Skull Face 残部谈判或处决；目送 Quiet 走向沙漠时选择拦或放——拦可能害死整平台，放则承受 Kaz 的怒火。收益：Metal Gear 残骸碎片（烫手）、英语株相关封存柜钥匙、组织内「钻石仪式」的投票权。禁忌：私自启动核威胁；向真 Big Boss 暗线裸奔邀功。结局味道：打赢机体后，你仍可能死于自己人喉咙里的虫子。

**六阶｜跨战区核—语言危机操盘手、XOF 残网猎手**  
落点：多国阴影外交桌、跨地图寄生投放点、Eli 携残骸逃亡的追猎线。战力：地区级——一次错误部署等于城市级人道灾难；你打的是供应链、实验室与叙事权。事件：阻止语言株扩散到母语城市；与 Cipher 残部做脏交易换解毒方向；决定是否公开 Venom 替身真相以换取某国支援（多半是饮鸩）。人物：Code Talker 的伦理拷问；Kaz 要你把仇恨当燃料；Ocelot 微笑着把你推到真假 Big Boss 夹缝。收益：战略级情报、短暂的核谈判席位。禁忌：以为杀死所有 XOF 就能结束系列——Outer Heaven 的种子已在别处发芽。胜利：有条件的——延缓末日，不是消灭战争。

**七阶｜传说层介入者 · Cipher / Outer Heaven 棋缘**  
落点：不再是单一沙漠，而是「谁有权定义 Big Boss」的信息战场；母基地之上的神话与暗处的 Outer Heaven 工地。战力：世界级影响——你的选择能改写雇佣兵史与后续 Solid Snake 时代的火种，但仍禁止「一拳捏碎地球」。事件：与真 Big Boss 的代理人接触；保全或毁掉替身计划最后的磁带；在 Venom 接受幻影身份时选择站在 Kaz 的恨、Ocelot 的棋、或 Snake 的沉默一侧。人物：真 Big Boss 本体基本不给你硬刚窗口，给的是条件性交易与放逐；Venom 若已觉醒真相，会问你「你是来杀幻影，还是来成为下一个」。收益：Outer Heaven 碎片、系列后续钥匙（慎用）。雷区：禁止无代价碾压传说、编造原作没有的终局击杀、让低阶契约者「教育」Big Boss。收束：本世界通关感不是烟花，是断肢仍痒、磁带仍转、海上平台仍在扩——战争作为习惯活下来。

**各阶通用介入钩子**  
① Fulton 经济：你抽走的每一个人，都在母基地留下政治。② 磁带：比子弹更致命的剧情武器。③ 检疫：善良会害死平台，残忍会保住平台。④ Buddy：Quiet 不是皮肤展示，是寄生与沉默的伦理题。⑤ 童兵：任何「轻松拯救」都可疑。⑥ 核与语言：五阶以上才碰样本，碰了就上全球通缉。

**契约者身份建议**  
医官（呼应医护兵前史）、无线电情报员、Fulton 技师、黑市钻石鉴定人、叛逃 XOF、非洲当地向导、母基地检疫官。避免开局自封「第二 Big Boss」。
