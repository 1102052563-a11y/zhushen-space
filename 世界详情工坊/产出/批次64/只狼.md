## 剧情

**【作品来源】**
《只狼：影逝二度》（Sekiro: Shadows Die Twice）由 FromSoftware 开发、Activision 发行，宫崎英高监制，2019 年发售的第三人称动作冒险游戏。日文标题「セキロ：シャドウズ ダイ トゥワイス」，中文通称「只狼」。舞台为架空战国末期的苇名国，融合忍术、侍道与「龙胤」不死诅咒。文风克制、残酷、悲剧性强：忠诚与铁则对撞，不死并非恩赐而是灾厄。本档案服务轮回乐园世界库「主库」条目，覆盖一至七阶；卡片锚点对应「苇名城破 / 怨恨之鬼」——内府赤备军全面入侵、城池火光冲天、佛雕师堕为怨恨之鬼于战场边缘肆虐。信息密度优先，忠于游戏正典与公开设定，不编造终局与宝物来历。

**【世界定位】**
架空战国末期的苇名国：一方因「樱龙」神威而浸透不死之力的边陲之地。主角是独臂忍者「狼」（后被苇名一心赐名「只狼」），使命是守护龙胤少主平田九郎，并在「斩断不死」「净化」「龙之归乡」与「修罗」诸结局之间抉择。契约者切入后，是这片被内府铁蹄与龙胤诅咒双重碾压的舞台上的变数，不是免死金牌。

**【世界观 · 力量体系】**
力量本源并非修真灵气，而是三类交织的「不死与杀伐」：
1. **凡俗武技**：剑、枪、忍刀、铁炮、火绳枪、火药。苇名军、内府赤备、夜鹰众、贼众皆以此为底。顶尖剑士能以架势崩解与弹刀反杀压倒数量优势，但仍是「人的极限」。
2. **忍术与义手道具**：钩绳、手里剑、喷火筒、斧、伞、烟玉、神隐等。佛雕师（前代只猿）将「忍义手」装上断臂，使狼得以攀城、破盾、破兽。忍术的核心是时机、架势与杀招，而非法力数值。
3. **龙胤与伪不死**：
   - **龙胤（Dragon's Heritage）**：来自西方神龙（樱龙）的血脉恩宠。持有者（现任为少主九郎）可将血赐予他人，使其死后复活；频繁复活会引发「龙咳」，吸食周边生者寿元。
   - **变若水 / 变若之淀**：仙峰寺以龙胤衍生物「变若水」制造伪不死军与实验体；饮者红眼再生，理智与人形逐步崩坏。淀中之虫寄生体可令死者复起。
   - **不死斩**：能真正杀死不死者的刃。红刀「拜泪」由变若卿子持有，后归狼；黑刀「开门」由弦一郎取得，可献祭自身召唤逝者巅峰之姿。
   - **源之香 / 龙泪**：进入源之宫（仙乡）见樱龙的钥匙与断绝龙胤的核心素材。

死亡规则：凡人一刀即死；龙胤受赐者可复活，但会扩散龙咳；不死斩可永久击杀龙胤与伪不死；修罗之路则是杀戮本身吞噬心智。世界顶点不可无限复活——樱龙可被取其泪，剑圣可被斩倒，九郎可选择赴死。

**战力阶梯（以破坏力与战场表现为准，宁低勿高）**：
1. **足轻 / 贼兵 / 乡民**：冷兵器与铁炮的底盘。≈一阶。
2. **苇名精兵 / 夜鹰众 / 内府杂兵**：小队战力、攀城与伏击。≈一~二阶。
3. **名有所出的武士与忍者**（御子上众、七枪级军官、幻影之蝶级）：可单挑据点、弹刀破军。≈二~三阶。
4. **一方强者**（弦一郎前期、大忍枭、佛雕师未堕前、破戒僧、狮子猿）：毁建筑群、屠小队、以名号镇一面。≈三~四阶。
5. **领主级 / 灾厄体**（饮变若水后的弦一郎、怨恨之鬼、头颅狮子猿、真僧、源之宫贵族战力）：可令一城战局倾斜、伪不死再生。≈四~五阶。
6. **国之灾 / 神威侧影**（巅峰剑圣苇名一心、樱龙降临时的神域战场）：一战可定国运、改写局部战场规则（雷、神威、领域感）。≈五~六阶。
7. **本世界顶点**：剑圣一心（终局）与樱龙神威同列本世界可直面的最高战力；仍属「人形战略 / 神域侧」而非爆星。≈六~七阶。

乐园阶位映射：足轻~精兵≈一阶；精锐武士/忍者≈二阶；名物级剑客·忍者≈三阶；一方强者 / 精英 Boss≈四阶；领主·灾厄体≈五阶；剑圣巅峰·神龙侧影≈六阶；本世界顶点对决（剑圣一心终局 / 樱龙相关）≈七阶。宁低勿高；顶点忠于原作；**严禁**把顶点写成「战力归零式削弱」。

**【地理 · 舞台】**
舞台集中于**苇名国**及其神域延伸，由下而上：
- **破庙 / 佛师之所**：狼的据点。佛雕师在此雕佛、装义手；九郎曾藏于此。是情报交换与休整的「井底安全屋」。
- **平田宅邸**（记忆之地）：三年前九郎故居。幻影之蝶与枭在此布局夺龙胤；狼可借香炉铃铛进入「记忆」重走当年火夜。
- **苇名城**：一心盗国后的权力中心。天守、城下、水生邸、城前广场、内城水路构成主战场。弦一郎在此两次与狼决战；终局时内府火箭与火焰喷射器把城池点燃。
- **苇名城下 / 源之川下游**：平民、败兵、龙咳患者与商人混居。永真在此行医；龙咳扩散的人间惨状最直观。
- **仙峰寺（金刚山）**：僧兵与变若实验的温床。折屏猴子、破戒僧、变若卿子、无数失败实验体。不死斩「拜泪」藏于此。
- **水生谷**：狙击枪林立的断崖与洞穴，狮子猿盘踞；通往源之宫的隐秘路径之一。
- **源之宫（仙乡）**：樱龙栖居的神域宫殿。淤加美一族、水生贵族、宫中莲花、栖宿之石皆在此。点燃源之香后方可进入，取龙泪。
- **银草平原**：终局战场。弦一郎饮变若水后再战，并以黑不死斩献祭，唤出年轻剑圣一心。
- **外压**：内府（中央统一势力）从国境压入，赤备军、铁炮队、火焰兵器代表「新时代战争」。

AI 写场景须先定契约者落在哪一层：破庙周边是一阶欺压与龙咳；城下与仙峰外围是二~三阶资源与站队；城内与水生谷是四阶精英战；源之宫与银草是五~七阶神域与终局。

**【世界剧情线】**
① **前史 · 盗国与龙胤降临**  
西方神龙（樱龙）负伤东来，落于苇名高山，化作樱树与神水。其血脉恩宠渗入土地，形成龙胤、变若水与源之宫。苇名一心以「盗国」之战夺取此地，成为剑圣名号的起点。他收养战争孤儿弦一郎为孙，将国运与剑道一并托付。与此同时，大忍枭在战场捡到一名孤儿，取名「狼」，以「忍之铁则」——主人优先于一切——严苛养成。平田家少主九郎继承龙胤，成为当代「龙胤之子」。一心晚年病重，内府虎视，国运如风中残烛。

② **三年前 · 平田宅邸火夜**  
贼众与忍者突袭平田宅邸，目标直指九郎的龙胤。幻影之蝶作为枭安排的「师」、亦是狼的修罗试炼官，主持夺主。狼拼死护主，最终被「未知一刺」（实为假死布局的枭）贯穿；九郎以龙胤之血救活狼，二人主从羁绊自此铸死。宅邸化为焦土，九郎流落，狼在死亡边缘被龙胤绑定——这是「影逝二度」的起源。

③ **开篇 · 城门断臂**  
故事现在时：内府大军压境，一心卧病。弦一郎为续国命，决意夺取九郎龙胤以铸「不死之军」。一心反对邪道，令药师永真传讯，让狼携九郎逃离。弦一郎于城门拦下二人，与狼决斗。无论玩家架势如何，剧本要求狼败：左臂被斩，九郎被掳。佛雕师救下濒死的狼，装上忍义手，令其从「死忍者」重新爬回人间。一心于城中见狼，赐名「只狼」——独臂之狼。

④ **夺回少主 · 城中再战弦一郎**  
狼自破庙出发，穿越城下、水道与天守，与苇名军、夜鹰众、鬼形与不死实验体缠斗。途中可遇不死斩的传闻、仙峰寺的异变、水生谷的兽吼。狼攻上苇名城，于天守再次击败弦一郎。弦一郎饮下变若水残渣，以伪不死逃出生天，身上留下红黑疤痕与「为国不择手段」的执念。九郎被救出后拒绝单纯逃亡——他看见龙咳吞噬生者，决意**斩断龙胤**，求狼成全「不死斩断」之仪式。

⑤ **集齐断绝之具**  
断绝龙胤需要：不死斩、源之香相关素材（宫中莲花、栖宿之石等）、樱龙之泪。狼入仙峰寺，自变若卿子处取得红不死斩「拜泪」；卿子是变若实验唯一较「成功」的幸存者，体内寄宿龙胤碎片，性格清冷悲悯。狼再入水生谷、击败狮子猿（及后来的头颅狮子猿），取得通往神域的钥匙线索。期间可与永真、佛雕师、九郎反复对话，逐步拼出「前代龙胤持有者·丈」与女忍者「巴」曾试图断绝却失败的历史。

⑥ **義父归来 · 铁则与主从**  
大忍枭现身——他并未死于三年前，而是布局至今，要狼遵守铁则：放弃九郎，把龙胤交给「父亲」。此为路线分歧点：
- **弃主从枭**：踏入修罗道。狼于城中斩永真与病中一心，枭则处理弦一郎；最终狼与枭反目相残，狼堕为修罗，九郎绝望旁观，苇名化为修罗修罗场。
- **守主拒枭**：狼于天守斩杀義父枭（亦可在平田记忆中再战「枭之父」形态），铁则被主从之义碾碎。此为通往其余三结局的主干。

⑦ **源之宫 · 见龙取泪**  
守主路线上，狼点燃源之香，进入源之宫。神域水殿、贵族残影、淤加美神女与巨大鲤鱼构成非人风景。狼于樱树之前挑战**樱龙**，取其「龙泪」——这是削弱并最终断绝龙胤的核心。归途时，一心已病逝；内府总攻开始，火箭与火焰喷射器把苇名城烧成白昼。

⑧ **锚点卷 · 苇名城破 / 怨恨之鬼**  
内府赤备军全面入侵，城门崩裂，街巷成焦土。佛雕师（前代只猿）长年以「忍之杀业」雕刻佛像以求赎罪，心中修罗终于破闸，化为**怨恨之鬼**，于战场边缘挥舞火焰与杀意。狼可选择面对这一修罗化身——那是「若不斩断杀业，自己也会变成什么」的镜子。弦一郎于银草平原最后一次拦路：他已彻底饮下变若水，红眼再生，并以黑不死斩「开门」自尽式献祭，召唤出**年轻时的剑圣苇名一心**，把国运赌在祖父巅峰一战上。

⑨ **终局 · 剑圣与四种归宿**  
狼与剑圣一心决战于银草。一心明知弦一郎的邪道不可取，仍以武士之礼全力相迎——他要的是一场无悔的巅峰对决，也是对「盗国者」命运的谢幕。击败一心后，依狼所集之物与选择：
- **不死斩断（Immortal Severance）**：九郎服龙泪，狼以拜泪刺死少主，龙胤断绝；狼失去主从，回到破庙雕佛，接过佛雕师的位置，永真嘱「将来还会有新的忍者来求力量」。
- **复归常人（Purification）**：额外完成窃听、父亲铃铛、记忆中枭战、取得常樱等步骤后，以龙泪+常樱净化九郎，狼自我牺牲，九郎以凡人身份活下去。
- **龙之归乡（Return / Dragon's Homecoming）**：与变若卿子深化羁绊（蛇柿、冰泪等），卿子成为新容器吸纳濒死的九郎之龙胤，狼护送二人向西，将神龙恩宠归还故乡——常被视为最「圆满」的结局。
- **修罗（Shura）**：见⑥，国破、主亡、己堕。

⑩ **微观政治与可介入日常**  
除大决战外，世界高频事件：城下龙咳患者的隔离与药；仙峰寺抓孩做实验；夜鹰众与内府细作的情报战；水生谷猎人的兽材交易；破庙商人的铃钱（战斗经验货币）与牙狼钱；败兵抢劫、乡民逃难、佛像前的忏悔。契约者低阶时优先卷入这些，而不是直接叫板剑圣。

⑪ **蝴蝶效应边界**  
可改：救下某名龙咳患者、提前击退一路赤备、改变怨恨之鬼出现时的战场态势、协助或阻碍某次夺宝。不可无代价改：龙胤必须被「处理」的主题、一心与弦一郎的国运悲剧底色、狼「主从 / 修罗」的核心抉择、樱龙作为神域源头的逻辑。改局部后，因果以追杀、替代牺牲者、更惨烈的城破回流。

**【主要人物】**
- **狼 / 只狼（主角，独臂忍者）**｜性格：寡言、绝对忠诚、杀伐果断；铁则与主从冲突时会露出裂缝；不滥杀无辜却从不回避必要之杀。｜装备·能力：忍刀、忍义手（钩绳及多种忍具）、龙胤复活、后来的不死斩「拜泪」；战斗核心是弹刀、架势破坏与忍杀。｜人物弧光：断臂死忍→夺回少主的只狼→斩義父的主从者→见龙取泪者→剑圣之敌→四结局之一的归宿（雕佛 / 自牺牲 / 西行 / 修罗）。｜立场关系：九郎是主与光；枭是父与影；永真是医者与见证；一心是赐名者与最终试炼；弦一郎是国之执念的敌手。对契约者：先察其是否威胁少主，可用则默许同行，危及主则杀。

- **平田九郎（龙胤少主）**｜性格：早熟、悲悯、坚定；不逃不躲，主动求死以断灾厄。｜装备·能力：龙胤本体——可赐血复活他人、亦是一切争夺的根源；自身近战能力弱，价值在血脉。｜人物弧光：被夺的孩童→看清龙咳真相的觉悟者→四种结局中的牺牲者 / 被净化者 / 归乡的种子。｜立场关系：视狼为唯一可托付的刃；信任永真；畏惧却也理解弦一郎的「为国」；对契约者温和但会要求「勿扩散龙胤」。

- **苇名弦一郎（苇名家主战领袖）**｜性格：刚烈、爱国至偏执、愿以己身与道德为薪。｜装备·能力：太刀与长弓、苇名流剑术、雷引（一心所传）、后期变若水伪不死、黑不死斩「开门」。｜人物弧光：守护国的名将→夺龙胤的罪人→饮变若水的灾厄→献祭唤出剑圣的悲壮棋子。｜立场关系：敬爱一心却违背其道；视九郎为国器而非人；视狼为必须踏过的障碍。对契约者：若助其守国可短暂利用，若护九郎则不死不休。

- **苇名一心（剑圣，苇名开国者）**｜性格：豁达、好酒、好战、厌恶邪道；临终仍保持武士之礼。｜装备·能力：苇名流极致、枪剑切换、风与火的战场压制；终局「剑圣一心」为年轻巅峰态，雷枪剑三相位。｜人物弧光：盗国剑圣→病榻守道者→被孙献祭唤回的巅峰亡灵→银草上的谢幕。｜立场关系：爱弦一郎但不认同不死邪道；欣赏狼并赐名；托永真护走九郎。世界顶点级战力之一。对契约者：有趣则试剑，邪道则斩。

- **药师永真（一心的侍医）**｜性格：冷静、悲悯、藏锋；表面柔和，实则参与断绝仪式的关键设计。｜装备·能力：医术、解毒与龙咳观察、仪式知识；武力低于一线剑客，但情报与道具关键。｜人物弧光：侍医→狼的引导者→修罗道上的牺牲者（若狼堕）→结局的见证与嘱托者。｜立场关系：忠于一心遗志；护九郎；观察狼是否走向修罗。对契约者可交易药品与情报，厌恶扩散龙咳者。

- **佛雕师 / 只猿（前代独臂，忍义手之源）**｜性格：疲惫、悔罪、被杀业啃噬；最终可堕为怨恨之鬼。｜装备·能力：雕刻、改装忍义手、往昔修罗级杀技；堕鬼后火焰与巨刃。｜人物弧光：修罗→赎罪雕佛者→（可选）怨恨之鬼。｜立场关系：视狼为后继与镜子；与「幻影之蝶」等旧日有纠葛。锚点事件的修罗化身。

- **大忍 · 枭（薄井右近左卫门，狼的義父）**｜性格：伪慈、权欲、把铁则当枷锁。｜装备·能力：大忍术、毒、火、巨力、铃铛相关记忆战；可伪装死亡多年。｜人物弧光：养父与师→平田火夜的布局者→夺龙胤的野心家→被狼斩断的「父亲」。｜立场关系：要狼当刃不当人；与幻影之蝶合作；与一心国运线平行争夺。对契约者：可利诱入伙，背叛则杀。

- **幻影之蝶（女忍，平田火夜首领级）**｜性格：妖冶、残酷、以「培养修罗」为乐。｜装备·能力：幻术分身、毒雾、忍刀、操控贼众。｜人物弧光：枭的棋子与狼的试炼官→记忆中的死斗对手。｜立场关系：师徒与猎杀并存；服务枭的夺胤计划。

- **变若卿子（仙峰寺实验幸存者）**｜性格：清冷、悲悯、背负同胞尸山。｜装备·能力：持红不死斩「拜泪」、体内龙胤碎片、与「龙之归乡」相关的容器资质；可赐冰泪等。｜人物弧光：实验体→不死斩授予者→（归乡线）新容器与西行同伴。｜立场关系：托付狼斩断悲剧；与九郎命运镜像。

- **樱龙（神龙，龙胤之源）**｜性格：非人神性，无寻常善恶。｜装备·能力：神域威压、风暴与雷、龙泪；其存在定义源之宫与不死体系。｜人物弧光：负伤东来→被取泪→（归乡线）力量被护送西还。｜立场关系：不站队，是规则本身。高阶战力 / 神域规则体。

- **狮子猿 / 头颅狮子猿**｜性格：兽性、被怨与寄生驱动。｜装备·能力：巨力、恐怖嚎叫、再生（头颅形态更甚）。｜人物弧光：水生谷霸主→被斩仍复起的灾厄。｜立场关系：挡路之兽，亦是通往源之宫的试炼。

- **内府将领与赤备军**｜性格：冷、统一意志、视边陲为必须铲平的钉子。｜装备·能力：组织化铁炮、火箭、火焰喷射器、重甲赤备。｜人物弧光：终局城破的执行者。｜立场关系：苇名公敌；不在乎龙胤哲学，只要灭国。

**【势力图谱】**
- **苇名家**：宗旨——守国与盗国者荣光；地盘——苇名城及周边；代表——一心、弦一郎、七枪与家臣团；战力自足轻至剑圣跨度极大；对内府死战，对龙胤态度分裂（一心拒、弦一郎夺）。
- **内府（中央）**：宗旨——统一天下；地盘——国境之外压入；代表——赤备军官与铁炮队；战力以组织与火器见长；视苇名为必须攻陷的割据。
- **仙峰寺**：宗旨——以变若水触及伪龙胤 / 佛道实验；地盘——金刚山；代表——高僧、破戒僧、变若卿子；外表佛门，内里尸山。
- **忍群（枭一脉 / 夜鹰众等）**：宗旨——任务与铁则；地盘——暗处与屋檐；代表——枭、幻影之蝶、夜鹰；可被雇佣亦可背刺雇主。
- **源之宫淤加美与水生贵族**：宗旨——侍奉樱龙、维系神域；地盘——源之宫；对凡人冷淡，对入侵者杀无赦。
- **破庙中立点**：佛雕师、商人、偶居的九郎与狼；无军力，有情报与改装价值。

敌友逻辑：利益与「国 / 主 / 胤」三字优先；盟约可破；暴露能复活的龙胤之血=被全国追杀。

**【贵重物品】**
- **龙胤之血**：九郎所赐，令狼不死复活；来历为樱龙恩宠；暴露即被弦一郎、枭、仙峰寺争抢；频繁使用扩散龙咳。
- **忍义手**：佛雕师所装，可换装钩绳、斧、喷火、伞、羽、岚文等；是狼战力质变的工程学外挂。
- **不死斩·拜泪（红）**：变若卿子授予；可永杀不死者，亦是断绝仪式用刃。
- **不死斩·开门（黑）**：弦一郎持有；可献祭唤回死者巅峰（剑圣一心）。
- **龙泪**：击败樱龙取得；断绝 / 净化 / 归乡皆需。
- **源之香素材**（宫中莲花、栖宿之石等）：点燃后开启源之宫通路。
- **常樱花瓣**：净化线关键，与记忆中的「父亲的铃铛」等绑定。
- **冰泪 / 蛇柿**：归乡线与卿子羁绊相关。
- **铃钱 / 牙狼钱**：战斗成长与特殊兑换货币；牙狼钱可换复活相关与稀有物。
- **香炉与铃铛**：进入平田记忆的钥匙。

每件写清：谁想要——弦一郎要龙胤，枭要龙胤，仙峰要伪胤，狼要断绝之具；抢到的后果——国运、修罗或救赎；当前阶能否用——低阶拿不死斩易被夺，高阶才能完成仪式链。

**【隐藏剧情 · 伏笔】**
- **前代断绝失败**：丈与女忍巴曾试图斩断龙胤，巴习得引雷（后传至一心与弦一郎线），最终功败，拜泪被封存仙峰。
- **佛雕师的真名与修罗**：佛雕师即前代「只猿」，与「怨恨之鬼」是同一杀业的两面；狼若不自制，即是下一个鬼。
- **枭的假死与一刺**：三年前刺狼者是枭，为夺胤布局；铁则是驯狼的项圈。
- **变若水的伦理黑洞**：仙峰寺以孩童做容器，卿子是唯一接近成功者；「不死」在此是尸山统计。
- **龙咳的社会面**：复活不是个人福利，是对周围村人的吸血；九郎觉悟的核心证据。
- **一心的矛盾**：他盗国而建苇名，却拒绝以邪道续国——「国该亡时则亡」与「孙想救国」的悲剧张力。
- **四结局的主题分裂**：斩断（代价在主）、净化（代价在狼）、归乡（代价在旅途与容器）、修罗（代价在一切）。
- **卡片锚点的结构位置**：城破与怨恨之鬼不是孤立副本，而是终局内府总攻与杀业主题的同一夜。

**【世界剧情线·补详：断臂后的城下与城中】**
狼重返人间后，第一课是「如何用独臂活下去」。破庙外的山道有败兵与山贼，城下有龙咳隔离的草棚，永真在此记录患者与「复活者」的关联。狼若频繁复活，附近村民会咳血死去——这不是氛围点缀，而是龙胤的物理代价。城中水道与天守之间，夜鹰众以屋顶为路，苇名足轻以铁炮封锁阶梯；狼的钩绳把垂直城池变成忍道。此阶段可介入事件：替永真取药、暗杀一名搜捕九郎的夜鹰小头目、护送一名尚未咳血的孩子离开隔离区、或偷听弦一郎家臣关于「不死之军」的军议。名场面颗粒：城门残血未干、义手首次钩住飞檐、第一次因复活而听见远处传来的龙咳。

**【世界剧情线·补详：仙峰寺的佛与尸】**
金刚山仙峰寺外貌香烟缭绕，内里是变若水生产线。僧兵以「成佛」「度化」为名征集孩童，失败体被丢进谷或关进暗室，成功者无限接近「伪龙胤」。折屏猴子以幻术守护寺中秘径；破戒僧以巨力与贪婪守库。变若卿子独居于寺之深，持红刀拜泪，既是实验的「成果」也是实验的控诉者。狼在此不仅取刀，更取「不死该被杀死」的伦理许可。契约者若在此阶段介入：可救一批孩童改写卿子对「外人」的信任、可窃取变若水残方却招致全寺追杀、可协助或阻止某次「灌水」仪式。禁止把仙峰写成普通副本——它是龙胤主题的道德反面教材。

**【世界剧情线·补详：水生谷与兽王】**
水生谷是枪与崖的几何题：内府或苇名的狙击手占据高位，谷底有毒与兽。狮子猿起初是territorial的巨兽，斩首后因怨与虫再度起立，头颅形态更疯、更不死。打通水生谷，等于打通「人域→神域」的脏路。巴与丈的传说在谷中残响：引雷之技、失败的断绝、遗落的信物。契约者可作猎人、向导或夺宝者；与狼抢先杀猿会改变谁先掌握源之宫线索。

**【世界剧情线·补详：源之宫的非人礼仪】**
源之宫不是「更大的城」，而是神域。水殿倒映樱，贵族以舞与刃待客，淤加美侍奉樱龙如侍奉季节本身。宫中莲花与栖宿之石服务于源之香；没有香，人进不了这层。樱龙之战是取泪，不是屠神炫武——龙泪是断绝、净化、归乡三条线的共同燃料。归途时人间已变：一心死、城将破、时代用火箭改写武士的夜。

**【世界剧情线·补详：银草与剑的对话】**
银草平原是终局的舞台设计：开阔、风大、无可躲的政治。弦一郎饮尽变若水，把「爱国」烧成红眼；黑刀开门刺入自身，换年轻一心站起。一心的三相位（剑、枪、踏跃与风）是本世界武技天花板的展示，也是「盗国者」对「守国者」的最后回答：他不同意孙的邪道，但同意给孙一场无愧的赌。狼在此不是复仇者，是把「斩断不死」送进历史的刃。契约者若达此阶，任何插手都必须付出血价，且不能把一心写成虚弱的幻影。

**【主要人物·补详】**
- **不死斩相关的历史人物：丈与巴**｜丈为前代龙胤持有者，巴为女忍，二人试图断绝却失败；巴的引雷之技流入苇名剑术谱系。他们不是路人设定，而是九郎与狼路线的先例与警告：断绝曾失败过，刀曾被封存，代价曾有人付过。
- **折屏猴子与破戒僧**｜仙峰寺的守护与腐化两极；前者以戏法护秘，后者以贪嗔守物。是三~四阶舞台的具名障碍，不是无名小怪。
- **七枪与苇名家臣团**｜弦一郎的武力延伸；在城防与搜捕中执行「国先于人」。契约者在二~四阶最常与之冲突或合作。
- **破庙商人**｜铃钱与牙狼钱的兑换者，中立到近乎薄情；提供成长资源与情报碎片，不站队。

**【势力图谱·补详】**
苇名内部裂成「一心遗志（拒邪道、护九郎离开）」与「弦一郎路线（夺胤续国）」两条命令链，永真是前者的执行神经，家臣团多是后者的肌肉。内府不在乎龙胤哲学，只在乎地图上的苇名是否消失；因此他们会与任何削弱城防的人暂时交易，事后再灭口。仙峰寺与源之宫分别代表「人造不死」与「神授不死」——一条向下堕为尸山，一条向上逸为神域。枭一脉是第三条线：把龙胤当最高级忍器拍卖给野心。

**【贵重物品·补详】**
- **父亲的铃铛 / 香炉**：开启平田记忆，见到幻影之蝶与「枭之父」真相；净化线必需的历史钥匙。
- **牙狼钱**：稀有，可换取与复活、技能相关的关键物；来源多与强敌、头目相关，露富即被盯。
- **雷引相关传书**：连接巴→一心→弦一郎的雷技谱系；四阶以上才有实战意义。
- **源之香整套**：莲花、栖宿之石与九郎之血共同构成；缺一不可进宫，是五阶前后的战略级道具链。

**【隐藏剧情 · 伏笔·补详】**
- **修罗的定义**：不是「变强」，是杀业接管人格；佛雕师、狼、甚至弦一郎的偏执，都在同一光谱上。
- **龙胤的西来**：归乡线揭示力量不属于苇名，苇名只是伤龙的落脚处；「国运」建立在外来神伤上，本就脆弱。
- **内府火器的时代意义**：火箭与喷射器宣告冷兵器英雄史的终结；一心的谢幕同时是一个时代的谢幕。
- **九郎的主动权**：多数从者故事里孩子是被救者；此处孩子是下命令者——「杀我」是主君令。

**【地理 · 舞台·补详】**
破庙到苇名城是「垂直的阶级」：山下咳、城中搜、天守令。仙峰在侧翼抬高道德代价，水生谷在另一侧抬高肉体代价，源之宫在顶上抬高存在代价。银草在城外留白，专门给终局的剑与风。AI 叙事时勿把所有地点同时塞进一阶视野；低阶只见破庙与城下，中阶打开寺与谷，高阶才配见宫与银草。

**【世界定位·补详】**
本世界的冲突不是「谁最强」，而是「不死该不该存在」。弦一郎用国回答，枭用欲回答，仙峰用实验回答，九郎用死回答，狼用刀执行答案。契约者是第五种答案的候选，但世界会用龙咳、城破与修罗反噬任何轻浮的选择。

**【叙事基调 · 雷区】**
基调：残酷浪漫、沉默忠诚、战国腥风与神域鬼气并存；对白少而重，环境叙事多。  
忌把狼写成话痨英雄或滥好人。  
忌把龙胤写成无代价的「外挂复活」。  
忌低阶让剑圣、樱龙满地走；低阶只闻其名、见其余波（龙咳、赤备前锋、夜鹰哨探）。  
忌用「战力归零式削弱」解释一心或樱龙。  
忌把内府写成脸谱魔鬼而忽略其「统一战争」逻辑；也忌美化弦一郎夺孩的爱国。  
最早切入锚点：平田火夜前夜、或城门断臂前后、或内府总攻前的城下混乱。

## 阶位切入点

> 阶位↔原作强度：足轻~精兵≈一阶；精锐武士/忍者≈二阶；名物级剑客·忍者≈三阶；一方强者/精英Boss≈四阶；领主·灾厄体≈五阶；剑圣巅峰·神龙侧影≈六阶；本世界顶点对决≈七阶。世界顶点（剑圣一心、樱龙神威）为高阶存在，低阶切入一律「规避顶点·情报优先」。本世界仅覆盖：一、二、三、四、五、六、七阶。

**一阶（足轻与龙咳 · 破庙井底线）**
切入身份/时点：契约者以逃兵、药跑腿、破庙杂役或流浪武者身份切入，锚定「狼刚装上忍义手、九郎藏于破庙」前后。
初始事件：城下出现一批咳血的村民被苇名兵驱离；有人高价收购「能让人死而复生的血」，夜鹰哨探在屋顶盘问来历。契约者必须在「交出可疑情报」「护送一名龙咳患者逃进山」或「替破庙商人跑一趟铃钱买卖」中抉择。
开场白建议：「破庙的香灰落在你肩上时，远处苇名城还亮着灯。有人在咳，咳到肺叶像破风箱；有人在问你见没见过一个独臂的忍者。你手里那点刚配得上一阶的刀钱，突然变得又烫又轻。」
关键NPC立场：**狼**——沉默，只要你不靠近少主便可共处一庙；**平田九郎**——躲在内间，警惕却不凶；**佛雕师**——要你别吵他雕佛，可给你打磨兵器；**药师永真**——愿意用药换龙咳观察记录；**夜鹰众哨探**——查外人，查松则放，查紧则杀。
主线钩子/支线：九郎藏匿点是否暴露；支线＝送药、探城下兵势、替商人取牙狼钱线索、跟踪夜鹰的传讯路线。
危险度/规避：中——死于哨探、逃兵火并与龙咳恐慌；须规避弦一郎本人、城中武士队与任何「名物」级对手。
任务方向/奖励：活过第一周并摸清「龙胤」传闻真伪；奖励取向＝伤药、低阶忍具残件、通行口令、龙咳笔记。

**二阶（城下精兵 · 内府前哨线）**
切入身份/时点：以佣兵、苇名临时足轻或内府细作（双面）身份切入，锚定「狼开始向苇名城渗透、弦一郎加紧搜捕九郎」阶段。
初始事件：内府前哨与苇名巡逻在城下桥头交火，一名身负密函的夜鹰坠落河沟；密函涉及「少主可能藏在佛所」。契约者捡到密函，立刻成为双方要灭口或要买通的人。
开场白建议：「铁炮的硝烟贴着河面爬。你从死人手里抽出一封还没被血浸透的信——上面写着『佛』与『胤』两个字。桥的两端，都有人朝你抬起了枪。」
关键NPC立场：**狼**——若你不卖九郎，可默许你借道；**苇名弦一郎**的家臣——要密函，要人；**内府斥候**——出更高价买同样的字；**药师永真**——劝你别把孩子推向军阵；**破庙商人**——收脏物、换情报。
主线钩子/支线：密函是否送到弦一郎或内府；支线＝护送伤兵、破坏一座铁炮阵地、潜入水生邸外围探路。
危险度/规避：中高——死于铁炮齐射与忍者暗杀；须规避天守方向的弦一郎、幻影之蝶级老手。
任务方向/奖励：在两国夹缝活成「有用的中间人」；奖励取向＝精制刀装、火药、夜鹰面具、城中地图残页。

**三阶（名物之刃 · 仙峰寺外围线）**
切入身份/时点：以成名浪人、叛出仙峰的僧兵或受雇忍者身份切入，锚定「狼为不死斩与变若真相接近金刚山」前后。
初始事件：仙峰寺外围的村落失踪儿童案爆发，僧兵以「度化」为名强征孩童；一名逃走的实验失败体（红眼、再生）撞进契约者营地，后面跟着要灭口的破戒僧小队。
开场白建议：「山上的钟声被风撕碎。你看见那孩子的眼睛是红的，伤口以可见的速度合拢——而追他的僧兵念着佛号，手里的刃却比强盗更稳。」
关键NPC立场：**变若卿子**——远观，若你护孩则暗中示路；**狼**——同路者，可短暂协作夺「拜泪」线索；**仙峰僧兵头**——视你为窃密者；**佛雕师**——愿听你描述「红眼」，神色难看；**平田九郎**——若得知实验真相会更坚定断绝。
主线钩子/支线：是否助狼取得不死斩情报；支线＝救下一批孩童、潜入藏经处抄变若水配方残页、猎杀失控实验体。
危险度/规避：高——死于僧兵围杀与伪不死再生战；须规避寺深的强敌与任何完整变若水战力。
任务方向/奖励：掌握「伪不死」与「真龙胤」的区别；奖励取向＝寺产秘药、忍具蓝图、不死斩传说抄本、卿子所赠护符。

**四阶（精英 Boss 感 · 天守与水生谷线）**
切入身份/时点：以一方知名剑客、大忍外围、或狼的临时同盟身份切入，锚定「天守再战弦一郎前后 / 水生谷狮子猿为患」。
初始事件：弦一郎在天守集结精锐，下令死守通往少主的所有路径；同时水生谷猎人求援——狮子猿屠尽猎队。契约者被请去「二选一支援」：助城防，或助猎猿开路，两者都通向狼的主线节点。
开场白建议：「天守的旗在风里裂开。有人用雷纹装饰刀鞘，有人用猿骨镶枪柄。你知道自己已经站在『名字会被记进战报』的那一层——死与成名只隔一刀。」
关键NPC立场：**苇名弦一郎**——若你助守城则赐军功，若你助狼则列为斩杀目标；**狼**——要你别伤害九郎，其余不论；**狮子猿**——无交涉的灾厄；**大忍枭**（若已暗中现身）——出价买你背叛；**药师永真**——求你优先救人而非杀名。
主线钩子/支线：天守决战的战场变量；水生谷开路是否提前；支线＝夜袭夜鹰巢、夺雷引笔记残页、保护永真的临时医所。
危险度/规避：高——死于名物级对决与兽王；须规避饮变若水后的弦一郎完全态、剑圣本尊。
任务方向/奖励：在「国」与「主」之间留下自己的刀痕；奖励取向＝名刀、高级忍具、军中特权、兽材与雷纹装备。

**五阶（领主·灾厄 · 城破前夜线）**
切入身份/时点：以可影响一城战局的强者、内府中级将校或苇名侧方面大将身份切入，锚定「内府总攻倒计时、源之香将燃、一心将逝」。
初始事件：内府赤备前锋已炮轰城门；城内出现「怨鬼将生」的流言（佛雕师杀业失控的前兆）；九郎与狼准备点燃源之香。契约者掌握一支机动部队或一件足以改局部战局的道具，必须决定：掩护狼入神域、协助弦一郎做最后夺胤、或给内府开城换位。
开场白建议：「火箭划破夜空时，整座苇名城的影子都在抖。你听见有人在雕佛的木屑里磨刀，也听见有人在天守喊『只要不死，国就不会亡』。你的决定，会写进今夜谁先烧起来。」
关键NPC立场：**狼**——要通路，不要出卖九郎；**苇名弦一郎**——要你拖延内府并交龙胤线索；**内府赤备将**——要你开城；**佛雕师**——已接近失控，对话即是赌博；**变若卿子**——若你倾向归乡线，会托付冰泪相关线索。
主线钩子/支线：源之香点燃时机；怨恨之鬼提前或延后爆发；支线＝疏散城下民、抢运龙泪预备容器、暗杀或保护某一侧传令官。
危险度/规避：极高——死于赤备火器阵列、灾厄体与领主级对决；须规避银草级终局战力、樱龙本体。
任务方向/奖励：在城破之夜留下「可被历史记住」的变量；奖励取向＝军权信物、灾厄素材、神域通行残片、高阶疗伤与抗龙咳方。

**六阶（剑圣侧影 · 源之宫与银草前夜）**
切入身份/时点：以可踏入神域的超凡战力、或终局棋盘上的「第三刃」身份切入，锚定「狼已取龙泪返回、一心已逝、弦一郎将于银草献祭」。
初始事件：源之宫余波未平，龙泪在谁手成为公开秘密；弦一郎派人全城搜「能挡剑圣之刃的人」——他要为献祭后的祖父清场，或找人试刃。契约者被双方同时点名。
开场白建议：「龙泪的冷意隔着盒子都能渗进骨头。银草在风里翻白，像一片尚未染血的海。有人告诉你：接下来那一战，不是为赢国，是为赢一个死法。」
关键NPC立场：**苇名弦一郎**——邀你见证或挡狼；**狼**——邀你勿干涉少主的抉择，或在修罗边缘拉他一把；**剑圣一心**（将临）——只认剑，不认阵营；**药师永真**——求你保证仪式不被第三方污染；**樱龙**（余威）——神域规则仍可能反噬盗泪者。
主线钩子/支线：银草决战是否被第三方改写；支线＝守护龙泪、阻止修罗触发、夺取黑不死斩情报。
危险度/规避：贴近顶点——剑圣与神龙侧影＝高阶巅峰：存在·情报优先/条件性胜利，严禁硬刚本体与战力归零解释；须规避无准备挑战剑圣一心。
任务方向/奖励：获得影响结局分支的关键砝码；奖励取向＝龙泪相关残韵、剑圣流剑术体悟、神域素材、终局站队权。

**七阶（世界级侧翼 · 四结局棋盘）**
切入身份/时点：以本世界顶层强者、可与终局诸刃并立的存在切入，锚定「剑圣战前后、四结局分岔点」。
初始事件：弦一郎已倒，剑圣一心的剑气仍留在银草；九郎、狼、卿子、永真同时站在仪式边缘。契约者手中握有足以把结局从「斩断」推往「净化 / 归乡 / 修罗」的关键物或关键证言（例如常樱线索、冰泪、枭残党的铁则诏、或阻止狼落刀的力量）。
开场白建议：「风停的一瞬，你听见整座苇名在燃烧。孩子说『斩断我』，独臂说『是』，医者说『还有别的办法』，山上传来容器的呼吸。你若开口，历史就会改口。」
关键NPC立场：**平田九郎**——愿死，但也会听「能让更少人死」的方案；**狼**——主命优先，可被净化/归乡逻辑改写；**变若卿子**——归乡的钥匙；**药师永真**——净化线的共谋者；**大忍枭残党**——仍想诱狼入修罗；**剑圣一心**（若残响仍在）——只问你的剑是否无愧。
主线钩子/支线：四结局导向；怨恨之鬼是否被彻底镇灭；内府是否在仪式中途打入银草。
危险度/规避：极高/顶点侧——世界顶点＝剑圣一心与樱龙神威：条件性胜利/情报与布局，严禁硬刚与「战力归零式削弱」；即便同阶也须付出血价。
任务方向/奖励：参与或改写「不死如何结束」；奖励取向＝结局级权柄残片（断绝之证、归乡之路引、修罗名号、剑圣认可的一刀）、顶级忍具/名刀、可带离的叙事性遗产（非无代价龙胤）。
