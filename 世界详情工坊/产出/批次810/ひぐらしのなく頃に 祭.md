# ひぐらしのなく頃に 祭
<!--meta lib=休闲 tiers=休闲-->

## 剧情

**【作品来源】**
《ひぐらしのなく頃に 祭》（寒蝉鸣泣之时·祭）是 Alchemist 于 2007-02-22 发行的 **PlayStation 2** 主机版。在出题·解答本编基础上，强化语音、界面，并追加「祭」特有的追加剧本与多结局要素（含与主机用户向的分支体验）。气质：本编信赖主题 + 主机版「可参与的祭典感」。休闲档案仍以部活与情感为核，追加内容只写其**情感分支**意义。

**【世界定位】**
同一雏见泽，因 PS2「祭」的包装而更像「可反复参拜的夏日祭」：本编八话的情感仍在，额外剧本让玩家在「如果更早说出那句话」的分叉上多走几步。

**【世界观 · 舞台设定】**
舞台是昭和五十年代至五十八年的架空山村**雏见泽**（原型氛围近岐阜白川乡）：人口约两千，御三家（园崎·古手·公由）与村民以「绵流し」祭维系信仰与秩序。表层是蝉鸣、部活、祭典与「大家族」式的人情；内里有水坝抗争遗恨、连续怪死传闻、入江诊所与「雏见泽症候群」等阴影。

休闲档案**只取日常与情感侧**：教室里的惩罚游戏、沙都子的陷阱、祭典的棉花糖、圭一被当成笨蛋的笑声、梨花的「にぱ～☆」、诗音与悟史的青涩、レナ的「かぁいい」、魅音的「都是大姐头的功劳」。超自然与惨剧只作为「考验信赖的压力」出现——主题句永远是作者强调的：**不要短视，要相信伙伴，把烦恼说出口**。

**【地理 · 生活舞台】**
- **雏见泽分校教室／部活室**：圭一、レナ、魅音、沙都子、梨花的日常心脏；惩罚游戏、卡片、跪立是情感润滑。
- **古手神社与祭具殿**：绵流し会场；禁忌与好奇并存的名场面地。
- **园崎家与兴宫**：魅音／诗音身份与家族压力的舞台；安琪·莫特咖啡馆是诗音打工的温柔角落。
- **前原家**：圭一的落脚点；深夜吐槽与崩溃的私密空间。
- **垃圾山／玩具山**：レナ的「宝物」圣地，治愈与危险一体两面。
- **入江诊所**：沙都子与悟史的医疗／人情锚点；入江京介的女仆梗是轻松对照。
- **兴宫与学校外的商店街**：大石的咖喱、葛西的接送、普通人的昭和街景。
- **水库与团地**：祟杀／目明线中「想救沙都子」的绝望与诗音坠落的悲剧舞台——档案写其**情感功能**（保护欲、悔恨、救赎）。

**【故事主线 · 情感线】**
**本编情感线（与 PC 解／出题一致的核）**  
圭一转入雏见泽 → 部活五人的蜜月 → 绵流し与连续怪死传闻 → 各编以不信任收场 → 解篇以目明／罪灭／皆杀／祭囃完成理解与真结局。主题句不变：把烦恼说出口，相信伙伴。

**「祭」作为主机版的情感意义**  
「祭」不只是标题：语音让レナ的哭腔、魅音的笑、梨花的软萌与大人语落差、沙都子的「わ」成为可被「听见」的感情。追加剧本与多结局在体验上提供：  
- **更早介入的选项感**（相对原作一本道，主机版给「站队／说服／陪伴」的参与幻觉）；  
- **角色深度对话**加长，便于攻略向游玩——与レナ散步、听诗音说悟史、陪沙都子等到入江诊所、帮魅音办祭、给梨花买甜食；  
- 部分 BE／IF 强调「一句话没说出口」的悔恨，反衬祭囃し的 HE 有多珍贵。

**推荐情感游玩序**  
先沉浸部活日常 → 经历至少一出题悲剧以理解「不信任的代价」→ 进入目明理解诗音 → 罪灭学会「救」→ 皆杀看到团结 → 祭囃收下真夏日。追加分支作为「若你当时在场」的补完，不取代真结局的信赖胜利。

**情感总弧**  
与《解》相同的主弧，外加主机版「玩家被允许多说几句」的祭典式参与感。

**【可攻略角色 / 主要人物】**
前原圭一（主人公／转学生／部活中心）｜外貌：黑发学生装、阳光眉眼｜性格：嘴炮、重情、怕孤独却爱出风头｜角色类型：元气男主／信赖催化剂｜萌点：被沙都子整、对レナ认真、关键时刻的「相信」｜个人线：从怀疑到成为打破循环的钥匙｜与主角关系：玩家默认同行者；契约者可作转校伴读。

竜宫レナ（同学／部活）｜外貌：橙发短发、水手服、温和笑容｜性格：温柔、照顾人、内心有裂痕与「かぁいい」开关｜角色类型：天然／病气边缘的守护者｜萌点：剁肉刀反差、宝物、叫「圭一くん」｜个人线：罪灭し中被圭一拼死拉回；鬼隐的镜像心结｜与主角关系：最先可深交的温柔位。

园崎魅音（学姐／部活部长／园崎继承人）｜外貌：青绿长发、眼罩式刘海、大姐头气场｜性格：豪爽、腹黑游戏、真正温柔｜角色类型：大姐头／傲娇守护｜萌点：「都是大姐头的功劳」、惩罚游戏、对圭一的在意｜个人线：绵流／目明对照下的家族与恋心｜与主角关系：部活核心、可攻略的可靠学姐。

园崎诗音（魅音双生妹／兴宫）｜外貌：与魅音同脸、发型与气质更「都会」｜性格：任性、毒舌、深情、易走极端｜角色类型：双生另一面／病娇边缘纯爱｜萌点：安琪·莫特、对悟史的恋、对沙都子的复杂爱｜个人线：目明し的悲剧与「想被理解」｜与主角关系：需要被接住而非被审判的人。

北条沙都子（下级生／陷阱专家）｜外貌：双马尾、帽子、元气表情｜性格：好胜、爱整人、外强中干｜角色类型：元气妹／创伤儿童｜萌点：「わ」、陷阱、依赖「圭一」｜个人线：祟杀的保护欲与皆杀中全村站队救她｜与主角关系：想当「哥哥／家人」的对象。

古手梨花（下级生／巫女／循环的旅人）｜外貌：暗紫双马尾、巫女／制服｜性格：表面软萌「みぃ」、内里千年疲惫的大人语｜角色类型：腹黑萝莉／命运承载者｜萌点：にぱ～☆、甜食、突然的大人腔｜个人线：皆杀重燃希望、祭囃与羽入并肩｜与主角关系：最懂「再来一次」的人。

羽入（オヤシロ本体／梨花的伴生）｜外貌：角、紫发、巫女装｜性格：软弱、善良、从旁观到介入｜角色类型：神明系幼驯染／自我牺牲｜萌点：「ですの」、与梨花的抬杠｜个人线：祭囃し中走上舞台｜与主角关系：第四面墙式的共感对象。

北条悟史（沙都子之兄／失踪的少年）｜外貌：柔和少年、球棒意象｜性格：温柔、压抑、过度保护妹妹｜角色类型：不在场的恋人／兄｜萌点：诗音的初恋、沙都子的「にーにー」｜个人线：各编阴影与终局的等待｜与主角关系：透过诗音／沙都子触达的心结。

重要配角：大石藏人（刑警，咖喱与执念）；鹰野三四（护士，真相核心，情感侧写其扭曲的「被承认欲」）；入江京介（诊所所长，女仆梗与救赎）；赤坂卫（暇溃し的公安，父爱与预言）；富竹（摄影师）；葛西（园崎侧）；知惠老师（教室的平常心）。

**【人际关系网 / 社团势力】**
- **部活五人组**：圭一—レナ—魅音—沙都子—梨花；惩罚游戏是爱的语言。
- **园崎双生**：魅音／诗音身份交换与家族压力；悟史是诗音的情感锚。
- **北条家阴影**：铁平／玉枝的虐待 vs 部活与入江的保护网。
- **御三家与村**：お魎、公由喜一郎等「大人的规则」vs 孩子们的信赖。
- **诊所与山狗**：入江—鹰野—富竹的表里；休闲侧只写「谁在利用信赖」。
- **梨花—羽入**：循环与旁观的双人组，最终一起相信伙伴。

**【隐藏剧情 · 真结局 · 伏笔】**
- PS2《祭》是商业移植里程碑，后续 DS《绊》等继续扩展。  
- 追加剧本与 CG 以官方／攻略记录为准；与后日《礼》《奉》篇章有概念重叠处，本档案不把《奉》独占章强行算进《祭》。  
- 真结局精神仍指向祭囃し：赢下六月、迎来可毕业的明天。  
- 语音与音乐强化「日常／惨剧」切变，是情感演出的一部分。

**【氛围基调 · 雷区】**
基调：昭和部活的明亮 × 悬疑致郁 × 催泪的信赖主题。BGM 从轻松到不协和的落差是招牌。NSFW：本编全年龄向血腥恐怖有，情感向不写露骨性描写。  
忌：把寒蝉写成纯斩杀；忌忽略部活日常；忌让角色无理由崩坏；忌「战斗向措辞／对决升级／等级排位」；忌剧透式说教压过「相信」的体验。  
最适切入：圭一转学后的部活蜜月期，或祭典前夜「还笑得出来」的几天。

**【本版焦点】**  
《祭》＝「能听见的雏见泽」。写角色时优先引用可被语音记住的口癖与名场面，强调祭典筹备、部活、咖啡馆等日常舞台。

**【故事主线 · 情感线｜补述：部活作为爱情与信赖的语法】**
雏见泽的「攻略」不是好感度数值，而是你是否愿意在惩罚游戏里输得起、在祭典上帮忙搬道具、在对方眼睛变了的时候仍然叫出名字。圭一吸引人的点在于他会出风头也会道歉；レナ的萌点与恐怖同源——她把「喜欢」活成了整个人生的执念；魅音用大姐头嗓门盖住继承人的孤独；诗音用任性盖住被家族扔掉的孩子气；沙都子用陷阱盖住「不要丢下我」；梨花用にぱ盖住一百年的疲倦；羽入用软弱盖住神明的罪疚。解／祭／粋／奉的差异在载体与收纳，不在「要不要相信」这一句答案。

部活的一局游戏往往比长篇对白更能推进关系：你故意输掉让沙都子笑、你认真听レナ讲垃圾山的宝藏逻辑、你答应魅音「祭典当天听大姐头的」、你给梨花带巧克力、你在诗音嘲笑圭一时没有跟着起哄而是问「悟史是什么样的人」。这些琐事在出题编里会被怀疑污染，在解答编里会成为救命的伏笔——罪灭し的圭正是因为记得「她曾经那么相信我」才冲进教室。

大人角色同样走情感线：大石的咖喱与执念是「没有被好好结束的友情」；赤坂的暇溃し是「工作与家庭之间选错一次就永远的痛」；入江的女仆玩笑下是对孩子们的医学与人情责任；鹰野的恶是「想被世界承认」走到灭绝逻辑的镜面。休闲入世若与他们互动，目标应是听完故事、递出一碗不带条件的热食，而不是审判。

**【可攻略角色｜互动补强】**
与**竜宫レナ**的攻略切入：一起在放学后绕远路，允许她的沉默，不在她提父亲与继母时插玩笑。与**园崎魅音**：接受她的捉弄再私下认真道谢。与**园崎诗音**：兴宫的雨天共伞，只听悟史，不逼她回本家。与**北条沙都子**：当她的陷阱伤到人时先护她再讲理，成为「不会消失的家人」。与**古手梨花**：甜食贿赂有效，但更有效的是在她露出大人语气时回答「我在听」。与**羽入**：承认她的存在本身就是最大好感。与**前原圭一**：不是攻略对象时也可做最强同盟——让他保持「嘴炮但可靠」。

**【地理补强：名场面坐标】**
教室窗边的夕阳＝部活解散的日常；神社石阶＝告白与阴谋的双关；团地顶楼＝诗音线的绝望与自我了断的痛（写痛，不写猎奇）；诊所后院＝沙都子等待兄长的时间静止；水库的广阔＝祟杀线保护欲的空茫；绵流し的河灯＝每年被粉饰的恐惧与美。

**【隐藏与后日谈的情感用法】**
真结局之后的「普通」比高潮更难写：赛杀し问的是幸福是否可定义；Outbreak 问的是绝境里是否还把人当人；神姦し问的是欲望是否能不伤害——《奉》把这些问题「奉」还给玩家。无论哪一版，回到教室继续被罚，仍是最高级的 HE。

雏见泽的「攻略」不是好感度数值，而是你是否愿意在惩罚游戏里输得起、在祭典上帮忙搬道具、在对方眼睛变了的时候仍然叫出名字。圭一吸引人的点在于他会出风头也会道歉；レナ的萌点与恐怖同源——她把「喜欢」活成了整个人生的执念；魅音用大姐头嗓门盖住继承人的孤独；诗音用任性盖住被家族扔掉的孩子气；沙都子用陷阱盖住「不要丢下我」；梨花用にぱ盖住一百年的疲倦；羽入用软弱盖住神明的罪疚。解／祭／粋／奉的差异在载体与收纳，不在「要不要相信」这一句答案。

部活的一局游戏往往比长篇对白更能推进关系：你故意输掉让沙都子笑、你认真听レナ讲垃圾山的宝藏逻辑、你答应魅音「祭典当天听大姐头的」、你给梨花带巧克力、你在诗音嘲笑圭一时没有跟着起哄而是问「悟史是什么样的人」。这些琐事在出题编里会被怀疑污染，在解答编里会成为救命的伏笔——罪灭し的圭正是因为记得「她曾经那么相信我」才冲进教室。

大人角色同样走情感线：大石的咖喱与执念是「没有被好好结束的友情」；赤坂的暇溃し是「工作与家庭之间选错一次就永远的痛」；入江的女仆玩笑下是对孩子们的医学与人情责任；鹰野的恶是「想被世界承认」走到灭绝逻辑的镜面。休闲入世若与他们互动，目标应是听完故事、递出一碗不带条件的热食，而不是审判。

**【可攻略角色｜互动补强】**
与**竜宫レナ**的攻略切入：一起在放学后绕远路，允许她的沉默，不在她提父亲与继母时插玩笑。与**园崎魅音**：接受她的捉弄再私下认真道谢。与**园崎诗音**：兴宫的雨天共伞，只听悟史，不逼她回本家。与**北条沙都子**：当她的陷阱伤到人时先护她再讲理，成为「不会消失的家人」。与**古手梨花**：甜食贿赂有效，但更有效的是在她露出大人语气时回答「我在听」。与**羽入**：承认她的存在本身就是最大好感。与**前原圭一**：不是攻略对象时也可做最强同盟——让他保持「嘴炮但可靠」。

**【地理补强：名场面坐标】**
教室窗边的夕阳＝部活解散的日常；神社石阶＝告白与阴谋的双关；团地顶楼＝诗音线的绝望与自我了断的痛（写痛，不写猎奇）；诊所后院＝沙都子等待兄长的时间静止；水库的广阔＝祟杀线保护欲的空茫；绵流し的河灯＝每年被粉饰的恐惧与美。

**【隐藏与后日谈的情感用法】**

## 休闲切入点

> 本世界为休闲/情感向（寒蝉·祭），核心不是厮杀而是**部活、信赖与悲剧—真相的情感侧**。契约者以日常身份融入雏见泽。

切入身份：转学生同伴／前原家远亲／部活编外人员／兴宫暂时借住的少年少女。最自然是与圭一同期转入，第一周就被罚游戏。

切入时点：PS2《祭》开局：圭一转学后的第一周部活，语音与祭典筹备并行。

初始处境：借住前原家附近的空屋或亲戚家；白天去分校「旁听／转学」，立刻被拉进部活惩罚游戏；认识**前原圭一**、**竜宫レナ**、**园崎魅音**，放学后神社石阶与商店街是固定动线。

开场白建议：「蝉声密得像雨。你站在雏见泽分校门口，看见橙发的女孩笑着说什么「かぁいい」，青绿长发的大姐头已经在发惩罚用的卡片，双马尾的小一号正把水桶挪到门后。有人一把揽住你：「迟到的也算部员！」——你还不知道，这个夏天的笑声，有多值得拼了命去守。」

可攻略对象：
- **竜宫レナ**：一起「宝物」或买零食；吃温柔与认真；心结＝家庭裂痕与被怀疑的恐惧。  
- **园崎魅音**：部活与祭典筹备；接住大姐头的玩笑再认真；心结＝继承人压力与对圭一的在意。  
- **园崎诗音**：兴宫咖啡馆；听她说悟史；心结＝被家族放逐的怒与爱。  
- **北条沙都子**：帮拆／躲陷阱，当可靠的「家人」；心结＝虐待与被抛弃恐惧。  
- **古手梨花**：甜食与神社打扫；听懂软萌下的疲惫；心结＝循环与「这次想赢」。  
- **羽入**（解／奉更清晰）：在梨花身边被「看见」；心结＝从旁观到介入的勇气。

日常玩法钩子：
1. **部活周间赛**：卡片、跪立、沙都子陷阱——用玩乐建立好感。  
2. **祭典筹备**：帮魅音／梨花布置神社，听大人讲「规则」时选择站在孩子们一边。  
3. **兴宫咖啡馆**：听诗音说悟史，不打断、不审判。  
4. **信赖事件**：当谣言与惨剧压力来临，优先「把话问出口、把人拉到阳光下」，推进罪灭／皆杀式情感选择，而非独自用暴力解决。

氛围/雷区：保持昭和部活温度与「相信伙伴」主题；惨剧只作压力测试。忌战斗任务化、忌战斗向措辞／对决升级／等级排位措辞、忌把沙都子／レナ的创伤当噱头。

补充节奏建议：第一周只玩部活与买零食，不碰怪谈；第二周参加祭典筹备，听御三家与「规则」；第三周当压力事件出现时，强制自己执行「先找同伴说」的流程——这是寒蝉情感线的正确通关法。与**园崎诗音**的好感在兴宫比在村里涨得快；与**古手梨花**的好感在神社打扫与甜食；与**北条沙都子**在共同对抗「被大人随意处置」的体验中锁定家人线。若世界进入解／祭囃时点，把**羽入**从「看不见的吐槽役」升级为可对话对象，是情感向特典。

禁忌再强调：不把症候群与山狗写成升级打怪；不把沙都子的创伤写成萌点消费品；不让契约者凭「外来者优越」羞辱村民信仰——要做的是理解、陪伴、把话带到阳光下。
