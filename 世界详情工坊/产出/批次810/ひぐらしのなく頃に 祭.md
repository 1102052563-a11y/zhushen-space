# ひぐらしのなく頃に 祭

## 剧情

**【作品来源】**
《ひぐらしのなく頃に 祭》（寒蝉鸣泣之时·祭）是 Alchemist 于 **2007-02-22** 发行的 **PlayStation 2** 主机版。在出题·解答本编基础上，强化语音、界面，并追加「祭」特有的追加剧本与多结局要素（含主机用户向的分支体验）。气质：本编信赖主题 + 主机版「可参与的祭典感」。休闲档案仍以部活与情感为核，追加内容只写其**情感分支**意义。

**【世界定位】**
同一雏见泽，因 PS2「祭」的包装而更像「可反复参拜的夏日祭」：本编八话的情感仍在，额外剧本让玩家在「如果更早说出那句话」的分叉上多走几步。一句话：**祭典灯笼下，把能说的话提前说完。**

**【世界观 · 舞台设定】**
舞台仍是昭和山村雏见泽与兴宫。主机版用**全语音**让レナ的哭腔、魅音的笑、梨花的软萌与大人语落差、沙都子的「わ」成为可被「听见」的感情。社会氛围：村祭筹备、部活、咖啡馆、诊所、警局咖喱——日常被麦克风放大。超自然与循环设定同本编，档案只写情感侧：分支不是「选对数值」，而是「你有没有在崩溃前开口」。

**【地理 · 生活舞台】**
- **雏见泽分校教室／部活室**：惩罚游戏、卡片、跪立；笑声是整部作品的情感润滑剂。初遇、和好、再结盟都在这里发生。
- **古手神社与祭具殿／石阶**：绵流し会场；禁忌好奇与祭夜名场面。巫女装的梨花、棉流的河面、烟火与蝉鸣。
- **园崎本家与兴宫 Angel Mort**：魅音的当主课与诗音的打工甜笑；女仆装端布丁的温柔角落。
- **前原宅**：转学生的落脚点；深夜吐槽、崩溃与「要不要说出口」的私密空间。
- **垃圾山／玩具山**：レナ的圣地；治愈寻宝与危险一体两面。
- **入江诊所**：沙都子、悟史的医疗与人情锚；「Maid in heaven」玩笑是轻松对照。
- **兴宫商店街／警局附近**：大石的咖喱店、普通人的昭和街景。
- **水库遗址与团地**：保护欲、绝望与诗音悲剧的**情感功能**舞台（写悔恨与救赎，不写猎奇细目）。
- **山道与蝉林**：日常放学路；蝉声一起，气氛就从甜切到怖。
- **圣露西亚相关回忆空间**（诗音线）：束缚与逃出的对照，强调「被当装饰」的心结。

- **PS2「祭」特有情感舞台**：语音回想室般的再体验；追加剧本中的「更长的双人对话」——与レナ散步、听诗音说悟史、陪沙都子等到入江诊所、帮魅音办祭、给梨花买甜食；部分 BE／IF 强调「一句话没说出口」的悔恨，反衬祭囃し HE 有多珍贵。

**【故事主线 · 情感线】**

**本编情感线（与 PC 出题／解一致的核）**  
圭一转入 → 部活五人蜜月 → 绵流し与连续怪死传闻 → 各编以不信任收场 → 解篇以目明／罪灭／皆杀／祭囃完成理解与真结局。主题句不变：把烦恼说出口，相信伙伴。

**「祭」作为主机版的情感意义**  
「祭」不只是标题：  
1. **听见**：语音让名场面从「读」变成「在场」。  
2. **参与幻觉**：相对原作一本道，主机版给「站队／说服／陪伴」的参与感。  
3. **祭典装置**：绵流し的灯笼、烟火、谁拉谁的手，被放在体验中心。

**推荐情感游玩序**  
先沉浸部活日常 → 经历至少一出题悲剧以理解「不信任的代价」→ 进入目明理解诗音 → 罪灭学会「救」→ 皆杀看到团结 → 祭囃收下真夏日。追加分支作为「若你当时在场」的补完，不取代真结局的信赖胜利。

**追加分支的情感类型（概括）**  
- **早开口 IF**：在怀疑爆发前把秘密交给伙伴 → 局部 HE 气味。  
- **晚一步 BE**：同一句话晚一天 → 关系断裂的悔恨。  
- **角色深度加长**：诗音、沙都子、梨花个人戏加厚，便于攻略向。  
- **祭夜特写**：走散与重逢、浴衣、石阶托付。

**情感总弧**  
与《解》相同的主弧，外加主机版「玩家被允许多说几句」的祭典式参与感。

**【可攻略角色 / 主要人物】**
（本版《祭》侧重：语音可感的口癖、祭典名场面、追加分支的「早开口／晚一步」。）

- **前原圭一（转学生／部活核心）**｜外貌：黑发学生装、表情丰富的少年脸｜性格：热血、嘴炮、重义、好奇心强，被戏称「语言魔术师」｜角色类型：元气男主／信赖催化剂｜萌点：惩罚游戏惨败时的夸张反应、对朋友会突然认真到可怕、不会做菜却硬要帮忙｜个人线剧情：从「有趣就上钩」的转学生，被部活五人吞没；各编中经历怀疑、保护欲、崩溃与再选择。攻略关键是**先听完再行动**，不替对方做决定。HE：成为「说出口就能被接住」的人，赢下可毕业的夏天；BE：先下手为强、孤立无援；Bittersweet：救了某人却来不及救全体｜与主角关系：可代入、损友或同班转学生。

- **竜宫レナ／龙宫礼奈（同班／罪灭视点）**｜外貌：橙色短发、水手服、裙子深开叉的个性便装｜性格：温柔照顾人、害羞、认真时眼神骤变；口癖「～吗？～吗？」「かぁいい」｜角色类型：天然治愈／病气阴影的双面｜萌点：垃圾山寻宝、料理上手、对「可爱」暴走｜个人线剧情：心结是茨城时期的家庭阴影与「想被好好当作人」。罪灭し编中因鹰野文件陷入被害妄想，把世界看成敌；圭一以命相搏去救她。攻略＝不嘲笑她的认真、陪她在垃圾山捡「宝物」、在她眼神变了时仍叫她「レナ」。HE：被抱住、放下凶器回到部活；BE：孤立中崩坏；Bittersweet：事情平息后仍怕自己再失控｜与主角关系：同班、可攻略核心。

- **园崎魅音（上级生／班长／部活部长）**｜外貌：青绿长发、便装常带模型枪｜性格：大姐头、护短、不会看气氛、自称「大叔」｜角色类型：元气大姐头／隐藏大小姐｜萌点：「都是大姐头的功劳」、合气道与多才多艺、其实会害羞｜个人线剧情：园崎家下任当主的压力与「想只当魅音」的日常。绵流し相关线中被身份错位与家族规则撕扯。攻略＝把她当普通人一起办祭、一起被沙都子整，而不是只敬畏「御三家」。HE：在部活里被叫名字而不是头衔；BE：被规则吞没、与诗音身份彻底错位的悲剧理解；Bittersweet：守护了村却失去轻松笑容｜与主角关系：部长、青梅式损友。

- **园崎诗音（兴宫／双子／目明视点）**｜外貌：与魅音同貌异发型、Angel Mort 女仆装｜性格：端正、伶俐、占有欲强、爱恨分明｜角色类型：病娇边缘／恋爱偏执｜萌点：咖啡馆甜笑、谈悟史时的柔软、与魅音互换身份的恶作剧｜个人线剧情：圣露西亚式束缚→逃回兴宫→爱上北条悟史→悟史失踪后把恨指向规则与家族。目明し编是理解她的必修。攻略＝认真听她说悟史、不把她当「魅音的复制品」、在她动手前给她台阶。HE（理解向）：被当作诗音本人被爱；BE：电梯与坠落的自我毁灭；Bittersweet：罪被看见却无人能接住｜与主角关系：兴宫打工相遇、可攻略。

- **北条沙都子（下级生／祟杀情感核）**｜外貌：双马尾、元气脸、常设陷阱｜性格：胜气、口癖「～わ」、怕输又不服输｜角色类型：捣蛋妹妹／创伤儿童｜萌点：教室陷阱、讨厌南瓜、分不清西兰花与花椰菜｜个人线剧情：父母亡、叔父铁平归来的虐待阴影；部活是她唯一能笑的地方。祟杀／皆杀线的核心是「谁来保护她」。攻略＝陪她放学、帮她对抗大人而不替她犯罪、记住她怕黑与怕被抛弃。HE：全村站在她一边、叔父被正道处理；BE：被带走或误会杀死信任的人；Bittersweet：哥哥悟史仍未归，但她能睡安稳｜与主角关系：需要兄长式陪伴的可攻略对象。

- **古手梨花（下级生／循环旅人）**｜外貌：紫蓝长发、巫女气质、软萌表情｜性格：外表天真、内里疲惫冷静；口癖「～なのです」「にぱ～☆」「みぃ」｜角色类型：软萌／千年疲备双声部｜萌点：甜食、大人语气落差、与羽入的拌嘴｜个人线剧情：古手末裔、绵流し巫女；死则由羽入送往下一世界。皆杀／祭囃中坦白循环，求伙伴相信。攻略＝买甜食、不把她当吉祥物、在她露出大人眼神时认真回话。HE：与羽入、部活一起赢下六月；BE：又一次独自死去；Bittersweet：赛杀し式「普通」仍难｜与主角关系：可攻略、神隐式倾诉对象。

- **羽入（古手远亲／オヤシロ本体）**｜外貌：角状发饰（一侧伤痕）、巫女服、怯生生｜性格：懦弱害羞、口癖「あぅあぅ」；旁观百年后终于上场｜角色类型：神明系幼馴染／守护者｜萌点：怕辣、爱奶油点心、被梨花腹黑捉弄｜个人线剧情：从「只能看」到「愿意站到舞台上」。祭囃し是她的毕业式。攻略＝请她吃甜点、承认她的存在、不逼她立刻万能。HE：以「古手羽入」转学、与大家一起笑；BE：继续透明旁观；Bittersweet：被看见却仍怕自己无能｜与主角关系：可攻略的超自然侧（只写情感）。

- **北条悟史（失踪的哥哥／诗音心结）**｜外貌：温和少年｜性格：温柔、靠不住的「唔…」、被虐待家庭压垮｜角色类型：缺席的锚｜个人线：为保护沙都子而崩溃失踪；入江诊所地下室的沉睡是真相侧。休闲档案写其**被思念的温度**，不写猎奇。攻略向：通过诗音与沙都子的回忆「见到」他｜与主角关系：间接。

**【人际关系网 / 社团势力】**
- **部活五人组**：圭一—レナ—魅音—沙都子—梨花；惩罚游戏是爱的语言，谁缺席谁就会被世界问责。
- **园崎双生**：魅音／诗音身份交换、お魎的规则、葛西的接送；悟史是诗音的情感锚。
- **北条家阴影**：铁平／玉枝虐待 vs 部活与入江的保护网。
- **御三家与村**：古手·园崎·公由；「大人的规则」vs 孩子们的信赖。
- **诊所与山狗表里**：入江—鹰野—富竹；休闲侧只写「谁在利用信赖、谁在装成朋友」。
- **梨花—羽入**：循环旅人与旁观神，最终一起相信伙伴。
- **大人侧援军**：大石、赤坂、知惠——祭囃し里「肯站队的大人」改变命运重量。
无传统后宫争宠；多角张力来自**谁先被怀疑、谁先被托付秘密**。

**【情感事件 · 名场面】**
1. **转学第一天**：圭一被部活吞没，惩罚游戏的笑声盖过蝉鸣。  
2. **レナ「かぁいい」**：下一秒认真眼神的落差——日常与崩坏的分界线。  
3. **魅音护短**：嘴上「大姐头的功劳」，夜里悄悄收拾烂摊子。  
4. **沙都子陷阱成功**：得意之后夜里的不安，要人陪走夜路。  
5. **梨花にぱ～☆**：软萌后突然的大人语气，像百年疲备漏出来。  
6. **绵流し夜**：棉流河面、祭具殿禁忌、谁拉住谁的袖口。  
7. **Angel Mort**：诗音谈悟史，布丁甜、话却苦。  
8. **罪灭し**：圭一以命相搏，换レナ放下——「相信」第一次被写成行动。  
9. **皆杀**：全村站队保护沙都子，感动压过恐惧。  
10. **祭囃**：羽入从旁观走上舞台，信赖击破「注定的六月」。  
11. **后日谈夏天**：惩罚游戏再次响起，普通比高潮珍贵。  
12. **寒蝉声**：听觉切换键——同一条路，甜与怖只差一声蝉。

另增主机版印象：**祭夜走散与重逢**；**部活全员浴衣**；**神社石阶的告白或托付**；语音里レナ哭腔与魅音护短的呼吸声。

**【隐藏剧情 · 真结局 · 伏笔】**
- 出题／解答对应：鬼隐↔罪灭、绵流↔目明、祟杀↔皆杀；祭囃し为总收束。  
- 梨花循环与羽入＝オヤシロ；贝伦卡斯特尔诗句是更高一层旁白意象（点到为止）。  
- 鹰野动机是「让养祖父研究被承认」的扭曲爱——可理解不可认同。  
- Tips、スタッフルーム补推理，但不替代「相信」主题。  
- PS2《祭》是商业移植里程碑，后续 DS《绊》等继续扩展。追加剧本与 CG 以官方／攻略记录为准；与后日《礼》《奉》篇章有概念重叠处，本档案不把《奉》独占章强行算进《祭》。真结局精神仍指向祭囃し。
