# ひぐらしのなく頃に 粋

## 剧情

**【作品来源】**
《ひぐらしのなく頃に 粋》（寒蝉鸣泣之时·粋）是 Alchemist 于 **2015-03-12** 发行的 **PlayStation 3／PS Vita** 高清合集向作品，整合本编与多项主机版内容，强调「粋（粹）」——把寒蝉的精髓以高清立绘、全语音与便利功能再提示给玩家。气质：回顾式的完整旅途 + 精致化的日常演出。

**【世界定位】**
这是给「想一口气走完从转学到赢下夏天」的玩家的精装雏见泽：情感线完整，演出更清楚，适合以攻略角色与信赖为主的休闲入世。一句话：**把所有碎片读成一个人间，再好好毕业。**

**【世界观 · 舞台设定】**
同一雏见泽，高清重制让教室木窗的纹理、神社石阶的磨损、蝉的背景层更「可住」。全语音让情感名场面不必靠脑补。合集便利（回想、Tips）鼓励玩家像重读书信一样重读「她当时为什么哭」。世界规则与本编一致；档案强调**可沉浸的日常密度**，而不是新黑幕。

**【地理 · 生活舞台】**
- **雏见泽分校教室／部活室**：惩罚游戏、卡片、跪立；笑声是整部作品的情感润滑剂。初遇、和好、再结盟都在这里发生。
- **古手神社与祭具殿／石阶**：绵流し会场；禁忌好奇与祭夜名场面。巫女装的梨花、棉流的河面、烟火与蝉鸣。
- **园崎本家与兴宫 Angel Mort**：魅音的当主课与诗音的打工甜笑；女仆装端布丁的温柔角落。
- **前原宅**：转学生的落脚点；深夜吐槽、崩溃与「要不要说出口」的私密空间。
- **垃圾山／玩具山**：レナ的圣地；治愈寻宝与危险一体两面。
- **入江诊所**：沙都子、悟史的医疗与人情锚；「Maid in heaven」玩笑是轻松对照。
- **兴宫商店街／警局附近**：大石的咖喱店、普通人的昭和街景。
- **水库遗址与团地**：保护欲、绝望与诗音悲剧的**情感功能**舞台（写悔恨与救赎，不写猎奇细目）。
- **山道与蝉林**：日常放学路；蝉声一起，气氛就从甜切到怖。
- **圣露西亚相关回忆空间**（诗音线）：束缚与逃出的对照，强调「被当装饰」的心结。

- **粋的体验舞台**：高清部活室的木光；可反复跳转的「重读她的哭」；合集菜单像把八年旅程装进一只箱子。

**【故事主线 · 情感线】**

**合集的情感旅程（完整主线）**  
1. **出题四编**：鬼隐（圭一的怀疑）、绵流（诗音／魅音错位的刀）、祟杀（保护沙都子的扭曲善意）、暇溃（赤坂与梨花的预言与悔）。每一编都先给足部活甜，再抽走地板。  
2. **解答四编**：目明（理解诗音的爱与恨）、罪灭（圭一救レナ＝救曾经的自己）、皆杀（梨花看见团结）、祭囃（羽入介入、信赖胜出）。  
3. **粋的体验侧重**：高清重制让空间可住；全语音让感情可听；便利功能鼓励重读。合集不是新故事，是**精装的完整人情**。

**角色情感线在合集中的可读性**  
- レナ：从「かぁいい」到罪灭的崩溃与被抱住。  
- 魅音／诗音：大姐头与双生阴影，目明是必修。  
- 沙都子：陷阱喜剧 vs 虐待现实主义，皆杀的全村保护是催泪点。  
- 梨花／羽入：软萌与循环疲备，祭囃是她们的毕业式。  
- 圭一：从嘴炮转学生变成「相信」的实践者。

**与《奉》《祭》的差异（情感定位）**  
- 《粋》＝主机上的精装重温与完整旅途。  
- 《祭》＝PS2 时代「可参与的祭」。  
- 《奉》＝PC 完全收纳＋新作章。  
契约者在《粋》应先当「肯一起放学的人」，不必先当侦探。

**情感总弧**  
蜜月部活 → 多重视点悲剧 → 理解 → 一人不够 → 众人与神同行 → 粋然的夏日（精髓被保存的日常）。

**【隐藏剧情 · 真结局 · 伏笔】**
- 出题／解答对应：鬼隐↔罪灭、绵流↔目明、祟杀↔皆杀；祭囃し为总收束。  
- 梨花循环与羽入＝オヤシロ；贝伦卡斯特尔诗句是更高一层旁白意象（点到为止）。  
- 鹰野动机是「让养祖父研究被承认」的扭曲爱——可理解不可认同。  
- Tips、スタッフルーム补推理，但不替代「相信」主题。  
- 《粋》定位为主机高清合集，内容覆盖本编精髓；具体收录列表以包装／官方为准。与《奉》（PC 完全版＋新章）不同。真结局精神仍是祭囃し的信赖胜利。可对照 MangaGamer 英译文本，语音情感以日版主机为佳。
