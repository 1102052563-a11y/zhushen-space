# MELTY BLOOD Actress Again

## 剧情

**【作品来源】**
《MELTY BLOOD Actress Again》（メルティブラッド Actress Again，简称 MBAA）是 TYPE-MOON 与 French-Bread、Ecole Software 联合开发的《月姬》元续作系列第四作。街机 2008-09-19（NAOMI）；PS2 2009-08-20；后继 Current Code 街机 2010-07-29、PC 2011-12-30、Steam 2016-04-19（Arc System Works 发行）。剧本奈须蘑菇，人设武内崇，音乐 Raito。气质：幻影夏日重演、远野屋敷人情、巷弄同盟、被改写的记忆与救赎。本档案只取**情感与日常一侧**。

**【世界定位】**
《月姬》与前作《Melty Blood》一年后的三咲町：瓦拉几亚之夜本已落幕，人们却忽然觉得「忘了谁」。契约者应把这里当作月姬群像的情感剧场——在屋敷走廊、公园喷泉、教会与巷弄里，选择记得谁、陪伴谁。

**【世界观 · 舞台设定】**
三咲町的夏夜：蝉、车站交叉路口、公园、教会、废弃楼与「开始之丘」。表层是又一次「Hologram Summer」——旧日幻影按剧本重演；深层是新存在「奥西里斯之砂」（另一时间线完全吸血化的西昂）试图把现实封进幻想、把人类炼成永恒记录。休闲侧只强调温度：远野志贵被秋叶／翡翠／琥珀／阿尔奎德／希耶尔拉扯的日常；西昂从「被遗忘」到「被巷弄同盟记住」；莉丝归来后的并肩走路。月之形态在原作是操作差，此处仅作「同一人不同心情的面」之象征，不写数值对决。

**【地理 · 生活舞台】**
- **远野屋敷**：早饭、廊下、琥珀的茶、翡翠的静默、秋叶的兄控命令。  
- **公园／喷泉夜**：阿尔奎德与志贵确认「你还在」的约会感舞台。  
- **教会**：希耶尔的信仰、厨艺与吃醋。  
- **车站交叉路口**：西昂持枪剪影与「路人从不叫她名字」的孤独。  
- **巷弄街角**：西昂·五月·莉丝的同盟生活（Current Code 1.07 后日谈更暖）。  
- **开始之丘／魔女之夜舞台**：与月姬、青子线呼应的情感原点。

**【故事主线 · 情感线】**
**共通：被擦掉的名字**  
参与过前两次幻影夏日的人失去对西昂与部分事件的记忆，像那一年从未发生。被改写者按旧剧本行动；西昂、都古、白莲、青子等察觉违和。情感主轴：**谁还记得西昂？谁愿意在假夏日里伸手？**

**西昂线**  
系列情感主轴。从冷静炼金术师的自我封闭，到承认「想被志贵与莉丝需要」。官方指南将其故事模式标为时间线相关。结局侧写她重新被钉回世界与巷弄同盟。

**莉丝线**  
四年前死去的伙伴以有意识的残留归来。A 线重逢；1.07 追加在西昂与五月结局成立后的街头生活。名场面是终于能并肩走，而不是交锋。

**志贵线**  
被所有人需要的杀生人。在阿尔克、希耶尔、秋叶、女仆、西昂之间重新选择「记得谁」。官方时间线相关。

**阿尔奎德／真祖面**  
日常是找志贵玩与被希耶尔瞪；Archetype: Earth 完整故事写「公主承认孤独与恋心」。红阿尔克是偏执的反面。

**希耶尔／完全武装面**  
说教、咖喱、吃醋；Powered Ciel 独立结局写「武装到极致仍想保护他」。

**秋叶／翡翠／琥珀**  
兄妹与主仆的家庭情感网：命令、无言的茶、笑面的腹黑。制服秋叶强化校园日常侧。

**五月与都古**  
五月补上月姬的亏欠，被同盟当人看；都古以幼时缘要「再比一次」的阳光日常。

**真终局**  
拒绝「灭绝换永恒记录」的假救赎；记住有限却真实的夏日。巷弄三人后日谈是最暖收束。

**情感总弧**  
遗忘 → 重演 → 违和 → 重逢 → 抉择记得谁 → 拒绝假永恒 → 屋敷与巷弄的日常温度。

**【可攻略角色 / 主要人物】**
远野志贵（主人公／远野归人）｜外貌：黑发、学生装、眼的意象｜性格：吐槽、重情、想过普通日子｜角色类型：日常系男主｜萌点：被阿尔克缠、被秋叶骂仍回家｜个人线：在假夏日选择「记得」｜与主角关系：默认同行。

阿尔奎德·布伦史塔德（真祖）｜外貌：金长发红瞳、白裙｜性格：天真任性、执着志贵｜角色类型：天然公主｜萌点：公园夜、食欲反差｜个人线：猎物→恋人；真祖面认孤独｜与主角关系：核心恋慕。

希耶尔（代行者）｜外貌：黑发冷艳｜性格：严谨毒舌、料理好｜角色类型：修女系守护｜萌点：咖喱、吃醋｜个人线：武装面仍护志贵｜与主角关系：救赎与被救赎。

远野秋叶（当主／义妹）｜外貌：黑长直大小姐｜性格：严厉傲娇兄控｜角色类型：妹系支配｜萌点：「哥哥」、红赤朱时的脆弱｜个人线：家庭与血统的情感｜与主角关系：家人线。

翡翠（女仆）｜外貌：青系少表情｜性格：寡言尽职｜角色类型：无口治愈｜萌点：静静端茶｜个人线：守护安稳｜与主角关系：主仆信赖。

琥珀（女仆）｜外貌：茶色笑面｜性格：腹黑、执着秩序｜角色类型：腹黑姐姐｜萌点：茶与药｜个人线：玩笑包装的温柔｜与主角关系：被捉弄也依赖。

西昂·艾尔特纳姆·阿特拉西亚（阿特拉斯）｜外貌：紫发眼镜长靴｜性格：冷静孤独渴望被承认｜角色类型：知性女主／被遗忘者｜萌点：咖啡与毒舌下的认真｜个人线：被遗忘→被记住｜与主角关系：最想被记得的人。

莉丝·拜菲（骑士）｜外貌：淡色长发、圣盾意象｜性格：忠诚直率｜角色类型：骑士百合羁绊｜萌点：叫西昂的名字｜个人线：重逢与巷弄生活｜与主角关系：经西昂连接。

弓塚五月（少女）｜外貌：棕发学生｜性格：善良内向｜角色类型：病气治愈｜萌点：想当普通女孩｜个人线：同盟接纳｜与主角关系：需温柔对待。

有间都古（拳）｜外貌：运动短发｜性格：好胜单纯｜角色类型：元气｜萌点：再比一次｜个人线：童年缘｜与主角关系：幼时羁绊。

配角：莲／白莲、青崎青子、轧间红摩、猫阿尔克系、两仪式（客串）、奥西里斯之砂（终局象征）。

**【人际关系网 / 社团势力】**
远野屋敷网（秋叶—翡翠—琥珀—志贵）；双女主张力（阿尔克↔希耶尔）；巷弄同盟（西昂—五月—莉丝）；教会信仰侧；幻影「旧剧本」触发心结；青子—梦侧轻松对照。

**【情感事件 · 名场面】**
1. 西昂不被叫出名字。2. 公园「你还在」。3. 希耶尔厨房。4. 秋叶廊下晚归训。5. 翡翠的茶。6. 琥珀笑说真相。7. 莉丝重逢。8. 五月被当同伴。9. 都古的阳光约架（玩耍）。10. 1.07 巷弄后日谈。11. 拒绝假永恒。12. 「Everlasting Destiny」式余韵。

**【隐藏剧情 · 真结局 · 伏笔】**
官方指南锚定志贵／莉丝A／西昂／五月等；奥西里斯＝无人接住的西昂镜像；PS2 两仪式与 Boss Rush 等 if；1.05／1.07 补完武装希耶尔与真祖面；巷弄漫画补日常。

**【氛围基调 · 雷区】**
月夜浪漫＋轻度致郁＋主仆甜涩＋巷弄人情。忌对战教科书、忌月形态数值化、忌工具人西昂、忌「战斗向措辞／对决升级／等级排位」。最适切入：假夏日刚起、西昂刚察觉被遗忘时。

**【故事主线 · 情感线｜补述：部活作为爱情与信赖的语法】**
雏见泽的「攻略」不是好感度数值，而是你是否愿意在惩罚游戏里输得起、在祭典上帮忙搬道具、在对方眼睛变了的时候仍然叫出名字。圭一吸引人的点在于他会出风头也会道歉；レナ的萌点与恐怖同源——她把「喜欢」活成了整个人生的执念；魅音用大姐头嗓门盖住继承人的孤独；诗音用任性盖住被家族扔掉的孩子气；沙都子用陷阱盖住「不要丢下我」；梨花用にぱ盖住一百年的疲倦；羽入用软弱盖住神明的罪疚。解／祭／粋／奉的差异在载体与收纳，不在「要不要相信」这一句答案。

部活的一局游戏往往比长篇对白更能推进关系：你故意输掉让沙都子笑、你认真听レナ讲垃圾山的宝藏逻辑、你答应魅音「祭典当天听大姐头的」、你给梨花带巧克力、你在诗音嘲笑圭一时没有跟着起哄而是问「悟史是什么样的人」。这些琐事在出题编里会被怀疑污染，在解答编里会成为救命的伏笔——罪灭し的圭正是因为记得「她曾经那么相信我」才冲进教室。

大人角色同样走情感线：大石的咖喱与执念是「没有被好好结束的友情」；赤坂的暇溃し是「工作与家庭之间选错一次就永远的痛」；入江的女仆玩笑下是对孩子们的医学与人情责任；鹰野的恶是「想被世界承认」走到灭绝逻辑的镜面。休闲入世若与他们互动，目标应是听完故事、递出一碗不带条件的热食，而不是审判。

**【可攻略角色｜互动补强】**
与**竜宫レナ**的攻略切入：一起在放学后绕远路，允许她的沉默，不在她提父亲与继母时插玩笑。与**园崎魅音**：接受她的捉弄再私下认真道谢。与**园崎诗音**：兴宫的雨天共伞，只听悟史，不逼她回本家。与**北条沙都子**：当她的陷阱伤到人时先护她再讲理，成为「不会消失的家人」。与**古手梨花**：甜食贿赂有效，但更有效的是在她露出大人语气时回答「我在听」。与**羽入**：承认她的存在本身就是最大好感。与**前原圭一**：不是攻略对象时也可做最强同盟——让他保持「嘴炮但可靠」。

**【地理补强：名场面坐标】**
教室窗边的夕阳＝部活解散的日常；神社石阶＝告白与阴谋的双关；团地顶楼＝诗音线的绝望与自我了断的痛（写痛，不写猎奇）；诊所后院＝沙都子等待兄长的时间静止；水库的广阔＝祟杀线保护欲的空茫；绵流し的河灯＝每年被粉饰的恐惧与美。

**【隐藏与后日谈的情感用法】**
真结局之后的「普通」比高潮更难写：赛杀し问的是幸福是否可定义；Outbreak 问的是绝境里是否还把人当人；神姦し问的是欲望是否能不伤害——《奉》把这些问题「奉」还给玩家。无论哪一版，回到教室继续被罚，仍是最高级的 HE。

教室窗边的夕阳＝部活解散的日常；神社石阶＝告白与阴谋的双关；团地顶楼＝诗音线的绝望与自我了断的痛（写痛，不写猎奇）；诊所后院＝沙都子等待兄长的时间静止；水库的广阔＝祟杀线保护欲的空茫；绵流し的河灯＝每年被粉饰的恐惧与美。

**【隐藏与后日谈的情感用法】**
真结局之后的「普通」比高潮更难写：赛杀し问的是幸福是否可定义；Outbreak 问的是绝境里是否还把人当人；神姦し问的是欲望是否能不伤害——《奉》把这些问题「奉」还给玩家。无论哪一版，回到教室继续被罚，仍是最高级的 HE。


**【故事主线 · 情感线｜补述：部活作为爱情与信赖的语法】**
雏见泽的「攻略」不是好感度数值，而是你是否愿意在惩罚游戏里输得起、在祭典上帮忙搬道具、在对方眼睛变了的时候仍然叫出名字。圭一吸引人的点在于他会出风头也会道歉；レナ的萌点与恐怖同源——她把「喜欢」活成了整个人生的执念；魅音用大姐头嗓门盖住继承人的孤独；诗音用任性盖住被家族扔掉的孩子气；沙都子用陷阱盖住「不要丢下我」；梨花用にぱ盖住一百年的疲倦；羽入用软弱盖住神明的罪疚。解／祭／粋／奉的差异在载体与收纳，不在「要不要相信」这一句答案。

部活的一局游戏往往比长篇对白更能推进关系：你故意输掉让沙都子笑、你认真听レナ讲垃圾山的宝藏逻辑、你答应魅音「祭典当天听大姐头的」、你给梨花带巧克力、你在诗音嘲笑圭一时没有跟着起哄而是问「悟史是什么样的人」。这些琐事在出题编里会被怀疑污染，在解答编里会成为救命的伏笔——罪灭し的圭正是因为记得「她曾经那么相信我」才冲进教室。

大人角色同样走情感线：大石的咖喱与执念是「没有被好好结束的友情」；赤坂的暇溃し是「工作与家庭之间选错一次就永远的痛」；入江的女仆玩笑下是对孩子们的医学与人情责任；鹰野的恶是「想被世界承认」走到灭绝逻辑的镜面。休闲入世若与他们互动，目标应是听完故事、递出一碗不带条件的热食，而不是审判。

## 休闲切入点

> 本世界为休闲/恋爱向（月姬情感侧）。契约者以**日常身份**融入，核心＝relationship + 屋敷／巷弄日常。

切入身份：三咲町转校生／远野远亲寄宿生／教会临时帮手／巷弄店员。与志贵同龄最易并入夏日游荡；西昂线可设「书信往来的助手」以第一时间发现遗忘异常。

切入时点：Actress Again 开场，幻影夏日再起、莉丝尚未稳定重逢的几天。

初始处境：学生公寓或屋敷客房；白天学园或打工，傍晚公园与车站；起点认识**远野志贵**、**琥珀**、街角的**西昂**。

开场白建议：「三咲的蝉比城里密。你出站时，紫发少女在交叉路口对旧地图，路人走过却无人叫她的名字。公园喷泉旁有金色长发在等人；黑发代行者抱着纸袋走向教会。你忽然觉得——有人正从这世界的记事本上被轻轻擦掉。」

可攻略对象：
- **西昂·艾尔特纳姆·阿特拉西亚**：从「记得她」开始；咖啡与信任；心结＝遗忘与诅咒。  
- **阿尔奎德·布伦史塔德**：公园夜；吃任性；心结＝真祖的孤独。  
- **希耶尔**：教会／学园；夸料理；心结＝信仰与恋心。  
- **远野秋叶**：守时与拆傲娇；心结＝怕失去兄长。  
- **翡翠／琥珀**：家务茶会；安静 vs 接住恶作剧。  
- **莉丝·拜菲／弓塚五月**：巷弄重逢与接纳。

日常玩法钩子：
1. 屋敷家务周与晚饭点名。2. 公园双女主吃醋散步。3. 巷弄找资料、买东西、等莉丝「住下来」。4. 收集「谁还记得西昂」的证言，推动情感收束。

氛围/雷区：月夜浪漫与人情；可轻度致郁但给出口。忌对战任务、忌战斗向措辞／对决升级／等级排位、忌出招表化角色。

补充节奏建议：第一周只玩部活与买零食，不碰怪谈；第二周参加祭典筹备，听御三家与「规则」；第三周当压力事件出现时，强制自己执行「先找同伴说」的流程——这是寒蝉情感线的正确通关法。与**园崎诗音**的好感在兴宫比在村里涨得快；与**古手梨花**的好感在神社打扫与甜食；与**北条沙都子**在共同对抗「被大人随意处置」的体验中锁定家人线。若世界进入解／祭囃时点，把**羽入**从「看不见的吐槽役」升级为可对话对象，是情感向特典。
