## 剧情

**【作品来源】**
《MELTY BLOOD Actress Again》（メルティブラッド Actress Again，简称 MBAA／MBAACC 系）是 TYPE-MOON 与 French-Bread、Ecole Software 联合开发的《月姬》元续作系列第四作。街机 2008-09-19（NAOMI）；PS2 2009-08-20；后继 *Actress Again Current Code* 街机 2010-07-29、PC 2011-12-30（随 *Carnival Phantasm* 限定等）、Steam 2016-04-19 由 Arc System Works 发行。剧本奈须きのこ，人设武内崇，音乐来兎。系列气质：幻影夏日、远野屋敷人情、巷弄同盟、被改写的记忆与救赎。本档案只取**情感与日常一侧**，不写对决数值与胜负评级。

**【世界定位】**
《月姬》与前两度「幻影之夏」落幕后约一年的三咲町。瓦拉几亚之夜本已结束，人们却忽然觉得「忘了谁」。一句话：**在被擦掉名字的夏夜里，选择记得谁、陪伴谁、拒绝用「永恒记录」换虚假救赎。** 契约者应把这里当作月姬群像的情感剧场——屋敷走廊、公园喷泉、教会咖喱香、巷弄咖啡，而不是竞技场。

**【世界观 · 舞台设定】**
时代锚定 2000 年代日本地方都市「三咲町」的盛夏：蝉、燥热、车站交叉路口的霓虹、深夜便利店灯光。表层是又一次「Hologram Summer」——旧日幻影按剧本重演；深层是新存在「奥西里斯之砂／Dust of Osiris」（另一时间线完全吸血化、未曾到访日本的西昂最终形态）试图把现实封进幻想，把人类炼成可永久保存的记录。休闲侧只强调温度与伦理：  
- **被遗忘的痛**：西昂走过车站，路人从不叫她的名字。  
- **被需要的吵闹**：阿尔奎德缠志贵「去玩」、秋叶命令「早点回家」、希耶尔端咖喱时的吃醋。  
- **巷弄同盟**：西昂·弓塚五月·莉丝·拜菲在街角互相确认「我们还在」。  
超自然只服务情感：直死之眼是「看见尽头」的孤独隐喻；月之形态（满月／半月／新月）在档案中仅作「同一人不同心情的面」之象征，不写操作差。世界的治愈感来自：远野家早饭的筷子声、琥珀的茶香、翡翠无言的陪伴、西昂被第一次认真称呼全名时的停顿。

**【地理 · 生活舞台】**
- **远野屋敷**：早饭、廊下、秋叶的兄控命令、琥珀的茶与腹黑玩笑、翡翠静静端药——家庭情感主舞台。  
- **公园／喷泉夜**：阿尔奎德与志贵确认「你还在」的约会感；真祖面承认孤独的长谈。  
- **教会与希耶尔的据点**：说教、咖喱、信仰与吃醋；完全武装面仍想「保护他」的名场面。  
- **车站交叉路口**：西昂持枪剪影与「路人从不叫她名字」的孤独定场。  
- **巷弄街角／Back-Alley**：西昂·五月·莉丝的同盟生活；Current Code 1.07 后日谈更暖。  
- **开始之丘／魔女之夜舞台**：与月姬、苍崎青子线呼应的情感原点。  
- **学校与制服日常**：制服秋叶线的校园侧；都古约「再比一次」的运动场。  
- **废弃楼与幻影重演点**：旧日事件按剧本重演——档案写「像被强迫看自己的录像带」的违和感。  
- **阿特拉斯相关意象（远景）**：西昂的出身与被追捕的孤独，不展开机构对决。  
- **契约者临时落脚**：车站旅馆或远野家客房一侧——能闻到屋敷的味噌汤，也能走到巷弄。

每个场景的情感功能：初遇＝「你记得我吗」；升温＝固定时段的见面与共餐；冲突＝记忆被改写、吃醋、家族命令；和解＝叫出真名；告白＝关灯后的走廊或喷泉边「我想记住的是有限的你」。

**【故事主线 · 情感线】**

**共通线：被擦掉的名字**  
参与过前两次幻影夏日的人，失去对西昂与部分事件的记忆，像那一年从未发生。被改写者按旧剧本行动；西昂、有间都古、白莲、苍崎青子等察觉违和。情感主轴：**谁还记得西昂？谁愿意在假夏日里伸手？** 共通阶段不拼胜负，拼三件事：学会在幻觉里分辨「谁在演戏」、记住每人习惯的称呼、在专业距离（代行者／当主／女仆）与心动之间划线。

**第一阶段·违和**  
蝉鸣比往年更吵。志贵在夜里巡逻时觉得「少了谁」。阿尔奎德仍来缠他，却对某些梗反应空白；秋叶训人时漏掉一句本该记得的事；琥珀的记录本上有被涂掉的名字。西昂独自走在车站，计算「被遗忘」的概率，却第一次算不出答案。

**第二阶段·重演与重逢**  
幻影按剧本重演旧日冲突与重逢。莉丝·拜菲以有意识的残留归来——四年前死去的伙伴再次叫出「西昂」。五月在同盟里被当人看，而不是「失败的样本」。都古甩着「再比一次」冲来，像童年契约未被时间取消。青子与莲／白莲线带来「梦与使魔」的情感对照：谁被当作工具，谁被当作孩子。

**第三阶段·冲突／误会**  
奥西里斯之砂的「永恒记录」诱惑：若人类终将毁灭，把所有人的血炼成记录是否更仁慈？角色分裂为：想拒绝虚假永恒的人、被幻影困住的人、想利用异变达成私愿的人。秋叶的兄控与当主责任冲突；希耶尔的信仰与私心冲突；西昂的理性与「想被需要」冲突。没有传统反派演讲，只有**「被记住」与「被保存为数据」**的伦理拉扯。

**第四阶段·选择记得谁**  
各人故事模式给出不同答案。志贵线：在阿尔克、希耶尔、秋叶、女仆、西昂之间重新选择「记得谁」——官方指南标为时间线相关。西昂线：从自我封闭到承认「想被志贵与莉丝需要」，重新被钉回世界。莉丝线：A 线重逢；Current Code 1.07 追加在西昂与五月结局成立后的街头生活。名场面是**终于能并肩走，而不是交锋**。

**第五阶段·真终局与后日谈**  
拒绝「灭绝换永恒记录」的假救赎；记住有限却真实的夏日。巷弄三人后日谈是最暖收束：咖啡、吐槽、普通地活着。屋敷侧：早饭多一双筷子、秋叶仍命令却耳尖红、翡翠多看一眼、琥珀少下一味怪药。

**各角色路线结局（情感向）**

- **西昂·艾尔特纳姆·阿特拉西亚线**：心结＝被学院抛弃、被世界遗忘、半死徒身份的自我厌恶。攻略＝认真叫全名、请她喝咖啡、承认她的计算里可以有「无理解的感情」。HE＝重新被钉回世界与巷弄同盟；BE＝彻底死徒化／成为奥西里斯式孤独；Bittersweet＝被记住却选择离开三咲。  
- **莉丝·拜菲线**：心结＝「我已死，是否还有资格站在她身边」。攻略＝不把她当道具、让她保护也让她被保护。HE＝与西昂、五月的街头日常；BE＝作为 TATARI 消散；Bittersweet＝只在夏夜幻影中重逢。  
- **远野志贵线**：心结＝杀生人与「想过普通日子」。攻略＝陪他回家吃早饭、不把他当凶器。HE＝在假夏日仍选择有限的人；BE＝被七夜人格或遗忘吞没；Bittersweet＝救了所有人却记不清自己。  
- **阿尔奎德线／真祖面（Archetype: Earth）**：心结＝公主的孤独与「只被当成真祖」。日常是找志贵玩与被希耶尔瞪；真祖面完整故事写「承认恋心与孤独」。HE＝猎物→恋人的确认；BE＝红阿尔克式偏执独占；Bittersweet＝千年城回忆里的独白。  
- **希耶尔线／完全武装面（Powered Ciel）**：心结＝信仰、赎罪与私心。攻略＝吃她的咖喱、听完说教、承认吃醋也是真心。HE＝武装到极致仍想保护他；BE＝只剩代行者职责；Bittersweet＝目送他回屋敷。  
- **远野秋叶线／制服秋叶／赤主面**：心结＝当主、血统、兄控与「想被当妹妹」。攻略＝遵守门禁也认真回「我回来了」、不把命令只当任性。HE＝家庭与恋心两立；BE＝红赤朱失控的恐惧成真；Bittersweet＝送他出门上学的背影。  
- **翡翠线**：心结＝寡言尽职下的「想被需要」。攻略＝接过她的茶、说谢谢、不逼她多话。HE＝主仆信赖升级为被选择的陪伴；BE＝只被当家具；Bittersweet＝廊下擦肩的一瞬笑。  
- **琥珀线**：心结＝用玩笑与怪药包装的温柔与执念。攻略＝拆穿她却不当众羞辱、喝她的茶前先问成分。HE＝腹黑姐姐的真心被接住；BE＝远野家「帝国」玩笑变成孤独；Bittersweet＝留下一瓶无害的新茶。  
- **弓塚五月线**：心结＝月姬的亏欠与「想当普通女孩」。攻略＝同盟接纳、不把她当失败样本。HE＝被当人看的日常；BE＝再次被排除；Bittersweet＝只在巷弄有名字。  
- **有间都古线**：心结＝童年缘与「再比一次」。攻略＝陪练、记住她的胜负欲是亲近方式。HE＝阳光再契约；BE＝被当小孩子打发；Bittersweet＝约好下次却各自长大。  
- **莲／白莲／青子等相关线**：使魔、梦、师徒——档案强调「谁被当工具」。HE＝被当作孩子与伙伴；BE＝被丢回梦的边角。  
- **两仪式等客串向**：点到为止，不抢三咲主舞台。

**情感总弧**  
遗忘 → 重演 → 违和 → 重逢 → 抉择记得谁 → 拒绝假永恒 → 屋敷早饭与巷弄咖啡的日常温度。

**【隐藏剧情 · 真结局 · 伏笔】**
- 奥西里斯之砂＝另一时间线西昂的「人类之血」残留所成；目的是保存灵长类记录。  
- Current Code 1.07：真祖阿尔克、完全武装希耶尔完整故事；莉丝在西昂／五月结局后的巷弄路线。  
- 琥珀记录本涂改、西昂被涂掉的名字、屋敷多出来的咖啡杯——回收时都是温柔炸弹。  
- 与 *Type Lumina* 无故事连续（重制世界线），本档案锚定 **Actress Again／Current Code**。  
- True 情感核：有限、可死、可被忘记的人生，比完美记录更值得选。
