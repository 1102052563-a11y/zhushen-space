# 女子校生監禁日記-365日

## 剧情

**【作品来源】**
《女子校生監禁日記-365日》是轮回乐园休闲库收录的**条目名**（库内检索键保留原题），**并非**对某一真实「监禁未成年人」作品的逐字改编。本档案把题面里的「365日」锚定为：**为期一年、可随时自由退出的自愿共居日记实验**；把「日記」锚定为**共同书写与各自私本并行的记录权**；把「監禁」一词在正文中**彻底改写为封闭感舞台装置**——即「关上门的合租/宿舍式共居」，而不是真实拘禁、绑架或非法限制人身自由的美化。气质上可对照公开可见的「密闭共居」「同居日记」「封闭空间心理」氛围作品（如短篇密室 ADV、共居实验题材的叙事游戏与 VNDB 中 confinement / under-the-same-roof 标签下的成人向作品），但角色一律重写为**大学生与社会人**，舞台一律改写为**大学城寄宿宿舍 / 合租一户建 / 研究型共居实验屋**。搜笔趣阁等长篇站以本条目标题检索无对应出版长篇书页；本条以「一年日记实验 + 同意伦理 + 自由退出」为专属锚点，成人向只记关系结构与同意边界，不写露骨色情。

**【世界定位】**
大学城边缘、一栋可刷卡进出的三层木造寄宿屋「日历寮」。契约者以**实验观察员 / 新入住室友 / 日记整理员 / 伦理监督志愿者**之一的身份入场。一句话：**第 365 个叉之前，笔永远在本人手里；门永远能从里面打开。**

**【世界观 · 舞台设定】**
时代为现代日本地方大学城。没有超能力对决，也没有「真的把人锁死」的犯罪玩法。世界的核心规则不是暴力，而是**协议、日历、日记权、退出权**四件套：

1. **协议书**：所有入住者在入寮前签署《一年共居日记实验知情同意书》。条款明文写：可随时口头/书面退出；退出后 24 小时内协助搬离与资料销毁选项；禁止藏钥匙、断网、藏手机、伪造「不能出门」；任何身体接触必须可撤回的当场同意。
2. **日历墙**：客厅有一面巨大白板日历。每天可划叉，也可**用白笔涂掉重写**——「划叉」只是仪式感，不是刑期。第 365 日不是「必须熬完」，而是「若双方仍愿续签，可开第二年；不愿则体面结束」。
3. **日记权**：每人一本封面不同的私本；另有一本放在茶几上的「共本」。**私本不可被他人擅自翻开**；共本只写双方都同意公开的句子。谁握笔、谁撕页、谁销毁，是本世界的情感进度条。
4. **退出通道**：玄关电子锁对内常开；对外需要密码，但密码贴在冰箱磁贴上（实验故意如此，防止「假封闭」变成真封闭）。邻居、辅导员、伦理热线号码贴在门背后。

世界的温度来自：共用洗衣机的等待、便利店关东煮的蒸汽、窗缝漏进的雨声、笔尖在共本上停住又落下。世界的压迫感只来自**关系里的沉默、误会、羞耻与想逃却又想留下的犹豫**——绝不是来自真正的锁链。若出现「像被关着」的主观感受，叙事必须立刻给出**可执行的出口**（开门、打电话、找邻居、终止协议），并把「选择留下」写成主体决定，而不是强迫结果。

软规则（写作与入世都必须遵守）：
- ① 全员成年：大学生（≥18，默认 20+）/ 研究生 / 社会人；禁止任何未成年外形、年级、校服性化。
- ② 「女子校风」只允许作为**服装风格/怀旧角色扮演**出现在成人角色自愿穿搭中，且必须伴随「这是角色扮演，本人是大学生/社会人」的明确信息；禁止写成真实女高中生。
- ③ 日记本所有权归书写者；偷读私本 = 重大信任危机事件，不是浪漫。
- ④ 可划掉日期重启；可提前终止实验；合法外部介入优先于「为了剧情继续关着」。
- ⑤ 本世界只写日常、关系、心理与同意谈判，不写强弱对决或评级闯关。

**【地理 · 生活舞台】**
- **日历寮·一楼客厅**：白板日历、共本茶几、懒人沙发。几乎所有「今天还要不要继续」的对话发生在这里。墙角有一盆总也养不活的绿萝——实验开始时种下，第几个月终于发芽，是无声的时间标记。
- **日记室（二楼东侧）**：原本是和室，铺了地毯和矮桌。白天是各自写私本的地方；晚上若两人同意，可变成共读共本的空间。台灯色温偏暖，是「安全色」。
- **厨房与狭长餐桌**：泡面、煎蛋、分食便利店便当的主场。谁先洗碗、谁总忘记买牛奶，比任何告白台词更能推进关系。冰箱门上贴着退出热线与实验条款摘要。
- **窗缝与小阳台**：能看见电车尾灯和远处体育馆灯。共看雨、共看初雪、凌晨并排坐着不说话——名场面高发地。阳台门锁坏过一次，修好时两人都松了口气：原来大家潜意识里仍需要「能出去」。
- **玄关**：鞋盒分左右；电子锁对内常开。**真开门**是本世界最重要的画面之一——不是逃狱，是确认「我随时可以走，而我选择回来」。
- **洗衣房 / 浴室前走廊**：等待烘干时的尴尬闲聊；沐浴露香味成为记忆锚。
- **隔壁·邻居千夏（チカ）的家**：围裙、多做的菜、爱多管闲事的好心人。她是**外部眼睛**与合法报信通道，不是「妨碍恋爱的坏人」。
- **大学城咖啡馆「空白页」**：线下网友雪（ユキ）常约在这里。远程支援线的实体化地点；墙上贴满撕下的便签「今天也想活着出门」。
- **学生咨询室 / 区派出所对照点**：不作为「恐怖机关」，而作为**合法出口的现实坐标**。角色可以走进去求助，叙事不得惩罚这种选择。
- **车站前药妆店与 24h 便利店**：买新笔、买生理用品、买关东煮；「笔没水了」这类微观事件从这里开始。

**【故事主线 · 情感线】**
**共通线：谁握住笔，谁保有门**  
实验第 0 日，寄宿屋张贴招募：「一年共居日记实验——记录亲密、边界与日常，可随时退出。」报名者经面谈与冷静期后入住。主角（契约者视角默认）可能是：  
- A. 与日记主人葵（アオイ）同时入住的**对等室友**；  
- B. 中途顶替退出者的**继任室友**；  
- C. 由伦理委员会派出的**观察/整理员**（不参与身体关系，只参与日记与日常）。  

共通前 30 日节奏：熟悉动线 → 制定家务表 → 第一次共本合写 → 第一次差点越界又停住 → 第一次认真讨论「如果有一方想走怎么办」。气质是**慢热心理日常**，不是猎奇。

**阶段一 · 第 1–60 日：划叉与试探**  
葵开始每天在日历上划叉，笔迹发颤。她声称自己是「想用一年证明可以信任陌生人」的社会学系大四生；真实动机线可随路线揭露为：前一段关系里被剥夺过表达权，想用「自己握笔」的方式复健。主角若偷读她的私本，会触发「门锁未坏、心锁先裂」事件：她不会被关住，但会要求重签协议或直接退出。正确推进是：**还笔、道歉、公开销毁偷拍/偷读记录**。此阶段名场面：「把笔推回她手心」。

**阶段二 · 第 61–150 日：共本变厚，私本变轻**  
两人开始在共本写「今天吃了什么」「今天谁先笑」。冲突来自生活摩擦：谁总占浴室、谁把窗关死导致闷热、谁把「我有点累」听成「你很烦」。邻居千夏送来多做的咖喱，顺便问「门锁还好用吗」——这是世界在提醒：外部仍在看，退出仍合法。网友雪在线上发来《逃出清单》改版为《自我保护清单》：充电宝、证件复印件、紧急联系人、终止协议模板。清单不是教人「越狱」，是教人**保有主体性**。

**阶段三 · 第 151–240 日：第 200 叉危机**  
日历上叉越来越多，有人开始把「划叉」误读成「刑期」。葵可能突然把第 200 个叉涂白，写上「今天不算」。这是路线关键分歧点：  
- 若主角把涂白当成背叛 → 信任崩盘，BE 向；  
- 若主角把涂白当成「她在夺回定义权」→ 关系升级，进入深度共情。  
停电夜、假出门（走到车站又回来）、真实开门（买完东西堂堂正正进门）三连事件在此阶段集中出现。

**阶段四 · 第 241–330 日：亲密与边界的精细谈判**  
成人向内容只允许写到：是否牵手、是否同床异被、是否公开关系、是否允许拍摄日常照片。每一次升级必须有**可观察的同意**（点头、复述条款、设安全词、允许随时喊停）。若出现胁迫、下药、藏门卡等行为，世界规则判定为**实验作废 + 角色走法律/咨询路线**，不给「用暴力换好感」的奖励。雪的清单在此阶段多了一条：「喜欢也可以离开；离开不等于失败。」

**阶段五 · 第 331–365 日：空白页还是续签**  
第 365 日不是强制高潮 consummation，而是三种收束可选：  
1. **空白日历 HE**：撕掉旧叉，贴上新的空白月历；两人决定「以后按日子生活，不再用实验框架」。  
2. **续签一年**：重签协议，条款更平等，观察员角色可退出。  
3. **体面终止**：拥抱或握手后分开；日记由葵决定出版为心理互助读物、封存或销毁。  

**アオイ（葵）线**  
核心欲望：夺回叙事权与身体边界的自主。攻略关键不是「把她留住」，而是「让她相信离开也安全，所以愿意留下」。  
- **HE**：日历空白页；她的笔迹从抖到稳；共本最后一页写「门开着，我在家」。  
- **BE**：主角越界读私本/阻止她出门；她合法退出；客厅只剩绿萝与未用完的白板笔。  
- **True 向补充**：日记出版为**警示与互助**而非猎奇；或她亲手销毁，选择不被围观。

**邻居千夏线（外援向，可淡攻略）**  
好心、嘴碎、做菜过量。路线不是把她拉进寮内幽闭，而是让主角学会「接受外部注视」。  
- **HE**：她成为长期邻里支持，实验结束后仍一起逛超市。  
- **危险信号**：把她的关心当成「破坏二人世界」而切断联系——世界会变假封闭，触发伦理警报。

**ラインのユキ（雪）线（远程声援向）**  
只以语音、消息、咖啡馆短会形式出现。她提供清单、冷静期建议、必要时陪同去咨询室。  
- **HE**：成为稳定友人；证明「亲密实验」不需要与世隔绝。  
- **BE**：主角拉黑所有外援，世界气氛转为不健康依赖（系统应提示 OOC/雷区）。

**观察员/伦理监督线（若契约者选此身份）**  
禁止成为「假借研究的加害者」。胜利条件是：记录真实、保护退出权、在越界苗头出现时叫停。可与葵发展信任，但信任建立在**专业边界清晰**上。

**微观日常事件池（入世优先刷这些，而不是一上来高潮）**  
笔没水、泡面口味投票、谁负责垃圾日、绿萝换盆、阳台门锁维修、共本墨水洒了重抄、假出门、真实开门、雨天共伞到车站又各自上课、学期末论文季的作息错位、生日时在日历上画小星而不是叉。

**蝴蝶效应边界**  
契约者可改局部：更早归还偷拿的笔、替葵拒绝不礼貌的访客、帮她预约咨询、坚持把退出条款贴在更显眼处。不可无代价改写：同意原则、自由退出、全员成年、禁止真监禁美化。任何把「门从里面锁死且无法打开」写成浪漫的分支，应被叙事判为 BE/世界崩坏预警。

**总收束**  
本世界的胜利条件不是「关满 365 天」，而是：**在随时能离开的前提下，仍然选择彼此；并让笔与门都回到本人手中。**

**【可攻略角色 / 主要人物】**
- **葵／アオイ（日历寮日记主人·大学四年级·社会学）**｜外貌：黑色短发或低马尾，常穿宽松针织与牛仔裤；若出现「女子校风」制服式穿搭，仅作为成人角色扮演且当场说明本人年龄与身份｜性格：被削过仍倔，表面平静，边界被侵时会突然冷静地终止对话｜角色类型：叙事权核心／慢热自立｜萌点/魅力：夺回笔时的眼神、把叉涂白的勇气、泡面分一半的别扭温柔｜个人线剧情：从「用实验证明信任」到「不需要实验也敢信任」；心结是曾被剥夺表达权｜与主角关系：对等共居者；吃「尊重退出权」与「不把脆弱当把柄」。她的笔迹抖→稳是最可靠的好感与安全感指标；她害怕的不是孤独，是「被写成别人的故事」。

- **契约者视点（可设为同龄大学生／研究生／社会人观察员）**｜姓名外貌自定｜性格成长轴：从偷读/控制欲残留 → 学会请求与等待 → 能主动打开门也能主动回来｜角色类型：介入者／室友／记录协助者｜萌点：会修阳台锁、会把密码磁贴贴回去、会在共本只写对方也同意的句子｜个人线：身份三选一对关系节奏影响很大｜与葵：最优是「笔的归还者」而不是「日历的看守者」。

- **邻人·佐藤千夏（チカ）（隔壁社会人·食堂厨务）**｜外貌：围裙、短发、笑起来有法令纹的松弛感｜性格：多事但心地好，讨厌「年轻人搞些危险封闭」｜角色类型：外部眼睛／生活派｜萌点/魅力：多做的菜、突然出现的雨伞、把伦理热线又贴厚一层｜个人线剧情：从怀疑实验 → 看见同意条款后有限支持 → 在危机日成为报信与收留选项｜与主角关系：外援；吃「诚实」忌「敷衍她的担心」。

- **网友·雪／ユキ（另一所大学心理援助社社员）**｜外貌：线上头像是光斑猫；线下 thrift 外套与圆框镜｜性格：急性子、清单癖、反对任何美化强制的叙事｜角色类型：远程声援／理性制动器｜萌点/魅力：《自我保护清单》、语音留言的气音、咖啡馆「空白页」的固定座｜个人线剧情：从匿名网友 → 线下见一面 → 可成为长期友人；不进入「把她也拉进寮里关一年」的歪曲｜与主角关系：声援与监督双功能。

- **伦理委员会对接人·森崎（社会人·兼职辅导员）**｜外貌：西装外套+运动鞋的割裂感｜性格：公文腔，但关键时刻站退出权一边｜角色类型：制度出口｜个人线：非恋爱向；提供终止协议模板与咨询室预约｜与主角：提醒「实验不是躲法外的帐篷」。

- **前室友·遥（已退出者·仅回忆/电话出现）**｜外貌：不详或旧合影里的长发｜性格：温和、果断｜角色类型：退出先例｜作用：证明「提前离开是被允许的」；她的空床位是契约者可能的入场切口｜与葵：曾经的共本写手，分开后仍互相尊重。

- **便利店夜班·小田（路人温度）**｜总记得葵常买的关东煮口味；在假出门夜给过一句「回去也没关系，明天再来也没关系」。

- **绿萝（非人角色·时间标记）**｜从快死到发芽；谁浇水、谁换盆，成为无台词的关系计量器。

**【人际关系网 / 社团势力】**
- **日记权中心**：葵持有私本最终解释权；共本需双方署名式认可（可简为日期+昵称）。主角不拥有「编辑她人生」的权限。  
- **双外援轴**：千夏（物理邻里）+ 雪（信息与心理清单）。切断双外援 = 高危依赖结构，应被写成问题而非甜蜜。  
- **制度轴**：森崎与咨询室/派出所对照点——存在即威慑「真监禁」写法。  
- **生活共同体**：洗衣、垃圾、购物、电费 AA；比情敌结构更重要。  
- **无「监视者崇拜」**：若出现旧文本里的「监视者侧之声」，本档案一律改写为**不可美化的压力源/可报警对象**，不是攻略目标。  

**【情感事件 · 名场面】**
1. **笔还她**：主角把私本与笔推回葵手心——第一胜，不是征服。  
2. **划掉第 200 叉**：她涂白「刑期幻觉」；他选择相信这是复健。
