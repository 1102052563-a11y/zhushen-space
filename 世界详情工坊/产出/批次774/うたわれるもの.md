# うたわれるもの
<!--meta lib=休闲 tiers=休闲-->

## 剧情

**【作品来源】**
《うたわれるもの》（英译 Utawarerumono / 散りゆく者への子守唄）是 **Leaf / AQUAPLUS** 于 2002 年 4 月 26 日发售的 Windows 成人向视觉小说＋战棋 RPG。制作：下川直哉；监督：鷲見努；剧本：菅宗光；原画：甘露樹。后经 PS2《散りゆく者への子守唄》（2006）、PSP、PS4/Vita HD 重制与 Steam 版等平台移植，成人内容在主机版中删除或改写。同世界观续作有《偽りの仮面》《二人の白皇》，以及《ロストフラグ》《モノクロームメビウス 刻ノ代贖》等派生。2006 年 OLM 动画化 26 话，另有 OVA 与续作动画。气质：**假面男建国史诗 × 兽耳族人情家庭剧 × 责任与牺牲的悲壮恋歌**。本档案按休闲轨道写：**关系、家族、责任、名分与日常**为主，战争只作叙事背景，不写强弱排行或闯关表。

**【世界定位】**
这是一片以阿伊努文化为灵感、兽耳兽尾的「人」族栖居的架空大地。契约者不是来刷副本，而是以旅人、药师学徒、文书、近侍或归乡者身份，走进 **ハクオロ** 从山村走到建国的人情史。一句话：在假面与炉火之间，看一群人如何把「保护」写成国家、把「家人」写成命运。

**【世界观 · 舞台设定】**
地表文明表面是前工业的农耕、铁器、神职与诸国并立：山村交粮、城砦收税、神使调停、商路与战火交替。住民几乎皆有兽耳兽尾（狼、虎、翼人等部族特征各异），语言、祭礼与地名带浓厚北方原住民色彩。超自然元素在本档案中只作 **情感与伦理装置**：森林神兽的死亡带来的哀伤；不可摘下的假面带来的身份悬置；神使血脉带来的「被托付」的重量；远古「神」与人类遗业的真相则埋在后期，用来解释「为什么他必须离开」而非用来开无尽升级。

世界的温度来自三处：① 药草与汤药的气味——**エルルゥ** 的药箱永远比剑更先打开；② 炉边与寝殿的称呼——从「先生」到「父上」到「陛下」，每改一次称呼，关系就重一分；③ 假面下的沉默——**ハクオロ** 不解释过去，只以行动回答「你会不会丢下我们」。社会氛围是「小共同体优先」：山村先救人后问身份；建国后仍把「谁还没吃上饭」放在「谁该跪」前面。成人向原作中的亲密线存在，但本档案严格区分：**成年角色之间的情感与身体可按原作成人向把握；アルルゥ等幼年角色禁止任何性描写，情感仅限慕父、姐妹、被守护的安心。**

**【地理 · 生活舞台】**
- **ヤマウラ（山村）**：故事起点。林缘药圃、木屋、溪边洗衣处、村口粮仓。**トゥスクル** 的药炉与 **エルルゥ** 的药箱是最初的安全屋；假面男在此被命名为 **ハクオロ**。情感功能：收留、命名、第一次被当作家人。
- **山间密林与神兽领地**：药草、陷阱、神兽 **ムティカパ** 的领域。**アルルゥ** 与白虎幼崽 **ムックル** 的羁绊在此扎根。情感功能：失去与收养——「保护」第一次具体到一只小兽与一个孩子。
- **ケナシコウルペ治下的城砦与税道**：旧支配者的收税、驻军与酷吏空间。压迫不是抽象背景，而是粮袋被拿走、亲人被带走的日常。情感功能：共同愤怒如何变成共同责任。
- **新生国トゥスクル的居城与政厅**：**ハクオロ** 建国后的政治中心。文书堆、寝殿、议事厅、训练场边的廊下。**ベナウィ** 在此叹气催批公文；**エルルゥ** 在此递汤药与日程。情感功能：从「一起活」升级为「一起承担别人的活」。
- **オンカミヤムカイ（翼人神使国）的神殿与飞廊**：**ウルトリィ**、**カミュ** 的故乡。白色建筑、经文、祭仪。情感功能：信仰、外交婚姻式的托付、姐妹之间的保护欲。
- **商港与船难海岸**：**カルラ** 被解放的舞台。盐味、酒气、铁枷的冷。情感功能：从「被拥有」到「自己选择站在谁身边」。
- **エヴェンクルガ等部族居地与边境驿路**：**トウカ** 一类武人的出处；也是同盟与背叛交错的走廊。情感功能：忠诚从「主君」迁移到「值得把后背交给的人」。
- **城下祭典广场与温泉/澡堂（日常池）**：建国后的喘息处。假面在酒杯边仍不摘，却有人敢在他面前笑出声。情感功能：权力人物被还原成「家人的一员」。

**【故事主线 · 情感线】**

**开幕阶段 · 假面坠落：被捡到的无名者**
遍体鳞伤的假面男子倒在山间。药师学徒 **エルルゥ** 与妹妹 **アルルゥ**、祖母 **トゥスクル** 将他抬回 **ヤマウラ**。他失忆，假面不可摘，连名字都没有。**トゥスクル** 以已故亲人的名字为他命名——**ハクオロ**。命名即收养：他不是俘虏，也不是神谕，而是「先活下来、再慢慢成为谁」的人。**エルルゥ** 负责换药、喂食、骂他乱动伤口；**アルルゥ** 起初怕生，很快把「这个不会突然消失的大人」当成可以挨着睡的安心源。此阶段的情感主轴是 **单向的被照顾** 与 **假面带来的距离**：他能感受温度，却无法用「完整的脸」回报信任。

**阶段1 · 山村日常：炉火比刀先亮**
伤愈后的 **ハクオロ** 并未急着称王。他帮农事、改灌溉、谈铁器与收成，把「外来者的知识」换成「村里的饭」。**エルルゥ** 从看护者变成搭档：她管药与人的情绪，他管「明天还怎么活」。**アルルゥ** 跟在身后学着叫他；当森林神兽 **ムティカパ** 因外界暴力而惨死后，她收养幼崽 **ムックル**——这是孩子第一次练习「我也可以保护谁」。压迫性的税与暴行逐渐逼近：**ケナシコウルペ** 的统治把山村从「穷」推向「无法再忍」。情感转折不在第一场胜仗，而在 **トゥスクル** 为护人而倒下：祖母的死把「感谢收留」改写成「必须替她护住剩下的人」。

**阶段2 · 起义与建国：名字成为国家**
**ハクオロ** 被推到前头。不是因为他喊得最响，而是因为他能让不同脾气的人站到同一条粮道上。旧日匪气未脱的 **オボロ** 率双剑与忠心加入，称他为兄长般的主；病弱的妹妹 **ユズハ** 与侍从双生射手 **ドリイ／グラア** 一并进入这个扩大的「家」。原属旧体制的冷静武人 **ベナウィ** 与刚猛的 **クロウ** 在认清「苛政没有未来」后倒戈，成为日后最会骂皇帝偷懒却最不肯离开的左右。起义成功后，新国以祖母之名 **トゥスクル** 立号——国名即墓志铭，也是誓言：政权的合法性来自「被保护过的人愿意继续保护别人」。

**阶段3 · 集结家人：盟约写成同居**
建国不是终点。周边诸势力或来犯、或来谈、或来试探。**オンカミヤムカイ** 派来神使 **ウルトリィ**（后幼妹 **カミュ** 也闯入），她们带来信仰与外交的重量：接近 **ハクオロ** 不只是恋爱，更是「神意是否站在此国」。被铁枷束缚的女武者 **カルラ** 在船难与解放后选择留下，巨大黑刀与酒壶是她的语言——她用「我自己决定侍谁」重写奴隶身份。**エヴェンクルガ** 的 **トウカ** 一度因主君被野心家 **ニウェ** 所害而敌对，后在真相与人格吸引下转为近侍式守护，严肃外表下藏着笨拙的柔软。此阶段的情感结构是 **后宫式的多线亲近** 与 **家族式的共居**：每个人都对 **ハクオロ** 抱有不同形状的恋慕或忠诚，但叙事真正黏合他们的，是「在同一屋檐下分担他的沉默」。

**阶段4 · 战争作为背景的人情损耗**
外敌与阴谋不断。本档案不展开军略数值，只写战争对关系的啃噬：谁在出征前把药包塞进怀里；谁在捷报后先问「孩子有没有做噩梦」；谁因政略婚与名分而嫉妒却仍递上情报。**エルルゥ** 的占有欲与吃醋常以喜剧出现，内核却是恐惧——怕这个从死里捡回来的人再次「被更大的命运叫走」.**アルルゥ** 的慕父日益清晰：她要的不是恋人的位置，而是「父上还会回来吃晚饭」。**オボロ** 在妹妹 **ユズハ** 的病体与家族血脉责任之间拉扯；**ベナウィ** 用公文与谏言把「你是皇帝」钉在「你仍是人」之上。每一次同盟、每一次牺牲，都在问：**保护是否必须不断扩大战线，直到保护本身变成无法停手的诅咒。**

**阶段5 · 假面下的真名与终局的摇篮曲**
后期世界观从奇幻滑向远古科技与「神」之本质的揭示：**ハクオロ** 的假面与失忆并非单纯车祸式空白，而是与被传唱的存在 **ウィツァルネミテア** 相关的半身／容器命运。敌对侧存在以严酷「爱」推动争斗的另一半意志（常以 **ディイ** 等形象出现），把「进化需要冲突」与 **ハクオロ**「想让孩子们活在炉火边」的温柔对立起来。终局不是称霸地图，而是 **自我封印式的退场**：他选择停下神之循环，把国与家人托付，自己沉入必须被传唱才能等待归来的沉睡。**エルルゥ** 守社、守约、守「他还会回来」的日常；**アルルゥ** 在父性缺席中长大一截，却仍把那句称呼留在嘴里。曲名与主题句「それは、散りゆく者への子守唄」在此落地——摇篮曲不是催眠敌人，而是给将散之人与留下之人的同一首温柔。

**情感线分述（可攻略／核心关系走向）**
- **エルルゥ线（主ヒロイン · 守护与被守护的对调）**：从看护伤者→行政近侍→恋人／后宫中心。心结是「我能不能成为他摘下面具后仍敢依靠的人」。HE 倾向：确认互选，即使终局分离也以守约完成爱；BE 倾向：只把他当皇帝工具，或被嫉妒烧成不信任。
- **アルルゥ线（慕父 · 严格非恋爱）**：孩子的安心与成长。允许的情感是撒娇、同行、恐惧离别、学会照顾 **ムックル** 与他人。**禁止任何性线、禁止把慕父写成恋慕替代品。**
- **ウルトリィ／カミュ线（神意与姐妹）**：外交式靠近与年幼神使的闯入喜剧；真核是「信仰对象是否仍是人」。
- **カルラ线（解放后的自由选择）**：酒、刀、不跪的自尊；爱是「我留下」而不是「我被收编」。
- **トウカ线（忠诚迁移）**：从旧主亡故的恨到新主值得的守护；告白困难，行动先于言语。
- **オボロ／ユズハ侧（义兄弟与病弱之柔）**：男性义理与对妹妹的守护；**ユズハ** 的病床戏是「弱者也能改变兄长选择」的情感核，戏核在病室而非沙场。
- **后日谈／续作接口**：国仍在，人换代；假面与白皇的余波在《偽りの仮面》《二人の白皇》中以新主角视角回响。本档案以初代建国史为入世主舞台，续作仅作隐藏接口。

**【可攻略角色 / 主要人物】**

**ハクオロ（假面的失忆者→トゥスクル初代皇帝）**｜外貌：高挑成年男性，深色发，不可摘的角状假面遮住面容；常服朴素，战时铁扇｜性格：沉稳、腹黑式坏笑、懒于公文却敏于人心；对弱者极温柔，对「必须流的血」异常果断｜角色类型：假面霸王 / 义理父亲 / 被多方恋慕的中心｜萌点/魅力：不解释过去，只用「还在」回答不安；肯为孩子的晚饭停战一刻｜个人线剧情：被救→改村→起义→建国→集结家人→直面神之半身命运→自我封印以换取炉火长明｜与众人关系：对 **エルルゥ** 是可对等的伴侣候选；对 **アルルゥ** 是父；对 **オボロ** 是兄长式主君；对 **ベナウィ** 是被骂仍被守的皇帝；对女武者们是「值得把自由交给的人」。

**エルルゥ（ヤマウラ药师学徒→皇帝近侍／主ヒロイン）**｜外貌：灰黑系长发，兽耳兽尾，药箱与朴素着物；眼神认真时像要骂人，笑时很软｜性格：温柔却强势、爱吃醋、责任感爆棚；会把「你先喝药」说成最高指令｜角色类型：大和抚子底色的照顾型ヒロイン / 轻度嫉妒喜剧｜萌点/魅力：从单方面看护变成被他需要时的手足无措；吃醋时仍把药煎好｜个人线剧情：拾救假面男→共同生活→随军行的情感升温→名分与后宫压力→终局守约｜与主角关系：最核心的成人向情感轴；攻略关键是尊重她的主体与恐惧，而不是享受她的奉献。

**アルルゥ（エルルゥ之妹 · 幼年）**｜外貌：幼小身形，相似的耳尾特征，常跟白虎幼崽 **ムックル**｜性格：害羞、安静、偶尔顽皮；对认定的人极粘｜角色类型：妹系幼女 / 慕父｜萌点/魅力：轻轻一声「父上」能让朝堂上的假面皇帝瞬间变回山村人｜个人线剧情：怕生→依赖→收养 **ムックル**→在战争阴影下练习勇敢→目送父亲式存在的离去｜与主角关系：**仅家庭与守护**。任何成人向描写均属违规。

**トゥスクル（ヤマウラ长老药师 · エルルゥ／アルルゥ的祖母）**｜外貌：年迈、目光清亮，药炉边的权威｜性格：慈而刚，识人；命名 **ハクオロ** 时已在赌一个未来｜角色类型：祖母 / 精神国母｜萌点/魅力：死前护人的决断让国名有了重量｜弧光：从山村长老成为国之名｜与主角关系：救命恩人与命名者；她的缺席是建国伦理的起点。

**オボロ（双剑浪人→トゥスクル重臣）**｜外貌：精悍，双剑，表情火爆｜性格：冲动、重义、嘴硬心软；对 **ユズハ** 有近乎执念的守护｜角色类型：义弟 / 热血武者｜萌点/魅力：叫「兄者」时比叫「陛下」真｜个人线：匪气→家臣→在病妹与主君之间学会成熟｜与主角关系：最早把「跟着你」说出口的男人之一。

**ユズハ（オボロ之妹）**｜外貌：病弱苍白，常在寝台｜性格：柔和、敏锐、不怨自己的身体｜角色类型：病弱系 / 以柔转刚｜萌点/魅力：一句话能让兄长收剑｜弧光：证明「不能上阵的人也在推动历史」｜与主角关系：被护的家人；情感戏在病室与书信，不在战场。

**ベナウィ（原旧国武将→トゥスクル柱石）**｜外貌：冷静整洁，常皱眉｜性格：逻辑、尽责、毒舌；最受不了皇帝摸鱼｜角色类型：贤臣 / 吐槽役｜萌点/魅力：骂完仍把文书整理好放回案上｜弧光：从体制内清醒者变为「自愿背锅的宰相型」｜与主角关系：最不像家人、却最像「国家理性」的家人。

**クロウ（ベナウィ侧近）**｜外貌：体格强健，寡言｜性格：直、猛、忠｜角色类型：刚猛副手｜功能：用行动翻译 **ベナウィ** 说不出口的义理。

**カルラ（被解放的女武者）**｜外貌：英气，颈上曾有枷；巨刃与酒｜性格：豪放、好酒、厌恶虚伪束缚｜角色类型：女剑豪 / 自由恋人型｜萌点/魅力：刀比誓约先落地，「我留下」三个字比求婚响｜个人线：奴隶→自由人→自选的伴侣／盾｜与主角关系：最不跪、也最敢醉后说真话的人。

**トウカ（エヴェンクルガ武人→近侍守护）**｜外貌：严肃武服，举止偶有笨拙｜性格：认真、死板、内心细｜角色类型：忠犬武者 / 别扭守护｜萌点/魅力：敬礼完美，脸红失败｜个人线：敌对→真相→守护；爱意藏在站岗的位置里｜与主角关系：把「主君」一词重新定义后的人。

**ウルトリィ（オンカミヤムカイ神使）**｜外貌：翼人特征，神职装束，气场清贵｜性格：沉静、虔诚、重责任｜角色类型：神姬 / 外交新娘感｜萌点/魅力：信仰与私情打架时仍先把祭仪做完｜个人线：奉命接近→看见假面下的人→选择站在「人」一边｜与主角关系：把神意翻译成陪伴。

**カミュ（ウルトリィ之妹）**｜外貌：幼于姐的翼人少女，活泼｜性格：淘气、魔力资质高、怕寂寞｜角色类型：神使妹 / 捣蛋治愈｜萌点/魅力：偷渡出嫁式剧情里的笑声｜注意：年龄感偏幼时同样禁止性描写，写姐妹与被保护。

## 休闲切入点

> 本世界为休闲／人情向奇幻家庭史诗，无「靠升级解决一切」的生存闯关压。契约者以**日常身份**融入，核心玩法＝relationship（家人／恋／义理）推进 + 山村与政厅日常，而非任务厮杀。战争只作背景鼓声。

切入身份：推荐四选一——① **ヤマウラ药圃见习**（与 **エルルゥ** 同炉煎药，最自然）；② **トゥスクル政厅见习文书**（被 **ベナウィ** 扔笔墨，近距离看假面皇帝摸鱼）；③ **商路通译／船运书记**（在海岸线遇见 **カルラ** 的自由选择）；④ **神使随行记录者**（随 **ウルトリィ** 使节团入城，吃外交宴与信仰冲击）。身份合理处：新兴国缺人，山村不拒肯干活的外来者；你不是来篡位的，而是多一双能递药、抄粮册、在孩子害怕时讲故事的手。

切入时点：锚定 **命名之后、祖母尚在或刚逝、起义前后到建国初期** 的窗口——**ハクオロ** 已是家人尚未完全是传说；**エルルゥ** 的心意刚从照顾滑向依赖；**アルルゥ** 开始稳定叫出亲近称呼；**オボロ** 已至或将至；**カルラ／トウカ／ウルトリィ** 可陆续登场。此点可改写日常选择，又不必一上来承接终局神恩真相。

初始处境：住在山村木屋阁楼或政厅侧廊小间；一日节奏是清晨帮药圃／抄录 → 午前随 **エルルゥ** 巡诊或随 **ベナウィ** 被骂 → 午后看 **アルルゥ** 与 **ムックル** 在院中打滚 → 黄昏炉边听 **ハクオロ** 用假面后的声音讲「明天先修水渠」。社交起点：先被 **エルルゥ** 规定「生人先洗手再碰药」；被 **アルルゥ** 观察三天后分到半块饼；被 **オボロ** 用酒试胆；被 **ベナウィ** 用错别字羞辱并因此记住你。关键道具不是神器，而是 **药箱钥匙备份、未批粮册、孩子画的假面涂鸦、可共写的守夜值日表**。
