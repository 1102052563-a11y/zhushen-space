# 石世界-觉醒黎明
<!--meta lib=主库 tiers=一-->

## 剧情

**【作品来源】**
《Dr.STONE 石纪元》（稻垣理一郎原作、Boichi作画）于周刊少年Jump连载并动画化多季。核心意象：全人类石化、科学重建文明。本档锚点「**觉醒黎明**」——石化终结后的最早复苏窗口、石神千空苏醒与科学火种初燃。属主库，覆盖阶位：**一**。

**【世界定位】**
公元5738年前后的石化地球。文明归零，石像林立。主角**石神千空**以科学为武器重启人类。一句话：**在全是石头的黎明，先造出第一滴硝酸与第一缕文明烟。**

**【世界观 · 力量体系】**
本世界战斗力来源极克制：人体机能、武器工艺、战术、资源控制；几乎无传统玄幻法术。
1. **未复苏／刚复苏的普通人**：体力差、无工具≈乐园观感**一阶及以下**。
2. **石之世界原住民战士（矛弓）**：部落级武力≈**一～二阶**观感。
3. **科学王国成长期的工程／战斗骨干**：工具与火药优势≈**二阶**。
4. **狮子王司级超人身体能力**：远超常人≈更高阶观感。
5. **石化装置／Why-Man相关**：规则级，**超阶**，一阶绝不可硬碰。
**乐园阶位映射**：刚复苏求生与村级冲突≈**一阶**；王国骨干≈二阶；司≈更高；石化装置／Why-Man＝超阶。本档仅覆盖**一阶**——宁低勿高。

死亡规则：石化可被硝酸配方等科学手段逆转，但误操作、酸液争夺、野兽与原住民冲突均可致死；无无限复活。特殊系统：百物语式科技树、复苏优先级＝政治。

**【地理 · 舞台】**
- **石化平原／石像森林**：开局视觉冲击。
- **千空的秘密基地／树屋实验室**：最早工坊。
- **河边与洞穴**：取水、硝石、实验。
- **石之世界村方向（远景）**：后续人力。
- **杠苏醒点相关废墟**：伙伴轴。
- **海岸线远景**：文明扩张伏笔。
一阶活动半径极小：水源—食料—酸液—安全夜宿四点连线。

**【世界剧情线】**
① **石化之日（前史）** 光罩笼罩全球，人类成石像三千七百年。
② **觉醒黎明（本档核）** 石神千空自力维持意识至复苏；用原始化学制出复苏液；点名科学路线。
③ **杠与「科学之助」** 小川杠等关键伙伴被选择性复苏；性格冲突与信任建立。
④ **生存工业0.1** 陶器、衣、枪的前身、通讯雏形——每一步都是名场面级发明。
⑤ **遭遇原住民世界** 矛与科学的第一面；琥珀、铬等名字进入舞台（时点依进度）。
⑥ **科学王国萌芽** 旗号与制度出现（更高阶档详写）。
⑦ **司的理念对撞（远景）** 一阶只闻其名，不写决战。
⑧ **装置与Why-Man（远景）** 情报碎片级。

阶段结局（一阶窗口）：契约者活过第一个月科学求生，并至少参与一次成功复苏或一次关键发明节点，且未导致配方外泄失控。

**【主要人物】**
石神千空（科学狂人／领袖）｜性格：理性、嘴毒、心怀人类｜装备·能力：头脑与实验｜人物弧光：一人科学→王国｜立场：可用契约者当助手，厌恶反科学。
小川杠（嗓门与心）｜性格：热血、重义｜装备·能力：体力／感召｜弧光：伙伴核｜立场：先信人。
浅雾幻（日后）｜骗术与外交｜一阶或未登场则作情报名。
铬｜村之工匠／矿｜科学搭档。
琥珀｜侦察／战士｜速度与直觉。
钴｜铬之姊相关｜村政。
狮子王司｜理念敌｜一阶仅远景压迫。
金石岚等｜以卷在场为准。
契约者｜乐园投放的「早期助手／漂流复苏者」。

**【势力图谱】**
- **千空小队（黎明期）**：人数极少，科技唯一。
- **石之世界村**：人力与冲突源。
- **司帝国（远）**：武力净化思想。
- **未苏石像**：沉默的政治资源。

**【贵重物品】**
硝酸／复苏液配方与样瓶；原始发电机零件；硝石与硫磺；陶器；石片样本；粮种；记录科学步骤的石板／树皮纸。来历均须科学过程，禁玄学化。

**【隐藏剧情 · 伏笔】**
Why-Man；石化光源机制；千空的时间表与「全人类复苏」野心；杠与千空的幼时羁绊。

**【大事记时间线】**
石化光 → 千年石像 → 千空苏醒 → 制酸 → 复苏杠 → 基础科技 → 接触村 → 王国 → 对司 → 世界真相。
