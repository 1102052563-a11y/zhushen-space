# KiraKira

## 剧情

**【作品来源】**
《KiraKira》（キラ☆キラ）为 OVERDRIVE 2007 年 11 月 22 日发售的 PC 成人向青春恋爱摇滚视觉小说；剧本瀬戸口廉也，原画片仓真二，音乐 milktub。PS2 版《キラ☆キラ 〜Rock'n'Roll Show〜》（2009）、续作/FD《キラ☆キラ カーテンコール》（2008）、移动版等。气质＝**乐队、巡演、贫穷与梦想、残酷与闪耀并存的青春**。本档案聚焦 d2b 四人情感与旅途日常。

**【世界定位】**
ミッション系学校**欧美学园**的**第二文芸部**面临废部。**前島鹿之助**、**椎野きらり**、**樫原紗理奈**、**石動千絵**组成朋克乐队 d2b（第二文芸部バンド），文化祭成功后被名古屋邀约，开破旧面包车开始日本巡演。核心是旅途中的恋爱与「太快活、太年轻死」式青春。

**【世界观 · 舞台设定】**
现代日本：私学欧美学园、部室、文化祭、全国ライブハウス（名古屋·大阪·冲绳·熊本等）。乐器与现场知识真实向（Fender 贝斯、Les Paul、SG、桶当鼓练）。无超自然；残酷来自贫困、家族、疾病、业界。世界的温度在：练习室的汗、服务区的关东煮、女装贝斯手的假发、副声部的和音。

**【地理 · 生活舞台】**
- **欧美学园·第二文芸部室**：起源。  
- **文化祭舞台**：第一次成功。  
- **破旧ワゴン车内**：移动的家，吵架与睡着的名场面。  
- **各地ライブハウス**：名古屋 Panic 冲突、大阪古暮智、等。  
- **きらり打工的店 / 贫家**：债务阴影。  
- **樫原家豪宅与乡下**：祖父正次、女仆斎木。  
- **千絵的家**：家族审判与原谅。  
- **服务区·海边·烟花大会**：告白高发地。

**【故事主线 · 情感线】**
鹿之助退部失恋，遇きらり，与幼驯染部长千絵、大小姐紗理奈组乐队。文化祭后本应解散备考，却因网络直播被邀巡演，在 STAR GENERATION 的八木原、殿谷健太帮助下出发。共通线＝旅途成长与四人羁绊。

**椎野きらり线**  
天才嗓音、贫苦、高语汇却不善交际。Normal 与 True 分叉残酷：债、家火、父、业界面试等（公开资料有 BE 向描写）。True 涉及更沉重的伦理选择与原谅，后日谈她成歌手仍与鹿之助交往。情感核：保护欲与无能感。

**樫原紗理奈线**  
病弱大小姐、芯强、父遗吉他。祖父反对乐队；旅途后禁止通学。海边告白、手机回应、叔父祐司协助、说服祖父；她未必立刻回东京，但关系被承认。情感核：阶层与自由。

**石動千絵线**  
留级幼驯染、部长、鼓手、表面严格内里胆小。父出轨与同父弟弟、母诉讼影响学业。她自幼喜欢鹿之助；在帮助下原谅家庭。情感核：一直在的人。

**后日谈与カーテンコール**  
毕业后きらり职业歌手、千絵福祉、紗理奈会社役员、鹿之助社畜仍触琴；另有二代 d2b 与 HAPPY CYCLE MANIA 故事，可作延伸日常。

**【可攻略角色 / 主要人物】**
- **前島鹿之助 / 女装「前島鹿子」**｜贝斯｜失恋退部、为全女装乐队女装｜类型：成长男主。  
- **椎野きらり**｜主唱兼吉他｜アホ毛、打工、天才嗓｜类型：元气贫民窟天使｜个人线见上。  
- **樫原紗理奈**｜主音吉他｜病弱大小姐、劳斯莱斯通学｜类型：大和抚子芯｜个人线见上。  
- **石動千絵**｜鼓｜泣きぼくろ、部长、不吃兽肉｜类型：幼驯染严格系｜个人线见上。  
- **殿谷健太（トノヤン）**｜STAR GENERATION 吉他、冷淡热音。  
- **村上**｜元网球搭子、围棋部、教朋克、后 HAPPY CYCLE MANIA。  
- **八木原**｜SG 主唱、伯乐。  
- **美雪&舞**｜部员、制服制作。  
- **吉沢先生**｜修女教师。  
- **前島祐子**｜妹、后カーテンコール贝斯。  
- **樫原正次 / 斎木**｜祖父与女仆。  
- **屋代彰 / 古暮智 / 佐藤** 等巡演路人贵人。

**【人际关系网 / 社团势力】**
d2b 四人核心；STAR GENERATION 导师位；家族（贫/富/裂）为阻力。无社团斗争，只有「能不能继续响」。

**【情感事件 · 名场面】**
1. 部室决定组乐队。  
2. 文化祭一曲终了的耳鸣式安静。  
3. 假发与女装第一次上台。  
4. ワゴン车抛锚的夜。  
5. 烟花下きらり「可以交往」的预告。  
6. 大阪的吉他与「サトっち」。  
7. 紗理奈海边与电话。  
8. 千絵鼓棒敲在鹿之助肩上的告白。  
9. 各线 ED 曲情绪（a song for... / travelers / go on a trip）。  
10. カーテンコール 的重逢演出。

**【隐藏剧情 · 真结局 · 伏笔】**
- きらり True 需看完三条 Normal。  
- 鹿之助生父早逝。  
- 紗理奈父亦爱摇滚。  
- 现实 CD/直播与虚构乐队连动是作品外彩蛋。

**【氛围基调 · 雷区】**
青春残酷物语+摇滚高扬；きらり线有致郁与伦理重量。NSFW18 从略。忌：把巡演写成无压力旅行；回避贫困与病；把女装当纯搞笑忽略羞耻与团结。最适切入：废部决定后；文化祭次日；出发前夜的部室。

**【巡演情感地理】**
名古屋：被视觉系嘲笑女装→四人更抱团。大阪：贵人与旧伤。冲绳：海与外国人同事。熊本：叔父与家族史。每一站都让某一条路线的心结露出一角。公路上的沉默比舞台上的嘶吼更决定恋爱分叉：谁坐副驾、谁负责点歌、谁在服务区先下车买关东煮，都是好感的刻度。

**【四人关系力学】**
きらり是太阳，紗理奈是月，千絵是地，鹿之助是接线。攻略一人时，其余三人仍是镜子——忽略乐队整体会让个人线变假。文化祭后的解散危机、巡演中的资金见底、家族电话的突然接入，都会把「喜欢」逼成「是否还一起走下去」。

**【きらり线情感细述】**
她的语汇像词典，对话却常卡壳；她把打工排满，只为让家里少一点催债的声音。Normal 线的残酷在于：爱无法直接偿债，舞台的光也照不进厨房。True 线要求玩家先走完其他路线，理解「保护」有时是僭越，有时是共犯，有时是一起活下去的唯一方式。后日谈里她在舞台上发光，鹿之助在台下当学生——关系没有童话化，但还在。

**【紗理奈线情感细述】**
劳斯莱斯与部室的对比是笑点也是刀。她病弱却倔，吉他是亡父的形状。祖父正次代表「阶层的正确」，叔父祐司代表「血脉里的摇滚」。海边告白与电话回应拆开空间：爱可以隔着乡下与东京。结局不是立刻自由，而是被承认「可以喜欢这个人」。

**【千絵线情感细述】**
她留级的理由写在家庭法庭的影子里。鼓棒是命令也是心跳。她从小看着鹿之助恋爱、失恋、女装、出丑，仍把位置留在他侧后。路线高潮是原谅父与「另一个孩子」——不是为父，是为让自己继续敲得下去。

**【日常可扮演密度】**
调音十分钟的对视；安可前的手汗；便利店关东煮分食；修女教师吉沢的沉默许可；美雪与舞缝制服时的闲话；村上教「什么是朋克」时的笨拙认真。这些场景不推进大事件，却决定好感能否从「队友」变成「恋人」。

**【きらり线情感细述】**
