## 剧情

**【作品来源】**
1992年美国科幻恐怖电影《Alien 3》（风格化 *Alien³*，中文通译《异形3》，港译《异形三集》），二十世纪福斯发行，Brandywine Productions 制作。导演大卫·芬奇（David Fincher，长片处女作），剧本由大卫·吉勒（David Giler）、沃尔特·希尔（Walter Hill）、拉里·弗格森（Larry Ferguson）执笔，故事概念源自文森特·沃德（Vincent Ward）；角色源自丹·奥班农（Dan O'Bannon）与罗纳德·舒塞特（Ronald Shusett）开创的《异形》宇宙；监制戈登·卡罗尔（Gordon Carroll）、吉勒、希尔；配乐艾略特·戈登塔尔（Elliot Goldenthal）；摄影亚历克斯·汤姆森（Alex Thomson）。主演雪歌妮·韦弗（Sigourney Weaver）饰艾伦·里普利（Ellen Ripley），查尔斯·S·达顿（Charles S. Dutton）饰伦纳德·狄龙（Leonard Dillon），查尔斯·丹斯（Charles Dance）饰乔纳森·克莱门斯（Jonathan Clemens），布莱恩·格洛弗（Brian Glover）饰哈罗德·「哈里」·安德鲁斯（Harold "Harry" Andrews），拉尔夫·布朗（Ralph Brown）饰弗朗西斯·亚伦（Francis Aaron，外号「85」），保罗·麦甘（Paul McGann）饰沃尔特·高力（Walter Golic），丹尼·韦伯（Danny Webb）饰罗伯特·摩斯（Robert Morse），兰斯·亨利克森（Lance Henriksen）声演残破毕晓普并饰「毕晓普二号/迈克尔·毕晓普」。本片为系列第三部正传，紧接《异形2》苏拉科号返航，把「陆战队火力战」收束为「无枪监狱·宗教末世·宿主自焚」。文风：芬奇式阴冷、宗教忏悔、工业熔炉、企业黑深残。院线约114分钟，2003年汇编剪辑版（Assembly Cut）约145分钟（牛宿主等差异，终局骨架一致）。本档案服务轮回乐园世界书常青读取，剧情与人名以公开电影、中英维基及 Xenopedia 可核为准。

**【世界定位】**
公元2179年前后：艾伦·里普利与纽特、希克斯、半毁毕晓普在苏拉科号休眠返航途中，舰内异形卵孵化触发火警；逃生舱坠落**菲奥里纳「狂怒」161（Fiorina "Fury" 161）**——一座铅矿铸造厂兼最高设防C级男监，住满有反社会倾向、已皈依「末日宗教」的囚犯。纽特溺亡、希克斯被支架刺死；里普利独活，却发现狗（院线）/牛（汇编剪辑）宿主诞生的**奔跑者异形（Runner/"Dragon"）**在无枪设施内猎杀，而她腹中另有**女皇胚胎**。契约者切入的是一场**「坠毁检疫—无枪猎杀—铸造厂铅水诱饵—公司回收队登陆—宿主自焚」**的封闭杀局：主角线是 **Ellen Ripley** 与囚犯精神领袖 **Leonard Dillon**、医官 **Jonathan Clemens**、尖酸幸存者 **Robert Morse** 在信仰与企业之间求「不把样本交出去的死」。一句话：没有脉冲步枪的监狱星球上，女皇在腹中倒计时，公司要活体，熔炉要宿主，活着的人两边都不是甲方。

**【世界观 · 力量体系】**
本世界无灵气修仙，战力轴是：**人体素质 / 囚犯暴力与宗教组织力 / 医官与行政权限 / 工业环境杀（毒废、铅水、熔炉、热胀冷缩） / 异形生物阶梯 / 公司武装回收队**。死亡规则极硬：无复活外挂；休眠舱只跨时间不跨死亡；酸血可蚀穿金属与人体；胸破即死；女皇胚胎在宿主体内时成体异形会**放过宿主**（护种本能）。菲奥里纳161**无制式枪械**——这是本片与《异形2》最大的战力断崖：胜负手不是弹药，是诱饵路线、熔炉阀门与是否把身体交给公司。

强度阶梯（按实际破坏力，宁低勿高）：
1. **普通囚犯 / 底层勤务 / 无组织散兵**：徒手、管道扳手、工业工具；一爪一酸可致命。乐园观感≈**一阶**。
2. **有组织囚犯小队 / 通风清理组 / 铸造厂劳工班组**：熟悉管道与铸造流程，可设伏、诱敌，仍无重火力。乐园观感≈**一~二阶**。
3. **精神领袖狄龙级组织力 / 医官权限 / 典狱长行政控制**：可动员全监、决定火化与封锁、掌握通讯上报；非个人武力，是场景内的「规则权」。乐园观感≈**二~三阶**（组织与情报层）。
4. **奔跑者异形（四足·狗/牛宿主）**：极速、管道猎杀、单只屠杀松散人群；酸血使近战昂贵。乐园观感≈**四阶**。
5. **女皇胚胎宿主状态的里普利（场景限定）**：成体不杀她，换来「活体样本」价值与公司全副武装回收——不是个人战力拔高，是**战略人质级**。乐园观感≈**五阶事件锚**（生物—政治复合）。
6. **铸造厂铅水浇筑 + 喷淋热胀冷缩**：工业级灭杀单只奔跑者；依赖完整诱饵链与阀门操作。乐园观感≈**五阶**（环境杀上限，本片中段生物顶点解法）。
7. **韦兰-尤坦尼武装回收队 + 女皇胚胎出世风险 + 熔炉自焚级选择**：公司要活体女皇，里普利跳炉同归于尽——非「个人开无双」但决定地球侧是否再得生物武器。乐园观感≈**七阶事件**（本片世界级复合灾难）。

乐园阶位映射（破坏力对齐，宁低勿高）：囚犯散兵·勤务·无武装平民≈一阶；有组织劳工班·通风组·监舍暴力骨干≈二阶；狄龙动员力·克莱门斯医权·安德鲁斯典狱权·铸造流程专家≈三阶；奔跑者异形成体单只≈四阶；铅水—喷淋工业灭杀链 + 女皇胚胎宿主政治价值≈五阶；（本世界不开放六阶切入）公司武装回收 + 女皇胚胎出世/熔炉自焚复合≈七阶。世界顶点＝**奔跑者异形 + 腹中女皇胚胎 + 公司「样本优先」回收链 + 铸造厂熔炉**，写**存在·情报优先/条件性胜利**，严禁「异形被削弱所以战力为零」。低阶切入一律**规避顶点·情报优先**。

**【地理 · 舞台】**
主舞台锁在「苏拉科号—EEV逃生舱—菲奥里纳161铸造厂监狱—垃圾场—医务室—通风管道—铸造模具区—公司登陆场」一条死亡折返线：
- **USS Sulaco（苏拉科号）**：前作终局休眠舰；本片开场火警与EEV弹射发生处（多数场面以残骸/黑匣子回溯）。
- **Type 337 EEV / 逃生舱**：载里普利、纽特、希克斯、毕晓普坠入菲奥里纳；舱内医疗扫描设备可检出女皇胚胎；酸蚀痕迹是「有异形同行」的物证。
- **Fiorina "Fury" 161**：贫瘠炼铅/铸造星球；表面荒凉、风沙与工业废热；外景气质接近英国工业废岸。
- **菲奥里纳161 C级劳动矫正设施（Class C Work Correctional Unit）**：全男监，囚犯多具暴力基因倾向与宗教皈依（双Y等设定在公开资料中作为「反社会倾向」背景提及）；**无枪支**，只有工业工具与信仰纪律。
- **医务室（Infirmary）**：克莱门斯主场；纽特尸检、克莱门斯被杀、异形放过里普利的关键舞台。
- **垃圾场 / 废弃物倾倒区**：残破毕晓普被弃处；里普利重接其电源读取苏拉科黑匣数据。
- **通风管道与地下通道网络**：奔跑者主猎场；高力精神崩溃的触发空间；清理风扇工墨菲之死（酸液致盲卷入风扇）的名场面层。
- **食堂 / 集会厅**：安德鲁斯被拖入通风口的公开处刑舞台；囚犯议事与「我们打还是等公司」的政治场。
- **铸造厂模具区 / 炼炉 / 铅水闸门**：卡片锚点「铸造厂诱饵战」主舞台——囚犯充当诱饵，引异形入模，浇铅，再喷冷水热胀冷缩炸裂甲壳。
- **巨型熔炉 / 龙门吊平台**：终局里普利抱女皇幼体跳入之处；公司回收队对峙的垂直舞台。
- **公司救援舰登陆点**：韦兰-尤坦尼武装小队与「毕晓普二号」着陆处；枪械首次系统进入本片战场。

势力地盘：典狱方握通讯上报与日常纪律；狄龙握囚犯灵魂与动员；公司握救援船与地球侧叙事权；异形握管道与黑暗；里普利握「真相」与腹中人质。AI写场景先定层：坠毁海滩/医务室 → 监舍与管道 → 铸造厂诱饵 → 熔炉对峙，再匹配冲突强度。

**【世界剧情线】**

① **前史·苏拉科号上的第二枚卵**  
《异形2》终局：里普利、纽特、重伤希克斯、半毁毕晓普在苏拉科号入休眠返航。本片开场揭示：至少一枚（扫描后证实与里普利体内相关的链路指向**两只抱脸体量级**的灾难）异形卵/抱脸体在舰内活动，酸液腐蚀触发火警与紧急程序；仍在休眠舱中的四人被弹入EEV逃生舱，脱离母舰。因果：军事胜利后的「二次入侵」在舰内完成——不是女皇本体登舰（前作已抛入太空），而是**残余样本/卵**把返航变成坠落。

② **坠毁菲奥里纳·三人已死（开篇）**  
EEV坠入菲奥里纳161。囚犯与管理人员无防护打捞。结果：**德韦恩·希克斯**被安全支架刺死；**丽贝卡·「纽特」·乔丹**溺亡；毕晓普残骸被当垃圾丢弃；唯**艾伦·里普利**被医官**乔纳森·克莱门斯**救醒。里普利要求对纽特尸检——怕胸破；克莱门斯执行后未发现胚胎。尽管典狱长**哈罗德·安德鲁斯**与助手**弗朗西斯·亚伦（「85」）**反对，仍在「传染病风险」话术下将纽特与希克斯遗体投入熔炉火化。葬礼上，精神领袖**伦纳德·狄龙**致悼词；与此同时，抱脸体寄生监狱的狗 Spike（院线）或牛（汇编剪辑），**奔跑者异形**破体而出。世界状态：官方叙事是「飞船事故」；真实状态是「无枪监狱已迎来完美猎手」。

③ **独活者的检疫焦虑·毕晓普黑匣子**  
里普利在陌生全男监中剃发求安（狄龙建议），仍遭**泰德·「少年仔」·吉拉斯（Junior）**等囚犯企图性侵，被狄龙喝止救出。她在垃圾场找回毕晓普，强行启动；残破合成人确认：苏拉科上有抱脸体，且**韦兰-尤坦尼知情链路**已指向「样本」。毕晓普请求永久关机。里普利据此警告安德鲁斯——典狱长不信「龙」，只信纪律与上报流程，并把她的存在本身视为打破囚犯宗教平衡的污染。阶段因果：真相已在，权力拒绝听。

④ **猎杀开始·墨菲、管道与高力**  
异形成熟后猎杀：清理通风的**托马斯·墨菲**被酸液溅盲，卷入风扇而死；**爱德华·波格斯、丹尼尔·雷恩斯**等在通道中遇害；目击者**沃尔特·高力**精神崩溃，回到连环杀手式的失神状态。安德鲁斯把死亡归咎于囚犯骚动与高力旧疾，拒绝里普利的「外星生物」框架。公司对坠毁报告的异常重视让安德鲁斯起疑，却选择把里普利软禁在医务室方向——要的是秩序，不是真相。

⑤ **医务室之夜·放过宿主**  
克莱门斯向里普利坦白自己的囚犯前科与留任医官的轨迹，两人短暂建立信任与亲密。异形突袭医务室，**杀死克莱门斯**，却在逼近里普利时停住、退去——护种：她体内已有**女皇胚胎**。里普利冲入食堂警告正在开会的众人；异形从通风口拖走**安德鲁斯**，当众处刑。典狱权力真空；亚伦无力接盘；囚犯陷入「等公司船」与「自己动手」的撕裂。

⑥ **毒废引燃计划失败·扫描真相**  
里普利提出：将易燃有毒废料注入通风系统并点燃，把异形驱入废弃核废料贮罐。计划因异形中途袭击而提前爆炸，多名囚犯死于火灾与混乱，异形逃脱。里普利胸痛，使用EEV医疗扫描仪——影像显示**女皇胚胎**在生长。她顿悟：苏拉科上不止一个威胁；公司救援的真实目的是**回收她腹中的样本**做生物武器，而非救人。成体异形不杀她，正因她是移动的巢。

⑦ **与狄龙的死约·铸造厂诱饵战（卡片锚点）**  
里普利求狄龙杀了自己；狄龙答应——**前提是先联手杀死成体异形**。亚伦拒绝参与，押注公司船。狄龙动员残存囚犯：以自身为诱饵，把异形引入铸造厂模具区，由**罗伯特·摩斯**操作闸门浇下熔铅。追逐中**彼得·格雷戈尔、大卫·波斯特尔思韦特**等大批囚犯死于管道与陷阱；汇编剪辑中**Junior**等亦有不同牺牲节点。最终**狄龙**在模具中以肉身钉死异形站位，被异形杀死；摩斯浇铅。异形负伤爬出；里普利启动喷淋系统——**热胀冷缩**令浇铅甲壳炸裂，奔跑者毁灭。名场面因果：没有枪的世界，用工业与人命换一次条件性胜利。

⑧ **公司登陆·熔炉终局**  
韦兰-尤坦尼小队抵达，领队是与毕晓普同貌的男人（片中称毕晓普二号；衍生资料常作**迈克尔·毕晓普/Michael Bishop**），声称是合成人设计者、要「手术取出胚胎并销毁」。里普利看穿话术：公司要的是**活体女皇**。亚伦终于醒悟，挥扳手砸向公司人，被突击队击毙。摩斯协助里普利登上熔炉上方平台；公司开枪打伤摩斯的腿。里普利拒绝交出身体，从平台**纵身跃入熔炉**；女皇幼体破胸而出的瞬间，她将其按在怀中一同焚毁。设施随后关闭；**摩斯**作为全监（含里普利）唯一人类幸存者被押走。片尾回响《异形》中里普利的最终日志录音。

⑨ **汇编剪辑版差异（同一终局骨架）**  
2003年 Assembly Cut：宿主可为牛而非狗；高力对异形的病态「守护」更长，可破坏封锁计划；Junior 等有额外牺牲戏；部分人物生死节点微调。写世界状态时以**院线终局为准**（铅水+喷淋灭奔跑者、里普利跳炉、摩斯独活），汇编差异作可选支线皮肤，不改公司要样本与宿主自焚的核心因果。

⑩ **贯穿逻辑（契约者用）**  
- **企业线**：坠毁报告一出，公司船比真相更快；「取出并销毁」=公关句，「生物武器部门」=真句；毕晓普二号是伯克的升级皮。  
- **宗教/囚犯线**：狄龙的末日信仰把「赎罪」变成可动员的纪律；无枪条件下，信仰是唯一的编制。  
- **宿主线**：里普利从「杀异形的人」变成「异形要保护的温床」；放过=诅咒；跳炉=她夺回对身体的最后解释权。  
- **空间规则**：无枪、管道、酸血、铅水、喷淋、熔炉是叙事物理；硬刚奔跑者不如诱饵与工业杀。

任务方向/奖励：活过第一夜、遗体安全火化、物证不被亚伦销毁；奖励取向＝熔炉区通行、基础防护面罩、克莱门斯的临时助手身份（非胚胎、非异形组织）。

**二阶（通风班组·管道阴影 · 墨菲前后）**

切入身份/时点：以**通风清理班/屠宰与畜舍相关劳工（对接狗或牛宿主线）/监舍纠察**切入，锚定**宿主已破体但官方仍称意外、墨菲事件前后**。

初始事件：班长派你进扇叶通道除垢，你在格栅上摸到新鲜酸蚀。频道里有人笑「又是高力发病」。下一秒，尖叫声被风扇吃掉——**托马斯·墨菲**的死讯传来。**沃尔特·高力**满身血地爬出来只重复「龙」。安德鲁斯要你签字证明「高力行凶」；里普利（若已介入）要你带路去垃圾场找合成人残骸。

开场白建议：「管道里的热气把呼吸变成刀。手电扫过处，金属像被舌头舔过。你听见上方有四足落地的轻响，轻得像在学你们的班组口号。报告表只有两栏：工伤，或凶杀。没有一栏写：有东西把人当畜舍。」

关键NPC立场：**沃尔特·高力**——崩溃且危险，可能把你拖进他的「龙」叙事；**哈罗德·安德鲁斯**——要你的签名当遮羞布；**艾伦·里普利**——用你当向导，厌谎言；**伦纳德·狄龙**——要稳定，不接受恐慌传教；**奔跑者（未正面）**——二阶不可猎，只可逃。

主线钩子/支线：主线＝活着带出「非人为」死亡证据并送达克莱门斯或里普利；支线＝锁死畜舍避免二次宿主、给高力注射镇静而非私刑、偷改上报文案延迟公司注意力（双刃）。

危险度/规避：高——单只奔跑者可清班；严禁独自深管道追声；禁止把「签名诬陷」当长期策略。

任务方向/奖励：班组减员最小化、证据链建立；奖励取向＝管道地图、简易酸液冲洗剂、狄龙社区的有限接纳。

**三阶（医官与领袖·食堂权力真空 · 安德鲁斯之死前后）**

切入身份/时点：以**医务助手、狄龙信任的执事囚犯、或典狱通讯值守**切入，锚定**克莱门斯仍在至食堂安德鲁斯被拖走**——真相开始公开，组织力决定谁死。

初始事件：你在医务室值班，听见克莱门斯与里普利低语前科与检疫。阴影里有东西停住——它杀医官，却绕开里普利。你若尖叫乱跑，食堂会在无准备时被突袭；你若冷静传讯，**伦纳德·狄龙**能先控制通道。十分钟后，安德鲁斯在集会中被拖入通风口，血滴在汤盆里。

开场白建议：「警报不是响，是人声一层层碎掉。典狱长还在强调纪律，天花板却先回答了他。你看见那东西的头冠像湿润的骨盔，动作却比祈祷更快。有个剃光头的女人站在血泊边说：它放过我，因为我比你们更糟。你第一次希望她在说谎。」

关键NPC立场：**艾伦·里普利**——要以真相动员，厌「等公司」；**伦纳德·狄龙**——可结盟的动员核；**弗朗西斯·亚伦**——会反对一切「囚犯拿主意」；**乔纳森·克莱门斯**（死前）——托付药品与记录；**哈罗德·安德鲁斯**（死时）——秩序的尸检。

主线钩子/支线：主线＝在食堂惨案后帮助建立临时指挥（狄龙—里普利轴）并清点毒废库存；支线＝保护扫描仪电缆、阻止私刑高力、切断亚伦对外「一切正常」的通讯。

危险度/规避：高——成体已公开猎杀；须规避无组织的英雄主义冲管道。

任务方向/奖励：临时指挥链成型、毒废计划获得投票；奖励取向＝钥匙串、医械、狄龙的「执事」名义、里普利的有限信任。

**四阶（奔跑者猎场·毒废与火灾 · 成体主宰段）**

切入身份/时点：以**毒废灌注组核心、诱饵预备队、或里普利认可的「懂异形的外人」**切入，锚定**毒废引燃计划执行至爆炸失败、扫描确认女皇胚胎之前/之时**——奔跑者是战场主宰。

初始事件：你拧开毒废阀，计量刚好。异形从侧道扑出，有人提前点火。走廊变成白焰与尖叫。计划失败后，里普利按住胸口去EEV；扫描仪的光映出胚胎轮廓。她看你的眼神变了：你若流露「保护样本」的职业习惯，她会先找刀。

开场白建议：「毒废的气味像甜的腐蚀剂。火焰该按剧本走，剧本却被四条腿改写。烧伤的人在地上叫领袖的名字，领袖在数还剩谁能走。女人从逃生舱下来，脸色比熔炉更冷：它在我肚子里。而公司的船，已经在路上。」

关键NPC立场：**奔跑者异形**——四阶顶点生物，陷阱与工业优先于硬刚；**艾伦·里普利**——宿主自觉后目标转为「先杀龙，再死」；**伦纳德·狄龙**——要求你用命换锁链；**弗朗西斯·亚伦**——仍幻想公司；**罗伯特·摩斯**——开始显得有用。

主线钩子/支线：主线＝在火灾后收拢残兵并保住铸造厂闸门操作人员；支线＝销毁一份可能被公司截获的胚胎扫描拷贝、救出一名重伤诱饵、确认喷淋系统未关。

危险度/规避：极高——单只奔跑者可撕开编制；女皇胚胎使里普利成为「不可杀的战略体」，也使她成为公司磁铁。

任务方向/奖励：残存战力转入铸造厂方案；奖励取向＝闸门操作权、防火服、摩斯级流程口令——严禁活体样本奖励。

**五阶（铸造厂诱饵·铅水与喷淋 · 工业杀龙）**

切入身份/时点：以**模具区闸门主操、狄龙并排的诱饵核心、或持有熔炉总阀权限的高阶契约身份**切入，锚定**死约已立至铅水浇筑与喷淋炸裂**——本阶必须完成对奔跑者的**条件性胜利**。

初始事件：狄龙把铁管拍在你手上：「你跑得过它，或死得有用。」摩斯在闸台上骂人。诱饵链条一环断，全盘崩。异形入模后，狄龙选择留下；你必须在「冲下去救领袖」与「执行浇铅」之间按原作逻辑站队——救错，铅水浇空，龙出模，全员灭。

开场白建议：「模具像张开的铁喉。你们把脚步踩成鼓点，把恐惧踩成路线。它追进来时，铅水还是安静的银。领袖在坑底张开双臂，像最后一次讲道。你听见闸门咬合的声音，比祈祷响。然后是水——冷的——打在滚烫的甲壳上，整条龙在热胀冷缩里开花。」

关键NPC立场：**伦纳德·狄龙**——以死完成钉位，**存在·条件性胜利**的人柱；**罗伯特·摩斯**——闸门与浇铅执行；**艾伦·里普利**——要求你先完成杀龙再谈她的死；**奔跑者**——五阶生物顶点，工业杀而非单挑；**弗朗西斯·亚伦**——缺席或旁观，押注公司。

主线钩子/支线：主线＝引异形入模并完成铅水+喷淋灭杀；支线＝减少诱饵死亡人数、确保狄龙的死换来有效站位、预埋熔炉平台通路供终局使用。

危险度/规避：贴近生物顶点——胜利条件是**闸门、铅、水、路线**，严禁「无工业条件肉搏削血条」；必须规避浇铅前内讧、关闭喷淋、把异形「留活口给公司」。

任务方向/奖励：奔跑者确认死亡、铸造区暂时安全；奖励取向＝五阶工业反生物经验、熔炉总阀权限、全监残存指挥碎片——**禁止活体异形/胚胎作为装备**。

**六阶（样本回收前夜·熔炉控制与公司谎言）**
切入身份/时点：以掌握公司航路的通讯技术员、熔炉总工程师、狄龙认可的协调者，或伪装成矫正局稽核的契约者身份切入，锚定奔跑者被毁后、公司登陆队抵达前的短暂空档。
初始事件：毕晓普残骸留下的飞行记录与 EEV 扫描结果指向同一件事：来船不是救援。亚伦要恢复联络迎接公司，摩斯要先修好闸门，里普利则要求把能证明胚胎存在的记录烧掉。你必须决定保留哪一条证据链，以及谁持有熔炉平台钥匙。
开场白建议：「铅水已经冷成一张灰色的脸，通风管却仍在滴酸。远处的舰影像一枚落向监狱的手术刀，频道里有人把它叫救援。」
关键NPC立场：**艾伦·里普利**——样本不得离星；**罗伯特·摩斯**——先让自己活过下一小时，但懂得闸门比口号可靠；**弗朗西斯·亚伦**——仍愿相信公司；**兰斯·毕晓普**——只提供事实，不替任何人作决定；**迈克尔·毕晓普**——将以手术与安全话术争夺宿主。
主线钩子/支线：主线＝在登陆前封存或销毁扫描、黑匣与熔炉权限，令公司无法同时拿到宿主和完整记录；支线＝为摩斯设置撤离通路、拆除一台可供回收队使用的升降机、把亚伦的上报记录留作反证。
危险度/规避：贴近顶点——没有成体不等于安全，武装队和腹中女皇构成新的杀局；规避把里普利当作可谈价资产，规避让单一人持有全部控制钥匙。
任务方向/奖励：守住销毁路线、获得熔炉区权限或公司行动情报；奖励为反生物武器程序、工业设施控制权与罪证，不含异形组织。

**七阶（公司回收·熔炉主权 · 条件性胜利）**

切入身份/时点：以**韦兰-尤坦尼生物武器观察员（表面）、矫正局特派稽核、熔炉总工程师或乐园灾难干预专员**切入，锚定**公司舰着陆至里普利跳炉**——世界顶点以**女皇胚胎 + 武装回收 + 熔炉自毁选择**三重复合出现。

初始事件：你同时收到两份指令——A.「保护里普利体内资产，活体优先，可牺牲全部囚犯」；B.「样本不得离开菲奥里纳重力井，必要时销毁宿主」。平台上，**毕晓普二号**温言劝手术；**亚伦**挥扳手；**摩斯**腿上中弹仍爬向控制杆。里普利看你：「你是来剪脐带的，还是来按火的？」

开场白建议：「救援舰的灯光比熔炉更假。穿防护服的人端着枪，却把「治疗」说得像圣餐。女人站在吊台边缘，腹中有一整座未来战争。公司要她躺下。监狱要她消失。她选第三种：带着那东西，一起成为炉渣。你站在按钮与扳机之间，第一次明白芬奇式终局——有时英雄不是活下来的人，是不让样本活下去的人。」

关键NPC立场：**艾伦·里普利**——跳炉线的执行者，对回收派先拒后死；**迈克尔·毕晓普/毕晓普二号**——七阶政治顶点，枪与谎言；**罗伯特·摩斯**——唯一可配合的幸存者键；**弗朗西斯·亚伦**——晚到的醒悟与横死；**女皇胚胎**——破胸即世界失败条件；复合顶点＝**存在·情报优先/条件性胜利**，严禁数值削峰。

主线钩子/支线：主线＝确保女皇胚胎与成体均不得被公司带离 + 里普利（或等价销毁方案）完成熔炉销毁；支线①击穿「手术销毁」话术并公开录音；支线②护送摩斯活到见证位；支线③删除公司舰上的活体运输协议。

危险度/规避：极高/贴近顶点——必须规避：无销毁条件下的「保护宿主撤离」、相信毕晓普二号、在女皇破胸后仍试图活捉。胜利条件应是**熔炉、销毁、拒绝交付**，而非单挑公司全队。

任务方向/奖励：地球免遭本星女皇样本、公司回收失败、摩斯级见证链保全；奖励取向＝七阶反生物武器协议、企业罪证核、熔炉级销毁授权——**严禁把女皇胚胎/活体异形当宠物或常规装备发放**。
