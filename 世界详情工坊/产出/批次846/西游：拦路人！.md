## 剧情

**【作品来源】**
《西游：拦路人！》作者九月病句，起点中文网连载古典仙侠／西游同人系统流长篇（书号1046179008，约2025年10月上架，连载中，公开进度常在六百章上下、约一百四十万字量级，章节随更新浮动）。气质偏半文半白、贴近《西游记》回目腔，又叠一层现代穿越者的规则博弈：系统按「阻拦取经人滞留天数」发奖，智取布局优先于硬刚碾压。文风有乐子、有危机感，人物智商多在线，神佛不降智。**作品仍在连载，本档案只写至已公开剧情与可核设定，不编造终局谁胜谁负、西天是否被改写。**

**【世界定位】**
三界仍是《西游记》底盘：东土大唐、西天灵山、天庭、龙宫、地府与沿途妖国城池并存；取经大劫是既定气运工程。主角敖徒为穿越者，穿成泾河龙王第五子、西天大雷音寺司钟之龙，被阿难尊者陷害下界充当「取经劫难」。他绑定【西游拦路人系统】，以拖延唐僧师徒、阻止佛教大兴换取术法、精血与灵宝，从血脉不纯的四爪黄龙走成可布局三界的真龙棋手。一句话：在「八十一难」的棋盘上当拦路人，用时间和规则把取经路拖成八百难。

**【世界观 · 力量体系】**
本界以仙佛妖魔共处的神话修真为总框架。力量来源包括：先天血脉（龙族、妖族、先天灵根）、后天修炼（丹道、神通、法宝认主）、功德与气运（取经、香火、因果债）、系统外挂（拦路时长结算）。死亡对低阶妖人近乎永久；高阶有元神、真灵、法身莲台等留后手，但因果反噬、香板、天条、佛门戒律仍可抹杀。复活不是日常设定，被收服、炼化、入莲台、入宝葫芦等「非死」结局更常见。

**凡人／妖卒层**：乡绅、兵卒、小妖、土地城隍属吏。战力=冷兵器与微末法力，可被契约、谣言、田契、官府公文左右。是拦路人早期最重要的「时间武器」——拖一天也是一天。

**地仙／天仙边**：山精水怪、地仙散修、天庭小将、龙宫偏将。能裂石断河、兴云布雨、变化人形，怕正牌神通与法宝。典型：早期夜叉追杀、土地神、小妖头目。战力≈街区至据点级，一城可掀局部灾难。

**天仙**：正式入仙籍或妖中成气候者。御空、领域雏形、法宝初成。司钟小龙原属此层边缘；阿难派来的部分打手也在此。战力≈城市级威胁，可屠城寨、困凡人军。

**玄仙**：天仙之上的质变。法力绵长、神通可成形，龙族血脉在此开始「真龙化」。敖徒吸收真龙精血后由天仙后期破入玄仙，是早期智取的本钱。战力≈地区级：移山填海初现、可正面斗普通妖王与部分天将。

**金仙**：不朽雏形，受三灾约束减弱；真龙金仙可生第五爪、项下龙珠孕小世界雏形。敖徒在高老庄线后炼化真龙龙珠、渡劫雷成就金仙道果，蜕为五爪金龙。战力≈世界级局部：可抗衡菩萨法身、困压齐天大圣一时、布大阵改写一难格局。

**太乙／大罗向**：金仙之上的更高道果；先天灵宝、大阵本源、古佛法脉、祖龙血脉多在此层发力。大罗、如来、部分古佛与先天大能属世界顶点带。战力≈行星／规则级：言出法随、掌一方气运工程。

**特殊系统：西游拦路人系统**
- 绑定条件：主角进入西游大劫、下界为「拦路」身份后解锁主线。
- 计时规则：唐僧师徒进入指定劫难区域（观音禅院、高老庄、黄风岭、通天河……）开始累计滞留天数；离开区域则停表结算；**进度可跨区域继承**（如前一难多余天数转入下一难）。
- 主线任务：阻拦取经人，阻止佛教大兴。
- 奖励逻辑：按连续／累计拦截时长阶梯发放，偏实用成长向，而非无脑无敌签到。已知阶梯（公开简介与早期正文一致处）：
  - 拦截三天 → 移山之术
  - 拦截七天 → 六转金丹
  - 拦截半个月 → 真龙龙珠
  - 拦截一个月 → 后天灵宝·五火神焰扇
  - 拦截一季 → 神通·五色神光
  - 拦截半年 → 祖龙精血
  - 拦截一年 → 先天灵宝·散魄葫芦
  - 拦截三年 → 先天灵宝·河图洛书
  - 拦截十二年 → 混沌钟
  - 拦截六十年……（更高未尽公开）
- 后期公开结算例：通天河一类长阻可达数百天级，奖先天八卦神通、周天星斗大阵阵旗等；女儿国线可用河图洛书布九曲黄河大阵。
- 代价与纪律：系统鼓励布局、信息差、契约与结盟；过早暴露真身硬刚孙悟空／菩萨级常等于任务失败。奖励暴露即杀劫，须保密。

**龙族血脉线**：四爪黄龙（不纯）→ 真龙精血提纯 → 真龙龙珠／祖龙精血 → 五爪金龙。血脉决定寿元、龙息属性（如锐金毫光）、变化上限与龙珠小世界潜力。

**取经气运线**：八十一难是灵山与天庭共谋的功德工程；每难本有「既定妖怪与收服节奏」。敖徒的存在把「剧本」改写成可超时、可改派、可反杀的动态战场。阿难意图收服他换功德，是个人恩怨与制度性压榨的叠合。

乐园阶位映射（对照破坏力，宁低勿高）：凡人乡绅／小妖卒≈一～二阶；地仙～天仙边≈三～五阶；天仙≈五～六阶；玄仙≈六～七阶；金仙≈七～八阶；太乙～大罗向／如来级＝超阶（情报优先／条件性胜利，严禁把顶点写成「镇印后战力归零」）。本库覆盖一～八阶；低阶侧重龙族小劫、凡俗侧契约与谣言布局，勿把金仙战直接塞进一～三阶开局。

破坏力补充锚：凡人械斗与小妖偷袭≈一～二阶街巷；地仙移云布雨≈三～四阶据点；天仙司钟、夜叉围剿≈五阶城市边缘；玄仙移山与龙息≈六阶地区；金仙法身对轰、阵困齐天≈七～八阶世界局部；如来法座与大罗道果＝超阶，只可迂回、交易因果或借规则，不可当杂兵清场。

**【地理 · 舞台】**
地图以《西游记》西行线为脊：东土大唐国境 → 观音禅院／法门一线 → 高老庄 → 黄风岭 → 流沙河 → 四圣试禅心诸地 → 五庄观万寿山 → 白虎岭 → 平顶山莲花洞 → 乌鸡国 → 火云洞／枯松涧 → 黑水河 → 车迟国 → 通天河 → 金兜山／金兕洞 → 女儿国／西梁 → 火焰山 → 小雷音／小西天 → 荆棘岭、镇海寺等后续难地。终点意象是西天极乐、大雷音寺；开篇与幕后则反复回到灵山钟楼、大雄宝殿与如来法座。

**西天大雷音寺**：敖徒原职「司钟之龙」所在。钟声报时是佛门仪轨一环；误钟即犯天条式戒律。阿难在此设局，如来在此判罚三千香板与下界为劫。写灵山：金碧、戒律冷光、近侍弟子的笑面杀机。

**泾河／四海龙宫**：父亲泾河龙王已在前史被算计致死（呼应原著袁守诚／魏徵斩龙线的余波与本书填充）；兄弟分赴各处任职；舅舅西海龙王曾把敖徒安插到灵山「求个安稳差事」——安稳反成陷阱。龙宫是血脉、人情与龙族政治的后台。

**观音禅院**：金池长老、广智广谋等凡僧舞台；袈裟、香火与官府介入是早期拖时工具。

**高老庄**：高太公一家、田地契约、乡民舆论。凡人规则在这里比金箍棒更致命——因为取经人不能对无辜乡民大开杀戒。

**黄风岭**：黄风怪（貂鼠精）洞府、三昧神风故地；灵吉菩萨飞来弹压的战场。五火神焰扇在此改写「菩萨随手解围」的剧本。

**通天河／陈家庄一带**：八百里水域、灵感大王原著位、老鼋渡口；本书中成长阻战场，数百天级结算的地理容器。

**女儿国／西梁女国**：河图洛书与九曲黄河大阵的舞台，把「情劫」改写成阵法锁路。

**火焰山／翠云山／积雷山**：牛魔王、铁扇公主、红孩儿因果网；公开章节中有多回「路阻火焰山」，涉及五色神光、古佛宝珠、观音被擒等强冲突描写（以正文为准）。

**小雷音／镇海寺等**：假佛、古寺妖氛；连载后期常见「遇怪」连章，适合高阶切入。

写场景分层：灵山戒律金光、龙宫潮腥冷宫、禅院香火、庄户泥土、黄风砂石、通天河冰雪、火焰山赤霞——同一取经路，海拔与规则完全不同。AI写环境先写「这一难原来的功德剧本是什么、谁负责收尾」，再写山水。

**【世界剧情线】**
① **前史：泾河龙死与灵山安插**  
泾河龙王因与术士／天庭因果被斩（本书在原著框架上补龙族侧视角），五子敖徒等兄弟失怙。西海龙王安排敖徒入西天为大雷音寺司钟之龙，求的是「远离是非」。穿越魂入体后，他本可隐忍混日子，却撞上阿难的功德算盘。世界状态：取经尚未发车或将发，灵山表面庄严，内部近侍已在挑「下界劫难」的棋子。

② **司钟陷害与下界为妖（天仙边）**  
阿难尊者施法使敖徒昏睡，误了司钟时辰，再向如来告发。敖徒当堂揭穿仍难翻盘——如来偏近侍，判三千香板并罚其下界充当取经路上的劫难。阿难威胁：即便身死也要龙魂受苦；又假报「逃遁」，领命「收服」，实则想独占收服功德。敖徒含恨下界，**西游拦路人系统**开启：阻拦取经人、阻止佛教大兴。新手礼真龙精血已吸，修为自天仙后期入玄仙。本阶段因果：没有阿难，就没有拦路人；没有系统，就没有与齐天大圣同台的本钱。世界状态：唐僧尚未出长安或刚出，拦路人先于取经人入场。

③ **观音禅院：谣言、官府与二十七天（玄仙）**  
第一难区锁定观音禅院。敖徒不以龙身硬刚，而**化身／冒充阿难尊者**诱导凡僧与地方舆论，把袈裟、香火、金池长老的贪念与官府介入拧成绳，迫使取经人滞留约**二十七天**（未满月，多余进度约十二天可继承）。结算移山之术、六转金丹、真龙龙珠等。名场面逻辑：佛门近侍的脸被用来砸佛门安排的禅院——信息差与身份戏是早期核心战法。阶段结局：有术有丹有珠，仍远不是悟空对手，但「时间可以换钱」的模式跑通。

④ **高老庄：假唐僧、猪八戒与田契围堵（玄仙→金仙门槛）**  
继承禅院进度冲刺「一个月档」。途中灭阿难所派夜叉（天仙／地仙级），龙息锐金毫光显形神俱灭之威。入高老庄后，敖徒**变作唐僧模样提前收服猪八戒**，再以土地买卖、田契纠纷煽动乡民围堵真正的师徒，滞留三十余天量级，兑现后天灵宝·五火神焰扇等。阶段主题：凡人不能杀、契约不能毁、舆论不能压——取经人的道德枷锁就是拦路人的城墙。炼化真龙龙珠前后，境界松动，天降劫雷，成就**金仙道果**，蜕鳞为**五爪金龙**，项下龙珠现小世界雏形。世界状态：八戒入伙时间线被搅乱；阿难「演戏赏金花」实则利用，恩怨升级。

⑤ **黄风岭：结盟黄风怪、扇灭灵吉（金仙）**  
任务区更新。敖徒以灵山旧识等话术联合黄风怪，用三昧神风与布局困悟空，再祭出五火神焰扇——七禽翎羽与五火合成的后天灵宝，可灰山石、煎海水、克五行遁，**烧毁灵吉菩萨法身莲台并擒之**。原著「菩萨一弹指解围」被改写成拦路人与妖王的反杀。奖励线继续指向五色神光、祖龙精血等。阶段结局：第一次在「菩萨级出场」上取得可见战果，三界开始重新给「那条黄龙」定价。

⑥ **中段诸难：白骨、乌鸡、红孩儿、车迟（金仙稳固）**  
公开回目可见：三打白骨精线中，白骨精假身被悟空打散后向敖徒汇报／配合（他甚至能在田间与凡人「乔儿」一类角色共事，把布局沉到泥土里）；乌鸡国线他曾授当地道士法门，后被悟空所得，说明其手段已渗入「助国除妖」的官方叙事缝隙；大战红孩儿多章，神通与三昧真火／火云洞因果纠缠；车迟国斗法中化身太西真人等身份与虎力、鹿力、羊力及悟空赌胜，争的是**信仰与香火因果**，直接对冲佛教在东土的扩张。黑水河、平顶山、五庄观等原著难地按取经顺序被改写为可超时的任务区——手段仍是：结盟本地妖、改契约、换身份、布阵、用系统法宝卡时间。世界状态：八十一难的「标准工期」全面延误。

⑦ **通天河：三百八十七天与大阵资本（金仙高阶）**  
通天河成为长阻标杆：公开资料记约**三百八十七天**结算，奖励含**先天八卦神通**、**周天星斗大阵阵旗**等。这些资本在后续金兕洞等战场用于困压孙悟空与下界神将，把「齐天大圣＋天兵」从无解压成可围可耗。阶段主题：时间本身成为战略资源；拦路人从「捣乱的龙」变成「改工期的总工」。

⑧ **女儿国、火焰山、小雷音与更高难（金仙→更高）**  
女儿国线：以河图洛书布**九曲黄河大阵**锁取经人，情劫变阵劫。火焰山多回「路阻」：涉及牛魔王势力网、五色神光、古佛宝珠，公开描写中有擒观音等强冲突（细节以正文为准，写卡时勿无依据夸大日常）。小雷音、荆棘岭、镇海寺等连载后期连章，任务区继续解锁更高奖励（先天灵根扶桑木、十二都天神煞大阵阵图、古佛法脉等公开提及项）。盟友网扩张：黄风怪、牛魔王、万圣公主等；与哪吒、二郎神等正神有接触／博弈，非简单敌我。阿难线贯穿：夜叉、阴间状告、功德争夺，车迟国后再起波澜。

⑨ **远景钩子：八百难仍未至西天（连载进行中）**  
简介与公开钩子：若干年后，如来问观音取经人历几难，观音答「已经八百难了」——仍未达西天。总矛盾清晰：灵山要功德工程按剧本完工；阿难要个人收服账；取经人要成佛；敖徒要活、要强、要改命、要让龙族与自己不被当棋子。**作品连载未完：不写终局胜负，不编造如来是否退位、取经是否废除、敖徒是否证大罗；只标世界处于「取经严重逾期、三界重新洗牌中」的进行态。**

⑩ **微观戏贯穿始终**  
司钟时辰、香板数目、田契文书、乡民围堵、假身份法号、区域计时边界、奖励结算提示、龙鳞蜕换、劫雷计数、阵旗落点——这些比大招更常出场。写这个世界，先写「这一难的工期被拖了几天、谁在为延期负责」，再写金箍棒与龙爪。

⑪ **智取纪律与失败态（写作硬约束）**  
成功的拦路几乎总满足三条：①任务区边界清晰（师徒未离区才计时）；②取经人道德枷锁被利用而非被无视（屠庄＝任务偏航＋气运反噬）；③阿难／灵山反制被预留一手（假逃遁、阴间状告、假赏金花）。失败态同样具体：提前暴露系统面板、硬刚未结算完的齐天大圣、为抢奖励杀唐僧导致主线崩盘。连载中任何「一掌扇死如来」都属 OOC；正确高光是工期改写、阵旗落子、菩萨级被拖入消耗。

**【世界剧情线·分难细目】**

**难前·灵山司钟**  
起因：阿难要功德棋子。事件链：昏睡误钟 → 如来判罚 → 香板 → 假逃遁 → 下界。名场面：大雄宝殿对质。可介入：救钟、揭阿难、替罪或同罪。

**观音禅院**  
起因：系统第一任务区。事件链：化身阿难 → 谣言／贪袈裟 → 官府 → 滞留27天 → 结算术丹珠。因果：模式验证。

**高老庄**  
起因：冲刺月档奖励。事件链：杀夜叉 → 假唐僧收八戒 → 田契乡民围堵 → 五火神焰扇 → 金仙蜕变。因果：战力质变。

**黄风岭**  
起因：金仙后的第一场「菩萨局」。事件链：联黄风怪 → 困悟空 → 扇灭灵吉莲台并擒。因果：定价上调。

**白骨岭／白虎岭**  
起因：经典心性难。事件链：白骨精配合／汇报 → 田间凡人线并行 → 搅乱三打节奏。因果：取经人内部分歧被拉长。

**乌鸡国**  
起因：国运与全真／佛道争。事件链：授道士法门 → 被悟空侧复用 → 工期再拖。因果：手段渗入「合法除妖」。

**红孩儿／火云洞**  
起因：三昧真火与观音收善财原剧本。事件链：大战多章、神通对轰。因果：火焰山前史被改写。

**车迟国**  
起因：香火与信仰。事件链：太西真人等马甲斗法 → 对冲佛兴。因果：阿难线再交锋。

**通天河**  
起因：长阻最优地理。事件链：约387天 → 先天八卦／周天星斗阵旗。因果：大阵资本到位。

**女儿国**  
起因：情劫改阵劫。事件链：河图洛书 → 九曲黄河大阵。因果：锁路不锁心亦可。

**火焰山**  
起因：必经火路与牛魔网。事件链：多回路阻、五色神光／古佛珠等。因果：高阶冲突公开化。

**小雷音／镇海寺等**  
起因：假佛与古寺。事件链：连载后期连章遇怪。**止于已公开信息，不写结局。**

画风：半文半白神话腔＋现代布局脑；可有乐子，不可降智；神佛威严与阴私并存。忌把孙悟空写成无脑莽夫；忌把如来写成随便斩的杂兵；忌无系统结算就发先天灵宝；忌低阶开局直接扇灭菩萨；忌用「封印削弱」解释顶点战力；忌编造连载终局。最早可切入锚点：泾河后、司钟前夕的龙族小劫；或观音禅院前夜的凡僧流言场；契约者亦可作「另一路被派下界的劫」但勿抢敖徒系统唯一性（系统仅主角绑定，契约者走原作规则与自身能力）。

## 阶位切入点

> 阶位↔境界：一阶≈凡人乡绅／小妖卒；二阶≈精怪／兵痞超凡边；三阶≈地仙边缘／山寨妖头；四阶≈地仙稳固／龙宫偏将；五阶≈天仙边～天仙（司钟小龙、夜叉级）；六阶≈天仙圆满～玄仙初（禅院～高老庄前期）；七阶≈玄仙～金仙门槛／金仙初（高老庄结算～黄风岭）；八阶≈金仙稳固～金仙高（通天河、女儿国、火焰山）。世界顶点（如来、大罗向古佛等）为**超阶巅峰：存在·情报优先／条件性胜利**，严禁写成「镇印后战力归零即可击杀」。低阶只写龙族小劫与凡人侧布局，不硬塞金仙战。

**一阶（凡人庄户·田契与谣言）**

切入身份／时点：契约者以东土流民、庄客、行脚商或衙役文书身份切入，锚定**高老庄田契纠纷前夜**或**观音禅院香火争端初起**——取经人未至或将至，地方已为袈裟／婚事／田界吵成一锅粥。

初始事件：你经手的一份田契或一份诬告状被人用「佛祖近侍」的名义加盖了不存在的威压；乡民要你在「保田」与「迎僧」之间选边，拒绝则三日后田契作废、全家被指为妖党。

开场白建议：「高老庄的土还是湿的，契纸却已经传了三姓。有人说西边来了真佛，有人说昨夜钟声里有龙气。你只是个在田埂上盖手印的人——可手印一落，取经的路就会在这里多停一天。」

关键NPC立场：**高太公**（护女护田，可用可骗）；**金池长老**（贪袈裟，怕官府）；乡里文书与捕快（怕事）；远处传说中的**猪八戒**尚未定心。

主线钩子／支线：保住或烧毁那份能拖住「真唐僧」的田契；支线——替广智广谋跑腿送假消息；支线——在禅院后门听见「阿难」声口不对。

危险度／规避：低～中。规避显形龙、硬碰行者；莫在佛前发誓后立刻背誓被因果盯上。

任务方向／奖励：凡俗人脉、地契情报、禅院内部平面图；奖励取向为情报费、护身符、引荐去龙王庙／土地祠，不发灵宝。

**二阶（小妖卒·山寨眼线）**

切入身份／时点：黄风岭外围小妖、流沙河巡河小卒或车迟国侍香小道，锚定**黄风怪招兵**或**车迟国三仙立国师前**。

初始事件：寨主要你去刺探「是否有龙气结阵」；你带回的一句假情报会让取经人多走三天冤枉路，也会让你被对立山头剥皮。

开场白建议：「风里有金屑，像谁把龙鳞磨进了砂。寨主说，拖住那只猴的人会赏内丹；土地公公却说，多嘴的小妖活不过下一声佛号。你把刀藏在笑里，决定今晚吹哪边的风。」

关键NPC立场：**黄风怪**（可投名状）；土地／山神（怕两边）；**虎力大仙**门下杂役（争香火）。

主线钩子／支线：送假军情拉长黄风岭对峙；支线——偷换车迟国祭器；支线——给拦路人（敖徒）留一封不具名的风向条。

危险度／规避：中。规避火眼金睛直视、菩萨法旨；莫贪行者身上毫毛。

任务方向／奖励：小妖功法、三昧神风边角料、香火钱；为人脉与「可被大妖点名」的资格。

**三阶（地仙边·龙族小劫）**

切入身份／时点：泾河余脉巡河校尉、西海龙宫外遣、或地府勾魂小吏，锚定**泾河龙死余波、敖徒入灵山前**的龙族分赃与问罪。

初始事件：你奉命清点泾河库藏，却发现少了一枚本应随五公子西行的「安家龙符」；西海来使要你三日内交出替罪者，否则以「通天庭」论处。

开场白建议：「河底的骨还在响，像龙王的名字被魏徵的梦又斩了一次。你握着空符匣，听见西海的潮声与灵山的钟声叠在一起——有人要把一条小龙送去敲钟，有人要把你送去填账。」

关键NPC立场：**西海龙王**（务实护族）；泾河旧部（恐慌）；地府勾魂使（公事公办）；未来的**敖徒**尚在司钟路上。

主线钩子／支线：护下或调换龙符，改变敖徒入灵山的时间窗；支线——查清斩龙背后多头分账；支线——拦下阿难提前递来的「要龙」文书。

危险度／规避：中。规避天庭钦差、地府铁面；莫在龙王灵前撒谎被龙威反噬。

任务方向／奖励：龙宫水法残卷、避水珠、龙族人情；为后续灵山线买门票。

**四阶（地仙稳·夜叉与偏将）**

切入身份／时点：天庭夜叉队副、龙宫偏将或灵山外门护法，锚定**阿难派夜叉追杀下界敖徒**前后。

初始事件：你领到缉拿「逃钟孽龙」的金牌，第一眼却看出金牌上的逃遁时间被改过——若照章办事，你是阿难刀；若抗命，你是下一具香板下的魂。

开场白建议：「金牌是热的，像刚从功德炉里夹出来。天上说那龙逃了，河里却有血腥的锐金气。你把令旗插在云里，决定是报假军情，还是把真相钉进下一道表章。」

关键NPC立场：**阿难尊者**（要你背锅）；**敖徒**（可杀可放可交易）；同行夜叉（怕事）；**如来**远在法座（不可直谏）。

主线钩子／支线：放水让敖徒逃到高老庄；支线——窃取阿难假报逃遁的原稿；支线——与西海暗通款曲。

危险度／规避：中高。规避当堂顶撞如来；莫被锐金龙息正面轰中。

任务方向／奖励：天兵制式兵刃、土遁符、一线「可被真龙记住名字」的因果；勿发先天灵宝。

**五阶（天仙·司钟与禅院）**

切入身份／时点：大雷音寺司钟副手、禅院护法神、或下界监察香火的天仙，锚定**误钟案前后**与**观音禅院二十七天**。

初始事件：钟楼缺一响，你被点名连坐；阿难微笑着让你「指证敖徒」，同时有一道不属于灵山的系统气机在下界闪了一下——你不知那是什么，只知再错一时辰，香板有你的份。

开场白建议：「钟杵比你的话重。阿难师兄的笑容比戒尺软，也比戒尺毒。你听见下界有人用你师父的声口说话，把一整个禅院的工期往泥里按——你要报警，还是把这声假阿难当成救命的缝？」

关键NPC立场：**敖徒**（刚下界，玄仙在望）；**阿难尊者**（要干净证词）；**金池长老**（贪）；**观音**尚未下场救火。

主线钩子／支线：在误钟案中做伪证或反杀伪证；支线——把禅院滞留多拖三天；支线——偷录阿难威胁龙魂的话。

危险度／规避：高。规避如来法眼、观音净瓶；系统气机不可妄探。

任务方向／奖励：天仙功法、避香板符箓、灵山地图边角；为「知道取经是工期」的知情权。

**六阶（玄仙·假唐僧与月档奖励）**

切入身份／时点：游方玄仙、龙族旁支真传、或车迟国「客卿」，锚定**高老庄假唐僧收八戒**至**五火神焰扇结算**。

初始事件：你识破田间那个「唐僧」龙气不稳，他却用一份完美的田契反锁你的道场——要么帮他把真唐僧多堵一旬，要么明日乡民告你「与妖婚」。

开场白建议：「钉耙在月光下痒，像天蓬又做了一次梦。假的佛爷在堂上念经，真的佛爷在庄外吃土。你掐指一算，多拦七天，便有人成丹、有人成仇——而你的道场契，落在人家袖里。」

关键NPC立场：**敖徒**（可盟可竞）；**猪八戒**（已被先收）；**孙悟空**将至（杀机）；**阿难**派来的后续杀手。

主线钩子／支线：助假唐僧圆满月档；支线——偷学移山与龙珠药力边角；支线——在金仙劫雷夜护法或偷雷。

危险度／规避：高。规避行者棒影、劫雷误伤；莫夺系统主奖励硬抢五火扇。

任务方向／奖励：玄仙丹方、后天法宝残件、龙族人情；可旁观金仙蜕变但不可冒充主角。

**七阶（金仙门槛·黄风与灵吉）**

切入身份／时点：金仙散修、妖王座下军师、或灵山外派罗汉级监察，锚定**黄风岭扇灭灵吉**前后。

初始事件：灵吉菩萨法旨到山，要你做「定风内应」；黄风怪却拿敖徒的帖子请你布第二道风眼——选错边，莲台下无完尸，扇火里也无完尸。

开场白建议：「砂石里有莲香，也有焦糊的五火。菩萨的弹指还没落下，龙扇已经先响了一声。你把定风丹含在舌下，尝出它是甜的，也是毒的——因为有人要烧的不是风，是剧本。」

关键NPC立场：**敖徒**（金仙五爪龙）；**黄风怪**（盟主姿态）；**灵吉菩萨**（原解围位）；**孙悟空**（被困怒战）。

主线钩子／支线：改写灵吉下场（擒／放／换）；支线——夺定风丹；支线——把战报改成「妖风自散」骗过观音一日。

危险度／规避：极高。规避五色神光误摄、五火扇正面；**超阶**如来／观音本体勿硬刚，情报优先。

任务方向／奖励：金仙心得、定风类宝物、妖盟席位；阵法残片为下一难通天河铺路。

**八阶（金仙高·长阻与八百难）**

切入身份／时点：金仙后期大能、牛魔王座上宾、天庭特使或地仙之祖门下，锚定**通天河长阻**、**女儿国九曲黄河阵**或**火焰山路阻**。

初始事件：你收到两道并行的符：一道要你「助取经人破阵立功」；一道是拦路人阵旗上的星斗坐标——站错位，你会被周天星力绞成祭旗。

开场白建议：「河冰下压着三百多天的工期，阵旗上钉着齐天的怒意。有人已经在西天问出『八百难』，有人还在东边借芭蕉扇的风。你踏入星斗缝，听见龙珠小世界里潮生潮灭——这一步，要么名勒灵山，要么名勒龙册。」

关键NPC立场：**敖徒**（布局总工）；**孙悟空**（消耗战狂）；**牛魔王**／**万圣公主**（盟友网）；**观音**／**哪吒**／**二郎神**（干涉变量）；远景**如来**（超阶，只可情报）。

主线钩子／支线：助长阻满季／满年档；支线——偷学先天八卦一爻；支线——在火焰山决定站牛魔还是站灵山。

危险度／规避：贴近顶点。**超阶巅峰（如来等）：存在·情报优先／条件性胜利**；严禁写「镇印后战力归零」式取胜。规避人种袋、金铙、混沌钟误震、九曲黄河阵灭顶。

任务方向／奖励：阵旗残柄、先天材料边角、三界双面间谍身份；奖励止于金仙高～太乙边，不发大罗道果。
