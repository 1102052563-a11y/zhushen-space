## 剧情

**【作品来源】**
《黑兽～被白浊沾污的高傲圣女～》（日文：黒獣 ～気高き聖女は白濁に染まる～）由 Liquid 制作，2010年4月发售的王国蹂躏·强制侍奉向AVG；脚本浅生詠/和泉万夜，原画日陰影次；衍生含魔人Studio OVA、小说与续作《黑兽Ⅱ》。舞台在塞尔努斯大陆西部厄俄斯（亦作Eostia/Eos）平原：北方黑暗精灵魔军与南方人类—高等精灵同盟以七砦对峙数百年。高等精灵圣女**塞蕾丝汀·卢克鲁斯**（セレスティン・ルクルス / Celestine Lucullus，通称塞蕾丝汀；亦见塞勒斯缇娜译）被同盟尊为女神**拉伦蒂娅**（ラーレンティア / Larentia）转生，坐镇**白之城**（白の城，七盾中的圣座）以预知调度全军，并亲手凝聚**七盾同盟**（プリンセスナイツ/七盾公主骑士）。**本档案「黑兽魔王觉醒-女神堕落」**只截取并放大原作圣女—教廷线：**预知失灵 → 护卫链断裂 → 大司教甩锅「骗国魔女」 → 光之结界与白之城名分崩坏 → 女神象征被公开踩碎。** 对应OVA标题意象「女神堕落的早上」。写信仰、预知、教产、名分战争与条件性胜利，**不写色情细节**。与同批「大陆统一」（砦旗军政）、「沃尔特进化」（魔王加冕）严格区分：本档镜头锁死白之城侧殿与神意死亡。

**【世界定位】**
时间锚=黑犬佣兵团背刺同盟之后、侍奉国家法统外溢之际、**白之城尚未彻底法统改写但已进入名分终盘**的数日到两周。地理锚=白之城大圣堂—圣女侧殿—预知室—护卫走廊—展示广场—教廷文书库—城外难民棚。大陆统一战役是背景鼓点；主戏是「神意先死于文书，后死于广场」。契约者在此档的胜利几乎全是**政治与象征**：保预知残片、揭伪广播、拆教廷话术、护圣女离开展示半径。一句话：这是一场关于「谁有权代表神」的内战，不是热血屠魔王副本。

**【世界观 · 力量体系】**
无修仙境界。本档力量来源按「能否改写军令合法性」排序：
1. **预知**：塞蕾丝汀的战略外挂。真预知能让七砦互援半步领先；被污染后同一夜发出相反军令，同盟自相踩踏。
2. **神术与光之结界**：白之城圣座外围以高等精灵神术维持的防护层与「不可亵渎」心理墙。结界碎裂=民心先碎半截。
3. **圣器与骑缝章**：军令、降表、互援令须侧殿骑缝章；缺章则前线可拒收。
4. **护卫剑**：克劳迪娅·莱凡坦等把「人」挡在「神」前面；盾缺人时神先死。
5. **教廷话术与伪神意广播**：大司教布道词、假预言喇叭、伤兵营谣言。能在不流血的情况下杀死圣女的合法性。
6. **黑犬武勇与展示政治**：沃尔特要的是「女神跪」的画面完整度，不是首级。

战力分层（破坏力优先，宁低勿高）：
1. 民兵、难民、圣堂杂役、伤兵≈一阶：可堵巷、可传谣，单兵不堪。
2. 精锐卫士、低阶神职武装、黑犬清场队≈二阶：守门、押送、清广场。
3. 砦主级骑士、护卫链长、伪广播技工头目≈三阶：决定侧殿能否撑一夜。
4. 塞蕾丝汀预知指挥态、克劳迪娅满编护卫、佣兵王沃尔特单体≈四阶：决定名分战走向。
5. **女神象征被公开踩碎后的信仰崩盘 + 白之城名分战争高潮 + 统一军压境心理场**≈**五阶**：地区秩序级事件；个人仍强，真正恐怖是「没人再肯为神意死」。

乐园阶位映射：民兵/难民≈一阶；精锐卫士/清场队≈二阶；砦主/护卫长/伪广播头目≈三阶；圣女预知指挥与佣兵王单体≈四阶；**女神堕落周的信仰崩盘与名分战争完整态≈五阶（本世界覆盖）**。顶点仍是沃尔特体系+伪神意机器；条件性胜利=鉴墨/盗稿/毁喇叭/护书记/送真页/拦孤军/拆布道词。严禁「被封印所以战力为零」「圣女突然无敌」。

**【地理 · 舞台】**
- **白之城大圣堂**：公开布道与「骗国魔女」宣判预演地；钟声是政治信号。
- **圣女侧殿**：塞蕾丝汀起居与签章处；密道两出口（钟楼/下水道）。
- **预知室**：禁火、特供墨、原文册；矛盾预言最先在此成双。
- **护卫走廊**：四步一停靴钉节奏；乱步夜=内奸夜。
- **教廷文书库**：布道词三稿、骑缝章登记、与黑犬旧通信可能藏此。
- **展示广场**：木架按「全城可见」而非「可处死」设计；女神之晨主舞台。
- **六砦军械 / 七砦伤兵营**：换防表与谣言源头；鉴墨蒸馏角在伤兵营。
- **城外难民棚与粥棚**：民心副锚；普莉姆若现身会被优先同框。
- **地下水道—洗衣房出口**：书记逃亡线。

**【世界剧情线】**
① **神意同盟（前史压缩）**  
厄俄斯平原百年拉锯。黑暗精灵女王**奥尔加·迪斯克伦蒂亚**魔力衰竭，总力战窗口打开。塞蕾丝汀以拉伦蒂娅转生之名凝聚七盾，雇佣**沃尔特**的黑犬佣兵团为前卫。她本人并非最强剑，却是同盟的**大脑**：预知决定谁增援、谁弃线、谁保民。白之城因此既是圣座也是总参谋部。民心公式简单：「圣女看见了，所以我们还能赢。」

② **黑之城易主与预知第一次污染**  
黑犬攻陷黑之城、俘奥尔加后，不归还同盟，迁根据地建国「侍奉国家」，并对旧雇主开战。塞蕾丝汀的预知从「可调度的胜机」变成「越看越跪」的泥沼：同一战场出现两种未来，笔一顿，前线就多死一队。她开始自限「只发保民令不发决战令」，大司教立刻把这解释成失职与怯懦。神意死亡的第一刀，是**指挥迟疑**。

③ **护卫链危机与换防表**  
**克劳迪娅·莱凡坦**分身乏术：城墙与侧殿不能同时满编。**格拉夫·莱凡坦**家法线与换防表篡改，使关键夜「盾」缺人。护卫靴钉乱步、尸衣番号错乱、军械三钥少一把——任何一条都能在堕落夜放大成侧殿失守。契约者若在此阶段拿回换防表原件，等于给圣女买五分钟。

④ **教廷分裂：脸与产**  
教廷从来不是铁板。一派要真神（圣女预知仍可救命），一派要教产与活命。**大司教**在军事崩溃前就起草「骗国魔女/预知不准」布道词：温和稿→嫁祸稿→钉死魔女稿。第三稿与展示日绑定。钟声被他提前停=甩锅日临近；圣女强行乱钟开会=最后一次公开对质窗口。神意先死在文书库，后死在广场。

⑤ **光之结界碎裂与伪神意广播**  
白之城外围光之结界在双重压力下裂：外是黑犬展示部队与伪广播喇叭（外形圣器、内里黑之城工坊编号），内是教廷停钟、伤兵营「圣女卖城」三字谣（源头常指向**托马斯·比尔兹利**）。结界碎不是单纯法术失效，而是**「不可亵渎」心理墙 simultaneous 崩**。军士看见圣堂灯灭，比看见砦旗倒更快溃。

⑥ **名分战争与女神堕落名场面**  
沃尔特不要圣女首级，要画面：**女神之晨**。伪神意广播 + 展示性押送 + 预知公开碎裂 + 神职笼训「低头角度」。同盟军士气断崖，降表开始比刀快。艾丽西娅若孤军来救，中点辱旗诱饵正等着她；普莉姆若在粥棚，会被抓去同框。堕落完成=侍奉秩序获得「神也跪了」的广告。

⑦ **终局三分支（可被契约者改写完整度，不可无代价抹掉逻辑）**  
A. 教廷改投侍奉秩序，圣女成哑神（能说话，不能再签发合法军令）。  
B. 圣女被救离开展示半径，但预知永久损毁，残部保留「未来再组织」种子。  
C. 白之城玉碎，名分连同城砖一起碎，统一加速。  
蝴蝶边界：可改展示完整度、可改谁先死谁先跑、可改布道词当场穿帮；不可无代价改「侍奉国家已存在」「沃尔特要画面」「教廷有人要卖神保产」。

关键NPC立场：**塞蕾丝汀·卢克鲁斯**——求你别让预言再被用来杀人，可给残页与自限令原文；**克劳迪娅·莱凡坦**——用命换侧殿五分钟，要换防表真相；**大司教**——买你的笔证与第三枚印；**沃尔特**——要完整女神之晨，不急杀你；**艾丽西娅·阿尔库托鲁斯**——若知圣女危会孤军来，需你指路或拦路；**托马斯·比尔兹利**——卖你第三份更假的预言；**希克斯**——清场与押送倒计时；**克劳斯·柯蒂斯**——鉴墨换女儿出城；**预知室书记**——活着=真链。

主线钩子/支线：主线=阻止或改写「女神之晨」展示完整度。支线：①盗布道词三稿；②护书记走下水道；③定位并毁/搬伪广播源；④送真预言给一砦；⑤换克劳斯公开证词；⑥夺回骑缝章；⑦证光核钥匙时间线；⑧护普莉姆离粥棚；⑨截第三枚主教印。

危险度/规避：极高。展示日广场=死亡漏斗。须规避：无准备冲沃尔特本阵/展示中心；无证据空喊复国。条件性胜利优先。

任务方向/奖励：目标=圣女离开展示半径 / 伪广播源摧毁或当众拆穿 / 教廷话术当众破产（三选二即阶段胜）。奖励：预知残页、侧殿符、教廷黑材、护卫旧部人情、密道蓝图、鉴墨试剂——不发「一刀斩魔王」神器。

五阶补充钩子A：假预言试探**大司教**「骗国魔女」话术；成功侧门通行，失败进神职笼训练名单。

五阶补充钩子B：鉴墨——圣堂特供墨 vs 松烟墨。实验室在七砦伤兵营蒸馏角。**克劳斯·柯蒂斯**可合作；公开鉴墨需护他女儿出城。

五阶补充钩子C：护预知室书记走下水道，洗衣房有眼线。书记活着=真预言链不断；死=伪令横行。

五阶补充钩子D：伪神意喇叭外形圣器、内里黑之城编号。毁之降展示完整度；搬到教廷大厅反杀话术。

五阶补充钩子E：拦**艾丽西娅·阿尔库托鲁斯**孤军：中点辱旗处给真或假路线。真→侧殿；假→展示部队。

五阶开场变体：脚手架木工窃听**希克斯**排练押送队形；锯梁延期或刻旧盟纹激民心。

五阶阶段胜：圣女离开展示半径 / 伪广播摧毁或拆穿 / 教廷话术破产——三选二。残胜：四选二带出城（残页/孤儿名簿/侧殿账本/密道蓝图）。

五阶雷区：不写色情；不无脑圣女；不无准备冲广场；不写成大陆统一砦旗战或沃尔特血旗加冕。

五阶行动节拍1：入场后先确认「谁在发令、谁在收尸、谁在记账」。发令者决定你能不能活过第一小时，收尸者决定失败态，记账者决定奖励是否被吞。

五阶行动节拍2：第一笔交易必须小——情报换路条/鉴墨半天，而不是直接碰展示场上的沃尔特体系。露富即杀劫在本阶成立。

五阶行动节拍3：建立至少一个具名盟友与一个具名仇敌；两边都空的契约者会在第二日被系统当作「无主物资」。

五阶行动节拍4：主线节点到来前完成一次侦察：入口、逃口、人质、证据四样至少两样到手。

五阶行动节拍5：阶段结算看条件性胜利指标，不看击杀榜。奖励池锁定：残页/侧殿符/教廷黑材/护卫人情/密道蓝图。

五阶失败教育：硬刚顶点、无准备冲展示/仪式、把关键NPC当群像工具人——三种写法禁止。失败具象：神职笼、伪章横行、书记死、剑姬死于辱旗、第三稿成立。

五阶台词锚：本地人不会相信你能「一个人推翻秩序」；他们最多信你能让某一夜少死一些人或多死一些人。

五阶时间压力：倒计时明确（停钟/木架/神职笼/女神之晨）。拖延也是策略，但拖延会改奖励表与完整度。

五阶关系网：优先点名原作真名互动，禁止用「群像/牙人/红颜」替代人名；交易对象必须能被追踪到势力。

五阶战场读图：先画三层——外围难民棚与双税点、中层文书库与护卫走廊、核心侧殿/预知室/展示广场。契约者默认从外围进，只有证据齐才进核心。

五阶证据链：文书、钥匙、物证、证人四选二才能触发「公开揭露」类胜利；只有热血演说无效。

五阶展示完整度机制：圣女在场、神职低头、伪广播不断、孤军未至、民心同框五项；每破一项完整度-1；低于阈值则改期或整肃——阶段胜窗口。

五阶与邻档隔离：不结算砦旗数，不写血旗加冕；只结算名分与神意死亡相关指标。
