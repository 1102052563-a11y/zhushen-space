# 黑兽魔王觉醒-女神堕落
<!--meta lib=主库 tiers=五-->

## 剧情

**【作品来源】**
同 Liquid《黑兽》系列（2010 AVG + 魔人OVA等）。**本档案「黑兽魔王觉醒-女神堕落」**聚焦动画/原作圣女线核心：**高等精灵圣女塞勒斯缇娜·卢克鲁斯（女神转生）在统一战争中预知失灵、护卫链断裂、教廷甩锅，最终作为「神意」象征堕落/被公开踩碎。** 对应OVA标题意象「女神堕落的早上」。只写信仰、预知、护卫与政治，**不写色情细节**。

**【世界定位】**
舞台仍在厄俄斯，但镜头锁死**白之城—圣女侧殿—六七砦护卫链**。大陆统一是背景鼓点；主戏是「神意死亡」。契约者在此档的胜利条件几乎总是**政治与象征**，不是屠城。

**【世界观 · 力量体系】**
同黑兽军政体系，本档额外强调：
- **预知**：塞勒斯缇娜的战略外挂；被污染则全军调度崩溃。
- **神术/圣器**：教廷权柄，可被伪广播。
- **护卫剑**：克劳迪娅等把「人」挡在「神」前面。
分层同一～五阶映射；本世界覆盖**五阶**——女神堕落发生在统一高潮，地区秩序级事件。
乐园阶位映射：民兵≈一阶；精锐≈二阶；砦主/护卫长≈三阶；圣女预知指挥与佣兵王≈四阶；**女神象征被公开踩碎后的信仰崩盘+统一军压境≈五阶**。顶点仍是沃尔特体系；条件性胜利=保预知残片/揭伪广播/护圣女离展示半径。

**【地理 · 舞台】**
白之城大圣堂与侧殿；圣女预知室；护卫走廊；六砦军械与七砦伤兵营；教廷文书库；黑犬临时祭坛（伪神意广播源可能在此）；城外展示广场。

**【世界剧情线】**
① **神意同盟**  
塞勒斯缇娜以女神转生身份凝聚七盾，预知让同盟在总力战中占调度优势。她不是花瓶，是**大脑**。

② **黑犬背刺后的预知污染**  
黑之城易主、侍奉法广播开始后，她越预知越看见「跪地的未来」。指挥从果断变迟疑，前线收到矛盾指令。

③ **护卫链危机**  
克劳迪娅分身乏术：城墙与侧殿不能同时满编。格拉夫家法线、换防表篡改，使「盾」在关键夜缺人。

④ **教廷甩锅**  
大司教起草「骗国魔女/预知不准」布道词，时间表可能早于军事崩溃。神意死亡先发生在文书里。

⑤ **女神堕落名场面**  
伪神意广播+展示性押送+预知公开碎裂。同盟军士气断崖。沃尔特要的是画面，不是圣女首级。

⑥ **终局三分支**  
教廷改投侍奉秩序 / 圣女被救但成哑神 / 白之城玉碎。大陆统一因此加速或变形。

**【主要人物】**
- **塞勒斯缇娜·卢克鲁斯**（圣女）｜性格：悲悯、自责、不肯弃民。｜能力：预知、神术调度。｜弧光：大脑→祭品→（可选）哑神。｜立场：求契约者「救的是人不是神位」。
- **克劳迪娅·莱凡坦**｜护卫剑；失职=堕落加速器。
- **大司教**｜甩锅主谋；要教产不要真神。
- **沃尔特**｜要画面的征服者。
- **艾丽西娅·阿尔库托鲁斯**｜最可能孤军救驾的剑。
- **普莉姆·菲尔丽热**｜民心副锚。
- **克劳斯·柯蒂斯**｜证词可炸教廷话术。
- **格拉夫·莱凡坦**｜家法内奸。
- **托马斯·比尔兹利**｜谣言与假预言。
- **希克斯/基恩**｜押送与清场。
- **奥尔加·迪斯克伦蒂亚**｜对照「象征先死」。
- **白之城预知室书记**｜记录预言原文的关键证人。

**【势力图谱】**
白之城教廷（脸 vs 产）、圣女护卫链、七砦残部、黑犬展示部队、民心与难民潮。

**【贵重物品】**
圣女预知原文册；伪广播法器；侧殿通行符；换防表原件；布道词草稿；圣器残件；「女神之晨」仪式日程板。

**【隐藏剧情 · 伏笔】**
- 预知污染可能来自黑之城圣器逆向工程。
- 大司教与某黑犬干部有旧通信。
- 塞勒斯缇娜曾预见契约者——视为「未写入的变量」。
- 救下圣女不等于复国，只等于保留「未来再组织」的种子。

**【大事记时间线】**
七盾凝聚 → 雇黑犬 → 背刺 → 预知污染 → 护卫缺口 → 布道词定稿 → 展示/堕落 → 信仰崩盘 → 统一加速。

**【叙事基调 · 雷区】**
冷、悲、制度。忌色情；忌把圣女写成无脑圣母；忌五阶外随意请神灭魔王。最早切入：预知首次自相矛盾的那周。

**【女神堕落·预知双墨】**
真预言用圣堂特供墨，伪广播用松烟墨。侧殿墙灰可检。

**【女神堕落·钟声政治】**
停钟=大司教日程；乱钟=圣女强行开会。

**【女神堕落·护卫靴钉】**
克劳迪娅规定四步一停。乱步夜=内奸夜。

**【女神堕落·布道词版本】**
至少三稿：温和、嫁祸、彻底钉死魔女。第三稿与展示日绑定。

**【女神堕落·展示木架】**
木架尺寸按「可被全城看见」而非「可处死」。沃尔特要画面。

**【女神堕落·孤军救驾窗】**
艾丽西娅从一砦到白之城最快骑程已知；黑犬在中点设辱旗诱饵。

**【女神堕落·书记逃亡线】**
预知室书记走地下水道，出口在七砦伤兵营洗衣房。

**【女神堕落·假神意喇叭】**
伪广播法器外形像圣器。拆开是黑之城工坊编号。

**【女神堕落·民心粥棚】**
普莉姆若出现在粥棚，堕落日会被优先抓去同框展示。

**【女神堕落·哑神定义】**
预知永久损毁后圣女仍可说话，但不能再给军令合法性。

**【女神堕落·侧殿密道】**
密道出口两处：钟楼与下水道。下水道被大司教加锁。

**【女神堕落·骑缝章】**
圣女军令骑缝章丢一枚=伪令可乘虚。

**【女神堕落·伤兵谣言】**
「圣女卖城」三字从伤兵营传出，源头是比尔兹利。

**【女神堕落·神职笼】**
展示前神职被单独关训「低头角度」。

**【女神堕落·预知室禁火】**
禁火是为了保护墨与纸；伪令常在有火把的侧廊写成。

**【女神堕落·信仰战场日志1】**
堕落倒计时第1时辰：预知室收到第2类矛盾指令。**塞勒斯缇娜·卢克鲁斯**试图以「只救平民不调度军」自限，反而被**大司教**解释为失职。**克劳迪娅·莱凡坦**申请侧殿满编被驳回——军械库缺人。**托马斯·比尔兹利**在伤兵营散布「圣女卖城」。契约者可做：鉴墨、盗稿、护书记、送真页给一砦、定位伪喇叭。每成功一项，展示日完整度下降一成；完整度低于五成则**沃尔特**可能改期——这是阶段胜。禁止在展示广场无准备冲锋。本日志条目互相独立，供不同时点切入。

## 阶位切入点

> 阶位↔：同黑兽映射；本世界仅**五阶**。顶点=沃尔特体系+信仰崩盘场。超阶表述不适用；五阶内条件性胜利。

**五阶（女神堕落周 · 预知与名分）**

切入身份/时点：契约者以「白之城外籍神官书记 / 护卫链临时剑 / 教廷审计员 / 黑犬伪神意技术奴」切入，锚定**预知已连续三天自相矛盾、六七砦换防表异常、展示广场木架开工**的堕落前夜。

初始事件：预知室书记把你拉进侧殿——桌上两份「圣女亲笔」预言，墨迹不同，指令相反：一份令一砦死守，一份令一砦弃城救白之城。**大司教**派人来收「假的那份」；**克劳迪娅**派人来收「真的那份」。你必须在十息内判断哪份是伪广播产物，并决定交给谁。

开场白建议：「侧殿的烛火照着两张都像圣女笔迹的纸。城外木锤敲展示架，像给神意钉棺材。大司教的人在左廊，克劳迪娅的靴钉在右廊。你手里没有剑，只有选择：让哪一句谎言先杀死同盟。」

关键NPC立场：**塞勒斯缇娜**——求你别让预言再被用来杀人；**克劳迪娅**——用命换侧殿五分钟；**大司教**——买你的笔证；**沃尔特**——要完整画面，不急杀你；**艾丽西娅**——若知圣女危会孤军来，需你指路或拦路；**托马斯·比尔兹利**——卖你第三份更假的预言；**希克斯**——清场倒计时。

主线钩子/支线：主线=阻止或改写「女神之晨」展示。支线：①盗布道词草稿；②护书记撤离；③定位伪广播源；④送真预言给一砦；⑤换克劳斯公开证词。

危险度/规避：极高。展示日广场=死亡漏斗。规避无准备冲沃尔特本阵。

任务方向/奖励：目标=圣女离开展示半径 / 伪广播源摧毁 / 教廷话术当众破产（三选二即阶段胜）。奖励：预知残页、侧殿符、教廷黑材、护卫旧部人情。

五阶补充钩子A：假预言试探**大司教**「骗国魔女」话术；成功侧门通行，失败进神职笼训练名单。

五阶补充钩子B：鉴墨——圣堂特供墨 vs 松烟墨。实验室在七砦伤兵营蒸馏角。**克劳斯·柯蒂斯**可合作。

五阶补充钩子C：护预知室书记走下水道，洗衣房有眼线。书记活着=真预言链不断。

五阶补充钩子D：伪神意喇叭外形圣器、内里黑之城编号。毁之降展示完整度；搬到教廷大厅反杀话术。

五阶补充钩子E：拦**艾丽西娅·阿尔库托鲁斯**孤军：中点辱旗处给真或假路线。

五阶开场变体：脚手架木工窃听**希克斯**排练押送队形；锯梁延期或刻旧纹激民心。

五阶阶段胜：圣女离开展示半径 / 伪广播摧毁 / 教廷话术破产——三选二。

五阶雷区：不写色情；不无脑圣女；不无准备冲广场。

五阶行动节拍1：入场后先确认「谁在发令、谁在收尸、谁在记账」。发令者决定你能不能活过第一小时，收尸者决定失败态，记账者决定奖励是否被吞。

五阶行动节拍2：第一笔交易必须小——情报换路条，而不是直接碰展示场上的沃尔特体系。露富即杀劫在本阶成立。

五阶行动节拍3：建立至少一个具名盟友与一个具名仇敌；两边都空的契约者会在第二日被系统当作「无主物资」。

五阶行动节拍4：主线节点到来前完成一次侦察：入口、逃口、人质、证据四样至少两样到手。

五阶行动节拍5：阶段结算看条件性胜利指标，不看击杀榜。奖励池锁定：残页/侧殿符/教廷黑材/护卫人情。

五阶失败教育：硬刚顶点、无准备冲展示/仪式/门心、把关键NPC当群像工具人——三种写法禁止。

五阶台词锚：本地人不会相信你能「一个人推翻秩序」；他们最多信你能让某一夜少死一些人或多死一些人。

五阶时间压力：本阶剧本默认倒计时明确（砦陷/展示/加冕/开门/纹路合拢），拖延也是策略，但拖延会改奖励表。

五阶关系网：优先点名原作真名互动，禁止用「群像/牙人/红颜」替代人名；交易对象必须能被追踪到势力。

五阶战场读图：先画三层——外围补给、中层冲突、核心仪式/展示/门心。契约者默认从外围进，只有证据齐才进核心。

五阶证据链：文书、钥匙、录音、证人四选二才能触发「公开揭露」类胜利；只有热血演说无效。
