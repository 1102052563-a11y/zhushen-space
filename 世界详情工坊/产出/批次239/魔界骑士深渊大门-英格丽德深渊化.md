# 魔界骑士深渊大门-英格丽德深渊化
<!--meta lib=主库 tiers=六-->

## 剧情

**【作品来源】**
对魔忍宇宙·魔界骑士**英格丽德**相关作品线（Lilith Soft系列游戏/动画/RPG差分）。**本档案「魔界骑士深渊大门-英格丽德深渊化」**专写：**英格丽德在深渊门能量/仪式/上位实验作用下，从骄傲的魔界骑士质变为深渊污染体**——战力上升、可控性下降、忠诚与自我被改写。聚焦权力、仪式、可逆分支与六阶止损战；**不写色情凌辱细节**。

**【世界定位】**
时间可与大门开启重叠或紧随其后。空间=门缘仪式阵、魔界临时圣所、人界封印设施。主戏=**英格丽德个人线质变**。

**【世界观 · 力量体系】**
同对魔忍/魔界体系。深渊化特则：
- 深渊魔素改写半魔结构；
- 战力冲向六阶上限（地区灾难感）；
- 命令抗性上升，可能连魔界上级也压不住。
乐园阶位映射：……；**深渊化英格丽德≈六阶顶点级战力（本世界覆盖）**。条件性胜利=断供能/三分钟窗口插入净化/封印为活体门钥；严禁「打掉理智就变弱」。

**【地理 · 舞台】**
深渊纹路祭坛；门缘供能柱；净化池（对魔忍侧）；魔界锁链监牢；都市废弃线；记忆信物保管处。

**【世界剧情线】**
① **骑士常态**  
英格丽德以实力与契约行事，猎人对魔忍，完成魔界任务。

② **污染导入**  
门开回流 / 强制契约 / 实验注入 / 战斗吸入。纹路开始爬上甲与肤。

③ **半深渊**  
仍能对话，逻辑更非人；可能主动加速开门或屠杀「弱者」。

④ **止损战争**  
人界要封印；魔界要回收或废弃；企业要取样。

⑤ **三终局**  
恢复 / 完成深渊化灾厄 / 活体门钥封印。

⑥ **余波**  
无论分支，门与深渊议题永久化；英格丽德名号成为传说级警告。

**【主要人物】**
- **英格丽德**｜深渊化主体；六阶核心。
- **埃德温·布莱克**｜评估利用或废弃。
- **井河朝阳**｜最可能执行止损斩击的人界顶点之一。
- **井河樱**｜羁绊/救援线可能点。
- **八津紫**｜情报与支援。
- **深渊祭司**｜仪式主持。
- **Nomad取样队长**｜要活体样本。
- **魔界锁链官**｜回收部队。
- **记忆信物保管者（版本相关，不详则写职责不写假名）**｜唤回关键。

**【势力图谱】**
深渊仪式派；魔界回收派；对魔忍止损队；企业取样队；门自体意志（非人）。

**【贵重物品】**
深渊纹路抑制剂；净化刃/符；未污染骑士徽；记忆信物；供能柱核心；封印锁链；取样针（企业）。

**【隐藏剧情 · 伏笔】**
- 深渊化可能是上位者清洗「太有自我的骑士」的局。
- 半深渊阶段她仍可能遵守「与强者的决斗约定」——可谈判窗口。
- 活体门钥结局会把她绑死在门上，成为后续所有门任务的BOSS装置。
- 与「大门开启」档案联动：门越稳，她越难恢复。

**【大事记时间线】**
骑士任务期 → 门开/实验 → 纹路出现 → 半深渊 → 止损战 → 三终局。

**【叙事基调 · 雷区】**
悲剧性武力、冷对话。忌无脑狂化脸谱；忌色情；忌「打一顿就好」。最早切入：第一道纹路未合拢的夜。
