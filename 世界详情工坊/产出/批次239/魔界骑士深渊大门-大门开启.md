# 魔界骑士深渊大门-大门开启
<!--meta lib=主库 tiers=六-->

## 剧情

**【作品来源】**
日本 Lilith Soft / Black Lilith 旗下对魔忍（対魔忍 / Taimanin）宇宙的场景切片档案。系列以 2005 年《対魔忍アサギ》为起点，扩展至《对魔忍紫》《阿莎姬 2/3》《雪风》《ZERO》《决战竞技场》及 OVA《魔界骑士英格丽德》（Makai Kishi Ingrid / Hell Knight Ingrid）等。本条目**不是**阿莎姬本传全本复述，而是以「人界—魔界交界处的战略级位面通道：深渊大门从封闭到开启」为唯一主轴的六阶危机档案。核心人物沿用系列真名：魔界骑士**英格丽德**、诺玛德创始者**埃德温·布莱克**、对魔忍**井河阿莎姬 / 井河樱 / 八津紫**、诺玛德干部**胧 / 弗斯特**、魔医**桐生佐马斗**等。气质：冷酷特务战、门钥与结界工程、组织博弈与条件性胜利；**禁止**色情凌辱细节，战斗写到刀光、魔焰、 Horde 潮汐与疏散即可。公开资料可核：VNDB 英格丽德词条、英文维基 Taimanin 条、ANN / MAL 的《魔界骑士英格丽德》条目、Lilith 官网对魔忍产品线。

**【世界定位】**
近未来日本暗部与魔界（Dark World）长期渗透的双层世界中，本切片锁定**门缘战场**——一处被多方命名为「深渊大门」的战略通道。它不是日常任务里随处可见的小裂隙，而是足以投放地区级灾难、改写一国暗部格局的大门级位面接口。表面可能落在废弃地铁枢纽、旧军事实验室、尤米哈拉（Yomihara）外延空洞或东京湾底人工岛的结界薄弱带；地下则是结界柱阵列、门钥保管库、诺玛德门务实验室与魔界侧桥头堡。契约者被乐园投入的瞬间，世界状态已是「开门倒计时」：门尚未全开，但序列已启动；英格丽德作为魔界骑士与诺玛德二把手，要的是**可控开启**；对魔忍要的是**延迟或关闭**；诺玛德实验室与部分干部要的是**私门/分流**；深渊回流本身则倾向**失控扩大**。本世界覆盖顶点档为**门开启完成态及其地区灾难投影（≈六阶）**，不硬开七阶以上魔界深层。

**【世界观 · 力量体系】**
世界基本规则：人界与魔界由契约、血祭、企业通道与结界共同约束。大规模投射长期被「结界网 + 对魔忍猎杀 + 政府暗部配额」压制，因此小裂隙频发、大门级通道则成为各方必争。死亡对凡人与下忍即终结；高阶魔族可寄生、改造、换体续命，皆有人格崩坏或永久弱点代价。对魔忍以血脉忍法 + 对魔忍装 + 体术/忍刀作战；魔族侧有吸血鬼血权、魔焰剑术、那伽毒素、半兽人蛮力、梦魔精神干扰等；诺玛德把这些能力**产品化**——改造手术、魔素浓缩、门钥复制、结界校准仪。

本切片独有的「门务力学」（写正文必须遵守的可观察规则）：
1. **结界柱阵列**：通常以三至七根主柱锚定门缘。柱体过载会先冒冷雾与紫色静电，再出现金属嘶鸣；三号柱往往最旧、最先过载。校准钥匙可延长或缩短柱寿命，但日志会留下操作者魔纹/指纹。
2. **门钥与伪钥**：真钥在开启序列前约两小时发烫；假钥始终凉。伪信标可短时扭偏门向，但会被写入门日志，后续被魔界侧记仇。
3. **魔素潮汐 / Horde 潮汐**：开启过程分波。第一波多为低阶涌出体（试压）；第二波带军官与编制；第三波可能出现上位投影或领域雏形。把第一波当胜利=自杀。
4. **通讯死区**：门缘约五公里内常规通讯失效；废弃线 B3 中继是已知可抢修的中继点之一。
5. **门心回流**：门心禁区站桩超过约三十秒会遭位面回流撕扯；只允许拆锚冲刺，不允许「站在门心硬刚」。
6. **时刻表差**：魔界骑士团内部时刻表常比人界表快约七分钟——以门脉冲为准，不以手表为准。

战力阶梯（按实际破坏力，宁低勿高；本档案只把任务舞台推到六阶）：
1. **凡人·街头炮灰**：警员、疏导员、黑帮打手。热武器可致命；在门缘是资源不是战力。乐园≈一阶。
2. **下忍·低阶魔物**：见习对魔忍、半兽人杂兵、诺玛德外围打手。碎砖裂石、近战压凡人小队。乐园≈二阶。
3. **中忍·成型魔物**：独当一面的对魔忍、诺玛德中层执行者、魔界正规士兵。端据点、乱军取首。乐园≈三阶。
4. **上忍·精英 Boss**：井河阿莎姬级剑术天才、胧级咒术忍、诺玛德精锐干部。掀建筑群、硬抗常规热武器。乐园≈四阶。
5. **城市级·领主威胁**：能让一城治安与政治同时崩溃的魔灾；魔界骑士精锐、城市级结界崩溃。乐园≈五阶。**英格丽德本体约在五～六交界**：门缘一对一可压多数上忍，但面对「门本身 + 上位投影 + 组织火力」仍须战术。
6. **地区灾难·门开启态**：**深渊大门开启完成态 / 地区级 Horde 投放 / 埃德温·布莱克级组织权柄与本体阴影**。凡俗军队与热武器近乎无效；对话、夺钥、拆锚、扭门向有时比开战更有效。乐园≈**六阶（本世界覆盖）**。

**乐园阶位映射**：街头凡人/见习≈一阶；下忍·低阶魔≈二阶；中忍·成型魔≈三阶；上忍·阿莎姬/胧级≈四阶；城市级魔灾/骑士精锐≈五阶；**深渊大门开启态与诺玛德顶点阴影≈六阶**。宁低勿高；六阶以上魔界更深层、宇宙重置级手段不在本切片硬开。世界顶点（埃德温·布莱克 + 门本身）为跨阶/条件性存在：低阶不写，六阶亦以**情报优先 / 拆锚夺钥 / 扭门向 / 延迟开启**为合法胜利，严禁「被封印所以战力为零」式削弱。

**【地理 · 舞台】**
- **门缘空洞主战场**：地下巨型穹顶，地面嵌有发光结界纹路；中央是尚未完全成形的门环。气压随脉冲起伏，像有巨大的肺在门后呼吸。此处是六阶主舞台。
- **结界柱廊**：环形或放射状排列的主柱与辅柱。每根柱有铭牌、校准接口、过载泄压阀。三号柱历史最久，最先报警；五号柱靠近诺玛德侧实验室通道，常被企业私改。
- **门钥保管库（双锁）**：一锁归魔界骑士仪仗系统，一锁归诺玛德门务实验室。真钥转移时必有武装护送；假钥常被混入车队以钓对手。
- **诺玛德门务实验室 / 黑暗宫殿外延**：合法企业外壳下的魔素浓缩与门钥复制设施。弗斯特系技术、桐生佐马斗式改造黑医痕迹可能同时出现——前者偏「组织实验」，后者偏「个人复仇与亵渎」。
- **对魔忍临时指挥所**：多设在门缘五公里外的废弃线车站、学校地下室或公安暗部前哨。井河阿莎姬若介入，指挥风格是「先保疏散与证据，再谈斩首」。
- **废弃线 B3 中继**：门缘通讯死区里的关键中继点。谁控制 B3，谁就能在 Horde 潮汐中指挥己方；八津紫等情报向对魔忍会优先争此点。
- **魔界侧桥头堡**：门另一侧的骑士团临时营地。英格丽德的骑士徽可通行；人界证件在此是死刑状。
- **都市疏导线**：门缘地表五公里内的撤离走廊。凡人警官与防灾广播是「世界还在假装正常」的标尺；一旦广播改口承认「异象」，政治层已经决定甩锅对象。
- **尤米哈拉 / 东京王国式地下都市（互文）**：系列中诺玛德与魔族共治的法外地带。本切片可借其物流与黑市，但主战场仍是门缘，勿把整场写成东京王国观光。

**【世界剧情线】**
① **前史·结界时代与大门禁忌**  
人界暗部与魔界贵族长期默认一条默契：小裂隙可交易、可猎杀、可当黑市通道；**大门级通道**则是禁忌。对魔忍体系（井河家等血脉忍里 → 后期学园化/国家化）负责猎杀失控涌出；政府暗部把对魔忍当刀，也把诺玛德当「必要的恶」。诺玛德（Nomad）由埃德温·布莱克以跨国企业形态运营：合法资本为壳，魔族技术、改造、竞技场、政治渗透为里。魔界骑士团本是魔界贵族护卫建制——英格丽德出身黑龙骑士团一脉，曾为骑士团领袖级战力，后因对布莱克的绝对忠诚（与对贵族制的幻灭）离开魔界，成为诺玛德公开侧的二把手、秘书与近卫。此阶段世界规则已定：**门是资产，不是风景**；谁掌握门钥与结界柱，谁就掌握下一次地区灾难的开关。

② **裂隙频发与「开门工程」立项**  
近数年小裂隙异常增多：码头、废弃神社、地铁隧道连续出现魔素浓度尖峰。对魔忍侧判定为「有人在校准大门坐标」；诺玛德内部则出现两派——**可控开启派**（以英格丽德为代表：门必须由骑士纪律与布莱克利益框住）与**私门分流派**（实验室头目、部分弗斯特系技术官、想绕过骑士团记账的企业干部）。埃德温·布莱克本人极少亲临门缘；他更可能通过晚宴、政要与「评估报告」决定是否允许开启。立项文件里不会写「毁灭东京」，而会写「战略通道测试 / 魔素物流优化 / 对魔忍压力测试」——语言本身就是伪装。

③ **门钥转移夜（危机可见化）**  
开门工程进入实操：真钥必须从保管库转移到门心校准台。车队分真假两路；英格丽德亲自或派直属骑士押真钥；胧的胧忍军残部/合作部队可能被派去「清理目击者」；桐生佐马斗一类黑医若介入，目标往往不是开门本身，而是**借门缘混乱猎取高价值样本**（对魔忍、骑士、甚至英格丽德本人）。对魔忍侧由井河阿莎姬（或同级上忍）接获公安/暗部急电：必须在 T-12h 前确认真钥路线。井河樱可能被派作快速侦察；八津紫负责柱体数据与 B3 中继。此阶段名场面种子：**夜雨中的假钥车爆炸，真钥车在另一条隧道无声通过**；契约者若只追爆炸，会错过真正的钥匙。

④ **结界柱过载与四角站队**  
T-6h 起，三号柱过载报警成为不可逆信号。门缘气压升高，通讯开始死亡。四方同时到位：
- **英格丽德**：要求按魔界时刻表执行开启；失控涌出优先斩杀；凡人疏散「可以 sympathize，但不能改时刻表」。
- **井河阿莎姬**：要求死撑到疏散完成；必要时炸辅锚延迟开启；拒绝把平民当可接受损耗。
- **诺玛德私门派**：试图提前崩柱、开小门分流魔素与 Horde，以便企业独占物流。
- **深渊回流 / 门自体**：无人格，却有「扩大开口」的物理倾向；任何站桩门心的人都会被当成燃料。
此阶段不是单纯「正邪对决」，而是**四角斗**。契约者手中的校准钥匙能让柱「多撑一小时」或「提前崩十分钟」——转左是魔界的门，转右是人界的命，中间是会被两边同时猎杀的变数。

⑤ **开启序列启动（T-2h～T0）**  
真钥插入门心校准台后发烫达到峰值。魔素浓度阶梯上升：+3%、+8%、+15%……第一波 Horde 涌出，多为试压体。英格丽德以魔剑黑焰清扫失控单位，纪律像军规；阿莎姬则切开疏散走廊，紫装在魔素雾里像一道不肯退的刀光。胧若在场，会优先制造人质与催眠混乱，让对魔忍无法专注拆锚。弗斯特系技术官盯着数据板，关心的是「门日志是否完整」——完整日志=可复制的下一扇门。桐生佐马斗可能在混乱中投放诅咒/改造装置，试图削弱英格丽德以报复或要挟布莱克。名场面因果：**开门不是按钮，是一场必须同时打赢「工程、斩杀、疏散、反间谍」的复合战役**。

⑥ **开启瞬间（T0）**  
门环成形。气压塌陷后反冲，门缘五公里内玻璃震裂，防灾广播被迫改口。第二波 Horde 带军官与编制涌出；地区灾难评级正式坐实——此即本世界**六阶舞台开张**的标志。凡俗军队推进会变成送菜；热武器对门心无效，只对先锋潮有限有效。英格丽德的目标从「按表开启」转为「控门权」：她可以暂时与对魔忍交换「杀私门工程换延迟」，因为她视凡人糟蹋门为亵渎。阿莎姬的目标从「阻止开启」转为「最小化平民死亡 + 夺或毁复制钥」。布莱克的眼线开始记账：谁是有趣的变数，谁是该抹除的噪音。

⑦ **门权争夺（T+0～T+2h）**  
门已开，但**谁控制门向**仍未定。合法胜利路径包括：
- **拆辅锚 / 第二锚**：无法立刻关门，但可把开口压回「半开可控」。
- **夺钥 / 毁复制钥**：阻止诺玛德量产第二扇门。
- **伪信标扭门向**：把 Horde 先锋诱偏到无人区或海面——会被英格丽德记入门日志，敌意+1，却可能救下一座城。
- **交易延迟**：向英格丽德提供私门派罪证，换取时刻表后移；向阿莎姬提供柱体数据，换取撤离窗口。
- **公开证据**：把「企业私门 + 政治默许」证据交给能短暂联手的政府暗部，换取地区级支援（非歼灭顶点）。
非法/自杀路径：门心站桩决斗、宣称「一人推翻诺玛德」、无准备硬刚布莱克本体。

⑧ **英格丽德弧光：从门卫执行者到门权棋手**  
英格丽德在本切片中的核心不是「被俘凌辱」（OVA 线另有其作，本库不写），而是**骑士纪律与组织肮脏之间的撕裂**。她忠于布莱克，厌恶胧与弗斯特的下作手段，更厌恶企业把大门当提款机。开启夜她可能：① 斩杀失控 Horde 以证明骑士仍守规则；② 与阿莎姬短暂停火以清私门；③ 拒绝布莱克眼线提出的「用平民当祭品加速开启」；④ 在门日志上亲手划掉一份伪造时刻表。她的软肋是忠诚本身——布莱克的一纸评估，比千军万马更能让她收剑。对契约者：碍门可控=敌；助她控门且不亵渎=临时工具；试图用感情戏收编=被一剑拍开。

⑨ **对魔忍弧光：消防队长与证据战**  
井河阿莎姬在本切片的定位是「人界止损的顶点刀」。她未必能在开启瞬间关上大门，但能决定**这座城死多少人、证据链是否完整、下一代对魔忍是否还有家可回**。井河樱适合快速侦察与影遁撤离；八津紫适合抢 B3、扛柱体数据、断后。她们与英格丽德的关系是「同场敌人 / 临时共敌」：可以并肩砍私门派，下一秒仍会为柱寿命对砍。公安暗部则反复在「用对魔忍」与「弃对魔忍」之间摇摆——门开之后，甩锅名单比救援名单更长。

⑩ **诺玛德与布莱克：股东不在工地**  
埃德温·布莱克是古吸血鬼、诺玛德创始者，战力与组织权柄双线碾压。本切片中他更常以**阴影与评估**存在：玻璃塔晚宴、加密指令、对「阿莎姬是否值得成为更长久课题」的兴趣。他允许开门，不等于他站在门心；他救人（例如其他时间线中对英格丽德的保全）也不等于温情——那是资产维护。六阶任务若把他写成「被主角一刀斩于门前」，即违顶点规则。合法互动：赴宴谈条件、截获评估报告、用证据逼他收缩私门预算、在他注视下完成条件性胜利后撤离。

⑪ **终局方向与开放残局**  
本档案默认三种可写终局（皆非「世界和平」）：
- **半开永久化**：门成为长期舞台装置；地区灾难常态化，任务书可续写。
- **延迟后更大反弹**：本夜救下城，三周后第二序列更猛——强调「延迟是胜利，不是结局」。
- **可控开启成功**：魔界骑士夺回门权，人界以惨重代价换来「至少知道谁握着钥匙」。
无论哪条，诺玛德不灭、布莱克不灭、英格丽德仍站在门与忠诚之间。契约者可改局部（某柱寿命、某批钥归属、某次疏散成败、某份门日志内容），不可无代价抹掉顶点或让大门「因为被封印而战力归零」。

⑫ **可介入的微观任务（贯穿开启夜）**  
护送抗魔素药剂与伤员；抢修 B3 中继；鉴定真假门钥；投放/回收伪信标；盗取柱体日志；伪造一份让私门派内斗的订单；掩护井河樱撤离；向英格丽德呈交私门罪证；销毁会被政治层利用的录像；在疏导线建立临时掩体；从门心冲刺拆第二锚后三十秒内撤出。AI 优先这些可玩事件，而不是一上来顶点决斗。

⑬ **蝴蝶效应边界**  
可改：具体柱校准结果、某次 Horde 潮汐伤亡、假钥车队成败、B3 归属、英格丽德与阿莎姬是否短暂停火、复制钥是否被毁。慎改：布莱克作为诺玛德顶点的长期在场、英格丽德对布莱克的忠诚内核、大门作为战略资产的存在本身、对魔忍「先保人后逞强」的伦理底线。硬改核心会触发：骑士团通缉、诺玛德点名回收、对魔忍敌视外来变数、乐园任务失败判定。记住：这个世界的「胜利」常常是「让门开得慢一点、偏一点、可控一点」；情报、钥匙、柱与中继，往往比蛮力更决定胜负。

⑭ **名场面速查（供 AI 落笔）**  
- 假钥车在雨夜爆炸，真钥车无声过隧道。  
- 三号柱铭牌上，英格丽德的剑尖点着校准口：「时刻表不能改。」  
- 耳机里阿莎姬的声音只剩断续：「疏散还剩……撑住柱。」  
- 真钥插入的一瞬发烫，门后传来不像风的呼吸。  
- B3 中继绿灯亮起的同时，第二波 Horde 军官踏出门环。  
- 英格丽德对阿莎姬说：「今晚我可以杀企业，不杀你的疏散队——前提是你别碰主钥。」  
- 玻璃塔方向，布莱克的评估报告多了一行：「门缘出现有趣的变数。」  
任意时点问三句：真钥在谁手里？三号柱还能撑多久？B3 是谁的？答得清，才能写对「现在的世界状态」。

## 阶位切入点

> 阶位↔原作强度：凡人/见习≈一阶；下忍·低阶魔≈二阶；中忍·成型魔≈三阶；上忍·阿莎姬/胧级≈四阶；城市级魔灾/骑士精锐≈五阶；**深渊大门开启态与诺玛德顶点阴影≈六阶**。宁低勿高。本世界**仅覆盖六阶**。世界顶点（埃德温·布莱克 + 门本身）=存在·情报优先 / 条件性胜利；严禁硬刚本体或「战力归零式削弱」。

**六阶（深渊大门开启夜 · 门缘夺权）**

切入身份/时点：契约者以「抗魔素工程承包商 / 被买通的结界技术员 / 对魔忍外围协力刀 / 魔界侧搬运骑士随员 / 乐园特准深层变数」切入，锚定**门开 T-6h 至 T+2h**。你已具备地区级战场生存力，但**不等于**能单挑门心或布莱克。

初始事件：结界柱三号过载报警。你手里的校准钥匙能让柱「多撑一小时」或「提前崩十分钟」。通道尽头，**英格丽德**的剑尖点在铭牌上，要求你「按魔界时刻表崩」；对讲里**井河阿莎姬**的声音在数疏散还剩多少人，命令「死撑到列车离站」。与此同时，诺玛德实验室头目通过加密频段出价买你「提前崩」；**八津紫**要柱体原始数据；**胧**的眼线在疏导线制造恐慌。门不要你的立场，只要你的键——而键一转，门日志就会记下你的名字。

开场白建议：「地下的风是热的，像有什么巨大的肺在门后面呼吸。英格丽德的剑尖点在结界柱铭牌上，说时刻表不能改。耳机里阿莎姬的声音在数疏散还剩多少人。你的校准钥匙烫手——转左是魔界的门，转右是人界的命，中间是你会不会被两边同时猎杀。」

关键NPC立场：
- **英格丽德**：门可控高于一切；可交易「杀私门换延迟」；伪信标会被她记仇。
- **井河阿莎姬**：人界止损；接受同级协力，拒绝把你当救世主。
- **八津紫**：争 B3 与柱体数据权；可靠但要结果。
- **井河樱**：可能在影中护送伤员；被俘风险高。
- **胧**：要混乱与人质；视你为余兴或刻印材料。
- **弗斯特 / 实验室头目**：买你的键；怕门日志。
- **桐生佐马斗**（若现身）：猎样本，不站队。
- **埃德温·布莱克**（阴影/眼线/晚宴线）：**超阶巅峰：存在·情报优先/条件性胜利**；记账变数，不轻易下场被砍。
- **门钥祭司（仪仗）**：杀擅改主序列者。
- **山本系暗部联络人**：要报告也要甩锅。

主线钩子/支线：
- 主线：开门瞬间站队与扭向——延迟开启 / 半开可控化 / 夺或毁复制钥 / 完成疏散后撤离。
- 支线①：伪信标投放（救城+被记日志）。
- 支线②：夺门钥复制体并选择销毁或上交。
- 支线③：抢修或炸毁废弃线 B3 中继。
- 支线④：与英格丽德交易——呈交私门罪证换时刻表后移。
- 支线⑤：护井河樱撤离并截获胧的催眠布置。
- 支线⑥：门心三十秒拆第二锚冲刺（超时=回流）。
- 支线⑦：赴布莱克阴影下的评估会，用情报换「不扩大 Horde」。

危险度/规避：贴近顶点。门心禁止久留；第二波 Horde 带军官后凡俗编制失效。必须规避：布莱克本体硬刚、门心决斗站桩、无抗魔素药硬扛三小时、把第一波试压当最终胜利、相信黑市假药。阶段失败教育：硬刚顶点、无准备冲仪式、把关键 NPC 当群像工具人——三种写法禁止。

任务方向/奖励：
- 目标：延迟开启 / 半开可控化 / 夺钥毁钥 / 最大化疏散 / 公开足以让暗部短暂联手的罪证。
- 奖励：门钥碎片、抗魔素配方、骑士临时通行证（有时限与仇敌标记）、对魔忍协力章、柱体日志副本、乐园结算用的「组织级战果」而非弑神证明。
- 阶段胜判定看条件性胜利指标（柱寿命、门向、钥归属、疏散人数、日志完整性），不看击杀榜。

六阶行动节拍（执行参考，非灌水）：
1. 入场先确认「谁在发令、谁在收尸、谁在记账」。发令者=英格丽德/阿莎姬/实验室；收尸者=Hor de 与胧；记账者=布莱克眼线与门日志。
2. 第一笔交易必须小——情报换路条或一小时药剂，不直接碰主钥。
3. 建立至少一个具名临时盟友与一个具名仇敌（例如：紫的数据同盟 + 实验室头目的追杀）。
4. 主线节点前完成侦察：入口、逃口、人质、证据四样至少两样到手。
5. 真钥发烫后进入序列态：优先保证 B3 与抗魔素，再谈扭向。
6. 第二波后改打「走廊战与拆锚」，停止无意义清潮。
7. 若与英格丽德交易成功，必须在约定窗口内兑现，否则骑士信用归零。
8. 撤离时带走日志副本；空手撤离=政治层可把你写成纵火犯。

六阶战场读图：外围补给（药剂/疏导）→ 中层冲突（柱廊/B3/假钥）→ 核心（门心校准台/主钥）。契约者默认从外围进，证据与键齐才进核心。台词锚：本地人不会信你能「一个人关掉深渊」；他们最多信你能让某一夜少死一些人或让门偏一点。
