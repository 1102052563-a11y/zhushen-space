# 魔界骑士深渊大门-大门开启
<!--meta lib=主库 tiers=六-->

## 剧情

**【作品来源】**
对魔忍系列（Lilith Soft / 对魔忍宇宙）：以现代日本为表、魔界与超凡忍者为里的黑暗动作世界；含《对魔忍アサギ》等正传、RPG、外传与魔界骑士相关作品。**英格丽德（イングリッド）**为魔界侧著名「魔界骑士」：金发半魔、剑与魔力、执行魔界意志于人界。**本档案「魔界骑士深渊大门-大门开启」**聚焦战略级位面通道**深渊大门从封闭到开启**的危机，舞台为人界-魔界交界的门缘战场。文风：冷酷特务战+超凡暴力；写门、结界、组织与条件性胜利，**不写色情凌辱细节**。

**【世界定位】**
现代都市地下/旧实验室/结界薄弱点。主线=**深渊大门开启事件**。英格丽德是门务关键执行者之一，未必已深渊化。契约者是被乐园丢进「开门倒计时」的变数。

**【世界观 · 力量体系】**
- **对魔忍**：忍者血脉+粒子/忍术武装，专精猎魔。
- **魔族/半魔**：魔力、变身、上位契约。
- **魔界骑士**：魔界建制武勋，英格丽德为其代表级。
- **企业/Nomad等**：科技+魔素实验，常是开门帮凶或竞品。
- **深渊门**：位面通道；开启后可投放地区级灾难。
分层：
1. 普通人、底层打手≈一阶。
2. 训练有素特工、低阶魔物≈二阶。
3. 正式对魔忍、魔界士兵≈三阶。
4. 精英对魔忍、魔界军官≈四阶。
5. 顶级对魔忍、魔界骑士精锐≈五阶。
6. **深渊大门开启态的门缘战场与上位投射**≈**六阶（本世界覆盖）**：地区级灾难、凡俗军队近乎无效。
乐园阶位映射：如上；**门开启完成态≈六阶**。顶点含上位魔/门本身；条件性胜利=拆锚/扭方向/延迟，严禁「门被削弱所以没了」。

**【地理 · 舞台】**
门缘地下空洞；结界柱阵列；Nomad式实验室；对魔忍临时指挥所；都市疏导线；魔界侧桥头堡；门钥保管库。

**【世界剧情线】**
① **裂隙前史**  
人界结界长期压制魔界大规模投射；小裂隙不断，大门级通道被多方觊觎。

② **开启工程**  
祭品、魔素、门钥、坐标、薄弱点对齐。英格丽德可能被派监督「开门必须由魔界可控」。

③ **人界反制**  
对魔忍与政府暗部试图炸锚、夺钥。**井河朝阳**等顶级战力可能登场作消防。

④ **开启瞬间**  
气压、魔素、第一波 Horde、通讯死。六阶战场正式开张。

⑤ **门权争夺**  
魔界要可控门；企业要私门；对魔忍要关门；深渊要门失控。四角斗。

⑥ **终局方向**  
开启成功/半开永久化/延迟后更大反弹。本档默认门成为长期舞台装置。

**【主要人物】**
- **英格丽德**（魔界骑士）｜性格：傲、强者逻辑、任务优先。｜能力：剑+魔力，门务战专家≈五~六阶交界。｜弧光：门卫执行者→门权棋手。｜立场：契约者若碍门可控=敌；若助魔界控门=临时工具。
- **埃德温·布莱克**（魔界支配相关上位，系列关键）｜性格：冷、布局。｜能力：远超门缘炮灰。｜弧光：门的真正股东之一。｜立场：情报优先，勿硬刚。
- **井河朝阳**（对魔忍）｜性格：坚韧、人界守护。｜能力：顶级对魔忍战力。｜弧光：开门事件消防队长。
- **井河樱**｜对魔忍侧重要战力/羁绊点。
- **八津紫**｜情报与战斗支援型对魔忍。
- **Nomad研究员头目（任务具名）**｜开门技术侧。
- **门钥祭司/魔界仪仗官**｜仪式执行。
- **都市疏导警官（凡人）**｜平民标尺。
- **低阶魔族先锋队长**｜第一波。

**【势力图谱】**
魔界骑士团与支配体系；对魔忍组织；邪恶企业；政府暗部；深渊污染自体（非人）。

**【贵重物品】**
深渊门钥；坐标信标；结界柱核心；抗魔素药剂；英格丽德骑士徽；对魔忍装；伪信标（可扭门向）。

**【隐藏剧情 · 伏笔】**
- 开门可能被上位者设计为「清洗不听话的魔界骑士」的局。
- 企业复制门钥会导致多门连锁。
- 英格丽德厌恶凡人糟蹋门——可成临时同盟点。
- 半开比全开更适合长期任务书。

**【大事记时间线】**
结界时代 → 裂隙频发 → 开门工程 → 对魔忍介入 → 门开 → 四角夺权 → 新常态。

**【叙事基调 · 雷区】**
特务冷、超凡暴力、都市崩坏感。忌把对魔忍写成无敌超级英雄；忌低阶关六阶门；忌色情。最早切入：门钥转移车队夜。

**【大门开启·结界柱编号】**
三号柱最旧，最先过载。校准钥匙通用但日志不通用。

**【大门开启·门钥温度】**
真钥在开启前两小时发烫。假钥始终凉。

**【大门开启·疏导半径】**
门缘五公里通讯死。对讲中继在废弃线B3。

**【大门开启·魔界时刻表】**
英格丽德的表比人界表快七分——以门脉冲为准。

**【大门开启·企业私门】**
Nomad想开小门分流。英格丽德视为亵渎，可临时同盟。

**【大门开启·Horde潮汐】**
第一波弱，第二波带军官，第三波可能有上位投影。

**【大门开启·抗魔素药剂】**
六阶战场一小时一支。黑市假药致幻。

**【大门开启·骑士徽通行】**
英格丽德徽可过魔界桥头堡，对人界关卡是死刑状。

**【大门开启·伪信标】**
可扭门向，但会被记入门日志，后续敌意+1。

**【大门开启·门心禁区】**
站桩超三十秒必遭回流。只允许拆锚冲刺。

**【大门开启·门缘小时志1】**
开启序列第1步：魔素浓度+3%，Horde压力上升。**英格丽德**巡柱，斩失控涌出优先于救人。**井河朝阳**线要求死撑疏散。**八津紫**要柱体数据。契约者可：调柱、投伪信标、夺复制钥、交易延迟、炸第二锚。六阶规则：门心不站桩；上位投影出现则情报撤离。阶段胜=门开但方向扭/或延迟到援军。本条对应不同序列步，细节不重复粘贴同一句。

## 阶位切入点

> 阶位↔：凡人~打手≈一~二；正式对魔忍/魔兵≈三；精英≈四；顶级对魔忍/魔界骑士≈五；**深渊门开启态≈六阶**。仅覆盖**六阶**。顶点=门+上位：存在·情报优先/条件性胜利。

**六阶（深渊大门开启夜 · 门缘夺权）**

切入身份/时点：契约者以「抗魔素工程承包商 / 被买通的结界技术员 / 对魔忍外围协力者 / 魔界侧搬运骑士随员」切入，锚定**门开T-6h至T+2h**。

初始事件：结界柱三号过载报警。你手里的校准钥匙能让柱「多撑一小时」或「提前崩十分钟」。**英格丽德**出现在通道尽头，要求你「按魔界时刻表崩」；对讲里**井河朝阳**线要求「死撑到疏散完成」。门不要你的立场，只要你的键。

开场白建议：「地下的风是热的，像有什么巨大的肺在门后面呼吸。英格丽德的剑尖点在结界柱铭牌上，说时刻表不能改。耳机里朝阳的声音在数疏散还剩多少人。你的校准钥匙烫手——转左是魔界的门，转右是人界的命，中间是你会不会被两边同时猎杀。」

关键NPC立场：**英格丽德**——门可控高于一切；**井河朝阳**——人界止损；**八津紫**——要你的柱体数据；**埃德温·布莱克**眼线——记录变数；**Nomad头目**——买你提前崩；**门钥祭司**——杀擅改者。

主线钩子/支线：主线=开门瞬间站队与扭向。支线：①伪信标投放；②夺门钥复制体；③护疏散列车；④与英格丽德交易「杀企业换延迟」；⑤炸第二锚。

危险度/规避：贴近顶点。门心禁止久留。规避上位本体现身轴心。

任务方向/奖励：目标=延迟开启 / 半开可控化 / 夺钥。奖励：门钥碎片、抗魔素配方、骑士临时通行证、对魔忍协力章。

六阶补充钩子A：伪信标诱偏先锋，被**英格丽德**记仇敌意+1。信标写入门日志不可擦。

六阶补充钩子B：三号柱校准——撑一小时或提前崩十分钟；**井河朝阳**要撑，英格丽德要魔界时刻表（快人界七分）。

六阶补充钩子C：真钥开启前两小时发烫，假钥始终凉；Nomad常混假钥。

六阶补充钩子D：与英格丽德交易杀Nomad私门工程换延迟——她视凡人糟蹋门为亵渎。

六阶补充钩子E：守或炸废弃线B3中继；**八津紫**争柱体数据权。

六阶开场变体：工程承包商药箱混假抗魔素药，先自检再站队；假药致幻在门缘等于自杀。

六阶阶段胜：延迟开启 / 半开可控化 / 夺钥。门心站桩超三十秒必遭回流。

六阶规避：上位投影轴心、**埃德温·布莱克**本体线、门心决斗。情报优先于击杀数。

六阶Horde潮汐：第一波弱、第二波带军官、第三波可能上位投影——别把第一波当胜利。

六阶行动节拍1：入场后先确认「谁在发令、谁在收尸、谁在记账」。发令者决定你能不能活过第一小时，收尸者决定失败态，记账者决定奖励是否被吞。

六阶行动节拍2：第一笔交易必须小——情报换路条，而不是直接碰深渊门与上位。露富即杀劫在本阶成立。

六阶行动节拍3：建立至少一个具名盟友与一个具名仇敌；两边都空的契约者会在第二日被系统当作「无主物资」。

六阶行动节拍4：主线节点到来前完成一次侦察：入口、逃口、人质、证据四样至少两样到手。

六阶行动节拍5：阶段结算看条件性胜利指标，不看击杀榜。奖励池锁定：门钥碎片/抗魔素配方/骑士临时通行/协力章。

六阶失败教育：硬刚顶点、无准备冲展示/仪式/门心、把关键NPC当群像工具人——三种写法禁止。

六阶台词锚：本地人不会相信你能「一个人推翻秩序」；他们最多信你能让某一夜少死一些人或多死一些人。

六阶时间压力：本阶剧本默认倒计时明确（砦陷/展示/加冕/开门/纹路合拢），拖延也是策略，但拖延会改奖励表。

六阶关系网：优先点名原作真名互动，禁止用「群像/牙人/红颜」替代人名；交易对象必须能被追踪到势力。

六阶战场读图：先画三层——外围补给、中层冲突、核心仪式/展示/门心。契约者默认从外围进，只有证据齐才进核心。

六阶证据链：文书、钥匙、录音、证人四选二才能触发「公开揭露」类胜利；只有热血演说无效。
