## 剧情

**【作品来源】**
《黑兽～被白浊沾污的高傲圣女～》（日文：黒獣 ～気高き聖女は白濁に染まる～）由 Liquid 制作，2010年4月23日发售的王国蹂躏·强制侍奉AVG；脚本浅生詠/和泉万夜，原画日陰影次；衍生含魔人OVA、小说及续作《黑兽Ⅱ》。主角侧视点人物为黑犬佣兵团首领**沃尔特**（Vult / Vault，亦作佣兵王）。**本档案「黑兽魔王觉醒-沃尔特进化」**从原作征服线中截取并放大**个体质变弧**：佣兵领袖→侍奉国家国主→战争意志与个人武勇完成「魔王化」加冕。焦点=卡里斯马、编制截胡、个人武勇质变与法统加冕；**不写色情细节**；与「大陆统一」（战役全景）、「女神堕落」（圣女线）严格分工。

**【世界定位】**
舞台=塞尔努斯大陆西部·厄俄斯平原。时间锚=**统一高潮前后的「魔王化窗口」**（侍奉国家已立、七盾至少过半崩解、白之城名分战前后数日）。地理核=黑之城阅兵台 / 血旗祭坛 / 干部议事厅 / 法库 / 战俘坑 / 入城大道。契约者面对的不是「再多打一个Boss血条」，而是**体系完成体**：个人武勇仍强，真正恐怖是「法统+军团+恐怖平衡」叠成的地区级指挥灾难。一句话：看一个卖刀的首领如何把自己写成法律。

**【世界观 · 力量体系】**
无修仙境界。力量来自：个人武勇与剑术、魔法与神术、亚人血脉（黑暗精灵魔力、高等精灵神性）、佣兵战术与编制截胡、虫使与工程战、宗教预知与政治名分。死亡即死；「神」是政治-信仰符号。

战力分层（破坏力优先，宁低勿高）：
1. 民兵、溃兵、逃奴、街垒志愿兵≈一阶：聚众可断小路，单兵不堪一击。
2. 精锐士兵、低阶骑士、黑犬骨干、税关武装≈二阶：守门、劫商、清街。
3. 砦守护者、具名女骑士、虫使精锐、半身人工程指挥≈三阶：一砦之主级压迫。
4. 圣女预知全军指挥、**佣兵王沃尔特单体武勇与卡里斯马**、破城决战、魔女王残存魔力≈四阶：决定战线走向。
5. **沃尔特进化完成态**（个人武勇质变 + 侍奉法统 + 黑犬/新魔军指挥链 + 血旗加冕）≈**五阶**：地区级秩序重写；个人仍可斩将，但体系让反抗转地下。

乐园阶位映射：民兵≈一阶；精锐/佣兵骨干≈二阶；砦主/具名骑士≈三阶；圣女预知与佣兵王单体≈四阶；**魔王化完成态沃尔特+体系≈五阶（本世界覆盖）**。顶点=进化完成态沃尔特；条件性胜利=**斩首指挥链 / 夺号令与法卷真本 / 断补给与仪式供能 / 策反干部裂卡里斯马**，严禁「被封印所以战力归零」「被削弱变弱智」。

**【地理 · 舞台】**
- **黑之城**：原黑暗精灵王都，现黑犬根据地与魔王化仪式核；阅兵台、法库、酒窖旧契约箱、干部议事厅、战俘坑同城。
- **血旗祭坛**：加冕用真血旗升旗处；混入奥尔加旧旗丝的「真旗」与假旗政治意义不同。
- **白之城方向入城大道**：名分终盘与展示性押送通道；进化宣告也可在此办。
- **法库与侍奉法卷库**：真本、抄本、朗读班教材三分；书记可动笔即能动法统完整度。
- **商路与粮道（二砦向）**：进化后秩序的补给脊骨；断粮=断镇压节奏。
- **山地/虫域外缘**：莎木哈杂预算与非对称反制并存。
AI写场景先定：契约者距阅兵台几步、能否接触法库钥匙、是否在干部三步圈内。

**【世界剧情线】**
① **佣兵王阶段：卖刀的首领**  
沃尔特以黑犬佣兵团起家。逻辑是契约、份额、恐惧与可兑现的胜利。他对部下不讲骑士道，讲「谁能分到一成」。个人武勇已在同盟与魔军之间被传说化，但身份仍是**可被雇佣的刀具**——这是后来一切加冕要抹掉的旧罪。

② **背刺与建国：从雇主关系跃迁立法者**  
七盾同盟在黑暗精灵女王**奥尔加·迪斯克伦蒂亚**魔力衰竭窗口雇佣黑犬为前卫。黑犬攻下黑暗精灵国后，不归还同盟，而将根据地迁入黑之城，建立「侍奉国家」。沃尔特从「拿钱办事」变成「自己写法」。这一跳是魔王化的政治前提：没有法统，只有屠城，加冕会变成笑话。

③ **编制截胡：把别人的军变成自己的手**  
进化不是突然长角，而是**截胡指挥链**的连续成功：夺砦旗与烽火令、改换防表、吞战俘编入新魔军、用份额骰与投名状锁干部。**希克斯**是鞭子，**基恩**是刀与账，**约翰·曼德维尔爵士**是话术包装，**莎木哈杂**与**莫尔塔德拉兄弟**是非常规战与工程。沃尔特的卡里斯马=「跟他能赢、能分、敢杀不听话的」。

④ **统一推进中的个人武勇质变**  
砦旗连锁倒、象征人物跪，不是单纯数值涨，而是**战场证明**：谁能在关键节点亲自斩将、谁能在示众场不眨眼，谁的法才被当真。佣兵王单体≈四阶；进化窗口中他开始把「个人斩将」与「法条宣读」叠成同一次出场——这是五阶观感的来源。

⑤ **进化仪式/宣告：再称他佣兵是死罪**  
血旗、法卷真本、跪地象征（圣女/公主/骑士长席位的政治意义）、干部效忠、旧佣兵契约的封存或当众撕毁。进化完成=身份法统化。仪式可延期、可玷污、可因空席与假旗降完整度，但**不可靠「封印降智」解释失败**。

⑥ **进化后秩序**  
地区级镇压能力；反抗转地下；干部争宠与账本互咬；「进度官」每日报砦旗与法统完整度。终局：进化不可轻易逆转；可做的是推迟、玷污法统、救出火种、公开「不过是佣兵」的旧罪证以裂卡里斯马。

⑦ **蝴蝶边界**  
可改：仪式延期、法卷缺页、干部火并、象征空席、旧契约公开、地桩倾斜笑场。不可无代价改：沃尔特的征服逻辑、侍奉国家作为制度已存在、黑深残军政气质。与「大陆统一」档案分工：本档不铺七砦战役全景，只写进化弧上的战役切片；与「女神堕落」分工：本档不写圣女个人堕落细部，只写其作为仪式象征的政治功能。

**【主要人物】**

- **艾丽西娅·阿尔库托鲁斯**（剑姬｜一砦）｜性格：刚直、耻感。｜能力：剑技冠绝同盟。｜弧光：最硬阻力；辱名可逼其出城。

**【隐藏剧情 · 伏笔】**
- 进化完整度依赖「足够多的象征跪地」——少一席可推迟或降级为整肃大会。

- 契约者若在进化前夜窃走法卷真本，入城式会变成致命笑话。

## 阶位切入点

> 阶位↔原作：民兵≈一阶；精锐/佣兵骨干≈二阶；砦主/具名骑士≈三阶；圣女预知与佣兵王单体≈四阶；**魔王化完成态沃尔特+体系≈五阶**。本世界仅覆盖**五阶**。顶点=进化完成态沃尔特：情报优先/条件性胜利（斩首指挥链、夺号令与法卷、断补给与仪式供能、裂卡里斯马），严禁战力归零式削弱。

**五阶（魔王化窗口 · 血旗与法卷）**

切入身份/时点：契约者以「血旗绣工 / 法库书记 / 外籍观礼佣兵 / 干部侍从 / 工程队销工」之一切入，锚定**进化仪式T-72h至T+1h**（日程板已写「三日后」或真本抄写夜）。

初始事件：你在法库抄写侍奉法真本时，**基恩**与**希克斯**先后到来：基恩要你「少抄一条关于份额的」；希克斯要你「一字不改」。缺页会在仪式上被当众朗读——你动哪一笔，决定干部是否当众翻脸。门外进度官在等完整度日报。

开场白建议：「黑之城的夜比刀亮。血旗在风里像一块还没晾干的罪。基恩把账本拍在你左手，希克斯把刀鞘拍在你右手。沃尔特的加冕要一句完美的法，而完美的法从来不是抄出来的，是人选出来的。」

关键NPC立场：**沃尔特**——仪式不容耻，视你为刀具或障碍；**希克斯**——谁搞砸谁死，认染料与三步圈；**基恩**——肯用你裂内廷，握死士本与账本；**曼德维尔**——要对外讲稿漂亮，收翻译封口费；**奥尔加**——恨供能；**克洛伊**——守夜钥与阀门；**艾丽西娅/克劳迪娅残部**——买你的日程板；**塞勒斯缇娜/普莉姆**——象征席政治意义。

主线钩子/支线：主线=推迟或玷污进化宣告，或在完成态下制造指挥链裂缝。支线：①偷死士本名册；②换血旗为无旧丝假旗；③放跑象征贡品一席；④公开旧佣兵契约；⑤换阅兵台地桩销（莫尔塔德拉工程弱点）；⑥换曼德维尔讲稿页；⑦粮道晚点连动镇压节奏。

危险度/规避：贴近顶点。仪式现场硬刚=死。优先外围、文书、工程、观众席政治杀伤。规避：无准备闯入三步圈、无后手抢真本出城。

任务方向/奖励：目标=仪式延期 / 法统当众裂 / 救走象征 / 指挥链断裂。奖励：名册副本、旧契约残页、干部把柄、出城死路图、半枚旧徽、日程板抄件。不发「一刀斩魔王」神器。

五阶补充钩子A：持旧徽半枚混内圈须交投名状军旗；**希克斯**查染料。假旗三日褪色，真旗植物染。过关可近侧前文书位，仍禁入三步圈。

五阶补充钩子B：偷死士本名册可策反可逼清洗，进化改期。名册分公开本/干部本/死士本，死士本只有**基恩**与**沃尔特**。暴露则全城搜。

五阶补充钩子C：换阅兵台地桩销使宣言台倾斜，笑场裂卡里斯马后整肃。地桩为**莫尔塔德拉兄弟**工程编号M-1起；窗口在T-89h加固夜。

五阶补充钩子D：酒窖旧佣兵契约投观众席，辱「魔王也曾卖刀」。贵族窃笑、士兵动摇、沃尔特杀人立威三线并行——耻感成立即阶段胜条件之一。

五阶补充钩子E：放跑象征贡品席一人（**塞勒斯缇娜·卢克鲁斯** / **普莉姆·菲尔丽热** / **克劳迪娅·莱凡坦** / **艾丽西娅·阿尔库托鲁斯**择一），空席即法统伤口；祭司点名不可用替身。

五阶开场变体：血旗绣工被令绣**奥尔加·迪斯克伦蒂亚**旧纹丝；绣错、换布、藏真丝三种手法对应三种风险。祭司火试丝，希克斯湿布试染。

五阶阶段胜：延期 / 法统裂 / 象征被救 / 指挥链断 / 耻感成立。禁正面持久战进化完成态沃尔特。

五阶干部缝：基恩账本角与希克斯灌铅骰互卖可内讧；**约翰·曼德维尔爵士**会收「翻译封口费」；预算夜莎木哈杂可被买通放虫扰彩排。

五阶加冕讲稿：曼德维尔三版（对贵族/对兵/对俘）念错版会致命笑场——可偷换讲稿页；侧前站位可摸匣。

五阶行动节拍1：入场后先确认「谁在发令、谁在收尸、谁在记账」。发令者决定你能不能活过第一小时，收尸者决定失败态，记账者决定奖励是否被吞。

五阶行动节拍2：第一笔交易必须小——情报换路条，而不是直接碰进化完成态沃尔特。露富即杀劫在本阶成立。

五阶行动节拍3：建立至少一个具名盟友与一个具名仇敌；两边都空的契约者会在第二日被系统当作「无主物资」。

五阶行动节拍4：主线节点到来前完成一次侦察：入口、逃口、人质、证据四样至少两样到手。

五阶行动节拍5：阶段结算看条件性胜利指标，不看击杀榜。奖励池锁定：名册副本/旧契约/干部把柄/死路图/半枚徽。

五阶失败教育：硬刚顶点、无准备冲展示/仪式/门心、把关键NPC当群像工具人——三种写法禁止。失败态示例：进度官点名虚报牵连、希克斯验出假旗、三步圈插队被当场斩。

五阶连动补给：若同步让二砦向粮道晚点6小时，城内朗读班与展示队列失序，完整度日报必烂——不削弱单体，只断表演。

五阶残部线：莱凡坦旧部剑士买日程板；成交后可能灭口。可用假日程反卖黑犬，诱其清洗错目标。
