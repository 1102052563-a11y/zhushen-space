# 黑兽魔王觉醒-沃尔特进化
<!--meta lib=主库 tiers=五-->

## 剧情

**【作品来源】**
同 Liquid《黑兽》系列。**本档案「黑兽魔王觉醒-沃尔特进化」**聚焦征服者本体质变：**佣兵王沃尔特在完成关键征服节点后，从「卖刀的首领」进化为「立法的魔王/新秩序顶点」**——战争机器、侍奉法统、黑犬干部结构与个人武勇叠成五阶顶点形态。不写色情；写加冕、血旗、干部政治与条件性胜利。

**【世界定位】**
时间锚在**统一高潮前后的魔王化窗口**。地理仍可在黑之城阅兵台或白之城入城式。契约者面对的不是「多打一个Boss血条」，而是**体系完成体**。

**【世界观 · 力量体系】**
同黑兽分层。本档强调质变：
- 进化前：佣兵王单体≈四阶，可斩将破城。
- 进化后：个人仍强，但权柄来自**法统+军团+恐怖平衡**，整体观感≈**五阶地区级灾难指挥**。
乐园阶位映射：……；**沃尔特进化完成态≈五阶顶点（本世界覆盖）**。条件性胜利=断仪式/夺法卷真本/策反干部/公开其「不过是佣兵」的旧罪证以裂卡里斯马。严禁「削弱归零」。

**【地理 · 舞台】**
黑之城阅兵台；血旗祭坛；干部议事厅；战俘坑；侍奉法库；入城大道；刺客可能的屋顶。

**【世界剧情线】**
① **佣兵王阶段**  
沃尔特以黑犬打天下，逻辑是契约、份额、恐惧。

② **建国**  
侍奉国家宪法让他从雇主关系跃迁为立法者。

③ **统一推进**  
砦旗倒、象征跪，卡里斯马膨胀。

④ **进化仪式/宣告**  
血旗、法卷、跪地象征、干部效忠。进化完成=「再称他佣兵」会成为死罪。

⑤ **进化后秩序**  
地区级镇压能力；反抗转地下；干部争宠。

⑥ **终局**  
进化不可轻易逆转；可做的是推迟、玷污法统、救出火种。

**【主要人物】**
- **沃尔特**｜进化主体；顶点。
- **希克斯**｜欢迎进化的鞭子。
- **基恩**｜担忧个人崇拜的刀。
- **约翰·曼德维尔爵士**｜话术包装魔王。
- **莎木哈杂**｜非常规战预算受益者。
- **奥尔加/克洛伊**｜燃料与阀门。
- **塞勒斯缇娜/艾丽西娅/普莉姆/克劳迪娅**｜可能的仪式「贡品象征」（政治意义）。
- **莫尔塔德拉兄弟**｜阅兵台工程。
- **残存刺客首领（具名随任务生成，常用莱凡坦旧部剑士）**｜反进化。

**【势力图谱】**
魔王直属、干部派系、法统文官、战俘与新奴、地下残盟。

**【贵重物品】**
血旗布；法卷真本；黑犬全员名册；进化仪式日程；旧佣兵契约残页（可辱卡里斯马）；干部投名状。

**【隐藏剧情 · 伏笔】**
- 进化可能需要「足够多的象征跪地」——少一个象征可推迟。
- 基恩藏有可证「份额分配不公」的账本。
- 奥尔加残晶是仪式供能之一。
- 契约者若在进化前夜窃走法卷真本，入城式会变成笑话——致命的笑话。

**【大事记时间线】**
佣兵发迹 → 背刺同盟 → 建国 → 统一高潮 → 进化宣告 → 地区新秩序。

**【叙事基调 · 雷区】**
冷、压迫、制度加冕。忌中二魔王台词过度；忌进化=突然无敌外挂无过程。最早切入：进化日程板写下「三日后」之时。

**【沃尔特进化·血旗纤维】**
真血旗混奥尔加旧旗丝。假血旗无丝，仪式会「差点意思」——足够裂卡里斯马。

**【沃尔特进化·名册三本】**
公开本、干部本、死士本。死士本只有基恩与沃尔特。

**【沃尔特进化·投名状】**
观礼要同盟军旗。旗可假，但希克斯认染料。

**【沃尔特进化·加冕讲稿】**
曼德维尔写三版：对贵族、对兵、对俘。念错版会笑场式致命。

**【沃尔特进化·账本角】**
基恩账本记「份额超领」。公开可让希克斯反噬。

**【沃尔特进化·地桩编号】**
阅兵台地桩莫尔塔德拉编号，抽掉两根则台在宣言时倾斜。

**【沃尔特进化·旧契约箱】**
沃尔特佣兵时代契约藏在黑之城酒窖。可辱「魔王也不过卖刀」。

**【沃尔特进化·象征贡品席】**
圣女/公主/骑士长席位可空一席——空席=完整度下降。

**【沃尔特进化·干部站位】**
距沃尔特三步内是希克斯，五步是基恩，曼德维尔在侧前。插队者死。

**【沃尔特进化·血契刀】**
进化用刀来自黑犬建团夜。被盗则改期。

**【沃尔特进化·加冕倒计时1】**
T-89h：工程队加固阅兵台地桩（莫尔塔德拉编号M-1）。祭司试音血旗。**沃尔特**练一句短宣言。干部开始互咬「谁站得更近」。契约者今日窗口：换地桩销、换讲稿页、偷死士本名、放跑象征贡品、投旧佣兵契约到观众席。每成功项使进化「法统完整度」-1；完整度低于阈值则加冕变成整肃大会——仍危险，但体系裂缝出现。AI须写清谁在场、谁缺席。

## 阶位切入点

> 阶位↔：同黑兽；仅**五阶**。顶点=进化完成态沃尔特+体系。

**五阶（魔王化窗口 · 血旗与法卷）**

切入身份/时点：契约者以「血旗绣工 / 法库书记 / 外籍观礼佣兵 / 干部侍从」切入，锚定**进化仪式T-72h至T+1h**。

初始事件：你在法库抄写侍奉法真本时，**基恩**与**希克斯**先后到来：基恩要你「少抄一条关于份额的」；希克斯要你「一字不改」。缺页会在仪式上被当众朗读——你动哪一笔，决定干部是否当众翻脸。

开场白建议：「黑之城的夜比刀亮。血旗在风里像一块还没晾干的罪。基恩把账本拍在你左手，希克斯把刀鞘拍在你右手。沃尔特的加冕要一句完美的法，而完美的法从来不是抄出来的，是人选出来的。」

关键NPC立场：**沃尔特**——仪式不容耻；**希克斯**——谁搞砸谁死；**基恩**——肯用你裂内廷；**曼德维尔**——要对外讲稿漂亮；**奥尔加**——恨供能；**克洛伊**——守阀门；**艾丽西娅残部**——买你的日程板。

主线钩子/支线：主线=推迟或玷污进化宣告。支线：①偷名册；②换血旗为旧盟旗布；③放跑象征贡品；④公开旧佣兵契约；⑤炸阅兵台地桩（莫尔塔德拉工程弱点）。

危险度/规避：贴近顶点。仪式现场硬刚=死。优先外围与文书。

任务方向/奖励：目标=仪式延期 / 法统当众裂 / 救走象征。奖励：名册副本、旧契约、干部把柄、出城死路图。

五阶补充钩子A：持旧徽半枚混内圈须交投名状军旗；**希克斯**查染料。假旗三日褪色，真旗植物染。

五阶补充钩子B：偷死士本名册可策反可逼清洗，进化改期。名册分公开本/干部本/死士本，死士本只有**基恩**与**沃尔特**。

五阶补充钩子C：换阅兵台地桩销使宣言台倾斜，笑场裂卡里斯马后整肃。地桩为**莫尔塔德拉兄弟**工程编号。

五阶补充钩子D：酒窖旧佣兵契约投观众席，辱「魔王也曾卖刀」。贵族窃笑、士兵动摇、沃尔特杀人立威三线并行。

五阶补充钩子E：放跑象征贡品席一人（**塞勒斯缇娜**/**普莉姆**/**克劳迪娅**择一），空席即法统伤口。

五阶开场变体：血旗绣工被令绣**奥尔加·迪斯克伦蒂亚**旧纹；绣错、换布、藏真丝三种手法对应三种风险。

五阶阶段胜：延期 / 法统裂 / 象征被救。禁正面持久战进化完成态沃尔特。

五阶干部缝：基恩账本角与希克斯灌铅骰互卖可内讧；**约翰·曼德维尔爵士**会收「翻译封口费」。

五阶加冕讲稿：曼德维尔三版（对贵族/对兵/对俘）念错版会致命笑场——可偷换讲稿页。

五阶行动节拍1：入场后先确认「谁在发令、谁在收尸、谁在记账」。发令者决定你能不能活过第一小时，收尸者决定失败态，记账者决定奖励是否被吞。

五阶行动节拍2：第一笔交易必须小——情报换路条，而不是直接碰进化完成态沃尔特。露富即杀劫在本阶成立。

五阶行动节拍3：建立至少一个具名盟友与一个具名仇敌；两边都空的契约者会在第二日被系统当作「无主物资」。

五阶行动节拍4：主线节点到来前完成一次侦察：入口、逃口、人质、证据四样至少两样到手。

五阶行动节拍5：阶段结算看条件性胜利指标，不看击杀榜。奖励池锁定：名册副本/旧契约/干部把柄/死路图。

五阶失败教育：硬刚顶点、无准备冲展示/仪式/门心、把关键NPC当群像工具人——三种写法禁止。

五阶台词锚：本地人不会相信你能「一个人推翻秩序」；他们最多信你能让某一夜少死一些人或多死一些人。

五阶时间压力：本阶剧本默认倒计时明确（砦陷/展示/加冕/开门/纹路合拢），拖延也是策略，但拖延会改奖励表。

五阶关系网：优先点名原作真名互动，禁止用「群像/牙人/红颜」替代人名；交易对象必须能被追踪到势力。

五阶战场读图：先画三层——外围补给、中层冲突、核心仪式/展示/门心。契约者默认从外围进，只有证据齐才进核心。

五阶证据链：文书、钥匙、录音、证人四选二才能触发「公开揭露」类胜利；只有热血演说无效。
