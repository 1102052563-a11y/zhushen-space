## 剧情

**【作品来源】**
《成龙历险记》（英文原名 *Jackie Chan Adventures*）是美国—香港合拍动作冒险动画，由 John Rogers、Duane Capizzi、Jeff Kline 等开发，Adelaide Productions、Columbia TriStar / Sony Pictures Television、The JC Group、Blue Train Entertainment 制作，2000 年 9 月 9 日至 2005 年 7 月于 Kids' WB 首播，共 **5 季 95 集**。主角以真实功夫巨星成龙为原型（片中成龙为考古学家兼特工，片尾另有真人成龙短片答疑）。文风为家庭向功夫喜剧 + 中国神话魔幻：口头禅「倒霉倒霉倒霉」「魔法必须打败魔法（Magic must defeat magic）」「一呀！（Aiya!）」贯穿全剧。大陆通称《成龙历险记》，核心冲突为**十二符咒 / 圣主 / 黑手帮 / 八大恶魔 / 鬼影兵团 / 黑气巫师 / 恶魔小龙**。本档案以动画正典为准，服务轮回乐园世界卡与正文常青读取；查不到写不详，严禁编造原作没有的终局与宝物。卡片锚点：**圣主复活 / 恶魔宫殿**——圣主集齐十二符咒恢复真身，建立恶魔宫殿，成龙与老爹在符咒神力与黑影兵团围攻下以魔咒再封印。

**【世界定位】**
当代地球表层（旧金山、香港、全球考古现场）与**气（Chi）魔法 / 恶魔地狱（Netherworld）/ 鬼影位面**并行的「都市神话冒险」世界。主角线：考古学家**成龙**在十三区协助下，与侄女**小玉**、表叔**老爹**、后改邪归正的**特鲁**等组成龙小组（J-Team 核心），轮番拦截黑手帮、圣主、刀龙、塔拉库多、恶魔小龙对符咒、地狱门、鬼影面具与魔气的争夺。一句话：功夫 + 气法 + 生肖符咒的环球封印战，契约者是变数，不是天选免死金牌。

**【世界观 · 力量体系】**
力量本源分四层，可叠加，不可互相替代：
1. **凡俗武技 / 军警装备**：拳脚、兵器、枪械、车辆、十三区科技。决定「能不能在巷战活下来」。
2. **气（Chi）**：东方式生命力 / 法力。正气（老爹一脉：符咒、阵法、圣物、大蒜与啰嗦式 mentorship）与黑气（刀龙一脉：腐化、强化爪牙、召唤）对冲。铁律：**魔法必须打败魔法**——纯拳头打不过真正的魔咒级威胁，但气法往往需要时间、材料与仪式，窗口期由武者硬扛。
3. **神器赋能**：十二符咒、盘古宝盒、八仙信物、鬼影（Oni）面具、岁月史书、魔气容器等。持有即生效或近乎即效，门槛低、上限高，是本世界「普通人一夜变灾星」的主因。
4. **恶魔血脉 / 元素权柄**：八大恶魔巫师各自掌火、月、山、风、雷、水、地、天等大范围异能；圣主另有符咒本源与阴谋人格；其子恶魔小龙可**吸收**兄弟姐妹残留魔气，战力可叠至世界级威胁。

死亡与封印规则：凡人一刀一枪可死；符咒「狗」可近乎不死、「马」可治愈；恶魔本体极少被彻底抹消，常规终局是**再封入恶魔地狱 / 石化 / 面具封印**。圣主曾被圣斗士**洛佩**以卷轴魔咒剥去十二种力量化为石像；「被封印」在本世界是**正典手段**，但**不等于战力归零**——残魂可附身、可改史、可借刀龙复活、可与子嗣联手。低阶切入只闻其名，高阶才见影，终局才见局。

战力阶梯（按破坏力表现，宁低勿高）：
1. **路人 / 黑帮打手 / 新特工**：徒手制服平民，热武器致命。乐园≈**一阶**。
2. **精英武者 / 成龙常态（无符咒）/ 黑手帮三傻单挑级**：碎砖裂门、以一敌多、借环境脱险。≈**一~二阶**。
3. **符咒单件强化 / 阿福级武痴 / 特鲁蛮力**：牛符咒力大无穷、兔符咒超速、鸡符咒漂浮等，可掀翻小队与装甲车级别目标。≈**二~三阶**。
4. **多符咒组合 / 气法中坚 / 鬼影兵团小队**：隐身+爆破、念力+速度等组合拳；老爹常规驱魔；黑影忍者群袭。≈**三~四阶**。
5. **黑气巫师刀龙 / 单尊释放态恶魔（未满功率）/ 鬼影王塔拉库多统御态**：街区级异变、大规模召唤、腐化爪牙。≈**四~五阶**。
6. **圣主真身（集齐或回收符咒）/ 八大恶魔中上层全力 / 恶魔宫殿与龙军雏形**：城市至地区灾难，可改写局部历史（岁月史书节点）。≈**五~六阶**。
7. **叠加魔气的恶魔小龙（终局超充能）/ 多恶魔同时出世的末日窗口**：一国至大陆级威胁，需门户放逐与联手布局。≈**六~七阶**。

乐园阶位映射（对照《阶位战力图鉴》，宁低勿高）：凡人打手~新特工≈一阶；成龙常态武技 / 黑手帮精锐≈一~二阶；单符咒强化 / 阿福 / 特鲁≈二~三阶；多符咒 / 气法中坚 / 鬼影小队≈三~四阶；刀龙 / 单恶魔释放 / 塔拉库多≈四~五阶；圣主真身·恶魔宫殿 / 八魔中上层≈五~六阶；超充能恶魔小龙·多魔同世末日窗口≈六~七阶。世界顶点（集齐权柄的圣主、终局超充能小龙、多魔联动）= 本世界七阶顶；**不设八九阶**。超阶级「概念抹杀」不在本作战力表内；写顶点用**条件性胜利（封印、放逐、拆权柄、改史回滚）**，严禁把圣主写成「被削弱后战力为零的沙包」。

**【地理 · 舞台】**
- **旧金山 · 老爹古董店**：日常据点、气法工坊、小玉闯祸起点；店内堆满「看起来像垃圾的圣物」。
- **十三区（Section 13）**：布莱克警长掌管的秘密机构，地下金库长期存放符咒、面具、恶魔相关证物；既是庇护所也是被渗透的靶心。
- **全球考古 / 博物馆 / 遗迹**：符咒最初散落点（如鸡符咒藏于巴伐利亚盾牌一类文物中）；成龙本职舞台。
- **香港与亚洲都会**：家族线、黑气势力出没、圣主旧日统治亚洲的历史回声。
- **恶魔地狱门 / 门户现场**：盘古宝盒定位的七道（外加圣主相关）地狱门，开启则对应恶魔出世；第二季主舞台。
- **恶魔宫殿 / 圣主旧都**：第一季终局圣主复活后试图建立的统治中枢，黑影兵团与龙军叙事的视觉锚点——**卡片锚点可玩区**。
- **鬼影位面 / 永恒黑暗威胁**：塔拉库多目标「以九枚鬼影面具召满兵团、令世界陷入永夜」。
- **岁月史书所在的秘境 / 图书馆式空间**：第二季中段改史危机的规则杀战场。
- **未来时间线残影**：恶魔小龙自未来而来，提示「若封印失败，未来会怎样」。

地图层级：街区巷战 → 都市金库攻防 → 环球夺宝 → 地狱门战场 → 位面/史书规则层。本地人若听闻「一阶要去硬刚圣主真身」，会当疯子——正确反应。

**【世界剧情线】**
① **前史：八魔统治与洛佩封印**  
远古，**八大恶魔巫师**（火·圣主、月·咒蓝、山·波刚、风·啸风、雷·中苏、水·巴莎、地·地魁、天·西木）割据大地与月轨，被**八仙**以专属信物封印入恶魔地狱。圣主曾诱使同胞送自己回人间却独吞权柄、压迫亚洲；约九百年前，圣斗士**洛佩**以卷轴魔咒将圣主**石化**，并把其十二种神力剥离为**十二枚生肖符咒**，散藏世界各地。因果：后世所有「夺符咒=复活圣主」「开地狱门=兄弟姐妹回归」的主线，都从这次剥离开始。

② **开篇：鸡符咒与黑手帮（第一季前段）**  
考古学家**成龙**在任务中接触藏有**鸡符咒**（漂浮 / 念力）的文物，旧识**布莱克警长**将其拉入十三区。犯罪组织**黑手帮**首领**瓦龙**受石化态圣主蛊惑，以黑影兵团与打手**阿奋、拉苏、周**（后有武痴**阿福**、巨汉**特鲁**）全球搜刮符咒，换取「失落宝藏」承诺。成龙侄女**小玉**从香港闯来，冲动却关键地多次改写战局；表叔**老爹**以气法与唠叨成为真正的魔法支柱。名场面：第一次符咒激活、古董店被砸、十三区金库初启。世界状态：一~二阶巷战为主，圣主仍是「会说话的石像」。

③ **符咒争夺战与圣主复活（第一季后段 · 卡片锚点）**  
十二符咒陆续归位十三区，也陆续被黑手帮夺走。**虎符咒**曾将成龙分裂为善恶两面，恶面导致金库失守——圣主得以集齐符咒**恢复真身**，升起**恶魔宫殿**意图征服，黑影兵团铺天盖地。成龙一方夺回关键符咒（尤其**鼠符咒**的「化静为动 / 再活化」在石化逻辑中极为关键），老爹施展对应魔咒，小玉以**龙符咒**等手段摧毁石像形态的后手。阶段结局：圣主再被压制，符咒重回金库，但残魂未灭。**此卷是中阶核心可玩副本**：入口是「金库失守 / 符咒集齐前夜」，争夺物是十二符咒与圣主本体，压迫者是真身圣主与黑影潮，出逃/分赃时有瓦龙背刺与龙军余波。

④ **地狱门与八魔回归（第二季主线）**  
圣主残魂**附身瓦龙**，借**盘古宝盒**定位并开启兄弟姐妹的地狱门。成龙、老爹、小玉与已改邪归正的**特鲁**（拜老爹为师）组成稳定小队，后扩为**龙小组**（含墨西哥摔角手**牛战士**、前女盗**小蛇**等）。每开一门即一场元素灾难：月之咒蓝改引力、山之波刚暴食巨化、风雷水土天各自屠城级异能。封印依赖**八仙信物**与气法阵眼，窗口期由武者与符咒持有者硬抗。中段高潮：**岁月史书**被圣主改写，历史变成「八魔从未坠落、圣主永握符咒」的黑暗时间线；唯小玉因特殊原因保留记忆，联手把史书改回。季末黑气巫师**刀龙**登场，觊觎符咒之力。世界状态：三~五阶，门户战场与改史规则杀并存。

⑤ **符咒粉碎与高贵生肖（第三季）**  
刀龙渗透十三区；为免符咒彻底落入魔手，成龙一方**摧毁实体符咒**，力量逸散进入十二种**高贵动物宿主**（生肖后裔）。刀龙以黑气强化爪牙全球猎捕；部分力量（如鸡、猪）一度被夺。为染指已无活龙承载的**龙之力**，刀龙**复活圣主**——圣主立刻背叛，吞回诸力再度暴走。老爹重演洛佩式石化咒，符咒以新形态回归金库。主题：销毁不等于终结；力量会找容器；与恶魔做交易必被反噬。

⑥ **鬼影面具与塔拉库多（第四季）**  
刀龙误唤醒**鬼影王塔拉库多**。九枚**鬼影（Oni）面具**各召唤一类黑影兵团，戴上者被人格侵蚀；集齐则世界永夜。成龙队与被面具控制的打手、塔拉本体赛跑。未来火魔**恶魔小龙**（圣主之子）现身试图救父/夺权，一度被关入十三区。世界状态：召唤海 + 人格污染，五~六阶威胁感，战场从金库扩到全城阴影。

⑦ **魔气争夺与终局放逐（第五季）**  
恶魔小龙逃脱，猎取封印七大恶魔后残留的**魔气圣物**，逐一吸收兄弟姐妹权柄，终局形态可压过其父。成龙、小玉、老爹、特鲁与龙小组以门户、计谋、临时同盟周旋；高潮中甚至出现**暂时释放圣主对抗其子**的险棋。终局（系列终章 *The Powers That Be*）：小龙被诱入/逼入恶魔门户，与诸魔一并放逐回地狱；地狱内诸魔逼小龙分享所吞魔气，形成「魔内再平衡」。人间侧销毁关键门户媒介，龙小组回归日常。结局方向：世界暂安，符咒/面具等危险物仍需看管；英雄没有成神，只是又一次把盖子按回去。

⑧ **微观日常与可介入事件（贯穿）**  
除大场面外，高频事件：古董鉴定真伪、博物馆失窃、小玉逃学闯祸、黑手帮三傻抢劫升级、阿福来挑战、符咒误触（猴变形、羊出窍、蛇隐身恶作剧）、十三区安检、气法材料采购（老爹要新鲜材料与安静）、黑影夜袭、面具半戴的人格分裂、未来残影警告。AI 写低阶正文应优先这些，而不是一上来圣主宫殿决战。

⑨ **蝴蝶效应边界**  
契约者可改：某枚符咒暂时归属、救下某场巷战人质、策反特鲁更早、毁掉某扇未开的地狱门钥匙复制件。不可无代价改写：洛佩封印的历史地位、十二符咒作为圣主本源的设定、终局「门户放逐」的逻辑、成龙「不愿滥杀、倒霉但可靠」的底色。抢了核心符咒不交十三区 = 黑手帮、刀龙、圣主残魂同时点名；提前杀死瓦龙可能让圣主更快换宿主。

⑩ **分季细目·第一季符咒篇（低中阶主舞台）**  
发现顺序大致为：鸡（漂浮/念力）→ 牛（神力）→ 蛇（隐身）→ 兔（超速）→ 羊（出窍）→ 龙（爆破）→ 鼠（活化）→ 马（治愈）→ 猴（变形）→ 狗/猪（永生向与热视，常同集出现）→ 虎（阴阳）。每一枚的现世都伴随一场「文物点—黑手帮—成龙小队」三角战：巴伐利亚盾牌、亚洲遗迹、美洲展馆、海底或沙漠废墟皆可能。黑影兵团（Shadowkhan）由圣主借给瓦龙，是「凡人犯罪组织突然拥有超凡步兵」的标志；契约者若在此季只看到西装三傻，说明尚未触发圣主直接调度。虎符咒分裂成龙是质变节点：善面仍守原则，恶面效率极高且会把金库拱手让人——写正文时必须让读者分清「哪一张脸在说话」。圣主复活后不只打架，还宣告旧日亚洲统治的复辟；宫殿、龙军与黑影潮同框，才是卡片锚点该有的压迫感。再封印依赖「拆其符咒权柄 + 老爹魔咒 + 团队配合」，不是成龙一人打到 BOSS 血条归零。

⑪ **分季细目·第二季恶魔篇（中高阶主舞台）**  
圣主残魂寄生瓦龙后，人格会从「黑帮老板」滑向「古老暴君」：口音、词藻、对宝藏承诺的轻蔑都会露馅。盘古宝盒是地图，不是武器；打开地狱门的现场往往在现代城市薄弱点（码头、地铁、博物馆地基、偏远峡谷），一开即元素天灾。八魔性格与战法不同：**咒蓝**改引力与月轨，战场「上下」失效；**波刚**巨口吞物，地形与载具会被吃掉；**啸风**风暴切割；**中苏**雷电封锁空域；**巴莎**水域淹没；**地魁**地震与遁地；**西木**飞行与奸诈；**圣主**火与阴谋。封印必须信物对口，拿错圣物=仪式失败。岁月史书篇是规则杀巅峰：被改写后，龙小组可能变成奴隶、通缉犯或不存在，小玉的「记得原史」是唯一回滚锚——契约者若在此节点介入，优先保护小玉与史书，而不是去硬刚已改史成功的圣主朝堂。季末刀龙登场，把威胁从「恶魔血脉」扩到「人类黑气巫师也可掀桌」。

⑫ **分季细目·第三至五季（高阶到终局）**  
第三季「砸碎符咒」是伦理与战术双重赌博：短期防盗，长期制造全球动物宿主竞赛。刀龙强化手下（含对三傻/阿福的黑气化）后，单点战力可压过无符咒的成龙，迫使龙小组更依赖气法与计谋。圣主被刀龙复活后的第一件事永远是背叛——写刀龙线勿让其「控得住圣主」。第四季面具：每一枚鬼影面具对应一类黑影兵种与人格污染；戴上面具的友军比敌人更可怕，因为你不忍下死手。塔拉库多的胜利条件是「永夜」，不是击杀成龙个人。恶魔小龙自未来带来的信息是「某条线你们已经输了」，他既想救父也想超父。第五季魔气吸收是叠层威胁：每吞一魔，小龙的元素权柄+1，终局超充能可压过圣主。险棋「放圣主打小龙」必须付出信任税——圣主随时会改目标。终章门户放逐后，地狱内诸魔逼小龙吐出/分享魔气，人间销毁媒介，系列收束为「日常暂回，看守永不下班」。

⑬ **名场面因果速查（写正文优先调用）**  
- 鸡符咒初启：普通人发现「东西会飞」，布莱克拉成龙入局。  
- 古董店被砸：老爹被迫从「卖古董」转为「公开用咒」。  
- 特鲁投诚：武力倒戈+情感归属，黑手帮失去最强打手。  
- 虎符咒分裂：内在黑暗外化，金库失守，圣主集齐。  
- 恶魔宫殿：地区级征服宣言，卡片锚点主舞台。  
- 岁月史书：历史被写死，小玉记忆回滚。  
- 砸符咒：实体毁灭，力量入动物。  
- 刀龙复活圣主：交易反噬模板。  
- 九面具永夜：城市级阴影污染。  
- 终章门户：放逐 > 击杀；条件性胜利范本。

**【势力图谱】**
- **十三区**：秘密执法与危险物仓储；代表布莱克、成龙（协力）。战力：凡俗精锐 + 金库神器。与古董店联盟，与黑手帮敌对。
- **老爹古董店 / 正气一脉**：气法据点；代表老爹、特鲁（弟子）。战力：仪式与克制黑气。收徒极严。
- **龙小组（J-Team）**：松散英雄编制；成龙、小玉、老爹、特鲁、牛战士、小蛇等。任务型集结。
- **黑手帮（Dark Hand）**：跨国犯罪；瓦龙、三傻、阿福、早期特鲁。战力：人海 + 黑影租借 + 偶尔神器。利益至上，可被圣主/刀龙租用。
- **八大恶魔与恶魔地狱**：前史统治者；圣主为核心搅局者。战力：元素权柄。彼此亦会内斗。
- **刀龙黑气门徒**：黑气巫师团；强化打手。与正气死敌。
- **鬼影兵团 / 塔拉库多**：面具与阴影军团。目标永夜，与所有希望见太阳的人敌对。
- **博物馆 / 黑市文物圈**：灰色中间商；常被各方当枪使。

敌友逻辑：符咒与面具比钞票更硬；暴露「能稳定使用多枚符咒」= 全员追杀；盟约在圣主面前默认作废。

**【贵重物品】**
- **十二符咒（Talismans）**：洛佩自圣主剥离的生肖石，八边形掌心大小，持有即效，可叠用（如鸡+兔≈超音速飞行）。**鼠**化静为动（石像、玩偶、兵马俑级）；**牛**力大无穷；**虎**阴阳平衡（可分裂善恶面，亦为全套平衡锚）；**兔**超速；**龙**爆破炎能（圣主本源向）；**蛇**隐身；**马**治愈；**羊**灵魂出窍（躯体空门大开）；**猴**变形（人/物转动物）；**鸡**漂浮/念力；**狗**永生向抗死（不使人无痛）；**猪**热视/射线。谁想要：黑手帮、圣主、刀龙、投机客、甚至一时冲动的英雄。抢到的后果：短期无敌、长期被全世界点名；集齐并归还圣主 = 第一季终局级灾难。低阶可用单枚，高阶才碰「集齐」。第三季实体可毁，力入高贵动物；再封后可凝回金库。
- **盘古宝盒（Pan'ku Box）**：定位并开启地狱门的关键道具；第二季钥匙级。持有即被圣主残魂与龙小组双向追杀。
- **八仙信物**：封印对应恶魔的圣物（扇、鼓、莲、花篮等，以正典出场为准）；门户战必需，拿错=仪式空转。
- **岁月史书（Book of Ages）**：改写历史的规则神器；小玉保留记忆回滚是名场面。低阶禁止「随手改一笔让圣主从未出生」——原作证明改史会反噬整个现实。
- **鬼影面具（Oni Masks）×9**：各召唤一类黑影；戴者人格被蚀，摘下亦可能留残伤；集齐服务塔拉永夜。
- **魔气容器 / 七魔残留圣物**：第五季小龙吸收的对象；持有即被火魔追杀；是终局叠层威胁的燃料。
- **洛佩卷轴 / 石化咒材料**：再封圣主的仪式核心；老爹可重演，需要时间与掩护。
- **十三区金库权限卡 / 安检协议**：身份与通行；比枪更重要；金库既是保险箱也是靶心。
- **黑气法器 / 刀龙爪牙强化物**：腐化向；用则沾因果，战力涨、人性跌。
- **圣主石像残片 / 宫殿构件**：卡片锚点战场掉落物；可能残留火气与黑影吸引力，勿当普通建材。

启用原则：当前持有方（金库 / 散落 / 魔手）、启用代价、是否会引来黑影或恶魔点名——写进每一场争夺的「谁在看」。

**【隐藏剧情 · 伏笔】**
- 圣主「独吞」历史解释了为何兄弟姐妹恨他，也解释终局魔内分食小龙的合理性——恶魔之间没有永恒同盟。
- 虎符咒分裂善恶、岁月史书改史、面具蚀心——三条线都在问「你还是不是你」；契约者若使用同类物，应付人格检定而非只加算战力。
- 特鲁投诚证明黑手帮并非铁板；瓦龙被附身证明「雇主」随时变「皮囊」。
- 恶魔小龙来自未来，暗示某条失败时间线里圣主线并未被按死；他带来的不只是战力，还有「你们已经输过一次」的信息战。
- 符咒粉碎→动物宿主→再凝形：力量守恒式循环，销毁实体从不等于消灭概念。
- 八仙与洛佩代表「人类/仙侧曾胜利过」；但胜利形态永远是封印而非抹杀，为续作式危机留门。
- 片尾真人成龙段落是第四面墙玩笑，正文世界内勿让「演员成龙」串台替代「角色成龙」。

S5 → 小龙逃脱并吸收七魔魔气；超充能；临时释放圣主的险棋；终章门户放逐；地狱内分食权柄；人间销毁媒介；龙小组回归日常。

**【叙事基调 · 雷区】**
画风：轻快功夫喜剧底色 + 真有灭世危机的魔幻；打戏讲究环境互动与俏皮，不是无脑屠城；亲情与 mentorship 是情绪锚，不是无逻辑光环。  
忌把成龙写成冷酷杀手或全知气法大师。  
忌低阶让圣主/超充能小龙满街散步并被一拳打爆。  
忌把「石化/放逐」解释成战力归零式削弱——那是**条件性胜利**，残魂与再起是正典。  
忌编造原作没有的第十三符咒或让八仙全体下凡当保姆。  
忌用编号清单堆砌内容，也忌跨世界修仙套话（灵石、结丹、飞升、元婴）。  
忌把小玉写成纯拖后腿花瓶——原作里她多次是破局关键。  
最早切入：鸡符咒刚出现、小玉刚到旧金山；或圣主集齐前夜的金库警戒升级。

## 阶位切入点

> 阶位↔原作强度：一阶≈黑帮打手/新特工/路人武者；二阶≈成龙常态功夫/黑手帮精锐；三阶≈单符咒强化·阿福·特鲁；四阶≈多符咒组合/气法中坚/鬼影小队；五阶≈刀龙·单恶魔释放·塔拉统御；六阶≈圣主真身·恶魔宫殿·八魔中上层；七阶≈超充能恶魔小龙·多魔同世末日窗口。宁低勿高；顶点（圣主真身/终局小龙）条件性胜利：封印·放逐·拆权柄，情报优先，严禁硬刚本体与「战力归零」解释。本世界仅覆盖一～七阶。

**一阶（旧金山巷战 · 鸡符咒余波）**
切入身份/时点：契约者以十三区见习、博物馆安保、古董店临工或被黑手帮临时雇佣的跑腿切入，锚定**鸡符咒刚刚易手、小玉刚到旧金山**前后。
初始事件：夜巡博物馆时，阿奋/拉苏/周试图抢走一件「会自己浮起来」的展品复制品箱；真符咒已不在箱内，但黑影忍者试刀，你必须选择报警、私吞线索还是跑去古董店报信。
开场白建议：「旧金山的雾里混着大蒜和轮胎味。三名穿着不成套西装的家伙正把一根钢钎撬进保险柜，柜门却自己轻轻抬了半寸——像有人用看不见的手托着。你的对讲机只剩杂音，巷口屋顶闪过一排无脸忍者。有人在你耳麦里骂『倒霉』，那声音还很远。」
关键NPC立场：**成龙**——先把你当需要保护的平民，立功后才肯听解释；**小玉**——好奇你是不是「自己人」，可能偷偷塞你一个对讲频道；**老爹**——怀疑你碰过黑气，命令你站门口别动他的符；**阿奋**——觉得你碍事，可吓可骗。
主线钩子/支线：鸡符咒争夺的余波情报；支线＝护送展品回十三区、在古董店打杂换老爹一句「有点用」、从三傻嘴里套瓦龙集合点。
危险度/规避：中——死于枪械与坠落；须规避真身级魔法与完整黑影军团。
任务方向/奖励：活着把情报送到布莱克或成龙；奖励取向＝十三区临时通行证、基础防护符纸、小额赏金、对「符咒会动」的认知（不发完整符咒）。

**二阶（环球夺宝 · 符咒半表）**
切入身份/时点：契约者以独立武术顾问、黑市鉴定人 or 十三区外聘特工切入，锚定**第一季中段、多枚符咒已现但未集齐**。
初始事件：在香港或欧洲某拍场，牛符咒与兔符咒的假货同时出现；阿福为夺「真的那一块」向全场宣战，你被夹在拍卖行、黑手帮与成龙小队之间，必须在六十秒内判断哪块是真、是否出手。
开场白建议：「木槌落下的瞬间，半个拍卖厅的人从座位上弹起来。穿唐装的武痴已经把两名保镖丢进香槟塔，嘴里喊着夸张的招式名。成龙从吊灯上翻下来，对你挤出一个『别插手——算了你已经插手了』的表情。你掌心那块生肖石烫得像刚出锅。」
关键NPC立场：**成龙**——认可你的身手但警告勿戴符咒过夜；**阿福**——点名要与你「堂堂正正」单挑；**瓦龙**——出价买你手里的线索；**布莱克警长**——要你把石头交公，否则以走私论处。
主线钩子/支线：阻止任一关键符咒落入瓦龙完整计划；支线＝识破假符咒流水线、护送小玉摆脱绑匪、与特鲁（若仍属黑手帮）在巷战中照面。
危险度/规避：中高——死于武痴与黑影小队；勿试图集齐多枚炫耀。
任务方向/奖励：上交或安全转移 1 枚符咒；奖励取向＝十三区正式协作名分、气法入门笔记复印件、单次治愈符（仿马符咒效果的弱化版，非真符）。

**三阶（金库失守前夜 · 虎符咒阴影）**
切入身份/时点：契约者以十三区特勤或龙小组临时外援切入，锚定**虎符咒分裂危机前后、圣主即将集齐**。
初始事件：金库值班夜，你目睹「成龙」做出违背原则的交符行为——实为恶面；善面被困。你要在拉响全面戒严、追恶面、或先保护小玉之间做抉择，每一秒都在给圣主凑进度。
开场白建议：「地下三层的警报是红的，通道却出奇地空。监控里的成龙微笑着把一盘生肖石推进对方手里，那笑容你在他脸上从没见过。老爹的电话只丢下一句『虎符咒！阴阳！别信任何一张嘴！』然后断线。你背后的阴影里，有人用圣主的腔调轻轻鼓掌。」
关键NPC立场：**成龙（善面）**——恳求你别杀恶面、要复合；**小玉**——坚称「那不是舅舅」；**特鲁**——若已动摇，可在门禁上放你一马；**瓦龙**——趁乱要收割最后几枚；**圣主（石像态）**——低语许诺力量。
主线钩子/支线：虎符咒复合 / 阻止最后交割；支线＝关闭金库分区、抢救被黑影围住的研究员、夺回鼠符咒防石化逆转被滥用。
危险度/规避：高——恶面成龙与黑影同步；严禁与即将复活的圣主谈条件。
任务方向/奖励：金库不全灭、核心符咒不全送；奖励取向＝虎符咒相关封印知识、十三区高级权限、老爹亲制防护阵一次性卷轴。

**四阶（恶魔宫殿外围 · 卡片锚点）**
切入身份/时点：契约者以气法助手、龙小组战力或被胁迫的黑手帮叛逃者切入，锚定**圣主真身复活、恶魔宫殿升起**的锚点卷。
初始事件：宫殿外围，黑影兵团围成三层；龙符咒的爆破在夜空划开缺口。老爹需要人在阵眼点燃/放置封印媒介，成龙负责引开圣主视线——你被点名冲阵眼，途中特鲁用身体挡下第一波手里剑。
开场白建议：「城市的天际线被一座不属于这个世纪的宫殿切开。石龙脊骨上爬满无面忍者，火光里有人用古老的语言宣读新王朝。老爹把一枚烫手的符纸拍进你掌心：『放在第三根柱子，数三个一呀，跑。魔法必须打败魔法——你的腿负责打败距离。』」
关键NPC立场：**成龙**——把你当可托后背的同级战力；**老爹**——只关心阵眼是否完成，完成后才记得夸你；**小玉**——可能擅自带龙符咒支援；**圣主**——视你为可焚尽的蚁；**瓦龙**——已开始怀疑「宝藏」是否存在，可能卖队友。
主线钩子/支线：完成宫殿外围封印节点 / 夺回 1～2 枚关键符咒；支线＝救出被俘特工、摧毁龙军孵化点、确保鼠符咒不在圣主爪中。
危险度/规避：高——真身威压与军团；**圣主本体＝条件性胜利对象，禁止硬刚，走封印与拆符路线**。
任务方向/奖励：宫殿崩溃或圣主再石化的贡献认定；奖励取向＝战斗中遗落的单符咒短期使用权（任务后须上交）、气法进阶、龙小组提名。

**五阶（地狱门争夺 · 岁月史书余波）**
切入身份/时点：契约者以门户封印专员或改史事件幸存者切入，锚定**第二季地狱门高潮或岁月史书回滚后夜**。
初始事件：一道地狱门在码头裂开，咒蓝或波刚的元素前兆已现（潮汐倒灌或建筑被「吃」）；盘古宝盒碎片在三方之间传递。同时有人发现史书回滚后仍残留「从未发生过的记忆」，你的存在被部分人当成错误。
开场白建议：「海水爬上三层楼高的仓库墙，却没有风。布莱克的人在念你的名字，档案里却没有你的指纹。小玉抓住你的袖子：『如果史书再被改一次，你可能先消失——所以现在就帮忙把门关上！』远方月影里，有什么东西在笑。」
关键NPC立场：**老爹**——要你守住八仙信物直到仪式完成；**小玉**——把你当改史同盟；**刀龙**（若已出场）——开价买你的「错误记忆」；**圣主（附身态）**——诱导你打开下一扇门；**特鲁**——用身体当门栓。
主线钩子/支线：协助封印当集地狱门；支线＝保护史书残页、清除残留黑暗时间线锚点、阻止刀龙黑气化信物。
危险度/规避：高——单恶魔释放≈城市灾难；规避多门同开。
任务方向/奖励：门户关闭功勋；奖励取向＝八仙信物仿制护符、正气修习资格、对恶魔真名与克制的完整口述。

**六阶（面具永夜 · 圣主再起窗口）**
切入身份/时点：契约者以驱魔武者或十三区高层特派切入，锚定**第四季鬼影面具高潮，或第三季圣主吞回诸力暴走**。
初始事件：九面具已集其六，半个城区陷入不散的夜；或圣主刚背叛刀龙、符咒之力回流真身。你奉命与成龙分兵：一路夺面具，一路掩护老爹重读洛佩咒。阿福戴着半张面具向你挑战，招式名比以往更疯。
开场白建议：「街灯还在，光却到不了地面。黑影从排水沟、橱窗、人的瞳孔里往外溢。老爹的嗓子已经喊哑，符粉在风里画不出完整的圆。成龙把你推上消防车云梯：『上面那张脸是塔拉的将军——别让它凑齐。我去拖住火龙。』」
关键NPC立场：**塔拉库多**——要你成为下一枚面具的容器；**刀龙**——恨你坏事但可临时停火；**圣主**——许诺分你一条亚洲；**成龙 / 老爹**——只要你不戴面具就信任你的战力；**恶魔小龙**（初现）——嘲笑你们「未来已经输过」。
主线钩子/支线：截获第 7 枚以后的面具 / 完成再石化仪式侧翼；支线＝净化被蚀心的人质、摧毁黑影发生井、隔离小龙与圣主会面。
危险度/规避：极高——军团 + 真身级火魔；顶点条件性胜利，禁止硬刚本体。
任务方向/奖励：永夜中止或圣主再封的关键助力；奖励取向＝面具残片研究权（封印向）、高级气法、可请求一次龙小组联防。

**七阶（魔气终局 · 门户放逐）**
切入身份/时点：契约者以终局战力或「未来残线回归者」切入，锚定**第五季恶魔小龙超充能、系列终章门户放逐**前后。
初始事件：小龙已吞数道魔气，街区在脚下折叠；有人提出险棋——**暂时释放圣主**以子制子。你必须在「同意险棋」「强攻门户」或「牺牲金库诱敌」中选路，并保证小玉的诱敌计划有人断后。
开场白建议：「天空的颜色像被龙焰洗过。十三区的大门第一次对你完全敞开，不是信任，是已经没有什么好藏。老爹把最后一枚定位符按在你心口：『这次不是打败他，是送他回家——地狱的家。数到一呀之前别松手。』远处，父亲与儿子的火撞在一起。」
关键NPC立场：**恶魔小龙**——视你为可吞的额外零食或值得一战的障碍；**圣主**——可能与你短暂同盟，事后必背刺；**成龙**——把你当并列的终局棋子；**小玉**——执行诱敌，需要你信她；**老爹 / 特鲁**——守门人，完成即炸毁媒介。
主线钩子/支线：完成门户放逐链条；支线＝防止魔气落地污染、保证人间侧无人被拖入地狱、销毁盘古/门户复制件。
危险度/规避：贴近顶点——超充能小龙 / 圣主 = 条件性胜利（放逐、分权、封印），**严禁战力归零式削弱解释**；低一档角色不得「一掌拍死」。
任务方向/奖励：世界暂安的 Survivors 证明；奖励取向＝终局目击级情报（可换乐园侧高评价）、残存正气圣物、与龙小组的永久友谊标记（非无限复活权柄）。
