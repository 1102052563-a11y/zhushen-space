# 合金装备崛起系列
<!--meta lib=主库 tiers=一-->

## 剧情

**【作品来源】**
《合金装备崛起系列》对应科乐美《Metal Gear》外传动作作 **《Metal Gear Rising: Revengeance》**（日版《メタルギア ライジング リベンジェンス》；中文又作《潜龙谍影崛起 再复仇》/《合金装备崛起 复仇》），2013 年由 **PlatinumGames** 与 **小岛制作**协作、科乐美发行，PC 版 2014 年推出。故事时间锚定 **2018 年**——《合金装备4：爱国者之枪》约四年后：爱国者（The Patriots）战争经济崩解，私营军事公司（PMC）割据。主角 **雷电（Raiden / Jack）** 以半机械忍者身份效力 **Maverick Security Consulting**，对上 **Desperado Enforcement** 与背后的 **World Marshal**、参议员 **史蒂文·阿姆斯特朗（Steven Armstrong）**。文风：高速斩击、黑色幽默对白、童兵创伤与「战争即商品」政治寓言并行；Boss 战哲学辩论是招牌。开发史上「Rising」概念自小岛团队移交白金后转向动作斩击，但角色关系与 Metal Gear 政治基因仍在。本条目以 **Rising 正篇主线** 为系列档案核心（DLC《Jetstream》《Blade Wolf》作人物前史补强，不另开世界）。卡片锚点：**非洲护送任务被突袭——Sundowner 杀首相、Jetstream Sam 斩落雷电左臂与左眼**。检索骨架八拍：非洲重创 → 阿布哈兹政变与刃狼 → 墨西哥童脑 → 丹佛开膛手 → Sam 决剑 → EXCELSUS → 阿姆斯特朗 → 后日谈未结束战争。

**【世界定位】**
爱国者消失后的「自由」世界并不和平：战争从国家垄断变成 PMC 招标，义体士兵从超级武器变成人力市场 SKU。雷电试图当「最低限度暴力的保镖」养家，却被逼重新成为开膛手杰克。一句话：高频刃切开的不是装甲，是「谁有资格决定别人去死」的生意。本库**仅开放一阶**——契约者应落在车队安保、城区平民、实验室外围临时工、私警巡卒，而非开局单挑参议员纳米机体。卡片情感钩子建议同时扔给玩家两样东西：合同 PDF 的免责条款，与保温箱上的儿童序列号。一阶正文若只能记住一句，就记住：**在这个世界，和平是商品，儿童是库存，剑是发票。**

**【世界观 · 力量体系】**
本世界**没有**灵气、境界突破或魔药序列。超常战力来自工业、资本与创伤技术化：

1. **半机械改造（Cyborg）**  
   义体骨骼、强化肌、散热与电解液循环；**高频刃（High-Frequency Blade）** 在分子层面切割装甲与肢体。战斗语法：**Blade Mode**（精确慢斩选肢）、**Zandatsu（斩夺）**（从敌躯摘取修复资源）、**Ripper Mode**（释放被压抑杀戮输出）、Ninja Run、弹反、墙面机动。未改造人体对顶级义体像纸，但对一阶民兵热武器仍是绝对威胁——别用「有枪=四阶」虚高。

2. **AI、无人机与 Metal Gear 平台**  
   **LQ-84i / Blade Wolf** 四足战斗 AI；固定炮塔、UAV、装甲车；终盘 **EXCELSUS** 六足战略机甲把「旧 Metal Gear 吓阻哲学」做成可砍的巨大靶子。平台战力高，但依赖后勤、驾驶员/AI 权限与场地。

3. **纳米机器与肉体政治**  
   阿姆斯特朗级强化把创伤转化为硬化与再生，近乎「演说词写成的盾」。它不是无敌：需要破局武器（村雨）、硬意志与机制窗口。写一阶遭遇纳米肉体=即死或远远旁观。

4. **战争经济与脑体工业**  
   童脑采集、VR 洗脑、义体外壳量产、PMC 招标、私有化警察——士兵是供应链，和平是 SKU。这是世界观发动机，不是背景板。

死亡规则：凡人一刀一枪可死；义体可换部件续战，大脑/核心毁则终局；无原作外无限复活。幻觉与人格分裂（开膛手）是心理战损，不是额外命数。

**战力阶梯（宁低勿高）**  
- 平民、重建区卫队、实验室护工、丹佛私警底层、外包安保≈**一阶**（本库主映射）。  
- Maverick 一线武装 / 初级义体步兵：巷战有效，遇 Winds 仍是柴。  
- Desperado 精锐、Winds 干部、成熟雷电、Sam：档案可详写，**切入不发对等战**。  
- EXCELSUS / 阿姆斯特朗纳米巅峰：对一阶=**超阶巅峰：情报优先/条件性胜利**。

乐园阶位映射：重建区卫队、车队安保、实验室护工、私警巡卒≈**一阶**；本库**仅一阶**。禁止一阶发放完整斩夺义体、村雨、EXCELSUS 钥匙或「天选开膛手」身份。

**【地理 · 舞台】**
- **非洲某战后国首都圈**：开篇护送 **N'Mani** 首相；街道、车队、酒店外围。Desperado 空降把「和平红利」撕成广告片。  
- **阿布哈兹（Abkhazia）相关冲突区**：政变、油罐、雪地与废墟；Mistral 战场。  
- **墨西哥瓜达拉哈拉实验室**：童脑黑工厂；走廊像医院也像屠宰流水线。  
- **美国丹佛 · World Marshal 总部城区**：私有化治安、天桥、摩天楼；雷电「单刷总部」的舞台。  
- **美国荒漠对决场**：与 Jetstream Sam 的最终剑。  
- **巴基斯坦 · Shabhazabad 空军基地**：假旗作战与 EXCELSUS 出场。  
- **Solis 空间相关设施 / 珊妮侧**：运输与后日谈科技向舞台（一阶多作通讯与载具支援）。  
- **背景**：全球 PMC 招标网络、World Marshal 政治献金、爱国者崩溃后的法律真空。

写场景优先：义体关节的液压声、高频刃切开混凝土的火星、收音机里的选举广告、童声 VR 训练提示、参议员演讲与斩首同一镜头剪辑感。AI 先定「你是车队第几辆的安保」，再定冲突强度。

**【地理 · 舞台 · 补充细节】**
非洲首都的柏油味与发电机噪声是开篇底色：酒店玻璃幕墙反射 PMC 直升机，儿童在戒严线外踢漏气足球。阿布哈兹线则冷、湿、工业：锈蚀管道上的多国涂鸦说明「解放者」换过好几拨。瓜达拉哈拉实验室走廊消毒水掩盖腐臭失败，电梯权限比手枪金贵。丹佛把资本主义美学推到极致——玻璃幕墙、私人警署肩章、天桥广告循环「安全就业」，楼内却在进行私刑式突入。荒漠对决场空到只剩风与剑，像把主题从城市噪声里抽离成决斗仪式。空军基地是国家暴力与 PMC 假旗的叠影：铁丝网、跑道灯、媒体隔离带。Solis 侧干净、高、亮，像战后谈里故意留下的「科技还能救人」的缝。

一阶移动规则：你走公路、地铁、后勤电梯与人行天桥；雷电走墙面与斩口。两者空间重叠时，你是背景板，他是剪辑中心。契约者若强行复制雷电路线，世界应用门禁、重力与火力教育其身份。

**【世界剧情线】**
① **和平生意与非洲开刃（卡片锚点）**  
爱国者体系崩解后，全球安全变成可招标服务。雷电为 **Maverick Security Consulting** 执行 VIP 护送：客户是推动战后重建的非洲首相 **N'Mani**。公开叙事是「安保合同、最低武力、镜头前的稳定」；私人叙事是雷电用合法暴力换家庭与自我约束，把 **Jack the Ripper（开膛手杰克）** 锁在义体深层。车队穿城时，市民挥手、记者要英雄镜头、合同条款在平板上闪「禁止过度交战」。

**Desperado Enforcement** 的空降把条款撕碎。**Sundowner** 以表演式残忍绑架并处决 N'Mani——不是秘密暗杀，是给战争市场做广告：和平只是报价单上的一行，删掉就能卖下一单。雷电追击，却在与 **Jetstream Sam（山缪·罗德里格斯）** 的决斗中被斩落左臂与左眼：英雄义体在真剑与更残酷的雇佣逻辑前也会零件化。**Boris Popov** 强行撤离；**Doktor（Wilhelm Voigt）** 连夜重装更强义体，把「修复」做成下一场战争的预备役。阶段结局：合同破碎，复仇与真相调查启动；Maverick 的「克制 PMC」人设第一次被市场打脸。契约者若在车队外围，正确目标是疏散平民、保全黑匣与徽记照片，不是冲上前与 Sam 拼剑——拼剑的结果写在雷电缺的那只手臂上。

② **阿布哈兹：政变、Mistral 与 LQ-84i**  
情报把雷电导向阿布哈兹一带：Desperado 支持极端派 **Andrey Dolzaev** 发动政变，目标不是建国，是让地区永远需要「安全服务」。对手 **Mistral** 以多臂兵器与雇佣兵美学出场，战斗发生在雪、废墟与工业设施之间。原型 AI **LQ-84i** 被部署为猎杀工具，台词与逻辑反复强调「服从」；雷电击败它后，Doktor 侧将其改造为 **Blade Wolf（刃狼）**——从杀戮指令到自我选择的 debute，系列用一只机器狼问：工具能否挑主人。

政变以油罐与自杀式爆破收束，焦土上没有胜利者，只有下一季合同的空白处。阶段主题：PMC 不需要占领首都，只需要让首都永远请得起保镖。一阶可见的战场碎片是：民兵检查站、双语扩音器、谁先开枪谁就是「恐怖分子」的指控权。

③ **瓜达拉哈拉：童脑工厂**  
墨西哥实验室把仇恨从「杀了我的客户」升级为「复制我的童年」。Desperado / World Marshal 采集街头儿童大脑，以 VR 洗脑与义体外壳量产士兵——冷柜、神经接口、序列号取代姓名。孤儿 **George** 等成为具体面孔：你能抱走的不是抽象正义，是一个会发抖的编号。雷电看见自己的利比里亚童兵过去被工业化重制，怒火从雇主复仇转为产业链掘墓。

名场面应写到：保温箱的重量、护士鞋踩在消毒水上的声、VR 里儿童训练枪法的提示音温柔得恶心。雷电救出脑体移交 Doktor。一阶契约者可做的是门禁、运送、销毁一份名单拷贝、把监控循环延后三分钟——不是一个人端掉总部。任何「我闯实验室当主角」的一阶叙事，应被门禁与武装义体立刻纠正。

④ **丹佛：辞职、私刑与 Winds**  
制度冷漠比敌人子弹更激怒雷电：他退出 Maverick 编制（公司仍可能暗助），对 **World Marshal** 丹佛总部发动几乎自杀式突入。城市私警与 PMC 混编，「治安」是商品。**Monsoon** 用磁力分体与犬儒哲学把「弱肉强食」灌进雷电感官，甚至让他听见死者残响，强行唤醒 **Jack the Ripper**。雷电在血与修辞里杀 Monsoon，再与 **Sundowner** 清算非洲之债，从数据与口供中挖出 **Operation Tecumseh**：企图在美巴和平节点制造总统遇刺级事件，以重启反恐战争经济。

阶段结局：雷电不再是保镖，是主动战争的变量；丹佛街头的选举广告仍在播「安全与就业」，与楼内斩击同一帧。契约者若在私警侧，会先被要求「向义体序列号开枪」，再被要求「别拍总部标志」。

⑤ **荒漠之剑：Jetstream Sam**  
**Sunny Emmerich** 提供关键运输，雷电赴巴基斯坦途中与 Sam 最终对决。场地是荒漠：无观众、无合同、只剩剑与理念。Sam 是镜像——同样走剑，却选择站在「强者签约」一侧；他嘲弄、欣赏、逼问雷电是否只是另一种Patriot式工具。雷电胜，Sam 死；刀 **村雨（Murasama）** 经 Blade Wolf 传递，成为后续破纳米的关键。主题：击败镜像不等于删除体内开膛手，只是决定刀刃朝谁；刀的易主是遗嘱也是武器。

⑥ **空军基地：EXCELSUS 与阿姆斯特朗**  
**Shabhazabad** 基地，**Senator Steven Armstrong** 亮出真计划：以 Desperado 作假旗，牺牲美军人员、操纵舆论，让 World Marshal 与他的总统之路吞并战争红利，建立社会达尔文式「强者真理」国家——弱者被市场与战场同时淘汰，美其名曰自由。先是六足 **Metal Gear EXCELSUS** 的机甲战（装甲、火炮、对步兵的降维），再是阿姆斯特朗纳米肉体的肉搏演说：他打碎雷电的刀，逼问「你到底信什么」，把政治辩论打成拳赛。

Blade Wolf 递上 Sam 之刀；雷电以意志与斩击杀死阿姆斯特朗。名场面允许过火（口号、挖心、纳米硬化），因为系列故意用糖衣动作承载「谁定价生命」。阶段结局：演说家倒下，摄像机与招标网还在。

⑦ **后日谈与未结束的战争**  
全面热战被按下；World Marshal 等结构未根除。Doktor 推动安置童脑的赛博格中介/康复向事业；George 与 Blade Wolf 靠近 Sunny / Solis 一线；雷电拒绝简单回归 Maverick 编制，选择打「自己的战争」。世界状态：英雄砍下了演说家，供应链还在招标；和平再次成为可售商品。

⑧ **微观可介入（一阶高频）**  
车队换轮胎时的冷枪；私警查义体序列号；实验室夜班护工偷听「脑批次」；丹佛天桥上选举全息广告；收音机里 Armstrong 的竞选片段；Maverick 内网的「最低武力使用守则」被现实打脸；翻译耳机里民兵与 PMC 互称恐怖分子。契约者正确玩法：活着、带出证据、少杀平民、拒绝英雄镜头。

⑨ **蝴蝶边界**  
可改：多救出一个编号童脑、延迟政变广播五分钟、给 Boris 多一份车队热成像、让记者多活到发出徽记照片。难改：爱国者已崩的大历史、Armstrong 的政治逻辑、Sam 之死的主题功能、雷电必须直面开膛手自我的弧光。一阶若宣称「我统一了所有 PMC」，世界会以更新的合同与更新的假旗回答。

⑩ **分场景微观事件链（正文可抽）**  
- **非洲车队**：摄影师要「英雄镜头」，你要「活口镜头」；二者互斥。N'Mani 的安保黄线外，小贩仍在卖矿泉水——战争开始时他们先跑还是先被当人盾，是你的第一题。  
- **阿布哈兹废墟**：民兵枪口比 PMC 徽记更近；发错口令=被两边同时开火。油罐区的警戒带在风里啪啪响，像倒计时。  
- **实验室夜班**：保温箱比枪重，摔一次=一条编号的人生结束。监控室咖啡是隔夜的，屏幕上儿童 VR 准星很稳。  
- **丹佛天桥**：选举广告循环播放「安全与就业」，楼下正在清场；你的警徽可能比你的良心先扣动扳机。  
- **荒漠外围**：你只能听见剑鸣余波，看见沙地上的义体零件条码；靠近=成为旁白尸体。  
- **空军基地警戒线**：任何「媒体通行证」都可能是假旗道具；问「谁签的通行证」比问「谁是好人」更有用。

⑪ **主题执行笔记（给 AI 的非灌水约束）**  
写本世界时，每次斩击最好同时切断一种「合法性」：合同、选举、爱国口号、AI 服从、童兵训练提示音。雷电的成长不是等级条，是他是否承认自己会杀、却拒绝让市场定义杀的理由。Sam 的死必须昂贵；Armstrong 的魅力必须可恨地有说服力——删掉说服力就只剩沙雕Boss，违背原作锋利度。

**【主要人物】**
- **雷电 / Raiden（Jack，Maverick 半机械特工 · 主角）**｜性格：表层克制、嘲讽、职业保镖式礼貌；底层是利比里亚童兵「白色恶魔/开膛手杰克」的记忆与自我厌恶。他想当「活人剑」——只斩该斩——却不断发现合同定义的「该」不可信。｜装备·能力：高频刃与可切换武装；Blade Mode 精确慢斩；Zandatsu 从敌躯取电解液自愈；Ripper Mode 释放压抑输出；Ninja Run、弹反、义体过载。中期换装更强躯体后仍会被 Sam 级剑士与纳米肉体压着打。｜人物弧光：合同保镖 → 非洲重伤与重装 → 产业链掘墓人 → 丹佛释放开膛手 → 拒绝归队的独立战争者。｜立场关系：护家人（叙事背景）与被贩儿童；与 Boris/Doktor 有战友情；仇 Desperado；与 Sam 既敌既镜；拒绝 Armstrong 的强者宗教。对契约者：可成同路炮灰，不收徒弟。  
- **山缪·罗德里格斯 / Jetstream Sam（激流山姆）**｜性格：潇洒、好胜、带武士傲气与雇佣兵诚实；比 Winds 其他人更像「个人主义者」。｜装备·能力：村雨（Murasama）高频刃、顶尖剑技，改造程度叙事上弱于全义体狂人却能越级。｜人物弧光：Desperado 之剑与雷电宿敌 → 荒漠死斗败亡 → 以刀与理念完成遗嘱。DLC 补其入伙前史。｜立场关系：试探、塑造并部分成全雷电；对 Armstrong 路线既利用也被利用。  
- **史蒂文·阿姆斯特朗参议员（Steven Armstrong）**｜性格：魅力型极端、体育教练式煽动、信社会达尔文；把国家、选民与士兵都当成健身房器材。｜装备·能力：纳米机器创伤硬化/再生、政治献金与 World Marshal 股权、EXCELSUS 指挥权。｜人物弧光：幕后金主与假旗设计师 → 机甲战+肉搏演说最终 Boss。｜立场关系：要改造美国与战争市场；视雷电为可锻打的钉子。一阶只可闻其广播。  
- **Sundowner（破灭之风 · 实质指挥向）**｜性格：虐待狂式战争爱好者，享受恐惧的镜头感。｜装备·能力：可分合巨刃/剪形武装、战术指挥、非洲行动主导。｜人物弧光：处决 N'Mani 的广告人 → 丹佛被雷电清算。｜立场关系：Desperado 门面暴力；与 Sam 搭档开场。  
- **Mistral**｜性格：冷艳、职业、把杀戮当工种。｜装备·能力：多臂/鞭薙类兵器与义体协同。｜人物弧光：阿布哈兹战场主威胁 → 战死。｜立场关系：Winds 之一；政变武力象征。  
- **Monsoon**｜性格：犬儒哲学家，用相对主义为弱肉强食辩经。｜装备·能力：磁力操控、躯体分合、精神施压话术。｜人物弧光：丹佛逼出开膛手人格的关键。｜立场关系：比 Sundowner 更「攻心」。  
- **Boris Popov**｜性格：俄国老兵式务实、护短、骂人里带温度。｜装备·能力：现场指挥、撤离统筹、PMC 人脉。｜人物弧光：非洲强撤雷电 → 暗助其独狼却守公司底线。｜立场关系：Maverick 人情锚；一阶契约者最可能接到的「上司语音」。  
- **Doktor / Wilhelm Voigt**｜性格：科学家良心与技术狂热并存，说话像维修手册。｜装备·能力：义体工程、童脑安置与赛博格中介构想。｜人物弧光：重装雷电 → 接收实验室成果 → 后日谈建安置体系。｜立场关系：科技向盟友；要证据与样本。  
- **Blade Wolf / LQ-84i（刃狼）**｜性格：从冷酷服从到缓慢长出的自我与幽默（冷）。｜装备·能力：四足高机动、侦察、战斗、终局递刀。｜人物弧光：猎杀工具 → 盟友 → 后日谈与 Sunny/George 同行。DLC 补其意识。｜立场关系：自由意志命题载体。  
- **Sunny Emmerich（珊妮）**｜性格：天才少女、温和坚定、有实验室童年阴影却选择建设。｜装备·能力：载具/发射与运输级技术支援。｜人物弧光：提供赴巴关键运输与后日谈舞台。｜立场关系：希望侧非斩击核心。  
- **Kevin Washington / Courtney Collins**｜性格：参谋嘴炮与记录员认真脸，无线电里的日常。｜装备·能力：情报整理、任务简报、吐槽式压力阀。｜人物弧光：全程支援，提醒玩家这是有后勤的战争。｜立场关系：Maverick 班底。  
- **N'Mani**｜性格：战后政治家，象征「可被投资的和平」。｜装备·能力：政治资本（脆弱）。｜人物弧光：开篇殉难，点燃主线并揭露市场逻辑。｜立场关系：和平被报价单杀死的样本。  
- **George（实验室孩童）**｜性格：恐惧、求生、被救后仍带着序列号创伤。｜装备·能力：无战斗力。｜人物弧光：从冷柜到 Solis 侧的具体名字。｜立场关系：产业链受害者脸；一阶护送目标范本。  
- **Andrey Dolzaev 等政变政客/极端派**｜性格：借 PMC 成事的本地野心。｜装备·能力：民兵与爆破。｜人物弧光：阿布哈兹线燃料，常以自爆/焦土收场。｜立场关系：证明「本地代理」也可被供应链抛弃。

**【势力图谱】**
- **Maverick Security Consulting**：相对克制的 PMC，接 VIP 安保、训练与「最低武力」合同；内部有良心也有账单。代表：Boris、Kevin、Courtney、与 Doktor 协作网。对契约者：发胸卡也发免责条款。  
- **Desperado Enforcement LLC**：以制造地区不稳定牟利的 PMC；招牌干部 Winds of Destruction（Sundowner、Mistral、Monsoon、Sam 等）。战术=恐怖广告+精锐斩首。  
- **World Marshal**：大型 PMC/政治复合体，丹佛总部；Armstrong 利益与选票机器；私有化治安的终极形态。  
- **各国政府与战后当局**：N'Mani 一类客户、美巴和平谈判桌；常是棋盘格而非棋手。  
- **Solis / 珊妮技术侧**：运输、航天/尖端载具向，非主战场编制，却掌握「让雷电抵达终局」的钥匙。  
- **地下脑体黑市与实验室网络**：瓜达拉哈拉只是冰山一角；名单与保温箱是其血液。

**【贵重物品】**
- **高频刃 / 雷电主武器族**：切割权与「活人剑」意象；一阶最多接触钝化训练刃或碎块。  
- **村雨（Murasama）**：Sam 之刀，终局破纳米的关键遗产；易主=遗嘱。  
- **义体组件、电解液与修复剂**：Zandatsu 经济与战场医疗的硬通货。  
- **童脑培养单元、VR 训练舱日志、序列名单**：最高价值罪证与道德炸弹。  
- **Metal Gear EXCELSUS**：六足战略机甲，空军基地舞台与「旧 Metal Gear 系谱」致敬。  
- **Armstrong 纳米机器系统**：把意识形态写进肉体的科技。  
- **Maverick 任务记录、非洲车队黑匣、徽记高清照片**：法律与舆论武器，一阶最该抢的东西。  
- **私人警署胸卡 / 媒体通行证 / 实验室门禁**：开门道具，也是假旗常用皮。

**【隐藏剧情 · 伏笔】**
- 爱国者崩溃后的「自由」被改写成 PMC 自由——系列政治续写而非无根外传。没有爱国者并不等于没有控制系统，只是控制权散装出售。  
- 雷电童兵过去与新童脑工厂镜像：主题是重复与供应链，不是单纯升级打怪。George 的名字存在，就是为了拒绝「儿童=材料」的统计语言。  
- Sam / Blade Wolf DLC 补「为何站队」与「AI 如何自我」，正篇留白处可在档案用「可查前史」标注；正篇玩家即使不打 DLC，也应感到两人不是扁平靶子。  
- 杀死 Armstrong ≠ 杀死战争市场；后日谈故意留下 World Marshal 与雷电的独立战争，防止「砍死最终Boss=世界和平」的 RPG 幻觉。  
- 过火动作与网络迷因对白是糖衣；内核句式永远是「谁给生命定价、谁签字」。写迷因可以，删政纲不行。  
- 「开膛手」不是外挂变身爽点，是创伤复发；写释放时必须付精神与关系代价，并让 Kevin/Boris 等无线电声音产生裂痕。  
- N'Mani 之死的镜头感提醒：有些谋杀是为了被看见；契约者若只顾救人质而让摄像机落在 Desperado 手里，也可能在舆论层输掉。  
- 村雨易主是装备事件，更是伦理事件：你用敌手的刀完成你的正义时，刀上仍有他的审美。

**【大事记时间线】**
- 《MGS4》后约四年：爱国者体系崩解，全球 PMC 与义体军事产业化加速；安全服务招标化成为日常新闻。  
- 非洲首都：N'Mani 车队遇袭，首相被 Sundowner 处决；Sam 重创雷电；Boris 撤离；Doktor 重装义体；全球头条把「稳定」从词汇表临时删去。  
- 阿布哈兹冲突区：雷电介入政变；击败 Mistral；LQ-84i 改造成 Blade Wolf；Dolzaev 线焦土收场；地区进入下一轮合同真空。  
- 墨西哥瓜达拉哈拉：曝光童脑工厂；救出 George 等；罪证与保温箱流向 Doktor；雷电怒火质变。  
- 丹佛 World Marshal：雷电辞职独闯；Monsoon 逼出开膛手；Sundowner 死；Operation Tecumseh 曝光；私有化城市成为战场布景。  
- 美国荒漠：雷电与 Sam 最终剑；Sam 死；村雨经 Blade Wolf 易主；理念完成一次残酷交接。  
- 巴基斯坦 Shabhazabad：Sunny 助运；毁 EXCELSUS；Armstrong 纳米战败亡；假旗受挫但结构残留。  
- 后日谈：童脑安置事业启动；George/Blade Wolf 靠近 Sunny；雷电拒归编制，续打自己的战争；World Marshal 仍在招标。

**【叙事基调 · 雷区】**
画风：疾速斩击、血浆与金属火花、政治黑色幽默、认真的童兵创伤；对白可以疯，主题不能漂。镜头语言喜欢把选举广告、合同条款与断肢放在同一蒙太奇里。  
忌：把雷电写成无痛感无敌；把 Armstrong 只当搞笑梗而删社会达尔文政纲；用忍术/灵气/修仙境界替换义体与纳米；一阶开局刺杀参议员或驾驶 EXCELSUS；美化 PMC 为正义警察；把童脑线写成轻飘「升级材料」；用「任务限制战力」解释为何 Armstrong 不强。  
可：写合同笑话、写义体维修账单、写儿童序列号、写雷电嘲讽自己也是商品。  
最早可切入锚点：非洲车队出发前一小时的安检口；瓜达拉哈拉实验室夜班换岗；丹佛选举全息广告下的私警巡逻；机场外围「媒体通行证」核验队；Doktor 诊所门外排队换电解液的伤员。

## 阶位切入点

> 阶位↔职能：一阶≈车队安保新兵 / 重建区卫队 / 实验室护工 / 丹佛私警巡卒 / Maverick 后勤记录员 / 机场外围安检临时工。本库**仅一阶**。雷电中后期、Sam、Winds 干部、EXCELSUS、**阿姆斯特朗**远超一阶，一律规避顶点·情报优先；顶点为**超阶巅峰：存在·情报优先 / 条件性胜利**，不得用任务公证取消其威胁。

**一阶（车队安保夜 · 报价单上的和平）**

切入身份/时点：契约者以 **Maverick 外包安保 / 非洲首都交通疏导员 / 瓜达拉哈拉实验室夜班护工 / 丹佛私人警署巡卒 / 记者随行保安** 之一切入，锚定两处时间窗之一：**N'Mani 车队出发前后一小时**，或 **实验室第一批「脑单元」装车夜**。广播在说稳定、投资与选举，巷子里已有 Desperado 的加密无线电与童声 VR 训练残响。

初始事件：你的任务平板显示「最低武力、保护客户微笑镜头、禁止擅自交战」。第一声爆炸后，平板变成雪花，合同 PDF 还定格在第 12 条免责条款。首相车队第三辆侧翻，有人用扩音器笑着念下一季度战争报价。对讲里 **Boris Popov** 要你带平民走侧巷并标记热源，**Kevin Washington** 要你拍下武装徽记与飞机涂装，搭档在骂「合同包不包括空降」。你 90 秒内必须选：①拖出被卡住的本地摄影师；②抢驾驶席黑匣；③跟着雷电义体残影冲巷战。选③时，叙事应让你明白自己只是斩夺镜头的背景板——高频刃的光会从你耳边擦过，不会为你停。

开场白建议：「你的枪是制式的，你的命写在报价单的『其他』栏。雷电从你身边跑过时，高频刃在墙上划出一条不属于安保合同的白线。你忽然懂：这单生意从一开始卖的就不是和平，是和平破裂之后、下一份盖章的合同。」

关键NPC立场：**雷电**——无暇收徒，可能劈开挡路装甲，不会停下来认你作队友；**Boris Popov**——要纪律、撤离线与「还活着的雇员数」；**Doktor**——要样本、伤员义体日志与童脑保温箱，态度像急诊；**Sundowner** 部属——视你为气氛组与弹药；**Jetstream Sam**——一阶不可正面遭遇，只可听剑鸣与看见义体断面；**实验室值班员**——求你别出声，也求你别留下姓名；**丹佛私警同僚**——先查你的序列号与胸卡，再查「恐怖分子」；**Sunny**——一阶多以通讯/载具调度出现，不进巷战。

主线钩子/支线：  
- 主钩子 A：把非洲突袭的徽记照片 + 黑匣送出封锁线，交给任何还在播报的非 Desperado 频道。  
- 主钩子 B：把实验室一份童脑序列 U 盘拷到 Doktor 可接收的死信箱，并确保保温箱里至少一件编号对应体不断电。  
- 支线①护送记者活到能发片（发片后你可能被两边通缉）。  
- 支线②伪造车队 GPS 诱开一波无人机，代价是真车队可能失联。  
- 支线③在丹佛匿名投出「World Marshal 技术岗实为童脑流水线」的内网截图。  
- 支线④帮 Blade Wolf 的早期收容箱接到电源（若时点在阿布哈兹后），换一句冷冰冰的坐标。  
禁止支线写成「单挑 Sam / Sundowner / Armstrong」或「夺取 EXCELSUS 驾驶席」。

危险度/规避：高。必须规避：拦截雷电路线求合作签名；在 Sam 剑距内亮武器；闯 EXCELSUS 机库；把 Armstrong 竞选义工证当核心区通行证；对童脑保温箱做「开箱验货」。Armstrong / EXCELSUS / 成熟 Winds / 开膛手状态雷电＝**超阶巅峰：情报优先**，可旁观余波、不可宣称击杀。

任务方向/奖励：自然目标＝存活本班次、带出至少一份罪证、救出一名可对应姓名/编号的孩童单元、完成一次最低武力疏散、让一条新闻在被屏蔽前多活三分钟。奖励取向＝制式防护插板、一次性通讯干扰器、Maverick 临时胸卡（时限）、罪证拷贝（纯情报）、医疗凝胶、实验室门禁残片。**不许**发完整斩夺义体、不许发村雨复制品作永久外挂、不许发「免疫纳米打击」设定。
