# 七剑下天山
<!--meta lib=主库 tiers=一、二、三、四-->

## 剧情

**【作品来源】**
梁羽生（陈文统）新派武侠奠基作之一，1956年2月15日至1957年3月31日连载于香港《大公报》「小说林」专栏，全书共楔子+30回，后有晨风出版社等单行本与多次修订版。属梁羽生「天山系列」中段：上承《白发魔女传》《塞外奇侠传》，下接《江湖三女侠》《冰川天女传》等，与武当、回疆、天地会、南明遗民线交织。文风重词章、家国与儿女悲情，武打写意，常以诗词开篇收束。影视改编极多（1959港片、2005徐克电影《七剑》、2006《七剑下天山》剧等）——**本世界书以小说正文为准**；徐克系「七把神剑」魔改为改编，不得覆盖原著「七剑指人」与抗清终局。

**【世界定位】**
清康熙初年（三藩、回疆、京城、五台、西藏并立）的历史江湖：天山派（晦明禅师一脉）与武当北支、白发魔女门下、天地会、南明忠烈后裔，在满清高压下以剑行侠、以情结缘。主线围绕**凌未风**（原名梁穆郎，天山神芒）、**易兰珠**（杨云骢与纳兰明慧之女）、**楚昭南**（叛门大内高手）及一众「天山七剑」展开。一句话：家国恨、师门裂、儿女情，七剑下天山，恩仇了了隐天山。

**【世界观 · 力量体系】**
本世界为梁羽生式「历史+侠义」人间武学：无仙道飞升、无系统面板、无现代热武器体系（鸟枪偶见，不主导胜负）；胜负取决于内力厚薄、剑法精粗、暗器与轻功、心性与布局，以及朝廷兵权与江湖义气的交织。死亡基本不可逆；「假死」「替身」「血书」「装扮入宫」皆为人间计谋。武功可极强，但仍属人体极限内的超凡——一剑可败多名高手，不能夷平城池、不能爆星。

力量来源与成长逻辑：
- **天山剑法与晦明一脉**：**晦明禅师**（本名**岳鸣珂**，霍天都弟子）创天山北支，集各派之长成天山剑法（传有三百六十一式之说）、拂尘拂穴、暗器**天山神芒**，并重铸**游龙剑**、**断玉剑**。关门弟子**凌未风**得其真传，又自创**大须弥剑式/掌法**（芥子纳须弥，重拙大、守中带攻，狭小空间亦可成剑圈）。
- **武当北支与腾蛟剑**：**卓一航**一系与**石天成**等；**桂仲明**得**腾蛟剑**（剑阁瀑布后秘藏，号称极锋），开武当北支，配合达摩剑法、五禽剑法等。
- **白发魔女门下**：**练霓裳**弟子**飞红巾**（哈玛雅一系称呼亦见资料）、**武琼瑶**等，剑术轻灵狠厉，多涉回疆与草原。
- **少林/达摩心法与医侠**：**韩志邦**失意中偶得**达摩心法**；**傅青主**（傅山）医武双绝，可治心疾、识破宫廷秘事。
- **左右互搏、正反剑与痉挛旧疾**：**凌未风**擅左右互搏与正反天山剑；旧疾痉挛是关键破绽——可在恶斗中突发，被**楚昭南**所乘。
- **朝廷武力与江湖的交叉**：大内高手、绿营、喇嘛寺迷宫、布达拉宫禁锢，是「武功再高也难敌组织与地利」的写实层。
- **死亡规则**：无系统复活。韩志邦替死、纳兰明慧自尽、三格格牺牲、辛龙子力战死、楚昭南自杀——悲剧不可逆。

力量阶梯（按正文破坏力与江湖地位，宁低勿高）：
1. **江湖新人 / 庄丁侍卫 / 绿营兵丁 / 三流剑客**：冷兵器格斗，踹门破窗级。≈一阶初中期。
2. **二流好手 / 会党骨干 / 小派护法 / 成名镖师**：能单挑小队、镇一场客栈或山道。≈一阶后期～二阶初期。
3. **一流成名 / 天地会分舵级 / 大内中坚**：**韩志邦**（达摩心法前）、一般大内侍卫头目、**张华昭**成长中、**冒浣莲**/**武琼瑶**中段、回疆成名侠。≈二阶。
4. **顶尖一流 / 准顶点**：**傅青主**、全盛**韩志邦**、**桂仲明**、**飞红巾**、成熟**易兰珠**、**张华昭**、**李思永**等「七剑」中坚。一剑可败多名高手、可闯天牢外围。≈三阶。
5. **当世顶点近峰**：**凌未风**（游龙剑+天山神芒+大须弥）、**楚昭南**（大内顶尖、游龙剑前持有者）、全盛**晦明禅师**（前史/坐化前海内剑宗）。一怒可改江湖与局部朝局，仍属人间武学极限，无夷平城池。≈三阶后期～四阶。

**乐园阶位映射**（对照《阶位战力图鉴》，宁低勿高）：江湖新人/兵丁/三流≈**一阶**；会党骨干/二流～一流中坚≈**二阶**；七剑中坚/傅青主/飞红巾量级≈**三阶**；凌未风·楚昭南·晦明禅师全盛≈**三阶后期～四阶**（人间绝顶剑客，未达掀平建筑群的质变）。**世界峰值≈四阶**，无五阶以上、无超阶。

**【地理 · 舞台】**
舞台横跨中原、塞外与雪域，以「抗清网络」串联：
- **天山（北高峰/天山派）**：晦明禅师道场；凌未风、易兰珠学艺之地；终局归隐与「光大天山剑派」的精神原点。
- **杭州 / 钱塘江畔（楔子）**：纳兰明慧与多铎大婚前夜；杨云骢托孤；梁穆郎投江未果、携婴上山的起点。
- **五台山清凉寺**：天地会刺多铎大戏；轿中实为纳兰明慧；顺治出家为僧的隐秘；康熙杀父保位的惊天黑幕。
- **京城 / 天牢 / 鄂王府**：张华昭被擒、易兰珠入京营救；纳兰容若词章线；易兰珠刺多铎；明慧自尽；凌未风闹天牢；飞红巾助救。
- **云南 / 幽谷 / 山道**：李自成后人**李思永**会合；**桂仲明**精神错乱与治愈；群豪结义「五剑」雏形。
- **回疆 / 草原 / 瀚海**：飞红巾盟主舞台；辛龙子蛊惑与悔悟；武琼瑶与李思永情线；边塞恶战。
- **西藏 / 布达拉宫迷宫**：凌未风被囚、逼交拳经剑谱；血书认身份；韩志邦替死；群雄杀入；楚昭南自杀。
- **武当剑阁瀑布后**：腾蛟剑秘藏；桂仲明获剑的关键地理符号。
- **喇嘛寺 / 舍利子线**：清廷抢夺西藏圣物；凌未风仗义败楚昭南却中暗器的中段节点。

AI 写场景时应先定「本幕在五台刺杀 / 京城营救 / 云南结义 / 回疆会盟 / 布达拉宫破局 / 天山归隐」哪条链，再定冲突强度与可用 NPC。

**【世界剧情线】**

### 前史 · 天山五剑与钱塘托孤（楔子前至楔子）
《白发魔女传》《塞外奇侠传》余波：晦明禅师、卓一航、白发魔女练霓裳于天山授徒，江湖遂有「**天山五剑**」：**杨云骢**（晦明大弟子，助哈萨克抗清）、**楚昭南**（晦明二弟子，后堕落）、**飞红巾**（白发魔女大弟子，回疆各族盟主）、**辛龙子**（卓一航大弟子，哈萨克，正邪之间）、**凌未风**（时尚未成名，后为晦明三弟子/关门）。杨云骢与满族贵女**纳兰明慧**相爱，生女（后名**易兰珠**/宝珠）；明慧被迫许配清宗室**多铎**（鄂王）。大婚前夕，杨云骢潜入杭州将军府欲夺回所爱，终与仇家**纽枯鲁**等同归于尽于钱塘；临终将幼女托付给因误会恋人、欲投江自尽的少年**梁穆郎**。穆郎打消死念，携女婴上天山，拜晦明禅师为师，改号学艺。此阶段世界状态：南明余烬未灭，天地会暗流，天山静修，清廷巩固康熙朝权。

### 第一阶段 · 五台出手与顺治之秘（约第1–3回）
十六年后，梁穆郎已成「**天山神芒**」**凌未风**，易兰珠亦成天山剑弟子，立志杀多铎报父仇。天地会得悉多铎将赴**五台山清凉寺**礼佛，部署刺杀。半路杀出的易兰珠误判轿中之人——实为王妃**纳兰明慧**——打乱全盘，甚至倒戈护明慧，致**张华昭**（南明名将**张煌言**之子）负伤被擒。**傅青主**与养女/弟子**冒浣莲**在古刹中发现：五台老僧竟是失踪的**顺治帝**；**康熙**为保帝位暗杀亲父。众人以此秘要挟康熙，换得群雄放行下山。名场面：一女独寻仇、五台山上震三军、隐身古刹空灵幻灭。阶段结果：刺多铎失败；张华昭入京囚笼；易兰珠内疚；宫廷黑幕成为江湖手中「可燃物」。世界状态：天山剑气第一次大规模切入中原政治。

### 第二阶段 · 旧侣刀痕与舍利子恶斗（约第4–6回）
易兰珠独自进京救张华昭。凌未风因事件重会十六年前恋人**刘郁芳**——此时已是**天地会（铁枪会）女总舵主**——却因脸上刀痕毁容与性格执拗，抵死不认自己是梁穆郎。副舵主**韩志邦**暗恋郁芳多年，三人同道上京，三角情感压抑如弦。途中清廷卫士抢夺西藏**舍利子**，凌未风仗义出手，**力败师兄楚昭南**，自身亦中暗器烧伤。郁芳悉心照料；韩志邦黯然离去，却在绝望中意外悟入/习得**达摩心法**，武功跃升。名场面：同门决战、难受温柔岂为新知忘旧好、方窟得真经。阶段结果：楚昭南与凌未风师门仇彻底公开；韩志邦从「情伤配角」变成可托生死的强者；舍利子线把西藏拉进棋盘。世界状态：大内与天山进入热战前夜。

### 第三阶段 · 云南结义与桂仲明治愈（约第7–12回）
群豪转进云南幽谷与山道：凌未风、刘郁芳结识李自成侄孙**李思永**，会合傅青主、冒浣莲，并遇武功奇高却因**误杀生父**而精神错乱的**桂仲明**。傅青主点拨，冒浣莲以深情与陪伴治愈其心疾，二人结为爱侣；桂仲明于剑阁瀑布后得**腾蛟剑**，奠武当北支。凌未风赌技夺金、一剑败三魔等情节，把「五剑」雏形与物资、名声一并做实。名场面：幽谷缔良缘、一女灵机获宝剑、宝剑配神砂。阶段结果：抗清核心从「天地会单线」扩为天山—武当—医侠—闯王后人的联合；桂冒情线成为全书暖色支柱之一。世界状态：西南山中义军气势暂振。

### 第四阶段 · 京华词剑与多铎之死（约第13–19回）
桂仲明、冒浣莲进京探讯；满洲第一词人**纳兰容若**（纳兰明珠之子，与明慧、兰珠有亲缘）为冒浣莲才学倾倒，诗词往还，构成梁羽生标志性的「词章武侠」段落。鄂王妃明慧终认出易兰珠是女儿，恳求多铎勿伤；热恋张华昭的**三格格**牺牲自己，换张、易脱险，二人共患难成婚。多铎出征，易兰珠行刺；多铎几乎手刃她时，因明慧嘱托与兰珠酷似少时明慧而不忍，**最终死在明慧怀中**。易兰珠被擒入天牢；母女相认后爱恨交织，明慧心力交瘁**绝望自尽**，临终与赶来的**飞红巾**尽释前嫌、结为姐妹。凌未风大闹天牢失败，后得飞红巾相助救出易兰珠。名场面：百尺楼头飞剑影、挟符闯狱、王妃梦断恨绵绵。阶段结果：父仇得报却撕开更大伦理伤口；七剑网络因飞红巾正式汇合而接近完整。世界状态：京城血债暂时结清，焦点西移回疆与西藏。

### 第五阶段 · 回疆会盟与辛龙子悔死（约第20–26回）
刘郁芳、傅青主等潜入回疆；群雄聚会草原。李思永与大侠**武元英**之女、白发魔女关门弟子**武琼瑶**相恋。卓一航弟子**辛龙子**一度被楚昭南蛊惑为虎作伥，终在大义前**悔悟力战而死**——原「天山五剑」再损一角。边塞逃亡、荒漠奇缘、石窟诛怪、天山神剑伏奇人等章回，把塞外奇侠气质从《塞外奇侠传》直接注入本书中后段。名场面：边塞逃亡荒漠奇缘、牧野飞霜碧血金戈。阶段结果：回疆义旗与天山剑派深度绑定；楚昭南彻底从「可劝」变为「必诛」。世界状态：抗清重心在塞外，宫廷线转为追杀与诱捕。

### 第六阶段 · 布达拉宫与血书替死（约第27–29回）
凌未风在恶斗中**突发痉挛旧疾**，被楚昭南所擒，押往**西藏布达拉宫迷宫**，百般折磨逼交晦明所遗**拳经剑谱**。越狱失败后，凌未风自度将死，以**血书**向刘郁芳承认：自己就是当年的梁穆郎。郁芳悲痛欲绝。韩志邦铤而走险，化妆入宫说动活佛，救出已近丧失功力的凌未风，**自己假扮其模样留下**；后被楚昭南识破，力战不敌**身死**——全书最沉的悲情落点之一。傅青主、刘郁芳、易兰珠等率众杀入迷宫；武功大进（兼修正反天山剑法）的**易兰珠剑逼楚昭南**，楚昭南无奈**自杀**。名场面：十年幽梦禁迷宫、舍己为人甘替死、绝招雪恨勇除奸。阶段结果：师门叛徒伏诛；韩志邦以死成全「天山」与「天地会」的共同道义；凌未风身份与情感线在血与雪里坦白。世界状态：大内顶尖爪牙崩解，西藏节点收复于侠义一方。

### 第七阶段 · 恩仇了了隐天山（第30回·终局）
群雄收束恩仇：原天山五剑中杨云骢、楚昭南、辛龙子皆已亡故；新「**天山七剑**」确立为——**凌未风、飞红巾、桂仲明、冒浣莲、易兰珠、张华昭、武琼瑶**；**刘郁芳**被尊为「**天山之友**」（非严格七剑之列，却是义旗与情感的关键一环）。七剑与天山之友重整旗鼓，光大天山剑派，仗义行侠，**继续抗清**，而以天山为归隐与再起的家。凌未风与刘郁芳的情感并非传统「洞房大团圆」：血书认身后知己义重、家国优先，悲情英雄气质保留——梁羽生故意把「圆满」留给家国未竟之志，把「残缺」留给私情，使终局读来苍凉而亮。武琼瑶与李思永成婚，把闯王后裔与魔女门风并入天山谱系；易兰珠与张华昭夫妇并剑，南明忠烈血脉与断玉游龙气脉相续；桂仲明与冒浣莲琴剑相偕，武当北支与医侠词章成为七剑中的「文心」。回疆与中原的联络不会因归隐而断：飞红巾仍可在瀚海扬旗，刘郁芳仍可在会党传号。全书以「生死茫茫侠骨柔情埋瀚海，恩仇了了英雄儿女隐天山」收束——不是改朝换代的沙盘胜利，而是剑在、人在、志在，雪山下仍有下山的路。

### 回目气口与叙事节奏（供任意时点定位）
原著回目对仗工整，可当时间戳用：前十回偏「五台—重逢—云南」的结义升温；十一至二十回偏「京华词剑、天牢、王妃梦断」的伦理高潮；二十一至三十回偏「回疆—迷宫—替死—定名」的悲壮收束。AI 若需快速判断「现在书读到哪」，可先问：顺治秘辛是否已揭？多铎是否已死？血书是否已出？七剑名单是否已定？四问可锚定大半进度。人物情感上另有三条并行线不可混：①凌—郁芳—韩的三角成全；②兰珠—张华昭—三格格的牺牲成全；③桂—浣莲的治愈成全。写错谁爱谁、谁替谁死，会直接 OOC。

## 阶位切入点

> 阶位↔江湖战力：一阶≈江湖新人/兵丁/三流剑客；二阶≈会党骨干/二流～一流中坚；三阶≈七剑中坚/傅青主/飞红巾量级；四阶≈凌未风·楚昭南·晦明全盛。世界顶点（凌未风、楚昭南、晦明禅师）为四阶顶，低阶切入一律**规避顶点·情报优先/条件性胜利**；顶点战力按原作全盛书写，不得用「任务侧硬削战力」解释。痉挛旧疾等仅作**条件破绽**（原作病发窗口），禁止把顶点写成无故失去全盛。

切入身份/时点：契约者以天地会记名弟子、客栈小二、进香香客、绿营新兵或南明遗民远亲切入，锚定**五台山清凉寺刺多铎前后至群雄被放行下山**。

初始事件：你被塞了一支暗号香，吩咐「看清鄂王轿帘」；山风里剑光一闪，有人喊「轿中是王妃」——计划碎了，一名负伤青年（张华昭）被官兵拖走，有人塞给你半块令牌要你送去山下医馆。

开场白建议：「清凉寺的钟声压着雪。你本只是送香的，却看见轿帘一角露出女子的凤钗，不是王爷的顶戴。有人骂『坏了事』，有人拔剑，有人把血热的令牌拍进你掌心——你突然知道，今晚的香灰里埋着刀。」

关键NPC立场：**易兰珠**冲动且愧疚，可能拉你当目击者或挡箭；**张华昭**重伤时会托信；**傅青主**冷静收局，可收你做跑腿；**冒浣莲**心善，或替你包扎；大内耳目视你为可灭口的香客；**刘郁芳**若在外围会以会党规矩检验你。

主线钩子/支线：主线——是否把「轿中是纳兰明慧」的目击卖给会党或官府（蝴蝶点：加速或延缓康熙线与放行谈判）。支线：①护送半块令牌到山下而不被官差截获；②在混乱中救下一名被误伤的僧人换取寺内人情；③拒绝伪造「多铎已死」的谣言单。

危险度/规避：中低。规避与任何成名剑客硬刚；见「天山神芒」或大内侍卫头目即逃。无四阶贴脸。

任务方向/奖励：活着离开五台并带回一条可验证的会党/宫廷流言；奖励取向：三流通用剑谱残页、伤药、会党外围腰牌、傅青主或冒浣莲的一次性人情（非核心剑诀）。

**二阶（成名好手·舍利子道与云南谷）**

切入身份/时点：契约者以二流剑客、天地分舵护法、镖师或回疆商队护卫切入，锚定**舍利子争夺至云南幽谷结义、桂仲明获腾蛟前后**。

初始事件：你护送的「佛骨」匣在官道被截，红光里两名同门剑客互砍——有人叫他们楚昭南与凌未风；或你在云南谷中撞见一个疯癫青年把「杀父」二字刻满树皮，红衣女子（冒浣莲）正以琴声稳他心神。

开场白建议：「舍利子三个字比金子烫。你还没摸到匣扣，剑风已经削断马鬃。左边的人骂『叛门』，右边的人一声不吭，只有暗器在月下闪成芒星——你意识到，站错半步，名字就会写进邸报的盗匪栏。」

关键NPC立场：**凌未风**重义可结盟，但不喜被追问旧名；**楚昭南**会开价收编或灭口；**韩志邦**压抑而可靠，可引你入会党深层；**桂仲明**疯时危险、愈后可结生死；**冒浣莲**以心交友；**李思永**重义军纪律。

主线钩子/支线：主线——舍利子真相链上选择交给喇嘛、会党或双重倒卖（蝴蝶：影响楚凌伤势与西藏伏笔）。支线：①协助冒浣莲稳定桂仲明一次疯病发作；②在瀑布剑阁外围挡住夺剑小贼；③为李思永送一份云南义军粮草伪账。

危险度/规避：中。规避单独拦击凌未风或楚昭南；不进大内密札交易核心；「治心病」现场可能被官差当作邪教窝点。

任务方向/奖励：拿到可证明楚昭南投清或舍利子去向的物证；奖励：准一流身法心得、解毒膏、腾蛟传说绘图一角、韩志邦或傅青主的有条件承诺。

**三阶（一流顶尖·京华天牢与回疆会盟）**

切入身份/时点：契约者以散修一流、小派掌门、会党舵主级或回疆盟事切入，锚定**易兰珠刺多铎至明慧自尽、飞红巾救天牢，及回疆辛龙子悔死前后**。

初始事件：你被请作「天牢外应」或「草原盟会证人」；夜半听到王妃哭声与剑鸣，有人传「鄂王死在王妃怀里」；或草原上辛龙子突然调转枪头对着楚昭南的旗帜，血先于话落地。

开场白建议：「京城的夜把刀藏在词里。你本去听纳兰公子的新词，却听见天牢方向有人喊『女儿』。草原上又是另一种听法——盟火一灭，就有人要用叛门的剑重新点着。你握紧剑柄，知道自己已不是看客。」

关键NPC立场：**易兰珠**刚烈，认友极快也认敌极快；**飞红巾**盟主气度，可赐你草原路引；**纳兰容若**可提供非武力门路；**张华昭**重诺；**辛龙子**悔悟前危险、悔悟后可托孤式托事；**武琼瑶**/**李思永**重双修情义与义军。

主线钩子/支线：主线——天牢救援中选择主攻、诱敌或护送明慧遗物（蝴蝶：影响兰珠心结与飞红巾结义）。支线：①替纳兰容若送一首可能惹祸的词稿出宫；②在回疆揭穿一份假「楚昭南劝降书」；③见证辛龙子临终并带回其悔书给凌未风。

危险度/规避：高。规避与楚昭南单挑至死；天牢内无内应勿深闯；回疆盟会中勿当众辱飞红巾。顶点若现身以情报与条件破绽周旋，**条件性胜利**，禁止写其无故失去全盛。

任务方向/奖励：救出/保全一名七剑相关人物或拿到明慧/辛龙子关键遗物；奖励：一流剑理指点、回疆良马与暗器袋、飞红巾或易兰珠的短期同盟、达摩心法外围吐纳（非完整韩志邦所得）。

**四阶（人间剑顶·布达拉宫与七剑定名）**

切入身份/时点：契约者以当世一流巅峰、西域剑客、会党总舵核心或「准七剑」盟友切入，锚定**凌未风被囚布达拉宫、韩志邦替死、楚昭南自杀至天山七剑定名**。

初始事件：你收到一封尚未干透的血书抄件——「我是穆郎」——或活佛殿外有人用你的脸做面具（韩志邦线）；迷宫深处楚昭南以拳经剑谱为饵，问你「投清还是陪凌未风死」。

开场白建议：「雪落在布达拉宫的金顶上，像谁写完的句号。血书只有三字真名，却重过游龙剑。有人已经决定替死，有人已经决定自杀，而你站在迷宫口，知道此夜之后，江湖会记住『七剑』，也会记住那些没能走进七剑名单的人。」

关键NPC立场：**凌未风**功力受损时仍是人格顶点，可托付后事；**韩志邦**已决死，只需你保证郁芳与大局；**楚昭南**在绝境仍要谱与面子；**易兰珠**剑锋对准叛徒；**刘郁芳**悲愤中统军；**傅青主**布局破阵；终局**七剑全员**可能请你做「天山之友」侧见证。

主线钩子/支线：主线——破迷宫营救中选择「换囚」「断粮道」或「直取楚昭南」三策之一（蝴蝶：韩志邦能否少付代价、拳经剑谱是否外流）。支线：①护送血书真迹到刘郁芳手中不被调包；②阻止朝廷焚毁晦明遗谱；③在七剑定名仪式上拒绝或接受「编外之剑」的位置。

危险度/规避：极高/贴近顶点。与全盛楚昭南、恢复中的凌未风对决须**情报优先/条件性胜利**（用地利、人数、病发窗口、谱卷要挟等原作逻辑）；禁止写顶点无故失去全盛。世界峰值即此层，无更高阶可召唤。

任务方向/奖励：参与逼死或逼退楚昭南、保全凌未风与遗谱、见证七剑定名；奖励：游龙/断玉/腾蛟级剑理观摩（非直接夺剑）、大须弥剑意入门比喻、天山或天地会核心盟约、史诗级人脉（七剑联名）——不发毁城级装备，不突破四阶上限。
