# 真・恋姫†夢想-革命- 劉旗の大望

## 剧情

**【作品来源】**
《真・恋姫†夢想-革命- 劉旗の大望》是 BaseSon（NEXTON 旗下）「恋姫†」系列「革命」三部曲的蜀篇，2019年7月26日发售的 18 禁恋爱冒险＋SLG 要素作品。系列前作含《真・恋姫†夢想-革命- 蒼天の覇王》（魏篇，2017）、《孫呉の血脈》（吴篇，2018），本作聚焦蜀汉阵营。人设沿袭片桐雛太等原画班底，气质为：少女三国志、主公养成、后宫向纯爱与群像羁绊。本档案只取**人物魅力、情感线与日常／据点生活侧**，不写军略对决。

**【世界定位】**
这是「刘旗」飘扬的少女三国：契约者应以**北郷一刀式主公／军师伴读**身份，在桃园、军帐与村镇日常中经营与蜀将们的信赖与恋心，而非沙场数值。

**【世界观 · 舞台设定】**
架空的三国乱世被 BaseSon 重写为「全是少女武将」的华丽舞台：黄巾余波、群雄割据的大势仍在，但档案重心落在**据点生活、结义、恋爱与人心**。蜀线的温度来自：桃园结义的誓言、仁德之名带来的追随、军中饭桌与篝火夜谈、出征前夜的告白与归还后的重逢。超自然／奇谋设定若出现，只作「命运与羁绊的隐喻」，不写成对决升级。乱世只是压力背景——真正主题是「愿与谁共守一方桃源」。

**【地理 · 生活舞台】**
- **桃园／结义之地**：刘・关・张情感原点；誓约与重温誓言的名场面地。
- **军帐与议事厅**：孔明献策、主公决断；日常则是文书、茶点与吐槽。
- **训练场／马厩**：赵云、马超等武将的日常修炼与并肩流汗。
- **村镇市井与农舍**：仁政侧写；与民同乐、节日与救济是情感润滑。
- **城墙夜巡与月下**：告白、误会和解的经典舞台。
- **行军途中的篝火**：闲谈、酒、回忆与「明天还要并肩」的温柔。
- **蜀中山水与临时行宫**：休整期的约会与后日谈感。

**【故事主线 · 情感线】**
**共通基调（革命・蜀篇）**  
玩家作为与蜀旗绑定的主公位存在（系列传统的「北郷一刀」系主公／或同构的转生主公），在乱世中聚集蜀将，以仁义凝聚人心。革命三部曲各以魏／吴／蜀为主视角；本作让**刘备（玄德）**与蜀众成为情感中心：从起兵、结义、三顾／求贤、到荆州・益州经营的情感版图，落脚于「旗下之人愿为谁而战、愿为谁而活」。

**情感总弧**  
相遇与结义 → 据点日常中的信赖升温 → 出征／离别的不安 → 归还与告白 → 各将个人线的心结回收 → HE：与选定对象共守蜀旗下的日常（后宫向作品亦有多线并进的温存结局）。

**刘备（玄德）线**  
心结是「仁德是否只是软弱」「能否保护所爱之人」。攻略核心是：在她自我怀疑时肯定其温柔，而非逼她变成冷酷霸主。名场面多为月下谈志、为伤员守夜、与主公并骑回望桃园。结局：共治／共枕的主公与主君，旗与心同在。

**关羽（云长）线**  
义与恋的拉扯：对主公的忠诚与私情如何并存。攻略靠并肩、守约、在她「义」的坚持前不退缩。名场面：夜巡交心、单骑相送、红颜一怒为谁。HE：义与恋合一，青龙偃月旁是并肩的人。

**张飞（翼德）线**  
豪爽外表下的细腻与吃醋。日常是酒、练武、被哄与被夸奖。攻略吃「真心夸奖＋一起闹＋关键时刻的认真」。HE：最吵闹也最安心的伴侣位。

**诸葛孔明线**  
智者的孤独与「被需要」的渴望。攻略在议事厅深夜、观星、替她分担「算无尽的人心」。HE：军师卸下全知之重，与主公共享日常琐碎。

**赵云（子龙）线**  
完美武将的自我压抑与「想被当作普通人」。攻略在训练后的休息、护卫任务中的反向守护。HE：银枪放下后的柔软日常。

**马超／黄忠／其他蜀将线**  
各有家族、年龄、功名心结；情感落点均为「被主公看见作为『人』的一面」。结局统一为：乱世未尽，但两人的日常已立。

**【可攻略角色 / 主要人物】**
北郷一刀／玩家主公（主人公）｜外貌：系列传统青年武者／书生感可变｜性格：仁义、会吐槽、重情｜角色类型：后宫中心／结义核｜萌点：被武将们围着仍认真听每个人说话｜个人线：从外人到蜀旗灵魂｜与主角关系：契约者默认同位。

刘备・玄徳（蜀主／仁德之君）｜外貌：柔和长发、华丽武饰、温和眉眼｜性格：温柔、优柔却在关键时坚定｜角色类型：大和抚子／主君系｜萌点：结义誓言、为部下操心到自己倒下｜个人线：仁与恋｜与主角关系：最核心的情感锚。

関羽・雲長｜外貌：飒爽长身、红系意象、凛然美｜性格：重义、寡言深情｜角色类型：飒爽义侠／傲娇边缘｜萌点：义字当头却会为恋心动摇｜个人线：义与私情｜与主角关系：可攻略的绝对信赖位。

張飛・翼徳｜外貌：英气、豪放着装｜性格：直率、吵闹、重情｜角色类型：元气／豪快｜萌点：酒与练武、突然的认真告白｜个人线：从结义姐妹到恋人｜与主角关系：最容易亲近的热闹位。

諸葛亮・孔明｜外貌：羽扇纶巾意象的美少女军师｜性格：聪慧、腹黑轻、内心孤独｜角色类型：天才军师／冷静系｜萌点：观星、吐槽主公、算无恋爱｜个人线：卸下全知｜与主角关系：议事厅深夜的独占。

趙雲・子龍｜外貌：银白系武饰、端丽｜性格：忠谨、自我要求极高｜角色类型：完美骑士／禁欲温柔｜萌点：护卫时的距离感与私下的松懈｜个人线：想被当普通人｜与主角关系：并肩护卫变双向守护。

馬超・孟起｜外貌：西凉风、飒爽｜性格：好胜、重家族｜角色类型：傲娇武姬｜萌点：练武较劲、被温柔接住｜个人线：家族与归属｜与主角关系：从对手感到眷属。

黄忠・漢升｜外貌：成熟武将美｜性格：稳重、不服老｜角色类型：年长可靠｜萌点：弓与酒、默默守护｜个人线：被承认「仍在盛时」｜与主角关系：半师半侣。

重要配角：蜀中文臣武将群（依原作登场）、魏吴侧作为外交／对照的群像（曹操・孙权等，本篇作客串情感对照，不展开军争）。

**【人际关系网 / 社团势力】**
- **桃园三角**：刘・关・张结义核心；主公介入后变成「主公＋三姐妹」的情感磁场。
- **军师与主君**：孔明—刘备—主公的信任三角。
- **五虎级武将圈**：赵云・马超・黄忠等与主公的「被看见」关系网。
- **据点日常组**：炊事、巡逻、文书的生活羁绊，比军议更催恋爱。
- **革命三部曲对照**：魏／吴篇的旗与人，在蜀篇以书信、使者、回忆出现，强调「各旗之下皆有所爱」。

- 系列传统的后日谈／CG 回收结义与主公位选择。  
- 隐藏对话多在据点反复访问、特定将好感阈值后触发。  
- 本档案不剧透全部分支 BE；情感原则：BE 多来自「背弃仁义或忽视她的心结」，HE 来自「看见她作为人」。

**【补记 · 情感节奏】**
从「初识的礼貌距离」到「可以沉默相对」通常需要数周校园／据点时间。档案写作时优先安排：共同任务 → 私话 → 危机中的选择 → 告白后的生活细节。避免一章之内从陌生跳到终身许诺；让角色的口癖、小动作与固定场景反复出现，读者才能记住「这是她」。季节变化（樱、蝉、雾、雪）应用来标记关系阶段，而非堆砌风景。

**【补记 · 对话与称呼】**
称呼变化是情感进度条：姓→名→昵称→只有两人场合才用的叫法。配角在场时的「演」与两人独处时的「素」要有落差。禁止用「女主」「那孩子」代替真名。吵架戏必须有具体事由（日程冲突、隐瞒、旧伤被戳），和解要有一方先低头的具体动作（递红茶、留灯、把伞塞过去）。

**【补记 · 日常物件清单】**
每条线至少绑定 2～3 件「信物级」日常物：一把扇子、一只多出来的杯子、一枚书签、一条封缄札、一盒便当、一张旧海报。物件在高潮戏重现时，比任何台词更催泪。写 FD／后日谈时让物件磨损、被修理、被一起收纳——关系就有了时间的重量。

**【补记 · 群像不抢戏】**
多女主作品的共通线要让每个人都有「被看见的三分钟」，但个人线必须敢于冷落其他人而不写成恶意：用出差、部活、归省、生病等自然缺席。情敌感点到为止，主题永远是「选择与被选择的郑重」，不是踩低他人。
