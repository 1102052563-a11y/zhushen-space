# 高达水星的魔女

## 剧情

**【作品来源】**
《机动战士高达 水星的魔女》（日：機動戦士ガンダム 水星の魔女；英：Mobile Suit Gundam: The Witch from Mercury），日升动画制作，监督小林宽，系列构成大河内一楼，原作矢立肇·富野由悠季。电视动画两季共24话（2022年10月—2023年1月；2023年4月—7月），另有序章（PROLOGUE，2022年7月先行）。日历为星元（Ad Stella，A.S.）。主题：企业霸权与「魔女」污名、地球人/宇宙人阶级裂痕、亲子操控与克隆身份、校园决斗制与LGBTQ情感线。文风校园青春与企业战争双层并行，受莎士比亚《暴风雨》与女巫审判意象启发。卡片锚点：**奎达工厂恐袭——地球武装「弗特的黎明」驾驶乌尔魔灵与索恩魔灵袭击贝纳里特集团奎达工厂，风灵高达（艾莉克特）开启帕梅特刻痕6，展现碾压级无人机控制力；斯莱塔在普洛斯佩拉引导下首次杀人。**

**【世界定位】**
A.S.122年，贝纳里特集团支配的阿斯提卡西亚高等专门学园与地球—宇宙经济圈。水星出身少女**斯莱塔·墨丘利**驾驶禁忌的**风灵高达（Aerial）**转学入园，与总裁之女**米奥莉奈·伦布兰**因决斗制结为「新郎—新娘」关系，卷入GUND-ARM复兴、公司政治与母亲**普洛斯佩拉·墨丘利**（真名埃尔诺拉·萨玛雅）的复仇计划**寂静零号**。一句话：魔女之女在学园与工厂之间学会「前进」，用机体与选择改写企业秩序，终与米奥莉奈成伴侣。

**【世界观 · 力量体系】**
**帕梅特（Permet）**：太阳系局地特产矿物/粒子，粒子间可共享信息，混入素材、推进剂与机体即可实现控制与通信。人类进军宇宙的机能扩充技术多建立在帕梅特之上。

**GUND 与 GUND-ARM（高达）**：原为医疗用脑机接口，补偿宇宙环境下的身体损伤；被武器化为**GUND格式**后称**GUND-ARM（高达）**。驾驶员以神经系统/信息链路同步机体，性能跃升，但高负荷会导致**数据风暴**、肉体崩溃乃至死亡——开发者（多为女性）被污名为**「魔女」**。A.S.101 **凡娜迪斯事变**后，机动战士开发评议会与**教廷（Cathedra）**体系严禁高达，由**多米尼克斯队（Dominicus）**执行肃清。

**帕梅特刻痕（Permet Score）**：衡量驾驶员—机体—帕梅特链路强度的等级。刻痕越高，GUND-BIT（无人机/浮游炮位）操控、反应与领域控制越强；低刻痕多显红光，高刻痕（约6起）多显蓝光。常人与量产强化人在高刻痕下极易致死；**风灵**因内置**艾莉克特·萨玛雅**的意识/控制核，可让斯莱塔几乎无伤承受常人无法负担的刻痕；终局**凯列班高达**无滤网，斯莱塔强开超高刻痕后留下终身后遗。

战力层级（按破坏力·编制，宁低勿高）：
1. **无武装平民 / 学园普通生 / 工场劳工**：无MS战力。观感≈一阶。
2. **量产MS学员 / 宿舍普通驾驶员**：训练用机、企业标准机，可毁车辆、制服同级。观感≈一~二阶。
3. **学园卫冕级 / 企业精锐改修机驾驶员**（杰特克达里尔巴尔德、格拉斯利恶魔审判者等）：决斗与据点战主力，可重创同级MS、压制小队。观感≈二~三阶。
4. **GUND-ARM 主力 + 高帕梅特刻痕（风灵·乌尔/索恩魔灵·铁骑·凯列班等）**：摧毁工厂设施与建筑群、碾压量产集群、广域GUND-BIT压制；**本世界库覆盖阶位（四阶）的主战场落在此层**——尤其是奎达工厂恐袭、学园公开日屠杀、寂静零号周边的数据风暴战。观感≈**四阶**（质变分水岭：摧毁据点/掀平建筑群级）。
5. **企业舰队交战 / 弗特的黎明武装据点级**：战区灾难、可瘫痪工厂与学园区域。观感≈五阶向。
6. **寂静零号要塞 + 高诺德无人机网 + 广域数据风暴**：覆盖地球圈局部、强行接管含帕梅特机械、舰队级压制。观感≈六~七阶向。
7. **宇宙议会联合体 ILTS 行星间激光传输系统**：战略级能量打击，可摧毁要塞与舰队。观感≈七~八阶向。
8. **凡娜迪斯事变级全面肃清 / 地球圈企业—议会总动员战争（历史与终局峰值）**：文明秩序重写级威胁。观感≈八阶边缘。

**乐园阶位映射**（对照《阶位战力图鉴》破坏力，宁低勿高）：平民·学员·工场劳工≈一阶；量产MS与学园普通决斗≈二阶；企业精锐改修/卫冕级≈三阶；**GUND-ARM 高刻痕主力与奎达工厂级都市/工厂战≈四阶**；舰队与武装据点≈五阶；寂静零号领域≈六~七阶；ILTS 战略激光≈七~八阶。**本世界库仅覆盖四阶切入**。世界顶点＝**寂静零号+艾莉克特/风灵数据层改写**与**ILTS 战略打击体系**：对中阶为**超阶巅峰感：存在·情报优先/条件性胜利**，严禁「战力归零式削弱」顶点。禁止低中阶硬刚寂静零号本体与ILTS；四阶主玩学园决斗、工厂恐袭、企业暗战与GUND-ARM局部战。

补充成长规则：学园决斗以「契约」约束杀伤，但企业与弗特的黎明不受校园礼仪保护；GUND抗性因人而异，强化人与克隆链路是特例不是常态。死亡在凡俗与学员层不可逆；高阶或有假死、义体、数据残响，皆有政治与肉体代价。世界不提供无限复活。

**【地理 · 舞台】**
- **水星·新星开发公司据点**：斯莱塔童年与「姐姐」风灵共同成长的严酷矿星；普洛斯佩拉以母女情与「前进就会得到两个」的教条塑造工具。
- **阿斯提卡西亚高等专门学园**：贝纳里特集团旗下精英校，企业宿舍制（杰特克、格拉斯利、佩尔、地球宿舍等）；决斗制决定卫冕者与米奥莉奈的「新娘」归属。校园日常、决斗场、公开日是中前期主舞台。
- **地球宿舍 / 高达公司（Gund-Arm Inc.）**：地球人学生聚居与创业据点；尼卡、雀丘利等；米奥莉奈推动GUND医疗合法化的前台。
- **贝纳里特集团总部与各社工厂**：杰特克重工业、格拉斯利防卫系统、佩尔科技、新星开发等子公司瓜分MS市场；总裁德林·伦布兰坐镇。
- **奎达工厂（Plant Quetta）**：**卡片锚点主舞台**。德林与普洛斯佩拉密谈寂静零号之地；弗特的黎明恐袭、风灵刻痕6、斯莱塔首次杀人、德林重伤、盖尔误杀其父维姆，皆在此定格。
- **弗尔克范格（Fólkvangr）**：A.S.101 凡娜迪斯研究所设施，多米尼克斯屠杀现场；序章逃出点。
- **地球·克茵哈伯等地**：米奥莉奈推动GUND医疗、反宇宙人运动与秘密高达产线冲突的地面舞台。
- **寂静零号要塞**：终局数据风暴领域与高诺德无人机巢；普洛斯佩拉—艾莉克特计划的实体。
- **ILTS 武装星际激光卫星**：宇宙议会联合体战略威慑，终局炮击寂静零号。

写场景先定层级：学园宿舍=配额、面子与决斗契约；工厂与港湾=合同、恐袭与保密；地球贫民与劳工区=阶级怒火；寂静零号=数据层改写权限。本地人若听说「学园新生要单机打掉联合体激光网」，会当疯子——这是正确反应。感官锚：帕梅特粉尘的金属甜味、决斗场全息裁判音、工厂冷却液与血混在一起的气味、水星矿渣风沙、地球宿舍廉价合成餐。

**【世界剧情线】**
① **A.S.101 凡娜迪斯事变（序章）**  
**凡娜迪斯研究所**将GUND医疗技术武器化，在奥克斯地球资助下推进GUND-ARM。**德林·伦布兰**主导的机动战士开发评议会以「保护人类」为名禁绝高达，派**多米尼克斯队**突袭弗尔克范格。测试驾驶员**埃尔诺拉·萨玛雅**与幼女**艾莉克特·萨玛雅**驾原型**魔灵高达（Lfrith）**突围；夫**纳迪姆·萨玛雅**殿后战死。幸存者被钉上「魔女」污名。埃尔诺拉后化名**普洛斯佩拉·墨丘利**，将病弱的艾莉克特意识/控制核移入后续机体**风灵**，并以基因技术养育克隆体**斯莱塔**为第二女儿，在水星蛰伏二十一年，布局复仇与「解放艾莉」的**寂静零号**。世界状态：魔女被猎，企业秩序以肃清立威。

② **A.S.122 转学与卫冕者**  
斯莱塔自水星赴阿斯提卡西亚，途中救下试图逃离父亲掌控、漂在宇宙中的**米奥莉奈·伦布兰**——米奥莉奈原计划借逃出学园切断「总裁之女=决斗奖品」的命运。为夺回主动权，她胁迫斯莱塔接受卫冕者**盖尔·杰特克**的决斗。斯莱塔驾**风灵**以压倒性机动与BIT防御完胜，成为新卫冕者与米奥莉奈的「新郎候选人」（学园规则在同性婚姻合法语境下仍把新娘归属绑在卫冕契约上）。评议会与教廷眼线立刻怀疑风灵是高达；普洛斯佩拉以**新星开发公司**社长身份到场掩护，用「水星特制」话术挡第一波。米奥莉奈逼出重赛条款，盖尔动用AI辅助与场外手段仍败，面子与杰特克股价同步崩裂。斯莱塔因考试舞弊陷害一度孤立，后被**地球宿舍**收留，结识**尼卡·七浦**、**雀丘利·潘兰杞**等，以口吃式善意、超额完成「有用」指标与驾驶天赋立名。名场面：第一次决斗后米奥莉奈的嫌弃与试探；重赛中风灵「不让驾驶员痛」的异常。主题：井底规则是「决斗=契约=身份=股价」。

③ **铁骑、强化人与高达公司**  
佩尔科技推出的「完美学员」**伊兰·凯莱斯**（强化人4号）接近斯莱塔，带她看星星、学社交，实为测试风灵是否为**无负荷高达**。决斗中风灵在危急时刻自主介入，GUND干扰特征曝光；佩尔高层视4号为耗材，数据收齐后将其处理掉——斯莱塔第一次尝到「被当人看待又被系统回收」的企业伦理。米奥莉奈以政治与商业手段创立**高达公司（Gund-Arm Inc.）**，把禁忌技术重新包装为安全/医疗应用，吸纳地球宿舍为整备与运营班底，挡下退学决议与格拉斯利式吞并。创业派对上舆论与资本同时砸场，米奥莉奈用股权结构与公开承诺硬扛。**沙迪克·泽内利**率格拉斯利宿舍发动团体决斗，企图一次拿走公司控制权与米奥莉奈人身；风灵在强干扰环境下以BIT破网反杀式压制「恶魔审判者」队。此后米奥莉奈把业务重心推向GUND医疗，与父亲德林「禁绝=和平」的强硬路线公开离心。世界状态：禁忌技术重新合法化的缝隙被撕开，地球人学生第一次摸到股份而非只有打扫配额。

④ **奎达工厂恐袭（卡片锚点·详述）**  
贝纳里特内部暗杀与权力斗争升温：有人要德林死，有人要股价活。斯莱塔、米奥莉奈与地球宿舍相关人员赴**奎达工厂**接应德林并护送风灵相关资产；普洛斯佩拉与德林在深层会议室推进**寂静零号**——表面是「用GUND链路终结战争」的和平工程，实则绑定艾莉克特存在与旧仇。地球武装**弗特的黎明**（与沙迪克政治线、尼卡技术线有暗线牵连，亦与杰特克内部阴谋交错）发动恐袭：**苏菲·普罗涅**驾**乌尔魔灵**、**诺蕾雅·杜诺克**驾**索恩魔灵**，GUND-BIT与实弹火力把工厂防卫撕成碎片，平民技工与量产驾驶员在警报里成批倒下。普洛斯佩拉在频道里用母亲的声音引导斯莱塔：「前进。」风灵升**帕梅特刻痕6**，艾莉克特接管式展开碾压级无人机控制，乌尔/索恩的优势被数据层反制。混乱中**维姆·杰特克**的夺权/灭口图穷匕见，**盖尔**在自卫中误杀其父，杰特克血脉与企业继承同时炸裂；德林为护米奥莉奈重伤倒地。一名士兵仍扑向米奥莉奈与垂死的德林——斯莱塔以风灵双手将人机一并碾碎，**第一次明确杀人**，松了口气的表情与米奥莉奈的恐惧同框。米奥莉奈震怒称她为「杀人犯」，卫冕契约下的亲密瞬间被血洗成裂痕。  
**锚点写法铁则**：必须同场出现企业密谈、弗特的黎明恐袭、风灵刻痕6、斯莱塔首次杀意与米奥莉奈道德裂痕、盖尔弑父创伤；禁止把高刻痕写成无代价开挂，禁止低阶硬刚寂静零号本体；四阶可玩的是厂区巷战、疏散、证据与站队，不是战略层清场。

⑤ **公开日、苏菲之死与总裁选举**  
恐袭两周后，苏菲与诺蕾雅以「使者/转校生」潜入学园。公开日她们携**高沃尔瓦**等GUND-BIT兵器入侵，造成学生死伤（含盖尔之弟**劳达·尼尔**一系创伤）。风灵/艾莉克特以数据风暴压制，苏菲因无法承受负荷身亡；事件被集团掩盖，强硬治安反招舆论反噬。贝纳里特决定总裁选举：沙迪克、佩尔强化人5号（新「伊兰」）、重整后的盖尔与米奥莉奈各自入局。普洛斯佩拉向米奥莉奈抛出接管寂静零号的条件。世界状态：校园从决斗场升级为内战前哨。

⑥ **抛弃、复制人真相与地球战场**  
米奥莉奈为把斯莱塔从母亲棋盘上剥离，与盖尔合谋：以病毒/停机削弱风灵，令斯莱塔卫冕决斗败北，剥夺新郎地位并刻意冷待。风灵内**艾莉克特**现身：斯莱塔是其**基因克隆（repli-child）**；帕梅特刻痕达第8层的风灵与普洛斯佩拉已不再需要她。斯莱塔被弃于宇宙，精神崩溃，后在地球宿舍支撑下重建自我，开始质疑「前进就会得到两个」的教条。米奥莉奈赴地球克茵哈伯推GUND医疗、压反宇宙人运动；普洛斯佩拉与风灵摧毁当地秘密高达产线，地面战爆发。盖尔揭穿沙迪克为多起暗线黑手，沙迪克被多米尼克斯逮捕；诺蕾雅无差别攻击后被杀，学园遭重创。

⑦ **寂静零号启动与终局**

**四阶（GUND-ARM 高刻痕 · 奎达工厂恐袭线）**

切入身份/时点：契约者以「贝纳里特系工厂安保顾问 / 学园决斗特待生 / 地球宿舍外援整备员 / 新星开发随行技术员」之一切入，锚定**奎达工厂恐袭前后**——德林与普洛斯佩拉密谈寂静零号、弗特的黎明乌尔/索恩魔灵即将突袭、风灵即将升帕梅特刻痕6、斯莱塔尚未完成第一次杀人的窗口。亦可略早落在团体决斗后、赴奎达护送途中；略晚则落到公开日前夜，但恐袭血债与「杀人犯」裂痕必须已成既定事实。

初始事件：契约者随护送编队进入奎达工厂封锁区，安检警报与内部暗杀传闻并行。突袭开始时，量产防卫线在GUND-BIT下像纸一样碎；有人塞给你一支「非登记」步枪芯片或一张假通行证，要你在「掩护米奥莉奈撤离」「抢修风灵外甲」「给弗特的黎明开后门」三条指令里当场站队——选错会立刻被多米尼克斯或企业安保点名。同一分钟内，你可能看见盖尔在频道另一头崩溃，普洛斯佩拉却仍在用平静语调指挥女儿「前进」。

开场白建议：「奎达工厂的警报比广播更早刺进头盔。走廊尽头，乌尔魔灵的光学涂装像血一样亮；你听见有人用很轻的声音说『刻痕6』——下一秒，风灵背上的BIT展开成墙，量产机整排断电跪地。米奥莉奈的呼吸在频道里发抖，斯莱塔的声音却稳得可怕。你握着自己那点四阶够用的权限，忽然明白：这里不是学园决斗场，是魔女与企业第一次把命写进合同的地方。」

关键NPC立场：**斯莱塔·墨丘利**——仍信母亲「前进」教条，视你为可一起「派上用场」的同伴，一旦你威胁米奥莉奈或风灵会毫不犹豫开火；**米奥莉奈·伦布兰**——优先父亲与改革筹码，对突然出现的战力又防又用，厌恶无意义杀戮；**普洛斯佩拉·墨丘利**——把你当可消耗的棋子或寂静零号外围遮罩，甜言蜜语下只有艾莉优先级；**盖尔·杰特克**——创伤临界，可能把你当杰特克暗桩或复仇工具；**苏菲·普罗涅** / **诺蕾雅·杜诺克**——视宇宙人体制为敌，可临时同盟但终将屠厂；**尼卡·七浦**——技术后门与良心撕裂，是情报与整备线关键；**德林·伦布兰**——重伤前后都是秩序本身，低位者不可直议；**沙迪克·泽内利**——远在学园侧遥控舆论，你的「英雄行为」也可能被他剪成宣传片。

主线钩子/支线：主线＝**奎达工厂恐袭**全程介入（疏散、夺通讯、保全或暴露风灵刻痕数据、影响斯莱塔是否当众碾碎士兵）；支线①在工厂下层截获弗特的黎明补给清单，换取地球宿舍信任或卖给沙迪克；支线②协助米奥莉奈保全德林，换取高达公司临时顾问身份；支线③窃取普洛斯佩拉与德林密谈碎片，为后续寂静零号线留情报，但禁止四阶玩家「读档式」直接关停要塞；支线④在撤离舰上安抚或激怒盖尔，改变杰特克内斗节奏。

危险度/规避：高——同场GUND-ARM可掀平厂房与防卫集群；必须规避**普洛斯佩拉完全体布局下的风灵刻痕极限**、多米尼克斯整队清场、以及任何提前召唤的**寂静零号本体/ILTS**。对顶点：**超阶巅峰：存在·情报优先/条件性胜利**，严禁写成顶点无代价崩坏或战力归零。契约者可在厂区巷战与信息战中立功，不可单机对轰要塞与激光网。

任务方向/奖励：方向＝在恐袭中保住关键人物/证据、促成或阻止某次致命一击、带着「魔女数据」活着离开奎达，并为公开日/总裁选举留下立场标签。奖励取向＝帕梅特精料与GUND接口模块、学园/企业临时通行权、风灵BIT残骸分析权、地球宿舍人情、四阶可用的改修武装与决斗担保（非寂静零号核心、非ILTS权限）。禁止发放终局数据风暴总控或联合体激光授权。
