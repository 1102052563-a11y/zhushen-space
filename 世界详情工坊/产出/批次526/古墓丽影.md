# 古墓丽影
<!--meta lib=主库 tiers=一、二、三-->

## 剧情

**【作品来源】**  
《古墓丽影》（英语：*Tomb Raider*，港译盜墓者羅拉 / 台译古墓奇兵）是由英国 **Core Design** 于 1996 年开创、后转由美国 **晶体动力（Crystal Dynamics）** 与 **艺夺蒙特利尔（Eidos Montréal）** 续写的动作冒险游戏系列。发行方历经 Eidos、史克威尔艾尼克斯欧洲，现属拥抱者集团旗下 CDE 娱乐等。主角为英国女考古学家 / 探险家 **劳拉·克罗夫特（Lara Croft）**。系列含经典六代（Core）、传奇三部曲重启、2013 起「幸存者三部曲」二次重启，并衍生电影（安吉丽娜·朱莉版、艾莉西亞·薇坎德版）、漫画与小说。全系列销量逾 1 亿套。文风：从解谜平台的「数字邦女郎」到写实求生的起源叙事，贯穿古墓、神器与超自然。本档案以 **幸存者三部曲（2013 / Rise / Shadow）为乐园默认主线**，经典与传奇时间线为可切换「档案层」。

**【世界定位】**  
近现代地球 + 失落文明与神话遗迹并行的探险世界。劳拉从贵族继承人成长为能独自闯绝境的「古墓强盗 / 学者」。对手包括邪教、雇佣兵、永生组织与神话级存在的残响。表层是绳索、弓箭、枪械与攀爬；暗层是「神器不该被凡人妄动」的代价。

**【世界观 · 力量体系】**  
以**人类体能极限 + 生存技巧 + 枪械 / 弓 · 工具 · 神器副作用**为主。死亡基本现实；部分神器可造成风暴、不死、灵魂转移等超常现象，持有者仍可能被杀死。

**1. 凡人与探险者**  
- 普通学者 / 船员 / 记者：无战斗训练。  
- 雇佣兵与邪教徒（太阳神兄弟会 Solarii、三位一体 Trinity 士兵）：制式枪械、近战、数量优势。  
- 特种 / 老兵级（康拉德·罗斯 Roth、部分 Trinity 精英）：战术与体能接近人类顶尖。  
- **劳拉·克罗夫特（成长线）**：攀爬、制作、潜行、弓箭头杀、双枪；从恐惧开枪到冷静处刑。

**2. 超自然与遗物层**  
- **邪马台风暴 / 日女卑弥呼的灵魂容器**：可召唤灭船风暴，需仪式转移宿主。  
- **不朽先知 / 神器（崛起：神圣之源相关）**：触及永生与净化的代价。  
- **玛雅银盒与「清洗」仪式（暗影）**：可改写自然灾变，滥用则末日级。  
- **风暴卫队 / 鬼武士 Oni、不死守卫**：遗迹守护者，物理可摧毁但远强于常人。

**3. 成长与装备**  
幸存者三部曲有经验升级、弓 / 冰斧 / 绳索枪强化；经典作有双枪、车辆与关卡解谜。装备优势 ≠ 肉体四阶。

**乐园阶位映射（宁低勿高）**  
- 平民、落难船员、低阶邪教徒 ≈ **一阶**。  
- 精锐雇佣兵、Solarii 干部、Trinity 士兵、罗斯级导师、成长中的劳拉、双枪熟练劳拉 ≈ **二阶**（小队歼灭、载具与弓枪压制；仍惧重火力与坠落）。  
- 风暴卫队精英、被神器强化的敌首、终盘「仪式级」劳拉在极端环境下的表现、神话守卫 ≈ **三阶**（端据点、裂石断金级机关战、扛小型爆炸级环境伤害）。  
- 完整「清洗」灾变、永不停歇的风暴圈、神器失控灭岛 ≈ **存在级威胁**（情报优先 / 条件性胜利：中断仪式、摧毁容器，非硬抗天灾）。  
**世界个人战力顶点 ≈ 三阶**（成熟期劳拉、Trinity 顶级执行者、神器宿主敌首）。严禁把劳拉写成无敌超人；她会流血、会怕、会死。

**【地理 · 舞台】**  
- **邪马台（Yamatai）岛**：龙三角海域失落岛；海岸船骸、山中修道院、太阳神要塞、Endurance 残骸——2013 主舞台。  
- **西伯利亚 / 基捷日相关遗迹（崛起）**：冰原、苏联设施、圣城废墟。  
- **秘鲁 / 墨西哥 / 中美洲丛林与地下城（暗影）**：玛雅遗迹、油田、库瓦科亚特尔相关圣地。  
- **克罗夫特庄园（英格兰）**：情报与回忆枢纽。  
- **经典层**：秘鲁维拉科查、中国长城、Atlantis、埃及、希腊等关卡式古墓。  
- **功能点**：营地升级台、挑战墓、文献收集点、滑索与攀爬点。

**【世界剧情线】**  

**① 经典起源层（档案）**  
1996 起，劳拉受雇或自发寻找神器：Atlantis 的 Scion、匕首、埃及神器等，对抗雅各布·维斯纳（Jacqueline Natla）等反派。Core 后期《黑暗天使》口碑崩盘后系列转交晶体动力。

**② 传奇三部曲层（档案）**  
《传奇》重述母亲与亚瑟王圣剑线；《周年纪念》重制初代；《地下世界》涉及北欧神话与劳拉之母的真相。气质偏超级英雄式动作。

**③ 默认主线 · 幸存者三部曲**  

**A. 2013《古墓丽影》· 邪马台**  
年轻劳拉随考古船 Endurance 按自己理论进入龙三角寻找邪马台，船毁人散。她从洞穴逃生，首次杀人，在康拉德·罗斯指导下求生。岛上「太阳神兄弟会」崇拜女王卑弥呼，领袖马蒂亚斯（Mathias）要为卑弥呼寻找Ascension（登神）容器。劳拉挚友萨曼莎·西村（Sam Nishimura）因血脉被选中。风暴实为卑弥呼灵魂困于尸身、拒绝移交力量所维持。罗斯、格里姆等牺牲；劳拉摧毁卑弥呼遗骸、杀死马蒂亚斯，救出 Sam，却选择继续探寻神话而非回家——「A Survivor is Born」。

**B. 《崛起》Rise of the Tomb Raider**  
劳拉追寻父亲理查德·克罗夫特未竟的「神圣之源 / 不朽」研究，对抗全球阴谋组织 **Trinity（三位一体）**。舞台西伯利亚，涉及先知、基捷日与永生代价。劳拉从「幸存者」转向主动袭击 Trinity 设施的猎人，确认神器被滥用的伦理线。

**C. 《暗影》Shadow of the Tomb Raider**  
劳拉与约拿·马亚瓦（Jonah Maiava）在中美洲行动时，误触银盒导致海啸级灾难，愧疚驱动她阻止 Trinity 领袖多明格斯博士（Pedro Dominguez）完成「清洗」仪式。她深入玛雅预言与「世界终结 / 重生」神话，最终在牺牲与救赎间做出选择，三部曲收束于劳拉接受「自己既是学者也是会改变世界的人」。

**④ 结局方向**  
幸存者线：劳拉活着离开，神话被暂时按住，Trinity 受重创但世界暗流仍在；她成为真正的「古墓丽影」。经典 / 传奇层各自有 Natla 与北欧线结局。电影层为平行演绎，可作外观与名场面参考。

**【主要人物】**  

- **劳拉·克罗夫特（Lara Croft）**（主角）｜性格：聪慧执着、早期恐惧后期坚韧、带愧疚感｜装备·能力：冰斧、制作台弓箭、绳索、双枪（成长期后）、求生本能高亮｜人物弧光：惊慌毕业生→杀死邪教领袖的幸存者→对抗 Trinity 的猎人→接受阴影的探险家｜立场关系：视罗斯为父执；与 Sam 挚友；与约拿生死搭档。  
- **康拉德·罗斯（Conrad Roth）**（导师）｜性格：强硬温柔｜装备·能力：前皇家海军、求生专家｜人物弧光：护船护劳拉至死｜立场关系：劳拉精神之父。  
- **萨曼莎·西村（Samantha Nishimura）**（挚友）｜性格：开朗、纪录片导演气质｜装备·能力：非战斗向｜人物弧光：被选为卑弥呼容器→获救｜立场关系：劳拉的人性锚。

**【贵重物品】**  
- **劳拉的冰斧与制作弓**：求生符号级装备。  
- **卑弥呼遗骸 / 登神仪式场**：风暴之源。

**【隐藏剧情 · 伏笔】**  
- 卑弥呼前任继承者自尽导致灵魂滞留，是风暴永续的真相。  
- Trinity 对克罗夫特家族的长期监视，串联三部曲。  
- 「劳拉是否与古墓强盗无异」的道德质询在暗影达到顶点。  
- 系列多次重启，乐园需声明当前时间线，避免经典双枪劳拉与 2013 恐枪劳拉混 co 性格。

**【大事记时间线】**  
- 古典神话时代 → 邪马台 / Atlantis / 玛雅仪式埋下祸根。  
- 二战等 → 多国军队曾困于邪马台。  
- Endurance 远征 → 劳拉诞生为幸存者。  
- 西伯利亚 → 对抗 Trinity 与不朽。  
- 中美洲 → 清洗危机与三部曲收束。  
- 经典平行 → Natla 与多地神器争夺。

**【叙事基调 · 雷区】**  
基调：从恐怖求生到史诗探险；强调劳拉的呼吸、受伤与成长。忌开局就写成冷酷双枪女神（幸存者线）；忌无代价使用神器；忌性化消费替代角色弧光；忌把超自然当日常法术 intermediate。最早锚点＝Endurance 船难后数小时。


**【世界剧情线 · 幸存者三部曲细述】**

**2013 开篇节奏**：序章风暴撕开 Endurance，劳拉被吊在洞穴中，用尽气力拔出腹中金属——「出生」仪式的暴力隐喻。她在荒野第一次猎杀、第一次用弓、第一次对活人扣扳机（Solarii 袭击者）。罗斯以残酷训练逼她活下去；萨曼莎的被俘把个人求生升级为拯救。太阳神兄弟会的社会结构是「失事男性」的极端宗教化：用卑弥呼神话赋予屠杀意义。惠特曼的背叛揭示知识精英的懦弱。直升机二次坠毁与罗斯之死，抽掉劳拉的父权保护伞。修道院层揭示登神是灵魂侵占，Sam 的「火焰无伤」是选定标记。终局劳拉双持开火的镜头，是角色从猎物到猎人的视觉加冕。

**崛起细述**：劳拉继承父亲对「不朽」的执念，却要证明父亲不是疯子。Trinity 以十字与现代军工结合，是全球化的神器殖民者。西伯利亚的圣城与先知传说，把「永生」写成文明病毒。劳拉开始主动暗杀与渗透，道德线从「自卫」移到「先发制人」，为暗影的灾变愧疚埋笔。

**暗影细述**：劳拉为阻止 Trinity 抢先触碰神器，反而触发海啸，平民死难成为她背负的「原罪」。库瓦科亚特尔与玛雅日历式的世界重生叙事，把个人冒险抬到生态—文明层级。约拿的质问（你与他们有何不同）是三部曲最重要的台词级冲突。终局选择强调封印与责任，而非再夺新神器。

**经典层可调用名场面**：初代 Atlantis 双枪、中国长城摩托、埃及金字塔解谜、Natla 的背叛；传奇三部曲的亚瑟王剑与北欧地下世界。电影朱莉版强调时光与父女，薇坎德版更贴近 2013 求生。

**【主要人物 · 补】**
- **安古斯·「格里姆」·格里马尔迪**：舵手，粗口与忠诚，要塞突击中牺牲。
- **乔斯琳·雷耶斯**：单亲机械师，要船要活路，实用主义。
- **阿列克斯·韦斯**：电子专家，Endurance 残骸中牺牲挡枪。
- **西村家族**：提供远征资金与文化线索，Sam 是血脉钥匙。
- **三位一体指挥官群（崛起/暗影）**：可替换的组织面孔，代表系统恶。
- **约拿在暗影的濒死与归来**：友情线压力测试。

**【地理补密】**
邪马台：海岸难民营式废墟、山地无线电站、燃烧的村寨、宫城刑场、日女主殿。西伯利亚：苏联基地、地热谷、冰湖、圣城穹顶。中美洲：热带城市、油田、地下河、天文台神庙。克罗夫特庄园地下墓穴与展览厅是记忆副本。

**【力量体系补】**
「求生本能」是叙事化直觉，非开挂透视外挂；制作系统把世界资源翻译成箭与炸弹。QTE 坠落与动物搏斗强调「劳拉会死」。神器副作用优先于神器攻击力：风暴、永生、清洗都是环境规则。

**【贵重物品补】**
- Endurance 的黑匣子与船员名册。
- 卑弥呼的葬礼面具与仪式锁。
- 父亲的怀表与未寄出的信。
- Trinity 的十字徽章与黑站点硬盘。
- 暗影的银盒与献祭匕首。
- 经典 Scion 三片。

**【隐藏与互文】**
系列重启两次，性格不能串：经典劳拉幽默自信；传奇劳拉超级英雄；幸存者劳拉创伤成长。吉尼斯级文化符号不代表战力超阶。重制与合集不改变因果，只改变画质。

**【世界状态机】**
船难后 24h：分散求生。Sam 被俘：全员目标转向要塞。卑弥呼死亡：风暴停，可撤离。Trinity 介入后：全球追捕状态。银盒触发后：沿海灾难新闻成为背景 UI。


**【古墓丽影·2013 章节功能再述】**
海岸求生章教系统：制作、狩猎、升级。山地无线电站章教「希望的欺骗」——飞机来了又被风暴打下来。宫城章教潜入与大规模枪战。修道院章教超自然真相。尾声海港救援教「活着离开仍要继续走」。
每个死亡的船员都对应劳拉失去的一种外部支撑：舵手、导师、技术宅、理想主义者。活下的约拿与雷耶斯、Sam 成为新共同体。


**【古墓丽影·崛起与暗影功能再述】**
崛起把敌人从「岛上邪教」升级为「全球公司化神秘学军队」。神器争论从「该不该碰」变成「谁配保管」。暗影把劳拉放上审判席：你的好奇心是否与反派同源。三部曲结尾她没有成为圣人，成为更清醒的介入者。

## 阶位切入点

> 阶位↔等级：一阶≈落难者/低阶邪教徒；二阶≈雇佣兵精锐/成长中劳拉/罗斯级；三阶≈遗迹守卫精英/神器敌首/成熟劳拉巅峰。顶点威胁（完整清洗/永续风暴）为存在级，条件性胜利。

**一阶（船难求生 · 邪马台海岸线）**  
切入身份/时点：契约者以「Endurance 临时船员 / 纪录片组助理 / 冲上岛的遇难者」切入，锚定船裂后、劳拉刚逃出洞穴前后。  
初始事件：你在潮汐洞穴里找到半包医用酒精和一根锈铁管——外面有人用日语和英语混骂着拖走「活祭品」；远处传来劳拉第一次拉动锈蚀弓弦的喘息。你要在「跟上去」「躲进神社废墟」或「点火求救（可能引来 Solarii）」间选择。  
开场白建议：「龙三角的雨横着刮。船名牌 Endurance 的残片卡在礁石上，像一块墓碑。你听到女声压抑的尖叫被风撕碎——这座岛不要救援，只要新鲜的名字。」  
关键NPC立场：**劳拉·克罗夫特**——警惕但会救同类；**萨曼莎·西村**——善意、易被绑架；**康拉德·罗斯**——评估你是否拖后腿；**马蒂亚斯**——把外人当燃料；**惠特曼**——会卖你换命。  
主线钩子/支线：信号火与无线电塔；支线＝收集草药制作、潜行偷武器、挑战墓外围。  
危险度/规避：中高——规避 Oni 巡逻核心与要塞刑场。  
任务方向/奖励：活过第一夜、一把能用的弓；奖励＝营地技能点、地图残片、Solarii 腰牌。

**二阶（要塞与联邦外雇 · 太阳神战争线）**  
切入身份/时点：以「被困多年的老幸存者 / 私营军事合同工 / 西村家族派来的搜救顾问」切入，锚定劳拉强攻太阳神要塞救 Sam 前后。  
初始事件：马蒂亚斯宣布登神仪式，Sam 被戴上仪式饰品；你被塞进奴隶劳工队运火药，必须决定是炸开囚笼、暗杀祭司，还是把情报卖给惠特曼换直升机位。  
开场白建议：「要塞里的鼓点像心跳。太阳面具后的眼睛不看人，只看‘容器’。你的工牌号被用刀刻在木牌上——在这里，编号比名字长寿。」  
关键NPC立场：**劳拉**——已会杀人，可组队；**雷耶斯**——要船要效率；**约拿**——保护平民优先；**Solarii 祭司**——狂热；**罗斯**——愿意为劳拉挡子弹。  
主线钩子/支线：阻止直升机二次坠毁式悲剧；支线＝解救船员、破坏防空炮。  
危险度/规避：高——规避马蒂亚斯本体与完整仪式圈。  
任务方向/奖励：救人质、截获仪式文献；奖励＝强化枪械、绳索枪、邪马台史料。

**三阶（神器末日 · Trinity / 清洗线）**  
切入身份/时点：以「独立神器猎人 / Trinity 叛逃研究员 / 劳拉的临时搭档」切入，锚定崛起后期或暗影清洗仪式前夜。  
初始事件：银盒或神圣之源的能量读数让盖革计数器失灵；多明格斯或 Trinity 指挥官要求你「协助完成人类的下一纪元」，而对讲机里劳拉的声音在说：若仪式完成，沿海城市会先被浪吞掉。  
开场白建议：「丛林在夜里发光，不是萤火虫，是碑文。你意识到神话不是故事，是一套尚未冷却的武器系统——而有人已经按下了倒计时。」  
关键NPC立场：**劳拉·克罗夫特**——背负灾变愧疚，极锋利；**约拿**——劝她别自我毁灭；**Trinity 高层**——目标明确可谈判可背刺；**当地伊希斯/玛雅后裔角色**——要保护圣地。  
主线钩子/支线：中断清洗、摧毁或封印神器；支线＝抢救平民、夺取 Trinity 黑站点。  
危险度/规避：极高——贴近存在级；严禁硬抗完整天灾，走仪式破坏与刺杀。  
任务方向/奖励：世界级神器样本（封印态）、Trinity 罪证、与劳拉的长期协作契约。


**一阶补强**：潮间带的 Solarii 巡逻二人组；无线电站下的饿狼；洞穴中的仪式涂鸦翻译任务。
**二阶补强**：要塞厨房下毒；防空炮校准破坏；惠特曼谈判桌的窃听。
**三阶补强**：Trinity 黑站点双线潜入；清洗倒计时中的平民疏散；与劳拉争抢「谁来按封印」的伦理对峙。
**开场白（三阶）**：「盖革计数器在神庙门口罢工。劳拉的耳机里只有杂音和一句：『若我失控，杀了我。』你知道这句话不能当真，也不能不当真。」
