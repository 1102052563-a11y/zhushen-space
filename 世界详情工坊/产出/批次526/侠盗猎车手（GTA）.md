# 侠盗猎车手（GTA）
<!--meta lib=主库 tiers=一、二-->

## 剧情

**【作品来源】**  
《侠盗猎车手》（英语：*Grand Theft Auto*，简称 **GTA**）是由苏格兰爱丁堡 **Rockstar North**（前身为 DMA Design）主开发、**Rockstar Games** 发行的开放世界动作冒险游戏系列。系列由大卫·琼斯（David Jones）与迈克·戴利（Mike Dailly）开创，后由丹·豪瑟（Dan Houser）、萨姆·豪瑟（Sam Houser）与扎卡里·克拉克等核心团队塑形。自 1997 年俯视角初代起，历经 2D 世界、3D 世界（GTA III / 罪恶都市 / 圣安地列斯）与 HD 世界（GTA IV / V 及线上模式），至 2013 年《侠盗猎车手 V》与持续运营的 *GTA Online*，全系列累计销量逾 2.6 亿套。文风：黑色幽默讽刺美国都市文化、犯罪升迁叙事、成人暴力与媒体政治嘲弄并存。本档案以 **3D 世界至 HD 世界主线** 为底本，**以 GTA V 三主角线为乐园默认连续主线**，圣安地列斯 / 罪恶都市 / 自由城各作为可切换「年代档案副本」。

**【世界定位】**  
架空美利坚都市犯罪宇宙：主角们皆是大城市底层或中层罪犯，在帮派、黑手党、联邦机构与媒体资本的夹缝中完成「升职、背叛、复仇、大劫案」。表层规则是枪支、载具、通缉星级与现金；暗层是腐败警察、FIB/IAA 互咬、军火商与名人文化的共生。世界顶点不是超级英雄，而是**能调动私军、军火与情报的联邦代理人 / 毒枭 / 亿万富翁**——仍是可被枪弹杀死的人类。

**【世界观 · 力量体系】**  
本世界**无超凡修炼与魔法**。战力锚定在：**枪械 · 载具 · 体能 · 犯罪组织资源 · 联邦/私军火力 · 通缉系统**。死亡规则高度写实——一枪可致命；「复活」仅见于医院重生等游戏机制，非世界内复活术。

**1. 街头与帮派层**  
- **平民 / 店员 / 路人**：无战力，子弹与车祸即死。  
- **街头混混 / 低阶帮派成员（Grove Street 外围、巴勒帮小卒、三合会打手）**：手枪、近战、街头伏击。  
- **核心帮派干部（斯威特·约翰逊级、拉玛·戴维斯级）**：熟悉地盘、可组织巷战与驾车射击。  
- **职业劫匪 / 老兵型罪犯（麦克·迪圣塔、崔佛·菲利普、卡尔·约翰逊中后期）**：战术规划、多武器切换、载具追逐与反制。

**2. 执法与联邦层**  
- **洛圣都警局 / 自由城警方（LSPD 等）**：一至五星通缉；星级升高则增派载具、直升机与特种火力。  
- **FIB（联邦调查局仿写）探员**：戴夫·诺顿、史蒂夫·海因斯——以把柄胁迫罪犯执行黑任务。  
- **IAA（国际事务局仿写）**：与 FIB 对立的情报机构，掌握资金车队与实验室。  
- **梅利威瑟安保（Merryweather）**：私人军事公司，武装直升机、特种步兵，可被亿万富翁雇佣。

**3. 载具与特殊能力（非肉体跃迁）**  
- 汽车 / 摩托 / 直升机 / 固定翼飞机 / 潜艇等构成机动优势。  
- GTA V 三主角「特殊技能」：麦克子弹时间、富兰克林驾驶减速、崔佛狂暴——属叙事化人类极限表现，**不抬高至超凡阶**。  
- 莱斯特·克瑞斯特的黑客网络、股票操纵、劫案策划 = 信息战优势。

**乐园阶位映射（对照破坏力标尺，宁低勿高）**  
- 平民、店员、低星警察、街头混混 ≈ **一阶**（徒手/手枪级，子弹致命）。  
- 精锐帮派、职业劫匪、FIB/IAA 一线、梅利威瑟士兵、麦克 / 富兰克林 / 崔佛、CJ 后期、汤米·维尔塞蒂、尼克·贝里克 ≈ **二阶**（可单挑小队、公路枪战、近距规避第一波火力；肉体仍惧重火力与爆炸）。  
- 五星通缉全城围剿、梅利威瑟武装空中力量、FIB/IAA 机构级行动 = **组织/装备优势**，不单独把个体抬到三阶。  
- 化学武器失窃、核级设施、跨州军工阴谋 ≈ **存在级灾难威胁**（对契约者：存在·情报优先 / 条件性胜利——截获、销毁、改写投放，而非与装置肉搏）。  
**世界个人战力顶点 ≈ 二阶巅峰**（崔佛狂暴态、麦克战术射击、德凡·维斯顿 / 陈伟调度下的私军头目）。严禁把主角写成硬抗导弹的超人。

**【地理 · 舞台】**  
GTA 宇宙按 Rockstar 划分为**相对独立的 2D / 3D / HD 世界**，共享品牌与地名灵感，人物不完全互通。乐园任务默认锚定 **HD 世界 · 圣安地列斯州（GTA V）**，辅以 3D 世界可刷副本。

- **洛圣都（Los Santos）**：以洛杉矶为原型；好莱坞山、威斯普奇海滩、南洛圣都贫民区、岩地城、市中心摩天楼。麦克豪宅、富兰克林姨妈家、崔佛拖车营地构成三主角据点。  
- **布莱恩郡 / 大塞诺拉沙漠 / 佩立托湾**：乡村冰毒、摩托帮、银行与化工设施；崔佛「崔佛·菲利普企业」地盘。  
- **洛圣都国际机场 / 诺斯扬克顿（北扬克顿）**：序章鲁登道夫劫案舞台；终局回溯墓地真相。  
- **自由城（Liberty City，IV / III）**：以纽约为原型；经纪人区、阿尔冈琴、博汉、杜克斯；尼克·贝里克移民线。  
- **罪恶都市（Vice City）**：以 1980 年代迈阿密为原型；可卡因经济、游艇与电台文化；汤米·维尔塞蒂崛起。  
- **圣安地列斯州（SA，3D）**：洛圣都 / 圣辉洛 / 拉斯云组华三城 + 乡村；CJ 帮派统一线。  
- **功能节点**：弹药店、服装店、修车厂、酒吧、高尔夫球场、电影工作室、FIB 大楼、IAA 设施、梅利威瑟码头——构成「任务世界可刷地图」。

**【世界剧情线】**  

**① 3D 世界前史 · 罪恶都市崛起（1980s）**  
汤米·维尔塞蒂（Tommy Vercetti）自自由城出狱后被桑尼·福雷利派往罪恶都市发展毒品生意，却在交易中遭伏击失去货与钱。他逐步清除古巴帮、海地帮与上校科特斯线中的障碍，与肯·罗森伯格、兰斯·万斯等合作，最终吞并地产业与电台，杀死桑尼，成为罪恶都市地下之王。此线奠定系列「外来者吞城」语法。

**② 圣安地列斯 · 1992 帮派战争**  
卡尔·约翰逊（Carl "CJ" Johnson）因母亲被杀从自由城返回洛圣都格罗夫街。兄长斯威特仍固守 Grove Street Families，而大烟哥（Big Smoke）与莱德已暗中投靠腐败警察坦佩尼（Frank Tenpenny）与巴勒帮。CJ 在洛杉矶式暴乱阴影下重振帮派、学习飞行与盗窃、深入圣辉洛与拉斯云组华，揭露坦佩尼的勒索网络，最终在暴乱中解决大烟哥，坦佩尼坠亡。主题：忠诚、背叛与「家园」是否值得用血赎回。

**③ 自由城 III / 故事 · 沉默的复仇**  
克劳德（Claude）在自由城被女友卡特琳娜背叛枪击坠桥，失声后辗转里奥内家族、八角形馆与哥伦比亚卡特尔线，最终在卡特琳娜的逃逸直升机下复仇。自由城故事（LCS）则以托尼·塞普拉诺式黑手党青年托尼·西普里亚尼补完 1998 年前传。

**④ HD 世界 · 尼克的美国梦破碎（2008）**  
尼克·贝里克（Niko Bellic）自东欧战火来到自由城投奔堂兄罗曼，却被拖入高利贷、黑帮与「美国梦」骗局。他为铃木家族、保加利亚犯罪网与联邦线人做事，最终在「复仇」与「放过」之间抉择是否杀死黑暗人迪米特里·拉斯卡洛夫的幕后——主题是移民、战争创伤与无法洗净的手。资料片《失落与诅咒》《夜生活之曲》分别从约翰尼·克莱比茨与路易斯·洛佩兹视角补完同一城的摩托帮与夜总会权力。

**⑤ 主线默认 · GTA V 三主角（约 2013）**  
**序章**：2004 年北扬克顿，麦克·汤利、崔佛·菲利普与布莱德·施奈德抢劫失败；麦克在 FIB 探员戴夫·诺顿安排下「死亡」隐居，化名麦克·迪圣塔；布莱德实则被杀埋于麦克假墓；崔佛逃亡。  
**九年后**：麦克在洛圣都过着空虚豪宅生活；富兰克林·克林顿为车行收车时结识麦克，二人摧毁马丁·玛德拉索相关豪宅后被迫重操劫案，珠宝店得手。崔佛从电视新闻认出手法，逼麦克重聚。  
**中段压力**：FIB 的史蒂夫·海因斯胁迫三人打击 IAA、抢佩立托湾银行、盗化学武器、潜入 FIB 删档；德凡·维斯顿与电影业线纠缠麦克；崔佛扩张布莱恩郡黑市，硬刚失落摩托帮、欧尼尔兄弟、陈伟三合会与梅利威瑟；富兰克林在拉玛与史崔奇的巴勒帮仇杀中求生。  
**联合储蓄大劫案**：莱斯特策划抢黄金；崔佛发现布莱德已死、自己被骗，团队濒临破裂。  
**终局三选一（乐园默认「结局 C · 死亡愿望」为 canonical 走向）**：富兰克林可杀崔佛、杀麦克，或联合两人反杀陈伟、史崔奇、史蒂夫·海因斯与德凡·维斯顿，三主角维持扭曲友谊。  
**线上层（GTA Online）**：玩家自创角色在洛圣都从事差事、资产、DLC 剧情（末日豪劫、佩里科岛、合约等），与故事模式时间线松散并行，可作「契约者自建身份」沙盒。

**⑥ 结局方向**  
故事模式以「死亡愿望」为准：联邦叛徒与亿万富翁清除，三人未成圣人，只是暂时不被清算。世界本身不因主角胜利变好——媒体、毒品、腐败警察与枪支文化仍在运转。GTA VI 预告指向新罪恶都市双主角，属后续可扩展档案，本世界以 V 完结为当前锚点。

**【主要人物】**  

- **麦克·迪圣塔 / 麦克·汤利（Michael De Santa / Michael Townley）**（退休劫匪·三主角之一）｜性格：愤世嫉俗、中年危机、表演型自恋又护短｜装备·能力：战术射击、特殊技能「子弹时间」、豪宅军火库｜人物弧光：假死隐居→被迫复出→与家人复合并在终局选择继续作恶或收手｜立场关系：与崔佛亦敌亦友；收富兰克林为「精神义子」；被戴夫·诺顿与史蒂夫·海因斯拿捏。  
- **富兰克林·克林顿（Franklin Clinton）**（街头青年·三主角之一）｜性格：务实冷静、想「出人头地」｜装备·能力：驾驶天赋（特殊技能减速）、多种枪械｜人物弧光：车行打手→劫案核心→终局抉择者｜立场关系：敬重麦克；忌惮又依赖崔佛；与拉玛·戴维斯兄弟情；厌恶史崔奇。  
- **崔佛·菲利普（Trevor Philips）**（反社会罪犯·三主角之一）｜性格：暴虐、忠诚扭曲、情绪极端｜装备·能力：狂暴特殊技能、飞行、近战凶残｜人物弧光：以为麦克已死而 Reckless 经营布莱恩郡→重逢后崩溃→终局可被杀或被救赎式接纳｜立场关系：恨麦克的「背叛」；对维恩·丹尼尔斯等有畸形保护欲；与陈伟死敌。  
- **莱斯特·克瑞斯特（Lester Crest）**（黑客·军师）｜性格：尖刻、体弱、智力碾压｜装备·能力：入侵、计划劫案、股票操纵｜人物弧光：从麦克旧友到三人本的大脑｜立场关系：收费服务，但关键时刻站队主角。  
- **拉玛·戴维斯（Lamar Davis）**（帮派朋友）｜性格：话多、冲动、讲义气｜装备·能力：街头枪战｜人物弧光：巴勒帮仇杀中多次被救｜立场关系：富兰克林挚友；讨厌史崔奇。  
- **戴夫·诺顿（Dave Norton）**（FIB 探员）｜性格：疲惫、机会主义｜装备·能力：联邦权限、狙击与任务调度｜人物弧光：用麦克换功劳→被海因斯当枪使→终局求生｜立场关系：既是麦克的「保护人」也是枷锁。  
- **史蒂夫·海因斯（Steve Haines）**（FIB 明星探员）｜性格：自恋、媒体狂、背信｜装备·能力：联邦资源、电视形象｜人物弧光：胁迫三人为其清除政敌→终局被杀（结局 C）｜立场关系：全员公敌。  
- **德凡·维斯顿（Devin Weston）**（亿万富翁）｜性格：笑面虎、资本冷酷｜装备·能力：梅利威瑟雇佣权、产业｜人物弧光：与麦克电影线结仇→终局被杀（结局 C）｜立场关系：资本层终局 BOSS 之一。  
- **马丁·玛德拉索（Martin Madrazo）**（毒枭）｜性格：暴躁、要面子｜装备·能力：私人武装、南美背景｜人物弧光：豪宅被毁后勒索麦克｜立场关系：可交易的危险雇主。  
- **陈伟（Wei Cheng）**（三合会首领）｜性格：冷酷、扩张主义｜装备·能力：杀手组织｜人物弧光：欲招揽/清除崔佛→终局被杀（结局 C）｜立场关系：崔佛线主要黑帮压力。  
- **汤米·维尔塞蒂（Tommy Vercetti）**（罪恶都市之王·档案层）｜性格：冷酷精明｜装备·能力：1980s 枪械与地产帝国｜人物弧光：福雷利家族马仔→都市之主｜立场关系：副本级主角。  
- **卡尔·约翰逊（CJ）**（圣安地列斯主角·档案层）｜性格：重情、能屈能伸｜装备·能力：全技能成长、飞行与帮派战争｜人物弧光：浪子归乡→重振 Grove｜立场关系：与斯威特、塞萨尔、吴梓等。  
- **尼克·贝里克（Niko Bellic）**（IV 主角·档案层）｜性格：厌战、讽刺、致命｜装备·能力：欧洲战场经验、多种武器｜人物弧光：移民梦碎→选择是否复仇｜立场关系：罗曼、小雅各布、德米特里。

**【势力图谱】**  
- **格罗夫街帮 / 巴勒帮 / 家族帮（SA）**：街头地盘与毒品；代表 CJ、斯威特、大烟哥。  
- **福雷利 / 莱昂内等黑手党（自由城 3D）**：传统有组织犯罪。  
- **FIB / IAA**：联邦互咬；海因斯 vs 机构利益。  
- **梅利威瑟安保**：资本的枪；可被德凡或任务方雇佣。  
- **失落摩托帮 / 三合会 / 欧尼尔冰毒家族**：布莱恩郡与港口黑市。  
- **崔佛·菲利普企业（TP Industries）**：崔佛的非法 co-op 品牌化犯罪。  
- **洛圣都警局与腐败系统**：通缉星级即世界规则本身。

**【贵重物品】**  
- **联合储蓄黄金**：终盘大劫案目标，四吨级现金/金块叙事。  
- **珠宝店货品 / 佩立托湾银行现金**：中期资金链。  
- **IAA 化学武器样本**：联邦黑任务核心，灾难级道具。  
- **莱斯特的服务器与删档权限**：信息即武器。  
- **改装载具（火箭跑车、武装直升机等，线上更丰）**：机动与火力平台。  
- **罪城时代可卡因货与地契、SA 的喷气机与赌场筹码**：各副本「起家资本」。  
- **麦克假墓与布莱德尸体真相**：情感与背叛的物证，非装备却是剧情至宝。

**【隐藏剧情 · 伏笔】**  
- 麦克「死亡」是 FIB 与戴夫的长期剧本；布莱德未入狱而死是崔佛崩溃的引信。  
- FIB 与 IAA 互相抹黑，罪犯只是棋子——系列一贯「体制比黑帮更脏」。  
- 埃普西隆邪教、异世界彩蛋、飞碟与「叙述性不可靠」彩蛋构成 Rockstar 式元叙事，可作支线怪谈，不抬高战力。  
- 3D 与 HD 世界人物不直接连续，但地名与品牌互文；契约者跨档案需乐园说明「副本隔离」。  
- Online 持续更新改写洛圣都经济，故事模式「结束后的城市」仍在犯罪。

**【大事记时间线】**  
- 1980s → 汤米称霸罪恶都市。  
- 1992 → CJ 圣安地列斯帮派战争与暴乱。  
- 1998–2001 → 自由城黑手党与克劳德线。  
- 2004 → 北扬克顿劫案；麦克假死。  
- 2008 → 尼克自由城梦碎。  
- 约 2013 → 三主角重聚、联邦黑任务、联合储蓄、终局清洗。  
- 2013 后 → GTA Online 持续；VI 预告新罪恶都市（扩展中）。

**【叙事基调 · 雷区】**  
基调：讽刺、黑色幽默、残酷现实主义、公路与枪火美学。忌把主角写成正义超级英雄；忌美化无后果屠杀（世界有通缉与同伴反应）；忌混淆 3D/HD 人物直接串台而不加副本说明；忌把崔佛写成无动机的纯搞笑角色（其核心是被背叛的创伤）。最早切入锚点＝洛圣都底层（车行、帮派、小劫案）或北扬克顿序章前后。


**【世界剧情线 · 分作细述与因果链】**

**罪恶都市细部因果**：汤米抵达时，桑尼要求他讨回被伏击的毒品与钱。真正设局者牵涉到福雷利家族内部与当地势力的双重出卖。汤米先依附上校科特斯获得军事级人脉，再借地产大亨 Avery 的任务学会「用子弹改城市规划」。肯·罗森伯格从吸毒律师变成恐惧的合伙人；兰斯·万斯从盟友变为嫉妒的叛徒。电台、电影工作室与印刷厂等产业线不仅是玩法，更是汤米把「暴力」翻译成「合法控股」的过程。终局游艇上桑尼的出现，是自由城逻辑对罪恶都市新王的清算——汤米选择杀回马枪，完成系列最早的「外来者称王」。

**圣安地列斯细部因果**：CJ 下飞机即被坦佩尼勒索，确立「警察即黑帮」基调。格罗夫街因毒品分裂：斯威特要纯血统帮派荣耀，大烟哥要利润与坦佩尼的保护伞。塞萨尔·维尔潘多带来圣辉洛的联盟可能；吴梓、真偏执狂等支线把 CJ 推成多面手。「绿色军刀」任务揭露大烟哥与莱德的背叛，是情感核爆炸点。乡村逃亡段教会飞行与盗窃国家级别资源；返回洛圣都时暴乱已起，坦佩尼的直升机坠落不是英雄凯旋，而是腐败自噬。大烟哥之死发生在毒品宫殿，CJ 在兄弟情与处决之间完成成人礼。

**GTA IV 细部因果**：尼克为罗曼还债而卷入弗拉迪、米哈伊尔、保加利亚人与意大利家族的多层盘剥。小雅各布提供武器与良心旁白；帕克提供美式犬儒。暗影人物「黑暗人」许诺战争真相，实则是联邦与犯罪的共同谎言。终局二选一（杀死德米特里或杀死黑暗人相关路线的经典分歧）决定罗曼/凯特的生死，把「美国梦」写成零和。

**GTA V 任务链因果加密**：珠宝店劫案的电视转播成为崔佛的认亲信物；玛德拉索债是被迫复出的经济引擎；佩立托湾与屠宰场等任务把「兄弟团」重新焊上。FIB 与 IAA 的实验室、资金车、总部删档，是国家对罪犯的外包。崔佛挖开麦克假墓看到布莱德骨骸时，叙事从犯罪喜剧跌入创伤悲剧。终局 C 要求同时处理梅利威瑟攻势与四名点名处决，是沙盒对「友谊」的终极压力测试。

**线上与主线的关系**：Online 角色不认识三位主角的「电影结局」，但共享洛圣都地理与部分 NPC（莱斯特等）。契约者若以线上身份切入，应视为平行雇工时间线，避免直接改写单人终局，除非乐园任务声明「蝴蝶权限」。

**【主要人物 · 补遗与关系网】**
- **阿曼达·迪圣塔 / 崔西 / 吉米**：麦克的家庭战场；网球教练事件是珠宝店前的导火索；家庭治疗支线提供非枪战的角色厚度。
- **维恩·丹尼尔斯与崔佛的拖车社区**：展示崔佛的畸形保护欲与布莱恩郡生态。
- **佛莱迪·内特斯 / 乔许·伯恩斯坦等支线真名**：构成洛圣都名人讽刺群像，可作情报与幽默支线，不可替代主线因果。
- **坦佩尼、普拉斯基、赫南德兹（SA）**：腐败警察三角，是 3D 世界「体制反派」模板。
- **卡特琳娜·达·内（III）**：克劳德的背叛者，自由城复仇符号。
- **罗曼·贝里克、凯特·麦克雷里、小雅各布**：IV 的情感三角与道德参照。

**【地理 · 舞台补密】**
洛圣都可按剧情功能分区：摇滚邦的娱乐伪善、南洛的帮派实录、晨光村的富人偏执、塔通基的乡村暴力、帕莱托的港口走私。安全屋（高楼公寓、中产别墅、山区小屋）是角色阶级的建筑学表达。枪械店、衣服店、理发店与纹身店把「暴力」包装成消费——这是 GTA 讽刺美国的核心空间语法。

**【贵重物品 · 补】**
- 北扬克顿劫案的原装金条与运钞车：序章失败的圣杯。
- 麦克的精神医生与「治疗」账单：中年危机道具化。
- 崔佛的破斧与血腥围裙：恐怖符号。
- 富兰克林的白色座驾与恰普曼：阶级跃迁的轮子。
- 莱斯特的哮喘与纤维病躯体：智力超模的肉身代价。

**【隐藏剧情 · 补】**
- 「我才是主角」式元梗与电影《惊魂记》式彩蛋服务玩家文化，不改变因果。
- 德凡与梅利威瑟的合同关系证明：当代战争可被私人订购。
- 史崔奇与富兰克林的童年线是南洛圣都循环暴力的缩影。

**【叙事执行细部 · 给 AI 的世界状态机】**
任意时点判断：若珠宝店未发生，则崔佛未进城；若假墓未挖开，则三人仍可同桌；若结局 C 已完成，则海因斯与德凡已死、洛圣都进入「三巨头冷和平」。通缉星级是世界规则 UI：一星喝止，三星直升机，五星坦克级压力（各作表现略有差异）。现金与银行账户分离（V 起），抢劫会影响股市——莱斯特线把犯罪写成金融。


**【侠盗猎车手（GTA）·GTA III 至 SA 因果再述】**
克劳德在卡特琳娜开枪后坠桥失声，这个沉默不是人设萌点，而是把玩家变成「城市的刀」。他先后为里奥内家族、八角形馆姚氏、哥伦比亚卡特尔与西班牙领主做事，每一单都把自由城的族裔黑帮版图撕开一道口。最终在大坝直升机下完成对卡特琳娜的复仇，自由城并不感谢他。
自由城故事的托尼·西普里亚尼在1998年替父亲一代黑手党收拾残局，面对地产开发、毒品新贵与家族背叛，主题是「旧黑帮被新资本吞噬」。罪恶城故事的维克·万斯则在1984年为弟弟兰斯铺路，展示万斯兄弟悲剧的前史。
圣安地列斯中，CJ 的技能成长（枪、车、飞、肺）是身体资本主义：你练什么，城市就允许你抢什么。斯威特的监狱段落逼 CJ 独自建立跨城网络；真理者的迷幻支线把美国阴谋论写成喜剧；吴梓的五件套任务把「忠诚」写成互雇。终局暴乱不是正义胜利，是城市高烧。


**【侠盗猎车手（GTA）·GTA IV 与资料片再述】**
尼克的每一份工作都在问：战争技能在和平城市值多少钱？答案是「刚好够被所有人利用」。罗曼的结婚线提供温暖，也提供人质。小雅各布的牙买加帮伦理与贝尔的美式犬儒对照。黑暗人许诺的「名单」是情报拜物教。
《失落与诅咒》约翰尼·克莱比茨被自己的摩托帮与上家双重出卖，结局惨烈，并在 V 中以支线尸体形式出现，形成跨作残酷彩蛋。《夜生活之曲》路易斯在夜总会电梯与直升机之间做道德选择，展示「保镖也是刽子手」。


**【侠盗猎车手（GTA）·GTA V 任务段再述：从珠宝店到屠宰场】**
珠宝店方案可选潜行钻洞或强攻，选择改变警力与分红，是系列「玩法即叙事」的范例。玛德拉索债清后短暂喘息，崔佛入城把喜剧变成炸弹。托尼娅的拖车、弗洛伊德的懦弱、维恩的毒，构成布莱恩郡伦理泥潭。
三部曲式的「杀人练习」：猎杀巴勒、突袭欧尼尔农场、机场对梅利威瑟，逐步把三人焊成一套武器。莱斯特的杀人名单与末日预备役笑话，提前预告终局点名处决结构。
屠宰场与 FIB 总部双线任务要求玩家在角色间切换射击，是机械层面对「三人一体」主题的翻译。当崔佛挖开墓穴，玩家失去「笑着开枪」的权利。


**【侠盗猎车手（GTA）·Online 与都市生态】**
线上豪劫（监狱、人道、末日、赌场、佩里科、合约）把单人电影拆成可重复劳动。莱斯特、富兰克林（合约更新后）等NPC成为跨模式雇主。乐园若引入线上身份，应用「平行雇员时间线」，避免直接杀死故事模式三主角，除非任务标明高阶蝴蝶权限。
都市生态还包括：街头随机事件（劫案、情侣争吵、被追的妓女与皮条客等）、动物与荒野彩蛋、飞行学校与射击场训练。它们不是主线，却是洛圣都「活着」的证据。

## 阶位切入点

> 阶位↔战力：一阶≈平民/混混/低星执法与街头枪火；二阶≈职业劫匪/精锐帮派/FIB·IAA 一线/梅利威瑟士兵/三主角本级。世界个人顶点（崔佛·麦克·富兰克林及联邦顶级代理人）为二阶巅峰。低阶切入一律「规避顶点·情报优先」；机构级化学武器与全城五星围剿按存在威胁处理，条件性胜利。

**一阶（街头求生 · 洛圣都底层线）**  
切入身份/时点：契约者以「刚到洛圣都的外来工 / 车行杂工 / 帮派外围跑腿」身份切入，锚定富兰克林仍在西蒙车行收车、麦克尚未与其深度绑定前后。  
初始事件：一次普通的扣车任务变成枪战——车主是帮派关系户，巴勒帮或家族帮有人拦街；契约者必须在「把车交回老板」「私吞零件跑路」或「打电话求救」之间抉择，警笛已在两个街区外。  
开场白建议：「南洛圣都的柏油路被午后的阳光晒得发白。你刚学会在导航里标出威斯普奇的弯道，对讲机里却响起枪栓声——扣车单上的地址，原来写在别人的地盘上。」  
关键NPC立场：**富兰克林·克林顿**——务实，可合伙但讨厌拖后腿；**拉玛·戴维斯**——话多义气，容易把你拖进仇杀；**麦克·迪圣塔**——若早期接触，会以「人生导师」口吻试探你是否够狠；**西蒙**（车行老板）——剥削型雇主；**史崔奇**——笑面虎，出卖朋友的隐患。  
主线钩子/支线：扣车链升级为珠宝店外围眼线；支线＝街头竞速赌金、当出租车/救护车支线、低星通缉逃脱教程、电台与酒吧情报。  
危险度/规避：中——主要死于帮派火并与二星以上警察；须规避崔佛的无差别攻击、梅利威瑟据点与 FIB 大楼深层。  
任务方向/奖励：站稳脚跟、第一笔脏钱与一支可靠手枪；奖励取向＝载具钥匙、安全屋床位、低阶人脉（拉玛/车行）。

**二阶（大劫案与联邦棋盘 · 三主角线）**  
切入身份/时点：以「莱斯特认可的技术员 / 专业司机 / 前兵痞」身份切入，锚定珠宝店得手后、佩立托湾或联合储蓄筹备期。  
初始事件：莱斯特的电话把你召到某处废弃厂房——桌上是联合储蓄或 FIB 任务的蓝图；麦克要求纪律，崔佛要求「有趣」，富兰克林看你是否会在枪声里保队友；门外 Mel 里威瑟或 FIB 的车已经闪过一次。  
开场白建议：「莱斯特的氧气瓶嘶嘶作响，投影仪把金库结构打在墙上。麦克说按计划来；崔佛笑着问你‘死过没有’。你意识到：这不是电影，是洛圣都最贵的自杀方式——如果成功，也是最肥的一次。」  
关键NPC立场：**莱斯特·克瑞斯特**——按能力付费，背叛则封杀；**戴夫·诺顿**——给活路也给项圈；**史蒂夫·海因斯**——利用完即弃；**德凡·维斯顿**——资本微笑，背后梅利威瑟；**崔佛·菲利普**——忠诚极其私人，被骗则屠城；**马丁·玛德拉索**——债主型 BOSS。  
主线钩子/支线：联邦删档、化学武器、三结局分支（劝富兰克林选 C）；支线＝股票内幕（莱斯特）、电影制片线、崔佛与陈伟战争调停或拱火。  
危险度/规避：高——须规避同时激怒 FIB+梅利威瑟+三合会；终局点名目标（海因斯/德凡/陈伟）需团队协作。  
任务方向/奖励：完成一次「传奇级」劫案并活着分账；奖励取向＝高阶武器库、改装载具、莱斯特级情报网、三主角友谊或仇笺。


**一阶补强事件池（勿与二阶重复）**
- 威斯普奇海滩的毒品丢包：捡到即被两帮追杀，可交给拉玛换人情或交给警察换「不可靠保护」。
- 出租车司机目击杀人：是否做伪证，决定能否获得地下赛车入场券。
- 车行扣押的「关系车」其实是玛德拉索外围马仔的车，扣下会引来警告电话。

**二阶补强事件池**
- 联合储蓄筹备夜：莱斯特要求你在 FIB 监控盲区测试钻孔，崔佛却把实验变成烟花。
- 死亡愿望前夜：富兰克林的屋里两部手机同时震动——海因斯与德凡的报价单，契约者可影响他听哪一边。
- 陈伟绑架麦克时的码头枪战：必须在崔佛怒火与救援窗口之间同步。

**开场白备用（一阶）**：「加油站的收音机在播名人丑闻，你的口袋里只有一颗打瘪的子弹壳——刚才那一枪不是你开的，但血溅到了你的鞋。洛圣都的规则是：在场，就有罪。」

**开场白备用（二阶）**：「莱斯特把三维金库模型转到最后一层，说『这里会死四个人，名字待填』。麦克看你，崔佛看门，富兰克林看手机。你意识到自己的名字也可能在那四个空格里。」
