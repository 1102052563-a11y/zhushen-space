# 战地
<!--meta lib=主库 tiers=一、二-->

## 剧情

**【作品来源】**  
《战地》（英语：*Battlefield*，台译戰地風雲）是 **EA DICE**（Digital Illusions CE）开发、**艺电（EA）** 发行的第一人称射击游戏系列。2002 年《战地 1942》以二战陆海空载具大战奠基，后续涵盖越战、现代战争、未来战、一战、二战回归与近未来等。以**大规模地图、载具、破坏地形、小队协作**著称，与《使命召唤》的巷战节奏形成分野。文风：战争电影感、多阵营、部分作品含单人战役剧情。本档案以 **系列共性 + BF3/BF4 现代战役与 BF1/V 历史战役为叙事锚**，多人模式为「永恒沙盒战场」。

**【世界定位】**  
可切换的近现代—未来战争地球：玩家是士兵、特种或「无名炮灰」，在征服、突破、行动等模式中夺取旗帜与目标。单人战役提供具名主角与政治阴谋；多人无永久个人神话，只有军衔、载具与队友情。世界顶点是**航母、核弹级战略与气候灾难下的超国家战争**——对个体仍是「一发子弹」的真实。

**【世界观 · 力量体系】**  
完全军事写实向（科幻作有外骨骼与未来载具，仍非魔法）。

**1. 兵种**  
- 突击 / 工程 / 支援 / 侦察（各作名称微调）：步枪、炸药、弹药箱、狙击与无人机。  
- 驾驶员 / 飞行员：坦克、舰船、直升机、固定翼。  
- 指挥官（经典 BF2）：战略资源与扫描。

**2. 载具与破坏**  
- 陆海空全载具连锁；寒霜引擎建筑可坍塌（Bad Company 起强化）。  
- 乐高式摧毁改变掩体与战线。

**3. 战役剧情层**  
- BF3：美军与俄/中东线，迪马·马亚科夫等。  
- BF4：中国将领金杰与「中国崛起」背景的政治阴谋，Recker 等。  
- BF1：一战多视角「战争故事」。  
- BFV：二战「战争故事」与火线风暴。  
- 2042：气候难民「无国者」与专家系统。  

**乐园阶位映射（宁低勿高）**  
- 新兵、后勤、无载具步兵 ≈ **一阶**。  
- 老兵、特种、王牌驾驶员、战役主角作战表现 ≈ **二阶**（可清剿小队、载具击杀、据点攻防；肉体惧重火与爆炸）。  
- 航母打击、战术核弹、气候武器级事件 ≈ **存在级战场威胁**（条件性胜利：撤离、拆弹、夺控，非肉身抗轰）。  
**世界个人战力顶点 ≈ 二阶**。严禁写成一人灭旅而无载具/小队。

**【地理 · 舞台】**  
- 二战欧陆与太平洋（1942/V/1943）。  
- 越南丛林（Vietnam / BC2 资料片）。  
- 现代中东、德黑兰、巴黎、上海、新加坡、苏伊士等（BF3/4）。  
- 一战西欧、阿尔卑斯、阿拉伯、东线（BF1）。  
- 2042 全球灾害地带与传送门经典图。  
- 功能：部署点、旗帜、M-COM、医疗站、载具刷新点。

**【世界剧情线】**  

**① 系列语法起源 · 1942**  
盟军 vs 轴心，无深角色，只有战场与载具——「你是战争机器的一颗螺丝」。

**② 现代神话 · BF3**  
单人跟拍布莱克burn 等，涉及伊朗、巴黎恐袭、迪马反叛与真相；主题是「谁定义恐怖分子」。多人则是美俄全面战争幻想。

**③ 政治惊悚 · BF4**  
「墓碑」小队在中国上海开场，卷入金杰上将的政变与美中俄三角。战役短促，强调大场面倒楼与三阵营。多人「中国崛起」等 DLC 扩展亚洲战场。

**④ 历史人文 · BF1 / V**  
BF1 以多个「战争故事」展现一战残酷与人性；BFV 试图讲述非典型二战视角，含火线风暴模式。重点从「爽」转向「代价」。

**⑤ 近未来 · 2042**  
气候崩坏，美俄代理冲突，玩家选「专家」英雄单位；评价两极，但提供灾害动态天气战场。

**⑥ 叛逆连队支线**  
Bad Company 1/2：B 连寻金与「暗箭」武器，更电影小品。

**⑦ 结局方向**  
战役各有短暂胜利；多人无结局——战争永不结束，只有下一张地图。乐园可把「当前服务器」当作永恒现在时。

**【主要人物】**  

- **迪马·马亚科夫（Dima Mayakovsky）**（BF3）｜性格：犬儒、致命｜装备·能力：特战｜人物弧光：被利用的「恐怖分子」真相｜立场关系：战役道德灰心。  
- **亨利·布莱克本（Henry Blackburn）**（BF3）｜性格：服从到质疑｜装备·能力：美军中士｜人物弧光：审讯室叙述者。  
- **雷克（Daniel Recker）**（BF4）｜性格：沉默执行者｜装备·能力：墓碑小队｜人物弧光：选择中国未来走向的扳机。  
- **金杰（Huang "Hannah" Jin / 相关中方线与金杰上将剧情）**｜性格：爱国与权斗｜装备·能力：政治与军队｜人物弧光：BF4 政治核。  
- **爱尔兰 / 帕克等墓碑队员（Irish, Pac）**｜性格：忠诚吐槽｜装备·能力：战斗支援｜人物弧光：小队羁绊。  
- **BF1 各战争故事主角（克莱、佐拉等具名士兵）**｜性格：多样｜装备·能力：时代武器｜人物弧光：一战切片。  
- **Bad Company：科尔·马奎尔等 B 连**｜性格：痞｜装备·能力：破坏专家｜人物弧光：寻金逃死。  
- **2042 专家群（麦克凯、索拉等）**｜性格：设定卡｜装备·能力：专属小技能｜人物弧光：多人对战符号。  
- **无名士兵（系列真主角）**｜性格：由玩家定义｜装备·能力：兵种｜人物弧光：重生—死亡—再部署。

**【势力图谱】**  
美军、俄军、中国人民解放军（BF4）、欧盟、泛亚、一战协约/同盟、二战同盟/轴心、PMC、无国者。

**【贵重物品】**  
- 主战坦克、攻击直升机、战斗机、航母。  
- 爆破炸药与 C4、核武密码箱（战役）。  
- 「暗箭」电磁武器（BC2）。  
- 指挥官终端、卫星扫描。  
- 军衔解锁武器皮肤（多人成长符号）。

**【隐藏剧情 · 伏笔】**  
- 系列主题是「系统比个人伟大」：你死了，旗还在。  
- BF3/4 美俄中叙事服务多人玩具箱，政治简化。  
- 破坏系统是第三势力：楼一倒，战局重写。  
- 老兵系统跨作标记玩家资历。

**【大事记时间线】**  
1942 发售奠基 → 越战 / BF2 现代 → 2142 冰河未来 → Bad Company 破坏革命 → BF3/4 高峰 → BF1/V 历史 → 2042 争议 → 新作循环。

**【叙事基调 · 雷区】**  
基调：宏大、嘈杂、黑幽默重生。忌单人英雄灭国；忌无视友军复活/医疗机制；忌把真实历史惨剧纯娱乐化而无尊重（尤其一战故事）。最早锚点＝任意征服模式开局空降或 BF3 德黑兰街道。


**【战役与多人双层叙事细述】**

**BF1942 语法**：阵营、旗帜、载具刷新、重生——战争被做成可游玩的系统，个人故事让位于地图故事。罗马之路与秘密武器资料片扩展「如果兵器」的玩具箱。

**BF2 现代**：美中俄（及欧盟等）的全尺度战争幻想；指挥官模式把玩家变成半个 RTS 操作者；小队系统强迫协作。特种部队资料片缩小空间，强调室内与绳索。

**Bad Company**：B 连的黄金梦是系列少有的「角色喜剧」；全环境破坏让掩体成为消费品。BC2 的暗箭线把冷战遗产写成山崩海啸级武器。

**BF3 战役**：从新兵到被指控，布莱克本的审讯叙事框类似政治惊悚；迪马的莫斯科与巴黎线揭示「反恐」可被导演。多人地图如大壩、德黑兰公路、运营成为一代记忆。

**BF4 战役**：上海开场倒楼是技术演示也是政治隐喻；墓碑小队在中美俄之间运送「真相」。金杰的政变与最终选择影响过场，但多人仍是玩具箱。DLC 中国崛起、海军风暴等把战场推向亚洲近海。

**BF1**：战争故事（意大利高山、阿拉伯沙漠、弗兰德斯等）用短篇强调炮灰与友谊；行动模式模拟战线推移。

**BFV**：火线风暴大逃杀式异色；战争故事尝试女兵与抵抗视角，引发争议，但核心仍是二战枪感与破坏。

**2042**：专家英雄化、128 人、灾害天气、传送门怀旧；叙事弱于系统。Hazard Zone 抽逃模式是另一种「存在威胁」节奏。

**【兵种与载具作为力量体系细目】**
突击要突破，工程要反载具，支援要续航，侦察要信息。坦克需步兵清理反坦克手；飞机需防空与维修循环。破坏使「记忆中的掩体」消失，强制重新理解地图。这是战地与巷战 FPS 的本质差。

**【人物补】**
战役角色服务主题：迪马的俄语粗口与悲剧；爱尔兰的原则；雷克的沉默；BF1 的克莱在沙漠与理想之间。多人「人物」是军衔与狗牌。

**【地理补】**
垂直度：摩天楼、大坝、航母甲板。开阔度：荒漠、雪原、农田。城市：巴黎、上海、柏林式街巷。历史：堑壕与气球。

**【贵重补】**
旗点本身是圣物；M-COM 是爆破信仰；核弹密码与暗箭是战役麦高芬；载具击杀数是荣誉。

**【隐藏】**
主题曲旋律跨作召唤「老兵」身份；系列营销与政治简化常被批评，写作时可用士兵视角消解宣传。模组与私服是 PC 文化层。

**【状态机】**
开局部署：兵力全满。中期失旗：重生点被压缩。建筑倒塌：路线重算。战役关键过场后：解锁下一地图叙事。多人无永久死亡，只有回合结束。


**【战地·模式即世界观】**
征服=领土欲；突破/行动=战线史诗；团队死斗=巷战压缩；突袭=爆破信仰；火线风暴=末日抽逃；危险区域=抽资料撤离。模式告诉你战争被怎样消费。


**【战地·BF3/4 场景符号】**
德黑兰高速、巴黎地铁、上海高楼、航母甲板、苏伊士、新加坡，是「新闻里的战争」变成可玩地图。倒楼不是CG，是可互动的战局重写器。


**【战地·历史作伦理】**
一战二战故事应用泥、虱、错令、友军炮火，避免卡通纳粹或卡通英雄。士兵可以黑幽默，但不嘲弄死者。


**【战地·载具生态链】**
步兵克制无护卫坦克后部；防空克制飞机；工程修理延长战场寿命；侦察标记让整队变聪明。单人英雄电影镜头可以有，系统胜利必须靠链条。

**【战地·破坏后的战术发明】**墙倒了出现新枪线；楼塌了出现新制高点；桥没了步兵必须游泳或绕路。世界在对玩家说：记忆不可靠，观察才可靠。
**【战地·战役短片的用法】**用真名推动一次任务，用多人模式消化余波。例如上海倒楼后，契约者可在同一地形打征服，感到「新闻变成日常」。
**【战地·兵种人格】**突击急性子；工程强迫症修车；支援妈妈型扔弹药；侦察孤独症。允许刻板，好识别。
**【战地·载具人格】**坦克傲慢；轻型吉普滑头；直升机焦虑；战斗机贵族；舰船官僚。击杀日志是它们的墓志铭。
**【战地·伦理红线】**可写残酷，不写虐杀欣赏；可写黑话，不嘲弄真实惨剧群体；战略武器任务走拆除与撤离。

**一阶（新兵上线 · 征服模式前夜）**  
切入身份/时点：契约者以「刚部署的列兵 / 战地记者嵌入小队 / 工程营学徒」切入，锚定任意现代或二战地图开局。

开场白建议：「降落伞破了个洞，世界以每秒五十米的速度变大。你还没记住枪编号，HUD 上的旗帜已经从蓝变红——战争不等人读完教程。」

**二阶（王牌与墓碑 · 战役 / 大规模作战）**  
切入身份/时点：以「墓碑预备队员 / 王牌飞行员候选人 / 反抗军顾问」切入，锚定 BF4 上海倒楼或 BF3 巴黎夜。  
初始事件：摩天楼开始倾斜，金杰的广播与美军指挥重叠；你要在「撤平民」「夺数据包」或「驾驶坦克断后」中选一条，倒计时九十分钟。  
开场白建议：「玻璃雨比真雨密。有人在频道里唱走调的国歌，有人在报坐标。你发现所谓正义，只是频道加密方式的不同。」  
关键NPC立场：**雷克 / 爱尔兰**——要任务完成；**金杰阵营**——要政治胜利；**无名敌兵**——要活命。
