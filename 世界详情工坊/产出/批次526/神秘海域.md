# 神秘海域
<!--meta lib=主库 tiers=一、二-->

## 剧情

**【作品来源】**  
《神秘海域》（英语：*Uncharted*，台译秘境探险）是 **索尼互动娱乐** 旗下 **顽皮狗（Naughty Dog）** 为 PlayStation 平台打造的动作冒险系列，由 **艾米·亨尼格（Amy Hennig）** 等主创塑造。2007 年《神秘海域：德雷克的命运》（Drake's Fortune）起，经《纵横四海》《德雷克的骗局》《盗贼末路》完成内森·德雷克主线，另有前传《黄金深渊》、独立篇《失落的遗产》（克洛伊主角）及合集重制。系列销量逾四千万，以电影感关卡、角色互动与枪战攀爬著称。文风：普尔普冒险、伙伴喜剧、宝藏与诅咒并重。本档案以 **内森四部曲 + 失落的遗产** 为连续主线。

**【世界定位】**  
近现代地球上的「未被标上地图」角落：航海家遗物、失落之城、海盗共和国与神话诅咒并存。主角 **内森·德雷克（Nathan Drake）** 自称弗朗西斯·德雷克后裔，与导师 **维克多·沙利文（Victor Sullivan）**、记者 **埃琳娜·费希尔（Elena Fisher）** 及搭档 **克洛伊·弗雷泽（Chloe Frazer）** 等周游世界盗宝、拆弹、拆伙又重聚。敌人是雇佣兵、海盗、邪教式私军与被诅咒的突变体——顶点仍是「可被枪打死的人 + 必须封印的神器」。

**【世界观 · 力量体系】**  
**人类冒险者体能 + 枪械掩体战 + 攀爬跑酷 + 载具 + 诅咒遗物**。无修仙境界；诅咒可造成突变、幻象、生物武器级瘟疫。

**1. 凡人层**  
- 平民、导游、博物馆员。  
- 海盗与雇佣兵（埃迪·拉贾帮、拉扎尔维奇私军、Shoreline 等）：枪、载具、数量。  
- 老手冒险家（萨利、克洛伊、纳丁·罗斯）：近战与枪法达人类顶尖。  
- **内森·德雷克**：攀爬、即兴枪战、历史直觉、嘴炮；非超人，常挂彩。

**2. 组织与战争层**  
- 犯罪资本家（加布里埃尔·罗曼、佐兰·拉扎尔维奇、劳夫·阿德勒）雇佣军队级火力。  
- 海盗共和国莱伯塔利亚相关武装、叛军与 PMC（如 Shoreline）。  

**3. 诅咒 / 遗物层**  
- 黄金国偶像（变异瘟疫）。  
- 香巴拉 / 生命树（Cintamani）相关。  
- 伊拉姆的亚特兰蒂斯式装置与Djinn传说。  
- 亨利·埃弗里宝藏与海盗诅咒岛。  
- 印度诃罗神器与「失落遗产」相关力量。  

**乐园阶位映射（宁低勿高）**  
- 平民、普通海盗、博物馆安保 ≈ **一阶**。  
- 精锐雇佣兵、Shoreline 士兵、内森 / 萨利 / 埃琳娜 / 克洛伊 / 山姆·德雷克 / 纳丁 ≈ **二阶**（小队歼灭、火车坦克级场面中的个人战力；肉体惧重机枪与坠落）。  
- 突变守卫群、遗物完整释放的瘟疫 / 城市级崩塌 ≈ **存在级威胁**（条件性胜利：击沉偶像、摧毁装置、放弃宝藏）。  
**世界个人战力顶点 ≈ 二阶巅峰**。严禁把内森写成硬抗坦克炮塔无伤。

**【地理 · 舞台】**  
- 巴拿马海域与南太平洋无名岛（1 代）。  
- 伊斯坦布尔、婆罗洲、尼泊尔战城、藏区香巴拉（2 代）。  
- 伦敦、哥伦比亚、法国叙利亚城堡、也门、鲁卜哈利沙漠（3 代）。  
- 意大利、苏格兰、马达加斯加、印度洋岛屿与莱伯塔利亚（4 代）。  
- 巴拿马丛林（黄金深渊）、印度西高止山（失落的遗产）。  
- 波士顿 / 新奥尔良等「回家」节点。

**【世界剧情线】**  

**① 德雷克的命运（1）**  
内森与埃琳娜打捞弗朗西斯·德雷克棺，得日记指向黄金国。实为被西班牙人运走的黄金偶像。罗曼与纳瓦罗追债；萨利中枪假死。众人登岛，发现纳粹与西班牙人皆因偶像瘟疫突变。纳瓦罗欲出售病毒；内森击溃其并击沉偶像，与埃琳娜感情升温，萨利带若干宝藏离开。

**② 纵横四海（2）**  
内森被拉入寻找马可·波罗失落舰队与香巴拉。叛徒哈利·弗林、军火商拉扎尔维奇与女主雀儿娅线交织。名场面：崩塌的尼泊尔城、行驶火车枪战、雪山大桥。最终在香巴拉阻止拉扎尔维奇滥用力量，埃琳娜重伤但存活，主题从「寻宝」转向「人比宝藏重要」。

**③ 德雷克的骗局（3）**  
聚焦「亚特兰蒂斯 of the Sands」伊拉姆与弗朗西斯·德雷克骗局。童年闪回中萨利收留孤儿内森；现时马洛与塔尔博特的神秘教团追猎。沙漠飞机失事、游轮大战等。终局揭示骗局与Djinn装置，内森选择与埃琳娜婚姻修复，表面挂靴。

**④ 盗贼末路（4）**  
退休的内森被兄长 **山姆·德雷克（Samuel Drake）**（曾以为已死）拉回场，追寻海盗亨利·埃弗里与莱伯塔利亚。对手为富商劳夫·阿德勒与 PMC 领袖 **纳丁·罗斯（Nadine Ross）**。主题：兄弟羁绊、婚姻信任、是否再一次为宝藏出卖生活。终局宝藏泡影式收束，内森真正回归家庭，与埃琳娜育有一女。

**⑤ 失落的遗产**  
克洛伊与纳丁合作在印度寻找伽内什相关遗物，对抗叛军军阀。独立女主叙事，补完克洛伊弧光。

**⑥ 结局方向**  
主线以《盗贼末路》为准：内森挂靴（相对），山姆存活线取决于玩家关键选择记忆，但系列强调「家庭 > 黄金」。诅咒遗物多被摧毁或沉海。

- **维克多·「萨利」·沙利文（Victor Sullivan）**（导师）｜性格：狡猾温柔、老烟枪｜装备·能力：枪法、飞行、谈判｜人物弧光：骗子老爹→真父职｜立场关系：内森的锚。  
- **埃琳娜·费希尔（Elena Fisher）**（记者 / 妻子）｜性格：正直、勇敢、吐槽役｜装备·能力：枪战后期不输内森｜人物弧光：跟拍记者→并肩配偶｜立场关系：道德罗盘。

- **加布里埃尔·罗曼 / 阿托克·纳瓦罗**（1 代反派）｜性格：贪婪｜装备·能力：雇佣兵｜人物弧光：瘟疫结局。

- **凯瑟琳·马洛 / 塔尔博特**（3 代）｜性格：邪典领袖｜装备·能力：教团与幻象｜人物弧光：骗局核心。  
- **劳夫·阿德勒（Rafe Adler）**（4 代）｜性格：富二代执念｜装备·能力：买下战争｜人物弧光：决斗式败北。  
- **埃迪·拉贾（Eddy Raja）**（海盗）｜性格：旧识油滑｜装备·能力：海盗帮｜人物弧光：1 代葬身突变。

**【贵重物品】**

**【隐藏剧情 · 伏笔】**  
- 「德雷克」姓氏与孤儿身份的自我神话。  
- 每次宝藏都有文明级代价，系列反「贪婪」。  
- 山姆假死与监狱线是 4 代情感核。  
- 电影版（汤姆·赫兰德）为松散改编，乐园默认游戏连续宇宙。

**【大事记时间线】**  
巴拿马打捞 → 黄金国岛 → 香巴拉 → 伊拉姆沙漠 → 退休 → 莱伯塔利亚 → 印度遗产；童年萨利收留穿插。

**【叙事基调 · 雷区】**  
基调：冒险喜剧 + 突然的残酷。忌把内森写成沉默硬汉（他必须碎碎念）；忌无伙伴独行全片；忌让诅咒变成可控法术外挂。最早锚点＝巴拿马打捞夜或 4 代退休家中。


**【四部曲剧情细述与名场面因果】**

**1 代节拍**：开场海底棺→海盗袭船→萨利「中弹」→岛上要塞→海关文件→修道院→德国地堡→偶像真相→码头决斗。突变体「后代」是西班牙殖民与纳粹实验的双重遗产，把寻宝片写成生物恐怖。纳瓦罗的演讲点明：真正值钱的是武器化瘟疫，不是黄金。

**2 代节拍**：伊斯坦布尔博物馆夜盗→哈利背叛→尼泊尔战争城市→火车→婆罗洲→雪原修道院→香巴拉。拉扎尔维奇追求的是「不死的士兵」，与 1 代瘟疫镜像。雀儿娅的牺牲与幸存版本影响情感浓度；埃琳娜中弹迫使内森重新定义胜利。

**3 代节拍**：伦敦酒馆斗殴→闪回孤儿院与萨利→法国城堡→叙利亚→游轮→也门→沙漠空难→伊拉姆。马洛教团用仪式与药物制造「Djinn」幻觉，揭示许多超自然是可解释的残暴。婚姻危机线把冒险成本写进家庭。

**4 代节拍**：退休生活→山姆现身→意大利拍卖→苏格兰→马达加斯加开放岛→海盗新城→海中沉船。阿德勒是「有钱的内森」阴影；纳丁是职业镜像。莱伯塔利亚的理想国早已腐烂，宝藏要么诅咒要么空空，主题落在「停手」。

**失落的遗产**：克洛伊与纳丁的印度山城与古庙，军阀与神器，证明系列可无内森仍成立。

**【人物补】**
- **哈利·弗林**：嘴炮盟友与叛徒。
- **雀儿娅·因泰达**：游击队与信念。
- **查理·卡特**：3 代喜剧火药桶。
- **马洛与塔尔博特**：母子/主仆式邪典压迫。
- **劳夫的私人剑术与决斗偏好**：资本家的骑士病。
- **诺兰·诺斯的即兴**：角色语音人格的一部分，写作时应保留碎碎念。

**【地理补】**
每个主要关卡都有「垂直性」：悬崖、钟楼、水井、电梯竖井。开放岛（4 代）允许载具与多路径潜行，是线性电影关卡的进化。城市战（尼泊尔）把平民区变成枪战舞台，需注意叙事上的残酷。

**【贵重物品补】**
- 德雷克戒指的拉丁铭文与坐标传统。
- 马可·波罗的残页。
- 埃弗里的盐与金币传说。
- 萨利的怀表与债务本。
- 埃琳娜的摄像机与新闻伦理。

**【隐藏】**
「内森杀过多少人」的粉丝质询与游戏玩法暴力的张力，可在角色对话里自我嘲讽，但不要做成系统忏悔。电影版年轻内森与萨利第一次冒险，可作一阶副本。

**【状态机】**
未遇埃琳娜：内森更浪。已婚后：高风险任务需说服。山姆存活与否影响 4 代尾声电话。偶像沉海后 1 代瘟疫线关闭。


**【神秘海域·1-2 代情感工程】**
1代把「搭档喜剧」焊死：内森与埃琳娜拌嘴是呼吸。萨利中弹的假死训练玩家不信任过场。2代把背叛交给哈利，把理想交给雀儿娅，把婚姻可能性交给埃琳娜中弹，使寻宝的笑变成咬牙的笑。


**【神秘海域·3-4 代情感工程】**

切入身份/时点：契约者以「埃琳娜的摄像助理 / 包船水手 / 博物馆临时工」切入，锚定德雷克棺打捞成功、海盗袭击前后。

主线钩子/支线：护送日记残页；支线＝岛上宝藏微光收集、避开突变体幼巢。  
危险度/规避：中——规避偶像室与雇佣兵指挥部。  
任务方向/奖励：活着上岸、一支手枪与绳索；奖励＝小额宝藏、萨利的人情。

**二阶（私军与末路 · 香巴拉 / 莱伯塔利亚线）**  
切入身份/时点：以「专业译码员 / 叛逃 PMC / 山姆的狱中联络人」切入，锚定火车任务或马达加斯加夺宝期。  
初始事件：拉扎尔维奇或阿德勒的人把你和内森锁在同一节车厢；桥梁将炸，克洛伊在无线电里倒计时；你必须选择炸锁救全车或只切一条逃命绳。  
开场白建议：「车轮下的山谷没有护栏。内森喘着说‘放轻松’，枪口却没从你肩后移开——信任在这行是假的，直到你们一起流过血。」  
关键NPC立场：**内森**——关键时刻护人；**山姆**——目标偏执；**纳丁·罗斯**——尊重强者；**埃琳娜**——道德否决权；**劳夫·阿德勒**——有钱能使枪。  
主线钩子/支线：阻止遗物武器化；支线＝兄弟和解、Shoreline 内讧。  
危险度/规避：高——规避完整遗物释放与坦克级火力正面对轰。  
任务方向/奖励：分得「够用」的宝藏或更珍贵的情报网；奖励＝高端枪械皮肤级装备、与克洛伊 / 纳丁的合作契。


**一阶补强**：码头海关的贿赂；被遗弃的旅游团相机里有要塞地图；海盗逼你唱「入伙歌」试胆。
**二阶补强**：火车车顶换轨；Shoreline 合同的电子签名伪造；与纳丁的非致命近战考核。
**开场白（二阶）**：「马达加斯加的湿气把烟卷成条。山姆说『像以前一样』，萨利说『以前你们差点死』。你检查弹匣，发现少了两发——有人已经先开过枪。」

**一阶完整场（巴拿马夜）**
身份：潜水助理。事件：海盗要日记复印件。开场白：「海水进了面罩，你分不清是盐还是泪。甲板上有人喊Drake，有人喊开枪。」
NPC：**内森·德雷克**、**埃琳娜·费希尔**、**维克多·沙利文**、海盗头目。
钩子：把真日记藏进救生衣；危险中。奖励：呼吸管、手枪、萨利人情。


**二阶完整场（尼泊尔巷战/马岛）**
身份：前PMC医官。事件：卡车冲进市场，要你选救人还是抢卡车。
开场白：「香料与火药一个味儿。内森拉你一把，说‘别想了，跑！’」
NPC：**内森**、**克洛伊·弗雷泽**、**拉扎尔维奇军官**或**纳丁·罗斯**。
