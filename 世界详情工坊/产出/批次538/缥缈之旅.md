## 剧情

**【作品来源】**
原作正式书名为《飘邈之旅》（亦常写作《飘渺之旅》；清单与本条目标题作「缥缈之旅」，【作品来源】特此标明即萧潜所著《飘邈之旅》）。作者萧潜，本名刘晓强，南京人。2003 年 10 月至 2005 年 2 月前后于网络连载完结，约 28 集 280 章、近 191 万字；台湾鲜网曾出 28 本分册，大陆南海出版公司 2006 年前后出 7 册实体（误入天庭 / 阗殛魔杖 / 逆天宝镜 / 赤明魔尊 / 神之战魂 / 厉禁天君 / 神罚之眼）。与《诛仙》《小兵传奇》并称早期「网络三大奇书」，被视作现代修真体系奠基作之一：储物法器、境界阶梯、丹药法宝、雷劫飞升、传送阵、拍卖会等后世模板多自此定型。2018 年前后入选「中国网络文学二十年江苏 20 部优秀代表作品」一类回顾名单，评价轴心是「开创」而非后期文笔密度。文风幽默轻松、重兄弟情与星际奇观，作者自谓「茶余饭后消遣」，不以黑深残为主；光明网等评论亦指出其在「中式玄幻」兴起前的里程碑位置。官方后续有《歧天路》（赤明一系弟子线）；坊间另有《飘渺神之旅》等衍生，口碑与正典不可混。以小说正典为准。

**【世界定位】**
中华修真脉络与星际科幻并置的「星际修真」宇宙：地球只是起点，主角李强以商人出身被傅山改造后误入天庭星，一路游历坦邦星、霖明星、封缘星乃至仙魔诸界与原界，凭商业头脑、义气与修真，从凡人成长为原界之主、神人级存在。世界不是单一王朝修真地图，而是「多星球 + 多界域」旅行结构：凡俗战争、晶石能源、门派圣地、仙令魔雾与原界权柄层层抬升舞台。情感线刻意克制，兄弟义气的戏份密度高于后宫。一句话：地球商人在无尽星空的飘邈修真之路。

**【世界观 · 力量体系】**
力量来源以真气 / 真元修炼为主，辅以功法、丹药、法宝、心炼与各类劫难；修真者与凡人、异族、妖仙、散仙、仙人、魔头、灵鬼、神人同处同一星际舞台。传送阵、定星盘、玉瞳简等跨星工具把「门派—星球—界」串成可旅行的大地图。死亡：低阶身死即灭；高阶可有元神、散仙、转世、原界之力等迂回，但皆有代价，不可写成无限读档。凡人王朝与修真门派长期并存：绿色盆地三国每隔十数年征战，真正定局的往往是背后供奉与宗门；坦邦星则把**晶石**当能源与货币，出现飞战团一类「科技外壳 + 修真内核」的编制，战争逻辑不同于纯法宝对轰。

正统修真境界（由低到高，各境常分上下；后世网文「炼气—筑基—金丹」模板多可对照此表，但本世界书**必须保留原名**）：
1. **旋照**：入门感知真气、炼体炼气起点，战力约强于凡人顶尖，可徒手开石、御低阶符器。
2. **开光**：开窍通脉，真气外放雏形，可对敌伤人，勉强抗一支凡俗精锐小队。
3. **融合**：气与体、意进一步合一，战技与法器操控成型，一城内可称好手。
4. **心动**：心魔关口突出，道心不稳则走火入魔；过关后意志与真元质变，开始接触「修心即修真」的主旨。
5. **灵寂**：内丹 / 金丹向阶段，真元凝实，可远程手段与区域压制，毁据点、镇矿脉一级。
6. **元婴**：元婴结成，有身外元婴、小宇宙雏形；可越阶续命、双修梳理、远程夺舍威胁上升。
7. **出窍**：神识 / 元神可出窍远游，侦查与远程杀伤跃升，一地战局可被单人搅动。
8. **分神**：分神化身、多线操控，战场覆盖与保命能力大增，适合星际追杀与反追杀。
9. **合体**：神与体、法与道进一步合一，一域顶尖，移山填海初现。
10. **渡劫**：天劫、仙劫、神劫、丹劫等劫难考验，过则质变、败则身死道消；渡劫期是「人形战略武器」门槛。
11. **大乘**：凡修顶峰，可飞升仙界；之上另有散仙、仙人、古仙、远古上人、始隐、神人（含九神天等称谓）更高层级。

修炼路径分支：修仙、修佛、修妖、修魔、**修神**等；早期修真界格局曾被《缥缈论》一类总结改写，但主角线真正掀桌的是《**修神天荐章**》——仙界奇宝，修炼有天兆，把个人成长直接暴露给仙魔上层。法宝分宝器 / 仙器 / 神器等品级；资源常见灵石、晶石（坦邦科技能源）、仙石 / 黄金仙石。心魔、劫难、界域规则（仙界令、黑魔逆行通道、原界权柄）均会改写战局。另有灵剑体、楚巫族、纳波人、绿族、裂兽族等体质 / 种族修正，不可全按「人类修士」模板写。

破坏力体感（宁低勿高）：旋照~开光≈冷兵器超凡、裂石级；融合~心动≈一城好手 / 小队灭杀；灵寂≈区域强者、可毁据点与矿脉；元婴~出窍≈一城至一地级威胁；分神~合体≈地区级、移山填海初现；渡劫~大乘≈国级至大陆级；仙人 / 魔尊 / 神人 / 原界之主向≈世界级乃至跨界规则级，属本条目七阶及顶点。

乐园阶位映射：旋照~开光≈一阶；融合~心动≈二阶；灵寂≈三阶；元婴≈四阶；出窍~分神≈五阶；合体~渡劫≈六阶；大乘~散仙 / 初入仙魔≈七阶；仙界主宰、魔尊巅峰、神人 / 九神天 / 原界之主＝**超阶：存在·情报优先 / 条件性胜利**（顶点以存在感与情报战处理，不得用「战力归零」式降智）。

同境仍有上下之分，丹药、法宝与心炼差距可造成「同境碾压」；星际旅行意味着资源与情报差比同星球门派斗争更致命——不知道定星盘与传送阵规则的「土著大乘」也可能在跨星局中失位。写任务时优先描述破坏力、编制与信息差，少堆裸数字。

**【地理 · 舞台】**
- **地球 / 火星**：李强商人起点，都市背叛与追杀构成凡人序章；傅山以紫炎心改造其身。火星有古传送阵，本拟送往封缘星，是「地球—星际」唯一公开跳板。
- **天庭星**：误传落点，全书前段主舞台。原始森林合抱参天、戈壁与红岩石化山脉、寒冰原、最肥沃的**绿色盆地**。盆地三国——**故宋国**（相对弱，含林城、都城、供奉堂）、**丽唐国**、**大汉国**（最强）——十数年征战不息，皇室贵族多与修真界挂钩。盆地外有暴风原望明国、清风国（纳善出身）、秦吴楚夏商等族群活动带；**星星宫**（莫怀远被困与元婴线）、暗影堡、黑旗军戈壁营地构成早期副本感场景。
- **坦邦星**：冤魂海分隔**坦邦大陆**与**西大陆**。坦特国黑狱 / 黑营（苦囚挖晶石、无政府弱肉强食）、邦奇宁国（绿族主导、飞战团）、拉都国（岚湫公主等线）并立；**天籁之城**孤悬冤魂海上，终年霜雪、淡蓝仙境，修真与晶石科技最先进的交汇点；阪寿商行等黑市把持跨陆贸易。西大陆坎波儿大神信仰塑造绿族神使体系（卡本）。
- **霖明星**：修真圣地，门派与**千宝阁**类商会林立，奇龙城、苍极宗、修真联合会等；大罗上仙令曾在此把「抓李强」变成全星政治任务。
- **封缘星**：傅山所属**重玄派**核心所在，古剑院 / 天缘城一类高阶修真中心，终局聚首与故人线收束地。
- **诸秘境与界**：幻星神阵、幻神殿、古神藏、鑫波神藏、极玄冰眼、冤魂海、紫魂星等；层级上有**仙界**、**灵鬼界**、**黑魔界**、**原界**（李强后期开启并为主）。另有潜杰星（百黄老人恶霸组织）、幻树星、巴达星（元始门相关）、勾蓝星 / 战圈大陆等点缀后半星际版图。
感官：星际传送的眩晕与阵纹光、黑狱矿尘与晶石灯、绿色盆地刀兵与供奉堂香火、天籁城霜雪与仙器嗡鸣、魔血煞雾的腥热、原界权柄下的寂静虚空。契约者入世时，应先判断「当前在哪颗星 / 哪一层界」——同境界在天庭星宫廷与在霖明星圣地、黑魔逆行通道口，话语权完全不同。

**【世界剧情线】**
网络版二十八集可粗分为八个因果阶段（实体七册标题夹注），时间顺序如下，便于任意时点判断「世界处于哪一段」。

① **地球背叛与紫炎心（第1集前史 · 误入天庭开篇）**  
资产过千万的贸易商人**李强**遭女友与好友**曾伟光**背叛，愤怒伤人后万念俱灰、面临追捕。西汉时代修真者**傅山**（字傅崇碧，号青峰真人，重玄派核心）为寻找能承载**紫炎心**能量的宿主重返地球——紫炎心不是普通丹药，而是可改造成长底盘的高能核心。李强先拒后因绝境应允；傅山以真气 / 紫炎心助其筑基，改造成高大俊美之躯，兄弟相称，经火星古传送阵拟往**封缘星**重玄本部。妖女**花媚娘**纠缠傅山并搅乱方位，传送未定，李强被单独甩出，落于未知星球——**天庭星**。名场面：都市绝境应诺、火星阵纹亮起、花媚娘一闹改写一生轨迹。主题：背叛、被迫入道、星际误传。因果：无花媚娘则李强直接入封缘体系，后文黑狱兄弟团与「独自成势」全部不成立。

② **天庭星：故宋、黑旗军与商业破局（第1–2集 · 误入天庭 / 星星宫·寒冰原）**  
李强于含林城一带结识商人**赵豪**（后拜师，狂刀向），联手击退**黑旗军**屠城之危——此战确立「义气 + 算账」双开局。故宋国召见，结识**梅晶晶**（妞妞，灵蟠门 / 回春谷一系，炼丹大师**梅游冰**之孙女；因灵丹被觊觎全家遇袭，曾流落七叉岭为匪首）、**侯霹净**（原名赵岳，师尊赐名，大智威圣亲王，元始门，故宋皇帝之兄，年轻时弃帝位求道）。李强以地球商业、军需、制度经验化解故宋对外战争与内政危机：粮草、商路、供奉堂与宗门利益被他一层层拆开，声名鹊起，也树敌于丽唐细作、本国供奉与跨国恶霸。  
支线：**星星宫**、寒冰原；**莫怀远**被困与元婴线；与莫怀远、**倚君煞（琦君煞）**结生死兄弟；**百黄老人**（潜杰星恶霸组织，半白半黄长发，麾下雷天笑「小雷雷」、斗战王司徒雍、紫仙王紫影等）构成早期跨星压力。阶段结局：李强在天庭星站稳、收徒结义，修为自旋照向上攀；宫廷与门派的人情债同时到期，为后续暗算埋雷。

③ **坦邦星：黑狱争锋与晶石战争（第3集核心 · 黑狱争锋，并延至天籁前）**  
因利益与仇家布局，李强真元运转受制，被投入**坦特国黑狱**挖晶石——黑狱是无政府的矿奴地狱，也是全书最重要的「结义熔炉」。他以组织力与义气集结上千苦囚： **纳善**（天庭星清风国人，秃头独眼，入狱极早，监区老大，直率大大咧咧，后称李强「老大」仍拜师）；**帕本**（西大陆商人，天生胆小，货被坦特天击兵没收送狱，后拜师，其妻女财物遭**阪寿商行**谋夺，引出复仇线）。暴动夜与清剿部队血战突围，是中前期最大名场面之一：镐尖、晶石灯、口号与修真残招混战。  
出狱后立刻卷入坦特与**邦奇宁**的晶石矿争夺（邦奇宁绿族、飞战团）；西大陆信仰与**卡本**（坎波儿大神使者，绿肤、眉心血痕、墨绿长袍与圆牌）结盟；冤魂海偷渡、**天籁之城**霜雪科技修真、拉都国宫廷线并行。阶段结局：黑狱兄弟随李强修真，李强恢复并提升境界，世界观从「一国王朝」升级为「双陆 + 能源战争」。因果：阪寿商行仇恨、绿族人情、天籁城线索全部进入中盘。

④ **法宝与门派风云（第4–10集 · 释魂龙戒 / 十八灭魔手 / 天籁之城 / 风云变幻 / 阗殛魔杖 / 大幻佛境 / 重返天庭）**  
主线进入法宝与势力深水区。**释魂龙戒**：重玄派核心兄弟的身份与责任象征，银色环节戒指，戴上即背负传承——李强一系与封缘星的法理纽带在此显性化。**十八灭魔手**等秘术扩充战斗清单。天籁之城继承、城主与科技修真利益纠葛，使「晶石文明」与「门派文明」第一次对等谈判。  
**阗殛魔杖**卷：阗殛老祖**李俞苇**（大乘后修魔，三元神）的魔杖级威胁；剧情含护送公主、冤魂海劫掠、莽原 / 佛宗 / 天戟峰等多方卡位，李强在「保人」与「夺宝」之间反复站队。**大幻佛境**考验心性与佛魔界限，防止主角线滑成纯魔修。其后**重返天庭**了结故宋—供奉—黑旗余孽旧账，名号多星流传。阶段因果：兄弟团扩大，法宝级依仗到手，与仙魔边缘人物结下善恶两面债，为仙界点名做铺垫。

⑤ **逆天宝镜与修神之祸（第11–13集 · 逆天宝镜 / 修神之祸 / 灵鬼双尊）**  
获得核心神器**逆天宝镜**（元婴双修、空间 / 反制类神通相关，书卷级名物），修为与身价齐飞。接触并修炼《**修神天荐章**》（仙界奇宝，与**青帝**关联），天兆大放，仙界侧目——「修神之祸」不是单一反派，而是上层规则对「另辟神人路线者」的系统性打压与招安并存。**灵鬼双尊**等跨界强者登场，交易、猎杀、合作同一张桌上发生。  
名场面：宝镜认主、天荐章天兆、灵鬼界规则与修真界礼仪冲突。主题：神器在手、路线分叉（正统大乘飞升 vs 修神）、被上层世界点名。因果：此后李强再也无法当「地方成名散修」，一举一动牵动仙魔。

⑥ **大罗上仙令与赤明魔尊（第14–18集 · 大罗上仙令 / 逆行通道 / 赤明魔尊 / 危机重重 / 神之战魂）**  
仙界**大罗上仙令**下达霖明星诸派，围堵李强；千宝阁拍卖、黄金仙石、圣地门派全部政治化，「抓人或护人」成为站队题。**逆行通道**撕开，连通**黑魔界**；**赤明魔尊**（黑魔界排行前列，赤红脸、艳红眼、落腮胡、骨头项链，魔血煞雾等手段；其上尚有开皇、延康等更强序列，正典以赤明与李强互动最深）与李强对撞后被收服 / 同行修神——魔头改修神，是本书价值观的硬核表达之一。  
**神之战魂**线中，战魂刀真名**九衍鎏**（空间操控向神器，大范围光幕等）逐步被李强掌握，初期因修炼岁月不足未能尽用。**孤星**等存在揭示天荐章与**青帝（元木上人）**渊源。阶段结局：李强拜青帝等为师，魔尊成队友，仙魔追杀与合作并存；津阳城等地点曾受魔血煞雾威胁，大战乾善庸等名场面巩固「兄弟团 + 魔尊战力」编制。

⑦ **始隐、贝冶、厉禁天君与古神传说（第19–24集 · 心鉴之花 / 始隐初现 / 贝冶丹鼎 / 厉禁天君 / 古神传说 / 极玄冰眼）**  
**心鉴之花**等心性宝物线后，**始隐者**一系初现，提示「比仙人更老」的观察者层级。李强为救人炼制高阶丹时，无意从**贝冶丹鼎**中救出被困的**天真上人**——丹鼎本身是炼器 / 炼丹神器，救人即开传承。**厉禁天君**线：孕丹、神灵劫、九衍鎏深挖与天君级压力，渡劫戏份密集。  
古神传说、**极玄冰眼**、地图残片把线索拧向「古神藏 / 鑫波」。此阶段李强在大乘门槛与仙人门槛之间反复渡劫，人脉从星球门派升到仙魔始隐层级；每一次渡劫失败风险都是「真死」，禁止写成刷经验。

⑧ **原界之主与神罚之眼（第25–28集 · 鑫波神藏 / 原界之主 / 守护神兽 / 神罚之眼）**  
**鑫波神藏**与古神藏开启，李强在青帝、**天姑**等帮助下开启**原界**，成为**原界之主**，可调动原界之力，修为至**九神天**等神人级。收束线并行：助**莫怀远**修成散仙并转世、地球寻转世后的倚君煞与故人、**守护神兽**契约、封缘星重玄故人、赵豪等旧部汇报。终章**神罚之眼**：历经神罚之眼神劫成就神人；众人于傅山处聚首把酒，收于「修真无尽、星空无尽」。  
作者后补：**云钰**（慧蘅宫、好钻研的恬静弟子，双修道侣向）为李强主要情感线，因修行观未走凡人圆满。世界终态：李强一系稳居原界与星际高层；赤明一系由官方续作《歧天路》延展；正典本体以聚首与开放星空作结，而非灭世或称帝。全书情绪底色是「还能再往前走」：神罚之眼之后不是谢幕，而是无尽星空的新起点——写终局任务宜留远航与聚首，不宜强行世界末日。

**【隐藏剧情 · 伏笔】**
- 紫炎心与傅山西汉身份、重玄派在封缘星的完整布局，开篇仅露一角，终局聚首才收束——早期「大哥带弟」其实是跨千年布局。
- 花媚娘搅局表面是闹剧，实为李强「独闯多星」叙事引擎：无此则无黑狱兄弟、无独立成势、无原界之路。
- 《修神天荐章》与青帝、孤星、仙界天兆：修神不是简单升级，而是惊动上层的路线选择；天兆=暴露坐标。
- 赤明从魔尊到修神同伴，正典内完成「魔可修神」对照，并为官方续作《歧天路》埋线。
- 莫怀远肉体毁灭→散仙→转世，云钰纯阴体元婴助梳理：显示「元婴小宇宙 / 转世」规则，早于后世网文流行写法。
- 古神藏 / 鑫波神藏 / 原界：古神传说不是背景板，而是终局权柄来源；令牌、残图、冰眼皆为钥匙碎片。
- 黑魔界序列（开皇、延康等）与仙界老家伙：世界天花板远高于「星球大乘」，写七阶任务须保留上方阴影。
- 作者补笔云钰感情与「修真无尽」：刻意拒绝凡人圆满，保持星际开放结局。
- 跨作品：仅《歧天路》作官方延展；勿把《飘渺神之旅》等低评衍生当正典战力与结局。

**【叙事基调 · 雷区】**
画风：轻松幽默的星际修真冒险，重兄弟义气与奇观探索，商战智慧可破局，不走无尽阴谋与黑深残；对话可痞可豪，忌过度现代网络烂梗淹没年代感。  
忌：① 把李强写成冷血杀伐或无脑舔狗——他豁达重义但有商人底线；② 把境界名乱换成「炼气筑基」而丢掉旋照—大乘原名（可对照解释，不可替换正史）；③ 低阶硬刚青帝 / 原界之主 / 神罚之眼本体；④ 用《飘渺神之旅》等衍生覆盖正典结局；⑤ 把黑狱写成无意义受虐副本——它是结义与组织力的成型场。  
最早可切入锚点：李强刚落天庭星、尚未入故宋宫廷之前；或黑狱暴动前夜。卡片生成时优先锁定「哪一颗星 + 哪一集因果」，避免把仙界令与含林城守城写成同一难度。

## 阶位切入点

> 阶位↔境界：一阶≈旋照~开光，二阶≈融合~心动，三阶≈灵寂，四阶≈元婴，五阶≈出窍~分神，六阶≈合体~渡劫，七阶≈大乘~散仙/初入仙魔。世界顶点（青帝、神人/九神天、原界之主李强终局态）为**超阶：存在·情报优先/条件性胜利**，低阶切入一律规避顶点硬刚。

**一阶（旋照~开光 · 天庭星落地与故宋边缘线）**
切入身份/时点：契约者以「刚落入天庭星的外乡散修 / 商队护卫」身份切入，锚定李强初到含林城、与赵豪击退黑旗军前后。
初始事件：黑旗军逼近含林城外，商队炸营；你与一个自称见多识广的地球口音青年（李强）被裹进守城民兵，城主府急征懂账册与军需的人——有人出高价买「外乡人」的命去填戈壁。
开场白建议：「绿色盆地的风里有血腥和粮价。你站在含林城垛口，旁边那个刚结识的高个子商人正飞快盘算守城的粮草缺口——他说自己叫李强，可他看刀兵的眼神，不像这颗星上土生土长的人。」
关键NPC立场：**李强**——初来乍到却敢用商道破局，可结盟换情报；**赵豪**——本地豪爽刀客，对能打仗的外人友善；**梅晶晶**——顽皮落难少女，可能把你卷进灵蟠门旧仇；**侯霹净**——高高在上的宗亲修士，视有用者为棋；**黑旗军头目**——屠城利益优先，见外乡修士必夺宝。
主线钩子/支线：黑旗军攻城、故宋宫廷召见、供奉堂与回春谷丹药线；支线＝戈壁护送、七叉岭匪窝周旋、星星宫外围探路。
危险度/规避：中——主威胁是凡俗军阵与开光级打手；须规避侯霹净师门长辈、任何灵寂以上供奉。
任务方向/奖励：在故宋立住脚、换到入门功法与储物简；奖励取向＝旋照/开光功法、低阶法器、故宋人情、丹药边角料。

**二阶（融合~心动 · 故宋破局与暗算前夜）**
切入身份/时点：以故宋客卿、供奉堂外围弟子或跨国商队掌柜身份切入，锚定李强用商业智慧缓解故宋战事、树敌渐多之时。
初始事件：丽唐与故宋边境商路被劫，宫廷要你与李强联合查账；查到一半发现账册指向宗亲府与某修真供奉的利益输送——心动关口的心魔幻境与真实暗杀叠在一起。
开场白建议：「故宋的朝堂比戈壁更难走。你刚把一份军需清册按在案上，窗外飞来的不是信鸽，是灭口的符箭——李强在隔壁哈哈笑着说：看来咱们查对了。」
关键NPC立场：**李强**——需要可信盟友分摊朝堂火力；**侯霹净**——可赐名分也可卸磨杀驴；**梅晶晶**——丹药与情报掮客向；**百黄老人**眼线——试探并招揽或做掉新晋；**曾与李强结怨的供奉**——必报复。
主线钩子/支线：故宋—丽唐战争议和、供奉堂夺权、心动心魔考验；支线＝丹方走私、寒冰原采集、暗影堡潜入。
危险度/规避：中高——须规避百黄一系大宗师与元婴供奉。
任务方向/奖励：过心动、拿稳客卿位；奖励取向＝融合/心动功法、心魔类秘术解法、中阶丹药、商路特权。

**三阶（灵寂 · 黑狱与晶石战线）**
切入身份/时点：以被押入坦特黑狱的苦囚、晶石矿工头或邦奇宁探子身份切入，锚定李强真元运转受制、组织狱中暴动前后。
初始事件：黑狱塌方，晶石脉暴露出旧阵纹；狱卒要活埋知情者，**纳善**控制的监区突然对你抛出条件——加入今晚的暴动，或被钉在矿壁上。
开场白建议：「晶石的冷光照着每个人脸上的灰。李强的声音从隔壁洞传来，平静得可怕：『兄弟们，今天不出去，就永远是矿渣。』你握紧偷藏的镐尖，听见远处清剿营的战靴。」
关键NPC立场：**李强**——暴动大脑，重义但要你服从指挥；**纳善**——区老大，认老大不认规矩；**帕本**——胆小却知商路与西大陆情报；**卡本**——绿族神使，后期可成外援；**坦特狱吏 / 阪寿眼线**——出卖与镇压。
主线钩子/支线：黑狱暴动突围、坦特—邦奇宁晶石矿大战、帕本复仇阪寿商行；支线＝冤魂海偷渡、天籁城外围交易、西大陆信仰冲突。
危险度/规避：高——清剿营与灵寂以上矿脉守卫；须规避天籁城城主级与跨境魔修。
任务方向/奖励：活着出黑狱并站队晶石战争；奖励取向＝灵寂功法、晶石能源技术残页、黑狱兄弟人脉、中阶法宝残件。

**四阶（元婴 · 法宝与重返天庭线）**
切入身份/时点：以元婴期游历修士、重玄外围或天籁城客卿身份切入，锚定释魂龙戒—阗殛魔杖—重返天庭阶段。
初始事件：拍卖会拍出疑似**释魂龙戒**的仿品，真品线索指向重玄故人；同场有人悬赏「李强的元婴小宇宙数据」，你被误认为中间人，必须在逃亡与鉴宝之间选边。
开场白建议：「拍卖槌落下的瞬间，半个会场的神识锁在你身上。有人笑着说逆天宝镜还在后面——可你只想先分清，哪枚戒指是真的，哪句悬赏是假的。」
关键NPC立场：**李强**——已有元婴底蕴与兄弟团，对鉴宝者又疑又用；**莫怀远**——元婴兄弟，可托死生；**云钰**——慧蘅宫，双修与典籍线索；**阗殛老祖残余势力**——夺杖不择手段；**重玄派使者**——认戒不认人。
主线钩子/支线：释魂龙戒归属、阗殛魔杖护送/争夺、大幻佛境心性试炼、重返天庭了断；支线＝十八灭魔手残卷、佛宗—魔修谈判、公主线护送。
危险度/规避：高——须规避大乘边缘的阗殛级与佛境守门人。
任务方向/奖励：拿到一件真正法宝级依仗；奖励取向＝元婴功法、法宝残核、重玄人情、双修辅佐类秘术（勿越级发神器完整体）。

**五阶（出窍~分神 · 逆天宝镜与修神之祸）**
切入身份/时点：以出窍/分神期成名散修或霖明星门派真传身份切入，锚定李强获逆天宝镜、修炼《修神天荐章》惊动仙界之时。
初始事件：夜空天兆大放，门派长老宣读半道来历不明的「协查令」；你的分神在千宝阁看见逆天宝镜的镜像残片被人竞价——出价最高的不是人，是一道仙压。
开场白建议：「神识刚出窍，就被一道不属于此星的威压按回泥丸。有人在街角低声说：修神之祸来了。你抬头，看见李强站在光柱边缘，像是又高兴又麻烦。」
关键NPC立场：**李强**——被仙魔点名，需要分神级帮手挡枪；**青帝一系眼线**——观察而非立刻收徒；**灵鬼双尊相关存在**——交易与猎杀并存；**霖明星门派长老**——执令围堵或包庇；**孤星线索持有者**——卖天荐章真相。
主线钩子/支线：逆天宝镜争夺余波、修神天荐章天兆、灵鬼界接触、大罗上仙令前兆；支线＝千宝阁鉴宝、黄金仙石护送、分神化身演戏骗围杀。
危险度/规避：极高——仙压与灵鬼双尊级；须规避青帝本体、完整仙令降临。
任务方向/奖励：在修神之祸中活下来并摸到天荐章边角；奖励取向＝出窍/分神秘术、宝镜残识、仙石、跨界情报。

**六阶（合体~渡劫 · 赤明魔尊与厉禁天君线）**
切入身份/时点：以合体期一方尊者或渡劫期求药者身份切入，锚定逆行通道、赤明魔尊、贝冶丹鼎、厉禁天君阶段。
初始事件：黑魔界逆行通道撕开霖明星郊外，**赤明**的魔血煞雾吞掉半座城；你渡劫所需的丹成色被污染，唯一解法指向李强阵营正在争夺的**贝冶丹鼎**。
开场白建议：「魔雾里有骨项链碰撞的轻响。赤明魔尊站在通道口，像在等一个敢跟他谈条件的人——而李强已经在你身后拍了拍你的肩：『兄弟，劫难跟魔尊，今儿个一块儿过。』」
关键NPC立场：**李强**——收服/同行赤明的主导者，要你选边；**赤明**——可战可降，降则同行修神；**天真上人**——困于丹鼎，救之得传承；**厉禁天君相关势力**——以劫相逼；**青帝**——远距授法，不轻易下场。
主线钩子/支线：收服赤明、神之战魂与九衍鎏认主、贝冶救人、厉禁天君孕丹/神灵劫；支线＝魔血解毒、始隐者接触、古神传说残图。
危险度/规避：极高——渡劫失败即死；须规避天君全力、青帝试探性一击。
任务方向/奖励：渡劫成功并绑定魔仙任一阵营；奖励取向＝渡劫辅助宝、九衍鎏使用残篇、丹鼎炼一炉、魔尊级人情。

**七阶（大乘~散仙/初入仙魔 · 原界与神罚之眼）**
切入身份/时点：以大乘飞升在即的星域名人、散仙或黑魔/仙界边缘客卿身份切入，锚定原界开启至神罚之眼终局。
初始事件：古神藏裂开，原界之门只认「有缘与有力」；李强已在门内握住权柄，门外仙魔混战——你手中半块鑫波神藏令牌突然发烫，成为两边抢夺的钥匙。
开场白建议：「门后没有天，只有比天更大的安静。有人已经叫他原界之主。你站在神罚之眼的光晕外，知道再近一步就不是星球战争，而是神劫。」
关键NPC立场：**李强（原界之主）**——终局顶点之一，可邀共守不可硬夺权柄；**傅山**——聚首东道，调和旧人；**青帝**——超阶师长，条件性合作；**赤明**——修神同伴，战力恐怖；**云钰 / 莫怀远转世线索**——人情收束支线；**神罚之眼规则本身**——无差别降劫。
主线钩子/支线：原界权柄分配、神罚之眼神劫、地球转世亲友、封缘星聚首；支线＝守护神兽契约、歧天路前史伏笔（赤明弟子向）、无尽星空远航。
危险度/规避：贴近顶点——青帝、神人、原界之主＝**超阶巅峰：存在·情报优先/条件性胜利**；不得用「顶点无战力/可随便碾压」式降智，胜利须来自站队、劫难规则与代价。
任务方向/奖励：在终局站队并带走一丝原界/神劫余韵；奖励取向＝大乘飞升资粮、散仙法门、原界碎片级权限、神器残识、跨星门派话事权（不可直接发完整原界主权）。
