## 剧情

**【作品来源】**
《魔界拍卖所-定期競売》为轮回乐园休闲库收录的**魔界拍卖所·定期落札会与物语交易伦理**情景档案。清单标题点出「魔界拍卖所／定期競売」母题，**无与标题逐字对应的单一出版长篇**；气质整合 DLsite 同人圈常见的「魔界オークション」「闇オークション」「落札」系公开作品惯例，以及音声／CG 中把拍卖写成「被看见、被竞价、被温柔领走」的暧昧舞台。可公开对照氛围源包括：SnowCable《魔界闇オークション》（RJ01258460，吸血鬼女主人竞拍与宠物式宠爱）、同人漫画《魔界オークションへようこそ！》一类「魔界拍场」设定、以及 DLsite「オークション／魔界」关键词页中大量以预约、展柜、加价声为核心的作品。媒介印象：同人音声／CG／短篇跨媒介；搜笔趣阁以本条目标题检索无对应长篇小说书页。本档案以「定期競売」为专属锚点，写**目录册、举牌、落槌后的茶会与是否能在报价之外说停**，不写强制掠夺美化，不写数值比拼或闯关式推进。

**【世界定位】**
夜都灰市一栋改装成拍场的旧银行「落札馆」。契约者以**见习书记／目录整理员**身份进入：每两周一次的「定期競売」把魔界风俗化的拍品——记忆碎片匣、旧契约残页、半透明翅膀标本、匿名告白信——放上转盘，而真正被交易的往往是**参与者愿意被谁记住**。一句话：槌声落下之前，所有人都还能改口。

**【世界观 · 舞台设定】**
时代为轻度奇幻外壳的当代生活感：电车、便利店、魔界签证贴纸与人类区通勤并存。社会氛围温和而讲究口碑：灰市允许「魔界系服务业」挂牌，但必须公示拒客权与冷静期。落札馆不是战场，也不是无规则黑市；它是预约制文化空间——进门要核名、拍前要问禁忌、落槌后有三十分钟反悔茶。

本世界**没有**强弱对决逻辑，也不用等级解释亲密。唯一的「规则」是拍场现实逻辑：
1. **任何加价与接触前须有可观察同意**（举牌、口头确认、主动留下联络卡）；醉酒举牌无效，沉睡中代拍 = 黑线。
2. **「停」「撤拍」「反悔」永远有效**：主理会当场撤项，不羞辱撤拍者。
3. **禁止偷拍／录音要挟／伪造竞价记录**；发现即除名并通报灰市联席。
4. **配偶与家人不是背景板**：他们有名字、有疲惫、有自己的失败，不可写成「活该被绿」。

世界的温度来自：目录册边角被翻白的那一页、槌柄上磨亮的拇指印、茶会里多摆的一副杯碟、以及有人把「起拍价」写成「今天只想被叫名字」。治愈感不靠奇迹，靠重复：同一句「先问状态再举牌」、同一支退色蓝墨水写的黑名单、同一盏只在落槌后亮起的琥珀灯。

**【地理 · 生活舞台】**
- **落札馆本馆**：灰市站步行六分钟的旧银行。一楼「目录厅」与鞋柜、主题区「定期ホール」、私密区「目録室」、缓冲「落札後談話」、茶水角、储物角与小型露台。玻璃门贴「定期競売・少人数・可拒否」。
- **定期ホール**：公开示范与主流程；举牌声与呼吸交叠是升温主舞台。
- **目録室**：熟客加钟式「深度鉴赏」多发生于此；适合慢热告白与练习说不。
- **落札後談話／茶水角**：崩溃与和好常在这里；排期表与「本周主题色」贴在保险柜门上。
- **灰市河堤长椅**：会后散步与秘密短谈的默认舞台。
- **駅前连锁对照店「クイック落」**：便宜、快速、没人记得你的禁忌史——衬托落札馆的「慢」。
- **各人自宅玄关**：个人线后期「上门送目录」——只到垫子，尊重家庭边界。
- **周末市集**：采购展柜绒布、季节花与小礼物；灯笼下可发生告白级对白。
- **外派点（合作画廊）**：藤崎灯里常驻鉴识，带来另一套节奏。

每个场景的情感功能：初遇＝被塞进书记空位；升温＝固定时段续约整理目录；冲突＝家庭日程与拍卖周撞车、或被邻居看见共伞；和解＝雨天加开一场「修复鉴赏」；告白＝关馆后只剩一盏琥珀灯与未收的号牌。本条专属标语写在更衣室镜角：**先问状态，再碰槌柄。**

**【故事主线 · 情感线】**

**共通线·目录油墨未干**
你入职落札馆。主理**黒瀬璃华**把新印的目录拍在柜台上：「价写在纸上，人写在眼睛里。」你认识：书记前辈**白峰奏**、估价见习**朱里柚子**、常客**高城澪音**、外派鉴识**藤崎灯里**、后勤**御堂真白**、场内主持**一之瀬涼**。璃华的丈夫**风间透**是夜班电工，常在闭馆前后才回家。

三扇可选择的门：
1. 澪音爽约，璃华让你送「恢复目录」——只到玄关还是进门？
2. 灯里外派前夜多装一副杯碟邀你去河堤——灯很亮。
3. 有人要求越界「代拍强迫」——帮腔、沉默，还是停槌记黑？

共通线中段：奏在预约本多写一行＝表扬；柚子第一次独立估价时手抖，你扶住展柜而不是扶她的腰；涼把欢迎词喊到一半突然哑掉，你递水而不笑场；真白只在储物角才肯说「我其实怕槌声」。后段：闲话、透撞见送人、连锁店挖角、「拍卖＝可以暂时不做妻子／不做自己」的幻觉。终盘分线取决于：你是否在河堤承认越界、是否协助婚姻对话、是否选择离开、以及有没有把任何人的隐私写成拍品说明。

**黒瀬璃华线（核心）**
前银行柜员，婚后把旧行改成落札馆。爱透也恨其旅馆式回家；她把温柔都给了客人与展品，把自己活成永远的提供者。流程→茶水→河堤三段；HE 须对透诚实，可共同挂名或礼优先婚内修复；BE 秘密崩盘，她降为仅前台，关系冻结。萌点：落槌前会下意识擦槌柄；说谎时先整理目录角。

**藤崎灯里线**
两头跑的鉴识员，家中情感空缺，用工作填满不想回家的夜晚。外派日便当永远多一副杯碟。攻略：不把同情当交换，外场禁止把店内黑话当调情。HE：减少夜宿、建立稳定约会并协商婚姻；Bittersweet：独自整理，留下未用完的鉴识证与「谢谢你让我学会撤拍」。

**风间透（可互动配偶）**

**共通后日谈**
无论选谁，落札馆继续四季定期会；未攻略角色保持友好，不恶意拆家。True 氛围＝「一室多人、各得其所」或「一人专属私人鉴赏」——由玩家择。店规三条件写清：必须双方同意、可随时撤拍、禁止代约与强制加钟。

**微观日常事件池**
连锁挖角时璃华只回「我们慢一点」；雨天共伞谁先开口谁撑；黑名单写在签到本末页，退色蓝墨水；透家的猫曾闯入目録室，全员笑场十分钟；拒客后空气清一周；盛夏加会仍留一个体验空位；外派日灯里的便当多杯碟；关馆铃三响：收物、道别、留给还想说的人。目录不够、温控失灵、雨天屋顶湿滑、过敏报备、会员睡着打鼾、丈夫电话打来时的屏息、签到本被涂改——每一件都可转化为：谁先蹲下收拾、谁替谁圆场、谁在事后单独道谢。

**定期競売专属流程备忘**
1. 开场：前台核名 → 禁忌三问（过敏／疲劳／「今天可不可以举牌」）→ 选 定期ホール／目録室。
2. 中段：正规流程进行到一半，允许一次「暂停重来」而不扣费——璃华立下的馆规。
3. 收尾：甜羊羹、号牌归还、预约本只写自愿的下一格。
4. 本周特别项：禁止把「定期競売」写成空洞口号，只写成可完成的一件小事；遗失一件相关道具，当晚全员复盘，不罚钱只谈「谁在慌」。
5. 外派日：跟灯里走合作路线时，禁止把馆内黑话带到外场；违者由璃华停班一周（情感冷处理）。
6. 雨天预案：河堤关闭时改走落札後談話；共伞只送到站口，不送进玄关。
7. 月末复盘：每人说一句「这周我被谁记住了」，并追加一句与本周拍品相关的小事。

**结局方向**
- **共事 HE**：越界被刹住；璃华夫妻关系修复或体面调整；你保有尊严的书记位置。
- **分居后恋爱 HE**：璃华与透协议分开，你与璃华在阳光下交往。

目录厅的石柱还留着旧银行时期的编号，璃华从不刷掉——她说「让人记得这里曾经清点过别人的存款，现在清点的是自己的呼吸」。见习第一周，你会被要求把每本目录的折痕对齐到毫米级：不是洁癖，是训练「手在碰别人东西之前先稳」。朱里柚子第一次独立估价时，会把放大镜在掌心转三圈才敢开口；你若在她开口前抢答，好感会静默下降。白峰奏的清茶永远多泡一杯空位：那不是邀请你喝，是邀请你学会「为还没进门的人留位置」。一之瀬涼主持时的声线与私下说话完全是两个人——前者像铜管，后者像被布包住的铃；攻略她的关键不是拆穿，而是在关馆后问一句「现在用哪一副嗓子说话比较轻松」。御堂真白会把展柜绒布洗到发白，却不敢把自己的名字写进志愿者栏；当你把笔递给她而不是代她写，她会第一次直视你的眼睛超过两秒。高城澪音的号牌永远端正，紧张时却会把「可以撤拍」那一行摸到起毛——那是她的摩斯码。藤崎灯里外派归来时鞋底总有画廊的木屑，她会下意识在鞋柜前停顿，像把「外面的自己」脱下来；你若嘲笑这停顿，线会冻，你若把拖鞋尖朝外摆好，线会暖。风间透的安全帽凹痕来自一次没能修好的巷口灯——璃华因此开了拍场夜里的琥珀灯；这不是浪漫传说，是两人共同的失败纪念。灰市河堤在涨潮时会淹没最低一级台阶，告白若选在那一级，叙事应让浪声盖过台词一半，逼角色把话再说一遍——重复即诚实。月末复盘会上，若有人说不出「被谁记住」，允许沉默，但禁止用业绩数字代替；数字在这里是冷的，名字是热的。连锁店「クイック落」的店员有时会来偷师折目录，璃华会给对方一杯茶然后送客：她不怕被学走流程，怕被学走「慢」。黑名单用退色蓝墨水，是为了让除名也有被时间软化的可能——除非偷拍与伪造，那两项永不用退色墨。雨天共伞的规矩写在员工手册第 11 条：谁先开口谁撑伞，另一人负责看路；若两人同时开口，就一起淋到站口，笑场比湿衣更重要。目録室的门锁是旧式铜钥，钥匙只有璃华与当值书记各一把；任何「临时配一把」的请求都是红旗事件。茶会的甜羊羹按人数切，多出来的一角永远留给「今天说了停的人」——这是馆内唯一的奖励机制。你成长的刻度不写在等级表上，写在：你能在槌柄被握住时仍问出「现在可以吗」。

**追加名场面与因果**
- 柚子把主题色选成「雾灰」那周，全馆沉默了半天，直到奏说「雾灰是允许看不清」——全员才松气。

- True 线安装「可改主意」黄铜牌时，螺丝会拧歪一次，必须有人笑着重来：允许失败写进结局。


## 休闲切入点

> 本世界为休闲／拍场伦理恋爱向，无生存压力与比拼主轴。契约者以**日常身份**融入，核心玩法＝relationship 选择 + 边界经营，而非强制冲突推进。

切入身份：落札馆见习书记／目录整理员。自然融入拍场班底，不需要假冒魔王或超自然征服者。若玩家想走「周末市集临时帮手」支线，可在第二周被真白拜托铺绒布，仍属日常身份。

切入时点：定期競売周第二天——已学会禁忌三问，尚未发生河堤邀请；透仍高频夜班；澪音尚未爽约；连锁挖角信尚未到。最理想的第一幕落点是「目录油墨未干的早晨」或「第一次合法举牌示范的午后」。

初始处境：
- 工位近茶水角与保险柜；璃华直属
- 奏教你扫描旧目录；尚无私人加钟权
- 抽屉有空白号牌与退色蓝墨水
- 认识柚子与涼（点头之交）；未与透深谈
- 手机可有馆内群，没有澪音私聊——私聊靠事件解锁

开场白建议：「旧银行的石阶还留着昨夜雨渍。主理把新印的目录拍在柜台上，说价写在纸上，人写在眼睛里。定期ホール的转盘空着，号牌在抽屉里轻轻响。有人在河堤方向的海报下站了很久，像在练习撤拍。你的工作不是让谁被高价领走，而是让槌声落下之前，所有人仍能说停。」

可攻略对象：
- **黒瀬璃华**：从目录与禁忌三问的可靠下属做起；好感起点：礼貌距离；钩子：槌柄手帕、旧银行伤、河堤诚实；吃「尊重其停」，不吃「我比你丈夫懂你」
- **藤崎灯里**：从外派禁黑话与多杯碟开始；好感起点：同事信任；钩子：鉴识证与减夜宿
- **高城澪音**：从听说明与玄关投递开始；好感起点：紧张；钩子：「你有在听吗」
- **白峰奏**：从接受沉默开始；好感起点：警惕式关心
- **朱里柚子**：从交给主题色开始；好感起点：同辈玩笑
- **御堂真白／一之瀬涼**：角落与六十分钟国境线

日常玩法钩子：
1. **目录与号牌线**：每日折叠、涂改、归还；长期伪造记录降 True。
2. **合法互助线**：送恢复目录到玄关、雨天共伞到站口、市集采购——信任的合法积累。
3. **夫妻修复／分居分支**：推动对谈或体面分开，而不是单纯夺取。
4. **口碑与隐私线**：处理闲话、挖角、拒客副署。
5. **河堤诚实线**：离开门牌号的告白／拒绝／道歉。
6. **黄铜牌后日谈线**（True 后）：共同钉上「可改主意」。

氛围/雷区：克制潮湿的拍场伦理剧；允许成人向亲密，但必须清醒同意。忌迷奸偷拍、忌把 NTR 当无后果开后宫、忌跟踪透上下班。开局口诀：**先问状态，再碰槌柄；先能撤拍，再靠近。**

优先戏：目录拍桌、禁忌三问、玄关投递、河堤杯碟、茶会澄清、透的酒友夜；不是强制落槌、公开羞辱或破门。

在璃华（或任一对象）明示邀请并保持清醒之前，禁止进入目録室 Overnight 场景。

若选择伤害透尊严的炫耀式剧情，应导向 BE 而非奖励。

第一周内若连续三次「代人举牌且未二次确认」，奏线强制提前触发警告事件。

补充节奏：白天多写目录厅与市集建立合法共事感；夜间才允许目録室戏，且每次加钟后都要给「停／续／公开版」三选一。冷漠与克制是两回事——正文应通过奏或柚子的一句闲聊点破。
