# 触手遊園地-新アトラクション

## 剧情

**【作品来源】**
《触手遊園地-新アトラクション》为轮回乐园休闲库收录的**触手主题游乐园·新项目试运行与安全同意伦理**情景档案。清单标题点出「触手遊園地／新アトラクション」母题，**无与标题逐字对应的单一出版长篇**；气质整合 DLsite 同人圈常见的「触手遊園地」「アトラクション」「触手」系公开作品惯例——把触手写成可上下车的感官游乐设施，而非不可逃的猎食。可公开对照：夢みもち本舗《触手遊園地へようこそ★》（RJ369957，迷路少女须体验各触手设施才能离开）、DLsite「触手」题材页大量以安全词、粘液、旋转木马／咖啡杯为意象的短篇 CG／音声，以及成人向「主题乐园」设定中「必须先听说明再坐」的流程感。媒介印象：同人漫画／CG／短篇；搜笔趣阁以本条目标题检索无对应长篇小说书页。本档案以「新アトラクション」为专属锚点，写**试乘腕带、安全说明、急停按钮与闭园后是否还想牵手**，不写强制掠夺美化，不写数值闯关。

**【世界定位】**
海雾町临海的预约制主题园「触手乐园（テンタクル・パーク）」。契约者以**新项目试运行周见习安全员／票务协助**身份进入：园方本周上线「新アトラクション」试乘，员工与少量熟客一起调试**可叫停的共生护理菜单**——触手在此是可感知温度与压力的游乐媒介，不是战争兵器。一句话：安全带扣上之前，所有人都能下车。

**【世界观 · 舞台设定】**
时代为轻度奇幻外壳的当代海滨町：电车、盐味风、便利店关东煮与园内雾气并存。社会氛围开放而讲规则：海雾町允许「触手主题娱乐」挂牌，但卫生所与观光课要求公示急停、年龄限制与拒乘权。乐园不是地牢，也不是无规则猎场；它是预约制游乐场——进门核腕带、坐前听说明、中途可拍急停。

本世界**没有**强弱对决逻辑，也不用等级解释亲密。唯一的「规则」是乐园现实逻辑：
1. **任何触碰与缠绕前须有可观察同意**（腕带绿灯、口头确认、主动握住引导杆）；醉酒试乘无效。
2. **急停与「停」永远有效**：安全官会当场切断触手回路，不羞辱下车者。
3. **禁止偷拍／隐藏镜头／用粘液痕迹勒索**；发现即除名并通报町役场。
4. **配偶与家人不是背景板**：他们有名字、有疲惫、有自己的失败。

世界的温度来自：腕带内侧被磨亮的号码、安全说明板上被指尖点过无数次的「可拒否」三字、夜间整备库多泡的一壶茶、以及有人把「想再坐一次」写成「想被你记得我的呼吸」。治愈感靠重复：同一句「先问状态再扣带」、同一支荧光黄笔写的检修日志、同一盏只在急停解除后亮起的青灯。

**【地理 · 生活舞台】**
- **触手乐园本园**：霧見台站步行八分钟。入口票务、鞋柜、「新アトラクション試乗口」、私密调试区「設計室」、缓冲「夜間整備庫」、茶水角、储物角与临海露台。玻璃门贴「新アトラクション・少人数・可拒否」。
- **試乗口**：公开示范与主流程；呼吸与雾气交叠是升温主舞台。
- **設計室**：熟客加钟式「深度试乘」多发生于此；适合慢热告白与练习说不。
- **夜間整備庫／茶水角**：崩溃与和好；排班表与「本周粘液香型」贴在工具柜上。
- **海边栈桥**：会后散步与秘密短谈的默认舞台。
- **駅前连锁对照店「クイック霧」**：快、吵、没人记得你的禁忌——衬托乐园的「慢」。
- **各人自宅玄关**：上门送试乘报告——只到垫子。
- **商店街与周末市集**：采购备用绒套、季节花；灯笼下可告白。
- **合作诊所（外派点）**：春野理央常驻的材料实验室。

每个场景的情感功能：初遇＝被塞进安全员空位；升温＝固定班次续约调试；冲突＝家庭日程与试运行撞车、或被邻居看见共伞；和解＝雨天加开一场「修复试乘」；告白＝闭园后只剩一盏青灯与未收的腕带。专属标语写在更衣室镜角：**新アトラクション可以慢，不可以瞒。**

**【故事主线 · 情感线】**

**共通线·腕带还热**
你入职触手乐园新项目组。园长**七海穗乃**把试用腕带拍在桌上：「绿灯才能缠，红灯谁都得停。」你认识：安全官**青柳雫**、试乘员**月代杏**、设计**春野理央**、票务**泽渡美玖**、夜间整备**小森纱耶**。穗乃的配偶**外川诚**在邻港做轮机，常在闭园前后才回来。邻店咖啡师**岸本遥**常送雨天外带，顺便听闲话。

三扇门：
1. 杏爽约，穗乃让你送「恢复腕带」——玄关还是进门？
2. 理央外派前夜多装一壶茶邀你去栈桥——灯很亮。
3. 有人要求关掉急停「真・共生」——帮腔、沉默，还是拍停记黑？

中段：雫在检修日志多写一行＝表扬；美玖把欢迎声喊到一半自羞；纱耶只在整备库才说「我其实怕粘液味道」；杏在設計室突然落泪，你递毛巾而不追问。后段：闲话、诚撞见送人、连锁乐园挖角、「试乘＝可以暂时不做妻子」的幻觉。终盘：栈桥承认、协助沟通、离开、或泄密。

**时间节拍**
- 第 1 周：安全说明、腕带折叠、第一次合法试乘旁站。
- 第 2–3 周：续约调试、共伞到站口、茶会多杯。
- 第 4 周：栈桥邀请与上门只到玄关。
- 第 5–6 周：闲话／撞见／挖角／拒乘事件。
- 第 7 周后：闭园茶会分线。

**七海穗乃线**：前护士，婚后接手家族主题园并改造成触手乐园。爱诚也恨其旅馆式回家。流程→茶水→栈桥；HE 须对诚诚实；BE 秘密崩盘降为仅票务。萌点：扣带前会下意识数「三秒可悔」。

**春野理央线**：两头跑的材料设计，外派日永远多一壶茶。外场禁黑话。HE：减夜宿；Bittersweet：学会拍急停后离开。

**月代杏线**：把试乘当不会出错的功课，却总屏息。试探「你有在看仪表吗」。攻略：夸呼吸而非身体曲线。HE：细水长流；BE：闲话暂停试乘。

**青柳雫线**：话少因为被要求坚强；設計室会决堤。接受沉默。HE：晨间私人检修不看表。

**泽渡美玖线**：元气掩饰迷茫。请她定本周雾气歌单。HE：转正票务长，阳光恋爱。

**小森纱耶线**：害羞，镜子与粘液反光是敌人。从不展览她，等她主动点設計室。HE：栈桥第一次不躲眼神。

**外川诚**：轮机员，指节有机油纹。酒友可能；促成三人谈则 HE 互动；背刺则冷。

**岸本遥（配角）**：邻店耳目，不恋爱主轴，但是闲话与送伞的关键 NPC。

**共通后日谈**：乐园继续试运行转正；「共生护理菜单」写清三条件：双方同意、可随时急停、禁止代约与强制加钟。True＝一园多人各得其所，或一人专属夜场调试。

**微观事件池**
挖角时穗乃回「我们慢一点」；伞谁开口谁撑；黑名单荧光黄笔；诚家的猫闯入整备库笑场；拒乘后空气清一周；冬日加场仍留体验空位；关园铃三响。粘液不够、温控失灵、雨天栈桥湿滑、过敏报备、试乘员睡着打鼾、配偶电话时的屏息、腕带被涂改——谁先蹲下收拾，谁就多一分被信任。

**新アトラクション专属流程**
1. 核名 → 禁忌三问（过敏／疲劳／「今天可不可以缠」）→ 选 試乗口／設計室。
2. 中段允许一次「暂停重来」不扣费——穗乃园规。
3. 收尾：清茶试饮、腕带归还、日志只写自愿下一格。
4. 本周特别项：禁止把新项目写成口号，只写成可完成的一件小事；遗失急停罩当晚复盘。
5. 外派日跟理央：禁止把园内黑话带到实验室；违者停班一周。
6. 雨天：栈桥关闭改整备库长谈；共伞只到站口。
7. 月末：每人说「这周我被谁记住了」+ 一句与新项目相关的小事。

**结局**：共事／分居恋爱／搬离／BE／True（急停三条件写进园规牌）。

**【隐藏剧情 · 真结局 · 伏笔】**
- 穗乃护士时期因「过度照护」被调岗——开乐园是为了把照护变成可上下车的娱乐。

- 连锁乐园报价想去掉「可拒否」字样，True 必须公开拒签。
- True：零偷拍、至少一次公开拒乘副署、至少一次协助婚姻对话。
- 后日谈：入口多一块「可改主意」青牌；诚学会说「今晚回得早」；你把备用急停罩放在显眼处。

**【氛围基调 · 雷区】**
海雾、青灯、粘液反光与栈桥风声；克制成人向。忌强制缠绕、偷拍、纸片配偶、以「乐园氛围」豁免同意。忌把触手写成比拼破坏力的兵器、忌任务厮杀。切入：新项目试运行周第二天，腕带油墨未干。


**【细节长卷·触手乐园肌理】**
海雾町的风总把入口旗吹成侧影，七海穗乃说那是「提醒游客：你看见的永远只是侧面」。见习安全员第一周要背急停罩的三种故障音：咔、嘶、闷——分别对应机械、回路、人为遮挡。青柳雫会在你背错时不骂，只把荧光背心拉链拉到顶，意思是「今天你还不配松」。月代杏的试乘服内侧用记号笔写「可拒否」，墨水被洗淡了她会重写，从不嫌烦；你若觉得这幼稚，线会冷。春野理央的样品袋里永远有失败的粘液配方，她当众展示失败品而不是成功品——这是她的信任测试。泽渡美玖嗓门一亮就会耳根红，票务广播她录了七次才过关；你若拿去当笑话，她会把你的班次调到最晒的入口。小森纱耶怕粘液味道，却把夜间整备做得最干净：怕与专业可以并存。外川诚的机油纹洗不掉，他握杯时会下意识躲开白桌布——穗乃因此在员工休息室铺了深色桌布，这细节比任何情话更像婚姻。岸本遥的雨伞永远多一把，却从不问你要去谁家；她是舆论的过滤器，不是传声筒。試乗口的雾气香型每周轮换，过敏报备表贴在最显眼处；有人想「香型也是情趣」强行关闭报备，穗乃会当场拒乘并记黑。設計室的门上有两道锁：一道机械，一道「口头二次确认」——第二道才是真正的锁。急停键的高度按最矮员工手指设计，手册写明「尊严不因身高打折」。闭园后的青灯只亮在急停解除之后，意味着「今晚我们都安全下了车」。栈桥木板在雨后会翘起一角，告白若踩上去，应先让对方站稳再说话——身体安全先于台词漂亮。月末复盘不比客流，只比「有多少人中途拍停仍被温柔对待」。连锁乐园「クイック霧」的海报写着「更快更刺激」，穗乃的回应永远是多留一个体验空位给「第一次说停的人」。你的成长刻度是：能否在全场起哄「继续」时，仍然把急停键按下去。

**追加名场面**
- 杏第一次中途拍停，全场没有嘘声，只有雫的一句「漂亮」。
- 理央把失败粘液倒进展示柜，写「今天也没成功」——穗乃加了一行「所以明天还来」。
- 美玖广播念错「可拒否」被自己笑场，你若帮她重录而不发群，她会请你喝最苦的咖啡表示信任。
- 诚说「她在病房也会数三秒」时，不要抢答「我比你懂」，要问「你希望她现在还数吗」。
- True 线钉青牌时海风会吹掉一张图纸，必须有人去追：追回的不是纸，是「还可以重来」。



**【角色日常切片（非循环）】**
七海穗乃晨会只讲三件事：过敏、急停、今天谁可以说不。她的白大褂口袋永远有两支笔，一支写检修，一支写人情——两支笔不能混用。青柳雫午休在整备库吃饭团，从不边吃边看仪表，说「嘴和眼不能同时对机器撒谎」。月代杏试乘前会在洗手间对着镜子做三次深呼吸，第三口最长；你若在门外计时并笑话她，线冷，若递温水不说话，线暖。春野理央的失败粘液柜有日期标签，像日记；她允许你闻，不允许你评价「恶心」。泽渡美玖的广播室贴满便利贴，全是「可拒否」的不同读法练习。小森纱耶夜班结束会把手套翻面晾干，像把恐惧也晾干。外川诚偶尔带港口的盐饼来，穗乃收下却不在游客面前吃——婚姻的私密零食。岸本遥送伞时会把伞柄朝向接伞的人，从不朝向自己：细节即性格。

**冲突升级阶梯（写作用，非数值）**
1. 闲话：商店街「乐园员工是不是也……」——用公开茶会澄清。
