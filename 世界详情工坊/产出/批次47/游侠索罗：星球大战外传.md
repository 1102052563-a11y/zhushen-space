# 游侠索罗：星球大战外传
<!--meta lib=主库 tiers=三-->

## 剧情

**【作品来源】**
《游侠索罗：星球大战外传》（Solo: A Star Wars Story；台译《星际大战外传：韩索罗》，港译《韩索罗：星球大战外传》）为2018年美国太空歌剧冒险电影，卢卡斯影业／迪士尼发行；导演朗·霍华德（中途接替菲尔·洛德与克里斯托弗·米勒），编剧劳伦斯·卡斯丹、强·卡斯丹。属《星球大战》外传系列第二部，讲述韩·索罗青年时代如何遇见丘巴卡、蓝道·卡利西安并赢得千年隼，以及科舍尔航道十二秒差距传说的由来。文风：亡命徒公路片＋火车劫案＋赌场老千＋帝国扩张期的灰黑江湖。

**【世界定位】**
帝国治下，科雷利亚造船世界的孤儿靠偷窃求生。韩与恋人奇拉窃取超光速燃料科亚烯（Coaxium）意图逃跑，奇拉被捕后韩加入帝国又被逐，在明坂战场结识罪犯托拜厄斯·贝克特与伍基族丘巴卡，卷入赤色黎明犯罪辛迪加的科亚烯劫案，最终以赌博赢得千年隼并驶向塔图因。一句话：**姓氏是招募官随口一叫，传说是一次次背叛里活下来的航行日志。** 本条目仅覆盖**三阶**——犯罪团伙核心、精锐雇佣与据点级劫案带。

**【世界观 · 力量体系】**
无修仙体系。战力＝枪法＋驾驶＋欺诈＋人脉＋偶尔原力边缘（本片达斯·摩尔远程现身属顶点警示）。死亡真实；帝国征兵与犯罪辛迪加比原力更常见地决定生死。科亚烯未精炼时极不稳定，本身即战略爆炸物。

战力阶梯：
1. 科雷利亚街童／苦力≈一阶（参照）。
2. 帝国步兵／底层船员≈二阶（参照）。
3. **贝克特团伙核心、精锐走私者、云骑士精锐、韩与丘巴卡任务态≈三阶（覆盖）**：可劫帝国列车、渗透矿星、在混战中夺取货仓。
4. 蓝道级赌徒船长、赤色黎明高管德莱登·沃斯≈四～五阶指挥／资源级。
5. 歼星舰与舰队≈六阶；达斯·摩尔（赤色黎明幕后）≈七阶至超阶边缘。

**乐园阶位映射**：街童≈一阶；帝国步兵≈二阶；**劫案核心与王牌飞行员雏形≈三阶**；辛迪加高层与传奇船长≈四五阶；舰队≈六阶；摩尔＝超阶边缘，情报优先。

**【地理 · 舞台】**
科雷利亚：造船与黑帮，韩与奇拉故乡。明坂：泥泞战区，韩遇贝克特与丘巴卡。凡道一号：帝国科亚烯列车劫案地。沃斯游艇／赤色黎明：奢靡犯罪宫廷。凯瑟矿星：奴隶与未精炼科亚烯。科舍尔航道：星团、引力涡、巨型触手生物，十二秒差距捷径。萨弗林：精炼与背叛舞台。塔图因：结局航向，贾巴线预告。

**【世界剧情线】**
**① 科雷利亚出逃**：韩与奇拉偷科亚烯贿赂海关，奇拉被抓，韩发誓回来救她；无姓，招募官赐姓「Solo」。**② 帝国三年**：军校抗命被逐，明坂当步兵；胁迫加入贝克特团伙，与笼中丘巴卡结盟逃亡。**③ 列车劫案**：凡道一号抢科亚烯，云骑士恩菲斯·内斯特截胡，队友里奥·杜兰特与瓦尔阵亡，货毁。**④ 赤色黎明**：贝克特向德莱登·沃斯交代；韩重逢已成沃斯左右手的奇拉；提议凯瑟盗未精炼科亚烯。**⑤ 蓝道与千年隼**：牌局萨巴克，蓝道作弊赢船但仍欠人情出任务；L3-37机器人解放矿星奴隶引发混乱；丘巴卡解救同族；撤离时L3重创，核心并入导航。**⑥ 科舍尔十二秒差距**：韩驾驶穿星团，十二秒差距完成凯瑟航道，稳定科亚烯。**⑦ 萨弗林背叛链**：云骑士实为反抗军雏形；贝克特出卖韩；奇拉杀沃斯却向达斯·摩尔谎报；韩先开枪击毙贝克特；奇拉远去；韩丘送科亚烯给恩菲斯；再赌赢千年隼，航向塔图因。

**【主要人物】**
韩·索罗｜性格：嘴硬重诺、赌徒胆｜能力：驾驶、枪法、临场｜弧光：孤儿→士兵→走私雏形。
丘巴卡｜性格：义气｜能力：武力、维修、伍基怒｜弧光：囚徒→大副。
奇拉｜性格：生存理性｜能力：社交工程、枪｜弧光：恋人→赤色黎明→摩尔线。
托拜厄斯·贝克特｜性格：导师型叛徒｜能力：策划劫案｜弧光：引路人→死于韩枪下。
蓝道·卡利西安｜性格：风流老千｜能力：牌技、魅力、船｜弧光：失隼→仍周旋。
L3-37｜性格：权利宣言机器人｜能力：导航全图｜弧光：解放→牺牲→成为船的一部分。
德莱登·沃斯｜性格：优雅残暴｜能力：赤色黎明资源｜弧光：雇主→被奇拉所杀。
恩菲斯·内斯特｜性格：面具下的理想｜能力：云骑士指挥｜弧光：海盗→反抗火种。
达斯·摩尔｜性格：黑暗残存｜能力：原力、犯罪帝国｜顶点。
里奥·杜兰特、瓦尔：贝克特队牺牲者。

**【势力图谱】**帝国征兵与占领；赤色黎明（摩尔遥控）；云骑士／早期反抗资源链；贝克特自由犯罪小组；科雷利亚黑帮摩洛克线。

**【贵重物品】**科亚烯（精炼/未精炼）；千年隼；萨巴克牌组与作弊装置；L3导航核心；贝克特的「先开枪」哲学；恩菲斯面具；沃斯权杖与游艇。

**【隐藏剧情 · 伏笔】**奇拉—摩尔线接动画《反抗军组织》等正史；十二秒差距成为韩的招牌吹嘘；L3成隼的灵魂；贝克特临死肯定韩的选择；塔图因预告贾巴。

**【大事记时间线】**科雷利亚出逃→帝国→明坂→列车失败→沃斯→凯瑟→科舍尔→萨弗林背叛→赢得千年隼→塔图因。

**【叙事基调 · 雷区】**亡命徒成长而非绝地神话。忌把青年韩写成无敌；忌奇拉单纯白莲；忌击杀摩尔；忌否定背叛的痛感。最早切入：明坂入伙或列车劫案前夜。本条目仅三阶。


**【战役细节档案·斯卡里夫立体分层】**
轨道层：同盟舰队与帝国护盾门、歼星舰对射；拉达斯将军的指挥风格是「用舰体换窗口」。护盾门关闭意味着地面全灭，因此地面必须先夺通讯再求轨道砸门。大气层：运输船闯门、TIE与X翼缠斗、爆破点选择。地面层：堤道、碉堡、AT-ACT步行机、数据塔电梯与档案库。数据库内：标题检索「星尘」、重力井式档案架、K-2SO单机防守走廊。海滩：义军以血肉吸引火力，席鲁穿越弹雨的镜头是本片原力主题的凡人版——他不是绝地，但信原力。塔顶：图纸发射、克伦尼克的白披风沾血、死星碟影挡住阳光。海边：琴与卡西安的沉默比口号更像胜利。

**【制度与灰区】**
同盟暗杀令证明「反抗军也会做帝国做过的事」；索的酷刑室证明「极端反抗可变成另一种压迫」；塔金用礼貌语气接收军权，展示帝国精英的官僚暴力。契约者若只想当英雄，会被这套制度吞掉；若只想当逃兵，会错过磁带。

**【人物微表情与关系补充】**
琴对「星尘」的反应是父爱触发器；卡西安看琴的眼神从评估资产变成评估同伴；博迪重复「我是飞行员」是用身份压住恐惧；席鲁摸手杖确认方向时巴兹总会在侧后方；克伦尼克对维达的恐惧比对塔金的嫉妒更真实；莱娅接过磁带时的「希望」不是乐观，是责任。

**【可锚定的小时序】**
T-72h 杰达巷战；T-48h 圣城毁灭；T-36h 伊杜轰炸；T-24h 雅文辩论；T-12h 私自发舰；T-6h 闯护盾；T-2h 数据库战；T-0 图纸离星；T+短 维达屠杀；T+衔接 莱娅航向塔图因。



**【工程伦理与「星尘」】**
盖伦无法阻止死星建成，只能选择把毁灭变成「可被勇敢者关掉的机器」。他把弱点藏在反应堆模块的结构共振里，使精确打击可连锁。帝国质检看数字，看不懂一个父亲的语法。琴在数据库里输入「星尘」的瞬间，工程、父爱与军事目标重合为同一个词。

**【与正典的咬合点】**
本片结束即《新希望》开始：磁带→莱娅→塔图因→卢克。任何改写若让图纸灭失，则正史希望线断裂；若让维达被击退，则西斯压迫感崩坏。契约者最优贡献是「冗余备份」与「多送出一秒钟」。

**【环境伤害表（三阶相关）】**
爆能枪：一至二发可致命未着甲目标。手雷：掩体后仍可能破片。AT-ACT：踩踏与主炮覆盖堤道。护盾门：能量墙，非门禁卡不可过。死星低功率：城市级蒸发，无个人抗性。维达：原力扼喉与光剑清走廊，三阶单位接触即任务失败条件。
