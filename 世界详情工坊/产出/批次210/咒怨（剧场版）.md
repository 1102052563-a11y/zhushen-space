# 咒怨（剧场版）
<!--meta lib=主库 tiers=一、二、三、四、五-->

## 剧情

**【作品来源】**
《咒怨》（日语：呪怨，英译 Ju-On: The Grudge）是清水崇编剧并执导的日本超自然恐怖片，2002年10月18日于美国 Screamfest 首映，2003年1月25日日本院线上映。它是《咒怨》系列第三部、首部院线长片（前两部为录像带直发《惊恐极限OVA》与续作）。主演奥菜惠（西奈理佳）、伊东美咲（德永仁美）、市川由衣、藤贵子（佐伯伽椰子）、尾关优哉（佐伯俊雄）、松山鹰志（佐伯刚雄）。文风为非线性叙事的 J-Horror：静默、日常崩坏、诅咒不可协商。续作有《咒怨2》、美版《不死咒怨》系列、剧集《咒怨之始》等。本档案锚定**2002剧场版主线与诅咒规则**，服务轮回乐园常青读取。

**【世界定位】**
东京练马区一栋普通民宅里，因家庭灭门诞生的「咒怨」会把任何踏入者拖入死亡循环；社工西奈理佳与警察、居民、少女们在非线性时间片中被诅咒逐一吞噬。一句话：没有英雄逆转的怨灵规则世界，契约者介入等于主动报名进咒。

**【世界观 · 力量体系】**
本世界几乎无「修炼境界」，力量来源是**怨念固化为诅咒法则**：人在极度愤怒与恐惧中死去时，其死所会残留「咒」，再吸引并杀死后继进入者，死亡地点又成为新的咒源，呈病毒式扩散。怨灵可短暂附身、模仿生者面容、扭曲时间感知，使目击者看见「未来惨剧的预兆」。凡人无法以热武器、符咒速解（剧场版内无成功净化结局）。

等级与表现（按破坏力/威胁落位，宁低勿高）：
1. **普通人 / 未染咒**：日常通勤、社工、警员。乐园观感≈一阶初期。
2. **轻度接触者**：听见猫叫、看见苍白孩童、短暂幻觉。精神崩坏前仍是凡人。≈一阶中后期。
3. **中度染咒**：被跟踪到职场/公寓，周围开始出现关联死亡。≈二阶。
4. **重度染咒 / 濒死**：身体被拖入死所、影像眼睛被涂黑、重演伽椰子之死。≈二～三阶。
5. **怨灵·佐伯俊雄**：孩童形态、猫叫、诱引生者上楼。个体破坏力有限，但不可常规杀死。≈三阶。
6. **怨灵·佐伯刚雄**：附身男性、诱发暴力与嫉妒。≈三～四阶。
7. **怨灵·佐伯伽椰子**：长发遮面、爬行、咯咯死音；可跨地点追杀、改写染咒者命运为「重演自己之死」。领域感覆盖整栋佐伯宅及染咒者死亡点。≈四～五阶（城市级精神恐怖，非物理屠城）。
8. **咒源·佐伯宅本身**：进入即计时的规则杀场；烧毁亦难根绝（远山未遂）。≈五阶规则级节点。

乐园阶位映射：普通人≈一阶；轻度接触≈一～二阶；染咒追杀阶段≈二～三阶；俊雄/刚雄≈三阶；伽椰子≈四～五阶；佐伯宅咒源≈五阶规则节点。顶点=佐伯伽椰子+咒源宅（五阶顶，非行星级）；低阶切入一律规避顶点·情报优先。宁低勿高。

**【地理 · 舞台】**
主舞台为**东京都练马区佐伯宅**（剧中诅咒核心）：二层民宅、阁楼藏尸、楼梯栏杆（俊雄常观望处）、壁橱与走廊是惊吓动线。关联扩展点包括：德永家居住期、德永胜也的公司与妹妹仁美的公寓、西奈理佳所属福利事务所、小学（真理子家访路线）、远山家（退休警探隐居处）、东京街头失踪人口海报铺地的末世意象。势力无帮派战，只有「宅内咒源—染咒者—警察/社工」三角。AI写场景先定契约者是否已踏入佐伯宅或接触染咒者；未入宅可走外围调查，入宅则进入高危规则杀。

**【世界剧情线】**
① **咒源起源（主线前数年）**  
佐伯刚雄发现妻子佐伯伽椰子暗恋他人后，在宅内杀害伽椰子、家猫マー与幼子佐伯俊雄。惨剧凝成「咒怨」：全家人以怨灵形态复现，伽椰子之灵反杀刚雄。此后任何进入该宅者终将被吞噬；被吞噬者死亡之处又成为新的咒，继续感染后继者。此阶段世界状态：一栋看似普通的练马民宅已是永久杀场。

② **德永一家入住**  
最新住户为上班族德永胜也、妻子德永和美、病母德永幸枝。和美迅速被咒吞噬；胜也受刚雄怨灵情绪感染后亦死。伽椰子之灵追随胜也的妹妹德永仁美至其公司，杀死保安，再至仁美公寓将其杀害。名场面：日常家庭空间在无预警下变成停尸场，尸体后被发现于阁楼。

③ **社工西奈理佳入宅**  
上司广桥派社工西奈理佳上门照护幸枝。理佳发现俊雄，并目睹幸枝被伽椰子杀死，惊吓昏厥。广桥寻至并报警。警探中川与五十岚在阁楼发现胜也、和美尸体，随后串联仁美失踪与公司保安死亡。广桥本人亦死，理佳开始被怨灵纠缠。因果：调查链条把警察、社工一并拖入咒网。

④ **警探追查与远山线**  
中川、五十岚查阅佐伯宅旧案后联系退休警探远山。远山恐惧重提旧案，却仍前往试图焚宅；他听见楼上少女群声——实为未来预兆：远山之女远山逸美（小泉）成年后与友人入宅，仅她逃走而友人被杀。伽椰子追杀中，中川与五十岚死，远山遁为隐居者，最终仍死于咒，留下幼女。名场面：时间线折叠，现在的远山「看见」未来的女儿惨剧。

⑤ **真理子家访与理佳之死**  
理佳试图回归日常。友人、小学教师浮气真理子因学生「俊雄」从不出席而家访佐伯宅。理佳赶来已迟。伽椰子现身，理佳短暂看见自己变成伽椰子的面容，意识到自己将重演伽椰子的死法。俊雄在楼梯栏杆观望时，刚雄之灵下楼以同样方式杀死理佳。新闻后确认理佳尸体于佐伯宅被发现。

⑥ **远山逸美线与终局意象**  
青年逸美与友人入宅，她逃走、友人被杀（对应远山当年幻视）。两周后友人送照片，新闻播报理佳遗体；照片中逸美与已死友人眼睛被涂黑。逸美因抛弃友人而偏执，母亲亦受咒影响。友人离去后，逸美见亡父幻象与友人怨灵，终被伽椰子拖入诅咒。几乎荒废的东京街面散落失踪海报；理佳尸体在阁楼以伽椰子式长发姿态「醒来」发出死亡咯咯声——诅咒未灭、循环继续。

⑦ **叙事结构说明（给 AI）**  
原片以章节式非线性呈现，上列按因果时序整理。任意时点世界状态可判为：①咒源已成立；②德永家覆灭中/后；③理佳调查期；④警探与远山介入；⑤理佳死；⑥逸美线收束。结局方向：**无人净化成功，诅咒扩散，都市失踪潮意象收尾**。

**【主要人物】**
西奈理佳（社工·视点主角）｜性格：尽责、共情、面对超自然时仍试图救人｜装备·能力：无异能，仅社工权限与勇气｜人物弧光：从照护老人到意识到自己将重演伽椰子之死｜立场关系：受广桥派遣；与真理子为友；被伽椰子锁定。

佐伯伽椰子（咒源核心怨灵）｜性格：怨念凝结后无谈判、无怜悯｜装备·能力：爬行追杀、面容置换、跨地点现身、死音｜人物弧光：被害妻子→不可灭的咒｜立场关系：杀刚雄；吞噬一切入宅者；令染咒者重演己死。

佐伯俊雄（孩童怨灵）｜性格：静默诱引｜装备·能力：猫叫、现身诱导上楼｜人物弧光：被害幼子→咒的「诱饵」｜立场关系：常与伽椰子联动。

佐伯刚雄（施暴者·怨灵）｜性格：嫉妒、暴力｜装备·能力：附身诱发杀意｜人物弧光：灭门凶手→被伽椰子反杀的怨灵｜立场关系：对德永胜也等男性施加影响。

德永胜也（新住户）｜性格：上班族日常感｜装备·能力：无｜人物弧光：入住→被咒与刚雄影响而死｜立场关系：和美之夫、仁美之兄、幸枝之子。

德永和美（新住户）｜性格：家庭主妇｜装备·能力：无｜人物弧光：最早被宅吞噬者之一｜立场关系：胜也之妻。

德永幸枝（病母）｜性格：体弱｜装备·能力：无｜人物弧光：理佳照护对象，死于伽椰子｜立场关系：德永家。

德永仁美（胜也之妹）｜性格：职场女性｜装备·能力：无｜人物弧光：被咒追至公司与公寓而死｜立场关系：咒扩散的关键节点。

广桥（福利事务所上司）｜性格：公务式｜装备·能力：无｜人物弧光：派理佳入宅后自身亦死｜立场关系：理佳上司。

中川（警探）｜性格：尽职调查｜装备·能力：警职｜人物弧光：发现阁楼尸体→联系远山→死于咒｜立场关系：与五十岚搭档。

五十岚（警探）｜性格：配合追查｜装备·能力：警职｜人物弧光：同中川｜立场关系：搭档。

远山（退休警探）｜性格：恐惧旧案、仍有责任感｜装备·能力：旧案记忆｜人物弧光：试图焚宅失败→隐居→死于咒｜立场关系：逸美之父。

远山逸美（小泉·远山之女）｜性格：愧疚、偏执｜装备·能力：无｜人物弧光：弃友逃走→照片涂黑→被拖入咒｜立场关系：连接「未来预兆」的关键。

浮气真理子（小学教师）｜性格：关心学生｜装备·能力：无｜人物弧光：家访俊雄→死｜立场关系：理佳之友。

**【势力图谱】**
- **佐伯宅咒源**：无组织形态，规则即法律；代表=伽椰子/俊雄/刚雄；战力=四～五阶规则杀。
- **东京都警相关（中川、五十岚、远山）**：调查旧案与失踪；凡人体系，无法对抗怨灵。
- **福利事务所（广桥、理佳）**：照护入口，无意中把新人送入咒。
- **德永家 / 远山家 / 逸美友人圈**：染咒扩散的社会单元。
敌友关系不适用正邪阵营，只有「未染—染咒—已死」。

**【贵重物品】**
- **佐伯宅本身**：最大「道具」，进入即契约；烧毁未遂。
- **阁楼尸骸与旧案档案**：警察调查核心物证。
- **被涂黑眼睛的照片**：逸美线诅咒显形媒介。
- **失踪人口海报**：终局都市扩散意象。
- **家猫マー**：起源惨剧元素（非可交易宝物）。
本世界几乎无传统神器；契约者若「搜刮」，更可能带走染咒而非装备。

**【隐藏剧情 · 伏笔】**
非线性时间使「现在」可见「未来」：远山焚宅时所闻少女声即逸美友人线。理佳看见自己成为伽椰子，揭示诅咒的**角色置换**机制——受害者可被改写为咒的新载体。系列互文：录像带前作、美版《The Grudge》、剧集《咒怨之始》可作旁支，但本档案以2002剧场版终局为准：诅咒未解。

**【大事记时间线】**
佐伯灭门 → 咒源成立 → 德永家入住覆灭 → 仁美职场/公寓被杀 → 理佳入宅见幸枝死 → 警探发现阁楼尸 → 远山焚宅未遂、警探死 → 真理子家访死、理佳被刚雄以伽椰子死法杀害 → 逸美入宅弃友 → 理佳遗体新闻 → 逸美被拖入咒 → 街头失踪海报与阁楼「醒来」。

**【叙事基调 · 雷区】**
基调：静、冷、日常感突然断裂；忌写成可打怪升级的驱魔爽文。忌把伽椰子写成可恋爱攻略对象或「被封印就没战力」。忌给理佳强行圆满净化结局（原作无此）。忌热血喊话驱散怨念。最早可切入锚点：德永家入住前后或理佳被派往佐伯宅前夜。

**【世界剧情线·德永宅日记式复盘】**
德永胜也作为上班族把母亲幸枝接来同住，本是孝道日常。和美最先感知「家里多了一个孩子」：冰箱门自开、猫砂盆无人却有抓痕、二楼传来不应存在的脚步。她试图告诉胜也，却被当作产后或压力幻觉。胜也加班归来时，刚雄的怨念开始渗透——剧本呈现为男性对控制权的病态执着：他开始对和美歇斯底里，又在片刻后茫然。和美消失后，胜也的死亡被处理成「阁楼静物」，直到警察撬开盖板。仁美接到兄长异常电话赶往公司，保安在走廊被无形之物扭断常识；仁美逃回公寓，电视雪花里爬出长发。这一支线证明：咒不限于宅基地，**染咒者走到哪，哪就可能成为新的死所**。契约者若在仁美公司层活动，必须假设整层楼已进入感染倒计时。

**【世界剧情线·警察程序如何失效】**
中川与五十岚的调查路径完全符合写实刑侦：失踪人口、双重命案、社工昏迷、上司横死。他们调出佐伯宅旧档，发现多年前的灭门以「狂夫杀妻儿」结案，凶手刚雄亦死于现场。逻辑上应结案，情感上远山拒绝重提。当两位警探把远山拖回现场，规则杀再次覆盖公权力——枪、对讲机、增援都来不及改写「进入即被写进诅咒剧本」。远山的焚宅是全片最接近「主动攻击咒源」的行为，却被未来少女的笑声打断：时间在宅内不是直线。AI描写警察线时，应强调**程序正义对规则杀无效**，而不是警察无能笑话。

**【世界剧情线·理佳的职业伦理陷阱】**
西奈理佳的悲剧核心不是「好奇心」，而是社工伦理：失联老人必须探视、儿童户籍必须核对、同事有难必须支援。广桥把最难的单派给她，办公室里还有未喝完的茶。理佳在宅内发现俊雄时，第一反应是保护儿童而非逃命——这让她看见幸枝被杀。苏醒后她仍去上班、仍与真理子见面，试图用日常覆盖创伤，直到真理子说出「我有个学生叫俊雄」。姓名重合把职业伦理变成催命符。理佳冲向佐伯宅的奔跑，是全片最接近英雄行为的段落，结局却是身份被改写：她在幻觉中成为伽椰子，被刚雄以相同方式杀死。契约者若扮演「更专业的社工」，不能靠勇气通关，只能靠**拒绝单独入宅、双人探视、实时定位**等反规则SOP。

**【世界剧情线·逸美愧疚机制】**
远山逸美（小泉）线解释诅咒如何利用「生者的愧疚」。她与友人入宅时逃跑，友人死；照片眼睛被涂黑是视觉化的「你已被勾销」。亡父幻象不是安慰而是牵引。友人怨灵与伽椰子联手，完成对「抛弃者」的处刑。新闻里理佳遗体被发现，把两条时间线钉在同一舆论场。终局街头失踪海报铺地，暗示咒已从一宅扩散为都市病。阁楼中理佳长发遮面「醒来」的咯咯声，是系列标志性音效的剧场版定格：循环没有结束，只是观众停在最黑的一帧。

**【力量体系·怨灵行为学细则】**
俊雄：低攻击、高诱引，负责把生者带向楼梯、阁楼、壁橱等「死法舞台」。刚雄：对男性角色施加嫉妒与暴力脚本，使其重复自己的灭门逻辑。伽椰子：全图追杀、面容置换、跨场景闪现，死亡演出常伴随爬行与死音。咒的传播条件公开资料强调「进入死所/接触染咒者」，不是随便看见照片就死（逸美照片是已染咒后的显形）。净化手段在本片内**未成功演示**；美版与续作另有不同结局，本世界卡以2002日版为准。

**【地理·练马民宅的空间语法】**

危险度/规避：中。严禁深入阁楼；规避**佐伯伽椰子**正面接触。
任务方向/奖励：情报、社工证件、邻里人脉；不发「驱魔神器」。

**二阶（轻度接触·警探协力线）**
切入身份/时点：以私家侦探/记者/警局临时翻译身份切入，锚定中川、五十岚发现阁楼尸体前后。
初始事件：你被请去辨认阁楼两具尸体的身份，录音笔自动录下猫叫；撤离时照片里多出一个长发女人的残影。
开场白建议：「法医帐篷外，中川把烟掐灭。他说阁楼很挤，却像有人刚爬过。你的相机自动对焦到楼梯栏杆——那里空无一人，取景框却闪过苍白孩童的脸。」
关键NPC立场：**中川**（要线索）；**五十岚**（警惕外人）；**远山**（拒谈旧案但眼神恐惧）；**德永仁美**（若仍存活则已偏执）。
主线钩子/支线：说服远山交出旧案笔记；支线=护送仁美离开公寓窗口；支线=查广桥死亡现场是否成新咒源。
危险度/规避：高。规避夜宿任何染咒公寓；勿焚宅蛮干。
任务方向/奖励：旧案档案副本、生存向护身习惯（非神功）。

**三阶（染咒边缘·理佳同期）**
切入身份/时点：以理佳的同事或大学同期友人切入，锚定真理子决定家访「俊雄」前后。
初始事件：真理子约你同行家访；你在校名簿上看到俊雄住址正是佐伯宅，理佳电话里只剩断续呼吸声。
开场白建议：「校门口的风把真理子的教案吹乱。她说那个孩子从没来上课，可户籍上他住在练马。你的手机震了一下——理佳发来一张楼梯照片，栏杆后有一双童鞋，没有脚。」
关键NPC立场：**浮气真理子**（坚持家访）；**西奈理佳**（已染咒，想救人）；**佐伯俊雄**（诱引）；**佐伯伽椰子**（猎食者）。
主线钩子/支线：拦截真理子入宅；支线=帮理佳确认「面容置换」幻觉含义；支线=转移逸美童年期远离宅。
危险度/规避：极高。正面硬刚伽椰子=送死；情报优先。
任务方向/奖励：短暂拖延连锁死亡、锁定咒扩散地图。

**四阶（可周旋·远山焚宅夜）**
切入身份/时点：以远山旧识的民俗顾问/退役特殊部队医疗兵切入，锚定远山决定烧宅之夜。
初始事件：汽油味混着少女笑声从二楼传来；中川要求你掩护，五十岚已在楼梯口失联。
开场白建议：「远山的手在抖，火柴却划得很稳。他说这房子不该存在。楼上忽然响起好几个少女的嬉闹——你记得档案里，那些女孩要在几年后才失踪。」
关键NPC立场：**远山**（求毁宅）；**中川/五十岚**（执法但无力）；**远山逸美**（未来线投影）；**佐伯伽椰子**（超阶压迫感）。
主线钩子/支线：是否点火（可能加速咒怒）；支线=救出「未来声」中的一个投影体；支线=保护逸美幼年。
危险度/规避：极高。超阶巅峰：伽椰子·情报优先/条件性胜利（切断入宅链、隔离染咒者），严禁写「封印后变弱」。
任务方向/奖励：时间线扰动情报、隔离协议模板。

**五阶（贴近咒源·终局循环）**
切入身份/时点：以「已理解规则仍选择入局」的猎奇研究者/乐园老兵切入，锚定理佳死后、逸美被拖入咒前后。
初始事件：东京街头失踪海报铺地；你在阁楼看见理佳长发遮面的尸体睁眼，发出伽椰子的咯咯声。
开场白建议：「新闻里理佳的名字被念错一次。阁楼木板下陷，你与一具‘不该醒来’的尸体对视。她的头发比死亡证明上的照片更长。猫叫从地板下升起，像整座城市的喉咙。」
关键NPC立场：**佐伯伽椰子**（顶点）；**西奈理佳**（已成咒载体意象）；**远山逸美**（愧疚崩溃）；**佐伯俊雄**（旁观）。
主线钩子/支线：尝试建立「未入宅者隔离带」；支线=销毁涂黑照片传播链；支线=记录咒规则供乐园存档（非灭杀顶点）。
危险度/规避：贴近顶点。超阶巅峰：存在·条件性胜利；严禁削弱式解释。
任务方向/奖励：诅咒规则完整档案、扩散地图、生存SOP；不发灭神级装备。
