# 天子传说（天子传奇）
<!--meta lib=主库 tiers=一、二、三、四、五、六-->

## 剧情

**【作品来源】**
《天子传奇》（亦称《天子传说》）是香港漫画家黄玉郎于1993年出狱后创办玉皇朝时的主打长篇系列。原拟以《义勇门》开业，后采纳首徒祁文杰建议，改以「天子」为题打头阵，取其好兆头。系列将中国历代开国或关键帝皇的史实骨架，与港漫式神功、魔功、仙域、妖道、神兵与宫廷权谋深度缝合，早期部分篇章有黄易等参与编剧。文风：热血夸张、招式特效满屏、正邪对决与天命命格并重；尺度与血腥在早期尤甚。已公开篇章约九部：①开周篇（周武王姬发）②秦皇篇（秦始皇嬴政）③流氓天子（汉高祖刘邦）④大唐威龙（唐太宗李世民）⑤如来神掌（龙戈儿主线·宋太祖赵匡胤为真命天子）⑥洪武大帝（明太祖朱元璋）⑦三国骄皇（刘备·曹操·孙坚等）⑧苍天霸皇（元太祖铁木真）⑨商汤天子（系列更早时代延伸）。2010年前后有网游《天子传奇Online》改编。本档案以系列共通设定与各部公开主线为据，服务轮回乐园任务世界注入；跨界企划（如与《神兵玄奇》联动）仅作背景注记，不以神兵本传为准。

**【世界定位】**
这是一个「历史拐点被武学神话重写」的中华架空武侠宇宙：真实王朝更迭仍发生，但决战场上的决定性力量不是单纯兵马粮草，而是天子命格、女娲/释迦/黄帝一系神功、天魔天妖邪统、以及认主神兵。每一代「真命天子」几乎都要在少年或青年期经历追杀、奇遇、失亲、夺宝、会英雄、最后以神功决战改写朝堂。一句话：得天命与绝学的人，在历史的刀刃上以武入帝。

**【世界观 · 力量体系】**
本世界没有统一的「修仙境界」表，战力由**内功神功层级、命格气运、神兵认主、魔功吸噬、阵法与战器**共同决定。跨部有可传承的「天子武学」与「妖魔邪功」两条主轴。

1. **天子武学（正统神功）**
   - **先天乾坤功**：传自黄帝河图一系（广成仙派等），护体有金刚真身等；杀招乾坤七绝「天惊地动」分五击（风兮破地、火兮焚空、水兮滔天、山兮崩裂、雷兮天地碎等表述因版略异），是开周篇西岐侧根基。
   - **浑天宝鉴**：女娲所留，以**十枚天晶**为载体，不靠书本秘籍，须持晶冥想吸纳自然能量逐层突破。十层与天干/自然象对应，常见表述为：白云烟、玫霞荡、土昆仑、碧雪冰、紫星河、玄混沌、靛沧海、金晨曦、血苍穹（血穹苍）、**玄宇宙**。前层偏内功气劲与控物，中后层可裂土碎石、冰火星河、血气苍穹；第十层玄宇宙极难，公开资料中正式练成全十层的关键人物包括第五部一邪宫主**皇甫极**。嬴政悟「九式逆运浑天宝鉴」可逆转返璞，是秦皇篇破局关键。
   - **如来神掌**：释迦系佛门绝学。大唐威龙中与**六神诀**、佛兵法器绑定（李世民得「伏魔尊者」命，学得数式）；如来神掌篇中龙戈儿自李无名处得三式起修，逐步推至九式乃至**万佛朝宗**等终极式，并可兵解佛兵、正邪合一。各港漫作品对招式名与式数有差异；本档案以天子系列内表现为准。
   - **六神诀**：大唐威龙核心之一（雷神疾等至如来破），与如来神掌互补，伏魔正宗气质。
   - **天子剑法**：李淳风等创，九式至「君临天下」紫龙气，赵匡胤一脉关键。
   - **风云双绝**：刘邦侧（风绝步、云绝掌等），智谋型天子的轻功掌法代表。
   - **明神武典 / 日月双炁**：洪武大帝中朱元璋融合明教先天元阳炁（日）与后天玄阴炁（月）成紫气天子功的代表表述。
   - **苍狼歼霸拳 / 长生天神功 / 九阴真经（系列内用法）**：苍天霸皇与洪武中蒙古侧与中原侧武学交叉点。

2. **妖魔邪功**
   - **天魔神功（天魔功）**：上古大天魔所留至阴魔功，含天魔五绝、**天魔四蚀**（蚀肉/蚀骨/蚀经/蚀魂）与天魔极乐等。资质不足者易形变走火；成功者可吸精元、长生与暴增战力，但心性残暴，且有大天魔元神碎片反噬之险。开周中**元始天魔**、纣王、附身后的姬考；隋唐中**杨广、李元吉**等均为传人或沾染者。
   - **天妖屠神法**：混沌天妖一系，气芒常呈绿骷髅象，九式（吸阴式、刁魂破、妖魂冲霄、吸天蚀日、千魂屠城、妖锁囚神、幻魅妖球、妖乱天地、妖极灭世等），配合**狼牙妖棒 / 天妖战甲**。开周中**妖帅**；唐篇中李建成等。天妖与天魔传人因上古仇恨常互为宿敌，亦可在极端剧情中魔妖合体。
   - 其他邪统：极乐正宗（摩诃叶）、魔光/青鉴秘卷、黑死毒功、神游物外大法、魔弥陀飞龙异血等，各篇独立又互文。

3. **命格与气运**
   九九命数、九五至尊、紫龙皇命、偷龙转凤、九鼎奇法转命、李元霸以血定李唐二十代皇命、输血改命格等，是「君权神授」的硬设定：武功再高，若气运被夺或命数未转，仍可能夭折或失帝位。

4. **神兵与战器**
   **天剑**（周武一系，可万雷/真龙等变体，后落刘邦）、**雷刀 / 霸王雷刀**（项羽遗物，摩诃叶布局关键）、太渊刀与柔晶剑、如来佛兵（九把与火云圣衣等）、浑天十晶、武神/飞将战器、光武系列、苍狼刀、十二金人、天翔五灵（万象真藏、太极玄棺、聚宝盆等）……神兵常带灵性，认主或反噬。

5. **凡俗军武与门派**
   军中精锐、侯府将尉、武林门派（武当七侠、丐帮十二生肖、全真、蜀山、稷下、天母门、一邪宫、双飞堡、三绝观等）构成中间层。正一纯阳、虎啸皇拳、战阵七式、五龙道、冰火五重天、七旋斩、如意禅等是「非天子亦可达绝顶」的百家武学。

6. **死亡与超脱规则**
   凡人一刀一枪可死；高阶可吸功、夺舍、元神附身、转世伏笔（如项羽→杨玄感等公开互文）。天子亦死于阴谋与决战（洪武杀功臣、项羽乌江、姬考惨死皆在设定内）。仙域天帝级存在对本档案六阶切入属**超阶情报**，低阶勿硬碰。

**战力阶梯（对照破坏力，宁低勿高）**
- 普通人 / 兵卒 / 市井混混：无超凡。≈一阶初。
- 训练有素老兵 / 帮派打手 / 江湖三流：冷兵器格斗。≈一阶中后。
- 门派好手 / 侯府将尉 / 初入内功：空手裂石边缘、轻功。≈**二阶**。
- 一方宗师 / 干部级高手 / 妖帅九大高手中下游 / 射将乐将级：裂石断金、阵中取首。≈**三阶**。
- 绝顶名家 / 浑天中层（土昆仑～玄混沌）/ 如来初式稳固者：毁建筑群、对抗魔将。≈**四阶**。
- 浑天高层 / 如来多式 / 魔君级 / 天魔四蚀成型 / 一城战场覆写：领域雏形、吸功、城级战。≈**五阶**。
- 血苍穹圆满 / 玄宇宙 / 开国天子巅峰决战 / 天魔天妖传人巅峰：地区级灾难、改写王朝战争。≈**六阶**。
- 天帝本尊、女娲释迦级神话体、跨篇终极仙魔：超阶，仅情报。

**乐园阶位映射**：江湖军武·市井≈一阶；门派好手·初入内功≈二阶；一方宗师·将帅干部≈三阶；绝顶名家·浑天中层≈四阶；魔君·如来数式·浑天高层≈五阶；血苍穹·玄宇宙·开国巅峰决战≈六阶；天帝/女娲本尊级＝超阶（低阶切入一律规避本体）。**宁低勿高**；本档案任务世界切入封顶六阶。

**【地理 · 舞台】**
舞台随篇章切换，契约者入场前必须先锁定时间线：
- **开周篇**：商都朝歌、西岐侯府与战场、西域云雾中的**缥缈城**（酷刑城主、四星君）、广成仙派、南楚、修罗幻域、蓬莱/仙域边缘、白狄魔族相关战场。
- **秦皇篇**：战国秦赵燕楚、咸阳宫廷、蜀山、稷下、天母门、剑父一脉、六国公子府。
- **流氓天子**：沛县市井、博浪沙、咸阳、十二金人秘藏、东海射鲛、楚汉中原与乌江。
- **大唐威龙**：隋末长安与江都、瓦岗、风尘三侠游历线、秦皇陵万世潜龙穴、玄武门、突厥「天可汗」线。
- **如来神掌**：五代十国后汉/后周/南唐/辽边界、一邪宫、双飞堡、三绝观、天竺修炼线、玄宇宙相关秘境、陈桥驿。
- **洪武大帝**：元末中原、白莲教/明教日宗月宗、皇觉寺与丐帮、大都、鄱阳湖、武当、紫禁城洪武朝回忆框架。
- **三国骄皇 / 苍天霸皇 / 商汤**：东汉末、蒙古草原、更早上古商汤——可作扩展线，低阶优先前六部。

共通空间逻辑：**朝堂权谋层 + 江湖门派层 + 仙魔妖三界层**叠图。同一地点可同时发生「朝议」与「神魔斗法」。

**【世界剧情线】**

① **开周篇：姬发·武王伐纣（系列奠基，约163期）**  
商末，纣王昏暴，妲己与妖魔帅系横行，民不聊生。西伯侯姬昌次子**姬发**历「三年孕期」降生，负**九九命数**（易夭折或大劫）。姬昌以**九鼎奇法**将其命格转为**九五至尊**，直接刺激纣王斩草除根——自此西岐与朝歌的杀机不再是政治，而是命格战争。姬发师承广成仙派，修**先天乾坤功**（护体金刚真身，杀招乾坤七绝「天惊地动」分风火水山雷诸击），少年期即被追杀与奇遇反复淬炼。  
关键转折在西域云雾中的**缥缈城**：约三十年前胡人建城，二十余年前一名武林高手单骑攻入、激战一日一夜斩近千人，被拥立为「酷刑城主」。城主得女娲所留**浑天宝鉴**（十天晶）称霸西域，与姬昌分庭抗礼；为冲击第九层**血苍穹**而活取童脑、放毒蜂、驱战俘与饿熊搏斗后浇油点火，雪宫「三道菜」成为港漫式暴虐名场面。麾下四大星君：苍龙（亲子，靛沧海）、白虎（养子，原戈巴克夫，玄混沌，野心勃勃）、朱雀（女，土昆仑，朱雀刀，对姬发有意）、玄武（龟甲玄功）。总管胡威、幽冥双魅、三鹰、哼哈二怪、四飞女（血滴子）等构成酷刑统治网。姬发误入/被诱入城中为质，城主欲以他为饵灭西岐并借口结亲引姬昌入瓮。西岐**绣尉**为效忠牺牲贞节离间城主父子（城主强占绣尉激化与苍龙矛盾，绣尉后死于苍龙误伤）；**绿毛老祖**（姬昌叔父辈，雷电门，绝学惊雷恶电/雷电霹雳）与姬昌联手攻城，城主与绿毛「穹苍无悔 vs 雷电霹雳」两败俱伤后，苍龙偷袭击杀城主，白虎再杀苍龙，终与朱雀同归于尽线收束。城池总设计师之子**白毛虎**重情重义，为姬发盗天晶，获女娲授予有限次数的血穹苍气劲——**浑天宝鉴**由此转入姬发之手，逐步修至血苍穹，成为伐纣的核心资本。  
兄**姬考**线是魔功悲剧：上古大天魔败于天帝，临死以血刻**天魔神功**于岩壁；**元始天魔**欲篡商反被纣王布局，皇城歼魔之役遭魔帅五先锋、妖帅、魔君、大祭司、纣王、一忧子等围剿，败于「天惊地动」后被吸功断肢囚天牢，再以移魂大法附身姬考。附身后的姬考吸功成魔君级，南楚之役吸尽魔君、红毛老祖、晴儿、天母圣姬等，野心攻朝歌间接重创纣王，却在吸功骄傲时死于天诛转嫁。父姬昌亦在系列中后期殉国/战死。姬发继西伯侯位，会诸侯，**姜尚**（斗转星移等奇门）辅佐，西岐两相两将两尉（礼相、数相、射将、乐将、书尉、绣尉）在破九鼎与守城中大量牺牲。  
邪统另一极是**妖帅**：于南蛮得天妖盔甲碎片，修罗幻域得全甲与元神附身，使**天妖屠神法**九式与狼牙妖棒；麾下九大高手（妖哥、蜂魅、钓叟、伶鬼、巨僧、残魂、小妖、猪童、**燕九妹**——后者后嫁姬发）。西岐之役妖帅看准绿毛后继无力而击杀之，终死于姬发**天剑**下。神话层：天帝遭天妖偷袭，元神入天剑等。伐纣成功，姬发建周，燕九妹等为后妃线。本篇是系列武学宇宙的「创世纪」：浑天、天剑、天魔、天妖、先天乾坤全部登场；融合《封神》皮相而重心在港漫神魔对决。

② **秦皇篇：嬴政·十二月王孙（约67期）**  
战国末，楚大灵巫**项魁**卜得「秦生子，十二月」之谶，发动对十二月怀胎王孙**嬴政**的连环追杀——追杀本身即气运与武林的双重绞杀。吕不韦「奇货可居」扶异人（子楚），赠**浑天宝鉴**使政在亡命中自强。嬴政返秦后与名分兄弟**成蟜**夺嫡：成蟜拜越**剑父**与天地双绝，修**冰火五重天**并推浑天至**玄宇宙**，独眼后暴戾，辱天母门主**春雨姬**致其自杀，终被嬴政悟出的**九式逆运浑天宝鉴**（逆转返璞归真）击败流放。  
朝堂与武林双轨并行：**赵高**因意外伤及下体仍忠政，修**青鉴秘卷**；王翦、环渊等将相；燕太子**姬丹**持先天乾坤功与政情仇交织；渭水**剑帝**、蜀山琴画魔、鱼氏刁氏、平原君玄冥幻功等六国高手穿插。稷下、天母门、战国四公子神功体系构成「非天子亦可绝顶」的对照。情节紧凑、悲剧色浓，统一天下过程被神功化，但雄猜暴烈、焚书坑儒观感的秦皇像仍在——本篇强调「悟性与命数」胜过单纯武力堆砌。

③ **流氓天子：刘邦·楚汉（约63期，全系列偏短）**  
秦末，沛县流氓**刘邦**因债务累及父母，追捕大盗**任横行**（瘟神气质）反被卷入刺秦。与**张良、田霸**等博浪沙刺秦，败于秦始皇浑天宝鉴的绝对碾压，刘张分散逃亡。刘邦奇遇得周武王神兵**天剑**，**徐福**暗线相助，入咸阳结识**项羽**，合谋盗**十二金人**熔铸兵器，出海射鲛，合力斩杀秦始皇——「流氓与霸王联手弑皇」是本篇最港漫的史诗翻转。  
秦亡后群雄并起，楚怀王旗下刘项分道：刘邦能用人（韩信、张良、萧何系），武功非最强却靠天剑+智囊+气运；项羽勇冠天下，却在失**范增**与**虞姬**后偏激失控。四面楚歌、垓下、乌江自刎收束。风格偏市井搞笑与权谋，是「非最强武力亦可成天子」的样本；天剑象征「周武天命转移至汉」。项羽遗物**雷刀**为隋唐篇埋线；项羽转世为杨玄感等互文在后续公开设定中出现。

④ **大唐威龙：李世民·玄武与摩诃叶（约102期）**  
隋末，**极乐正宗摩诃叶**扶持**杨广**弑父夺位，并谋以项羽遗物**雷刀**称帝；对李唐施「偷龙转凤」，针对真命**李世民**。李世民（含替身/身世痛苦线）被抛入江湖，招讨瓦岗，结**程咬金**（三道板斧）、**虬髯客**（虎啸皇拳）、**李靖**、风尘三侠等，悟「**伏魔尊者**」天命，得**六神诀**（雷神疾等至如来破）与**如来神掌**数式。佛兵法器线：心光法令、万华如意、禅震法杖、梵音陀铃等与掌式对应，摩诃迦叶与世民分持，对决中兵解净化魔气。  
入**秦皇陵万世潜龙穴**争宝是中段高潮；**李元霸**以身血定李家二十代皇命，是命格硬设定的顶点操作之一。诛隋炀后李渊建唐；**玄武门之变**中世民诛兄**李建成**（得天妖战甲妖棒）、弟**李元吉**（天魔功），魔妖合体等高潮后登太子/帝位，再战摩诃叶与突厥「天可汗」线。终成太宗，却终生笼罩摩诃叶阴影——「赢了天下，赢不了那个影子」。本篇结构分章清晰，评价较高；如来神掌与佛兵体系在此奠基，并下启第五部。

⑤ **如来神掌：龙戈儿与赵匡胤（约214期）**  
五代十国乱世，**一邪宫、双飞堡、三绝观**三大势力分别服务辽、南唐、后汉等政权，武林成为分裂政权的外挂军团。李唐后人**龙戈儿**天生**紫龙皇命**，自**李无名**处得三式**如来神掌**起步，行侠仗义；结识**赵匡胤**（真命天子，主修**天子剑法**至君临天下紫龙气），习**如意禅**，受**柴荣**功力与气运灌顶。中段名场面密集：杀胡林、**魔弥陀**（飞龙异血）、**皇甫极**（一邪宫主，公开资料中史上关键「浑天十层**玄宇宙**」正式练成者，可化浑天邪剑、灭绝皇拳、浑天神掌等）、**项无恨**（紫雷刀第九击）、极恶龙戈儿分身、佛兵兵解、天竺修炼、天道仙府、火云圣衣等。  
关键命格操作：龙戈儿曾**输血**救赵匡胤，改变双方命格结构——侠客以血改天子运，是本篇主题句。赵匡胤**陈桥兵变**称帝建宋，与诸强大战后逐步统一；龙戈儿正邪合一、兵解佛兵，完成非常规主角的佛武巅峰。本篇主角不是直接坐龙椅的人，武功破格多，读者评价两极，但是如来与浑天交叉、正邪辩证法的巅峰舞台。契约者切记：**真命天子是赵匡胤，龙戈儿是神掌主视角**。

⑥ **洪武大帝：朱元璋·反元开国（潮华等版约186本/550章量级）**  
公开框架以**洪武二十八年**紫禁城处决叛众、朱元璋亲手杀死智者**刘伯温**的残酷场景开场，再倒叙元末崛起——一开篇就钉死「鸟尽弓藏」基调，忌洗白为单纯明君传。元朝统治近百年，中原武学遭毁灭性打击，汉人义军四起：**白莲教（韩山童等，神游物外大法）**、**明教日宗/月宗**、在野「军」前辈培训志士。蒙古为挽颓势，派出本土最强编制第三代**十三翼**入关（继承历代名人名号的战士团），其首**苍狼·铁木真**是后裔/名号继承者（非苍天霸皇中的成吉思汗本人），练**长生天神功**，妻白鹿·帖木儿为军师，子有朮赤、拖雷等；哲别、札木合/毒王、合撒儿、赤老温等翼将环绕。苍狼与韩山童同归于尽级对决，是元末天花板群像。  
各方争夺南宋文天祥所留**天翔五灵**（万象真藏、太极玄棺、聚宝盆等）。少年**朱重八/朱元璋**贫苦出身，父母早亡，失恋人**巧巧**，与元廷亲王**罕雷**结仇；入皇觉寺修行，入丐帮，与已成为「再世诸葛」的**刘伯温**重逢。融合明教**先天元阳炁（日王）**与**后天玄阴炁（月王）**，获紫气天子之相，成**明神武典**。联合**徐达、常遇春**（武神战器传人）、**汤和、马秀英**（白莲天女，后为后）等，经黄河起义、大都之战、鄱阳湖之战、武当灭门（拖雷黑死毒）等，扫张士诚、陈友谅（含窝阔台伪装+邪术等架空）诸雄，推翻元朝建明。武当七侠、丐帮十二生肖、宋帝昺/张君宝潜伏太极玄棺等线丰富江湖侧。本篇重现「元后天子武学再兴」，并强制保留杀功臣、杀伯温的残酷。

⑦ **三国骄皇 / 苍天霸皇 / 商汤天子（扩展三部）**  
**三国骄皇**（约52期）：东汉末十常侍乱政，阉党、女流（妲己传系）、仙道（南华老仙转生等）操纵朝局。**刘备**得刘邦魂启异能，结关羽、张飞等；**曹操、孙坚**同步崛起，开创「三皇并立」系统，匠心重述三国。光武神兵等为关键道具。  
**苍天霸皇**（约70期，玉皇朝20周年纪念作）：**铁木真**自「卑污血脉」争议与父死后部族离散中崛起，获**苍狼刀**、九阴真经等，练苍狼歼霸拳、苍狼七杀刀、北斗七星剑指、空明拳、黄祸、降魔大手印等；幼受**王重阳**（全真）影响。与义弟**札木合**由安答至决裂是情感主轴；答兰版朱思破十三翼、札答兰战役、怒破北斗七星剑阵、大罗天与灭世黑死毒等为武戏高峰。敌对面有**龙藏大宗**（西藏大梵天宗，欲夺大金龙脉）、神火圣君宇航（波斯拜火教）、毒帅、西魔等；爱人艾美黛线。主题：「天生皇者宿命 vs 弱肉强食的孤独成皇」。  
**商汤天子**：系列时间轴回到更早商汤时代，作起源向延伸。三部可作高阶或专题线；默认低阶任务仍以前六部为主。

⑧ **跨篇主题与互文**  
神兵与功法跨代传承（浑天十晶、天剑、雷刀、如来与佛兵、天子剑、苍狼刀）；转世与遗物复仇（项羽—杨玄感、雷刀入隋唐）；天子必付代价（失亲、入魔阴影、玄武门、杀功臣）；正邪可合一（龙戈儿）；命格可偷、可转、可输血改写。天魔与天妖因上古仇恨互为宿敌，亦可在极端剧情魔妖合体。元后中原武学断层→洪武再兴，是系列内部的「黑暗时代」设定。与《神兵玄奇》等有跨界企划（如南宫问天用四式唐版如来对龙戈儿九式等粉丝/跨作讨论），但本任务世界默认锁在天子本传历史轴，不强制引入神兵本传主角。

**【对契约者的世界规则摘要】**
1. 先锁篇章时间线，再选身份；跨时代穿越须乐园任务明文允许。  
2. 神功层数与命格气运双轨：只加武力不加气运，难坐龙椅；只沾气运无武力，难活过刺客。  
3. 吸功、夺舍、兵解、输血改命皆有剧情代价，禁止无代价复制。  
4. 土著默认不知「轮回乐园/契约者」；天子侧智者（姜尚、刘伯温、张良）可敏感到「气运异物」但不会叫出乐园术语。  
5. 神兵认主前是争夺焦点，认主后是身份绑定；强夺易反噬。  
6. 开国后的清洗期（洪武）与阴影期（太宗 vs 摩诃叶余影）同样是合法任务舞台，不只有「打天下」。

## 阶位切入点

> 阶位↔本世界战力：一阶≈兵卒市井，二阶≈门派好手初入内功，三阶≈一方宗师将帅，四阶≈绝顶/浑天中层，五阶≈魔君/如来多式/浑天高，六阶≈血苍穹·玄宇宙·开国巅峰。乐园阶位映射见上。世界顶点（天帝/女娲本尊级）为超阶情报；低阶切入一律规避顶点本体。宁低勿高。

**一阶（兵卒市井 · 沛县/西岐外围/战国边城线）**  
切入身份/时点：契约者以「沛县混混跟班 / 西岐外围民兵 / 战国边城斥候」身份切入，锚定刘邦尚未得天剑、或姬发尚在西岐少年、或嬴政遭追杀而江湖尚未全面神魔化的凡人层。  
初始事件：一场债务追讨、征粮抓丁或刺客擦肩而过——你亲眼见「大人物」的气劲掀翻街棚，却只能捡回一条命；当夜有人塞给你一枚来路不明的令牌/残页，要你送进某侯府或盗伙据点。  
开场白建议：「中原的夜总是尘土味。你听见远处有人笑骂『真命天子』四个字，像在说一个与你无关的笑话——直到刀光从巷口掠过，你才发现，笑话里的血，溅到了你的鞋上。」  
关键NPC立场：**刘邦**（此时仍痞气，可拉拢可出卖）；**任横行**一类大盗（利益至上）；西岐**书尉/绣尉**外围眼线（忠义但可死）；边城秦卒（恐惧谶语）。  
主线钩子/支线：送信/带路引发蝴蝶；目击刺秦或征兵惨剧；支线＝市井赌债、盗墓边缘、军中吃空饷。  
危险度/规避：中低——须规避任何已显浑天气劲的高手与朝堂点名追杀。  
任务方向/奖励：活着摸到「天命传闻」的边；奖励取向＝入门刀法残本、轻功皮毛、一枚可交易的令牌、贵人一面之缘。

**二阶（门派好手 · 广成外围/瓦岗边缘/丐帮十二生肖外围）**  
切入身份/时点：以「广成仙派记名弟子 / 瓦岗外围好汉 / 丐帮末代生肖跟班」切入，锚定姬发习先天乾坤初期、或李世民江湖历练、或朱元璋入丐帮前后。

开场白建议：「师父说江湖的水比黄河浑。你护着那只沉重的木箱跑过山梁时，才明白：箱里装的不是货，是会把人切成两半的『气运』。」

危险度/规避：中——须规避魔帅本尊、摩诃叶直系与浑天中层以上。  
任务方向/奖励：站稳门派/义军脚；奖励取向＝成型内功入门、兵器认主边缘、义军军职。

**三阶（一方宗师 · 将帅干部 / 妖帅九大 / 射将乐将级）**
