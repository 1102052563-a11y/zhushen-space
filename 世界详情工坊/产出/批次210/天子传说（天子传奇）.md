## 剧情

**【作品来源】**
《天子传奇》（又称天子传说）是香港漫画家黄玉郎1993年玉皇朝创业主打作，黄易等参与编剧，连载并分篇出版多部。已公开篇章包括：1开周篇（姬发）、2秦皇篇（嬴政）、3流氓天子（刘邦）、4大唐威龙（李世民）、5如来神掌（赵匡胤/龙戈儿线）、6洪武大帝（朱元璋）、7三国骄皇（刘备等）、8苍天霸皇（铁木真）、9商汤天子等。文风：港漫神功魔功、历史人物架空武侠化、神兵天命。2010年有网游改编。本档案以系列共通设定+各篇主线骨架服务AI，细节以开周/秦皇/流氓/大唐/如来/洪武等公开梗概为准。

**【世界定位】**
中华历代「天子命格」与神兵、魔功、仙域纠缠的长篇武侠神话：从武王伐纣到秦皇一统、汉楚争霸、唐初玄武、宋初陈桥、明初反元等，皆被写成神功决战史。一句话：得天命与浑天/如来等绝学者，在历史拐点上以武入帝。

**【世界观 · 力量体系】**
力量来源：内功神功（先天乾坤功、浑天宝鉴九层、如来神掌、天魔功、魔光七重天等）、命格（九五/九九）、神兵认主、仙域魔域传承。死亡：高阶可吸功、夺舍、元神出窍，但天劫与反噬常见；天子亦会死于阴谋与决战。

境界/层级（综合各篇战力表现，宁低勿高）：
1. **江湖三流 / 军中精锐**：刀枪弓马。≈一阶。
2. **门派好手 / 侯府将尉**：成型武技、轻功。≈二阶。
3. **一方宗师 / 干部级**（射将、乐将、妖帅九大高手中下游）：裂石断金、阵法辅助。≈三阶。
4. **绝顶名家 / 浑天中层**（土昆仑～玄混沌等）：毁建筑群、对抗魔将。≈四阶。
5. **浑天高层 / 如来数式 / 魔君级**：城战场覆写、吸功与领域雏形。≈五阶。
6. **血穹苍圆满 / 玄宇宙 / 天帝级传承一角 / 开国天子巅峰战**：地区级灾难、改写王朝战争。≈六阶。
7. 更高仙域天劫、跨篇神话体：本档案切入封顶六阶，标为超阶情报。

乐园阶位映射：江湖军武≈一～二阶；宗师将帅≈三阶；绝顶/浑天中≈四阶；魔君/如来数式/浑天高≈五阶；血穹苍/玄宇宙/开国决战≈六阶；天帝本尊级=超阶（低阶规避）。宁低勿高。

**【地理 · 舞台】**
随篇章切换：商末朝歌与西岐、缥缈城、蓬莱仙域、白狄魔族；战国秦赵燕楚与咸阳；秦末沛县、乌江、十二金人；隋唐长安、瓦岗、秦皇陵；五代辽唐汉周与双飞堡；元末明教白莲与紫禁；三国东汉末；蒙古草原等。共通逻辑：朝堂+江湖+魔妖仙三界叠图。AI须先锁定契约者处于哪一篇时间线。

**【世界剧情线】**
① **开周篇：姬发与伐纣**  
商纣暴政。西伯侯姬昌次子姬发历三年孕期降生，负九九命数，姬昌以九鼎奇法转其命格为九五。纣王杀机起。姬发承广成仙派先天乾坤功，于缥缈城得女娲系绝学**浑天宝鉴**，结识英雄，痛失兄姬考与父姬昌。继西伯侯位后会诸侯，与纣王展开神功魔功决战，终建周。关键道具与人物：燕九妹、白毛虎、绿毛老祖、姜尚、元始天魔附身之姬考、妖帅、天帝与天剑等。

② **秦皇篇：嬴政一统**  
楚国大灵巫项魁得「秦生子，十二月」之谶，追杀十二月怀胎的王孙嬴政。政得吕不韦所赠浑天宝鉴自强，历经燕太子丹、成蟜夺嫡等，悟九式逆转返璞归真，破成蟜玄宇宙，登秦王位并走向一统。双绝、剑帝、蜀山琴画魔等构成武林侧。

③ **流氓天子：刘邦与楚汉**  
沛县流氓刘邦为赎父母，追捕大盗任横行，反被卷入刺秦。与张良、田霸等博浪沙败于秦始皇浑天之下；刘邦得周武王神兵**天剑**，入咸阳，结识项羽，盗十二金人。后合力射鲛斩秦皇，却迎来群雄混战。项羽失范增与虞姬后偏激，刘邦与之血战，项羽乌江自刎，汉室奠基。

④ **大唐威龙：李世民**  
极乐正宗摩诃叶扶杨广弑父，谋以项羽遗物雷刀称帝。李世民被偷龙转凤，历瓦岗、识程咬金虬髯客李靖，悟佛门伏魔尊者命与如来神掌数式。秦皇陵万世潜龙穴中李元霸以身定李家二十代皇数。玄武门变中世民诛建成元吉，又失挚爱入魔，终败摩诃叶成太宗，却终生笼罩其影。

⑤ **如来神掌：龙戈儿与赵匡胤**  
五代乱世，邪宫、双飞堡、三绝观服务辽、南唐、后汉。龙戈儿得三式如来神掌，会皇甫极、识赵匡胤，经杀胡林、魔弥陀、玄宇宙、万佛朝宗、火云圣衣等线，正邪分合。柴荣死后赵匡胤陈桥称帝，与项无恨等大战，皇甫极出山，宋一统。

⑥ **洪武大帝等后续**  
元末白莲明教、十三翼、「天翔五灵」争夺；朱元璋回忆杀廖永忠、刘伯温等残酷开国（公开梗概）。三国骄皇写刘备得刘邦启发；苍天霸皇写铁木真；商汤篇续系列。共通主题：天子命数与魔功神兵互相吞噬。

**【主要人物】**

姬昌（周文王）｜性格：隐忍卜算｜装备·能力：先天乾坤、九鼎奇法｜人物弧光：护子转命→殉西岐｜立场关系：发、考之父。

姬考｜性格：野心膨胀｜装备·能力：天魔传人、吸功｜人物弧光：兄长→魔君级→攻商惨死｜立场关系：一度敌发。

嬴政（秦始皇）｜性格：雄猜、悟性极高｜装备·能力：浑天至血穹苍、九式逆转｜人物弧光：被追杀王孙→秦皇｜立场关系：吕不韦、成蟜。

吕不韦｜性格：奇货可居、权谋｜装备·能力：锋藏颺等｜人物弧光：扶异人→操秦政｜立场关系：政之助力与阴影。

刘邦｜性格：流氓豁达、能用人｜装备·能力：天剑、智囊加持｜人物弧光：沛县混混→汉高祖｜立场关系：张良、项羽。

项羽｜性格：勇猛寡谋后期偏激｜装备·能力：西楚武力、雷刀相关后续｜人物弧光：反秦霸主→乌江｜立场关系：虞姬、范增。

李世民｜性格：英武、身世痛苦｜装备·能力：如来神掌、天子命｜人物弧光：被调换的世子→太宗｜立场关系：摩诃叶阴影。

摩诃叶｜性格：阴谋称帝｜装备·能力：极乐正宗、雷刀布局｜人物弧光：操隋唐→败｜立场关系：世民一生之敌。

赵匡胤｜性格：枭雄天子｜装备·能力：天子剑法｜人物弧光：结识龙戈→陈桥称帝｜立场关系：宋开国。

朱元璋｜性格：多疑残暴开国｜装备·能力：乱世武略（篇内神功体系）｜人物弧光：反元→洪武杀功臣｜立场关系：刘伯温等。

**【隐藏剧情 · 伏笔】**
命格可转移；浑天可逆转返璞；天子常以魔功阴影为代价。系列跨篇神兵与人物转生/遗物呼应（如雷刀）。与《神兵玄奇》有跨界企划（公开资料），本档案以天子本传为主。

**【大事记时间线】**
姬发降生转命 → 伐纣建周 → 嬴政逆境称王 → 刘邦得天剑灭秦楚汉 → 李世民玄武与败摩诃叶 → 龙戈儿/赵匡胤宋一统 → 朱元璋洪武 → 三国与蒙元等篇延伸。

**【叙事基调 · 雷区】**
港漫夸张神功+历史皮；忌写成严谨正史。忌把所有天子写成完人（洪武杀伯温等须保留残酷）。忌跨篇战力无限通吃而不分时代。最早切入：开周篇姬发幼年或秦皇被追杀期。

**【开周篇细部因果】**
姬发九九命数意味着夭折或大劫，姬昌以九鼎奇法转为九五，直接刺激纣王斩草除根。广成仙派先天乾坤功与昆仑姜尚洛书系形成河图洛书对照。缥缈城「酷刑城主」持浑天，苍龙、白虎、朱雀、玄武四星君分掌；姬发与白毛虎盗宝、女娲传劲，城主父子相残，白虎死于朱雀等线构成港漫式恩怨。绿毛老祖惊雷恶电、雷震子、西岐两相两将两尉（礼相、数相、射将、乐将、书尉、绣尉）在破九鼎与守城中大量牺牲，绣尉离间城主父子等戏是「忠义代价」。元始天魔欲篡商，反被纣王布局吸功断肢；附身姬考后考成魔，吸魔君与南楚高手，攻商却死于天诛转嫁。妖帅得天妖卸甲，西岐杀绿毛，终死姬发天剑下。天帝遭天妖偷袭元神入天剑，姬发毁肉身。伐纣成功，燕九妹为后，周朝立。

**【秦皇篇细部因果】**
项魁谶语驱动楚系追杀。吕不韦「奇货可居」扶异人，赠浑天使政自强。赵高因意外伤及下体仍忠政，修青鉴。成蟜拜越剑父与天地双绝，修冰火五重天与玄宇宙，独眼后暴戾，杀春雨姬，终被政九式逆转所败流放。渭水剑帝、蜀山琴画魔、鱼氏刁氏等构成秦国武林侧。平原君玄冥幻功等六国高手穿插。统一战争在漫画中被神功化，但政治仍落在「称王—称帝—焚书坑主观感」的暴烈天子像。

**【流氓天子细部】**
刘邦欠债累父母，抓任横行反被带去刺秦；张良、田霸加入。秦始皇浑天碾压，刘张分散。刘邦得天剑，徐福暗助，入咸阳识项羽。盗十二金人充兵器。射鲛斩秦后群雄并起。楚怀王旗下刘项分道，范增虞姬死后项羽失稳，垓下与乌江收束。天剑象征「周武天命转移到汉」。

**【大唐威龙细部】**
摩诃叶扶杨广，选雷刀为天子神兵，偷龙转凤李世民。瓦岗识程咬金、虬髯客、李靖。伏魔尊者命与如来神掌两式。秦皇陵万世潜龙穴李元霸以血肉定李家二十代。诛炀帝后摩诃叶扶李渊建唐。玄武门：建成元吉藏妖魔功，欲先清摩诃叶再杀世民，世民再诛兄弟。天可汗线失挚爱入魔，终败摩诃叶，却一生活在其阴影——漫画对「贞观明君」的暗面注解。

**【如来神掌细部】**
龙戈儿从李无名得三式如来，会皇甫极夺雷刀，识赵匡胤。如意禅与柴荣传功。杀胡林斩耶律德光破一邪谋辽。魔弥陀飞龙异血横扫，后汉国师；皇甫极玄宇宙破佛法无边，极恶龙戈儿击杀魔弥陀。万佛朝宗、九大佛兵兵解、火云圣衣。赵匡胤天子剑法、陈桥兵变、与项无恨死战，皇甫极出山收束，宋一统。

**【洪武与后续篇提纲】**
元末白莲明教、「军」培训汉人、蒙古十三翼、文天祥天翔五灵争夺。开篇倒叙朱元璋斩廖永忠并杀刘伯温，回忆残酷。三国骄皇：刘备得刘邦启发异能。苍天霸皇：成吉思汗。商汤天子：更早神话史。系列用「天子命数」把历史拐点串成神功连续剧。

**【力量·浑天九层与逆转】**
公开资料与读者综述常提浑天分层至血穹苍，成蟜线玄宇宙化劫归真，政以九式逆转返璞破之。如来神掌式数递增，万佛朝宗为佛武顶。天魔四蚀、魔光多重天、先天乾坤天惊地动多式等是各篇天花板技能。六阶切入可接触这些名词，但完整修成应付重大代价与剧情位。

**【神兵表补充】**
浑天宝鉴、天剑、雷刀、祭司权杖、万魂幡、朱雀刀、玄冰宝刃、雷电霹雳、仙斧、铁塔、三尖戟、风火轮、十二金人、火云圣衣、玄宇宙晶柱、天翔五灵等。每件绑定人物命运，不是商城刷物。

**【地理切换规则】**
契约者入世必须锚定篇名时间：商末≠唐初。跨篇携带神功需乐园规则允许，否则按「遗物传说」降效。朝歌、咸阳、长安、汴梁、应天府是各自政治心脏；仙域魔域是超自然心脏。

**【人物关系网速记】**
发—昌—考三角；政—不韦—成蟜三角；邦—羽—良三角；世民—摩诃叶—建成元吉；戈儿—皇甫极—匡胤—项无恨。写对话用真名，忌「天子甲乙」。

**【列国战场观察·4462】**
在天子传奇的历史皮之下，真正决定胜负的是神功是否认主、命格是否被九鼎或魔功改写、智囊是否还活着。军队人数提供炮灰与合法性，名将提供阵法与斩首，但一篇的高潮几乎总是「两个人把天打裂」。契约者在六阶以下应优先保护命格载体、神兵与智囊，而不是与血穹苍对拳。王朝更替后江湖不会清空：败者门人、魔功残卷、神兵失踪，会成为下一篇的种子。

主线钩子/支线：送密信出城；支线=赌命赏金；支线=偷听九鼎仪式。
危险度/规避：中。规避妖帅本人。
任务方向/奖励：基层人脉、入门武技。

**二阶（侯府将尉伴当）**
切入身份/时点：西岐两将两尉体系随从，或张良侧书童。
初始事件：破九鼎阵前夜，射将命你运送羽箭，妖帅九大高手阴影在城外。
开场白建议：「箭囊比你的命重。射将说今夜阵风会吃人，你只要把箭送到台前。城外狼牙棒的风声，已经比号角早到一步。」
关键NPC立场：**射将/乐将**；**妖哥**；**张良**（若汉线）。
主线钩子/支线：守阵一角；支线=救绣尉；支线=博浪沙刺秦外围。
危险度/规避：高。
任务方向/奖励：军阵经验、名将荐书。

**三阶（缥缈城/刺秦外围高手）**
切入身份/时点：缥缈城夺宝者，或刺秦三人组外围。
初始事件：城主酷刑堂开审，白毛虎盗天晶；或秦始皇浑天一掌拍碎刺秦阵。
开场白建议：「刑架还滴着昨天的人。白毛虎从梁上丢下一句：宝鉴不认怂货。你跟着血味跑，跑进女娲留下的光里。」
关键NPC立场：**姬发**；**白毛虎**；**嬴政**；**张良**。
主线钩子/支线：助盗或夺浑天残页；支线=护送刘邦逃咸阳；支线=十二金人转运。
危险度/规避：高。
任务方向/奖励：功法残篇、神兵线索。

**四阶（魔族/南楚战场）**
切入身份/时点：魔族七将混战期或南楚鄂氏线。
初始事件：姬考吸功场展开，你被波及经脉逆流；蜂魅求你送血救一忧子。
开场白建议：「毒雾有八色，你只认得哪种让眼睛流血。姬考在高台上笑，像已经坐上魔君的位子。有人拉你的袖：救一忧子，或看西岐明天被谁吞。」
关键NPC立场：**姬考**；**蜂魅**；**魔君**；**姬发**。
主线钩子/支线：打断吸功仪式一角；支线=护送天女线情报；支线=雷电门援军。
危险度/规避：极高。
任务方向/奖励：抗毒抗魔功、阵营声望。

**五阶（如来/玄武门级）**
切入身份/时点：龙戈儿再战魔弥陀前后，或玄武门之变夜。
初始事件：佛法无边压城，或李建成李元吉暗藏妖魔功要先杀摩诃叶再杀世民。
开场白建议：「宫门的门钉倒映着三兄弟的刀光。有人念如来，有人念魔号。你站在门轴后，决定今晚让谁的血先溅到龙椅。」
关键NPC立场：**龙戈儿**；**皇甫极**；**李世民**；**摩诃叶**。
主线钩子/支线：护住传国关键物；支线=阻止魔弥陀吸收佛兵；支线=玄武门弓弩位。
危险度/规避：极高。
任务方向/奖励：佛武残式、政变情报。

**六阶（开国决战）**
切入身份/时点：武王牧野神魔决战、秦皇成蟜终战、或洪武屠戮广场回忆战。
初始事件：血穹苍与魔功对撞，天象裂开；或朱元璋在广场问你刀是否够快。
开场白建议：「鼓声停时，只有神功的风还在刮。天子在对面，魔在头顶。你听见有人说九五，有人说九九——你只握住自己的兵器，走进历史被改写的那一页。」
关键NPC立场：**姬发/嬴政/朱元璋**等篇主；对应最终敌（纣王/成蟜/叛众与阴影）。
主线钩子/支线：守护命格阵眼；支线=夺走敌方神兵一秒；支线=记录开国杀戮供后世。
危险度/规避：贴近顶点。天帝级=超阶情报优先；条件性胜利。
任务方向/奖励：开国级战利、神功领悟契机。
