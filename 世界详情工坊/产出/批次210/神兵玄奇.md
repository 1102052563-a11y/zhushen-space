# 神兵玄奇

## 剧情

**【作品来源】**
《神兵玄奇》（英文常译 *Weapons of the Gods*）是香港漫画家黄玉郎于玉皇朝集团推出的奇幻武侠漫画，1999年3月27日创刊第一部，美术总监邱福龙等参与兵器与人物造型，主编构图含黄玉郎、邝彬强等，编剧先后有文敌、胡家宝、锺英伟等，主要绘制黎智昌等。第一部约146期（1999–2002），第二部约100期，其后有《神兵3》《神兵3.5》《神兵前传》多辑、《神兵外传》《神兵玄奇F》（清末/二战）、《创世神兵》《神兵4》《神兵问天》《天妖玄奇》《皇朝君临》等衍生与跨界。每期常附赠或销售袖珍神兵模型（拆信刀精品），掀起港漫兵器模型潮。2007年有央视动画《神兵小将》改编（少儿向，设定大幅净化）。本档案以漫画正传壹、贰的神魔兵主线为核，兼录前传/F系为情报层，不把动画当正史。文风：早期血腥草根粤语江湖→中后期神魔玄幻与跨神话神域。媒介核对：公开检索与维基/百科均指向港漫，非同名网络小说；搜笔趣阁类站以 keyword 检索通常不返回小说全本页。

**【世界定位】**
乱世武林（约两晋十六国/南北朝皮）之上叠一层创世神话：元始天尊与元祖天魔的亿年对峙，女娲、蚩尤、伏羲、轩辕、大禹等上古神祇所铸「十大天神兵」与天魔精元魔珠所化「十大魔兵」在人间现世、认主、破碎、新生。南宫问天由北冥山庄伙头逆袭为天晶传人与武林盟主，与南宫铁心、北冥雪、玄天邪帝、南宫太平及天地盟、大罗刹宗、神道盟、邪道盟等纠缠，阻止天魔复活并守护神州。一句话：神兵认主的乱世，一刀可改天运，一念可入魔。

**【世界观 · 力量体系】**
力量来源三层叠加：
1. **内功武学**：世家剑诀、心剑神诀、剑皇诀、上天下地至尊功、罗刹魁神功、玄邪罡、天刀邪诀、鬼谷神算等，决定「能否握住神兵而不立刻反噬」。
2. **神兵/魔兵异能**：天神兵由神祇以神材与元神铸成，各代表一种美德或神力（仁爱、凶暴、正义、智慧、怜悯、邪狠、复仇、公正、忠贞、救世等）；魔兵由魔珠+强烈负面情感+物质催生，对应恨、疚、妒、绝望、空虚、执念、贪等，异能往往比神兵更「针对心性」。
3. **血统与魔气**：元祖天魔一口魔气可投胎为玄天邪帝；问天与铁心之子南宫太平身怀魔籽/天魔血统；神魔同体、人可入魔、魔可被感化是主题，而非单纯黑白对立。

死亡与再生：高阶武者可重伤复出，但神兵对撞可夷平岛屿（序章剑岛）；神兵破碎后可「新生」变形再临，异能近似；魔兵毁灭后异能可以图纹附体，留下人体铭刻隐患；天魔级难灭，需完美天晶合璧与诸神兵共鸣等条件性胜利。

战力阶梯（综合破坏力表现，宁低勿高）：
1. **江湖杂鱼 / 山寨 / 伙头打杂**：刀枪与粗内力。≈一阶。
2. **门派精英 / 世家护院 / 天地盟基层 / 四绝外围**：成型武技、地神兵试用。≈二阶。
3. **青年英杰 / 十大英杰下游 / 地神兵熟练持有者 / 法王外围**：可裂石断金、参与神兵争夺。≈三阶。
4. **世家家主级 / 一级神兵初成 / 四法王 / 天地双使下游**：毁建筑群、布阵屠会。≈四阶。
5. **天神兵主力兵主 / 大罗刹宗主级 / 玄天邪帝交锋 / 武林盟主巅峰**：城战场覆写、魔气领域、刀剑双绝。≈五阶。
6. **完美天晶合璧、天魔/魔籽决战、神域边缘联合作战**：地区级灾难、改写乱世格局。≈六阶。
7. 女娲/盘古/元始天尊/元祖天魔本尊级神话本源：超阶情报，本档案切入封顶六阶。

乐园阶位映射：江湖杂鱼/伙头≈一阶；门派精英/护院/盟基层≈二阶；青年英杰/地神兵主≈三阶；家主/法王/双使下游≈四阶；天神兵主力兵主/大罗刹宗主/邪帝交锋≈五阶；完美天晶与天魔/魔籽决战≈六阶；神话本源（女娲、盘古、天尊、天魔本尊）=超阶，低阶一律规避顶点·情报优先。宁低勿高。持有天神兵可越阶作战，但心性不稳则反噬甚至灭门，不可把「捡到神兵」写成稳升两阶。

**【地理 · 舞台】**
- **南宫剑岛（太湖）**：正道南宫世家旧本营，序章因天晶对撞被夷平，后重建话语权；剑冢、剑坟、家主大殿是权力与诅咒的双重象征。
- **北冥山庄 / 铸兵山庄（北地）**：天下兵库，北冥正主政，炉火与精铁是财富；问天在此当伙头学打铁烧菜，玉法王托孤处。
- **东方渤海盐贸线**：东方世家海贸与盐利，东方雄（心剑）夺回遗产的主战场。
- **西城（西域孔道）**：西城世家扼丝路，四绝护院（昆仑绝刀、天山魔斧、楼兰石怪、乌兹箭神）镇关；西城秀树等青年英杰出身地。
- **天地盟：森罗绝域、铁壁铜场、外征府与内务府**：邪道第一大势力的官僚魔窟；铜场铸兵、绝域关妖兽军。
- **蚩尤秘窟 / 灭邪玄网**：上古凶神遗迹，封印与天魔线索交汇，中期主战场。
- **西域大罗刹宗**：地位等同早期天地盟的西域霸主，宗主与炎神孛勒台（卧底线）交织。
- **神道盟六门故地**：刀、斧、剑、玄、雪、火等门派旧址，邪帝屠盟后的血债地图。
- **苗疆蛊族 / 火家 / 火焰山周边**：用毒、火玉燕血脉、铸毒与铸兵支线。
- **衍生层（前传/F）**：北魏乱世、婆罗门、奥林匹斯与亚萨神域、清末与二战平行线——主库任务默认锚定壹贰乱世，跨时代须乐园规则允许并降效为「遗物传说」。

地图逻辑：神兵气息一泄漏，正道世家、天地盟、邪道余孽、西域宗门会在数日内齐聚；舞台随神兵现世移动，而非固定王城。

**【世界剧情线】**
① **创世层与神兵铸造（前史）**  
元始天尊与元祖天魔的创世之战奠定正邪骨架。女娲以五彩神石与元神凝练**天晶**（子母双剑，仁爱，之首，开天辟地之能，亦受异魔诅咒：心术不正者「富贵绵绵，灾难滔滔」）；蚩尤以天外异妖与战虎骨骼炼**虎魄**（凶暴嗜血，预言「虎魄重光，天地称皇，谁可匹敌，唯独天晶」）；伏羲与龙马系铸**十方俱灭**（智慧、天机、大智慧空间）；轩辕黄帝为抗虎魄，九天玄女投身熔炉，成**太虚**（正义，轮/战车形）；罗刹骸骨+地心毒火成**噬魂**（邪狠，双截/三节棍）；大禹以九片凤凰银甲铸**凤皇**（救世，控水与息壤）；雷公电母系**惊邪**（公正，雷电叉）；黑母以九子血肉铸**天诛**（复仇弓与九箭，箭尽则废）；女娲神臂所化**神舞**（忠贞，琵琶音波）；神农奇玉**神农尺**（怜悯，医毒一体，纹路藏盘古无极功线索）。十兵各有铸造悲剧与美德，非商城刷物。

元祖天魔被封印前吐出精元化十颗魔珠；遇强烈负面情感结合物质，则生成**十大魔兵**（帝恨、星宿劫、疚疯、无妄、裂天破地、悲怒权杖、虚无魔镜、执念障、夔·帅/魔帅、十全魔刀/十全魔兵等），对应恨火、时光操控、感官错乱、模仿异能、反震锐锥、分离万物、虚无隐形、弱化圈、贪念傀儡、统合九能等。魔兵故事常比神兵更「人味」：由角色悲剧催生。

② **序章：天晶献礼与剑岛浩劫（约公元318年）**  
朝廷以天晶赠南宫世家家主**南宫逸**。东方雄（心剑，南宫逸侧室/相关线）警告诅咒，逸仍出鞘。天地盟天尊使持惊邪类/特制兵器夺兵，神兵对撞，能量灾难夷平剑岛。东方雄率残部；火玉燕（问天生母相关）难产而死；幼年**南宫问天**与女婴（问菜/问雅）出现在北冥山庄门口——问天以内力/血哺妹，草根生存本能先于任何剑诀。外公**玉法王**（天地盟四法王之末）暗中托孤，并在问天左臂植入天晶精元绿珠，使其与天晶投缘。此事件定义全书规则：顶级神兵对撞=战略武器；心性不配=自毁（南宫逸自私触发诅咒的教训贯穿）。

③ **第一部前半：伙头入江湖与虎魄重光**  
十八年后，问天在北冥长大，打铁烧菜，身份低微。四大家族联姻与会盟遭天地盟重创：地尊使**凶阎王**等削弱西、北势力，与**北冥正**激战中掳走娇气小姐**北冥雪**。问天为救雪踏入江湖，无意卷入**虎魄**现世引发的武林血洗。早期文风血腥、粤语江湖味浓，神兵争夺是「谁的血喂得饱刀」。问天险死还生，命运与天晶绿珠同步觉醒。同期线：**南宫铁心**（卓铁心）女扮男装曾掌南宫，实为东方雄与神算子**卓不凡**之女，十大青年英杰之首，武器由银鹰剑→龙腾剑→寒霜剑等演变，武功心剑神诀、剑皇诀。问天与铁心结缘，亦与北冥雪情感线并存（后为正室/平妻结构）。

④ **第一部中后：蚩尤秘窟、灭邪玄网、天晶之首**  
战场拉入上古：蚩尤秘窟、灭邪玄网，元祖天魔封印线索浮出。编剧调整后玄幻加重，**玄天邪帝**与**大罗刹宗主**入场，口碑回升。问天逐步面对/集结十大天神兵：天晶、虎魄、太虚、十方俱灭、神农尺、噬魂、天诛、惊邪、神舞、凤皇。持有者更迭频繁：虎魄多名兵主仍难完全镇压；太虚可被魔兵击碎再由西城秀树等重铸；惊邪可落入邪手；神舞战绩不佳曾被虎魄所伤；天诛箭尽成鸡肋；神农尺逆天治愈但需充能。问天最终成为天晶剑主，被推为武林盟主，守护神州的「正道顶点」成形——但正道内部仍有联姻政治、铸兵野心与伪善盟主。

⑤ **人物与势力绞杀（并行主线）**  
- **玄天邪帝（独孤星夜）**：独孤凛次子，元祖天魔一口魔气投胎；木姥姥以子眼换其魔眼压抑；成人礼被冷迎风弄盲反促魔眼重生；杀父母、毁神道盟、创邪道盟；被鬼谷先师阵法封印约百年后复出。持魔兵**星宿劫**（控周身时光）与**疚疯**（放大情感弱点、感官错乱），武功玄邪罡、天刀邪诀、三刀三剑三神技。魅力与毁灭并存，收**南宫太平**为义子，是系列人气反派/灰主角。  
- **燕王（慕容儁相关线）**：毒杀灭穹苍夺天地盟主之位，伪秩序；绝望催生魔兵**裂天破地**（裂天盾反震+破地锥强化），与灭鬼神等死斗。  
- **天地盟**：灭穹苍创盟；外征府天地双使（暴轰雷、炎神孛勒台实为罗刹卧底、凶阎王等）+金银铜玉四法王；内务府近卫、总务（雪总管等卧底线）、刑法、密侦、圣手、库财；森罗绝域妖兽军；铁壁铜场炼氏铸兵。问天曾假冒或正式任盟主，意图借其力抗魔，后推动邪转正/济世化。  
- **大罗刹宗**：西域最强，两使三圣结构，宗主与邪帝有巅峰对决；武勇、问菜等线交织。  
- **神道盟六门**：蛰伏百年后的正道顶尖之一，被邪帝屠灭后成血债与复仇引擎。  
- **四大家族**：南宫（剑与正统）、北冥（铸兵财富）、东方（海贸与心剑）、西城（孔道与四绝）。联姻宴会是天地盟最爱的屠宰场。

⑥ **第二部：十大魔兵与天魔复活之战**  
魔珠吸恨、惧、妒、绝望等负面情感生成魔兵；异魔/西方邪灵之主收集以释放天魔。神道盟复仇、邪帝与宗主决战为支线高潮。铁心妒忌问天等情感可催生**无妄**（模仿其他兵器异能，一次一种）；帝恨以怨毒烈焰熔物；悲怒权杖分离万物；虚无魔镜隐形聚光；执念障弱化圈；夔·帅贪念傀儡；十全魔兵统合九能，威胁天晶。高潮：问天以天晶子母剑合璧「**完美天晶**」，联合诸天神兵对抗天魔与魔籽**南宫太平**（问天与铁心之子，长生不死倾向，曾魔霸，后魔性被驱散）。魔兵毁灭后异能图纹化，人间留下「人体铭刻」后患。问天斩杀/镇压元祖天魔相关威胁，感化与杀伐并存，非单纯屠魔机器。

⑦ **第三部、前传与跨界（情报层）**  
数百年后主角**豪杰**集五柄小天晶重组新天晶对抗妖兵；中期硬融隋唐与《天子传奇》人物，口碑两极，讥评「神兵不三不四」。前传五辑补：邪帝/武勇巅峰战、婆罗门恩怨、乱世时间穿越、邪帝入魔前史、万神之神（宙斯/奥丁等神话扩展）。F系列清末至二战平行世界，天魔一族与历史人物客串。创世神兵、神兵4、科幻纪等为架空/未来层。**主库任务默认锁定壹+贰神魔兵主线**；跨时代须声明时间锚并降效。

⑧ **动画《神兵小将》差异（防OOC）**  
动画将神兵兽化、净化暴力与复杂身世：女娲造神兽，天魔派天地盟主魔化；问天得天晶兽，妹妹问雅净化魔兽。可用作低龄皮或支线彩蛋，**不可覆盖漫画正史的血腥、联姻政治、魔籽与邪帝灰线**。

**【主要人物】**
- **南宫问天**｜身份：南宫逸私生子，北冥伙头出身，天晶传人/剑主，武林盟主｜性格：正直嫉恶、草根重情、守正辟邪，成长中学会借力天地盟而不被吞没｜装备·能力：左臂绿珠/晶石之臂、天晶子母剑与完美天晶、天仁剑极等自创绝招、铸兵与厨艺｜人物弧光：哺妹孤儿→救雪入江湖→天晶剑主→抗天魔与魔籽的父亲｜立场关系：铁心正室、雪平妻、太平之父；与邪帝敌友纠缠；曾掌天地盟。

**【势力图谱】**
- **四大世家（正道骨架）**：南宫（剑统与盟主话语）、北冥（铸兵与财富）、东方（渤海盐贸）、西城（西域孔道与四绝）。宗旨各异，联姻可瞬间变屠场。
- **天地盟**：早期江湖第一大邪道势力。宗旨：上天下地、以力压武林。地盘：森罗绝域、铁壁铜场、外征/内务官僚网。代表：灭穹苍、燕王、天地双使、四法王、凶阎王。敌友：对四族是屠刀；对天魔是可被借力的刀；问天任盟主时出现「邪转正」张力。
- **邪道盟**：玄天邪帝所创，单靠邪帝即可横扫的顶尖邪盟。宗旨：血债与魔性秩序。代表：邪帝、刀剑双绝体系。
- **神道盟（六门）**：蛰伏数百年的正道顶尖，被邪帝屠灭后成复仇与悔恨之源。
- **大罗刹宗**：西域等同天地盟的霸权，两使三圣，与邪帝恩怨是第二部支线核弹。
- **拜火教十二宫 / 苗疆蛊族 / 鬼谷门 / 昆仑派等**：提供用毒、神算、铸兵、秘境支线，非主线可忽略但可作任务素材。
- **神话神域（前传/F）**：奥林匹斯、亚萨、婆罗门、天外天魔——主库低阶勿直接开战，作传说与跨界情报。

主线矛盾：正道世家 vs 天地盟/天魔系，神兵为催化剂；次线矛盾：邪帝血债 vs 神道与问天守护；隐线：神魔同体（太平）与兵器伦理（认主/反噬）。

**【大事记时间线】**
创世神魔战 → 十大天神兵铸造与魔珠埋设 → 约318年天晶献礼、剑岛灭、问天托孤北冥 → 十八年后四族会血洗、北冥雪被掳、虎魄重光 → 问天入江湖、结识铁心、神兵争夺升级 → 蚩尤秘窟/灭邪玄网、天晶意义显现 → 邪帝解封、大罗刹交锋、天地盟易主与问天借盟 → 魔兵连环催生、异魔集珠 → 完美天晶合璧对天魔/魔籽太平 → 魔性驱散与乱世暂定 → 数百年后豪杰重组小天晶（3/3.5）→ 前传补邪帝前史与万神 → F系清末二战平行。

**【叙事基调 · 雷区】**
港漫神兵热血：早期血腥脏话与草根粤语味，中后神魔玄幻与跨神话。忌写成严谨两晋正史；忌神兵人人能用、无反噬；忌把天魔/女娲写成一阶可斩；忌用动画《神兵小将》覆盖铁心/邪帝/魔籽成人向线；忌跨时代把宙斯奥丁直接扔进壹贰乱世当街战；忌把问天写成滥好人——他守正，但会掌天地盟、会借刀。最早切入锚点：问天仍在北冥伙房，或虎魄初现江湖、雪被掳前后。

**【序章空间毁灭机制】**
天晶母子刃出鞘使劣兵自碎，与天地盟特制惊邪类兵器对撞，能量灾难夷平剑岛——顶级神兵对撞是战略武器级，不是「两个人比武」。东方雄因性别与生育政治失宠，却在浩劫后成为南宫残部实际领袖。玉燕难产死，问天以血/内力哺妹，把「内力」写成生存本能而非炫技。北冥山庄作为天下兵库收留孤儿，为问天提供铸兵文化浸淫：他懂炉火，才懂神兵不是从天上掉下来的数字。

**【第一部结构：腥膻到玄幻】**
早期强调暴力与草根：虎魄要吃血才能醒，武林以命饲刀。中期蚩尤秘窟灭邪玄网把战场拉入上古凶神遗迹，元祖天魔从传说变成可被释放的实体威胁。后期天晶真正意义与邪帝、大罗刹入场，锺英伟等梳理后玄幻武侠色彩加深。神兵破碎后再以新生造型登场，异能近似，服务剧情与兵器周边——契约者世界里可表现为「碎片认主」「新生异形」，不可表现为无代价的满配刷新。

**【十大天神兵能力摘要】**
天晶：母子刃，诅咒与至强斩击，合璧完美天晶。虎魄：蚩尤系嗜血增能。太虚：黄帝正义轮/战车。十方俱灭：天机与大智慧空间。神农尺：医毒一体，充能。噬魂：罗刹精神/魂吸。天诛：九种异能箭，箭尽则废。惊邪：雷电音速。神舞：音波催眠。凤皇：控液与息壤。持有者战力可越一至两个阶层，心性不稳则反噬；「捡兵=升阶」是死亡旗。

**【十大魔兵与第二部】**
魔珠吸收恨、惧、妒、绝望、空虚、执念、贪等生成魔兵；异魔收集以释放天魔。帝恨烈焰；星宿劫时光三尺；疚疯感官错乱；无妄模仿异能；裂天破地反震+锐锥；悲怒权杖分离；虚无魔镜隐形；执念障弱化圈；夔·帅贪念傀儡；十全魔兵统合九能。完美天晶=子母合璧，联合其他天神兵，是对抗天魔与魔籽的关键仪式武器。魔兵毁灭后异能图纹化，留下人体铭刻隐患——任务可出现「剥离图纹」支线。

**【玄天邪帝完整弧光】**
独孤凛次子，天魔气投胎；木姥姥以子眼换其魔眼压抑；成人礼被冷迎风弄盲反促魔眼重生；杀父母毁神道盟建邪道盟；鬼谷先师阵法封印百年；复出后魅力与毁灭并存，收太平为义子，三刀三剑三神技与星宿劫、疚疯双兵。他不是「为恶而恶」的纸片反派，而是问天的黑暗镜像：同样重情，路径是血债与魔性。契约者与其互动优先「今夜不屠城」「护住太平神性」类条件目标，而非五阶以下硬刚击杀。

**【天地盟官僚魔窟】**
灭穹苍创盟，燕王毒师夺位。外征府：天地双使+金银铜玉四法王（玉法王为问天外公）。森罗绝域妖兽军与无常。内务府近卫、总务、刑法、密侦、圣手、库财——多层卧底使情报战与正面战同样重要。铁壁铜场炼氏铸兵，亦是裂天破地催生地。问天任盟主时，组织可从邪道屠刀滑向「抗魔工具」，滑移本身即剧情。

**【四大家族地缘】**
南宫太湖剑岛（后重建话语权）；北冥北地铸兵财富；东方渤海盐与海贸；西城扼西域孔道，四绝护院镇关。联姻与会盟是天地盟的屠宰场选择。西城秀树可重铸太虚，说明「正气+铸兵术」能修复神兵，但修复本身是大战代价后的叙事奖励。

**【第三部与跨界使用规则】**
数百年后豪杰集小天晶；硬融隋唐与天子传奇角色导致口碑分裂。前传五辑补百年前神道、婆罗门、乱世、邪帝、万神。F清末二战跨时代。写主线任务优先壹贰；衍生作单独声明时间锚。跨界神域人物在六阶以下只作传说、壁画、邪帝口中的「万神之战」情报。

**【契约者经济与兵器伦理】**
神兵不是掉落装：强夺轻则反噬重则灭门。北冥可定制地神兵，品质低于天神兵。魔兵图纹交易黑市危险。铸兵师、火凤凰用毒、苗疆蛊等支线提供非神兵成长路径。乐园币/魂币在土著眼中不存在——当地以精铁、盐引、西域宝石、铸兵契约计价（当地货币规则）。

**【神兵现世连锁反应】**
每当一件天神兵气息泄漏，正道世家、天地盟、邪道余孽、西域宗门会在数日内齐聚。先到者未必能认主，后到者可能捡走碎片。问天路线强调「心性与守护」；邪帝路线强调「魔与债」；燕王路线强调「伪秩序」。契约者插入时选择站边即选择敌人列表。六阶以下不要宣称能镇压天魔，应争取「让完美天晶合璧仪式不被打断」「护住太平神性一瞬」「夺回一枚魔珠」这类条件胜利。

**【铸兵与认主机制（AI可执行）】**
认主条件公开叙述常绑心性、血脉、绿珠类媒介与「是否愿付代价」。铸兵需炉火、奇金奇玉、息壤、神材与情感触媒（魔兵尤甚）。地神兵可量产倾向；天神兵/魔兵不可量产。破碎→新生可改造型但保留核心异能关键词。反噬表现：走火入魔、屠尽亲友、兵器反杀兵主、诅咒应验（天晶系）。

**【情感三角与家庭线】**
问天—铁心—北冥雪：不是轻浮后宫，而是乱世联姻、救命之恩与并肩作战的沉淀；铁心妒忌可催生无妄，说明爱情亦是魔兵燃料。问天—太平—邪帝：父子与义父子的神魔教育权之争。问菜/问雅线：问天护妹原点，动画强化净化，漫画中与邪帝疚疯情感相关。写对话用真名，忌「男主女主反派甲」。

**【战斗叙事口径】**
数值只进乐园系统指令；正文用神兵异能名、内功招式名、城毁岛沉的感官描写。天神兵对撞写「战略灾难」；邪帝星宿劫写「三尺内时光乖离」；疚疯写「感官背叛」；完美天晶写「子母合拢白昼熄灭」。禁止用现代热武器类比稀释港漫神兵神话感。

**【任务世界土著认知】**
土著知世家、盟、神兵传说，不知「契约者/乐园」。神兵现世时的恐慌、膜拜、抢夺是常态。问天成名后「伙头逆袭」成为江湖话本，可被说书人夸大。低阶契约者冒充天晶传人会被真内行（北冥铸师、英杰、法王）一眼拆穿。

**【名场面清单（可改编任务）】**
剑岛对撞灭岛；问天哺妹入北冥；凶阎王掳雪；虎魄饲血夜；四族会惊邪撕宴；铁心女扮男装露锋；蚩尤窟灭邪玄网；邪帝魔眼扫城；星宿劫斩宗主级；无妄振翅模仿凤皇；裂天破地炉中催生；完美天晶合璧对魔籽；魔兵图纹烙体。每场提供「护人/夺兵/断仪式/送情报」四类任务钩，而非只给击杀。

**【与天子传奇的边界】**
同属黄玉郎/玉皇朝宇宙，可有跨界企划，但神兵世界核心是「兵器认主与神魔同体」，天子世界核心是「命格与浑天/如来」。契约者不可把浑天宝鉴当成本世界默认掉落；跨界遗物须降效并标注来历。

**【世界观一句话备忘】**
神铸十兵镇魔珠，魔珠化兵借人心；伙头握天晶，邪帝握时光，魔籽握未来——谁先认主，谁就改写乱世。

**【地理移动与情报网】**
神兵现世点往往先出现「劣兵自鸣、禽兽躁动、铸炉异火」等前兆。西城四绝负责丝路封锁；北冥负责鉴定真伪；天地盟密侦最快；鬼谷/卓不凡系预言最贵。契约者可卖情报、护运图纸、假扮铸工，不必人人抢当兵主。

**【势力敌友速查】**
问天友向：铁心、雪、西城秀树（阶段性）、部分改造后的天地盟、铸兵义人。问天敌向：凶阎王系、燕王伪秩序、异魔、失控魔籽、心性堕落的神兵持有者。邪帝友向：邪道盟死士、太平（义父子）、部分「只恨神道」的遗民。邪帝敌向：神道遗孤、大罗刹宗主、早期问天。可转化：燕王、部分法王、武勇——取决于任务选择。

**【神兵持有者心态】**
天晶系：守护与诅咒并存，怕自己变成南宫逸。虎魄系：越杀越强越不像人。太虚系：正气自居，易被现实政治背叛。十方俱灭系：聪明到孤独。神农尺系：救人亦能下毒，道德灰色。魔兵系：每件兵都是一段悔恨，持有者常爱兵亦恨兵。

**【第二部终局条件胜利模板】**
1. 护住完美天晶合璧不被打断；2. 夺回/摧毁至少一枚关键魔珠；3. 在魔性吞没前唤回太平神性一瞬；4. 挡住十全魔兵统合完成；5. 护送神农尺完成战场急救链。六阶任务优先条件胜利，禁止无代价「一刀斩天魔」。

**【前传与F系使用边界】**
前传可写邪帝入魔前、武勇巅峰、婆罗门、乱世穿越，作五～六阶回忆副本。F系清末二战仅在乐园允许跨时代时开启，且神兵以「遗物/仿品/平行天晶」降效。科幻纪/大世界不作主库默认舞台。

**【语言与称谓】**
人名用南宫问天、南宫铁心、玄天邪帝、北冥雪、北冥正、东方雄、卓不凡、西城秀树、燕王、灭穹苍、凶阎王、玉法王、大罗刹宗主、武勇、南宫太平、豪杰等真名。兵器用天晶、虎魄、太虚、十方俱灭、神农尺、噬魂、天诛、惊邪、神舞、凤皇及魔兵全名。组织用天地盟、邪道盟、神道盟、大罗刹宗、四大世家。忌「系统」「升级」「经验」等出戏词进入土著口吻。

**【开局三选一（给AI选锚）】**
A. 北冥伙房：问天尚微，虎魄谣言刚起。B. 四族会血夜：护院视角，惊邪撕宴。C. 邪帝复出前后：城将破，魔眼将睁。三者皆可接神兵争夺，但敌人列表与情报层级不同。

**【世界之源与结算导向】**
推动世界之源的行为：阻止一次神兵对撞级灾难、促成完美天晶条件、驱散/抑制魔籽、揭露燕王伪盟、修复太虚、切断异魔集珠链。负向：无脑饲虎魄、把魔兵卖给异魔、对剑岛级对撞袖手。结算评级看乱世是否仍有「人间」可守，而非杀人数。

**【细节补完：绿珠与晶石之臂】**
玉法王植入的天晶精元绿珠是问天与天晶投缘的生理钥匙，后可演化为左臂晶石之臂等形态。它解释「为何伙头能握至强神兵」——不是运气，是血泪布局与女娲系认主。契约者若试图外科摘取绿珠，应触发诅咒/反噬/问天敌意，而非轻松获得天晶权限。

**【细节补完：无妄与铁心】**
无妄由铁心对问天的妒忌等负面情感催生，材料含地神兵凤翔与剑坟万剑，能模仿其他兵器异能（一次一种）。它把「女主吃醋」写成魔兵级世界观事件：在神兵宇宙，强烈情感是战略资源。铁心持无妄可出心剑/剑皇体系的爆发，亦可能被魔性反噬——写她时保留英杰之尊与情感代价双面。

**【细节补完：裂天破地与燕王】**
燕王无法保护妻女的绝望，在铁壁铜场融铁炉中催生裂天破地（盾+锥可合）。攻守兼备，曾助其击败灭鬼神级对手，自身亦伤重。这是「伪盟主也有人味」的样本：契约者可恨其夺位，亦可在其绝望点介入，改变魔兵催生与否。

**【细节补完：星宿劫与疚疯】**
星宿劫：焦躁情感+天诛箭等材料，控持剑者周身约三尺时光，近身几乎无敌。疚疯：悔疚疯狂情感，刀鸣扰乱感官，逼人与假象搏死。邪帝双兵合用可出「一剑藏空·千刀不尽」类完美神技。五阶以下正面对这套体系=送死；应设计抢时间、破焦点、护平民疏散。

**【细节补完：完美天晶】**
子母剑长期分开是平衡；合璧则威力暴涨至可对抗天魔/魔籽的仪式级。合璧需要心性、时机、诸神兵共鸣与战场掩护。六阶切入点的核心不是「你比天魔强」，而是「你让合璧成功」。

**【乱世社会面】**
两晋十六国皮：门阀、胡汉、流民、坞堡与武林世家叠图。神兵现世会打断正常战争与通商；盐引与精铁比黄金更硬。说书人把问天写成神话，把邪帝写成魔童，把虎魄写成会吃小孩的刀——谣言本身是战场。

**【反派逻辑层级】**
基层：抢兵发财（山寨、散修）。中层：以兵争霸（法王、使）。高层：以兵改命（燕王、宗主）。顶点：以兵复活天魔（异魔、魔籽失控）。写反派动机时按层选，避免所有小兵都喊「为天魔万岁」。

**【伙伴与对手可转化表】**
西城秀树：友，修复向。武勇：可竞可盟。大罗刹宗主：敌，可在抗邪帝时临时盟。燕王：敌，绝望点可转化。邪帝：敌，护太平时可短暂默契。太平：子辈，魔性时敌、神性时友。铁心/雪：核心友，妒忌/被掳时产生支线危机。

**【禁止事项清单】**
禁止无反噬使用天晶屠城后毫无代价；禁止低阶契约者一章内集齐十大天神兵；禁止把魔籽太平写成无脑黑化工具人；禁止用现代枪械轻易击毁神兵；禁止把天地盟写成单一反派脸谱（有卧底、有改邪、有问天任盟主）；禁止忽略铸兵经济只写打打杀杀。

**【推荐任务链示例】**
一阶：伙房举报细作→护住问菜→获铸兵边角。二阶：四族会突围→夺回一张神兵图纸。三阶：虎魄祭夜救饲刀学徒→英杰榜露名。四阶：灭邪玄网破一层→获法王级线索。五阶：邪帝压城夜疏散→换取「今晚不屠」默契。六阶：合璧仪式守阵眼→条件胜利驱散魔性。

**【世界观收束句】**
在这片神兵与魔兵互相照见的乱世里，真正决定认主的不是手掌的力道，而是一颗是否还愿意为他人流血的心——南宫问天以此握住天晶，玄天邪帝以此握住血债，南宫太平以此在神魔之间跌倒又爬起。契约者入世，只是又一只伸向神兵的手；手伸出去之前，先问自己：你是来饲刀，还是来护人。

## 阶位切入点

> 阶位↔：一阶≈山寨伙头；二阶≈护院精英/盟基层；三阶≈青年英杰/地神兵主；四阶≈家主/法王/双使下游；五阶≈天神兵主力兵主/邪帝交锋；六阶≈完美天晶与天魔/魔籽决战。顶点（女娲/天尊/天魔本尊级）超阶，低阶规避·情报优先。乐园阶位映射见剧情段，宁低勿高。

**一阶（北冥伙房·饲刀前夜）**
切入身份/时点：北冥山庄伙头同事或运炭小工，**南宫问天**尚在灶间打杂、绿珠未显威之时。
初始事件：铸兵炉夜半异响，虎魄重光的谣言随西城商队传入；天地盟细作夜探粮仓，欲在四族会前摸清北冥兵力。
开场白建议：「铁屑拌进粥里都有人吃。问天用手指给女娃喂血的旧事，厨子还当笑话讲。今夜山庄警铃响了——不是催班，是催命。你抓起烧火棍时，才发现棍子在抖，像也听见了远方那柄要吃人的刀。」
关键NPC立场：**南宫问天**（拉你守后门或护妹）；**北冥雪**（尚娇气，不知自己将被掳）；**北冥正**（家主，视你为可消耗的耳目）；天地盟探子（可抓可放，放则引二阶灾）。
主线钩子/支线：护住幼妹问菜/问雅不被细作摸清；支线=举报或反向卖假情报；支线=偷听铸师争论「虎魄是否已醒」。
危险度/规避：中。规避任何天神兵真身与法王；见惊邪电光立刻趴下。
任务方向/奖励：铸兵边角料、进入世家外围名册、问天的人情（比银子硬）。

**二阶（四族会盟护卫）**
切入身份/时点：西城或东方护院、四绝外围弟子，四族聚会遭天地盟突袭时。
初始事件：寿酒未敬完，兵器自鸣；天尊使/地尊使级压力压场，惊邪电光撕开宴会，**北冥正**与**东方雄**同时拔剑。
开场白建议：「东方雄的袖剑出鞘半寸，已有人倒下。你的刀在神兵面前像烧火棍——但烧火棍也能挡一次致命。西城秀树在长桌那头吼：护住家主图纸！你听见自己回答：收到——声音却像别人的。」
关键NPC立场：**东方雄**（冷傲下令突围）；**西城秀树**（英杰向，可结盟）；**北冥正**（护女失败的前奏）；**凶阎王**（掳雪执行者，二阶不宜硬刚）。
主线钩子/支线：护送家主突围；支线=夺回一张神兵/铸兵图纸；支线=掩护北冥雪车驾（失败则衔接到救雪线）。
危险度/规避：高。规避双使正面与虎魄真身；优先救人与保图。
任务方向/奖励：英杰推荐信、地神兵试用权、四族之一的短期客卿身份。

**三阶（青年英杰榜·虎魄祭夜）**
切入身份/时点：十大青年英杰周边挑战者、苗疆用毒客、或刚认主一件地神兵的新兵主。
初始事件：武林以血喂刀，虎魄半醒；**南宫铁心**以剑锋指你：让路或并肩；**南宫问天**在血雾里找北冥雪的下落。
开场白建议：「榜上有名的人眼睛都红。虎魄要吃血才能醒。你分不清自己是来成名，还是来送血。铁心说：站我这边，就别把手伸向问天的命；站另一边，就先过我的剑。」
关键NPC立场：**南宫铁心**（英杰之首，试你心性）；**南宫问天**（救雪主线，可拉你入伙）；**西城秀树**；**卓不凡**（远处算子，卖预言极贵）；古柔/灵剑子等英杰炮灰位。
主线钩子/支线：助问天接近虎魄真相或救下雪；支线=苗疆用毒破饲刀阵；支线=阻止山寨把学徒抛去饲刀。
危险度/规避：高。规避邪帝（若已解封则本阶直接降为逃）；虎魄反噬区只可外围作战。
任务方向/奖励：成名战记录、神兵残识（线索非认主）、地神兵稳固认主。

**四阶（法王/灭邪玄网）**
切入身份/时点：灭邪玄网或四法王作战圈的成名高手、铸兵大师护卫、或天地盟反正的内应。
初始事件：银法王万德威类强者布阵，森罗妖兽出闸；**玉法王**在暗处观察问天左臂绿珠是否亮起；**燕王**的伪盟主令从铜场传出。
开场白建议：「蝠影盖住月亮。法王的笑声像银币碰撞。问天在你身侧低声：神兵会选人——你先别死，才有被选的资格。灭邪玄网的纹路在地下爬，像要把整座山缝进天魔的胃。」
关键NPC立场：**南宫问天**（已近天晶剑主之路）；**玉法王**（可助可试）；**燕王**（伪秩序，可揭露）；**炎神**/**大罗刹**卧底线相关；森罗妖兽群。
主线钩子/支线：破一层玄网或守住一个阵眼；支线=揭露燕王毒师夺位证据；支线=护送神农尺残页到伤员营。
危险度/规避：极高。规避天魔真身与完美邪帝双兵；法王可战但需地形与内应。
任务方向/奖励：法王级功法碎片、天神兵线索、天地盟内应身份、铸兵奇材。

**五阶（邪帝与宗主·魔眼压城）**
切入身份/时点：玄天邪帝解封后的邪道盟扩张期，或大罗刹决战前夜；你已是天神兵候选兵主或一方盟临时将。
初始事件：星宿劫出鞘，百年魔气压城；疚疯刀鸣让守城军自相残杀；**南宫太平**的魔性在血味中躁动。
开场白建议：「城墙上的人跪了一片，不是礼，是腿软。邪帝说他来收神道欠的血债。你站在问天与邪帝之间，像站在神与魔的刀锋——你的任务或许不是赢，是让这座城天亮时还在。」
关键NPC立场：**玄天邪帝**（可杀可谈，五阶难真杀）；**南宫问天**（需要你守侧翼）；**大罗刹宗主**（可临时同盟抗邪帝或相反）；**武勇**；**南宫铁心**（无妄可能已现）。
主线钩子/支线：阻止屠城/疏散；支线=保护太平不被魔性吞噬；支线=在星宿劫三尺外抢一秒送出合璧材料。
危险度/规避：极高。超阶边缘情报：勿挑战女娲/天魔本尊叙事位；对邪帝优先条件目标。
任务方向/奖励：魔兵图纹抗性、邪帝暂时默契、天神兵认主考验、武林盟文书。

**六阶（完美天晶 vs 天魔/魔籽）**
切入身份/时点：第二部终局，完美天晶合璧对天魔或魔籽**南宫太平**失控之夜；你是守阵英杰、持一兵共鸣者或问天托孤的最后棋子。
初始事件：十大神兵共鸣，子母刃将合未合；异魔催动魔珠链；太平魔气吞天，铁心喊他的名字喊到失声。
开场白建议：「子母刃合拢的刹那，白天熄了。问天喊出太平的名字，像喊自己的命。你把最后一枚神兵扔进光柱——不是为了成神，是为了让人间还在。远处邪帝的魔眼眯起，不知是在笑，还是在哭。」
关键NPC立场：**南宫问天**（合璧核心）；**南宫太平**（魔/神双态）；**南宫铁心**/**北冥雪**（唤回神性的情感锚）；**玄天邪帝**（义父线可助可乱）；**元祖天魔/异魔**（顶点，条件性胜利）。
主线钩子/支线：维持神兵阵一角直到合璧完成；支线=唤醒太平神性；支线=切断一枚魔珠或挡住十全魔兵统合；支线=用神农尺保住问天/铁心不倒。
危险度/规避：贴近顶点。天魔=超阶条件性胜利；严禁削弱式开挂灭神；失败表现应为「人间崩一角」而非玩家无敌翻盘。
任务方向/奖励：劫后神兵残片、乱世和平契约叙事、图纹剥离术、武林史册留名（及乐园侧世界之源高评价）。

**三阶补场·虎魄饲刀学徒**
武林人士抓铸工学徒饲刀。你可救下学徒换北冥铸师人情，或眼睁睁看虎魄更醒——后者会加速四阶灾难。关键NPC：**南宫铁心**（可助一剑）、饲刀头目（可杀可审）。

**五阶补场·无妄振翅夜**
铁心妒忌与战火催生无妄，剑甲展开模仿凤皇软化息壤，助某人（甚至邪帝）破局。你的选择：帮铁心稳住心性，或利用无妄抢一息壤通道。关键NPC：**南宫铁心**、**南宫问天**、**玄天邪帝**。

**四阶补场·铜场绝望**
燕王被投入融铁炉的绝望边缘。你可递绳、递刀、或递一句「你妻女尚在」——可能阻止或改写裂天破地催生。相关NPC：**燕王**、铜场囚徒铸师、灭鬼神追兵。
