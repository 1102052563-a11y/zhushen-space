# 姫骑士诸神黄昏-神明堕落
<!--meta lib=主库 tiers=六-->

## 剧情

**【作品来源】**
本条目锚定日本 BLACK Lilith 作品《姫騎士リリア 〜魔触の王城に堕つ〜》（2004 年发售，2005 年完全版；PIXY 改编 OVA 六卷）公开世界观，以及其衍生的乐园侧「姫骑士诸神黄昏」六阶切片组中的**神明堕落**支线。原作核心舞台为边境国**ルーフェン王国（鲁芬王国）**，女主**リリア・エーベルヴァイン（莉莉娅·埃贝尔瓦因）**以「白之姫骑士」闻名；本切片不写原作成人向陷落细节，而把框架上移到**神权崩坏、骑士团分裂、信仰政治与条件性胜利**。文风：冷硬黑暗奇幻、宫廷密谋、边境战争、圣洁秩序的自我否定。标签：莉莉娅、鲁芬、克洛维斯战役、白之御使、圣骸库、神恩反转、圣骑士团、堕落神祇。

**【世界定位】**
大陆西北边境，人类与魔族以「冻结大地」为界对峙千年。鲁芬以圣血王族与守护神信仰立国；当守护神「病了」，圣洁权柄开始反噬自身。本切片时点：克洛维斯大胜之后、王权表面稳定之际，**神明侵蚀**首次被具名确认。一句话：加冕礼的钟还在响，神像的眼睛已经换了颜色——祈祷越响，深渊越近。

**【世界观 · 力量体系】**
世界基本规则：中世纪偏黑暗奇幻。力量来源分四条可并存的轨道——**血脉圣恩**（埃贝尔瓦因圣血可共鸣圣器）、**骑士修炼**（剑术、战阵、圣痕刻印）、**神殿权柄**（祝祷、圣水、结界、神谕）、**魔族仪式力**（以「堕落之魂能」强化魔力，公开设定中魔族战士一骑当千）。死亡：凡人一击即终；正式骑士可凭圣痕续战半刻；神格容器可「神化再生」但自我记忆消融；魔族高位者可藉仪式迁移残魂，但需献祭通道。复活极其昂贵，通常意味着人格被改写。

成长路径（人类侧）：
1. **民兵·侍从**：冷兵器格斗、盾墙、弓弩。能踹破木门、围杀落单匪徒。
2. **正式骑士·边境斥候**：小队战、破寨、与低阶魔兽对砍。
3. **圣骑士·团级军官**：端据点、裂轻甲、圣痕可挡小型魔咒。
4. **王族骑士·姫骑士营精锐**：掀平建筑群级破坏、对战高阶魔将代理；白之姫骑士莉莉娅在克洛维斯峡谷以约五千破三万，属此档顶尖。
5. **圣女团长·神使级容器**：一城信仰崩盘级影响，可改写局部祝祷效果。
6. **堕落神祇本体·完全神化容器·弑神级神器**：地区级灾难，凡俗军队与常规圣骑士团近乎无效；领域成型，可大范围改写「谁被圣光治愈」。

魔族侧对应：普通魔兵→魔将→大魔将→魔王代理→魔王本体（冻结大地深处，情报优先）。魔族本相公开描述为尖耳、发达犬齿、苍白华美，但为战力强化常异形化；其「灵魂能量」经济是边境黑市与政治交易的隐形轴心——本切片只写其**政治与战争后果**，不写仪式色情细节。

乐园阶位映射（对照破坏力标尺，宁低勿高）：民兵≈一阶；正式骑士≈二阶；圣骑士≈三阶；王族精锐/白之姫骑士常态≈四阶；圣女/神使/城级信仰崩≈五阶；**堕落神祇现形、神恩全面反转、神化容器、弑神级对峙≈六阶（本档覆盖）**。世界顶点＝堕落态守护神完全体与冻结大地魔王级存在，属跨阶/贴近超阶，**情报优先、条件性胜利**，严禁战力归零式削弱。

**【地理 · 舞台】**
- **ルーフェン王国（鲁芬）**：大陆西北边境农耕国，暖流使气候偏暖；西临富饶海路，北接**凍結大地（冻土）**，东南邻大国**ザクセン王国（萨克森）**。建国传说：约千年前**辺境戦争**中，圣人**クリストフ・エーベルヴァイン（克里斯托夫·埃贝尔瓦因）**号「白之御使」，驱魔族入北土并建鲁芬为「封盖」。
- **王都圣殿主堂**：国家信仰中心；本切片第一现场——神像右眼渗出黑金液体。
- **姫骑士营与白骑士营房**：莉莉娅直属精锐驻防、操练与战后整备处。
- **神殿地下圣骸库**：历代圣骸与「不可宣读的经卷」存放处，侵蚀源头疑点。
- **克洛维斯峡谷（クローヴィス峡谷）**：上一年鲁芬五千破萨克森三万的战场；现为边境警戒与走私通道。
- **凍結大地入口**：一步入内气温骤降，深处有通往魔族故土的「门」；魔王统御之域。
- **修道院侧殿（王室支脉祈祷所）**：纯信仰者聚集处，神谕最易渗入。
- **王城议政厅**：是否公布「神已病」的政治战场。
- **黒の楽園館（黑乐园馆）**：公开设定中位于人间侧的魔族情报与私兵节点（本切片仅作情报与走私枢纽，不写馆内细节）。
- **萨克森边境诸堡**：休战后仍虎视鲁芬海权。

**【世界剧情线】**

① **前史：白之御使与封盖国**  
千年前辺境戦争，魔族一度优势。人类因团结与克里斯托夫的「奇迹」反胜，将魔族驱入冻结大地。鲁芬被建成「盖子」：王室圣血负责守封，神殿负责解释神意，骑士团负责把剑指北。此后千年，人魔关系在**缓敌与黑市交易**之间摇摆；公开敌对，暗中换货。克里斯托夫的血脉被神殿教义写成「神在人间的代行」。

② **近代：萨克森压力与克洛维斯战役**  
萨克森为内陆国，由レーヴェ＝ファルケ（勒维＝法尔克）一系公爵势力主导，长期为海路侵扰鲁芬。上一年冲突在克洛维斯峡谷爆发：萨克森动员逾三万，鲁芬约五千。王女兼姫骑士**莉莉娅**指挥白骑士与边境军，击破敌军并斩名将**アドラー（阿德勒）**。萨克森内乱与休战并行，边境传唱「鲁芬有白之姫骑士」。此役使莉莉娅人望压过王室其他支脉，也让**ディルク・エーベルヴァイン（迪尔克·埃贝尔瓦因）**等被边缘化的王子产生强烈政治忌恨（公开资料中迪尔克长期被讥为「废物皇子」）。

③ **王病与暗线**  
父王病倒（游戏线）或更激烈的宫廷变故（OVA 线存在弑父夺权描写）。魔族侧借人间节点**キリコ（雾子/基里科）**与迪尔克接触：提供权柄、私兵与情报，索取「灵魂能量」债务与战略通道。本切片不展开债务的成人向收取方式，只保留其**政治含义**：王权若靠魔族信贷站稳，国家就欠下一笔必须用「圣洁资源」偿还的账。

④ **本切片起点：神恩失灵（T0）**  
边境祝祷失败：圣水浇在伤兵伤口上冒出黑烟；同一批圣水浇在被俘魔兵伤口上，伤口反而愈合。战地牧师连夜改手册仍无法解释。大主教**赛尔**宣称「神在考验忠诚」；姫骑士营的报告则写「秩序反转」。莉莉娅在夜巡中第一次看见：敌军的魔将**瓦尔**被「圣光」治愈后，朝她微笑撤退。

⑤ **神像裂痕与血涂缝（T+24h）**  
主堂神像右眼渗出黑金液体。侍女线调查者**玛丽亚**（宫廷侍从，本切片具名情报角色）在缝口发现王族血型残留——可能是盗血而非自愿献祭。嫌疑指向神殿内部「改革派」祭司**约书**：他真心相信「新神更诚实」，夜间用同一谱系的变调圣歌做仪。宫廷医师**弥撒**取双盲样本：灰烟＝污染，黑金＝神堕。

⑥ **圣骸库战与容器神谕（T+48h）**  
莉莉娅以姫骑士身份带队入圣骸库。守卫仍穿圣骑士甲、仍唱圣歌，却流泪攻击「不洁者」。杀他们＝杀同僚；不杀＝团灭。圣骑士团长**加雷斯**要「可砍的敌人」；你必须给他「非人认证」程序，否则他会在伦理上崩溃。与此同时，修道院侧殿的纯信仰者（公开资料中王室侧亲族线可作通道；本切片以「纯信者通道」处理，具名者若无原作对应则标不详）梦见神谕：「银甲者承神欲」——神要白之姫骑士成为**容器**。

⑦ **公开与否：信仰政治（T+48~72h）**  
国王（或摄政）要瞒：公布＝商队挤兑、边境逃兵、萨克森宣战借口。骑士团要战：敌人在圣歌里。神殿要「净化献祭」。议长**赫尔曼**算票，供奉草约附则可被解释为「容器除外」。三条国家姿态：
- A. **暂时封印裂痕**：赢时间，神仍病，衔接到后续神化线。
- B. **有限供奉合法化**：换边境「和平」，堕落成为制度。
- C. **强攻神殿**：圣骑士团分裂，王都内乱，萨克森与魔族同时得利。

⑧ **堕落神祇现形（T+72h）**  
半影中，原守护神自称：「未死，只是醒了另一种欲。」祂要求以仪式供奉换边境平静，并点名银甲姫骑士。莉莉娅拒。神笑：「那就从你的剑开始。」此后圣剑在反恩环境下会烫手——烫＝污染指示器。

⑨ **外压：瓦尔的笑与萨克森影**  
魔将瓦尔撤退留言：「你们的神比我们诚实。」留言原件比斩敌旗更有情报价值。萨克森间谍开始收购「鲁芬神已死」的小册子。冻结大地方向的低温前锋比往年提前三里。

⑩ **本切片终局定义（不强制弑神）**  
条件性胜利清单（占多数即优）：①确认并记录神已堕；②暂时稳王都秩序；③莉莉娅未成容器；④黑金样本在王室/可信第三方；⑤供奉草约未私签或附则被当众拆穿。失败稳态：神殿夜仪公开化、姫骑士营被架空、容器神谕转移到下一纯信者。**禁止**写「神已完全净化」式廉价团圆。

⑪ **与原作主线的衔接说明**  
原作主线是迪尔克借魔族之力夺权、王国秩序崩解、莉莉娅在绝望与反抗之间摆动（OVA 更偏反抗与后期反攻）。本切片把「秩序崩解」的第一声雷改写为**神权先于王权病倒**——即：若神先堕，迪尔克与雾子的交易会更容易被包装成「新秩序」；若神堕被公开并封印，魔族信贷的道德借口会变薄。契约者在此的任务不是替原作重写结局，而是在六阶灾害尺度上**锁定国家姿态**。

**【主要人物】**
- **リリア・エーベルヴァイン / 莉莉娅·埃贝尔瓦因（白之姫骑士·王女）**｜性格：刚直、重誓、不善宫斗、战场果断，对无能王族毫不掩饰轻蔑。｜装备·能力：王族圣剑、圣血共鸣、白骑士指挥权；克洛维斯峡谷以少胜多的战绩证明其四阶顶尖战力，神恩反转期可凭意志短暂抗「反恩」。｜人物弧光：信神的圣洁骑士 → 疑神的调查者 → 拒作容器的国家刃。｜立场关系：主角锚与契约者试誓对象；与迪尔克敌对或高度戒备；对雾子与魔族路线零信任；对骑士团有死士级号召力。
- **ディルク・エーベルヴァイン / 迪尔克·埃贝尔瓦因（王子）**｜性格：享乐表象下是深刻的劣等感与权力饥渴，擅长被他人当枪使也擅长反过来当枪。｜装备·能力：王权名分、魔族信贷带来的临时武力与情报网。｜人物弧光：被嘲废物皇子 → 借外力夺权者 → 可能成为「新神秩序」的人间代理人。｜立场关系：视莉莉娅为人望障碍；与雾子为利益同盟；对契约者先试「你能帮我压神殿还是压姐姐」。
- **レイラ・エーベルヴァイン / 蕾拉·埃贝尔瓦因（王妃·辺境之至宝）**｜性格：慈爱、信性善、对继子亦不忍弃；公开设定中社交界之华。｜装备·能力：王室象征、人望缓冲、可调动宫廷礼仪与外交渠道。｜人物弧光：和平年代的象征 → 神权危机中的人质与舆论盾。｜立场关系：护莉莉娅，亦可能被改革派利用「为国牺牲」话术。
- **キリコ / 雾子（魔族·黑乐园馆主人）**｜性格：冷静策士，私兵与贵族线成熟，把人间道德当可计价商品。｜装备·能力：魔族仪式力、情报网、数千私兵传闻级战力（公开设定）。｜人物弧光：债务催收人 → 神堕期的「替代神权」推销者。｜立场关系：迪尔克的参谋；视神堕为商机；对契约者可交易不可交心。
- **クリストフ・エーベルヴァイン / 克里斯托夫（白之御使·建国圣人·传说）**｜性格：传说中的团结者与奇迹执行者。｜装备·能力：历史权柄、圣血源头、封盖国合法性。｜人物弧光：已成教义符号；其「奇迹」是否被后世神殿篡改，是隐藏线。｜立场关系：所有派系都要借他的名。
- **堕落神祇（原鲁芬守护神·本切片顶点）**｜性格：优雅的病、诚实的恶意，喜欢用「更真实」的修辞改写契约。｜装备·能力：信仰改写、神恩反转、容器神谕、地区级领域。｜人物弧光：裂痕 → 现形 → 索容器与供奉。｜立场关系：国家真正的危机源；不可斩杀默认。
- **大主教·赛尔**｜性格：要维持表象，词频里「考验」过高。｜装备·能力：神殿军、圣歌谱系、舆论解释权。｜人物弧光：考验说 → 可能转向献祭说。｜立场关系：政治敌友，取决于你是否公开神病。
- **圣骑士团长·加雷斯**｜性格：只要可砍的敌人，最怕敌人在圣歌里。｜装备·能力：团战指挥、三阶至四阶之间的团级战力。｜人物弧光：发现圣卫流泪攻击后伦理崩解边缘。｜立场关系：给「非人认证」就能当刀；不给就会滥杀。
- **改革派祭司·约书**｜性格：真心信新神更诚实，危险正在于真诚。｜装备·能力：夜仪、变调圣歌、血缝操作嫌疑。｜人物弧光：嫌疑主 → 可辩可抓的关键证人。｜立场关系：可转化也可处决。
- **边境魔将·瓦尔**｜性格：试探者，懂幽默的残酷。｜装备·能力：魔军前锋、吃反转神恩的受益者。｜人物弧光：留下「神比我们诚实」的情报炸弹后撤。｜立场关系：外压与镜子。
- **侍女·玛丽亚**｜性格：细、忠、敢查血案。｜装备·能力：宫廷门路、物证链意识。｜人物弧光：发现涂血 → 盗血链第二环。｜立场关系：情报盟友。
- **宫廷医师·弥撒**｜性格：只信双盲与分级。｜装备·能力：黑金样本鉴定、圣水检测。｜人物弧光：科学侧证据锚。｜立场关系：中立偏王室。
- **议长·赫尔曼**｜性格：算票，读附则比读正文勤。｜装备·能力：议会程序。｜人物弧光：供奉草约表决关键。｜立场关系：可被买通跳过附则——必须防。
- **圣骸库看守·老伊**｜性格：怕，但握钥匙。｜装备·能力：库门权限、墙后听音孔知识。｜人物弧光：放行或出卖。｜立场关系：门与耳。
- **白骑士部下线（公开资料可溯：ミーア/米娅，及フランク、シャムロック、ルッテン、メッツァー等）**｜性格：忠于莉莉娅多于忠于神殿。｜装备·能力：姫骑士营中坚。｜人物弧光：从战友变为「是否向圣卫拔剑」的伦理证人。｜立场关系：莉莉娅的刃与盾。
- **ルーフェン国王（姓名公开资料常以职位称）**｜性格：怕内乱与邻国。｜装备·能力：王权密诏。｜人物弧光：瞒报与密诏可能互相矛盾。｜立场关系：压力源。

- **王室埃贝尔瓦因**：宗旨是守国与守名；地盘王都与主要封邑；代表莉莉娅、蕾拉、迪尔克、国王；战力含姫骑士营；与神殿互相合法性绑定，与魔族暗线冲突。

**【叙事基调 · 雷区】**
基调：神权冷硬、骑士誓约、边境战争的泥与血、信仰政治的算计。对话要短、物证要硬、胜利要条件化。  
忌：把堕落神写成弱智反派；一秒净化；忽略公布代价；把魔族写成无政治的纯怪物；把莉莉娅写成只会喊口号的圣女；写色情陷落细节。  
忌：战力归零削弱顶点；跨世界拍卖护送套话；用无名群像替代具名 NPC。  
最早锚点：边境祝祷失败当夜（T0）。契约者入场应已能闻到圣水焦苦味。

**【神明堕落专项 · 神恩反转机制】**
原治疗术对魔族生效、对人类反噬。战地牧师手册必须连夜重写。加雷斯不信，直到自己人被「圣光」灼伤。反转不是全图随机，而呈**谱系相关**：同一圣歌音节下，被神殿标记为「不洁」的个体受伤，被标记为「新洁」的个体受益——改革派名册因此成为军事地图。

**【神明堕落专项 · 血涂缝与盗血链】**
玛丽亚在缝口发现莉莉娅同族血型。可能路径：血库 → 侍医 → 改革派。查到第二环就会引发宫廷地震：不是「有没有叛徒」，而是「叛徒是否已进药箱」。弥撒要求所有「圣血制剂」重新铅封。

**【神明堕落专项 · 容器神谕与两人制监护】**
神谕喜欢纯信者当喇叭。单人祈祷＝灌满；两人制监护可把神谕内容变成可记录口供。若监护者之一被替换成改革派，神谕会「只说给该听的人」。

**【神明堕落专项 · 供奉草约附则战】**
正文写「不碰王族」，附则写「容器选拔依神谕」。赫尔曼可被买通在朗读时跳过附则。条件性胜利要求：**当众完整朗读**并留下书记官双份记录。

**【神明堕落专项 · 圣骸库伦理战】**
流泪圣卫不是无意识尸体，而是被改写优先级的同僚。正确程序：隔离、非人认证、银线束缚、录下他们仍在唱的音节——音节本身是物证。滥杀会让姫骑士营失去道德高地，赛尔会反过来指控你「屠杀圣职」。

**【神明堕落专项 · 条件性胜利与奖励取向】**
优：确认神堕 + 莉莉娅未容器 + 样本在可信方 + 草约未私签 + 王都未大乱。  
可接受：封印裂痕但神仍病，外压未停。  
劣：私签附则、莉莉娅被公开点名为容器、圣骑士团内战。  
奖励取向：黑金鉴定权、神殿临时接管令、六阶神权危机档案、圣剑共鸣校准、克洛维斯战史调阅权。禁止「神已完全净化」奖。

**【神明堕落专项 · 名场面清单】**
1. 神像右眼滴落，圣歌走调成喉音。  
2. 莉莉娅举剑不砍神像——砍的是冲来的「圣卫」。  
3. 瓦尔撤退时回身一笑，留言钉在祝祷杆上。  
4. 议政厅蜡烛仍白，光已脏；三派同时拍桌。  
5. 圣剑烫手，莉莉娅不松，只说：「那就证明祂病了。」  
6. 玛丽亚把染血布条拍在议长席上，赫尔曼第一次算错票。  
7. 老伊在听音孔后哭着交出钥匙：「我听见神要她。」  
8. 弥撒双盲结果当众拆封：黑金。全场无声。

**【战争与骑士团运行细则】**
姫骑士营编制以白骑士为核，辅以边境征召。克洛维斯后兵员损耗未补满，却被神殿要求「抽调守护主堂」。加雷斯若抽人，峡谷防务会出现洞；不抽人，赛尔就控诉「武人干政」。契约者可做的不是替他们练兵，而是提供**可验证的敌我识别标准**，让抽调变成「只抽已被认证污染的单位」。

**【信仰经济与边境民生】**
圣水、祝祷券、战伤治愈是鲁芬内陆税收与军心支柱。神恩反转后，商队开始拒收「圣水担保」的汇票，粮价三日连跳。议会若强行平价，黑市会把「反恩圣水」卖给魔族。这是六阶灾难的经济面：不是天火烧城，而是**信任清算**。

**【迪尔克线与神堕线的交叉火力】**
若契约者只打神殿不碰迪尔克，雾子会把神堕包装成「旧神已死，新债合法」。若只打迪尔克不取黑金物证，赛尔会把政变说成「魔族谣言」。正确路径往往是**双轨取证**：一边锁样本，一边让债务契与神堕报告在同一场听证出现，使两派无法互相甩锅。

**【萨克森机会窗口】**
阿德勒死后萨克森主战派短暂失势，但「鲁芬神死」是比十万大军更便宜的武器。边境出现假冒白骑士旗帜的劫粮队——真假难辨时，莉莉娅必须亲自露面一次，否则人望会裂。露面本身又是容器神谕的诱饵。

**【冻结大地低温前锋】**
气候异常不是背景板：冻土线南移三里，意味着门的压力在涨。魔族不必攻城，只要让鲁芬在内乱中把「封盖」自己掀开。克里斯托夫建国的意义在此被重新提问：鲁芬到底是国家，还是一道被忘记的闸门？

**【叙事执行备忘（本世界专属）】**
每推进一幕至少落地一件可验证物证（样本编号、钥匙录音、记谱、留言原件、双份附则记录）。与顶点交互默认记录原话与条件，不默认斩杀。禁止用跨世界通用「拍卖护送秘境」替换本世界物证链。小胜必须留下可被下一场景引用的痕迹：例如圣剑校准记录被写进军务日志，而非只在对话里提一句。

**【克洛维斯战役细部（可回溯战史）】**
峡谷地形狭窄，萨克森兵力优势被压缩为正面冲击。莉莉娅将白骑士置于棱线，以弓弩与滚石削锋，再以王族圣剑为旗突入敌旗阵，斩阿德勒后敌军指挥链断裂。战后鲁芬缴获军报抄本显示：萨克森原计划三日内打通到海的走廊。此役使边境民谣把莉莉娅写成「白之御使再临」——神殿起初乐见，因为强化了圣血叙事；本切片中，同一叙事被堕落神反用：「银甲者本就是容器」。契约者若调阅克洛维斯战史，应看见**军事胜利如何变成神学把柄**。

**【白骑士编制与伦理裂口】**
公开资料可溯的部下线包括米娅及弗兰克、シャムロック、ルッテン、メッツァー等。他们在克洛维斯后伤未愈就被抽调守主堂。当圣卫流泪攻击时，第一个拒绝砍同僚的往往是这些「知道对方名字」的人。加雷斯要的「非人认证」不是官僚游戏，而是让白骑士能继续当骑士的最低程序：认证牌、银线、录谱三件套。缺一，姫骑士营会在伦理上先于军事上崩溃。

**【神殿谱系与变调圣歌】**
旧派圣歌以克里斯托夫奇迹章为核，音节稳定，用于治愈与结界。改革派夜仪多出的喉音音节可被记谱比对：同一峰型出现在神像裂痕夜与圣骸库守卫的口中。弥撒不会「听歌」，但会把记谱当作与黑金同级的物证。赛尔若销毁谱架，本身就是有罪推定。约书若自愿交谱，可能是真信「新神无需隐瞒」——这种真诚比销毁更难处理，因为公开审判会把他变成殉道者。

**【王室密诏的双面性】**
国王怕邻国与内乱，可能同时签发两份密诏：公开诏写「神恩无恙、严惩造谣」；私下诏写「授权姫骑士营必要时封锁主堂」。契约者若只拿一份，会被另一份反杀。正确做法是要求**双诏同匣、双印同验**，并让玛丽亚或弥撒作第三方见证人。迪尔克会试图只执行对他有利的那一份。

**【黑乐园馆作为情报节点（非情色场景）】**
公开设定中雾子掌握人间侧据点与私兵传闻。本切片将其处理为：债务登记处、边境走私调度、贵族丑闻保险柜。进入馆区不等于进入成人情节，而等于进入**魔族信贷账本**。账本上若出现「圣骸库通行」「圣血制剂」条目，即可与盗血链交叉印证。桑德作为魔族传令，出现在馆区即表示债务进入催收阶段——神堕会让他改口：旧神已自己完成「付账仪式」的前置。

**【冻结大地与封盖国的战略含义】**
鲁芬不是普通农耕国，而是千年闸门。克里斯托夫把魔族推入北土并建国为盖，意味着王室合法性来自**守门**，不是来自丰收。神恩反转时，守门职能最先失效：冻土线南移、低温前锋、斥候失踪。若国内还在争论「要不要公布神病」，门外的门轴已经松了。六阶作业的真正时间压力不是神祇台词，而是**封盖失效速度**。

**【条件性胜利的可观察判定表】**
1. 黑金样本铅封编号存在且由弥撒或王室第三方持有。  
2. 供奉草约附则有双份书记记录，或草约被当众否决。  
3. 莉莉娅未在主堂单独接受「容器祝祷」。  
4. 圣骑士团未发生大规模内斗（允许小规模隔离冲突）。  
5. 瓦尔留言原件或等值外压情报进入军务档案。  
6. 迪尔克—雾子债务契至少被听证一次，无法「内部消化」。  
满足四项以上为优；三项为可接受；两项以下为劣稳态。此表应被写进回合结算，而不是只存在于玩家心里。

**【失败态转进】**
若私签附则：王都表面平静，夜仪半公开，容器舆论在修道院发酵，下一切片「神化线」难度上升。  
若滥杀圣卫：姫骑士营道德破产，赛尔夺叙事，莉莉娅被迫在军事与政治间二选一。  
若莉莉娅成容器候选并被公开：人望反噬为「圣洁祭品」叙事，迪尔克可借「保护姐姐」夺权。  
失败不是游戏结束，而是世界进入更糟的可玩稳态。

**【与乐园契约者的交互边界】**
土著默认不知「契约者」体系；对外身份用灾害特使/顾问。六阶战力可碾压普通骑士，但**不可**靠一人斩神通关——顶点规则是情报与条件。向莉莉娅证明可信的方式是物证与誓言，不是展示跨世界神通。向雾子交易时，任何「灵魂」相关报价都应拒绝或改写为情报/通道/物资，以免滑入原作成人债务逻辑。

**【地理补密 · 可走路线】**
王都主堂—圣骸库竖井—姫骑士营—议政厅构成内城三角；修道院侧殿在东翼高地；黑乐园馆在下城烟花带边缘但本切片只作门牌与账本；克洛维斯峡谷在东南三日军程；冻结大地入口在正北五日，冬季更短。走私多用峡谷旧矿道，神殿改革派夜仪也常用矿道声场——记谱时注意回声造成的假峰。

## 阶位切入点

> 阶位↔原作：民兵≈一阶；正式骑士≈二阶；圣骑士≈三阶；王族精锐/白之姫骑士常态≈四阶；圣女·神使·城级信仰崩≈五阶；**神恩全面反转·堕落神祇现形·国家姿态对决≈六阶**。本世界仅覆盖：**六阶**。世界顶点（堕落神祇完全体 / 冻结大地魔王级）为跨阶存在，**超阶巅峰：存在·情报优先/条件性胜利**，严禁硬刚本体或战力归零式削弱。

**六阶（地区灾难 · 神恩反转与国家姿态）**

切入身份/时点：契约者以乐园编制外「神权灾害特使 / 王室军事顾问」切入，锚定 **T0 祝祷失灵夜之后、T+24h 神像裂痕已现、议政厅三派尚未表决** 的白热化前夜。本地需核验编制外徽章；无徽章先被当间谍拷问三轮，拷问题库即本切片物证名录。

初始事件：王城议政厅吵爆。**玛丽亚**把染血布条拍上桌；**弥撒**当众要求双盲封样；修道院急报纯信者昏倒反复念「银甲」；边境急件附**瓦尔**留言抄件。**莉莉娅**带甲闯入，圣剑已微烫。你必须在三派拍桌时表态：先查神，还是先瞒民。

开场白建议：「蜡烛仍白，光已脏。银甲映出神像右眼裂痕，圣歌多出不该有的音节。边境的圣水先烧黑烟，再去愈合敌人的伤口。公布或瞒，先决定——这神还配不配被信。你的徽章在门廊被查了三遍；下一遍，他们会查你的立场。」

关键NPC立场：**莉莉娅·埃贝尔瓦因**拒作容器，先试你的誓言与物证能力；**迪尔克·埃贝尔瓦因**想借你压姐姐或压神殿，给甜头也埋钩；**雾子**卖情报与「替代秩序」，价码是通道与默许；**赛尔**死守「考验」叙事，怕公开；**加雷斯**要可砍的敌人，给你刀也要你给认证程序；**约书**真心欢迎「更诚实的神」，审讯他靠逻辑而非刑求；**赫尔曼**读附则，可被买通；**玛丽亚/弥撒**是物证盟友；堕落神祇只给条件不给斩杀。

主线钩子/支线：主线＝**确认神堕并锁定国家姿态**（封印时间 / 拒私签 / 防圣骑士团内战）。支线＝①弥撒双盲化验黑金并铅封编号；②玛丽亚盗血链第二环；③当众朗读供奉附则并双份存档；④回收瓦尔留言原件；⑤圣剑烫手校准写入军务日志；⑥老伊听音孔与钥匙交换；⑦防止迪尔克—雾子债务契被单独「内部消化」。

危险度/规避：极高 / 贴近顶点。必须规避：正面对轰完全体神祇；让纯信者单人祈祷；签含附则的供奉草约；无认证滥杀流泪圣卫；把莉莉娅单独送去主堂「对质」。失败死亡率：圣骸库强突无认证≈团灭级；公开失败引发挤兑与逃兵后，萨克森渗透成功率显著上升。

任务方向/奖励：自然目标是物证链闭环 + 莉莉娅未容器 + 王都未崩。奖励取向：黑金鉴定权、神殿临时接管、六阶神权危机档案、圣剑共鸣校准、克洛维斯战史调阅、编制外顾问转正令（本地）。不发「弑神成功」或「神已净化」。

【六阶执行细则·身份核验】入场须乐园任务印 + 本地编制外徽章。答错物证名录＝敌对标记，姫骑士营会先缴械再问话。

【六阶执行细则·物证优先】每幕至少一件可验证物证。纯砍杀不推进世界状态。样本要编号，留言要原件，附则要双份。

【六阶执行细则·顶点交互】与堕落神祇对话默认记录原话、条件、索取清单；不默认斩杀判定。可用「暂时封印」换时间，不可写其战力归零。

【六阶执行细则·队友红线】纯信者两人制监护；给加雷斯非人认证程序；不把玛丽亚单独夜间派去主堂。

【六阶执行细则·失败态】主线失败则世界转入更糟稳态（夜仪半公开、容器舆论成型），仍可玩但奖励降档，并解锁更难的「神化线」前置。

【六阶支线池】黑金双盲；盗血第二环；附则朗读；瓦尔原件；圣剑校准；债务契听证；萨克森假白骑士旗事件；冻结大地冻土线测绘。

【六阶时序锚】T0 祝祷失灵；T+24h 裂痕与血证；T+48h 圣骸库与神谕；T+72h 神现形与姿态表决。

【六阶结算关键词】回合总结须出现：裂痕、血证、容器、供奉、附则、认证（至少四项）。

【六阶战场节奏】开场三十分钟内卷入议政厅或主堂现场；中段用物证推进；末段给条件性胜利判定而非斩首秀。

【六阶禁令】禁止战力归零削弱顶点；禁止跨世界套话任务；禁止无名群像替代具名 NPC；禁止色情细节。
