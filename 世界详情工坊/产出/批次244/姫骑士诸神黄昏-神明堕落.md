# 姫骑士诸神黄昏-神明堕落

## 剧情

**【作品来源】**
《姫骑士莉莉亚》（姫騎士リリア）系奇幻陷落题材衍生；乐园「诸神黄昏」为六阶神权崩坏切片组。本条目**神明堕落**锚定：守护王国的**神明被侵蚀**，圣洁转向堕落，信仰与神器开始变质。人物锚：**莉莉亚**（姫骑士/公主）、妹妹**莉雪尔**、侍女**玛丽亚**、圣骑士团、神殿、堕落神祇。文风：神权政治+信仰崩坏+骑士伦理。不写色情细节。

**【世界定位】**
加冕礼的钟还在响，神像的眼睛已经换了颜色。一句话：祈祷越响，深渊越近。

**【世界观 · 力量体系】**
世界基本规则：中世纪奇幻王国以「圣光信仰」维系王权与骑士团；神殿、圣骑士、王族血脉构成圣洁秩序。诸神黄昏线中，守护国神被未知侵蚀，圣洁权柄反转为堕落神力：信仰、神器、圣女、公主皆可成为容器。死亡：凡人一击即终；圣骑士可凭圣痕续战；神格容器可「神化再生」但自我消融。成长：骑士修炼→圣光加持→神器共鸣；堕落侧：侵蚀→容器→神化→新神权柄。

战力阶梯：
1. **民兵·侍从**：冷兵器格斗。≈一阶。
2. **正式骑士·低阶魔族**：小队战。≈二阶。
3. **圣骑士·中阶魔将**：端城寨。≈三阶。
4. **王族骑士·高阶魔王代理**：掀平建筑群。≈四阶。
5. **圣女团长·堕落神使**：一城信仰崩盘。≈五阶。
6. **堕落神祇本体·神化容器·弑神级神器**：地区级灾难，军队无效。≈**六阶（本档）**。

乐园阶位映射：民兵≈一阶；骑士≈二阶；圣骑士≈三阶；王族精锐≈四阶；圣女/神使≈五阶；**堕落神祇/神化莉莉亚/弑神战≈六阶**。顶点（堕落神祇完全体）情报优先/条件性胜利。

**【地理 · 舞台】**
- **王都圣殿主堂**：神像开裂、圣光变浊的第一现场。
- **姫骑士营**：莉莉亚驻防与操练场。
- **神殿地下圣骸库**：侵蚀源头疑点。
- **边境要塞**：魔族试探性进攻，检验神恩是否还在。
- **莉雪尔修道院侧殿**：妹妹信仰最纯处，也最易碎。
- **王城议政厅**：要不要公布「神已病」。

**【世界剧情线】**
① **神恩失灵**  
边境祝祷失败，圣水烧出黑烟。大主教宣称「考验」；莉莉亚在战场看见敌魔族反而被「新神恩」治愈——秩序反转。

② **神像裂痕**  
主堂神像右眼渗金黑液体。玛丽亚发现夜间有人用王族血涂缝。嫌疑指向神殿内部改革派。

③ **莉莉亚调查**  
以姫骑士身份带队入圣骸库，遭遇「圣洁」攻击——神殿守卫已被侵蚀却仍唱圣歌。莉雪尔哭着说听见神要姐姐「成为容器」。

④ **公开与否**  
国王要瞒；骑士团要战；神殿要献祭「净化」。公布=信仰崩盘内乱；不公布=侵蚀加速。

⑤ **堕落神祇现形**  
半影中神自称「未死，只是醒了另一种欲」。要求王国以仪式供奉，换边境和平。莉莉亚拒，神笑：「那就从你的剑开始。」

⑥ **终局分叉（本切片）**  
A. 暂时封印裂痕，神仍病，赢时间衔接到神化线。  
B. 接受有限供奉，和平换堕落合法化。  
C. 强攻神殿，圣骑士团分裂，王都内乱。  
本切片不强制完成弑神，重点是**确认神已堕并选择国家姿态**。

**【主要人物】**
- **莉莉亚（姫骑士·公主）**｜性格：刚直、重誓、不善宫斗。｜装备·能力：王族圣剑、六阶边缘的神恩/反恩抗性。｜弧光：信神→疑神→拒容器。｜立场：主角锚，对契约者先试誓言。
- **莉雪尔（妹妹）**｜性格：纯、敏感、易被神语引诱。｜装备·能力：祈祷共鸣强。｜弧光：听见选容器神谕。｜立场：保护对象/风险源。
- **玛丽亚（侍女）**｜性格：细、忠、敢查血案。｜装备·能力：宫廷门路。｜弧光：发现涂血。｜立场：情报。
- **堕落神祇（原守护神）**｜性格：优雅的病。｜装备·能力：信仰改写、神恩反转。｜弧光：裂痕→现形→索容器。｜立场：顶点。
- **大主教·赛尔**｜性格：要维持表象。｜装备·能力：神殿军。｜弧光：考验说→可能献祭说。｜立场：政治敌友。
- **圣骑士团长·加雷斯**｜性格：只要可砍的敌人。｜装备·能力：团战。｜弧光：发现敌人在圣歌里。｜立场：武力。
- **国王（莉莉亚之父）**｜性格：怕内乱。｜装备·能力：王权。｜弧光：瞒报。｜立场：压力。
- **神殿改革派祭司·约书**｜性格：真心信「新神更诚实」。｜装备·能力：仪式。｜弧光：涂血嫌疑。｜立场：可辩可抓。
- **边境魔将·瓦尔**｜性格：试探。｜装备·能力：魔军。｜弧光：吃到反转神恩。｜立场：外压。
- **议长·赫尔曼**｜性格：算票。｜装备·能力：议会。｜弧光：公布表决。｜立场：程序。
- **圣骸库看守·老伊**｜性格：怕。｜装备·能力：钥匙。｜弧光：放行或出卖。｜立场：门。
- **宫廷医师·弥撒**｜性格：化验黑金液。｜装备·能力：鉴定。｜弧光：科学侧证据。｜立场：物证。

**【势力图谱】**
王室；神殿（旧/改革）；圣骑士团；姫骑士营；边境军；魔族试探军；修道院。

**【贵重物品】**
裂痕黑金样本；王族血案布条；圣骸库钥匙；莉莉亚圣剑；神谕梦记（莉雪尔）；公布诏书草稿；有限供奉草约；反转神恩战场报告。

**【隐藏剧情 · 伏笔】**
侵蚀是否由魔族引入还是神自发；容器神谕是否只点名莉莉亚；约书是真信还是被改写。

**【大事记时间线】**
祝祷失灵 → 神像裂 → 血涂缝 → 圣骸库战 → 神现形 → 封印/供奉/内乱。

**【叙事基调 · 雷区】**
神权冷硬与骑士誓；忌把堕落神写成弱智反派；忌一秒净化；忌忽略信仰政治。最早锚点：边境祝祷失败当夜。

**【神明线·神恩反转机制】**
原治疗术对魔族生效、对人类反噬。战地牧师手册要连夜重写。加雷斯不信，直到自己人被「圣光」灼伤。

**【神明线·血涂缝案】**
玛丽亚在缝口发现莉莉亚同族血型。可能是盗血，不是自愿。调查=宫廷地震。

**【神明线·容器神谕】**
莉雪尔梦中反复：「银甲者承神欲」。莉莉亚若远离妹妹，神谕会找下一纯信者。

**【神明线·公布代价】**
公开神病：商队挤兑、边境逃兵、邻国宣战借口。瞒：神殿继续夜间仪式。

**【神明线·条件性胜利】**
占三：① 确认并记录神已堕；② 暂时稳王都秩序；③ 莉莉亚未成容器；④ 黑金样本在王室；⑤ 供奉草约未私签。

**【神明线·名场面】**
神像右眼滴落，圣歌走调成喉音。莉莉亚举剑不砍神像——砍的是冲来的「圣卫」。

**【神明线·圣骸库】**
地下骨堆发光。守卫流泪攻击：「神要洁净你们。」杀他们=杀同僚；不杀=团灭。

**【神明线·有限供奉条款】**
每月仪式、不碰王族、开放边境贸易。条款二「不碰王族」有附则可被解释为「容器除外」。赫尔曼会读附则。

**【神明线·瓦尔的笑】**
魔将撤退时留言：「你们的神比我们诚实。」情报价值高于一场胜仗。

**【神明线·奖励】**
黑金鉴定权、神殿临时接管、六阶神权危机档案、圣剑共鸣校准。禁止「神已完全净化」。

【神明·圣水检测】黑烟分级：灰=污染；黑金=神堕。弥撒要双盲样。

【神明·圣歌音节】多出的音节可记谱。约书的夜仪用同一谱。

【神明·盗血链】血库、侍医、改革派。玛丽亚查到第二环。

【神明·莉雪尔监护】两人制。单人=神谕灌满。

【神明·附则朗读】供奉草约必须当众读附则。赫尔曼可被买通跳过——防。

【神明·边境报告】瓦尔留言原件比斩敌旗重要。

【神明·圣剑校准】反恩环境下剑会烫手。烫=神恩污染指示器。

【神明·国王怯场】他怕邻国。给你的密诏可能与公开诏相反。

【神明·加雷斯怒】他要杀流泪圣卫。你要给他「非人认证」程序免其崩溃。

【神明·赛尔演讲】「考验」词频过高=撒谎指标。

【神明·评价】确认神堕+莉莉亚未容器+样本在手=优。

【神明·角色要点】莉莉亚要誓；莉雪尔要保护；玛丽亚要真相；神要容器。

【神明·夜间戒严】主堂夜禁。破禁者可能是调查员也可能是涂血者。

【神明·修道院墙】莉雪尔侧殿墙后有听音孔。老伊知道。

【神明·战后衔接】本切片胜利=赢到「神化线」开场的时间，不是弑神。
