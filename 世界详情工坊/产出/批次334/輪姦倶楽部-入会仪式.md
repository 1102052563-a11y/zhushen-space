# 輪姦倶楽部-入会仪式

## 剧情

**【作品来源】**
《輪姦倶楽部》（Rinkan Club）为成人 OVA／相关游戏作品，MAL 等库有收录（如 id 相关条目），主题是封闭「俱乐部」式的集体关系与入会仪式。清单后缀「入会仪式」点题。正文写社会结构与角色心理，不展开露骨集体描写。角色细节以公开页为准，不详标不详。搜笔趣阁未收录。

**【世界定位】**
表面普通的学园／社区地下俱乐部：入会仪式决定成员等级与「被共享」的规则。一句话：在会员证与眼罩之间，把「自愿」问到发颤。

**【世界观 · 舞台设定】**
活动室、地下仓库、成员联络用的暗号、放学后的空教室。规则是入会、升级、退会不能。无超自然。温度来自：眼罩布料、签字笔、仪式后的沉默归路。

**【地理 · 生活舞台】**
- **入会房间**：仪式。
- **空教室**：日常伪装。
- **归宅路**：后悔与依赖。
- **退会失败的传闻墙**：恐惧。

**【故事主线 · 情感线】**
**入会仪式主线**  
主角或女主被邀请／胁迫接触俱乐部→参观→仪式→成为成员后关系网重组→试图退会或反过来改变规则。情感核：从属感、耻辱与扭曲亲密。公开详细人名有限，结构优先。

**【可攻略角色 / 主要人物】**
新入会者（不详·可攻略）｜仪式焦点。

干部成员（不详）｜规则维护者。

旧成员（不详·可攻略）｜后悔的前辈。

外部知情者（不详）｜是否报警的压力。

**【人际关系网 / 社团势力】**
俱乐部等级制＋外部道德。

**【情感事件 · 名场面】**
1. 邀请函。 2. 眼罩。 3. 签字。 4. 第一次「活动」后的归路。 5. 退会谈判。

**【隐藏剧情 · 真结局 · 伏笔】**
原作尺度极重；档案只保留仪式结构与心理。

**【氛围基调 · 雷区】**
封闭、仪式感、黑暗。忌美化；忌战斗。

## 休闲切入点

> 本世界为休闲／恋爱向档案（成人向原作取其人物与情感舞台侧），无生存压力与战斗力体系。契约者以**日常身份**融入，核心玩法＝relationship 攻略 + 日常事件，而非任务厮杀。

切入身份：转校生／记者实习生／被误发邀请的局外人。

切入时点：下一场入会仪式前 72 小时。

初始处境：拿到一张只有时间地点的卡片。

开场白建议：「卡片上没有署名，只有一个时间与「眼罩自备」。教室后门的划痕像数过的人数。你把卡片夹进书里，书却怎么也合不拢。」

可攻略对象：
- **新入会者**：吃退出的实质帮助。

1. 调查卡片。 2. 跟踪归路。 3. 退会谈判是否介入。
