## 剧情

**【作品来源】**
《エルフの里-侵略者》是轮回乐园休闲库收录的**条目名**（检索键保留原题），**并非**对某一单一商业长篇的逐字改编，也**不**把「侵略」写成真实屠村或无意志征服。气质贴近日系同人「エルフの里／オーク／侵略」题材氛围：公开可溯源可对照 DLsite 同人《エルフの里とオーガ軍》（色点屋，RJ152252）的「里被外族军队压境」构图印象，以及《淫欲のエルフ姫アリア》（スタジオ cute，RJ353001）「护林失败后的关系崩解与重建」类短篇的情感骨架；Steam 上《Fallen Elf Freya》等「战后对话—可拒绝亲密」交互也提供「入侵后如何相处」的日常侧参考。搜笔趣阁（sobqg.com）以关键词「エルフの里」「侵略者」检索**无与本条目标题同名的长篇小说书页**——本条目不以盗版全文为底本，而把「侵略者」锚定为：**误闯里门的外来使者／停战观察员／赔偿谈判见习**，把「侵略」重写为**边界危机后的共居与文化冲突**，核心是名字、语言、餐桌礼仪与可撤回同意，不美化屠杀，不写未成年精灵，不把「种族征服」当通关奖励。整体气质：苔藓、风铃、树液灯、双语告示与未喝完的花茶。媒介印象：同人 CG／短篇情景／轻量 ADV；成人向 18+，正文写关系后果与边界，不写露骨过程细目。

**【世界定位】**
银叶里——被古树环抱的成年精灵聚落，正值「停战观察月」。契约者以**外来使者见习／停战记录员／语言交换志愿者／赔偿谈判旁听**身份入里。一句话：**刀先放下，名字先学会；门可关，出口必须仍在。**

**【世界观 · 舞台设定】**
本世界是**温和奇幻日常**，不是清场闯关。银叶里没有「数值对决主轴」，戏剧来自：语言不通时的误会、餐桌礼仪冲突、谁先学会对方的名字、停战条款是否被口头涂改、以及「外来者」会不会把「好奇」做成「越界」。社会氛围：成年精灵以树冠层与根廊双层生活；里门有双语告示；访客须在冷静期后领取**可自管的通行牌**（牌可还，不可被单方没收）。

软规则（情感侧，非对决）：
1. **全员成年**：精灵与外来者默认生理与社会成年；禁止幼态性化与「幼女精灵」路线。
2. **停战优先于征服叙事**：任何「占领／臣服」台词只能作为被批判的旧习或反派压力，不能成为 HE 条件。
3. **名字权**：被叫全名＝亲密升级；被改成编号／蔑称＝信任危机。
4. **同意可撤回**：共餐、共浴、共宿、触碰都必须可当场中止；中止词入里规。
5. **通行牌自管**：没收通行牌＝重大违规，触发长老听证与可合法离里。
6. **文化不是借口**：不得以「精灵风俗如此」强迫外来者或土著做违背个人底线的事。
7. **记录透明**：停战备忘与访客名册由指定记录者主写；禁止代笔涂改离里权条款。
8. **外部眼睛合法**：商队、邻里旅人、听证旁听都可成为「假封闭」的破口，叙事不得惩罚求助。

世界的温度来自：树液灯在黄昏转暖、风铃被晚风撞响、双语词典被翻到起毛的那一页、有人第一次用你的母语说「请坐」。若出现「旧战场」「断矛」「焦黑树皮」等意象，只作记忆与伤痛装置，**不写**强弱评级、闯关榜、硬核对决主轴。银叶里的「余响」是一种软超自然：当有人用胁迫语气强行抬亲密进度时，近旁树液灯会短暂熄灭——用来警示，不是用来开挂。

**【地理 · 生活舞台】**
- **银叶里门与双语告示廊**：初遇与点名。告示写「可拒绝／可离里／通行牌自管」。告示廊木柱上钉着往年离里申请的空白表。
- **长老会圆桌（根廊）**：停战条款、听证、名字登记。长老希尔维娅主场。圆桌中央常年放一盏小灯，象征「条款比情绪更先说话」。
- **语言交换小屋**：词典、黑板、错别字笑话。莉娅的主场。窗台藤蔓活过来那天常被写成关系进度。
- **共餐桌与露天灶**：分食树果粥、谁先洗碗、谁总把盐当糖。长凳不够坐时要肩并肩。
- **风铃回廊**：夜间散步、未说完的半句话、雨声。告白高发地。风铃绳每月检修一次。
- **药草棚**：艾琳的绷带与草药味。内棚贴「未经允许勿入」。
- **访客宿屋（可上锁、可从内开）**：契约者私人起点；通行牌挂在门内钩上——**钩在内侧**是视觉母题。
- **旧边界石与苔藓坡**：停战月「纪念散步」路线；石上刻着已模糊的双语「停」。
- **里外交易亭**：补给、谣言、外部眼睛；商队向导马可提供对照日常。
- **树冠观星台**：True 线收束地——「明天还教我三个词」。
- **根廊洗衣台与共用水井**：谁占太久、谁帮忙绞干，比情话更能推进关系。
- **远哨休息亭**：能看见里门灯火；适合谈「来不及」的恐惧。

**【故事主线 · 情感线】**

**共通线：停战月的名字与餐桌（相遇→文化摩擦→条款危机→双签）**

第一周·误闯与登记：契约者因乐园偏差或正当信使身份抵达银叶里门。卫士**菲娜**拦下他，不是用刀尖，而是用一块写着「先放下武器登记」的木牌。长老**希尔维娅**要求：三日内学会十个精灵语日常词；通行牌自管；任何身体接触写入《访客行为备忘》。女猎手**赛尔菈**冷眼旁观，认定「外来者＝下一场火」。药草师**艾琳**递来凉茶，只问「有没有擦伤」。语言交换员**莉娅**把词典塞进他手里：「先别谈赔偿，先学会说谢谢。」商队向导**马可**在交易亭远远看了一眼，像在估量「这个新人会不会把里内事当酒话」。

第二至三周·升温：共餐桌成为主舞台。谁把盐当糖、谁拒绝「试吃未知果」、谁在雨中把斗篷让出半边。菲娜开始在交接班时多停三秒；莉娅的黑板出现契约者母语的歪扭拼写；艾琳允许他帮忙捣药但禁止擅自进内棚；赛尔菈仍称呼他「入侵种」，却在他迷路时远远丢来一枚发光苔球引路。共通拼图是三件事：**通行牌是否仍在本人手里**、**中止词是否被全员承认**、**停战条款有没有被商队／鹰派口头涂改**。希尔维娅开始在程序句末尾多问一句「你今天吃饭了吗」——这是她疲惫的泄露，也是破冰口。

第四周·冲突：外里商队或鹰派精灵提出「用高额赔偿换永久驻军观察权／把外来者编成劳役编号」。有人试图没收通行牌「以免逃跑影响谈判」。真相并非猎奇屠村，而是旧停战文书里本来就有**冷静期与离里权**，被口口相传的「外来者没有拒绝权」冲淡。冲突焦点：你会不会用「为了和平」覆盖同意？会不会把文化课变成羞辱课？会不会在听证上替别人代签？

收束·停战备忘双签：修订扉页——**通行牌不得单方没收；名字不得改编号；中止词全员会用；共居条款可随时口头终止并在二十四小时内协助离里**。结局方向取决于契约者是否尊重拒绝权、是否把精灵当战利品收藏、是否把「学会语言」当成控制工具。

**希尔维娅线（长老 · 责任与疲惫）**
把整座里的安全扛在肩上，用程序句掩饰恐惧。攻略关键不是「推翻她」，而是承认其压力，帮她把「安全」从「锁死出口」改回「出口可见」。她会在圆桌散会后独自磨墨，墨香是她的情绪温度计。
- HE：她在圆桌当众把备用通行牌规章贴回告示廊，说「和平不是把人关在里面」。
- Bittersweet：她卸任观察长，仍每周寄来新词表。
- BE：你当众夺权羞辱／偷改条款 → 她终止访客资格，只留一封程序化驱逐函。

**菲娜线（门卫 · 傲娇守门人）**
肩甲擦得发亮，嘴硬心软。怕的不是外来者本身，而是「再一次来不及开门放人」。交接本是她的日记替代品。
- HE：她把你的名字写进门岗交接本「允许夜归」栏；交接时笑骂「别再迷路」。
- BE：你强闯里门／藏她的哨笛 → 她申请调去远哨，不再多停三秒。

**赛尔菈线（猎手 · 创伤与偏见）**
旧边界火灾的幸存者视角。称呼「入侵种」是铠甲。攻略关键是不急着纠正用词，先在行动上证明「我不会放火」。苔球是她给信任的人的路标，不是武器。
- HE：她第一次叫你的本名，并把发光苔球正式交给你「自己照路」。
- BE：你嘲笑其创伤／把旧边界当约会景点炫耀 → 她切断同行，边界石再不见其身影。

**艾琳线（药草师 · 照料边界）**
温柔到容易自我耗尽。攻略关键是让她也学会说「今天不接待」。药杵的节奏乱了，说明她在勉强。
- HE：共同修订药棚值班表；她把「可中止照料」写进备忘。
- BE：你把她的温柔当无限额度 → 她请长假，药棚只留干燥花束。

**莉娅线（语言交换 · 桥）**
元气、话多、词典永远缺页。把「学会对方的语言」当成最高浪漫。公开词表末页空白只留给双方同意的昵称。
- HE：观星台互教三个新词；她把你的母语昵称写进公开词表（经你同意）。
- BE：你嘲笑其口音／把双语课变成羞辱 → 她收回词典，只做书面翻译不再对视。

**马可线（人类商队向导 · 外部对照，可淡攻略）**
碎嘴、现实、带外来补给。提供「里外都是人」的对照。
- HE：交易亭长期合作；他不再用玩笑盖住「你有没有把通行牌给别人」。
- BE：你拿里内隐私当谈资 → 他改派别人送货。

**阶段节奏（情感而非闯关）**
- 第 1 周：登记、十词、通行牌内侧挂钩；第一次共餐桌。
- 第 2 周：盐糖事故与道歉笑；菲娜多停三秒；艾琳允许捣药。
- 第 3 周：赛尔菈丢苔球；条款流言；马可认真问牌。
- 第 4 周：没收牌危机；旧文书真相；听证。
- 收束月：备忘双签；各线 HE／BE／Bittersweet。

微观日常事件池（可重复推进关系）：词典缺页、风铃绳松、盐糖拿错、苔球没光、通行牌掉漆、雨中共斗篷、观星台露水、药杵节奏、黑板错字、告示廊新贴、门岗哨笛、词表被灵兽踩脏、水井排队、洗衣绞干、远哨热茶、墨磨太浓、藤蔓发芽、梯子借还、干粮分半、斗篷补丁、双语空白表被雨打湿再重抄、共本词表被猫科灵兽踩脏后的重抄夜谈。

**【可攻略角色 / 主要人物】**

**菲娜（里门卫士 · 交接班负责人）**｜外貌：深绿短发、擦亮肩甲、哨笛、站姿像门栓｜性格：傲娇、守时、嘴上「只是规定」｜角色类型：傲娇 · 守门人｜萌点/魅力：交接本上多写的半行字；夜归时假装不在意的侧目｜个人线剧情：从拦人 → 允许夜归 → 把名字写入交接本｜与主角关系：第一张脸 → 可攻略｜攻略忌：强闯门、藏哨笛、当众拆穿其心软以羞辱。

**赛尔菈（女猎手 · 边界巡行）**｜外貌：亚麻马尾、浅疤在眉骨、皮甲、发光苔球｜性格：刺、直接、创伤铠甲厚｜角色类型：冷娇 · 创伤猎手｜萌点/魅力：骂「入侵种」却丢路标；第一次本名时的卡壳｜个人线剧情：偏见 → 同行 → 交出路标权｜与主角关系：敌视 → 共担巡行班表的恋人｜攻略忌：消费其火灾记忆、逼立刻改口、把旧边界当情趣场景。

**艾琳（药草师）**｜外貌：浅褐软发、围裙、指尖草渍、声音轻｜性格：温柔、自耗、界限模糊｜角色类型：治愈系 · 照料者｜萌点/魅力：先问擦伤再问来意；拒绝时也会把茶放凉到适口｜个人线剧情：无限照料 → 学会中止 → 共同值班表｜与主角关系：照料者 → 可互惠恋人｜攻略忌：予取予求、闯内棚、把其拒绝当「撒娇」。

**莉娅（语言交换员）**｜外貌：金杏短发、墨水指、永远缺页的词典｜性格：元气、话密、怕被笑口音｜角色类型：元气 · 桥｜萌点/魅力：歪扭拼写；互教词时的耳尖红｜个人线剧情：词典 → 公开词表 → 观星台三词｜与主角关系：老师兼同学 → 可甜宠｜攻略忌：嘲口音、偷改词表羞辱、逼公开未同意的昵称。

**马可（人类商队向导 · 交易亭）**｜外貌：风尘斗篷、账本、笑纹｜性格：碎嘴、现实、有底线｜角色类型：日常对照 · 损友外置｜萌点/魅力：多带的干粮；认真问通行牌｜个人线剧情：补给 → 信任 → 长期合作｜与主角关系：外部眼睛／淡攻略｜攻略忌：泄里内隐私、让他背锅、把交易亭当压迫工具。

**契约者视点（外来使者见习／停战记录员 · 玩家）**｜姓名外貌自定；默认愿意学语言、敢在备忘上写「可离里」｜成长：从「证明我无害」到「承认对方可以不需要我」｜立场：名字先于编号；出口先于亲密；条款先于冲动。

**【人际关系网 / 社团势力】**
- **程序派（希尔维娅核心）**：条款、听证、名册完整；口号「印章不可戏」。
- **门岗派（菲娜）**：班表、夜归、哨笛；口号「先登记再说话」。
- **边界派（赛尔菈）**：巡行、旧伤、对外警惕；口号「火不会自己灭」。
- **生活派（艾琳·莉娅）**：药与词、共餐桌；口号「先吃饭再开会」。
- **外部商队（马可）**：补给与谣言；口号「别把里内事当酒话」。
三角张力：菲娜与赛尔菈对「外来者可不可夜归」争执；莉娅公开词表可能激怒程序派；契约者连续三天与同一人共餐桌≈流言站队。希尔维娅必须在「效率」与「尊严」之间公开表态。各派非善恶二分，而是对「如何被记住」的不同答案。

**【情感事件 · 名场面】**
1. **里门木牌**：菲娜递「先放下武器登记」——第一句规矩。
2. **十词测验**：莉娅黑板；契约者母语歪扭回教。
3. **盐糖事故**：共餐桌道歉笑，希尔维娅程序句破功。
4. **发光苔球**：赛尔菈丢路标。
5. **通行牌危机**：有人要没收；你选择还是拒绝。
6. **雨中斗篷**：必须决定先让给谁半边。
7. **观星台三词**：True 向收束——「明天」「一起」「可以吗」。
8. **备忘双签后第一句**：不是征服宣言，而是「你饿不饿」或「叫我的名字」。
9. **药棚拒绝日**：艾琳第一次说「今天不接待」，你选择尊重。
