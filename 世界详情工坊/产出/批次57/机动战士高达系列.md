## 剧情

**【作品来源】**
《机动战士高达系列》（日语：ガンダムシリーズ，英语：Gundam）由矢立肇・富野由悠季原作，日昇动画（Sunrise / 万代南梦宫映像）自1979年电视动画《机动战士高达》起持续展开的跨媒介军事科幻宇宙。媒介含TV、OVA、剧场版、小说、漫画、游戏与高达模型（Gunpla）。主时间轴为宇宙世纪（Universal Century，UC）；另有多条独立纪年平行宇宙（FC/AC/AW/CC/CE/AD/AG/RC/PD/AS等），主题共通而史实不互通。文风总基调：写实机器人、战争群像、无绝对正邪、少年驾驶员被国家与资本征用；反战与「人类是否进化」的哲学并置。**本档案为系列总览**，负责UC主轴串联与主要分支定位；单作细部（如一年战争航迹、SEED三舰同盟战役表）以对应单作档案为准，此处不整段复制。

**【世界定位】**
近未来（或替代史）地球圈—殖民卫星文明：人类把战争搬进轨道与都市，机动战士（Mobile Suit，MS）成为主力兵器，「高达」多为联邦/抵抗方的高性能原型机符号。UC线以阿姆罗・雷与夏亚・阿兹纳布尔的镜像对决为精神中轴；分支线各自重写「地球 vs 宇宙/基因/资本/战争经济」。一句话：契约者切入的是**有补给、有军法、有难民与名牌**的MS战场，不是无敌超级机器人游乐场。

**【世界观 · 力量体系】**
全系列无灵气修真。战力由四层叠加：①人体素质与军训；②机体世代、装甲与武装；③飞行员资质（普通人→王牌→特殊进化人类）；④战略兵器与政治权柄（殖民卫星坠落、GENESIS、GN武装、阿赖耶识系统等）。死亡即死：无复活；精神感应/Innovator/X-Rounder等不能让机体免疫主炮。米诺夫斯基粒子（UC）使雷达失效，近战与光学观测主导——此规则塑造了整个写实机器人类型。

力量阶梯（跨作破坏力对齐，宁低勿高；本库切入仅一、二阶）：
1. **平民・难民・舰内勤务**：徒手/轻武器，坠舱、冲击波、配给断绝可致死。乐园≈一阶初期。
2. **常规士兵・宪兵・舰员**：步枪、对空炮、舰炮位；无MS时被量产机屠杀。≈一阶中后。
3. **量产MS新兵**：扎古II、吉姆、金/银普通机、量产旗鱼等；毁坦克与街区建筑，热武器对座舱仍致命。≈二阶。
4. **熟练驾驶员・小队王牌**：多次出击存活，可左右据点攻防。≈三阶（本库不切入）。
5. **战争中期王牌・新人类/协调者萌芽**：局部战役级；可毁舰桥、撕开防线。≈五阶向。
6. **后期特殊进化驾驶员・巨型MA/战略机**：要塞区段级；新人类对决、GN粒子域、阿赖耶识满负荷等。≈六~七阶向。
7. **战略兵器与政治顶点**：太阳射线、殖民卫星坠落、GENESIS、超大型轨道武器、战争经济中枢；个人肉体非爆星，掌握即文明灾难。≈八阶向。
8. **跨纪年神话层（极少）**：如∀的月光蝶级文明重置——仅对应单作终局讨论，系列总览不鼓励低阶接触。

特殊人类标签（各纪年名称不同，机制近似「感知/反应/共感超常」）：
- **新人类（Newtype，UC）**：空间适应与精神感应；弗拉纳冈机关等将其武器化。
- **强化人（Cyber-Newtype）**：药物与手术强造，精神易崩。
- **协调者（Coordinator，CE）**：基因改造超人；与自然人（Natural）对立。
- **Innovator / 革新者（AD）**：GN粒子长期暴露后的进化形态。
- **X-Rounder（AG）**、**阿赖耶识适格者（PD）**、**Gundam Meister**等：各作战斗资格证，勿混用设定名。

乐园阶位映射（系列总览用）：平民/新兵≈一阶；量产MS新兵≈二阶；熟练MS/小队王牌≈三阶；中期王牌/特殊进化萌芽≈五阶；后期特殊机/巨型兵器≈六~七阶；战略兵器与最高统帅权≈八阶。**本世界库条目仅覆盖一、二阶切入**；更高阶事件作背景气压与情报，禁止一介新兵正面对轰太阳射线/GENESIS/满输出00。世界顶点随切入纪年变化（UC：基连・萨比与战略兵器 / 逆袭后的夏亚与阿姆罗级对决；CE：扎拉派与地球军核武双方；AD：ELS接触与革新者战争等）——低阶一律**规避顶点・情报优先**；**超阶巅峰**写条件性胜利（夺发射权、截获密钥、政治瓦解），严禁「战力归零式削弱」。

**【地理 · 舞台】**
**共同骨架**：地球本体 + 近地轨道殖民卫星/宇宙殖民地 + 月球（部分作）+ 木星能源线（UC等）。写场景先定契约者在「殖民地居住区 / 地球占领或贫民街 / 军舰内部 / 残骸带 / 中立港」哪一层。

**主要分支舞台（点到为止）**：

底层日常：配额、假证件、黑市零件、征兵、难民舱位。中层：MS出击、基地攻防、间谍交换。高层：战略兵器发射权、议和条约、战争经济账本。本地人若听闻码头新兵要单挑「红色彗星」或抢GENESIS控制权，会当疯子——正确反应。

**【世界剧情线】**
以下按「系列史观」写：先UC主轴完整串联，再列主要平行纪年的定位与因果；细部战役见单作档案。

**〇・总前提：人类为何打MS战争**  
殖民卫星使「地球住民 vs 宇宙住民」成为结构性矛盾。资源、投票权、重力井下的歧视与殖民独立思想反复点燃战争。高达系列的残酷在于：双方士兵多是普通人；「进化人类」被当成兵器；胜利方往往立刻腐败，为下一场战争施肥。

**①・宇宙世纪前史与一年战争（UC 0068—0080）**  
吉翁・兹姆・戴昆倡导殖民地自治与新人类理念；其死后萨比家夺权建立吉翁公国。UC 0079.1.3一年战争爆发：一周战争屠杀、英帕尔斯殖民卫星坠落重创澳洲、鲁姆战役与南极条约、吉翁地球降下占领约三分之二地表。联邦V作战在Side 7诞生RX-78-2；少年阿姆罗・雷登机，白色基地在布莱德・诺亚指挥下逃亡作战。卡尔马・萨比战死、兰巴・拉尔队覆灭、奥德萨反攻、所罗门陷落（德兹尔与大扎姆）、阿・巴瓦・空终战：基连发射太阳射线蒸发己方与敌方舰队一部并弑父，基西莉亚杀基连后死于夏亚，阿姆罗与夏亚两败俱伤。UC 0080吉翁败北。**名场面锚**：白色恶魔、红色彗星、太阳射线漂白宇宙。主题：战争吞噬少年；敌人有脸。单作细部→《机动战士高达 0079》等档案。

**②・战后余烬与星尘记忆（UC 0083）**  
迪拉兹舰队等吉翁残党发动星尘作战，再次威胁殖民卫星坠落。联邦内部强硬派借机推动「泰坦斯」思想温床。世界状态：胜利者清洗、残党复仇、舆论被操纵。契约者可介入护航、假投降、残骸带打捞——仍属低中阶战场。

**③・格里普斯战役（UC 0087，《Z高达》）**  
泰坦斯以「维护秩序」对殖民地施暴；反地球联邦组织AEUG与卡拉巴武装起义。卡缪・维丹驾驶Z高达；夏亚化名库瓦特罗・巴吉纳加入AEUG。帕普提马斯・西罗克、哈曼・卡恩（阿克西斯）多方混战。格里普斯2殖民卫星被改造成武器。基调比初代更黑：友军内斗、强化人悲剧、政治屠杀。阿姆罗在此时期受监视与边缘化，政治气压高于个人英雄。阶段结局：泰坦斯崩解，阿克西斯/新吉翁势力坐大，为ZZ与逆袭铺路。

**④・第一次新吉翁抗争（UC 0088，《ZZ高达》）**  
哈曼・卡恩主导新吉翁；主角转为裘多・阿西塔与尚・波岛少年群。达卡尔演说后的权力真空、殖民地被当成人质、高达队在废墟与黑市间游击。基调混杂荒诞与残酷：战争日常化到「孩子开MS抢食物」。哈曼路线失败后，吉翁系意识形态转入夏亚的下一手。

**⑤・逆袭的夏亚（UC 0093）**  
夏亚・阿兹纳布尔公开以新吉翁领袖身份回归，推动「让地球寒冷化、强迫人类进入宇宙」的阿克西斯坠落计划。阿姆罗・雷率隆德・贝尔队驾驶ν高达迎击。终局阿克西斯冲击：两人精神与机体对撞，传统史观中双双下落不明（后续作以「传说」处理）。主题：理想主义走到极端是否等同暴政；宿敌最后一次「理解」。

**⑥・拉普拉斯之盒与独角兽（UC 0096—0097）**  
《机动战士高达UC》：巴纳吉・林克斯与独角兽高达；「拉普拉斯之盒」指向宇宙世纪宪章中被抹去的殖民地自治承诺。フル・フロンタル（满腹・弗龙塔尔）作为「夏亚的再来」统率袖章队。联邦腐败、毕斯特财团、新人类可能性再次被政治化。续作《NT》在0097年处理独角兽余波与Phenex。世界状态：CCA三年后，传说未死，制度更脏。

**⑦・哈萨维与后期UC（UC 0105起）**  
布莱德之子哈萨维・诺亚走向反地球联邦恐怖/革命光谱（《闪光的哈萨维》）。更后期《F91》（0123）、《V高达》（0153）等描写联邦衰落后的小国混战与新世代MS——战争形态从「两大阵营」碎成「地方军阀+新型机」。系列总览只需记住：**一年战争是原罪，越往后个人越难撼动结构，越容易被结构碾碎**。

**⑨・跨作母题链（AI写任何切入点都要守）**  
1. MS是兵器不是圣剑：弹尽、关节损坏、推进剂、座舱击破。  
2. 少年兵与征用：国家用「保护家园」话术合法化。  
3. 进化人类的政治化：新人类/协调者/Innovator先被捧成希望，再被做成兵器或清洗对象。  
4. 胜利者腐败：联邦、地球联合、地球圈联邦皆然。  
5. 殖民卫星=可砸的武器与可屠的城市。  
6. 红色与白色的镜像（UC）：夏亚与阿姆罗定义了「理解敌人却仍扣扳机」。

**⑬・一年战争因果链（总览压缩・供任意时点判定）**  
开战首周：毒气与MS奇袭清空多座Side，人口与舰队腰斩级损失；英帕尔斯未砸中贾布罗却抹平悉尼级都市，使「殖民地=质量兵器」成为全UC创伤后应激。鲁姆战役证明MS可屠舰，南极条约禁用核生化与再坠落，反而把战争锁进MS近战——吉翁的世代优势被延长，联邦被迫用V作战赌原型机。地球降下后占领约三分之二地表：配给、宪兵、通敌审判与反抗军并存。Side 7之后，白色基地的航迹是「联邦唯一活跃的V兵器投放点」被全军通缉的逃亡史：北美追杀、卡尔马之死点燃基连演说、兰巴・拉尔以军人荣誉猎白鲸而死、奥德萨常规军反攻扭转地球资源线、贾布罗量产吉姆、再入宇宙破所罗门（大扎姆）、阿・巴瓦・空与太阳射线。终战不是浪漫决斗收场：萨比家自噬、要塞投降广播与残骸带并存，联邦胜利证书上写着下一场清洗的墨水。

**⑭・Z—ZZ—CCA 政治递进**  
泰坦斯把「秩序」做成对殖民地的公开暴行，AEUG的合法性来自受害者叙事而非血统。卡缪的愤怒是私仇与公义缠在一起；西罗克把新人类论扭成支配神学；哈曼以阿克西斯守护者自居，实则把少年与殖民地当燃料。夏亚在Z以库瓦特罗身份站在「反暴政」一侧，到0093却选择用坠落强迫移民——同一人完成从革命者到极端理想主义者的滑落。阿姆罗在逆袭中代表「仍要拦下坠落的人」，两人最后一次对撞是理念而非单纯击坠数。总览写作时：Z期气压=内斗与清洗；ZZ期气压=战争日常化到孩子开MS；CCA气压=宿敌终局与地球存亡。

**⑮・CE 与 AD 的结构对照（防串设定）**  
CE用基因把人类劈成协调者/自然人：血色情人节核弹与中子干扰器让地球陷入能源与通讯双崩；基拉在赫利奥波利斯被迫驾驶强袭，大天使逃往阿拉斯加途中与阿斯兰多次对决；拉克丝盗出自由、三舰同盟在双方超武（核与GENESIS）对射前卡死扳机。DESTINY让真・飞鸟走仇恨线再醒悟，证明停战条约挡不住下一轮动员。AD用轨道电梯与化石能源枯竭重画地图：天人四大师以高达强制介入局部战争，国家军用伪GN与人海反扑；第二季地球圈联邦的A-Laws把「统一」做成新压迫，刹那经双驱动与Innovator之路走到与ELS的非人类对话。写CE禁米诺夫斯基粒子叙事；写AD禁新人类测定器——用该作自己的粒子与制度名。

**⑯・铁血・魔女・∀ 的低阶可玩面**  
PD的契约感最强：佣兵合同、童工兵、地球圈贵族的保险与赔偿条款写得比理想清晰；三日月的枪口对准「妨碍团员活下去的人」。AS的战场常伪装成学院竞赛与企业并购，GUND格式的人体代价被公关稿盖住；斯莱塔的「听从指示」是武器说明书语气。∀的正历地表是废土领主政治，月之民的科技优势像天降；月光蝶是终局级文明抹除，一/二阶只闻其名、只做遗迹边缘与领主税契。三者都适合低阶：合同纠纷、学院考试作弊被军需眼线盯上、废土商队护送——不必立刻开战略终局。

**【叙事基调 · 雷区】**
基调：金属与泥、臭氧与呕吐物、家书与名牌；可热血但必须付账。忌：①超级机器人式无油无弹无敌；②把新人类写成修真神通无限复活；③吉翁全体狂信徒或联邦全体正义；④串宇宙设定（扎古打GN、协调者解释新人类）；⑤一/二阶硬刚战略兵器与超阶顶点；⑥后贴元写作段或编号灌水段。最早切入：任一纪年的殖民地日常、新兵舱、避难闸门、量产机第一次出击前夜。

## 阶位切入点

> 阶位↔战力：一阶≈平民/新兵/舰内勤务/难民；二阶≈量产MS新兵。熟练王牌以上、新人类顶格对决、战略兵器与政治顶点均为背景气压。世界顶点随纪年变化（如UC基连・太阳射线 / 逆袭夏亚与阿姆罗 / CE超武双方 / AD革新者战争），低阶切入一律「规避顶点・情报优先」；条件性胜利，严禁战力归零式削弱。本世界仅覆盖：一阶、二阶。

**一阶（平民・新兵・难民 · 名牌与闸门）**

切入身份/时点：契约者以殖民卫星学生、码头装卸工、舰内勤务、联邦/吉翁（或其它纪年等价）新兵入伍生身份切入；推荐锚点——UC Side 7遇袭前夜、一年战争地球占领区配给点、CE血色情人节后的中立港、AD轨道电梯下层贫民街、PD火星佣兵招募站外围（勿直接阿赖耶识手术台）。

初始事件：警报或征兵广播撕裂日常。扎古/量产机的震动传来，避难闸门只开一半；你必须在三十秒内选择——挤进闸门、捡起倒下宪兵的射频袖章冒充士兵、或跟着芙劳・波一类带孩子的平民往撤离艇跑。不选也会被冲击波选。若在CE/AD等纪年，将「扎古」替换为该作量产机与宪兵体系，冲突结构不变：配额、假证件、谁先关门。

开场白建议：「避难广播沙哑循环，传感器在粒子干扰里变成雪花。窗外量产机的脚掌碾过你邻居的姓氏门牌。有人把沾血的袖章塞进你手里：戴上，你是士兵；不戴，你是统计数字。远处轨道上，有人正把殖民地的灯光当成靶纸。」

关键NPC立场：**阿姆罗・雷**（若UC开篇：慌乱要手册不要英雄叙事，防你也用你）；**芙劳・波**（拉人避难，问失散孩子）；**布莱德・诺亚**（上舰则登你为消耗品编制）；**夏亚・阿兹纳布尔**（侦察中对平民无话，目击其假名活动则灭口）；**基拉・大和**（CE中立殖民地崩溃时护友优先，厌战但会拦你乱开枪）；**奥尔加・伊兹卡**（PD招募外围：给面包换卖命，契约写得极清）；当地宪兵士官（要证件与配额）。

主线钩子/支线：钩住「是否登上白色基地/大天使/天人相关撤离线/铁华团外围运输」的分支；支线——寻找失散亲属名牌、倒卖净化滤芯与假军籍、为任一阵营送一次假情报（被抓即军法）。勿碰太阳射线/GENESIS/GN真驱动核心。

危险度/规避：中。死于踩踏、流弹、征用与通敌嫌疑。须规避任何MS正面对射、萨比家/扎拉派/A-Laws高层出行路线、阿赖耶识强制征召。

任务方向/奖励：活过首日、取得避难身份或新兵番号；奖励底层证件、零件边角、人脉与「还活着的名字」，无光束步枪整机。

**二阶（量产MS新兵 · 第一次出击）**

切入身份/时点：联邦吉姆训练生、吉翁扎古新兵、ZAFT/地球军量产机学员、联合军普通驾驶员、铁华团外围雇佣机手等；锚定UC白色基地地球逃亡期或奥德萨战前、CE大天使追击战外围、AD私兵与国家军第一次大规模交火外围、PD浊流战场边缘——皆为「量产机当耗材」时段。

初始事件：模拟器里循环播放「红色机体/自由型高机动/00粒子雾」的击破录像；实战第一出击，友机被切开，通讯只剩杂音。你要在三选中落地：救友机、保自己撤退、或执行「记录特殊机体数据」的上峰密令。密令报酬高，存活率写在附录小字里。

开场白建议：「座舱是汗和臭氧。瞄准镜里一抹异常涂装一闪即逝——军中传说里那台不该由你这种编号去碰的机。手指在扳机上发抖：射击，你可能成为击破统计的小数；不射击，你连编号都留不下。补给官在频道里只重复一句：推进剂剩百分之四十。」

关键NPC立场：**凯・西汀**（嘲你怕死，扔来扳手）；**小林隼人**（拉你编队）；**兰巴・拉尔**（若属吉翁：考军礼与纪律，可敬亦可令你送死）；**阿斯兰・萨拉**（CE：军规与友情撕裂，视你为可争取的同僚或炮灰）；**洛克昂・斯特拉托斯**（AD：要你提供弹道数据，不保证救你）；**三日月・奥格斯**（PD：无废话，完成护送或滚）；**玛・库贝**类资源军官（用指标压你）。

主线钩子/支线：量产机编队遭遇「白色/自由/天人」特殊机时的存活与报告战；支线——抢修腿部驱动、战俘交换当翻译、中立港走私冷却剂、伪造出击记录吃空饷（东窗事发=军法）。

危险度/规避：中高。死于王牌碾压、友军炮火误射、弹尽。须规避兰巴队全力时段、奥德萨炮兵覆盖中心、GENESIS/太阳射线试射窗口、阿赖耶识过载机体近距离暴走。

任务方向/奖励：三次出击存活或完成一次护航；奖励中级零件、军阶点、被某具名军官点名一次（人脉）、自定义涂装许可（无换代机体）。
