# 赛博PK
<!--meta lib=主库 tiers=二-->

## 剧情

**【作品来源】**
《赛博PK》为轮回乐园世界库**原创任务世界**（非单一商业 IP 移植），定位于近未来义体格斗产业与地下博彩共生的「合法／非法擂台生态」。公开锚点文案：「擂台上，义体拳手开启了过载模式，一拳轰碎了对手的钛合金下颚，冷却液喷洒全场。」文风：赛博朋克体育暴力、合同与器官分期、直播审判。本档案据此扩展可运行的舞台、人物与产业规则，供契约者二阶切入。

**【世界定位】**
都市名**铬湾市（Chrome Bay）**：上层是持牌「联赛格斗联盟（CGL）」的霓虹场馆，下层是废弃地铁改成的**锈圈擂台**。选手靠义体过载、神经加速与赞助合同换取晋级，战败常等于器官回收。一句话：这里的 PK 不是游戏术语，是签了生死免责的职业——契约者是突然空降的「外卡拳手／经纪人／修械师」。

**【世界观 · 力量体系】**
1. **义体等级**：民用／竞技许可／军用走私；许可芯片决定能否上台。
2. **过载（Overclock）**：短时破限，代价是神经灼伤与冷却液沸腾。
3. **神经加速与预判插件**：违规则禁赛，地下赛鼓励。
4. **裁判 AI「公正」**：表面公正，实际可被博彩公司买通补丁。
5. **死亡**：台上可「技术性死亡」后义体重启；真死发生在合同漏洞与回收车里。

战力（本库**仅二阶**）：
- 未改造市民 ≈一阶环境
- 持许可竞技拳手／锈圈常客／联盟执法 orthotics ≈**二阶**
- 军用义体冠军、城市安保机械化部队 ＞二阶，作压迫景，不开放更高切入

乐园阶位映射：铬湾常规 PK 生态≈**二阶**（碎砖裂颚、掀翻装甲护栏、扛轻武器）。军用冠军／安保机甲＝规避对象。

**【地理 · 舞台】**
- **CGL 主场馆「白热穹顶」**：直播、赞助商、贵宾盒。
- **锈圈（The Rust Ring）**：地下淘汰赛，无医护。
- **义体分期商场「关节仓」**：贷款与回收。
- **冷却液河**：工业区排污，常漂浮断肢义体。
- **博彩中枢「赔率塔」**：数据与打手。
- **诊所「回滚」**：改芯片、洗战绩、假死证明。

**【世界剧情线】**
① **产业兴起**  
战后劳动力过剩，政府把义体格斗合法化征税；黑市同步做无许可死斗。英雄叙事掩盖器官链。

② **外卡与合同**  
新人签「三败回收」条款。经纪人**薇拉·科恩**专吃外卡。修械师**老郑**能让报废膝再生一晚。

③ **联赛丑闻**  
冠军**雷克斯「钛颚」奥兰多**被曝使用军用过载芯；联盟主席**霍尔登**甩锅地下赛。直播中途裁判 AI 死机，观众以为是赛制。

④ **锈圈起义**  
地下拳手**妮可·「断线」帕克斯**拒绝回收，打穿联盟执法队，把一场死斗变成劳工直播。博彩公司要她的头与她的流量。

⑤ **契约者窗口**  
二阶任务期通常落在「外卡选拔周」或「钛颚丑闻爆发夜」：你要么上台，要么修械，要么运一箱未登记冷却液芯——每条路都通向同一张合同网。

⑥ **可能结局方向**  
A 联赛改革（表面）；B 锈圈被清剿；C 军用芯流入街头；D 你带着未改的战斗日志离开，成为唯一真实录像。无唯一真结局，产业会自我修复。

**【势力图谱】**
CGL 联盟；锈圈松散同盟；赔率塔博彩；关节仓金融；都市安保；军用芯走私线；药企赞助商。

**【贵重物品】**
许可芯片；军用过载芯；未篡改战斗日志；冷却液配方；裁判 AI 根密钥；冠军腰带（含定位追踪）；三败回收合同原件。

**【隐藏剧情 · 伏笔】**
联赛本是退役军人消化计划；「公正」最初为防止死亡而写，后被资本改写；妮可的神经索原型来自军方项目。

**【大事记时间线】**
合法化格斗 → 锈圈并行 → 钛颚五连冠 → 军用芯曝光 → 选拔周／丑闻夜（锚点） → 清剿或改革分叉。

**【叙事基调 · 雷区】**
冷却液、闪光灯、合同字体。忌修仙化义体；忌无合同上台；忌把二阶写成毁城；忌美化器官回收。锚点：外卡选拔周。
