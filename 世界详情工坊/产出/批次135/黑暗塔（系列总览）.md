# 黑暗塔（系列总览）
<!--meta lib=主库 tiers=一、二、三、六、七-->

## 剧情

**【作品来源】**
《黑暗塔（系列总览）》（The Dark Tower）为斯蒂芬·金（Stephen King）黑暗奇幻·西部·科幻长篇系列，含正传七卷＋《穿过钥匙孔的风》等，1982–2012。灵感来自白朗宁诗《公子罗兰来寻黑塔》、魔戒、亚瑟王与镖客电影。本档案为**系列总览**，分卷细节见各分册条目。文风：命运（Ka）、枪侠、多世界、塔与玫瑰。卡片锚点：**荒漠尽头塔影——罗兰·德斯钦追黑衣人并重抽卡泰特**。

**【世界定位】**
中世界（Mid-World）「已经前移」：封建枪侠文明残存，旧世界科技与魔法并存，时间乱流。最后的枪侠**罗兰·德斯钦**追寻万物枢纽**黑暗塔**，沿途抽来**艾迪·迪恩**、**苏珊娜·迪恩**、**杰克·钱伯斯**与学舌兽**奥伊**，对抗**兰道尔·佛拉格／黑衣人**与**血腥之王**。一句话：为塔可弃一切——契约者是卡泰特边缘的变数，不是塔的主人。

**【世界观 · 力量体系】**
1. **枪侠之道**：训练、意志、 deb  deb  deb  deb  deb——枪是手的延伸。
2. **Ka 与卡泰特**：命运编组。
3. **光束与门户**：六光束撑塔，十二门户与守护者（含赛博格替代）。
4. **钥匙、玫瑰、黑十三**：跨世界与诱惑。
5. **旧世界科技**：正电子学公司残骸、单轨 AI、毒气。
6. **死亡**：血肉可死；塔顶循环可重置罗兰记忆——非免费复活。

战力阶梯：
1. 中世界平民／纽约普通人 ≈一阶
2. 强盗、低级突变、警察 ≈二阶
3. 学徒枪侠、吸血鬼猎人卡拉汉级 ≈三阶
4. （四、五本库总览不覆盖为切入主档）
5. 光束级灾、狼群战争、破碎者工营 ≈六阶压迫
6. 塔周、血腥之王层面 ≈七阶压迫
顶点血腥之王／塔本身＝超阶条件性

乐园阶位映射：一阶≈凡人；二阶≈武装威胁；三阶≈枪侠学徒成长期；六阶≈卡拉／破碎者战争级；七阶≈近塔终局。覆盖一、二、三、六、七。

**【地理 · 舞台】**
荒漠、塔尔城、西海、门户之门、光束之径、卢德、托皮卡、卡拉村、纽约多年代、霹雳地、玫瑰田、黑暗塔。

**【世界剧情线】**
① **枪侠前史**：罗兰早考、梅吉斯苏珊之死、基列陷落、耶利哥丘。  
② **卷一枪侠**：追黑衣人，弃杰克。  
③ **卷二三张牌**：抽艾迪、苏珊娜，救杰克悖论。  
④ **卷三荒原**：沙迪克、光束、卢德、布莱恩。  
⑤ **卷四巫师与玻璃**：梅吉斯回忆、粉红球。  
⑥ **卷五卡拉之狼**：狼群、卡拉汉。  
⑦ **卷六苏珊娜之歌**：米娅、莫德雷德、共业公司。  
⑧ **卷七黑暗塔**：破碎者、艾迪／杰克之死、罗兰登塔循环，号角变数。  
⑨ **钥匙孔的风**：插话故事层。

**【主要人物】**
罗兰；艾迪；苏珊娜／欧黛塔／黛塔；杰克；奥伊；佛拉格／华特／马登；血腥之王；苏珊·戴嘉多；艾伦；卡斯博；科特；卡拉汉；莫德雷德；派崔克·丹维尔；滴答人；伯廉；史蒂芬·金（书中作者）；黄衣人低等。

**【势力图谱】**
枪侠残部；大善人法尔森；公会与狼群；Sombra／北中正电子；共业公司；深红之王势力；吸血鬼与低民。

**【贵重物品】**
罗兰的枪；梅林彩虹／黑十三；钥匙；玫瑰；号角；查理火车书；灵魂与名字。

**【隐藏剧情 · 伏笔】**
多世界互文；十九；作者即角色；循环与号角。

**【大事记时间线】**
基列亡 → 追衣人 → 抽三 → 荒原 → 玻璃 → 卡拉 → 苏珊娜之歌 → 塔 → 循环。

**【叙事基调 · 雷区】**
冷硬西部＋恐怖。忌把罗兰写成无私圣人；忌塔战力归零；忌忽略 Ka 代价。锚点：荒漠追迹或门户之门。

**【Ka 的残酷美学】**
Ka 像轮：成全塔亦碾碎爱人。苏珊之死、杰克之弃、艾迪之殁皆是轮齿。契约者救一人，Ka 会从别处讨债——正文要写「讨债方式」而非取消命运。

**【语言】**
Thankee-sai；丹-特特；卡泰特；Ka-shume。中高阶可增加高语碎片，低阶少而精。

**【光束可见时】**
空气中有方向的光路，像世界自己指出路。沙迪克之死是科技神残可被杀的证明。

**【纽约层】**
1977／1987 等年代门：毒品、种族、残疾政治、出版业。艾迪的毒、苏珊娜的民权记忆、杰克的私立学校——现代戏与中世界同等重要。

**【狼群与机器人】**
卡拉的狼是科技与神话的合体恐怖；胜利靠训练村民与枪侠协同，不是罗兰一人秀。

**【破碎者】**

任务方向/奖励：水、地图残片、一句 Thankee。

**二阶（门户之门 · 抽牌夜）**

切入身份/时点：西海海滩门户出现时，以「被门误吸的纽约边缘人／海滩拾荒」切入。

初始事件：门上写着囚犯／阴影／推人者之一；你目睹艾迪被拖出，或必须递一支真枪给罗兰。

开场白建议：「门后是另一个年份的汽车尾气。枪侠缺了手指，却仍比你更像完整的人。」

关键NPC立场：**罗兰**；**艾迪**；**欧黛塔／黛塔**；海关式危险（现代侧）。

主线钩子/支线：帮助完成一次抽牌不崩溃；支线＝弄药、挡毒枭、安抚人格分裂。

危险度/规避：高。规避：关上门让卡泰特缺人。

任务方向/奖励：现代药品、弹药、初步枪训。

**三阶（光束之径 · 卡泰特成型）**

切入身份/时点：杰克回归前后，说话环至河渡镇。

初始事件：双记忆悖论使罗兰／杰克濒疯；你被要求在开门时守住环外。

开场白建议：「孩子说他死过。枪侠说他没死过。二者同时为真时，世界的牙在咯吱响。」

关键NPC立场：**杰克**；**苏珊娜**；**艾迪**；**奥伊**；梦魔。

主线钩子/支线：护送开门；支线＝刻钥匙、护玫瑰梦。

危险度/规避：高。规避：在环内交易灵魂。

任务方向/奖励：木钥匙仿品、光束方向感、卡泰特边缘承认。

**六阶（卡拉之狼 · 战争）**

切入身份/时点：布莱恩·史特吉斯卡拉迎狼前夜，以武装协力／匠人切入。

初始事件：**卡拉汉**展示狼面具；村民要你站队「交童」或「战」。

开场白建议：「机器人的眼在面具后亮了。神父说：枪不救灵魂，但今夜灵魂需要枪。」

关键NPC立场：**罗兰**；**卡拉汉**；村长；狼群执行体；艾迪／苏珊娜。

主线钩子/支线：守村夜战；支线＝改陷阱、救双胞胎、夺机器人零件。

危险度/规避：极高。规避：单独追狼入门。

任务方向/奖励：狼械零件、枪侠认可、卡拉公民权。

**七阶（近塔 · 条件性）**

切入身份/时点：霹雳地至玫瑰田至塔前，以幸存者／画家助手／共业公司特工切入。

初始事件：艾迪已死或将死的战场烟；你要保护**派崔克**或玫瑰不被低民踩灭。

开场白建议：「塔比故事更高。血腥之王在阳台上笑，像红线钉进世界的图钉。」

关键NPC立场：**罗兰**；**苏珊娜**；**派崔克·丹维尔**；**莫德雷德**；**血腥之王**（远）。

主线钩子/支线：条件性胜利＝保全玫瑰／协助抹去王的完整影像／送罗兰号角；支线＝葬奥伊、传信共业公司。

危险度/规避：顶点。**超阶巅峰：存在·情报优先／条件性胜利**，严禁战力归零。

任务方向/奖励：玫瑰瓣封印、号角残响、离开循环的旁支门（代价自定）；不发塔的所有权。
