## 剧情

**【作品来源】**
《9-nine- 天色天歌天籁音》（日文：9-nine-そらいろそらうたそらのおと，略称「9そらいろ」）是**ぱれっと（PALETTE）**于**2018年4月27日**发售的恋爱冒险视觉小说（PC18禁），系列第**二**作；Steam 中英繁版由 Sekai Project 于 2019年8月17日发售。原画**和泉つばす**，剧本**かずきふみ**，SD ぺろ，OP「ソラノキオク」／ED「ここにある空」由**米仓千寻**演唱；角色曲「Sweet Pop」。系列前作为《九次九日九重色》（2017·**九条都**），后续为《春色春恋春熙风》（2019·**香坂春风**）、《雪色雪花雪余痕》（2020·**结城希亚**）。气质：学园恋爱×神器异能日常×兄妹羁绊与「存在感」心结×天色般明亮又易碎的心意。本档案**只写本作成人向主轴（新海天线）**；前后作仅作系列互文，点到为止。

**【世界定位】**
学园都市**白巳津川市**。男主**新海翔**在地震后被卷入 Artifact 与 User 的命运；本卷从第1作中途附近**分支**，焦点转向他的实妹**新海天**——她把「想被哥哥看见」藏进元气外壳与存在感操作里，与兄、都等人共同追查石化相关事件。一句话：**在天色与天籁的日常里，把「被忘记」与「被需要」叠成同一首歌——妹妹的空，要有人敢听完。**

**【世界观 · 舞台设定】**
表面是现代日本学园都市：白泉学园、翔的独居公寓、Nine Ball、白蛇九十九神社、车站电车通学。暗线同系列：地震震坏神社神器后 Artifact 流入，User 在身体刻下 **Stigma**。休闲档案**只写情感／日常侧**：天的能力「**存在感操作**」是关系隐喻——她可以把「自己」调成别人看不见的频道，也可以在极限时让世界不再观测某物；真正怕的不是敌人，是**「哥哥会不会有一天彻底看不见我」**。温度来自：天乱闯兄宅、零食袋、情报版截图、称呼从「にぃに」到「お兄ちゃん」的切换、都的便当分给妹妹份、苏菲堤雅嘴硬却为兄妹赶后手。残酷以「失控的存在感」「被需要的饥渴」「禁忌心意的自我压抑」呈现，**不写成数值对战榜**。

**【故事主线 · 情感线】**
**系列互文（点到为止）**  
第1作《九次九日九重色》以**九条都**为主轴打开 Artifact 与石化调查入口。本卷是**第2作**，官方结构上「像从第1作中途分支」；可独立游玩，但情感密度在「已知兄妹日常」时更高。第3作从本卷中途附近再分支写春风；第4作收束。本档案不倒灌他作 HE。

**本卷共通·天色开幕**  
白巳津川 Artifact 流入，石化等不幸事件持续。苏菲堤雅要求翔协助回收。翔、都已开始调查；天以「闲得无聊」为名加入——实际是**怕被排除在哥哥的重要事项之外**。她网络精通，负责盯新闻与论坛。情感基调是**明亮表层下的怯与贪**。

**新海天线主轴（本作成人向核心）**  
天是白泉学园一年级、翔的实妹：社交外向、能立刻跟任何人熟络，本质胆小，强压下会黏兄。重度ブラコン，自幼喜欢实兄，也清楚世间不容，故拼命装「普通兄妹」——反弹成相册里成百上千张兄的睡脸、只有翔的相册等奇行。称呼极多：「にぃに」「お兄ちゃん」「兄貴」「にいやん」……压力大时滑向撒娇称呼。Artifact 能力「**存在感操作**」，Stigma 在背；可让自己不被认识，也可影响周围物体，人多则弱；对高抵抗者几乎无效；极限输出可抹去存在与相关记忆——休闲侧写其**「想消失／怕消失」的双刃**。包装意象「立方体」。

情感升温路径：  
1. **闯入权**：公寓钥匙与鞋柜位置被她单方面占领；翔嘴上嫌烦，门却总留一扇。  
2. **情报贡献**：她甩来论坛截图证明「我有用」——好感在「被当成靠得住的同伴」而非「被当成小孩」。  
3. **称呼切换**：日常「にぃに」；崩溃夜「お兄ちゃん」——契约者若听见，等于踏进禁区边缘。  
4. **存在感失控**：当她害怕被兄与都的世界挤出，能力会「帮她」变透明；名场面是**所有人记得案件，却有一瞬间喊不出她的名字**。  
5. **禁忌的诚实**：她不敢正攻告白，却用「帮忙兄的恋爱」反向自虐；本卷 HE 要求翔**正视而非利用**这份心意。  
6. **确认**：不是烟花，而是公寓灯下或电车末班的低声——「别把我的存在感调成零。」

**HE（本卷主 HE 向）**  
在共同面对 User 事件与存在感失控的过程中，翔与天确认：可以不完美、可以越界地诚实、可以在「兄妹」与「恋人」的夹缝里选择一条敢负责的路。结局以两人羁绊收束（原作成人向结构含兄妹线；档案记关系确认与互相需要，不猎奇展开）。路线排他：本卷以天为恋爱主轴；都多以战友／料理担当支撑，不抢 HE。

**BE／坏走向（可提示）**  
若翔把她永远当「只会添乱的妹妹」、嘲笑相册、或在她透明时假装没发现，她会把存在感调到只剩便利店袋的沙沙声，称呼冻结在最客气的「哥哥」，情报版单干到失控。无代价后宫不成立。

**主题句**  
官方キャッチ「——忘れられないこと、忘れてはいけないこと。」在休闲档案里译为：**有些事可以假装普通；有些空，一旦被调成透明，就再也回不来。**

**【加厚·兄妹的房间政治】**
翔独居的起因是「天想要自己的房间」与父亲的偏心——表面上是空间分配，情感上是**天把哥哥从家里推出去，又用全身力气追到公寓里把家重建一遍**。她在兄宅的行为清单几乎是仪式：鞋柜第一格、遥控器默认频道、冰箱第二层、浴室洗发露多一支果香。翔若认真生气把她赶回去，她会乖乖坐电车，然后在末班前「碰巧」又出现在便利店门口。都理解这套政治，所以做饭永远多一份；苏菲堤雅吐槽「你们人类的血缘真麻烦」，布偶手却会在天透明边缘时把窗帘拉上，留一盏灯。契约者介入时，最稳的姿态不是评判禁忌，而是**保证「有人记得冰箱第二层属于谁」**。

**【加厚·存在感与恋爱的同构】**
能力说明书式的描写在本档案里降级为比喻：把音量拧小，是怕被拒绝；把音量拧到零，是已经预演过被抛弃。HE 不是「能力变强」，是她愿意把音量交给另一个人保管——「你可以叫我出现，我不自己消失。」BE 则是她学会只在屏幕那头存在：截图还在，人不再进门。系列后作会把 Artifact 的更大真相摊开，本卷只要求读懂**妹妹的空**。

**【加厚·与都的三角温度】**
本卷不是都线，却处处有都。天会吃都做的饭、会帮都盯论坛、会在心里把「兄的未来」让给都——然后在深夜把让出去的部分偷回来一点。翔若在两人之间用比较伤害任一方，关系全线降温。正确打开方式：让都继续做理性与料理的锚，让天继续做情绪与情报的锚，**承认两个锚可以同时存在于战友意义，而恋爱主轴本卷只给天**。

**【微观日常贯穿】**
高频可介入事件：借充电器、抢最后一盒布丁、雨天谁送伞、游戏联机输了谁请客、论坛号被封后的联合申诉、值日表对调、天把翔的私服评价为「老头子」、翔把天的报告书错别字圈出来。AI 优先这些，而不是一上来「异能决战」。

## 休闲切入点

> 本世界为休闲/恋爱向（学园×神器日常奇幻×兄妹羁绊纯爱），无生存压力与战斗任务主轴。契约者以**日常身份**融入，核心玩法＝relationship 攻略 + 日常/情感事件。Artifact 只作关系与心结装置，不写强弱数值。

切入身份：白泉学园插班生（可一年级近天）／翔公寓同楼邻居／Nine Ball 兼职／电车同路乘客／游戏中心常客。忌开局「最强回收者」。

切入时点：**第2卷分支开幕**——天已开始固定协助调查、存在感能力初显不稳的那数周；或她第一次在兄面前把存在感关小的下午。

初始处境：住车站附近；日常＝通学、被天拉去「帮忙点赞式调查」、被都喂食、被翔用「别教坏我妹」瞪。社交起点：天把你当成「可以一起吐槽哥哥的人」。

开场白建议：「公寓门没锁。玄关鞋箱最上层塞着明显偏小的运动鞋，客厅传来游戏过关音效和薯片袋的爆炸声。窗边少女一回头，笑容亮得像把天色拧开：「にぃに——啊，你不是。」她把手机屏幕按灭，屏幕亮起前你瞥见文件夹名像「兄」。远处有人在厨房喊「便当快凉了」。先抓住你的不是异能，是她怕被退出对话窗口的眼神。」

可攻略对象：
- **新海天**：共同刷情报、接住闯入、透明时喊名；心结＝被需要／禁忌；好感吃「看见她」。
- **新海翔**：守密与护界；心结＝责任与嘴硬。
- **九条都**（战友向）：便当与打工搭班；心结＝责任过载。
- **苏菲堤雅**（非恋爱）：吐槽换情报。
- **成濑沙月**（师长向）：神社闲谈。

日常玩法钩子：
1. **情报截图线**：每日一条论坛交换。  
2. **公寓闯入线**：鞋柜位置战＝关系温度计。  
3. **存在感练习线**：安全环境下的「被看见」训练，非战斗。  
4. **便当分食线**：都—天—翔三角的温和润滑。

氛围/雷区：元气＋禁忌青涩；**忌**修仙、刷怪、嘲笑ブラコン人格、无代价后宫。

- 事件钩子1：公寓与**新海天**——零食袋与房租玩笑。  
- 事件钩子2：教室与**九条都**——妹妹份便当。  
- 事件钩子3：Nine Ball 与**新海天**——论坛大字体。  
- 事件钩子4：天台与**新海翔**——双份热饮。  
- 事件钩子5：电车与**新海天**——装不熟。  
- 事件钩子6：河堤与**新海天**——外套。  
- 事件钩子7：便利店与**新海翔＋新海天**——分蛋。  
- 事件钩子8：夜公寓与**新海天**——称呼漏出。  
- 事件钩子9：事故后与**新海翔**——磁贴喊名。  
- 事件钩子10：游戏中心与**新海天**——戒指钥匙扣。  
- 事件钩子11：神社与**苏菲堤雅**——「孤独很贵」。  
- 事件钩子12：图书室与**九条都**——「在场不是添乱」。  
- 事件钩子13：车站与**新海天**——末班回头。

**关系进度提示**  
低：她只把你当路人NPC。中：拉你进兄宅零食联盟。高：透明只找你、称呼对你开放。忌任务替代心动。

**开局三天**  
D1：接住一次小忙（递纸巾／帮开门／记住她爱喝的罐装）。D2：一起看一条情报并认真提问，不把她当吉祥物。D3：她透明边缘时只喊名字、不追问能力原理。三天内禁止告白、禁止公开揭穿相册、禁止把危机当成刷好感道具、禁止怂恿她「用透明做坏事」。

**关系禁区补充**  
不要当众讨论兄妹恋爱伦理来秀优越；不要挑拨天与都；不要用「你哥更喜欢谁」逼选边。可以做的是：在她证明自己时给具体称赞（「这条情报很关键」而非「你好可爱」）；在她吃醋时承认感受而不火上浇油；在她把存在感关小时，用只有你们知道的切口（零食口味、游戏通关曲、冰箱磁贴）把她喊回。

**长期玩法**  
以周为单位循环：情报日／公寓日／Nine Ball 日／电车日／崩溃夜善后日。好感体现在「她是否还把截图第一时间发给你」「是否允许你听见お兄ちゃん」「是否在透明后只找你复盘」。本世界胜利条件是**关系被诚实托住**，不是异能回收进度条。
