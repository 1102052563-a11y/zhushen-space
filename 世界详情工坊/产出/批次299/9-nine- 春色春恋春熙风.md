## 剧情

**【作品来源】**
《9-nine- 春色春恋春熙风》（日文：9-nine-はるいろはるこいはるのかぜ，略称「9はるいろ」）是**ぱれっと（PALETTE）**于**2019年4月26日**发售的恋爱冒险视觉小说（PC18禁），系列第**三**作；Steam 中英繁版由 Sekai Project 于 2020年9月1日发售。原画**和泉つばす**，剧本**かずきふみ**，SD ぺろ，OP「ハルトキ～Spring Moment～」／ED「そして愛になる」由**米仓千寻**演唱；角色曲「floating Heart」。系列前作为《九次九日九重色》（2017·**九条都**）、《天色天歌天籁音》（2018·**新海天**），后续为《雪色雪花雪余痕》（2020·**结城希亚**）。气质：学园恋爱×神器异能日常×双人格的春色心事×御宅与猫与「心想事成」的温柔危险。本档案**只写本作成人向主轴（香坂春风线）**；前后作仅作系列互文，点到为止。

**【世界定位】**
学园都市**白巳津川市**。男主**新海翔**在地震后被卷入 Artifact 与 User 的命运；本卷从第2作中途附近**分支**，焦点转向白泉学园三年级前辈**香坂春风**——内向主人格与女王副人格共住一具身体，项链形态的 Artifact 让「思考」有时会变成「现实」。一句话：**在春色与春恋的风里，把「两个形状」叠成同一个人——她的世界有两个入口，钥匙都是被认真喜欢。**

**【世界观 · 舞台设定】**
表面是现代日本学园都市：白泉学园、翔公寓、Nine Ball、电车通学、同人与动画网站、野猫出没的小巷。暗线同系列：Artifact 与 **Stigma**、User 组织「**リグ・ヴェーダ（梨俱吠陀）**」与管理侧「セフィロト」交错。休闲档案**只写情感／日常侧**：春风的能力「**思考的具现化／心想事成**」是心结装置——妄想过强会泄漏成现实，副人格曾是论坛人设被能力「养」出来；真正的危险不是招式名，是**「不敢用主人格的声音说喜欢」**。温度来自：主人格的小声与失言、副人格的大胆与护短、猫、漫画堆、与希亚聊动画的共鸣、都的便当被两人格轮流夸、天当恋爱参谋却吃醋。残酷以「能力失控的妄想」「组织立场的撕裂」「被当成怪孩子」呈现，**不写成数值对战榜**。

**【地理 · 生活舞台】**
- **白巳津川市**：学园都市总舞台。
- **白泉学园**：翔二年级、天一年级、春风三年级前辈；教室、天台、无人走廊是人格切换高发地。
- **新海翔公寓**：本卷恋爱升级与崩溃善后的私密舞台；漫画与游戏成为共同语言。
- **咖啡店 Nine Ball**：都打工；春风可被拖来「人类观察」；希亚圣代位作后作伏笔。
- **电车车厢／车站**：春风与天早有面识；通学是主人格练习「不逃」的考场。
- **野猫小巷／公园沙地**：猫控圣地；与希亚兴趣重合。
- **春风的房间（御宅巢穴）**：漫画、动画BD、语音作品、论坛「アガスティアの葉」——副人格的诞生土壤。

- **リグ・ヴェーダ相关据点（模糊）**：组织侧阴影；本卷写「立场选择」多于据点攻略。

**【故事主线 · 情感线】**
**系列互文（点到为止）**  
第1作以**九条都**打开调查；第2作以**新海天**写兄妹与存在感。本卷是**第3作**，结构上像从第2作中途分支；可独立游玩。第4作以**结城希亚**收束并强烈建议通关前三作。本档案不倒灌他作 HE。

**本卷共通·春色开幕**  
石化与 User 事件持续，调查小队（翔・都・天・苏菲堤雅）与自称司令官的**高峰莲夜**、冷酷的**GHOST**、以及组织「リグ・ヴェーダ」交错。春风曾以代号「**エンプレス（Empress）**」与组织见解部分重合——「User 不该被随便束缚」——却厌恶暴力，事实中立，后离反到翔一侧。情感基调是**春风式的摇晃：逃与冲、羞与欲、两个「我」都想被选。**

**香坂春风线主轴（本作成人向核心）**  
春风是白泉三年级：主人格内向、话少、怕男性（小学霸凌阴影），观察力却强；副人格大胆、女王气、会在通学途中用能力弄出「被围着走」的戏码——那多是主人格「想被喜欢」的妄想被当真，而非副人格真的男控。两人格**记忆共享、取向大体一致**，与解离性障碍教科书不同；副人格源自动画／论坛人设被能力具现。重度御宅，漫画从少女到青年通吃，游戏动画语音全开，与莲夜在粉丝站早有交流，与希亚因猫与二次元投缘。Artifact 为项链，「心想事成」；Stigma 在左胸；消耗大，长时只能稳在人格相关用途；后期可把能力「染成自己的颜色」达完全控制——休闲侧写为**「第一次承认两个我都是我」**。包装意象「拼图块」。

情感升温路径：  
1. **误认开局**：能力曾被误会成「魅惑」；澄清过程是信任的第一块拼图。  
2. **主人格的失言**：她会突然把性癖或妄想说漏，然后当机；翔若嘲笑则锁死，若接住则升温。  
3. **副人格的护短**：女王模式帮主人格挡麻烦，也逼主人格面对「你其实想要」。  
4. **组织裂痕**：リグ・ヴェーダ与调查小队立场冲突时，她必须选「理念」还是「在乎的人」。  
5. **猫与同人**：一起蹲猫、交换推荐作品——比战斗更像恋爱。  
6. **确认**：不是只接受副人格的华丽，也不是只怜惜主人格的软弱——**两个形状，同一份春恋**。

**HE（本卷主 HE 向）**  
在离反组织、收束能力失控与心意坦白后，春风与翔确认：不需要用妄想代替现实，也不需要杀掉某个人格才能被爱。结局以两人羁绊收束；副人格仍在，但不再只是逃避通道。路线排他：本卷以春风为恋爱主轴；都／天多战友与妹妹位，希亚作友人向，不抢 HE。

**BE／坏走向（可提示）**  
若翔只消费副人格的大胆、逼主人格「正常一点」、或把她的能力当许愿机，她会把项链藏进最深层抽屉，主人格消失在人群里，副人格只在论坛冷笑。无代价后宫不成立。

**主题句**  
官方キャッチ「——2つのカタチ、2人のセカイ。」在休闲档案里译为：**两个形状可以同居；两个世界要同一条风，才能叫春。**

**【加厚·双人格的房间公约】**
春风房间墙上没有「人格排班表」，却有心照不宣的公约：主人格负责白天的学校与道歉；副人格负责夜间的果断与回击；共同负责猫粮钱与动画追更。翔若只在副人格时段出现，主人格会把门锁扣上；若只怜惜主人格而命令副人格「消失」，项链会冷掉一整周。正确的共处是**开会**：两个声音都可以发言，决议写在同一张便签——「今天想被怎么对待」。契约者最稳的介入是当会议记录员，而不是当法官。

**【加厚·组织与恋爱的拉扯】**
リグ・ヴェーダ提供「不用道歉的强者幻想」；调查小队提供「要道歉也要一起活」的日常。春风的离反不是热血演说，是一串小选择：拒绝一次暴力、归还一次不该具现的妄想、在莲夜的中二台词里听出孤独却仍说「我不站这一边」。HE 的政治含义是**她把自己从代号里解雇，雇回真名**。

**【微观日常贯穿】**
高频事件：推荐番剧、借漫画、项链保养布、猫罐头品牌之争、主人格做饭失败的黑锅、副人格点外卖的报复、论坛被炎上后的共同潜水、考试周项链「不许发光」誓言。AI 优先这些。

## 休闲切入点

> 本世界为休闲/恋爱向（学园×神器日常奇幻×双人格前辈纯爱），无生存压力与战斗任务主轴。契约者以**日常身份**融入，核心玩法＝relationship 攻略 + 日常/情感事件。Artifact 只作关系与心结装置，不写强弱数值。

切入身份：白泉学园插班生（可近三年级／图书委员）／翔公寓同楼／Nine Ball 兼职／同人交流会临时工／猫巷常客／动画网站版务熟人。忌开局「最强许愿者」。

切入时点：**第3卷分支开幕**——春风与调查小队接触加深、能力误认尚未完全澄清或刚澄清的前后；或她在组织与小队之间开始动摇的那个春天感季节。

初始处境：住车站附近；日常＝通学、被拉去「人类观察」、被天拉去恋爱会议、被都喂食。社交起点：春风主人格把你当成「不会突然靠近的安全人」，副人格把你当成「可以斗嘴的对象」。

开场白建议：「天台风把窗帘似的云扯薄。栏杆边站着三年级的前辈，项链在锁骨处一闪。她先是小声到几乎听不见的「……那个」，忽然语气一变，笑得像女王巡视：「找我？排队。」同一张脸，两种春色。远处有人喊便当凉了，有人抱着同人纸袋跑过。先抓住你的不是异能，是她不知该用哪一副表情说『喜欢』的慌。」

可攻略对象：
- **香坂春风**：双人格平等、御宅共鸣、蹲猫、拒许愿机；心结＝逃与整合；好感吃「两个都要」。
- **新海翔**：守密与不利用；心结＝责任嘴硬。
- **九条都**（战友向）：便当打工。
- **新海天**（妹妹向）：情报与参谋。
- **结城希亚**（友人向）：猫与动画。

日常玩法钩子：
1. **人格会议线**：便签决议「今天怎么相处」。

4. **离反善后线**：立场选择后的日常修补，非战斗结算。

**关系进度提示**  
低：主人格逃、副人格嘲。中：允许你看切换、共享追番表。高：会议便签写你的名字、主人格肯独自说完心声。忌任务替代心动。

**开局三天**  
D1：小忙＋不突然靠近。D2：聊一部作品或一只猫。D3：一次切换时两边都称呼「春风」。禁告白、禁命令某人格消失、禁许愿测试、禁战力化。

**关系禁区补充**  
不要把副人格当「更好用的皮肤」；不要逼主人格表演勇敢；不要在莲夜与小队之间逼她当场站队秀忠诚。可以做的是：记录她的失言却不传播；在妄想泄漏成小麻烦时先善后人格尊严；送猫罐头而不是送「能变强的道具」。

**长期玩法**  
周循环：追番日／猫日／Nine Ball 日／人格会议日／组织余波谈话日。胜利条件是**两个形状都被爱着的日常**，不是回收清单清空。
