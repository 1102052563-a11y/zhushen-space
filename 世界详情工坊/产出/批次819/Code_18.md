## 剧情

**【作品来源】**
《code_18》（Code_18）是 Cyberfront 于 2011年9月29日发售的恋爱 ADV（PSP／Xbox 360，2012年12月21日 PC），infinity 系列再启动之作。角色原画赤津豊，剧本原案日暮茶坊，Run&Gun 执笔，音乐 milktub。气质：学园恋爱＋时间回环；封闭空间脱出主题被刻意拿掉，改为**文化祭前的一个月与五位少女**。本档案聚焦**恋爱线与人物**，回环设定只作情感结构。

**【世界定位】**
2018年东京·浅草附近的**龙成学园**（大规模单位制高校）。男主**飞野逸人**是第二科学部发明少年；与**春奈ひかり**等少女在文化祭前交织恋与秘密。契约者进入的是**部室、屋顶、祭典准备与循环中的心意**。

**【世界观 · 舞台设定】**
单位制大学式高校：班级归属感弱，部活与个人课程更重要。第二科学部从科学部分裂而出，顾问**神原玄九郎**；部室常变成点心茶会。アガスティアの石嵌入智能手机接收「未来语音」——只译为**命运的旁白与父辈的指引**，不写武器化。ドラゴン号飞行单元是青春发明象征。回环：文化祭高潮后时间回到 9月11日，逸人逐次与不同ヒロイン深入，最终章解谜。温度：部室零食、屋顶风、文化祭灯、番茄、未听清的语音。

**【地理 · 生活舞台】**
- **龙成学园屋顶**：ドラゴン号试验、有里佳坠落相遇。  
- **第二科学部部室**：发明、点心、恋爱咨询。  
- **教室／单位制走廊**：偶遇与错过。  
- **春奈家附近**：几乎独居的生活照顾。  
- **馆川家／商店街**：お嬢様与オタ活动。  
- **文化祭会场**：演唱会、展示、烟花式收束。  
- **スカイタワー相关（终盘）**：情感上的「送女儿回家」之路。

**【故事主线 · 情感线】**
**结构**  
固定顺序：共通プロローグ → **春奈ひかり** → 回环 → **馆川珠姫** → 回环 → **空木柚子** → 回环 → **鸟仓菜菜莉** → 回环 → **时任有里佳**终章。选项主要避 BE，非自由选线；休闲扮演可强调「每次循环的恋爱认真度」。

**春奈ひかり线**  
中学相识、料理照顾逸人、前子役、因家庭不和无法再唱。情感：修家庭与自信，文化祭ライブ成功后确认恋心——回环抹去结果，她在后续循环中默默退让，却在最后一次行动中救下关键。终章暗示她是未来母亲。HE 感：歌与恋都回到声音。

**馆川珠姫线**  
名家お嬢様、高飞车皮、重度オタ（コスプレ／BL 公言）。情感：被看见「不体面的爱好」仍被喜欢；文化祭コスプレ企画。HE：摘下傲娇，邀请共同参展。

**空木柚子线**  
ボーイッシュ长女、五人姐弟、脚比嘴快。情感：不擅长直球，用踢与骂遮羞；弟志虎过保护的喜剧。HE：踢完之后的道歉是告白。

**鸟仓菜菜莉线**  
眼镜、寡言努力家、实为神原之女。情感：内向被误认为冷淡；父女与发明主题共鸣。HE：主动牵手的一毫米。

**时任有里佳／飞野あかり终章**  
坠落少女、敌视逸人。真身：2040 的女儿**飞野あかり**，因工作狂父亲偷走时间机器。情感核：父女和解、与ひかり的命运闭环、未来语音的本体论悖论式关爱。HE／True：送她回未来，循环解除，家庭在时间两端同时修好。

## 休闲切入点

> 本世界为休闲/恋爱向，无生存压力与战斗。契约者以**日常身份**融入，核心玩法＝relationship 攻略 + 日常事件，而非任务厮杀。

切入身份：龙成学园**转校生／第二科学部新部员／文化祭实行委员**。部员身份最自然：点心会＋发明失败喜剧。

切入时点：**2018年9月11日前后**——ドラゴン号试验日或第一次接住有里佳之后；文化祭倒计时一个月开启。

初始处境：单位制选课与逸人实验室时段重叠；被晴夏塞点心、被义人搭话、被神原称为「另一个会熬夜的笨蛋」。住学校附近公寓，第一周在屋顶看试飞。

开场白建议：「龙成学园的屋顶风很大，大到能把发明少年的护目镜吹歪。你帮着扶住叫做ドラゴン号的飞行框架，旁边飞野逸人念着数据，口袋里的手机突然吐出一串沙沙的语音——谁也听不清，他却说『老师讲这是未来』。更远处，天空像破了个口，有个少女的轮廓向下落。你还来不及害怕，逸人已经张开双臂。文化祭的横幅在楼下晃，你忽然觉得这个月会很长，也会很短。」

可攻略对象：
- **春奈ひかり**：部室与料理；吃「认真听她唱歌的欲望」；钩子＝家庭与ライブ。  
- **馆川珠姫**：オタ=======公开；吃「不嘲笑癖好」；钩子＝コスプレ文化祭。  
- **空木柚子**：互怼与姐弟；吃「接住别扭」；钩子＝踢与告白。  
- **鸟仓菜菜莉**：图书馆与实验；吃「给内向者时间」；钩子＝父女阴影。  
- **时任有里佳（あかり）**：敌视软化；吃「稳定的父亲式温柔（非血缘扮演也可）」；钩子＝回家与原谅。

日常玩法钩子：
1. 第二科学部点心例会。  
2. 文化祭准备与ライブ协力。

5. 终盘送行前的全家福式合影（情感）。
