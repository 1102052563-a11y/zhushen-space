# 触手寺院-修行完了
<!--meta lib=休闲 tiers=休闲-->

## 剧情

**【作品来源】**
本档案对应世界库休闲条目「触手寺院-修行完了」，**无单一可考原作小说/动画/galge**。设定气质取自东亚寺院日常、日本佛教寺院生活节奏与温和奇幻恋爱向二次创作常见的「山中结业季」母题；触手元素仅作寺院灵物/结界符号，**禁止**猎奇恐怖与战斗叙事。档案服务轮回乐园休闲世界卡：信息密度优先，写僧院生活、修行伙伴情感与结业仪式日常。参考公开资料见文末来源（佛教通识、日本佛教寺院制度、寺院建筑与修行节奏等）。

**【世界定位】**
山中古刹「柔灯寺」进入一年一度的「修行结业」季：一批学徒将完成功课、领取路引、在晚钟与灯火中告别或留下。契约者若切入，落点是结业前数周的温吞日常——扫廊、抄经、共膳、夜谈、典礼彩排，以及与修行伙伴之间迟来的、说不出口的情愫。世界温度来自雾气、木廊、热汤、并肩劳作，而非对抗。

**【世界观 · 舞台设定】**
舞台为架空东亚山国深处的「柔灯寺」。寺名来自传说：山雾里曾有柔软、发微光的「灯触」守护迷路香客，后被历代住持收为「护法灵物」——它们形态似触手藤蔓，却不伤人，只在晨课、温泉、结业礼中作为「温柔结界」出现：托起灯盏、拂去经卷尘、在人疲惫时轻轻托住后背。寺规明确：**灯触是伙伴与象征，不是敌人，也不作比武题材**。

时代基调接近前现代晚期至近代交界：有电灯尚未普及，寺内以油灯、纸灯笼与灯触自身的淡金微光照明；山脚有小镇「雾津」，每月两次市集。社会氛围安稳，香客多为还愿、求安、送子女入寺短期修行。学徒制度分三层：新入（扫廊与膳房）、常修（功课与仪轨）、结业候（完成「四季功课册」后等待结业礼）。本档案锚定**结业候**阶段——功课已近尾声，人心却反而最软、最乱：有人怕下山、有人怕留下、有人怕把「喜欢」说出口后破坏同袍情谊。

奇幻只服务情感侧：灯触会在人说谎时轻轻点额（像提醒）、在两人并肩沉默时把同一盏灯推到中间。不展开咒术对决、不写修行比武或强弱排行。治愈感来自：准时的粥、雨后廊檐共伞、抄经时指尖相碰、结业夜山风与晚钟。

寺院日课节奏（可当日常剧本表）：寅时起身、漱洗、山门静立片刻；卯时早课诵读与灯触巡廊；辰时粥食、分派洒扫；巳时至未时为功课——抄经、仪轨、园艺、膳房轮值；申时茶歇与自由交谈；酉时晚课；戌时洗浴或温泉（结业前三日特开）；亥时熄大灯，仅留小铃一类夜灯。结业季在此表上叠加「终课」：彩排、路引誊抄、香客接待演练。没有人用「谁更强」衡量彼此，评价标准是功课册是否诚实、同袍是否被好好对待。

**【地理 · 生活舞台】**
- **山门与石阶**：四百余级青石，两侧苔痕与风铃。结业季清晨，候礼学徒须赤足拾级「静心一遍」；初遇、重逢常发生在半山腰的歇脚亭。
- **大殿与回廊**：木构回廊环绕中庭，雨天最适合并肩擦地板。灯触常沿梁游走，把掉落的银杏叶卷回庭中。
- **藏经阁二层南窗**：光线最好，是抄经与「功课册互批」的固定席。名场面：两人争同一支秃笔、被墨点染到对方袖口。
- **膳堂与灶房**：大锅粥、咸菜、偶尔的山菌汤。灶火边是坦白秘密的地方——蒸汽遮住脸，话就容易说。
- **温泉别院「柔汤」**：寺内私汤，结业前三日开放给候礼学徒。雾气中只能看见轮廓；灯触会把木桶递到岸边，像沉默的侍者。氛围暧昧，但不写成猎奇。
- **钟楼与观星台**：晚课之后可登楼。结业礼前夜，许多人在此决定「走」或「留」。
- **雾津小镇**：茶屋、纸店、卖糖的婆婆。下山试穿俗世衣的半日行程，是情感升温的经典约会舞台。
- **后山竹林小径**：通往无名石佛。适合谈心、和解、递手作念珠。
- **染衣房与晾廊**：结业前要换一季新净衣。晾廊风大，衣袂猎猎，是「帮对方整理衣领」的经典亲密镜头发生地。
- **香积库与柴间**：铃兰的地盘延伸处；雨夜躲雨时，柴香与人声很近，秘密容易落地。
- **客堂侧门**：雾津来人登记处。澄真归山、契约者挂单，都从这扇门被写进名簿。

**【故事主线 · 情感线】**
**共通线·结业季前奏（约四至六周）**  
故事从「功课册盖完最后一个印」开始。候礼学徒名单贴在膳堂门板：青莲、白雾、绯樱、墨渊、铃兰、澄真（外归）等。住持静心宣布：结业礼定在满月夜，此前每日一轮「终课」——抄经、仪轨彩排、山门迎客、为灯触梳理「光丝」（以清水与软布擦拭灵物表面，类似照料植物，完全非暴力）。  

日常推进靠小事：谁起得最早谁负责开窗；谁把粥舀得太满被笑；谁在藏经阁睡着，脸压出经文印。灯触「小铃」特别亲近青莲，总把同一盏灯滚到青莲与白雾之间——像在起哄。绯樱则总把「结业后去看海」挂在嘴边，却在夜深人静哭。墨渊表面冷淡，功课册却被发现夹着所有人的名字与小涂鸦。  

**情感升温**  
雨季来临，廊下共伞成为常态。青莲发现自己会下意识等白雾的脚步；白雾则把青莲忘在膳堂的手帕洗干净叠成方块，塞回对方功课册。绯樱拉着铃兰去雾津买糖，归途在石阶上谈「如果不想分开，能不能一起申请留寺」。澄真从城中归来，带来俗世的书信与衣料气味，搅动旧日承诺——他曾与青莲约「结业后一起下山」，如今却犹豫。  

**冲突与误会（非战斗）**  
结业彩排日，名册被墨弄脏，有人误以为是「被除名」。其实是小铃碰翻砚台。绯樱以为墨渊故意不告诉她「留寺名额有限」，冷战三天；和解发生在灶房——两人一起淘米，水声盖过道歉，却比任何大道理更真。青莲误读白雾给静心的信，以为白雾已决意远赴他寺任职；真相是白雾在为青莲争取「双人行脚」资格——可结伴云游一年再决定去留。  

**告白与选择**  
满月前夜，观星台。灯触把整片回廊的灯一一点亮，像给人间铺一条光路。各条情感线在此分叉：有人说「想和你一起听明年的钟」；有人说「先下山，信会寄到」；有人只是把额头轻轻抵在对方肩上，让灯触绕腕一圈当作无声的誓。  

**结业礼·结局走向**  
满月夜，大殿外设灯阵。静心宣读路引：有人接俗世文牒，有人接「留寺功课」。典礼不比高低，只比是否诚实面对心。灯触升空如缓慢的烟花（无爆炸、无杀伤），众人合十。结局基调为**温柔多结局**：可结伴留下、可结伴下山、可约定三年后再会；核心不是输赢，而是「修行完了，情感才刚开始」。

**个人线概要**  
- **白雾线**：温柔引导者学会为自己争取。结局多为双人行脚或共同留寺，HE。  
- **绯樱线**：元气外表下的分离焦虑被接住。结局可结伴去海边小镇开茶屋，或短期留寺再启程，HE。  
- **墨渊线**：冷淡外壳下的细腻记录被看见。结局常是共同整理寺史、成为下一任藏经助手，安静HE。  
- **铃兰线**：灶房日常里的踏实陪伴。结局可共同掌厨、招待香客，烟火气HE。  
- **澄真线**：旧约与新心的和解。结局可重续下山约，或坦然放开成挚友，HE/温和开放结局。  
- **静心线**：非恋爱向亦可敬重亲近——师徒信任、被认可「可以带着心去生活」。

**分周细谱（共通线加厚）**  
第一周「印落之后」：名单公示，众人表面平静，夜半却有人在廊上反复踱步。青莲被分到与白雾同组终课，第一次被近距离看见「辅导时的认真侧脸」。绯樱拉人练习迎客辞，笑场三次；墨渊在旁纠正用词，语气冷，耳朵却红。铃兰加餐「安神甜粥」，小铃把碗一个个推正。  

第二周「雨季与手帕」：连阴雨。共伞事件发生；手帕往返成为无声情书。澄真归山，客堂侧门湿脚印一直拖到膳堂，旧约被重新提起。青莲失眠，钟楼遇见墨渊——对方只丢来一句「功课册夹页别被雨泡了」，第二天却发现夹页已用油纸包好。  

第三周「误会与淘米」：名册墨污事件；绯樱—墨渊冷战；白雾书信被误读。灶房和解夜，灯触把两只碗碰出轻响，像替人鼓掌。静心不插手感情，只多批了一行字在全员功课册扉页：「结业不是结束喜欢的资格。」  

第四周「柔汤与观星」：温泉别院开放三日。雾中对话只留声音，最易说「想继续」。满月前夜观星台灯阵，告白或抵额；路引意向条投入木匣。  

满月日「路引与后日」：典礼、灯升、袖与袖相触。后日谈可接雾津市集重逢、一年后回寺听钟、双人行脚途中的第一封家书（寄回柔灯寺膳堂，铃兰会代收并念给留下的人听）。  

**【可攻略角色 / 主要人物】**

**青莲（候礼学徒·本季结业核心视点）**｜外貌：墨色短发微卷，深棕瞳，常穿洗旧的灰蓝僧衣，腕上系一圈褪色红线（幼时母亲所留）；笑时右颊浅涡。｜性格：温和、怕麻烦别人、观察力强，关键时刻会突然勇敢。｜角色类型：温柔主人公 / 迟钝系（对感情）。｜萌点/魅力：认真抄经时咬唇、被灯触顶手心会不好意思、会默默记住每个人爱喝的粥。｜个人线剧情：结业前在「留下」与「下山」间摇摆；与白雾、绯樱、墨渊等关系随选择分叉；学会把「想在一起」说出口。｜与主角关系：契约者常可叠合/并行其位置，或以同袍身份介入其选择。

**白雾（常修偏师·结业辅导）**｜外貌：浅银长发低束，灰蓝瞳，衣摆总沾一点粉笔灰与檀香；手指修长，常夹一支未削完的木簪。｜性格：沉静、照顾人到忘我、不擅长谈自己的愿望。｜角色类型：大和抚子 / 温柔姐系。｜萌点/魅力：替人掖被角、下雨先把伞让出、被夸奖会耳尖发红。｜个人线剧情：心结是「辅助别人完成修行，却不敢给自己留位置」；攻略关键是让她承认「也想被选择」；结局双人行脚或共留。｜与主角关系：先是可靠前辈，后成可并肩的伴侣。

**绯樱（候礼学徒·同届）**｜外貌：赤褐马尾，琥珀瞳，僧衣袖口总卷高一截，踝上有跑山磨出的浅疤。｜性格：元气、直球、怕寂寞却死撑「我才不怕」。｜角色类型：元气 / 口是心非。｜萌点/魅力：喊「修行完了就去看海」、把糖藏在功课册、哭的时候还要笑着说「灰进眼里了」。｜个人线剧情：分离焦虑；需被接住「想一直在一起」的任性；结局海边或共留。｜与主角关系：损友式升温，最快破冰也最易吃醋。

**墨渊（候礼学徒·同届）**｜外貌：黑发齐耳，墨色瞳，总把领口系到最严；左手有长期握笔的茧。｜性格：寡言、毒舌但句句中肯、记性好到可怕。｜角色类型：冷淡系 / 外冷内柔。｜萌点/魅力：功课册夹页涂鸦、默默把你的经卷补抄完、被拆穿关心会移开视线。｜个人线剧情：心结是「用距离保护自己」；攻略靠共同整理藏经与夜谈；结局安静的长期陪伴。｜与主角关系：先保持一臂距离，后成为最稳的锚。

**铃兰（膳堂主事学徒·结业候）**｜外貌：栗色丸子头，圆杏眼，围裙永远干净，指尖有姜与柚皮气味。｜性格：踏实、嘴甜、用食物说话。｜角色类型：治愈厨娘系。｜萌点/魅力：凌晨揉面、把烧焦的饼笑着吃掉、用「再喝一碗」代替安慰。｜个人线剧情：怕自己「只会做饭不够修行」；被认可「烟火即功课」后释然；结局共掌厨或开山脚小馆。｜与主角关系：从共灶到共灯，节奏最稳。

**澄真（外归同袍·曾同期）**｜外貌：深栗短发，眉眼清朗，已改半俗装：深色短褂外罩旧僧衣，像两段人生叠在一起。｜性格：爽朗、重诺、对山外世界又向往又不安。｜角色类型：幼驯染 / 归乡者。｜萌点/魅力：记得旧日约定细节、会把城中见闻讲成故事、沉默时搓袖口。｜个人线剧情：旧约「结业后一起下山」与现下心意冲突；需诚实重谈；结局续约或挚友式放开。｜与主角关系：旧识重逢，带俗世气息的对照线。

**静心（柔灯寺住持）**｜外貌：白发整齐盘起，眼尾细纹，目光清亮；手持竹杖却很少真正拄地。｜性格：慈悲而清醒，玩笑里藏功课。｜角色类型：师长 / 智慧长辈。｜萌点/魅力：用粥的咸淡讲人生、允许年轻人犯错、结业礼上会偷偷抹泪。｜个人线剧情：非恋爱主线，可走敬重与被认可支线；揭示灯触起源的温和传说。｜与主角关系：引路人、结业礼主持、情感选择的旁证者。

**小铃（灯触灵物·人格化陪伴）**｜外貌：淡金半透明触须团，端部常卷成小灯盏形；情绪好时会泛粉光。｜性格：好奇、黏人、起哄高手（无恶意）。｜角色类型：吉祥物 / 气氛担当。｜萌点/魅力：把灯推到两人中间、点额提醒说谎、困了会缠在腕上当手镯。｜个人线剧情：象征「被温柔接住」；只作陪伴与氛围，不参与任何冲突性比试。｜与主角关系：全程陪伴的奇幻符号，情感催化剂。

**暮穗（年长灯触·静心随行）**｜外貌：光色偏琥珀，触须更长更稳，常托一盏旧铜灯。｜性格：沉稳、少起哄、在典礼时负责「定场」。｜角色类型：长辈灵物。｜萌点/魅力：会轻轻拍人后背示意「别慌」。｜个人线剧情：可讲述灯触起源的温和版本。｜与主角关系：典礼见证者。

**阿薰（雾津茶屋婆婆）**｜外貌：花白发髻，围裙印茶渍，笑时眼成缝。｜性格：热心、爱撮合年轻人。｜角色类型：小镇月老型配角。｜萌点/魅力：免费续热茶、塞糖。｜个人线剧情：提供约会舞台与「下山生活预演」。｜与主角关系：友善NPC。

**阿弥（雾津纸店少年）**｜外貌：浅褐短发、指尖常沾墨、总夹着一卷未裁的宣纸。｜性格：羞涩好学，看见僧衣会下意识鞠躬。｜角色类型：路人助力。｜萌点/魅力：偷偷在纸边画小灯触、结业季给候礼学徒送加厚宣纸。｜个人线剧情：市集支线跑腿与「第一封下山信」的邮差式存在。｜与主角关系：雾津向导、纸笔补给。
