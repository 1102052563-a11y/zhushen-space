# 魔法少女ミオ-空間歪曲

## 剧情

**【作品来源】**
《魔法少女ミオ-空間歪曲》（中文可称「魔法少女美绪·空间歪曲」）为**原创/情景向休闲世界**，非某一单一商业原作的直译改编；世界库条目以「魔法少女日常系 + 恋爱校园」气质为锚，参考日系魔法少女类型史中的日常系传统（变身与双重生活、吉祥物伙伴、校园友情与恋爱）而独立构筑。气质定位：**青涩纯爱 · 轻度奇幻日常 · 治愈偏甜，带一点秘密身份的轻紧张**。无正式单行本/动画出版纪录可查时，本档案以世界库命名与公开类型资料为据，**禁止把力量升级或战斗任务写成主轴**。卡片用途：恋爱/日常代入与可攻略角色情感线。

**【世界定位】**
一座靠海的日本地方都市「澄海市」里，高中二年级的**藤原美绪**在某一天获得了「能让身边一小片空间变得柔软、错位、缩短距离」的奇妙能力——当地人私下把它叫作「空间歪曲」。本世界不是英雄救世界的战场，而是：**把魔法藏进书包、把告白藏进雨天伞沿、把秘密身份与恋爱同时经营的日常恋爱舞台**。核心玩法＝与美绪及身边人物的 relationship + 季节节日与校园事件，而非任务厮杀。

**【世界观 · 舞台设定】**
澄海市是典型的温和现代日本地方都市：电车十分钟能到站前，骑车二十分钟能到海边，学校后山有神社与旧观景台。表面上一切按普通人的作息运转——上课铃、便利店关灯、周末祭典、期末考前夜的集体补习。异常只发生在「极小尺度」：教室门后多出半步的阴影、雨天两人之间伞的距离被悄悄拉近、走廊尽头的拐角似乎比昨天更短一点。这类现象被设定为**情感与心绪的外溢**，而不是用来清怪或攀比强弱的资源。

「空间歪曲」在本世界的日常侧含义是：当美绪强烈地想靠近、想躲开、想把某一瞬间延长或缩短时，身边会短暂出现「只对当事人可见的错位」。旁人通常只会觉得「今天走廊好像特别挤」「放学路怎么一下子就到站了」。真正知道秘密的人很少，保密本身构成情感张力：美绪害怕被当成怪人，更害怕自己在意的人被卷入解释不清的麻烦。

季节基调以**春末到初冬的一学年**为主轴。春：樱花与转校/社团招新；夏：海边与烟火大会；秋：文化祭与天台风；冬：圣诞灯与期末考。社会氛围安静、人情味足，老师会记住学生的绰号，便利店夜班大叔会记住常客的饭团口味。世界的温度来自：**有人愿意为你撑伞，有人愿意替你圆谎，有人愿意在你快要说破秘密时先把话题岔开**。奇幻元素只作「把心意可视化」的柔光滤镜，不写成战斗力榜单。

**【地理 · 生活舞台】**
- **澄海市立第二高校**：主舞台。三年级楼、二年级走廊、天台铁门、保健室、音乐教室、图书室深处靠窗的双人桌。初遇、误会、和好多半在这里完成。
- **二年级 A 班教室**：美绪的座位靠窗第三排；邻座与后排人物关系在此慢慢织密。雨天窗雾是名场面高发地。
- **天台**：铁丝网与储物间之间的夹缝位。午休带便当、文化祭排练偷闲、差点告白又咽回去的地方。
- **轻音部活动室 / 手工社角落**：美绪有时被拉去帮忙布景或借场地；社团不是战队据点，而是「能合法待到晚自习前」的避风港。
- **站前「月影咖啡」**：老板娘神崎结衣经营的二层小店，二楼靠窗是约会与谈心默认点。空间歪曲偶尔会让楼梯多出一级——结衣装作没看见。
- **澄海岸堤与旧观景台**：夏夜烟火、冬日风很大的告白备选地。观景台栏杆上有一代代学生刻的缩写，美绪从不刻，只摸着别人的字发呆。
- **藤原家（美绪住所）**：两层公寓，客厅总有未叠完的衣服。母亲藤原遥晚班回家时会留便当便签。秘密身份的压力在家门关上后才真正落地。
- **市立图书馆与旧书店「青页」**：资料搜集与偶遇男主/女配的软场景；书架尽头有时会「多出」一条只通向某人心情的小径——叙事上作比喻与轻度超自然，不作迷宫副本。
- **夏日祭会场（神社参道）**：浴衣、捞金鱼、烟火。情感事件密度最高的一夜。

**【故事主线 · 情感线】**
本世界主线按「学年情感弧」推进，不以打怪章节划分。

**① 相遇 · 秘密的第一天（春）**  
藤原美绪在放学后的空教室里第一次触发空间歪曲：她想逃开班上关于「喜欢的人」的起哄，结果发现自己与教室门之间的距离被折叠，一步就到了走廊。随后出现的是只会说人话的毛绒感小生物**珀尔（Perle）**——外形像团棉花糖狐狸，自称是「把歪曲缝合回去的缝线精灵」。珀尔不教战斗，只唠叨：别在人多的地方乱折空间，别把别人的秘密折进自己口袋。  
同一时期，转学生**高城莲**从大城市转到澄海二高二年级 A 班，坐在美绪斜后方。他观察力极强，第一次就注意到美绪「进教室的方式不太对」。美绪的青梅竹马**三浦葵**则用玩笑把话题盖过去。情感线起点：美绪对「被看穿」又怕又隐隐期待。

**② 日常相处 · 双面生活（春末—初夏）**  
美绪白天是成绩中游、体育一般、会帮老师搬资料的普通女生；傍晚在珀尔监督下练习「把歪曲收得更小」。她用能力做的事全是小事：帮赶电车的同学把到站口的距离缩短半分钟；在雨天把伞下空间撑大一点让更多人挤进来；把弄丢的钥匙「折」回原处附近。  
高城莲开始在图书室与美绪同桌自习，借口是「本地人带路找资料」。三浦葵吃醋式地插队。同班的**白石凪**（冷静文艺委员）默默记下美绪异常的时间点，却选择不揭穿，只在美绪快撑不住时把保健室钥匙塞给她。  
此阶段情感升温靠细节：共用耳机、替对方挡起哄、在便利店门口分吃最后一个关东煮。美绪发现自己对高城莲的在意已经超过「怕被发现」。

**③ 升温 · 夏夜与第一次几乎说破（夏）**  
夏日祭。美绪穿母亲旧浴衣，发带歪了；高城莲伸手帮她扶正，空间歪曲在那一瞬间把两人间的半步距离抹掉，额头差点相撞。烟火升起时，美绪差点说出「其实我能……」，被珀尔在袖口里轻轻咬了一下。  
同晚，美绪看见**神崎结衣**（月影咖啡老板娘，二十多岁）在祭典后收拾摊子时对某个打电话的人说「我在这边很好」——暗示结衣也有自己的「离开原生活」的故事，与美绪的秘密形成镜像：都是把真心藏在日常岗位里。  
三浦葵在祭典后坦白：他从小就喜欢美绪，但更怕失去「能随便互骂的关系」。美绪无法立刻回答，空间在她周围轻微颤动，像心情本身在发抖。

**④ 冲突 · 误会与保密的代价（秋）**  
文化祭前，班上需要密室逃脱风格的游园设施。美绪为了赶工，过度使用歪曲把布景房间「多挤出」一个储物角，结果被摄影部拍到异常片段。流言变成「藤原好像会什么超能力」。高城莲为了帮她圆谎，主动把异常揽到自己身上（「是我剪辑搞砸了」），两人第一次真正并肩对外部压力。  
三浦葵却以为高城莲在利用美绪的秘密接近她，两人冷战。白石凪把关键U盘藏起，逼三人在图书室谈开：不是谁对谁错，而是**秘密会改写信任的形状**。  
美绪崩溃点：她发现自己用歪曲「缩短」了太多与高城莲相处的路，却拉长了与葵之间的沉默。她第一次主动向葵道歉，并向高城莲承认「我有不能说的事，但不是不信任你」。高城莲的回应是：「那就等你想说的时候。我可以先当你普通的邻座。」这句话成为本世界最重要的情感锚。

**⑤ 和解 · 告白与选择（秋末—冬）**  
文化祭当天，美绪负责的教室剧《会缩短的走廊》意外成为人气节目——剧本其实就是她自己的隐喻。终场谢幕时她在掌声里看清：能力可以藏，心意不必永远藏。  
冬日初雪，旧观景台。美绪对高城莲说出半句真相：「有时候，我能让世界对我好一点。但我想对你好的时候，反而最不敢用。」高城莲握住她的手：「那就用普通的方式。走路远一点也没关系。」  
若走**高城莲线 HE**：两人确认交往，美绪仍保守能力细节，但分享「我有时会害怕自己」的脆弱；珀尔成为只有两人知道的第三者吐槽役；葵以损友身份留下，学会祝福。  
若走**三浦葵线 HE**：美绪意识到青梅竹马的安心感不可替代，在河堤坦白「我想继续和你并排走，不是因为习惯，是因为选择」；高城莲优雅退为可靠朋友，偶尔用观察力帮她挡流言。  
若走**白石凪线（偏百合/深交 HE）**：凪在保健室摊牌「我整理你异常记录不是要揭穿你，是想在你倒下前接住你」；关系从守护走向对等依恋；恋爱节奏更慢、更安静。  
**共通结局方向**：能力不消失，但美绪学会「少用、慎用、为谁而用」；真正解决危机的从来不是歪曲，而是沟通、道歉与陪伴。无世界末日，无最终魔王战——最终Boss是**期末考与如何介绍对方给家长**。

**⑥ 真结局倾向（多线交汇后的日常）**  
当美绪对至少一人完全坦白能力后，空间歪曲会「稳定」为更温和的现象：只在强烈共感时轻微发生。珀尔说这叫「心缝合好了，线就不用绷那么紧」。世界进入后日谈式日常：一起打工、一起跨年、一起把去年文化祭的布景板烧掉。情感主题落点：**喜欢一个人，不必先变成完美的魔法少女；先做愿意迟到也要等的人。**

**【可攻略角色 / 主要人物】**

**藤原美绪（主角 / 澄海二高二年级 A 班）**｜外貌：栗色中长发，常扎低马尾；瞳色暖棕；校服外套袖口爱藏手；左腕有一条磨损的红绳手链（母亲所赠）。｜性格：外柔内倔，怕麻烦却总先帮人；秘密让她敏感，也让她更会察言观色。｜角色类型：元气偏内向的「秘密系女主」。｜萌点/魅力：认真道歉的样子、雨天把伞倾向别人、说半截话就脸红。｜个人线剧情：从「用能力回避关系」学到「用语言承担关系」；心结是「被当作怪人」与「伤害青梅」。｜与契约者关系：若契约者以转校生/邻座身份进入，她会先防备后依赖，好感来自「不追问却在场」。

**高城莲（可攻略 / 转学生 / 二年级 A 班）**｜外貌：黑发微卷，眼神冷静；身高偏高；冬日围巾永远叠得很整齐。｜性格：观察型温柔，话少但准；不喜欢起哄文化，却会为在意的人主动背锅。｜角色类型：高岭系沉稳男主 / 可靠邻座。｜萌点/魅力：替人圆谎时的平静、共用耳机时先问曲目、记住美绪不吃香菜。｜个人线：学会从「分析异常」转向「尊重边界」；结局与美绪交往后仍不逼问能力全貌。｜与主角关系：斜后方座位起步，图书室同桌升温。

**三浦葵（可攻略 / 青梅竹马 / 同班）**｜外貌：浅茶短发，笑起来有虎牙；运动外套常搭在肩上。｜性格：吵闹护短，嘴硬心软；吃醋时会更吵。｜角色类型：元气青梅 / 损友型男主。｜萌点/魅力：吵架后塞一瓶热可可、运动会为谁加油喊到嗓子哑。｜个人线：从「默认你会选我」到「认真追求你」；BE 风险是把秘密当把柄开玩笑。｜与主角关系：小学起的上下学同伴。

**白石凪（可攻略 / 文艺委员 / 同班）**｜外貌：黑长直，眼镜，制服永远熨帖；说话轻。｜性格：理性温柔，行动派守护；不擅长直球表白。｜角色类型：文静知性 / 隐性攻略角色。｜萌点/魅力：把创可贴和薄荷糖分装小袋塞进美绪笔袋、用书页夹留言。｜个人线：从记录异常到选择共守秘密；HE 是并肩坐在保健室窗边承认「我想成为你的普通日常」。｜与主角关系：先是「知道太多的人」，后是「可以沉默相对的人」。

**神崎结衣（可攻略向重要角色 / 月影咖啡老板娘 · 约26岁）**｜外貌：亚麻色披肩发，围裙上永远有咖啡渍；笑时眼下有细纹。｜性格： intergenerational 的成熟包容，嘴上嫌弃学生占座，手上下双倍奶油。｜角色类型：年长治愈 / 姐姐系。｜萌点/魅力：打烊后请喝热牛奶、听完秘密只说「明天还开门」。｜个人线：不推动校园恋爱，提供「成人侧的倾听」；若攻略则节奏偏慢热同居感日常（打工、关店、雨夜留宿沙发——点到氛围）。｜与主角关系：美绪的精神避难所与兼职雇主。

**珀尔（非攻略吉祥物 / 缝线精灵）**｜外貌：巴掌大的乳白狐狸团，耳朵尖粉。｜性格：毒舌管家，讲规矩，关键时刻心软。｜角色类型：吐槽役 + 保密 debuff 解除键。｜萌点：躲在领口骂人、吃布丁时不说话。｜剧情功能：限制美绪滥用能力，推动「用沟通代替折空间」。｜与主角关系：契约式伙伴，不是战斗教官。

**藤原遥（配角 / 美绪母亲）**｜外貌：短发利落，护士鞋声音很响。｜性格：忙但细，便签作家。｜人物作用：家庭线温度；不知女儿能力，却直觉女儿在恋爱。｜与主角关系：晚归的拥抱与「喜欢就带来家里吃饭」。

**星野优马（配角 / 摄影部 / 同级）**｜外貌：棒球帽，相机不离身。｜性格：八卦但不坏，被白石凪压着删素材。｜作用：误会的导火索与后来的「证据销毁同盟」。

**川上芽衣（配角 / 美绪闺蜜 / 邻班）**｜外貌：双马尾，发卡多。｜性格：恋爱雷达灵敏，起哄专业户。｜作用：把美绪推向选择，也在她难过时塞巧克力。

**森下诚（配角 / 班主任）**｜外貌：瘦高，眼镜反光。｜性格：表面严格，知道学生秘密时选择装睡。｜作用：校园秩序的柔软边界。

**【人际关系网 / 社团势力】**
- **核心三角**：藤原美绪—高城莲—三浦葵。葵拥有过去，莲拥有现在的并肩；美绪的选择决定三角是「恋爱+损友」还是「恋爱+优雅友人」。
- **守护斜线**：白石凪单方面守护美绪 → 可转为对等。星野优马制造危机 → 被凪与莲共同收拾。
- **成人侧支持**：神崎结衣对美绪；藤原遥对家庭；森下诚对班级舆论降温。
- **闺蜜线**：川上芽衣拉美绪参加联谊又帮她逃联谊，是喜剧缓冲。
- **「社团」不是战队**：轻音部/手工社/摄影部/文艺委员会构成日常事件池；月影咖啡是跨年级社交点。没有魔法少女总部，没有敌人组织——最大的「势力」是**班级起哄文化**与**期末考压力**。

**【情感事件 · 名场面】**
1. **空教室第一步**：美绪第一次折空间逃开起哄，与高城莲的视线在走廊相撞。  
2. **雨天伞沿**：伞下空间被不自觉撑大，三个人挤成两个半——葵吃醋名场面。  
3. **夏祭扶发带**：额头差点相撞，烟火声盖过心跳。  
4. **摄影部U盘危机**：莲背锅，美绪第一次意识到「有人为你撒谎」。  
5. **图书室三人对质**：凪逼谈开，秘密从「一个人的」变成「关系的」。  
6. **文化祭话剧隐喻**：观众笑，美绪哭笑不得地鞠躬。  
7. **初雪观景台半告白**：「想对你好的时候最不敢用能力」。
