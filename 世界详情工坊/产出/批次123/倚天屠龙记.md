# 倚天屠龙记

## 剧情

**【作品来源】**
金庸（查良镛）著长篇武侠小说，1961—1962年连载于香港《明报》，后收入「射雕三部曲」终章（前接《射雕英雄传》《神雕侠侣》）。定本四十回，回目以柏梁体诗联成；1976年修订、2003年新修。新修版约一百一十七万字。气质：元末江湖恩怨与反元大义交织，正邪难辨、情义缠绵、权欲与屠龙号令的讽刺。跨媒介：多版影视（邵氏电影、港台陆三地电视剧）、游戏、漫画；本档案以小说定本/新修公开情节为准。

**【世界定位】**
元末中原武林：明教（被正道目为魔教）与六大门派、丐帮、朝廷汝阳王府多方博弈；屠龙刀、倚天剑牵引「武林至尊，号令天下」。主角张无忌由冰火岛孤儿成长为明教第三十四代教主，终卸任与赵敏归隐蒙古。契约者切入的是冷兵器江湖、门派倾轧与起义前夜，非仙侠飞升。

**【世界观 · 力量体系】**
力量本源为内力、外家拳脚、兵器与轻功；无灵气境界飞升，靠秘笈、师承、苦修与奇遇。死亡即死，无复活系统；残废（如俞岱岩经脉尽断）、失明（谢逊、成昆）、走火入魔（韦一笑寒冰绵掌）为常态代价。医毒双绝（胡青牛、王难姑）可延命，不能起死。

武学层级（以破坏力/江湖地位表现，宁低勿高）：
1. **江湖走卒 / 镖师 / 杂役**：刀剑对拼、破门伤人。≈一阶初。
2. **门派弟子 / 一流好手**：碎砖裂石、单挑小队、点穴擒拿。≈一阶中后期。
3. **一流高手 / 长老级**：裂石断金、乱军取首、内力远程伤人（气劲、指力）。≈二阶。
4. **绝顶高手 / 掌门 / 法王级**：一掌重创名宿、狮吼震乱心神、轻功瞬移取敌；能左右一城武林格局。≈二阶巅峰~三阶初。
5. **当世巅峰**：张三丰太极拳剑、张无忌九阳+乾坤大挪移+太极、少林渡难/渡劫/渡厄金刚伏魔圈——可压服六大派掌门联手，左右「屠狮大会」级场面。≈三阶。
6. **神功/秘笈维度**（非境界，是战力乘数）：九阳神功（纯阳内力、疗伤驱毒）、乾坤大挪移（挪移借力、激发潜力）、九阴真经（含九阴白骨爪等）、七伤拳（伤敌先伤己）、圣火令上古波斯武学、玄冥神掌（阴寒入骨）、倚天剑/屠龙刀（锋锐与号令象征）。

乐园阶位映射：江湖弟子~一流好手≈一阶；一流高手~门派长老/法王≈二阶；当世巅峰（张三丰、巅峰张无忌、少林渡字辈联手级）≈三阶。世界顶点=张三丰（德望与修为当世第一）及「刀剑秘藏+义军」的格局级变数；低阶切入一律规避巅峰对决·情报优先。本世界无行星爆星级，封顶三阶。

**【地理 · 舞台】**
- **中原核心**：武当山（武当派）、少室山少林寺（金刚伏魔圈、屠狮大会）、峨眉山（峨眉派）、昆仑山明教总坛光明顶、蝴蝶谷（胡青牛医仙）、江南龙门镖局/王盘山岛（屠龙刀英雄宴）、大都万安寺（六大派被囚）、濠州（张无忌周芷若婚礼变故）。
- **边陲与海外**：冰火岛（谢逊隐居、无忌出生）、灵蛇岛（黛绮丝/金花婆婆）、波斯明教总教（圣火令、十二宝树王、风云月三使）。
- **政治轴**：元大都、汝阳王府（察罕帖木儿/赵敏）、义军前线（徐达、常遇春、朱元璋、陈友谅、韩林儿）。
- **前史舞台（序章）**：少林寺、襄阳附近武当山、华山论剑后的南宋遗脉（郭襄、张君宝）。

势力地盘：正道六大派（少林、武当、峨眉、昆仑、华山、崆峒）占名山；明教占光明顶与五行旗、天鹰教（殷天正自立支派后回归）；丐帮游走江湖；蒙古朝廷以兵权与番僧高手压武林。

**【世界剧情线】**

① **序章·九阳分流与武当起源（南宋末~元初）**  
宋理宗景定元年后，郭襄寻杨过、小龙女不遇，少室山遇昆仑三圣何足道挑战少林。觉远与弟子张君宝以九阳神功震退何足道，却因无师自学犯寺规被追杀；郭襄偕逃。觉远坐化前背诵《九阳真经》，无色、郭襄、张君宝各记一段，后分衍为少林九阳、峨眉九阳、武当九阳根基。张君宝入武当山隐修，改名张三丰，创内家拳，武当与少林并称。江湖流传：「武林至尊，宝刀屠龙。号令天下，莫敢不从。倚天不出，谁与争锋。」实则刀剑内藏《九阴真经》《武穆遗书》等（新修版为玄铁片指向桃花岛取书），铸自郭靖、黄蓉熔杨过玄铁重剑与君子/淑女剑，寄望后人复国。

② **屠龙祸起·王盘山与冰火岛十年（元顺帝至元二年起）**  
俞岱岩得屠龙刀，遭暗算瘫痪；张翠山查案识殷素素，知龙门镖局灭门系殷素素所为。王盘山英雄宴上谢逊夺刀，胁张翠山、殷素素出海，三人困冰火岛十年：翠山素素成婚，生子张无忌，谢逊收为义子。十年后一家返中原，群雄逼问谢逊与刀下落；张三丰百岁寿宴上，翠山因护义兄、又知妻伤三师兄俞岱岩，羞愤自刎，殷素素殉夫。无忌中玄冥二老玄冥神掌，寒毒入骨。

③ **蝴蝶谷·杨不悔·九阳奇遇**  
张三丰携无忌求医少林被拒；途救常遇春与周芷若，常遇春荐蝶谷医仙胡青牛。无忌学医却难根除寒毒。纪晓芙因杨逍生女杨不悔，拒杀杨逍为灭绝师太所杀，托无忌送女。无忌送杨不悔至昆仑明教，险遭何太冲夫妇与朱长龄、朱九真骗取谢逊下落；坠崖入谷，于猴腹得完整《九阳真经》，数年练成，寒毒尽去，内力冠绝同辈。

④ **光明顶之役·教主之位**  
成昆挑拨六大派围攻光明顶。无忌化名曾阿牛介入，以九阳与乾坤大挪移（于秘道得阳顶天遗下心法，数时辰练至第六层）力挽狂澜，与周芷若、武当诸侠、外公殷天正相认，被推为明教第三十四代教主。灭顶危机暂解，无忌名震天下。

⑤ **万安寺·赵敏·刀剑再失**  
六大派回程遭元廷伏击囚于大都万安寺，倚天剑入赵敏之手。无忌率明教高手救出六大派；武当危局中张三丰授太极拳剑。无忌应赵敏「借看屠龙刀」之约寻义父，辗转灵蛇岛：金花婆婆实为紫衫龙王黛绮丝，欲借刀雪灭绝剑下之耻。波斯总教使者至，小昭为救众人被迫继任波斯教主，东归永隔。

⑥ **荒岛刀剑互斫·周芷若黑化**  
避中原群雄时流落荒岛。周芷若为完成灭绝遗命，迷倒众人，盗刀剑互斫得秘笈，练九阴白骨爪与白蟒鞭法，并杀/重创殷离、嫁祸赵敏。无忌与赵敏感情升温，赵敏叛出王府追随。

⑦ **屠狮大会·成昆败露·黄衫女**  
成昆擒谢逊囚少林作饵，唆空闻方丈开屠狮英雄大会。张无忌、赵敏、杨逍、黄衫女子（杨过后人）等破局；谢逊于金刚伏魔圈井中废成昆武功、剜其双目。周芷若劣行被黄衫女以正宗九阴「摧坚神爪」揭破。无忌终明刀剑之秘：复国兵书与武学，非单纯「号令」神器。

⑧ **义军与卸任·终局**  
无忌以《武穆遗书》授徐达，义军连败元兵、攻陷大都。朱元璋渐夺兵权；无忌卸教主予杨逍，与赵敏归隐蒙古。明教兵权终入朱元璋之手，驱元建明。张无忌性格「只记人好」、优柔寡断贯穿始终——武功当世仅次张三丰，却极少下杀手，感情在赵敏、周芷若、小昭、殷离间被局势推着走，终局选定赵敏。

支线要节点：宋青书窥寝杀莫声谷叛武当、为俞莲舟所废；陈友谅与成昆合谋害丐帮史火龙；范遥自毁容貌潜入汝阳王府为苦头陀；阳顶天练乾坤大挪移见妻与成昆私通走火入魔而死——明教衰微根源。

**【主要人物】**

- **张无忌（男主·明教第三十四代教主）**｜性格：宅心仁厚、记恩忘怨、优柔寡断，非生死关头不下重手。｜装备·能力：九阳神功、乾坤大挪移（至第六层）、武当太极拳/太极剑、圣火令武功；医术得胡青牛真传。武功当世除张三丰外难逢敌手。｜人物弧光：冰火岛孩童→寒毒少年→光明顶救世主→教主→卸任归隐。｜立场关系：谢逊义子；赵敏终局道侣；与周芷若有婚约却逃婚；对小昭、殷离皆有情义；对外公殷天正、杨逍诸人亦师亦友。

- **赵敏（敏敏特穆尔·绍敏郡主）**｜性格：机智果决、敢爱敢恨、能文能武。｜装备·能力：家学与范遥所授武学，善用计、指挥番僧与玄冥二老。｜人物弧光：剿灭反贼的郡主→万安寺主谋→为爱叛父→归隐蒙古。｜立场关系：察罕帖木儿之女、王保保之妹；与周芷若情敌；终局与无忌同行。

- **周芷若（峨眉第四代掌门）**｜性格：初清丽温婉，后为师命与情伤转冷酷阴鸷，终渐悔。｜装备·能力：峨眉九阳功、倚天剑（阶段性）、九阴白骨爪、白蟒鞭法。｜人物弧光：汉水孤女→灭绝亲传→掌门→荒岛夺秘→婚礼被弃→劣行败露。｜立场关系：无忌青梅式情愫与婚约；视赵敏为死敌。

- **谢逊（金毛狮王）**｜性格：惨痛后疯厉，冰火岛后因无忌而人性回温，终皈依。｜装备·能力：七伤拳、狮子吼、混元一气功、屠龙刀（长期持有）。｜人物弧光：成昆害死全家→滥杀逼师现身→夺刀漂流→复仇剜成昆双目→出家。｜立场关系：无忌义父；与张翠山义兄弟；成昆死仇。

- **张三丰（武当开派·张君宝）**｜性格：冲虚圆通、护短弟子、武德极高。｜装备·能力：当世第一；纯阳无极功、太极拳剑、倚天屠龙笔法（创授翠山）等。｜人物弧光：少林叛出→创武当→百岁寿宴丧爱徒→授无忌太极。｜立场关系：视七侠如子；与郭襄有旧；拒少林门户之见却终身不得入寺。

- **殷素素 / 张翠山**｜性格：素素美艳果决（天鹰教手段）、翠山儒雅重诺。｜装备·能力：素素毒针暗器；翠山银钩铁划、倚天屠龙笔法。｜人物弧光：夺刀结怨→冰火岛成家→寿宴双双自尽。｜立场关系：无忌父母；谢逊羁绊核心。

- **成昆（圆真·火工头陀）**｜性格：阴毒善谋、以正派面目祸乱武林。｜装备·能力：幻阴指、混元功、霹雳拳等；少林空字辈级阴谋家。｜人物弧光：私通阳顶天夫人→灭谢逊全家→挑六大派与明教→屠狮会败露→被废。｜立场关系：谢逊师；陈友谅师；全书幕后黑手。

- **杨逍 / 范遥（逍遥二仙）**｜性格：杨逍孤傲智计；范遥深情隐忍（自毁容为苦头陀）。｜装备·能力：乾坤大挪移前两层（杨）、弹指神通等；范遥博通武学。｜人物弧光：明教栋梁→无忌卸任后杨继教主大位却难阻朱元璋。｜立场关系：纪晓芙/杨不悔；范遥曾恋黛绮丝。

- **灭绝师太**｜性格：刚愎嗜杀、正邪二分到病态。｜装备·能力：倚天剑、灭剑绝剑、峨眉九阳、佛光普照。｜人物弧光：寻屠龙刀→围光明顶→万安寺拒救摔死。｜立场关系：周芷若师；杀纪晓芙；恨明教入骨。

- **黛绮丝（紫衫龙王/金花婆婆） / 小昭 / 殷离（蛛儿）**｜黛绮丝：波斯圣女叛教、灵蛇岛主；小昭：无忌丫鬟→波斯教主；殷离：无忌表妹、千蛛万毒手、曾立誓为妻。三人分别代表异域、牺牲、初恋式执念。

- **其他关键**：殷天正（白眉鹰王）、韦一笑（青翼蝠王）、胡青牛与王难姑、常遇春、徐达、朱元璋、陈友谅、宋青书、俞莲舟、殷梨亭与杨不悔、黄衫女子、玄冥二老（鹿杖客、鹤笔翁）、何太冲夫妇、朱长龄朱九真、阳顶天。

**【势力图谱】**
- **明教**：反元、光明顶总坛；教主—左右光明使—四大法王—五散人—五行旗；支派天鹰教（殷天正）；海外波斯总教（圣火令、教规严酷）。与六大派世仇，后因无忌调解暂缓。
- **武当派**：张三丰—七侠—宋青书等；内家宗门，偏正却护明教后人无忌。
- **峨眉派**：郭襄所创；灭绝—周芷若；持倚天剑，灭明教执念最强。
- **少林派**：空闻/空智/空性等；渡难渡劫渡厄隐世；门户之见深，亦被成昆利用。
- **昆仑/华山/崆峒**：六大派成员；何太冲、崆峒七伤拳谱等。
- **丐帮**：史火龙死后陈友谅篡权线；与成昆勾结。
- **汝阳王府/元廷**：察罕帖木儿、赵敏、番僧、金刚门阿三等；以武力与阴谋控武林。
- **义军**：韩林儿、徐达、常遇春、朱元璋——从明教战友到夺权建明。

**【贵重物品】**
- **屠龙刀 / 倚天剑**：郭靖黄蓉所铸；内藏武穆遗书、九阴真经等（或玄铁片指引）；「号令天下」实为兵书与武学，非神谕。
- **《九阳真经》**：写于《楞伽经》夹缝；完整版在昆仑谷猴腹，无忌练成。
- **《乾坤大挪移》**：明教镇教神功，藏光明顶秘道；无忌速成，阳顶天死于第四层走火。
- **《九阴真经》**：刀剑中秘；周芷若速成白骨爪，黄衫女正宗路子克之。
- **《武穆遗书》**：兵法；无忌授徐达，助北伐。
- **《七伤拳谱》**：崆峒镇派；谢逊强练致性格愈厉。
- **圣火令**：波斯古武记载；无忌临阵领悟破风云月三使。
- **玄冥神掌 / 寒冰绵掌 / 千蛛万毒手**：阴毒奇功，分别标玄冥二老、韦一笑、殷离的代价。
- **黑玉断续膏**等灵药：治俞岱岩等重伤的关键消耗品。

**【隐藏剧情 · 伏笔】**
- 刀剑秘密直至后段才揭：争夺「号令」者多死于贪欲，兵书才是复国关键。
- 成昆一条线串联阳顶天之死、谢逊灭门、六大派围攻、丐帮内乱、屠狮大会。
- 九阳三分（少林/峨眉/武当）与完整九阳对比：完整版成就无忌，分传成就三派根基。
- 射雕互文：郭襄—张三丰、黄衫女「终南山后，活死人墓，神雕侠侣，绝迹江湖」、玄铁剑熔铸。
- 新修版后记：金庸暗示无忌若不同选择可能周芷若/小昭/殷离线；强调「人为」与「只记好处」。
- 朱元璋夺权：正史投影，明教「胜利」后政治清洗的伏笔。

**【大事记时间线】**

## 阶位切入点

> 阶位↔武学：一阶≈江湖弟子~一流好手；二阶≈一流高手~法王/掌门级；三阶≈当世巅峰（张三丰、巅峰无忌、渡字辈联手级）。世界顶点（张三丰与「义军+刀剑之秘」格局）低阶一律规避硬刚·情报优先。

**一阶（江湖走卒·镖局/门下弟子线）**

切入身份/时点：契约者以龙门镖局护镖伙计、天鹰教外围香主、或武当山杂役道童身份切入，锚定俞岱岩得屠龙刀前后至王盘山英雄宴前夜。

初始事件：你押送一位「重伤贵客」（实为俞岱岩线索相关）夜行江南官道，前哨火把尽灭——紫衣蒙面人（殷素素一线）留下蚊须针与挑拨少林、武当的伪证；你必须在「报官/报武当/吞下线索卖钱」之间抉择，而王盘山英雄宴的请柬已在茶馆传开。

开场白建议：「浙东的雨把镖旗打湿了。车上那人喘得像拉风箱，路边茶棚却有人低声念：武林至尊，宝刀屠龙。你摸了摸腰间半吊钱——今晚若走错一步，镖灭人亡，名字都不会上话本。」

关键NPC立场：**殷素素**——为刀不择手段，可利用你当烟雾，也可灭口；**张翠山**——查案儒侠，重诺，对滥杀反感却渐迷殷素素；**谢逊**——尚未现身，但金毛与狮吼的传闻已让海沙派、巨鲸帮胆寒；**白龟寿**——玄武坛主，王盘山幸存者线索源；**都大锦**——龙门镖局相关，怕事爱财。

主线钩子/支线：王盘山英雄宴观礼或杂役；龙门镖局灭门真相；屠龙刀转运线。支线：江南假屠龙刀骗局；少林与武当为灭门案结怨的传话；冰火岛出海前的船引。

危险度/规避：中——一流高手随手可杀你；须规避谢逊现身时的狮吼范围、殷素素灭口、成昆早期暗子。

任务方向/奖励：活着离开江南、换到入门级内功或名帖；奖励取向＝基础内功秘本、解毒丸、门派记名弟子身份、屠龙刀假货情报（真刀勿碰）。

**二阶（一流高手·光明顶前后线）**

切入身份/时点：以明教五行旗副使、六大派真传、或汝阳王府挂名护卫身份切入，锚定六大派围攻光明顶前后至万安寺囚俘。

初始事件：你被编入攻顶左翼/守殿右翼，中军传令「成昆密信：今日必屠魔」；火光中少年「曾阿牛」（张无忌）以空前内力连破掌门，你的旗帜倒向决定：帮明教突围，还是趁乱夺一份「乾坤残页」谣言中的好处。

开场白建议：「光明顶的风里全是烧焦的经幡。六大派的名号在山道上排成诛魔的墙，可你看见那少年一掌卸开灭绝师太的剑锋时，才明白——有人把『正邪』写成了笑话，而笑话要拿命来听。」

关键NPC立场：**张无忌**——愿劝和不愿滥杀，可结盟；**灭绝师太**——杀魔执念，视外人助力为污；**杨逍**——孤傲，认实力不认情面；**赵敏**——布局囚六大派，视你为棋子或弃子；**韦一笑**——寒毒发作需血，危险不可控；**周芷若**——刺无忌一剑后情愫暗生，立场摇摆。

主线钩子/支线：光明顶劝和或火上浇油；万安寺劫囚；倚天剑易主。支线：蝴蝶谷医毒交易；成昆身份排查；波斯使者前哨。

危险度/规避：高——掌门/法王交手余波可致死；须规避玄冥神掌、灭绝全力一剑、成昆幻阴指。

任务方向/奖励：在正邪之间换取教职或郡主府通行证；奖励取向＝二流~一流兵刃、九阳残章线索、明教香主/门派长老席。

**三阶（绝顶边缘·屠狮与终局线）**

切入身份/时点：以成名大侠、义军军师、或波斯归来的暗桩身份切入，锚定屠狮大会至无忌授《武穆遗书》、卸任前后。

初始事件：少室山屠狮大会擂台下，谢逊被囚井中，成昆在袈裟后发笑；你收到三封信——朱元璋邀「共图大业」、赵敏换「义父性命」、周芷若以峨眉名义邀「共诛邪教」。你的站队将影响刀剑之秘是否早于原作揭开。

开场白建议：「少室山的钟声被万众的『杀』字淹没。屠龙刀的传说在今日要用人血写完，而你手里那卷旧纸，墨迹未干：武穆遗书——有人要它复国，有人要它称帝，有人要它只换一人的自由。」

关键NPC立场：**张无忌**——已倦权位，可托付兵书或被夺权；**谢逊**——复仇后求佛，不欲再起杀业；**成昆**——垂死反扑；**黄衫女子**——持正宗九阴，专破邪路；**朱元璋**——枭雄，胜利后必清洗；**徐达**——良将，得书则战局改写；**张三丰**——德望顶点，不轻易下场但一言九鼎。

主线钩子/支线：屠狮破局；刀剑之秘公开与否；授书北伐；无忌卸任。支线：宋青书清理门户余波；波斯圣火令残党；赵敏归蒙古护送。

危险度/规避：极高——贴近顶点；须规避张三丰本体对决、渡字辈金刚伏魔圈、义军与朝廷野战争锋；**超阶不适用，本世界封顶三阶**；对张三丰、巅峰无忌仅可情报/条件性协作，严禁硬刚。

任务方向/奖励：左右义军与武林话语权；奖励取向＝武穆遗书抄本（残）、九阴/九阳对照笔记、义军兵符、归隐或入朝的二选一结局资源。
