# 连城诀
<!--meta lib=主库 tiers=一、二、三、四-->

## 剧情

**【作品来源】**
金庸（查良镛）武侠中篇，原名《素心剑》，1963年连载于香港《东南亚周刊》，后改题《连城诀》以突出“连城宝藏/剑诀”主线，收于《金庸作品集》，约20余万字，已完结。全书十二章加后记：乡下人进城、牢狱、人淡如菊、空心菜、老鼠汤、血刀老祖、落花流水、羽衣、「梁山伯·祝英台」、「唐诗选辑」、砌墙、大宝藏。无明确纪年，名物与官制暗示清代江湖。主题为人性之贪、背叛与伪善，金庸自言“对坏人描写得最淋漓尽致”，文风冷峻压抑，少英雄救国、多冤狱与自噬。修订中血刀门由“西藏”改为“青海黑教”，雪山由“藏边”改为“川边”，并补梅念笙与《鹿鼎记》吴六奇同派等互文。**以小说正文为准**。全本可核：[搜笔趣阁·金庸作品集（含连城诀）](https://www.sobqg.com/p/1523230.html)；对照 [中文维基·连城诀](https://zh.wikipedia.org/wiki/%E8%BF%9E%E5%9F%8E%E8%AF%80)、[百度百科·连城诀](https://baike.baidu.com/item/%E8%BF%9E%E5%9F%8E%E8%AF%80/11557)。

**【世界定位】**
一座被“连城宝藏”毒化的江湖：表面正邪分明，实则正派名宿为图梁元帝藏宝图自相残杀。主角**狄云**从湘西乡下“空心菜”到狱中冤囚、雪山亡命、最终看破人性，携戚芳遗孤归隐雪谷。一句话：冤狱、背叛、贪念即毒药，素心与宝藏不可兼得。

**【世界观 · 力量体系】**
本世界无仙道飞升、无系统面板。力量来源为**内功真气 + 剑刀枪拳招式 + 毒功/暗器/铁甲**。生死残酷：琵琶骨锁死、断指、缢死、毒杀、雪崩与饥寒皆可夺命；唯一近似“起死回生”的手段是**神照功**极境内力强行续命，仍须当场及时、代价极大。无复活规则，重伤靠内功、医术或乌蚕衣等物。官刑与江湖武功同样能废人：江陵大牢的琵琶骨与断指，是“公权力”对武者的降维打击。

江湖战力按表现归纳（原作无官方境界名）：
1. **乡野新人 / 狱卒捕快 / 普通门人**：会几路躺尸剑法或家传拳脚，斗殴护院。破坏力≈徒手制民、冷兵器格斗。≈一阶。
2. **门派精英 / 绿林好手**：万震山门下弟子（鲁坤、周圻等）、血刀门弟子**宝象**、太行吕通吕威、耿天霸、马大鸣等。能单挑小队、破门夺命、用毒设局。≈二阶。
3. **一流高手 / 名宿掌门级**：**万震山**（五云手）、**言达平**（陆地神龙）、**戚长发**（铁锁横江）、**南四奇**单人（陆天抒、刘乘风、水岱、花铁干）、**凌退思**（知府兼帮会头目）、铃剑双侠中的水岱级。裂石断金、乱军取首、一招定生死。≈三阶。
4. **当世顶点近峰**：**梅念笙**（铁骨墨萼，未尽传三徒前）、**丁典**（神照功大成）、**血刀老祖**（血刀经内功刀法）、终局**狄云**（神照功圆满+血刀心法）。人间武学极限：一击毙血刀老祖、狱中独步、雪山以一敌众；未达毁城夷地。≈三阶后期～**四阶**。

关键功法逻辑：
- **神照经 / 神照功**：梅念笙所持至高内功，口诀“练成神照功，天下无对手”。可疗重伤、续断气、通筋脉；丁典狱中大成救活缢死狄云；狄云雪山被扼颈反通最后关窍，一掌毙血刀老祖。内功纯度与续命能力是本书战力天花板的真正标尺。
- **连城剑法 / 唐诗剑法**：梅念笙成名剑学，招式以唐诗命名（如“孤鸿海上来”“落日照大旗”等意象），剑谱与宝藏密码一体。三徒未得完整真传；戚长发故意教错为**躺尸剑法**（“哥翁喊上来”“落泥招大姐”等谐音恶搞），华而不实。言达平曾以三招真诀点破伪剑。
- **血刀经 / 血刀刀法**：青海黑教（修订前西藏血刀门）镇派绝学，内功+怪异刀法，每招从不可能方位劈砍。血刀老祖第四代掌教；狄云因误穿宝象僧袍、雪谷机缘兼习心法。
- **中平枪法**：花铁干“中平无敌”成名枪；雪山怯战后心态崩坏，枪名成讽刺。
- **五云手 / 铁锁横江 / 陆地神龙**：万、戚、言三人成名绝技，皆一流，却因心术不正在终局互噬。
- **乌蚕衣**：丁典所赠软甲，刀枪难入，狄云雪山与终局赖以保命；属装备位战力，不增攻击而保命线。
- **金波旬花毒**：剧毒无解，凌退思用以涂棺杀丁典；象征“美而致命”，武功再高亦难抗。

乐园阶位映射（对照《阶位战力图鉴》，宁低勿高）：乡野/狱卒/躺尸剑法门人≈**一阶**；门派精英/宝象/万门弟子≈**二阶**；万震山·言达平·戚长发·南四奇单人·凌退思≈**三阶**；丁典（神照大成）/血刀老祖/终局狄云/梅念笙巅峰≈**三阶后期～四阶**（人间绝顶、毁据点级，未达夷平城池）。**世界峰值≈四阶**，无五阶以上。

**【地理 · 舞台】**
舞台为清代中南江湖网络，以荆州官衙—湘西乡野—川边雪山—寺观藏宝为四极，辅以武昌菊会与江湖索宝走廊：
- **湘西沅陵 / 麻溪铺一带南郊**：狄云、戚长发、戚芳隐居乡下，青梅竹马与“空心菜”绰号起点；躺尸剑法的教学场；老实人被拽入江湖的零点。戚长发旧居后亦成言达平掘地寻宝之所。
- **荆州 / 江陵**：全书权力与阴谋中心。**万震山府**寿宴是构陷起点，夹墙是万震山藏尸癖与狄云以其人之道还治的双重符号；**死牢**是狄云—丁典命运交汇点，琵琶骨与神照续命同场；**知府凌退思衙署 / 凌府**囚禁丁典、逼问连城诀、害死凌霜华，狱窗对菊是全书最干净的风景；**天宁寺**（江陵城南偏西）大殿佛像下藏梁元帝巨宝，终局互屠场；城墙曾被用来公布连城剑诀引敌。
- **武昌 / 汉口菊花会**：丁典与凌霜华一见倾心之地；“人淡如菊”的地理锚点。
- **长江三峡**：九年前梅念笙遭三徒围攻投江、丁典相救传功的前史战场。
- **川边大雪山**（修订；初版藏边；原型近贡嘎/海螺沟一带）：血刀老祖掳水笙逃入，雪崩困绝谷；南四奇覆灭、血刀老祖死、狄云神照圆满、水笙情愫生根；终局归隐地。乌蚕丝产地传说亦挂靠此山系。
- **破庙 / 驿道 / 太行外围**：狄云误杀（毒杀）宝象换袍、被正邪追杀的亡命走廊；吕通、吕威、耿天霸、马大鸣等过场角色的死亡背景板。
- **雪谷羽衣 / 冰洞**：绝谷内饥寒、吃尸、诬陷与求生伦理崩坏的封闭舞台；亦是水笙等候的终点。
- **菊圃合葬地**：丁典与凌霜华终葬之所，狄云了却遗愿，与天宁寺毒宝形成“情”与“贪”的空间对位。

**【世界剧情线】**

### 一、乡下人进城（开局·湘西→荆州寿宴）
湘西沅陵麻溪铺一带，**狄云**随师父**戚长发**（铁锁横江）、师妹**戚芳**习一套看似花哨实则无用的**躺尸剑法**——招名被改成“哥翁喊上来，是横不敢过”“落泥招大姐，马命风小小”等方言谐音，实为戚长发藏私、防止弟子窥破唐诗剑法真诀。狄云与戚芳青梅竹马，互称“空心菜”。师徒三人赴荆州给师伯**万震山**（五云手）贺寿。席间万门八弟子轻侮乡人；狄云因故与人冲突，得**言达平**暗中点拨三招真连城剑意，反胜数人，埋下嫉恨。当夜再遭群殴报复。次日戚长发与万震山因旧事——实为**连城诀/宝藏**——暗中冲突：表面“戚长发行刺万震山后逃走”，实则二人各怀鬼胎。万震山与子**万圭**、妾**桃红**及弟子合谋：诬狄云盗窃并非礼桃红，捕快下狱。名场面：乡下人进城、寿宴比剑、三招破躺尸、诬奸入罪。阶段结果：狄云入狱；戚芳被万圭以“营救”为饵控制；世界状态——正派名宿已开始用官刑与流言杀人。

### 二、牢狱（冤案成型·约四年）
狱中狄云遭酷刑：**右手五指被削断**（或主要食指中指，视版本）、**琵琶骨被铁链锁穿**，行动维艰、武功尽废。万圭表面承诺营救，暗中重金买通官府，欲以死罪/长囚逼戚芳就范。戚芳在绝望与胁迫下嫁万圭，并生一女（后亦绰号“空心菜”）。同牢狂囚**丁典**误认狄云为知府**凌退思**派来套取连城秘密的细作，日日虐待、拳脚交加。狱中不时有江湖人（血刀门善勇、胜谛，雪山派等）潜入逼问“藏宝图”，丁典让狄云假扮自己、自己暗中击杀来人，初步展露绝顶实力。名场面：琵琶骨锁、断指酷刑、万圭伪善探监、戚芳出嫁消息传入死牢。阶段结果：狄云彻底失去故人与名誉；丁典仍未信他。世界状态：官匪一家，正义只在刑具上。

### 三、人淡如菊 · 神照续命
狄云闻戚芳改嫁，心死上吊。恰逢丁典**神照功**大成，以内力救活已断气的狄云——章名“人淡如菊”既写**凌霜华**风骨，亦映狄云素心未灭。丁典察其无伪，结生死交，授**神照功**心法，助其从琵琶骨重伤中恢复经脉与内力，并在击杀雪山派来人后，从其尸身剥下**乌蚕衣**（大雪山乌蚕丝所织软甲，刀枪难入，前后两片扣合）赠狄云。丁典自述身世：他本可越狱，只因狱窗对面凌霜华每日换菊传情、毁容守志，甘愿留狱受苦只为见她一眼。二人谋划真正越狱探望霜华。名场面：缢死回生、狱中传功、乌蚕衣授受、隔窗菊花。阶段结果：狄云得当世顶级内功启蒙；丁典第一次有了可托付之人。世界状态：死牢反而成了真相与武学的温室。

### 四、空心菜 · 霜华棺毒（越狱与丁典之死）
越狱后丁典奔凌府寻挚爱**凌霜华**，见其已死入棺（实被父活埋；棺内指甲刻“丁郎，丁郎，我们来世再做夫妻”等血泪字，棺盖暗藏连城诀部分真文）。他不顾一切吻别棺中人，中凌退思预布之**金波旬花毒**（无药可解）。临终丁典向狄云尽吐前史并嘱合葬菊圃：九年前，万震山、言达平、戚长发三师兄弟于长江三峡围攻师父**梅念笙**（铁骨墨萼）夺**连城诀**与**神照经**；梅重伤投江，被丁典所救，临终传功并道破宝藏秘密后身亡。江湖遂传丁典握藏宝图，追杀不断，丁典被迫经商避祸。**武昌/汉口菊花会**上他识凌霜华，两情相悦；此后半年每日隔窗相望、只换一盆花、从无轻薄。凌家迁走后他寻一年余，于江陵重逢，夜夜接她出府旷野畅谈，仍守礼如初，甚至把连城秘密告知于她。**凌退思**表面翰林知府，实为帮会（龙沙帮一类）首脑，为寻梁元帝荆州巨宝谋得知府、拒升迁，假意允婚后以金波旬毒倒囚丁典，每月逼供。凌霜华毁容抗婚（自划十余刀），终被父害死，以尸身作毒饵。丁典临终说“她为我死了，我也就为她死……心里很快活”，拒治毒。丁典死后，狄云得知戚芳已育女“空心菜”，心如死灰离江陵。名场面：棺吻中毒、棺内血字、丁典托友、霜华毁容守贞。阶段结果：连城秘密与合葬遗愿移交狄云；丁典—凌霜华线悲剧收束。世界状态：贪念已杀死一对至情之人。

### 五、老鼠汤 · 血刀僧袍（亡命误入邪派）
狄云颠沛中误入破庙，撞见血刀门弟子**宝象**烹食“老鼠汤”（实为人肉）。冲突中宝象中毒身亡（狄云误用毒物/环境机缘），狄云换上其血衣僧袍御寒，从此被正道当作血刀门人围杀。青海黑教（修订；初版西藏血刀门）掌教**血刀老祖**途经，误认其为门下，出手解围，并掳走“铃剑双侠”中的**水笙**（父水岱，表哥兼未婚夫**汪啸风**），裹挟狄云逃往川边大雪山。途中狄云几次欲救水笙、又被身份与武力所困，正邪两边皆不容。名场面：人肉锅、血衣错位、血刀老祖掳美、正道通缉。阶段结果：狄云身不由己入邪派叙事；水笙线启动。世界状态：正邪标签比真相更快传播。

### 六、落花流水 · 雪谷崩坏（南四奇覆灭）
中原正道**南四奇**（落花流水）——**陆天抒**（仁义陆大刀）、**花铁干**（中平无敌）、**刘乘风**（柔云剑）、**水岱**（冷月剑）——率众追入雪山。雪崩将血刀老祖、狄云、水笙与南四奇等困入绝谷。血刀老祖以怪异刀法杀陆天抒；使计令花铁干误杀结义兄弟刘乘风；砍断水岱双足，水岱不甘受辱，求狄云以木棒结果自己，以免落敌手。花铁干跪求饶命苟活，被血刀老祖戏弄后暂时留命。名场面：落花流水名存实亡、水岱求死、花铁干下跪。阶段结果：南四奇实质覆灭；“正道”招牌在饥寒中碎裂。世界状态：绝谷成为人性实验室。

### 七、羽衣 · 神照圆满 · 血刀伏诛
绝谷奇寒，血刀老祖欲以水笙取暖泄欲，狄云以乌蚕衣与血肉之躯护之，几被老祖扼死——生死关头**神照功最后关窍贯通**，反手一击毙血刀老祖。狄云于老祖遗物中得**血刀经**心法，正邪顶级内功始得兼修。章名“羽衣”写雪谷御寒与水笙衣物相关的生存细节；“梁山伯·祝英台”写两人绝境相依、以衣物/身体取暖却守礼的伦理张力。花铁干饥寒交迫**分食同伴尸体**，并多次谋害狄云与水笙以灭口、掩盖跪地吃尸丑事。水笙自相处中看清狄云外冷内善，情愫暗生。名场面：扼颈通关、一掌毙魔、雪谷吃尸、羽衣御寒。阶段结果：狄云跃升当世顶点级战力；血刀门群龙无首。世界状态：邪派首恶已死，伪君子花铁干成为新威胁。

### 八、「唐诗选辑」· 砌墙（真相与复仇）
雪融获救。花铁干立刻诬陷狄云与未嫁水笙有私，正道舆论再压；水笙被接回，狄云独自下山。他查到言达平在戚长发旧居掘地寻宝，救言达平得解蝎毒药；化装见戚芳，不忍见其苦，赠解药并留下夹蝴蝶的《唐诗选辑》（连城剑谱载体）。潜入万府窃听后，全部真相落地：师父**戚长发**是当年害师元凶之一，隐居乡下教错剑法为藏私与自保；万震山有**砌墙藏尸/梦游砌墙**癖，曾以“砌墙”制造戚长发已死假象；万圭、桃红构陷细节一一对上。万震山父子欲杀戚芳夺谱时，狄云出手，将二人打昏，**以其人之道砌入夹墙**。戚芳心软，**暗挖墙放出万圭**，反被万圭一刀杀害，临终托孤“空心菜”于狄云。狄云杀万圭，清算万震山。后为丁典与凌霜华合葬时，于棺盖得更完整连城剑诀，或书于江陵城墙引敌毕至。名场面：砌墙困兽、梦游砌墙癖、戚芳心软致命、托孤空心菜、合葬菊圃。阶段结果：万门崩盘；狄云对“情义可救恶人”的幻想彻底粉碎。世界状态：复仇完成一半，更大的贪念场在天宁寺等待。

### 九、大宝藏（天宁寺终局）
众人破解《唐诗选辑》密码：浸湿显数字，连城剑招次序对应唐诗字句，拼出“江陵城南偏西，天宁寺大殿佛像，向之虔诚膜拜……”一类指引。**凌退思**、**花铁干**、**汪啸风**、装死现身的**戚长发**、**言达平**及万门余党等齐聚天宁寺，掀开佛像下梁元帝巨宝。戚长发杀万震山（若其尚存）夺先；群豪抢宝时不知宝物通体涂剧毒——中毒者发狂自相残杀，戚长发、凌退思、花铁干、汪啸风等尽死于贪念。狄云目睹“正道”与“邪人”同一副嘴脸，悟出**贪念才是最大的毒**，拒绝染指宝藏。他携戚芳遗孤，重返川边雪谷归隐；入谷时，**水笙**已默默等候。宝藏重埋或无人敢取，江湖传说仍在，人心却已写尽。名场面：唐诗密码、佛下藏宝、毒宝互屠、雪谷重逢。结局：狄云、水笙、空心菜隐居雪谷；丁典凌霜华合葬心愿了结；连城宝藏成为无人敢碰的警示；梅念笙三徒与南四奇、血刀老祖、凌退思等尽数死亡或身败。

**【主要人物】**

- **狄云（湘西乡人 / 戚长发弟子 / 神照功与血刀心法传人）**｜性格：单纯木讷、重情重义、吃尽苦头仍不愿学坏，外号“空心菜”；后期沉默冷峻，看破却不滥杀，拒宝如拒毒。｜装备·能力：初期躺尸剑法（无用）；得言达平三招真连城剑意；狱中得**神照功**、**乌蚕衣**；雪山兼修**血刀心法/刀法**；终局正邪双修、一掌毙血刀老祖，当世顶点级。右手曾断指，琵琶骨曾被锁穿，均靠神照功恢复战力。｜人物弧光：乡下少年→冤狱死囚（约四年）→越狱传人→雪山亡命→砌墙复仇→拒宝归隐。｜立场关系：爱戚芳而失之，收其遗孤；丁典生死之交与传功之师；终与水笙隐居；杀万圭、血刀老祖；拒戚长发与宝藏；对花铁干之诬不屑争辩而用行动证明。

- **戚芳（戚长发之女 / 万圭之妻）**｜性格：温柔善良而软弱，易被形势与亲情绑架，关键时刻心软致命；对狄云旧情未绝却无力反抗万府。｜装备·能力：躺尸剑法入门，战力有限；能识万震山梦游砌墙之诡。｜人物弧光：青梅竹马→被迫嫁万圭→育女空心菜→收狄云解药与唐诗选辑→挖墙救夫反被杀。｜立场关系：狄云初恋；万圭之妻与受害者；临终托孤狄云；戚长发之女（至死未尽知父之恶）。

- **丁典（梅念笙真传 / 狱中狂侠 / 情侠）**｜性格：外狂内烈、至情至性、嫉恶如仇，对霜华一往情深；可因一盆菊甘愿坐牢十年。｜装备·能力：**神照功**大成，“练成神照功，天下无对手”；曾号菊花剑客一类江湖称谓；乌蚕衣得自雪山派来人后转赠狄云。｜人物弧光：救梅得传→经商避祸→菊会遇霜华→为爱入狱不逃→传功狄云→棺吻中金波旬毒而死，嘱合葬菊圃。｜立场关系：凌霜华恋人；狄云师友；凌退思死敌；梅念笙衣钵；杀善勇、胜谛及多名索宝者。

- **凌霜华（凌退思之女）**｜性格：清雅坚贞，“人淡如菊”，为爱毁容守志，狱窗换菊十余年不辍。｜装备·能力：无争武名，以气节立身；棺内血书与棺盖暗藏连城诀片段。｜人物弧光：菊会遇丁典→隔窗换花半年→江陵重逢夜游守礼→父逼为饵→自划毁容→被父活埋，尸作毒棺。｜立场关系：丁典至爱；凌退思之女与受害者；狄云合葬执行对象。

- **水笙（水岱之女 / “铃剑双侠”之一）**｜性格：初傲而明理，绝境中辨人真心，情深不妒。｜装备·能力：家传剑法，铃剑双侠之一，战力不及顶尖。｜人物弧光：被血刀老祖所掳→雪谷识狄云→遭花铁干诬陷→雪谷等候归隐。｜立场关系：水岱之女；汪啸风未婚妻（名存实亡）；狄云终侣。

- **万圭（万震山之子）**｜性格：阴柔伪善、色厉内荏，善用官刑与婚姻夺人所爱。｜装备·能力：万门武学，远逊一流顶尖。｜人物弧光：构陷狄云→娶戚芳→被囚获释后杀妻→死于宝藏之毒或狄云之手（情节以原作为准：复仇线杀万圭，余党多死于天宁寺）。｜立场关系：戚芳之夫与凶手；狄云死仇。

- **万震山（五云手 / 梅念笙大弟子）**｜性格：貌善心贪、城府极深，寿宴主人实为构陷主谋；有砌墙藏尸癖与梦游砌墙症状，阴森过武。｜装备·能力：五云手一流，门下八弟子（鲁坤、周圻、万圭、孙均、卜垣、吴坎、冯坦、沈城等）成势。｜人物弧光：害师夺诀→构陷狄云→做戏“被刺/砌死戚长发”→被狄云砌墙→夺宝局中身亡。｜立场关系：言达平、戚长发师兄弟；凌退思同谋级贪徒；万圭之父。

- **戚长发（铁锁横江 / 梅念笙三弟子 / 狄云师父）**｜性格：阴险狡诈、装死潜伏、教错剑法以藏私，全书最深的“身边之恶”；对女儿亦工具化。｜装备·能力：铁锁横江成名技，真连城剑法知悉者之一；躺尸剑法的制造者。｜人物弧光：害师→隐居湘西教伪剑→装死（万震山砌墙戏码的共谋/利用对象）→天宁寺夺宝中毒狂死。｜立场关系：狄云、戚芳之师与背叛者；杀万震山夺先；与言达平互抢。

- **言达平（陆地神龙 / 梅念笙二弟子）**｜性格：贪婪而略少于万、戚之阴毒，曾以三招真诀点破躺尸剑法，对狄云有片刻“露真”却无改夺宝本性。｜装备·能力：陆地神龙身法武功一流。｜人物弧光：害师→寿宴暗助/点拨狄云三招→旧居掘地→夺宝局中身亡。｜立场关系：梅门三叛徒之一；狄云短促的“真剑”启蒙者。

- **梅念笙（铁骨墨萼）**｜性格：侠义而识人不清，对三徒防范不足；持连城与神照两大密，成为天下贪念的靶心。｜装备·能力：神照经、连城剑法、梅花拳、大力鹰爪等，正道近顶；第三版互文称与吴六奇同派。｜人物弧光：持诀遭三徒围攻于三峡→托丁典→身亡。｜立场关系：万/言/戚之师；丁典恩师；全书灾难的“种子持有者”。

- **血刀老祖（青海黑教第四代掌教）**｜性格：好色残忍、狡诈好杀，视人命如草芥，善用计分化南四奇。｜装备·能力：**血刀经**内功刀法怪异至极，可一敌南四奇（用计+实力）。｜人物弧光：掳水笙入雪→杀陆天抒等→扼狄云反被神照圆满一击毙命。｜立场关系：宝象等之师；狄云阶段性“假师父”与死敌；水笙的直接施暴者。

- **花铁干（中平无敌 / 南四奇）**｜性格：外勇内怯、伪君子标本，饥则吃尸、危则下跪、安则诬人，是“正道”二字最响的讽刺。｜装备·能力：中平枪法，南四奇中相对最弱。｜人物弧光：误杀刘乘风→跪血刀→吃尸→诬狄云水笙→天宁寺中毒死。｜立场关系：南四奇之耻；狄云舆论死敌；水笙名节的加害者。

- **陆天抒 / 刘乘风 / 水岱（南四奇其余三人）**｜陆天抒：仁义陆大刀，死于血刀老祖，是四奇中较有侠名者。刘乘风：柔云剑，被花铁干误杀，死于“自己人”。水岱：冷月剑、水笙之父，断足后求狄云结果自己，以免受辱——把最后一点尊严托付给被当成血刀门人的狄云。四人合称“落花流水”，追血刀而尽灭，象征正道联盟的脆断。

- **凌退思（荆州知府 / 帮会首脑）**｜性格：道貌岸然、为宝杀女，官匪一体；假允婚、真下毒，把亲情写成夺宝道具。｜装备·能力：武功与权术并重，金波旬花毒使用者；死牢与衙署双线控制。｜人物弧光：囚丁典十年→杀女设毒→天宁寺夺宝致死。｜立场关系：霜华之父与凶手；丁典死仇；与万门争同一张藏宝图。

- **汪啸风（铃剑双侠之一 / 水笙表哥兼未婚夫）**｜性格：正道青年，终难逃贪念场；对水笙有名分而无雪谷共患难。｜装备·能力：铃剑双侠剑法。｜人物弧光：失水笙→卷入夺宝→中毒死。｜立场关系：水笙名义未婚夫；与狄云形成“名分 vs 真情”对照。

- **宝象（血刀门弟子）**｜性格：凶残吃人。｜装备·能力：血刀门武学。｜人物弧光：破庙吃人→中毒死→僧袍连累狄云，是身份错位的导火索。

- **桃红（万震山妾）**｜性格：势利，参与诬奸剧本。｜后遭抛弃，为构陷棋子，体现万府连“自己人”也工具化。

- **空心菜（戚芳与万圭之女）**｜狄云旧绰号的血缘延续；终局由狄云（与水笙）抚养归隐，象征素心在下一代偷生，而非江湖翻盘。

- **万门弟子群**：**鲁坤、周圻、孙均、卜垣、吴坎、冯坦、沈城**等——寿宴帮凶与门派中坚，周圻等死于冲突，吴坎等有内斗被杀记载，体现万门“名门”实为鹰犬窝。

- **血刀门其余 / 索宝过客**：**善勇、胜谛**死于丁典；**耿天霸、马大鸣**等绿林亦在索宝链上被清理，衬托“连城”二字召来的是一整条食腐食物链。

**【势力图谱】**
- **梅念笙一系（已碎）**：神照经与连城剑法源头。代表：梅念笙、丁典（真传）、狄云（再传）。三徒万震山、言达平、戚长发叛师后自相吞噬。
- **万震山府 / 万门**：荆州地方武学名门外壳，实为夺宝与构陷中枢。代表：万震山、万圭、桃红、八弟子。与官衙、凌退思利益纠缠。
- **荆州官衙—帮会（凌退思 / 龙沙帮一类）**：知府身份掩护的夺宝组织，金波旬花毒与死牢为工具；拒升迁只为盯荆州宝藏。代表：凌退思、衙役捕快、帮会打手。
- **雪山派（过场）**：曾派人入狱索宝，被丁典击杀，乌蚕衣由此落入狄云之手。
- **南四奇（落花流水）**：中原正道标杆组合，追剿血刀。代表：陆天抒、花铁干、刘乘风、水岱。雪山后名存实亡，暴露正道虚名。
- **铃剑双侠**：水岱、水笙、汪啸风一系，家传剑与姻亲绑定，被血刀线撕碎。
- **青海黑教 / 血刀门**：邪派刀法集团。代表：血刀老祖、宝象、善勇、胜谛等。老祖死后星散。
- **太行等外围绿林**：吕通、吕威、耿天霸、马大鸣等，多在丁典或夺宝周边被清理，衬托江湖层层食腐。
- **北四怪（风火云龙）**：仅提及对照的北方名号，本篇戏份极淡，作江湖坐标。

**【贵重物品】**
- **《连城诀》/《唐诗选辑》/连城剑谱**：表面唐诗剑法或诗选，浸湿显数字，按剑招序读出“江陵城南偏西，天宁寺大殿佛像……”等藏宝指引。梅念笙持有→三徒争夺→丁典知晓→霜华棺盖补全→终局多人破解。梁元帝（南朝）荆州巨宝之钥；武学与贪欲同一本书。
- **《神照经》**：至高内功秘籍，梅→丁典→狄云。疗伤续命、缢死可救、战力质变；口诀强调练成则“天下无对手”。
- **《血刀经》**：血刀门内功刀法总诀，刀招从不可能方位劈砍；狄云雪谷自老祖遗物中得心法，与神照双修成终局面板。
- **躺尸剑法**：戚长发篡改的伪唐诗剑法，谐音恶搞招名（“哥翁喊上来”“落泥招大姐”“两个黄梨拧脆了”等），华而不实，专坑弟子、藏真诀；言达平三招真诀可破之。
- **乌蚕衣**：大雪山乌蚕丝软甲，刀枪难入、剪刀难断，前后两片扣合；原雪山派人物之物，丁典击杀索宝者后赠狄云，雪谷与终局多次保命。
- **金波旬花**：剧毒花卉（名取佛教魔王波旬），无药可解，涂于霜华棺盖杀丁典；亦象征美与死。凌退思夺宝工具之一。
- **天宁寺佛下宝藏**：梁元帝所藏金银珠宝，通体剧毒（防盗），触者发狂互屠；狄云拒取，主题物化。
- **血刀 / 中平枪 / 铃剑**：血刀老祖兵刃、花铁干“中平无敌”成名枪、水笙一系铃剑，剧情杀伐与身份符号。
- **菊花（意象道具）**：菊会、狱窗换菊、菊圃合葬，贯穿丁典—凌霜华线，是本书唯一不被毒化的“宝物”。

**【隐藏剧情 · 伏笔】**
- 连城剑法=藏宝密码：唐诗招名顺序即地图句读，武学与贪欲同一本书；浸湿显字是典型“书中书”机关。
- 躺尸剑法是师父的“爱”与背叛：戚长发教错既防弟子抢宝，也保证自己永远高弟子一等；言达平三招则是裂缝。
- 花铁干=正道镜像：下跪、吃尸、诬奸，说明“落花流水”名号救不了人心；其最终死于毒宝，与邪派同坑。
- 金波旬棺吻：至爱被制成杀器，父女伦理与官匪伦理同时崩坏；棺盖补全连城诀，连死者都被榨取情报。
- 空心菜双重指称：狄云绰号→戚芳之女，素心在下一代偷生，不靠称霸翻盘。
- 砌墙对位：万震山梦游砌墙藏尸癖 ↔ 狄云以其道砌回万氏父子，正义以恐怖形式现身。
- 乌蚕衣来路：雪山派索宝者死物转赠，说明“连城”召来的杀手装备反而护住了素心者。
- 第三版互文：梅念笙与吴六奇同派，把本书轻轻钩进《鹿鼎记》宇宙，但不改本篇封闭悲剧。
- 主题句：贪念是最大的毒——天宁寺毒物只是物化的人心；狄云不碰宝=唯一解毒法。
- 水笙等候：全书罕见的“未被贪念腐蚀的等待”，与戚芳心软、霜华殉情形成三种女性命运对照（软弱致死 / 坚贞致死 / 等待得生）。
- 丁典不逃：武功可越狱却因一窗菊自囚十年，是“情”对“力”的倒置，衬出满江湖有力无情。

**【大事记时间线】**
前史九年 → 梅念笙遭万震山、言达平、戚长发围攻；丁典得神照经与连城秘密；梅死。  
前史数年 → 丁典与凌霜华相恋；凌退思囚丁典逼宝；霜华毁容被害。  
开局 → 狄云师徒贺万震山寿；构陷入狱；戚芳嫁万圭。  
狱中 → 狄云缢死被神照救活；学神照、得乌蚕衣；越狱。  
江陵 → 丁典棺吻中毒死；狄云知空心菜出生后离开。  
亡命 → 破庙毒死宝象；血刀老祖掳水笙入川边雪山。  
雪谷 → 南四奇覆灭；狄云神照圆满毙血刀老祖；花铁干吃尸诬陷。  
下山 → 砌墙困万门；戚芳放万圭被杀；狄云复仇。  
终局 → 天宁寺破译唐诗密码；群豪触毒互屠；狄云携孤归隐；水笙雪谷相候。

**【叙事基调 · 雷区】**
画风：冷硬写实武侠，冤狱与雪谷戏极压抑，偶有躺尸剑法谐音黑色幽默；不歌颂称霸，只逼问“好人如何活”。  
忌：把狄云写成快速黑化的复仇魔王或争霸主角；把戚长发写成慈师偶尔犯错；把花铁干写成一路刚烈的真英雄；用毁城爆星夸张神照/血刀；把水笙写成单纯花瓶或抢宝女配；让连城宝藏无毒可安然发家（破坏主题）；时间线跳过狱中传功与雪谷吃尸（二者是人性转折）。  
最早可切入锚点：万震山寿宴前夜、狄云刚入狱、越狱当夜、破庙换袍前后、雪崩入谷前、砌墙复仇前、天宁寺夺宝前夜。

## 阶位切入点

> 阶位↔江湖战力：一阶≈乡野新人/狱卒捕快/躺尸剑法门人；二阶≈万门精英弟子/宝象级/绿林好手；三阶≈万震山·言达平·戚长发·南四奇单人·凌退思；四阶≈丁典（神照大成）/血刀老祖/终局狄云/梅念笙巅峰。世界顶点为四阶人间绝顶，低阶切入一律**规避顶点·情报优先/条件性胜利**；顶点按原作全盛书写，不得用“任务限制/突然变弱”解释。

**一阶（乡下入城·寿宴构陷）**

切入身份/时点：契约者以湘西/荆州一带镖队趟子手、万府下人、酒楼跑堂或麻溪铺乡勇切入，锚定**狄云师徒抵达万震山寿宴前后至狄云刚刚入狱**。

初始事件：你在寿宴杂役席听见万门弟子讥讽“躺尸剑法”，又见乡人狄云以拙剑反胜；夜半捕快突至，桃红哭喊“非礼”，戚芳被万圭“劝回”内宅，街巷戒严押送死牢。

开场白建议：「荆州的灯比沅陵亮十倍，可你总觉得每盏灯后都有人在记账。空心菜——他们这么叫那个乡下小伙子——被铁链拖过石板时，还回头找师妹的影子。你手里只剩一碗没送出去的寿面，已经凉了。」

关键NPC立场：**狄云**蒙冤却仍想解释，可成证人线；**戚芳**惊慌无措，需要可信的外人递消息；**万圭**温言拉拢杂役作伪证；**万震山**杀气藏在笑里；**桃红**按脚本表演；**戚长发**已“失踪”，实则暗处观察。

主线钩子/支线：主线——是否在公堂/牢门递出“桃红与万圭密谈”的证据，改变狄云量刑速度。支线：①偷听万言戚三人为“剑谱”争吵的只言片语；②护送戚芳一封未送出的家书；③拒写诬陷画押换赏银。

危险度/规避：中低。规避与万震山、言达平正面冲突；死牢深处的丁典仅作传闻，勿硬闯。

任务方向/奖励：保全一名证人或救出半条狄云的活口消息；奖励：荆州路引、三流剑谱残招、万府腰牌、戚芳的信物与“空心菜”绰号来历情报（非神照经）。

**二阶（死牢与血衣·精英夹缝）**

切入身份/时点：契约者以万门记名弟子、刑房牢头、太行绿林二流或血刀门外围切入，锚定**狄云狱中传功前后至破庙换宝象僧袍、血刀老祖现身**。

初始事件：你奉命给死牢送刑具，听见狂囚丁典怒吼与少年狄云的沉默；或在城外破庙撞见“老鼠汤”人肉锅，血衣僧袍尚热，官军与黑教同时喊杀。

开场白建议：「琵琶骨的铁链锈进肉里时，神照功的呼吸声却稳得吓人。你不知道该怕狱里的疯子，还是怕狱外的知府。城门口一张新告示：血刀门僧人过境，格杀勿论——而你刚捡到的，正是一件还滴血的僧袍。」

关键NPC立场：**丁典**只信真心，试探则打；**狄云**可结盟但信息不全；**凌退思**视一切外来者为可灭口的探子；**宝象**吃人灭口；**血刀老祖**喜怒无常，认袍不认人；**万圭**继续用戚芳线封口。

主线钩子/支线：主线——越狱夜选择放行丁典狄云、通风报信给凌府，或只偷走半页“唐诗”线索。支线：①查金波旬花来源；②阻止宝象再猎路人；③向水笙一系提前警告血刀路线。

危险度/规避：中高。规避单独对上丁典或血刀老祖；凌府内宅=高危。

任务方向/奖励：换取乌蚕衣碎片情报、狱图、血刀门动向；奖励：二流刀法心得、解毒残方（非金波旬解药）、越狱人情、半阙唐诗剑招名（尚未成谱）。

**三阶（雪山落花·一流名宿局）**

切入身份/时点：契约者以散修一流、小派掌门、南四奇随行高手或铃剑双侠旁支切入，锚定**血刀老祖掳水笙入川边雪山至雪谷南四奇覆灭、血刀老祖伏诛前后**。

初始事件：你随陆天抒或水岱追入雪线，雪崩封谷；耳边是花铁干的中平枪名号与刘乘风的惨呼，眼前是血刀老祖把水笙抛在雪上当饵。

开场白建议：「落花流水四个字，在雪里冻成了哭声。有人下跪，有人吃肉，有人求你一棒打死他。你握着的刀还算一流，可你分不清——此刻该杀的是魔头，还是那个喊着‘中平无敌’的同伴。」

关键NPC立场：**血刀老祖**敌对，仅群策可扰；**狄云**护水笙、可并肩但沉默；**水笙**初疑后信；**花铁干**会拉你作伪证；**水岱**托死后事；**陆天抒/刘乘风**若仍存活需救援窗口极短；**汪啸风**正道立场强硬。

主线钩子/支线：主线——雪谷内是否揭穿花铁干误杀与吃尸，改写“狄云奸邪”的舆论。支线：①护水笙至雪融；②夺血刀经残页但不练邪功；③记录水岱遗言交给水笙。

危险度/规避：高。血刀老祖：条件性胜利（地利、偷袭、狄云神照关窍将通时的配合），禁止写成无故削战力。规避雪谷独行夜。

任务方向/奖励：保全水笙或南四奇之一的名誉证据；奖励：准一流内功笔记、防寒羽衣、血刀刀意观察记录、水笙铃剑一式指点（非完整神照）。

**四阶（砌墙与天宁寺·顶点边缘）**

切入身份/时点：契约者以当世一流顶尖、隐世前辈、梅门旁系或荆州地下舵主级切入，锚定**狄云下山砌墙复仇至天宁寺大宝藏开启**。

初始事件：江陵巷弄半夜起墙封门，万震山父子被困；或天宁寺大殿佛像眼中渗出金光，唐诗选辑的湿痕数字在火把下显形，戚长发含笑现身。

开场白建议：「佛像抬起来的时候，所有人都跪了——不是为佛，是为了金子。你闻见一种甜得发苦的香，像金波旬，又像人血。狄云站在最外一圈，眼里没有宝，只有坟。你忽然明白：今晚死的人，都会死在同一句唐诗里。」

关键NPC立场：**狄云**拒宝，可同盟但拒绝称霸；**戚长发**最危险的伪师，笑里藏刀；**凌退思**官威+帮会；**花铁干**拼命洗白；**言达平**抢先手；**戚芳**（若仍在砌墙段）心软是变数；**水笙**在雪谷线外等候，可作终局退路；**丁典/梅念笙**已死，仅遗志与经义可“对话”。

主线钩子/支线：主线——是否在触宝前喊破“毒”字，或帮狄云完成丁典与凌霜华合葬、空心菜托孤。支线：①截下唐诗选辑真本防再泄；②阻止汪啸风/花铁干煽动屠城式搜宝；③护送狄云水笙归隐路线不被江湖跟踪。

危险度/规避：极高/贴近顶点。丁典级神照、血刀老祖级刀法、终局狄云：本世界峰值四阶——**存在·情报优先/条件性胜利**（毒宝、舆论、砌墙困局、群贪互屠），按原作全盛写，不得写成封印削弱。天宁寺内人人可能疯狂。

任务方向/奖励：促成拒宝归隐线或保住空心菜活路；奖励：神照功呼吸法门观察（非完整经书）、乌蚕衣修复线索、连城密码作废声明的江湖传播权、四阶以下兵刃与“素心”名声——**不发可单吃毒宝或稳胜终局狄云的越级功法**。
