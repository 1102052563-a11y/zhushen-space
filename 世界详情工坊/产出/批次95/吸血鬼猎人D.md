# 吸血鬼猎人D
<!--meta lib=主库 tiers=一、二、三、四、五、六、七-->

## 剧情

**【作品来源】**
菊地秀行小说系列《吸血鬼猎人D》（吸血鬼ハンター"D"），1983 年起出版，天野喜孝插画，长篇连载数十卷。改编：1985 年芦田丰雄执导 OVA（对应《吸血鬼新娘》系）；2000 年前后川尻善昭执导剧场版《Vampire Hunter D: Bloodlust》（对应《妖杀行》），原作者评价更高；另有漫画等。文风：西部荒野、哥特贵族、科幻废土、孤独剑客。卡片锚点：麦耶尔·林克的马车——D 在荒原追逐贵族马车，对抗野蛮族三兄弟（影子操纵、变形、树木融合），银质长剑斩断幻术。

**【世界定位】**
西历约一万二千年后的地球：核战后废墟上，预知核战的「贵族」吸血鬼以中世纪美学重建统治，用基因与妖术制造妖兽，科技可造星际船。人类在边境农场与城寨苟活，雇佣猎人对抗贵族与怪物。D 是吸血鬼王之子与人类所生的混血，行走边境猎杀贵族，亦被双方排斥。世界库以剧场版《妖杀行／Bloodlust》为主轴（林克马车与卡蜜拉城塞），OVA 李伯爵线作低中阶对照。

**【世界观 · 力量体系】**
力量本源：贵族血脉妖术、超科学遗物、混血身体能力、人类枪械与人数、左手共生存在。
1. 边境农民／市民：火器简陋，惧夜。
2. 人类猎人团（马可斯兄妹级）：枪械、协作、药剂，可杀中低贵族护卫。
3. 蛮族／异能佣兵：影子杀、拟态、兽化跳跃等专精。
4. 普通贵族吸血鬼：超速再生、魅惑、夜行优势，惧阳光与特殊剑技。
5. D（Dhampir）：白日可行动（有阳光症风险）、超凡剑速、左手辅助吸血再生与吐槽情报。
6. 古代贵族／卡蜜拉级残留意志：幻术、吸血复活、精神压制。
7. 吸血鬼王相关血脉顶点：叙事顶格，情报优先。
死亡：人类真死；贵族可被斩首／破心／特殊手段消灭；D 可重创但极少被简单杀死。顶点条件性胜利：利用阳光、幻术破绽、情感隙、星际船时限等，严禁写顶点被封印故战力为零。
乐园阶位映射：农民民兵≈一阶；人类猎人≈二阶；蛮族高手≈三阶；普通贵族≈四阶；D 常规任务态≈五阶；卡蜜拉城塞级≈六阶；王血／古代顶点≈七阶。宁低勿高。

**【地理 · 舞台】**
边境小镇与农场、荒原道路、贵族城堡与避难所、卡蜜拉传说城塞伊特（含星际飞船）、雷雨树洞、墓园后日谈。卡片锚点：荒原马车追逐与野蛮族三兄弟战。气味：尘土、马汗、旧血、城堡冷石、火箭燃料。

**【世界剧情线】**
① OVA 对照线：朵莉丝被李伯爵「死之吻」，雇 D；拉米卡反对父亲吸血扩张；丽银星等受煽动与 D 战斗；D 以孤独剑技了结贵族阴谋。作低中阶教学：雇佣契约、贵族傲慢、混血歧视。
② 剧场版开端：夏洛特·艾尔邦被报「掳走」，实则自愿与麦耶尔·林克私奔，欲逃向星海尽头平静生活。艾尔邦家双雇 D 与马可斯兄妹（波可夫、蕾拉、葛罗普、凯尔、诺多）保险式追杀。
③ 荒原追逐（卡片锚点）：D 追林克马车；野蛮族三兄弟拦截——影子杀、环境拟态、狮头腹化跳跃敏锐。马可斯成员逐个战死或重创；蕾拉重伤被 D 所救。
④ 真相与雨夜：D 确认夏洛特自愿；蕾拉在雷雨中助 D 埋土度过阳光症危机，两人树洞对话：恨意与混血身份。蕾拉仇视贵族，D 称若以猎杀为乐则人也将不人。
⑤ 卡蜜拉城塞：林克与夏洛特入伊特，落入卡蜜拉复活阴谋。幻术使林克误判夏洛特离弃，卡蜜拉斧伤林克并吸血夏洛特。波可夫被控，葛罗普药剂同归于尽。D 以王血力量反制卡蜜拉精神压迫并重创之；林克复活救夏洛特杀卡蜜拉，但夏洛特失血死于怀中。
⑥ 终战与戒指：D 为任务与「贵族对人类施血牙之罪」与林克决战占上风；蕾拉目睹爱情真相，扔出夏洛特戒指。D 剑刺偏心脏仅中左肩，用剑挑起戒指证明「目标已死、任务完成」。林克携遗体登火箭赴星海。
⑦ 后日：数十年后蕾拉葬礼，其孙女见山坡树下 D；D 为履行「墓前献花」而来，见已有许多花，微笑策马离去。
任意时点：雇佣接单／荒原追车／蛮族战／雨夜树洞／入城塞／神殿式幻术战／火箭台／墓园后日。蝴蝶可救某一猎人、改戒指时机；不可无代价抹去混血孤独与贵族衰亡的时代底色。

**【主要人物】**
- D：混血猎人，寡言，银长剑，左手共生；弧光从杀戮机器到对契约与情的一丝缝隙。
- 左手：吐槽与吸血辅助，情报口。
- 麦耶尔·林克：贵族，爱夏洛特，愿为私奔对抗猎人们。
- 夏洛特·艾尔邦：自愿私奔，失血死。
- 蕾拉·马可斯：人类猎人，恨贵族，终理解并改命运一剑。
- 波可夫／葛罗普／凯尔／诺多：猎人团，逐个牺牲。
- 卡蜜拉：古代残留，幻术与复活阴谋。
- 李伯爵、朵莉丝、拉米卡、丽银星：OVA 线关键。
- 亚伦·艾尔邦：夏洛特之兄，雇主侧。
- 野蛮族三兄弟：护卫专精。

**【势力图谱】**
贵族残存城堡与城塞；人类边境聚落；猎人公会／佣兵团；蛮族护卫；古代意志（卡蜜拉）；星际逃离线（火箭）。

**【贵重物品】**
D 的银长剑；左手能力；夏洛特戒指（终局关键）；卡蜜拉城塞与星际船；野蛮族异能；阳光症与埋土求生法；雇佣契约金与艾尔邦家徽。

**【隐藏剧情 · 伏笔】**
D 为吸血鬼王之子；贵族用基因造妖兽与宇宙船；阳光症是混血代价；自愿私奔颠覆「掳走」叙事；戒指一掷是人类情感改写剑轨；后日献花是系列罕见温暖。

**【大事记时间线】**
核战 → 贵族统治 → 边境雇佣文化 → OVA 李伯爵事件 → 林克与夏洛特私奔 → 双猎人追杀 → 蛮族战 → 卡蜜拉阴谋 → 戒指与火箭 → 蕾拉葬礼 D 献花。

**【叙事基调 · 雷区】**
基调：冷硬西部＋哥特悲恋，孤独与风沙。忌把 D 写成热血少年漫主角；忌忽略混血歧视；忌低阶硬刚古代意志；忌把夏洛特写成无主动人质。最早切入＝接雇佣前后或荒原见马车尘烟时。

## 阶位切入点

> 阶位↔原作：一阶≈边境农民；二阶≈人类猎人；三阶≈蛮族高手；四阶≈普通贵族；五阶≈D 任务态；六阶≈卡蜜拉城塞；七阶≈王血／古代顶点。顶点情报优先／条件性胜利。

**一阶（边境 · 雇佣传闻线）**
切入身份/时点：农场帮工或驿站伙计，D 入镇前后。
初始事件：夜风带来马蹄与甜腥；镇长贴出贵族袭击悬赏，有人低声说「叫 D」。
开场白建议：「风车吱呀像旧骨架。你把干草叉握热，却知道叉子刺不穿贵族的夜。尘土里一道黑影帽子压得很低——左手在手套里动了动。」
关键NPC立场：**D**——只谈报酬与目标；**左手**——嘲讽人类胆小；**镇民**——又怕又求；**贵族眼线**——记录谁雇猎人。
主线钩子/支线：是否参与集资雇 D；支线＝护送孩童进屋、辨认咬痕真伪。
危险度/规避：中；规避夜间单独出镇。
任务方向/奖励：接触猎人网络；银弹劣货、地图、D 的侧目。

**二阶（猎人 · 马可斯协作线）**
切入身份/时点：马可斯团临时枪手，双雇追杀启动时。
初始事件：艾尔邦家同时付两份契约；蕾拉警告你「别挡 D 的剑」。
开场白建议：「两份合同，一个目标。枪油味盖不住贵族香水。荒原尽头马车扬尘，像一道不肯愈合的伤口。」
关键NPC立场：**蕾拉**——务实；**波可夫**——冷酷效率；**D**——并行不合作；**亚伦·艾尔邦**——只要结果。
主线钩子/支线：马车首轮接触；支线＝补给水、侦察蛮族脚印、隐瞒夏洛特自愿的信件。
危险度/规避：高。
任务方向/奖励：猎人协作经验；制式枪械、药剂、团内位置。

**三阶（蛮族 · 三兄弟战线）**
切入身份/时点：刀盾手或斥候，锚定影子／拟态／兽化拦路。
初始事件：同伴影子被切断倒地；树影里有人与树皮同色；腹中狮吼震马。
开场白建议：「银剑出鞘的光比太阳薄。D 说瞄准的不是影，是影的主人。你满嘴沙尘，第一次相信『斩断幻术』是字面意思。」
关键NPC立场：**D**——主攻；**蕾拉**——负伤掩护；**野蛮族三兄弟**——死战；**林克**——护车先行。
主线钩子/支线：破三兄弟联防；支线＝救蕾拉、夺马车零件、保护夏洛特不被流弹所伤。
危险度/规避：高。
任务方向/奖励：破异能经验；蛮族武器残片、D 的短暂认可。

**四阶（贵族 · 自愿真相线）**
切入身份/时点：谈判者或艾尔邦家仆，锚定确认夏洛特自愿时。
初始事件：夏洛特亲口说爱林克；任务从「救人」变成「破坏爱情还是完成契约」。
开场白建议：「城堡窗关着，心却开着。她说不要被救。你的合同在风里响，像一张过期的谎。」
关键NPC立场：**夏洛特**——求你停手；**林克**——警戒；**D**——仍执行契约逻辑；**蕾拉**——恨意动摇。
主线钩子/支线：是否把真相传回雇主；支线＝藏戒指、伪造「已救回」消息、拖延马可斯火力。
危险度/规避：高。
任务方向/奖励：真相权；贵族遗物、契约解释权。

**五阶（D 级 · 雨夜与城塞入口）**
切入身份/时点：与 D 短暂同行者，阳光症发作与入伊特前。
初始事件：D 斩敌后晕厥，需埋土；蕾拉被复活头颅抓住；落雷行刑。
开场白建议：「土填到他唇边时，左手还在骂。雷把世界撕白。你第一次看见猎人被『太阳』打败。」
关键NPC立场：**D**——被迫受助；**蕾拉**——还人情；**左手**——催促；**卡蜜拉眼线**——引你们入城。
主线钩子/支线：入城塞或劝返；支线＝采集阳光症土法、标记飞船库方向。
危险度/规避：极高。
任务方向/奖励：与 D 的互债；高级药剂、城塞地图碎片。

**六阶（城塞 · 卡蜜拉阴谋线）**
切入身份/时点：突入小组，锚定幻术吸血与猎人团溃灭。
初始事件：林克被幻术动摇，夏洛特血被吸；波可夫反噬；葛罗普同归。
开场白建议：「墙在呼吸。有人用你母亲的声音叫你放下剑。D 的瞳孔里有不属于这个世纪的光。」
关键NPC立场：**卡蜜拉**——要血与臣服；**林克**——救侣；**D**——反制王血；**蕾拉**——最后人类火力。
主线钩子/支线：破幻术保夏洛特残命；支线＝毁复活祭坛、夺飞船钥匙。
危险度/规避：极高。
任务方向/奖励：参与打倒古代残留；城塞核心构件、王血情报。

**七阶（顶点 · 戒指与火箭线）**
切入身份/时点：终战见证者，锚定 D 与林克决战、蕾拉扔戒指。
初始事件：剑指向心脏，戒指在空中旋转；火箭点火倒计时。
开场白建议：「荒原风把悼词吹散。D 的剑为契约，林克的牙为爱。戒指落点决定谁还能看见星星。」
关键NPC立场：**D**——任务将成；**林克**——只求带走遗体；**蕾拉**——改写一剑；**夏洛特遗体**——沉默的裁决。
主线钩子/支线：是否助偏剑、是否拦火箭；支线＝保护蕾拉孙女时间线（后日）、记录王血觉醒。
危险度/规避：贴近顶点——条件性胜利／情报优先。
任务方向/奖励：终局见证与后日花约；传说级信誉、星际尘埃样本、D 的微点头。
