# 火焰之纹章：新·暗黑龙与光之剑
<!--meta lib=主库 tiers=一、二、三、六、七-->

## 剧情

**【作品来源】**
《火焰之纹章 新·暗黑龙与光之剑》（日语：ファイアーエムブレム 新・暗黒竜と光の剣，英：Fire Emblem: Shadow Dragon）为 Intelligent Systems 开发、任天堂发行的任天堂 DS 战略角色扮演游戏，系列第11作，是1990年FC初代《暗黑龙与光之剑》的重制。2008年8月7日日本发售，后于欧/北美/澳洲发行，系系列初代首次正式海外化（8-4本地化）。总监堀川将之、俵正树，制作人成广通、山上仁志，音乐辻横由佳与春日沙树，主视觉士郎正宗与井塚大介。文风：王道解放战争 + 永久死亡铁则 + 兵种棋盘；故事简洁直给，重点在军团经营与关卡结构。续作重制为《新·纹章之谜》，本档案只覆盖暗黑龙篇（马尔斯第一次解放战争）。

**【世界定位】**
阿卡内亚大陆。远古英雄安里以神剑法尔西昂与封印之盾「火焰纹章」击败暗黑龙梅迪乌斯；百年后邪术师卡涅夫复活梅迪乌斯，重建多鲁亚帝国并侵略诸国。阿利提亚王子马尔斯在祖国沦陷、父王科尼利厄斯战死、姊艾莉丝被俘后流亡塔利斯，再与希达等盟友率军反攻，夺回火纹、法尔西昂，击破卡涅夫与梅迪乌斯。一句话：少年王在永久死亡规则下集结杂牌军，把大陆从暗龙爪下撬开。

**【世界观 · 力量体系】**
力量本源：兵种训练、武器技艺、魔法与龙族血统（玛那克特/神龙）。战斗在网格地图上回合制进行；**武器三角**（枪克剑、剑克斧、斧克枪）决定近战优劣；弓克飞行，魔法分理/光/暗等（本作体系相对早期）。武器有耐久，可锻造与合并。角色升到一定等级可用大师证转职；本作新增**兵种转换（Reclass）**。铁则：**单位永久死亡**——阵亡即退出战役，外传角色在减员严重时补位（诺尔内、雅典娜、那基等）。

战力阶梯：
1. **村民·伤兵·平民**：≈一阶。
2. **新兵·低级海盗/山贼**：≈一~二阶。
3. **正规士兵·骑士·弓手·僧侣**：≈二~三阶。
4. **精锐骑士·飞马骑士·剑士·魔道士**：可左右一城攻防。≈三~四阶。
5. **将军·圣骑士·龙骑士·大贤者**：方面军骨干。≈四~五阶。
6. **黑骑士卡缪级名将 / 米奈娃级龙骑将**：一国王牌，可挡全军锋。≈五~六阶。
7. **卡涅夫（邪术师）**：操控帝国与禁咒，战略级威胁。≈六~七阶。
8. **梅迪乌斯（暗黑龙）**：龙地顶点，需法尔西昂与火焰纹章条件战。≈七阶。

乐园阶位映射：平民/伤兵≈一阶；新兵/山贼≈一~二阶；正规军≈二~三阶；精锐≈三~四阶；将官≈四~五阶；名将卡缪/米奈娃≈五~六阶；卡涅夫≈六~七阶；梅迪乌斯≈**七阶**。本条覆盖一、二、三、六、七阶。宁低勿高；严禁「暗龙被削弱所以随便砍」。

**【地理 · 舞台】**
阿卡内亚大陆诸国：阿利提亚（马尔斯母国）、格拉（叛变邻国）、塔利斯（流亡岛国）、奥雷利斯、格鲁斯特、卡达因（魔道都市）、马其顿（龙骑士之国）、多鲁亚（暗龙帝国）、拉曼神殿、港湾沃伦、克诺达市场、赫勒拿城等。战役从塔利斯出征→解放中部→奥雷利斯同盟→取火纹→卡达因→夺回阿利提亚→对黑骑士→马其顿内战→卡涅夫→龙地终战。

**【世界剧情线】**
① **远古与复活**  
安里斩暗龙，大陆和平百年。卡涅夫复活梅迪乌斯，多鲁亚再起，格拉背叛，阿利提亚陷落。科尼利厄斯战死，艾莉丝被俘，马尔斯由忠臣掩护逃塔利斯。

② **序章·逃亡（重制新增）**  
正常难度下四章序章详述逃亡、父死、姊俘，建立「王子几乎一无所有」的起点。杰刚、凯因、阿贝尔、高顿、多洛亚等旧臣与弓手构成最初棋子。

③ **塔利斯出征**  
两年后马尔斯与塔利斯公主希达、佣兵奥格玛班（巴兹、波德、科德等）击退海盗，正式踏上大陆。招募纳巴尔、朱利安、莉娜、马利克等名场面：盗贼救修女、剑士对决、幼年魔道友人重逢。

④ **奥雷利斯与火焰纹章**  
与奥雷利斯王弟哈丁及狼骑士团（沃尔夫等）会师，自妮妮公主处取得**火焰纹章**——政治正统与封印钥匙。此章是「从流亡军变成联合军」的质变。

⑤ **中部战役与港湾**  
勒夫坎迪斯隘口、沃伦港、皮拉西火龙等关卡推进补给线；外传在减员时开启，补诺尔内等角色。

⑥ **马其顿姐妹**  
龙骑士公主米奈娃、玛丽亚、帕拉、卡秋雅、艾丝特线：米奈娃为救妹与兄长米谢尔对立，最终可加入马尔斯。马其顿内战是中盘情感与兵力双重高潮。

⑦ **王宫与叛变**  
无龄王宫、叛徒价值外传等：阿卡内亚宫廷政治、米迪娅等禁卫、阿斯特拉姆等剑士。琳德（米洛亚之女）与奥拉魔道线补齐对卡涅夫的法理对抗。

⑧ **卡达因与阿利提亚光复**  
魔道之国卡达因、木马骑兵、悲伤之地；与温德尔、乔治等名弓会师。阿利提亚夺还战让马尔斯短暂站上故土。

⑨ **黑骑士卡缪**  
格鲁斯特黑骑士团长卡缪：忠诚与正义撕裂的悲剧名将。马尔斯军与之决战，卡缪败而不失风骨，是系列「高贵敌人」原型。

⑩ **龙地与终局**  
马其顿天空骑士战、暗黑教主卡涅夫（需星光破伊姆霍鲁禁咒）、龙之血脉地、祭坛外传（那基等）、命运抉择终章：马尔斯持法尔西昂与火焰纹章击破梅迪乌斯。和平恢复，希达等结局分支；希达若死，终局马尔斯对妮妮的悼词更沉重（重制强化）。

⑪ **铁则与蝴蝶**  
永久死亡使「谁活到终章」成为世界状态机。契约者可救下原作可死角色、提前劝降，但不可无代价改梅迪乌斯需神器条件的规则，也不可让马尔斯无军团硬刚暗龙。

**【主要人物】**
- **马尔斯**（阿利提亚王子·领主）｜性格：温柔坚毅、重义。｜装备·能力：领主剑、后手法尔西昂。｜弧光：流亡→联合→解放→斩龙。｜立场：联军核心。
- **希达**（塔利斯公主·飞马）｜性格：勇敢直率。｜装备·能力：飞枪、翼枪系。｜弧光：出征伙伴→可成王妃。｜立场：马尔斯情感锚。
- **杰刚**（老骑士）｜性格：守护者。｜装备·能力：高初始低成长「杰刚榜样」。｜弧光：护送王子。
- **凯因 / 阿贝尔**（绿/红骑士）｜性格：对照搭档。｜弧光：阿利提亚双骑士传说起点。
- **奥格玛**（佣兵队长）｜性格：寡言可靠。｜装备·能力：剑。｜弧光：塔利斯佣兵核。
- **纳巴尔**（剑士）｜性格：冷面求强。｜弧光：被说服加入的杀手。
- **朱利安 / 莉娜**（盗贼/修女）｜弧光：救人入队的经典对。
- **马利克**（魔道）｜性格：友人学者。｜装备·能力：艾克斯卡利伯系。
- **哈丁**（奥雷利斯）｜性格：豪爽盟友（续作黑化伏笔，本篇为友）。
- **米奈娃**（马其顿）｜性格：武士荣誉。｜装备·能力：龙枪奥特克莱亚。
- **卡缪**（黑骑士）｜性格：悲壮忠诚。｜弧光：高贵敌对。
- **卡涅夫**（邪术师）｜性格：操纵暗龙。｜装备·能力：伊姆霍鲁。
- **梅迪乌斯**（暗黑龙）｜性格：支配。｜装备·能力：龙地顶点。
- **艾莉丝 / 妮妮 / 琪姬 / 戈托**等：姊、正统象征、神龙族少女、导师神龙。
- **诺尔内、雅典娜、那基**等重制追加：减员补位与外传战力。

**【势力图谱】**
阿利提亚解放军；塔利斯；奥雷利斯；格鲁斯特（敌后可挖角）；马其顿分裂；卡达因魔道；多鲁亚帝国与暗龙教团；阿卡内亚王室正统（妮妮）。

**【贵重物品】**
法尔西昂；火焰纹章（封印盾）；星光；伊姆霍鲁；艾克斯卡利伯；奥拉；格拉迪乌斯；墨丘利；帕尔提亚；奥特克莱亚；奥姆之杖；大师证；翼枪。

**【隐藏剧情 · 伏笔】**
哈丁日后黑化在《纹章之谜》；卡缪→泽利（新纹章之谜玩家角色线）；神龙族与地龙族仇恨；减员外传的「替补哲学」。本篇胜利≠永恒和平。

**【大事记时间线】**
安里斩龙→百年→卡涅夫复活梅迪乌斯→阿利提亚灭→马尔斯流亡→出征→火纹→中部→马其顿→卡达因→光复→卡缪→卡涅夫→梅迪乌斯→和平。

**【叙事基调 · 雷区】**
王道热血、对话偏古典。忌无视永久死亡；忌让领主单刷全图无代价；忌跳过神器条件硬杀暗龙；忌把卡缪写成无脑反派。最早切入：塔利斯出征或阿利提亚逃亡夜。

**【战役细目与可观察细节】**
网格上的森林回避、城门封锁、海盗船甲板、龙穴高温、黑骑士披风、飞马羽音、神殿铃、火焰纹章移交仪式的沉默。减员时外传村的新兵眼神、仓库空了的武器耐久、锻造屋账单，都是AI写「战争日常」的素材。

**【军团经营】**
战前整备换装、仓库、竞技场刷级、兵种转换上限、借将（联机时代）。契约者若只当独行侠，会与「必须凑齐职业链」的世界逻辑冲突。

**【终局状态机】**
是否已离塔利斯；是否已取火纹；米奈娃是否入队；卡缪是否已战；是否持法尔西昂；梅迪乌斯是否已灭；关键角色谁死谁活。

**【章节战役全表】**
序章详述阿利提亚陷落：科尼利厄斯战死、艾莉丝被俘、马尔斯经杰刚等护送逃塔利斯。C1出征清周边威胁。C2加尔达海盗战，奥格玛、巴兹、波德、科德入队，塔利斯佣兵核成型。C3「齿间」遭遇纳巴尔，剑士对决与说服是系列经典招募。C4草原推进至奥雷利斯。C5与哈丁及沃尔夫、塞德加、罗谢、维兰德等狼骑士会师。C6妮妮授予火焰纹章，流亡军升格为联合正统。C6x等外传在减员时开启，诺尔内等补位。C7勒夫坎迪斯隘口地形绞杀。C8沃伦港争夺补给线。C9皮拉西火龙预演龙系威胁。C10米奈娃线启动：为救玛丽亚与兄米谢尔对立。C11克诺达市场城镇战。C12无龄王宫宫廷与禁卫。C12x叛徒价值。C13木马骑兵攻城。C14悲伤之地。C15卡达因魔道绿洲，温德尔、马利克线升温。C16阿利提亚光复，马尔斯踏归故土。C17星与救世主。C17x赫勒拿。C18黑骑士团。C19玛那克特公主琪姬。C20卡缪悲壮一战。C20x火之巢。C21-22马其顿天空与骑士。C23卡涅夫，须星光破伊姆霍鲁。C24-24x龙地与祭坛。E命运抉择：法尔西昂与火焰纹章条件斩梅迪乌斯。

**【招募与永久死亡】**
莉娜被救→朱利安入队；纳巴尔可被说服；米奈娃因玛丽亚与荣誉加入；卡缪本篇不可劝降；戈托终局导师；琪姬神龙族关键。减员触发外传是规则：全员存活会错过部分角色，AI勿写成系统惩罚情绪。死者名字应在后续对话残留。

**【神器链与军团后勤】**
铁→钢→银→名武（格拉迪乌斯、墨丘利、帕尔提亚、奥特克莱亚等）→星光→法尔西昂。火焰纹章同时是政治诏书与封印钥匙。武器耐久、锻造、仓库、竞技场、兵种转换构成「战争会计」。领主马尔斯必须上场却又是优先被狙目标。

**【地理行军质感】**
塔利斯海风与渔港；登陆后的泥与焚村；奥雷利斯开阔骑战；隘口枪林；港湾侧袭；卡达因沙与塔；阿利提亚城墙缺口；格鲁斯特黑甲列阵公路；马其顿悬崖适合飞龙；龙地硫磺与骨堆。每处应决定兵种克制而非背景板。

**【名场面因果】**
自愿流亡的王子在塔利斯接受希达同行；火纹移交时全军改称；米奈娃拒做暴君之刃；卡缪说忠诚与正义不可两全；卡涅夫笑安里血脉只是影子；马尔斯以影子斩夜。希达若死，终局对妮妮的话更冷——重制强化了这条线。

**【与《新·纹章之谜》边界】**
本篇终局哈丁仍是豪爽盟友。其称帝、暗珠、影之英雄克里斯属下一档案，禁止在本篇E章剧透黑化。

**【可介入微观事件】**
假法尔西昂护身符骗局；村庄被山贼收税；劝降窗口只有对话格；治疗被狙导致连锁崩盘；飞马被弓成片点名；龙骑士俯冲破枪阵；神殿铃响时龙地门开。契约者可改变谁死谁活与局部胜负，不可取消「斩龙需神器」与永久死亡铁则。

**【军团编制示例】**
前排铠骑/重装；侧翼剑骑；后排弓与魔道；飞兵侦察；僧侣贴身；盗贼开门与开宝。职业链断裂时用兵种转换补洞，但转换有数量上限。

**【战力观感对照】**
村民与溃兵≈一阶；山贼海盗≈二阶；王国正规≈三阶；精锐骑士飞马≈三~四阶观感（本条不单开四阶）；卡缪米奈娃≈六阶；卡涅夫梅迪乌斯≈七阶。六七阶切入必须带军团与神器情报，禁止独行秒龙。

**【战役调度总则】**
AI落笔先定：本章胜利条件（歼灭/夺据点/突破）、在场具名角色至少三名、一件可损耗武器或可夺走的宝箱、失败后二十四小时内会少谁。禁止用跨世界套话。马尔斯的「温柔」表现为记名字与拒屠城，不是圣母光环无敌。

**【海盗章细节】**
甲板湿滑，斧比剑先手。奥格玛不鼓励抢人头，鼓励活着。巴兹的笑是压力阀。科德波德是劳动力与盾。此章教会玩家：佣兵也有纪律。

**【纳巴尔章细节】**
剑士站在「齿」状地形，像等一个值得交手的人。说服比击杀更难，因为击杀只需骰子，说服需要马尔斯的信誉。

**【火纹章细节】**
妮妮的手在抖，不是怕暗龙，是怕把正统交给一个几乎无国的少年。哈丁的笑声很大，像要用声音填满同盟的裂缝。

**【米奈娃章细节】**
龙枪奥特克莱亚的重量是「妹妹还活着」。米谢尔要的是马其顿天空只属于强者。三姐妹飞骑的三角攻击是后话伏笔，本篇先写姐妹失散。

**【卡缪章细节】**

切入身份/时点：阿利提亚陷落后的难民/塔利斯渔村青年，锚定马尔斯抵达塔利斯前后。

初始事件：港口挤满溃兵，有人高价卖「假法尔西昂」护身符；格拉追兵的斥候在打听金发少年王子。

开场白建议：「船板还在晃，岸上已经在收尸。你听见女祭司低声念安里的名字，像怕暗龙听见。渔网里缠着一面烧焦的阿利提亚旗。」

关键NPC立场：**希达**要你别挡路；**杰刚**盘问来历；**马尔斯**拒绝让平民送死；山贼头目要丁钱。

主线钩子/支线：主线=协助隐蔽王子一行；支线=送药、辨假货纹章、引开追兵。

危险度/规避：低~中。规避格拉正规骑士正面。

任务方向/奖励：口粮、情报、塔利斯通行证。

**二阶（塔利斯出征 · 新兵）**

切入身份/时点：塔利斯守备或奥格玛佣兵见习，锚定海盗讨伐章。

初始事件：加尔达海盗劫港，你被编入清剿队，隔壁床位的同伴可能在本章永久消失。

开场白建议：「号角响时，奥格玛只说一句：活着回来的才算佣兵。你握剑的手全是汗，对面甲板上的斧光比太阳还白。」

关键NPC立场：**奥格玛**要纪律；**巴兹**笑着让你别抢人头；**马尔斯**记你的名字；海盗头目要血。

主线钩子/支线：主线=护航出征；支线=救莉娜线前的村民、夺回被抢粮仓。

危险度/规避：中。规避纳巴尔级剑士单挑（除非剧情劝降）。

任务方向/奖励：铁剑、伤药、入队推荐。

**三阶（大陆正规战 · 士兵）**

切入身份/时点：奥雷利斯或阿利提亚残军士官，锚定火纹前后。

初始事件：隘口争夺战，狼骑士团从侧翼杀出；你的中队被要求「拖住直到王子到场」。

开场白建议：「枪林在晨雾里像移动的栅栏。传令兵喊哈丁大人来了，士气才从牙缝里挤回来。你看见火焰纹章的红在公主手中一闪，像整条战线忽然有了名字。」

关键NPC立场：**哈丁**要功；**妮妮**要正统；**马利克**要魔道书；敌将军要首级。

主线钩子/支线：主线=护送火纹；支线=劝降纳巴尔式强敌、保护补给车。

危险度/规避：中高。规避龙骑士俯冲与魔道范围。

任务方向/奖励：钢兵器、坐骑、尉官任命。

**六阶（名将战场 · 卡缪/米奈娃）**

切入身份/时点：联军精锐百夫长或马其顿叛将参谋，锚定黑骑士决战或米奈娃入队周。

初始事件：黑甲连队挡住整条公路，卡缪单骑叫阵；或米奈娃以妹为质的夹缝里要你选「救玛丽亚还是攻城」。

开场白建议：「黑披风落地无尘，像战争本身在行礼。马尔斯说：他是值得尊敬的敌人。你的枪却还是对准了他的咽喉——因为尊敬不能替死人呼吸。」

关键NPC立场：**卡缪**要格鲁斯特荣誉；**米奈娃**要妹与国；**米谢尔**要龙骑霸权；**马尔斯**要最小牺牲。

主线钩子/支线：主线=突破黑骑士团/促成米奈娃入盟；支线=保护帕拉三姐妹航线、夺奥特克莱亚。

危险度/规避：高。规避无策硬刚卡缪本体。

任务方向/奖励：银兵器、名马、将令旗。

**七阶（暗龙终战 · 卡涅夫/梅迪乌斯）**

切入身份/时点：联军核心干部，锚定星光破咒至龙地终章。

初始事件：卡涅夫以伊姆霍鲁免疫物理，星光未到手则全军砍不进；梅迪乌斯在龙地苏醒，法尔西昂未归位则斩龙失败。

开场白建议：「神殿的空气像烧焦的铜。卡涅夫笑称王只是安里的影子，而影子打不死真龙。马尔斯抬起火焰纹章，红光把你的盔面照成血色：『影子也足够斩开夜。』」

关键NPC立场：**卡涅夫**要暗龙永续；**梅迪乌斯**要支配；**戈托**要神龙平衡；**琪姬**要族运；**马尔斯**要结束战争。

主线钩子/支线：主线=取星光→破卡涅夫→法尔西昂斩梅迪乌斯；支线=外传减员补那基、救艾莉丝。

危险度/规避：贴近顶点。条件性胜利：神器链完整、军团减员可控。严禁宣称无神器秒龙。

任务方向/奖励：神器残光认可、阿利提亚勋章、战后领地秩序席位（非暗龙本体权能）。
