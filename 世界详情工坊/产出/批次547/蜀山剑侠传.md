# 蜀山剑侠传
<!--meta lib=主库 tiers=一、二、三、四、五、六、七-->

## 剧情

**【作品来源】**
还珠楼主（李寿民，字善基）创作的超长篇神魔剑侠章回小说，1932年起于天津《天风报》等报刊连载，延至约1948–1949年因战乱与时局两度辍笔，**终未完结**。系列包括前传《长眉真人传》（约24回）、正传《蜀山剑侠传》（常见称约309–329回，好读典藏等版本记本传五十集加后传五集共约三百廿九回）、别传《峨眉七矮》（9回）、后传与新传等，总字数约五百万言量级。文风文白夹杂，写景雄深、斗法瑰奇，融神话、志怪、剑仙与武侠，思想以儒释道合流为骨。后世影视（徐克《新蜀山剑侠》《蜀山传》、无线《蜀山奇侠》系、大陆《蜀山战纪》等）多有魔改——**本世界书以小说正文为准**，不取徐克「血魔丁引」等改编人名覆盖原著「血神子邓隐」。

**【世界定位】**
以四川峨眉为玄门正宗中枢的神魔剑侠世界：正道峨眉派与青城、五台、武当、昆仑等派，对魔道、妖尸、旁门左道，在「第三次峨眉斗剑」与「道家四九天劫」两大未竟主轴下，由「三英二云」「七矮」等年轻弟子学艺斩妖、开府护道。一句话：紫青双剑未合璧终局，斗剑与天劫尚未写到，仙侠仍在半途。

**【世界观 · 力量体系】**
本世界为还珠楼主式「剑仙—神魔」宇宙：力量来自内丹修真（炼精化气、凝运元神、孕育元婴）、外功行善积德、飞剑法宝、阵图禁制与天地灵药机缘；儒释道法术可互通，佛门神僧与玄门剑仙常并肩。死亡规则严厉——元神可兵解/尸解转劫，但天劫失败可至神魂俱灭；无系统面板、无现代热武器主导；斗法可翻江搅海、移山填谷，仍属「仙道人间」尺度，**勿写成爆恒星/毁星系**。

核心术语与成长逻辑：
- **元神**：灵魂与三魂七魄总称，主宰思想动作，须寄托肉身；魔道可炼三尸元神等邪种。
- **元婴**：道行高深剑仙体内灵胎，与元神合后大小由心，可自泥丸出窍，是脱体、护劫与飞升的关键。
- **身外化身**：额外元神，需**定珠/玄牝珠/雪云珠**等宝珠承载护持。
- **尸解 / 兵解**：功行或危急时脱壳；兵解借兵刃开颅隙使元神遁走，以便转劫再修。
- **合籍双修**：有缘眷侣并修，重道心与因果，非单纯色欲捷径。
- **天魔 / 阴魔 / 人造天魔**：魔道体系；阴魔以元命牌诱人出卖灵魂；人造天魔以元神生魂合精血炼成，极难灭尽。
- **飞剑与法宝**：御剑为剑仙标志；神雷、神火、神针、宝珠、阵图、灵兽蛊虫并重。正邪秘笈各有谱系（如正道《九天玄经》、邪道《血神经》《天魔神功》等）。

力量阶梯（按正文破坏力与仙道地位，宁低勿高）：
1. **江湖武者 / 未入道少年 / 村镇兵丁**：冷兵器与粗浅气功，踹门破寨级。≈一阶。
2. **初入仙门剑侠 / 半仙门槛**：辟谷、短距御剑、元神暂离体；「三英二云」学艺早期、普通门下。≈二阶。
3. **成名剑仙 / 小派护法 / 一域斩妖手**：飞剑成型、神光神雷雏形，可破妖巢、镇山门。≈三阶。
4. **一流剑仙 / 开府前后的后辈翘楚**：李英琼得紫郢与重宝后、余英男严人英齐灵云周轻云全盛中段、七矮精英；可参与大破邪巢、守府大战。≈四阶。
5. **散仙 / 地仙中坚 / 正教长老级**：元神或元婴稳固，御空长程，一怒可移山填海初现、夷平魔宫一角；如醉道人、追云叟白谷逸、矮叟朱梅、部分东海三仙门下长老。≈五阶。
6. **散仙地仙顶尖 / 准金仙门槛 / 魔道巨擘**：四九天劫相关、开府盛会可镇场；如极乐童子李静虚、白眉和尚、神驼乙休、绿袍老祖全盛、血神子邓隐、圣姑伽因量级。≈六阶。
7. **天仙 / 金仙目标 / 开山祖师与第一玄门顶点**：长眉真人（飞升后余威与遗宝）、妙一真人齐漱溟掌教全盛、第一神僧白眉之流；一怒毁国级战场、初掌规则皮毛。≈七阶。**世界峰值≈七阶**；书中「金仙圆满飞升灵空」多作目标与传说，未写到可稳定毁行星的八阶表现，**禁止虚高到爆星**。

**乐园阶位映射**（对照《阶位战力图鉴》，宁低勿高）：江湖武者/未入道≈**一阶**；初入仙门剑侠/半仙≈**二阶**；成名剑仙/一域斩妖≈**三阶**；一流剑仙/三英二云成长期≈**四阶**；散仙地仙中坚/正教长老≈**五阶**；散仙顶尖/魔道巨擘/白眉李静虚量级≈**六阶**；长眉遗泽·妙一掌教·准金仙顶点≈**七阶**。**世界峰值≈七阶**；第三次斗剑与四九天劫终局未写出，顶点以「存在·情报优先/条件性胜利」处理，禁止用非原作理由抹去全盛战力。

**【地理 · 舞台】**
舞台以巴蜀峨眉为轴，辐射中原、东海、苗疆、极地与海外仙岛：
- **峨眉金顶 / 凝碧仙府（凝碧崖）**：峨眉派本山与主开府地；长眉开山、妙一掌教、群仙盛会之核。
- **幻波池（依还岭万山中）**：圣姑伽因旧府，后李英琼开府；五行仙遁制阵、幻波心灯；清剿艳尸崔盈与守府大战的主舞台。
- **紫云宫 / 东海紫云相关**：二云等曾开东海紫云宫为峨眉行宫；另有紫云三女与破宫线，属势力扩张节点。
- **慈云寺**：早期正邪大战魔巢/邪寺；绿袍老祖等登场的名场面地。
- **天荙山灵峤仙宫 / 赤杖真人一线**：灵峤三仙（阮纠、甘碧梧、丁嫦）等圣地。
- **北极陷空岛 / 东海玄龟殿 / 铜椰岛·金银岛·月儿岛**：海外仙府与奇遇岛链。
- **天外神山小光明境 / 不夜城 / 静琼谷 / 小寒山 / 葬苍山 / 大荒山**：正旁门隐修与斗法点。
- **苗疆红木 / 百蛮山阴风洞 / 青螺峪**：苗疆与魔道边缘。
- **火云岭神剑峰阿修罗宫 / 万劫之门赤身魔宫**：魔宫主巢。
- **青城、五台、武当、昆仑、华山、崆峒、青海、雪山、黄山**：并列玄门或旁支地理；青城有极乐童子李静虚一脉重镇。

AI 写场景应先定「本山开府 / 幻波守府 / 慈云破寺 / 东海行宫 / 苗疆猎魔 / 海外寻宝 / 天劫外围」哪条链，再定冲突强度与可出场前辈。

**【世界剧情线】**

### 前史 · 长眉开山与血神孽缘（《长眉真人传》及正传回叙）
峨眉开山祖师**长眉真人**（任寿）与同门**连山大师**等立下第一玄门根基，传《九天玄经》一脉剑仙之学。同门师弟**邓隐**犯戒被逐，得《血神经》等魔籍，炼血影神光与血身，号**血神子**，成为峨眉心腹大患——长眉曾多次擒放，恩怨未了。长眉预言「**吾道之兴，三英二云**」，并留**紫郢剑**（雄）等重宝于机缘。长眉功成飞升/退隐于传承中，第二代以**东海三仙**为核心：**玄真子**、**苦行头陀**、掌教**妙一真人齐漱溟**及其夫人**荀兰茵**（入道前眷侣、入道后道侣），另有屠龙师太、晓月禅师、白云大师、餐霞大师、髯仙李元化、万里飞虹佟元奇、风火道人吴元智、坎离真人许元通、醉道人、元觉/元敬/元元/顽石诸大师等撑起门墙。第二代四大弟子**诸葛警我、笑和尚、申屠宏、阮征**已露头角。世界状态：正邪尚未总决，四九天劫周期在散仙地仙中悄然运行，峨眉蓄势待「三英二云」出世。

### 第一阶段 · 入蜀避祸与慈云寺乱（约第1–40回）
开篇：**李宁**（后拜**白眉和尚**为徒）携女**李英琼**等入川避祸，月夜棹舟、巫峡啼猿、栈道逢侠，结识醉道人等前辈，烟雨上峨眉。同期江湖与仙道交叠：多臂熊、蛇怪、色界天淫孽等妖邪横行。正邪在**慈云寺**爆发大规模斗法——正道有追云叟**白谷逸**、矮叟**朱梅**、醉道人、极乐童子**李静虚**、顽石大师等；邪方有**绿袍老祖**（百毒金蚕蛊、修罗幡、挖心吸血之恶名）、毒龙尊者等。大破慈云寺成为全书第一声号角，同时抛出「紫青双剑」「三次峨眉斗剑」预言与无数恩怨线头。**周轻云**（周淳之女，师黄山餐霞大师）等后辈开始学道辟邪。名场面：火焚色界天、大闹慈云寺、斩绿袍身（绿袍在此线受重创/被正道合力诛除于相关战局，余孽与魔道网络仍在）。阶段结果：正道小胜但魔焰未熄；李英琼尚在成长前夜；「三英二云」预言进入读者视野。世界状态：巴蜀仙道从「传闻」变为「可步行抵达的战场」。

### 第二阶段 · 三英二云聚义与紫郢出世（约第41–100回）
**李英琼**正式成为叙事重心：父李宁入白眉门下后，她孤身万里历险——古寺斗僵尸、擒龙得剑缘、斩巨人、马熊报恩、摘**朱果**、收**神雕佛奴**与猩猩**袁星**、诛木魃杀猛虎，终得长眉所遗**紫郢剑**（雄剑，紫青双剑之一），修为根基暴涨。**余英男**幼遭孤露，被恶亲逼出，冻骨后得救，转入峨眉，持**南明离火剑**、**离合五云圭**等，与英琼亲如姐妹。**齐灵云**（妙一夫妇次女）敦厚识大体，道心坚定，后辈尊「大师姊」，持**青罡剑**、**九天元阳尺**等，与白侠**孙南**有情缘。**周轻云**沉稳敏捷，持**青索剑**（雌，与紫郢成对传说）等，与**严人英**有三世眷侣缘。**严人英**（妙真观主**瑛姆**侄孙，师醉道人）稍后登场，持**银河剑**、**修罗刀**等，谦和英爽。五人合称**三英二云**，与**峨眉七矮**（**阮征**为首，**齐金禅**、**石生**、**易鼎**、**易震**、**甄良**、**甄兑**）及四大弟子构成峨眉新生代主力。李英琼陆续得**紫清神焰率兜火（兜率火）**、白眉禅师**定珠**（佛门定慧之珠，可护元神、照破隐形、承载化身）、前世挚友**圣姑伽因**所赠一甲子功力等，福缘之厚冠绝同辈。名场面：万里走孤身、玩梅花擒龙得宝剑、鬼哄森林飞剑斩妖。阶段结果：预言中的「三英二云」实体化；紫郢在手，峨眉后辈战力质变。世界状态：斩妖除魔从「前辈出手」变为「小辈可独当一面」。

### 第三阶段 · 东海行宫、紫云宫与异派纠葛（约第100–180回）
三英二云与同门出海陆四方行道：开**东海紫云宫**为峨眉行宫；穿插紫云三女小传；峨眉大破/经略**紫云宫**相关据点，扩张旗帜。**岳雯、陆蓉波、邓八姑（女殃神）、天狐宝相夫人、秦紫玲、秦寒萼、司徒平、廉红药、朱文、癞姑、女神婴易静、凌云凤、墨凤凰申若兰**等带艺投师或结盟。与**仙都二女谢璎、谢琳**及**李洪**等至交往来。旁门与亦正亦邪人物（**韩仙子、陷空老祖、水母矶璇、赤城子、青海教祖藏灵子、黑伽山主兀南公、鸠盘婆**等）时敌时友，因果网急剧膨胀。铜椰岛等事件埋下与**天痴上人、红发老祖、辛凌霄**夫妇等怨隙，遥指幻波池。名场面：海外仙岛夺宝、神火神雷对轰、行宫立旌。阶段结果：峨眉从「一座山」变成「多府联动的玄门集团」；小辈人脉与仇家同步暴涨。世界状态：第三次斗剑的「棋子」被摆上半张盘。

### 第四阶段 · 峨眉开府凝碧与群仙盛会（约第200–230回）
峨眉于**凝碧崖/凝碧仙府**大举开府，群仙毕至，近乎玄门阅兵与结盟大典。血神子**邓隐**等邪擘觊觎滋事，在开府线中被正道合力压制（资料称其在此线收场偏草草，但魔种与血神经余毒未绝）。小辈分派占据各洞府据点，「蜀山地图」插满峨眉旌旗。开府既是上半部总结，也是下半部「各自守土、分头渡劫」的起点。名场面：群仙朝金顶、神光映凝碧、魔焰犯山门。阶段结果：峨眉组织形态从师门变为「开府—行宫—分府」网络；李英琼等被赋予更重守土责任。世界状态：正邪由流寇式冲突转入阵地与府邸攻防。

### 第五阶段 · 幻波池三入、诛艳尸与英琼开府（约第230–300回核心）
**幻波池**线是后半部最完整单元之一：池为**圣姑伽因**旧府，擅五行禁制与乾罡神雷；衣钵弟子**崔盈**堕为**玉娘子（艳尸）**，盘踞为祸。峨眉众人**三入幻波池**，破解**五行仙遁制阵**，诛杀艳尸崔盈，补全圣姑因果。其后**李英琼**奉命开府幻波池，为池主，率同门布防。沙红燕、辛凌霄等来犯，更引**兀南公（丌南公）**、**九烈神君**等强敌猛攻；英琼困中参悟、功力再进，得老前辈暗助守府成功。名场面：三探幻波、心灯破魅、守府血战、开府立主。阶段结果：李英琼从「福缘最厚的小辈」升格为「可独镇一方的府主」；幻波池与凝碧并称峨眉双枢。世界状态：第三次斗剑所需的「后辈领袖」已成型，但总决之期仍被作者搁笔拦住。

### 第六阶段 · 苗疆与海外余波、天劫阴影（正传后段至后传/新传可引）
苗疆斗**红发老祖**等无长辈坐镇的恶战，检验小辈独立指挥；海外岛链、大荒二老（枯竹老仙、卢妪）、星宿老魔、欢喜神魔**赵长素**、妖尸**谷辰**/**穷奇**、雪山老魅等线交织。散仙地仙面临约四百九十年一度的**道家四九天劫**（天风、天魔、天雷等关），峨眉多方护法；地仙另有更漫长末劫之说。后传、新传、别传《峨眉七矮》补写七矮奇遇与旁支，但**正传承诺的「第三次峨眉斗剑」总决与四九天劫总收束均未写出**。名场面：苗疆毒瘴与神火对轰、护劫阵前、七矮别传笑中带刀。阶段结果：世界停在「大战前夜的漫长准备」；读者与契约者都面对一部**结构性未完结**的史诗。

### 七矮与旁支气口（别传可引·正传交叉）

## 阶位切入点

> 阶位↔境界：一阶≈江湖武者/未入道；二阶≈初入仙门剑侠/半仙；三阶≈成名剑仙；四阶≈一流剑仙/三英二云成长期；五阶≈散仙地仙中坚；六阶≈散仙顶尖/魔道巨擘；七阶≈妙一掌教·白眉·长眉遗泽级顶点。世界顶点（长眉遗威、齐漱溟全盛、李静虚、白眉等）为**七阶**，低阶切入一律**规避顶点·情报优先/条件性胜利**；顶点按原作全盛存在书写，禁止用非原作理由抹去全盛战力。

**一阶（凡俗入蜀·栈道啼猿与避祸夜）**

切入身份/时点：契约者以船工、栈道脚夫、避祸商户子弟、山村猎户或未入道香客切入，锚定**李宁父女入川、月夜棹舟至烟雨上峨眉前后**。

初始事件：巫峡夜猿声里，你替一条客船掌篙，舱中父女低语「峨眉」；岸上有人化名拦路，剑光一闪却被醉态道人一指震退——有人塞给你半块峨眉山路引，要你天亮前送到山脚茶棚。

开场白建议：「江风把猿啼拍进你的领口。船舱里的女孩还不到及笄，却把剑囊抱得死紧。岸上有人喊『李宁留下』，又有人笑『醉了也认刀』。你本只想挣一笔船钱，却发现船舷外的水，反着的不是月，是剑光。」

关键NPC立场：**李英琼**年少刚烈，可能把你当可托信的凡人；**李宁**戒备外来者；**醉道人**玩世，或随手点你一记吐纳；栈道盗匪视你为可灭口的目击者；峨眉外围香火道士只认香钱不认侠名。

主线钩子/支线：主线——是否把「李宁父女入蜀」的行踪卖给追兵或报给山门（蝴蝶：改变英琼早期遇险次数）。支线：①夜送路引避开毒蛇涧；②替客船瞒过一队假官差；③在茶棚辨认谁是绿袍外围探子。

危险度/规避：低～中。规避任何成名剑仙与魔头；见飞剑即伏地。无五阶以上贴脸。

任务方向/奖励：活着送达路引或保住父女一夜平安；奖励：凡俗伤药、粗浅吐纳口诀残句、峨眉山门香客腰牌（外围）、醉道人一句「你有仙缘」的戏言（非真传）。

**二阶（初入仙门·辟邪学剑与山门记名）**

切入身份/时点：契约者以峨眉记名弟子、黄山餐霞外围、青城跑腿或半仙门槛游方道人切入，锚定**周轻云学道、英琼拜山前后至三英二云未齐**。

初始事件：你被派去辟邪村送符，却撞见少女周轻云一剑斩蛇妖；回山路上又有人传「长眉紫郢将出世」，师兄要你盯紧所有「带紫光的剑」。

开场白建议：「山门的晨钟还没散尽，你的袖里已经塞进三道符。村里的妖气像潮，潮头上站着一个沉静的女童，剑尖一点，蛇血溅在你的新鞋上。她头也不回：『报师门，有人在找紫郢。』」

关键NPC立场：**周轻云**冷静可结盟；**齐灵云**厚道，愿指点阵基；**李英琼**火暴但重诺；**阮征**以七矮规矩察你；山门管事重纪律；旁门牙人会开价买紫郢消息。

主线钩子/支线：主线——紫郢出世流言中选择护密、泄密或假泄（蝴蝶：影响英琼得剑顺逆）。支线：①陪周轻云清理辟邪村余孽；②替齐灵云送九天元阳尺的剑油配方；③拒绝血神经残页黑市。

危险度/规避：中。规避绿袍余党头目与血神外围；不闯凝碧禁地。

任务方向/奖励：记名转外门/获得基础飞剑口诀；奖励：三阶以下飞剑胚、避邪符、同门人情、朱果「传闻图」而非真果。

**三阶（成名剑仙·斩妖建功与慈云余烬）**

切入身份/时点：契约者以成名游仙、小派护法、正道讨魔队成员切入，锚定**慈云寺大战余波至英琼得紫郢、收灵兽**。

初始事件：你奉命清扫慈云寺残殿，殿中百毒未散，有人低呼「绿袍的金蚕还在」；或你在森林鬼火里看见紫光一剑，少女李英琼正与妖人恶斗，神雕掠过你的头顶。

开场白建议：「残寺的香灰里埋着指甲盖大的金蚕壳。你本以为绿袍已死，壳却还在蠕。林子另一头紫光起落，像有人把长眉的名字重新写进夜空——你握紧剑诀，知道『成名』二字，从今夜起要拿命续。」

关键NPC立场：**李英琼**可收你为临时剑伴；**白谷逸**/**朱梅**考你胆色；**余英男**友善；绿袍残部要蚕壳要你命；**血神子**远端投影级压力（不宜正面对抗）。

主线钩子/支线：主线——慈云残毒与紫郢护持二选一深耕（蝴蝶：余毒是否流入苗疆）。支线：①助英琼收服受惊的袁星；②护送朱果情报到山门而不被截；③烧毁一份伪《血神经》抄本。

危险度/规避：中高。规避绿袍级余威与血神亲至；不独吞紫郢。

任务方向/奖励：斩妖功牌、准一流剑招指点、避毒丹、神雕掉落的一枚 equ 翎（信物级）。

**四阶（一流剑仙·三英二云并肩与东海行宫）**

切入身份/时点：契约者以一流剑仙、行宫护法、开府候选盟友切入，锚定**三英二云齐聚、东海紫云宫、得兜率火/定珠前后**。

初始事件：紫云宫外潮声如鼓，齐灵云命你布水线阵眼；潮里有人夺青索剑影，岸上李英琼兜率火一开，半边夜烧成紫金。

开场白建议：「东海不讲理的时候，先讲剑。你踩在行宫阵眼上，潮水倒竖成墙，墙后有人笑『峨眉也有今日』。紫火从身后掠过你的肩，李英琼的嗓音又烈又稳：『阵还在，人就在。』」

关键NPC立场：**齐灵云**委你重任；**周轻云**以青索剑意试你心；**李英琼**认战友极快；**严人英**重礼；**孙南**或白侠一脉可引为外援；鸠盘婆等旁门开价极黑。

主线钩子/支线：主线——行宫保卫战中守阵、夺岛或护定珠移交（蝴蝶：定珠是否提前入英琼手）。支线：①助余英男炼合离火剑意一夜；②识破假紫青双剑赝品；③护送圣姑旧物残简。

危险度/规避：高。规避血神子本体、李静虚全盛赌斗；与英琼抢宝必遭门规。

任务方向/奖励：行宫供奉级法宝零件、四阶飞剑强化、三英二云联名保举、兜率火「观摩一度」（非传承）。

**五阶（散仙中坚·凝碧开府与分府守土）**

切入身份/时点：契约者以散仙、地仙初成、正教外盟长老级切入，锚定**凝碧开府群仙会至分府插旗**。

初始事件：金顶钟鸣，你被邀观礼；魔焰突袭山门，邓隐血光在云上写字讨债；掌教命你守一侧崖门，不得退入禁中半步。

开场白建议：「开府的礼花是神光，礼花背后是血光。你站在崖门，听齐漱溟一句『此门即峨眉面目』。云上有人用血写旧日同门的名字——你忽然懂，仙府落成之日，便是债单到期之时。」

关键NPC立场：**齐漱溟**/**荀兰茵**以大局用你；**诸葛警我**等四大弟子考校军纪；**血神子邓隐**视你为可破的盟约薄弱点；**李静虚**可能远程驰援；小辈三英二云视你为「前辈尺度」。

主线钩子/支线：主线——开府夜选择死守崖门、诱敌入两仪微尘阵或护民撤山（蝴蝶：开府伤亡与舆论）。支线：①截获血神经运宝队；②为幻波开府预备阵基材料；③调停青城与峨眉座次争执。

危险度/规避：高。顶点齐漱溟、白眉若下场仅可**情报协同/条件性胜利**；禁止写其无故失去全盛。

任务方向/奖励：散仙护体丹、开府级阵图残页、五阶法宝、峨眉外盟长老名牒。

**六阶（散仙顶尖·三入幻波与诛艳尸）**

切入身份/时点：契约者以散仙顶尖、地仙、佛道双修高人切入，锚定**三入幻波池、诛崔盈、英琼即将开府**。

初始事件：你第二次入池仍迷在五行遁中；第三次，幻波心灯亮起，艳尸崔盈笑你「也想当圣姑」——池心李英琼紫郢出鞘，问你「助我，还是让开」。

开场白建议：「池水不是水，是五行叠成的谎。你每走一步，就有一个自己在镜里劝你回头。灯一亮，谎碎了，碎屑里站着艳尸与紫郢。李英琼的火烧在剑上：『第三次了。要么破阵，要么成池泥。』」

关键NPC立场：**李英琼**可结生死；**圣姑**余念/遗产考验道心；**崔盈**诱你堕尸魔；**齐灵云**/**周轻云**外围接应；**兀南公**已在远处盯上下一任池主。

主线钩子/支线：主线——破五行阵的路线选择（火克、双修合阵或佛门定珠照破）决定伤亡（蝴蝶：崔盈是否留下诅咒残核）。支线：①护送定珠入池心；②阻止旁门盗挖圣姑遗骨；③为英琼开府草拟守则三条。

危险度/规避：极高。血神子、九烈神君级若介入须条件战；圣姑级余威**存在·情报优先**。

任务方向/奖励：幻波池客卿令、五行遁入门、六阶灵珠、与英琼的开府同盟契。

**七阶（近顶点·幻波守府与天劫外围）**

切入身份/时点：契约者以近顶点散仙/准金仙门槛、正教可镇场长老或魔道叛出巨擘切入，锚定**李英琼开府后守幻波、兀南公等来犯，及某次四九天劫护法外围**；**第三次斗剑仍未开打（原著未载）**。

初始事件：兀南公攻府，九烈神君魔焰压顶，你被请为「对等战力」守东线；或天劫云轮在海外聚成，峨眉要你护法一名渡劫散仙——失败则神魂俱灭，成功则金仙路近一步。

开场白建议：「府旗在紫火里猎猎，像峨眉把半壁未来钉在一根杆上。云上有人叫战，云下有人渡劫。你知道书页在这里撕开了一道口：斗剑尚未写到，天劫却已轮到眼前。你抽剑，不是为写结局，是为让结局仍有人可写。」

关键NPC立场：**李英琼**已是府主，视你为同阶盟友或试金石；**齐漱溟**远程节制；**白眉**/**李静虚**可能各至一次；**邓隐**借劫生事；渡劫散仙把命托你；**长眉**仅余谶言与剑意，不可召之即来。

主线钩子/支线：主线——守府总攻夜或护劫三灾中择一主线深做（蝴蝶：幻波是否失守、某散仙是否成道）。支线：①保全紫青双剑传说不落入魔手；②阻止伪「第三次斗剑」提前开打的阴谋；③记录一份天劫法则供后辈（非完结剧透）。

危险度/规避：贴近顶点。与齐漱溟、白眉、李静虚、长眉遗泽级对等冲突须**情报优先/条件性胜利**（阵图、天劫规则、因果债、法宝相克等原作逻辑）；禁止把顶点写成无故失去全盛。世界峰值即七阶，无八阶可召唤。

任务方向/奖励：七阶观摩级剑意/神火一度、开府联防盟约、护劫功德名牒、史诗人脉（三英二云+掌教线）——不发毁大陆级外挂，不突破七阶上限，**不伪造第三次斗剑原文结局**。
