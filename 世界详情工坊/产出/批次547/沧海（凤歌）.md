# 沧海（凤歌）

## 剧情

**【作品来源】**
凤歌（本名向麒钢）所著大陆新武侠长篇，属「山海经」三部曲第二部（前接《昆仑》，后续《灵飞经》）。约 2006 年起于《今古传奇·武侠版》连载，后由重庆出版社、长江文艺等陆续分卷出版；2012 年前后有修订版（知音动漫 / 长江等，卷名如《海涯天劫》《东岛西城》《金刚法藏》《周流万物》），人物弧光、紫禁城之战、荒岛悟功、万归藏不杀陆渐之因等有增补，**本档案以修订版主线为准，旧版差异作脚注气质处理**。文风：双主角（仁 / 智）并进，商道、医道、抗倭史实、东瀛与远洋并入传统江湖；武学重「显脉 / 隐脉」「劫力 / 黑天劫」「周流六虚」等体系创新，格局大于一般门派恩怨。公开资料可核：[百度百科·沧海](https://baike.baidu.com/item/沧海/7855150)、[百度百科·陆渐](https://baike.baidu.com/item/陆渐/3128827)、[豆瓣书目·沧海](https://book.douban.com/subject/2032651/)；[搜笔趣阁检索·沧海](https://www.sobqg.com/searchBook.html?keyword=沧海) 2026-07 可作书目入口，正文以百科与分卷长评互证。

**【世界定位】**
明嘉靖年间，倭寇扰东南，江湖却另有两座「非朝廷」的庞然大物：**西城八部**与**东岛**。西城祖师梁思禽（昆仑主角梁萧之孙）曾以周流六虚功助复汉室、后因「抑儒术、限皇权」政见败走西域，于昆仑建「帝之下都」式西城，分八劲立八部，留八幅祖师画像，遗训「八图合一，天下无敌」——八图暗藏灭世级神器「潜龙」线索。东岛与西城百年宿怨。双主角：**陆渐**（东南渔家少年，憨厚仁厚，被炼为劫奴后以金刚一脉与黑天书破局，自创「海之道」）与**谷缜**（东岛少主，智计飞扬，蒙冤入狱后洗冤收复东岛，自悟周流六虚）。主线是：劫奴枷锁 → 九幽绝狱结义 → 八图与南京风波 → 抗倭与洗冤 → 天柱山大成 → 紫禁城双王对决 → 灵鳌岛「论道灭神」→ 环球寻宝 → 万归藏陨于天罚剑，东西城归于沉寂。主题：天道无亲 / 人道有情、「谐」与「不谐」、民生与权谋、仁智兄弟共破劫数。

**【世界观 · 力量体系】**
本世界为「人间武学 + 隐脉玄理」：无飞升成仙面板，有炼体 / 化气 / 炼神之境与神识渗透；死亡基本不可逆，劫奴与劫主的生死绑定是硬规则。破坏力天花板落在「一城可倾、海域可撼、神器可改天象」的人间顶点，**不写行星爆星**；梁思禽、梁萧属前史天花板，正文时代巅峰为万归藏、谷神通、后期陆渐 / 谷缜、鱼和尚量级。

力量来源与成长逻辑：
- **显脉与隐脉**：常人真气走奇经八脉（显脉）；**劫力**走三十一隐脉，由《黑天书》体系开掘。显脉主「真气」，隐脉主「劫力」，后期可互补循环。
- **黑天书 · 劫力 · 黑天劫**：花镜圆与风怜一系于东岛风穴参研隐脉所创，总纲「有无四律」：无主无奴、有借有还、无休无止、有往有来。劫主以内力打通劫奴隐脉，劫奴得四体通 / 五神通等异能，过用则发「黑天劫」（奇痒空虚、痛不欲生，须劫主真气化解；劫主亡则劫奴危）。陆渐被宁不空炼为劫奴；与宁凝互救、吸纳周流内力等过程中破四律，终成「补天劫手」与黑天神功。
- **大金刚神力**：金刚一脉单传，三十二身相 / 六代祖师本相，鱼和尚「大愚大拙之相」，陆渐「海纳百川之相」；可化法相生大力，久后化相存神，端坐伤人。与黑天书并称绝学，有「西城之主、东岛之王、金刚怒目、黑天不祥」之谣。
- **周流六虚功**：源自《昆仑》梁萧，梁思禽臻极致；以天、地、山、泽、风、雷、水、火八劲应自然，重「谐」忌「不谐」（不谐者易遭天损 / 六虚劫）。西城八部各传一支；万归藏自商道悟通、谷缜荒岛毒化机缘自悟兼修；「天无尽藏」等变招属同系。
- **剑意与家传**：姚家**断水剑法**；东岛**风穴剑意**（公羊剑意，承《昆仑》公羊羽归藏剑一脉余韵）；陆渐后期木剑断水与海之道合流。
- **商道 · 医道 · 望气**：万归藏以财神指环与商道布局半壁江湖；医理（花晓霜一脉隐脉典籍）与天子望气术等可越级影响战局。
- **神器前史**：「潜龙」为梁萧所造可扰天象 / 海流的灭世级机关；八图合一即寻潜龙。非日常可刷副本。

力量阶梯（按正文实战破坏力与地位，宁低勿高）：
1. **渔户 / 市井 / 普通倭寇兵 · 一阶初中期**：冷兵器、无内力或稀薄。
2. **江湖三流 · 一阶后期～二阶初**：门派外门、小帮打手、初级劫奴未稳。
3. **成名好手 · 二阶**：一方剑客、八部执事、东岛中坚、断水入门、初开隐脉。
4. **一流高手 · 三阶**：八部高手、东岛四尊外围、金刚身相初成、劫力可战群豪；姚晴、施妙妙活跃期多在此档上下浮动。
5. **顶尖一流 / 准顶点 · 四阶**：鱼和尚量级金刚怒目、沈舟虚天部主、宁不空火部、白湘瑶等；陆渐破黑天书后、谷缜悟周流前中期。
6. **人间顶点 · 五阶**：万归藏（西城城主 · 周流六虚 + 商道布局）、谷神通（东岛之王）、后期陆渐（金刚六相 / 海之道 / 黑天集大成）、后期谷缜（周流六虚兼修）、梁思禽前史传说。全力可倾城撼海、改一役格局；潜龙属前史 / 终局神器，不作常驻 NPC 战力。

**乐园阶位映射**：渔户市井～江湖三流≈一阶；成名好手 / 初开隐脉≈二阶；八部高手 / 东岛中坚 / 金刚身相初成≈三阶；部主级 / 金刚怒目 / 破黑天书前后的陆渐≈四阶；万归藏、谷神通、终局陆渐与谷缜≈五阶。世界战力天花板＝五阶顶尖，无超阶；前史梁萧 / 梁思禽可作传说背景，不写入局可直战的日常顶点。

**【地理 · 舞台】**
- **东南沿海渔村**：陆渐出身地；宁不空隐居牵连倭寇掳人的起点。潮声、渔火、简陋木屋是「仁」之底色。
- **东渡海路与日本列岛**：倭寇海船、桶狭间前后织田信长线；宁不空成谋士、阿市情线；陆渐为劫奴流离，遇鱼和尚。
- **东岛 · 九幽绝狱**：东岛权力核心与地牢；谷缜蒙冤被囚，陆谷结生死交之地。风穴与黑天书渊源相关。
- **南京**：姚晴化身「丑奴儿」盗地部画像后的追杀场；陆、谷、姚三人抗八部与东岛四尊，苦战逃离。
- **抗倭前线（戚继光线）**：谷缜与沈舟虚赌约、助戚继光重获重用的历史舞台；倭寇四大线索与洗冤并行。
- **天柱山**：陆渐破解《黑天书》、武功大成、金刚一脉得传的关键战场；谷神通现身、谷缜「假死」名场面。
- **紫禁城**：谷神通与万归藏惊天对决；谷神通与沈舟虚同归于尽；陆渐身世（母商清影、生父沈舟虚）揭晓处。
- **灵鳌岛**：九九之期「论道灭神」；梁思禽前史「有不谐者吾击之」石刻余威；群雄汇聚、谷缜收复东岛、与万归藏对峙。
- **西城（昆仑「帝之下都」意象）**：八部驻地、八图秘藏、万归藏权柄中心；商道与武学并重。
- **荒岛（艾伊丝弃谷缜 / 施妙妙处）**：谷缜将六虚之毒化为周流六虚功的机缘场。
- **远洋寻宝航线**：循八图环球远征；潜龙与天罚剑终局；锚起帆张、东方红日的收束意象。
- **大明宫廷与市井**：天子望气、粮价商战、临江斗宝等「武侠外」的权谋场，万归藏野心在此显形。

**【世界剧情线】**

**前史 · 昆仑余波与西城立基**
《昆仑》末，梁萧携花晓霜远走大洋，以格物与武学极致造「潜龙」——可扰天象、改海流的灭世级机关，为避战祸隐于沧溟。其子梁饮霜一系海外繁衍；孙**梁思禽**尽得祖父学识，周流六虚功修至极致，元末返中土。灵鳌岛上，梁思禽以周流连败东岛九大高手，石刻「有不谐者吾击之」，震慑海上百年。他助朱元璋驱元复汉，却因「抑儒术、限皇权」政见与帝决裂，携残部西遁昆仑，建恢弘巨城，武林比之东岛，号**西城**（「帝之下都」意象）。身边七弟子加贴身婢女共八人，梁思禽将周流一分为八——天、地、山、泽、风、雷、水、火——分授八部，共守西城。临终留**八幅祖师画像**，暗藏潜龙线索，遗训「**八图合一，天下无敌**」。东岛与西城由此结百年宿怨：一边是海上王权与旧恨，一边是西域巨城与祖师理想的异化。至嘉靖，西城城主**万归藏**以商道与周流并重，野心由「守遗业」滑向「倾天下」；东岛之王**谷神通**坐镇海上。黑天书一系（隐脉不祥）与金刚一脉（怒目单传）在江湖另成传说坐标。写前史时只作口碑与碑刻，勿让梁萧 / 梁思禽以满状态下场群殴嘉靖群雄。

**第一阶段 · 海涯天劫：渔村、劫奴与东渡**
嘉靖东南，倭寇如梳。渔家少年**陆渐**淳朴憨厚，识字不多却重然诺，与姚家女**姚晴**青梅竹马，曾跟学断水剑法皮毛。西城火部高手**宁不空**因故隐居渔村，一身烈火般的部传绝学藏于渔火之后。倭寇夜袭，陆渐因宁不空牵连被掳上船东渡——这是他人生第一次「被大海吞没」。途中宁不空为续战力、亦为控制，将陆渐炼为**劫奴**：强开三十一隐脉、种入劫力、钉死有无四律。陆渐初得四体通 / 五神通之利，亦初尝黑天劫的空虚奇痛——没有劫主真气，他就是一具会行走的劫数。至日本，宁不空阴差阳错入**织田信长**幕府，成为谋士，并与**阿市**生出情愫；桶狭间前后的东瀛乱局，是陆渐作为「工具人」目睹的第一场大国权谋。战后陆渐流离，遇明国金刚传人**鱼和尚**。鱼和尚见其仁心未泯，以自身寿元与功力续其命、传大金刚神力十六身相以压制黑天劫，嘱其携**舍利**返明，寻破劫正法，并严嘱金刚一脉「正眼法藏，横绝古今，所传非人则宁绝」。陆渐归途却失陷**东岛**，被投入**九幽绝狱**——与蒙冤被囚的东岛少主**谷缜**隔栏相对。谷缜智计飞扬、口吻刻薄心肠却热，被继母**白湘瑶**等构陷；两人一个仁、一个智，一个被劫力钉住、一个被权位钉住，结为生死之交。此段主题：仁者入劫、智者蒙冤；枷锁是成长的第一口井，结义是全书情感骨架。

**第二阶段 · 东岛西城：南京、画像与逃离**
狱外，姚晴为查陆渐下落、亦为撬动西城秘密，盗得西城**地部画像**（八图之一），化身「**丑奴儿**」潜入南京：白日华贵绝色、行动时易容成丑，反差本身就是轻功与胆色。画像案如投湖之石，西城八部与东岛四尊同时扑向金陵。陆渐、谷缜脱狱后与姚晴重逢，三人在秦淮夜色与官巷追杀中联手突围——这一战确立「陆谷姚」铁三角：陆渐硬抗、谷缜破局、姚晴牵线。逃离南京后，谷缜执着追查「四大倭寇」以洗冤，线索却屡被暗棋掐断；他转而与西城天部主**沈舟虚**立下赌约：助**戚继光**重获兵权、抗倭立功，则以军功与公义倒逼东岛还他清白。家国线由此楔入江湖线——「贼过如梳，兵过如篦」的民生痛感，是凤歌有意写进武侠的重量。白湘瑶算计加深，东岛内鬼与西城眼线交织；姚晴重伤被宁不空掳走；陆渐黑天劫将发，仍咬牙赶往**天柱山**。**施妙妙**进入谷缜视野：英气娇美，既是情缘亦是东岛政治的软肋。主题：八图从传说变成可盗、可追、可死人的实物；个人情义被整锅扔进东西城棋局。

**第三阶段 · 金刚法藏：天柱山大成与假死迷局**
天柱山是陆渐的成人礼战场。面对救人或自毁的极限，他以金刚身相硬抗劫力反噬，参透《黑天书》总纲与元体、玄用，**补天劫手**成型——非体非神、亦体亦神，隐脉与显脉开始自洽循环。后得浑和尚等补全另十六相，六代祖师本相可幻化，终至六相合一，「海纳百川」本相立。东岛之王**谷神通**现世，威压如海啸；谷缜却在众目睽睽下被父亲一掌「击毙」。陆渐目睹兄弟脑浆迸裂般的假象，悲愤欲绝——读者与他同悲，是本阶段必须守住的视角纪律。依谷缜「遗愿」，陆渐寻到谷缜的商道与武学之师、西城城主**万归藏**报丧。万归藏从容接讣，却借陆渐之手完成一场「脱劫」：枭雄借仁者之力卸下自身枷锁，野心从此脱缰。万归藏信「天道无亲、天道无私、天道无情」，商道上可翻手为云，武学上周流大成、天无尽藏；他对谷缜近乎父子，对陆渐则因分魔 / 神识渗透「你中有我」而多次无法痛杀（修订版专补此因）。主题：破劫即成人；假死拆散兄弟，却把棋盘从天柱直接推到紫禁城。

**第四阶段 · 周流万物：紫禁城、身世与荒岛悟功**
谷神通与万归藏百年宿敌，终在**紫禁城**摊牌。双王对决惊动庙堂：一边是东岛海上王权的孤注，一边是西城商武一体的倾国之志。战局尾声，谷神通与**沈舟虚**同归于尽——天部主倒下的同时，陆渐身世之锁应声而开：母**商清影**，与谷缜同母异父；生父竟是沈舟虚。兄弟相认不是爽文揭秘，而是两人对「父」与「王」的再定义：理解谷神通的苦心，却不必认同他的掌法；承认沈舟虚的血脉，却不必继承他的算计。万归藏野心至此彻底暴露——权倾天下、颠覆大明、以八图合潜龙，完成梁思禽未竟却已变质的「限权」幻梦。西财神**艾伊丝**奉命「临江斗宝」，商战即兵战；她挟持**施妙妙**，将谷缜与施妙妙弃于荒岛，意在折断谷缜智计与人心。荒岛上六虚之毒侵蚀脏腑，谷缜反以商道悟性与生死体验，把毒炼成**周流六虚功**——智者补上武学短板，八劲之「谐」第一次真正被他摸到。并行线：粮价操纵可饿死一城；东岛四尊内斗；西城八部有人愚忠有人投机；**宁凝**（沈舟虚所炼劫奴）与陆渐互救，有无四律出现裂口。主题：天道无情 vs 人道有情；兄弟各持「周流」与「海之道」的一半钥匙，才能开终局的锁。

**第五阶段 · 论道灭神与环球终局**
九九之期，**灵鳌岛**群雄毕至。「论道灭神」不是清谈：梁思禽石刻下，道与武、权与情、谐与不谐同时开考。谷缜以智勇收复东岛，清洗白湘瑶一系余孽，却在登顶时已萌生「解散」之念——他不要做一个新的谷神通。陆渐持金刚 + 黑天 + 海之道，与谷缜并肩挡在万归藏面前。危局中双方定下**寻宝决胜**之约：循八图环球远征，以潜龙归属定胜负，避免中土生灵涂炭于双城全面战争。远洋航线上谜团叠谜团——暗礁、异邦、八图拼接的星图与海图、潜龙苏醒的征兆。终局，万归藏倒在**天罚剑**下：周流六虚被黑天书等「不祥」之学破其「谐」的傲慢，剑与人有同沉海底之意象。枭雄之败不在技穷，而在「不谐」遭天损，以及他始终舍不得拉谷缜殉葬、不忍对陆渐下死手的一念之仁——仁恰恰是他理论里最不该有的变量。战后东岛、西城命运归于沉寂：谷缜解散东岛，携施妙妙远游气质；陆渐或暂摄天部 / 西城残局，终与姚晴等归于相对平静；锚起帆张，东方红日喷薄。主题：八图合一不是称霸，是把灭世钥匙沉回沧海；仁智双全，方破天劫。

**综合结局状态（供 AI 判断任意时点）**

**武学与江湖生态补述**
「西城之主、东岛之王、金刚怒目、黑天不祥」是江湖对顶尖战力的四象限谣言，写战斗时优先用此氛围，而非现代化战力表。同阶之内：周流重顺势借力与八劲相生相克；金刚重法相爆发与神意伤人；黑天重隐脉奇攻、感官神通与反噬风险；断水 / 风穴剑意重水势与风算。劫奴身份一旦暴露，会引来西城控制欲与东岛忌惮；八图碎片是全图最大通缉令。商道战场（粮、宝、指环）可与武戏等权——饿死一城的罪，不比一掌拍死人轻。土著（嘉靖军民、倭寇、织田家臣）默认不知「轮回乐园 / 契约者」——仅江湖与东西城逻辑。若玩家强索完整周流六虚或潜龙操控权，应付出「不谐」代价或被万归藏 / 谷神通量级碾压警示；陆渐可传断水残式与金刚入门心法气质，谷缜可传谋略与周流皮毛，鱼和尚在世时金刚全传极苛。双主角并存时，忌把戏份写成「陆渐无脑冲、谷缜永远正确」——仁会吃亏，智会算错，两人互相补锅才是原作节奏。

**【势力图谱】**
- **西城八部**：天、地、风、雷、山、泽、水、火分劲而治；城主万归藏总揽武学与商道；天部沈舟虚掌望气与庙堂接口，火部宁不空掌烈性战力与海外线。宗旨由「守护祖师遗业、八图秘藏」滑向「八图合一、权倾天下」；地盘含昆仑西城本体、中土密网、商路与粮道。八部之间可协作亦可倾轧，部主之位是四阶玩家最敏感的政治雷区。
- **东岛**：谷神通为王，修订版以**四尊**为柱（旧版五尊说并存时以修订为准），少主谷缜；九幽绝狱、风穴（黑天书渊源地）、海上舰队与岛链防务。与西城百年仇，内有白湘瑶式权斗。谷缜收复后选择解散，是势力线最大反转。
- **金刚一脉**：九如祖师以降单脉严传，六代鱼和尚、七代陆渐；中立慈悲，忌所传非人；战力符号「金刚怒目」，不站东西城永久队。
- **黑天书一系**：隐脉不祥之学，劫主 / 劫奴网络可渗透东西城与市井；四律是社会性枷锁，不只是个人 debuff。
- **明廷与抗倭力量**：嘉靖朝廷、戚继光军与沿海卫所；不知西城全貌，却是谷缜洗冤与万归藏谋朝的外部棋盘；兵锋可改变江湖舆论。
- **倭寇与东瀛势力**：沿海倭患、织田信长线；宁不空与陆渐东渡舞台；可写历史氛围，勿写成现代战争。
- **商道联盟 / 西财神**：艾伊丝等；斗宝、粮价、财神指环——武侠外的第二战场，饿殍与剑一样能杀。

**【贵重物品】**
- **八幅祖师画像（八图）**：梁思禽所留，合一则「天下无敌」/ 指向潜龙；地部画像由姚晴盗出引爆主线；持图者即全图通缉对象。
- **潜龙**：梁萧所造灭世级机关神器，可扰天象海流；终局寻宝核心，不可轻予玩家永久掌控或当刷材料。
- **天罚剑**：终局克制万归藏周流的关键兵器意象，与其同沉；象征「天道损有余」。
- **黑天书**：总纲有无四律 + 元体 + 玄用；开隐脉、生劫力、铸补天劫手；得书不等于免劫。
- **金刚舍利 / 身相传承**：鱼和尚以命所托，陆渐破劫与传法的信物；旁人强修易走火。
- **财神指环**：万归藏商道权柄象征；可调粮价与死士。
- **断水剑与木剑**：姚家剑道与陆渐海之道的日常兵器符号；木剑后期亦可断水。
- **灵鳌岛石刻**：「有不谐者吾击之」——梁思禽前史威压的地理铭文，论道场的「裁判席」。

**【隐藏剧情 · 伏笔】**
- 陆渐与谷缜同母异父、沈舟虚为陆渐生父——中前期严禁全知泄露。
- 谷缜天柱「死亡」为假死布局——陆渐视角应真悲，不可提前 OOC 安慰「他没死」。
- 万归藏不杀陆渐：神识渗透 / 心魔分二（修订版）——非简单圣母，是枭雄理论的自我裂痕。
- 八图合一真正指向是潜龙与「天道之谐」，非个人皇权永续；强行称帝即「不谐」。
- 周流重「谐」，万归藏「不谐」故遭天损——终局主题句，可在论道对白中回响。
- 黑天四律可被互救与双脉循环打破——陆渐 / 宁凝线是机械性伏笔。
- 《昆仑》梁萧出海与《灵飞经》后续——本世界可作开放边界，勿剧透灵飞经主线。
- 新版谷萍儿记忆等细节可淡化，以免与旧版读者记忆冲突；阿市线以东瀛阶段为主，勿强行拖入中土终局抢戏。

**【大事记时间线】**
- 前史（宋元～明初）：梁萧出海造潜龙；梁思禽返土、灵鳌败东岛、助明、建西城、分八部、留八图。
- 嘉靖初：倭患；宁不空隐渔村；陆渐被掳炼奴东渡；织田线；鱼和尚传功；九幽结义。
- 中期：姚晴盗图；南京逃离；抗倭赌约与戚继光复出；天柱破书；谷缜假死；万归藏脱劫。
- 高潮：紫禁城双王战；谷神通与沈舟虚同归；身世揭晓；临江斗宝；荒岛周流。
- 终局：灵鳌论道灭神；收复东岛；环球寻宝；万归藏陨于天罚剑；东西城沉寂；锚起帆张。

**【叙事基调 · 雷区】**
基调：仁智双雄、家国与江湖并重、枭雄可敬可恨、天道与人道辩题；抗倭有历史重量，勿戏说成无代价打闹；商道与民生（粮、倭、兵）与剑同样重要。雷区：忌把陆渐写成无脑开挂（他多次惨败近死）；忌把谷缜写成无底线阴谋家（面狠心善）；忌万归藏扁平脸谱反派；忌早泄身世与假死；忌把周流 / 黑天写成修仙飞升；忌忽略「贼过如梳，兵过如篦」；忌五阶以上战力与现代热武器体系；忌用潜龙开局秒天秒地。最早切入锚点＝东南渔村、陆渐尚未被掳或刚入劫奴之时。

## 阶位切入点

> 阶位↔武学：一阶≈渔户市井 / 江湖三流；二阶≈成名好手 / 初开隐脉；三阶≈八部高手 / 东岛中坚 / 金刚身相初成；四阶≈部主级 / 金刚怒目 / 破黑天书前后；五阶≈万归藏 / 谷神通 / 终局陆谷。世界顶点＝五阶顶尖；低阶切入一律规避顶点、情报优先。

**一阶（渔户 · 市井 · 倭患边缘线）**
切入身份 / 时点：契约者以「东南沿海渔村少年 / 船工 / 小商贩」身份切入，锚定陆渐仍在渔村、宁不空隐居未彻底爆发、姚晴尚未盗图的最早开局。
初始事件：夜里倭寇火光烧到邻船，有人喊西城高手的名号；你与陆渐一同被卷进逃散与救人，发现村尾那位「隐士」绝非善茬——他可能把你也当成「材料」。
开场白建议：「潮水退下去的时候，渔网里不止有鱼，还有刀。你听见邻船的哭喊，也看见陆渐那张还没被海风刮硬的脸——有人说，今晚过后，这村子就不会再姓陆。」
关键 NPC 立场：**陆渐**——可深交的渔家少年，仁厚但不知劫；**姚晴**——尚未化丑奴儿，护村护人；**宁不空**——高危隐士，视人为棋；村中长老与倭寇哨探——生存压力来源。
主线钩子 / 支线：躲倭 / 被掳边缘、接近宁不空真相、护送姚晴离开海岸；支线＝渔市换粮、给戚家军通风报信的最基层。
危险度 / 规避：中——规避任何八部高手正面、东岛来使、完整黑天炼奴仪式。
任务方向 / 奖励：活过开局夜、换到离村船票或门派引荐；奖励取向＝断水入门残式、渔村人情、抗倭情报。

**二阶（劫奴边缘 · 南京风声 · 初入江湖线）**
切入身份 / 时点：以「刚开隐脉的外围弟子 / 南京跑商护卫 / 东岛逃奴」身份切入，锚定陆渐已为劫奴或九幽将出、姚晴画像案发酵之时。
初始事件：你在客栈听见「地部画像」与「丑奴儿」的悬赏，西城眼线与东岛细作同一桌喝酒；有人塞给你半块与八图纹样相似的绣片，说「送出去就能活命」。
开场白建议：「南京的夜市比刀还密。你刚学会把真气压进一条不该存在的脉，就听见有人笑：劫奴也好，跑商也好，只要碰过那幅画，命就不值一吊。」
关键 NPC 立场：**姚晴（丑奴儿）**——可同盟的盗图者；**陆渐 / 谷缜**——若已结义则是最强变量伙伴；八部执事——追捕与招安双开；东岛四尊外围——杀俘不问缘由。
主线钩子 / 支线：护送或截画像情报、帮谷缜传倭寇线索、躲避黑天劫发作的同伴；支线＝商队护镖、假扮织田线回国密使。
危险度 / 规避：高——规避部主亲至、鱼和尚级交手、万归藏召见。
任务方向 / 奖励：在南京活着离开并换到「东西城都不完全恨你」的身份；奖励取向＝隐脉安抚药、断水剑中阶、八图残片情报。

**三阶（八部 / 东岛中坚 · 抗倭与天柱前夜线）**
切入身份 / 时点：以「西城一部执事 / 东岛供奉 / 戚军特勤剑客」身份切入，锚定谷缜洗冤抗倭、天柱山决战前的中盘。
初始事件：沈舟虚与谷缜的赌约传开，前线要一份能左右戚继光复出的「倭寇名册」；同时天柱方向黑气夜夜涌动——有人说陆渐的黑天劫要反噬半座山。
开场白建议：「军报与江湖帖子叠在同一张案上。你既要给戚将军一条活路，也要决定：明天上山，是帮劫奴破书，还是帮城主收狗。」
关键 NPC 立场：**谷缜**——洗冤急于求成，可结盟；**陆渐**——濒临大成，需护法；**沈舟虚**——赌约与算计；**白湘瑶**——要谷缜死；**宁不空**——仍握姚晴或劫主之权。
主线钩子 / 支线：抗倭名册、天柱护法或抢功、假死局的边缘知情；支线＝粮价商战初露、宁凝劫奴互救。
危险度 / 规避：高——规避谷神通 / 万归藏正面、完整周流八劲齐至。
任务方向 / 奖励：在家国与江湖之间站队成功；奖励取向＝一部真传残卷、金刚身相入门、抗倭军功换的朝廷虚衔。

**四阶（部主级 · 紫禁～荒岛 · 双王余波线）**
切入身份 / 时点：以「一部之主候选人 / 金刚旁支大成 / 西城商道总管」身份切入，锚定紫禁城战后、谷缜荒岛悟功、万归藏野心毕露阶段。
初始事件：紫禁城余波未平，圣旨与江湖追杀令齐飞；艾伊丝要你协助「临江斗宝」或弃岛封口，而陆渐持金刚怒目来问沈舟虚的遗物。
开场白建议：「皇城根下的砖还热着。有人说东岛之王死了，有人说西城之主刚刚才真正活过来——你夹在财神指环与金刚舍利之间，必须选一个说谎的方式。」
关键 NPC 立场：**万归藏**——招安或灭口；**陆渐**——追身世与公道；**谷缜 / 施妙妙**——荒岛生还后的变数；**艾伊丝**——执行刀；**宁凝**——四律破裂的证人。
主线钩子 / 支线：斗宝站队、寻谷缜下落、护或夺八图剩余、阻止粮战屠城；支线＝神识心魔初感（万归藏对陆渐）。
危险度 / 规避：极高——可与四阶交手，须规避万归藏全力、潜龙启动。
任务方向 / 奖励：在双王死后的权力真空里活成「第三势力」；奖励取向＝周流单劲、黑天玄用篇残页、财神支线权柄。

**五阶（顶点 · 灵鳌论道 · 寻宝终局线）**
切入身份 / 时点：以「与城主 / 岛主同级的隐世宗师 / 八图合一道的见证者」身份切入，锚定灵鳌岛九九之期或环球寻宝决胜。
初始事件：灵鳌石刻「有不谐者吾击之」在夜雨中渗血色；万归藏约陆渐、谷缜论道，八图已近合一，海上有人看见潜龙影子。
开场白建议：「沧海无边，岛却只有一座容得下「灭神」。你站在梁思禽的字前，听见万归藏笑：天道无情——而陆渐只是把木剑插在沙里，说：那人道呢？」
关键 NPC 立场：**万归藏**——终局枭雄，可谈不可信；**陆渐 / 谷缜**——仁智合体破局；**姚晴 / 施妙妙**——人性锚；东岛西城残部——为存续或为散。
主线钩子 / 支线：论道灭神站位、寻宝航线领航或破坏、天罚剑归属、东西城解散善后；支线＝潜龙是否启动的伦理抉择。
危险度 / 规避：极高——此阶即顶点对决，无更高可躲；禁止写超阶外援一键秒杀万归藏。
任务方向 / 奖励：参与终局并决定「沉海还是称霸」；奖励取向＝海之道残篇、周流「谐」之解、八图封印权、东西城遗产分配权——皆须付出血与「不谐」代价。
