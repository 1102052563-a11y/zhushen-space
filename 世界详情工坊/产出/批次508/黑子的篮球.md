# 黑子的篮球
<!--meta lib=主库 tiers=一-->

## 剧情

**【作品来源】**
藤卷忠俊原作少年漫画《黒子のバスケ》（中文通译《黑子的篮球》，台译《影子篮球员》，港译《幻影篮球王》），以高中篮球为题材。2008年末起于集英社《周刊少年Jump》连载（正式连载刊号多记为2009年2号起），2014年9月1日完结；话数单位为「第N节（Quarter）」，全275话+外传6话，单行本全30册，累计发行突破3000万册量级。前身短篇曾刊于《赤丸Jump》并获Jump十二杰新人漫画赏相关荣誉。关联作品：轻小说《黑子的篮球-Replace-》（平林佐和子著、藤卷插画，全6册）及其漫画化《Replace PLUS》；续篇短篇连载《Extra Game》（全2册/8话）；TV动画三季（Production I.G，2012–2015，共75话+OVA）及总集篇剧场版三章；2017年新作剧场版《LAST GAME》（Extra Game篇）。文风：热血竞技、团队羁绊、夸张招式与青春心理并重；比分与战术写得沉重，日常则轻喜剧。

**【世界定位】**
近现代日本高中篮球圈——以东京周边虚构强校为舞台的夸张竞技世界。主角黑子哲也是帝光中学「奇迹的世代」中「幻之第六人」，升入诚凛高校后与从美国归来的火神大我组成「影子与光」，立志让篮球重新变回「快乐」并击败已分裂的奇迹世代五人。世界没有修真、魔法或系统面板；冲突全部发生在体育馆、县预选、IH（全国高校综合体育大会）与冬季杯（W.C.）赛场，以及帝光时期回忆与街球对决中。所谓「变强」=特训、配合默契、ZONE心流与一场一场赢下来的经验。乐园任务 thrills 来自上场权、比分压力与「影子是否该被看见」，而非屠龙。

**【世界观 · 力量体系】**
本世界「力量」=人体机能×篮球技术×战术素养×心理韧性×团队配合，再叠加漫画式夸张招式与极少数「心流/预知」级表现。一切高光仍可还原为：弹跳与核心力量、脚步与变向、投篮手感与出手点、传球视野、防守选位、犯规控制、第四节仍能执行战术的心肺与意志；黑子的「无存在感」是极端人气与视线诱导训练的结果，并非真正的超自然隐身。成长路径清晰：①日常体能与基础技术；②位置专精——控卫重组织，得分后卫重投射，小前锋重全能，大前锋重篮板与终结，中锋重护筐；③招式体系——奇迹世代各有招牌（青峰「无形式篮球」、绿间「全场命中率」、黄濑「完美复制」、紫原「绝对防守与高度」、赤司「天帝之眼」与双人格）；④ZONE（心流）——极少数人进入后速度/力量/判断力跃升，火神可一战两次进入并曾触发「直接配合型ZONE」；⑤团队化学反应——诚凛「影子+光+外线+组织+内线」缺一不可。
规则骨架是日本高中联赛：都县预选→IH出线名额；冬季杯全国舞台；犯规五次离场、走步、干扰球、恶意犯规与罚球，每一条都能直接改写比分。伤病与「过度使用身体」是真实风险：黑子视力诱导过度会崩溃、青峰过劳、紫原与诚凛的肉搏、赤司手腕与帝光时期心理创伤等都会决定整场。
等级可按高中篮球层级理解（非官方修炼境界，仅为乐园映射用）：
1. 普通高中生/未系统训练者：体能一般，基本不会打正规比赛。
2. 校队边缘/替补：会基础运投传，上场易被高度与速度针对。
3. 县内常规主力：有稳定位置技术，能打满一节以上。
4. 都县强校王牌/IH常客主力（笠松幸男、日向顺平、木吉铁平等）：攻防有招牌，可左右一校格局。
5. 「奇迹的世代」级/全国顶级（青峰大辉、绿间真太郎、黄濑凉太、紫原敦、赤司征十郎、火神大我巅峰态）：身体与球商接近本作高中天花板，招式夸张但仍是人类运动员尺度的漫画强化。
6. 教练与体系层：相田丽子、白金须须美、原一彰、实秀等以情报、轮换与心理塑造队伍上限。
破坏力尺度：徒手竞技与身体对抗上限；冲撞、落地、手指与膝踝伤是主要风险；无毁建筑、无超音速、无远程异能。对照《阶位战力图鉴》，全体角色落在「凡俗段·一阶」——人类顶尖运动员级速度与力量，热武器与军队无关，重伤仍可致命。
**乐园阶位映射**：普通高中生~校队替补≈一阶初期；都县强校主力≈一阶中期；奇迹世代/火神巅峰/洛山体系≈一阶后期·巅峰。世界顶点（赤司征十郎·洛山与奇迹世代巅峰态）仍为一阶巅峰，不设超阶。篮球高中生整体≈一阶。

**【地理 · 舞台】**
主舞台锚定日本关东高中篮球圈。诚凛高校为近年新建篮球队的新兴校：旧体育馆、相田丽子执教、木吉铁平与日向顺平、伊月俊等元老撑起骨架，因黑子与火神入学而跃升。都内/周边权力版图清晰：海常高校（黄濑凉太+队长笠松幸男）是IH常客级外线强队；秀德高校（绿间真太郎+高尾和成）以投射与配合著称；桐皇学园（青峰大辉+若松孝辅等）以个人能力碾压；阳泉高校（紫原敦+大坪泰介等）以内线高度与防守墙闻名；洛山高校（赤司征十郎统帅，实渕玲央、叶山小太郎、根武谷永吉、黛千寻等「无冠之世代」）是冬季杯最强王朝级存在。帝光中学是前史圣地：奇迹世代与黑子共同夺三连霸的母校，教练白金须须美的「胜利至上」塑造了分裂的种子。生活侧舞台包括学校屋顶、合宿宿舍、街球场、医院与冬季杯主场馆；街球线延伸至美国风格的Jabberwock对决舞台（Extra Game）。势力地盘=各校篮球部预算、球探注意力与IH/W.C.名额；「全国」=夏季IH与冬季杯构成的日本高中篮球金字塔顶。

**【世界剧情线】**
① 诚凛立队与影子的光（开局至海常首战前）
帝光中学篮球社社员过百，其中五人被媒体与球探并称为「奇迹的世代」：赤司征十郎（控场与领导）、青峰大辉（无形式突破）、绿间真太郎（超高命中投射）、黄濑凉太（完美复制）、紫原敦（高度与绝对防守）。队中还有几乎无出场记录、观众与对手都记不住的「幻之第六人」黑子哲也——身高约168cm，身体素质在正式五人中垫底，却把「存在感稀薄」练成武器：用视线诱导把防守注意力引向队友，自己只做传球中继点，球几乎不在手上停留。赤司在中学时期主动开发这套影子打法，刻意不教黑子投篮，因为持球会抬高存在感。帝光以压倒性实力拿下中学联赛三连霸，代价是白金须须美式的「胜利至上」：弱者被边缘化，强者彼此较劲到失去笑容。中学末奇迹世代分崩离析，各自升入不同高中；黑子对「篮球已不再快乐」痛彻心扉，选择几乎无人知晓的诚凛高校篮球部——那里没有王朝光环，只有新建队的灰尘与旧木地板。
诚凛篮球社前一年才由木吉铁平、日向顺平、伊月俊、水户部凛之助、小金井慎二等创社，相田丽子任教练。第一年IH东京都预选曾挺入决赛联赛，却被当时的「三大王者」级强校轮流击溃，无缘全国。故事本年开始，美国归来的火神大我入学：身高约190cm，灌篮与盖帽天赋夸张，街头与美国训练背景让他习惯高强度对抗，却对日本高中联赛「节奏慢、身体软」一度失望。黑子在屋顶与体育馆观察到火神的执念——打败奇迹世代、成为日本第一——认定这就是下一个「光」。二人立约：影子让光更耀眼，光则带着影子去赢。日向顺平以队长身份用「每投失一球就折断战国武将手办」的残酷心理契约锤炼关键三分；伊月俊以冷笑话缓解更衣室气压，场上负责变速与组织；木吉铁平本是内线定海神针，却在季前重伤长期缺席，诚凛禁区一度只剩火神硬顶。水户部与小金井撑起板凳与气氛。诚凛从「新建校」变成「影子+美国怪物+元老射手」的不定时炸弹：训练吵闹、配合生疏，却第一次拥有能威胁强校的矛。

② IH都预选：海常·秀德·桐皇（第一次直面奇迹）
夏季IH东京都预选是诚凛冲击全国席位的唯一窗口。对海常高校：黄濑凉太以模特外表行走校园，球场上却能「完美复制」对手刚使出的招式，甚至在系列战中进化到可叠加多技；队长笠松幸男脾气暴烈、防守与气势顶级，专治黄濑的懒散与花架子。诚凛靠黑子无存在感传球撕开轮转，火神用暴力灌篮与盖帽换分，日向外线牵制，伊月串联节奏，最终险胜——黄濑第一次把诚凛记为「值得认真复制的对手」，笠松则记住了「那支新建队的眼神」。对秀德高校：绿间真太郎的投射命中率夸张到「进了射程就像进了」，高尾和成以「鹰眼」视野喂球做空接；诚凛用节奏变化、包夹触发点与日向关键三分咬住比分，火神冲击禁区迫使绿间离开舒适区，黑子在绿间视线死角完成致命传导。诚凛再胜，绿间的「绝对自信」出现第一道裂痕，高尾则开始真正把诚凛当决赛级对手。对桐皇学园：青峰大辉——黑子帝光时期的「光」——已把篮球打成「无形式」：不按教科书脚步与配合，只凭本能突破、变向与终结，若松孝辅等队友常常沦为看客。此战是全书第一次「影子与旧光」正面对决：黑子情绪剧烈波动，存在感与视线诱导失效，传球被读穿；火神独力硬刚青峰，体力与经验双双见底。诚凛败北，无缘夏季全国大赛。更衣室同时撕裂：天堂是「我们能跟奇迹咬全场」，地狱是「还差一整套团队与心」。赛后黑子与青峰的对白揭开帝光分裂的核：胜利至上榨干了快乐，青峰对篮球一度冷漠到只剩本能；黑子发誓要在高中证明「篮球可以再次快乐」——这句话成为后续全部特训与冬季杯的主题句。

③ 夏季败后特训与冬季杯预选·阳泉战
IH出局后，诚凛没有散伙。黑子针对「只会传球」的枷锁开发新技：消失的切入（让防守产生「人消失了」的错觉从而突破）、加速传球（以掌击球直线传出）、幻影射球（出手点与节奏欺骗，使球在防守视野中短暂「消失」）——后者让他取得故事中的首次正式得分，标志影子开始被看见。火神补强中距离与滞空控制，流星灌篮（罚球线起跳级滞空灌篮）成为对抗高大内线的王牌。日向练成「不容侵犯的射球」：不改射姿却把重心后移，制造「距离更远」的错觉。伊月与黑子磨合双组织；木吉铁平伤愈复出，内线协防与经验回补。对阳泉高校：紫原敦以身高与「绝对防守」封锁禁区，大坪泰介等构筑高度墙，紫原本人懒到能在场上走神，一旦认真则攻防一体极难撼动。诚凛用外线与黑子绕传拉开空间，火神以流星灌篮硬撼，日向惩罚收缩，木吉卡位换二次进攻。紫原被逼出认真模式后仍不敌团队轮转，诚凛胜出。此战舆论转向「诚凛能赢奇迹」；紫原也从「不想动」转向「愿意为值得的对手认真」——为后来Extra Game全员集结埋下性格伏笔。并行线：黄濑在笠松斯巴达管理下从花瓶变海常真核；绿间与高尾空接体系再进化；青峰被逼正视「一个人赢不了完整的队」；赤司征十郎在洛山统合实渕玲央、叶山小太郎、根武谷永吉、黛千寻等「无冠之世代」——他们没有奇迹名号，整体强度却被写成能压过多数「有冠」队，洛山成为冬季杯夺冠最大热门。

④ 冬季杯全国：再战海常·再战秀德·洛山决战
冬季杯把舞台拉到全国。诚凛再战海常：黄濑复制已进化到可连续叠加奇迹级招式，笠松把「复制」编进体系轮转；黑子祭出拟·天帝之眼——只能预判队友下一步，建立在长期信赖上——与火神首次稳定进入的ZONE（心流：速度、力量、判断力同步跃升）配合反制，诚凛再次险胜。再战秀德：绿间与高尾的空接与投射网更密，诚凛靠团队轮转、日向关键球与火神冲击再胜，绿间终于承认「有人能破绝对」。半决赛/决赛对洛山：赤司以天帝之眼预判全场行动，双人格在心理线上拉扯——帝光时期的冷酷「必须赢」赤司，与更早的温和统帅赤司交替主导；无冠之世代四人各司投射、速度、内线与补位，形成没有明显破绽的王朝阵。决战呈地狱难度：火神进入ZONE仍被赤司预判压制；黑子存在感消耗到极限，几乎无法再当影子；日向在关键回合扔进决定气势的三分；伊月完成最冷静的传导；木吉用经验换关键篮板与犯规控制。终盘火神触发「直接配合型ZONE」——ZONE的进化版，使配合对象（黑子）与自己完全同步——影子与光第一次在生理与战术上融为一体，诚凛逆转洛山，夺得冬季杯优胜，媒体称之为「奇迹的新星」。赤司败后双人格趋向整合；帝光因果在此战阶段性回收：不是影子独自打倒光，而是影子与新的光、连同非奇迹的队友，一起证明团队篮球可以战胜「胜利至上」的幽灵。

⑤ 帝光篇补完与正篇收束
正篇以回忆、对白与关键战役穿插，完整补完帝光三年：白金须须美的制度性压迫；赤司从温和到「必须赢」的人格分裂（天帝之眼与控制欲同源）；青峰对篮球失去兴趣后只剩无形式本能；绿间的绝对自信如何成为盔甲；黄濑「只会模仿」的自卑与复制背后的努力；紫原的懒散如何掩盖天赋；黑子被塑造成影子却始终在心里问「这样真的快乐吗」。正篇结局方向明确：诚凛冬季杯夺冠；黑子与火神的影子/光约定成立且被全国看见；奇迹世代各人重新理解篮球的意义（竞争、信赖、快乐可并存）；主要角色均在，无人死亡；伤病与心理崩溃是最大「死亡」隐喻。世界层面：关东与全国高中篮球圈多了一个永远被提起的「影子冬天」。

⑥ Extra Game / 剧场版 LAST GAME（街球终局）
正篇后的番外与2017剧场版：美国街球队Jabberwock受邀访日，对日本代表队Strky大比分羞辱，并留下将日方贬为「猴子没资格打篮球」一类挑衅。相田景虎（丽子之弟，组织/经纪线关键人物）极度不满，推动集结奇迹世代全员+黑子哲也+火神大我，组成梦幻队Vorpal Swords（V.S.）应战。Jabberwock以Nash Gold Jr.等街球顶级身体、技巧与场外心理战碾压常规高中阵容；V.S.则靠奇迹世代久违联手、黑子传球网络与火神ZONE完成复仇级胜利。此篇把「日本高中最强」放到国际街球尺度再测一次，招式与身体仍属人类运动员的漫画夸张，不引入超自然毁场；乐园侧映射仍为一阶巅峰团体战。主题回收是「曾经分裂的光与影子，可以再次站在同一边」。

⑦ 因果链与结局含义
帝光胜利至上→奇迹世代分裂与黑子「篮球不再快乐」→诚凛遇火神立约→IH胜海常、秀德却负桐皇→败后特训开发新技与木吉复出→冬杯破阳泉、再胜海常秀德→洛山决战整合赤司与夺冠→Extra Game国际街球验证。结局不是一人成神捧杯特写，而是：影子被看见；光学会与影子同步；非奇迹的日向、伊月、木吉、笠松、高尾等人证明体系与意志可托起天才；奇迹世代重新找到快乐。谁活着：主要角色均在；世界变成什么样：高中篮球圈的传说里，「黑子的影子」与「奇迹的新星」成为固定词条，后来者仍在IH与冬季杯的地板上追赶那个冬天的高度。

**【主要人物】**

初始事件：你在诚凛体育馆走廊差点撞上一个存在感低到像空气的蓝发一年级生——黑子哲也平静地说「请小心」，转头却把球精准传到正在暴扣的火神大我手中。日向顺平吹哨骂伊月俊的冷笑话影响训练；丽子把登记簿拍在你胸口，要你统计新人弹跳与折返。数日内，海常的笠松幸男带黄濑凉太来交流赛施压；又有传言桐皇的青峰大辉根本不认真打联赛。你必须在「申请试训／帮黑子做存在感掩护练习／给火神捡球并纠正中距离／向丽子提交海常侦察草图」之间抉择：试训成功进入12人名单边缘；帮黑子会被火神当成「影子的影子」；侦察草图则会在IH首战被日向点名表扬。此事件结束后，你已被写进诚凛的冬天。

开场白建议：「体育馆的灯管闪了一下，你才发现身边多了一个蓝头发的一年级生——他好像一直站在那儿，只是没人看见。篮下，红黑发的大个子把球砸进筐里，木地板震得脚麻。队长日向把计时器按停：『新人？先活过今天的折返——以及，别在火神灌篮时走底线，会吃球。门外好像停着海常的车。』」

关键NPC立场：**黑子哲也**——把你当可配合的「临时光/临时掩护」，语气平淡但记恩；破坏其传球节奏会立刻被礼貌疏远；**火神大我**——只认球场上的强度，请吃饭（大胃王）比说教有用，嘲讽其课业会引爆冲突；**日向顺平**——以训练态度与礼仪录用你，关键球信任极难拿；**伊月俊**——冷笑话测试你的耐心，组织局可深交；**木吉铁平**——伤愈前后都是定心丸，适合内线基本功；**相田丽子**——掌握出场建议与情报，实际权限极大；**黄濑凉太**——交流赛可见，会复制你的习惯动作作弄人；**笠松幸男**——骂醒型对手，尊重拼命者；**绿间真太郎**/**高尾和成**——秀德战接触，忌在赛前迷信争端里站队出错；**青峰大辉**——IH顶点压力源，校外约单挑多半被懒拒；**紫原敦**——阳泉战前可用零食换一句实话；**赤司征十郎**——冬杯决赛级存在，情报优先，禁止校外硬刚；Extra Game时**相田景虎**可拉你进记录组或热身陪练。

主线钩子/支线：主线蝴蝶点——IH海常战黑子存在感是否被你用掩护多续两分钟；秀德战绿间热区你是否提前提醒日向；桐皇战火神独木是否被你在暂停递上「传给黑子」的战术板；阳泉战紫原认真模式启动时机；洛山决战直接配合型ZONE时你是否在弱侧终结点；Extra Game·V.S.集结时你递水还是递情报。支线：①帮黑子练幻影射球并数命中；②替丽子做桐皇·洛山两份侦察；③凌晨陪火神练中距离；④给日向保管战国手办（心理责任重大）；⑤与伊月练挡拆发牌；⑥木吉复出后的对抗陪练；⑦海常交流赛被黄濑复制后的「反复制」习惯改造。

危险度/规避：日常中等（加练伤、校园冲突）；正式比赛高（冲撞、落地、手指与膝踝、黑子过劳性崩溃）。必须规避：模仿恶意犯规；怂恿伤员隐瞒伤情；校外群架「提前解决」奇迹世代。世界顶点**赤司征十郎／洛山整体／奇迹世代全员联手（V.S.）**为一阶巅峰团体战力：存在·情报优先／条件性胜利——靠轮换、体能分配、战术执行与关键球处理取胜，禁止校外单挑「削平王朝」，禁止宣称对方状态被任务或非原作手段压低。

任务方向/奖励：短期＝进入正式报名名单或经理情报组、获得垃圾时间分钟、学会一项可上场技术（弱侧底角、挡拆顺下、掩护质量、全场紧逼轮转）；中期＝在海常／秀德／桐皇／阳泉／洛山任一节点贡献可统计正负值或一次致命掩护／传球；长期＝见证或参与诚凛「影子被全国看见」的冬天，可选支线参与Extra Game记录组。奖励取向＝丽子战术笔记抄本、合宿名额、球鞋护具、都内球探名片、与诚凛主力的信任羁绊、冬季杯相关推荐线索——严禁发放毁场级异能或越阶功法，奖励上限停在人类运动员与情报资源。
