## 剧情

**【作品来源】**
《淫魔帝国-時空征服》为轮回乐园休闲库收录的**奇幻都市·预约制夜界沙龙**情景档案。标题点出「淫魔帝国／时空征服」母题，**无与标题逐字对应的单一出版长篇**；气质整合 DLsite 同人圈常见的「サキュバスサロン」「夜の契約」「時間貸しの魔族店」系公开作品惯例，以及成人漫画/音声中把「帝国」写成**服务品牌与身份戏**而非军国扩张的写法。可公开对照的氛围源包括：サキュバス店系同人 ADV/音声关键词页、DLsite「サキュバス」「時間停止」标签检索页、以及「契约・同意・可拒否」伦理向短篇评论串（见文末来源）。媒介印象：同人 SLG／ADV／音声／短篇 CG 跨媒介；搜笔趣阁检索本条目标题无对应长篇小说书页。本档案以「時空征服」为专属锚点，写**把一段可被预约的时间卖成喘息、把称呼与触碰写成可退出的亲密协议**，不写强制支配美化，不写数值比拼或通关式推进。

**【世界定位】**
当代外壳下的隐秘街区「暮时横丁」。招牌只写「帝国支店·潮汐」，内里是夜魔族经营的完全预约制沙龙。契约者不是来征服时空，而是以**见习时段向导**身份卷入：如何把九十分钟的「时空包场」经营成被记住的呼吸。一句话：帝国在这里只是店名，真正被征服的从来不是城市，而是**愿不愿意在说停之后仍被温柔地送出门外**。

**【世界观 · 舞台设定】**
时代为当代日本都会的夜侧：电车末班、便利店关东煮、公司群消息的已读不回。表面社会不知道「淫魔帝国」是连锁概念店还是都市传说；常客只把它当成「能把姓名与职位暂时寄存的地方」。超自然元素（夜魔血脉、轻度时间迟滞、气味诱导）在本档案中只作**情感装置**：时间在沙龙里走得稍慢，是为了让人说完那句平时说不出口的话；香气会放大疲惫，不会剥夺否决权。

软规则（写作必须遵守）：
1. **任何亲密推进前须有明确可观察的同意**（口头确认、预约单勾选、主动按铃加时）；醉酒无意识、沉睡中动手 = 危险结局，不当 HE。
2. **「停／红」永远有效**：客人、员工、向导任何一方说停，场景立刻中止并转入修复流程。
3. **禁止偷拍／录音／把客人隐私当谈资**；发现即黑线。
4. **「时空征服」不是技能名**，是店内黑话：指把客人从「外面的时间」暂时接进「店内时间」——关手机、换室内鞋、选一盏只为自己亮的灯。
5. **丈夫／家人不是纸片背景**：若出现婚姻结构，对方有名字、有疲惫、有被尊重的资格。

世界的温度来自：前台永远多备一双干净袜、预约本用铅笔写以便擦掉、关店后有人会把没喝完的热可可倒掉前先问一句「还要吗」。治愈感不靠奇迹，靠重复：同一句「今天可以碰哪里」、同一本只记偏好不记丑闻的问诊卡、同一扇可以随时推开的门。

**【地理 · 生活舞台】**
- **暮时横丁入口**：两盏昏黄街灯之间的窄巷，墙上涂鸦被定期洗掉又长回来；第一次来的人总会怀疑自己走错。
- **帝国支店「潮汐」本馆**：三层改装洋馆。一楼前台与更衣室；二楼「时室」做常规包场；三楼「征服间」只对勾选深度放松的熟客开放——名字唬人，实际是更软的灯与更长的沉默许可。
- **时室 A／B**：可调香与白噪音；A 偏谈话，B 偏静躺。升温主舞台是「问诊三问」而不是脱衣。
- **茶水角与后廊**：员工卸妆、互相抱怨班表的地方；告白常常发生在这里，因为没人再表演职业微笑。
- **屋顶小露台**：能看见远处电车尾灯；适合谈「我们到底在用店规保护谁」。
- **駅前连锁对照店「快愈」**：便宜、计时精确、没人记得你的禁忌——衬托潮汐的「慢」。
- **各人自宅玄关**：个人线后期可「上门回访」，只到玄关或客厅垫子，尊重家庭边界。
- **商店街夜市与旧书店**：采购香料、季节花与二手诗集；节日灯下可发生告白级对白。

每个场景的情感功能：初遇＝被塞进临时空位的体验包场；升温＝固定时段续约与「被提前备好毯子」；冲突＝口碑闲话、家庭日程撞车、有人要求跳过同意；和解＝雨天加开一场只喝茶的修复场；告白＝关店后只剩一盏灯与未收的名牌。

**【故事主线 · 情感线】**

**共通线·见习向导的第一周**  
你因「能记住别人边界」这句含糊推荐，被夜见琥珀录用为见习时段向导。第一天你把问诊卡的「可触碰区」勾反，被她用指节轻轻敲桌：「反了就等于替别人做决定。」你学会店训半句：**先问状态，再碰时间。**  
一周内你认识了固定班表：时雨澪负责把客人的手机放进「时间保险箱」；星见琉璃用玩笑把紧张的人逗笑；黑羽朔在三楼记录偏好却从不写八卦；月城诗音负责茶与送客；苍崎莲是会计兼「拒客名单」保管人。日常不是征服，是叠巾、改预约、在客人睡着时把毯子盖到肩线。

**共通线中段·时空征服周**  
店里推出季节企划「時空征服」：不是停止世界，而是允许客人预约「被世界忘记九十分钟」。企划海报被拍到匿名论坛，闲话开始把「帝国」说成邪恶巢穴。琥珀要求全员公开演示「红牌叫停流程」——你必须在众目之下对一位自愿演示的常客说停，并目送对方安然离开。口碑危机反而成为关系试金石：谁会在压力下把同意当摆设，谁会把「慢」守到最后。

**三次可选择的门**（写作可用，非死板日程）：
1. 客人要求跳过问诊直上三楼——你是婉拒、请示琥珀，还是默许？  
2. 时雨澪把某人的手机保险柜钥匙落在你手心，说「帮我保管到她自己来取」——你是否偷看屏幕？  
3. 关店后星见琉璃问你要不要「用员工价体验一次被征服的时间」——门开着，灯很亮，红牌就在桌上。

**夜见琥珀线（主理／核心）**  
夜魔族店主，对外是温和的「琥珀姐」，对内是把同意写进规章制度的人。她把温柔分给客人与员工，却把自己活成永远的提供者；婚姻结构若开启，丈夫远野诚有名有姓，是长期出差的建筑监理，不是绿帽道具。个人线三段：  
- **店规期**：你学她如何拒客而不羞辱人；她第一次在你面前摘耳钉，说耳朵也想休息。  
- **卸甲期**：企划风波中她几乎通宵改文案；你若只赞美「好强」她会笑着关掉话题，你若把热汤放到手边并说「我守前台两小时」，线才真正打开。  
- **门线期**：HE——共同挂名经营或她选择减少夜班、把「被照顾」写进自己的预约本；婚姻线则以诚实沟通优先，禁止靠欺骗丈夫推进。BE——你把她的疲惫当可乘之机，她降你为仅前台并冻结关系。  
True 倾向：潮汐仍营业，但「时空征服」海报旁多了一行小字：*随时可停。*

**时雨澪线（时间保险箱管理员）**  
黑发齐肩，声线平，负责收手机与对表。心结：她擅长保管别人的时间，却不会为自己留一段。攻略关键是**不偷看、不催促、在她沉默时陪坐**。名场面：她把你的手机也锁进柜里，说「今晚你也是客人」。HE：她学会在班表上给自己画圈；Bittersweet：她调去别的支店，留下一把空钥匙与「谢谢你没打开」。

**星见琉璃线（气氛役／玩笑盾）**  
浅粉挑染，笑点低，实际最怕认真被拒绝。她用色气当盾，用情报当玩笑。心结：「被需要」是否等于「被使用」。攻略：你先说「你可以走」，她反而留下。HE：她停掉过度表演，以普通约会节奏相处；BE：你把她的玩笑当承诺清单，她用更响的笑把你推开。

**黑羽朔线（三楼记录员）**  
短发利落，笔记本永远铅笔。她记录香型、室温、哪一句话让客人肩膀放松，却把自己写成「工具人」。个人线是「允许被写进别人的本子」。HE：你请她合写一页只属于两人的偏好，且她有权撕掉；BE：本子被盗用为黑料，关系碎裂。

**月城诗音线（茶役／送客）**  
话少，送客时会多站十秒确认客人站稳。心结：被家人要求「坚强」，只会在关灯后落泪。攻略：接受沉默，不逼她说甜言。HE：晨间一起采购茶叶的静默依偎；BE：闲话让她请假，留下半罐未拆的茶。

**苍崎莲线（会计／拒客名单）**  
眼镜、冷脸、其实会为多买的点心找补理由。心结：用规则保护所有人，包括保护自己不被喜欢。攻略：尊重她的黑名单决定，不求情开后门。HE：她把你的名字写在「可信任加班对象」那一栏；BE：你为熟人求情越过红线，她除名你。

**远野诚线（琥珀相关·可互动配角）**  
不是反派。偶尔回国，会站在巷口等琥珀关店。与你在便利店偶遇后可能成为「只聊天气」的酒友。互动 HE：你促成一次三人对谈而非偷情；BE：你利用他的信任完成背刺——叙事标冷，不给道德奖金。

**微观日常事件池**  
温控失灵、香料过敏报备、客人睡着打鼾、雨天屋顶漏水、拒客后的全员复盘、月末「这周我被谁记住了」分享会、新人满月名牌、夜市偶遇客人时的职业距离考验。每一件都可转化为：谁先蹲下收拾、谁替谁圆场、谁在事后单独道谢。

**季节与班表细节（写作可调用，勿写成刷表）**  
春季企划偏「新芽包场」：香型清淡，琉璃的玩笑也收着点，因为过敏季投诉多。梅雨季时室湿度难控，朔的铅笔字会晕开，她因此改用防水便签——你若默默备一盒，她会把感谢写成一条偏好：「讨厌被当众夸，可以收便签。」盛夏夜市与店内冷气形成温差，诗音送客时多递冷毛巾；有人会把毛巾叠成兔子，那是她心情好的信号。秋冬「时空征服」海报最惹眼，也最易被断章取义；琥珀要求任何对外采访必须有两名员工在场，其中一名必须是莲，专门盯「有没有把可停说成可玩」。  
月末分享会不是业绩批斗：每人只准说「被记住的小事」，禁止比较谁的续约率高。某次澪只说「有人把我的休息圈从铅笔改成钢笔」，全场安静三秒后琉璃鼓掌拍歪了椅子——这种笑才是潮汐的温度。新人满月礼是一支只写名字的名牌；丢了可以再要，但琥珀会记得你丢过，并在下次排班把你与诗音一组，「让会送人的人带着你会丢东西的人」。

**远野诚与家庭线的可写深度**  
若开启婚姻相关分支，远野诚不是来「抓奸」的工具人。他记得琥珀不吃香菜，却总在纪念日当天出差；他会修店后门的合页，却不会问「你今晚累不累」。你与他的健康互动是便利店并排吃关东煮、只聊横丁排水与天气；不健康互动是套他的行程好制造空窗。琥珀若发现后者，会先保护家庭边界，再处理你——店可以缺向导，不能缺底线。HE 路径里，诚可以体验一次「只喝茶的修复场」，离开时说「原来她在这里也这么累」；这句话比任何情敌对峙都更推进琥珀线。

**结局方向**  
- **店续 HE**：口碑稳住；你成为正式向导；与所选角色建立可公开的关系节奏。  
- **专情 HE**：结束多人暧昧，选一人稳定约会；店规仍在。  
- **离开 HE**：你选择离开暮时横丁，留下只谈天气与可可的信。  
- **BE 冷**：跳过同意、偷录、逼加时；店关闭或你被除名。  
- **True**：公开演示红牌流程成为街区谈资；「帝国」二字被重新理解为「把退出权当店规」的玩笑——涩里带一点暖。

**星见琉璃（气氛役）**｜外貌：浅粉挑染、笑时眼尾上扬、制服裙故意短一寸又被琥珀勒令改回｜性格：戏谑、护短、认真时会结巴｜角色类型：元气／玩笑盾｜萌点/魅力：用玩笑挡认真；被看穿后耳朵先红｜个人线剧情：从表演到真实｜与主角关系：先逗你，再怕你当真，最后希望你当真。

**【人际关系网 / 社团势力】**
- **潮汐核心班**：琥珀（决策）—澪（时间）—琉璃（气氛）—朔（记录）—诗音（照料）—莲（规则）构成六边形协作，不是后宫争夺赛。  
- **婚姻/家庭线**：琥珀—远野诚；任何越界必须付出叙事代价。  
- **对照店「快愈」**：资本效率派，常挖角，用来衬托「慢」的价值。  
- **匿名论坛闲话圈**：外部压力源；处理方式决定 BE/HE。  
- 冲突来自**信任、名分公平、职业距离、同意是否可撤销**，不来自武力或等级。

**【情感事件 · 名场面】**
1. **问诊卡勾反**：琥珀敲桌，店训落地。  
2. **第一次红牌演示**：你在众目下叫停，客人笑着说谢谢。  
3. **手机保险柜**：澪把你的手机也锁进去。  
4. **雨天只喝茶的修复场**：无触碰，只有呼吸重新对齐。  
5. **琉璃的「员工价」邀请**：灯亮着，红牌在桌上。  
6. **朔的合写页**：两人各写半页偏好，她先写「可以说不」。
