# 淫妖虫母巢觉醒-铃音融合

## 剧情

**【作品来源】**
「淫妖虫母巢觉醒」系列切片之**铃音融合**。骨架取自日本 TinkerBell（Cyberworks）成人向退魔/寄生系列《淫妖蟲》（In'youchuu / Inyouchuu）公开母题：退魔屋本舗·黒猫支店、寄生妖虫、蚀变与组织溃败、都市/岛/病院等舞台的虫潮战。乐园侧将其结构化为**六阶母巢觉醒阶段任务**之一。

**「铃音」并非必有的原作固定角色名**，在本切片中定义为：母巢选中的**人侧接口（Human Interface Node）**的任务代号/融合节点名——即「人格尚存、可对人类说话，同时已能调度虫潮」的那一个体。可理解为系列女退魔师在被寄生后被母巢提拔为接口的节点状态；叙事上可把接口宿主锚定为退魔一线人员（例如以**白鳥深琴**式剑斗退魔体质为模板的高亲和宿主，或**九重鈴子**式灵血亲和宿主的任务化投影），但**本档写的是接口个体本身**，不是「女皇」本体。

公开可核角色真名（任务中必须具名出现，禁止全用外号顶替）：
- **橘木ヤマト**（橘木大和）：黒猫支店侧主角级退魔师，半妖血脉被封印的复合体，队内决策轴。
- **白鳥深琴**：主力退魔师，雷光系刀技与突击；高亲和宿主候选之一。
- **白鳥武**：深琴双生姐妹，结界·札术·情报侧；冷静分析。
- **香山水依**：ヤマト青梅竹马，支援/治愈向退魔见习至正式。
- **夜（Yoru / 夜·葵）**：黒猫支店负责人，化け猫/鵺系上司，懒散却掌握旧封印与父辈债。
- 周边可调度：**月白桜花 / 月白菊花**（式神双生·诅咒线）、**桐嶋菫**、**野原小春**（教职/市民侧）、**桃野**家系医护侧等。

文风强制：**生物恐怖 + 组织战 + 都市防卫**。只写感染机制、退魔编制、隔离政治、声波/地铁虫网调度、斩接口/摘冠抉择。**禁止色情细节**；查无的原作章节、页码写「不详」，不编造出版页码。与同批次「女皇诞生」严格区分：女皇=都市级繁殖主权与纳贡编制；**本档=融合接口个体的戴冠/调度/人格叠合窗口**。

**【世界定位】**
近现代日本都市外表仍运转：地铁准点、便利店灯亮、电视台播报「局部卫生事件」。地下则是停车场—地铁廊—下水总干—净水厂取水口连成的**母巢廊道网**。退魔屋本舗·黒猫支店与对策科、警视厅特班、厚生省卫生特任组形成四层应对，互不信任。

**铃音融合**阶段的一句话定义：**她开口时，整座城的虫一齐转头。**  
接口个体（任务代号「铃音」）不再只是感染者，而是母巢写入「人语协议」后的调度终端：保留可辨识的人类声音与部分记忆，同时把街区虫潮密度、路径、停攻/改道当成可点名的指令。城市仍在，主权尚未完成物种更替——那是「女皇诞生」的后段；**本档停在「人巢边界」被撕开的九十分钟到七十二小时。**

**【世界观 · 力量体系】**
① **淫妖虫（寄生种群）**  
卵—幼体—工兵—战士成体—区域母体节点—母巢核心。感染途径在任务中简化为：创伤创口、污染水源、气溶胶雾团、已被接口点名的「同步街区」二次吸入。虫体不靠魔法发光，靠湿度、血铁、地铁振动频率繁殖。

② **蚀变**  
宿主体能上升、痛觉改写、嗅觉对同类虫液过敏或依赖、逐步接入群体网络。蚀变早期可伪装人类；中期出现指令延迟（筛查仪可读）；晚期人格溶解或被提拔为节点。

③ **退魔体系**  
黒猫支店：咒具、雷光刀（深琴系）、结界札（武系）、银线网、退魔装。夜掌握旧封印目录与「不可写进报告」的父辈债。对策科：生物样本、触媒安瓿、共生派/斩首派政治。警视厅特班：热武、戒严、对退魔不信任。卫生特任：净水加药、隔离许可、舆情口径。

④ **母巢意志**  
分布式。核心可迁移、可分芽，有冷却。不依赖单一大脑；但**人侧接口**能极大降低「向人类语言与城市基建翻译」的成本——所以母巢要铃音。

⑤ **融合体 / 接口体（本档顶点战力面）**  
保留部分人格的高阶节点。戴「融合冠」（虫丝编织的调度冠）后获得：**点名调度**（指定街区虫潮密度×3）、**声波指令**（占电视台/地铁广播）、**人语安抚**（对未感染市民说「请撤离」同时让虫让路或改道）。六阶地区灾难级，但**不是女皇**：不能长期编制全城纳贡，不能宣布物种主权；她是「键盘」，女皇才是「政权」。

死亡与再生：凡人感染后期人格溶解即「社会性死亡」；退魔师战死即终；接口被斩→母巢短时狂暴无主（城市更糟）或切换备用接口（需触媒与时间）；母巢核心被毁则局部网崩，可分芽再生。

战力阶梯（破坏力优先，宁低勿高）：
1. 市民 / 警员 / 未觉醒学徒 ≈ 一阶  
2. 初期感染者 / 见习退魔 ≈ 二阶  
3. 成体战士虫 / 正式退魔 ≈ 三阶  
4. 精英退魔班（深琴·武级） / 区域母体 ≈ 四阶  
5. 城区失守级感染节点 / 退魔本部临时指挥 ≈ 五阶  
6. **融合体铃音 + 母巢调度 / 戴冠后的接口窗口 ≈ 六阶（本档覆盖）**  
母巢本源全醒、女皇羽化 = 更高阴影，本档不写为可斩目标。

**乐园阶位映射**：市民≈一；初期感染≈二；成体虫/正式退魔≈三；精英退魔/区域母体≈四；城区失守级≈五；**铃音融合·接口戴冠·都市虫潮点名调度≈六阶**。宁低勿高。世界顶点（母巢本源/女皇）为跨阶存在，本档切入一律**规避本体·情报优先**；条件性胜利，严禁「战力归零式削弱」。

**【地理 · 舞台】**
- **隔离保护病房（市立中央病院地下 B3 特护）**：铃音被保护性隔离处。双层玻璃、负压、虫丝监测电极贴在太阳穴与腕内侧。走廊永远有消毒水与甜腥叠味。
- **退魔屋本舗·黒猫支店**：「夜」的据点。外表像老旧占卜屋/便利店后仓，内里是札库、刀架、封印目录柜。ヤマト、深琴、武、水依的行动起点。
- **对策科地下本部**：触媒库房、共生派会议室、斩首令打印机。白峰医官（任务具名）的安瓿从这里出库。
- **地铁母巢廊（○号线废弃侧线）**：虫丝冠编织处；戴冠仪式的「地下圣坛」投影。电梯在戴冠瞬间全市同步停在 13 层（民俗恐怖符号任务化，非超自然必然律，而是母巢对楼控系统的同步劫持）。
- **净水厂取水口与加药间**：感染路径之一；抑制剂投放点；市民「自来水能不能喝」的恐慌中心。
- **电视台主控室 / 备用发射塔**：声波战主战场。融合体用广播下群体指令；退魔残部用反向频率芯片干扰。
- **避难学校（市立第三小学）**：结衣教师（任务具名）护童线；道德标尺；儿童更易被接口声波同步。
- **黑市「虫药」巷（旧仓库区）**：壳（任务具名）卖抑制剂与假检测条；两头吃退魔与感染者。
- **警视厅特班戒严指挥车**：犬饲（任务具名）要戒严与对退魔问责。
- **家属会见室**：遥（任务具名，接口宿主的亲属/弟妹配置）是人格残片的软肋锚。

**【世界剧情线】**
① **融合前夜（T−24h～T0）**  
铃音被保护性隔离。退魔侧公开口径：「高价值感染者，可控。」监测却显示：她的脑波波峰与○号线列车进站振动、下水干线泵频同步。白鳥武把频谱图钉在黒猫支店墙上，夜只说一句：「有人在教虫说人话。」橘木ヤマト申请进入病房对话，被对策科以「污染风险」挡回；深琴在门外擦刀，雷光符未燃。

② **退魔内鬼与触媒（T0）**  
对策科出现「共生派」：主张与母巢签约换城市存活，代表医官**白峰**。斩首派以**黒崎**队长（任务具名，退魔/对策联合斩首班）为核心。内鬼向隔离病房注入**融合触媒安瓿**（库房编号可追）。触媒作用不是「变强」，而是打开「人语协议写入窗」——九十秒到三分钟，足够母巢把调度协议烙进接口。水依在加药间发现同一批号安瓿少了两支；她报警，报告被「卫生特任」压成普通损耗。

③ **戴冠仪式（T+0～T+90min）**  
母巢以虫丝在地铁廊编织「融合冠」。冠落地时，全市电梯同步停 13 层；电视台台标闪一下虫纹。铃音戴冠瞬间，她用人类声音说「别让我戴上」，扩音器里同时响起同一句话的虫鸣混响——**人还是巢**的名场面启动。虫潮不攻退魔本部，先切断补给线：弹药库电梯、净水加药车、黒猫支店后门的银线补给箱。深琴突击想斩冠，被点名调度的战士虫墙挡在 B2；武的结界在玻璃外裂成蛛网状，不是破了，是被「改道」了——虫不撞结界，绕去未结界的通风管。

④ **名场面：求死与笑脸虫潮（T+90min）**  
契约者若进入病房：铃音流泪求杀，同时窗外虫潮在玻璃上排列成她的笑脸轮廓。杀她 → 母巢短时无主狂暴，城市更糟，但调度中断；留她 → 城市成可点名的苗床；摘冠 → 需广播换气窗 90 秒 + 反向频率 + 近身，人格未必恢复。ヤマト会问：「她还在喊自己的名字吗？」武会回答：「她在喊，但地铁时刻表也在她心电图里。」

⑤ **声波战与三派会议室（T+2h～T+24h）**  
电视台被占。融合体用广播下群体指令：某区「请市民进入地铁」、某区「虫潮改道绕开学校」。表面像仁慈，实为把人流赶进母巢廊。退魔残部由工程师**三波**（任务具名）抢备用发射塔，投放反向频率芯片。频率过强震碎人耳膜，必须选空楼投放。黒崎要斩首；白峰要你签「共生停火协议」（附件含纳贡人口条款，必读附件）；夜与剃刀老顾问（任务具名，传统封印派）要大封印札封 B3 整层。三派会议室是政治副本：签字、撕协议、抢打印机、抢触媒编号单。

⑥ **结局方向（条件性，无廉价净城）**  
- **斩融合**：接口死，母巢暂乱，城市残；备用接口倒计时开始。  
- **断冠保人**：摘冠成功，铃音昏迷，调度半径崩，母巢怀恨；人格残片可能可谈。  
- **完全融合**：接口稳定，女皇前兆拉满，叙事可接「女皇诞生」切片（本档不展开羽化）。  
- **共生协约**：城市纳贡式停火，道德溃败；白峰胜利，黒猫支店分裂。  

⑦ **微观任务链（可写进回合）**  
隔离许可骑缝章、触媒编号连库房、净水加药记录、广播频率执照、虫药黑市价签、家属签字、电梯 13 层黑匣子日志、泪样两小时失效送检、反向频率芯片序列号。

⑧ **蝴蝶效应**  
可改：触媒是否注入成功、广播是否抢回、铃音是否被斩、遥是否被用作软肋测试、协议附件是否被公开。  
不可改：母巢要人侧接口；融合不可无代价逆转；禁止「一瓶药全城净化」；禁止六阶瞬间灭母巢本源。

⑨ **与「女皇诞生」焦点切割（写手铁则）**  
女皇诞生写：羽化台、纳贡广场、服从场、24h 甲壳骨化、编制军队。  
**铃音融合只写**：戴冠、点名调度、人巢边界、摘冠 90 秒窗、接口个体的泪与指令叠合。若正文出现「女皇宣布编制全市」，即串片，须拉回接口窗口。

**六阶（戴冠九十分钟 · 人巢边界）**

切入身份/时点：契约者以退魔外援、隔离病房警卫编入、频率工程师助手、或黒猫支店临时协力者切入，锚定**触媒注入后、戴冠仪式开始前至戴冠后 90 分钟窗口**（T0～T+90min 最佳；最迟 T+24h 广播战前夜）。

初始事件：市立中央病院地下 B3，病房玻璃内侧开始凝出虫丝冠的轮廓。任务代号「铃音」的宿主用人类声音说「别让我戴上」，扩音器同时送出同一句的虫鸣混响。黒崎把咒刃横在走廊：「帮斩，或让开。」白峰托盘里安瓿轻响，出示「共生停火协议」要签字（附件第 7 条未翻开）。门外：橘木ヤマト被挡，白鳥深琴雷光符未燃，白鳥武抱着三线频谱图，香山水依手心是触媒库房缺支复印件。夜的电话只给一句：「整点换气九秒十——去抢塔。」

开场白建议：「走廊消毒水味盖不住甜腥。负压灯绿得发冷。铃音的心电图滴答与○号线进站广播重合。黒崎的斩首令还带着热敏纸的温；白峰的安瓿碰撞声像倒计时。你听见表层的人声求死，也听见底层有什么东西在学她的嗓音说：戴上。」

关键NPC立场：  
**铃音**——求限制她/求死与调度指令叠合；  
**橘木ヤマト**——要对话与摘冠保人；  
**白鳥深琴**——先救后斩；  
**白鳥武**——要数据与窗口；  
**香山水依**——触媒链与净水；  
**夜**——换气窗情报换旧账；  
**黒崎**——斩；  
**白峰**——签协议；  
**遥**——救人软肋；  
**三波**——抢备用塔；  
**结衣**——护童；  
**剃刀**——封 B3；  
**冠丝**——逼融合；  
**犬饲**——戒严问责；  
**壳**——卖药。

主线钩子：阻止完全融合，或把融合代价压到城市可承受——**摘冠 / 可控斩接口+毁备用名单 / 拒签附件的有限停火** 三选一为主轴。  
支线：①截安瓿追编号内鬼；②护遥稳住人格残片；③反向频率抢塔；④学校疏散与校铃频率保卫；⑤公开或销毁协议附件；⑥亲和名单保全；⑦13 层黑匣子取证。

危险度/规避：极高。规避母巢本源本体与女皇羽化线；戴冠后正面对轰接口死亡率极高。严禁战力归零削弱顶点；条件性胜利见剧情清单。

任务方向/奖励：抑制剂、频率芯片校准权、封印札借用、隔离密钥副本、泪样报告、频谱图、触媒证据链、黒猫人情。不发一键净城、不发女皇首级。

【六阶执行细则·身份核验】入场出示乐园任务印与本地编外徽章/黒猫临时协力条。无条则先被犬饲或黒崎当间谍拷三轮——拷问内容即物证名录（安瓿号/附件/芯片序列），答错=敌对标记。

【六阶执行细则·物证优先】每推进一幕至少一件可验证物证。纯砍杀不推进。

【六阶执行细则·顶点交互】与母巢本源/冠丝对话默认记录原话与条件，不默认斩杀判定。

【六阶执行细则·队友红线】勿逼深琴无差别斩杀仍在求死的接口；勿向铃音承诺「可逆摘核变回完全人类」；勿在遥面前表演处决。

【六阶执行细则·失败态】完全融合稳定或附件第 7 条生效=更糟稳态，仍可玩，奖励降档。

【六阶支线池·1】校准武的频谱并当众演示脑波—地铁—泵频三线重合。  
【六阶支线池·2】与三波卡整点换气窗切备用塔。  
【六阶支线池·3】偷复印协议附件第 7 条并选择公开/勒索白峰。  
【六阶支线池·4】护送遥会见，换取接口指令延迟升高（更像人）的窗口。  
【六阶支线池·5】夜的目录涂黑行：用旧账换「冠的弱点在广播换气」完整版。  
【六阶支线池·6】壳处鉴定抑制剂真伪（底部封印墨）。  
【六阶支线池·7】结衣校铃频率保卫，防止儿童同步。  
【六阶支线池·8】触媒库房台账骑缝与隔离许可日期对账。

【六阶开场白变体】若跳过默认开场：「备用发射塔风很大。三波把反向频率芯片拍在你掌心：『九秒十。错过就听她指挥整座城。』楼下电梯显示屏全部停在 13。」

【六阶结算关键词】戴冠或摘冠、触媒、点名调度、广播窗、人格残片、协议附件、备用接口。

【六阶时序锚】T0 触媒/戴冠；T+90min 笑脸虫潮名场面；T+2h～24h 声波战与会议室；T+72h 完全融合或备用接口风险阈值。

【六阶对话样本】铃音：「别让我戴上。」黒崎：「斩令已打印。」ヤマト：「她还在喊名字。」白峰：「请看第 7 条。」夜：「虫在学人话。」

【六阶战场节奏】开场 30 分钟内卷入 B3 或抢塔；中段用物证推进；末段给条件性胜利判定而非斩首秀。

【六阶禁令复读】禁止战力归零削弱顶点；禁止跨世界套话任务；禁止无名群像替代具名 NPC；禁止串片女皇纳贡。

【淫妖虫母巢觉醒-铃音融合·切入补强】开局必须落到 B3 病房 / 备用塔 / 对策科会议室等具体地点与具名 NPC（ヤマト、深琴、武、水依、夜、黒崎、白峰、铃音）。写清报酬/风险/失败死亡率。顶点只给情报与条件。观察细节：徽章核验、物证编号、时序 T 值、结算关键词。与剧情段条件性胜利清单对齐。
