# 魔界骑士深渊大门-牺牲之门

## 剧情

**【作品来源】**
日本 Lilith Soft / Black Lilith 对魔忍（対魔忍 / Taimanin）宇宙的场景切片档案。系列自 2005《対魔忍アサギ》起，含《对魔忍紫》《阿莎姬 2/3》《雪风》《ZERO》、RPG/动作手游及 OVA《魔界骑士英格丽德》（Makai Kishi Ingrid）。本条目**不是**阿莎姬本传或英格丽德 OVA 情色线的复述，而是「深渊大门」系列中的**关闭协议切片**：当前置切片「大门开启」已使门环成形、Horde 潮汐常态化后，残缺协议解密出一条残酷规则——**稳定关闭或压回半开，需要高阶生命写入「牺牲钥」**。核心真名沿用系列：**英格丽德**、**埃德温·布莱克**、**井河阿莎姬 / 井河樱 / 八津紫**、**胧 / 弗斯特**、**桐生佐马斗**等。气质：骑士悲剧、组织算计、地区级灾难与条件性胜利；**禁止色情凌辱细节**，写钥槽、名册石、誓约、疏散与斩杀即可。资料可核：VNDB 英格丽德与《对魔忍紫》页、英文维基 Taimanin 条、ANN/MAL《魔界骑士英格丽德》、Lilith 产品线公开设定。

**【世界定位】**
近未来日本暗部与魔界（Dark World）渗透的双层世界中，本切片锁定**门环祭坛与关闭协议**。门已开（或半开永久化），城市仍在流血；五车学园/对魔忍残部、诺玛德观测、魔界骑士仪仗同时截获同一份残缺协议：关闭需要「自愿或被誓约强制的高阶生命」写入钥槽。非自愿写入成功率低且反噬更大——门会吐回尸体并加大污染。标题「牺牲之门」意指：**不是能不能关，是谁把自己写进钥里**。契约者六阶以钥祭司、诺玛德特使、五车敢死队长、反牺牲派间谍等身份入场；世界顶点仍是布莱克阴影＋门自体，合法胜利是**改写候选、毁替身骗局、压回半开、换取延迟**，不是门前秒杀古吸血鬼。

**【世界观 · 力量体系】**
人界与魔界由契约、血祭、企业通道与结界约束。对魔忍：血脉忍法＋对魔忍装＋忍刀；诺玛德：改造、药剂、光学迷彩、门务产品化；魔界骑士：魔装、魔焰剑、誓约与仪仗；深渊大门：通道协议，可污染、改写誓约、回收残滓。死亡：凡人/下忍即终；高阶可改造续命；**写入牺牲钥者**意识与灵核被门吃掉，难逆——这是本切片独有的「政治死亡＋灵核死亡」。

本切片独有的「牺牲钥力学」（写正文必须遵守）：
1. **名册石**：浮现候选真名；金字＝协议认可的高阶牺牲；血改名＝可强行替换但反噬与污染暴增。
2. **钥槽**：门环中央；写入需自愿誓约或强制誓约；伪自愿（精神控制）会被门识别并吐尸。
3. **替身批量**：黑市「伪自愿」技术，弗斯特系常参与；短期看似省人，实则加大 Horde 潮汐。
4. **誓约纹章**：英格丽德与布莱克、门仪仗绑定；她是「最贴门」的候选，也是最懂门礼仪的执行者。
5. **关闭≠和平**：即使成功写入，门可能只压回半开；完全关闭常是宣传用语。
6. **布莱克条件**：他要的往往不是门关，而是「谁愿意为秩序死」的数据与政治证明。

战力阶梯（宁低勿高，本档案主战场六阶）：
1. 凡人·疏导员·黑帮≈一阶。
2. 下忍·低阶魔物·替身贩子打手≈二阶。
3. 中忍·成型魔·仪仗兵≈三阶。
4. 上忍·阿莎姬级·诺玛德精锐干部≈四阶。
5. 城市级魔灾·骑士精锐联合≈五阶。
6. **魔界骑士全力、大门级灾难、牺牲仪式完成态**≈**六阶（覆盖）**。布莱克魔神化/深渊本源＝更高阴影，情报优先。

**乐园阶位映射**：凡人≈一；下忍≈二；中忍≈三；上忍≈四；城市联合≈五；**骑士/大门/牺牲仪式≈六阶**。顶点条件性胜利，严禁「被封印所以战力为零」。

**【地理 · 舞台】**
- **门环祭坛**：钥槽、名册石、仪仗火盆；六阶主舞台。气压随门脉冲起伏。
- **人界临时防线**：五车/对魔忍与神田旅团式编制交错；疏散走廊与拒马。
- **魔界侧回流廊**：骑士徽可通行；人界证件是死刑状。
- **诺玛德观测舰/特许港**：布莱克眼线与数据板；股东不在工地，在玻璃塔。
- **英格丽德誓约纹章投影厅**：可核验誓约真伪与「自愿」是否被改写。
- **平民避难区**：广播改口后的政治甩锅场。
- **黑市「替身钥」作坊**：弗斯特系技术外泄点；伪自愿生产线。
- **废弃线 B3 中继**（互文开启篇）：死区通讯；关闭夜仍是指挥命脉。
- **五车学园外围前哨**（时点许可时）：敢死队集结点；阿莎姬拒绝交人的政治前线。

**【世界剧情线】**

① **开启后的残局（承接大门开启）**  
门环已成形或半开永久化。Horde 第二、三波间歇涌出；凡俗军队推进仍是送菜。对魔忍把目标从「阻止开启」改为「止损＋找关闭条件」；英格丽德从「按表开启」转为「控门权」；诺玛德私门派则趁乱复制钥。布莱克的评估报告多了一行：「关闭成本尚未定价。」此阶段世界状态：地区灾难常态，协议残页开始在黑市与暗部同时流通。

② **关闭协议解密**  
五车对策科与诺玛德门务实验室几乎同时拼出残缺协议：关闭或压回可控半开，需要高阶生命写入牺牲钥。条款写明：自愿优先；强制誓约次之；非自愿成功率极低且反噬更大。英格丽德沉默——她比任何人都懂「门吃人」的字面义。阿莎姬的反应是：「把名单给我看，再谈谁去死。」公安暗部的反应是：「最好写对魔忍，别写政要。」

③ **候选名单三栏**  
名册石与纸质副本同步出现三栏：  
- **A 栏：英格丽德**——誓约最贴门，写入成功率最高，布莱克资产损失也最大。  
- **B 栏：井河阿莎姬**——人界象征；写入=对魔忍士气与政治双重崩盘。  
- **C 栏：替身批量**——黑市伪自愿；门会识别并惩罚。  
可选支线 D 栏：血亲/妹妹阴影（若前序「妹妹陷落」已发生）——用血亲换骑士的更脏算术。布莱克微笑：「选有趣的。」五车拒绝交阿莎姬；英格丽德不发言，手按剑柄。

④ **替身钥骗局**  
弗斯特系与黑市作坊出售「伪自愿」：精神控制＋伪造誓约纹章。第一批替身写入后，门吐回尸体，污染指数暴涨，第三波 Horde 提前。名场面：作坊灯火通明，数据板显示「成功率 92%」——门日志写的是「亵渎」。契约者若买替身，会被英格丽德与阿莎姬同时列为共敌。合法路径：端作坊、夺控制核心、公开罪证给可控开启派。

⑤ **名场面：钥槽点名**  
名册石浮现金字：**イングリッド**。英格丽德拔剑欲自书——骑士逻辑：纪律要求她成为钥匙。甲河胧残党与深渊教团（或私门狂热派）抢先用血改名，试图写成**井河アサギ**。门环爆炸级污染，六阶战场展开：骑士迷彩与黑焰清教团；阿莎姬光速剑清场；八津紫抢 B3；井河樱撤避难区。双方仍要面对同一句：**必须有人写进去，或必须找到协议漏洞。**

⑥ **布莱克的真正条件**  
加密指令抵达：证明「骑士比五车更懂牺牲」。这是政治，不是战术。布莱克要数据：谁自愿、谁被逼、谁用替身、谁敢改名册。英格丽德若自书，他会「惋惜资产」并记一笔忠诚；若她把阿莎姬推上去，他会更「感兴趣」。契约者可向他提交替代方案：伪信标压潮汐＋毁复制钥＋半开永久化，换取「暂不强制写入」——条件性胜利的谈判桌在玻璃塔，不在门心站桩。

⑦ **反牺牲派与协议漏洞**  
对策科「共生派」主张与母门签约换城市存活（脏）；「反牺牲派」主张找第二锚/辅钥，用工程代替人命。漏洞线索：门日志显示「双写入可互相抵消半拍」；或「仪仗火盆熄灭时名册石短暂可擦」。紫与诺玛德叛出技术官可协作。漏洞成功=延迟而非永生；失败=双倍污染。

⑧ **英格丽德弧光：愿死与不许亵渎**  
本切片核心不是俘囚，而是**她是否把自己写成钥匙**。她厌恶替身与血改名，因那是对门的亵渎；她也厌恶布莱克把牺牲当数据，却仍可能服从。可写分支：自书前被契约者拦下并改用工程漏洞；自书成功门压半开，骑士团缺位；拒书并斩替身作坊，换来与阿莎姬短暂共敌。软肋仍是忠诚——布莱克一纸评估可令她收剑或举剑。

⑨ **对魔忍弧光：拒绝交人**  
阿莎姬的底线：五车/对魔忍不是祭品配额。她可与英格丽德停火清教团，可派樱撤平民，可让紫扛数据，但不会把名簿上的自己或学员乖乖送进钥槽。她的胜利定义是「人还在、证据在、门至少不更开」。与开启篇一致：消防队长，不是热血偶像。

⑩ **终局方向（皆非世界和平）**  
- **骑士自书**：门压半开，英格丽德灵核重创或消失；布莱克记忠诚数据；人界惨胜。  
- **工程漏洞成功**：无人写入，门暂稳，三周后反弹（接续任务）。  
- **血改名被阻止＋替身网端掉**：候选回到谈判桌；政治战延长。  
- **阿莎姬被强写（坏结局向）**：对魔忍崩盘，门却未必关——强调亵渎无效。  
禁止无代价「门因封印消失」；禁止秒杀布莱克。

⑪ **可介入微观任务**  
护送反牺牲派技术官；端替身作坊；鉴定名册石金字真伪；抢修 B3；向英格丽德呈交弗斯特罪证；向阿莎姬呈交「双写入漏洞」残页；熄灭/守护仪仗火盆；销毁会被甩锅的录像；从钥槽冲刺擦名（三十秒回流规则仍适用）；玻璃塔递替代方案。

⑫ **蝴蝶边界**  
可改：谁被点名、替身网是否曝光、英格丽德是否落笔、漏洞是否成功、疏散伤亡。  
慎改：布莱克顶点地位、英格丽德忠诚内核、门吃灵核的规则、对魔忍拒交人底线。  
硬改核心→骑士通缉、诺玛德回收、乐园失败判定。

⑬ **名场面速查**  
名册石金字浮现；英格丽德剑尖抵钥槽又抬起；血改名时门环尖叫；替身被吐回的雨夜；阿莎姬：「五车不交人。」；布莱克评估：「牺牲数据完整。」；B3 绿灯与 Horde 第三波同步；伪信标诱潮入海，门日志记下操作者。  
任意时点三问：名册上是谁？替身网还在吗？漏洞残页在谁手里？

**【主要人物】**

- **英格丽德**（魔界骑士·诺玛德二把手）｜性格：严肃倔强、骑士荣誉；对布莱克绝对忠诚；厌下作与亵渎门务｜装备·能力：魔焰长剑、迷彩、五～六阶交界斩击；深层黑龙力量有灵魂代价｜弧光：门卫→门权棋手→**牺牲候选/自书者/拒亵渎者**｜立场：助她守门礼仪＝临时工具；推替身或血改名＝敌。

**【势力图谱】**
魔界骑士/黑龙一脉（纪律与门礼）；诺玛德（可控开启派 vs 私门/替身派）；对魔忍/五车（拒交人）；政府暗部（配额与叙事）；黑市替身网；深渊回流（非人）；反牺牲派（脆弱同盟）。敌友随名册与罪证切换。

**【贵重物品】**
牺牲钥坯；名册石拓片；钥槽校准钉；替身控制核心；伪誓约纹章；门日志关闭章；B3 模块；抗魔素药剂；英格丽德骑士徽；恶鬼六文；弗斯特实验记录；双写入漏洞残页；仪仗火盆种火；伪信标。

**【隐藏剧情 · 伏笔】**
关闭协议可能是布莱克压力测试；双写入漏洞或为门自体诱饵；血亲栏是否出现取决于前序切片；英格丽德自书后灵魂是否可被布莱克「资产维护」回收不详；半开永久化更适合长期任务书；门日志会记下谁拒绝牺牲——政治资产。

**【大事记时间线】**
大门开启/半开 → 残缺协议流通 → 候选三栏 → 替身骗局与吐尸 → 钥槽点名与血改名战 → 布莱克要数据 → 工程漏洞或自书或谈判 → 半开/延迟/崩盘残局 → 续写门务任务。

**【叙事基调 · 雷区】**
骑士悲剧与组织算计；口吻冷、可执行。忌色情；忌门前秒杀布莱克；忌把阿莎姬写成自愿祭品无反抗；忌替身成功无代价；忌「被封印门消失」。最早切入：协议残页同时到五车与诺玛德之夜；或名册石第一次浮现金字时。


**【关闭协议全文残页（可考据式重构）】**
协议自称「深渊接口稳定条款·第七附则」。可读残句包括：①高阶生命之灵核可作为负熵锚；②自愿誓约优先于强制；③伪自愿视为亵渎，惩罚系数为污染×2；④双写入若相位相反可短暂抵消；⑤股东（未具名，门务圈解读为布莱克级）保留评估权。残页缺页码，真伪需与门日志交叉。契约者伪造完整协议=政治核爆，也可能被门直接反噬。

**【牺牲候选的政治经济学】**
英格丽德＝高成功率＋高资产损失＋骑士叙事完整。阿莎姬＝中成功率＋对魔忍崩盘＋人界宣传灾难。替身＝低成功率＋污染惩罚＋黑市利润。血亲＝叙事最脏、布莱克最「感兴趣」。公安暗部内部报价单甚至给「牺牲单位」标了预算码——六阶任务可以偷这张表，不必先拔剑。

**【门环祭坛的微观工程】**
