# 対魔忍RPG-对魔忍全堕
<!--meta lib=主库 tiers=五-->

## 剧情

**【作品来源】**
日本成人游戏品牌 Lilith（BLACK Lilith / Anime Lilith）旗下「対魔忍（Taimanin）」系列衍生线上作品《対魔忍RPG / 対魔忍RPGX》（2018 年 9 月 25 日于 DMM 上线，RPG 为全年龄向、RPGX 为成人向）所延伸的「全堕」任务世界。系列原作自 2005 年《対魔忍アサギ》起，历经《対魔忍アサギ2》《対魔忍ムラサキ》《対魔忍ユキカゼ》《対魔忍アサギ3》《対魔忍アサギZERO》及卡牌作《対魔忍アサギ～決戦アリーナ～》等，人物原案カガミ、企画笹山逸刀斎。RPGX 以决战竞技场重建后的平行世界为舞台，玩家视点为五车学园一年级生风魔小太郎。本档案锚定「对魔忍组织系统性溃败、核心成员接连陷落」的全堕线，文风硬核黑暗、组织崩坏与人体改造并存，服务轮回乐园常青读取。

**【世界定位】**
近未来日本被魔界渗透后的暗战地球。政府暗部组建对魔忍，以忍术与粒子兵器对抗魔族、诺玛德等邪恶势力；五车学园培养下一代。本世界处于「全堕」临界：东京王国与地下都市尤米哈拉的黑暗经济吞噬前线，胧与诺玛德的连锁陷阱使井河浅葱、水城雪风、八津紫等顶梁柱相继失陷，风魔小太郎与残存小队在学园与沦陷街区之间挣扎。契约者切入的是「组织尚未彻底覆灭、但已开始批量失去王牌」的五阶城市级崩坏期。

**【世界观 · 力量体系】**
力量本源有三：忍法（血脉与修炼共鸣的粒子操控）、魔族血统与契约、人体改造与药物（混沌竞技场、诺玛德实验）。对魔忍穿特制紧身衣强化粒子传导；魔族以肉体强度、异能与魔界规则碾压凡人。死亡规则残酷：低阶对魔忍与士兵阵亡即止；被改造、刻印、洗脑者即使存活也往往人格崩坏；魔族高层可凭借魔核、分身或契约残存，但并非无限复活。

等级表现（以破坏力与战场角色为准）：
1. **普通人 / 警员 / 学园杂务**：冷兵器与制式枪械，街区巷战级。乐园观感≈一阶。
2. **见习对魔忍 / 下忍 / 黑道打手**：初开忍法或强化体术，可秒杀普通人小队。≈二阶。
3. **正式对魔忍 / 中忍 / 魔族杂兵头目**：裂石断金、粒子刃与中距离术式，端据点级。≈三阶。
4. **上忍精英 / 魔族将校 / 改造体勇士**：摧毁建筑群、硬抗常规热武器，一城仅数人。≈四阶。
5. **传说级对魔忍（浅葱位阶）/ 魔族领主级干部 / 诺玛德执行层**：一击夷平街区、领域雏形或大范围术式，可左右东京级战场。≈五阶。
6. **魔界大公 / 古魔 / 系列终局幕后**（本任务世界不作为常规可击杀目标开放）：地区至战略级。≈六阶及以上，本档只作情报顶点。

乐园阶位映射：普通人~见习≈一阶；正式对魔忍≈二~三阶；上忍精英≈四阶；浅葱·雪风·紫级传说战力与诺玛德执行层≈五阶；魔界大公与系列幕后古魔=本世界情报顶点（规避本体·条件性胜利）。宁低勿高；本条目仅覆盖五阶切入。

世界顶点在本档表现为「已觉醒并介入人间的高阶魔族与诺玛德决策层」——不可用封印削弱解释其战力，只能情报、离间、截断仪式与撤离。

**【地理 · 舞台】**
舞台以日本列岛都市圈为核心，暗线连向魔界裂隙。
- **五车学园（Gosha）**：对魔忍培养与指挥枢纽，宿舍、训练场、地下资料库与紧急避难层齐全。全堕线中学园表面维持教学，地下已转为收容、审讯与残存指挥所。
- **东京 / 东京王国**：表层为现代首都，底层为犯罪与魔族共生的「东京王国」地下社会，赌场、改造工房、人口买卖与情报黑市交织。
- **尤米哈拉（Yomihara）**：地下都市，魔族与人类罪犯混居，诺玛德与叛忍据点密集，是雪风篇与 RPGX 多事件的黑暗中枢。
- **混沌竞技场（Chaos Arena）**：地下角斗与人体改造表演场，浅葱早期噩梦的延伸；全堕线中成为「俘获对魔忍公开处刑与改造」的象征舞台。
- **诺玛德（Nomad）设施链**：表面跨国企业，实为魔族渗透的军火与生物兵器网络，实验室、收容舱与传送锚点遍布都市。
- **魔界裂隙与前线村镇**：关东郊外与废弃工业区常见小型裂隙，魔兽溢出与对魔忍清剿在此循环。

全堕时点的地图特征：学园灯火仍亮，但外出任务回收率暴跌；东京王国霓虹下多了「对魔忍俘虏拍卖」的暗语；尤米哈拉封锁线一周一变。

**【世界剧情线】**
① **前史·对魔忍成立与浅葱时代**  
魔界与人间默契破裂后，日本政府秘密组建对魔忍部队，以伊贺·甲贺等忍血与现代兵器融合应战。井河浅葱以天才剑术与忍法「杀阵华」（分身杀阵）成为最强对魔忍，妹妹井河樱同行。前对魔忍胧叛变，催眠刻印与情报出卖使浅葱陷入长期对抗。浅葱曾在混沌竞技场被俘并遭受身体改造，性感带敏感度被提升至常人约三千倍——这一创伤成为全系列「强者亦可被规则与药物击穿」的母题。浅葱最终在多次战役中与胧、魔族干部反复交锋，确立「对魔忍即国家暗部利刃」的传说。

② **支线展开·紫、雪风与红**  
八津紫（对魔忍紫）线：傀儡与调教主题，展示叛忍与魔族如何把「忠诚的身体」改造成武器。水城雪风线：与母亲水城不知火、搭档秋山凛子相关，舞台深入尤米哈拉，揭示地下都市的人口与药物产业链。心愿寺红等后继角色扩展「血族·恶魔契约·独立任务」支线。这些篇章在 RPGX 中以任务卡、活动与角色剧情并行，构成「前线不断失血」的背景噪声。

③ **RPGX 主舞台·五车与风魔小太郎**  
决战竞技场世界线重建后，风魔小太郎作为五车学园一年级生登场，组建任务小队，与井河浅葱（教官/传说）、井河樱、水城雪风、秋山凛子、八津紫、甲河明日香、神村舞华等并肩或交错。主线任务覆盖夺回生物兵器、清剿裂隙、潜入东京王国、对抗诺玛德分公司与魔族贵族。小太郎的风魔法与统御力使他成为「能把散沙对魔忍捏成队」的枢纽——全堕线正是从「枢纽尚在、但王牌接连被点名」开始崩坏。

④ **全堕导火索·连锁失陷**  
本任务世界的关键转折：诺玛德与胧系残余、魔族贵族联合发动「对对魔忍的点名猎杀」。第一波针对情报与后勤——任务简报被调包、支援延迟、解毒剂被替换。第二波针对个人软肋：亲属、弟子、旧伤改造接口、未清除的催眠刻印。井河樱在掩护撤退时被俘并移送改造舱；八津紫因傀儡术反噬短暂失控；水城雪风在尤米哈拉纵深任务中与凛子失散，通讯只剩断续坐标。浅葱强行单骑突入东京王国救人，中伏于多重刻印与药物，战力未减但指挥链被切断。学园内开始出现「谁还敢出任务」的裂痕，政府暗部讨论是否放弃五车、收缩为更小的暗杀编队。

⑤ **全堕高潮·学园戒严与公开拍卖**  
尤米哈拉与东京王国同步举行「堕忍展示」：被俘对魔忍的紧身衣编号被剥下，改挂奴隶铭牌；混沌竞技场重开专场。五车学园进入准戒严，风魔小太郎被临时赋予残存机动队指挥权，任务从「歼灭」转为「夺回与毁证」。关键名场面包括：地下铁隧道内与改造体追兵的阻滞战；资料库自毁倒计时前抢出弟子名册；浅葱被囚影像在暗网播放引发的士气崩溃与反向动员。魔族侧则推动「把五车整座迁入裂隙」的狂妄计划，需以多名传说级对魔忍的粒子作为锚点燃料——全堕若完成，日本暗部将失去对魔手段。

⑥ **残局与可介入终局方向**  
原作多线并不存在唯一「全员永久堕落」的官方终局；本世界将「全堕」定义为进行中的灾难状态：部分核心已陷、部分仍战、仪式未完成。契约者可推动的方向包括：斩断锚点仪式、救出特定目标、策反中间层魔族、向政府暗部换取战术核级清除授权（代价是街区毁灭）、或与小太郎合作重建地下抵抗网。失败态为五车沦陷、名单公开、对魔忍一词变为黑市商品分类。成功态亦非童话——救出者可能已遭不可逆改造，组织重建需数年月。

⑦ **日常可介入层**  
除大战外，全堕期高频事件：学园门禁与内鬼排查、伪造身份潜入东京王国赌场、黑市购买解毒剂与粒子稳定剂、护送伤员至秘密诊所、窃听诺玛德中层酒会、销毁落入手中的改造手术记录。AI 叙事应优先这些「还能救人的窗口」，而非一上来与魔界大公对轰。

**【主要人物】**
- **风魔小太郎（ふうま 小太郎）**｜身份：五车学园一年级、任务小队核心｜性格：外表随和、内核极硬，善于统御与扛锅｜装备·能力：风魔血脉忍法、小队指挥与临机战术｜人物弧光：普通新生→残存机动队枢纽→全堕线上的抵抗旗手｜立场关系：对浅葱敬畏、对同学保护欲强；对契约者先审后用，讨厌空降指挥。
- **井河浅葱（井河アサギ）**｜身份：最强对魔忍、传说级剑士｜性格：冷静克制、责任过重、创伤深藏｜装备·能力：忍刀、忍法杀阵华（近战分身杀阵）、粒子强化体术；旧改造后遗｜人物弧光：国家利刃→多次被俘与夺回→全堕线中成为敌我双方争夺的「象征」｜立场关系：视保护后辈为底线；对胧杀意与因果纠缠并存。
- **井河樱（井河さくら）**｜身份：浅葱之妹、对魔忍｜性格：明朗外向、重情｜装备·能力：体术与忍法（各版本表现略异）、与姊共同作战默契｜人物弧光：被保护的妹妹→独当一面→全堕中优先被猎的软肋｜立场关系：绝对信任浅葱；对小太郎小队友好。
- **水城雪风（水城ゆきかぜ）**｜身份：对魔忍、尤米哈拉相关任务常客｜性格：好胜、毒舌、外刚内热｜装备·能力：雷系忍法、高速突袭｜人物弧光：新世代王牌→地下都市深潜者→全堕中失联风险最高者之一｜立场关系：与秋山凛子搭档情深；对不知火母女线敏感。
- **秋山凛子（秋山凜子）**｜身份：对魔忍、雪风搭档｜性格：沉稳可靠、刀锋内敛｜装备·能力：剑技与忍法协同｜人物弧光：搭档支柱→失散后的搜救锚点｜立场关系：任务中优先保全雪风。
- **八津紫（八津紫 / 対魔忍ムラサキ）**｜身份：对魔忍、傀儡相关剧情核心｜性格：坚韧、一度被命运碾碎仍求反抗｜装备·能力：忍法与反傀儡意志｜人物弧光：忠诚武器→被操控的悲剧→全堕中「谁掌握她的线」成关键变量｜立场关系：对背叛者恨极。
- **胧（おぼろ）**｜身份：前对魔忍、叛变者｜性格：残忍、玩弄人心、享受堕落过程｜装备·能力：忍法催眠刻印、钩爪、情报网｜人物弧光：同伴→猎手→全堕工程的设计者之一｜立场关系：对浅葱执念病态；对契约者若有利用价值会笑着伸手。
- **水城不知火（水城不知火）**｜身份：雪风之母、强力对魔忍（各作状态有别）｜性格：成熟锋利｜装备·能力：高阶忍法与实战经验｜人物弧光：母辈传说→母女线悲剧/救赎的交点｜立场关系：护女优先。
- **甲河明日香（甲河アスカ）**｜身份：对魔忍、机械义肢相关设定｜性格：认真、略别扭｜装备·能力：义肢强化战力｜人物弧光：人造与人性的边界样本｜立场关系：可编入小太郎队。
- **神村舞华（神村舞華）**｜身份：火遁众相关、对魔忍｜性格：豪爽｜装备·能力：火系忍法｜人物弧光：火力手与气氛组｜立场关系：战场可靠、私下吵闹。
- **心愿寺红（心願寺紅）**｜身份：独立色彩浓厚的对魔忍｜性格：高傲、有自己的正义｜装备·能力：血与暗相关战力（随作）｜人物弧光：非五车主流却能左右局部战局｜立场关系：合作需谈条件。
- **诺玛德干部（具名随任务卡，公开资料常以组织层出现）**｜身份：企业外壳下的魔族渗透层｜性格：利益与实验优先｜装备·能力：资金、改造科技、魔族私兵｜人物弧光：全堕供应链｜立场关系：可贿可杀，杀后有替补。
- **五车学园校长/暗部联络官（公开资料多以职能出现，姓名随版本）**｜身份：体制接口｜性格：官僚与现实夹缝｜装备·能力：权限与情报｜人物弧光：决定是否放弃学园的人｜立场关系：对契约者冷淡，只看战果。

**【势力图谱】**
- **对魔忍 / 五车学园**：宗旨为对抗魔族与黑暗组织；地盘为学园与秘密据点；代表人物井河浅葱、风魔小太郎；战力从见习到传说级；全堕中内部分裂为「死守」与「疏散」。
- **日本政府暗部**：提供资金与合法杀人权，随时可抛弃棋子；与对魔忍是利用关系。
- **诺玛德**：跨国犯罪企业，军火、毒品、改造；与魔族深度勾结；全堕的物流与实验室后台。
- **东京王国 / 尤米哈拉黑市**：无统一领袖的生态圈，规则是钱与拳头；可买情报也可卖你。
- **魔族贵族与魔界来客**：弱肉强食，视人间为猎场与牧场；高阶个体即战略武器。
- **胧系叛忍与叛徒网络**：人数不多但懂对魔忍弱点；全堕手术刀。

敌友逻辑：利益与生存优先；盟约可破；暴露「能破催眠刻印的方法」或「仪式锚点坐标」会同时招来友军征用与敌军灭口。

**【贵重物品】**
- **对魔忍紧身衣与粒子传导组件**：战力倍增器，被俘后常被改造为控制介面。
- **忍刀（浅葱爱用等）**：传说级近战媒介，拍卖场高价目标。
- **催眠刻印相关触媒 / 反制符咒**：胧系核心技术；有则能救人或害人。
- **混沌竞技场入场与地下医疗舱权限**：进出黑暗经济的钥匙。
- **诺玛德生物兵器样本**：任务夺回物，失控即街区灾难。
- **五车弟子名册与任务黑匣**：全堕线中比宝石更贵的情报体。
- **魔界裂隙稳定锚 / 仪式核心**：完成则学园可被「拖入」非人领域；毁之可打断全堕终盘。

**【隐藏剧情 · 伏笔】**
- 浅葱三千倍敏感改造是全系列「强者如何被规则击穿」的原型，全堕线会复用同类手术于他人。
- 胧的执念不仅是个人仇恨，更服务更大魔族棋盘。
- RPGX 与旧作时间线有平行/重建差异，AI 应以「五车+小太郎」为当前锚，旧作事件作回忆与伤疤。
- 政府暗部可能早已准备「弃子方案」：用导弹抹平尤米哈拉与学园灭口。
- 全堕仪式需要「自愿或破碎的对魔忍粒子」，因此敌人会优先摧毁意志而非肉体。

**【大事记时间线】**
- 2005 起 → 浅葱篇：最强对魔忍登场，与胧对立，混沌竞技场创伤。
- 后续数年 → 浅葱2、紫、雪风等：东京王国与尤米哈拉黑暗面展开。
- 浅葱3 / ZERO 等 → 时间跨度拉长，世界观补全。
- 2014 决战竞技场 → 卡牌网游，角色大集结。
- 2018 RPG/RPGX → 五车学园、风魔小太郎主视角。
- 全堕任务时点 → 王牌连环失陷、学园戒严、锚点仪式倒计时。
- 终局分叉 → 抵抗重建 / 组织商品化 / 焦土清场（依契约者介入而定）。

**【叙事基调 · 雷区】**
基调：黑暗特工、肉体与精神双重凌虐威胁、组织伦理崩坏、仍有人选择死守。忌把浅葱写成无弱点的无敌剑圣；忌把全堕写成轻浮后宫游戏；忌用「一颗药丸全体恢复」消解改造后遗；忌让低阶契约者正面硬刚魔界大公。最早可切入锚点＝五车学园戒严令下达前后、第一次「堕忍展示」消息传回的那一夜。

## 阶位切入点

> 阶位↔战力：五阶≈传说级对魔忍 / 诺玛德执行层 / 魔族领主干部（城市级·可夷平街区）。世界顶点（魔界大公、古魔、系列幕后）为更高阶情报存在，本阶切入一律「规避顶点·情报优先」，严禁战力归零式削弱。

**五阶（传说对魔忍崩坏期 · 全堕抵抗线）**

切入身份/时点：契约者以「政府暗部临时编制的协力对魔忍 / 五车外援教官 / 国际暗部交换人员」身份切入，锚定风魔小太郎刚接过残存机动队指挥权、井河浅葱失联或被俘影像首度泄露的四十八小时内。

初始事件：五车地下指挥室警报连响——东京王国地下拍卖行预告「今夜上拍编号 A-01 井河相关物资与活体」。小太郎组建强袭队，名额只剩最后一个外援位；同时学园后门发现被丢回的弟子名牌，内侧刻着胧的催眠纹样。契约者必须在「参与强袭救人」与「留守学园防内应」之间立刻表态，否则权限将被回收。

开场白建议：「五车的灯还亮着，可任务板上的红叉已经连成一片。通讯里有人用笑腔念出井河浅葱的编号，像在念拍卖目录。风魔小太郎把一份外援臂章拍在你胸口——『你要是怕脏，现在就离开学园；若留下，今晚我们可能都回不来。』」

关键NPC立场：**风魔小太郎**——需要战力与可信之人，厌恶空降指挥，用战果换信任；**井河浅葱**——若已失联则成为营救目标，若短暂夺回则命令优先于情感；**水城雪风**——可能失联于尤米哈拉，坐标是诱饵也可能是真；**胧**——会以旧识口吻远程嘲讽，诱导你用「错误的救人方式」启动更大刻印；**秋山凛子**——索要支援时只报坐标不报情绪，可靠的刀；**诺玛德现场主管（任务具名）**——视契约者为可回收实验体。

主线钩子/支线：主线＝阻断「堕忍拍卖」并截获仪式锚点零件；支线＝清洗学园内鬼、黑市换取抗催眠针剂、护送伤员突围、销毁落在记者手中的改造影像。蝴蝶点：是否在拍卖场当众揭穿买方为政府暗部某一派——揭穿可乱敌，也可能让暗部提前弃子屠城。

危险度/规避：极高。须规避魔界大公级存在与已完成的群体催眠场中心；对胧近距离对视超过数秒需抗性检定。超阶巅峰仅作情报：存在·条件性胜利，不靠削弱解释。

任务方向/奖励：方向＝救人、毁锚、重建指挥链；奖励取向＝高阶忍法卷宗残页、反刻印组件、诺玛德实验数据、五车最高行动权限、传说级武器维修零件。禁止直接发「秒杀魔界」级道具。
