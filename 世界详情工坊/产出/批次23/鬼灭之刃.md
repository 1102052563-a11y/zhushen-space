# 鬼灭之刃
<!--meta lib=主库 tiers=一、二、三、四-->

## 剧情

**【作品来源】**
《鬼灭之刃》（日语：鬼滅の刃，英文：Demon Slayer: Kimetsu no Yaiba）为日本漫画家**吾峠呼世晴**创作的少年漫画，2016年2月15日至2020年5月18日于集英社《周刊少年Jump》连载，全205话、单行本全23卷。类型：大正时代黑暗奇幻、时代剧、猎奇战斗与兄妹羁绊。文风：残酷斩鬼与人情温度并存，呼吸法刀技华丽、血鬼术猎奇，主线紧扣「让妹妹变回人类」与「斩杀鬼王」。衍生：ufotable动画（立志篇、无限列车篇/剧场版、**游郭篇**、刀匠村篇、柱训练篇、无限城篇等）、舞台剧与游戏。本档案以**漫画正典主线**为准。卡片锚点（世界库提要）：**花街吉原篇**——音柱**宇髓天元**与**灶门炭治郎**小队潜入吉原，对抗上弦之陆兄妹**妓夫太郎**与**堕姬**，战斗余波摧毁半个花街，谱面完成前的绝死时刻。

**【世界定位】**
日本大正年间：表面电车、电话与西服并行，暗处「鬼」食人、畏日光，由**鬼舞辻无惨**以血制造与统御。人类隐秘组织**鬼杀队**以日轮刀与呼吸法斩鬼，最高战力称「柱」。主角**灶门炭治郎**在家人被无惨屠杀、妹妹**灶门祢豆子**变鬼后，拜**鳞泷左近次**为师，入队寻使祢豆子变回人类之法并报灭门之仇。一句话：兄妹在大正夜色里斩鬼，从最终选拔走到吉原上弦之战。契约者是变数，不是免死金牌；日轮刀、紫藤花与柱的刀锋随时改写命运。

**【世界观 · 力量体系】**
力量本源：**呼吸法**（全呼吸强化心肺与肌肉，衍生水·炎·雷·风·岩·花·虫·音·霞·蛇·恋等流派，源头为**继国缘壹**的「日之呼吸」/火之神神乐）、**日轮刀**（刀匠村特殊砂铁矿，斩鬼颈可令鬼灰化；刃色随持有者变）、**血鬼术**（鬼以血发动的异能，上弦尤甚）、**斑纹**（极境觉醒，寿命代价）、**透骨视觉**等。鬼规则：食人成长、断肢可再生、畏日光即死、颈被日轮刀斩断则难再生；十二鬼月（上弦一～六、下弦一～六）眼中刻字，持无惨血印。死亡：人类与鬼皆可真正死亡（炼狱杏寿郎、宇髓重伤引退、上弦被斩等有明证），禁止无限复活。特殊系统：鎹鸦传令、隐部善后、蝶屋医疗与毒学（胡蝶忍一脉）、刀匠村锻刀、产屋敷一族统领鬼杀队。

战力阶梯（按破坏力表现，宁低勿高）：
1. **凡人 / 见习队士前**：徒手或农具，毁木门、放倒平民。乐园观感≈一阶初期。
2. **最终选拔合格队士 / 水之呼吸初成**：日轮刀斩下等鬼，裂石级斩击，单挑村落级食人鬼。乐园观感≈一～二阶。
3. **十人队士精锐 / 善逸一之型·伊之助兽气**：可斩下弦级外围，乱战取首，街区巷战级。乐园观感≈二～三阶。
4. **柱级 / 上弦之陆·无限列车猗窝座级**：一击毁楼群、掀平花街建筑、与上弦正面对砍；音柱爆竹与毒镰可改写街区地形。乐园观感≈三～四阶（质变分水岭·精英Boss）。本世界库覆盖止于四阶；再上（上弦之壹～参、鬼舞辻无惨本体、继国缘壹级）为本世界**顶点向**，低阶只闻名号、**规避顶点·情报优先**，严禁硬刚。

乐园阶位映射：凡人～选拔前≈一阶；合格队士～水呼初成≈一～二阶；十人精锐～下弦对战≈二～三阶；柱 / 上弦之陆 / 无限列车级≈三～四阶。宁低勿高。世界顶点（鬼舞辻无惨、上弦之壹黑死牟、继国缘壹传说）低阶切入一律**规避顶点·情报优先**；超阶向写条件性胜利，严禁「被封印所以战力为零」。

**【地理 · 舞台】**
- **灶门家山中炭窑 / 狭雾山**：开篇灭门与鳞泷修行地；炭窑炊烟与雪夜尸体对照，是「平凡被鬼撕开」的第一课。最终选拔「藤袭山」——山中遍植紫藤，将食人鬼囚于内侧，选拔生须七日内存活；山腹有手鬼等「专用粮仓」级老鬼。
- **浅草**：炭治郎初遇无惨与珠世、愈史郎之地；电车铃声、夜市灯火与西服人群，是鬼王最爱的「掩护层」——在此拔刀等于惊动宪兵与无数目击者，隐部事后极难圆场。
- **京乐家鼓宅**：响凯以鼓面改写室内空间结构的封闭杀场；楼梯与房间可被鼓点重组，适合「规则杀」叙事。
- **那田蜘蛛山**：下弦之伍累的蜘蛛家族巢穴；丝网跨谷、亲族鬼分级（父·母·兄·姐），地形垂直，适合小队拆队作战；水柱富冈义勇在山缘收场。
- **产屋敷本邸**：鬼杀队主公府邸，紫藤环绕、香气特殊；柱合会议庭院与祢豆子审判回廊是政治戏舞台。主公病卧深处，外人难至。
- **蝶屋（胡蝶屋）**：胡蝶忍主导的伤员疗养与毒学中心；香奈乎、神崎葵、寺内清、中原澄、高田菜穗等看护轮值；药园、病房、训练庭三分。
- **无限列车沿线**：连接都市与地方的夜行特快；车厢即迷宫，魇梦以睡境把乘客钉在「幸福梦」里；车顶与车头是炎柱对上弦的移动战场。
- **吉原游郭（花街）**：卡片锚点主舞台——东京深处合法烟花地；京极屋（蕨姬/堕姬）、荻本屋、鸨屋（鲤夏线）等屋形并列；屋顶可跑、地下有囚室与水道；音柱爆竹与毒血一经点燃，半个花街楼群可连锁倒塌起火。战后废墟由隐部与官方以「火灾/地震」口径掩盖。
- **刀匠村**：隐藏在山中的锻刀聚落，铁穴与神祠并存；上弦之肆·伍曾突袭（四阶后时间线，低阶仅作刀坯产地情报）。
- **霞柱宅 / 恋柱相关路线 / 岩柱寺院**：柱训练期舞台，本覆盖可作「柱不在吉原时」的远程气压。
- **无限城**：无惨与十二鬼月的异空间——走廊无限、门扉乱接；本世界覆盖阶外，只作终局名号与高阶恐惧传说。

写场景：先定「山村 / 城镇 / 游郭 / 柱宅」层级，再定冲突强度；底层是失踪人口、游女失踪、鎹鸦急件；中层是下弦与队士讨伐；高层是柱与上弦、主公安危。吉原夜战须同时写：游女尖叫、灯笼爆裂、毒雾、建筑垮塌四层感官。

**【世界剧情线】**
① **灭门之夜与狭雾山（一阶叙事）**  
大正年间，灶门家以卖炭为生。家主**灶门炭十郎**病逝后，长子**灶门炭治郎**撑起母亲**灶门葵枝**与弟妹**竹雄、花子、茂、六太**及姐姐**祢豆子**。某雪夜炭治郎在山脚被老人**三郎**留宿——三郎警告「夜里有鬼」——次日返家见家人被杀，血泊中唯祢豆子存活却已鬼化，额生角、牙出，却在本能与亲情撕扯间未食兄长。水柱**富冈义勇**本欲一刀斩祢豆子，见兄妹互相挡刀，改而试探炭治郎的刀意与慈悲，终荐其投奔前水柱**鳞泷左近次**。狭雾山两年：劈开巨岩、恒久全呼吸、水之呼吸十型基础；祢豆子被鳞泷施以暗示，视人类为「应守护的家人」，昼眠于特制木箱，以睡眠代食人。最终选拔于藤袭山七日：炭治郎救下同批少年少女，斩杀手鬼（曾吞食鳞泷多代弟子、以「好恨」为口头禅的老鬼），合格入队。刀匠**钢铁冢萤**锻出其日轮刀（刃色特殊），鎹鸦**天王寺松卫门**报到。世界状态：普通队士不知无惨面容；「柱」如天神；灭门案被隐部以野兽害记档。

② **初任务至浅草·珠世同盟**  
沼鬼篇：炭治郎于沼泽地带救下被沼鬼（可分身、以沼空间囚人为食）纠缠的兄妹，凭嗅觉辨「真身沼」与假面，完成首个正式讨伐。途中结识队士**村田**等前辈，见识「普通队士也会怕」。浅草：在夜市人潮中嗅到与灭门之夜同源的冰冷气味，初遇易容的**鬼舞辻无惨**；无惨当街杀人取乐、对耳饰露出憎恶，炭治郎不敌其再生与威压。变鬼医师**珠世**与侍从**愈史郎**以血鬼术短暂逼退无惨分身/本体注意，带走炭治郎。珠世坦陈：她曾是医师，被无惨所变，现研究「使鬼变回人类」之药，需要炭治郎作为「能接近战场的队士」搜集十二鬼月之血与无惨情报；炭治郎以祢豆子的未来为赌，结盟。因果：主角线从「单纯复仇」叠上「科学式救赎」；无惨记下「日轮花札耳饰少年」；愈史郎成为情报中转。

③ **鼓宅·善逸·伊之助·那田蜘蛛山**  
任务转入京乐家旧宅：鼓鬼**响凯**以血鬼术「鼓」改写房屋结构与攻击节奏，炭治郎在崩塌楼梯间学会听「破绽的气味」。其后于任务中结识雷之呼吸使·**我妻善逸**——金发、胆小到跪地求饶，却有超常听觉；其师**桑岛慈悟郎**（前鸣柱）只传他一之型，清醒时几乎只会这一招，陷入沉睡则雷光爆发——以及兽之呼吸使·**嘴平伊之助**——野猪头套、双刀、山中长大的好战少年，皮肤感知锐利。那田蜘蛛山：下弦之伍**累**以丝线与「亲族」鬼（父·母·兄·姐皆为累改造或役使）织成猎场，专吃队士。炭治郎在绝境中跳出灶门家祖传舞蹈剑型「火之神神乐」（实为日之呼吸的家传残型），祢豆子觉醒血鬼术「爆血」灼烧丝网；累以极致再生与丝茧反杀，水柱**富冈义勇**于山缘一刀斩累之颈。战后隐部与柱围住「携鬼同行」的炭治郎，押往主公府。**产屋敷耀哉**病弱而声柔，令祢豆子于日下当众受审；风柱**不死川实弥**以刀伤试其是否伤人，祢豆子忍痛不反击；主公裁可：祢豆子可随队，若伤人则由义勇与炭治郎亲手斩之。柱中炼狱杏寿郎等表示理解，胡蝶忍留下「毒与药」的观察。炭治郎、善逸、伊之助入蝶屋康复，在**栗花落香奈乎**与**神崎葵**督促下练「全集中·常中」——呼吸不止于拔刀瞬间，而贯于行走坐卧。

④ **无限列车·炎柱陨落（三～四阶门槛）**  
鎹鸦传令：无限列车连续失踪，炎柱**炼狱杏寿郎**已登车。炭治郎四人同行。下弦之壹**魇梦**以强制入梦的血鬼术把乘客钉在「最幸福的假现实」里，并在梦中具现「一级·二级」等人格分身。炭治郎在梦中与亡父之影对峙后斩断梦境；醒后四人与炼狱合力斩魇梦。黎明前，上弦之参**猗窝座** 杀上列车顶：武痴，要强者「变鬼与之长搏」。炼狱以炎之呼吸护住后辈与乘客，与猗窝座对轰至车厢变形；终被贯穿胸口，仍挡在炭治郎前，留下「成为柱、守护弱者、心志如炎」的遗言后身亡，未及斩上弦。猗窝座因日将近而退。因果：上弦被钉死为「柱以命未必能换杀」的标尺；炭治郎背上炎柱的期待；无惨侧确认灶门兄妹与「斑纹/神乐」异常。隐部回收列车，报纸只写事故。

⑤ **花街吉原篇（卡片锚点卷·四阶质变副本）**  
无限列车后约四个月：音柱**宇髓天元**追查游女连环失踪，派三名妻室（原忍）**雏鹤、牧绪、须磨**分头潜入吉原屋形，逾期未归。宇髓杀到蝶屋，要强征看护**神崎葵**作女装潜入；炭治郎、善逸、伊之助拦下，请缨替代。三人易装（炭治郎女装、伊之助假扮少爷、善逸心惊胆战）进入游郭，分探京极屋、荻本屋、鸨屋等。鸨屋花魁**鲤夏**线提供人间气味的对照；线索逐渐指向京极屋当红「蕨姬」——实为上弦之陆**堕姬**（人类时名**梅**，因容貌受辱与执念被无惨之血所变，嗜「可爱」与赞美，能以腰带操纵人体、夺取他人面容与记忆片段）。堕姬识破并掳走善逸；宇髓下令后辈撤离以免拖累，炭治郎抗命追击。伊之助凭皮肤感知自地下掘入，发现被囚的善逸、牧绪、须磨，以及堕姬分出的「腰带鬼」看守；宇髓杀到，以音之呼吸与爆竹忍具轰塌腰带鬼。腰带鬼回归堕姬本体，炭治郎在座敷与街道间苦战；**祢豆子**长大化出战，爆血压制，却因大量人血气味几乎失控噬人，宇髓以明确指令与炭治郎的呼唤令其重新沉睡。宇髓斩落堕姬之头并出言嘲讽其「不华丽」——堕姬崩溃悲鸣，从体内唤出同为上弦之陆的兄长**妓夫太郎**：贫民街出身的兄妹，人类时靠妹妹卖身与互相啃食般的贫困活着，化鬼后兄以剧毒镰刀与恐怖再生成为真正战力，妹负责诱食与潜藏；两人必须「同刻斩首」才难再生。混战升级：宇髓对妓夫太郎，毒镰见血即腐肉；炭治郎、善逸、伊之助与恢复的堕姬及毒雾周旋。最终炭治郎与重伤的宇髓联手斩断妓夫太郎之颈，善逸（雷之一之型变奏）与伊之助同时斩堕姬，兄妹灰化。音之呼吸的爆竹链、毒血腐蚀与楼阁连锁倒塌，**半个吉原化为火海废墟**；游女与客人在隐部与残存屋形人员组织下疏散，仍有大量平民伤亡——这是锚点卷的「胜利税」。宇髓失去一眼与一臂、全身剧毒，战后宣布引退，把「华丽」留给后辈。主公**产屋敷耀哉**闻报喜极而泣：百年来鬼杀队首次确认斩杀上弦，断定此代可见无惨之终。无限城中，无惨召集剩余上弦，对空席震怒，清洗与重编杀意——直接通向刀匠村袭击与无限城决战。名场面：灯笼巷嗅觉追踪、女装潜入的屈辱与信息差、善逸被掳时的听觉绝望、祢豆子街头巨大化、宇髓「得分」式爆破街战、妓夫太郎毒雾下的兄妹羁绊、双首同落、废墟上的黎明。世界状态：四阶核心副本；上弦可杀，柱亦可能退役；无惨战略目光转向「能在日光下存活的鬼」祢豆子。

⑥ **刀匠村至终局方向（本覆盖外情报，供时间线定位）**  
刀匠村：炭治郎等协助抵御上弦之肆**半天狗**（分身情感）与上弦之伍**玉壶**（血鬼术壶中异界）；霞柱**时透无一郎**、恋柱**甘露寺蜜璃**参战；祢豆子于战斗中**克服日光**。无惨得知后突袭产屋敷邸，耀哉以爆炎与命相搏；珠世之药使无惨短暂变人，无惨启动**无限城**把鬼杀队与上弦拖入决战。上弦之壹**黑死牟**（继国岩胜）、贰**童磨**、参猗窝座等与柱捉对惨胜；珠世死于无惨之手。炭治郎于濒死幻视中见祖先**炭吉**观摩**继国缘壹**之日之呼吸，悟十二型连携；无惨遭毒与群攻后被黎明逼入绝境，一度附身炭治郎，终被祢豆子与珠世遗药拉回人形。无惨灰灭，世上恶鬼尽，鬼杀队解散，后辈进入无鬼的昭和日常。写手勿编造未公开宝物与私设终局；低阶禁止提前「通关无限城」。

⑦ **微观政治贯穿**

- 产屋敷一族与无惨有血缘诅咒式纠缠（设定向），主公香气既统队也引鬼。
- 童磨曾居上弦之陆，后升贰，陆之位由妓夫太郎兄妹补上——席位会递补、会内斗。
- 十二鬼月眼中刻字由无惨赐予，亦可剥夺。
- 终局后无鬼，队解散；现代后裔梦中仍见日轮刀光（尾声），证明创伤与传承未灭。

近世～大正前 → 鬼杀队与十二鬼月拉锯；下弦上弦编制成型。  
大正·灶门灭门夜 → 祢豆子化鬼；炭治郎遇义勇，投鳞泷。  
两年修行 → 劈岩；最终选拔；斩手鬼；入队。  
沼鬼·浅草 → 初任务；遇无惨；珠世同盟。  
鼓宅·结伴 → 响凯；善逸、伊之助入队缘。

**【叙事基调 · 雷区】**
画风：残酷斩首、断肢中毒与温暖人情同框；呼吸法可有声彩特效，但肉体损伤必须真实。  
忌把炭治郎写成无脑无敌超人（他常哭、常重伤、靠嗅觉与同伴）。  
忌低阶让无惨、黑死牟、童磨满街走。  
忌「上弦被削弱/封印才可击败」——吉原之胜来自联手、毒抗、牺牲、双首同落与战场选择，不是战力归零。  
忌编造原作没有的呼吸流派名、乱改谁死谁活。  
忌游郭篇写成轻松逛窑子——失踪、人体操纵、毒与肢解是底色；花魁线写权力与诱饵，不写低俗凑字。  
忌忽略隐部与舆论掩盖：大正日本「公开斩鬼」会引发宪兵与恐慌。  
最早切入锚点：灭门后赴狭雾山途中，或**吉原潜入前夜**（卡片锚点）。

## 阶位切入点

> 阶位↔原作：一阶≈凡人～选拔合格前后；二阶≈正式队士～下弦外围；三阶≈十人精锐～下弦之伍/列车前；四阶≈柱协同～上弦之陆/吉原决战。世界顶点（鬼舞辻无惨、上弦之壹～参）为跨阶存在，低阶切入一律「规避顶点·情报优先」。

**一阶（凡人～最终选拔 · 狭雾山入门线）**
切入身份/时点：契约者以「赴狭雾山求斩鬼之法的山民/浪人」或「选拔前年幼见习」身份切入，锚定灶门炭治郎刚拜入鳞泷门下、祢豆子尚在木箱沉睡的最早开局。
初始事件：大雪封山的夜路，你与一个背着木箱、额有伤疤的少年（炭治郎）同宿山脚民家；次日他要上狭雾山，而山道传来「选拔用」的鬼的锁链声——有人贩卖「免死名额」，要你选：举报、买名额、或跟炭治郎一起见鳞泷。
开场白建议：「雪把山路抹成一条白痕。你听见木箱里传来均匀的呼吸——那少年说那是他的妹妹，可箱缝里溢出的气味绝不是病人。狭雾山的紫藤还没开，鬼的眼睛已经在树后面亮了。」
关键NPC立场：**灶门炭治郎**——会把你当可求助的人，但绝不让你开箱伤祢豆子；**灶门祢豆子**——昼眠，夜可能攻击「威胁兄长者」；**鳞泷左近次**——严师，测你是否只想借刀杀人；**富冈义勇**——偶过山门，话少刀快，视可疑者为潜在鬼；**三郎**——山脚老人，知鬼怕留人，可提供一夜庇护。
主线钩子/支线：劈岩修行旁观、最终选拔藤袭山、手鬼栖息地；支线＝紫藤花采集、山中迷路孩童、假冒鬼杀队收钱。
危险度/规避：中——夜间鬼与选拔鬼是主威胁；须规避任何「像普通人却无呼吸」的都市来客（无惨级只闻名）。
任务方向/奖励：活过入门、拿到基础呼吸口诀或一把劣质日轮刀坯；奖励取向＝紫藤粉末、嗅觉训练法残页、鳞泷评语（队内通行证雏形）。

**二阶（正式队士 · 初任务至浅草线）**
切入身份/时点：以「刚获日轮刀的新人队士」或「紫藤驿站护卫」切入，锚定炭治郎沼鬼任务前后至浅草遇无惨前夜。
初始事件：鎹鸦投下任务——某镇连续失踪，现场有沼泥与童鞋；你抵达时炭治郎已在场，沼面浮出多张脸要「选一个陪葬」。是抢人头、联手，还是先撤平民？
开场白建议：「日轮刀还新得反光。镇子井盖下的水黑得像墨，你的鎹鸦只说了一句：斩颈，别听鬼哭。桥那边，有个背木箱的队士已经把鼻子凑近了沼气。」
关键NPC立场：**灶门炭治郎**——欢迎联手，厌恶虐杀；**我妻善逸**（若时点稍后）——尖叫拖后腿，危机会睡；**珠世**——浅草暗巷里的「医生」，用情报换你护送愈史郎；**愈史郎**——嘴毒，验你是否无惨的眼线；**天王寺松卫门**——炭治郎的鎹鸦，传讯不客气。
主线钩子/支线：沼鬼讨伐、浅草人群掩护下的气味追踪、与珠世第一次交易；支线＝假鬼灾骗保、走私「鬼骨」药、保护变装旅行的祢豆子箱。
危险度/规避：中高——下弦未至但无惨可能擦肩；见「冷白皮肤、憎恶阳光的上流人」立刻撤离，勿对刀。
任务方向/奖励：完成首个独立讨伐、换取蝶屋治疗名额；奖励取向＝正式队士俸禄、水/雷入门指导、珠世的解毒试药。

**三阶（精锐队士 · 蜘蛛山至列车前夜线）**
切入身份/时点：以「有多次斩鬼记录的十人队士」或「蝶屋外勤护卫」切入，锚定那田蜘蛛山战役前后、柱合裁判已过、无限列车出发前。
初始事件：蜘蛛山丝网上挂着还活着的队士；累的「母亲」鬼发出亲族召集。你的鎹鸦命令与灶门小队汇合——同时风柱一派仍有人要斩祢豆子，你被夹在「军令」与「主公裁可」之间。
开场白建议：「丝线割开树皮的声音像细雨。你看见童脸的鬼叫『父亲』，而真正的下弦在巢心笑。远处水柱的羽织一闪——这一战，不是你一个人的军功簿。」
关键NPC立场：**灶门炭治郎**——已会火之神神乐雏形，重伤也要救人；**嘴平伊之助**——抢首级，可被激将；**累**——以家庭枷锁杀人，可被「亲情话术」扰动但极危；**富冈义勇**——收场斩杀，不啰嗦；**胡蝶忍**——战后收伤员，顺便查你身上的鬼血污染；**不死川实弥**——敌视一切「护鬼者」。
主线钩子/支线：蜘蛛山破阵、柱合会议余波、全集中常中特训；支线＝护送伤员回蝶屋、刀匠取刀、调查列车失踪预告。
危险度/规避：高——下弦之伍可屠小队；须规避上弦级（猗窝座若提前出现则逃），勿拦炼狱的列车任务名额硬抢。
任务方向/奖励：参与斩下弦或确认击杀、获柱级评语；奖励取向＝常中训练法、毒抗药、较好刃纹的日轮刀、蝶屋长期床位。

**四阶（柱协同 · 花街吉原锚点线）**
切入身份/时点：以「音柱临时编入的潜入组」或「有游郭地下关系的编外协助者」切入，锚定宇髓天元妻室失联、炭治郎三人请缨代神崎葵潜入吉原前后——卡片锚点正中。
初始事件：吉原夜市灯笼如昼，京极屋新晋花魁「蕨姬」座敷接客频率异常；你的联络人雏鹤断线前只留下半块手帕与一句「腰带会动」。宇髓要你在「撤平民」与「跟炭治郎进地下」里二选一——选错，半街爆炸时你在里面。
开场白建议：「脂粉味盖不住腐甜。游女笑着拉袖，你的刀却在鞘里发烫——有什么东西在屋檐 inductive 走，像一条活着的腰带。音柱在屋顶说：今夜要华丽地活，或者华丽地死。」
关键NPC立场：**宇髓天元**——指挥独裁，认「有用的华丽」；**灶门炭治郎**——坚持救人优先；**我妻善逸**——易被花魁线诱捕，救他即支线；**嘴平伊之助**——地下开路最优选；**灶门祢豆子**——战力与失控风险一体；**堕姬**——慕强，可被「赞美/侮辱」扰动；**妓夫太郎**——无谈判定，毒镰见血；**雏鹤·牧绪·须磨**——救援目标，知屋形密道。
主线钩子/支线：游郭潜入大作战、腰带鬼地下牢、兄妹同时斩首、花街崩坏疏散；支线＝假扮客人套话、转移中毒游女、为珠世盗一管上弦毒血（极难）、掩护宇髓起爆点。
危险度/规避：极高——双上弦之陆；半个花街级破坏力属场景天灾。须规避无惨本体与其他上弦；**超阶巅峰（无惨）存在·情报优先/条件性胜利**，严禁硬撼。宇髓重伤线可改细节，不可当「柱无敌」。
任务方向/奖励：斩上弦参与认定、救人清单、产屋敷侧谢礼；奖励取向＝音之呼吸观摩残谱、抗毒血清、上等日轮刀、隐部级善后权限、花街废墟中的情报残卷（指向无限城名号即可，勿发终局神器）。
