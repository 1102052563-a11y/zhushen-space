# 空洞骑士
<!--meta lib=主库 tiers=一、二、三、四、五、六-->

## 剧情

**【作品来源】**  
《空洞骑士》（*Hollow Knight*）由澳大利亚独立工作室 **Team Cherry**（阿瑞·吉布森 Ari Gibson、威廉·帕普 William Pellen、杰克·文斯 Jack Vince 等）开发并发行的 2D 银河恶魔城动作冒险。2017 年 2 月 24 日 Steam/GOG 首发，后登陆主机；DLC 含 *Hidden Dreams*、*The Grimm Troupe*、*Lifeblood*、*Godmaster*。像素剪影风、手绘场景与克里斯托弗·拉金（Christopher Larkin）原声构成「已死王国」气质。续作《空洞骑士：丝之歌》另档。本档案锚定 **圣巢（Hallownest）本体 + 四部官方 DLC 主线与真结局分支**，不含模组。

**【世界定位】**  
地下虫国 **圣巢** 已因感染与王室崩坏沦为废墟。玩家扮演 **小骑士（The Knight）**——无面甲的空洞容器，自遗忘十字路醒来，深入王城、深渊与梦境，追查 **苍白之王（The Pale King）** 与 **辐光（The Radiance）** 的旧账。一句话：在死去的国度里，用钉子与护符重写「容器能否容纳虚空」的答案。卡片锚点：**感染蔓延的圣巢** 与 **辐光/虚空心结局分叉**。

**【世界观 · 力量体系】**  
力量本源：**灵魂（SOUL）** 可从敌人汲取并用于法术与聚焦治疗；**钉子（Nail）** 近战；**护符（Charms）** 槽位构筑；**梦钉（Dream Nail）** 读记忆、入梦战；**虚空（Void）** 为深渊产物，可对抗光之感染。死亡：小骑士「死亡」掉落暗影并在长椅重生——叙事上是容器残响，非普通人复活术。感染（Infection）由辐光意志驱动，使虫民狂热、甲壳渗出橘光。

战力阶梯（破坏力表现，宁低勿高）：  
1. **爬虫 / 幼虫 / 普通感染虫** — 近身叮咬、弱弹。≈一阶。  
2. **精英守卫 / 螳螂战士 / 真菌高地精锐** — 可组战、有招架节奏。≈一~二阶。  
3. **领主级 Boss（魂之暴君、螳螂领主、守梦者三贤等）** — 场地战、阶段转换。≈二~三阶。  
4. **梦境强化体 / 梦魇之王格林 / 白色宫殿平台杀局** — 高速弹幕与精确平台。≈三~四阶。  
5. **纯粹容器 / 辐光（标准战）** — 王国级意志对决，光弹覆盖舞台。≈四~五阶。  
6. **绝对辐光（Absolute Radiance）/ 虚空化终局表现** — 神主级弹幕与虚空海吞光。世界个人顶点≈**六阶**（王国/概念神战表现；不抬七阶以上星体摧毁）。  

**乐园阶位映射**：爬虫与十字路杂兵≈一阶；螳螂/蜂巢/水晶等区域精英与常规 Boss≈二阶；守梦者、梦境 Boss、格林剧团≈三阶；白色宫殿、纯粹容器≈四阶；辐光本体≈五阶；绝对辐光与虚空心终局表现≈六阶顶点。切入覆盖一~六。对顶点：**存在·情报优先 / 条件性胜利**（护符构筑、梦境规则、虚空共鸣），严禁把顶点写成「无威胁空壳」。

**【地理 · 舞台】**  
- **遗忘十字路（Forgotten Crossroads）**：教程与感染前后对照；假骑士、母蛆。  
- **苍绿之径 / 真菌荒地 / 雾之峡谷**：酸液、孢子、乌姆。  
- **螳螂村（Mantis Village）**：中立到敌对可转化的战士族。  
- **泪水之城（City of Tears）**：永恒雨的王城；灵魂圣所、守望者尖塔。  
- **水晶山峰 / 帝王之谷 / 王国边缘**：采矿与地理尽头。  
- **深巢（Deepnest）**：黑暗、织者、赫拉。  
- **蜂巢 / 皇家水道 / 古老盆地**：垂直分层。  
- **深渊（The Abyss）**：虚空海、阴影生物、黑卵记忆。  
- **白色宫殿（White Palace）**：梦中王室平台杀局。  
- **神居（Godhome，Godmaster）**：众神像与万神殿挑战。  
- **Dirtmouth**：地上小镇，长椅与 NPC 枢纽。

**【世界剧情线】**  
① **坠落与十字路**  
小骑士自井道进入圣巢，于十字路苏醒。商人 **斯莱（Sly）**、老人 **老人甲虫（Elderbug）** 等提示王国已死。假骑士（False Knight）占据甲壳，揭露「灵魂与感染」的双重剥削。击败后十字路可感染化，世界状态改变——进度会改写地图。

② **绿径、真菌与城门**  
苍绿之径遇 **霍瓦（Hornet）** 第一次交手：她试探容器是否「足够空」。真菌荒地可促成螳螂领主同盟（杀叛徒后中立）。泪水之城展示王权遗骸：贵族、乞丐、灵魂学者 **苏尔（Soul Master）** 把活虫炼成灵魂。

③ **守梦者与黑卵圣殿**  
三位 **守梦者（Dreamers）**——**赫拉（Herrah）**、**莫诺蒙（Monomon）**、**卢瑞恩（Lurien）**——以梦封锁 **黑卵圣殿** 内的 **空洞骑士（The Hollow Knight）**。小骑士以梦钉杀其梦中身，解除封印。此过程伴随霍瓦监视、乌姆与教师档案、守望者之城的雨中潜行。

④ **深渊真相**  
古老盆地与深渊揭示：**苍白之王** 为对抗辐光，用虚空批量制造容器；「完美容器」被选中封印辐光，即殿内空洞骑士——实则仍残留意念，封印失败。小骑士是被抛弃的容器之一。虚空心（Void Heart）护符在深渊记忆中觉醒，改变结局权限。

⑤ **霍瓦、蜂巢与血统**  
霍瓦为深巢女王赫拉与苍白之王交易所生，身负「守护者」与「王室血」双重身份。蜂巢线处理感染与蜂群忠诚；皇家水道与收藏家展示王国卫生与怪物收集的黑色幽默。

⑥ **格林剧团（DLC）**  
灯火召唤 **格林（Grimm）** 与剧团，举行梦魇之王仪式：收集火焰、与格林/梦魇之王交战，可选择完成仪式或阻止——关乎梦魇心与灯笼。

⑦ **神主（Godmaster）**  
神探求者把小骑士拉入神居，以万神殿复现众 Boss。最终面对 **绝对辐光**。此线把「玩法挑战」写成虫国诸神的审判场。

⑧ **结局分叉（须写明）**  
- **空洞骑士结局**：不取虚空心，取代前辈进入黑卵——循环封印。  
- **封印破碎**：霍瓦介入，二人共战殿内骑士，辐光仍在。  
- **梦无止境**：虚空心 + 入梦击杀辐光，虚空吞没神光。  
- **拥抱虚空（Godmaster 相关）**：虚空存在升起，更彻底地终结辐光。  
乐园默认可玩「梦无止境」为 canonical 解放向；循环封印作悲剧 IF。

⑨ **王国社会残响**  
钉子匠 **纳吉（Nailsmith）** 可被杀死或放生；画家、诗人、磨钉师、银行家米利布等构成废墟经济。感染不是简单「刷怪」，而是文明级精神瘟疫。

⑩ **结局后状态**  
Dirtmouth 人数随进度增减；神居可无限挑战；圣巢地理不因结局自动绿化——解放是意志层面的，废墟仍在。

**【主要人物】**  
- **小骑士 / 容器（The Knight）**（可操作主角）｜性格：沉默、行动即语言｜装备·能力：钉子、法术（复仇之魂等）、护符、冲刺/二段跳/幽暗披风等技能树｜弧光：弃子→弑神工具→虚空意志载体｜立场：契约者默认可映射为其或同行容器。  
- **霍瓦（Hornet）**（半血公主·守护者）｜性格：锋利、尽责、寡言｜装备·能力：针与丝线｜弧光：试探→阻拦→终局可并肩｜立场：防「不完美容器」害圣巢。  
- **苍白之王（The Pale King）**（旧神·王）｜性格：理智到残酷｜能力：虫国建立、容器工程｜弧光：为灭光而造虚空子嗣，终在白宫留白｜立场：已死/梦残。  
- **白皇后（White Lady）**（根之皇后）｜性格：悔恨、疏离｜能力：王室之根｜弧光：自我束缚于花园｜立场：可赐王徽碎片。  
- **辐光（The Radiance）**（光之旧神）｜性格：被遗忘的暴怒｜能力：感染、梦境统治、光弹神战｜弧光：被王取代→以感染复仇→可被虚空抹除｜立场：世界精神顶点级敌对。  
- **空洞骑士 / 纯粹容器（The Hollow Knight / Pure Vessel）**（封印容器）｜性格：残存痛苦与忠诚｜能力：钉、感染爆发、王室教习战技｜弧光：完美幻觉破灭｜立场：殿内囚徒。  
- **赫拉（Herrah the Beast）**（深巢女王·守梦者）｜性格：母性与交易｜弧光：以梦换霍瓦未来｜立场：守梦者。  
- **莫诺蒙（Monomon the Teacher）**（教师·守梦者）｜性格：求知｜能力：档案与水母奎拉（Quirrel）线｜立场：求真相被看见。  
- **卢瑞恩（Lurien the Watcher）**（守望者·守梦者）｜性格：忠于王｜立场：尖塔俯瞰泪水之城。  
- **奎拉（Quirrel）**（旅行者）｜性格：温和好奇｜弧光：护面甲→忆起莫诺蒙→湖畔告别｜立场：友。  
- **斯莱（Sly）**（商人·大钉子圣哲）｜性格：精明｜能力：售货与隐秘战力｜立场：Dirtmouth 经济。  
- **泽特（Zote the Mighty）**（自封英雄）｜性格：吹嘘无能｜弧光：可救可弃可再战｜立场：喜剧镜。  
- **格林（Troupe Master Grimm）**｜性格：戏剧性、仪式｜能力：火与舞｜立场：DLC 中立偏交易。  
- **乌姆（Uumuu）/ 灵魂大师等**｜区域 Boss 真名锚点，见势力与战场。

**【隐藏剧情 · 伏笔】**  
- 容器「无念无欲」教条本身是苍白之王的失败命题——残余意念导致感染反噬。  
- 霍瓦与小骑士可能同属虚空/王室实验光谱的两端。  
- 辐光是被虫群遗忘的飞蛾族神，遗忘即弑神。  
- 神居暗示「挑战本身」在喂养某种更高观赏。  
- 泽特的「生命尽头」与巨兽等彩蛋是苦中滑稽。  
- 丝之歌属另一时空/角色（霍瓦），不并入本顶点。

**【大事记时间线】**  
远古：辐光受飞蛾族崇拜 → 苍白之王崛起夺信 → 建圣巢 → 感染复发 → 容器计划与黑卵封印 → 王消失、守梦者沉睡 → 小骑士归来 → 破梦者 → 深渊觉醒 → 辐光战 / 虚空终局 →（可选）神居加冕。

**【叙事基调 · 雷区】**  
基调：冷诗意、废墟孤独、少量温暖 NPC、高难度精确动作。  
忌：把小骑士写成话痨龙傲天；把感染当普通红名怪无叙事；剧透时不标进度；把绝对辐光写成杂兵；忽略护符槽与技能门门。  
最早切入：Dirtmouth 井口或十字路长椅。

**【世界剧情线 · 区域因果细述】**  
十字路的「感染后」版本会替换敌人与背景音乐，提示圣巢是活的病体。泪水之城永雨来自上方水库与王的地理工程，雨水本身是文明遗迹。深巢的黑暗逼迫依赖萤光护符，把「可见性」做成资源。白色宫殿没有传统血条叙事，而是把王的自责写成尖刺与电锯的几何——通过平台杀即阅读史书。黑卵内战：殿内骑士既是兄长镜像也是国家罪孽的肉身化；霍瓦的插入结局强调「圣巢不需要第二个沉默的囚徒」。虚空海在终局中升起时，无数阴影手是被弃容器的集体意志，而非单一英雄光环。

**【主要人物 · 补遗】**  
- **伊斯玛 / 蒂索 / 奥罗** 等泪水之城与水道相关亡魂，提供钥匙与哀悼。  
- **面具师** 售卖血量面具碎片链。  
- **盐与泪的船夫** 连接盆地。  
- **收藏家** 与罐中生物，黑暗喜剧。  
- **磨钉师奥罗 / 钉子匠** 路线影响 Dirtmouth 人口与「匠人是否自杀」的伦理。  
- **布里塔 / 米利布 / 莱吉** 等市民事务，构成低阶任务板。  
- **诺克斯 / 神探求者** 服务 Godhome 元叙事。

**【地理 · 功能再注】**  
长椅 = 重生与读档伦理点；热熔岩与酸池 = 环境杀；梦境入口 = 平行战场；电车与stag 甲虫车站 = 解锁式快速旅行，象征王国基建残留。AI 写场景须先报区域名与是否已感染变体。

**【势力与战争伦理】**  
灵魂圣所把「研究」写成活体炼金；螳螂族把「外敌」定义为荣誉测试；蜂巢把「女王」写成感染前后的忠诚悲剧。契约者若屠城式清图，会与霍瓦的「守护」逻辑冲突——她杀的是威胁平衡者，不是经验包。

**【贵重物品 · 技能门】**  
蛾翼披风、晶心、暗影披风、帝王之翼、嚎叫哀号等能力对应地图门控：无冲刺勿进特定刺道，无强壳勿硬吃骑士马。苍白矿石升级钉子是全程 DPS 轴。梦境 Boss 掉落本质（Essence）解锁梦钉觉醒与守梦者定位。

**【隐藏 · 再补】**  
苍白之王与白皇后的繁殖是「计算」，霍瓦是「交易例外」。深渊灯与黑卵铭文是可阅读的史书碎片。格林的仪式可能服务于更高梦魇存在（Nightmare's Heart）。绝对辐光战的舞台从神居升至虚空外，是「游戏性神性」的顶点演出。

**【叙事执行 · 区域遭遇表】**  
低阶：十字路假骑士、绿径黄蜂、真菌孢子 Ambush。中阶：魂暴君、战士之墓、守护者。高阶：守梦者、格林、白宫。顶点：殿内骑士与辐光。每条遭遇写清「先听歌再看招」——本世界 Boss 战是音乐与模式识别。

**【契约者纪律】**  
你可成为第二容器、霍瓦的临时共犯、或 Dirtmouth 的清道夫。你不可无代价宣布「圣巢已恢复农耕乌托邦」——地理废墟与种族凋零是长周期设定。奖励应是护符、矿石、地图、stag 站，而非突然的现代枪械。

**【剧情加厚 · 黑卵政治】**  
三守梦者并非单纯障碍：赫拉要女儿未来，莫诺蒙要知识不被埋，卢瑞恩要秩序残影。杀梦者=杀三位历史上的「自愿牺牲者」的余晖，伦理重量应在文中留下痕迹。乌姆守护的是教师档案，不是无脑触手。

**【剧情加厚 · 虚空哲学】**  
虚空在官方叙述中接近「无光之海」，可淹没神名。小骑士取得虚空心后，不再只是工具，而是海的焦点。拥抱虚空结局把「个人英雄」扩成「物种级否定光」。乐园高阶任务可围绕「是否让虚空登陆地表」给世界加新伤。

**【剧情加厚 · DLC 咬合】**  
格林剧团把「火焰传递」写成巡回祭礼；Lifeblood 区域提供血蓝替代医疗哲学；Hidden Dreams 的白防御者与失落亲眷补王室武士史；Godmaster 则承认玩家已成「杀神者」并给神居加冕。

钉子有短前摇；法术耗灵魂；聚焦治疗占时间；冲刺无敌帧；护符改规则（如刺之握、迅刃、萨满之石）。写战斗要写「灵魂够不够再放一次撕吼」。面具即生命格，碎片收集是长线。

**【结局四分支操作定义】**

## 阶位切入点

> 阶位↔进度：一阶≈十字路/绿径杂兵期；二阶≈城与区域领主；三阶≈守梦者/梦境Boss/格林；四阶≈白宫/纯粹容器；五阶≈辐光标准战；六阶≈绝对辐光与虚空终局。顶点≈六阶。低阶严禁硬刚辐光本体。

**一阶（Dirtmouth 与十字路 · 学钉）**  
切入身份/时点：新坠落的容器或拾荒虫，锚定假骑士前后、十字路未完全感染变体。  
初始事件：长椅旁老人甲虫絮叨王国已死；假骑士挥锤砸塌通道，感染甲虫从地缝涌出。  
开场白建议：「你从井底爬出时，钉子还锈着。Dirtmouth 的风比地下暖一点——直到十字路的石头里渗出橙色的光，有什么东西在甲壳下笑。」  
关键NPC立场：**老人甲虫**——信息与乡愁；**伊塞尔达/科尼弗**——卖地图；**斯莱**——要你证明会打架才开全货；**假骑士**——敌对。  
主线钩子/支线：击破假骑士、买十字路地图、取毛噜/短冲刺前置；支线=泽特营救选择。  
危险度/规避：中低——规避深巢无图硬闯、黑卵、深渊。  
任务方向/奖励：基础护符、Geo、第一块面具碎片。

**二阶（泪水之城与区域领主）**  
切入身份/时点：已开城门的猎虫者，锚定魂之圣所、螳螂、真菌、水晶峰进度。  
初始事件：灵魂大师把实验体抛向你；或螳螂村要求你先杀叛徒再谈同盟。  
开场白建议：「泪水之城的雨洗不掉钉子上的灵魂残渣。有人在圣所里喊『飞升』，有人在蘑菇地里喊『入侵者』——你都得回答。」  
关键NPC立场：**奎拉**——可同行；**螳螂领主**——荣誉审判；**苏尔/灵魂暴君**——敌对学者；**霍瓦**——首次或再次试剑。  
主线钩子/支线：城内钥匙链、区域Boss、王徽收集；支线=钉子匠委托。  
危险度/规避：中——规避三守梦者硬闯、白宫、深渊无准备。  
任务方向/奖励：法术、中级护符、城内交通。

**三阶（守梦者与剧团）**  
切入身份/时点：持梦钉的破封者，锚定赫拉/莫诺蒙/卢瑞恩之一前后，或格林灯火已点。  
初始事件：你在梦中刺入守梦者；现实里黑卵裂缝增大；或格林邀请你加入火焰收集。  
开场白建议：「梦里的雨是彩色的。守梦者睁眼看你时，不像恨，更像在确认——封印再撑不久了，孩子。」  
关键NPC立场：**赫拉/莫诺蒙/卢瑞恩**——梦中敌对/哀矜；**霍瓦**——警告你别成下一个囚徒；**格林**——交易与试炼；**乌姆**——档案门神。  
主线钩子/支线：破三位梦者、格林仪式分支；支线=梦境竞技场。  
危险度/规避：高——梦战与剧团火；规避无虚空心碰结局战。  
任务方向/奖励：梦者精华、剧团护符、黑卵准入。

**四阶（白宫与深渊权限）**  
切入身份/时点：集齐国王印记相关、准备国王之魂的容器。  
初始事件：白色宫殿的光锯启动；或深渊记忆回放「被弃的你」。  
开场白建议：「王的宫殿不欢迎活物，只欢迎精确。你在尖刺上重新学会呼吸——每一下都像在签名：我不是完美的空。」  
关键NPC立场：**苍白之王残影**——无对话的结构压强；**白皇后**——悔恨赐予；**霍瓦**——血脉侧观察；**深渊阴影**——可响应。  
主线钩子/支线：白宫通关、虚空心、王徽终极；支线=神居初入。  
危险度/规避：高——平台即死；规避绝对辐光。  
任务方向/奖励：虚空心、结局权限、高级钉。

**五阶（黑卵与辐光）**  
切入身份/时点：黑卵内战与辐光标准战窗口。  
初始事件：殿内空洞骑士跪地再起，感染从甲壳爆开；或你借梦钉进入光之神域。  
开场白建议：「锁碎开的声音像整座城在吸气。黑卵中央的影子曾是『完美』——它看你，像看一面镜，镜里全是裂缝。」  
关键NPC立场：**空洞骑士**——悲敌；**霍瓦**——可插入结局；**辐光**——神敌；**奎拉**——或已湖畔谢幕（视时点）。  
主线钩子/支线：三结局分支选择；支线=清剩余梦Boss。  
危险度/规避：极高——辐光弹幕；**条件性胜利**：学阶段、护符、虚空。  
任务方向/奖励：结局旗标、神级纪念、圣巢历史改写权（有限）。

**六阶（绝对辐光 / 虚空心终局）**  
切入身份/时点：Godhome 万神殿或虚空终局战。  
初始事件：神居观众无声，绝对辐光降临；或虚空海在神光下升起无数手。  
开场白建议：「这一次光不再装成仁慈。弹幕写下旧神的名字，而你的影子比名字更长——圣巢在地下听着，谁先被遗忘。」  
关键NPC立场：**绝对辐光**——顶点敌；**神探求者**——观礼；**虚空存在**——可同化胜利；**霍瓦**——世界外的丝线（勿强行塞进神居除非 IF）。  
主线钩子/支线：绝对辐光击破、拥抱虚空；支线=万神殿全清。  
危险度/规避：贴近顶点。**六阶巅峰：存在·情报优先/条件性胜利**，严禁削成杂兵。  
任务方向/奖励：神主纪念、虚空权能碎片、苛政挑战解锁。


**一阶补充钩子**：科尼弗的哼唱作寻人任务；假骑士锤裂地面救出被压的弱虫可换Geo；泽特喊「伟大的泽特」时选择嘲讽或救助将改后续竞技场。  
**二阶补充钩子**：灵魂圣所实验记录可交给伊塞尔达换情报；螳螂同盟后出现「外客可过」的荣誉标记；霍瓦二战可能改在城或盆地。  
**三阶补充钩子**：三位梦者的杀序影响霍瓦对话温度；格林焰收集可与主线穿插；乌姆战前可读莫诺蒙残页降低信息差。  
**四阶补充钩子**：白宫分段死亡是预期内；白皇后花园的自我束缚根须可作隐藏对话；深渊灯谜要求安静移动。  
**五阶补充钩子**：黑卵战允许「替换封印」与「霍瓦共斗」两套开场白；辐光阶段转换要写光翼与剑雨。  
**六阶补充钩子**：神居观众无声是压迫源；绝对辐光后的虚空海要用「群手」而非单人光环；禁止把辐光写成可随意秒杀的杂兵。
