# 饥荒

## 剧情

**【作品来源】**  
《饥荒》（*Don't Starve*）由加拿大 **Klei Entertainment** 开发发行，2013 年正式版；单机后推 *Reign of Giants*、*Shipwrecked*、*Hamlet* 等 DLC。联机版 **《饥荒联机版》（Don't Starve Together, DST）** 持续更新，扩角色与Boss。气质：维多利亚怪谈 + 生存黑幽默 + 蒂姆·伯顿式剪影。本档案以 **威尔逊开局的「恒常」世界规则 + DST 常见主线Boss** 为底，兼顾单机经典事件。

**【世界定位】**  
被 **麦斯威尔（Maxwell）** 用 Codex Umbra 拉入的 **恒常（The Constant）** 生存位面：季节残酷、黑暗噬人、科学与魔法并行。玩家从 **威尔逊（Wilson P. Higgsbury）** 等角色中选一具名幸存者。一句话：活过冬天，别活成怪物的晚餐。卡片锚点：**暗影王座与麦斯威尔的交易**、**远古织影** 级威胁。

**【世界观 · 力量体系】**  
力量本源：科学机器/炼金引擎科技树、魔法（暗影、月）装备、烹饪、护甲与武器、角色特质（薇洛放火、温蒂与阿比盖尔等）。**理智（Sanity）** 与 **饥饿、生命** 三槽是核心资源；理智过低召唤暗影生物。黑暗中无光源会被查理（Charlie）秒杀级攻击。

战力阶梯：  
1. **兔子 / 蜘蛛 / 猎犬基础波** ≈一阶。  
2. **树精守卫 / 发条骑士 / 季节巨人（鹿角鹅、熊獾、龙蝇、麋鹿鹅等）** ≈二阶。  
3. **远古织影者 / 天体英雄 / 燃料织工等 DST 终盘级、麦斯威尔王座规则** ≈三阶顶点。  

**乐园阶位映射**：入门到首冬前≈一阶；巨人与遗迹≈二阶；远古/天体/王座终局≈三阶。宁低勿高，不写毁大陆。

**【地理 · 舞台】**  
程序生成：草原、森林、沼泽、岩石地、绿洲沙漠（RoG/DST）、月岛、洞穴与遗迹、远古入口、海洋（船难/DST海）。地皮决定资源与怪物。猪人村、兔人、鱼人构成原生社会。

**【世界剧情线】**  
① **交易与坠落**  
麦斯威尔以「秘密」诱骗威尔逊签字，威尔逊被拉入恒常，身旁只有两只火堆用火柴逻辑。第一夜必须光——这是世界铁律。

② **科学立命**  
砍树、挖矿、造科学机器，解锁背包、斧镐升级、农场与炼金。蜘蛛巢、蜂窝、猪人曼恩雷（Pig King）交易金块，建立经济。

③ **季节暴政**  
春雨蛙雨、夏热自燃、秋丰、冬冻与驯鹿——巨人 DLC 把季节做成Boss日历。猎犬袭击定时敲门。

④ **洞穴与遗迹**  
向下进入洞穴：无昼夜但有地震；遗迹中远古科技与暗影文化。单机冒险模式与麦斯威尔之门讲述「谁坐王座」。

⑤ **王座与查理**  
经典线：击败麦斯威尔后，**玩家坐上王座，麦斯威尔解放，查理（暗影女王侧）接掌威胁**——权力转移而非世界变好。DST 中查理成为暗影侧核心叙事。

⑥ **DST 扩展战争**  
织影者、天体词条、月亮与暗影阵营、海盗、海难残骸等更新把恒常写成长期战争沙盘。结局开放：永不「通关结束」，只有「又活过一季」。

**【主要人物】**  
- **威尔逊（Wilson）**｜科学家、剃胡得理智｜弧光：自负到求生。  
- **薇洛（Willow）**｜纵火狂、怕暗有打火机。  
- **沃尔夫冈（Wolfgang）**｜力量随饥饿变。  
- **温蒂（Wendy）**｜与鬼魂阿比盖尔（Abigail）。  
- **WX-78**｜机器人吃齿轮。  
- **薇克巴顿（Wickerbottom）**｜图书管理员、书术。  
- **伍迪（Woodie）**｜变海狸/鹅/麋鹿形态。  
- **温斯顿/麦斯威尔（Maxwell）**｜旧王、暗影魔术师。  
- **薇诺娜、沃尔特、沃恩、沃尔特等 DST 角色** 各具名特质。  
- **查理（Charlie）**｜黑暗一击与暗影女王叙事。  
- **猪王（Pig King）**｜交易中枢。  
- **远古织影者（Ancient Fuelweaver）** 等Boss 真名。

**【势力图谱】**  
猪人、兔人、鱼人、发条军（远古自动机）、暗影生物、月亮生物、巨人类、玩家营地联盟（联机）。

**【贵重物品】**  
暗影双刃、铥矿甲、 thrule 等远古装、懒人护符、传送类法杖、烹饪锅顶级菜（曼德拉汤等）、火魔杖、唤星者法杖、boss 掉落独特装备。

**【隐藏剧情 · 伏笔】**  
麦斯威尔曾是威廉·卡特，舞台魔术师交易失败；查理是其助手被暗影改写；恒常由「Them」更高存在窥视；月与暗影双路线。官方漫画与动画短片补史。

**【大事记时间线】**  
签约 → 坠落 → 首个科学机器 → 首冬 → 洞穴 → 遗迹 → 王座易主 → DST 长期版本战争。

**【叙事基调 · 雷区】**  
基调：滑稽残酷、饥饿喜剧、黑暗瞬杀。  
忌：无视理智与季节；把查理写成可聊天路人（默认可敌对规则）；无限资源开局无代价。  
最早切入：第一晚火堆旁。

**【求生百科细述】**  
食物腐败、怪物肉理智惩罚、烹饪锅公式、保温与降温、雨露与防水、盔甲耐久——全部是「可写进正文的死亡原因」。巨人战要风筝与地形，不是站桩。

**【角色选择即世界观】**  
每个角色开局道具与惩罚不同：薇克巴顿不能睡等规则改变策略。联机是多主角共用一个世界时钟。

**【麦斯威尔线细述】**  
冒险模式六章展示他者与王座；DST 角色麦斯威尔可玩，是「暴君成为幸存者」的反讽。

**【月亮与暗影】**  
月岛突变、玻璃与启蒙、暗影魔术消耗理智——选边会改可造物品与敌对表。

**【状态机】**  
日数、季节、是否入洞、是否击杀当季巨人、暗影/月进度。

**【名场面】**  
第一晚差点被查理拍死；猪王扔金；巨鹿进营地；王座上的麦斯威尔；织影者骨堆。

**【契约者纪律】**  
奖励给生存科技与季节装备；三阶才碰织影/天体。失败回流=档期死亡与基地烧毁，不是读档笑话（硬核）。

**【地理再补】**  
沼泽触手、沙漠蚁狮、龙蝇沙漠、仙人掌、盐矿、火山（船难）——按所开DLC/世界设置启用。

**【收束】**  
《饥荒》的故事是 **「你能在被设计来杀死你的世界里坚持签名之后的第几天」**。


**【世界剧情线 · 恒常的残酷日历】**  
签约场景（经典）：麦斯威尔的收音机/影子交易把威尔逊从「科学自负」扔进黑暗教学法。第一晚规则永久有效：无光则查理攻击，攻击在叙事里接近斩杀。科学机器与炼金引擎划分时代；魔法栏与暗影灯、火魔杖把理智当燃料。饥饿迫使你建立农场、炊锅、肉架；怪物肉与厨具公式是一门黑色营养学。蜘蛛巢分级、鱼人与猪人的外交（喂肉变友、满月变兽）构成土著政治。

季节是真Boss：冬无保温则冻死，夏无降温则自燃，春雨带来雷电与繁殖爆炸，秋是备战窗口。巨人（鹿角鹅、熊獾、龙蝇、麋鹿/麋鹿鹅等名随版本）按季登场，摧毁基地逻辑，逼迫风筝与牙阵。猎犬波按天数为表，不看你是否准备好。洞穴层改写光与资源：无昼夜但有地震掉矿；遗迹层展示远古文明的铥与宝石，以及暗影生物的生态位。

麦斯威尔线（单机冒险/叙事）：多层世界展示「他者」与王座。击败麦斯威尔后经典反转——你坐上王座，他自由，威胁转移到查理与暗影侧。DST把查理写成可持续反派资源，并把月与暗影做成可选科技树：启蒙、突变、玻璃、织影者骨廊、天体英雄等终盘内容把「活过一季」升级为「选边打神」。船难与哈姆雷特等DLC把地图换成海与猪城邦，规则仍是饥饿与季节（或等价暴政）。

**【角色即世界观（具名）】**  
威尔逊剃胡回理智；薇洛烧东西回精神、怕暗但有打火机；沃尔夫冈饿了变弱、吃饱变强；温蒂与阿比盖尔的鬼魂输出；WX-78吃齿轮、怕水（视版本调整）；薇克巴顿读书施法、低理智惩罚与睡眠限制；伍迪变身Beaver/Moose/Goose；麦斯威尔可玩后召唤暗影仆从却自削；薇诺娜建投石与发电机；沃尔夫冈之外还有沃姆伍德、沃特、沃尔特、旺达（时间表）、沃尔特等——每人开局物与惩罚都是一篇小伦理。

**【地理与生态】**  
地皮决定：草原牛、森林树精、沼泽触手与鱼人、岩石地矿、沙漠蚁狮与龙蝇、月岛突变、盐矿、火山（船难）。猪王（Pig King）白天交易金块，是经济央行。蜂王/蜂群、高脚鸟、电羊、钢羊等提供资源与风险。发条骑士/主教/战车是远古自动军。

**【贵重物品与科技】**  
暗影双刃、铥甲、皇冠类、懒人护符、传送魔杖、唤星者、火/冰魔杖、烹饪锅顶级菜（培根煎饼、曼德拉汤、蜜汁火腿等）、 tent/siesta 睡眠回理智、冰块机与灭火、避雷针、墙与牙阵、Boss 掉落专属（如巨人手杖类）。远古伪科学站解锁终盘。

**【隐藏史】**  
威廉·卡特→麦斯威尔的舞台交易；查理助手身份被暗影改写；「Them」更高存在的窥视；官方漫画与短动画补角色前史。月与暗影不是善恶简单二分，而是两套消耗与变异。

**【叙事基调】**  
滑稽残酷：角色讲话气泡幽默，死亡却很干净利落。写火光半径、饥饿音效、黄昏色变。忌无限食物无腐败；忌查理变闲聊NPC（除非任务特批）；忌无视季节表。

**【联机社会学】**  
多人共享天数与Boss；有人专职厨师、有人清野、有人作死引巨人。服务器回档政治是元规则。乐园应声明：是否开启洞穴、巨人、PVP、回档。

**【状态机】**  
天数、季节、是否入洞、遗迹进度、暗影/月等级、是否击杀织影/天体、王座IF。

**【名场面】**  
第一晚火柴慌；猪王扔金；巨鹿进营；麦斯威尔广播嘲讽；查理暗袭；织影者骨爆；月岛突变尖叫。

**【契约者边界】**  
可改基地位置、巨人击杀次序、角色选择。不可无代价关停「黑暗即死」铁律。失败=档死与基地焚毁叙事。

**【威尔逊与麦斯威尔的叙事轴】**  
威尔逊代表「科学可解释一切」的傲慢，被扔进不可完全解释的暗影法则。麦斯威尔代表「交易成功者」的诅咒：王座是权力也是囚笼。查理从黑暗一击成长为暗影侧人格化，使威胁有脸。DST让多人同时成为「被选中的实验体」，世界天数是共享暴政。

**【巨人逐只】**  
鹿角鹅（春）巢与蛋逻辑；龙蝇（夏）沙漠热战；熊獾（秋/冬相关）毁田；麋鹿鹅等——按世界设置启用。击杀给独特掉落，不击杀则持续拆迁。二阶正文应以「巨人逼迁」为常用钩子。

**【织影者战阶段感】**  
骨廊、影子、燃料、复活结构——三阶要写阶段，不写秒杀。天体英雄则是月侧终盘的走位考试。

## 阶位切入点

> 阶位↔：一阶≈首季求生；二阶≈巨人/遗迹；三阶≈织影/天体/王座规则。顶点≈三阶。

**一阶（火与饥饿）**  
切入身份/时点：任意幸存者第1–10天。  
初始事件：黄昏降至，查理在暗处抬手；库存只有浆果与树枝。  
开场白建议：「火柴还能划几根？科学机器的图纸在你脑子里发烫——但首先，别让火灭。」  
关键NPC立场：**猪人**——可友可敌；**猪王**——交易；**蜘蛛**——敌；**麦斯威尔**——广播嘲讽（视版本）。  
主线钩子：建营、过首冬准备；支线=挖坟、钓浮木。  
危险度：中——规避巨人、洞穴深层。  
奖励：科学/魔法初级、稳定营地。

**二阶（季节巨人与地下）**  
切入身份/时点：有盔甲与烹饪锅的中期队长。  
初始事件：鹿角鹅孵化场或熊獾毁田；或洞穴地震砸开通道。  
开场白建议：「地面在震，不是雷——是季节有名字的那种饿。」  
关键NPC立场：**巨人Boss**；**发条骑士**；**兔人**；队友角色真名。  
主线钩子：击杀当季巨人、遗迹探险；支线=Boss 掉落装。  
危险度：高——规避织影无准备。  
奖励：巨人大战利品、铥/宝石科技。

**三阶（王座与远古）**  
切入身份/时点：终盘远征队。  
初始事件：远古入口开启，织影者自骨与燃料成形；或天体英雄自月力凝聚。  
开场白建议：「暗影在地板下签名。有人告诉你王座只需要一个坐着的人——问题是，谁心甘情愿？」  
关键NPC立场：**远古织影者**；**查理**；**麦斯威尔**；天体侧Boss。  
主线钩子：织影战、阵营抉择、王座IF；支线=全图节日事件。  
危险度：极高。**三阶巅峰条件性胜利**。  
奖励：终盘武器、叙事易主权、世界法则碎片。
