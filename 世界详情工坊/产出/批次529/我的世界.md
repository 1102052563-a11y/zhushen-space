# 我的世界
<!--meta lib=主库 tiers=一、二、三-->

## 剧情

**【作品来源】**  
《我的世界》（*Minecraft*）由 **马库斯·佩尔松（Markus “Notch” Persson）** 创始，**Mojang Studios** 开发，现属 **微软（Microsoft）** 发行。2009 年经典版原型，2011 年 11 月 18 日正式版；分为 **Java 版** 与 **基岩版** 等。体素沙盒：采集、建造、合成、探索与有限的「终末主线」。文风可从轻松创造到硬核生存。本档案以 **官方生存模式主线（下界—末地）+ 广泛认知的生态与结构** 为准，不含特定模组宇宙。

**【世界定位】**  
无限（或巨型）程序生成的方块世界：主世界、下界、末地三层维度。无固定电影主角名——乐园以 **契约者本人** 为「玩家实体」视点；世界叙事锚为 **末影龙（Ender Dragon）** 与 **远古守卫/凋灵** 等 Boss 级威胁。一句话：从第一夜活到杀死龙，再决定当建设者还是继续挑战。卡片锚点：**击败末影龙** 与 **下界启程**。

**【世界观 · 力量体系】**  
力量本源：方块与物品的合成树、附魔、酿造、信标金字塔、装备材质（木石铁金钻下界合金）。无修仙境界。死亡：硬核永久死或普通掉落重来——高度可配置。经验值用于附魔与修砧。

战力阶梯（宁低勿高）：  
1. **动物 / 僵尸 / 骷髅 / 爬行者（初夜）** — 个体威胁街区级以下。≈一阶。  
2. **铁傀儡 / 全套钻石 / 下界早期 / 烈焰人塔** — 可清府邸、据点。≈二阶。  
3. **凋灵（Wither）/ 末影龙 / 监守者（Sculk 深暗之域）** — 区域到维度级 Boss 战表现。世界顶点≈**三阶**（不抬到毁星；龙与凋灵是「可被精良装备与工程击败的巨型敌」）。  

**乐园阶位映射**：初夜到铁器≈一阶；钻石—下界合金与海底神殿等≈二阶；凋灵、末影龙、监守者战≈三阶顶点。组织级红石与城建是工程优势，不单独抬个体到四阶。

**【地理 · 舞台】**  
- **主世界生物群系**：平原、森林、沙漠、雪原、丛林、恶地、蘑菇岛、樱花林等。  
- **村庄 / 掠夺者前哨 / 古城 / 试炼密室**：社会结构与战利品。  
- **矿洞与深板岩层**：资源带。  
- **下界（The Nether）**：地狱岩、堡垒、猪灵、疣猪兽、要塞残骸。  
- **末地（The End）**：主岛龙战、末地城、外岛、潜影贝、鞘翅。  
- **海洋与海底神殿、沼泽、红树林**：垂直内容。

**【世界剧情线】**  
① **第一夜**  
契约者生成于某生物群系，用拳与木工具砍树挖石，在日落前挖洞或搭屋。僵尸敲门、骷髅射击、爬行者的嘶嘶声定义「生存恐惧」。床跳过夜，设置重生点。

② **铁器与村庄政治**  
烧炼铁、合成盾与桶；发现村庄则进入交易与袭击（Raid）逻辑：不祥之兆、劫掠兽、卫道士。铁傀儡是村的武装。掠夺者前哨提供不祥旗帜与跨村战争。

③ **钻石、附魔与下界门**  
深矿找钻石，附魔台用青金石与书架；黑曜石门点亮进下界。下界是高温资源维：烈焰棒（酿造）、恶魂泪、下界石英、古代残骸→下界合金。猪灵以金易货，逻辑是「金本位外交」。

④ **要塞与末地门**  
主世界用末影之眼定位要塞，填满末地传送门。进入末地=承认进入屠龙阶段。

⑤ **屠龙**  
摧毁末地水晶（含铁笼水晶），弓箭与近战交互攻击末影龙。龙死后产生返回传送门、龙蛋与大量经验——**官方「通关」演出**，但世界不停服。

⑥ **后龙纪元**  
外岛寻末地城、鞘翅与潜影盒；可造信标、刷怪场、红石都市。召唤凋灵（灵魂沙+凋灵骷髅头颅）是玩家主动打开的灾难。深暗之域监守者惩罚「喧哗挖宝」。创造模式是乐园可开关的「规则词条」，默认叙事用生存。

⑦ **结局方向**  
无单一帝国结局：通关诗（End Poem）以「玩家被宇宙独白」作哲学收束，然后可回主世界无限建造。胜利=你还活着且世界被你改写。

**【主要人物】**  
注：官方无固定「小说主角团」，下列为 **具名实体/设定角色与可交互类型真名**。  
- **契约者 / 玩家（Steve/Alex 默认皮可换）**｜性格：由扮演定义｜能力：合成与工程｜弧光：难民→屠龙者→建筑师。  
- **村民（Villager）** 各职业（图书管理员、盔甲匠、牧师等）｜性格：交易逻辑｜能力：经济与铁傀儡｜立场：可被袭击。  
- **流浪商人**｜随机旅行商。  
- **猪灵 / 猪灵蛮兵**｜金与巴塞尔外交。  
- **卫道士 / 劫掠兽 / 唤魔者**｜袭击方。  
- **末影人（Enderman）**｜搬块、恨直视。  
- **末影龙**｜末地主岛Boss。  
- **凋灵**｜玩家召唤的三头灾难。  
- **监守者（Warden）**｜深暗之域声波杀机。  
- **铁傀儡 / 雪傀儡**｜可造生物。  
- **Notch / 官方传说中的「玩家神话」** 仅作彩蛋层，不进入战力。

**【势力图谱】**  
- **村庄文明**：交易、繁殖、袭击防御。  
- **灾厄村民系（Illager）**：木府、前哨、袭击。  
- **猪灵部落**：下界金权。  
- **末地生态**：龙、末影螨、潜影贝、末影人。  
- **古城市遗民痕迹**：深暗之域——无对话，只留结构与监守者。  
- **玩家城邦**：信标半径内的事实政权。

**【贵重物品】**  
- **下界合金装备**、**鞘翅**、**龙蛋**、**信标与金字塔矿物**、**附魔金苹果**、**图腾（不死图腾）**、**末影箱**、**恢复药水与力量药水**、**三叉戟/重锤** 等版本武器。  
- **末影之眼 / 烈焰棒 / 潜影壳**：维度进度门控。

**【隐藏剧情 · 伏笔】**  
- 通关诗（由 Julian Gough 等参与的文本）暗示玩家与「宇宙读者」关系——元叙事。  
- 末影人搬块、龙与水晶的循环，暗示末地曾有文明工程。  
- 远古城市与监守者：有人试图在深暗沉睡某物。  
- 红石是「可物理实现的魔法电路」。  
- 模组与服务器玩法是平行宇宙，默认不抬战力。

**【大事记时间线】**  
世界生成 → 第一夜 → 铁/村 → 钻石附魔 → 下界 → 要塞 → 屠龙 → 外岛鞘翅 → 可选凋灵/古城 → 无限沙盒。

**【叙事基调 · 雷区】**  
基调：沙盒自由、恐怖夜、工程浪漫、可选史诗。  
忌：强制单一剧情铁路；把末影龙写成毁星神；忽略维度门控；把监守者当小怪群刷；OOC 成纯创造无视饥饿与怪物。  
最早切入：第一日清晨出生点。

**【生存技术史细述】**  
木石铜（若启用）铁钻石链是文明史缩影；农业（小麦、胡萝卜、养牛）把「饥饿」变成可管理资源；附魔把经验转化成词条运气；酿造台把下界资源翻译成战场药水。刷怪塔与铁农场是工业革命，使个人战力被后勤放大——乐园应允许工程胜利，而非只认近战。

**【维度政治】**  
下界门是殖民通道：谁控制堡垒烈焰人房，谁控制药水工业。猪灵栏金交易是跨物种关税。末地则是「一次通关+反复农场」的猎场，龙可重生（终界水晶）在部分版本/机制下成为可重复内容——以当前版本官方逻辑为准书写。

**【村庄与灾厄】**  
袭击波次把村庄变成塔防；英雄之村效果是声望。封锁村庄或屠村会改变可用交易，是道德与机械双重选择。契约者可成为「雇庸骑士」或「灾厄同路人」。

**【深暗之域】**  
古城箱子有迅捷潜行等诱惑；喧闹值召唤监守者。此战鼓励潜行与声波箭，不鼓励钢铁直球——三阶也可用条件性胜利。

**【建造与叙事】**  
建造是《我的世界》真正的终局：大教堂、红石计算器、像素画、生存城市。AI 应把「墙是否完整、灯是否防刷怪」写成生存细节。

**【多人服务器社会学】**  
领地插件、经济、战争（若有）把沙盒变成政治服。乐园默认可单人硬核或合作服，须在开场声明规则：是否保留物品、是否 PvP。

**【主要结构清单】**  
废弃矿井、要塞、海底神殿、林地府邸、古迹残骸、试炼密室、村庄三种演化、沙漠神殿、丛林神庙——每个结构是一篇小型地城剧本。

**【生物伦理】**  
狼、猫、鹦鹉可驯；狐狸、美西螈等构成生态萌点；同时爬行者爆炸拆家——「可爱与恐怖同一世界规则」。

**【契约者纪律】**  
奖励给材质升级、鞘翅、图腾、信标，不给超出三阶映射的「创造模式秒杀一切」除非任务明示规则词条。末影龙=三阶顶点条件战：拆水晶、床爆、弓骑等策略皆可。

**【状态机】**  
A 无床无铁；B 铁村；C 钻石附魔；D 下界合金；E 龙前；F 龙后；G 鞘翅信标。禁在 A 发龙蛋。

**【名场面】**  
第一声爬行者；第一次下界门点燃；第一次被恶魂炸回门；龙死时的声效与经验雨；第一次飞出鞘翅；第一次不祥之兆进村。

**【与泰拉瑞亚边界】**  
同为沙盒，MC 主线更短、建造权重更高、战力天花板按清单仅到三阶。勿写成无限 Boss 链。

**【收束】**  
世界剧情的「完」只是诗与龙；真正的故事是契约者在方块上刻下的城与路。


**【世界剧情线 · 从第一夜到鞘翅的技术史诗】**  
第一日的目标不是英雄，是「活过日落」：木镐、工作台、煤炭火把、土屋或半地下掩体。饥饿值系统把「吃」写成行动力；和平模式可关怪物，但乐园默认生存难度下，夜是政治。床的出现改变战争：跳过夜晚等于停火协议，爆炸 crouch 附近的床在末地则是战术武器。铁器阶段解锁盾牌，使爬行者与骷髅的威胁从「随机死刑」降为「可管理风险」。

村庄是第一个他者社会：职业站点、繁殖、铁傀儡自动国防。不祥之兆与袭击把玩家拖入「英雄之村」声望政治——你可保卫，可抛弃，可屠村换战利品，道德与机械结果分叉。灾厄村民的林地府邸是大型合作地城：唤魔者、卫道士、掠夺者构成阶级军队。海底神殿用远古守卫的挖掘疲劳惩罚「硬挖」，要求牛奶与战术清场。

钻石层与黑曜石门是工业起飞：附魔台把经验翻译成锋利、保护、无限、经验修补；铁砧是昂贵的维护税。下界门点燃瞬间，叙事从「行星表面生存」跳到「殖民地狱维度」。堡垒的烈焰人产出烈焰棒，酿造台由此解锁；恶魂的火球可反弹；猪灵的金靴外交是跨物种关税。古代残骸与下界合金把装备上限再抬一档，使主世界钻石时代显得「古典」。

要塞由末影之眼定位——眼睛的合成需要烈焰粉与末影珍珠，强迫玩家完成下界功课。末地传送门满眼后，主岛屠龙成为官方「通关仪式」：摧毁水晶（含铁栏内），弓箭与床爆、末影珍珠抛射构成工程学打法。龙死经验雨、返回门、龙蛋，以及通关诗的宇宙独白，把沙盒突然抬成哲学课，然后立刻交还自由：你仍可回去盖城。

后龙纪元的真正奖赏是外岛：末地城潜影贝、鞘翅飞行权、紫颂果与潜影盒物流革命。信标金字塔用矿物块换状态光环，把个人武装扩成领地光环政治。凋灵是玩家自选的核按钮：灵魂沙与头颅的召唤式，对主世界建筑极不友好，掉落下界之星做信标芯。深暗之域古城用幽匿感测与监守者惩罚噪声采矿——三阶威胁中最强调「潜行条件胜利」的一环。试炼密室等新结构把挑战做成可重复的保险库。

**【地理 · 生物群系与结构政治】**  
平原适合开局与马；森林供木；沙漠神殿有TNT陷阱与战利品；雪原冻伤逻辑（细雪）；丛林神庙与竹子；恶地矿富；蘑菇岛无敌对——罕见的和平飞地。海洋纪念碑（神殿）控制海洋心与海绵。下界的疣猪兽兽栏、堡垒、要塞残骸、玄武岩三角洲各自提供不同死亡方式。末地虚空坠落是最终环境杀。AI 写场景必须先报：维度、生物群系、是否已屠龙、难度与游戏规则。

**【主要人物 / 实体档案扩写】**  
村民职业链：农民、图书管理员（附魔书经济）、工具匠、武器匠、盔甲匠、牧师（末影珍珠与红石）、制图师、皮匠、屠夫、石匠、渔夫、牧羊人、制箭师、牧师等——交易是和平战力。流浪商人与羊驼队提供随机稀缺。铁傀儡可人工堆南瓜与铁块召唤，是村与玩家的共享国防工业。雪傀儡产雪与吸引仇恨。狼、猫（爬行者克制）、鹦鹉、狐狸、美西螈构成可萌可战的生态工具。猪灵蛮兵不可交易，是金权外交的失败面。卫道士的斧与破盾、劫掠兽的冲撞、唤魔者的恼鬼与獠牙图腾，定义灾厄美学。末影人恨直视，可被引入敌对。监守者无眼却以声波定位，是黑暗的司法。

**【势力再写】**  
村庄 vs 灾厄是主世界文明战争模板；猪灵是下界金本位部落；古城市无对话居民，只留「曾有人试图让某物沉睡」的建筑论证；玩家信标半径是事实政权；末地是龙与末影人的虚空领地。

**【贵重物品扩写】**  
下界合金套、鞘翅+烟花飞行、龙蛋（纪念与炫耀）、不死图腾（图腾防死）、附魔金苹果、末影箱跨维物流、潜影盒便携仓储、三叉戟忠诚引雷、重锤风弹（版本）、恢复/力量/神龟Master级药水、满级信标（速度力量抗性跳跃再生）、经验修补+耐久的「永动装」。进度门控物：末影之眼、烈焰棒、紫颂果、回响碎片（定位古城）。

**【隐藏与元叙事】**  
通关诗以第二人称「你」对玩家说话，是少有的官方文学层。末地水晶与龙的循环暗示工程文明。远古城市中心的那一格「奇怪的结构」鼓励玩家不要乱挖——恐吓即叙事。红石与活塞使魔法等于电路。创造模式、旁观模式是规则词条。模组（工业、魔法模组）另宇宙。

**【大事记细年】**  
日0生成→夜1首尸→铁与村→钻石→门→下界桥与堡垒→眼睛→要塞→屠龙→外岛→信标→可选凋灵/古城→无限建造。

**【叙事基调执行】**  
写方块放置的咔哒声、夜声、门后的嘶嘶。允许幽默（苦力怕拆家）与恐怖（矿洞深处）。忌把末影龙写成行星爆破；忌无视饥饿与耐久；忌在无下界前提发鞘翅。

**【工程与战争经济学】**  
刷怪塔、铁农场、黄金农场（猪灵）、经验农场把个人战力后勤化。隧道掘进机式挖掘与TNT dup（若规则禁止则勿用）属于服务器政治。乐园任务应标明是否允许刷场。

**【契约者可改边界】**

## 阶位切入点

> 阶位↔进度：一阶≈木石铁/首几夜；二阶≈钻石—下界—府邸神殿；三阶≈凋灵/末影龙/监守者。顶点≈三阶。

**一阶（第一夜到铁器）**  
切入身份/时点：新生成幸存者，锚定尚未进下界。  
初始事件：日落倒计时，三只僵尸捶门；远处村庄的灯火与掠夺者巡逻二选一钩子。  
开场白建议：「太阳沉得比你砍树快。木门挡住第一只手，却挡不住爬行者的倒计时——你需要石头、煤，以及一张床。」  
关键NPC立场：**村民**——可交易或无视；**铁傀儡**——护村；**灾厄巡逻队**——敌对；动物中立。  
主线钩子/支线：活过第三夜、建农场、换铁套；支线=探矿洞。  
危险度/规避：中低——规避深暗、下界、深海神殿。  
任务方向/奖励：铁器、盾、床、最初村庄声望。

**二阶（钻石、下界与结构战）**  
切入身份/时点：附魔钻石或下界合金起步的探险者。  
初始事件：烈焰人塔火力网；或林地府邸唤魔者召唤恼鬼；或海底神殿远古守卫给挖掘疲劳。  
开场白建议：「黑曜石门另一侧是红天下的地狱岩。猪灵盯着你的金靴——在下界，外交有时比剑快。」  
关键NPC立场：**猪灵**——金；**烈焰人/恶魂**——敌；**唤魔者**——府邸核心；村民——高级交易。  
主线钩子/支线：烈焰棒、末影之眼、府邸/神殿战利品；支线=金农场外交。  
危险度/规避：中高——规避未准备的龙、凋灵、监守者。  
任务方向/奖励：下界合金、附魔书、图腾、药水工业。

**三阶（维度Boss 与深暗）**  
切入身份/时点：满配探险队，锚定屠龙/凋灵/古城。  
初始事件：末地水晶激光sweep；或凋灵在主世界被召唤后炸村；或你在古城踩响声纳。  
开场白建议：「末地主岛没有昼夜，只有龙的翅风。水晶还在亮——你数过几颗，就知道今晚是否够箭。」  
关键NPC立场：**末影龙**——顶点；**凋灵**——人造灾；**监守者**——深暗顶点级；**末影人**——可被利用仇恨。  
主线钩子/支线：屠龙、鞘翅、凋灵星、古城宝；支线=龙后外岛线。  
危险度/规避：高—极高。**三阶巅峰：条件性胜利**（工程、药水、拆机制），严禁无准备硬刚。  
任务方向/奖励：龙蛋纪念、鞘翅机动、信标满功率、深暗战利品。


**一阶补**：教村民造船交易链；第一次爬行者拆墙后的连夜重建；煤矿与火把政治。  
**二阶补**：烈焰人房间清场工程；金靴换终末之珠；府邸三层攻坚分工；神殿潮汐与夜视药水。  
**三阶补**：水晶计数喊话；床爆安全距；凋灵在基岩层或末地的「污染隔离」战术；古城羊毛静音与声波箭；鞘翅烟花物流网。
**一阶（扩写七字段密度）**  
切入身份/时点：第1–5日清晨的新生契约者，难度默认普通/困难，坐标未备份。  
初始事件：西侧森林传来爬行者嘶声，东侧似有村庄钟声；背包只有合成表与空胃。  
开场白建议：「太阳比你的木斧更着急。把第一块圆石放进熔炉时，你听见门外有东西在数你的心跳——其实是骷髅在上弦。」  
关键NPC立场：**村民**——可交易可无视；**铁傀儡**——护村不护你；**灾厄巡逻**——给不祥；**流浪商人**——高价。  
主线钩子/支线：活过第三夜、建麦田、换铁；支线=探洞取煤与铁。  
危险度/规避：中低；规避下界门、深暗、深海。  
任务方向/奖励：铁套、盾、床、村声望。

**二阶（扩写）**  
切入身份/时点：附魔钻石或下界起步的探险队长。  
初始事件：堡垒烈焰人火力网封锁走廊；或府邸唤魔者召唤恼鬼撕辅助。  
开场白建议：「金靴子是下界护照。猪灵的鼻子嗅到金属，才肯把终末之珠推过栏杆。」  
关键NPC立场：**猪灵**；**烈焰人**；**唤魔者**；**图书管理员村民**。  
主线钩子：烈焰棒、眼睛、府邸/神殿；支线=金农场。  
危险度/规避：中高；规避龙与凋灵与监守者。  
任务方向/奖励：合金、附魔书、图腾、药水。

**三阶（扩写）**  
切入身份/时点：满配屠龙/隔离队。  
初始事件：末地水晶激光连成网；或古城喧闹值爆表。  
开场白建议：「你数过水晶吗？漏一颗，龙就会在你换箭时回满——末地不讲武德，只讲工程。」  
关键NPC立场：**末影龙**；**凋灵**；**监守者**；**末影人**。  
主线钩子：屠龙、鞘翅、凋灵星、古城；支线=外岛城。  
危险度：极高；条件性胜利。  
任务方向/奖励：龙蛋、鞘翅、信标、深暗物。
