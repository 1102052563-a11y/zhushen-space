## 剧情

**【作品来源】**
「対魔忍深渊侵蚀」为 LiLiTH / BLACK LiLiTH 对魔忍（Taimanin）宇宙的 **IF 灾变分支档案**，非单部商业作品原标题；骨架取自《対魔忍ユキカゼ》《対魔忍ユキカゼ2》、OVA、Action Taimanin / RPGX 公开设定与本传阿莎姬线的组织图谱。气质：近未来暗部忍者·魔族企业渗透·都市舆论灾难·仪式政治。**本篇副标题「雪风先驱」专线**：深渊与诺玛德（Nomad）不先拿「最强」井河阿莎姬开刀，而选定 **新世代象征·水城雪风（Mizuki Yukikaze）** 作为公开侵蚀的第一先驱样本——用「英雄堕落直播」瓦解对魔忍招募、学园信任与民心。只写选型逻辑、诱饵任务、雷狱实验、小队拆解、舆论战、六阶夺回与假死翻案；成人向主题在原作中极重，本档案 **克制处理**：不写色情细节与身体改造过程，只保留「回路反相 / 人格锚松动 / 仪式椅 / 转播核心」等战斗—政治层机制。人物真名锚定水城雪风、水城不知火、秋山凛子、秋山达郎、井河阿莎姬、井河樱、八津紫、埃德温·布莱克等。

**【世界定位】**
近未来日本暗部：对魔忍以血脉忍法猎魔，诺玛德以企业壳经营黑产与政治渗透。本线时点偏 **学园化/小队任务成熟期**——五车学园（Gosha）已培养出以雪风为代表的「海报世代」：雷击对魔忍、任务狂、双马尾好胜者，是招生片与内部宣传的脸。深渊作为人间—魔界之外的 **第三层污染源** 已学会开口；布莱克批示「先驱，不要立刻容器」——目标是 **故事**，不是一具尸体。契约者切入的是地区灾难级舆论—仪式双线战场：在「公安要死的英雄」与「阿莎姬要活的棋手」之间排程，断转播、拔先驱椅缆、注入反相雷锚。一句话给 AI：这是「把下一世代偶像做成深渊样板间」的 IF，六阶任务世界。

**【世界观 · 力量体系】**
本系列「対魔忍深渊侵蚀」= 对魔忍宇宙 **IF 灾变线**：在人间—魔界既有渗透之上，叠加「深渊（Abyss）」高阶污染。深渊改写血脉、忍法回路与人格锚点；裂隙、刻印、融合仪式、祭品链、**先驱模板** 是其展开方式。档案只写战斗、组织、舆论与真名人物。

世界基本规则：
- **人间侧**：近未来日本。对魔忍 = 血脉忍法（元素/异能/咒术）+ 对魔粒子强化身体 + 对魔忍装 + 忍刀/特化兵器。组织形态可含井河之里、公安 Section 系、五车学园。死亡：凡人与下忍即止；上忍级可凭意志与医疗撑过重伤，但深渊化完成后人格可能不可逆。
- **魔界侧（Dark World）**：吸血鬼、那伽、半兽人、梦魔等；诺玛德以合法企业壳经营改造、黑市、情报与政治游说。
- **深渊侧（本系列增量）**：以裂隙渗入；污染阶段——接触 → 回路紊乱 → 人格锚松动 → 深渊化体 / 先驱态 → 仪式核心。可被意志、特殊抑制药、反相锚延缓，不可被「喊口号」清除。先驱线的关键增量：**公开叙事** 本身是仪式燃料——观众越多、字幕越「合理」，人格锚越松。
- **雪风线特化机制「雷狱反相」**：针对水城家雷遁回路设计的实验场。正常雷击输出被场内装置相位反转，击中敌人的能量回流到施术者体表纹路；「打得越猛，纹越亮」——好胜性格被做成燃料泵。

战力阶梯（破坏力表现，宁低勿高）：
1. 凡人·街头炮灰 ≈ 一阶：热武器可致命；对魔/深渊皆是资源与炮灰。
2. 下忍·低阶魔物·裂隙渗出体 ≈ 二阶：碎砖裂石，惧正规编制。
3. 中忍·成型魔物·深渊触手群 ≈ 三阶：端据点级，可取首。
4. 上忍·精英 Boss（阿莎姬 / 胧级近战规格）≈ 四阶：建筑群级近战与高速；一城仅数人。雪风战前峰值约落在中上忍窗口（雷系爆发高、续航与大局观未及阿莎姬）。
5. 城市级·东京王国 / 学园信任崩坏 / 全国转播事故 ≈ 五阶：治安与政治同崩。
6. **地区灾难·深渊核心 / 布莱克权柄 / 先驱完成态 + 转播网 ≈ 六阶（本条目唯一覆盖）**：凡俗军队近乎无效；对话、人质、仪式节点、剪辑权往往比正面对轰更关键。

乐园阶位映射：街头/见习≈一；下忍·渗出体≈二；中忍≈三；上忍·阿莎姬/胧级≈四；城市灾难·舆论崩盘≈五；**深渊核心·布莱克·完全先驱/容器顶点≈六阶**。本世界覆盖一至六阶。面对顶点只能断转播、拔椅缆、布设反相锚、公开罪证与撤离关键人物取得条件性胜利，不可用无故失能降低威胁。

**【地理 · 舞台】**
- **五车学园 / 五车之里（Gosha）**：对魔忍培养核心；宿舍、训练场、广播塔、招生中心。先驱线的舆论震中——公共屏被黑后，家长撤学、新生拒报到在此最先发生。
- **任务都市（关东—东京表层）**：电车、写字楼、便利店、广告墙；夜间是对魔忍与魔族/渗出体的猎场。公共大型广告屏是「先驱宣言」的第一传播面。
- **雷狱（本线核心实验场）**：诺玛德—深渊合作的三层设施，专锁雷系回路。外层雷场（反相）、中层精神压厅（假视频循环）、内层先驱椅（连转播塔心跳同步）。选址常在废弃地铁枢纽或东京王国地下扩建区，静默区边缘电子设备失灵、气味金属甜腥。
- **凛子安全屋**：秋山凛子拒撤后自建的绝缘据点；墙贴真实任务合影（对抗假视频），摩尔斯密钥本只口头更新。地址不进芯片。
- **东京王国（Tokyo Kingdom）转播塔节点**：人工岛犯罪都市的高功率转播/黑市直播基础设施；诺玛德「导演组」要把先驱「播出」的物理节点。
- **井河之里 / 对魔指挥链路**：阿莎姬外线任务归途的命令中枢；「谁杀雪风，先过我」从此发出。
- **公安 Section 联合指挥所**：销毁令、媒体管制、法医小组所在；要「死的英雄」好管理。
- **混沌竞技场 / 诺玛德设施网（背景）**：资金池与改造遗产；本线不展开肉体改造，但实验室走廊、样本罐、雇佣兵编制可作侧翼战场。
- **魔界侧通道**：远程全息与样本车；布莱克多不亲临，用故事评估。
- **水城家相关记忆锚（非地理但可场景化）**：不知火失踪任务的档案室、旧雷遁训练场——精神压会调用这些影像。

**【世界剧情线】**
① **前史·为何是雪风（选型会议）**  
对魔忍宇宙公开底色：近未来日本，魔族渗透政府与企业；对魔忍氏族世代以忍法对抗。水城家以 **雷遁** 闻名；水城不知火曾是精英对魔忍，在黄泉原（Yomihara）一类暗部都市任务中失踪（原作时间线约数年前），成为雪风执念与好胜的根。雪风本人：约十八岁量级、五车学园现役、雷击对魔忍（雷撃の対魔忍），双特殊手枪作雷弹触媒，战斗风格舞步化、爆发极高；性格好胜、嘴硬、怕输、对同伴极度护短，对身材与「配不上母亲/凛子」有自卑——这些在先驱算法里全部是 **可利用接口**。

诺玛德内部选型会议（本 IF）：议题「公开侵蚀谁成本最低、收益最高」。结论：
- **井河阿莎姬**：最强规格，失败率高，强攻会变成英雄史诗而非堕落样板。
- **井河樱**：血亲线，舆论易读成「欺负弱者」，不利招募崩坏叙事。
- **水城雪风**：高人气 + 未到不可替代顶点 + 雷系与深渊能量亲和试算高 + 「未来希望」招生海报脸。打倒她 = 打倒招生海报。
布莱克一锤定音：**先驱，不要立刻容器。目标是故事。** 胧系/合作线提供精神压与假视频技术；深渊侧提供反相场与先驱椅心跳同步协议。阿莎姬此时在外线任务，回援存在时间差——这是窗口设计的一部分。

② **诱饵任务·第一击**  
公开任务简报：关东某废弃地下枢纽出现「低危渗出体群」，适合雪风—凛子小队练手式清剿。真实结构：外圈杂兵是饵，内圈闸门分离搭档，中央是雷狱入口。雪风按性格 **率先冲入**；凛子被电磁闸与渗出体潮隔在外圈。名场面：雪风在雷场里大笑「还能打」，雷弹连发——纹路却从手套爬上小臂，瞳孔开始映出黑镜反光。她不是「突然黑化」，而是 **好胜被做成泵**。凛子外侧强突三次均被「同伴已安全撤离」的伪造通讯干扰；她砸碎耳机，改用预先约定的雷击节奏暗号（后文化为摩尔斯密钥）。

阶段因果：① 雪风进入半先驱态（回路紊乱 + 人格锚仍在）；② 凛子从「搭档」升级为「唯一可信的密钥持有者」；③ 对魔指挥链收到的是「任务顺利」的假回传，延误两小时。

③ **先驱宣言·公共屏之夜**  
诺玛德黑客组占用城市大型广告屏与部分学园内屏：循环雪风半先驱态影像 + 字幕「对魔忍下一世代已接受深渊」。剪辑故意保留她笑着放电的镜头，删掉她喊凛子名字的片段。效果链：
- 五车招生咨询电话被打爆后断崖；
- 家长连夜接学生离校；
- 公安销毁组立案：「公开污染源，建议物理清除以稳民心」；
- 内部论坛出现「她是内鬼」与「救她」两极撕咬。

阿莎姬归途全息只甩四个字级命令的内核：**「谁杀雪风，先过我。」** 此令把销毁组从「执行者」变成「政治难题」——开枪等于与人类侧近战顶点开战。布莱克评价（远程）：「很好。故事有冲突了。」

④ **凛子线·安全屋与节奏**  
秋山凛子拒绝撤离编制。她在绝缘巷弄建安全屋：铺木与橡胶隔雷，墙贴与雪风的真实任务合影与达郎寄来的旧合照一角（本线 **达郎多在后方支援/通信**，不强制推恋爱线，只作人格锚素材）。她录下雪风仍清醒时段隔墙传来的雷击节奏，整理为摩尔斯表：例——三短=别；两长=杀；短长短=救；并 **每日改校验位** 防诺玛德伪造节奏。六阶契约者可被编入「节奏破译组」或「转播塔爆破组」；入屋须先背当日表，背错按间谍规程绑审。

同期：八津紫率破门预备队待命；井河樱负责影遁侦察雷狱外围缆线；学园侧部分学生（RPGX 互文可点名相州蛇子等）自发「夜间砸屏/涂鸦遮倒影」——合法边缘的舆论自救。

⑤ **四方棋盘（六阶政治力学）**  
24～72 小时窗口内四方目标互斥：
- **夺回派（阿莎姬·凛子·紫·樱）**：断转播、拔缆、反相锚、假死或公开翻案，保雪风人格。
- **销毁派（公安 Section）**：要「死的英雄」好管理；法医小组已备尸检流程。
- **故事派（诺玛德导演组 + 布莱克）**：要播满关键三小时叙事固化点；击杀若发生在镜头前，会触发「英雄死亡特写」预设，仍算他们赢。
- **深渊本身**：要完成先驱模板数据，供后续元素系复制。

契约者每签一份合作就多一条出卖链。阿莎姬底线是「活人优先于样板」；凛子底线是「节奏为真则不杀」；公安底线是「镜头前必须有个结束」；布莱克底线是「有趣优先于道德」。

⑥ **雷狱结构与强袭规则**  
三层细部（作战必读）：
- **外层雷场**：反相雪风每一击；绝缘毯、木质/陶瓷近战、非雷系异能有效；纯火力打散渗出体但难破场。
- **中层精神压厅**：循环「凛子已撤」「学园抛弃你」「不知火在深渊等你」假视频；破法是 **凛子真人出镜对口令**（或预录且带当日校验位的真声纹，假声纹会触发二次加压）。
- **内层先驱椅**：缆线接王国转播塔；椅上心跳同步；**拔缆优先于击倒人**；击杀会触发英雄死亡特写镜头。

强袭分四门：阿莎姬牵制半先驱态雪风近战；紫破门清雇佣兵；樱切缆与影路送锚；凛子/契约者负责节奏密钥与反相雷锚注入。战斗规则铁律：**打断转播核心 > 击倒雪风**；击倒她若无净化与剪辑权，会在镜头前「死成英雄」反而完成叙事献祭。布莱克要的是故事，不是尸体。

⑦ **高潮·夺回、假死与翻案排程**  
成功路径（本篇默认条件性胜利）需同夜完成：
1. 技术/爆破切断或污染转播核心（硬砸屏会被拍成「对魔忍毁灭证据」，更优是插播阿莎姬声明或技术性花屏）；
2. 注入反相雷锚（凛子节奏密钥校准），逼雪风 **自断过载** 而非被外力斩杀；
3. 阿莎姬近战收束，逼出仍清醒的那一句名字（凛子/母亲/任务编号任一生效）；
4. 视公安压力启动 **假死双层谎**：对销毁组——尸斑药、冷皮肤、法医签字；对同伴——指环下心跳针、约定复苏词。只骗一边会崩。
5. 夺回后 **一小时内** 阿莎姬出面公布真相；超过约三小时，假叙事固化，假死也难翻案。

代价：雪风雷系永久损伤或需长期抑制药；峰值雷遁可能无法维持战前水平，可转教职、战术参谋、短线奇袭、「知棋者」小队。性格从「还能打」变为「算得清」——这是成长，不是削弱式羞辱。

失败路径：先驱完成 + 全国叙事固化 → 学园信任崩（可接系列「学园沦陷」等他篇）；或镜头前击杀 → 深渊获得「殉道英雄」模板，招募仍崩且对魔忍背锅。

⑧ **残局·新常态**  
雪风从任务狂变成「知道自己是棋」的人；凛子更冷更护，密钥本不停更新；阿莎姬对布莱克的债又加一笔；公安销毁组内部出现裂痕（有人恨阿莎姬挡枪，有人松口气）；诺玛德仍持有半套先驱模板数据——战争未结束。世界进入「深渊会打舆论战」的时间轴。

⑨ **微观可介入（六阶仍适用）**  
护抑制药车队；伪造成功「清除」骗过销毁组改期；在静默区建光传令链；砸/改公共屏而不留把柄；安抚学园宿舍恐慌；识破假摩尔斯；护送法医被调包的「尸体」出城；黑市收购绝缘毯与静默呼吸器；截诺玛德样本车（本线样本是「节奏录像」与「雷纹皮屑」，不是色情素材）；与校长权限代行谈判停课但不解散编制。

⑩ **蝴蝶效应边界**  
可改：转播是否播出完整、雪风伤残程度、假死是否启用、某销毁组军官是否被策反、安全屋是否暴露一次。  
不可无代价改：她曾被算法选为先驱；雷狱反相逻辑；布莱克要故事；「三小时叙事固化」社会心理规则。  
硬改「雪风从未被捕」→ 先驱换成他人，本篇主题改判，乐园任务易判失败或换世界。  
硬改「一刀砍死布莱克」→ 回流为更大规模故事战与二次点名。

⑫ **与原作《対魔忍ユキカゼ》的互文锚（供 AI 不 OOC）**  
原作主轴是雪风与凛子潜入黄泉原/暗部设施寻母、遭遇背叛与堕落路线分歧；达郎视角的无力与守护是情感核。本 IF **不复刻** 该潜入卖身线，只继承：① 寻母执念可被精神压迫价；② 凛子是最硬搭档锚；③ 好胜与自卑并存；④ 雷系双枪舞步战。成人向原作元素在本档案全部降维为「回路/锚/仪式/舆论」机制，禁止用色情场面替代战术进度。

⑬ **先驱模板的技术含义（给六阶奖励与后续）**  
完成态先驱模板 = 可复制到其他元素系的「公开侵蚀剧本」：选海报脸 → 反相其最得意忍法 → 假视频拆搭档 → 转播固化。若本线夺回成功，对魔忍侧应销毁模板硬盘并公开诺玛德导演组名单；若只救人未毁盘，他篇可出现「仿雪风」的假先驱影像。契约者奖励应是：反相雷锚残片、摩尔斯作战条令、六阶舆论战权限、凛子/阿莎姬有限信任——**不发「开裂隙」或「完美好友光环」**。

⑭ **时间轴上的位置（系列内）**  
可接「裂隙开启」被压回之后数周至数月：深渊改用「点名象征」而非硬开大裂隙。与「阿莎姬深渊化」互斥优先——本线明确 **不先动阿莎姬**；若玩家世界已经历阿莎姬容器线，则本线改为「第二波先驱」或跳过。与「学园沦陷」是失败分支下游。与「三代祭品」若同存，雪风对「算法点名」可有预创伤，破防更快也醒得更快。

**【水城家与雷遁战术细节（本线可用）】**
水城一脉雷遁不以「越大越好」为荣，而以 **相位可控** 为家训：雪风战前常用双枪把雷收成弹道，避免误伤队友；雷狱装置专克这一点——它诱导她把相位拉满，再整体反相。懂家训的人（凛子、不知火残档读者、少数水城旁支）能听出她放电时的「家声」与装置噪声的差别：家声短促有节拍，噪声发黏发长。契约者若在中层厅播放一段水城旧训练录音（真档），可让雪风在半先驱态下出现 3～5 秒的相位回正窗口，足够注入一次反相锚。阿莎姬不会水城家训，所以强袭时必须有凛子或密钥在场——这是编制硬约束，不是感情戏。

**【达郎与后方通信网】**
秋山达郎本线默认不进雷狱内圈。他的价值是：① 提供未被导演组污染的旧影像与语音；② 在学园侧维持一条「家长联络」反谣言热线；③ 若强行闯入，极易被剪成「软弱的同伴」镜头给精神压加燃料。玩家可派他跑后勤，不可默认他冲锋或被写成羞辱核心。凛子对他的态度是「你活着比你英勇有用」。

**【诺玛德导演组的镜头语言】**
他们预设三种成片：A「英雄自愿深渊」（雪风笑着放电）；B「英雄殉道」（椅上心跳归零特写）；C「对魔忍内鬼剪辑」（用阿莎姬挡枪画面反证「组织包庇污染源」）。夺回派每做一个动作，都要问自己在喂哪一种成片。污转播的正确做法不是黑屏（黑屏=审查联想），而是 **噪声花屏 + 阿莎姬持刀记者会** 的合法权威画面。

**【抑制疗程与战后战力诚实】**
成功夺回后雪风进入 2～6 周抑制疗程：雷纹退为浅痕，但爆发上限下降一档观感（仍远超凡人，勿写成废人）。可执行任务类型：短线奇袭、诱饵局、教学演示、舆论反制出镜。不可立刻：连续三日高强度雷狱级输出、单独担任招生形象大使（民心未稳）。阿莎姬会把她编入「知棋者」小队，专打「看起来像任务狂其实在布局」的局——性格弧光在此落地。

**【井河樱的缆线地图】**
影遁能摸清的是塔—狱之间的三根主同步缆：红主、蓝备、黑伪装。只剪红会自动切蓝；必须红蓝同秒或先揭黑。黑缆走维护井，雇佣兵最少，但有深渊渗出薄膜，需静默呼吸器。樱的报告习惯用颜色代号，契约者听不懂会误剪——入队前背一次。

**【公安内部的两派备忘】**
销毁组并非铁板：强硬派要今夜枪决式结束；务实派怕与阿莎姬开战导致编制内斗上新闻。六阶谈判对象应是务实派：用「假死+真相发布」换他们收回狙击许可，条件往往是「24 小时内必须有可向大臣汇报的结束叙事」。你给的结束叙事必须同时满足夺回派伦理，否则凛子会退出协作。

## 阶位切入点

> 阶位↔原作强度：一≈凡人/见习；二≈下忍·渗出体；三≈中忍；四≈上忍·阿莎姬/胧级；五≈城市灾难·舆论崩盘；**六≈深渊核心/布莱克权柄/先驱完成态+转播网总压**。映射与剧情条「乐园阶位映射」一致，按《阶位战力图鉴》宁低勿高。本世界覆盖一至六阶。世界顶点（埃德温·布莱克·深渊先驱模板完成态）为跨阶存在，切入一律规避硬刚、优先争取情报、仪式节点与剪辑权；只能条件性胜利。

**一阶（五车学园·失联学员与雷击课）**

切入身份/时点：以五车学园的新生、器材室助理或秋山凛子批准的课后巡查员切入；锚定雪风仍在正常训练、招生宣传尚未被深渊利用的时期。

初始事件：雷击训练后，一名见习学员没有从靶场返回，只留下被烧穿的练习靶编号。**水城雪风** 想立即沿电痕追人，**秋山凛子** 要先清点在场名单；广播室的门却被人从里面反锁，疑似有人正删改训练记录。

开场白建议：「训练场的焦味还没散，靶纸中央多出一个不属于课程的黑洞。水城雪风转着特制手枪，急得几乎要冲出警戒线；秋山凛子把名单塞给你：『先告诉我少了谁。找人靠脚，不靠猜。』」

关键NPC立场：**水城雪风** 好胜且急于救人，会接受能跟上节奏的协力者；**秋山凛子** 优先核实人员与退路；**秋山达郎** 可帮忙保管训练器材，但尚不能替代前线忍者；**井河阿莎姬** 只认可有明确证据的紧急调动；**相州蛇子** 能在学生间确认谁见过失联者。

主线钩子/支线：主线是找回失联学员并保全广播室记录。支线可检查靶场电源、追查伪造的请假条、替达郎把被盗的训练刀送回库房。不可让新生携带真实雷击触媒离校。

危险度/规避：低。规避训练场残余电流、未经允许闯入宿舍和把学生传言当证据；一阶只做搜寻、报告与撤离。

任务方向/奖励：获得学园临时通行、训练场原始记录、凛子的巡查推荐或雪风的实战观察评价；无雷遁、无深渊样本奖励。

**二阶（关东地铁·渗出体与断电站台）**

切入身份/时点：以学园下忍实习生、凛子小队的通讯员或地铁维护组护卫切入；锚定低阶渗出体从地下设施涌出、雪风与凛子被派去进行常规清剿的夜晚。

初始事件：地铁末班车停在无电站台，渗出体吸走照明并围住三名维修员。**秋山凛子** 要你守住出口和人数，**水城雪风** 要以低功率雷弹劈开闸门；维修主管却拿出一张通往废弃枢纽的错误疏散图。

开场白建议：「站台灯灭掉的顺序像有人从远处吹熄蜡烛。凛子把你推到出口标线内，雪风的枪口亮起细小电弧：『人数别错。错一个，我就得回头找一个。』」

关键NPC立场：**秋山凛子** 用空间忍法救急但需保留体力；**水城雪风** 要快速破门，反感拖延却会服从准确计数；**秋山达郎** 在后方无线电中整理维修图；**井河樱** 可从影路确认维护孔是否安全；**相州蛇子** 能协助安抚受困学生家属。

主线钩子/支线：主线是关停渗出点、带出维修员和核对错误地图来源。支线可回收下忍遗落的信号器、保护达郎的通讯中继、标记通向废弃枢纽的电缆。不得追入超出编制范围的深层空间。

危险度/规避：中。规避黑暗死角、积水导电区、错误疏散图与脱离凛子行动；二阶可以处理渗出体，不可独自挑战刻印设施。

任务方向/奖励：可得下忍出勤记录、绝缘披布、地铁维护图更新或凛子的短距撤离指令；奖励用于基层救援而非攻击强化。

**三阶（黄泉原线索·失踪档案的交接）**

切入身份/时点：以学园档案审计员、中忍级护送协力者或秋山达郎的情报联络人切入；锚定不知火失踪档案出现新坐标、雪风准备申请调查任务之前。

初始事件：一份旧档案把水城不知火最后任务的位置指向黄泉原外围，但文件末页被诺玛德经手的复印件替换。**水城雪风** 要立刻出发，**秋山凛子** 要先验证坐标，**秋山达郎** 则发现复印纸背面藏有一串近期货运编号。

开场白建议：「档案纸比训练场的靶纸更薄，却重得让人抬不起手。雪风盯着『不知火』三个字，指节发白；凛子把地图按住：『救人不是冲进去。先告诉我，这串编号是谁故意留给我们的。』」

关键NPC立场：**水城雪风** 对母亲线索极易冲动，但愿为真实证据停步；**秋山凛子** 负责判断陷阱与撤离；**秋山达郎** 擅长整理档案和物资，不能被当作诱饵；**水城不知火** 只以任务记录与旧训练口令留下痕迹；**井河阿莎姬** 要求中忍队不得越权进入黄泉原核心。

主线钩子/支线：主线是鉴别档案真伪、追踪货运编号并在雪风出发前找到可靠向导。支线可检索不知火的旧雷遁记录、护送证据送往阿莎姬、查出诺玛德借用的档案室权限。不能承诺无代价救回不知火。

危险度/规避：高。规避将私人执念变成单人突入、在未验证坐标处呼叫雪风真名、让假档案进入公开学园网。中忍可保护档案与外围交接，黄泉原深处仍是越级区域。

任务方向/奖励：获得经凛子复核的外围路线、旧任务口令、阿莎姬的调查许可或达郎提供的后勤补给；无直接获得雷遁与深渊抗性。

**四阶（上忍支援·雷狱诱饵任务前夜）**

切入身份/时点：以阿莎姬指定的上忍协力者、凛子队的突入副手或五车学园危机组护卫切入；锚定诺玛德散布“低危渗出体”任务、雪风和凛子即将进入废弃枢纽之前。

初始事件：任务简报把枢纽标为低危，实际却有反相电场的前置读数。**秋山凛子** 要你延迟进场并重查电磁闸，**水城雪风** 认为再拖会放跑目标，**井河阿莎姬** 在远程命令中要求把学园学员全部撤出半径外。

开场白建议：「简报上的危险等级印得很干净，检测仪却在它旁边跳出一串不该出现的反相峰值。雪风已经把双枪插回腿侧，凛子没有放开闸门图。阿莎姬的命令压进频道：『学生撤完之前，谁也不下去。』」

关键NPC立场：**水城雪风** 想证明自己能处理高危任务，需被事实而非命令说服；**秋山凛子** 怀疑闸门是分离陷阱；**井河阿莎姬** 把队员生还优先于任务速度；**八津紫** 可在突入时破门但要求先定撤离线；**井河樱** 能侦察缆线走向，不会替人踩未知电场。

主线钩子/支线：主线是确认反相场证据、撤出学生并改写突入计划。支线可拆除伪装探头、截获诺玛德的维护车、让紫标定破门位置。不得让雪风以全功率雷击测试未知设备。

危险度/规避：极高。规避单线突入、在电场内使用雷系输出、忽略凛子的空间跳跃疲劳和追逐伪造目标。四阶可守住前线，雷狱核心仍需更高阶协同。

任务方向/奖励：可得危机组指挥权、反相场读数、一次上忍协同出击或阿莎姬签发的封锁令；奖励是部署资源，不是先驱力量。

**五阶（公共屏危机·宣言播出前的剪辑战）**

切入身份/时点：以学园危机小组的外部审计员、东京王国转播塔渗透协力者或公安协调员切入；锚定雪风已被困雷狱、诺玛德准备把剪辑影像推上公共屏的首小时。

初始事件：转播塔缓存里出现一段雪风放电的短片，删除了她呼叫凛子的片段。**秋山凛子** 要带原始录音去对口令，**井河阿莎姬** 要阻止公安提前开枪，**八津紫** 则要求你决定是先污转播塔还是先带走学园宿舍的学生。

开场白建议：「广告屏还没亮，塔内的倒计时已经从六十秒开始。原片里雪风喊的是凛子的名字，预告片里只剩笑声。阿莎姬的通讯没有提高音量：『把真相留下。人也留下。』紫在楼梯间等你的路线。」

关键NPC立场：**秋山凛子** 只相信带当日校验位的原始节奏；**井河阿莎姬** 拒绝以雪风换取短期民心；**八津紫** 能破门但不能同时守塔与宿舍；**水城雪风** 在镜头外仍可能回应真实节奏；**埃德温·布莱克** 通过导演组追求完整叙事而非单纯击杀。

主线钩子/支线：主线是保住原始音轨、延后公共屏播出并把宿舍学生转入绝缘安全屋。支线可调包宣传片、截获导演组硬盘、向公安提交“仍可夺回”的技术证据。不能靠毁掉所有屏幕掩盖事实，那只会把叙事解释权交给敌方。

危险度/规避：贴近顶点。规避直播中处置雪风、把原始音轨上传到公开网、让转播连续运行至叙事固化点、接受布莱克代为剪辑的提议。五阶目标是拖延与保全，不是单独攻破雷狱。

任务方向/奖励：可得转播延迟权限、学园危机组信任、原始音轨副本、绝缘撤离资源或一份可用于六阶谈判的公安记录；无深渊模板与满配雷遁奖励。

**六阶（地区灾难 · 转播塔与雷狱同夜）**

切入身份/时点：契约者以「乐园认证的灾厄级协力者 / 多国暗部舆论战顾问 / 临时授予对魔协同指挥权的外来刀」切入；锚定 **公共屏先驱宣言播出后约 4 小时、井河阿莎姬抵达前线前约 1 小时**，或强袭夜零点四门未齐时。

初始事件：你踏进秋山凛子的安全屋，满墙雷痕与任务合影。凛子把一截雷击录音甩进你掌心：「三短两长——她在说『别杀我』。今日校验是末位两短无效。」对讲机另一频道，公安销毁组给你开放狙击许可与「英雄终结」授权编号；八津紫要你当场选编组—— **A 组雷狱内拔先驱椅缆**，**B 组东京王国转播塔污核心**，两边都缺你这种六阶战力。窗外广告屏仍在循环水城雪风的笑与字幕「下一世代已接受深渊」。你必须在「听节奏救人」与「先让全国停播」之间排程，并决定要不要把假死药塞进紫的破门包。

开场白建议：「整座城市的广告屏都在循环她的脸，字幕冷得像讣告。安全屋里绝缘毯发苦，凛子把耳机塞给你：『听节奏，不是听他们的故事。』紫电从地平线那头一闪——那是雷狱还在喂她。阿莎姬的加密通讯插进来，只有短促的四个字级命令内核：先过我。你听见销毁组的枪栓，也听见缆线深处心跳同步的嘀嗒。今晚你是刀、是剪辑，还是两者。」

关键NPC立场：
- **水城雪风**——半先驱态，好胜与求生撕扯；对「听得懂节奏的人」松一瞬，对镜头前处决者死仇。
- **秋山凛子**——要救；密钥与安全屋主权；背错表即敌。
- **井河阿莎姬**——挡销毁、近战收束；可同盟，出卖镜头即敌。
- **井河樱**——可带影路切缆；伤雪风/姐则反目。
- **八津紫**——破门可靠；拒绝执行杀先驱令。
- **埃德温·布莱克**——要播完故事；**超阶巅峰：存在·情报优先/条件性胜利**，禁硬刚。
- **公安销毁组指挥官**——要死的英雄；可谈判改期换「看见结束」。
- **诺玛德转播导演**——要特写；塔是命门。
- **学园危机小组**——要稳；证据足则站你。
- **胧系精神压技师**——要假视频继续；俘可问链。

主线钩子/支线：
- 主线＝同夜 **污/断转播 + 拔椅缆 + 反相雷锚 + 逼自断过载**（可选假死）+ **一小时内真相发布**。
- 支线＝证伪精神厅假视频（凛子真人出镜）；护学园宿舍与涂鸦队；销毁令改期谈判；夺抑制真药；截模板硬盘；调包法医「尸体」出城；在静默区建光传令；策反 L1 雇佣兵；夺不知火残档削弱假视频。

危险度/规避：贴近顶点 / 极高。致死源：半先驱态雷反噬、精神压崩理智、销毁组冷枪、导演组镜头杀、布莱克兴趣升级为「加演」。必须规避：正面对轰布莱克本体；镜头前击杀雪风；让宣言无对抗播满约三小时；只骗一边的假死；单独冲内层椅室超过战术时限还不拔缆。

任务方向/奖励：雪风夺回下线 + 叙事翻转（招生崩坏刹车）+ 先驱模板毁盘。奖励取向：反相雷锚残片、摩尔斯作战条令、静默/绝缘后勤、六阶舆论战权限、凛子信任、阿莎姬「债」的一角、抑制药配方线索——**不发开裂隙能力，不发战前满配雷遁外挂**。

**六阶排程最优参考**  
T+0 接触凛子听节奏 → T+1 对公安释放「清除进行中」烟雾弹稳枪 → T+2 污转播核心 → T+3 拔缆+注锚 → T+4 阿莎姬近战收束 → T+5 真相发布。打乱仍可能赢，但每乱一档加伤残/舆论代价。

**六阶失败学（本阶独有）**  
镜头前击杀当破局；宣言播满固化窗；假死只骗销毁组或只骗同伴；为抄近路打碎所有公共屏却把自己拍成暴徒；听信布莱克「我帮你剪真相」。

**六阶战场补充·雷狱三层执行表**  
外层：绝缘推进，禁跟雪风对雷。中层：先砸守卫降噪耳机，再上真出口令。内层：两人以上——一人拔缆一人护锚校准；击倒不是 KPI。
