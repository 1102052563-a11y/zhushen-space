## 剧情

**【作品来源】**
「対魔忍深渊侵蚀」为对魔忍（Taimanin / 対魔忍）宇宙的**IF灾变分支档案**，非商业单作原标题。骨架取自 LiLiTH / BLACK LiLiTH 自 2005 年《対魔忍アサギ》起的本传、外传《混沌竞技场》、续作《阿莎姬2 淫谋的东京王国》《对魔忍紫》《阿莎姬3》及 RPGX / Action Taimanin 等公开设定：近未来日本、对魔忍血脉忍法、诺玛德企业、埃德温·布莱克、胧、井河姐妹与「内在恶鬼 / 半魔化」母题。气质：暗黑忍者·都市灾难·意志战·组织政治。**本篇副标题「阿莎姬深渊化」专线**：在「裂隙曾被压回」之后数周至数月，深渊改策略为**点名最强血脉作为行走裂隙的第一容器**——只写战斗、意志战、容器仪式、清除表决与条件性自刃，**不写色情细节**（原作改造/竞技场等成人设定仅保留「长期精神负荷与战术弱点」级叙事功能）。

**【世界定位】**
近未来日本暗部仍在运转：对魔忍猎魔，诺玛德经营黑产。本线时点锚定**井河阿莎姬仍是人类侧近战顶点、东京王国结构仍在、但她已被深渊选中为第一容器**。表面街市如常，暗部已进入「最强之剑是否还是自己人」的政治审判。契约者面对的不是砍倒一个 Boss，而是**延缓 / 逆转 / 切割容器化**的条件战：药、锚、钉、冷枪、表决与布莱克交易桌同时转动。

**【世界观 · 力量体系】**
本系列在人间—魔界既有渗透之上，叠加名为「深渊（Abyss）」的第三层侵蚀。深渊不是单纯魔界扩张，而是会改写**血脉回路、忍法轨迹与人格锚点**的高阶污染源；裂隙、刻印、融合仪式与祭品链是其展开方式。档案只写战斗、组织、仪式政治与真名人物。

世界基本规则：
- **人间侧**：近未来日本。对魔忍以血脉忍法（元素 / 异能 / 咒术）+对魔忍装+忍刀对抗魔族。组织形态依时点可含忍里、公安 Section 系、后期五车学园等。成长逻辑是血脉觉醒→忍法成型→实战淬炼→组织权限，不是打怪升级。
- **魔界侧（Dark World）**：吸血鬼、那伽、半兽人、梦魔等；诺玛德以企业壳经营改造、黑市与政治渗透。
- **深渊侧（本系列增量）**：以裂隙渗入；污染分阶段——接触→回路紊乱→人格锚松动→深渊化体→仪式核心。可被意志与特殊抑制药延缓，不可被口号清除。凡人与下忍死亡即止；高阶魔族可寄生 / 改造续命；深渊化者若完成仪式，人格可能不可逆。
- **与本传「内在恶鬼」的叠加**：对魔忍谱系中存在极端创伤 / 半魔化触发的内在恶鬼（Inner Demon）母题。本线深渊化**不是**简单套用半魔皮，而是把「内在恶鬼可被宿主部分驾驭」的缝隙当作接口——深渊诱降话术常伪装成「给你真正的力量去复仇」，实则要吞掉人格锚。阿莎姬若曾掌握部分恶鬼化控制，本线表现为**双轨争夺方向盘**：一边是她自己的恶鬼化经验，一边是深渊镜像的诱降。

战力阶梯（破坏力表现，宁低勿高）：
1. 凡人·街头炮灰≈一阶：热武器可致命；对魔 / 深渊皆是资源。
2. 下忍·低阶魔物·裂隙渗出体≈二阶：碎砖裂石，惧正规编制。
3. 中忍·成型魔物·深渊触手群≈三阶：端据点级，可取首。
4. 上忍·精英 Boss（阿莎姬 / 胧级）≈四阶：建筑群级近战与高速；一城仅数人。阿莎姬「最强对魔忍」名号落在此档顶尖。
5. 城市级·东京王国 / 学园沦陷级灾难≈五阶：治安与政治同崩；领域雏形。
6. **地区灾难·深渊核心 / 布莱克权柄 / 完全深渊化顶点≈六阶（本条目唯一覆盖）**：凡俗军队近乎无效；对话、人质、仪式节点往往比正面对轰更关键。

乐园阶位映射：街头 / 见习≈一阶；下忍·渗出体≈二阶；中忍≈三阶；上忍·阿莎姬 / 胧级≈四阶；城市灾难≈五阶；**深渊核心·布莱克·完全深渊化顶点≈六阶**。本世界覆盖一至六阶。面对顶点只能以断仪式、封裂隙、夺祭品链、公开罪证、撤离关键人物与拖满谈判窗取得条件性胜利，不可用无故失能降低其威胁。

**【地理 · 舞台】**
- **东京及关东都市表层**：电车、写字楼、便利店；夜间是对魔忍与魔族 / 深渊渗出体的猎场。临界夜被建议「别看镜子」。
- **井河之里 / 对魔忍据点**：训练场、祠堂、武器库、长老议事所；内鬼与清除令的温床。
- **井河宅地下「心核室」**：**本线核心**。原武器库改造：四面无镜、吸音砂、红光谱顶灯；墙面四条镇魂钉槽（东南西北）；外闸认编制，内闸认血脉或阿莎姬亲口点名。阿莎姬被强制静养与观察处；墙面逐渐浮现深渊纹。
- **联合作战室 / 表决厅**：讨论「预防性清除容器」的政治舞台；清除令、误伤半径附件、媒体直播权在此交锋。
- **黑庭（本线领域）**：阿莎姬失控时展开的局部半显领域，建筑像被墨浸；通讯死寂，雨可反重力，影子可实体化。半径从二十米雏形扩至半城级即临界。
- **东京王国（Tokyo Kingdom）**：人工岛犯罪都市；改造厂、黑市、情报交易所；本线常作「任务中黑庭爆发」的触发地。
- **混沌竞技场（Chaos Arena）**：地下展示与资金池；本线可作胧系残党藏匿锁刻印装置的地点，不写展演细节。
- **诺玛德设施网**：合法写字楼、生物实验室、海上转运站；伪抑制药出货与真药半页配方的来源。
- **深渊裂隙带（系列）**：旧封印阵、废弃地铁、东京湾海底隧道、井河祠堂地脉交点；边缘「静默区」——电子设备失灵、气味金属甜腥。本线裂隙多已压回，但容器体内即新的门。
- **公安封锁与狙击楼**：清除派冷枪位；常选黑庭边缘——里面他们也怕。

**【世界剧情线】**
① **前史·容器逻辑（接「裂隙开启」残局）**  
第一道稳定裂隙「湾底·黑镜」被压回后，深渊学会了人间的门牌号。进阶理论（本 IF）：裂隙是门，容器是腿。第一容器需同时满足——极强意志、极强血脉、曾接触裂隙内缘。井河阿莎姬三者全中：她以隼之术切入过黑镜十米内，恶鬼六文砍出过「静止的噪音」，又是人类侧近战顶点。埃德温·布莱克远程评估：「她若完全化，比我开十个分公司有用。」胧残党则想把容器变成报复展品，用催眠刻印「锁死容器人格」。对魔忍高层分裂为「救姐派」与「清除派」。世界规则：容器化一旦进入 V 临界（行走裂隙），东京无险可守；但过早斩杀容器也可能让体内裂隙炸成二次开口。

② **征象·最强的裂（第 0～12 日）**  
阿莎姬从湾底任务归来，恶鬼六文入鞘时发出非人嗡鸣，她当即要求检查，被部分长老以「疲劳」驳回。任务中出现：隼之术轨迹拖出黑残影；砍中目标后伤口不流血而「静音」；睡眠中说出非人音节。井河樱影遁追上，发现姐姐在数自己的心跳——间隔被外力拉长。八津紫带回战场报告：阿莎姬曾单人清场渗出体潮，却在结束后对同伴拔刀——意志压回后她要求给自己戴上镇魂铐。名场面：最强对魔忍把自己铐上，说「下次若叫不出樱的名字，杀我。」第 12 日，黑庭雏形出现在废弃立交桥下，半径约二十米，三辆巡逻车同时熄火；阿莎姬自己用镇魂钉钉进柏油路把领域收回。阶段结局：观察期开始，信任开始裂。

③ **政治·清除表决（第 7～18 日）**  
联合作战室从非正式讨论升级为表决「预防性清除容器」。清除派核心论据：容器一旦完成，东京无险可守；宁可失去最强，不可失去一城。救姐派论据：失去阿莎姬等于失去对魔忍象征与实战柱，后续招募与外勤全崩。中立派要「再观察七天」，实为等布莱克报价。公安要直播权以便事后甩锅。樱怒翻桌子；紫拒绝执行清除令；阿莎姬本人无表决权——她是议题不是议员——却要求旁听，并在旁听席上戴铐，用行动投票。布莱克派人送来「合作抑制方案」——伪药标签写着对魔忍医疗处，实为诺玛德外包；真药苦金属味，伪药偏甜，且伪药也「有效」约三小时以建立信任再反噬。第 18 日樱尝苦味识破，阿莎姬把药瓶砸在表决桌上。契约者六阶身份可获一票观察权或临时否决建议权，但否决需用战果抵押（七日内交出一次封隙 / 毁实验室级成果，否则票自动转清除）。

④ **黑庭展开·任务中的方向盘争夺**  
一次针对东京王国残存改造厂的清剿中，阿莎姬强制展开黑庭：半径街区通讯死寂，渗出体从她影子涌出又被她自己斩碎——她在与体内深渊抢方向盘。胧系锁刻印者趁乱接近，试图用催眠刻印锁核；失败则刺激深渊化进度。阶段因果：**外界每优先「利用阿莎姬战力当免费清扫机」，深渊化就快一寸**。六阶战场规则：黑庭内声音优先被吞，其次无线电，再次气味；血腥味会倒流回伤者鼻腔；影子可实体化，怕强光与镇魂钉共鸣；本体与「镜中阿莎姬」不同步——本体砍杀渗出，镜像劝降；镜像能说话，本体有时失语。领域边界呈墨渍扩散，碰到水面会沿河爬，故临界夜前须断城市喷泉与开放水库。禁止「以毒攻毒」使用深渊样本，否则领域瞬扩。

⑤ **心核仪式争夺·四件套**  
抑制真药配方在诺玛德与对魔忍研究所各半；需同时取得：樱的血脉共鸣、紫的护阵、阿莎姬自愿接受「记忆锚」（语音锚是仪式组件，不是煽情道具）、外阵镇魂钉按位。语音锚工程要求多份备份：樱声、泽木恭介遗音、紫战报、阿莎姬自己读血脉名录。深渊会优先污染「最温柔的那条」——因此不能只靠恋人记忆。布莱克提出交易：交出一半东京王国物流线，换完整配方——阿莎姬拒绝「用城市买自己」。他从不以「投降」开口，而以「优化」：把速度峰值卖给实验室、或「只要一滴心核血」；拒绝后他鼓掌离开，并在数小时后让伪药出现在友军医疗箱。条件性胜利定义之一：让他在谈判桌停留满约七分钟且不签字——七分钟够外阵钉位完成；签任何字都可能被写成「自愿容器化同意书」。六阶核心抉择：帮她抢真药硬抗，或帮清除派准备「一击断头」却可能让深渊炸开更大裂隙。

⑥ **高潮·临界夜与自刃主弦**  
第 21 日起进入临界夜倒计时：城市无声戒严——不是宵禁，是「今晚别看镜子」。黑庭覆盖向半城扩张。阿莎姬选择自刃心核纹——不是自杀，是用恶鬼六文以浅角度挑断深渊回路的主弦。主弦位于心核纹中央偏左，非解剖心脏；深则真死，浅则无效；窗口约九秒，黑庭会在主弦暴露时自动收缩保护。成功条件必须同时满足：①外阵三枚以上镇魂钉按位；②语音锚最大音量；③冷枪位被压制；④布莱克谈判桌有人占用其注意力。成功后阿莎姬会短暂休克级停跳，需樱影遁送医、紫断后；此间清除派最可能「补刀称安乐」。本篇默认**条件性成功：深渊化回退到可控慢性，阿莎姬失去部分速度峰值，但人格锚保住**。失败：阿莎姬成为行走裂隙，系列滑向东京陷落类终局（他篇）。

⑦ **残局·带病作战**  
清除派失势或转入地下（可能与胧系 / 诺玛德合流，制作「阿莎姬已堕」假录像）；樱成为「姐姐的语音锚」法定监护战友，可否决部分针对姐姐的任务令；布莱克记下「人类拒绝成为完美工具」的数据；世界多了一个「半深渊的最强对魔忍」——既是盾也是定时问题。慢性深渊化需每周抑制疗程；药由对魔忍自产，拒绝诺玛德供货；阿莎姬被限制单独出高危任务，必须双人编制。本篇终局选择「带病作战」而非「完美清除」。

⑧ **容器化阶段医学 / 战术表（六阶必读）**  
- **I 接触**：黑残影、梦呓；可出任务，需双人监护。  
- **II 回路**：忍法输出不稳，或超常或自伤；禁单独行动。  
- **III 人格松**：偶发敌我误认；需语音锚与镇魂铐。失语临界标志：她开始用第三人称说「阿莎姬」，或问「樱是谁」——出现后者立即执行语音锚 + 主弦预案，不要等表决。
- **IV 半显**：黑庭可被情绪触发；城市级事故。  
- **V 临界**：行走裂隙；本篇要阻止到达。  
治疗不是「睡一觉」，是真药 + 锚 + 自刃主弦 + 外阵钉的四件套。缺一，进度只暂停不回退。与本传改造后遗叠加时：痛觉通道可被深渊征用——她越压抑痛，黑庭越稳；她若放任痛，深渊化加速但短期输出暴涨。队友必须识别「战术压抑」与「失语临界」。

⑨ **微观可介入（六阶仍适用）**  
护录音锚设备与多份备份匣；识破伪抑制药并砸上表决桌；在黑庭内用光信号 / 约定手势引导平民（口令如「摸墙左转」）；阻止媒体直播加速恐慌共鸣；与清除派狙击手对耗；伪造「容器可控」战果抵押；截获锁刻印装置；在病床与狙击线之间站岗防「安乐补刀」；销毁会被深渊共鸣的录像。

⑩ **蝴蝶效应边界**  
可改：表决票数、真药来源与到手时机、自刃是否留永久伤、某栋狙击楼是否被拔除、布莱克是否被拖满七分钟、某份清除令是否被公开。  
不可无代价改：阿莎姬曾被选为容器、深渊点名逻辑、布莱克兴趣、她曾接触裂隙内缘。硬改「阿莎姬从未沾染」→深渊改点名樱或雪风（他篇主题改判）。硬改「一针治愈完美复原」→伪药陷阱或二次爆发。

⑪ **名场面速查**  
- 自铐：「叫不出名字就杀我。」  
- 表决厅翻桌与砸伪药瓶。  
- 黑庭里阿莎姬斩自己的影子。  
- 镜中阿莎姬劝降，本体失语。  
- 恶鬼六文挑心核主弦的一响。  
- 布莱克鼓掌离开后，友军医疗箱出现甜药。  
任意时点问四句：容器阶段到了几？真药还是伪药？录音匣备份几份？清除令是否已盖章？答得清，才能写对「现在的世界状态」。

⑫ **六阶政治力学与文件战**  
六阶不是「人多打赢」。清除令编号、可接受误伤半径、对「井河樱是否连坐」的附件——契约者常在文件桌而非刀尖上决胜。救姐派需要战果证明「容器可控」；清除派需要一次黑庭事故录像。你扔给媒体的每一秒画面都在改表决。阿莎姬的底线是「平民优先于样本 / 容器效率」；布莱克的底线是「有趣优先于道德」；清除派的底线是「城市优先于英雄」。三线不可同时满足——这是本篇的残酷物理。对阿莎姬的斩首承诺在暗部可被记录为「安乐处决授权」：事后反悔且她失控会被追责；执行过早，樱会成为终生敌人。重量在伦理与时机，不在伤害数字。

⑬ **与系列其他 IF 的边界**  
本篇终局停在「慢性可控 + 人格锚保住」。不兑现：姐妹双核锁链（见姐妹共堕）、雪风先驱直播（见雪风先驱）、三代祭品链、裂隙首次开口 72 小时（见裂隙开启）。可互文一句：深渊点名顺序可能是「最强→血亲→学园象征」，但本篇只打「最强」这一环。

**【心核室规程 · 本线独有】**
守卫分三班：樱班——语音锚与情绪安抚；紫班——对外拒入与抗狙击；轮值上忍班——失控时「延迟处决」（等语音锚最后一轮播完，不是仁慈）。契约者常编入第四班「文件与药」：识伪药、挡清除令、与布莱克连线拖时。外来六阶每次进入须阿莎姬重新点名，防夺舍冒名。心核室禁止镜面（含水面与亮屏）；任务平板须磨砂膜。

**【黑庭内存活要点 · 本线独有】**
闭无线电，用约定手势；不追影子分身；听到她用第三人称自称时提高警戒；退出跟着紫的破门路线，不跟着「温柔的恭介声」。对完全化临界体：只断主弦与外阵，不砍要害——要害死亡可能炸开新裂隙。平民疏散靠口令与摸墙，不靠手机地图。清除派冷枪爱架在领域边缘。

**【与本传因果的对齐】**
本传已定：阿莎姬曾斩胧并承担恭介之死的昂贵胜利、曾清算腐败长老、改造后遗是长期精神负荷而非废战力。本 IF 不推翻这些因果，只在裂隙后叠加容器点名。写「阿莎姬因本传改造已经废了所以必然深渊化」= OOC；正确写法是：**正因为意志与规格仍在顶点，她才被选中，也才有资格自刃主弦。**

**【容器点名：为何不是别人】**
深渊的选择算法（本 IF 可观察结论，非口号）可拆成四权：①血脉纯度与井河统领系残留权限——能打开旧封印图的手纹匹配；②曾切入裂隙内缘并存活——湾底黑镜十米内的纪录几乎无人能复刻；③意志峰值——能日夜压制本传级精神负荷者，对深渊而言是「壳够硬、值得灌」；④象征价值——最强倒下的舆论杀伤力高于十个下忍据点。水城雪风人气高但规格未到「行走裂隙」阈值；井河樱血脉近却缺内缘接触与顶点输出；胧系堕忍壳已碎，装不下完整门。故本篇必须写阿莎姬，不能换成路人容器充数。布莱克内部备忘（情报侧可截获的碎片）将其标为「Prime Vessel-01」，备注栏只有一句话：Do not break the shell before the door learns to walk.

**【伪药供应链与真药战争】**
伪药出货分三层：表层标签「对魔忍医疗处应急批号」、中层外包诊所冷链、底层诺玛德魔科。甜味剂不是口味问题，是标记——三小时后反噬时，深渊纹会沿同一代谢路径亮起，方便远程计量「壳的耐受」。真药苦金属味来自镇魂钉粉末与特定血脉血清的共熔，产量极低：一剂够压 I–II 阶段约七日，III 以上只能换「可手术窗口」而非治愈。对魔忍研究所若接受诺玛德「援助原料」，批次号会被写成共同专利——残局铁则因此是「自产线优先，拒绝供货合同」。契约者劫车时要分清：劫到甜药=证据；劫到苦药=生命线；两箱混装=经典陷阱（甜药在苦药箱底层）。

**【黑庭扩散的城市工程】**
临界夜前 48 小时，救姐派默认执行「断镜·断水·断直播」三断：商场试衣镜与玻璃幕墙贴磨砂；喷泉、开放水库、景观河道临时抽干或投镇魂砂；公共屏与无人机航拍由公安与对魔混编接管。失败案例（可写进正文）：某短视频平台把黑庭边缘当「都市传说取景」，弹幕共鸣导致领域沿信号塔多爬三条街。紫的对策是物理砸塔，不是网上删帖。阿莎姬若在黑庭内听见自己的名字被广场音响循环播放，会进入「被展览」应激——与本传竞技场记忆叠合——进度条肉眼可见上跳。故六阶任务含「沉默城市」后勤，不只是砍怪。

**【表决厅附件与连坐条款】**
清除令正文常只有「授权对容器采取最终手段」一句，危险在附件：误伤半径（是否含井河宅周边居民楼）、樱是否列入「次级风险血亲」、契约者是否被写成「授权执行人」（事后背锅位）。救姐派律师型上忍（若编制存在）会要求把「最终手段」改成「四件套失败后的最后手段」，并写明主弦自刃优先于外部斩首。中立派的「再观察七天」附件里，往往夹着布莱克报价的匿名 PDF。公开附件比暗杀更有效——六阶可以是泄密战。

**【主弦手术级细节（供 AI 写高潮）】**
阿莎姬自己报角度：「浅于复仇，深于犹豫。」意思是：刀意不能带斩胧时的杀意（会过深），也不能带引退时的留恋（会飘浅）。语音锚在九秒窗内的播放顺序建议：先名录（血脉自我识别）→再樱的短句本名呼唤→最后恭介遗音只作背景层而非主轨。外阵钉位完成信号是三声铃（静默区替代无线电）；第四声铃=冷枪已压。布莱克侧拖时话术：问契约者「你更怕她死，还是怕她不再是她」——答错会被记入下一份优化方案。成功后的第一句若仍是第三人称，说明主弦未断干净，需二次浅挑（风险倍增，本篇默认一次成功）。

**【残局编制与「可托付」评价】**
人格锚保住后，阿莎姬对契约者的评价只有三档可写进任务结算：①「可托付」——允许进入心核室内闸、可共同出双人高危；②「可用」——外勤协力但不给承诺权；③「观察」——等同清除派候选人。获得①的条件通常是：拒签布莱克、识破至少一批伪药、挡过一次补刀或冷枪、且未对樱撒谎关于斩首授权的去向。慢性疗程日她会检查你是否仍带着那份未执行的承诺书——撕毁或上交樱保管均可，藏着当把柄则评价降档。

**【东京王国改造厂触发战（完整一幕）】**
清剿名义是端掉残存改造流水线，实则诺玛德故意留「半关的厂」诱阿莎姬出静养。厂区三层：地面合法仓储、夹层刻印车间、地下渗出培养槽。阿莎姬下到夹层时黑庭首次非自主扩张，影子从培养槽爬出成形。樱本应按预案切断厂区总电，却发现电闸被焊死——紫破门改走冷却管爆破。契约者若在场，可选：①护樱拆焊；②在地面伪造「阿莎姬已失控」的假直播喂给清除派（险着，可换取对方提前暴露狙击位）；③冲谈判全息拖布莱克。本幕结束标志不是厂毁，而是阿莎姬是否还能报出「樱」的读音——报得出=III 未破；报不出=立即撤离进心核室预案。

**【与「内在恶鬼」对话的禁区】**
深渊镜像会借用恶鬼化话术：「融合吧，你早就不是纯粹人类。」正确应对不是否认血脉，而是重申锚点：「我是井河阿莎姬，目标是活着砍断你。」任何「那就让恶鬼来做主」的让步都会被记为 IV→V 跳级许可。系列互文里部分时间线存在吸血鬼化 / 恶鬼化变体；本篇默认阿莎姬拒绝永久恶鬼接管，自刃是对「被接管」的物理否决，不是向恶鬼投降。

对阿莎姬：「我拒签。真药路线。斩首权我拿着但不先动。」——可加分。  
对樱：「备份匣在我这第二份，苦的我尝过。」——降敌意。  
对紫：「狙击楼东侧红外，我去拔，你守内闸。」——专业同盟。  
对清除派：「附件误伤半径含居民楼，我拒当执行人。」——文件战。  
对布莱克：「优化免谈。七分钟聊天气也行。」——拖时正确姿势。  
对药剂师：「箱底那排甜的，当庭拆。」——识破。

**六阶胜利定义（本世界）**  
人格锚保住 + 黑庭未永久半城化 + 关键人物（阿莎姬 / 樱 / 紫）存活 + 你未成为书面刽子手或布莱克合同乙方。击杀榜与「完全净化」均非胜利条件。残局允许慢性深渊化与速度峰值损失——这是代价不是失败。
