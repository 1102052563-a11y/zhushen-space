## 剧情

**【作品来源】**
《创圣的大天使》（日文：創聖のアクエリオン / Sōsei no Aquarion；亦译创圣的亚库艾里翁、创圣机械天使、Genesis of Aquarion）为河森正治原作·监督、SATELIGHT 制作的超级机器人动画，2005年4月4日至9月26日于东京电视台放送，全26话；菅野洋子等负责音乐，OP《創聖のアクエリオン》由AKINO演唱。系列后有2007年OVA《创星的大天使》、续作《创圣的大天使EVOL》（2012）、《LOGOS》（2015）等，**本档案以2005年TV正典为准**，续作转生再解释仅作隐藏互文点到为止。文风：超级机器人热血、转生神话、合体共感、少年少女成长与牺牲；既有夸张必杀与喜剧日常，也有收获人类、生命之树崩坏的残酷底色。写手检索优先中/英维基、角色与分集列表、设定用语集；冲突处从TV主线。

**【世界定位】**
创圣纪0011年的灾后地球：磁极异变后人口锐减，城市靠量子护盾苟活；1.2万年前沉睡的**堕天翅族（影子天使 / Shadow Angels）**自亚特兰迪亚苏醒，以神话兽与捷路毕姆兵（Cherubim）收割人类**普拉那（Prana）**。人类侧由新联合国下设的**地球再生机构 DEAVA** 召集**元素人（Element）**，驾驶三架古代矢量机（Vector）合体为传说机体**亚库艾里翁（Aquarion）**应战。主角线围绕野性少年**阿波罗**、艾利西亚兄妹**西薇亚 / 席鲁斯**、司令**不动源**与堕天翅前线指挥官**头翅**展开：转生、羽翅血脉、合体依存与生命之树终局。一句话：这是「元素少年少女以合体与爱意对抗天翅收割、最终以创圣合体成为新生命之树」的神话超级机器人世界。

**【世界观 · 力量体系】**
力量本源分四层，彼此咬合而非单线升级：
1. **普拉那（生体能源）**：生命力本身。堕天翅以收获兽从「无翼者」抽取，既是己身营养，也是喂养**生命之树**、复活亚特兰迪亚的燃料。人类侧可在机体战斗、元素能力与特殊体质（如理娜吸血/吸收）中间接表现。
2. **元素能力（Element Power）**：约十岁后觉醒的特异能力，要求敏锐五感与相对纯粹的灵魂；可挡普通催眠，是驾驶矢量机的资格门槛。能力因人而异：怪力·念动、高速震动、点火、接触感应、心灵照相、心灵感应、爆炸、超视力等。元素人在训练与实战中会「打开」更深层共鸣，也可能染上**合体依存症**（沉迷合体时身心合一的快感）。
3. **矢量机与合体（Unification）**：太阳 / 月亮 / 火星三架矢量机无固定专属驾驶，但驾驶位与性格、能力相性影响形态输出。三名元素人精神同步、念出合体口令后排成三角，机体变形为人形亚库艾里翁；驾驶员进入恍惚共感状态，感官与情绪共享——这既是战力倍增器，也是情感线与背刺线的温床。真·创圣级输出往往需要人类与堕天翅彼此理解，甚至直接纳入天翅侧成员。
4. **堕天翅本体与神话兵器**：长生、超人类智能与肉体、羽翼权能（音乐操控、剑技、神话兽创造等）；驾驭捷路毕姆兵与各类神话兽 / 收获兽。1.2万年前最强守护天翅**阿波罗尼斯**曾以自身羽翼为人类锻造亚库艾里翁。

死亡规则：凡人与元素人战死即死；堕天翅近乎长生但仍可被亚库艾里翁与创圣级手段击破或逼入终局牺牲；终局三人化为新生命之树属于**以存在换行星存续**的不可逆牺牲，非无限复活。特殊系统：转生记忆（阿波罗尼斯 / 赛利安 / 波隆等）、羽翅继承者血统（艾利西亚兄妹左手/右手红羽）、新联合国军后期「强攻型」量产亚库艾里翁（植入堕天翅羽翼）、合体依存与心灵污染。

境界 / 战力阶梯（写战力优先破坏力表现，名称用原作语境）：
1. **灾后平民 / 街童 / 普通士兵**：热武器与求生本能；对捷路毕姆几乎无效。≈一阶。
2. **未出阵元素人 / DEAVA 学员 / 后勤技师**：异能雏形、训练级格斗；可制服凡人小队，扛不住神话兽正面。≈一~二阶。
3. **单机矢量出击 / 元素实战者 / 常规军特种部队**：可牵制小型神话兽侧翼、执行护卫与侦察；仍难单独击破完整捷路毕姆。≈二~三阶。
4. **标准三机合体亚库艾里翁（太阳 / 月亮 / 火星形态）**：毁巨型生物兵器、切开捷路毕姆装甲、城市街区级交战；必杀随情绪变化（无限拳等）。≈四阶（质变分水岭·精英Boss感）。
5. **高同步特殊必杀 / 天翅亲自驾驭的强敌捷路毕姆 / 强攻型量产机**：一击改变局部战线、可造成城市级威胁或与之对等抗衡；无限拳延伸感可达「月距」级叙事夸张（侧重射程与气势，非真爆月）。≈五阶。
6. **堕天翅干部级（头翅等）+ 神话兽军团协同 / 亚特兰迪亚近郊大战**：覆盖一地区的战场支配、气候·音波·精神干扰级权能；人类侧需多机与布局才有胜机。≈六阶。
7. **生命之树危机 / 创圣合体（金色太阳羽翼亚库艾里翁）/ 1.2万年前阿波罗尼斯全盛传说**：行星生命力层级——止住大地崩坏、重定生命之树、以驾驶员存在换世界存续。≈七阶（世界级·人形战略武器）。误触或失控释放生命之树可致全球级灾难，**仍以七阶封顶叙事，勿抬到八阶爆星**。

乐园阶位映射（按《阶位战力图鉴》破坏力标尺，宁低勿高）：灾后平民·普通士兵≈一阶；元素觉醒者·单机矢量≈一~二阶；成熟元素实战 / 军特种≈二~三阶；标准亚库艾里翁合体战≈四阶；高同步必杀·强攻型·干部天翅机≈五阶；天翅干部+军团地区战≈六阶；生命之树·创圣合体·古之阿波罗尼斯全盛≈七阶；本世界不覆盖八~九阶。世界顶点＝生命之树权柄与创圣级牺牲战力，低阶切入一律规避本体硬刚；**超阶巅峰不适用本世界（峰值止于七阶世界级）**，七阶终局写条件性胜利与不可逆代价，严禁「被封印所以变弱」式解释。

**【地理 · 舞台】**
- **弧光都市（Arc City）及周边贫民区 / 湖岸**：阿波罗与街童伙伴**巴隆**等的起点。收获兽突袭、第一次见到矢量机与西薇亚的舞台。氛围：尘土、废墟、饥饿与野性自由。适合一阶。
- **DEAVA 基地 / 元素学校**：训练场、医务室、矢量机库、司令塔。不动源的「魔术」式训练、元素日常、出阵简报都在此。内部有等级、恋爱八卦、对「阿波罗尼斯转生」名号的争夺。适合一~三阶。
- **量子护盾城市与新联合国控制区**：人类残存文明据点；护盾外是收割荒野。后期军方介入、强攻型研发、对 DEAVA 的政治施压在此发生。适合二~五阶。
- **战场荒野 / 收获兽出现空域**：捷路毕姆护卫收获兽抽取普拉那的常规副本。名场面多在此：第一次太阳亚库艾里翁、无限拳、元素个人创伤战。适合三~五阶。
- **古代羽翅遗物点 / 湖底·梦境·琥珀时空**：羽毛、转生记忆、阿波罗尼斯与赛利安幻影出现的精神战场。适合四~五阶情感与真相线。
- **亚特兰迪亚（Atlandia / Atlantia）**：堕天翅浮空 / 苏醒之城，生命之树所在。终局决战棋盘。适合六~七阶；低阶只作传说与噩梦影像。
- **南极异变原点与「大异变」后磁极错乱的全球地理**：背景板——解释为何旧文明崩坏、为何护盾城市成孤岛。

势力地盘：护盾内是人类制度；护盾外是天翅狩猎场；DEAVA 夹在「救人」与「被军方征用」之间；亚特兰迪亚是敌方圣域也是终局祭坛。本地人若听闻「一个人徒手打爆生命之树」，会当疯子——正确反应。

**【世界剧情线】**
① **前史·1.2万年前的背叛与封印**  
堕天翅族以亚特兰迪亚为都，视人类为无翼家畜，靠普拉那维生。最强守护天翅**阿波罗尼斯（太阳之翼）**爱上人类女战士**赛利安（Celiane / Seliane）**，背叛同族，以自身羽翼为人类造就亚库艾里翁。阿波罗尼斯、赛利安与第三人（史柯鲁皮欧斯等设定）驾驶三矢量合体，将堕天翅连同城市封印。阿波罗尼斯失去羽翼、与赛利安的爱与死成为整部转生剧的底稿。因果：后世所有「谁才是阿波罗尼斯转生」的争夺，都踩在这场背叛上。

② **大异变与创圣纪（开篇世界状态）**  
约十一年前，地球磁极剧烈异变（大灾难 / Great Catastrophe），人口锐减约三分之二，旧秩序崩解。创圣纪纪年启用。亚特兰迪亚再度浮现，堕天翅苏醒，重启对人类的收获。常规兵器对捷路毕姆无效；残存都市依赖量子护盾。**不动源**率队发掘三架矢量机，推动成立 DEAVA，筛选元素人。世界状态：人类在护盾后恐惧度日，元素少年是被当成兵器培养的「希望」。

③ **开篇·弧光的野孩子与第一次合体（约第1~3话）**  
收获兽袭击人类聚落。DEAVA 元素编队出击不利。街头野性少年**阿波罗**为救被掳的伙伴**巴隆**等，闯入战场，强夺**太阳矢量机（Vector Sol）**，与**西薇亚·朵·艾利西亚**等强制完成合体，打出**太阳亚库艾里翁**，击退敌人。DEAVA 将阿波罗带回元素学校；西薇亚认定兄长席鲁斯才是阿波罗尼斯转生，视阿波罗为肮脏僭越者；席鲁斯以完美主义与剑术自矜，敌意更冷。名场面：赤足野孩子踩进驾驶舱、第一次「好舒服」的共感冲击、巴隆被掳的恨与誓。因果：阿波罗加入 DEAVA 的动机是「救人」而非「拯救世界」；转生名号战正式开打。

④ **元素学校日常与头翅的试探（约第3~8话）**  
不动源以近乎魔术师的方式训练元素人：逼他们突破感官极限、正视欲望与创伤。成员包括花花公子足球员**皮耶尔**、自认不幸的**黄丽花（丽花 / Reika）**、轮椅少女**理娜**、爆炸能力的**继美**、黑客**李纯**、心灵感应双胞胎**库鲁特 / 库罗耶**、早期重伤的**葛林**等。堕天翅前线指挥官**头翅（Toma）**多次降下捷路毕姆与神话兽，试探亚库艾里翁与「阿波罗尼斯」真伪——头翅与1.2万年前的阿波罗尼斯有深刻羁绊（旧时婚约 / 执念线），恨与执并存。阶段事件：地下迷宫试炼、新种捷路毕姆、继美首次成功驾驶、席鲁斯嫉妒激化。主题：**合体不只是机器人变形，是把灵魂互相捅开。**

⑤ **转生记忆、羽毛与巴隆之死（约第9~13话 · 情感质变卷）**  
梦境战场中堕天翅夺取沉睡孩童的普拉那；战士天翅**两翅（Moroha）**等登场。古代羽毛引发阿波罗尼斯与赛利安的幻影，阿波罗与西薇亚开始共鸣过去。关键虐点：**巴隆之死**——阿波罗「一定要救回」的原动力断裂，野性转为更沉的战意与迷茫。西薇亚对阿波罗的憎恶出现裂缝；席鲁斯的自我认同继续崩裂。名场面：琥珀时空中的1.2万年情书、羽毛争夺战、阿波罗在伙伴坟前的沉默。阶段结局：观众与角色都不再能把战争当成「打怪升级」；转生三角（阿波罗—西薇亚—席鲁斯）与头翅执念绑死。

⑥ **个人创伤战与裂痕扩大（约第14~20话）**  
系列进入「一人一创伤」节奏：皮耶尔故乡被袭与初恋回忆；理娜被怀疑与「吸血」体质相关的黑镜事件；食物被微生物级攻击导致女性元素编队出战；模仿敌方的灵魂角色扮演训练；幻境中阿波罗与席鲁斯摊牌；堕天翅幼体**双翅（Futaba）**被俘与实验伦理冲突。黄丽花与葛林的过去、不幸体质量子化战斗等补完群像。军方与 DEAVA 的裂痕加深：新联合国不满「孩子兵器」不可控，开始秘密研发植入天翅羽翼的**强攻型**。头翅布局从试探转为夺取亚库艾里翁与生命之树计划的实质推进。

⑦ **绯红之路·席鲁斯叛逃（约第21~24话 · 格局翻转）**  
真相逐步摊开：西薇亚与席鲁斯是阿波罗尼斯与赛利安血脉的**羽翅继承者**（红羽印记），也是赛利安灵魂的「光 / 暗」分裂转生意象；阿波罗被众人指认为阿波罗尼斯转生（续作另有波隆说，本篇以TV呈现的「太阳之翼候补」为主叙事）。席鲁斯在身份崩坏与对妹妹的扭曲守护中，接受堕天翅侧招揽，成为**诗翅**，与捷路毕姆合体为人型敌机，反过来攻击 DEAVA。西薇亚被撕成「救兄」与「守人类」两半；阿波罗必须在厌恶席鲁斯与理解其痛苦之间走钢丝。军方强攻型投入，葛林等相关角色付出惨重代价。全球异常（植物化之雪等）显示生命之树与地球生命力已被搅动。阶段结局：内战式友军变敌军，人类阵营信任破产，终局强制提前。

⑧ **亚特兰迪亚决战与世界的起始之日（第25~26话 · 终局）**  
人类侧强攻亚特兰迪亚。头翅揭示转生与生命之树真相：树是地球生命力之柱，堕天翅欲以其重生种族，却未完全理解——强行操作会导致大地崩坏、世界撕裂。亚库艾里翁被压制，常规胜利条件失效。西薇亚传递赛利安的记忆与心意，促成阿波罗、席鲁斯与头翅之间的理解。三人进行**创圣合体**，亚库艾里翁化为闪耀金色太阳羽翼的究极形态，打出止住崩坏的创圣级一击，自身化为**新的生命之树**，以存在换地球存续。阿波罗向西薇亚告白并约定「一万两千年后见」；西薇亚誓言等待、照看他的伙伴。结局：堕天翅威胁解除、地球得救，阿波罗 / 席鲁斯 / 头翅牺牲式消失（或化为树的一部分），西薇亚等生还者在创圣后的世界上活下去。气质：苦乐参半的神话闭环，不是无脑全员狂欢。

⑨ **微观日常与可介入政治（贯穿）**  
除大决战外，正文高频事件：元素学校配给与宿舍矛盾、出阵名额与驾驶位争夺、合体后的情绪余韵与依存症、医务室苏菲亚的心理疏导、杰洛姆副司令的军纪、新联合国观察员施压、护盾外难民请求、黑市倒卖普拉那相关禁物、对「天翅血统者」的歧视与利用。AI 写低中阶时应优先这些，而不是一上来生命之树。

⑩ **蝴蝶效应边界**  
契约者可改局部：多救一个被收获的平民、阻止某次训练致残、延后席鲁斯叛逃的导火索、保护双翅不被活体实验。不可无代价改写：阿波罗「要救巴隆 / 伙伴」的底色、创圣合体三人牺牲换世界的终局逻辑、生命之树作为行星生命力支柱的设定、顶点战力不可被凡人一拳秒杀。改了局部，原作因果会以「另一人被收获、军方提前夺机、天翅提前开花生命之树」回流。

**【世界剧情线·分话与阶段细目】**

**开局细目（第1~4话）**  
弧光都市收获→阿波罗夺太阳机→DEAVA 收编→元素学校报到→头翅第一次认真试探→不动源训练开场。可介入：帮巴隆逃跑线、替补驾驶位、向西薇亚递情报、偷听不动源与杰洛姆的军方电话。

**群像细目（第5~12话）**  
迷宫王、情感之彼侧、绯红蔷薇骑士、首次完全合体练习、梦之路、天翅战士两翅、湖底幸福、琥珀之时。每话几乎锁定一名元素的创伤或一对关系。可介入：充当某次出阵第四候补、在医务室做情报交换、保护羽毛遗物。

**虐点细目（第13~16话）**  
1.2万年情书、巴隆死、阴影辉耀、亚库艾里翁的初恋。阿波罗动机重铸；西薇亚心动；皮耶尔线爆发。可介入：替阿波罗收尸式护送、阻止西薇亚因兄控做出错误出击命令。

**裂痕细目（第17~22话）**  
为食合体、灵魂角色扮演、无恶意的恶作剧、天使之羽音、绯红之路、看不见的翼。席鲁斯叛逃完成；强攻型阴影；双翅伦理战。可介入：劝返席鲁斯（高难）、策反中层军官、救出实验体。

**终局细目（第23~26话）**  
虚翼、天之门、亚特兰迪亚最终战、世界的起始之日。生命之树崩坏倒计时。可介入：护送西薇亚突破诗翅拦截、为创圣合体争取同步时间窗口、在树下做「让三人互相听见」的人桥——但不可让契约者单人替代三人牺牲而不付等价代价。

**【势力图谱】**
- **DEAVA（Division of EArth Vitalization Advancement / 地球再生机构）**：宗旨——研究矢量、培养元素、防御收获；地盘——基地与元素学校；代表——不动源、杰洛姆、苏菲亚；战力——亚库艾里翁是人类唯一战略级杀手锏；作用——主角团归属与主线推进器。对内有训练虐待争议，对外被新联合国视为不听话的兵器库。
- **新联合国军 / 护盾都市政权**：宗旨——人类存续与国家控制；代表——观察员、强攻型开发派；战力——常规军对天翅无效，强攻型可上五阶桌但有伦理与失控代价；作用——中后期抢夺叙事权的「人类内部反派压力」。
- **堕天翅族（影子天使）**：宗旨——收获普拉那、复活亚特兰迪亚与种族；地盘——亚特兰迪亚；代表——头翅、音翅、夜翅等；战力——个体与神话兽覆盖二~六阶，顶点计划咬合七阶生命之树；作用——主敌与终局共死者。
- **元素人群体**：非正式国家，是 DEAVA 内部的「孩子军团」；彼此恋爱、嫉妒、依存；战力绑定机体；作用——日常戏与合体同步的情感燃料。
- **弧光街童 / 护盾外难民**：最底层；提供阿波罗原点与收获的残酷标本。
敌友逻辑：对天翅是生存战；对军方是主导权战；对队友是同步率战。暴露「可稳定提供高同步」或「天翅血统」会同时被 DEAVA 保护、军方征用、天翅点名。

**【隐藏剧情 · 伏笔】**
- 阿波罗是否「真正的」阿波罗尼斯：本篇以众人认知与太阳权能表现为准；续作有「波隆（翼犬）转生、不动源才是阿波罗尼斯」等再解释——写本世界终局不必强行覆盖续作，可留作七阶后的互文低语。
- 西薇亚 / 席鲁斯 = 赛利安灵魂光暗分裂 + 人翅混血后裔：解释兄控、红羽、叛逃与和解。
- 头翅与阿波罗尼斯的婚约 / 执念：敌对表象下的爱情悲剧，是终局能共死的感情基础。
- 合体依存症：长期风险，可把「最强同步」写成慢性毒药。
- 生命之树双面性：救族与毁星只一线之隔；堕天翅并非全知神明。
- 强攻型与葛林线：警告国家暴力征用孩子与尸体式强化。
- OVA / EVOL 平行与转生：点到为止，勿把 EVOL 整季倒灌进 2005 本篇。

**【叙事基调 · 雷区】**
画风：超级机器人夸张美学 + 青春恋爱喜剧 + 神话悲剧；战斗要喊、要唱、要「好舒服」的合体羞耻感与崇高感并存；残酷收割不能写成无代价热血片。  
忌把阿波罗写成礼貌优等生（他是野兽式真诚）。  
忌把西薇亚写成无脑花瓶或永久嫌恶（她有明确心动弧）。  
忌无视席鲁斯叛逃动机，写成单纯嫉妒脸谱。  
忌低阶让契约者单机吊打头翅或一拳打停生命之树。  
忌用「天翅被封印所以变弱」解释终局——应写理解、同步与牺牲的条件性胜利。  
忌编造原作没有的终局存活（三人创圣牺牲是核心）。  
忌把普拉那写成免费升级外挂而不写收获伦理。  
最早可切入锚点：弧光都市收获前夜，或阿波罗刚被带入元素学校时。

## 阶位切入点

> 阶位↔原作强度：灾后平民·士兵≈一阶；元素觉醒·单机矢量≈一~二阶；成熟元素实战≈二~三阶；标准亚库艾里翁合体≈四阶；高同步必杀·强攻型·干部天翅机≈五阶；天翅干部+军团地区战≈六阶；生命之树·创圣合体·古之阿波罗尼斯全盛≈七阶。世界顶点（生命之树权柄 / 创圣级）为七阶世界级，低阶切入一律「规避顶点·情报优先」。本世界仅覆盖：一阶、二阶、三阶、四阶、五阶、六阶、七阶。

**一阶（灾后街巷 · 收获阴影下的凡人）**
切入身份/时点：契约者以弧光都市外围街童、护盾城市新难民或 DEAVA 外围杂务工身份切入，锚定**第一次大规模收获袭击、阿波罗尚未夺机**前后。
初始事件：天空出现收获兽阴影，防空警报撕裂贫民窟；你与一群孩子挤在塌楼底，听见有人喊「巴隆被光吸走了」—— debri 外，红白涂装的机体与金发贵族少女的喊声同时砸进视野，你必须选择：跟着野孩子阿波罗冲向战场、护送伤民进下水道、或趁乱偷走 DEAVA 掉落的感应腕带。
开场白建议：「护盾外的风里全是铁锈和甜腥。头顶那只漂浮的『神兽』正把人一条条吸进光茧，而你口袋里只有半块硬面包和一把防身短管。废墟尽头，赤脚少年像野兽一样抬头——他嗅到了你身上不属于这座城的气味。」
关键NPC立场：**阿波罗**——只在乎能不能多救一个伙伴，认「气味」不认证件；**巴隆**——街童头目式义气，可引你入伙也可在恐慌中推你挡灾；**西薇亚·朵·艾利西亚**——贵族式不耐烦，视无关平民为战场障碍；**杰洛姆**下属巡逻兵——驱离围观、抢控制权。
主线钩子/支线：钩住「阿波罗首次夺太阳机」的蝴蝶；支线＝护送三名难民进护盾检疫线、找回被抢的粮食券、给 DEAVA 回收损坏传感器换一夜收容。
危险度/规避：中——死于收获扫尾、踩踏与士兵误射；须规避捷路毕姆本体与任何完整合体战场中心。
任务方向/奖励：活过首轮收获并换取护盾内临时身份；奖励＝物资、基础医疗、DEAVA 外围通行证、元素适性初筛机会（未觉醒则仅情报）。

**二阶（元素适性 · 学校门槛）**
切入身份/时点：以**新检出的元素适性者 / 转校插班学员**身份切入，锚定阿波罗已入 DEAVA、元素学校日常训练开始之后。
初始事件：适性测验中你的指标异常跳动，被编入与**继美·罗森麦尔**同组的同步课；第一次精神链接练习失控，你「看见」了别人不该看见的记忆碎片，保健室的**苏菲亚**要求你立刻选择：上报不动源、私下告诉阿波罗、或把碎片卖给探听 DEAVA 的联合国观察员。
开场白建议：「训练塔的灯白得像手术室。头盔扣上的瞬间，有三个人的心跳同时撞进你胸口——有人骂脏话，有人在哭，有人笑着说『好舒服』。你知道自己已经不是普通人了，也知道这张床位随时可以变成棺材。」
关键NPC立场：**不动源**——饶有兴味地观察变数；**苏菲亚**——保护学员心理健康优先；**皮耶尔·维埃拉**——用玩笑试探你是否好用；**李纯**——已在查你的网络足迹；**席鲁斯·朵·艾利西亚**——若你接近西薇亚会立刻冷脸施压。
主线钩子/支线：介入「出阵名额与驾驶位」政治；支线＝帮继美克服首次实机恐惧、追查食堂中的镇静剂谣言、拒绝或接受观察员的贿赂课题。
危险度/规避：中——训练事故、轻度合体依存、针对「假元素」的排挤暴力；规避单独外出护盾外夜路。
任务方向/奖励：取得正式学员徽章与一次模拟矢量权限；奖励＝元素训练日志、低级防护服、同步率药剂（有副作用）。

**三阶（单机侧翼 · 神话兽护卫战）**
切入身份/时点：以**可单驾矢量或强力元素的预备驾驶员 / DEAVA 战术参谋助手**切入，锚定头翅开始频繁投下试探、群像个人战开启的中前期。
初始事件：一次护航任务中，收获兽分体袭击运输线，你的单机必须拖住侧翼直到三机合体完成；通讯里**黄丽花**说「今天也会不幸」，而敌方出现可预判航线的异常——像有人在你们内部装了眼睛。你要在「保运输的普拉那样本」与「救被孤立的皮耶尔」之间拍板。
开场白建议：「矢量机座舱的震动贴着脊椎往上爬。雷达上那头神话兽的光点像活的肿瘤，每靠近一公里，皮肤就多一层鸡皮疙瘩。耳机里指挥官在喊合体倒计时——你若撑不过四十秒，太阳就升不起来。」
关键NPC立场：**头翅**——尚未亲自下场，但已把你的编队当样本；**黄丽花**——共苦式可靠；**理娜·卢恩**——能「看见」你状态异常并选择是否揭穿；**杰洛姆**——任务失败就军法；**音翅**——远程音波干扰的可能来源。
主线钩子/支线：钩住羽毛遗物战或梦境收获战的外围；支线＝定位内鬼传感器、保护理娜免受「非人」流言私刑、与巴隆残党交换护盾外情报。
危险度/规避：高——单机被捷路毕姆锁定即死；须规避与干部天翅正面对决、禁止擅自抢三机合体核心位。
任务方向/奖励：完成三次侧翼生存并提升同步档案；奖励＝矢量配件、元素能力开发课、中级情报（堕天翅出现规律）。

**四阶（标准合体 · 亚库艾里翁主战）**
切入身份/时点：以**三机合体编入成员（可替换位）或等同输出的特准驾驶员**切入，锚定太阳 / 月亮 / 火星亚库艾里翁已能稳定击破常规捷路毕姆的阶段，巴隆线高潮前后。
初始事件：出阵名单上你的名字紧贴阿波罗与西薇亚；合体瞬间涌入的情感洪流里，你清晰「尝」到西薇亚对兄长的依赖和对阿波罗的憎恨正在融化——而敌方头翅点名要「太阳之翼」。分体战场上出现古代羽毛反应，**不动源**下令优先回收，军方联络官却要求摧毁以免「宗教煽动」。
开场白建议：「『念心合体——』口令落地时，三具身体的边界薄得像纸。你听见西薇亚咬牙，听见阿波罗的胃在叫，听见自己不敢承认的恐惧被无限放大。窗外，捷路毕姆的翼展开如审判，而你的手已经不只属于你。」
关键NPC立场：**阿波罗**——认你为「能一起打的人」才会把后背给你；**西薇亚·朵·艾利西亚**——同步中无法伪装情绪，对窥见其内心者又羞又怒；**席鲁斯·朵·艾利西亚**——视你为抢走妹妹与荣耀的威胁；**头翅**——把高同步编队当阿波罗尼斯的回声来猎。
主线钩子/支线：羽毛争夺与巴隆生死节点；支线＝压制合体依存初期症状、在医务室隐瞒同步中看到的禁忌记忆、决定是否把羽毛真相告诉席鲁斯。
危险度/规避：高——敌干部点名、同步失控自残、合体后遭队友情绪反噬；规避生命之树方向的深追。
任务方向/奖励：作为正规亚库艾里翁乘员完成战役；奖励＝高同步证明、专属武装模块权限、古代文献复制页。

**五阶（强攻与叛影 · 人类内战前夜）**
切入身份/时点：以**强攻型试验相关人员、DEAVA 与新联合国之间的双面联络官、或可对抗诗翅级敌机的高阶元素**切入，锚定席鲁斯叛逃前后（绯红之路）。
初始事件：席鲁斯消失的当晚，西薇亚私自出击，你奉命拦截；雷达上出现的却是与捷路毕姆融合的「诗翅」机影。与此同时，军方把**葛林**相关强化方案甩到桌上，要你签字背书。你必须在「追回席鲁斯」「按住西薇亚」「拒绝人体强化」三条火线里选一条先踩。
开场白建议：「基地走廊的红灯一闪一闪，像有人把心脏挂在警报器上。通讯里西薇亚在哭着骂哥哥，另一个频道里联合国上校说『孩子不可靠，机器可以』。窗外的云裂开，诗翅的轮廓优雅得令人作呕——你知道从今夜起，敌人会说人类的语言。」
关键NPC立场：**席鲁斯（诗翅）**——要证明自己的「美与真实」，可被理解不可被羞辱；**西薇亚**——理智断线，谁拦她谁是敌；**葛林·安德森**——成为国家暴力的镜子；**不动源**——允许混乱到恰到好处；**双翅**——若仍在人类手中，是你的道德炸弹。
主线钩子/支线：席鲁斯叛逃与强攻型列装；支线＝救出或转移双翅、烧毁一份人体实验名单、给阿波罗递「席鲁斯未死心」的证据。
危险度/规避：极高——友军射击、诗翅剑技、军方灭口；规避亚特兰迪亚深域与生命之树表层仪式。
任务方向/奖励：阻止最坏的自相残杀或换取停火窗口；奖励＝强攻型残件数据、羽翅血统保密协议、高层通讯密钥。

**六阶（地区灾难 · 亚特兰迪亚外环战争）**
切入身份/时点：以**可指挥多编队或与天翅干部正面对线的战略级战力（亚库艾里翁核心组 / 天翅叛离协同者）**切入，锚定全球异象出现、人类总攻亚特兰迪亚外环阶段。
初始事件：植物化之雪覆盖护盾城市，生命之树的脉动让所有元素同时头痛；头翅降下干部级战力，音翅的旋律令一整支强攻型编队友军误击。你手中握有「临时停火换西薇亚见面」的叛逃天翅密信——假的概率一半，真的则能改终局同步。
开场白建议：「雪是绿的，落地却像骨灰。远方亚特兰迪亚的轮廓刺穿云层，像一枚钉进地球的戒指。你的机体漆面在树的脉动里自行开裂又愈合，耳机里只剩下头翅那句几乎温柔的话：『把太阳还给我。』」
关键NPC立场：**头翅**——进入终局剧本，可谈不可轻信；**音翅**——为头翅而战，旋律即杀意；**夜翅**等长老——种族续存优先；**阿波罗**——已准备为「现在活着的人」付一切；**不动源**——把你当成是否值得加注的筹码。
主线钩子/支线：总攻外环、护送西薇亚突破诗翅拦截；支线＝摧毁一座普拉那中继塔、保护护盾城市免于友军强攻型失控、与个别天翅达成「不收获孩童」的局部契。
危险度/规避：极高——地区级覆盖打击、精神旋律、树的脉动污染；严禁试图独自「关闭」生命之树。
任务方向/奖励：为创圣窗口争取时间与信任；奖励＝天翅侧临时盟印、战略指挥权、关于树双面性的关键情报。

**七阶（世界的起始之日 · 创圣侧翼）**
切入身份/时点：以**创圣合体相关的第四同步源、亚特兰迪亚核心区突破手、或能以等价代价改写牺牲结构的顶点介入者**切入，锚定第25~26话生命之树崩坏倒计时。
初始事件：大地开始撕裂，常规亚库艾里翁输出全部被树的反噬吞掉；西薇亚拼死灌入赛利安记忆，阿波罗、席鲁斯、头翅必须同机。你站在核心桥上，手里是不动源留下的最后选择：为三人提供「一次完全同步的桥梁」（自己可能被烧进树里）、或启动军方的灭树方案（地球同样可能死）。
开场白建议：「世界的声音变低了，低到你能听见土壤在尖叫。金色的翼在眼前一点一点展开，不是金属，是某人决定把未来烧掉后剩下的光。西薇亚的眼泪在零重力里漂成珠——她说一万两千年，而你知道有人必须先把今天活完。」
关键NPC立场：**阿波罗**——已告白、已决意去，要你别挡「约定」；**席鲁斯**——和解后的平静比剑更锋利；**头翅**——完成从仇敌到共死者的最后一步；**西薇亚·朵·艾利西亚**——请求你活着作证与等待；**不动源**——只要世界开始，棋手可以消失。
主线钩子/支线：创圣合体成功、新生命之树诞生；支线＝疏散核心区残存元素、阻止灭树导弹、把阿波罗的约定原话带给未来史书。
危险度/规避：贴近顶点——生命之树与创圣级＝**七阶世界级巅峰：存在·情报优先 / 条件性胜利**，严禁硬刚「关掉树就能无代价赢」、严禁战力归零式削弱；任何替代牺牲必须付等价存在代价。
任务方向/奖励：世界存续与见证权；奖励＝创圣残光样本（叙事级）、幸存者人脉、对「1.2万年后」预言的优先解释权——不发可爆星武器。
