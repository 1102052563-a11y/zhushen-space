# →ショタ←おね

## 剧情

**【作品来源】**
全称《→ショタ←おね》（清单同）。媒介：成年向「お姉さん×ショタ」主题作品（动画／游戏系标签向）。气质＝年长女性的主导与照料、箭头从姐姐指向年下的亲密喜剧。**档案铁则：契约者与可攻略对象一律按成年角色书写**；「ショタ」作原作标签保留，入世玩法改写为**年下感外观／被照料欲**的成年关系，不涉及未成年。

**【世界定位】**
都市公寓／便利店／姐姐的房间：拖鞋、围裙、冰箱贴。一句话：这是「箭头明确从お姉さん指向你，照料与支配混成甜蜜」的年差日常世界。

**【世界观 · 舞台设定】**
无超自然。软规则：
1. **お姉さん＝主动**：邀约、做饭、定界。
2. **年下感＝被接住**：你被允许撒娇，也被要求成长。
3. **箭头可视化**：原作标题箭头即主题——她先喜欢。
4. **房间＝安全区**：门外社会，门内规则她定。
温度：味噌汤、洗衣柔顺剂、夜灯。NSFW 18 禁年差成人向；过程略。

**【地理 · 生活舞台】**
- **姐姐的公寓**：主场。
- **厨房／餐桌**：喂食与谈心。
- **便利店深夜**：偶遇与共犯购物。
- **洗衣房／阳台**：日常亲密。
- **你家／合租**：她来访的反转。
- **电车站**：送你与接你。

**【故事主线 · 情感线】**
① 你以邻居／弟弟朋友／合租者身份进入她生活圈。② 她以「照顾」名义介入作息。③ 箭头明确：她先告白式行动（便当、钥匙、留宿许可）。④ 你从被动接受到主动回应。⑤ 冲突：外界眼光「年差」与她的不安。⑥ HE：对等成年恋爱；BE：她收回箭头只剩照顾空壳。

**【可攻略角色 / 主要人物】**
- **お姉さん（核心·真名公开页不详则以身份称）**｜主导照料者｜外貌：成熟妆容、居家服与外出反差｜性格：强势温柔、怕被当妈妈｜类型：お姉さん／年差｜萌点：揉你头发却脸红；把「因为你可爱」说出口｜个人线：从照料→欲望承认→要你叫名字不叫姐姐｜与主角：箭头发起者。
- **女同事／女友候选（弱）**｜对照线。
- **邻居太太（弱）**｜舆论。
- **男主（可玩家化）**｜年下感成年｜学会对等。
- **便利店店员**｜见证深夜购物。
- **合租损友（弱）**｜起哄。

**【人际关系网 / 社团势力】**
她的房间规则 vs 外界目光。势力＝钥匙与冰箱使用权。
