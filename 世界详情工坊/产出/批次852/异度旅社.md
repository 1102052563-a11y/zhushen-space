## 剧情

**【作品来源】**
《异度旅社》作者**远瞳**（阅文白金作家，科幻代表作含《深海余烬》《异常生物见闻录》等；本书勿与盗版/同名旅馆文混淆），起点中文网连载（书号 **1041604040**，分类科幻·时空穿梭，2024-09-20 起更）。公开体量约 **263 万字 / 838 章**（连载中，最新可见章题含「女巫和孩子们」）。标签气质：脑洞、诡异、探险、轻松、无限流、诡秘养成。文风＝松弛二次元吐槽壳 + 硬核异域/规则/科幻核：日常崩坏与怪谈并存，门后真相层层抬升，不写无脑无敌。**连载中，禁止编造未公开终局**；「又开一扇门」「岁月门厅」「于生第 N 次死而复生」只作前沿气压与结构母题，不作完结定论。

**【世界定位】**
现代都市表象之下，与现实镜像/并存的**异域**层层叠压。普通人**于生**推开家门后，熟悉城市坍塌为似是而非的**界城**：路灯下三四米漆黑高瘦影子、楼顶蒸汽车头、空教室读书声、墙上画门。雨夜被多眼巨蛙穿心——然后**死而复生**。他与画中脱出的高等人偶**艾琳**、九尾银狐**胡狸**等以**梧桐路 66 号**为堡垒型基地组建**旅社**，与官方**特勤局**合作处理实体与异域，后期升格**异度旅社号**飞船、触及失乡号/域外碎片/岁月门厅。一句话：**死亡已死后，余生开门；旅社既是家，也是通往世界根源的门廊。**

**【世界观 · 力量体系】**
本世界「力量」不来自传统灵气飞升境界表，而来自四层叠合：**异域规则理解**、**个体异变/血脉天赋**、**开门与通道控制**、**组织编制与装备**。战斗逻辑优先「懂规则 + 可控门 + 庇护权限」，其次才是火力。任何变强须能落到可观察代价：血、门、许可、实体残留、官方登记。

**一、异域与实体（舞台与危险源）**
1. **异域**：与现实并存的诡异空间，规模越大规则越复杂。普通人接触多数死亡，约十分之一会成为处理异域事务的「专家」气质幸存者。
2. **类型（公开设定常见四分）**：
   - **旷野型**：无明确边界的自然/荒野空间（如可改造的山谷）。
   - **国度型**：有大量人造痕迹与相对稳定出入口（如博物馆类）。
   - **堡垒型**：单一封闭建筑、规则复杂——**梧桐路 66 号**即属此类。
   - **异常型**：抽象概念化（情绪、记忆、故事）——**「童话」异域**为代表，带主动恶意。
3. **实体**：异域中有活性的存在（血肉怪物、规则系如「实体-饥饿」——以饥饿与疯狂影响人，倒下者可被并入）。可被击杀，但**往往无法被彻底消灭**；受**晦暗天使**影响会活化/狂暴。
4. **晦暗天使**：来自世界之外/新宇宙边缘的威胁源。其降临或投影可造成环境剧变、精神污染、规则紊乱、失控与大规模死亡。类型多样（如精神干扰的树天使、惰性巨眼、高速跃迁+晶簇污染的衍星体等）。崇圣隐修会等极端势力与其触媒研究纠缠。
5. **交界地 / 裂隙**：现实与异域的薄弱处；特勤局监控重点。

**二、于生系核心能力（主角模板，非人人可复制）**
1. **死而复生 + 身体异变**：执念驱动多次死亡后回归（目录反复出现「于生第 N 次死而复生」）。恢复力极强（断骨速愈级），接触异域后力量/敏捷提升，可吞噬/利用怪物肢体增强自身——**复活不是无代价无敌**，每次死亡都是叙事与认知的升级节点。
2. **开门人 / 通道控制**：利用现实门或凭空造门，通往异域节点；可引发时空震动、稳定链接；经血测试等实验后可做到「成功复现，通道可控」。乱开门＝自杀级操作（艾琳早期铁律：「别乱开门」）。
3. **血液与庇护**：血液对敌可有侵蚀/控制向效果；对友方可敞开**庇护荒原**等权限，赋予灵魂/存在层面的庇护——旅社「收容伙伴」的硬规则根基。
4. **感知异样**：能看见常人不可见的实体与违和结构；部分人永远看不见。

**三、伙伴/种族天赋（多体系并存）**
- **人偶系（艾琳、露娜等）**：油画/钢铁躯壳、量产分身、能量吸收释放、预制门、古老知识库；艾琳后期可指挥人偶军团（「满坑满谷的人偶」「艾琳的王牌军团」）。
- **兽灵/妖狐系（胡狸）**：狐火、尾巴飞弹/储物、高速、敏锐嗅觉、尾巴可再生；火力与后勤双线。
- **童话组织成员（小红帽等）**：童话异域规则加持，深潜/狩猎/团战适配。
- **修仙系（玄澈/千峰灵山等）**：基础法门可对外族开放学习，与科幻/怪谈并置不冲突。
- **特勤局深潜员/专家**：官方装备、边境通讯、登记许可、编制火力。

**四、组织编制与成长阶梯（按破坏力/可控范围，宁低勿高）**
1. **普通人 / 误入界城者**：无超凡，只能绕路、记录、逃；热武器对高阶实体往往无效。
2. **活过规则的初死者**：理解「假门/恶意天气/实体」后的幸存者；初获异变或复活认知。
3. **可控通道者**：能稳定复现门/通道，完成血测试与开门实验；旅社雏形。
4. **大队编制级**：与特勤局二大队等合作，处理成规模异域事件；旅社注册合法组织。
5. **跨异域 / 域外碎片层**：触及黑森林外新城、失乡号、黑星、域外碎片、星际航线。
6. **岁月门厅 / 多次复活临界**：根须天幕、断层、古圣灵、岁月残响、又开一扇门——地区级灾难以上的结构战。

**五、死亡、规则与纪律**
- 乱开门、无视门规、把实体当普通怪物硬刚 → 高概率团灭。
- 实体可杀不可尽灭；晦暗天使优先「封堵/切断脐带/触媒对抗」，禁止写成一刀砍死的普通 BOSS。
- 官方登记、非人类身份、血液权限、旅社房契/门钥都是可观察资源。
- 连载前沿之后的「世界根源」全貌**不详**，写作只保留门后气压。

**乐园阶位映射**（对照破坏力，宁低勿高）：普通人误入界城/仅能观察绕路≈**一阶**；活过规则·初死复活认知·能躲实体≈**二阶**；通道可控·血测试成功·旅社雏形≈**三阶**；特勤局大队编制·旅社成军·成规模清异常≈**四阶**；跨异域·域外碎片·飞船/失乡号节点≈**五阶**；岁月门厅/根须天幕/多次复活临界·结构级威胁≈**六阶**。世界当前可见顶点＝古圣灵/晦暗天使投影/岁月门厅类结构威胁；**无爆星超阶战力作为常规 NPC**；低阶切入一律「规避顶点·情报优先/条件性胜利」，严禁把晦暗天使写成可被凡人小队无代价斩杀。

**【地理 · 舞台】**
**界城（开局异化都市）**
- 似是而非的现代城：理事塔取代熟悉地标（公开叙述有博源大厦→理事塔一类错位）、四元街商店变墙、电话亭画风错位、楼顶蒸汽车头、空教室读书声。
- **路灯黑影**：三四米漆黑高瘦影子，常人不可见；绕开不等于安全。
- **恶意冻雨 / 镜面地面**：选中个人的天气猎杀；多眼雨蛙从镜面升起，舌可拐弯穿心。
- **墙上假门 / 画门**：猎杀装置与异域入口伪装。

**梧桐路 66 号（旅社核心堡垒）**
- 堡垒型异域住所/基地；早期仅接触过于生血液者可见/可进入。
- 功能：起居、餐饮招待、安顿伙伴、实验开门、情报汇总；后期可「便携化」、注灵建跨异域门，甚至出现「便携式梧桐路 66 号」应用。
- 感官锚：旧铁门油漆味、上锁房间、油画框、走廊门规告示、人偶与狐火共存的违和温馨。

**特勤局 / 官方侧**
- 监控裂隙、调查实体、非人类登记、对抗邪教；**藏龙卧虎二大队**为早期合作接口之一。
- 装备与通讯（边境通讯手机等）是中期战力补丁。

**黑森林与童话相关**
- 黑森林该溜子社团、孤儿院/孩子们、小红帽与狼外婆线、童话组织；后期「黑森林边的新城」。
- 异常型童话异域：故事规则可反噬现实。

**中后期扩张舞台**
- **仙梭坠毁点**、猎人邀请、静林疗养院、茧之厅、焦点之山。
- **异度旅社号**飞船与星际节点；**失乡号**、黑星现象、瓦尔哈拉气质节点、爱丽丝树。
- **域外碎片**、边境之外消息、根须天幕、深井、断层、**岁月门厅**、林中阴影、破碎王城/碎裂之境（连载前沿）。

**空间使用提示**：一阶黏在界城街道/路灯/冻雨；二阶黏在假门—上锁房间—初死现场；三阶黏在 66 号开门实验；四阶上特勤局编制与黑森林/童话清场；五阶起才抬到飞船/失乡号/域外；六阶触及岁月门厅与结构残响。地理移动要付代价：门权限、血、官方许可、实体污染。

**交界地与裂隙工作带**
特勤局在城市与异域之间划出监控网格：裂隙探针、临时封条、深潜出发层。契约者低阶最容易死在「跟着闪光点走进裂隙」——闪光点可能是出口提示，也可能是实体诱饵。写出口寻找时可用公开规则气质：违和物品、高反光处、与周围画风不一致的门框；找到出口不等于安全，门外可能是另一层异域。

**感官地理速查（按阶）**
一阶：雨腥、镜面反光、路灯嗡鸣、购物袋塑料声。二阶：血温、假门油漆、空房间回声。三阶：油画颜料、狐火焦香、血试管标签字。四阶：森林湿雪、通讯忙音、编制靴踏泥。五阶：船舷低频震动、碎片罐密封胶味、黑星观测窗的暗。六阶：门厅无钟的静、根须摩擦声、复活后第一口空气的金属味。


**感官与文书铁则（写正文必用）**
优先气味/声景：冻雨如钉、镜面反光、蛙舌穿胸的湿冷、旧油漆门、油画颜料、狐火焦香、人偶关节轻响、边境通讯忙音。任一强化须能落到三类可观察物至少两类——**器物**（门钥、油画、血样、通讯器、实体残肢）、**文书**（特勤局登记、旅社名册、门规告示、血测试记录）、**惩罚**（乱开门伤亡、污染、复活计数、权限回收）。缺代价的升级视为灌水。

**阶段〇 · 失家与界城（开篇）**
于生自认普通二十多年，直至某次清晨推门（公开叙述有买橘/日常外出气质）后**永失其家**：熟悉城市变成更大、无名、似是而非的界城。身份证明与住址仍可能有效，却困在更大「森林」。他开始看见只有他看得见的不对劲：电话亭错位、蒸汽车头、空教室、路灯黑影。此阶段功能：把「日常」钉成已崩坏的表象；穿越或感知异变两假说并列，不急于给唯一答案。

**阶段一 · 雨与初死（一～二阶核心教学）**
章题链：雨 → 无人受害 → ……。于生绕开黑影走老旧小道，冻雨提前，地面成镜，行人蒸发感，墙上假门，**多眼雨蛙**从镜面升起，舌拐弯穿心。死亡教学完成：「优雅用词救不了心脏」。随即**复活**——世界状态从「怪谈观察」切到「你已死过一次」。无人受害表象与个人地狱并存：旁人可能完全无感。写作铁则：初死必须写出感官与规则，禁止秒变超级英雄。

**阶段二 · 上锁房间与画中艾琳（伙伴契约）**
章题：上锁的房间 → 房间里没人 → 画中的艾琳 → 友好交流的第一步 → 艾琳的脱困方案 → 别乱开门 → 一点真相 → 花开二度 → 事不过三。于生在上锁/空房间线索中接触油画内的**艾琳**——哥特油画气质的高等人偶，活泼聒噪、爱吐槽、馊主意与关键时刻可靠并存。脱困方案落地，**「别乱开门」**成为旅社宪法第一条。花开二度/事不过三提示：异常会复发，门后没有「一次解决永久安全」。

**阶段三 · 胡狸、饥饿与调查人员（旅社雏形）**
章题：这地方有人！？ → 碰面 → 胡狸 → 困于此地 → 饥饿 → 门 → 调查人员 → 回家。破庙/废墟气质场景中解救饥饿的九尾银狐少女**胡狸**；「饥饿」同时是生存问题与**实体-饥饿**规则预告。调查人员上门——官方视线进入私人异常。回家与情报建议（艾琳的情报与建议）把「临时避难」升级为「可经营的据点」。梧桐路 66 号作为堡垒型基地确立。

**阶段四 · 开门人与通道可控（三阶质变）**
章题：镜中景 → 梦中狐 → 入梦深处 → 跟人偶相处的日子 → 开门人 → 血的测试 → 延迟生效？ → 成功复现，通道可控！ → 藏龙卧虎二大队。于生完成从「乱开门会死」到「可控通道」的实验链：血测试、复现、延迟生效等失败/半成功节点必须保留。特勤局**藏龙卧虎二大队**入场，旅社从私人收容所变成可与官方对话的编制接口。塑造/身体筹备（筹备身体的第一步、塑造、自由的人偶）为艾琳实体化与人偶线埋线。

**阶段五 · 天使、童话与黑森林（四阶成军）**
中期章题簇：第一天使 → 孤儿院/孩子们 → 黑森林该溜子社团初具雏形 → 小红帽的狼外婆 → 陌生的「小红帽」 → 天使的名字/脐带 → 仙梭坠毁点 → 猎人的邀请 → 风雪越界。**晦暗天使**从概念变为可点名的威胁；**小红帽**作为特勤局深潜员气质角色进入合作/对峙光谱；童话异域把故事角色（狼外婆等）变成可战斗、可污染、可收编的变量。黑森林成为中期主舞台之一：社团、孩子们、猎人、风雪越界把「城市怪谈」抬到「森林规则战」。旅社侧日常（艾琳惊世厨艺、胡狸做的「饭」、66 号人杰地灵）与前线交叉，保持轻松壳。

**阶段六 · 人偶军团、入侵与童话战线（四～五阶抬升）**
约 500 章前后：来自百里晴的情报 → 静林疗养院 → 极端学术组织 → **满坑满谷的人偶** → 茧之厅 → 大规模入侵 → 狐冲直撞 → 童话战线。**百里晴**（特勤局局长气质）提供情报与注册/合作接口；艾琳人偶产能从个体助手扩到军团级；大规模入侵把编制战推上台面。洛、茧之厅等节点提示：异域内部还有更深层结构。此阶段奖励与仇恨同步最大化——旅社已是靶子。

**阶段七 · 失乡号、黑星与域外碎片（五阶跨域）**
约 700 章前后：黑星现象/研究 → **古圣灵，失乡号，以及黑星** → **域外的碎片** → 黑森林边的新城 → 瓦尔哈拉气质节点 → 边境之外消息 → 爱丽丝树。舞台从行星表面异域抬到星区/域外：失乡号与旅社号形成「船级」对比；古圣灵（爱丽丝相关体系的高位存在）入场；新城建设＝安家落户与边境经营。玄澈等修仙侧人员可加入飞船探索。写作纪律：跨域不等于无敌，黑星/域外碎片优先污染与信息战。

**阶段八 · 岁月门厅与又开一扇门（六阶前沿，不作终局）**
连载前沿章题：艾琳的王牌军团 → 碎裂之境 → 高墙之下 → 破碎王城 → 岁月消退之后 → 远方来客 → 死神的退休生活 → 古圣灵也是有各种性格的 → 断层现状 → 再探深井 → **根须天幕** → 尽头 → **岁月的门厅** → 林中阴影 → 岁月的残响？ → 血色一闪 → **于生的第 N 次死而复生** → 毛了一波大的 → **又开一扇门** → **女巫和孩子们**。结构从「清一个副本」递归为「时间/岁月/根源门厅」；复活计数成为角色标签与代价暗示；新门开启＝叙事永不封顶。**禁止**写成已揭完的世界根源终局或全员无敌大团圆。女巫与孩子们是最新可见气压，只作钩子。

**阶段九 · 可介入日常事件池（AI 优先写这些）**
界城绕路记录黑影；雨夜护送/假门排查；66 号上锁房清点；艾琳脱困协助；给胡狸找食与防饥饿实体；血测试志愿/记录；二大队联合清场；黑森林带路/孤儿院疏散；童话规则谈判；人偶分身后勤；静林疗养院潜入；入侵警报下的门控值班；失乡号/旅社号甲板勤务；域外碎片取样封存；岁月门厅外围测绘。失败惩罚层次：轻度——门权限暂时回收、污染症状、官方警告；中度——实体标记、伙伴反噬风险、复活计数+1 的叙事压力；重度——乱开门团灭、晦暗触媒泄漏、旅社据点暴露。

**蝴蝶效应边界**
可改：某次绕路成败、某扇门是否开、是否救下某次事件中的孩子/狐/人偶、与特勤局合作深度、是否提前暴露血液权限。不可无代价改：异域类型法则、实体难尽灭、乱开门致死级风险、晦暗天使的域外威胁属性、于生复活机制的存在本身（可改触发条件与代价表现，不可改成无限无敌）。乱改回流为：裂隙扩大、官方收容、伙伴失控、门厅污染、匹配难度上调。

**门规与收容伦理（贯穿全线）**
旅社不是无敌堡垒，是「收容与风险」的平衡装置。艾琳脱困、胡狸困饿、露娜忠诚、孩子们安置——每一份收容都绑定门规与食物/权限成本。官方调查人员代表秩序登记；崇圣隐修会代表「修复世界」话术下的极端实验；爱丽丝小屋/古圣灵代表更古老的人偶—圣灵谱系。于生第 N 次死而复生是系列母题：读者与角色都在计数，写作须让每一次死有信息增量。


**阶段补充 · 界城日常崩坏细节（开局可写池）**
界城不是「换皮地球」：同一条街的店招字体可能差半个字号，公交站名多一个生僻偏旁，便利店冷柜嗡鸣频率像故意调慢。于生式观察法＝先记三处违和，再决定绕路还是敲门。路灯黑影的规则教学必须分层：远看像杆影，近看有关节，被它「注意」时空气变粘——但一阶禁止写成可斩杀的小怪，最多是「被注视后冻雨概率上升」。假门的油漆未干味是关键嗅觉锚；摸上去若温热，说明内侧有东西在呼吸。写「无人受害」时要给对照：监控里没有尸体，路边没有警戒线，只有你衬衫上洗不掉的雨腥。

**阶段补充 · 旅社经营与门规日常（三～四阶可写池）**
梧桐路 66 号的「温馨」是违和的温馨：人偶关节轻响当背景音，狐火偶尔燎到门框，门规告示用红笔加粗「别乱开门」并被艾琳吐槽字丑。餐饮招待章题气质说明旅社会对外（或对官方访客）提供餐食——写场景时食物是安全感，也是饥饿实体的诱饵。血权限管理要像钥匙管理：谁进过门、谁碰过血样、谁被划入庇护名单，都应有可点名的名册行。调查人员上门不是龙傲天刷脸，是盘问、取样、临时封条与「你们最好配合」的压迫感；旅社侧用情报交换与清场战果换注册空间。

**阶段补充 · 童话/黑森林作战纪律（四阶）**
童话异域的危险在于「故事逻辑覆盖物理逻辑」：狼外婆可以是亲缘也可以是猎手，孩子们的笑声可能是安全屋也可能是诱饵。小红帽的冷静是职业外壳，不等于把契约者当自己人；联合行动时指挥链冲突（旅社门规 vs 特勤局纪律）本身就是戏剧点。风雪越界不是天气特效，是边界失效：雪线一侧是城市规则，另一侧是森林规则，夹在中间的人会「同时被两套规则惩罚」。天使脐带/名字类情报一阶二阶只可耳闻，四阶可参与外围切断，禁止低阶写斩天使本体。

**阶段补充 · 船级与域外写作纪律（五阶）**
黑星优先污染与观测异常，不优先爆炸特效。域外碎片的标签「勿直视、勿听、勿开门」是三禁，违反任一条都应立即结算代价（幻觉、权限回收、实体标记）。失乡号与异度旅社号并置时，强调「乡愁/失家」主题回环——于生推门失家的母题在船名上复发。新城建设戏要有物资、许可、污染监测，禁止「一夜盖成无敌城」。古圣灵有性格差异（章题「古圣灵也是有各种性格的」），不可写成单一神性模板。

**阶段补充 · 门厅前沿气压（六阶，禁终局）**
岁月门厅、根须天幕、断层、残响，写的是时间被结构看见，而不是主角突然学会时间停止。第 N 次死而复生要让队友的表情与计数发生变化：玩笑变短、门规更严、艾琳骂人更脏、胡狸把食物往于生面前推。又开一扇门是结构承诺：本世界故事引擎是门，不是升级条。女巫和孩子们只给钩子与感官（笑、脚步、不合时宜的童谣），不给身世全解与最终BOSS战剧本。

- 百里晴超长履历背后的时间/门厅异常。

## 阶位切入点

> 阶位↔原作：一阶≈普通人误入界城·仅观察绕路；二阶≈活过规则·初死复活认知；三阶≈通道可控·旅社雏形；四阶≈特勤局大队编制·旅社成军；五阶≈跨异域·域外碎片·船级节点；六阶≈岁月门厅/根须天幕·多次复活临界。世界顶点（晦暗天使投影/古圣灵/岁月门厅结构威胁）为跨阶存在，低阶切入一律「规避顶点·情报优先/条件性胜利」，严禁「被规则锁死削弱」式硬撼。

**一阶（普通人 · 雨前界城绕路线）**

切入身份/时点：契约者以「刚推门失家、仍以为自己在熟悉城市迷路的普通人」身份切入，锚定于生尚未被雨选中、只在商业街与老旧小道之间躲路灯黑影的最早开局。

初始事件：你在似是而非的街道上看见路灯下三四米高的漆黑瘦影——旁人直直穿过，只有你瞳孔发紧。冻雨气味提前渗来，墙上出现一扇油漆未干的假门；购物袋还在手里，回家的路已经不对。

开场白建议：「界城的雨比记忆里沉。你绕开路灯下那道只有你看得见的高瘦黑影，鞋底却踩上开始反光的路面——电话亭的画风错了一帧，楼顶停着不该存在的蒸汽车头。购物袋还在手里，家门已经不在原来的地方。」

关键NPC立场：**于生**——同期误入者/可能的同伴，吐槽里藏谨慎，尚未死过，对「只有我们看得见」既恐惧又想验证；路人——完全无感，会把你的警告当神经病；路灯黑影——环境威胁，不主动解释规则。

主线钩子/支线：活着摸回「像家的地址」、记录三种违和物、确认假门是否可触；支线＝空教室读书声排查、电话亭错位拍照、与于生交换第一条「我看得见」暗号。

危险度/规避：中高——须规避恶意冻雨锁定、假门内侧、任何主动招惹黑影的行为；禁止把一阶写成已能杀蛙。

任务方向/奖励：生存与第一手异常笔记；奖励取向＝界城地图碎片、门规传闻、与于生的初步信任，不发通道权限。

**二阶（初死者 · 雨蛙与复活线）**

切入身份/时点：以「刚经历穿心之死、带着未干的雨腥复活」的初死者切入，锚定假门事件当夜至「无人受害」表象被撕开前后。

初始事件：多眼雨蛙的舌从镜面地面拐弯刺穿胸腔；你在「优雅的遗言」说到一半时醒来，心脏狂跳，而街道上的人行色如常——仿佛无人受害。上锁房间的线索与墙上假门同时指向下一扇不该开的门。

开场白建议：「舌头是凉的，笑话却卡在喉咙里。你死过一次——多眼雨蛙、假门、冻雨，像一套写好的猎杀程序。醒来时界城仍在下雨，行人撑伞走过，像什么都没发生。你知道自己再也不能装成普通人了。」

关键NPC立场：**于生**——同为初死体验者或即将同步体验者，开始用实验思维对待死亡；**艾琳**（画中阶段）——隔着油画观察、评估谁值得对话；雨蛙/假门规则——无谈判，只执行。

主线钩子/支线：接受复活现实、追查上锁房间、验证「无人受害」是否全局；支线＝收集冻雨样本、标记假门出现条件、第一次对实体残留产生恐惧性好奇。

危险度/规避：高——须规避二次被雨选中、无准备触碰假门、向无感路人展示尸体级证据导致被当疯子隔离。

任务方向/奖励：复活认知与房间线索；奖励取向＝异变萌芽、实体残肢研究入口、进入上锁房的钥匙级线索。

**三阶（旅社雏形 · 艾琳脱困与通道可控线）**

切入身份/时点：以「被允许踏入梧桐路 66 号、参与开门实验的编外助手」身份切入，锚定画中艾琳交流之后、血测试至「通道可控」前后。

初始事件：油画中的艾琳提出脱困方案，第一句警告是「别乱开门」；胡狸在饥饿边缘发出危险的嗅闻。血测试台上，你的或于生的血被用来复现通道——延迟生效、半成功、突然开大的门缝里，有什么东西在呼吸。

开场白建议：「梧桐路 66 号的铁门油漆味很旧，旧得像早就等人来住。画框里的艾琳歪头看你：『友好交流的第一步，是先确认谁会吃人。』走廊尽头，胡狸的尾巴扫过地板，肚子叫了一声——那声音让你想起『饥饿』不只是形容词。」

关键NPC立场：**于生**——实验主导者，门控强迫症发作中；**艾琳**——脱困中/刚脱困，毒舌但给方案；**胡狸**——困饿，食物与善意可换忠诚；调查人员——上门登记，合作与收管一线间。

主线钩子/支线：协助艾琳脱困、参与血测试与通道复现、制定门规告示；支线＝给人偶找身体材料、给狐找食并防实体-饥饿、应付调查人员问询。

危险度/规避：高——须规避擅自开门、血权限外泄、让饥饿实体沾上厨房。

任务方向/奖励：伙伴契约与门规席位；奖励取向＝66 号进入许可、基础通道协助权、旅社名册登记。

**四阶（成军 · 二大队与黑森林/童话线）**

切入身份/时点：以「旅社注册成员或特勤局编外协作员」身份切入，锚定藏龙卧虎二大队对接之后、黑森林该溜子社团/小红帽狼外婆至风雪越界阶段。

初始事件：二大队要求联合清场；黑森林方向传来孤儿院与孩子们的异常报告，陌生的「小红帽」在通讯里报出深潜坐标。风雪越界之夜，童话规则开始改写地图，你必须在「官方纪律」与「旅社门规」之间选一个当场有效的指挥链。

开场白建议：「通讯器里跳出『藏龙卧虎二大队』的编制码，像一句不好笑的玩笑。黑森林的风雪却认真得很——小红帽的声音冷静：『狼外婆不是比喻。』你看了一眼于生，又看了一眼门规告示：乱开门的人，今晚别指望有人给你收尸。」

关键NPC立场：**于生**——旅社决策核，愿合作但拒交血权限；**艾琳**——战场吐槽与人偶支援；**胡狸**——开大助攻；**小红帽**——深潜纪律优先，可结盟亦会按规上报；**百里晴**——远端授权与情报，态度冷而准。

主线钩子/支线：联合清场、黑森林疏散、童话规则谈判/斩断；支线＝该溜子社团接触、狼外婆线索、天使名字/脐带级情报的初步耳闻（只听不碰高位）。

危险度/规避：极高——须规避晦暗天使投影本体、童话核心规则反噬、无编制闯入官方封锁区。

任务方向/奖励：编制许可与前线战功；奖励取向＝特勤局协作装备、童话抗性道具、旅社火力升级材料。

**五阶（跨异域 · 域外碎片与船级线）**

切入身份/时点：以「异度旅社号/失乡号相关节点的船员、研究员或边境代理人」身份切入，锚定黑星研究、古圣灵—失乡号、域外碎片至黑森林边新城阶段。

初始事件：黑星现象让仪器失灵；一份标注「域外的碎片」的密封样本在甲板间失窃或泄漏。百里晴的情报与玛琳式推测同时到达，新城工地需要人手——而失乡号方向传来「人杰地灵」的反常热闹，像另一座会走路的 66 号。

开场白建议：「船舷外不是海，是被黑星舔过的暗。失乡号的轮廓在通讯全息里一闪，像一句写错的乡愁。你把域外碎片的封印罐抱紧：标签写着『勿直视、勿听、勿开门』——第三项是旅社传统，前两项是刚学的。」

关键NPC立场：**于生**——跨域调度者，传统艺能仍是死而复生与开门；**艾琳**——人偶军团后勤与甲板防御；**胡狸**——高速侦察；**百里晴**——情报分级；**玄澈**——修仙侧技术支援；古圣灵/失乡号变量——高位，态度不明。

主线钩子/支线：碎片封存与溯源、失乡号接触任务、黑森林边新城建设/防御；支线＝黑星采样、爱丽丝树巡礼、边境之外消息核验。

危险度/规避：极高——须规避未授权域外直连、黑星污染核心、古圣灵意志冲突；顶点相关＝情报优先。

任务方向/奖励：跨域素材与船级权限；奖励取向＝域外样本份额、飞船舱段权限、新城地契级立足点。

**六阶（岁月门厅 · 第 N 次复活与又开一扇门）**

切入身份/时点：以「门厅外围测绘员/旅社核心随员/多次死过仍站着的老资格」身份切入，锚定根须天幕、岁月门厅、于生第 N 次死而复生至又开一扇门、女巫和孩子们的连载前沿。

初始事件：根须从天幕垂下，断层里渗出岁月残响；于生在血色一闪后再次倒下又站起——计数器在所有人心里 +1。新的一扇门自行开启，门后有女巫的笑与孩子们的脚步声。百里晴罕见地承认：这超出她近百年怪异履历的处理模板。

开场白建议：「岁月门厅不挂钟，却让每根头发都想起自己的年龄。你听见于生第 N 次把气喘匀，艾琳骂了一句很脏的古语，胡狸的尾巴炸成刷子。门又开了——传统艺能。门后是女巫和孩子们的气息，像童话终于长出了牙。」

关键NPC立场：**于生**——复活临界的开门人，疲态与执念并存；**艾琳**——王牌军团尽出；**胡狸**——仍饿、仍打；**百里晴**——第一次公开「懵」；远方来客/古圣灵/女巫与孩子们——前沿变量，立场以正文为准，不可写成已收编的工具人。

主线钩子/支线：门厅测绘与残响采样、护住第 N 次复活后的窗口期、决定是否踏入「又开一扇门」；支线＝碎裂之境收尾、深井再探、死神退休生活式的荒诞侧写（轻松壳保留）。

危险度/规避：贴近顶点——须规避门厅核心结构硬闯、与古圣灵/女巫规则正面对撞；**超阶巅峰：存在·情报优先/条件性胜利**，严禁削敌卖血。

任务方向/奖励：门厅外围权限与不可逆航线情报；奖励取向＝岁月残响样本、复活窗口护持技术、新门坐标（高污染）。
