# 骑马与砍杀2：霸主

## 剧情

**【作品来源】**
《骑马与砍杀 II：霸主》（Mount & Blade II: Bannerlord，亦译领主）由土耳其 TaleWorlds Entertainment 开发发行，作曲 Finn Seliger；实体版曾与 Prime Matter 合作。2012 年公布，2020 年 3 月 27 日 Steam 抢先体验（当年 Steam 最大体量上线之一），2022 年 10 月 25 日全平台正式版（Windows / PS4 / PS5 / Xbox One / Series X|S）并支持创意工坊。系列定位：2008《骑马与砍杀》→ 2010《战团》的**前传**——时间线约在《战团》**173 年前**（社区亦常称约 200 年），写帝国崩解与列国成型。文风：中世纪沙盒 + 轻微历史奇幻氛围，但**无魔法体系**；旗帜、氏族、攻城、经济与政治共同驱动。标签：卡拉迪亚、卡拉迪克帝国三分、龙旗、彭德莱克之战、氏族与影响、围城器械、工坊、流浪者同伴。DLC《战帆 War Sails》（海军与诺德系新势力向，2025 年）属后续扩展，本档案以基线正式版卡拉迪亚大陆为主。媒介以游戏正典与游戏内百科为准；维基/社区对领主口述的整理作交叉参考。

**【世界定位】**
架空大陆**卡拉迪亚（Calradia）**正值**卡拉迪克帝国（Calradic Empire）**日薄西山：皇帝**阿伦尼科斯（Arenicos）**遇刺身亡后，帝国裂为**北 / 南 / 西**三个互称正统的帝国；西有封建重骑**瓦兰迪亚**，西北山林**巴旦尼亚**，东北雪林**斯特吉亚**，东草原**库赛特**，南沙漠**阿塞莱**——五大王国环伺，雇佣兵与匪帮填满缝隙。玩家自自定义出身的小人物起步，组战团、建氏族、争采邑、扶一统或自立为王。一句话给 AI：**这是「帝国余烬里用骑砍、氏族与旗帜写沙盘史诗」的世界——主线有龙旗骨架，终局仍是沙盒谁先插满地图。**

**【世界观 · 力量体系】**
力量来源是**凡人冷兵器军事 + 后勤 + 政治**，不是灵气或法术。核心变量：
1. **个人武勇**：属性（活力/控制/耐力/狡诈/社交/智力）与技能树（单手/双手/长柄/弓/弩/投掷/骑术/运动/医疗/工程/铁匠/战术/统御/魅力/交易/偷窃等）；高熟练可单挑小股，但无法空手裂城。
2. **部队质量与编制**：各文化兵种树（帝国铁甲甲骑/射；瓦兰迪亚骑士；巴旦尼亚菲安长弓；斯特吉亚斧盾；库赛特骑射；阿塞莱混成骑步）；伤病、士气、粮食、工资决定能不能打下一场。
3. **氏族（Clan）与王国**：氏族等级限制同伴数、工坊数、军团规模；**影响力（Influence）**用于提案政策、召集军团、强行分配采邑；政策改税赋、忠诚、民兵、君权强弱。
4. **经济**：商队、工坊（酿酒/铁匠/织布/橄榄油等，受城镇繁荣与供需约束）、赎金、掠夺、关税。
5. **攻城**：战图上建造投石机/攻城塔/冲车轰墙开缺口；入城后城垛/门楼可毁，守方有杀伤孔与狭窄巷道优势。
6. **死亡与继承**：士兵死即死；领主可被俘勒索或处决；玩家战败可被俘；开启子嗣后可由子女继承部队与采邑（角色死亡换代）。无复活神术。
7. **特殊叙事物**：**龙旗（Dragon Banner）**是帝国法统象征，碎片可拼合，用于开启建国立业与阴谋线——它是政治炸弹，不是魔法灭国器。

战力阶梯（破坏力按《阶位战力图鉴》宁低勿高；本世界几乎全落在凡俗军事）：
1. **难民 / 村民 / 新兵 / 小偷小摸**：徒手或农具，一刀可致命。≈一阶初期。
2. **山贼 / 逃兵 / 酒馆打手 / 雇佣兵新锐 / 民兵**：冷兵器格斗、踹破木门、截商队。≈一阶中后期。
3. **正规军精锐 / 小氏族战团 / 流浪者名手**：单挑小队、端匪巢、野战侧翼。≈二阶。
4. **名城守军 / 大领主军团 / 满编文化精锐（骑士、菲安、甲骑射等）**：野战歼灭、围城破门、踏平山寨级。≈三阶。
5. **国王 / 女皇 / 皇帝亲征军与帝国军团合编**：持续灭国战争、连城易手。≈四阶。
6. **统一卡拉迪亚或重建帝国的霸主级势力**：大陆军事政治顶点，仍是凡人旗帜最多者。≈五阶。

乐园阶位映射：新兵·匪·民兵≈**一阶**；精锐士兵·小领主·成名同伴≈**二阶**；大领主军团·名城攻防≈**三阶**；君主主力·三帝亲征≈**四阶**；统一帝国/取代帝国的军事霸权≈**五阶**。世界顶点＝重建或取代卡拉迪克帝国的**军事政治巅峰（五阶）**；**无超阶魔法存在**。同阶因装备、兵种克制、地形与指挥可上下浮动观感，但禁止写成「一骑毁城」「言出法随」。

**【地理 · 舞台】**
卡拉迪亚为半岛状大陆，地貌与文化分区清晰；游戏内**无官方首都机制**，下列「主城」按开局统治氏族驻地与社区共识标注：
- **北帝国（Northern Empire）**：元老寡头法统；主城常指 **Diathma**；文化帝国式甲骑、矛阵、弓手；代表氏族 **Osticos（吕松）**。
- **南帝国（Southern Empire）**：帝系血脉继承；主城常指 **Lycaron**（另有 Onira 等）；代表氏族 **Pethros（拉盖娅）**；帝都文明残余、竞技场与贵族宴会氛围浓。
- **西帝国（Western Empire）**：军选皇帝、「军队即国家」；主城常指 **Jalmarys**；代表氏族 **Comnos（加里奥斯）**；军营、老兵聚落、边防感强。
- **瓦兰迪亚（Vlandia）**：西海岸封建重骑；主城常指 **Galend**（亦有人推 Pravend / Charas 为历史重镇）；灵感诺曼/法兰克/汪达尔—哥特混合；兵种：重甲骑士、弩手、矛步。
- **巴旦尼亚（Battania）**：西北/中西山林；主城 **Marunath**；灵感凯尔特；菲安（Fian）长弓、镰刀斧突击、林地伏击；氏族名前缀 **fen**。
- **斯特吉亚（Principality of Sturgia）**：东北雪林湖沼；主城常指 **Balgard**（港口大城如 Varcheg 亦重）；灵感维京/罗斯；斧盾步兵、盾墙；湖匪「湖鼠」盘踞湖岸。
- **库赛特汗国（Khuzait Khanate）**：东部草原；主城 **Makeb**；灵感匈奴—突厥—蒙古系；骑射、长矛骑；定居化与游牧传统张力大。
- **阿塞莱苏丹国（Aserai Sultanate）**：南部纳哈萨沙漠绿洲；主城常指 **Quyaz**（Sanala 等同氏族重镇）；灵感阿拔斯/马穆鲁克前伊斯兰部落；骑步均衡、骆驼与商路。
- **缝隙地带**：匪窝、走私港、森林兄弟会地盘、沙漠游牧草场、草原黑库赛特营地；商队路线是早期生死线。
氛围关键词：泥、雪、沙、旗帜、攻城塔、酒馆招募、氏族宴会、赎金契约。AI 写场景先定「玩家在哪一文化区 + 战争是否开打」，再定冲突强度。

**【世界剧情线】**
本世界是**沙盘史诗**：有可追溯的前史与「龙旗」任务骨架，但无电影式唯一结局。下列按时间顺序写清「任意时点世界处于什么状态」。

① **前史：帝国霸权与边军逻辑（远景）**  
卡拉迪克帝国长期以银币、公民权与边军编制收编外族：瓦兰迪亚骑士当边骑，斯特吉亚人当斧兵，阿塞莱与库赛特提供雇佣骑射，巴旦尼亚人在林线两侧既是盟友也是叛乱源。帝国文化（希腊—罗马—拜占庭意象）提供法典、银币、军团与「龙旗」符号；边疆则积累百年血债——抢牛、毁村、人质、背誓。写这一层时：本地人把帝国既当文明灯塔也当收税机器；「帝国人」身份在边疆可救人也可招杀。

② **1077 · 彭德莱克之战 / 「内雷泽斯的愚行」（Neretzes’ Folly）**  
皇帝 **德罗西奥斯·内雷泽斯（Drosios Neretzes）** 率帝国主力并雇佣阿塞莱、库赛特部队，远征巴旦尼亚方向的 **彭德莱克（Pendraic）** 一带，对抗 **斯特吉亚 + 巴旦尼亚 + 瓦兰迪亚** 联军。这是开局前约一世代的「大陆级创伤」，几乎所有大贵族都能讲一段，且**口述互相矛盾**——正是设计意图。

前奏：巴旦尼亚前至高王 **埃里尔（Aeril）** 曾与内雷泽斯有停战誓约；埃里尔「狩猎失踪」，养子/婿 **卡拉多（Caladog）** 继位（杀君传闻缠身）。卡拉多毁约，应氏族「百年血债」之请联斯特吉亚。内雷泽斯召瓦兰迪亚履行旧防务誓约；瓦兰迪亚男爵观望，内雷泽斯斥其为懦夫叛徒——**侮辱把国王德瑟特逼到对立面**。库赛特可汗 **蒙楚格** 允诺雇佣；阿塞莱苏丹 **安奇德** 为银与荣耀放行年轻英雄 **尼姆尔·班努·萨兰（Nimr Banu Sarran）** 等。

战斗重构（综合领主口述）：  
- **林道伏击**：内雷泽斯令前军（含青年军官 **加里奥斯·科姆诺斯**、阿塞莱矛投兵）抢关；**卡拉多** 号角一响，菲安箭雨从两侧林中倾泻；阿塞莱举盾反击后镰刀斧手收割，前军崩溃；加里奥斯等少数遁入林中听头颅作奖品的呼啸。  
- **谷地骑战**：尼姆尔追击过猛被拉散；瓦兰迪亚骑士（如年轻的 **因加尔瑟·德·科尔坦 Ingalther dey Cortain**、**阿尔德里克 Aldric** 等）冲散阿塞莱；库赛特骑射（**克尔吉特** 部 **索伦 Noyan Solun** 等）射马反制；瓦兰迪亚弩手（**温瑟里·德·阿罗芒** 等按德瑟特部署）将骑射压回，索伦阵亡，克尔吉特男丁近乎灭族级损失（遗孀 **梅苏伊 Mesui** 一系叙事极重）；帝国甲骑再冲，双方俱伤。  
- **主阵与营垒**：斯特吉亚大公 **瓦丁斯拉夫（Vadinslav）** 对帝国主阵盾墙推进；帝国退入加固营垒投矛石与「咒火」；斯特吉亚架梯强攻，瓦丁斯拉夫死于城头锤击；**奥列克老人（Olek the Old）** 冲破缺口，自内雷泽斯尸旁夺下**龙旗**。**阿伦尼科斯·佩特罗斯** 收拢御林残部（含青年参谋 **吕松·奥斯蒂科斯**）自后方撤出——帝国唯一成建制退兵。  

战后：瓦丁斯拉夫之子 **拉甘瓦德（Raganvad）** 赶到索旗；奥列克老人当众折断旗杆掷还——斯特吉亚内部裂痕与「巴旦尼亚只顾割头不冲主阵」的骂战同步爆发；德瑟特自认「几乎像战败的胜利」；卡拉多称「受祝福之战」；加里奥斯认定「老东西不配再领兵」；阿塞莱侧 **尼姆尔** 归国炫耀却因与基尔德氏族女儿私情被**班努·基尔德**公开处死，萨兰—基尔德世仇固化；库赛特克尔吉特部衰弱被其他氏族吞食。**龙旗碎片**散落匪窝与传说，成为数十年后玩家主线的钥匙。

世界状态（战后）：帝国军事神话破裂；三边蛮族/王国各怀怨；阿伦尼科斯因救残军与稳定局势被元老院推为新帝。

③ **阿伦尼科斯短暂中兴与 1084 前后遇刺**  
阿伦尼科斯（佩特罗斯氏族）以务实稳住帝国，延缓崩解，娶达尔士系出身的 **拉盖娅**，育女 **伊拉（Ira）**。他死后（游戏文本为**可疑谋杀**，凶手无官方唯一答案），法统真空瞬间爆炸：  
- **拉盖娅·佩特罗斯**：亡夫遗孀，主张**血脉继承**，为己与女儿伊拉争冠冕；要强君终结无尽内战。  
- **吕松·奥斯蒂科斯**：元老贵族，主张**元老院选举皇帝**，认为失控强君才是帝国之病。  
- **加里奥斯·科姆诺斯**：军中英雄（彭德莱克幸存者），士兵拥立，主张**军队选帝**、给老兵土地，以武力说话。  

于是卡拉迪克帝国裂为**北帝国 / 南帝国 / 西帝国**，三帝互不承认，同文化同兵种，打的是**继承权内战**；周边五王国趁火打劫，吞边城、收附庸、卖雇佣。

④ **开局时点：帝国三分已成、列国战争常态（约 1084–1087 叙事窗口）**  
玩家创建角色与**氏族**，自带少量亲属与初始技能。世界状态：  
- 三帝国在中部与湖海走廊互相撕咬；  
- 瓦兰迪亚西扩与帝国西境摩擦；  
- 巴旦尼亚据林自重，与斯特吉亚旧怨未消；  
- 斯特吉亚大公拉甘瓦德权威受杯葛，湖匪与森林民活跃；  
- 库赛特向西压牧与劫掠；  
- 阿塞莱控商路与绿洲税，部落与苏丹宫廷博弈。  
早期微观：清匪、护送商队、比武、酒馆招**流浪者（Wanderer）同伴**、买第一间**工坊**、攒第一笔赎金。

⑤ **主线骨架：调查「内雷泽斯的愚行」→ 拼合龙旗**  
自动任务链（可推迟但无法从设定中抹去其历史重量）：  
1. 向约十余位亲历/听闻彭德莱克的贵族问询，集满足够口述（约 10 人即可推进，其余锁定以保留视角差异）。  
2. 前往 **Marunath** 见 **阿尔扎戈斯（Arzagos）**——反帝国、要帝国永碎的密谋者；前往 **Epicrotea** 见 **伊斯蒂亚娜（Istiana）**——亲帝国、要龙旗重铸法统的女策士。  
3. **拼合龙旗（Assemble the Dragon Banner）**：按两人指引清剿指定匪窝，夺回旗面碎片并组装。  
4. 之后分岔：扶**帝国重建**（伊斯蒂亚娜路线）或**摧毁帝国法统、扶外族/自立**（阿尔扎戈斯路线）；再进入王国级「阴谋 / 联盟」阶段——其他大势力会因你举旗而合纵连横。  
注意：沙盒允许**完全无视主线**只做商人/佣兵/一国之主；但龙旗一旦拼合，政治桌面的赌注会抬高。

⑥ **氏族崛起：影响、政策、婚姻、采邑**  
- 氏族升级：更多同伴位、工坊上限、军团指挥权。  
- **影响力**：胜仗、完成领主任务、政策与魅力相关专长产出；用于提案、否决、强分城、开军团。  
- **政策样例**（写政治戏可用）：神圣君威（Sacred Majesty）抬高统治者日影响力；王室卫队（Royal Guard）扩君王部队；元老院/贵族枢密抬高高层氏族；农奴制/债务赦免等改忠诚与产出；国家专营损工坊利君税。  
- **婚姻与子嗣**：与 **伊拉、科瑞恩（Corein）** 等贵族联姻可绑势力；子女可继承。  
- **附庸与叛变**：关系、文化、采邑分配决定领主是否倒戈；处决俘虏改变恐怖与忠诚。

⑦ **战争沙盒：野战、围城、灭国与重建**  
常见战役形态：  
- 野战：骑兵冲散、盾墙推进、骑射风筝、菲安箭雨、甲骑对冲。  
- 围城：先轰墙开缺口→云梯/攻城塔/冲车→城门与巷战；工程技能与攻城器械质量决定节奏。  
- 军团（Army）：多氏族党联合编，消耗影响力维持；缺粮即溃。  
- 叛乱：错误文化总督、低忠诚、苛政会引爆城镇起义。  
名场面应写：**旗帜、号角、泥里的断矛、城墙上的投石机、酒馆里庆祝或哭丧的老兵**——不要写魔法光柱。

⑧ **常见玩家路径（非唯一 canon，但是 AI 最常需要的「阶段剧本」）**  
A. **佣兵—附庸—大领主**：先给任一王国打工攒影响力与关系，再宣誓获封，逐步吞邻国。  
B. **南帝国扶龙**：娶伊拉/效忠拉盖娅，以血脉法统统一三帝国，再外打五王国。  
C. **元老院法统**：扶吕松，玩寡头政治与政策战。  
D. **军选帝国**：投加里奥斯，老兵土地与武力统一。  
E. **外族灭帝**：瓦兰迪亚骑士团式西征、库赛特西牧、阿塞莱北上商路帝国、巴旦尼亚林海独立、斯特吉亚雪国一统。  
F. **龙旗自立**：拼旗后自建王国，用阿尔扎戈斯/伊斯蒂亚娜之一作意识形态外壳，招流浪者封臣。  
G. **纯经济**：工坊垄断 + 商队 + 铁匠，战争只当贸易工具。  

⑨ **终局形态（沙盒，无强制电影尾）**  
- 三帝之一重铸帝国；  
- 外族王国吞尽帝国残疆；  
- 玩家旗帜插满卡拉迪亚；  
- 或永恒混战：每十年一次改朝换代。  
「谁活谁死」由战争与处决决定：canon 开局活人可在沙盒中死绝；写档案时保留**开局默认存活**的统治者名单，不预设玩家必杀。

⑩ **微观贯穿线（每回合都可写）**  
氏族宴会与比武；村庄征兵与民兵；粮食价格与饥荒；土匪刷新；同伴性格互憎；工坊被战火或政策打穿；贵族任务（训练、追匪、解决纠纷、走私）；竞技场角斗（伊拉曾以贵族之身下场的传闻可作性格注脚）。

⑪ **蝴蝶效应边界**  
可改：谁统一、哪国先灭、哪位君主被俘斩、龙旗归属、玩家是否称帝。  
不可改成：出现魔法灭世、把中世纪兵种写成现代热武器、把「深渊」超凡体系硬塞进卡拉迪亚（本世界无此设定）。兵种与文化气质应保持：菲安仍是林弓，库赛特仍是骑射，帝国仍是甲骑与纪律步。

- **南帝国（Pethros）**：血脉法统；地盘南卡拉迪亚帝国故地；代表拉盖娅、伊拉；战力帝国兵种；敌北/西帝国，视外族为蛮。  
- **北帝国（Osticos）**：元老法统；代表吕松；政策与贵族政治戏重。  
- **西帝国（Comnos）**：军选法统；代表加里奥斯；老兵与边军气质。  
- **瓦兰迪亚**：封建骑士国家；德瑟特；西海岸；重骑破阵；与帝国西境、巴旦尼亚摩擦。  
- **巴旦尼亚**：山林王国；卡拉多；伏击与长弓；对帝国世仇叙事强。  
- **斯特吉亚**：雪林公国；拉甘瓦德；盾墙斧兵；内有波雅尔杯葛。

- **阿塞莱**：沙漠苏丹国；安奇德；商路与绿洲；氏族荣誉与世仇。

**【贵重物品】**
- **龙旗（Dragon Banner）及其碎片**：帝国皇冠级象征物；彭德莱克后破碎流落匪窝；拼合后用于建国立业与法统宣称。**非魔法灭国**，是政治核按钮。下落：主线任务可取。  
- **卡拉迪克冠冕 / 帝位法统**：抽象权柄，比任何单件武器更贵；三帝争夺的真正目标。  
- **优质战马与甲骑装备**：帝国甲骑甲、瓦兰迪亚骑士枪马、库赛特良马——野战质变。  
- **菲安级长弓与帝国复合弓**：远程质变装备。  
- **工坊所有权文书**：城镇内生产设施，被动银币来源；战争与政策可毁掉收益。  
- **采邑敕封（城/堡/村）**：权势本身；比任何单件神器重要。  
- **领主赎金契约**：活人比死人贵；早期资本原始积累核心。  
- **铁匠传奇武器（玩家打造）**：游戏机制允许锻造高数值武器，叙事上应写成「名匠之作」而非神器降世。  
- **文化特色战利品**：巴旦尼亚头颅战利叙事、斯特吉亚斧、阿塞莱香料与马具、库赛特马饰。  
禁止编造原作没有的「灭世魔剑」；稀有以历史名物、优质套装、采邑与龙旗为顶。

**【隐藏剧情 · 伏笔】**
- **彭德莱克多视角口述**：同一战役十几种真话与假话；契约者调查时应允许矛盾并存，勿写成单一官方战报。  
- **埃里尔之死**：卡拉多杀君与否是巴旦尼亚政治幽灵。  
- **阿伦尼科斯凶手**：无唯一 canon 答案；三帝皆可被政敌指控——适合阴谋线。  
- **龙旗双顾问**：伊斯蒂亚娜 vs 阿尔扎戈斯是意识形态分岔，不是简单好坏。  
- **萨兰—基尔德仇、克尔吉特衰亡、斯特吉亚功臣辱君**：开局几十年前的血仍在流。  
- **与《战团》互文**：约 173 年后的列国（斯瓦迪亚、诺德、库吉特等）是本时代势力演化的后代；写霸主正文勿剧透战团具体国王，可暗示「帝国碎后还会再碎」。  
- **War Sails / 诺德海上线**：属后续 DLC 扩展层；基线档案以大陆八势力为主，海上维京系可作为边角威胁而非必写主轴。  
- **机制即叙事**：影响力不足开不起军团、粮食崩盘、忠诚叛乱——比「幕后大魔王」更卡拉迪亚。

**【大事记时间线】**
远古—数百年前 → 卡拉迪克帝国扩张，边军与外族编入；龙旗成为法统符号。  
约 1077 → **彭德莱克之战**：内雷泽斯战死；瓦丁斯拉夫死；龙旗被夺后破碎；阿伦尼科斯突围。  
1077 后数年 → 阿伦尼科斯称帝，短暂中兴；拉盖娅成后，伊拉出生并成长；加里奥斯、吕松积累资历。  
约 1084 → 阿伦尼科斯遇刺；帝国三分：吕松（北）、拉盖娅（南）、加里奥斯（西）。  
开局窗口（约 1084–1087） → 玩家氏族创建；列国战争常态；匪帮与雇佣兵活跃。  
玩家早期 → 清匪、商队、工坊、酒馆同伴；调查内雷泽斯愚行。  
玩家中期 → 见伊斯蒂亚娜与阿尔扎戈斯；拼合龙旗；宣誓/自立；围城夺第一座城。  
玩家后期 → 军团灭国战；三帝存亡见分晓；政策与叛乱并至。  
终局沙盘 → 统一 / 新帝国 / 外族霸权 / 持续混战；约 173 年后的《战团》格局在此埋根。

**【叙事基调 · 雷区】**
画风：泥与血的中世纪旗帜史诗；可残酷可热血，但保持**凡人军事逻辑**；对话可带各族口音与荣誉观（帝国修辞、草原直白、沙漠诗意、林人诅咒、雪国冷硬）。  
忌写成高魔奇幻或武侠内力（无飞剑、无法术屠城）。  
忌把三帝写成卡通脸谱：每人皆有自洽法统（血脉 / 元老 / 军选）。  
忌忽略后勤：无粮无饷的「百万雄师」是 OOC。  
忌低阶发「统一全国」奖励或开局单挑君主满编军团。  
忌用现代民族国家逻辑抹平氏族私斗；卡拉迪亚先是**氏族**，后是王国。  
忌把龙旗写成魔法神器秒杀；它是称王的政治钥匙。  
最早可切入锚点：开局创建氏族后第一周——某城酒馆招募、城外匪窝、或帝国边境第一次遭遇战。

## 阶位切入点

> 阶位↔战力：一阶≈村民新兵匪帮；二阶≈精锐士兵与小氏族战团；三阶≈大领主军团与名城攻防；四阶≈国王/三帝主力与灭国战役；五阶≈统一帝国或取代帝国的军事霸权。世界顶点（重建/统一卡拉迪亚的霸主）为**五阶凡人军事巅峰**，无超阶；低阶切入一律「规避满编君主军·情报与站队优先」。

**一阶（新兵·匪患·商路 · 落地求生线）**
切入身份/时点：契约者以「初建氏族的流浪武者 / 小商队护卫 / 刚离村的应征新兵」身份切入，锚定游戏开局、龙旗尚未拼合、三帝国内战已打响但玩家尚未被大贵族点名之时。
初始事件：在瓦兰迪亚或帝国边境某镇外，一支商队遭山贼（或森林兄弟会/逃兵）拦截；守军来迟，契约者必须在「拼死护车换赏金」「趁乱抢货」「活捉匪首换贵族好感」中抉择——第一笔银币与第一场关系账同时落下。
开场白建议：「卡拉迪亚的土路上，旗帜比法律多。你刚把氏族的破旗插上营地，远处商队的铃铛就乱了——林子里伸出的不是友谊，是矛尖。你手里只有一把还算趁手的武器，和几个比你更怕死的追随者。」
关键NPC立场：**德瑟特**——远在加伦德的国王，此时只是悬赏榜上的名字，对小人物无感除非你立功；**拉盖娅 / 吕松 / 加里奥斯**——三帝皆在远方撕咬，不会为你停战；酒馆**流浪者（如 Bloodaxe、the Red）**——要钱卖命，可招；地方**氏族男爵/波雅尔**——给清匪任务，完成后记一笔人情；匪首——早期可杀可俘。
主线钩子/支线：第一次清匪与商队护送；竞技场挣第一桶金；买下第一间**工坊**；打听彭德莱克老兵的酒话（为龙旗线铺垫）。支线勿复用高阶「灭国」句：本阶是粮、工资、士气。
危险度/规避：中低——满编领主党可碾死你；规避任何国王亲征军与大型军团，被俘交赎金即可，勿硬刚。
任务方向/奖励：活过第一季、攒到正规编制；奖励取向＝银币、劣中质武器马匹、初级同伴、工坊、对某男爵的关系。

**二阶（精锐战团·附庸门槛 · 旗帜初响线）**
切入身份/时点：以「成名佣兵团长 / 小贵族氏族族长」切入，锚定已能野战吃掉中等匪帮、开始被王国征召为雇佣兵之时。
初始事件：某王国（瓦兰迪亚或南帝国常见）的领主发来**雇佣合同**：随军团攻打敌方城堡外围。战场上你的精锐第一次和帝国甲骑或瓦兰迪亚骑士同列——战后要在「续约换钱」「宣誓效忠换未来采邑」「临阵倒戈」间选择。
开场白建议：「你的人不再一碰就散。城墙上的守军开始用名字叫你——那是好事，也是诅咒。今晚军帐里，一份雇佣契约和一柄象征效忠的剑摆在同一张桌上。」
关键NPC立场：**加里奥斯**——爱勇士，雇佣转效忠概率高；**拉盖娅**——要能打仗的附庸护伊拉的未来；**卡拉多**——林中法则，强者可谈；**安奇德**——商路利益可买；**伊斯蒂亚娜 / 阿尔扎戈斯**——若已启动调查，开始在暗处打量你是否配碰龙旗。
主线钩子/支线：雇佣→宣誓；追击敌方小领主；婚姻线启动（接触**伊拉**或**科瑞恩**需关系）；继续收集彭德莱克口述。支线：走私、解决村庄纠纷、铁匠单。
危险度/规避：中——可打赢中小野战，仍须规避多氏族军团围殴；勿在无攻城器械时强攻名城。
任务方向/奖励：拿到第一块封地或稳定雇佣金；奖励取向＝精锐兵模版、中质甲骑装、魅力/统御技能突破、氏族升级。

**三阶（大领主·围城者 · 龙旗拼合线）**
切入身份/时点：以「王国重要附庸 / 独立大领主」切入，锚定已能指挥多队、参与名城攻防，并推进至**拼合龙旗**前后。
初始事件：你按指引攻入某大型匪巢夺回龙旗碎片；出巢时，帝国与某王国的侦骑都在——龙旗消息走漏，**伊斯蒂亚娜**与**阿尔扎戈斯**同时派人传话：站帝国法统，或站碎帝之路。
开场白建议：「旗面在火光里拼合的时候，帐篷外的马嘶突然静了。你知道从今夜起，卡拉迪亚没有人会再把你只当成一个运气好的佣兵——三座帝国的文书吏已经在写你的名字。」
关键NPC立场：**伊斯蒂亚娜**——给你帝国重建剧本与贵族门路；**阿尔扎戈斯**——给你反帝联盟与地下刀；**吕松**——问你是否尊重元老院；**拉盖娅**——问你是否愿护血脉；**加里奥斯**——问你是否敢为士卒夺地；**蒙楚格 / 拉甘瓦德**——视你为可雇可灭的新变量。
主线钩子/支线：龙旗拼合；第一次指挥攻城器械破墙；在王国议会用**影响力**抢采邑；镇压低忠诚城镇。支线：小派系雇佣（湖鼠/吉尔曼等）改变兵种结构。
危险度/规避：高——你已是暗杀、联军与叛变目标；规避同时对三帝国宣战；粮道被切即崩。
任务方向/奖励：合法称王或成为王国柱石；奖励取向＝高级兵种满编、名城治理权、龙旗政治资本、顶级同伴封臣化。

**四阶（君主级·灭国战役 · 三帝存亡线）**
切入身份/时点：以「一国之王 / 三帝之一的首席军阀 / 重建中的帝国元帅」切入，锚定大陆进入连续灭国战争、至少一位开局君主已陨落或被俘之时。
初始事件：你的军团兵临某帝国主城（Lycaron / Diathma / Jalmarys 之一）；城内使者提出条件——劝降某帝、联姻、或献出龙旗残党名单。城外粮草只够旬日，城内可能发生贵族开门或死战。
开场白建议：「投石机的臂杆每次甩下，城砖就少一块。你听得到墙后的帝国号角——那曾是统治卡拉迪亚的声音。如今它在问你：要皇冠，还是要焦土？」
关键NPC立场：**拉盖娅**——可降可死，降则血脉法统可被你吞并利用；**吕松**——宁讲条件不讲膝；**加里奥斯**——倾向城下决战；**德瑟特 / 卡拉多 / 蒙楚格** 等——或趁火打劫你的后方，或索要战后分赃；**伊拉 / 科瑞恩**——若已联姻则成内政与军事双刃。
主线钩子/支线：三帝内战终局战；第一次真正「灭一国」；政策堆叠（君威 vs 贵族反弹）；处理功臣争赏。支线：安置移民、改文化总督、防叛乱。
危险度/规避：极高——多线开战即崩；规避无盟友情况下的两线灭国；注意军团影响力耗尽。
任务方向/奖励：皇冠级权柄、广袤采邑再分配权；奖励取向＝王国级政策主导、精锐禁卫编制、战败国宝库与名马。

**五阶（霸主·统一 / 新秩序 · 卡拉迪亚终局线）**
切入身份/时点：以「卡拉迪亚实际霸主 / 重建卡拉迪克帝国的皇帝 / 外族征服王朝的最高汗·王·苏丹」切入，锚定地图大半同色、仅余零星抵抗与海外/边角威胁之时。
初始事件：最后一座敌对王都谈判破裂或投降；你在龙旗（或你的新王旗）下举行献俘与册封——旧氏族跪拜或流亡，新政策一纸改变半个大陆税率与征兵。与此同时，边境传来新匪帮或海上掠袭者消息：统一不是休息，是更大的治安与继承问题。
开场白建议：「从雪林到纳哈萨，斥候的回报第一次用同一种颜色画地图。你把龙旗插在殿门时，听见的不是神话，是书记官问你：明天的税，按帝国旧律，还是按你的剑？」
关键NPC立场：幸存的**开局君主**——阶下囚、附庸或坟茔；**伊斯蒂亚娜**——若帝国重建成功则成理论家，失败则流亡策反；**阿尔扎戈斯**——若帝国被碎则称目标达成，转而防你成为新帝；各国氏族——在「归顺 / 叛乱 / 联姻」三态间摇摆；你的**子女继承人**——五阶真正的软肋是继承而非单挑。
主线钩子/支线：法统典礼与大赦/大清洗二选一；最后叛乱与忠诚危机；为《战团》时代埋下的制度种子（分封过碎或集权过猛）。支线：海上商路、边境屯垦、竞技场庆典稳民心。
危险度/规避：贴近顶点——世界内已无更高军事阶位；威胁来自叛乱、继承、经济崩溃与多线起义，而非「更强的魔王」。不存在超阶；禁止用封印削弱解释任何君主。
任务方向/奖励：大陆霸权叙事结算、王朝法统、传奇兵器与禁卫番号；奖励取向＝统治级人脉与制度、传说级装备（仍凡人）、史诗称号——**不发超凡法术**。
