## 剧情

**【作品来源】**
《眷思量》（外文名 The Island of Siliang，别名《眷思量之烟霞海客》）是由赵禹晴执导、编剧，炎央文化与企鹅影视联合出品的原创国漫网络动画。第一季于2021年6月14日在腾讯视频独播，全15集；第二季《眷思量之风烟迭起》于2024年6月1日上线，全13集。类型为奇幻、冒险、情感与权谋交织的新中式玄幻。作品以异界仙岛「思量岛」为舞台，将神族流放结界、家族权斗、青梅竹马与凡人寻仙线熔于一炉，美术走工笔服饰与荧光晕染的新中式路线，气质偏「瑰丽阴谋 + 成长救赎 + 暗恋守护」。曾获华语电影短片大赛最佳动画短片、中国网络视频学院奖最佳网络动画、腾讯金鹅荣誉年度原创动画等。本稿人物与主线以公开百科与动画剧情为准，不以小说同名作品替代。

**【世界定位】**
这是一个「被神族结界锁死的流放仙岛」世界：神、仙、人分界而居，思量岛位于神界与天界交界处，专囚犯错神仙及其后裔；岛民以凡人寿数尝生老病死，海水沾身可令谪仙灰飞烟灭。主角线以天机神族少年镜玄（幼名觞小福）探寻父母觞阵、璎陌「出岛惨案」真相为轴，凡人少女屠丽（丽娘）为完成养父母离岛遗愿造船出海，南梁皇族凡人萧霁误入岛屿搅动风云，三线纠缠揭开「天赎」考验、家族合谋与逃出生天的秘密。一句话：在锁岛结界里，查案、恋爱、权谋与破界同时发生。

**【世界观 · 力量体系】**
宇宙生灵以神、仙、人为各界代表，分居不同时空。神族掌管世间运行法则；仙家有血脉与法术传承；凡人无先天神力，却可凭机缘、修习与「岛上物质」积累获得异于常人的体魄。思量岛是神族为惩罚犯错神、仙所建的流放之地，结界封锁，谪仙沾海水即有灰飞烟灭之危，且以凡人寿数在岛上受苦。传说每五百年降临一次「天赎」：被困神仙或其后人通过考验可重返天界；亦有人不等天赎，铤而走险破除结界逃出生天。

力量与身份大致分层（原作无严格修仙数字境界，以下按公开能力表现归纳，供乐园映射，宁低勿高）：
1. **凡人层**：无神力或仅有体魄优势。代表：初登岛的萧霁一行、未吸纳岛力的普通外人。屠丽虽为凡人，因自幼食用岛上水粮，力气奇大并具备修仙基础，介于凡人与初修之间。
2. **仙家后裔·初入**：能驱使基础法术、操控家族器物、参与望神节竞技与出海护卫。代表：少年期程炫、须灵犀等家族子弟。
3. **仙家骨干·岛内战力**：可独立对抗海兽、傀儡守卫、参与议事堂武力威慑；家族公子程炜、得力家丁与护卫属此层。
4. **长老级·岛内顶点之下**：长老会成员及其等价存在。须莫海（须家）、程灼（程家）等以家族底蕴与长年修为坐镇；程染（天机神族赘婿）温润有礼但线索缠身，战力不可小觑。
5. **神族后裔·高天资**：镜玄为天机神族后裔，寒沁剑认主，水系法术呈冰晶状，可幻化布匹装扮、假扮恒凌虚，战力远超同龄，后期身份暴露时足以搅动全岛。
6. **上古神兽·岛内近顶点**：奉眠，灵凰血脉，寿命漫长，容颜驻妙龄，长老会三大支柱之一，布局出岛，实力深不可测，岛众忌惮。
7. **破界/天赎级（世界顶点方向）**：当年觞阵强开法阵试图送亲离岛失败酿成惨案；真正破除结界、通过天赎或对抗南溟玄武级镇海存在，属于全岛博弈与超凡冲突的顶点档。

**乐园阶位映射（最高五阶，宁低勿高）**：凡人与初入岛民≈一阶；仙家后裔初入与家族年轻弟子≈二阶；家族骨干、镜玄成长中期、屠丽体魄+手工战力≈三阶；长老会主力、程灼/须莫海级≈四阶；奉眠（灵凰）、神族高阶与破界/天赎级冲突≈五阶顶点。严禁以「遭禁制/条件受限/实力未减·仅策略规避/不可越阶硬刚/乐园规则外强制降权」解释顶点——顶点始终是**存在·情报优先/条件性胜利**。

**【地理 · 舞台】**
- **思量岛**：全剧核心舞台。被神族结界封锁的流放仙岛，岛上仙家后裔聚族而居，以长老会治理，设议事堂、藏典阁、望神节广场集市等公共空间。海水为致命禁忌线，出海需特殊船只与航路知识。
- **恒水居**：南方朱雀-柳宿恒凌虚一脉居所。恒凌虚故去后，镜玄假扮「恒老」在此收养屠丽；第二季镜玄以真身入住，与屠丽共同生活。清修、制药、花草氛围，实为镜玄藏线与屠丽造船的大本营。
- **藏典阁**：岛上禁地档案库，存案宗与法宝。镜玄夜探触发傀儡与爆炸，震惊全岛，是查案线的起点。
- **议事堂**：长老会开会处。须、程、奉三大势力各怀算盘，隐瞒藏典阁真相，以望神节奖品转移视线。
- **殇家/觞阵旧居与火场遗迹**：十年前惨案现场，屠丽曾回去打捞养父母遗物（衣料、手工痕迹），万象方等关键道具与此关联。
- **外海·暗礁与沉船区**：屠丽打渔、救萧霁、藏凡人水手之处；沉船勾起儿时记忆；手串触发海水异动。
- **南溟海域**：镇守思量岛外海的南溟玄武出没，查船队曾遭危难，屠丽奋战、镜玄出手共斩巨兽。
- **焱洞**：程灼侧可疑活动地点，镜玄调查线深入后触及。
- **凡间·南梁**：萧霁出身地。萧氏皇族，萧霁奉皇命出海寻仙，为救质子父母建功立业，经虫洞/风暴误入思量岛。
- **神界/天界（背景层）**：结界外的「生天」方向，天赎归途与破界目标，岛民终生执念。

**【世界剧情线】**

**前史：流放岛与天赎传说**
自宇宙有生灵，神、仙、人分界而居。神族建思量岛流放犯错神、仙，结界锁海。岛上谪仙以凡人寿数受苦，每五百年一次天赎考验可重返天界；亦有人选择破界逃亡。岛上仙家后裔按星宿家族聚居，形成以须家（北方玄武-女宿，女尊）、程家（西方白虎-觜宿，火性冶炼）、恒家（南方朱雀-柳宿，清修制药）与外来长老奉眠（灵凰）为核心的权力结构。

**前史·殇家惨案（约十年前）**
天机神族觞阵与戮神神族璎陌在岛上成家，育有子觞小福（后名镜玄）。海盗屠霍携幼女屠丽误入岛屿，夫妇收养屠丽，与小福共同成长。璎陌身体出现异样，觞阵为送亲人离岛，触犯禁忌强开法阵。布阵代价惨烈，璎陌投海，觞阵与幼子亦坠海；幼子被误入岛上的奉眠所救，抹去「殇家人」身份，赐名镜玄，后以恒水居主人恒凌虚身份掩护生活，并收养屠丽。岛上公开叙事将惨案归咎于出岛失败，程灼等人则视屠丽为「根源隐患」。万象方为觞阵研究的魔方式装置（木金混合，嵌欧珀石），儿时小镜玄与丽娘曾当玩具，实为出岛/阵法研究关键。

**第一季前期：藏典阁爆炸与望神节玉牌（约对应第1–5集脉络）**
十年后深夜，神秘少年镜玄潜入藏典阁查案宗，触发机关与傀儡，引发爆炸，全岛震动。长老会须老、程老、奉老互相猜忌。屠丽是岛上唯一凡人，被恒老收留，靠出海打渔为生，正筹备离岛；她潜入议事堂偷听，被发现后得程染说情脱身，回恒水居未见「恒老」房内血迹与异状。

镜玄被奉眠拦阻，仍坚持查父母被害真相。长老会隐瞒藏典阁真相，将阁中法宝（含万象方）作望神节奖品转移视线。屠丽为夺回养父母遗物须得优先选奖的玉牌，与程炜争夺；天秤选中她的海鱼而非程家火芝，文书偏袒程家，逼她打来两丈大鱼。屠丽出海搏鱼，坠入沉船勾起儿时记忆，捡到手串引发海水震动，救起落海的南梁小侯爷萧霁，将其与水手藏于暗礁并供给粮水。她以大鱼赢下玉牌；程炜欲半路抢牌被程染喝止。程灼命程染盯紧屠丽，认定她是十年前惨案根源。

镜玄在恒水居查阵法线索，假扮恒老；奉眠传话警告勿再私行。镜玄夜入屠丽房为其疗伤，从她做的小玩意中发现解读案件的线索。屠丽不知「上仙镜玄」即儿时小福哥哥，亦不知「恒老」实为镜玄所扮。

**第一季中后期：凡人登岛、查船与南溟玄武（公开脉络归纳）**
萧霁谎称暴风雨迷路，实则奉皇命寻仙，为救质子父母建功。岛民探究凡人登岛秘密，派须隐带头查船，遭遇镇守外海的南溟玄武。屠丽奋勇对抗，镜玄出手相救，并肩斩杀玄武巨兽。须隐视镜玄为救命恩人请至岛上；庭审时镜玄思路敏捷瞒过众人，仍回恒水居。萧霁与须灵犀线并行发展（官配），须灵犀外表高岭、内心渴望自由；须灵珑（真相弟）童言无忌常戳破大人伪装。程炜跋扈、程炫温润，兄弟对照；程染作为赘婿口碑佳，却在镜玄调查中多次现身于旧案线索。

主线在「查案—藏身份—望神节遗物—凡人搅局—海战」中推进：镜玄对须、程两家均持怀疑；奉眠既保护又约束，似有利用其出岛的布局；屠丽造船、手工、航海尺与大平衡舟成为离岛物质准备。第一季末公开走向包括身份危机加深、真相碎片拼合、大战前夜的氛围（具体集数细节以动画为准）。

**第一季尾声至第二季衔接：身份裂缝**
镜玄真实身份逐渐被程灼等察觉；屠丽仅心存疑虑，未尽知全部真相。镜玄希望查明真相并保护屠丽，甚至为陪伴凡人屠丽考虑留岛，而屠丽仍执着完成养父母遗愿去看外面世界——情感与目标第一次结构性冲突。万象方、案宗标号、焱洞可疑活动成为下一阶段钥匙。

**第二季《风烟迭起》：同居三年后的风烟**
公开资料称第二季中镜玄与屠丽关系更亲密，已同居约三年；镜玄借助奉眠等人帮助以真实身份入住恒水居。须、程两家已知其身份；程灼当面拆穿。剧情重心转向：身份暴露后的全岛站队、家族权斗升级、出岛/天赎路线白热化、萧霁—须灵犀情感与凡仙身份张力、奉眠出岛之约的真相逼近。公开预告与词条摘要提及镜玄身份暴露后纵身跃崖、生死未卜，大战将启——契约者切入时可将此作为「高阶蝴蝶节点」，但勿写成顶点被废战力。第二季因预算与时长对部分情节有压缩，晦涩理论概念被简化，叙事更通俗，但核心仍是「阴谋与救赎、挣脱枷锁」。

**可玩主线结构（供乐园展开，忠于设定不编造未公开结局细目）**
1. **查案主线**：藏典阁案宗 → 万象方与阵法 → 殇家惨案责任链 → 须/程/奉谁利用了觞阵 → 焱洞与程灼 → 奉眠真实交易。
2. **离岛主线**：屠丽造船与航路 → 手串/沉船记忆 → 海水禁忌破解线索 → 天赎 vs 破界两路线选择。
3. **身份主线**：镜玄恒老伪装 → 上仙身份公开 → 屠丽知情与否 → 跃崖节点后的救援/追查。
4. **凡人皇命线**：萧霁寻仙救亲 → 与须灵犀 → 南梁与岛上利益交换或冲突。
5. **家族权斗线**：望神节资源、玉牌、火芝、冶炼军备、女尊须家规矩 vs 自由渴望。

**结局方向（公开信息边界）**
第一季完结于「真相未尽、风波已起」；第二季推进至身份暴露与大战将启。完整大结局以动画后续为准；世界书不编造未播出终局谁死谁活，但可写明世界终态的两种张力：要么通过天赎/破界让部分人「生天」，要么结界仍在、岛民继续以凡人寿数轮回执念。契约者介入应推动「可观察的因果改变」，而非宣告原作未写死的终局。

**【主要人物】**

- **镜玄（男主·天机神族后裔｜幼名觞小福｜化名恒凌虚）**｜性格：清冷寡言、老成隐忍，外冷内炽，执着真相与守护。｜装备·能力：寒沁剑（神族法器，启动态蓝宝石剑身，隐藏态半透明苍蓝雪纱；可随主属性变化）；水系法术呈冰晶状；法力变幻布匹秒切装扮；高天资自学阵法与案宗解读。｜人物弧光：被宠爱的神族幼子 → 惨案幸存者被奉眠改名庇护 → 假扮恒老收养屠丽并暗恋守护 → 上仙身份渐露 → 真身同居与身份暴露 → 跃崖级冲突节点。｜立场关系：爱屠丽；敬畏又防备奉眠；怀疑须、程；对萧霁警惕；与程灼对立加深。

- **屠丽 / 丽娘（女主·凡人）**｜性格：开朗灵巧、独立坚强、重情重义、爱憎分明，手工狂人与技术宅。｜装备·能力：大平衡舟、小平衡舟、航海尺；养父所授手工；因吸纳岛上物质力气过人并具修仙基础；肚兜纹样源自生父遗物「五星出东方利中国」织锦意象。｜人物弧光：海盗之女误入岛 → 觞阵璎陌养女 → 目睹养父母坠崖 → 恒水居长大 → 秘密造船离岛 → 救萧霁、夺玉牌、战玄武 → 与「上仙」重逢而不识小福。｜立场关系：官配镜玄；感激程染说情；与程炜对立；视须灵珑为亲近后辈；对长老会戒备。

- **萧霁（南梁萧氏皇族·凡人小侯爷）**｜性格：活泼乐观、玩世不恭外表下有建功救亲的决心。｜装备·能力：骑射与凡间武艺；随从斌叔等；无神力，靠智计与人脉。｜人物弧光：奉皇命出海寻仙 → 风暴/虫洞误入 → 被屠丽所救 → 搅动岛上政治 → 与须灵犀官配发展。｜立场关系：依赖屠丽援助；被镜玄警惕；倾心须灵犀。

- **须灵犀（仙家后裔·北方玄武女宿·须家大小姐）**｜性格：外表高傲如带刺玫瑰，卸下伪装是渴望自由时光的「傻姑娘」。｜装备·能力：琴棋书画女工习字舞等世家教养；家族法术与规矩束缚并存。｜人物弧光：被安排一生的女尊继承人 → 遇见萧霁与岛外风波 → 寻求自我。｜立场关系：萧霁官配；须灵珑之姐；父母须隐、须妍，祖母须莫海。

- **奉眠（上古神兽灵凰·长老会三大支柱之一）**｜性格：高深莫测、波澜不惊，一切部署为早日出岛恢复自由。｜装备·能力：灵凰血脉，寿命漫长，实力深不可测；曾执教，以条件交换登长老位。｜人物弧光：误入思量岛被困 → 救镜玄并改名抹身份 → 约束其查案同时布局出岛 → 与程老暗斗。｜立场关系：镜玄的师父/养育者，亦可能利用之；岛众敬称「奉老」。

- **程染（天机神族·程家赘婿）**｜性格：温润有礼、处事得当，口碑佳，心思未必尽显。｜装备·能力：神族底蕴；访古探史、制香；多次出现在旧案线索中。｜人物弧光：入赘程家调和火性门风 → 说情屠丽 → 被程灼驱使盯梢 → 真相漩涡中的灰色枢纽。｜立场关系：程炜程炫之父辈角色；夹在程家与镜玄调查之间。

**【势力图谱】**
- **长老会**：岛上最高议事机构。三大支柱：须系（女尊、规矩、祖训）、程系（火性、冶炼、军备与资源垄断）、奉眠（外来灵凰、条件上位、出岛布局）。功能：隐瞒危机、分配望神节资源、裁定登岛凡人处置。
- **须家（北方玄武-女宿）**：以女为尊，严苛培养女孩；代表须莫海、须妍、须隐、须灵犀、须灵珑。与程家既共治又互疑。
- **程家（西方白虎-觜宿）**：火性门风，冶炼锻造；代表程灼、程熔、程染（赘婿）、程炜、程炫。对屠丽与镜玄态度最激进。
- **恒水居/恒系残脉**：清修制药，政治存在感弱，却是主角大本营。
- **殇家（已毁）**：觞阵、璎陌、小福；技术与阵法遗产（万象方）仍搅动当下。
- **奉眠势力**：不靠血缘家族，靠实力、寿元与情报；与程老暗斗，对镜玄既护又用。
- **南梁萧氏（凡间）**：皇族寻仙线，萧霁为代表；目标救质子父母、建功立业，与岛上仙家后裔形成文化冲击。
- **神族结界/天界秩序（背景势力）**：设定流放与天赎规则的更高存在；岛民无法直接对话，只能通过考验或破界对抗其规则。
- **海中镇守（南溟玄武等）**：非政治势力，但是地理与军事障碍，阻碍查船与离岛。

**【贵重物品】**
- **万象方**：觞阵研究制作的魔方状装置，木质与金属混合，启动可见嵌欧珀石；非启动全木。望神节奖品中的殇家遗物，屠丽必争，解读案件与阵法的钥匙级道具。
- **寒沁剑**：镜玄神族法器，启动态/隐藏态双形态，随主属性变化，水系冰晶战力核心。
- **藏典阁案宗与法宝**：记录岛上过往禁忌与惩罚的档案；爆炸后部分法宝流入望神节奖池。
- **玉牌**：望神节优先选奖凭证，资源与面子的符号，程炜与屠丽争夺焦点。
- **火芝**：程家进贡/竞赛类珍物，败给屠丽海鱼的对照物。
- **手串（外海）**：屠丽搏鱼时捡得，引发海水震动，关联结界/沉船记忆。
- **大平衡舟 / 小平衡舟 / 航海尺**：屠丽离岛工程核心，手工与航海技术结晶。
- **屠丽肚兜织锦纹样**：生父遗物意象，情感信物，非战力却是角色锚点。

**【隐藏剧情 · 伏笔】**
- 觞阵出岛是否「被利用」：镜玄多年怀疑父母遭人利用，须、程皆在嫌疑名单。
- 奉眠救镜玄并抹身份的真实交易：出岛之约、与何人有约、是否需镜玄血脉/阵法完成破界。
- 程染多次出现在旧案线索：温润外表下的知情程度。
- 屠丽作为「岛上唯一凡人」却力气过人：岛上物质改造凡人的边界与程灼恐惧的「根源」。
- 天赎五百年周期 vs 破界捷径：两条意识形态，长老会内部分歧的深层原因。
- 海水灭仙规则的例外与漏洞：沉船、手串、玄武航道可能藏着规则缝隙。
- 第二季身份暴露与跃崖：公开走向「大战将启」，幕后黑手是否仅止于程家仍待动画展开；契约者可查不可替原作终裁。
- 文化纹样伏笔：服饰商周青铜感、汉纹玩偶酒壶，暗示流放史与更古老神话层（观众猜想级，写作时标为「岛民传说层」）。

**【大事记时间线】**
- 远古 → 神族建思量岛流放犯错神、仙，设结界与天赎规则。
- 岛史漫长 → 仙家后裔聚族，须/程/恒等宿姓成型；奉眠误入以条件换长老位。
- 约屠丽3岁 → 屠霍船队误入，屠丽被觞阵、璎陌收养，与觞小福共长。
- 约镜玄6岁前后 → 璎陌病重/异状，觞阵触禁布阵，夫妇坠海惨案；奉眠救子改名镜玄；屠丽后被「恒老」收养。
- 惨案后十年内 → 镜玄假扮恒凌虚，闭关自修，暗查案宗；屠丽学手工、打渔、筹备离岛。
- 第一季当下深夜 → 藏典阁爆炸；议事堂权斗；望神节用万象方等转移视线。
- 望神节前 → 玉牌之争；屠丽出海；救萧霁；赢大鱼得牌。
- 凡人登岛风波 → 查船遇南溟玄武；镜屠共战；庭审瞒身份。
- 第一季末—第二季前 → 线索指向程家/焱洞；身份裂缝扩大。
- 第二季（约同居三年后） → 镜玄真身与屠丽同住；须程知情；程灼拆穿；身份暴露与跃崖级冲突；风烟迭起大战将启。
- 未竟 → 天赎/破界/真相终裁以动画后续为准。

**【叙事基调 · 雷区】**
基调：新中式瑰丽奇幻 + 青春成长 + 暗恋守护 + 家族权谋；画面工笔华美，情感1V1官配（镜玄×屠丽、须灵犀×萧霁），不写后宫三角。叙事宜「美中见冷」：海水禁忌、寿数流逝、出岛执念压在甜与燃之上。
忌：把思量岛写成普通修仙门派升级地图；把镜玄写成无脑后宫男主；把奉眠写成无条件圣母；用「封印削弱」取消顶点战力；编造未公开的第二季终局死人名单；把屠丽写成花瓶（她是造船与战斗行动派）。
最早可切入锚点：藏典阁爆炸当夜；或屠丽出海夺玉牌/救萧霁前后；高阶可切身份暴露与跃崖前夜。


**【世界剧情线·分阶段细述】**

**阶段甲：岛规与望神节政治（爆炸后的七天）**
藏典阁爆炸后，长老会第一反应不是缉凶，而是「稳人心」。须莫海强调祖训与秩序，程灼强调武力震慑与资源管控，奉眠则提议用望神节奖品把注意力从禁地引开。三方达成表面一致：不公开案宗失窃细节，不承认有人能夜破藏典阁。广场上玉牌兑换排起长队，火芝、海产、手工奇巧轮番上场，屠丽的海鱼最终压过程家珍品——这一结果在岛民口中迅速变成「凡人也能赢」的危险信号。程熔闻声震怒，程炜在巷口集结家丁，程染以「家风」压下暴力，实则把监视网织得更密。屠丽赢牌后并不张扬，她要的是万象方，是养父母遗物里那一块能拼上记忆的几何体。镜玄在恒水居对照案宗标号与儿时玩具结构，第一次确认：父亲不是单纯「疯了出岛」，而是在做一套可复现的装置研究。

**阶段乙：沉船记忆与手串异动**
屠丽外海搏两丈大鱼时撞入沉船，船舱纹样与儿时玩偶、酒壶的汉纹叠合，唤醒「误入—被收养—火光与海」的碎片。手串入手后海水震动，像结界被轻叩。公开剧情未把手串写成万能神器，它更像「规则缝隙的敲门砖」：提醒观众与契约者，离岛不只靠蛮力，也靠对海禁的理解。萧霁被救后以迷路掩盖皇命，斌叔教他「先当弱者换同情」。屠丽给粮给水却拒绝他立刻上岛换盘缠——她清楚凡人登岛等于把火扔进火药库。这一阶段的戏剧张力在于：善意救人与政治风险同时成立。

**阶段丙：南溟玄武与庭审瞒天**
须隐查船是长老会程序正义的表演，也是对「凡人如何穿过结界」的恐惧。南溟玄武一出，程序变成求生。屠丽以凡人身体扛住第一波冲击，镜玄以寒沁切开水幕——两人配合像幼年玩万象方时的默契重演，只是屠丽仍不知对方是小福。战后须隐感恩，庭审上镜玄以逻辑与沉默完成瞒天：他救了人，却不能让人救「他的名字」。萧霁在岛民眼中从「灾星」变成「可利用的凡间线索」；须灵犀在家族礼法与心动之间摇摆；须灵珑则在宴席上随口点破「上仙的剑不像恒家」。三阶切入者最容易在此改写证词与站队。

**阶段丁：焱洞、程家与身份裂**
镜玄调查越深，程染身影出现频率越高；焱洞炉火与兵器锻造暗示程家不止「护岛」，也可能在为某种大战备战。程灼拆穿身份不是突然的灵感，而是长期盯梢与旧案对照的结果。第二季公开信息中，须、程已知情而屠丽未尽知——这意味着最大的情感炸弹不在战场，而在恒水居饭桌：当「恒老」「上仙」「小福」三张脸叠成一张，屠丽的离岛遗愿与镜玄的留岛守护会正面相撞。奉眠在此阶段的态度是关键变量：她可以压下程家刀锋，也可以放任冲突加速破局，只要结果利于出岛之约。

**阶段戊：跃崖与风烟（高阶节点）**
身份暴露后的公开走向指向镜玄跃崖与大战将启。写作与扮演时须守三条：①不写顶点战力被废；②不替未播终局点名处刑；③把「条件性胜利」落在阵眼、同盟、信息与代价交换上。屠丽的舟、万象方、寒沁、奉眠灵凰与结界纹，是五阶舞台的四件「可见道具」；契约者应推动其中至少两件发生可观察互动，而非空喊破界。

**【主要人物·补述（关系网与可扮演切口）】**
- **镜玄×屠丽**：官配1V1。竹马+养成+天降叠层；第二季同居三年。扮演切口：知情权、离岛/留岛、身份坦白时机。
- **须灵犀×萧霁**：凡仙官配。扮演切口：女尊规矩 vs 凡间皇命，是否用婚姻/盟约换资源。
- **奉眠×镜玄**：师徒/棋手。扮演切口：出岛条件清单、是否允许查案触及奉眠旧约。
- **程灼×屠丽**：清除逻辑。扮演切口：能否用「屠丽有用」暂时换命。
- **程染**：灰色枢纽。扮演切口：一次说情、一次沉默、一次泄密即可改线。
- **须灵珑**：情报童子。扮演切口：宴席一句真话引发连锁。

**【地理补述·可遭遇表】**
恒水居后山药圃可藏船材；广场集市可听流言；议事堂侧廊可偷听；藏典阁废墟夜间有傀儡残响；暗礁区涨潮改航道；焱洞外围有程家巡逻暗号；南溟雾带能见度极低，适合伏击与误认。

**【贵重物品·使用边界】**
万象方非即战力神器，强行启动可能重演惨案代价；寒沁认主，他人强持难尽全力；玉牌是政治筹码；平衡舟怕火怕程家锻打干扰；手串异动不可在长老眼前复现。

**【叙事执行要点（给AI）】**
写战斗时突出环境：海、雾、结界纹、舟板、炉火；写权谋时突出名分与寿数焦虑；写情感时守官配、禁后宫；写顶点时情报与条件优先。任何「乐园规则外强制降权战力」表述禁止出现。


**【势力互动与可玩冲突矩阵】**
须家要规矩与女尊传承，怕萧霁把灵犀「拐」成凡间故事；程家要资源与清剿权，怕镜玄血脉唤起旧案清算；奉眠要出岛，怕任何一方提前撕破结界导致她失去谈判筹码；屠丽要看外面的世界，怕被当成灾星软禁；镜玄要真相与她的命，怕坦白后失去守护位置；南梁要仙缘与质子，怕空手而归被朝堂问罪。契约者插入任一缝隙都能形成任务：调停宴、伪证、护送、盗图、换俘、谈条件。注意：冲突推进靠选择与代价，不靠「系统锁战力」。

**【名场面清单（可复现/可改写）】**
藏典阁夜探与爆炸；议事堂三大长老互疑；望神节天秤选海鱼；外海搏大鱼与沉船；暗礁藏萧霁；夜疗伤见手工线索；南溟玄武共战；庭审瞒身份；程灼拆穿；真身同居饭桌沉默；跃崖前对望。扮演时应让名场面「有变数」：同一场面因契约者介入可延迟、加速或换人承受代价，但人物性格不可OOC。

## 阶位切入点

> 阶位↔原作：一阶≈凡人/初入岛民与误入者；二阶≈仙家后裔初入与家族少年；三阶≈家族骨干、成长中的镜玄与屠丽战力；四阶≈长老会主力（程灼、须莫海等）；五阶≈奉眠（灵凰）与破界/天赎级冲突（世界顶点）。顶点为**存在·情报优先/条件性胜利**，严禁「遭禁制/条件受限/实力未减·仅策略规避/不可越阶硬刚」式解释。最高五阶，宁低勿高。

**一阶（凡人/初入岛 · 误入与立足线）**

切入身份/时点：契约者以凡间水手、南梁随行、或风暴后漂流者身份，切入屠丽出海夺玉牌、救起萧霁前后。

初始事件：你在暗礁区醒来，看见屠丽把晕厥的萧霁拖上岸，又把粮水塞进沉船缝；远处程家巡船火光闪动，有人低声说「凡人沾海也别想乱岛规」——你必须立刻选择：帮藏人、告密换活路，或装死漂回外海。

开场白建议：「咸风灌进喉咙时，你听见少女骂了一句『这侯爷可真沉』。暗礁背后，南梁衣冠的少年还在吐水，而岛的方向，望神节的灯已经亮了。她把半袋干粮塞进你怀里：『别出声。出声就都得死。』你这才明白——你不是来寻仙的，你是来活过今晚的。」

关键NPC立场：**屠丽**愿换物资换沉默，重诺但防长老；**萧霁**需要向导与盘缠，会套近乎；**斌叔**怂恿利用丽娘；**程炜**若发现凡人藏匿会暴力夺牌；**程染**可能说情也可能盯梢。

主线钩子/支线：主线——帮屠丽把萧霁藏稳并送大鱼回岛赢玉牌；支线——打听万象方来历、用凡间见闻换须灵珑情报、避开程家巡海。

危险度/规避：中低。规避正面对抗程灼级；海水对谪仙致死规则勿用血肉试探；勿在议事堂曝光屠丽离岛船坞。

任务方向/奖励：立足 freepass、航海常识、基础岛规、手工材料与人情（屠丽/萧霁），不发长老级法器。

**二阶（仙家后裔初入 · 望神节与家族规矩线）**

切入身份/时点：以旁系子弟、望神节文书助手或须/程家外围弟子身份，切入玉牌争夺与藏典阁爆炸余波。

初始事件：天秤当众选中屠丽的海鱼而非程家火芝，文书要你在「偏袒程家」与「按秤公断」之间落笔；程炜已带人堵巷口，须灵犀在廊上冷眼看你如何坏规矩或守规矩。

开场白建议：「玉牌在秤盘里轻得像一句谎话。程家公子笑着说『两丈大鱼——打不来就滚』，可广场那头，丽娘已经扛起渔叉。你手里的朱笔还悬着：写偏，你是程家的人；写正，你今晚可能没有家。」

关键NPC立场：**程炜**要面子与牌；**屠丽**要遗物；**须灵犀**观察外来者是否懂女尊礼数；**须灵珑**会童言拆穿你；**奉眠**远观，不轻易下场。

主线钩子/支线：主线——护送或截胡玉牌相关人证；支线——替须家采办庆典、查爆炸流言、接近恒水居而不惊动「恒老」。

危险度/规避：中。规避夜闯藏典阁硬刚傀儡；规避当众质疑长老会瞒报。

任务方向/奖励：家族入门术法、望神节赏格边角、火芝/海产贸易权、须程两边的「可用之人」评价。

**三阶（家族骨干/镜屠成长 · 查船与南溟线）**

切入身份/时点：以查船队护卫、临时盟约客卿或「上仙」同行者身份，切入须隐查船遭遇南溟玄武前后。

初始事件：外海起雾，玄武掀浪，屠丽的平衡舟几乎折断；镜玄寒沁出鞘的瞬间，有人认出他的剑式不像恒家——你必须在「暴露他」与「并肩斩兽」之间选边，同时萧霁的凡间船队在漩涡边哭喊。

开场白建议：「第一浪拍碎了旗，第二浪拍碎了胆。须隐吼着收绳，丽娘却迎着兽瞳冲上去。冰蓝剑光从侧翼切开水幕时，你看见那少年眼里没有英雄梦，只有『不能让她死』。玄武的鳞甲反光里，整座岛的秘密都在晃。」

关键NPC立场：**镜玄**要瞒身份也要救人；**屠丽**行动优先；**须隐**记救命之恩亦记疑点；**萧霁**恐惧与崇拜并存；**程灼**事后追责「谁引来祸」。

主线钩子/支线：主线——海战存活并影响庭审证词；支线——打捞手串共鸣点、查焱洞流言、助屠丽加固离岛舟。

危险度/规避：高。规避单独挑衅南溟级海兽；规避在庭审上戳破镜玄=恒老。

任务方向/奖励：海战军功、水系/舟战技巧、部分案宗抄本边角、与镜屠的有限信任。

**四阶（长老会主力 · 身份裂与权斗线）**

切入身份/时点：以长老会特使、程家/须家客卿高手或奉眠「棋子」身份，切入程灼拆穿镜玄、须程两家已知情阶段。

初始事件：议事堂重开，程灼抛出「殇家余孽」铁证，奉眠只饮茶不表态；屠丽被挡在门外，万象方在堂上的光一闪——你被点名：是站程家清剿，是站奉眠换条件，还是把屠丽带离火线。

开场白建议：「程灼的声音像锻锤：『他不是恒凌虚。』满堂寂静里，只有茶盏碰托的轻响来自奉眠。你忽然意识到，这座岛真正的兵器不是剑，是『名字』。名字一破，崖下就是海，海里是灰。』」

关键NPC立场：**程灼**要斩草；**奉眠**要出岛布局完整；**须莫海**要祖训与稳定；**程染**左右为难；**镜玄**护屠丽优先于自辩；**屠丽**若知情会爆发行动。

主线钩子/支线：主线——影响长老会表决与镜玄去留；支线——换取焱洞真相碎片、拦截对恒水居的抄检、与须灵犀—萧霁交换凡间退路。

危险度/规避：极高。规避与程灼+家族战力正面团灭战；对奉眠只谈条件不谈道德绑架。

任务方向/奖励：长老级情报、阵法残页、有限法器维护权、站队后的庇护或追杀令（双刃）。

**五阶（灵凰/破界·天赎 · 顶点风烟线）**

切入身份/时点：以破界计划参与者、天赎观察使或「跃崖后救援/追查」的顶级契约者身份，切入身份暴露、镜玄跃崖、大战将启节点。

初始事件：悬崖上寒沁的剑意散作冰雾，屠丽的舟在海禁忌线外起不来帆；奉眠的灵凰虚影第一次不再藏，程家炉火与须家祖钟同鸣——结界纹路在夜空显形。你的任务不是「打死顶点」，而是在破界/天赎/救人间找到**条件性胜利**的缝。

开场白建议：「他回头只说了一句『别下来』，人已经进了风里。海面没有溅起该有的血，却浮出一层像神谕的纹。奉眠的声音很轻：『五百年的天赎，等得起的人已经死绝了。』你握着不知从哪一季漂来的万象方残件，第一次看清——岛不是家，是锁。锁的钥匙，从来都烫手。」

关键NPC立场：**奉眠**（顶点之一）只认出岛契约；**镜玄**生死未明但意志未灭；**屠丽**会赌命救人/离岛；**程灼**趁乱扩大清剿；**萧霁/须灵犀**可成凡仙联盟变量。

主线钩子/支线：主线——跃崖后的生死检索与结界纹路解读；支线——拼合觞阵法阵原设计、阻止屠丽无准备冲海、与奉眠谈「条件性」合作而非臣服。

危险度/规避：贴近顶点。**超阶巅峰处理：存在·情报优先/条件性胜利**；严禁写奉眠/结界「遭禁制/条件受限故战力为零」。可赢在信息、阵眼、同盟与代价交换，不可写无代价斩杀顶点。

任务方向/奖励：破界/天赎相关关键认知、万象方完整启动条件、寒沁共鸣残识、改变岛史走向的因果权重（非越级神器轰炸）。
