# 假面骑士（1971）
<!--meta lib=主库 tiers=一、二、三、五、六-->

## 剧情


**【作品来源】**
《假面骑士》（仮面ライダー，Kamen Rider）1971年特摄电视剧，石森章太郎原作，东映×每日放送制作，1971年4月3日至1973年2月10日播出，全98集。主演藤冈弘、佐佐木刚、千叶治郎、小林昭二；旁白中江真司；音乐菊池俊辅。类型：特摄超级英雄、科幻恐怖、改造人正义剧。系列开山之作，后接《假面骑士V3》等昭和假面骑士。卡片锚点：ショッカー（Shocker）怪人作战高峰期——本乡猛/一文字隼人双骑士联手前后，立花藤兵卫与滝和也协力反击。

**【世界定位】**
1970年代日本为舞台的暗黑改造人战争世界：纳粹残党与秘密结社组成的ショッカー以绑架、改造、洗脑制造动物型改造人，企图征服世界。城南大学生化系学生兼机车手本乡猛被改造为假面骑士1号后逃出洗脑，后救出摄影师一文字隼人成为2号，双骑士与立花赛车俱乐部、FBI特工滝和也共同作战。一句话：改造人英雄以肉体与机车对抗世界征服组织的昭和特摄原点。

**【世界观 · 力量体系】**
力量来源：①改造手术（动物/昆虫基因+机械义体）赋予超人力量、跳跃、装甲与必杀技；②机车（サイクロン号等）高速机动；③情报与组织（少年假面骑士队、FBI）；④怪人侧的兵器、基地与干部指挥。死亡规则：常人一击即死；战斗员可被骑士徒手击溃；怪人被Rider Kick等必杀后爆炸身亡；干部可多次复活/再登场（剧中后期Destron线等跨作），但本季内多数一死即终。无魔法复活。特殊系统：变身腰带（タイフーン）需风力/能量启动；双骑士合体技；ショッカー→ゲルショッカー重组。

战力阶梯（宁低勿高）：
1. **普通人/记者/学生**：无战力。≈一阶初。
2. **训练有素的特工/赛车手/空手道黑带**：可打倒单个战斗员边缘。≈一阶中后。
3. **ショッカー/ゲルショッカー战斗员**：强于常人，群体威胁。≈一~二阶。
4. **一般怪人（动物改造人）**：可毁车辆、压制凡人小队，需骑士变身应对。≈二~三阶。
5. **干部怪人本体（狼男/イカデビル/ガラガランダ/ヒルカメレオン等）**：指挥一方基地，战力高于普通怪人。≈三~五阶观感。
6. **假面骑士1号/2号（双骑士）**：稳定击破怪人，后期可联手灭组织。≈五阶。
7. **顶点·ショッカー首领/ゲルショッカー首领**：组织级、跨季存活阴影，非单集可永久抹杀的本体感。≈六阶·情报优先。

乐园阶位映射：普通人~特工≈一阶；战斗员~初阶怪人≈二阶；强怪人/地区基地≈三阶；**本世界不覆盖四阶**；双骑士成名期≈五阶；首领与组织终局≈六阶。顶点=首领级，**超阶未达**；低阶切入规避首领本体与基地自爆。宁低勿高。

**【地理 · 舞台】**
- **日本本土都市与郊外**：怪人每周袭击工厂、研究所、学校、公路。
- **Snack Amigo咖啡馆→立花オートコーナー/立花レーシングクラブ**：早期据点与人情舞台。
- **ショッカー各地支部与秘密基地**：地下设施、实验手术室、自爆装置。
- **欧洲/瑞士等本乡海外作战暗示**：本乡一度赴欧打击支部。
- **ゲルショッカー重组后的新基地**：紫黄战斗员、杂交怪人。
舞台由「街头怪人事件→基地突入→干部决战」循环扩展；契约者先定自己是凡人协力者还是改造边缘人。

**【世界剧情线】**

① **本乡猛觉醒（第1~13集前后）**  
城南大学学生本乡猛被ショッカー绑架，改造成蝗虫系改造人，手术完成前被恩师绿川弘救出。绿川被蜘蛛男杀害。本乡以假面骑士身份开战，变身口令与Rider Kick定型。立花藤兵卫（赛车队经理）成为精神支柱；绿川ルリ子从误解到协助。世界状态：日本普通人不知组织全貌，怪人事件被当奇案。名场面：蜘蛛男、初次变身、机车追逐。因果：正义改造人对抗邪恶改造人的主题立住。

② **怪人周刊与立花据点**  
ショッカー连续派出蝎子男、サラセニアン、かまきり男等怪人执行绑架科学家、投放毒物、破坏设施。本乡在Snack Amigo与立花赛车俱乐部成员（野原ひろみ、ライダーガールズ等）之间往返，一边保护市民一边追查基地。滝和也（FBI）加入，武术协助。世界状态：一骑士苦战，组织仍在扩张。

③ **本乡赴欧与一文字隼人登场（约第14集起）**  
本乡赴海外打击ショッカー，摄影师一文字隼人被改造，本乡在洗脑完成前救出，隼人成为假面骑士2号，守护日本。サイクロン号与2号机体作战风格略异。世界状态：双骑士体制开启。名场面：2号初战、本乡托付。因果：系列「传承」模式成立。

④ **双骑士与干部战（卡片锚点期）**  
ゾル大佐（狼男）、死神博士（イカデビル）、地狱大使（ガラガランダ）先后坐镇日本支部，指挥大规模怪人作战。本乡归国后与隼人联手，Rider Double Kick等合体技出现。立花藤兵卫开设车行与俱乐部，少年假面骑士队（第74集起）在全国发展，孩子们用通信徽章报告怪人迹象。世界状态：组织与骑士进入全面战争。名场面：干部变身决战、友人被俘、基地爆破。

⑤ **ショッカー覆灭与ゲルショッカー**  
地狱大使败死后，首领摧毁旧组织残部，与ゲルダム残党合并为ゲルショッカー。杂交怪人（两种生物融合）与紫黄战斗员更强。ブラック将軍（ヒルカメレオン）指挥。双骑士再战。世界状态：敌人升级，和平假象破灭。

⑥ **终局**  
双骑士击败ゲルショッカー首领侧战力，组织崩溃；滝和也返回美国，表面和平恢复——实则首领阴影未绝，衔接V3的デストロン。结局：骑士胜利但是「暂时的」；改造人身份的孤独仍在。

⑦ **微观日常贯穿**  
绑架科学家、假事故、学校与工厂袭击、毒物瓦斯、战斗员街头群殴、机车追逐、咖啡馆情报交换、少年队通信——AI优先这些，而非每集末日。

⑧ **蝴蝶效应边界**  
可改：救一名被改造者、提前炸毁一处支部、保护立花店。不可无代价改：双骑士诞生的改造因果、首领跨季存活、ショッカー→ゲルショッカー重组趋势。

**【主要人物】**
- **本乡猛 / 假面骑士1号**｜性格：热血、正义、责任感强，改造后仍保留人类心。｜装备·能力：タイフーン变身、蝗虫跳跃与力量、Rider Kick/Punch/Chop、サイクロン号。｜人物弧光：被改造→叛逃→独战→赴欧→归国双骑。｜立场关系：对立花如父；对隼人战友；对契约者先察是否ショッカー余党。
- **一文字隼人 / 假面骑士2号**｜性格：冷静、摄影记者式观察、战斗果断。｜装备·能力：2号变身与机车、与1号合体技。｜人物弧光：被改造前夜获救→守日本→与本乡联手。｜立场关系：双骑士之一；可托付后背。
- **立花藤兵卫**｜性格：豪爽、烟斗、父亲般。｜装备·能力：无超人战力，情报与据点、少年队总裁。｜人物弧光：赛车经理→骑士最大支援者。｜立场关系：骑士精神家园。
- **滝和也**｜性格：FBI务实、武术强。｜装备·能力：非改造人，可空手战战斗员。｜人物弧光：调查→并肩→归美。｜立场关系：国际线接口。
- **绿川弘**｜性格：学者良知。｜装备·能力：生化权威，曾被迫为组织工作。｜人物弧光：救本乡→被蜘蛛男杀。｜立场关系：骑士诞生的代价。
- **绿川ルリ子**｜性格：从仇恨到理解。｜装备·能力：凡人。｜人物弧光：误认本乡杀父→知真相协助→随本乡欧战暗示。｜立场关系：早期情感锚。
- **野原ひろみ**｜性格：活泼支援。｜装备·能力：Snack Amigo店员线。｜人物弧光：日常协力。｜立场关系：据点人情。
- **ショッカー首领**｜性格：冷酷、可牺牲部下。｜装备·能力：组织权柄、跨形态（后续作揭露岩石大首领等）。｜人物弧光：幕后→ゲル首领→跨季。｜立场关系：**六阶顶点·情报优先**。
- **ゾル大佐 / 狼男**｜性格：伪装高手、中东支部出身。｜装备·能力：狼形态、可化妆成滝。｜人物弧光：坐镇日本→被2号击败→V3中短暂复活。｜立场关系：干部级敌。
- **死神博士 / イカデビル**｜性格：冷算。｜装备·能力：乌贼形态、触手鞭。｜人物弧光：瑞士支部→日本指挥→被1号击败。｜立场关系：干部级敌。
- **地狱大使 / ガラガランダ**｜性格：残暴、鞭与铁爪。｜装备·能力：响尾蛇形态、地中移动。｜人物弧光：东南亚支部→日本最高干部→败死引爆组织重组。｜立场关系：ショッカー末期门面。
- **ブラック将軍 / ヒルカメレオン**｜性格：ゲル侧指挥官。｜装备·能力：水蛭+变色龙、吸血操控、隐身。｜人物弧光：ゲルダム并入后指挥→败。｜立场关系：终盘干部。
- **ショッカー战斗员**｜性格：无个性工具。｜装备·能力：黑服超人肉体。｜人物弧光：每集炮灰。｜立场关系：一~二阶日常敌。
- **ライダーガールズ与少年队（ナオキ、ミツル等）**｜性格：勇气与通信。｜装备·能力：凡人+徽章电台。｜人物弧光：情报网。｜立场关系：可结盟协力。

**【势力图谱】**
- **ショッカー**：纳粹残党系世界征服组织；改造人工厂；战斗员+怪人+干部+首领。
- **ゲルショッカー**：旧组织+ゲルダム合并；杂交怪人。
- **假面骑士侧**：1号2号+立花据点+滝+少年队。
- **日本警察/社会**：常后知后觉，需骑士收拾。
敌友：利益与洗脑；改造完成=失自我；暴露骑士身份会牵连平民。

**【贵重物品】**
- **タイフーン变身腰带**：骑士核心，失=无法变身。
- **サイクロン号等专用机车**：高速与武装。
- **少年队徽章电台**：情报网节点。
- **怪人改造手术台/改造液**：组织核心科技，抢=伦理炸弹。
- **ショッカー徽章扬声器**：首领传令。
- **双骑士合体技条件**：信任与同步，非道具但等同战略资源。

**【隐藏剧情 · 伏笔】**
- 首领未真正死绝，V3デストロン是续篇。
- 漫画版与TV版剧情分歧（本乡重伤、12ショッカーライダー等）——**本档案以TV准**。
- 改造人「人类之心」主题：洗脑完成前一刻的救赎是系列母题。
- 后续平成/令和骑士宇宙可互文，但不倒灌本季。

**【叙事基调 · 雷区】**
画风：昭和特摄热血+怪人恐怖+机车摇滚；可黑暗不可无脑搞笑消解。忌：①把骑士写成魔法少女式无代价变身；②低阶单挑干部本体；③忽略改造手术的人体实验残酷；④用平成腰带梗污染1971；⑤让首领被一阶秒杀。最早切入：本乡刚逃出改造室前后，或隼人被改造前夜。

**【剧情增密·怪人作战档案】**
ショッカー怪人多为「原人类+动物基因」：蜘蛛男负责初期绑架链；蝙蝠男、蝎子男、蜂女等执行暗杀与毒物；后期地狱大使系出动ジャガーマン、ごきぶり男、シオマネキング等，能力从地中移动到寄生操控不一。ゲルショッカー改为双生物杂交：ガニコウモル、サソリトカゲス、クラゲウルフ等，单兵素质整体上调。每集结构常为：怪人计划→市民受害→骑士调查→中盘苦战→识破弱点→Rider Kick爆破。契约者介入点在「计划曝光前」价值最高。

**【剧情增密·立花据点与少年队】**
Snack Amigo是早期情报交易所：店员、车手、孩子带来怪人目击。立花オートコーナー后成为物资与伤员中转。少年假面骑士队全国化后，徽章电台让「小学生的目击」变成战略预警——这是本世界独特的「凡人战力」。滝和也作为队长级存在，把FBI调查手法带入日本线。写日常优先：修车、送零件、藏伤员、教孩子逃生，而不是每集末日。

**【剧情增密·双骑士战术】**
1号擅长正面突破与精神感召；2号擅长渗透、摄影取证与冷静收割。合体时Rider Double Kick/Chop可越级处理干部边缘战。本乡赴欧期间日本线由隼人独撑，压力更大。契约者若以「第三骑士苗子」自居，会被立花先骂后帮，会被组织优先猎杀。

**【地理增密】**
工厂区夜班、港湾集装箱、山间公路、废弃矿山基地、都市下水道、赛车场——皆是怪人出没经典布景。ショッカー支部常伪装成企业或研究所。ゲルショッカー基地更具军事感。海外线仅作背景压力，正文主舞台在日本。

**【人物增密】**
绿川ルリ子的「从仇到援」是早期情感教学；野原ひろみ与ライダーガールズ提供非战斗清扫与通信；ナオキ与ミツル是少年队脸面。干部三人（ゾル/死神/地狱大使）性格从伪装、冷算到暴虐递进，对应骑士成长阶段。ブラック将軍带来「组织可再生」的绝望感。

**【势力增密】**
ショッカー内部有支部竞争与干部夺权；失败怪人常被抛弃。ゲルショッカー清洗旧部时制造森林屠杀等惨剧，显示首领对生命的绝对工具化。骑士侧无正式编制，靠信任运转——这也是弱点：俘获立花或少年队员即可逼骑士现身。

**【道具增密】**
改造手术记录、洗脑装置、怪人强化药、自爆开关、假警察证件、毒物瓦斯罐、机车维修零件、双骑士联络信号——皆可成为任务目标。低阶抢到变身腰带也无法使用，强行安装等于送死。

**【伏笔增密】**
首领真身在Stronger等后续才完全揭示；本季只需保持「扬声器后的不可名状」。漫画Big Machine等要素不写入TV正典主线。跨作复活干部在V3第27-28集出现，可作六阶周边事件。

**【日常事件池】**
科学家失踪、学校运动会遭袭、公路巴士劫持、水库投毒、发电厂破坏、假车祸掳人、战斗员夜训、咖啡馆窃听、少年队误报、赛车场对决——每条都可独立成局。

**【再增密·改造伦理】**
ショッカー的手术不只是赋予力量，更是删除拒绝权。洗脑完成前的「最后一秒」是本乡与隼人共同的原罪式幸运。契约者若试图「自愿接受改造变强」，会落入组织脚本：力量换忠诚。绿川弘的死证明——科学家良心在组织里没有退休。写改造场景要有血、钢、动物内脏意象与麻醉失效的痛苦，而不是酷炫升级动画。

**【再增密·机车与街道战】**
サイクロン号不仅是坐骑，是骑士机动哲学：公路即战场，引擎声即宣战。都市追逐中常人车辆、电车轨道、桥梁皆成障碍。立花的维修技术让机车起死回生，等同后勤神迹。二阶以下契约者抢骑士机车=被摔死或被组织锁定。

**【再增密·音乐与符号】**

初始事件：你掌握「首领传令扬声器」的物理位置，摧毁它可致指挥链瘫痪，但会触发全国支部连环自爆——平民伤亡预估在你桌上。

开场白建议：「地图上的红点同时亮起。双骑士已冲进中央通道，你的手按在总开关上。灭组织，或救一城。首领在笑：他赌你会犹豫。」

关键NPC立场：**双骑士**主攻；**ブラック将軍**死战；**首领**条件性胜利对象；**立花藤兵卫**要你优先平民。

主线钩子/支线：终局指挥链斩断；支线：疏散、假死撤离、为V3线留「首领未死」情报。

危险度/规避：贴近顶点。**六阶巅峰：情报优先/条件性胜利**；严禁战力归零削弱首领。

任务方向/奖励：组织覆灭结算、骑士勋章级认可、跨季情报（デストロン前兆）。
