# 苟在两界修仙

## 剧情

**【作品来源】**
《苟在两界修仙》作者笔名「文抄公」（阅文大神，代表作含《巫界术士》《苟在妖武乱世修仙》《星空职业者》《神秘之劫》《神秀之主》等），仙侠·修真文明长篇，2026 年 1 月 1 日起于起点中文网连载，状态连载中（公开统计约百八十万至近两百万字量级，章节六百余章浮动，最新章名随更新变化）。气质：多界穿梭苟道、真炁与金位果位、道统权柄、稳健发育；爽点在信息差与账本，不在无脑横推。官方简介要旨：年岁丰凶、灵氛运转、阴阳嬗变皆「值岁」微末之绩；少年穿越两界修仙，摸索成为【值岁】。标签常见：苟、双穿、证金、真炁、道统。媒介以网络小说正典为准。本档案据百科、书讯与公开评述可核设定整理；搜笔趣阁书页若环境 403 则以百科与起点书讯为准；**连载推进中，不编造终局**。

**【世界定位】**
以「值岁」权柄与多世界道统为核心的修真宇宙。主角**方青**携前世记忆与**道生珠**，在古蜀服气道、元始天炼气道、大周炁功界、灵界、帷幕世界等多界之间穿梭，低调收集真炁、神通与金位，逐步逼近十二值岁之一的位置。人间疾苦与太古权柄同框：低层在瘟疫与逃难里抢一口粮，高层在金位与道统里抢一年。一句话：双界（多界）苟修，珠藏天机，目标不是称霸一城，而是摸到「年岁」本身的权柄——先活过每一个冬天，再谈值哪一月。

**【世界观 · 力量体系】**
本世界**多体系并行**，不可压成单一升级条。方青可兼修、转化，以道生珠返本归元为枢纽。

**一、服气道（古蜀世界·仙道）**  
服气境 → 道基境 → 紫府境 → 金丹境。强调服气吐纳、道基、紫府开府与证金。密藏域（大雪山道统）另有：僧侣 → 上师 → 法王 → 即身佛。妖族：妖兵 → 妖将 → 妖王 → 妖圣。金丹与「金位」深度绑定：主/从/顺/客/缺等分等，对应天仙/地仙等意象；晋升可用摄、隐、借、养、篡、禅等方式控制他位。

**二、元始天·炼气道（东海修仙界）**  
炼气期 → 筑基期 → 结丹期 → 元婴期 → 化神期。妖兽一至五阶对照。方青公开进度中有炼气道化神初期等表述（随连载浮动）。

**三、大周世界·炁功**  
外功：练力 → 练劲 → 练神。内功（炁功）：先天 → 胎息 → 内景 → 外景 → 天人。偏人间武道与炁感，适合「另一界低调刷资源」。

**四、灵界**  
先天期 → 心动期 → 灵璇期 → 化婴期 → 分神期 → 炼虚期 → 洞玄期 → 大乘期 → 真仙 → 道主/法主。尺度更高，是中后期开图。

**五、帷幕世界**  
蒙昧者 → 呢喃者 → 通晓者 → 长生者 → 飞升者；另有隐秘「蚀世者」层级（藏骸主、牧场主、大群之主等公开名向）。气质偏神秘污染与认知危害，与服气/炼气的「清修」对照。

**六、真炁与道统**  
真炁分九阶四品三十六等级：最高「元始之炁」，九阶上品含大日紫炁、太阴青炁等。道统阴阳五德共二十八家（大日、金德、火德、太阴、水德、土德、木德等）。太古正魔以道统划分。

**七、值岁与金位**  
十二值岁对应十二月星宿位（一月虚日、二月牛金……十二月室火等）。太古大日值岁兼位失败陨落，金位四散，太古终结——是世界史前史。证金、空证、金位争夺是中高层主线燃料。

**八、金手指：道生珠**  
混沌色珠，三大公开功能：  
1）**返本归元**：回炼能量为元炁再化生；  
2）**无所不至**：穿梭两界/多界门户；  
3）**如在算中**：免疫天机推算、可编织虚假命格身份。  
写珠必须写「苟」：暴露穿梭与免疫推算会引来道统级猎杀。

**乐园阶位映射**（宁低勿高，对照破坏力）：服气/炼气低层、炁功练力练劲≈**一阶**；服气中层/炼气中、炁功先天胎息≈**二阶**；道基/筑基、可屠寨≈**三阶**；紫府/结丹初、内景≈**四阶**；金丹稳固/元婴边缘、外景≈**五阶**；元婴高/化神初、天人边缘≈**六阶**；化神稳/灵界中高、城际灾难≈**七阶**；真仙/值岁边缘/多界权柄≈**八阶**；完整值岁、蚀世主、元始之炁执掌者＝**超阶**（情报优先/条件性胜利）。同境因真炁品阶、金位主从、是否兼修可上下浮动观感，但映射以典型破坏力为准。本库覆盖一～八阶。严禁「顶点战力归零」式解释。

**【地理 · 舞台】**
- **古蜀/巴蜀流民线·三水坳**：方青开篇地。河谷、湿瘴、疫棚、乱葬与逃难土路。服气道人间底色在此写实：气是药也是税，命比草贱。适合一阶。  
- **服气道城邑与道门**：青砖门墙、灵根台、真炁市坊、紫府洞府外的迎客松。金位传闻从茶馆传到刑堂。适合二～五阶。  
- **密藏域·大雪山**：海拔与戒律双重寒冷；法号、转经、东渡船。密僧看中原像看待渡的河。适合三～六阶支线。  
- **元始天·东海修仙界**：岛链、潮汐灵脉、宗门水牢与妖兽回游路线。化神尊号在海湾回响。适合四～七阶。  
- **大周王朝**：官道、漕运、兵镇、炁功武馆与夜禁。飞剑非法，拳头与印信合法。适合二～五阶第二身份。  
- **灵界通道与洞府**：光与法则更「硬」的天空；道主府的名册比刀锋利。适合六～八阶。  
- **帷幕世界**：街道可以是记忆的褶皱；呢喃者在墙缝交易「多知道一点」。适合五～八阶高危。  
- **值岁虚位与太古遗迹**：季节乱序的遗址、金位四散的祭坛、没有脸的月份。终局级，低阶只闻其名。  
写场景先定：**在哪一界、哪一道统、是否暴露道生珠、当前金位是否被人盯上、今天的假名是谁**。同一地理名词在不同界可能同名异实——这是穿梭文常见陷阱，也是方青利用的伪装。

**【世界观 · 力量体系·补述】**
真炁九阶四品不是装饰：品阶决定同境质量，像武道内力六重的「质」。元始之炁是传说顶，大日紫炁、太阴青炁是九阶上品样本。道统二十八家把元素与伦理绑在编制上——入大日与入水德，资源包不同，仇家包也不同。太古正魔以道统分，不等于今天的正派邪派脸谱；写「魔」时要问是道统标签还是帷幕污染。方青的水灵根与箕水命格，使他天然靠近水德/太阴相关叙事，但道生珠允许他「洗」出别的路，洗的代价是时间与暴露风险。丹师副业提供合法收入与社交面具：四阶下品丹师够在市坊立足，不够在道主面前吹牛。口舌类天赋（言出法随边缘）适合谈判与战场扰乱，不是无敌嘴炮，要写反制与冷却。

**【世界剧情线】**
① **瘟疫逃难与道生珠觉醒**  
方青以流民少年之身（前世记忆逐渐苏醒）经历三水坳瘟疫、亲人零落、随老叔逃难。采气谋生、遭追杀等危机中激活网购遗物意象的**道生珠**，获得返本、穿梭与瞒天能力。细节链：疫区的气味、逃难队的抽签、第一次把「气」当饭、珠亮时周围推算落空的诡异安静。此阶段世界状态：人间疾苦、服气低层遍地，值岁只是童谣。因果：苟道人格定调——能跑不硬刚，能瞒不炫富；道生珠三功能全部埋钩，但只用到活命所需的最小集。

② **服气道起步与道基**  
在古蜀仙道规则下服气、立道基，接触门派、市坊、妖修与密僧传闻。中品水灵根、命格箕水等公开设定落地；真炁开始「地泉淬真」一类低中阶积累。门墙内外是两套语言：墙外讲粮与命，墙内讲根骨与供奉。方青以「普通好苗子」伪装，把珠的奇迹翻译成「运气好」。可介入：逃难车队护运、瘟疫余波夺药、低层道门收徒骗局、市坊真炁掺假案。

③ **首次稳定穿梭与双界发育**  
道生珠「无所不至」打开第二界（炼气道或大周炁功界公开向）。方青学会身份隔离：此界小修士、彼界炁功客，账本分开；库存分仓，仇家分册，连修炼日程都按两界时辰表排。返本归元把战利品洗成元炁，再投入缺资源的那一侧。主题：时间差与信息差是最大法宝。失败样本也要写：有人模仿穿梭死在门缝，有人返本洗不掉道统印记被追杀——用来衬托珠的不可复制。

④ **紫府、神通与金位阴影**  
紫府境开府，四神通与金位玄妙挂钩。箕水缺位等公开进度出现；证金不再是传说。道统二十八家的招揽与暗杀同步上升。「如在算中」开始频繁用于骗过推算与做假身份。名场面气质：别人算天机算到空白，方青已经在另一界结完丹；空证拍卖上买家吐金粉暴毙，司仪归咎天道，方青归咎人。因果：从「修士」变成「编制候选人」。

⑤ **炼气道化神与东海棋局**  
炼气道线推进至化神初期等公开节点；《水经注》《碧海功》《吞海功》等功法名进入战斗与经营。东海宗门、妖兽阶位、结丹元婴战场提供更高破坏力样本。方青仍以「震明离炁持法真君」等尊号在局部建立声望，但核心策略仍是不把所有筹码放同一界——尊号是工具，不是身份证。可介入：妖潮挡账、伪尊号、真炁商队、丹师战时订单。

⑥ **大周炁功、灵界与帷幕侧写**  
炁功界提供另一套身体与组织资源；灵界打开真仙/道主视野；帷幕世界引入蚀世者级污染威胁。多线不是开地图游戏，而是「哪条线暴露成本最低、回报最高」的会计题。琴如雪、许黑、展等公开主要角色在此阶段成为稳定钉点：有人管情，有人管刀，有人管眼。

⑦ **值岁之路（连载推进中）**  
十二值岁虚位、太古大日兼位失败史、金位四散后的近古争夺，构成长线。方青目标从「活下去」变为「摸到年岁权柄」。太古史不是背景板，是仍在计息的债。**结局不编造**：只保留「多界苟成值岁」的主题方向；谁死谁活、最终值哪一月，以连载为准。

⑧ **贯穿微观：苟的技术细节**  
假身份、假命格、分界存货、真炁品阶表、丹师副业（公开有四阶下品丹师向）、口舌类天赋法术（言出法随边缘）、尊号与真身分离——正文应优先这些「会计师式修仙」，而不是天天喊打值岁。每一次露富，都要写对应的灭口或搬家。

****【世界剧情线 · 阶段详述】****

**阶段甲：三水坳与逃难**  
瘟疫如何撕碎宗族；老叔与流民队的伦理；第一次采气失败与成功；道生珠首次发光的代价感。可介入：药引争夺、瘟疫锅检查、逃难津渡买命钱。

**阶段乙：服气门墙**  
外门杂役式日子、灵根检测、命格箕水被老人看重或看轻、市坊真炁材料价。可介入：替人炼废丹背锅、密藏域东渡僧人路过引发的恐慌。

**阶段丙：双界账本**  
建立第二身份；两界时差；返本归元洗赃；第一次用虚假命格骗过低阶推算。可介入：两界同时被仇家找、必须选救哪边。

**阶段丁：紫府与神通考**  
开府材料、四神通对应金位、箕水缺位争夺战的外围。可介入：空证骗局、借位契约、篡位传闻中的炮灰位。

**阶段戊：东海化神压强**  
炼气道宗门大比、妖兽潮、化神初期立足。可介入：护送《吞海功》残卷、假扮散修卖元炁。

**阶段己：炁功王朝与灵界门**  
大周朝廷征炁士；灵界裂隙临时工。可介入：以炁功身份潜入不想暴露仙道的场合。

**阶段庚：帷幕与蚀世阴影**  
呢喃者集会、长生者交易、蚀世主名讳。低阶只闻其名；八阶才有「如何不灭」的叙事资格。可介入：送错帷幕信件、回收污染样本。

**阶段辛：值岁虚位争夺的前夜（连载）**  
十二月位传说、太古遗物、金位合并仪式外围。不写终局胜负。

**服气道人间志（扩写·事实向）**  
古蜀地脉湿、水多、瘟多。服气士把「气」当粮也当税：门派按灵根收徒，市坊按真炁品阶标价。方青中品水灵根不耀眼，命格箕水却像一把钥匙——钥匙不保证开门，只保证有人想抢。道基境要的是稳，不是响；紫府境要的是府中四神通与金位玄妙对齐，空证者死于推算反噬的新闻会定期刷新，用来恐吓想走捷径的人。金丹境在本体系里不只是「结个丹」，而是服金登位：主位天仙最优，从/顺/客/缺各有政治含义。摄、隐、借、养、篡、禅六法，是上层把金位当股权操作的说明书。契约者若把金丹只理解成战斗力数字，会在谈判桌上先死。

**炼气道东海志**  
东海修仙界更「传统网文」：山门、大比、妖兽阶、结丹元婴化神。方青把《水经注》《碧海功》《吞海功》等水脉功法与服气道水德真炁互喂，形成跨体系复利。化神初期是公开进度中的重要标尺——足以在局部海域称尊号，却远不够在值岁棋盘上亮真身。东海的敌人往往不是一个人，而是「宗门+妖潮+真炁通胀」。返本归元在这里最赚：战场残骸进珠，出元炁，再买下一次战场的入场券。

**大周炁功志**  
大周提供「不会法术的身体」：练力练劲练神的外功，与先天到天人的内景路。朝廷征炁士平叛、门派收保护费、江湖卖秘籍。方青的第二身份适合办那些「仙道身份办不了」的事：进官场、进军伍、进不让飞的城。炁功战力映射偏低阶到中阶，但情报与组织力可杠杆仙道资源。写大周戏要烟火：酒肆、兵符、漕运、镖局，而不是动辄飞剑灭城。

**灵界与帷幕志**  
灵界是「天」的方向：洞玄、大乘、真仙、道主法主。规则更硬，面子更贵，假身份成本更高。帷幕则是「侧」的方向：蒙昧到飞升的认知阶梯，蚀世者把世界当牧场。方青若在帷幕暴露免疫推算，会被当成最肥的羊或最危险的狼。两界对照：灵界杀你用令，帷幕杀你用「让你明白不该明白的事」。

**道生珠使用纪律（给正文的硬规则）**  
1）穿梭前先问：另一界是否有人正盯着我的尸体位置？  
2）返本前先问：这股能量是否带道统印记，洗不干净？  
3）瞒天前先问：假命格是否与当前金位冲突？  
4）任何公开场合，珠是「从来没有的东西」。  
违反纪律的剧情可以发生，但必须付出血的利息——这是苟文的契约。

**证金社会学**  
金位不是奖杯，是编制。有编制才有道统供奉、真炁配额、话语权。缺位像编制空缺，吸引投机者；主位像铁饭碗，吸引暗杀。方青走箕水相关路径时，每一步都会碰到「要不要入某家道统」的选择题：入则资源，不入则自由与风险。琴如雪、许黑、展等公开主要角色，是这张关系网上的钉——钉可挂画，也可钉人。

- 东海：妖兽潮、宗门大比、化神挡账、真炁商队。

- 帷幕：呢喃集会、污染样本、蚀世代行路过。  
- 跨界：门缝泄漏、两界仇家会合、假珠骗值岁虚影。  
每条事件写清：在哪一界、暴露成本、方青是否在场、报酬是真炁还是金位线索。

**世界状态速查**  
①道生珠是否已激活？②是否已稳定双界？③紫府是否已开？④是否已沾金位/缺位？⑤炼气道是否已化神级公开？⑥是否已触灵界/帷幕？⑦是否已进入值岁虚位叙事？七问决定 NPC 口径与危险预算。

**人物关系补述**  
方青与琴如雪：公开并列的情感/道业双线入口，正文应写互助与保密的张力，忌降智成花瓶。方青与许黑、展：行动与情报三角，可分可合。老叔线是早期道德锚，中后期可能只以回忆与忌日出现。道统使者没有固定脸，但有固定态度——要编制内的狗，不要编制外的龙。密僧看方青像看「可渡的外道」或「该收的魔」。东海长老看方青像看「可征的战力」。帷幕长生者看方青像看「会走路的样本」。值岁虚影不看人，只看「年」。

**修炼日程范例（中期双界）**

**道统二十八家速写（写作用）**

**【隐藏剧情 · 伏笔】**

**【叙事基调 · 雷区】**
画风：冷静苟道、账本式修仙、多界身份喜剧与杀机并存；爽点在「算赢」与「活到下一界」，不在「喊打」。对话可冷幽默，不可无脑中二。场景切换要有「卸妆」：换衣、换名、换呼吸法。  
忌：逼方青无脑单挑值岁；让道生珠当众表演穿梭；把多体系写成混乱数值膨胀；未完结写「已成完整十二值岁主宰大团圆」；用「顶点战力归零」削弱太古残影；把琴如雪等写成无能力花瓶；忽略瘟疫开篇的残酷底色。

**给入世正文的三不**  
不解释完所有道统；不让路人喊值岁小名；不让一阶任务奖励九阶真炁。

**【长写·瘟疫与气】**
三水坳的早晨没有鸡鸣，只有咳。瘟疫把宗族的谱系烧成灰：有名的先死，因为先被抬去求神；无名的后死，因为没人记得送药。方青在记忆苏醒的那夜数过屋顶的星——星不救人，但让他想起前世手机屏上那枚「道生珠」订单。珠在识海亮起时，没有系统音效，只有一种「账本被打开」的清脆。他用最小功能：短距门躲刀，返本半口浊气续命。老叔以为他走了邪运，流民队以为他走了鬼运；只有他知道，运是假的，纪律是真的。

**【长写·门墙经济学】**
服气门墙外，灵根是彩票，命格是期货。中品水灵根换不来筵席，箕水命格却能换来绑票。方青把请柬与恐吓信叠在一起压床板，像压两张到期日不同的债。市坊真炁掺假案每周上演：买家吐血，卖家换招牌，牙人抽成。他以丹师学徒身份卖中品水丹，收入合法，流言也合法——「那小子药稳」，比「那小子会穿梭」安全一万倍。

**【长写·双界时差表】**
甲界子时，乙界可能午时。方青的袖里永远有两套时辰牌。仇家按甲界夜袭搜山时，他在乙界的丹房看火候；乙界官差按户籍抓人时，户籍是假的，人在甲界听空证拍卖。返本归元的仓库分三格：干净元炁、带印记待洗、洗不掉待弃。弃不是扔，是找无人的门缝扔进死界——扔错会引来道统巡逻。双界发育的本质是会计：每一枚丹、每一缕真炁、每一次门缝，都有科目。

**【长写·金位不是奖杯】**
证金那天没有烟花，只有名册。缺位像空出来的编制，吸引投机者用血填。方青走箕水相关路径时，每一次「进一步」都伴随一次「被看见」。看见他的有道统使者、空证骗子、密僧、炼气道说客。他用假命格让一半人看见空白，用真进度让另一半人愿意投资。投资即枷锁：资源进来，条款进来，暗杀也进来。琴如雪在公开叙事里是钉，钉可挂画也可钉人——写她时应保留这种双刃，而不是写成只会担心的旁白。

**【长写·东海尊号】**
「震明离炁持法真君」之类尊号在海湾回响时，方青人在账房。尊号是工具：挡妖潮、挡索贡、挡联姻。真身在另一条潮汐线上洗妖丹。东海宗门把化神当山头柱石，他却把化神当「可调用的天气」。天气不能透露自己从哪片云来。有人要他以尊号入籍某山，他笑着答应附属条款里的「永不入籍」——条款在乙界的公证人那里。

**【长写·帷幕的冷】**
帷幕世界的冷不是温度，是「你不该明白」。呢喃者在墙缝卖半句真名，长生者在灯下买一整段记忆。方青若在此暴露免疫推算，会被当成会走路的样本。他只做三件事：送错的信、回收的样本、假死的档。蚀世主的名字像季节，提起就会换天。低阶契约者在此阶段的正确胜利是「活着离开且忘记一点」。

**【长写·值岁前夜】**
十二月位虚影没有脸，却有农时。它要的不是跪，是「候选人把道生之物放上季节」。方青给假珠，自己去真线索。三息的长度在太古遗址里比在人间长。条件性胜利的写法：虚影被引走，线索到手，人没死，珠没交——而不是虚影跪地求饶。档案在此停笔：连载未完，值哪一月未定。

**【补足·值岁会计学】**
值岁不是职业称号，是年的编制。方青靠近它时，每一界的账本都要重做：哪一月的虚影要假珠，哪一家道统要真丹，哪一条门缝要封死。契约者若只看见「升级」，会在季节里迷路；若看见「科目」，才能与方青同行三息而不死。

值岁之路上，活着本身就是年报。
先活过冬天，再谈值哪一月。

## 阶位切入点

> 阶位↔对照：一阶≈服气/炼气低、炁功练力练劲；二阶≈服气中/炼气中、先天胎息；三阶≈道基/筑基；四阶≈紫府/结丹初；五阶≈金丹稳/元婴边；六阶≈元婴高/化神初；七阶≈化神稳/灵界中高；八阶≈真仙/值岁边缘。完整值岁与蚀世主＝超阶。低阶规避顶点·情报优先；严禁顶点战力归零。

**一阶（流民采气 · 瘟疫尾声）**

切入身份/时点：三水坳外流民队苦力/采药童，锚定方青刚醒记忆、道生珠未稳时。

初始事件：药队要进疫区收「还阳草」，队长要抽签；你抽中，而少年方青用半块面饼换你的签——他要进山，却不只为草。

开场白建议：「土路上的灰比药香还浓。方青把面饼塞进你手里时，眼睛已经在看山那边的雾。他说：你腿伤，别死。你没问他为什么不怕——后来才知道，他怕的是另一种死法。」

关键NPC立场：**方青**（要安静进山，可换人情）；流民队长（要草要人）；老叔职能长辈（要侄子活）；瘟鬼传闻制造者（要钱）。

主线钩子/支线：疫区采药存亡；支线——逃难津渡买票、伪药贩子、第一次见人「服气」。

危险度/规避：中。规避疫区夜宿无防护；勿抢珠。

任务方向/奖励：基础服气口诀残句、草药、流民队身份，不发金丹。

**二阶（服气门墙 · 灵根命格）**

切入身份/时点：外门记名/市坊跑腿，方青灵根命格初显前后。

初始事件：检测台报出「中品水灵根、箕水」，有人当场要收徒，有人当场要绑票；你被夹在中间当证人。

开场白建议：「铜镜照出的不是脸，是命。方青袖里的东西轻轻一烫——你看不见珠，只看见他笑得比谁都乖。收徒帖与绑票绳同时伸过来，你得选站哪边的阴影。」

关键NPC立场：**方青**（要脱身不结死仇）；门派代收（要苗子）；绑匪眼线（要钱）；市坊牙人（要抽成）。

主线钩子/支线：脱身入门或假入门；支线——假灵根药、命格合婚骗局。

危险度/规避：中高。规避当众逼问道生珠。

任务方向/奖励：服气辅药、外门腰牌、水灵材。

**三阶（道基/筑基 · 双界门缝）**

切入身份/时点：道基期外务弟子/大周边境炁功学徒（双身份之一），方青首次稳定穿梭后。

初始事件：你在甲界追债，仇家却从乙界的门缝里摸出来——只有方青知道门在哪；他要你帮忙「堵门」三息。

开场白建议：「墙后不该有海潮声。方青按住你的肩：数三息，把这袋石灰洒出去，别问灰是什么。你数到二时，潮声变成刀声。」

关键NPC立场：**方青**（要门缝机密）；甲界债主；乙界巡逻炁士；偶然撞见的密僧。

主线钩子/支线：堵门换双界人情；支线——返本失败的黑市元炁、假穿梭骗局。

危险度/规避：高。规避跟着冲进未标定的界门。

任务方向/奖励：道基材、两界地图残页、小笔元炁。

**四阶（紫府 · 神通与空证）**

切入身份/时点：紫府境外围护法/神通考巡场，方青开府前后。

初始事件：有人卖「空证金位」名额，买家暴毙于推算反噬；方青要你混进拍卖记录谁在布网。

开场白建议：「拍牌举到一半，那人笑着吐出金粉，然后没了呼吸。司仪说是天道不容伪。方青在你掌心写：伪的不是天，是人。」

关键NPC立场：**方青**（要名单）；拍卖行；空证骗子残党；道统观察者。

主线钩子/支线：名单换箕水线情报；支线——四神通材料劫案、假命格被拆穿现场。

危险度/规避：高。规避购买任何来路不明的金位。

任务方向/奖励：紫府辅剂、神通残篇、反推算香（低阶）。

**五阶（金丹/元婴边 · 箕水位）**

切入身份/时点：金丹境客卿/东海散修盟外务，方青箕水相关金位进度公开化时。

初始事件：缺位争夺的外围祭坛要活祭「水命人」；你与方青的假身份都在名单上。

开场白建议：「祭坛的水是逆流的。司仪念到你的假名时，方青的真名在水底亮了一下又灭。他咬字：今晚我们都得是别人。」

关键NPC立场：**方青**（要保命保位）；金位竞争者；密藏域观礼僧；炼气道说客。

主线钩子/支线：调包祭名；支线——借位契约谈判、篡位谣言战。

危险度/规避：极高。规避主位天仙级下场者。

任务方向/奖励：金丹稳固方、水德真炁中品、客卿合约。

**六阶（化神初 · 东海压强）**

切入身份/时点：化神初期战团参谋/真炁商队总管，方青炼气道化神节点前后。

初始事件：妖兽潮顶着化神压境，宗门要方青出手亮尊号；他要你导演一场「别人的化神」来挡账。

开场白建议：「海平面比昨天高三尺。方青把一枚伪造的令牌给你：今天我是你的副手，你是震字辈的谁。别念错尊号，念错潮会笑。」

关键NPC立场：**方青**（要隐藏上限）；宗门长老（要战果）；妖王（要岸）；真炁商人（要涨价）。

主线钩子/支线：伪尊号挡账；支线——《吞海功》残页争夺、丹师订单战时加急。

危险度/规避：极高。规避与完整值岁遗物共振。

任务方向/奖励：化神辅材、海属真炁、战团股份。

**七阶（灵界/多界权柄边）**

切入身份/时点：灵界临时契约工/道主府外务，方青多界账本已厚时。

初始事件：道主府要清查「免疫推算者」；方青的「如在算中」成为焦点，你需在册上把他做成「已死于帷幕」。

开场白建议：「名册的墨会咬人。你写『卒』字时手在抖，方青只说：写漂亮点，死人值得一笔好字。远处道主的法旨像潮，却没漫过你的假坟。」

关键NPC立场：**方青**（要假死档）；道主府书吏；帷幕通晓者；值岁猎人。

主线钩子/支线：假死档与真货转移；支线——蚀世样本泄漏、灵界洞玄期遗迹门票。

危险度/规避：贴近顶点。情报优先。

任务方向/奖励：灵界通行令、高阶真炁、反蚀药剂。

**八阶（值岁边缘 · 太古残局）**

切入身份/时点：金位联盟观察员/太古遗迹破译员，值岁虚位争夺前夜。

初始事件：十二月位之一的虚影苏醒，要「候选人献道生之物」；方青要你用假珠引开，自己去夺真正的年岁线索。

开场白建议：「虚影没有脸，却有季节。它问你珠在哪。你掏出仿品时，春天在左手，冬天在右手。方青的声音从另一界传来：撑住三息，三息后我还你一个活的年。」

关键NPC立场：**方青**（条件性胜利思维）；值岁虚影；二十八家特使；蚀世主代行。

主线钩子/支线：假珠引开与年岁线索；支线——太古大日失败现场还原、金位合并仪式破坏。

危险度/规避：贴近顶点。**超阶巅峰：存在·情报优先/条件性胜利**；严禁硬刚完整值岁与蚀世主本体；严禁顶点战力归零话术。

任务方向/奖励：值岁相关残页、九阶真炁线索、多界永久门户坐标（叙事向），不发完整值岁权柄。
