# 神采炼金名匠
<!--meta lib=休闲 tiers=休闲-->

## 剧情

**【作品来源】**
《神采炼金名匠》（日文：神採りアルケミーマイスター，假名：かみどりアルケミーマイスター）是日本品牌 **Eushully（エウシュリー）** 于 **2011 年 4 月 22 日** 发售的 18 禁 PC 游戏。官方定位为「知识采集＋工房运营＋战术模拟 RPG」，并非纯 ADV 恋爱作，但情感与角色魅力是叙事核心。原画 **やくり、鳩月つみき、みつき**；剧本 **高杉九郎、八雲意宇、若葉**；音乐 **クワイア**；OP「**Ars Magna**」、ED「**虹の彼方**」由 **オリヒメヨゾラ** 演唱。舞台同属 Eushully 共用奇幻世界 **ディル＝リフィーナ（德尔＝里法纳）**，时间线落在《战女神 VERITA》终盘至初代《战女神》之间。Getchu 2011 上半年销量第 4、全年第 6；萌系游戏大赏 2011 月间赏、大赏银奖。气质：工房日常＋多族共存的温厚奇幻恋爱，主线偏「认真工匠的成长与理解」，H 存在但不取代情感主轴。

**【世界定位】**
这是一座以 **工房都市ユイドラ（尤伊德拉）** 为中心的「物作り日常世界」：新米工匠 **ウィルフレド・ディオン** 继承双亲遗下的工房，与三位女护卫及众多亚人伙伴一起采集知识、接委托、开店、换装、谈心。冲突来自误解、环境、政治阴影与心结，解决靠 **对话、制作、共存与信赖**，不是比武逞强。契约者切入后玩的是 relationship、工房节奏与多族日常，而非杀戮任务。

**【世界观 · 舞台设定】**
时代为德尔＝里法纳世界中的 **拉乌尔巴什大陆（ラウルバーシュ大陸）南方塞特托里（セテトリ）地方**。北方有 **ミケルティ王国連合（米克尔提王国联合）**：二十年前市民推翻王政，七座自治都市组成联合体，经济尚可、对外却脆弱，旧贵族仍暗中活动。尤伊德拉是联合体内陆山岳都市——农业商业不突出，却以 **矿脉与加工技术** 闻名；市内工房林立，工匠资格由 **工匠会** 严格管理，无资格经营工房会受重罚。南侧大国 **ディスナフロディ神権国（迪斯纳芙洛迪神权国）** 人口约二百八十万，文化呈和风武具与瓦葺建筑气质，技术相对滞后，近年饥馑，对扩张抱有危险野心。

超自然设定在本作被日常化：**精灵、水精、木精（ユイチリ）、土精、睡魔（マーテル种）、狐炎兽（サエラブ）、天使、龙族、幻兽、歪魔** 等与人类共存于同一地理圈。主角的工作不是「打怪升级」，而是 **采集知识与素材 → 炼金/工匠制作 → 店铺销售 → 完成市民委托 → 获得评价与资金**。工匠会纳税与生活费构成温和生存压力：资金见底会 GAME OVER，逼玩家把「认真经营」写进人生节奏。女性角色服装可作为工房制品更换，改变外观与部分事件 CG——恋爱与制作系统被同一套「为她做衣服」的逻辑绑在一起。

世界温度来自：认真的人会得到多族信任；误解可以被调查解开；「鬼教官」的严厉背后是共存理想；工房的店铺、居室、工房、庭园四区可摆家具，既提效率也触发温泉、双人床等情感事件。天使侧会讨论人类的「开拓命运之力」，但档案侧只取其 **情感与伦理议题**（人类成长会否伤及自然、监视者如何学会爱），不写数值对抗。

**【地理 · 生活舞台】**
- **ウィルの工房兼住居**：店铺／房间／工房／庭园四区。店铺接客与定价；工房炼金突变与材料节约；庭园可设畑、木、水晶窟等自动采料；居室是谈心与亲密的终点。扩展工房还会解锁称号与更多家具位。
- **工房都市ユイドラ市区**：工匠会、市场、委托板、街巷与市民评价的主舞台。**ロサナ・レビエラ** 作为领主兼工匠会长在此运筹。
- **ティアンの酒場**：光头老板 **ティアン** 的据点；威尔父母故旧，他是失怙后的后见人。**セラウィ** 最初在此与威尔相遇。
- **闘技場（ジェーン管理）**：各色人等进出，情报网发达；**ジェーン** 喜欢强者气质的工匠，对威尔的成长格外上心。
- **セーナル教会（ハンナ）**：奉商业神セーナル的修道女 **ハンナ** 在此卖恢复药，却因工匠城信奉者少而叹气；Append 可加入巡礼故事。
- **ユイドラ鉱山／鉱脈**：资格工匠才能深入；**ユエラ** 为寻姐而必须进入，故与威尔形成「护卫换入场」的利害同盟。深处有神殿与封印相关传说。
- **尖晶樹の森**：**セラヴァルウィ・エンドース** 故乡侧的精灵相关舞台，也是环境问题与共存议题的象征地。
- **シセティカ湖**：**スイナ（水那）** 诞生地；湖畔魔兽对立与「父亲情念」的情感线发生处。
- **近郊の森**：**クレール／クレアンヌ** 姐弟的木精领域；恶作剧驱人与「被反复问理由」的喜剧起点。
- **火山帯・狐炎獣圣域**：**狐伯蓮** 与 **サエラブ（永恒）** 的领地；威尔以诚意而非征服获得承认。
- **山頂廃教会／ミサンシェル聖堂**：**メロディアーナ** 与 **エリザスレイン** 的天使侧舞台；从监视到同行、从敌视到理解的情感弧线。
- **地底湖**：**パラスケヴァス** 栖息处；误解解除后的合作象征。
- **フェマ山脈侧**：**ガプタール** 等龙族与魔流入相关事件的边缘舞台。
- **ディスナフロディ方向的文化阴影**：和风武具、饥馑、扩张压力；**ユエラ** 出身与后期人质/外交相关收束会触及此地。

**【故事主线 · 情感线】**

**〇、开局：新米工匠与三位护卫**  
**ウィルフレド・ディオン**（爱称 **ウィル**）在尤伊德拉长大，父母一年前去世。为继承双亲工房，他通过工匠试验，成为新米工匠。因不善战斗，他需要护卫——于是故事迎来三位命运交汇的女性：

1. **ユエラ（月爛）**：ディスナフロディ出身的放浪剑士，为寻找失踪的姐姐来到尤伊德拉。矿山需工匠资格才能进入，她以护卫换取同行。口拙、义理重、只会用剑表达自己。  
2. **エミリッタ**：贵族令嬢，为逃避政略结婚离家；曾见过威尔父母的名匠风范，循线投靠。明朗任性却努力自立；工匠之路因手拙而挫败，转向魔法。  
3. **セラヴァルウィ・エンドース（セラウィ）**：尖晶树之森的精灵女性，罕见地主张与人类共存。为调查近年环境污浊，在酒場遇见威尔，自愿兼任护卫与教官——训练极严，有「鬼教官」之评。

共通线前半的情感主题是：**三人同住工房的磨合**、接市民委托、采集素材、提升评价、把空屋变成「有人气的家」。威尔认真、思虑深，对敌对者亦先求理解，这一性格成为打开多族心扉的钥匙。

**一、共通线：工房日常与多族连结**  
日常循环：制作商品上架 → 付税与生活费 → 接委托 → 出采地采集/解决事件 → 归来强化装备与女主衣装。伙伴多通过 **本筋外的交流任务** 加入；加入后不仅改变日常对话，也会解锁对应属性配方——「理解她／他」直接等于「能做出新东西」。中盘根据威尔的选择与好感倾向，明确分支 **ユエラ线／エミリッタ线／セラウィ线**；未选中的女主会退让，但工房仍可能保留亚人伙伴的亲密事件（原作机制如此；档案侧强调主线 HE 的情感专一度，亚人线作可选日常彩蛋）。

**二、ユエラ路线**  
情感核：**只会战斗的不器用少女，如何学会被需要、被理解，而不是只被当刀。**  
ユエラ因寻姐执念与矿山准入，与威尔绑定。相处中她话少、直球、讨厌歪理；威尔用行动证明「你可以不只靠剑存在」。路线中后期触及 **ラグスムエナ**——外表酷似姐姐的「死神」灵体；若有其同行，会追加与姐姐执念相关的事件。ディスナフロディ的政治阴影与尤伊德拉的外交压力上升时，**レグナー・アーシェス**（挚友兼竞争对手、ゴーレム使い）可能被作为人质派往ディスナフロディ——本人知情并接受，象征「为城市未来的苦涩选择」。结局 HE：ユエラ在寻姐心结得到面对与安置后，选择留在威尔身边，把「护卫契约」改写成「共同生活的伴侣」；她仍不擅长甜言蜜语，却会用守护与同行表达爱。

**三、エミリッタ路线**  
情感核：**从政略结婚的逃兵，到能自立、能选择爱的女人。**  
エミリッタ表面自由散漫，实则拼命积累经验以免再被家族当棋子。她曾想当工匠却因不器用放弃，魔法成为她的自尊出口。威尔不把她当「麻烦大小姐」，而是认真回应她的好奇心与学习欲：一起做配方、一起见世面、一起面对贵族残留的阴影。路线特色是轻快日常与成长喜剧并重；衣装与制作系统让「为她做衣服」成为求爱语言。结局 HE：她不再只是「逃出来的令嬢」，而是能站在威尔工房里说「我选择这里」的伙伴与恋人；政略婚姻的恐惧被「被当作人对待」的日常治愈。

**四、セラヴァルウィ・エンドース路线**  
情感核：**寿命与价值观不同的精灵，如何把「共存理想」落成对一个人的爱。**  
セラウィ最初以教官姿态保持距离：严格、温柔、经验丰富。环境污浊、人类工房扩张与自然的摩擦，是本线背景议题。威尔以调查、制作环保向道具、调解多族误解等行动证明「人类可以不靠掠夺成长」。她逐渐从「观察对象」变成「想并肩的对象」；鬼教官的严厉里渗出偏爱。结局 HE：两人确认跨越种族寿命差的感情；共存不再是抽象口号，而是每天在工房与森之记忆之间往返的生活。可制作物与可前往地会因本线略有变化（原作机制）。

**五、亚人／伙伴情感支线（非三主线互斥日常）**  
- **アト**：威尔借セラウィ之力制造的土精；因杂质形态不稳、不会说话，却依恋「需要自己」的威尔；除杂后呈女性型，成长为能并肩的存在。  
- **スイナ（水那）**：シセティカ湖水精，由威尔父亲珍视湖泊的情念诞生；人见人羞，称威尔「お義兄さん」；同行确认父亲安否后，心意转向威尔本人。  
- **クレール／クレアンヌ**：木精双胞胎；弟好奇人类、姐想永远守着故乡与弟弟；在「外出成长 vs 留下守护」的矛盾中与威尔建立信赖。  
- **シャルティ**：睡魔マーテル种少女，为「食事」与有妇之夫交涉而遭女性敌视；威尔庇护后她报恩同行，表面无邪地收集光物。  
- **狐伯蓮／サエラブ（永恒）**：仙狐与炎狐；从戏弄、监视到承认威尔的心意；人外社会的面子与神兽威信成为工房的隐形后盾。  
- **メロディアーナ／エリザスレイン**：天使监视者与激进派之长；前者从冷淡到爱慕，后者从敌视人类发展到愿理解「人类力量的源泉」。  
- **ミレーヌ・プロア**：歪魔姫；因遗失物结缘，预告袭击却在败后被威尔用「去学更多乐趣」留下——享乐主义被日常知识改写。  
- **フィニリィ**：精灵王女气质的强大精灵，幻雾戏人却被威尔发明破术，于是好奇同行。  
- **アスモデウス**：色欲魔神（堕天炽天使位阶背景）；以盟约与腕轮试探，最终因威尔对其「为成全神而选择必败」的矜持产生兴趣而借力。  
- **パラスケヴァス／ガプタール／ラグスムエナ** 等：误解解除型合作；ラグスムエナ在ユエラ线有额外情感回响。

**六、城市线与后日谈气质**  
**ロサナ・レビエラ** 物色后继者；威尔因声望与多族人脉被看好，终局可至领主，周回甚至可及「匠王／大匠王」荣誉（系统向，叙事上=被城市托付）。**レグナー** 既是挚友也是路线中的政治棋子与技术镜像。Append／予約特典可追加 **ロカ・ルースコート、リフィア・マーシルン、エヴリーヌ、ハンナ巡礼** 及女仆吉祥物 **エウシュリーちゃん** 等，属周回与彩蛋，不改三条主 HE 的情感结论。

**七、收束原则**  
三条主女线各自 HE：在一起、共同经营工房与城市命运、心结被认真对待。Bad 向来自经营失败（破产）或关键事件处理失败，不是「打不过」。无全员强制后宫 True Route；主线恋人确定后其他主女退让，亚人亲密事件为可选扩展。

**【可攻略角色 / 主要人物】**

**ウィルフレド・ディオン（工房都市ユイドラ新米工匠｜爱称ウィル｜无语音）**  
外貌：青年工匠装束，气质认真清爽（原画やくり）。性格：诚实、勤奋、探究心强，优先用理解与制作解决问题。角色类型：成长型工房主人公／多族调和者。萌点/魅力：把「认真」贯彻到对敌人也先调查误会；工房与恋人衣装的亲手制作。个人线：视三主线分支，终局可成领主级人物。与众人：ユエラ／エミリッタ／セラウィ的雇主兼恋人候补；レグナー的挚友对手；ロサナ眼中的后继苗子；ティアン的被后见人。

**ユエラ／月爛（ディスナフロディ放浪剑士｜护卫｜CV ユリア｜原画やくり）**  
外貌：利落剑士风，战斗向装束；口数少时更显锐利。性格：厌恶歪理、重义理、口拙不器用，除武艺外近乎空白。角色类型：不器用剑士／口拙忠犬／寻亲旅人。萌点：话少却守约；只会用并肩与剑表达心意；寻姐执念。个人线：护卫契约→矿山同行→面对ラグスムエナ与姐姐阴影→ディスナフロディ相关政治苦涩→HE 留在威尔身边。与主角：最初是利害同盟，后成「被允许软弱的归处」。Append 衣装可化用战女神セリカ风（系统彩蛋）。

**エミリッタ（贵族令嬢｜逃亡中｜CV 真宮ゆず｜原画みつき）**  
外貌：明亮大小姐气质，服饰华丽可随工房衣装系统更换。性格：开朗任性、求知欲强、努力自立。角色类型：逃婚大小姐／元气学徒／魔法侧恋人。萌点：手拙工匠 contender 转魔法；对世界的好奇；被认真对待时的柔软。个人线：投靠威尔→共同生活与学习→摆脱政略婚姻阴影→HE 选择工房与爱。与主角：父母名匠的缘→室友→恋人。

**セラヴァルウィ・エンドース（尖晶樹の森精灵｜教官兼护卫｜爱称セラウィ｜CV かわしまりの｜原画鳩月つみき）**  
外貌：端丽精灵，成熟温柔；战斗时弓与魔法并重的印象。性格：平日沉稳体贴，指导时极严（鬼教官）。角色类型：大和抚子型精灵／鬼教官／共存理想主义者。萌点：严与柔的落差；对人类不放弃的耐心；偏爱时的微露破绽。个人线：酒場相遇→教官同行→环境与共存课题→确认跨越种族的爱→HE。与主角：教官→理解者→恋人。

**アト（土精｜工房造物｜CV 春日アン｜原画みつき）**  
外貌：初期形态不稳，后稳定为女性型。性格：依恋威尔，从无言到成长。角色类型：人造生命／成长女儿感伙伴。萌点：被需要而存在；除杂后的自我确立。个人线：被制造→除杂女性化→特训成长→对威尔的慕情。与主角：创造者与被需要的家人。

**スイナ／水那（シセティカ湖水精｜CV 篠原ゆみ｜原画やくり）**  
外貌：水乙女清澈感，常显怯生生。性格：害羞、不疑人、纯粹。角色类型：害羞水精／义理姐姐幻想。萌点：お義兄さん称呼；湖与父亲情念的出身。个人线：被救→确认父亲安否→心动威尔；可遇分裂的冰精事件。与主角：由父缘到本缘。

**クレール（ユイチリ少年｜CV 无｜原画やくり）／クレアンヌ（姉｜CV 夏野こおり）**

## 休闲切入点

> 本世界为休闲/恋爱向（工房日常＋多族情感），无「以武力定生死」的压力。契约者以**日常身份**融入，核心玩法＝relationship 攻略 + 工房经营与委托日常，而非任务厮杀。

切入身份：最自然是 **工房都市ユイドラ的新米工匠／威尔工房的共同经营者或住店学徒／工匠会登录的见习资格者**。也可弱化为「酒場ティアン介绍来的住客帮工」「矿山登记的采集助手」「竞技场珍妮引荐的临时护卫翻译（文职向）」。务必保持逻辑：你靠 **制作、销售、委托、交际** 立足，不靠外挂屠城。若不想顶替 **ウィルフレド・ディオン**，可作「同期合格的邻家工匠」或「寄宿在威尔工房的远亲学徒」，与三女主从「同屋／同事」开始熟络。
