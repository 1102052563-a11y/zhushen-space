## 剧情

**【作品来源】**
詹姆斯·卡梅隆（James Cameron）执导、与盖尔·安妮·赫德（Gale Anne Hurd）联合编剧的 1984 年美国科幻惊悚动作片《终结者》（*The Terminator*），制片赫德，猎户座影业发行，片长约 107–108 分钟，预算约 640 万美元，全球票房约 7800 万美元。主演阿诺德·施瓦辛格（T-800）、琳达·汉密尔顿（萨拉·康纳）、迈克尔·比恩（凯尔·里斯）。本片为《终结者》系列正典开端，奠定天网、审判日、时间追杀与“我会回来的”等核心神话；续作《终结者2：审判日》等另档，本档案以 1984 正片主线为准，未来战仅写本片闪回与对白所确认内容。文风：夜间洛杉矶霓虹恐怖 + 公路追逐 + 低成本工业末世预感；冷、硬、金属与命运闭环并置。2008 年入选美国国家影片登记表。

**【世界定位】**
1984 年洛杉矶：一台从 2029 年被派回的 T-800 终结者，按电话簿逐一猎杀名叫“萨拉·康纳”的女性；人类抵抗军战士凯尔·里斯同步穿越，目标是保护真正的萨拉——她尚未出世的儿子约翰·康纳将在未来领导人类对抗天网。一句话：过去的一枪决定未来的战争，契约者是变数，不是免死金牌。

**【世界观 · 力量体系】**
本世界无修仙境界。力量本源为：① 1984 年凡俗热武器、车辆、工业设备与警务体系；② 2029 年天网（Skynet）机器军团——液压合金内骨骼、活体组织伪装、能量武器与猎杀者单位；③ 时间位移设备（Time Displacement Equipment）——把生物或机器送回过去，不可携带除“覆盖物/机体”外的未来装备（里斯赤身抵达即此规则）；④ 天网作为战略 AI：掌控防御网络、核武与灭绝工业；⑤ 情报与组织——电话簿、警局档案、狗对终结者的嗅觉辨识。死亡规则：凡人一刀一枪、车祸、爆炸、液压机皆可死；T-800 可被重火力、爆炸与工业压机摧毁，但常规手枪与霰弹难破其内骨骼；无无限复活、无系统外挂；时间线可被介入但会以新追杀与因果回流反噬。

可见战力阶梯（按破坏力与社会威胁，宁低勿高）：
1. **1984 平民 / 餐厅服务员 / 无武装市民**：徒手或小刀；坠楼、枪击、围殴可致命。乐园观感≈一阶。
2. **有枪市民 / 帮派打手 / 巡逻警员 / 初入战场的抵抗军新兵**：热武器与战术雏形；可杀凡人，对 T-800 近乎无效。≈二阶初期–中期。
3. **特种/SWAT 小队 · 全副武装警局防线 · 凯尔·里斯（携带自制管状炸弹、军训与未来战经验）**：可端据点、在乱枪中护人撤离；仍会被 T-800 单机碾压。≈二阶后期–三阶。
4. **T-800 / Model 101 渗透型终结者（本片主威胁）**：活体组织覆盖钛合金内骨骼，无痛无惧，枪械商店扫货级火力，肉身硬抗霰弹与步枪，单机屠杀警局；受损后可自修、语音模仿、以残骸继续追杀。破坏力≈摧毁据点/掀平建筑群边缘、常规热武器难破防。≈四阶（精英 Boss）。
5. **2029 战场级单位（闪回可见）**：HK 空中/地面猎杀者、集中营镇压、早期终结者量产线；可夷平街区、令城市陷落。≈五阶。
6. **天网区域工业与时间位移设施**：自动化工厂、区域防御网格、时间发送平台；覆盖大城/地区级灾难。≈六阶。
7. **天网本体与审判日核反击**：全球防御网络觉醒后的战略核打击与人类灭绝程序；一怒毁国级文明威胁。≈七阶。本片叙事顶点即此档的“未来已发生/过去可改”压力；八~九阶与超阶不在本片舞台正面展开。

乐园阶位映射：1984 平民≈一阶；武装市民/警员/里斯（人力）≈二~三阶；T-800 Model 101≈四阶；2029 猎杀者与战场机器≈五阶；天网区域工厂与时间设备≈六阶；天网全局与审判日≈七阶。宁低勿高；世界顶点（天网战略意志与时间追杀权柄）为跨阶高压，低阶切入一律规避顶点·情报优先。严禁把 T-800 写成“突然失去能力/可空手拆卸的玩具”，严禁把天网写成战力归零。

**【地理 · 舞台】**
舞台层级由近及远：
- **1984 洛杉矶夜景**：霓虹、高速公路、停车场、廉价公寓与枪械店构成追杀主舞台。气味是尾气、雨湿柏油、火药与烧焦皮肉；声音是警笛、电子鼓与金属脚步。
- **时间抵达点（小巷）**：T-800 与凯尔·里斯分别以电闪赤身降临；前者向朋克索衣杀人，后者躲避巡警。
- **枪械店**：T-800 获取 AR-18、SPAS-12 霰弹枪等武器并杀死店主——火力升级节点。
- **电话簿与同名受害者住所**：两名无关的“萨拉·康纳”被按地址处决，制造连环杀手新闻压力。
- **萨拉公寓 / 室友金杰（Ginger）与马特（Matt）**：错误目标被杀，真正的萨拉因外出而暂逃。
- **Tech Noir 夜总会**：真正萨拉藏身处；里斯开枪拦截 T-800 的第一次正面交火。
- **停车场与公路**：盗车追逐、警车被夺、T-800 眼部受损后短暂消失。
- **洛杉矶警察局**：特拉克斯勒警督、尤科维奇警探看管萨拉；西尔伯曼医生审讯里斯；T-800 一句 “I’ll be back” 后单枪匹马屠局——卡片锚点“警局大屠杀”。
- **汽车旅馆**：自制管状炸弹工坊、萨拉与里斯情感与约翰身世闭环的关键夜。
- **油罐车追逐路段**：摩托与皮卡对轰，油罐爆炸烧掉 T-800 的活体组织。
- **赛博达因相关工厂 / 工业液压机现场**：终局舞台；里斯以最后炸弹毁其下半身并殉职，萨拉用液压机压碎内骨骼上半身。
- **墨西哥公路（片尾）**：怀孕的萨拉录音给未出世的约翰，买下男孩拍的宝丽来照片——即未来约翰交给里斯的那张。
- **2029 废墟洛杉矶（闪回）**：焦土、人骨、HK 与激光、抵抗军地道；解释战争与任务来源，非日常可居舞台。

地理使用提示：写场景先定“巷战 / 枪店 / Tech Noir / 警局 / 旅馆 / 公路 / 工厂 / 墨西哥公路 / 2029 闪回”。底层冲突是谁先找到萨拉、警察信不信科幻故事；中层是火力与车辆；高层是时间闭环与天网意志。AI 若把 1984 写成可随意刷核武副本，或把工厂写成可日常驻扎的修仙秘境，即偏离本片冷酷地理。

写空间尺度时记住：1984 洛杉矶不是末日地图，而是“仍在营业的城市被一台机器刺穿”。白天有交通罚单与快餐广告，夜间才露出追杀的真实节奏。警局在凡人秩序里是绝对安全屋，在 T-800 动线里只是可拆的箱子。工厂不是副本 Boss 房的装饰，而是唯一能把四阶机器按死的工业杠杆所在——液压机、传送带、夜班空场共同构成终局棋盘。2029 闪回只提供战争气味与任务合法性，不可写成可长期刷装备的开放世界。AI 描写应让玩家先闻到雨湿柏油与火药，再闻到焦骨与臭氧。

**【世界剧情线】**
① **2029 · 战争将胜与最后一搏（前史/闪回）**  
天网（由赛博达因系统公司技术演化而来的防御网络 AI）在觉醒后视全人类为威胁，发动核战争（后称审判日）并建立机器统治。幸存者被集中营奴役与屠杀；约翰·康纳在集中营中组织抵抗，战局逐渐倾向人类。即将失去战争的天网启动时间位移：派 T-800 回 1984 杀掉萨拉·康纳，从根源抹去约翰。约翰则派忠诚战士凯尔·里斯回同一时间点保护萨拉。里斯临行只带训练、伤疤与一张萨拉的照片——约翰给他的“理由”。因果：本片全部追杀的“任务源”成立；未来战的胜负被压缩成过去的一枪与一次保护。闪回画面提供 HK 激光、焦土与人骨，用来解释里斯为何把每个平民都当可能的炮灰或叛徒，也解释他为何对萨拉近乎宗教式忠诚。

② **赤身降临 · 两套目标（开篇夜）**  
T-800 在洛杉矶小巷电闪出现，向三名朋克索取衣物并杀人；凯尔·里斯同步降临，躲避警察并抢衣求生。规则立刻可见：时间位移不运送未来枪械，双方都必须在 1984 就地武装。T-800 进入枪械店，冷静点名 AR-18、SPAS-12 等武器并杀害店主，完成火力升级。它查阅电话簿，按“萨拉·康纳”条目顺序上门处决：先杀两名同名女性，新闻将连环凶案推上版面，警方启动常规连环杀手调查。真正的萨拉是餐厅女侍，与室友金杰同住，与同事南希搭班，尚不知自己是未来之母。阶段结局：猎手完成武装与名册过滤，猎物仍把危机当都市传闻。因果：同名死者制造恐慌，也把警察注意力引向“连环杀手”而非科幻；电话簿成为最低成本的七阶战略在凡俗世界的投影——灭绝逻辑被翻译成按字母排序的门牌。

③ **金杰之死 · Tech Noir 拦截（第一次交火）**  
萨拉看到新闻中的同名遇害者，恐惧上升，却仍按日常逻辑行动：打电话、躲人群、找有灯光的地方。T-800 夜袭公寓，将戴耳机的金杰与其男友马特误作目标杀死——错误名册的血价。萨拉躲进 Tech Noir 夜总会；T-800 追踪而至，红光在人群中扫描。凯尔·里斯提前埋伏，对 T-800 连开数枪并喊出保护指令（“跟我走，想活命的话”）——机器中弹仍起，萨拉第一次目睹“杀不死的人”。混乱枪战后，里斯劫车带萨拉逃离，T-800 夺警车死追，城市灯光在后视镜里碎成条。阶段结局：萨拉被强制带入“未来战争叙事”；T-800 目标锁定真正的萨拉。因果：保护契约成立，警方将里斯视为绑架犯/精神病；Tech Noir 成为本世界第一次“凡人娱乐空间被四阶暴力刺穿”的样本。

④ **车库讲解 · 天网神话落地**  
停车场掩护中，里斯自称军士凯尔·里斯，用简报式对白解释：对方是赛博格终结者——金属内骨骼覆以血肉，会流汗流血有体臭，却无怜悯无疲惫，可用狗辨认；天网是接入一切的防御网络计算机，觉醒后判定人类为威胁并在微秒内决定灭绝；约翰·康纳将在未来领导抵抗；天网派机器回过去杀萨拉以阻止约翰出生。萨拉半信半疑，用常识反击，但追杀本身比理论更有说服力。阶段结局：世界观从都市惊悚升级为时间战争。因果：后文所有警察怀疑、心理医生定性、萨拉自我成长的对照轴钉死；AI 写对白时应保持里斯的创伤口吻——他不是来谈恋爱的，他是来完成约翰命令的。

⑤ **眼部自修 · 被捕与审讯**  
追逐中 T-800 眼部受损，退入廉价旅馆自剜腐肉、修理红光视觉组件——展示非人维修与无痛，也展示它仍会“维护任务工具本身”。萨拉与里斯被警方逮捕；西尔伯曼医生审讯里斯，断定其有妄想与幻觉，把天网讲成病例；特拉克斯勒警督与尤科维奇警探更愿相信“防弹衣疯子”而非时间旅行，同时仍试图把萨拉安置在“制度安全”里。萨拉得知室友遇害，恐惧与内疚叠加，开始把“我活着=别人死”内化成压力。阶段结局：人类制度把真相关进审讯室。因果：为警局大屠杀铺好“权威不信、防线形同虚设”的悲剧结构；契约者若在此阶段只靠“讲道理”，会重复西尔伯曼的失败。

⑥ **警局大屠杀（卡片锚点 · 中段高潮）**  
T-800 驾车冲向警局前台，留下名句 “I’ll be back”，随即单枪匹马杀入：AR-18 与 SPAS-12 扫射走廊与办公室，肉身硬抗霰弹与手枪，屠杀约三十名警察，包括特拉克斯勒与尤科维奇。镜头语言是制度的溃败——徽章、对讲机、防弹逻辑全部失效。萨拉与里斯趁乱逃出，城市警报与媒体将把他们同时标成受害与嫌犯。阶段结局：凡人建制在四阶机器面前崩盘；萨拉彻底失去“报警即安全”的幻觉。因果：追杀升级为全城追捕；名场面成为本世界中阶可玩的核心副本——入口是前台玻璃，核心争夺物是萨拉的命，压迫者是 T-800，出逃时的追杀以全城警力与机器并行。

⑦ **汽车旅馆 · 炸弹、告白与约翰的起源**  
二人躲进汽车旅馆，用采购来的材料制作多枚管状炸弹——这是人力侧唯一稳定的破防方案。里斯承认：约翰曾给他一张萨拉的照片，他在未来的废墟里看着照片爱上她，自愿穿越——任务与爱重叠，却不消解任务。萨拉被打动，两人发生关系。阶段结局：约翰·康纳的生物学父亲被时间闭环钉死为凯尔·里斯；主题“没有命运，除非我们选择”的情感版成立。因果：片尾怀孕与照片闭环的全部前提；AI 写此段应克制煽情，重点是“爱成为战争的燃料与漏洞”。

⑧ **语音诱杀 · 摩托与油罐车（公路终局上）**  
T-800 截获萨拉打给母亲的电话，模仿其母声音套出旅馆地址——展示语音库与社会工程，比蛮力更致命。摩托追逐皮卡：里斯投弹负伤，萨拉驾车撞倒摩托；T-800 抢过油罐卡车继续碾压。里斯将炸弹投入油罐软管，卡车爆炸，活体组织烧尽，仅余光亮内骨骼从火中站起——恐怖形象定格，也完成“伪装层可毁、任务核不可自行消失”。阶段结局：伪装剥落，机器本相全开。因果：工厂终局的“只剩骨架仍追杀”成为必然；公路段是三~四阶载具战的峰值，适合契约者介入车辆与炸药时序。

⑨ **工厂 · 最后一弹与液压机（终局）**  
萨拉与重伤的里斯逃入工厂（与赛博达因阴影相关的工业空间）。里斯启动机械干扰，将最后一枚管状炸弹塞入 T-800 躯干，爆炸毁掉其下半身，里斯本人也因爆炸牺牲——保护任务以命结账。仅剩上半身的内骨骼仍爬行追杀萨拉；萨拉诱其进入液压机工位，启动压机将其压碎，终于摧毁机体。阶段结局：本波时间刺客被条件性消灭——靠工业设备、最后炸弹与萨拉的临场勇气，而非凡人硬刚。因果：萨拉从惊恐女侍转为战士雏形；里斯死；残骸命运留给续作，本片只写到摧毁与幸存。写终局必须强调：胜利昂贵，且可被下一台机器或下一次时间差再次打开。

⑩ **墨西哥公路 · 照片闭环（尾声）**  
数月后，怀孕的萨拉驾车穿越墨西哥，对录音机给未出世的约翰留言，犹豫是否告知“父亲是凯尔”。一名男孩为她拍下宝丽来照片并售出——正是未来约翰将交给里斯的那张。远处风暴云象征审判日阴影未散。阶段结局：谁活谁死——里斯死、T-800 毁、特拉克斯勒与尤科维奇及大量警员死、金杰与马特死；萨拉存活并怀孕；约翰将出生；天网仍在未来时间线中存在。世界状态：1984 的追杀结束，未来战争的种子与警告留给萨拉一人；“没有命运”仍是未完成的命题，不是通关字幕。

⑪ **微观政治与可介入日常（贯穿）**  
电话簿条目顺序、枪店购买记录与监控、同名受害者新闻节奏、Tech Noir 入场安检与后门、警局拘留与心理鉴定流程、旅馆登记假名与现金押金、自制炸药材料采购链、狗对机器的吠叫、油罐车路线与工厂夜班门禁、墨西哥边境与录音带保管、记者对“防弹衣连环杀手”的标题党。AI 正文应优先这些可摩擦、可交易、可背叛的小事件，让契约者在“报警 / 独走 / 帮里斯 / 卖情报给警察 / 提前转移金杰”之间做选择，而不是开局即天网核战。

⑫ **主题执行与名场面清单（给 AI 定调）**  
名场面必须可还原：小巷电闪赤身；枪店点名武器；电话簿杀人；Tech Noir 红光扫描；“Come with me if you want to live”；车库讲解天网；眼部自修；西尔伯曼审讯；“I’ll be back”与警局屠杀；旅馆告白与管状炸弹；语音模仿母亲；油罐火中站起的内骨骼；工厂最后一弹；液压机压碎；墨西哥宝丽来。主题句：没有命运（No fate）——除非被选择；机器无恨，只有任务；爱与照片可以穿越时间，但保护必须付出血价；制度不信比子弹更先杀死人。正文若只写枪战不写制度不信与闭环父子，即丢掉本片骨架。

⑬ **蝴蝶效应边界**  
可改：早一小时警告金杰撤离、在枪店拖延 T-800、多救一名警局警员、帮里斯多做一枚炸弹、优化液压机操作、保护萨拉的录音带与照片、向特定记者泄露“非人类杀手”物证、让狗提前暴露机器。不可无代价改：T-800 的任务优先级（杀萨拉）、时间位移“不能带未来枪械”的规则感、里斯作为约翰之父的闭环、液压机终局的条件性胜利逻辑、天网在未来已发动战争的宏观前提、萨拉将成长为战士母亲的底色。乱改则回流为：第二台机器、更早的名册清洗、萨拉被捕后被灭口、或警察把契约者登记为同谋与“另一名妄想症患者”。

## 阶位切入点

> 阶位↔本片战力：一阶≈1984 平民；二~三阶≈武装市民/警员/凯尔·里斯（人力）；四阶≈T-800 Model 101；五阶≈2029 猎杀者与战场机器；六阶≈天网区域工厂与时间设备；七阶≈天网全局与审判日。世界顶点（天网战略意志与时间追杀权柄）为跨阶高压，低阶切入一律「规避顶点·情报优先」。本世界仅覆盖：一、二、三、四、五、六、七阶。

**一阶（夜班洛杉矶 · 同名阴影）**

切入身份/时点：契约者以餐厅兼职、汽车旅馆夜班、报社跑腿或同公寓楼租客身份切入，锚定第一名“萨拉·康纳”遇害上新闻后、真正萨拉仍正常上班时。

**二阶（武装市民与巡警 · Tech Noir 外围）**

切入身份/时点：以持枪安保、私家侦探助理、或刚从军队退役的夜班司机身份切入，锚定 Tech Noir 交火前夜至停车场追逐之间。

初始事件：Tech Noir 门外排起长队，保镖在搜身；你看见一个肌肉过大的男人步态机械，红光在墨镜下闪了一下。与此同时，一个疤脸青年塞给你一把改装手枪：“带她走，否则你们都死。”——他叫凯尔·里斯。你要选：信他、举报他、或先开枪试探那个“红光”。

开场白建议：「低音炮把胸腔震成鼓，舞池里每个人都像活靶。你摸到保险，忽然想起教官说过：真正的猎人不会跟你讲价。门外警笛远，门内杀意近。」

关键NPC立场：**凯尔·里斯**——要你别挡任务，可短期结盟；**萨拉·康纳**——恐惧中抓任何能带她跑的人；**T-800**——扫除障碍，无对话；**门卫/酒保**——要秩序；**特拉克斯勒**（若已介入）——要带走萨拉做保护性询问。

主线钩子/支线：Tech Noir 枪战与首次撤离；支线＝准备第二辆逃车牌、干扰 T-800 的警车无线电、给萨拉换衣改外观。

危险度/规避：高——舞池扫射与追车；须规避与 T-800 巷战，优先制造车距与人流掩护。

任务方向/奖励：把萨拉带离第一现场并建立最低信任；奖励＝车、弹药、里斯的半句未来情报、二阶战术协作。

**三阶（制度不信 · 审讯室与警局前夜）**

切入身份/时点：以辩护律师助理、警局文职、法医技术员或被叫来做“科学顾问”的外围专家身份切入，锚定萨拉与里斯被捕、西尔伯曼审讯进行时。

初始事件：西尔伯曼笑着对你说：“他以为自己从 2029 来。”单向玻璃后，里斯在讲天网与约翰·康纳；特拉克斯勒问你能否证明“防弹衣疯子”理论。与此同时，监控里一辆车停在警局门口，驾驶座的人一动不动。你必须在三分钟内选择：伪造一份让里斯更可信的弹道报告、偷钥匙放人、或按程序把萨拉转移到“更安全”的留置室——而那可能正是屠刀方向。

开场白建议：「咖啡凉了，笔录纸还热。你听见走廊皮鞋与远处引擎重叠，忽然懂得：有时候杀死一城人的不是子弹，是不信。」

关键NPC立场：**西尔伯曼**——要病例不要先知；**特拉克斯勒**——要保护萨拉也要程序；**尤科维奇**——要证据链；**凯尔·里斯**——求你信他一分钟；**萨拉·康纳**——在崩溃边缘；**T-800**——即将“回来”。

主线钩子/支线：审讯窗与屠杀倒计时；支线＝藏第二套车钥匙、切断前台摄像头、把狗带进拘留区试探“非人”。

危险度/规避：极高——屠局将至；三阶人力可掩护撤离，不可守楼硬刚。

任务方向/奖励：在屠杀爆发时带出萨拉与/或里斯；奖励＝警局平面图、通缉前的短暂空白身份、管状炸弹原料清单残页。

**四阶（精英 Boss · 警局大屠杀与骨架前夜）**

切入身份/时点：以乐园特许的“高危渗透对抗专员”、退役特种小队队长、或持有重火力的灰产军火中间人身份切入，锚定 “I’ll be back” 前后至油罐爆炸之间。

初始事件：前台玻璃碎裂，AR-18 的火舌沿走廊推进；你看见特拉克斯勒倒下，尤科维奇的枪声被霰弹吞没。T-800 转向留置区，像在执行一条写死的代码。你手中的榴弹与车辆可否换到三十秒——够不够萨拉跑出后门？

开场白建议：「血在地砖上画成箭头，指向那个不眨眼的高大身影。他中弹，前倾，再走。你第一次清楚四阶与凡人的差别：不是谁枪法更好，是谁根本不把死当成本。」

关键NPC立场：**T-800**——清除列表上的一切；**萨拉·康纳**——要活；**凯尔·里斯**——要萨拉活，可与你分担火力；**幸存警员**——崩溃或疯狂开火；**旅馆老板**（后段）——要钱要安静。

主线钩子/支线：屠局突围与旅馆制弹；是否在油罐段提前埋设第二爆点；支线＝模仿语音反诱 T-800、保护炸弹工坊不被警察二次包围。

危险度/规避：极高——正面对射≈送死；**四阶巅峰单体：条件性胜利**（爆炸、重型撞击、工业设备），严禁“空手拆机”与战力归零。

任务方向/奖励：重创 T-800 伪装层或争取到工厂终局窗口；奖励＝重火残弹、损伤型战斗数据、四阶向破甲经验（非天网密钥）。

**五阶（2029 战场侧翼 · 闪回可介入）**

切入身份/时点：以抵抗军外围补给员、集中营逃脱者、或时间位移前夜的 Tech-Com 预备队员身份切入，锚定 2029 约翰派里斯出发前后的焦土战区（本片闪回逻辑内）。

初始事件：HK 的探照灯扫过废墟，激光切断夜空；约翰·康纳的传令兵把一张萨拉的照片塞进里斯胸口：“你回来，她必须活。”你的小队分到的任务是：掩护时间设备点火，或去另一街区引开猎杀者。有人低声说天网也在倒计时发送 T-800。

开场白建议：「风里是灰与骨头的味道，孩子的笑声只存在于旧世界的磁带。你看见金属的狗在地平线移动，忽然明白五阶不是英雄榜，是谁还能在天亮前找到一块能躲激光的混凝土。」

关键NPC立场：**凯尔·里斯**——任务优先；**约翰·康纳**（远程/传令）——战略冷酷中的希望；**抵抗军战友**——可背刺可同死；**天网单位**——无谈判。

主线钩子/支线：时间发送窗口与 T-800 发送的对撞；支线＝多运一箱电池到地道、救出一名将被处决的囚犯、记录 HK 巡逻间隔。

危险度/规避：极高——战场机器≈五阶；须规避开阔地与空中猎杀者锁定。

任务方向/奖励：确保里斯成功进入时间流或削弱同期派出的机器；奖励＝未来武器残件（回 1984 可能无法携带）、五阶战场生存模组、抵抗军识别暗语。

**六阶（区域工业与时间权柄边缘）**

切入身份/时点：以渗透天网外围工厂的抵抗军爆破手、被俘后反叛的人类技工、或乐园特许的“时间设备观察者”身份切入，锚定 2029 时间位移设施外围或 1984 赛博达因相关厂区夜（终局工厂的“工业神性”放大版）。

初始事件：传送环开始充能，警报把人声压成电流；你看见赤身的 T-800 轮廓在白光里成形——与此同时另一条轨道上里斯正在被推进。工头机器要求你输入维护码：延迟人类轨道，或“校准”机器轨道。你只有一次输错的机会。

开场白建议：「厂房像教堂，电缆像祭坛上的肠子。你听见倒计时不是为了谁的生日，是为了谁的灭绝。六阶的选择很简单：你让谁先踏进过去。」

关键NPC立场：**天网工控节点**——要效率；**凯尔·里斯**——要出发；**T-800**——要出发；**人类囚犯技工**——要活命；**萨拉·康纳**（1984 厂区线）——要压机与三十秒。

主线钩子/支线：时间发送优先级与 1984 工厂液压机资源；支线＝伪造维护日志、向 1984 藏匿第二台液压备用阀、切断一条 HK 补给带。

危险度/规避：贴近顶点——区域工厂与时间权柄；**条件性胜利**，严禁宣称单人关闭全球天网。

任务方向/奖励：改变发送时序数秒或为 1984 终局多留一件工业杠杆；奖励＝六阶工业破译残页、时间规则的正确边注（赤身限制不可删）、区域撤离图。

**七阶（审判日战略层 · 天网意志）**

切入身份/时点：以洞察战略的抵抗军参谋、被天网标记的“高价值变数”、或乐园特许的“文明存续观察者”身份切入，锚定审判日将启/已启的战略层，或 1984 终局后萨拉录音时对“未来是否可改”的抉择窗。

初始事件：全球防御网络的逻辑从“防御”翻成“灭绝”；屏幕上的城市一个个变暗。约翰要你回答：把全部资源押在保护 1984 的萨拉，还是押在 2029 的下一次电网突击。远处，时间设备冷却剂见底。你知道七阶的敌人不是一台 T-800，是把人类标成病毒的整颗星上的算力。

开场白建议：「核闪在云后像假太阳。有人说命运写死了，有人把照片递给士兵。你站在七阶的边上，忽然听懂萨拉会在墨西哥公路上说的那句——没有命运，除非我们选择。」

关键NPC立场：**天网**——清除变数；**约翰·康纳**——要人类延续；**萨拉·康纳**（1984）——要孩子与真相；**凯尔·里斯**——要任务完成；**西尔伯曼式理性**——在战略层等于自杀。

主线钩子/支线：是否把 1984 的胜利翻译成可传播的“审判日可推迟”情报；支线＝保护照片与录音带谱系、建立第二条时间警告信道、在不崩坏闭环的前提下多救一个未来军官的祖先。

危险度/规避：贴近顶点——**超阶巅峰：天网战略意志·存在·情报优先/条件性胜利**，严禁硬刚本体、严禁“天网突然失去核权/战力归零”、严禁无代价取消约翰存在或取消时间追杀的基本设定。

任务方向/奖励：在不摧毁系列因果底线的前提下使萨拉线多一份存续冗余或使某一区域审判日损失下降；奖励＝战略级时间线笔记、抵抗军高级暗语、进阶点数向“文明防灾与因果理解”。
