# 鹿鼎记
<!--meta lib=主库 tiers=一、二、三-->

## 剧情

**【作品来源】**
金庸（查良镛）最后一部长篇武侠小说，1969年10月24日至1972年9月23日连载于香港《明报》，共五十回，回目集自清代诗人查慎行《敬业堂诗集》对句，已完结。新修版通常分五册，约百万字量级。背景锚定清初康熙年间（约1668–1689），历史事件（明史案、鳌拜专权、三藩之乱、雅克萨之战、《尼布楚条约》、俄国索菲亚摄政等）与虚构情节交织。文风反传统侠义、市井厚黑、反讽权谋，主角非武功英雄而是机缘小人。关联《碧血剑》人物（九难即长平公主阿九等）。**以小说正文为准**。全本可核：[搜笔趣阁·鹿鼎记书页](https://www.sobqg.com/book/ludingji/)（金庸武侠·在线分章）。

**【世界定位】**
清初江山：宫廷权斗、反清秘密会党、邪教海岛、边陲罗刹与台湾郑氏，四线并进。主角**韦小宝**生于扬州丽春院，武功低微却凭机变、义气、运气与厚黑学周旋康熙、天地会、神龙教之间，官至鹿鼎公，最终忠义难两全携七妻归隐。一句话：市井小人物的权谋江湖、历史缝隙里的双面间谍、鹿与鼎的权力隐喻。

**【世界观 · 力量体系】**
本世界无仙道飞升，力量来源为**内功真气 + 招式武学 + 奇门毒功/暗器 + 朝廷军权与情报**。生死残酷：一刀一枪可致命，重伤靠医术与运气；无系统面板、无复活。韦小宝一系的“战力”往往不靠硬碰，而靠**宝衣、匕首、迷药、化尸粉、人情与谎言**。

江湖与宫廷通行战力分层（非原作官方境界名，按实际破坏力归纳）：
1. **市井凡夫 / 宫女太监 / 普通兵丁**：徒手制民、冷兵器格斗，遇高手即溃。≈一阶初期。
2. **绿林响马 / 会党普通兄弟 / 侍卫精锐**：能打能跑，单挑小队士兵，无内家亮点。≈一阶中后期。
3. **门派弟子 / 香主骨干 / 神龙教中层 / 一等侍卫**：内力初成，裂石断金级，乱军取首。如**徐天川**、**风际中**、**多隆**等。≈二阶。
4. **一流高手 / 教主级副手**：成型功法、可远程气刃级，端据点、压一场。如**吴六奇**、**何铁手**、**胖头陀/瘦头陀**、**陆高轩**等。≈二阶后期～三阶初期。
5. **当世顶点（人间武学极限）**：**陈近南**（天地会总舵主）、**九难**（铁剑门·神行百变传人）、**洪安通**（神龙教主·豹胎易筋丸异功）、**归辛树**夫妇、**冯锡范**（一剑无血）等。可裂石断金、乱军取首、踏平山寨级，**未达夷平城池、未达四阶质变**。

关键功法与保命手段：
- **神行百变**：铁剑门绝技，木桑→袁承志→九难一系，步法变化多端，擅闪避逃命。韦小宝学得最认真的一门（因便于跑）。
- **百胜刀法**：美刀王**胡逸之**刀法，狠辣迅捷。
- **英雄三招 / 美人三招**：神龙教教主夫人**苏荃**所授，韦小宝略会皮毛。
- **大擒拿手 / 大慈大悲千叶手**：太监**海大富**所授，韦小宝半吊子。
- **神龙教丹药（豹胎易筋丸等）**：催功却埋心性与毒控隐患；教众受三尸类控制逻辑约束。
- **韦小宝四宝**：**金丝软甲/宝衣**（刀枪难入，替康熙挡九难一剑）、**鳌拜匕首**（削铁如泥）、**化尸粉**（灭迹）、**迷药/蒙汗药**（市井出身惯用）。

乐园阶位映射（对照《阶位战力图鉴》，宁低勿高）：市井/太监/普通兵丁/韦小宝本尊≈**一阶**；会党精英/一流侍卫/门派弟子/神龙中层≈**二阶**；陈近南、九难、洪安通、归辛树、冯锡范等当世顶尖≈**三阶**。**世界峰值≈三阶**，无四阶以上；韦小宝本人武功≈一～二阶初期，靠权谋与宝物生存。

**【地理 · 舞台】**
舞台为清初中国与北境罗刹、台湾郑氏之交汇网络，以宫廷—会党—邪教—边陲为四极，交通靠漕运、驿道、海船与会党地下线：
- **扬州丽春院**：韦小宝出身地；母亲**韦春花**（亦称韦春芳）接客处；衣锦还乡与“大被同眠”收妻名场面；全书首尾市井气的根。瘦马、茶楼、运河纤夫构成其性格底色。
- **京师紫禁城 / 康亲王府 / 韦府**：康熙与“小桂子”结义舞台；擒鳌拜、假太后线、建宁公主线、炮轰韦府皆在此展开。宫墙内是戏台，宫墙外是刑场。
- **庄家（湖州南浔）**：明史案遗眷；**庄三少奶**赠双儿；神龙教追杀与地图线交汇点；文字狱创伤的空间象征。
- **五台山清凉寺 / 少林寺**：顺治出家为**行痴**；喇嘛围攻；韦小宝剃度掩护；康熙父子相认。佛门清静与政治追杀并置。
- **神龙岛（蛇岛）**：神龙教总坛；白龙使任命；苏荃怀孕引爆内乱；洪安通与教众互噬。孤岛即密闭权斗实验室。
- **云南平西王府**：吴三桂势力；建宁和亲吴应熊；**陈圆圆**揭阿珂身世；九难制服吴三桂。藩王坐大的西南棋眼。
- **鹿鼎山 / 雅克萨**：满清宝藏与“龙脉”埋藏传说地；罗刹要塞；尼布楚谈判前哨。书名“鹿鼎”的地理落点。
- **莫斯科**：索菲亚公主政变；韦小宝以戏文知识自比“罗刹诸葛亮”。中西权谋对照镜。
- **通吃岛（书中后称与钓鱼岛相关）**：七妻儿女逍遥数年；陈近南遇害后的暂栖；历史大势外的桃花源假象。
- **台湾延平王府**：郑克塽、冯锡范；陈近南被暗算；施琅平台的历史线。正统空壳与背叛。
- **大理（终局隐居地）**：韦小宝携家归隐；康熙六下江南寻访无果。主动退出历史叙事。
- **蒙古 / 西藏接点**：**噶尔丹**、**桑结**与韦小宝结义，边陲政治筹码；盟约随国运起伏。

**【世界剧情线】**

### 一、明史案与入宫（开局·扬州→京师）
清初文字狱阴影下，**庄廷鑨**《明史辑略》案牵连甚广，鳌拜一系借机高压，江南士绅人头落地，市井亦人心惶惶。扬州丽春院少年**韦小宝**市井油滑，骰子、骂街、顺手牵羊无一不精，却在官兵追杀中救下江湖人**茅十八**——一半是义气，一半是听戏听多了“英雄救好汉”。二人北上京师，口称要去“打鳌拜”，实则韦小宝连鳌拜长什么样都不清楚。

尚膳监总管太监**海大富**将二人擒入宫。韦小宝以毒致盲海大富、杀死小太监**小桂子**并冒其名，再设计放走茅十八。宫中规矩森严，他却用丽春院练出的察言观色活下来：谁贪银子、谁怕责打、谁爱听评书，一一摸清。结识自称“**小玄子**”的少年与刁蛮的**建宁公主**——小玄子实为康熙帝玄烨。韦小宝以骰子赌博、戏文段子、市井话术讨得欢心，成为康熙心腹玩伴；建宁则以拧耳朵、打骂为亲昵，两人孽缘从此结下。

康熙欲除权臣**鳌拜**（满洲第一勇士，辅政专权、杀苏克萨哈等）。韦小宝出谋：以少年侍卫布下戏局，假作玩闹诱鳌拜入套，绳索布袋齐下生擒。名场面：布袋擒鳌拜、君臣少年来往如兄弟、宫灯下分食点心论国事。阶段结果：韦小宝站稳“小桂子”身份；康熙初掌实权；鳌拜下狱。世界状态：宫廷权柄易手，江湖与会党尚未正式登场，但《四十二章经》与后宫异样已埋线。

### 二、杀鳌拜·天地会香主（京师·会党线启动）
康熙命韦小宝去杀囚中鳌拜，既试忠心，也灭口患。天地会刺客同夜潜入，混战中韦小宝误杀/乱刀结果了鳌拜，反被天地会掳走。总舵主**陈近南**（真名陈永华，国姓爷郑成功旧部军师，中原开山立会）见其“手刃鳌拜”，以为替前任青木堂香主报仇，收为关门弟子，任**青木堂香主**。韦小宝磕头时心里想的是“又多条命”，嘴上却把反清口号背得山响。从此身兼康熙心腹与反清会党香主，白天跪皇帝，夜里对香堂——双面人生正式开启。

天地会与沐王府因南明正统问题龃龉：谁才是“正朔”，刀口比嘴仗更硬。郡主**沐剑屏**被绑送入宫，藏于韦小宝处，天真认他作亲人；师姐**方怡**入宫行刺康熙（意图嫁祸吴三桂），亦被他救下，情愫暗生。假太后**毛东珠**（神龙教、明将毛文龙之女）多次欲杀韦小宝灭口，因他撞见过多后宫不可说之事。海大富与假太后对质时被杀，韦小宝得知：顺治帝尚在人间、后妃王子之死与假太后有关，康熙生母一线亦被阴影笼罩。康熙强忍丧母之痛，派他往五台山打听顺治下落——既是寻父，也是把“小桂子”支到局外。阶段结果：双面身份钉死；沐剑屏、方怡进入其命运；《四十二章经》与宫廷秘密露头。世界状态：会党、宫廷、邪教三线绞合，任何一边拆穿都是死局。

### 三、双儿·五台山·神龙白龙使
出宫途中，韦小宝一行在庄家遇神龙教追杀。明史案遗眷**庄三少奶**感其曾与“杀鳌拜”相关，相救并赠丫头**双儿**。双儿武功不俗、性静志坚，后成为韦最信任之人；亦拜**何铁手**为师，与**吴六奇**结为义兄妹，多次在绝境中拼死相救、拼合藏宝地图。五台山清凉寺，找到出家的顺治（法号**行痴**），韦小宝以市井话把宫廷惨变说得顺治泪下。喇嘛来犯，少林僧人驰援，庙宇刀光与木鱼声并存。

回程被方怡以情诱至**神龙岛**。方怡、沐剑屏已被迫入神龙教，身中教中毒物。韦小宝凭一张嘴帮教主**洪安通**与夫人**苏荃**平定内乱，被封**白龙使**，奉命回宫继续搜寻《四十二章经》——经皮夹层藏满清宝藏/“龙脉”地图碎片，原分交八旗旗主，除国君外或以龙脉说辞震慑八旗坐吃山空。康熙知顺治所在，令韦小宝先赴少林剃度掩饰身份，再转清凉寺住持。途中遇**王屋派**袭击，结识**曾柔**；以宝刀宝衣反制，再以赌局“放人换彩头”释放众人，赢得曾柔好感——韦小宝的“武功”从来是赌桌与人心。

剃度后遇美貌倾城的**阿珂**（九难弟子），一见倾心到神魂颠倒，阿珂却只把他当油嘴滑舌的小和尚。清凉寺再遭喇嘛围攻，韦小宝施计脱逃，促成康熙与顺治父子相认——帝王跪父，是全书罕见的家国温情。**九难**（白衣尼，前明长平公主）行刺康熙，韦小宝以宝衣挡剑被掳；在铁剑门逻辑与家国仇恨之间，他取得九难有限信任，潜入宫中揭穿假太后毛东珠，救出真太后。阿珂倾心台湾延平郡王世子**郑克塽**，为后文情伤与师门蒙羞埋伏。阶段结果：双儿、曾柔、阿珂登场；白龙使身份；假太后线收束；顺治线落地。世界状态：神龙教与宫廷仇恨公开化，台湾线的纨绔阴影已现。

### 四、云南和亲·阿珂身世·吴三桂
韦小宝回宫禀报，真太后复位，康熙对“小桂子”信任再上一层，亦开始隐隐察觉其江湖气味。康熙命其护送**建宁公主**和亲云南，嫁平西王世子**吴应熊**——实为安抚亦为牵制吴三桂。途中建宁与韦小宝私通，到云南后建宁狠阉吴应熊，和亲变羞辱，藩王府里刀光冷笑。九难派阿珂行刺**吴三桂**失手被擒。韦小宝从**陈圆圆**处得知：阿珂实为陈圆圆与**李自成**之女，九难误以为吴三桂所出——家国仇人谱系错位，美人命运更荒诞。九难以绝顶武功制服吴三桂，众人胁持吴应熊离滇。穿插寻经、戏耍郑克塽（阿珂面前出丑）、天地会香堂事务，并与蒙古**噶尔丹**、西藏**桑结**结义纳质，为康熙羁縻边陲预留筹码。阶段结果：建宁实质归韦；阿珂身世揭开；吴三桂线埋下三藩之乱历史背景。世界状态：藩王尾大不掉，边陲与台湾线升温，韦小宝官职与会党职责同步膨胀。

### 五、罗刹政变·衣锦还乡·七妻齐聚
康熙派韦小宝剿神龙教。他被洪安通所擒，靠双儿拼死救出，北逃至罗刹国雅克萨（鹿鼎山附近）——地图上的藏宝坐标，此刻变成冰雪绝境。随**索菲亚公主**赴莫斯科，以《三国》《水浒》戏文里的权谋知识助其政变夺权（自比“罗刹诸葛亮”），火枪与弓箭之间，市井智慧意外通杀宫廷。带回特使与清朝修好，康熙大喜，晋其爵、令衣锦还乡。

扬州丽春院，韦小宝与母亲**韦春花**重聚，母子抱头痛哭又立刻吵银子与面子。以迷药与机缘“大被同眠”，先后纳入阿珂、双儿、曾柔、方怡、沐剑屏、苏荃等，后加建宁，共**七位夫人**——不是传统侠义后宫，而是一串由救命、欺骗、和亲、教变拧成的命运绳结。回京途中遇**归辛树**一家欲刺康熙，韦小宝双面操作：一面报会党布局，一面通报康熙。归家被杀，他双手沾血却两边都没能救下。康熙点破其天地会身份，准备炮轰韦府；韦小宝借建宁之助逃脱并救众，君臣少年情第一次彻底裂开。再被神龙教擒回岛——洪安通发现苏荃怀了韦小宝的孩子，教义、嫉妒与毒控总爆发，教内互噬，洪安通等尽灭，苏荃幸存。众人转往**通吃岛**逍遥。台湾郑家、天地会、施琅水师先后追至；陈近南被**郑克塽**（借**冯锡范**“一剑无血”之手）暗算身亡，韦小宝哭恩师、安葬、逼郑写欠条，第一次真正恨到想屠尽延平王府，却仍被大势推着走。阶段结果：罗刹功业、七妻成型、恩师惨死、神龙教覆灭。世界状态：三藩与台湾将平，韦小宝忠义裂缝扩大到不可缝合。

### 六、雅克萨·尼布楚·终局归隐
通吃岛数年，七妻生子（子女名多由骰子决定，如韦虎头、韦铜锤、韦双双等），渔火炊烟，像把历史关在岛外。清廷平三藩、平台湾（**施琅**等）。康熙多次召回，圣旨与私信交替，恩威并施。因他曾助索菲亚，被派东北出征罗刹，再用计（仿周瑜反间一类）促成有利于清的**《尼布楚条约》**——边界、逃人、贸易，一条条谈下来，韦小宝的“戏文外交”写进了正史缝隙。

康熙知其天地会卧底身份，不再满足于装糊涂：放风称其杀陈近南以激会党仇杀，又允回扬州探母以示恩。途中**顾炎武**等文士以春秋大义劝其举义。夹在“小玄子”兄弟情与天地会江湖义之间，韦小宝两不相帮——他看清自己既做不成岳武穆，也做不成完美香主。假装全家遭劫杀，携七位夫人、儿女与母亲**韦春花**隐姓埋名，终隐**大理**一带。康熙不信其死，派人查找并六下江南寻访，石沉大海。“自此世上不复有奇人韦小宝矣。”结尾：韦小宝问生父是谁，母亲言年轻时恩客汉满蒙回藏皆有，**父不详**——首尾呼应多民族混融的隐喻，也消解“非我族类”的简单仇恨。结局：韦小宝归隐；康熙雄主在位；陈近南死；洪安通死；神龙教灭；台湾入清；雅克萨—尼布楚定界；七妻随隐；宝藏仍埋鹿鼎山，欲望的终点空着。

**【主要人物】**

- **韦小宝（丽春院少年 / 假太监小桂子 / 天地会青木堂香主 / 神龙教白龙使 / 鹿鼎公）**｜性格：油滑贪财好色、重义气讲交情、怕死善逃、市井厚黑却有底线（不真正害康熙、敬陈近南、护妻儿母亲）；骂得难听也哭得痛快，是金庸笔下最“活”的反英雄。｜装备·能力：武功低微（神行百变逃命、英雄美人三招皮毛、半吊子擒拿）；**宝衣、匕首、化尸粉、迷药**；赌术、戏文权谋、多语种哄人、多重身份切换。战力≈一～二阶，存活靠运气与人脉。｜人物弧光：妓院小鬼→双面间谍→一等公→忠义难全归隐；从想当英雄到拒绝当任何一方的刀。｜立场关系：康熙挚友兼臣；陈近南徒弟；七妻之夫；神龙教白龙使（后敌）；郑克塽仇隙；茅十八、吴六奇、多隆等兄弟线。

- **康熙玄烨（清帝 / “小玄子”）**｜性格：英明多疑、重情亦重权、少年时真挚、成年后帝王术纯熟；对韦小宝的感情是全书最动人也最残酷的一根线。｜装备·能力：君权与情报网、精锐侍卫；武功非常人可比但非江湖顶点。｜人物弧光：受制鳌拜的少年天子→擒鳌拜亲政→平三藩平台湾定尼布楚→逼韦小宝表态；从“咱们是兄弟”到“你只能是朕的臣”。｜立场关系：视韦小宝为玩伴兄弟后变君臣；杀鳌拜、除假太后；与天地会势不两立。

- **陈近南（天地会总舵主 / 陈永华）**｜性格：忠义沉毅、反清复明志向坚定、重师徒名分；英雄末路仍不肯放下大义。｜装备·能力：当世顶尖武功，可压一流群雄，≈三阶。｜人物弧光：国姓爷军师→中原开山立会→收韦小宝→被郑克塽暗算身亡；死于“自己人”比死于清廷更讽刺。｜立场关系：韦小宝恩师；与沐王府有隙；轻信郑氏终遭祸。

- **建宁公主**｜性格：蛮横泼辣、喜虐与被虐、占有欲强、关键时刻护夫。｜装备·能力：宫中身份；武功有限。｜人物弧光：假太后之女（名义公主）→赐婚吴应熊→私通韦小宝并阉吴→助韦出逃→七妻之一（新修版康熙钦定正妻倾向）。｜立场关系：康熙“皇妹”（实非）；韦小宝妻；敌视情敌又同穴。

- **双儿**｜性格：温柔忠心、沉静机敏、生死相随。｜装备·能力：庄家丫鬟底子 + 何铁手传授，武功在七妻中实用；拼合《四十二章经》地图。｜人物弧光：明史案遗孤→赠予韦小宝→多次救夫脱险→归隐。｜立场关系：韦小宝最信任；吴六奇义妹；何铁手徒弟。

- **方怡**｜性格：机巧聪慧、情感摇摆、曾两次欺瞒。｜装备·能力：沐王府武学，入神龙教后受毒控。｜人物弧光：入宫行刺→被救→被迫入神龙教→终归韦。｜立场关系：沐剑屏师姐；韦小宝初恋意味；神龙教棋子时期。

- **沐剑屏（沐王府郡主）**｜性格：天真纯善、少心机。｜装备·能力：沐家武学基础。｜人物弧光：被绑入宫→藏于韦处→入神龙教→归韦。｜立场关系：沐王府；方怡师妹；七妻中年龄偏小。

- **苏荃**｜性格：千娇百媚、城府与武功皆深、曾为教主夫人。｜装备·能力：神龙教高层武功，授韦英雄/美人三招，≈二阶顶尖至三阶边缘。｜人物弧光：洪安通妻（洪不近女色）→迷药后有孕→教内崩盘幸存→归韦。｜立场关系：洪安通旧人；韦小宝妻；神龙教覆灭关键引爆点。

- **曾柔**｜性格：温柔斯文、外柔内正。｜装备·能力：王屋派弟子武学。｜人物弧光：偷袭被韦以赌局反制→感其不杀之恩→归韦。｜立场关系：王屋派；七妻之一。

- **阿珂**｜性格：美貌绝伦、心高气傲、情路坎坷。｜装备·能力：九难弟子武学。｜人物弧光：倾心郑克塽→失身于韦后被郑弃→知身世（陈圆圆×李自成）→归韦。｜立场关系：九难徒弟；陈圆圆之女；郑克塽旧情；韦小宝妻。

- **洪安通（神龙教主）**｜性格：狂傲多疑、近乎偏执、权欲与教义扭曲。｜装备·能力：豹胎易筋等异功，当世顶点级≈三阶。｜人物弧光：坐镇神龙岛→封韦白龙使→因苏荃有孕互噬而死。｜立场关系：苏荃前夫；假太后同路；韦小宝先主后敌。

- **九难（长平公主 / 阿九）**｜性格：家国仇恨深重、冷峻有原则、武学传承者。｜装备·能力：**神行百变**等铁剑门绝学，≈三阶；刺帝、制服吴三桂。｜人物弧光：明亡巨变→为尼→行刺康熙被韦挡剑→揭假太后→阿珂之师。｜立场关系：与《碧血剑》线互文；阿珂师父；一度掳韦。

- **鳌拜**｜性格：跋扈专权、满洲第一勇士之傲。｜装备·能力：勇士级武力，≈二～三阶之间。｜人物弧光：专权→被康熙计擒→狱中死于混乱（韦小宝之手）。｜立场关系：康熙权敌； equip 宝衣匕首后归韦。

- **毛东珠（假太后）**｜性格：阴狠、潜伏深、神龙教棋子。｜装备·能力：毒计与宫廷渗透；武功不可小觑。｜人物弧光：杀真后妃、软禁真太后→海大富对质线→被揭身亡。｜立场关系：神龙教；建宁生母疑云；康熙宫廷大患。

- **茅十八**｜性格：江湖义气、粗豪。｜装备·能力：三流至二流好手。｜人物弧光：越狱被韦救→带其上京→后线淡出。｜立场关系：韦小宝江湖引路人。

- **吴六奇（大力将军 / 雪中铁丐）**｜性格：义薄云天、会党与朝廷身份复杂。｜装备·能力：一流至绝顶之间。｜人物弧光：天地会骨干；与双儿结义；后悲剧收场（版本细节涉及误会身亡）。｜立场关系：陈近南一系；双儿义兄。

- **多隆**｜性格：豪爽护短、满洲侍卫性格。｜装备·能力：一等侍卫级。｜人物弧光：韦小宝宫廷兄弟、多次出任务搭档。｜立场关系：康熙侧；韦小宝挚友。

- **郑克塽 / 冯锡范**｜性格：郑纨绔无能；冯阴鸷“一剑无血”。｜装备·能力：冯≈三阶刺客型。｜人物弧光：阿珂旧情线；暗算陈近南；后降清。｜立场关系：台湾郑氏；韦小宝仇隙。

- **陈圆圆**｜性格：红颜遗恨、明澈。｜装备·能力：不详/非战。｜人物弧光：告知阿珂身世；吴三桂侧历史符号。｜立场关系：阿珂之母；李自成线。

- **索菲亚公主**｜性格：野心勃勃、善用外援。｜装备·能力：罗刹宫廷权术。｜人物弧光：借韦政变摄政→遣使修好。｜立场关系：韦小宝“罗刹功业”关键人。

- **归辛树 / 归二娘 / 归钟**｜性格：反清死士、一门刚烈。｜装备·能力：华山一系顶尖，≈三阶。｜人物弧光：刺帝失败→家破。｜立场关系：与韦双面操作相关。

- **海大富**｜性格：老辣太监、知情者。｜装备·能力：大擒拿手等。｜人物弧光：擒韦入宫→被毒盲→对质假太后被杀。｜立场关系：韦小宝半吊子师父；假太后死敌。

- **韦春花（韦春芳）**｜性格：市井母亲、护子。｜装备·能力：无。｜人物弧光：丽春院撑持→子贵→随隐；点破父不详。｜立场关系：全书市井根与终章隐喻。

**【势力图谱】**
- **清廷（康熙朝）**：宗旨为皇权稳固、开疆定乱。地盘：京师—全国州府—东北边陲。代表：**康熙**、**索额图**、**多隆**、**施琅**等。战力层级：军队+侍卫体系为主，顶尖个人武者少于江湖顶点。与天地会敌对，与神龙教敌对，与郑氏先战后纳，与罗刹谈判定界。
- **天地会**：反清复明秘密会党，总舵主**陈近南**，下设各堂（青木堂香主韦小宝等），骨干**吴六奇**、**徐天川**、**风际中**（实为清廷卧底）等。与沐王府有正统之争，与郑氏有历史渊源却遭背刺。
- **神龙教**：蛇岛邪教，教主**洪安通**、夫人**苏荃**，龙使分色，假太后**毛东珠**为宫廷内应。借《四十二章经》图谋宝藏与龙脉话语权；内乱后覆灭。
- **沐王府**：南明沐氏余脉，**沐剑屏**、**方怡**、**沐剑声**等；与天地会既抗清又互争正统。
- **台湾延平王府**：郑氏政权，**郑克塽**、**冯锡范**；平台后入清版图。
- **罗刹国**：雅克萨驻军与莫斯科宫廷，**索菲亚**一系；尼布楚条约对手。
- **边陲盟好**：**噶尔丹**（蒙古）、**桑结**（西藏）与韦结义，服务康熙羁縻与韦个人人脉。
- **门派散修**：铁剑门（九难）、王屋派（曾柔）、少林、华山归家、胡逸之等，提供武学与刺客变量。

**【贵重物品】**
- **《四十二章经》八部**：表面佛经，封皮夹层藏羊皮藏宝图碎片；满清入关掠宝埋于关外**鹿鼎山**（近雅克萨），碎片分交八旗旗主；对外或称关乎龙脉以防八旗坐吃山空。韦小宝智取集齐，**双儿**拼合；韦因怕坏康熙气运未挖宝。串联宫廷、会党、神龙、九难多方争夺。
- **金丝软甲 / 宝衣（黑背心）**：鳌拜处抄得，刀枪难入；韦小宝替康熙挡九难一剑的关键道具。
- **鳌拜匕首**：削铁如泥，韦小宝贴身兵器。
- **化尸粉**：得自海大富一系，毁尸灭迹，宫斗江湖两用。
- **迷药 / 蒙汗药**：丽春院出身标配，多次破局（含扬州收妻名场面）。
- **神龙教丹药（豹胎易筋丸等）**：催功与控制，毁于教内崩溃。
- **鹿鼎山宝藏**：地图所指关外巨藏；原作中未真正掘开，象征“欲望终点的空位”。
- **爵位与权柄**：韦爵爷、鹿鼎公印信、尚方与兵权——在本世界中比任何神兵更可移国。

**【隐藏剧情 · 伏笔】**
- **父不详**：结尾汉满蒙回藏皆可能，消解单纯华夷叙事，呼应多民族中国主题。
- **《四十二章经》龙脉谎言**：宝藏真、龙脉话语权假——权力用神话管理八旗。
- **假太后与神龙教**：宫廷腹心之患，连结邪教与明遗；建宁血缘疑云。
- **阿珂身世**：陈圆圆×李自成，九难误认吴三桂，家国仇与情爱错位。
- **风际中卧底**：天地会内部清廷钉子，会党并非铁板。
- **与《碧血剑》互文**：九难（阿九）、神行百变传承，金庸宇宙串联。
- **康熙逼表态**：兄弟情最终让位于帝王术；韦小宝“两不帮”是对侠义与忠君双重话语的拒绝。
- **陈近南之死**：反清事业毁于郑氏不肖，讽刺“正统”空洞。

**【大事记时间线】**

- 约康熙初 → 韦小宝离扬州入京、冒小桂子、结识小玄子与建宁。
- 擒鳌拜 → 康熙亲政关键节点；韦杀鳌拜、入天地会、任青木堂香主。
- 五台山 → 见顺治行痴；少林/清凉剃度掩护；父子相认。
- 神龙岛 → 白龙使；《四十二章经》争夺白热化。
- 假太后败露 → 真太后出；九难刺帝被挡。
- 云南和亲 → 建宁阉吴应熊；阿珂身世明；吴三桂线。
- 雅克萨—莫斯科 → 助索菲亚政变；清俄接触。
- 扬州衣锦 → 七妻渐齐；归家事件；韦府炮轰出逃。
- 神龙教灭 → 苏荃有孕引爆；通吃岛逍遥。
- 陈近南死 → 郑克塽/冯锡范暗算。
- 平台湾、平三藩（历史并行）→ 施琅等；韦被召回。
- 雅克萨之战与尼布楚条约 → 韦用计促成有利定界。
- 终局 → 韦假死归隐大理；康熙寻访不得；父不详收束。

**【叙事基调 · 雷区】**
画风：市井喜剧外壳 + 历史权谋内核 + 反英雄反讽；可荤可谑，但人物动机须自洽。口吻宜带评书与市井气，不写仙侠金手指开挂。忌把韦小宝写成传统侠义圣人或无敌高手（他怕死、贪财、好色，却讲义气）；忌把康熙写成纯暴君或纯玩伴（少年真挚与帝王术并存）；忌神龙教、天地会脸谱化；忌四阶及以上毁城描写（本世界顶点止于三阶人间顶尖）；忌把《四十二章经》写成魔法书（它是地图与权力话语）；忌无视七妻各自性格强行工具人。最早可切入锚点：扬州丽春院—茅十八北上前后；或康熙擒鳌拜前夜的宫中少年局。

**【再补·韦小宝的市井武学】**
韦小宝武功低微，却有“逃命步”“混赖摔”“喊冤功”与神行百变的皮毛——真正外挂是厚黑、运气、多身份与七个老婆的情报网。写战斗应是桌翻、锅扣、烛灭、人喊，而不是掌风裂石。顶尖高手陈近南、九难、海大富等才接近三阶天花板。

**【再补·多线身份】**
清廷侍卫/太监假身份、天地会青木堂香主、神龙教白龙使、丽春院韦小宝——同一人四张皮。契约者任务可从“帮他圆谎”切入，而不是比武招亲。康熙与陈近南两端拉扯是主轴，雅克萨与《尼布楚条约》是国运戏，通吃岛归隐是终局。

**【再补·七夫人】**
双儿忠、建宁蛮、阿珂别扭、苏荃智、方怡弱、沐剑屏稚、曾柔静——每人应用真名与一句关系，禁止“后宫们”。大被同眠是喜剧也是政治联姻的市井版。

**【再补·名场面】**
杀鳌拜（借康熙与计）、五台见顺治、少林、神龙岛、雅克萨、通吃岛假死归隐。每段写清因果：谁逼宫、谁卖友、谁救谁。结局：忠义难两全，携妻母隐大理，康熙寻访不得。

## 阶位切入点

> 阶位↔原作战力：一阶≈市井/太监/普通兵丁/韦小宝本尊；二阶≈会党精英/一流侍卫/门派弟子/神龙中层；三阶≈陈近南、九难、洪安通、归辛树、冯锡范等当世顶点。世界顶点为三阶人间武学极限（完整战力），低阶切入一律“规避顶点·情报优先/条件性胜利”，不得用“战力归零”解释强敌。

**一阶（市井与宫门 · 假太监/小人物线）**

切入身份/时点：契约者以扬州跑腿、进京货郎、或新入宫的小太监/杂役身份切入，锚定**韦小宝刚随茅十八北上、尚未冒名小桂子**，或**小桂子身份初稳、康熙仍受鳌拜压制**前后。

初始事件：京师街巷或宫门值房里，一场明史案余波的搜捕与勒索把你卷进人群——你撞见满口戏文的瘦小少年（韦小宝）正帮江湖汉子茅十八打圆场；官兵逼近，你必须立刻选择：举报换赏、帮忙支开、还是趁乱摸走谁的腰牌。

开场白建议：「扬州的脂粉气还没从鼻子里散尽，京师的风已经刀一样刮在脸上。你看见那个叫韦小宝的小子拽着个外号茅十八的汉子往宫墙影子里钻——后面马蹄声响，有人高喊“鳌拜大人有令”。你手里只有半块饼和一张假身份，却已经站在了大清与江湖的缝上。」

关键NPC立场：**韦小宝**——此时全靠嘴皮与运气，可结盟可坑你，但讲义气时真敢两肋插刀；**茅十八**——粗豪欠人情，带你进局也敢拖你下水；**海大富**——老太监眼线，视下人为棋子，知情过多者死；**康熙（小玄子）**——若在御花园碰见，只是爱玩的少年，尚未显帝王全貌；**建宁公主**——蛮横惹祸，靠近易挨打也易得靠山；**多隆**——侍卫里的好人缘，可当宫廷导航。

主线钩子/支线：协助或旁观**智擒鳌拜**布局；宫中假太监身份存续；《四十二章经》风声初起时的传话与掉包。支线：丽春院旧识讨债、太监派系跑腿、为庄氏遗眷递一口信。

危险度/规避：中低——普通刀剑与廷杖可致命；须规避鳌拜本人、假太后毛东珠的直接视线，以及任何“当场拆穿小桂子”的冲动。

任务方向/奖励：活过宫门第一周、换到可靠腰牌与一层人脉；奖励取向＝迷药配方、入门刀法、宫廷地图碎片、与韦小宝的人情债（比银子更值钱）。

**二阶（会党与龙岛 · 双面香主线）**

切入身份/时点：以天地会新晋兄弟、沐王府护卫、或神龙教外围药人/使者身份切入，锚定**韦小宝已成青木堂香主且身兼白龙使**、假太后将败未败、五台—神龙—云南三条线并行之时。

初始事件：一次青木堂接头在茶楼二楼进行，暗号刚对上半句，风际中式的“自己人”突然反水——刀光里有人塞给你半部经皮残页，要你在“交给陈近南、交给韦香主、还是交给宫里”之间三选一，巡城营的脚步已在上楼。

开场白建议：「茶汤里的茉莉盖不住血腥气。你袖中那块经皮摸着像普通佛纸，可堂口的人都说它能换一座山。楼下有人喊“钦犯在此”，楼上有人低声叫你“兄弟”——你不知道该先抱拳，还是先跑。」

关键NPC立场：**陈近南**——忠义导师，欣赏能办事的人，讨厌投机；对契约者先观后用；**韦小宝**——双面压力极大，愿换情报与人情，也可能两头卖；**双儿**——温柔却护主，对威胁韦者刀下不留情；**方怡 / 沐剑屏**——神龙毒控下的摇摆，可救可利用；**洪安通**——多疑嗜权，投效易得位亦易灭口；**苏荃**——智计深，交易条件苛刻但守诺；**九难**——家国大义压过私情，行刺线附近极度危险。

主线钩子/支线：**揭假太后**前后的宫变；神龙岛内乱前的运毒运经；云南和亲车驾的刺局与护局。支线：帮吴六奇送一份会党密信、与阿珂/郑克塽三角边缘擦枪、王屋派赌命局。

危险度/规避：高——须规避洪安通单挑、九难认真一击、以及康熙亲军的围杀；双面身份一旦坐实，会党与宫廷两边都可能赐死。

任务方向/奖励：在会党或龙教拿到正式职司、保住一条命与一张地图残页；奖励取向＝神行百变入门步法、神龙丹药（慎用）、宝衣残片情报、边陲通行信物。

**三阶（边陲与终局 · 雅克萨/忠义难全线）**

切入身份/时点：以关外参领、罗刹通事、台湾来使护卫、或成名江湖高手（归家/铁剑门旁支级）身份切入，锚定**神龙教将灭或已灭、陈近南尚在或刚死、雅克萨战事与尼布楚谈判前后**，康熙开始逼韦小宝表态。

初始事件：雅克萨城下冻土上，清军与罗刹对峙，你被临时指派为“懂戏文也懂谈条件”的中间人——账篷里韦爵爷掷骰子定攻城时刻，索菲亚的使者递来拉丁文信件，天地会旧部在营外要你给陈近南的仇一个交代。你必须在天亮前决定：帮清军、帮和谈、还是帮一个准备假死的人出城。

开场白建议：「鹿鼎山的风里有血和雪。城头罗刹火枪冒烟，营里鹿鼎公笑着说‘兄弟你来掷一把’。你知道他笑里藏着会党的刀和皇帝的炮——这一把骰子，赌的不是城，是他还能不能继续当两头的人。」

关键NPC立场：**韦小宝**——已至爵爷，七妻儿女在通吃岛或随行，正被忠义撕裂，愿用宝藏情报与你换“全家活路”；**康熙**——雄主，可赏可杀，逼宫式表态不认兄弟只认臣；**陈近南**（若时点仍在）——责徒不忠，亦知大势已去；**冯锡范 / 郑克塽**——卖师仇家，谈判桌上笑里藏剑；**索菲亚一方使节**——只认利益；**施琅**——平台实干，讨厌会党空谈；**归辛树余脉或九难**——刺帝意志未消，可能把你当挡箭牌。

主线钩子/支线：**雅克萨攻防与尼布楚条约**条件博弈；陈近南死因真相与郑氏欠条；韦小宝**假死归隐**的护送或揭穿。支线：鹿鼎山藏宝是否开挖的伦理与利益；噶尔丹/桑结盟约的续约；七妻中某人落单遇险。

危险度/规避：极高——贴近世界顶点；须规避与陈近南、九难、冯锡范的致命对决，以及康熙“炮轰韦府”式的政治绞杀。**超阶巅峰不适用本世界**；顶点皆为三阶，胜利条件多为情报、人质、条约与撤退，而非击杀帝王或教主硬刚。

任务方向/奖励：促成一条对己有利的边界条款、护送或截留归隐车仗、取得拼合后的藏宝图抄本（未必敢挖）；奖励取向＝边陲军权文书、罗刹火器样本、三阶名家残篇武学、与康熙或会党清算后的“活路”。
