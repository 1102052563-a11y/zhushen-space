# 火凤燎原
<!--meta lib=主库 tiers=一、二、三-->

## 剧情

**【作品来源】**
《火凤燎原》（英文常译 The Ravages of Time）是香港漫画家**陈某**（陈海峰）自2001年起连载的三国历史策略长篇漫画，先后于香港《新少年》、台湾《新少年快报》等刊出，单行本已逾八十卷，连载仍在进行。作品以正史为骨、演义为纬，大量原创设定（残兵刺客团、水镜八奇、袁方等）与现代商战/人性视角重写汉末群雄，文风冷峻、智斗层层反转，黑页金句点题，画风兼具日系人物线条与港漫战场气势。衍生含Q版外传《火凤Q·我的水镜学园》、角色外传、设定集与2023年起的网络动画等。本档案以漫画正典主线为准，不写成修仙/魔法世界。

**【世界定位】**
东汉末年崩坏的中原乱世：天子失权、军阀割据、粮草即命脉、智谋与刺客决定城池存亡。双主角为**燎原火**（残兵之首、后被世人与刘备侧称为赵云/赵火）与**司马懿**（河内司马家二公子、残兵幕后主使）；书名「火」指燎原火、「凤」指司马懿。故事从少年司马懿的「终局之梦」开场，再倒回董卓入洛、关东讨董，以残兵视角串联洛阳大火、长安兵变、徐州屠戮、下邳白门楼、官渡、赤壁直至潼关、汉中等未完长线。一句话给AI：这是「冷兵器三国谋略武侠、顶尖武者约三阶」的常青档案，核心主题是义利之辩与时间对野心的灼烧。

**【世界观 · 力量体系】**
本世界**没有**灵气修仙、法术毁城、内力飞升一类超凡体系。力量来源是：①体魄与武艺（枪剑戟弓、马术、苦无、徒手）；②神经/感官残缺带来的非常规优势（残兵）；③情报、粮道、人心与层层计谋；④地形、水火、瘟疫与补给。死亡即死亡——无复活、无元神夺舍；偶有假死/易容/替身，但代价是剧情与信任，不是系统外挂。

战斗观感分层（按实际破坏力，对照乐园图鉴宁低勿高）：
1. **民壮 / 新兵 / 坊市混混**：徒手或短兵，踹门、械斗、劫道。≈一阶初期。
2. **郡兵 / 青州兵等正规军卒 / 山贼骨干**：阵列、盾矛、骑射基础；百人规模作战中的一员。≈一阶中后期。
3. **精锐部曲 / 陷阵营士卒 / 清风帮一类佣兵好手**：可破小寨、单挑数人至十数人。≈一阶后期～二阶初期。
4. **百人敌级将领 / 残兵骨干（郭昂、张雷、三船、小孟等）**：乱军中斩将、夜袭破营、一骑当关；可掀翻小队、裂盾断矛。≈二阶。
5. **万人敌 / 名将中坚（关羽、张飞、张辽、典韦、许褚、马超中后期等）**：裂石断金感的武艺展示、踏平据点、于万人阵中取首级；仍受箭雨、火攻、地形与人数制约。≈二阶后期～三阶初期。
6. **顶尖武者 / 战神级（吕布、巅峰燎原火、部分巅峰名将对决）**：乱军中近乎无人可正面硬挡，可左右一场局部会战的士气与胜负；**不能**一掌夷平洛阳、不能隔空灭国。≈三阶（世界顶点）。
7. **谋士与后勤**：水镜八奇、司马懿、荀彧、贾诩、郭嘉、庞统、周瑜、诸葛亮等**不按拳脚定阶**，但能用计灭军、用粮困死城池——对契约者的威胁往往高于蛮力。

特殊「体系」说明：
- **残兵**：口号「残存亦没路，兵败如山倒」。成员多有残疾，却把残缺炼成战技（无痛觉、单臂锤、刃足、易容、聋哑居合等）。出手以刺杀与奇袭为主，外界称「刺客之首残兵」。
- **水镜八奇**：水镜先生司马徽门下军师集团，制式笠帽面帘；一奇袁方、二奇荀彧、三奇贾诩、四奇郭嘉、五奇周瑜、六奇庞统、七奇诸葛亮、八奇赵昇（后期揭晓）。以理念与布局左右乱世，非统一门派武功。
- **南华八怪**等方外线：涉及黄巾、左慈、于吉等道门/异人元素，偏奇诡与毒计，仍非毁城魔法。
- **兵器与马**：方天画戟、青龙偃月刀、蛇矛、长枪、苦无、铁锤链、赤兔等名物有剧情重量，但本质是冷兵器与良驹，不是神器爆星。

乐园阶位映射：民壮/郡兵≈一阶；百人敌与残兵骨干/精锐将领≈二阶；吕布、巅峰燎原火、顶尖万人敌名将≈三阶。世界顶点＝三阶（冷兵器+谋略武侠），禁止把本世界写成超凡毁城档。低阶切入一律「规避顶点·情报优先」；顶点武者始终可战可杀可计取，但低阶须付出血价与条件，不得用「战力归零」式借口抹掉吕布/燎原火。

**【地理 · 舞台】**
舞台即汉末中国北方与荆扬江东，随连载缓慢扩张，AI 写场景先定「哪一年、哪座城、谁的粮道」。
- **洛阳**：东汉旧都。董卓专权、火烧洛阳、迁都长安的核心舞台；残兵刺许临、城下一聚、关东联军外围博弈均在此与周边展开。废墟与皇权象征并存。
- **长安 / 西凉通道**：董卓西迁后的朝廷所在；王允、吕布、李傕郭汜等权斗与兵变。
- **河内 / 司马家庄园与商路**：司马氏以商贾与情报立身的根基；残兵补给与人质线常回扣此处。司马家血洗是全书情感与立场转折地。
- **兖州 / 濮阳 / 徐州 / 下邳**：曹操立足、吕布夺徐、水淹下邳、白门楼斩吕的血腥棋盘；小孟之死、张辽降曹、刘备漂泊皆系于此。
- **官渡 / 河北袁氏领地（邺城等）**：袁绍与曹操决战主场；乌巢、颜良文丑、袁方布局与败亡改写北方归属。
- **许都 / 许昌**：曹操挟天子后的政治中枢；荀彧、司马懿等文臣谋士的角力场。
- **荆州 / 新野 / 当阳长坂 / 赤壁 / 南郡**：刘备寄人篱下、三顾草庐意象、长坂坡赵云救阿斗、孙刘抗曹火攻的水陆大战场。
- **江东（吴郡、柴桑等）**：孙策开基、周瑜经营、孙权继业；水军与世家政治。
- **潼关 / 关中 / 汉中方向（后期）**：马超复仇、曹操西征、八奇与燎原火/司马懿再交织；入川与襄樊等仍在连载推进中。
地理规则：城池靠粮、关隘靠兵、江河靠船；一场大火或一场水淹比「内力对轰」更能决定战役。契约者自称「飞升」会被当疯子或细作。

**【世界剧情线】**
① **终局之梦与开场定调（序幕）**  
漫画以少年司马懿的梦境开场：时间被拉到极远的将来，权倾天下的司马氏与「火」相关人物的结局阴影一闪而过——燎原火与司马懿两位主角的命运被预先灼烧在读者眼前，再倒叙回汉末。主题句已立：**时间会燎原一切野心，也会考验一切义气。** 此阶段世界状态：尚未进入正式年表叙事，但AI应记住「结局阴影已存在」，写正文时可用梦、谶、黑页金句暗示，不可剧透成已发生的现实。

② **洛阳·残兵出手与讨董序章（约189年起）**  
董卓入主洛阳，废立天子，西凉铁蹄压城。河内司马家以商贾身份在乱世求利，二公子**司马懿**虽年轻，却已是家族实质智囊，将战场政治当作商场算计。为阻董卓系军师**许临**对商路与家族利益的威胁，司马懿调动私兵刺客组织**残兵**，由首领**燎原火**执行刺杀。许临之死把司马家与残兵从幕后推到乱世棋盘的明处。

同期：关东诸侯以袁绍为盟主起兵讨董；**刘备、关羽、张飞**以微末之师卷入；**吕布**在董卓侧以武压场；残兵为救人质（司马朗等）潜入洛阳，与吕布、西凉猛将、朝廷权臣多方碰撞。**小孟**以易容与弓术执行美人计/刺探，常以「貂蝉」意象出现；**郭昂、张雷、三船**等残兵骨干负责攻坚、诱饵与断后。名场面包括：残兵夜刺、燎原火与吕布级武者的生死交锋、刘备一方救民时将燎原火误认为「赵云」、火烧洛阳与迁都前的混乱。

「**城下一聚**」是早期标志性长计：董卓侧（含李儒等）以迁都、焚城、诱敌互噬等多重布局，试图一石数鸟；司马懿与尚未正式出山的**诸葛亮**（水镜七奇）等智者从不同角度破解「繁衍/聚歼」真意，二人惺惺相惜的伏笔由此埋下。阶段结局：洛阳残破，董卓西迁长安，曹操等在讨董中崭露枭雄面目；残兵任务阶段性完成，司马家暂得保全，但已被各路豪杰记住。

③ **长安兵变与董卓之死**  
长安朝廷成为新棋盘。**王允**等设计联吕布除董；**吕布**在义与利、父子名分与天下野心之间摇摆后杀董卓。西凉余部李傕、郭汜等反扑，长安再乱。残兵与司马懿在此阶段继续以情报与刺杀换取家族生存空间；吕布失去靠山后开始流浪争城。水镜八奇中的**贾诩**等以「黑暗兵法」在西凉余波与后续张绣线中显威。阶段因果：董卓时代结束，权力真空扩大，兖徐青冀成为下一主战场。

④ **兖州·徐州与司马家血洗（第一部中段关键转折）**  
曹操收青州兵、经营兖州，屠城与严法并行，逐步成为北方最危险的「最强者」候选。吕布夺徐州、刘备失徐州又再起，三方拉锯。司马家因粮草、商路与残兵行动被卷入反曹或疑似反曹的连锁（含庞统等连环布局的波及），曹操派兵清算。

**许定**（许临之子、许褚之兄）潜伏残兵、自封首领意象，率部**血洗司马家**：族人、**郭昂**等死难，**张雷**守门重伤再残，庄园化为尸场。燎原火赶回救援，司马懿等核心得以逃生，却已家破。司马懿由此确立更冷酷的长期战略：**与其长年混战涂炭生灵，不如助最强者（曹操）速平天下，再取而代之改政**——「助纣为虐」式的大义自洽。他将部分布局与「制衡曹操」的棋子交给燎原火：去投靠能与曹抗衡的明主。理念裂痕在此加深：燎原火越来越听得懂「义」，司马懿越来越只信「利与时间」。

⑤ **下邳·白门楼与小孟之死（第一部收束核心）**  
吕布据下邳，曹操引沂泗之水灌城。高顺陷阵营夜袭、吕布突围与内叛（魏续等）交织；陈宫拒降，张辽在刘备说项下降曹。白门楼，吕布被缢杀，战神时代落幕。

**小孟**潜行刺曹失败，身份暴露，受辱后在枪林中死去；临终心念指向燎原火。司马懿在家族存亡与曹营信任之间做出令燎原火无法原谅的取舍（「出卖」换存续的叙事核心）。燎原火寻尸、暴走斩杀、与司马懿相对无言——**义利之辩**从伙伴之争变为半生宿敌伏笔。燎原火此后更深绑定**刘备**阵营，赵云之名逐渐坐实于世人口中；残兵旧时代随小孟与郭昂等凋零，张雷等转入抚养后辈（郭淮、王双、燎原广等）的余烬线。第一部（约单行本1–32卷量级）在「战神陨落、火凤分道」中收束。

⑥ **官渡上下：袁曹决战与八奇时代升温（第二部前半）**  
袁绍据河北，寄望于私生子兼水镜一奇**袁方**的总布局；曹操据许都，得**荀彧**养兵治国、**郭嘉**黑暗决策、**贾诩**等先后入局。官渡之战被写成多层计中计：颜良、文丑之死（关羽、张辽等武戏），乌巢焚粮，袁氏内斗，杨修等刺袁方导致一奇坠崖败亡。郭嘉病逝前后的布局与「奉孝遗产」继续影响南征节奏。司马懿在曹营从边缘智囊向可托付的隐患棋子爬升，隐忍不发。燎原火侧则随刘备在荆州边缘求存，执行救援、刺杀与护送。阶段结局：袁绍败亡，北方大势归曹；水镜八奇从「传说军师」变为各方死抢的实体棋手。

⑦ **当阳·赤壁·华容与荆州再分配（第二部中核）**  
曹操南下，刘琮降，刘备弃新野走当阳。**长坂坡**：**赵云（燎原火）**单骑救阿斗，张飞据桥断后，关羽水军接应——武戏与义的高光。诸葛亮出山后的联孙抗曹成为智斗主轴；**周瑜**主水军火攻，连环船与风向、假降、反间等被多方（含贾诩、庞统等）改写与再反写，而非演义式单线借东风。赤壁大火后曹军北还，华容道等线涉及放曹与否的义利选择题。荆州四郡争夺中，赵云桂阳等线、樊氏相关悲剧、黄忠等老将登场，为后续入川与襄樊埋伏笔。孙刘联盟貌合神离：周瑜、鲁肃、孙权各有算盘；刘备侧关张赵与军师团扩大。

⑧ **南郡至潼关·汉中方向（火凤时代·连载中）**  
赤壁后争南郡、借荆州、交锋不绝。曹操西征，**马超**为父报仇于潼关展「新战神」武威；许褚、徐晃等死战；贾诩、司马懿等计破关中联盟。燎原火与司马懿在更广阔的棋盘上再度间接交手：一边是蜀汉势力雏形（刘备、诸葛亮、庞统入川布局），一边是曹魏中枢的隐忍与清洗。八奇童年线、姜维等后辈、荀彧铜雀台后的理念之死等，把「谁才是真正为天下」的辩论推向更冷的高度。连载至八十余卷时，真正三分与祁山北伐仍遥远——**时间本身就是火凤**。

⑨ **贯穿支线A：残兵兴衰与二代**  
刘大创残兵→燎原火时代巅峰→许定潜伏与血洗→小孟郭昂陨落→张雷抚养**郭淮、王双、燎原广、黄皓**等二代。残兵从「司马家私兵」逐渐变为散在乱世的刺客余烬与义子网络。契约者可介入护送义子、追查许定、争夺残兵名号，但须付出血价。

⑩ **贯穿支线B：水镜八奇与水镜府**  
司马徽办学育人，八奇各怀理念：袁方「不战屈人」、荀彧「王佐兴汉」、贾诩「黑暗求生」、郭嘉「屠一城降十城」、周瑜「江东烈火」、庞统「连环翻盘」、诸葛亮「隆中与天时」、赵昇「天象宗教与野心」。八奇非铁板一块，常互坑互敬；司马懿是「遗珠」式竞争者，与诸葛亮的镜像关系是全书智斗审美中轴。

⑪ **贯穿支线C：刘备仁义 vs 曹操霸道 vs 孙权江东自保**  
刘备以仁义聚关张赵与士人，常败常起；曹操以法、粮、士与八奇速统北方，亦有屠城与多疑；孙权继承父兄基业，水军与世家平衡。三方都不是脸谱：失败者往往只是「棋差一招」。写正文忌把反派写成无能。

⑫ **结局方向（连载未完·仅能据开篇梦与史向推断）**  
正典未完结。开篇梦与史向共同指向：司马氏最终取魏；燎原火/赵云一系在义的路上燃烧自身；诸葛亮尽瘁；三国归晋的「时间胜利」属于凤，而火的意义在于曾燎过不义的平原。AI 写终局相关只能用「谶梦、史书残页、老人讲述」做氛围，**禁止**提前写成已完成的全剧终实体剧情。

时间顺序锚点（便于判断世界状态）：讨董洛阳 → 长安除董 → 兖徐拉锯与司马家血洗 → 下邳白门楼/小孟死 → 官渡袁败 → 赤壁火攻 → 荆州四郡/南郡 → 潼关马超 → 入川与后续（连载中）。任意时点先问：吕布是否已死？司马家是否已血洗？燎原火是否已公开为赵云？赤壁是否已烧？

**洛阳细目（可写进正文的微观链）**  
许临谨慎巡街→残兵苦无与剑影→赵贤护主战死意象→司马朗为人质的谈判桌→小孟以歌伎/侍从身份递毒针情报→关东联军各怀鬼胎不肯齐攻→董卓焚宫迁驾→难民潮中刘备一行收容百姓、误呼赵云→燎原火第一次被「义」的目光刺中。契约者若在此抢许临首级或密信，会同时被司马家、西凉余部和关东细作三重追杀。

**血洗与白门楼细目**  
许定以残兵名号反噬故主；郭昂锤断门闩仍不敌围杀；张雷刃足再残；司马懿在尸堆里完成「助强者」的自我论证。下邳城内水位上涨、鼠雀皆尽；高顺夜袭的马蹄与魏续开门的钥匙声叠在一起；小孟箭矢擦过曹操肩甲却换不回燎原火的原谅；白门楼上吕布要死得像战神，却死在绳与人心。写此阶段忌把悲剧写成无脑误会——每一步都是选择。

**官渡·赤壁细目**  
袁方在河北布「不战屈人」的网，杨修的刀与悬崖改写八奇排序；乌巢火光里许攸的一句话比千军可怕；长坂坡上阿斗的啼哭与赵云枪尖的血是同一条义脉；赤壁的船索、苦肉、反间、风向被八奇与江东诸将反复改写，火起时没有人是唯一的英雄。潼关之后，马超的枪是给曹操看的仇，也是给天下看的「新战神」广告。

**【隐藏剧情 · 伏笔】**
- 开篇终局梦：司马氏执权与火的结局阴影，全书最大跨度伏笔。
- 燎原火＝赵云的身份层：真假赵云、替身陆霜等线，用来讨论「名与实」。
- 小孟之死与司马懿「出卖」：火凤分道的情感核，后续每次重逢都带余烬。
- 许定潜伏残兵与血洗：刺客组织自我吞噬；许褚家族与司马家血债。
- 水镜八奇灭门/散伙与赵昇真面目：八奇非永恒同盟。
- 司马懿与诸葛亮的「镜像」：谁更懂天时，谁更懂人心。
- 「助最强者再取代」理论：为史向晋篡魏提供动机自洽，而非单纯性恶。
- 南华八怪与左慈于吉：方外线干扰江东与华容，防止世界观变成纯军史。
- 连载未完：入川、襄樊、夷陵、祁山等仅能作「将至的烽烟」，不可当已完结全剧写死。

**【叙事基调 · 雷区】**
基调：冷色智斗、层层反转、失败者亦可敬；武戏狠而短，谋略长而黑；黑页金句点题；人性在义利间灼烧。口吻宜克制、文言白话夹杂可少量，忌现代网络爽文嘴炮。  
忌：①写成修仙内力、毁城法术、系统面板；②把吕布写成无脑匹夫、把曹操写成纯反派小丑、把刘备写成只会哭；③提前剧透未连载完的完整终局为既成事实；④残兵/八奇用「群像」代替真名；⑤低阶契约者开局硬刚吕布或巅峰燎原火还无伤。  
最早可切入锚点：董卓刚入洛阳、残兵刺许临前后；或兖州商路护送、下邳围城前夜、官渡粮道、赤壁联孙密议等——皆有具体人名地名冲突。

## 阶位切入点

> 阶位↔原作战力：一阶≈民壮/郡兵/坊市好手；二阶≈百人敌、残兵骨干、精锐将领；三阶≈吕布、巅峰燎原火、顶尖万人敌名将。世界顶点为三阶冷兵器武者，不存在更高破坏力档的超凡毁城者。低阶切入一律规避顶点·情报优先；顶点可条件性胜利，禁止把顶点写成无威胁。

**一阶（民壮·郡兵·坊市线 · 乱世求活）**

切入身份/时点：契约者以河内/洛阳/兖州一带的民壮、运粮夫、商队护卫、新编郡兵或清风帮外围脚力切入；锚定**董卓入洛后、关东讨董前后**，或**曹操经营兖州、商路不宁**的阶段——洛阳尚未烧尽或余烬未冷，天下已乱而「战神」仍是传闻。

初始事件：所随商队（挂司马家或地方豪商旗号）在洛阳外官道被西凉游骑与山贼前后夹击；押运的粮册与一封无署名密信成为抢手货。混战中有人喊「残兵要货」，独眼或蒙面刺客一闪而过；契约者必须在「弃货逃命 / 护信求赏 / 把信卖给关东细作」之间当场抉择，从而被**司马朗**门客或**老贾**一类司马家联络人盯上。

开场白建议：「官道上的土是灰的，洛阳方向的天也是灰的。西凉人的马蹄像砸在你胸骨上，商队的牛车翻进沟里，粮袋裂开，密信却还贴在你内襟——有人用听不清的口音喊残兵，有人喊关东盟主有赏。你握着一根不算兵器的矛杆，第一次明白：乱世里活下去，比读过的忠义都贵。」

关键NPC立场：
- **司马懿**：尚未对你展露全部心机；视你为可抛的棋子或可用的眼线，给钱给路，不给真相。
- **燎原火**：若擦肩，只问「信在谁手」；不废话，杀意专业，可能因你护住妇孺而改口留一命。
- **小孟**：易容成歌伎或书僮试探；笑着递水，水可能有迷药。
- **刘备**：微末时愿分粮救人；拉你入伍是为仁义招牌，不是为认你当兄弟。
- **曹操**部下小校：要的是粮与户口册；降或死，少谈理想。
- **华雄**部游骑：视你如草芥；硬刚＝死。

主线钩子/支线：主线可介入「残兵刺许临」外围的信使与替死鬼环节，或洛阳难民潮中的救司马朗支线。支线：①清风帮**王刚**押镖被紫衣刺客盯上；②关东联军粮台被盗，背锅或立功；③王允府外买通门子听「貂蝉」流言（实为小孟线）。

危险度/规避：中高。规避**吕布**正面、**燎原火**夜刺、西凉铁骑冲阵；一阶禁止挑战「武者之最」。超阶不存在；顶点三阶＝存在·情报优先。

任务方向/奖励：护送密信/粮册、救出质、换取司马家或刘备军的安身之所；奖励为良矛皮甲、少量金银、残兵口令残片、地方军功身份——**不发**名将兵器本体。

**二阶（百人敌·残兵骨干·将领线 · 入局）**

切入身份/时点：契约者以州郡良将、佣兵头目、残兵外围「准成员」、或刘备/曹操/吕布某营曲军侯切入；锚定**司马家血洗前后**或**下邳围城至白门楼前夜**——火凤将分道，吕布将尽。

初始事件：受雇护送「司马家支脉商队」入兖，夜宿时许定线内应放火；**郭昂**铁锤开路突围，**张雷**刃足断后，契约者被迫与残兵临时联手杀出。突围后**司马懿**亲自（或隔帘）考问：「你要钱、要官，还是要一个能看见五年后的位置？」答错可能被灭口，答对则被塞进下邳战场当细作或死士。

开场白建议：「火光里司马家的匾掉下来，砸碎半个庭院。单臂的汉子甩锤砸开寨门，刃足的汉子在你身边旋出一道血弧。有人在烟里问你要钱还是要五年后的位置——你忽然听懂残兵的口号：残存亦没路。路，得自己杀出来。」

关键NPC立场：
- **司马懿**：收你为刀，测你是否会因「义」抗命；可结盟于利，必背刺于义。
- **燎原火**：若你救过残兵伤员，给三分不杀；若你碰小孟，杀无赦。
- **小孟**：下邳城内可能拉你做刺曹或送药的助手；情感上只认燎原火。
- **吕布**：赏识勇者，轻视无胆者；可赐一战之荣，难赐信任。
- **陈宫**：想听你是否懂「吕布不可辅却必须辅」的悲哀。
- **张辽**：降曹前后都可能成为你的考官或刀下鬼。
- **曹操**：要的是结果；献吕布首级或小孟行踪者赏，心软者坑。

主线钩子/支线：主线咬住**司马家血洗余波**与**白门楼刺曹**外围。支线：①高顺陷阵营敢死夜袭的向导；②为**典韦**死士营送刀，反水或同归于尽；③帮**庞统**传反曹同盟密信，同时被贾诩线反间。

危险度/规避：高。规避与**吕布**单挑至死、与**燎原火**无情报硬刚、站在白门楼枪林正中。顶点三阶存在·情报优先/条件性胜利（火攻、围杀、内应、毒、水淹）。

任务方向/奖励：夺取残兵名号碎片、救出**郭淮**幼弱、换取曲军侯印、缴获良马与百锻刀；可接触二阶武艺传授（枪阵、易容入门），不可直接领走方天画戟。

**三阶（顶尖武者·名将线 · 火凤对局）**

切入身份/时点：契约者以一方名将、残兵公认高手、水镜门客武职、或孙刘曹某侧「可入史书」的万人敌切入；锚定**官渡决战前后**、**长坂—赤壁**或**潼关马超反**——八奇时代与火凤时代重叠，每一次出手都牵动北方与荆扬。

初始事件：官渡粮道或乌巢外围，**袁方**余计与**郭嘉**死计对撞；契约者被双方同时抛出的「假粮官」身份砸中，必须在焚粮前选择站队。同步，**燎原火**奉命截杀某袁氏智囊，**司马懿**则要保同一人活着送假情报——你成为火凤之间的活绳结：帮火则曹营记你一笔血账，帮凤则刘备侧义士骂你，中立则两方杀。

开场白建议：「乌巢的夜比刀锋还薄。左边是河北的旗，右边是许都的火，你怀里那枚粮印是真是假，连你自己都快分不清。枪尖从暗处递到喉前——独眼的残兵首领只说三个字：让开，或死。更远处，有人把这一切写进五年后的账本。你突然笑了：原来三阶的战场，杀的不是人，是选择。」

关键NPC立场：
- **燎原火（赵云）**：认你为可交手的对手；若你护民护幼，可暂同盟；若你害刘备骨肉，必死战。
- **司马懿**：视你为「可加速统一的变量」；合作必须签在利益上，情感债他会记但不会还。
- **曹操**：可授你高位；叛一次则三族式清算（按乱世规则）。
- **刘备**：以仁义挽你；要你的枪，也要你的名。
- **诸葛亮**：试你是否懂天时；可能借你破局，不会把隆中全盘交你。
- **周瑜**：赤壁时视你为火攻链一环；江东利益优先。
- **马超**（潼关时点）：认勇不认阵营；可邀你冲曹操大营。
- **吕布**（若选白门楼前高阶线）：只认胜负；败则死，胜则可能收你为左右。

主线钩子/支线：主线可选**官渡焚粮站队**、**长坂救幼龙外围**、**赤壁火船敢死**、**潼关单骑救援/狙击**。支线：①与**张辽**约定三合之约；②护**荀彧**汉室密疏出许都；③追查八奇**赵昇**天象教众对赵云亲眷的暗杀。

危险度/规避：极高 / 贴近顶点。与**吕布**、巅峰**燎原火**、**马超**对决必须付出血价与条件（地形、毒、围、内应）；**禁止**无代价秒杀顶点。世界无超阶神魔；三阶巅峰＝存在·情报优先/条件性胜利。

任务方向/奖励：名将级兵书残卷、良驹、可写入军志的战功、残兵或八奇某支的暂定同盟、一柄「可传代」的百战枪——仍非毁城神器。奖励取向：人脉、声名、部曲编制、情报网，而非越级法宝。
