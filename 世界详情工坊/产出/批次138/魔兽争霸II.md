# 魔兽争霸II
<!--meta lib=主库 tiers=八-->

## 剧情

**【作品来源】**
《魔兽争霸II：黑潮》（英文：Warcraft II: Tides of Darkness；台译黑暗浪潮）是暴雪娱乐开发的即时战略游戏，1995 年 12 月北美发售（DOS／后移植 Mac、Windows；另有主机合集 The Dark Saga、1999 Battle.net 版、2024 Remastered）。续接《魔兽争霸：兽人与人类》第一次大战；资料片《黑暗之门》（Beyond the Dark Portal）接战后两年，本档案**以黑潮正篇第二次大战为主**，黑暗之门仅作结局衔接点名。设计师罗恩·米拉尔，剧本克里斯·梅森（Chris Metzen）等，音乐 Glenn Stafford。本档人物与战役以游戏手册／战役文本与暴雪二次大战设定为准，后续《魔兽世界》修订处必要时注明，但不整本倒灌现代魔兽编年。

**【世界定位】**
艾泽拉斯第二次大战：兽人部落在新大酋长**奥格瑞姆·末日之锤（Orgrim Doomhammer）**率领下跨海北征；暴风城残军在**安度因·洛萨（Anduin Lothar）**带领下逃往洛丹伦，联合人类、高等精灵、矮人、侏儒组成**洛丹伦联盟**。一句话：**海陆空全面战争，洛萨之死与黑暗之门前的终局**。卡片锚点：暴风沦陷难民北渡 → 联盟结成与部落扩张 → 古尔丹叛离寻墓 → 黑石山洛萨陨落 → 图拉扬接管、黑暗之门战役 → 门户被毁。

**【世界观 · 力量体系】**
力量本源：剑与魔法的中世纪奇幻军事；金矿、木材、石油三维资源；陆地／海军／空军单位；法师与死亡骑士等施法者；科技升级与建筑前置。战争迷雾：未探索漆黑，已探索无单位处仅见地形。死亡观：单位战死即亡，英雄单位死亡导致任务失败。特殊系统：石油平台、运输船、飞艇／龙、圣骑士与食人魔魔法师法术差异。

战力阶梯（本世界**仅覆盖八阶**，宁低勿高——大战可席卷大陆与门户级空间设施，按图鉴「行星／星空级·一界之主」观感落八阶）：
- **八阶**：联盟与部落的大战军团、洛萨／末日之锤级统帅、古尔丹等顶级术士、黑暗之门作为跨界设施的战役级对抗。表现：灭国级推进、跨海征服、摧毁或重创跨界门户。乐园观感≈**八阶**。  
  本档无九阶／超阶覆盖。顶点（黑暗之门战役、末日之锤与联盟最终对决）＝八阶顶：**存在·情报优先／条件性胜利**（需舰队、空中优势、英雄存活、资源与科技），严禁「大酋长被随便封印战力为零」。

乐园阶位映射：第二次大战主力与门户战＝八阶。普通步兵／苦工叙事上是军势部件，不单开低阶覆盖。

**【地理 · 舞台】**
- **艾泽拉斯南部／暴风王国废墟**：一代终局，难民起点。
- **无尽之海**：北渡与舰队战场。
- **洛丹伦王国**：联盟政治中心与主战场。
- **奎尔萨拉斯等精灵领地、卡兹莫丹矮人地、侏儒工坊**：盟军来源。
- **黑石山**：洛萨陨落关键战。
- **黑暗之门／南部焦土**：终局与跨界裂隙。
- **德拉诺（资料片）**：战后兽人退路，本档点到为止。

**【世界剧情线】**

**① 一代终局与难民潮**  
第一次大战：部落攻破暴风城，莱恩国王死，洛萨收拢残部跨海北上洛丹伦。部落在布莱克汉德死后由奥格瑞姆夺权，整编氏族，准备北征。因果：战争从「一国沦陷」变成「大陆同盟 vs 扩张部落」。

**② 联盟结成**  
洛萨说服洛丹伦等人类王国，并联合高等精灵、矮人、侏儒，组建洛丹伦联盟。新能力：海军、空军、更强施法者。名场面：多族旗帜同屏、造船与采油的战争经济。

**③ 部落扩张与氏族政治**  
奥格瑞姆统合部落，吸收森林巨魔、食人魔、地精等盟友，军势几乎所向披靡。内部仍有氏族野心与古尔丹的暗线。

**④ 古尔丹叛离**  
胜利前夕，古尔丹及其追随者抛弃盟友，去寻找沉没的萨格拉斯之墓与远古力量。部落力量被抽空，被迫收缩。名场面：术士私欲压过战局——第二次大战的转折点之一。

**⑤ 黑石山与洛萨之死**  
联盟反推。黑石山惨烈会战中，奥格瑞姆击杀洛萨，短暂重振兽人士气。但洛萨之死未摧毁联盟决心，反而成为旗帜。

**⑥ 图拉扬接管与黑暗之门**  
洛萨副官**图拉扬（Turalyon）**接管联盟军。最终在艾泽拉斯南部黑暗之门周边决战：联盟歼灭部分氏族、俘虏大酋长残部，并摧毁黑暗之门，试图杜绝再入侵。战役允许玩家打人类或兽人线，正史走向以联盟胜利、门户被毁为主流叙述（兽人战役为平行「如果部落赢」）。

**⑦ 战后衔接**  
兽人败退，门之裂隙未尽——引出资料片耐奥祖与德拉诺线。本档止于黑潮终局。

**⑧ 蝴蝶效应边界**  
契约者可改：某场海战、是否截住古尔丹分队、某英雄护送成败。不可无代价改写：洛萨之死的历史重量、古尔丹叛离对部落的削弱、黑暗之门被毁的终局方向（若走正史）。改了局部，因果以「部落更早崩盘／联盟内战／门未毁导致三战提前」回流。

- **图拉扬**｜性格：忠诚、迅速接过重担。｜装备·能力：圣骑士气质统帅。｜人物弧光：副官→洛萨死后最高指挥→门前胜利者。
- **奥格瑞姆·末日之锤**｜性格：冷酷高效、重纪律。｜装备·能力：大酋长、击杀洛萨。｜人物弧光：夺权→北征→终局被俘／战败。
- **古尔丹**｜性格：野心、私欲。｜装备·能力：顶级术士。｜人物弧光：部落智囊→叛离寻墓→削弱部落。
- **洛丹伦王室与人类诸侯**｜性格：各异，需被说服出兵。｜装备·能力：王国军团。｜人物弧光：从观望到全面参战。
- **高等精灵／矮人／侏儒代表将领**｜性格：各自文明立场。｜装备·能力：弓骑、火炮、飞行器等。｜人物弧光：盟军支柱。
- **部落氏族酋长与巨魔食人魔盟友**｜性格：好战、氏族优先。｜装备·能力：陆空海杂糅军势。｜人物弧光：扩张与内斗。
- **苦工／农民**｜性格：战争经济的血肉。｜装备·能力：采集。｜人物弧光：资源战的无名者。

**【势力图谱】**
- **洛丹伦联盟**：人、精灵、矮人、侏儒；守土与复仇。
- **旧部落**：兽人氏族＋巨魔＋食人魔＋地精；征服。
- **古尔丹派系**：术士私军。
- **黑暗之门／德拉诺残余**：跨界威胁源。
敌友逻辑：氏族可叛；盟军可争执指挥权；资源点即前线。

**【贵重物品】**
- **黑暗之门**：跨界设施，终局摧毁目标。
- **石油平台与舰队**：海权关键。
- **圣骑士与死亡骑士装备／法术**：高端单位。
- **末日之锤（武器与名号）**：大酋长象征。
- **萨格拉斯之墓线索**：古尔丹目标。
- **金矿／木材／石油**：战争血液。

**【隐藏剧情 · 伏笔】**
- 古尔丹寻墓连接更早燃烧军团暗线（点到为止）。
- 人类／兽人双战役是「历史的两张牌」，正史取联盟胜。
- 后续资料片与魔兽世界对第二次大战有修订，写作以黑潮战役情感为准。
- 单位点击彩蛋与音乐是暴雪风格，不入正史。

**【大事记时间线】**
暴风沦陷 → 洛萨北渡 → 联盟结成 → 部落跨海扩张 → 古尔丹叛离 → 联盟反攻 → 黑石山洛萨死 → 图拉扬接管 → 黑暗之门决战 → 门毁、部落残部被俘／败退 → （衔接）黑暗之门资料片。

**【叙事基调 · 雷区】**
基调：史诗 RTS 战争、英雄牺牲、氏族与同盟政治；海陆空立体战场。  
忌：把二代写成三代混乱之治剧情；忌无视石油与海军；忌让洛萨轻易不死而无替代史诗代价；忌用仙侠境界词；忌把苦工农民写成八阶个体。  
最早切入：难民刚抵洛丹伦、联盟会议前后；或古尔丹即将叛离的前线。

**【第二次大战战役细部】**

北渡：无尽之海上的运输船与兽人舰队遭遇战决定多少暴风血脉能活到洛丹伦。洛萨在甲板上不是传奇，是会计——每一艘沉船都是一张贵族名单。

结盟：人类诸侯要安全保证，精灵要边境不被烧，矮人要铁矿与尊严，侏儒要实验场。洛萨用「下一个就是你」把他们捆上同一条船。

部落侧：奥格瑞姆用纪律替换布莱克汉德的混乱，氏族旗帜仍在内斗。胜利让所有人更饿。古尔丹的影子在萨满与术士之间拉锯。

叛离：当部落以为洛丹伦将破时，古尔丹把精锐抽走。前线突然缺了一块骨头。奥格瑞姆的怒火烧向内贼，却来不及补洞。

黑石山：洛萨之死是镜头中心。剑折、旗不倒。兽人狂欢一夜，人类把名字刻进盾里。

门前：图拉扬的命令更短、更狠。摧毁门不是荣耀，是恐惧管理——怕第三次。

**【单位与战争经济】**

农民／苦工是八阶战争的毛细血管。石油让大海变成可计算的前线。飞龙与狮鹫让高地不再安全。法师与死亡骑士是稀缺决胜手。英雄单位是任务的心跳：他死，你重来——叙事上即不可替代的人物弧光。

**【双战役说明】**
人类战役走正史胜利；兽人战役是「若部落团结且古尔丹不叛」的血色假想。契约者可体验任一，但世界状态默认正史。

**【人物补录】**
洛丹伦国王泰瑞纳斯等人类君主提供政治舞台；乌瑟尔等圣骑士传统在二代气质中萌芽（具体头衔以战役文本为准）；部落侧各氏族酋长作为军势脸谱出现时尽量用具名。麦迪文一代阴影仍在背景，不抢二代镜头。

**【地理补录】**
冬季地图冰面可通行改变陆军路径；森林是木材也是墙；油井在近海。黑石山垂直战场；门户周边焦土与裂隙光。

**【宝物补录】**
联盟纹章旗、部落氏族旗、洛萨佩剑传说、石油库存清单、门户爆破装置、古尔丹的恶魔契约残页。

**【隐藏补录】**
《黑暗之潮》小说可作对照，但本档以游戏战役为主。重制版不改变二次大战骨架。

**【八阶战争语感】**
写摧毁门户、沉没舰队、焚城，不写爆恒星。八阶是「一界之战」，不是宇宙终焉。

**【可介入节点】**
护侨船队；截获古尔丹分队情报；黑石山死战中抢回洛萨遗体或旗帜；门户爆破倒计时；俘虏管理与兽人营地暴动。

门在光中崩解。海平静得不诚实。洛萨的位置空着，图拉扬的命令还在。部落的鼓声远了，但德拉诺的影子近了。第二次大战结束，艾泽拉斯学会害怕门。



**【第二次大战人物志扩写】**

洛萨在洛丹伦宫廷不是来乞求施舍，是来展示名单：每一行名字背后是沉船与焚城。他的演说短，因为长了会被当成戏剧。图拉扬学会在洛萨身后重复命令，也学会在洛萨死后把命令缩短到动词。奥格瑞姆相信纪律能把部落从「掠夺者」锻成「征服者」，他恨古尔丹不是恨邪恶，是恨失控。古尔丹爱力量像爱镜子，萨格拉斯之墓是他给自己的加冕礼，哪怕加冕礼烧掉整条前线。

人类诸侯把联盟当成保险合同：条款写着出兵，附录写着利益。高等精灵的弓阵昂贵而精准，他们参战要边境承诺。矮人的炮火改变攻城时间表。侏儒的飞艇让「高地安全」成为过时观念。森林巨魔加入部落，把游击与残忍写进林地。食人魔冲击步兵线。地精把爆炸变成商业。

**【海陆空三维战争】**

海：运输船是难民与军团的血管，油井是心跳。谁控制近海平台，谁决定下一次登陆点。空：龙与狮鹫让城墙矮了一截。陆：农场既是人口上限也是墙。雾战让地图成为情报商品。资源耗尽逼玩家把森林砍成走廊，也把战略变成几何。

**【洛萨之死的叙事处理】**

不要写成无意义的Boss战失败。要写成旗帜转移：兽人得到一夜狂欢，人类得到一句永不投降的姓氏。图拉扬接过的不是剑，是账本与棺材。

**【门户终局】**

摧毁门是恐惧管理。联盟怕第三次，部落怕归路。俘虏营里的兽人眼睛仍红，人类牧师在争论赎罪是否适用于异世界。海平静，焦土不平静。

**【双战役与正史】**

人类胜为正史；兽人胜为血色if。契约者可玩if，但世界书默认门毁、部落残。资料片耐奥祖线是下一扇门的预告，本档不展开德拉诺全战役。

**【可观察细节库】**
燃烧的石油海面、冰上可走的冬季浅滩、被反复点击仍骂人的步兵语音（氛围，非剧情）、氏族旗在内斗中更换、联盟会议桌上的地图钉、黑石山的烟、门户的绿光、苦工的号子、圣骑士的祷词与死亡骑士的嘲笑对位。



**【正史收束句】**第二次大战以联盟胜利、黑暗之门被毁、部落残部被俘与败退写入艾泽拉斯的恐惧词典。洛萨的空位没有被填满，只被继承。海恢复蓝色，人没有恢复天真。下一场鼓声将从门的另一侧传来，但那是另一本书的事。

## 阶位切入点

> 阶位↔原作：八阶≈第二次大战军团与黑暗之门级战役。顶点（门前决战／末日之锤）＝八阶顶，条件战·情报优先。

**八阶（第二次大战 · 联盟与部落）**
切入身份/时点：洛丹伦新编军团军官、暴风流亡骑士、部落氏族先锋、石油船队护卫、古尔丹派系叛逃术士；锚定联盟结成后至门户战。
初始事件：你的运输船在无尽之海上看见兽人油烟。甲板上的洛萨残部在数活人。岸上洛丹伦的使者问：你们是难民，还是开战争端的火种？
开场白建议：「海比城墙更诚实——它沉没你所爱的，也送来你所恨的。远处部落的鼓点与联盟的号角抢同一阵风。你的剑还带着暴风的灰，已经要决定刺向氏族，还是刺向犹豫的诸侯。」
关键NPC立场：**洛萨**——要纪律与希望；**图拉扬**——要执行；**奥格瑞姆**——要征服与秩序；**古尔丹**——要私欲；**人类诸侯**——要安全；**精灵矮人侏儒将领**——要条件；**苦工／农民**——要活。
主线钩子/支线：护侨；海战夺油；黑石山死战；截古尔丹；门户爆破；支线＝氏族内斗情报、盟军补给谈判。
危险度/规避：极高——**门户与大酋长级＝八阶顶：存在·情报优先／条件性胜利**；须舰队空中与英雄存活条件；严禁单骑砍门。
任务方向/奖励：活过门毁；奖励＝军衔、舰队指挥权、氏族或联盟政治筹码、门户碎片研究权（高危）。

**八阶·第二开场（古尔丹叛离夜）**
切入身份/时点：部落侧氏族百夫长或联盟间谍。
初始事件：前线正要破城，精锐却奉命掉头。有人口中念着「坟墓」。奥格瑞姆的怒火比人类的剑更近。
开场白建议：「胜利的味道突然像叛变。你不知道该追古尔丹，还是该挡联盟反攻。鼓停了一拍，历史就在这一拍里拐弯。」
关键NPC立场：**古尔丹**——要你追随坟墓；**奥格瑞姆**——要你追叛贼或死守；**联盟先锋**——要趁乱。
主线钩子/支线：截击术士车队；死守缺口；支线＝藏起氏族幼崽与奴隶。
危险度/规避：高——内斗与外敌同时。
任务方向/奖励：氏族命运的投票权、术士残卷、奥格瑞姆的暂时信任或死亡令。

**八阶·第三开场（门前爆破）**
切入身份/时点：爆破工兵指挥官或圣骑士护卫队。
初始事件：图拉扬的命令只有一句：门必须在黎明前消失。兽人残部反扑，石油着火把海烧成白天。
开场白建议：「你把炸药背在比信仰更沉的地方。洛萨的名字在耳边，像催促，也像债。门一亮，世界的边界就松动。」
关键NPC立场：**图拉扬**——要成功；**奥格瑞姆残部**——要保门；**工程师侏儒**——要时间。
主线钩子/支线：护爆破点；拖住英雄单位；支线＝疏散平民奴隶。
危险度/规避：贴近顶点——门户能量反噬。
任务方向/奖励：门户终战勋章、战后重建话语权、对德拉诺的恐惧与准备。


**八阶·第四开场（黑石山）**
切入身份/时点：洛萨亲卫或部落敢死队；锚定洛萨陨落战。
初始事件：山道被尸体填平。你听见统帅的号角断了一拍。有人喊「洛萨倒下了」，有人喊「杀光他们」。你必须在崩溃前决定抢旗还是抢遗体。
开场白建议：「黑石山不需要诗。它只需要你还站着。旗倒了可以扶，人走了只能记。」
关键NPC立场：**洛萨（濒死／已死）**——旗帜；**奥格瑞姆**——胜利者；**图拉扬**——继承者；**双方士卒**——恐慌与狂欢。
主线钩子/支线：夺回统帅信物；组织撤退；支线＝烧毁己方机密地图。
危险度/规避：极高——统帅级交锋余波。
任务方向/奖励：洛萨之旗、图拉扬信任、黑石山伤痕勋章。

**八阶·第五开场（俘虏营暴动）**
切入身份/时点：联盟战俘官或部落卧底。
初始事件：门毁后的第一个月，俘虏营里鼓声再起。有人要用兽人苦工重启战争，有人要用人道法令把自己洗白。
开场白建议：「胜利比失败更难管理。你看着铁栏后的红眼睛，不知道自己是狱卒还是下一场战争的燃料官。」
关键NPC立场：**图拉扬派**——要秩序；**强硬派贵族**——要处决；**部落酋长残部**——要越狱；**牧师**——要救赎叙事。
主线钩子/支线：镇压或谈判；支线＝防止古尔丹残党策反。
危险度/规避：高——暴动可燎原。
任务方向/奖励：战后政策话语权、氏族情报、道德污点或勋章。
