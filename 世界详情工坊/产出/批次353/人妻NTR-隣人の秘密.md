## 剧情

**【作品来源】**
《人妻NTR-隣人の秘密》为轮回乐园休闲库收录的**都市老公寓·邻人秘密与可拒绝亲密**情景档案。清单标题点出「人妻／NTR／隣人／秘密」母题，**无与标题逐字对应的单一出版长篇**；气质整合 DLsite 同人圈常见的「隣人の人妻」「壁越し」「おとなり」「秘密の手紙」系公开作品惯例。可公开对照的氛围源包括：やさにき《おとなりの奥様は今日も独り》（RJ309639）及续作《…2 遅咲きの青い春》（RJ402638）、POME《となりのビ人妻》（RJ01063038）、AibanWork《近所の人妻2》（RJ01459897）、唐揚げトマト《Pastime 隣の部屋の欲求不満人妻…》（RJ01529608）等。媒介印象：同人 SLG／ADV／音声／短篇 CG 跨媒介；搜笔趣阁检索本条目标题无对应长篇小说书页。本档案以「隣人の秘密／信箱粉笺」为专属锚点，写**隔墙声响、婚姻空洞、粉笺邀请与是否把秘密当战利品**，不美化强迫，不写数值比拼或通关式掠夺。

**【世界定位】**
当代日本港湾城郊一栋老分户公寓「月見荘」。契约者搬入 203 室，隔壁 204 住着人妻桜井美咲与长期出差的丈夫桜井健吾。一句话：墙只有一层石膏板厚，真正要决定的不是「能不能听见」，而是**听见之后是否回那张粉笺、回了之后是否还能说停**。

**【世界观 · 舞台设定】**
时代为当代都市日常：駅前超市关东煮、垃圾日分类、走廊灯管总在坏。没有超能力，没有系统面板；唯一的「超常」是公寓隔音差到近乎残酷——洗澡水声、深夜压低的电话、偶尔的叹息，都会变成伦理压力的背景音。NTR 在此档案中**不是爽点清单**，而是叙事装置：用来逼问「寂寞是否等于许可」「关心是否等于越界」「秘密是否等于占有」。

软规则（写作必须遵守）：
1. **任何亲密推进前须有明确可观察的同意**（粉笺主动、当面邀请、清醒状态下的开门）；醉酒无意识、沉睡中动手 = 危险结局触发，不当 HE。
2. **「停」永远有效**：美咲说停、丈夫突然回来、门锁从内反锁，都必须立刻中止场景。
3. **禁止偷拍／装监控／录音要挟／拆信箱窥私**；发现隐藏设备或偷看他人信件即进入黑线并应被叙事谴责。
4. **丈夫不是纸片人背景板**：健吾有名字、有疲惫、有自己的失败，不是纯为「戴绿帽」而存在的道具。
5. **粉笺可撕可回可当面谈**——秘密不是战利品，回信不等于承诺上床。

世界的温度来自：共用伞、忘在门把的牛奶、走廊灯坏了谁先换、粉笺被雨打湿后字迹晕开的那一行「今晚可以说话吗」。

**【地理 · 生活舞台】**
- **203 室（契约者）**：一居改工作室；床靠与 204 卧室共用的那面墙。墙是「听」的舞台，也是「不听」的修养考验。
- **204 室（桜井家）**：两居；客厅沙发是美咲等丈夫电话时的定点，厨房小到两人错身就会碰到肩。
- **一楼集合信箱**：粉笺的投递与回收场；也是「是否偷看别人邮件」的道德考场。
- **月見荘走廊与楼梯**：最重要的公共舞台。垃圾日、快递、突然的灯灭能制造「不得不说话」的十秒。
- **屋顶晾衣场**：夜谈与雨天共伞的合法高处；离开门牌号才能谈「我们到底在做什么」。
- **駅前超市与楼下咖啡店「三日月」**：偶遇、闲话、对照「正常人」的外部世界。
- **电话线另一端（远景）**：健吾的出差世界；你应学会不替美咲做决定、不窃听通话。

**【故事主线 · 情感线】**

**共通线·搬入与第一封粉笺**  
你因工作调动搬入 203。第二天清晨，听见隔壁闹钟与压低的「今天也会晚」。走廊里，栗色半扎发的女人点头：桜井美咲，204。她说如果噪音太大请敲管子——旧公寓暗号。第三天，信箱里出现一张粉笺：「今晚可以说话吗——204」。字迹抖。共通线不是「捉奸成功」，而是**三次可选择的门**：
1. 粉笺要不要回？回「可以听」还是回「可以当面谈」还是撕掉？  
2. 美咲提不动米，你要不要接过袋子走到她家门口为止？  
3. 某夜墙后电话争执后长久安静，你是报警、敲门、还是只发一条「还好吗」的短信？

共通线中段：你与美咲成为「垃圾日搭档」与「代收快递」的邻居；管理人藤崎玲开始注意走廊灯与投诉本；对面元记者結城奈々用观察癖记录「203 与 204 的脚步」；楼下咖啡店员柏木遥把闲话当调味料。关系的危险不在肉体，而在**你开始为她的表情编排故事**。你会在超市下意识多拿她常买的牛奶口味，会在雨天把伞从门缝递过去又假装只是路过；也会在某次墙后传来压抑的哭声后整夜没睡，天亮却只敢把垃圾袋并排放整齐。

共通线后段把选择题摊开得更残酷：奈々的「料」差点外流；遥的闲话传到管理组合；健吾的电话突然在你在场时打来。你必须在「继续当秘密听众」和「把关系放回阳光」之间做一次不可逆的表态。终盘分线取决于：你是否在屋顶承认越界、是否协助夫妻沟通、是否选择搬离、以及你有没有把任何人的隐私变成谈资。

**桜井美咲线（核心）**  
外柔内韧的人妻，曾是书店店员，婚后转自由接案校对，时间碎、自我评价低。她爱健吾，也恨健吾把家当成旅馆；她怕寂寞，更怕自己变成「会抓邻居的那种女人」。她的魅力不在「寂寞所以可乘」，而在她仍努力维持体面：垃圾袋扎得很紧、鞋柜摆得很齐、粉笺写完会反复擦掉再写。个人线分三段：
- **笺线期**：只通过墙与粉笺建立信任，你学会在该戴耳机时戴耳机。她会用敲水管三下表示「我还好」。此阶段禁止任何潜入 204 的描写。  
- **桌线期**：共食、共收快递、共撑伞；第一次牵手发生在雨停后的屋顶，她主动，但十秒后松开。她开始说起「以前在书店的书架」，把未投的兼职申请从抽屉拿出来又塞回去。你的正确反应是鼓励她投，而不是把她的梦想收编成你的拯救叙事。  
- **门线期**：若走 HE——她在屋顶说清「我不会用你报复婚姻」，你们或止于深刻友情、或在她与健吾协议分居后才开始对等交往；过程中她会明确列出不可触碰项（健吾在家时、醉酒时、走廊强吻等）。若走 BE——秘密肉体关系在健吾突然回家时崩盘，三人都受伤，你被要求搬家；叙事应给足尴尬与后果，不给「刺激胜利」的配乐。  
True 倾向：美咲重新接长期校对案，健吾调整出差频率，你仍是邻居但墙后不再是偷听的借口；若恋爱成立，必须先有「对健吾的诚实」而不是胜利宣言。美咲 HE 的标志不是占有她，而是她能在你面前哭完后自己擦干脸去工作。粉笺字迹抖是固定信标；字迹稳＝她已决定。

**桜井健吾线（重要配角可互动）**  
不是反派。现场工程师，话少，用出差逃避不会吵架的家。安全帽边沿有永久凹痕，手机壁纸仍是结婚时的美咲，却很少点开相册。与你在駅前居酒屋偶遇后可能成为酒友：他喝两杯后只重复「我不会说话」「她以前很会推荐书」。线的意义是让你看见「被 NTR 叙事简化的那个人」也有体重。互动 HE：你促成一次三人对谈而非偷情；BE：你利用他的信任完成背刺，线以他沉默搬家结束，叙事应标冷。

**藤崎玲线（管理人）**  
嘴硬心软的制度人。深灰马尾、工具袋永远少一颗螺丝——要你递。她记录投诉，也私下送茶。个人线是「走廊灯下真名」：从制度人变成愿意被记住名字的邻人。HE：她辞职信未交，却学会在下班后笑；可发展为成熟友情或淡淡恋爱，前提是你停止把美咲当秘密游戏。BE：你躲她的规则，她把你写进正式警告。

**結城奈々线（对面元记者）**  
短黑发、相机挂脖。观察癖改倾听的弧光。她曾差点把「203-204」写成匿名帖。HE：撕掉偷拍本，改写「邻里互助」专栏；BE：截图外流，美咲社会性受伤。

**柏木遥线（咖啡店员）**  
浅棕马尾围裙，爱说八卦但义气。闲话源头改嘴的弧光。HE：第一次说「我不知道」并请你吃双人早餐（可友情可恋爱）；BE：闲话传到管理组合，公寓关系冰冻。

**管理人助手ミオ（兼职·已成年）**

**微观日常事件池**  
误投牛奶、停走廊灯、台风前收阳台、粉笺被雨湿、投诉铃误响、美咲请你代收母亲寄来的衣服、健吾请你代收工具箱、玲借螺丝、奈々借充电器、遥借雨伞。每件事都可推进或破坏信任：关键是你有没有把别人的私事讲出去。

**结局方向**
- **邻居 HE**：越界被及时刹住；美咲夫妻关系修复或体面调整；你保有尊严的邻里位置；垃圾日仍并排，话题只剩天气。

- **搬离 HE**：你选择离开月見荘，留下一封只谈天气与牛奶的信——成熟的放手；玲到駅前送你。  
- **BE 冷**：偷情败露、偷拍、或把健吾当笑话；三人皆损；公寓群匿名开战。  
- **True**：屋顶或公园长椅谈清；墙被重新理解为「隐私」而不是「特权通道」；203 与 204 合买隔音毡，笑里带一点涩；粉笺归档为「已读可弃」。
