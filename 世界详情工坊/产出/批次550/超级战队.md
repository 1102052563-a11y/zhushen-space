# 超级战队
<!--meta lib=主库 tiers=一、二、三、四、五-->

## 剧情

**【作品来源】**
《超级战队》（スーパー戦隊シリーズ，Super Sentai Series）是东映制作、朝日电视台播出的日本特摄超级英雄团队系列，自1975年4月5日《秘密战队五连者》首播起，以约一年一部的节奏连播至令和时期第49部《No.1战队五兽者》（2025–2026）后进入阶段性休止，累计正史TV逾2400集，并含剧场版、V-Cinema、VS跨界与超级英雄大战等。创作者谱系：前期石之森章太郎（《五连者》《JAKQ电击队》），1979年起以八手三郎（东映集体笔名）为主；玩具与机体由万代等驱动。系列与《假面骑士》同属「超级英雄时间」周日档，并以北美《金刚战士》（Power Rangers）改编母本闻名。本档是**系列总览任务宇宙**，写共通机制与代表作脉络；单部深挖（如恐龙战队兽连者）以分册条目为准，禁止把总览写成单部复读。文风：彩色队形热血、周刊怪人节奏、巨大战高潮，外壳下藏组织阴谋、牺牲与「齐心」伦理。

**【世界定位】**
表层为现代日本（或各作自立的地球／星海／异界舞台），地下则有侵略帝国、暗组织、异次元军团与巨型怪物轮番袭击城市。色彩编码的战士小队以变身装置完成「变身（Henshin）」，人型战击退后怪物巨大化，召唤并合体巨型机器人决战。总览世界**不是单一连续正史**（除电人战队／太阳战队等少数共享、以及海贼战队豪快者等周年串线作），而是**共享语法的任务宇宙**：五色（及扩展色）队形、点名亮相、周刊怪人、巨大化、机体合体。一句话：你在「本周袭击」与「机体出动前的疏散」里活下来，而不是一上来单骑拆掉历代全部巨大机器人。

**【世界观 · 力量体系】**
共通规则（跨作可复用，细节随年度主题换皮）：

1. **色彩战士与队形**  
   红战士通常为队长／主视角，蓝冷静、黄／粉／绿／黑等承担性格与职能分工；初期常3人或5人，中后期追加第六战士（金／银／白等）与强化形态。头盔全罩、紧身战斗服、个人武器＋共享队技。变身后的**点名亮相（ロールコール）**是仪式：报颜色、报称号、摆队形——不是可省略的过场，而是士气与身份确认。

2. **变身装置与口令**  
   手环、腰带、手机、钥匙、徽章、列车通行证等年度道具＋口令（「变身！」「合身！」「出发！」等）完成变身。早期五连者偏「転換」式着装显现，现代多电子／魔法驱动。装置绑定意志与编制，仿品常失灵或反噬。第六战士道具往往来源更黑、代价更大。

3. **周刊怪人与量产兵**  
   恶役组织派「本周怪人／怪物」执行绑架、实验、城市破坏；量产兵（ゾルダー、战斗员等）清场与人海。人型阶段以格斗、枪械、队技终结；怪人败北后常**巨大化**（成长液、复活咒、自爆膨胀、干部投放等），逼出巨型战。

4. **巨大机器人与合体**  
   自《战斗狂热J》起确立「个人机／兽机 → 合体巨神 → 必杀」语法；此后每作核心卖点。人型再强也难长期抗巨大体，**机体出动＝战力层级跃迁**。合体需队员协同、能量同步、指挥口令；断一色可能合失败。强化合体、最终机体、列车／恐龙／引擎等主题机型是中高阶压力源。

5. **恶役组织语法**  
   组织多有大首领／皇帝、数名干部、科学／魔法技术、移动要塞或异空间据点。目标：征服地球、采集资源、消灭战队、改写历史。干部轮换推动中期剧情；终局常揭首领真身或第二形态。部分作恶役侧有「可同情的怪人」或叛徒，但总览默认组织为敌。

6. **死亡与复活**  
   市民与多数量产兵真死；怪人可巨大化「续命」一次；干部有再生／分身；大首领与帝国级常有备份、要塞核心、宇宙舰队。战队侧偶有牺牲与归来（周年作尤甚），但非无限读档。无「打到没血即清空全系列宇宙」。

战力阶梯（总览用，对照《阶位战力图鉴》，**宁低勿高**）：

1. **普通人／目击者／疏散员**：无超凡。≈一阶以下～一阶初。  
2. **量产兵／组织外围特工**：冷热兵器、绑架、巷战。≈**一阶**。  
3. **本周怪人（人型）／新锐战士基础态**：毁车裂屋、碾压凡人小队。≈**一～二阶**。  
4. **稳定变身战士／中坚怪人／初期个人机驾驶员**：街区威胁、可压基础怪人。≈**二～三阶**。  
5. **强化形态／第六战士／干部怪人**：多街区～城区边缘。≈**三～四阶**。  
6. **合体巨神常规战／组织中坚干部指挥下的巨大战**：城市级局部。≈**四～五阶**（本档覆盖上沿）。  
7. **最终机体、帝国旗舰、大首领终局形态、跨作军团总动员**：地区～世界阴影。≈五阶上沿～更高阶分册／**超阶阴影**。

乐园阶位映射：量产兵～目击者≈一阶；基础战士与常规怪人≈二阶；强化态／干部／个人机≈三阶；合体巨神常规战≈四阶；城市级巨大战与中坚终局压力≈**五阶**（本档峰值，宁低勿高）；历代全队齐聚、帝国灭星舰队、神级大首领本体＝更高阶条目与**超阶阴影**。顶点：**超阶巅峰写存在·情报优先／条件性胜利**，严禁「被封印所以战力为零」。

**【地理 · 舞台】**
- **昭和系舞台包**：EAGLE等防卫组织日本支部、黑十字军式地下基地、东京近郊工业区与海岸要塞；巨大战常在空旷填埋场／山地避免写死全城。  
- **平成系舞台包**：各作自立都市（忍者隐村、魔法界入口、宇宙警察署、海盗船靠港的海岸、兽拳道场、侍屋敷等）——总览只作「可切换主题包」。  
- **令和系舞台包**：机械异世界通道、王权列岛、爆走公路、编号竞技场等；语法仍是「城市被袭 → 疏散 → 巨大战」。  
- **共通空间语法**：夜间商业街、学校操场、发电厂、码头集装箱、组织圆形标志大厅、爆破后的烟柱、巨神落地扬尘。  
- **跨作／周年**：历代战队现身时舞台可「叠层」，但本地市民记忆不必然连续；AI须声明本周主题包。

AI写场景先选**时代包＋本周组织名＋本周怪人真名**，再决定是否进入巨大化阶段。

**【世界剧情线】**
① **原点·石之森与五色队形（1975–1977）**  
《秘密战队五连者》定下系列基因：国际防卫组织EAGLE遭**黑十字军**摧毁日本各支部后，幸存者**海城刚（赤连者）**、**新命明（青连者）**、**大岩大太／后任熊野大五郎（黄连者）**、**佩姬松山（桃连者）**、**明日香健二（绿连者）**在司令**江戸川権八**下结成秘密战队。变身偏「転換」着装，手环主通讯；队技**五连者风暴／飓风**以传球踢击终结怪人；载具有飞行舰等，但尚未定型「每集合体巨神」。《JAKQ电击队》以扑克牌四色＋后期追加，走电子犯罪组织线，收视承压后进入约一年空窗。主题：秘密部队、色彩分职、组织复仇。

② **复活与巨大机器人语法确立（1979–1980s 昭和中后期）**  
东映与漫威合作特摄《蜘蛛侠》导入巨型机器人概念后，《战斗狂热J》把「人型战 → 巨大化 → 机体战」钉成系列骨架；《电子战队电磁人》《太阳战队太阳火神》等（部分与漫威协作）巩固科幻防卫队路线。《闪电战队变身人》《超电子生化人》《电击战队变化人》《超新星闪光人》《光战队面具人》《超兽战队生命人》等逐年更换主题（科学、生物、星、光、兽），完成「一年一主题、一队一机体」工业流程。生命人起更强调同学／青春与背叛阴影。此阶段结束时，超级战队已是东映特摄年轮支柱。

③ **平成开篇至「恐龙」与忍者潮（1989–1995）**  
《高速战队涡轮连者》跨入平成；《地球战队五人组》《鸟人战队喷射人》把家庭与爱情悲剧写进热血壳。《恐龙战队兽连者》以恐龙守护与美版《金刚战士》改编成为全球符号——**总览只记其历史位置与「恐龙主题＋第六战士」工业影响，剧情细节交分册**。《五星战队大连者》《忍者战队隐连者》推东方奇幻与忍者潮；《超力战队王连者》机器人文明压迫感加重。VS剧场与跨队共斗开始成为商业与叙事常规。

④ **喜剧、数码、宇宙与救难（1996–2000）**  
《激走战队车连者》自嘲式喜剧松绑沉重；《电磁战队百万连者》数码虚拟；《星兽战队银河人》星兽羁绊；《救急战队GOGO V》家族消防救难；《未来战队时间连者》时间警察与「官方超级战队」品牌在2000年前后正式收束早期两作进系列谱系。语法稳定：主题道具经济、中期追加战士、终盘干部崩坏与首领决战。

⑤ **平成中期·兽拳忍者警察魔法冒险（2001–2010）**  
《百兽战队牙吠连者》兽魂；《忍风战队破里剑者》忍者三＋暗之追加；《爆龙战队暴连者》暴龙与「觉醒」；《特搜战队刑事连者》宇宙警察署；《魔法战队魔法连者》家族魔法；《轰轰战队冒险者》遗迹冒险；《兽拳战队激气连者》拳法；《炎神战队轰音者》引擎公路；《侍战队真剑者》侍与封印；《天装战队护星者》天使与行星。此十年把「职业／文化主题」吃满，并为35周年铺垫「历代遗产可被引用」。

⑥ **周年海盗与平成后段实验（2011–2019）**  
《海贼战队豪快者》是总览级关键作：宇宙海贼小队持**连者钥匙**变装历代战队，对抗**残虐帝国赞基亚克**，把「超级战队史」本身做成主线道具与主题——自由、宝藏、继承。队长**キャプテン・マーベラス（豪快红）**、**ジョー・ギブケン（豪快蓝）**、**ルカ・ミルフィ（豪快黄）**、**ドン・ドッゴイヤー（豪快绿／博士）**、**アイム・ド・ファミーユ（豪快粉）**及后加入的**伊狩铠（豪快银）**等构成串线枢纽。《特命战队Go-Busters》任务部队写实风；《兽电战队强龙者》再恐龙；《烈车战队特急者》想像力列车；《手里剑战队忍忍者》忍；《动物战队兽王者》王者 cub；《宇宙战队球连者》九人星座；《快盗战队鲁邦连者VS警察战队巡逻连者》双队对立；《骑士龙战队龙装者》骑士龙。平成尾声用双队、多人数、列车与星座等实验队形上限。

⑦ **令和：机械、王权、爆走与编号（2020–2026）**  
《魔进战队煌辉者》宝石魔进；《机界战队全开者》全战队机械世界与平行；《暴太郎战队咚兄弟》桃太郎型人格崩坏喜剧；《王样战队国王者》虫王政治；《爆上战队奔奔者》载具爆走；《No.1战队五兽者》编号竞技至系列阶段性休止。朝日／东映宣布超级英雄时间档由新企划（如 Project R.E.D. 等）承接后，超级战队进入休止窗口，但跨作资产、剧场与未来重启可能仍存。总览锁定：**年度换队、主题换皮、语法不变**。

⑧ **跨作语法与蝴蝶边界**  
可改：本周怪人计划、救出被掳市民、延后一次巨大化、协助机体零件补给、让某场人型战易主。不可无代价改：取消「巨大化→巨神」工业语法、抹掉全系列历代存在、让一阶契约者单人永久摧毁所有帝国大首领、把Power Rangers美版设定直接覆盖日本正史队名。周年串线（豪快者、全开者、超级英雄大战）是「假期事件」：战力上限突然抬高，低阶应转疏散与后勤。

- **各作「本周怪人」与量产兵**｜必须以原作／该作设定真名登场（如黑十字军假面怪人各具名个体），禁止「怪物甲」。  
- **历代红战士群（总览调用时具名）**｜如喷射人的结城凯、真剑者的志叶丈瑠、强龙者的桐生大吾等——出场须报全名与所属队，禁止「红的那个」。

**【势力图谱】**
- **防卫／协力侧**：EAGLE式地球防卫、宇宙警察署、时间局、侍家、魔法家族、消防厅、任务部队等——各作自立，总览统称「战队侧编制」。提供基地、机体库、疏散权。  
- **侵略／暗组织侧**：黑十字军、残虐帝国赞基亚克、以及各作年度帝国（机械帝国、地下帝国、宇宙忍者团、恶魔族等）——干部＋怪人＋巨大化技术。  
- **灰色与跨界**：连者钥匙黑市、机体零件走私、VS电影临时同盟、超级英雄大战反派联合。  
- **市民与媒体**：疏散对象、目击证人、夸大战队或抹黑组织的舆论场。  
契约者最稳入口是**防卫组织外围／疏散编成／机体地勤**，而非直接抢红战士编制或大首领权杖。

**【贵重物品】**
- **变身手环／腰带／手机／钥匙等年度装置**：编制身份证明；失窃＝全队警报。总览禁止开局发完整正史队长器。  
- **连者钥匙（レンジャーキー）**：豪快者核心；可临时变装历代战队。总览中高阶可接触「一枚失窃钥匙」事件，不可开局图鉴全收集。  
- **合体巨神与个人机**：玩具工业与战场双重核心；驾驶资格＝战力跃迁。零件、能源槽、合体密钥是四～五阶争夺物。  
- **成长液／巨大化装置／复活咒具**：组织侧战略物资；夺取可推迟巨大战。  
- **组织徽章、干部指挥杖、要塞核心**：潜入与终局钥匙。  
- **队徽与点名背景卡**：仪式道具，亦是士气战。  
每件须写清：谁想要、抢到的后果、当前阶能否驾驶／启动。

**【隐藏剧情 · 伏笔】**
- 系列主题反复：**一个人英雄不够，要齐心**；颜色是分工不是种姓；巨大化是组织逼你升级战争的手段。  
- 「超级战队」品牌本身在2000年前后被官方回溯收编五连者／JAKQ，总览写作允许早期两作进谱系。  
- 豪快者把「历代」实体化为钥匙，全开者等令和作继续玩平行／全史引用——串线是遗产管理，不是抹平各作终局。  
- 美版《金刚战士》改名换角换剧情，**日本正史队名与人物优先**；可作平行影射，不可覆盖。  
- 玩具驱动导致形态膨胀：正文优先TV主形态与主合体，剧场限定机体当假期事件。  
- 2026休止窗口：世界状态可写「年度队空窗、旧组织余党、民间仿战队犯罪」，勿断言永久灭亡。

**【大事记时间线】**
1975 五连者开播，黑十字军战 → 1977 JAKQ → 1978 空窗 → 1979 战斗狂热J确立巨大机器人 → 1980s 昭和主题连作 → 1989 涡轮连者入平成 → 1992 兽连者／全球金刚战士影响 → 1990s 忍者·王连·车连·数码 → 2000 时间连者期品牌收束 → 2001–2010 牙吠至关星 → 2011 豪快者35周年串线 → 2012–2019 实验队形 → 2020 煌辉者入令和 → 2021 全开者全史玩法 → 2023 国王者政治巨构 → 2025–2026 五兽者与系列阶段性休止。判断世界状态用：本周怪人是否已巨大化、机体是否已出动、是否周年动员周。

**【叙事基调 · 雷区】**
基调：彩色爆炸特摄＋队形仪式＋巨大战高潮；可热血可悲壮，忌写成无队的个人骑士戏。  
忌把全系列捏成单一连续正史（市民人人记得五十年前赤连者）；忌一～二阶秒杀合体巨神或大首领；忌无变身无编制硬开正史机体；忌用「红蓝黄粉绿群像」代替真名；忌把恐龙战队分册剧情整段粘进总览；忌美版角色名覆盖日本队。  
最早切入：任意年度「本周怪人首次袭击、尚未巨大化」的商业街疏散夜。

**【共通机制深述·变身仪式】**
变身不是「穿上衣服」的过场，而是从市民时间进入战争时间的闸门。镜头语法通常是：装置高举或贴腰 → 口令 → 色块爆发 → 头盔锁定 → 队形点名。点名顺序多由红起或按固定队歌卡点。打断点名会动摇协同率，合体会出噪声。一～二阶契约者更常「目击变身」而非完成变身；若获临时协力装备，须有编制官或队长口头授权，否则组织侧会当仿冒者清除。

**【共通机制深述·人型战与队技】**
人型阶段强调分工：红突进、蓝牵制、黄破防、粉／绿机动或支援、黑／第六战士奇袭。队技前需要削弱：打掉怪人武器、破坏再生核、引离人质。秒杀会破坏特摄节奏，也让巨大化失去戏剧理由。量产兵战是契约者最安全的输出窗口：缴械、封路、救人，而不是抢最终踢。

**【共通机制深述·巨大化与巨神】**
巨大化触发后，战队侧优先**疏散**再合体。巨神战场地默认郊外或已清空街区；在市中心硬开必产生平民伤亡事件，影响任务评级。合体失败常见原因：队员失联、能源不足、钥匙／徽章被夺、干部干扰。五阶契约者可参与地勤同步、副驾驶、诱饵飞行器，仍忌独自驾驶最终机体无授权。

**【共通机制深述·恶役组织运营】**
组织不是只会喊征服：有资源配额、干部政争、实验伦理（拿市民当材料）、对战队的情报战（假新闻、假投降、仿战士）。黑十字军式「假面怪人主题」与赞基亚克式「宇宙帝国主义」是两种模板——前者偏秘密结社美学，后者偏舰队与殖民。写干部时给真名与军阶，禁止「干部A」。

**【代表作脉络·五连者锚点】**
EAGLE日本支部连环被毁，五名幸存者在零食店式秘密基地与司令连线，周刊对抗黑十字军假面怪人。队技传球风暴把「齐心」做成物理动作。姓氏头文字可拼出仙后座等伏笔，终局对総統与要塞。总览用此作定义：秘密部队、五色、组织复仇。

**【代表作脉络·巨大战工业化】**
战斗狂热J以降，每年机体设计驱动叙事中期「强化合体」与终盘「最终巨神」。电磁人、太阳火神等巩固防卫队编制感；生命人把青春友情与背叛写进机体座舱。总览任务可复用：「机体试作机被抢」「合体密钥运输」「巨大化液工厂爆破」。

**【代表作脉络·平成主题超市】**
喷射人的爱情与牺牲、兽连者的恐龙守护（分册）、隐连者的忍者喜剧、刑事连者的宇宙警察程序、魔法连者的家族咒语、真剑者的侍与封印——主题不同，周刊结构同构。总览切入时只取「一句主题＋真名队长＋本周组织」，不整季剧透分册。

**【代表作脉络·豪快者总览钥匙】**
赞基亚克毁地球防卫后，海盗们为宝藏与自由而战，却在收集钥匙的过程中被迫理解「战队是什么」。马伟拉斯拒绝被帝国与「正确」收编；伊狩铠用宅知识把遗产翻译给观众。连者钥匙失窃案是四阶黄金任务：钥匙能开历代能力，也能引来帝国与旧队员同时追杀。

**【代表作脉络·令和实验】**
全开者把机械世界与全战队遗产再玩一次；咚兄弟拆解英雄人格；国王者把「王」与国家战争抬到政治剧；奔奔者回归载具速度感。休止期可写：旧机体封存库被盗、民间「伪战队」直播打怪、残党干部招募失意青年。

**【地理包切换规则】**
开场必须声明：昭和黑十字巷战／平成宇宙警察署／海贼靠港夜／令和王权列岛等。切换包＝切换组织名、变身器名、机体名。不可让黑十字军量产兵无说明地开赞基亚克飞船，除非是串线假期事件并付代价。

**【力量体系·编制与能源】**
战力不只看个人格斗：编制权限决定你能否呼叫机体、动用队库武器、进入基地。能源槽、合体同步率、市民疏散完成度是隐性资源。仿变身器黑市货在一～二阶可当诱饵，三阶以上真货争夺会引来整队。

**【主要人物·补全调用说明】**
写海城刚时带队长口吻与红鞭意象；写马伟拉斯时带海贼直球与「宝藏」口头禅；写ドン时带维修工具与害怕巨大化仍上舰；写黒十字総統时用变声器与要塞阴影，不写日常闲聊。每人出场带道具与一句立场。

**【贵重物品·总览级下落】**
五连者手环可在EAGLE废墟通讯室；连者钥匙可在赞基亚克回收舱或黑市冷柜；合体密钥常在机体库双人认证保险柜；巨大化液桶有辐射标与组织纹样。真队长变身器绑定意志，转让即剧情级事件。

**【隐藏·互文与工业】**
石之森队形伦理 → 八手三郎工业年更 → 豪快者遗产自觉 → 令和玩解构与政治。超级英雄大战与假面骑士跨界是假期，不覆盖各TV终局。Power Rangers是平行改编产业链。

**【大事记扩写判断】**
用三个开关判断世界状态：①怪人是否已巨大化；②巨神是否已合体成功；③是否周年／VS动员。开关全关＝一～二阶日常袭击；开①＝三阶危机；开②＝四～五阶；开③＝全队阴影，低阶转后勤。

**【叙事雷区扩写】**

**【系列语法战役志】**
每一集几乎是完整小战争：诱拐或实验、怪人展示、战队苦战、发现弱点、队技、巨大化、巨神必杀、留下下周伏笔。一阶活在诱拐与报警，二阶活在人型并肩，三阶活在强化与干部，四阶活在合体边缘，五阶活在城市级巨大战与帝国中坚。把节奏写乱就失去特摄感。

**【跨时代误入规则】**
若契约者持错误时代道具（昭和巷战掏出连者钥匙），世界以「异常反应／组织回收／历代相关者感应」处理，不自动开挂。串味必须付代价：被两方同时追杀、钥匙过热、机体拒认。

**【组织一句志】**
黑十字军：秘密假面征服；赞基亚克：宇宙残虐殖民；宇宙警察对立犯罪组织：程序与越权；侍之敌封印族：怨念轮回；机械帝国：效率抹杀个性。写组织先写宗旨再写怪人。

**【人物弧光速查】**
海城刚：幸存者→原点队长；佩姬：技术粉战士；黒十字総統：原点首领阴影；马伟拉斯：海贼自由→遗产继承；ジョー：帝国刀→伙伴剑；ドン：怕死维修工→海上博士；伊狩铠：宅→银；江戸川：司令父。调用时化成对白与道具。

**【宝物与权限】**
旋风式个人载具钥匙、队库通行证、连者钥匙单枚、巨大化液样品、机体涂料样本、组织干部名簿残页。真合体权杖级道具仅五阶任务末段短暂接触，用后须归还或自毁。

**【伏笔总览】**
颜色＝分工；点名＝意志对齐；巨大化＝战争升级按钮；巨神＝齐心的工业身；钥匙＝历史可被持有也可被盗。五十年换皮不换骨：齐心对组织。

**【周刊怪人命名纪律】**
本周敌人必须有可呼叫的名字与主题（面具、动物、物件、星、引擎……）。击败后巨大化要有触发物或台词。禁止「未知怪物突然出现又无因消失」。

**【第六战士与强化态】**
第六战士常带来编制震荡：原五人是否接纳、力量是否来自敌方技术、市民是否只认五色。强化态（武装、超、最终）是中期玩具波，正文给冷却与代价：能源耗尽、失控、暴露身份。

**【市民侧真实】**
战队世界市民会逃、会拍、会误会、会成立应援、会卖假周边。疏散完成度影响巨神能否在市区外决战。契约者救下的记者可能把你拍成「神秘第七人」引来组织。

**【机体库政治】**
谁有权批准出动、谁背平民伤亡责任、谁维护最终机体，是四～五阶人际冲突。技术人员比红战士更早知道能源不够合体。抢功的政治家干部比怪人更难打。

**【周年动员规则】**
历代现身时：战力上限上升、镜头拥挤、低阶角色应去救市民与守机库。Movie结束主线回弹，但建筑损毁与牺牲可保留。禁止用周年当借口让一阶打赢大首领。

**【休止期世界态】**
官方队空窗≠和平。余党、仿队、黑市钥匙、封存机体被盗是主菜。可出现「民间防卫团」邀请契约者，但其变身器合格率极低。

**【与假面骑士跨界边界】**
超级英雄大战等允许同框，总览仍以战队语法为主：有队才有巨神。骑士的个人踢击不能替代合体必杀的叙事功能，除非剧情明确借用。

**【Power Rangers平行声明】**
美版改名、换演员、改剧情。本世界书默认日本队名与人物；若玩家提及美版，作平行影射或「海外纪录片误译」，不覆盖正史。

**【契约者身份白名单】**
目击者、记者、消防／警察外围、防卫组织见习、机体厂技术员、黑市中间人、被救人质、编外协力者、临时追加试验体（高风险）。红战士正史编制需剧情长线挣，不靠开局抽取。

**【失败与代价】**
人型战失败→人质损失；疏散失败→巨大战评级崩；合体失败→城区毁一角；钥匙失控→错误历代能力反噬；硬刚大首领→团灭级事件。允许惨胜，不允许无代价开挂。

**【对话口吻参考】**
红：短句命令与鼓舞；蓝：数据与冷静纠正；黄：大嗓门或义气；粉：关怀与刺；绿：吐槽或技术；海贼红：直球宝藏与自由；司令：先救人。禁止全员同一励志腔。

**【音乐与仪式感（叙述用）】**
变身BGM、点名卡点、合体歌、必杀前静止帧——叙述中用「卡点」「合唱感」「鼓点」暗示，不必写真实曲名版权。打断仪式＝打断士气。

**【总览与分册边界再申明】**
恐龙、侍、警察、魔法等单作的完整人物弧与终局以分册为准。总览只提供可切换舞台包、共通机制、串线枢纽与阶位标尺。写总览任务时点到即止，避免整季剧透复读。

## 阶位切入点

> 阶位↔原作：一阶≈量产兵／目击者／疏散层；二阶≈基础变身战士与常规怪人巷战；三阶≈强化态／第六战士／干部怪人与个人机；四阶≈合体巨神常规战与组织中坚战役；五阶≈城市级巨大战与帝国中坚终局压力（本档峰值）。历代全军、灭星舰队、神级大首领本体＝更高阶／超阶阴影。低阶切入一律规避顶点·情报优先；**超阶巅峰：存在·情报优先／条件性胜利**。

**一阶（量产兵巷战·目击者疏散）**

切入身份/时点：便利店夜班、消防见习、防卫组织外围通讯员、学校疏散志愿者；锚定任意年度「本周怪人首次袭击、尚未巨大化」前后，尤适昭和黑十字军活动期或平成某作开播第1–3话窗口。

初始事件：商业街警报响起，黑十字军式（或该作）量产兵封锁路口，怪人点名要带走一名工程师。你的对讲机里闯进**江戸川権八**式司令频道（或该作基地广播）：「先带市民进地下，别逞英雄。」彩色烟雾在街角炸开，你还没看清战士的脸，先看清自己有没有退路。

开场白建议：「警报把玻璃窗震成鼓面。黑色面罩的量产兵端枪推开人群，怪人的主题纹样在路灯下反光。远处有人高喊变身口令，五色光还没聚齐——你手里只有扩音器和对讲机，第一任务写着：活着，并把孩子带进地铁口。」

关键NPC立场：**江戸川権八**（或该作司令）要疏散数据不要送死；**海城刚**若在场会先救人再盘问；量产兵队长要清场；**本周怪人**视你为人质或材料；市民只求活；记者想拍英雄不顾你死活。

主线钩子/支线：完成一条街疏散、拍摄基地入口却不深入、给战队报信怪人弱点线索；支线＝假新闻辟谣、护送工程师家人、缴获量产兵通讯器。

危险度/规避：中。避干部与大首领；避主动冲进人型决战圈；避偷变身器。

任务方向/奖励：协力者袖章、疏散功勋、组织行动时间表残页、基础防护面罩。禁止正史变身器与机体钥匙。

**二阶（基础战士并肩·人型歼灭）**

切入身份/时点：编外协力者、逃亡实验体、机体厂夜班保全、宇宙警察署临时翻译；锚定战队已稳定五人变身、第六战士未至、干部尚未下场的窗口。

初始事件：你持半枚仿队徽混进组织外围仓库，惊动本周怪人。红战士落地要求你选：当诱饵引怪人离开居民区，或带路去剪巨大化液管线。仓库顶灯映出组织纹章，倒计时滴答。

开场白建议：「变身气流掀起你的衣领。红战士只问一句：平民撤了没？你口袋里的仿徽烫得像罪证——选错边，下一秒队技光里不会有人记你的名字。」

关键NPC立场：**海城刚／本周红**冷静分配；**新命明／本周蓝**要路线图；**佩姬松山／本周粉**要技术样本；干部还未现身但通讯里低语；量产兵增援在路上。

主线钩子/支线：与战队夹击怪人、破坏巨大化液、夺回被盗通讯手环（交还非自用）；支线＝假战士骗局、学校被占为人质、黑市卖仿变身器。

危险度/规避：中高。避巨大化后的正面硬抗；避干部本体。

任务方向/奖励：正式协力编制、队库基础武装借用、怪人弱点笔记。禁合体驾驶权与最终形态。

**三阶（强化态／第六战士／个人机边缘）**

切入身份/时点：临时追加试验者、第六战士相关技术员、个人机地勤、被组织盯上的「适任者」；锚定中期强化武装登场或第六战士加入前后。

初始事件：组织干部投放「强化怪人」，基础五人队技无效。你被塞进实验外骨骼或半成品变身器，同步率只有六成——要么当诱饵拖住干部，要么把能源罐送到个人机弹射轨。金／银光在天际划过，第六战士尚未决定信不信你。

开场白建议：「外骨骼关节咬合声像倒计时。干部的声音从扩音器落下：把适任者带回来。你看见个人机的眼睛亮起，红战士在频道里吼：能走就走，走不了就拖——别死在强化光里。」

关键NPC立场：**伊狩铠**式追加者狂热却守规则；**ドン・ドッゴイヤー**要你别乱改线路；干部要活捉；原五人队对「编外力量」态度分裂；市民把你的半成品装甲当成新怪人。

主线钩子/支线：协助第六战士初次联手、夺回强化配件、阻止干部回收你的适任数据；支线＝个人机试飞事故、仿强化武装流入黑市、队友不信任仲裁。

危险度/规避：高。避大首领；避无授权启动合体。

任务方向/奖励：临时变身时限卡、个人机副驾驶一回、强化零件样本（须上交一半）。禁最终巨神主驾驶。

**四阶（合体巨神常规战）**

切入身份/时点：机体库同步官、副驾驶候补、合体密钥押运、帝国侧反正的技术官；锚定「怪人已巨大化、主合体可出击」的标准巨大战周。

初始事件：巨大化液在码头爆开，怪人踩碎起重机。五色战机升空，缺一名同步官——你的生物波刚好匹配。密钥箱被量产兵抢走三成路程，不夺回则合体失败、城区失守一角。

开场白建议：「警报从人声变成舰桥蜂鸣。巨神的影子还没合拢，城市的灯却一排排灭。红战士在频道里报点名，最后空出来的那个位置，光标停在你的名字上。」

关键NPC立场：**キャプテン・マーベラス**会问你是逃跑还是上船；**ジョー・ギブケン**用剑指你是否叛徒；**江戸川権八**式司令要伤亡最小化；帝国干部要击落合体中的战机；市民在避难所看直播尖叫。

主线钩子/支线：夺回合体密钥、完成同步、执行一次标准必杀协助；支线＝假密钥调包、能源槽泄漏、干部分身袭击座舱。

危险度/规避：高。避最终机体与大首领终局形态；城市中心硬刚需先完成疏散指标。

任务方向/奖励：副驾驶认证、机体涂料／数据残片、组织干部行动预测。禁永久主驾驶最终巨神。

**五阶（城市级巨大战·帝国中坚）**

切入身份/时点：战队侧特任协力指挥官、失窃连者钥匙回收专员、帝国中坚干部谈判破裂现场的第三方；锚定周年动员或帝国旗舰压境、主合体吃紧需强化合体／多机体的窗口（仍低于灭星舰队与神级首领本体）。

初始事件：赞基亚克式巡洋舰投影笼罩海湾，中坚干部放出第二巨大体。历代钥匙反应让你体内／背包中的单枚钥匙过热——不用则巨神能源不足，用则引来旧队员与帝国双线锁定。马伟拉斯在频道里只丢一句：「宝藏可以晚点找，城市不能晚。」

开场白建议：「旗舰的阴影把白昼压成黄昏。巨神的装甲上全是旧战的刮痕，同步灯乱闪。你掌心的连者钥匙烫得像要穿透皮肤——它在叫某一队的名字，而帝国的第二只巨手已经抬起。」

关键NPC立场：**キャプテン・マーベラス**要结果不要借口；**ジョー・ギブケン**防你叛变；**ルカ・ミルフィ**护平民优先；**アイム・ド・ファミーユ**坚持底线；**伊狩铠**能背出钥匙对应队但不知你的来历；帝国中坚要钥匙与你的命；**黑十字総統**级存在仅作通讯阴影勿下场。

主线钩子/支线：在限定时限内完成强化合体或双机体夹击、回收／归还钥匙、炸断旗舰降落锚；支线＝假投降谈判、市民人质轨道电梯、历代影像战士短暂援助后消失。

危险度/规避：极高。避大首领本体、灭星级舰队总攻、全历代无代价齐聚开挂。**超阶巅峰：存在·情报优先／条件性胜利**，严禁把顶点写成战力归零。

任务方向/奖励：特任编制、单枚钥匙保管权（任务后强制上交或封存）、强化合体数据副本、帝国中坚掉落的指挥纹章。禁止开局灭世兵器与永久全钥匙图鉴。

一阶扩展：假新闻指你是战队同伙；量产兵要征用你的卡车；孩子把你认成黄战士要签名。  
二阶扩展：仿变身器拍卖；人质学校；剪巨大化液被干部通讯警告。  
三阶扩展：适任者名单泄露；第六战士考验；个人机拒认你的生物波。  
四阶扩展：密钥调包；同步率被黑入；城区一角成为必杀牺牲辩论。  
五阶扩展：钥匙过热叫错队名；旗舰锚；谈判桌下的第二巨大体。
