## 剧情

**【作品来源】**
《魔兽世界：巨龙之魂》（Dragon Soul，简称 DS）是暴雪娱乐 MMORPG《魔兽世界》大灾变资料片最终团本，随补丁 4.3.0「巨龙之魂」于 2011 年 11 月 29 日上线，10/25 人难度，入口叙事经时光之穴接入龙眠神殿战役。设定以游戏本体冒险指南、副本日志、首领台词、官方社区文案、小说《Aspects 的冲锋》《萨尔：暮光之龙》及编年史为准；前序为暮光之刻五人本与永恒之井/时光之末链，后日谈衔接熊猫人之谜前夜。文风：末日史诗、龙族权柄、旧神触须、飞空舰追猎、麦尔斯特罗姆终战。本档案只锚定「巨龙之魂战役」主舞台与直接前史/即时后果，不把整本大灾变或巨龙群岛倒灌为同一场。

**【世界定位】**
诺森德龙骨荒野，泰坦为龙族所建的**龙眠神殿**遭死亡之翼大军围攻；萨尔携自时光之穴取回的**巨龙之魂**（古称恶魔之魂）与红蓝绿青铜四位守护巨龙（阿莱克丝塔萨、卡雷苟斯、伊瑟拉、诺兹多姆）集结于此，试图以神器反制黑龙魔王。契约者切入的是：从神殿外围解围、潜入无面者裂口、夺聚焦虹膜、为巨龙之魂充能、登上天火号追击、攀死亡之翼之脊，直至麦尔斯特罗姆「疯狂的死亡之翼」终战。一句话：**这是艾泽拉斯以龙族权柄与凡人远征军联手「抹除」大地守护者的末日战役，不是普通矿洞刷装。**

**【世界观 · 力量体系】**
力量本源：泰坦授予的龙族守护权柄（生命/魔法/时光/梦境/大地）、元素与奥术、旧神腐蚀、元素金属（源质装甲）、圣光/萨满自然力、凡人团队战术。死亡规则：凡人与多数精英战死即死；死亡之翼在脊背战被打穿后坠入麦尔斯特罗姆，疯狂阶段被巨龙之魂「抹除」——原作终点为**真正死亡**（非简单放逐）；守护巨龙在终战后失去不朽权柄、化为凡人。特殊系统：① 10/25 人团队协同；② 巨龙之魂充能与聚焦虹膜稳定；③ 天火号舰载炮与伞降；④ 脊背装甲板剥离；⑤ 守护者权柄通道（治疗/护盾/减速等场景 buff）；⑥ 大灾变声望与龙魂掉落装（含源质与暮光材料）。

战力阶梯（按破坏力表现，宁低勿高）：
1. **龙眠外围民兵 / 暮光苦力 / 新兵**：冷兵器与火枪极限。≈一阶初。
2. **暮光教徒 / 黑龙军团外围斥候 / 神殿底层守卫**：有组织斧盾与暗影。≈一～二阶。
3. **龙眠中层精英 / 暮光龙兽骑手 / 无面者腐化者**：可清小队、控据点。≈二～三阶。
4. **摩诺克级门槛前卫 / 地怒者与暮光攻城单位**：一域堡垒压迫。≈三～四阶。
5. **摩诺克、战争大师佐诺兹、不眠的约萨希、缚风者哈格拉**：团本级，可屠灭未准备团队。≈四～五阶。
6. **奥卓克希昂、战争大师黑角、死亡之翼之脊**：区域灾难至方面级追猎。≈五～六阶。
7. **疯狂的死亡之翼（麦尔斯特罗姆终战态）与充能后的巨龙之魂联手一击**：可改写大陆与位面裂口。≈六～七阶。
8. **全盛死亡之翼（大灾变开幕撕开艾泽拉斯）/ 守护巨龙全盛态**：行星/方面级对照。≈七～八阶。
9. **旧神本体（尤格-萨隆等）、泰坦级**：超阶，不在 DS 可击杀表。

乐园阶位映射：龙眠外围杂兵≈一～二阶；神殿内精英与无面者中层≈二～四阶；DS 前中段首领≈四～六阶；**奥卓克希昂/黑角/脊背≈六阶**；**疯狂的死亡之翼≈七阶（本档顶点可打目标）**；大灾变开幕全盛死亡之翼与旧神级=超阶·情报优先。本条目覆盖一至八阶；八阶写「权柄消散后的余波与旧神阴影」，非让契约者在 DS 内单挑未削弱的开幕态死亡之翼。

**【地理 · 舞台】**
- **龙骨荒野 / 龙眠神殿外围**：冻结荒原、泰坦之路、暮光攻城线；契约者低阶日常舞台。
- **龙眠神殿本体 / 龙眠之巅**：红蓝绿青铜黑五座龙殿环绕的尖塔；守护者与萨尔充能点。
- **毁灭之喉 / 苟拉斯之喉 / 舒玛之喉**：旧神触须裂口，无面者将军与血肉软泥涌出。
- **永恒之眼（龙魂版本）**：原玛里苟斯圣所，哈格拉设伏夺聚焦虹膜处。
- **冰封海上空 / 天火号（The Skyfire）**：联盟炮舰追击死亡之翼的空中战场。
- **死亡之翼之脊**：源质装甲板、腐蚀触须、畸变融合体；伞降落点。
- **麦尔斯特罗姆（大漩涡）**：艾泽拉斯中央漩涡；终局岛屿与触手平台。
- **时光之穴（前史入口）**：取回巨龙之魂的时间锚，本档不展开五人本细节。

AI 写场景：先定契约者在「神殿外围 / 无面者裂口 / 永恒之眼 / 天火号甲板 / 脊背 / 漩涡岛」哪一层，再选冲突——外围是补给与教徒渗透，中层是无面者与暮光龙，深层是死亡之翼本体。

**【世界剧情线】**

**一、前史：大地守护者的堕落与恶魔之魂**
奈萨里奥原为泰坦赋予的大地守护者，因旧神低语与大地重负而疯狂，改称**死亡之翼**。上古之战中他锻造**巨龙之魂**，吸取其他龙族权柄，后被摧毁。大灾变中他自深岩之洲撕裂世界之柱，重塑艾泽拉斯地貌，并推动「暮光审判」——解放旧神、灭绝生灵。其子奈法利安、奥妮克希亚等黑龙线与暮光龙军团、暮光之锤教团构成其爪牙网。世界状态：大陆裂开，元素领主与教团四起；守护巨龙在海加尔与暮光高地连战，仍无法正面击杀全盛死亡之翼。

**二、取回巨龙之魂**
在《Aspects 的冲锋》与补丁 4.3 叙事中，**萨尔**与守护者经**时光之穴**取回毁灭前的巨龙之魂。神器仍带死亡之翼的大地本质，凡人强行全功率使用可能反噬致死。**卡雷苟斯**提出以永恒之眼的**聚焦虹膜**稳定输出。契约者低阶主舞台：护送萨尔车队、防暮光刺客劫魂、龙眠外围声望与补给。

**三、围攻龙眠：摩诺克**
战役自暮光之刻结束后直接衔接。死亡之翼军围神殿；**德弗莱斯塔兹领主**率凡人与龙族联军反击。地基被**摩诺克**（宝石巨像/大地元素暴君）捶击，意图推倒尖塔。英雄击破摩诺克后，神殿暂时稳住。名场面：泰坦之路上元素地震与暮光攻城器交叉。阶段结局：外围牙齿被拔，真正威胁转入地下裂口。

**四、无面者裂口：佐诺兹与约萨希**
**泰莉苟萨**警告：死亡之翼召唤极强无面者。**战争大师佐诺兹**自苟拉斯之喉出击，操纵「不灭虚空」弹道；**不眠的约萨希**自舒玛之喉涌出，召唤六色血肉软泥（酸/钴/猩红/暗影等），组合即团灭公式。阶段结局：两处旧神裂口被压制，神殿可内推进。因果：旧神触须证明死亡之翼背后仍是「深渊逻辑」。

**五、永恒之眼：缚风者哈格拉**
守护者欲充能巨龙之魂，暮光之锤先知先机，派**缚风者哈格拉**率萨满精锐在永恒之眼设伏。冰晶绑定、雷电相位与聚焦虹膜争夺战。哈格拉败后虹膜归队。名场面：冰封平台上雷暴与冰墙交替，虹膜被暮光符文锁死前必须打断。阶段结局：稳定装置到手，充能可开始。

**六、龙眠之巅：奥卓克希昂**
守护者向巨龙之魂注入权柄时，死亡之翼派终极造物**奥卓克希昂**——暮光龙实验顶点，专职在此刻杀死守护者、完成暮光审判。天幕变暮光，龙群先被清扫，随后奥卓克希昂本体降临。击败后巨龙之魂充能完成，萨尔一击打落死亡之翼背部一块源质装甲。死亡之翼摧毁一艘部落炮舰后逃向麦尔斯特罗姆，意图经裂隙遁入深岩之洲。

**七、天火号：战争大师黑角**
幸存联军登上联盟炮舰**天火号**，**天空上尉斯威泽**指挥追击。途中**战争大师黑角**（暮光之锤，骑暮光龙戈里欧纳）率龙骑士与破坏者袭击引擎。英雄须守甲板、击落龙骑、阻止工兵引爆，最终击杀黑角。名场面：舷侧炮火、伞降预备、引擎舱抢修。阶段结局：航线恢复，追上死亡之翼。

**八、脊背与疯狂：终局**
英雄伞降至**死亡之翼之脊**，剥除源质装甲板，杀死腐蚀触须与畸变融合体，为萨尔打开射击窗口；第二发巨龙之魂洞穿胸膛，死亡之翼坠入麦尔斯特罗姆。众人以为结束，**诺兹多姆**警告未完——**疯狂的死亡之翼**自漩涡升起，触手、利爪、翅膀与 Tentacle 平台多阶段战。守护者倾尽剩余权柄，经巨龙之魂将其**抹除**；四位守护者失去不朽，化为凡人；巨龙之魂被时光之力送回取走的时间点。阶段结局：死亡之翼死；大地守护者权柄空缺；艾泽拉斯进入「无守护者」时代。

**九、结局方向与边界**
- **本档胜利条件**：解围龙眠、压制无面者、取得虹膜、充能巨龙之魂、击破奥卓克希昂与黑角、剥脊、抹除疯狂的死亡之翼。
- **不在本档必演**：奈萨里奥早年全史、巨龙群岛觉醒、拉希奥黑龙复兴——可作情报远景。
- **政治后果**：部落与联盟因共同末日短暂协作；守护者凡人化改写龙族政治；旧神未灭，仅最大爪牙倒下。

**十、可介入事件链（供任务生成，皆具名）**
暮光密探混入龙眠补给线 → 假聚焦虹膜黑市案 → 摩诺克地震时有人故意拆神殿支撑符文 → 佐诺兹虚空弹被第三方研究机构偷测 → 约萨希软泥样本流入炼金黑市 → 哈格拉冰晶锁虹膜时内奸向暮光报时 → 奥卓克希昂降临前有人试图私下「谈判」卖守护者位置 → 天火号引擎被工兵安放第二组炸药 → 脊背装甲板碎片争夺（源质黑市） → 麦尔斯特罗姆终战后有人打捞「未熄的腐蚀血」作旧神祭品。

① 龙眠尖塔在暮光炮火中摇晃；② 摩诺克双拳砸地基；③ 佐诺兹「不灭虚空」弹墙；④ 约萨希六色软泥融合；⑤ 哈格拉冰雷相位切换；⑥ 奥卓克希昂天幕暮光；⑦ 萨尔第一发射落装甲板；⑧ 天火号甲板龙骑俯冲；⑨ 伞降钉上死亡之翼之脊；⑩ 腐蚀触须从装甲缝钻出；⑪ 第二发洞穿胸膛坠入漩涡；⑫ 疯狂的死亡之翼自麦尔斯特罗姆升起；⑬ 四守护者倾尽权柄；⑭ 抹除后的寂静与凡人化。

**十二、契约者行为边界**
可改：救人、抢非核心掉落、让某次攻城易主、提前泄露哈格拉设伏、选择源质碎片归属、伪造或真实护送虹膜的节奏。不可无代价改：让死亡之翼无条件投降并在大灾变开幕前消失、抹除旧神、杀死未进入龙魂战役的全盛开幕态死亡之翼、把守护者写成终战后仍保有完整不朽权柄。胜利若来得太干净，世界会用下一场元素喷发、下一支暮光余党或下一次旧神低语把账讨回。

**【主要人物】**

- **死亡之翼 / 奈萨里奥**（黑龙之王，大地守护者堕落体，本档顶点）｜性格：暴虐、痛苦驱动的憎恨、视生灵为折磨的延伸。｜装备·能力：源质装甲、熔岩吐息、地震与裂空、暮光龙与无面者军团、旧神赋能；开幕态可撕大陆，龙魂战役中为可被条件性击杀的终局态。｜人物弧光：大地守护者 → 旧神腐蚀 → 恶魔之魂 → 大灾变 → 龙眠围攻 → 脊背与疯狂 → 被抹除。｜立场关系：契约者是「待焚的昆虫」；对萨尔与守护者是死敌；对旧神是工具性效忠。

- **战争大师黑角**（暮光之锤将领）｜性格：好战、舰战专家。｜装备·能力：骑龙戈里欧纳、甲板突袭。｜人物弧光：天火号航线拦路者。｜立场关系：空战政治——拖住天火号即放跑主子。

- **德弗莱斯塔兹领主**（红龙，神殿防卫指挥）｜性格：刚烈。｜装备·能力：红龙火力与凡军调度。｜人物弧光：外围解围关键 NPC。｜立场关系：低中阶任务板。

- **天空上尉斯威泽**（天火号舰长）｜性格：硬派、嘴硬心热。｜装备·能力：舰载火力指挥。｜人物弧光：追击阶段接口。｜立场关系：契约者登舰需服从军纪。

- **泰莉苟萨**（蓝龙，情报）｜性格：冷静。｜装备·能力：奥术侦测。｜人物弧光：警告无面者入侵。｜立场关系：中阶情报线。

**【势力图谱】**
- **死亡之翼直属（黑龙军团 / 暮光龙 / 源质军工）**：宗旨=暮光审判与旧神解放；地盘=深岩裂口、龙骨攻城线、麦尔斯特罗姆逃路；代表=死亡之翼、奥卓克希昂、脊背腐蚀体。
- **暮光之锤**：宗旨=旧神崇拜与末日仪式；代表=哈格拉、黑角、教徒渗透网；与无面者协同。
- **无面者（旧神爪牙）**：宗旨=腐蚀与裂口扩张；代表=佐诺兹、约萨希；苟拉斯/舒玛之喉。
- **龙眠联军（四守护者 + 萨尔 + 凡军）**：宗旨=充能巨龙之魂、抹除死亡之翼；代表=阿莱克丝塔萨等与德弗莱斯塔兹。
- **联盟 / 部落远征分队**：提供炮舰、步兵与政治合法性；天火号为联盟资产。
- **大地之环**：萨尔所属萨满组织；元素侧支援与战后安抚。

**【贵重物品】**
- **巨龙之魂（恶魔之魂）**：上古神器；来历=死亡之翼锻造；本档由萨尔持有充能；暴露即引全世界夺宝。
- **聚焦虹膜**：永恒之眼核心；稳定巨龙之魂输出；哈格拉线关键。
- **源质装甲板 / 源质碎片**：死亡之翼脊背材料；黑市与锻造硬通货。
- **龙魂团队装备（含泰坦级饰品武器）**：改变全服 T12/T13 结构。
- **暮光龙卵 / 无面者软泥样本**：禁研材料；教团与炼金疯抢。
- **天火号通行与舰载权限**：身份与后勤。
- **守护者权柄残响（终战后）**：凡人化后的残留祝福，高阶稀有。

**【隐藏剧情 · 伏笔】**
- 巨龙之魂取回时间线本身是诺兹多姆允许的「必要悖论」；乱改取魂点可崩青铜龙承诺。
- 死亡之翼的痛苦部分来自大地权柄与旧神双压——他不是无脑火龙，是「想卸下地球的疯子」。
- 奥卓克希昂的设计目标是「此刻杀死守护者」，说明暮光审判有精确时间表。
- 终战后守护者凡人化，为后续龙族政治与巨龙群岛留下空位——本档点到为止。
- 旧神未死；无面者裂口是「深渊仍睁眼」的证据。

**【大事记时间线】**
- 上古 → 奈萨里奥铸巨龙之魂；后被毁。
- 大灾变开幕 → 死亡之翼撕裂世界之柱，重塑大陆。
- 4.3 前 → 时光之穴取回巨龙之魂；暮光之刻护送萨尔。
- 龙眠战役 → 摩诺克→佐诺兹/约萨希→哈格拉→奥卓克希昂。
- 追击 → 天火号→黑角→脊背。
- 终局 → 麦尔斯特罗姆疯狂的死亡之翼被抹除；守护者凡人化。
- 之后 → 无守护者时代开启（后日谈）。

**【叙事基调 · 雷区】**
画风：末日硫磺与极地冷风、龙吼压过军号、旧神触须的湿冷、炮舰甲板上的油烟。忌：① 把死亡之翼写成无脑火球，丢掉「堕落守护者+旧神工具」史诗感；② 让低阶契约者单挑开幕全盛死亡之翼；③ 用「他被削弱所以战力为零」解释胜利——胜利应来自巨龙之魂充能、虹膜稳定、团队战术与守护者倾尽权柄；④ 终战后仍写守护者完整不朽；⑤ 编造原作没有的「死亡之翼存活暗桩」为主结局。最早切入锚点：龙眠神殿围攻前夜，或暮光之刻结束后萨尔携魂抵达时。


**十三、大灾变政治与龙魂战役的位置**
大灾变主线横跨元素位面、海加尔、暮光高地、奥丹姆与火焰之地。巨龙之魂是这些线的收束：当拉格纳罗斯在火焰之地被永久击杀、奥拉基尔在风神王座倒下、古加尔在暮光堡垒败亡之后，艾泽拉斯仍无法正面解决死亡之翼——他不是「又一个元素领主」，而是被旧神改写的泰坦级代理人。龙眠战役因此具有「文明级总攻」性质：部落与联盟在炮舰甲板上暂时把旗帜放低，大地之环把萨尔推到神器之前，龙族把最后的权柄押进一枚金盘。契约者若只把 DS 当成「T13 农场」，会错过正文真正的戏剧张力——每一次充能都是一次文明认赌。

**十四、巨龙之魂的技术与伦理**
巨龙之魂（恶魔之魂）的本质是「吸取并重定向龙族权柄」。萨尔能用它，是因为死亡之翼把自己的大地本质铸进了盘中，而萨尔作为萨满与「世界的伤」同频；但卡雷苟斯反复警告：全功率释放可能把持有者烧成焦炭。聚焦虹膜的作用不是「增伤饰品」，而是把输出曲线压进可存活区间。写正文时，虹膜被夺=充能中断=守护者暴露在奥卓克希昂的猎杀窗口；假虹膜、裂虹膜、被暮光符文污染的虹膜都是中高阶可玩骗局。伦理上，使用敌人的神器反杀敌人，会在龙族内部留下「我们是否重复了奈萨里奥的傲慢」的争论——阿莱克丝塔萨赞成，部分青铜龙保留异议。

**十五、无面者与旧神线索**
佐诺兹与约萨希不是随机小怪升级。苟拉斯之喉与舒玛之喉是旧神触须在龙骨荒野的物理开口；六色软泥的组合逻辑暗示「深渊实验」而非单纯军事。契约者若带走软泥样本，可能引来炼金公会、肯瑞托残部与暮光余党三方追杀。死亡之翼死后，这些裂口不会自动愈合——八阶切入点的「余波」正是处理「主子死了、触须还在蠕动」的政治。诺兹多姆在终局的警告「他起来了」既指死亡之翼二次升起，也可读作旧神嘲笑：你们只烧掉了爪牙。

**十六、天火号与阵营符号**
天火号是联盟炮舰，却搭载部落英雄与龙族——这是大灾变末期罕见的「跨阵营甲板」。天空上尉斯威泽的军纪与部落萨满的仪式在同一条走廊里打架：谁有权下令开火、谁有权靠近萨尔、源质碎片战后归谁，都会在甲板上变成可观察冲突。黑角的战术目标不是「击沉一艘船」那么简单，而是拖住追击窗口，让死亡之翼逃进麦尔斯特罗姆的安全距离。契约者若在甲板战中私吞引擎零件或故意延误修理，等同于放跑世界顶点——乐园任务会以「文明级失败」结算。

**十七、脊背战的身体性**
死亡之翼之脊是艾泽拉斯团本史上最「身体化」的战场之一：你不是站在平台上打血条，而是钉在一条活着的、燃烧的山脉上，脚下是源质，耳边是风，眼前是腐蚀触须从装甲缝里挤出来。装甲板的剥离顺序、畸变融合体的爆炸、腐蚀之血的清理，全部要求团队把「个人英雄」让位给「标记与节奏」。萨尔的第二发不是「大招动画」，而是在你用命撕开的窗口里完成的文明一击。写正文优先调用硫磺味、靴底融化、伞绳勒痕、源质反光刺眼——少写抽象「很强」。

**十八、疯狂阶段与凡人化代价**
疯狂的死亡之翼自麦尔斯特罗姆升起时，身体已经不是完整的龙，而是旧神腐蚀与大地权柄互相撕咬的肉块：触手、利爪、翅膀、Tentacle 平台轮转。守护者必须把剩余权柄全部灌进巨龙之魂，才能完成抹除——这不是「耗蓝」，而是「烧命」。终战后四位守护者化为凡人：他们仍强大，但不再不朽，不再自动锚定世界法则。这一后果比「BOSS 死了」更深远：艾泽拉斯进入无守护者时代，龙族政治真空，后续一切元素与龙岛故事都从这里裂开。契约者若试图「偷一点权柄残响带回乐园」，会同时被青铜龙与大地之环标记。

**十九、与前序五人本的因果锁**
时光之末、永恒之井、暮光之刻三条五人本不是可跳过的过场。永恒之井让玩家目击上古之战中恶魔之魂的原始语境；时光之末展示「若死亡之翼胜利」的荒芜未来；暮光之刻把萨尔与巨龙之魂护送到龙眠门口。巨龙之魂团本的第一秒，默认这些因果已成立。契约者若从低阶切入却声称「我刚从时光之穴出来带着原版魂」，必须支付时间线代价——诺兹多姆的眼线会先于暮光教徒找到你。

**二十、掉落、声望与后勤**
龙魂掉落定义大灾变末的装备上限：源质、暮光织布、泰坦级饰品与武器改写全服职业结构。声望与正义/勇气点是「谁有资格站在脊背上」的阶级门槛。假抗暗影药、过期药水、被掉包的虹膜稳定剂，都是中阶骗局。材料换图纸逻辑在龙魂以「源质碎片换锻造」重演——露富即杀劫在龙眠酒馆一样成立。

**二十一、可观察的日常微观事件**
除大场面外，正文高频：龙眠下层的伤兵配额、暮光假难民渗透、源质黑市秤重、虹膜护送的班次表、天火号甲板上的阵营口角、软泥样本的密封罐泄漏、终战后诗人与密探同时抵达的新闻战。AI 优先这些小事件，而不是一上来麦尔斯特罗姆大战。

**二十二、奈萨里奥的痛与旧神交易**
公开史诗与小说补充：奈萨里奥并非天生嗜杀，大地权柄的重负与旧神低语长期撕扯其神智。他相信旧神能卸下「托举世界」的酷刑，于是把黑龙军团、暮光龙实验与元素领主盟友押上赌桌。写死亡之翼时，暴虐台词下应保留「痛苦驱动」的底色——这不是为他开脱，而是避免 OOC 成无脑火球。契约者若在终局对话窗口试图「劝降」，只会换来熔岩与嘲笑；可交易的是情报（裂口位置、暮光名单），不是他的忠诚。

**二十三、摩诺克到哈格拉的战术语文**
摩诺克考地震与分摊；佐诺兹考虚空弹反弹与站位；约萨希考软泥组合阅读；哈格拉考冰雷相位切换与水晶打断。这四场构成「龙眠语文课」：不会读机制的团队，没有资格听萨尔讲解充能。契约者中阶主舞台正是把这些语文翻译成可执行的口令与标记，而不是个人 DPS 榜。

**二十四、奥卓克希昂的时间表**
奥卓克希昂被设计为「在守护者专心充能时完成猎杀」。因此战场上的真正资源是注意力：清龙群、挡吐息、保萨尔与虹膜通道，必须并行。有人若在此阶段挑起阵营冲突或抢掉落，等于帮死亡之翼完成暮光审判。英雄难度与普通难度的叙事差异可写为「实验体是否释放全部暮光领域」，但不要用「削弱」解释普通模式胜利。


**二十五、终局后的世界状态（可写正文）**
死亡之翼被抹除后，龙骨荒野的炮声停了，但世界没有「变好」成田园。大地裂缝仍在冒烟，暮光教徒改名换姓潜入旅店，源质黑市价格先暴涨后崩盘。守护者凡人化的消息比捷报传得更快：有人庆祝「神也有死期」，有人恐慌「谁还托着大陆」。萨尔离开龙眠后仍要面对元素动荡；联盟与部落很快会把共同末日抛到脑后，重新在卡利姆多与东部王国拔刀。契约者八阶任务的正确气质是「胜利后的清算与缝合」，不是再打一场更大的龙。

**二十六、人物对话气质备忘**
死亡之翼：短句、痛苦、焚尽隐喻。萨尔：沉重、少口号、多代价。阿莱克丝塔萨：生命与牺牲并置。卡雷苟斯：技术词汇与冷静警告。伊瑟拉：梦境意象。诺兹多姆：时间与遗憾。斯威泽：军舰口吻。黑角：嘲讽与斩首逻辑。无面者：非人节奏、重复与饥饿。写对话忌现代网络梗，忌把龙写成轻浮反派。

**二十七、地理感官清单**
龙骨：干冷、龙骨白、焦土缝。神殿：石阶震颤、龙息热浪、旗帜烧边。裂口：湿冷、酸气、触须黏液。永恒之眼：奥术蓝、冰晶反光、虹膜嗡鸣。天火号：机油、硝烟、缆绳勒手。脊背：源质烫、风割脸、血蒸发。麦尔斯特罗姆：海水倒灌声、硫磺雨、光在碎。每场戏至少落一到两个感官钉，避免空镜。

**二十八、与火焰之地/暮光堡垒的对照（勿混档）**
火焰之地终点是元素领主被杀；暮光堡垒终点是古加尔与暮光锤核心被拔；巨龙之魂终点是大地守护者被抹除。三者都是大灾变，但舞台、敌人类型与政治后果不同。写 DS 时不要让拉格纳罗斯突然出现在龙眠，也不要让古加尔在脊背上喊话。跨本互文只允许以「捷报/伤兵/材料短缺」方式出现。

**二十九、团队社会学（龙魂版）**
十人与二十五人编制改变指挥密度：十人更依赖个人机制阅读，二十五人更依赖标记与分组。DKP、ROLL、进度夜与农装夜同样存在。指挥权威在脊背战达到顶点——抗命者会被踢出伞降名单。契约者若以「外来英雄」身份挑战指挥权，必须先证明自己能在约萨希软泥组合里活过三轮。

**三十、卡片锚点：龙眠充能前夜**
最适合生成世界卡的锚点是「虹膜已到手、奥卓克希昂尚未降临」的十二小时窗口：神殿内外同时备战，萨尔在顶层调息，暮光密探最后一次渗透，源质与药水被清点到每一瓶。契约者可在此窗口选择：守虹膜、查内奸、抢黑市、或偷偷测量巨龙之魂的辐射——每条路都通向不同的六至七阶后果。


**三十一、补充名场面因果**
摩诺克倒下后地基裂纹被红龙法师暂时焊死——若焊口被暮光再炸开，尖塔仍可能在奥卓克希昂战中倾斜。佐诺兹的不灭虚空若被错误反弹进友军，会造成「自己人灭团」的政治甩锅。约萨希软泥中的暗影球会沉默治疗，这是教团故意泄露的「克龙眠」配方。哈格拉败亡时虹膜表面会留下暮光蚀痕，卡雷苟斯必须现场净化，否则充能会吸引更多无面者。奥卓克希昂死后的暮光尘会污染龙眠下层水源，需要绿龙梦境净化。脊背战后坠落的源质板若砸中麦尔斯特罗姆航线，会改变渔村命运——八阶可写「坠落物索赔」。

## 阶位切入点

> 阶位↔战力：一阶≈龙眠外围凡人/新兵；二阶≈暮光教徒与山外老兵；三阶≈神殿中层精英；四阶≈摩诺克门槛；五阶≈佐诺兹/约萨希/哈格拉级；六阶≈奥卓克希昂/黑角/脊背；七阶≈疯狂的死亡之翼终战；八阶≈权柄消散后余波与旧神阴影。世界顶点（疯狂的死亡之翼）为七阶可条件性击杀目标；开幕全盛死亡之翼与旧神本体为**超阶巅峰：存在·情报优先/条件性胜利**，低阶一律规避。

**一阶（龙骨荒野外围 · 冻土求生）**

切入身份/时点：契约者以佣兵、难民向导或大地之环跑腿身份出现在龙骨荒野南缘，时点为龙眠围攻已起、巨龙之魂尚未公开充能。

初始事件：一支押运抗暮光药剂的商队在泰坦之路被教徒截下，幸存者塞给你半枚烫手的黑龙鳞片，要你把口信送到龙眠外围营地——你还没进殿，名字却可能已进暮光耳目网。

开场白建议：「风是铁色的，雪里混着硫磺。龙眠尖塔在烟幕后像一截还在燃烧的钉。有人把血糊的布条塞进你手里：上面只有半个雷霆符文和一句兽人脏话——『别让暮光的狗把信烧了。』」

关键NPC立场：**暮光截道头目**（要药要命，可贿可杀）；营地招募官（给最低阶悬赏）；**大地之环**学徒（收元素扰动报告）。

主线钩子/支线：钩子=把鳞片口信送达并换取「外围测温差事」。支线：① 找回被劫的抗暗影药剂半成品；② 护送伤兵离开龙尸坑；③ 辨认鳞片真伪。

危险度/规避：低～中。规避无面者裂口与任何成体暮光龙；见源质反光即撤。

任务方向/奖励：斥候徽记、冻土护符、外围通行证；奖励以物资与人脉为主。

**二阶（神殿外环武装 · 攻城阴影）**

切入身份/时点：契约者以雇佣剑士/盗贼身份潜入龙眠外环通道，时点为摩诺克仍在捶地基。

初始事件：你受托「借阅」一份暮光攻城器部署草图，却在交叉火线里发现第三伙人——有人要同一份图去向部落与联盟双边卖价。

开场白建议：「巷道里全是锤声与龙息余温。草图上的红线指向尖塔根部，两拨刀已经先动手。你还没碰到羊皮纸，就有人低吼：大地在疼——迟到的信差只配喂尸虫。」

关键NPC立场：**龙眠防卫军士官**（死守测绘）；**双边情报贩**（要图议价）；伤兵医官（仇外但可买通补给）。

主线钩子/支线：钩子=三方夺图。支线：① 调开巡逻；② 伪造草图栽赃；③ 卖一份假图给双方。

危险度/规避：中。规避摩诺克本体与无面者成体。

任务方向/奖励：测绘残页、防卫徽章、初级抗暗影药剂配方线索。

**三阶（神殿中层 · 无面者预警）**

切入身份/时点：小队级冒险者，冲击无面者腐化者外围。

初始事件：**泰莉苟萨**的影像刚警告完裂口扩大，神殿戒严；你必须在触须加密前摸到苟拉斯方向的温度记录——德弗莱斯塔兹需要「旧神仍活跃」的证明。

开场白建议：「蓝龙的声音像冰碴刮耳朵。石像的眼睛亮了又灭，像一排嘲笑。地底深处有什么在呼吸：不是龙，是更湿的旧神。有人塞给你一支测温晶簇——碰一下，掌心就起泡。」

关键NPC立场：**德弗莱斯塔兹领主**（要证明）；**无面者腐化者**残部（可绕可打）；中立地精修理商（要高价）。

主线钩子/支线：钩子=取得裂口活跃证明。支线：① 救下被献祭的民兵；② 记录无面者军语换情报费；③ 破坏一场攻城补给。

危险度/规避：中高。规避直接叫阵佐诺兹；勿无编制硬闯裂口。

任务方向/奖励：测温数据、神殿通行残令、精良抗暮光武器。

**四阶（摩诺克 · 推塔之战）**

切入身份/时点：持团队资格的先锋，首次在泰坦之路硬刚摩诺克。

初始事件：开战瞬间，地基震动已让后排掉进裂隙——有人泄露了你开荒的时间表；第一场战斗在「解围爽感」里变成反伏击，摩诺克的践踏尚未结束，已有人被水晶钉死。

开场白建议：「领主只说了三个字：别散开。震动把牙关咬响，脚底的石头像还活着。对面巨像已经举起双拳：死亡之翼说，今天的尖塔要倒——最好带着你们的骨头当砂浆。」

关键NPC立场：**摩诺克**（终局敌本阶）；**德弗莱斯塔兹领主**（指挥接口）；团队同盟领袖（要纪律与站位清单）。

主线钩子/支线：钩子=在反伏击中击破摩诺克并稳住地基。支线：① 查出泄密者；② 夺取第一批源质边角料；③ 建立临时灵魂安放点。

危险度/规避：高。规避双分身英雄机制未准备时硬开；勿单独拆支撑符文。

任务方向/奖励：源质边角、团队定位权、T12 过渡装。

**五阶（无面者与虹膜 · 裂口双王）**

切入身份/时点：成熟团队核心成员，推进佐诺兹、约萨希、哈格拉。

初始事件：约萨希的六色软泥有人故意打错组合，酸与暗影同时融化治疗阵；哈格拉的冰晶同时锁住聚焦虹膜——有人必须在双灭团边缘按下打断与传弹口令。

开场白建议：「虚空弹的轨迹是黑色的句号。软泥的气味是甜腥的，虹膜在冰里像一只未眨的眼。有人喊传弹，有人喊驱散，有人只顾把钴球——不对，那是下一只——你先听见自己的血在耳膜里烧。」

关键NPC立场：**战争大师佐诺兹**；**不眠的约萨希**；**缚风者哈格拉**；**卡雷苟斯**（要虹膜）。

主线钩子/支线：钩子=压制双裂口并夺回聚焦虹膜。支线：① 清缴软泥样本；② 记录哈格拉符文换大地之环声望；③ 阻止有人偷走虹膜仿品。

危险度/规避：极高。软泥错误组合=团灭；未满抗性禁硬吃虚空弹。

任务方向/奖励：T12/T13 向、虹膜护送权、抗暮光装。

**六阶（奥卓克希昂与天火号 · 充能与追击）**

切入身份/时点：已过哈格拉，参与充能与登舰。

初始事件：奥卓克希昂降临前有人试图向暮光「出售」守护者站位；天火号引擎被安放第二组炸药——同时萨尔第一发巨龙之魂打落装甲，死亡之翼已掉头逃向漩涡。

开场白建议：「天幕变成淤青的紫。奥卓克希昂的影子比尖塔还长。舰长吼着升帆，萨尔的双手在神器上烧出裂纹。有人喊：炸药！有人喊：射击窗口！——风把所有口号煮成同一股白汽。」

关键NPC立场：**奥卓克希昂**；**战争大师黑角**；**萨尔**；**天空上尉斯威泽**；**阿莱克丝塔萨**。

主线钩子/支线：钩子=击破奥卓克希昂、守住天火号、击杀黑角。支线：① 拆除第二组炸药；② 护送萨尔稳定输出；③ 截获戈里欧纳龙鳞。

危险度/规避：极高。无编制禁硬吃奥卓克希昂；甲板落水=任务失败。

任务方向/奖励：龙魂武器向、舰载勋章、源质装甲板线索。

**七阶（脊背与疯狂 · 麦尔斯特罗姆）**

切入身份/时点：伞降编制齐全，全团终局。

初始事件：脊背装甲剥至最后一层，腐蚀触须点到你的指挥；死亡之翼坠入漩涡后，诺兹多姆警告「他起来了」——疯狂的死亡之翼自麦尔斯特罗姆升起，触手平台要求你们在守护者倾尽前打进抹除线。

开场白建议：「伞绳割破掌心，源质在脚下像滚烫的甲板。第二发洞穿之后，世界安静了一息。然后漩涡睁开：更大的声音把海按进地里。守护者的光开始碎——有人必须在碎光落尽前按下最后的集火。」

关键NPC立场：**死亡之翼**（脊背/疯狂双形态）；**萨尔**；**诺兹多姆**；**阿莱克丝塔萨**；**卡雷苟斯**；**伊瑟拉**。

主线钩子/支线：钩子=剥脊、洞穿、抹除疯狂的死亡之翼，保护权柄通道不被打断。支线：① 优先击杀愈合触手；② 保护被点名的治疗位；③ 抢在政敌前封存源质主碎片。

危险度/规避：贴近顶点。对疯狂的死亡之翼可条件性胜利（巨龙之魂+守护者倾尽+团队战术）；对开幕全盛仍情报优先。

任务方向/奖励：终局掉落、源质主碎片、抹除死亡之翼的政治声望、守护者凡人化目击证明。

**八阶（权柄消散之后 · 余波与阴影）**

切入身份/时点：已见证抹除的传奇级契约者，处理龙族权力真空与旧神余党。

初始事件：麦尔斯特罗姆安静的第七夜，龙眠传来「谁继承大地」的密会，暮光余党与无面者残部同时要你「解释」裂口是否会再次被拉开；一份写着深岩之洲的热图在萨尔的保险柜里发烫——有人已经在策划「追进元素位面」。

开场白建议：「捷报开始发霉，诗人却把它写成永恒。龙眠尖塔仍在跳——不是龙，是地心的伤在鼓掌。有人把烧焦的羊皮纸拍在你桌上：上面画着一座你从未踏入的要塞，标注只有一个词——未完。」

关键NPC立场：**萨尔**（要元素安抚）；**卡雷苟斯**（要龙族秩序）；暮光余党头目（要复仇）；无面者残部（要裂口）。

主线钩子/支线：钩子=稳定龙眠权力真空，防止裂口再开与源质知识外泄。支线：① 封印残余触须；② 仲裁源质碎片归属；③ 追踪旧神情报锚（只情报，不决战旧神本体）。

危险度/规避：极高～超阶边缘。**旧神本体与开幕全盛死亡之翼残响=超阶巅峰：存在·情报优先/条件性胜利**；宜借势不硬刚。

任务方向/奖励：龙眠停战契约、封存的裂口钥匙、通往深岩/旧神时间线的情报锚（非强制进入）。
