## 剧情

**【作品来源】**
《装甲骑女伊莉丝》（装甲騎女イリス / Armored Warrior Iris / Armored Knight Iris）为 Black Lilith（Lilith Soft）于 2006 年 3 月 24 日发售的成人向 ADV（DLsite RJ022698；包装版约 3 月 31 日），企划监督笹@、剧本そのだまさき、原画 ZOL、装甲骑机械设定やしガニ；主角声优春日アン（伊莉丝）、鸟瀬くるみ（梅莉）。MangaGamer 于 2014 年推出英译下载版。PIXY 于 2007–2009 年推出同名成人 OVA 四卷：vol.01《超空的降下作战》、vol.02《娼馆的女战士》、vol.03《从战士到悦虐娼妇》、vol.04《被虐战士的生还》。原作客选择系统含**肉体改造系**路线；OVA 强化「从战士到被改造服务体」的坠落阶梯。**本档案取「魔铠融合」分支：把改造线推到驾驶员与装甲骑残骸/异物因子强制神经合体的军事工业极端——只写接口、控制权、人格窗口、联邦伦理分裂与战场用途，不写色情细节。**

**【世界定位】**
本世界锚定**人机边界被暴力改写**：伊莉丝·雷贝尔不再只是「被监禁的驾驶员」或「娼馆资产编号」，而是被做成**可遥控的展示兵器 / 魔铠实验体**。冲突从「救不救得出娼馆」升级为「救出来的还是不是伊莉丝」。梅莉·纳セリ的科学能力在此线从「援救工具」变为「逆融合唯一希望」。与「骑女沦陷」（单人商品化）、「妹妹援救」（梅莉夺还）、「双堕深渊」（双人尽没）共享前史，但高潮必须落在**融合度、补丁与控制权**上。

**【世界观 · 力量体系】**
远未来，人类离开地球向星际扩张，与多种族组成松散的**宇宙联邦政府**。联邦对行星自治控制力有限：法律覆盖的都市与反联邦都市可同处一颗行星。联邦保安军 **SSF**（资料中亦写作 Space / Soviet Security Force 等转写）负责治安维持、反奴隶贸易与区域冲突介入。核心兵器是人型机动兵器**装甲骑**（Armored Knight / Armored Warrior，ADT 系列等），由驾驶员神经链接操作；驾驶员在机内可完成据点攻坚与街区级机动打击，**离机后几乎等同受训特种兵，无法与完整装甲骑正面对抗**。

魔铠分支的标尺不是把原作机体夸张成舰队决战武器，而是看它在都市战能做什么：ADT 系装甲骑可突破据点、压制装甲目标并承担突入任务；驾驶员一旦离机，优势会立刻消失。故本档把完整装甲骑与改造完成体放在四阶战场，而把中队包围、黑市技术链和遥控权限当作四阶顶压力。公开设定没有把此作写成行星毁灭战场，契约者不得将其扩写为高阶宇宙战争。

**战力阶梯（破坏力对齐《阶位战力图鉴》，本条目仅开放四阶切入）：**
1. **非武装平民 / 被俘缴械士兵**：徒手与冷兵器级，≈一阶及以下；战场上面对装甲骑无有效反抗。
2. **常规步兵 / 轻型载具 / 百万都市奴隶警备 / 改造工房护卫**：可清据点杂兵，≈二阶～三阶边缘。
3. **SSF 特种装甲骑驾驶员（伊莉丝·雷贝尔、梅莉·纳セリ等）驾驶正役机**：据点攻坚、建筑群级毁伤，典型≈**四阶**。
4. **装甲骑中队协同 / 敌方主力舰炮 / 重型异种战力 / 军阀私兵装甲骑**：街区级毁伤，≈四阶中上～四阶顶。
5. **本世界叙事顶点**（百万军阀核心、魔铠完成体、被完全控制的王牌展示体）：四阶顶～五阶门槛；仍按四阶世界结算。魔铠完成体输出可摸到四阶顶，但伴随人格延迟与遥控权限——其威胁在于**把四阶火力交给敌方指挥链**，而非凭空变成五阶毁城神。

**特殊系统与死亡/俘获/融合规则：**
- **神经链接驾驶**：装甲骑战力绑定驾驶员适配与机体完整度；机体大破后驾驶员进入「离机即降阶」状态。
- **爆弹项圈（恶魔项圈）**：敌后制度核心。逃离或强拆失败即死；在魔铠线上，项圈进一步升级为**神经总线接口**，兼拘束与指令上行。
- **奴隶拍卖与产权晶片**：ミリオン把战俘变成可交易动产；改造工房购入后产权从「服务资产」转为「作战平台资产」。
- **肉体改造订单 → 军事工业订单**：原作「客人订单」改写身体规格；本分支宏观化为作战部订单——身体不再为「服务」改规格，而为「作战平台」改规格。本质仍是「人被规格化」。
- **魔铠融合阶梯**：接口植入 → 残骸嵌合 → 服从固件 → 人格冲突窗口 → 完成体。融合度可量化观测（军语延迟、对「中尉」称呼的反应、射界自主偏移）。
- **逆融合补丁**：以熟悉神经节奏覆盖服从固件的技术手段；可能锁死驾驶天赋、模糊记忆、或写入术者/盟友的反应模式。
- **上行遥控与链路密钥**：真正的 Boss 不是魔铠火力，而是遥控器在谁手里。多方可同时争夺。
- **宣传符号战**：装甲骑女既是精锐也是广告；魔铠被投入战场后，宣传可剪成「联邦英雄已成敌方兵器」。

乐园阶位映射：平民≈一阶；常规军≈二～三阶；装甲骑女主战力≈四阶；世界顶点≈四阶顶（宁低勿高）。超阶巅峰不适用；若遇魔铠/完全支配个体，按四阶 Boss＋精神/系统污染规则处理，条件性胜利。

**【地理 · 舞台】**
- **行星レフト（Left）**：极端女性蔑视风习残留，人身买卖网络盘根错节。伊莉丝与梅莉日常任务是强袭并扑灭奴隶贸易节点。猎奴成功时她们是秩序的锤；猎奴失败时同一套物流链会把她们当货物。
- **联邦辖区都市群**：受联邦法律保护，与レフト上反联邦都市对立。宣传部循环播放装甲骑女的胜利影像；本分支后期还会出现「销毁/回收」争议会议室。
- **区域冲突前线 / ホリック降下作战空域**：联邦对反联邦都市联军的空降与突入战区。对空网、诱饵航迹与兽人清道夫共同把「特种作战」改写成「俘获经济」。
- **敌对势力都市ミリオン（Million）·通称ダークタウン**：黑市、军阀、奴隶拍卖与娼馆经济共生的暗黑都市；伊莉丝被俘后主要囚禁地。权力以竞价、编号与项圈频率运行。
- **SSF 第 101 特殊装甲中队基地**：通称「装甲骑女」部队驻地，女队长编制，最新锐装甲骑列装。食堂空座、机库空位与宣传片同框；本分支还出现「拒绝叫她目标体」的情感分裂。
- **奴隶拍卖会场 / 娼馆拘束设施**：商品化中转；本分支中它们是通往改造工房的前站，而非终局舞台。
- **敌方改造工房深层手术台**：魔铠计划核心。手术灯、融合度仪表、神经索桥接舱并列；记录员、术者与军阀代表分席。
- **神经索桥接舱**：把驾驶员中枢与装甲骑驱动索强制桥接的专用舱；短窗口失控常在此处首次出现。
- **装甲骑残骸库 / 月影拆解线**：ADT-74 月影残骸被当作外骨骼素材拆解的产线；出厂编号比军籍更完整，是本分支视觉符号。
- **边境工业城测试场**：够展示四阶火力、又不够触发联邦战略报复的「城市测试」舞台；宣传片拍摄区与平民疏散区重叠。
- **联邦「销毁/回收」争议会议室**：鹰派与鸽派、伦理官与宣传部对「魔铠伊莉丝」法律定性的场所。
- **タゾック邸等レフト黑产宅邸**：战前猎奴名场面发生地；战后残党可向改造工房供货或竞价。

本分支额外强调：**改造工房深层手术台**、**神经索桥接舱**、**月影残骸外骨骼拆解线**、**联邦销毁/回收争议会议室**、**短窗口失控的城市测试场**、**上行遥控器三备份存放点**。

**【世界剧情线】**

① **惑星レフト：猎奴者的日常（共享前史）**  
人类扩张至星际后，宇宙联邦政府以多种族松散结盟维持秩序。联邦保安军 SSF 第 101 特殊装甲中队因女队长与几乎全女队员、列装最新锐装甲骑，被通称为「装甲骑女」。中尉**伊莉丝·雷ベル**驾驶 **ADT-74 月影（ゲツエイ；英译亦作 Griffin）**，搭档**梅莉·纳セリ**（埃尔芬族）驾驶 **ADT-73 金刚（コンゴウ；英译亦作 Chimera）**。两人在惑星レフト反复强袭人身买卖网络：对她们而言，扑灭奴隶贸易既是任务，也是「对恶人可以冷酷到底」的正义出口与压力阀。

レフト的人身买卖呈分层结构：码头收货人负责「货」的装卸与伪装；武装中介负责护送与灭口；宅邸大物（如**タゾック**）负责订单、洗钱与政治庇护。伊莉丝惯用月影中庭突入与近战超振动刀具完成斩首；梅莉以金刚高空压制与传感器网封锁逃逸。公开资料中出现过对タゾック邸的急袭——梅莉以装甲骑巨铳抵住タゾック头部的名场面，体现装甲骑女「治安维持＋反恐」双重职能。猎奴成功后的拌嘴日常（伊莉丝孩子气护短、梅莉知性吐槽）是后续坠落的情感落差来源。

② **区域战争与ホリック降下作战（共享前史）**  
レフト上反联邦都市结盟，对联邦侧都市发动区域战争。战争经济与黑产共生：俘虏、残骸零件、识别码与宣传素材都可出售。ホリック降下作战中，先行部队暴露，空降队遭对空炮火。伊莉丝为掩护梅莉机，切断自动回避系统改全手动，把自己机体滑入射线上——这是她「孩子气却绝对护短」的驾驶伦理，也是战术上把自身暴露给敌方火力与俘获链的关键失误点。激战中两机先后重创；伊莉丝苏醒时已落入战场鬣狗——**オーク人（兽人）战场清道夫**之手。

オーク清道夫不按意识形态作战：他们认徽章、残骸完整度与「女驾驶员溢价」。俘获是战场经济，不是信念审判。伊莉丝的军籍徽章、驾驶服识别与机体涂装在这一刻从荣誉变成价签。梅莉下落在主线中悬置：失联、被俘或转入外线，成为伊莉丝后续决策的精神锚与情报黑洞。

③ **从猎奴者到商品：押送与拍卖（共享前史，本分支快速过渡）**  
伊莉丝被当作奴隶押往敌对都市**ミリオン（通称ダークタウン）**。她此前一直在猎杀奴隶商人，如今自己成为待售商品——官方剧情明确写出这一讽刺。押送分三段：荒野中转剥徽章与军籍证明；走私货柜把人与货物同级堆叠；入城后安装**爆弹项圈**。军籍被剥即象征性除籍；ADT-74 月影残骸被拖走解析。

公开拍卖中，伊莉丝被当作高价值联邦王牌展示与竞价。展示话术强调她的猎奴战绩以抬价。**博ズク（ボズク）**一类奴隶商人/娼馆利益链人物是关键推手。**本分支的关键分叉发生在此之后**：常规沦陷线会把她编入娼馆劳动编制；魔铠融合线则出现**技术买家 / 改造工房**以更高价或军阀授意截走产权——不是为了「服务资产」，而是因为她与月影的**异常适配数据**被解析为「异常优秀」。

④ **材料发现：异常适配（本分支独有起点）**  
敌方研究所与装甲骑残骸库解析 ADT-74 月影黑匣子时，发现伊莉丝的神经链接适配曲线远超常规驾驶员样本：链路延迟极低、冲突抑制极稳、近战超振动机动中的应激波形可复现。改造工房提出计划：以项圈为接口，把装甲骑神经索与驾驶员中枢强制桥接，并混入异种生物因子提升再生与服从。名称在敌方文件中记为「魔铠」或等价代号——档案采用「魔铠」作为分支通用名。

异常适配使月影残骸从「可卖零件」升级为「外骨骼素材」。拆解线上，肩甲出厂编号比军籍更完整——视觉上完成「人被规格化」的第一刀。军阀要可控兵器；术者要完成数据；博ズク若能分时租赁完成体，则要分成。三方利益在「把伊莉丝做成平台」这一点上短暂合流。

⑤ **手术阶梯：只写阶段功能（本分支主轴）**  
魔铠改造不是单次手术，而是可观测的阶梯：

- **接口植入**：项圈从拘束装置升级为总线。爆弹功能保留，但新增上行指令通道与遥测。伊莉丝仍能说话，但军语应答开始出现半拍延迟——这是融合度仪表上第一个可记录指标。
- **残骸嵌合**：月影装甲片/驱动索被改造成半永久外骨骼层。嵌合部位以肩线、脊索与驱动接口为主；视觉上「驾驶员」与「机体」边界模糊。离机即降阶的旧规则被改写：她本身成为机甲的一部分。
- **服从固件**：战斗指令优先于自我语言。固件不是洗脑口号，而是优先级表——当「开火」与「我是伊莉丝」冲突时，开火先执行。窗口期的自我语言被压缩成短脉冲。
- **人格冲突阶段**：正义感与服从程序抢控制权，出现「短窗口失控」。可观察表现包括：偏转主炮射界只毁空建筑；用军用点名频道喊出只有第 101 知道的暗语求援/求死；对「中尉」称呼产生短暂夺权反应。
- **完成体阶段**：可执行四阶顶输出，但自我名称应答延迟——她要先想「我是不是伊莉丝」。完成体是展示兵器与实战兵器的交界：火力够用，人格是「瑕疵」或「卖点」取决于买家。

每一阶段都有记录员、术者签名与军阀验收节点。契约者应先改记录与链路，再改火力。

⑥ **联邦分裂：销毁派 vs 回收派**  
情报泄露后，联邦内部撕裂。鹰派主张销毁魔铠伊莉丝（视为被污染的战略泄密体与舆论炸弹）；鸽派主张救回研究逆转；伦理官要求先定性「装备 / 伤员 / 资产」再行动；宣传部两头为难——既怕承认「英雄已成敌方兵器」，又怕销毁画面流出更糟。第 101 中队出现分裂：有人仍称她「中尉」，有人已称「目标体」。食堂空座从哀悼变成辩论。

销毁授权与回收授权可能同时下发到不同渠道，造成契约者手中同时握着「补丁」与「销毁令」的经典开局。法律战的关键证据可能是一句仍能挤出的「我是伊莉丝」，以及旧军籍芯片。

⑦ **梅莉的逆融合补丁**  
若梅莉仍自由或能以技术链路介入：她是最可能写出「逆融合补丁」的人——利用埃尔芬科技与对伊莉丝神经习惯的了解。补丁原理是用熟悉的神经节奏覆盖服从固件，而非简单断电。代价可能包括：锁死伊莉丝的驾驶天赋、模糊部分战术记忆、或让她永远无法再链接任何装甲骑。更尖锐的伦理问题是：补丁是否会把梅莉自己的部分反应模式写进伊莉丝，造成另一种「融合」。

救回「人」与救回「王牌」不可兼得。梅莉的弧光在此线从搭档变为「主刀者式拯救者」，并恐惧自己成为新的改造术者。若梅莉已以リルル人格存续于敌后，则补丁作者位可能悬空，契约者必须另找埃尔芬科技圈或工房叛逃学徒。

⑧ **城市测试：被投入攻打联邦**  
魔铠被投入攻打联邦侧边境工业城。选址逻辑冷酷：够展示四阶火力，又不够触发战略报复。宣传片拍摄区与平民疏散区重叠——军阀要的是「联邦英雄轰击联邦都市」的舆论片。

伊莉丝在短窗口失控中可能：
- 偏转主炮射界，只毁空建筑，使宣传片穿帮；
- 用军用点名频道喊出只有第 101 知道的暗语求援/求死；
- 完全执行命令，造成街区级伤亡（四阶顶表现）。

契约者必须在「击毁 / 捕获 / 注入补丁 / 夺遥控」中做战术伦理选择。窗口期误判会导致击杀可救对象；遥控器落入第三方会把问题从「救她」变成「谁配拥有她」。

⑨ **控制权战争：真正的 Boss**  
真正的 Boss 不是魔铠的火力，而是**遥控器在谁手里**。多方同时争夺上行链路：
- **改造术者**：要完成数据，可拿手术日志换补丁，但拒绝永久销毁核心；
- **军阀指挥官**：要可控兵器，不要「有人格的瑕疵品」；
- **联邦鹰派**：要销毁，视任何保留为通敌；
- **联邦鸽派/伦理官**：要回收研究，可能事后把契约者当弃子；
- **梅莉侧/第 101 情感派**：要人，优先人格锚；
- **博ズク**：若魔铠可租可卖，他要分时租赁分成。

遥控器通常有三备份：主控、工房备份、军阀备份。只毁其一等于没毁。契约者稳玩法是经济/情报刺客：伪造验收、换密钥、毁母带、劫手术日志，而不是与完成体硬拼毁城火力。

⑩ **リルル变量**  
若存在リルル（容貌酷似梅莉、性格迥异的少女；官方与 OVA/游戏关键，涉及梅莉创伤防御/记忆封锁），她可能被用作「人格锚」或「伪锚」：让伊莉丝在窗口期把服从对象认成梅莉，从而加速完成体化，或反过来成为逆融合触发器。契约者不可仅凭外貌确认身份。リルル提供的「真/伪人格锚语句」是高风险情报道具。

⑪ **终局分叉**  
- **人胜**：逆融合成功，伊莉丝失去部分驾驶能力但恢复主体；可能退役为教官，月影成忌讳词。  
- **兵器胜**：完成体化不可逆，成为敌方四阶顶展示兵器；联邦宣传崩溃或转入舆论战。  
- **同毁**：链路过载，月影残核与驾驶员意识一同熄灭；鹰派宣称胜利，第 101 内部永久裂痕。  
- **第三方夺权**：契约者或某势力夺得遥控器，问题从「救她」变成「谁配拥有她」；夺遥控者可能成为新军阀。  
- **法律生还**：肉体未归，但军籍芯片与一句「我是伊莉丝」成为法庭关键证据，禁止销毁授权生效，转入长期回收谈判。  
- **幽灵猎杀**：完成体失控后以四阶顶火力猎杀前中尉相关人员，成为「系统污染 Boss」条件性胜利对象。

⑫ **与改造系原作逻辑的对应**  
原作「肉体改造系客人订单」在此线被宏观化为**军事工业订单**：身体不再为「服务」改规格，而为「作战平台」改规格。本质仍是「人被规格化」。客选择系统中的改造后果（驾驶适配下降、识别码失效、无法再合法归队）在魔铠线上被放大为：驾驶适配被强制锁死在敌方接口上，识别码被改写成敌方资产编号，归队资格在法律上变成「装备回收」而非「伤员救治」。

⑬ **与系列其他分支的边界**  
本分支不是单纯监禁，也不是单纯营救，而是**人机伦理战**。与「骑女沦陷」「妹妹援救」「双堕」共享前史（レフト猎奴、ホリック失利、ミリオン拍卖），但高潮必须落在融合度、补丁与控制权上。禁止把四条线写成同一事件复读。魔铠融合的独特性在于：**异常适配把战俘变成平台**、**项圈升级为总线**、**联邦鹰鸽为销毁/回收分裂**、**真正的 Boss 是遥控器**。

关键NPC立场：**伊莉丝·雷贝尔**仍是失联中尉，只能作为追查对象；**梅莉·纳セリ**会追问机体残骸下落；**タゾック**的旧网络想把标签变成价码；**オーク中间人**只认运费；**SSF 登记官**害怕事件曝光。

主线钩子/支线：主线是确认残骸与俘虏是否同一物流线；支线可伪造货单、护送登记副本、追查月影去向。不同于拍卖线，此阶只争“人是否仍在链上”。

危险度/规避：危险度低。规避武装车队、项圈母钥和完整装甲骑；不要把一阶角色写成能正面夺机。世界顶点尚未入场，情报优先。

任务方向/奖励：奖励取向为港区通行、残骸标签、走私路线和 SSF 人情；不得获得月影控制权或改造技术。

主线钩子/支线：主线是夺回或污染产权记录，支线可调换黑匣子、建立假买家身份、把讯号送出城。该阶的胜负是文书与路线，不是强拆拘束装置。

危险度/规避：危险度中。规避项圈测试、公开拍卖和工房深层；完整装甲骑与军阀仍应规避、情报优先。

**三阶（改造工房外围 · 链路证据线）**

开场白建议：「夜班记录每隔十秒打印一行，机器把她写成样本，唯独最后一行仍保留军衔。门禁倒计时跳到零前，你得决定让这行字成为证据，还是成为下一道销毁命令。」

关键NPC立场：**伊莉丝·雷贝尔**的应答是人格仍存的证据；**梅莉·纳セリ**优先取得原始记录；**改造术者**要完成试验；**博ズク**在意资产能否转售；**第101特殊装甲中队队员**可能冒险接应。

危险度/规避：危险度高。规避深层试验舱、军阀护卫和任何完成体正面战；四阶目标一律情报优先、条件性胜利。

任务方向/奖励：奖励为链路日志、工房门禁权限、月影编号对照表与第101中队信任；不得取得完整上行遥控器。

**四阶（魔铠融合 · 人格窗口与控制权线）**

切入身份/时点：契约者以**改造工房叛逃技师**、**联邦黑计划审计官**、**梅莉临时联调员**、或**城市防空链路工程师**身份切入。锚定时点为**伊莉丝处于人格冲突阶段，完成体化倒计时已启动**——融合度刚突破安全阈值。叛逃技师可碰手术日志与工房备份遥控；黑计划审计官可同时收到销毁授权与回收授权；联调员持有梅莉侧补丁半成品；防空工程师掌握城市测试场射界与平民疏散窗。

初始事件：实验舱警报狂响。半融合状态的伊莉丝突然占用军用频道，用第 101 中队才知道的点名方式喊出你的假名（或梅莉给你的临时代号）。三分钟内你必须选择：注入逆融合补丁、拔除核心（高概率致死）、放她出去并承担街区被轰风险、或抢夺上行遥控器把她改挂到你的权限下（伦理灾难）。与此同时，销毁授权与补丁被同时塞进你手中，第三人（军阀或博ズク中介）开始竞价遥控。第一选择会被三方（联邦鹰鸽、工房、梅莉侧）永久记入立场。

关键NPC立场：

- **第 101 队员**：拒绝叫目标体，愿以机体肉身挡射界；或恐惧到请求击落。  
- **军阀指挥官**：要遥控器，不要「有人格的瑕疵品」。  
- **リルル**：提供真/伪人格锚语句；不可仅凭外貌信任。  
- **博ズク**：出价购买「可租用魔铠时段」。  
- **民防官 / 工房叛逃学徒**（场景位）：求疏散窗或用日志换钥匙。

支线③：保护城市测试场平民疏散，同时制造宣传片穿帮。  
支线④：夺取并销毁全部三备份遥控器。  
支线⑤：让军籍芯片与一句「我是伊莉丝」成为法庭上禁止销毁的关键证据。  
支线⑥：阻止博ズク把完成体合同卖给第三方，避免控制权再次外流。

危险度/规避：危险度**高～极高**。四阶顶火力、项圈总线同归于尽、遥控器第三方夺权与补丁写错（永久植物人）并存。必须规避：①窗口期误判导致击杀可救对象；②遥控器落入第三方；③补丁写错造成永久植物人；④只毁一个备份就宣称胜利；⑤暴露乐园能力被格为「新实验体」；⑥与完成体本体硬拼毁城火力。对完成体以瘫痪链路为主；世界顶点级军阀本体**情报优先**，不硬刚。无超阶巅峰；条件性胜利路径含：后门过载、宣传片穿帮导致法律冻结、固件反向炸指挥链。

任务方向/奖励：目标自然落在「保住伊莉丝这个名字的主体性」或「完成最小化伤害的销毁/回收」。奖励取向贴合四阶：逆融合补丁（残份）、接口破解器、月影后门密钥、伦理档案（可换联邦高层人情）、魔铠核心残片（四阶素材）、手术日志（可换工房内应）、伪军籍芯片工艺、城市测试射界图。禁止发放可遥控任意四阶机体的完整上行系统；禁止发放五阶以上毁城兵器。

备选开场（可替换主开场白）：「肩甲上的出厂编号比军籍更完整。她用延迟了半拍的呼吸挤出『中尉』点名，融合度仪表同时跳红。销毁令与补丁在你左右口袋里互相发烫——而第三只手，已经伸向遥控器。」
