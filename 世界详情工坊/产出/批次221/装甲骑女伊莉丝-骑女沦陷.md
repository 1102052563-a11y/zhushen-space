# 装甲骑女伊莉丝-骑女沦陷
<!--meta lib=主库 tiers=四-->

## 剧情

**【作品来源】**
《装甲骑女伊莉丝》（装甲騎女イリス / Armored Knight Iris）为 Black Lilith 于 2006 年 3 月 24 日发售的成人向 ADV（DLsite RJ022698），企划监督笹@、剧本そのだまさき、原画 ZOL、装甲骑机械设定やしガニ，声优春日アン（伊莉丝）、鸟瀬くるみ（梅莉）。类型标签为「娼馆监禁 ADV」。PIXY 于 2007–2009 年推出同名 OVA 四卷：vol.01《超空的降下作战》、vol.02《娼馆的女战士》、vol.03《从战士到悦虐娼妇》、vol.04《被虐战士的生还》，角色原案 ZOL。系列属「美少女装甲骑＋敌后沦陷」军事科幻。**本档案只写组织、战争、俘获、拍卖与监禁制度下的军人尊严崩解节点，不写色情细节。**

**【世界定位】**
本世界锚定**伊莉丝·雷贝尔单人主沦陷线**：惑星レフト反奴隶贸易日常→ホリック降下作战失利→兽人战场清道夫俘获→ミリオン（ダークタウン）拍卖→娼馆监禁制度下的意志磨损。重点是「从猎奴者变成商品」的制度反讽，以及 SSF 宣传符号与真实战俘身份的撕裂。

**【世界观 · 力量体系】**
远未来，人类向星际扩张。多种族组成松散的**宇宙联邦政府**。联邦保安军 **SSF** 负责治安维持、反奴隶贸易与区域冲突介入。核心兵器是人型机动兵器**装甲骑**（ADT 系列等），由驾驶员神经链接操作。

**战力阶梯（破坏力对齐《阶位战力图鉴》，本条目仅开放四阶切入）：**
1. **非武装平民 / 被俘缴械士兵**：徒手与冷兵器级，≈一阶及以下；战场上面对装甲骑无有效反抗。
2. **常规步兵 / 轻型载具 / 百万都市奴隶警备**：可清据点杂兵，≈二阶～三阶边缘。
3. **SSF 特种装甲骑驾驶员（伊莉丝·雷贝尔、梅莉·纳塞利等）驾驶正役机**：机动摧毁装甲目标、突入都市与轨道设施、突破音障级机动，掀平建筑群与据点攻坚，典型≈**四阶**。
4. **装甲骑中队协同 / 敌方主力舰炮 / 重型异种战力 / 军阀私兵装甲骑**：街区级毁伤，≈四阶中上～四阶顶。
5. **本世界叙事顶点**（百万军阀核心、改造兵器、魔铠完成体、被完全控制的王牌展示体）：仍落在四阶顶～五阶门槛之间；**本条目按四阶世界处理**，禁止以六阶以上外挂入场秒杀。

死亡与俘获规则：驾驶员离机后几乎等同常人；爆弹项圈把「突围」改写成「自杀」；改造手术与神经接口可永久改写战斗身份。  
乐园阶位映射：平民≈一阶；常规军≈二～三阶；装甲骑女主战力≈四阶；世界顶点≈四阶顶（宁低勿高）。超阶巅峰不适用；若遇魔铠/完全支配个体，按四阶 Boss＋精神/系统污染规则处理，条件性胜利。

**【地理 · 舞台】**
- **行星レフト（Left）**：极端女性蔑视风习残留，人身买卖网络盘根错节；伊莉丝与梅莉日常任务是强袭并扑灭奴隶贸易节点。
- **联邦辖区都市群**：受联邦法律保护，与レフト上反联邦都市对立。
- **区域冲突前线 / ホリック降下作战空域**：联邦对反联邦都市联军的空降与突入战区；兽人（オーク）战场清道夫出没。
- **敌对势力都市ミリオン（Million）·通称ダークタウン**：黑市、军阀、奴隶拍卖与娼馆经济共生的暗黑都市；伊莉丝被俘后主要囚禁地。
- **SSF 第101特殊装甲中队基地**：通称「装甲骑女」部队驻地，女队长编制，最新锐装甲骑列装。
- **奴隶拍卖会场 / 娼馆拘束设施 / 改造手术室**：俘虏商品化与意志磨损的中转与终点。
- **敌方研究所与装甲骑残骸库**：解析 ADT 黑匣子、项圈频率、魔铠接口的技术节点。
本分支额外强调：**拍卖会场的权力仪式**、**娼馆产权编号**、**联邦宣传部仍在播放胜利影像**的时间差。

**【世界剧情线】**
① **惑星レフト：猎奴者的日常**  
人类扩张至星际后，宇宙联邦政府以多种族松散结盟维持秩序。联邦保安军 SSF 第 101 特殊装甲中队因女队长与几乎全女队员、列装最新锐装甲骑，被通称为「装甲骑女」。中尉**伊莉丝·雷贝尔**驾驶 **ADT-74 月影（ゲツエイ）**，搭档**梅莉·纳塞利**（埃尔芬族）驾驶 **ADT-73 金刚（コンゴウ）**。两人在惑星レフト反复强袭人身买卖网络：对她们而言，扑灭奴隶贸易既是任务，也是「对恶人可以冷酷到底」的正义出口与压力阀。公开资料中出现过对大人物**タゾック**邸的急袭——梅莉以装甲骑巨铳抵住タゾック头部的名场面，体现装甲骑女「治安维持＋反恐」双重职能。

② **ホリック降下作战：秩序崩溃**  
レフト上反联邦都市结盟，对联邦侧都市发动区域战争。装甲骑女被派往前线。ホリック降下作战中，先行部队暴露，空降队遭对空炮火。伊莉丝为掩护梅莉机，切断自动回避系统改全手动，把自己机体滑入射线上——这是她「孩子气却绝对护短」的驾驶伦理。激战中两机先后重创；伊莉丝苏醒时已落入战场鬣狗——**オーク人（兽人）战场清道夫**之手。

③ **从猎奴者到商品：结构性反讽**  
伊莉丝被当作奴隶押往敌对都市**ミリオン（通称ダークタウン）**。她此前一直在猎杀奴隶商人，如今自己成为待售商品——官方剧情明确写出这一讽刺。军籍徽章被剥，ADT-74 月影残骸或被拖走解析。此节点完成「骑女」从主体战斗者到「战俘资产」的身份降格。

④ **恶魔项圈：突围选项被规则删除**  
押送途中或入城后，伊莉丝被安装**爆弹项圈（恶魔项圈）**：逃离即死。制度设计把特种兵最擅长的「硬突围」改写成自杀，从而把后续冲突从武力问题改写成**制度与情报问题**。任何营救方若不知项圈频率，强突等于杀死人质。

⑤ **拍卖会：商品化仪式（OVA vol.01 节点）**  
公开拍卖中，伊莉丝被当作高价值联邦王牌展示与竞价。档案只记权力羞辱与情报风险：机体数据、中队战术、联邦识别码、宣传形象的可利用度。**博兹克（ボズク）**一类奴隶商人/娼馆利益链人物是关键推手；娼馆主竞得「所有权」后，将其编入娼馆劳动编制。军人自尊与商品编号并置，是本分支核心残酷点。

⑥ **娼馆监禁下的意志磨损（主线中段，只写制度）**  
原作以「客选择系统」推进状态：大类分为**人类系 / 非人类系 / 肉体改造系**，选择影响结局。**本档案禁止展开性描写**，只保留可玩戏剧节点：  
- 是否向联邦发出可定位的微弱信标；  
- 是否用中队战术情报换取项圈暂时解锁；  
- 是否接受「伪自愿」表演以换取梅莉生死消息；  
- 是否组织其他战俘暴动；  
- 是否拒绝改造手术而触发更严酷的拘束升级。  
肉体改造在原作中是「按客人订单改身体」的制度环节，档案中只记为**身份与战斗能力的永久改写风险**（驾驶适配下降、识别码失效、无法再合法归队）。

⑦ **SSF 侧时间线：宣传与营救的裂缝**  
联邦宣传仍循环播放装甲骑女的胜利影像；第 101 中队内部就是否投入营救分裂——营救可能中埋伏，放弃则等于承认王牌可被商品化。梅莉若存活于外线，将成为私人违令营救的火种（详见「妹妹援救」条目，本分支仅作外线气压）。

⑧ **深度节点：友军羞辱与伪替身（人类系压迫）**  
公开资料与 OVA 指出，部分压迫来自**敌方收编或敌视装甲骑女的人类士兵**，以「假伊莉丝」叙事进行人格羞辱。档案只记：伊莉丝必须在「承认自己是伊莉丝」与「以替身身份求生」之间做身份博弈；任何承认都可能被录成敌方宣传素材。

⑨ **非人类系与改造系的制度功能**  
异种种族与改造订单在原作中用于「磨碎骄傲」。档案只写功能：非人类系制造不可用联邦军规谈判的沟通障碍；改造系把身体变成可交易规格。两者共同把「特种中尉」降格为「可配置资产」。

⑩ **沦陷完成与可选终局**  
- **深度沦陷终局**：军人自我叙事崩溃，成为敌方展示「联邦英雄已堕」的活广告，中队士气受创，战争转入舆论战。  
- **残留火种终局**：身体与编制被控，但她暗记拍卖网络与军阀后勤名单，等待梅莉或契约者触发。  
- **暴动失败终局**：暴动被镇压，项圈升级，营救窗口关闭。  
- **OVA vol.04 向「生还」岔路**：若外线装甲骑女部队飞来且内部出现协助者（如容貌酷似梅莉的**リルル**相关线），可转入营救——但本分支默认玩家切入时仍处于「单人沉没」主轴。  
契约者应把**制度**当 Boss，而非单一反派。

⑪ **与系列其他分支的边界**  
本分支强调伊莉丝**单人**沉没；梅莉主视角夺还见「妹妹援救」；双人尽没见「双堕深渊」；驾驶员与机体/异物强制合体见「魔铠融合」。禁止把四条线写成同一事件复读。

**【主要人物】**
- **伊莉丝·雷贝尔（イリス・レベル）**（主角 / SSF 中尉，地球人类）｜性格：明朗快活、对后辈温和，略孩子气；对犯罪者冷酷；护短到会切断安全系统挡枪｜装备·能力：王牌机 **ADT-74 月影**；格斗与驾驶顶尖；离机后战力骤降｜人物弧光：猎奴英雄→战场被俘→拍卖商品→娼馆编制下的意志战｜立场关系：梅莉的搭档与精神锚；第 101 中队符号；敌方高价值战利品。
- **梅莉·纳塞利（メイリィ・ナセリ）**（搭档 / SSF 中尉，埃尔芬族）｜性格：外表幼态，人类年龄换算超百岁；知性高、科学力强、种族骄傲易惹麻烦｜装备·能力：**ADT-73 金刚**；传感器与战术辅助｜人物弧光：本分支多为外线失联/潜在营救者｜立场：伊莉丝最重要之人。
- **博兹克（ボズク）**（奴隶商人 / 娼馆利益链关键人物，OVA 具名）｜性格：贪婪冷酷，把人当库存｜能力：俘获押运、拍卖渠道、娼馆管理｜立场：把伊莉丝商品化的核心推手。
- **タゾック（Tazock）**（人身买卖网络大物，官方剧情出现）｜性格：奴隶帝国式老板｜能力：レフト侧黑产网络节点｜立场：战前被装甲骑女强袭的对象，象征「猎奴者与猎物」翻转前的旧秩序。
- **娼馆主（ミリオン侧经营者，公开资料多以职位称）**｜性格：资产思维｜立场：拍卖竞得伊莉丝后以项圈与编制控制其行动半径。
- **オーク人战场清道夫头目（公开资料以种族/职能称，个体名不详）**｜性格：趁乱掳人｜立场：ホリック混战中的「战场鬣狗」，完成初次俘获。
- **第 101 特殊装甲中队队长（公开页未稳定给出汉化全名，标不详）**｜性格：严厉护短｜立场：营救批准权与政治责任承担者。
- **联邦宣传官 / 情报官（职衔）**｜性格：要「英雄不死」的叙事｜立场：可能拖延承认被俘以保士气。
- **ターフル（Tāfuru，公开资料中与「仇视/迷恋伊莉丝形象的人类士兵」相关压迫节点）**｜性格：把英雄形象扭曲为占有欲｜立场：人类系压迫的具名压力源之一（档案只写羞辱结构，不写过程）。
- **リルル（リルル）**（容貌酷似梅莉、性格迥异的少女，官方与 OVA 出现）｜性格：与梅莉相反的顺从/沉溺姿态｜立场：在伊莉丝眼中是「假梅莉」式冲击源，可成为营救线谜题或心理崩溃加速器。
- **敌方军阀指挥官 / 改造术者（分支 Boss 位）**｜性格：视装甲骑女为政治羞辱工具与实验体｜能力：项圈、改造手术、装甲骑残骸解析｜立场：与联邦开战的利益结合体。
- **被俘联邦士兵群**｜性格：崩溃/硬抗/叛变光谱｜立场：伊莉丝是否仍能组织「军人自尊」的镜子。

**【势力图谱】**
- **宇宙联邦政府 / 联邦保安军 SSF**：多种族秩序维护者；装甲骑女是精锐也是宣传符号。
- **第 101 特殊装甲中队**：特种突入与反奴隶贸易；伊莉丝与梅莉为双王牌。
- **惑星レフト反联邦都市联军**：区域战争发动方；战争经济与黑产共生。
- **ミリオン（ダークタウン）黑市共同体**：拍卖、娼馆、军阀私兵一体化。
- **奴隶商人网络（博兹克、タゾック等）**：连接战场俘获与都市产权。
- **埃尔芬族势力**：高知性科技种族；影响梅莉能否获得族援（本分支弱化）。

**【隐藏剧情 · 伏笔】**
- 联邦是否故意迟缓营救以维持「英雄不死」宣传，是可挖黑幕。
- 月影黑匣子若流入黑市，敌方装甲骑技术将跃进一代。
- 项圈频率与联邦识别码同源的可能性指向内部叛卖。
- リルル与梅莉的关系（替身、克隆、改造体或独立人物）是营救线关键谜题；本分支只抛出「容貌相同、人格相反」的冲击。
- 伊莉丝对犯罪者的冷酷记忆，是她最难被完全改写的内核。

**【叙事基调 · 雷区】**

忌：任何色情描写、器官细节、性暴力过程；用「商品化 / 意志战 / 接口 / 编制」等制度语言替代。
