# 装甲骑女伊莉丝-骑女沦陷

## 剧情

**【作品来源】**
《装甲骑女伊莉丝》（装甲騎女イリス / Armored Knight Iris）为Lilith（Black Lilith）于2006年3月24日发售的成人向ADV（DLsite RJ022698），企划监督笹@、剧本そのだまさき、原画ZOL，声优春日アン（伊莉丝）、鸟瀬くるみ（梅莉）等。PIXY于2007–2009年推出同名OVA四卷：《vol.01 超空的降下作战》《vol.02 娼馆的女战士》《vol.03 从战士到悦虐娼妇》《vol.04 被虐战士的生还》，监督高箸漏助等，角色原案ZOL。系列属「美少女装甲骑 + 敌后沦陷」军事科幻；**本档案只写组织、战争、沦陷主线节点与营救/堕落分支，不写色情细节。**

**【世界定位】**
本世界取**伊莉丝单人主沦陷线**：战场被俘→拍卖→娼馆监禁制度下的军人尊严崩解过程。重点在组织性压迫、项圈与拍卖制度、SSF营救迟滞，而非感官描写。

**【世界观 · 力量体系】**
远未来，人类扩张至星际，多种族组成松散的**宇宙联邦政府**。武力核心之一是人型机动兵器**装甲骑（ADT系列等）**，由联邦保安军SSF编制运用。

**战力阶梯（破坏力对齐乐园图鉴，本世界条目仅开放四阶切入）：**
1. **非武装平民 / 被俘缴械士兵**：凡俗一阶以下，战场上无反抗能力。
2. **常规步兵 / 轻型载具 / 奴隶警备**：二~三阶边缘，可清剿据点杂兵。
3. **SSF特种装甲骑驾驶员（伊莉丝、梅莉等）驾驶正役机**：机动摧毁装甲目标、突入都市与轨道设施，典型表现≈**四阶**（掀平建筑群、据点攻坚、突破音障级机动力）。
4. **装甲骑中队协同 / 敌方主力舰炮与重型异种战力**：四阶中上~四阶顶；可造成城市街区级毁伤。
5. **本世界叙事顶点（敌方军阀核心、改造兵器、魔铠完成体等）**：仍落在四阶顶到五阶门槛之间，**本条目按四阶世界处理**，契约者不可按六阶以上战力入场。

乐园阶位映射：平民≈一阶；常规军≈二~三阶；装甲骑女主战力≈四阶；世界顶点≈四阶顶（宁低勿高）。超阶巅峰不适用；若遇「魔铠/完全支配」个体，按四阶Boss+精神/系统污染规则处理，条件性胜利。

**【地理 · 舞台】**
- **联邦领土 / SSF第101特殊装甲中队基地**：通称「装甲骑女」部队，队长与队员几乎全为女性，装备最新型装甲骑，负责高危特种作战。
- **敌对势力都市「百万（Million）」**：黑市、军阀、奴隶拍卖与娼馆经济共生的暗黑都市，伊莉丝被俘后主要囚禁地。
- **轨道降落战区 / 超空战场**：OVA vol.01开场空降与混战区，兽人（オーク）等战场清道夫出没。
- **奴隶拍卖会场与拘束设施**：俘虏被商品化的中转节点（只写制度与权力结构）。
- **敌方研究所 / 改造工房**：肉体改造、项圈爆弹、装甲骑残骸解析与「魔铠」实验相关。
- **联邦后方指挥链**：营救窗口受政治与情报延迟制约，是「妹妹援救」「双堕」等分支的时间压力来源。

**【世界剧情线】**
① **超空降下与中队传说**  
联邦与敌对军阀冲突白热化。SSF第101特殊装甲中队以全女队员与最新型装甲骑著称，被称作「装甲骑女」。中尉伊莉丝·雷贝尔驾驶ADT-74月影，搭档梅莉·纳塞利驾驶ADT-73金刚，屡次完成高难度突入，成为联邦宣传中的英雄符号。

② **混战俘获（沦陷触发点）**  
一次轨道/超空降下作战中，战场秩序崩溃。伊莉丝与中队失散，遭趁乱捞人的兽人佣兵与奴隶商人捕获。ADT-74月影重创或被夺，军籍徽章被剥。此节点是「骑女」从主体战斗者变为「战俘资产」的结构性转折。

③ **押送百万都市与恶魔项圈**  
伊莉丝被押往敌对都市百万。敌方以**爆弹项圈**限制行动：逃离即死。制度设计把「勇敢突围」改写成「自杀」，从而瓦解特种兵最习惯的解题方式。

④ **拍卖会：商品化仪式**  
在公开拍卖中，伊莉丝被当作高价值联邦王牌展示与竞价（档案只记权力羞辱与情报泄露风险：机体数据、中队战术、联邦识别码）。娼馆主竞得「所有权」，将其编入娼馆劳动编制。军人自尊与商品编号并置，是本分支核心残酷点。

⑤ **娼馆监禁下的意志磨损（主线中段）**  
在客选择式压迫结构中（原作以多类型「客人」推进状态），伊莉丝经历从硬抗、谈判、逃跑失败到认同资源被抽空的过程。**本档案禁止展开性描写**，只保留可玩的戏剧节点：  
- 是否向联邦发出可定位的微弱信标；  
- 是否出卖中队战术换取项圈暂时解锁；  
- 是否接受「伪自愿」表演以换取见梅莉遗物/消息的机会；  
- 是否组织其他战俘暴动。

⑥ **SSF侧时间线**  
联邦宣传仍在播放装甲骑女的胜利影像；第101中队内部就是否投入营救产生分裂——营救可能中埋伏，放弃则等于承认王牌可被商品化。梅莉若存活于外线，将以私人行动突破指挥链。

⑦ **沦陷完成节点与可选终局**  
- **深度沦陷终局**：伊莉丝的军人自我叙事崩溃，成为敌方展示「联邦英雄已堕」的活广告，中队士气受创，战争进入更肮脏的舆论战。  
- **残留火种终局**：身体与编制被控，但她藏起一份完整的拍卖网络与军阀后勤名单，等待任何契约者/梅莉触发。  
- **暴动失败终局**：暴动被镇压，项圈升级，营救窗口关闭。  
契约者切入时应把「制度」当Boss，而非单一反派。

⑧ **与系列其他分支关系**  
本分支强调**伊莉丝单人沉没**；梅莉主视角见「妹妹援救」；双人尽没见「双堕深渊」；驾驶员与机体/异物强制合体见「魔铠融合」。

**【主要人物】**
- **伊莉丝·雷贝尔（イリス・レベル）**（主角/SSF中尉）｜性格：明朗快活、孩子气、正义感强，对罪犯冷酷；被俘后尊严与军人身份是剧情核心张力｜装备·能力：最新型装甲骑**ADT-74 月影（ゲツエイ）**王牌驾驶员；格斗与驾驶顶尖｜人物弧光：联邦英雄→战场被俘→在爆弹项圈与拍卖制度下被迫放弃军人身分认同，完成「骑女沦陷」主线节点｜立场关系：梅莉的搭档与精神支柱；SSF第101中队核心；敌方视作高价值战利品与象征性征服对象。
- **梅莉·纳塞利（メイリィ・ナセリ）**（搭档/SSF中尉，埃尔芬族）｜性格：外表幼态实则人类年龄换算超百岁；知性高、科学力强、种族骄傲易惹麻烦｜装备·能力：新锐装甲骑**ADT-73 金刚（コンゴウ）**；技术解析与战术辅助｜人物弧光：王牌搭档→前线失联，成为营救或后续作品的火种（本分支中多为外线）｜立场关系：伊莉丝最重要之人；埃尔芬族立场常与人类联邦摩擦。
- **第101特殊装甲中队队长（具名依资料作「队长」编制，公开页未稳定汉化全名者标不详）**｜性格：严厉、护短｜立场：中队母舰/基地指挥，营救行动批准者。
- **博兹克（ボズク）**（OVA关键敌方角色）｜性格：奴隶商人/敌方利益链中人，贪婪冷酷｜能力：俘获、押运、拍卖渠道｜立场：把伊莉丝商品化的关键推手之一。
- **娼馆主（百万都市侧经营者，公开资料以职位称）**｜性格：把俘虏当资产｜立场：购入伊莉丝后以爆弹项圈等手段控制行动范围。
- **兽人佣兵/战场清道夫群体**｜性格：趁乱掳人｜立场：超空混战中的「战场鬣狗」。
- **敌方军阀指挥官 / 改造术者（分支Boss）**｜性格：视装甲骑女为实验体与政治羞辱工具｜能力：项圈、改造手术、魔铠融合实验｜立场：与联邦开战的意识形态与利益结合体。
- **联邦情报官 / 营救队队长（不详真名则保持职衔）**｜性格：务实｜立场：权衡是否牺牲一人保全战线。
- **埃尔芬族联络人**｜性格：傲慢或理性派不一｜立场：梅莉族群是否介入营救的变量。
- **被俘联邦士兵群**｜性格：崩溃/硬抗/叛变光谱｜立场：伊莉丝在敌后是否仍能组织「军人自尊」的镜子。
- **ADT-74月影 / ADT-73金刚（机体拟人化叙事时作装备角色）**｜能力：四阶主战装甲骑｜下落：战场残骸、被夺、或魔铠化素材，随分支变化。

**【势力图谱】**
- **宇宙联邦政府 / 联邦保安军SSF**：维持多种族秩序；第101特殊装甲中队为精锐宣传符号，「装甲骑女」偶像化与真实伤亡形成反差。
- **第101特殊装甲中队**：女队长编制，特种突入；伊莉丝与梅莉为双王牌。
- **敌对军阀 + 百万都市黑市共同体**：战争经济依赖奴隶、情报与联邦战俘羞辱。
- **奴隶商人网络（含博兹克一类）**：连接战场俘获与都市拍卖。
- **埃尔芬族势力**：高知性科技种族，对人类态度复杂，影响梅莉是否获得族援。
- **改造工房/魔铠教团式实验派**（分支强化）：企图把驾驶员与装甲骑、异种因子强制融合。

**【贵重物品】**

- **爆弹项圈（恶魔项圈）**：逃离则引爆的拘束装置，定义敌后行动边界。
- **装甲骑女识别徽章 / SSF军籍证明**：身份象征，被剥夺即象征性「除籍」。
- **拍卖芯片与奴隶契约晶片**：百万都市产权规则的物化。
- **改造手术套件 / 魔铠核心**：分支关键，能把驾驶员与外骨骼/异物永久嵌合。
- **中队紧急信标**：营救线的时间轴道具，能否发出决定联邦是否定位。

**【隐藏剧情 · 伏笔】**

- 伊莉丝孩子气外表下的冷酷执法记忆，是她最难被完全改写的内核。

## 阶位切入点

> 阶位↔原作：四阶≈装甲骑王牌驾驶员与主战机体交战级。世界顶点为军阀核心与完全控制下的「展示性战力」，按四阶顶处理。低配切入禁止开五阶以上外挂。

**四阶（骑女沦陷·项圈都市线）**
切入身份/时点：契约者以SSF临时特遣、黑市掮客或被击落的中立佣兵身份切入，锚定伊莉丝已被押入百万都市、拍卖前后。
初始事件：你在拍卖会外围接到两份互相矛盾的委托——联邦要你销毁伊莉丝以免战术泄露，娼馆中介要你抬价保住「活体王牌」；同时项圈信号显示她还在试图拆弹。
开场白建议：「百万都市的穹顶广告循环播放联邦英雄的笑脸，而实物正在拍卖台上被报价。你的通讯器里同时跳进两条指令：毁灭她，或买下她。倒计时与项圈红灯一起闪。」
关键NPC立场：**伊莉丝·雷贝尔**——仍以军人逻辑谈判，对「买下」本身感到羞辱但可能利用；**博兹克**——只在乎利润与是否再供货；**娼馆主**——把你当新客户或竞品；**梅莉·纳塞利**（若外线连入）——要求你优先救人而非任务；**第101中队指挥链**——要成果不要丑闻。
主线钩子/支线：拍卖竞价/劫标、项圈破解、黑匣子夺还；支线＝策反战俘、伪造军籍、向联邦假报阵亡以换营救窗口。
危险度/规避：高——四阶机体残骸火力与军阀安保；规避暴露真实乐园能力导致被格为「新实验体」。
任务方向/奖励：决定伊莉丝沦陷深度或撕开营救口；奖励取向＝装甲骑部件、项圈解码数据、黑市路线图、SSF人情或敌方权限。
