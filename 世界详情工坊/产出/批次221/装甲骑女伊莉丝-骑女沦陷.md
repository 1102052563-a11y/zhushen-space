# 装甲骑女伊莉丝-骑女沦陷

## 剧情

**【作品来源】**
《装甲骑女伊莉丝》（装甲騎女イリス / Soukou Kijo Iris / Armored Warrior Iris / Armored Knight Iris）为 Black Lilith（Lilith Soft 旗下品牌）于 2006 年 3 月 24 日发售下载版、3 月 31 日发售包装版的成人向 ADV（DLsite 等渠道常见编号 RJ022698），类型常被标为「娼馆监禁 ADV / 女机师沦陷」。企划·监督笹@；剧本原案·剧本そのだまさき等；原画 ZOL；装甲骑机械设定·图形 やしガニ / チームやしガニ；演出 EDEN·栗栖。主角声优：春日アン（伊莉丝）、鸟瀬くるみ（梅莉）。MangaGamer 于 2014 年推出英译下载版（Armored Warrior Iris）。PIXY / Anime Antenna Iinkai 于 2007–2009 年推出同名成人 OVA 四卷：vol.01《超空の降下作戦》、vol.02《娼館の女戦士》、vol.03《戦士から悦虐娼婦へ》、vol.04《被虐戦士の生還》。系列核心命题是「美少女装甲骑驾驶员在敌后被俘、商品化、制度性磨损」；**本档案只写组织、战争、俘获、拍卖与监禁制度下的军人尊严崩解节点，不写色情细节。**

**【世界定位】**
本世界锚定**伊莉丝·雷贝尔单人主沦陷线**：惑星レフト反奴隶贸易日常 → ホリック降下作战失利 → 兽人战场清道夫俘获 → ミリオン（ダークタウン）拍卖 → 娼馆监禁制度下的意志磨损。重点是「从猎奴者变成商品」的制度反讽，以及 SSF 宣传符号与真实战俘身份的撕裂。梅莉·纳塞利在本分支多作外线失联/潜在营救气压，不抢主视角（梅莉夺还见「妹妹援救」；双人尽没见「双堕深渊」；机体强制融合见「魔铠融合」）。

**【世界观 · 力量体系】**
远未来，人类离开地球向星际扩张，与多种族组成松散的**宇宙联邦政府**（Space Federation）。联邦对行星自治的控制力有限：法律覆盖的都市与反联邦都市可同处一颗行星，边境治安靠军警与特种部队维持。联邦保安军 **SSF**（资料中亦写作 Space / Soviet Security Force 等转写，以「联邦保安军」统一指称）负责治安维持、反奴隶贸易与区域冲突介入。核心兵器是人型机动兵器**装甲骑**（Armored Knight / Armored Warrior，ADT 系列等），由驾驶员神经链接操作；驾驶员在机内可完成据点攻坚与街区级机动打击，**离机后几乎等同受训特种兵，无法与完整装甲骑正面对抗**。

装甲骑战斗的实际表现决定乐园映射：正役特种机可机动摧毁装甲目标、突入都市与轨道设施、突破音障级机动并掀平建筑群——对应图鉴四阶「摧毁据点 / 掀平建筑群」。中队协同、敌方主力舰炮、重型异种战力与军阀私兵装甲骑可把毁伤抬到四阶中上～四阶顶。本世界**没有**稳定的行星爆破级战力作为日常棋子；叙事顶点（ミリオン军阀核心、改造兵器、被完全控制的王牌展示体）仍落在四阶顶～五阶门槛之间。**本条目按四阶世界处理**，禁止以六阶以上外挂入场秒杀。

**战力阶梯（破坏力对齐《阶位战力图鉴》，本条目仅开放四阶切入）：**
1. **非武装平民 / 被俘缴械士兵**：徒手与冷兵器级，≈一阶及以下；战场上面对装甲骑无有效反抗。
2. **常规步兵 / 轻型载具 / 百万都市奴隶警备**：可清据点杂兵，≈二阶～三阶边缘。
3. **SSF 特种装甲骑驾驶员（伊莉丝·雷贝尔、梅莉·纳塞利等）驾驶正役机**：据点攻坚、建筑群级毁伤，典型≈**四阶**。
4. **装甲骑中队协同 / 敌方主力舰炮 / 重型异种战力 / 军阀私兵装甲骑**：街区级毁伤，≈四阶中上～四阶顶。
5. **本世界叙事顶点**（百万军阀核心、改造兵器、完全控制下的王牌展示体）：四阶顶～五阶门槛；仍按四阶世界结算。

**特殊系统与死亡/俘获规则：**
- **神经链接驾驶**：装甲骑战力绑定驾驶员适配与机体完整度；机体大破后驾驶员可能被弹射或拖出，进入「离机即降阶」状态。
- **爆弹项圈（恶魔项圈）**：敌后制度核心。逃离或强拆失败即死，把特种兵最擅长的「硬突围」改写成自杀；营救方若不知频率，强突等于杀死人质。
- **奴隶拍卖与产权晶片**：ミリオン把战俘变成可交易动产；编号、腕带、契约晶片构成法律拟制上的「所有权」。
- **肉体改造订单**：原作以「客人订单」改写身体规格。档案只记制度功能——驾驶适配下降、识别码失效、无法再合法归队——不写过程。
- **宣传符号战**：装甲骑女既是精锐也是广告；被俘本身可被剪成「联邦英雄已堕」的舆论武器。
- **双重人格/记忆封锁（梅莉—リルル线）**：原作中梅莉以「リルル」人格作为创伤防御机制出现；本分支只作身份冲击源与营救谜题，不把双堕写进主轴。
- **客户选择状态机（游戏机制）**：人类系 / 非人类系 / 肉体改造系三类压迫节点影响结局分叉；档案只保留「是否外泄情报 / 是否拒绝改造 / 是否伪自愿表演换消息」等可玩戏剧，不展开感官内容。

乐园阶位映射：平民≈一阶；常规军≈二～三阶；装甲骑女主战力≈四阶；世界顶点≈四阶顶（宁低勿高）。超阶巅峰不适用；若遇魔铠/完全支配个体，按四阶 Boss＋精神/系统污染规则处理，条件性胜利。

**【地理 · 舞台】**
- **行星レフト（Left）**：极端女性蔑视风习残留，人身买卖网络盘根错节。伊莉丝与梅莉日常任务是强袭并扑灭奴隶贸易节点。猎奴成功时她们是秩序的锤；猎奴失败时同一套物流链会把她们当货物。レフト不是单一国家，而是联邦辖区与反联邦都市交错的殖民地行星——同一片大气下并存两套法律与两套价格表。
- **联邦辖区都市群**：受联邦法律保护，与レフト上反联邦都市对立。宣传部在这里循环播放装甲骑女的胜利影像，与敌后现实形成时间差。征兵海报上的伊莉丝笑容，往往比她本人更「不死」。
- **区域冲突前线 / ホリック降下作战空域**：联邦对反联邦都市联军的空降与突入战区。对空网、诱饵航迹与兽人清道夫共同把「特种作战」改写成「俘获经济」。此处是骑女身份翻转的第一现场。
- **敌对势力都市ミリオン（Million）·通称ダークタウン**：黑市、军阀、奴隶拍卖与娼馆经济共生的暗黑都市；伊莉丝被俘后主要囚禁地。权力以竞价、编号与项圈频率运行，而非军法。穹顶广告与地沟货柜共用同一套物流编号体系。
- **SSF 第 101 特殊装甲中队基地**：通称「装甲骑女」部队驻地，女队长编制，最新锐装甲骑列装。食堂空座、机库空位与宣传片同框，是联邦侧气压。
- **奴隶拍卖会场**：商品化仪式的中心舞台。观众含军阀、技术买家、羞辱联邦的富商与宣传承包商。展示台高于观众席，编号灯亮于姓名灯。
- **娼馆拘束设施 / 改造手术室**：俘虏商品化与意志磨损的中转与终点。档案只写编制、权限门禁与手术预告单，不写场景细节。下层水循环与门禁地图是静默撤离的关键情报。
- **敌方研究所与装甲骑残骸库**：解析 ADT 黑匣子、项圈频率、驾驶适配数据的技术节点；月影残骸若流入此处，敌方装甲骑技术可能跃进一代。
- **タゾック邸等レフト黑产宅邸**：战前猎奴名场面发生地；战后残党可在拍卖会以「报复性竞价」现身。

本分支额外强调：**拍卖会场的权力仪式**、**娼馆产权编号**、**联邦宣传部仍在播放胜利影像**的时间差、**项圈把突围改写成自杀**的规则空间。

**【世界剧情线】**

① **惑星レフト：猎奴者的日常**  
人类扩张至星际后，宇宙联邦政府以多种族松散结盟维持秩序。联邦保安军 SSF 第 101 特殊装甲中队因女队长与几乎全女队员、列装最新锐装甲骑，被通称为「装甲骑女」。中尉**伊莉丝·雷贝尔**驾驶 **ADT-74 月影（ゲツエイ；英译亦作 Griffin）**，搭档**梅莉·纳塞利**（埃尔芬族）驾驶 **ADT-73 金刚（コンゴウ；英译亦作 Chimera）**。两人在惑星レフト反复强袭人身买卖网络：对她们而言，扑灭奴隶贸易既是任务，也是「对恶人可以冷酷到底」的正义出口与压力阀。

レフト的人身买卖呈分层结构：码头收货人负责「货」的装卸与伪装；武装中介负责护送与灭口；宅邸大物（如**タゾック**）负责订单、洗钱与政治庇护。伊莉丝惯用月影中庭突入与近战超振动刀具完成斩首；梅莉以金刚高空压制与传感器网封锁逃逸。公开资料中出现过对タゾック邸的急袭——梅莉以装甲骑巨铳抵住タゾック头部的名场面，体现装甲骑女「治安维持＋反恐」双重职能。猎奴成功后的拌嘴日常（伊莉丝孩子气护短、梅莉知性吐槽）是后续坠落的情感落差来源：观众（与联邦宣传）记得的是英雄广告，不是俘虏编号。

日常任务节奏可复原为「情报切入—突入—清点—移交—宣传剪辑」五段：SSF 先收线人民与传感器异常，装甲骑女负责不可谈判的武力段，再把活口与账本移交联邦司法或殖民警；宣传部只剪最后三十秒的机体立姿与胜利口号。伊莉丝把「对恶人可以冷酷」当作职业伦理，因此她在レフト积累的敌人名单极长——码头、中介、宅邸残党都会在她被俘后以看守、竞价者或线报贩子的身份回流。梅莉则以埃尔芬式长期视角记录每一次任务的技术参数：对空网更新、走私货柜合金成分、项圈黑市原型是否已在レフト试装。这些参数在ホリック之后全部变成营救或沦陷的变量。

猎奴成功时，两人把「任何手段对付恶」当作准绳；这一准绳在她自己成为商品后会反噬——她无法用同一套冷酷对待「伪自愿」表演，也无法轻易原谅自己为求生而做的妥协。日常段的功能不是温情填充，而是建立**符号溢价的来源**：联邦越把她们剪成广告，敌方越把俘获她们写成可回收资产项目。

② **区域战争前夜：符号被磨成刀**  
レフト上反联邦都市结盟，对联邦侧都市发动区域战争。战争经济与黑产共生：俘虏、残骸零件、识别码与宣传素材都可出售。第 101 中队被派往前线时，伊莉丝与梅莉已不只是战术单位，更是联邦「女性精锐可镇压黑暗」的活广告。广告越亮，俘获溢价越高——敌方从一开始就把「猎杀王牌」写成可回收的资产项目，而非单纯歼灭。

战争前夜的情报战同样重要：反联邦侧释放虚假航迹，诱导联邦把装甲骑女投入对空网最密的空域；联邦宣传部则要求「必须有影像可剪」，逼中队在战术上多做一次高风险过顶。伊莉丝的护短伦理与梅莉的传感器谨慎在此出现裂痕——前者要先救人，后者要先确认诱饵。裂痕本身不是背叛，却在ホリック日成为可被敌方利用的缝。

③ **ホリック降下作战：秩序崩溃**  
ホリック降下作战中，先行部队暴露，空降队遭对空炮火。对空网强度超预期；梅莉机偏航或陷入不利射界。伊莉丝为掩护梅莉机，切断自动回避系统改全手动，把自己机体滑入射线上——这是她「孩子气却绝对护短」的驾驶伦理，也是战术上把自身暴露给敌方火力与俘获链的关键失误点。激战中两机先后重创；伊莉丝苏醒时已落入战场鬣狗——**オーク人（兽人）战场清道夫**之手。

オーク清道夫不按意识形态作战：他们认徽章、残骸完整度与「女驾驶员溢价」。俘获是战场经济，不是信念审判。伊莉丝的军籍徽章、驾驶服识别与机体涂装在这一刻从荣誉变成价签。梅莉下落在主线中悬置：失联、被俘或转入外线，成为伊莉丝后续决策的精神锚与情报黑洞。

ホリック空域的失败可拆成三个可观察失误：先行部队暴露导致对空网提前点火；自动回避与护短伦理冲突，伊莉丝选择人而非系统；弹射/拖曳窗口被清道夫抢先。联邦事后复盘若只写「敌火力超预期」，会掩盖真正可改的变量——诱饵航迹、护短协议、清道夫分账点。契约者若切入作战前夜，最有价值的不是再加一门炮，而是改这三项中的任意一项，使「俘获经济」接不住人。

④ **从猎奴者到商品：结构性反讽**  
伊莉丝被当作奴隶押往敌对都市**ミリオン（通称ダークタウン）**。她此前一直在猎杀奴隶商人，如今自己成为待售商品——官方剧情明确写出这一讽刺（「奴隶商人を狩っていた自分が……売られる商品になる」）。押送分三段：荒野中转剥徽章与军籍证明；走私货柜把人与货物同级堆叠；入城后安装**爆弹项圈**。军籍被剥即象征性除籍；ADT-74 月影残骸或被拖走解析。此节点完成「骑女」从主体战斗者到「战俘资产」的身份降格。

押送途中的信息剥夺同样关键：伊莉丝不知道梅莉生死、不知道联邦是否已宣告她阵亡、不知道月影黑匣子是否已被拆。特种兵最依赖的「态势感知」被制度性清空。她仍会用任务语言给自己编号——观察门禁、数巡逻间隔、记押运员口音——但每一次成功的观察都可能被项圈遥测反向标记为「高风险库存」。

⑤ **恶魔项圈：突围选项被规则删除**  
项圈制度把特种兵的解题模板整表删除：不能硬突、不能约时间窗口强攻、不能用「我能打」替代「我知道频率」。伊莉丝仍以敌后渗透心态解题——观察门禁、记录巡逻、寻找可定位信标——但每一次失败都会被制度记分。任何营救方若不知项圈频率，强突等于杀死人质。契约者切入时，真正的 Boss 往往不是某个打手，而是**频率、产权与宣传**三条锁链。

项圈的权力语言是「可验证的死亡威胁」：它不需要时时施暴，只需要让所有相关方相信「拆弹失败=当场死亡」。因此破解项圈从来不是纯技术问题，而是技术＋信誉＋分账：谁持有母钥、谁敢在拍卖后改频、谁会因为你的破解动作失去利润。四阶稳玩法是先拿到「谁靠项圈赚钱」的名单，再谈物理接触。

⑥ **拍卖会：商品化仪式（OVA vol.01 节点）**  
公开拍卖中，伊莉丝被当作高价值联邦王牌展示与竞价。展示话术强调她的猎奴战绩以抬价，同时摧毁其自我叙事：你越是英雄，价越高，羞辱越完整。观众含军阀、技术买家、羞辱联邦的富商与宣传承包商。**博兹克（ボズク）**一类奴隶商人/娼馆利益链人物是关键推手；娼馆主竞得「所有权」后，将其编入娼馆劳动编制。军人自尊与商品编号并置，是本分支核心残酷点。

拍卖不只是交易，更是政治展演：联邦各殖民地若看到「英雄也可编号」，征兵叙事会裂，黒产会涨。タゾック残党可能报复性竞价，把旧日被强袭的耻辱买回来。技术买家则盯着驾驶适配数据与黑匣子，试图把伊莉丝从「人」拆成「接口样本」。

拍卖厅的空间秩序本身就是权力语言：展示台高于观众席，编号灯亮于姓名灯，武装门禁朝内不开向外。竞价轮次通常先炒「猎奴战绩」再炒「机体数据」最后炒「宣传可利用率」——三轮价格分别对应军人自尊、技术机密与舆论武器。博兹ク擅长把第三轮卖给军阀宣传承包商，使伊莉丝即使不被「使用」，也被「播出」。对契约者而言，打断第三轮（母带、信号、剪辑权）往往比抢走第一轮成交价更能改终局。

⑦ **娼馆监禁下的意志磨损（主线中段，只写制度）**  
原作以「客选择系统」推进状态：大类分为**人类系 / 非人类系 / 肉体改造系**，选择影响结局。**本档案禁止展开性描写**，只保留可玩戏剧节点：  
- 是否向联邦发出可定位的微弱信标；  
- 是否用中队战术情报换取项圈暂时解锁；  
- 是否接受「伪自愿」表演以换取梅莉生死消息；  
- 是否组织其他战俘暴动；  
- 是否拒绝改造手术而触发更严酷的拘束升级。  

肉体改造在原作中是「按客人订单改身体」的制度环节，档案中只记为**身份与战斗能力的永久改写风险**（驾驶适配下降、识别码失效、无法再合法归队）。人类系压迫的关键结构之一，是敌方收编或敌视装甲骑女的人类士兵以「假伊莉丝 / 友军羞辱」进行人格攻击——伊莉丝必须在「承认自己是伊莉丝」与「以替身身份求生」之间做身份博弈；任何承认都可能被录成敌方宣传素材。非人类系制造不可用联邦军规谈判的沟通障碍；改造系把身体变成可交易规格。三者共同把「特种中尉」降格为「可配置资产」。

监禁期的时间感被刻意打碎：没有可靠日历，只有班次表与项圈脉冲。伊莉丝用「任务日」给自己记时——第 N 次观察门禁、第 M 次尝试记录频率——这是她拒绝被改写成「库存」的最后语言。契约者若在此阶段切入，必须先学会读她的任务语言，而不是用乐园式劝降话术；她会把「同情」读成买家话术，把「命令」读成联邦毁证。

⑧ **リルル冲击：容貌相同、人格相反**  
在博兹克侧设施中，伊莉丝会遇到容貌酷似梅莉、性格迥异的少女**リルル**。公开资料与游戏主线指出：リルル与梅莉的关系涉及创伤防御、记忆封锁与身份分裂——梅莉本人可能以リルル人格存活于敌后体系中。本分支不把「双堕」写满，但必须保留冲击：伊莉丝若把リルル当假货，可能错过唯一内应；若把リルル当真梅莉，可能被伪自愿结构利用。身份确认本身就是高风险情报战。

リルル的存在改写了「营救」的定义：你要救的是记忆完整的梅莉，还是当前可用的内应，还是两者都不是、只是一个被体系训练出的接口？联邦若收到リルル影像，可能宣布「梅莉已死、影像皆伪」以止损——从而在政治上抛弃真人。本分支把リルル定位为**单人沉没线上的身份谜题与钥匙**，不把双俘联锁写成主轴。

⑨ **SSF 侧时间线：宣传与营救的裂缝**  
联邦宣传仍循环播放装甲骑女的胜利影像；第 101 中队内部就是否投入营救分裂——营救可能中埋伏，放弃则等于承认王牌可被商品化。宣传官要「英雄不死」的叙事；情报官可能压住拍卖单以免士气崩溃；队长在「派谁去死」与「保中队建制」之间撕裂。梅莉若存活于外线，将成为私人违令营救的火种；若梅莉已在敌后，则本分支气压转向「你救的是人还是符号」。

联邦侧最危险的不是冷漠，而是**切割算法**：若承认伊莉丝被俘，征兵与同盟信用受损；若坚持她阵亡，营救队就没有合法窗口。契约者带回的任何证据都可能被两派同时利用——营救派要活人，切割派要「已污染不可归队」的鉴定。因此「救回」与「归队」在本世界是两个不同任务。

⑩ **经济流与制度 Boss**  
俘获经济链路：オーク抽成 → 商人押运 → 拍卖税 → 娼馆编制 → 改造回流 → 军阀宣传。每一环都有记录员、受益人与可打断点。四阶稳玩法往往是**经济刺客**：伪造账、换频、毁母带、劫黑匣子，而不是在拍卖厅正面对轰。军阀私兵与残骸火力仍致命，但真正锁死终局的是项圈频率与舆论母带。

经济流的可观察证据包括：俘获分账账本（徽章溢价规则）、拍卖税单（政治庇护费）、娼馆编制表（编号与权限）、改造预告单（归队资格死刑状）、宣传母带清单（第三轮竞价标的）。契约者不必全拿，但至少要切断其中两环，才能把「商品」重新读成「人质」。

⑪ **沦陷完成与可选终局**  
- **深度沦陷终局**：军人自我叙事崩溃，成为敌方展示「联邦英雄已堕」的活广告，中队士气受创，战争转入舆论战。  
- **残留火种终局**：身体与编制被控，但她暗记拍卖网络与军阀后勤名单，等待梅莉或契约者触发。  
- **暴动失败终局**：暴动被镇压，项圈升级，营救窗口关闭。  
- **OVA vol.04 向「生还」岔路**：若外线装甲骑女部队飞来且内部出现协助者（リルル/梅莉相关线），可转入营救——但本分支默认玩家切入时仍处于「单人沉没」主轴。  
- **数据生还岔路**：肉体未归，但黑匣子与拍卖网络名单外泄，联邦发动定点清除。  

契约者应把**制度**当 Boss，而非单一反派。

游戏多结局与 OVA 四卷共同给出可映射的状态机：人类系压迫把「承认身份」变成宣传素材；非人类系压迫把军规谈判失效；改造系压迫把归队资格永久改写。真结局方向通常要求在完全被改写前找到梅莉/リルル线的裂口，并完成至少一次可外泄的情报动作（名单、频率、黑匣子）。坏结局则是编号固化、中队神话崩坏、联邦选择切割。OVA vol.04 的「生还」不是无代价奇迹，而是外线装甲骑女部队与内部协助同时成立时的窄门——本分支默认该门未开，玩家必须自己造门。

⑫ **与系列其他分支的边界**  
本分支强调伊莉丝**单人**沉没；梅莉主视角夺还见「妹妹援救」；双人尽没见「双堕深渊」；驾驶员与机体/异物强制合体见「魔铠融合」。禁止把四条线写成同一事件复读。骑女沦陷的独特性在于：**猎奴者被同一物流链商品化**、**项圈删除突围**、**拍卖把英雄变成编号**。

## 阶位切入点

> 阶位↔原作：四阶≈装甲骑王牌驾驶员与主战机体交战级（掀平建筑群/据点攻坚）。世界顶点为ミリオン军阀核心与完全控制下的「展示性战力」，按四阶顶处理。低配切入禁止开五阶以上外挂；情报与项圈规则优先于蛮力。世界无超阶巅峰；若遇完全支配/改造完成体，按四阶 Boss＋条件性胜利处理。乐园阶位映射与剧情条目一致：平民≈一阶；常规军≈二～三阶；装甲骑女主战力≈四阶；世界顶点≈四阶顶。

**四阶（骑女沦陷 · 拍卖与项圈都市线）**

切入身份/时点：契约者以 **SSF 临时特遣审计员**、**ダークタウン黑市掮客**、或**被击落的中立佣兵技师**身份切入。锚定时点为**伊莉丝·雷贝尔已被押入ミリオン、拍卖会前后 48 小时内**——爆弹项圈已安装，ADT-74 月影黑匣子尚未完全被解析。SSF 审计员可碰战俘财产清单与军籍注销草案；黑市掮客须先洗一笔レフト赃款才能进内场；中立技师可获项圈母钥三十秒级物理接触窗口。三种身份都会在开拍前被三方同时试探：联邦要你「处理污染资产」，博兹克要你「守场子规矩」，第 101 中队要你「带回活人」。

初始事件：你在拍卖会外围同时接到两份互相矛盾的委托。联邦渠道加密指令要求你「优先销毁目标，防止战术数据扩散」；娼馆中介则要你抬价保住「活体王牌」并护送交接。与此同时，项圈遥测显示伊莉丝仍在尝试以特种兵手法拆弹——她把拆弹当最后一次「任务」。开拍倒计时内你必须决定：劫标、假买、放任、或向第 101 中队发出会暴露你身份的信标。第一选择会被三方（联邦、博兹克体系、潜在营救链）永久记入立场。若你沉默超过一个竞价轮次，タゾック残党会以报复性抬价抢走话语权，把「猎奴者」当众标成旧仇纪念品。

开场白建议：「ミリオン穹顶广告循环播放联邦英雄的笑脸，而实物正在拍卖台上被报价。你的通讯器里同时跳进两条指令：毁灭她，或买下她。倒计时与项圈红灯一起闪。你听见有人用商品编号喊出『伊莉丝·雷贝尔』四个音节——那一刻，猎奴者的名字第一次被标了价。」

关键NPC立场：  
- **伊莉丝·雷贝尔**：仍以军人逻辑谈判；对「买下」本身感到羞辱，但可能利用买家作为突围道具；最怕你是联邦来毁证的人。  
- **博兹ク**：只在乎利润与供应链信誉；你若坏了他的场子会立刻标杀；你若守规矩加价，他可暂时当你客户。  
- **娼馆主**：把你当新客户或竞品；愿用权限换你交出联邦识别漏洞。  
- **梅莉·纳セリ**（若外线连入）：要求你优先救人而非任务完成度；会用埃尔芬传感器验证你是否说谎。  
- **リルル**：可能提供假解锁顺序或真内应路径；不可仅凭外貌信任。  
- **第 101 中队指挥链 / 宣传官**：要成果不要丑闻；可能事后把你当弃子。  
- **オーク清道夫中间人**：可卖出俘获路线图，价码是联邦弹药或机体零件。  
- **タゾック残党**：以复仇买家身份抬价，可能把你拖进旧仇清算。

主线钩子/支线：  
主线＝拍卖竞价/劫标、项圈频率破解、月影黑匣子夺还或销毁、决定伊莉丝沦陷深度（深度沦陷 / 火种 / 静默撤离 / 数据生还）。  
支线①：伪造「已阵亡」报告给联邦，换取秘密营救窗口。  
支线②：潜入タゾック旧部网络，用旧仇换取ミリオン内部地图与水循环图。  
支线③：接触リルル相关传闻，判断她是诱饵、创伤人格还是裂口。  
支线④：策反被俘联邦士兵，组织无爆炸风险的「静默撤离」。  
支线⑤：夺下或调换旧胜利广告母带，反制「英雄不死」宣传或阻止羞辱片外泄。  
支线⑥：截获オーク—商人分账账本，反推押送节点与内鬼佣金，切断下一轮「王牌猎杀」订单。

危险度/规避：危险度**高**。四阶机体残骸火力、军阀私兵与项圈同归于尽风险并存。必须规避：①对项圈盲拆导致人质死亡；②暴露乐园能力被格为「新实验体」；③为救人出卖完整中队战术导致战线崩溃；④在拍卖厅对轰引发产权方与军阀联手清场；⑤把リルル当梅莉本体公开点名，触发敌方「双货联锁」升级。世界顶点级军阀本体**情报优先**，不硬刚。

任务方向/奖励：目标自然落在「撕开营救口」或「确认沦陷不可逆并回收/销毁黑匣子」。奖励取向贴合四阶：装甲骑部件与驾驶日志、项圈解码数据、ダークタウン路线图、SSF 人情或敌方临时权限、レフト猎奴档案（可用于勒索黑产）、伪军籍芯片工艺、俘获分账账本副本。禁止发放可越阶毁城的战略兵器。若走数据生还，奖励以情报资产与定点清除授权为主，而非「当场救回王牌」的爽文结算。

开局节奏建议（非灌水清单，而是本阶独有事件链）：第 1 回合确认双指令真伪并选定公开身份；第 2 回合进入博兹ク试探酒局或等价验资场合，交出赃款/审计徽章/技师执照之一；第 3 回合必须回应项圈异常脉冲——那是伊莉丝在拆弹，你的沉默会被她读成「联邦毁证」。若选择假买，须在产权晶片落印前植入可撤销条款；若选择劫标，须先切断拍卖厅对外直播，否则军阀会把劫标剪成「联邦承认英雄已成赃物」。成功撤离的最低证据包是：伊莉丝存活信号、项圈频率片段、月影黑匣子或已销毁证明——三者缺一，联邦仍可能按「任务失败/目标已污染」切割你。
