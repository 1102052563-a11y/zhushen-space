# 装甲骑女伊莉丝-妹妹援救

## 剧情

**【作品来源】**
《装甲骑女伊莉丝》（装甲騎女イリス）Black Lilith ADV（2006，RJ022698）＋ PIXY OVA 四卷（2007–2009）。企划监督笹@，剧本そのだまさき，原画 ZOL，机械设定やしガニ。主角搭档为**伊莉丝·雷贝尔**与**梅莉·纳塞利**。OVA vol.04 标题《被虐战士的生还》明确出现「装甲骑女部队飞来夺还」的急转结构。**本档案取梅莉主视角援救线，只写情报战、违令突入、项圈破解与夺还代价，不写色情细节。**

**【世界定位】**
本世界锚定**「妹妹」式搭档梅莉·纳塞利的援救主轴**：伊莉丝已落入ミリオン后，梅莉突破指挥链实施夺还。核心冲突不是「能不能打赢一场机战」，而是**项圈规则、宣传战伪讯、以及救回身体却救不回军魂**的三重代价。埃尔芬族政治债与联邦军纪处分是贯穿压力。

**【世界观 · 力量体系】**
远未来，人类向星际扩张。多种族组成松散的**宇宙联邦政府**。联邦保安军 **SSF** 负责治安维持、反奴隶贸易与区域冲突介入。核心兵器是人型机动兵器**装甲骑**（ADT 系列等），由驾驶员神经链接操作。

**战力阶梯（破坏力对齐《阶位战力图鉴》，本条目仅开放四阶切入）：**
1. **非武装平民 / 被俘缴械士兵**：徒手与冷兵器级，≈一阶及以下；战场上面对装甲骑无有效反抗。
2. **常规步兵 / 轻型载具 / 百万都市奴隶警备**：可清据点杂兵，≈二阶～三阶边缘。
3. **SSF 特种装甲骑驾驶员（伊莉丝·雷贝尔、梅莉·纳塞利等）驾驶正役机**：机动摧毁装甲目标、突入都市与轨道设施、突破音障级机动，掀平建筑群与据点攻坚，典型≈**四阶**。
4. **装甲骑中队协同 / 敌方主力舰炮 / 重型异种战力 / 军阀私兵装甲骑**：街区级毁伤，≈四阶中上～四阶顶。
5. **本世界叙事顶点**（百万军阀核心、改造兵器、魔铠完成体、被完全控制的王牌展示体）：仍落在四阶顶～五阶门槛之间；**本条目按四阶世界处理**，禁止以六阶以上外挂入场秒杀。

死亡与俘获规则：驾驶员离机后几乎等同常人；爆弹项圈把「突围」改写成「自杀」；改造手术与神经接口可永久改写战斗身份。  
乐园阶位映射：平民≈一阶；常规军≈二～三阶；装甲骑女主战力≈四阶；世界顶点≈四阶顶（宁低勿高）。超阶巅峰不适用；若遇魔铠/完全支配个体，按四阶 Boss＋精神/系统污染规则处理，条件性胜利。

**【地理 · 舞台】**
- **行星レフト（Left）**：极端女性蔑视风习残留，人身买卖网络盘根错节；伊莉丝与梅莉日常任务是强袭并扑灭奴隶贸易节点。
- **联邦辖区都市群**：受联邦法律保护，与レフト上反联邦都市对立。
- **区域冲突前线 / ホリック降下作战空域**：联邦对反联邦都市联军的空降与突入战区；兽人（オーク）战场清道夫出没。
- **敌对势力都市ミリオン（Million）·通称ダークタウン**：黑市、军阀、奴隶拍卖与娼馆经济共生的暗黑都市；伊莉丝被俘后主要囚禁地。
- **SSF 第101特殊装甲中队基地**：通称「装甲骑女」部队驻地，女队长编制，最新锐装甲骑列装。
- **奴隶拍卖会场 / 娼馆拘束设施 / 改造手术室**：俘虏商品化与意志磨损的中转与终点。
- **敌方研究所与装甲骑残骸库**：解析 ADT 黑匣子、项圈频率、魔铠接口的技术节点。
本分支额外强调：**走私航线与传感器盲区**、**第 101 中队指挥舰通讯室**、**娼馆外围的撤离走廊**、**埃尔芬族私人实验室（临时改装金刚传感器）**。

**【世界剧情线】**
① **双王牌前史：护短与争吵**  
官方样本对话显示，伊莉丝与梅莉在任务后会为琐事拌嘴（如头发），战场上却会互相把机体挡进射线上。伊莉丝孩子气、梅莉胜气；埃尔芬长寿使梅莉常觉人类短视，却唯独信任伊莉丝的驾驶直觉。这种关系是援救线的情感燃料：梅莉不是在救「同事」，而是在救「唯一愿意为她切断安全系统的人」。

② **ホリック之后：MIA 与违令**  
超空/ホリック作战后伊莉丝通讯中断。官方初判战场 MIA，宣传部要求低调。梅莉根据残骸弹道、オーク迁徙路线与奴隶贸易季节航线，独自得出「被俘贩运」结论，与指挥部「等待正式侦察」方针冲突。她驾驶 **ADT-73 金刚**突破航路管制，进入通往ミリオン的走私航线——援救线正式开端。

③ **情报战：拍卖已结束**  
进入ダークタウン情报层后，梅莉确认：伊莉丝已被拍卖，爆弹项圈已装。正面强突娼馆＝引爆人质。她必须先拿到**解码母钥 / 同步频率 / 人质置换筹码**三者之一。敌方同步释放「伊莉丝已自愿堕落」的影像战，专门打击「埃尔芬式理性」：若梅莉相信伪讯，会把营救改成处决任务。

④ **リルル冲击：容貌相同的裂口**  
公开资料出现容貌酷似梅莉、性格迥异的**リルル**。对伊莉丝，这是「搭档已堕落」的心理武器；对梅莉，这是身份盗窃与诱饵。援救线必须回答：リルル是改造体、替身、独立人物，还是博兹克馆的「名物」？错误判断会导致梅莉把真伊莉丝当伪目标，或把诱饵当妹妹。

⑤ **营救方案树（结构，非 Sensational）**  
- **技术线**：潜入改造工房盗取项圈母钥；金刚近身电磁干扰掩护撤离。  
- **交易线**：以联邦战术残卷或机体部件换人，道德与军法代价极高。  
- **暴动线**：煽动战俘与底层奴隶同步起事，混乱中抢人，平民伤亡风险大。  
- **埃尔芬线**：以族群传感器阵列换取军阀暂时中立，梅莉欠下族内政治债。  
- **宣传线**：反向释放「伊莉丝仍在执行渗透任务」的联邦伪讯，争取指挥部正式授权。

⑥ **突入名场面设计（对齐 OVA vol.04「部队飞来」）**  
若梅莉成功说服第 101 中队部分队员，援救可升级为「小规模装甲骑女突入」。档案强调：突入必须与项圈解锁**同秒**，否则巷战火力会先杀死人质。金刚的科学优势用于破译频率；伊莉丝的驾驶直觉用于在无机体状态下指出建筑结构弱点（她在监禁中被迫记住了娼馆与拍卖场的动线）。

⑦ **夺还当下的二次崩溃**  
成功接触伊莉丝时，最危险的不是敌军，而是**身份确认失败**：伊莉丝可能拒绝相信「真梅莉」会出现；梅莉可能因伊莉丝的状态改变而犹豫是否仍该救。档案要求把这一幕写成**口令、旧争吵梗、只有两人知道的护短事件**三重验证，而不是简单抱头痛哭。

⑧ **撤离与军法**  
撤离成功后，联邦军法可能以「违令出境、泄露航线、私自开战」起诉梅莉；宣传部可能要求两人配合「英雄生还」造假纪录片。伊莉丝则可能因改造痕迹与项圈残留而**失去归队资格**。援救线的真正终局问题是：人回来了，军籍还在不在。

⑨ **失败分支（仍属本世界）**  
- 梅莉被俘，转入「双堕」；  
- 项圈爆炸，任务转为夺回黑匣子与为伊莉丝正名；  
- 救回身体，但伊莉丝拒绝再驾驶，中队失去一名王牌。  
契约者可介入任一节点，改变成败，但不能无代价抹掉项圈物理规则。

⑩ **与原作系统的衔接**  
原作 ADV 的「客选择影响结局」在援救线中转化为**时间压力**：伊莉丝在监禁中被推进的状态越深，解锁项圈所需密钥层级越高。梅莉每多耽误一天，技术线难度上升一级。
