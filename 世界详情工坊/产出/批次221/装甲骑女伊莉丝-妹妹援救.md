# 装甲骑女伊莉丝-妹妹援救
<!--meta lib=主库 tiers=四-->

## 剧情

**【作品来源】**
《装甲骑女伊莉丝》（装甲騎女イリス / Armored Warrior Iris / Armored Knight Iris）为 Black Lilith（Lilith Soft）于 2006 年 3 月 24 日发售的成人向 ADV（DLsite RJ022698；包装版约 3 月 31 日），企划监督笹@、剧本そのだまさき、原画 ZOL、装甲骑机械设定やしガニ；主角声优春日アン（伊莉丝）、鸟瀬くるみ（梅莉）。MangaGamer 于 2014 年推出英译下载版。PIXY 于 2007–2009 年推出同名成人 OVA 四卷：vol.01《超空的降下作战》、vol.02《娼馆的女战士》、vol.03《从战士到悦虐娼妇》、vol.04《被虐战士的生还》。**本档案取「妹妹」式搭档梅莉·纳塞利主视角援救线，对齐 OVA vol.04「装甲骑女部队飞来夺还」的急转结构，只写情报战、违令突入、项圈破解、身份确认与夺还代价，不写色情细节。**伊莉丝单人沉没见「骑女沦陷」；双人尽没见「双堕深渊」；机体强制融合见「魔铠融合」。

**【世界定位】**
本世界锚定**梅莉·纳塞利违令夺还伊莉丝·雷贝尔**的主轴：ホリック降下作战后伊莉丝落入ミリオン，梅莉突破指挥链实施营救。核心冲突不是「能不能打赢一场机战」，而是**爆弹项圈规则、宣传战伪讯、埃尔芬族政治债、军法处分，以及「救回身体却救不回军魂」的三重代价**。猎奴者被同一物流链商品化的反讽仍在，但叙事镜头从「被俘者意志战」切到「营救者如何在规则锁链上开锁」。

**【世界观 · 力量体系】**
远未来，人类离开地球向星际扩张，与多种族组成松散的**宇宙联邦政府**。联邦对行星自治的控制力有限：法律覆盖的都市与反联邦都市可同处一颗行星，边境治安靠军警与特种部队维持。联邦保安军 **SSF**（资料中亦写作 Space / Soviet Security Force 等转写）负责治安维持、反奴隶贸易与区域冲突介入。核心兵器是人型机动兵器**装甲骑**（Armored Knight / Armored Warrior，ADT 系列等），由驾驶员神经链接操作；驾驶员在机内可完成据点攻坚与街区级机动打击，**离机后几乎等同受训特种兵，无法与完整装甲骑正面对抗**。

装甲骑战斗的实际表现决定乐园映射：正役特种机可机动摧毁装甲目标、突入都市与轨道设施、突破音障级机动并掀平建筑群——对应图鉴四阶「摧毁据点 / 掀平建筑群」。中队协同、敌方主力舰炮、重型异种战力与军阀私兵装甲骑可把毁伤抬到四阶中上～四阶顶。本世界**没有**稳定的行星爆破级战力作为日常棋子；叙事顶点（ミリオン军阀核心、改造兵器、被完全控制的王牌展示体）仍落在四阶顶～五阶门槛之间。**本条目按四阶世界处理**，禁止以六阶以上外挂入场秒杀。

**战力阶梯（破坏力对齐《阶位战力图鉴》，本条目仅开放四阶切入）：**
1. **非武装平民 / 被俘缴械士兵**：徒手与冷兵器级，≈一阶及以下；战场上面对装甲骑无有效反抗。
2. **常规步兵 / 轻型载具 / 百万都市奴隶警备**：可清据点杂兵，≈二阶～三阶边缘。
3. **SSF 特种装甲骑驾驶员（伊莉丝·雷贝尔、梅莉·纳塞利等）驾驶正役机**：据点攻坚、建筑群级毁伤，典型≈**四阶**。
4. **装甲骑中队协同 / 敌方主力舰炮 / 重型异种战力 / 军阀私兵装甲骑**：街区级毁伤，≈四阶中上～四阶顶。
5. **本世界叙事顶点**（百万军阀核心、改造兵器、完全控制下的王牌展示体）：四阶顶～五阶门槛；仍按四阶世界结算。

**特殊系统与死亡/俘获/营救规则：**
- **神经链接驾驶**：装甲骑战力绑定驾驶员适配与机体完整度；机体大破后驾驶员可能被弹射或拖出，进入「离机即降阶」状态。
- **爆弹项圈（恶魔项圈）**：敌后制度核心。逃离或强拆失败即死，把特种兵最擅长的「硬突围」改写成自杀；营救方若不知频率，强突等于杀死人质。**本分支胜负手 = 解码母钥 / 同步频率 / 同秒突入**。
- **奴隶拍卖与产权晶片**：ミリオン把战俘变成可交易动产；编号、腕带、契约晶片构成法律拟制上的「所有权」。营救若只劫人、不处理产权记录，军阀可把「被劫商品」写成「联邦承认盗窃」的舆论片。
- **肉体改造订单**：原作以「客人订单」改写身体规格。档案只记制度功能——驾驶适配下降、识别码失效、无法再合法归队——不写过程。时间拖得越久，解锁项圈所需密钥层级越高。
- **宣传符号战**：装甲骑女既是精锐也是广告；被俘本身可被剪成「联邦英雄已堕」的舆论武器。营救方可用反向伪讯争取授权，也可被伪讯逼成「处决任务」。
- **双重人格/记忆封锁（梅莉—リルル线）**：原作中梅莉可能以「リルル」人格作为创伤防御机制出现于敌后。本分支将リルル保留为**身份盗窃 / 诱饵 / 内应 / 创伤人格**四选一谜题：外部营救的梅莉必须先回答「容貌相同者是谁」，否则会把真伊莉丝当伪目标，或把诱饵当妹妹。
- **军法与紧急避险**：违令出境、私自开战、传感器技术外泄均可起诉；成功夺还不等于军籍保全。营救终局问题常是「人回来了，驾驶资格还在不在」。

乐园阶位映射：平民≈一阶；常规军≈二～三阶；装甲骑女主战力≈四阶；世界顶点≈四阶顶（宁低勿高）。超阶巅峰不适用；若遇魔铠/完全支配个体，按四阶 Boss＋精神/系统污染规则处理，条件性胜利。

**【地理 · 舞台】**
- **行星レフト（Left）**：极端女性蔑视风习残留，人身买卖网络盘根错节。伊莉丝与梅莉日常任务是强袭并扑灭奴隶贸易节点。猎奴成功时她们是秩序的锤；猎奴失败时同一套物流链会把她们当货物。
- **联邦辖区都市群**：受联邦法律保护，与レフト上反联邦都市对立。宣传部在这里循环播放装甲骑女的胜利影像，与敌后现实形成时间差。
- **区域冲突前线 / ホリック降下作战空域**：联邦对反联邦都市联军的空降与突入战区。对空网、诱饵航迹与兽人清道夫共同把「特种作战」改写成「俘获经济」。本分支复盘起点。
- **敌对势力都市ミリオン（Million）·通称ダークタウン**：黑市、军阀、奴隶拍卖与娼馆经济共生的暗黑都市；伊莉丝被俘后主要囚禁地。权力以竞价、编号与项圈频率运行，而非军法。
- **SSF 第 101 特殊装甲中队基地 / 指挥舰**：通称「装甲骑女」部队驻地，女队长编制，最新锐装甲骑列装。本分支关键舞台包括**指挥舰通讯室**、**机库违令出航闸口**、**军法预审室**。
- **奴隶拍卖会场 / 娼馆拘束设施 / 改造手术室**：俘虏商品化与意志磨损的中转与终点。档案只写编制、权限门禁、撤离走廊与手术预告单，不写场景细节。
- **敌方研究所与装甲骑残骸库**：解析 ADT 黑匣子、项圈频率、魔铠接口的技术节点；月影残骸若流入此处，敌方装甲骑技术可能跃进一代。
- **走私航线与传感器盲区**：通往ミリオン的分段承包航路；假巡逻、二次倒卖与「猎救星」赏金机甲出没。本分支主航线。
- **埃尔芬族私人实验室 / 联络站**：可临时改装金刚传感器阵列、提供解码算力；代价是族内政治债与技术外泄风险。
- **タゾック邸等レフト黑产宅邸**：战前猎奴名场面发生地；战后残党可卖航线、内通或报复性情报。

本分支额外强调：**违令前十二小时的指挥舰**、**走私航段的时间压力**、**娼馆外围的同秒突入走廊**、**归队后的军法体检台**。

**【世界剧情线】**

① **双王牌前史：护短与争吵**  
人类扩张至星际后，宇宙联邦政府以多种族松散结盟维持秩序。联邦保安军 SSF 第 101 特殊装甲中队因女队长与几乎全女队员、列装最新锐装甲骑，被通称为「装甲骑女」。中尉**伊莉丝·雷贝尔**驾驶 **ADT-74 月影（ゲツエイ；英译亦作 Griffin）**，搭档**梅莉·纳塞利**（埃尔芬族）驾驶 **ADT-73 金刚（コンゴウ；英译亦作 Chimera）**。两人在惑星レフト反复强袭人身买卖网络：对她们而言，扑灭奴隶贸易既是任务，也是「对恶人可以冷酷到底」的正义出口与压力阀。

レフト的人身买卖呈分层结构：码头收货人负责「货」的装卸与伪装；武装中介负责护送与灭口；宅邸大物（如**タゾック**）负责订单、洗钱与政治庇护。伊莉丝惯用月影中庭突入与近战超振动刀具完成斩首；梅莉以金刚高空压制与传感器网封锁逃逸。公开资料中出现过对タゾック邸的急袭——梅莉以装甲骑巨铳抵住タゾック头部的名场面，体现装甲骑女「治安维持＋反恐」双重职能。

任务后的拌嘴日常是援救线的情感燃料：伊莉丝孩子气护短，梅莉知性吐槽；埃尔芬长寿使梅莉常觉人类短视，却唯独信任伊莉丝的驾驶直觉。官方样本对话显示，两人会为头发一类琐事拌嘴，战场上却会互相把机体挡进射线上。**梅莉不是在救「同事」，而是在救「唯一愿意为她切断安全系统的人」。**

② **区域战争前夜：符号被磨成刀**  
レフト上反联邦都市结盟，对联邦侧都市发动区域战争。战争经济与黑产共生：俘虏、残骸零件、识别码与宣传素材都可出售。第 101 中队被派往前线时，伊莉丝与梅莉已不只是战术单位，更是联邦「女性精锐可镇压黑暗」的活广告。广告越亮，俘获溢价越高——敌方从一开始就把「猎杀王牌」写成可回收的资产项目，而非单纯歼灭。

③ **ホリック降下作战：掩护与失联**  
ホリック降下作战中，先行部队暴露，空降队遭对空炮火。对空网强度超预期；梅莉机偏航或陷入不利射界。伊莉丝为掩护梅莉机，切断自动回避系统改全手动，把自己机体滑入射线上——这是她「孩子气却绝对护短」的驾驶伦理，也是战术上把自身暴露给敌方火力与俘获链的关键失误点。激战中两机先后重创；伊莉丝苏醒时已落入战场鬣狗——**オーク人（兽人）战场清道夫**之手。

本分支的关键分叉在此：**梅莉机未与伊莉丝同批被俘**，或虽重创却成功脱离射界，转入指挥舰外线。通讯中断后，官方初判伊莉丝战场 MIA，宣传部要求低调。梅莉根据残骸弹道、オーク迁徙路线与奴隶贸易季节航线，独自得出「被俘贩运」结论，与指挥部「等待正式侦察」方针冲突。**援救线正式开端。**

ホリック空域的失败可拆成三个可观察失误：先行部队暴露导致对空网提前点火；自动回避与护短伦理冲突，伊莉丝选择人而非系统；弹射/拖曳窗口被清道夫抢先。联邦事后复盘若只写「敌火力超预期」，会掩盖真正可改的变量——诱饵航迹、护短协议、清道夫分账点。梅莉在违令前夜把这三项写成座舱笔记，不是为了追责，而是为了证明「人还活着」。

④ **违令前十二小时：证据先于出航**  
梅莉不会无脑冲锋。她先取三类证据：オーク信标残片、奴隶季热成像、伊莉丝护短通话波形。座舱里写下一封未发送的族书：私人身份去，勿向联邦索赔。指挥舰通讯室同时堆着三份互斥文件——禁止出境令、埃尔芬科技换债合同草案、未签名的突入名单（中队队员愿跟她走）。伪讯频道开始循环播放「伊莉丝自愿留下」的剪辑，专门打击「埃尔芬式理性」：若梅莉相信伪讯，会把营救改成处决任务。

违令本身是军法事件：出境、开战、传感器技术可能外泄。梅莉把违令写成「紧急避险」的唯一出路，是事后交出可验证的人质存活证据与项圈频率片段。契约者若在此节点切入，最有价值的不是再加一门炮，而是帮她把证据链补到「军法预审官无法一票否决」的密度。

⑤ **传感器航线：时间等于推进深度**  
通往ミリオン的走私航分段承包。私货传感器阵列闻离子尾迹；途遇假巡逻、二次倒卖、猎救星赏金机甲。每一段延误都等于伊莉丝在监禁中被推进更深一层——原作以「客选择系统」推进状态，人类系 / 非人类系 / 肉体改造系选择影响结局；**本档案禁止展开性描写**，只保留制度含义：时间拖得越久，解锁项圈所需密钥层级越高，归队体检被否决的概率越高。

梅莉在航线上做三件事：校准金刚临时加装的埃尔芬传感器；用タゾック残党或走私船长买「无税吏走廊」；截获伪讯母带的中继节点。任何「英雄生还」的宣传冲动都会暴露航迹。她必须像科学家一样解题，而不是像复仇者一样加速。

⑥ **情报层：拍卖已结束，项圈已装**  
进入ダークタウン情报层后，梅莉确认：伊莉丝已被拍卖，爆弹项圈已装，产权晶片已落印。正面强突娼馆＝引爆人质。她必须先拿到**解码母钥 / 同步频率 / 人质置换筹码**三者之一。敌方同步释放「伊莉丝已自愿堕落」的影像战；联邦宣传部则可能仍在播放旧胜利广告——两边叙事都在删掉「营救」这个选项。

タゾック残党可能因旧恨卖内通，也可能把梅莉的行踪二次倒卖给军阀。オーク中介认钱不认主义，可卖俘获路线图。娼馆技师是最可贿的齿轮：他怕死，可用活命交换母钥物理接触时间。**真正的 Boss 往往不是某个打手，而是频率、产权与宣传三条锁链。**

⑦ **リルル冲击：容貌相同的裂口**  
公开资料出现容貌酷似梅莉、性格迥异的**リルル**。对伊莉丝，这是「搭档已堕落」的心理武器；对外部营救的梅莉，这是身份盗窃与诱饵。原作暗示リルル可能就是创伤防御机制下的梅莉本人；本分支允许四种解释并存，由切入后证据裁决：
- **创伤人格说**：梅莉曾短暂被俘并分裂，外部「梅莉」是记忆回流后的主人格或并行营救体（叙事上需严格自洽，避免双梅莉无解释）。
- **替身/改造体说**：敌方用埃尔芬样本或整容接口制造诱饵。
- **独立人物说**：リルル是馆内「名物」，外貌巧合被博兹克利用。
- **内应说**：リルル主动埋线，等待口令。

错误判断会导致梅莉把真伊莉丝当伪目标，或把诱饵当妹妹。身份确认必须靠**口令、旧争吵梗、只有两人知道的护短事件**三重验证，而不是外貌。

⑧ **营救方案树（结构，非感官）**  
梅莉与可能的共犯（契约者、愿违令队员、族联络官、情报贩）面对五条主路径，可组合不可无代价抹平：
- **技术线**：潜入改造工房盗取项圈母钥；金刚近身电磁干扰掩护撤离。成功率绑定传感器与技师接触窗口。
- **交易线**：以联邦战术残卷或机体部件换人，道德与军法代价极高；博兹克愿谈，价码常是金刚传感器数据。
- **暴动线**：煽动战俘与底层奴隶同步起事，混乱中抢人，平民伤亡风险大，宣传部事后最难粉饰。
- **埃尔芬线**：以族群传感器阵列换取军阀暂时中立，梅莉欠下族内政治债；族联络官只认契约条款，不认人类感情。
- **宣传线**：反向释放「伊莉丝仍在执行渗透任务」的联邦伪讯，争取指挥部正式授权；可与技术线叠加，成熟组合常是「宣传打底＋技术或族债执行」。

⑨ **同秒协议：突入名场面（对齐 OVA vol.04）**  
若梅莉成功说服第 101 中队部分队员，援救可升级为「小规模装甲骑女突入」。档案强调：**突入必须与项圈解锁同秒**，否则巷战火力会先杀死人质。时间轴可复原为：
- T-30：干扰项圈基站与拍卖厅直播；
- T-10：破墙 / 切断产权门禁；
- T-0：注入频率 / 母钥；
- T+5：口令三重验证；
- T+20：伊莉丝指出建筑动线弱点（她在监禁中被迫记住了娼馆与拍卖场的结构）；
- T+60：接应机与撤离走廊汇合。

金刚的科学优势用于破译频率；伊莉丝的驾驶直觉用于在无机体状态下当「活地图」——她不是包裹，是撤离的第二指挥官。友军识别敌我码必须提前灌入，否则会在窄巷误击。

⑩ **夺还当下的二次崩溃**  
成功接触伊莉丝时，最危险的不是敌军，而是**身份确认失败**：伊莉丝可能拒绝相信「真梅莉」会出现，把营救当成审讯手段；梅莉可能因伊莉丝的状态改变而犹豫是否仍该救。应激下伊莉丝甚至可能攻击营救者。档案要求把这一幕写成三重验证，而不是简单抱头痛哭：军语点名、拌嘴梗、ホリック骂笨蛋语气。通过后仍可能出现「救回身体、拒绝再驾驶」的半成功。

⑪ **撤离与军法：人回来了，军籍呢**  
撤离成功后，联邦军法可能以「违令出境、泄露航线、私自开战」起诉梅莉；宣传部可能要求两人配合「英雄生还」造假纪录片。伊莉丝则可能因改造痕迹与项圈残留而**失去归队资格**。军法体检台是第二战场：撒谎或可保军籍，骄傲抗拒撒谎；可解释创伤报告成为军医与宣传官博弈的筹码。

成功夺还的最低证据包是：伊莉丝存活信号、项圈频率片段、月影黑匣子或已销毁证明——三者缺一，联邦仍可能按「任务失败/目标已污染」切割两人。梅莉的违令出航日志既是罪证，也是「我愿意为她犯罪」的筹码。

⑫ **终局分支（仍属本世界）**  
- **完美夺还**：人回、频率回、黑匣子回；军法以紧急避险轻判；中队神话强化。代价是族债或交易线留下的技术外泄。  
- **半成功**：只救一人；丢黑匣子；伊莉丝除籍成幽灵驾驶员。比完美 HE 更适长期线。  
- **宣传生还**：身体回，但必须配合假纪录片；两人成为联邦符号，失去私人叙事权。  
- **失败—双堕**：梅莉被俘，转入「双堕深渊」世界逻辑。  
- **失败—项圈爆炸**：任务转为夺回黑匣子与为伊莉丝正名；营救变追悼。  
- **失败—军魂已死**：救回身体，伊莉丝拒绝再驾驶，中队失去一名王牌；梅莉赢了军法、输了搭档。  

契约者可介入任一节点改变成败，但不能无代价抹掉项圈物理规则。原作 ADV 的「客选择影响结局」在援救线中转化为**时间压力**：伊莉丝在监禁中被推进的状态越深，解锁项圈所需密钥层级越高。梅莉每多耽误一天，技术线难度上升一级。

⑬ **与系列其他分支的边界**  
本分支强调**梅莉主视角违令夺还**；伊莉丝单人沉没见「骑女沦陷」；双人尽没见「双堕深渊」；驾驶员与机体/异物强制合体见「魔铠融合」。禁止把四条线写成同一事件复读。妹妹援救的独特性在于：**违令前的证据战**、**项圈同秒协议**、**リルル身份裂口**、**军法与归队资格的二次战场**。

## 阶位切入点

> 阶位↔原作：四阶≈装甲骑王牌驾驶员与主战机体交战级（掀平建筑群/据点攻坚）。本分支顶点压力来自「项圈同归于尽＋军阀私兵装甲骑＋伪讯舆论战」，仍按四阶顶处理。情报、破解与撤离优先于毁城。世界无超阶巅峰；若遇完全支配/改造完成体，按四阶 Boss＋条件性胜利处理。低配切入禁止开五阶以上外挂。

**四阶（妹妹援救 · 违令夺还线）**

切入身份/时点：契约者以 **第 101 中队临时联调员**、**埃尔芬族传感器技师**、或**走私航线领航员**身份切入。锚定时点为**梅莉已确认伊莉丝在ミリオン、正式营救尚未获批**——金刚已加装私货传感器，违令出航倒计时 6 小时。联调员可见禁止出境令红章与军法预审草稿；技师可藏改装日志并接触族债合同；领航员知道走私船长价码与无税吏走廊节点。

初始事件：梅莉把你拽进金刚的副座（或通讯位），丢给你三份互斥文件：指挥部的「禁止出境令」、埃尔芬联络官的「科技换债合同」、以及一份未签名的突入名单（中队队员愿跟她走）。座舱安全带仍是伊莉丝的尺寸；仪表盘有ホリック留下的焼蚀痕。与此同时，伪讯频道开始循环播放「伊莉丝自愿留下」的剪辑，剪辑里有近似声纹却错了口型。你必须在起飞前决定：帮她伪造授权、帮她谈判族债、把她绑回禁闭室「保护性拘留」，或先截伪讯母带再谈出航。第一选择会被三方（指挥链、埃尔芬、ミリオン情报贩）永久记入立场。

开场白建议：「金刚的座舱里全是红灯：航路管制、军法警告、还有一条被压成雪花点的私人频率——那是伊莉丝以前跟她拌嘴时才会用的短码。梅莉头也不回地说：你要么帮我把人带回来，要么现在就从舱里滚下去。你听见远处宣传屏还在播装甲骑女的胜利广告，而倒计时已经跳进六小时。」

关键NPC立场：  
- **梅莉·纳塞利**：把你当工具人也可以当共犯；最怕你向指挥部告密；会用传感器验证你是否说谎。  
- **伊莉丝·雷ベル**（远端）：可能不信任任何「营救者」；需要口令三重验证；应激下可能攻击闯入者。

- **リルル**：态度不明，可能给钥匙也可能给假钥匙；不可仅凭外貌信任。  
- **第 101 中队队长**：若你能提供「可宣传的生还脚本」，她可能补盖章；否则通缉。
