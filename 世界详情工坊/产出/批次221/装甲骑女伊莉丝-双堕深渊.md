# 装甲骑女伊莉丝-双堕深渊

## 剧情

**【作品来源】**
《装甲骑女伊莉丝》Black Lilith ADV（RJ022698）与 PIXY OVA 四卷。原作以伊莉丝被俘监禁为主轴，梅莉行踪为悬念；OVA 与官方文案反复强调「二人能否从犯罪都市生还」。**本档案取「双堕」分支：伊莉丝与梅莉先后或同时落入ミリオン体系，重点写双王牌符号的政治毁灭、互相牵制的人质结构与联邦舆论崩盘，不写色情细节。**

**【世界定位】**
本世界锚定**双王牌尽没**：不是一个人的沦陷故事，而是「装甲骑女」这一品牌被整体撕碎的事件。伊莉丝与梅莉互相成为对方的项圈——救一个会炸另一个，反抗一个会伤害另一个。敌方目标从「获得一名驾驶员」升级为「摧毁联邦最耀眼的女性精锐神话」。

**【世界观 · 力量体系】**
远未来，人类向星际扩张。多种族组成松散的**宇宙联邦政府**。联邦保安军 **SSF** 负责治安维持、反奴隶贸易与区域冲突介入。核心兵器是人型机动兵器**装甲骑**（ADT 系列等），由驾驶员神经链接操作。

**战力阶梯（破坏力对齐《阶位战力图鉴》，本条目仅开放四阶切入）：**
1. **非武装平民 / 被俘缴械士兵**：徒手与冷兵器级，≈一阶及以下；战场上面对装甲骑无有效反抗。
2. **常规步兵 / 轻型载具 / 百万都市奴隶警备**：可清据点杂兵，≈二阶～三阶边缘。
3. **SSF 特种装甲骑驾驶员（伊莉丝·雷贝尔、梅莉·纳塞利等）驾驶正役机**：机动摧毁装甲目标、突入都市与轨道设施、突破音障级机动，掀平建筑群与据点攻坚，典型≈**四阶**。
4. **装甲骑中队协同 / 敌方主力舰炮 / 重型异种战力 / 军阀私兵装甲骑**：街区级毁伤，≈四阶中上～四阶顶。
5. **本世界叙事顶点**（百万军阀核心、改造兵器、魔铠完成体、被完全控制的王牌展示体）：仍落在四阶顶～五阶门槛之间；**本条目按四阶世界处理**，禁止以六阶以上外挂入场秒杀。

死亡与俘获规则：驾驶员离机后几乎等同常人；爆弹项圈把「突围」改写成「自杀」；改造手术与神经接口可永久改写战斗身份。  
乐园阶位映射：平民≈一阶；常规军≈二～三阶；装甲骑女主战力≈四阶；世界顶点≈四阶顶（宁低勿高）。超阶巅峰不适用；若遇魔铠/完全支配个体，按四阶 Boss＋精神/系统污染规则处理，条件性胜利。

**【地理 · 舞台】**
- **行星レフト（Left）**：极端女性蔑视风习残留，人身买卖网络盘根错节；伊莉丝与梅莉日常任务是强袭并扑灭奴隶贸易节点。
- **联邦辖区都市群**：受联邦法律保护，与レフト上反联邦都市对立。
- **区域冲突前线 / ホリック降下作战空域**：联邦对反联邦都市联军的空降与突入战区；兽人（オーク）战场清道夫出没。
- **敌对势力都市ミリオン（Million）·通称ダークタウン**：黑市、军阀、奴隶拍卖与娼馆经济共生的暗黑都市；伊莉丝被俘后主要囚禁地。
- **SSF 第101特殊装甲中队基地**：通称「装甲骑女」部队驻地，女队长编制，最新锐装甲骑列装。
- **奴隶拍卖会场 / 娼馆拘束设施 / 改造手术室**：俘虏商品化与意志磨损的中转与终点。
- **敌方研究所与装甲骑残骸库**：解析 ADT 黑匣子、项圈频率、魔铠接口的技术节点。
本分支额外强调：**双人并置展示台**、**互相可见的监室结构**、**联邦紧急听证会会场**、**第 101 中队空编后的士气崩塌点**。

**【世界剧情线】**
① **神话的制造**  
联邦宣传把伊莉丝与梅莉打造成不可分割的双王牌：人类热情与埃尔芬理性、月影与金刚、护短与吐槽。レフト猎奴影像被剪成英雄广告。神话越亮，坠落越值钱。

② **第一次坠落：伊莉丝**  
ホリック作战后伊莉丝被オーク俘获并送入ミリオン，路径同主线。不同点在于：敌方从一开始就设局捕捉「第二只」。他们释放可追踪的伪信标，诱导梅莉违令前来。

③ **第二次坠落：梅莉入瓮**  
梅莉按「妹妹援救」逻辑突入，却踏入双俘陷阱：项圈频率与金刚被锁定，或リルル诱饵将她引入无支援的巷战。两名中尉同时出现在ダークタウン——对博兹克与军阀而言，这是可拍卖的「成套藏品」。

④ **人质结构：互为引信**  
双堕的核心规则：  
- 伊莉丝反抗 → 梅莉项圈示警/惩罚；  
- 梅莉破解 → 伊莉丝被转移至更深层；  
- 联邦强攻 → 两人被推上公开处刑或公开展示的舆论绞索。  
档案只写结构：任何单线英雄主义都会被制度反噬。

⑤ **政治战：联邦听证会**  
双王牌尽没迫使联邦召开紧急听证会：是承认失败并组织营救，还是切割「个人违令」以保中队建制？宣传部主张切割；第 101 中队主张全军夺还；埃尔芬族则可能要求优先撤出本族成员梅莉——这一提案会摧毁伊莉丝对盟友的信任。

⑥ **展示与毁灭神话**  
敌方举办「双骑女」公开活动（档案只记权力展演）：目的是让联邦各殖民地看到「英雄也可编号」。タゾック残党与新军阀借此招股战争经济。伊莉丝与梅莉若被拍到互相搀扶或互相冷漠，都会被剪成不同叙事武器。

⑦ **内部选择：牺牲算法**  
两人被允许进行一次「只能活一个」的虚假谈判（经典双人质压力测试）。伊莉丝倾向自己留下；梅莉倾向用技术同归于尽换数据外泄。契约者切入时，最可玩的是**打破虚假二选一**，而不是站队谁更该死。

⑧ **リルル与身份污染**  
当リルル以「第三张同脸」出现，双堕升级为身份污染：联邦无法确认影像中哪个是梅莉。埃尔芬族可能宣布「梅莉已死、影像皆伪」以止损，从而在政治上抛弃真人。

⑨ **终局分叉**  
- **双灭神话终局**：两人被彻底编入敌方体系，第 101 中队解散或改名。  
- **一人生还终局**：牺牲结构成立，生还者带着创伤与军法继续战争。  
- **数据生还终局**：肉体未归，但黑匣子与拍卖网络名单外泄，联邦发动定点清除。  
- **同归火种终局**：两人合作完成一次无生还保证的情报爆破，成为地下传说。  

⑩ **与其他分支边界**  
单人沉没见「骑女沦陷」；梅莉成功夺还见「妹妹援救」；机体融合见「魔铠融合」。双堕的独特性在于**人质互锁与品牌毁灭**，禁止写成两个单独沦陷的简单相加。
