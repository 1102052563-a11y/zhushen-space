# 死侍

## 剧情

**【作品来源】**
《死侍》（Deadpool）为漫威漫画反英雄角色，本名韦德·温斯顿·威尔逊（Wade Winston Wilson），由罗布·利菲尔德（Rob Liefeld）与法比安·尼谢萨（Fabian Nicieza）创作，1991年2月首次登场于《新变种人》#98。最初定位为超级反派/刺客，后在《X特攻队》与1997年起连载中逐渐成为以「嘴炮佣兵（Merc with a Mouth）」闻名的反英雄：毁容、近乎不死的自愈因子、枪剑格斗专家、打破第四面墙。  
本世界档案以**电影主干**为可玩时间线主轴，漫画为设定与人物补强：  
- 《恶棍英雄：死侍》（Deadpool，2016，提姆·米勒执导，莱恩·雷诺斯主演，R级）  
- 《死侍2》（Deadpool 2，2018，大卫·雷奇执导）  
- 《死侍与金刚狼》（Deadpool & Wolverine，2024，薛恩·李维执导，漫威电影宇宙第五阶段）  
关联作品含 X 战警系列电影宇宙（二十世纪福斯线）与 MCU 接轨后的时间变异管理局（TVA）设定。文风：黑色幽默、R级暴力、吐槽、反英雄情感内核；服务「轮回乐园」世界库主库，覆盖阶位一、二、三。卡片气质：佣兵夜生活、X武器黑实验、高速公路猎杀、冰囚堡与囚车劫、虚无废土——不是正统英雄片，是嘴炮与血的复仇/救赎喜剧。

**【世界定位】**
近现代北美都市 + 变种人阴影社会：特种部队退役佣兵、酒吧黑市、非法「治疗」实验室、X战警学园边缘与变种人监狱并存。主角**韦德·威尔逊／死侍**在癌症末期被骗入阿贾克斯的改造实验，获得自愈却毁容，化身红衣佣兵复仇、护爱、护少年，最终卷入多元宇宙与时间管理局的棋局。一句话：嘴炮不死佣兵的爱、恨与第四面墙。契约者切入的是**低阶街头—佣兵—变种人黑产**舞台；世界顶点（高阶英雄/宇宙级实体）在本库仅覆盖至三阶，低阶一律规避顶点·情报优先。

**【世界观 · 力量体系】**
力量本源（电影+漫画共通逻辑）：  
1. **变种基因／异变**：天生变种人（钢人、青少女弹头、多米诺、火拳等）或实验激活潜在变种能力。  
2. **武器 X／黑产改造**：以血清、酷刑、缺氧箱等手段强行激发自愈或超级士兵素质；韦德线即此。  
3. **科技与军火**：枪械、武士刀、炸药、未来兵器（机堡）、抑制项圈（冰囚堡束能圈）、时光设备（机堡时光表／TVA 时间平板）。  
4. **特殊叙事权能（死侍专属气质）**：打破第四面墙（知晓自己在故事中）、多重人格对话；写正文可作吐槽与元叙事，**不得**当成「随便改写世界法则的无敌外挂」——它不直接等于物理战力。  
5. **漫画补强（高阶背景，本库低阶少碰）**：萨诺斯诅咒相关的「难以真正死亡」、魔法背包瞬间移动等；电影线以自愈+武技+装备为主。

死亡与复活：凡人一刀一枪即死；有自愈者可断肢再生、爆炸后拼回，但抑制项圈会关闭能力并让癌症回潮；真正「死」在情感线上仍昂贵（凡妮莎中弹、韦德挡枪）。世界无无限免费复活；时间旅行改局有代价与悖论。

**战力阶梯（按实际破坏力，宁低勿高）**：  
1. **凡人／街头**：平民、脱衣舞俱乐部、出租车司机、普通帮派。徒手或冷兵器互殴，热武器致命。乐园观感≈一阶初期。  
2. **佣兵／特种兵**：退役特种部队、黄鼠狼酒吧常客、阿贾克斯手下雇佣兵。枪械、近战、小队战术；可端据点、清车队。≈一阶中后期～二阶初期。  
3. **强化体／初阶变种**：沙尘天使级蛮力、部分武器 X 实验体、戴抑制圈后的「半残」超凡。裂车掀人、硬扛常规火力一段时间。≈二阶。  
4. **稳定超凡·街头天花板**：死侍（满自愈+枪剑）、钢人（钢铁之躯怪力）、青少女弹头（自身爆炸）、多米诺（概率幸运）、火拳（火焰）、红坦克（蛮力冲撞，本库视为三阶压迫者）。可屠一支车队、打穿建筑结构、重创升空母舰支架级设施，**达不到**夷平整城或战略核当量。≈二阶后期～三阶。  
5. **本库顶点（三阶峰值）**：机堡（半机械、重火力、时间设备）、阿贾克斯（无痛觉+改造技术掌控）、冰囚堡体系与红坦克级囚犯。一怒可毁设施/小队，仍属地区级以下。  
6. **超阶（本库不写可挑战，仅作背景气压）**：完整 X 战警顶配、萨诺斯、时间变异管理局高层、卡桑德拉·诺瓦、碎时器级装置、多元宇宙骨干部——低阶切入**情报优先／条件性胜利**，严禁「被封印所以战力为零」式削弱。

乐园阶位映射：凡人·街头混混≈一阶；特种佣兵·初级实验体≈一～二阶；死侍／钢人／沙尘天使／多米诺／火拳≈二～三阶；机堡·阿贾克斯·红坦克·冰囚堡体系≈三阶峰值；宇宙级／TVA／诺瓦／碎时器=**超阶**（存在·规避）。宁低勿高；本世界任务库最高切入三阶。

**【地理 · 舞台】**
- **北美都市夜生活区**：韦德与凡妮莎相遇的脱衣舞俱乐部、街巷、公寓；一阶日常与情感线主场。  
- **黄鼠狼的佣兵酒吧**：情报、军火、死亡轮盘「死侍」命名出处、悬赏与嘲讽的社交场；一～二阶任务板。  
- **盲妇爱儿的合租屋**：韦德养伤、换红衣、日常互怼；低阶安全屋气质。  
- **阿贾克斯移动实验室／氧舱设施**：非法变种激活实验、毁容与自愈诞生地；二阶黑产副本。  
- **高速公路车队猎杀现场**：韦德截击阿贾克斯车队、钢人与青少女弹头介入；二阶名场面。  
- **废料场·除役升空母舰**：阿贾克斯挟持凡妮莎、三方混战、母舰坠落；一作终盘。  
- **X 宅邸（X 学园）**：钢人收留、训练、变种人青少年；二作情感与收容线。  
- **埃塞斯变种人教养院**：虐待变种少年的机构，罗素·柯林斯（火拳）仇恨源头；二～三阶。  
- **冰囚堡（Ice Box）**：关押变种人的监狱，抑制项圈，黑汤姆·卡西迪等狱霸；二～三阶。  
- **囚车公路／高空跳伞劫车现场**：X 特攻队首秀与惨烈减员；三阶。  
- **虚无（The Void）与 TVA 设施**（《死侍与金刚狼》）：被时间管理局放逐的废土与官僚空间；对一～三阶契约者属**超阶气压区**，只宜情报/逃亡支线，不宜硬刚诺瓦或碎时器。  
- **10005 号宇宙故乡都市**：韦德卖二手车、朋友生日派对、凡妮莎关系线收束舞台。

势力地盘由街头佣兵网 → 武器 X 黑产 → X 战警边缘 → 变种人监狱系统 →（背景）TVA 层层碾压；本地人若听闻一阶要屠机堡本体，会当疯子——正确反应。

**【世界剧情线】**
① **佣兵韦德与凡妮莎（一作前史·情感锚）**  
前特种部队佣兵韦德·威尔逊退伍后以「打抱不平」式临时打手为生，嘴碎、好色、否认自己是英雄。在常去的俱乐部结识脱衣舞娘**凡妮莎·卡莱尔**，热恋并谈婚论嫁。名场面：俱乐部搭讪、同居日常、求婚气氛。世界状态：纯凡人情感喜剧；暴力仅限街头纠纷。因果：凡妮莎是韦德全部软肋与日后复仇、自毁、救赎的引擎。

② **癌症与「史密斯干员」的契约**  
韦德确诊癌症末期。为不拖累未婚妻，他夜逃求医。自称**史密斯干员**的招募员（韦德戏称其像《黑客帝国》史密斯）游说：组织科技可治病。韦德签约，进入由**阿贾克斯（弗朗西斯·弗里曼）**与**沙尘天使**管理的移动实验室——实为非法变种改造与超级士兵黑产，目标是改造后卖作奴隶，不是治病救人。

③ **氧舱、毁容与死侍诞生**  
阿贾克斯注射激活变种基因的血清；韦德不停嘲讽其本名「弗朗西斯」，激怒对方。阿贾克斯用氧舱极端手段逼出自愈因子：癌症被压住，皮肤却因缺氧严重毁容腐烂。韦德偷火柴引爆设施，得知阿贾克斯是唯一可能「治好脸」的人。逃出后无颜见凡妮莎，投奔老友**黄鼠狼**，以酒吧死亡轮盘名「死侍」为号，戴面具当佣兵复仇；与盲妇**爱儿**合租，在她劝说下换上红色战服。名场面：氧舱酷刑、实验室爆炸、面具初戴、红衣定装。

④ **高速公路猎杀与 X 战警搅局（一作中盘）**  
死侍酷刑拷问阿贾克斯线人以定位目标，在高速公路截击车队，屠尽大部分手下并擒住阿贾克斯。正要逼问「解药」时，**钢人**与**青少女弹头**赶到制止私刑。阿贾克斯趁乱逃走；钢人劝韦德加入 X 战警，韦德锯断被铐的手逃逸并再生。主题：反英雄与正统英雄伦理冲突；「解药」谎言埋钩。

⑤ **凡妮莎被绑与废料场终战（一作终盘）**  
阿贾克斯从黄鼠狼酒吧取走韦德与凡妮莎合影，绑架凡妮莎，约废料场除役升空母舰赎人。韦德以「答应考虑加入 X 战警」换钢人、青少女弹头助战。三方混战：钢人对沙尘天使，青少女弹头以自身爆炸重创母舰支架导致坠落，韦德在舰上对决阿贾克斯。阿贾克斯坦白**根本没有恢复容貌的解药**；韦德无视钢人劝阻枪杀阿贾克斯。战后凡妮莎愤怒于被抛弃，韦德摘下面具坦白毁容与爱意，两人和解热吻。片尾死侍打破第四面墙预告续集有机堡。结局状态：阿贾克斯死、凡妮莎复合、韦德仍是嘴炮佣兵而非官方英雄。

⑥ **两年间的国际佣兵与纪念日悲剧（二作开篇）**  
一作约两年后，韦德以死侍身份接国际佣兵单，与凡妮莎计划成家。纪念日当晚，曾被韦德放过的俄国毒枭带人寻仇，**凡妮莎中枪身亡**。韦德屠光仇敌后六周自毁：堆油桶自爆仍因自愈失败「死成碎片」，被钢人拼回并拖入 X 宅邸强制「训练」。韦德仅剩凡妮莎送的铅制 skee-ball 游戏币作遗物。主题：爱的丧失、自杀与假英雄课。

⑦ **火拳罗素、冰囚堡与机堡降临**  
韦德随钢人、青少女弹头出任务，制止在埃塞斯变种人教养院门口纵火的少年变种人**罗素·柯林斯（火拳）**。发现罗素遭教养院虐待后，韦德枪杀施虐职员，被钢人放弃维护。两人送入**冰囚堡**，戴抑制项圈：韦德癌症回潮，遭**黑汤姆·卡西迪**欺凌。来自未来的半机械士兵**机堡（内森·萨默斯）**穿越而来——未来的罗素成暴君杀其妻小，机堡要斩草除根。监狱冲突中项圈损坏，韦德与机堡缠斗并双双被炸出监狱；凡妮莎幻象指引韦德必须保护罗素。

⑧ **X 特攻队与囚车惨剧**  
韦德联合黄鼠狼招募**多米诺、喧嚣、碎星、消影人、时代精神**与普通人**彼得**组成「X 特攻队」劫囚车救罗素。跳伞遇强风，除韦德与多米诺外几乎全灭（彼得等「炮灰喜剧死亡」）。机堡再至；罗素放出冰囚堡最强囚犯**红坦克**，红坦克一拳毁囚车并将韦德撕成两半。韦德长回下半身期间与机堡谈条件：给时间劝降罗素。钢人一度拒绝援助。

⑨ **教养院终局、挡枪与时间线补丁（二作终盘）**  
罗素回教养院追杀校长；红坦克难敌，钢人率青少女弹头与其女友**雪绪**赶到，合力将红坦克电晕。韦德戴上抑制圈向罗素示诚；机堡射击时韦德挡枪「死去」，感化罗素，机堡因时间线改变见女儿泰迪熊复原而知家人得救。机堡用尽能源回溯，把游戏币垫在韦德中枪处使其免死。罗素被纳入韦德「大家庭」；教养院孩童由 X 方收留；嚣张校长被韦德召来的出租车司机**阿杜**撞死。韦德偷修时光设备：救回凡妮莎与彼得，并整活枪杀《X 战警：金刚狼》里的「11 号武器」死侍与签约《绿光战警》前的莱恩·雷诺斯本人（元叙事玩笑）。阶段结局：凡妮莎复活版时间线、罗素得庇护、机堡滞留当代。

⑩ **六年后的二手车与 TVA（《死侍与金刚狼》主线·高气压）**  
韦德曾潜往 616／神圣时间线求入复仇者，被快乐·霍根拒绝。约六年后与凡妮莎分手，与彼得卖二手车。生日派对上被**时间变异管理局**逮捕，见**悖论先生**：因「骨干人物」金刚狼已死，韦德所属 10005 号宇宙将缓慢崩解；悖论欲用**碎时器**加速毁灭。韦德偷时间平板试图复活金刚狼，墓中仅剩亚德曼合金骸骨，遂穿越多元宇宙抓「最差」金刚狼变体，与之被悖论放逐**虚无**。

⑪ **虚无、卡桑德拉·诺瓦与碎时器（终局背景）**  
虚无中韦德与金刚狼遇霹雳火等弃子，被**火狂**等押见**卡桑德拉·诺瓦**（心灵感应，查尔斯·泽维尔孪生相关设定）。诺瓦杀霹雳火、玩弄记忆；两人逃出后加入抵抗势力（幻影杀手、刀锋战士、牌皇、X-23 劳拉等）。火狂实为悖论卧底；诺瓦获悉碎时器后欲毁诸宇宙只留虚无，并召唤死侍军团。韦德与金刚狼以身作导体短路碎时器，奇迹生还；猎人 B-15 逮捕悖论。10005 宇宙因两人行动停止崩解；韦德与凡妮莎重修旧好，金刚狼与 X-23 融入其朋友圈。片尾彩蛋重播韦德「没诬陷霹雳火」的证据。  
对乐园一～三阶：本段为**超阶棋盘摘要**，契约者可沾边跑腿、护送朋友、在都市侧收尸，**禁止**写成低阶硬刚诺瓦/碎时器/TVA 高层。

⑫ **漫画主干补强（与电影并行的背景史）**  
韦德早年家庭破碎（记忆中父母版本混乱，巴特勒等反派操控心灵的设定见于漫画）、加拿大特种作战与佣兵生涯；为治癌症接受武器 X 实验，植入类金刚狼自愈却与癌细胞互斥导致毁容与精神不稳。曾与**模仿猫**等关系纠葛；与**死亡女神**有感情线而遭萨诺斯嫉恨（不死诅咒版本）。好友**黄鼠狼**、跟班**九头蛇特工鲍勃**、技术与情报网贯穿多系列。道德灰色：常杀黑帮与恐怖分子，仍被制裁者、部分英雄厌恶。可与蜘蛛侠互怼、与 X 特攻队/雷霆特攻队等有任务交集。写正文时电影线优先落地，漫画人名与设定作支线与彩蛋，避免与已定电影结局直接打架。

⑬ **微观可介入事件（按阶，勿跨阶混用）**  
一阶：俱乐部看场、替黄鼠狼跑腿送货、跟踪出轨雇主、出租车阿杜线情感吐槽、帮凡妮莎处理骚扰客（非超凡）。  
二阶：截击阿贾克斯外围车队、潜入移动实验室外围、在高速公路战役捡漏、为死侍提供换装与医疗、教养院外围取证虐待。  
三阶：冰囚堡越狱协作、囚车劫掠中的生存位、教养院终盘侧翼、阻止机堡误杀少年的谈判、虚空边缘逃亡（不挑战诺瓦）。  
蝴蝶边界：可救路人、改一场小枪战归属、让某次绑架延迟；不可无代价抹掉凡妮莎之死的情感重量（除非已进入二作时间线补丁后的平行状态且任务允许）、不可让一阶契约者单挑机堡/红坦克/诺瓦。

⑭ **一作名场面因果链（细述，供任意时点判定）**  
韦德在俱乐部以「互损式调情」拿下凡妮莎后，生活短暂像浪漫喜剧：同居、互送廉价礼物、讨论未来。确诊场景切断笑声——医院走廊的白光比任何反派更冷。夜逃不是英雄独行，是愧疚驱动的懦夫选择：他怕凡妮莎守一个慢慢烂掉的人。招募员的话术精准戳中「还能变回值得被爱的样子」。实验室内，阿贾克斯把「治疗」包装成流程：注射、观察、加压、再加压；韦德的嘴炮在此是唯一反抗，也是激怒施虐者的燃料。火柴引爆是机会主义逃生，不是完美计划；废墟外路人对毁容的指点比刀伤更深，逼出面具刚需。高速公路战役里，死侍的战术是佣兵式清场而非英雄式救人；钢人介入把「私刑正义」钉上公共伦理。废料场母舰坠落时，青少女弹头的爆炸既是解围也是误伤风险模板——变种人能力没有友好模式。枪杀阿贾克斯的一枪，是韦德拒绝「被规训成英雄」的句号；和解之吻则承认：爱不需要一张完好的脸，但需要诚实。

⑮ **二作主题句与减员喜剧的残酷**  
凡妮莎死后，韦德的自毁不是耍帅：油桶、碎片、钢人一枚枚捡回的肢体，是「不死」作为诅咒的视觉化。X 宅邸训练段落讽刺「把反英雄塞进英雄学校」——说教解决不了丧偶。冰囚堡项圈让观众（与契约者）重新理解自愈：外挂可关，关了就是癌末凡人。X 特攻队跳伞是类型片自嘲：征召广告骗人、普通人彼得天真赴死、消影人隐形却冻死等笑点，本质是「超英团队片减员有多随便」。红坦克毁车撕人把喜剧瞬间扳回恐怖。挡枪感化罗素时，抑制圈+凡人脆弱=韦德第一次用「可能真死」换信任；机堡耗尽能源回溯，是反派动机被「家人还在」软化后的同盟化。时间线补丁救凡妮莎/彼得后，世界进入「幸福但韦德仍嘴炮」的稳态，直到三作 TVA 再次打碎日常。

⑯ **三作宇宙存亡与「最差金刚狼」**  
骨干人物理论把个人死亡升格为宇宙病：罗根已死，10005 缓慢崩解。悖论先生的诱惑是官僚版阿贾克斯——「我帮你进神组，顺便毁你的家」。韦德抓来的金刚狼变体被讥为多元宇宙最差，却在虚空冲突与碎时器共毁中完成「再当一次英雄」。抵抗势力（幻影杀手、刀锋、牌皇、X-23）是福斯时代弃角的集体回声，写正文可作彩蛋队友，勿让一阶契约者指挥全员。死侍军团被彼得破梗停手，延续「普通人意外关键」的喜剧物理。碎时器双人导体是条件性胜利范本：不是单人无敌，是分担伤害+刚好活下来。B-15 拒绝改金刚狼过去——创伤造就此刻能救人的他——与二作「改变死亡」形成对照：有的线可补，有的线必须留下。

忌：忽略凡妮莎/罗素情感线只堆砍杀。  
忌：代称凑人数、灌水标记、跨世界套话。  
最早可切入锚点：韦德确诊前后夜生活；或实验室出逃后黄鼠狼酒吧；或二作纪念日悲剧前夜；三作仅建议都市侧/TVA 外围。

## 阶位切入点

> 阶位↔力量：一阶≈凡人街头·普通佣兵；二阶≈强化体/初阶变种·死侍前期复仇战；三阶≈冰囚堡·机堡·红坦克·教养院终盘。世界顶点（卡桑德拉·诺瓦、碎时器、TVA 高层、完整宇宙级实体）为**超阶**，低阶切入一律「规避顶点·情报优先／条件性胜利」，严禁战力归零式削弱。

**一阶（凡人夜生活·佣兵酒吧跑腿线）**

切入身份/时点：契约者以酒吧保镖、俱乐部保安、私家侦探助手或出租车同行身份切入，锚定**韦德确诊前～刚夜逃求医前后**，或黄鼠狼酒吧日常接单期（阿贾克斯实验室尚在阴影中）。

初始事件：黄鼠狼酒吧里，一场「教训家暴雇主」的佣兵单出错——目标其实是阿贾克斯外围供货商的白手套。枪响之夜，你与嘴碎的韦德·威尔逊抢同一条巷子的逃生路线；凡妮莎在俱乐部后门抽烟，追债的人往这边来。

开场白建议：「霓虹把积水染成粉色。黄鼠狼酒吧的门一开，笑声、火药味和过期啤酒一起涌出来。吧台后的男人冲你扬下巴：『单子很简单——别死，别问，把信封送到第三街。』街角那个穿皮衣、嘴比枪快的佣兵朝你眨了眨眼：『搭个伴？我请客——用你的命。』」

关键NPC立场：**韦德·威尔逊**（尚未全毁容或刚确诊）——把你当临时炮灰兼吐槽对象，给钱就干，动凡妮莎立刻翻脸；**凡妮莎·卡莱尔**——护自己人，讨厌被当弱者；**黄鼠狼**——卖情报与枪，两头吃；**阿杜**——载你逃命，收双倍车费；**招募员（史密斯干员）**——在医院走廊散发「治愈」名片，危险的微笑。

主线钩子/支线：主线——阻止或目击韦德签下实验室契约的「最后一晚」（蝴蝶：延迟签约/调换地址）。支线：俱乐部保护费冲突；替黄鼠狼追假军火；帮阿杜处理跟踪狂前男友（凡人级）。

危险度/规避：危险度中（热武器、帮派）。规避阿贾克斯本体与完整实验室核心、任何「变种激活」手术台。

任务方向/奖励：护送、取证、救人质；奖励凡人军火、酒吧信誉、黄鼠狼的一次免费情报、与韦德的「损友位」。

**二阶（实验室出逃·高速公路猎杀线）**

切入身份/时点：契约者为佣兵、私家武装、或刚逃脱的实验外围人员，锚定**韦德出逃毁容后～高速公路截击阿贾克斯车队前后**。

初始事件：你在废仓库清点阿贾克斯流失的血清箱时，红衣面具人踹门而入——死侍要「名单上的下一张嘴」。交火三分钟后钢人的金属脚步震落灰尘：X 战警不打算让私刑直播继续。

开场白建议：「面具后的声音像在给你做脱口秀：『别紧张，我只是来问话——用刀问。』你还没答，窗外高速公路的车灯连成一条血河。对讲机里沙尘天使的呼吸很稳：『目标在东匝道。』你意识到：今晚要么当死侍的临时人质，要么当阿贾克斯的弃子。」

关键NPC立场：**死侍（韦德）**——要阿贾克斯下落与「解药」线索，可组队但随时开黄腔与背刺玩笑；**阿贾克斯**——视你为灭口对象或可利用的中间人；**沙尘天使**——武力镇压优先；**钢人**——要你放下致命武力，否则一并制伏；**青少女弹头**——嫌烦但关键时刻炸场；**盲妇爱儿**——你若带伤回家，她骂你比骂韦德还凶；**黄鼠狼**——加价卖车队路线图。

主线钩子/支线：主线——高速公路战役介入：救 equipage 上的实验体、截一箱血清、或在钢人铐住死侍时选择帮谁。支线：为死侍定制/修补红衣；追查「解药」谣言来源；保护凡妮莎不被照片线定位（若时点已到绑架前夜）。

危险度/规避：危险度高。规避与沙尘天使长时间正面硬刚、不引爆炸药车连坐平民；阿贾克斯本体系二阶 Boss，宜设伏与情报，不宜无计划决斗。

任务方向/奖励：变种血清样本（高危）、佣兵级枪械改造、死侍的一次「人情」、钢人的临时通行证（进 X 相关外围）、阿贾克斯实验室平面图残页。

**三阶（冰囚堡·火拳与机堡线）**

切入身份/时点：契约者为变种人囚犯、狱警内鬼、X 特攻队应征者或未来科技拾荒者，锚定**冰囚堡关押期～囚车转移～教养院终盘**（机堡已出现）。

初始事件：抑制项圈勒进你的皮肤。食堂里黑汤姆的人围住新来的少年火拳；死侍用癌痛嗓音讲冷笑话吸引仇恨。警报响时，穿着重型武装的机堡破墙而入：「罗素·柯林斯。死刑。」你手中的汤勺还没放下，选择已经变成：挡枪、开门、或卖位置。

开场白建议：「冰囚堡没有春天。项圈的指示灯一闪，你的能力像被拔掉的电源。走廊尽头，半机械的男人肩抗未来，光学瞄准点落在少年额心。红衣囚犯抬手：『嘿，未来大爷，杀孩子这事能不能排到我的脱口秀之后？』你听见自己的心跳——和囚车引擎在监狱外预热的声音。」

关键NPC立场：**机堡**——任务是杀罗素，可谈「先证明能改未来」但耐心极短；**罗素·柯林斯／火拳**——缺信任，谁肯当家人谁就能拉住他；**死侍**——嘴上说不在乎，挡枪时从不含糊；**多米诺**——相信幸运，行动派盟友；**钢人**——要合法与感化，讨厌滥杀；**红坦克**——释放即战场天灾，宜引开不宜硬刚；**教养院校长**——可暗杀的凡人魔头；**雪绪／青少女弹头**——终盘增援。

主线钩子/支线：主线——囚车劫案中保罗素存活并阻止机堡致命一击；或教养院终盘协助电晕红坦克、促成韦德示诚。支线：销毁一批抑制项圈密钥；救出教养院其他孩童；帮彼得这种普通人在跳伞前「假死离队」（改写炮灰命运）。

危险度/规避：危险度极高，贴近本库顶点。红坦克、满状态机堡=三阶峰值；**卡桑德拉·诺瓦、碎时器、TVA 裁决层=超阶**，若时间线滑到三作，只做虚空边缘逃亡与朋友圈护送，**情报优先，条件性胜利**，禁止硬刚。

任务方向/奖励：抑制项圈破解经验、多米诺式战术协作信誉、火拳的信任（人脉）、机堡未来弹药残件（不可复制量产）、X 宅邸临时收容权；禁止发放碎时器/完整时光表级奖励。
