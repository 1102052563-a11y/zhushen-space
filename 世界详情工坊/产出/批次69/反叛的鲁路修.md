# 反叛的鲁路修

## 剧情

**【作品来源】**
日升动画（Sunrise）制作、大河内一楼系列构成、谷口悟朗监督的原创电视动画。第一季《Code Geass 反叛的鲁路修》2006–2007 年播映 25 话；第二季《Code Geass 反叛的鲁路修 R2》2008 年播映 25 话；另有 OVA《亡国的阿基德》、漫画《双貌的奥兹》、剧场版三部曲与续作《复活的鲁路修》等。正典主轴以 TV 两季为准。文风：政治权谋、机甲战争、道德灰度与黑色悲剧并重；口号式热血与冷酷算计同框。卡片锚点（世界库提要）：成田连山崩坏——红莲二式首次实战，引发泥石流吞没不列颠尼亚军团，兰斯洛特在崩塌山体中机动，Knightmare 王牌对决。

**【世界定位】**
平行史观下的近未来地球：神圣不列颠尼亚帝国以「强者吞噬弱者」为国是，以 sakuradite（樱石）与Knightmare 机甲征服世界；被吞并的日本改称 Area 11，国民被蔑称「Eleven」。流亡皇子鲁路修·vi·不列颠尼亚化名鲁路修·兰佩路基，与不朽契约者 C.C. 立约获得「绝对服从」的 Geass，戴上面具成为「Zero」，组建黑之骑士团，为妹妹娜娜莉与「更温柔的世界」发动叛乱。契约者切入后是棋盘上的变数，不是免死天选。

**【世界观 · 力量体系】**
本世界无修仙灵气，战力由三层叠加：**常规武力**（枪械、装甲、情报与后勤）、**Knightmare 机甲（Knightmare Frame，KMF）**、**超常权能 Geass / Code**。

**一、 sakuradite 与工业底座**  
樱石是超导矿物，驱动 Yggdrasil Drive（世界树驱动）与能量填充器，是 KMF 高性能与帝国扩张的命脉。日本（Area 11）拥有世界最大樱石矿床，故 2010 a.t.b. 被帝国以约一个月闪电战吞并。失去樱石补给的战线会在数周内退化到「铁与血」的前 KMF 时代。

**二、Knightmare 世代阶梯（破坏力表现）**  
1. **步兵 / 治安部队 / 贫民区武斗**：徒手、冷热兵器、轻装甲车。≈乐园一阶。  
2. **精锐步兵 / 荣誉不列颠人小队 / 初期抗日游击**：战术素养与轻火力压过平民，仍怕重火力与 KMF。≈一~二阶。  
3. **四世代量产 Glasgow、抗日改型 Burai**：使主战坦克过时，可端据点、碾压街区巷战。≈二~三阶。  
4. **五世代 Sutherland、Gloucester、抗日 Gekka 等**：对 KMF 特化，精锐军官与「四圣剑」级老兵可在一城战场左右局部胜负。≈三~四阶。  
5. **七世代定制机（兰斯洛特早期、红莲贰式、加文等）**：单机可撕裂量产编队、掀翻建筑群、近乎突破音障机动；王牌驾驶员是一城仅数人的「精英 Boss」。≈四~五阶。  
6. **八~九世代与圆桌骑士机（兰斯洛特·征服者 / 阿尔比恩、红莲圣天八极式、加拉哈德、Percival 等）+ 初期 F.L.E.I.J.A.**：地区级灾难投影——可覆盖大城战役、空中超机动、能源翼；F.L.E.I.J.A.（Field Limitary Effective Implosion Armament）单发即可抹去东京租界级城区。≈五~六阶。  
7. **达摩克利斯要塞多联装 F.L.E.I.J.A.、皇帝级指挥权 + 大规模 Geass 动员、世界之 C / 诸神黄昏连结的规则层干预**：国家与文明级威慑；一怒可改写政治版图乃至人类集体潜意识。≈六~七阶。  

**三、Geass（王权之力）与 Code**  
Geass 由持有 Code 的不朽者（如 C.C.、V.V.）以契约授予。因人而异、通常自单眼鸟纹开始；过度使用可能「失控」常驻开启。已知正典实例：  
- **鲁路修**：绝对服从——视线接触 + 口头命令，对同一对象通常仅一次生效；可令自杀、遗忘、永久「活下去」。  
- **毛**：绝对听音——大范围读心，终致精神崩溃。  
- **罗洛·兰佩路基**：绝对中止——使范围内对象主观时间停滞（对无机物无效），代价是心脏负担。  
- **查尔斯·zi·不列颠尼亚**：记忆改写；双目可控的「完成体」。  
- **玛丽安娜·vi·不列颠尼亚**：意识转移（死后寄宿他身，如阿尼亚）。  
- **俾斯麦·瓦尔德施泰因**：数秒预知。  
Code 持有者近乎不死、免疫 Geass，可将 Code 交给 Geass 完成进化者从而恢复可死之身。世界之 C 是人类集体无意识海洋；「诸神黄昏连结（Ragnarök Connection）」企图以阿卡夏之剑杀死「神」并强制人类心灵合一——这是本世界的规则级顶点手段。

**四、死亡与复活**  
凡人与驾驶员一死即死；KMF 有弹射座舱但战场存活率仍残酷。Code 持有者可从致死伤再生，但非无限「游戏读档」。Geass 不能随意复活死者。顶点战的「条件性胜利」指切断连结、夺取政治合法性、以自己的死完成叙事等，严禁用「战力归零式软化」解释顶点。

乐园阶位映射：无 KMF 平民~治安部队≈一阶；精锐步兵/初期游击≈一~二阶；四世代量产 KMF 与普通驾驶员≈二~三阶；五世代精锐与军官王牌≈三~四阶；七世代定制机王牌（成田期兰斯洛特/红莲）≈四~五阶；八~九世代圆桌级 + 单发 F.L.E.I.J.A.≈五~六阶；达摩克利斯战略网 / 皇帝级 Geass 动员 / 世界之 C 干预≈六~七阶；诸神黄昏连结成功态与 Code 不朽棋手的规则层对决＝本世界顶点（七阶巅峰，条件性胜利，可记超阶概念威胁）。宁低勿高；顶点忠于原作。

**【地理 · 舞台】**
时间坐标 a.t.b.（Ascension Throne Britannia，不列颠尼亚登基历）。舞台是被三大势力切割的地球：  
- **神圣不列颠尼亚帝国**：美洲为腹地，首都潘德拉刚；以数字「Area」编号殖民地。Area 11＝旧日本，行政中心为**东京租界（Tokyo Settlement）**，日本人被驱入新宿等**贫民区（Ghetto）**。成田连山、神根岛、富士山周边、芦花公园、阿什弗德学园皆为剧情要地。  
- **中国联邦**：亚洲大陆霸权，皇帝（天子）为幼女蒋丽华，实权常在大宦官；后因零之政变与黎星刻而震荡，终并入超合国框架。  
- **E.U.（欧罗巴联合）**：欧洲—非洲一侧抗帝力量，与帝国长期消耗。  
- **樱石矿脉与京都**：日本旧财阀「六家」通过京都向抗日武装秘密输血，是黑之骑士团早期后勤命脉。  
- **海上与空中**：加文、达摩克利斯等浮游要塞把战场抬到立体；最终决战在富士山—达摩克利斯轴线。  
契约者写场景须先定：贫民区巷战 / 租界上流 / 学园日常面具 / 连山野战 / 国际外交桌 / 世界之 C 精神域，再选冲突强度。本地人若听闻「一个 Eleven 徒手推翻皇帝」会当疯子——这是正确反应，除非对方有 Zero 级叙事与 Geass。

**【世界剧情线】**

① **流亡与仇恨的底色（前史 · 2009–2010 a.t.b.）**  
皇妃玛丽安娜遇刺，皇子鲁路修与幼妹娜娜莉被流放日本；兄妹受首相之子枢木朱雀家庇护。2010 a.t.b. 帝国发动第二次太平洋战争，以 Glasgow 等 KMF 闪电征服日本，改称 Area 11。鲁路修目睹「强食」国是下的亡国，立誓摧毁不列颠尼亚；娜娜莉双目失明、下肢瘫痪，成为他全部温柔的寄托。朱雀之父枢木源太郎在占领后自杀，朱雀选择成为「荣誉不列颠人」以求从内部改变帝国——两条路线自此分叉。

② **新宿契约 · Zero 诞生（2017 a.t.b. 开篇）**  
鲁路修以平民学生身份就读阿什弗德学园，暗中赌博收集情报。新宿贫民区，大和路抗日小组（扇要、红月卡莲等）夺取装有毒瓦斯的胶囊——其中实为被实验的 C.C.。总督克洛维斯·la·不列颠尼亚下令屠杀平民灭口。鲁路修被卷入，遭行刑前与 C.C. 立约，获得绝对服从 Geass。他指挥反抗军逆转局部战局，又用 Geass 令克洛维斯自述玛丽安娜之死线索后将其处决。枢木朱雀驾驶实验机兰斯洛特介入，双方首次在战场交错。鲁路修戴上面具，自称「Zero」，以「为弱者伸张正义」为旗，逐步把扇组改组为**黑之骑士团**。

③ **立名与镜像（湖川事件、日本解放战线、朱雀线）**  
Zero 通过公开处刑毒枭、酒店人质事件等塑造「侠盗英雄」形象；情报官迪特哈尔特·里特、京都六家资源、卡莲的红莲系机体陆续入局。副王柯内莉亚·li·不列颠尼亚上任，以铁腕清剿。日本解放战线（JLF）藤堂镜志朗与四圣剑在军事上更专业，却被帝国与 Zero 双向利用。鲁路修与朱雀在学园仍是好友，战场上却是 Zero 与兰斯洛特的死敌；「杀死对方」与「无法真正下手」并存。此阶段世界状态：Area 11 治安战升级，Zero 名号开始值钱也开始招死。

④ **成田连山崩坏（卡片锚点卷 · 质变副本）**  
柯内莉亚围攻盘踞成田连山的 JLF。Zero 率黑之骑士团作为第三方介入。卡莲驾驶**红莲贰式**首次实战，以辐射波发生器与钻头臂破坏山体水脉，引发大规模泥石流，吞没不列颠尼亚军团阵地，亦波及平民（含舒尔茨家相关悲剧伏笔）。藤堂与四圣剑一度逼近柯内莉亚本阵；**兰斯洛特**在崩塌山体中极端机动护主，与红莲展开王牌对决。C.C. 接触兰斯洛特，以 Code 相关幻象冲击朱雀精神。黑之骑士团借此役扬名，JLF 却元气大伤，日后残部多被吸收。名场面要素：连山地形杀、红莲首秀、泥石流政治账、兰斯洛特救主、第三方收割名誉。对契约者而言，这是中阶可玩的「多方混战副本」：入口是情报与站队，核心争夺是柯内莉亚首级 / JLF 存续 / Zero 声望，出逃时面临帝国增援与泥石流二次灾害。

⑤ **修利的悲剧与毛事件 · Geass 的伦理税**  
泥石流间接导致同学舒尔茨·费内特之父死亡；鲁路修的「为大局牺牲」第一次被熟人血泪反噬。前 Geass 契约者毛以读心逼近 C.C. 与娜娜莉，暴露 Geass 失控者的末路；鲁路修不得不以更残酷的命令收场。世界规则对玩家提示：Geass 是王权也是诅咒，用一次就在人格上记账。

⑥ **特殊行政区与血染的尤菲米娅（SAZ）**  
公主尤菲米娅·li·不列颠尼亚推动「日本特别行政区」，试图让 Eleven 重新称自己为日本人并享有权利。鲁路修曾考虑政治收编；却在私人对话中 Geass 失控，对尤菲米娅下达「杀掉日本人口」的命令。尤菲米娅在典礼上屠民，Zero 亲手击毙她以止损，并对外把事件钉成「帝国假面下的屠杀」。和平路线断裂，日裔怒火与黑之骑士团动员力爆表。此节点是全季道德崩坏的中轴：理想主义公主之死、Geass 不可控、Zero 宣传战。

⑦ **黑色叛乱（第一季终盘）**  
黑之骑士团攻入东京租界，占领关键节点，以学园为临时枢纽。鲁路修以 Zero 之姿击败并审讯柯内莉亚，追问玛丽安娜真相。娜娜莉被掳往神根岛；鲁路修弃前线去救妹，指挥链断裂。杰雷米亚、尼娜核弹威胁、C.C. 与朱雀等多线绞杀。朱雀揭开 Zero＝鲁路修，对射收束第一季：叛乱失败，干部被俘或星散，Area 11 进入更严酷的戒严。

⑧ **R2：记忆删除与假弟罗洛（一年后）**  
皇帝查尔斯以 Geass 抹去鲁路修关于 Zero、皇室血脉、C.C.、娜娜莉等记忆；他以普通学生生活，身边多了「弟弟」罗洛——实为 V.V. 系刺客，拥有时停 Geass。C.C. 以吻与契约记忆唤醒鲁路修。Zero「复活」于监狱处刑秀，黑之骑士团重建。娜娜莉以总督身份现身且一度恢复行动能力，朱雀任其骑士——兄妹对立台面化。

⑨ **超合国与国际棋盘**  
鲁路修策动中国联邦：绑架/护送天子蒋丽华、联手黎星刻清洗大宦官，促成**超合国（U.F.N.）**，黑之骑士团成为其军事臂膀。战场从 Area 11 扩到全球。圆桌骑士、罗洛的忠诚与背叛、卡莲的身份撕裂、迪特哈尔特的摄像机真理同时运转。第二次东京战役中 F.L.E.I.J.A. 首次实战，城区蒸发，娜娜莉下落成谜，鲁路修心性再被碾碎。

⑩ **黑之骑士团倒戈与罗洛之死**  
修奈泽尔向黑之骑士团公开 Zero 的皇族身份、Geass 操控史与「利用日本」的证据。扇要、藤堂等选择逮捕 Zero。罗洛以时停拼命救出鲁路修，力竭心脏停跳而死——这个被制造的「弟弟」用生命完成自我定义。鲁路修失去军队，转入与父母的终局。

⑪ **世界之 C · 拒绝诸神黄昏**  
鲁路修与 C.C. 进入精神域，对质查尔斯与寄宿意识的玛丽安娜。父母的理想是发动 Ragnarök，杀死神、消除谎言与个体隔阂，强制人类合一。鲁路修拒绝「没有未来的温柔」，切断连结，抹去父母在世界之 C 中的存在。随后他以智略与 Geass 夺取皇位，成为第 99 代皇帝，任命朱雀为「零之骑士」。

⑫ **零之镇魂曲（Zero Requiem · 终局）**  
皇帝鲁路修故意行暴政、集中全世界仇恨于己身；控制达摩克利斯与 F.L.E.I.J.A. 网，以暴制乱，把战争收敛成「对暴君的十字准星」。阅兵处刑日，假死身份的**新 Zero（面具下为枢木朱雀）**当众刺死皇帝。鲁路修含笑而亡，宣布自己既破坏也重建了世界。仇恨随暴君死去而落幕，超合国与新秩序得以喘息；朱雀必须永远当 Zero，弃个人幸福守住和平；娜娜莉在兄长血泊中读懂牺牲；C.C. 继续漫长人生。结局铁则：核心是「暴君之死换世界喘息」，契约者可改局部战役与谁活谁伤，但不可无代价把「零之镇魂曲」改成无血童话或让顶点无条件崩塌。

⑬ **微观政治贯穿（日常可介入）**  
配额与通行证、荣誉不列颠人考试、贫民区物资黑市、樱石走私、学园祭面具下的情报交换、新闻导演（迪特哈尔特式）、Knightmare 零件黑市、Geass 实验体失踪案、圆桌骑士巡游造成的交通管制。AI 正文应优先这些高频事件，而不是一上来皇帝与达摩克利斯。

⑭ **蝴蝶效应边界**  
可改：救下某场泥石流边缘的平民、截获一批 Sutherland、让某次公开演说多一个镜头、推迟某干部倒戈。难改/高代价：尤菲米娅之死的因果核、黑色叛乱指挥链断裂逻辑、F.L.E.I.J.A. 技术一旦出现的扩散、零之镇魂曲的结构。改了局部，原作因果会以「帝国换指挥官清场、黑之骑士团提前倒戈、Geass 追捕优先级上升」回流。

**【主要人物】**

- **鲁路修·vi·不列颠尼亚 / 鲁路修·兰佩路基 / Zero**（主角·第 11 皇子→第 99 代皇帝）｜性格：极度聪慧、表演型人格、为目的不择手段却把「温柔的世界」全部押在娜娜莉上；对棋如对人生，既傲慢又常自我厌恶。｜装备·能力：Geass「绝对服从」；指挥加文、蜃气楼等；后期皇权与战略武器网；智力与宣传战是真正主武器。｜人物弧光：流亡学生→假面革命家→被祖国与战友双重抛弃者→自愿成为人类仇恨容器的暴君→以死完成镇魂。｜立场关系：视 C.C. 为唯一知全盘的契约搭档；视朱雀为镜像挚友/死敌；对卡莲有利用与真心的混杂；对修奈泽尔是棋逢对手；对契约者先当棋子评估，能用则用、威胁娜娜莉或揭破面具则杀。

## 阶位切入点

> 阶位↔原作强度：无 KMF 平民/治安≈一阶；精锐步兵与初期游击≈一~二阶；四世代量产 KMF≈二~三阶；五世代精锐与军官王牌≈三~四阶；七世代定制机王牌（成田期兰斯洛特/红莲）≈四~五阶；八~九世代圆桌级+单发 F.L.E.I.J.A.≈五~六阶；达摩克利斯战略网/皇帝级 Geass 动员/世界之 C≈六~七阶。世界顶点（查尔斯·玛丽安娜的诸神黄昏连结、Code 不朽棋局、零之镇魂曲级政治终局）为跨阶规则威胁，低阶切入一律「规避顶点·情报优先」。本世界仅覆盖：一~七阶。

**一阶（贫民区与租界边缘 · 井底生存）**
切入身份/时点：契约者以 Area 11 贫民区居民、黑市跑腿或阿什弗德外围勤杂身份切入，锚定 2017 a.t.b. 新宿屠杀前数日。
初始事件：新宿区突然戒严，不列颠尼亚巡逻队以「搜毒瓦斯」为名清场；你被夹在扇要小组撤离路线与帝国治安墙之间，必须选择带路、告密或趁乱偷一份通行证芯片。
开场白建议：「警报把新宿的铁皮屋顶震得发颤。扩音器里总督的声音在说『保护良民』，可激光瞄准点已经在 Eleven 的背上跳舞。你口袋里只有半块压缩口粮和一张假通行证——而街角那个紫发女人正被士兵抬上军用车。」
关键NPC立场：**扇要**——需要可靠跑腿，会给面包也敢在危急时丢卒；**红月卡莲**——敌视不列颠人面孔，对「肯挡枪的外人」短暂认可；**克洛维斯部下军官**——视你为可屠配额；**米蕾·阿什弗德**的巡回转播车组——可能把你拍成背景政治宣传。
主线钩子/支线：新宿事件边缘逃生；支线＝伪造荣誉申请材料、樱石碎料走私、给学园送私货时目睹鲁路修赌博圈。
危险度/规避：中——死于清场炮火与告密链；须规避任何 KMF 交战半径与皇族车队。
任务方向/奖励：活过戒严夜、换到合法身份碎片；奖励＝通行证、贫民区人脉、初级情报耳机，不发 Geass。

**二阶（荣誉不列颠人 / 抗日细胞 · 站队开端）**
切入身份/时点：以新晋荣誉不列颠人士兵或抗日细胞联络员切入，锚定 Zero 名号刚出现、湖川药剂事件前后。

开场白建议：「头盔里全是自己的呼吸声。对面的面具在霓虹里反光，像在嘲笑整座租界的法律。通讯器同时塞进来两套命令：帝国要你开火，贫民区的孩子要你让路——你的手指已经扣在扳机上。」
关键NPC立场：**枢木朱雀**——荣誉不列颠人前辈，逼你回答「正义是否等于军服」；**迪特哈尔特·里特**——愿用镜头换你的证词；**玉城真一郎**——冲动抗日分子，可能把你当奸细；**纯血派军官**——怀疑所有非纯血。
主线钩子/支线：Zero 早期形象战；支线＝协助/破坏湖川人质舆论、潜入阿什弗德舞会窃取嘉宾名单。
危险度/规避：中高——政治身份比子弹更易杀你；规避柯内莉亚本人与兰斯洛特试作出击。
任务方向/奖励：选定阵营标签；奖励＝制式武器改良件、双面证件、低层 KMF 模拟舱时数。

**三阶（量产 KMF 驾驶员 · 成田外围）**
切入身份/时点：以 Sutherland / Burai 驾驶员或 JLF 外围协力者切入，锚定成田连山战役爆发前 48 小时。
初始事件：你的小队接到 截击任务——切断山道补给。黎明时侦察显示黑之骑士团与帝国主力将在同一谷地交汇；泥石流风险报告被上级压下。你要选择执行、抗命疏散下游村庄，或把报告卖给第三方。
开场白建议：「成田的晨雾里，量产机的关节在滴冷凝水。参谋说『例行包围』，可地质传感器一直在尖叫。你看见远方一点赤红涂装掠过林线——那不是 Glasgow 该有的速度。」
关键NPC立场：**藤堂镜志朗**——要你守住侧翼军纪；**红月卡莲**（红莲贰式）——视量产机为移动标靶，但会认「敢救平民的驾驶员」；**柯内莉亚近卫**——只认命令链；**京都六家联络人**——高价收战场实时坐标。
主线钩子/支线：成田崩坏的蝴蝶（救不救下游）；支线＝抢修被辐射波烧融的关节、护送 JLF 伤员突围、抢拍红莲首秀影像。
危险度/规避：高——泥石流与王牌机碾压；须规避与兰斯洛特/红莲正面对轴、远离柯内莉亚旗机。
任务方向/奖励：在名场面边缘留下「你的版本」；奖励＝五世代零件、战场医函、黑之骑士团或帝国的见习徽章（二选一）。

**四阶（精锐军官 / 四圣剑级 · 成田至黑色叛乱）**
切入身份/时点：以 Gloucester 级精锐、四圣剑残员或黑之骑士团中队长安切入，锚定成田战后至 SAZ 典礼窗口。
初始事件：你奉命护送尤菲米娅视察特别行政区筹备点；人群中混有帝国强硬派刺客与 Eleven 极端分子。Zero 的广播突然接入，提出「把公主交给正义审判」的挑衅式停火。你要在护主、谈判与借刀之间三秒抉择。
开场白建议：「公主的笑容干净得不像这个 Area。你的瞄准镜里同时出现三枚不同阵营的徽章，而假面的声音从所有喇叭里同时响起——像是整个日本在问你：枪口对准谁才叫忠诚。」
关键NPC立场：**尤菲米娅·li·不列颠尼亚**——真心信任护卫，不知自己是风暴眼；**鲁路修 / Zero**——评估你是否可被 Geass 或演说收编；**柯内莉亚·li·不列颠尼亚**——若公主受伤会夷平你所属建制；**扇要**——希望你「至少别朝孩子开枪」。
主线钩子/支线：SAZ 血剧前后的站位；支线＝追查 Geass 失控传言、保护/绑架仪式主持人、切断迪特哈尔特直播。
危险度/规避：高——政治暗杀与 Geass 规则杀；规避正视可疑发光眼、不与皇帝直系长谈。
任务方向/奖励：改变局部平民存活数或公主安保质量；奖励＝精锐机涂装认证、特区通行证、一份被涂黑的 Geass 实验档案首页。

**五阶（七世代王牌侧翼 · 黑色叛乱到 R2 重建）**
切入身份/时点：以可驾七世代定制机的王牌、圆桌骑士侍从驾驶员或黑之骑士团零小队辅翼切入，锚定黑色叛乱攻入东京租界或 R2 Zero 监狱救援之夜。
初始事件：租界塔楼争夺战中，你的机位正好卡在「兰斯洛特突击路线」与「Zero 旗舰指挥链路」之间；娜娜莉转移车队的假情报与真情报同时传来。你选择堵截朱雀、掩护 Zero，或两头卖以换取撤离码。
开场白建议：「东京租界的玻璃幕墙在炮火里变成白色的雨。你的定制机仪表全是红光，通讯里有人喊『为了 Elevens』，有人喊『All Hail Britannia』——而你知道，今晚活下来的人会改写明天的教科书。」
关键NPC立场：**枢木朱雀**——只问你是否伤害无辜；**C.C.**——懒洋洋地暗示「别挡在契约者的棋盘中心」；**杰雷米亚·戈特瓦尔德**——情绪化敌意或后期式狂热忠诚（视时间线）；**娜娜莉·vi·不列颠尼亚**——若你接近护送队，她会成为你道德判决的镜子。
主线钩子/支线：黑色叛乱指挥链是否因你而多撑一小时；支线＝抢修能源填充器、拦截 Siegfried、在学园地下掩体疏散学生。
危险度/规避：极高——王牌互杀与皇族 Geass；规避查尔斯亲临、不碰神根岛深层遗构。
任务方向/奖励：获得名场面「侧写权」；奖励＝七世代武器挂架、假死身份一套、与黑之骑士团或圆桌的一次性呼号。

**六阶（圆桌 / F.L.E.I.J.A. 阈值 · 国际战争）**
切入身份/时点：以圆桌骑士候补、超合国联合参谋或 F.L.E.I.J.A. 相关技术护送官切入，锚定第二次东京战役前后至超合国成立窗口。
初始事件：你持有一枚未授权的 F.L.E.I.J.A. 保险键备份；修奈泽尔的特使与黎星刻的密使同一夜找到你，分别许诺「地区和平」与「民族解放」。城市在头顶燃烧，你必须决定销毁、交付或公开拍卖这份末日。
开场白建议：「保险键冷得像一块冰。窗外的蘑菇云还没散尽，新闻却已经在讨论下一枚该落在谁的首都。有人说你是英雄，有人说你是文明的叛徒——而你只知道，按下或折断它的人，会从六阶被历史记成七阶的注脚。」
关键NPC立场：**修奈泽尔·el·不列颠尼亚**——微笑着给你最优条件与最美锁链；**黎星刻**——用军人荣誉换你的良心；**罗洛·兰佩路基**——可能时停抢键；**拉克莎塔·乔拉**——只关心技术是否再被疯子使用。
主线钩子/支线：F.L.E.I.J.A. 伦理与超合国投票；支线＝护送天子蒋丽华、识破假娜娜莉情报、在神虎与兰斯洛特战场开一条撤离走廊。
危险度/规避：极高——战略武器与顶层阴谋；规避世界之 C 仪式场、不与查尔斯/V.V. 谈判 Code。
任务方向/奖励：左右一枚战略弹的归属；奖励＝八世代接口权限、超合国观察员身份、被涂改的圆桌提名文件。

**七阶（皇帝棋盘侧翼 · 世界之 C 与镇魂）**
切入身份/时点：以皇帝侧策士、超合国大将、Code 现象观察者或「假面执行人」候选人切入，锚定鲁路修称帝至零之镇魂曲阅兵日。
初始事件：新帝下令集中全世界仇恨，你的名字出现在「公开处刑名单」与「秘密赦免名单」两份文件上；C.C. 抛来半块披萨般的选择题——你是选择成为仇恨的燃料、镇魂的刀，还是试图偷改达摩克利斯的发射逻辑以拯救某个不该被牺牲的人。
开场白建议：「潘德拉刚的旗在新的风向里猎猎作响。暴君的笑容被投影在所有城市的天空，像神在示威。你知道有一把刀已出鞘，你不知道握刀的是朱雀还是命运——你只知道，若今晚还有人能改半步棋，那个人可能是你。」
关键NPC立场：**鲁路修·vi·不列颠尼亚（皇帝）**——要你理解并执行「可憎的和平」；**枢木朱雀（新 Zero）**——要你在刺出前后守住秘密；**C.C.**——以百年视角问你「死后名声是否重要」；**娜娜莉·vi·不列颠尼亚**——若得知真相会动摇整场镇魂；**修奈泽尔**——败局中仍想夺回叙事。世界顶点（查尔斯残影/诸神黄昏逻辑/Code 不朽层）＝**超阶巅峰：存在·情报优先/条件性胜利**，严禁硬刚本体与战力归零解释。
主线钩子/支线：零之镇魂曲是否多一名知情共犯；支线＝掉包处刑用剑、伪造皇帝 Geass 录像、在阅兵人群中拦截真正想「救下暴君」的变数。
危险度/规避：贴近顶点——死于政治、Geass、战略武器与历史本身；须规避重启 Ragnarök、夺取 C.C. 的 Code、在世界之 C 宣称自己为神。
任务方向/奖励：参与「结束仇恨」的终局结构；奖励＝后镇魂时代的新身份（和平侧高官/永世假面执行者/被抹去记录的自由人）、达摩克利斯残骸中的非活性技术样本、与 C.C. 的一次性问答权——不发完整 Code。
