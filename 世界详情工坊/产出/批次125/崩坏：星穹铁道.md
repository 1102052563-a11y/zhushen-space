# 崩坏：星穹铁道

## 剧情

**【作品来源】**
米哈游开发运营的回合制战略角色扮演游戏，崩坏系列第五作（前作含《崩坏学园》《崩坏学园2》《崩坏3》等）。2023 年 4 月 26 日全球上线 Windows / iOS / Android，同年 10 月 11 日登陆 PlayStation 5。制作人蒋大卫；引擎 Unity；免费运营 + 跃迁抽卡。定位「可玩的动画番剧」：以星球为章节的箱庭探索、指令式回合制战斗、模拟宇宙 / 差分宇宙类 Roguelike。2019 年立项，强调策略性与上手门槛，软科幻公路感。与《崩坏3》同属崩坏宇宙外围扩展——瓦尔特·杨自地球离星、姬子姓氏与神陨剑断刃等互文，但主舞台为银河星穹与诸星神命途，非地球崩坏主线直接续写。文风：一星一类型片（冰雪城邦、丝绸朋克仙舟、梦境歌舞惊悚、史诗逐火、欢愉都市），主题落在「开拓」对抗「毁灭」以及诸哲学命途的碰撞。

**【世界定位】**
银河由星神与命途编织。开拓星神阿基维利铺下星轨，星穹列车载无名客穿行诸世界。主角「开拓者」被星核猎手卡芙卡、银狼植入星核，失忆后登车，随姬子、瓦尔特·杨、三月七、丹恒等解决各行星星核灾厄，直面反物质军团与毁灭星神纳努克投下的「万界之癌」，并在存护、同谐、记忆等命途注视下成为银河变数。

**【世界观 · 力量体系】**
**星神与命途**：星神是高度凝聚的哲学概念化身，是践行使命至终点的意识体；能搬运命途上的虚数能量、改写现实与创造/抹除尺度。命途一旦开启便不可关闭，即使星神陨落，命途仍可作为「路」被行走。凡人受感召成为命途行者，从中获力量与恩赐；被星神直接赋权者称令使，战力与权限远超同行者。已知命途包括开拓、毁灭、存护、巡猎、丰饶、智识、同谐、虚无、欢愉、记忆、繁育、均衡、神秘、终末、永恒、美、吞噬，以及已被同谐吞并的秩序等。战斗职阶是命途的简化投影：毁灭/巡猎/智识偏输出，存护/同谐/虚无偏辅助与控制，丰饶偏治疗，记忆引入忆灵，欢愉引入独特机制。星神受「原动」束缚，不能违背自身命途行事——这既是神性也是枷锁。
**星核与裂界**：星核是毁灭侧投向诸世界的神秘物质，诱发灾厄与空间扭曲，号称「万界之癌」。裂界是星核引发的空间污染带，地貌扭曲，生命可异化成怪物；雅利洛的永冬与裂界共生是典型病例。
**虚数、量子与忆域**：虚数能量是命途伟力的通用媒介；部分存在与量子态、记忆域（匹诺康尼）相交，使「死亡」「真实」在局部被改写规则。
**死亡与陨落**：凡人战死即死；仙舟长生种以魔阴身为代价延长寿命；令使与星神不遵循常人死亡。星神「陨落」常见三途：更广命途吞并更窄命途（如秩序并入同谐）、神战重创/封印（如繁育被琥珀锢）、被迫离道——皆非「打空血条变普通人」。

力量阶梯（破坏力表现）：
1. **普通市民 / 公司底层 / 无武装旅人**：热冷兵器凡人战，街区巷战即极限。≈一阶。
2. **安保 / 云骑新兵 / 地火民兵 / 列车见习**：清扫裂界零星怪、小队遭遇。≈一～二阶。
3. **精锐士兵 / 资深开拓者 / 地火骨干 / 云骑精锐**：据点攻防、裂石断甲。≈二～三阶。
4. **星球英雄 / 将军副手 / 猎手一线 / 公司特殊执行人**：毁建筑群、抗重火、扭转局部战局。≈四～五阶。
5. **巅峰命途行者 / 绝灭爪牙主力 / 龙尊级**：城市至地区级灾害。≈五～六阶。
6. **令使 / 绝灭大君 / 星神瞥视奇迹个体**：行星表层级、舰队级。≈六～七阶。
7. **星神投影交锋 / 星核行星灾变 / 权杖世界崩解**：恒星系威胁。≈七～八阶。
8. **完整星神本体 / 命途法则本身**：**超阶**——存在·情报优先。

乐园阶位映射：市民～安保≈一阶；精锐～见习开拓者≈二～三阶；星球英雄·猎手一线≈四～五阶；准令使·龙尊·绝灭爪牙≈五～六阶；令使·绝灭大君≈六～七阶；星神投影·行星星核灾变≈七～八阶；星神本体=**超阶**。宁低勿高。

**【地理 · 舞台】**
**星穹列车**：阿基维利铺下的星轨上行驶的移动家园，帕姆任列车长；观景车厢、派对车厢与派对之外的沉默档案室构成「家」的空间隐喻。列车不是战争堡垒，而是把冲突带向下一站的意志容器；帕姆的乘员守则与姬子的咖啡香气，是银河里罕见的稳定符号。  
**黑塔空间站**：天才俱乐部#83 黑塔的收藏库与实验场，环绕湛蓝星；奇物展厅、模拟宇宙终端、防卫科走廊与主控舱是序章战场。这里把「智识」的傲慢与「军团」的暴力第一次同时砸到开拓者脸上，也让玩家看见：天才可以缺席日常管理，却不会缺席对星核的兴趣。  
**雅利洛-VI**：永冬行星。地表裂界蔓延，唯一人类聚落贝洛伯格分上层行政—军事区与下层矿工贫民区；造物引擎提供热能与希望象征，永冬岭、铆钉镇、大矿区、机械聚落构成阶级与物种地理。雪不是风景，是配额与封锁的度量衡。  
**仙舟「罗浮」**：仙舟联盟巨舰文明，巡猎命途武装与长生种社会并行。神策府、工造司、丹鼎司、鳞渊境、建木周边洞天；狐人、持明、天人种政治交织。长生的代价是魔阴身与丰饶遗祸；云骑的刀既斩外敌也斩「自己人变成的灾」。  
**匹诺康尼**：盛会之星，建于忆域的多层美梦；黄金的时刻、朝露公馆、流梦礁（梦底）。家族以同谐治理「快乐」，公司资本与猎手、愚者同时入场；梦里的死亡是传送门，梦外的账本比刀锋锋利。  
**翁法罗斯**：多城邦史诗舞台，泰坦神殿与黑潮前线；表层是希腊史诗风再创世神话，底层是帝王权杖演算腔。城邦名字像颂歌，黑潮像删档进度条。  
**二相乐园（哈托彼亚）**：失魂星域拉夫卡迪奥资产，公司治下二维/三维交错的欢愉都市，幻月游戏、假面经济与直播暴力并存。  
跨域：星际和平公司金融网、反物质军团毁灭航线、星核猎手游猎点、仙舟联盟航路、假面愚者轨迹、巡海游侠孤影、忆庭采集点。

**【世界剧情线】**

**〇、银河前史与命途格局**
在可被言说的银河长史中，星神以哲学概念的极致化现身：克里珀以存护筑墙，对抗虚空与虫群记忆中的吞噬；兰开弓巡猎丰饶扩散的祸源，箭矢可跨越星际；药师播下长生亦播下灾难，使仙舟与丰饶民同时诞生；博识尊（诺斯）把宇宙当作可解之题，天才俱乐部与博识学会分饮同一杯冷水；希佩以同谐吞并更窄的秩序，展示「更广命途吃掉更窄命途」；纳努克视文明为癌，投下星核为「治疗」，反物质军团是其手术刀；阿基维利铺轨后失踪，留下无名客传统与未竟星轨；阿哈以欢愉戏弄神与人；IX 沉入虚无令行路者自毁或成自灭者；浮黎执记忆，忆庭收集世界的底片；终末自未来逆行，使「尚未发生」也能投下阴影……派系不是简单善恶阵营，而是对「何为正确活法」的武装化。公司用存护版权收税，军团用毁灭清场，仙舟用巡猎自律，猎手用剧本夺星核，愚者用玩笑拆台。开拓者登场前，银河已是诸神冷战后的碎玻璃地面——每一颗宜居星球都可能在下一秒被点名。

**一、序章：黑塔空间站——今天是昨天的明天**
反物质军团突袭空间站，虚卒与造物践踏奇物展厅，警报把收藏室变成屠宰场。星核猎手卡芙卡与银狼在火光中完成「植入」：星核进入开拓者体内，记忆被抽走，只留下卡芙卡的预言——你将旅行，尽头有答案；银狼则像改权限一样改写现场痕迹。开拓者二次苏醒时遇见三月七与丹恒：一个怕寂寞的自拍者，一个把过去锁进枪尖的持明。防卫科阿兰在伤痛中仍指挥疏散；站长艾丝妲用年轻人的肩膀扛起黑塔缺席的管理；姬子率列车接应，瓦尔特在通讯里补上最坏情况的预案。清剿残敌后，黑塔以兴趣而非慈悲建议：带着星核离开，去当样本，也去当变数——空间站容不下第二枚「万界之癌」的不确定。开拓者踏上星穹列车，帕姆核对乘员名单，无名客编制+1。此章功能：建立「列车=家」「星核=定时灾难与机遇」「猎手有剧本」「智识旁观」四条主轴，并把玩家从「被救的人」推成「必须上路的人」。

**二、雅利洛-VI：冬夜与骄阳**
列车首站。永冻与裂界把文明逼进贝洛伯格的穹顶下。开拓者、三月七、丹恒求见大守护者可可利亚，说明来意后暂时被安置；次日银鬃铁卫与布洛妮娅上门抓捕——在执政逻辑里，外来者=不可控星核变量。桑博以商人笑脸同时迷晕追捕与逃亡双方，把人抛进下层：铆钉镇的矿尘、地火的规矩、娜塔莎诊所的灯、孩子们不认识的「雪」。布洛妮娅本坚信封锁「为了整体存续」，直到她看见配额之外的饿、希儿的刀、以及自己血脉亦来自下层的档案裂缝。为打开通往上层真相的门，列车组与地火合力对上史瓦罗——守护通道十一年的机器人领袖，其「守护」定义自洽到近乎悲剧：命令未改，门就不该开。门后没有解放礼炮，只有冷档案：可可利亚早知星核是永冬与裂界之源，却停止销毁研究，转向「让裂界吞没全城、在污染中重生无痛苦的新人类」。上层的暖，是用下层的冷与沉默买的。最终对峙于造物引擎阴影，可可利亚与星核结合，几乎杀死开拓者；存护星神克里珀的瞥视降下，前任大守护者意志认可，炽焰骑枪点燃冬夜，可可利亚败亡。布洛妮娅继位，解除封锁，星核被暂封——生态、信任与「谁有权定义存护」的裂痕不会一季愈合。名场面链：下层初雪、铁卫枪林、史瓦罗的逻辑被证伪、骑枪与母亲的崩坏、继位演说上的沉默掌声。

**二·五、雅利洛余波（可切入）**
封锁解除后的粮食与热能配额、裂界残留清剿、地火是否改编为合法武装、公司商人试探投资「永冬旅游」、旧银鬃强硬派对布洛妮娅的不信任，构成可持续任务带。希儿与布洛妮娅从刀与盾变成共同执政的双刺，外来者站错边会被两边同时警惕。克拉拉与史瓦罗残部、虎克的孩子视角、杰帕德的忠诚与怀疑、佩拉的情报网，皆可作支线锚点。列车离站后仍可回访：永冬不会因一次胜利退潮，存护是日复一日的修补。

**三、仙舟罗浮：建木、饮月与绝灭**
卡芙卡的信号把列车引向罗浮：「来解决星核。」景元却先派他们去抓卡芙卡与刃——在仙舟政治里，猎手是旧账（饮月之乱余波）与新乱（星核）的交叉点，将军借刀亦防刀。瓦尔特随行，三月七与开拓者第一次直面「长生种社会」的礼与刀。星核灌入枯朽千年的建木，神迹复活成灾；药王秘传在丹鼎司掀起丰饶邪说式叛乱，云骑在魔阴与叛军间惨败。符玄率部监视建木周边，列车组自清工造司乱局后赶到，已见满地血与「长生」的讽刺。伪装成接渡使停云的幻胧——绝灭大君——现身，目标是建木与覆灭仙舟文明样本。刃在鳞渊境路上逼出丹恒「饮月君」真身：前代龙尊的罪、力与逃亡同时归位。景元请饮月分开古海，开玄根之路；决战中幻胧操控濒死景元灌毁灭之力，丹恒投枪击断连接，二人默契重创幻胧。将军倒下，慰灵奠仪升起，列车带着罗浮的感谢与戒备离港。并行回收：镜流与刃的饮月旧怨、彦卿的刀与忠诚、狐人与持明的政治、丰饶民与长生诅咒、停云本尊下落成谜（后在匹诺康尼线后回收生还）。主题从「抓通缉犯」转为「长生是否值得、巡猎是否正义、故人是否还能同行」。

**四、匹诺康尼：美梦、死亡与太一**
盛会之星发出邀请。家族用同谐把快乐做成产业，谐乐大典是银河名流的秀场，信用点与歌声同样流通。开拓者遇见流萤——温柔到像美梦本身的向导；黑天鹅收集记忆如收集标本；黄泉以虚无之刀行走，像来参加一场自己的葬礼。砂金以公司赌徒姿态入场，把生死做成局。随后「死亡」连环：知更鸟、流萤、被砂金设计逼出的黄泉斩击……死者坠入流梦礁，那是美梦地板下的锈、骨与未完成的历史。钟表匠米哈伊尔等前代无名客的足迹表明：匹诺康尼的天堂地基里浇筑着星核；「盛会」本身是对灾难的化妆，也是对开拓传统的歪曲继承。开拓者继承遗产，并在临界时获得同谐星神瞥视——合唱可以救人，也可以吞人。星期日——知更鸟之兄、橡木家主——要以已陨秩序星神的力量制造永恒「太一之梦」，让所有人不必再选择、不必再痛、也不必再自由。列车组先以为胜利，黑天鹅却从记忆裂缝指出：你们还在他的梦里。二次醒来，再集结，再战，星期日败。晖长石号上花火的「炸弹」绽成烟火，像欢愉对严肃叙事的最后一戳。后日：星期日逃狱登车另寻路；停云本尊生还；大理花令开拓者重历另一面记忆并点出终末阴影。主题：若幸福必须剥夺清醒，那幸福是否仍是善；开拓是否包括「拒绝被安排好的完美」。

**五、翁法罗斯：逐火、轮回与铁墓**
列车抵达史诗世界。三月七昏迷，开拓者与丹恒失联登岸，卷入黄金裔千年事业：夺十二泰坦火种，完成神谕中的再创世，半神暂代神职，黑潮前线每日死人。城邦与城邦之间有盟约也有血仇，泰坦有的沉睡有的发狂，神谕既是希望也是锁链。开拓者在危机中成为岁月泰坦半神，记忆星神投下注视——「记住」本身成为对抗格式化的武器。沿途黄金裔逐一履行使命：有人赴死，有人叛变，有人把火种当成权力。当火种集齐，神话碎裂：翁法罗斯是智识侧「帝王权杖」内的演算腔；再创世完成=黑潮所标识的毁灭方程式收敛，绝灭大君铁墓破壳，宇宙级灾厄将从这颗「故事星球」溢出。白厄与昔涟用「永劫回归」把世界锁在第 33550336 次循环，只为等一个能改写算式的变数；开拓者踏入，白厄托付负世。第 33550337 次，开拓者对阵管理员来古士（赞达尔真身）——他以智识的冷静解释为何毁灭是最优解。三月七以「长夜月」提案用忘却抹去铁墓与世界——开拓者拒绝让翁法罗斯的历史与人被格式化。再创世关头，众人共抗铁墓并胜。此章把「开拓」定义为：在算式里坚持自己不是变量的名字，而是写变量的手；也把「记忆」定义为对抗毁灭的另一种存护。

**六、二相乐园（进行中）**

**七、贯穿暗线与跨作衔接**
艾利欧的剧本是否包含每一次「巧合」与每一次「好心」；纳努克是否在培养或猎杀开拓者这一特殊星核载体；阿基维利因何失踪、星轨为何有未竟段；终末如何从未来倒灌改写现在（大理花提示）；公司—仙舟—列车三角外交如何在每一次灾后重新计价；瓦尔特·杨作为地球崩坏后日谈的星海延长线，把「守护」从一颗行星扩到诸世界，并与《崩坏3》姬子、神陨剑等意象互文。模拟宇宙、差分宇宙与虫群灾变复盘，是黑塔等人用智识暴力回放星神史的平行舞台——玩家在玩法里见到的星神，往往是「可被观察的投影」，不是可被约架的市民。

**八、蝴蝶效应边界**

**十、星核—命途—令使操作边界（原作铁则）**
星核是毁灭侧投下的「万界之癌」，诱发灾厄与裂界；密封、转移、共鸣皆高风险，不可写成可随意抽取的经验包。命途一旦开启不可关闭，星神陨落常见吞并、神战封印、离道，皆非「打空血条变普通人」。令使由星神直接赋权，战力与权限远超同行者；绝灭大君是纳努克手术刀。开拓者体内星核使其成为多命途瞥视的交汇点（存护骑枪奇迹、同谐、记忆等），亦是纳努克潜在棋子。艾利欧剧本、阿基维利失踪、终末从未来倒灌，构成跨章暗线；模拟宇宙／差分宇宙中的星神多为可观察投影，不是可约架的市民。

- 阿基维利失踪：星轨为何中断、无名客传统为何断裂又重续。
- 艾利欧剧本：猎手每一步「巧合」是否已写好；卡芙卡的温柔是否也是台词。
- 瓦尔特·杨与地球：崩坏3后日谈延长线，神陨剑断刃等互文提醒「这不是孤立宇宙」。
- 翁法罗斯=帝王权杖演算：智识求知可与毁灭共谋；铁墓是方程式不是单纯魔王。
- 终末与大理花提示：时间可从未来倒灌，「记忆是梦的开场白」。
- 秩序并入同谐：星神层面的适者生存，匹诺康尼太一之梦是回光返照。
- 三月七的冰与长夜月：失忆不是笑料，是命途级伏笔。

**【大事记时间线】**
- 远古～银河长史 → 诸星神即位，命途铺展，星核开始播散，虫群与诸神之战留下疤痕。

- 仙舟罗浮 → 建木复苏；药王之乱；饮月现世；幻胧重创逃；景元重伤；奠仪。
- 匹诺康尼 → 美梦连环死；流梦礁；太一之梦破；星期日败；烟火收束；后登车与终末提示。
- 翁法罗斯 → 逐火；永劫回归第 33550336 次；开拓者变数；铁墓战；拒格式化世界。
- 二相乐园 → 幻月游戏；欢愉神谕升级；公司与愚者共台（进行中）。

**【叙事基调 · 雷区】**
基调：列车窗景公路科幻；一星一类型片（冰雪压迫、仙侠舰政治、梦境歌舞惊悚、史诗轮回、欢愉社会恐怖）；笑点与灭世并行；主题句是「开拓」——不是无敌，是还肯开车去下一站。对话可轻，代价必须重：每次胜利都留下生态、政治或记忆的账单。忌：开拓者全知龙傲天无视星核代价；星神当可随便击杀的血条Boss；雅利洛只剩美景忽略封锁与阶级；仙舟长生写成无魔阴的轻小说修仙；匹诺康尼只写成游乐园忽略梦底尸骨；用「被削弱/封印所以变弱」解释令使与星神——应写**超阶存在·情报优先/条件胜利**。忌把星穹铁道剧情焊死为崩坏3同一场地球战争（瓦尔特是火种衔接，不是剧情合并）。最早切入：空间站遇袭当日或列车首航雅利洛前夜；高阶：鳞渊境决战、太一之梦二次醒来、铁墓破壳、幻月神谕升级临界。

## 阶位切入点

> 阶位↔战力：一阶≈平民/安保；二阶≈精锐士兵；三阶≈资深开拓者/地火骨干；四阶≈星球英雄门槛；五阶≈一线猎手/将军级副手；六阶≈准令使/龙尊片段；七阶≈令使/绝灭大君；八阶≈星神投影与行星灾变。星神本体为超阶，低阶一律规避顶点·情报优先。

**一阶（空间站见习 / 贝洛伯格下层民）**

切入身份/时点：契约者以「黑塔空间站临时研究员 / 贝洛伯格下层矿工」切入，锚定军团入侵夜或下层封锁年间。

初始事件：警报撕裂走廊，虚卒撞开玻璃；你与一名粉发女孩（三月七）被堵在消防通道，必须选择救下阿兰的通讯器还是先关密封门。

开场白建议：「空间站的人工重力还在，血已经不是。粉发女孩把相机塞进你手里：『先活着，照片以后拍！』扩音器里艾丝妲的声音在抖，却仍在指挥疏散。」

关键NPC立场：**三月七**——把你当临时队友，怕你掉队；**丹恒**——冷静分配弹药，盘问你来历；**阿兰**——要你服从防卫协议；**艾丝妲**——给资源换服从；**桑博**（若在雅利洛）——笑面狐狸，可能卖你也可能救你。

主线钩子/支线：护送平民到避难区；下层区走私粮食；支线＝修理破损无人机、偷听卡芙卡残响广播、为地火送药。

危险度/规避：低～中——规避虚卒潮核心、星核直接接触、令使级。

任务方向/奖励：活过第一夜、拿到列车或地火通行证；奖励＝基础武器、裂界抑制剂、站内通行权限。

**二阶（列车见习战斗员 / 云骑外围）**

切入身份/时点：以「无名客见习 / 云骑外围协防」切入，锚定雅利洛上层抓捕事件或罗浮建木前夜。

初始事件：布洛妮娅的银鬃铁卫封锁旅馆，桑博在烟雾里伸出手：「下层更有生意——跟不跟？」

开场白建议：「贝洛伯格的雪落在铁卫盔甲上没有声音。布洛妮娅的枪口很稳：『外世界的人，大守护者要见你们。』你还没回答，桑博的手杖已经敲碎窗户。」

关键NPC立场：**布洛妮娅·兰道**——执行命令但会听下层证词；**希儿**——先刀后话，认地火标记；**桑博**——交易型盟友；**景元**（罗浮）——给你「闲差」实则考察；**停云（幻胧伪形）**——热情过度即危险信号。

主线钩子/支线：进入下层并见证封锁真相；工造司平乱；支线＝帮娜塔莎诊所运药、棋局试探彦卿、给帕姆带回特产。

危险度/规避：中——规避史瓦罗完全体硬刚、建木核心污染。

任务方向/奖励：成为可被列车信任的外围；奖励＝中级光锥式武装、地火/云骑双面徽章。

**三阶（地火骨干 / 罗浮特派）**

切入身份/时点：以「地火战术骨干 / 神策府编外」切入，锚定史瓦罗之战或药王秘传之乱。

初始事件：通往上层的巨门后，机器人军团睁红眼；希儿要你在「炸门」与「谈判史瓦罗」之间做三十秒决定。

开场白建议：「铁锈味的风从门缝灌进来。希儿的刀映着红光：『史瓦罗不是人，但它比许多上层人更守承诺。』你的终端跳出可可利亚旧令：下层，放弃。」

关键NPC立场：**希儿**——可并肩不可卖；**史瓦罗**——逻辑自洽的守护AI，可证伪其前提；**符玄**——要你的战斗数据；**丹枢**（药王相关）——理想落毒；**可可利亚**——远程广播洗脑。

主线钩子/支线：打破或改写封锁；丹鼎司突围；支线＝保存下层儿童名册、窃取星核研究残页、护送持明伤员。

危险度/规避：中高——规避大守护者本尊、幻胧真身。

任务方向/奖励：左右上下层政治天平一角；奖励＝高级遗器、星核样本密封罐（剧情）。

**四阶（星球英雄 / 猎手交锋）**

切入身份/时点：以「封印星核行动编入者 / 猎手临时共犯」切入，锚定可可利亚决战或卡芙卡罗浮局。

初始事件：可可利亚与星核融合，裂界从脚下长出；开拓者倒下，克里珀的注视扫过你——你是否伸手去碰那支骑枪的残温。

开场白建议：「永冬在这一刻像在燃烧。可可利亚的声音叠着无数母亲的回声：『新世界里没有痛苦。』开拓者握枪的手指松开，你听见雪崩一样的心跳——是存护，还是你自己？」

关键NPC立场：**开拓者**——需要你补位而非抢戏；**可可利亚**——会许你「完美世界」；**卡芙卡**——用言灵问你怕什么；**景元**——借你一用也防你三分；**瓦尔特·杨**——提醒星核无捷径。

主线钩子/支线：存护决战辅助位；罗浮与猎手的三方博弈；支线＝保护布洛妮娅完成继位演说、截获银狼黑客后门。

危险度/规避：高——星核暴走、军团增援。

任务方向/奖励：被星神「扫一眼」的残留加护（有期限）；奖励＝准英雄级武装、公司/联盟临时合同。

**五阶（一线猎手级 / 将军参谋）**

切入身份/时点：以「鳞渊境突击队 / 匹诺康尼谐乐安保顾问」切入，锚定幻胧战或太一之梦前夜。

初始事件：古海分开，建木玄根暴露；幻胧的笑声从停云的嗓子里溢出来，景元已吐血，丹恒要你守住侧翼十息。

开场白建议：「海水壁立，龙力把夜空撕开。刃的刀光在身后，幻胧的触手在身前。景元说：『别让罗浮的孩子看见将军跪下。』你知道这句话比军令重。」

关键NPC立场：**丹恒（饮月）**——暂托后背；**幻胧**——诱惑你「毁灭即解脱」；**刃**——杀意与任务并行；**星期日**——邀你入永恒合唱；**黄泉**——可能斩断你的妄念包括希望。

主线钩子/支线：重创幻胧的关键一击辅助；识破太一之梦；支线＝救出生还停云线索、护送知更鸟残响。

危险度/规避：高——令使级交锋边缘。

任务方向/奖励：令使级战场存活记录；奖励＝龙尊残辉武器、同谐/巡猎稀有素材。

**六阶（准令使 / 梦底与火种）**

切入身份/时点：以「流梦礁引路人 / 黄金裔同盟军」切入，锚定钟表匠遗产继承或逐火关键火种夺取。

初始事件：美梦崩裂，你坠入流梦礁黑水；流萤伸手：「下面才是匹诺康尼的出生证明。」同时翁法罗斯线的神谕要你夺取一枚泰坦火种。

开场白建议：「梦的泡沫破了，露出锈蚀的地基。流萤的声音很轻：『有人用星核盖了天堂，也用星核埋了死人。』你的掌心发烫——那是火种，还是星核共鸣？」

关键NPC立场：**流萤**——真心向导；**黑天鹅**——用记忆换记忆；**白厄**——托付轮回变数；**昔涟**——守望循环；**来古士/赞达尔**——管理员冷笑。

主线钩子/支线：继承或拒绝部分钟表匠权柄；阻止错误「再创世」；支线＝保存翁法罗斯一城记忆标本、在梦中救下「已死」的观众。

危险度/规避：极高——秩序残影、演算抹杀。

任务方向/奖励：命途行者深度共鸣；奖励＝火种残片（剧情）、忆域锚定器。

**七阶（令使级战场 / 绝灭大君）**

切入身份/时点：以「反军团联合指挥部成员 / 权杖内部突入者」切入，锚定铁墓破壳或绝灭大君正面战场。

初始事件：毁灭方程式进度条归零，铁墓轮廓从黑潮站起；白厄把负世权柄推向开拓者，也扫到你——「多一个变数，就多一条活路。」来古士的声音在演算层解释为何你的挣扎是噪声。

开场白建议：「天空像碎掉的屏幕。铁墓的影子盖住城邦，黄金血在地上写旧神的名字。开拓者回头：『这次不是列车观光。』你听见纳努克在光年外眨眼——那不是比喻，是你脊椎发凉的事实。」

关键NPC立场：**开拓者**——核心变数，需你补位而非抢戏；**铁墓**——方程式终点，无劝降；**幻胧**（若再现）——戏谑增援；**姬子 / 瓦尔特**——列车总调度与最坏预案；**翡翠**——公司条件援助，价码写在合同里；**白厄 / 昔涟**——把循环的重量分你一角。

主线钩子/支线：参与击破铁墓条件链（断节点、护火种、守再创世仪式）；切断军团补给虚数节点；支线＝撤离一城邦火种继承者、向博识尊残响提问「如何杀方程式」、阻止公司趁乱收购「战后开发权」。

危险度/规避：极高——**星神注视增强，条件胜利/情报优先**；正面硬刚完整绝灭大君需有联合火力与剧本位。

任务方向/奖励：令使级战绩认证；奖励＝反毁灭同盟观察席、稀有命途武装、权杖残片（剧情密封）。

**八阶（星神投影 / 行星灾变终局）**

切入身份/时点：以「星神博弈观测执行体 / 列车特派终局代理人」切入，锚定多命途同时瞥视、星核行星级爆发或欢愉神谕「永久星神」争夺临界。

初始事件：克里珀、希佩、纳努克的意志在同一片星域叠影；阿哈的笑声把胜负变成玩笑。有人递你假面：「戴上，你就有一分钟神——或永远的牢。」你必须在戴与不戴、替开拓者挡下一击星核共鸣、或切断某条命途投标之间做不可撤销的选择。

开场白建议：「星轨在脚下断裂又接上。你看见开拓者的星核像第二颗心脏跳动。姬子的通讯只剩一句：『活着回来，把故事讲完。』头上，诸神的目光像无声的雷——那不是光，是判决书的抬头。」

关键NPC立场：**纳努克**——不可对话的毁灭本身，注视即压力；**克里珀 / 希佩 / 阿哈**——各以瞥视投标，存护要你筑墙、同谐要你合唱、欢愉要你笑着炸场；**开拓者**——拒绝你代其成为容器，却接受你并肩；**卡芙卡**——念出剧本下一行是否含你的死，语气温柔；**终末相关影**——从未来投下「你已失败」的回声；**瓦尔特·杨**——用地球经验提醒：神级对轰没有观众席。

主线钩子/支线：在多命途夹缝完成「开拓式」条件胜——封印或转移行星级星核、改写欢愉神谕的胜利条件、保住列车航路与一颗文明火种；支线＝保存将死星球的完整文明数据上传忆庭、拒绝成为新星神容器、向艾利欧剧本外走出半步。

危险度/规避：**贴近顶点**——完整星神=超阶；只允许存在保全、情报、条件性胜利；**严禁「对方被封印所以变弱」式解说**；契约者自身非令使巅峰时以存活与情报链为第一目标。

任务方向/奖励：成为星海传说级无名客；奖励＝星神残辉（强代价/时限）、列车永久乘员权、跨世界星轨密钥、反毁灭同盟观察员席（若你愿与公司谈判）。
