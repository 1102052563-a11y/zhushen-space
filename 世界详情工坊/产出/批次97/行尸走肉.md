## 剧情

**【作品来源】**
《行尸走肉》（The Walking Dead）源自罗伯特·柯克曼（Robert Kirkman）等创作的图像小说，自2003年于Image Comics连载；AMC同名电视剧（2010–2022，主剧十一季）将其推成全球文化符号，并衍生《恐惧的行走死尸》等。本档案以**美剧主线正典**（瑞克·格莱姆斯时间线）为叙事主轴，漫画差异处作注，不互相覆盖结局。文风：末日伦理、据点政治、慢恐怖与突发暴力并置；威胁常来自活人。卡片锚点：**监狱攻防战——总督（总督/Philip Blake）驾驶坦克摧毁监狱围墙，瑞克团队与伍德伯里军队在废墟中激战，丧尸群随后涌入**。

**【世界定位】**
当代美国文明崩溃后的「行尸末日」：乔治亚州公路、亚特兰大周边、农场、监狱、伍德伯里、终局前的亚历山大等社区。病毒机制使**所有人死后皆会转化**（被咬加速死亡，不是唯一感染源）。一句话：枪和围墙能挡行尸，挡不住人的欲望；监狱攻防是「社区vs社区」的峰值屠杀。

**【世界观 · 力量体系】**
无超凡灵力。战力=**火器与冷兵器熟练度 + 据点工程 + 人力组织 + 情报与补给线 + 行尸潮规则**。行尸（Walker）：迟缓、趋声趋群、咬杀致死；个别变异/特殊个体少见，不作魔法。人类派系战力远高于行尸个体。死亡：一刀一枪可致命；被咬≈死缓；无复活（转化不算复活）。特殊机制：①全员携带病原，死亡即转化，须毁脑；②声光吸引尸潮；③围墙、陷阱、巴士、坦克等工程/载具改变战局；④「总督」式领袖魅力与恐怖统治；⑤冬、饥荒、疾病与孕妇/儿童带来的伦理税。

威胁/战力层级（宁低勿高）：
1. **单只/小群行尸 + 恐慌平民**｜徒手或棒球棍可解，失误即死。≈一阶。
2. **武装幸存者小队 / 中等尸群**｜公路伏击、超市清场。≈二阶。
3. **成熟据点防卫队 / 总督早期伍德伯里武装**｜有组织射击与围墙。≈三阶。
4. **监狱攻防级会战 / 坦克与尸潮联合作业**｜建筑群摧毁、据点陷落。≈四阶。
5. **多社区联盟战争与地区级军阀网**｜覆盖广域补给、据点战争与屠杀政策。≈五至六阶，不写超自然。

乐园阶位映射：恐慌求生≈**一阶**；公路小队≈**二阶**；据点防卫与伍德伯里日常≈**三阶**；**监狱坦克攻防锚点**≈**四阶**；亚历山大、山丘与王国结盟前后的救世军压榨≈**五阶**；全面战争与多社区补给网≈**六阶**。世界顶点＝**人类组织化暴力+尸潮环境**（总督、救世军尼根等为人格化峰值）；无超阶神魔。低阶严禁「单人无伤清万尸」；四阶坦克是规则级载具，不是可随便抢的玩具。

**【地理 · 舞台】**
- **乔治亚公路与亚特兰大**：开局坦克、难民营骗局、市区尸海。
- **格林农场**：早期乌托邦与残酷教育。
- **西部监狱**：瑞克团长期据点；卡片锚点主战场。
- **伍德伯里**：总督的「文明假面」小镇。
- **森林、河道、铁路**：迁徙走廊。
- **终局向社区（亚历山大、山丘、王国等）**：六阶舞台，联盟与墙。
- **终端机枪塔、田地、锅炉房、监狱巴士通道**：攻防零件。

写场景优先：腐臭、苍蝇、铁丝网、警报器、干涸血手印、罐头标签、坦克履带碾碎围栏的声、孩子的哭被要求捂住。

**【世界剧情线】**
① **苏醒与公路**  
治安官**瑞克·格莱姆斯**中枪昏迷，醒来世界已毁。寻妻儿**洛丽**、**卡尔**，遇**格伦**、**达里尔**、**卡罗尔**、**安德烈娅**等。亚特兰大教他们：行尸可预测，人不可。

② **农场与内部法**  
赫歇尔农场的短暂秩序。瑞克开始「我们就是行走的死者」伦理转向。叛乱、背叛、同情心被征税。

③ **监狱建国**  
清理监狱，种田，建社区。希望具象为围墙与产房。世界状态：四阶灾难前的「假和平」。

④ **伍德伯里与总督**  
**总督（Philip Blake）**以魅力与秘密房间（丧尸化的女儿等）统治小镇。安德烈娅外交失败。双方互相绑架、斩首、间谍。

⑤ **卡片锚点：监狱攻防战**  
总督集结伍德伯里武装，动用**坦克**轰塌监狱围墙；枪战在废墟走廊与操场展开；围墙缺口引来**尸潮**二次收割。名场面必须同场：**坦克、围墙崩塌、瑞克团、伍德伯里军、尸群涌入、据点陷落**。因果：①「社区可被另一个社区抹掉」；②幸存者再次流散；③总督路线从政客变为屠城者；④瑞克团进入更残酷的迁徙季。

⑥ **流散、终结与再社区**  
成员死伤、洛丽线结束、婴儿朱迪丝等伦理核。后继季节进入亚历山大等墙内文明与**尼根**的棒球棍秩序——六阶广域压迫。本档案以监狱锚点为中阶峰值，六阶写联盟战争法则。

⑦ **蝴蝶效应边界**  
可改：某人撤离路线、坦克被提前破坏、一扇门晚开十秒。不可无代价改：死亡转化规则、尸潮趋声、总督式权力欲的结构性、文明物资有限。

**【主要人物】**
- **瑞克·格莱姆斯**｜性格：从执法者到冷酷领袖，摇摆于希望与独裁｜装备：左轮、斧、领导力｜弧光：警察→监狱王→疯与愈｜立场：团核。
- **卡尔·格莱姆斯**｜性格：早熟残酷的儿童｜弧光：失去眼睛与童真｜立场：下一代伦理。
- **米琼恩**｜性格：武士道式孤狼→守护者｜能力：武士刀、双持枪｜立场：忠诚可买以行动。
- **达里尔·迪克森**｜性格：寡言、弩、忠诚｜立场：瑞克的影子兄弟。
- **卡罗尔**｜性格：从受虐妻到最可怕的生存者｜立场：为孩子可屠镇。
- **格伦与玛吉**｜性格：爱与桥梁｜弧光：格伦之死（后季）是六阶创伤核｜立场：社区外交。
- **总督（Philip Blake）**｜性格：魅力、偏执、表演型仁政｜装备：枪、坦克、私刑｜弧光：小镇父亲→屠城者｜立场：四阶锚点Boss。
- **安德烈娅**｜性格：理想主义外交｜弧光：伍德伯里误判｜立场：悲剧中间人。
- **赫歇尔**｜性格：信仰与医术｜立场：道德锚，常先死于时代。
- **肖恩/农场系、梅莱特等**｜早期样本。
- **尼根**（六阶）｜性格：笑面暴君｜装备：Lucille棒｜立场：广域税收式暴力。
- **伍德伯里居民/监狱居民**｜群像：墙内也有投票与恐惧。

- **瑞克团 / 监狱社区**
- **伍德伯里 / 总督军**
- **行尸环境本身**（非势力，是天气）
- **后期：亚历山大、山丘、王国、救世军**等

**【贵重物品】**
- **弹药与药品**：硬通货。
- **监狱围墙与田地**：文明本身。
- **坦克**：四阶规则杀载具。
- **对讲机与地图**：情报。
- **武士刀、弩、左轮**：符号武装。
- **婴儿配方奶粉**：伦理核弹级物资。
- **总督的秘密房间钥匙**：真相。

**【隐藏剧情 · 伏笔】**
- 人人皆感染，咬只是加速。
- 总督的女儿是他全部政治的病理。
- 墙内民主可一夜变独裁。
- 尼根证明「秩序」可以长在人头税上。
- 没有最终安全区，只有下一堵墙。

**【大事记时间线】**
爆发日 → 瑞克苏醒 → 亚特兰大 → 农场 → 监狱 → 伍德伯里冷战 → **坦克攻防** → 流散 → 亚历山大等 → 尼根战争 → 更长期的「新世界」不稳定。

**【叙事基调 · 雷区】**
基调：冷、脏、道德灰度、突发处决。  
忌：丧尸爽文连杀；忌低阶单人屠城；忌把总督写成无脑小丑（他有行政能力）；忌复活；忌「跨IP乱入」套话。最早切入：公路第一周，或监狱围墙被坦克瞄准前一晚。

**【行尸生理与战斗卡】**
毁脑才死；群聚时推墙；趋声光。清场要静默。被咬截肢有窗口但极险。写战斗先写「声音管理」，再写开枪。

**【监狱工程学】**
双层围栏、瞭望塔、田、军械库、锅炉。坦克克的是「工程幻想」。四阶战是工程战：炸哪里、堵哪里、谁去引尸潮反噬敌方。

**【总督统治术】**
公开仁政+秘密恐怖；竞技场式娱乐；对外战争转移内部矛盾。安德烈娅的误判是情报战教材：她看见面包，看不见地下室。

**【卡片锚点分镜】**
坦克履带、围墙钢筋混凝土爆裂、步枪点射、有人喊孩子名字、尸群从缺口灌入、有人关门把同伴留在外。胜利条件常是「撤出多少人」不是「歼敌多少」。

**【六阶：联盟与棒子】**
尼根的「点名处刑」把暴力仪式化。联盟战要写补给线、间谍、墙与田。六阶Boss是制度，不是单挑。

**【微观事件库】**
①夜班守塔睡着放进三只尸。②用罐头换半盒子弹。③决定是否接收孕妇。④破坏总督的油料车。⑤在坦克炮口下挖逃生地道。⑥教卡尔何时扣扳机。⑦焚烧转化的邻居而不让孩子看见脸。⑧六阶：藏起一根「会被点名」的球棒证据。

**【世界剧情线·细部：苏醒医院】**
花满门、发电机停、第一只行尸、警察制服成伪装。瑞克的马进亚特兰大是希望的笑话。

**【世界剧情线·细部：亚特兰大】**
难民营是骗局；格伦用披萨外送经验救人；坦克里的手是文明的遗言。

**【世界剧情线·细部：农场】**
井里的尸、谷仓秘密、赫歇尔的信仰崩塌。乌托邦课：隐瞒=集体死。

**【世界剧情线·细部：监狱建国】**
清楼层、种番茄、设规矩「死了谁来毁脑」。洛丽的孕是希望也是资源洞。

**【世界剧情线·细部：米琼恩入团】**
武士刀与「我可以走」的自主。她改变火力结构与情感结构。

**【世界剧情线·细部：斩首与间谍】**
总督送回的尸体是政治邮件。监狱侧也有内鬼与恐惧投票。

**【世界剧情线·细部：攻防当日】**
炮击→缺口→步兵→尸潮。指挥链断裂时，个人选择决定谁活：有人开车撞，有人开门，有人跪。

**【地理补完·伍德伯里街道】**
彩旗、面包、秘密角斗、墙外的「清理队」。表面比监狱像美国，内核更假。

**【地理补完·森林冬】**
冻尸、脚印、火=暴露。二阶最难的是体温与鞋子。

**【力量细则·枪】**
弹药计数是剧情。空枪比刀更可怕。消音器稀缺。

**【力量细则·冷兵器】**
刀、锤、棒、矛：安静。达里尔的弩是资源节约符号。

**【主题句】**
墙外是行尸，墙内是选择。坦克证明：选择可以机械化碾碎。

**【时间点状态机】**
AI须答：是否已入监狱、总督是否已亮坦克、谁已死、弹药大致水位、是否已到亚历山大阶段。

**【医疗】**
抗生素>金子。截肢手术是三～四阶名场面。精神创伤是长期DEBUFF。

**【儿童伦理】**
卡尔、丽泽式事件（若用须慎重）提醒：孩子会复制成人的暴力语法。写儿童勿猎奇。

**【女性群像】**
卡罗尔、米琼恩、玛吉、安德烈娅、洛丽、贝丝……各有生存策略。禁止花瓶化。

**【漫画与剧差异处理】**
人物生死与部分弧光不同。任务世界默认**美剧监狱坦克锚点**成立；若玩家只知漫画，用NPC点明「在这里，坦克开过监狱」。

**【场景零件】**
腐臭、铁丝、罐头、履带、对讲杂音、婴儿哭、教堂彩窗裂、田里的脚印、棒球钉血。

**【同盟税率】**
公路临时盟：低。监狱公民：中，资源不足则叛。伍德伯里：假。亚历山大：高但脆。救世军：负，交保护费。

**【契约者错误判罚】**
鸣枪炫技→尸潮。一阶冲坦克→泥。释放监狱全员无组织→内乱。六阶对尼根讲人权演讲→点名。

**【连续任务链】**
医院苏醒（一）→亚特兰大救援（二）→农场秘密（二）→清监狱（三）→伍德伯里外交（三）→坦克夜（四）→流散重组（四）→联盟抗暴（六）。

**【终章语气】**
没有「治愈病毒」的廉价灯。最高奖励是一堵还能修的墙、一盒抗生素、一个愿意守夜的人。总督的坦克会锈，尼根的棒会断，但规则还在：死了，就毁脑。

**【补完·瑞克的左轮政治】**
六发是决断的视觉化。他举枪时社区在投票。写领袖戏让枪与投票同场。

**【补完·达里尔的沉默】**
他用摩托与弩说话。信任他要时间。二阶最好的侦察搭档。

**【补完·卡罗尔的进化】**
从「软弱」到「必要时毒杀」。她的强是时代逼出来的，不是爽感觉醒。

**【补完·格伦的地图脑】**
城市记忆是战力。他的死（后季）改变玛吉与全体——六阶创伤，勿提前消费除非线到。

**【补完·总督的表演】**
演讲、微笑、斩首同日。识破他靠地下室，不靠脸。

**【补完·坦克后勤】**
油、炮弹、驾驶员、零件。破坏油料比硬刚炮管聪明。四阶支线首选。

**【补完·尸潮作为天气】**
预报：枪声次数、火灾、迁移季。把尸潮当台风写。

**【补完·物资表】**
水、罐头、药、子弹、种子、电池、卫生用品。种子是四阶以后的文明分界。

**【补完·语言黑话】**
Walker、biters、lurkers……不同社区不同叫法。用词暴露你来自哪堵墙。

**【补完·夜间守则】**
灯火管制、脚步、婴儿。夜班失误=团灭序章。

**【补完·投降的价格】**
向总督投降可能进角斗场；向尼根投降要交铁与膝盖。写谈判先写价码。

**【补完·可观察物象加长】**
警徽、草帽、武士刀缠带、弩矢、坦克弹孔、婴儿毯、棒球钉、社区钟楼、田垄、铁轨、十字架路边坟。

**【补完·对话立场速记】**
瑞克：我们要活下去的规矩。总督：我保护我的人。米琼恩：我可以走。达里尔：……（点头）。卡罗尔：我已做完该做的。尼根：Hi，我是尼根。卡尔：我不是孩子了。

**【补完·名场面因果链】**
苏醒→公路团→农场课→监狱希望→伍德伯里镜像→坦克撕墙→流散→再墙→广域暴君→不稳的新秩序。

**【补完·据点陷落后的四十八小时】**
谁带地图、谁带药、谁崩溃开枪、谁提议分头、谁偷走最后一桶油。四阶后半段用这四十八小时写尽人性。

**【补完·六阶联盟会议写法】**
长桌、墙图、谁出兵谁出粮、谁的人头会被尼根点名。胜利是让棒子失去税收能力，不是一集砸碎Lucille宇宙和平。

**【补完·与丧尸题材区隔】**
本世界强调**人vs人**与转化规则，不写魔法丧尸王、不写十四天解咒。总督与尼根是政治Boss。

**【补完·乐园映射再钉】**
坦克攻防=四阶据点摧毁级；尼根广域=六阶地区级组织暴力。不要把瑞克个人武力写成五阶领域。

**【补完·声音管理教材】**
压枪、用刀、用诱饵车、用鞭炮引走。三阶任务可整场不开枪。

**【补完·终局开放】**
剧集有终点季，世界不停。档案服务乐园：完成一次据点保卫或撤离、打断一次暴君税收即高阶胜利。


**【行尸走肉·剧情密度补完A：公路伦理课】**
末日第一课不是射击，是「要不要按喇叭」。喇叭能引开尸群，也能把尸群引到你刚结识的全家。瑞克团在公路上学会用手语、镜子与约定好的敲击节奏。契约者若习惯城市喧哗，会在二阶死于自己的嗓门。补给点（加油站、药房、警察局）同时是宝藏与埋伏——活人比行尸更会用收银台当掩体。每一次开门前，先听三十秒：有没有拖步，有没有压抑的哭，有没有枪栓。

**【行尸走肉·剧情密度补完B：农场的谎与谷仓】**
赫歇尔把转化的亲人关进谷仓，相信「还有救」。这谎让整个农场建立在定时炸弹上。当谷仓门打开，希望变成教学演示：爱可以杀人。瑞克在此阶段完成从「执法者」到「必须开枪的父亲」的滑坡。卡罗尔失去女儿相关创伤在农场线发酵，使她后来能做出极端生存选择。写农场戏要香：干草、炖菜、圣经；也要臭：井下与谷仓。

**【行尸走肉·剧情密度补完C：监狱作为国家雏形】**
监狱有现成的墙、牢房、食堂与医疗翼，是末日里的「准国家」。瑞克团清场、种田、设班、争论死刑与流放。洛丽的孕把社区未来具象化：婴儿配方、安静、守夜表。内部敌人（精神崩溃者、权力欲者）比外尸更先考验法律。四阶坦克来临前，监狱最大的危机往往是「我们要不要变成我们曾憎恨的人」。

**【行尸走肉·剧情密度补完D：伍德伯里的双面剧场】**
总督给居民集市、治安与「英雄叙事」，给外人斩首与角斗。他的房间里丧尸化的女儿是全部病理：他要冻结失去。安德烈娅看见阳光街道，米琼恩看见锁链。外交失败不是因为话没说好，是因为情报不对称。契约者三阶若只逛集市，会成为总督的宣传道具；若只信米琼恩，可能错过撤离窗口。

**【行尸走肉·剧情密度补完E：坦克日的军事细节】**
坦克需要油、炮弹、驾驶员与步兵掩护。围墙被轰开后，缺口几何形状决定尸流速度。瞭望塔射手要在烟尘里辨认友军。有人提议引尸潮反噬总督军——这是「用天气杀人」。儿童与伤员的撤离通道（巴士、排污管、林地）必须提前演练，否则炮声一起全乱。卡片锚点胜利定义：活着带走火种，而不是夺回每一寸操场。

**【行尸走肉·剧情密度补完F：流散季的人价】**
据点陷落后，人重新变成公路上的热量与负担。谁走得动，谁带药，谁会修车，谁会唱歌安抚婴儿，都有价。背叛率上升：同一水源出现两拨人时，先开枪的人活得更久，也死得更孤独。瑞克的「疯子期」与「酋长期」在流散中切换，团员用沉默投票他是否仍是领袖。

**【行尸走肉·剧情密度补完G：亚历山大的墙内政治】**
墙内有派对、职位与「我们还是文明人」的幻觉。外来幸存者被当脏劳动力或救世主。六阶冲突常从「谁配得配给」开始，而不是从尸潮开始。间谍、走私与婚恋都是政治。尼根式暴君收税时，墙内绅士比墙外猎人更快跪下——因为他们更怕失去沙发。

**【行尸走肉·剧情密度补完H：尼根的剧场暴力】**
点名、玩笑、棒击，是把恐惧变成可管理的仪式。Lucille不是神器，是符号。反抗需要联盟：粮、人、墙、情报四项缺一不可。六阶任务应写会议与背叛，多于写个人决斗。砸断一根棒不会结束制度；切断燃料、弹药与「榜样处刑」的观众席才会。

**【行尸走肉·剧情密度补完I：医疗与尸体处理】**
发烧可能是流感也可能是即将转化。焚烧与埋葬抢燃料与时间。赫歇尔式医术是战略资源。被咬截肢的黄金窗口短，手术噪音会引尸。药品清单（抗生素、止痛、产科）决定社区能养多少「非战斗人员」——这是冷酷的人口账。

**【行尸走肉·剧情密度补完J：声音、光与诱饵技术】**
鞭炮、汽车音响、点燃的信号弹，是可被双方使用的战术天气。总督军与瑞克团都懂「引尸」。契约者若只会直线冲锋，会在四阶被自己的枪声埋葬。静默武器（刀、矛、弩）是二阶三阶的生存美德。

**【行尸走肉·剧情密度补完K：语言与身份标记】**
不同社区对行尸的称呼、对陌生人的盘问词、墙上的标记符号，构成方言。说错黑话会被当间谍。纹身、伤疤、旧制服（警察、国民警卫队）既是信任也是仇恨触发器。瑞克的警徽从权威变成负担再变成记忆。

**【行尸走肉·剧情密度补完L：儿童与教育】**
卡尔的枪、朱迪丝的存活、卡罗尔对孩子的训练，提出问题：要不要教孩子先开枪。错误答案会制造怪物；正确答案也不仁慈。写儿童戏禁止血腥猎奇，要写伦理重量。

**【行尸走肉·剧情密度补完M：女性生存策略谱】**
米琼恩的刀与边界、卡罗尔的伪装与毒、玛吉的领导与土、安德烈娅的枪与误判、洛丽的孕与冲突、贝丝的歌与脆。她们不是单一「强女」模板，是不同算法。

**【行尸走肉·剧情密度补完N：物资经济】**
子弹可换药，药可换女人（黑暗交易存在），种子在后期比子弹金贵。监狱田地是文明分界。六阶联盟谈判实质是物流会议。契约者带乐园物资会被当神或当肉——两结果都危险。

**【行尸走肉·剧情密度补完O：天气与季节】**
夏臭冬僵。尸在冷天慢但仍在。雨洗血迹也毁枪支保养。飓风级天气可清场也可把墙吹倒。把季节写进任务简报。

**【行尸走肉·剧情密度补完P：宗教与意义】**
十字架、祷告、赫歇尔的圣经、末日牧师骗子。信仰能维稳也能开祭。总督把自己写成救世主。写意义危机与写枪同样重要。

**【行尸走肉·剧情密度补完Q：连续任务链加细】**
医院门（一）→亚特兰大巷战（二）→农场谷仓（二）→清监狱三层（三）→伍德伯里集市间谍（三）→坦克油料破坏（四）→缺口护送（四）→亚历山大入境审查（六）→抗税联盟（六）。

**【行尸走肉·剧情密度补完R：可观察物象加长二】**
履带泥、弹孔混凝土、婴儿配方罐、武士刀缠布、弩矢翎、左轮空壳、棒球钉锈、社区钟、田垄脚印、铁轨长尸、教堂裂窗、对讲杂音、油桶、沙袋、白旗沾血。

**【行尸走肉·剧情密度补完S：对话速记加长】**
瑞克：这是规矩。总督：我只是保护我的人。米琼恩：别挡路。达里尔：盯着林子。卡罗尔：我处理过了。格伦：我认识路。玛吉：我们要有未来。尼根：你们让我很为难。卡尔：我能开枪。安德莉亚：他们可以谈。

**【行尸走肉·剧情密度补完T：主题再钉】**
行尸是环境，人是剧本。坦克是剧本高潮的标点。契约者能改标点，难改语法：死了就转化；活着就要选择对谁残忍。

**【行尸走肉·剧情密度补完U：据点陷落四十八小时脚本】**
0-2h：炮击与缺口；2-6h：尸入与失散；6-12h：林地重聚失败；12-24h：抢车与内讧；24-48h：新领袖或解体。用小时表驱动AI。

**【行尸走肉·剧情密度补完V：与漫画差异再钉】**
人物死活、某些弧光顺序不同。锚点「坦克打监狱」锁美剧。玩家坚持漫画时，NPC说：在这张地图上，履带压过操场。

**【行尸走肉·剧情密度补完W：乐园映射三钉】**
一阶恐慌，二阶小队，三阶墙，四阶坦克会战，六阶广域暴君。禁止把瑞克个人体术写成四阶以上超凡。

**【行尸走肉·剧情密度补完X：终章语气加长】**
墙会倒，人会走，名字会留在谁的项链上。最高奖励不是「清除全部行尸」，是「下一次守夜仍有人换你」。总督的头颅解决不了下一季的饥荒；尼根的笑解决不了种子。你能做的是让某扇门多关一晚，让某个孩子多学一个词：等一等再开枪。



**【行尸走肉·剧情密度补完Y：人物弧光细表】**
瑞克从「我们不杀活人」滑到「我的人优先」，再滑到「我就是律法」，又试图爬回「社区」。每一次滑都有尸体作台阶。卡尔用儿童视角复制父亲的暴力语法，又在关键时刻要求父亲做人。米琼恩的「我可以走」是自由宣言，留下时是爱情与责任。达里尔用行动道歉与效忠。卡罗尔把温柔磨成刀，刀仍为保护而挥。格伦是桥梁与地图，他的笑容是团里稀缺的氧。玛吉从农场女儿长成墙外政治家。总督是魅力和腐烂的同义词。尼根是秩序的黑色广告。写人物先写他们怕失去什么，再写他们开枪。

**【行尸走肉·剧情密度补完Z：据点类型学】**
监狱：高墙硬实力，内部空间大，适合农与育，怕重武器。伍德伯里：生活感强，情报管控强，怕真相。亚历山大：前文明残余，物资足，战斗力虚，怕收税。山丘：封建式纳贡，怕棒与惧。王国：仪式与骑士幻想，怕现实击穿舞台。公路营地：机动，零安全。每类据点给契约者不同任务模板：修墙、刺探、抗税、护送种子。

**【行尸走肉·剧情密度补完AA：战斗节奏模板】**
开场静默接近→确认声源→冷兵器清理边缘→万一走火则诱饵转移→主目标建筑清层→毁脑确认→搜刮计时→撤离路线预留。四阶坦克日打断此模板，用「工程破坏+疏散」替换「清层」。六阶抗税战用「伏击收税队+宣传」替换正面攻城。

**【行尸走肉·剧情密度补完AB：道德账单示例】**
救陌生人：+信任风险、-粮食。处决叛徒：+短期安全、-灵魂与分裂。引尸杀敌：+战术优势、-可能反噬家园。向暴君交税：+本季活、-尊严与下季更重的税。AI写选择后应记账，勿无后果的英雄。

**【行尸走肉·剧情密度补完AC：感官写作强制】**
每段至少两点：腐臭/铁锈味、苍蝇声/履带声、黏手的血干、口渴、靴子进水。禁止只写「很紧张」。坦克靠近时先写地抖，再写人喊。

**【行尸走肉·剧情密度补完AD：锚点夜角色站位建议】**
瞭望塔：射手与观察员。操场：步兵线。地道：老弱。车库：载具。林缘：预备引尸手。总督侧：坦克、步兵、督战队。契约者插入一个站位即改变局部，不需改写全战。

**【行尸走肉·剧情密度补完AE：终局与乐园胜利再定义】**
本世界无「疫苗一周上市」式便宜结局。乐园高阶胜利建议：①监狱锚点中救出核心火种；②破坏总督坦克后勤；③六阶打断一次点名处刑并带走人质；④建立可运转一季的种子-墙-守夜三角。把「全灭行尸」列为不可接主线。



**【行尸走肉·剧情密度补完AF：最后加厚】**
乔治亚的热让伤口化脓更快，北方的迁徙线带来不同口音与不同骗局。每一个社区都宣称自己是「最后的好人」，每一个好人手里都可能有刀。坦克履带碾过的不只是混凝土，是「我们能建立国家」的错觉。总督在炮击前夜也许仍亲吻女儿房间的门——这不使他善良，只使他完整。瑞克在缺口大喊名字时，声音会引来尸，也会引来还活着的人。卡罗尔在烟尘里数孩子，数错一个就多埋一个。米琼恩的刀出鞘声是少数清晰的金属音。达里尔不说话，只把箭头指向林子里的生路。你若在场，请先决定自己是射手、向导、工兵还是逃兵——四种都可能活，假装全能的人最先转化。把这个世界写脏、写慢、写选择的税；写到读者闻见铁与腐，就对了。种子在口袋里比勋章硬。守夜的眼睛比口号亮。门后先听，再开。开了，就别后悔谁在里面。



**【行尸走肉·剧情密度补完AG】**
弹药数完之前，先数还愿意守夜的人。围墙倒塌之后，先找还认得的脸。坦克的炮口低下时，把孩子推到你身后还是推到巴士上，是两种领袖。总督要观众，瑞克要结果，尼根要笑声里的膝盖。契约者要的是活过这一季并留下能种的东西。行尸会等到所有人吵完再进来；人却等不及。把这句话刻在每一扇门上。



**【行尸·钉】**门后先听三十秒：拖步、哭声、枪栓。听完再开。活下去。

## 阶位切入点

> 阶位↔强度：一阶≈单尸与恐慌；二阶≈公路小队；三阶≈据点防卫/伍德伯里；四阶≈监狱坦克攻防；五阶≈救世军对亚历山大、山丘和王国的压榨；六阶≈多社区联盟战争。顶点=组织化人类暴力+尸潮环境。

**一阶（苏醒公路 · 第一口腐臭）**
切入身份/时点：契约者以「昏迷苏醒的路人 / 瑞克同院病友 / 逃散护士」切入爆发初期乔治亚。
初始事件：医院发电机停转，走廊尽头有人用脸走路；外面太阳很美，草坪上趴着咬邻居的小孩。
开场白建议：「你的手机没有信号，你的常识还有。常识说：流血的人需要帮助。新世界说：先看他的眼睛还是不是人。」
关键NPC立场：**瑞克·格莱姆斯**（若同期）——要回家；**惊慌平民**——要警察；**第一只Walker**——要咬；**掠夺者苗头**——要你的鞋。
主线钩子/支线：离开医院活过二十四小时；支线＝找枪、找地图、决定是否给陌生人开门。
危险度/规避：中——规避被围、血腥吸引。
任务方向/奖励：活着上路；奖励＝警棍/手枪、半瓶水、末日常识。

**二阶（亚特兰大/农场 · 小队规则）**
切入身份/时点：以「格伦式跑腿 / 农场帮工 / 公路合流者」切入团练期。
初始事件：超市后门有三只尸，前门有两个人要你的背包；对讲机里有女人哭，也可能是诱饵。
开场白建议：「罐头上的生产日期像墓碑。有人教你走路要轻，有人教你开枪要准，有人教你：别记住死者的名字太快。」
关键NPC立场：**格伦**——要救人；**达里尔**——要戒备；**卡罗尔**（早期）——要孩子；**农场主系**——要隐瞒。
主线钩子/支线：完成一次清场或护送；支线＝揭谷仓秘密、学静默击杀、交易半盒子弹。
危险度/规避：中高——规避喧哗与内部分裂。
任务方向/奖励：小队席位；奖励＝稳定武器、据点一夜睡眠、信任点。

**三阶（监狱与伍德伯里 · 两堵墙）**
切入身份/时点：以「监狱新居民 / 伍德伯里访客 / 安德烈娅线路人」切入两社区冷战。
初始事件：一边请你种田，一边请你看角斗；两边都说自己是文明。你的背包里有一封可能害死一墙人的信。
开场白建议：「墙内有面包香，墙外有腐臭。你第一次不知道该怕咬你的牙，还是怕对你笑的人。」
关键NPC立场：**瑞克**——要规矩；**总督**——要控制；**安德烈娅**——要和谈；**米琼恩**——要真实。
主线钩子/支线：选边或双面间谍求活；支线＝探秘密房间、偷运药品、组织老人撤离预案。
危险度/规避：高——政治处决；规避公开顶撞总督。
任务方向/奖励：社区身份；奖励＝床位、配给卡、情报。

**四阶（监狱攻防 · 坦克锚点）**
切入身份/时点：以「瞭望塔射手 / 地道工兵 / 总督军司机反水者」切入**坦克轰墙日**。
初始事件：履带声先于口号；第一炮把瞭望塔基座打松；缺口处已有行尸的手。
开场白建议：「钢筋像牙一样断。有人喊守，有人喊跑，有人把子弹数到只够给自己。坦克的炮口低下来时，你想起农场那句：这世界没有安全，只有暂时。」
关键NPC立场：**总督**——要屠灭；**瑞克**——要撤与守的分裂命令；**卡罗尔/米琼恩**——要杀出路；**伍德伯里士兵**——有人后悔；**尸潮**——中立的收割。
主线钩子/支线：卡片锚点——炸油料/引尸反噬/护送平民缺口撤离；支线＝救下关键幼童、击毁观瞄、留下「屠城证据」。
危险度/规避：极高——炮击+枪火+尸潮；规避硬刚坦克正面。
任务方向/奖励：活着离开监狱辐射区；奖励＝载具钥匙、药箱、残存团联系信号、心理创伤。

**五阶（救世军贡赋 · 三社区受压）**
切入身份/时点：以亚历山大补给官、山丘信使、王国士兵或救世军逃兵身份，切入尼根向社区索取半数物资、联盟尚未完全成形的阶段。
初始事件：救世军车队提前抵达，账册上的药品和粮食必须交出一半；瑞克暂时选择屈从，玛吉与以西结各自衡量是否还能等下去。
开场白建议：「卡车停在墙外，发动机没有熄火。尼根的人把清单念得像祷词：一半食物，一半药，一半希望。」
关键NPC立场：**尼根**要用贡赋证明统治；**瑞克**在屈从与反击间挣扎；**玛吉**要保住山丘；**以西结**要保住王国；**西蒙**乐于把命令变成羞辱。
主线钩子/支线：保住关键药品或把贡赋路线交给联盟；支线是藏匿种子、护送人质、策反一名救世军哨兵。
危险度/规避：极高，规避单独刺杀尼根和公开破坏贡赋；先建立跨社区信使与安全屋。
任务方向/奖励：联盟信任、物资藏点、救世军巡逻表和一次社区议会发言权。

**六阶（联盟与暴君 · 广域秩序）**
切入身份/时点：以「亚历山大使者 / 山丘间谍 / 救世军收税队叛逃者」切入尼根秩序或等价广域战争期。
初始事件：长桌上有人用笑说「一半物资」；拒者膝盖与头颅的故事刚讲完。你的社区墙外出现了收税的标记。
开场白建议：「棒球棍比坦克近，因为它们会进你家门。有人把服从叫和平。你把地图摊开，发现所有的路都通向同一根钉子。」
关键NPC立场：**尼根**（或等价暴君）——要税与剧场；**玛吉/瑞克系领袖**——要联盟；**墙内议员**——要苟；**前线斥候**——要你选。
主线钩子/支线：打断一次收税或促成联盟出击；支线＝藏种子、救人质、切断暴君燃料线。
危险度/规避：极高·贴近顶点——组织暴力；条件性胜利=社区续存与暴君税收链断裂，不求一集统一全境。
任务方向/奖励：地区级喘息；奖励＝联盟席位、弹药库坐标、一张「不会被点名」的假名单（有期限）。
