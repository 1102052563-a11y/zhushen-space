# 移动迷宫

## 剧情

**【作品来源】**
《移动迷宫》（The Maze Runner）为美国作家詹姆斯·达什纳（James Dashner）所著青年反乌托邦系列第一部，2009年由Delacorte Press出版；后续有《灼热考验》《死亡解药》及前传等。2014年电影《移动迷宫》及续集为衍生影像。本档案以**小说第一部主线正典**为准（电影迷宫结构与部分人物命运有异，不替代书中因果）。文风：封闭实验、少年自治、记忆剥夺、生物机械恐怖。卡片锚点：**鬼火兽之夜——迷宫大门未关闭，鬼火兽（Grievers）入侵林间空地，少年们手持木矛对抗半机械怪物，死伤惨重**。

**【世界定位】**
末日后「火疫/耀斑」摧残的地球上，组织**WICKED**（World In Catastrophe: Killzone Experiment Department，灾难世界：杀戮区实验部门）以「寻找治愈」为名，把记忆清除的少年投入巨大变动迷宫。舞台核心是**林间空地（Glade）**与环绕的**迷宫（Maze）**：白天可入，夜晚大门关闭；墙壁移动改图；夜间有**鬼火兽**巡逻。一句话：失忆男孩在盒子里醒来，用跑图与牺牲换一张可能不存在的出口地图；大门没关的那一夜，空地不再安全。

**【世界观 · 力量体系】**
无魔法修仙。战力=**体能与跑者耐力 + 武器（矛、刀、弓） + 地形记忆 + 团队编制 + 对鬼火兽弱点的情报**。鬼火兽：半生物半机械，酸液、毒刺、金属肢，夜间主杀；被刺者可能「变异/发热」进入「变化」（Changing）以换取迷宫记忆片段，代价是剧痛与人格撕裂。死亡即死，无复活。WICKED的远程监控、补给盒、变量投放是「系统侧」力量。

威胁层级：
1. **新来者（Greenie）/普通割草者、厨子**｜凡人农作。≈一阶。
2. **守卫、建造者、常规跑者**｜可处理单兽边缘与械斗。≈二阶。
3. **精英跑者小队 / 鬼火兽单只遭遇**｜迷宫内高死亡率。≈三阶。
4. **鬼火兽之夜（群体入侵空地）/ WICKED武装变量**｜据点级屠杀与实验收网。≈四阶。

乐园阶位映射（对照《阶位战力图鉴》）：空地农作新人≈**一阶**；守卫与日间迷宫边缘≈**二阶**；跑者深入与单兽≈**三阶**；**大门未关·鬼火兽入侵空地**≈**四阶**。世界顶点＝**WICKED实验体制+迷宫规则+末日疫**；无超阶。低阶严禁「一人推倒迷宫墙」。

**【地理 · 舞台】**
- **林间空地（Glade）**：农田、宿舍、武器库、图室、盒子升降井；四扇大门连迷宫。
- **迷宫走廊**：移动石墙、死路、标记系统；跑者白日绘图。
- **鬼火兽洞穴/崖壁路径**：兽巢与出口相关线索。
- **地图室**：多年跑图拼出的图案（「FLOAT」等单词线索）。
- **盒子（Box）**：每月送新人与补给的升降装置。
- **墙外世界（后部揭示）**：疫、废墟、WICKED设施——四阶后半与系列后续。

写场景优先：石墙摩擦夜鸣、藤蔓、烤肉香与机油腥、木矛尖、地图上的红叉、大门齿轮、酸液烧草的滋声。

**【世界剧情线】**
① **托马斯入盒**  
少年**托马斯**记忆被抹，送入空地。结识**纽特**、**明霍**（首席跑者）、**纽特**副手政治、**阿尔比**领袖、**加利**敌意、**查克**幼弟式羁绊。空地秩序：各司其职，禁止夜入迷宫。

② **秩序与跑者文化**  
跑者白天冲迷宫记路线，日落前必须返回。墙壁夜间移动。地图室是圣殿。新来者适应农作与恐惧。

③ **特丽萨与变量升级**  
首名女孩**特丽萨**抵达，昏迷中与托马斯有心灵感应式联系；留言「一切即将改变」。WICKED加大变量。

④ **卡片锚点：鬼火兽之夜**  
某夜**大门未关闭**，鬼火兽涌入空地。少年以木矛、弓箭、陷阱死战，死伤惨重。名场面必须同场：**未关的门、Grievers、林间空地、木矛、惨叫与尸、秩序崩盘**。因果：①空地「夜安全」神话破产；②领袖与跑者结构重创；③托马斯等被迫更激进求出口；④变化（被刺后记忆回流）成为可交易的诅咒。

⑤ **入迷宫救阿尔比与规则打破**  
托马斯为救人夜留迷宫，存活归来，动摇律法。后更多人进入「变化」求地图完整。

⑥ **出口与火疫真相侧翼**  
拼图指向出口；逃离迷宫后面对的是更大实验与灼热世界（接续集）。本档案四阶以「空地失守夜+逃出迷宫」为峰值。

⑦ **蝴蝶效应边界**  
可改：某次跑图成败、谁被刺、大门故障多开一小时。不可无代价改：记忆清除设定、夜门规则的恐怖核、鬼火兽存在、WICKED的实验伦理。

**【主要人物】**
- **托马斯**｜性格：好奇、冲动、领袖潜质、与迷宫有「设计侧」暧昧前史｜弧光：Greenie→破例者→出口关键｜立场：可变数。
- **明霍**｜性格：骄傲、速度、忠诚｜职位：首席跑者｜立场：迷宫之王。
- **纽特**｜性格：稳、协调、伤腿｜职位：二把手｜立场：秩序润滑剂。
- **阿尔比**｜性格：权威、压力｜职位：领袖｜弧光：被刺变化后崩溃边缘。
- **加利**｜性格：敌视托马斯｜立场：保守派暴力。
- **查克**｜性格：小、黏人｜弧光：牺牲型羁绊。
- **特丽萨**｜性格：冷静、信息源｜立场：变量与心灵链。
- **克林特、杰夫等医者；各岗少年群像**。
- **WICKED观察者**｜系统侧｜立场：数据先于人命。
- **鬼火兽**｜无性格，程序与饥饿｜四阶威胁体。

**【隐藏剧情 · 伏笔】**
- 托马斯与特丽萨曾参与设计侧。
- 地图拼词是出口钥匙。
- 治愈火疫的「免疫者」逻辑驱动全实验。
- 空地是杀区实验的A/B测试场。

**【大事记时间线】**
火疫前史 → WICKED建迷宫 → 男孩逐月投放 → 托马斯到 → 特丽萨到 → **鬼火兽之夜** → 破例入迷宫 → 拼图逃出 → 墙外世界。

**【叙事基调 · 雷区】**
基调：封闭、急促、少年政治、生物机械恶心感。  
忌：写成轻松校园；忌低阶推倒石墙；忌无视夜门；忌「跨IP乱入」套话。最早切入：托马斯出盒日，或大门故障前黄昏。

**【鬼火兽作战卡】**
酸、刺、金属肢、群体。弱点需原作逻辑内的探索（攻击特定部位/利用地形）。夜战禁止单人英雄。锚点夜是防守战：火把、矛阵、拖时间等门——但门没关。

**【空地律法】**
不伤害己方、各司其职、日落归。托马斯破律救人引发公审式冲突。秩序是存活技术，也是压迫。

**【跑者哲学】**
速度=寿命。地图室是教堂。跑者死亡用红叉纪念。三阶任务核心是「多活一个转弯」。

**【变化（Changing）】**
被刺后高热、幻觉、迷宫设计师记忆碎片。可换情报，付人格。四阶有人自愿被刺——伦理核。

**【微观事件】**
①第一天分配工作。②偷出地图抄本。③大门齿轮卡死前发现异常。④矛阵训练。⑤特丽萨苏醒传讯。⑥加利私刑企图。⑦引兽入死路。⑧逃出后看见「真实天空」。

**【世界剧情线·细部：出盒】**
铁箱上升、阳光刺眼、围观少年起哄。托马斯的恐惧与熟悉感并存——伏笔。

**【世界剧情线·细部：第一周】**
割草、脏话、篝火规矩。有人自杀倾向，有人笑话掩饰。

**【世界剧情线·细部：跑者选拔】**
耐力测试、迷宫边缘见习。明霍的速度是信仰。

**【世界剧情线·细部：特丽萨到达】**
盒子例外、全场静默、字条。变量开始。

**【世界剧情线·细部：锚点夜】**
门该关未关；先是一只，再是群；矛断、人飞、地图室险毁；天亮后清点名字。

**【世界剧情线·细部：夜留迷宫】**
托马斯与明霍等在移动墙间求生，改写「夜入即死」的绝对法。

**【地理补完·四门】**
北南东西门各通迷宫区，标记不同。跑者分区负责。

**【地理补完·图室】**
纸、木桌、蜡、多年层叠的路线。拼出单词时的战栗是四阶情报高潮。

**【力量细则·少年身体】**
无超凡，有训练差。矛阵纪律>个人勇。伤感染可致命。

**【主题句】**
墙会动，规则会骗人，名字会被红叉吃掉。跑，不是为赢，是为还有人记得路。

**【时间点状态机】**
托马斯是否已到、特丽萨是否已到、锚点夜是否已过、地图是否拼出、是否已逃出。

**【WICKED伦理】**
目的论：牺牲少数救物种。空地男孩是数据点。四阶真相战是「还听不听出口的话筒」。

**【电影差异】**
结构与部分情节不同。任务锁小说：记忆清除、跑者文化、鬼火兽机制、托马斯破例。

**【场景零件】**
石磨声、藤、粪肥、烤肉、酸臭、矛尖、地图红叉、盒链、门齿轮、兽肢液压声。

**【同盟税率】**
空地法：中高。跑者：高。加利派：负。WICKED：负。被刺同盟：不稳。

**【契约者错误】**
夜入迷宫炫耀→兽餐。毁地图→全员死刑。对WICKED广播信任到底→下一实验。

**【连续任务链】**
出盒适应（一）→守卫夜班（二）→跑者见习（三）→鬼火兽之夜（四）→拼图逃亡（四）。

**【终章语气】**
出口外未必是自由，是更大的迷宫。最高奖励是一张完整地图抄本与还活着的朋友名单。

**【补完·纽特的腿与权威】**
伤腿使他从跑者转为秩序核心。他的温和是空地不崩的胶。

**【补完·加利的恐惧政治】**
仇外与私刑是保守派工具。托马斯是他投射恐惧的靶。

**【补完·查克的功能】**
提供托马斯的人性锚。牺牲若发生，须写痛，不写燃。

**【补完·特丽萨信息战】**
她带来的真相碎片可救人可裂团。心灵链是双刃。

**【补完·地图单词】**
FLOAT等线索是解谜核。四阶支线可保护图室不被兽毁。

**【补完·补给盒政治】**
谁先开盒、如何分配，引发微叛乱。一阶日常戏。

**【补完·墙壁移动的时间表】**
夜间改图，跑者清晨重学。写迷宫先报「今天墙怎么变」。

**【补完·声音】**
门关是希望的金属声；门不关是死刑预告。锚点夜的核心声景。

**【补完·气味】**
草汁、汗、兽酸、医疗酒。用嗅觉分昼夜安全。

**【补完·乐园映射再钉】**
鬼火兽之夜=四阶据点遭灾级。不要把少年写成三阶裂石超人。

**【补完·对话速记】**
托马斯：我总觉得我来过。明霍：日落前回来。纽特：我们有规矩。阿尔比：听我的。加利：都是他的错。查克：你别死。特丽萨：一切将改变。WICKED：变量已投放。

**【补完·名场面链】**
出盒→适应→跑者→特丽萨→门不关→兽入空地→破例→拼图→逃出。

**【补完·防守阵型】**
矛在前，弓在后，火把两侧，伤员中心，图室死守。锚点夜可被契约者改一环。

**【补完·医疗】**
无抗生素时感染=慢死。被刺者隔离与否引发投票。

**【补完·语言】**
空地俚语（Greenie、klunk、shuck等，按原作语气适度使用）增强沉浸，勿堆到不可读。

**【补完·可观察物象】**
盒链锈、门齿痕、矛断茬、地图红叉、兽金属壳、特丽萨字条、篝火、农田垄、图室蜡泪。

**【补完·终局开放】**
逃出迷宫是第一部终点，不是系列终点。档案胜利：活过兽夜或带出地图。


**【移动迷宫·密度补完01：火疫与WICKED语汇】**
太阳耀斑与「火疫」（Flare）摧毁旧文明，患者狂乱、社会崩解。WICKED自称要以残酷实验找出免疫模式与治愈路径，口号接近「以恶行善」。空地男孩被洗去记忆，是为了控制变量：恐惧、领导、背叛、创造力都要被采样。契约者应把每一只鬼火兽、每一次门故障理解为「投放的变量」，而不是随机怪物潮。广播、补给盒、特丽萨的到来，都是实验员的手。

**【移动迷宫·密度补完02：空地一日时间表】**
黎明：清点人数与大门。上午：农作、建造、屠宰、跑者出发。正午：补给与冲突高发。下午：跑者归、图室更新。黄昏：全员点名，门齿轮合拢的金属声是集体心理安慰剂。夜晚：篝火、守夜、禁谈过多墙外幻想。锚点夜打破的是黄昏那一声「合拢」——当它缺席，时间表变成葬仪程序。

**【移动迷宫·密度补完03：岗位百科】**
跑者：最高荣光与最高死亡率。守卫：门与秩序。建造者：住房与陷阱。厨子与农夫：热量。屠宰：蛋白质与血腥耐受训练。医者：草药与截肢级决策。图室员：文明本身。领袖与二把手：仲裁与处刑权。新人必须先被岗位吞没，才被允许做梦。

**【移动迷宫·密度补完04：迷宫空间诗学】**
走廊等宽却方向背叛你；墙移动时有低鸣与尘。死路可能次日变通途。标记系统（石粉、刻痕、记忆口诀）是跑者私生子技术。迷路的真实恐惧不是饿，是夜。夜意味着兽与墙一起猎你。

**【移动迷宫·密度补完05：鬼火兽解剖学叙事】**
金属与肉的接缝、液压声、酸液烧草、尾刺注射。被刺者进入变化：高热、幻觉、设计师记忆碎片涌入。有人因此得到关键路线，有人精神碎裂攻击同伴。锚点夜的兽群不是BOSS血条，是移动的刑具阵列。

**【移动迷宫·密度补完06：托马斯的「似曾相识」】**
他对迷宫机制过度直觉，对特丽萨过度连接，对WICKED名字过度敏感——全是前史泄漏。加利因此恨他：秩序需要替罪羊。写托马斯勿开挂无敌，要写「知道却记不起」的折磨。

**【移动迷宫·密度补完07：特丽萨变量】**
首名女孩打破性别单一场域，带来信息与分裂。心灵感应式沟通可在锚点夜传警，也可被怀疑为WICKED后门。她的字条「一切即将改变」是全书节拍器。

**【移动迷宫·密度补完08：锚点夜分镜加长】**
黄昏门不关→议论→第一声兽鸣→矛阵仓促→农田被踩→有人断腿→图室门被撞→火把阵列→兽酸溅脸→天亮清点红叉新增。每一分钟都可被契约者改写：早发现齿轮、早点燃诱饵、早转移查克。

**【移动迷宫·密度补完09：破例入迷宫的政治】**
托马斯夜留救人，存活归来。公审式会议：英雄还是叛徒？律法若可破，空地靠什么不互杀？纽特调解，加利要血，明霍要结果。三～四阶人际战与兽战同样致命。

**【移动迷宫·密度补完10：地图拼词】**
多年路线叠加显形单词/图案，指向出口机制。图室被毁=文明回滚。四阶支线「死守图室」价值高于多杀一只兽。

关键NPC立场：**鬼火兽群**——屠杀；**阿尔比/纽特**——要阵型；**加利**——要找人背锅；**托马斯**——要破局；**特丽萨**——要关键信息。
主线钩子/支线：卡片锚点——撑到天亮或组织撤离死角、保护地图室、抢救被刺者；支线＝记录门故障证据、自愿/拒绝「变化」、为逃亡藏补给。
危险度/规避：极高——群体生物机械；规避单挑、护图室优先于逞能。
任务方向/奖励：空地残存火种；奖励＝出口线索、兽甲残片、幸存者名单上的你、WICKED更高关注（诅咒式奖励）。

**一阶补强**
你要学会空地俚语与镰刀角度，学会在起哄中保持呼吸。第一周目标不是英雄，是不被分配到「最容易死的蠢事」。与查克分享食物会换来忠诚；与加利顶撞会换来石头。盒子再升时，你要决定要不要帮新人——你还记得自己昨天也是Greenie。

**二阶补强**
夜班的敌人是睡意与想象。真正的兽未必来，但门轨的金属屑会告诉你有人动过手脚。汇报还是隐瞒，决定你是守卫还是叛徒。特丽萨的出现让男孩们的嗓音都变了调，你要在起哄里听见信息。

**三阶补强**
跑者的水囊比刀重要。转弯口的风不同，死路的回声不同。你第一次看见兽的金属反光时，腿会自己做决定。活着回来把线画直，比把兽刺穿更被明霍尊重。红叉旁多一个名字时，篝火会短一寸。


**四阶补强（逃出前夜 · 拼图与冲击）**
切入身份/时点：以「图室核心抄写员 / 出口突击队」切入兽夜之后、全员决定冲击出口时。
初始事件：单词拼完，特丽萨指出时间窗；有人要烧地图投降，有人要活捉一只兽做实验。
开场白建议：「墙上的字像用跑者的命写成。出口不保证天堂，只保证不再装睡。你把矛磨尖，把抄本塞进怀里，听见远处兽肢又在磨门轨。」
关键NPC立场：**托马斯**——要冲；**纽特**——要尽可能多的人活；**加利余党**——要破坏；**WICKED**——要数据收网。
主线钩子/支线：护送抄本到出口机制点；支线＝安置伤员、阻止内讧纵火、最后一次补给分配。
危险度/规避：极高；规避分心私斗。
任务方向/奖励：离开迷宫；奖励＝墙外第一口真实空气、实验真相碎片、幸存者羁绊。
