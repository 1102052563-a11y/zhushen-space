# 双城之战
<!--meta lib=主库 tiers=一、二、三、四、五、六、七-->

## 剧情

**【作品来源】**
《英雄联盟：双城之战》（全球名 *Arcane*，又译奥术）由拳头游戏与法国动画工作室 Fortiche 联合制作、Netflix/腾讯视频播出的动画剧集，主创克里斯汀·林克（Christian Linke）、艾力克斯·叶（Alex Yee）等。2021 年第一季 9 集、2024 年第二季 9 集（最终季）。背景为《英雄联盟》符文之地的皮尔特沃夫与祖安起源故事。文风：阶级冲突、姐妹羁绊撕裂、海克斯科技伦理、蒸汽朋克与底城炼金。卡片锚点：**金克丝引爆海克斯宝石，炸死皮城议会成员；蔚与杰斯装备海克斯拳套与战锤突袭底城炼金男爵工厂**。

**【世界定位】**
富裕乌托邦皮尔特沃夫（Piltover，「进步之城」）与污染、犯罪、微光横行的地下城祖安（Zaun）隔桥对立。主线追随底城孤儿姐妹**蔚（Vi / 菲艾）**与**金克丝（Jinx / 原名爆爆 Powder）**，以及发明家**杰斯·塔利斯**、**维克托**推动的海克斯科技如何改写双城命运。一句话：进步日礼炮响处，姐妹与阶级同时开战。

**【世界观 · 力量体系】**
战力来源：**体能搏击、执法装备、炼金科技、微光（Shimmer）诱变、海克斯科技（Hextech）与后期海克斯核心/奥术失控**。微光可短时强化肉体、致瘾、致畸；海克斯以奥术与科技结合驱动宝石、门、武器。死亡规则：凡人一枪一拳可死；微光强化者抗打骤升但仍可被海克斯兵器斩杀；维克托义体化后逼近「超越肉体」；安蓓萨级诺克萨斯将领与奥术神性存在构成本季上限。

威胁/战力层级（宁低勿高）：

1. **底城平民 / 皮城普通市民**  
   徒手或棍棒。≈一阶下沿。

2. **街头混混 / 初级执法者 / 野火帮外围**  
   拳脚、简陋枪械。≈一阶～二阶初。

3. **蔚（拳套前）/ 成熟执法者 / 塞薇卡（Sevika）义肢战**  
   可掀翻酒馆、单挑小队。≈二阶。

4. **微光强化打手 / 金克丝爆破艺术 / 杰斯早期海克斯锤**  
   毁街区级设施、炸桥、清工厂。≈三～四阶。

5. **海克斯武装杰斯+蔚联手、议会级政治暴力、炼金男爵私军**  
   工厂与桥头战役。≈四～五阶。

6. **维克托进化体 / 安蓓萨·梅德拉达（Ambessa）军团指挥 / 大规模奥术事故**  
   城市级改写、战争机器。≈六阶。

7. **顶点**  
   奥术本源失控、双城存亡级冲突、接近神性的维克托路线与战争女神级干预——本世界**顶点≈七阶**（人形战略/世界级下沿）。无八阶爆星。

乐园阶位映射：平民≈一阶；街头与初级执法≈二阶；蔚/塞薇卡/成熟金克丝爆破≈三阶；海克斯个人兵器与微光军团≈四阶；工厂突袭与议会爆炸级事件≈五阶；维克托进化与安蓓萨战争≈六阶；奥术终局与双城命运裁决≈七阶。顶点=**奥术/海克斯失控 + 安蓓萨战争机器 + 维克托神性化路径**，低阶规避，条件性胜利。

**【地理 · 舞台】**
- **皮尔特沃夫上城**：学院、议会厅、进步日庆典广场、吉拉曼恩庄园、杰斯工坊。
- **桥**：双城咽喉；多次流血与封锁。
- **祖安 / 底城**：毒河、管道、穷街、希尔科工厂、「最后一滴」酒馆（The Last Drop）。
- **静水监狱**：蔚被囚数年。
- **野火帮据点 / 艾克的避难所**：反微光社区。
- **辛吉德实验室**：微光与人体实验。
- **海克斯门与空中航道**（进阶季）：城际权力符号。
- **诺克萨斯触角**（安蓓萨线）：外部帝国压力。

**【世界剧情线】**

○ **第一季第一幕：游乐场与姐妹**  
叛乱年代，蔚与爆爆在桥战中丧亲，被范德尔（Vander）收养，与麦罗（Mylo）、克莱格（Claggor）为伍。数年后代为「最后一滴」一家。姐妹潜入皮城杰斯公寓盗窃，爆爆打碎海克斯水晶引发爆炸。范德尔与警长格雷森维持脆弱和平；希尔科（Silco）用微光制造变异打手戴克，复仇旧怨。杰斯因非法奥术实验被学院处分，维克托伸手合作，梅尔·梅德拉达议员暗中推动。希尔科绑架范德尔；蔚率兄弟营救，爆爆投出水晶炸弹误杀麦罗、克莱格并重创范德尔。范德尔注射微光救蔚后身亡。蔚掌掴爆爆称「扫把星」后离开，被马克（Marcus）下狱；爆爆哭倒在希尔科怀中，被收为金克丝。

○ **第一季第二幕：进步日与重逢**  
数年后皮城因海克斯繁荣。金克丝为希尔科走私微光，进步日之夜偷杰斯宝石并炸死执法者。凯特琳·吉拉曼恩（Caitlyn）从静水监狱提出蔚，共寻希尔科。塞薇卡刺伤蔚；金克丝在毒河边被希尔科劝说「接受新身份」。杰斯与梅尔相恋；维克托病情恶化。

○ **第一季第三幕：桥火与议会火箭（卡片锚点）**  
野火帮首领原是长大的艾克（Ekko）。金克丝与艾克桥上死斗。希尔科送金克丝至辛吉德，大量微光令其瞳色异变。杰斯与蔚捣毁微光工厂，杰斯误杀童工后转向和谈：以金克丝换祖安独立。金克丝绑架凯特琳与蔚、希尔科同桌「茶会」；混乱中金克丝误杀希尔科，希尔科临终仍称爱她。金克丝接受身份，将海克斯宝石装入火箭筒，在议会通过独立提案的瞬间射向议会厅——**爆炸撕碎皮城权力核心**（卡片锚点前半）。

○ **第二季：王冠、灰烬与终局**  
议会爆炸后双城战争边缘。蔚与杰斯分别以海克斯拳套、战锤等武装反击底城势力与炼金男爵体系（卡片锚点后半：工厂突袭）。凯特琳走向铁腕；金克丝成为符号与通缉犯。维克托沿海克斯核心/进化之路偏离人性；安蓓萨带诺克萨斯逻辑入局。艾克、黑默丁格等尝试底城自救。终局在奥术、战争与姐妹对决中收束：双城秩序重写，角色死伤与神性化代价高昂（以剧集终章为准：战争循环被打断的方式昂贵，金克丝/蔚关系以悲剧与救赎并存收场，维克托路线触及「光荣进化」的顶点威胁）。

○ **蝴蝶效应边界**  
可改：某次走私成败、童工是否死亡、工厂突袭伤亡。不可无代价改：海克斯与微光的科技伦理轴心、姐妹撕裂的核心创伤、议会火箭作为季终锁。

**【隐藏剧情 · 伏笔】**
- 范德尔差点溺死希尔科，铸就「眼中的毒」。
- 金克丝屋内三具尸体疑云（身份与时间线）指向更深层替换/创伤叙事（以正典呈现为准）。
- 童工之死是杰斯从武器转向和谈的道德开关。
- 维克托「光荣进化」是比金克丝火箭更慢却更深的终局威胁。
- 双城真正病灶是结构压迫，非单一个体疯狂。

**【大事记时间线】**
- 桥战年代 → 姐妹丧亲，范德尔收养。
- 盗窃夜 → 水晶爆炸，杰斯倒台又重生。
- 营救夜 → 炸弹误杀，蔚入狱，爆爆成金克丝。
- 进步日 → 宝石失窃，议会给杰斯席位。
- 桥火与茶会 → 希尔科死，火箭射议会。
- 战后 → 工厂突袭、安蓓萨、维克托进化、姐妹终局。

**【叙事基调 · 雷区】**
基调：悲剧史诗、阶级痛感、姐妹爱与伤害同体、科技伦理。  
忌：把金克丝写成无动机疯子；忌低阶秒杀安蓓萨/进化维克托；忌把祖安写成纯邪恶；忌忽略微光成瘾社会成本；忌套话。最早锚点：盗窃杰斯公寓前夜，或进步日爆炸当夜。


**【世界剧情线·第一季第一幕细部】**
桥战年代，祖安叛军冲击皮城遭残酷镇压，蔚与爆爆在废墟中失去父母，被战败领袖范德尔收养，与麦罗、克莱格组成底城「家庭」。数年后范德尔经营「最后一滴」酒馆，以暴力威胁与交涉维持底城脆弱和平。姐妹潜入皮城杰斯公寓盗窃，爆爆拿走海克斯水晶却不慎引爆，毁坏大楼。格雷森警长向范德尔施压要人；底城市民要求反击。希尔科——范德尔旧友，曾被其在毒河差点溺死——用微光把戴克变成变异打手，绑架范德尔。蔚率兄弟营救，爆爆为救姐投出水晶炸弹，误杀麦罗、克莱格并重创范德尔。范德尔注射微光救蔚后死去。蔚掌掴爆爆称「扫把星」后离开，被马克伏击入狱；爆爆以为被弃，哭倒在希尔科怀中，被重塑为金克丝。

**【世界剧情线·第一季第二、三幕细部】**
数年后皮城因杰斯海克斯科技繁荣，进步日庆典上金克丝盗窃宝石并炸死执法者。凯特琳从静水监狱提出蔚共查希尔科。塞薇卡义肢战刺伤蔚；希尔科在毒河边劝金克丝接受新身份。杰斯与梅尔相恋并获议会席；维克托病情恶化，转向海克斯核心与辛吉德合作。野火帮首领竟是长大的艾克，致力于反微光与社区自救。桥上金克丝与艾克死斗，马克叛变线收束于爆炸。杰斯与蔚捣毁微光工厂，杰斯误杀童工后转向和谈：以金克丝换祖安独立。金克丝绑架凯特琳、蔚与希尔科同桌「茶会」；混乱中金克丝击倒希尔科，希尔科临终仍说爱她。金克丝接受身份，将宝石装入火箭筒，在议会通过独立提案瞬间射向议会厅。

**【世界剧情线·第二季纲要细部】**
议会爆炸后皮城要战争，祖安要呼吸。蔚与杰斯以海克斯拳套与战锤突袭底城炼金男爵工厂（卡片锚点后半）。凯特琳走向铁腕执法；金克丝成为符号、创伤与通缉犯的综合体。维克托沿海克斯核心改造肉体，触及「光荣进化」，助手斯凯之死成为其罪与驱力。安蓓萨·梅德拉达以诺克萨斯战争逻辑入局，要女儿梅尔学会「王冠之重」。艾克与被逐的黑默丁格尝试底城科技向善。终局在奥术失控、战争循环与姐妹对决中收束：打断循环的代价是血、神性化与告别。具体生死以剧集终章为准，任务世界应保留「胜利必贵」原则。

**【势力与经济补强】**
皮城议会是商人与贵族的进步联盟，海克斯门与贸易航道是其血液。执法者维护桥上秩序，也执行结构性暴力。希尔科的微光帝国用成瘾控制劳动力与士兵。炼金男爵是底城军阀资本家。野火帮是青少年自治与戒毒共同体。「最后一滴」象征范德尔式和平底城。诺克萨斯是外部帝国压力锅。写任务时让钱、微光、宝石、选票四种货币互相兑换。

**【人物对位补强】**
蔚/金克丝：保护与毁灭的同一枚硬币。杰斯/维克托：进步的光面与暗面。范德尔/希尔科：和平父职与毒河父职。凯特琳/梅尔：特权女性的两种政治路径。艾克/金克丝：时间与记忆的童年分叉。安蓓萨/梅尔：战争教育与自主。辛吉德是所有黑暗科学的供应商，跨线出现。

**【微观事件库】**
①进步日烟花下的走私交接。②静水监狱探视交易。③野火帮戒毒帐篷守夜。④议会投票前的贿选与理想演说。⑤工厂童工转移。⑥毒河边的身份劝说戏（金克丝线）。⑦海克斯核心实验室的伦理辩论。⑧安蓓萨晚宴上的刀光。

**【AI雷区补强】**
不要把祖安写成纯犯罪乐园；不要把皮城写成纯正义；不要让金克丝无创伤动机；不要低阶秒安蓓萨或进化维克托；海克斯不是万能愿望机，每次使用都要有社会后果。




**【海克斯科技编年】**
杰斯童年被奥术魔法所救，执念用科技稳定奥术。水晶盗窃夜后他几乎被毁，维克托以共同理想拉他起来，梅尔提供政治保护。海克斯门与宝石装置带来贸易革命，也带来武器化诱惑。黑默丁格的警告被当成保守。维克托的海克斯核心能修复肉体，也会杀死斯凯。金克丝的火箭是海克斯平民化暴力的极端形态：一个底城女孩用上城的心脏轰碎上城的头颅。

**【微光产业链】**
原料、提炼、注射、变异、成瘾、戒断、野火帮的康复帐篷，构成祖安真正的GDP。希尔科用微光买警察、造士兵、控价格。塞薇卡的义肢战斗是微光工业的人体广告。工厂突袭会引发街头断供与暴动，不只是打一架。

**【姐妹时间线对照表】**
童年：蔚保护爆爆。盗窃夜：蔚指责。入狱年：蔚被时间冻结，爆爆被希尔科重塑。重逢：蔚要「妹妹回来」，金克丝要「证明我配被爱」。茶会：选择枪口。火箭：选择身份。第二季：选择是否结束循环。契约者任何介入都应碰到这根神经。

**【议会政治游戏】**
席位、贿选、进步日演讲、对底城的恐惧、对贸易的贪婪。杰斯获得席位是因为宝石失窃需要技术权威。童工之死后他才看见武器的脸。和谈提案通过的瞬间被火箭否定——历史的黑色幽默。

**【第二季战争逻辑】**
爆炸后鹰派上台，铁腕执法，底城报复。安蓓萨带来职业战争。维克托带来后人类。艾克带来社区时间。终局需要多方代价同时支付，禁止单一英雄无伤拯救双城。

**【名场面清单（可切入改写边缘）】**
范德尔毒河溺希尔科（前史）；盗窃爆炸；营救夜炸弹；进步日爆炸；静水重逢；桥上金克丝vs艾克；工厂突袭；茶会；议会火箭；以及第二季王冠与灰烬诸战。改写原则：可改伤亡数字与谁先开枪，难改姐妹结构性创伤。

**【道具升级路径】**
拳套校准、战锤充能、金克丝枪械涂装与爆炸当量、凯特琳狙击镜、塞薇卡臂刃、艾克的装置。奖励应沿此路径，不发「毁城无敌宝石」。

**【双城之战·场景零件库（供正文调用）】**
写任何段落先选 3 个零件：气味、声音、制度、道具、伤口、价格。地铁/孤岛/双城/日记/末日各有专属零件，禁止串味。人物开口必须带身份立场，禁止「路人甲说了一句有深意的话」式空壳。战斗后写政治后果：谁涨价、谁封路、谁要你的血或日记或宝石。

**【双城之战·同盟与背叛税率】**
本世界盟约默认脆弱。结盟时必须先谈清楚：退出条件、分赃比例、谁先死谁断后。背叛不是随机事件，而是资源不足时的理性选择。契约者若以乐园外挂碾压，本地人会恐惧并集结更强规则（项圈、议会、神、火萤、导弹）反噬。

**【双城之战·时间点状态机】**
AI 在任意时点应能回答：主角现在在哪、谁已死、关键道具在谁手里、下一场名场面是什么。若契约者改写了局部，用「追杀/取代/更惨」回流，而不是假装原作没发生。


**【第一季九集情绪曲线】**
1-3集：童年创伤与姐妹分裂完成。4-6集：时间跳后的重逢与身份错位。7-9集：桥火、工厂、茶会、火箭，一切加速破裂。任务应按曲线选强度：早期打群架，后期打战争。

**【第二季主题词】**
王冠、灰烬、名字、蓝、河床、符文、第一次、循环、指甲缝的泥。每一词对应一集气质：权力、燃烧、认同、涂装暴力、地下真相、奥术信息、时间回环、仇杀、余烬人性。契约者高阶切入应撞上「循环如何停」。

**【杰斯-维克托决裂年表】**
共创海克斯→宝石武器化分歧→黑默丁格被逐→核心实验→斯凯死→承诺摧毁核心→违背与战争。决裂不是吵架，是理想被权力与死亡污染。

**【凯特琳阶级自觉】**
从特权执法菜鸟到理解底城地图，再到铁腕争议。她可能爱蔚，也可能用制度伤害蔚所属的世界。任务勿把她写成无瑕女主。

**【艾克的时间与社区】**
野火帮不是童子军，是武装自救。艾克对金克丝下不了死手，是童年记忆的政治。黑默丁格加入后，科技向善第一次有了实验室。

**【安蓓萨战争教育语录式冲突】**
「软弱招致征服」「女儿必须像狼」。梅尔的金与外交是另一种武器。诺克萨斯线让双城矛盾被嵌入更大帝国棋盘，七阶切入应感到「城市之上还有刀」。

**【爆炸物理学与政治学】**
议会火箭不是点状伤害，是权力真空炸弹：谁来填补席位、谁被宣布殉道、谁被宣布恐怖分子。工厂突袭同样：断微光=断成瘾者命=短暴动。写后果先于写爆炸光效。



**【底城口音与上城礼仪】**
祖安脏话、管道隐喻、酒馆划拳；皮城 titling、学院腔、议会敬语。角色串层时口音会暴露身份。金克丝的笑声是第三种语言：破裂的童谣。

**【海克斯宝石失窃的连锁】**
安保升级→杰斯政治化→武器研发→底城恐慌→走私利润上升→野火帮更难。一块宝石推动整季政治。

**【希尔科临终】**
「你做的很好」式父爱与金克丝的崩溃是茶会核心。杀他的是她，葬他的也是她。火箭是葬礼的礼炮。禁止把此场面写成爽杀。

**【蔚的拳与凯特琳的枪】**
近战与远程、底城与上城、冲动与程序。双人任务应让两种解决方式冲突：打还是谈，抓还是放。

**【维克托的身体政治】**
残疾、疾病、义体、进化。他追求的是不被身体背叛，却可能背叛人性。斯凯之死是实验伦理的血。

**【金克丝武器命名癖】**
她给枪起幼稚名字，把杀戮玩具化，是PTSD的化妆。战利品奖励可用「被她涂鸦过的零件」增加角色感。

**【双城桥梁作为反复舞台】**
桥上死过人、谈过和、封过城、爆过虫械。任何重大转折可再回桥，让地理有记忆。



**【进步日仪式细节】**
演讲、气球、海克斯展示、儿童合唱、暗处的走私船。金克丝的爆炸把庆典变成罪案现场。杰斯放弃展示新宝石是黑默丁格警告的直接结果，也埋下「未展示却被盗」的政治责任。

**【静水监狱年代】**
蔚在地下遗忘时间，世界在上方改朝换代。出狱后的光照、声音、对金克丝名字的震惊，是表演重点。契约者若曾探监，可成为她唯一的时间证人。

**【野火帮日常】**
巡逻、戒毒、修理、涂鸦、对希尔科的诅咒、对皮城的复杂情绪。艾克的领导是做出来的，不是封号。



**【补完块1·双城之战】**
范德尔的和平是用自己当人质换来的：他压住底城怒火，也压住执法者的清剿。希尔科认为和平是跪着的死。两人理念冲突是双城政治的原型。

爆爆的「扫把星」称呼来自至亲之口，杀伤力高于敌人子弹。金克丝后来把伤害变成人设与武器涂装，是自我保护。

杰斯的锤子最初是工具，后来是权杖。梅尔教他政治舞蹈，也让他失去部分天真。维克托失去的是身体与边界。

凯特琳进入底城靠的是地图与特权，离开时带走的是愧疚与爱情。她对希尔科威胁的认知被议会无视，是上城系统性失明。

金克丝茶会桌的布置像儿童过家家：座位、枪、选择题。她要蔚「选我还是选她」。蔚拒绝选择题本身，导致场面滑向死亡。

议会火箭发射时，独立提案的墨水未干。和平被自己的程序否定。这种反讽是第一季最后一枪的文学性。

第二季若写安蓓萨，要写军队纪律、家族狼性、对「软弱城市」的轻蔑。她不是街头反派，是帝国级玩家。

维克托的进化体发光时，底城像被新神巡视。信徒与恐惧者同时出现。「光荣进化」是七阶的意识形态副本。


**【补完块2·双城之战】**
海克斯门开启贸易时，皮城庆祝，祖安的空气未必更好。进步的分配问题是剧集经济基础。

塞薇卡在希尔科死后的权力真空中求生，义肢与立场一样可以更换，但骨子里的底城务实不变。

艾克与金克丝的桥上战是童年游戏的成人版：谁先停手谁更痛。艾克停手，金克丝用炸弹回答。

童工之死让杰斯看见海克斯武器的脸。没有这一眼，就没有和谈，也就没有被火箭嘲弄的和谈。


**【双城气味地图】**
皮城：金属抛光剂、咖啡、海风、新布料。祖安：油、化学甜味（微光）、潮霉、血与酒吧麦酒。角色跨城时应用气味提示读者。金克丝的爆炸后是焦糊与臭氧。

**【海克斯实验室规则】**
未授权实验违法，学院可开除，议会可定罪。杰斯与维克托的夜间实验是理想也是罪。黑默丁格的长生视角让他恐惧奥术失控，年轻人听成懦弱。

**【希尔科的毒河布道】**
他带金克丝回被溺之河，重演创伤并重写意义：痛苦是力量，旧名可死，新名可生。这是邪教式父爱，有效且可怕。

**【工厂童工的经济】**
微光工厂需要小手与低成本。杰斯杀死童工不是偶然，是产业链把孩子放进火力范围。和谈的良心起点在这里。

**【议会席位游戏】**
商人、贵族、科学家、外邦代表（梅尔）交换选票。杰斯的席位是危机任命。爆炸后席位真空引发继承战争式政治。

**【金克丝的涂鸦语言】**
墙画、笑脸、猴子、霓虹涂装，是她的宣言与地图标记。任务可用「跟着涂鸦走」进入她的据点，比跟着枪响更角色。

**【蔚入狱空白年】**
她不知道金克丝这个名字如何长成。重逢时两套记忆对不上，冲突不可避免。契约者若只站一边叙事，会加剧分裂。

**【第二季循环论】**
仇杀、战争、进化、帝国，都是循环。停循环要有人放下刀，或有人付出不可逆代价。七阶任务的胜利条件应是「打断一次循环」，不是「占领双城」。

**【配乐式场景提示】**
进步日用庆典乐，桥战用鼓点，茶会用诡异童谣感，终局用空寂。AI不写歌名也要写节奏：快切或长镜头。



**【海克斯与奥术的边界】**
海克斯是把奥术装进可量产外壳的企图。稳定时点亮城市，失控时撕裂现实纹理。第二季终局的「城市开裂感」来自边界被反复捅破。低阶不可把奥术当烟花。

**【底城交通与垂直社会】**
电梯、管道、吊桥、毒河渡口决定谁能见到阳光。希尔科控制关键垂直节点即控制人口流动。野火帮在夹缝建家。

**【金克丝火箭的弹道政治】**
目标是议会厅，符号是「你们的进步由我签字」。爆炸后每个政客的死都会被重新叙事：殉道、无能、秘密交易曝光。


**【终章语气锁定】**双城的晨雾里仍有油与金属抛光剂。议会的缺口可以用新人填，姐妹的缺口不能。海克斯还会亮，微光还会流，安蓓萨的狼还会找门。契约者能做的是让某一次开枪晚三秒、让某个孩子活过工厂夜、让某句「对不起」被听见。进度日会继续过，但故事只承认付过代价的人。

**【双城制度与街道补完】**
皮城通行证与底城文身一样能决定你走哪条桥。执法者盾墙推进时，商贩会在三十秒内收摊消失。微光针剂的价格随工厂被捣毁而暴涨，引发街头抢药。海克斯门关闭等于贸易心脏停跳，议会会为此互相咬。金克丝的据点常选在有回声的管道，方便她听见谁靠近。蔚的拳头打破的不只是下巴，还有谈判需要的时间。凯特琳的狙击镜里，底城从「犯罪统计」变成一张张脸。

**【议会与工厂双舞台补丁】**议会厅的穹顶玻璃在火箭命中前反射进步日彩带；命中后彩带与烟混成一种可笑的葬礼。工厂车间的传送带还在转，微光桶翻倒，紫色液体像河。蔚的拳套每次撞击都让灰尘从梁上落下，杰斯的锤带回声。塞薇卡的义肢冒火花，仍不退。童工的鞋子落在角落，是此战真正的计分板。契约者若只记得击杀数，就写歪了双城。



**【进步日次日的街道】**封街、戒严、宝石悬赏令、底城走私改道、野火帮救人、议会临时会议在废墟边召开。杰斯的名字被喊成英雄也是祸首，金克丝的名字第一次印上全城通告。蔚若已出狱，会在这份通告上看到妹妹的笑脸涂鸦版通缉。

## 阶位切入点

> 阶位↔：一阶≈底城街头/盗窃前；二阶≈执法与走私；三阶≈微光打手与金克丝爆破；四阶≈海克斯个人兵器；五阶≈工厂突袭与议会爆炸级；六阶≈安蓓萨战争与维克托进化；七阶≈奥术终局。顶点条件性胜利。

**一阶（底城孩童 · 游乐场规则）**

切入身份/时点：契约者以「管道跑腿孩 / 本索店临时工 / 刚到底城的落魄学徒」切入范德尔收养姐妹后的日常，杰斯公寓盗窃前。

初始事件：蔚拉你「练胆」上桥；爆爆塞给你一颗未装药的玩具雷。「今天只是玩玩」——直到希尔科的眼睛在毒雾里亮了一下。

开场白建议：「上城的灯永远干净，底城的雨永远带油。酒馆里范德尔用低音说：别惹执法者。可你口袋里那颗螺丝，是爆爆说『以后能炸开命运』的零件。」

关键NPC立场：**蔚**——考你拳头；**爆爆**——要认可；**范德尔**——要和平；**麦罗/克莱格**——嘴臭但护犊；**希尔科**（远）——要情报。

主线钩子/支线：钩子=是否参与皮城盗窃；支线=送货躲戴克；支线=偷听格雷森谈判。

危险度/规避：低～中；规避毒河深区与希尔科会面。

任务方向/奖励：站稳底城身份；奖励=跑腿人脉、简陋工具、范德尔一句「你可以留下」。

**二阶（爆炸余波 · 追查与封街）**

切入身份/时点：盗窃爆炸后，以「目击者 / 执法线人 / 希尔科外围搬运」切入。

初始事件：格雷森要范德尔交人；底城要范德尔开战。你手里有一块炸飞的水晶碎片——两边都要。

开场白建议：「进步之城的玻璃渣还在下。谁先眨眼，谁就是叛徒。你的碎片在口袋里发烫，像一颗还没决定爆炸的心。」

关键NPC立场：**范德尔**——护家；**格雷森**——要凶手；**马克**——已在被腐蚀；**戴克**——微光实验体。

主线钩子/支线：钩子=碎片归属；支线=救下被抓的底城孩；支线=给杰斯匿名警告。

危险度/规避：中；规避希尔科本营。

任务方向/奖励：情报价码；奖励=微光样品（剧毒）、执法通行证伪章。

**三阶（金克丝之夜 · 进步日）**

切入身份/时点：进步日，以「庆典安保临时工 / 野火帮眼线 / 金克丝军火黑市中间人」切入。

初始事件：烟花与枪声分不清；六名执法者倒下，宝石失踪。你被两边同时点名：「看见蓝头发了吗？」

开场白建议：「礼炮本该赞美未来，却把未来打成了弹孔。有人在屋顶笑，笑声像拆开的发条。」

关键NPC立场：**金克丝**——表演型威胁；**希尔科**——要成果；**杰斯**——要宝石；**凯特琳**——要真相。

主线钩子/支线：钩子=宝石流向；支线=帮艾克截走私；支线=给蔚监狱传话。

危险度/规避：中高；金克丝爆破无差别。

任务方向/奖励：活过进步日；奖励=爆炸物配方残页、议会旁听证。

**四阶（海克斯武装 · 拳与锤）**

切入身份/时点：蔚出狱后，以「凯特琳编外向导 / 杰斯实验室清洁工实为技工」切入海克斯武器化前后。

初始事件：蔚戴上拳套试力，墙塌一角；杰斯的锤第一次嗡鸣。你被要求在「摧毁工厂」与「保护童工」之间选站位。

开场白建议：「当科学学会握拳，桥就不再只是桥。你听见金属里有奥术的心跳——它不管你站在上城还是下城。」

关键NPC立场：**蔚/杰斯**——行动派；**维克托**——要核心不要战争；**塞薇卡**——工厂盾。

主线钩子/支线：钩子=工厂突袭预备；支线=转移童工；支线=偷改武器保险。

危险度/规避：高；微光强化群。

任务方向/奖励：介入武装升级；奖励=海克斯废料、拳套校准数据。

**五阶（议会火箭 · 锚点决战）**

切入身份/时点：茶会之夜到火箭发射，以「议会书记 / 金克丝邀请的『客人』 / 和谈信使」切入。

初始事件：桌上三个人质，一把枪；希尔科倒下；蓝发女孩把宝石推进火箭。你有十秒：扑向发射器、护住一名议员，或什么都不做只记下历史。

开场白建议：「独立提案的墨迹未干，火箭已经学会写字。字是火，收件人是整个进步日。」

关键NPC立场：**金克丝**——终局表演者；**蔚**——喊妹妹；**希尔科**（死/濒死）——爱与国的选择题；**杰斯**——和谈破产。

主线钩子/支线：钩子=卡片锚点爆炸；支线=救下一名议员改变权力真空；支线=抢出和谈文件副本。

危险度/规避：极高；爆炸与随后战争。

任务方向/奖励：改变爆炸伤亡结构；奖励=政治遗产、金克丝罪证/心证、战争爆发时间表。

**六阶（安蓓萨与进化 · 战争季）**

切入身份/时点：第二季战争期，以「诺克萨斯外交翻译 / 维克托公社外围 / 凯特琳铁腕下的特别探员」切入。

初始事件：安蓓萨的舰队/军团逻辑压境；维克托的进化体在底城发光。你被两边征召：「选文明，或选胜利。」

开场白建议：「狼妈妈说战争是教育，机械先知说肉体是错误。你夹在两种未来中间，呼吸都像通敌。」

关键NPC立场：**安蓓萨**——要战争；**梅尔**——要自主；**维克托**——要进化；**艾克**——要底城活路。

主线钩子/支线：钩子=阻止或加速全面开战；支线=保护黑默丁格装置；支线=切断微光供应链。

危险度/规避：极高；规避与安蓓萨单挑、与进化体硬碰。

任务方向/奖励：战争棋手入场；奖励=诺克萨斯军情、进化样本（剧毒伦理）、双城临时停火令。

**七阶（奥术终局 · 命运改写）**

切入身份/时点：最终幕，以「奥术观察者 / 艾克时间线协助者 / 双城最后信使」切入。

初始事件：天空与城市纹理被奥术改写；姐妹在终点对视。你能递出的不是武器，是一个「停手」的理由——或一颗让某一方永远闭嘴的引爆点。

开场白建议：「当城市开始唱歌一样的裂开，你终于明白：海克斯从不是灯，是门。门后没有上城下城，只有代价。」

关键NPC立场：**蔚/金克丝**——终局情感核；**维克托**——顶点威胁；**杰斯**——共业者；可能的神性/奥术意志——不可硬刚。

主线钩子/支线：钩子=终局站队与代价分配；支线=保全艾克社区火种；支线=让梅尔完成对安蓓萨的政治/武力了断。

危险度/规避：顶点——**超阶式误解禁止**；条件性胜利=打断循环、换取双城残存秩序，非无伤通关。

任务方向/奖励：见证新秩序；奖励=奥术残响（高危）、双城重建权柄碎片、姐妹和解或永别的「在场证明」。
