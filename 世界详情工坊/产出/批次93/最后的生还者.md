## 剧情

**【作品来源】**
《最后的生还者》（The Last of Us）由 Naughty Dog 开发、索尼发行的动作冒险游戏，创意总监尼尔·德鲁克曼（Neil Druckmann）、游戏总监布鲁斯·斯特雷利（Bruce Straley）。2013 年 PS3 首发，后有 Remastered、Part I 重制；DLC《遗落》；续作《第二部分》2020。另有 HBO 剧集等。本档案以**第一部游戏主线正典**为默认世界（卡片锚点在比尔小镇高中巨无霸战），续作人物作延展标注。文风：末日公路、父女羁绊、道德灰、感染恐怖。卡片锚点：**乔尔与艾莉在比尔的小镇高中遭遇巨无霸感染者，孢子与甲壳需燃烧瓶与散弹应对**。

**【世界定位】**
2013 年前后，变异虫草属真菌（Cordyceps）全球爆发，宿主被转化为「感染者」。二十年后，美国四分五裂：军政府隔离区、火萤反抗军、猎人与食人部落并存。走私者**乔尔·米勒**失去女儿莎拉后麻木求生，受命护送对真菌免疫的少女**艾莉**穿越废土，前往火萤实验室以求疫苗——终点却是他私心的谎言。一句话：世界要她的命换解药，他选择她。

**【世界观 · 力量体系】**
无魔法。战力=**枪械与近战、潜行、工艺制作（燃烧瓶、钉棒）、感染者阶段强度、人类派系火力**。感染阶段：奔跑者（Runner）→潜行者（Stalker）→咔咔（Clicker，超声波定位、硬脸甲）→巨无霸（Bloater，厚甲+孢子囊）。人类侧：隔离区士兵、火萤、匹兹堡猎户、大卫食人部落等。死亡即死；艾莉免疫不代表不被咬伤痛苦/二次风险叙事。

威胁/战力层级（宁低勿高）：

1. **普通幸存者 / 隔离区平民**  
   徒手或棒。≈一阶下沿。

2. **有枪平民 / 初级巡逻兵 / 奔跑者**  
   小队交火。≈一阶～二阶。

3. **熟练走私者（乔尔中期）/ 咔咔 / 猎户小队**  
   建筑清场、伏击。≈二～三阶。

4. **巨无霸 / 重装军 / 大卫式领袖+部落**  
   需燃烧与霰弹；可屠据点。≈三～四阶。

5. **（本档跳过五阶）**  
   工单无五阶切入；世界仍存在城市级冲突，但不单列。

6. **顶点（六阶）**  
   **火萤手术体系 + 乔尔单人闯实验室的战力叙事 + 疫苗伦理作为「改变人类命运的开关」**；地区/城市级决定权。本世界**顶点≈六阶**（无七阶毁国超凡）。乔尔不是超人，但是「愿意为一人开战的顶点人类」。

乐园阶位映射：平民≈一阶；武装巡逻与奔跑者≈二阶；乔尔/咔咔/猎户≈三阶；巨无霸与食人部落首领≈四阶；火萤总部手术与终局闯关≈六阶。顶点=**火萤「杀艾莉取药」的文明选择题 + 乔尔的拒绝**，条件性胜利=带走艾莉/揭露谎言/另寻免疫样本，严禁把火萤写成纸老虎。

**【地理 · 舞台】**
- **奥斯汀（爆发夜）**：莎拉之死。
- **波士顿隔离区**：军管、黑市、玛琳接头。
- **城外废墟与地下**：感染者巢、地铁。
- **林肯小镇（比尔的镇）**：陷阱之城；高中体育馆=卡片锚点。
- **匹兹堡**：猎户伏击区。
- **杰克逊县**：汤米公社。
- **科罗拉多大学 / 实验室废墟**：火萤旧点。
- **盐湖城医院**：终局手术室。
- **公路、水坝、季节变换**：旅程骨架。

**【世界剧情线】**

○ **爆发夜**  
乔尔、弟弟汤米带女儿莎拉撤离，军警误杀莎拉。乔尔人格封闭。

○ **二十年后：波士顿委托**  
乔尔与搭档苔丝为生计干脏活。火萤领袖玛琳以武器交易托付护送免疫少女艾莉出城。途中苔丝发现自己被咬，殿后牺牲；乔尔被迫独自（后与艾莉）上路。

○ **比尔的小镇（卡片锚点）**  
偏执生存者比尔用陷阱武装整座空城。为修车/取得零件，乔尔与艾莉进入高中建筑，遭遇**巨无霸**：厚重菌甲、近战碾压、喷射孢子囊。需用燃烧瓶软化甲壳再以散弹/火力输出——此战定义「感染者顶点单位」的恐惧感（卡片锚点）。后揭示比尔伴侣弗兰克的悲剧，比尔给车与物资送他们上路。

○ **匹兹堡与亨利、萨姆**  
猎户城市；乔尔艾莉与亨利兄弟合流。萨姆被咬后感染，亨利崩溃杀死弟弟再自杀。旅程的「孩子与保护者」主题再碎一次。

○ **杰克逊与汤米**  
汤米建立水电公社。乔尔一度想把艾莉扔下，最终承认羁绊，继续西行。

○ **冬季：大卫**  
乔尔重伤。艾莉独自狩猎，遇大卫的「友好」共同体——实为食人部落。大卫对艾莉的掠食与说教，艾莉以暴力挣脱，心理留下永久刻痕。乔尔雪中找到她。

○ **春：盐湖城**  
火萤截获二人。玛琳告知：制疫苗需摘取艾莉脑部（致死）。乔尔屠穿实验室，击毙主刀医生与阻拦者，救出麻醉中的艾莉，杀死玛琳。公路上乔尔谎称「火萤已有许多免疫者、手术无望、他们放过我们」。艾莉怀疑但暂时接受。世界失去（可能的）解药，换得父女继续活着。

○ **延展（第二部分提要，非本档主舞台）**  
数年后杰克逊生活被艾比复仇打断：艾比杀乔尔（因其父即当年医生）。艾莉与戴娜赴西雅图复仇；循环在圣巴巴拉近乎互相毁灭后，艾莉放手。本档案切入默认第一部时间线。

○ **蝴蝶效应边界**  
可改：比尔镇路线伤亡、是否救下某路人、匹兹堡伏击细节。不可无代价改：艾莉免疫、火萤手术需致死取样的设定、乔尔撒谎的终局题眼。

**【叙事基调 · 雷区】**
基调：压抑公路片、父女爱、暴力沉重、没有爽感救世主。  
忌：把乔尔写成超级英雄无代价；忌让艾莉无痛感免疫；忌低阶团灭火萤总部；忌把感染者写成无脑数值怪而忽略潜行资源；忌套话。最早锚点：波士顿出城前夜，或比尔小镇入口。


**【世界剧情线·细部展开一：爆发与波士顿】**
虫草真菌通过孢子与啃咬传播，宿主被接管神经，阶段演化为奔跑者、潜行者、咔咔、巨无霸。文明在数周到数月内崩塌。二十年后波士顿隔离区由军政府FEDRA高压统治，火萤作为反抗军存在。乔尔与苔丝靠走私为生，人格冷硬。玛琳以交易委托他们护送免疫少女艾莉出城。出城途中感染与人类火力交织，苔丝被咬后确认艾莉免疫的真实性，选择殿后牺牲，把「活下去」的命令留给乔尔。

**【世界剧情线·细部展开二：比尔镇与巨无霸】**
林肯小镇被比尔改造成陷阱迷宫，门外挂着警告与尸体。为获得车辆零件，乔尔与艾莉进入高中建筑。体育馆与走廊存在巨无霸：厚甲抗枪，近战秒杀，喷射孢子囊迫使防毒与拉开距离。标准战术是燃烧瓶软化甲壳，再用霰弹或高火力输出。此战是旅程中期的恐惧高峰，也证明感染者生态有「顶点单位」。事后比尔线揭示其与弗兰克的爱恨与遗弃，比尔最终提供车辆与军火，让公路旅程真正开始。

**【世界剧情线·细部展开三：匹兹堡、杰克逊、冬季】**
匹兹堡被猎户控制，过路者即猎物。乔尔艾莉与亨利、萨姆兄弟合流，经历城市巷战与背叛式信任。萨姆被咬后感染，亨利杀死弟弟再自杀——「保护孩子」主题的黑暗预演。杰克逊县汤米建立水电公社，展示微光文明可能。乔尔一度想把艾莉留下，最终承认自己需要她，继续西行。冬季乔尔重伤，艾莉独自面对大卫的食人部落；大卫以文明话语包装掠食，艾莉以极端暴力脱身，心理留下不可逆刻痕。

**【世界剧情线·细部展开四：盐湖城终局】**
火萤在盐湖城医院准备以摘取艾莉脑部的方式研制疫苗。玛琳认为牺牲一人可救人类。乔尔拒绝成为「把第二个女儿送上手术台的人」，单人闯关，杀死士兵与主刀医生，带走艾莉，并杀死试图阻拦的玛琳。公路上他撒谎：火萤已有很多免疫者，手术无望，他们放人。艾莉怀疑但暂时接受。世界失去可能的解药，换得父女继续存在。这个谎言是系列道德核心，也是续作复仇循环的种子。

**【战斗与资源细则】**
弹药稀缺，近战与砖块吸引是常规。潜行勒杀可清咔咔前置。孢子区必须戴面罩。工艺台升级武器。AI写战斗应先报：子弹几发、燃烧瓶几枚、是否有面罩、敌人是人还是感染者。对人战斗往往比感染者更残酷——猎户与食人者会喊话、会乞求、会突然开枪。

**【微观事件库】**
①隔离区配给暴动。②黑市假药识别。③比尔陷阱误伤的道德补偿。④高中教室找到学生遗物。⑤匹兹堡狙击点反杀。⑥杰克逊站岗一夜。⑦冬季鹿猎与大卫帐篷。⑧手术室是否按下录音键留下真相。

**【AI雷区补强】**
乔尔很强但会被伏击败；艾莉会怕会哭会狠；火萤不是纯邪也不是纯善；疫苗成功与否本就不确定；不要把末日写成爽快割草；不要忽略季节与饥饿。




**【感染者阶段战术手册】**
奔跑者：速度快，怕火与噪音管理失败。潜行者：埋伏。咔咔：盲但听，要用砖块声东击西，近战从后方，正面硬刚易死。巨无霸：燃烧破甲+霰弹/燃烧箭，拉扯柱子与门，禁止近身连刀。孢子区：面罩滤毒，破损即换。尸潮与人类伏击常叠加，最危险的是「清完感染者刚换弹时被人爆头」。

**【公路旅程物资表】**
子弹、布、酒精、刀刃、电池、药品、食物。冬天额外要保暖与卡路里。比尔镇后有车，匹兹堡可能丢车。工艺升级顺序影响中后期手感。契约者带来的现代知识（例如更高效燃烧瓶）可以是合法奖励，但会被乔尔用眼神计价。

**【父女羁绊节拍】**
不信任→共同杀戮→笑话与磁带→差点遗弃→承认→为她屠尽实验室→谎言。每一节拍都可插入契约者：你是促成承认的人，还是促成谎言的人，或是把真相塞进艾莉背包的人。

**【火萤伦理辩论稿】**
正方：一人换人类。反方：不确定成功、未征得同意、革命组织也有私刑。乔尔的选择不是正确答案，是父亲答案。玛琳的选择不是纯恶，是指挥官答案。任务应让玩家脏手。

**【续作阴影（可选支线，不锁主线）】**
若世界时间推进到第二部分，艾比的复仇、西雅图三天、圣巴巴拉奴隶营地等可作高阶延展，但默认切入仍在第一部。提到续作时保持悲剧循环主题：暴力制造暴力。

**【比尔镇高中内部结构】**
停车场陷阱、教室涂鸦、更衣室黑暗、体育馆中央巨无霸、二楼可投掷点、紧急出口可能被堵。卡片锚点战应写成「资源检查表」：瓶×2，弹×N，面罩完好？艾莉位置？比尔在对讲机里骂什么？

**【城市人格】**

开场白建议：「城市把骨头都长在外面。脚步声一乱，墙后的东西就知道你是热的。」

关键NPC立场：**乔尔/苔丝**——效率优先；**艾莉**（若已会合）——嘴硬怕黑；感染奔跑者——数量恐惧。

主线钩子/支线：钩子=活过第一夜；支线=取回物资包；支线=标记孢子区。

危险度/规避：中；规避咔咔巢穴。

任务方向/奖励：废土生存课；奖励=消音器零件、自制弓。

**三阶（咔咔楼 · 潜行课）**

切入身份/时点：以「乔尔临时搭档 / 艾莉的监护备份」切入多栋感染建筑清场期。

初始事件：咔咔的超声波咔咔作响；你必须用砖头声东击西。艾莉问：「如果我被咬了呢？」空气凝固。

开场白建议：「它们没有眼睛，却比有眼睛的人更会找你。你屏住呼吸时，连心跳都像叛徒。」

关键NPC立场：**乔尔**——教你勒杀；**艾莉**——学着开枪；**猎户侦察兵**——人类更可怕。

主线钩子/支线：钩子=无伤穿楼；支线=救下被困幸存者；支线=录下火萤频率。

危险度/规避：中高；群咔咔与伏击。

任务方向/奖励：潜行认证；奖励=钉棒、砖块战术、艾莉的一句谢谢。

**四阶（高中巨无霸 · 锚点战）**

切入身份/时点：比尔小镇，以「修车队第三人 / 比尔不情愿收留的外来者」切入高中体育馆线。

初始事件：体育馆门后是巨无霸。比尔在对讲机骂「我说过别进学校」。燃烧瓶只有两枚，散弹更少。你要负责拉仇恨、投瓶，还是护艾莉撤梯？

开场白建议：「菌毯从天花板垂下来，像婚礼的纱。巨无霸一踏步，篮筐框架都在落灰。比尔的陷阱管不到这里——这里要火，要铅，要你们还想活的那点狠。」

关键NPC立场：**乔尔**——指挥火力；**艾莉**——投瓶与恐惧；**比尔**——远程冷嘲但给弹药；**巨无霸**——四阶标尺怪物。

主线钩子/支线：钩子=卡片锚点击杀/逼退巨无霸；支线=取电池与零件；支线=发现弗兰克遗物交给比尔。

危险度/规避：高；近战即死，优先燃烧破甲。

任务方向/奖励：小镇放行；奖励=车用零件、比尔军火、燃烧瓶配方精通。

**六阶（盐湖城医院 · 父与文明）**

切入身份/时点：以「火萤外围医护 / 乔尔潜入的影子 / 玛琳谈判代表」切入手术即将开始时。

初始事件：医生说「打开她的颅骨」。乔尔的呼吸在防毒面罩里像风暴。你站在走廊：按警报让全层戒备，还是打开消防通道让他过去，或偷走麻醉记录让艾莉日后知道真相。

开场白建议：「消毒液的味道盖不住枪油。世界把解药放在一个女孩的脑里，男人把女孩放在自己的谎里。你的手按在哪边门把上，历史就往哪边倒。」

关键NPC立场：**乔尔**——不惜屠尽；**玛琳**——要疫苗；**主刀医生**——职业信念；**艾莉**——麻醉中的不知情；火萤卫队——六阶体系火力。

主线钩子/支线：钩子=终局闯关/改写谎言结构；支线=保留一份免疫组织样本（非脑）；支线=让艾莉醒来前听到真话。

危险度/规避：极高·贴近顶点——火萤整层军事力量；条件性胜利=带走艾莉或拿到可验证的替代方案，不求「完美拯救全人类」。

任务方向/奖励：终局在场者；奖励=火萤研究残卷、乔尔的亏欠、艾莉未来追问的钥匙、永远争议的道德重量。
