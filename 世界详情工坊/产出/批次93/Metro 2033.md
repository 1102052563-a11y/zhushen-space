# Metro 2033

## 剧情

**【作品来源】**
《Metro 2033》（俄语《Метро 2033》）为俄罗斯作家德米特里·格卢霍夫斯基（Dmitry Glukhovsky）2002 年起网络连载、2005/2007 年纸质出版的后启示录科幻小说；2010 年由乌克兰 4A Games 改编为同名第一人称生存射击游戏（THQ 发行，后有 2014 Redux 重制）。本档案以**游戏主线正典**为默认舞台（卡片锚点=导弹打击黑怪），小说设定作世界观补强。系列后续含《地铁：最后的曙光》《地铁：离乡》与小说《2034》《2035》。文风：核冬天、隧道政治、资源匮乏、人性与异类的道德抉择。

**【世界定位】**
2013 年全球核战后二十年，莫斯科地表成辐射废土，幸存者困居莫斯科地铁站台聚落。主角阿尔乔姆（Артём / Artyom）自展览馆站（原 VDNKh，游戏称 Exhibition）出发，携游骑兵猎人（Hunter）的狗牌前往波利斯（Polis）求援，对抗被称为「黑怪 / 黑暗者（Dark Ones）」的神秘异种。一句话：隧道里的人类内战与地表核尘之间，一次可能误杀「新智慧」的导弹抉择。卡片锚点：**阿尔乔姆引导导弹击中黑怪巢穴，核弹再次在废土上爆炸**（亦可在慈悲路线下砸毁激光制导，放生黑怪）。

**【世界观 · 力量体系】**
本世界没有灵气修真，战力来自**枪械与军用子弹、防毒面具与滤芯、辐射耐受、变异体生物强度、以及黑怪的精神冲击**。货币即战前 5.45×39mm 军用级子弹（可交易也可当高伤子弹）。地表与部分隧道必须佩戴防毒面具并更换滤芯，否则窒息/中毒。死亡规则：人类一枪一刀可致命；变异体靠数量与再生；黑怪以幻觉与精神压迫瓦解心智，非常规枪战可解。

威胁/战力层级（对照《阶位战力图鉴》，宁低勿高）：

1. **站台平民 / 商贩 / 孩童**  
   无武装或持简陋刀具。乐园对照≈一阶下沿。

2. **站台卫兵 / 商队护卫 / 普通匪徒**  
   改装手枪、霰弹、自制步枪；能守关卡、劫掠隧道。乐园对照≈一阶～二阶初。

3. **红线正规军 / 第四帝国士兵 / 有组织匪帮**  
   制式步枪、机枪巢、装甲列车片段；可发动站台级攻防。乐园对照≈二阶。

4. **游骑兵（Rangers of the Order）精英 / 猎人级单兵**  
   高训练、精良改装武器、地表作战经验；可在辐射区与变异群中穿插。乐园对照≈三阶。

5. **大型变异体与隧道灾厄**  
   鼠潮、守护者（Library Guardians / 图书馆巨兽）、诺萨利斯（Nosalis）群、地表雪原猛兽：能撕开小队、封锁整段隧道或图书馆大厅。乐园对照≈三～四阶。

6. **战略设施与黑怪群体**  
   秘密导弹库 D6、奥斯坦金诺塔激光制导、黑怪巢穴与其精神网络：可改写莫斯科北缘战局，导弹齐射覆盖植物园级区域。乐园对照≈五～六阶。

7. **世界顶点（本档仅开六阶）**  
   **黑怪集体 + 人类导弹反击能力**构成地区级灾难：一怒可令北线站台心理崩溃，或让废土再燃核火。本世界**顶点≈六阶**（地区级·镇国档下沿）。**无七阶及以上**。

乐园阶位映射：平民与杂兵≈一阶；站台武装与红线/帝国步兵≈二阶；游骑兵与猎人≈三阶；图书馆守护者与大型变异≈四阶；D6 再启动与导弹齐射≈五阶；黑怪巢穴/精神网络 + 核打击终局≈六阶。世界顶点=**黑怪群体（及掌握其坐标的导弹权）**，低阶切入一律规避顶点·情报优先；终局胜利可为「击毁」或「放生」的条件性选择，严禁把黑怪写成「被削弱的零战力」。

**【地理 · 舞台】**
- **展览馆站 / Exhibition（原 VDNKh）**：阿尔乔姆故乡；黑怪幻觉与袭击的前线，猎人在此托付狗牌。
- **里加方向与中继站**：商队与走私路线，波旁（Bourbon）接头点。
- **地表莫斯科废墟**：冰雪、辐射尘、废弃街垒；短距穿越需滤芯与导航。
- **红线（Red Line）**：斯大林主义集权站台群；安德鲁铁匠（Andrew the Blacksmith）可提供偷渡。
- **第四帝国（Fourth Reich）**：新纳粹控制区；种族清洗与处决台，帕维尔（Pavel）、乌尔曼（Ulman）救援点。
- **波利斯（Polis）**：地铁「首都」，图书馆与知识贵族；议会否决救援展览馆，米勒（Miller）上校提出 D6 方案。
- **莫斯科国立图书馆地表废墟**：寻找 D6 地图，遭遇图书馆守护者。
- **D6（莫斯科地铁二号线传说中的秘密指挥所）**：导弹与战略设施，再启动后可打击植物园巢穴。
- **奥斯坦金诺塔（Ostankino Tower）**：安装激光制导、俯瞰黑怪巢穴的终点舞台。
- **植物园 / 植物园站方向巢穴**：黑怪聚居；导弹落点即卡片锚点。

写场景强调：煤油灯、滤芯倒计时、军用子弹叮当声、隧道滴水与远处号角、红线标语与帝国臂章、地表永恒黄昏。

**【世界剧情线】**

○ **前史：2013 核火与地铁方舟**  
世界大战的核交换在数小时内抹去地表文明。莫斯科地铁因冷战民防设计成为最大避难所。一代人在站台长大，不知蓝天；子弹成为货币，蘑菇与猪肉养殖维生。意识形态裂成汉萨同盟（Hanza）贸易网、红线集权、第四帝国排外、以及中立调停的游骑兵团。地表则被辐射与变异生物占据，偶有「异常」与幻觉区。

○ **展览馆告急与猎人的托付**  
2033 年，展览馆站屡遭黑怪精神冲击与实体袭击。游骑兵猎人抵达，评估威胁后决定向上级汇报，临行将狗牌交给养子阿尔乔姆：若他不归，阿尔乔姆须把狗牌送到波利斯。猎人失踪/失联后，阿尔乔姆决心完成使命。

○ **商队、波旁与可汗**  
阿尔乔姆随商队赴里加方向，途中遭遇精神冲击，唯他受影响较轻。走私者波旁带他穿越站台与地表，却在匪帮伏击中身亡。神秘旅者可汗（Khan）救出阿尔乔姆，带他穿过「闹鬼」隧道与交战站台，讲述隧道中的「声音」与黑怪或许并非单纯魔物。可汗建议他联系红线下的安德鲁。

○ **红线偷渡与帝国绞架**  
安德鲁协助阿尔乔姆潜入/溜出红线控制区，却被第四帝国俘获，押上处决台。游骑兵帕维尔与乌尔曼劫法场；帕维尔在护送中牺牲。阿尔乔姆独自继续，途中加入阻挡变异潮冲击波利斯外围的防线，救出男孩，换取前往地表与波利斯的通路。

○ **波利斯议会否决与 D6 备选**  
乌尔曼将阿尔乔姆引见游骑兵上校米勒。波利斯议会拒绝出兵救援展览馆。米勒提出备选：启动传说中的 D6 导弹库，摧毁植物园方向的黑怪巢穴。阿尔乔姆加入游骑兵，与米勒等在国立图书馆废墟夺回 D6 地图，遭图书馆守护者追杀。

○ **再启动 D6 与奥斯坦金诺**  
小队潜入并重启 D6 指挥中心。阿尔乔姆与米勒攀登奥斯坦金诺塔安装激光制导。黑怪发动强烈幻觉，试图沟通——内容暗示它们可能是突变后的新智慧生命，试图与人类建立联系而非单纯屠杀。

○ **终局：导弹或慈悲（卡片锚点）**  
- **正典/默认结局**：阿尔乔姆放任导弹发射，黑怪巢穴被毁，核火再一次在废土上升起——人类「安全」了，却可能消灭了唯一能理解/拯救彼此的新物种。  
- **替代结局（需全程慈悲道德点）**：阿尔乔姆砸毁制导装置，拒绝再开核火，承认黑怪的沟通意图。  
两结局均昂贵：前者是种族级误杀的原罪，后者是冒全地铁风险的赌命。卡片锚点对应前者。

○ **蝴蝶效应边界**  
契约者可改：某次商队是否全灭、某名匪首生死、是否救下某站孩童、道德点走向。不可无代价改写：核战已发生的既定前史、军用子弹货币逻辑、黑怪作为「可沟通异类」的终局题眼、D6 导弹真实存在。

**【主要人物】**
- **阿尔乔姆（Artyom）**｜性格：沉默、执拗、对战前明信片怀旧，逐渐学会选择而非只服从。｜装备·能力：改装步枪/霰弹、左轮、气动弩、防毒面具与滤芯、猎人狗牌。｜人物弧光：站台青年→波利斯求援者→游骑兵→决定黑怪存亡的扳机手。｜立场关系：忠于展览馆与养父一脉；对米勒既敬又疑；对黑怪从恐惧到动摇。
- **猎人（Hunter）**｜性格：冷酷专业、使命优先。｜装备·能力：游骑兵顶级单兵装具。｜人物弧光：展览馆评估→托付狗牌→失踪成为阿尔乔姆出发的原因。｜立场关系：阿尔乔姆的引路与精神标尺。
- **米勒（Colonel Miller）**｜性格：务实强硬、以人类存续为最高伦理。｜装备·能力：游骑兵指挥权、D6 作战方案。｜人物弧光：议会失败后的备选推手→塔上并肩者。｜立场关系：招揽阿尔乔姆；推动导弹方案。
- **乌尔曼（Ulman）**｜性格：干练、黑幽默。｜装备·能力：游骑兵战术。｜人物弧光：帝国救人→引见米勒→图书馆与 D6 行动队员。｜立场关系：阿尔乔姆在体制内的可靠接点。
- **帕维尔（Pavel）**｜性格：豪爽、重承诺。｜装备·能力：突击与近战。｜人物弧光：帝国救场→护送途中牺牲。｜立场关系：以命换阿尔乔姆前路。
- **波旁（Bourbon）**｜性格：精明走私者、嘴硬心软。｜装备·能力：地下路线与交易网。｜人物弧光：雇佣向导→匪帮之死。｜立场关系：阿尔乔姆离开展览馆后的第一任向导。
- **可汗（Khan）**｜性格：神秘、哲思、倾听隧道「声音」。｜装备·能力：异常区生存知识、精神耐性。｜人物弧光：救波旁残局→带路→暗示黑怪非单纯魔物。｜立场关系：打开阿尔乔姆认知的钥匙人物。
- **安德鲁铁匠（Andrew the Blacksmith）**｜性格：务实、反感红线高压。｜装备·能力：锻造与地下关系。｜人物弧光：红线内应→协助偷渡。｜立场关系：可汗线的下家。
- **展览馆站长（苏霍伊 / Sukhoi 等养父系角色，游戏叙述中的站台权威）**｜性格：护短、守土。｜装备·能力：站台防卫指挥。｜人物弧光：送养子远行。｜立场关系：阿尔乔姆的情感锚。
- **黑怪 / 黑暗者（Dark Ones）**｜性格：非人、试图沟通、被人类解读为攻击。｜装备·能力：精神幻觉、群体感应、巢穴级存在。｜人物弧光：从「怪物」到「可能的新人类/新智慧」。｜立场关系：终局的「敌」或「被误杀的他者」；**六阶顶点侧**。
- **红线军官与政委群**｜性格：纪律、宣传、清洗异己。｜装备·能力：正规军编制。｜人物弧光：隧道内战一方。｜立场关系：可被利用的通道，不可信任的主人。
- **第四帝国军官**｜性格：种族主义、残虐。｜装备·能力：处决与排外武装。｜人物弧光：阿尔乔姆差点死于其绞架。｜立场关系：绝对敌对。
- **汉萨商人与站台议长们**｜性格：利润与稳定优先。｜装备·能力：贸易网与关税。｜人物弧光：波利斯政治的背景噪声。｜立场关系：可买通情报，难买通出兵。

**【势力图谱】**
- **展览馆站**：北线前线聚落；求生第一，政治第二。
- **游骑兵团（Rangers / Order）**：中立精锐，守波利斯与战略秘密；代表米勒、猎人、乌尔曼。
- **波利斯**：知识与政治中心；议会保守，惜力不援展览馆。
- **红线**：集权意识形态政权；控制若干枢纽站，与帝国死敌。
- **第四帝国**：新纳粹站台联盟；清洗「不纯」人口。
- **汉萨同盟**：贸易与关税联盟；中立赚钱。
- **匪帮与奴隶商**：隧道寄生者；波旁死于此类。
- **黑怪巢穴**：地表/植物园方向的异种社会；人类官方叙事中的终极威胁。

**【贵重物品】**
- **猎人的狗牌**：通行游骑兵体系的信物，阿尔乔姆身份证明。
- **军用级 5.45 弹药**：硬通货与高伤弹药，交易与战斗双重用途。
- **防毒面具与滤芯**：地表与毒区刚需，滤芯倒计时=生命倒计时。
- **D6 地图与启动权限**：图书馆夺得的战略钥匙，决定导弹是否可用。
- **奥斯坦金诺激光制导装置**：终局开关；砸毁或启动决定黑怪命运。
- **气动武器与改装枪械**：地铁特色军工，噪音与弹药经济的权衡。
- **战前明信片**：阿尔乔姆的精神图腾，象征已死的地表世界。

**【隐藏剧情 · 伏笔】**
- 黑怪的幻觉实为沟通尝试；正典结局的残酷在于「胜利即灭绝可能的盟友」。
- 可汗线暗示隧道「异常」与黑怪精神场相关，人类意识形态内战反而更致命。
- D6 作为冷战遗产，说明旧世界毁灭机器仍可被新世界年轻人扣下扳机。
- 道德点系统隐藏：布施、倾听、不滥杀影响能否看见「放生」选项。
- 小说与游戏细节差异（站名、同伴）存在；以游戏流程写任务，以小说补「地铁社会学」。

**【大事记时间线】**
- 2013 → 全球核战，莫斯科地表弃守，地铁成为方舟。
- 2013–2030s → 站台国家化：汉萨、红线、帝国、游骑兵成型。
- 2033 初 → 展览馆遭黑怪冲击；猎人托付狗牌。
- 2033 旅程 → 阿尔乔姆经里加线、红线、帝国、波利斯、图书馆、D6。
- 2033 终 → 奥斯坦金诺制导；导弹摧毁巢穴（正典）或砸毁制导（慈悲）。
- 此后 → 地铁政治继续；黑怪存灭成为阿尔乔姆永久原罪或赌注。

**【叙事基调 · 雷区】**
基调：阴冷、压抑、资源焦虑、政治讽喻与存在主义；枪战服务于恐惧而非爽感。  
忌：把地铁写成爽快清怪副本；忌无视滤芯/辐射规则；忌把黑怪写成无智慧丧尸；忌低阶让玩家手搓核弹；忌抹掉红线/帝国的意识形态残酷；忌用空话套话代替站名与人名。最早切入锚点：猎人离开展览馆前后，或阿尔乔姆刚拿到狗牌之时。


**【世界剧情线·细部展开一：展览馆与猎人】**
展览馆站原属莫斯科地铁卡卢日斯科-里日斯卡亚线 VDNKh 站，核战后被居民改称为「展览馆」。站内靠蘑菇种植、猪肉养殖与军用子弹交易维生。黑怪的精神冲击表现为突如其来的幻觉、耳鸣与集体恐慌，部分人因此自杀或发狂。猎人作为游骑兵精锐到站评估后，判断黑怪威胁超出单站防卫能力，决定上报波利斯。他将狗牌交给阿尔乔姆并留下指令：若自己未归，阿尔乔姆必须把狗牌送到波利斯交给上级。猎人离站后失联，站内再次遭遇冲击，阿尔乔姆决定离开故乡完成托付。此段确立主题：个人对故乡的义务，高于隧道政治的口号。

**【世界剧情线·细部展开二：商队、波旁、地表与可汗】**
阿尔乔姆加入前往里加方向的商队，途中遭遇精神冲击，同伴短暂失能而他受影响较轻——暗示他与黑怪存在特殊联系（小说与游戏均强调其「被选中」感）。走私者波旁以带路费为条件引导阿尔乔姆穿越站台与地表废墟；地表满是冰雪、辐射尘与废弃车辆，滤芯倒计时成为生存节拍器。波旁在匪帮伏击中身亡。神秘旅者可汗救出阿尔乔姆，带他经过被认为「闹鬼」的隧道段落，讲述隧道中的声音、异常与「不要开枪打空气」的生存哲学。可汗暗示黑怪或许不是单纯魔物，人类内战才是更致命的毒素。他指引阿尔乔姆联系红线控制区内的铁匠安德鲁。

**【世界剧情线·细部展开三：红线、第四帝国与游骑兵救援】**
红线以斯大林主义宣传与纪律维持站台群，对外战争与对内清洗并行。安德鲁协助阿尔乔姆偷渡，却在离开红线后被第四帝国俘获。第四帝国以种族纯净为意识形态，将「不纯」者与俘虏送上处决台。游骑兵帕维尔与乌尔曼劫法场救出阿尔乔姆；帕维尔在后续护送中牺牲，用生命换通路。阿尔乔姆途中加入阻挡变异潮冲击波利斯外围的临时防线，救出男孩，换取地表通行与进入波利斯的资格。此段展示地铁社会学：意识形态、种族清洗、精锐中立力量与平民互助并存。

**【世界剧情线·细部展开四：波利斯否决、图书馆与 D6】**
波利斯是地铁的「首都」，由知识贵族与多派代表组成议会。阿尔乔姆出示猎人狗牌并陈述展览馆危机，议会仍拒绝出兵——北线救援成本高于政治收益。游骑兵上校米勒提出备选方案：启动冷战遗产 D6 秘密指挥所的导弹，摧毁植物园方向的黑怪巢穴。阿尔乔姆加入游骑兵，与米勒、乌尔曼等进入莫斯科国立图书馆废墟寻找 D6 地图。图书馆内盘踞「守护者」类巨型变异体，书架与穹顶成为垂直战场。夺图后小队重启 D6，导弹系统从沉睡中苏醒——旧世界的毁灭机器再次听命于新世界的年轻人。

**【世界剧情线·细部展开五：奥斯坦金诺与双重终局】**
阿尔乔姆与米勒攀登奥斯坦金诺电视塔安装激光制导。黑怪发动强烈幻觉，画面与情感冲击试图传达「我们不是敌人」。正典结局中阿尔乔姆放任导弹发射，巢穴被毁，核火再一次升起，展览馆暂时安全，人类却可能灭绝了唯一能沟通的新智慧。慈悲结局要求玩家在全程做出足够的人道选择（布施、倾听、不滥杀等），才能在塔上选择砸毁制导装置。两结局都昂贵：前者是种族级误杀原罪，后者是把全地铁命运押在未知沟通上。卡片锚点对应正典导弹打击。

**【地理补遗·站台政治生态】**
汉萨同盟以贸易与关税维生，尽量中立赚钱；红线与第四帝国长期交战；匪帮在无人区设卡收「过路费」；游骑兵在波利斯与战略点维持最低限度秩序。军用级 5.45 弹药既是货币也是高伤弹药，导致「开枪就是烧钱」。防毒面具滤芯、手电筒手摇充电、气体检测成为日常科技。地表异常区可造成精神伤害，与黑怪精神场可能同源。写场景时应让 AI 先问：契约者在站内还是地表？滤芯还剩多久？附近是红线、帝国还是汉萨？

**【人物关系网补强】**
阿尔乔姆对养父与展览馆的忠诚是其行动第一引擎；对猎人是敬畏与继承；对米勒是上下级与伦理冲突；对可汗是认知打开者；对乌尔曼是战友式可靠；对帕维尔是亏欠；对波旁是短暂雇佣中的人性温度。黑怪作为集体存在，不与单一人类建立「反派头目」式对话，而是以幻觉网络施压。红线政委与帝国军官是可替换的制度面孔，真正持久的威胁是制度本身与导弹按钮。

**【贵重物品使用细则】**
猎人狗牌：在游骑兵体系中等同介绍信，遗失则阿尔乔姆难入核心行动。军用弹药：可在商人处换武器改装、过滤器、药品。D6 地图：图书馆夺得后成为终局钥匙，副本中应被多方觊觎（匪帮、帝国间谍、红线侦察）。激光制导装置：安装完成前可被破坏、校准偏移或被黑怪精神干扰。气动武器：噪音低但需充气，适合潜行；霰弹与燃烧瓶在对付大型变异时更有效。战前明信片：无战斗数值，却是阿尔乔姆拒绝彻底野蛮化的精神锚。

**【可介入微观事件库（展览馆—波利斯段）】**
①商队护卫：保护弹药箱不被匪帮劫走，报酬是滤芯。②红线证件伪造：安德鲁需要一张通行证底版。③帝国处决名单：偷偷改掉一个名字，可能救下未来的向导。④图书馆禁书：取回一本战前生物学期刊，为慈悲结局提供「黑怪同源突变」的叙事证据。⑤波利斯议会旁听：用情报交换让一名议员改变投票（通常失败，但可埋线）。⑥可汗的静默试炼：在异常隧道中坚持不开枪，换取精神防护提示。⑦D6 内部道德：是否处决投降的旧世界自动防卫残存操作员（若存在）。⑧塔上最后三十秒：夺米勒对讲、砸装置、或亲手确认坐标。

**【叙事执行要点（给 AI）】**
优先写资源焦虑与政治，而非无脑清怪。黑怪出场以幻觉与不安为主，少写「血厚 BOSS 条」。任何「拯救人类」的口号都要被可汗式质疑打断一次。终局必须让玩家感到手脏：开火是罪，不开火也可能是罪。




**【站台日常生活档案】**
展览馆的一天从清点滤芯与弹药开始。孩子们在轨道上踢球，老人讲述战前公园的喷泉。广播偶尔插播红线胜利或帝国暴行的谣言。商人用天平称军用子弹，价格随隧道战事浮动。夜间有人听见「敲管子」的暗号，可能是商队，也可能是匪帮。黑怪冲击来时，灯光乱闪，有人跪地哭，有人开枪打墙壁。阿尔乔姆收集战前明信片，试图在纸上重建蓝天。这种日常是旅程的对照：外面更冷，但里面也并不仁慈。

**【红线与帝国的战争美学】**
红线站台挂着领袖像与标语，队列整齐，审查口音与成分。第四帝国涂着十字与口号，检查外貌与血统叙事。双方交火时平民躲进隧道盲肠，用棉被堵弹孔。游骑兵穿行其间像穿过两头野兽的牙缝。契约者若穿错臂章，会被两边同时当间谍。战争没有胜利，只有下一次换防。

**【图书馆战役分镜】**
进入图书馆前要换滤芯、检查弹匣、约定集合点。第一层是散落的书籍与小型变异；中层出现守护者的脚步震动；上层地图室光线从破穹洒下，美得像陷阱。夺图时应用潜行优先，强突会导致队友死亡与地图损坏。逃出时地表风雪灌入防毒面具，镜片结雾，米勒的手电成为唯一方向。

**【D6 内部结构想象（任务可用）】**
锈蚀电梯、冷战海报、仍在闪烁的服务器、需要手摇发电的闸门、可能存在的自动机枪巢。重启导弹需要多人同时授权：米勒、技术员、阿尔乔姆。契约者若持技术身份可成为第三授权，从而拥有延时/取消的杠杆。取消会被视为叛国；延时可能让黑怪先发精神冲击。

**【黑怪沟通内容的叙事边界】**
幻觉可呈现：童年、死者、展览馆的安好、以及「我们想说话」。不可写成黑怪突然说流利现代俄语开记者会。沟通是残缺的、压迫的、令人崩溃的。正典阿尔乔姆仍开火，是因为恐惧与米勒的人类中心伦理压过了不确定的善意。

**【道德点对应表（任务设计）】**
救助乞讨者、聆听可汗、不杀投降匪徒、给孩子玩具/子弹（非武器）等行为累积「倾听」权重；反之滥杀、嘲讽难民、对幻觉开枪成瘾则锁定正典导弹。契约者可成为道德点的外部催化剂。

**【Metro 2033·场景零件库（供正文调用）】**
写任何段落先选 3 个零件：气味、声音、制度、道具、伤口、价格。地铁/孤岛/双城/日记/末日各有专属零件，禁止串味。人物开口必须带身份立场，禁止「路人甲说了一句有深意的话」式空壳。战斗后写政治后果：谁涨价、谁封路、谁要你的血或日记或宝石。

**【Metro 2033·同盟与背叛税率】**
本世界盟约默认脆弱。结盟时必须先谈清楚：退出条件、分赃比例、谁先死谁断后。背叛不是随机事件，而是资源不足时的理性选择。契约者若以乐园外挂碾压，本地人会恐惧并集结更强规则（项圈、议会、神、火萤、导弹）反噬。

**【Metro 2033·时间点状态机】**
AI 在任意时点应能回答：主角现在在哪、谁已死、关键道具在谁手里、下一场名场面是什么。若契约者改写了局部，用「追杀/取代/更惨」回流，而不是假装原作没发生。


**【终局双结局并行脚本（任务官用）】**
正典轨：制导锁定植物园巢穴→倒计时→导弹升空→闪光与冲击波→展览馆欢呼与阿尔乔姆的空白。任务成功条件可设为「保护制导直至发射」或「记录发射全过程并护送阿尔乔姆返回」。失败条件：制导被毁、米勒死亡导致授权中断、黑怪精神冲击使小队自相残杀。
慈悲轨：需前置人道行为达标→塔上出现砸毁互动→黑怪幻觉转为平静→地铁陷入对未知的恐惧。任务成功条件：「砸毁且撤离」。失败条件：被米勒开枪制止、装置自毁失败仍发射。
两轨奖励不同：正典给军功与阴影；慈悲给异常耐性与政治追杀（被骂叛徒）。

**【音频与视觉母题】**
滴水、远处号角、军用弹叮当、面具呼吸、俄语口语、红线广播、帝国军靴、书页翻动、塔风。视觉：煤油暖黄、雪地冷蓝、血与锈、明信片褪色。AI应反复调用这些母题保持世界统一。

**【与续作衔接的伏笔位】**
导弹后的政治真空、游骑兵声誉、阿尔乔姆的创伤、黑怪残存个体（若慈悲）均可接《最后的曙光》气质任务，但不强制进入续作剧情。



**【阿尔乔姆视角日志体补遗】**
「猎人走的第三天，展览馆的灯又闪了。有人说看见黑影在隧道口招手。我把明信片压在枕头下，像压住一种会飞走的东西。狗牌冰凉。米勒后来告诉我，游骑兵不相信招手，只相信弹道。可汗却说，有些招手是说话。我在塔上才懂：说话和弹道不能同时按下。」此类第一人称碎片可作开场白素材，但正文仍以第三人称为主。

**【商人与价格样例】**
一支滤芯=若干军用弹；改装瞄准镜更贵；信息（红线换防时间）可卖到一顿热汤加三发子弹。契约者若用乐园币硬闯经济，会引发「假币」怀疑与抢劫。建议用物资与人情交易。

**【变异体名录补充】**
诺萨利斯类掘隧道兽、鼠潮、图书馆守护者、地表雪原巨兽、毒雾中的盲行物。不同隧道生态不同，禁止写成统一「地铁怪」。黑怪外观常被描述为接近人形却异质，眼神与精神压是其主威胁。

**【波利斯图书馆政治】**
波利斯不仅有枪，有书。书是战前知识与身份。议会用知识傲慢拒绝北线求救，是文明的另一种残酷。米勒的D6方案是军人绕过文人的捷径，也是把文明重新交给导弹。

**【塔上三十秒的身体细节】**
风、冻伤、瞄准镜雾、阿尔乔姆的呼吸、米勒的命令、幻觉中的黑怪触碰、手指是否松开保险。终局应放慢，让选择有重量。



**【汉萨关税与中立的虚伪】**
汉萨标榜中立贸易，实则对弱势站台抽成，对强权让利。阿尔乔姆若无弹药，会被拦在关卡外活活等死。中立是强者的语言。

**【可汗的哲学句可复用】**
「子弹解决不了的东西，存在于子弹之间。」「有人把恐惧叫做怪物，好开枪。」这些句子可在异常区回响，不必每次解释。



**【补完块1·Metro 2033】**
展览馆防卫队的编制通常包括门岗、隧道巡逻、民兵支援与儿童疏散组。黑怪冲击来临时，标准作业是熄灯、戴防护、避免对幻觉开枪浪费弹药。阿尔乔姆作为青年民兵参与过多次无效射击，这也是他后来能在精神冲击中保持相对清醒的经验来源之一。猎人评价展览馆「还能守，但守不了永远」，这句话推动了求援。

波旁带路时会讲黑市笑话、抱怨红线关税、警告「别碰地上的玩偶」。这些闲聊是世界的血肉。他的死应突然：笑声还在，血已经在。可汗接手后气氛从走私喜剧转为灵异公路片，节奏必须切换。

第四帝国处决台的仪式包括宣读「不纯」罪状、群众（或士兵）注视、行刑。帕维尔与乌尔曼的救援是游骑兵招牌行动：短促、准、少废话。帕维尔之死让阿尔乔姆明白游骑兵不是不死英雄。

D6 的导弹不是科幻激光，是旧世界战略武器，带着锈与官僚编号。发射时地下会震动，站台会误以为又一次核战。正典终局的残忍在于：人们欢呼的「胜利」听起来像2013年的回放。

契约者六阶任务结算时，应给出「你让世界更安全还是更孤独」的非数值评价，而不是只给经验。黑怪巢穴的烟与展览馆的酒同样重要。

## 阶位切入点

> 阶位↔威胁：本世界工单仅覆盖**六阶**。六阶≈D6 导弹再启动、奥斯坦金诺制导、黑怪巢穴与核打击终局。世界顶点（黑怪集体 + 导弹权）为六阶存在，切入一律「条件性胜利 / 情报与抉择优先」，严禁贬低顶点式硬刚。

**六阶（D6 与奥斯坦金诺 · 黑怪终局线）**

切入身份/时点：契约者以「游骑兵新晋队员 / 波利斯派往 D6 的技术顾问 / 可汗引荐的异常区观察员」身份切入，锚定波利斯议会否决出兵之后、米勒启动 D6 方案前后，直至奥斯坦金诺塔激光制导安装完成。

初始事件：你随小队在国立图书馆废墟夺图，巨兽在书架间咆哮；地图到手时通讯里传来展览馆又一次精神冲击的惨叫。米勒下达死命令：重启 D6，把坐标打在植物园巢穴。可汗（或他的传话人）却塞给你一块刻着奇怪符号的弹壳——「它们在说话，你听不听？」你必须在「人类安全」与「可能的种族灭绝」之间，抢在导弹点火前做出可见行动。

开场白建议：「莫斯科的风里全是铁和雪。防毒面具起雾的那一秒，你听见不是枪声，是有什么东西在颅骨内侧轻轻敲了三下。米勒的手套按在你肩上，说游骑兵不靠幻觉打仗。可当你爬上奥斯坦金诺的钢铁脊骨，激光红点落在植物园方向时，你突然分不清：准星尽头，是怪物，还是另一种孩子。」

关键NPC立场：**阿尔乔姆**——执着送狗牌与救故乡，终局动摇时可被说服或被煽动；**米勒**——人类中心主义，视放生黑怪为背叛；**乌尔曼**——执行命令但保留冷幽默与人性缝隙；**可汗**——主张倾听，反对无脑开火；**黑怪**——以幻觉施压/沟通，非谈判桌上的人类政客；波利斯议会残余代表——要结果不要良心。

主线钩子/支线：钩子=卡片锚点「导弹打击黑怪巢穴」的触发或阻止；支线 A=在 D6 内部保全或销毁某组备用弹头数据；支线 B=把展览馆幸存者名单送入波利斯广播；支线 C=收集足够「慈悲证据」（救助平民、拒绝虐俘）以解锁砸毁制导的叙事权限；支线 D=从图书馆取回战前生物学档案，证明黑怪神经结构与人类同源突变。

危险度/规避：危险度极高·贴近顶点——图书馆守护者、D6 内部防卫、地表辐射与黑怪精神场叠加；**超阶式误解禁止**：黑怪不是「战力归零的靶子」，导弹也不是玩具。条件性胜利：改写制导、调偏弹着点、伪造「已摧毁」报告、或在正典线上确保阿尔乔姆心理不被完全碾碎。

任务方向/奖励：决定巢穴存灭与地铁道德史；奖励取向=游骑兵认证、D6 权限碎片、异常区耐受训练、军用弹药储备、可汗一脉的精神防护物语（非无敌光环）、终局见证者身份（影响后续《最后的曙光》时间线的入场姿态）。


**【切入加厚·六阶战术与道德清单】**
D6 行动阶段可拆为三幕：图书馆夺图（变异与黑暗）、地下指挥所重启（机关、辐射、旧军械）、奥斯坦金诺安装（高空、风雪、精神冲击）。契约者若持技术身份，可在重启阶段输入「延时发射」或「弹着点偏移」代码，代价是被米勒视为叛徒。若持可汗引荐身份，可在幻觉中主动「回应」黑怪，换取短暂沟通片段，但可能被同伴当作已被精神污染。正典线上，任务成功条件可以是「确保导弹命中并记录罪证」；慈悲线上，成功条件是「砸毁制导且让阿尔乔姆活着离开」。奖励避免发「黑怪宠物」；应发游骑兵认证、异常耐性训练、军用弹药库坐标、展览馆疏散方案等。

补充开场白备选：「D6 的指示灯一颗颗亮起来时，你听见的不是希望，是2013年的回声。米勒说这是为了孩子。可汗说孩子也可能长成黑的。你握着校准扳手，指节发白——你知道自己在校准的不是激光，是人类还配不配活下去。」

补充NPC微观立场：**米勒**会在你质疑导弹时提高嗓门，但若你拿出展览馆儿童伤亡名单，他会沉默三秒再下令；**乌尔曼**会在行动后递烟，用笑话掩盖恐惧；**阿尔乔姆**若看见你善待难民，会在塔上多看你一眼才做决定。规避清单：不要在图书馆用火把无脑引怪；不要在地表摘下面具秀耐力；不要对黑怪喊「投降」式中二台词。
