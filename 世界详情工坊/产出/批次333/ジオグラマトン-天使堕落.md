# ジオグラマトン-天使堕落

## 剧情

**【作品来源】**
《ジオグラマトン》（Geographroll／Geographaton 等转写）为成人向作品名，清单后缀「天使堕落」点题天使／圣职存在坠入欲望的情感轴。公开详细条目与角色表极少，档案以「圣洁象征×堕落弧光」类型学撰写，真名不详处明确标注。搜笔趣阁未收录。

**【世界定位】**
架空圣域与人间夹缝：记载世界的「地理／圣典」装置让天使与人类视线相交，圣洁在注视中崩解。一句话：在彩窗光尘里，把「翼」的脱落写成「愿意留下」。

**【世界观 · 舞台设定】**
大教堂、抄经室、云上回廊、人间贫民街对照。超自然（天使、圣痕）只写身份羞耻与自由渴望。温度来自：羽毛掉落、圣水冰凉、第一次赤足踩地。

**【地理 · 生活舞台】**
- **圣堂彩窗下**：觉醒。
- **抄经室**：禁忌对话。
- **云阶／回廊**：逃亡。
- **人间巷口**：堕落完成或重生。

**【故事主线 · 情感线】**
**主线结构**  
某天使因「被人类看见」或触犯天律开始堕落→与人类主角产生依赖→翼与光轮的象征性剥离→选择返回或留在人间。成人向场景对应堕落节点。角色名不详。

**【可攻略角色 / 主要人物】**
人类见证者（不详·男主）。

堕落天使（不详·可攻略）｜高洁→渴望被触摸。

尚忠诚的天使同僚（不详）｜拉回或处决的压力。

圣职者（不详）｜解读圣典的人。

**【人际关系网 / 社团势力】**
天使—人类—天律三角。

**【情感事件 · 名场面】**
1. 彩窗初遇。 2. 第一根羽毛。 3. 抄经室夜谈。 4. 翼落。 5. 巷口选择。

**【隐藏剧情 · 真结局 · 伏笔】**
资料不足；标题中「ジオグラマトン」或与地理／圣名书写有关，待考。

**【氛围基调 · 雷区】**
圣堕美学、诗性。忌无情感的渎神堆砌。
