# 魔界骑士英格丽德-完全堕落
<!--meta lib=主库 tiers=三-->

## 剧情

**【作品来源】**
**英格丽德（イングリッド）**为 Lilith《对魔忍》宇宙核心魔族角色：初登场于《対魔忍ムラサキ 〜くノ一傀儡奴隷に堕つ〜》（2008），为吸血鬼始祖**エドウィン・ブラック**的护卫兼秘书，号「**魔界骑士**」；PIXY OVA《魔界骑士イングリッド》全四话（2009–2010）以其视点重述ムラサキ事件，弧光收束于屈辱与惨め终末。原画カガミ。**本卡片锚点「完全堕落」**锚定：骑士名誉、身份与自我叙事**双崩**后的终点态——不是单纯战败，而是她再也无法用「骑士道」解释自己的处境与选择。成人向条目只写剧情、战力、组织与任务介入；不写色情过程描写。

**【世界定位】**
人界与魔界门洞开的近未来日本。**ノマド**以**ブラック**为顶点；**カオス・アリーナ**为地下斗技与改造黑市。对魔忍**八津紫**、**井河さくら**、**井河アサギ**执行扫讨；**桐生佐馬斗**以寄生融合延命。英格丽德作为ブラック側近执行监视与肃清。卡片锚点：**完全堕落——名誉条归零后的骑士**。一句话：披风还在，骑士已经死了。

**【世界观 · 力量体系】**
力量本源：魔界贵族魔力、骑士武技、对魔忍忍法、魔科改造与妖魔融合、吸血鬼始祖权能。死亡：人类可死；妖魔融合可假死延命；ブラック系接近不死。

力量阶梯：
1. 普通人／观众≈一阶。
2. 下忍／低级佣兵／オーク≈二阶。
3. 五车生级对魔忍／英格丽德常规骑士态≈**三阶**（主战场）。
4. アサギ级／桐生融合后≈四阶边缘。
5. ノマド干部重镇≈五阶。
6. **エドウィン・ブラック**≈超阶巅峰：存在·情报优先／条件性胜利。

**乐园阶位映射**：杂兵≈一阶；佣兵下忍≈二阶；紫／さくら／英格丽德常规≈**三阶**；アサギ／融合桐生≈四阶边缘；ノマド重镇≈五阶；ブラック≈超阶。仅覆盖**三阶**。

**【地理 · 舞台】**
カオス・アリーナ（公开碎裂主舞台）、东京キングダム（逃亡与黑市）、五车学园（对魔忍侧）、ノマド隠れ家（改造与汇报）、魔界贵族领（已回不去的出身）。感官：血灯、沙、断裂的披风扣、项环、融合肉肿味、摄像机红点（「公开」的物理来源）。

**【世界剧情线】**
① **骑士身份的高点**  
英格丽德以一骑讨、忠义、敌卑劣自居；视朧为对照；对ブラック绝对忠诚。

② **ムラサキ事件中的挫败**  
追查ブラック细胞；アリーナ逼桐生；被融合紫击退；目标逃亡。挫败尚可用「敌卑劣」解释。

③ **完全堕落定义（卡片锚点）**  
完全堕落＝**骑士道叙事无法自我修复**：公开场合名誉碎裂＋本人被迫接受或执行曾最鄙视的手段（人质、药、群殴、拍摄、出卖同袍）＋ブラック侧评价从「可用骑士」滑向「仍可使用的工具」。不是战力归零，是身份归零。

④ **OVA 四话弧光对应**  
无残→被虐视点→屈辱誓约→惨め终末。任务世界用名誉条／工具条量化：完全堕落＝名誉条触底、工具条仍可被ノマド调用。

⑤ **对魔忍与桐生**  
紫／さくら／アサギ仍是正义侧扫讨；桐生仍是卑劣核心；他们不负责「拯救骑士」，只负责各自目标。

⑥ **终局**  
英格丽德可存活；ブラック不必口头宣判——一个眼神或一道新命令即可确认「骑士」头衔已成空壳。契约者可选择：留下公证记录、协助逃亡、或完成ノマド的回收。

⑦ **可介入节点**  
切断转播、销毁母带、伪造「仍在执行任务」的报告、救出被她波及的人质、把朧的阴谋证据交给第三方。

⑧ **蝴蝶边界**  
可改局部公开范围与逃亡成败；不可无代价抹去ブラック或改写始祖权能。

**【主要人物】**
- **イングリッド（英格丽德）**｜性格：骑士道崩解后的空洞、自尊残留、对朧的恨仍在。｜装备·能力：武技仍在≈三阶，但战术选择被胁迫改写。｜弧光：側近骑士→挫败→公开碎裂→**完全堕落态**。｜立场：契约者若能给她「最后一次合法一骑讨」可能短暂合作，否则视同围观者。
- **エドウィン・ブラック**｜超阶顶点；评价者与所有者。
- **桐生佐馬斗**｜卑劣核心；融合与实验。
- **八津紫**｜斧与不死；击退英格丽德的关键战力。
- **井河さくら**｜人质环线与影遁。
- **井河アサギ**｜后台总队长。
- **朧**｜推动公开羞辱与规则杀的对照干部。
- **八津九朗**｜软肋参考。
- **赌盘主／转播技师**｜「公开」的执行层。
- **キングダム蛇头**｜逃亡通道贩子。

**【势力图谱】**
ノマド（ブラック—英格丽德—朧）、对魔忍／五车、桐生个人线、竞技场资本、キングダム地下网。完全堕落发生在多方视线交叉处：观众要戏、朧要丑、桐生要体、对魔忍要敌、ブラック要结果。

**【贵重物品】**
骑士披风与纹章扣（碎裂物证）、ブラック细胞样本、项环与遥控器、融合核、转播母带、ノマド走廊权限卡。三阶奖励：母带残片、通行、临时医疗、低阶魔界金属。禁止始祖血。

**【隐藏剧情 · 伏笔】**
- 英格丽德对ブラック的恋心在完全堕落后仍可能是唯一未碎的轴。
- 朧是否在借公开羞辱削弱ブラック身边的骑士。
- OVA 终末的惨め是情感终点；任务世界允许「碎裂后仍可被记录为曾是骑士」。
- 桐生逃亡キングダム后，完全堕落的英格丽德是否被派去回收，是后续钩子。

**【大事记时间线】**
アリーナ扫讨 → 桐生融合 → 紫受创 → さくら人质 → 英格丽德对峙受挫 → 公开规则杀／拍摄／胁迫连环 → **完全堕落确认** → ブラック新命令或沉默 → キングダム方向。

**【叙事基调 · 雷区】**
黑暗动作＋骑士悲剧。忌色情过程；忌三阶斩始祖；忌「削弱战力」解释顶点；忌把完全堕落写成无代价黑化爽文。最早切入：转播灯亮起前十分钟，或名誉条第一次公开扣到红线时。

**【完全堕落 · 可验证物证】**
1. 转播母带时间码。
2. 断裂纹章扣。
3. 项环遥控日志。
4. 赌盘赔率突变记录（骑士从「必胜」被改成「必辱」）。
5. ノマド内部「工具化」调令抄本。
契约者带走任两件，即可在后续任务主张「曾见证完全堕落」。

**【与系列锚点边界】**
骑士陷落＝规则击败骑士道；妹妹威胁＝软肋；公开凌辱＝公开碎裂过程；姐妹相认＝羁绊抉择；**完全堕落＝碎裂后的终点身份**。

## 阶位切入点

> 阶位↔原作强度：杂兵≈一阶；佣兵下忍≈二阶；紫／さくら／英格丽德常规≈**三阶**；アサギ／融合桐生≈四阶边缘；ブラック≈超阶。宁低勿高。顶点存在·情报优先。仅覆盖三阶。

**三阶（转播灯亮 · 名誉触底）**

切入身份/时点：契约者以转播助理／赌盘审计／キングダム蛇头跑腿身份切入，锚定英格丽德被推入「公开碎裂」流程前夜。

初始事件：你拿到两样东西——**断裂纹章扣**的高清照片，以及**转播母带**的空白带仓钥匙。**朧**要你把灯全开；**イングリッド**用眼神求你「至少别开脸」；**桐生佐馬斗**的人只要实验体还活。

开场白建议：「红点在镜头上亮起来的时候，英格丽德还在整理披风角。观众已经开始喊要看骑士跪下。你拇指按在母带仓上——按下去，她会变成工具；弹开，你会被ノマド记名。」

关键NPC立场：**イングリッド**——要最后一点骑士体面；**朧**——要公开丑；**桐生佐馬斗**——要活体；**八津紫**——要打断桐生，顾不上拯救骑士；**井河さくら**——人质环线；**赌盘主**——要赔率。

主线钩子/支线：主线：阻止或完成「完全堕落」的公开确认，并带走物证；支线：剪灯、换带、伪造调令、给キングダム开一口。

危险度/规避：高；规避**ブラック**；勿在镜头前暴露契约者身份。

任务方向/奖励：母带／纹章扣／调令抄本；奖励＝通行、低阶魔界金属、蛇头人情。不发始祖血。
