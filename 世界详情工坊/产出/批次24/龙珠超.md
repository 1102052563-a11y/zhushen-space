# 龙珠超

## 剧情

**【作品来源】**
《龙珠超》（日语：ドラゴンボール超，Dragon Ball Super）为鸟山明原案、东映动画制作的电视动画（2015年7月5日—2018年3月25日，全131话），漫画由丰太郎作画于集英社《V Jump》连载（2015年8月号起，动画与漫画分线并行、部分剧情有差）。承接《龙珠Z》魔人布欧篇约四年后，正式把破坏神、侍从天使、全王、多重宇宙与超级龙珠推入正史。关联作品含剧场版《龙珠Z 神与神》《龙珠Z 复活的「F」》《龙珠超 布罗利》《龙珠超 超级英雄》；本档案以动画主线为骨架，漫画独有篇（银河巡警·魔罗、幸存者葛拉诺拉、超级英雄延伸）作补充。文风：热血格斗、神级降维、吃货日常与宇宙存亡并置。卡片锚点：**比鲁斯指弹**——破坏神比鲁斯于界王星轻轻一指弹飞超级赛亚人3孙悟空，气势碰撞即令界王星崩溃，神与人的次元差距。

**【世界定位】**
魔人布欧战后的第七宇宙：地球、那美克星、界王星、破坏神行星、全王宫殿与十二宇宙并立。主角**孙悟空**以武道狂热为轴，从界王星被比鲁斯一指弹飞起步，经超级赛亚人之神、金弗利萨、宇宙6武道会、黑悟空与扎马斯、力之大会、布罗利、魔罗、葛拉诺拉等事件，把「地球小队」推上神与全王棋盘。一句话：从指弹飞超赛3的神人次元，一路打到力之大会与全王注视下的宇宙存亡。

**【世界观 · 力量体系】**
力量本源：**气（战斗力）**、神之气息、破坏之力、天使技法、龙珠许愿、波特拉耳环合身、自在极意功、唯我极境等。赛亚人濒死复苏可暴涨战力；界王神与破坏神成对——界王神死则对应破坏神亦亡。死亡规则：凡人/战士可被龙珠复活（有次数与龙种限制）；神仙、破坏神、全王体系另有规则；全王可一念抹杀整宇宙/整条时间线。特殊系统：侦察机战斗力读数（高阶神气常无法准确读）、时空戒指、超级龙珠（七颗各如行星大小，许愿近乎无限制）、力之大会「掉出擂台即出局」的规则杀。

战力阶梯（按破坏力与原作表现，宁低勿高）：
1. **地球凡人 / 无气武道家**：徒手格斗、枪械、胶囊科技。≈一阶。
2. **地球气功武道家 / 低阶外星杂兵**：能飞、放气功波、掀车裂石；克林、天津饭、亚姆查常态。≈二阶。
3. **地球最强层 / 弗利萨军中下级军官 / 赛亚人常人态中段**：端据点、毁街区级；初期比克、悟饭求学期常态、中级侦察机读数。≈三阶。
4. （本世界不覆盖四阶。）
5. **超级赛亚人1~2 / 金弗利萨前段 / 行星威胁级**：一怒可毁城夷地、逼近行星表面级；超赛3在卡片锚点中仍被比鲁斯指弹秒杀。≈五阶。
6. **超级赛亚人之神 / 超级赛亚人蓝入门 / 宇宙6武道会主力**：神之气息、跨星际交锋；希斯、卡芙拉、击退金弗利萨后的蓝悟空/贝吉塔。≈六阶。
7. **自在极意功未熟·兆 / 吉连中盘 / 破坏神候补托普破坏态 / 布罗利暴走前中段**：世界级到行星级碾压、规则皮毛。≈七阶。
8. **破坏神比鲁斯常战 / 维斯未尽全力 / 全盛吉连 / 自在极意功极意 / 黑弗利萨级**：爆星、掌破坏与神之权柄、跨宇宙战力标尺。≈八阶。
9. **大神官 / 全王（禅） / 维斯全力示警层 / 合身扎马斯逼出全王抹杀级事件**：文明·时间线·宇宙可被抹除；属九阶及**超阶**。

乐园阶位映射：地球凡人~低阶气功≈一阶；地球成名武道家~外星杂兵≈二阶；地球最强~弗利萨军中层≈三阶；超赛1~2/行星威胁≈五阶；超赛神~蓝/宇宙6主力≈六阶；自在极意未熟~吉连中盘≈七阶；破坏神常战~全盛吉连≈八阶；全王/大神官/时间线抹杀级≈九阶及超阶。顶点（全王、大神官、维斯全力层）=**超阶巅峰：存在·情报优先/条件性胜利**，严禁「被封印/削弱所以战力为零」。卡片锚点比鲁斯指弹：超赛3孙悟空在破坏神面前≈五阶顶被八阶神秒杀。

**【地理 · 舞台】**
- **地球（第七宇宙）**：西都、撒旦城、龟仙屋、布尔玛胶囊公司、悟空家、那巴那沙漠武道会场、无人岛（17号巡林）。凡人不知神战，胶囊科技与天下第一武道会并存；神级对轰常被「天然灾害/记忆消除」盖过。一~三、五阶主舞台。
- **界王星**：北界王居所与10倍重力修行地；**比鲁斯指弹名场面发生地**，神之气势碰撞可致星球结构崩溃，是「人级最强仍是神级玩具」的教学现场。五~六阶警示舞台。
- **破坏神行星 / 比鲁斯星**：比鲁斯与维斯居所，美食仓库与神之修行场；重力、破坏余波、天使示范并存。悟空、贝吉塔神之修行主场。六~八阶。
- **那美克星（新）**：龙珠与波伦加龙神；复活F、魔罗篇许愿与囚犯释放的关键龙系节点。
- **宇宙空间擂台 / 虚无空间**：宇宙6武道会、力之大会主场；规则杀核心是「掉出擂台即出局」，与纯战力屠杀不同。六~八阶。
- **全王宫殿**：大神官接待厅，十二宇宙破坏神与界王神觐见；力之大会观赛席，孩童全王一念可改宇宙存亡。八~九阶。
- **第十宇宙神界**：见习界王神扎马斯与戈瓦斯师徒线；黑悟空前史与「伪正义弑神」起点。
- **未来特南克斯时间线**：人造人后、黑悟空与合身扎马斯肆虐的废土；可被全王整条时间线抹杀，是超阶「条件性解围」的范本。七~九阶。
- **西利亚尔星**：狙击民族故土、弗利萨军旧征服地、西塔一族黑产棋盘；葛拉诺拉篇主场，巴达克史考特关键物出土地。
- **弗利萨军旧星域 / 赛亚人故星残骸**：星际奴隶贸易与灭族史背景板；佣兵、残党、赏金猎人出没。
- **第十二宇宙等免战宇宙**：平均战斗力高、力之大会免参加，对照「低水平宇宙可被抹」的残酷规则。
- **地狱 / 冥界侧**：弗利萨被捞出力之大会的通道；力之大会临时同盟的荒诞来源地。

**【世界剧情线】**
① **破坏神比鲁斯篇（卡片锚点·神人次元）**  
魔人布欧战后约四年，地球用龙珠抹去布欧相关记忆，表面恢复「撒旦英雄叙事」。孙悟饭与比迪丽成婚，布尔玛在胶囊公司办大型派对；撒旦先生获世界和平大奖一亿，受之有愧转赠悟空，悟天怂恿父亲收下后上界王星修行。另一条线：破坏神**比鲁斯**长眠苏醒，侍从天使**维斯**陪其巡游诸星寻美食并执行「该破坏的星球」清单；界王神侧已察觉其苏醒。比鲁斯听闻「打倒弗利萨的赛亚人」，循气至界王星。悟空久未逢敌，不顾**北界王**哭劝，以**超级赛亚人3**出战——比鲁斯几乎未认真，**轻轻一指弹飞悟空**，双方气势碰撞令**界王星球体结构濒临崩溃**。此即卡片锚点：超赛3在破坏神面前不是「差一点」，而是次元差。  
比鲁斯转赴地球，派对上因**布丁**与魔人布欧争执被激怒，决意毁灭地球。悟空经神龙提示，集结贝吉塔、悟饭、悟天、特南克斯等赛亚人血脉举行义血仪式，短暂觉醒**超级赛亚人之神**，与比鲁斯展开跨海陆空的神级对轰；终局比鲁斯认可其战意与潜力，收手离场，地球得保。维斯此后成为悟空、贝吉塔的神之修行导师；「破坏神喜怒可毁球」成为第七宇宙永久悬顶规则。名场面链：界王星指弹 → 地球布丁危机 → 超赛神觉醒 → 神与神对拳 → 比鲁斯打哈欠式收场。世界状态：凡人仍过日常，知情者开始恐惧「天上的猫」。

② **复活的「F」篇（金弗利萨·复仇军）**  
约半年后，弗利萨军残党**索鲁贝**等潜入地球，逼迫与诱骗并行，集齐龙珠许愿复活**弗利萨**。复活后的弗利萨拒绝立刻报仇，选择闭关苦练四月，压出**黄金形态（金弗利萨）**，再率大军压境。地球侧悟空、贝吉塔已在维斯门下触及神之气息与蓝色超级赛亚人；战场从登陆点蔓延至地表大规模对轰。金弗利萨战力一度压过蓝形态，性格急躁与能量效率问题成为破绽；终局仍败（动画与电影在谁补刀、地球是否短暂被毁等细节有差，核心不变：弗利萨再败、地球危机解除）。  
因果：弗利萨确认「修行有效」，暴君开始认真变强；为后文力之大会悟空从地狱捞人埋下人格逻辑（悟空要强敌与战力，弗利萨要复活与复仇舞台）。地球再次被星际暴君标记为「可反复登陆的玩具箱」。可介入点：龙珠护送、残党小队清剿、疏散西都、窃取弗利萨军通讯。

③ **第六宇宙篇（超级龙珠与全王随口）**  
比鲁斯与第六宇宙破坏神**象帕**为争面子与超级龙珠，私办宇宙间武道会：各派五名战士，胜方获地球所有权及**超级龙珠**使用权。第七宇宙阵容以悟空、贝吉塔、比克、魔人布欧等为核心，并塞入吉祥物**蒙纳卡**（比鲁斯侧操作）。第六宇宙出场**希斯**、**卡芙拉**、波塔摩等；悟空与希斯交锋揭示赛亚人跨宇宙同源与不同进化路径。赛制强调擂台规则，纯杀戮不是唯一解。第七宇宙险胜。  
赛后**全王**突然到场「参观」，随口提出：下次把全部宇宙选手聚起来办大赛。悟空兴奋到想交朋友，比鲁斯与维斯如临大敌——全王的随口一句，在后文变成力之大会与宇宙抹杀规则。因果：超级龙珠进入公开争夺视野；全王「孩童兴趣」成为比任何反派更危险的剧情发动机；宇宙6赛亚人线立住，为后续交流与对照埋钩。

④ **「未来」特南克斯篇（黑悟空·扎马斯·时间线抹杀）**  
未来特南克斯时间线：人造人与布欧危机后人类仍未迎来和平。外貌酷似悟空的**黑悟空**系统屠戮残存人类；未来布尔玛在完成单程时光机相关努力后死于其手，未来小舞重伤掩护特南克斯逃往「现在」。现在时间线调查揭示真相：第十宇宙见习界王神**扎马斯**极度憎恶人类，被戈瓦斯赏识却走向极端；他胁迫或利用智者祖诺情报，杀师，借超级龙珠夺取悟空之身成黑悟空，并联手未来时间线的自己，许愿不死身，以**超级赛亚人玫瑰**肆虐。因未来界王神已死，未来比鲁斯亦亡，破坏神制衡真空，黑悟空得以屠神链扩张。  
比鲁斯先摧毁「现在」的扎马斯以断因果；悟空、贝吉塔、特南克斯赴未来苦战。名场面包括：魔封波因封条带错失败、贝吉塔蓝形态压制、波特拉耳环合身、合身扎马斯分裂铺满天。终局悟空动用全王所赠呼叫器召来**未来全王**——未来全王见满目疮痍与无数扎马斯脸，**一念抹杀整条未来时间线**。众人逃回本时间线；悟空把未来全王带来与现在全王做朋友；特南克斯与未来小舞转赴另一平行未来。因果：全王抹杀被实证为「不可解战斗题」；时空戒指、超级龙珠、界王神—破坏神生死绑定全部摊牌；「条件性胜利」=引来更高维度孩子气裁决，而非硬刚合身扎马斯本体到死。

⑤ **宇宙生存篇 / 力之大会（宇宙存亡与17号许愿）**  
悟空为催促全王兑现「全宇宙大赛」，不顾比鲁斯、维斯警告求见。大神官召集各宇宙破坏神与界王神；第十一宇宙傲兵隊**托普**、**吉连**等亮相。全王真正意图包括：平均战斗力偏低的宇宙可被清理；力之大会败者所属宇宙全灭，胜者获超级龙珠许愿权。第一、五、八、十二宇宙因水平够高免战；其余必须参赛。  
第七宇宙十人席：悟空、悟饭、贝吉塔、克林、天津饭、比克、龟仙人、人造人17号、18号；布欧休眠，悟空从地狱捞出**弗利萨**顶替。比赛在虚无空间擂台进行，掉落即出局。多宇宙按动画顺序相继被抹（第九、十、二、六、四、三等，漫画顺序略有差）；战场名场面含17号与弗利萨的算计、悟空**自在极意功（未熟·兆）**对吉连、托普破坏之正义审判。终盘悟空、弗利萨、吉连同落场外，诈死的**17号**成为最后幸存者，向超级神龙许愿**复活所有被抹除的宇宙**。第七宇宙平均水平排名改善，但真正的「赢」是善意许愿而非征服。因果：悟空与全王、大神官、吉连、托普永久结缘；弗利萨获得再次活动的政治资本；「许愿措辞」比「打死最强」更接近九阶解法。

⑥ **布罗利篇（传说赛亚人·剧场版衔接）**  
力之大会后短暂和平被弗利萨再次打破。流放的传说级赛亚人**布罗利**与其父帕拉格斯线被弗利萨利用：战力在交锋中无尽攀升，绿色暴走席卷行星级战场。悟空与贝吉塔最终以合身（戈吉塔）级手段应对；事后布罗利被带离可控环境，弗利萨再度退场却未死心。因果：赛亚人「战斗中变强」被推到极端样本；弗利萨军与赛亚人旧账（灭族、奴隶、实验）继续为葛拉诺拉篇供能。

⑦ **银河巡警囚犯篇 / 魔罗篇（漫画·吸气与双修路线）**  
银河巡警因远古魔导士**魔罗**越狱，试图动用沉睡于布欧体内的**大界王神**之力，甚至不择手段麻醉悟空贝吉塔。魔罗能吸收气、驱使宇宙监牢囚犯、与星球融合；那美克星龙神许愿曾让其恢复全盛并释放囚犯。悟空在危机中掌握**自在极意功**神之力向；贝吉塔拒绝复制悟空，向亚德特拉星人学「把气分给他人」；大界王神觉醒并牵扯坏布欧转世线索。维斯点拨下，众人于地球融合危机中击败魔罗。因果：自在极意（悟空）与破坏神路线/唯我前史（贝吉塔）明确分岔；银河巡警证明「执法组织在神级面前仍需借力」；吸气与分气成为可玩的战术母题。

⑧ **幸存者葛拉诺拉篇（漫画·复仇与黑弗利萨）**  
西利亚尔星灭族幸存者**葛拉诺拉**与那美克星人**莫奈特**相依。他以龙珠耗去约150年寿命换「宇宙第一」之力，只为向弗利萨与赛亚人复仇；实则被西塔一族**艾列克**当刀——西塔专干「雇凶灭族→倒卖行星」的黑产，灭族主谋并非单纯赛亚人冲动。悟空、贝吉塔与葛拉诺拉三方混战中逐渐识破利用链；贝吉塔悟出**唯我极境**；莫奈特揭开**巴达克**曾救幼年葛拉诺拉母子的真相（史考特录音）。艾列克另许愿强化其弟**贾斯**，以生命燃烧堆战力。终局**黑弗利萨**降临，秒杀棋子、重创悟空贝吉塔后带走可用之才离去。因果：弗利萨成为未收回的顶级隐患；巴达克形象补全；「龙珠许愿耗寿/耗命」展示超级龙珠外的残酷交易。

⑨ **超级英雄篇及延伸（悟饭·比克·红缎带残党）**  
漫画/剧场版超级英雄线把镜头更多交给**孙悟饭**、**比克**与地球科技侧：红缎带残党、海德博士盗尸改造人造人、终极赛鲁等；悟天、特南克斯以「赛亚超人」玩英雄游戏，皮拉夫等负责后勤与翻车。悟饭在护短爆发中展现野兽形态等新高度；布罗利被引导控制超级赛亚人；比克亦有战力跃进。悟空父子与贝吉塔、布罗利在比鲁斯星的「最强切磋」演变成闹剧，最终仍要接小芳放学——神级战力与地球日常再次并置。世界状态：神之修行日常化，但红缎带式本土危机仍可杀人；契约者在地球侧仍有低中阶任务池。

⑩ **蝴蝶效应边界与可玩政治**  
契约者可改局部：救下弗利萨军炮火下的街区、截获索鲁贝小队、帮布尔玛抢修时光机零件、在宇宙6赛场边缘传递掉落预警、替17号多挡一次围猎、把巴达克史考特复印件交给葛拉诺拉更早。  
不可无代价改：比鲁斯「没吃到就毁」的任性底层、全王抹杀权、力之大会败者宇宙消灭规则、悟空因好战催生大赛的性格因果、扎马斯憎恶人类的神界病灶。  
改写回流方式：破坏神提前巡视地球、扎马斯提前夺身、力之大会名单被替换导致某宇宙提前被抹、弗利萨因情报差更早黑化或更早屠城、全王因「比赛变无聊」直接改规则。AI叙事优先：龙珠争夺、侦察机读数、擂台规则、许愿措辞、破坏神饮食情绪，而非一上来硬刚全王。

- **弗利萨军飞船 / 跃迁数据**：五~六阶抢夺物，能换情报与逃命通道。

**【隐藏剧情 · 伏笔】**
- 界王神与破坏神生死绑定：未来辛身亡→未来比鲁斯死→黑悟空屠神链前提；任何「先杀界王神」的阴谋都等于拆掉破坏神保险丝。
- 全王「孩子气」与大神官执行力：真正终局威胁不是反派台词，是观赛席一念；力之大会本质是清理低水平宇宙的游戏化。
- 悟空好战双刃：催生力之大会把多宇宙拖入抹杀风险；也称全王朋友换来未来全王解围——同一性格写出灾与救。
- 赛亚人无尽成长：布罗利、吉连战中变强、贝吉塔唯我极境、悟饭野兽，反复证明「到顶」是假命题。
- 弗利萨黑形态：葛拉诺拉篇末未收回的刀，漫画后续阴影；档案勿写成已死或已改邪。
- 动画与漫画分线：动画131话止于力之大会后；漫画续魔罗/葛拉诺拉/超级英雄。可并置引用，勿硬接成无缝同一集数。
- 卡片锚点「指弹」永久标尺：任何超赛3级自满在比鲁斯面前仍是井底；神之气息是质变门槛，不是发型变色。
- 布丁与美食政治：比鲁斯情绪管理=行星外交；布尔玛敢吼破坏神的名场面说明「神也可被生活细节撬动」，但不等于可驯服。
- 17号许愿：证明超级龙珠胜负的最优解可以是「利他措辞」；九阶任务应优先许愿语言学而非输出比拼。

画风：热血格斗 + 生活搞笑 + 神级降维恐怖；光柱满天时西都便利店仍在营业。叙事口吻宜干脆、有气势碰撞与吃货细节，忌修仙文腔硬套。  
忌：把比鲁斯写成可被劝退的老好人而忽略毁球任性；忌把全王当可击败最终BOSS；忌「封印/削弱所以神没战力」；忌洗白悟空好战招灾；忌四阶硬凑（本世界不覆盖）；忌把龙珠写成无限免费复活外挂而无次数与代价。

## 阶位切入点

> 阶位↔战力：一阶≈地球凡人/无气武者；二阶≈地球气功武道家；三阶≈地球最强~弗利萨军中层；五阶≈超赛1~2/行星威胁；六阶≈超赛神~蓝/宇宙6主力；七阶≈自在极意未熟~吉连中盘；八阶≈破坏神常战~全盛吉连；九阶≈全王/大神官/时间线抹杀级。世界顶点（全王、大神官、维斯全力层、比鲁斯动真格）为超阶存在，低阶切入一律「规避顶点·情报优先」。**无四阶。**

**一阶（地球凡人 · 西都巷弄线）**
切入身份/时点：契约者以西都或撒旦城普通市民/打工者/胶囊公司外围临时工身份切入，锚定布尔玛大型派对前数日、街面尚平静。
初始事件：胶囊公司运货路上，你目睹不明飞行物掠过（实为维斯带比鲁斯觅食的前兆扰动），随后警察与记者封锁街区；一个自称「找布丁配方」的古怪传闻在夜市传开，有人出高价买「地球甜品地图」。
开场白建议：「西都的霓虹还亮着，胶囊车在高架上排成长龙。你刚把最后一箱货搬进后门，就听见天上裂开一声不属于飞机的嗡鸣——有人抬头许愿，有人已经开始囤水。没人知道，真正的危机不叫魔人，叫一位没吃到布丁就会炸行星的神。」
关键NPC立场：**布尔玛**——天才却暴脾气，缺人手时会临时雇人，对异常能量比警察敏感；**撒旦先生**——门面英雄，爱面子，能弄到奖金与记者通道；**克林**——警察身份巡逻，对「天上的事」心知肚明却懒得对凡人解释；**贝吉塔**——若在胶囊公司露脸，视凡人如无，但动他家人会立刻杀意。
主线钩子/支线：布丁与甜品供应链（间接触发比鲁斯情绪）；龙珠雷达零件失窃传闻；支线＝护送胶囊运输、夜市情报倒卖、避开弗利萨军早期探子。
危险度/规避：低~中——致命来自枪械与外星人杂兵；须规避任何「金色巨人/紫色猫神」目击现场。
任务方向/奖励：活过神级路过、换到胶囊科技边角与人际关系；奖励取向＝龙珠残片线索、防弹胶囊、布尔玛签的临时通行证。

**二阶（地球气功 · 龟仙流外围线）**
切入身份/时点：以龟仙流编外弟子/天下第一武道会外围选手身份切入，锚定悟空上界王星前后、地球武道家仍是「本地天花板」。
初始事件：武道会预选赛场突发气功对轰，一名弗利萨军逃兵伪装成选手砸场抢龙珠情报；你与**天津饭**、**亚姆查**被卷进同一场混战，必须在警察封锁前把逃兵逼出馆外。
开场白建议：「擂台还飘着尘土，你的手刀刚震开对方护具，远处海平线却闪过一道不属于地球武学的光。天津饭低声说：别抬头太久——天上的事，我们管不了；地上的龙珠，有人已经在数了。」
关键NPC立场：**龟仙人**——老不正经但眼毒，可授基础气功与战术；**克林**——同门可靠，关键时刻会拉你一把；**天津饭**——严肃，厌恶无谓牺牲；**比克**——若现身则压迫感极强，对外来武者先敌意后观察。
主线钩子/支线：截获弗利萨军探子通讯；武道会名额与赞助商黑钱；支线＝仙豆流言追查、龟仙屋杂役换指导。
危险度/规避：中——须规避超赛级对轰余波与比鲁斯接近地球时的气绝压。
任务方向/奖励：稳定气功与飞行；奖励取向＝中级气功心法、侦察机残片、仙豆半颗。

**三阶（地球最强层 · 龙珠争夺线）**
切入身份/时点：以成名散修武者/那美克星访客向导/银河巡警编外线人身份切入，锚定复活F前夜或龙珠被索鲁贝盯上时。
初始事件：你受托护送一颗龙珠穿越荒漠，半路遭遇弗利萨军小队与地球防卫临时联盟抢珠；**悟饭**或**比克**其中一人路过「顺便」清场，却把更麻烦的宇宙通讯频率暴露给你。
开场白建议：「龙珠在背包里沉得像一颗会呼吸的星。沙尘暴里亮起红色侦察机光点时，你知道这不是天下第一武道会的延长赛——是星际奴隶主的购物清单，而地球只是货架。」
关键NPC立场：**孙悟饭**——护民优先，可临时合作；**比克**——冷酷高效，讨厌拖后腿；**索鲁贝**——弗利萨忠犬，情报与杀意绑定；**邓斯特**等残党——炮灰但人多。
主线钩子/支线：阻止或延迟弗利萨复活所需的最后一颗龙珠；支线＝那美克星方言翻译、假龙珠调包、给布尔玛送雷达校准数据。
危险度/规避：高——弗利萨军中层可屠城；须规避金弗利萨本体与任何破坏神气。
任务方向/奖励：介入复活F序章；奖励取向＝龙珠一颗（烫手）、气感训练、银河巡警联络暗号。

**五阶（超赛阈值 · 界王星余波/金弗利萨前线）**
切入身份/时点：以能变身或抗住超赛余波的外星佣兵/赛亚人旁支/北界王推荐修行者身份切入，锚定**比鲁斯指弹之后**或金弗利萨登陆地球当日。
初始事件：你被临时编入「拖住弗利萨军侧翼」的敢死任务，目睹金弗利萨一击抹平山脊；**孙悟空**与**贝吉塔**蓝光冲天前，有人把平民疏散路线图塞进你手里——选救人还是抢军残技术。
开场白建议：「超级赛亚人的金光曾是传说，现在只是开战信号。你在冲击波里抱住一个孩子，耳鸣中听见弗利萨慢条斯理的笑声：行星级的玩具，不该反抗。」
关键NPC立场：**孙悟空**——会为强者眼睛一亮，也可能把你当热身；**贝吉塔**——命令式合作，完成任务才给眼色；**弗利萨**——视一切为玩具；**北界王**——远程惨叫预警（界王星相关记忆创伤）。
主线钩子/支线：金弗利萨战场救人/窃取飞船数据；卡片锚点回响：有人复述「指弹飞超赛3」以恐吓新兵；支线＝收集弗利萨军战损残骸。
危险度/规避：极高——须规避比鲁斯、维斯、金弗利萨正面对轰中心。
任务方向/奖励：在神级与暴君夹缝存活；奖励取向＝超赛级气感样本、飞船跃迁残码、贝吉塔一句「还算能打」。

**六阶（神之气息 · 宇宙6武道会线）**
切入身份/时点：以第七宇宙候补选手/破坏神临时点名的「凑数战力」/象帕侧间谍反水者身份切入，锚定宇宙6武道会开赛前后。
初始事件：虚无空间擂台外，**希斯**与**卡芙拉**的气让你膝盖发软；比鲁斯命令你在赛场边缘盯防「超级龙珠坐标泄漏」，同时**蒙纳卡**（吉祥物）被人群挤到你怀里——护住他或看破伪装都是死局。
开场白建议：「两个宇宙的破坏神在贵宾席打哈欠，擂台上却已经打出行星都装不下的气浪。有人告诉你：赢了有超级龙珠，输了——地球易主。你的名字，刚好在替补名单最后一行。」
关键NPC立场：**比鲁斯**——喜怒无常，任务失败可能直接消灭；**维斯**——微笑给提示，从不白帮；**希斯**——冷傲，可因武道惺惜相惜；**卡芙拉**——好战，挑衅必应；**孙悟空**——只想和下一个强敌打。
主线钩子/支线：超级龙珠情报战；赛场规则杀（掉落即败）；支线＝宇宙6赛亚人文化接触、赌盘欺诈。
危险度/规避：极高——须规避破坏神动真格与全王突然到场时的跪伏礼仪失误。
任务方向/奖励：宇宙级舞台露脸；奖励取向＝神之气息入门指引、超级龙珠方位残片、维斯的点心（别问味道）。

**七阶（力之大会中盘 · 宇宙存亡线）**
切入身份/时点：以第七宇宙十人名单边缘替换者/被悟空从别处「推荐」的战士/17号岛上盟友身份切入，锚定力之大会开战至中盘多宇宙已灭时。
初始事件：擂台边缘，你与**人造人17号**背靠背挡住第十一宇宙队员的围猎；远处**吉连**一拳震开空域，**托普**宣布对邪恶零容忍——你的「外来者」气息可能被判定为应清除对象。
开场白建议：「计时器在虚无中走动，每少一个宇宙，观赛席就少一排破坏神。有人还在为荣誉而战，有人已经只为让家人继续存在而战。你脚下一滑，就是整条命与整个故乡的 co-pay。」
关键NPC立场：**吉连**——强敌与标尺，无废话；**托普**——正义审判，立场极端；**弗利萨**——同盟时可靠、事后必背刺风险；**人造人18号**——护兄优先；**孙悟饭**——指挥向，重视配合。
主线钩子/支线：力之大会存活与联动许愿；支线＝给17号制造诈死窗口、阻止某宇宙绝望反扑误伤。
危险度/规避：贴近顶点——须规避全王不悦、大神官警告、吉连全力。
任务方向/奖励：成为许愿链条上的一环；奖励取向＝自在极意「兆」的体感残影、宇宙级战绩公证、17号的信任。

**八阶（破坏神常战 · 比鲁斯星/魔罗后期）**
切入身份/时点：以比鲁斯星特准修行者/银河巡警特级顾问/魔罗篇被维斯点名的「气之容器」身份切入，锚定魔罗吸收气危机或布罗利级行星战场。
初始事件：**魔罗**在地表铺开吸收法阵，你的气被抽走一半；**贝吉塔**扔来「分气」训练口诀，**孙悟空**自在极意光影一闪而过——你必须在枯竭前把平民空洞区的坐标送给银河巡警。
开场白建议：「破坏神的星球上，重力与美食同等致命。维斯轻轻一挥杖，时间像被捏过的布丁——他对你说：别学悟空去招惹全王，先学会在被抽干气之前，把任务做完。」
关键NPC立场：**维斯**——导师与天花板；**比鲁斯**——可因一顿饭改变任务优先级；**魔罗**——吸气魔导，古老邪恶；**布罗利**——若同场则环境灾害本身；**大界王神**（布欧内）——关键辅助，非战斗主力。
主线钩子/支线：魔罗法阵破解；破坏之力与自在极意两条路的见闻；支线＝护送银河巡警、回收囚犯。
危险度/规避：贴近顶点——比鲁斯动真格、维斯全力、魔罗全盛=超阶边缘，情报与条件胜优先。
任务方向/奖励：神之修行入门资格；奖励取向＝分气/防御特训、破坏之息皮毛（勿滥用）、巡警最高通行。

**九阶（全王注视 · 时间线/许愿终局）**
切入身份/时点：以被大神官临时传讯的「观察者」/持有未来全王呼叫器残响的关联者/超级龙珠许愿现场存活者身份切入，锚定未来时间线抹杀前后或力之大会许愿瞬间。
初始事件：你跪在全王宫殿的光面上，**大神官**宣读规则；远处未来时间线的天空裂开，**合身扎马斯**的脸铺满天幕——有人把呼叫器塞给你，说：「现在，决定你是按按钮的人，还是被抹掉的背景。」
开场白建议：「十二宇宙的破坏神在此也不敢大声喘气。全王笑着问你想不想看更有趣的比赛；你知道正确答案不是战力，是别让那孩子觉得无聊——也别让他觉得你该消失。」
关键NPC立场：**全王**——超阶顶点，喜恶即宇宙生死；**大神官**——唯一有效沟通层；**未来全王**——抹杀执行实例；**合身扎马斯**——意识形态终焉之敌；**孙悟空**——唯一敢称全王朋友的疯子，也是你活下来的可能纽带。
主线钩子/支线：时间线保全/许愿措辞（17号式「复活诸宇宙」级善意）；支线＝阻止错误许愿、护送特南克斯撤离抹杀倒计时。
危险度/规避：顶点——**超阶巅峰：存在·情报优先/条件性胜利**；严禁硬刚全王、大神官、维斯全力；胜利来自措辞、站位与被悟空/17号等原作因果「顺路带上」。
任务方向/奖励：终局见证与许愿余波；奖励取向＝超级龙珠许愿残权（极危）、全王礼物级玩具（可能是宇宙法则漏洞）、活着离开宫殿本身。
