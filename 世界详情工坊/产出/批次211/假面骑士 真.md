# 假面骑士 真

## 剧情

**【作品来源】**
《真・假面骑士 序章》（真・仮面ライダー 序章；英：Shin Kamen Rider: Prologue）为东映1992年2月20日发行的OV（非戏院首映）作品，片长约88分钟，纪念假面骑士诞生20周年。原作石之森章太郎；导演辻理；编剧宫下隼一、小野寺丈；主演石川功久（风祭真）、野村裕美（明日香爱）、石滨朗（风祭大门）等；石之森本人友情客串财团人士。定位「成人向特摄」：血腥、暴力、裸露、恐怖镜头显著，命名「真」因被视为石之森心目中「真正」的假面骑士；称「序章」原拟续作未成。卡片锚点：ISS人体实验→蝗虫化觉醒→爱人横死→血腥复仇。

**【世界定位】**
近现代日本：表面是临床免疫与绝症药物研究，实则国际神秘「财团」日本支部ISS以基因工程制造改造人士兵（Cyborg Soldier），服务军事。城南大学体育学部毕业的风祭真IQ250、体育万能，为助父亲风祭大门的研究自愿成为受试者，实已被改造成蝗虫基因LEVEL 3完成体。情绪激动时异变为半人半蝗怪物／假面骑士真，在爱人明日香爱挡枪死后展开残酷复仇。一句话：没有喊变身口令的血肉骑士，对军工财团的成人向复仇悲剧。

**【世界观 · 力量体系】**
力量本源：①蝗虫基因改造与细胞再生（LEVEL 3完成体可急速愈合且保有自我）；②LEVEL分级改造人士兵：Lv.2（豪岛：机械臂强化、无再生）／Lv.3（鬼塚失败品失理智；真＝成功体）；Lv.1／4不详；③情绪触发异变，无需腰带口令；④肉搏为主，无传统骑士踢美学，可拔头碎脊。死亡：凡人枪械可杀；改造人可被更强再生者撕碎；财团可继续量产。无宇宙神魔，顶点是财团体系与LEVEL完成体互杀。

战力：普通人／研究员≈一阶（本世界不覆盖一、二阶切入）；武装保安与Lv.2≈三阶；Lv.3完成体（真）与失控Lv.3≈五阶；财团武装总力与「完美兵器」企划层≈五~六阶。清单仅三、五、六。

乐园阶位映射：ISS外围武装／Lv.2≈三阶；假面骑士真（稳定杀意）／鬼塚失控体≈五阶；财团本级清场与真的终局复仇战≈六阶。顶点＝财团意志＋顶级改造兵器，条件性胜利／情报优先。宁低勿高。

**【地理 · 舞台】**
城南大学与体育会系；风祭家与研究室；ISS日本支部实验室（手术室、饲养／观察笼、枪械库）；都市夜巷与停车场（肉搏战场）；泳池／私密空间（成人向情感戏，叙事点到为止）；片尾公路（真抱子「风祭新」离去）。CIA线作为外部窥伺势力偶尔触达。整体封闭、阴冷、实验室味重。

**【世界剧情线】**
① **药物谎言与自愿实验**  
大门博士被ISS所长欺骗，以为研发电解绝症药物；真为父自愿受试。财团真目的：开发可量产的改造人士兵。真在不知情中完成蝗虫基因融合，成为LEVEL 3第一号成功体。

② **异变与身份崩坏**  
真情绪激动时皮肤外骨骼化、复眼与肢节显现，理性与怪物性拉锯。爱人明日香爱与其结合并怀孕；关系戏含成人向镜头（成品叙事避免色情描写，只保留「羁绊与代价」）。

③ **背叛、枪杀与挡弹**  
财团内部清洗、CIA介入抢夺成果、失败品鬼塚义一失控。明日香爱为真挡子弹而死于其面前。名场面：血、枪火、真的完全异变。

④ **复仇肉搏**  
真不再寻求「变回人」的天真，以怪物之躯猎杀相关者：鬼塚（Lv.3失败）、豪岛（Lv.2）等。终结处对豪岛「连头带脊拔起」的残酷镜头，标志与传统骑士美学决裂。

⑤ **基因生命体与未完序章**  
片尾基因生命体婴儿（后设定名风祭新）与真同行；本片无续，残留「真之后」的空白。漫画版等周边有延伸，TV档案以OV本篇为准。

⑥ **蝴蝶边界**  
可改：提前救出个别受试者、销毁一份实验数据、让CIA档案曝光度变化。不可无代价改：爱之死所铸杀意、LEVEL 3再生与失控逻辑、财团仍存在的结构性邪恶。

**【主要人物】**
- **风祭真／假面骑士真**｜性格：高智商运动员的自信碎裂后转为冷酷复仇；保有自我的怪物。｜装备·能力：Lv.3再生、蝗虫化肉搏、无腰带直接变身。｜弧光：受试者→觉醒→丧偶→复仇者→带孩子上路。｜关系：对财团绝对敌对。
- **明日香爱**｜性格：深爱真，以身挡弹。｜能力：凡人。｜弧光：爱人→牺牲→遗腹子线。｜关系：真的人性锚，死后化为杀意。
- **风祭大门**｜性格：学者被骗的悔恨。｜能力：免疫学权威。｜弧光：研究推手→真相崩溃。｜关系：父债子偿式悲剧。
- **鬼塚义一**｜性格：科学家，失败改造后失理智。｜能力：Lv.3失控体。｜弧光：同僚→怪物对手。｜关系：可杀可比照。
- **豪岛**｜性格：财团鹰犬。｜能力：Lv.2机械臂。｜弧光：执行者→被拔头。｜关系：中层BOSS。
- **ISS所长／财团干部**｜性格：军工冷酷。｜能力：权限与枪械。｜弧光：骗局核心。｜关系：结构反派。
- **莎拉·深町等周边**｜性格：关系网。｜能力：不一。｜弧光：信息与背叛可能。
- **冰室严等**｜性格：压迫侧。｜能力：组织力。｜弧光：清场。
- **风祭新（婴儿）**｜性格：无。｜能力：基因生命体潜质。｜弧光：片尾象征。｜关系：续作空白。
- **石之森客串的财团人士**｜性格：元叙事彩蛋。｜能力：无战力。｜弧光：一闪。

**【势力图谱】**
财团（国际军工）→ISS日本支部（生物兵器）；CIA窥伺抢成果；大学与家庭被当掩护；改造人士兵内部等级森严，失败品即废弃。无「正义骑士组织」。

**【贵重物品】**
蝗虫基因培养基与LEVEL评估档案；再生细胞样本；大门研究笔记；ISS门禁与枪械库；爱的遗物（信／戒指级，情感）；婴儿风祭新（活体延续）。

**【隐藏剧情 · 伏笔】**
- 「真」＝石之森回归生物恐怖原点，否定围巾英雄无害化。
- 无机车骑士是故意设计：变身后是怪物不是偶像。
- 序章无续＝世界停在「复仇之后仍被追杀」的开放黑暗。
- 与BLACK的蝗虫意象互文，但更血腥、更无救赎。

**【大事记时间线】**
大门接ISS案 → 真自愿实验 → 基因融合完成 → 异变初现 → 爱怀孕／关系深化 → 清洗与枪战 → 爱挡弹死 → 真全开复仇 → 撕鬼塚／豪岛 → 抱子离去。

**【叙事基调 · 雷区】**
成人向、残酷、实验室恐怖；少儿特摄口吻禁止。忌：美化拔头为「帅气必杀」而无血腥代价；给真强行加围巾踢；低阶无资源硬闯ISS核心；色情化爱的戏份。最早锚点：真已完成改造、尚未失去爱之时，或爱死后复仇夜。

**【细述】**骗局：药物立项掩盖改造人士兵。崩坏：情绪异变、爱怀孕、真相碎裂。爆炸点：清洗+CIA+爱挡弹。复仇：再生肉搏、鬼塚对照失败品、豪岛拔脊。开放结局：抱风祭新上路，财团仍在。

**【等级】**Lv.2量产无再生；Lv.3失败失理智；Lv.3成功再生+自我；1/4黑箱。再生非无敌。

**【人物】**莎拉、冰室、企业干部监视网；石之森客串非战力。

**【事件】**偷档案、炸培养室、挡弹夜、公开数据按钮、高速护送。

## 阶位切入点

> 阶位↔：三阶≈ISS外围/Lv.2；五阶≈Lv.3真与失控体；六阶≈财团清场级。无一二四阶。顶点情报优先。

**三阶（实验室外围·受试者逃亡）**
切入身份/时点：ISS清洁工、安保叛逃、或被抓的预备受试者；锚定真已改造、爱仍在世前后。
初始事件：你偷到一份「LEVEL 3-01 风祭」档案拷贝，豪岛带队封锁走廊；墙后传来非人的骨骼摩擦声——真正在第一次完全异变边缘。
开场白建议：「消毒液掩盖不了血的甜腥。档案袋上的蝗虫示意图被印成红章。对讲机里豪岛说：活捉受试者，死掉的算样品。你握着U盘：这东西比枪更致命。」
关键NPC立场：**风祭真**（不稳定）；**豪岛**；**风祭大门**（悔）；**明日香爱**（找真）。
主线钩子/支线：送出档案；掩护爱离开监视圈；炸毁一间培养室。
危险度/规避：高。规避真完全失控时的正面；不闯所长层。
任务方向/奖励：LEVEL技术残页、逃生路线、大门的道歉录音。

**五阶（挡弹之夜·复仇开始）**
切入身份/时点：能与Lv.3短暂交手的改造体／契约者；锚定爱死后数小时。
初始事件：停车场枪战刚停，爱倒在真怀里；鬼塚失控体从暗处扑向围观密探。你必须选择：帮真撕开包围，或先抢CIA要带走的婴儿培养数据。
开场白建议：「雨和血分不清。真的复眼里没有英雄，只有空洞。鬼塚的笑声像坏掉的锯。你迈出一步，就等于在财团的死亡名单上签字。」
关键NPC立场：**风祭真**；**鬼塚义一**；**豪岛**；**CIA特工**；**财团枪手**。
主线钩子/支线：撕开包围；夺回／销毁婴儿数据；阻止鬼塚冲入市区。
危险度/规避：极高。真的再生使其难杀，亦难控；勿当「可收服宠物」。
任务方向/奖励：与真的临时互不伤害默契、Lv.3战斗样本、爱的遗物转交。

**六阶（拔脊之后·财团反扑）**
切入身份/时点：终局夜的幸存者向导；锚定豪岛被杀、财团准备卫星式追杀时。
初始事件：真带着婴儿出现在高速公路入口，财团武装直升机与第二波Lv.2小队抵达；你持有大门留下的「公开全数据按钮」——按下则全球曝光，亦会让真永远被全人类当怪物。
开场白建议：「直升机探照灯把蝗虫的影子拉得像神。真把婴儿护在胸甲骨下，对你只说一个词：走。按钮在你掌心发烫：真相是光，也是另一种解剖刀。」
关键NPC立场：**风祭真**；**风祭新**；**财团干部**；**大门**（若仍存活则遥控）；**CIA**要抢独家。
主线钩子/支线：条件性胜利＝让真突围并决定是否公开数据；不「净化」其怪物之躯。
危险度/规避：顶点级。**存在·情报优先／条件性胜利**；财团不是被压制的魔王，是可持续派兵的体系。
任务方向/奖励：序章结局分支见证、基因生命体一线生机、ISS覆灭度（局部）。

**三阶补场（培养室停电）**
切入身份/时点：发电机维修工。
初始事件：停电30秒Lv.2暴走，你拉手闸换真的逃亡窗。
开场白建议：「应急灯把蝗虫影拉到天花板。闸刀冰凉：拉下就是选边。」
关键NPC立场：**豪岛**射击；**风祭真**异变边缘；**风祭大门**哭喊停止；**受试者**求放。
主线钩子/支线：拉闸；开污道；拷贝LEVEL日志。
危险度/规避：高。
任务方向/奖励：污道图、日志。

**五阶补场（CIA黑车）**
切入身份/时点：被夹击的司机。
初始事件：黑车运成功体样本，真拦路，你决定撞谁。
开场白建议：「保险杠对怪物，后视镜是星条旗文件夹。没有第三选项。」
关键NPC立场：**风祭真**；**CIA**；**财团枪手**；**鬼塚义一**远处狂笑。
主线钩子/支线：连环撞车掩护；夺婴儿培养数据销毁。
危险度/规避：极高。
任务方向/奖励：短暂同盟、数据灰烬。

**六阶补场（公开按钮）**
切入身份/时点：持有大门最终权限的人。
初始事件：直升机围真与婴儿，按钮可全球曝光ISS。
开场白建议：「探照灯下蝗虫影像神。按钮烫手：真相是光也是解剖刀。」
关键NPC立场：**风祭真**；**风祭新**；**财团干部**；**CIA**抢独家。
主线钩子/支线：条件性胜利＝突围+是否公开；不净化怪物之躯。
危险度/规避：顶点。存在·情报优先/条件性胜利。
任务方向/奖励：结局分支见证、ISS局部崩坏度。
