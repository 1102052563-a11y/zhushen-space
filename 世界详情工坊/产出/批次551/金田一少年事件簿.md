# 金田一少年事件簿
<!--meta lib=主库 tiers=一-->

## 剧情

**【作品来源】**
日本本格推理少年漫画《金田一少年の事件簿》。早期脚本金成阳三郎，后由天树征丸（天樹征丸）接手原作/原案，作画佐藤文也；1992 年起在讲谈社《周刊少年 Magazine》连载，单行本累计发行逾一亿册。系列结构跨 File 系列、短篇 File、Case 系列、明智系列、新系列、20 周年、Returns（R）、30 周年、以及成人线《金田一 37 岁之事件簿》《金田一爸爸之事件簿》。改编覆盖动画（1997–2000 本篇与后续 R）、多部剧场版与特别篇、1995 年起多代主演的真人电视剧与电影、游戏与小说。气质为封闭空间连环杀人、建筑/机关/时间差诡计、都市传说与悲剧动机并重；读者可与主角同步推理。本档案以**高中生金田一一主线（File/Case 及经典宿敌线）**为常青核心，37 岁/爸爸线作后续时间层简述。

**【世界定位】**
1990 年代前后的日本现实社会表层：高中、警视厅、孤岛旅馆、暴风雪山庄、机关宅邸、列车与学园——在普通人的日常下反复裂开「不可能犯罪」。主角金田一一（金田一耕助之外孙）以懒散差生的外壳掩盖 IQ180 的推理力，与青梅竹马七濑美雪、警部剑持勇、警视明智健悟一起，把一件件被传说包装的谋杀拆回人间动机。一句话：凡俗本格推理世界——枪、刀、绳、药、机关与谎言，没有超能力，只有证据与人性。

**【世界观 · 力量体系】**
本世界**无超凡修炼、无异能、无复活**。冲突手段完全落在现实物理与心理层面：钝器/利器/毒/缢死/坠落/火灾/溺水；建筑机关（机关宅、冰桥、舞台吊杆、列车行李仓、蜡像馆通道）；时间差与不在场证明（录音、替身、路线捷径、时钟与停电）；身份伪装与心理诱导（怪人面具、童谣、占星/塔罗主题、歌剧魅影式演出）。「强」的定义不是毁城，而是：**能否在封闭圈里完成杀人而不被当场抓获**，以及**能否在证据链上压倒对方**。

可操作的「能力阶梯」（均为凡俗技能，非境界）：
1. **普通人/被害者侧**：无特殊技能，依赖警察与运气；一刀一枪、坠楼、围殴即可致命。
2. **高中社团/业余爱好者**：摄影、戏剧、魔术、登山、机关宅参观等，常被凶手利用为舞台。
3. **现场刑警（剑持勇型）**：搜查、控制现场、柔道制伏、人情网；推理偶有偏差，执行力强。
4. **精英刑事（明智健悟型）**：法务与多语、物证逻辑、跨辖协作；能独立结案，但会栽在过于完美的「艺术犯罪」上。
5. **少年侦探（金田一一型）**：超常观察、记忆、还原时间线与建筑剖面；口头禅「爷爷的名义」「谜题全部解开」；弱点是体能一般、易被情感牵制、常先被怀疑。
6. **犯罪艺术家（高远遥一型）**：把谋杀当作品，擅长规划「完美犯罪」并交给执行者、自己抽身或伪装；多次越狱与远程指挥，是智力对抗顶点。

死亡规则：人死即死；无魂体复仇。所谓「雪夜叉」「歌剧院怪人」「猎头武者」「鬼兵」等，**全部是人为扮演或机关制造的恐怖符号**。证据优先于传说。

乐园阶位映射：全体凡俗人≈一阶（平民 Lv.1–3；刑警/格斗好手/高中体育系≈一阶中期 Lv.4–7；金田一式推理顶点、高远式犯罪规划、明智级精英≈一阶后期 Lv.8–10）。**世界峰值＝一阶**（徒手/冷兵器/热武器与诡计层面的人类对抗，无超凡破坏力）。映射宁低勿高，禁止把「IQ180」写成二阶异能。

**【地理 · 舞台】**
日本列岛被切成无数「一次性密室」。东京是回流点，地方是行刑场；铁道、渡船、暴风雪与停电负责切断外界。

- **东京·私立不动高中**：金田一一、七濑美雪就读；旧校舍、屋顶、社团教室构成「日常封闭」。学园七不可思议、绞首学园等案证明：不必出海，钟声与点名表也能锁人。谜研会、戏剧部是常驻社交与嫌疑人池。
- **金田一家与美雪家周边市区**：短篇与日常过渡发生地；一逃课、打游戏、被美雪拎去补习的生活面，用来对照下一秒的尸体。
- **警视厅搜查一课与地方署**：剑持勇、明智健悟的制度舞台。跨县协查时，少年侦探以「参考人/协助者」身份挤进封锁线——世界规则允许「野孩子破案」，但笔录与记者会仍归警察。
- **歌剧院旅馆（孤岛）**：File1 与后续歌剧院系列的哥特母巢；舞台、乐池、吊杆、化妆间是动线教材。暴风雨取消渡船＝天然宵禁。
- **异人馆村（六角村系）**：分尸木乃伊传说与封闭村落；教堂、地下室、分家血缘把整村变成慢热密室。外人婚礼是入场券，也是诱饵。
- **北海道背水村等暴风雪地**：雪夜叉；本馆/别馆双建筑 + 冰谷/冰桥，地理本身就是诡计零件。取暖、燃料、电话线成为生存资源。
- **悲恋湖周边山岳旅馆**：湖面、栈桥、游船与旧日船难传闻；夏季避暑团与冬季封路团两种封闭法。
- **秘宝岛 / 墓场岛**：潮汐洞、战争遗迹、生存游戏营地；「寻宝」与「鬼兵」把少年冒险皮套在连环杀上。
- **飞驒机关宅邸**：传统木构 + 机关键与暗道；猎头武者传说是移动的恐怖符号，真杀器是建筑知识。
- **塔罗山庄 / 蜡人形城 / 黑死蝶宅 / 魔神遗迹**：主题乐园化的死亡布景——牌意、蜡像、蝶纹、遗迹咒文全是凶手的分镜脚本。
- **魔术列车（死骨原方向）**：移动密室；行李仓、车厢连接部、站台换乘是高远美学的轨道画布。
- **函馆异人馆旅馆、歌岛度假村、京都花道场、摩天住宅（37 岁线）**：品牌化「馆·岛·雪·城」在成人社交圈复用，证明本世界永不缺下一间锁上门的房。

写场景口吻：雨打旅馆窗、发电机嗡鸣、面具缝里的人眼、粉笔字「犯人就在你们之中」、剑持的吼与明智的冷笑、美雪被绑在下一间房——恐怖来自「隔壁就是凶手」，不是鬼。

**【世界剧情线】**

① **开局：歌剧院杀人事件（File1）**  
不动高中戏剧部赴孤岛歌剧院旅馆集训，排练与《歌剧魅影》气质互文的舞台剧。暴风雨取消唯一渡船，通讯与补给一并切断。蒙面「歌剧院怪人」按剧中角色顺序点名杀人，死者妆造与台词本被当作死亡预告。金田一一在同学的嘲笑与恐惧之间第一次完整摆出推理剧场：录音制造「死者还在说话」的时间差，舞台机关、钓鱼线与远程发射物把「不可能」拆回力学。真相落在过去——恶作剧导致的硫酸毁容与恋人自杀，把爱拧成复仇脚本。歌月（友森）作为怪人侧核心，动机残酷而私人。阶段意义：钉死系列公式 **封闭空间 + 怪人形象 + 悲剧动机 + 少年侦探登场**；世界状态：一仍是被教师嫌弃的差生，警方尚未建立制度性信任，结案更像「被少年说中」。

② **猎奇与分尸：异人馆村杀人事件（File2）**  
一与美雪等因婚礼进入通称「异人馆村」的封闭聚落。七具干尸/木乃伊传说先于第一具新尸出现；地下室无头干尸、酒宴分尸把节奏拉到恐怖片速度。核心戏剧性诡计：将有限尸体拆解重组，制造「人数与身份」的认知崩坏；教堂密室则调用建筑高度与搬运逻辑挑战「谁能进出」。真相侧涉及村民为掩盖非法作物等利益的旧日灭门，以及凶手在扭曲母爱下被训练成复仇机器（六星龙一系叙事）。尺度大，动画完整移植困难。阶段意义：金田一被拖出校园本格，进入 **哥特·猎奇·村社会**；也暴露早期「证据偏软、靠心理崩坏」的写法，入世叙事宜补强物证。

③ **雪夜叉与对手登场：雪夜叉传说杀人事件（File3）**  
国民偶像速水玲香的节目组取景北海道背水村，暴风雪把本馆与别馆锁成双囚笼。「雪夜叉」面具在两馆之间「瞬移」行凶，现场留下自画像式误导与钥匙/时间表矛盾。金田一一抓住**冰桥捷径**：凶手借冰面抄近路，再毁桥伪造不在场——地理诡计的教科书。警视**明智健悟**初登场，以精英姿态宣布速破，却在物证上被一压过，自此成为抬杠式长期对手。凶手绫辻真理奈的动机是空难中「见死不救/夺生路」的旧罪；本案重要在于 **凶手未死、被感化** 的收束模板首次立住。玲香洗冤后对一萌生好感，美雪侧情敌线启动。阶段意义：雪庄公式 + 警视厅精英线接入 + 半固定角色玲香入队。

④ **校园怪谈：学园七不可思议杀人事件（File4）**  
战场回到私立不动高中。旧校舍「七不思议」被逐条兑现为杀人预告与现场装置；放学后魔术师式演出、镜面与连通空间把都市传说钉上物证板。一在「自家地盘」反而最危险——同学既是证人池也是嫌疑人池，美雪与谜研同伴的信任成为唯一安全网。阶段意义：证明日常学园的点名表、铁门、值日生制度足够封闭；系列开始把「传说清单」当分集目录用。

⑤ **寻宝与双重身份：秘宝岛杀人事件（File5）**  
以寻宝为名的海岛远足迅速恶化为封闭杀。藏宝图、潮汐洞、暗号与双重身份让每个人都像叛徒。一与美雪在涨潮线赛跑时意识到：财宝是诱饵，真目标是改写继承与灭口知情者。阶段意义：少年冒险皮 + 本格硬核；「地图本身是谎言」成为后世寻宝案母题。

⑥ **湖畔传说：悲恋湖传说杀人事件（File6）**  
悲恋湖旅馆浸泡在船难与殉情传闻里；杀手造型借鉴恐怖片「杰森」式面具与利器，游客与关系户被迫困在湖畔。动机反复敲打系列母题——**见死不救、集体排挤、幸存者的报复**。一被亲友事务牵连入局，必须在「同情」与「逮捕」之间完成伦理站位。阶段意义：把「加害者曾是受害者」写进读者预期，同时禁止美化行凶。

⑦ **旅馆推理之夜：异人馆旅馆杀人事件（File7）**  
函馆系「异人馆旅馆」与 File2 村案同名不同地，强调品牌化舞台。毒杀、红胡子圣诞老人形象与意外性凶手让「最不可能的人」成立。阶段意义：短封闭、高误导的旅馆本格样板；圣诞季氛围与尸体的反差成为视觉记忆点。

⑧ **地狱摇篮：绞首之学园杀人事件（File8）**  
寄宿制学园被「地狱摇篮曲」符号笼罩，缢死连环发生。夜间点名、宿舍门锁、绳结与替身层叠，一甚至接近「被处刑者」位。名场面常是倒吊的身体与童谣录音。阶段意义：校园恐怖峰值；时间表诡计与「全员不在场却全员有嫌疑」的压迫感拉满。

⑨ **猎头武者：飞驒机关宅邸杀人事件（File9）**  
飞驒传统机关宅对外开放或家族聚会时，猎头武者传说被唤醒。暗道、翻板、刀架与斩首式演出要求侦探同时是**建筑系学生**。家族遗产、养子/庶出与旧仇在梁柱后面发霉。阶段意义：机关宅本格代表作；后世凡写「宅邸杀」几乎都在致敬本案的剖面图思维。

⑩ **被嫁祸：金田一少年的杀人（File10）**  
证据链被设计成指向金田一一本人。媒体与警方惯性把他从英雄打成凶手，一被迫逃亡式侦查，美雪与剑持的信任是唯一绳索。读者体验与角色体验同步——「我们是不是看错人了」。阶段意义：元叙事高人气篇；**侦探成为猎物**后，一与警视厅关系质变为「体制不得不借用的不规则」。

⑪ **塔罗与玲香过去：塔罗山庄杀人事件（File11）**  
暴风雪山庄，死者姿态按塔罗牌意布置；速水玲香的失踪/求救信把一强行拉回「私人案件」。双重凶手、连通门与身份掉包并行，玲香童年被拐与养父速水雄一郎相关阴影被挖开。一必须在限时救人与完整解谜之间切割注意力。阶段意义：玲香从「客串偶像」变成有长线伤口的半固定角色；美雪的嫉妒与担忧同步显性化。

⑫ **蜡像与视觉诡计：蜡人形城杀人事件（File12）**  
蜡像城展厅里，Redrum 式主题杀让「展品与尸体」边界消失。面具、加热变形、通道与参观动线被改造成不在场证明。阶段意义：视觉恐怖本格；证明展览建筑与歌剧院舞台一样适合做杀人剧场。

⑬ **怪盗绅士的杀人（File13）**  
怪盗宣言、预告函与失窃案和谋杀叠在一起。一要先回答：这是表演、盗窃，还是借怪盗皮做的灭口。阶段意义：扩展「怪人」谱系——复仇鬼之外，增加戏谑绅士面具，丰富封闭圈的角色功能。

⑭ **鬼兵与孤岛生存：墓场岛杀人事件（File14）**  
南国孤岛上，大学生生存游戏遭遇「二战鬼兵」传说。伪装军装、莫尔斯信号、双重凶手与复仇动机在补给耗尽时发酵；不动高中同行者与外部大学生团互相猜疑。阶段意义：生存恐怖 + 孤岛；把「游戏规则」写成杀人时间表。

⑮ **宿敌诞生：魔术列车杀人事件（File15）**  
地狱邀请函与扭曲人偶寄达相关者；开往死骨原方向的魔术列车成为移动密室。幻想魔术团团长被杀，尸体在行驶中消失，再以人偶姿态于他处再现。真凶 **高远遥一** 以地狱傀儡师姿态揭幕：为母近宫玲子被徒弟背叛夺技致死而复仇，宣称谋杀是艺术、魔术是辅助。重量差、行李置换、气球与视觉死角构成魔术思维诡计。高远一度被捕，旋即越狱，成为贯穿系列的智力宿敌。阶段意义：世界从「一案一凶」升级为 **有人在幕后量产案件**。

⑯ **File 后半至 Case：主题杀的量产期**  
- **黑死蝶杀人事件**：蝶宅、母女与家族诅咒感；情感密度极高，常被列为「最虐」。  
- **佛兰西银币杀人事件**：银币母题串联伪证与连锁。  
- **魔神遗迹杀人事件**：遗迹诅咒落成人事与密室，氛围压迫大于花巧。  
- **速水玲香诱拐杀人事件**：玲香被绑，一被指定运送巨额赎金；艺能圈黑暗与高远系策划阴影并置，私人关系被武器化。  
- **银幕之杀人鬼（Case 系）**：电影研究社按恐怖片分镜死亡，银幕预告谁是下一个；影像本身成为凶器与不在场工具。  
同期还有魔犬森林、天草财宝、雪影村、露西亚人偶、怪奇马戏团、歌剧院再访等。公式熟练化为固定节拍：**传说包装 → 交通切断 → 连续死 → 金田一还原 → 凶手自白/崩溃 → 感谢或死亡**。读者与契约者都应学会在「第二具尸体前」开始画平面图。

⑰ **高远线的长期对抗**  
越狱后的高远遥一很少只当刀手；他更常为怀恨者**绘制完美犯罪蓝图**，自己当观众或导演，留下 Good Luck 式签名。一的回应是「用自己的骄傲抓住你」——不是乐园币式悬赏，而是智力伦理对决。露西亚人偶等案出现近乎平行线的哲学对话：犯罪可否成艺术、侦探是否只是破坏作品的人。37 岁线中，高远可在特殊收容内以「宙斯」等代号远程指挥，也可再越狱编织「奥林匹斯十二神」式共犯网。世界状态：只要他在逃或在狱中能传话，任何「完美案件」都可能是画布。

⑱ **新系列 / R / 30 周年：少年时间线再点火**  
吸血鬼传说、狱门塾、蔷薇十字馆、雪鬼、亡灵校舍、人偶岛、香港九龙财宝等继续输出封闭本格；画风与节奏现代化，规则不现代化——仍无超能力。20 周年、Returns、30 周年把 17 岁的一反复拉回舞台，证明品牌核心永远是**差生侦探与美雪同行**。

⑲ **成人线：37 岁与爸爸**  
《金田一 37 岁之事件簿》：一进入社畜/公关现实，推理热情被加班与房贷磨损；美雪推进婚姻与职业（乘务精英）。案件舞台换成摩天住宅贵妇圈、京都美人花道、函馆酒店、狼人杀聚会、天空密室等**成人封闭社交**。剑持退休仍会冲现场；明智升至高阶警官，资源更多也更官僚。高远的棋没有下完。《金田一爸爸之事件簿》：再数年，一与美雪育儿（九十九等子嗣线），私家侦探身份与父亲身份互抢时间。系列**未完结**；少年线可随时被作者重启。

⑳ **结局方向（常青用法）**  
没有「魔王死亡世界和平」的总结局。每一长篇以凶手逮捕、自杀、被杀或感化为局部句号；宏观上：一与美雪从朋友以上恋人未满走向婚姻家庭；一从差生变为警视厅默许的不规则；高远作为永恒镜子存在。契约者应把本世界读成 **可反复生成的本格案件宇宙**，选准时间锚点后只推进该锚点的封闭圈，勿强行十年剧情一天内播完。

**【势力图谱】**
- **警视厅搜查一课**：国家司法正面。代表 **剑持勇**（现场、人情、制伏）、**明智健悟**（精英指挥、跨辖逻辑）。与一是非官方共生：要他的脑，又要为程序、记者会、未成年人介入擦屁股。
- **地方警察署 / 县警**：每案属地势力；初期常抓「外来少年/可疑旅客」，被打脸后移交或协作。是契约者最容易起冲突也最容易借证件的层级。
- **私立不动高中（校方·戏剧部·谜研会）**：日常势力。社团既提供旅行/集训的入场券，也提供嫌疑人名单；校方优先名誉，可能压消息。
- **艺能事务所与偶像工业（速水玲香侧）**：资本与形象管理。诱拐案暴露保护不足、炒作与封口；媒体机位本身改变凶手行为。
- **封闭村社 / 财阀分家 / 机关宅主人**：地方宗法与遗产法。异人馆村、飞驒宅、黑死蝶宅、魔神遗迹「守护者」——外人婚礼、参观券、继承会议是常见入场借口。
- **幻想魔术团等艺能/旅团**：魔术列车案的封闭乘客团；技能本身是诡计零件。
- **犯罪侧「怪人」与高远网**：无统一组织时是一案一伙；高远出现后形成 **设计师—执行者** 结构，后期主题化共犯（奥林匹斯意象）。与警方、一三方拉锯。

**【世界剧情线 · 经典事件补述】**

**歌剧院旅馆**  
戏剧部孤岛集训，暴风雨切断交通；「歌剧院怪人」面具杀人。金田一一第一次以「爷爷的名义」完整解谜，确立封闭空间＋悲剧动机公式。美雪遇险、剑持封锁现场、学生互相猜疑——一切后续大案的原型。

占星术主题的分尸重组与村中旧怨；建筑与仪式双重诡计。把「传说」拆成物理通道与时间差，是系列对民俗恐怖的标准处理。

背水村暴风雪、别馆电话线断、速水玲香与明智登场。冰面、绳索、取景组动线构成机关；明智「三小时结案」的自信与一的哈欠形成对照。玲香成为长期羁绊人物。

**魔术列车**  
行驶密室、行李架、地狱邀请函；**高远遥一**作为犯罪艺术家全面登场。把谋杀当作品、把金田一当唯一观众的宿敌关系自此钉死。

**塔罗山庄／蜡人形城／墓场岛等**  
机关宅、蜡像通道、孤岛童谣……公式熟练后，重点转向动机的社会悲剧（遗产、欺凌、战争遗恨、偶像工业）。写任务时择一即可，勿堆砌全部机关说明书。

**高远线长尾**  
越狱、远程指挥、诱拐玲香、与一的对称「遗产」（祖父之名 vs 母之死）。37 岁线中高远仍可再布局——系列未因少年线收束而消灭宿敌。

少年线可循环开局；37 岁社畜线与爸爸线是同一人物未来层。契约者选定锚点后勿混龄。

## 阶位切入点

> 阶位↔原作：一阶≈全体凡俗（平民～刑警～少年侦探～犯罪艺术家）。世界顶点为**高远遥一式智力犯罪与金田一式推理对决**（仍属一阶后期，破坏力不超过热武器/机关杀伤，无超凡）。切入无需「规避超阶」；须规避**被当犯人击毙、封闭圈灭口、高远棋局炮灰、暴风雪与海难环境死**。

**一阶（凡俗本格 · 封闭杀人与少年侦探线）**

切入身份/时点：契约者任选贴地身份——①私立不动高中转校生/戏剧部或谜研会临时会员；②剑持班底实习巡查或县警协勤；③速水玲香节目组场务/艺能临时工；④山庄·旅馆·列车餐车季节工；⑤秘宝岛或墓场岛旅行团编外向导。时间锚点优先：戏剧部赴**歌剧院旅馆**当日；**雪夜叉**取景团刚抵背水村、暴风雪警报拉响；**魔术列车**发车前夜（地狱邀请函已寄达）；高风险可选 **File10 全国通缉夜**（一被嫁祸）或**玲香诱拐赎金交接夜**。

初始事件（开局第一幕必须立刻入局）：  
- **歌剧院锚点**：渡船班次取消公告刚贴上，发电机第一次熄火；后台镜中掠过怪人面具，同屋同学的剧本角色名被红笔划掉——女声尖叫从走廊炸开。你要在「跟着金田一一冲进化妆间」与「听从剑持封锁、先做证人笔录」之间选边，选边决定你在封闭圈里的站位。  
- **雪夜叉锚点**：别馆电话线被确认掐断，本馆窗上出现雪花掌印；**明智健悟**当众限时结案，**金田一一**却盯着谷间反光发呆。你在柴房捡到半截烧焦绳头与湿靴印——这证物让你立刻从路人变成「必须被灭口或被保护」的变量。  
- **魔术列车锚点**：行李架滴下黏稠红色，团长从行驶密室蒸发；笑容过分礼貌的魔术师（高远侧影子）邀请你当舞台志愿者。拒绝=被标记为不配合的观众；答应=被写进节目单与死亡时间表。  
- **嫁祸/诱拐锚点（可选）**：电视字幕打出「少年侦探涉嫌杀人」或「玲香被绑·赎金一亿」——你若出手藏匿一或护送赎金，就与警视厅和媒体同时开战。

开场白建议：「暴雨把码头吞没的时候，你才明白『集训』是一张单程票。旅馆吊灯摇晃，有人用变了调的嗓子唱着剧里的咏叹调——金田一一揉着眼睛说『好困』，七濑美雪已经拉住你的袖口：别单独行动。走廊尽头，歌剧院怪人的面具对着你轻轻鞠躬，像在邀请你上场。」

关键NPC立场：  
**金田一一**——表面不可靠，实则最快把你当「线索载体」；不瞒物证则护你，也强行把你拽去最前线；若你撒谎，他会比对时间表拆穿。  
**七濑美雪**——先礼貌后警惕；对一怀恶意者记仇；遇险时既是人质位也是情报位，能提供社团名册与房间分配。  
**剑持勇**——要证件、要笔录、要你别添乱；救人/缴获凶器后把你当临时部下罩着；最烦你「私自冲进现场」。  
**明智健悟**——默认你是累赘或嫌疑人；你提出不输逻辑的假说后改为敬语式嘲讽，并可能征用你做实验性验证。  
**高远遥一**（列车及之后锚点）——视有趣外来者为颜料、观众或棋子；夸你「有品位」时最危险。  
**速水玲香**（雪夜叉/塔罗/诱拐）——公众笑容，私下易托付；靠近她=站上靶心与镜头。  
**金田一二三**（若短篇/后期锚点）——毒舌小孩，可能先于大人发现你藏的小动作。

主线钩子/支线：  
主线——歌剧院怪人按角色表杀人；雪夜叉冰桥不在场；魔术列车尸体转移与地狱傀儡师真身；金田一少年被嫁祸逃亡；塔罗山庄限时救玲香；玲香诱拐的赎金夜路与高远阴影；异人馆村「第七具」人数崩坏；飞驒机关宅猎头武者；银幕预告下的电影研究社灭口。  
支线——帮美雪保管被掉包的钥匙/房间牌；替剑持盯住擅自离队者并做双人互证；与明智赌「谁先指出伪证」；拒绝高远的艺术合作并活过终章；在暴风雪夜维护发电机与取暖燃料；为墓场岛生存游戏改写「淘汰规则」以拖延下一刀；保护知情者活到自白场景。

危险度/规避：中～高（凡俗顶格）。致命源＝凶手灭口、警方误判开枪、暴风雪/海难/机关宅夹杀、毒酒、高远布局替死、媒体曝光后的私刑风险。无超阶存在，**禁止**用封印/削弱解释任何人。规避：夜间独行、无见证对质、吞不明饮食、在全员集合前单挑凶手、对高远秀「我拆穿你了」却无退路与录音。

任务方向/奖励：活过封闭圈；保全 **七濑美雪** / **速水玲香**；协助 **金田一一** 完成可呈堂的物证链；协助 **剑持勇** 合法逮捕；阻止 **高远遥一** 的作品完成或至少截获其蓝图残页。奖励取向严格一阶：现场勘查笔记、平面图复原术、不在场证明拆解经验、机关宅局部图纸、魔术识破手法、警方协力人情、剧团/旅馆内部通行证；**禁止**超凡功法、二阶战力、复活、真诅咒道具。
