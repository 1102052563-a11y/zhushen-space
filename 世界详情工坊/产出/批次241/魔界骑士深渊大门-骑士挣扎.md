# 魔界骑士深渊大门-骑士挣扎
<!--meta lib=主库 tiers=六-->

## 剧情

**【作品来源】**
「魔界骑士深渊大门」系列切片之**骑士挣扎**。骨架取自Lilith《对魔忍》宇宙公开设定：魔界骑士**イングリッド（英格丽德）**、吸血鬼魔神**埃德温·布莱克**、组织**诺玛德（Nomad）**、对魔忍五车体系（井河阿莎姬/樱、八津紫等），以及OVA《魔界骑士イングリッド》。**深渊大门**为乐园多阶段协议：魔界—人界异常通道，开启后污染、契约改写、骑士誓约可被深渊化反噬。本档锚点不是宏大战场全景，而是**英格丽德个人在忠君、骑士荣誉与深渊侵蚀之间的内斗与近战挣扎**。卡片锚点：骑士誓约濒临改写·迷彩下的单骑突围。

**【世界定位】**
人界东京暗部与魔界门廊交界的“窄走廊”战场。宏大沦陷与最终防线在其他切片展开；此处镜头贴在英格丽德的呼吸、伤口与命令冲突上：布莱克要她守门/清障，故乡要她勤王，深渊要她改旗。一句话：**最强骑士的战场，有时只有一把剑长。** 契约者六阶以「诺玛德监军 / 五车特务 / 魔界残骑士联络人」贴身入场。

**【世界观 · 力量体系】**
- **对魔忍**：血脉忍法+对魔忍装+忍刀/体术，对抗魔族与人魔外道；组织上连结五车学园、政府暗部。
- **诺玛德**：布莱克主导的跨国暗组织，魔族能力产品化（改造、斗士、药剂、光学迷彩）。
- **魔界骑士**：魔界武勋爵位；魔剑/鞭、魔装、迷彩、誓约绑定主君。英格丽德对布莱克近乎绝对忠诚，亦是其最锋利的刀。
- **深渊大门协议**：通道异常；污染可改写誓约纹章，使骑士从“为主君而战”变为“为深渊而渴”。**深渊化**是本档核心威胁——不是瞬间变Boss，而是荣誉感被一寸寸蛀空。
- 死亡：下忍/凡人即死；高阶魔族可改造续命；大门侧回收残滓为污染源。

战力阶梯：凡人≈一；下忍/低魔≈二；中忍/成型魔物≈三；上忍（阿莎姬、胧级）/干部魔将≈四；东京级城市威胁≈五；**魔界骑士全力、深渊化临界、地区污染、布莱克阴影≈六阶（本档）**。布莱克魔神化与深渊本源更高，六阶情报优先。

乐园阶位映射：凡人≈一；下忍≈二；中忍≈三；上忍≈四；城市联合作战≈五；**骑士挣扎/深渊化临界/门廊近战≈六阶**。顶点=存在·情报优先/条件性胜利。

**【地理 · 舞台】**
- **门廊狭窄段**：污染最浓、能见度最低，适合迷彩与伏击。
- **诺玛德临时指挥车**：布莱克远程指令入口，干净到残酷。
- **五车观测盲区**：阿莎姬队无法及时支援的“十分钟空洞”。
- **废弃地下铁/下水道战场**：英格丽德单骑突围的典型舞台。
- **誓约纹章共鸣点**：骑士个人纹章与门廊共振处，深渊化发作高发。
- **人界平民街区边缘**：挣扎失败则污染溢入日常。

**【世界剧情线】**

① **命令冲突的早晨**  
英格丽德同时收到三份指令：布莱克的模糊令（“维持有趣的平衡”）、魔界残部的勤王密信、门廊传感器的深渊浓度警报。她选择先到门廊——不是因为最爱门，而是因为**失控的门会让另外两份命令都失去意义**。

② **第一次流鼻血**  
污染改写誓约时，骑士先出现生理信号：流鼻血、幻听主君笑声、武器握感“变甜”。英格丽德在巷战中斩杀三只溢魔后发现自己**没有对最后一击感到厌恶**——这是深渊化早期。她用自残式痛觉打断幻觉（刀背击伤），暂时夺回主权。

③ **同袍影子战**  
深渊化的旧骑士同僚不正面挑战荣誉，而用**她熟悉的队形**围攻，逼她用“对魔界的刀”。光学迷彩第一次用于逃出友军包围而非暗杀敌人。她点名不点杀：点到谁，谁短暂清醒——这是骑士残留的教范。

④ **フュルスト的“治疗”**  
魔科医**フュルスト**提出把深渊污染“医疗化”：植入抑制针、改造成可控狂化。条件是英格丽德接受对布莱克更彻底的工具化。英格丽德拒绝完整方案，只拿走一剂**短时清醒药**——她要挣扎，不要被做成产品。

⑤ **五车的十分钟**  
**井河阿莎姬**给出有限窗口：十分钟火力掩护，换英格丽德把污染核心标记钉入门廊。**井河樱/八津紫**负责收容与断后。若超时，五车撤退，骑士必须独自完成。这是国家机器与个人骑士的残酷分工。

⑥ **名场面：迷彩下的独白**  
英格丽德关闭迷彩，让敌人看见她的脸，只为说一句：“我仍是魔界骑士。”随即再开启迷彩斩断污染触须。这一秒不是战术最优，但是**身份声明**——本档主题句。

⑦ **妹妹与软肋线（系列互文）**  
系列中英格丽德与血亲/妹妹相关威胁在其他切片展开；本档只保留压力：布莱克与敌人都知道**软肋存在**。契约者若用软肋要挟，会立即被英格丽德列为灭口级目标——除非你用软肋去**救**而非控。

⑧ **阶段结局方向**  
- **稳住**：深渊化被压回临界下，门廊暂封，英格丽德带伤向布莱克复命，内心裂缝扩大。  
- **半堕**：她完成任务但手段已是深渊式屠杀，五车与残骑士同时忌惮。  
- **挣脱失败**：迷彩失控，骑士成为门廊代行，直接接“英格丽德深渊化”类切片。  

⑨ **微观政治**  
清醒药剂量、十分钟授权书、布莱克录音的法律效力、污染浓度读数造假、谁有权宣布“骑士已堕落”。

⑩ **蝴蝶**  
可改：十分钟是否延长、清醒药真伪、是否点名救下某同袍。不可改：深渊改誓约的生理机制、布莱克冷棋、英格丽德荣誉内核（可碎不可无中生有成圣母）。

**【主要人物】**
- **イングリッド（英格丽德）**｜性格：冷彻、忠君、极重骑士荣誉；深渊化时自尊与渴互相撕。｜装备·能力：魔装、光学迷彩、魔剑/鞭、六阶地区级近战。｜弧光：接冲突令→生理预警→斩同袍边缘→声明身份→稳住/半堕/失败。｜立场：六阶核心；契约者先表明“不把她当产品”。
- **埃德温·布莱克**｜性格：求刺激与平衡，对刀有占有无温情。｜能力：吸血鬼魔神化，跨阶。｜弧光：远程下模糊令，观察挣扎本身。｜立场：情报优先。
- **フュルスト**｜性格：崇拜布莱克、轻视英格丽德。｜能力：魔科医、改造刻印。｜弧光：推销可控狂化。｜立场：阴谋医疗。
- **井河阿莎姬**｜性格：责任与决断。｜能力：光速剑体、上忍顶点向。｜弧光：给十分钟，不给无限。｜立场：可临时同仇。
- **井河樱**｜性格：护短教官。｜能力：影系。｜弧光：收容伤员与人质线。｜立场：中坚。
- **八津紫**｜性格：冷静怪力。｜能力：大斧、校医。｜弧光：断后与医疗。｜立场：可靠。
- **甲河朧 / 邪恶胧**｜性格：悲壮或残虐。｜能力：空蝉/催眠刻印等。｜弧光：深渊实验常客。｜立场：假友风险。
- **甲河明日香**｜性格：钢铁死神。｜能力：超粒子炮、义肢兵器。｜弧光：清场变量，可能误伤政治。｜立场：高火力。
- **桐生佐马斗**｜性格：前魔科医。｜能力：魔界医术。｜弧光：清醒药真伪鉴定钥匙。｜立场：灰市。
- **深渊化旧骑士（具名任务生成）**｜性格：曾荣耀。｜能力：骑士级+污染。｜弧光：围攻与被点名清醒。｜立场：可救可杀。
- **门廊代行幼体**｜性格：非人。｜能力：污染喷吐。｜弧光：环境Boss。｜立场：优先标记摧毁。
- **诺玛德监军**｜性格：要报告。｜能力：权限与通讯。｜弧光：逼英格丽德工具化。｜立场：契约者可能的假身份。

**【势力图谱】**
诺玛德指挥链；五车特务窗；魔界残骑士密信网；深渊门廊代行；魔科医改造派。敌友：英格丽德同时被三方当刀；谁先把她当“人/骑士”谁先获得短暂信任。

**【贵重物品】**
短时清醒药；未污染誓约纹章碎片；十分钟授权伪/真章；门廊标记钉；布莱克录音晶片；迷彩发生器核心；污染浓度黑盒。

**【隐藏剧情 · 伏笔】**
挣扎是深渊化切片的前史；布莱克可能需要“将碎未碎的刀”；五车十分钟是政治上限不是战斗上限；迷彩下的独白会被某人录下成为日后审判证据。

**【大事记时间线】**
三令齐至 → 门廊初战流鼻血 → 同袍围攻 → フュルスト提案 → 五车十分钟 → 迷彩独白 → 稳住/半堕/失败 → 复命或失踪。

**【叙事基调 · 雷区】**
近身、窒息感、荣誉悲剧。忌：把英格丽德写成无脑忠犬；六阶秒杀布莱克；忽略深渊化生理细节；用成人向凌辱当唯一剧情（可暗线不可替代主轴）。锚点：她擦掉第一滴鼻血、决定继续持剑的瞬间。



**【骑士挣扎·专项战役档案】**

**档案一：三令并行**
布莱克模糊令、魔界勤王密信、门廊浓度警报同时到达。英格丽德的选择顺序即人物弧光：先门廊（失控会废掉一切），再处理忠与乡。契约者若逼她先复命布莱克，会加速工具化；若逼她先勤王弃门，污染溢入人界。

**档案二：深渊化生理谱**
阶段0无感；阶段1流鼻血与幻听；阶段2武器握感变甜、杀意愉悦；阶段3纹章变色、迷彩失控；阶段4代行化。本档主战场在1到2。痛击、短时清醒药、被信任者点名，可暂时压回。

**档案三：同袍围攻教范**
深渊化旧骑士用旧队形围攻，逼英格丽德对魔界刀。她点名不点杀：点到谁谁短暂清醒。契约者可学这套点名救援，但点错名会触发集体狂化。

**档案四：フュルスト医疗陷阱**
抑制针等于可控狂化合同。完整方案拒绝，只取短时药。药真伪需桐生佐马斗或紫级医疗鉴定。假药会把阶段1直接推到3。

**档案五：五车十分钟政治**
阿莎姬给的是政治窗口不是战斗极限。超时五车撤，骑士独留。樱负责收容，紫断后。授权书原件是战后追责关键物证。

**档案六：迷彩独白名场面**
关迷彩露脸声明“我仍是魔界骑士”，再开迷彩斩触须。有人录音等于日后审判非堕落证据。契约者抢录音或毁录音改变英格丽德政治命运。

**档案七：软肋线压力**
系列中妹妹与血亲威胁在其他切片爆发；本档只保留压力源。用软肋要挟等于灭口级敌意；用软肋去救等于短暂最高信任。

**档案八：门廊地形杀**
铅白天、甜锈味、酸滴、幻视主君微笑。六阶环境持续掉状态。标记钉必须在喷口收缩微窗打入。

**档案九：结局三种写法**
稳住：临界下压，复命，裂缝在。半堕：任务成，手段成深渊，双方忌惮。失败：迷彩失控，接深渊化切片。

**档案十：微观道具清单**
清醒药剂量刻度、污染仪、纹章碎片、标记钉、布莱克录音晶片、迷彩核心、十分钟授权章、密信火漆。

**【可介入事件·六阶】**
1. 在地下铁为英格丽德断后十秒。
2. 鉴定清醒药。
3. 点名救一名旧骑士。
4. 切断诺玛德实时摄像头。
5. 护送标记钉到喷口。
6. 录下迷彩独白。
7. 伪造一份“已复命”报告帮她拖时间。
8. 拦下フュルスト手术车。
9. 向阿莎姬申请第11分钟（极难）。
10. 在半堕边缘用痛、名、药三选一拉回。

**【人物立场细表】**
英格丽德要荣誉完整。布莱克要有趣平衡。阿莎姬要人界少死人。フュルスト要产品。旧骑士要被点名或被解放。监军要报告。佐马斗要证明医术可逆。明日香要清场效率。胧要实验数据。

**【感官与对话钩子】**
甜锈味是污染；干净地板味是诺玛德；消毒水是五车医疗。英格丽德短句：“突围。”“数到十。”“我仍是。”布莱克远程：“别让我无聊。”阿莎姬：“十分钟，一分钟都不多。”

**【与系列其他切片接口】**
前接大门开启与人界危机压力；旁接魔界沦陷勤王；后接深渊化或最终防线。本档不写宏大沦陷全景，镜头贴呼吸与伤口。

**【对魔忍宇宙锚点补强】**
五车学园与对魔忍装是人界侧制度；诺玛德把魔族能力产品化；魔界骑士是爵位与誓约。深渊大门协议让三者在门廊狭窄段被迫同一帧出现。写六阶时保持“近身窒息感”，不要写成大军团沙盘。


**【临界骑士的小时志·续0】**
00:00 三令齐至。英格丽德把布莱克录音晶片与勤王密信并排放在膝上，翻译本身就是站队。

00:40 门廊甜锈味。第一滴鼻血。她用刀背击伤确认痛觉仍属于自己。

01:10 旧骑士队形出现。迷彩开启。点名第一人，对方眼底闪过清醒又沉回去。

01:55 フュルスト手术车灯。她拒完整方案，夺一剂短时药。药瓶刻度被她用指甲划痕标记，防调包。

02:30 五车频道接通。阿莎姬只给十分钟。樱要坐标，紫要伤型预判。

03:10 标记钉突入。喷口收缩微窗。环境酸滴腐蚀魔装漆。

03:40 迷彩关闭。独白。录音笔红点在某处亮起——可能是契约者，也可能是监军。

04:00 窗口结束。五车撤。她仍在。稳住或半堕在这一刻分叉。

深渊化不是魔王变身条，是荣誉被蛀空的过程。写六阶要让每次杀敌都问：这刀是骑士的还是深渊的。

诺玛德监军要的是“可用报告”，不要故乡。五车要的是人界少死，不要骑士荣誉完整。勤王密信要的是旗，不要布莱克。英格丽德站在三角中心。

软肋线：系列其他切片的妹妹威胁在此作气压。你若用软肋控她，她会把你当深渊同谋；你若用软肋救她，她会给你背对背的三分钟。

与魔界沦陷切片分工：沦陷写宏观府邸与旗；本档写呼吸、伤口、十秒断后。不要重复黑日旗大场面，写隧道灯一盏盏死掉。

与最终防线切片接口：若本档失败，污染前锋提早抵达人界街垒。若稳住，最终防线多出半日准备。

道具真伪学：清醒药见光、纹章碎片会烫手、授权章有暗纹、密信火漆有旧魔界语拼写陷阱。

切入身份/时点：以「诺玛德随行记录官 / 五车十分钟联络员 / 残骑士密信使者」切入，锚定**英格丽德已出现流鼻血预警、同袍围攻将起**。

初始事件：你在地下铁隧道撞见迷彩闪灭的英格丽德——她左手血，右手剑，面前三个戴旧骑士团纹章的“友军”正举刃。她对你只说：“数到十，选边。”污染甜锈味涌来，你的污染仪指针爆表。

开场白建议：「隧道灯一盏一盏死掉。英格丽德的迷彩像碎玻璃，拼不出完整的人影。她问你是监军、是五车，还是故乡的信。你答错一个字，剑就会先落在你肩上——不是因为恨，是因为她正在用痛确认自己还是骑士。」

关键NPC立场：**英格丽德**（要清醒、要完成门廊标记）；**埃德温·布莱克**（远距观赏）；**井河阿莎姬**（守时撤退）；**フュルスト**（卖药）；**深渊化旧骑士**（围杀或被点名）；**八津紫**（医疗窗口）。

主线钩子/支线：主线=帮她把标记钉打入门廊并撑过十分钟。支线：鉴定清醒药真伪；录下独白作为日后非堕落证据；救出一名可清醒的旧骑士；切断诺玛德逼她工具化的实时摄像头。

危险度/规避：极高。规避深渊本源喷口、布莱克若现身、英格丽德半堕时的无差别斩击。环境持续掉SAN/状态。

任务方向/奖励：目标=临界压回或可控半堕的政治定义权。奖励：纹章碎片、迷彩残核、五车临时通行证、英格丽德“债”或“杀意标记”。



扩展执行段：六阶开局你在隧道数秒。英格丽德要你在监军、五车、密使里认一个。认监军：她把你当监视器，刀背相向。认五车：她要你倒计时报秒。认密使：她要看火漆，假信直接斩。

战斗中优先保护她的声明窗口——关迷彩露脸的那三秒若被狙击，身份崩溃加速深渊化。你的胜利可以是“让她说完那句话并活着打开迷彩”。

奖励示例：纹章碎片可作通行、短时药残剂、五车临时通道、英格丽德一句“你不是产品”。禁止奖励永久免疫深渊化。

补充支线：追回被深渊化同袍抢走的迷彩核心备份；在诺玛德指挥车植入假报告；给樱的收容点送止血剂；用污染仪找出真正喷口而非幻视喷口。


**【六阶切入执行加厚·续0】**
身份三选一后果表写进开场。监军被刀背；五车被要求报秒；密使被验火漆。

初始事件：三个旧骑士举刃，污染仪爆表，英格丽德说数到十。你的第一个动作决定她是否把你当人。

战斗优先级：护声明窗口>护标记钉>击杀。击杀过多会推她阶段2。

药检小游戏：摇、闻、滴血试纸（设定化）。假药推阶段。

录音线：独白三秒，你在场或不在场改变审判结局。

撤离：五车走后你可留可撤。留则高危高回报。

奖励与惩罚：人情债/杀意标记；纹章碎片；迷彩残核；禁止永久抗污染。
