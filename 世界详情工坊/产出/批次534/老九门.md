# 老九门
<!--meta lib=主库 tiers=一、二、三、四-->

## 剧情

**【作品来源】**
《老九门》南派三叔著，为《盗墓笔记》宇宙前传向作品；2016年爱奇艺/东方卫视剧集《老九门》（The Mystic Nine）扩大公众认知（陈伟霆张启山、张艺兴二月红、赵丽颖尹新月等）。本世界以**民国长沙九门提督格局+矿山之谜+抗日暗线**为主，人物真名以小说/剧集公开设定对齐。**搜笔趣阁检索未收录本书页**，来源以维基与公开剧情介绍为准。

**【世界定位】**
民国长沙：九门镇守倒斗江湖。**张启山（张大佛爷）**为查鬼车与矿山，联手**二月红**、**齐铁嘴**等，对抗日谍、内鬼与矿山异象，并与北平**尹新月**结缘。它是盗墓笔记世界观的“上一代战争与规矩”。

**【世界观 · 力量体系】**
1. **九门技艺**：各门绝活——狗五嗅觉犬、二月红竹竿走壁与戏曲掩护、陈皮铁弹与九爪勾、半截李残兵体系、霍家女软功等。
2. **军事武装**：张启山布防官身份，枪械、兵队、城防。
3. **矿山异象**：超自然机关/疫病/心智污染级威胁（按区域灾难理解，不爆星）。
4. **情报与钱**：新月饭店、盘口、三点天灯散财。
**乐园阶位映射（宁低勿高）**：跑腿/新兵/小贼≈**一阶**；门内骨干/张日山级≈**二阶**；二月红、陈皮阿四、齐铁嘴谋略场≈**三阶**；张启山巅峰与矿山核心灾厄≈**四阶**。**顶点≈四阶**。

**【地理 · 舞台】**
长沙城与火车站；城外矿山；北平新月饭店；白乔寨/黑乔寨；东北张家线索；战场与戏台双重空间。

**【世界剧情线】**
① **鬼车入长沙**：神秘列车，布防官张启山奉命查；齐铁嘴同行。
② **矿山与日谍**：矿山被日本人窥伺；国家大义压过江湖金盆。
③ **求二月红**：二月红因爱妻**丫头**重病已洗手；张启山北上新月饭店求药。
④ **三点天灯**：为丫头散尽家财连点天灯；识**尹新月**，暗生情愫，得助返长沙。
⑤ **丫头之死与觉醒**：药无效，丫头故；二月红悲极后发现祖辈与矿山关联，重出与张联手。
⑥ **九门内斗与外敌**：陈皮阿四心狠、霍三娘立场摇摆、陆建勋背叛、裘德考与田中良子等外敌线；矿山之谜层层剥开。
⑦ **情感与家国**：张尹姻缘；二月红情义；陈皮对师娘扭曲情感酿祸。
⑧ **结局方向（剧集主线）**：矿山危机与抗日暗线收束中，九门人物命运分水——为盗墓笔记世代埋下佛爷、红官、狗五、解九等传说。

**【势力图谱】**
老九门上中下三门；国民党地方军政；日本特务与伪合作者；新月饭店资本；乔寨土司；东北张家。

**【贵重物品】**
矿山核心物（异象源）；天灯与求药；戏台下盗洞机关；各门兵器暗器；佛爷府大佛象征。

**【隐藏剧情 · 伏笔】**
矿山与九门祖辈契约；陈皮杀老四门规；丫头之死是否被延误；裘德考与盗墓笔记正传的跨代恶意；佛爷与二月红情义传说如何传到吴邪一代。

**【大事记时间线】**
鬼车 → 矿山疑云 → 北上求药三点天灯 → 丫头死 → 九门重聚破局 → 内外奸清算 → 家国战役收束。

**【叙事基调 · 雷区】**
民国江湖热血、戏与墓、家国。忌：把九门写成无敌仙侠门派；忌消费丫头之死无后续；忌与正传张起灵强行同一人设糊弄。锚点：火车站鬼车开门的雾夜，或新月饭店天灯点起时。

## 阶位切入点

> 阶位↔层级：一阶≈跑腿新兵；二阶≈门内骨干；三阶≈红官/陈皮/齐铁嘴场；四阶≈佛爷与矿山核心。顶点四阶。

**一阶（长沙街巷学徒）**

切入身份/时点：火车站脚夫、戏园跑堂、布防部文书；鬼车抵达当夜。

初始事件：鬼车开门滚出无面尸，齐铁嘴只看一眼就走；张副官抓住你：今晚目击名单上有你的名字，跟查或灭口二选一。

开场白建议：「蒸汽与湘江夜雾搅在一起，铁轨像两根伸向矿山的黑骨。有人在你掌心按了个刀疤印：九门的事，知道太多与太少都死。」

关键NPC立场：**张日山**、**齐铁嘴**、**张启山**、站长顾庆丰。

主线钩子/支线：钩子=跟随初查鬼车；支线=送密信给二月红戏园、盯日籍商人。

危险度/规避：中。矿山核心勿进。

任务方向/奖励：短枪/匕首、九门外围人情。

**二阶（门内行动队）**

切入身份/时点：张府护卫或红官戏班武行；求药线前后。

初始事件：丫头咳血，张启山要北上；陈皮冷笑药在仇人手上。你被派先探新月饭店底——听奴已记下你的心跳。

开场白建议：「京胡拉到一半突然哑了。二月红卸了戏妆，眼睛却比台上更红：药若换我一条命，你换不换？」

关键NPC立场：**二月红**、**丫头**、**尹新月**、**陈皮阿四**。

主线钩子/支线：钩子=三点天灯资金链护运；支线=识破陆建勋的绊子、救霍三娘临时同盟。

危险度/规避：高。日谍与内鬼双线。

任务方向/奖励：精良暗器、饭店令牌、戏班密道图。

**三阶（九门联手）**

切入身份/时点：佛爷特使；丫头死后矿山总攻筹备期。

初始事件：九门议事桌上缺两席，矿山方向的狗突然集体暴毙；吴老狗袖中犬三寸丁狂吠不止。

开场白建议：「大佛影子压在长桌上，像另一位无声的门主。张启山拍案：今夜之后，不是门与门，是活与死。」

关键NPC立场：**张启山**、**二月红**、**解九**、**半截李**、**黑背老六**。

主线钩子/支线：钩子=破矿山外围机关；支线=清理陈皮私兵、截获田中良子密电。

危险度/规避：极高。

任务方向/奖励：门内中层地位、军械、矿山部分真相。
