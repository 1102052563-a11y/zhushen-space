## 剧情

**【作品来源】**
《我在美国搞内战》（亦常标《我在美国搞内战(韦恩)》）作者**捕梦者**，起点中文网都市·异术超能连载（书号约 1047777980）。文风属「美利坚斩杀线」一脉：把美国底层生存危机、帮派、退役军人成瘾、尸体产业链与医疗绝境写成可操作的地下城冒险，系统以魔域映射现实，讽刺与爽感并行。约 106 万字量级、三百余至近四百章仍在更新（公开目录可见「西雅图冰雨夜」「收尸人」「王庭反抗者」至中后期「借调洛杉矶」「德鲁伊之力」「韦恩遇袭」等）。作者另有《异常收藏家》《真实末日游戏》《末世狂喵》等。气质不是纯无脑升级，而是「资料感很重的黑色幽默」：抗生素买不起、鱼药顶替、收尸有报价单，系统却认真弹出「哥布林街巷确认」。**本档案以起点公开简介、目录与可核早章信息为准；连载未完结，不编终局、不伪造分卷名。**

**【世界定位】**
当代美利坚底层被「地下城冒险者系统」强制重写为**魔域地下城**：流浪汉、瘾君子、退役老兵、街头边缘人被扫描成哥布林、食尸鬼、退役恶魔骑士、堕落黑暗精灵、黑暗龙骑等；统治层被概括为**黑暗王庭**，顶点为**金色魔王**。主角**布鲁斯·韦恩**（通称**韦恩**）是中国人穿越进前童星、三婚破产、抑郁病弱的底层流浪汉躯体，开局即在西雅图冰雨夜踩着**斩杀线**求生。主线任务写死：**击败魔王，毁灭魔域地下城**——在写实层面即从底层抱团、技能拾取、威望与势力扩张，把「内战」式动荡从街头烧到更大舞台。一句话：斩杀线下的流浪汉，用地下城规则点燃美利坚。卡片生成时优先锚：**冰雨帐篷 + 系统装载 + 第一次收尸/巷战**，不要一上来写全国内战。

**【世界观 · 力量体系】**
本世界没有仙侠境界表，成长轴是**系统冒险者路径 + 现实热武器/组织/资源**，再叠加系统给出的奇幻映射标签。写战力时必须双轨：左边是现实（枪、药、人、钱、证件），右边是系统（技能专精、经验、威望、成就、战宠）。

1. **系统装载与环境判定**  
   系统扫描周围后确认「哥布林街巷」等魔域环境，装载**地下城冒险者系统**。面板含职业（初级冒险者、未选路径）、体力/血量、经验、技能栏、战宠、状态（极度饥饿、大病虚弱等）。金币可兑商店药水、卷轴。成就如「唯一超凡」提供地位与威望向奖励。环境标签会改写你对街巷的认知：同一条巷，游客看见涂鸦，系统标「哥布林街巷」；废弃堂区变「教堂地下墓场」；成瘾者窝点变「食尸鬼巢穴」。标签不改变物理，改变**任务与掉落逻辑**。

2. **斩杀线（生存铁律）**  
   血量跌破阈值（早章示例：HP 4/60）触发**斩杀线**：平均每小时再掉 1 点血，直至彻底斩杀。映射现实＝无抗生素、无食物、无取暖即死。低阶契约者入世第一课不是打怪，是**把血量拉回安全区**。任何「帅气硬扛」若导致重伤断粮，都会被斩杀线秒杀叙事。

3. **拾取 / 搜刮尸体**  
   对尸体执行搜刮可获得技能专精与记忆残片。公开桥段：搜刮**退役恶魔骑士**类尸体得**【M4A1 专精】**；开局搜刮**老约翰**得**【战术匕首使用（专精）】**并灌入中东作战记忆。技能可融合、可升级（中后期目录可见技能融合、炼器、灵媒等向）。拾取有伦理与法律双风险：现实里是盗尸/盗物，系统里是标准地下城操作——这是本作黑色幽默的核心摩擦。

4. **净化 / 挽救 / 引导**  
   挽救街头**堕落黑暗精灵**得经验；发现流浪**退役黑暗龙骑**可「引导其前往东方神秘国度」；对边缘人「净化」与立规矩，是抱团扩张的核心手段，而非单纯杀戮刷级。韦恩路线偏「圣徒式冒险者」：救人换经验与口碑，但救人也会引来王庭与黑帮的共同注视。

5. **威望与通缉**  
   与**黑暗王庭**对抗会触发通缉，同时**威望 +**。通缉不是单纯负面，而是系统承认你已进入「王庭棋盘」的凭证。威望可换话语权、任务品质与部分商店权限；过高威望则提高遇袭率（中后期「韦恩遇袭」即证明）。

6. **战宠与辅助**  
   开局**荒原猎犬**（串串小狗）入战宠栏，低等级但可警戒、撕咬、陪睡保暖。中后期有空间背包、德鲁伊向能力等系统扩展（以连载公开章名为锚，细节随正文）。战宠饿死、走失、被帮派打死，都会直接削弱生存面。

7. **现实战力层（与系统标签叠合）**  
   - 赤手空拳的斩杀线流浪汉、饿病交加的拾荒者：凡人炮灰，一刀一枪可死。  
   - 持刀、有街头经验、能组 2～3 人抢尸/抢药：小范围威胁，仍怕热武器。  
   - 有**枪械专精**、战术记忆、能带小队清巷：正式战力，可压一个街区的乌合之众。  
   - 帮派头目、有组织火力与地盘、系统高阶路径、触及「内战」级动员：已超出本库一～二阶硬刚范围。  
   - **金色魔王 / 黑暗王庭核心**：主线终局级，**超阶情报**，低阶只做规避与情报。  
   医疗与后勤是隐藏战力：能稳定搞到抗生素与热食的人，比只会开枪的人更能带队过冬。

**乐园阶位映射（对照《阶位战力图鉴》，宁低勿高）**：  
斩杀线边缘流浪汉 / 无专精拾荒者 / 街头哥布林级冲突 ≈ **一阶**；  
持枪械专精、战术记忆、稳定小队清巷 / 小帮派打手头目边缘 ≈ **二阶**；  
城市级帮派头目、系统高阶职业路径、跨城动员与「内战」级战事、黑暗王庭中高层、金色魔王 ＝ **超阶或本库外情报优先**（本库只覆盖一、二阶，禁止低阶硬刚）。

**【地理 · 舞台】**
- **西雅图（开局锚点）**：第一章「西雅图冰雨夜」。破帐篷、涂鸦巷、码头垃圾与霓虹「欢迎来到西雅图」并置；一街之隔是游客区暖光，一街之内是斩杀线。系统扫描出哥布林街巷、教堂地下墓场、食尸鬼巢穴等「副本名」，实为底层街巷、废弃堂区、成瘾者聚集点的魔域标签。适合一阶全流程：求药、收尸、第一次巷斗、结识韦恩。  
- **收尸产业链节点**：生物科技公司式「收尸人」收购（公开信息提及类「耶稣会圣玛丽生物科技公司」向渠道）——卖尸换现金/药，是早期生存经济。地点多在工业冷链、停尸转运、电话对接，不在霓虹主街。  
- **西雅图医神口碑圈**：不是医院，是黑市药、系统药水与「找韦恩能活」的帐篷—车库—安全屋网络。二阶任务多从这里发出：运药、护药、假药排查。  
- **洛杉矶及更大美利坚底层网**：中后期有「借调洛杉矶」等章，舞台从西雅图扩到南加州与跨城关系网；监狱、帮派走廊、退役军人社区、街头「黑暗精灵」聚集带均可成为任务点。娱乐工业残骸与前童星人设对照，适合身份线支线。  
- **系统语义下的魔域全图**：黑暗王庭治下的「魔域地下城」＝整座被资本与暴力分层的美利坚；金色魔王所在层不在一、二阶地图上展开坐标，只作主线北极星。  
写场景时抓住：**冰雨湿冷、帐篷霉味、枪油与廉价抗生素、霓虹反光在水坑里碎成金币色**——系统弹窗与现实气味必须同框。禁止写成中世纪地牢而删掉美元、医保、帮派刺青与警车。

**【世界剧情线】**
连载未完结，下列按**可公开阶段**梳理因果链，供 AI 判断任意时点世界状态；不写假结局。每个阶段写清：起因、事件链、名场面感、阶段结局、对后续因果、可介入口。

**① 西雅图冰雨夜 · 斩杀线觉醒（开局）**  
起因：中国人意识坠入**布鲁斯·韦恩**躯体——二十三岁前童星人设、三婚破产、抑郁病弱、睡在西雅图冰雨帐篷。  
事件链：**荒原猎犬**舔醒他 → 旁侧**老约翰**（退役老兵、强化剂/阿片成瘾、家庭破裂）用残缺巫毒仪式「献祭」过原身，临死交出 **USB**，托付找到女儿**艾丽**并给报酬，警告「他们在找你」→ 老约翰断气 → 系统完成环境扫描，装载地下城冒险者系统，宣布主线**击败魔王，毁灭魔域地下城**，支线交 USB 给艾丽 → 韦恩搜刮尸体，获**战术匕首使用（专精）**与中东战术记忆碎片 → HP 跌破斩杀线 → 两名系统标注为「肮脏的哥布林」的拾荒者逼近帐篷，意图收尸倒卖 → 第一次挑战：杀死/击败/赶走。  
名场面感：游客区霓虹与帐篷腐甜同框；系统冷字与冰雨齐下；狗比人先站队。  
阶段结局：一阶最底，资源＝刀、狗、手机、几块钱、一份遗嘱式支线；活过第一夜才有后续。  
可介入：帮抢/护 USB、联手打跑拾荒者、抢先搜刮技能、偷卖尸体、报警（通常无效）。

**② 收尸人 · 他乡故知 · 约翰·威克式立威（早期生存）**  
起因：血量与体温要求立刻变现。  
事件链：联系收尸渠道卖尸体换生存金 → 用手机、技能记忆与狗完成第一场巷战级冲突 → 目录「他乡故知」「你就是约翰·威克」：底层传说给干净利落处理威胁者贴「威克」标签，完成**威望原始积累**（不是真成电影角色）→ 药、热汤、干衣、抗生素成为比经验更紧的硬通货。  
阶段结局：暂时脱离小时级斩杀倒计时，但任何重伤/断粮都会重新触发。  
可介入：当收尸中间人、伪造尸体完整度、用「威克」谣言吓人、用真药换人情。

**③ 王庭反抗者 · 生死之交 · 抱团雏形**  
起因：个人求生不够，系统与现实同时逼人站队。  
事件链：第五章「王庭反抗者」把**黑暗王庭**从系统旁白拉进人事 → 底层出现自认或被系统标注为反抗王庭的人/团伙 → 韦恩以「挽救、净化、立规矩」而非单纯屠戮扩大圈子 → 简介中的**堕落黑暗精灵**（经验 +30）、**退役黑暗龙骑**（可引导向东方）成为此阶段模板事件 → 生死之交在物资与情报共享中结成。  
阶段结局：一阶后期～二阶门槛，开始有「小队」概念。  
可介入：替反抗者运刀、把黑暗精灵从「经验包」改造成线人、谈龙骑东行条件、在韦恩与反抗者之间吃两头。

**④ 技能融合 · 医神口碑 · 西雅图势力芽**  
起因：救人比杀人更能稳定产经验与任务。  
事件链：技能融合、灵媒、炼器等系统横向 → 「西雅图医神」式口碑：系统商店 + 拾取医技 + 黑市药 → 「找韦恩能活」成为街区暗号 → 信誉＝任务源＝经验＝枪械与防具来源 → 核心矛盾：救人越多，越容易被黑暗王庭与现实黑帮同时盯上。  
阶段结局：安全屋与运药线成型；假药、内鬼、举报成为日常。  
可介入：鉴假药、护运抗生素、打掉抢药小队、用口碑换枪械专精载体（尸体/教官）。

**⑤ 通缉与威望 · 唯一超凡**  
起因：影响力越过「巷内传说」阈值。  
事件链：遭到黑暗王庭通缉，威望 +20 → 解锁成就**唯一超凡** → 系统承认稀缺身份 → 「唯一」意味着同规则超凡席位被占，后续第二超凡如何出现成为中期戏剧点 → 小队枪械专精成型，开始跨街区行动。  
阶段结局：二阶可玩核心区；韦恩从「能打的病号」变成「棋盘上的子」。  
可介入：卖通缉情报、帮洗通缉痕迹、借威望接高难支线、制造「第二超凡」谣言搅局。

**⑥ 南下 / 借调洛杉矶 · 跨城棋盘**  
起因：西雅图地盘与资源见顶，或任务强制外派。  
事件链：「借调洛杉矶」等章把舞台拉到南加州 → 跨城＝任务 + 人情 + 帮派走廊 + 王庭耳目三重压力 → 洛杉矶流浪汉生态、拉丁裔/其他族裔帮派、娱乐工业残骸（与前童星人设对照）提供新魔域标签与拾取源。  
阶段结局：二阶高位任务密集；任何「内战」级动员仍属更高阶情报。  
可介入：当跨城向导、护送「借调」物资、用前童星档案勒索或保护韦恩、在南北帮派间卖路线。

**⑦ 教父、德鲁伊、空间背包 · 体系横向扩张**  
起因：纯枪械拾取已不够应对组织战。  
事件链：公开章名「树上的空间背包」「教父最爱的孩子」「严厉的父亲」「德鲁伊之力」「花月杀手」「韦恩遇袭」→ 体系扩展到自然系/德鲁伊向、空间储物、类黑手党家族结构、刺杀与反刺杀 → 韦恩从帐篷病号变成可被遇袭的有价值目标。  
阶段结局：仍以系统与组织战为主，未到公开击败金色魔王；**连载中，终局不写死**。  
可介入：争空间背包控制权、嵌入「教父」家族秩序、拦截花月杀手、在遇袭现场救人换最高人情。

**⑧ 主线远景（仅方向，非结局）**  
系统主线始终悬挂：踏碎魔巢、击败**金色魔王**、毁灭魔域地下城。写实对应＝动摇黑暗王庭所映射的统治结构，把底层「斩杀线」从个人命运变成集体起义式的长期战争。本库一、二阶只负责**点燃与护送火种**，不负责终局斩王。契约者若在低阶宣称「本周杀魔王」，应被叙事与危险度直接惩罚。

**【世界剧情线·阶段细目与可介入账本】**

**细目A · 斩杀线经济学**  
一剂抗生素、一罐热汤、一床干被、一顶不漏的帐篷，在一阶的价值高于「经验 +30」。任何任务奖励若只给虚名不给热量，玩家会在叙事上死亡。AI 写一阶务必先写：**钱从哪来、药从哪来、今晚睡哪**。

**细目B · 尸体产业链双面**  
收尸是合法/灰色商业：报价、冷链、完整度、身份抹除。系统鼓励搜刮，现实鼓励倒卖。契约者可同时做「系统拾取」与「现实卖尸」，但一旦被拍下视频或被帮派抽成，就从猎人变猎物。第 2 章「收尸人」与后续「收尸人的求救」类章说明这条线会贯穿，不是一次性教程。

**细目C · 映射词典（写正文时保持一致）**  
流浪汉/拾荒者≈哥布林；成瘾崩溃者≈食尸鬼或堕落者；中东/战区归来的绝望退伍兵≈退役恶魔骑士；街头边缘超凡感人物≈黑暗精灵；重型战斗记忆流浪者≈黑暗龙骑；组织统治层≈黑暗王庭；顶点≈金色魔王。映射是系统 UI，不是种族洗白：打赢「哥布林」仍可能是打死一个饿疯的人——本作要保留这层不适与选择。

**细目D · 抱团规则**  
韦恩系早期规矩通常包括：分食、报信、不卖同伴尸体给仇家、重伤优先药。破坏规矩者即使战力高也会被孤立。契约者想空降当老大，必须先证明能带来药与枪，而不是只带来「我很强」。

**细目E · 通缉的双重记账**  
现实通缉/帮派悬赏与系统通缉可能不同步。系统加威望时，现实可能只是「多了一个难缠的流浪汉」；反之，现实上了新闻，系统才补成就。二阶任务常卡在这缝里：你要系统奖励，就得承担现实曝光。

**细目F · 洛杉矶不是升级副本**  
跨城会重置人际关系与药线，旧西雅图口碑只能部分兑现。把洛杉矶写成「经验更多的西雅图」是 OOC；应写成**规则相似、庄家不同**的新桌。

**细目G · 德鲁伊/灵媒/炼器的位置**  
中后期超凡手段是枪械路线的补充，不是替代。德鲁伊之力解决不了子弹账单与帮派包围；炼器与灵媒提供情报与道具，仍怕热武器与斩杀线。低阶发「成体系自然神通」属于越级发奖。

## 阶位切入点

> 阶位↔原作：一阶≈斩杀线边缘流浪汉/无专精拾荒与巷斗；二阶≈枪械专精持有者/稳定小队清巷与小帮派边缘头目。世界顶点（金色魔王·黑暗王庭核心·内战级动员）为**超阶巅峰：存在·情报优先/条件性胜利**，低阶切入一律规避顶点，严禁「封镇/削弱所以可硬刚」。

**一阶（斩杀线流浪汉 · 西雅图冰雨求生线）**

切入身份/时点：契约者以**刚坠入底层的新流浪汉 / 与韦恩同巷的拾荒者 / 收尸线跑腿**身份切入，锚定第一章「西雅图冰雨夜」——系统装载前后、老约翰断气前后、两名「肮脏的哥布林」逼近帐篷之时。

初始事件：冰雨灌进领口，你闻到邻帐篷药味与腐甜；一个临死老兵把血污 USB 塞进你或韦恩手里，系统同时在视网膜弹出「魔域地下城已确认」。两名要收尸卖钱的拾荒者挡住去路——你必须在**抢尸、让尸、联手赶走、报警却无人来**之间当场抉择，且血量/体温正在掉。

开场白建议：「西雅图的雨不是雨，是冰渣子。帐篷布漏成筛，你和一条瘦狗共有同一口热气。邻铺的白人老兵刚把一块粘血的 USB 塞进你掌心，系统字就在暗里亮起来——斩杀线，每小时扣一点命。巷口两个黑影已经在估你们两具身体的行情。」

关键NPC立场：**韦恩**——同期斩杀线上的变数，可结盟抢药也可抢先拾取关键尸体技能，防心重但认「能一起活」的人；**老约翰**（若仍剩最后一口气）——只求 USB 到女儿手，谁承诺谁就是临时神；**荒原猎犬**——认喂食与抱团者，吠警可比人可靠；**两名哥布林式拾荒者**——纯利益，可吓退可收买，难感化；**收尸渠道对接人**——只认完整尸体与报价，不问正义。

主线钩子/支线：主线钩＝护送/争抢老约翰 USB 交付链的第一环，并在系统第一次挑战中活下来；支线＝卖尸换第一笔抗生素钱、在游客区边缘行乞/表演（吟游表演初级）、探教堂地下墓场标签下的真实避雨点、打听「他们在找谁」。

危险度/规避：中（环境与饥饿高于强敌）——须规避持枪帮派清场、警察扫荡收容式「消失」、任何黑暗王庭具名鹰犬；金色魔王名讳仅作系统任务栏远景，禁止主动猎王。

任务方向/奖励：把血量抬离斩杀线、拿到第一件可复用武器技能、与韦恩或底层一人建立互助；奖励取向＝战术冷兵器专精、低级治疗药水/真药、战宠口粮、USB 任务分成、系统初级金币。

**二阶（枪械专精小队 · 王庭耳目与跨巷火并线）**

切入身份/时点：契约者以**已获一门枪械专精的小队核心 / 西雅图医神圈子外围打手 / 王庭反抗者联络员**身份切入，锚定「王庭反抗者」之后、通缉与「唯一超凡」前后，或「借调洛杉矶」前夜——巷战已能打赢，但组织战刚开始。

开场白建议：「M4 的抛壳在雨幕里烫手。你已经不是那个为了一罐汤发抖的人，可系统仍把对面标成『王庭的狗』。耳机里反抗者在骂你叛徒，巷尾韦恩的人在等这箱抗生素——你扣下保险的时候，威望数字跳了一下，像有人在黑暗里给你记账。」

关键NPC立场：**韦恩**——已从病号变为抱团核心与「唯一超凡」候选人/持有者，招揽有规矩的枪，讨厌无脑屠城；**黑暗精灵线同伴（正文真名）**——感念挽救，可做巷内眼线，怕被再当怪物卖掉；**退役黑暗龙骑线人物（正文真名）**——战力硬、脾气倔，引导「东方」的条件可谈可炸；**王庭反抗者头目（章内真名）**——要刀枪不要圣人，可盟可在赏金面前卖契约者；**艾丽**（若支线已推进）——民用身份，怕「他们」，可能是情报锁也可能是人质雷。

主线钩子/支线：主线钩＝在黑暗王庭通缉网下保住小队与一条运药/运枪走廊，并决定是否押注韦恩的超凡路线；支线＝完成或截胡黑暗龙骑「东行引导」、竞标收尸公司高价单挖黑幕、为洛杉矶借调做预卧底、查「花月杀手」类刺杀预告是否对准韦恩。
