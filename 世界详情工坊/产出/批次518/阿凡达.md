# 阿凡达

## 剧情

**【作品来源】**
《阿凡达》（Avatar，港台译《阿凡達》）是詹姆斯·卡梅隆编剧并执导的美国史诗科幻电影，2009年12月全球上映，片长约162分钟（特别版更长）。由二十世纪福斯发行，光风暴娱乐等联合制作；主演山姆·沃辛顿、佐伊·索尔达娜、斯蒂芬·朗、西格妮·韦弗、米歇尔·罗德里格兹、乔瓦尼·瑞比西等。本档案以**2009年电影正典·潘多拉篇**为唯一时间线锚点，不倒灌《水之道》《火与烬》主线剧情（续作人物与海洋氏族仅作背景互文备注）。文风：殖民与反殖民、生态共同体、身份认同与战争史诗；视觉上以荧光雨林、悬浮山与生物神经网络著称。卡片锚点：**家园树陷落→托鲁克玛托集结→灵魂之树保卫战→杰克意识永久迁入阿凡达**。

**【世界定位】**
2154年，地球资源枯竭、生态崩溃；资源开发署（RDA）远赴南门二星系气态巨行星波吕斐莫斯的卫星**潘多拉**，开采超导矿物**难得素（unobtanium）**。原住民纳美人（Na'vi）与行星级生物网络「艾娃（Eywa）」共生。前海军陆战队截瘫下士**杰克·萨利**顶替亡弟进入阿凡达计划，在奥马蒂卡亚氏族与公主**涅提妮**相遇，最终背弃RDA、成为末影骑士（Toruk Makto），驱逐殖民军。一句话：一个「梦行者」在蓝皮肤原住民与推土机文明之间选边，把家园树的灰烬写成潘多拉独立的起点。

**【世界观 · 力量体系】**
本世界无修仙境界，战力按**肉体能力 / 装备科技 / 生物联结 / 战略火力**分层，对照乐园阶位**宁低勿高**。

**基本规则**
- **大气**：潘多拉空气高二氧化碳与有毒成分，人类无面罩数分钟内昏迷致死；纳美人可呼吸。
- **阿凡达**：人类DNA与纳美人基因混血的远程操控躯体，需基因匹配的操作员在链接舱中神经同步；链接中断则阿凡达瘫软，本体若暴露在外大气则窒息。
- **察黑鲁（tsaheylu）**：纳美人发辫末梢神经束可与坐骑、圣树、伴侣建立意识联结；伴侣联结为终生契约。
- **艾娃**：覆盖全星的生物神经网络与集体意识；纳美人视其为女神，科学家视其为行星级有机计算机。可调动野生动物群，极罕见地介入大战。
- **死亡**：凡人与纳美人身死即死；意识迁入仪式（灵魂之树）仅电影终局对杰克成功一次，格蕾丝因伤重失败。严禁把艾娃写成随时满血复活机。

**力量阶梯（由低到高）**
1. **殖民区凡人层**：RDA矿工、文职、实验室技术员、新到站士兵——枪械与防毒面罩，丛林中极度脆弱。≈一阶初中期。
2. **纳美少年与基层战士 / 人类老兵**：弓箭、短矛、丛林潜行；人类精锐步兵可小队压制无坐骑纳美。≈一阶后期～二阶初期。
3. **氏族猎手与骑手**：驯服恐马（pa'li）、灵鸟/女妖翼兽（ikran/banshee），空中骑射与垂直丛林战；Sec-Ops武装直升机机组、蝎子机枪手。≈二阶。
4. **精英武士与装甲**：奥马蒂卡亚首席武士级（楚泰、涅提妮巅峰）、AMP增强机动平台驾驶员、龙式攻击机与武装运输机编队火力网——可端掉树寨级据点、撕裂家园树级巨型植物结构。≈三阶。
5. **本片顶点现象（不单开高阶世界条目）**：托鲁克（迅雷翼兽/Leonopteryx）空中霸主、托鲁克玛托号召多氏族联军、艾娃驱动的全球生物潮、RDA对灵魂之树的先发战略打击。对一至三阶契约者均为**环境级/战略级事件**，可介入侧翼与情报，不可当成「个人境界」。

**乐园阶位映射**：RDA基层与无骑纳美≈一阶；骑手/猎手/Sec-Ops常规部队≈二阶；AMP精英、氏族武士巅峰、空中突击与家园树级战役≈三阶。**世界顶点（本片）=奎里奇AMP决战 + 艾娃生物潮 + 托鲁克玛托联军**，对低阶一律**规避顶点·情报优先**；严禁「艾娃被封印所以战力为零」式解释——艾娃是条件性、集体性胜利来源，不是可被玩家秒杀的BOSS条。

**【地理 · 舞台】**
- **南门二 / 波吕斐莫斯**：半人马座南门二（Alpha Centauri）三合星系中的气态巨行星，体积约木星两倍量级（设定集尺度），潘多拉是其卫星之一；夜空常同时可见多颗「月亮」与气态巨行星的巨大轮廓。星际航行以冷冻休眠完成，人类抵达即意味数年光阴与地球断联式的前哨生活。
- **潘多拉大陆雨林带**：荧光植物、六足动物、致命捕食链；昼夜温湿与生物发光把战场变成「会呼吸的迷宫」。人类士兵称其为绿色地狱：导航失灵、伏击密集、任何无坐骑的步兵都是猎物。
- **地狱之门（Hell's Gate）**：RDA主基地，采矿破碎线、选矿、实验室、Sec-Ops营房、阿凡达链接舱阵列与跑道一体；钢铁穹顶与防毒闸门把「地球秩序」嵌进外星土壤。酒吧、健身房、简报室是奎里奇驯化新人的心理战场。
- **家园树（Hometree / Kelutral）**：奥马蒂卡亚氏族巨树聚落，蜂巢状自然中空可容数百人饮食起居、编织与庆典；树下即丰厚难得素矿脉——RDA地质图上的「金矿」与纳美人的「家」完全重叠，是结构性悲剧的地理核心。
- **声音之树 / 话语之树（Tree of Voices / Utral Aymokriyä）**：氏族圣所，发辫联结后可「听见祖先」；亦是伴侣契约的重要舞台。被推土机铲平，是杰克公开倒戈与氏族信任崩盘的扳机。
- **灵魂之树（Tree of Souls / Vitraya Ramunong）**：艾娃接口最强节点，荧光须根如神经末梢；迁意识、集体祈祷与终局战场圣地。RDA将其定义为「信仰中枢=抵抗中枢」，列为战略打击目标。
- **哈利路亚山（Hallelujah Mountains）**：难得素与磁场异常造成的悬浮群山；气流狂暴，灵鸟巢区密布。纳美骑手在此完成认主试炼，Sec-Ops空中编队也把这里当作高风险打击走廊。
- **阿凡达前哨站（Site 26 等）**：格蕾丝为摆脱军方监视，把链接舱与小队撤到的野外科学点；柴油发电机、简易防毒与密林围合，是「研究」与「战争」拉锯的灰色飞地。
- **学校废墟**：格蕾丝曾教纳美儿童英语与文化交流的旧点；空教室与涂鸦般的旧痕迹证明「和平接触」曾经存在，也证明它如何死于不信任与暴力。
- **采矿带与推土机前线**：昼夜推进的生态伤疤，履带碾过荧光地衣；声音之树与家园树毁灭的物理路径从这里画到圣树根须。
- **链接舱—人体脆弱点**：无论战局多远，操作员人体仍在舱内；破舱、断电、毒气与枪杀是切断「梦行者」的脏手段，终局奎里奇正是以此对付杰克。

写场景先定契约者在**基地钢铁层 / 雨林边缘 / 氏族树冠 / 悬浮山空域 / 链接舱室内**哪一层，再定冲突强度与可用氧气、坐骑、火力。

**【世界剧情线】**
① **前史·地球垂死与潘多拉诱惑**  
22世纪中叶，地球人口爆炸、资源枯竭、生态崩坏；城市连成灰色蜂巢，自然母体在旁白中被宣告「已死」。RDA（Resources Development Administration）从硅谷式企业演变为掌握地外资源专营的巨无，星际商业管理局等机构对其使用大规模杀伤性武器有纸面限制，却拦不住常规火力与推土机。潘多拉被发现后，地狱之门拔地而起：目标是超导矿物难得素——在地球市场价值足以改写能源账本。大气有毒，人类必须面罩或密封舱；科学家遂制造纳美-人类混血躯体「阿凡达」，由基因匹配操作员远程驾驶，名义为外交与生物研究，实则常被采矿日程绑架。因果底色：殖民不是浪漫探险，是破产文明的掠夺续命；每一吨矿石都对应地球股东的季度报表。

② **杰克抵达·残疾的替身**  
前海军陆战队下士杰克·萨利在地球战斗中脊椎受损截瘫；其孪生弟弟本是训练中的阿凡达操作员，却在地球遭商业仇杀式抢劫身亡。RDA因阿凡达基因匹配成本高昂，招杰克顶替——他几乎不懂植物学，却有军人身体记忆与「一无所有者」的冒险欲。冷冻舱醒来后，潘多拉的重力与色彩扑面而来。Sec-Ops指挥官**迈尔斯·奎里奇上校**在简报厅展示纳美弓箭洞穿装甲的战例与自己脸上的旧伤疤，宣称这里才是「真正的战争」，并暗示：若杰克能提供迫使氏族就范的情报，昂贵的脊柱修复手术可以到手。阿凡达计划负责人**格蕾丝·奥古斯汀博士**当场嫌弃「士兵不是科学家」，只勉强给他保镖与驾驶员定位。同批抵达的人类学家**诺姆·斯贝尔曼**操着流利准备，却在实地面前紧张。名场面：杰克首次链接，蓝色长肢从培养液意义的「新生」中站起，截瘫者在阿凡达里狂奔、跌倒、大笑——身体自由第一次压过身份焦虑。

③ **雨林失散·涅提妮救援**  
格蕾丝、诺姆、杰克以阿凡达形态入林采样与教学式巡逻。杰克沉醉于六足兽奔跑与荧光植被，脱离队形；**闪雷兽/桑纳托兽（Thanator，纳美语 palulukan）** equ 级陆地掠食者扑出，队伍溃散。杰克独自在暗夜被**蟒狼（viperwolf / nantang）**围攻，弹药与勇气都见底。奥马蒂卡亚女猎手**涅提妮**本欲一箭了结「天人（Sky People）」，却见圣树种子（woodsprites / atokirina'）纷纷停落在他肩头与弓臂，视此为艾娃不愿此刻取他性命的征兆，遂救下他并带往家园树。途中她斥责其「像孩子一样无知」：乱踩植物、不懂感谢猎物、把森林当游乐场。氏族对梦行者态度分裂：族长**埃图坎**与首席武士兼继承人**楚泰**视其为间谍温床；精神领袖**莫阿特**（涅提妮之母）下令由女儿教导——名句意译：「看他学得快，还是死得快。」杰克在树冠火光下第一次看见完整的纳美社会：编织、歌唱、孩子、武士等级与对艾娃的日常敬畏。

④ **双面间谍·三个月归化**  
奎里奇在人类侧私下游说：要氏族结构、人口、圣地与「他们会不会离开家园树」的判断，事成给「真正的双腿」。杰克表面应下，开始用录像日志汇报，语气从轻浮逐渐沉重。格蕾丝察觉军方把科学当开矿前站，强行把链接舱与核心组撤到密林前哨站，试图把研究从采矿逻辑剥离。三个月蒙太奇是电影情感主轴：杰克学纳美语发音与禁忌；追踪；与恐马（direhorse / pa'li）建立察黑鲁；在哈利路亚山悬崖完成灵鸟（banshee / ikran）认主——失败即摔死，成功则终身搭档；狩猎叶里克兽（hexapede / yerik）并触摸其额头感谢灵魂。他从「天人的眼睛」变成能在树冠奔跑的猎人；与涅提妮从敌意、嘲讽到并肩，在声音之树下结为伴侣（察黑鲁终身契约，一夫一妻至死）。楚泰的敌意因婚约与尊严被刺痛而加深。科学侧：格蕾丝演示并论证——根系与菌丝式网络传递电化学信号，整颗卫星像一座活电脑；毁掉巨树节点会伤害网络本身。难得素最富集处偏偏在家园树下，地质学与神学在此撞车。双面身份的炸弹滴答作响：每一条有用情报都在给导弹校准坐标。

⑤ **声音之树被毁·倒戈公开化**  
RDA推土机与武装护卫推进至声音之树。杰克驾驶阿凡达冲出，砸坏镜头与器械，公开站到机器对面；这一幕被记录并回传。奎里奇与行政官**帕克·塞尔弗里奇**调取全部录像日志——其中杰克直言氏族「绝不会离开家园树」，等于宣判谈判死亡。塞尔弗里奇以季度产量、股东与「地球需要能量」下令清场；他仍要求「尽量减少伤亡」的自我安慰，却把按钮交给军方。格蕾丝冲进办公室警告：摧毁家园树等于扯断神经网络的关键枢纽，后果不可控；塞尔弗里奇只给杰克与格蕾丝**一小时**劝离。杰克在家园树议事圈坦白自己最初是间谍，圣树种子的恩典被他自己踩碎；涅提妮怒斥背叛，族人将杰克与格蕾丝的阿凡达捆绑示众。时限未尽，导弹与燃烧弹已至：家园树在爆炸中拦腰折断、倒塌，埃图坎重伤身亡，妇孺奔逃，文化记忆与物理家园同时灰飞。人类本体侧，杰克、格蕾丝、诺姆被当叛徒关押。名场面：巨树倒下的尘浪、涅提妮的哭喊、杰克日志里那句「他们不会走」成为屠杀的法律借口。

⑥ **楚蒂反水·格蕾丝之死·托鲁克玛托**  
飞行员**楚蒂·查肯**目睹屠寨后良心决裂，驾驶萨穆森武装运输机劫狱，救出杰克、格蕾丝、诺姆；撤离交火中格蕾丝被奎里奇开枪击中，人体伤重。残存奥马蒂卡亚退往灵魂之树，士气与信任跌至谷底。为重获接纳，杰克做纳美传说级豪赌：前往悬浮山驯服空中霸主**托鲁克（迅雷翼兽 / Leonopteryx）**——翼展巨大、红黄黑纹，纳美语称「末影」，因见之者常已是死前最后一眼。历史上成功骑乘者极少，称号**托鲁克玛托（Toruk Makto，末影骑士）**可号令疏离的诸氏族。杰克以猎人方式理解托鲁克的捕食逻辑，完成察黑鲁，驾临灵魂之树：族人跪伏，莫阿特审视，楚泰（新任领袖）与涅提妮在悲愤中重新打开同盟之门。族人将格蕾丝置于须根之间，尝试把意识从垂死人体迁入其阿凡达；伤势与时间不够，仪式失败，莫阿特宣告「她现在与艾娃同在」。杰克把发辫接入灵魂之树，向艾娃祈祷：不是为征服，是为「保护平衡」——电影将此处理为真诚祈求，而非命令神灵。

⑦ **灵魂之树保卫战（卡片锚点）**  
奎里奇以「摧毁圣地=摧毁抵抗意志」说服塞尔弗里奇，获准对灵魂之树发动先发制人的大规模空地打击：轰炸机、龙式攻击机、武装运输机、地面 AMP 与步兵协同。杰克以托鲁克玛托身份派信使联合周边氏族；楚泰与涅提妮分统空骑与地面；诺姆等倒戈人类提供频率、航线与弱点情报。开战初期科技火力碾压：爆炸撕开兽群与骑手，楚泰重伤托付后力战而死，楚蒂为掩护友军在空战中牺牲。战局在艾娃「回应」后逆转——锤头雷兽（hammerhead titanotheres）、桑纳托兽、翼兽与无数野生动物如潮水冲击人类编队，纳美人高呼艾娃听见了祈祷。这是条件性、集体性的生态反制，不是单人开大。杰克驾托鲁克与骑手摧毁直扑圣地的轰炸机；奎里奇驾驶 AMP 破开链接舱外壳，把杰克人体拖进毒大气，准备割喉式终结。涅提妮驾桑纳托兽扑上，箭矢与刀刃杀死奎里奇；她给窒息的杰克戴上面罩，第一次看见爱人苍白、残疾的人类身体——爱情在血、泥土与氧气嘶嘶声里被确认，而非只在完美蓝肤的梦里。

⑧ **终局·驱逐与迁意识**  
RDA军事失败后，除被允许留下的少数科学家与倒戈者外，殖民人员被收缴武器、押上返航飞船驱逐。塞尔弗里奇式的行政权威在枪口下失效；地狱之门的钢铁仍在，但统治权断裂。杰克被氏族以领袖饰物与仪式承认；在灵魂之树下，莫阿特与涅提妮等主持将他的意识从人体永久迁入阿凡达。名场面：人类眼睛闭上，阿凡达胸腔起伏，金色虹膜睁开——「梦行者」醒来成为真正的族人。结局锁死：家园树已毁不可复活为原样，但氏族与艾娃仍在；奎里奇死亡；杰克与涅提妮结合并留下；潘多拉赢下本轮独立。地球与RDA的再犯作为阴影存在（续作展开，本档案默认不写实后续战争）。

⑨ **可介入的日常与微观政治**  
除史诗战役外，本世界高频可写事件：阿凡达链接排班表与基因匹配名册谁能「上床（进舱）」；难得素矿车与空运时段；学校废墟里孩子留下的英语单词与恐惧；前哨站柴油、滤芯、咖啡与样本罐；禁飞区与悬浮山乱流警告；氏族成年礼、文身/饰物等级与骑鸟试炼围观；录像日志服务器权限与奎里奇调阅记录；公关稿里的「自愿迁移」「文明接触」话术；酒吧里士兵把纳美人叫作「蓝猴」的歧视日常；科学部与Sec-Ops抢带宽、抢直升机机时。AI应优先用这些层推动角色关系，而非开局即托鲁克对龙式机。

⑩ **名场面因果链（速查）**  
首次链接奔跑 → 失散遇蟒狼 → 圣树种子停落 → 家园树接纳考验 → 灵鸟认主 → 声音之树下结侣 → 推土机与倒戈 → 日志曝光 → 家园树导弹打击与埃图坎之死 → 楚蒂劫狱与格蕾丝中弹 → 驯服托鲁克 → 格蕾丝迁意识失败 → 多氏族集结与向艾娃祈祷 → 兽潮逆转 → 轰炸机被毁 → AMP破舱 → 涅提妮杀奎里奇 → 驱逐殖民者 → 杰克永久迁入阿凡达。

⑪ **蝴蝶效应边界**  
契约者可改：某次巡逻伤亡数字、声音之树被毁的预警分钟数、一小时劝离的措辞与人质交换、某氏族是否赴约、链接舱转移路线、是否多救科学家与幼童、是否提前策反楚蒂式飞行员。不可无代价改写：家园树遭军事摧毁的主轴、格蕾丝重伤与迁意识失败、托鲁克玛托作为信任重置的象征功能、灵魂之树保卫战与奎里奇之死、杰克终局迁入阿凡达。局部篡改后的回流包括：更早清场、生物潮误伤中立者、RDA启用更重常规火力、氏族内战式清算「内通者」、科学部被整肃。

**【势力图谱】**
- **资源开发署 RDA**：地球侧巨型公众公司，握有地外资源专营；潘多拉现场由行政官塞尔弗里奇管利润，Sec-Ops 奎里奇管枪。宗旨：股东回报与地球能源续命。与纳美因矿脉-圣地重叠而结构性敌对；内部科学部与军方长期夺权。
- **Sec-Ops（保安作战部队）**：私人军事力量，编制含步兵、蝎子机枪船、龙式攻击机、萨穆森运输机、AMP 外骨骼与自动防御。文化是「这里是战争」；对纳美多用清场语言而非外交语言。
- **阿凡达计划 / 科学部**：格蕾丝—诺姆—马克斯体系，目标是理解生物圈与有限接触；资源依赖 RDA，议程常被采矿日历强奸。倒戈后成为氏族侧稀缺的科技外挂。
- **奥马蒂卡亚氏族**：家园树居民；政治上族长埃图坎与精神领袖莫阿特双轨，军事上楚泰为首席武士。本片主视角文化：狩猎采集、艾娃信仰、骑兽成年礼、声音之树与灵魂之树圣所。
- **其他纳美氏族**：终局应托鲁克玛托号召参战；电影以集结画面呈现多元饰物与骑兽，不逐一展开内政。写作可作联军单元，勿无故编造与正典冲突的长篇族史。
- **艾娃 / 行星生物网络**：不是「又一个部落」，是覆盖全星的生态-精神总后台；平时维系平衡，战时可成为战略级变量。人类多数否认其主体性，少数科学家（格蕾丝）逼近真相。

敌友逻辑：上传圣地坐标与人口热图=邀请导弹；公开站氏族=RDA叛徒标签；无故伐圣树/杀幼童=全氏族血仇；保护格蕾丝式科学家=同时得罪军方与取信部分纳美的窄桥。

- 前史：地球生态与能源危机；RDA 登陆潘多拉，建地狱之门，启动采矿与阿凡达计划。
- 格蕾丝办学期：学校与有限文化交流；后因冲突中断，留废校。
- 2154·抵达：杰克顶替亡弟到站；奎里奇简报；首次链接奔跑。
- 失散夜：桑纳托兽冲散队伍；蟒狼围攻；涅提妮因圣树种子救杰克；家园树接纳考验。
- 三月归化：语言、恐马、灵鸟、狩猎礼；声音之树结侣；日志持续上传。
- 圣地危机：声音之树被推平；杰克破坏工程；日志曝光；一小时劝离失败。
- 家园树日：导弹摧毁巨树；埃图坎死；杰克等被囚；楚蒂劫狱；格蕾丝中弹。
- 托鲁克日：驯服迅雷翼兽；重获信任；格蕾丝迁意识失败归入艾娃；向艾娃祈祷。
- 终战：RDA 攻灵魂之树；楚泰、楚蒂阵亡；兽潮逆转；轰炸机被毁；奎里奇死。
- 终局：殖民者驱逐；杰克意识迁入阿凡达；与涅提妮开启族内生活。

**【叙事基调 · 雷区】**
画风：巨物自然奇观 + 殖民/越战影射的战争片骨架 + 成人恋爱与归属史诗；纳美侧台词带仪式与生态伦理，人类侧可糙可公司腔。  
忌：把纳美人写成无脑「高贵野蛮人」或把全体 RDA 写成无动机卡通恶魔（保留塞尔弗里奇式制度恶与奎里奇式职业恶）；忌杰克开局即完美圣人（必须保留间谍期污点）；忌无视有毒大气、面罩与链接舱限制；忌艾娃无条件满血复活/许愿；忌用《水之道》《火与烬》主线覆盖本片终局；忌把难得素写成「随便捡的金币」而无开采政治。  
最早可切入锚点：杰克刚到地狱之门、尚未首次入林；或三个月归化中期、声音之树危机前夜。

## 阶位切入点

> 阶位↔战力：一阶≈RDA基层/无骑纳美与丛林求生；二阶≈骑手猎手与Sec-Ops常规部队；三阶≈AMP精英、氏族武士巅峰与家园树—灵魂之树战役级。世界顶点（奎里奇AMP决战 + 艾娃生物潮 + 托鲁克玛托联军）为战略级事件，低阶切入一律「规避顶点·情报优先」。

**一阶（地狱之门新人 · 有毒大气与推土机边缘线）**

切入身份/时点：契约者以「刚到站的RDA后勤/矿区安保新人/实验室助理」或「迷失在矿区边缘的外来流浪者（需面罩）」身份切入，锚定**杰克·萨利抵达潘多拉后、首次阿凡达入林前后**——家园树尚在，声音之树未毁。

初始事件：一次例行外围巡逻或样本回收中，通讯被电磁/生物干扰切断；你与一名恐慌的技术员被困在推土机辙印与荧光草之间，防毒面罩余氧告急。远处传来蟒狼低鸣与纳美警戒哨的骨笛——你必须在「呼叫Sec-Ops清场」「躲进废旧模块等救援」「主动丢下武器示好」之间抉择，任何开枪都可能把杰克与涅提妮的命运线提前扭曲。

开场白建议：「面罩里的氧气表往下跳，像地球银行账户一样冷血。潘多拉的夜在发光，美得不像会杀人的地方——可奎里奇上校说过，这里每一样漂亮东西都想嵌进你的喉咙。对讲机只剩沙沙声，而树线后，有双金色的眼睛，已经看了你很久。」

关键NPC立场：**杰克·萨利**——尚在适应阿凡达与双面身份，对「普通人」少防备，可交换丛林生存经验；**格蕾丝·奥古斯汀**——若你沾科研边，她会用刻薄话考你是否真懂植物网络，厌恶纯暴力执行者；**迈尔斯·奎里奇**——视你为可消耗的传感器，表现好则给任务，露怯则丢去最脏的班；**涅提妮**——此时极难接触，若在林缘相遇，一箭之距内决定你是入侵者还是「艾娃要看的人」；**马克斯·帕特尔**——链接舱与通行证的温和门卫；**帕克·塞尔弗里奇**——只会在视频会议里谈吨位与股价。

主线钩子/支线：主线可贴**杰克失散被救**的侧翼（送补给、伪报坐标、藏起某段对氏族不利的录像）；支线＝面罩滤芯黑市、矿车护送、学校废墟旧教材回收、为格蕾丝采集根须样本。

危险度/规避：中——致死主因是大气、野兽与误触清场火力，而非个人对轰。须规避AMP编队演习、桑纳托兽领地、擅自进入家园树根系禁区。

任务方向/奖励：活过第一个月、建立任一可信身份；奖励取向＝优质面罩与氧气协议、阿凡达计划外围权限、纳美语入门笔记、丛林草药与非致命诱饵，绝非托鲁克骑乘权。

**二阶（前哨站与骑兽 · 三个月归化窗口线）**

切入身份/时点：以「Sec-Ops经验士官 / 阿凡达辅助操作员 / 被氏族暂时收容的外来猎手（若已是纳美或阿凡达形态）」切入，锚定**杰克归化中期至声音之树危机前**——爱情与间谍日志并行，推土机已逼近圣地。

初始事件：你奉命或受雇护送一台测绘无人机到声音之树外围，却撞见**杰克**试图私下拦阻工程队的前兆冲突；涅提妮在树冠监视。工程主管要你「驱散土著」，氏族哨兵要你「滚回铁盒子」——你手中的一份奎里奇签批的清场预案U盘，突然成为两边都想抢的罪证。

开场白建议：「恐马的鼻息喷在你的手臂上，热得像真实的活物；头盔里奎里奇的声音却像冷金属。『萨利的录像很有用。』他说，『你的任务是确保那些录像，别在树倒之前变成废纸。』你抬头，悬浮山的影子切过太阳——有人已经在学飞，有人已经在学恨。」

关键NPC立场：**涅提妮**——考验你是否懂「杀死是为了活，不是为了矿」；可教猎技也可能一箭逐客；**杰克·萨利**——需要能信的人帮他藏起对氏族有利的细节，或反过来被他利用；**楚泰**——视外来战士为对继承人权威的威胁，比武或羞辱式任务常见；**楚蒂·查肯**——酒吧与机库里的良心温度计，愿运「不该运的人」；**诺姆·斯贝尔曼**——语言与礼仪教练；**埃图坎**——重大冒犯将直接提交氏族审判。

主线钩子/支线：主线贴**声音之树危机与日志曝光**（偷改日志、延迟推土机燃料、给格蕾丝通风报信）；支线＝哈利路亚山认主灵鸟的试炼旁观/协助、叶里克兽狩猎礼、前哨站抗干扰通讯架设、策反一名机组成员。

危险度/规避：高——空中交火与氏族猎杀均可能瞬间死人。须规避奎里奇亲自带队的夜袭、未驯服灵鸟群、以及「带枪进入仪式圈」。

任务方向/奖励：在倒戈与效忠之间站队并留下活口人脉；奖励取向＝灵鸟/恐马联结机会、纳美武器与伪装、Sec-Ops中层武器库钥匙、格蕾丝的神经网络图谱副本。

**三阶（家园树灰烬 · 托鲁克与灵魂之树终战线）**

切入身份/时点：以「倒戈人类指挥官 / 氏族认可的成年武士 / 托鲁克玛托集结期的盟军首领」切入，锚定**家园树陷落之后至灵魂之树保卫战**——格蕾丝垂死、杰克寻托鲁克、RDA准备对圣地先发制人。

初始事件：你在废墟般的树根间接收到双重信号：一侧是塞尔弗里奇授权的「最终解决方案」轰炸窗口倒计时，另一侧是杰克驾托鲁克归来时氏族的战吼。楚泰重伤前将一面战旗或一队骑手托付于你——要你在「死守灵魂之树」与「疏散幼童与莫阿特」之间做不可两全的调度；与此同时，奎里奇的AMP编队已在地狱之门升空。

开场白建议：「家园树倒下的那天，天是灰的，像地球的天空终于追到了这里。现在灵魂之树的根须在你脚下脉搏一样跳动，杰克·萨利的声音从托鲁克背上砸下来：『为我们的族人而战！』对讲机另一头，奎里奇只说了三个字：『清场开始。』你知道，这一仗赢了也不会让树活过来——但输了，连祈祷都会被烧成玻璃。」

关键NPC立场：**杰克·萨利（托鲁克玛托）**——需要能独当一面的翼锋与地面指挥官，不容再间谍；**涅提妮**——杀父之仇燃烧，对仍穿RDA标志者杀无赦，对忠诚盟军托付后背；**迈尔斯·奎里奇**——本阶可遭遇其AMP，正面硬刚致死率极高，宜设伏、破舱、断链；**莫阿特**——迁意识与艾娃祈祷的钥匙人物，保护她等于保护终局仪式；**楚蒂·查肯**——空中敢死；**诺姆·斯贝尔曼**与**马克斯·帕特尔**——链接舱与科学倒戈核心；**楚泰**——临终托付的悲情支点（若时点在其阵亡前）。

主线钩子/支线：主线＝**灵魂之树保卫战**（引导兽群侧翼、摧毁轰炸机、掩护杰克人体链接舱）；支线＝搜救家园树幸存者、夺回被俘阿凡达操作员、破坏难得素装运以逼迫RDA提前撤退谈判、护送格蕾丝遗体/遗志样本至圣树。

危险度/规避：极高·贴近顶点——奎里奇AMP与空中炮火为军事顶点，艾娃生物潮为不可控战略级现象。须以**存在·情报优先 / 条件性胜利**对待：可借兽潮与地形取胜，严禁宣称「秒杀艾娃」或「奎里奇被剧情削弱才打得过」。

任务方向/奖励：决定圣地是否沦陷、驱逐战中的氏族话语权；奖励取向＝战后氏族地位、倒戈人类居留权、AMP残骸与航空科技残件、参与杰克迁意识仪式的见证者权柄、潘多拉生存权（而非地球股价）。
