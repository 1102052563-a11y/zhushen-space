# 轩辕剑叁：云和山的彼端
<!--meta lib=主库 tiers=一、二、三、四、五、六-->

## 剧情

**【作品来源】**
《轩辕剑叁：云和山的彼端》（Xuan-Yuan Sword III: Mists Beyond the Mountains / Beyond the Cloud and Mountain）是大宇资讯 DOMO 小组 1999 年 12 月 4 日发行的 Windows 角色扮演游戏，系列 2D 巅峰作之一，也是轩辕剑正传首次在中国大陆正式发行的作品。制作人蔡明宏，编剧鲍弘修，程序黄志荣，美术郭炳宏，音乐吴欣叡；封面由日本漫画家皇名月绘制。故事以约公元 750 年前后（法兰克丕平执政、阿拉伯阿拔斯革命、唐天宝至安史之乱前夜）为轴，主角自威尼斯东行，横跨法兰克、阿拉伯、西域与大唐，串入怛罗斯之战、石国灭国、阿拔斯革命、安史之乱等史事——部分描写与史书有出入，**以游戏剧情为准**。2017 年 iOS 加强版与 2023 年 Steam / Switch 高清重置版补强中国篇与部分对白。主题句贯穿全作：**世上无战争不败之法；王道是仁慈与秩序，是尽量避免战争。**

**【世界定位】**
可被契约者切入的完整任务世界。法兰克混血骑士**赛特**（Septem / Set，可改名；东方人与日耳曼血统混血，异色瞳一眼褐一眼蓝）受执政王**丕平三世**之命，东寻「战争不败之法」。威尼斯救阿拉伯商人时，神秘巫师为其召唤爱哭爱闹、失忆的女魔将**妮可**（Nicole，可改名；前世为《轩辕剑叁外传：天之痕》中的**独孤宁珂**，魔性被伏羲琴净化九十九年后转生）。一行经移动岛、蓝人冰山、大马士革、阿拔斯革命、怛罗斯、天山、长安，终识王道，封印**撒旦**（人间转世为护教团队长**麦尔斯**）。一句话：东行骑士以王道取代不败战法，用文明碰撞回答「为何而战」。

契约者不是天选，是变数。低阶体验信仰暴力与商路风险；中阶卷入革命、古墓与怛罗斯；高阶触及反曼陀罗阵与撒旦觉醒。本地人若听闻低阶要屠撒旦，会当疯子——这是正确反应。

**【世界观 · 力量体系】**
力量本源：骑士武技与奇术、炼妖壶（东 / 西方祭坛炼化妖物）、神魔血脉与魔将底蕴、六芒星阵、反曼陀罗阵、轩辕剑仙之剑化仙力、撒旦魔威与恐惧之王权柄。弱肉强食与「信仰 / 军功 / 学问」三重话语并存——欧洲用异端火刑，阿拉伯用革命与海军，大唐用官职与烽火。

死亡规则：凡人与普通士兵战死即止；神魔可残魂、转世或再赋肉身，皆有代价。赛特曾被撒旦元神杀死，**轩辕剑仙**寻回其魂、以仙力重塑肉身，仅余约三十年寿；妮可终因助封撒旦有功，由神界安排在欧洲转世为人。世界顶点不可无限复活。

特殊系统：
- **炼妖壶**：系列核心至宝。本作分**西方祭坛**（威尼斯修道院获六芒星阵后开启）与**东方祭坛**（天山得李靖所赠封神坛后开启）；同材料异产物，体现文明双轨。可收化敌人、炼化物品与妖物；终局将撒旦封印入壶（至少九十九年内不得出）。
- **神魔异事录**：战斗图鉴，记录妖魔属性与特征。
- **反曼陀罗阵**：赛特前世为天使时与撒旦共同反叛天界所设计，可扰乱因果、涉及时空；撒旦欲借此统治人界。
- **时空潜质**：赛特前世能力残存，移动岛上被石像考验时触发；后被蜃楼王利用造「虚幻五年」。

境界 / 战力阶梯（写战力优先破坏力表现，宁低勿高）：
1. **市民 / 商贩 / 新兵** ≈ 一阶。徒手或冷兵器，火刑与巷战中可致命。
2. **护教团骑士 / 海盗精锐 / 边军老兵** ≈ 一～二阶。单挑小队、破门劫狱。
3. **赛特中前期 / 薇达部曲将校 / 成名冒险者** ≈ 二～三阶。裂石断金级武技，可端据点。
4. **蓝魔神部将 / 古墓机关与中型魔 / 慧彦高僧战力** ≈ 三～四阶。可左右一域战场，抗常规军阵。
5. **恐惧之王康纳里士 / 觉醒前麦尔斯 / 半神魔将妮可（魔将底蕴）** ≈ 四～五阶。升岛、恐惧领域、魔法成体系。
6. **撒旦本体（八臂魔帝）/ 轩辕剑仙 / 反曼陀罗阵级** ≈ 五～六阶。城市至地区级灾难；终局封印需阵眼、炼妖壶与同伴协作，严禁低阶硬刚。

乐园阶位映射：市民～士兵 ≈ 一阶；骑士精锐 / 海军将校 ≈ 二阶；成名冒险者 / 中型魔 / 高僧 ≈ 三阶；恐惧之王 / 名将李靖级 ≈ 四阶；半神魔将 / 觉醒前麦尔斯 ≈ 五阶；撒旦 / 轩辕剑仙 / 反曼陀罗完整启动 ≈ 六阶。**世界顶点 = 撒旦与轩辕剑仙（六阶巅峰）**；低阶切入一律规避顶点、情报优先；条件性胜利（破阵眼、开壶口、借剑仙），严禁战力归零式削弱。

**【地理 · 舞台】**
地图由西向东展开，文明三色（建筑、服饰、音乐、对白）必须随地切换，禁止三地同一套武侠腔。

**欧洲 · 威尼斯与法兰克**：运河、钟楼、修道院、护教团教堂密道、青苔石迷宫与海洞。开篇是信仰暴力与情报采购：丕平要「战争不败之法」，护教团以抓异端为业。气味是咸潮、蜡与烧焦的木架。

**移动岛**：妮可前世府邸，由会说话的石像掌舵。恐惧之王**康纳里士**自海底升起，困威尼斯与护教团舰队。岛上城堡是魔界棋盘，石像只认妮可主权——「去阿拉伯」是政治选择，不是景点。

**蓝人冰山**：蓝人与**蓝魔神**。阿拉伯海军女将**薇达**率舰队追击亦被困。冰封与魔神战提供奇幻战力标尺。

**中东 · 丝绸之路西段**：贝鲁特、大马士革、叙利亚森林、**塔德莫尔（培德莫尔）古墓**、肯迪之家、火炎迷宫。阿拔斯推翻倭马亚的革命巷战、黄道十二宫机关、水火地风四象门。学问链在此：肯迪大师已逝，孙**小肯迪**续教。

**西域 · 怛罗斯与天山**：巴格达、石国故地、**怛罗斯**战场、天山。唐将高仙芝使诈灭石国，石国王子**远恩**组联军（含大食黑衣军）反攻；唐与大食在怛罗斯对撞。**慧彦**横锡杖于沙，以命阻战。蜃楼王幻术与轩辕剑仙指引亦在此段。

**大唐**：长安（王思月家、坊市、宫城）、旧长安废墟（反曼陀罗阵眼）、洛阳、骊山、少林寺（慧彦骨灰归处）。加强版中赛特曾临时任唐军要职，看见粮草与腐败，使「止战」更难也更必要。

**【世界剧情线】**

① **威尼斯：护教团、妮可与东行王命**  
赛特为丕平收集东方战情。威尼斯「法兰克护教团」以抓异端为名滥杀。赛特潜入教堂密道救出阿拉伯商人时，神秘巫师召唤出爱哭爱闹、记忆缺失的妮可；妮可缠其同行。情敌兼好友麦尔斯率护教团阻挠；妮可误伤赛特暗恋对象**莉莲**。赛特从修道院取得航海图与六芒星阵（开启炼妖壶西方祭坛），妮可召唤炎魔神姆斯比尔对抗麦尔斯。众人乘阿拉伯海盗船东航。  
主题：战争的正义话术有多廉价；人道主义行动里塞进魔界棋子，讽刺成立。  
名场面：运河夜救商、教堂密道、莉莲被误伤、六芒星阵入手。

② **移动岛：莉莲之死与石像三问**  
海上遭遇威尼斯执政与护教团海战。康纳里士升起移动岛困双方；护教团入城堡尽死。赛特追入，与麦尔斯合力击败康纳里士，但莉莲在邪恶仪式中身亡。城堡后园石像精灵确认妮可身份，考验赛特（触发其潜在时空穿梭能力），同意岛屿东移。  
主题：两边的死长得一样；失忆魔将的喜剧外壳下是净化悲剧。  
名场面：升岛、城堡尽灭、石像掌舵谈判、时空潜质初醒。

③ **蓝人冰山与薇达**  
蓝人冰山撞击移动岛。赛特入冰山，解救被困的薇达舰队。击败并封印蓝魔神后，移动岛失能。薇达通领先祖劲旅，因女性身份遭上级排斥发配边疆，不守阿拉伯对女性的诸多限制；直率敢爱敢恨。一行乘其船赴大马士革。妮可因吃醋一度离开。  
主题：性别政治与军功；同盟要先放下骑士优越感。  
名场面：冰山恶战、蓝魔神封印、薇达邀请同行。

④ **大马士革：肯迪、革命与卡玛 / 安卡**  
拜访肯迪大师闻其逝世，赛特随小肯迪学习。卷入阿拔斯王朝推翻倭马亚的革命。官员**麦尔丹**企图释放古墓中的「恶魔」镇压革命。赛特与小肯迪进入塔德莫尔古墓，破解黄道十二宫 / 四象机关，发现「恶魔」实为印度爱情精灵**卡玛**及其宠物埃及有翼黑猫**安卡**（名取自生命象征安赫）。卡玛为薇达的孪生灵魂姐妹 / 守护精灵，因执着人间情缘背弃部分誓约；安卡比卡玛年长，冷静吐槽。麦尔丹遭报应。卡玛与安卡加入队伍。革命前夕王宫遇唐使**黄雷**及其机关兽，偶遇大唐高僧慧彦。  
主题：「镇压工具」叙事的谎言；爱情精灵不是炮灰。  
名场面：黄道机关、石棺密室、卡玛安卡现身、黄雷伏击。

⑤ **王道初授与怛罗斯**  
慧彦言：世上无战争不败之法，最好是避免战争——即**王道**（仁慈与秩序）。巴格达闻唐军使诈灭石国，远恩组联军反唐。麦尔斯再现，以黑魔法击伤赛特。怛罗斯战场，黄雷埋伏；大食与唐军激战。慧彦让赛特带远恩上山，自己以一己之力阻止两军开战，最终英勇阵亡。临终嘱赛特将骨灰（舍利子）运回少林，勿弃王道。薇达找回赛特一行。  
主题：王道从口号变成尸体与骨灰托付。  
名场面：锡杖横沙、两军不敢先射、慧彦掩护溃退、骨灰托付。

⑥ **蜃楼王、虚幻五年与天山返时**  
赛特晕倒，蜃楼王利用其时空能力制造虚幻五年：赛特与薇达结婚、成波斯执政官甚至发兵攻唐。慧彦魂魄现身打破幻境，带赛特至天山。赛特在麦尔斯等人诱惑下坚持前行，得轩辕剑仙指引，收伏蜃楼王。剑仙引见慧彦前世——仙人**李靖**（唐开国功臣），李靖加入，带赛特返回五年前，共赴大唐。东方祭坛（封神坛）在此开启。  
主题：幻境里的「胜利」是王道的反面教材；返时是代价而非金手指。  
名场面：虚幻五年、慧彦亡魂破局、剑仙指路、李靖归队。

⑦ **长安：安史前夜、撒旦觉醒与死而再肉身**  
抵达长安，康纳里士与麦尔斯已先至。众人借传教住**王思月**家。加强版中赛特曾临时任唐军要职。闻**安禄山**即将起兵，赶往皇宫，揭麦尔斯即撒旦已完全觉醒。赛特摧毁其肉体，却被撒旦元神杀死。一年后，轩辕剑仙寻回赛特灵魂，以仙力赋予新肉身（仅能再活约三十年）。  
主题：安禄山是历史倒计时，撒旦是神魔倒计时，两者在皇宫夜对齐；胜利税写进寿命。  
名场面：皇宫觉醒、肉身尽毁、元神杀赛特、剑仙赋身。

⑧ **终局：反曼陀罗、封撒旦入壶与王道传播**  
赛特返回长安（汉长安废墟），击败企图用反曼陀罗阵统治人界的撒旦（可先战夺肉身的撒旦，再入洞战八臂本体；隐藏可选蚩尤、牛魔王等），将其封印入炼妖壶。妮可助战有功。赛特领悟王道精髓，告别友人返回欧洲传播。队伍解散：妮可因功转世欧洲为人；卡玛回守护薇达（赛特与薇达结婚，约十年后薇达去世）；李靖留守护唐；安卡与轩辕剑仙继续跟随赛特未完的旅程。多年后赛特找到转世妮可，共度余生。  
主题：不败之法的答案是「少开战」；多段感情是人间税，不是后宫清单。  
名场面：阵眼争夺、八臂撒旦、炼妖壶封印、返欧传王道、转世重逢。

⑨ **微观政治贯穿（日常可介入）**  
威尼斯异端营救与火刑旁观；移动岛石像三问与航向谈判；冰山停战与俘虏交换；大马士革革命夜担架与假情报；古墓「释放 / 囚禁 / 自选」伦理；怛罗斯传讯、救伤兵、假骨灰诱饵；长安坊市撤侨、军职公文、密报安禄山粮草；反曼陀罗阵眼抢修（高阶）。AI 优先这些小事件，而不是一上来顶点大战。

⑩ **蝴蝶效应边界**  
契约者可改局部：救一个阿拉伯商人、让某场巷战易主、护送骨灰时调包、破坏外围阵眼。不可无代价改写：慧彦怛罗斯殉道的重量、赛特死而再肉身的寿命税、撒旦终局被封入壶、王道定义、妮可转世因果。改了局部，原作因果会以追杀、取代、更惨的方式回流。

**【主要人物】**
- **赛特（Septem / Set）**｜身份：法兰克混血骑士，丕平情报官 / 养子系亲信；前世为与撒旦共同反叛天界的天使，设计反曼陀罗阵并获时空能力；官方后设为《天之痕》宇文拓后裔（非转世）。｜性格：寡言、逻辑强、重大事不拘小节，无神论倾向，反感迷信；从追求「不败之法」到传播王道。｜装备·能力：骑士战技、炼妖壶、时空潜质、后得剑仙赋身（寿三十年）。｜人物弧光：情报使者 → 东行求真 → 死而再肉身 → 王道传播者。｜立场关系：友妮可 / 薇达 / 慧彦·李靖 / 卡玛·安卡 / 剑仙；敌撒旦系；对契约者先防后用，软肋是「是否仍信不败之法」。诗词意象：「为求真理东方行……祈愿尘世行王道。」

- **妮可（Nicole）**｜身份：撒旦派来的女魔将使者，移动岛主人；前世独孤宁珂，古月仙人置魂入伏羲琴净魔性九十九年。｜性格：爱哭爱闹、单纯深情、记忆缺失。｜装备·能力：魔将底蕴、暗系法术与弓技、炎魔神姆斯比尔等召唤；敏捷高、血防弱。｜人物弧光：净魂少女 → 恋赛特 → 助封撒旦 → 欧陆转世为人与赛特终老。｜立场关系：任务是拉拢赛特入魔界，终反撒旦；与卡玛吵嘴。

- **薇达（Widad）**｜身份：阿拉伯海军女豪杰，祖传劲旅统帅，因性别被排斥发配边疆。｜性格：沙场直率、敢爱敢恨、不守对女性的诸多限制。｜装备·能力：海军统帅战力、舰队指挥。｜人物弧光：边军女杰 → 赛特妻子 → 约十年后去世。｜立场关系：卡玛孪生灵魂姐妹；对契约者若摆骑士优越感则立刻翻脸。

- **卡玛（Kama）**｜身份：印度爱情精灵，薇达守护精灵 / 孪生灵魂。｜性格：执念红尘、见证无数悲剧爱情。｜装备·能力：精灵法术、预知与赛特命运相关的「看戏」动机。｜人物弧光：背约游荡 → 古墓被释 → 随队 → 终回守薇达。｜立场关系：与安卡一体；对赛特是命运观察者而非单纯战力。

- **安禄山**｜身份：唐边将，安史之乱发动者。｜性格：野心。｜能力：军政。｜弧光：长安倒计时压力锅。｜立场关系：历史线，非神魔顶点。

**【势力图谱】**
- **法兰克 / 丕平政权**：订单是「战争不败之法」；赛特是采购员，不是自由旅人。
- **梅罗文加护教骑士团（麦尔斯部）**：以抓异端为业；信仰暴力的人间脸。
- **威尼斯执政与商路**：运河、火刑、海盗船；东行船票与情报市。
- **阿拉伯海军（薇达）与阿拔斯 / 倭马亚**：革命巷战、女将排斥、海军封锁。
- **大唐朝廷 / 安禄山 / 高仙芝系**：怛罗斯与安史双战场；加强版军职线见腐败。
- **少林**：慧彦骨灰归处，武学与止战并置。
- **魔界撒旦系（康纳里士等）**：升岛、觉醒、反曼陀罗。
- **轩辕剑仙一脉**：条件性秩序侧，赋身有寿命税。
敌友逻辑：利益与信仰优先；盟约可破；暴露炼妖壶或时空潜质 = 被魔界与人界同时盯上。

**【贵重物品】**
- **炼妖壶**：收化、炼化、终局封撒旦；东西方祭坛同材异果。谁想要：魔界与野心家。抢到后果：持壶者心性决定壶是狱还是胃。当前阶可用，但终局封印不是「吸收成自己力量」。
- **六芒星阵**：开西方祭坛。修道院入手。
- **封神坛**：开东方祭坛。天山李靖侧入手。
- **反曼陀罗阵**：时空 / 统治级；六阶守阵眼，禁止当永久外挂拾取。
- **慧彦骨灰（舍利子）**：圣物级任务道具；两边军队都可能抢「杀死高僧的证明」或「英雄遗物」。
- **移动岛石像操控权**：航向政治。
- **轩辕剑（剑仙形态）**：顶点级，低阶只闻其名。
- **神魔异事录**：情报与图鉴，中低阶硬通货。
- **航海图 / 军职令 / 通行符**：身份与通行，露富即杀劫。

**【隐藏剧情 · 伏笔】**
- 赛特前世与撒旦同反天界，反曼陀罗与时空能力是终局钥匙。
- 赛特 = 宇文拓后裔（官方后设）；主线对白不必反复官宣，避免破坏自我认知弧光。
- 妮可 = 独孤宁珂净魂；净化九十九年是喜剧外壳的来源。
- 轩辕剑仙与皇甫暮云神识线点到为止，勿整本倒灌《云之遥》。
- 原定结局更悲剧（赛特返欧被当异教徒烧死等），部分灵感移至《天之痕》；正式结局为传播王道 + 转世重逢。
- 加强版唐职线：公文、校场、密报，少用无敌斩。
- 隐藏战：蚩尤、牛魔王等（高阶可选，不改主线封印逻辑）。

**【大事记时间线】**
前史：天界造反与反曼陀罗；宁珂净魂成妮可；宇文拓血脉至赛特。  
开篇：赛特受命东行 → 威尼斯救商与妮可 → 败麦尔斯东遁。  
移动岛：康纳里士升岛 → 莉莲死 → 时空潜质初醒 → 岛向阿拉伯。  
冰山：蓝魔神 → 薇达同盟。  
阿拉伯：小肯迪 → 阿拔斯革命 → 古墓卡玛·安卡 → 慧彦授王道。  
西域：石国与远恩 → 怛罗斯慧彦殉道 → 蜃楼虚幻五年 → 剑仙·李靖返时。  
大唐：长安王思月家 →（加强版军职）→ 安禄山将反 → 麦尔斯觉醒为撒旦 → 赛特死。  
终局：剑仙赋身（寿三十年）→ 破反曼陀罗 → 封撒旦入壶 → 返欧传王道 → 薇达卒 → 妮可转世终老。

**【叙事基调 · 雷区】**
史诗东行 + 王道思辨 + 文明碰撞细腻。忌：把赛特写成无脑十字军或滥好人；忽略慧彦牺牲重量；低阶硬刚撒旦本体；把历史事件当精确教科书（以游戏为准）；把安卡写成无战力花瓶；把薇达 / 妮可写成后宫清单；三地同一套口头禅；编造原作没有的终局死亡与宝物来历；战力归零式削弱顶点。  
最早切入锚点：**威尼斯护教团抓捕异端当夜**，或**赛特刚得东行王命**。

**【世界剧情线 · 威尼斯到移动岛细目】**
赛特为丕平收集战情，在威尼斯撞上护教团以信仰为名的屠杀。救商人、得妮可、败麦尔斯，被迫东遁。海战中康纳里士升岛，把人间战争变成魔界棋盘。石像只认妮可主权，岛的移动是谈判不是自动驾驶。甲板上拉丁祷文、阿拉伯航海符、未燃尽的火油并置——两边的死长得一样。此段主题：战争的正义话术有多廉价；可介入 = 运河救商、密道带路、石像三问、航向投票。

**【世界剧情线 · 薇达军中的排斥与战功】**
她有祖传劲旅却被发配边疆，上级用性别与礼法做刀子。赛特若以法兰克骑士的优越感说话，会立刻失去这位盟友。薇达后成为妻子并十年而卒，要求正文给足相处的日常与战场互相救命，而不是结局字幕一句「结婚」。可介入 = 冰山停火、俘虏交换、舰队补给、替薇达挡一次「女将不得入厅」的羞辱。

**【世界剧情线 · 卡玛与安卡的古墓伦理】**
阿拔斯官员称她为镇压用恶魔，实际是爱情精灵。释放她可能乱，囚禁她可能罪。三阶任务的好结局是「让她自己选是否离开墓穴」，而不是玩家当判官处死。安卡的吐槽是队伍良心：提醒卡玛别把赛特当又一场悲剧爱情的戏码。

**【世界剧情线 · 怛罗斯的两军与一僧】**
慧彦横锡杖于沙，不是战力碾压，是道德勒索：谁先射谁先亵渎自己的神。他的死把骨灰托付变成圣物级任务道具。四阶切入应让契约者在「传话 / 肉盾 / 救伤兵 / 假骨灰诱饵」中选，而不是「一掌扇飞两军」。黄雷埋伏与远恩人质线并行。

**【世界剧情线 · 蜃楼五年与王道反面教材】**
幻境里赛特「赢了」：结婚、执政、攻唐——正是不败之法的极致，也是王道的反面。慧彦亡魂破局后，赛特目标更硬。写此段忌把幻境当免费外挂时间；破局后身体与心性都有税。

**【世界剧情线 · 长安双重倒计时】**
安禄山将反是历史倒计时；撒旦觉醒是神魔倒计时。两者在皇宫夜对齐。赛特毁肉身而死，再肉身只有三十年，把胜利税写进寿命。妮可助封有功转世，赛特在薇达卒后与转世妮可终老——多段感情是王道旅途的人间税。可介入 = 坊市撤侨、军职密报、宫城外围破觉醒外阵、阵眼抢修、壶口协作。

**【世界剧情线 · 赛特身份四层】**
表层：混血骑士。中层：丕平情报官。深层：反曼陀罗前世与时空能力。系列层：宇文拓后裔（非转世）。写对话时分层透露，低阶只给表层；异色瞳标记「两边都不完全接纳」——在法兰克被当东方种子，在大唐被当胡人骑士，在阿拉伯被当可用的外人，王道是他唯一能站的「第三边」。

**【世界剧情线 · 丕平订单的政治风险】**
赛特东行不是个人游学，是法兰克执政的情报采购。王道答案否定订单本身——最好的战法是少开战。返欧若宣称「战争不败之法不存在」，可能被丕平视为辱命。王道传播需要盟友：转世妮可的人间身份、薇达旧部残存、修道院改革派。六阶奖励应是「话语权与免死」，不是魔帝级法术。

**【文明三色写法】**
威尼斯：水与钟、商语与拉丁诅咒、蜡与火刑架。  
大马士革：巷与经文、军令与学问、沙尘与革命夜火把。  
长安：坊与宫、官话与烽火、公文与密报。  
禁止三地同一「侠士」口头禅；音乐服饰对白随地变。

**【时点三问（轩三）】**
1. 妮可是否已跟队？  
2. 慧彦是否已殉？  
3. 赛特是否已死而再肉身？  
三问决定当前可用 NPC、可用宝物与危险度。

**【可介入任务表】**
运河救商、岛三问、冰山停火、革命夜担架、古墓谈判、怛罗斯挡箭 / 传讯、骨灰伪装护送、长安撤侨、军职密报、阵眼抢修、壶口协作、隐藏蚩尤 / 牛魔王（仅高阶可选）。

**【丝绸之路作为舞台骨架】**
本世界不是「几个孤岛副本」，而是一条可走的欧亚商路：威尼斯港货 → 地中海 / 移动岛航线 → 贝鲁特与大马士革巷市 → 巴格达与石国遗民 → 怛罗斯风沙 → 天山关隘 → 长安坊市。商队、译语人、假货香料、军票与革命传单，是中低阶最高频的可玩资源。契约者若只打怪不碰商路政治，会错过王道主题的人间面——止战从来不只是战场上的锡杖，也包括谁在运粮、谁在抬价、谁在用「异端」话术抢货。

**【炼妖壶双祭坛的文明隐喻】**
西方六芒星阵与东方封神坛不是换皮 UI：同材料在两边祭坛炼出不同神魔 / 道具，逼玩家用「两套世界观」思考。AI 写炼化场景时，须让本地人认出符号（修道院修士认六芒星，唐僧道认封神坛），禁止两地通用「炼丹口诀」。壶终局封撒旦，是收纳为狱，不是吸经验成自己的魔帝；与系列其他作「壶中仙」对照时，点到持壶者心性即可。

**【结局群像与开放式收尾】**
封印后队伍解散有明确分工：妮可欧陆转世；卡玛回守薇达（婚后约十年薇达卒）；李靖留守护唐；安卡与轩辕剑仙随赛特继续未完旅程；远恩弃仇转向西域音乐；王思月一线可接义勇抗乱。赛特返欧传播王道，面对丕平「订单未完成」的政治风险，最终在薇达身后与转世妮可共度余年。写结局忌大团圆无税：寿命三十年、人间离散、历史安史仍可能爆发——王道改变的是「如何对待战争」，不是抹掉所有烽火。

**【与系列互文边界】**
可点：宇文拓后裔、独孤宁珂净魂、轩辕剑仙与暮云神识。不可：把《天之痕》《云之遥》整本剧情倒灌进本世界主线；契约者低阶不应「全知系列设定」。本世界自洽优先，互文是彩蛋不是主菜。

**【战斗与神魔生态（写手用）】**
常规遭遇随地段切换：威尼斯侧为护教团兵、海盗、海怪与低阶欧陆魔；移动岛与冰山为石像灵、蓝人与蓝魔神部属；中东古墓为机关兽、陵卫与被误称「恶魔」的精灵类；怛罗斯为两军士兵与黄雷机关；长安为唐兵、乱兵与撒旦系高魔。战力描写宁低勿高：士兵互砍是一阶叙事，恐惧之王升岛才是四～五阶气压，八臂撒旦与反曼陀罗是六阶终盘。妮可暗系法术对撒旦本体可能无效甚至反补，终局应强调协作、阵眼与炼妖壶，而不是单一输出循环。神魔异事录是情报硬通货——契约者可用「记全某类妖」换商人信任或军中通行证。

**【本地货币与资源】**
乐园币不进土著报酬。欧洲用银钱第纳尔系，阿拉伯商路用第纳尔 / 迪拉姆与货换货，大唐用铜钱绢帛与军票。硬通货还包括：香料、马匹、机关零件、炼妖材料、通行符、假身份文书。露富即杀劫；安卡与卡玛对「把爱情当交易」极度敏感，勿用买通精灵推进主线。

## 阶位切入点

> 阶位↔本世界战力：一阶≈市民士兵；二阶≈护教团骑士 / 海军将校；三阶≈成名冒险者 / 中型魔 / 高僧战力门槛；四阶≈恐惧之王 / 李靖级 / 怛罗斯可左右战局；五阶≈半神魔将 / 觉醒前麦尔斯；六阶≈撒旦 / 剑仙 / 反曼陀罗。世界顶点（撒旦与轩辕剑仙）为六阶巅峰，低阶切入一律「规避顶点 · 情报优先」；条件性胜利，严禁硬刚本体。

**一阶（市民 · 士兵 · 威尼斯井底线）**  
切入身份 / 时点：契约者以威尼斯码头苦力、修道院杂役、护教团新兵或阿拉伯商队学徒身份切入，锚定赛特刚得东行王命、护教团抓捕异端当夜之前或当夜。  
初始事件：运河边一场「抓异端」演成火刑预备；你被点名作证或抬柴，目睹赛特潜入救商、神秘巫师召唤妮可——你手里只有绳索、短刀与一张能换命的假通行证。  
开场白建议：「威尼斯的潮水拍着木桩，钟楼响过三遍。护教团的人说今夜要净化运河——可你看见那个异色瞳的骑士把阿拉伯商人往密道里推，而一个爱哭的女孩突然出现在火把底下。」  
关键 NPC 立场：**赛特**——此时只是情报官，未必信你；**麦尔斯**——护教团长，视你为可消耗的证人；**妮可**——刚被召唤，缠人却无完整记忆；码头牙人——卖船票与假身份。  
主线钩子 / 支线：救商或告密的二选一；六芒星阵到手前后的修道院潜入；弄到东行船票。支线 = 火刑旁观记录、运河走私、替莉莲传一句未说完的话。  
危险度 / 规避：中——凶险来自护教团刀与火刑；须规避康纳里士、完整麦尔斯战力与任何魔将正面对抗。  
任务方向 / 奖励：活着离开威尼斯或混上船；奖励取向 = 通行符、初级药剂、炼妖壶相关线索（非壶本体）、少量乐园币等价硬通货（本地币以银钱为准）。

**二阶（骑士 · 将校 · 移动岛 / 蓝人线）**  
切入身份 / 时点：以护教团叛逃骑士、海盗船副官、薇达部曲外来顾问身份切入，锚定移动岛升起或蓝人冰山撞击前后。  
初始事件：岛升起后两边舰队变标本；石像要航向，妮可记不起魔将岁月；你被推去「跟石像说话」或「进冰山找薇达」。  
开场白建议：「甲板裂开，海底抬起一整座岛。拉丁祷文和阿拉伯符咒烧在同一片火油里——石像开口问：去哪里？你发现没有人敢先回答。」  
关键 NPC 立场：**妮可**——岛主权却失忆；**康纳里士**——四阶边缘压迫，宜规避正战；**薇达**——可结盟，忌性别歧视话术；**麦尔斯**——情敌与暂时共敌。  
主线钩子 / 支线：石像三问与航向；冰山停火 / 蓝魔神外围；莉莲仪式的干扰（难改必死时可改为多救一人）。支线 = 岛上搜刮、俘虏交换、为薇达舰队弄补给。  
危险度 / 规避：高——须规避恐惧之王全力与蓝魔神本体硬刚。  
任务方向 / 奖励：促成东航或救下薇达舰队一角；奖励取向 = 中阶武具、海军令、炼妖材料、西方祭坛炼化入门。

**三阶（成名冒险者 · 革命与古墓线）**  
切入身份 / 时点：以大马士革佣兵、小肯迪的游学同伴、革命巷战中的灰线联络人身份切入，锚定阿拔斯革命与塔德莫尔古墓开启前后。  
初始事件：官员麦尔丹要放「古墓恶魔」镇压革命；你与赛特 / 小肯迪一并被卷进黄道机关——深处不是恶魔，是卡玛与安卡。  
开场白建议：「大马士革的巷子里，口号比刀快。有人说古墓里有能结束革命的恶魔；你跟着那位异色瞳的法兰克人走进黄道门——石棺打开时，飞出来的是一只会说话的黑猫。」  
关键 NPC 立场：**小肯迪**——学问与向导；**卡玛**——要自决；**安卡**——吐槽与战力辅助；**麦尔丹**——可杀可逐；**慧彦**——王道初授前夜出现。  
主线钩子 / 支线：古墓伦理（释放 / 囚禁 / 自选）；革命夜担架与假情报；王宫黄雷伏击的预警。支线 = 四象机关解谜、炼妖壶收化中型魔、为薇达传军情。  
危险度 / 规避：高——须规避黄雷机关兽群与任何撒旦系高魔。  
任务方向 / 奖励：让卡玛安卡以「自选」方式入队或离队；奖励取向 = 精灵系道具、古墓祭器、革命派通行、中阶法术残卷。

**四阶（名将门槛 · 怛罗斯 / 天山线）**  
切入身份 / 时点：以联军传令官、唐军或大食军中立观察者、护送高僧的护院身份切入，锚定怛罗斯开战与慧彦殉道前后；或蜃楼破局后天山剑仙指引时。  
初始事件：锡杖横沙，两军对峙；你必须在「传话 / 肉盾 / 救伤兵 / 带远恩上山」中选一条路——慧彦将死，骨灰托付需要活人接力。  
开场白建议：「怛罗斯的风沙里，唐军与大食的旗帜一样红。一位少林僧把锡杖横在阵前，说：谁先射，谁先亵渎自己的神。你站在中间，手里只有一面旗和一条命。」  
关键 NPC 立场：**慧彦**——殉道前托付；**远恩**——人质与政治；**黄雷**——埋伏；**薇达**——战后收容；**轩辕剑仙 / 李靖**——天山后半段才可触。  
主线钩子 / 支线：骨灰伪装护送（真假骨灰诱饵）；蜃楼幻境的破局辅助；东方祭坛开启条件。支线 = 救伤兵、联军情报、石国遗民安置。  
危险度 / 规避：极高——须规避两军主帅对轰与撒旦系黑魔法全力。  
任务方向 / 奖励：完成骨灰东送的第一段或促成返时条件；奖励取向 = 高僧遗物（非骨灰本体支配权）、封神坛线索、李靖级战技残篇、地区级声望。

**五阶（半神魔将 · 长安觉醒前夜线）**  
切入身份 / 时点：以长安方士、唐军中层将校（加强版语境）、传教团护卫或王思月家门客身份切入，锚定安禄山将反、麦尔斯即将完全觉醒为撒旦前夜。  
初始事件：宫城外围恐惧领域初显；康纳里士与麦尔斯已先至；你要在「撤侨 / 密报粮草 / 破觉醒外阵」中选，赛特将毁麦尔斯肉身并被元神杀死——你改的是外围，不是必死本身（除非六阶条件链）。  
开场白建议：「长安的坊鼓乱了。有人说安禄山要反，有人说宫里有比反贼更可怕的声音。异色瞳的骑士拔剑冲向一个你曾在威尼斯见过的护教团长——那人的影子，长出了不属于人的角。」  
关键 NPC 立场：**赛特**——赴死前夜；**妮可**——情感与魔将底蕴双线；**王思月**——撤侨锚点；**康纳里士**——恐惧领域；**安禄山**——历史压力锅，非神魔顶点。  
主线钩子 / 支线：坊市撤侨；军职密报；破坏觉醒外阵；保护妮可不被魔界回收。支线 = 杨国忠系政治、义勇军萌芽、少林骨灰安放后续。  
危险度 / 规避：极高——须规避撒旦元神与反曼陀罗完整启动。  
任务方向 / 奖励：降低长安平民伤亡或保住赛特再肉身的外围条件；奖励取向 = 唐军令、高阶法器、魔将级材料、剑仙出现的情报链。

**六阶（顶点 · 反曼陀罗与封壶线）**  
切入身份 / 时点：以赛特再肉身之后的同行者、剑仙认可的阵眼守护者、炼妖壶协作封印者身份切入，锚定旧长安废墟反曼陀罗阵与八臂撒旦决战。  
初始事件：阵眼以残酷材料布成；撒旦欲统治人界；你与赛特、妮可、李靖等协作——胜利条件是**破阵 + 开壶封印**，不是单挑吸魔帝当经验。可附带隐藏蚩尤 / 牛魔王，但不改封壶终局。  
开场白建议：「汉长安的废墟下，反曼陀罗阵像一只睁开的眼。赛特只剩三十年寿，妮可的魔将底蕴在暗处发烫——轩辕剑仙说：壶口一开，要么封神魔，要么人界先崩。」  
关键 NPC 立场：**赛特**——寿命税下的决死者；**妮可**——助封有功将换转世；**轩辕剑仙**——条件性顶点援手；**撒旦**——六阶巅峰本体；**安卡**——仍随队；**卡玛 / 薇达 / 李靖**——结局分流前最后并肩。  
主线钩子 / 支线：阵眼抢修 / 破坏；壶口协作；隐藏魔神战；为返欧传王道预留政治后手。支线 = 结局群像（远恩弃仇、王思月义勇等）的因果修补。  
危险度 / 规避：顶点级——禁止无协作硬刚八臂撒旦；禁止把反曼陀罗当永久外挂搬走。  
任务方向 / 奖励：封印成功与王道话语权；奖励取向 = 炼妖壶封印协力证明、剑仙级一次性护符、返欧免死与传播权、转世线情报（妮可）——**不是**魔帝本体力量。
