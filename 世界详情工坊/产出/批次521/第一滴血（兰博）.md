# 第一滴血（兰博）

## 剧情

**【作品来源】**  
《第一滴血》（英语：*First Blood*，港译《第一滴血》/ 新译《兰博：第一滴血》；台译《第一滴血》；新马旧译《杀出孤独岗》）是美国动作·心理惊悚片，1982 年 10 月 22 日上映。导演泰德·柯契夫（Ted Kotcheff），主演席维斯·史特龙（Sylvester Stallone）饰约翰·兰博（John Rambo），李察·克雷纳（Richard Crenna）饰山姆·崔普曼上校（Colonel Sam Trautman），布莱恩·丹内利（Brian Dennehy）饰威尔·提索警长（Sheriff Will Teasle）。改编自大卫·莫瑞尔（David Morrell）1972 年同名小说；剧本核心由迈克尔·科佐尔、威廉·沙克汉奠基，史特龙参与改写。制片卡罗科（Carolco）/ Anabasis，猎户座发行。全球票房约 1.25 亿美元（预算约 1500 万），开启「兰博 / 第一滴血」系列：续集含《第一滴血 2》（1985）、《第一滴血 3》（1988）、《第一滴血 4 / 热血回归》（2008）、《蓝波：最后一滴血》（2019），以及在制前传《John Rambo》（约 2027）。文风：写实丛林游击、PTSD 与越战老兵创伤、冷兵器与热武器近战、反战社会批判（正传）与后期动作升级。本档案以**电影正典第一部因果为常青主轴**，小说差异与续集脉络作互文与舞台延伸；搜笔趣阁未收录该小说全本（非网文），剧情以维基百科、角色列表与公开影评可核资料为准。

**【世界定位】**  
1980 年代初美国西北小镇与周边山地：表层是「希望镇」（Hope）一类民风保守的边境社区与州警、国民警卫队建制；底层是越战后遗症、流浪罪、警察暴力与特种作战技能回流民间。主角**约翰·詹姆斯·兰博**——绿扁帽 / MACV-SOG 退役荣誉勋章得主——在寻访战友、被镇警羞辱后爆发，一人对抗搜捕队。一句话：没有异能、没有复活外挂的**人体极限 + 丛林求生 + 游击陷阱**世界；契约者切入的是热武器与冷兵器写实战，顶点是「被逼入绝境的荣誉勋章特种兵」，不是灭城魔神。

**【世界观 · 力量体系】**  
本世界**无修真、无异能、无系统面板**。战力 = 特种训练 + 丛林求生 / 游击战经验 + 武器与地形利用 + 意志与创伤应激。死亡即死亡；重伤靠包扎与医疗，不能靠「突破境界」回满。

**1. 凡人 / 平民层**  
市民、餐馆雇员、普通路人、无作战经验青年：徒手或偶发手枪，一枪可致命。≈一阶初期。

**2. 地方执法 / 普通警员**  
希望镇副警长、州警巡逻、国民警卫队新兵：有枪械与编制，但缺乏丛林追击经验，易被陷阱与伏击拆散。≈一阶中期。

**3. 老练执法 / 猎人 / 国民警卫队骨干**  
如猎犬队、有实战经验的副警长、州警精锐：能组织直升机与犬搜，但仍难在密林中锁死特种兵。≈一阶后期～二阶初期。

**4. 正规军特种 / 绿扁帽级**  
接受过生存、侦察、爆破、近战体系训练的特种部队成员：可在人数劣势下用地形、陷阱、冷热兵器组合持续压制地方武力。≈二阶中期。

**5. 顶点：约翰·兰博（电影正典表现）**  
荣誉勋章得主、Baker Team 长程侦察幸存者、战俘营逃脱者。第一部中：徒手突破警局、摩托遁入山、以石击落直升机致副警长坠亡（间接）、用陷阱重创整支搜捕队却**刻意不杀**多数警员，后携 M60 回镇炸油站、断电、重创警长。破坏力 = 单人清场级枪战 + 车辆 / 油站爆炸 + 密林陷阱，**达不到**夷平城池、抗导弹。  
**政治 / 编制顶点**：上校级指挥官（崔普曼）、州国民警卫队、联邦羁押体系——个人武力未必最强，但可调动建制火力与法统。对契约者 = **情报与编制碾压**，不是魔法。

**关键规则（比「境界」更重要）**  
- **PTSD 闪回**：酷刑、密闭、剃刀等感官刺激可触发越战战俘记忆，导致失控爆发。  
- **流浪罪 / 歧视越战老兵**：1980 年前後部分地区仍可借仪容与「麻烦分子」标签驱逐、逮捕。  
- **荣誉勋章与特种档案**：一旦身份曝光，执法层会震惊但仍可能因私人恩怨继续升级。  
- **丛林求生（bushcraft）**：陷阱、伪装、利用雨雾与地形，是兰博对地方武力的核心优势。  
- **「他们先流第一滴血」**：电影名场面台词，定调冲突责任归属与叙事道德轴。  
- **小说与电影分轨**：小说兰博杀伤极重、终局与警长双亡且被崔普曼击毙；电影刻意收束为间接致死 + 投降收押。乐园任务默认以**电影正典**为常青时间线，小说作「平行残酷轨」可选。

**乐园阶位映射（对照《阶位战力图鉴》，宁低勿高）**  
平民 / 普通警员 / 国民警卫队炮灰≈**一阶**；职业特种、绿扁帽、州警精锐、国民警卫队指挥官≈**二阶**；世界顶点 = **约翰·兰博（二阶巅峰游击 / 枪斗）+ 崔普曼上校所代表的特种指挥与联邦法统**。低阶切入一律**规避与兰博正面对冲、规避国民警卫队火箭弹级火力**，情报与地形优先。严禁用「任务限制战力 / 剧情削弱」解释顶点——顶点始终是完整战力的特种兵，低阶只能规避或条件性周旋。

**【地理 · 舞台】**  
- **希望镇（Hope）**：电影主舞台（实景多取加拿大不列颠哥伦比亚省 Hope 镇与周边峡谷，片中设定为华盛顿州）。主街、警局、加油站、枪械店、电力变压器、警长办公室天窗——终局枪战与崩溃独白发生地。桥与公路出入口构成天然封锁线，镇民被要求留在室内时，整座镇变成提索与兰博的私斗场。  
- **镇郊公路与城界标志**：提索「送出城」的羞辱起点；兰博折返触发逮捕。此段公路是契约者最早可目击「第一滴血如何被点燃」的位置。  
- **查普曼峡谷 / 密林山地（片中称 Chapman Gorge 一类）**：直升机追击、悬岩、陷阱、猎犬搜山、兰博重创搜捕队的主战场。雨雾、陡坡、断枝与泥地让地方警员的编制优势失效，特种兵的生存训练在此完全展开。  
- **废弃矿坑 / 通风井 / 积水坑道**：国民警卫队以 M72 LAW 火箭弹轰塌洞口、误判兰博已死；兰博经坑道与通风井逃出。片中曾计划用蝙蝠，后因可控性改为老鼠——潮湿、黑暗、回声是这一段的感官关键词。  
- **军用卡车与补给线**：兰博劫车得 M60，返镇前的火力升级节点；卡车冲入加油站引发连环爆炸，切断追兵公路。  
- **枪店与警长办公室**：回镇后的火力补给与私人恩怨终点；天窗坠落的提索是视觉名场面。  
- **小说平行舞台**：肯塔基州玄武岩县麦迪逊镇（Madison, Basalt County）——莫瑞尔以宾州 Bellefonte 为原型；私酒作坊、矿洞、全镇纵火等更血腥节点。  
- **续集延伸舞台（互文）**：越南战俘营与三角洲（第二部）、阿富汗—巴基斯坦边境圣战补给线（第三部）、缅甸克伦邦与萨尔温江（第四部）、亚利桑那鲍伊牧场与墨西哥卡特尔边境（第五部）。本世界卡以希望镇事件为核心；续集作「同一人后续人生」的可选远征舞台。

**【世界剧情线】**  

**〇 · 前史（越战与 Baker Team）**  
约翰·詹姆斯·兰博生于亚利桑那鲍伊一带（系列后续补全出生信息；墓碑显示父 Reevis、母 Helga），少年时期离开有暴力阴影的家庭，入伍后经游骑兵 / 特种训练，编入崔普曼上校主导的特种长程侦察单位（通称 Baker Team）。队友包括德尔玛·巴里（Delmar / Delmore Barry）、约瑟夫「乔伊」·丹福思（Joseph "Joey" Danforth）、曼努埃尔「疯子」·奥尔特加（Manuel "Loco" Ortega）、保罗·梅斯纳、德尔伯特·克拉克豪尔、朱塞佩·科莱塔、拉尔夫·乔根森等。战场上兰博目睹乔伊被改装擦鞋盒诡雷炸断下半身、死在自己怀里——此记忆贯穿终局崩溃独白。1971 年前后单位遭北越伏击，幸存者被俘，兰博经历电击、刀烙、水刑与密闭囚禁后逃脱；他主动要求重返前线，直至 1974 年 9 月前后正式退役。回国后遭遇反战羞辱（被骂「婴儿杀手」、遭投掷垃圾）、工作难寻、漂泊无依。荣誉勋章与百万美元级装备操作经验，换不来代客泊车一类稳定工作——这句控诉后来成为系列最广为引用的社会批判金句之一。第一部开场前，他步行寻访最后一位战友德尔玛，得知其因橙剂（Agent Orange）致癌已死——Baker Team 在世者只剩他与崔普曼一脉。

**① · 进城与逮捕（1981 年 12 月 · 电影正典）**  
兰博长发、军外套、背囊与求生刀，沿公路进入华盛顿州希望镇，只想找一餐饭。警长**威尔·提索**视其为「不受欢迎的流浪者」与潜在麻烦，驱车载至城外丢下，明确表示「这里不欢迎你」。兰博感到被羞辱，折返入镇；提索当场逮捕，押回警局。  
副警长**亚瑟「亚特」·高尔特（Art Galt）**主导羞辱式处理：殴打、高压水冲、强迫清洗与剃刮。其他副警长多半附和或旁观；唯有**米奇·罗杰斯（Mitch Rogers）**等少数人流露不适。剃刀贴面时，兰博闪回北越战俘营酷刑，PTSD 爆发，徒手与冷兵器反击，打倒多名警员，夺回求生刀，冲出警局，劫摩托车遁入郊外山地。提索驾车追击失败，撞车后开始升级搜捕。

**② · 密林猎杀与「第一滴血」**  
提索组织副警长、猎犬与直升机搜捕。高尔特违令从低空直升机上向兰博开枪，甚至威胁飞行员保持航线以便射击，无视提索对讲机里的「活捉」命令。兰博被逼至悬崖，跃下树丛受伤后，以石击裂直升机风挡；直升机失控，高尔特坠崖身亡——**全片兰博唯一间接致死**。提索得知兰博为绿扁帽、荣誉勋章得主仍拒收手，将搜捕私人恩怨化。  
兰博试图投降却遭开火，遂以丛林陷阱、伪装与近战逐一重创搜捕队员：猎犬被解决、警员腿伤与坠崖险情、提索被刀抵喉警告「不要再追」。电影版刻意让兰博**多数时候致伤而非击毙**，与小说的大规模杀戮形成对照。无线电中，崔普曼上校现身劝降；兰博回以名句：**「他们先流了第一滴血，不是我。」**（They drew first blood, not me.）

**③ · 国民警卫队与矿坑**  
州警与华盛顿国民警卫队增援，封锁扩大。崔普曼向提索建议：先放松戒备让兰博下山，再以临检抓捕——提索拒绝，认为那是示弱。兰博退入废弃矿坑；警卫队以 **M72 LAW** 火箭弹轰塌洞口，现场误判其已死并开始收队。兰博经通风井与积水坑道逃出，劫持路过军用卡车，迫使司机跳车，携 **M60 通用机枪**返回希望镇——冲突从「山里的人猎」升级为「镇内的一人战争」。

**④ · 回镇与崩溃收场**  
兰博炸毁加油站截断公路、切断变压器令全镇停电、在枪店与警局区域交火，以 M60 重创提索（腿伤）。即将补枪时，**山姆·崔普曼上校**闯入，命令他停手。兰博爆发长段控诉：政府把人训练成能开武装直升机、开坦克、操作百万美元装备的战士，再丢回社会；反战者吐痰扔垃圾；战友被诡雷炸碎的画面夜夜重播——「有时醒来，不知身在何处；我不跟任何人说话……有时一天，有时一星期……我忘不了。」  
崔普曼安抚后护送其走出，兰博被联邦羁押。定格：兰博回望围观人群。提索上救护车。电影结束。  
（备用结局 / 特典：兰博求死、崔普曼开枪——拉斯维加斯试映观众强烈反对后，制片改用「活着收押」版；该死局仍存于部分碟片特典，并在第四部闪回中残影出现。）

**⑤ · 小说残酷轨（平行 · 非电影默认）**  
莫瑞尔原著中：地点为肯塔基麦迪逊；提索为朝鲜战争老兵、长期与精神压力斗争、妻子刚离开，并非单纯反派；警局无大规模酷刑羞辱，逮捕主因是流浪罪与仪容引发的「瘾君子」怀疑；兰博杀伤大量警员与国民警卫队；终局双方枪战互重创，兰博欲引爆炸药自尽时被崔普曼击毙，提索亦死。作者强调「建制」与「被社会遗弃者」双主角对位，章节在兰博与提索视角间切换。乐园常青档案以电影为准，小说作高残酷可选分支。

**⑥ · 续集因果链（系列互文 · 详述）**  
- **第二部（1985）**：狱中劳改约三年的兰博被崔普曼以「早释换任务」召回，赴越侦察战俘营；官僚**默多克（Murdock）**要求「只拍照不救人、不交战」。伞降丢失装备后，兰博与当地向导**高宝（Co Bao）**发现战俘并救出**班克斯（Banks）**；撤离被拒，兰博被俘受苏顾问**波多夫斯基（Podovsky）**与**尤辛（Yushin）**酷刑。高宝假扮营救，二人逃亡时高宝中弹死；兰博以爆炸箭等手段血洗营地，夺取直升机，火箭筒击毁波多夫斯基，返基地机枪扫指挥部并威胁默多克必须继续救战俘，再度漂泊泰国。  
- **第三部（1988）**：兰博在泰国修庙、参与棍斗；崔普曼赴阿富汗补给圣战者被苏军上校**扎伊森（Zaysen）**俘。兰博经**格里格斯（Griggs）**非官方渠道入境，得**穆萨·加尼（Mousa Ghani）**与少年**哈米德（Hamid）**协助两度闯要塞，救出崔普曼与其他战俘，终局与圣战者骑兵冲锋，以坦克撞击击杀扎伊森直升机。兰博将高宝项链赠哈米德。  
- **第四部（2008）**：兰博在泰缅边境捕蛇、摆渡；传教士**迈克尔·伯内特**与**莎拉·米勒**请其送医药物入缅。途中海盗欲索莎拉，兰博击杀海盗；村落遭塔玛多军血洗，传教士被俘。牧师雇**路易斯**等雇佣兵，兰博锻造新刀同行，弓箭与重机枪清场，刀杀少校**丁（Pa Tee Tint）**，终返亚利桑那「R. Rambo」邮箱的牧场路。  
- **第五部（2019）**：兰博经营亡父牧场，与**玛丽亚·贝尔特兰**及其孙女**加芙列拉**共同生活。加芙列拉赴墨寻生父遭卡特尔绑架；兰博入墨与记者**卡门·德尔加多**联手，以地道陷阱屠灭**乌戈 / 维克托·马丁内斯**团伙，终局重伤存活。  
续集战力与血腥度逐步升级，但仍属**人体 + 热武器**，不跨入城毁级异能。

**【主要人物】**

**【叙事基调 · 雷区】**  
基调：阴冷写实、雨雾密林、沉默的肉体暴力，第一部尤其是**社会批判惊悚**而非无脑肌肉爽片。忌把兰博写成无脑屠杀狂（电影版他多数时候在「停火—被打—再反击」）；忌消解 PTSD 与战友之死；忌把提索写成卡通纯恶而忽略其「建制傲慢」的真实；忌用异能 / 魔法解释特种训练；忌低阶任务让契约者正面硬刚国民警卫队火箭弹与兰博本尊。最早可切入锚点 = **兰博尚未进城、或刚被载到城界的公路时刻**；亦可锚定搜山中段「陷阱战」与终局前「回镇前夜」。


**【世界剧情线 · 补充细节（电影正典关键因果）】**  
希望镇事件的时间密度极高：从进城到投降大约横跨两昼夜。第一夜是警局羞辱与摩托入山；第二日是直升机、猎犬、陷阱与崔普曼抵达；矿坑火箭弹与回镇发生在搜捕队以为「已经结束」之后。契约者若要判断「世界处于什么状态」，可按下表：

1. **平静窗口**：德尔玛已死、兰博尚未入镇——公路上只有一个背囊男人。  
2. **点燃窗口**：提索送出城界到警局剃刀——冲突可被第三方打断或记录。  
3. **密林窗口**：高尔特坠崖后——提索私人化、国民警卫队将至，平民应撤离山道。  
4. **假死窗口**：LAW 轰洞后——全镇松一口气，兰博其实已在坑道里改道。  
5. **回镇窗口**：油站爆炸与停电——非战斗员必须躲入室内，电网与通讯混乱。  
6. **收场窗口**：崔普曼出现——唯一合法「软着陆」通道打开，硬刚双方都会输。

**电影版兰博的作战伦理（写给 AI 的行为锚）**：他会警告、会致伤、会拆编制，但第一部默认**不主动屠杀镇民与多数警员**；高尔特之死是间接与违令射击的结果。若任务把兰博写成无差别屠镇，即 OOC。小说轨才允许大规模击杀与双亡终局。

**崔普曼的战术建议为何正确**：他主张「放他下山再抓」，是因为绿扁帽在密林中对地方武力有不对称优势；提索拒绝的本质是面子与高尔特之死的复仇，不是治安最优解。契约者若站在州 / 联邦一侧，应优先支持「隔离—谈判—收押」而非「清剿—火箭弹」。

**续集如何改变世界温度**：第二部起兰博从「被猎者」变为「被国家再次利用的刀」；第三部地缘战争；第四部种族清洗现场的超暴力清场；第五部家庭保卫战。本世界卡常青主轴仍停在第一部收押定格，续集仅在玩家主动远征或时间跳跃时展开。

**与乐园任务的接口提示**：  
- 一阶任务多在「点燃 / 密林外围 / 目击者」层，奖励是情报与人脉。  
- 二阶任务可介入「假死 / 回镇前夜 / 谈判半径」，奖励可含特种训练与高级枪械，仍禁止灭城级道具。  
- 任何阶位若要「击败兰博」，须设计为谈判、陷阱、人质交换、联邦编制压制等**条件性胜利**，不得写成属性碾压秒杀。

**【地理 · 舞台补充】**  
希望镇并非「西部荒野」，而是 1980 年代初有加油站、枪店、电力与州际公路连接的小城。兰博的威胁感来自**把越战丛林带回美国街道**，而不是超自然。峡谷与矿坑的真实取景在加拿大弗雷泽谷地一带，但叙事上应保持「华盛顿州希望镇」设定，避免把拍摄地直接写成剧情国界。镇民在封锁令下的恐惧、媒体尚未完全涌入前的口耳相传、以及联邦身份档案通过无线电突然降临的震惊，都是可写的氛围层。

**【主要人物补充弧光】**  
提索与兰博在电影里是性格对立：一个把秩序等同于「把不想看见的人赶出去」，一个把生存等同于「他们先动手我才还手」。崔普曼则是第三人称的「造物主视角」——他既为兰博骄傲，也为自己训练出无法回归社会的战士而恐惧。米奇·罗杰斯代表体制内尚未被仇恨吞没的缝隙；高尔特代表把暴力当娱乐的毒细胞。把这些人写成标签（「好人警察」「坏人警察」）会失去第一部的社会批判厚度。

**【贵重物品与情报资产补充】**  
除刀与枪外，第一部真正值钱的「战利品」是：警局虐囚的目击证词、兰博荣誉勋章档案复印件、崔普曼的军用电台频段、矿坑通风井地图、以及「提索拒绝停火」的决策链记录。契约者若以记者 / 律师 / 联邦观察员身份切入，这些比 M60 更符合一阶奖励逻辑。

## 阶位切入点

> 阶位↔战力：一阶≈平民 / 普通警员 / 国民警卫队新兵 / 小镇帮工；二阶≈州警精锐 / 特种退伍 / 绿扁帽级 / 兰博本尊游击巅峰。世界顶点（约翰·兰博 + 崔普曼所代表的特种指挥与联邦法统）为跨阶存在，低阶切入一律「规避顶点·情报优先」。严禁用任务限制或剧情削力解释顶点；对顶点采取存在·情报优先 / 条件性胜利。

**一阶（希望镇日常与搜捕外围 · 流浪者 / 警员 / 目击者线）**

切入身份/时点：契约者以「过路拖车司机 / 加油站夜班 / 刚调来的实习副警长 / 来镇采访越战老兵的地方记者」切入，锚定**提索刚把长发男人载出城界、兰博尚未折返**的数小时窗口，或搜山开始后的外围封锁线。

初始事件：你在希望镇主街餐馆窗口看见警长车停又走；电台随即传来「警局有人袭警逃逸、嫌犯持刀入山」。封锁升起时，你的车 / 班次 / 记者证把你卡在检查点——高尔特派系的副警长要你指认「是否见过军外套长发男」，同时有人低声说「那人只是想吃饭」。你必须在「配合搜捕」「偷偷给山里送水粮」「装作什么都没看见」之间选边，而选边会在 24 小时内被双方记住。

开场白建议：「华盛顿州的雨把希望镇的霓虹洗成一片模糊的红。你看见警长把一个背囊男人丢在城界标志下，男人只是站了一会儿，又往回走。电台刺啦一声：袭警、逃逸、入山。有人开始发枪，有人开始发抖——而你的车，正好卡在唯一出镇的路上。」

关键NPC立场：**威尔·提索**——视外来者为麻烦，需要「听话的眼睛」而非英雄；**亚瑟·高尔特**——虐囚派，厌恶同情嫌犯者，危险；**米奇·罗杰斯**——反感酷刑，可能是体制内唯一愿意听你说「那人是老兵」的缝隙；**约翰·兰博**（远距）——此时只想离开，对平民无主动敌意，但受刺激即不可预测；**山姆·崔普曼**（尚未到场）——无线电与军用档案即将介入的影子。

主线钩子/支线：阻止或促成「第一滴血」升级（高尔特是否上直升机）、给兰博或搜捕队传递错误 / 正确情报、保护被误当帮凶的流浪汉；支线＝采访德尔玛遗属线索、记录警局虐囚证据、在封锁线倒卖雨衣与电池（事件与人名须本阶独有，勿与二阶雷同）。

危险度/规避：中——主要死于流弹、误伤、违令飞行员与恐慌开枪；须规避直升机旋翼半径、猎犬撕咬、兰博布置的绊索区。兰博本尊 = 情报优先，勿持枪追入密林「立功」。

任务方向/奖励：活过封锁夜、拿到可呈堂的录音 / 照片、与米奇或记者网建立信任；奖励取向＝警用防刺衣、合法枪证线索、希望镇地理手绘、越战老兵互助会联络方式（一阶勿发 M60 或特种档案原件）。

**二阶（密林对峙与回镇前夜 · 特种 / 州警 / 中间人线）**

切入身份/时点：契约者以「州警战术组顾问 / 国民警卫队排级军官 / 崔普曼提前派出的观察员 / 退役绿扁帽旧友」切入，锚定**高尔特已坠崖、提索拒绝收手、崔普曼抵达谈判**的时段，或矿坑火箭弹误判「击毙」之后、兰博劫军卡回镇之前。

初始事件：你随车队进入查普曼峡谷外围，对讲机里崔普曼的声音比提索更冷：「他还活着，你们死的人已经够多了。」下一秒，前方猎犬哀鸣，一名副警长腿上插着自制矛尖——兰博的警告信。提索要你带队「清掉这个疯子」；崔普曼私下塞给你一频段：「只观察，别当第一个流血的人。」你必须在**执行提索命令**与**执行崔普曼潜规则**之间做一次无法两全的选择，而兰博的刀，可能已经在你侧后方三米。

开场白建议：「雨停在树冠上，像成千上万把未出鞘的刀。你踩进泥里的瞬间，对讲机同时响起两个声音：警长要结果，上校要活口。林子里没有第三种声音——直到一根断枝以不属于风的角度弹起，你才明白：第一滴血，其实一直在等谁先开枪。」

关键NPC立场：**约翰·兰博**——对「不开枪的军人」可谈判，对「继续猎杀者」无慈悲；**山姆·崔普曼**——唯一能令兰博停手的人，需要你提供提索不会给的真实坐标；**威尔·提索**——私人恩怨已压过程序，会牺牲你的编制面子；**国民警卫队火力组**——手痒想射 LAW，可能再次误判；**米奇·罗杰斯**（若仍存活）——可作停火证人。

主线钩子/支线：矿坑「假死」情报战（是否上报已击毙）、阻止第二次火箭弹误射、在兰博回镇前切断油站与变压器（或相反：保护平民电网）、护送崔普曼进入终局谈判半径；支线＝回收高尔特坠崖点物证、与州检察线建立「虐囚」平行调查、截获军卡防止 M60 入镇（或确保其不落入第三方）。

危险度/规避：高～极高——陷阱、冷枪、LAW 误伤、回镇枪战流弹；须规避与兰博正面枪斗、规避提索情绪化指挥。世界顶点兰博 = 二阶巅峰人体极限，须以**存在·情报优先 / 条件性胜利**对待：停火、谈判、地形隔离，严禁一人硬刚传说。

任务方向/奖励：促成投降收押而非屠镇、保全崔普曼谈判窗口、拿到特种档案节选与希望镇事件完整时间线；奖励取向＝特种求生刀同级冷兵器、战术电台与密频、州警 / 国民警卫队通行证、崔普曼线任务推荐信（二阶可给高级枪械与训练，勿给灭城级装备）。
