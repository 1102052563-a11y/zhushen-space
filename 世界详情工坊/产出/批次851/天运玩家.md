## 剧情

**【作品来源】**
《天运玩家》作者**失落叶**（丁宗磊，数学系出身，网游数据流老作者，曾以《网游之盗版神话》等网游三部曲成名），起点中文网连载（书号 **1049375385**，分类游戏·游戏系统）。公开体量约 **13.15 万字 / 42 章**（截至 2026-07-20，最新章「贪欲战旗」），标签：游戏异界、气运流、穿越、系统流、升级流。官方调性金句：「众所周知，运气和爆率是游戏的第一生产力。一个玩家如果失去运气，将举步维艰。而我，气运加身，天选之子！」文风：2002 校园怀旧口吻（QQ、会员卡、一指禅打字、流星花园时代 BGM）与群星废土血腥生存并置；爽点落在**气运可见化爆率**、**心声情报碾压**、**灵魂萃取技能卡**与现实/废土双线切换。**连载中，禁止编造未公开终局**；日冕星人总攻与信号源切断完整流程仍属远景，写作只保留使命压力与已展开机制。

**【世界定位】**
双层嵌套世界：表层是 **2002 年江宁**（985 江宁大学信息与计算科学大一校园青春）；里层是信号源长成的 **群星世界**——废土、丧尸、红苔感染、绿洲城邦、拾荒队、巡查官编制与修行者体系。主角**林池鱼**本是 **2032 年**被日冕星人占领后的幸存者选拔玩家，经时间机器回拨至 **2002-03-18**，任务是**提前通关信号源生成的游戏，切断外星定位链路，拯救地球**。时间线污染使本应 14:00 开服的种田页游《群星OL》变成可侵入现实的真人模拟系统。一句话：**气运加身的天选玩家，用虫洞在 2002 教室与灾变纪元之间抢爆率、抢活路、抢切断信号源的先手。**

**【世界观 · 力量体系】**
本世界「力量」来自三层叠合：**系统规则**（任务/面板/储物/装备槽）、**肉体强化与技能卡**（属性点、技能萃取与装备）、**气运与心声**（爆率与信息差）。任何变强段落必须同步看见系统提示、冷却、体力/饥饿代价或感染风险，否则即 OOC。

**一、信号源与双世界宪法**
1. **日冕星人**：2032 入侵并占领地球的外星势力。以**可成长信号源**定位地球；信号源扎根网络，逐步生成游戏形态。提前通关对应阶段游戏即可切断该信号源与外星母体的联系。
2. **《群星OL》→群星世界**：2002 时间线第一阶段信号源本应是 **14:00 开服的网页游戏**；因时间线污染，引擎「升级」后直接侵入现实，解锁人物/技能/任务/储物等系统，把玩家拉进废土真人模拟。
3. **虫洞发生器**：绑定主角的跨界装置，冷却约 **1 小时**，在现实与群星登录/废土之间建立稳定虫洞。冷却未到、体力崩溃、感染失控都是切换失败条件。
4. **异次元储物空间**：系统解锁的随身仓库；可转移部分物资，但**希望币等镀层货币**现实侧不能直接花，须经提炼变现。
5. **任务双轨**：**群星任务系统**（废土/副本内）与**世界任务系统**（现实侧，如 S 级「永不挂科」）。主线远景包括通关群星、击杀关键节点（公开信息有「击杀白神」类终极气质任务，连载未完整展开，写作只作远景压力）。

**二、人物面板与装备**
1. **等级与属性点**：击杀丧尸/感染体得经验；升级给自由属性点（公开试读：每级约 3 点，可投入力量/耐力/敏捷等）。
2. **装备品级**：破碎 → 标准（白）→ 精良（蓝）等递进；词条含锋利值、力道、力量/敏捷加成与撕裂流血等特效。
3. **装备槽（八槽）**：武器 / 项链 / 戒指×2 / 徽章 / 图腾 / 面具 / **战旗**。图腾与战旗偏光环；战旗可主动展开，在身后形成旗帜形态。
4. **技能卡**：由灵魂萃取器从死者灵魂中萃取技能，形成可装备的技能卡；有成功率、灵魂配合度、能量消耗。技能发动消耗体能/热量（速度爆发后狂吃双汇火腿肠与方便面是公开名场面）。

**三、核心外挂**
1. **灵魂萃取器（天命宝物）**：来自 **2029 EWC 冠军礼包**，形态为血红/死亡沙漏。可检测逸散灵魂，检索技能列表（如庇护所建造、速度爆发、基础烹饪、单手斧基础等），配合度影响成功率；能量有限（一次萃取可耗 50%）。早期可萃 lv1–4 向技能。
2. **心声（次级天赋）**：来自 **2030 WEG 冠军礼包**。可听人/丧尸思维碎片——情报碾压与反伏击核心，也是校园抄作业、识破谎言的外挂。
3. **气运**：非嘴炮光环，而表现为**可见化爆率事件**——红苔开花预警、撞大运掉装、小地图红点、蓝装战旗等。气运可被观测、可被觊觎，不是绝对无敌。

**四、废土成长阶梯（按破坏力宁低勿高）**
1. **2002 校园普通人 / 系统刚解锁**：徒手、学生体质；危险来自社会与学业死线，不是开山。
2. **废土新人拾荒**：能砍衰退期丧尸、用煤气罐陷阱炸 lv10 丧尸；热武器与活跃期丧尸仍致命。
3. **技能成型 / 绿洲见习巡查**：装备速度爆发、耐力强化等，能 1V4、带队伏击、执勤隔离区；可对精英感染体用编制火力（电刑椅、车载机枪、腐蚀喷雾）。
4. **修行者 / 裂爪级**：练力体系（公开出现抱罡、龙力 lv4 等突破气质）；裂爪兽结晶等材料级战利品；个人破坏力抬到端据点/硬刚精英 BOSS。
5. **绿洲主力 / 气运碾压**：正式巡查编制、贡献点与南北派系资源、蓝装光环战旗带队；一城级冲突中可当尖刀，但仍惧尸潮与高阶寄生。
6. **切断信号源线高阶**：逼近信号源核心与日冕侧布局；战力从个人抬到「通关规则 + 情报 + 气运收割反制」。**无爆星超阶描写**。

**五、感染与死亡**
- **红苔花粉 / 星云毒素气质感染**：发烧、嗜血、自囚；日记主人汤阳即教材。衰退期丧尸弱而慢，活跃期快而狠；红苔开花伴随笑声与致死风险。
- **寄生突变**：宿主可炸裂成血藤触手形态（脑/眼嵌体、高腐蚀喷液），小地图以红点显示；精良蓝 BOSS 如休·克顿 lv25。
- **死亡**：废土死亡极真实；灵魂萃取依赖「正在逸散的灵魂」。现实侧死亡=任务失败级灾难。没有廉价满血复活。

**乐园阶位映射**（对照破坏力，宁低勿高）：2002 校园普通人 / 系统刚解锁 / 网吧开局≈**一阶**；废土新人拾荒 / 初装技能卡 / 煤气罐与白装斧≈**二阶**；绿洲见习巡查 / 技能成型 / 小队指挥与隔离区执勤≈**三阶**；修行者 / 裂爪兽结晶级 / 抱罡·龙力气质≈**四阶**；绿洲主力 / 正式巡查编制 / 气运与蓝装光环碾压≈**五阶**；切断信号源高阶线 / 日冕与信号源核心博弈≈**六阶**。世界当前可见顶点＝日冕星人 + 可成长信号源 + 群星通关规则；**低阶切入一律「规避顶点·情报优先/条件性胜利」**，严禁把日冕本体写成可一刀砍翻的路边 NPC，严禁无视感染与冷却抬战力。

**阶段〇 · 2032 派遣与回拨（前史）**
2032 地球遭日冕星人入侵占领。可成长信号源经网络定位地球并生成游戏；幸存者组织选拔玩家回拨过去，以「提前通关」切断信号。林池鱼中选，时间锚点定为 **2002-03-18 江宁**。他携带的不是旅游记忆，而是**冠军履历污染**：2029 EWC、2030 WEG 连续冠军礼包在新手阶段一并解锁，证明时间线已乱、未来战绩可「提前兑现」。此阶段世界状态：现实仍是平静 2002，群星页游尚未「开服」，主角唯一优势是情报与即将到手的系统。

**阶段一 · 神秘玩家 · 欣欣网吧开局（一阶核心）**
深蓝电光落地江宁大学外商业街。林池鱼按记忆冲向网吧，计划 14:00 登录《群星OL》。欣欣网吧无证无押金，老板不给机；他借正在抄乐谱的女生**陈曦**的电脑与网费，满屏 404 与搜不到开服信息。约五点道歉、互报姓名后，陈曦离开——系统光幕才升起：「游戏引擎升级完成……游戏正在启动……人物系统已解锁……技能系统已解锁……」页游计划被时间线污染成**真人模拟**。解锁：**群星任务系统、世界任务系统、异次元储物空间**；新手礼包 + 两份冠军礼包。

新手礼包公开内容气质：一代身份证（杭州住址）、现金约 200 元、江宁大学休学报告、**虫洞发生器**（1 小时冷却，连现实与群星）。冠军礼包：**灵魂萃取器**（天命宝物）、次级天赋**心声**。此阶段因果：主角从「找不到游戏」被迫接受「游戏已长进现实」；陈曦成为开局人情债与后线相关伏笔；身份证/休学报告解决 2002 身份落地。

**阶段二 · 新手村不存在 · 汤阳日记与灵魂萃取（二阶入口）**
首次虫洞进入群星废土：无新手村、无新手保护。地下庇护所内，**汤阳**（拾荒者）日记记录红苔花粉感染——发烧、嗜血上升、自锁客厅，求发现者给痛快。厨房后的丧尸即汤阳；心声听到「饿……吃你的脑子」。林池鱼堵门、砸天窗、拧煤气、爬出环壕，三次点火引爆，击杀 **丧尸 lv10**，经验暴涨（试读：直接 1→5 级气质），掉落**锋利单手斧**（标准/白，锋利+5、力道+5、力量+2，撕裂流血，需等级 5）。灵魂萃取器首次启动：检索汤阳技能（庇护所建造、速度爆发、基础烹饪、单手斧基础等），配合度从 50% 因对话与安息意愿拉到 100%，成功萃得**速度爆发 lv1** 技能卡；萃取耗能量 50%。技能发动后狂奔，却因能量守恒陷入空洞饥饿，狂吃方便面与火腿肠。

红苔开花章：衰退期丧尸经验极低；苔—藤—岩融合的杂交体伏击；血红苔中金色花苞与星红光晕、笑声、远处喊「快跑」。气运第一次以**死亡诱饵式奇遇**可见化。阶段因果：主角完成「学生→拾荒杀手」暴力教育；心声+萃取+煤气陷阱定下作战模板；庇护所爆炸引来尸群，坐标不再安全。

**阶段三 · 耐力强化、拾荒小队与 208 血洗（二阶成型）**
目录链：因为爱情 → 耐力强化 lv2 → 别开枪是我 → 离散数学（现实插叙）→ 灾变纪元 → 拾荒小队 → 伏击 → 1V4 → 208 惨遭血洗 → so easy。废土侧：耐力强化抬生存阈值；与拾荒队短暂协作又遭遇伏击/背叛气质事件；1V4 与血洗确立「气运+技能卡+心声」可越级。现实侧：赶课触发 S 级世界任务**永不挂科**（学期末全部通过，奖大量经验+超稀有技能卡×1）；室友**周凡**（208，挑染蓝发、叫「老鱼」、爱包夜传奇）、**杨学峰**、**袁刚**提供校园喜剧与作业渠道；**黄深**篮球线与「绿帽/女友」玩笑拉青春张力。阶段因果：双线节奏钉死——白天离散数学与球赛，虫洞 CD 一到钻回灾变；「别开枪是我」成为废土口令级名场面。

**阶段四 · 林叶、灰吹法与撞大运（现实经济线）**
**林叶**登场：学霸气质，补习数学分析/离散/连续可导，是永不挂科任务的关键 NPC。**灰吹法**章：林池鱼自称「江宁大学数学系首席炼金术师」气质，用木炭鼓风把群星带出的**希望币**（镀银/银铅合金或镀金颗粒）炼成可卖银铺的银/金，解决 2002 生活费；对应 C 级任务「灰吹法，牛刀小试」气质。撞大运、感染、进入绿洲——气运与风险同步抬升。阶段因果：游戏货币≠现实购买力，必须有**技术变现**；林叶成为校园线最稳补习与情感接口；感染危机把「心声能听丧尸」升级为「自己也可能变丧尸」。

**阶段五 · 进入绿洲 · 姜琳与老黄遗产（三阶核心）**

初始事件：雨夜里，裂爪兽的结晶还在背包里发烫；有人请你喝酒祝贺「突破」，酒过三巡才发现席上另外两个新人已经变成「考核耗材」——养蛊开始了。
开场白建议：「练力练到抱罡的人，笑起来比丧尸干净。他们说你是天才，说结晶认主，说只要咬死同桌就能进内门。你摸到杯底的残渣时，心声里响起第三个人的盘算：今晚死的，最好是你。」
关键NPC立场：**林池鱼**——用萃取叠技能、拒当枪，可做反养蛊同盟；修行者引路人——要苗子也要祭品；**姜琳**——体制内强者，可能插手也可能冷眼；雨夜狂魔级个体——猎杀或试炼。
主线钩子/支线：破养蛊局；裂爪结晶归属战；龙力/高阶技能卡情报；支线＝生命合剂配方交换、雨夜护送、给「健仁」加油式的炮灰救赎。
危险度/规避：极高——修行者内斗与精英兽；须规避无编制单挑狂魔，须规避把结晶当街拍卖。
任务方向/奖励：站住修行者门槛或干净退出；奖励取向＝裂爪结晶加工件、高阶技能卡、反侦察心声训练、四阶向护具。

**五阶（绿洲主力 · 贪欲战旗执勤线）**
切入身份/时点：以「巡查小队骨干/外援尖刀」切入，锚定休·克顿寄生审讯、贪欲战旗掉落前后，林池鱼已能指挥车载火力与腐蚀喷雾。
初始事件：审讯室电刑椅上的男人流利背出母亲生日与班长名字，下一秒胸腔炸开血藤；小地图红点炸成一片，江岸在喊「未知寄生」，有人把一面还在滴经验光的蓝旗塞进你手里——光环已经亮了。
开场白建议：「工作的时候叫职务，活下去的时候叫运气。绿洲外城的雨刮器刮不干净血雾，车载机枪的弹链比口号好听。你看见那面贪欲战旗在队友背后展开时，十米内所有人都像被同一只手推着往前——包括敌人的瞄准。」
关键NPC立场：**林池鱼**——持旗指挥者，气运与编制双在线；**江岸**——执行与恐慌的士兵接口；研究所收样人——要活体残肢；南北站观察员——要战果归属。
主线钩子/支线：精英蓝 BOSS 清剿；战旗归属与光环编队；红点预警网建设；支线＝寄生样本走私、执勤区风雨夜救援、职务冒名。
危险度/规避：极高——蓝 BOSS 与未知寄生链；须规避无光环硬抗腐蚀喷液，须规避战旗暴露引发裁决者/派系哄抢。
任务方向/奖励：正式编制与贡献跃迁；奖励取向＝精良蓝装、战旗/图腾光环件、小队指挥权、五阶向生命合剂配额。

**六阶（切断信号源线 · 气运反收割）**
切入身份/时点：以「逼近群星通关节点的高阶玩家/双线身份已暴露边缘的契约者」切入，锚定连载远景——信号源核心、日冕侧布局与「天选被瞄准」同时成立之时（**不编具体终局战果**）。
初始事件：你的面板与废土天空同时跳出同一枚红点；心声里出现非人语言的定位噪声，有人出天价买你的「运气」。虫洞冷却被未知规则拉长，永不挂科的学期成绩单与通关进度条在视野里重叠。
开场白建议：「天选之子的另一种写法，是被选中开炮。信号源在网络与废土底下同时跳动，像一颗不属于 2002 也不属于 2032 的心脏。你知道通关能切断它——也知道切断之前，所有眼红你爆率的人，都会先切断你。」
关键NPC立场：**林池鱼**——同路或竞速通关者，气运最亮也最危险；日冕棋子/代理人——要信号不断；裁决者高阶——收割天选；陈曦/林叶——现实软肋，可被挟持。
主线钩子/支线：信号源阶段节点争夺；反气运收割；双线人质交换；支线＝冠军礼包真相、时间线闭环证据、白神级节点情报（仅线索）。
危险度/规避：贴近顶点——**超阶巅峰压力：日冕/信号源＝存在·情报优先/条件性胜利**，严禁写路边斩杀日冕本体，严禁「信号源自己坏了」的廉价收场。
任务方向/奖励：通关进度与切断权限碎片；奖励取向＝六阶向规则级道具（限时）、稳定虫洞、气运伪装、现实侧永久身份与地球存续相关成就（表述保持开放，不写死结局）。
