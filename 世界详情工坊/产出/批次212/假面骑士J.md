## 剧情

**【作品来源】**
《假面骑士J》（仮面ライダーJ）是东映、东映视频与万代合作、石之森章太郎原作的特摄剧场版，1994年4月16日作为东映超级英雄嘉年华作品上映，片长约46～47分钟。导演雨宫庆太，编剧上原正三，音乐川村荣二，主演望月祐多（瀬川耕司）、野村佑香（木村加那）；J皮套演员冈元次郎。卡赖（ガライ）由神威杏次饰演，祖（ズー）由万里洋子饰演，亚基度（アギト）由栗原敏饰演；贝利（ベリー）声优相川梨绘；雾之母体声优丸山真帆（片中记名かやままりほ）。主题歌《心つなぐ愛》、插曲《Just One Love》由 BYUE 演唱。本片承接前一年《假面骑士ZO》，常被视为昭和骑士系谱末段剧场作，亦是石之森章太郎生前参与的最后相关故事之一。文风：环境公害调查外壳 + 外星寄生母体末日威胁；骑士可巨大化至约40米与母舰级敌人对决。特别篇有《假面骑士世界》《ZO & J VS 影月》等联动；Decade 等平成作品中亦有客串。美国版《Masked Rider》曾借用本片怪人素材，勿与本世界观混写。

**【世界定位】**
近现代日本。世纪末全球异常气象频发，实为宇宙怪物集团「雾（フォッグ / Fog）」再度来袭的前兆：其女帝「机械兽母舰·雾之母体（フォッグ・マザー）」曾以大孵化参与恐龙灭绝，约七千五百万年后回归，欲以人类为祭品哺育新一窝幼兽并吞食全人类。热爱自然的摄影师／环境记者瀬川耕司（26岁）在污染湖畔为救幼友木村加那挺身，被雾干部亚基度摔下悬崖身亡；地中种族「地空人」与大地精灵认可其勇气与护地球之心，以复苏改造手术注入「J力量（Jパワー）」并授予辉石「J精神（Jスピリット）」，令其作为假面骑士J复活。在知性蝗虫使者贝利引导下，J击破三干部与母体，救出加那。一句话：被复活的自然守护骑士，对抗雾之母体的祭品猎杀与周期灭绝。

**【世界观 · 力量体系】**
力量来源分两极。**守护侧**：地空人／大地精灵赋予的 **J力量**——寄宿于大地的精灵能量；变身核心为埋于腰带／腹部的红色辉石 **J精神**。变身者展露辉石、集中意识吸收自然之力，肉体与装甲融合；战斗意志消失可解除变身。等身形态约身高194cm、体重84kg、跳跃力约150m，具备 J 天线（感知约5km）、J 眼（雾中视约1km）、J 粉碎器式防毒呼吸与强化肌骨，必杀包括 **J Kick**、J Punch、J Chop、J Elbow；可结 **J Sign** 聚能。日常摩托注入 J 力量化为 **J穿越者（ジェイクロッサー / J-Crosser）**，耐约1000℃高温，极速设定向超千公里级，可空中战与 **J Strike**。危急时，地空人、贝利与大地沉睡精灵集中能量，J 可进入 **巨大形态（ジャンボフォーメーション / Jumbo Formation）** 约40m／约736t，必杀 **Jumbo Rider Kick / Punch**——原作更近「奇迹灌注」而非无限连发技能。

**侵略侧**：**雾** 为外星寄生／机械兽混合文明。雾之母体本体即金色机械兽母舰（约70m／12000t），兼母舰与女王，体内携带数千卵；约每千年完成仪式后大孵化，幼兽吞食行星生命。她喷吐黄雾枯死植被、污染水体，制造磁异常干扰气象与重力；武装含弹幕、翼剪带电、眼部光线、反重力射线、飞行切割器、钻头、加特林、钳臂与虫肢等；「神体」面庞可发号令，能将敌人吞入消化腔。三名最长子嗣可拟人并现真身：亚基度＝蜥蜴男（毒牙、束缚尾）；祖＝蜂女（翼、手针、赤红丝带）；卡赖＝眼镜蛇男／雾之王子（激光剑、拘束鞭、爪球），主持祭品仪式。

无传统修真境界。战力按威胁与破坏力划分（宁低勿高）：
1. **常人／摄影师／孩童／警员**：无超凡战力。≈一阶。
2. **初期变身J／三干部人类态或基础怪形态**：碎岩、高速战、毒牙／针／鞭级超凡。≈二阶。
3. **稳定J＋J-Crosser协同／三干部完全怪形态**：城镇级冲突，可击杀单体干部。≈三阶。
4. （本世界库不覆盖四阶）
5. **J 巨大形态／雾之母体子嗣孵化期攻势**：约40米级对战，区域灾难。≈五阶。
6. **雾之母体本体（机械兽母舰）＋新一窝苏醒意图**：种族灭绝级顶点；条件性胜利靠救出祭品、摧毁母体与新生幼兽。≈六阶。

乐园阶位映射：常人≈一阶；初期J／干部基础态≈二阶；稳定J／干部完全态≈三阶；巨大化J／孵化攻势≈五阶；雾之母体本体≈六阶。顶点＝雾之母体体系，**超阶边缘按六阶封顶写条件性胜利**，严禁「战力归零式削弱」或「被封印所以为零」。

补充战力表现：常人遇干部即死；初期J可再战亚基度；稳定J可逐破祖与卡赖但仍需战术；巨大化具备街区～城市对决级破坏但依赖精灵能量灌注与伦理代价；母体武装可犁城，胜利条件永远是「加那存活＋母体崩毁＋J脱出」，不是契约者单刷母舰。

**【地理 · 舞台】**
**污染山间湖／休闲地开发工地**：瀬川耕司调查开发排污与死鱼油膜的起点；加那被盯上、耕司被杀的第一现场。芦苇荡、泥岸鞋印、异常雾墙是一阶可观察锚。**悬崖与谷底**：亚基度摔死耕司处；亦是地空人发现遗体、实施复苏改造的入口附近。**地空人领域／地中通道**：不耐强光、下身与大地相连的种族栖所；J力量注入与J精神授予的仪式感空间，契约者通常只能接触外围感应而非核心宫殿。**雾域公路与雾墙**：黄雾枯树、磁异常致罗盘失灵；干部拟人态出入的走廊。**雾之母体要塞内部**：神体面庞所在的神殿式指挥核、祭坛、孵化室（加那被投放处）、消化腔通道、卵囊舱。**邻近都市街区**：母体攻城舞台——激光切开楼顶、疏散人潮、军方临时射击线。**J-Crosser 骑行道**：山道、高架、可空中跃迁的追击线。**战后平静草地／湖畔余韵**：耕司抱加那脱险、贝利远观之处。写场景强调：潮腥雾气、金属与土壤的腥甜、儿童哭声突然整齐如循环播放、巨大化后城市比例反转。

补充地理细目：湖区开发宣传牌与死鱼对照构成环境主题视觉；悬崖栏杆断裂是警方「事故」立案的伪证；地中苔藓通道在月光下会反光；要塞外壁金色反光可从数公里外误认「第二轮落日」；孵化室温度与湿度高于舰内其它区；消化腔有节律收缩，被吞者需在节律谷底找破口；变电所与医院是攻城次级受灾点；媒体直播车停在安全线外却最先被雾淹没。

**【世界剧情线】**

① **异常气象与雾的预兆**  
世纪末各地异常气象、植被枯死、水体污浊。公开叙事仍是「公害／厄尔尼诺」；私下，雾之母体已完成祈祷仪式，命最长子嗣卡赖与干部祖、亚基度外出猎取理想人类祭品，供未孵化弟妹食用，继而灭绝人类。恐龙灭绝被暗示为旧一窝大孵化的历史后果——周期猎食文明，非一次性入侵。

② **湖畔调查与祭品锁定**  
瀬川耕司以自然摄影师／环境记者身份调查休闲地开发导致的山间湖污染，遇见同样热爱自然、会为公害死去的动物建墓的少女木村加那。三干部锁定加那为理想祭品：幼弱、无武装、与守护者情感连接可最大化绝望。世界状态：普通人被猎杀，无骑士介入前几乎无解。

③ **死亡与掳走**  
耕司保护加那逃亡时，爬虫系干部亚基度（人形态为粗暴中年男性）将其摔下悬崖杀死；虫系干部祖（人形态为年轻女性）掳走加那。脚印被雾抹平，监控只剩白雾，警方立失踪／意外案。死亡是复活线的硬前提，不可用「假死」糊弄。

④ **地空人复活·假面骑士J**  
地中友好种族「地空人」发现耕司遗体，认可其勇气、正义感与护地球之心，施行复苏改造手术，注入大地精灵能量 **J力量**，授予神秘辉石 **J精神**，令其成为大自然的战士 **假面骑士J**。知性蝗虫 **贝利** 任使者与向导。雾之母体感应到「死人」仍活动，派亚基度斩草除根。

⑤ **击破亚基度与祖**  
J 与亚基度再战，亚基度现出蜥蜴男真身（毒牙、束缚尾），被 J 以 Rider Punch 等击杀坠崖。祖为兄复仇，以蜂女真身（翼、手针、赤红丝带）缠斗，最终与 J 冲撞雾之母体舰体而受致命伤，于卡赖面前消散入雾。此阶段确立：单体干部可破，母体仍远。

⑥ **卡赖仪式·加那入孵化室·攻城**  
卡赖以青年男性姿完成祭品仪式，将受精神控制的加那送入孵化室；真身为眼镜蛇男，持激光剑、拘束鞭与爪球。贝利介入试图解除加那的控制，被卡赖击倒。加那坠入死境，雾之母体开始攻击邻近都市，为幼兽苏醒清场。哺育倒计时与城市灾难并行。

⑦ **杀卡赖·被吞·杀新生·巨大化终局**  
J 登舰击杀卡赖，却发现母舰本身即雾之母体的一部分，被消化系统吞入。他吸收周围生命能量，杀死新生幼兽，脱出后在地空人、贝利与大地精灵的能量结集下展开 **Jumbo Formation**，从母体中救出加那，以 Jumbo Rider Kick／Punch 摧毁雾之母体。母体自爆拖拽时，J 完成最终一击并脱出。耕司带着加那逃至平静之地，贝利远距离守望。结局方向：本轮灭绝危机解除，祭品获救，代理人完成托付；不写母体立刻刷新续命。

⑧ **主题与可介入日常**  
主题：环境与生命、祭品伦理、复活者的自我、自然之力对抗机械寄生、巨大化奇迹的伦理代价。可介入：污染湖调查、雾天失踪案、阻止祭品仪式、要塞潜入、城市疏散、巨大化能量锚、残骸防复苏、公开真相与否。特别篇联动（与 ZO 对影月等）不改写本片主线终局。

**【世界剧情线·分幕细目】**

**幕一：雾与公害之间**  
耕司的镜头与水质试纸把故事锚在「看得见的污染」与「看不见的外星猎手」之间。湖面油膜、死鱼、异常雾墙是一阶可观察异常。三干部先以人类姿接近：卡赖冷峻如祭司，祖妩媚而迅捷，亚基度粗暴如中年打手。祈祷仪式是猎杀前的程序口令，不可当成可谈判的外交。开发商宣传牌与枯树同框，环境主题不是口号，是现场气味。

**幕二：加那被选中**  
木村加那会为公害死去的动物堆石立墓——这份「自然之爱」在雾的标准里被读成「理想祭品」的纯度。掳走过程强调速度与绝望：普通人无法对抗。耕司之死必须发生，否则 J 力量复活线不成立；契约者若在此点介入，最多延缓，难以同时阻止死亡与掳走。鞋印被雾抹平是标志性感官：证据链断裂。

**幕三：地空人的选择**  
地空人不是人类政府，复活带有「地球代理人」逻辑。他们不耐强光、与大地相连，科技与精灵术混合。贝利是沟通接口：提供情报、提醒母体弱点、在关键时刻挡刀，也会用试炼语气考验协力者。J 精神是变身核心；J Kick 是必杀。巨大化需精灵能量结集与周围生命力——有伦理代价，不可无限制使用。

**幕四：三干部逐破**  
亚基度＝毒牙与束缚尾的近战刽子手，擅用地形死角；祖＝飞行、毒针、丝带拘束的空中猎手，擅高空抛落；卡赖＝仪式主持与最强单体，眼镜蛇男武装完整。击破顺序影响加那存活窗口：拖得越久，孵化室越接近喂食时刻。祖在卡赖面前消散的悲剧感服务主题，不提供感化选项。

**幕五：母体与城市**  
雾之母体是金色机械兽母舰：加特林、眼部激光、圆锯／翼剪、钳臂、虫肢、反重力与飞行切割。她能感知机体内外攻击，损伤即痛。城市攻击是种族灭绝准备，不是无意义破坏。巨大化 J 约40米，与母体约70米形成比例对决。疏散与误射风险同步上升。

**幕六：救出与爆炸**  
救出加那是胜负手：只炸母体不救人＝失败。母体自爆拖拽 J 时，需完成最终一击并脱离。战后耕司恢复人类态，抱加那至平静之地，贝利远观——开放式余韵，代理人完成托付，世界暂回日常，雾的记忆仍在。

**幕七：因果总表**  
异常雾／公害表象 → 湖畔调查 → 猎祭品 → 耕司死／加那掳 → 地空人复活 J → 杀亚基度 → 杀祖 → 卡赖仪式 → 加那入孵化室 → 城市攻击 → 杀卡赖 → 被消化 → 杀新生幼兽 → 巨大化 → 救出加那 → 摧毁母体 → 逃出爆炸 → 贝利守望。

**幕八：人物关系网**  
耕司对加那是守护与愧疚双轴；对地空人是受命代理人；对贝利是战友与被引导者。卡赖—祖—亚基度是哺育共同体而非可感化的人类家庭。雾之母体视子嗣高于一切行星生命。人类社会（警员、市民、媒体）无知且脆弱，却构成道德压力源。任何「牺牲加那换毁舰」方案都违背本片主题。

**幕九：战斗规则与可操作细节**  
J 等身战：超跳、必杀踢、摩托协同；可被母体吞入。亚基度破法：拉开距离、破坏落点、要害窗口。祖破法：逼降、切断丝带、利用舰体碰撞风险。卡赖破法：打断祭坛法阵、保护贝利、在加那入室前抢时间。母体破法：救人优先于毁舰，巨大化落点避开居民区。黑暗／雾中 J 眼有效；黄雾对植被与水源是慢性杀伤。

**幕十：结局后余波（可作支线）**  
母体残骸若被军方或黑市回收，可能催生「第二降临」研究；耕司是否公开复活真相决定社会恐慌；加那的雾中心理阴影可作长期支线；地空人不会发奖章，贝利的远观即是认可。与 ZO 的互文仅作系列背景，勿把基因改造「完全生命」线写进本片主线力量源。

**【主要人物】**

**【巨大化伦理与战后处理】**
Jumbo Formation 在原作是地空人、贝利与大地精灵的能量奇迹，档案中允许写「周围生命力被牵引」的可观察代价：疏散人群短暂虚脱、伤员监护仪报警、协力者眼前发黑。契约者五～六阶核心抉择不是「能不能巨大化」，而是「允许抽多少生命去完成一脚」。救出加那是硬胜利条件。残骸监测、公开真相、加那心理修复是战后三条合法支线。禁止把母体写成「下周再来一艘」的刷怪器；周期威胁用长期阴影表现，不用立刻刷新 BOSS。

**【对话气质（供开场）】**

**【与系列边界】**

**【物品与情报经济】**
本世界最值钱的不是「万能腰带复制品」，而是：黄雾与非地球蛋白样本、祈祷韵律录音、祭坛／孵化室结构、母体武装模块弱点、贝利方位苔藓、地空人感应点坐标、残骸防复苏数据。低阶奖励＝证据与生存工具；中阶＝干部样本与共乘许可；高阶＝残能与结构情报。禁止低阶发放完整 J 精神或稳定巨大化权限。持有幼兽组织或母体核心碎片会同时吸引军方、黑市与残余雾感应——持有即剧情钩子。

**【人物弧光再补（真名）】**
瀬川耕司从「用镜头指控公害的人」变成「被地球选中的兵器」，正义感仍在，表达压缩成短句与必杀；他对加那的愧疚是持续动力，不是恋爱线强行升级。木村加那的「为动物建墓」既是性格也是被选原因，获救后需处理雾中控制的记忆残留。贝利在挡刀前后完成从考官到战友的位移，战后选择远观而非居功。卡赖把哺育程序执行到死；祖的消散带给卡赖的不是眼泪而是加速仪式；亚基度两度行刑失败定义「死人」威胁等级。雾之母体的台词核心永远是下一窝与吞食，不提供改邪归正分支。地空人保持沉默雇主位：给力量、给奇迹、不给人类勋章。

**【终局执行节奏（不可乱序）】**
卡赖已死 → J 确认母舰即母体 → 被吞入消化腔 → 腔内击杀新生幼兽 → 地空人／贝利／精灵能量结集 → Jumbo Formation → 从母体拔出木村加那 → Jumbo Rider Kick／Punch 毁舰 → 自爆拖拽中脱出 → 抱加那至平静之地 → 贝利远观。契约者合法席位：能量锚、疏散、外侧爆破通道、误射劝阻、残骸监测、落地接应——**不可抢最终一击代替 J**。任何打乱「先救人后毁舰」的写法都会把本片改成无主题的怪兽对轰。

## 阶位切入点

> 阶位↔强度：一阶≈常人／湖畔调查线；二阶≈初期J／干部基础态；三阶≈稳定J／干部完全态；五阶≈巨大化J／孵化攻势；六阶≈雾之母体本体。顶点规避硬刚，破局看救出木村加那与摧毁母体。本世界不覆盖四阶，禁止写入四阶段落。

**一阶（常人 · 湖畔调查线）**
切入身份/时点：契约者以「环境周刊实习记者／当地向导／开发工地临时测绘员」切入，锚定怪异雾初现、三干部尚未公开猎杀或刚完成祈祷。
初始事件：山间湖面漂着死鱼与油膜，浓雾里传来祈祷般的低语；木村加那的鞋印在泥地里突然中断，瀬川耕司抓着你的录音笔：「这不是公害——有东西在选人。」
开场白建议：「雾比天气预报更早到来。录音笔里只剩下小孩子的笑声被掐断的一截。瀬川耕司把你拉进芦苇荡：别出声，他们在念某种……菜单。」
关键NPC立场：**瀬川耕司**——需要证人与助手，尚未死亡；**木村加那**——潜在祭品，信任伸手的人；**亚基度（人形态）**——已在附近猎杀；**祖（人形态）**——接近孩童；**当地警员**——只当失踪／污染投诉案。
主线钩子/支线：完成污染取样却发现非地球蛋白；护送加那离开湖区；辨认雾中人影与假游客。支线：开发商封口、失踪儿童海报、异常体温的「游客」、罗盘失灵带。
危险度/规避：中——常人遇干部即死；规避夜间单独对峙，优先撤离孩童与切断仪式目击链。
任务方向/奖励：加那未当场被掳或争取到报警／转移窗口；奖励取向＝黄雾样本、仪式录音残片、地空人传说剪报、水质异常报告。

**二阶（复活夜 · 初次变身线）**
切入身份/时点：以「目睹复活的目击者／地空人外围感应接触者／悬崖事故『唯一生还证人』」切入，锚定耕司刚被复活为 J、亚基度奉命再杀。
初始事件：悬崖下，死去的摄影师睁眼，甲壳在月光下合拢；贝利落在你肩上：「带他去找那个孩子——雾之母体已经感应到他了。」
开场白建议：「泥土从他指缝里落下，却不再是死人的温度。假面骑士J的声音沙哑：加那在哪。草蜢向导贝利的复眼映着你：回答错了，下一个祭品就是你。」
关键NPC立场：**瀬川耕司／J**——目标单一（救加那）；**贝利**——情报与试炼；**亚基度**——必杀回归者；**雾之母体**——远程施压；**卡赖**——尚未正面，只催促仪式。
主线钩子/支线：首次 J Kick 外围掩护；追踪祖掳人路线；学习「雾为信号」的预警。支线：警方误认 J 为怪物、媒体直播陷阱、地中通道入口标记。
危险度/规避：中高——干部可杀调查者；规避卡赖本体与母体内部消化腔。
任务方向/奖励：确认 J 友军；奖励取向＝抗毒口罩、雾预警标记、J 信任标记、有限抗针剂。

**三阶（三干部战 · 城镇／要塞外围）**
切入身份/时点：以「J 认可的协力者／J-Crosser 临时共乘向导」切入，锚定亚基度已灭、祖复仇战前后，或卡赖仪式前夜。
初始事件：蜂女的毒针钉穿路灯，丝带卷起加那的头发丝——她已被送到要塞边缘；卡赖的声音从雾墙后传来：仪式开始了。
开场白建议：「针落在柏油路上还在冒烟。假面骑士J跨上J穿越者，轮胎在雾里烧出两道焦痕：挡住祖三十秒。——你听见要塞深处有孩子在哭，哭声突然变得很整齐，像被按了播放键。」
关键NPC立场：**J**——前线；**祖**——复仇与掳护；**卡赖**——仪式优先；**贝利**——弱点提示；**木村加那**——半被精神控制。
主线钩子/支线：祖空战外围；破坏仪式法阵一角；夺回加那失败后的二次潜入准备。支线：城市开始出现雾墙、居民昏迷、金色舰影被误报为「不明飞行物」。
危险度/规避：高——完全态干部；规避母体消化腔与巨大化前的蛮干登舰。
任务方向/奖励：击破或重创一名干部、标记要塞入口；奖励取向＝干部组织样本、要塞地图残片、J-Crosser 临时共乘许可、祭坛结构草图。

**五阶（巨大化 · 孵化与攻城线）**
切入身份/时点：以「城市疏散指挥协力／生命能量伦理决策者／地空人临时候补锚点」切入，锚定加那已入孵化室、母体攻击都市、J 准备或已进入 Jumbo Formation。
初始事件：四十米的影子覆盖街道，母体的激光切开楼顶；贝利告诉你：巨大化要吸走周围生命力——包括伤员与你自己的一部分。
开场白建议：「警报器与雾鸣叠在一起。假面骑士J的轮廓突然高过塔吊，城市在他脚踝边像玩具。——木村加那还在那艘金色母舰的肚子里，而你必须决定：让他吸走多少『生命』，才够一脚踢碎灭绝。」
关键NPC立场：**巨大化J**——需要窗口与方向指引；**雾之母体**——攻城与消化并行；**新生幼兽（孵化中）**——哺育倒计时；**贝利**——能量与结构提示；**市民／伤员**——道德压力源；**军方临时单位**——可能对 J 误射。
主线钩子/支线：引导 Jumbo Kick 落点；从孵化室外侧爆破救人通道；阻止军方误射；自愿能量锚与否。支线：医院断电、二次雾墙、祭品替身谣言、残骸坠落带。
危险度/规避：极高——区域灾难；**禁止与母体本体持久硬刚**，优先救人＋弱点模块打击＋控制落点。
任务方向/奖励：加那被移出孵化室或母体武装被毁一门；奖励取向＝巨大化残能样本、城市疏散功劳、地空人认可、幼兽组织封存罐。

**六阶（母体终局 · 灭绝阻止线）**
切入身份/时点：以「终局见证者／地空人临时代理人／加那落地接应者」切入，锚定卡赖已死、J 被消化后脱出并巨大化决战。
初始事件：母体钳臂抓住巨大化 J，自爆程序启动；加那在你与 J 之间被抛出——你接不住她，城市就接住她的坠落。
开场白建议：「金色的母舰在夜空里像一颗要落地的太阳。假面骑士J的拳从她胸腔里拔出那个孩子，雾之母体的声音却仍在说：下一窝，还会来。——你抱紧木村加那，看骑士的身影被爆炸吞没又冲出：这不是奥特曼童话，是地球把代理人赌在一个人身上。」
关键NPC立场：**雾之母体**——顶点，灭绝逻辑不可教化；**J**——条件性胜利执行者；**木村加那**——必须存活的胜负手；**贝利**——战后守望；**地空人**——力量源，不直接下场；**军方／媒体**——真相与误射双刃。
主线钩子/支线：确保加那安全落地；为最终 Kick 清理障碍；记录母体残骸防止二次复苏；决定是否公开复活与外星真相。支线：耕司身份暴露后的社会处置、加那心理修复、黑市残骸争夺。
危险度/规避：贴近顶点——**超阶边缘：存在·情报优先／条件性胜利**；胜利＝救加那＋毁母体＋J 脱出，而非契约者单刷母舰。禁止「封印削弱归零」。
任务方向/奖励：本轮灭绝危机解除；奖励取向＝J 力量残片级信物、雾之母体结构情报、系列「地球代理人」声望（不发完整母体核心／完整巨大化权限）。
