# 使命召唤：二战
<!--meta lib=主库 tiers=一、二-->

## 剧情

**【作品来源】**
《使命召唤：二战》（*Call of Duty: WWII*）由 **Sledgehammer Games** 开发、**Activision** 发行，2017 年 11 月 3 日登陆 PS4 / Xbox One / Windows。系列第十四部主作，亦是自 2008 年《世界战争》后首度全面回归二战欧陆战场的正作。文风：靴子落地式步兵战、小队互助、战役写实与战争暴行并陈；多人引入师部（Divisions）与总部（Headquarters）社交枢纽；合作模式 **Nazi Zombies** 另有 Geistkraft 科幻线。关联：同厂《高级战争》；系列僵尸谱系与 Treyarch / Infinity Ward 僵尸宇宙独立。搜笔趣阁无稳定长篇小说书页（游戏 IP），本档案以维基与可核战役流程为骨。

**【世界定位】**
1944–1945 年西线欧洲：玩家以美国陆军第 1 步兵师（大红一）**二等兵 Ronald "Red" Daniels** 为主视角，随排经历诺曼底登陆、眼镜蛇行动、解放巴黎、亚琛、许特根森林、突出部战役、雷马根大桥直至深入德国搜救战友。一句话：从奥马哈海滩泥沙到贝尔加集中营铁丝网，兄弟情与命令之间不断撕裂。顶点是装甲、火炮与集中营体系——个人仍可被一发子弹放倒。

**【世界观 · 力量体系】**
本世界**无超自然修炼**（战役主线）。战力锚在：人体、二战枪械与爆炸物、装甲车辆、空中支援、小队职能补给。战役取消自动回血：须拾取医疗包或依赖卫生兵队友；弹药/手雷由补给兵补充；可呼叫迫击炮与标记敌人轮廓。死亡写实。

**层级（由低到高）：**
1. **平民 / 被疏散者 / 俘虏**：几乎无武装。
2. **新兵与常规步兵**：M1 加兰德、卡宾枪、冲锋枪；可清散兵与机枪巢。
3. **老兵排级（Daniels 小队成熟期）**：协同破障、俘敌审讯、拖救伤员、坦克协同。
4. **装甲与空中资产**：**M4 Sherman（Staff Sergeant Augustine Pérez 的 745 坦克营）**、**P-47 Thunderbolt（Matthew Weber）**——改变局部战场，仍惧反坦克炮与防空。
5. **德军精锐与重装**：机枪阵地、**虎 II（虎王）**、装甲列车、V-2 运载、盖世太保网络。
6. **制度性顶点**：集中营与死亡行军体系、德军总退却与莱茵防线——属**组织与工业杀戮顶点**，非个人超人。

**乐园阶位映射（对照阶位战力图鉴·宁低勿高）**：新兵/后勤/被困平民≈**一阶**；一线步兵排与坦克协同、空中对地≈**二阶**（可清街区、打穿加强连级防御；惧集火、炮火与重装甲正面）。世界顶点＝战争机器与集中营工业，**个人火力封顶二阶**。严禁写成现代特种无伤或魔法回血。

**【地理 · 舞台】**
- **奥马哈海滩（Omaha Beach）**：诺曼底登陆血门；后期多人总部社交空间亦取解放后三日景象。
- **马里尼（Marigny）与诺曼底内陆**：眼镜蛇行动解放乡镇。
- **巴黎**：盖世太保驻地、抵抗组织接头；**Camille "Rousseau" Denis** 潜入线。
- **亚琛（Aachen）**：酒店地下室平民、撤退卡车与火力覆盖。
- **许特根森林 493 高地**：虎王、过早突击与 **Joseph Turner** 牺牲点。
- **突出部 / 阿登**：包围、非裔工兵 **Marcus Howard** 协助呼叫空中支援。
- **雷马根 / 鲁登道夫桥**：最后一座莱茵桥争夺。
- **贝尔加（Berga）集中营与死亡行军路线**：Zussman 被囚与搜救终章。
- **德州家乡**：终章归乡与兄长 Paul 墓碑。

**【世界剧情线】**
**① 1944-06-06 奥马哈**
PFC **Ronald "Red" Daniels** 随第 16 步兵团登陆。小队核心：**Robert Zussman**、**Frank Aiello**、**Drew Stiles**、技术军士 **William Pierson**、中尉 **Joseph Turner**。Zussman 被德军刺伤，Daniels 拖其至掩体——兄弟情主轴启动。

**② 眼镜蛇行动（7 月）**
协同 745 坦克营解放马里尼。Daniels 得知 Pierson 曾因卡塞林山口战役鲁莽导致部下死亡而被降职——埋下「命令 vs 救人」冲突。

**③ V-2 装甲列车与巴黎（8 月）**
上校 **Davis** 派小队与英军 SOE **Arthur Crowley**、**Vivian Harris** 及法抗 **Rousseau** 拦截运载 V-2 的德军装甲列车。Rousseau 潜入巴黎驻地取炸药，识破盖世太保 **Heinrich** 后将其刺杀，为解放巴黎打开突破口。

**④ 亚琛（10 月）**
酒店解放后发现地下室平民。Turner 坚持疏散，Pierson 反对。Daniels 救出女孩 **Anna**，卡车遭射击，Anna 姐姐 Erica 身亡；Pierson 令卡车无护撤离——道德裂痕扩大。

**⑤ 许特根 493 高地（11 月）**
Pierson 过早突击。摧毁火炮后遭虎王反击；Daniels 负伤，**Turner 牺牲掩护撤离**。Pierson 接管排，提拔 Daniels 为下士。

**⑥ 突出部战役（12 月）**
被围；**Marcus Howard** 协助联络空中支援。玩家短暂操控 **Matthew Weber** 的 P-47。审讯俘虏获悉德军计划炸毁雷马根桥。

**⑦ 雷马根失败与 Zussman 被俘**
摧毁在运炸药后，空袭基地失败，**Zussman 被俘**。Daniels 违令追卡车负伤，被 Pierson 开除出排。Daniels 获铜星却拒绝回家，坚持救友；对质后得知 Pierson 降职真相是曾抗命试图救部下——二人和解，Daniels 归队。

**⑧ 莱茵与贝尔加（1945-03）**
夺取最后大桥，深入德国搜到废弃贝尔加，沿死亡行军路线救出 Zussman。欧战结束后 Daniels 归乡见妻与新生儿，将勋章放在兄长 **Paul** 墓上（童年猎狼失手的创伤闭环）。

**⑨ 多人与僵尸支线（世界可并行态）**
师部系统、战争模式（War）、总部枢纽构成「永续二战沙盘」。Nazi Zombies 线：**Marie Fischer** 等对抗 **Peter Straub** 的 Geistkraft 亡灵军（与战役正史平行，可作限时灾变，勿与 Daniels 线混为同一物理因果）。

**结局方向**：战役闭环为「活着回家但仍背负死者」；战争本身无终局。乐园宜截取「单场战役目标 + 小队伦理抉择」，勿宣称凭一人之力结束二战。

**【主要人物】**
- **Ronald "Red" Daniels（主角）**｜性格：固执、重诺、德州泥土气息｜装备·能力：M1 等步兵武器、拖救与小队协同｜弧光：为兄长阴影入伍→违令救友→归乡放勋章｜立场：盟军，对 Pierson 从恨到理解。
- **Robert Zussman**｜性格：忠诚、幽默压恐惧｜弧光：登陆重伤→再上战场→集中营幸存者｜立场：Daniels 最铁兄弟。
- **Joseph Turner**｜性格：原则、护兵｜弧光：带队→493 高地牺牲｜立场：人道优先于战果数字。
- **William Pierson**｜性格：酗酒、刻薄、隐藏创伤｜弧光：卡塞林污名→逼死 Turner 阴影→承认曾抗命救人｜立场：要胜，也怕再失去人。
- **Frank Aiello / Drew Stiles**｜性格：务实/年轻｜弧光：排骨血肉｜立场：听从与吐槽并存。
- **Camille "Rousseau" Denis**｜性格：冷静致命｜能力：伪装潜入、刺杀 Heinrich｜立场：法抗，可结盟。
- **Arthur Crowley / Vivian Harris**｜SOE 情报与爆破｜立场：任务优先。
- **Augustine Pérez**｜坦克长｜立场：火力支援可靠。
- **Marcus Howard**｜非裔工兵｜弧光：突出部关键通讯｜立场：专业，对抗种族隔离现实。
- **Davis 上校**｜命令源头｜立场：把人当棋也给人铜星。
- **Anna**｜亚琛平民女孩｜立场：战争代价的人脸。
- **Heinrich**｜盖世太保｜立场：绝对敌对。

**【势力图谱】**
- **美军第 1 步兵师 / 16 团与 745 坦克营**：主视角编制。
- **法国内政部队 / 抵抗组织**：巴黎线。
- **英军 SOE**：V-2 任务。
- **德军国防军、装甲部队、盖世太保**：战术到政治敌人。
- **集中营体系（贝尔加）**：工业罪恶顶点。
- **多人盟军/轴心沙盘**：玩法阵营，叙事需区分党卫与普通士兵争议（档案按原作呈现，不美化）。

**【贵重物品】**
- **医疗包与小队补给链**：战役规则级「命」。
- **V-2 火箭与运载装甲列车**：战略级目标，宜拦截/摧毁。
- **雷马根桥炸药**：决定莱茵进军速度。
- **铜星勋章**：Daniels 荣誉与愧疚的物化。
- **虎王坦克**：二阶战场霸权符号。
- **Paul 的墓与勋章安放**：情感奇物，非战力。

**【隐藏剧情 · 伏笔】**
- Pierson 卡塞林真相与「抗命救人」镜像 Daniels。
- 童年猎狼：Daniels 未能开枪导致 Paul 死——贯穿不敢再「晚一步」。
- 亚琛平民车：命令正确性的灰色。
- 集中营段落：系列少见的暴行直面，忌娱乐化处刑描写。
- 僵尸 Geistkraft 与 Barbarossa 之剑：平行宇宙能源神话。

**【大事记时间线】**
1944-06-06 奥马哈 → 07 眼镜蛇/马里尼 → 08 V-2 与巴黎 → 10 亚琛 → 11 许特根 Turner 死 → 12 突出部与情报 → 1945-03 雷马根与贝尔加救 Zussman → 欧战结束归乡。

**【叙事基调 · 雷区】**
基调：泥、血、电台噪音、兄弟骂街与沉默；写实枪感。忌：无限弹药无敌英雄；自动回血无解释；把集中营当刷怪副本娱乐；把 Pierson 写成单纯反派。最早切入锚点＝奥马哈登陆艇舱门落下前 3 分钟，或亚琛酒店地下室发现平民时。

**【阶段档案·登陆舟波与滩头经济学·补1】**
第一波和第二波的区别不是勇气评分，而是机枪射界是否被烟与尸体改写。契约者若在滩头站直射击，会先死于几何而非「命中率」。军医包的位置、障碍物铁蒺藜的剪断顺序、潮汐退去露出的通路，都是可观察细节。Daniels 拖人时的负重会让瞄准失效——正文应写出喘息与血沫，而非无敌冲锋。

**【阶段档案·小队职能即生存接口·补1】**
卫生兵、补给兵、火力标记与迫击炮请求构成战役 UI 的叙事化：你不是一个人在打全图，你是排机器上的一个齿轮。切断任意接口（Pierson 赌气不给支援、Turner 坚持救人导致火力分散）都会改变下一幕死亡率。乐园任务可把「谁拿到最后一个医疗包」做成道德检定。

**【阶段档案·坦克与步兵的彼此恐惧·补1】**
Sherman 在开阔地是移动掩体，在巷战是活靶。Pérez 的炮口指向决定步兵敢不敢过街；反坦克手与虎王的出现会瞬间把二阶任务压回「找掩体」。禁止写步兵手雷无脑秒虎王，除非原作关卡给出爆破点与协作动画。

**【阶段档案·抵抗组织的城市语法·补1】**
巴黎线的胜负在证件、下水道、德语口令与炸药引信，不在正面刚枪。Rousseau 的刺杀是信息战：Heinrich 死，驻防混乱，美军才有破门窗口。契约者若只会美式冲锋，会在盖世太保办事处触发全城封锁。

**【阶段档案·森林与冬季的感官·补1】**
许特根的树爆、碎木片、泥雪吸声与突然出现的虎王履带，是恐惧设计。突出部的寒冷让枪机卡住、手指失去扣扳机精度。AI 叙事应优先温度与能见度，其次才是击杀数。

**【阶段档案·俘虏与情报的灰色·补1】**
德军俘虏给出的雷马根情报是转折，但审讯手段与「是否相信」构成伦理。错误情报会把排带进埋伏。乐园可做「验谎失败」分支，而不是自动正确字幕。

**【阶段档案·种族隔离下的同一条战壕·补1】**
Marcus Howard 的专业在突出部救场，同时提醒：盟军内部并非平等神话。正文可写食堂、晋升与同行者的目光，避免把历史写成全员和谐海报。

**【阶段档案·集中营段落的叙事纪律·补1】**
贝尔加与死亡行军是见证而非通关彩蛋。重点在找到 Zussman、记录空营、理解工业杀人流程的冷，而不是战斗得分。契约者若以「刷掉所有卫兵」为乐，应遭到乐园任务评价惩罚（人性扣减）。

**【阶段档案·归乡不能治愈·补1】**
终章的新生儿与墓碑并置：活着的奖赏与死掉的兄长同时在场。勋章放上墓是私人仪式，不是国家颁奖蒙太奇。后续日常任务可以「写信」为形式，而非再开一场大战。

**【阶段档案·多人总部的记忆错位·补1】**
奥马哈被改造成社交枢纽时，战役记忆与娱乐空间叠合——契约者可能在曾流血的沙滩开箱。可用作元叙事：战争被消费，但 NPC 老兵仍会在潮汐声里沉默。
