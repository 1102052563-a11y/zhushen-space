# 对魔忍（系列通用）
<!--meta lib=主库 tiers=三-->

## 剧情

乐园阶位映射：本世界战力按破坏力对齐《阶位战力图鉴》，宁低勿高；详见【世界观 · 力量体系】。

**【作品来源】**
对魔忍（Taimanin / 対魔忍）系列由 Lilith（Black Lilith）出品，2005《对魔忍阿莎姬》起。含视觉小说、OVA、RPGX、Action Taimanin 等。策划笹山逸刀斎，原画含カガミ、葵渚等。**本档案只写战斗、组织、任务政治，不写色情场面**；改造/契约仅记战术后果。卡片锚点：五车任务简报——对魔忍即将潜入黄泉原类无法地带。

**【世界定位】**
近未来日本：人类与魔界暗通。对魔忍=魔族血脉忍者，清剿魔与人魔勾结。旗帜角色井河阿莎姬。一句话：国家弃子忍者对抗魔界与腐败资本。

**【世界观 · 力量体系】**
忍法（雷火风空水幻粒子等）+ 体术 + 触媒武器 + 魔科改造。一般人≈一；黑帮兽化≈二；正式对魔忍≈三（本档覆盖三阶）；精英阿莎姬级≈四五；上位魔≈六+超阶边缘。本世界切入仅三阶：正式任务忍水平。

**【地理】** 五车村/学院；东京地表；黄泉原地下都市；Under Eden 类据点；魔界门城远景。

**【世界剧情线】**
魔界入侵与犯罪企业化→政府建对魔忍→阿莎姬与朧宿命对抗→各作主角（雪风/紫/红等）执行潜入与歼灭→黄泉原为高频地狱舞台→改造与契约是敌方武器→真结局强调任务完成与代价残留。

**【主要人物】** 井河阿莎姬；井河樱；朧；水城雪风；秋山凜子；水城不知火；黑井龙司（梦魔王相关）；各作魔科医与干部。

**【势力】** 五车；政府；诺玛德等企业；魔界贵族；黄泉原黑帮。

**【贵重物品】** 忍装触媒；契约咒式；EVE 类芯片（作战干扰）；魔导武器。
