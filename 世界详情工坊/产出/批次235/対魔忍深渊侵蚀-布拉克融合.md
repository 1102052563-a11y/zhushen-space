# 対魔忍深渊侵蚀-布拉克融合

## 剧情

**【作品来源】**
日本 LiLiTH／BLACK LiLiTH「对魔忍系列」架空分支档案。系列自 2005 年《対魔忍アサギ》起，由笹山逸刀斋监制、カガミ等原画，展开近未来くノ一对抗魔族企业诺玛德的暗黑忍传；本条目**「布拉克融合」**为世界库独立 IF 线——以诺玛德创始者**埃德温·布莱克**与深渊本源发生融合为轴，不替代《阿莎姬3》真 END 灭布莱克正史，而写「布莱克选择成为深渊容器」的平行高压世界。气质：组织阴谋、吸血鬼权柄、深渊仪式、条件性胜利。本档案只写战斗、组织、祭仪与真名人物，不写色情细节。

**【世界定位】**
近未来日本已学院化对魔忍（五车学园）与企业化魔族（诺玛德）并立。深渊裂隙在东京—魔界通道上扩大后，布莱克不再满足于「评估威胁自己的人类」，而试图以自身生而吸血鬼之躯**融合深渊**，成为「不朽的地区灾难」。契约者切入的是六阶组织战与祭仪破局，不是街头见习日常。

**【世界观 · 力量体系】**
世界基本规则：近未来日本与魔界（Dark World）长期渗透。对魔忍以忍者血脉觉醒忍法（Ninja Art），配合体术、忍刀、特制对魔忍装与政府暗部任务配额对抗魔族与人魔外道。诺玛德等企业把魔族能力产品化：改造手术、斗士量产、黑市药剂、契约通道。「深渊侵蚀」线在此之上叠加第三层威胁——**深渊**作为魔界更下层/外域的侵蚀性本源，以裂隙、黑雾、祭仪、宿主融合改写人间与对魔忍血脉。死亡：凡人与下忍层即终结；高阶魔族可寄生换体、克隆、改造续命；深渊宿主可「局部再生」但人格被改写。成长逻辑：血脉觉醒→忍法成型→实战淬炼→组织权限；深渊线额外变量＝侵蚀抗性与祭仪节点。

忍法分类：元素系（雷/火/水，雪风线典型）、异能系（阿莎姬「隼之术/疾风」「光阵华」、樱「影遁」、紫怪力）、咒术系（胧「催眠刻印」）。通用：隐身、侦察、陷阱、分身残影。魔族侧：吸血鬼血权、那伽毒素、半兽人蛮力、梦魔精神干扰。深渊侧：黑雾领域、血脉反转、融合体、裂隙召唤。

战力阶梯（按实际破坏力，宁低勿高）：
1. **凡人·街头炮灰**：黑帮打手、普通警员、无觉醒学徒。热武器可致命。乐园≈一阶。
2. **下忍·低阶魔物/初染者**：见习对魔忍、半兽人杂兵、街头魔族、深渊雾中的「初染」平民。碎砖裂石。乐园≈二阶。
3. **中忍·成型魔物/雾卫**：独当一面的对魔忍、诺玛德中层、竞技场常规斗士、深渊卫士小队。裂金断甲、端据点。乐园≈三阶。
4. **上忍·精英Boss**：井河阿莎姬级、胧级、雪风/凛子精锐小队、深渊使徒。掀平建筑群、硬抗常规热武器。乐园≈四阶。
5. **城市级·领主威胁**：东京王国级地下王国主宰、魔界骑士、深渊裂隙都市化节点。一城治安与政治同时崩溃。乐园≈五阶。
6. **地区灾难·镇国/深渊顶点**：埃德温·布莱克等古吸血鬼、深渊融合体、最终仪式主祭。凡俗军队近乎无效；对话与祭仪有时比开战更危险。乐园≈六阶（本条目覆盖）。

乐园阶位映射：凡人/见习≈一阶；下忍·初染≈二阶；中忍·雾卫≈三阶；上忍·阿莎姬/使徒≈四阶；城市级裂隙≈五阶；布莱克/深渊融合顶点≈六阶。宁低勿高。世界顶点（埃德温·布莱克及深渊融合态）为跨阶存在，低阶切入一律规避·情报优先；**超阶巅峰**写条件性胜利，严禁「战力归零式削弱」。

**【地理 · 舞台】**
- **诺玛德最上层玻璃塔／融合圣所**：布莱克晚宴与融合仪式主舞台；冷白灯光、无影走廊、血契电梯。
- **东京地下裂隙网**：旧地铁、废弃实验室、与魔界相连的黑雾竖井；融合进度以雾密度计量。
- **五车学园外围封锁带**：对魔忍总部被政治与深渊双重施压，内外交通需特别通行证。
- **东京王国残影**：浮岛黑产仍在转运「融合触媒」——魔族血样、对魔忍基因、深渊黑晶。
- **魔界侧暗厅**：吸血鬼贵族与深渊使徒谈判处；人间侧不可久留。
- **德蒙兹竞技场旧址**：可被临时征用为「融合稳定性测试场」。

**【世界剧情线】**
① **前史·诺玛德与深渊接触**  
埃德温·布莱克以合法资本为壳统治人间暗面，长期对井河阿莎姬等「可能威胁自己的人类」抱试验欲。深渊裂隙首次被诺玛德科学班记录为「可吸收的外域魔力」；フュルスト一系魔科医（若仍在世或技术遗产）留下融合理论残页。世界状态：对魔忍以为最大敌是魔族企业，尚未理解深渊不是「可雇佣的种族」。

② **融合提案·玻璃塔晚宴**  
布莱克向政要与魔族贵族发出邀请：议题表面是「对魔忍价值评估」，实质是宣布将以自身为核启动**深渊融合**。井河阿莎姬被点名为「对照样本」——他要确认最强对魔忍在融合完成后是否仍能构成威胁。契约者若在此阶段入场，首要任务是获取邀请函真伪、座次与「触媒清单」，而非拔刀。

③ **触媒猎取·城市级阴影**  
融合需要三件硬通货：深渊黑晶（裂隙核心）、高纯度对魔忍血脉样本（井河/八津/水城系优先）、魔界古契约书。诺玛德干部与深渊使徒分头行动；东京王国拍卖、五车名录失窃、地下竖井黑晶盗采同时发生。八津紫可带队截击样本车；水城雪风与秋山凛子执行潜入；井河樱以影遁护撤离。名场面：样本车与假车并行，真货在第三条物流线上。

④ **阿莎姬对峙·条件而非决斗**  
阿莎姬携恶鬼六文赴塔外缘，拒绝成为「融合对照」。布莱克不与她比剑速，而以人质（被初染的生徒/被契约的政治家）与「融合后东京是否还存在」作筹码。任何写成「阿莎姬一刀斩融合态布莱克」的叙述都违背本线六阶规格；正确路径是破坏触媒、切断竖井供能、公开罪证迫使人间政府短暂联手。

⑤ **融合启动·地区灾难**  
仪式开始后，玻璃塔外云层呈兽形跪姿，黑雾覆盖数区。布莱克本体进入半融合：雾化、吸血、生命夺取与深渊再生叠加，凡俗军队与热武器近乎无效。对魔忍侧目标从「击杀」转为「打断稳定率」——毁三座触媒锚点中的两座即算阶段胜利。胧残党或邪恶胧技术遗产可能被用来干扰仪式，亦可能被深渊反噬。

⑥ **终局分叉**  
A. 触媒尽毁，布莱克融合失败但未死，诺玛德收缩，深渊裂隙暂封——昂贵的条件性胜利。  
B. 融合完成，布莱克成为深渊代行，东京进入「黑雾治世」，对魔忍转入地下反抗（衔接到「反抗军」线世界）。  
C. 契约者以自身为伪触媒骗过仪式，换阿莎姬等人撤离，自身面临侵蚀——乐园任务可用「牺牲式成功」结算。  
禁止无代价秒杀顶点。

⑦ **微观任务贯穿**  
护送抑制药与抗雾面罩、鉴定黑晶真伪、伪造晚宴身份、切断竖井供电、从科学班服务器拷融合日志、与イングリッド式忠臣交涉「是否仍忠于未融合的布莱克」。

⑧ **蝴蝶效应边界**  
可改：触媒运输成败、某名生徒初染是否可逆、晚宴上谁先翻脸。慎改：布莱克作为诺玛德顶点的长期在场、阿莎姬拒绝成为筹码的底色、融合需要多重触媒的规则。硬刚本体=任务失败或世界线崩坏。

⑨ **名场面速查**  
- 请柬一行字：今晚谈论「最强对魔忍」的未来——背面是融合倒计时。  
- 电梯无影：契约者发现自己的影子被黑雾吃掉半截。  
- 阿莎姬横刀：「他要的不是我的死，是确认我会不会成为他的同类。」  
- 三锚点地图：毁掉两个，第三个会自己爆炸。  
任意时点问：融合进度百分之几？触媒还剩几件？阿莎姬是否已入塔？

**【主要人物】**
- **埃德温·布莱克（Edwin Black）**（诺玛德创始者·融合核）｜性格：礼貌优雅冷酷；视生命危机为极致兴奋；融合是他主动的「升级实验」而非失控。｜装备·能力：生而吸血鬼、无传统弱点、组织权柄；融合后叠加深渊再生与黑雾领域。｜弧光：幕后顶点→宣布融合→半融合地区灾难→失败收缩或完成代行。｜立场：**世界顶点**；对契约者先评估利用价值；情报优先。
- **井河阿莎姬**（最强对魔忍·对照样本）｜性格：冷静决断、拒绝被当筹码；对布莱克兴趣了然于胸。｜装备·能力：恶鬼六文、隼之术/光阵华、刀脚；上忍顶点。｜弧光：被点名→对峙→破局或撤离。｜立场：可临时同盟，不可被「保护主义」收编。
- **井河樱**｜性格：姉想い、执行护送与影遁撤离。｜装备·能力：影遁、色差忍刀；抑制药依赖史。｜弧光：名录保卫战核心。｜立场：低信任门槛入口。
- **八津紫**｜性格：沉稳可靠、攻坚断后。｜装备·能力：怪力、大斧向规格。｜弧光：样本车截击队长。｜立场：优质队友。
- **水城雪风**｜性格：好胜直率任务狂。｜装备·能力：雷系忍法。｜弧光：潜入触媒仓库。｜立场：小队突击。
- **秋山凛子**｜性格：冷静大姊头。｜装备·能力：剑技稳健。｜弧光：雪风搭档护卫。｜立场：纪律优先。
- **イングリッド**（魔界骑士·布莱克侧）｜性格：忠于布莱克、冷对他人。｜装备·能力：骑士级、光学迷彩。｜弧光：融合前后忠诚测试——忠「人」还是忠「融合后的神」。｜立场：可策反窗口极窄。
- **胧（残党/技术遗产）**｜性格：复仇与玩弄；本线多为遗产或残存意志。｜装备·能力：催眠刻印系。｜弧光：干扰或助纣。｜立场：不可信。
- **诺玛德科学班主任**｜性格：数据至上。｜装备·能力：融合日志与拘束技术。｜弧光：触媒清单持有者。｜立场：可买可劫。
- **深渊使徒（具名代号：黑牙）**｜性格：非人、只推进仪式。｜装备·能力：雾卫指挥、裂隙开门。｜弧光：六阶中层压迫者。｜立场：杀或骗，无谈判温度。
- **山本信繁**（事务/政治联络）｜性格：在监视下支援对魔忍。｜装备·能力：公文与预算。｜弧光：公开罪证的钥匙。｜立场：任务板。
- **心愿寺红**（半魔猎人·互文）｜性格：孤高、执血脉宿命。｜装备·能力：半魔剑技。｜弧光：理解布莱克血脉的另一钥匙。｜立场：灰色同盟。

**【势力图谱】**
- **诺玛德**：融合计划执行体；代表布莱克、科学班、イングリッド；与深渊从「利用」滑向「共生或被吞」。
- **对魔忍／五车**：破局方；代表阿莎姬、樱、紫、雪风、凛子；内部有主战与避战派。
- **深渊使徒团**：仪式推进者；非传统魔族贵族，不认人间契约。
- **日本政府暗部／公安**：要表面治安，可弃刀；罪证公开后短暂联手。
- **东京王国黑产网**：触媒物流；认钱不认阵营。
敌友逻辑：利益与侵蚀抗性优先；盟约可在雾中作废。

**【贵重物品】**
- **深渊黑晶**：融合触媒；暴露即被双方追杀。
- **对魔忍基因样本箱**：井河/八津/水城血样；销毁往往比持有更安全。
- **融合日志硬盘**：科学班核心；可换政府联手。
- **恶鬼六文**：阿莎姬身份与战力象征。
- **晚宴真请柬**：血纹水印；假请柬进错层即死。
- **抗雾面罩与抑制药**：六阶生存硬通货。
- **古契约书**：魔界侧开门钥匙。
- **伪触媒（契约者可造）**：骗仪式用，风险自噬。

**【隐藏剧情 · 伏笔】**
- 布莱克融合不是疯狂，是「永续兴奋」的理性选择。
- イングリッド的忠诚裂缝是真破局点之一。
- 深渊使徒可能在利用布莱克，而非被其雇佣。
- 阿莎姬被选为对照，因她曾多次在「差点成为同类」的边缘停步。
- 与《红》血脉线互文：半魔是否更抗融合。

**【大事记时间线】**
深渊裂隙被诺玛德记录 → 触媒理论成型 → 玻璃塔晚宴宣布融合 → 触媒猎取战 → 阿莎姬对峙 → 融合启动 → 三锚点攻防 → 失败收缩／完成代行／伪触媒骗局。

**【叙事基调 · 雷区】**
冷白玻璃塔与黑雾竖井的压迫感；组织战优先于热血决斗。  
忌：六阶直接秒杀布莱克；把融合写成可恋爱感化；忽略触媒规则；用色情代替祭仪与情报；群像/牙人代称；把本线与阿莎姬3真END强行同一时间线无矛盾合并。  
最早切入锚点：晚宴请柬发出夜，或第一件黑晶被盗采时。


**【融合线·触媒计量学】**
黑晶纯度以「雾吸光度」标定：吸光度≥0.7方可作主触媒，0.3–0.7仅能作辅锚。诺玛德科学班用便携分光计；对魔忍侧可用雷系忍法瞬时电离空气作土法检测——雪风能做，但会暴露位置。伪触媒需同时骗过吸光度与血契纹，缺一被仪式反噬。

**【融合线·玻璃塔层序】**
B1物流、M1宴客厅、M2观礼廊、M3融合圣所、R顶投影室。真请柬电梯只停M1与M3；假请柬会在B1被安保「热情接待」。阿莎姬若入塔，习惯走消防楼梯反清——楼梯间是契约者最佳汇合点。

**【融合线·イングリッド裂缝】**
她忠的是「能让布莱克保持优雅的秩序」。融合若使布莱克失态（例如当众雾化吞噬宾客），她会出现0.5秒的迟疑——足够塞入一句暗号或一枚干扰器。迟疑只有一次，用过即封。

**【融合线·阿莎姬对照逻辑】**
布莱克要的数据是：最强对魔忍在「同类化压力」下会崩还是会砍。因此他不会在融合完成前杀死阿莎姬；这反而给破局留窗——她是活体对照，不是一次性祭品。

**【融合线·三锚点破坏次序】**
先毁供能锚（竖井）、再毁样本锚（基因箱中转）、最后动总晶——反序会触发紧急融合，稳定率反而上升。紫带爆破队必须带时序器，不能「看见就炸」。

**【融合线·政治曝光包】**
山本信繁要的不是英雄故事，是：融合日志摘要+三位政要签名复印件+一张竖井坐标。三件齐，议会才肯临时授权对魔忍「国内灾害级别」行动权。
