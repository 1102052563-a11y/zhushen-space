# 対魔忍深渊侵蚀-布拉克融合
<!--meta lib=主库 tiers=六-->

## 剧情

**【作品来源】**
日本 LiLiTH／BLACK LiLiTH「对魔忍系列」架空分支档案。系列自 2005 年《対魔忍アサギ》起，由笹山逸刀斋监制、カガミ等原画，展开近未来くノ一对抗魔族企业诺玛德（Nomad）的暗黑忍传；正史轴心人物包括井河阿莎姬、井河樱、八津紫、水城雪风、秋山凛子、胧、埃德温·布莱克等。本条目**「布拉克融合」**（布莱克／布拉克同指 Edwin Black）为世界库独立 IF 线——以诺玛德创始者**埃德温·布莱克**主动选择与**深渊本源**融合为轴，不替代《阿莎姬3》真 END 灭布莱克正史，而写「布莱克把自身当作容器与实验核」的平行高压世界。气质：组织阴谋、吸血鬼权柄、深渊祭仪、条件性胜利。本档案只写战斗、组织、祭仪与真名人物，**不写色情细节**（原作成人设定仅保留「改造/契约作为组织手段与精神负荷」级叙事功能）。

**【世界定位】**
近未来日本：对魔忍学院化（五车学园）与魔族企业化（诺玛德）并立，东京王国式黑产浮岛仍转运违禁物。深渊裂隙在东京—魔界通道上扩大后，布莱克不再满足于「评估可能威胁自己的人类」，而试图以生而吸血鬼之躯**融合深渊**，成为「不朽的地区灾难·深渊代行」。契约者切入的是**六阶组织战与祭仪破局**，不是街头见习日常；胜利定义为打断融合稳定率、公开罪证、保全关键人物，**不要求弑神式击杀顶点**。

**【世界观 · 力量体系】**
世界基本规则分三层：

1. **人间侧**  
   近未来日本。对魔忍以忍者血脉觉醒忍法（Ninja Art），配合体术、忍刀、特制对魔忍装与政府暗部任务配额，对抗魔族与人魔外道。组织形态依时点可含忍里、公安 Section 系、后期五车学园等。成长逻辑：血脉觉醒→忍法成型→实战淬炼→组织权限，不是打怪升级。死亡：凡人与下忍层即终结。

2. **魔界侧（Dark World）**  
   吸血鬼、那伽、半兽人、梦魔等；诺玛德以合法资本为壳经营改造手术、斗士量产、黑市药剂、契约通道与政治渗透。高阶魔族可寄生换体、克隆、改造续命。埃德温·布莱克作为古吸血鬼与诺玛德顶点，常规弱点（阳光/木桩/圣水类）对其近乎无效——系列中他长期以「优雅实验者」姿态出现，把生命危机当极致兴奋。

3. **深渊侧（本系列 IF 增量）**  
   深渊不是单纯魔界扩张，而是会改写**血脉回路、忍法轨迹与人格锚点**的高阶外域污染源；以裂隙、黑雾、祭仪、宿主融合展开。污染分阶段：接触→回路紊乱→人格锚松动→融合体→仪式核心。可被意志、特殊抑制药、破坏触媒延缓，不可被口号清除。深渊宿主可「局部再生」但人格被改写。本线关键：**布莱克不是被深渊绑架的受害者，而是主动申请成为容器的实验者**——这使「感化他放弃融合」几乎无效，破局必须走触媒、供能、罪证与忠诚裂缝。

忍法分类：元素系（雷/火/水，雪风线典型）、异能系（阿莎姬「隼之术/疾风」「光阵华」、樱「影遁」、紫怪力）、咒术系（胧「催眠刻印」）。通用：隐身、侦察、陷阱、分身残影。魔族侧：吸血鬼血权、那伽毒素、半兽人蛮力、梦魔精神干扰。深渊侧：黑雾领域、血脉反转、融合体、裂隙召唤、雾化预注册（对「可吸收个体」的标记）。

战力阶梯（按实际破坏力，宁低勿高）：
1. **凡人·街头炮灰**：黑帮打手、普通警员、无觉醒学徒。热武器可致命。乐园≈一阶。
2. **下忍·低阶魔物/初染者**：见习对魔忍、半兽人杂兵、街头魔族、深渊雾中的「初染」平民。碎砖裂石。乐园≈二阶。
3. **中忍·成型魔物/雾卫**：独当一面的对魔忍、诺玛德中层、竞技场常规斗士、深渊卫士小队。裂金断甲、端据点。乐园≈三阶。
4. **上忍·精英Boss**：井河阿莎姬级、胧级、雪风/凛子精锐小队、深渊使徒。掀平建筑群、硬抗常规热武器。乐园≈四阶。
5. **城市级·领主威胁**：东京王国级地下王国主宰、魔界骑士、深渊裂隙都市化节点。一城治安与政治同时崩溃。乐园≈五阶。
6. **地区灾难·镇国/深渊顶点**：埃德温·布莱克（含半融合/完全融合态）、深渊代行、最终仪式主祭。凡俗军队近乎无效；对话与祭仪有时比开战更危险。乐园≈六阶（**本条目唯一覆盖**）。

**乐园阶位映射**：凡人/见习≈一阶；下忍·初染≈二阶；中忍·雾卫≈三阶；上忍·阿莎姬/使徒≈四阶；城市级裂隙≈五阶；布莱克/深渊融合顶点≈六阶。宁低勿高。世界顶点（埃德温·布莱克及深渊融合态）为跨阶存在，低阶切入一律规避·情报优先；**超阶巅峰**写条件性胜利，严禁「战力归零式削弱」。

**【地理 · 舞台】**
- **诺玛德最上层玻璃塔／融合圣所**：布莱克晚宴与融合仪式主舞台。冷白灯光、无影走廊、血契电梯。层序：B1 物流、M1 宴客厅、M2 观礼廊、M3 融合圣所、R 顶投影室。真请柬电梯只停 M1 与 M3；假请柬在 B1 被「热情接待」。消防楼梯是对魔忍侧反清与汇合首选。
- **东京地下裂隙网**：旧地铁、废弃实验室、与魔界相连的黑雾竖井；融合进度以雾密度与吸光度计量。主竖井在港区地下。
- **五车学园外围封锁带**：对魔忍总部被政治与深渊双重施压，内外交通需特别通行证；名录备份若仍在此则是高价值撤离目标。
- **东京王国残影／连陆残桥仓库**：浮岛黑产转运「融合触媒」——魔族血样、对魔忍基因、深渊黑晶；样本中转锚常设于此。
- **魔界侧暗厅**：吸血鬼贵族与深渊使徒谈判处；人间侧不可久留；古契约书常存于此。
- **德蒙兹竞技场旧址**：可被临时征用为「融合稳定性测试场」；亦可能藏胧系残党干扰装置。
- **议会与公安联席厅**：罪证公开后短暂授权对魔忍「国内灾害级别」行动权的政治舞台。
- **科学班地下冷库**：融合日志服务器、拘束架、失败实验体处理区；拷贝优先截图四关键字段。

**【世界剧情线】**

① **前史·诺玛德与深渊接触**  
埃德温·布莱克以合法资本为壳统治人间暗面，长期对井河阿莎姬等「可能威胁自己的人类」抱试验欲——系列正史中他多次以观察、改造、交易桌而非单纯斩杀处理对魔忍。深渊裂隙首次被诺玛德科学班记录为「可吸收的外域魔力」；魔科医一系（フュルスト技术遗产或同型实验室）留下融合理论残页：吸血鬼再生回路可与深渊雾化回路「并联」，代价是人格被外域语法改写。布莱克读完残页后的批注只有一行：「永续兴奋。值得。」世界状态：对魔忍仍以为最大敌是魔族企业与胧系堕忍，尚未理解深渊不是「可雇佣的种族」，而是会把雇主也写成燃料的本源。イングリッド等忠臣被要求准备「高层评估晚宴」，不知议题已从并购改为成神。

② **融合提案·玻璃塔晚宴（T-48h～T0）**  
布莱克向政要、魔族贵族与少数「有趣的人类」发出请柬：议题表面是「对魔忍价值评估与能源合作」，实质是宣布将以自身为核启动**深渊融合**。请柬正面一行字谈「最强对魔忍」的未来，背面渗出黑雾冷凝水——血纹水印方向决定电梯停层。宾客分三桌：政要桌要「可控的深渊」与连任经费；魔族贵族桌要通道份额；诺玛德中层桌要股票与活体数据。先举杯者往往已签血契，后举杯者仍可策反；第三桌「看表」者多是科学班联络，持有触媒时序。井河阿莎姬被点名为「对照样本」——布莱克要确认最强对魔忍在融合完成后是否仍能构成威胁；**因此他不会在融合完成前杀死阿莎姬**，这反而给破局留窗。契约者若在此阶段入场，首要任务是获取邀请函真伪、座次、触媒清单与日志字段，而非拔刀。名场面：冷白电梯里影子被雾吃掉半截——雾在预注册「可吸收个体」；阿莎姬用忍刀刃面照你的脸：「还能看见瞳孔就还有时间。」

③ **触媒猎取·城市级阴影（T-24h 前后高峰）**  
融合需要三件硬通货：  
- **深渊黑晶**（裂隙核心；吸光度≥0.7 方可作主触媒，0.3–0.7 仅辅锚）；  
- **高纯度对魔忍血脉样本**（井河／八津／水城系优先）；  
- **魔界古契约书**（开门与法理锚定）。  
诺玛德干部与深渊使徒分头行动；东京王国拍卖、五车名录失窃、地下竖井黑晶盗采同时发生。八津紫可带队截击样本车；水城雪风与秋山凛子执行潜入；井河樱以影遁护撤离。名场面：样本车与假车并行，真货在第三条物流线上——任何「看见就炸」都会炸到空车。伪触媒工艺：外壳须真黑晶碎屑烧结，内核为惰性魔核；还要在血契纹上摹刻布莱克旧签名笔压——イングリッド见过真迹，是最大识破风险，也是最大迟疑窗口。

④ **阿莎姬对峙·条件而非决斗**  
阿莎姬携恶鬼六文赴塔外缘，拒绝成为「融合对照」。布莱克不与她比剑速，而以人质（被初染的生徒／被契约的政治家）与「融合后东京是否还存在」作筹码。任何写成「阿莎姬一刀斩融合态布莱克」的叙述都违背本线六阶规格；正确路径是破坏触媒、切断竖井供能、公开罪证迫使人间政府短暂联手。阿莎姬可观察台词短而冷：「他要确认我会不会成为同类。」「先选你卖什么。」「对照样本的价值在活着。」她不发表长篇演说，只用刀与任务分配表达立场。若契约者试图「保护主义收编」她，会被当场拒绝——她是决策轴，不是被护送的花瓶。

⑤ **融合启动·地区灾难（T0～T+2h）**  
仪式开始后，玻璃塔外云层呈兽形跪姿，黑雾覆盖数区。布莱克本体进入半融合：雾化、吸血、生命夺取与深渊再生叠加，凡俗军队与热武器近乎无效。常规抗雾面罩只能降低深渊雾对凡人的初染速度，对吸血鬼本体雾化无效；对本体短时有效的是「光频干扰片」（红线半魔血可激活）。对魔忍侧目标从「击杀」转为「打断稳定率」——三锚点拓扑：主竖井（港区地下）、样本中转（旧东京王国连陆残桥仓库）、总晶（玻璃塔 R 层投影室正下方）。**破坏次序铁则：先毁供能锚（竖井）、再毁样本锚、最后动总晶**；反序会触发紧急融合，稳定率反而上升（先毁总晶约 +15%）。胧残党或催眠刻印技术遗产可能被用来干扰仪式，亦可能被深渊反噬。イングリッド忠诚裂缝：她忠的是「能让布莱克保持优雅的秩序」；若融合使布莱克当众失态（例如雾化吞噬宾客），她会出现约 0.5 秒迟疑——足够塞入一句暗号或一枚干扰器；**迟疑只有一次，用过即封**。

⑥ **终局分叉**  
A. **触媒尽毁／稳定率打断**：布莱克融合失败但未死，诺玛德收缩，深渊裂隙暂封——昂贵的条件性胜利。失败收缩后他不会「变弱到可斩」，而转入更隐蔽的企业战争：收购抗雾专利、收购对魔忍编制预算、用媒体把阿莎姬打成恐慌制造者；下一周任务从祭仪转为舆论与供应链。  
B. **融合完成**：布莱克成为深渊代行，东京进入「黑雾治世」（电车按可呼吸时段运行，公开编制解散为分区保安）——本世界主线失败态，直接导向「反抗军」线。  
C. **伪触媒骗局**：契约者以自身或伪造物为伪触媒骗过仪式，换阿莎姬等人撤离，自身面临侵蚀——乐园可用「牺牲式成功」结算。  
禁止无代价秒杀顶点；禁止恋爱感化放弃融合。

⑦ **政治曝光与议会窗口**  
山本信繁要的不是英雄故事，而是物证包：融合日志摘要（吸光度、血契纹版本、对照样本编号、失败实验体处理方式四字段优先）+ 三位政要签名复印件 + 一张竖井坐标。三件齐，议会才肯临时授权对魔忍「国内灾害级别」行动权。公关线（具名：白雪）会把融合写成「能源革命」；拆穿需同步直播与硬盘，单靠口述无效。

⑧ **微观任务贯穿**  
护送抑制药与抗雾面罩；鉴定黑晶真伪（分光计或雷系瞬时电离土法——雪风能做但会暴露位置）；伪造晚宴身份；切断竖井供电；从科学班服务器拷融合日志；与イングリッド交涉「是否仍忠于未融合的布莱克」；校准爆破时序器；在 M2 观礼廊偷拍政要签字；护送井河樱影中转移名录备份出塔区。

⑨ **蝴蝶效应边界**  
可改：触媒运输成败、某名生徒初染是否可逆、晚宴上谁先翻脸、イングリッド迟疑是否被利用、伪触媒是否过检、议会授权是否当天下达。  
慎改：布莱克作为诺玛德顶点的长期在场、阿莎姬拒绝成为筹码的底色、融合需要多重触媒的规则、三锚点破坏次序。  
硬刚本体=任务失败或世界线崩坏。硬改「布莱克从未接触深渊」→本线主题改判。硬改「一刀斩神」→规格崩坏。

⑩ **名场面速查**  
- 请柬一行字：今晚谈论「最强对魔忍」的未来——背面是融合倒计时。  
- 电梯无影：契约者发现自己的影子被黑雾吃掉半截。  
- 阿莎姬横刀：「他要的不是我的死，是确认我会不会成为他的同类。」  
- 三锚点地图：毁掉两个，第三个会自己爆炸——但次序错了会全部锁死。  
- イングリッド 0.5 秒：刀尖停在半空，眼中第一次出现「这不是他」。  
- 投影室：布莱克微笑宣布「从今夜起，对照结束」。  
任意时点问：融合进度百分之几？触媒还剩几件？锚点毁了几个、次序对不对？阿莎姬是否已入塔？日志四字段到手没有？

T-6h：竖井加强戒。  
T-1h：宾客安检。

契约者入场时点决定能碰哪一段；T+2h 后仍坚持「斩首布莱克」的计划默认团灭。

⑫ **条件性胜利清单**

> 阶位↔原作强度：一阶≈凡人/见习；二阶≈下忍·初染；三阶≈中忍·雾卫；四阶≈上忍·使徒/阿莎姬级；五阶≈城市级裂隙；六阶≈埃德温·布莱克/深渊融合顶点。宁低勿高。世界顶点为跨阶存在，切入一律「规避硬刚·情报/祭仪优先」。本世界仅覆盖：**六阶**。

**六阶（地区灾难 · 布拉克深渊融合）**

切入身份/时点：契约者以「能与诺玛德高层对话的灾厄级协力者／乐园特准深层刀／议会观察特使」切入，锚定**布莱克宣布深渊融合前后**——推荐 **T-48h 请柬发放夜** 至 **T0 投影宣布** 窗口；亦可卡在 T-24h 样本车出动或 T+30m 锚点锁定前夜。

初始事件：你同时收到两封信。一封血纹真柬邀你上玻璃塔「评估井河阿莎姬的未来价值」；一封匿名附三锚点坐标与爆破次序警告：「先竖井，再样本，最后总晶——反序即送他成神。」电梯门开前，**井河阿莎姬**通讯只留一句：「先选你卖什么——命、情报，还是整个对魔忍。」走廊尽头冷白灯下，你的影子缺了半截。

开场白建议：「玻璃塔外的云像跪着的兽。请柬正面只有一行字，背面却渗出黑雾冷凝水。阿莎姬把恶鬼六文横在膝上，不看你的眼睛，只看你的影子：『他要确认的不是我会不会死，是我会不会成为他的同类。你若进去，先想好自己是触媒，还是毁触媒的人。』电梯显示 M3，血纹水印在手腕内侧发烫。」

关键NPC立场：**埃德温·布莱克**——超阶巅峰·情报优先，评估你是否值得纳入融合方程，交易开口永远是「优化」不是投降；**井河阿莎姬**——临时同盟但拒绝当你的筹码，要你交一条真实目的；**八津紫**——可执行锚点爆破，强求时序；**イングリッド**——忠诚裂缝极窄，识破伪触媒与笔压，0.5 秒窗口不可复用；**水城雪风／秋山凛子**——潜入侧翼与土法测晶；**井河樱**——影中转移名录／样本；**科学班次席·加藤**——卖日志四字段换命；**山本信繁**——物证包换议会授权；**深渊使徒·黑牙**——无温度推进仪轨。

主线钩子/支线：主线＝打断深渊融合稳定率（正确次序毁触媒／公开日志换授权／伪触媒骗局／利用イングリッド一次迟疑）。支线＝M2 偷拍政要签字；校准紫的时序器；向イングリッド只说半句「失态」试探裂缝；护送樱出塔区；截获第三箱黑晶；在港区竖井安放供能干扰；识破公关白雪的「能源革命」直播脚本。

危险度/规避：贴近顶点——**超阶巅峰：存在·情报优先／条件性胜利**，严禁硬刚本体或写「战力归零式削弱」。必须规避：无准备的正面对轰、T+2h 后仍冲圣所、反序爆破总晶、被吸入最终祭仪核心、把同伴当可消耗燃料、对布莱克雾化本体只戴常规抗雾面罩、无光频片却规划「刺布莱克」、签任何可能被写成自愿容器化的优化同意书、重复消费イングリッド迟疑。

任务方向/奖励：阶段性和平、关键人物存活、诺玛德局部收缩或议会临时授权；奖励＝六阶魔族／深渊克制物、融合日志残卷、抗雾血脉线索、光频干扰片、竖井地图、乐园组织级战果——**非弑神证明**，不发布莱克血脉继承与无条件永生。

执行细则（六阶专属）：开局第一幕必须出现真／假请柬或三锚点坐标之一；禁止用「拍卖假货护送」通用句替换。若选择赴宴，安检戏要写清血纹水印方向与电梯停层；若选择毁锚，爆破次序必须先竖井后样本。与布莱克的任何接触以对话、评估、交易为主，战斗仅作撤离掩护。与阿莎姬结盟条件是交出真实目的一条。条件性胜利占三即可，不要求布莱克死亡。

六阶行动清单（融合）：①鉴别真假请柬与水印方向；②确认小时表落在 T-48h～T+2h 哪一段；③清点触媒三件与吸光度；④按序部署竖井→样本→总晶；⑤截取日志四字段；⑥试探イングリッド一次；⑦准备光频片与抗雾分层；⑧物证包递山本；⑨阿莎姬是否入塔的实时确认；⑩禁止反序与硬刚。

六阶失败学（融合）：反序炸总晶＝稳定率+15%；假请柬进 B1＝团灭开局；只靠抗雾面罩冲雾化本体＝送死；把阿莎姬当祭品交易＝主线道德失败且她会反杀交易方；忽略议会物证只砍人＝阶段后被媒体绞杀；伪触媒缺笔压＝当场反噬；T+2h 后仍宣称斩首＝规格外自杀。

六阶对话样本（可直接用）：

六阶胜利定义（本世界）：融合稳定率被打断或伪触媒骗过关键窗口 + 关键人物（阿莎姬／樱／紫等）存活率达标 + 未成为布莱克合同乙方或对照祭品。击杀榜与「完全净化东京」均非胜利条件。残局允许布莱克未死与企业战周——这是代价不是失败；代行完成则判定主线失败态。
