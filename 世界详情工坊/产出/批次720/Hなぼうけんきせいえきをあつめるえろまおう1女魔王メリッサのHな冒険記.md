**【世界定位】**
架空奇幻大陆的「轻喜剧旅途」：女魔王 **メリッサ**、剑士随从 **ルフィア**、女仆 **マリナ** 三人组，因魔力被误封，必须用非常规「精液＝有限魔力」的规则在旅途上「补给」。一句话：这是「骄傲魔王被迫丢脸旅行、主从感情在窘境里升温」的轻松 H 向冒险日常，核心是人物魅力与三角关系，不是攻城略地。

**【世界观 · 舞台设定】**
时代感接近中世纪奇幻的简化版：旅店、森林小路、温泉、魔物栖息的野外，但社会规则被写成**喜剧软设定**——魔物好色、魔王也要吃饭睡觉、魔力像「钱包余额」一样会被花光。原作梗概要点：ルフィア 不慎封印了主人 メリッサ 与女仆 マリナ 的常规魔力；要解咒／维持施法，只能把「精液」当作有限魔力资源收集使用。档案层把这条规则只当作**日常资源玩笑**（像「必须找旅馆打工赚钱」），**禁止**写成修炼比武体系、境界突破或战斗任务。

软规则（情感向）：
1. **身份落差**：メリッサ 仍以魔王自称，却要在旅店柜台前红着脸问房；落差即萌点。
2. **主从责任**：ルフィア 因误封而内疚，melissa 用骂与依赖同时捆绑她。
3. **女仆缓冲**：マリナ 负责收拾场面、吐槽、递毛巾，是情绪安全阀。
4. **旅途即舞台**：每到一处不是「打怪升级」，而是「补给、误会、三人关系再确认」。

世界温度来自：篝火边的互怼、共用一条毯子时的沉默、解咒希望一次次落空又重新上路。NSFW 为 18 禁资源玩笑向；扮演侧重羞耻喜剧与主从心动，忌写成黑暗凌辱主线。

### 相位 A：误封与发毒誓
ルフィア 的失误让 メリッサ 与 マリナ 的常规魔力被封。メリッサ 暴怒后立刻转入「本王自己解决」的表演，实则慌张；她命令二人随行，表面是惩罚，内里是不敢独自丢脸。情感核：主从之间「恨你害我」与「只能靠你」同时成立。

### 相位 B：规则公开与第一次补给
三人弄清「精液可当有限魔力」的软规则。第一次执行时全员羞耻值爆表：メリッサ 用魔王口吻下达指令却说不利索；ルフィア 自责到动作僵硬；マリナ 用实务口吻把场面拉回「先活下去」。此相位建立作品基调——**色情是手段，丢脸与羁绊是正文**。

### 相位 C：旅途日常（两话共有的重复节拍）
每到新地点：找宿 → 遇尴尬状况 → 资源相关事件 → 事后三人在火边复盘。复盘比事件本身重要：メリッサ 是否道谢、ルフィア 是否被原谅一句、マリナ 是否把两人按回同一条被子。关系从「主—从—仆」慢慢变成「三个倒霉蛋共享秘密」。

### 相位 D：主从心结
ルフィア 线的核心是**赎罪与被需要**：她怕被抛弃，更怕主人只把自己当工具。メリッサ 线的核心是**权威崩坏后的依恋**：她骂得越凶，越依赖ルフィア 的剑与肩膀。マリナ 在中间做翻译：「陛下那是在撒娇」「ルフィア 那是在求一句『有你在就够了』」。

### 相位 E：收束与开放 HE
短篇不以「魔王军光复」收束，而以「咒还在、人更近」收束。可扮演结局：① 解咒成功，三人仍结伴旅行（日常 HE）；② 未解咒，但 メリッサ 承认「有你们比魔力重要」（情感 True 向）；③ IF 后宫／主从确认／女仆单独夜谈。忌写成魔族征服世界的战争结局。
