## 剧情

**【作品来源】**
《黒獣》系列。本档案「七公主宴」指七盾同盟七位女性统治者（圣女加六砦姬与骑士长等象征）在军事上被逐一击破后，奉仕国家举行的「征服之宴」政治仪式：展示、册封伪职、分赃、强迫出席。焦点是宴会政治与象征暴力，不是日常筵席描写，不写色情。

**【世界定位】**
当七砦序列完成后，沃鲁特需要一次「可被史官书写的胜利」。宴会=新秩序的加冕礼。契约者切入点常在宴会筹备、入场名册、毒酒与逃生廊。本线物证是请柬、地契、名牌，不与神镜或月溪名册重复。

**【世界观 · 力量体系】**
塞勒努斯大陆西方埃奥斯平原：冷兵器、魔法、神术、亚人血脉与佣兵战术并存。死亡即死，无无限复活；俘虏与政治工具化比「当场斩首」更常见。

1. **民兵/溃兵/领民自卫**：徒手或农具，一刀可致命。乐园≈**一阶**。
2. **精锐士兵/低阶骑士/佣兵骨干**：碎砖裂甲、单挑小队。乐园≈**二阶**。
3. **砦守护者/具名女骑士/精英佣兵**：裂石断金、端据点。乐园≈**三阶**边缘。
4. **圣女预知指挥、砦姬单体决战、佣兵王沃鲁特与破城决战**：摧毁据点/掀平建筑群、一城仅数人级。乐园≈**四阶**（本条目覆盖阶）。

乐园阶位映射：民兵≈一阶；精锐士兵/低阶骑士≈二阶；砦守护者≈三阶边缘；圣女预知、砦姬决战与佣兵王沃鲁特≈四阶（宁低勿高）。世界顶点=沃鲁特+黑犬战争机器，条件性胜利=斩首链/夺象征人物/断补给/策反，严禁「战力归零式削弱」。
宴会场内武力被礼仪与人质结构压制：强者也难当场翻盘，因为七人分散、观众是军队。梁上弓手专射「起立鼓掌太慢」者——礼仪即杀意。条件性胜利优先于正战：毁地契、开侧廊、劫一人、抢家谱页。

**【地理 · 舞台】**
奉仕王都临时大殿（或黑之城改造金殿）是主场。七席展示台按破城顺序排座。侧廊与厨房链是下毒与换人通道。广场绞架与庆功火堆烧家谱抄本。地下酒窖藏旧同盟酒与密道，通向广场外粪车路线——臭，但可行。外族佣兵看客区与商会竞价席构成观众政治。

**【世界剧情线】**
① 七砦尽破，象征人物或真或伪被集中。请柬编号规则：金字必须到，银字可派代表，黑字已是俘虏；持金字假请柬安检更严。

② 基恩发请柬：领主、商会、伪神官、外族雇佣兵皆须到场「见证和平」。请柬火漆含追踪粉。某席可能是自愿出席换城中孤儿活命。

③ 宴会流程：献旗→点名→伪册封→分地契→强迫祝酒。沃鲁特致辞只三分钟，其余时间交给基恩读分赃名单——名单才是真权力。

④ 名场面：空席与替身同时存在——某砦姬未到，用假人与军旗填席，观众必须鼓掌。七席顺序：一阿莉西亚、二普莉姆、三辉夜、四玛雅、五露露、六七克劳迪娅，首座留给圣女。

⑤ 可能的转折：奥莉嘉系倒戈、玛雅拒酒、克劳迪娅挣锁、圣女预知残片示警、乐手第三曲「丰收」实为旧同盟集结号（奏完整=侧廊开门三十秒）、厨役总管（前二砦厨师）以刀工切锁换「先救孩后救人」承诺。

⑥ 宴后：地契生效，游击合法空间归零，直到有人烧掉地契册。地契墨用魔血，烧了会尖叫报警，需特制灰覆。宴会后的余兴是游街——比宴会更好劫人，但观众更多。基恩备用方案：若有人劫宴，立刻处死 A 席，因此你必须让他相信「无劫宴」或先移开 A 席人质。

席位道具政治：辉夜席前无镜只有碎镜镶盘；普莉姆席前堆花，花下是税册；阿莉西亚枪挂梁上展览，取枪等于宣战；克劳迪娅双席合并为一，强调「最后一块骨头」；玛雅伪册封为边境治安官，逼她用旧面孔镇压旧同事；露露席前摆玩具攻城锤模型，模型里可藏小型爆破符；奥莉嘉被安排贵宾席而非囚席，政治侮辱大于铁链；克洛伊侍立链短到不能举杯，递酒动作可藏纸条；塞勒斯缇娜若出席，预知被药物压成碎片，只能给单字暗示。

**【势力图谱】**
奉仕宫廷、败军俘虏、被迫观礼的领民、外族佣兵看客、地下反抗（厨役与乐手）。外族佣兵下注谁先崩溃，可下注换情报。庆功火堆烧七砦家谱抄本，抢出一页=一族血脉证明。

**【贵重物品】**
七席名牌、伪地契册、祝酒杯（内壁微刻「奉仕」，转三圈药剂活化；不转杯只抿可能活）、破城军旗七面、酒窖钥匙环、乐队总谱（暗语）、家谱残页、追踪粉请柬、特制灭火灰、玩具攻城锤内爆破符。

**【隐藏剧情 · 伏笔】**
请柬火漆含追踪粉。某席自愿出席换孤儿活命。乐手节奏=逃生信号。基恩备用处死 A 席方案说明「无完美英雄救美」。宴后游街才是更高收益也更高暴露的劫人窗。本宴与单砦陷落篇物证不重复。

**【大事记时间线】**
七砦破 → 请柬 → 宴会夜 → 地契生效 → 游击转入完全地下 → 游街余兴 → 史官书写新秩序。

**【叙事基调 · 雷区】**
政治仪式恐怖。忌写成欢乐派对；忌无代价全员救出；忌忽略观众军队的威慑；忌四阶在致辞三分钟内硬刚沃鲁特。最早锚点：请柬夜至祝酒前。

**【宴会流程分钟表】**
入场安检（金字更严）→ 献旗 → 点名（空席填旗）→ 伪册封 → 分地契（魔血墨）→ 沃鲁特三分钟致辞 → 基恩读名单 → 强迫祝酒 → 可能余兴游街。契约者行动窗：乐手第三曲三十秒、厨役切锁、粪车密道、抬价搅局、覆灰毁册。成功标准：至少一名象征人物离开金殿视野，或地契第 7 页以上被毁且未触发全场屠杀。失败标准：A 席被处死、名单读完且地契盖印、你被梁上箭钉在鼓掌席。

**【象征暴力词典】**
空席、假人、碎镜盘、花下税册、梁上枪、双席合并、贵宾囚、短链侍立、玩具锤、鼓掌竞价、家谱火堆——写场景时至少点出两种，避免空喊「黑暗」。气味：香遮不住铁锈；声音：鼓槌三下起拍；触感：名牌烫手因镀金下是旧砦铁。

## 阶位切入点


> 阶位↔：一阶≈庆典杂役、平民与外城守卫；二阶≈信使、护卫与黑犬斥候；三阶≈砦防骨干、具名骑士与仪式护卫；四阶≈沃鲁特主持的砦级征服战。沃鲁特是顶点威胁，超阶巅峰：存在·情报优先/条件性胜利。

**一阶（战后王都外缘·请柬传递）**
切入身份/时点：契约者以厨房杂役、马厩帮工、外城信使或被征发居民身份切入，锚定黑犬反叛、战后庆典消息开始沿埃奥斯平原传播的阶段。
初始事件：盖着新印的请柬要送入城内，送信人却被流民拦住。请柬命令参加庆典，却没有说明城门是否换旗；你可以替他送信、藏起请柬，或先帮助封城外的家属。
开场白建议：「城门上刚挂起陌生的旗，马厩里已经堆满写着名字的请柬。你把那张纸捂在衣襟里，外城的钟却敲得像庆祝，也像提醒谁开始有资格进门。」
关键NPC立场：**沃鲁特**要让胜利被看见；**基恩**把来宾和粮秣列入同一记录；**希克斯**监视城门信使；**塞蕾丝汀·卢克鲁斯**尝试维持通信；**普莉姆·菲奥里雷**关切被困领民。
主线钩子/支线：主线是决定请柬与城门消息落到谁手中；支线可带走家属、辨认伪印、找回被扣马匹或把外城状况传给同盟。
危险度/规避：低至中。搜查和流民冲突足以致命；规避仪式内场及高阶将领视线。
任务方向/奖励：奖励取向为城门通行、外城人情、可靠路线和庆典日程，不提供贵族地位。

**二阶（筹备期·名单与护卫）**
切入身份/时点：契约者以宴会护卫、商队账房、乐师随从或同盟密使身份切入，锚定征服仪式筹备、各砦消息纷乱抵达王都的时期。
初始事件：两份受邀名单日期相同、签名不同；一份增派黑犬护卫，另一份把同盟旧臣安排外席。持名单者已经失踪，你可查验印信、护送乐师离城，或把矛盾名单交给会利用它的人。
开场白建议：「大殿还没有点灯，走廊已挤满搬酒、磨刀和练习鞠躬的人。你从账房桌下抽出第二份名单，墨迹没干，名字却少了三个。」
关键NPC立场：**基恩**希望名单按新秩序完成；**希克斯**甄别并清除可疑者；**玛雅**警惕被迫参与的旧部；**克劳迪娅·雷凡坦**若能联络会优先寻找同盟人员；**奥莉嘉·迪斯科迪亚**旧部可能把仪式视作反扑机会。
主线钩子/支线：主线是确认真实名单与护卫调度；支线可保护证人、调换物资单、找回被扣乐器，或从外席带出同盟消息。
危险度/规避：中。护卫和斥候可进行小队战；规避把名单当作能一举推翻沃鲁特的万能证据。
任务方向/奖励：奖励取向为仪式日程、名单副本、通行口令和少量人脉，不发放王都控制权。

**三阶（仪式前夜·救人与证词）**
切入身份/时点：契约者以精锐护卫、砦主副官、潜入者或受雇调查者身份切入，锚定庆典前夜、黑犬以胜利展示巩固占领秩序的阶段。
初始事件：从七盾砦链逃出的证人被押往仪式场，声称掌握黑犬反叛后收编魔军的过程。营救会暴露残部，沉默则让沃鲁特版本成为唯一公开记忆。
开场白建议：「窗外的鼓点正在排练胜利，地下通道里却有人背诵砦名。你知道他若在明天开口会死；他若不开口，明天的旗帜就会替所有人讲另一种故事。」
关键NPC立场：**沃鲁特**需要可传播的胜利叙事；**基恩**以文书让占领常态化；**希克斯**截断反抗联络；**塞蕾丝汀·卢克鲁斯**需要可信证词；**阿莉西亚·阿克图鲁斯**与**普莉姆·菲奥里雷**的守军处境影响证词价值。
主线钩子/支线：主线是救出证人或把证词送至安全处；支线可伪造出城许可、破坏囚室警报、安排乐队制造混乱，或保护撤离队。
危险度/规避：高。戒严会迅速升级；规避沃鲁特本人，以人、文书和记忆链取得条件性胜利。
任务方向/奖励：奖励取向为同盟证词、王都路线、可信联络人和抵抗情报，不给予统治七砦的奖励。

**四阶（宴会夜·七席之前）**

切入身份/时点：乐手、厨役、伪商会代表、外族佣兵看客、侍酒奴；锚定请柬夜至祝酒前。

初始事件：你得到第三席名牌与地契册第 7 页其中之一。梁上弓弦已上。沃鲁特还未入席。乐手看你一眼，鼓槌点了三下——旧日集结号起拍，也是今晚唯一的三十秒预告。

开场白建议：「金殿的香遮不住铁锈。七个位子有人，有旗，有假人。乐手看你一眼，鼓槌点了三下——那是旧日集结号的起拍，也是你今晚唯一的三十秒。」

关键NPC立场：**沃鲁特**要完美仪式。**基恩**要名单读完。**希克斯**要杀起立者。七象征人物要活或要死得有意义。**厨役总管**要孩。商会要地。**奥莉嘉/克洛伊**可成倒戈火花。**大司教**要祝祷不被打断。

主线钩子/支线：主线=劫一人或毁地契或用第三曲开廊；支线①换杯避药；支线②抢家谱页；支线③假抬价搅商会。勿复制三砦神镜任务或月溪名册任务。

危险度/规避：极高；不在致辞三分钟硬刚沃鲁特；注意 A 席人质连坐。游街阶段收益更高但暴露更大。

任务方向/奖励：名牌、地契页、密道图、乐队总谱、一名幸存者指挥权、家谱残页、灭火灰配方。



**四阶补充（宴会夜七席前）**
切入：乐手/厨役/伪商会/外族看客/侍酒奴。
时点：请柬夜至祝酒前。
初始：第三席名牌或地契第7页；弓弦已上。
开场：金殿香遮不住铁锈；鼓槌三下三十秒。
NPC：沃鲁特、基恩、希克斯、七象征、厨役总管、商会、大司教、奥莉嘉、克洛伊。
主线劫人毁契开廊；支线换杯抢谱抬价。
极高危险；注意A席连坐；游街窗高收益高暴露。
奖励名牌地契页密道图总谱指挥权家谱页灭火灰。

执行备忘：先物证后谈判，先人质后名声，先可观察结果后道德宣言。危险度极高。规避沃鲁特或等价军阀本体正战。奖励贴合本阶物证与人情，禁止终局权柄。开场白保持第二人称画面感。关键NPC加粗真名 3～6 人，立场各异可背刺可结盟。
