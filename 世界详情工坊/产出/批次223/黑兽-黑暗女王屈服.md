# 黑兽-黑暗女王屈服
<!--meta lib=主库 tiers=二、四-->

## 剧情

**【作品来源】**
同《黑兽》Liquid 系列。**本档案「黑暗女王屈服」：焦点暗精灵女王奥莉嘉·迪斯科迪亚在黑之城被沃鲁特奇袭拘束、魔力衰败被放大为政权崩溃、魔军被吞编为新魔军的过程；只写王位、种族政治、弱化具与倒戈，不写色情细节。** 与圣女线对位：彼=白之城大脑；本线=**黑之城王座易主**。

**【世界定位】**
奥莉嘉北陆君临数百年，率魔军与七盾同盟长期对峙。魔力衰退被塞勒斯缇娜视为窗口，雇佣黑犬先登。沃鲁特不杀女王，而以弱化具拘束，收编魔军，宣奉仕国家。本线=**王座被佣兵逻辑接管**。

**【世界观 · 力量体系】**
セレヌス大陆西方エオス平原：剑、魔法、神术、亚人血脉与佣兵战术并存，无修仙境界，战力按编制与个人武勇分层。
1. 民兵、溃兵、逃奴≈一阶：单兵难敌正规骑士，聚众可堵粮道。  
2. 精锐士兵、低阶骑士、黑犬佣兵骨干、税关武装≈**二阶**：可守门、劫商队、清扫街垒。  
3. 砦守护者、具名女骑士、虫使精锐、半身人工程战≈三阶边缘：一砦之主级压迫。  
4. 圣女预知全军指挥、佣兵王沃鲁特个人武勇与卡里斯马、破城决战、魔女王残存魔力≈**四阶**：决定战线走向。  
本条目覆盖清单阶位（见 meta）。顶点按**沃鲁特+新魔军/奉仕国家战争机器**取四阶顶；条件性胜利=斩首链、夺人质、断补给、公开内奸证据、倒戈残部，**严禁**「被封印所以战力为零」。

乐园阶位映射：民兵≈一阶；精锐士兵/低阶骑士/佣兵骨干≈二阶；砦守护者/具名骑士≈三阶边缘；圣女预知指挥与佣兵王沃鲁特≈四阶（宁低勿高）。

**【地理 · 舞台】**
黑之城王座厅与地下魔库；城周亚人营地；弱化具安置密室；克洛伊巡逻廊；新魔军整编场；白之城雇佣契约原件存放处（远程政治）。

**【世界剧情线】**
① **衰退与窗口**：奥莉嘉魔力衰，前线收缩。塞勒斯缇娜签雇佣约，沃鲁特率黑犬突入黑之城。  
② **拘束夜**：弱化具上身，女王尚存威仪但无法全开魔力。克洛伊死战护主，战力被数量与战术淹没。  
③ **吞编演说**：沃鲁特对魔军讲「强者逻辑」：旧女王衰，新王给战与粮。种族仇恨被战利品暂时压住。  
④ **屈服的政治含义**：不是内心认输，而是** equip 被卸、印玺被挪、诏令改由佣兵王副署**。暗精灵贵族分裂：殉王派、实用派、逃北派。  
⑤ **对同盟反噬**：新魔军回戈七盾；奥莉嘉被作为「活战利品」稳定魔军中的暗精灵编制。  
⑥ **终局**：夺弱化钥恢复有限魔力并分裂新魔军 / 女王成永久副署印章 / 克洛伊换钥失败双亡象征。条件性胜利看钥与编制，不看决斗神话。

**【主要人物】**
- **奥莉嘉·迪斯科迪亚**（暗精灵女王）｜性格：高傲冷冽、数百年统御习惯。｜装备·能力：魔力（衰退中）+王位法。｜人物弧光：北陆之主→被拘束的合法性符号→（可选）残火复燃。｜立场关系：信克洛伊；恨人类佣兵；对沃鲁特是战术评估非单纯仇恨。  
- **克洛伊**（女王骑士｜混血）｜性格：绝对忠诚、憎人类主奴史。｜装备·能力：近侍剑技。｜人物弧光：护主失败→赎罪突袭。｜立场关系：契约者若人类需先过她的刀。  
- **沃鲁特**｜新王座实际支配者。  
- **希克斯 / 基恩**｜整编与情报。  
- **塞勒斯缇娜·卢克鲁斯**｜雇佣决策的责任方。  
- **暗精灵贵族三派代表**（殉王/实用/逃北）｜编制政治。  
- **魔军兽人千夫长 / 虫使联络**｜整编阻力或助力。  
- **弱化具看守官**｜钥链。  
- **白之城契约书记**｜雇佣约原文。

**【势力图谱】**旧魔军、新魔军（黑犬核心+吞编部）、暗精灵贵族、七盾同盟、奉仕国家宣传。

**【贵重物品】**弱化具与解码钥；黑之城印玺；雇佣契约副本；王座副署笔；克洛伊主从誓环；北陆逃遁星图。

**【隐藏剧情 · 伏笔】**奥莉嘉早知衰退仍拒退位，因退位=贵族内战。塞勒斯缇娜预知「黑犬必胜」却未预知「胜后叛」——或预知而赌。实用派贵族中有人早已通黑犬。

**【大事记时间线】**衰退显化→雇佣→破城→拘束→吞编→反噬同盟→钥战终局。

**【叙事基调 · 雷区】**黑暗王座政治。忌色情；忌把奥莉嘉写成无意志道具；忌二阶取钥成功无代价。

**【弱化具制度】**
具分项、环、座三件，分持才全效。沃鲁特留一件给基恩防叛。夺一件=魔力回一成，夺三件才近战威胁四阶。

**【吞编演说修辞】**
不讲正义，讲「谁能让你们吃肉」。暗精灵听印玺，兽人听粮，虫听战场样本。契约者拆编要对口许诺。

**【女王·印玺副署流程】**
旧制：奥莉嘉亲盖。新制：沃鲁特口授，基恩笔录，印玺由看守官盖。缺口授则印无效——二阶可制造「伪口授」混乱编制。

**【女王·克洛伊巡逻表】**
拘束后克洛伊仍被允许有限巡逻以安抚殉王派。表上空白格是突袭窗，也是诱杀窗。

**【女王·贵族三派会议暗语】**
殉王派「北风」，实用派「热锅」，逃北派「星图」。会议若三词齐出=当夜有刺客。

**【女王·雇佣约原文漏洞】**
约写「攻陷黑之城并解除魔军威胁」，未写「不得建国」。沃鲁特钻的是法条空心。书记留有草案涂改——可作同盟内责圣女的弹药，亦可作反宣传。

**【女王·魔力衰退观测日志】**
宫廷术士曾记「满月衰三成」。日志在地下魔库。沃鲁特按满月排展示，衰时最稳。

**【女王·兽人千夫长价码】**
要「旧女王亲口免罪」才倒戈。没有奥莉嘉声音，粮允诺不够。

**【女王·四阶钥战】**
T-0：三钥分持位置更新。T+2h：若只夺一钥，引发整编场骚乱但难脱。T+8h：逃北派星图窗口关闭。

**【女王·条件性胜利】**
钥≥2 + 实用派公开改旗 + 克洛伊存活（可选加分）。杀沃鲁特非必须。

**【女王·人类契约者门槛】**
克洛伊第一刀试探：报出「非黑犬、非七盾官员」身份并交出武器。硬闯=敌。

**【女王·宣传战】**
奉仕国家称「女王自愿副署」。反证需要印玺泥与口授笔迹不符。

**【女王·地下魔库层级】**
上库存战利品，中库弱化具件，下库北陆秘卷。下库门要女王血或旧印——衰退后血仍可用。

**【女王·塞勒斯缇娜责任】**
圣女线若并行，本线提供「雇佣决策」的黑锅与证据链，但不复制白之城正文。

**【女王·逃北星图】**
逃北派要女王本人或血瓶启动。无女王则星图只是纸。

**【女王·新魔军军纪】**

**【印玺、口授与三派暗语】**
副署印文与旧诏不合处，是伪政权的指纹：旧诏用「血月印」，新副署用「犬首印」叠盖。印泥热度可辨真伪——新印泥松香比例高，冷却慢。实用派暗语「热锅」=今夜议降；「冷锅」=装死；「翻锅」=倒戈杀监。殉王派暗语更短，多为血缘旧诗末字。看守官只关心钥不丢，可用「钥链生锈需打磨」骗取短暂离岗。粮官倒戈条件通常是「孩子先送出城」而非金银。

**【克洛伊作为近侍的双重要求】**

**【二阶与四阶任务边界】**
二阶：盗印泥样、传暗语、伪口授扰乱、偷巡逻表、护送一名逃北孩童。禁止闯整编场中心。四阶：弱化钥争夺、整编场开缺口、与近侍协同夺主短暂离场、烧毁副署名册关键页。顶点沃鲁特仍在，条件性胜利=钥+主声+至少一条北线，不要求复国当日完成。

**【与侍从双堕、七公主宴的接口】**
本档物证是印泥、口授残页、星图；不与双室誓环、宴会地契重复。若战役时间线推进到宴会，奥莉嘉可能被安排贵宾席侮辱——那是另一档案的象征暴力，本档止于整编期屈服政治。隐藏伏笔：某份「屈服诏」其实是奥莉嘉故意露出的伪诏，用来钓出实用派名单；契约者若全信伪诏会把名单交错人。

## 阶位切入点

> 阶位↔：二阶≈魔库杂役/营地斥候；四阶≈弱化钥战与整编场。顶点沃鲁特条件性胜利。

**二阶（印玺·口授·三派）**
切入身份/时点：黑之城被占后的库房搬运、营地译员、逃北派外围；锚定拘束后七日整编期。
初始事件：你搬到一箱印泥，发现副署印文与旧诏不合；实用派联络人塞给你「热锅」暗语，要你今晚选边。
开场白建议：「王座上空着，印却比从前盖得勤。克洛伊在廊上数你的脚步，像数刀。你手里的印泥还热——热的是谁的政权，你得自己闻。」
关键NPC立场：**奥莉嘉·迪斯科迪亚**——要钥与声；**克洛伊**——要主平安；**基恩**——要副署效率；**实用派贵族**——要活路；**殉王派**——要血债；**看守官**——要钥不丢。
主线钩子/支线：伪口授扰乱、盗印泥样、传暗语；支线＝粮官倒戈、星图残页。
危险度/规避：中高；规避整编场阅兵正中。
任务方向/奖励：印泥样、暗语本、巡逻表、贵族人情。

**四阶（钥与编制）**
切入身份/时点：倒戈千夫长联络、旧宫廷术士、同盟奇袭队；锚定满月衰夜。
初始事件：三钥分持图更新，沃鲁特本旗在侧——你只能夺钥不能夺旗。
开场白建议：「满月把弱化环照得像第二道王冠。奥莉嘉的眼睛仍是女王的眼睛，只是魔力浅了。克洛伊说：先钥，后恨。」
关键NPC立场：**沃鲁特**——要活印章；**奥莉嘉·迪斯科迪亚**——要可逆拘束；**希克斯**——要防逃；**克洛伊**——要突袭位；**兽人千夫长**——要免罪声；**塞勒斯缇娜**远程——要赎雇佣之罪的机会。
主线钩子/支线：夺钥≥2、改旗演说、护女王入下库；支线＝炸整编鼓、夺副署笔。
危险度/规避：极高；条件性胜利。
任务方向/奖励：钥、印玺、残部指挥权、星图启动权。
