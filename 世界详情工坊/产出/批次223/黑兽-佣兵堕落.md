# 黑兽-佣兵堕落
<!--meta lib=主库 tiers=四-->

## 剧情

**【作品来源】**
同《黑兽》Liquid 系列。**本档案「佣兵堕落」：焦点四砦女佣兵玛雅——黑犬旧部、对沃鲁特怀武者敬与旧情、因贵族米歇尔·潘蒂埃尔逆恨与卖砦交易而被点名收押的路线；只写佣兵伦理、旧团法、卖砦合同，不写色情细节。** 与其他砦线区分：本线=**佣兵团内部道德破产**。

**【世界定位】**
玛雅曾是黑犬「姐」字辈，离团后接四砦防务。沃鲁特建国后，旧识重逢变成猎捕。米歇尔以「交砦换人」求沃鲁特。本线问：**佣兵的誓是卖给契约还是卖给王？**


**【世界观 · 力量体系】**
セレヌス大陆西方エオス平原：剑、魔法、神术、亚人血脉与佣兵战术并存，无修仙境界，战力按编制与个人武勇分层。
1. 民兵、溃兵、逃奴≈一阶：单兵难敌正规骑士，聚众可堵粮道。  
2. 精锐士兵、低阶骑士、黑犬佣兵骨干、税关武装≈**二阶**：可守门、劫商队、清扫街垒。  
3. 砦守护者、具名女骑士、虫使精锐、半身人工程战≈三阶边缘：一砦之主级压迫。  
4. 圣女预知全军指挥、佣兵王沃鲁特个人武勇与卡里斯马、破城决战、魔女王残存魔力≈**四阶**：决定战线走向。  
本条目覆盖清单阶位（见 meta）。顶点按**沃鲁特+新魔军/奉仕国家战争机器**取四阶顶；条件性胜利=斩首链、夺人质、断补给、公开内奸证据、倒戈残部，**严禁**「被封印所以战力为零」。

乐园阶位映射：民兵≈一阶；精锐士兵/低阶骑士/佣兵骨干≈二阶；砦守护者/具名骑士≈三阶边缘；圣女预知指挥与佣兵王沃鲁特≈四阶（宁低勿高）。


**【地理 · 舞台】**
四砦佣兵营与酒窖；黑犬旧团法石板（残）；米歇尔贵族行馆；卖砦合同签字席；沃鲁特接见旧部时的「怀旧营火」陷阱场。

**【世界剧情线】**
① **旧团岁月**：玛雅与沃鲁特、希克斯、基恩共享过营火与分赃规矩；她离团因「不想建国，只想打仗」。  
② **四砦防务**：她以佣兵方式守砦，效率高、军纪野。  
③ **逆恨合同**：米歇尔多次求欢被拒，转以砦易人。沃鲁特笑收——他要的是「旧部归队」宣传。  
④ **怀旧陷阱**：营火叙旧，合同在背后生效；四砦门开，玛雅成「归队标本」。  
⑤ **堕落定义（制度）**：不是私德审判，而是**她的名字被写上黑犬新编制、旧团法被改成国法**——佣兵自由死。  
⑥ **终局**：撕合同夺砦旗重建自由佣兵连 / 名入新编制成「王的姐」符号 / 杀米歇尔曝合同引发贵族恐慌。

**【主要人物】**
- **玛雅**（四砦女佣兵｜黑犬旧部）核心。  
- **沃鲁特**旧识与猎手。  
- **米歇尔·潘蒂埃尔**卖砦贵族。  
- **希克斯 / 基恩**旧同事。  
- **四砦佣兵副手**可继旗。  
- **旧团法石板看守残兵**。  
- **阿莉西亚 / 克劳迪娅**远程可雇玛雅的客户型人物。  
- **奉仕国家编制官**写新名册者。

**【势力图谱】**四砦佣兵连、黑犬核心、贵族逆恨网、奉仕编制部、潜在雇主砦。

**【贵重物品】**旧团法石板拓片；卖砦合同正本；玛雅离团时的「无旗」徽章；营火陷阱用地图；新编制名册空页。

**【隐藏剧情 · 伏笔】**玛雅离团真正原因含一次「不分给弱者」的内部分赃争执。沃鲁特记得，并在怀旧时提起。米歇尔合同有双份，一份已送大司教派作政治保险。

**【大事记时间线】**离团→守四砦→逆恨→怀旧营火→合同生效→三终局。

**【叙事基调 · 雷区】**佣兵黑色幽默与背叛。忌色情；忌把玛雅写成单纯恋爱脑；仅四阶。

**【旧团法要点】**
战前可退；建国后不许退。玛雅踩的是时间差。拓片可证明「建国非法改法」。

**【怀旧营火编成】**
外圈酒，内圈伏弩，合同在帐篷后。进内圈=政治死亡。


**【佣兵·离团文书】**
旧法要求三人见证。玛雅的见证是希克斯与一名已死团员。死无对证时，编制官可宣称「离团无效」。拓片+酒窖记账可补证。

**【佣兵·卖砦合同条款】**
条款三：交人后贵族得「四砦税一年」。条款七：若人已「自愿归队」则税减半——沃鲁特诱导「自愿」话术的法源。

**【佣兵·营火座位图】**
沃鲁特对面座=尊；侧座=捕。玛雅若知图可拒侧座。图在基恩靴筒。

**【佣兵·副手继旗条件】**
副手继「无旗」需玛雅腰牌或拓片。无则连队散为土匪。

**【佣兵·米歇尔行馆】**
合同正本锁在情书盒——他以为浪漫，实为物证笑柄。二道锁是家族纹，可借贵族仇家开。

**【佣兵·条件性胜利】**
合同正本销毁或改写+四砦旗仍属无旗连+玛雅名未入新名册。

**【佣兵·希克斯私义】**
他可放水一次，价码是「别当众辱王」。玛雅难接受。

**【佣兵·雇主市场】**
阿莉西亚愿出疗伤药箱雇旧情报；克劳迪娅愿出通行符。价码改变她是否肯回战场。

**【佣兵·编制官笔】**
新名册用特种墨，见水不化。泼酒无效，需酸液——酒窖蒸馏可制。

**【佣兵·四阶战力】**
玛雅个人≈三~四阶交界，入编后被「示范旧部」限制行动。胜利靠合同与旗，不靠单挑王。

**【佣兵·宣传稿】**
「姐归队」传单已印。抢印刷所可延迟名册生效三日。

**【佣兵·终局连队】**
撕合同成功则「无旗连」成抗奉仕符号；入编则其他砦佣兵效仿投降。

**【佣兵·与表妹线接口】**
玛雅可卖夜袭习惯，但不在本线展开亲缘戏。

**【佣兵·禁套话】**
不开拍卖假货；开场必须合同/石板/营火座位图。

**【佣兵·时间表】**
