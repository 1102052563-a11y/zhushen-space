# 黑兽-表妹堕落

## 剧情

**【作品来源】**
《黑兽～气高的圣女们白浊地染上～》（黒獣 〜気高き聖女は白濁に染まる〜）Liquid（Nexton）2010年4月23日发售，后有 ERE、黑兽·改、OVA 与续作黑兽II。舞台为セレヌス大陆西方エオス平原。**本档案「表妹堕落」专线：焦点一砦姬骑士阿莉西亚·阿克图尔斯与其二砦表妹普莉姆·菲奥里雷的亲缘链——开城前后表姐的求援、迟援、人质化与家族纹章政治；只写战争、内奸、亲缘伦理与补给，不写色情细节。** 与「公主凌辱」区分：彼写二砦内部商人开城与伪令治理；本线写**一砦—二砦亲缘轴**与阿莉西亚侧的救援失败学。

**【世界定位】**
普莉姆是阿莉西亚的表妹（いとこ），花月姬君以德望维系二砦民心；阿莉西亚以圣アイリス骑士团与剑技守一砦。奉仕国家压境后，モルタデッラ兄弟以普莉姆为筹码开城，黑犬用「让表姐亲眼看见表妹沦为标本」作为瓦解一砦士气的心理战。本线=**亲缘被武器化**。


**【世界观 · 力量体系】**
セレヌス大陆西方エオス平原：剑、魔法、神术、亚人血脉与佣兵战术并存，无修仙境界，战力按编制与个人武勇分层。
1. 民兵、溃兵、逃奴≈一阶：单兵难敌正规骑士，聚众可堵粮道。  
2. 精锐士兵、低阶骑士、黑犬佣兵骨干、税关武装≈**二阶**：可守门、劫商队、清扫街垒。  
3. 砦守护者、具名女骑士、虫使精锐、半身人工程战≈三阶边缘：一砦之主级压迫。  
4. 圣女预知全军指挥、佣兵王沃鲁特个人武勇与卡里斯马、破城决战、魔女王残存魔力≈**四阶**：决定战线走向。  
本条目覆盖清单阶位（见 meta）。顶点按**沃鲁特+新魔军/奉仕国家战争机器**取四阶顶；条件性胜利=斩首链、夺人质、断补给、公开内奸证据、倒戈残部，**严禁**「被重创所以战力为零」。

乐园阶位映射：民兵≈一阶；精锐士兵/低阶骑士/佣兵骨干≈二阶；砦守护者/具名骑士≈三阶边缘；圣女预知指挥与佣兵王沃鲁特≈四阶（宁低勿高）。


**【地理 · 舞台】**
- **一砦阿克图尔斯领地**：圣アイリス骑士团驻地、家族纹章厅、出援整备场、阿莉西亚剑冢。  
- **二砦花月城**：花园议事厅、商队仓库、粮道侧门、陷落后开放城区。  
- **一—二砦驿路**：求援密信必经的三处烽火台（东岗、柳桥、断碑），可被内奸熄灯。  
- **黑犬前哨与移交车队宿营地**：把「表妹」运往展示场的节点。  
- **白之城远程**：塞勒斯缇娜预知闪片可到，援军实体难到。

**【世界剧情线】**
① **亲缘前史**：阿莉西亚视普莉姆为「必须护住的花」；普莉姆尊表姐为「剑与规矩」。两砦民心互相借用——一砦的军纪传说给二砦壮胆，二砦的仁政神话给一砦遮羞。  
② **黑之城「胜利」余波**：沃鲁特拘束奥莉嘉·迪斯科迪亚、吞魔军、宣奉仕国家。七砦连锁承压；一砦大臣贝亚兹利已对阿莉西亚起歹意，二砦商人兄弟同步算盘。  
③ **求援窗打开**：普莉姆察觉商人异常，派学徒携花月密信与半枚家徽印走驿路；东岗烽火「两短一长」=求亲援。阿莉西亚收信即整装，却被贝亚兹利以「一砦不可空」拖延半日。  
④ **开城夜与亲缘断裂**：コンティ与ポンティ·モルタデッラ开侧门交人；黑犬不急屠城，而急着让「表姐的名字」出现在伪令抬头——用亲缘合法性征收。阿莉西亚先锋在柳桥遇伏，死伤的是她最信任的见习骑士。  
⑤ **心理战升级**：前哨队长故意放活一名一砦斥候，令其带回「表妹尚存、但纹章已盖在征收令上」的消息。阿莉西亚若强攻=表妹被转移；若谈判=承认伪秩序。  
⑥ **双线终局可选**：  
  - 救出：夺回普莉姆与原印，一—二砦地下抵抗合流，贝亚兹利账本被公开。  
  - 工具化：普莉姆名号巡展，阿莉西亚剑折于救援途中，一砦随后因大臣通敌沦陷。  
  - 反杀：账本+密信当众对质，兄弟火并，沃鲁特侧近希克斯收编残局——亲缘仍碎，但商人网崩。  
禁止无代价团圆；亲缘一旦被当众工具化，民心不会回到「花月姬与姬骑士」的童谣时代。

**【主要人物】**
- **普莉姆·菲奥里雷**（二砦花月姬君｜阿莉西亚表妹）｜性格：心软、信人、怕连累表姐却更怕领民饿死。｜装备·能力：德望与行政合法性（印信），本人武力有限。｜人物弧光：仁政象征→人质符号→（可选）抵抗旗或空壳名号。｜立场关系：求表姐救又怕表姐为她空砦。  
- **阿莉西亚·阿克图尔斯**（一砦姬骑士｜圣アイリス骑士团长）｜性格：严己宽人、剑重于情却在表妹线破功。｜装备·能力：剑技与骑士团编制，≈三阶边缘个人武勇。｜人物弧光：守护神话→迟援愧疚→复仇或殉道。｜立场关系：对贝亚兹利由信转杀；对沃鲁特是仇敌。  
- **コンティ·モルタデッラ / ポンティ·モルタデッラ**（豚人族宫廷商人兄弟）｜性格：见风转舵、好货色。｜装备·能力：账本、开城钥匙、粮道图。｜人物弧光：内奸成功→分赃内讧风险。｜立场关系：卖表妹换特权。  
- **贝亚兹利**（一砦大臣）｜性格：伪忠、欲。｜装备·能力：调度权、拖延文书。｜人物弧光：用「空砦论」耽误救援，并行出卖阿莉西亚。｜立场关系：与商人网可互通。  
- **沃鲁特**（黑犬佣兵王→奉仕国家）｜性格：好战卡里斯马。｜装备·能力：四阶顶武勇与多族联军统御。｜人物弧光：把亲缘当心理武器。｜立场关系：要标本与士气崩溃，不急杀。  
- **希克斯 / 基恩**（沃鲁特侧近）｜性格：斥候务实 / 参谋冷静。｜装备·能力：情报与军务。｜人物弧光：执行移交与宣传。｜立场关系：对契约者可谈效率。  
- **塞勒斯缇娜·卢克鲁斯**（白之城圣女｜盟主）｜性格：悲悯、预知负重。｜装备·能力：预知指挥（城陷后碎裂）。｜人物弧光：远程闪片预警，实体难援。｜立场关系：嘱阿莉西亚「剑不要只为一人折」。  
- **克劳迪娅·莱凡坦**（六七砦骑士团长）｜性格：护主务实。｜装备·能力：同盟骑士团。｜人物弧光：可派小队协防驿路。｜立场关系：尊重阿莉西亚剑名。  
- **玛雅**（四砦女佣兵｜黑犬旧部）｜性格：强势姐感。｜装备·能力：佣兵战术。｜人物弧光：可卖旧情报给救援队。｜立场关系：敬沃鲁特武，厌其国。

**【势力图谱】**
- **一砦阿克图尔斯—圣アイリス**：军纪门面；内部已有贝亚兹利蛀洞。  
- **二砦花月领民与商会**：仁政外壳；商会被兄弟绑架。  
- **黑犬→奉仕国家**：用亲缘合法性做征收与宣传。  
- **白之城神殿远程**：预知与信仰声援，缺实体兵力。  
- **驿路烽火民兵**：求援生命线，也是熄灯内奸舞台。

**【贵重物品】**
- **花月家徽印（原印/伪印）**：原印=抵抗合法性；伪印=征收令。  
- **半枚纹章密信**：普莉姆求援物证，阿莉西亚腰间另一半可拼合验证。  
- **东岗—柳桥—断碑烽火码本**：两短一长=亲援；三长=已开城勿来。  
- **モルタデッラ兄弟联号账本**：记通敌粮价与买路钱，可引爆军纪审判。  
- **阿莉西亚剑穗信物**：表妹幼时所赠；出现在敌营=羞辱或诱饵。  
- **贝亚兹利拖延文书原件**：证明「空砦论」是人为。

**【隐藏剧情 · 伏笔】**
- 贝亚兹利与商人兄弟早有银线互通，不止「各自通敌」。  
- 塞勒斯缇娜预知中见过「剑穗落泥」却仍雇黑犬——负罪与理性并存。  
- 开城夜放走的儿童里有一人能背全本烽火码，是后期游击通讯核。  
- 沃鲁特不急杀普莉姆，因「活着的表妹」比死更能砍一砦士气。

**【大事记时间线】**
百年对峙 → 雇黑犬破黑之城 → 奉仕宣言 → 二砦求援密信发出 → 贝亚兹利拖延 → 开城交人 → 柳桥伏击 → 伪令盖花月印 → 救援窗开闭 → 救出/工具化/反杀三终局 → 一砦是否连锁沦陷。

**【叙事基调 · 雷区】**
亲缘伦理+驿路战争。忌色情；忌把本线写成纯二砦行政副本（那是公主凌辱）；忌二阶硬刚沃鲁特；忌无代价让表姐表妹双全。最早切入：密信发出夜，或柳桥伏击前六小时。

**【亲缘政治细目】**
花月童谣与アイリス军歌在民心中是一对——童谣停，军歌就哑。黑犬伪令抬头写「奉花月姬君安民」比写「沃鲁特命令」更省箭。阿莉西亚若当众撕伪令，等于宣布表妹已死或已叛；若默认，等于承认亲缘被征用。契约者每一次「公开/隐匿」选择都在改两砦士气曲线。

**【驿路作战卡】**
方案A：护密信原件到一砦，换阿莉西亚精锐夜袭粮道，不碰主城。  
方案B：伪造「三长」烽火骗前哨放松，真队走侧门夺印。  
方案C：公开贝亚兹利文书，逼一砦内清理后再出援。  
方案D：买通庞提弃康提，用分赃裂痕换移交延误。  
二阶铺垫（熄灯、账本、证人），四阶收束（夺车队、会剑姬）。

**【与公主凌辱边界】**
不复用「占领区日常税关美学」为主轴；本线主轴=表姐迟援、烽火码、剑穗信物、贝亚兹利空砦论。商人开城是背景因果，不是正文唯一Boss戏。

**【可介入事件（本线专属）】**
1. 东岗守烽火兵被买通熄灯——你可重燃或改码。  
2. 学徒携带半印倒在柳桥——救人或夺信。  
3. 贝亚兹利要你在文书上副署「不宜出援」——签或不签。  
4. 前哨放回斥候，袋中有剑穗——真伪鉴定。  
5. 花月儿童背码本求加入一砦——收编或藏匿。  
6. 四阶夜：移交车队三岔口，阿莉西亚正门吸火，你走河谷。

**【终局后亲缘状态】**
救出：两砦旗并列成抵抗符，阿莉西亚剑名染血更重。工具化：童谣变禁歌，一砦士气雪崩。反杀：商人网崩，亲缘创伤仍在，可挂钩圣女线信仰分裂但不抄正文。


**【表妹线·烽火台制度】**
东岗管「亲援」码，柳桥管「遇伏」码，断碑管「勿来」码。三台须同日校准油料；内奸只需买通一台即可让密信变废纸。契约者若任职见习烽火手，第一夜就会被塞两份互相矛盾的油票。

**【表妹线·纹章拼合术】**
半枚花月印与阿莉西亚腰间半枚必须以特定角度拼合，火漆才显「亲援有效」。伪印拼合会渗黑——这是老管家设的防伪，黑犬仿制时漏了这一步。二阶任务常是「在火边验证渗色」。

**【表妹线·空砦论会计】**
贝亚兹利用「一砦库存粮仅支十二日」「圣アイリス不可分兵」两张表拖救援。表中有三行被改过日期；契约者若会算术，可在议事厅当众拆穿，换半日行程。

**【表妹线·柳桥伏击编成】**
伏击队含黑犬骨干一个小队+二砦伪民兵向导。向导脖子上有商会烙印。击毙向导可减后续开城效率；留活可问出侧门班次。阿莉西亚先锋若不知向导身份会先救人再辨敌——这是她的性格破绽。

**【表妹线·伪令抬头战】**
占领后第一道令抬头必须出现「花月」二字，否则领民不交粮。宣传官与希克斯为此吵过：前者要仪式，后者要速度。契约者可伪造抬头错误版制造抗税窗口。

**【表妹线·剑穗诱饵识别】**
真剑穗内侧绣极小「ア」字（阿莉西亚幼时针脚）；假穗用机绣。带回斥候的人若强迫你「立刻给姬骑士看」，多半是要拍摄她崩溃表情作传单。

**【表妹线·儿童口述网】**
开城夜出逃儿童以井边、面包炉、旧学堂三角传话。他们不认军阶，只认「谁还敢唱花月童谣」。二阶融入靠童谣后半段接唱，接错会被当作奸细。

**【表妹线·一砦食堂政治】**
阿莉西亚出援前后，食堂分成「救表妹桌」与「守一砦桌」。贝亚兹利的人给后者加肉。肉的来源账可查到商会——又一条银线。

**【表妹线·四阶移交时刻表】**
T-12h：伪印盖征收令。T-6h：普莉姆转移至车队中厢。T-2h：剑穗传单散发一砦外围。T-0：三岔口分兵。T+4h：若未截，进入「展示巡回」不可逆。

**【表妹线·条件性胜利定义】**
救表妹≠杀沃鲁特。成功标准：原印到手、人离开展示半径、贝亚兹利文书公开至少一项。三者占二即算阶段胜；硬闯中军帐取沃鲁特首级=任务失败模板。

**【表妹线·与圣女远程】**
塞勒斯缇娜预知闪片显示「桥断、穗落、印歪」三镜头，无时间码。神殿书记把镜头画成三张卡，分给可能出援的骑士。契约者若持有「印歪」卡，可在伪令现场对质。

**【表妹线·玛雅情报价码】**
玛雅卖「旧黑犬夜袭习惯」：喜攻侧翼、轻视童谣民心。价码是一砦库存的一箱疗伤药或对沃鲁特「仍称佣兵王而非国王」的口头承诺——她要的是身份尊严。

**【表妹线·克劳迪娅协防条款】**
六七砦可借一个小队守断碑烽火，条件是阿莉西亚书面承认「亲援优先不代表放弃圣女护卫链」。文件本身会成为日后政治把柄。

**【表妹线·领民三派】**
保护派藏姬君旧物；告密派换粮；分润派给商人跑腿。三派人数随伪令是否「盖歪」波动——盖歪日保护派增。

**【表妹线·终局审查】**
若救出，白之城会问阿莉西亚：「你为表妹分兵时，是否预见一砦风险？」答「是」则记功也记过；答「否」则被视作失格骑士。契约者证词可改判决。

## 阶位切入点

> 阶位↔原作：二阶≈精锐士兵/驿路民兵/税关武装；四阶≈破城精锐与沃鲁特阴影。顶点佣兵王=条件性胜利（夺人/断链/公开文书），严禁宣称佣兵王战力可无条件归零。

**二阶（密信·烽火·空砦论）**
切入身份/时点：契约者以驿路见习烽火手、花月学徒护卫、一砦文书副署生切入；锚定普莉姆密信发出夜至柳桥伏击前。
初始事件：你拿到半枚花月印与「两短一长」码本抄页，同时贝亚兹利派人要你在「不宜出援」文书上按手印——按或不按，都会在天亮前被两边记名。
开场白建议：「东岗的油味盖过花香。学徒把半枚印塞进你掌心，说表妹还在等表姐的剑。身后大臣的墨已干，只差你的指印。你选哪边的天亮。」
关键NPC立场：**普莉姆·菲奥里雷**——要亲援不要空话；**阿莉西亚·阿克图尔斯**——要真印与路线；**贝亚兹利**——要你副署拖延；**コンティ/ポンティ·モルタデッラ**——要密信消失；**花月学徒**——要活着送到；**东岗烽火长**——要油与命。
主线钩子/支线：护密信到一砦、重燃被熄烽火、当众拆穿空砦表；支线＝儿童口述网、伪印渗色鉴定、剑穗真伪。
危险度/规避：中；规避沃鲁特本体，规避在柳桥正中喊「我是乐园来的」。
任务方向/奖励：半印、码本、贝亚兹利文书抄件、民心接点、阿莉西亚短期信任。

**四阶（柳桥后·移交截击）**
切入身份/时点：残存骑士、圣アイリス特遣、玛雅牵线的旧佣兵、魔军倒戈联络人；锚定开城后移交车队夜。
初始事件：斥候回报「表妹在中厢、剑穗传单已发、沃鲁特旗在侧翼」——你必须在正门吸火与河谷截击之间拍板，并决定是否让阿莉西亚知道传单内容。
开场白建议：「三岔口的尘里混着花月伪印的火漆味。阿莉西亚的剑穗出现在敌方传单上，像一记耳光。号角响起时，你听见她问你：先救人，还是先撕那张纸。」
关键NPC立场：**沃鲁特**——要亲缘崩溃标本；**阿莉西亚·阿克图尔斯**——要人不要神话；**普莉姆·菲奥里雷**——要停用伪印；**希克斯/基恩**——要准时移交；**玛雅**——要价码结清；**前哨队长**——要活口绩效。
主线钩子/支线：河谷夺厢、公开联号账本逼火并、护送原印回地下；支线＝炸侧翼补给、夺剑穗底版、策反伪民兵向导。
危险度/规避：极高；条件性胜利截人断链，严禁宣称沃鲁特被重创可斩。
任务方向/奖励：原印、账本、移交时刻表、两砦抵抗旗、短暂指挥权（互斥项：曝光贝亚兹利会失去一砦部分补给）。

**二阶补强（表妹专属物证）**
开局必须出现：半枚花月印 / 烽火码本 / 空砦论涂改表 三者之一。禁止通用「拍卖护送假货」开场。

**四阶补强（亲缘截击细则）**
T-6至T+4每小时更新：车队位置、传单覆盖村数、阿莉西亚怒值（影响她是否听劝）。失败即时代价：表妹转移展示巡回、一砦提前遭贝亚兹利开城。


**二阶执行备忘（表妹）**：先验半印渗色，再进一砦议事厅；先改烽火油票，再谈出援。失败写进当夜烽火日志。
