# 正义联盟（扎导版）

## 剧情

**【作品来源】**
《正义联盟（扎导版）》（Zack Snyder's Justice League，通称扎导剪辑版 / Snyder Cut）为华纳兄弟与 HBO Max 于 2021 年 3 月 18 日上线的导演剪辑长片，片长约 242 分钟，R 级。导演扎克·施奈德（Zack Snyder），编剧克里斯·特里奥（Chris Terrio）等，配乐 Tom Holkenborg（Junkie XL）。本片是对 2017 年院线版《正义联盟》的全面重构：不采用乔斯·韦登补拍素材，恢复并扩展角色弧光、阿波柯利普斯神话、达克赛德与反生命方程线，并显著抬升维克多·斯通（钢骨）的戏份。前史紧密衔接《蝙蝠侠大战超人：正义黎明》（2016）中超人战死、莱克斯·卢瑟警告「他来了」、以及布鲁斯·韦恩收到「你不是一个人」的信号。文风：史诗冷峻、神话史诗感与丧亲救赎并存，战斗残酷，母盒与新神科幻—神话混血。本档案**以扎导剪辑版正典为唯一正文准绳**，院线版差异仅作对照注释，不把整个 DCEU 后期乱线灌入。

**【世界定位】**
现代地球在超人之死后陷入真空。阿波柯利普斯的新神将军斯泰本沃尔夫为赎罪，率副魔大军前来夺取三枚母盒，企图完成「合一」（The Unity），把地球改造为阿波柯利普斯的副本，并向主君达克赛德呈上刻写在地表的「反生命方程」。布鲁斯·韦恩与戴安娜·普林斯（神奇女侠）招募巴里·艾伦、维克多·斯通、亚瑟·库瑞，并最终复活克拉克·肯特，结成正义联盟。一句话：这是「丧神之后人类被迫成神、在母盒合一前夜拼凑出地球最后防线」的神话超英世界；契约者是变数，不是天选免死金牌。

**【世界观 · 力量体系】**
力量本源并不统一，而是多层叠合：①氪星人在黄太阳下的细胞能量转化；②奥林匹斯旧神血脉与亚马逊、亚特兰蒂斯的神裔体质；③速度力（Speed Force）作为时间—动能的超维介质；④母盒（Mother Box）作为阿波柯利普斯的「变化引擎」，可按主人意志重排物质与生命；⑤新神（New Gods）层级的生理与军械，远超地球常规物理。死亡规则：凡人与多数变种一死即终；母盒可在极端条件下重建肉体（维克多）甚至复活死者（克拉克），但有代价、有失控风险、且不可无限复用；新神可被斩首、肢解，但阿波柯利普斯的统治结构不会因一名将军阵亡而崩。特殊系统：母盒「合一」可瞬间摧毁并异形改造行星；反生命方程可抹除自由意志、奴役全宇宙有感生命；砰管（Boom Tube）实现跨星际传送。

战力分层（以破坏力与战场地位写，不套修仙境界名）：
1. **凡人军警 / 特工 / 科学家**：热武器与组织力。毁门、清街区边缘；对副魔是炮灰。乐园观感≈一阶。
2. **精英特种 / 黑市武装 / 莱克斯科技安保**：训练与装备拉满，仍惧变种。乐园观感≈一~二阶。
3. **亚马逊精锐 / 亚特兰蒂斯卫队 / 早期变种雏形**：冷兵器开坦克级、水压与神裔力量；可守一城门一时。乐园观感≈二~三阶。
4. **正义联盟核心（戴安娜、亚瑟、巴里、维克多、布鲁斯以谋略与装备补足）与斯泰本沃尔夫级新神将军**：毁据点、掀建筑群、硬抗常规导弹与副魔潮、近战近乎瞬移级；一城仅数人。**乐园观感≈四阶（质变分水岭·精英 Boss）**——本世界开放切入的唯一阶位。
5. **黄太阳下的完全体超人（克拉克·肯特）**：可单方面碾压斯泰本沃尔夫、高速星际级机动雏形；在本片中是「联盟质变钥匙」，破坏力已摸城市—地区级边缘，但仍受剧情与同伴协作约束。乐园观感≈四阶顶尖至五阶门槛（本世界不开放五阶切入，写正文宁低勿高，作「同阶顶尖+」处理）。
6. **达克赛德本体、完整反生命方程、三母盒成功合一后的行星改造**：跨世界、跨意志的规则级权柄。乐园观感＝**超阶巅峰**。低阶切入一律规避本体；对达克赛德与反生命只允许情报、布局、条件性干扰，严禁「被封印削弱战力归零」式解释。

乐园阶位映射（对照《阶位战力图鉴》破坏力，宁低勿高）：凡人~军警≈一阶；特种与初级神裔卫队≈二阶；亚马逊/亚特兰蒂斯主力与未成团变种≈三阶；正义联盟核心成员与斯泰本沃尔夫·副魔军团指挥层≈**四阶**；完全体超人≈四阶顶尖（边缘触五，本世界不单开五阶）；达克赛德 / 反生命方程 / 母盒完全合一＝**超阶**。世界顶点忠于原作真实顶点，不许压低；四阶切入时达克赛德仅以砰管映像、梦魇预兆、间接命令出现。

**【地理 · 舞台】**
- **哥谭 / 韦恩庄园废墟 / 蝙蝠洞**：布鲁斯的基地与罪疚舞台；后半决定把废墟庄园改建为联盟总部。氛围：雨、蝙蝠信号、富人废土。
- **大都会 / 星球日报 / 超人纪念碑**：克拉克的人间身份与公众信仰中心；超人之死在此留下空洞。
- **塞米斯基拉（Themyscira）**：亚马逊岛国，守护一枚母盒；斯泰本沃尔夫第一击发生于此。女王希波吕忒的王座与悬崖神殿是史诗入口。
- **亚特兰蒂斯海域**：第二枚母盒所在；梅拉与守卫、亚瑟的半血身份冲突在此爆发。
- **S.T.A.R. 实验室（中城）**：人类保管/研究第三枚母盒；赛拉斯·斯通的工作地，维克多重生与人质危机的核心。
- **克里普顿侦察舰 / 创世舱（Genesis Chamber）**：复活超人的科技圣所；羊水池与母盒能量交汇处。
- **斯莫尔维尔·肯特农场**：克拉克记忆与人性的锚点；露易丝·莱恩在此唤醒他。
- **波扎尔诺夫（Pozharnov，俄罗斯废弃核城）**：斯泰本沃尔夫建要塞、三母盒合一的终局战场；辐射、废墟塔楼、砰管裂口。
- **阿波柯利普斯（Apokolips）**：达克赛德的工业地狱行星，仅通过砰管与映像现身；火海、熔炉、副魔生产线。低阶只作传说与噩梦。
- **速度力（Speed Force）**：巴里·艾伦进入的橙红能量维；可局部逆转关键瞬间，是终局翻盘的钥匙。

AI 写场景先定契约者在哪一层：街头与实验室是人情与科技；塞米斯基拉与深海是神话；波扎尔诺夫是末日；阿波柯利普斯与反生命是超阶棋盘。本地人若听闻「单人硬刚达克赛德」，会当疯子——这是正确反应。

**【世界剧情线】**
① **前史·第一次入侵与母盒三分（数千年前）**  
尚名乌克萨斯（Uxas）的年轻达克赛德亲率副魔与三枚母盒入侵地球，企图以合一改造行星并夺取刻在地表的反生命方程。地球史无前例地结成联军：旧神（宙斯、阿瑞斯、阿尔忒弥斯等）、亚马逊、亚特兰蒂斯人、人类诸王军队，以及绿灯侠亚兰·古尔（Yalan Gur）。阿瑞斯重创乌克萨斯肩部，合一被打断，入侵军溃退。三枚母盒被留下并分开守护：一归塞米斯基拉亚马逊，一归亚特兰蒂斯，一归人类。达克赛德此后在无数世界征战中一度迷失地球方位，却从未放弃反生命。名场面：神与人并肩的史诗会战、宙斯之力撕裂天空、绿灯构建物被撕碎、年轻暴君带伤遁走。因果：地球成为「唯一打退过达克赛德」的原始星球；母盒休眠等待唤醒信号。

② **近代·人类母盒与世界大战阴影**  
人类一枚母盒在近代辗转：战争年代被多方觊觎，最终落入美国防务与科研体系，编为政府物件，沉睡于实验室。它不可测年、能量不稳，被当作禁忌标本。此阶段世界状态：神话退入传说，科学自负地以为能「保管」神明遗物。

③ **《蝙蝠侠大战超人》余波·超人之死与母盒苏醒（直接前史）**  
克拉克·肯特与布鲁斯·韦恩的冲突、莱克斯·卢瑟制造末日怪兽、超人与怪兽同归于尽——超人之死成为全球信仰真空。三枚母盒同时苏醒，如节日灯火般亮起，向宇宙发出「地球无守护神」的信号。莱克斯在狱中对布鲁斯暗示「他来了」；布鲁斯收到来自未来的警告与「你不是一个人」的联络。布鲁斯开始带着罪疚与恐惧寻找其他变种。世界状态：英雄神话崩塌，媒体与政府各说各话，普通人仍上下班，但天空已在换气。

④ **塞米斯基拉失守·第一枚母盒（开篇）**  
斯泰本沃尔夫经砰管降临塞米斯基拉。他是达克赛德的叔父兼将军，曾背叛主君，被罚征服十万（或数万，片中强调「只剩最后几颗」）世界方可归乡；地球是他赎罪清单上的关键一颗。副魔潮淹没亚马逊防线，斯泰本沃尔夫以战斧与盔甲碾压精锐，夺走亚马逊母盒。女王希波吕忒拼死传信给女儿戴安娜：敌人来了，且与古老噩梦同源。名场面：悬崖火光、亚马逊骑兵与斧光对撞、母盒被强夺时的低频轰鸣、希波吕忒望向人界的决绝。因果：戴安娜走出半隐居，主动联络布鲁斯。

⑤ **招募线·布鲁斯与戴安娜拼凑联盟**  
布鲁斯以「地球需要一支队伍」为题，戴安娜以亲历战争的清醒配合。两人分工：布鲁斯负责科技、资金、情报网与「找人」；戴安娜负责神话解读、武力背书与对亚瑟、维克多的说服。  
- **巴里·艾伦**：中城少年天才，父亲亨利·艾伦蒙冤入狱，母亲命案是他心中刺。布鲁斯以「你想过得更快吗」与父亲线索打动他；巴里几乎是唯一主动兴奋入队者，带来速度力与幽默缓冲。  
- **亚瑟·库瑞**：半人类半亚特兰蒂斯，拒绝王座与人界责任，酗酒渔民外表下是王者血脉。初拒招募；直到亚特兰蒂斯母盒告急。  
- **维克多·斯通**：大学橄榄球明星，车祸重创后被父亲赛拉斯用母盒与义体重建，成为半机械生命。恨父亲「没问过他想不想活成这样」，自我封闭，却是唯一真正「听得懂」母盒语言的人。  
阶段主题：**没有超人，凡人与半神必须学会站在一起。** 契约者若在此阶段介入，最自然的位置是情报掮客、S.T.A.R. 外围、哥谭黑道与军方的夹缝——不是一上来与斯泰本沃尔夫对斧。

⑥ **亚特兰蒂斯失守·第二枚母盒**  
斯泰本沃尔夫突袭深海金库，屠杀守卫，夺下第二枚母盒。梅拉以水力与矛对抗，亚瑟被迫应战并最终选择站到人界一侧。两枚母盒入手后，斯泰本沃尔夫在合一预演中窥见地表上的反生命方程，立刻通过德萨德向达克赛德报喜：他找到了「那颗曾经反击的原始星球」，反生命就刻在地球表面。达克赛德兴趣被重新点燃，下令完成合一后他将亲临。世界状态：神话战争从传说变成新闻与失踪人口；军方开始捕获副魔样本却无力理解。

⑦ **哥谭港之战与人质·维克多入队**  
副魔为猎取第三枚母盒绑架 S.T.A.R. 人员，包括赛拉斯·斯通。联盟初阵在哥谭港下水下—码头混战：戴安娜开路，巴里救人，布鲁斯以载具与战术支撑，维克多被迫出手展露机炮与系统入侵，亚瑟自深海杀出救援。此战确立「五人尚未成军、但已能并肩」的雏形。维克多取回自己藏匿的人类母盒，并向众人解释：母盒是「变化机器」，无善恶，只执行主人意志——它曾拆开又重组他的身体，故也可能复活克拉克。

⑧ **复活超人·创世舱与失控**  
联盟掘出克拉克遗体，置入克里普顿侦察舰创世舱，以人类母盒与巴里的速度力电荷启动重生。克拉克苏醒时记忆空白，以本能攻击「威胁」，与戴安娜、亚瑟、巴里、维克多、布鲁斯苦战，险些酿成二次灾难。露易丝·莱恩赶到，以情感与名字将他拉回；一行转至斯莫尔维尔肯特农场，玛莎·肯特与土地记忆完成人性复位。克拉克稍后换上黑色战衣，象征哀悼与重生并存。代价线：复活过程暴露母盒位置与能量签名；赛拉斯为让维克多能追踪母盒，以实验室高能激光超载加热母盒并牺牲自己——父爱以「科学殉道」完成，维克多的恨与爱同时烧穿。

⑨ **第三枚母盒失落·波扎尔诺夫要塞**  
斯泰本沃尔夫趁乱取走最后一枚母盒，在俄罗斯废弃核城波扎尔诺夫建起要塞与合一祭坛，副魔如蚁。达克赛德、德萨德通过砰管监视。联盟五人（超人尚未归队）强攻要塞：布鲁斯驾驶飞行载具开路并承受火力，戴安娜斩副魔潮，亚瑟控水与近战，巴里救人与清场，维克多尝试接入合一中的母盒系统。名场面（卡片锚点向）：地下—要塞通道中斯泰本沃尔夫 impale 与斧压，戴安娜与亚瑟难以破防，布鲁斯以夜行者战车苦苦支撑，超人尚未抵达的绝望窗口——这是本世界「四阶精英 Boss 战」的标准气压。

⑩ **合一边缘·速度力逆转与斩首（终局战）**  
克拉克及时赶到，以绝对力量压制斯泰本沃尔夫，但合一仍启动：大地异形、建筑崩解、生命被改写。巴里冲入速度力，局部逆转关键数秒，为维克多充能并打开窗口；维克多以自身为接口侵入母盒网络，在「机器想成为人、人想成为机器」的意志对撞中分开三盒。联盟合力击杀斯泰本沃尔夫（扎导版更残酷：戴安娜斩击、亚瑟穿胸、最终斩首），尸身被抛入通向阿波柯利普斯的砰管。达克赛德首次以完整形貌现身映像中，得知母盒尽毁后宣称将以「古老方式」——舰队与战争——重返地球，因为反生命仍在。德萨德确认失败。阶段结局：地球暂时得救；真正的战争才刚被点名。

⑪ **余波·联盟定型与人间线**  
- 布鲁斯与戴安娜决定以韦恩庄园废墟为联盟基地，阿尔弗雷德·彭尼沃斯提供后勤与毒舌稳定器。  
- 克拉克回归大都会双面人生；露易丝暗示有孕，人间锚更深。  
- 亚瑟与梅拉、沃尔科会谈后去见生父，半血王者开始承担责任。  
- 巴里获得中城警局工作，向狱中父亲报喜，速度第一次用来「过正常生活」。  
- 维克多接受赛拉斯留下的信息，拥抱「成为桥梁」的使命，不再只当事故残骸。  
世界状态：公开层面「有一队人在守护」；暗地里砰管裂痕与副魔残骸仍在被掩盖。

⑫ **尾声钩子·卢瑟、丧钟、梦魇与火星猎人**  
莱克斯·卢瑟越狱，在游艇上会见斯莱德·威尔逊（丧钟），揭露布鲁斯·韦恩即蝙蝠侠——为后续「英雄互猎」埋雷。布鲁斯梦见「梦魇未来」（Knightmare）：达克赛德以反生命奴役超人，亚瑟、戴安娜、露易丝等陨落，废土上布鲁斯率领残存者（含巴里、维克多、梅拉、丧钟甚至小丑）反抗；这是**条件性未来**，不是已发生的定局，却是反生命一旦落地的真实预演。片末，卡尔文·斯旺维克显现真身火星猎人（Martian Manhunter），感谢布鲁斯集结队伍，并承诺协助备战达克赛德。终局气质：**胜仗只是序章；地球坐标已被标记。**

⑬ **蝴蝶效应边界（契约者须知）**  
可改局部：救下某名亚马逊/实验室员、推迟某次副魔袭击、帮巴里带一封家书、协助赛拉斯争取时间。不可无代价改写：三母盒最终必须被威胁到「合一边缘」（否则达克赛德线不成立）、克拉克复活的伦理代价、斯泰本沃尔夫作为四阶门神的压迫感、达克赛德与反生命的超阶地位。若契约者提前销毁单枚母盒，剧情会以「另一枚加速苏醒 / 砰管提前 / 人质升级」回流；若试图单挑达克赛德本体，世界应以「映像一瞥即灵魂重压、肉体非同阶」回应，而非削弱暴君。

- **正义联盟（初建）**：布鲁斯、戴安娜、克拉克、巴里、维克多、亚瑟。宗旨——在超人真空期及达克赛德阴影下守护地球。地盘——哥谭—大都会轴心，基地拟设韦恩庄园。战力——四阶精英团；缺一不可的协作型。

**【贵重物品】**
- **三枚母盒（Mother Boxes）**：阿波柯利普斯「变化引擎」。分置亚马逊、亚特兰蒂斯、人类；可重排物质、疗愈/改造生命、开启砰管相关能量；三者合一＝行星级异形改造并方便收割反生命。苏醒信号＝超人之死。终局被维克多与联盟拆解/毁用，迫使达克赛德改采舰队入侵。谁想要：斯泰本沃尔夫（赎罪）、达克赛德（征服）、赛拉斯/维克多（救人与理解自我）、军方（失控的武器梦）。契约者触碰单盒＝被四方锁定。

- **三叉戟（亚瑟）**：亚特兰蒂斯王权与水战核心。
- **蝙蝠装与夜行者等载具（布鲁斯）**：凡人对抗神战的工程学答案；隧道战与要塞突防的关键。

每件写清归属与抢夺后果：母盒与反生命是剧情核弹；其余是四阶战争的合法战利品与剧情道具。

**【隐藏剧情 · 伏笔】**
- **反生命在地表**：地球不是普通征服目标，而是方程载体；这也是古战达克赛德为何亲征。
- **梦魇未来（Knightmare）**：反生命成功后的废土时间线——超人被奴役、联盟残存。它是警告不是定局；露易丝之死是该线的关键扳机之一（与前作梦境互文，点到为止）。
- **火星猎人潜伏**：斯旺维克真实身份揭示，说明地球早有「外神侧观察者」；后续抗达克赛德战的暗线盟友。
- **莱克斯×丧钟**：蝙蝠侠身份公开化，为「英雄内战/猎杀」留口，不在本片内完成。
- **维克多与母盒的共生**：他是人与阿波柯利普斯科技的活体接口，既是钥匙也是隐患——若心志被夺，合一可被反向利用。
- **绿灯侠亚兰·古尔的战死**：宇宙警务力量曾介入古战却失败，暗示地球级危机超出常规灯戒编制。
- **跨作互文边界**：可轻触《蝙蝠侠大战超人》的罪疚与梦境，**禁止**把未在扎导版落地的整条 DCEU 后期剧情（院线混乱续作等）倒灌为本世界「已发生史」。

**【叙事基调 · 雷区】**
基调：神话史诗的沉重、丧亲与救赎、团队从「不信任的强者拼盘」到「愿意为彼此死」。战斗血腥、节奏沉缓而镜头郑重，允许冷幽默（多来自巴里与阿尔弗雷德），拒绝院线版轻飘段子感。  
忌：把扎导版写成 2017 院线喜剧拼盘；忌削弱斯泰本沃尔夫为「只会喊口号的杂兵」；忌让达克赛德开场就被凡人近身砍伤；忌用「任务限制/封印」解释达克赛德不强；忌复活克拉克无代价、无失控；忌把维克多降级为纯炮台；忌捏造原作没有的神器名与成员死亡。  
最早可切入锚点：超人之死后、塞米斯基拉失守前夜；或哥谭港人质危机前；四阶标准锚点为「要塞战前夜、超人尚未归队」。

## 阶位切入点

阶位↔原作战力：四阶≈正义联盟核心成员与斯泰本沃尔夫级新神将军（精英 Boss·毁据点/掀建筑群/硬抗常规热武器与副魔潮）；凡人军警≈一阶、神裔卫队≈二~三阶本世界不单开切入。世界顶点达克赛德与反生命方程、母盒完全合一＝**超阶巅峰**，四阶切入一律「规避顶点·情报优先 / 条件性胜利」，严禁硬刚本体，严禁用「任务压峰值/战力归零」式借口解释顶点。

**四阶（正义联盟集结期 ~ 波扎尔诺夫合一前夜 · 母盒争夺与成军）**

切入身份/时点：契约者以「国际安保承包商 / S.T.A.R. 外围顾问 / 哥谭—大都会灰色情报人 / 深海考古与军方联络的夹缝专家」等四阶战力身份切入；锚定时点优先三选一——①塞米斯基拉失守后、戴安娜刚联络布鲁斯时；②哥谭港人质危机当夜前后；③人类母盒用于复活计划已定、要塞方位尚未锁定时。禁止开局就站在达克赛德王座前。

初始事件：你在大都会地下管线或哥谭港货运区截获「会飞的虫形生物」残骸与一段低频信号，信号与 S.T.A.R. 封存物「6-19-82」频谱同构；同时韦恩企业壳公司发出高价「非官方征募」——要的不是杀手，是「能在神级冲突边缘活着带出人质与数据的人」。第一夜：副魔突袭实验室外围，**赛拉斯·斯通**的团队撤离路线被掐，**维克多·斯通**在监控里沉默不语，**布鲁斯·韦恩**的声音从加密频道传来：「如果你能活着进来，就别再装普通人。」你必须在「救人 / 抢频谱核心 / 把信号卖给莱克斯残党」之间做一次立刻见血的抉择。

开场白建议：「超人为这座城市死过一次以后，夜晚就变得很诚实。你在锈蚀的集装箱顶看见绿色的东西撕开铁皮，像昆虫，又像装甲；耳机里有人用钱和恐惧同时说话，叫你带一份‘会唱歌的方盒数据’出来。远处蝙蝠标没有亮，但你知道有人已经在屋顶上把你量过一遍。别抬头看阿波柯利普斯——你还没资格死在真正的神面前。先活过这条隧道。」

关键NPC立场：
- **布鲁斯·韦恩**：把你当可控资产与潜在叛徒双重审查；给装备与任务，收生物特征与把柄；你若救平民会加分，你若碰母盒本体会变猎杀目标。
- **戴安娜·普林斯**：以战士礼问动机；认可勇气，厌恶拿神器做生意；可在战场临时结盟。
- **维克多·斯通**：对「把我当接口」极度敏感；尊重真正把赛拉斯当人看的外援；可给系统权限碎片。
- **巴里·艾伦**：最易成为人情入口；会拉你救人，也可能因心软把你暴露在副魔视线里。
- **亚瑟·库瑞**：初期不信任西装与「地表人」；你若在深海或港口帮过亚特兰蒂斯一侧，他会以行动而非口头认账。
- **斯泰本沃尔夫**：不与你谈判；你出现在母盒半径内＝转化或碾死；四阶门神，须联盟协作或极端布局才有条件性胜机。
- **赛拉斯·斯通**（存活窗口内）：可给科学同盟；他的殉道线一旦启动，你只能争「多抢几十秒」。
- **莱克斯·卢瑟**（间接）：买你的情报、卖你的位置；绝非盟友。

主线钩子/支线：
- 主线：介入「三母盒争夺—复活超人—波扎尔诺夫拆合」链条，在隧道/港湾/要塞三场关键战中提供四阶战力缺口（掩护、破防、救人、电子战协同），推动联盟成军而不取代原作核心抉择。
- 支线A：协助亨利·艾伦案的外围证据流转，换取巴里的速度护送与信任。
- 支线B：在塞米斯基拉难民/伤员通道上帮希波吕忒传出第二封密信，换亚马逊短期庇护。
- 支线C：从副魔转化坑中抢回被茧化的科学家，获取砰管残留坐标，却引来德萨德侧的远程注视（情报级，非本体降临）。

危险度/规避：危险度＝**高至极高（精英 Boss 战）**。必须规避：**达克赛德本体、反生命方程的主动解读/试用、三母盒强行合一实验**。斯泰本沃尔夫可战但须条件（地形、队友、母盒干扰、超人窗口）；对达克赛德仅允许映像压力下的撤离与情报。**超阶巅峰：存在·情报优先/条件性胜利**。副魔潮可消耗弹药与理智；被茧化≈角色死亡或非人化。

任务方向/奖励：目标自然落在「护送 S.T.A.R. 撤离、夺取副魔指挥频段、为复活舱争取冷却时间、要塞外围开路、保护露易丝抵达农场」等。奖励取向：阿波柯利普斯合金碎片与副魔生物材料（需隔离）、韦恩定制四阶作战装与载具权限、维克多开放的只读系统接口、亚马逊或亚特兰蒂斯的短期通行徽记、关于反生命「存在于地表」的残缺星图（情报，非方程本身）。禁止发放完整母盒或反生命方程实体作为通关奖。
