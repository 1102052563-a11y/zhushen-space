# 对魔忍：红
<!--meta lib=主库 tiers=一、二、三、四、五-->

## 剧情

**【作品来源】**
《对魔忍：红》（日文：対魔忍 紅／TAIMANIN KURENAI）为日本成人游戏品牌 BLACK LiLiTH 于 2015 年 9 月 25 日发售的 18 禁视觉小说，属 Lilith「对魔忍」系列通算第 15 作、以心愿寺红为主角的独立篇章。企画含笹山逸刀斎（世界设定监修）与饭田和彦；原作·剧本饭田和彦；原画 ZOL；监督春山武蔵。时间线锚定在《对魔忍浅葱》主线之前的「对魔忍组织化初期」：五车学园尚未成为统一据点，风魔一族与政府直属机关的合作仍处磨合期，红的后继位置日后由井河浅葱一系接掌。文风：近未来都市暗黑、人魔边界崩坏、组织与血脉双重压迫下的硬核战斗与逆转；本档案**只写剧情、战斗、组织与设定**，成人向描写一律压缩为「俘获 / 改造 / 调教设施」等剧情节点，不写色情细节。

**【世界定位】**
近未来日本东京的「表世界」与「里世界」双重都市。表世界是普通市民的日常；里世界由黑手党、杀人者、违法企业、魔族与黑市交织。人与魔之间古来的「人魔不干涉」早已形骸化。政府将能以忍术对抗「魔」的忍者末裔组织化为「对魔忍」，用作刃与盾。主角心愿寺红是吸血鬼半魔却身在对魔忍、且列风魔八将的异端——她以歼灭魔族为执念，却被自身血脉与地下都市的阴谋拖入命运深渊，最终在觉醒与决断后选择「既非人亦非魔」的独立战路。

**【世界观 · 力量体系】**
世界基本规则：人类世界与魔界（Dark World）长期并存；魔族可经间隙、契约、走私与地下都市渗透人间。对魔忍是能以「忍法 / 异能」对抗魔族的特殊忍者，穿特制对魔忍战斗服，按门流与氏族传承术式。死亡规则：凡人与低阶魔族一死即终；高阶魔族、吸血鬼本源可极强再生与寄生，但非无限复活——重伤、斩首、针对血脉的术式仍可致命。特殊系统：①**忍法体系**（元素遁术与异能系）；②**「眼」之异能**（风魔一脉多门传眼力）；③**魔界医疗 / 人体改造**（科学与魔导融合，可强化肉体、刻印、改造器官，亦可用于支配）；④**半魔 / 吸血鬼血脉**（肉体强化、再生、觉醒后质变）；⑤**组织序列**（风魔八将、政府指令链、地下都市黑市阶梯）。

力量阶梯（由低到高，按本篇与系列共通战力表现归纳）：
1. **凡人·黑道打手 / 底层佣兵**：热武器与格斗，无异能。战力：街头火并、据点守卫；对正式对魔忍几乎无效。
2. **见习对魔忍 / 低阶魔族走狗**：忍术雏形或低级魔能。战力：可杀凡人小队、潜入与暗杀；热武器仍威胁。
3. **正式对魔忍 / 中坚魔族**：成型忍法、可远程杀伤。战力：端掉据点、乱军取首；魔族侧为地下都市常见「爪牙」。
4. **精英对魔忍 / 风魔一门高手**：一域知名，术式与体术并重。战力：掀平建筑群边缘、硬抗常规武装；风魔八将候补级。
5. **风魔八将级 / 心愿寺红（战前常态）**：一门之将，复合术式与双刀。战力：高速近战、真空刃、单体歼灭；「歼血的红姬」名号来自旋风阵后满身敌血。
6. **魔改强化体 / 地下都市「商品」级战斗奴隶**：经魔界医疗改造后的强化肉体。战力：视改造程度可越级，但精神与自由常被锁死；战斗上可威胁精英对魔忍。
7. **吸血鬼半魔觉醒体（红·觉醒后）**：血脉全开，再生与力量质变。战力：一击可终结人间侧「豪商级」武装与中坚魔族；对同血缘仍有伦理与杀意的双重撕裂。
8. **魔族少女·费利西亚（全力）**：血统高贵、经改造与训练的战斗体。战力：可与觉醒红死斗并互成重创；非战略级毁城，但是「血脉棋子」级威胁。
9. **天堂数马的武装网 / 中华联合相关黑市**：人侧权势+魔侧技术。战力：不靠个人单挑，靠情报、陷阱、改造设施与拍卖网络；可捕获八将级目标。
10. **埃德温·布莱克（Edwin Black）**：天生吸血鬼、诺玛德首魁。战力：系列顶点级存在之一——再生、寄生、超凡耐久与智谋；日光与传统弱点对其无效。本篇他更多以「血脉源头与观测者」出场，而非全程街巷乱战。

乐园阶位映射（对照《阶位战力图鉴》，宁低勿高；本世界覆盖一～五阶，**无六阶**）：凡人·黑道≈一阶；见习对魔忍·低阶魔族爪牙≈一～二阶；正式对魔忍·中坚魔族≈二～三阶；精英对魔忍·风魔高手·改造强化体≈三～四阶；风魔八将常态·觉醒前红≈四阶；觉醒红·全力费利西亚·地下都市顶级设施战≈四～五阶；埃德温·布莱克＝**五阶顶点（城市级·领主感，准地区级压迫）**。顶点始终保有完整战力与智谋，低阶切入一律**规避顶点·情报优先 / 条件性胜利**，不得把顶点写成「无害旁观」。

**【地理 · 舞台】**
- **东京表世界**：霓虹、通勤、政府机构与合法企业。对魔忍任务常伪装为「公安相关」或夜间失踪案。普通市民不知魔族存在，或将失踪案归咎于黑帮。氛围：光鲜下的不知情；契约者若在表世界公开施术，会先引来警察与媒体，而非魔族。
- **东京里世界**：黑帮、走私、人体与情报黑市、魔族酒吧与据点。边界在夜巷、废弃工场、地下通道、封闭车站。里世界有自己的「规矩」：不在表世界暴露魔、欠债用命偿、对魔忍是「会走路的灾厄」。
- **地下都市·黄泉原（ヨミハラ / Yomihara）**：本篇主舞台。人魔混居的巨大地下城，合法与非法分界模糊；情报、改造、拍卖、佣兵、违禁能源全在此流通。街道分层：上层偏娱乐与交易，中层是工场与改造诊所，深层接近魔界渗透口。对魔忍潜入即高风险——本地人能从步态与杀气辨认「上面的刃」。
- **贪婪之家（グリードハウス / Greed House）**：天堂数马经营的暗黑欢愉与交易设施，覆盖人口贩卖、器官、改造「商品」与拍卖。建筑内部多隔离层、拍卖厅、医疗改造层与贵宾通道；政府高官亦有秘密出入传闻；背后有**中华联合**影子。红与菖蒲被俘后的核心地点，也是本篇最大「副本」。
- **风魔一门领地 / 心愿寺一脉据点**：风魔八将议事、训练与分配任务的政治空间；半魔的红在此被孤立，会议室里的沉默比刀更锋利。心愿寺一门保留眼之传承的训练场与刀冢。
- **政府直属机关（对魔忍指令链）**：向对魔忍下发任务的人间侧枢纽；文件、预算、灭口权限集中于此。苫利礼一郎等特工在此夹在上层与现场之间。机关走廊干净得像手术室，与黄泉原的湿冷形成对照。
- **魔界边缘（本篇多作背景）**：魔族来源地；布莱克与诺玛德的真正根基在更远的暗处，本篇只露其爪牙与血脉。低阶角色把魔界当作「更深的黄泉原」，高阶才知那是另一套文明与弱肉强食。
- **表里交界的具体场景池**：废弃物流仓库、地下停车场拍卖预展、霓虹街的情报酒吧、对魔忍临时安全屋、被清空的地铁隧道。写开场优先用这些可观察地点，避免空喊「黑暗都市」。

**【世界剧情线】**
① **人魔边界崩坏与对魔忍召集（前史）**  
东京之夜分成两个世界：无辜之民的表世界，与黑手党、杀人者、违法企业及「魔」横行的里世界。古来「人魔不干涉」已成空文——人因欲望堕入黑暗，魔则向光亮处扩张势力。政府将能对抗魔的忍者末裔组织化，编为「对魔忍」，用作讨伐之刃。风魔一族作为忍者大族被编入合作框架，其头目众「**风魔八将**」为实力顶点序列。心愿寺一门以「眼」之传承著称；先代八将**心愿寺幻庵**将双小太刀「**白神**」「**红魔**」传予**心愿寺红**。红的母亲**心愿寺枫**曾与天生吸血鬼**埃德温·布莱克**有过交集并诞下子嗣——红因此成为**吸血鬼半魔**，在对魔忍内部被贴上「魔之血」的标签，却以洁癖与执念选择站在「人」的一侧，将魔族视作必须歼灭的对象。

② **歼血的红姬：孤独的八将（开局状态）**  
红以风遁与双刀复合的「**绝技·旋风阵**」闻名：以眼力察知空气中风的歪曲，双刀高速切入，真空之刃膨胀为小型龙卷，将目标撕裂——战后必然满身敌血，故得「歼血的红姬」异名。她列风魔八将，却因半魔身份难以真正掌握部下，在一族「利益优先」的风气中格格不入。唯一忠诚的腹心是**槙岛菖蒲（槇島あやめ）**：擅情报与支援，风遁读风可极致狙击，眼之异能将视觉扩展至近乎「先读」的程度，近战亦可枪火纵横。菖蒲对红的忠诚越出任务，近乎信仰。政府侧对接人是**苫利礼一郎**：原同心愿寺一门的忍者，因待遇不满转任政府特工，夹在上层严控与现场洁癖的红之间，苦不堪言。世界状态：对魔忍已组织化但未达五车学园时代的统合；红是「政府之犬」与「一族异端」的双重孤独者。

③ **天堂数马的目光与贪婪之家（阴谋启动）**  
黄泉原的闇商人**天堂数马**经营「贪婪之家」，以改造与拍卖牟利，并与魔族、中华联合相关网络勾连。他盯上红体内「诅咒般的宿命」——半魔吸血鬼血脉可被当作最高级「商品」与实验素材。天堂不以正面对决八将，而以情报、诱饵、陷阱与魔界医疗技术布局。费利西亚以「天堂的魔族少女」身份出现：容貌绝美、表情虚空，实为经彻底改造的使役体，亦是红的**血脉之妹**（布莱克与枫的另一女儿），却被蒙在身份与记忆的雾中。

④ **俘获：任务坠入陷阱（主线转折）**  
红与菖蒲执行针对黄泉原/贪婪之家相关的讨伐或潜入任务时，落入天堂布下的网。八将级战力在「正面对决」上占优，但在情报战、药物、设施与人数优势下仍可被生擒。两人被押入贪婪之家：此后剧情进入对魔忍系列经典结构——**俘获 → 魔界医疗式肉体改造 / 支配手段 → 设施内的屈辱与意志考验 → 血脉或羁绊触发逆转**。本档案将此阶段记为：**红与菖蒲的身体与行动自由被设施与改造技术暂时剥夺，天堂试图把红锻成可拍卖的「最高级商品」，并把菖蒲用作施压与实验对照**。红以「这也是任务、屈辱千倍奉还」的忍道撑住精神；菖蒲则因对红的忠诚在改造中承受极端痛苦，几近崩溃。

⑤ **血脉真相：布莱克、枫、费利西亚（中盘揭秘）**  
在设施与对峙中，红得知自己是**埃德温·布莱克与心愿寺枫之长女**，费利西亚是其**亲妹**。布莱克并非单纯「魔界暴君」式路过——他作为天生吸血鬼与诺玛德首魁，对「杂交特异体」与「能否成为威胁」抱有观测兴趣；红的存在本身是棋子与实验。费利西亚对天堂的服从来自改造与洗脑式使役，其战斗力在释放时足以与红死斗。此阶段主题：**红最恨的「魔」写在自己的血里；最该保护的「人」站在她身边却正在被改造碾碎。**

⑥ **觉醒与逆转：天堂的终结（高潮）**  
红在极限中唤醒父亲一侧的吸血鬼血脉：力量、再生与杀意质变，水色战斗服与气质亦随血脉显化而改变（原作中觉醒后形象偏酒红与褐色肌等视觉标记）。觉醒后的红不再是「可被拍卖的商品」，而是能撕开贪婪之家武装网的灾厄。**天堂数马**作为人间侧黑手被清算——资料记载其最终被费利西亚等魔侧弃子式处理，布莱克本就只把他当作棋子；红侧则完成对「捕获自己之人」的武力终结。菖蒲在重伤与失血中濒死；红以唯一能救命的方式——**以吸血鬼之力使其转化 / 延续生命**——做出不可逆决断。两人因此共同跨出「人类对魔忍」的边界。

⑦ **姐妹死斗与「既非人亦非魔」（终局抉择）**  
费利西亚与红的对决是血脉与立场的双重悲剧：妹妹是魔侧棋子与改造产物，姐姐是半魔对魔忍的觉醒体。红在布莱克的注视下对费利西亚施加致命级打击，使其重创。吸血鬼路线（原作中接近「真结局」气质的选择）上，红拒绝继续当政府刀刃，也拒绝被井河等后续正统对魔忍体系完全收编；她与菖蒲下野，以「**元对魔忍的吸血鬼**」身份自立，继续与魔、与布莱克所代表的暗网作战。结局方向（吸血鬼向）：红与菖蒲存活并共生；天堂覆灭；费利西亚重创生死未在本篇完全写死；布莱克退入暗处，战火未熄；红的「后任」叙事空间留给日后浅葱时代。其他结局分支（堕落 / 支配失败等）存在于游戏选项中，属 IF 路线，本档案主线以**觉醒—逆转—下野续战**为默认世界状态。

⑧ **与系列时间线的衔接（后史定位）**  
本篇结束后，对魔忍组织继续演进：井河浅葱崛起为「最强对魔忍」，五车学园与更完整的政府合作体制成形；红在部分后续媒体中以「前辈 / 半魔吸血鬼」形象再现（如 RPGX 等平行或延伸时间线年龄与立场有调整）。本世界书**以 2015 年 PC 版《对魔忍 红》主线为准**；后续手游设定仅作互文，不覆盖本篇结局。

⑨ **组织战与战场生态（贯穿）**  
对魔忍任务形态：潜入黄泉原、猎杀越界魔族、切断人魔交易链、保护重要证人、回收改造技术、摧毁黑市实验室。敌人形态：魔族种族多样（兽人、吸血鬼、那伽、梦魔等，系列共通）、人间侧黑商、改造体、佣兵、被洗脑的前对魔忍（传闻级）。战场节奏：表世界要隐蔽，里世界可略放开杀伤，黄泉原内则处处是监控与人质。名场面因果链：旋风阵满身血的立名 → 八将却孤立 → 天堂布局 → 贪婪之家俘获 → 血脉揭秘 → 觉醒撕开设施 → 救菖蒲的不可逆选择 → 伤费利西亚 → 拒绝归队、独立续战。

⑩ **可观察的世界状态锚点（供任意时点判断）**  
- 红是否仍列风魔八将且听命政府？  
- 红与菖蒲是否已被贪婪之家俘获？  
- 红是否已知布莱克为父、费利西亚为妹？  
- 红是否已吸血鬼觉醒？天堂是否已死？  
- 红与菖蒲是否已下野为「元对魔忍的吸血鬼」？  
任意切入须先钉死上述锚点，再写势力态度与危险度。

⑪ **前史补密：心愿寺一门与风魔政治**  
心愿寺一门在风魔体系中以「眼」立足：能看见风的歪曲、杀气的轨迹、甚至战斗中的半拍未来感。这种能力使一门成为八将序列的常客，也使一门成为被拉拢与被忌惮的对象。幻庵时代，一门尚能以武统压住闲话；红继任后，半魔血统成为公开的把柄——同列八将不必拔刀，只需在任务分配上把红扔进最脏、最孤立的猎场，让她「自己证明忠诚」。红的回应是更干净、更彻底的歼灭：她不收黑钱、不卖人情、不让魔族活口留下把柄，于是「歼血的红姬」之名越响，她在会议室里越安静。菖蒲看清这一点，选择站到红身侧而不是站到多数一边：情报、狙击、善后，她把红从「一个人的刃」补成「两人的小队」。

⑫ **阴谋层：天堂如何猎杀八将**  
天堂数马不是剑士，是供应链。他的猎杀分四步：①用中华联合相关渠道买到「半魔血脉」与对魔忍行动规律的情报；②在黄泉原布置合法外壳（娱乐设施）与非法内核（改造层）；③用诱饵任务或伪情报把红与菖蒲引入可封锁的动线；④以药物雾、拘束阵、人数与设施火力完成生擒，而非堂堂决斗。对契约者而言，这意味着：在三阶以前，真正的敌人是「信息差」；在四阶，敌人是「设施规则」；在五阶，敌人才变成「血脉与顶点注视」。天堂的自信来自他从未失败的拍卖履历——直到他选中的商品自己醒过来。

⑬ **俘获章的战斗与组织意义（非色情）**  
俘获章在叙事上的功能不是展示屈辱本身，而是完成三件事：①打碎红「我只要更强就能杀光魔」的单线信念；②把菖蒲的忠诚从「任务服从」推到「可为之越界」；③把吸血鬼血脉从「歧视标签」变成「可觉醒的力量与诅咒」。魔界医疗在此呈现为：神经增敏、行动拘束、精神压迫装置、以同伴为筹码的实验对照。红的抵抗方式是忍道式记账——把每一分被迫承受的失去自由，记成必须连本带利讨回的债。菖蒲的抵抗方式是情报型：哪怕意识模糊，仍试图记住通道、守卫换班与设备噪音，留给未来的反攻。

⑭ **觉醒战的空间与节奏**  
觉醒爆发时，贪婪之家从「牢笼」变成「屠宰场反向开启」。真空刃不再只用于清杂兵，而是切开隔离层与武装节点；吸血鬼再生让红能在中弹与贯穿伤后继续推进。天堂的私人武装与改造体在四阶级可构成威胁，在觉醒红面前迅速沦为路障。费利西亚的拦截把高潮从「单方面复仇」扭成「姐妹伦理战」：红每一刀都在杀敌，也在杀血缘。布莱克不需要下场挥拳——他的存在本身改变战场气温：所有人（包括契约者）都必须把「撤退路线」与「是否被观测」列入战术。

⑮ **终局后的世界真空**  
天堂死后，黄泉原出现短暂的权力真空：其他黑商抢地盘，魔族试探新主人，政府急于灭口与改写报告。红与菖蒲的下野，等于从体制内抽走一把最锋利也最不听话的刃。风魔一门会有人暗自松口气，也会有人担心「半魔吸血鬼」成为新的不稳定因子。浅葱时代尚未到来，但「需要更强、更可控的对魔忍」这一政治需求，已在本篇终局的废墟上发芽。契约者若在终局后入世，面对的是：可结盟的红—菖蒲独立战线、正在重划的黄泉原黑市、以及从未离开的布莱克阴影。

⑯ **多结局与档案默认**  
原作存在多条结局分支（含意志崩坏向、支配失败向等）。本档案默认**吸血鬼觉醒—逆转—下野续战**为世界主状态，因其最完整回收血脉、组织与姐妹线，并与系列后续「红以半魔/吸血鬼姿态存续」的互文相容。若契约者强行导向其他 IF，须在叙事中明确「此为分岐世界」，避免与主库其他对魔忍条目时间线硬撞。

**【势力图谱】**
- **对魔忍（政府组织化忍者）**：宗旨讨伐魔族与维稳里世界；地盘东京及渗透点；代表红、菖蒲、指令链上的苫利；战力从见习到八将；与政府互相利用——政府给合法性与资源，对魔忍给刀；一旦失控，政府优先保自己。
- **风魔一族 / 风魔八将**：忍者大族；八将为头目众；内部是实力与利益的议会政治；红为其中异端半魔，战功与孤立并存。
- **心愿寺一门**：眼之传承与双刀武学；与风魔体系绑定；枫—红—幻庵为血脉与武统线索；一门荣誉与半魔污名在红身上撕裂。
- **政府直属机关**：人间侧预算、任务书与灭口权限；可抛弃「失控的刃」；苫利是现场与上层的传声筒。
- **贪婪之家 / 天堂数马**：黄泉原暗黑产业旗舰；改造、拍卖、人口与器官；与中华联合传闻相连；覆灭后地盘被瓜分。
- **诺玛德（Nomad）**：布莱克创立的跨国犯罪复合体；魔族与人间资本合流；本篇多作背景压迫与血脉源头。
- **中华联合（相关黑市）**：跨境技术与人口利益；贪婪之家的国际阴影；账册是高价值情报目标。
- **魔族诸派**：弱肉强食；在黄泉原与人间犯罪共生；对「半魔对魔忍」态度分裂——有的想杀，有的想抢血样。
- **独立战线（终局后·红与菖蒲）**：无编制、无预算，只有刀与情报；对魔与对政府双重警戒；是契约者五阶最可结盟的「非体制刃」。

**【隐藏剧情 · 伏笔】**
- 布莱克对「杂交子嗣能否成为威胁」的观测，是红命运的幕后引擎；他要的不是温顺女儿，而是可检验的刀。

- 红下野后，井河浅葱时代的「正统对魔忍」叙事将填补其空缺——本篇是浅葱之前的暗色前传。  
- 中华联合与贪婪之家的链路可接到系列其他作的东京王国 / 龙门等犯罪网（互文，勿在本篇硬写未出场人物的完整主线）。

- 多结局中「吸血鬼向」是档案默认主线；其他结局为 IF，写入时需标注分岐世界。

**【大事记时间线】**
远前史 → 人与魔划界，「人魔不干涉」尚有约束力。  
近前史 → 边界形骸化；政府组织对魔忍；风魔八将入编。  
血脉前史 → 枫与布莱克诞下红与费利西亚；费利西亚落入魔侧/改造网。  
武统交接 → 幻庵传「白神」「红魔」于红；红列八将，得「歼血的红姬」名。  
开局 → 红以八将身份讨魔，菖蒲随行，苫利传令；半魔孤立加剧。  
阴谋 → 天堂盯上红的血脉，贪婪之家布局诱饵与捕捉阵列。  
俘获 → 红与菖蒲中计被擒；改造与设施监禁；意志考验。  
揭秘 → 红得知父为布莱克、妹为费利西亚。  
觉醒 → 吸血鬼血脉觉醒，撕开设施；天堂势力崩解并被清算。  
救赎 → 红以血脉手段救濒死菖蒲，二人同越人魔边界。  
死斗 → 红重创费利西亚；布莱克注视并退入暗处。  
终局 → 红与菖蒲下野，以元对魔忍吸血鬼之身续战。  
后史 → 对魔忍体制向浅葱 / 五车时代演进（本篇外）；黄泉原权力再分配。

**【叙事基调 · 雷区】**
画风：近未来暗黑、霓虹与地下湿冷并存；战斗高速血腥（旋风阵的「满身血」是视觉签名）；组织对话冷、交易脏、血脉伦理残酷；对白短、刀光长。  
忌：①把红写成无脑沉沦（主线是意志、羁绊与逆转）；②把布莱克写成可随意街斗的杂兵，或否认其顶点压迫（须情报优先/条件性胜利）；③成人场景展开描写（只保留俘获/改造/设施作为情节节点）；④让浅葱、雪风等后辈在本篇主时间线常态共同行动（时代错位）；⑤用恋爱日常盖过讨魔与组织阴谋；⑥给本世界写六阶及以上战力。  
最早可切入锚点：红仍以八将身份执行黄泉原外围任务、天堂尚未收网之前；或贪婪之家刚对红产生兴趣、陷阱将合未合时。

## 阶位切入点

> 阶位↔原作战力：一阶≈凡人·黑道打手·底层佣兵；二阶≈见习对魔忍·低阶魔族爪牙·街头异能；三阶≈正式对魔忍·中坚魔族·黄泉原常见战力；四阶≈精英对魔忍·风魔高手·改造强化体·八将常态；五阶≈觉醒吸血鬼红·全力费利西亚·设施总攻·**埃德温·布莱克（顶点）**。世界顶点布莱克为五阶巅峰存在，始终保有完整战力与智谋；低阶切入一律「规避顶点·情报优先 / 条件性胜利」。本世界**无六阶及以上**。

**一阶（东京表里交界·黑市跑腿线）**

切入身份/时点：契约者以黄泉原外围「搬运工 / 线人 / 黑道小弟」身份切入，时点锚定天堂数马势力扩张、对魔忍尚未对贪婪之家发起正式突入之前。

初始事件：你在废弃地铁口替人送一只上锁冷藏箱，箱角渗出消毒液与血锈味；接货的牙人突然被「水色战斗服的女忍」无声放倒——**心愿寺红**的巡逻刃锋停在你喉前半寸，**槙岛菖蒲**的瞄准镜红点落在你眉心：箱子里是魔族走私的改造样本，而你的雇主账上写着「贪婪之家」。

开场白建议：「东京的夜分成两层。上层的人假装听不见地下的尖叫；下层的人用密码和假名换命。你只是跑腿的——直到冷藏箱在你手里发烫，风里传来双刀破空的锐响。『放下。』红姬的声音像刀背。『再动一下，你就不是证人，是赃物。』」

关键NPC立场：**心愿寺红**视你为可能的污染源，可留活口换情报；**槙岛菖蒲**更愿远程解决隐患，但对红的决定服从；黄泉原牙人会卖你两次；天堂的外围打手只认货不认人。

主线钩子/支线：钩子——冷藏箱样本与贪婪之家的采购单；支线①替对魔忍指认交接暗号；支线②在表世界警局「走失人口」档案里发现同款箱号。

危险度/规避：中。规避正面对抗红与菖蒲；规避主动打开改造样本。

任务方向/奖励：活着交出情报链；奖励为黄泉原地图碎片、低阶护具、对魔忍临时「可用线人」标记（非编制）。

**二阶（见习刃·指令夹缝线）**

切入身份/时点：契约者以见习对魔忍或政府合同佣兵切入，时点在**苫利礼一郎**下达「监视贪婪之家」却禁止打草惊蛇的阶段。

初始事件：苫利把你塞进红的支援名单边缘——「别惹怒她」。任务是跟踪一辆驶入黄泉原的改装车；车里不是货，是被下药的半魔相关实验记录。红拒绝与你组队，菖蒲却塞给你一只读风用的耳麦：「她讨厌多余的人。但她更讨厌任务失败。」

开场白建议：「政府的公文把魔写成『非法定居体』，把你写成『可消耗的刃』。黄泉原的电梯门打开时，霓虹广告里笑着拍卖『特殊血脉』。耳麦里菖蒲的声音很轻：『红姬已经进去了。你若拖后腿，我会先射你的腿。』」

关键NPC立场：**苫利礼一郎**要你当夹缝缓冲；**心愿寺红**对「组织派来的累赘」冷淡；**槙岛菖蒲**以任务效率评估你；天堂外围魔族视见习为狩猎练习。

主线钩子/支线：钩子——实验记录指向红的血脉档案被黑市标价；支线①截获中华联合相关运单；支线②保护一名逃离贪婪之家的证人到表世界。

危险度/规避：中高。规避单独闯入贪婪之家核心；规避向风魔八将同僚炫耀半魔情报。

任务方向/奖励：完成监视与一次成功撤离；奖励中阶忍具零件、风遁基础训练许可、政府黑卡额度。

**三阶（正式对魔忍·贪婪之家外围战）**

切入身份/时点：契约者为正式对魔忍，与红、菖蒲同一指令链，时点在天堂对红「感兴趣」已明确、陷阱将合未合。

初始事件：联合突入黄泉原外围赌场据点。红以旋风阵清场，血雾未散，菖蒲在钟楼狙击。你负责切后路——却发现后路通道被改造成「欢迎八将」的捕捉阵列：药物雾、拘束咒文、魔界医疗探针。天堂的声音在广播里笑：「歼血的红姬，终于肯光临拍卖预展。」

开场白建议：「旋风过处，魔族的甲片像雨。红的外套下摆已被血浸透，她却只是换了个握刀角度。『菖蒲，三点钟。』『已锁定。』下一秒，你脚底的地板翻开，探针刺入靴底——有人把『捕获对魔忍』做成了流水线。」

关键NPC立场：**天堂数马**要活的红，你是附赠；**费利西亚**首次露面时像人偶；**心愿寺红**在危机中仍优先摧毁改造设备；**槙岛菖蒲**会为掩护红强行透支眼力。

主线钩子/支线：钩子——阻止红被生擒（原作主线中失败则进入俘获章，成功则大幅改写）；支线①夺取改造探针样本；支线②黑入贪婪之家场刊，找到「半魔血脉」拍卖页。

危险度/规避：高。规避与天堂正面对赌「规则内决斗」；设施战优先破阵而非斩将。

任务方向/奖励：破阵列、救下同伴或夺得场刊；奖励精英级刀装、抗魔药物、黄泉原深层地图。

**四阶（八将同级·俘获与觉醒前夜）**

切入身份/时点：契约者战力贴近精英对魔忍/八将候补，时点锚定**红与菖蒲已被俘、改造进行中、血脉真相尚未完全公开**。

初始事件：你潜入贪婪之家维修层，听见隔离室里菖蒲的 equip 生命体征报警。红在相邻室被作为「最高级商品」准备，却仍在用气音盘问看守天堂与布莱克的关系。费利西亚抱着膝盖坐在走廊，对你说：「ふぇり什么都不懂……但爸爸说，姐姐很快会『漂亮地醒过来』。」

开场白建议：「地下的空调把血腥味吹成冷的。你不是来拯救公主的骑士——你是来拆掉一条血脉生产线的刃。隔离玻璃后，红的眼睛仍锋利：『别救我。先毁了那台能改写神经的机器。然后……告诉我，费利西亚到底是谁。』」

关键NPC立场：**心愿寺红**要破坏而非单纯逃生；**槙岛菖蒲**命悬一线，是红的软肋；**费利西亚**在「妹妹」与「兵器」间切换；**天堂数马**视你为闯入展厅的小偷；**埃德温·布莱克**的注视可能通过魔眼/代理人降临。

主线钩子/支线：钩子——血脉档案室的「枫—布莱克—红—费利西亚」谱系；支线①偷运抗改造抑制剂给菖蒲；支线②伪造拍卖流拍，拖延红被转移。

危险度/规避：极高。规避与布莱克本体交涉式硬刚；设施内优先情报与破坏关键设备。

任务方向/奖励：推动红接触真相与觉醒条件；奖励高阶兵装维修权、吸血鬼血脉研究残页（危险品）、菖蒲的狙击协作契约。

**五阶（觉醒逆转·下野前夜·顶点阴影）**

切入身份/时点：契约者以「可与觉醒体并肩的战力 / 独立魔猎者」切入，时点在**红吸血鬼觉醒、天堂清算、费利西亚死斗、红将选择下野**前后。

初始事件：贪婪之家上层崩塌。红的双刀拖出真空裂痕，天堂的权势在血脉威压下碎裂。费利西亚以全力魔族形态拦路：「姐姐……爸爸在看。」布莱克的气息如夜雾压城——他不立刻出手，只评价：「成为威胁了吗，红？」菖蒲倒在红身后，心跳将熄；红转向你，问的不是赢，而是：「若只有魔之血能救她，你还认我是对魔忍吗？」

开场白建议：「拍卖场的吊灯爆裂，像一场红色的雪。红姬站在废墟中央，曾经厌憎的血正在救她、也正在改写她。费利西亚的刀光与她对撞，姐妹的杀意里全是被他人写好的剧本。布莱克的声音从暗处落下，慢条斯理：『继续。让我看看，心愿寺的孩子值不值得活到下一纪。』」

关键NPC立场：**心愿寺红**在救菖蒲与斩断血脉棋局间抉择；**槙岛菖蒲**的存活依赖红的禁忌手段；**费利西亚**可被重创、可被夺回（难度极高）；**天堂数马**已无谈判价值；**埃德温·布莱克**为**五阶顶点**——始终保有完整战力与智谋，对策是存在·情报优先 / 条件性胜利（布局、撤离、破坏其棋子），不得写成可正面硬刚的杂兵；**苫利礼一郎**若仍在场，代表政府将「回收或抹杀失控之刃」。

主线钩子/支线：钩子——支持红下野为「元对魔忍的吸血鬼」并建立独立战线，或阻止政府灭口；支线①护送重创的费利西亚脱离布莱克回收；支线②公开贪婪之家与中华联合账册，换取人间侧政治空间。

危险度/规避：贴近顶点。必须规避布莱克本体持久战；胜利条件是「救下羁绊、切断设施、带情报撤离」，不是斩杀诺玛德首魁。

任务方向/奖励：觉醒战协作成功；奖励准五阶级兵装残片、吸血鬼共生相关秘传（高风险）、黄泉原深层主权空窗期的情报网、与红—菖蒲独立战线的同盟标记。
