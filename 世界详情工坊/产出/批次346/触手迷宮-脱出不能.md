**【世界定位】**
遗迹「**常標迷宮**」（本地人又称「无返廊」——名字吓人，规矩却温柔）。契约者不以清道夫或闯关者入场，而以**出口标记抄录员／小队绳手／迷路者登记处临时书记**身份卷入：制图师**サキ**发现墙上的出口粉笔画被雾气遮住了视线却并未被擦掉；易惊探险者**ノゾミ**一慌就看不见本该常亮的出口符；迷宫灵联络人**リツ**坚持「共居可以、锁门不行」；营地医师**カナ**把伤药与「说不」写在同一张纸上。一句话：所谓「脱出不能」，从来不是门被焊死，而是还没人敢在恐惧里把出口重新描一遍——核心玩法是关系、描标、可拒绝的共居与自由离场，不是评级闯关。

**【世界观 · 舞台设定】**
本世界的「迷宫」与「触手」不是互砍的阵营数值，而是两套生活礼仪与情感语言。常標迷宮以「**出口标记＝公开可见、恐惧＝视觉滤镜、共居＝可解约、离场＝随时合法**」维系秩序：墙面每隔固定间距都有前人用白垩与淡金粉画下的**出口符**（箭头、门形、真名链）；符本身几乎不会消失，但当人陷入失联、羞耻、被编号称呼或被强迫「必须深入」时，视线会自动滑过那些符——本地人把这叫「**惧盲**」。所谓「**脱出不能**」在本库被写成**误译与心理滤镜的合称**：古铭文原意是「**不可独自脱出**」（须有同伴真名与绳结），被后人误抄成「永远逃不出」；真正锁人的不是石门，是失联与惧盲。

超自然只服务情感与日常：触手在本库被写成**迷宫原生的湿润触须植被／廊壁灵**——能递水、扶人、把滑倒者轻轻托回通道、在有人说「停下」时缩回石缝；**强制命令会让触须整片缩回并记入「拒绝簿」**。触须不是清场兵器，也不是「完成深入指标」的工具；它们只响应「自愿靠近」「说不即停」与「出口符是否仍被看见」。火把余温能缓解惧盲却洗不掉记忆；绳结能系也能解；出口符在说谎（代签「自愿深入」、替别人说「她喜欢」）时会短暂发暗——测的是「你是否还允许别人离开／是否还允许别人说不」，不是血统纯度。

时间被切成「描标时」与「雾时」。描标时，**サキ**按制图班次巡墙，把被雾遮住的出口符重新描亮；雾时，**ノゾミ**最容易进入惧盲，需要有人在她耳边念出口符旁的真名链。真正可怕的不是死亡，而是被从队伍绳上解下、被改成「只记编号不记真名」、或永远只是「那个新来的、谁也不记得名字的抄录员」——惧盲会因此变成长期状态。

软规则三条（全员默认遵守，违反即触发廊内危机事件，用对话、描标与纸本解决）：
① **出口标记始终存在**——任何人不得故意涂死出口符；被雾遮住只许重描，不许否认「出口还在」。
② **可随时离场**——入口营地与真出口双向开放；无人有权锁门、没收绳、代签「自愿深入」。
③ **共居须合意**——与迷宫灵／同伴的亲密与共居可解约；「说不」即停，拒绝簿公开可查。

社交货币包括：共担火把班、帮重描一处出口符、是否拒绝强制亲密、是否替同伴投「回营」票、是否在惧盲发作时念对方真名而不是编号、是否在迷宫灵递水时先等三秒点头。AI 写场景时应先定「现在是描标时还是雾时、契约者袖口的登记牌有没有被谁别上淡金注、出口符在视线里亮不亮、绳结是否仍系着全员真名」，再决定人物语气。

本世界**不写**数值对战表、闯关榜、硬核对决主轴、强弱评级；若有超自然／异质元素，只作情感与压迫装置。

**【地理 · 生活舞台】**
- **入口营地「回声篝」**：安全与系绳中心。篝火圈、绳架、迷路者登记处、出口符样本墙。**サキ**在此发白垩笔；契约者常在此拿到第一截绳头与抄录板。承载：初遇、点名、第一次被全队叫真名、公开离场。
- **雾廊**：失散与惧盲高发区。出口符仍在墙上，慌时却「看不见」。喊真名可重聚；喊编号则雾更浓。承载：恐惧、重聚拥抱、真名的重量。
- **泉厅「暖石」**：**カナ**的主场。温泉雾、伤药台、可换衣与私谈。触须常在此递毯，先停三秒。承载：照顾、示弱、边界试探。
- **假出口厅**：墙上画着「很像出口」的装饰符，实为前人恶作剧或测试惧盲的练习墙。**サキ**爱在此教学「真符与假符差在真名链」。承载：识破、羞耻与成长同场。
- **真出口壁画廊**：True 线核心。壁画出口旁刻着前人的真名链；出口门常开，门轴上挂着「随时可走」的铜牌。承载：告白窗口、HE、后日谈入口。
- **绳结处／暗角**：绳手习惯藏备用绳的地方（本切片由**リツ**与登记处共用）。承载：灰区信任、多系一绳的义气。
- **共居侧室「湿苔房」**：迷宫灵**モエギ**与自愿共居者的休息侧室。触须铺成软席，拒绝簿挂门后。承载：合意亲密、解约谈话、非人被记得。
- **古铭文室**：**フミ**保管误译档案与「脱出不能」原文拓片。承载：真相回收、知识与恐惧的和解。
- **外客棚**：短期逗留的收藏家／强制深入派歇脚处。承载：外部压力、锁门企图、试金石对话。
- **火把交接点**：各廊交界的窄台。谁先递火把给谁，被全队解读为亲近信号。承载：日常升温与流言站队。

**【故事主线 · 情感线】**

**共通线：墙上的出口，与惧盲**
契约者因乐园偏差或以正当补位身份进入常標迷宮，恰逢「出口符大雾周」。制图师**サキ**把白垩笔塞进你手里：「描，别争谁更勇敢——出口一直在。」易惊探险者**ノゾミ**在雾廊第一次发作惧盲，抓住你袖口却叫不出方向；迷宫灵联络人**リツ**当众声明「共居可以谈，锁门我第一个剪绳」；营地医师**カナ**把伤药瓶口朝向全员，却把自己那份留到最后；古铭文员**フミ**低声说：「『脱出不能』是误译。」外客收藏家则提议「涂掉出口符逼人深入」——被全员默认软规则拦下。共通线拼：系绳与登记 → 雾廊失散与真名重聚 → 假出口教学 → 泉厅边界 → 有人企图涂符／代签 → 古铭文室揭误译 → 真出口壁画前公开选择（走／留／共居／解约）。阶段：相遇 → 日常描标升温 → 惧盲与涂符冲突 → 告白窗口 → 结局分支 → 后日谈。

**① 登记夜（相遇·第一日）**
篝火边，**サキ**主持系绳：「系所有人的名，包括你自己——编号只给货箱用。」**ノゾミ**手抖却仍把自己的名念完整。**リツ**把一截备用绳塞进你袖口，假装只是撞到。**カナ**先给别人上药。**フミ**递来拓片角：「有空来古铭文室。」**モエギ**的触须在营地边缘轻轻点了一下地面——像打招呼，等你点头才靠近递水。因果：新人＝可移动的「第五结」与「多一支描笔」；此后「与谁第一次共担火把班」被解读为亲近信号。

**② 雾廊与真名（第二至三日）**
雾廊失散。**ノゾミ**惧盲发作，坚称墙上什么都没有；契约者若抓住她的手把她的掌心按在出口符上，符会从她的触觉里「回来」。升温点：先喊谁的真名＝流言站队。**サキ**若发现有人用假符糊弄新人，会当众刮掉假符——手在抖，羞耻与原则同场。**リツ**可能「偷」走你的登记牌又还回来，只为确认你会不会追、会不会用编号骂人。

**③ 泉厅与共居边界（第三至四日）**
暖石泉。触须递毯，可拒绝。**カナ**的心结浮出：她习惯治愈别人，不会给自己留药。**モエギ**在湿苔房门口停三秒；有人替别人说「她说停下」时，触须会整片缩回并记拒绝簿。**リツ**讲解共居契约：可过夜、可解约、不可锁侧室门。契约者若逼**カナ**自愈（温柔地），或学叫**モエギ**的名字而不是「那团触手」，个人线开启。

**④ 涂符危机（冲突·第四至五日）**
外客收藏家（或传闻中的「强制深入派」）提出「涂掉出口符、锁入口营地、全员必须走到最深」的意向。流言分裂：一派说新人「不该插手」，一派求契约者在登记簿上「美化同意」。**サキ**与**フミ**当众冷战（描标责任 vs 误译考证进度）；**ノゾミ**失踪半日——其实是一个人在雾廊用触觉摸符；**カナ**要求代签「同意深入」被当场打断。名场面：假出口厅必须先刮谁画的假符；失败方的单独道歉；登记簿上的代签危机。**回营票若通过，入口篝火更亮；强行涂符且无视「说不」则触发清醒反弹——当事人头痛、对施压者厌恶、出口符对施压者暂时不可见。**

**⑤ 古铭文室与真出口前夜（告白窗口·第五至六日）**
**フミ**揭开拓片：「原文是『不可独自脱出』。」全场安静。**サキ**把制图班的白垩笔暂时放进契约者手中：「明天——你把第一笔描向谁的符旁真名，是你的公开答案；也可以放回空处。」此夜可触发各线告白：程序句、发抖句、共居解约句、只属于今晚的私情后要求次日仍做公正抄录员。真出口壁画廊的门轴铜牌在风里轻响——提醒所有人：门一直开着。

**⑥ 真出口日与结局（第六至七日）**
壁画廊不是黑屏终点。分支：
- **サキ HE**：公开承认「重描出口也是勇气」；日间共担描标班，夜间把白垩笔与绳放在同一侧；学会先问「你现在看得见符吗」。
- **ノゾミ HE／惧盲线**：学会在慌时先摸墙再喊人；契约者成为她惧盲时的「活体真名」；Bittersweet 变体：她先一步走出真出口去上层报「符还在」，留下自己描亮的那一截箭头。
- **リツ HE／共居伦理线**：把「可解约共居」写进登记处附录；剪断外客带来的锁链（象征性，不写暴力）；灰区信任转为「把退路留给所有人」。
- **カナ HE／治愈线**：终于给自己上药；你替她挡下「只治愈别人」的期待；安定 End 变体：留下当入口营地常驻医师。
- **モエギ HE／非人线**：从「环境装置」被记名为存在；合意共居可发生也可结束；True 协助：拒绝簿公开悬挂。
- **フミ HE／考证线**：误译条文公开更正；「脱出不能」招牌被改成「不可独自——请系绳」；知识服务生还。
- **True（真名链）**：推动「禁止涂符、禁止代签、禁止强制共居、禁止锁门」写入登记附录；壁画旁前人真名被读出时出口符全廊一齐发亮一瞬。
- **无名编号 End**：周期内从未认真倾听、从未系真名、从未帮人重描一笔，活着离开，登记处不记名。

**⑦ 后日谈**
走出或留下之后的舞台仍是日常：描标、系绳、处理新迷路者、教新人「惧盲时先摸墙」。余温与触须被新规约限制在「缓解焦虑、促进倾听、说不即停」；情感线转为如何在「队友／共居者／恋人」多重身份里保持边界感。真出口门轴铜牌仍在响——日常的新开端，不是通关画面。

**サキ线（制图·出口可见性）**
从「必须把所有符描到完美」到「承认自己也会惧盲」。攻略关键：信她的刮假符、不笑她手抖、涂符危机时站在她身侧重描而不是抢笔。HE：共担描标班；BE：她独自描到虚脱后把笔扔给你「你来，我看不见了」。

**ノゾミ线（惧盲·重获视线）**
从「墙是空的」到「我摸得到箭头顶端」。攻略关键：不骂她矫情、把她的掌心按向符、先喊她的真名。HE：她能自己看见出口仍选择牵你的手走；Bittersweet：她先走报信。

**リツ线（共居伦理·剪锁）**
从「油滑护短」到「把退路写成制度」。攻略关键：不抢她的备用绳、支持公开拒绝簿、拒绝外客锁门。HE：共居契约双签且可解；BE：她剪锁后自己离开去别处传规约。

**カナ线（自愈·安定港）**
从「药只给别人」到「也给自己」。攻略关键：温柔逼自愈、挡代签、泉厅让她先喝汤。HE：安定港；安定 End：常驻医师。

**モエギ线（被命名的迷宫灵）**
从编号到名字。攻略关键：学叫モエギ、替它拒买断、共居先停三秒。HE／柏拉图均可；不走人类恋人线也可成「被记得的存在」。

**フミ线（误译更正）**
从「只敢在拓片角写小字」到「公开改招牌」。攻略关键：认真读拓片、不把考证当扫兴、True 线请她执笔附录。HE：古铭文室双人灯火；知识线闭环。

**微观日常事件池（可循环触发）**
绳结磨损需重系；白垩笔用尽；假符混入真符墙；泉水半干时的分杯；回营票平票时的弃权权；触须递水被拒后轻轻缩回；登记牌背面淡金注「今晚可以说话吗」；真出口门轴新挂一个小铜牌；雾廊里有人先摸到符却不好意思说；古铭文室拓片受潮需重拓；外客棚熄灯后的道歉；湿苔房拒绝簿多了一行字迹；火把将尽时的交接沉默；有人把「脱出不能」旧海报当装饰贴回——必须揭下。
