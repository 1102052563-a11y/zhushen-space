## 剧情

**【作品来源】**
《仙业》作者鹓扶君，起点中文网连载古典仙侠长篇（书号1035985996，约2022年12月20日首发，连载中，公开进度常在数百万字量级，章节随更新浮动）。气质偏硬核古典：道藏术语、因果业力、法侣地财、宗门博弈与资源账本并重，杀伐果断而非圣母团宠。文风雅致克制，升级扎实，强调「前尘皆客，再世为人。这一次，只愿求长生」。**作品仍在连载，本档案只写至已公开剧情与可核设定，不编造终局谁胜谁负、世界如何落幕。**

**【世界定位】**
九州四海、阳世诸天与阴世幽冥并存的古典仙道宇宙：玄门八派、魔道六宗、天人外道、净土僧伽交错；胥都天等十六大天是常见舞台。主角陈珩为穿越魂附身玄真派同名少年，携金蝉与《神屋枢华道君说太始元真经》自水牢死局中起身，辗转入玉宸派真传周行殿，与陈玉枢血裔因果缠斗，求的是长生仙业而非一时名位。一句话：在因果网与宗门棋盘里，用一真法界与太始道基，把面首水牢里的死人熬成可争道的棋手。

**【世界观 · 力量体系】**
本界以正统仙道为至贵法统，灵肉并重，圭臬是「炼精化气、炼气化神、炼神返虚、炼虚合道」。并行路径包括佛道、神道（香火）、武道（罡煞）、幽冥鬼道、剑道等，后期多殊途同归。修士日常围绕**法侣地财**四缘：法=功法传承与箓籍；侣=道侣同修与因果纠缠；地=洞天福地与道场；财=丹药符器灵材。功德与业障影响气运与劫数；大宗垄断高阶资源，弱肉强食是常态，凡人王朝多依附修真势力。

**胎息**：入门第一步，绵绵若存，先天一点灵光／真阳显化，性根自现，寿约可至百五十，开始脱离纯凡。战力表现：能御低阶术法、压倒武夫，仍怕围剿与毒物。

**炼炁（羽士）**：炼精化炁，真炁分**九阶三十六品**；七阶及以上真炁方可支撑上品道基。此境是「采气、服饵、授箓、占验」的密集练习期。战力：可裂石断金、斗杀同辈、在小宗内门争魁首；高阶炼炁已能在坊市级冲突中定生死。

**筑基**：炁海生化 → 大小如意 → 龟蛇相抱（常需外药，阴阳交汇）。真炁凝基，寿元与续航质变，可立稳小道场、远行秘境。战力：一域中坚，能端山寨、毁坊市局部，是小宗长老／大宗内门的常见门槛。

**紫府（高功）**：万妙归根 → 超脱分形 → 换魂消魄。神识与法力进一步内化，分形、换魂等手段出现，政治价值陡升。战力：可掀平建筑群、在下院大比与跨宗任务里当核心，一城仅数人级。

**洞玄（炼师）**：龙虎炉鼎 → 摄取五精（优先先天五行之精） → 先天金汞（成丹之基）。是金丹前最关键的「材料与道基」关，龙宫／洞天类机缘常在此层。战力：城市级冲突主力，领域／大阵雏形可显，能左右一宗下院格局。

**金丹（真人）**：身根成象 → 渐法九还 → 神中有形；金丹分**九品**（九品黄白最次，一品龙虎为极）。品级决定同境上限与后续法相潜力。战力：地区级灾难，移山填海初现，可镇一域、争岁旦评／龙宫大比头名。

**元神（真人）**：金丹后凝结法相（下／中／上／至四等），共十二重障关。法相显化即人形战略武器级。战力：一怒可毁一国量级的局部，神识与神通高度融合，是宗门真传与魔道巨擘的常见主战层。

**返虚／纯阳（真君）**：元神入太虚淬炼，直面宿尘、天魔；纯阳有风火雷三灾，常需洞天基业。战力：世界级，规则皮毛可触，能改写区域战场与气运。

**合道（道君）**：与道合真，形体俱妙，离羽化证仙一步之遥；有九难，过后可直达仙业。相当于佛门菩萨、神道神君等。战力：行星／星空级一界之主向，掌本源权柄雏形。

**大冶／仙人及以上**：合道后为**大冶**（仙人，天地为公、造化为炉；佛陀、神王等各道异称）→ **太乙**（真流用先天大道，散数用后天；另有「空证」法门）→ **大罗**（混元无极大罗／金仙；至极者为**道主**，宰御先天大道，约四十九条）。成仙位业另有尸解仙（冥仙／鬼仙）、地仙、天仙等分层。**道主／大罗＝超阶**，情报优先／条件性胜利，严禁写成「封镇削弱才可胜」。

特殊系统一：**金蝉／一真法界**。陈珩重生携带的核心依仗，可开辟一真法界：模拟心相、推演功法、内外时间流速不同，等于把闭关与演法压缩进可携带的私域；代价是精气神消耗与越界风险，绝非无脑外挂。特殊系统二：**太始元真经**全名《神屋枢华道君说太始元真经》，号称「龙天通明，诸真总摄」，减轻采气烦忧、可炼化多种灵气。特殊系统三：**太素玉身**，肉身／真形向功法，与元真经互补，后期有太素真形、剑遁等表现。特殊系统四：**因果与人劫**。高阶修士常把后裔、道种、气运当资粮；陈玉枢一脉的「豢人」逻辑即此——众生如马牛，独我作龙象。特殊系统五：**洞天与先天五行之精**。龙宫大比、洪泽长生拔罪洞天等，是洞玄→金丹的硬通货入口。

死亡与代价：低阶身死即终；高阶或有元神逃遁、夺舍、尸解，但业力、雷劫、人劫、父辈转嫁之劫可直接抹杀。复活不是日常设定。战力破坏力对照（宁低勿高）：胎息～炼炁前中≈凡俗～街区；炼炁高～筑基≈山寨／坊市级；紫府≈据点／建筑群；洞玄≈城市级；金丹≈地区级；元神≈世界级局部；返虚纯阳≈世界级规则皮毛；合道≈行星／星空级；大冶／太乙／大罗／道主＝超阶。

乐园阶位映射：胎息～炼炁≈一～二阶；筑基≈三阶；紫府≈四阶；洞玄≈五阶；金丹≈六阶；元神～返虚≈七阶；纯阳～合道≈八阶；道主／大罗／高位仙人＝超阶（情报优先／条件性胜利，严禁顶点战力归零或「封镇削弱」式解释）。本库覆盖一～八阶。

**【地理 · 舞台】**
故事以**九州四海**为底图，阳世诸天（胥都天、虚皇天、洪鲸天、须延天等）与阴世幽冥并行。东弥州一带是早期人间舞台：**小甘山玄真派**坐落山中，离阑粱城不远，规模不大、五房分辖，是陈珩原身与重生后的第一牢笼与第一跳板。山下有水牢、药园、地渊入口一类「小宗日常＋凶险补丁」：水牢是晏飞臣迁怒的刑场；地渊阴森，鬼物、阴马、人面芝等资源与死劫并存。

中期主舞台上移至**玉宸派**辖区。**宵明大泽**是玉宸核心地理意象——泽国烟波、下院大比、四院魁首争夺都在此展开。**长离岛**是陈珩大比／入真传后获赐的道场，清幽宜闭关，也是他证道与养基的「地」。**希夷山**与道子**君尧**相关，是身世曝光后「求援／改命」的关键坐标。**周行殿**属玉宸真传体系内的殿署，陈珩被通烜道君收为真传后在此任执事，政治身份从「外来血裔」转为「有师承的体制内棋手」。

海域线以**东海龙宫**为核：岁旦评、玉泉市坊、龙宫大比／法会、**洪泽长生拔罪洞天**串成一条「夺先天五行之精、冲洞玄与小纯阳雷劫」的硬主线。回程围攻、通烜暗助等场面把「宗门护短」与「血裔仇恨」同时写实。外宇与更高诸天（洪鲸、须延等）在中后期作为探险与布局棋盘出现；首阳山、无形埒剑洞等古迹则提供剑道／秘宝支线。

写场景时分层：小甘山的山雾与水牢潮气、地渊的阴冷与鬼哭、宵明大泽的烟波与大比鼓噪、长离岛的清寂闭关、龙宫的深海威压与法会珠光、周行殿的体制冷光——同一套仙道规则，在不同海拔味道完全不同。AI写环境先写「这里资源怎么分配、谁在收因果税」，再写山水。

**【世界剧情线】**
① **水牢死局与金蝉重生（胎息启动）**  
原身陈珩容国陈族血脉，虚皇天神王陈裕一脉之后、魔道大能陈玉枢之子嗣线相关；因相貌「惊于天人」，被玄真派女修**晏蓁**强收为面首。晏蓁访友途中遇害后，其父长老**晏飞臣**迁怒原身，投入水牢折磨至死。穿越魂携**金蝉**附身同名少年，在死局中醒来：寒斗真炁／寒毒侵蚀、宗门敌意、身份污名三重压顶。他以安葬等为由暂离山门，在家父遗物中得核心功法《神屋枢华道君说太始元真经》，并以金蝉开启**一真法界**，开始以推演与苦修把「死人身份」改写成「可活的修士」。本阶段因果：没有水牢与晏家恨，就没有下山求生的第一推力；没有元真经与金蝉，就没有后续所有道基。世界状态：玄真派仍是小宗秩序，陈珩是可被抹掉的污点名额。

② **地渊求药与玄真内斗（炼炁）**  
寒毒需阳属大药，逼他主动入**地渊**采人面芝、阴马等险资，炼器炼丹、试探鬼物，手段果决（非圣母）。同期处理王端宝、炀山道人一类恩怨，结识**许稚**（赤龙许氏后、月庵圣母一脉相关，剑道天赋，早期最稳盟友），获剑术医书与道义互援。玄真派五房功德换资源、地渊符诏、内部倾轧是日常；「阴天子」一类命格／流言把他卷入更大因果。阶段主题：小宗里活下去＝会记账、会杀人、会藏底牌。阶段结局：炼炁有成，玄真已不够大，变故将至。

③ **玄真倾覆与玉宸入局（炼炁～筑基）**  
玄真派遭灭／大变（与外部散人、魔道暗线、血裔身份曝光等交织），陈珩辗转求生，阴差阳错进入**玉宸派**视野。初入**宵明大泽**，因陈玉枢子嗣身份遭诸真发难；**米景世**等助力，指点其赴**希夷山**求道子**君尧**。君尧深不可测，有意假死外游、空出道子之位，赠**阿鼻剑**、*兜术天王神宗玉书*等，为陈珩打开玉宸体制门缝。本阶段是「从罪人血裔变成可投资棋子」的定价过程：拒绝部分长老强收以守自身道统，同时必须借势。

④ **下院大比、长离岛与四院魁首（筑基～紫府）**  
陈珩参与玉宸下院大比，夺魁／成四院魁首序列，获赐**长离岛**为道场，习玉宸三经之一《玄中太无自然开元经箓》等，处理药园之战、四国事务、与怙照宗等魔道势力的摩擦。与**卫令姜**（赤明派真传，青鸟异象）并肩破阴谋，习先天大日神光等手段，情愫暗生；与**顾漪**（怙照宗真传）岁旦评／大比交锋，互称「妖女」与敌手，亦有复杂互援。**陈白**一类同血脉／同辈阴狠者，信「利我者可夺」，构成内部镜像反派。阶段结局：陈珩从外来户变成玉宸可打的名片，仇恨值与资源同步上升。

⑤ **东海龙宫、岁旦评与洞玄雷劫（紫府～洞玄）**  
东海龙宫线：经玉泉市坊得玄阙芝，结识陈婵等，入岁旦评与龙宫大比／法会，夺头名，交好颜熙，获准入**洪泽长生拔罪洞天**，夺优质**先天五行之精**，推太素玉身入元境，成洞玄修士并渡**小纯阳雷劫**。回程遭围攻，**通烜道君**暗助——为后文真传埋线。本阶段胜利定义常是「活着带走五精与品阶」，而非当众称霸诸天。世界状态：金丹门槛在望，父辈阴影与魔宗布局开始显形。

⑥ **通烜真传、周行殿与一品金丹（洞玄～金丹）**  
通烜道君亲收陈珩为**真传弟子**，任**周行殿**执事，政治身份质变。长离岛闭关（公开描写有数十年级闭关冲击）证更高境，金丹冲击指向**一品**而非黄白凑数；卫令姜同步冲高品金丹。魔道六宗、先天魔宗与玄门八派的棋盘在九州尺度展开，药园、市坊、洞天、人劫并行。阶段主题：有师承不等于安全——真传是刀也是靶。

⑦ **元神法相与陈玉枢人劫（金丹～元神）**  
金丹后凝法相、渡十二重障关的元神路。**陈玉枢**（元师／魔师）以《豢人经》等手段掠夺因果、炼化血裔气运以求更高道果，视后裔为资粮；陈珩作为关键「人劫」被逼入局。联合君尧余绪、通烜门下、卫令姜等盟友，对抗先天魔宗及其附庸；涂山狐族安置、古迹剑洞、秘宝炼制等支线补「地与财」。名场面应写清：神通对轰、因果反噬、血裔相残、道君级目光下的求生。世界状态：个人修为线并入九州气运线。

⑧ **返虚纯阳至合道前沿（连载进行中）**  
后期触及返虚、纯阳三灾、合道九难与仙业命题。总矛盾清晰：正统仙道用秩序与箓籍垄断通道；魔道用气运与豢人换捷径；空空道人一类「四乱」级存在超然搅局；陈珩要在棋盘上从棋子长成棋手，求的是「与日月齐光，与天地为常」的长生，而非替谁完成布局。**作品连载未完：不写终局胜负，不编造谁死谁飞升，不给「仙业」下假定论**；只标世界处于「高阶争夺加速、顶点未揭完」的进行态。

⑨ **微观仙道戏贯穿始终**  
功德账、符诏名额、大比座次、丹方真伪、寒毒与外药、面首污名、血裔档案、周行殿公文、龙宫法会资格——这些比大招更常出场。写这个世界，先写制度与因果如何勒人，再写人如何在勒痕里炼出太始道基。陈珩风格：能推演解决的不浪，能借势解决的不树全敌，但人劫与雷劫一到，必须拼命。

**【世界剧情线·分卷细目】**

**卷一：水牢与下山（胎息）**  
起因：晏蓁死 → 晏飞臣水牢。事件链：金蝉附身 → 寒毒显化 → 下山安葬 → 得太始元真经 → 一真法界初开。名场面：水牢潮冷、面首污名、遗物经文。世界状态：小甘山秩序未崩。可介入：水牢劫夺、寒毒药引、晏家追杀。

**卷二：地渊与玄真（炼炁）**  
起因：寒毒求阳药。事件链：地渊试探 → 人面芝／阴马 → 许稚结盟 → 炀山道人等恩怨 → 功德换资源。名场面：阴蚀红水对轰、紫弥宝衣、符器炼丹。因果：没有地渊资源就没有离山资本。

**卷三：玄真倾覆与投玉宸（炼炁～筑基）**  
起因：门派大变＋血裔曝光。事件链：逃杀 → 宵明大泽发难 → 米景世指路 → 希夷山君尧 → 阿鼻剑与玉书。名场面：诸真围诘、道子冷眼投资。因果：定价权从「罪人」改为「可养的刀」。

**卷四：下院大比与长离岛（筑基～紫府）**  
起因：玉宸需要魁首与战力。事件链：四院大比 → 夺魁 → 长离岛 → 药园／四国事务 → 卫令姜并肩、顾漪交锋。名场面：大比鼓噪、岛上闭关灯火。因果：地＝长离，侣＝情敌友开始成型。

**卷五：龙宫与五精（紫府～洞玄）**  
起因：成丹需先天五精。事件链：岁旦评 → 龙宫大比 → 洪泽洞天 → 太素元境 → 小纯阳雷劫 → 回程围攻、通烜暗助。名场面：法会珠光、雷火灌顶。因果：洞玄实至名归，真传门缝大开。

**卷六：周行殿真传与金丹（洞玄～金丹）**  
起因：通烜收徒。事件链：真传仪轨 → 周行殿执事 → 长离深闭关 → 一品金丹冲击 → 魔宗暗线升温。名场面：殿署冷公文与闭关轰雷并存。因果：体制内刀刃出鞘。

开场白建议：「小甘山的雾带着水牢的腥潮。山上人说那个惊艳误人的面首已经死了，可你在松林转弯处遇见的少年，明明还在呼吸——他袖中似有金光一闪，而背后，晏长老的亲传正沿石阶拾级而上。」  
关键NPC立场：**陈珩**——初生之犊却已会藏锋，可交易药材与情报，绝不轻露金蝉；**晏飞臣**——迁怒未消，视相关者为可灭口的污点；**许稚**——尚未深交时偏公道，可作剑道与医书线索；**玄真执事／五房管事**——功德换资源，见钱与见势。  
主线钩子/支线：水牢余波与安葬下山；寒毒阳药线索；地渊符诏名额。支线＝药园顶罪、坊市假丹、为陈珩跑腿换人情。  
危险度/规避：中——须规避洞玄派主与外来散人高手；勿在晏飞臣面前认领「面首余党」。  
任务方向/奖励：活过开局、换到入门功法与外药；奖励取向＝低阶符器、小白阳丹类、地渊入场线索、与许稚／陈珩的初期善感。

**二阶（炼炁高阶 · 地渊与玄真倾覆前夜）**  
切入身份/时点：以玄真内门竞争者或地渊承包散修身份切入，锚定陈珩地渊采药、恩怨清理、门派风雨欲来之时。  
初始事件：地渊符诏发放日，人面芝与阴马的消息把三拨人挤进同一洞口——鬼物先杀人，人再杀人；你发现陈珩的队伍短、后手却多，有人出钱买他的人头，价码写着「晏家愿结善缘」。  
开场白建议：「地渊的风是倒着吹的。符诏在手里发烫，人面芝的香混着血腥；你听见有人低声报价，买的不是药，是一个炼炁弟子的命——而那人的名字，叫陈珩。」  
关键NPC立场：**陈珩**——已能杀伐，愿换药不愿换命；**许稚**——可并肩入渊；**炀山道人／王端宝**——贪婪背刺型；**玄真五房长老**——要功德账不要英雄。  
主线钩子/支线：地渊夺药；清理炀山线；玄真倾覆前的撤离／投靠选择。支线＝炼器材料倒卖、阴马驯化、假符诏案。  
危险度/规避：中高——须规避筑基以上老怪与门派覆灭级战力余波。  
任务方向/奖励：带出阳药或活口情报；奖励取向＝高阶炼炁功法残篇、符器胚、与玉宸方向的逃路地图。

**三阶（筑基 · 宵明大泽入玉宸线）**  
切入身份/时点：以筑基散修、小宗余烬或玉宸下院记名身份切入，锚定陈珩入宵明大泽、血裔身份遭诸真发难、米景世指路君尧前后。  
初始事件：泽国烟波上，一场「审血裔」的公议变成半公开的围猎；你或被雇为见证、或被当成陈玉枢余孽的「旁证」，必须在站队、沉默与卖情报之间当场抉择。  
开场白建议：「宵明大泽的水汽能把人的声音泡软，却泡不软诸真的眼神。他们问的不是你修过什么功法，而是——你认不认识那个叫陈玉枢的名字，以及，你愿不愿意指证一个刚上岸的年轻人。」  
关键NPC立场：**陈珩**——拒跪部分长老以守道统，需要可信任的外围；**米景世**——务实援手，可换人情；**君尧**——希夷山冷眼看棋，投资需价值；**玉宸诸真**——排外且畏魔宗污名。  
主线钩子/支线：血裔公议；希夷山求见；阿鼻剑出现前后的护送／截夺。支线＝泽国市坊气运单、下院名额黄牛、假投名状。  
危险度/规避：高——须规避道君级目光与魔宗截杀队。  
任务方向/奖励：换到玉宸门缝或君尧侧情报；奖励取向＝筑基顶阶丹方、玉宸外门功法、长离岛方向的未来人情。

**四阶（紫府 · 下院大比与长离岛线）**  
切入身份/时点：以下院大比参赛者、长离岛客卿或赤明／怙照观察使身份切入，锚定四院魁首争夺、卫令姜并肩、顾漪交锋之时。  
初始事件：大比第三场，药园边的禁制突然失灵，有人借「陈玉枢余孽」之名搅局；卫令姜的青鸟异象掠过水面，顾漪的魔功已在对岸起手——你被夹在正魔裁判与魁首之争中间。  
开场白建议：「鼓声把宵明大泽的鸟都震起来了。四院座次尚未写定，药园的禁制却先碎了一角；有人喊血裔，有人喊公平，而你手中的号牌告诉你——下一场，你可能要跟魁首热门交手，或替他挡刀。」  
关键NPC立场：**陈珩**——冲魁首与长离岛，可临时同盟；**卫令姜**——破阴谋优先，善感慢热；**顾漪**——好胜，胜负外另有算计；**陈白**——利我者可夺，背刺高危。  
主线钩子/支线：四院大比；长离岛赐封；药园之战；四国事务站队。支线＝禁制修复合同、青鸟异象目击证词、魔功残痕鉴定。  
危险度/规避：高——须规避真传以上与魔宗真传全力。  
任务方向/奖励：大比名次或长离岛客卿席；奖励取向＝紫府功法、玉宸三经外传、与卫令姜／顾漪的立场标记。

**五阶（洞玄 · 东海龙宫与五精线）**  
切入身份/时点：以岁旦评选手、龙宫法会随员或洞天资格竞争者身份切入，锚定龙宫大比、洪泽长生拔罪洞天、小纯阳雷劫前后。  
初始事件：玉泉市坊里玄阙芝易主的瞬间，三方杀意同时抬起；你或是护送、或是竞价、或是被龙宫点名「见证」，下一站就是深海法会与洞天入口。  
开场白建议：「东海的风带着龙涎与铁腥。岁旦评的榜文墨迹未干，已有人在玉泉市坊为了一株芝打穿两间铺面；有人说陈珩会夺头名，有人说他死在回程更划算——而洞天的门，只为带五精出来的人而开。」  
关键NPC立场：**陈珩**——目标五精与洞玄，需可靠外围；**颜熙／陈婵**——法会政治交好对象；**龙宫执事**——要秩序与脸面；**通烜道君**——多在暗处，勿主动招惹。  
主线钩子/支线：岁旦评；龙宫大比夺魁；洪泽洞天取先天五行之精；回程围攻。支线＝玄阙芝真伪、法会座次贿赂、雷劫护道。  
危险度/规避：极高——须规避道君正面与纯阳雷劫余波；回程伏击＝绞肉。  
任务方向/奖励：洞天残精、龙宫人情、护劫份额；奖励取向＝洞玄材料、太素辅助外药、通烜门下见闻（情报级）。

**六阶（金丹 · 通烜真传与周行殿线）**  
切入身份/时点：以金丹真人、周行殿外协或赤明友派特使身份切入，锚定通烜收徒、周行殿执事任命、一品金丹冲击与魔宗暗线升温之时。  
初始事件：真传仪轨的香灰未冷，周行殿已下三道符令——一查血裔档案，二调长离岛闭关物资，三封杀一份来自魔宗的「豢人」残页；你被点名协办其中一道，发现残页上的笔意像极了空空道人的传说。  
开场白建议：「周行殿的灯火亮如白日，却照不进所有人的眼底。通烜道君一收真传，半个宵明大泽的报价单都改了行；你捧着那道符令，听见廊下有人低语：金丹九品，一品才配叫真人——而有人，愿意用九州的血裔，换一品之上的道。」  
关键NPC立场：**通烜道君**——师尊权威，任务即法旨；**陈珩**——真传新贵，闭关与公务两头烧；**卫令姜**——同步冲高品金丹；**陈玉枢眼线**——要档案与人劫时机。  
主线钩子/支线：真传落地；周行殿政务；一品金丹冲击；豢人残页案。支线＝长离岛物资劫案、品级鉴定作假、友派联姻／道侣政治。  
危险度/规避：极高——须规避道君斗法中心与魔宗元师级神降。  
任务方向/奖励：殿署功勋、金丹辅药、真传外围名分；奖励取向＝九还材料、法相预备法门、反豢人的因果护符线索。

**七阶（元神～返虚 · 人劫与法相线）**  
切入身份/时点：以元神真人、一域镇守或反魔联军参谋身份切入，锚定法相十二障关、陈玉枢炼血裔人劫公开化、涂山与古迹同盟成型之时。  
初始事件：某州气运夜半倒灌，血裔名册自行燃烧——陈玉枢的神念或伪神降在云层显影，要「收回」种子；陈珩法相初成，向你抛来的不是求饶，是一张可同死可同胜的联手契。  
开场白建议：「法相撑开的那一夜，九州有三处钟鸣无声。名册在烧，血在响，有人把后裔叫作资粮；你站在云层下，看清陈珩的法相不是用来炫耀的——那是他拒绝被炼化的牙齿。」  
关键NPC立场：**陈珩**——人劫核心，可结盟不可夺志；**陈玉枢**——超阶边缘的布局者，情报优先；**通烜道君／君尧余绪**——正道支点；**涂山葛等**——非人盟友，要安置也要战功。  
主线钩子/支线：法相十二障；血裔反炼化；九州联防；无形埒剑洞等古迹钥匙。支线＝伪神降辨识、气运堤坝、狐族迁徙护卫。  
危险度/规避：贴近顶点——陈玉枢与空空道人级＝超阶巅峰：存在·情报优先／条件性胜利；切忌硬撼本体。  
任务方向/奖励：破豢人局部法、法相障关辅材、联军指挥权；奖励取向＝元神顶阶神通残篇、返虚入门感悟、非人同盟契约。

**八阶（纯阳～合道 · 仙业前沿线）**  
切入身份/时点：以纯阳真君、合道试炼者或道君门下代行身份切入，锚定三灾九难、仙业命题与道主／大罗目光边缘（连载前沿，不写终局胜负）。  
初始事件：纯阳雷火与合道九难的预兆同夜降临，有人在太虚边缘拍卖「道主席的见闻」——假货居多，真货能换一条命；陈珩是否已至这一层以原作进度为准，你则必须决定：为谁护劫，向谁封锁道音。  
开场白建议：「太虚很静，静到能听见自己的道心裂开的声音。风火雷三灾不是天在生气，是路在收费；有人说仙业在合道之后，有人说道主四十九，一步一尸。你立在灾云下，第一次明白：所谓长生，是把名字从别人的资粮册上划掉。」  
关键NPC立场：**通烜道君等道君**——同境或更高，友亦是压；**陈玉枢**——若仍在局则为终局级对手之一；**空空道人**——超然变量；**卫令姜等道侣／道友**——护劫与情义绑定。  
主线钩子/支线：纯阳三灾；合道九难；仙业位业选择；道主级情报战。支线＝假道音拍卖、太虚淬炼护道、跨天同盟。  
危险度/规避：顶点边缘——道主／大罗＝超阶巅峰：存在·情报优先／条件性胜利，严禁「封镇削弱」式取胜。  
任务方向/奖励：合道资粮、灾劫减免阵图、仙业线索；奖励取向＝纯阳／合道法门、洞天升级、与道君级存在的有限盟约（非隶属）。
