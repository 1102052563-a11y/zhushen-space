# 故に人妻は寝取られた。 第一巻

## 剧情

**【作品来源】**
清单全名《故に人妻は寝取られた。 第一巻》。媒介：成人向动画第1卷，**Queen Bee** 出品，销售日 2020-02-28，约19分钟；原作漫画 **あらくれ**《故に人妻は寝取られた。》（コアマガジン，DLsite BJ178893，2019-01-31，232页）。监督・絵コンテ・演出 **伊藤史夫**；声优公开含 **手塚りょうこ／八鍬忍／一生独身／春風つばめ** 等。气质＝人妻 NTR 群像：弱点、酒、租金、邻里权力缝隙。**本档案锚定第一卷动画两话结构**，写关系与情感功能，不展开露骨过程。

**【世界定位】**
现代日本街巷与小店：丈夫缺席、酒意、租金、欢迎会——这些「日常借口」把婚姻的门缝撬开。一句话：人妻在「要守住的生活」与「被看穿的身体」之间被一点点改写立场的世界。玩法是秘密、愧疚、是否告诉丈夫，不是战斗。

**【世界观 · 舞台设定】**
完全现实向成人社会。软规则：①丈夫住院／不在场制造真空；②酒与睡眠模糊同意边界（扮演须玩家确认伦理边界）；③「守住店／守住面子」成为可被利用的语言；④第一卷是群像入口，不是单一女主终身线。世界温度／寒意：咖啡香与消毒水、欢迎会空瓶、手机来电显示「老公」、榻榻米与街灯。NSFW 18禁；档案写事后是否洗脸、是否回信、是否还能笑着营业。

**【地理 · 生活舞台】**
- **人妻NTR茶·喫茶店柜台／厨房**：营业中的公开羞耻场；租金与「守住这个地方」的台词钉在这里。
- **店后／仓库**：短暂躲开客人视线的夹缝。
- **南さん打工的店（居酒屋／小店类）**：欢迎会桌、关店后的地板。
- **南さん家／通勤路**：电话响起的地方。
- **医院方向（远景）**：丈夫所在，道德压力的来源。
- **公寓门与信箱**：秘密是否会变成信件。

功能：柜台＝营业与堕落同框；欢迎会＝酒的借口；电话＝现实刺回。

**【故事主线 · 情感线】**
据 DLsite 第一卷作品介绍：

**话段A「人妻NTR茶」**  
夫妇经营喫茶店，丈夫住院。人妻向大家请求宽限租金。大家伪装好人，趁营业以「你想守住这里吧」压住她的抵抗，使她在客人仍在的空间里成为泄欲出口。情感主轴：生存与店的存续压过羞耻；她学会在笑脸下分裂。

**话段B「南さんはお酒に弱い」**  
人妻 **南さん** 开始打工，欢迎会上醉倒。店长遣散员工后对她下手；过程中丈夫来电，她在朦胧中半醒，分不清梦与现实，行为却未停。情感主轴：酒＝免责幻想；电话＝婚姻的刺；事后她如何面对「记得多少」。

相位总览：A1 租金恳求 → A2 营业中越界 → A3 无抵抗化；B1 欢迎会 → B2 关店后 → B3 来电半醒 → B4 余韵。第一卷收束：两个女人各自把秘密带回生活；系列第二卷另档（原作尚有「人妻は2度犯される」「故に町内会長は…」等篇）。

结局方向：NTR 群像式，非传统纯爱 HE。主题：故に——「所以」往往不是爱，而是弱点被击中。

**【可攻略角色 / 主要人物】**
- **喫茶店人妻（第一话核心·姓名公开页未固定则写职能名）**｜身份：店主妻子，丈夫住院｜性格：要强、为店低头、羞耻中仍想营业｜角色类型：守店型人妻｜萌点：柜台笑脸与背后崩溃的落差｜个人线：租金→越界→是否继续开门｜心结：店与丈夫哪个先崩。

- **大家（第一话男侧）**｜身份：房东｜性格：伪善、善用「守住」一词｜角色类型：权力缝隙利用者｜功能：把生存语言变成命令。

- **南さん（第二话核心）**｜身份：人妻打工｜性格：酒弱、社交时好说话｜角色类型：酒后脆弱型｜萌点：来电时的半醒眼神｜个人线：欢迎会→关店→电话→如何记忆｜心结：要不要当作梦。

- **店長（南さん侧）**｜欢迎会主人／雇主｜遣散员工的那个人。

- **两位丈夫（远景）**｜住院者与来电者｜道德压力，不写脸也可以有存在感。

- **常连客／欢迎会同事（群像）**｜目击或差点目击的社会眼。

- **原作扩展人妻（美穂等）**｜漫画篇目人物，第一卷动画未收则仅作系列远景。

**【人际关系网 / 社团势力】**
婚姻对：人妻—丈夫。越界对：人妻—大家／店长。社会眼：客人与同事。无帮派；「势力」＝租金、职位、酒桌座位。

**【情感事件 · 名场面】**
1. 「你想守住这里吧」这句话落下。  
2. 柜台外有客人说话，柜台内世界已歪。  
3. 欢迎会最后一瓶空了。  
4. 关店卷帘落下。  
5. 手机屏亮起「老公」。  
6. 半醒的「……亲爱的？」不知对谁说。  
7. 次日店门照常开，笑脸慢半拍。

**【隐藏剧情 · 真结局 · 伏笔】**
丈夫是否会提前出院；租金是否成为长期契约；南さん记得多少。物证：催租短信、欢迎会收据、未接来电记录、柜台下的手印想象。第二卷篇目作系列钩子，本档不展开。

**【氛围基调 · 雷区】**
苦涩、背德、生活气息重。忌：浪漫化强迫；未成年；写成清剿任务或强弱排名。最适切入：租金恳求当日；欢迎会散场；来电震动的那一秒。
