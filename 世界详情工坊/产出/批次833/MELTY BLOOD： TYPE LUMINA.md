# MELTY BLOOD: TYPE LUMINA

## 剧情

**【作品来源】**
《MELTY BLOOD: TYPE LUMINA》（メルティブラッド：タイプルミナ，通称「月姬格斗：TYPE LUMINA」）由 TYPE-MOON 与 French-Bread（フランスパン）主导、Project Lumina（含 Aniplex 等）发行，2021 年 9 月 30 日全球同步登陆 Switch／PS4／Xbox One／Windows。剧情由奈须蘑菇撰写／监修，角色原案武内崇，对战系统由 French-Bread 重做；后日免费 DLC 扩充角色与追加剧情（含爱尔奎特／希耶尔 Story B、诺耶尔 Boss Rush、Fate 客串与猫阿尔克混沌线）。本档案锚定**TYPE LUMINA 故事模式／街机式个人线**所呈现的「月姬重制连续体」群像与情感舞台，气质为：总耶市深夜的重逢与擦肩、代行者与真祖的别扭对话、远野邸门禁与女仆茶点、诺耶尔的嫉妒与自我证明、以及「对战只是人物说话的方式」——侧重关系演绎、都市夜色与各人内心的裂缝，而非赛事或连招。

**【世界定位】**
《月姬 -A piece of blue glass moon-》本篇事件约**十天前**的 2014 年、日本都市**总耶市**。吸血鬼与教会、远野家、真祖与转生体的影子已在街角潜伏，却尚未完全掀开本篇的命运。核心舞台＝总耶夜街／远野邸／学校与公园／地下巢穴周边；核心玩法＝以日常身份走进某一条角色个人线，听他们在「开战」前后真正想说的话，推进关系与心结，而不是刷战力表。

**【世界观 · 舞台设定】**
表层是 2010 年代日本地方都市的秋冬前夜：便利店冷光、自动售货机、校园走廊、远野邸的长廊与后院、偶尔飘来咖喱香的学姐宿舍方向。人们把「连续失踪／吸血传闻」当都市传说口耳相传，白天仍照常上课、打工、打扫；夜里才有人摘下眼镜、穿上法衣、或把长发束起准备「去见不该见的人」。

里层只写情感侧：存在死徒（如弗洛夫·阿尔汉格尔）、教会埋葬机关的代行者（希耶尔、诺耶尔）、真祖公主（爱尔奎特·布伦史塔德）、转生体与蛇（迈克尔·罗亚·瓦尔丹琼）、远野血脉的异能与门禁（远野秋叶、远野志贵）、寄养家的妹妹（有间宫古）、以及偶尔从别的故事掉进来的「客人」（阿尔托莉雅等 Fate 侧、猫阿尔克）。这些解释「夜里为何有人影」「为何某人突然请假／代课／夜巡」，**不写成力量体系或阶位对照**。故事模式的「对战」在休闲档案中一律读作**关系冲突的戏剧化**：谁拦谁的路、谁想被谁看见、谁怕自己变成怪物。

与旧作《MELTY BLOOD》本篇（三咲町、希翁、瓦拉齐亚之夜）的关系：TYPE LUMINA 是**重制连续体重启**，舞台从旧作的三咲町挪到总耶市，时间点移到月姬重制本篇前约十日；希翁·艾尔特纳姆·亚特拉斯等旧主线核心因时间线未登场。本档案以 TYPE LUMINA 为准，旧 MB 仅作系列气质参考，不混写「一年后夏夜希翁线」为主桌。

世界的温度来自：志贵是否又被秋叶骂晚归、翡翠是否在门廊点灯、琥珀是否端出味道可疑却暖胃的汤、爱尔奎特是否第一次学会用自动售货机、希耶尔是否把咖喱分给「麻烦的后辈」、诺耶尔是否在代课讲台上假装镇定、宫古是否还在说「哥哥是我家的」。

**【地理 · 生活舞台】**
- **总耶市夜街／商店街**：传闻发酵地；志贵夜巡、爱尔奎特路过、诺耶尔跟踪、希耶尔观察的主走廊。街灯下的对话比任何「胜负」更重要。
- **公园与河岸／空地**：真祖出没与擦肩点；爱尔奎特与志贵命运性交会的「前摇」常在这里发生。
- **远野邸**：秋叶的门禁、翡翠的沉默迎接、琥珀的茶与「请平安回来」——归宅的情感锚点。诺耶尔在邸外徘徊、秋叶察觉外来者，都是宅邸政治的一环。
- **志贵的学校／教室／操场**：诺耶尔以代课教师身份潜入的舞台；白天普通、夜里可能被「忘乡之钟」类幻象扭曲（死徒诺耶尔 IF 中）。
- **希耶尔相关的公寓／教会联络点（对话中）**：咖喱的味道、代行者的报告、对诺耶尔「再证明一次」的无奈。
- **地下巢穴／死徒活动痕迹**：弗洛夫等的领域感；诺耶尔与弗洛夫冲突、志贵追凶的「不该久留」之处——读作心理压力舞台，而非地下城刷怪。
- **有间家方向**：宫古练拳、宣示主权、试图「救回哥哥」的青春残响。
- **轧间红摩相关的旧影（对话／遭遇）**：远野血脉创伤的具现之一，情感上＝志贵与秋叶都想回避却绕不开的过去。
- **Fate 客串舞台（平行一夜）**：Saber、玛修、牛若丸、基督山伯爵等「掉进总耶」的喜剧／史诗一夜——读作群像扩展，不抢月姬本桌的情感主权。

**【故事主线 · 情感线】**
TYPE LUMINA **没有单一贯穿主线**。每位可操作角色拥有独立的故事模式（多场遭遇＋视觉小说式对话＋个人结局）；各线发生在相近的总耶之夜，却常是**并行 IF／前日谈切片**——爱尔奎特线与希耶尔线不必是同一世界的同一夜。休闲档案把这种结构读成：**你可以走进谁的视角，就先听谁的心跳**。

**共通空气：本篇前十天的总耶**  
月亮显得不祥，吸血与失踪的传闻让敏感的人无法安睡。**远野志贵**在邸内与街上夜巡；**爱尔奎特**为追踪最大之敌而来到这座城市；**希耶尔**以学生身份潜伏、同时履行代行者职责；**诺耶尔**晚她约一个月抵达，嫉妒学姐的「学生伪装」，自己改以**代课老师**潜入；**远野秋叶**收紧门禁；**翡翠**与**琥珀**在宅内维持日常；**有间宫古**仍把志贵当「该被带回有间家」的哥哥；**弗洛夫**与**罗亚**的影子在地下与转生线上游移。故事模式的「胜负」只是让这些人不得不停下来对话的借口。

**远野志贵个人线（前日谈感）**  
志贵察觉不祥的月色，在远野邸周边巡逻，撞见擅自接近的**诺耶尔**，交谈后交锋；继而直面**弗洛夫**，指责其与镇上吸血鬼事件有关。弗洛夫否认「全是我干的」式简单罪名，志贵几乎被「看见吸血鬼」本身激起的愤怒淹没，却仍撑过这场遭遇。事后他宁可不想再面对此类存在——却在休憩时瞥见金发的女人（**爱尔奎特**）走过，某种东西「碎裂」，他隐约意识到：与吸血鬼相关的命运才刚刚开始。  
情感核＝「普通高中生想守住家人与日常」与「直死之魔眼带来的无法回头」。HE 向读法：他仍选择夜巡、仍选择把人当人；BE 向读法：愤怒与恐惧先于理性，把所有「非人」一刀切。

**爱尔奎特·布伦史塔德个人线**  
真祖公主为追最大之敌（罗亚相关）抵达总耶，命运性交会的少年就在这座城里，但故事模式故意呈现「更早的可能性」切片，避免剧透月姬本篇全貌。线中她在街上、公园、与代行者／死徒的遭遇里展现天真直球与真祖威严的落差；对志贵的在意会以「为什么要拦我／为什么你的眼睛很烦」的形式溢出。后日追加的 Story B（基于奈须草稿）补强她与希耶尔、与城市的情感往返。  
情感核＝「千年的孤独」撞上「想被当作一个人对待的一夜」。HE：她愿意为总耶多留一步、学会自动售货机与「明天见」；坏方向：只把城市当猎场，错过那双会痛的眼睛。

**红爱尔奎特（暴走侧）个人线**  
作为独立登场的「失控之月」，她把食欲、独占与破坏欲摊在台面上。休闲读法＝爱尔奎特内心被压抑面的戏剧化外放：若无人用温柔与界限同时接住她，她会把在意的人一并撕碎。与本篇爱尔奎特的关系不是「数值变体」，而是「同一份爱意的两种说法」。

**希耶尔个人线**  
埋葬机关代行者、以学生身份潜伏总耶。故事围绕调查死徒活动、监视罗亚相关、与真祖擦肩、以及如何处理「软弱却仍想证明自己」的后辈**诺耶尔**展开。后日 Story B 与爱尔奎特侧相互映照。Powered Ciel（完全武装希耶尔）DLC 线则把「职责压过一切」推到极限——包括更残酷的 IF 后日（半死徒化的志贵、真祖城堡等），情感上读作：**当守护变成处刑手册，她还能否记得咖喱的温度**。  
情感核＝信仰、职责、对志贵的复杂心意、对诺耶尔的恨铁不成钢。HE 向：她允许自己在任务缝隙里当学姐；坏方向：只剩「处刑名单」。

**诺耶尔个人线（本篇气质最强的「想被看见」线）**  
依据希耶尔的情报晚约一个月到总耶。得知希耶尔伪装成学生后强烈嫉妒——自己装不了学生，便以**代课老师**身份进入志贵所在学校。在远野邸外打探时被**远野秋叶**察觉并交锋，诺耶尔嗅到血味不敢入邸，落荒而逃。追查至地下巢穴遭遇**弗洛夫**；对方称无意与杂鱼缠斗、愿放过她，反激怒她「证明给希耶尔看」的执念。她一度占上风，弗洛夫认真后陷入绝境，最终由希耶尔收拾残局。结局侧：获教会批准约一年假期，放松时在巷中看见**弓塚五月**摇晃的身影，心情正好地上前——不祥的余韵指向本篇可能发生的事。  
情感核＝自卑、嫉妒、施虐与受害记忆缠在一起的「想被学姐／被谁肯定」。HE 向：假期里有人认真听她说话、不拿她和希耶尔比较；坏方向：用暴力证明存在，直到把自己证明成怪物。

**死徒诺耶尔（DLC IF）线**  
注射拟似原理相关药剂后死徒化，外貌退回少女模样，能力暴涨并以「忘乡之钟」在学校具现故乡法国教堂。她以狩猎为欢、饮血、甚至对远野邸相关者下手（IF 中含对琥珀等的残酷）。志贵为复仇而来反被蔷薇魔眼困住；**罗亚**上门互嘲——夸她是「丑陋的发明」又贬她仍执着弱者的课题、永远逊于艾蕾西亚／希耶尔。她击倒罗亚后身体开始崩坏；**希耶尔**现身，以代行者的「救赎」一击结束她，并为过去的同僚祷告。  
情感核＝「若变强就能被爱」的诅咒。结局残酷而清晰：没有人在这条 IF 里获得轻松的 HE；它反衬本篇诺耶尔线「还来得及被当人」的可贵。

**远野秋叶个人线**  
当主之妹收紧门禁，把一切外来者（诺耶尔、夜巡的哥哥、可疑的金发女人）都当成风险。线中的交锋读作「兄控与失去恐惧」：她骂得越凶，越说明一年一年守住的日常有多脆。与志贵的关系＝门禁战争背后的「不许死在外面」。

**翡翠／琥珀／翡翠＆琥珀个人线**  
宅内双子女仆。翡翠以沉默与洁癖照料志贵；琥珀以笑脸、药草与腹黑体贴维持宅邸运转。组合线常带喜剧与默契，情感核＝「主人平安比本人自觉更重要」。谁能帮她们分担家务、谁能听懂翡翠的短句与琥珀的假笑，谁就走进远野家的内侧。

**有间宫古个人线**  
寄养家的妹妹位，练八极拳，试图把志贵「救回」有间家或至少从危险里拽出来。Boss Rush 等以她为焦点的内容强化「小孩子气的认真」。情感核＝宣示主权的青春与无力感并存；吃「陪她练完一套再谈大道理」。

**轧间红摩、弗洛夫·阿尔汉格尔、迈克尔·罗亚·瓦尔丹琼**  
分别承担远野血脉创伤、死徒压迫感、蛇之转生冷嘲的「压力源角色」。休闲档案不写战力榜，只写：红摩让志贵／秋叶想起血的气味；弗洛夫让诺耶尔的自尊碎裂又燃烧；罗亚用语言把人剥成「容器」与「失败作」。与他们对峙的胜负，永远是「你是否还把自己当人」。

**苍崎青子、马里奥·贾洛·维斯蒂诺等 DLC 扩展**  
青子被雇处理潜伏总耶的罗亚相关、保护研究资料、途中处理死徒化诺耶尔等——把「魔法使的任性正义」丢进月姬夜色。马里奥作为教会侧少年枢机相关人物，提供另一套「利用／交易／笑容」的社交接口。情感上可作「外来者如何在总耶过夜」的支线桌。

**Fate 客串与猫阿尔克混沌线**  
Saber（阿尔托莉雅·潘德拉贡）、玛修·基列莱特、牛若丸、基督山伯爵等以一夜访客身份登场；Boss Rush 后期引入猫阿尔克与混沌交叉。读作群像嘉年华与吐槽阀门，不改写月姬本桌的情感因果。莲的剪影等后日伏笔指向系列延续（Twi-Lumina 等），本档案点到为止。

**多线并行的「结局」读法**  
- 各角色故事模式通关＝听见该角色的 HE／苦涩收束；全员通关解锁的 Boss Rush 是「群像压力测试」，不是等级墙。  
- 没有「同时攻略全员无代价」的官方 True：真结局感来自你选定优先陪伴的人，让她／他在本篇前十天里少碎裂一点。  
- 与月姬本篇的衔接：志贵瞥见金发女人、诺耶尔看见五月、希耶尔继续潜伏——TYPE LUMINA 是门廊上的灯，本篇是门后的夜。

**【可攻略角色 / 主要人物】**

**远野志贵（主人公位・高中生・直死之魔眼）**｜外貌：黑发，常戴眼镜；摘镜后眼神锐利而疲惫。CV（TYPE LUMINA）：金本凉辅。｜性格：老好人、不解风情、关键时异常固执；对「又要把谁拖进血里」自责，却无法对传闻坐视。｜角色类型：日常侧男主／笨拙守护者。｜萌点/魅力：先帮忙再问设定；被秋叶骂仍偷溜出门；对诺耶尔、爱尔奎特、希耶尔都能说出「你不是怪物／工具」一类的笨拙肯定。｜个人线剧情：夜巡→遇诺耶尔→对峙弗洛夫→瞥见爱尔奎特→命运开端。｜与主角关系：世界行动中心；契约者可作同学／夜巡同伴／「帮我瞒过秋叶」的共犯。

**爱尔奎特·布伦史塔德（可攻略・真祖）**｜外貌：金发红眼，白与金的轻装；夜色中过分干净。CV：长谷川育美。｜性格：天真直球、对「有趣的人类」上心、对代行者又戒又逗；独占欲以任性形式出现。｜角色类型：月之公主／任性恋人位。｜萌点/魅力：听不懂人类的绕弯却听得懂「你在痛」；护食一般护在意的人；学用售货机与「普通约会式散步」。｜个人线剧情：追敌至总耶→与志贵命运性交会的前摇→与代行者／死徒擦肩→Story B 补完心境。｜与主角关系：吃「陪她做普通都市夜游」「不拿她比较别人」「允许她任性一次再认真拦她一次」。
