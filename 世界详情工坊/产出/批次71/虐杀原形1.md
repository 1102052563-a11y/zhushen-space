## 剧情

**【作品来源】**
《虐杀原形》（Prototype，2009）由加拿大 Radical Entertainment 开发、Activision 发行的开放世界动作游戏，平台含 PC/Xbox 360/PS3，属「虐杀原形」系列正传第一作（续作为 2012《虐杀原形2》）。本作以第一人称记忆碎片「阴谋之网（Web of Intrigue）」与第三人称都市跑酷战斗并重，气质黑冷、阴谋感重、道德灰：主角不是传统英雄，而是由病毒重塑的「类人」。本档案以**第一作主线与可公开设定**为准，不把续作 James Heller 线倒灌进本作终局；卡片锚点（世界库提要）：伊丽莎白·格林（母体）之战——A 哥在时代广场对抗化身为巨型触手怪物的伊丽莎白·格林，军方火力与 Hunter 怪物混战。检索优先官方设定、Wikipedia 条目与游戏 wiki（Powers / Timeline / Bloodtox 等），冲突处从游戏主线。

**【世界定位】**
2008 年前后的现代美国，舞台几乎锁死在**被隔离的曼哈顿岛**。基因工程公司 Gentek 与黑衣生物战部队 **Blackwatch** 长期合作研发病毒武器；人类科学家 **Alex Mercer** 在宾夕法尼亚车站（Penn Station）释放 **Blacklight（黑光）病毒**样本后被枪杀，病毒以他的尸体为模板重塑出玩家操控的「原型体」——一个拥有吞噬、记忆吸收与形态武器的超级感染体。故事线是：失忆的「Alex」追查身份真相 → 曼哈顿爆发全面感染 → 与妹妹 **Dana Mercer**、病理学家 **Bradley Ragland** 周旋 → 母体 **Elizabeth Greene** 出笼并接管蜂巢意识 → 最终阻止 **General Peter Randall** 的核清除计划并干掉至高猎杀者。一句话：这是「病毒塑形+吞噬记忆」的现代都市灾难世界，契约者是外来变数，不是官方天选。

**【世界观 · 力量体系】**
力量本源**不是灵气、功法、仙道**，而是 **Blacklight / Redlight 系工程病毒对宿主基因组的重写**。病毒激活所谓「垃圾 DNA」，让细胞在极短时间内完成结构重组：多数人器官衰竭而死或变成无智感染者；极少数宿主（原型体、母体、特定猎杀者）获得超常体能、再生、形态武器与蜂巢指挥权。成长逻辑：**吞噬（Consume）** 是核心——吞食生物质回复伤势，并拷贝目标的外貌、技能、记忆与权限；「进化点」叙事上表现为原型体通过战斗与吞噬不断解锁形态模块。伪装（Disguise）是战术，不是魔法隐身：一旦展露利爪/护甲/毁灭技，伪装立刻破功。

死亡与复活：
- 普通人类：一枪、一次爆炸、一次猎杀者扑杀即可永久死亡。
- 普通感染者 / Walker：可被火力、斩击、碾压消灭；蜂巢与水源塔可再生出新的个体。
- 原型体（Alex）：常规热武器与坠落几乎无效；重创后靠生物质再生。核爆级能量可把身体打散，但仍可能以残骸+吞噬（如乌鸦）缓慢重建——**有代价、有时间窗、绝非无限满血读档**。
- 母体 Greene：作为 Redlight 完美宿主，可指挥蜂巢、催生猎杀者，被原型体完整吞噬后其人格与蜂巢指挥权被吸收，本体消亡。
- 至高猎杀者（Supreme Hunter）：由寄生虫被 Greene 排斥后形成的独立顶级个体，可吞噬精英、伪装人类，最终被 Alex 斩杀。

特殊系统（按原作）：
1. **形态武器**：Claws 利爪、Blade 刃臂、Whipfist 鞭拳、Hammerfists 锤拳、Musclemass 肌群强化；防御侧 Armor 重甲、Shield 生物盾。进攻与防御通常各占一槽，可组合。
2. **感官**：Infected Vision 感染视界、Thermal Vision 热视，用于在烟雾 / Bloodtox 雾中找目标。
3. **毁灭技 Devastators**（Critical Mass 状态）：Tendril Barrage 触手弹幕、Groundspike Graveyard 地刺坟场、Critical Pain 临界痛楚等，以大量生物质换取范围杀伤。
4. **载具与军械**：吞噬驾驶员后可劫持坦克、武装直升机、炮塔；枪械本身不构成超凡，但在伪装渗透时是必要道具。
5. **反制剂 Bloodtox（血毒，编号 A-113A 等）**：Blackwatch 研发的红色抗病毒雾剂，对感染生物质持续腐蚀，对未感染人类相对无害；可把地下感染逼上地面，是中后期战略钥匙。
6. **蜂巢 / 水源塔 / 据点**：感染侧用 Hive 与水塔产怪；军方向用基地、检查站、病毒探测器、超级士兵维持蓝区。

战力阶梯（按破坏力表现，名称用原作生态位，**禁止修仙词**）：
1. **曼哈顿平民 / 恐慌市民 / 基础新兵**：徒手或手枪级，坠楼、围殴、一次感染抓挠即可致命。乐园观感≈一阶。
2. **海军陆战队精锐 / Blackwatch 普通战斗员 / 早期 Walker**：班组火力、装甲车协防；感染者能撕开凡人，但仍惧重机枪与集火。乐园观感≈二阶。
3. **Blackwatch 精英小队 / 早期猎杀者雏形 / 初觉醒原型体（仅有基础吞噬+速度）**：可端掉街垒、掀翻轻型车辆；近距闪避第一波枪火。乐园观感≈三阶。
4. **成熟猎杀者 Hunter / 超级士兵（D-Code 等）/ 中期原型体（利爪+护甲+跑墙滑翔）**：摧毁据点、掀平建筑群边缘；肉身硬抗常规热武器与轻型导弹；城市巷战中的精英 Boss 感。乐园观感≈四阶。
5. **领袖猎杀者 Leader Hunter / 九头蛇级大型感染体 Hydra / 蜂巢核心指挥 / 后期原型体（刃臂鞭拳+毁灭技）**：一击夷平街区、可让整片防区陷落；蜂巢意识让整区感染者协同。乐园观感≈五阶（城市级）。
6. **母体形态 Elizabeth Greene（时代广场巨体）/ 至高猎杀者 Supreme Hunter / 终局 Alex Mercer / 核清除威慑本身**：覆盖大城核心区的灾难级战斗；肉身级对抗装甲集群与战略级选项（核弹）形成对峙。乐园观感≈六阶（地区级灾难·镇国感）。本作**无七阶及以上常驻个体**——曼哈顿核爆是文明级威慑工具，不是角色本体破坏力。

乐园阶位映射（对照《阶位战力图鉴》破坏力标尺，宁低勿高）：平民/新兵≈一阶；陆战队精锐·普通 Blackwatch·Walker≈二阶；精英小队·早期原型体·轻型猎杀者≈三阶；Hunter·超级士兵·中期 Alex≈四阶；Leader Hunter·Hydra·蜂巢核心·后期 Alex≈五阶；Greene 母体巨形·Supreme Hunter·终局 Alex·核清除对峙≈六阶。世界顶点（终局原型体 / 母体 / 至高猎杀者 / Randall 核选项）=**六阶顶**，低阶切入一律规避顶点·情报优先；条件性胜利靠布局、Bloodtox、伪装渗透与多方火力，**严禁「战力归零式削弱」**。

**【地理 · 舞台】**
舞台几乎等于一座被锁死的岛城：**纽约曼哈顿**。Blacklight 难以跨过大水体，疫情与主战场因此被海面圈定——这是世界规则，不是剧情巧合。岛上按控制权动态染色：
- **蓝区（军事控制）**：检查站、铁丝网、巡逻直升机、Blackwatch 与海军陆战队混编；平民被疏散或集中，宵禁与「可疑感染」就地处决是日常。
- **红区（感染控制）**：蜂巢、水源塔、变异建筑、成群 Walker 与 Hunter；街道扭曲、生物质覆盖墙面，人类进入=送生物质。
- **紫区（争夺区）**：炮火与尖啸并存，最适合契约者捡漏、也最容易被两头打。

关键地点（可玩、可写）：
- **Gentek 总部 / 曼哈顿中城实验室**：Greene 长期关押处、Blacklight 合成现场；Alex 失忆后首要潜入目标；Greene 出笼名场面发生于此。
- **宾夕法尼亚车站（Penn Station）**：原版人类 Alex 砸开病毒瓶、被枪杀的起点；阴谋之网记忆的核心锚。
- **停尸房 / 初期街道**：原型体苏醒、第一次中弹不死后逃亡的井底舞台。
- **Dana 的安全屋**：妹妹提供情报与情感锚；后期被 Leader Hunter 袭击、Dana 被掳走的关键点。
- **Bradley Ragland 的地下诊所 / 病理设施**：唯一能以医学手段处理「寄生虫」的人类专家据点；中期救治与武器化寄生虫的关键。
- **核心蜂巢 Core Hive**：Greene 的地下指挥节点；Alex 借热压坦克破门、注射寄生虫的副本。
- **时代广场及周边主干道（卡片锚点）**：Bloodtox 逼出地下感染后，Greene 化巨型母体，军方火力与 Hunter 混战的公开战场。
- **Blackwatch 基地 / 红冠指挥（Red Crown）**：隔离令、超级士兵投放、Bloodtox 泵送、核选项决策的神经中枢。
- **航空母舰 USS Ronald Reagan**：终局棋盘——核弹装载、Randall 坐镇、至高猎杀者伪装 Cross 登舰、Alex 将核弹带离大西洋引爆。
- **前史舞台 Hope, Idaho（爱达荷希望镇）**：1968–1969 Redlight 试验场；Greene 成为母体、Pariah 出生、Blackwatch「利他行动」灭口的原罪之地。低阶只作记忆与档案，不宜整图开放。

势力地盘：美军公开面（海军陆战队）守街；Blackwatch 黑衣人管真相与处决；Gentek 管实验室与论文谎言；感染侧以 Greene 为脑、蜂巢为腺。契约者若在一阶宣称「我要单挑母体」，本地人会当疯子——正确反应。

**【世界剧情线】**
① **前史·希望镇与红光（1960s）**  
1962 年前后，Blackwatch 作为生物战 / 遏制黑部队成型。Carnival 等项目在封闭社区试验病毒。1968 年爱达荷 **Hope** 镇爆发 **Redlight（红光）** 疫情：多数居民异化，**Elizabeth Greene**（约 1951 年生，当地居民）成为完美宿主——病毒在她体内进化，她获得对感染者的蜂巢指挥，并在混乱中产下代号 **Pariah** 的孩子。1969 年 **Operation Altruistic（利他行动）** 中，时任军官、日后将军 **Peter Randall** 率队抓获 Greene 与 Pariah，灭镇掩盖为暴乱。Greene 此后数十年被关押研究，衰老几乎停滞，身体持续产出新毒株；Pariah 另案实验（Project Crusade 等）。1976 年 **Raymond McMullen** 创立 Gentek；1989 年起 Greene 被移至曼哈顿 Gentek 设施，成为受试体 001，**Blacklight** 项目在此以她的血样为母本推进。因果：曼哈顿灾难的种子在四十年前已埋下；Blackwatch 的信条「为守红线，可烧自己人」从希望镇开始写进基因。

② **开篇·停尸房与失忆原型体**  
2008 年前后某日（游戏 Day 00）：Gentek 科学家 **Alex Mercer** 识破项目真相后，偷走一瓶 Blacklight 样本作「保险」。Blackwatch 在 **Penn Station** 围堵他；他愤而砸碎瓶子，病毒释放，本人被乱枪击毙。病毒经伤口进入尸体，以细胞级重建造出一个**相信自己是 Alex Mercer 的原型体**。玩家故事从停尸房苏醒开始：Blackwatch 清场、科学家被灭口，原型体中弹不死后撞破封锁冲上曼哈顿街头。他发现自己能跑墙、高跃、滑翔、再生，更能**吞噬**人类以获取外貌与记忆。此阶段世界状态：疫情尚未全城公开，军方以「恐怖袭击 / 生化事故」口径行动；强者如天的是持枪部队与未知感染体。名场面：停尸房苏醒、第一次吞噬士兵换装渗透、屋顶跑酷穿越封锁线。因果：失忆+复仇欲成型；「我是谁」成为贯穿主线的问题。

③ **立名·Dana、Karen 与 Gentek 潜入**  
原型体联系妹妹 **Dana Mercer**（调查记者气质的大学生）。Dana 此前已在查 Gentek，提供目标名单与避难所。原型体沿阴谋之网吞噬相关人员，拼出「Blacklight 与 Blackwatch」拼图。前女友 **Karen Parker**（Gentek 相关研究人员）短暂合作，诱导他采集样本、进入实验室——实则为 Blackwatch 布局。原型体潜入 Gentek 试图从 **Elizabeth Greene** 处逼问答案：Greene 被释放 / 挣脱收容，当场把士兵改造成猎杀者，并向原型体灌注希望镇残影。曼哈顿隔离升级，**Red Crown** 指挥建立，感染百分比逐日爬升。阶段结局：Alex 从「被追的怪物」变成双方都要猎杀的「Zeus（宙斯）」代号目标；Dana 成为他唯一近似人性的锚。

④ **寄生虫、Ragland 与背叛税**  
**Captain Robert Cross**（「The Specialist」）受 Randall 指派猎杀 Zeus。他在蜂巢相关行动中与 Alex 对峙，成功注入 **Blackwatch 寄生虫**，压制原型体能力。Karen 的背叛在此线兑现。濒死的 Alex 找到前 Gentek 病理学家 **Bradley Ragland**：Ragland 以残酷但有效的方式（涉及猎杀者生物质与手术）移除寄生虫，并把它改造成可投放的生物武器。此阶段主题：**人类盟友全不可默认**——有用者可救命，也可用你换命。名场面：寄生虫发作的无力感、Ragland 诊所的血腥手术、Cross 作为「人类天花板猎手」的压迫感。

⑤ **Dana 被掳与核心蜂巢（中期质变）**  
Greene 的蜂巢意识扩大。一只 **Leader Hunter** 袭击安全屋，掳走 Dana。Alex 为定位妹妹，吞噬领袖级猎杀者以接入蜂巢感知，锁定 **Core Hive**。他借助军方火力与热压坦克破开外围，在核心与 Greene 正面对峙，将改造寄生虫注入母体。Greene 的身体排斥寄生虫，排出大量生物质，诞下 **Supreme Hunter（至高猎杀者）** 雏形；Greene 本体逃走，Dana 被救出时已接近昏迷 / 紧张性状态，交由 Ragland 看护。阶段结局：Alex 战力质变（刃臂、护甲、更完整毁灭技解锁的叙事窗口），同时把「至高猎杀者」这颗定时炸弹放进棋盘。

⑥ **Bloodtox、上校溃逃与格局战**  
感染覆盖率飙升（中后期可至大半个岛）。匿名联络人（后期揭示线）向 Alex 提供情报：Blackwatch 的 **Bloodtox** 可逼出地下感染。Alex 协助或夺取 Bloodtox 泵送系统，把红区从下水道赶到街道。军方与 Blackwatch 内部分裂：**Colonel Ian Taggart** 恐慌违令撤离；Randall 坚持「守红线」并启动更极端预案。超级士兵投入巷战；蜂巢与基地互相拔点。此阶段个人恩怨被阵营战争吞没：你救一个街区，可能只是给核选项多一天舆论；你毁一个泵站，可能让红区吞掉蓝区。

⑦ **质变副本·时代广场母体之战（卡片锚点卷）**  
Bloodtox 生效后，**Elizabeth Greene** 被迫现身地表，在**时代广场一带**化作覆盖建筑群的巨型生物质母体：触手抽打街道，Hunter 与 Walker 如潮水，坦克与武装直升机倾泻火力却难以根除核心。Alex 在军方炮火与感染潮的夹缝中攀上母体、撕开防护、最终**完整吞噬 Greene**。吞噬带来希望镇全貌记忆与有限的蜂巢感知，剩余感染者协同性骤降，但城市已残。名场面要素（写正文必用）：入口门槛=Bloodtox 作业区与戒严线；核心争夺物=母体本体 / 蜂巢指挥权；原作级压迫者=Greene 巨形+环伺 Hunter；出逃/分赃时的追杀=吞噬后的军方「顺便灭掉 Zeus」。阶段结局：Greene 死（被吸收），公开战场暂时倒向「人类火力+怪物清道夫」的畸形合作；真正的终局阴影转向核选项与至高猎杀者。

⑧ **终局·里根号、核弹与至高猎杀者**  
吞噬 Greene 后，Alex 得知 Randall 准备以核弹「清洗」曼哈顿（Operation Firebreak 一类口径）。他沿情报链登上 **USS Ronald Reagan**，吞噬 **General Peter Randall**，掌握发射权限相关记忆。真正的联络人身份揭晓：此前以 **Robert Cross** 面目出现的「线人」，其实是已吞噬真·Cross 并伪装的 **Supreme Hunter**——它要吞掉 Alex，以获得扛过核爆的强度，再重启疫情。航母甲板上决战：至高猎杀者使用触手、地刺与高机动；Alex 以刃臂、投掷军械、毁灭技与生物质拉锯，最终斩杀对方。随后他带着核弹飞离曼哈顿，在大西洋上空 / 海域引爆，自己被爆炸撕碎，残骸漂回后吞噬乌鸦缓慢再生。公开叙事把胜利记在军方「反恐」账上；Alex 回到仍在燃烧的岛上，独白自己「比人类少一点，也比人类多一点」，工作未完。谁活谁死（本作终局）：Greene 死；真·Cross 死；Randall 死；Taggart 线以溃逃/被利用收场；Dana 存活并逐渐恢复；Ragland 存活；Karen 作为叛徒工具人退场（生死以任务分支/吞噬与否为准，写手勿强行洗白）；原型体 Alex 存活。世界状态：曼哈顿重创、Blackwatch 公开度上升、Blacklight 未根除——为续作埋线，但**本世界条目终局停在一作**。

⑨ **微观日常（可高频介入）**  
除大场面外，正文应优先：检查站盘问与就地枪决、平民抢水抢药、黑市卖假通行证、蜂巢清剿悬赏、水源塔爆破、伪装成士兵混入运输车、用病毒探测器误报嫁祸、帮 Ragland 运样本、给 Dana 找抗生素、在紫区抢直升机、偷 Bloodtox 罐、卖「谁是 Zeus」的假情报。AI 勿一上来母体大战。

⑩ **蝴蝶效应边界**  
可改：救下某街区平民、提前炸掉某蜂巢、让某次巡逻全灭、抢非核心样本、延迟 Karen 背叛被识破的时间。不可无代价改：Greene 作为母体的生态位、Penn Station 释放事件的历史事实、核选项的存在、原型体「吞噬记忆」的规则、一作终局「核弹离岛引爆+Alex 再生」。你若在低阶公开宣称自己会吞噬，Blackwatch 会把你标成第二头 Zeus。

- **美国海军陆战队 / 公开军事存在**：宗旨——表面上的戒严与民防。地盘：蓝区街道、检查站、火力网。代表：一线军官、Taggart 协同体系。战力：一~四阶为主。作用：开放世界炮灰与载具来源；常被 Blackwatch 当枪使。

- **感染蜂巢（Greene 势力）**：宗旨——扩散、同化、以母体为神。地盘：红区、Hive、水源塔、地下核心。代表：**Elizabeth Greene**、Leader Hunter、Hydra。战力：二~六阶。作用：城市灾害本体；时代广场锚点战的主力敌。
- **地下医疗 / 灰色个体（Ragland 线）**：宗旨——在体制外保留一点科学与人性。地盘：隐蔽诊所。代表：**Bradley Ragland**。战力：低，但知识是四~五阶剧情钥匙。作用：寄生虫手术、Dana 监护、样本解读。

**【隐藏剧情 · 伏笔】**
- 玩家操控的「Alex」**不是**原版人类人格的简单复活，而是病毒以尸体为模板生成的新个体——「我是怪物还是人」是主题，不是中二台词。
- 希望镇、Pariah、Carnival 项目证明 Blackwatch 的原罪早于曼哈顿；核清除不是第一次「为大局牺牲一座城」。
- Karen 的背叛与 Cross 的寄生虫是同一条人类猎杀链；Ragland 的手术把这条链拧成反杀 Greene 的刀，又意外造出 Supreme Hunter——因果闭环。
- 「匿名 Cross 线人」= 至高猎杀者伪装，是终局最大骗局；任何把 Cross 当永久队友的写法都 OOC。
- 核弹离岛后官方叙事归功军方，Zeus 被抹成恐怖分子或都市传说；黑光未绝，为系列后续留口，但本条目不强制续作剧情。
- 卡片锚点「时代广场母体战」不是孤立副本，而是 Bloodtox 战略 + 蜂巢地表化 + 军方火网三线交汇的结果。

## 阶位切入点

阶位↔原作生态位：一阶≈曼哈顿平民/恐慌市民/基础新兵；二阶≈海军陆战队精锐·Blackwatch 普通战斗员·Walker；三阶≈Blackwatch 精英小队·早期原型体·轻型猎杀威胁；四阶≈Hunter·超级士兵·中期 Alex（利爪护甲跑墙）；五阶≈Leader Hunter·Hydra·蜂巢核心·后期 Alex；六阶≈Greene 母体巨形·Supreme Hunter·终局 Alex·核清除对峙。世界顶点（终局原型体 / 母体 / 至高猎杀者 / Randall 核选项）为六阶顶，低阶切入一律「规避顶点·情报优先」。

**一阶（隔离初夜·平民求生线）**

切入身份/时点：契约者以疏散平民、外州来的志愿者、货车司机或刚下船的游客身份切入；锚定 **Penn Station 事件后 24~48 小时**，封锁令初下、官方仍称「恐怖袭击」。

初始事件：检查站突然拉响感染警报，前车一家三口被黑衣人拖走；同队列有人开始咳血，陆战队举枪。契约者必须在 3 分钟内选择：强闯铁丝网、贿赂士兵、或跟着人群撤回已出现 Walker 的地铁口。

开场白建议：「扩音器重复着‘保持冷静’，可防毒面具下的眼睛全在抖。曼哈顿的夜风里有铁锈和甜腥。你的通行证在红外灯下闪了两下就变红——前面那辆面包车的车门被撬开，有什么东西用不是关节的角度爬了出来。你还来得及装成吓傻的游客，或者现在就跑。」

关键NPC立场：**Dana Mercer** 在安全屋接听幸存者线报，对外来者防备但缺人手；一线陆战队士官只认通行证与探测器；黑市票贩子卖假腕带，收现金不救命；早期 Walker 无交涉可能。

主线钩子/支线：钩子——打探「Gentek 死人复活」传闻，为后续 Zeus 线铺路。支线：①替超市老板运一箱胰岛素过两个街口；②帮记者偷拍黑衣人拖尸换一部卫星电话；③在地铁找到失散小孩，换取安全屋坐标（勿与高层复制）。

危险度/规避：危险度低~中。规避任何「黑衣、无番号、戴红色识别」的 Blackwatch 小队；勿跟拍直升机降落点；禁止故意接触体液。

任务方向/奖励：活着离开封锁区 / 弄到真通行证；奖励=防毒面具、现金、假身份、低阶情报（「Zeus」只是电台噪点）。

**二阶（蓝区士兵·Walker 巷战线）**

切入身份/时点：契约者以征召预备役、PMC 临时工或陆战队支援技师身份切入；锚定 **Red Crown 建立、感染开始街区化** 阶段。

初始事件：小队奉命清一座公寓，三楼突然从天花板掉下两只 Walker；班长要求「全灭后点火」，而你在抽屉里发现还握着娃娃的小孩尸体——电台同时呼叫支援隔壁蜂巢边缘。

开场白建议：「头盔里全是粗喘和枪栓声。走廊喷了‘已消毒’的黄漆，可墙纸下鼓着脉络一样的东西。你的同伴笑着说今天加班费三倍，下一秒他的笑声被什么东西连头盔咬扁。探测器尖叫，你必须决定是开火、后撤，还是按黑衣顾问说的——‘留一个活体’。」

关键NPC立场：陆战队军官要战果；**Blackwatch** 顾问要样本；屋顶上的 **Alex Mercer**（远距掠过）视你们为移动生物质；平民幸存者跪求跟队。

主线钩子/支线：钩子——护送样本车去 Gentek 外围，途中遭遇第一次「士兵被活活改形状」的目击。支线：①抢修被砸的通信塔；②假传命令让某检查站放行医疗队；③从倒下的 Blackwatch 尸体搜出 Bloodtox 试管残液（不知用途）。

危险度/规避：危险度中。规避单独追入红区小巷；勿对空射击武装直升机；遇到利爪人影优先隐蔽而非挑衅。

任务方向/奖励：守住一个街垒一晚 / 上交「未破损样本」；奖励=制式枪械改装、装甲车钥匙卡、二阶战地医疗包、Blackwatch 内部代号碎片。

**三阶（精英渗透·Zeus 初猎线）**

切入身份/时点：契约者以 Blackwatch 合同猎手、Gentek 安保主管或「民间特异事件调查员」切入；锚定 **Karen Parker 仍在与 Alex「合作」、Cross 已接 Zeus 猎杀令** 前后。

初始事件：你收到一份加密简报：目标代号 Zeus，能变形，最后出现在中城。简报附赠寄生虫发射器使用说明——同页有 Karen 的签名。行动开始 10 分钟后，屋顶滴下不属于人类的影子。

开场白建议：「简报用词像在猎一只动物，可页脚的机密章比你见过的战争还密。你检查发射器，药剂在管内缓慢蠕动。对讲机里 Cross 的声音平稳得可怕：‘活捉优先。若不能，就让它疼到失去形态。’你抬眼，对面广告牌后的轮廓正在以错误的关节爬行。」

关键NPC立场：**Robert Cross** 把你当耗材尖兵；**Karen Parker** 对你笑，眼里只有自己的活命合同；**Alex Mercer** 若发现你持寄生虫会优先抹杀；**Dana Mercer** 可能误将你当救哥哥的人。

主线钩子/支线：钩子——参与一次对 Zeus 的围猎，见证寄生虫命中或失败。支线：①潜入 Gentek 档案室复制 Greene「受试体 001」页；②截获 Dana 的邮件但不交上级；③在蜂巢边缘测试自制燃烧瓶配比（本阶独有）。

危险度/规避：危险度中高。规避与 Alex 正面对波超过 30 秒；勿在 Cross 面前暴露同情心；Karen 的「旧情故事」一律当饵。

任务方向/奖励：寄生虫命中记录 / 活体组织；奖励=高级伪装套件、三阶爆能武器、部分阴谋之网文字档（非完整记忆）。

**四阶（猎杀者战场·中期质变线）**

切入身份/时点：契约者以独立超凡佣兵、被感染却未失控的「带毒者」、或叛出 Blackwatch 的前精英切入；锚定 **寄生虫已移除、Leader Hunter 活跃、Core Hive 未破** 的窗口。

初始事件：一只 **Hunter** 把坦克掀翻在你的任务街区；超级士兵小队空降，要求你当诱饵把猎杀者引进预埋炸药区——诱饵存活率写在任务单最下方小字。

开场白建议：「地面在抖，不是炮击，是什么东西用四肢轮流砸街。超级士兵的面罩反着红光，其中一个用枪口点点你：‘跑直线，别死太早。’你听见自己的心跳和更远处的尖叫叠在一起，利爪刮过巴士车厢的声音像在给曼哈顿拉弦。」

关键NPC立场：**Bradley Ragland** 可用手术与情报换你带回的猎杀者组织；**Alex Mercer** 视同阶争食者，可短暂同盟清 Hunter；Blackwatch 要你的血样；**Dana** 失踪前的安全屋留下半盘录音。

主线钩子/支线：钩子——协助或阻挠 Alex 追 Leader Hunter / 救 Dana 的时间线。支线：①从超级士兵尸体提取 D-Code 残样；②炸毁一座水源塔但保留地下通道；③给 Ragland 送冷冻箱，换「如何暂时抑制感染视界眩晕」的土法。

危险度/规避：危险度高。规避 Core Hive 深层；勿试图单独吞噬（你不是原型体规则受益者就不要硬学）；超级士兵的集火比 Hunter 更突然。

任务方向/奖励：猎杀者心脏级样本 / Dana 线索；奖励=四阶生物质稳定剂（叙事向）、重型武器权限、Ragland 联盟标记。

**五阶（蜂巢与 Bloodtox·城市级灾难线）**

切入身份/时点：契约者以 Bloodtox 工程监理、红冠情报分析官、或已被 Alex 半承认的「有用人类/半感染体」切入；锚定 **Core Hive 战后、Bloodtox 大范围泵送、母体尚未在时代广场完全显化**。

初始事件：你负责的三号泵车在紫区抛锚，地下管线回灌出红色雾气与尖叫；对讲机里 Taggart 在骂、Randall 在下令「泵坏了就人肉顶」，而远方蜂巢脉冲让你鼻腔流血。

开场白建议：「Bloodtox 把日落染成假的。红色的雾贴着沥青爬，Walker 在雾里像被盐撒过的蛞蝓。你跪着修阀门，手套腐蚀出孔，指节却还是人类的颜色——目前为止。头顶武装直升机投下的不只是探照灯，还有一句冷静的：‘母体在动。’」

关键NPC立场：**Peter Randall** 视你为可牺牲零件；**Ian Taggart** 已在盘算跑路，可被你勒索；**Alex Mercer** 需要泵网工作以逼 Greene；**Elizabeth Greene** 的意识开始点名「不听话的节点」；**Ragland** 警告你雾中待太久会不可逆。

主线钩子/支线：钩子——确保或破坏关键泵站，直接摆动时代广场决战的开启条件。支线：①保护一列难民火车穿过紫区（本阶独有人道主义压强）；②窃取 Firebreak 预案目录页；③与 Alex 交易：你修泵，他不去动你标记的医院。

危险度/规避：危险度极高。规避 Leader Hunter 巢心与 Hydra 出水口；Bloodtox 对感染体质契约者持续伤害；禁止在母体地表化后留在开放广场中心。

任务方向/奖励：泵网控制权 / 预案碎片；奖励=五阶防护服（Bloodtox 耐性）、区域指挥临时权限、与 Zeus 的有时限互不吞噬协议。

**六阶（时代广场母体战与里根号终局线）**

切入身份/时点：契约者以准顶点搅局者、黑光高度适应体、或同时被军方与原型体点名的「第三方灾厄」切入；锚定 **时代广场 Greene 母体战当日** 至 **USS Ronald Reagan 核弹离岛** 之间。

初始事件：时代广场巨影遮住广告屏，Hunter 潮与坦克阵列对射；你刚落地，就收到两个频道——Randall 要你把 Zeus 引进核弹瞄准校射区，Alex 要你拖住甲板上的「Cross」。两个人都没把你当会活过今晚的人。

开场白建议：「城市在流血，血是红雾和航空煤油。Greene 的触手把一辆巴士折成金属花，Alex 的刃臂在另一侧切开直升机旋翼。你的通讯器里叠着将军的命令和怪物的呼吸。核弹的倒计时不是数字，是航母甲板上升起的那一道冷白光——你还有一次选择：帮谁按下去，或把所有人的剧本撕开。」

关键NPC立场：**Elizabeth Greene**（母体）无谈判，只有吞噬与被吞噬；**Alex Mercer** 可结「条件性同盟」至多到核弹离岛；伪装 **Robert Cross** 的 **Supreme Hunter** 会用完美人类语气骗你；**Peter Randall** 死前仍要灭岛；**Dana Mercer** 在 Ragland 处昏迷，是 Alex 的软肋也是你的人质杠杆（用则结永久仇）。

主线钩子/支线：钩子——介入母体战收尾或里根号终局，决定核弹是否偏离曼哈顿（原作默认离岛引爆，契约者可付代价微调时点，不可无代价取消核物理）。支线：①在吞噬 Greene 的窗口偷一段蜂巢指挥残响；②揭穿 Cross 伪装并活着把证据送给仍是人类的军官；③保住 Ragland 诊所不被终局余波夷平。

危险度/规避：危险度贴近顶点。本阶顶点为六阶顶的 Greene 母体 / Supreme Hunter / 终局 Alex / 核选项——**情报优先、条件性胜利**，靠 Bloodtox、多方火力、伪装与登舰权限取胜，严禁把顶点写成莫名掉档或无代价可欺。规避与母体硬刚超过一个呼吸；规避在核弹爆心停留。

任务方向/奖励：曼哈顿免于当场核平 / 至高猎杀者死亡确认；奖励=六阶黑光稳定样本、终局情报包（希望镇完整档案摘要）、与存活 Dana/Ragland 的战后联系人、乐园结算向的「阻止灭岛」高权重业绩。注意：吞噬能力规则仍属原作顶点生态，契约者不能因剧情奖励直接变成第二头无代价 Zeus。
