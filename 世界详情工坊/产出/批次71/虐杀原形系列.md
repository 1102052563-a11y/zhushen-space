# 虐杀原形系列

## 剧情

**【作品来源】**
《虐杀原形》（*Prototype*，中文亦作「虐杀原形」）系列由加拿大 Radical Entertainment 开发、Activision 发行的第三人称开放世界动作游戏。正史两作：*Prototype*（2009，PC / PS3 / Xbox 360）与 *Prototype 2*（2012）。另有官方漫画、小说与 Web of Intrigue / Blacknet 碎片补完前史与组织黑料。本档案是**系列总览条目**：写 Blacklight 共同规则、跨作年表、1→2 因果衔接、七阶「城市级病毒神性」终局舞台；曼哈顿区划关卡、单 Boss 连招、技能树点数等细目由 `虐杀原形1` / `虐杀原形2` 各条承担，本条不整段倒灌单作流水账。文风：冷硬阴谋惊悚 + 生物恐怖 + 反英雄复仇；基调是「人是否还是人」与「谁才是真正的病原体」。卡片锚点（世界库提要）：红光病毒完全体——莫瑟或海勒进化为终极生物，其生物质足以覆盖整个曼哈顿，并向外扩散，将地球生态改造为病毒巢穴。

**【世界定位】**
虐杀原形宇宙是近未来美国（以曼哈顿 / New York Zero 为绝对主舞台）被**人造病毒**改写的生物灾难世界。力量不来自修仙气运，而来自 **Redlight / Blacklight** 两条病毒谱系、军方黑卫（Blackwatch）的封锁与火线、基因科技公司 Gentek 的人体实验。系列不是「英雄救世」，而是同一座岛上两次疫情：第一次由人类科学家 **亚历山大·J·莫瑟（Alex Mercer）** 在宾州车站砸开样本引发，病毒以他的尸体为模板重建出**第一原型体（Prototype / Zeus）**；第二次由已觉醒的莫瑟主动再放病毒，把曼哈顿改造成分区隔离的 **NYZ（New York Zero）**，并制造可伪装渗透的 **Evolved（进化者）**，最终被他亲手感染的 **詹姆斯·海勒（James Heller）** 吞噬收束。契约者以**七阶**切入时，世界已处于「城市可被生物质覆盖、病毒可向外扩张改写生态」的可感舞台——不是巷战清怪日常，而是国级灾难、核选项边缘、原型体神性对决层。一句话：以 Blacklight 为轴的残酷生物神性舞台，契约者是变数，不是免死金牌。

**【世界观 · 力量体系】**
力量本源是**病毒改写生物学**，不是灵气或血统贵族。核心病原体谱系：

1. **Redlight（红光）**  
   冷战期人造生物武器前身。1960 年代在爱达荷州小镇 **Hope** 做人体试验；多数感染者死亡或畸变，**伊丽莎白·格林（Elizabeth Greene）** 成为罕见「母体」宿主，能以蜂巢意识指挥感染体，并诞下特殊子体（与 **Pariah** 相关的黑料碎片）。Hope 失控后由 Blackwatch 以焦土手段抹除（**Operation Altruistic / Firebreak 式**全镇清除），格林被秘密关押研究数十年。

2. **Blacklight（黑光 / 后世亦称 Mercer 病毒）**  
   Gentek 以格林血液样本等素材合成的进阶嵌合反转录病毒（公开资料常见编号 DX-1118 系列）。机制粗写：插入宿主遗传信息、激活大量「垃圾 DNA」、重写细胞用途。约 99.99% 宿主死于器官衰竭 / 细胞过载；极少数在特定条件下获得：超强再生、感官强化、生物质塑形（爪 / 刃 / 鞭 / 锤 / 甲盾）、攀爬跳跃与高速机动、**吞噬（Consume）** 以获取生物质、记忆、技能与外貌伪装。传播靠体液接触、撕咬、环境蜂巢 / 水塔污染等；大水体是天然屏障，故两次大爆发被地理锁在曼哈顿岛。

3. **原型体（Prototypes）规则**  
   - **莫瑟型**：人类亚历山大·J·莫瑟在宾州车站被击毙后，病毒以其生物质**重建**出的智能体——意识主体是病毒，却继承 / 拼贴出「莫瑟」身份与记忆。他不是「复活的人」，是**以人的脸行走的 Blacklight 神性节点**。  
   - **海勒型**：詹姆斯·海勒被莫瑟感染后，因**抗性 DNA** 保留人类意识主导权，形成混合体；莫瑟难以直接吞噬他，反被他反噬。  
   - **Evolved**：莫瑟主动「点化」的次级智能感染体，可伪装成人、渗透 Gentek / Blackwatch，战力弱于完整原型体，但组织破坏力极高。  
   - **感染体阶梯（叙事用）**：Walker 级行尸 → Hunter / Brawler 等中大型突变体 → 母体 / 蜂巢核心 → 原型体 / 进化者 → 完全体（城市级生物质展开）。

4. **吞噬与伪装**  
   吞噬目标 = 治疗 + 情报（Web of Intrigue / Blacknet 式记忆碎片）+ 外形复制。伪装可骗过视觉与多数门禁，但高级扫描、近亲识别、病毒嗅探与「你不像他」的行为破绽仍会穿帮。吞噬关键人物会永久改写信息战格局。

5. **军方增强与「白光」**  
   Blackwatch / Gentek 并行研发：D-Code / **Project Orion** 超级士兵（用病毒片段强化士兵）、**Whitelight（白光）**——名义上的治愈 / 接种剂。在第二作中 Whitelight 被莫瑟血样污染，投放后会把人群推向感染或进化者化，是「以解药之名做全球播种」的终局钥匙。

6. **死亡与复活**  
   凡人一死即死。感染体可被彻底摧毁或再被吞噬。原型体再生近乎犯规，但核心生物质被同级原型体**完整吞噬**即人格消亡（莫瑟终局即此）。核爆 / 战略火线可清空地表，却未必根除全部样本——世界顶点不可无限读档。

7. **地理锁与扩张逻辑**  
   岛与水域暂时限制扩散；若原型体完全体把生物质铺满曼哈顿并建立稳定对外节点（船舶、航空、被污染物资、被点化的进化者离岛），即可把地球生态逐步改造成病毒巢穴——这是卡片锚点的物理含义。

战力阶梯（按**实际破坏力 / 地位**写，宁低勿高；本世界清单**仅覆盖七阶**）：
1. **凡人武装 / 普通海军陆战队 / 平民**：枪械与掩体，对低阶感染体有效，对原型体是炮灰。约一~三阶感——**本系列总览不写切入**，细节见各单作。
2. **Blackwatch 精锐 / 装甲单位 / 中型感染体（Hunter 等）**：可毁街区建筑、清小型蜂巢、制造城市恐慌。约四~五阶感——单作日常战场。
3. **超级士兵（Orion / D-Code）/ 大型突变体 / 地区级蜂巢母体**：可屠城一角、对抗整支特遣队、改写局部战场。约六阶感——单作中后期 Boss 层。
4. **完整原型体 / 城市级完全体 / 被污染的全球投放级病毒事件（本条主舞台）**：一怒可覆盖曼哈顿级都会、以生物质重塑街区为巢、吞噬同级对手后继承其力量与记忆、威胁把整座岛变成可向外输出的病毒神座。乐园观感**七阶**（人形战略武器·世界级）。包括但不限于：吞噬格林后的莫瑟、再放病毒并统御进化者的莫瑟、吞噬莫瑟后的海勒、以及「生物质足以覆盖曼哈顿并向外扩散」的完全体形态。
5. **若叙事触及「整颗行星生态被病毒规则改写」**：在本世界映射中仍收束在**七阶巅峰 / 条件性终局**，不单开八九阶切入。契约者对「正在铺满都市的完全体」一律情报优先，禁止写成可随便练功超越的经验包。

乐园阶位映射：凡人~普通军警≈一~三阶（单作层）；街区~城级感染灾害与精锐特遣≈四~六阶（单作层）；完整原型体 / 城市级完全体 / 可外扩的病毒巢穴威胁≈**七阶**；曼哈顿被生物质覆盖、核选项边缘、Whitelight 全球投放危机等表现均按七阶世界级锚定，宁低勿高；**超阶巅峰**若叙事触及「整颗行星被改写成单一病毒生态」——仅作条件性胜利 / 情报优先，严禁硬刚本体与「被削弱就战力归零」式降智。本世界清单**只开七阶**，切入点严禁写一~六阶。

**【地理 · 舞台】**
舞台几乎全部落在**曼哈顿及其周边隔离体系**，从实验室尸检台扩到「整岛即巢穴」。

- **曼哈顿 / 第一作疫区**：帝国大厦等地标可识别的开放都市；军事蓝区、感染红区、争夺紫区在地图上拉锯。宾州车站（Penn Station）是两次疫情的**零号引爆点**。
- **Gentek 设施与停尸房**：莫瑟「醒来」之处；表面是生物科技公司，实际是 Blackwatch 外包的病毒工厂与人体实验场。
- **Blackwatch 前线 / 航母与核选项节点**：含第一作终局相关的 **USS Ronald Reagan** 航线与 **Operation Firebreak** 式核清除预案——军方宁可炸掉一座城，也不肯承认自己养出了神。
- **Hope, Idaho（前史舞台）**：1960 年代 Redlight 试验镇，已被抹除；只以记忆、档案与格林的存在回响。七阶正文里它是「原型体神性从哪来」的考古层，不是可随便传送的旅游点。
- **NYZ · New York Zero（第二作主舞台）**：第二次疫情后整岛重命名与分区：
  - **Green Zone**：相对「正常」的高压管制区（常见设定对应史泰登岛一侧意象），表面有生活，实则监控与渗透战。
  - **Yellow Zone**：难民营、检查站、人满为患的隔离城，Blackwatch 把平民当耗材与实验素材。
  - **Red Zone**：曼哈顿核心废墟，感染体与生物质横行，莫瑟的王座与终局死斗多在此发生。
  - **Dead Zone 等禁入残区**：叙事上进一步锁死「岛即牢笼」。
- **蜂巢 / 水塔 / 生物质节点**：病毒的基础设施；摧毁或夺取它们等于改写区划控制权。
- **银行金库 / 高楼顶（终局棋盘）**：第二作终局莫瑟将 **黛娜·莫瑟（Dana Mercer）** 与 **阿玛雅（Amaya / Maya Heller）** 锁入金库，自己在红区楼顶与海勒展开原型体对决——卡片锚点的「覆盖曼哈顿的生物质」在此具象为街道级触手、刃与盾的神性互噬。

AI 写场景先定契约者处于：**管制绿区渗透层 / 黄区人间炼狱层 / 红区病毒神座层 / 前史 Hope 记忆层**哪一层。七阶切入默认红区与「完全体展开」可感，前两层已是余波与后勤战场。本地凡人若听闻「要去单挑完全体莫瑟 / 海勒」，正确反应是当你疯了。

**【世界剧情线】**

① **前史 · Hope 与 Redlight（全系列母题）**  
1962 年前后，**Blackwatch**（美国军方黑卫，生物战与生物威胁特遣）成型。1960 年代中后期，Redlight 在爱达荷 **Hope** 镇试验：人口大规模感染、畸变、失控。孕妇 **伊丽莎白·格林** 成为异常宿主——病毒不以杀死她为终点，而以她为**母体节点**指挥感染潮。Blackwatch 以焦土行动抹镇（Altruistic / Firebreak 系清除），把格林与相关样本秘密回收。  
因果：没有 Hope，就没有后来的 Blacklight 合成材料，也没有「军方宁可屠城也要守住红线」的组织人格。卡片锚点里的「红光」谱系从这里长出黑光。

② **Gentek 与 Blacklight 的诞生**  
**雷蒙德·麦克马伦（Raymond McMullen）** 创立 Gentek，表面科研，实际承接 Blackwatch 外包。科学家 **亚历山大·J·莫瑟** 参与 / 主持 Blacklight 相关研发；团队在伦理崩坏中推进「可武器化、可控制、可激活垃圾 DNA」的嵌合病毒。同事与相关人物包括后来退场或反水的 **布拉德利·拉格兰（Bradley Ragland）**、**凯伦·帕克（Karen Parker）** 等。格林被长期关押当作活体数据库。  
因果：Blacklight 不是天降天灾，是**国家—公司—实验室**共犯的产品。

③ **2008 · 宾州车站零号事件（第一作爆发）**  
莫瑟察觉 Blackwatch 将清洗项目科学家，偷走一管 Blacklight 样本作保险。在 **Penn Station** 被逼入绝境时砸碎样本并被击毙。病毒以其尸体为模板**重建**出第一原型体——醒来时失忆，躺在 Gentek 停尸房，以为自己仍是「人莫瑟」。他吞噬逃生，发现可变装、可变武器、可再生，并与妹妹 **黛娜·莫瑟** 建立脆弱同盟。  
阶段名场面：尸检台逃亡；第一次公开吞噬；在曼哈顿屋顶与巷战中同时对抗 **美军 / Blackwatch** 与感染潮。因果：原型体诞生；岛上疫情从地下实验室变成全城事件。

④ **第一作主线 · 格林、网络阴谋与核选项**  
失忆原型体一边查「我是谁」，一边被多方当猎物：**罗伯特·克罗斯（Robert Cross / The Specialist）** 等 Blackwatch 猎手、Gentek 高层、感染母体格林。他释放 / 对峙格林，导致感染潮升级；中寄生陷阱后由拉格兰协助处理，反手把寄生逻辑变成更危险的 **Supreme Hunter（至高猎手）** 级存在。Web of Intrigue 记忆碎片拼出真相：Hope、Redlight、Blacklight、他自己就是零号释放者。  
终局链：吞噬 **伊丽莎白·格林**；渗透至 **USS Ronald Reagan** 一线，阻止 **彼得·兰德尔将军（General Peter Randall）** 一系把核武当 Firebreak 砸向曼哈顿；击败伪装渗透的至高猎手；将核弹转移并在海上承受爆炸后再生。公开叙事被军方改写成「反恐胜利」，岛上实际死伤以百万计，原型体仍在城中行走——更强，也更不像人。  
阶段结局：第一波疫情被「表面控制」；莫瑟完成从失忆怪物到自觉 Blacklight 节点的质变。因果：他已证明自己可吞噬母体、抗核再生、操弄整座城的信息与生物质——**七阶神性的预告片**。

⑤ **1→2 衔接 · 信仰崩坏与第二次放毒**  
约一年后。莫瑟在第一作后的观察中得出结论：人类文明不可救药，Blackwatch / Gentek 只会重复 Hope。他不再扮演「查真相的反英雄」，而选择**主动再砸开 Blacklight**——再次以 Penn Station 为象征性零点，把曼哈顿推进第二次大爆发。岛被封锁并重划为 **NYZ** 绿 / 黄 / 红区；公众称病毒为「莫瑟病毒」。  
他同时「点化」一批 **Evolved**，安插进 Gentek 与 Blackwatch（含财务与科研线如 **萨布丽娜·加洛韦（Sabrina Galloway）**、科研线相关进化者等），准备把名义上的解药 **Whitelight** 污染成全球播种剂。目标不是简单屠城，而是把人类推向「一心一体」的病毒超个体——在他看来是第二次进化机会。  
因果：系列从「灾难求生」跃迁为「病毒先知的神国工程」。卡片锚点的「向外扩散、改造地球生态」在此从可能变成计划。

⑥ **第二作开局 · 海勒被点名感染**  
海军陆战队中士 **詹姆斯·海勒** 海外归来，发现妻子 **科莱特（Colette）** 等在疫情中丧生（女儿 **阿玛雅** 起初也被认为死亡）。他自愿加入红区送死巡逻，只为亲手宰莫瑟。莫瑟伏击其小队，欣赏其顽强，**主动感染**他并丢给 Blackwatch / Gentek 处理。海勒在焚化与实验台上因抗性 DNA 活下来，逃出实验室，获得原型体能力却**保住自我**。  
关键人物：**路易斯·格拉神父（Father Luis Guerra）** 提供情报与人性锚；上校 **道格拉斯·鲁克斯（Colonel Douglas Rooks）** 代表 Blackwatch 冷酷实务；**安东·柯尼希（Dr. Anton Koenig）** 等 Gentek 线人物（部分实为进化者）构成渗透网。  
阶段结局：第二原型体诞生，且天生克制「莫瑟式吞噬」。因果：莫瑟为自己选了掘墓人。

⑦ **第二作中盘 · 假同盟、Blacknet 与白光阴谋**  
海勒起初被莫瑟引导去打 Blackwatch / Gentek——证据显示军方与公司才是病毒史元凶。他黑入 **Blacknet**、吞噬目标取记忆、破坏 **Project Orion** 超级士兵线、在三区间穿梭执行破坏与营救。与莫瑟的「同盟」在发现**第二次 Penn Station 放毒录像**与 **Whitelight 被污染**后彻底破裂。格拉神父之死把海勒从「报仇工具」推成「必须截断神国计划的人」。  
他发现阿玛雅仍存活：先被 Blackwatch 掌握，后卷入进化者与莫瑟的棋局；黛娜·莫瑟从「哥哥的妹妹」变成可与海勒短暂协作的情报与情感支点。加洛韦先求生献出进化者名单，后因不被接纳而反水绑架阿玛雅献祭莫瑟。  
阶段结局：Whitelight 大规模投放被搅局；进化者网被海勒逐个撕咬；莫瑟与海勒的私人战争上升为「人类是否被改写成病毒神民」的路线之战。

⑧ **第二作终局 · 楼顶神杀与城市级收束（七阶主舞台）**  
红区废墟高楼。莫瑟将黛娜与阿玛雅锁入银行金库，自己吞并残余进化者（含加洛韦）补完生物质，以近乎**完全体**的姿态迎战海勒——触手、利刃、护盾、生化爆发全开，波及整片街区，生物质有覆盖都会之势。海勒凭借抗性与意志反复撕碎其肢体，最终**吞噬莫瑟**。莫瑟临终 entropic 冷笑：「Welcome to the top of the food chain.」  
海勒继承其力量与关键记忆，以庞大触手网清除区域内大部分残余感染体，实质终结第二波疫情的主宰结构。他救出阿玛雅与黛娜；三人立于俯瞰纽约的屋顶，黛娜问「现在怎么办？」——世界没有「病毒永绝」，只是暂时只剩**一个**站在食物链顶端的原型体。  
阶段结局：莫瑟神国流产；NYZ 感染潮被大幅压制；海勒成为系列收束时的城市级病毒神性持有者。因果：卡片锚点「莫瑟或海勒进化为终极生物，生物质足以覆盖曼哈顿并外扩」在终局同时被**演示**（莫瑟侧）与**继承/选择**（海勒侧）——七阶舞台的胜负手是「谁掌握完全体权柄」，不是谁枪法更准。

⑨ **跨作主线总轴（写给 AI 的串联句）**  
- **轴 A · 病毒谱系**：Hope Redlight → 格林母体 → Gentek Blacklight → 原型体 → Evolved / Whitelight。  
- **轴 B · 组织原罪**：Blackwatch 的红线逻辑（屠镇、屠城、核选项）与 Gentek 的资本—实验室共谋。  
- **轴 C · 身份哲学**：莫瑟＝病毒穿人皮（失忆→觉醒→神国）；海勒＝人驾驭病毒（复仇→截断神国→食物链顶端的父亲）。  
- **轴 D · 同一座岛两次末日**：Penn Station 两次引爆；Firebreak 阴影始终悬在上空；终局从「阻止核弹」升级为「阻止全球白光播种」。  
单作条目写每场任务怎么打；本条写**为何同一病毒必须把曼哈顿推向可外扩的巢穴**。

⑩ **微观日常与可介入间隙**  
大灾之间：黄区难民营抢水与通行证、绿区超市货架与宵禁广播、Blackwatch 检查站随机拉人进笼子做实验、黑市倒卖「未污染」抗生素与假通行证、神父据点收留孤儿、记者偷拍红区却被灭口。七阶切入时这些日常已被完全体气压压成「废墟经济学」，但仍可作支线呼吸口。

⑪ **蝴蝶效应边界**  
契约者可改局部：提前疏散某黄区难民营、炸掉一座未启动的 Whitelight 仓储、救下格拉神父据点的一批平民、给海勒多一条 Orion 士兵的巡逻情报、在红区切断一条生物质管道延缓完全体铺展。不可无代价改写：Hope 被抹除的历史事实、宾州车站两次放毒的核心因果、莫瑟作为病毒重建体的本质、海勒抗性 DNA 使其难以被莫瑟吞噬、终局「海勒吞噬莫瑟并成为顶端」的收束、完全体级破坏力逻辑。改了局部，原作会以「白光提前投放 / 进化者更多 / 核选项被更早授权 / 另一个完全体节点冒头」回流。

**【势力图谱】**

- **Blackwatch（黑卫）**：生物战黑特遣，宗旨是「红线之内一切合法」——屠镇、屠城、人体实验、核清除。代表：兰德尔、克罗斯、鲁克斯。七阶主敌也是主雇主；与他们合作默认你是耗材。
- **Gentek**：表面生物科技公司，实际 Blackwatch 的民用马甲与实验室。代表：麦克马伦、各项目科学家；第二作被 Evolved 深度渗透。
- **感染生态 / 蜂巢势力**：格林母体、Hunter 群、红区生物质结构——无外交，只扩张。
- **原型体节点**：莫瑟（1→2 反派化）、海勒（2 主角）；同时最多稳定存在的「神」级个体有限，吞噬即继承。
- **Evolved 网络**：莫瑟的地下教会兼间谍网，跨绿黄红三区。
- **美军常规部队 / 市政 / 难民营自治**：被当炮灰与背景板；七阶正文里他们是要保护或会拖后腿的凡人海。
- **地下抵抗与宗教庇护（格拉一线）**：无战略武力，有情报与道德合法性。

敌友逻辑：病毒身份暴露 = 全球通缉；Blackwatch 可对「可能感染者」就地处决；莫瑟的同盟默认是「用完即吞噬」；海勒的同盟默认是「家人优先」。暴露「我能稳定制造完全体 / 离岛传播节点」等于被所有方同时点名。

**【贵重物品】**

- **Blacklight 原始样本管**：宾州车站级钥匙。谁想要：所有派系。抢到的后果：被当成新的零号。七阶可作任务核心物，**禁止**开局白给「可全球投放」完整权柄。
- **Redlight / 格林相关活体样本与档案**：连接 Hope 的祖源。谁想要：Gentek 遗毒、想复制母体的疯子、想根治的医师。
- **Web of Intrigue / Blacknet 数据核**：吞噬与黑入所得的记忆—阴谋网。比枪更值钱的情报硬通货。
- **Whitelight 仓储与配方**：名义解药、实际可被污染成全球播种剂。终局级争夺物。契约者最多销毁、调包、揭露，不建议「研究一下自己用」。
- **Project Orion / D-Code 血清与超级士兵样本**：军方版半成品原型体。可临时增强，副作用与被追踪率极高。
- **寄生体制剂 / 至高猎手相关组织样本**：双刃剑，历史证明使用者会被反噬出更强敌。
- **区划通行证、扫描屏蔽件、假身份生物特征**：NYZ 生存硬通货；七阶渗透战的基本门票。
- **完全体生物质节点（红区王座、巨型触手根、蜂巢核）**：不是「捡了就装包」的道具，而是可被争夺、引爆、夺取控制权的**地形级神器**。对齐卡片锚点。

每件写清使用边界：七阶契约者最多短期借用、破坏、护送或阻止共鸣；禁止无代价永久持有「可稳定把地球改成病毒巢穴」的完整权柄。

**【隐藏剧情 · 伏笔】**

- 莫瑟从第一作「阻止核爆救岛」到第二作「再放毒改造人类」，不是人设崩坏，而是**病毒身份彻底自觉**后的逻辑终点。  
- 海勒能赢，不只靠热血，而靠**抗性 DNA + 莫瑟无法完成吞噬**的生物学锁——系列在规则层写了「反莫瑟」的钥匙。  
- Hope 的 Firebreak 与曼哈顿核选项是同一组织人格的两次表达；「人类比病毒先选择屠城」是贯穿伏笔。  
- Evolved 与 Whitelight 说明第二作战争已从「清怪」升级为「谁定义进化」。  
- Pariah 等前史碎片（格林子体相关）在官方资料中点到即止，本档案不编造未证实终局；写手勿把社区脑补当原作。  
- 系列无官方第三作收束；终局状态是：**莫瑟死、海勒立、NYZ 暂稳、病毒未绝、食物链顶端换人**。  
- 跨媒体漫画可补人物小传，但不改变两作主线胜负手。

**【大事记时间线】**
- 1962 前后 → Blackwatch 成型，生物战黑线启动。  
- 1960 年代中后期 → Hope, Idaho 的 Redlight 试验与失控；格林成为母体；镇子被焦土清除。  
- 1970–2000s → Gentek 成立并深化病毒项目；格林被长期关押研究；Blacklight 合成推进。  
- 2008（第一作）→ 莫瑟在 Penn Station 释放 Blacklight 并死亡；原型体重生；曼哈顿爆发；吞噬格林；阻止航母线核爆；表面疫情受控。  
- 约 2009（第二作）→ 莫瑟再放毒；NYZ 三区隔离；进化者渗透；海勒被感染并崛起；Whitelight 阴谋；Orion 超级士兵；楼顶决战；海勒吞噬莫瑟并大幅清除感染。  
- 终局后 → 黛娜问「现在怎么办？」；海勒成为现存主导原型体；外扩神国被打断，病毒生态仍可能死灰复燃。

**【叙事基调 · 雷区】**
基调：混凝土、血雾、警报与直升机螺旋桨下的生物恐怖——可以写黑色幽默式的吞噬吐槽，但**身份解体与屠城伦理**必须沉重。战斗描写重生物质质感（骨刃破甲、肌肉再生、触手抽楼）与伪装渗透，不要写成超级英雄闪光或修仙斗法。  
忌：把莫瑟写成无动机的「纯坏人工具」而删掉 Hope—Blackwatch 原罪；把海勒写成轻松原谅一切的好爸爸；让完全体被常规坦克编队轻松击毙；把 Whitelight 当普通回血药；在本条展开一~六阶巷战教程（那是单作的活）；用跨世界通用空话填空。  
与单作分工：`虐杀原形1` 写格林之战、克罗斯、核选项与失忆莫瑟；`虐杀原形2` 写三区、进化者、白光与楼顶对决细部；**本条写年表、Blacklight 共同规则、1→2 衔接与七阶城市级病毒神性舞台**。  
最早可切入锚点（本世界仅七阶）：NYZ 红区已现完全体征兆，或莫瑟 / 海勒之一即将以城市级生物质改写曼哈顿——对齐卡片「红光病毒完全体……覆盖整个曼哈顿……改造为病毒巢穴」。

## 阶位切入点

> 阶位↔原作强度：凡人与普通军警≈一~三阶（单作层，本世界不切入）；街区~城级感染灾害与精锐特遣 / 中型突变体≈四~六阶（单作层）；完整原型体 / 城市级完全体 / 可外扩病毒巢穴威胁≈**七阶**。世界顶点表现（吞噬母体后的莫瑟、再放毒并统御进化者的莫瑟、吞噬莫瑟后的海勒、生物质覆盖曼哈顿的完全体）按七阶世界级锚定；若叙事触及「整颗行星生态被改写成病毒巢穴」——**超阶巅峰：存在·情报优先 / 条件性胜利**，严禁硬刚本体与「战力无故归零」式降智。本世界清单**仅覆盖七阶**，严禁写一~六阶切入段。

**七阶（城市级病毒神性 · 曼哈顿完全体舞台）**

切入身份/时点：契约者以「NYZ 灾难编外特遣 / Blackwatch 黑科顾问（假身份）/ Gentek 叛逃项目员 / 红区观测与清除承包商 / 地下抵抗的武装信使」等与世界级生物灾匹配的身份切入，锚定下列窗口之一：**①** 第二作红区，莫瑟已再放毒并开始铺生物质、进化者渗透三区，Whitelight 投放倒计时前后；**②** 莫瑟与海勒楼顶决战前后，完全体生物质已波及整片街区；**③** 历史回响型高压副本——第一作核选项边缘与吞噬格林后的莫瑟同时可感（作为七阶「神性预告」窗，仍按世界级结算）。勿切入「只清 Walker 的黄区日常巡逻」。

初始事件：天空被血色雾与直升机分割，远处楼群表面「长」出不该属于建筑的肌肉与甲壳；广播在「保持冷静」与「就地处决感染嫌疑」之间切换。你在坍塌的高架或金库街区边缘撞见——或是披着莫瑟外皮、口中说着「进化」的**完全体级存在**，或是骂着脏话却护着撤离线的**海勒**，或是举着扫描枪、已授权火线清场的 **Blackwatch** 小队。地面生物质像潮水一样试探你的靴底。你必须立刻抉择：协助疏散残存平民并换取格拉线 / 黛娜线情报、加入截断 Whitelight 仓储的侧翼、或向原型体伪降骗取短暂存活——伪降的死亡率高于正面当炮灰。完全体即便「看起来还在说话」，破坏力仍按城市级本体结算，禁止写成残血可秒。

开场白建议：「曼哈顿的天际线先是缺了一块，然后缺的地方被某种温热的、会跳动的东西填上了。你的通讯器里同时塞进三种声音：军方要你立刻开火，难民营要你带水，还有一个听起来很像人类的声线在说『别怕，这只是蜕皮』。水泥路面从裂缝里伸出第一条细须，轻轻碰了碰你的鞋尖——像在确认你是食物、同类，还是障碍。」

关键NPC立场：**亚历山大·莫瑟**——视你为生物质、信徒或障碍，谈「进化」时最危险；**詹姆斯·海勒**——先骂后用，护女心切，帮他截胡进化者/白光极易结盟，拿阿玛雅要挟则必死；**黛娜·莫瑟**——情报与人质双价，可信但极度脆弱；**路易斯·格拉神父**（若窗口仍在其生前）——给路线与人性锚，要你别变成下一只怪物；**道格拉斯·鲁克斯**——可签血合同，默认把你当清场耗材；**萨布丽娜·加洛韦**——先卖情报再卖你，进化者式笑面；**伊丽莎白·格林**（回响窗）——蜂巢母性，不接受谈判只接受加入；**至高猎手 / Orion 超级士兵**——战场竞争者，不讲同盟。

主线钩子/支线：主线＝阻止 Whitelight 全球播种 / 延缓或引导完全体生物质铺展 / 在莫瑟—海勒神杀中选择侧翼（对齐卡片锚点「覆盖曼哈顿并外扩的病毒巢穴」）；支线＝炸毁红区一条生物质主根以换取六小时疏散窗、从黄区囚笼劫走 Gentek 记录了 Hope 真相的硬盘、护送阿玛雅转移时反杀加洛韦绑匪线、黑入 Blacknet 取出兰德尔时代 Firebreak 授权链公之于「还听得见的人」。

危险度/规避：贴近顶点——完全体莫瑟、吞噬后的海勒、母体回响、核 Firebreak 授权皆**世界级**；须规避：单独挑战楼顶神杀双方、抢饮 Whitelight「研究一下」、在完全体铺展时去红区「捡材料」、向莫瑟炫耀「我能帮你传播得更快」却无反制。历史中被核弹边缘与吞噬结算过的存在再临时仍按完整世界级结算。**超阶巅峰**若出现「离岛传播已不可逆、行星生态开始改写」：存在·情报优先 / 条件性胜利，严禁「战力无故归零」式降智。

任务方向/奖励：活过红区完全体展开日或等价的白光投放夜、拿到可验证的「截断外扩节点」贡献、与海勒线或地下抵抗建立可续约关系；奖励取向＝同阶病毒抑制剂 / 扫描屏蔽义体、Orion 残次增强的一次性注射（副作用写明）、Blacknet 高级密钥、NYZ 重建期的通行证与情报席位、可寄生武器化的**有限**生物质样本（非完全体权柄）。**禁止**开局发放完整 Blacklight 祖源管、完整 Whitelight 全球配方、稳定离岛传播节点或「一键覆盖曼哈顿」的完全体钥匙。

**七阶·支线事件池（终局舞台专用，勿与低阶巷战套话重复）**
1. 生物质涨潮疏散：红区边缘难民营生命读数随触手推进下降，需在限定时间内切断三条主根并护送幸存者到黄区掩体；失败则海勒态度从「合作」转「你不配靠近我女儿」。  
2. 白光假解药拍卖：绿区黑市用军用冷柜拍卖「未污染 Whitelight」，买家包括末日教派与进化者；你可抢、可炸、可做局让双方互撕——真货流出即加速神国。  
3. Orion 二期失控：以海勒 DNA 片段催生的超级士兵认友为敌，需在鲁克斯下令火线清城前将其吞噬或瘫痪；成功换取一次性生物爆发模块，失败多一个城级暴走体。  
4. 进化者微笑谈判：加洛韦式对象愿以「三名进化者坐标」换离岛；坐标里至少一处是伏击，拒绝则阿玛雅线风险上升。  
5. Hope 档案回声：在 Gentek 废墟读到 Firebreak 屠镇影像，不解开「人类先选屠城」的认知锁就会在幻觉里对平民扣扳机；奖励是莫瑟动机预测，不是杀他的枪。
