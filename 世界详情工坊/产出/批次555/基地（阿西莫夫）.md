# 基地（阿西莫夫）
<!--meta lib=主库 tiers=一、二、三、四、五、六、七-->

## 剧情

**【作品来源】**
《基地系列》（Foundation Series）为美国作家艾萨克·阿西莫夫（Isaac Asimov）创作的银河系社会学科幻史诗。核心三部曲《基地》（1951）、《基地与帝国》（1952）、《第二基地》（1953）由1942–1950年间《Astounding》杂志短中篇结集；后传《基地边缘》（1982）、《基地与地球》（1986）；前传《基地前奏》（1988）、《基地缔造者》（1993，遗作）。1966年获雨果奖「史上最佳系列」。气质：文明兴衰、心理史学宿命与个体抉择、政治/经济/科技/心灵感应组织博弈；少热血对轰，战力多为制度、贸易、核技术垄断与精神力。

**【世界定位】**
银河帝国末年，数学家哈里·谢顿以心理史学预言帝国将亡、三万年黑暗，遂设第一基地（端点星·物质科学）与第二基地（心灵科学）将黑暗缩为千年。主线沿谢顿计划展开：百科全书时代→科学宗教→商人霸权→帝国将军与骡→第二基地修补→盖娅与盖娅-星系抉择。一句话：以概率预测文明，用两座基地与一次集体意识选择，改写银河的未来史。

**【世界观 · 力量体系】**
本世界几乎无个人修真，冲突轴是**政治组织力、核科技与工业、贸易网络、心理史学概率、心灵感应与情感控制、行星级集体意识**。死亡规则：个体可被刺杀/战死；帝国与基地靠舰队、核动力与制度存续；「骡」类突变体可改写群体情绪；盖娅可共享全星感知。心理史学两前提：①研究对象数量达统计规模；②对象不知自身被纳入预测。后补：对象须为人类智人。预测以千分概率表述，越远越散；未知个体可令计划偏航，须第二基地修正。

力量/影响力层级（对照阶位战力，宁低勿高；本世界「战力」=可动员资源与改写局势能力）：

1. **帝国边陲平民 / 百科全书学者初期**  
   无舰队话语权；端点星贫矿农业殖民点。≈一阶。

2. **地方诸侯军 / 安纳克里昂等四王国舰队**  
   可围困端点、打局部星战；仍依赖帝国残余或基地核技。≈二阶。

3. **基地市长期 / 科学宗教扩张**  
   以核技术披宗教外衣控制周边；个人力场盾、旗舰遥控断电级手段。≈三阶。

4. **商人王子 / 贸易联邦区域霸权**  
   以禁运、黑市、外交颠覆灭国级客户；侯伯·马洛式经济战争。≈四阶。

5. **帝国末代将军 / 骡征服级危机**  
   贝尔·里欧斯可征伐；骡以情感控制五年席卷基地与周边，建「世界联盟」。≈五阶。

6. **第二基地心灵级 / 谢顿穹窿与计划修补**  
   心灵感应组织可定位、改写个体与群体情绪，修正谢顿计划；穹窿全息预言节点。≈六阶。

7. **盖娅 / 盖娅-星系级文明选择顶点**  
   行星集体意识→潜在全银河超级有机体；戈兰·特维兹的选择决定第二帝国形态。≈七阶（情报优先/条件性胜利）。

乐园阶位映射：一阶≈帝国边陲平民/基地初期学者；二阶≈地方诸侯军/商人时代前夜；三阶≈基地市长/科学宗教扩张期；四阶≈商人王子/区域霸权；五阶≈骡征服级危机；六阶≈第二基地心灵级/谢顿穹窿级；七阶≈盖娅/盖娅-星系级文明选择顶点。世界顶点=盖娅-星系（及机·丹尼尔·奥利瓦长程布局），低阶切入一律规避顶点·情报优先；严禁「遭禁制/条件受限/实力未减·仅策略规避」式降级顶点。

**【地理 · 舞台】**
- **川陀（Trantor）**：帝国首都，几乎全行星金属穹顶；图书馆、皇宫、斯璀璘大学；帝国崩溃后遭洗劫，第二基地隐于「群星尽头」实为川陀废墟上的知识与心灵核心。
- **端点星（Terminus）**：银河边缘贫瘠殖民星，第一基地所在；谢顿穹窿定时开启；由百科全书委员会→市长→商人寡头演变。
- **安纳克里昂（Anacreon）等四王国**：帝国边陲诸侯，早期威胁端点，后被科学宗教与贸易收编。
- **阿斯孔（Askone）、科瑞尔（Korell）**：商人时代扩张目标；宗教禁忌与帝国军火客户。
- **卡尔根（Kalgan）**：娱乐与军事要地，骡建「世界联盟」首都，自封联盟第一公民。
- **西维纳（Siwenna）**：旧帝国行省，贝尔·里欧斯与侯伯·马洛均涉此地。
- **盖娅（Gaia）**：集体意识行星，万物共享「盖娅」；后传核心舞台。
- **地球与外世界遗迹**：后传寻根线；机器人时代余波与机·丹尼尔长生布局。
写场景强调：川陀金属穹顶下的人群潮、端点穹窿红光、商人飞船货舱的核电池、卡尔根宫殿的恐惧忠诚、盖娅草叶与飞鸟同声呼吸。

- **赫利肯星（Helicon）**：谢顿故乡，武术与数学并重的边陲文化符号。
- **赛协拉等中转世界**（后传）：特维兹航线的社会标本，展示基地时代外缘多样性。
- **月球/地球卫星设施**：丹尼尔长眠与终局对话舞台。
- **新川陀**：帝国残余政治中心意象，与废墟川陀对照。


**【世界剧情线】**

○ **前史与前传：心理史学诞生（约银河纪元11988–12069 / 基地纪元前）**  
哈里·谢顿生于大角星区赫利肯星，中产，数学天分与武术底子兼备。银河纪元约12020年前后来川陀，于十载会议提出心理史学雏形，引起皇帝克里昂一世与宰相关注。记者兼向导夫铭（实为机·丹尼尔·奥利瓦）助其脱身，结识铎丝·凡纳比（后为妻，亦有机器人身份线）、养子芮奇、合作者雨果·阿马瑞尔。谢顿在川陀各区（帝国区、达瑞尔、麦曲生、覆亡荒原等）采集社会样本，将心理史学从假说推为可算体系。克里昂遇刺、帝国行政腐化加深；谢顿晚年任首相又退隐，组建「谢顿计划」人员，录制谢顿穹窿全息讯息，并暗设第二基地。帝国以「百科全书计划」为名将第一批学者流放端点星——实为谢顿故意促成的「安全流放」。谢顿卒于基地元年前后，成为传说中的「乌鸦嘴」。

○ **心理史学家篇：审判与流放（基地纪元0前后）**  
盖尔·多尼克赴川陀追随谢顿，见证公共安全委员会（林吉·陈等）审讯。谢顿公开预言：帝国将在三百年内崩溃，银河将入三万年蛮荒；若建立两座基地，可将黑暗缩为一千年。委员会不敢杀之以成烈士，遂批准「百科全书基金会」迁往端点星。此阶段定下主轴：第一基地在「群星尽头」的物质科学避难所，第二基地在「星空的另一端」修修补补。

○ **百科全书编者篇：第一次谢顿危机（约基地纪元50）**  
端点星资源贫乏，被安纳克里昂等四王国环伺。百科全书董事会（刘易斯·皮兰等）只顾编书，市长塞佛·哈定看清：帝国已无力保护，唯有政治。哈定与约罕·李发动政变，掌握实权。他挑动四国互相牵制，使端点的核技术成为谁都不敢独吞的「平衡砝码」。谢顿穹窿首次开启，全息谢顿揭穿：百科全书只是幌子，真正目的是建立第二帝国的种子；危机本身被心理史学算入。哈定名言气质：「武力是无能者最后的借口」——本阶段胜利靠均势与制度，不靠舰队硬刚。

○ **市长篇：科学宗教（约基地纪元约80）**  
基地以核技术为「神迹」，向外输出科学宗教：周边王国的反应堆、医疗与舰船由基地祭司阶层操控。安纳克里昂摄政温尼斯欲用修复后的帝国战列舰突袭端点。哈定早令技师埋下遥控切断，旗舰叛变，全星核动力熄灭；温尼斯在个人力场盾前溃败自尽。宗教扩张巩固基地生存，却埋下「信仰僵化」隐患——谢顿计划下一阶段将用贸易取代神权。

○ **商人篇：阿斯孔楔子（约基地纪元约135）**  
林马·波耶茨等商人以违禁核技与黄金诱饵，撬开阿斯孔宗教禁忌，救出特工埃斯克尔·戈洛夫，换取锡矿与市场。贸易成为新楔子：科技不再只披神袍，也披利润。

○ **商人王子篇：侯伯·马洛（约基地纪元约155）**  
三艘基地船在科瑞尔附近失踪。侯伯·马洛奉命调查，识破「传教士」实为科瑞尔密探的陷阱，拒绝被宗教借口拖入战争。他单探西维纳，见旧帝国衰败与仍能输出军械的帝国工厂。回端点后于公审中翻盘，当选市长，清洗政敌。科瑞尔开战，马洛以全面禁运使其经济崩溃——**贸易战取代圣战**。商人寡头时代开启，基地成为区域霸权。

○ **帝国将军篇：贝尔·里欧斯（基地与帝国·上）**  
残存帝国名将贝尔·里欧斯受命征讨「边陲异端」基地。他精明、忠诚却被宫廷政治捆住手脚；基地商人与情报网周旋。心理史学意义上，帝国对基地的「最后一击」注定因帝国自身腐朽而失败：补给、猜忌与皇帝的恐惧令里欧斯功败垂成。名场面是「死手」——历史规律比名将更硬。帝国军事威胁告一段落，基地信心膨胀。

○ **骡篇：计划最大破口（基地与帝国·下）**  
突变体「骡」能直接改写他人情感：令敌人恐惧、令部下狂热忠诚。他先控海盗，再夺卡尔根，五年内攻陷第一基地，建「世界联盟」，自封第一公民，定都卡尔根。谢顿穹窿开启时预言与现实脱节——**个体突变不在统计之内**。贝泰·达瑞尔、托兰·达瑞尔等卷入；「小丑」马格尼菲克（视觉奏鸣器艺人）实为骡伪装。贝泰在关键时刻以情感冲击打断骡的控制链。骡因畏惧传说中的第二基地而停止扩张，转而搜寻「群星的尽头」。

○ **第二基地篇：搜寻与反搜寻（第二基地）**  
骡派汉·普利吉与年轻心灵感应者（及贝泰之女阿卡迪线）追踪第二基地。第二基地以心灵科学隐于川陀（「群星尽头」字面与心理史学双关），首脑「第一代言人」等布局：诱导、记忆修改、舆论操控。骡最终被第二基地转化野心，沦为无害统治者，不久病逝；世界联盟瓦解。其后第一基地与第二基地互相猜忌：端点星发展「心盾」等反心灵技术，阿卡迪·达瑞尔以少女视角完成情报关键一环。第二基地制造「已毁灭」的假象，继续秘密监护谢顿计划。

○ **基地边缘：五百年后的分岔（约基地纪元约498）**  
议员戈兰·特维兹因质疑谢顿计划被放逐，历史学家詹诺夫·裴洛拉特同行。二人寻找「地球」与第二基地真相，途经赛协拉、盖娅等。盖娅是行星级集体意识：人、兽、草木、矿物共享心灵网络，个体仍保留人格但可接入整体。第二基地代言人史铎·琴诺维兹等亦介入。三方在盖娅对峙：第一基地的物质扩张、第二基地的心灵监护、盖娅的集体道路。特维兹被要求「以直觉做银河级选择」——他选盖娅，因直觉判断：仅物质或仅心灵监护都不足以防范更大未知（如外星智慧），需要更大尺度的生命联合。

○ **基地与地球：寻根与终裁**  
特维兹、裴洛拉特与盖娅人布莉丝（Bliss）寻访传说中的地球与外世界。历经强迫记忆、辐射废土、太空族遗迹等，最终在月球（或地球卫星基地设定）遇见长生机器人**机·丹尼尔·奥利瓦**：他已存活两万年，曾以夫铭等身份引导谢顿，并推动盖娅实验。丹尼尔揭示：人类银河之外可能存在威胁；盖娅应扩展为**盖娅-星系（Galaxia）**——全银河超级有机体。特维兹再次确认选择。结局开放而沉重：第二帝国或许不是帝国，而是星系级生命；谢顿计划被「更大棋局」覆盖，却未简单作废。

○ **结局状态（可锚定）**  
第一基地仍握物质科技与贸易遗产；第二基地心灵监护传统延续但被盖娅路径超越；骡已死、联盟已散；帝国残骸不再是主矛盾；盖娅-星系成为文明顶点方向。契约者可改局部人物生死与某次危机手段，不可无代价抹去：心理史学前提、骡作为统计外个体、两基地分工、盖娅集体意识与特维兹选择的题眼。


○ **阶段细述：第一次危机的因果链**  
端点星建城后约五十年，帝国边陲已名存实亡。安纳克里昂公国最先撕下帝国封号，派安瑟姆·霍特·罗德里克式使节到端点「视察」，实质是丈量征服成本。百科全书董事会以「帝国特许」自欺，刘易斯·皮兰拒绝动员。塞佛·哈定看穿：帝国大臣多尔温勋爵的造访不过是文明礼仪的尸体展览——中央已无力派舰队。哈定联合约罕·李控制通讯与卫队，在董事会会议上完成不流血夺权。随后他向较弱的三国暗示：若安纳克里昂独吞端点核技，三国将永居下游。三国联合通牒迫使安纳克里昂接受「核技术共享监管」框架。穹窿开启，谢顿说明一切在算中：基地必须学会在无帝国保护下用政治生存。此战未开一炮，却奠定「危机—抉择—穹窿认证」的仪式结构，后世每次危机都回响这一模板。

○ **阶段细述：科学宗教如何运转**  
基地向外输出的不是单纯发电机，而是「神职—技术」捆绑：反应堆由受训祭司启动，医疗核素由神庙配给，战舰引擎维修手册写成经文。周边王国上层获得舒适与军力，下层获得神迹叙事，基地获得原材料与战略纵深。温尼斯的反扑代表旧式武力对神权—技术复合体的最后一搏；旗舰切断与全星断电证明：真正的武器是供应链与后门，而非舰炮口径。哈定让温尼斯在个人力场盾前自杀，完成的是「不可战胜的基地」神话，也为后来宗教僵化付账——当神话太成功，改革者马洛必须更狠。

○ **阶段细述：从波耶茨到马洛的经济战争**  
阿斯孔禁核教条看似坚硬，实则上层需要黄金与地位。波耶茨用「铁变金」装置与录像勒索，把宗教禁忌撕开一个可贸易的口子。马洛走得更远：他拒绝被「传教士被害」的舆论绑架，证明所谓传教士是科瑞尔特工；他向民众展示帝国仍能输出武器，基地必须用贸易而非圣战回应。禁运使科瑞尔工厂停转、民生崩溃，政权屈服。马洛当选市长并逮捕苏特、曼利奥，标志权力从「神庙—市府」转向「船队—交易所」。谢顿计划在此阶段需要的是扩张速度与物质优势积累，好在帝国将军到来前足够强，又在骡到来前足够骄傲——骄傲是陷阱的一部分。

○ **阶段细述：里欧斯与「死手」**  
贝尔·里欧斯是帝国军事传统最后的荣光：他能打、能算补给、能识破基地商人的部分诡计。但他的失败不在战术，在帝国政治生态——皇帝与朝臣恐惧功高盖主的将军，胜过恐惧边陲基地。心理史学把这种「中央猜忌前线」写成必然：帝国越衰，越杀自己的名将。里欧斯线告诉读者：有时敌人最强的武器是你自己的制度缺陷。基地因此获得战略自信，开始低估「统计外个体」。

○ **阶段细述：骡的五年与情感物理学**  
骡不靠说教，靠直接改写情绪变量：恐惧使舰队投降，忠诚使海盗变禁卫，绝望使议会签字。他定都卡尔根，把娱乐星球变成军事首都，以「世界联盟第一公民」称号回避皇帝名号的历史包袱。攻陷端点时，谢顿穹窿的过时预言成为最大心理打击——计划「失灵」比战败更摧毁信仰。贝泰在关键节点以强烈情感冲击打断骡对托兰等人的控制，证明情感武器是双刃。骡停战五年搜寻第二基地，是因为传说中「心灵科学」是他的克星；他越接近真相，第二基地越能反制。最终转化不是肉体消灭，而是野心与攻击性的心灵重写——计划以「无害化」而非英雄斩杀收场，符合阿西莫夫对制度与心理的偏好。

○ **阶段细述：两基地互猎与阿卡迪**  
骡死后，第一基地要肃清「心灵阴影」，第二基地要隐藏自身并修复概率曲线。阿卡迪·达瑞尔以少女旅行者身份穿越政治与心灵战场，完成成人无法完成的信息传递：她的「无害」是伪装，也是心理史学利用个体的又一例——但她比骡可控。第二基地导演「自身灭亡」的假象，让第一基地放下屠刀，自己继续在川陀废墟与星空心理空间中监护千年工程。

○ **阶段细述：特维兹选择的哲学结构**  
五百年后，第一基地物质力量接近「第二帝国」外形，第二基地监护也近完成。戈兰·特维兹的怀疑本身被利用：他被放逐，却因此获得在盖娅做选择的资格。三方方案：①第一基地式扩张——自由但可能僵化与战争；②第二基地式监护——稳定但精英心灵专制；③盖娅——集体意识，牺牲部分「原子化自由」换整体智慧与防御。特维兹选③，后在地球线确认扩展为盖娅-星系。丹尼尔的出场把故事从「谢顿千年」拉到「机器人两万年」：心理史学是工具，人类整体存续是目的。结局不是加冕礼，是开放的文明形态投票。

○ **叙事执行要点（AI入世常读）**  
描写优先会议桌、货舱账本、穹窿红光与心灵静室，而非无敌对轰。危机到来时先给「统计必然」的压迫感，再给「个体意外」的裂缝。谢顿全息说话应克制、像工程师报告而非神谕咆哮。商人场景写清：缺什么原料、禁运几周后哪条生产线停、哪位议员转向。骡场景写清：同一句话在控制前后语气如何变。盖娅场景写清：个体仍能说「我」，但「我」背后有行星在听。


**【世界剧情线·补密·谢顿计划节点档案】**
心理史学将银河视作可统计的文明流体：个体自由意志在兆亿人口尺度上被抹平为趋势，而未知突变体与心灵组织则是趋势方程中的奇点。契约者切入时，应优先判断「当前处于谢顿危机前后哪一环」：百科全书伪装期、科学宗教扩张期、商人贸易联邦期、帝国将军征伐期、骡乱与第二基地修补期，或盖娅抉择期。每一环的胜利条件不同——有时是拖延围城，有时是开放贸易，有时是隐藏第二基地坐标，有时是选择是否加入行星集体意识。

端点星的资源贫瘠被谢顿刻意写入：没有金属与农业余裕，就无法以帝国式扩张自毁，只能靠知识与制度输出。川陀则相反，巨型城市行星把行政神经集中到一点，帝国一倒，档案与官僚体系同时崩解，黑暗时代从「中心失忆」开始。第二基地的心灵科学不是魔法炫技，而是对心理史学的人工修正器：当概率曲线因骡而断裂，他们以精神力与政治伪装把曲线扳回千年帝国的轨道。

骡的出现证明「计划」并非神学：任何统计模型都可能被样本外个体击穿。贝泰与托兰线的价值在于普通人的选择仍能影响奇点结局——不是靠个人武力碾压舰队，而是靠信任、逃亡路线与对第二基地的误导。后传中戈兰·特维兹的「选择」进一步把主题从服从计划推向「人类是否应成为盖娅式整体」：情报优先、条件性胜利，严禁用「顶点无力」解释盖娅或丹尼尔级存在。

切入身份/时点：契约者以「川陀底层移民 / 百科全书抄写员 / 端点星农业技术员」切入，锚定谢顿受审前后至第一次危机前夕。

初始事件：你所在的端点档案库接到安纳克里昂使节通牒——停止「非法核研究」或断粮。董事会只催百科全书截稿，市府走廊里塞佛·哈定的助手在拉人开会。你手中一份反应堆维护日志，可能成为四国争夺的把柄，也可能成为哈定谈判的筹码。

开场白建议：「端点星的风里有盐和锈。川陀的金属天穹你再也看不见，只看见贫瘠土壤上竖起的穹窿工地。通牒钉在公告板上，墨迹未干，有人已在赌安纳克里昂哪一天打来。你把日志藏进袖管——在这颗被流放的星球上，知识有时比枪更危险。」

关键NPC立场：**哈里·谢顿**（全息或回忆中的权威，不可妄测）；**塞佛·哈定**——要实权与均势，视空谈学者为累赘；**刘易斯·皮兰**——死守百科全书；**约罕·李**——哈定刀笔吏；安纳克里昂使节——要核技或要征服。

主线钩子/支线：钩子=第一次谢顿危机的情报与政变夜；支线A=保护某卷百科全书母版离星；支线B=伪造四国互相猜忌的外交文书；支线C=在贫矿区找到可交易的稀土以换食物。

危险度/规避：危险度中——诸侯军可屠城级威胁但尚无骡；规避帝国特务与董事会清洗。勿挑战不存在的「个人无敌」。

任务方向/奖励：活过第一次危机；奖励=端点居留权、基础核技学徒资格、哈定阵营信任、心理史学通俗讲座笔记（非完整方程）。

**二阶（地方诸侯军 / 商人时代前夜 · 边境火药线）**

切入身份/时点：契约者以「安纳克里昂舰上技师 / 四国边境走私船长 / 基地派驻祭司学徒」切入，锚定科学宗教成型前的军备竞赛期。

初始事件：你所在的诸侯旗舰刚接收「帝国遗产」武器，却发现关键线圈印着端点序列号。温尼斯一系军官要你鉴定能否对端点开火；基地地下联络人要你把序列号拍回端点。天亮前必须选边——或两边都骗。

开场白建议：「舰桥玻璃外是四王国共有的黑暗。有人把核芯叫神迹，有人叫军火。你握着线圈的手在出汗：一旦点火，端点会亮，安纳克里昂也会亮，而你会先变成两边共同的叛徒。」

关键NPC立场：**温尼斯**——要战争与摄政权威；**波利·维里索夫**——祭司兼大使；**塞佛·哈定**（远程）——要遥控权；地方舰长——怕死怕叛。

主线钩子/支线：钩子=旗舰暗门与全星断电前夜；支线A=护送核电池商队过封锁；支线B=在阿斯孔前夜建立黑市客户；支线C=救出被控异端的基地技师。

危险度/规避：危险度中高——局部星战与宗教审判；规避个人力场盾对轰幻想，重渗透与证据。

任务方向/奖励：改变某次边境交火结果；奖励=商人执照、祭司阶位伪装、四国通行令、小型力场发生器（消耗型）。

**三阶（基地市长 / 科学宗教扩张期 · 神迹与政变线）**

切入身份/时点：契约者以「端点市府分析官 / 安纳克里昂神庙高阶技师祭司 / 行动党线人」切入，锚定哈定第二次危机（温尼斯突袭）。

初始事件：穹窿开启倒计时三十日。温尼斯的战列舰「修复完成」，少年国王勒波德成为傀儡。你发现旗舰主控被植入远程切断——密码在哈定一系手中。行动党要你公开密码逼市长开战；哈定要你守口如瓶直到仪式日。

开场白建议：「神庙里的反应堆发出低鸣，像一尊金属的神在呼吸。信徒跪下去的时候，你听见的不是祈祷，是冷却泵。哈定在通讯里只丢下一句：让他们先开火。你知道真正开火的不是炮，是开关。」

关键NPC立场：**塞佛·哈定**——成竹在胸；**温尼斯**——疯狂豪赌；**勒波德一世**——恐惧；**塞夫·瑟马克**等行动党——要强硬；**西奥·阿波拉特**——旗舰祭司。

主线钩子/支线：钩子=旗舰叛变与温尼斯溃败；支线A=保护少年国王免死；支线B=夺取战列舰导航芯；支线C=改革神庙教育去迷信化（为商人时代铺路）。

危险度/规避：危险度高——政变与宗教战争；规避「一拳打碎战列舰」。条件性胜利：保全端点与核垄断叙事。

任务方向/奖励：市长信任状、科学宗教法典修订权、力场盾维护许可、四国传教护照。

**四阶（商人王子 / 区域霸权 · 禁运与公审线）**

切入身份/时点：契约者以「马洛船队大副 / 科瑞尔军工采购代理 / 端点议会反对派秘书」切入，锚定侯伯·马洛调查失踪飞船至禁运战。

初始事件：三艘基地船失踪。你随马洛降落科瑞尔，一名「约德·帕尔玛神父」哭求避难——人群已举起火把。马洛要你记录人群面孔；苏特一系要你作证马洛「出卖传教士」。你按下记录键的瞬间，战争与公审同时启动。

开场白建议：「科瑞尔的空气里有机油和香料。马洛说：宗教是旧刀，贸易是新刀。你看着那名神父的袖口露出军用编号，突然明白刀锋不在剑上，在证据链上。」

关键NPC立场：**侯伯·马洛**——要真相与市长之位；**乔兰·苏特**——要除掉马洛；**普布利斯·曼利奥**——外交机器；**奥努姆·巴尔**（西维纳）——旧帝国衰败证人；科瑞尔统治者——要帝国军火。

主线钩子/支线：钩子=公审翻盘与禁运灭国；支线A=西维纳工厂图纸窃取；支线B=建立绕开宗教派的纯贸易航线；支线C=营救被扣商船船员。

危险度/规避：危险度高——星际禁运与政治谋杀；帝国残余军工厂是背景板勿硬闯核心。

任务方向/奖励：贸易特许、商人王子级船队股份、禁运情报网节点、西维纳古星图。

**五阶（骡征服级危机 · 情感战争线）**

切入身份/时点：契约者以「基地舰队参谋 / 卡尔根宫廷乐师助手 / 达瑞尔一行同行商人」切入，锚定骡攻陷基地前后。

初始事件：谢顿穹窿开启，全息预言与现实完全错位——端点陷落的消息比穹窿的光更早传到。你在人群恐慌里看见一名丑陋的「小丑」调试视觉奏鸣器，军官们忽然泪流满面宣誓效忠「第一公民」。有人塞给你一枚心盾原型芯片：戴上，或跪下去。

开场白建议：「卡尔根的夜宴从不间断，仿佛战争只是背景音乐。你听见奏鸣器里有颜色在爬进血管。旁边的将军笑着说他从未如此热爱主人。你摸到芯片发烫——在这个时代，恐惧和忠诚都是武器，而且比核弹便宜。」

关键NPC立场：**骡**——要征服与寻第二基地，可转化你；**贝泰·达瑞尔**——可结盟的意志杠杆；**托兰·达瑞尔**；**马格尼菲克**（骡伪装）；基地残存军官——已被改写或假装被改写。

主线钩子/支线：钩子=基地陷落与世界联盟建立；支线A=护送贝泰接近骡；支线B=销毁/转移核武库以免被骡征用；支线C=伪装效忠潜入卡尔根情报圈。

危险度/规避：危险度极高——情感控制可破队伍；**禁止写骡遭禁制/条件受限变弱**；条件性胜利：保全关键人物心智、延缓联盟扩张、向第二基地递送坐标谣言。

任务方向/奖励：心盾改良、卡尔根通行证、反情感训练、骡军编制内的「无害」职位掩护。

**六阶（第二基地心灵级 / 谢顿穹窿级 · 修正与猎杀线）**

切入身份/时点：契约者以「第二基地学徒 / 第一基地心盾工程师 / 普利吉远征队书记」切入，锚定骡搜寻第二基地至被转化、以及两基地互猎期。

初始事件：你被带进一间无窗的房间，代言人问你三个问题，而你发现自己「想回答」的内容在被轻轻改写。与此同时，端点星议会通过扩大心盾部署的法案，视一切心灵者为敌。你必须在「监护计划」与「基地自由」之间留下可被历史读到的痕迹。

开场白建议：「川陀的废墟上，风穿过旧图书馆的骨架。有人说这里是群星的尽头，有人说尽头在心里。代言人的声音很软：谢顿计划不是信仰，是工程。你忽然想起骡的眼睛——工程也会怕一个统计不了的人。」

关键NPC立场：**第二基地第一代言人**——计划至上；**汉·普利吉**——骡的猎犬；**阿卡迪·达瑞尔**——信息关键；第一基地军政要员——要消灭第二基地；**骡**（转化前）——要先下手。

主线钩子/支线：钩子=转化骡 / 制造第二基地「灭亡」假象；支线A=心盾与心灵战的技术对耗；支线B=保护阿卡迪完成情报；支线C=在川陀图书馆夺回心理史学残卷。

危险度/规避：危险度极高·贴近顶点前阶——记忆与人格可被改写；规避「物理打死代言人」简单化；条件性胜利：计划重回轨道或暴露最小代价的真相。

任务方向/奖励：心灵防护训练、心理史学入门公理（残缺）、两基地双面间谍身份、穹窿维护权限碎片。

**七阶（盖娅 / 盖娅-星系 · 文明选择顶点线）**

切入身份/时点：契约者以「特维兹放逐船随员 / 盖娅观察者 / 裴洛拉特研究助理 / 丹尼尔接触候选人」切入，锚定《基地边缘》三方对峙至《基地与地球》寻根终局。

初始事件：盖娅的草叶在你脑中轻轻说「你好」。史铎·琴诺维兹的心灵压来，第一基地舰长的指弹在发射键上。特维兹被要求选择：物质帝国、心灵帝国，或盖娅。有人把「外银河威胁」的旧档案塞进你掌心——选择的权重突然超过谢顿。

开场白建议：「你分不清耳鸣与行星的呼吸。特维兹说他只相信自己的直觉，布莉丝说直觉也是盖娅的一部分。远处金属地平线上，有什么比帝国更古老的东西在等待被命名。你知道这一枪若打出去，打中的不是敌人，是未来的形状。」

关键NPC立场：**戈兰·特维兹**——抉择核心；**詹诺夫·裴洛拉特**——寻地球；**布莉丝**——盖娅；**史铎·琴诺维兹**——第二基地；**机·丹尼尔·奥利瓦**——终局揭示者；第一基地舰队——武力选项。

主线钩子/支线：钩子=特维兹选择盖娅与盖娅-星系确认；支线A=寻地球路线生存；支线B=阻止任何一方提前开火；支线C=与丹尼尔谈判「人类整体」定义。

危险度/规避：危险度贴近顶点——**七阶顶点：盖娅-星系级，情报优先/条件性胜利**；严禁「封印丹尼尔/削弱盖娅」；可胜利形态：影响选择措辞、保全同伴、换取盖娅观察者身份、为乐园留下可回溯的文明档案。

任务方向/奖励：盖娅临时链接（有代价）、地球线星图、丹尼尔给出的有限真相、第二帝国形态的「观察席」而非统治权。
