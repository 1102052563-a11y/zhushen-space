## 剧情

**【作品来源】**
《Magical Charming!》（日文读作まじかる ちゃ〜みんぐ！；中文正式译名《魔卡魅恋！》）是 Lump of Sugar（方糖社）第 10 作，2013 年 5 月 31 日发售的成人向恋爱冒险／恋爱策略 ADV。原画梱枝りこ，剧本しげた・てつじん・二葉つばさ・冬茜トム，音乐大川茂伸，OP/ED 由 ave;new feat.佐倉紗織 演唱。引擎 Ethornell。类型标签：魔法少女学园 + 卡牌收集策略 + 纯爱。结局数 16。2013 Getchu 美少女游戏大赏综合 19、系统 6、角色部门葉山秋音 11；萌系游戏大赏角色设计金赏。HIKARI FIELD 推出中文版（本篇一度受 Steam 审核影响，后于官网发售；番外《Magical Charming! Chronicles of Zero／魔卡魅恋！零之编年史》曾上 Steam）。搜笔趣阁未收录（Galge）。

**【世界定位】**
会用「只有女性才该会的魔法」的普通少年夏本律，被拖进幻创庭国伊斯塔丽嘉的维斯利魔法学院，用クロノカード（Chrono Card／魔卡）收集日常与恋爱的「段取」，与公主奥莉艾塔、男装青梅叶山秋音、妹妹夏本讽歌、学生会长白神姬百合等人一起解开时间循环与「事故」的真相。一句话：在魔法学园里用卡牌排恋爱日程，最后为拯救世界与所爱之人选择「此刻」。适合写学园日常、卡牌养成、青梅／公主／妹／会长多线纯爱的契约者入世。

**【世界观 · 舞台设定】**
世界分「人间界」与「幻创庭国伊斯塔丽嘉」（オリエッタ常叫它マホーツ界）。后者由魔法构筑，一般人无法进出甚至无法「看见」；魔法使需专用飞行艇往来。庭国中枢有「泉」维持机构，需魔力灌注；魔力衰退意味着对这个世界的「认知」也消失——严重时会从存在层面被「吃掉」。因此学院的目的不是培养战争兵器，而是：在魔力自然衰退前保护学生、教会控制，避免暴走。

魔法在本作首先是**情感与日常的介质**：恋爱学科研究「魔法与恋爱感情的关系」；クロノカード把律见闻的行动、场所、魔法、记忆写成卡，用来推进约会与事件。卡的稀有度与类别影响「魔力数值」（TRUE 线解题用），但正文重点是「收集哪张卡＝经历哪段关系」，而非数值碾压。轮回天星、绝对王言、神魔讨击等固有魔法写的是角色心结与代价：时间回溯、认知强制、能量暴走——都落在「她为什么害怕／为什么隐瞒」上。

温度来自：学园走廊的喧闹、图书馆的纸墨、温室与石像、女仆的猫耳、恋爱学讲师的尴尬博客、合コン被律骂成「恶魔之宴」的段取主义。魔法学园外表在人间界挂名「真东女子学园」，普通科与魔法科并存——日常与奇幻共用同一栋校舍的错位感，是方糖社式喜剧来源。

**【地理 · 生活舞台】**
- **幻创庭国伊斯塔丽嘉**：魔法世界本体；泉、王室记忆、建国者伊斯塔丽嘉的阴影都在此。
- **维斯利魔法学院（ウィズレー魔法学院）**：主舞台。入门班ノービスメイジ与最上级ミスティック；按习熟度分班而非年龄。讽歌年纪小也能上上级班。
- **真东女子学园（人间界挂牌）**：对外名门女校形象；律作为「男的魔法使」插班的违和感从校门开始。
- **宿舍／公主监视房间**：律醒来被奥莉艾塔「抓获」的起点；同居式监视催生日常口角与心动。
- **图书馆**：秋音常驻阅读处；兰铎尔夫当司书兼时空学讲师，TRUE 线情报密集。
- **温室与伊斯塔丽嘉石像**：外表是植物管理区，实为建国者沉睡／献祭的象征空间。
- **恋爱学科教室／职员室**：加涅特（田中梦子）的恋爱讲座与「超尴尬博客」名场面温床。
- **食堂、商店、飞行艇甲板、城镇边缘**：卡牌事件与约会选项的常见场景。

**【故事主线 · 情感线】**
**前提：暑假被重置**  
夏本律过完一个「没什么收获」的暑假后沮丧入睡，次日醒来发现时间回到暑假前一天，自己躺在陌生房间，被自称奥莉艾塔·库洛诺·伊斯塔丽嘉·吉娜斯緹尔的少女困住。她指控律的存在搅乱了她的日常；为查清原因并满足她的要求，律转入专为魔法少女开设的维斯利魔法学院——作为「男性却能使用魔法」的异常者被监视。

**共通线：段取主义与魔卡日常**  
律公开宣称「想要女朋友」，却坚持自己的恋爱观「段取主义」：必须按正确步骤推进关系，厌恶跳步骤的合コン。学院生活中，他通过クロノカード记录见闻；卡牌组合改变事件走向（包括秋音妊娠 END、学院支配者 END 等分支）。入学班有公主奥莉艾塔、男装转来的青梅叶山秋音；上级班有妹妹讽歌与会长姬百合。讲师阵容包括恋爱学加涅特、精灵学兼女仆夏珑、物质学玛丽、时空学兰铎尔夫——后三者真实身份是伊斯塔丽嘉的使魔（黑猫／花／魔导书）。

共通日常：上课、任务、收集卡、被奥莉艾塔使唤、被秋音「当同性朋友」相处、被讽歌又傲又黏、被姬百合的中性气场与「想更像女孩子」的反差打动。喜剧与温柔并行，危机感逐渐从「我为什么会魔法」扩到「这个世界在循环什么」。

**奥莉艾塔线**  
公主外皮下是喜怒形于色的孩子气；在意身材、会为吸引律而按网友偏方狂吃卷心菜（其实讨厌）。固有魔法「轮回天星」可回溯时间与空间——她对律的愤怒与依赖，都连着「曾经失去过你」的时间创伤。个人线：监视变同居变恋人；王室立场与学院身份的冲突；HE 在承认任性也要被接住。写正文抓她「命令式撒娇」与「突然认真得吓人」的切换。

**叶山秋音线**  
温厚聪明、会读空气的青梅，男装追随律入学。无意识发动固有魔法「绝对王言」：让周围默认她是男性；知道真相者自动解除。个人线核心：从「同性朋友感」到被看见真实性别与恋心；努力型（练球、练格斗游戏）的笨拙浪漫。HE：卸下伪装与律在一起。人气高（角色投票常年前列），写时忌把男装写成永久工具人。

**夏本讽歌线**  
律的妹妹，幼时起在学院生活，优等生、被信赖，讨厌被当小孩。与奥莉艾塔吵归吵心相通；憧憬室友姬百合。固有魔法「天降灵祈」沟通精灵与羊驼。个人线：重逢后的别扭反抗→想撒娇的本音→恋爱 HE（注意原作尺度与兄妹线雷区，契约者可淡化血缘改写为「名义上的义理」仅当玩家明确要求时——默认忠于原作设定但正文侧重学业与精灵日常）。HE：被当成对等的人被爱。

**白神姬百合线**  
会长、凛冽美人、女生徒崇拜对象；本人自卑于「不够像女孩子」，想当新娘。固有魔法「神魔讨击」曾在幼时暴走，她因此长期封印直到学会控制。使魔是植物人偶雪莉。个人线：伪装放松（墨镜吃汉堡）、训练与告白、HE 允许她软弱。写正文抓「公共形象 vs 私密愿望」。

**TRUE 线（真结局）**  
与体验版／《零之编年史》叙事咬合：体验版故事并非无关广告，而是本篇 TRUE 的障眼法与前置。律所持クロノカード在本篇为「真品」（体验版为复制品）；循环前他从伊斯塔丽嘉处受让真卡。建国者伊斯塔丽嘉（奥莉艾塔之妹）用「梦想灯火」操作记忆，并以自身化为石像维持泉；泉枯竭时会「吃掉」魔法使的历史（循环前律几乎被吃掉）。TRUE 需要特定卡牌与属性／数值知识推进，终局庭国崩解、魔法自世界消失，伊斯塔丽嘉被救出，姐妹以普通人身份重获生活。卡牌与魔法一同消散——胜利的代价是「再也没有魔法的日常」，但人还在。

**其他 END**  
16 结局含各女主 HE／BE 与特殊卡牌向 END（学院支配等）。写契约者攻略以四女主 HE + TRUE 为主轴，特殊 END 作「卡牌玩坏了」的黑色幽默支线即可。

**卡牌系统如何推动情感（正文必懂）**  
クロノカード不是抽卡氪金，而是「把今天过过的生活写成可调用的记忆碎片」。去图书馆可能刷出场所卡，对秋音说错话可能刷出记忆卡，看奥莉艾塔施咒可能刷出魔法卡。组合卡＝改写当天下午的事件树：同一天可以变成「认真约会」也可以变成「学院支配者闹剧」。稀有度与类别换算魔力值，主要服务 TRUE 线的解题节奏——四元素相克（火＜水＜风＜土＜火）与ヘブル数字读法是「通关知识」而非日常强弱榜。写正文时让玩家「为了见某个人而收集某张卡」，比让玩家「为了数值而刷卡」更贴原作。卡牌 UI 可以写成律的手帐：每张卡背面都有一句当天的心情，翻卡＝翻回忆。

**循环感的写法**  
律的违和来自「暑假被退货」：蝉还在叫，作业进度却不对，有人对他露出「你应该记得」的眼神。奥莉艾塔的怒气背后是「我已经失去过你一次」。学院里使魔三人组偶尔说出超龄的话，像活得太久的人。温室石像在夜色里「像在听」。这些都是 TRUE 的伏笔，共通线阶段只给不安，不给答案。

**各线情感节奏补笔**  
奥莉艾塔线的甜来自「命令你陪我」逐渐变成「可以请你留下来吗」；秋音线的甜来自「把你当兄弟」被打破的羞耻与解脱；讽歌线的甜来自「别把我当小孩」与「其实想被摸摸头」的拉锯；姬百合线的甜来自「大家面前的完美会长」只对你摘墨镜。四条 HE 都允许毕业后魔力衰退——爱要在「魔法变少」之后仍然成立，才算过关。

**【隐藏剧情 · 真结局 · 伏笔】**
- 体验版／《零之编年史》必须先玩才能完整理解 TRUE 的「障眼法」。  
- 律是奥莉艾塔使魔故能用卡；微量魔力来自长期承接秋音／讽歌／姬百合等人的「思念」。  
- クロノカード在轮回中携带记忆与经验穿越，半因奥莉艾塔的执念。  
- TRUE 后魔法消亡＝学园奇幻层关闭，后日谈应写人间界日常，勿无限续魔法战争。  
- 特殊卡牌 END 是系统玩法彩蛋，非正史主轴。

**【氛围基调 · 雷区】**
基调：方糖社式甜腻喜剧 + 中盘开始的时间循环哀愁；卡牌 UI 带来的「养成感」。PC 原版 R18，中文／部分移植注意版本尺度；NSFW 可点到关系确立，不冲淡学园与 TRUE。  
忌：把魔法写成等级碾压 PK；忌忽略段取主义把律写成无脑后宫；忌永久男装化秋音；忌无代价后宫同时攻略；忌跳过卡牌收集空喊真爱；忌 TRUE 后仍当魔法全开世界。  
最适切入：律被奥莉艾塔抓获的早晨，或入学第一周。

**角色互补与学院生态（再补）**  
ノービスメイジ是「大多数人毕业前都待在这」的入门池，喜剧密度最高；ミスティック是优等生与危险固有魔法的聚居地，讽歌与姬百合的压力戏多在此。恋爱学科把「心动」当研究对象，物质学与精灵学提供事件道具（花、召唤兽、羊驼），时空学图书馆是 TRUE 的情报井。使魔三人组构成成人侧喜剧三角：猫耳完美主义、赌徒废柴天才、知性腐向司书——他们服务公主与建国者，也对律的「男性魔法使」身份格外宽容。写群像时让每个人有「今天在忙的一件小事」，学院才像活的。

秋音刚速球：青梅用运动证明「我为你练过」，比直球告白更动人。  
讽歌的漫画原稿被撞散：帮她捡稿纸时读到自己出镜的故事。  
姬百合的雪莉使魔在会客室跳舞：严肃会长的治愈缝隙。  
TRUE 终盘魔法消散：咒语失效的瞬间，牵手仍在——用「失去奇幻」衬「留下恋爱」。  
体验版与本篇的「双故事」：先玩零之编年史再进本篇，读者会像拆礼物一样理解障眼法；契约者周目也可模拟「先过短篇暑假再重置入学」。方糖社招牌的「可爱皮、认真核」在本作体现为：前半用卡牌与口角逗笑，后半用时间与献祭逼你做选择——写正文务必保留这道落差，而不是从头甜到尾或从头虐到尾。若字数仍紧，优先补「四人各自的课表冲突」：同一下午奥莉艾塔要监视、秋音要练球、讽歌要交精灵报告、姬百合要开学生会——律只能选一个人陪，选择即攻略。这就是本作的日常戏剧核心所在啊。

## 休闲切入点

> 本世界为休闲／恋爱向（魔法学园 + 卡牌日常），无生存压力与战斗力数值体系。契约者以**日常身份**融入，核心玩法＝relationship 攻略 + 卡牌事件，而非任务厮杀。

切入身份：维斯利魔法学院插班生／转校的「特殊编入生」（最顺）；或人间界真东女子学园相关旁听、图书馆助手、恋爱学科实验志愿者、飞行艇见习乘务。优势是观察、记笔记、愿意按「段取」推进关系——正好契合クロノカード收集。

切入时点：夏本律被奥莉艾塔抓获并决定入学的当周（推荐）；或ノービスメイジ分班稳定、秋音男装转来之后——四人关系网刚铺开。想碰 TRUE 氛围可从「感觉暑假被重置过」的违和感开始，但先别剧透石像。

初始处境：住学院宿舍或被安排与监视对象同层；班级在ノービスメイジ；手里可能有一张入门卡或空白日记代替卡组。认识顺序建议：律（同班）→奥莉艾塔（监视口角）→秋音（青梅气场）→讽歌／姬百合（上级班偶遇）。

开场白建议：「你是在蝉声里醒来的。窗帘不是自己房间的颜色，床边坐着金发的小个子少女，正用公主的口吻宣判：『都是你害的。』走廊外有人练咒文，有人讨论恋爱学分，还有人把汉堡藏在墨镜后面吃。一张透明的卡片从你掌心亮起，上面写着今天的地点——图书馆。在这个只有被选中的人才能看见的国度里，恋爱似乎也要先抽卡。」

可攻略对象：  
- **奥莉艾塔·库洛诺·伊斯塔丽嘉·吉娜斯緹尔**：同班／监视；吃「接住任性 + 认真查循环 + 不拿身高身材开玩笑过火」；心结是时间与失去。  
- **叶山秋音**：同班青梅；吃「把她当朋友也当女孩 + 尊重男装期边界 + 陪练」；心结是绝对王言与自我认知。  
- **夏本讽歌**：学院走廊／精灵课；吃「不把她当小孩 + 认可成绩与漫画」；心结是兄妹距离与自尊。  
- **白神姬百合**：学生会；吃「允许她软弱 + 保守秘密（汉堡）」；心结是形象与神魔讨击阴影。  
- **加涅特**（喜剧向）：恋爱咨询，勿当真攻略主轴。

日常玩法钩子：  
1. **卡牌收集线**：按地点／行动触发新卡，组卡改写当日事件。  
2. **学园课程线**：恋爱学作业、精灵学实习、温室值班。  
3. **四人约会线**：分别推进好感，注意段取，避免一天内三连告白。  
4. **循环违和线**：重复的一天、被人遗忘的细节，导向 TRUE 线索。  
5. **使魔喜剧线**：猫耳女仆、赌徒讲师、魔导书司书的日常委托。  
6. **版本线**：先《零之编年史》再本篇 TRUE，可作「前史周目」玩法。

氛围／雷区：甜、闹、偶发催泪；魔法服务于恋爱与救赎。忌 PK 化魔法；忌 OOC 抹掉段取主义；忌强行战争化庭国；忌无代价后宫。R18 版本点到为止即可。契约者若非律，可作「另一名持卡见习生」，勿抢走 TRUE 钥匙卡的唯一性——可并列收集、最终把关键卡交给他。开局先学会学院礼仪与「不要对公主说你矮」，再谈轮回。

事件线示例：  
- 第 1 日：被奥莉艾塔抓获，领到学生证与空白卡槽，在食堂被全场注视。  
- 第 3 日：与秋音去图书馆，触发绝对王言的细微违和（有人对「她」用错称呼）。  
- 讽歌的精灵课：帮她找回走失的アルパカさん，换一张精灵相关卡。  
- 学生会委托：陪姬百合夜巡，路上经过快餐店她坚持绕路。  
- 循环征兆周：同一节恋爱学讲义出现两次，加涅特却毫无自觉——开始记「重复日志」。  
- TRUE 准备：按火→水→风→土的相克顺序整理卡组，与律分工查温室石像。  
- 追加日常：帮夏珑送点心给奥莉艾塔；替玛丽保管「绝不拿去赌」的零用钱；向兰铎尔夫借一本禁书却只敢读序章；在人间界校门假装普通高中生与游客擦肩。  
- 好感小技巧：奥莉艾塔吃大豆制品时陪她；秋音练球时当捕手；讽歌交精灵作业前帮她对答案；姬百合想女装采购时当人肉衣架——每条都对应一张可能的卡。开局口头禅建议：「先把今天的卡集齐，再谈喜欢。」
