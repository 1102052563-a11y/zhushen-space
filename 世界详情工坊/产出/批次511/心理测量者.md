## 剧情

**【作品来源】**  
日本原创电视动画《PSYCHO-PASS サイコパス》（中国大陆译名《心理测量者》，台港译名《PSYCHO-PASS 心灵判官》）。由 Production I.G 制作（第一季、第三季与多部剧场版；第二季由龙之子制作公司负责动画制作），故事原案虚渊玄（Nitro+），人物原案天野明，导演盐谷直义，总导演本广克行；音乐菅野祐悟。2012年10月—2013年3月于富士电视台 noitaminA 档首播第一季（全22话）；2014年播出第二季；2015年上映《PSYCHO-PASS 剧场版》；2019年推出剧场版三部曲《Sinners of the System》与第三季；2020年剧场版《3 FIRST INSPECTOR》；2023年十周年剧场版《PROVIDENCE（天意）》。类型为赛博朋克、犯罪、反乌托邦心理惊悚；文风冷峻思辨、血腥与哲学引用并重，强调「当法律无法裁定」时的伦理抉择。

**【世界定位】**  
22世纪日本被「西比拉系统（Sibyl System）」以声像扫描全面量化公民精神状态（心灵指数 / Psycho-Pass）与犯罪系数，形成「预防犯罪」的极权式安定社会。主线以厚生省公安局刑事课第一分队为舞台：新人监视官常守朱在执行官狡啮慎也等人协助下，追查免罪体质者槙岛圣护、复合体鹿矛围桐斗、以及更深层的彩虹桥与和平破坏者等暗影势力，追问「正义由谁定义」。

**【世界观 · 力量体系】**  
本世界几乎不存在超自然修炼，战力来自人体素质、格斗训练、火器与系统授权武器、无人机与信息战。核心规则是「西比拉」对生物力场的实时诊断——色相清浊、犯罪系数数值决定是否被执法、隔离或处决；主宰者（Dominator）只有在系统判定达标且持有者身份合法时才会解锁扳机。心灵危害（Psycho-Hazard）可使不稳精神污染周围人群色相，故执法现场本身就是二次危机源。职业适性由系统终身分配，「人尽其才」同时取消了选择错误的权利，也取消了选择正确的自由。  
**社会分层（非传统境界，但决定可战与否）：**  
- **色相清澈市民**：犯罪系数远低于警戒线，日常无武装；体能≈现实人类；靠全息与精神护理维持「美丽心灵」。  
- **精神护理对象 / 色相混浊者**：需接受治疗，若恶化则被送往矫正设施；可能被企业或黑市利用。  
- **潜在犯**：犯罪系数持续超标（常见阈值约100以上进入执法视野；300以上常触发致命排除）；多数被隔离，少数被公安局吸收为执行官。社会对潜在犯及其家属存在歧视（宜野座对征陆的恨即源于此）。  
- **监视官**：犯罪系数极低的精英刑警，指挥执行官，有权对失控执行官开火；任职十年可望升入厚生核心，但长期面对残酷现场易染浊降职。  
- **执行官**：具备理解犯罪、近战与追踪能力的「猎犬」，多为潜在犯或由监视官降职者；受监视、宿舍管制，值勤时持主宰者。  
- **免罪体质者（犯罪无症状者）**：声像扫描与犯罪心理不一致，犯罪系数可长期极低甚至可自控；主宰者对其往往无效，只能物理逮捕。出现率约二百万分之一；西比拉称其为「天生的罪人亦是圣人」，并筛其为系统大脑候选。  
- **系统级存在**：西比拉本体为大量「免罪体质」大脑组成的生物计算机集群（第一季时约247名构成员，约200名按表轮值监控全国），可强制改写主宰者模式、灭口知情者、决定政策；其「战力」体现在全网授权与制度暴力，而非单体破坏力。二期后导入集团性心灵指数，并清理部分高系数大脑。  
**武器与装备档：**  
- **主宰者（携带型心理诊断·镇压执行系统）**：联网后按目标犯罪系数切换——非致命麻醉枪 / 致命排除枪（系数≥300）/ 摧毁分解枪（对机械·无人机）；需眼球生物认证；断电或无网则成废铁；充能下排除约4发、分解约3发的量级限制。  
- **强袭型主宰者**：射程更远、可隔墙检测与射击，误伤风险高（第二季曾误杀监视官青柳）。  
- **主宰者SG原型（猎枪型试作）**：可锁定复数目标的散射追踪弹，冷却约23秒、耗电高，分解模式可威胁整栋公安局建筑，被局长判定过危而封印于机房暗盒，后成反攻彩虹桥的关键。  
- **普通热武器 / 左轮等旧时代枪械**：在系统盲区或海外仍致命（狡啮以此击毙槙岛）。  
- **军用·警用无人机、运输无人机、全息投影套装**：可战斗、欺敌、遮蔽监视器；雨火与异物可干扰全息。  
**典型战力表现（对照乐园图鉴、宁低勿高）：** 市民互殴与街头潜在犯≈徒手/冷兵；训练有素执行官与监视官≈单挑小队、近距火器对抗；强犯罪者与特种佣兵、重武装无人机≈碎砖裂甲、据点级交火；免罪体质者与系统代理人的威胁在于「无法被Dominator合法处决」与情报/制度层面，而非夷城毁国。  
**乐园阶位映射：** 普通人·色相清澈市民·初任监视官/执行官日常执勤·Dominator麻醉模式对普通潜在犯≈一阶；强犯罪者·老练执行官·特种部队/和平破坏者佣兵·重武装无人机·强袭型Dominator交火·免罪体质者近战与物理对抗≈二阶。世界顶点（西比拉系统制度本体、槙岛圣护/鹿矛围桐斗等叙事级威胁）=二阶顶附近的情报·制度对抗，**非**行星级破坏；低阶切入一律规避系统灭口与顶点硬刚，情报优先。

**【地理 · 舞台】**  
**日本本土（西比拉治下）：** 都市被全息投影覆盖外观，街道、建筑表皮甚至衣着都可实时改换「对心理健康有益」的纹理；家用AI投影负责日程与身心管理。扫描器无处不在，珂美莎（太郎/花子）吉祥物投影用于降低市民面对警察时的压力。与之对照的是废弃街区、无投影覆盖的「真实混凝土」、隔离矫正设施与潜在犯专用宿舍——执行官除值勤或监视官陪同外不得自由外出，却可在隔离区内享有奢侈嗜好空间（縢的街机房、征陆的画室等）。  
厚生省本部**诺娜塔（Nona Tower）**为城市地标与系统中枢：外墙彩色投影，对外公开至地下四层，地下五至二十层入口藏于方块墙后；全国心灵数据经塔顶接收处理。第三季暗示二十层以下仍有更深未知层，彩虹桥核心「蓝德罗宾」相关空间与之勾连。公安局刑事课各分队办公室、综合分析室（唐之杜志恩）、局长层与伺服器机房暗盒（SG原型封印处）构成「合法暴力」的垂直结构。  
**关键案件地：** 废弃区与标本事件现场；樱霜学园（王陵璃华子案、霜月美佳青春创伤）；病毒/粮食管理中心（北方稻田与设施，槙岛终局）；地下道（东金与鹿矛围互射）；矫正所与东金医院相关设施；高速运输公司与机场（三期狐狸线）；公安局大楼（被劫持的主舞台）。  
**海外与扩展舞台：** 东南亚联合（SEAUn）水上都市香巴拉浮城——日本出口西比拉与无人武装后的「实验田」；内战、游击队与系统移植腐败是2015剧场版主舞台。外务省行动课活跃于国境、国际通缉与和平破坏者战线。彩虹桥以国家为赌盘，触角伸入国会、道上与开拓者暗杀部队；选举辅佐、教团据点、冷冻睡眠运输等「系统盲点工艺」分散在全国。

**【世界剧情线】**  

**① 第一季：槙岛圣护与「当法律无法裁定」（约2112—2113）**  
常守朱以同期第一名、监视官适性A级身份进入公安局刑事课第一分队（行动代号后期为牧羊犬系）。初任务中，她对即将对潜在犯使用排除模式的执行官**狡啮慎也**开枪麻醉——事后狡啮反而感谢她阻止自己「杀人」，使朱决定留下。分队构成：监视官**宜野座伸元**（严厉划界、戴平光镜掩饰与父亲相似的眼睛）、执行官**征陆智己**（旧时代警探、宜野座之父、猎犬一号）、**縢秀星**（幼年即潜在犯、猎犬四号）、**六合冢弥生**（前乐手、猎犬二号）、分析官**唐之杜志恩**。  
前期案件链具名可循：废弃区诱拐与色相暴走（朱首次质疑「受害者是否该被排除」）；肢解与标本类连续杀人（槙岛幕后诱导实行犯，把人体当「作品」）；全息投影欺敌与黑客式犯罪；樱霜学园**王陵璃华子**案——校内风云人物实为连续杀人犯，学生**霜月美佳**早有察觉却因友人惨死留下创伤。这些案件共同指向幕后主使**槙岛圣护**：免罪体质，可自由压低犯罪系数，主宰者对其无效。他通过诱导他人犯罪、引用古典文学与哲学，试图证明「只听从系统的人生毫无价值」，并逼西比拉露出真面目。标本事件中狡啮昔日部下**佐佐山**被杀，是其由监视官沦为执行官的创伤源；此后狡啮 defer 治疗、系数上升，完成「牧羊犬→猎犬」坠落。  
中盘名场面：槙岛故意在朱面前暴露「测不出的恶」，迫使她正视主宰者逻辑的漏洞；分队在废弃设施与市区追逐中逐步逼近其网络。高潮：槙岛策动市民骚乱，引开警力，直指诺娜塔。朱与狡啮在塔顶捕获槙岛；縢秀星尾随同伙**崔九圣**进入地下，目睹系统真相——**公安局禾生局长**（系统对外人格之一）为灭口，以强制分子分解模式主宰者抹杀縢与崔，连遗体不留。高层对外将縢定为逃犯交第二分队「追查」。禾生私下邀请槙岛加入西比拉，槙岛拒绝并逃脱——系统因迷恋其独特性而未当场处决。  
狡啮认定上层不会公正处刑槙岛，收下征陆所赠安全屋钥匙与左轮，脱离公安局私自追杀。西比拉告知朱真相，要求活捉槙岛；朱以「撤销对狡啮的处决令」为条件合作，性格至此与狡啮的「人必须对扳机负责」同步，却仍选择在法内行动。北方粮食/病毒管理中心：槙岛企图在系统修复窗口期内以变异「五谷神病毒」瘫痪日本粮食自给链、令西比拉统治崩溃。宜野座中陷阱被重物压制；征陆为救子扑向炸药，未及扔远而重伤，宜野座撕开被压的左手、终于喊出「爸爸」。朱负伤追上槙岛仍无法阻止——狡啮在稻田旁用左轮击毙槙岛后失踪，成为国际狙杀要犯。宜野座因丧父与创伤沦为潜在犯，自愿降为执行官并装上义肢、摘下眼镜；朱成为第一分队核心监视官。主题收束于：法律测不出的恶，要靠人还是继续靠系统？

**② 第二季：鹿矛围桐斗与「你是什么颜色？」（约2114）**  
朱领导重组后的第一分队，推理时会点与狡啮同牌香烟、与「幻想中的狡啮」对话。宜野座为执行官（猎犬一号）；新人监视官**霜月美佳**（未成年破格入职、递补宜野座位，极度排斥潜在犯，唯对弥生例外）；执行官**东金朔夜**（猎犬三号，表面模仿狡啮训练与抽烟）、**雏河翔**（红发、药不离手、全息投影专家，称朱为「姐姐」）；弥生为唯一留任一期执行官。表面安定被自称**鹿矛围桐斗**的存在撕开：他能清洗他人色相、用全息与独创禁药制造「作案瞬间系数飙升、事后恢复正常」的棋子，诱捕监视官做人体实验、诱杀警员，甚至推动警民互屠以掌握主宰者使用规律，并以暗号「WC?（What Color?）」质问系统，同时用海外偷渡者冒名顶替政要。  
真相：鹿矛围是321空难后东金财团「复数移植」产物——七人大脑与约184具遗体拼接的集合体，多人格各擅才能；系统不承认集合体心灵，故其长期「不可见」。复仇动机指向西比拉上线过程中的政治牺牲与「地狱的季节」实验。东金朔夜实为免罪体质改造失败的实验体（幼年杀母未遂、东金医院）、史上极高犯罪系数记录者之一（矫正所诊断曾达769，死前更高），恋母并自封「西比拉之子」，企图染黑朱色相；三年内曾导致多名监视官系数破300被处决；假借鹿矛围之名杀害朱的祖母。  
装备与组织事故：强袭型主宰者投入使用，第二分队执行官**须乡徹平**在隔墙无法确认身份时误杀监视官**青柳**；后以同型武器救下酒酒井等，宜野座对其态度由恨转释。霜月私查东金却因触及系统机密被禾生（东金美沙子相关）告知真相，沦为「市民反应样本」，恐惧中向东金朔夜输送情报，事后极度悔恨并更忠于西比拉。宜野座发现长期心理咨询师「向岛陆」实为鹿矛围假扮。终局：西比拉接受朱的提案导入「集团性心灵指数」，使鹿矛围可被测量（系数约400）并被处决；东金朔夜与鹿矛围互射后死亡。朱房间人去楼空式的空虚，暗示她已把自我完全押在「在系统内修正系统」的道路上。

**③ 剧场版（2015）：SEAUn 与系统输出（约2116）**  
日本向海外战乱国出口西比拉与武装无人机。东南亚联合（SEAUn）水上都市香巴拉浮城短暂「和平」，实则政治腐败与镇压并行。恐怖分子袭击日本系统相关设施，其中身影勾连狡啮。朱率第一分队赴当地调查，发现**狡啮慎也**已成为反政府游击队战略顾问，坚信自己在为该国人民自由而战。事件牵涉政权、第三势力与日本输出利益；第一分队拆解腐败链条后，狡啮为促成和平会谈再次背负杀名远走，后接受外务省**花城弗蕾德莉卡**提案，以隐秘行动形式与之合作。主题：西比拉「幸福」输出到无法全覆盖扫描的土地时，会变成何种殖民式暴力；朱与狡啮的重逢确认两人道路分叉却仍互相理解。

**④ Sinners of the System 三部曲（2019）**  
Case.1「罪与罚」：聚焦宜野座、矫正设施与「必要之恶」式管理，霜月亦触及设施内不可告人的实验逻辑。Case.2「First Guardian」：须乡徹平军旅往事——陆军上尉、「原初守卫」、秘密任务「泥布作战」被迫抛弃挚友**大友逸树**；后国防省大楼无人爆破、征陆与青柳请其协查，揭出高层与逸树之妻燐的复仇线，征陆鼓励「若成潜在犯就当执行官」。Case.3「恩仇的彼方」：狡啮海外线与花城收束，完成从佣兵到可被国家工具化的「特殊战力」过渡。三片补完执行官们「为何仍做猎犬」的个人史，并为PROVIDENCE与第三季人事布局铺路。

**⑤ PROVIDENCE 天意（2023 · 第三季前传，约2118初）**  
朱已是总督察级前辈，出席高层会议时接报外国船只事故。事件背后是外务省行动课长期追查的准军事组织**和平破坏者（Peace Breaker）**。狡啮结束佣兵生涯回国协助公安局。案件牵涉外务与公安的权责切割、密约文书与「天意」式国家机密；**杂贺让二**等旧识再度出山，却被多方当作诱饵，最终付出生命代价并留下推动朱进入政府高层的遗言。本片将朱的立场从「分队监视官」推到必须与国家机器正面谈判的位置，并为第三季朱被拘禁的政治后果埋线。

**⑥ 第三季 + FIRST INSPECTOR：彩虹桥与新世代（约2110年代末期）**

初始事件：某商业区全息广告下，一名色相骤浊的市民劫持人质；运输无人机投下主宰者，**常守朱**与**狡啮慎也**到场。契约者因目击或误触扫描器被登记为「现场关系人」，犯罪系数因惊吓短暂上升——必须在「接受即时精神护理」「协助笔录」「被误判为潜在犯苗头」之间做选择，同时听到主宰者语音冷冰冰报出数值。  

开场白建议：「诺娜塔的全息外墙把晚霞滤成心理健康的颜色。运输无人机的螺旋桨声压过人群的惊叫——你看见一名监视官举起漆黑的主宰者，视网膜上跳动着陌生人的犯罪系数。你的手腕也被街头扫描器咬了一下：色相，正在混浊。」  

关键NPC立场：**常守朱**——愿意给人改过机会，对契约者态度公正但会观察色相；**狡啮慎也**——像猎犬一样评估你是否「有用或危险」，不在乎客套；**宜野座伸元**——要求你与执行官保持距离，怀疑一切来历不明者；**征陆智己**——愿用烟和旧警察经验交换情报，提醒你别太信任系统数字；**唐之杜志恩**——可用情报交易换取后台查询，但会记下把柄；**六合冢弥生**——冷静记录，对音乐/艺术话题略松口。  

主线钩子/支线：主线可介入「人质色相失控是否应立即排除」的伦理现场，蝴蝶效应影响朱对狡啮的第一印象；支线＝废弃区走私记忆芯片、珂美莎投影下的街头诈骗、精神护理诊所的过度用药、为色相「漂洗」而找黑市全息服务。  

危险度/规避：中——热武器与麻醉弹可致命，但尚无顶点下场；须规避犯罪系数≥300的排除现场、以及任何试图向你展示「诺娜塔地下真相」的热心人（一期后期灭口逻辑）。  

任务方向/奖励：稳住自身色相、获取合法或半合法ID、学会读懂主宰者语音与扫描礼仪；奖励取向＝精神稳定类药物方、初级格斗/射击训练名额、分析室临时访问权限、一张能让分队记住你的现场功劳。  

**二阶（强犯罪—免罪体质—系统暗部线 · 猎犬与狐狸）**  

切入身份/时点：契约者以「调入第一分队的执行官候选人 / 外务省合作调查员 / 海外归国的有犯罪系数污点的专家」切入，锚定**槙岛圣护现出水面之后**至粮食中心终局前，或**鹿矛围桐斗引发集团恐慌**、或**彩虹桥「狐狸」活跃**的任一高压阶段（任选一条时间锚，勿同时混三季大结局）。  

初始事件：一次对「犯罪系数几乎为零」的嫌疑人的围捕失败——主宰者拒绝对**槙岛圣护**（或同等免罪体质者）解锁；现场转而爆发无人机与强袭火力交火。契约者被要求在「执行物理抓捕」「保护监视官撤离」「擅自追踪免罪目标」中抉择，并收到上层互相矛盾的命令（活捉 / 灭口 / 移交）。  

开场白建议：「扳机扣不下去。不是你手软，是西比拉拒绝承认他有罪。雨刷刮开全息噪点，你看见那名白发男子微笑着引用一本书里的句子，而你的同伴已经在喊：改用实弹——在系统的眼睛瞎掉的地方。」  

关键NPC立场：**槙岛圣护**——欣赏有自主意志者，可能诱导你「选择」犯罪；**狡啮慎也**——目标明确到可以脱队，视契约者为助力或障碍；**常守朱**——坚持活捉与最少杀伤，是与系统谈判的唯一桥梁；**宜野座伸元**——程序正义与私怨撕扯；**霜月美佳**（若锚二期后）——恐惧泄密，可能上报也可能放你一马；**鹿矛围桐斗**（若锚二期）——把你当棋子清洗色相；**花城弗蕾德莉卡**（若锚外务/剧场版）——用国家利益收编你；**慎导灼 / 炯**（若锚三期）——一套白脸黑脸查你的来历。  

主线钩子/支线：主线节点含标本事件收网、诺娜塔骚乱、槙岛逃脱、北方病毒中心、WC恐慌、公安局劫持等——介入任一即可改变谁死谁知真相；支线＝强袭型主宰者误射调查、东金财团病历窃取、SEAUn武装无人机走私线、彩虹桥赌盘的一场「小注」、和平破坏者武器缓存。  

危险度/规避：高～极高——重火器、分解模式、系统强制改写模式均可致死；**西比拉灭口权限与槙岛/鹿矛围级存在 = 二阶顶的情报战，条件性胜利**，禁止宣称「正面打爆诺娜塔就胜利」。规避：在无授权情况下对监视官举枪、单独进入诺娜塔地下、相信「加入系统就不会死」。  

任务方向/奖励：活捉或标记一名免罪体质关联者、保全分队关键成员、取得可与西比拉/外务谈判的筹码；奖励取向＝强袭型装备临时权限、外务省黑签证、免罪体质研究残页、分析室最高级解密一次、与朱或狡啮的长期协作关系（人脉重于神器）。
