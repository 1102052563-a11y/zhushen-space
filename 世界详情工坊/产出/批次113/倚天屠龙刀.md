## 剧情

**【作品来源】**
《倚天屠龙记》（本世界书条目名「倚天屠龙刀」，与书名同指）为金庸（查良镛）所著长篇武侠小说，「射雕三部曲」第三部，承《射雕英雄传》《神雕侠侣》。1961年7月6日起在香港《明报》连载，至1963年9月2日；定本四十回，回目以柏梁体诗句成。有1979年修订版与2005年世纪新修版。文风：元末群雄、正邪对峙、刀剑秘辛与情义抉择交织；重人格、误会与历史余波，少神魔飞升。  
**全本检索**：[搜笔趣阁搜索·倚天屠龙记](https://www.sobqg.com/searchBook.html?keyword=%E5%80%9A%E5%A4%A9%E5%B1%A0%E9%BE%99%E8%AE%B0) → 书页 [yitiantulongji](https://www.sobqg.com/book/yitiantulongji.html)（金庸，已完结，约89.14万字）+ 中文维基情节与角色表 + 英文维基摘要核对。

**【世界定位】**
元末武林与江湖：蒙古朝廷高压、明教举义、六大派与「魔教」世仇、屠龙刀与倚天剑传说搅动号令。主角**张无忌**自冰火岛孤儿，历玄冥寒毒、九阳神功、乾坤大挪移，于光明顶力挽狂澜、就任明教第三十四代教主，后卸任与**赵敏**归隐。一句话：仁厚少年掌教主之位，刀剑之秘终归河山。卡片锚点（本条目仅开**四阶**）：**六大派围攻光明顶 → 张无忌掌教 → 万安寺救派 → 屠狮大会**一线的武侠顶层冲突。

**【世界观 · 力量体系】**
力量来源：内力、心法、兵器与招式传承；无飞升、无灵根系统。成长逻辑：名师指点、秘籍机缘、生死淬炼、内力深厚与招式融会。死亡规则：凡人必死；武功再高亦惧暗器、寒毒、围攻与政治绞杀；无「无限复活」。特殊系统：正邪门派规约、江湖号令传说、刀剑互斫取秘、朝廷高手与武林并行。

本世界无仙侠境界名，以**江湖战力档**描述（对照破坏力，宁低勿高）：
1. **三流好手 / 镖师弟子**：冷兵器格斗，一刀可致命；≈一阶。
2. **二流名门弟子 / 地方高手**：碎砖裂石、单挑小队；≈二阶。
3. **一流高手 / 门派长老级**：裂石断金、乱军取首、气刃伤人；如武当七侠中坚、明教旗使顶尖、峨眉长老；≈三阶。
4. **绝顶高手 / 掌教·法王·九阳乾坤巅峰**：一掌可掀平据点、破城门级建筑群、瞬息制敌；内力近乎「肉身硬抗寻常刀剑与小型火器级冲击」的武侠顶层。代表：**张无忌**（九阳神功圆满 + 乾坤大挪移高阶 + 太极拳剑 + 圣火令武功）、**张三丰**（当世第一）、**阳顶天**（生前教主巅峰）、**灭绝师太**（倚天剑加持）、**谢逊**（七伤拳·狮吼）、**殷天正**、**杨逍**、**范遥**、**成昆**、**周芷若**（九阴白骨爪后期）、少林空字辈与渡字辈高僧等。一城/一派仅数人至十数人。≈**四阶**。

本条目**只开四阶**：不写一～三阶切入；契约者入世即按「可与一流高手周旋、逼近绝顶」的四阶战力落位。世界顶点在本档内为**张三丰**与巅峰**张无忌**（及阳顶天生前、成昆阴谋链）；破坏力仍属武侠「据点/建筑群」级，**勿抬到五阶城市夷平或更高**。  
乐园阶位映射：江湖三流≈一阶；二流≈二阶；一流长老≈三阶；**绝顶高手·九阳乾坤·太极宗师·明教教主巅峰≈四阶**。顶点（张三丰、巅峰张无忌）= 四阶巅峰：**存在·情报优先 / 条件性胜利**，严禁「被封印所以战力为零」。

**【地理 · 舞台】**
- **武当山**：张三丰创派、武当七侠根基；张翠山夫妇自刎、张无忌童年惨剧发生地；太极拳剑初传处。山门松柏、紫霄殿、后山药圃皆可作对话与传功场景；寿宴厅是父母殉情的伦理爆点。
- **光明顶（西域昆仑）**：中土明教总坛；六大派围攻主战场；乾坤大挪移秘籍所在；张无忌就任教主处。栈道、圣火祭坛、秘道、旗营分防，适合写「一夫当关」与「内斗差点先于外敌」。
- **蝴蝶谷**：胡青牛、王难姑医毒圣地；张无忌养伤学医、纪晓芙托孤、杨不悔成长节点。谷中毒虫异花是医线与阴谋线的道具库。
- **冰火岛 / 灵蛇岛**：谢逊隐居与屠龙刀所在；黛绮丝（金花婆婆）巢穴；四女同舟、刀剑互斫的关键海上舞台。孤岛=信息封闭，最适合误会与灭口。
- **大都万安寺**：赵敏囚六大派高手处；张无忌率明教救人、百尺高塔名场面。塔层结构迫使「救人」变成多线渗透而非单挑。
- **少室山 / 少林寺**：金刚伏魔圈、屠狮大会；成昆与谢逊最终清算；黄衫女子现身。戒律与英雄帖提供政治合法性外壳。
- **峨眉山**：倚天剑传承、灭绝师太与周芷若权力线。尼庵规矩与灭魔执念同在。
- **江南 / 王盘山 / 濠州**：夺刀英雄宴、宋青书线、张无忌与周芷若大婚变故。王盘山是夺刀公开化起点；濠州是情与权同时崩盘的舞台。
- **大都汝阳王府 / 蒙古侧**：赵敏出身、朝廷高手（玄冥二老、阿大阿二等）与武林博弈。王府宴会可杀人于谈笑。
势力地盘：六大派各据名山；明教以光明顶为核、五行旗与天鹰教外延；朝廷以大都与诸王府控局；波斯明教总教自海外伸手。四阶契约者活动半径应集中在「名山—大都—海岛」三角，勿写成满地图无阻力瞬移。

**【世界剧情线】**
① **前史·九阳分流与武当峨眉之源（约南宋末～元初）**  
第三次华山论剑后，郭襄寻杨过小龙女不果。昆仑三圣何足道挑战少林，为觉远与少年张君宝的九阳内力所震。觉远坐化前背诵《九阳真经》，无色禅师、郭襄、张君宝各记一部分，分入少林、峨眉、武当一脉内功根基。张君宝后改名**张三丰**，创武当派，内家拳剑自成一宗，与少林并称。郭襄创峨眉，倚天剑为镇派之宝。郭靖、黄蓉于襄阳将危之际，熔铸杨过夫妇所赠玄铁重剑与君子、淑女剑为**屠龙刀**与**倚天剑**，内藏《九阴真经》《武穆遗书》等（新修版为玄铁片指引桃花岛取书），寄望后人复国。江湖遂传：「武林至尊，宝刀屠龙。号令天下，莫敢不从。倚天不出，谁与争锋。」世界状态：传说大于实战，刀剑成为百年争夺的核心符号。

② **夺刀惨剧·张翠山殷素素与冰火岛十年**  
元顺帝至元前后，武当俞岱岩得屠龙刀，途中遭暗算重伤瘫痪（背后牵涉殷素素蚊须针与朝廷金刚门阿三等）。张翠山江南查案，遇天鹰教紫微堂主殷素素；王盘山英雄宴上，金毛狮王**谢逊**夺刀，胁张、殷入海。三人漂至冰火岛十年：张殷成婚，生子**张无忌**，谢逊收为义子。张无忌十岁随父母返中原，谢逊留岛苦思刀中「号令」之秘。  
返中土即遭武林围诘谢逊与刀下落。武当百岁寿宴上，张翠山拒供义兄，又知妻子曾害师兄，羞愤自刎；殷素素殉夫。张无忌中玄冥二老玄冥神掌，寒毒入体。张三丰携其求医少林遭拒，途遇常遇春与孤女周芷若——常送无忌往蝴蝶谷，三丰荐周芷若入峨眉。因果：正邪世仇、夺刀妄念与门派偏见，把孤儿推上绝路。

③ **蝴蝶谷·昆仑寻父·九阳与乾坤**  
胡青牛「见死不救」原则下仍延张无忌性命，并传医术。纪晓芙与杨逍私生女杨不悔；灭绝逼杀纪晓芙，托孤张无忌送杨不悔至西域光明顶寻杨逍。途遇昆仑何太冲夫妇加害、朱长龄朱九真父女骗取谢逊下落。张无忌坠崖入谷，于白猿腹中得完整《九阳真经》，数年练成，尽驱寒毒，内力冠绝同辈。后于明教秘道得**乾坤大挪移**心法，数时辰内练至高阶（原作第六层等描述）。此时其战力已压过六大派与明教绝大多数高手，仅张三丰等极少数可稳压。世界状态：少年「曾阿牛」已是四阶档绝顶胚子。

④ **六大派围攻光明顶（四阶主舞台·名场面）**  
成昆（混元霹雳手，化名圆真等）挑拨正邪，教唆六大派围攻光明顶，欲灭明教。教中阳顶天已死多年（修炼乾坤见妻子与成昆私通走火），杨逍、韦一笑、殷天正等内部分裂，五行旗苦守。张无忌化名介入：以九阳与乾坤化解危局，与周芷若互生情愫（倚天剑刺胸而九阳护体）、与外公殷天正及武当亲人相认，排难解纷于六强之间。战后群雄归心，张无忌被推为**明教第三十四代教主**，约法三章、统合五行旗与天鹰教。名场面：光明顶血战、倚天长剑、青翼蝠王出没、教主继位。主题：仁厚者不欲杀戮，却被迫坐上权力之巅。

⑤ **万安寺·赵敏线与朝廷黑手**  
六大派回程遭元廷高手伏击，囚于大都万安寺；倚天剑落入绍敏郡主**赵敏**之手。张无忌率明教高手救人，百尺高塔、俊貌玉面甘毁伤（范遥苦头陀线揭开）等情节交织。赵敏与张无忌由敌转爱：她机智敢为，为情可叛蒙古身份。成昆与朝廷、武林暗线继续布局。世界状态：武林与朝廷全面绞杀；教主必须同时做统帅与人情机器。

⑥ **灵蛇岛·四女同舟·刀剑互斫**  
张无忌为赵敏所请寻屠龙刀，船至灵蛇岛重逢谢逊与金花婆婆（紫衫龙王**黛绮丝**）。波斯总教使者至，小昭为救众被迫任波斯明教教主，东西永隔。周芷若已继灭绝遗命，于荒岛迷倒众人，盗刀剑互斫，得九阴与武穆等秘（或玄铁片指引），嫁祸赵敏，并重创殷离（蛛儿）。周练成九阴白骨爪与白蟒鞭法，成一流以上高手，性情因爱与使命扭曲。名场面：四女同舟、刀剑齐失、冤蒙不白。

⑦ **屠狮大会·成昆败露·黄衫女子**  
成昆再设屠狮英雄大会，以谢逊为饵，欲令各派自相残杀。张无忌、赵敏、杨逍及**黄衫女子**（神雕侠侣后人）干预。少室山金刚伏魔圈中，谢逊与成昆井下恶战，谢逊挖其双目、断其筋脉，大仇得报。周芷若劣行被黄衫女子以正宗九阴手段制服并当众揭穿。张无忌终明刀剑之秘：复国兵书与武学精义，而非「号令」本身。名场面：屠狮有会、三松郁苍、天下英雄莫能当。

⑧ **大婚生变·武穆归徐达·卸任归隐**  
濠州等处张无忌与周芷若婚约生变，赵敏以谢逊安危等迫其随行；周因爱成恨重创赵敏。张无忌将《武穆遗书》交徐达，助义军败元、克大都。朱元璋等明教出身将领夺兵权、建明朝；张无忌卸教主予杨逍，与赵敏寄迹蒙古（新修版强调不回中土亦可有偶然变数）。谢逊皈依少林；宋青书叛派弑叔终被清理；殷天正等为救谢逊力战而逝。结局：河山易主，教主退隐，情归赵敏；周芷若、小昭、殷离各有归处与遗憾。世界状态：武林传奇落幕，历史进入朱明。

⑨ **分卷细目·光明顶前后（四阶必写细）**  
围攻前：成昆以少林身份与旧怨煽动「灭魔」；明教缺教主、内斗（杨逍与殷天正夺位旧隙、韦一笑寒毒吸人血污名、五散人意气）。围攻中：锐金旗庄铮等死于倚天剑下；灭绝师太刚愎；张无忌不愿见生灵涂炭，以乾坤挪移借力打力、九阳化解寒毒掌与多门绝学。围攻后：教主之位不是奖赏，是枷锁——须安抚正邪、约束五行旗滥杀、应对波斯总教与元廷。契约者介入正确姿势：情报、调停、护送伤员、揭露成昆，而非「一掌灭六派」。

⑩ **分卷细目·屠狮与刀剑真相（四阶必写细）**  
屠狮大会是成昆的终局赌局：谢逊=诱饵，各派=棋子，少林场地=合法性。张无忌须在「救义父」与「不引爆全面武林内战」之间走钢丝。黄衫女子的出现是射雕宇宙的伦理校正：九阴有正邪练法，周芷若邪路必败于正途。刀剑真相落地后，「号令天下」幻觉崩塌——真正可复国的是兵法与人心，不是凶器本身。朱元璋线提醒：武林教主≠开国之君；仁厚者易被枭雄架空。

⑪ **微观政治与可介入事件（仅四阶档）**  
光明顶：旗使补给线、假冒使者、劝降书、成昆眼线、倚天剑去向。万安寺：囚徒名单、赵敏命令链、范遥/苦头陀身份、塔顶救人。灵蛇岛：波斯三使、小昭去留、黛绮丝旧怨、刀剑护送。屠狮：英雄帖真伪、少林戒律与俗家冲突、周芷若灭口、谢逊囚禁点。写战斗：先内力消耗与招式克制，再政治后果；写张无忌：仁厚优柔，非滥杀；写赵敏：敢爱敢恨、谋略压阵；写周芷若：使命与私情撕扯；写张三丰：太极柔克刚，顶点威压不轻易下场。

支线择要：宋青书妒杀莫声谷、陈友谅利用武当内鬼、杨不悔与殷梨亭情债、胡青牛王难姑死于黛绮丝复仇、常遇春寿元代价、徐达得书用兵。时间顺序：前史九阳 → 夺刀冰火岛 → 父母自刎 → 九阳乾坤 → 光明顶掌教 → 万安寺 → 灵蛇岛 → 屠狮大会 → 卸任归隐。任意时点判断：看张无忌是否已练九阳、是否已掌教、刀剑是否已被互斫、成昆是否已败露。

⑫ **光明顶战役链（四阶细写·可观察细节）**  
围攻不是一天打完的口号战。六大派分营：少林以戒律与渡字辈威望压阵，峨眉以倚天剑开路，武当因张无忌血脉而最尴尬，昆仑何太冲夫妇记恨旧怨，华山崆峒各有门人死伤要账。明教侧：锐金旗以精钢兵器硬撼倚天；巨木旗设障；洪水烈火厚土各守一门；天鹰教驰援打乱「正邪分明」叙事。张无忌的关键不是「打赢所有人」，而是三次以上公开调停：一止灭绝滥杀伤兵，二阻杨逍与殷天正内讧升级，三以自身武功作保让六派退半步。乾坤大挪移在战场上的表现是「借力、挪劲、反震」，九阳则是「抗毒、续命、内力深不见底」——写战斗应先写内力对耗与招式相克，再写谁后退谁丢旗。教主推举场面：五散人意气、旗使哭祭亡者、张无忌约法三章（戒滥杀、戒内斗、共抗元），群雄归心带泪也带算计。

⑬ **万安寺与大都棋盘（四阶细写）**  
六派被擒不是单纯武功不敌，而是元廷高手+地形+药迷+政治合法性。赵敏要的是「武林名册与刀剑」，不是当场屠尽。张无忌救人必须同时面对：明教要不要与朝廷全面开战、救了六派是否换来感恩还是猜忌、赵敏个人态度是否可变。范遥苦头陀线揭开后，读者应感到「明教从未放弃潜伏」。塔上救人、毁容、易容、假死、人质交换，都是四阶可介入的具体事件，而不是玄幻秒杀。

⑭ **灵蛇岛至屠狮的因果加粗**  
黛绮丝要屠龙刀雪倚天剑之耻；波斯总教要乾坤与教权；周芷若要完成灭绝「取秘称霸/复国」扭曲遗命；张无忌要义父与和平；赵敏要人与真相。四股力在孤岛碰撞，刀剑互斫是技术解，也是伦理崩坏点——周芷若从此可与绝顶短暂过招，却失去道德高地。屠狮大会上，英雄帖是成昆的请君入瓮：谢逊=饵，少林=台，各派=刀。黄衫女子出场纠正九阴邪路，等于射雕宇宙对「为达目的不择手段」的终审。成昆被废后，武林内战燃料骤减，历史主线交给义军与朱元璋。

⑮ **武当内政与宋青书线（四阶配线）**  
宋远桥稳重掌事，俞莲舟威严居七侠之首，俞岱岩瘫痪后仍传艺，张松溪智谋探敌，殷梨亭情伤剑利，莫声谷耿直送命。宋青书「玉面孟尝」因周芷若与张无忌生妒，窥寝、弑叔、勾陈友谅，终在屠狮场被俞莲舟太极震碎头骨，张三丰清理门户。此线说明：四阶舞台不只光明顶，名门内部也会因情与权崩盘。契约者可作武当客卿见证清理门户，不可煽动灭门。

⑯ **义军政治收束（四阶末段）**  
常遇春义气减寿、徐达得《武穆遗书》用兵如神、韩林儿被利用与沉死、朱元璋诡术逼走教主并收兵权——武林「仁」与皇权「术」最终分道。写结局勿把张无忌写成失败者，而要写成「完成刀剑时代使命后主动离席」；赵敏是情归，也是他拒绝权斗的最后同盟。

**【势力图谱】**
- **明教（中土）**：宗旨「驱除胡虏、光明圣火」；地盘光明顶、五行旗与各地分香；代表阳顶天、张无忌、杨逍、范遥、紫白金青四法王、五散人（冷谦、说不得、张中、彭莹玉、周颠）；与六大派世仇后转联合反元；后期兵权旁落朱元璋。教内结构：教主—左右光明使—四法王—五散人—五行旗—天地风雷门。四阶介入点是「令旗与香主级任务」，不是一上来就夺教主印。
- **天鹰教**：殷天正因与杨逍争位出走所创，后并回明教为天鹰旗；代表殷素素、殷野王、白龟寿等坛主；江南夺刀与王盘山线关键。对武林「正邪」标签极敏感。
- **武当派**：内家正宗；张三丰与七侠（宋远桥、俞莲舟、俞岱岩、张松溪、张翠山、殷梨亭、莫声谷）；对明教由敌意到因无忌血脉而复杂。第三代宋青书是内鬼样板。
- **峨眉派**：倚天剑镇派；灭绝师太、周芷若、丁敏君、纪晓芙；灭魔最烈，亦是刀剑之秘的唯一正统知情链（掌门相传）。
- **少林派**：空字辈、渡字辈高僧；屠狮与金刚伏魔圈主场；内有成昆（圆真）毒瘤，故「名门」不等于「干净」。
- **华山、昆仑、崆峒与丐帮等**：六大派或外围；围攻光明顶参与者；昆仑何太冲夫妇与张无忌有私怨，可作局部冲突源。
- **元廷 / 汝阳王府**：赵敏、玄冥二老、西域金刚门阿大阿二等；以武林高手与权术控江湖，万安寺是其收网示范。
- **波斯明教总教**：圣火令、风云月三使、十二宝树王；要乾坤心法与教义统属，视中土明教为分支。小昭任教主是外交人质式结局。
敌友：正邪 ↔ 明教；明教 ↔ 元廷；成昆两边煽风；刀剑传说令全武林短时利益同盟。四阶叙事里，任何「永久盟友」都可能因英雄帖与刀剑而一夜翻脸。

**【隐藏剧情 · 伏笔】**
- 刀剑「号令天下」是误读：真秘为复国兵书与武学，非凶器统治。
- 成昆是串联阳顶天死亡、谢逊灭门、六大派围攻、屠狮大会的幕后主线黑手。
- 范遥毁容潜入汝阳王府，是明教潜伏与赵敏线的关键伏笔。
- 射雕宇宙互文：郭襄—张三丰、倚天屠龙来历、黄衫女子「终南山后，活死人墓，神雕侠侣，绝迹江湖」。
- 新修版后记：张无忌与赵敏去蒙古后，周芷若/小昭/殷离仍可能因偶然重逢——强调「人为」与心软。
- 朱元璋夺权：教主仁厚无法转化为皇权，历史吞没武林理想。

**【叙事基调 · 雷区】**
画风：元末江湖史诗 + 人情伦理；刀光剑影服务性格与误会，非无敌打怪升级。  
忌：把张无忌写成冷酷杀神或现代嘴炮；忌把明教写成单纯邪教魔头无政治；忌五阶以上毁城爆星；忌无视赵敏蒙古身份与周芷若使命撕扯；忌张三丰动辄下场秒天秒地（他是顶点威压，非常战工具）。  
最早可切入锚点（本条目仅四阶）：**六大派围攻光明顶前后**至**屠狮大会**——绝顶高手云集、教主权位与刀剑阴谋并行。

**【四阶战力观感补述】**
本世界四阶不是仙侠「一掌夷城」，而是武侠顶层：一掌可震塌殿门、掀翻校场旗阵、以内力硬抗寻常刀剑雨；轻功近乎掠屋如飞，近战观者只见残影。张无忌九阳护体后可硬接倚天剑刺胸而不死；乾坤大挪移可将敌劲挪移反震，使多对一变成可周旋。张三丰太极拳「以柔克刚」能化开绝顶刚猛，废宋青书即其门内示范。谢逊狮吼可乱心智，七伤拳伤敌亦伤己。灭绝倚天剑胜在锋锐与刚狠，庄铮等旗使「输在兵器」是原作明确因果。写四阶交手：先比内力深浅与招式克制，再比兵器与场地（光明顶栈道、万安寺塔、少室金刚伏魔圈），最后比人心与站队——胜负常改写一派兴亡，却改不了朱元璋式的历史收割。

王盘山夺刀；冰火岛结义与产子；武当寿宴双自刎；蝴蝶谷学医与纪晓芙托孤；白猿腹中九阳；秘道乾坤；光明顶退六派与就任教主；万安寺救派；灵蛇岛四女同舟与刀剑互斫；小昭赴波斯；屠狮大会与成昆被废；黄衫女子止周芷若；武穆授徐达；濠州逃婚；卸任归隐。任意名场面都可作四阶开局，但须交代「此前张无忌是否已掌教、刀剑是否已失、成昆是否已暴露」。

## 阶位切入点

> 阶位↔江湖战力：一阶≈镖师、帮众与初学弟子；二阶≈地方好手与名门弟子；三阶≈门派长老、旗使与一流高手；四阶≈绝顶高手档（九阳神功+乾坤大挪移巅峰、张三丰太极宗师、明教教主/法王/掌门顶尖对决）。世界顶点（张三丰、巅峰张无忌）为四阶巅峰存在，切入一律「规避硬刚顶点·情报与布局优先 / 条件性胜利」。

**一阶（江湖新手·江南夺刀与武当寿宴线）**

切入身份/时点：契约者以龙门镖局趟子手、王盘山英雄宴的末席武人，或武当山脚药铺学徒身份切入，锚定俞岱岩遇袭至张翠山一家归山的前后。

初始事件：王盘山雾夜，**谢逊**狮吼后满席英雄心神大乱；你在翻覆酒桌下发现一枚蚊须针和一张指向武当的运镖单，既可救一个昏迷的镖师，也可能因此被**殷素素**的人盯上。

开场白建议：「海雾盖住王盘山的灯火，屠龙刀只在火把间亮了一瞬。下一刻，所有人的喊声都变成了野兽似的喘息；你攥着那枚细针，忽然明白今夜活着离岛的人，都要替这把刀背一笔债。」

关键NPC立场：**张翠山**要查俞岱岩惨案而不肯失武当名节；**殷素素**护天鹰教秘密也会救有用之人；**谢逊**只求夺刀与复仇，不辨旁人死活；**俞岱岩**重伤后仍是证据链核心。

主线钩子/支线：追查龙门镖局灭门与俞岱岩伤势；支线为护送伤者上武当、在王盘山辨认天鹰教暗号、替张翠山送出未被截获的信。

危险度/规避：中；狮吼、海难和江湖灭口足以杀死新手，避开与**谢逊**、**殷天正**及玄冥二老正面对峙。

任务方向/奖励：保住人证或还原运刀路线；奖励取向为镖局人脉、基础吐纳、天鹰教或武当名帖、疗伤药与江湖情报。

**二阶（门派弟子·蝴蝶谷与杨不悔护送线）**

切入身份/时点：以武当外门、蝴蝶谷求医者或峨眉俗家弟子身份切入，锚定张无忌中寒毒、在胡青牛处学医并受托护送杨不悔的时期。

初始事件：蝴蝶谷外的病人因胡青牛的规矩被拒于门外，**常遇春**寒毒发作，**纪晓芙**又把幼女杨不悔托给张无忌；你只有一夜决定是帮忙配药、陪同西行，还是把母女行踪卖给峨眉追兵。

开场白建议：「谷口药香压不住血腥。胡青牛说不救外人，纪晓芙却把哭到失声的孩子塞进你怀里；远处峨眉剑鸣渐近，你第一次知道，行医和拔剑都救不了所有人。」

关键NPC立场：**张无忌**寒毒缠身却肯替人担险；**胡青牛**守规矩也惜医术传人；**纪晓芙**宁死不伤杨逍；**灭绝师太**只认师命与灭魔立场；**杨不悔**是各方追索的孩子。

主线钩子/支线：护送杨不悔至光明顶、查纪晓芙死因；支线为采谷中药材、照料常遇春、避开何太冲夫妇对谢逊下落的盘问。

危险度/规避：中高；寒毒、毒虫和门派追杀均可致命，勿介入**灭绝师太**与**杨逍**的正面冲突。

任务方向/奖励：完成托孤与医案，保留纪晓芙遗物；奖励取向为胡青牛医术笔记、峨眉或明教的人情、轻功入门与解毒资源。

**三阶（一流高手·昆仑九阳与光明顶前哨线）**

切入身份/时点：契约者以昆仑山成名散修、五行旗副旗使或六大派随行长老身份切入，锚定张无忌坠谷练成九阳后、六大派尚未总攻光明顶之时。

初始事件：昆仑山道上，**朱九真**以谢逊下落诱你入庄，明教五行旗同时要征用你的马队运送伤药；一封来自少林圆真的密信催促各派提前合围，你要判断它是战机还是成昆埋下的火种。

开场白建议：「雪线以下，朱家庄的灯笼红得像血。主人许诺给你金毛狮王的坐标，山另一侧却有五行旗伤兵等着药车；风里传来少林弟子的号角，所有人都在催你先选一边。」

关键NPC立场：**张无忌**刚得九阳而身份未明；**杨逍**要保光明顶却不信外人；**朱九真**图谋套出谢逊消息；**何太冲**夫妇记恨旧怨；**成昆**借圆真身份煽动围攻。

主线钩子/支线：识破朱家庄骗局、向光明顶递交成昆动向；支线为护旗补给、救治栈道伤员、在六派营地伪造或截获调兵文书。

危险度/规避：高；已进入绝顶人物的战场边缘，避开**张无忌**九阳全开、倚天剑锋芒与六派掌门围杀。

任务方向/奖励：保住补给线或取得幕后证据；奖励取向为一流内功残篇、五行旗令牌、昆仑山路线图与门派信用。

**四阶（绝顶高手 · 光明顶—掌教—屠狮顶层线）**

切入身份/时点：契约者以「成名江湖的一流至绝顶散修 / 明教外围高手 / 六大派客卿长老 / 汝阳王府编外武师」等身份切入，锚定**六大派围攻光明顶前夕至张无忌就任教主、或万安寺救派至屠狮大会**任一高峰时点（须具体到「围攻战鼓已响」或「屠狮英雄帖已发」）。

初始事件：你抵达西域昆仑山道时，六大派旌旗已在光明顶下连营；明教锐金、巨木诸旗死守栈道，峨眉倚天剑寒光划夜。有人塞给你两封矛盾的密信——一封要你「助正灭魔」，一封以谢逊旧部名义求你「保教中妇孺一条生路」。更远处，有人低声吐出一个被反复涂改的名字：**成昆**。你必须在黎明总攻前，选择站队、调停或浑水取刀剑情报。

开场白建议：「昆仑的风割脸。山下是六大派的火把，山上是明教的狼烟。有人说今夜过后魔教将灭；有人说今夜过后武林将再无宁日。你握紧兵刃，听见光明顶方向传来一声狮吼似的长啸——那不是谢逊，那是即将改写江湖座次的年轻人，正要被推上教主之位。」

关键NPC立场：
- **张无忌**——仁厚而战力冠绝同辈，要停战不要屠城；可结盟调停，厌滥杀，感情线易被利用。
- **赵敏**——朝廷郡主，智计压阵；可交易情报与人质，亦可翻脸囚你；爱张无忌后立场剧变。
- **周芷若**——峨眉新核/掌门路，持倚天与灭绝遗命；表面清正，为刀剑之秘可下死手。
- **杨逍**——光明左使，孤傲多疑；认实力与教务，不认空口仁义。
- **谢逊**——刀与复仇的符号；护张无忌，对成昆不死不休。
- **张三丰**——顶点宗师；罕出手，出手则定调；忌辱武当、忌逼死其徒后再辱其孙。
- **成昆**——幕后毒瘤；任何「灭魔大义」背后都可能是他的棋；可合作必背刺。
- **灭绝师太**（若时点在其死前）——刚愎灭魔；谈判成本极高。
- **殷天正 / 范遥 / 韦一笑**——法王与右使线；分别代表强硬救援、潜伏情报、奇袭轻功。
- **朱元璋 / 陈友谅**（稍晚时点）——军权与政治；视你为枪或障碍。

主线钩子/支线：
- 主线钩子：**六大派围攻光明顶**（停战/破间/保旗/推举教主的蝴蝶效应）；**万安寺救六大派**（朝廷与赵敏线）；**屠狮大会**（揭成昆、救谢逊、刀剑真相）。
- 支线：护送杨不悔类人质与伤员；追查圆真=成昆证据链；灵蛇岛刀剑下落与波斯三使；宋青书叛派与武当清理；武穆遗书交徐达前后的军情。

危险度/规避：极高——贴近本世界顶点。须规避与**张三丰**正面争胜、与巅峰**张无忌**死斗、在金刚伏魔圈内硬刚渡字辈+成昆布局；对**灭绝倚天剑**与**周芷若九阴白骨爪**不可轻敌。顶点规则：**四阶巅峰存在·情报优先/条件性胜利**，严禁写成「顶点战力归零」或「轻易秒杀宗师」。

任务方向/奖励：促成光明顶停战或降低屠戮；救出万安寺人质；在屠狮局中保住谢逊并坐实成昆罪状；取得刀剑之秘的「正确理解」（兵书/心法线索而非迷信号令）。奖励取向：九阳/乾坤/太极等心法残篇或指点机缘、明教令旗与人脉、医术与黑玉断续膏级资源、武穆兵法抄本线索、圣火令相关海外情报——**不越级发放毁城级神器**。

补充场景包（仍属四阶，可任选开局细化）：
1. **光明顶总攻夜**：你被派往厚土旗地道送火药或反送假情报；路上撞见**韦一笑**掠过如青影，他问你「人血」与「解药」哪个更急——答错可能被寒冰绵掌点中死穴。目标：活着把信送到**杨逍**案前，并让**张无忌**听见「成昆」二字。
2. **万安寺塔影**：你伪装成元军押粮队混入大都，**赵敏**一纸手令可救你也可杀你；塔上六派掌门目光如刀。目标：在**范遥**暗示下打开一处锁链，换取日后明教人情。
3. **屠狮英雄帖**：你持帖入少室，场中**谢逊**被囚、**成昆**假慈悲、**周芷若**伺机灭口。目标：在黄衫女子现身前后保护证人或偷出成昆罪证，避免全面火并。
4. **濠州大婚变**：鼓乐未停，**赵敏**以义父安危胁婚；你可助逃婚、可助峨眉护驾、可两边卖消息。目标：活口优先，不把四阶战场写成闹剧。

四阶人脉货币：明教令旗、武当名帖、峨眉师命、汝阳王府腰牌、少林斋戒牒——同一物在不同营盘可救命亦可催命。资源货币：内力培元药、伤药、兵器、刀剑下落情报、成昆罪证、武穆抄本线索。写开局务必让契约者在「三刻钟内必须选边」的压力下行动，禁止云游观光式开场。
