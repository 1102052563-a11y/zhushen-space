# 攻壳机动队：SAC Solid State Society
<!--meta lib=主库 tiers=一、二-->

## 剧情

**【作品来源】**
《攻壳机动队 S.A.C. Solid State Society》（SSS，2006，神山健治监督，Production I.G）承接 SAC 与 2nd GIG 之后。卡片锚点：固态社会——孤老人与收养网络下的傀儡师级意识扩张；巴特代理九课与离队素子重逢。

**【世界定位】**
约 2034 年日本：素子离队，巴特代理指挥，德古沙一度入官房系统。少子高龄化下，福利把人登记成节点。固态社会是可管理秩序，不是单纯地名。

**【世界观 · 力量体系】**
电子脑、义体、塔奇克马、政府数据库、社会工程。市民／社工≈一阶；九课与素子交锋≈二阶。仅一、二阶。规则杀＝固态网络核心。

乐园阶位映射：社工／警官／文员≈一阶；九课行动员／素子交锋≈二阶；固态网络核心＝规则杀。宁低勿高。本世界仅一、二阶。

**【地理 · 舞台】**
九课总部、孤老人住宅、收养移送点、官房设施、网络固态节点。空屋药味、摄像头红点、塔奇克马轮胎。

**【世界剧情线】**
①素子离队后九课重组。②孤老人异常死亡与收养数据异常。③政府侧德古沙发现交叉。④傀儡师以福利为壳扩张固态社会。⑤素子介入，与九课冲突合作。⑥拆网或监督化收场，关系重组。⑦日常：家访、审计、巡逻。⑧蝴蝶：可救个体；不可抹少子高龄与网络寄生逻辑。

**【主要人物】**
- **草薙素子**｜疏离｜全身义体网络游猎｜离队→介入｜自由人
- **巴特**｜代理队长压力｜强化义体｜代理→重逢｜九课中心
- **德古沙**｜正派｜推理与政府权限｜官房线再卷入｜良心
- **荒卷大辅**｜老练｜政治庇护｜守护九课
- **石川／斋藤／帕兹／鲍马**｜情报狙击枪火｜队员
- **塔奇克马**｜哲学童真｜思考战车｜伙伴
- **傀儡师（固态侧）**｜网络意志｜系统渗透｜规则敌

**【势力图谱】**
九课；政府福利系统；固态网；地下移送；一般警察社工。

**【贵重物品】**
福利库权限；收养名册；防壁；塔奇克马；监控记录。

**【隐藏剧情 · 伏笔】**
善意可武器化；素子离队是关系不是战力归零；儿童与老人是易吸收节点。

**【大事记时间线】**
离队→代理→异常→显形→再临→收场。

**【叙事基调 · 雷区】**
社会派冷峻；忌普通邪教化；忌低阶硬刚网络神；忌套话。最早切入：空屋家访或异常收养报告。
