# 宝可梦 银
<!--meta lib=主库 tiers=一、二、三、四、五、六、七-->

## 剧情

**【作品来源】**
《宝可梦 银》（日文：ポケットモンスター 銀，英文：Pokémon Silver Version）是 GAME FREAK 开发、任天堂发行的 Game Boy Color 角色扮演游戏，与《宝可梦 金》同为宝可梦主系列**第二世代**首发双版本。日版 1999-11-21 发售，北美 2000-10-15，欧洲 2001-04-06；2017-09-22 登陆 3DS Virtual Console。封面传说宝可梦为**洛奇亚**（金版为凤王）。本档案锚定**银版主线与城都→关都后日谈**，人物名以简体常用译为准（与 52poke/官方中文习惯对齐），不混写动画连续剧支线。搜笔趣阁无收录（非网文），资料以百科与流程向公开页为准。

**【世界定位】**
关都地区「赤红」击败坂木并解散火箭队约**三年后**，城都地区少年/少女自**若叶镇**启程：从博士得到御三家、夺回被红发少年盗走的另一只、挑战八道馆、粉碎火箭队残党、登顶石英高原击败冠军**阿渡**，再渡海进入三年后的关都，集齐十六枚徽章后于**白银山**对决传说训练家**赤红**。银版专属核心：电台长赠**银色之羽**，于**漩涡岛**直面洛奇亚；版本限定野生含六尾、喵喵、芭瓢虫、盔甲鸟等。一句话：在时间、亲密度与传说羽翼下，从若叶镇走到白银山顶的城都银之旅。

**【世界观 · 力量体系】**
**1. 训练家与宝可梦契约**  
战斗为回合制，场上通常一打一。训练家以精灵球收服野生宝可梦，以对战积累经验、学习招式、进化。败北规则：同行全灭则黑视送回最后登记的宝可梦中心，损失部分金钱——**不是原作意义上的训练家死亡**，但叙事可写「被抬回中心、颜面尽失、任务失败」。野生宝可梦战败可逃可收；传说级常具备强逃跑 AI（三圣兽）或极高种族值。

**2. 属性·招式·能力值**  
本世代将旧「特殊」拆为**特攻/特防**；新增**钢、恶**属性以制衡超能力；招式分物理/特殊仍按招式属性表（与后世物理特殊分离不同）。属性相克决定倍伤；携带道具、状态异常（中毒/麻痹/睡眠/冰冻/灼伤）、要害、命中与 PP 构成战术层。亲密度影响进化（波克比→波克基古等）与部分招式威力。

**3. 时间·昼夜·星期**  
机内时钟与现实同步。部分宝可梦只在白天/夜晚出现；捉虫大会限周二、周四、周六；部分事件与 NPC 再战绑定星期。宝可装置（Pokégear）整合地图、电话、收音机、时钟——电话可约训练家再战、收听情报与「宝可梦音乐」影响遇敌活跃度。

**4. 徽章与联盟阶梯**  
城都八道馆徽章解锁 HM 使用权限与服从等级上限；四天王+冠军为联盟试炼；关都再集八章徽章后获准进入白银山。徽章既是战力认证，也是社会通行权。

**5. 培育与球果**  
饲育屋可放两只同蛋群异性产出蛋；阿笔镇**钢铁**用球果制特殊精灵球（诱饵球、重量球、友友球等）。大师球为极稀捕获资源（若叶镇博士线赠送等）。

**6. 战力表现（对照破坏力，宁低勿高）**  
- 新人与道馆见习：摔跤级碰撞、小火苗、电击 pros 级，毁木箱/吓退路人。≈一阶。  
- 中坚训练家、阿速～小茜段道馆：可清扫小路障、群体野生，仍惧专业队与传说。≈二阶。  
- 成熟道馆馆主（松叶、阿四、阿蜜、柳伯等）、火箭队干部级：可控制据点级冲突、掀翻设施。≈三阶。  
- 小椿、四天王、冠军阿渡：地区最强训练家层，联盟级连战。≈三～四阶。  
- 赤红（白银山）：训练家顶点标尺，同行含高等级皮卡丘等。≈四～五阶。  
- 洛奇亚（银版漩涡岛）、凤王（金版铃铛塔）、暴走级红色暴鲤龙事件：海域/风暴/神兽级压迫，地区灾难观感。≈五～六阶。  
- 神话叙事中的「触怒洛奇亚/凤王、烧焦塔三圣兽起源」：接近世界级传说，**七阶观感但任务侧写情报与条件遭遇，禁止低阶硬刚本体**。时拉比仅活动配布，不作常规战力池。

**乐园阶位映射**（对照《阶位战力图鉴》，宁低勿高；本世界覆盖一～七阶）：  
路人/新人训练家/弱小野生 ≈ **一阶**；  
有徽章的旅行训练家/中坚野生 ≈ **二阶**；  
城都道馆馆主中坚～强馆/火箭队干部 ≈ **三阶**；  
四天王/冠军阿渡/关都强馆 ≈ **四阶**；  
赤红/完全体传说下沿（可控遭遇） ≈ **五阶**；  
洛奇亚正式战/红色暴鲤龙事件失控级 ≈ **六阶**；  
神话级神兽全力与「人与宝可梦断交」级灾厄叙事 ≈ **七阶**。  
顶点侧：洛奇亚/凤王/三圣兽/赤红——**超阶边缘不成立**；最高按七阶封顶，低阶一律「规避顶点·情报优先/条件性胜利」。

**【地理 · 舞台】**
**城都地区**（灵感来自日本关西）在关都西侧，以传统神社塔楼与现代都市并存。关键节点按主线推进：

1. **若叶镇**：主角家、**空木博士**研究所、29 号道路入口。启程与失窃案原点。  
2. **吉花市→桔梗市**：新手向城镇；桔梗市有**喇叭芽之塔**（修行僧与大葱鸭相关 HM 前置氛围）、阿速道馆（飞行）。  
3. **桧皮镇**：呆呆兽之井、**钢铁**的工房、阿笔道馆（虫）；火箭队切呆呆兽尾巴事件。  
4. **自然公园 / 34 号道路**：捉虫大会、饲育屋。  
5. **满金市**：城都最大都市——百货、地下通道、自行车店、**播音塔**、磁浮站、小茜道馆（一般）；火箭队占领电台的高潮舞台。  
6. **缘朱市**：烧焦塔（三圣兽苏醒）、铃铛塔（凤王相关，金版核心）、松叶道馆（幽灵）；历史与传说浓度最高。  
7. **浅葱市 / 湛蓝市**：灯塔电龙「电次」、阿蜜道馆（钢）；需从湛蓝取秘密药水救灯塔。阿四道馆（格斗）在湛蓝。  
8. **卡吉镇 / 愤怒之湖**：可疑商店下的火箭队基地、电波逼鲤鱼王进化、**红色暴鲤龙**、**阿渡**联手突入。柳伯道馆（冰）。  
9. **烟墨市**：龙之穴试炼、小椿道馆（龙）；龙之牙与升起徽章。  
10. **漩涡岛**（银版洛奇亚）：潮漩环绕的洞窟群，银色之羽开门钥。  
11. **石英高原（联盟）**：四天王一树/阿桔/希巴/梨花 → 冠军阿渡。  
12. **关都（三年后）**：水流号至枯叶市；红莲因火山被毁、华蓝洞窟坍塌、发电厂零件失窃、常青道馆馆主换为**青绿**、浅红道馆由**阿杏**继承等；终局**白银山**见赤红。

写场景须先判定：玩家处于「城都馆章 / 火箭队线 / 联盟 / 关都后日谈 / 神兽支线」哪一段，再选冲突强度。

**【世界剧情线】**

**① 若叶启程与红发盗贼**  
主角受空木博士委托，前往「宝可梦爷爷」处取蛋。路上自选御三家：**菊草叶 / 火球鼠 / 小锯鳄**之一。红发少年窥探研究所后盗走克制主角选择的那一只。主角首次对决该少年（玩家可命名，通称**阿银**），报案后博士将同行御三家正式交付，鼓励挑战城都道馆与联盟。主题：旅途、责任、与「用强者逻辑压人」的影子对照。

**② 桔梗—桧皮：道馆与火箭队初现**  
击败阿速获蓝色徽章后，博士助手送来宝可梦蛋（孵出波克比）。经 32 号道路、互连洞至桧皮：火箭队残党为卖钱切割呆呆兽尾巴。**钢铁**负伤仍求主角下井清剿。事毕获诱饵球等，并开启球果制球日常。击败阿笔获金色徽章；阿银再战。桧皮森林寻大葱鸭得 HM 居合斩，打通往满金之路。

**③ 满金—缘朱：都市、幽灵与树怪**  
满金击败小茜（平平徽章）后得喷水壶，可唤醒挡路的**树才怪**。缘朱：正辉修时光胶囊、松叶道馆（白色徽章）；烧焦塔触发**雷公 / 炎帝 / 水君**游荡（追猎玩法：保留 HP、反复遭遇削弱）。传说层开始压进日常旅行。

**④ 浅葱—湛蓝：灯塔与格斗**  
阿银透露阿蜜因灯塔电龙生病拒接道馆。主角冲浪经 40/41 号道路至湛蓝，击败阿四（风暴徽章），取秘密药水返回浅葱，电龙康复后阿蜜归馆，钢属性战得矿物徽章。

**⑤ 卡吉—愤怒之湖：电波与红色暴鲤龙**  
卡吉道馆被堵，北上愤怒之湖遭遇**红色暴鲤龙**（异色），战后得红色鳞片。斗篷男**阿渡**出现，揭露火箭队基地：以电波强制鲤鱼王进化。主角与阿渡突入地下基地，关闭机器。后可挑战柳伯（冰河徽章）。

**⑥ 满金播音塔：寻找坂木**  
空木来电：火箭队发出寻老大广播。播音塔被占，假台长实为火箭队；真台长囚于满金地下。阿银再度出现，仍坚持「靠暴力变强」却开始质疑待宝可梦之道。救出台长、肃清火箭队后，银版获**银色之羽**。组织再次实质瓦解，坂木未归。

**⑦ 神兽与烟墨**  
持银色之羽可闯漩涡岛挑战**洛奇亚**（金版对应虹色之羽与铃铛塔凤王）。冰雪小径至烟墨，击败小椿后须入龙之穴取龙之牙才授升起徽章。八章齐全，空木赠大师球。东行经关都 26/27 号道路、冠军之路，阿银终战于出口。

**⑧ 联盟**  
四天王：一树（超能）→ 阿桔（毒，前关都道馆）→ 希巴（格斗）→ 梨花（恶）。冠军阿渡龙队终战。大木博士与主持人祝捷，登录名人堂。

**⑨ 关都后日谈与白银山**  
空木给水流号船票。船上寻走失孙女事件后抵枯叶。关都自由收集八章：小刚、小霞（需先处理华蓝道馆火箭队员与零件线）、马志士、莉佳、阿杏、娜姿、夏伯（道馆迁双子岛）、青绿（在红莲废墟感慨后回常青开馆）。十六章后大木允入白银山，山顶无言**赤红**决战，再映职员表。结局：主角被承认为两地最强训练家之一；火箭队城都再起失败；人与洛奇亚的信任以捕获/对决形式被重新书写（玩家选择）。

**【主要人物】**
- **主角（若叶镇训练家，默认男主称「小田」类可命名｜游戏主人公）**｜性格：由玩家定义，原作演出偏坚韧、重情义、不放弃宝可梦。｜装备·能力：御三家之一、图鉴、宝可装置、逐渐集齐的 HM/徽章/银色之羽。｜弧光：跑腿少年→粉碎火箭队→联盟冠军→白银山挑战者。｜立场：与阿银对立又互相造就；受空木/阿渡指引。  
- **阿银（红发劲敌，可命名）**｜性格：傲慢、信奉强者、早期虐待式训练观。｜能力：盗走的御三家及一路强化的队伍。｜弧光：多次败北后开始反思「宝可梦是工具还是伙伴」。｜立场：对手；对火箭队不屑但曾并行于混乱。  
- **空木博士**｜性格：温和、研究狂、对蛋与培育痴迷。｜能力：科研与御三家管理，非前线战力。｜弧光：从被盗案受害者到托付大师球的师长。｜立场：主角后盾。  
- **阿渡（冠军/龙使）**｜性格：正义、果断、外冷内热。｜能力：快龙等龙队，四天王级以上。｜弧光：联手端火箭队基地→被主角从冠军位掀下。｜立场：盟友与最终联盟墙。  
- **阿速**｜飞行馆主，傲却讲礼貌；首枚徽章门槛。  
- **阿笔**｜虫系天才少年馆主；桧皮秩序象征。  
- **小茜**｜一般系，招式「百万吨重踢」名场面；爱哭但强。  
- **松叶**｜幽灵馆主，追寻传说的见证人。  
- **阿四**｜格斗馆主，与妻子修行；风暴徽章。  
- **阿蜜**｜钢馆主，重责任（灯塔电龙优先于道馆）。  
- **柳伯**｜冰馆主，老人与寒冬之路。  
- **小椿**｜龙馆主，高傲，逼主角入龙之穴试炼。  
- **钢铁**｜球果制球匠人，护呆呆兽的本地义人。  
- **火箭队干部与杂兵（含假台长等）**｜性格：贪婪、怀旧坂木。｜弧光：城都再起→电台失败→再解散。  
- **坂木**｜前火箭队首领，本作品中「缺席的顶点」；广播寻人永不归队。  
- **一树 / 阿桔 / 希巴 / 梨花**｜四天王，属性墙与战术教材。  
- **青绿**｜三年后常青馆主，多属性队伍，前代冠军余韵。  
- **赤红**｜白银山沉默顶点，皮卡丘等高等级同行；训练家神话本身。  
- **洛奇亚**｜海之传说，银版灵魂象征；信任被人类背叛过。  

**【势力图谱】**
- **空木研究所 / 宝可梦联盟体制**：正规成长路径，徽章与四天王认证。  
- **城都—关都道馆网**：地方权威与属性专项；馆主个人风格即地方文化。  
- **火箭队残党**：犯罪企业残余，经费来自虐待宝可梦与电波实验；目标召唤坂木「再伟大」。  
- **缘朱传说信仰圈**：铃铛塔/烧焦塔祭司与市民记忆，对凤王洛奇亚敬畏。  
- **烟墨龙族训练家**：龙之穴试炼，排外而重资质。  
- **满金资本与传媒**：播音塔话语权；被占时全城信息战。  
- **石英联盟**：两地区共有高峰，政治上连接城都与关都。

**【贵重物品】**
- **银色之羽**：银版击溃电台火箭队后台长所赠；漩涡岛见洛奇亚的钥匙级道具。  
- **虹色之羽**：金版对应物（本档银版仅作对照）。  
- **红色鳞片**：打倒红色暴鲤龙后获得，可向多情老人换经验相关奖励。  
- **秘密药水**：湛蓝药房，救浅葱灯塔电龙。  
- **喷水壶**：满金事件后唤醒树才怪。  
- **龙之牙**：龙之穴试炼物，换升起徽章。  
- **大师球**：空木赠，必中捕获；契约者级「一次定乾坤」资源。  
- **球果与特殊精灵球**：钢铁手作，情境加成捕获。  
- **宝可装置扩展卡**：地图/电话/收音机功能解锁。  
- **HM 系列**（居合斩、飞翔、冲浪、怪力、攀瀑等）：移动与战斗双用，徽章权限绑定。

**【隐藏剧情 · 伏笔】**
- 烧焦塔大火与三圣兽「雷、火、雨」复活，解释游荡神兽与人兽信任破裂。  
- 洛奇亚/凤王离开缘朱：因人类恐惧与疏离，羽之道具象征「再次被选中的人」。  
- 关都三年后地理创伤：红莲火山、华蓝洞窟坍塌、超梦失踪，暗示前代终局余波。  
- 阿银成长线：强者哲学破产与伙伴观觉醒，常在火箭队事件与决赛前对照主角。  
- 时拉比：仅活动配布，代表「时间」隐藏页，不进入常规图鉴闭环。  
- 版本限定与时光胶囊：与金版/第一世代通讯才能完图，叙事可写「世界在通讯缆另一头还有半本」。

**【大事记时间线】**
- 约 1500 年前 → 阿露福遗迹文明与未知图腾共处后迁徙（心金魂银补完向）。  
- 约 700 年前 → 缘朱双塔建成，凤王/洛奇亚栖息。  
- 约 150 年前 → 西塔遭雷击焚毁，三圣兽复活后逃走。  
- 关都篇终焉 → 赤红解散火箭队，坂木隐退。  
- 三年后·城都 → 主角启程；火箭队切尾、电波、占电台；联盟易主；银版见洛奇亚。  
- 同旅后日谈 → 水流号赴关都；十六章；白银山决战赤红。


**【世界剧情线·城都馆章细目】**
**若叶—吉花—桔梗**：空木委托→爷爷的蛋→阿银盗宠→报案。桔梗阿速道馆以飞行为门槛，喇叭芽之塔的修行僧用大葱鸭与居合斩哲学对照「弱者也能立身」。此段世界状态：徽章=0，火箭队尚未露脸，传说只在缘朱老人故事里。

**桧皮井与虫章**：呆呆兽被切尾是城都版「资本吃掉信仰」的第一刀。钢铁负伤仍把正义交给少年训练家——叙事上把「本地义人」与「外来旅人」绑成同一条绳。阿笔道馆胜后，阿银再战，队伍已显虐待式强化痕迹。桧皮森林大葱鸭任务换 HM 居合斩，地理上把「镇—林—满金」焊死。

**满金都市政治**：小茜的百万吨重踢是新手墙，也是「外表软、规则硬」的都市隐喻。喷水壶解树才怪，打开缘朱走廊。满金百货、地下、自行车、电台构成消费—物流—舆论三角；后文火箭队占电台，等于占领舆论器官。

**缘朱传说层**：烧焦塔触发三圣兽游荡，把 RPG 的「随机遇敌」升级为「全国追猎」。松叶幽灵馆与铃铛塔信仰并置：一边是对战规则，一边是不可对战的神话记忆。银版玩家此时仍差羽，洛奇亚线被压到电台之后。

**浅葱—湛蓝双城**：灯塔电龙病倒，阿蜜拒开馆，逼主角跨海取药。阿四格斗馆与冲浪权限绑定，海上 40/41 道路是中期体力税。秘密药水归来后矿物徽章到手，钢属性作为「新时代平衡」进入队伍构筑。

**卡吉电波战争**：愤怒之湖红色暴鲤龙是异色+强制进化双重奇观。阿渡以斗篷身份介入，把冠军级战力提前借给中期副本。基地关闭后柳伯冰章才开放——设计上让「叙事高潮」先于「馆章收集」。

**播音塔终局（城都）**：假台长、真台长、地下迷宫、阿银抢功，全部服务「寻找坂木」的空响。银版银色之羽落地，组织再解散。此战之后城都政治恐怖消退，传说支线与烟墨龙章成为收尾。

**烟墨—联盟—关都**：龙之穴试炼强调「资格」；八章后大师球是捕获权的顶点道具。联盟连战是资源管理考试。关都后日谈用地理创伤（红莲、华蓝洞窟、发电厂）续写第一世代因果，白银山赤红把「训练家神话」实体化。

**【地理补述·城镇与功能】**
- 若叶：家、研究所、时间设置教学。  
- 吉花：新手向导、宝可梦中心习惯养成。  
- 桔梗：首馆、塔、黑暗洞穴入口之一。  
- 桧皮：井、制球、虫馆、森林。  
- 满金：经济中心、磁浮、电台、地下。  
- 缘朱：双塔、传说、幽灵馆。  
- 浅葱：港、灯塔、钢馆、船。  
- 湛蓝：药房、格斗馆、冲浪跳板。  
- 卡吉：可疑店、湖、冰馆。  
- 烟墨：龙穴、龙馆、通往冠军之路的东出口。  
- 石英高原：联盟建筑。  
- 关都诸城：三年后差分地图；白银山为终局地标。

**【主要人物·补充档案】**
- **宝可梦爷爷**：蛋事件触发者，温厚，连接「培育世代」主题。  
- **正辉**：电脑存储与时光胶囊维护者，满金赠伊布线相关。  
- **多情老人**：红色鳞片兑换，城都人情网络节点。

- **电台长**：羽之授予者，人与传说的中介。  
- **阿杏**：继承浅红道馆的毒系馆主，关都世代交替象征。  
- **夏伯**：道馆迁双子岛，火系在废土边缘坚持。  
- **小刚 / 小霞 / 马志士 / 莉佳 / 娜姿**：关都馆章墙，各绑前置事件（零件、约会、电力等）。  

**【势力·再述】**
火箭队城都残党经费来源：切尾、电波实验、占电台寻首领。其意识形态是「再伟大」的怀旧法西斯式口号，实操是犯罪经济学。联盟与道馆则是合法暴力与竞技荣誉的双轨。缘朱塔信仰属于第三轨：不可货币化的敬畏。烟墨龙族用试炼维持血统与技术垄断。契约者切入时，三轨冲突比「单纯打怪升级」更重要。

**【贵重物品·流转】**
银色之羽：电台事件→台长→主角→漩涡岛。红色鳞片：湖→主角→多情老人。秘密药水：湛蓝→灯塔→阿蜜归馆。龙之牙：穴→小椿→升起徽章。大师球：空木→主角，对应「一次必中」的伦理选择题（对洛奇亚是否使用）。球果：野外采集→钢铁→情境球，把工匠经济写进日常。

**【隐藏设定·版本差异】**
银版封面与可遇洛奇亚、银色之羽绑定；部分野生（六尾、喵喵、芭瓢虫、盔甲鸟等）与金版对调。日韩与欧美在小小象/熊宝宝限定上曾有差异。时拉比不在常规流程。水晶版另加水君剧情与女主，不在本档展开，但可作平行世界注记。

**【叙事执行要点（AI用）】**
写战斗先写属性相克与徽章权限；写火箭队先写「对宝可梦的工具化」；写传说先写「人类曾令其失望」；写阿银先写「强者逻辑的破产过程」。契约者不是天选大师，除非剧情证明其队伍与选择配得上羽与山。
**【叙事基调 · 雷区】**
画风：王道少年旅行 + 都市犯罪残党 + 神社传说敬畏；战斗活泼但火箭队线有虐待动物阴影。  
忌：把阿银写成本质善良的开局白；忌坂木提前归队收编全局；忌低阶契约者秒杀洛奇亚/赤红；忌忽略昼夜与徽章权限硬飞全图；忌用「神兽被封印战力为零」解释遭遇战。  
最早切入锚点：若叶镇失窃当夜，或呆呆兽之井火箭队行动前两小时。

## 阶位切入点

> 阶位↔原作：一阶≈新人/弱小野生；二阶≈有章旅行者/中坚；三阶≈道馆馆主中坚与火箭队干部；四阶≈四天王/冠军阿渡；五阶≈赤红与可控传说下沿；六阶≈洛奇亚正式战与红色暴鲤龙灾厄级；七阶≈神话神兽全力叙事。世界顶点（洛奇亚/凤王/赤红）低阶一律规避·情报优先；禁止「封印削成凡人」。

**一阶（若叶—桔梗 · 启程与失窃）**

切入身份/时点：契约者以「空木研究所临工/29 号道路见习训练家/邮差」切入，锚定御三家交付前后、红发少年踩点当晚。

初始事件：研究所警报与破碎玻璃；一只御三家被劫走。警官要笔录，空木要人沿 29 号道路追「红发」。你口袋里只有一只租借的小球与伤药。

开场白建议：「若叶镇的风总像在催人出门。你赶到栅栏外时，只听见少年甩下一句『弱者没资格拥有宝可梦』——草丛里还留着被踩折的菊草叶气味。」

关键NPC立场：**空木博士**——焦急、可给任务与图鉴权限；**主角**——刚拿到御三家，警惕却缺人手；**阿银**——鄙视弱者，可能回头再踩一脚；**若叶镇妈妈们**——提供床铺与基础补给。

主线钩子/支线：追盗与第一次野外遇敌；支线＝帮宝可梦中心送药、在吉花市学捕获教学、电话登号。

危险度/规避：低～中——败只丢钱；须规避阿银全力与任何传说游荡。

任务方向/奖励：活过第一夜并取回线索；奖励取向＝普通精灵球、伤药、低阶同行委托、空木好感。

**二阶（桧皮—满金 · 尾巴与都市）**

切入身份/时点：以「钢铁学徒/满金百货保安/自然公园裁判助理」切入，锚定呆呆兽之井或小茜道馆开放周。

初始事件：井下传来哀鸣与火箭队喊价「美味尾巴」；或满金地下通道有人倒卖来路不明的球果。

开场白建议：「桧皮的井口飘着湿咸味。钢铁老人摔在台阶上还骂贼：谁砍呆呆兽的尾巴，谁就和这镇子为敌——而你手里的诱饵球，刚好还没扔过。」

关键NPC立场：**钢铁**——正义、可制球；**阿笔**——少年馆主，可见证；**火箭队杂兵**——人多欺人；**小茜**——胜后爱哭，但馆内规则严。

主线钩子/支线：清井与制球日常；支线＝捉虫大会作弊案、饲育屋蛋归属纠纷。

危险度/规避：中——火箭队围殴与道馆连战；规避干部真身与阿渡级战力。

任务方向/奖励：赶走切尾组并保全呆呆兽；奖励＝球果、特殊球、第二枚徽章相关通行权。

**三阶（卡吉—电台 · 电波与假台长）**

切入身份/时点：以「卡吉侦察员/播音塔音响师/愤怒之湖渔民」切入，锚定红色暴鲤龙出现当周或电台被占当日。

初始事件：湖面被异色暴鲤龙掀船；斗篷男人要你带路进可疑商店地下室。同步：满金电台突然只播「坂木老大请回家」。

开场白建议：「收音机里的寻人启事像诅咒。你在卡吉镇推开商店后门时，电波嗡嗡发烫——鲤鱼王在深处被迫进化，而有人把这叫做『复兴经费』。」

关键NPC立场：**阿渡**——强力临时同盟，不容背叛；**阿银**——要独自灭火箭队，可能抢人头；**假台长**——说谎与人质；**真台长**——解锁银色之羽的关键。

主线钩子/支线：关停电波机器 / 救出台长；支线＝红色鳞片流转、地下假货球市场。

危险度/规避：高——组织围杀与狂暴水型；须规避洛奇亚本体与联盟级问责。

任务方向/奖励：解散城都火箭队指挥链；奖励＝银色之羽线索、中坚装备、电台话语权碎片。

**四阶（联盟 · 石英高原连战）**

切入身份/时点：以「冠军之路向导/四天王见习观察员/联盟记者」切入，锚定主角冲馆前夜或阿银出口决战之后。

初始事件：胜利之路出口硝烟未散，四天王通道已排号；有人出钱买「梨花的恶系克制表」。

开场白建议：「石英高原的门不认金钱，只认八枚徽章。你听见一树的汤匙轻响、阿桔的毒囊开合——再往后是阿渡的龙吟，像在问：你是来观光的，还是来改写名人堂的？」

关键NPC立场：**一树**——优雅施压；**阿桔**——毒控与经验老辣；**希巴**——正面击溃；**梨花**——恶系心理战；**阿渡**——最终墙，可敬可惧。

主线钩子/支线：连战规则下的替补与道具；支线＝阿银终战观战、大木电台连线采访。

危险度/规避：高——连战资源耗尽即出局；规避白银山赤红。

任务方向/奖励：见证或协助新冠军诞生；奖励＝联盟级药、招式机器、成名机会。

**五阶（关都后日谈 · 十六章与赤红）**

切入身份/时点：以「水流号乘务/发电厂技术员/白银山巡山人」切入，锚定零件失窃或十六章集齐后。

初始事件：发电厂机件失踪导致磁浮停运；或白银山脚中心护士警告「再往上是死人的沉默」。

开场白建议：「红莲只剩焦土与中心灯光。青绿站在火山灰里说『这里曾经有个镇子』——而白银山顶，那个三年前解散火箭队的男人，仍一句话也不肯多说。」

关键NPC立场：**青绿**——馆主与前辈压迫感；**小霞/马志士等关都馆主**——各有前置事件；**赤红**——无言顶点，战力标尺；**大木博士**——放行白银山的钥匙人。

主线钩子/支线：零件与华蓝线、十六章；支线＝模仿少女磁浮站、破坏基因传闻。

危险度/规避：极高——赤红队碾压级；只可条件挑战/观战/逃命。

任务方向/奖励：完成关都秩序修复或取回观战记录；奖励＝高等级经验、稀有进化石、地区通行特许。

**六阶（漩涡岛 · 洛奇亚）**

切入身份/时点：以「持银色之羽的深潜向导/漩涡岛遗迹学者/银版特使」切入，锚定电台事件后、羽入手时。

初始事件：潮漩张开一瞬，海平面下有巨大阴影呼吸；电台长拜托「让它重新相信人类」。

开场白建议：「银色之羽在掌心发冷。漩涡岛的岩壁上刻满逃走的人们的抓痕——洛奇亚不是宠兽，是一场愿意停在你面前的风暴。」

关键NPC立场：**洛奇亚**——可对决可捕获，怒则海难；**真台长**——托付信任；**阿渡**——可提供龙系建议但未必同行；**火箭队残党**——可能来夺羽。

主线钩子/支线：羽之试炼与正式战；支线＝三圣兽追猎交叉、铃铛塔对照（金版世界线情报）。

危险度/规避：极高·贴近顶点——海域级灾难；**条件性胜利**：对决/捕获/建立契约，严禁写「削成普通飞行型」。

任务方向/奖励：完成银版灵魂事件；奖励＝传说遭遇权、海洋 HM 精通、地区级声望。

**七阶（神话回响 · 双塔与信任）**

切入身份/时点：以「缘朱祭司代理人/烧焦塔守夜人/记录三圣兽的史官」切入，锚定传说叙事层（可与主线并行的「深读」时点）。

初始事件：雷雨夜烧焦塔残木发光，三道兽影冲天；市民恐慌，有人主张再造兵器「防神兽」。

开场白建议：「一百五十年前的雨停在老人的故事里，却落在你今晚的肩上。凤王与洛奇亚离开，不是因为弱，是因为失望——你要做的不是屠神，是让失望停住。」
