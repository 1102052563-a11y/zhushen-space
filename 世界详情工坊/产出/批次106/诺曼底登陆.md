# 诺曼底登陆
<!--meta lib=主库 tiers=四-->

## 剧情

**【作品来源】**
历史事件「诺曼底登陆」（Normandy landings / D-Day），代号「海王星行动」（Operation Neptune），属盟军「霸王行动」（Operation Overlord）的登陆日阶段。时间：1944年6月6日；舞台：法国诺曼底海岸约80公里滩头。媒介：真实二战史实（跨影视/游戏/纪实再演绎时仍以史实编年与公开战史为准）。关联：霸王行动全战役（1944.6.6–8.30）、大西洋壁垒、坚忍欺敌、法莱斯包围、解放巴黎。文风：史诗残酷、工业战争、后勤与天气同为敌人；无超凡魔法，战力=组织火力/装甲/空海权。本档案服务AI常青读取——信息密度优先，忠于公开战史，查不到写不详，严禁编造从未发生的终局与虚构神器。

**【世界定位】**
1944年西欧第二战场开启日：英美加等盟军对纳粹德国占领下的法国诺曼底实施史上最大规模两栖登陆。没有「小说主角」单线升级——视点是整条盟军登陆链与德军西线防御链的对抗。一句话：工业时代的生死赌局，契约者是滩头上的变数，不是免死金牌；炮弹与潮汐不讲情义。

**【世界观 · 力量体系】**
力量本源：现代军事体系——步兵火力、装甲、火炮、航空、海军舰炮、工兵破障、通讯与后勤；无灵气修真、无法印魔法。死亡规则：一发步枪/炮弹/地雷即可致命；军官与精英部队同样会死；「复活」不存在，只有后送医院与替补兵员。特殊系统：欺敌与情报（坚忍行动）、气象窗口、两栖工程（桑葚港 Mulberry、冥王星油管 PLUTO）、特种坦克「趣味」（Hobart's Funnies）、空降失散后的小单位重组。

战力阶梯（按破坏力与战场角色，宁低勿高）：
1. **平民·民兵·后勤杂役**：徒手或冷兵器级自保，炮火下≈无战力。乐园观感≈一阶初期。
2. **普通步兵·海岸静态师新兵**：步枪机枪、单兵可杀数人；被机枪/火炮覆盖即灭。≈一阶中后期。
3. **老兵·工兵破障组·空降散兵重组小队**：能清雷破障、端碉堡射孔、巷战夺屋。≈二阶。
4. **连级精锐·突击队·游骑兵·坦克排**：可撕开一段防线、打穿街区级据点。≈三阶。
5. **营团级合成突击·装甲连·舰炮/重炮支援节点·王牌飞行员单机对地**：摧毁海岸炮台群、掀平村镇据点、短时间压制一片海滩。≈**四阶**（本世界主切入档：质变分水岭=「火力与组织」的精英Boss层，一城/一滩仅数个关键节点）。
6. **师军级战役集群·战略轰炸编队·舰队炮火全覆盖**：地区级灾难，夷平城区、封锁半岛。≈五~六阶观感（战役层，非个人）。
7. **集团军群/最高统帅部决策层**：改写战线走向，本身不直接「砍人」。≈六~七阶「制度顶点」。

乐园阶位映射：普通步兵/新兵≈一阶；老兵·破障·空降散兵≈二阶；精锐连排·游骑兵·坦克排≈三阶；营团合成·装甲连·舰炮节点·关键滩头指挥官层≈**四阶**；师军/战略轰炸/舰队≈五~六阶；艾森豪威尔—希特勒级决策权柄≈制度顶点（情报优先，禁止写成「个人无敌」）。世界真实顶点=纳粹德国国家战争机器与盟军联合远征军最高司令部的战略博弈，非单人超阶；低阶切入一律规避总部硬刚·情报优先。本世界**仅开放四阶切入**，写厚滩头与内陆推进中的精英节点。

**【地理 · 舞台】**
- **英格兰南部集结区**（朴次茅斯、南安普敦、怀特岛水域）：登陆舰编组、密封营地、假电台与假装备展示区；艾森豪威尔在此下达「Go」。
- **英吉利海峡**：登陆舰队与空降航线；风力、浪高、云底高决定D日可否执行；扫雷舰先开航道。
- **诺曼底海岸五滩**（西→东）：**犹他滩**（Utah，美军VII军，科唐坦东岸）、**奥马哈滩**（Omaha，美军V军，维耶维尔—科勒维尔崖岸，伤亡最重）、**黄金滩**（Gold，英军XXX军，阿尔罗芒什方向）、**朱诺滩**（Juno，加拿大第3步兵师，科尔斯河河口）、**宝剑滩**（Sword，英军I军，乌伊斯特勒昂至乌伊斯特勒昂东）。
- **奥克角**（Pointe du Hoc）：奥马哈与犹他之间的悬崖角，美军第2游骑兵营攀崖目标。
- **科唐坦半岛 / 瑟堡**（Cherbourg）：西翼深目标，深水港与半岛根部切断线。
- **卡昂**（Caen）与奥恩河—卡昂运河：东翼交通枢纽，D日计划目标，实际成月余绞肉机。
- **贝叶、圣洛、卡朗唐、维莱博卡日、法莱斯、尚布瓦、262高地**：内陆连片、树篱战、突破与口袋关键点。
- **大西洋壁垒**：混凝土碉堡（WN抵抗点）、雷区、铁蒺藜、比利时门、捷克刺猬、岸炮与雷区纵深。
- **德军纵深**：树篱（bocage）、预备装甲（第21装甲师、党卫军装甲师等）、B集团军群与西线总司令部驻地链路。

AI写场景先定：哪条滩、涨潮还是落潮、空降区还是内陆树篱、是否有舰炮/空中支援。底层冲突=清障、抢滩、夺出口公路；中层=连片滩头、夺港、卡昂僵局；高层=欺敌真假、希特勒装甲释放权、突破与法莱斯口袋。法国平民村庄同时是掩护、障碍与道德雷区——乱射会制造抵抗组织与盟军政治危机。

**【世界剧情线】**

① **前史·第二战场压力与计划成型（1941–1943）**  
苏德战争爆发后，斯大林持续要求英美在西欧开辟第二战场。1942年美苏曾公开谈及当年开第二战场，丘吉尔说服罗斯福推迟——兵力与两栖能力不足，大西洋航线仍在流血。1942年迪耶普突袭惨败：加拿大与英军突击队硬冲设防港口，海滩变成屠场，证明「直接夺取大型港口」在当时技术与火力下不可行，间接推动「先抢海滩再建人工港」的路线。1943年德黑兰会议后，跨海峡入侵成为确定战略；盟军最高司令部（SHAEF）组建，**德怀特·D·艾森豪威尔**任最高司令，**伯纳德·蒙哥马利**任地面部队总指挥（21集团军群），**伯特伦·拉姆齐**海军、**特拉福德·利-马洛里**空军、**亚瑟·泰德**副最高司令。登陆日代号D-Day，两栖突击本身称海王星（Neptune），整个诺曼底战役称霸王（Overlord）。因果：世界状态从「是否登陆」变为「何时何地如何欺敌」；任何契约者切入前史阶段，接触的是计划室、演练场与假情报网，不是已经开火的五滩。

② **欺敌与准备·坚忍行动与大西洋壁垒（1943–1944春）**  
盟军实施大规模军事欺骗「保镖行动」（Bodyguard），核心子项「坚忍」（Fortitude）：虚构巴顿指挥的「美国第1集团军群」，在英格兰东南布置假坦克、假电台与假调动，示形加来海峡，诱使德军精锐与岸防加强滞留加来方向。运输计划（Transport Plan）轰炸铁路枢纽、桥梁与编组站，削弱德军增援速度，亦造成法国平民伤亡——解放的道德账单提前开出。特种装备：两栖DD坦克、扫雷坦克、喷火坦克、架桥车（霍巴特趣味系列）；预制人工港「桑葚」；海底油管「冥王星」。德军侧：**阿道夫·希特勒**任命**埃尔温·隆美尔**指挥B集团军群，加强大西洋壁垒——雷场、滩头障碍、岸炮、注水淹没区；西线总司令**格尔德·冯·伦德施泰特**与隆美尔在「装甲部署靠岸还是纵深」上争执；装甲预备队调动常需希特勒批准，形成「看见登陆却调不动坦克」的制度悲剧。1944年4月「老虎演习」中德军鱼雷艇袭击盟军演练船队，暴露保密与护航风险，死伤惨重却被新闻管制压住。名场面：艾森豪威尔在恶劣天气中权衡——6月5日延期，6月6日窗口若再误则推迟两周以上（月相、潮汐、日出时刻三重约束）。因果：D日天气与欺敌成败成为战役第一变量；契约者若在此阶段介入，最高价值任务是护航、反侦、气象与口令安全，而非个人杀敌数。

③ **D日前夜·空降与轰炸（6月5日夜–6日凌晨）**  
约24,000名美英加空降兵在午夜后着陆。美军第82、第101空降师（任务奥尔巴尼、波士顿、芝加哥、底特律等）投向科唐坦半岛内侧，保护犹他滩侧翼、夺渡口与公路；风、云与高射炮导致极度失散，许多伞兵落在沼泽与洪水区，武器束丢失，小分队以玩具蟋蟀信号器与口令互认。英军第6空降师（东加行动）以滑翔机奇袭夺卡昂运河桥（后称「飞马桥」）与奥恩河桥，攻击梅维尔炮台——**约翰·霍华德**连的演练式精确突击成为东翼开门名场面。海军与空军对海岸实施轰击；部分轰炸因云层与提前释放弹着偏离，奥马哈火力点未被充分压制。德军最初判断混乱：空降被部分参谋视为佯动或牵制；隆美尔本人当时不在诺曼底（回德国为妻庆生），前线请示链条更长。阶段结局：桥梁与部分要地被空降兵控制，但建制混乱、友军误击与迷路普遍；犹他侧翼「门锁」半开，奥马哈侧翼几乎未减负。因果：滩头登陆前，内陆「门锁」已被撬开一半——契约者若空降切入，第一小时主题是「活着集合」而非「攻克师部」。

④ **D日·五海滩（6月6日约06:30起）**  
目标80公里海岸分五区同时抢滩。  
- **犹他滩**：美军第4步兵师为主；强风使登陆艇偏东，意外落在德军较弱段；伤亡相对较低；与空降部队逐步衔接。  
- **奥马哈滩**：美军第1、第29步兵师；高崖、完整的第352步兵师防御、障碍与交叉火力；首波几乎被钉死在潮线；工兵在弹雨中清障；军官与军士长带领小群士兵爬出「杀伤漏斗」，沿山脊撕开缺口；当日盟军伤亡最重滩头。  
- **奥克角**：第2游骑兵营攀崖，发现大口径炮已移走，后在内陆找到并摧毁；孤立坚守待援。  
- **黄金滩**：英军第50诺森伯兰师等；趣味坦克协助清障；夺取阿尔罗芒什一带，为桑葚港预留位置；与内陆村镇巷战。  
- **朱诺滩**：加拿大第3步兵师；潮汐与障碍导致部分艇触雷；推进较深，接近卡昂方向目标但仍未与宝剑完全按计划会合。  
- **宝剑滩**：英军第3步兵师；登陆后向卡昂推进，遭第21装甲师等反击阻滞；第6空降在东翼死守桥头。  

当日统计量级（公开战史常用区间）：盟军登陆约156,000人、海军人员约195,700；盟军伤亡约10,000+（确认死亡约4,414）；德军伤亡约4,000–9,000。五滩均建立滩头，但卡昂、圣洛、贝叶等关键城镇多未按计划陷落；朱诺与黄金当日相连，五滩完全连片约至6月12日。阶段结局：登陆成功≠战役结束，只是「把一只脚踩进西欧」。

④-b **五滩战术细谱（供AI按滩写场景）**  
**犹他滩**：落点偏东反而避开较密雷区与部分炮位；第4步兵师在**西奥多·罗斯福 Jr.**等军官调整下迅速重组，优先夺堤道与内陆接点，与101/82空降残部握手。危险来自淹没区、运河与零星88炮，而非奥马哈式全面屠杀。  
**奥马哈滩**：A–D段沙滩狭窄，高崖上WN（抵抗点）交叉射击；首波工兵装备落水，爆炸筒与班加罗尔鱼雷短缺；存活者沿「死者堆」找死角，由军士与尉官自发组织向上；午后缺口扩大，傍晚勉强站稳。写奥马哈必须写：潮线、铁丝网、提道、 fortification 射孔、友军尸体形成的掩体。  
**奥克角**：绳梯与钩爪攀岩，崖顶遭遇零星德军与已转移的炮位；游骑兵内陆搜炮、摧毁后遭反击，弹药水粮见底，靠信号与后续登陆衔接。  
**黄金滩**：趣味坦克开路，阿尔罗芒什海岸为桑葚港A选址；内陆莱阿梅尔等强点需逐屋清除；与朱诺侧翼靠拢是当日后勤优先项。  
**朱诺滩**：加拿大部队在科尔斯河与村庄间推进最深之一，但交通堵塞与残存炮火拖慢装甲；平民夹杂在交火带，政治象征（自治领军队重返欧陆）与战术目标并重。  
**宝剑滩**：向卡昂公路推进时遭遇第21装甲师局部反击，英军与空降桥头堡之间走廊一度危险；「当日取卡昂」计划破产，改为建立桥头并等待增援。

⑤ **巩固与纵深·瑟堡与卡昂僵局（6月中–7月）**  
西翼美军（**奥马尔·布莱德雷**指挥第1集团军）向科唐坦推进：先切断半岛根部，再北上瑟堡。德军奉命死守港口，爆破船坞、码头与航道。6月底瑟堡陷落，盟军得到「名义上的深水港」，实际清障与扫雷耗费数周——证明「夺城≠立刻有后勤」。树篱（bocage）使每块田都是微型要塞：坦克易被侧射，步兵需爆破手与火焰喷射器配合。东翼蒙哥马利反复对卡昂发动「鲈鱼」「埃普索姆」「查恩伍德」「古德伍德」等行动，城区化为废墟，德军装甲被大量吸住——客观牵制了德军机动预备队。德军**弗里德里希·多尔曼**第7集团军、**埃里希·马克斯**等军级指挥在混乱中组织反击；党卫军与国防军部队均投入。名场面：维莱博卡日战斗中德军装甲短促突击重创英军纵队（细节史家有争，但「树篱伏击」成为东翼符号）。6月19–22日英吉利海峡大风暴摧毁桑葚港相当部分构件，卸载量暴跌，盟军一度靠海滩直接卸货硬撑——天气作为「第二敌人」登场。因果：盟军用物资与火力换空间；德军用精锐装甲填线，战略预备被钉死在诺曼底。

⑤-b **抵抗运动、吉德堡与后方战**  
法国抵抗组织在D日前后按BBC暗语与盟军计划破坏铁路、电话与公路标志；德军报复性清剿与人质政策使乡村政治极度危险。吉德堡（Jedburgh）三人小组空投协调游击与正规军。契约者若接触抵抗线：信任靠口令与成果，不靠自我介绍；叛卖与双重间谍真实存在。后方战不提供「无敌民兵」，只提供情报、向导、破坏窗口与道德风险。

⑥ **突破·眼镜蛇与法莱斯（7月底–8月）**  
7月25日美军发动「眼镜蛇行动」（Cobra）：密集轰炸在圣洛以西撕开缺口，但也发生友军误炸（含高级军官伤亡），提醒「火力优势可反噬」。突破后摩托化与装甲涌入开阔地，德军防线由线变点。希特勒下令「吕蒂希」反攻莫尔坦方向，企图切断美军走廊，反而把装甲送进更危险的突出部。加军「总计」「温顺」与波兰第1装甲师等从北向南压，美军从南向上卷，形成**法莱斯口袋**。8月中下旬德军残部经尚布瓦、262高地（蒙奥梅尔）等地血战突围：人员部分逃出，坦克火炮大多遗弃。8月25日巴黎解放（勒克莱尔第2装甲师与抵抗运动入城，艾森豪威尔一度倾向绕过巴黎却因政治与起义改变节奏）。霸王行动阶段通常计至8月30日左右盟军进抵塞纳河一线。阶段结局：西欧第二战场彻底站稳；德军西线由「拒敌于滩」变为「战略溃退」。

⑥-b **战役损失与平民代价（终局量级）**

**四阶（营团合成·装甲连·舰炮节点 · D日精英线）**

切入身份/时点：契约者以「盟军联合特遣参谋观察员 / 加强连级特遣队长 / 海军岸轰校射军官 / 德军岸防营指挥链上的作战参谋」之一切入，锚定**1944年6月5日夜–6月6日黄昏**（空降失散重组→五滩抢滩→滩头出口争夺）或**6月中–7月**（瑟堡走廊 / 卡昂树篱 / 眼镜蛇前夜）任一高压窗口；默认推荐奥马哈「杀伤漏斗」突破小时或飞马桥—宝剑侧翼连接战。亦可以「游骑兵加强排指挥官」身份落在奥克角崖下，或「空降重组小队核心」落在圣梅尔埃格利斯外围。

初始事件：你所在的波次刚踏进齐腰海浪，前一辆DD坦克只露出炮塔沉没；左侧工兵班在比利时门下被机枪点名，右侧无线电只剩噪音。连长死亡，地图被血浸透——你必须在三分钟内决定：是沿潮线横滚寻找「死者间隙」爬上堤道，还是呼叫海上战列舰/巡洋舰炮火对一处还在喷火的混凝土碉堡校射（误报坐标会炸到友军）。与此同时，法裔向导坚称东侧出口有雷，军士长却要强冲西侧冲沟；一名浑身湿透的空降兵跌进你的弹坑，口令对了一半——他可能是失散友军，也可能是渗透者。

开场白建议：「1944年6月6日，诺曼底。海水是铁青的，天空被曳光弹划开。你的靴底踩到的不是沙滩，是谁的背囊和谁的手臂。扩音器早哑了，只有军士在吼：前进——待在这里只有死。远处峭壁上，大西洋壁垒的射孔一明一灭，像有人在眨眼睛。你的手表进了水，但潮汐不会等你。」

关键NPC立场：**德怀特·D·艾森豪威尔**——远在朴次茅斯/南威克，听不到你的枪声，但你的成败会变成他桌上的箭头；对「失踪的特遣」只看结果。**奥马尔·布莱德雷**——奥马哈若崩溃会考虑撤离该滩，你的突破报告比勋章重要。**伯纳德·蒙哥马利**——东翼要卡昂，视西翼惨胜为全局可接受代价，征用你部协防侧翼时不问你死活。**埃尔温·隆美尔**——若你走德军线，他要的是「把敌人赶下海」的即时报告；他不在场时，你的装甲请求会卡在希特勒的批准链。**诺曼·科塔**——奥马哈现场推人出滩的军官，认勇敢与主动，厌恶瘫在潮线装死的人。**约翰·霍华德**——飞马桥方向，要的是守桥与通联，可结盟不可抢功。**马克斯韦尔·泰勒 / 马修·李奇微**——空降区要的是集合号与渡口，对「来路不明的加强火力」先核查口令再握手。**伯特伦·拉姆齐**——若你走海军校射线，他要的是弹着修正与友军识别，不接受「大概炸那边」。

主线钩子/支线：主线=奥马哈或犹他—空降区连线、奥克角孤立游骑兵救援、飞马桥—宝剑会师、卡朗唐走廊、瑟堡港区爆破与反爆破、眼镜蛇前夜标示友军位置防误炸。支线①：缴获德军岸防图与雷场图送师部；支线②：保护桑葚港工兵免遭残存炮台点名；支线③：追查坚忍欺敌是否已在当面德军师部被识破（情报战，非砍杀）；支线④：在树篱地带为后续装甲标出「不可进」的反坦克死角。

危险度/规避：极高（贴近战役层）。必须规避：完整装甲师的正面冲击、未校射的舰炮覆盖区、己方战略轰炸地毯（眼镜蛇开幕误炸真实存在）、希特勒/SHAEF级决策场（禁止四阶角色「单挑元首或艾克」）。制度顶点=存在·情报优先/条件性胜利；德军强在体系与火力，弱在空海权与指挥迟滞——**不是**「总部失去战力」，而是决策链慢于滩头节奏。

任务方向/奖励：打通/守住一条滩头出口公路、摧毁或致盲一座岸防火力点、护送关键校射观察组、夺取完好的雷达/密码残件、协助空降部队夺回渡口。奖励取向：四阶可用的制式与缴获——改进型自动武器与瞄准镜、工兵爆破与探雷套件、校射电台与烟标、缴获的MG42与反坦克武器、战术地图与口令表、短期舰炮/空中优先呼叫权（一次性）、盟军或抵抗组织的人脉徽章、桑葚港/油料卸载优先票；禁止发放「改变整个霸王计划」的战略权柄或个人毁灭师团的外挂。
