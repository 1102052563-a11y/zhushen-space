# 战锤40K：星际战士
<!--meta lib=主库 tiers=一、二、三、四、五、六、七-->

## 剧情

**【作品来源】**
《战锤40,000》（Warhammer 40,000，通称 40K）由英国 Games Workshop 于 1987 年推出，是跨媒介科幻微型战棋与设定宇宙；规则书迭代至当代版本，叙事由 Black Library 小说、各军团/战团 Codex、战役书、动画与电子游戏共同延展。本档案聚焦 **阿德普图斯·阿斯塔特（Adeptus Astartes）**——人类帝国的「死亡天使」星际战士：从帝皇造基因原体与二十军团、大远征统一银河、荷鲁斯异端（Horus Heresy）兄弟相残，到异端后基里曼《圣典》拆军团为千人战团、M41 末原铸（Primaris）与不屈远征。文风标签：**grimdark**——「在遥远未来的残酷黑暗中，唯有战争」。非战棋平衡表，而写叙事战力、基因政治与修士军魂。

**【世界定位】**
银河系约四万年纪元。人类帝国以百万世界与神皇信仰维系统一，却被混沌、绿皮、虫族、死灵、灵族、钛帝国等内外撕扯。本世界的「主角」不是单一修仙逆袭者，而是 **阿斯塔特制度本身**：凡人少年被选、被改造、被洗脑成超人僧侣，以爆弹与动力甲为帝国续命。卡片一句话：**帝皇的天使在星海斩首异端与异形，却永远不够多；基因种子比生命更贵，忠诚比荣耀更难证。** 契约者是变数，不是免死金牌。

**【世界观 · 力量体系】**
**力量本源（多轨并行，以基因战士为主轴）**：
1. **凡人军事**：体能、训练、编制、火力、装甲与后勤。**星界军（Astra Militarum）** 以数量与炮火换命；行星防卫军（PDF）更弱一档。
2. **基因改造战士**：**阿斯塔特** 经基因种子（gene-seed）培养的多道改造手术（初生约十九器官；原铸另有额外植入），单兵≈凡人小队至排级突击尖刀。**禁军（Adeptus Custodes）** 更高一档，不属本档案主战场，仅作气压。
3. **灵能（Psyker）**：连通 **亚空间（Warp）**；战团图书管理员可杀敌、可通信，亦可被混沌侵蚀。帝国以黑船收缴灵能者、以审判庭监管。
4. **机械神教科技**：STC 残片、动力甲维护、雷鹰/降落舱、终结者甲、轨道武器；创新常被视作异端，复制旧式更安全。
5. **混沌恩赐**：叛变阿斯塔特（混沌星际战士）获变异、恶魔契约与亚空间具现；战力暴涨伴随理智与形体崩坏。
6. **基因原体**：帝皇以自身基因模板制造的二十（后记十八已知）半神将领；军团是其「稀释的儿子」。

**死亡与复活**：凡人一枪一刀可死。阿斯塔特可被爆弹、能量武器、熔解枪、泰坦级火力击杀；药剂师须从阵亡者身上回收 **祖腺（Progenoid）** 以续基因种子。恶魔被放逐回亚空间≠永久抹杀。**帝皇** 坐黄金王座维持星炬，既非「已死无战力」也非可随意下场的 NPC。**混沌邪神** 与 **帝皇** 属 **超阶巅峰：存在·情报优先/条件性胜利**，严禁「被封印所以战力为零」。

**改造与编制（阿斯塔特专属规则）**：
- **招募**：优先青春期男性（各战团仪式不同：极限战士考行政与战术，太空野狼在芬里斯冰原猎杀，圣血天使重血脉与艺术，暗黑天使重骑士隐秘）。成人改造极难、失败率极高；候选者经筛选后进入数年乃至十余年的植入周期，期间死亡、精神崩溃、器官排斥皆常见。
- **基因种子（Gene-seed）铁律**：基因种子是帝皇以原体 DNA 为模板炼成的「活体工业」，无法在工厂流水线量产；它随修士体内生长，成熟后由 **药剂师（Apothecary）** 从 **祖腺（Progenoid Glands）** 外科回收，再培养下一代植入器官。每名阿斯塔特通常有两枚祖腺（颈与胸），分别在服役不同年限后成熟；阵亡战场上「先救种子、再收遗体」是药剂师的第一伦理。帝国向各战团征收 **基因种子十一税**，送交火星与泰拉基因库检测纯度——纯度高的血统（如极限战士、暗黑天使系）获允大量派生子战团；缺陷严重者被监视、限制创立，甚至面临解散威胁。混沌最常劫的硬通货不是金币，而是未腐化的祖腺罐。
- **关键植入物（约十九道，初生阿斯塔特）**：第二心脏、骨化腺体（强化骨骼）、双肌强化、血造器（高效造血）、拉曼氏器官（速凝止血）、卡塔列普塞安节点（半醒睡眠）、前胃（毒素过滤与吞食异物）、多肺、视神经强化、黑甲壳（对接动力甲神经接口）、祖腺等。器官缺失或退化=战团基因缺陷的生理根源：例如部分战团丧失嗅觉/味觉相关腺体，部分无法生成稳定的贝彻氏腺体（酸碱唾液）。**原铸（Primaris）** 在此基础上增加额外植入与体型强化，整体更稳、更高大。
- **动力甲与武装**：动力甲是「第二骨骼」；黑甲壳未成熟者只能穿侦察甲。终结者甲=精英重装，多用于第一连与登舰突击。武器主轴：爆弹枪（反冲大到凡人难控）、链锯剑、动力拳/爪、熔解枪、等离子、重爆弹与各类遗物刃。技术修士（Techmarine）与机械神教共同维护圣物甲——甲上常刻历代阵亡者之名。
- **编制史**：大远征期为 **军团（Legiones Astartes）**，万人至十余万级，一人可掌半个远征方向；异端后 **罗保特·基里曼** 著《阿斯塔特圣典（Codex Astartes）》，推动 **第二次创立**，将忠诚军团拆为约 **千人战团（Chapter）**，典型十连制（第一连老兵与终结者、第二至五连战斗连、第六至九连预备/支援、第十连侦察与新兵），并规定战术教条、招募节奏、基因监测与「不得再集结军团级私兵」。部分战团拒圣典或大幅变体：**太空野狼** 保留大连/狼群结构，**黑圣堂** 以远征舰队式永不结束的十字军编制运作。
- **原铸阿斯塔特（Primaris）**：M41 末 **贝利萨里乌斯·考尔** 以万年工程与原体级基因资料（含桑普利姆斯·波图姆等相关遗产）炼成，经 **极限创立（Ultima Founding）** 批量出厂并补充旧战团；初生可经「卢比肯原铸」升级，风险极高。原铸与初生之间的摩擦（荣誉、编制、谁更「像父亲」）是 M42 正文高频政治线。
- **圣典战团 vs 非圣典**：圣典合规者以 **极限战士** 为样板——行政清晰、战术模块化、子战团极多。非圣典者保留文化与基因特权，帝国容忍其存在是因为「前线太缺天使」，而非真正信任。

**战力阶梯（按破坏力表现；本世界映射宁低勿高，切入封顶七阶）**：
1. **巢都底层民 / 新兵 / PDF 民兵**：徒手或劣质自动枪；毁木门、巷战求生。乐园≈**一阶**。
2. **星界军老兵 / 帮派精锐 / 钢军团步兵骨干**：能端据点、扛轻火力；热武器日常。≈**二阶**。
3. **风暴部队 / 士官 / 政委小队 / 星界军精锐连**：裂装甲、屠小队、乱军取首。≈**二~三阶**。
4. **阿斯塔特新兵满编 / 标准战术小队骨干**：毁建筑群、肉身抗常规火力；一城仅见数人至数十。≈**四阶**（本世界超凡质变分水岭）。
5. **资深修士 / 中士 / 战术小队指挥 / 英雄级凡人（极罕见）**：街区压制、斩首小目标。≈**四~五阶**。
6. **终结者 / 连长 / 图书管理员·牧师精英 / 战争帮主中坚**：夷平街区、领域感=战术压制。≈**五~六阶**。
7. **战团长 / 第一连王牌 / 战役级舰队协同核心**：覆盖战区、移山式火力与指挥。≈**六阶**。
8. **原体侧翼 / 大恶魔具现边缘 / 泰坦亲王会战节点**：毁国级至大陆级战役改写。≈**七阶**（本世界切入上限）。
9. **完整原体巅峰 / 恶魔原体 / 舰队会战与行星清洗核心**：行星表面命运。≈**七阶~超阶**（剧情可标，**不写八九阶切入**）。
10. **帝皇（王座态权能）/ 混沌邪神本源 / 跨星系灾难**：文明~恒星系级叙事压迫。≈**超阶：存在·情报优先/条件性胜利**。

**乐园阶位映射（本世界专用·与切入点一致）**：凡人≈**一阶**；星界军/精锐凡人卫军≈**二~三阶**；阿斯塔特单兵~小队≈**四~五阶**；终结者/连长/战团中坚≈**五~六阶**；战团长与战役指挥≈**六阶**；原体/恶魔原体常态与战役具现≈**七阶**；**巅峰原体、恶魔原体满开、帝皇、混沌四神、完整亚空间神战**＝**超阶：存在·情报优先/条件性胜利**。峰值可在剧情中如实标注超阶；**切入点只写一至七阶**，低阶一律规避顶点硬刚。

**【地理 · 舞台】**
- **泰拉（Terra）**：帝国首都，黄金王座与帝皇所在；行政、审判、舰队枢纽；异端终局围城主舞台；帝国之拳等仍视其为圣地。
- **火星（Mars）**：机械神教圣世界；动力甲与泰坦圣所；异端期火星分裂战曾陷暗机械。
- **乌尔塔玛 / 极限星域（Ultramar）**：**罗保特·基里曼** 治下模范五百世界；行政征服与《圣典》样板；卡尔斯等战场曾被怀言者血洗。
- **巴尔（Baal）**：**圣血天使** 母星；辐射废土与血脉缺陷共生；**但丁** 坐镇千年。
- **芬里斯（Fenris）**：**太空野狼** 的死亡世界；狼牙堡（The Fang）；大连与狼群文化。
- **卡利班残片「岩石」（The Rock）**：**暗黑天使** 机动要塞修道院；猎捕「堕落者」的隐秘中枢。
- **夜曲星（Nocturne）**：**火蜥蜴** 母星；火山与保护欲军魂。
- **伊斯特万 / 伊斯塔万星系（Isstvan）**：异端公开化与空降场大屠杀圣地；忠诚三军团近乎建制毁灭处。
- **达文（Davin）**：荷鲁斯被安纳塞姆之剑重创、蛇神会堂堕落的关键星域。
- **普罗斯佩罗（Prospero）**：千子家园；狼群焚星之地。
- **恐惧之眼 / 大裂隙（Eye of Terror / Great Rift）**：叛徒遁入与 M41 末银河撕裂后的航道灾难。
- **阿米吉多顿、卡迪亚（已碎）、地狱之触** 等：凡人绞肉机与阿斯塔特空降改写战役的对照舞台。
感官：动力甲伺服嗡鸣、圣油与机油、爆弹雷鸣、基因实验室防腐液、战团歌咏与「为帝皇」对撞、降落舱撕开大气层的白热。

**【世界剧情线】**
① **前史·统一战争与天使制造（约 M30）**  
纷争时代后，**人类帝皇** 以雷霆军团统一泰拉。雷霆武士残暴难控，帝皇转向更精致的方案：先造 **二十基因原体** 为半神将领，再以其基因模板稀释量产 **星际战士军团**。混沌诸神将尚在培养舱的原体播散到银河各星——帝国史书将此记为「失窃」。泰拉基因实验室与征服月球基因教团后，第一批军团在 **找不到父亲** 的情况下已开始为大远征流血。此阶段世界状态：帝国尚「年轻」，启蒙与傲慢并存；「阿斯塔特」尚未被神化成教堂壁画，而是征服机器。

② **大远征·寻回父亲与军团军魂（约 798.M30—005.M31）**  
帝皇发动 **大远征**：军团尖刀、帝国军、机械神教、泰坦军团、禁军与寂静修女协同，把离散人类世界纳入 **帝国真理**（公开禁神、理性、服从帝皇）。失落的原体被逐一寻回并交给对应军团：  
- **莱恩·艾尔·庄森** 收暗黑天使（第一军团）骑士纪律；  
- **罗格·多恩** 筑帝国之拳的城与盾；  
- **黎曼·鲁斯** 为狼群与「帝皇的刽子手」；  
- **圣吉列斯** 以天使容貌统圣血天使；  
- **罗保特·基里曼** 以行政与五百世界示范「征服后如何治理」；  
- **沃坎、科拉克斯、察合台·可汗、费鲁斯·马努斯** 等各塑军魂；  
- 叛徒侧前史：**荷鲁斯·卢佩卡尔** 的影月苍狼最快最宠、**洛加·奥瑞利安** 的怀言者最爱建庙、**安格隆** 的吞世者最嗜血、**马格努斯** 的千子最通灵能……  
**乌兰诺** 大捷后，帝皇立荷鲁斯为 **战帅**，把远征指挥权交给爱子，自己返回泰拉秘密推进 **网道工程**（试图摆脱亚空间航行依赖），并抬高凡人官僚的 **泰拉议会**——诸多原体感到被抛弃。洛加在被征服世界宣扬「帝皇即神」，帝皇当众摧毁其完美之城、强迫全军团跪悔；信仰粉碎的洛加转向亚空间「真神」，以牧师与 **战士会堂** 渗透诸军团。因果：大远征的辉煌同时埋下 **信仰真空、兄弟嫉恨、军团过大** 三枚雷。

③ **达文之伤·战帅堕落（约 005.M31）**  
在达文卫星，荷鲁斯被附身的前下属以 **艾瑞巴斯** 窃得的 **安纳塞姆之剑（Anathame）** 刺伤，伤口拒愈。蛇神会堂仪式中，艾瑞巴斯以幻象展示「未来帝国将神化帝皇、抹去荷鲁斯、阿斯塔特沦为看门人」的虚假预言。**马格努斯** 曾灵能闯入劝阻，荷鲁斯仍选择混沌的「治愈」与权柄——交易核心：把帝皇交给诸神，银河归战帅。影月苍狼更名 **荷鲁斯之子**，系统策反兄弟、舰队、泰坦与帝国军。名场面逻辑：**最完美的儿子先死于幻象，后死于泰拉**。

④ **普罗斯佩罗之焚与伊斯塔万公开化**  
马格努斯为警告帝皇荷鲁斯已叛，强行击穿皇宫防护，摧毁网道工程门户，恶魔涌入视界之城。帝皇震怒，命鲁斯缉拿马格努斯；荷鲁斯篡改命令为格杀。太空野狼与寂静修女 **焚毁普罗斯佩罗**，马格努斯与千子投入奸奇。帝皇被网道内战拖住，泰拉防务交 **玛尔卡多** 与 **罗格·多恩**。  
**伊斯塔万 III**：荷鲁斯将可能顽固忠诚的连队编入「平叛第一波」，以 **生命吞噬者病毒炸弹** 与全球火风暴清洗地表——约十二亿凡人液化，忠诚阿斯塔特部分钻入掩体幸存。**安格隆** 违令降陆肉搏，把「干净清洗」拖成绞肉。**纳撒尼尔·加罗** 夺 **艾森斯坦号** 突围告警；**加维尔·洛肯、索尔·塔维茨** 等成为忠诚派地表传奇。  
**伊斯塔万 V 空降场大屠杀**：钢铁之手、火蜥蜴、暗鸦近卫第一波强攻，第二波「增援」——**康拉德·科兹、佩图拉博、洛加、阿尔法瑞斯**——调转枪口。**费鲁斯·马努斯** 被 **富尔格里姆** 斩首；**沃坎** 被俘受折磨（永生体反复「死」）；**科拉克斯** 携残部逃。三忠诚军团近乎建制毁灭。阶段意义：大叛乱不可逆；「叛徒之手挥出时，带着一整支军团的力量。」

⑤ **银河内战扩散（约 006—014.M31）**  
荷鲁斯推行黑暗归顺。**卡尔斯** 上怀言者突袭极限战士；**阴影远征** 血洗乌尔塔玛方向；**西格纳斯** 上圣吉列斯抗恶魔伏击；**斯拉马斯** 上莱恩与科兹互撕；**察合台·可汗** 辨伪后杀回忠诚侧。破碎军团游击、黑盾刮尽徽章只认私仇。**贝塔-加蒙** 等「泰坦之门」级会战消耗双方。忠诚主力被风暴与伪令分散，泰拉防线主要靠帝国之拳、圣血天使、白色疤痕等先期到位者。此阶段世界状态：没有「前线」，只有谁的星语先到、谁的基因种子先被抢光。

⑥ **泰拉围城与黄金王座（014.M31）**  
叛徒舰队轰炸泰拉，恶魔与暗机械、叛变泰坦围攻皇宫。**罗格·多恩** 总指挥，**圣吉列斯**、**察合台·可汗** 等死守宫墙与外围。荷鲁斯最终降下 **复仇之魂** 护盾，诱帝皇登舰做「父子终局」。**圣吉列斯** 先入舰死于荷鲁斯之手——此死铸成圣血天使万年黑怒回响；帝皇与混沌加持的战帅对决，**击杀荷鲁斯**（灵魂亦被抹除级叙事），自身亦近乎陨落，被安放于 **黄金王座**：以每日千名灵能者献祭维持星炬与帝国残躯，并封堵网道裂口。叛徒溃入恐惧之眼。因果：帝国从「启蒙远征」坠入万年神权、恐惧与停滞；阿斯塔特从「征服之矛」变成「永恒守夜人」；「为帝皇」从口号变成国教与临终遗言。

⑦ **大清洗与圣典改革·战团时代**  
**大清洗（Great Scouring）** 追杀叛徒至亚空间边缘。**罗保特·基里曼** 以帝国之主/军事统帅身份推动 **《阿斯塔特圣典》** 与 **第二次创立**：忠诚军团拆为约千人战团，防再出军团级叛乱；舰队、凡人军与阿斯塔特权力拆分。帝国之拳初拒后服；太空野狼、黑圣堂等保留异制。基因种子按血统隔离库存，叛徒种子封存而非尽毁（因帝国仍缺战士，也因技术不可再生）。此后万年：**一千以上战团** 散布银河，各自招募、远征、隐秘与缺陷——圣血的红渴与黑怒、野狼的狼化、暗黑天使的堕落者猎捕、火蜥蜴的保护欲、极限战士的圣典教条、钢铁之手的厌肉……阿斯塔特成为帝国宣传中的天使与官僚眼中的不可控核武器。

⑧ **漫长的 M31—M41：天使不够用**  
黑十字军反复叩关；绿皮 WAAAGH！周期性吞星；虫族从边缘涌入；死灵苏醒。战团以连级、小队级投入凡人无法想象的斩首与守关：**但丁** 守巴尔与外域千年；**马纽斯·卡尔加** 为极限战士战团长样板；**洛根·格里姆纳** 一类野狼领主在芬里斯与虚空猎杀；黑圣堂永不结束的远征；灰骑士专司恶魔（编制隐秘，禁止写成「路边NPC」）。凡人战役是「配额与炮灰」；阿斯塔特降临后是「战团长的时间表」——来则改写战役，走则留下审判风险与「谁配同行」的政治。基因种子十一税、纯度丑闻、战团灭绝与紧急创立在此时代循环。

⑨ **M41 末—M42：大裂隙、原铸与原体回归**  
**艾泽凯尔·阿巴顿** 第十三次黑十字军碎裂卡迪亚，**大裂隙** 撕开银河，帝国航道与行政一分为二。**罗保特·基里曼** 被复活，任帝国摄政/帝国之主，发动 **不屈远征**，释放 **贝利萨里乌斯·考尔** 的 **原铸阿斯塔特** 与极限创立新战团，试图以行政与新血续命。**莱恩·艾尔·庄森** 亦在近年正典中重返，活跃于裂隙另一侧的帝国虚无域，猎堕与旧骑士逻辑并行。恶魔原体 **安格隆、莫塔里安、马格努斯、富尔格里姆** 等仍为战役级灾难源。此阶段世界状态：天使更多了，敌人也更多了；「忠诚」仍要用血与祖腺证明；初生与原铸的摩擦、圣典修订、双帝域星语延迟，是正文政治主线。

⑩ **圣典时代的著名战团军魂（契约者最常撞上的「天使品牌」）**  
- **极限战士（Ultramarines）**：**罗保特·基里曼** 血统；罗马/希腊式行政与战术教科书；乌尔塔玛五百世界为样板；子战团数量极多；**马纽斯·卡尔加** 为当代战团长样板。圣典教条既是力量也是枷锁——「按书打」能赢大半仗，也会在书外局面里僵硬。  
- **圣血天使（Blood Angels）**：**圣吉列斯** 血统；艺术、高贵与近战突击并存；**红渴（Red Thirst）** 与 **黑怒（Black Rage）** 是血脉诅咒——前者是嗜血冲动，后者会让修士在幻觉中重演原体死于荷鲁斯之手的终局。**但丁** 以千年疲惫维持巴尔与外域的「天使形象」。  
- **太空野狼（Space Wolves）**：**黎曼·鲁斯** 血统；芬里斯死亡世界文化；**犬齿螺旋（Canis Helix）** 相关基因使部分修士面临狼化风险；拒圣典千人制，以狼群/大连作战；猎杀、宴饮与「帝皇的刽子手」记忆并存。  
- **暗黑天使（Dark Angels）**：**莱恩·艾尔·庄森** 血统；骑士隐秘；核心秘密是 **堕落者（The Fallen）**——异端期卡利班分裂后叛逃的兄弟；内环组织为猎捕堕落者可对帝国其他机构撒谎、灭口、调兵。**岩石** 要塞修道院是移动的耻辱档案馆。  
- **帝国之拳（Imperial Fists）**：**罗格·多恩** 血统；筑城、攻坚、泰拉遗产；子嗣含 **黑圣堂（Black Templars）** 等永不结束的远征战团。  
- **火蜥蜴（Salamanders）**：**沃坎** 血统；夜曲星火山文化；相对更重视保护凡人（仍非滥好人）；锻造与火焰美学。  
- **暗鸦近卫（Raven Guard）**：**科拉克斯** 血统；阴影、渗透与残缺基因后的重建创伤。  
- **白色疤痕（White Scars）**：**察合台·可汗** 血统；速度、骑乘与草原式军魂。  
- **钢铁之手（Iron Hands）**：**费鲁斯·马努斯** 血统；厌恶肉体软弱、崇尚机械增强；伊斯塔万创伤铸成冷硬教条。  
- **其他高频编制**：**灰骑士** 专司恶魔（隐秘、基因极纯、不随处可见）；**死亡守望** 为跨战团猎杀小队编制；各创立期子战团以母战团缺陷与教条的「稀释版/极端版」出现。

⑪ **混沌星际战士（Heretic Astartes）镜像**  
叛徒军团在泰拉围城失败后遁入 **恐惧之眼** 等亚空间裂口，在扭曲时间中存活万年。他们不再是统一的「第二帝国」，而是：  
- **黑军团（Black Legion）**：**艾泽凯尔·阿巴顿** 统合荷鲁斯残部与多源战帮，拒绝自称「第二荷鲁斯」，却以十三次 **黑十字军** 对帝国执行慢性处决；碎裂卡迪亚是其当代高光。  
- **侍神军团**：**吞世者** 归恐虐（安格隆为恶魔原体）、**死亡守卫** 归纳垢（莫塔里安）、**千子** 归奸奇（马格努斯）、**帝皇之子** 归色孽（富尔格里姆）——恩赐带来变异、神力与理智/形体崩坏。  
- **未专一侍神者**：**钢铁勇士** 的围城与苦役、**午夜领主** 的恐惧统治、**怀言者** 的混沌神学、**阿尔法军团** 的渗透与双头谜题——各自以战争帮、海盗王国或邪教帝国形态存在。  
- **与忠诚侧的永恒交换**：基因种子、奴隶、舰船、圣物、情报；战场上「兄弟相残」既是战术也是神学。契约者切勿把混沌战帮写成无脑丧尸——他们有战略、有交易、有对「假帝皇」的漫长恨意。

⑫ **微观可介入日常（正文高频）**  
除史诗战役外：基因种子十一税与纯度检测、新兵试炼死亡率、药剂师回收祖腺的优先级、动力甲维修排队、战团内部连队荣誉决斗、对凡人「是否值得救」的教条辩论、审判庭对「接触混沌」修士的灭口、混沌战帮劫掠基因种子、原铸与初生之间的摩擦、与星界军协同时的坐标与人盾政治、圣典派与非圣典派在同一战场上的指挥权争执、暗黑天使「临时征用」友军后失联灭口。契约者优先卷入这些，而不是一上来单挑原体。

⑬ **蝴蝶效应边界**  
可改：某次新兵试炼成败、某小队斩首是否完成、某战团是否及时空降某巢、某药剂师是否抢回祖腺、某堕落者是否被暗黑天使灭口灭到你、某黑十字军支线补给线是否被截。不可无代价改写：荷鲁斯异端终局与帝皇上王座、圣典拆军团的历史功能、圣吉列斯之死对圣血天使缺陷的因果、大裂隙级天文政治、帝皇/邪神本尊、第二与第十一军团被抹除的禁忌本身。局部改写会以「战团视你为干扰斩杀、审判庭灭你整团、混沌 entice 你所在的连、暗黑天使把你写进堕落者名单」回流。

**【主要人物】**
- **人类帝皇（The Emperor of Mankind）**｜性格：叙事中近乎神话的统一者与神权核心；生前冷酷理性，王座上不可对话。｜装备·能力：无匹灵能、星炬、帝国信仰锚点、造物主级基因工程。｜人物弧光：统一泰拉→大远征→击败荷鲁斯→黄金王座万年。｜立场关系：**超阶巅峰：存在·情报优先/条件性胜利**；低阶不可见、不可杀、不可当任务 BOSS 硬刚。
- **荷鲁斯·卢佩卡尔（Horus Lupercal）**｜性格：魅力战帅→堕落弑父；曾是最被宠爱的儿子。｜装备·能力：战帅级原体、混沌加持后的亚空间权柄；旗舰复仇之魂。｜人物弧光：影月苍狼→战帅→荷鲁斯之子→泰拉被帝皇击杀。｜立场关系：异端之源；高阶回忆/亚空间幻象可遇，非日常。
- **罗保特·基里曼（Roboute Guilliman）**｜性格：立法者、行政天才、对帝国现状的厌恶与责任感。｜装备·能力：原体武技与战略、《圣典》作者、复活后帝国摄政与不屈远征统帅。｜人物弧光：大远征行政征服→异端期乌尔塔玛苦战→圣典改革→陨落沉眠→M41 末复活。｜立场关系：忠诚侧改革尖刀；**七阶~超阶** 边缘，低阶规避硬刚。
- **圣吉列斯（Sanguinius）**｜性格：高贵悲悯、预见死亡仍赴死。｜装备·能力：有翼原体、近战与灵能、圣血天使血脉源头。｜人物弧光：大远征天使→西格纳斯试炼→泰拉为帝皇挡刀死于荷鲁斯。｜立场关系：忠诚烈士；其死铸成黑怒回响。
- **罗格·多恩（Rogal Dorn）**｜性格：固执、筑城者、不善虚礼。｜装备·能力：原体级防御与攻坚、法阵克斯星堡、泰拉围城总指挥。｜人物弧光：大远征盾墙→泰拉守夜→大清洗后失踪/疑陨。｜立场关系：帝国之拳与诸多子战团精神父亲。
- **黎曼·鲁斯（Leman Russ）**｜性格：野性、忠诚、可被利用的「刽子手」骄傲。｜装备·能力：狼群原体、近战与执行清洗令。｜人物弧光：焚普罗斯佩罗→异端忠诚侧→失踪入恐惧之眼（狼时将归的预言）。｜立场关系：太空野狼永不圣典化的源头。
- **莱恩·艾尔·庄森（Lion El'Jonson）**｜性格：骑士、多疑、不善沟通。｜装备·能力：第一军团原体、剑术与隐秘指挥。｜人物弧光：卡利班悲剧与堕落者种子→异端远域血战→近年正典重返。｜立场关系：暗黑天使隐秘的中心；对「耻辱」零容忍。
- **马格努斯（Magnus the Red）**｜性格：傲慢智者、悲剧式自毁与不甘。｜装备·能力：最强灵能原体之一、千子、奸奇仆从后的恶魔原体。｜人物弧光：打破禁忌警告→普罗斯佩罗之劫→混沌。｜立场关系：高阶灵能灾难源；**七阶~超阶**。
- **安格隆（Angron）**｜性格：被钉痛奴工钉的永恒怒火。｜装备·能力：吞世者原体、恐虐恶魔原体。｜人物弧光：角斗奴隶→叛乱→多次作为战役级灾厄重临。｜立场关系：**超阶边缘**；低阶只闻其名。
- **艾泽凯尔·阿巴顿（Ezekyle Abaddon）**｜性格：冷恨、战略耐心、拒当「第二荷鲁斯」却执行对帝国的慢性处决。｜装备·能力：黑军团战帅、魔剑达拉克涅、十三次黑十字军。｜人物弧光：荷鲁斯之子→恐惧之眼之王→碎卡迪亚。｜立场关系：混沌军事顶点之一；情报优先。
- **但丁（Dante）**｜性格：圣血天使的疲惫高贵，千年战争压弯不折。｜装备·能力：战团长级阿斯塔特巅峰、空降与斩首指挥、直面缺陷的领袖。｜人物弧光：巴尔与外域永恒守夜；阿米吉多顿等战役临时最高军事权威之一。｜立场关系：凡人眼中的天使；修士中的老人。
- **马纽斯·卡尔加（Marneus Calgar）**｜性格：极限战士式刚正与战术教科书。｜装备·能力：战团长、动力拳等；原铸时代前后皆为样板。｜人物弧光：乌尔塔玛与银河多线增援指挥。｜立场关系：圣典派门面。
- **加维尔·洛肯（Garviel Loken）**｜性格：正直、重兄弟情、能怀疑战帅。｜装备·能力：异端初期影月苍狼连长级战力。｜人物弧光：哀悼四杰→伊斯塔万 III 忠诚→后续黑盾/骑士线（正典延续）。｜立场关系：契约者低~中阶最宜对照的「像人的修士」。

**【势力图谱】**
- **阿德普图斯·阿斯塔特（忠诚战团）**：宗旨=保卫帝皇与人类（口径因战团而异）；地盘=母星要塞修道院/舰队/远征航线；代表=**但丁**、**马纽斯·卡尔加**、**洛根·格里姆纳**、**阿兹莱尔** 等战团长；战力=四~六阶建制、七阶看是否有原体级介入；与星界军是「尖刀与血肉」关系，与审判庭是「合作与互疑」。
- **第一创立核心战团（具名）**：**极限战士**（圣典样板·乌尔塔玛）、**圣血天使**（缺陷与高贵·巴尔）、**暗黑天使**（隐秘与猎堕·岩石）、**太空野狼**（拒圣典·芬里斯）、**帝国之拳**（筑城与泰拉遗产）、**火蜥蜴**（护民·夜曲星）、**暗鸦近卫**（阴影与残缺基因）、**白色疤痕**（速度）、**钢铁之手**（机械增强）等；各自派生大量子战团，基因稳定性决定创立配额。
- **非圣典与特编**：**黑圣堂** 远征舰队式永不结束的十字军；**灰骑士** 恶魔对口（极稀有）；**死亡守望** 跨战团猎杀小队。
- **混沌阿斯塔特 / 黑军团等**：宗旨=侍奉混沌或私人帝国复仇；代表=**艾泽凯尔·阿巴顿**、各恶魔原体与战争帮主；与忠诚修士是镜像仇敌；常劫基因种子与奴隶舰。
- **星界军（Astra Militarum）与 PDF**：凡人战争主体；为阿斯塔特提供炮灰、炮火与坐标；政委体系维持「动摇即死」。
- **机械神教（Adeptus Mechanicus）**：动力甲、舰船、泰坦与基因实验室接口；要数据与圣遗物；原铸工程的共犯与监视者。
- **审判庭 / 国教**：异端、变异、恶魔与「知情过多」的灭口权；可调阿斯塔特，亦可给整团贴叛乱标签。
- **禁军与寂静修女**：帝皇直属；异端期关键，40K 日常少见；气压用。
敌友逻辑：利益与生存优先；盟约可破；暴露混沌接触=死；阿斯塔特不是契约者的朋友，是临时同一瞄准镜。

**【贵重物品】**
- **基因种子与祖腺（Progenoid）**：战团续命根本；成熟后由药剂师回收再培养植入器官；**十一税** 交火星/泰拉检测纯度；持有未授权种子=死罪级政治灾难；混沌最想抢的硬通货；低阶契约者碰一下就等于被三方追杀。
- **完整植入器官组（含黑甲壳）**：把凡人改造成阿斯塔特的流水线零件；黑甲壳未愈者无法真正「穿上」动力甲人生。
- **动力甲与遗物甲**：对接神经的「第二人生」；战团圣物甲常刻历代阵亡者之名；遗物甲常绑定连队荣誉与牧师祝福。
- **爆弹武器与动力武器**：凡人难控的反冲与能量场；连队荣誉刃可当身份证明；熔解/等离子是对装甲的「打开方式」。
- **终结者甲与雷鹰/空降舱**：精英突入象征；引导坐标本身是六阶任务级资产——星界军整团可能为护一个信标标而死。
- **《阿斯塔特圣典》抄本 / 战团自有教典**：战术与政治圣经；极限战士可能为注解开战，太空野狼可能当柴烧；**新圣典/原铸修订** 是 M42 意识形态火药。
- **安纳塞姆之剑（Anathame）等混沌神器**：荷鲁斯堕落触媒级；接触即腐化判定；叙事中属于「不该被凡人看见」的禁忌物。
- **达拉克涅（Drach'nyen）等战帅权柄物**：**艾泽凯尔·阿巴顿** 权柄象征；不可「捡来当主角金手指」无代价使用。
- **桑普利姆斯·波图姆相关原铸技术残样**：**贝利萨里乌斯·考尔** 线；机械神教、战团与审判庭争夺的未来；泄露=政治核弹。
- **战团基因库与要塞修道院核心**：比单件武器更贵——丢了等于战团灭绝倒计时。
每件写清：谁想要、抢到的政治后果、当前阶能否用（原体武器对凡人是死刑判决书；基因种子对任何未授权持有者是全银河通缉令）。

**【隐藏剧情 · 伏笔】**
- **第二、第十一军团**：记录被抹除；正典禁止「揭开」为廉价真相，只可作恐惧与禁忌气压——任何声称「我知道失踪原体真名」的 NPC 要么是骗子，要么该被灭口。
- **基因缺陷即军魂**：**红渴/黑怒**、**狼化**、暗黑天使的 **堕落者**、钢铁之手的厌肉、暗鸦的残缺——缺陷不是「削弱到能打」，是身份、仪式与悲剧引擎；禁止写成「吃药就好的小 debuff」。
- **战士会堂（Warrior Lodges）**：从军官俱乐部到叛乱温床；提醒一切「非正式兄弟会」「跨连队秘密酒会」可被 **艾瑞巴斯** 式渗透。
- **圣典的双刃**：防军团叛乱也造成战团山头与协调困难；**基里曼** 复活后修订（新圣典/原铸编制）引发初生与原铸张力，是 M42 政治主菜。
- **帝皇是否为神**：帝国真理已死，国教已立；阿斯塔特内部有无神派与虔诚信徒撕裂——同一战壕里可能有人视帝皇为父亲，有人视其为神，有人只视其为「命令源头」。
- **加罗告警若失败**：泰拉可能更迟反应——展示「小人物连级选择」如何撬动史诗；契约者低阶最宜复刻此类「送情报」线。
- **大裂隙后双帝域**：航道与忠诚的地理分裂，使「同一战团两个半边银河」成为常态悲剧；星语迟到等于战役判死刑。
- **原体回归的政治**：基里曼与莱恩的回归不是爽文外挂，是行政、隐秘与旧恨的再分配；恶魔原体重临是战役级灾难，不是可刷的副本 BOSS 列表。

## 阶位切入点

> 阶位↔编制：一阶≈凡人巢民/新兵；二~三阶≈星界军与精锐卫军；四~六阶≈阿斯塔特（新兵满编→终结者/连长→战团长）；七阶≈原体/恶魔原体战役边缘。世界顶点（**帝皇**、**混沌邪神**、巅峰**原体/恶魔原体/阿巴顿**）为跨阶存在，剧情可标超阶，低阶切入一律「规避顶点·情报优先」；**超阶巅峰：存在·情报优先/条件性胜利**，严禁战力归零式削弱。本世界切入只覆盖 **一至七阶**。

**一阶（凡人·征兵与仰望天使）**
切入身份/时点：契约者以巢都工厂新征民兵、战团领地农奴或星港苦力身份切入，锚定任意「阿斯塔特即将降临/刚刚离开」的凡人日常——如某顺从世界等待极限战士收编，或巴尔域外难民听闻天使空降。
初始事件：广播要求上交适龄男性「候选者名单」；你的街区有人把孩子藏进排污管，政委与战团凡人附庸同步清查；你必须在「告发邻居换口粮」与「帮藏人被当异端」之间选。
开场白建议：「防毒面具里全是藻泥味。天空裂开的不是神迹，是降落舱的制动焰——工头把激光枪拍在你胸口：天使不需要你勇敢，需要你把路让开，把儿子交上去。」
关键NPC立场：**星界军下士**——要填线与秩序；**战团凡人录事**——要合格基因与忠诚记录；**巢都神父**——要颂歌与什一；**同批新兵**——怕被选中也怕不被选中；远处的 **阿斯塔特侦察组**——视你为移动障碍或临时坐标。
主线钩子/支线：候选者护送/藏匿、空降场清场劳工、颂歌游行中的刺杀流言；支线＝黑市倒卖「假基因证明」、帮药剂师凡人助手搬防腐箱。
危险度/规避：中——须规避正式阿斯塔特交火区、混沌渗透仪式、任何「靠近原体级存在」的传言现场。
任务方向/奖励：活命、民兵名册、劣质枪与「见过天使」的口供价值；奖励取向＝防毒滤芯、战团徽章碎屑（非法收藏风险）、离星船票线索。

**二阶（星界军·炮灰协同）**
切入身份/时点：以星界军步兵、装甲乘员或政委外围士兵切入，锚定阿斯塔特空降前 72 小时至协同巷战——如某工业世界绿皮/叛军压境，战团承诺「一个连」。
初始事件：你的连队奉命为雷鹰清射界，友邻整排被叛军熔解枪抹掉；通讯里出现不像人类的平静声线要求你「在坐标 T-7 点燃信号，不论那里是否还有平民」。
开场白建议：「激光枪托砸在肩上已经没感觉。天空先白后红，通讯里那个声音说：『引导完成前你们没有名字，只有坐标。』坐标末端，是你头盔里的呼吸。」
关键NPC立场：**星界军上校**——要战报好看；**政委**——要动摇者的头；**阿斯塔特中士**——要效率不要感情；**平民代表**——要你抗命留桥；**混沌教团渗透者**——要你的坐标改指向友军。
主线钩子/支线：信号引导、掩护空降、假投降诱敌；支线＝抢修被毁的星语中继、活捉叛军军官换战团问话权。
危险度/规避：高——须规避终结者突入轴心、泰坦对轰带、已知恶魔具现点。
任务方向/奖励：正规编制嘉奖、较好激光/狙击资产、短暂「可同行」臂章（凡人版）；奖励取向＝优质防弹插板、战团丢弃的破片弹药（需申报）。

**三阶（精锐卫军·政委与斩首外围）**
切入身份/时点：以风暴部队、政委直属、行星防卫军精锐士官或战团直属凡人特工切入，锚定「斩首前夜」——如目标是混沌仪式主持人或叛军基因实验室，阿斯塔特要你先撕开外层。
初始事件：你带队突入地下巢层，发现叛军在用偷来的祖腺做拙劣移植；通讯半残，战团只给你四十分钟窗口，超时轨道打击不认友军。
开场白建议：「防腐液的甜味盖过血腥。玻璃罐里有东西在跳——那是不该出现在凡人手里的种子。计时器每跳一次，轨道上的光矛就更亮一分。」
关键NPC立场：**纳撒尼尔·加罗**（若锚异端/告警航道）——要忠诚证据送达泰拉方向，视你为可消耗信使；**马纽斯·卡尔加** 的凡人联络官（M41 战场）——要四十分钟窗口内的完成度，不在乎你的名字；**艾瑞巴斯** 余党祭司——要腐化你或抢回祖腺罐；本地 **星界军政委**——要处决动摇者与「接触种子者」；**审判庭密探**——要灭口所有看见非法基因罐的人，包括你。
主线钩子/支线：实验室清剿、祖腺夺还、四十分钟撤离；支线＝伪造「未接触混沌」的血检报告、在审判庭与战团之间选交证据对象。
危险度/规避：高——混沌轻型实体、叛变阿斯塔特斥候可能出现；须规避已知连长级叛徒。
任务方向/奖励：英雄编制、动力武器残件（需上缴或获特许）、情报网接入；奖励取向＝准军事仿生眼、战团临时赦免令。

**四阶（阿斯塔特新血·试炼与首战）**
切入身份/时点：契约者以 **刚完成关键植入、黑甲壳初愈的新阿斯塔特**（或同等战力的外来改造体被战团临时收编）切入，锚定某战团新兵连首次实战——极限战士的顺从世界平叛、野狼的芬里斯外域猎杀、圣血天使的血缘试炼后第一战等。
初始事件：你的小队奉命夺取叛军广播塔；爆弹在你肩上的后坐力第一次「像自己的心跳」；药剂师警告：若同伴阵亡，你要在三分钟内切下祖腺，否则整次任务在战团账上等于零。
开场白建议：「甲内圣油是热的，血液是冷的。中士只说一句：今天你不再有家，只有连队编号。广播塔在第三层，叛军在唱战帅的旧歌——你的爆弹，是对那首歌的唯一评论。」
关键NPC立场：**本连中士**——要纪律与战损比；**药剂师**——要种子不要英雄；**图书管理员（若在场）**——要侦测灵能污染；**叛军军官**——要活捉一个「天使」献祭；**初生老兵**——嘲讽/考验原铸或外来者。
主线钩子/支线：广播塔、第一次祖腺回收、新兵连荣誉标记；支线＝拒绝屠杀平民与圣典教条的冲突、野狼式决斗入伙。
危险度/规避：高——同阶叛变修士、重火力点；须规避战团长级决斗与任何原体名号。
任务方向/奖励：战术小队转正、动力甲涂装权、连队刃；奖励取向＝优质爆弹枪、战团赞歌记名、基因稳定性药剂（合法渠道）。

**五阶（资深修士·小队指挥与缺陷）**
切入身份/时点：以战术小队指挥、终结者候选人、或圣血/野狼等缺陷战团的资深修士切入，锚定一场「必须用缺陷换胜利」的战役——如黑怒边缘的斩首、狼化阴影下的追猎、暗黑天使猎堕时要你对凡人灭口。
初始事件：目标是叛徒连长级；你的一名兄弟在接敌前陷入黑怒/狼化前兆，牧师要你选择：锁进死连突击，或一枪结束以免污染任务。
开场白建议：「头盔内红光闪了三次——那是血在唱，不是警报。牧师的权杖抵在你的兄弟眉心：『你来选，中士。天使的仁慈，有时是爆弹。』」
关键NPC立场：**战团牧师**——要灵魂与纪律；**药剂师**——要活体样本与种子；**但丁** 或 **洛根·格里姆纳** 一类高层的远程军令——要结果；**堕落者/叛徒连长**——要拉你下水；**审判庭代表**——要战后报告里没有「缺陷失控」。
主线钩子/支线：斩首叛徒连长、处理失控兄弟、对凡人目击者灭口与否；支线＝原铸与初生争夺指挥权、抢在黑军团之前回收基因库。
危险度/规避：极高——连长级混沌阿斯塔特、轻型恶魔；须规避恶魔原体战场与阿巴顿本尊。
任务方向/奖励：终结者候选、连级指挥权、遗物甲零件；奖励取向＝圣物链锯剑、缺陷抑制剂（有副作用）、战团密令羊皮卷。

**六阶（连长/战团中坚·战役改写）**
切入身份/时点：以连长、第一连王牌、舰队打击群指挥或战团长副手切入，锚定战团级行动——如不屈远征某节点、巴尔保卫战外围、乌尔塔玛方向虫潮/混沌双线。
初始事件：星语显示增援不会来；你要在「放弃三个巢都保基因库」与「全线平摊战死」之间签字；机械神教同时索要战场上的原铸遗体「研究」。
开场白建议：「全息星图上的红点比兄弟的名字多。战团长的频道只剩断续：『你的连，就是今天的帝国边界。』边界之外，亚空间在笑。」
关键NPC立场：**马纽斯·卡尔加** / **但丁** / 本战团 **战团长**——要战略服从；**贝利萨里乌斯·考尔** 的代理人——要遗体与数据；**星界军集团军司令**——要你救他的城；**阿巴顿** 的战帮先知——要你的基因库坐标；**审判庭领主**——要否决权。
主线钩子/支线：基因库转移、弃城签字、原铸遗体政治；支线＝与友军战团争夺空降窗口、猎杀负责伪星语的千子巫师小队。
危险度/规避：极高——战役级火力、多名精英混沌阿斯塔特；须规避完整原体决斗与帝皇相关圣所硬闯。
任务方向/奖励：战团荣誉十字、临时战区指挥权、遗物级动力拳；奖励取向＝舰队优先补给令、合法基因种子调拨指标（极少）。

**七阶（原体/恶魔原体边缘·史诗气压）**
切入身份/时点：以战团长、原体直属冠军、不屈远征高级特使或混沌侧同等「仅次于半神」的存在切入，锚定原体/恶魔原体 **战役边缘**——如基里曼巡视前线的外围十二小时、莱恩猎堕的侧翼、安格隆降临前的撤离窗口；**不是**与帝皇/邪神本尊对轰。
初始事件：亚空间裂口扩大，星语尖叫着某个恶魔原体的真名；**罗保特·基里曼**（或 **莱恩·艾尔·庄森**）的指挥链覆盖你的战团，要求你在半神交锋前完成一件凡人做不到的事：封印次级门、刺杀仪式核心、或护送一件能让原体多撑一小时的圣物。
开场白建议：「你听见的不是雷，是世界在被重写。通讯里响起的嗓音比战团歌咏更古老：『完成它。不要看天空。天空此刻不属于你。』你知道那是原体——也知道自己仍只是棋盘上较锋利的一枚。」
关键NPC立场：**罗保特·基里曼**——要帝国续命与纪律；**莱恩·艾尔·庄森**——要隐秘与结果；**马格努斯 / 安格隆 / 莫塔里安**（择一为气压）——要门开得更大；**艾泽凯尔·阿巴顿**——要渔利；**人类帝皇**——不可对话的超阶背景辐射。
主线钩子/支线：次级门封印、仪式核心斩首、圣物护送；支线＝阻止友军战团「荣耀冲锋」去送死、在战后报告中抹去不该存在的第二/十一军团流言。
危险度/规避：贴近顶点——**超阶巅峰：存在·情报优先/条件性胜利**；严禁硬刚帝皇/邪神/满开恶魔原体；胜利形态=开门晚一小时、门关上一道、圣物送达、活着把情报送回泰拉方向。
任务方向/奖励：原体级嘉奖（政治核能）、战团永久荣誉名、跨星域调兵权碎片；奖励取向＝近遗物武装、原铸/基因稳定性极限资源、以及「被半神记住名字」的危险荣耀。
