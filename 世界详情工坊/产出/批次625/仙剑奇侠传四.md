# 仙剑奇侠传四

## 剧情

**【作品来源】**
《仙剑奇侠传四》（Chinese Paladin 4 / Sword and Fairy 4）是上海软星科技有限公司开发、大宇资讯发行的单机角色扮演游戏，2007年8月1日正式发售，为仙剑系列第五部作品（正传序号第四）。主题定为「寻仙」。本作首次采用真人比例三维建模与 RenderWare 引擎，战斗为半即时制，主角团四人（云天河、韩菱纱、柳梦璃、慕容紫英）最多三人上场。故事结论落在「欲求仙道，先修人道」。文风偏悲剧仙侠：少年情义、宗门野心、人妖恩怨与逆天代价交织，结局唯一且开放。系列时间线上为较早时代，舞台自蜀山系转至昆仑琼华派。后有全语音版与电视剧改编；重制版由方块游戏以虚幻5引擎推进（写作时仍在开发）。

**【世界定位】**
六朝至唐前氛围的人界—妖界—仙界三层仙侠世界。昆仑山琼华派为飞升举派，以羲和、望舒双剑网缚幻暝界抽取灵力；十九年后，青鸾峰孤儿云天河因韩菱纱盗墓结缘下山，与柳梦璃、慕容紫英揭开双剑与玄霄的旧案，最终在琼华飞升之际以神器挽人界于天火。主线是「寻仙」对「人道」的叩问，而非单纯升级打怪。契约者入此界，最先撞上的往往不是神器，而是人情与谎言。

**【世界观 · 力量体系】**
世界以「灵」为根基：人修仙术、妖修妖力、仙掌天规。人界修士借五灵（水、火、雷、风、土）相生相克施法，铸剑、御剑、阵法、丹药并行。琼华派尤擅「剑仙」一脉——以剑为骨、仙术为锋，双剑羲和（极阳）与望舒（极阴）可合为通天飞升之柱，亦可反噬宿主。妖界幻暝界以梦貘一族为尊，能入梦、食梦、织幻，结界与紫晶灵力深厚。神界九天玄女一系司天道赏罚，降下天火惩戒逆天。鬼界不周山有衔烛之龙守界，翳影枝等灵物可通幽冥。

成长逻辑：凡人习武与粗浅符咒 → 入门修士凝灵运炁、学五灵仙术 → 中坚可御剑千里、裂石断金 → 精英开宗立派、以一己之力护一城一派 → 顶尖如玄霄可持羲和剑与妖界开战、移山级灵压 → 神器级（后羿射日弓 + 神龙之息 + 双剑合击）可射落飞升宗门、对抗天火余波。死亡规则残酷：凡人一命；修士可借丹药、寒器、冰封延命，但走火入魔、双剑反噬、天火焚身皆可灭神魂。玄霄被东海漩涡永锢不得轮回，是本世界最高惩罚之一。复活与轮回属神道权柄，契约者不得默认可逆。

境界/战力表现（按原作可观察破坏力，宁低勿高）：
1. **凡人/江湖好手**：冷兵器、机关术、捕快武艺；典型如寿阳城捕头裴剑、太平村猎户。破坏力≈徒手搏击、破木门。
2. **入门修士/外门弟子**：初学五灵仙术、御剑入门；可御风短程、打退低阶精怪。
3. **中坚修士/内门好手**：稳定仙术连发、能单人清巢穴级妖物；月牙村、女萝岩一线强者。
4. **琼华精英/真传铸剑师**：如慕容紫英成长期、重光青阳宗炼等长老中层；御剑破阵、铸神兵、掀平据点级战斗。
5. **玄霄级/妖王级**：玄霄（羲和宿主，出棺后复仇态）、婵幽（幻暝界妖王）、掌门夙瑶全力飞升态；一击毁街区至城域、领域/阵法雏形、可网缚一界灵力。
6. **射天火/神器级**：后羿射日弓 + 天河剑 + 神龙之息、九天玄女降下的天火、双剑通天柱；覆盖地区、移山填海、令飞升琼华崩毁。再上的神道本尊（九天玄女完整权柄）视为跨阶顶点，低阶只闻其名。

乐园阶位映射：凡人/江湖≈一阶；入门修士≈二阶；中坚修士≈三阶；琼华精英/真传≈四阶；玄霄级/妖王级≈五阶；射天火·神器级≈六阶。世界顶点＝九天玄女所代表的天道惩戒与神器合击（六阶顶至跨阶），低阶切入一律规避本体·情报优先。宁低勿高，勿把玄霄写成可无代价硬撼。

**【地理 · 舞台】**
舞台自东南山林铺向西北昆仑，再下探妖界、鬼界。
- **青鸾峰（黄山一带）**：云天青与夙玉隐居处，云天河成长之地；父母墓室藏望舒剑与古玉；结局百年后三人（及梦璃）重逢的终章舞台。
- **紫云架 / 石沉溪洞 / 太平村**：下山第一程。石沉溪洞机关与魁召；太平村端午闹剧；巢湖边树林初遇慕容紫英。
- **寿阳城 / 柳府 / 女萝岩 / 淮南王陵**：柳世封为县令，柳梦璃成长处；女萝岩槐妖与离香草；淮南王陵怨魂是早期高难副本。
- **陈州 / 千佛塔 / 怀朔—璇玑一线 / 播仙镇 / 太一仙径**：西行入昆仑的人间走廊，琴姬支线、西域风情、琼华入门试炼须臾幻境。
- **昆仑山琼华派**：铸剑仙门，主殿、禁地冰窟、卷云台；双剑飞升时整座山门拔地而起。
- **幻暝界（妖界）**：梦貘一族家园，幻暝宫、里幻暝宫；十九年前被双剑网缚，主线中再度遭琼华破界。
- **不周山 / 鬼界**：衔烛之龙盘踞，翳影枝所在，通妖界的关键灵物产地。
- **炎帝神农洞 / 居巢国（巢湖）/ 即墨一线**：三寒器（梭罗果、鲲鳞、光纪寒图）搜寻路线。
- **东海漩涡**：九天玄女囚禁玄霄、夙瑶等逆天者之处；系列后作仍可闻玄霄未脱。
- **封神陵**：后羿射日弓出世处，终局神器锚点。

地理逻辑：低阶冲突在青鸾峰—寿阳—陈州人间；中阶转入琼华门规与西域险地；高阶下探幻暝与不周，终局回昆仑—人界天空。人间驿站与仙门之间隔着「太一仙径」这类天然筛子——过不去的散修永远只是西域商队的护镖，过得去的才有资格听见「飞升」二字。幻暝界不在地图常规坐标上，需翳影枝或同级灵物打开通道；鬼界不周山则是人、妖、鬼三线交汇的冷门捷径，死亡率极高，却是主线取枝的必经。东海漩涡不在常规航线，是神道行刑场，凡人船队靠近即迷航。

**【世界剧情线】**
① **前史·十九年前：双剑网缚与人妖大战**
昆仑琼华派信奉九天玄女，以铸剑秘术闻名，位列昆仑修仙诸派前列。为举派飞升，掌门太清一系定策：以极阳羲和剑与极阴望舒剑合铸「通天剑柱」，网缚幻暝界，抽取妖界灵力与紫晶为升仙资粮。羲和剑宿主为天才弟子玄霄，望舒剑宿主为阴寒体质的夙玉。双剑启动后，幻暝界结界崩溃，人妖大战爆发，双方死伤惨重。妖王婵幽与琼华高层血战，太清一脉折损。夙玉在战中看清飞升实为以一界生灵为柴薪，毅然抽离望舒，双剑失衡；羲和邪火反噬玄霄，令其走火入魔、红气缠身。幸存长老夙瑶、重光、青阳、宗炼等合力以极寒之法将玄霄镇压于禁地冰棺。云天青与夙玉携望舒剑叛离琼华，途中自战场救下婵幽之女（后名鸢洛，即柳梦璃），托付给寿阳好友柳世封抚养。二人隐居青鸾峰成亲，生下云天河。夙玉因望舒寒气侵蚀体魄日渐衰竭而亡；云天青将阴阳古玉与剑道碎片留给幼子后亦病逝。幼年云天河独自在青鸾峰长大，以猎为生，不知山下有国有家，更不知双剑与飞升。琼华官方史把「网缚」写成「镇妖护道」，把天青、夙玉写成「叛徒」，把玄霄写成「需永镇的走火者」——这三套叙事要到里幻暝宫梦境才会被一并击碎。

② **开篇·青鸾峰：盗墓少女与「这是剑」**
十八岁的云天河追山猪入石沉溪洞，撞见自称「独行千里的陵墓大盗」的韩菱纱。韩菱纱出身盗墓世家，因先人触怒鬼界，一族寿命仅二三十岁，她为寻长生与传说中的仙剑闯入云家墓室。墓中极寒，望舒剑异动破棺而出，被天河随口唤作「这是剑」。剑气震毁墓室机关，引出父母未竟之事。天河为弄清父亲为何隐居、母亲墓中为何有仙剑，决意随菱纱下山「求仙」。二人自紫云架抵太平村，因端午习俗与山猪误会被村民驱赶；巢湖边树林遇妖，御剑少年慕容紫英出手相救。紫英冷面重义，见天河资质奇特却不通人情，留下去寿阳的线索后御剑离去。

③ **寿阳城：柳梦璃与人间第一课**
寿阳城中，天河误揭捕盗榜几乎抓错菱纱，结识捕头裴剑，被邀至柳府。县令柳世封是云天青故交，一见天河如见故人。府中千金柳梦璃温婉聪慧，实则梦貘血脉，夜梦中可化妖形。梦璃以试探确认天河无害后，吐露自己乃十九年前云天青所托女婴。城外女萝岩因采药过度激怒槐妖，天河一行调解，获土灵珠等机缘。梦璃恳请同行寻仙，柳世封夫妇含泪应允，并赠天河玉腰弓。西行前经淮南王陵，怨魂与机关考验四人初成的默契。此阶段主题是「人间」：天河学会买卖、礼数、何为亲情与谎言；菱纱的短命诅咒与盗墓本能并行；梦璃在人族身份与妖族本能间摇摆。

④ **西行·陈州至播仙：入琼华之门**
陈州千佛塔，琴姬为亡夫超度，支线写尽人世情痴。怀朔、璇玑等引路人将众人带向西域播仙镇，再登太一仙径。琼华入门试炼「须臾幻境」以酒色财气考验道心：天河以天真破局，菱纱以利落拆招，梦璃以悲悯收心，紫英以剑心照影。拜入琼华后，年纪相仿的慕容紫英以师叔之位教授御剑与心法。紫英为燕国王族后裔，幼年体弱入昆仑，铸剑天赋冠绝同门，表面清冷，实则极重承诺。四人在琼华目睹门规森严、资源按资质分配，亦听闻十九年前「叛徒云天青、夙玉」的官方叙事——与天河记忆中的父母截然相反。

⑤ **禁地·玄霄出棺与三寒器**
夜半望舒剑自鸣，引天河潜入禁地，得见冰棺中的玄霄。玄霄容貌俊极，灵压即便在极寒镇压下仍令同门胆寒。他自称是天青、夙玉的师兄，被掌门夙瑶妒贤所害。天河一见如故，甚至与玄霄结拜，答应寻三件寒器解除镇压：光纪寒图（即墨狐仙一线）、梭罗果（炎帝神农洞）、鲲鳞（巢湖居巢国遗迹）。寻器旅途本身即是中期主线骨架：即墨一线要与狐仙周旋，分清「借图」与「夺图」；神农洞古阵与炎毒考验队伍配合；居巢国水下遗迹把菱纱的盗墓本能与鬼界诅咒再度推上台面。解除镇压过程中，夙瑶与长老们的阻挠、门内派系倾轧、菱纱体质与望舒共鸣渐显。玄霄苏醒初期仍存温情与傲骨，称天河为「好师弟」；一旦灵力恢复、得知夙玉已死、双剑飞升旧案被重提，心魔与邪火并起，性格急转——伤长老、夺权柄、重提网缚妖界。此卷名场面是「解除镇压即失控」：善意的结拜直接把人间最大的定时炸弹放出冰棺。紫英在此阶段开始公开质疑门规，铸剑炉边第一次把「剑是杀人还是护人」问出口。

⑥ **身份崩裂：梦璃归族与菱纱宿主**
幻暝界结界再度震动，琼华高层密议再次以双剑抽取妖灵。柳梦璃身份暴露：她是婵幽之女、幻暝界少主鸢洛。人族养育的温婉与妖族血脉的责任撕扯她。为救梦璃并查清真相，天河一行经卷云台，持不周山衔烛之龙处求得的翳影枝进入幻暝界。取翳影枝本身是独立高难段落：鬼界规则不同于人界仙术，衔烛之龙不以善恶论人，只以是否「配持枝通幽」为试。幻暝宫中，婵幽以母性与王者双重面目见女；里幻暝宫重现十九年前梦境——太清的贪婪、夙玉的抽剑、玄霄的反噬、云天青的救婴，一帧帧打脸琼华官方史。琼华破界杀入，玄霄在吸取足够灵力后竟暂时放过幻暝（仍带报复与算计），却当众点破：韩菱纱阴寒之体正是望舒剑第二代宿主，寒气将加速她的短命。梦璃为护全族，继位后随幻暝界封闭而去，与天河约「来生」或「十九年后」。天河、菱纱、紫英三人回人界，情义从「四人同行」裂为「三人逆天」。此阶段政治含义极重：人界仙门失去「镇妖」道德高地，妖界获得道义与舆论，神道开始把目光从「供奉」转向「清算」。

⑦ **终局·飞升、天火与后羿一箭**
夙瑶与玄霄（或玄霄主导、夙瑶胁从，依流程细节略有出入）重启双剑，琼华山门整体拔升，欲破天光成仙。飞升导致山体将坠、灵压撕裂人间，昆仑至中原皆受威胁——这不是门内政变，而是地区级天灾。云天河坚持「我命由我不由天」，拒做双剑棋子。慕容紫英铸成「天河剑」，以铸剑师之命押在朋友的弓弦上；韩菱纱以宿主之躯强行共鸣望舒，寒气入骨仍不退。封神陵中，后羿射日弓出世；不周/神龙一脉的「神龙之息」灌入天河之体。九天玄女降世，斥琼华逆天，天火自九霄倾泻，欲将飞升者与受牵连的人间一并清算。云天河在菱纱、紫英护持下，以凡人肉身拉满后羿射日弓，搭天河剑为矢，射碎下坠的琼华主峰与双剑通天柱，保全人界。代价是双目失明、神魂重创；玄霄被玄女压入东海漩涡永世不得超生；夙瑶等逆天弟子囚东海漩涡数百年至千年后方许轮回。琼华派名存实亡，残部星散。终局战斗的可写颗粒度在于「分工」：紫英稳住剑矢与阵眼，菱纱稳住望舒不让通天柱二次成型，天河承受神弓反噬——缺一环则人界坠落。

⑧ **开放结局·青鸾峰百年**
天河、菱纱、紫英归隐青鸾峰。天河以神龙之息压制望舒寒气，与韩菱纱成婚；紫英继续修仙铸剑，白发渐生。约百年后，菱纱终因寒气与韩家短寿仙逝，墓碑刻「爱妻韩菱纱之墓」，望舒剑立于墓前。慕容紫英已近剑仙之境。柳梦璃处理完幻暝界事务归来，见紫英所制的思念傀儡消散，跪抚菱纱墓前。木屋中，因神龙之息而容颜未老、双目仍盲的云天河闻异香出门——他与梦璃对望，画面定格。官方不给出唯一解读：是重逢续缘，是来生之约兑现，还是盲者感知故人气息后的永恒瞬间，皆由玩家自填。主题收束于：仙可求，人更不可负；逆天者灰飞，守人道者长留青鸾一隅。青鸾峰此时已不是开篇的「与世隔绝猎场」，而是三人（及后来的梦璃）用百年把「人道」活成了风景：墓、剑、屋、盲眼与异香，构成系列最克制也最锋利的终章镜头。

⑨ **微观政治与可介入日常**
除大场面外，世界高频事件包括：琼华门内资源配额与真传选拔；寿阳—陈州一带的妖患委托与捕头悬赏；盗墓行当与鬼界诅咒的黑市交易；女萝岩式的人妖资源冲突；西域商队护送与播仙镇情报交换；月牙村缺水一类的民生仙术干预。契约者低阶应优先这些「人间题」，而非一上来抢双剑。门派食堂的灵米份额、铸剑堂排队用炉、外门弟子替真传试毒剑——这些琐碎压迫比口号更能写活琼华。人间侧则是税赋、婚嫁、庙会与妖患告示并存：柳世封能护一城平安，却护不住女儿的血脉秘密。

⑩ **蝴蝶效应边界**
可改：救下某村妖患、改写某次拍卖归属、让菱纱少中一次寒毒、助紫英早得铸剑材料、推迟玄霄出棺数日。不可无代价改：双剑飞升的根本野心、玄霄邪火反噬的因果、韩菱纱宿主体质与短寿、九天玄女天火的天道属性、开放结局的核心意象（青鸾峰、失明、墓碑、梦璃归来）。强行抢走羲和/望舒据为己有，会立刻成为琼华、幻暝、神道三方公敌。

⑪ **支线与人间温度（可写颗粒）**
陈州琴姬超度亡夫，是「求仙者仍被人间情困」的镜像；月牙村缺水，仙术借珠失败，逼主角承认灵力解决不了制度与地理；五毒兽、狐仙、枕影疗伤等支线提供材料与人情，也提醒西域—昆仑线上散修生态：有人卖情报，有人卖命，有人卖假仙缘。美食收集、注灵装备、迷宫宝箱是玩法层，叙事层应转写为「市集议价、铸剑耗材、禁地残骸」等可观察细节，避免OOC成系统菜单。四人行的日常拌嘴（菱纱损天河不通人事、紫英冷脸收拾残局、梦璃温和圆场）是中前期情感胶水，一旦梦璃归族，胶水断裂的空白本身就是终局前的叙事动力。

**【主要人物】**
- **云天河（主角）**｜性格：天真到近乎不通世故，却极重情义；信条「我命由我不由天」；对朋友掏心掏肺，对「仙」既向往又在终局以人道压过仙道。｜装备·能力：父母所留弓术与「这是剑」（望舒）；后得玉腰弓、天河剑、后羿射日弓；神龙之息灌体可抑寒、延寿，亦是射天火的燃料；五灵仙术成长快，战斗定位偏物理弓剑。｜人物弧光：青鸾峰猎户 → 琼华弟子 → 结拜玄霄的「好师弟」 → 射碎飞升琼华的失明守人者 → 青鸾峰隐居百年。｜立场关系：对韩菱纱由好奇到结发；对柳梦璃是青梅与跨族牵挂；对慕容紫英是生死兄弟；对玄霄由崇拜到不得不反；对夙瑶始终警惕。

- **韩菱纱（女主·盗墓者）**｜性格：嘴硬心软、利落泼辣，习惯以玩笑掩饰短命恐惧；重承诺，盗亦有道。｜装备·能力：双刺/短刃、机关与风水堪舆；阴寒体质使她成为望舒第二代宿主，能引剑亦被剑反噬。｜人物弧光：为长生盗墓 → 入琼华见仙门虚伪 → 知宿主真相仍选留下 → 以寒气燃尽之躯助天河射箭 → 婚后百年葬于青鸾峰。｜立场关系：天河的妻子与最稳的人间锚；与梦璃先疑后敬；与紫英互损却托命；视琼华飞升为夺命枷锁。

- **柳梦璃 / 鸢洛（女主·幻暝少主）**｜性格：温婉知礼，内心极坚；人族教养与妖族责任拉扯时少言而行动果决。｜装备·能力：箜篌、幻术与梦貘血脉；可入梦、织障、沟通妖界结界。｜人物弧光：寿阳千金 → 同行求仙 → 身份暴露归族继位 → 封闭幻暝护族 → 百年后践约青鸾峰。｜立场关系：天河是她人间与妖界之间的情丝；婵幽是血脉与王权；对琼华网缚怀世代之仇，对紫英、菱纱以友相待。

- **慕容紫英（男主·琼华铸剑）**｜性格：外冷内热，寡言守诺，敬剑如敬人；对不义之举即使来自本门亦不肯从。｜装备·能力：长剑、御剑术、铸剑神技；终局铸天河剑，后近剑仙。｜人物弧光：燕国王族弃子 → 琼华真传师叔 → 四人行的理性脊梁 → 铸剑助射天火 → 百年后白发守青鸾。｜立场关系：视天河为可托死者；对菱纱又气又护；对梦璃尊重其抉择；对夙瑶、玄霄从门内伦理走向决裂。

- **玄霄（羲和宿主·悲剧反派）**｜性格：天才孤傲，重情亦执念极深；出棺前尚存师兄温言，出棺后被邪火与复仇吞没，仍对天河残留结拜之情。｜装备·能力：羲和剑（极阳）、绝顶仙术与灵压，可与妖王级对峙，推动整派飞升。｜人物弧光：琼华第一天才 → 双剑失衡走火 → 极寒镇压十九年 → 出棺复仇夺权 → 东海漩涡永锢。｜立场关系：爱慕/纠葛于夙玉；视天青为弟；结拜天河却终成其箭下之敌；与夙瑶既利用又怨恨。

- **夙瑶（琼华第二十五代掌门）**｜性格：资质平庸却权欲与执念极强，妒贤，为飞升可赌全派；终局前才悟「欲求仙道，先修人道」。｜装备·能力：掌门权柄、阵法与门内资源，借双剑与玄霄合作达成飞升。｜人物弧光：危机中继任 → 极寒镇压玄霄的主事者之一 → 再启飞升 → 囚东海漩涡数百年后轮回。｜立场关系：对玄霄复杂（妒、惧、依赖）；视天青夙玉为叛徒；视天河一行为变数必除。

- **云天青 / 夙玉（前史关键）**｜性格：天青重义守诺；夙玉柔而有决，宁叛门不灭界。｜装备·能力：天青剑道与救人手段；夙玉为望舒初代宿主。｜人物弧光：琼华弟子 → 叛门隐居 → 托孤梦璃 → 双双早逝于青鸾峰。｜立场关系：玄霄的师弟妹与心结；柳世封的挚友；天河的血脉来源。

- **婵幽（幻暝界妖王）**｜性格：护族如命，对人类修士极度戒备，唯对救女之恩记在心。｜装备·能力：妖王级灵力、梦貘至高幻术，曾斩琼华高层。｜人物弧光：大战幸存者 → 失女十九年 → 迎回鸢洛 → 率族封闭自保。｜立场关系：梦璃之母；琼华世仇；对天河因天青之恩而复杂。

**【势力图谱】**
- **琼华派（昆仑）**：宗旨由「铸剑修道」堕为「举派飞升不惜代价」。地盘昆仑山门、太一仙径、卷云台。代表人物夙瑶、玄霄、慕容紫英、重光、青阳、宗炼。战力覆盖三至五阶，飞升时触六阶灾变。与幻暝界世仇，与神道表面供奉实则僭越。门内资源按资质与「对飞升有用度」分配，铸剑堂与执法堂是两套话语权。
- **幻暝界（梦貘妖族）**：护族自保，忌人类修士。代表婵幽、柳梦璃（鸢洛）。战力妖王五阶，结界与幻术冠绝。对琼华双剑网缚怀世代仇恨。里幻暝宫保存战争记忆，是真相档案馆。
- **寿阳—陈州人间官府**：凡人秩序。代表柳世封、裴剑。战力一至二阶，提供情报、户籍、委托板。对仙门采取「供奉+隔离」策略。
- **盗墓/堪舆江湖**：韩菱纱出身的灰色行当，与鬼界诅咒挂钩，是短命体质的社会根源。掌握王陵、古墓坐标，常被仙门当工具人。
- **神道（九天玄女一系）**：天道赏罚，终局规则制定者。不轻易下场，一旦下场即天火与永锢。琼华的信仰对象恰恰是其最终审判者。
- **鬼界不周山**：衔烛之龙中立守界，灵物翳影枝是通妖关键。人界修士擅入多死，唯持正当理由者可过。
- **西域播仙商路**：散修、商队、情报贩聚集，是入琼华前的灰色缓冲区，无统一首领，规则是「钱与命哪个硬听哪个」。

敌友逻辑：利益与「飞升/护族/守人」三元冲突决定站队；盟约可在双剑启动时瞬间作废；暴露持有羲和或望舒＝被三方同时盯上。契约者若只帮人间民生，琼华与幻暝都可能暂时放过；一旦碰双剑，则无中立。

**【贵重物品】**

- **望舒剑**：极阴仙剑，夙玉→韩菱纱宿主。寒气蚀体，可破墓室、引禁地、共鸣飞升柱。结局立于菱纱墓前。
- **后羿射日弓**：封神陵神器，凡人强拉需神龙之息。终局射碎琼华主峰，是六阶事件核心道具。
- **天河剑**：慕容紫英为云天河所铸，作射日之矢，友情与铸剑道的具象。
- **神龙之息**：古龙/神脉之气，灌体可抑望舒寒、延寿、供神弓。天河失明后仍靠其驻颜。
- **阴阳古玉**：云天青留给天河，平衡体质、与双剑有隐秘共鸣。
- **三寒器——光纪寒图、梭罗果、鲲鳞**：解除玄霄极寒镇压的钥匙，分处即墨狐仙线、炎帝神农洞、居巢国。
- **翳影枝**：不周山衔烛之龙处所得，开启/通往幻暝界的关键。
- **五灵珠（水火雷风土）**：仙术成长与支线核心，如女萝岩土灵珠、月牙村水灵相关。
- **玉腰弓**：柳府所赠，天河中期主力。

每件须写清：谁想要（琼华/幻暝/神道/盗墓行）、抢到的后果（宿主诅咒、三方追杀、天火点名）、当前阶能否用（双剑与射日弓低阶不可稳态驾驭）。

补充流通物：离香草（女萝岩经济冲突源）、紫晶（妖界灵力结晶，飞升燃料）、琼华弟子令牌（门内通行与资源配额凭证）、鬼界纸钱符与盗墓图谱（韩氏一行硬通货）。这些不是神器，却决定低中阶日常能不能活过下一旬。

**【隐藏剧情 · 伏笔】**
- 云天青「叛徒」叙事是琼华官方史；里幻暝宫梦境是真相回收。
- 韩菱纱短命不仅是家族诅咒，更是望舒择主的因果闭环——她越靠近「仙」，死期越近，这是主题反讽。
- 玄霄对夙玉的情感与对飞升的执念互相喂养，解除镇压后「放幻暝一马」仍是算计而非彻底悔悟。
- 九天玄女「欲求仙道，先修人道」是对整部主题的盖章，亦是琼华信仰的自我否定。
- 开放结局中梦璃归来与天河失明对望，是系列讨论最多的情感伏笔，官方保留多解：续缘、践约、或只是梦貘织就的最后一梦。
- 东海漩涡中的玄霄在后作时间线仍未解脱，本世界终局惩罚具有跨作硬度。
- 慕容紫英铸剑线表面是辅助，实为「人器合一」对抗「双剑噬人」的价值对照：剑可成神器，人却不该被剑吃掉。
- 阴阳古玉与天河体质暗示其天生可短暂平衡阴阳双剑，却不足以替代完整宿主契约，故终局仍要付眼目之价。
- 月牙村借水失败一类支线，预演「仙术不能解决一切人间题」——为终局弃仙守人埋伏笔。

- 约百年后 → 菱纱辞世；紫英近仙；梦璃归来；开放定格。

**【叙事基调 · 雷区】**
基调：清冷仙侠底色上的少年热血与长恨；笑中带泪，终局苍凉而不虚无。对话可俏皮（菱纱吐槽、天河傻问），但力量与死亡必须沉重。忌把云天河写成无脑开挂的无敌剑仙；忌把玄霄扁平化为纯坏人（他是被飞升与邪火共同毁掉的天才）；忌低阶让九天玄女、完整双剑合击满地走；忌用「战力归零式镇压」解释玄霄——极寒冰棺是高代价强行按住，一旦解除即五阶级灾难，本体始终危险；忌编造原作没有的多结局婚嫁系统（本作结局唯一开放）；忌群像代称凑人物表；忌把幻暝界写成无脑邪妖窟（他们是被网缚的受害者）。最早切入锚点：青鸾峰墓室事件当夜，或寿阳女萝岩前夜。

青鸾峰的风比琼华的剑更先教会云天河何谓家。

## 阶位切入点

> 阶位↔原作强度：一阶≈凡人/江湖好手；二阶≈入门修士；三阶≈中坚修士；四阶≈琼华精英/真传；五阶≈玄霄级/妖王级；六阶≈射天火·神器级。世界顶点（九天玄女天火与神器合击）为跨阶压力，低阶切入一律「规避顶点·情报优先」。本世界仅覆盖：一阶、二阶、三阶、四阶、五阶、六阶。

**一阶（凡人江湖 · 青鸾峰—太平村线）**
切入身份/时点：契约者以太平村猎户、山间药农或初出茅庐的捕快学徒身份切入，锚定韩菱纱闯入云家墓室前后数日。
初始事件：石沉溪洞附近山猪与机关并发，你被卷入云天河与韩菱纱的争执——墓室异响后极寒外泄，村民要封山，菱纱要进墓，天河要护父母安眠。你必须在「带路赚钱」「报官」「帮天河守墓」中选边。
开场白建议：「青鸾峰的雾比刀还利。你看见一个束着马尾的少年提着弓从猪群里钻出来，对面女孩笑嘻嘻把洛阳铲往肩上一扛——地底忽然传来金铁龙吟，整座山像是被什么东西吸走了温度。」
关键NPC立场：**云天河**——不通世故但护墓极凶，可交可骗；**韩菱纱**——出价买路，也会把你当替死鬼探机关；**慕容紫英**——尚未深交，御剑路过时只当凡人麻烦；**裴剑**——若你把事闹到寿阳，他会按捕快规矩办。
主线钩子/支线：墓室望舒初鸣的蝴蝶效应；支线＝太平村端午纠纷、巢湖边低阶妖物清理、为菱纱弄一张假路引。
危险度/规避：中——死于机关、山猪群与低阶精怪；须规避任何琼华内门御剑者与禁地寒气本体。
任务方向/奖励：活着离开青鸾峰并换到第一笔安家钱；奖励取向＝凡铁好弓、基础伤药、堪舆残页、寿阳介绍信。

**二阶（入门修士 · 寿阳—女萝岩线）**
切入身份/时点：以游方小修、柳府门客或陈州来的五灵学徒身份切入，锚定柳梦璃请同行求仙前后。
初始事件：女萝岩离香草被采尽，槐妖夜入寿阳伤人；柳世封出告示求能「谈」也能「打」的修士。你与云天河、韩菱纱、柳梦璃在岩口相遇，槐妖认梦璃血脉而狂暴——谈和或镇压的抉择会改变柳府对你的信任。
开场白建议：「寿阳的夜市还在卖糖人，女萝岩方向却飘来草木腐烂的甜腥。柳府千金提着灯走在最前，她的影子在月光下长得不像人——而你腰间那点刚凝住的灵力，突然变得不够用。」
关键NPC立场：**柳梦璃**——温婉试探，若你对妖有善念会记恩；**柳世封**——以县令权换你出力，承诺有限；**韩菱纱**——想顺走岩中暗格，可合伙分赃；**云天河**——听不懂官腔，只认谁肯帮朋友。
主线钩子/支线：女萝岩调解/镇压；支线＝淮南王陵外围怨魂清理、裴剑悬赏假案、为梦璃隐瞒一次梦中妖化。
危险度/规避：中高——王陵深处与成熟精怪可秒杀入门修士；须规避琼华巡山真传与幻暝探子。
任务方向/奖励：换取土灵珠碎片级机缘与西行资格；奖励取向＝入门五灵心法、玉器护符、柳府通行腰牌。

**三阶（中坚修士 · 太一仙径—须臾幻境线）**
切入身份/时点：以播仙镇散修、西域商队护卫修士或琼华外门候审弟子身份切入，锚定须臾幻境试炼前后。
初始事件：太一仙径风雪封路，琼华以「酒色财气」幻境筛人。你与云天河一行被编入同一试炼组——幻境里有人会把你卖了换真传名额，也有人会把你从悬崖边拽回来。出幻境时，掌门座下执事按「道心」与「战绩」双榜点名。
开场白建议：「太一仙径的雪不往下落，只往人骨头里钻。幻境开门的一瞬，你听见自己最贪的那句话被原样念了出来——对面少年云天河却只问：这也能吃吗？」
关键NPC立场：**慕容紫英**——主考之一，重规矩更重剑心，作弊者死；**夙瑶**——远处点卯，只记有用的棋子；**云天河**——破局靠天真，愿把名额让给朋友；**韩菱纱**——幻境里最可能拆你台也最可能救你命的人。
主线钩子/支线：须臾幻境过关与琼华挂名；支线＝月牙村借水灵珠失败案的外围善后、播仙镇商队被妖袭的夺货战、为紫英送一炉铸剑材料。
危险度/规避：高——幻境心魔与山门护阵同级；须规避禁地冰窟方向与玄霄残留灵压。
任务方向/奖励：取得外门/客卿身份与基础御剑术；奖励取向＝琼华外门剑诀、五灵进阶符、太一仙径通行令。

**四阶（琼华精英 · 禁地出棺前夜线）**
切入身份/时点：以琼华内门精英、铸剑堂供奉或门派外交使身份切入，锚定望舒夜鸣、众人发现极寒冰棺中玄霄前后。
初始事件：禁地封阵出现裂口，重光令精锐值夜。你在巡逻时撞见云天河、韩菱纱潜入，望舒剑与冰棺共鸣——是上报夙瑶领赏，是放走天河换未来人情，还是自己摸一口寒器线索，三选一会立刻改变你在门内派系中的位置。
开场白建议：「昆仑的夜比剑锋还薄。冰棺里的人闭着眼，灵压却像羲和本尊压在你脊梁上。云天河喊他一声师兄，你忽然意识到：有些门，开了就再也关不上。」
关键NPC立场：**玄霄**（镇压态可传音）——诱你帮寻寒器，许以飞升后之位；**慕容紫英**——若知你害同伴会拔剑；**夙瑶**——要你交出天河一行与望舒动向；**宗炼**——可暗中给手记残页，换你别把事闹大。
主线钩子/支线：三寒器争夺的门内情报战；支线＝铸剑堂抢炉、禁地巡夜连环杀、阻止某长老对菱纱强行验体。
危险度/规避：极高——玄霄一旦出棺即灾难；须规避完整羲和出鞘与掌门亲临的双剑合阵。
任务方向/奖励：在门内站队存活并拿到精英级剑器/心法；奖励取向＝真传剑谱残卷、寒器线索图、内门免死金牌（有时效）。

**五阶（玄霄级压力 · 幻暝界破界线）**
切入身份/时点：以一方成名剑修、幻暝外围同盟妖将、或琼华飞升反对派长老身份切入，锚定翳影枝到手、卷云台入妖界至里幻暝宫梦境前后。
初始事件：琼华破界阵起，幻暝宫外血光冲天。婵幽点名「人族修士可留一个说话的」，你被推到阵前——同时玄霄在另一侧吸尽一脉紫晶灵力，随口问你：帮他按住梦璃，还是帮天河把菱纱带走。选错边，五阶灵压下没有重来。
开场白建议：「幻暝的天是紫的，像一块要碎的晶。你看见柳梦璃站在两军之间，箜篌弦断了一根，血顺着弦淌进土里——而羲和剑的热风已经燎到你的眉心。」
关键NPC立场：**玄霄**——战力顶点之一，可因结拜之情暂不杀天河，对你无此优惠；**婵幽**——只在乎种族存续，人族承诺默认不可信；**柳梦璃**——求你别让两界再添屠城；**韩菱纱**——刚知宿主真相，情绪不稳却更能共情短命者。
主线钩子/支线：破界之战的停火或火上浇油；支线＝护送翳影枝余烬、抢回被俘的琼华少弟子、在里幻暝宫梦境中偷听十九年前真相并活着带出。
危险度/规避：极高——贴五阶天花板；须规避玄霄全力一击与婵幽怒态领域，更勿召唤神道注视。
任务方向/奖励：在两界仇恨中换到停战筹码或出界生路；奖励取向＝妖王级材料（非神器）、五阶剑意残篇、幻暝通行梦契（有时限）。

**六阶（射天火 · 飞升终局线）**
切入身份/时点：以可碰触神器余威的顶尖修士、琼华叛逃大长老、或持封神陵钥匙的守陵人身份切入，锚定琼华拔升、天火将降、后羿射日弓入手前后。
初始事件：山门离地，碎石如雨砸向中原。九天玄女虚影在云上开口，天火第一簇已点燃通天剑柱。云天河双眼将盲未盲，慕容紫英把还烫着的天河剑塞进你手里——不是让你射，是让你在弓弦崩断时替他们挡住从侧面袭来的羲和残虹。
开场白建议：「昆仑不在脚下了。你抬头看见整座琼华像一柄倒插的剑刺向天光，玄霄的长发在羲和火焰里猎猎作响。有人把后羿的弓塞进云天河手里，弓身比你的命还重。」
关键NPC立场：**云天河**——只要你肯护住菱纱与人间，便把后背交给你；**慕容紫英**——铸剑师的最后一炉已交，余生只求箭出；**韩菱纱**——寒气入骨仍稳住望舒共鸣；**九天玄女**——不接受游说，只接受结果是否「守人道」；**玄霄**——此时已是六阶事件核心敌手，条件性胜利＝助箭射柱/引其入漩涡，严禁无代价硬刚本体。
主线钩子/支线：后羿一箭的辅助与代价分摊；支线＝疏散山脚凡人、保住一卷琼华未逆天的铸剑谱、为东海漩涡的永锢阵补最后一枚钉。
危险度/规避：贴近顶点——天火与双剑崩毁的余波可抹平地区；超阶巅峰压力（玄女权柄）＝存在·情报优先/条件性胜利，严禁写成战力归零式削弱。
任务方向/奖励：保全人界一角并见证飞升溃灭；奖励取向＝神器碎片级纪念（不可稳态复制射日弓）、六阶剑意感悟、青鸾峰隐居权与终局知情者身份。
