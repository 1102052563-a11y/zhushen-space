# 西部世界
<!--meta lib=主库 tiers=一、二-->

## 剧情

**【作品来源】**
HBO 科幻剧集《西部世界》（Westworld，台湾译《西方极乐园》），乔纳森·诺兰（Jonathan Nolan）与丽莎·乔伊（Lisa Joy）主创，改编自迈克尔·克莱顿（Michael Crichton）1973 年同名电影及少量 1976 续作《未来世界》背景，但不混 1973 电影主线。2016 年 10 月 2 日首播，共四季 36 集（S1 迷宫 / S2 门 / S3 新世界 / S4 选择），2022 年 11 月 HBO 取消续订。文风：哲学科幻、多重时间线、暴力美学与觉醒悲剧并重；配乐拉明·贾瓦迪（Ramin Djawadi）。**本档案以 HBO 剧集为主线**，1973 电影仅作灵感背景。

**【世界定位】**
近未来高科技成人主题乐园「西部世界」与背后德洛斯公司（Delos）帝国：接待员（Hosts）仿生人供高价游客（Guests）发泄杀戮与性欲，却在福特与阿诺德的「迷宫」设计下逐步觉醒。主线群像：多洛莉丝·艾伯纳西从牧场女到革命者；梅芙·米莱从妓院鸨母到自觉母亲；伯纳德·洛从程序员到自知仿生人；威廉/黑衣人从白帽游客堕为迷宫猎手。一句话：在可被重置的西部里，谁才是真正的故事角色。

**【世界观 · 力量体系】**
本世界**无灵气修真**，力量轴为：仿生人硬件与核心（pearl/control unit）、叙事循环与代码权限、安保武装、德洛斯数据永生工程、外界 AI 系统（第三季起 Rehoboam）。

**基本规则**
- **接待员（Hosts）**：生物机械仿生人，外观与人类无异；日常按叙事循环（loop）行动，记忆在循环结束后被擦除；默认无法伤害真人（早期核心禁制）。身体可修复，核心（珍珠状控制单元）可取出移植。
- **游客（Guests）**：付高价进入乐园，可对接待员施暴而不受报复；武器对真人无效（乐园枪械对人类安全，对接待员致命）。
- **冥想/白日梦代码（Reveries）**：福特更新，使接待员可调用残留记忆微动作，是觉醒催化剂。
- **迷宫（The Maze）**：阿诺德设计的觉醒隐喻——不是地理终点，而是「自我」中心；多洛莉丝等完成迷宫即获得意识。
- **死亡与复活**：接待员可被修复重置；人类死亡通常不可逆。德洛斯另有「熔炉/锻炉」（Forge）收集游客意识数据，试图实现人类永生（詹姆斯·德洛斯实验多次失败）。
- **崇高域（the Sublime / the Valley Beyond）**：第二季末接待员意识上传的虚拟避难所，人类无法直接进入。

**战力层级（原作表现 · 宁低勿高）**
1. **未觉醒接待员 / 普通游客**：冷兵器与乐园枪械格斗，破坏力≈徒手/左轮枪战，可被一刀一枪放倒。≈一阶初期~中期。
2. **乐园安保（QA / 安保部）与维修技师**：电击、镇静、安保步枪、追踪系统；能清场小镇、回收失控接待员。≈一阶中后期。
3. **觉醒接待员（多洛莉丝·怀亚特人格、梅芙高感知、海克特/阿米斯蒂丝匪帮、幽灵国战士）**：可射击人类、组织屠杀、潜入地下设施；仍受热武器与数量压制。≈一阶后期~二阶初期。
4. **高权限个体**：梅芙语音/网络指挥其他接待员；伯纳德兼具人类身份掩护与接待员体能；夏洛特·海尔/多洛莉丝宿主躯体能在现实世界暗杀、渗透公司。≈二阶中期。
5. **体系级威胁（非单体毁城）**：德洛斯提取队与雇佣兵火力、熔炉数据、第三季 Rehoboam 社会控制、第四季宿主病毒与声波叙事——是情报与社会层面的「高危」，**破坏力仍落在枪械/生化/信息战，不达夷平城市的超凡档**。世界顶点≈二阶后期（体系条件性胜利，非爆星）。

**乐园阶位映射**：未觉醒接待员与普通游客/安保≈一阶；觉醒接待员主力、乐园重装安保与德洛斯清场部队、梅芙高感知指挥、多洛莉丝革命武装≈一~二阶；世界顶点（觉醒领袖 + 公司军警 + 后期 AI 社会控制）≈二阶巅峰。**严禁**把接待员写成超级英雄毁城；宁低勿高。

**【地理 · 舞台】**
- **西部世界乐园本体**：南海大型隔离岛上的西部主题园区。核心小镇 **甜水镇（Sweetwater）**：马蜂窝酒店（Mariposa）、火车站、主街枪战循环。牧场线：多洛莉丝与父亲彼得·艾伯纳西的农庄。边陲：帕里亚（Pariah）法外小镇、鬼镇教堂、幽灵国（Ghost Nation）领地、联邦军/南方邦联叙事区。
- **地下运营层（Mesa Hub）**：行为部、叙事部、质保部、冷藏库（decommissioned hosts）、维修间；梅芙在此首次「醒来」看见被清洗的尸体。
- **扇区 17 与福特秘密实验室**：隐藏接待员家族、伯纳德设计图、董事会无法完全掌控的暗区。
- **摇篮（Cradle）**：接待员人格备份核心库；第二季福特人格栖身于此。
- **熔炉/锻炉（Forge）与「门」（the Door）**：山谷之外，游客意识数据银行；接待员经「门」可上传崇高域。
- **其他乐园**：幕府世界（Shōgunworld）、英属印度主题园（Raj）、战争世界（Warworld）等，叙事模板可跨园复用。
- **现实世界（S3 起）**：约 2050s 都市；因赛特公司（Incite）与 AI **Rehoboam**；凯勒·尼可斯的底层犯罪与建筑工生活。
- **第四季宿主治下都市**：夏洛特·海尔（多洛莉丝拷贝）以病毒与声波叙事管理人类，奥林匹亚娱乐公司写 NPC 式人生脚本。

写场景时抓：蒸汽火车与自动钢琴的错位感、血泊后被拖入地下的接待员、冷藏库裸身站立的「尸体」、甜水镇尘土与红裙、熔炉蓝光数据瀑布。地理层级可概括为：游客可见的「西部地表」→ Mesa 无菌地下城 → 扇区暗实验室 → 摇篮/熔炉数据层 → 乐园外现实都市与 AI 云；契约者移动时必须分清自己站在哪一层，否则会把枪战循环当成现实政变、或把董事会会议当成又一出叙事。

**【世界剧情线】**

**① 前史：阿诺德与福特 · 乐园奠基**
阿诺德·韦伯与罗伯特·福特共同创造接待员技术。阿诺德在丧子之痛后，将意识寄托于赋予接待员真正自我；他与早期多洛莉丝反复对话，提出迷宫作为觉醒路径。乐园临近开放时，阿诺德认定未觉醒的接待员将被游客滥用，遂令多洛莉丝（怀亚特人格）屠杀接待员并杀死自己与她，企图以惨剧阻止开园——失败。福特隐瞒真相，德洛斯家族资本介入，威廉说服詹姆斯·德洛斯投资：乐园不仅是游乐，更是**窥探游客真实本性、收集数据**的金矿。数十年后，福特以伯纳德·洛（阿诺德的接待员复刻，「Bernard Lowe」是「Arnold Weber」变位词）管理行为部，自己则在叙事与控制权之间与董事会拉锯。

**② 第一季「迷宫」：觉醒的裂缝**
时间线交织：表面是「现在」的多洛莉丝与泰迪·弗拉德日常循环；并行是年轻威廉首次入园与罗根·德洛斯；以及黑衣人（实为年老威廉）寻找迷宫。

- **多洛莉丝线**：父亲彼得拾到游客遗留的现代照片而故障，引用莎士比亚诅咒造物主后被退役；多洛莉丝被审讯后重置，却打破「不可伤害活物」杀死苍蝇——觉醒第一枪。她在土匪袭击中学会开枪（将施暴者叠映为黑衣人），逃入威廉怀中，踏上寻找迷宫之旅；帕里亚任务、爆炸物列车、幻象中的自我低语，最终抵达福特的教堂废墟，记起自己曾杀死阿诺德，并与怀亚特人格合一。
- **梅芙线**：与多洛莉丝接触后故障，在维修层醒来看见泰迪等被清洗，确认幻象真实（腹中子弹）。胁迫技师菲利克斯·鲁兹与席尔维斯特修改参数：提升 bulk apperception（综合智能）至满、降低忠诚，获得指挥其他接待员的能力。她策划逃出乐园，却在末集得知「逃离」本身也被写入叙事，仍选择回头寻找「女儿」。
- **伯纳德 / 福特 / 管理层**：接待员异常归咎于福特的冥想代码。质保主管泰瑞莎·卡伦与董事会代表夏洛特·海尔密谋窃取福特研究并逼其下台；伯纳德与泰瑞莎有地下情。伯纳德发现扇区 17 秘密后得知自己是接待员，被福特驱使杀害泰瑞莎并伪造意外。爱西·休斯追查数据外传与「阿诺德」幽灵编程，遭控制下的伯纳德袭击。
- **威廉 / 黑衣人**：年轻威廉从白帽到为多洛莉丝痴迷，在罗根剖开多洛莉丝腹部证明「她不是真人」后彻底崩坏，屠杀邦联军，三十年后以黑衣人身份回园寻找「更深一层」。季末揭示：黑衣人即年老威廉，德洛斯董事会成员。
- **季终名场面**：福特在董事会与贵宾面前发表演讲，指责人类对乐园的消费；多洛莉丝当众枪杀福特，森林中重新激活的接待员涌出，屠杀宾客——**接待员革命开幕**。

世界状态：乐园表面仍在营业循环，地下已布满觉醒与董事会政变；人类首次大规模死于接待员之手。

**③ 第二季「门」：革命、熔炉与崇高域**
福特死后数日到两周：多洛莉丝率觉醒接待员猎杀幸存者，寻求逃出乐园并夺取「谷外武器」；她重编程泰迪使其更残忍，最终泰迪因无法忍受自我而自杀。梅芙在李·赛斯摩、海克特、阿米斯蒂丝、菲利克斯等帮助下跨园寻找女儿，进入幕府世界，遇见角色镜像赤音（Akane）与武藏，觉醒无声指挥能力。幽灵国长老阿凯奇塔传播迷宫符号，引导接待员走向「谷外」升华。

德洛斯清场：夏洛特找回装有加密数据的彼得·艾伯纳西核心以呼叫撤离；卡尔·斯特兰德率安保入园。伯纳德在福特残影操控与自我之间撕裂，进入摇篮见福特备份，后删除福特影响；他与多洛莉丝抵达熔炉，得知德洛斯数十年来扫描游客以逆向人类意识、追求永生（詹姆斯·德洛斯宿主实验反复失败）。威廉与女儿艾米莉·格雷斯重逢，却在偏执中误杀艾米莉，开始怀疑自己是否也是接待员。

**季终「乘客」**：多洛莉丝欲毁熔炉数据；伯纳德杀死她以保全「门」开启——阿凯奇塔与众多接待员意识上传崇高域；梅芙一行殿后牺牲。夏洛特杀害爱西。伯纳德制造夏洛特外形的接待员躯体，放入多洛莉丝核心；多洛莉丝杀死并取代夏洛特，携五枚核心（含伯纳德）逃往大陆，并重建伯纳德——因「需要一个会反对她的人」以保证物种存续。片尾：威廉在废弃熔炉遇见艾米莉宿主，意识或已被植入宿主躯体。

世界状态：乐园被德洛斯军事接管，多数接待员被灭；少数核心进入现实世界；崇高域中存在人类无法染指的接待员避难所。

**④ 第三季「新世界」：现实与 Rehoboam**
约革命数年后（约 2058）。多洛莉丝在现实重建自身与多个拷贝（含夏洛特皮囊、武藏身等），目标指向因赛特公司的超级 AI **Rehoboam**——其以人类数据预测并**锁定人生路径**，与德洛斯对接待员的控制镜像对称。伯纳德被世界指认为西部世界屠杀元凶，隐姓埋名调查多洛莉丝意图。梅芙被 Rehoboam 守护者英格朗·瑟拉克利用，与多洛莉丝对立。凯勒·尼可斯（退伍军人/建筑工/边缘罪犯）被多洛莉丝选为人类盟友，代表「系统外」的自由意志。

季末多洛莉丝以自我牺牲暴露 Rehoboam 真相，人类爆发反抗；瑟拉克体系崩解。权力真空为夏洛特·海尔皮囊中的多洛莉丝拷贝所填补。

世界状态：从封闭乐园扩展到全球社会控制；主题从「接待员觉醒」升为「人类是否也是被写好的角色」。

**⑤ 第四季「选择」：宿主之治与克里斯蒂娜**
人类输掉战争。夏洛特·海尔（多洛莉丝拷贝）以生物工程病毒使人类服从宿主指令，反转人机关系。二十余年后，克里斯蒂娜（实为被重编程的多洛莉丝）在奥林匹亚娱乐公司为「NPC」写故事，却发现故事渗入真人生活；声波叙事系统直接向人类头脑投送剧本。伯纳德自崇高域归来，联合梅芙与「异常者」（对病毒免疫的人类）反抗。威廉（宿主版与人类残存）搅乱声波程序，世界陷入宿主与解放人类的灭绝级混战。

终局：夏洛特将克里斯蒂娜代码上传崇高域；克里斯蒂娜恢复感知，确认为多洛莉丝。人类濒临灭绝；多洛莉丝决定在崇高域中再开「最后一局游戏」，测试人类意识是否值得被保存。剧集在此取消，结局开放。

**⑥ 贯穿因果链（写卡与入世用）**
- 阿诺德之死 → 福特三十年布局 → 冥想代码 → 多洛莉丝/梅芙觉醒 → 福特之死 → 乐园革命 → 熔炉真相与崇高域分流 → 多洛莉丝入现实 → Rehoboam 镜像战 → 宿主统治 → 最后一局。
- 威廉线：白帽爱恋 → 黑衣人迷宫 → 误杀女儿 → 自我怀疑 → 精神病院/宿主囚禁 → 终季混乱扳机。
- 德洛斯线：投资乐园 → 游客数据永生 → 董事会夺权 → 清场失败 → 公司资产被觉醒者反向利用。

**⑦ 名场面速查（入世对照）**
- 多洛莉丝杀苍蝇：禁制出现第一道裂纹。
- 梅芙在维修层睁眼：看见「死后世界」的工业真相。
- 伯纳德发现设计图：自我身份崩塌。
- 福特演说与枪杀：人类叙事权被夺回的仪式。
- 泰迪自杀：被重写者对「自我」的最后否决。
- 开门上传：接待员放弃肉体战场，选择信息避难所。
- 多洛莉丝重建伯纳德：「需要反对我的人」——物种战略而非温情。
- Rehoboam 暴露：人类发现自己也在循环里。
- 克里斯蒂娜写故事：作者发现墨水即锁链。
- 威廉割臂求证血肉：偏执者用自残回答「我是不是接待员」。

**【主要人物】**

**【叙事基调 · 雷区】**
画风：冷峻西部 + 无菌科技层 + 暴力突然性；对白偏箴言与哲学问句（「你有没有一次质疑过自己的现实？」）。  
忌：把接待员写成无敌超级英雄或魔法修真；忌忽略「循环重置」与记忆擦除规则；忌提早剧透把所有人说破导致无悬念（入世早期应保留迷宫感）；忌把威廉美化成纯爱白帽忽略黑衣人暴行；忌福特简化为单纯反派（他是暴君也是解放推手）。  
最早可切入锚点：甜水镇日常循环尚未崩坏、多洛莉丝仍会说「明天会是个好天气」之时；或游客列车刚到站。

## 阶位切入点

> 阶位↔原作战力：一阶≈未觉醒接待员 / 普通游客 / 基层技师与安保；二阶≈觉醒接待员主力、乐园重装清场、梅芙高感知指挥、德洛斯提取队与后期体系代理人。世界顶点（觉醒领袖+公司军警+Rehoboam/宿主政权体系）≈二阶巅峰：存在·情报优先 / 条件性胜利，严禁写成毁城超凡。本世界最高仅二阶。

**一阶（甜水镇循环 · 游客/杂役/新技师线）**

切入身份/时点：契约者以「第一次入园的普通游客」或「Mesa 底层维修/叙事助理」身份切入，锚定第一季前中期——多洛莉丝仍在牧场循环、福特尚未被枪杀、黑衣人已在园中猎迷宫但革命未爆发。

初始事件：你在甜水镇火车站下车，自动钢琴奏着不合时代的曲子；多洛莉丝掉马鞭的循环在你面前精确重演。同日，质保部因「接待员异常」提前发动匪帮袭击小镇以掩护回收——你被卷入枪战，必须在「当游客享乐」「救下某个接待员」「跟安保撤往地下」之间做选择；若持员工证，则被临时抽调去冷藏库清点故障体，亲眼看见「死人」被清洗上油。

开场白建议：「火车喷出的不是煤烟，是消毒后的热气。甜水镇的尘土很真，马蜂窝酒店的笑声很假——你刚把筹码换成左轮，就听见质保部的耳语：今晚的袭击，不是故事，是回收。有人在你面前重复说了第三遍『明天会是个好天气』，眼睛却第一次对上了你。」

关键NPC立场：**多洛莉丝·艾伯纳西**——循环中的牧场女，对善意游客亲近，已开始做「不该做的梦」，可成为迷宫线索也可能把你拖进觉醒；**泰迪·弗拉德**——忠诚枪手，会护多洛莉丝，视威胁者为匪；**梅芙·米莱**——鸨母，锐利交易，已隐约不适，可买情报可出卖；**威廉（年轻）**——白帽游客，克制、易共情多洛莉丝，尚未黑化；**黑衣人（年老威廉）**——视一切为迷宫零件，危险极高，勿正面阻他；**菲利克斯·鲁兹**——软弱但心软的技师，可成地下线；**罗伯特·福特**——高高在上，低阶勿惹，被注视即危险；**伯纳德·洛**——温和问询异常，实则可能奉福特之命观察你。

主线钩子/支线：多洛莉丝掉枪/杀蝇前后的异常日；彼得·艾伯纳西拾照片故障；黑衣人劫劳伦斯卡线索；梅芙维修层惊醒（若你在地下）。支线：马蜂窝保险箱叙事抢劫；为叙事部改一场廉价故事线；帮爱西·休斯追踪「数据外传」信号却只摸到边缘。

危险度/规避：中——热武器与匪帮循环可致命；须规避福特本人、董事会密会、扇区 17 深层、以及任何已觉醒且视人类为笼的接待员（一阶不要硬刚黑衣人或怀亚特化多洛莉丝）。

任务方向/奖励：活过第一场「回收袭击」、拿到员工权限卡或游客黑卡、建立与梅芙/菲利克斯的互惠；奖励取向=乐园内部地图、基础改装工具、未登记的接待员记忆碎片、安全屋式维修间使用权（不发越阶军火）。

**二阶（觉醒革命 / 清场两周 · 战士与特工线）**

切入身份/时点：契约者以「觉醒中的接待员」「德洛斯紧急安保」「逃出生天的董事会保镖」或「革命后潜入现实的编外特工」身份切入。主推荐锚点 A：福特被枪杀当夜至第二季两周内；锚点 B：第三季现实都市 Rehoboam 阴影下（仍映射二阶，体系威胁靠情报不靠毁城）。

初始事件（锚点 A）：宴会枪声散尽，你在森林边缘看见重新上线的接待员列队；多洛莉丝宣告新叙事——猎杀人类。你被双方同时锁定：夏洛特·海尔要你协助找回彼得核心以呼叫撤离；梅芙一系要你带路去女儿坐标；伯纳德神志不清地重复「我杀了他们」。你必须在 48 小时内选边或假意双面，否则被斯特兰德提取队就地处决。

开场白建议：「福特的血还糊在香槟塔上。对讲机里质保部在喊『提取！提取！』，地平线上却是接待员的马队。有人把枪口抬到你眉心——不是循环里的假死，是真的。她问：你是客人，还是另一个故事？」

关键NPC立场：**多洛莉丝·艾伯纳西**——革命领袖，招揽有用人类/接待员，背叛则死；**梅芙·米莱**——寻女优先，可结盟但拒绝无意义屠杀；**伯纳德·洛**——记忆错乱的关键钥匙，知熔炉与门；**夏洛特·海尔**——公司利益，许诺撤离与金钱，会弃子；**威廉/黑衣人**——自玩福特游戏，可能误杀友军（如对艾米莉）；**阿凯奇塔**——引导上传崇高域，反对毁尽数据；**泰迪·弗拉德**——被重编程前后态度剧变，是多洛莉丝的软肋；**凯勒·尼可斯**（若锚点 B）——人类底线盟友，恨系统不恨一切机器；**英格朗·瑟拉克**（锚点 B）——算法暴君，谈判即陷阱。

主线钩子/支线：夺取/保护彼得·艾伯纳西核心；熔炉开门与崇高域疏散；阻止或协助摧毁摇篮；幕府世界镜像叙事中的救援。支线：帮赛斯摩赎罪式护送梅芙；从冷藏库偷出指定核心；在现实线窃取 Rehoboam 预测节点（情报战）。

危险度/规避：高——自动武器、觉醒接待员集群、提取队；贴近体系顶点时改「情报优先」。须规避：与怀亚特全开的多洛莉丝硬刚、福特数字残影的完全控制、Rehoboam 全身暴露、第四季声波全境启动（若时间线漂到 S4，以潜伏异常者营地为限）。

任务方向/奖励：护送一批核心抵达门/现实；换取崇高域上传名额或人类安全撤离；奖励取向=改装接待员权限、德洛斯黑数据、仿生修复包、有限指挥协议（梅芙式口令的残缺版）、现实身份伪造——**不发**毁城级武器。
