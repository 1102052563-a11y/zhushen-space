## 剧情

**【作品来源】**
《DICE》（韩文原题《다이스》，英文全称 *DICE: The Cube That Changes Everything*，中文常称**DICE骰子** / **命运骰子**）为韩国漫画家**윤현석（Yun Hyun-seok / 尹贤锡）**所作彩色网络漫画。2013年起于**Naver Webtoon**连载，2021年完结，全约**387话**（Anime-Planet 标 Ch.387；LINE Webtoon 英区 title_no=64，状态 COMPLETED）。文风：校园霸凌→RPG骰子系统→欲望与贪婪主题的心理·动作·生死游戏；前期学校日常与逆袭，中期公会/A级战与「最终骰」争夺，后期对局主**X**与「无影」线、苦乐参半终局。媒介以网漫正典为准；搜笔趣阁站内检索**未收录**完整中文译本，本档案以 Webtoon 作品页、Anime-Planet、TV Tropes 等可溯源公开资料交叉核对，**禁止编造原作未写出的人物与结局**。

**【世界定位】**
当代韩国都市校园为表层舞台：普通高中生的外貌、成绩、社交被「骰子」量化成可加点的属性。主角**동준（Dongtae / 董俊，亦译东俊）**原是学校「面包跑腿（bread shuttle）」——被霸凌、替人跑腿买东西的底层。转学生**태빈（Taebin / 太彬）**以完美外貌与能力登场，董俊跟踪后窃得一枚可改变命运的**骰子（Dice）**，自此被神一般的游戏主**X**拉入任务系统。一句话给AI：这是「被欺凌少年用骰子改写人生，却在欲望战争中直面系统之神与最终骰」的校园·生死游戏档案。

**【世界观 · 力量体系】**
世界表层是现实韩国校园与都市；超凡层由**骰子**与**X的游戏规则**支配。骰子可投入指定属性（外貌、力量、敏捷、智力、耐久等），把人改造成「游戏角色」；持骰者称**Dicer（骰子使 / 玩家）**。X通过手机式界面发布**任务（Quest）**，奖励更多骰子；欲望越强、越愿越界，任务与回报往往越丰。死亡时骰子会从体内爆出，成为他人抢夺的源头——**병철（Byeongcheol / 炳哲）**之死即校内骰子大扩散的导火索。

关键机制：
1. **属性加点**：骰子消耗后永久或阶段性改写身体与能力；董俊以**敏捷**拉满著称，形成高速高输出的脆弱速度型。
2. **任务与欲望**：X按「Desire（欲望）」分配任务；缺乏欲望者几乎无任务，好勇斗狠者反而吃香。完成超额可额外奖励。
3. **A级骰 / A-Ranker**：特殊能力骰。常见A技包括——**时间暂停（Time Pause）**（董俊招牌，发动时周遭单色）、**瞬移（Teleport）**（太彬）、**隐身（Cloaking）**（대웅 / Daewoong / 大雄，沾水失效）、**漂浮/飞行（Levitation）**、**念动力（Psychokinesis）**、**洞察/千里眼（Clairvoyant）**（미오 / Mio / 美乌：信息框、战斗最优解、后期可编辑状态）。收集多枚A骰可成**A级玩家**；X曾以「集齐6枚A骰许愿」推动PVP。
4. **古代骰 / 类S级道具**：董俊在X的随机机中抽得**古代骰（Ancient Die）**，附时间系扩展：**时间回溯（Time Rewind）**、**时间跳跃（Time Skip）**、**时间扭曲（Time Distortion）**、**时间切割（Time Cutter）**等；Time Cutter 可冻结受伤目标形成连控，代价常为生命力。
5. **最终骰（Final Die）**：可实现愿望的核心麦高芬，原在X手中；实为有自我意识的「欲望集合体」，内含死者灵魂直至消散。能量有限，用尽可缓慢恢复——终局被X彻底打碎后，骰子系统消亡。
6. **GM规则枷锁**：X作为Game Master，直接杀人受规则限制，常把死亡包装成「转化」、或借玩家互杀；正式挑战中可立惩罚规则。董俊利用「GM不能随意直接杀玩家」的漏洞与文字游戏反制。
7. **排名与S级传闻**：Mooyoung 线涉及「S级诞生近神」的情报；**무영（Mooyoung / 无影）**以夺舍、多灵魂融合、阴影武器与经验碾压成为现世级威胁。
8. **死亡与记忆**：骰子系统崩坏后，多数人失去相关记忆；董俊保留记忆，世界回到「近似原点」的日常。

战力阶层（按破坏力表现，宁低勿高）：
- **普通人 / 无骰学生**：徒手格斗、校园霸凌、刀具威胁可致命。
- **低点数Dicer**：碎砖、打趴壮汉、单挑小帮派；热武器仍危险。
- **高属性校园级**：裂门掀车、乱群穿插；隐身/念力初显。
- **A级玩家（单A技）**：时间停/瞬移/飞行级，可屠校级冲突、掀建筑局部。
- **多A / 古代骰强化 / 无影级**：跨街区追杀、团体战核心；可致命伤秒杀A级。
- **X（GM态）+ 最终骰本体**：世界顶点侧。玩具军团、规则改写、空间舞台；低阶切入只做**存在·情报优先 / 条件性胜利**。X即使受GM条款约束不能随意「直接抹杀玩家」，本体威胁与规则权柄仍在——禁止把顶点写成「因限制而毫无战力」。

乐园阶位映射（对照《阶位战力图鉴》，宁低勿高）：无骰凡人~低点数Dicer≈**一阶**；校园稳定Dicer / 公会打手≈**二阶**；A级玩家（单能力成型）≈**三阶**；多A级、古代骰董俊巅峰战、无影现世屠杀、X正式对局人间侧≈**四阶**。本世界清单最高**四阶**；X全盛GM与最终骰神性＝贴近顶点，四阶亦须规避硬刚本体。顶点忠于原作完结：最终骰被毁、系统消亡、苦乐参半收束。

补充成长曲线（同阶内细分，便于写任务难度）：刚得1~3枚骰、只敢加外貌与力量的新人≈一阶初中期；炳哲级「会查系统、会记笔记」的信息型≈一阶后期到二阶初；公会打手与隐身偷窥型大雄≈二阶中；单A时间停/瞬移/飞行≈三阶；洞察常开的美乌在信息战上可越半阶观感；无影与正式对局的X人间投影≈四阶顶。热武器与利刃对「只堆耐久不防穿刺」的Dicer仍有效——智恩的刀是设定提醒，不是笑话。

**【地理 · 舞台】**
- **董俊所在高中 / 教室与天台**：霸凌、「面包跑腿」、猎杀恩珠任务、公会争夺的主舞台。天台是跳楼、对峙、告白与崩溃的高频场景。
- **市区小巷与跟踪线**：董俊跟踪太彬发现骰子的起点；夜间任务、炳哲焚校企图的路径。
- **董俊公寓**：父母疏离后的独居点；美乌「过夜任务」与感情线关键地点。
- **女浴室排水管 / 校舍外墙**：炳哲坠落身亡处——骰子爆散、全校扩散的名场面地理锚。
- **医院**：教师春小峰（Chun Sobong）等相关生死、石顺与娜妮线、战后疗伤。
- **地铁与地下迷宫**：秀英一派围堵美乌等；都市战机动作。
- **公共浴场 / 随机机与蓝仙子（Blue Fairy）**：X的玩具关卡、古代骰抽取处。
- **游乐园 / 烟火约会点**：董俊与美乌正式感情高光。
- **X的竞技场 / 最终舞台**：过往地图拼贴的混合空间；最终游戏、hide-and-seek、愿望结算。
- **时间跳跃后的都市**：成人Dicer暴增、骰子神兄弟会（Brotherhood）、直播曝光「恐怖袭击」新闻化。

势力地盘：校园公会（8班家族→董俊公会→分裂）→校外A级猎手与黑暗兄弟→摩托车队 Team Charlie（崔秀英）→骰子神兄弟会→X的规则空间。写场景先定「是否已有骰子扩散 / 是否A级PVP / 是否X下场」。

**【世界剧情线】**

**阶段一 · 面包跑腿与第一枚骰子（开篇）**  
동준（董俊）在外貌、成绩、社交全面垫底，被현조（Hyunjo / 玄祖）等霸凌者役使跑腿买面包。转学生태빈（太彬）以「校园完美体」降临：外貌、运动、学业全优，并迅速接近董俊暗恋的은주（Eunju / 恩珠）。嫉妒与不甘驱使董俊跟踪太彬，撞见其收集神秘立方体——**骰子**。董俊偷得一枚，开始用骰子提升外貌与体能；X以孩童口吻的神之存在通过界面联络他，提供可选任务（道德光谱从灰色到漆黑）。一周内董俊外形大变，校园地位翻转。主题：改变外表容易，改变「被选中的欲望结构」很难。因果：第一枚骰=全书外挂开关；太彬=旧世代A级幸存者；X=真正庄家。

**阶段二 · 炳哲之死与骰子瘟疫（扩散转折）**  
董俊把骰子分给同为被欺凌者的병철（炳哲）。炳哲遭霸凌升级、精神崩溃，尝试用骰子复仇并接受「焚校」类极端任务；他曾试图黑入骰子系统并留下笔记。某夜X给董俊「去学校开窗救人」的任务，董俊因疲惫拒绝——炳哲悬挂在女厕窗外排水管上坠落身亡，体内骰子爆散，**全校学生捡拾**。名场面：坠落、骰爆、校园秩序崩坏的第一夜。因果：①董俊背负「见死不救」的原罪；②Dice从「个人外挂」变成「社会瘟疫」；③炳哲笔记成为早期情报源。

**阶段三 · 猎杀恩珠与A级觉醒（校园战争前哨）**  
恩珠拒绝使用捡到的骰子，触怒X。X向全校发布伤害/羞辱恩珠的限时任务；暗中又给董俊「保护她」的反向任务。董俊与太彬一度同盟守护，耗尽储备骰子硬抗一天。董俊因此获得**A级·时间暂停**。8班家族式Dicer帮派由대웅（大雄）坐庄，隐身、围殴、绑架企图层出；董俊击败大雄后试图把帮派改造成「共享骰子、禁止滥暴」的**公会（Guild）**。阶段结局：A级能力登场；公会理念与人性欲望第一次正面冲突。

**阶段四 · 公会崩坏、美乌与A级竞赛（中前期）**  
公会成员为骰子内讧；X减少合作向任务、增加互害任务。미오（美乌，原肥胖被嘲的「眼镜青春痘猪」）对改运前的董俊亦有好感，用骰子重塑外貌，完成「在董俊家过夜」任务后获**洞察（Clairvoyant）**。感情线：董俊心系恩珠，美乌单向深情，后成正式情侣却始终笼罩恩珠阴影。X公布：出现足够A级后开启**PVP**，先集齐规定枚数A骰者可向最终骰许愿。飞行者성철（Sungchul）、念力者길마（Gilma）等叛出；미주（Miju）等被杀，骰子再扩散。世界状态：学校已是小型战场，新闻开始出现「神秘暴力」。

**阶段五 · 恩珠成Dicer与意识形态分裂（中期）**  
恩珠在目睹杀戮与X的嘲笑后被迫成为Dicer，极强的「消灭骰子」欲望使她短时间内连夺多枚A技（念动、隐身进阶为幻象等），以极端手段猎杀A级。她与董俊形成**弥赛亚对决**：董俊想保留骰子但打倒系统（X）；恩珠想抹除骰子本身。太彬身体被무영（无影）侵蚀的真相渐露——无影曾灭杀上一代A级团队，仅留太彬，并夺舍其身以培养新A级再收割。Dawn等旧识登场。阶段主题：善意的两种互斥解法；「完美女神」恩珠的伪善指控与车祸伤疤、钢琴童年一并揭开。

**阶段六 · 战争任务（War Quest）与道德灰化（中后期）**  
X发动攻守阵营的**战争任务**：攻击方要伤害指定目标（含教师春小峰等），防守方保护。董俊慢性英雄病被X算计利用；教师김영숙（Kim Youngsuk）捡骰成恶师加入攻击方。春小峰最终不治。董俊开始接受「为守护而PK」；盟友채현（Che-Hyun）表面亲X、实则给防守方输送骰子。Hyunjo 在医院缺席数月后回归，被董俊所救后正式道歉、洗白。名场面：校内攻防、人质、任务文案的精确咬字（Exact Words）。

**阶段七 · 时间跳跃后：成人Dicer、兄弟会与悬赏（后期铺垫）**  
与X正面对撞后，董俊重伤失去几乎全部骰子，外貌回归「弱鸡」；美乌因X挑拨感情冷战/分手式疏远；全城悬赏主角团。时间跳跃后，有人公开直播骰子机制，**成人玩家**暴增；**骰子神兄弟会**以类天主教仪式崇拜X；崔秀英（游泳冠军因禁药丑闻转向骰子）率 Team Charlie；黑暗兄弟（Smart Brother / Strong Brother）以肤色歧视创伤驱动，愿望是「让全人类变黑」；정석순（Jeong Suksoon）与病弱나니（Nani）线展示骰子的医疗与杀戮双刃。董俊用古代骰时间系在绝境中挣扎求生。

**阶段八 · 对X的挑战、最终骰与捉迷藏终局**  
董俊集结残存A级与盟友，正式挑战X。X以玩具军团（火星王机器人、蓝仙子、巨型泰迪等）、分身、规则惩罚与公开处刑威胁镇压。最终骰显露自我意识：复活竞技场死者、迫使X「最后一次公平」。董俊洞察X童年创伤（序章被家暴的男孩即X）与「想被一起玩」的空洞，把最终对局定为**捉迷藏（Hide and Seek）**：按X自订规则——找不到人即负——全员选择「藏」而非「找」，X因童年阴影崩溃停赛，**整个骰子系统停摆**。X盛怒打碎最终骰；若仅耗尽电量本可恢复，打碎则永久。

**阶段九 · 愿望结算与苦乐参半结局**  
最终骰有限能量下，董俊尽量兑现众人愿望：太彬被从自杀时间点救下并转学重逢、秀英重返泳池、娜妮痊愈与石顺同住、无影安息、蔡贤因系统bug白嫖游戏、大雄得到荒诞苏丹梦、恩珠车祸相关生死线被改写（她自己也记不清原确认度）、지은（Jieun / 智恩）获得新身份后任董俊班教师、兄弟会领袖约翰仍瘫痪但教团变普通教会、美乌以强化身体复活并在结局挡开霸凌、X暗示过上普通家庭生活。骰子从未存在过，相关死亡被抹消；**唯董俊记得一切**。结局画面：他仍是面包跑腿，但这次选择站起来反抗，并重新与美乌、智恩、恩珠建立健康关系——**不是全能许愿的童话，而是带着记忆的再出发**。

**分卷细目与可介入节点**

**开篇卷细目**  
霸凌日常的微观压迫：扣钱、羞辱、强迫跑腿。太彬的「完美」是系统外挂的广告牌。跟踪夜是契约者最易「抢骰/被发现」的窗口——抢到骰=上X的观察名单。

**扩散卷细目**  
炳哲笔记=早期攻略书。拒绝夜间任务的选择是原罪节点：契约者若替董俊完成开窗，可能改写全校扩散，但X会换别的瘟疫触发器（原作逻辑上欲望总会找到出口）。

**猎杀恩珠卷细目**  
限时一天、全校敌对、双向任务。Exact Words：任务要求「伤害感情」而非单纯殴打——恩珠若不崩溃则任务不算。董俊A级时间停的诞生点。

**公会卷细目**  
共享理想 vs 贪婪。Gilma 念力地铁伏击、Sungchul 飞行在密闭空间的喜剧性失误、Miju 死亡与再扩散。美乌洞察常开、不耗骰的信息战优势。

**意识形态卷细目**  
恩珠三A猎杀、太彬/无影双人格、Dongtae 想「让所有人幸福」却被数据幽灵质问与恩珠有何不同。

**战争任务卷细目**  
攻守阵营、教师目标、恶师金、春小峰之死。Che-Hyun 双面运输。

**终局卷细目**  
古代骰抽取、公开处刑被古代骰救下、最终骰自醒、捉迷藏规则反噬、打碎最终骰、记忆清算与日常回归。

开场白建议：「警报不是消防铃，是任务倒计时。红名与蓝名在班牌上刷新，像把整栋教学楼做成了大逃杀服务器。你在三楼拐角撞见时间停止的余波——灰尘悬在半空，半秒后才落地。有人拉你袖口：选边，或者被两边一起猎。」

关键NPC立场：**董俊**（防守核心，要人别杀生又不得不杀）；**恩珠**（只取A骰，白名单极窄）；**美乌**（洞察扫你状态栏）；**蔡贤**（笑着卖骰两边赚）；**金永淑**（恶师，觉「有趣」）。

主线钩子/支线：主线——护送或刺杀指定教师目标（春小峰线）的蝴蝶效应。支线：帮娜妮换医；在无影阴影边缘只录影不交手；调解董俊与恩珠的一次屋顶谈判。

危险度/规避：危险度高。规避：**无影**本体交锋；X的「认输就放人」文字陷阱；对恩珠出示多枚A骰。

任务方向/奖励：A技熟练度、战争任务阵营勋章、一份「GM不能直杀」的规则笔记——不发最终骰权限。

**四阶（多A/古代骰/顶点阴影 · 对X与终局）**

切入身份/时点：多能力或古代骰外围协力者；锚定**董俊失骰低谷后的悬赏期**至**捉迷藏终局**之间。

初始事件：城市屏幕被劫持，X的笑声盖过广告。悬赏令上是董俊「弱鸡」模样的脸。随机机房的蓝仙子裂嘴笑：再转一次？远处竞技场像被儿童积木拼错的城市。

开场白建议：「你记得骰子该有重量，可当最终骰在地平线升起时，连空气都像欲望一样发粘。有人说许愿能改命，有人说许愿会吃人。童装尺码的神坐在王座上踢腿：『来玩啊。』——你若把这当成普通Boss战，会死在规则的标点符号里。」

关键NPC立场：**董俊**（偏执要让「所有人幸福」，需人提醒能量有限）；**X**（只对有趣的玩家给活路）；**无影**（收割强者）；**美乌**（生死边缘的洞察）；**最终骰（有意识）**（可对话的「欲望之海」）；**智恩**（刀子比演讲快）。

主线钩子/支线：主线——协助捉迷藏规则战（藏人、拖延、不让X找到「它」）；或在最终骰许愿窗口争取「抹除系统但保留一条人命」。支线：救出被兄弟会当祭品的低阶Dicer；阻止X公开处刑直播的信号塔。

危险度/规避：危险度极高/贴近顶点。**超阶巅峰：X与最终骰=存在·情报优先/条件性胜利**（规则漏洞、心理击溃、能量耗尽、文字契约）；X的GM权柄始终危险，禁止解释成「顶点毫无战力」。规避：独自闯最终竞技场中心；许愿时说「我要成为新的X」。

任务方向/奖励：终局站队成果、古代骰时间碎片（一次性）、记忆锚（你可能是少数记得骰子的人之一）——不发完整最终骰。
