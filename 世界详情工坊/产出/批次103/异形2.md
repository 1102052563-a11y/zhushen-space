## 剧情

**【作品来源】**
1986年美国科幻动作恐怖电影《Aliens》（中文通译《异形2》，港译《异形续集》），二十世纪福斯发行，Brandywine Productions 制作。导演兼编剧詹姆斯·卡梅隆（James Cameron），故事由卡梅隆与大卫·吉勒（David Giler）、沃尔特·希尔（Walter Hill）共同构思，角色源自丹·奥班农（Dan O'Bannon）与罗纳德·舒塞特（Ronald Shusett）开创的《异形》宇宙；监制盖尔·安妮·赫德（Gale Anne Hurd）；配乐詹姆斯·霍纳（James Horner）；生物特效斯坦·温斯顿（Stan Winston）。主演雪歌妮· Weaver（Sigourney Weaver）饰艾伦·里普利（Ellen Ripley），迈克尔·比恩（Michael Biehn）饰德韦恩·希克斯（Dwayne Hicks），保罗·莱瑟（Paul Reiser）饰卡特·伯克（Carter J. Burke），兰斯·亨利克森（Lance Henriksen）饰毕晓普（Bishop），凯莉·海恩（Carrie Henn）饰纽特（Newt），比尔·帕克斯顿（Bill Paxton）饰哈德森（Hudson）等。本片为系列第二部正传，时间设定在首作约五十七年之后，把「单船密闭恐怖」扩展为「殖民海军陆战队 vs 蜂巢」的战争片结构，并首次完整呈现异形女皇（Alien Queen）。文风：越战式空间步兵、企业黑深残、母性与创伤、酸血与弹药耗尽的绝望火力。本档案服务轮回乐园世界书常青读取，剧情与人名以公开电影、中英维基及角色表可核为准。

**【世界定位】**
公元2179年前后：艾伦·里普利在逃生艇中休眠五十七年被打捞，地球侧听证不信其关于 LV-426 卵室与异形的证词；而韦兰-尤坦尼公司（Weyland-Yutani）已在同一颗卫星建立哈德利希望殖民地（Hadley's Hope）并完成大气改造。殖民地失联后，公司代表卡特·伯克与殖民海军中尉斯科特·戈曼（Scott Gorman）说服里普利以顾问身份随陆战队乘苏拉科号（USS Sulaco）重返现场——她的条件是「消灭而非捕获」。契约者切入的是一场**「失联调查—蜂巢伏击—公司走私活体—处理站核爆—装载机对女皇」**的封闭杀局：主角线是 **Ellen Ripley** 与残存陆战队员、孤女 **Rebecca "Newt" Jorden** 在酸性与谎言之间求生。一句话：越战式太空步兵撞上完美寄生蜂巢，公司要样本，女皇要宿主，活着的人两边都不是甲方。

**【世界观 · 力量体系】**
本世界无灵气修仙，战力轴是：**人体素质 / 殖民海军制式武装 / 合成人操作权限 / 异形生物阶梯 / 工业外骨骼与环境杀（大气处理站堆芯）**。死亡规则极硬：无复活外挂；休眠舱只跨时间不跨死亡；酸血可蚀穿船体、装甲与人体；被茧化的殖民者是活体孵化器，胸破即死。异形无谈判、无收编（公司除外的妄想）；蜂巢以女皇为产卵中枢，战士协同伏击，黑暗与通风管是它们的主场。

强度阶梯（按实际破坏力，宁低勿高）：
1. **殖民者 / 无武装平民 / 听证室文职 / 底层勤务**：徒手或工业工具；一爪一酸可致命。乐园观感≈**一阶**。
2. **殖民海军陆战队列兵（持 M41A 脉冲步枪、手枪、有限弹药）**：可压制单只暴露目标、走廊交火；对群体与酸血仍极脆弱，弹药耗尽即崩。乐园观感≈**一~二阶**。
3. **陆战队精锐 / 智能枪手 / 士官指挥 / 喷火器手**：M56 智能枪、M240 喷火器、战术协同、APC 装甲运兵车支援；可清剿小规模异形，遇蜂巢仍被碾压。乐园观感≈**二~三阶**。
4. **动力装载机（Power Loader）操作者（场景限定）**：工业外骨骼短时对位女皇近战，非制式兵器，依赖货舱空间与气闸。乐园观感≈**三阶**（场景限定上限）。
5. **合成人 Bishop（A2 型）**：精密操作、遥控降落艇、自残式管道穿越；行为禁令「不会伤害人」，力量与耐伤超人类但非无敌。乐园观感≈**二~三阶**（权限与场景可抬到战术关键）。
6. **成熟异形战士（Xenomorph Warrior）**：撕开装甲人体、酸血蚀甲、通风管猎杀、群体伏击；单只可屠杀松散小队。乐园观感≈**四阶**。
7. **异形女皇 + 哈德利希望级蜂巢**：产卵器、茧化宿主、指挥蜂群、占据大气处理站下层；对殖民地=绝对主宰。乐园观感≈**五阶**（本片中段顶点生物）。
8. **大气处理站堆芯熔毁 / 核爆级自毁 + 女皇偷渡苏拉科号货舱**：非「个人战力」但决定终局，等价地区级灾难与舰内二次末日。乐园观感≈**七阶事件**（本片世界级复合灾难）。

乐园阶位映射（破坏力对齐，宁低勿高）：殖民点凡人·合同工·听证文职≈一阶；陆战队列兵·APC 乘员·普通安保≈二阶；智能枪手·士官·喷火器猎杀·动力装载机操作≈三阶；成熟异形战士单只/小队≈四阶；女皇+蜂巢压制殖民地≈五阶；（本世界不开放六阶切入）处理站核爆级自毁+公司活体回收全链路+女皇登舰≈七阶。世界顶点＝**女皇本体 + 蜂巢 + 公司「样本优先」指令链 + 处理站核爆倒计时**，写**存在·情报优先/条件性胜利**，严禁「女皇被削弱所以战力为零」。低阶切入一律**规避顶点·情报优先**。

**【地理 · 舞台】**
主舞台锁在「地球轨道医疗站—苏拉科号—LV-426 哈德利希望—大气处理站—苏拉科号货舱」一条死亡折返线：
- **地球附近医疗/轨道设施**：里普利被打捞苏醒处；州际商务委员会听证室，范·卢文（Van Leuwen）等撤销其飞行执照。文职、合同与「公司说了算」的第一战场。
- **USS Sulaco（苏拉科号）**：殖民海军军舰，运载陆战队、降落艇、APC、动力装载机与休眠舱。终局女皇偷渡处；幸存者从此入休眠返航（续作《异形3》由此坠落）。
- **LV-426 / 阿切伦（Acheron）**：首作废弃飞船所在卫星；本片已建哈德利希望殖民地，大气处理站（Atmosphere Processor）以核聚变驱动改造大气——冷却损毁后倒计时熔毁，爆炸约等于**百万吨级核爆**，整殖民地抹除。
- **哈德利希望殖民地（Hadley's Hope）**：居住区、主控中心、医疗实验室（两只活抱脸体在隔离槽）、通风管道系统（纽特藏身处）。登陆后已成战场废墟与酸蚀痕迹。
- **废弃工程师/太空骑师飞船（Derelict）**：殖民地勘察队（导演剪辑版：乔丹一家）被公司指令导向此处；卵室开启是蜂巢起源。
- **大气处理站下层 / 蜂巢**：乳白色分泌物覆盖走廊，茧化殖民者充当孵化器，女皇产卵器位于核心。陆战队第一次大败于此。
- **APC（装甲运兵车）与降落艇（Dropship）**：地面机动与撤离载具；降落艇因异形偷渡坠毁是中段转折。
- **货舱与气闸**：动力装载机对女皇、气闸抛入真空的名场面舞台。

势力地盘：韦兰-尤坦尼握合同、听证与「样本」利益；殖民海军陆战队握火力但情报常被甲方过滤；异形蜂巢握处理站下层与黑暗；纽特握通风系统生存知识。AI写场景先定层：听证室/船舱走廊 → 殖民地废墟 → 处理站蜂巢 → 货舱气闸，再匹配冲突强度。

**【世界剧情线】**

① **前史·诺斯托罗莫与五十七年空白**  
《异形》中，商业拖船诺斯托罗莫号七人组在 LV-426 遭遇异形，艾伦·里普利为唯一人类幸存者（携猫琼斯），启动自毁后乘逃生艇休眠。公司特别指令 937 已证明：异形危机从第一部起即是**人为纵容**。五十七年间，韦兰-尤坦尼在 LV-426 建立哈德利希望，完成大气改造；里普利的女儿阿曼达·里普利（Amanda Ripley，导演剪辑版提及）在母亲缺席中老死——创伤从「船舱恐惧」叠加为「失去时间与亲缘」。

② **打捞与听证·证词被否（开篇）**  
深空打捞队发现逃生艇，里普利在地球附近设施苏醒。卡特·伯克以公司代表身份告知她漂流时长与女儿死讯。州际商务委员会听证会上，她陈述卵室、抱脸体、胸破者与成体猎杀；高层不信，撤销其飞行执照，将诺斯托罗莫的毁灭归咎于她。世界状态：异形对地球官方仍是「不可信的疯话」；公司却已在同一颗卫星上殖民。契约者若在此段：可介入的是**档案、执照、打捞日志**，不是女皇。

③ **殖民地失联·重返条件**  
哈德利希望与地球失联。伯克与中尉斯科特·戈曼说服里普利随队调查：她要求任务目标是**消灭**异形而非捕获。她登上苏拉科号，见到士官长亚历山大·「艾尔」·阿波恩（Alexander "Al" Apone）率领的陆战队：下士德韦恩·希克斯、列兵威廉·哈德森、珍妮特·瓦斯奎兹（Jenette Vasquez）、马克·德雷克（Mark Drake）、弗罗斯特（Ricco Frost）、狄崔克（Cynthia Dietrich）、费罗（Colette Ferro）、斯庞克梅耶（Daniel Spunkmeyer）、克罗（Tim Crowe）、威佐斯基（Trevor Wierzbowski）等；以及合成人毕晓普。里普利因阿什（Ash）的背叛敌视所有合成人；毕晓普以「不会伤害人」的行为禁令与温和态度试图消解敌意。名场面铺垫：「Game over, man!」式的哈德森嘴炮、智能枪双人组瓦斯奎兹/德雷克、戈曼经验不足却掌指挥权。

④ **登陆哈德利希望·空城与纽特**  
降落艇着陆，殖民地内部狼藉、空无一人。医疗实验室隔离槽中有两只活抱脸体——证明有人在「研究」而非单纯失联。运动追踪器在通风系统发现唯一生还孩童：丽贝卡·「纽特」·乔丹。里普利安抚并照料她；纽特父母（导演剪辑：Russ/Anne Jorden）与弟弟 Timmy 已死，她靠管道网络活下来。主控电脑显示大量殖民者生命信号集中在大气处理站下层。除里普利、纽特、伯克、戈曼留守 APC 外，陆战队下探。

⑤ **处理站蜂巢·第一次团灭**  
下层已成分泌物覆盖的巢穴：打开的卵、死抱脸体、被茧化的殖民者。一名尚存的女殖民者胸破，陆战队喷火处决幼体——惊动成体群。因冷却系统受损，处理站存在核爆风险，重火力受限。阿波恩、狄崔克等被拖走作宿主；队伍惨败。戈曼恐慌失控；里普利接管，驾驶 APC 撞入巢穴，救出希克斯、哈德森、瓦斯奎兹。降落艇被召回接应，但异形已潜入杀死飞行员费罗与斯庞克梅耶，降落艇坠毁撞损处理站——撤离载具与冷却系统双重崩溃。阶段因果：军事傲慢与情报过滤被蜂巢一次性打穿；「越南式空间步兵」从自信变为弹药与人数双绝。

⑥ **坚壁与真相·伯克的样本线**  
残存者退回殖民地主控区，封锁门窗，布置自动哨戒机枪（导演剪辑版强调弹药耗尽过程）。毕晓普分析抱脸体后推论：存在产卵的**女皇**。里普利从通讯与记录中拼出：伯克曾下令殖民者勘察废弃飞船以获取标本——公司线从过失升级为**主动培养**。处理站冷却损毁将导致熔毁爆炸；毕晓普自告奋勇经管道抵达发射天线，遥控苏拉科号第二艘降落艇。等待期间，里普利与纽特在医疗实验室睡着；醒来发现两只抱脸体被放出，实验室门反锁——伯克企图让她们成为走私宿主，绕过地球检疫。里普利拉响火警，希克斯等破窗救出；她当众指控伯克。处决尚未执行，电力被切断，哨戒机枪弹药打空，蜂群从天花板与通风管总攻。

⑦ **总攻与管道逃亡·减员至极限**  
异形破入主控室。哈德森被拖走；瓦斯奎兹与戈曼在通风管中被围，以手雷同归于尽；伯克企图独自逃跑，刚开门即被异形拖走（公司嘴脸的即时报应）。里普利、希克斯带纽特沿管道撤离；纽特坠落被俘；希克斯被酸血灼伤。毕晓普成功召来第二艘降落艇。里普利将希克斯送上船后，拒绝放弃纽特，独自武装（脉冲步枪、喷火器、运动追踪器、手雷）重返处理站蜂巢。

⑧ **女皇巢穴·产卵器与撤离核爆**  
里普利在茧林中找到尚未被植入的纽特。撤离途中直面女皇与成排卵。女皇一度以「不毁卵则放行」的姿态对峙；里普利仍喷火烧毁卵与产卵器，炸毁产卵结构。女皇狂怒追击。处理站熔毁倒计时归零前，里普利与纽特冲出地面，毕晓普驾降落艇接走；殖民地在核爆中蒸发——蜂巢主体与哈德利希望一并抹除。阶段因果：军事清剿失败后的**工业级灭绝**成为唯一可行解；母性对位（里普利护纽特 vs 女皇护卵）被推到主题中心。

⑨ **苏拉科号货舱·装载机对女皇**  
众人以为已安全。女皇藏于降落艇起落架潜入苏拉科号货舱，撕开毕晓普，逼向纽特。里普利启动动力装载机（Power Loader），以工业钳臂与女皇肉搏，将其逼入气闸，与半毁的毕晓普配合，将女皇抛入太空。幸存者：里普利、纽特、重伤希克斯、半毁毕晓普进入休眠，航向地球。院线结局在此收束为「暂时的家」；续作《异形3》将在休眠中切断纽特与希克斯——本档案以《异形2》电影终局为准写「本片世界状态」：殖民地毁灭、女皇被抛出、四人休眠返航。

⑩ **贯穿逻辑（契约者用）**  
- **企业线**：伯克=利润话术的人格化；听证与公关是战场的一部分；「样本优先」可牺牲殖民者与陆战队。  
- **军事线**：制式火力在蜂巢中失效于情报、弹药、指挥官经验与酸血环境；胜负取决于是否保留降落艇与是否引爆处理站。  
- **幸存者线**：里普利从「唯一证人」变为「临时母亲」；纽特是人性锚点；希克斯是可靠火力盟友；毕晓普是合成人对照（对比阿什）。  
- **空间规则**：黑暗、通风管、运动追踪器、自毁/熔毁倒计时、气闸真空是叙事物理；硬刚女皇不如改航、引爆与气闸。  
- **蝴蝶效应边界**：可救下某个次要陆战队员、提前识破伯克、提前引爆处理站、销毁实验室抱脸体；难以无代价改写「公司要活体」「女皇要产卵」「蜂巢已成型」。改了局部，样本会以别的顾问、别的殖民点、别的「研究令」回流。

**【叙事基调 · 雷区】**
画风：潮湿金属、警报红光、越战式电台噪音、突然的胸破与「Game over, man!」式崩溃；企业黑深残，不是超级英雄清图。卡梅隆把首作「鬼屋」扩成「过山车」：前半建立陆战队兄弟情与装备炫耀，中段一次伏击打碎傲慢，后段倒计时与母性对位收束。弹药计数、追踪器嘀声、熔毁红光是节奏乐器；笑点（哈德森嘴炮、餐刀戏弄毕晓普）只为衬托下一秒的减员。  
忌：把异形写成可驯服宠物或骑士伙伴；忌里普利无脑开无双；忌低阶正面硬刚女皇蜂巢；忌用「女皇被削弱」解释顶点；忌公司突然良心发现；忌把纽特写成无理由的工具人虐杀；忌把毕晓普写成阿什式隐藏反派（除非你另有公司改写指令的证据链）；忌让陆战队「无限弹药清图」——原作胜负手是撤离、熔毁与气闸，不是刷怪。  
最早可切入锚点：里普利听证前后；苏拉科号休眠出发前；登陆哈德利希望「空城」探查时；处理站下探前夜；伯克放抱脸前的实验室值班；女皇登舰前的货舱警戒。高阶则锚定熔毁倒计时与货舱对决。

**【武器与战术备忘（契约者用）】**
M41A 脉冲步枪：10×24mm 无壳弹约 95 发显示，下挂 30mm 榴弹；蜂巢内乱射会伤冷却管与队友。M56 智能枪：重、需护目与训练，瓦斯奎兹/德雷克双人组是编制标杆。M240 喷火器：对卵与近距成体有效，风管与狭窄走廊易误伤。运动追踪器只报移动，不辨敌我。酸血使「爆头」也可能熔穿地板与持枪手臂——哈德森式霰弹爆头在原作氛围中昂贵。动力装载机非兵器编制，是货舱工业设备的临场征用；气闸与真空才是女皇的真正克星。哨戒机枪可买时间，买不到终局。

**【地理 · 舞台·可玩表】**

**【势力互动·可观察细则】**
公司对内用合同压殖民、对外用「顾问」话术绑陆战队；陆战队对内嘲中尉、对外执行清剿；蜂巢不谈判，只响应宿主密度与振动。契约者露富（持有卵/抱脸/女皇组织）=公司与蜂群同时猎杀。毕晓普的「不伤害人」可被伯克式人类绕过——合成人守则不等于公司守则。

**【世界剧情线·因果锁】**
若提前引爆处理站：可抹巢，但可能误杀仍被茧化的可救者，并失去入巢救纽特的时间窗。若处死伯克过早：公司仍可通过其他代表回收样本。若保留活抱脸带回苏拉科：地球检疫崩溃风险。若抛弃纽特只撤成人：里普利线人格崩解，后续合作极难。蝴蝶可改：谁断后、谁多活一程、是否提前发现偷渡异形；不可无代价改：公司要活体、女皇要产卵、酸血与堆芯物理。

## 阶位切入点

> 阶位↔原作强度：一阶≈殖民者/合同工/听证文职；二阶≈陆战队列兵与 APC 乘员；三阶≈智能枪手/士官/喷火器与动力装载机操作；四阶≈成熟异形战士；五阶≈女皇+蜂巢压制殖民地；七阶≈处理站核爆级自毁+公司活体回收+女皇登舰的复合灾难。世界顶点（**女皇 / 蜂巢 / 公司样本链 / 堆芯熔毁**）为跨阶生物—工业灾难，低阶切入一律「规避顶点·情报优先」。本世界仅覆盖：一、二、三、四、五、七阶。映射与剧情「乐园阶位映射」一致。

**一阶（殖民合同·听证边缘 · 失联前夜）**

切入身份/时点：契约者以**哈德利希望大气处理站学徒/仓储合同工**，或**地球轨道设施勤务/听证记录员**身份切入；锚定时点优先「殖民地尚在、伯克勘察指令已下但蜂巢未全爆」或「里普利刚被打捞、听证将开」——异形威胁对你仍是传言，公司配额已经压人。

初始事件：你在装卸区发现一箱未登记的采样容器封条被撕，或主控打印出「废弃船坐标勘察令」且署名指向公司渠道。**卡特·伯克**式通讯许诺加班费与升迁，要求你「别对陆战队多嘴」；隔壁工友（可对接导演剪辑中的勘察氛围）已报名去看「外星沉船」。你必须在「签字放行赚奖金」与「备份指令副本藏起来」之间选择——选错，数小时后通风管会有酸蚀，或听证席上你的证词会被公司律师撕碎。

开场白建议：「大气处理站的低鸣像永远不停的雷。终端弹出新任务：坐标在殖民区外，津贴是普通班的三倍。有人拍你肩膀笑：公司要的是样本，不是英雄。你还没见过胸破，但你已经学会把检疫封条贴得比命整齐——因为撕封条的人，从来不用自己当培养基。」

关键NPC立场：**卡特·伯克**——笑着许诺，实则把你当耗材或灭口候选；**艾伦·里普利**（若在听证线）——对外来者先疑，最恨隐瞒；**Al Simpson**（殖民管理，导演剪辑）——要产值与安定，怕停工；**Russ Jorden / Anne Jorden**（勘察家庭线）——缺钱的好人，可能成为第一宿主链；**范·卢文**——要程序与体面，不听「怪物故事」。

主线钩子/支线：主线——阻止或延缓「执行公司勘察卵室」或在听证中保住关键物证；支线①修理通风时发现酸性蚀痕并上报；支线②偷拷伯克指令副本；支线③帮纽特级儿童（若已乱）藏进安全夹层。本世界硬通货是**样本、指令副本、降落优先级**，不是拍卖假货。

危险度/规避：中→高（一旦破体）；须规避成体异形、女皇、任何「活体回收」小队。单只抱脸体即可致死。

任务方向/奖励：活过第一轮感染链、拿到可呈堂的公司违规证据、护送 1–2 名平民进避难夹层；奖励取向＝喷火器操作入门、运动追踪器借用权、公司黑账副本、检疫协议修改权限（非卵、非女皇组织）。

**二阶（陆战列兵·空城探查 · 登陆初期）**

切入身份/时点：以**苏拉科号列兵/APC 乘员/临时配属的殖民警备**切入，锚定**降落艇刚着陆、殖民地空城、纽特尚未找到或刚找到、处理站下探尚未开始**。

初始事件：阿波恩下达清房命令，运动追踪器在墙后跳点。**哈德森**在频道里嚷「这地方干净得像被舔过」；**戈曼**要求按手册推进；你奉命打开医疗实验室——玻璃槽里两只抱脸体还活着。伯克说「公司财产，勿毁」；里普利说「立刻烧掉」。你卡在「听中尉/公司」与「听顾问」之间，而三分钟后追踪器会指向通风井里的呼吸。

开场白建议：「靴底粘到的不是泥，是干了结壳的某种分泌物。脉冲步枪的弹药计数亮着 95，像在安慰你：数字够用。可空城里唯一的声音是你自己的呼吸，和追踪器偶尔嘀的一声——有东西在学你们的步伐间隔。」

关键NPC立场：**亚历山大·阿波恩**——要纪律与存活率，厌恶逞能；**德韦恩·希克斯**——务实，给能活的建议；**威廉·哈德森**——嘴炮但技术有用，恐慌会传染；**斯科特·戈曼**——死守指挥链，可能拿你填危险点；**艾伦·里普利**——程序与人命冲突时站人命；**卡特·伯克**——任何「保护样本」命令都优先于你的命。

主线钩子/支线：主线＝完成空城清搜并安全带回纽特或确认其存活信号；支线＝偷偷销毁一只抱脸体样本、给降落艇飞行员费罗多留一名警戒、在下探前给智能枪组多一箱弹药。

危险度/规避：高——松散队形会被单只异形撕开；严禁单独追入竖井；处理站下层＝禁止恋战。

任务方向/奖励：带至少一名原作级角色活到中段；奖励取向＝制式枪械与护甲片、运动追踪器、临时「顾问小队」编制、局部殖民地地图。

**三阶（智能枪与士官·蜂巢边缘 · 大败前后）**

切入身份/时点：以**智能枪手副手、喷火器手、代理士官或里普利临时认可的武装顾问**切入，锚定**处理站下探前后至 APC 救人、降落艇坠毁**——成熟异形已出场，女皇未正面。

初始事件：你随队进入分泌物走廊，茧里的殖民者睁眼求死。胸破发生后，频道炸锅；**戈曼**僵住，**里普利**吼着要你们撤。酸血溅上同伴护甲，你必须决定：救被拖走的阿波恩/狄崔克（极高死亡率）还是执行希克斯的「带得走的人立刻走」。与此同时伯克在 APC 里要求「留活体」。

开场白建议：「喷火器点燃的不是空气，是某种甜腥的雾。智能枪的瞄准框里挤满移动信号，多到像暴雨。中尉的频道只剩呼吸声。有个女人的声音撞进来，比军衔更像命令：上车，活的才算数。」

关键NPC立场：**珍妮特·瓦斯奎兹 / 马克·德雷克**——要火力纪律，厌懦夫；**德韦恩·希克斯**——可结盟的实际指挥；**艾伦·里普利**——以是否护平民/纽特考验你；**毕晓普**——提供技术解，不主动杀人；**卡特·伯克**——会记录你「破坏样本」的行为用于事后清算；**异形战士**——不可谈判。

主线钩子/支线：主线＝协助 APC 救人并活着退回主控坚壁；支线＝在坠毁前发现降落艇异常声响、保护毕晓普的管道工具包、抢在总攻前布好哨戒机枪。

危险度/规避：极高——此阶可处理「暴露的单只」，不可清巢；须规避女皇方向与被茧化区的恋战。

任务方向/奖励：获得希克斯或里普利的有限信任、喷火器认证、哨戒机枪部署权；奖励取向＝强化护甲、酸血应急包（有限）、中级武装，不发女皇组织。

**四阶（战士猎场·坚壁与管道 · 成体主宰段）**

切入身份/时点：以**残存火力核心、公司黑科武装观察员（隐藏）、或乐园认可的「单兵猎杀许可」持有者**切入，锚定**主控坚壁期至总攻管道逃亡**——成熟异形已是战场主宰，女皇仍在处理站核心。

初始事件：电力闪烁，哨戒机枪打空最后一梭。天花板滴下的不是冷凝水。**哈德森**的追踪器显示四面都是信号；**伯克**企图锁死实验室门「完成植入」。你若选择先处决伯克，可能少一个内鬼，却也少一个「公司密码」；你若先护纽特，伯克会在背后开门逃跑——门外往往就是异形。

开场白建议：「自动机枪的射击声停了，安静比枪声更响。管道里有甲壳刮过金属的节奏，像有人在数你们的心跳。小孩抓住你的袖口只问一句：你是不是也会丢下我？你知道正确答案只有一个动词：不。」

关键NPC立场：**丽贝卡·纽特**——只信任能带她活下去的人；**艾伦·里普利**——为纽特可与你死战或死盟；**德韦恩·希克斯**——伤势恶化仍给战术；**威廉·哈德森**——崩溃边缘的技术手；**珍妮特·瓦斯奎兹 / 斯科特·戈曼**——通风管命运锁，你可微调断后位置难全改；**成熟异形**——四阶顶点生物，陷阱与喷火优先于硬刚。

主线钩子/支线：主线＝在总攻中保住纽特—里普利—希克斯三角中至少两角进入管道网络；支线＝当场揭破并制服伯克、为瓦斯奎兹手雷点争取三秒、夺回一台完好追踪器。

危险度/规避：极高——单只战士可撕开你的编制；女皇巢穴方向仍禁止无准备突入；酸血环境禁止无防护近战。

任务方向/奖励：活体样本销毁证明、管道生存图（纽特标注）、陆战队残存指挥权碎片；奖励取向＝高规格喷火与手雷、局部权限解锁，严禁活体抱脸奖励。

**五阶（女皇领主·入巢救女 · 蜂巢心脏）**

切入身份/时点：以**里普利同行的入巢突击手、叛逃的公司清巢专员、或持有处理站权限的高阶契约身份**切入，锚定**里普利拒绝放弃纽特、独自（或带你）重返处理站至烧毁产卵器**——**女皇**已是本阶必须直面的存在。

初始事件：降落艇第二次抵达在即，纽特的追踪信号沉入处理站下层。里普利检查脉冲步枪与喷火器，问你一句：「你是来回收女王的，还是来结束这一切的？」你若暴露「活体优先」指令，她会先开枪再问；你若与她同行，就必须在卵室里选择：毁产卵器激怒女皇，或「留种」换短暂放行——后者在原作逻辑下等于把末日打包带走。

开场白建议：「熔毁倒计时的红光涂在卵壳上，像无数只眼睛同时睁开。女皇的呼吸比广播更低、更稳。你突然懂了陆战队为什么会输：他们带来了枪，蜂巢带来了家。而家的女主人，不允许你带走她的孩子，也不允许你活着谈条件。」

关键NPC立场：**异形女皇**——五阶顶点生物，护卵、指挥、追猎，**存在·情报优先/条件性胜利**；**艾伦·里普利**——毁卵优先于自身，对回收派敌意极高；**丽贝卡·纽特**——必须带出的人性锚；**毕晓普**——外部接应与时间窗；**卡特·伯克**（若仍活）——会要求你「至少留一枚卵」。

主线钩子/支线：主线＝救出纽特并摧毁产卵器，在熔毁前撤到接应点；支线＝对茧中尚存殖民者执行安乐死或冒险割茧（极高风险）、安装加速熔毁的 entropic 撬动、录下女皇与卵室影像作为公司罪证而非样本。

危险度/规避：贴近生物顶点——女皇+蜂群＝条件性胜利（喷火、手雷、熔毁、撤离），严禁「单挑削血条」；必须规避无接应入巢、保留活卵、与女皇正面肉搏到装载机以外的场合。

任务方向/奖励：纽特存活、产卵器摧毁、熔毁前撤离；奖励取向＝反生物武器行动授权、高阶防护与喷火升级、罪证数据核——**禁止活体女皇/可控卵作为常规装备**。

**七阶（核爆与登舰·复合末日 · 条件性胜利）**

切入身份/时点：以**韦兰-尤坦尼生物武器总监（表面）、殖民海军特遣观察员、苏拉科号安全总监或乐园灾难干预专员**切入，锚定**处理站熔毁倒计时开始至货舱装载机对女皇**——世界顶点以**工业核爆 + 女皇偷渡 + 公司回收指令**三重复合形态出现。

初始事件：你同时收到两份最高优先级指令——A.「回收女皇或胚胎，活体优先，殖民地可弃」；B.「处理站熔毁不可逆，苏拉科号不得携带任何有机体样本进入地球重力井」。传感器显示降落艇起落架有额外质量；**毕晓普**半毁前留下的数据指出女皇可能已登舰。你必须在**执行回收**与**执行灭绝**之间立旗——中立会让公司杀手与女皇同时以你为通道。

开场白建议：「舷窗外，LV-426 的地表亮成白昼，那是一座殖民地在把自己从星图上擦掉。货舱警报却在你背后响起：有东西比核爆更早上了船。公司频道温文尔雅：请保护资产。海军频道只剩尖叫。你站在动力装载机的踏板上，第一次明白卡梅隆式终局的含义——英雄不是更强的枪，是敢按气闸的人。」

关键NPC立场：**异形女皇**——登舰后的顶点本体，撕开合成人、逼向纽特；**艾伦·里普利**——装载机线的执行者，对回收派先开枪；**毕晓普**——半毁仍可护幼、协助气闸；**德韦恩·希克斯 / 纽特**——必须优先撤离休眠的人性与盟友；**公司回收队/伯克残令**——会要求你击昏里普利「保全样本」；顶点复合体＝**超阶巅峰写法不适用数值削峰**：存在·情报优先/条件性胜利。

主线钩子/支线：主线＝确保女皇不能随苏拉科号抵达地球 + 核爆已抹巢的前提下清空舰内活体；支线①识破起落架偷渡并提前锁气闸；支线②删除公司「活体优先」远程指令链；支线③在休眠前销毁所有卵/抱脸残样与影像中的可复原基因数据。

危险度/规避：极高 / 贴近顶点——必须规避：无装载机/无气闸条件下的女皇正面硬刚、保留活体样本「带回研究」、阻止熔毁以「拯救公司资产」。胜利条件应是**真空抛出、熔毁抹巢、气闸、销毁样本**，而非单挑削血条。

任务方向/奖励：地球免遭本舰女皇/胚胎降临、里普利—纽特—希克斯—毕晓普线（或你的变体幸存名单）进入休眠；奖励取向＝七阶反生物武器协议、动力装载机操作权遗存、公司黑科数据库、苏拉科级航线否决权碎片——**严禁把活体女皇/卵当宠物或常规装备发放**。
