# 棕色尘埃（Brown Dust）
<!--meta lib=主库 tiers=一、二、三、四、五、六-->

## 剧情

**【作品来源】**
《棕色尘埃》（Brown Dust，韩：브라운더스트）为NEOWIZ出品的移动端战术RPG，2017年前后韩服运营，2018年3月7日前后国际服上线；以高完成度二次元立绘、格子回合制、阵型与攻击顺序策略为核心。玩家组建佣兵团，在帝国崩坏后的大陆上推进主线战役、符文神殿、水晶洞窟、魔城等玩法。续作/关联生态有《Brown Dust 2》等，本档案以**一代 Brown Dust 佣兵战术世界观与公开主线框架**为准。文风：黑暗奇幻×佣兵群像×策略养成。

**【世界定位】**
帝国已亡，大佣兵时代开启。大陆被旧帝国残党、新兴王国、教会、魔族（含「六魔」与「支配八人组 Dominus Octo」等传说级存在）与无数佣兵团瓜分。玩家作为佣兵团指挥官，从废墟与边境据点起步，招募超过数百名具名佣兵，用阵型、职业克制与技能连锁在千余关卡中求生并追查「帝国为何崩塌」。一句话：帝国余烬上的佣兵棋盘，策略比等级更决定生死。

**【世界观 · 力量体系】**
力量本源：①佣兵固有技能与职业（攻/防/辅等）；②装备（武器、防具）与制作；③符文（Rune Temple 五系符文强化能力）；④觉醒材料（水晶洞窟等）；⑤阵型格子与攻击顺序——同战力因站位可完胜或团灭；⑥传说存在：六魔、Dominus Octo 等远超常人。战斗为经典回合制+格子移动。死亡：战役中佣兵可「战败退场」；剧情上凡人与普通佣兵会死；传说级 downstream 可灭城。无无限复活；核心链接式养成不改变「战场一次失误=团灭」的物理。

战力阶梯（破坏力，宁低勿高）：
1. **难民/农夫/新兵佣兵**：冷兵器格斗。≈一阶。
2. **成熟佣兵/小团精锐**：碎石裂阵、清剿匪窝。≈二阶。
3. **名佣兵/王国骑士级**：端据点、破城门。≈三阶。
4. **觉醒佣兵/符文满配团长侧**：毁建筑群、一方战场支配。≈四阶。
5. **传说佣兵/魔将级**：夷平街区～小城威胁。≈五阶。
6. **六魔/Dominus Octo/帝国崩坏级灾厄**：地区至王国级灾难。≈六阶。

乐园阶位映射：新兵≈一阶；常规佣兵≈二阶；名佣≈三阶；觉醒精锐≈四阶；传说侧≈五阶；六魔与八人组≈六阶。顶点=支配帝国崩坏的传说灾厄（情报优先）。宁低勿高。

**【地理 · 舞台】**
大陆世界地图：旧帝国核心废都、边境王国城塞、佣兵都市（酒馆·公会·黑市）、符文神殿、水晶洞窟、魔城（Evil Castle）、世界BOSS战场、公会战前线。层级：据点养成→战役章节推进→神殿/洞窟资源层→魔城高难→传说级灾厄。势力地盘随章节开图：王国征召、教会净化、魔族侵蚀、佣兵中立区重叠。

**【世界剧情线】**
① **帝国余烬·佣兵起步**  
帝国崩塌后秩序真空，盗匪与残军横行。指挥官在边境组建小队，接护送、清巢、夺粮任务，结识第一批具名佣兵（各国背景的剑士、法师、枪手、辅助等）。名场面：酒馆招募、首次团灭教学关、废都边缘遭遇帝国纹章怪物。因果：明白「阵型>蛮干」。

② **王国与教会的征召**  
新兴王国以金钱与爵位诱饵征召佣兵团镇压叛乱；教会以「净化魔尘」名义争夺圣遗物。指挥官在两派间接任务，发现双方都在隐瞒帝国崩坏真相的一角。名场面：城墙攻防战、圣物护送遇袭、友军背刺。

③ **符文神殿与力量质变**  
为对抗更高阶敌人，进入符文神殿收集五系符文，水晶洞窟挖觉醒材。神殿内机制战与「顺序杀」成为主流。名场面：错误站位被秒、符文共鸣、觉醒演出。阶段结局：佣兵团跨入四阶精锐门槛。

④ **魔城与六魔阴影**  
Evil Castle 高难本揭示魔族编制；六魔之名在口述传说与残页中反复出现。指挥官团首次接触「非人智慧」的敌将，伤亡换情报。名场面：魔城层主、古代币奖励、全灭重开。

⑤ **Dominus Octo 与帝国真相**  
主线深层指向帝国崩坏非单一政变，而与八名支配级存在、禁忌实验或魔尘扩散相关（公开资料强调「发现王国为何陨落、走出自己的命运与策略」）。终盘战役为多势力混战：王国、教会、魔族、佣兵公会抢夺「能改写战局的遗物」。名场面：八人组之一降临战场、指挥官用阵型完成条件性击退。

⑥ **大佣兵时代的「胜利」**  
无单一「加冕为王」的强制结局；主线强调在崩坏世界上以策略存活并推进真相。可理解为：核心灾厄被压制或封印碎片化，佣兵团成为新秩序的一极，帝国真相部分公开。连载/运营式内容持续追加角色与活动剧情。

⑦ **微观可介入**  
公会战、世界BOSS、远征、制作、觉醒材料争夺、佣兵个人故事（信赖/羁绊）。AI优先：缺粮、假任务、阵型间谍、符文赝品。

⑧ **蝴蝶效应边界**  
可改：单场战役胜负、救下具名佣兵、提前拿符文。不可无代价改：帝国已崩的历史前提、六魔/八人组战力逻辑、阵型规则。给低阶直接杀六魔=世界观崩坏。

**【主要人物】**
（游戏为多角色卡池；下列为世界观职能真名/职称锚点，具名佣兵以公开图鉴常见核心为例，查无则标职能。）
- **指挥官（玩家）**｜性格：冷峻或热血皆可，本质「用阵型说话」。｜装备·能力：编队权、战术指令。｜人物弧光：边境小队→传说级指挥官。｜立场关系：契约者可同位或副手。
- **初期剑士系佣兵（如常见主C位）**｜性格：直率好战。｜装备·能力：前排斩击。｜人物弧光：炮灰→觉醒。｜立场关系：前排锚。
- **法师/炮击系佣兵**｜性格：理性。｜装备·能力：范围魔导。｜人物弧光：清杂→破甲。｜立场关系：后排核。
- **圣职/辅助佣兵**｜性格：悲悯或狂热。｜装备·能力：治疗、增益、净化。｜人物弧光：教会关联线。｜立场关系：双面。
- **盗贼/刺客佣兵**｜性格：唯利。｜装备·能力：背刺、扰乱顺序。｜人物弧光：黑市线。｜立场关系：可背刺。
- **枪兵/坦克佣兵**｜性格：沉默守约。｜装备·能力：嘲讽、减伤。｜人物弧光：挡灾厄一击。｜立场关系：防核心。
- **六魔之一（具名随版本）**｜性格：非人傲慢。｜装备·能力：五～六阶灾厄。｜人物弧光：章节Boss。｜立场关系：硬刚=团灭。
- **Dominus Octo 成员**｜性格：支配者逻辑。｜装备·能力：六阶。｜人物弧光：真相钥匙。｜立场关系：情报优先。
- **王国将军（征召方）**｜性格：务实爱国。｜装备·能力：军团指挥。｜人物弧光：雇主力。｜立场关系：金主可反目。
- **教会审判官**｜性格：净化偏执。｜装备·能力：圣术。｜人物弧光：圣物线。｜立场关系：任务与审判。
- **佣兵公会会长**｜性格：老狐狸。｜装备·能力：情报网。｜人物弧光：经济线。｜立场关系：中介。
- **旧帝国残党首领**｜性格：复国执念。｜装备·能力：禁军遗术。｜人物弧光：废都副本。｜立场关系：可联可灭。
- **魔城城主**｜性格：残酷。｜装备·能力：高难机制。｜人物弧光：周常压迫。｜立场关系：刷本对象。
- **世界BOSS（灾兽）**｜性格：本能。｜装备·能力：多人战。｜人物弧光：服务器事件。｜立场关系：协作击杀。

**【势力图谱】**
- **佣兵公会**：中立接单、评级、黑名单。
- **新兴王国**：征召与领土战。
- **教会**：净化、圣遗物、审判。
- **旧帝国残党**：复辟与禁术。
- **魔族/六魔体系**：侵蚀与魔城。
- **Dominus Octo**：幕后支配级。
- **玩家佣兵团**：可变第三极。

**【贵重物品】**
- **五系符文**：神殿产出，强化能力。
- **觉醒素材**：水晶洞窟。
- **古代币**：魔城等，换稀有。
- **传说武器/防具图纸**：制作顶配。
- **帝国机密文书**：崩坏真相。
- **支配八人组相关遗物**：高阶争夺。
- **公会徽章与王国委任状**：身份通行。

**【隐藏剧情 · 伏笔】**
- 帝国崩坏与魔尘/实验/八人组多因叠加。
- 符文神殿可能是帝国遗制而非纯自然。
- 部分佣兵身世连到六魔或皇族。
- 运营活动剧情补全个人线，勿与主线终局冲突。
- 续作世界观演进不回溯否定一代「帝国已亡」。

**【大事记时间线】**
帝国盛世 → 崩坏（六魔/八人组/内乱） → 大佣兵时代 → 指挥官建团 → 王国教会征召 → 神殿洞窟质变 → 魔城与传说战 → 真相碎片公开 → 新秩序一极。

**【叙事基调 · 雷区】**
画风：华丽立绘下的残酷战损与策略冷感。忌：无阵型描写的无脑碾压；把六魔当杂鱼；忽略佣兵个人故事；写成纯抽卡无剧情。最早切入：建团首周、第一次王国征召令下达时。

## 阶位切入点

> 阶位↔：一阶新兵；二阶成熟佣兵；三阶名佣；四阶觉醒精锐；五阶传说侧；六阶六魔/八人组。顶点灾厄情报优先。

**一阶（建团·废都边缘）**
切入身份/时点：新佣兵/辎重员；开局章节。
初始事件：护送粮车遭匪，错误站位几乎团灭，公会考官冷眼记录。
开场白建议：「沙尘里有人喊‘后排放前面’。下一秒，你的前排就断成了两截——欢迎来到大佣兵时代。」
关键NPC立场：**指挥官**要战果；**公会会长**要评级；**初期剑士**要证明；**难民**要活。
主线钩子/支线：首胜评级；支线：假货符文、酒馆决斗。
危险度/规避：低；规避魔城。
任务方向/奖励：基础装备、公会登记。

**二阶（王国征召）**
切入身份/时点：正式团员；王国第一章。
初始事件：城门攻防，友军骑士抢功导致缺口。
开场白建议：「城墙上的旗是王国的，缺口里涌出的却是旧帝国的盔。将军说：守住，赏金翻倍——没说谁垫尸。」
关键NPC立场：**王国将军**催战；**圣职佣兵**救伤；**刺客**卖情报；**残党**要血。
主线钩子/支线：城墙胜负；支线：圣物箱调包。
危险度/规避：中。
任务方向/奖励：委任状、中级武器。

**三阶（名佣竞标）**
切入身份/时点：竞标团核心；多团争夺同一悬赏。
初始事件：悬赏「猎杀叛将」，三团同图，攻击顺序决定谁拿到首级。
开场白建议：「三支旗插在同一座桥上。桥对面叛将只开一枪——倒地的却是你侧翼的治疗。」
关键NPC立场：**名佣**要名声；**教会审判官**要活口；**黑市**要尸体。
主线钩子/支线：首级归属；支线：叛将密信。
危险度/规避：中高。
任务方向/奖励：名佣契约、高级图纸。

**四阶（符文神殿深部）**
切入身份/时点：觉醒候选；神殿高层。
初始事件：五系符文共鸣失控，友军被己方法术误伤。
开场白建议：「神殿的光按你的站位改颜色。站错一格，治疗会变成火葬。」
关键NPC立场：**法师佣兵**要稳定；**坦克**挡反噬；**神殿守护者**（机制敌）。
主线钩子/支线：拿满符文；支线：赝品符文案。
危险度/规避：高；勿硬闯魔城顶。
任务方向/奖励：满符文、觉醒材。

**五阶（魔城层主）**
切入身份/时点：讨伐先锋；Evil Castle。
初始事件：层主技能点名「后排」，需用替身与顺序骗招。
开场白建议：「魔城的门为古代币而开，也为葬队而开。层主念你名字的发音，像在点菜单。」
关键NPC立场：**六魔残影**压迫；**公会观察员**要录像；**指挥官**要通关。
主线钩子/支线：层主击破；支线：古代币换禁装。
危险度/规避：高；六魔本体仅情报。
任务方向/奖励：古代币、传说装。

**六阶（八人组碎片战）**
切入身份/时点：真相追猎者；终盘。
初始事件：Dominus Octo 之一投影降临，王国与教会临时停火。
开场白建议：「天色变成旧帝国仪典的金。八分之一的支配者只投下影子，三座城的警报就同时响了。」
关键NPC立场：**Dominus Octo**顶点；**指挥官**条件击退；**教会与王国**抢遗物。
主线钩子/支线：条件性击退；支线：崩坏文书公开。
危险度/规避：贴近顶点；**超阶边缘：情报优先/条件性胜利**。
任务方向/奖励：真相碎片、大陆级声望（非必杀八人组本体）。
