# 棕色尘埃（Brown Dust）

## 剧情

**【作品来源】**
《棕色尘埃》（韩：브라운더스트 / Brown Dust；国际服后称 **Brave Nine**；日：ブラウンダスト）由 **GAMFS** 开发、**NEOWIZ** 发行的移动端格子回合制战术RPG。韩服约 **2017年4月24日**上线；日服 **2018年3月7日**；台服约2018年10月；国际服 **2019年3月7日**（Brave Nine）；Steam 约 **2023年8月30日**；运营后期转向单机/离线完整版，并有「佣兵团终章」类收束更新。续作《棕色尘埃2 / Brown Dust 2》另起多宇宙与3×4战棋体系，**本档案以一代 Brown Dust / Brave Nine 的佣兵大陆主线与公开设定为准**。文风：华丽立绘下的黑暗奇幻×佣兵群像×阵型策略；友情、冒险与背叛交织。卡片锚点：**「日蚀之日」余波下的佣兵团崛起——帕维安之子隐藏身份追查父死真相，六魔星与八夜之主阴影笼罩大陆。**

**【世界定位】**
**卡里安帝国（Carian Empire）** 因名将 **帕维安（Pavian）** 政变与「**日蚀之日**」崩坏后，帝国权威碎裂，部族同盟、北部联邦、共和国与神圣帝国残躯并立，佣兵团成为真正的硬通货。玩家视点为**佣兵团团长**——公开资料强调其为**帕维安之子**：为复兴家门、查明父亲死因而隐姓埋名，在七百至一千二百余战役关卡中招募三百余名佣兵，以阵型、攻击顺序与技能连锁求生。一句话：**帝国余烬上的佣兵棋盘，站位比等级更决定生死；契约者可同位团长/副手/敌对团核心，但不可无代价改写日蚀史实与六魔星战力逻辑。**

**【世界观 · 力量体系】**
力量本源多层并存，无单一「灵气修仙树」：
1. **佣兵固有战技与职业定位**（前排斩击/坦克嘲讽、后排魔导炮击、治疗增益、扰乱顺序、魅惑控制等）。
2. **装备与制作**（武器、防具、图纸与强化）。
3. **符文系统（Rune Temple / 符文神殿）**：收集并强化五系符文，直接抬升佣兵能力上限。
4. **觉醒（Sealed Crystal Cave / 封印水晶洞窟等）**：材料觉醒后技能与造型质变。
5. **助手佣兵 / 羁绊强化**：与主佣兵有缘分的助手可抬高技能境界。
6. **阵型格子与攻击顺序**：同名战力因站位可完胜或团灭——本世界「物理定律」的核心。
7. **传说与灾厄存在**：六魔星、八夜之主（暗夜侧支配者）、女神侧（弗雷莱亚、艾达）等，远超常规佣兵团。
8. **运营后期追加系统（公开宣传）**：符卡扩展策略、神话佣兵、记忆板等——叙事上可理解为「终盘打开的更高棋盘接口」，低阶勿直接发奖。

死亡与复活：战役关卡可「战败重开」；剧情物理上凡人与普通佣兵会死、可残废、可被政治抹杀。传说级存在灭城、灭军、扭曲一域秩序。无「无限免费复活」；一次站位失误=团灭是常态。世界顶点（六魔星全盛、八夜之主、日蚀级灾厄）对低阶为**情报优先 / 条件性胜利**，严禁写成「被封印所以战力归零」。

战力阶梯（破坏力表现，宁低勿高）：
1. **难民 / 农夫 / 新兵佣兵 / 辎重**：冷兵器格斗、护送粮车、清匪。≈一阶。
2. **成熟佣兵 / 小团精锐**：碎石裂阵、端匪窝、守城门一段。≈二阶。
3. **名佣兵 / 王国骑士级 / 公会高评团**：破据点、城墙攻防决定胜负、猎叛将首级。≈三阶。
4. **觉醒满配 / 符文精锐团长侧**：毁建筑群、一方战场支配、神殿机制战通关。≈四阶。
5. **传说佣兵 / 魔城层主 / 六魔星残影级**：夷平街区～小城威胁、古代币级资源战。≈五阶。
6. **六魔星全盛 / 八夜之主 / 日蚀之日帝国崩坏级**：地区至王国级灾难，可左右「太阳是否落下」。≈六阶。

乐园阶位映射：新兵与边境杂役≈一阶；常规佣兵团≈二阶；名佣与王国征召主力≈三阶；觉醒+满符文精锐≈四阶；传说侧与魔城高压≈五阶；六魔星与八夜之主≈六阶。顶点=日蚀级灾厄与暗夜支配者（**情报优先，条件性胜利**）。宁低勿高。

**【地理 · 舞台】**
大陆以**旧卡里安帝国核心皇都—废都轴**为中心辐射：
- **神圣帝国卡里安残躯**：以皇都为中心维系「神圣帝国」之名，律令与教会权威并存，征召佣兵镇压边境。
- **部族同盟阿尔兰（Arlan）**：边境部族为反抗征服而联合，大酋长统御，骑兵与部落仪式战常见。
- **北部联邦史贝伦（Speiren）**：北部原藩属主张独立，联邦制军事与贸易并重。
- **里安共和国（Lian）**：帝国领土分裂独立的共和政体，议会、商会与民兵交错。
- **佣兵都市 / 公会城**：酒馆招募、评级、黑市、假货符文、决斗场与公会战前线——契约者最常落脚。
- **符文神殿**：五系符文产出地，机制战与顺序杀教学场。
- **封印水晶洞窟**：觉醒材料层，资源争夺与塌方、赝品材料并行。
- **恶魔城 / 魔城（Demon’s Castle / Evil Castle）**：高难分层，古代币与层主机制；六魔阴影与暗夜气息浓。
- **世界魔王战场 / 公会战地图 / 协力征伐场**：多人/多团协作与 PVP 政治。
- **暗夜侧世界名（公开设定）**：图阿尔（吸血鬼界）、利亚沙（梦魔界）、塔罗斯（不死界）、里班（逃亡者/堕落天使侧）、凯那尔（恶魔贵族侧）等——八夜之主统治域，低阶仅闻其名。

层级玩法：据点养成 → 战役章节推进 → 神殿/洞窟资源 → 魔城高压 → 传说/女神线/终盘真相。AI 写场景先定契约者在哪一层地图与哪一势力税契下，再定冲突强度。

地理细节补充（可玩锚点）：皇都外环是法令与税契最密的「纸面秩序」；废都走廊则是旧帝国军械库、纹章魔物与残党伏击带，适合低中阶护送与拾荒。佣兵都市的酒馆是信息交易所——合同分「明标」（公会挂牌）与「暗标」（桌下咬耳），暗标赏金高但常附带灭口条款。符文神殿并非单一建筑，而是多入口、多层回路的遗迹群，五系符文对应不同试炼厅，错系硬装会反噬技能循环。水晶洞窟潮湿塌方多，觉醒材与「看起来像觉醒材的石头」混卖。恶魔城垂直分层：越往上古代币越丰、层主机制越针对「固定站位癖」。暗夜侧地名在地上界多为禁书插图与酒馆鬼故事，真要入境需八夜许可或等价祭品——低阶写「听说」即可。

**【世界剧情线】**
① **前史·日蚀之日（Eclipse Day）**  
卡里安帝国名将兼贵族 **帕维安**，以「忠臣」之名发动政变。皇帝 **阿特欧二世（Arteo II）** 派出数以万计帝国军镇压；叛军人数远少，却有追随帕维安的六颗「邪恶之星」——**六魔星**——战力极强，帝国军陷于劣势。帕维安最终杀害阿特欧二世；世人将帝死比作太阳消失，史称 **「日蚀之日」**。其后帝国军反击，帕维安战死。叛乱虽被镇压，帝国已伤痕累累，统治力暴跌：边境结成 **部族同盟阿尔兰**；北部藩属合成 **北部联邦史贝伦**；部分领土独立为 **里安共和国**；皇都势力勉强保全 **神圣帝国卡里安** 之名。因果锁死：**帝国已崩、帝已死、帕维安已死**——契约者不可无代价改写这一历史前提。

② **七年真空·大佣兵时代开启**  
日蚀后约七年，大陆因各国纷争与内部混乱严重人力不足。有实力的佣兵团名声与财富暴涨，接征召、护送、清巢、城防与猎杀任务。公会评级、决斗场、公会战成为新秩序的「法庭」。名场面类型：酒馆合同、假任务埋尸、城门缺口友军抢功、粮车被双面间谍卖掉路线。因果：阵型文化压过蛮勇；「团长」成为可买卖的职业身份。

③ **开篇·隐名团长建团（井底）**  
视点人物——**帕维安之子**——隐藏身份组建佣兵团，目标：复兴家门、查明父亲真正死因（「杀害君主者、遮住太阳的月亮、六颗星的主人、无法呼唤的名字」等口述碎片指向日蚀黑幕）。首批佣兵来自各国背景：剑士、法师、枪手、圣职、刺客、枪兵坦克等。名场面：首次几乎团灭的站位教学关；废都边缘遭遇帝国纹章怪物；公会考官冷眼记录评级。因果：明白「阵型＞蛮干」；对「六魔星是否仍在世」仅有传闻。

④ **立名·王国 / 教会 / 共和征召**  
新兴与残存政权以金钱、爵位、圣遗物话语权征召佣兵团：镇压叛乱、护送圣物、夺粮道、猎叛将。**教会（含阿克莱特教团等背景线）** 以「净化」「守护圣物」为名争夺解释权；王国将军要战果；共和商会要路线垄断。指挥官在多方金主间走钢丝，发现各方都在隐瞒日蚀真相的一角。名场面：城墙攻防、圣物箱调包、友军骑士抢功导致缺口、三团同桥争首级。阶段结局：团在一域立住「不可轻易欺」的评级，开始被更高层次注视。

⑤ **质变·符文神殿与水晶洞窟**  
为对抗更高阶敌人，进入 **符文神殿** 收集五系符文，进入 **封印水晶洞窟** 挖觉醒材。神殿内错误站位可被己方法术误伤；符文共鸣与觉醒演出带来战力台阶。助手佣兵系统让「个人故事」与技能上限绑定。名场面：赝品符文案、神殿守护者机制敌、觉醒后造型与技能质变。阶段结局：佣兵团跨入精锐门槛（乐园观感≈四阶），可碰触魔城外围。

⑥ **魔城与六魔星阴影**  
**恶魔城** 高难分层揭示魔族/暗夜编制；**六魔星** 之名在口述与残页中反复出现。六魔星公开设定包括：  
- **安洁利卡（Angelica）**：「堕落之星」，持魔剑的修女，原阿克莱特教团魔剑守护者，因一把魔剑失踪几乎被处刑，被帕维安救出后追随；日蚀后仍寻另一把魔剑。  
- **艾瑞克（Alec）**：「破坏者」，前恶名海盗，为守护 **埃拉（Ella）** 与其所爱世界而追随帕维安。  
- **芮彼泰雅（Refithea）**：「贪婪」妖精公主，失落王国幸存者，外表幼女实则长寿；与帕维安有「重建王国」之约，日蚀后等待其子出现。  
- **纳罗达斯（Nartas）**：「灾祸之星」，少年貌实逾百岁的恶魔力魔法师，研究古代魔法致家人罹难后隐居，为见帕维安重现世间。  
- **西利亚（Celia）**：「诅咒之星」，诅咒魔女血脉，日蚀后仍因疑虑活跃于世（不同于多数遁隐）。  
- **葛兰西特（Granhildr）**：「虚无」单翼女武神，听命半神 **艾达（Eda / Ida）**，日蚀后守护艾达身侧。  
指挥官团首次接触「非人智慧」敌将与层主，伤亡换情报与 **古代币**。名场面：层主点名后排、全灭重开、古代币换禁装。

⑦ **格局战·四极与公会政治**  
神圣帝国、阿尔兰、史贝伦、里安与无数佣兵公会在资源与「日蚀解释权」上开战。决斗场、公会战、世界魔王、协力征伐把个人恩怨吞进阵营棋盘。团长从棋子变成能搅局的棋手：可同时接两边合同，也可因露富被全民围猎。名场面：公会战防守阵与进攻阵对撞、世界魔王周常、假投降与阵前倒戈。

⑧ **八夜之主与暗夜棋盘**  
更高阴影为 **八夜之主** 体系（暗夜侧支配者，公开具名包括）：  
**贝尔佩伦（Velfern）** 图阿尔之王；**阿斯摩德（Asmode）** 利亚沙梦魔领袖；**斐杰（Valze）** 图阿尔女王；**马莫尼勒（Mamonir）** 塔罗斯不死统治者；**卢修斯（Lucius）** 里班堕落天使王；**莱维亚（Rabia）** 禁咒研究者；**贝里阿斯（Belias）** 凯那尔贵族；**赛托（Seto）** 凯那尔以力称王者等。他们各自追求永恒、神杀、灵魂与禁术，对地上界的态度从诱惑、掠夺到征服不等。终盘战役常呈现：王国与教会临时停火，多势力抢夺「能改写战局的遗物/文书」，八夜之一投影或代理人降临。名场面：天色如旧帝国仪典之金、三城警报同响、条件性击退而非必杀本体。

⑨ **女神线与「回到过去」叙事接口（运营公开向）**  
公开宣传与后期剧情接口出现女神 **弗雷莱亚（Freylea）**、**艾达** 等：「尚未憎恶人类的弗雷莱亚」「尚未失去双翼的艾达」、神话佣兵、时空向章节。叙事功能：补全日蚀前后因果、给出「为何六魔追随帕维安」「神与暗夜的旧账」的碎片，而非让低阶团长一刀斩神。低阶切入只闻神名；中高阶可接触神殿侧代理人与试炼。

⑩ **终局方向·佣兵团长路与真相碎片**  
一代主线为运营式连载，无单一「加冕为王」强制结局；公开框架强调：在崩坏世界上以策略存活、推进「帝国为何陨落」、走出自己的命运。可归纳的终局态：  
- 核心灾厄被压制、封印碎片化或暂时击退；  
- 帕维安之子查到部分真相（政变动机、六魔契约、帝死与反攻中的多因叠加），家门名誉部分洗刷或彻底决裂；  
- 佣兵团成为新秩序一极；  
- 离线完整版强调「佣兵团漫长旅程的最终章」——允许以团长视角收束个人弧光，但不否定大陆仍处于多极佣兵时代。  
**胜利必须昂贵**：伤亡、政治背刺、符文反噬、与六魔/八夜的条件交易。严禁低阶硬刚六魔星全盛或八夜之主本体。

⑪ **微观可介入（日常高频）**  
缺粮、假任务、阵型间谍、符文赝品、觉醒材料黑市、佣兵个人信赖/羁绊故事、决斗场段位、公会黑名单、圣物护送、废都拾荒。AI 优先这些小事件，而非一上来顶点大战。

⑫ **蝴蝶效应边界**  
可改：单场战役胜负、救下具名佣兵、提前拿符文/觉醒材、让某城墙易主、公开一份非核心文书。  
不可无代价改：日蚀之日帝已死与帕维安已死的历史前提；六魔星/八夜之主战力逻辑；阵型与顺序规则；把六魔当杂鱼秒杀。改了局部，原作因果会以追杀、取代、更惨的方式回流。

⑬ **战役关卡与玩法叙事化（AI 写「正在发生什么」用）**  
公开宣传与商店页将内容拆为：战役/剧情（数百至千余关，多难度）、恶魔城、决斗场、地下竞技场、公会战、符文神殿、封印水晶洞窟、活动地下城、世界魔王、协力征伐、混沌之门等。叙事时不要写成菜单，而写成「本周大陆正在发生的事」：例如公会战周是各团在边境换防与偷袭补给线；世界魔王周是多团临时结盟、阵型从爆发切减伤；恶魔城周是古代币通胀与黑市哄抬；决斗场冲段位会直接改变酒馆里谁敢跟你说话。重复战斗/挂机向便利在世界观里可解释为「团有后备编制与轮换」，但剧情关键战仍须写伤亡与政治后果，禁止「无限免费重来所以无惧」。

⑭ **佣兵团内部政治（可长期线）**

任务方向/奖励：活下来并拿到公会登记；基础装备、少量符文残片、信誉。

**二阶（王国征召 · 城墙缺口）**
切入身份/时点：正式团员/副战术手；第一次王国或神圣帝国城防征召。
初始事件：城门攻防中友军骑士抢功，缺口涌出旧帝国盔甲残兵；将军承诺赏金翻倍却不说谁垫尸。
开场白建议：「城墙上的旗是王国的，缺口里涌出的却是旧日纹章。将军的手指点在地图格子上，像点菜单——你的名字刚好在『可消耗』那一栏。」
关键NPC立场：**王国将军**催战果；**圣职佣兵**要救伤与净化名分；**刺客佣兵**卖缺口情报；**旧帝国残党百夫长**要血债；**团长**要两面不得罪。
主线钩子/支线：钩子=城墙胜负与赏金是否兑现；支线=圣物箱调包、残党密信、友军背刺证据。
危险度/规避：中；规避六魔点名战场。
任务方向/奖励：委任状、中级武器、公会声望。

**三阶（名佣竞标 · 三团同桥）**
切入身份/时点：竞标团核心；多团争夺同一「猎杀叛将」悬赏。
初始事件：三支旗插在同一座桥上，叛将只开一枪，倒地的却是你侧翼治疗；攻击顺序决定谁拿首级与谁背锅。
开场白建议：「桥面太窄，容不下三份荣誉。叛将的枪烟散开时，你才发现——真正的敌人站在你旗杆旁边微笑。」
关键NPC立场：**敌对名佣团长**要名声；**教会审判官**要活口审讯；**黑市牙行老板**要尸体与装备；**芮彼泰雅的传闻眼线**（妖精侧探子，具名可作「芮彼泰雅使仆」）观察「是否帕维安血脉」；**团长**要首级与身份不露。
主线钩子/支线：钩子=首级归属与叛将密信；支线=三团停火协议破裂、密信中的日蚀关键词、公会黑名单威胁。
危险度/规避：中高；勿追叛将入魔城层。
任务方向/奖励：名佣契约、高级图纸、政治人情。

**四阶（符文神殿深部 · 觉醒前夜）**
切入身份/时点：觉醒候选/神殿承包团骨干；五系符文冲刺期。
初始事件：符文共鸣失控，友军被己方法术误伤；神殿守护者按站位改「光之颜色」，站错一格治疗变火葬。
开场白建议：「神殿的光不认你的军功，只认你站的那一格。符文在骨血里发烫——觉醒或者烧成焦炭，中间没有第三种客气。」
关键NPC立场：**法师佣兵**要稳定回路；**坦克佣兵**挡反噬；**神殿机制守护者**（敌）；**纳罗达斯相关灾祸残响**（环境/幻象级压迫，非本体）；**团长**要满符文与零减员。
主线钩子/支线：钩子=拿满五系符文并完成关键觉醒；支线=赝品符文案、助手佣兵信物争夺、神殿地下通向帝国遗制的门。
危险度/规避：高；勿硬闯魔城顶层与六魔本体。
任务方向/奖励：满符文、觉醒材、助手缘分、精锐评级。

**五阶（恶魔城层主 · 六魔阴影）**
切入身份/时点：讨伐先锋/古代币承包团；恶魔城中高层。
初始事件：层主技能点名「后排」，需用替身位与攻击顺序骗招；古代币奖励与葬队风险并存。
开场白建议：「魔城的门为古代币而开，也为葬队而开。层主念你名字的发音像点菜单——菜单背后，有人低声提起『六魔星』与『帕维安之子』。」
关键NPC立场：**魔城层主**要清场；**西利亚**（若活跃于世）以疑虑目光试探；**安洁利卡**寻剑途经可能擦肩；**公会观察员**要通关录像与评级；**团长**要古代币与情报；**艾瑞克**若出现则软肋指向埃拉相关传闻。
主线钩子/支线：钩子=层主击破与六魔动向情报；支线=古代币换禁装、魔剑下落线索、日蚀档案残页。
危险度/规避：高；六魔星全盛与八夜本体仅情报/投影，严禁硬刚。
任务方向/奖励：古代币、传说装、六魔相关情报、团内传说佣兵契约。

**六阶（八夜投影 · 日蚀真相碎片）**
切入身份/时点：真相追猎者/多国临时联军顾问；终盘多势力混战。
初始事件：**贝尔佩伦**或**卢修斯**等八夜之一投下投影，王国与教会临时停火；三城警报同响，遗物与日蚀文书成为夺命标的。
开场白建议：「天色变成旧帝国仪典的金。有人喊六魔，有人喊神杀，有人把你的阵图当成献给暗夜的祭品——你必须在『条件性击退』与『全灭』之间选一条缝活下去。」
关键NPC立场：**八夜之主投影（如贝尔佩伦/卢修斯/赛托等，择一降临）**——顶点压迫；**葛兰西特**护 **艾达** 红线；**弗雷莱亚**侧试炼官（若神线开启）；**团长（帕维安之子）**要真相与全团存活；**神圣帝国大主教/大将军**要正统叙事；**芮彼泰雅**要履约或毁约的政治价码。
主线钩子/支线：钩子=条件性击退投影 + 公开一份日蚀真相碎片；支线=神与暗夜旧账文书、帕维安政变动机证言、家门复兴或决裂二选一。
危险度/规避：贴近顶点；**超阶边缘：存在·情报优先 / 条件性胜利**，严禁「削弱归零」。
任务方向/奖励：真相碎片、大陆级声望、神话接口钥匙（非必杀八夜/六魔本体）、终章向团徽与历史定位。
