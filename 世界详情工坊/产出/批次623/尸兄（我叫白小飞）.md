# 尸兄（我叫白小飞）
<!--meta lib=主库 tiers=一、二、三、四、五、六-->

## 剧情

**【作品来源】**
《尸兄》为漫画家**七度鱼**（贾海波）创作的国产搞笑·丧尸·科幻长篇漫画，2011年11月22日起于腾讯动漫连载（部分资料亦记2010年6月28日相关节点），2020年6月13日完结，累计点击破数十亿，是腾讯动漫早期现象级国漫之一。动画第一季由卢恒宇和李姝洁工作室制作；第二季更名《我叫白小飞》，由彩色铅笔工作室与绘梦者联盟制作；2016年开心麻花推出同名舞台剧。另有小说化与周边衍生。气质：前期丧尸求生吐槽喜剧，中期势力混战与能力升级，后期跨城跨族的热血终局，夹杂社会讽刺与友情牺牲。本档案以**漫画正典主线**为骨，动画作视觉与名场面补充；禁止把舞台剧即兴桥段当正史。

**【世界定位】**
近未来现代中国为起点的丧尸末日世界：盗墓棺木坠入H市自来水厂污染水源，饮用者化为可吞噬进化的「尸兄」；少数保留理智者成为可成长的「理智尸兄/尸姐」。主角**白小飞**——H市屌丝平面设计师——因未喝自来水幸免于「被动尸变」，却在救女友与同伴的过程中卷入尸潮、军方、猎人协会、血莲教、东瀛、黑色火线与尸王**龙右**的全球棋局，最终自封「炎黄守护」，与克隆体**黑小飞**合体超神，将尸王封印并换回人间残存的团聚。一句话：**吐槽宅男在尸潮里一路升级，把「我还活着」喊成文明存续的口号。**

**【世界观 · 力量体系】**
力量来源多层叠合，不是单一修仙气感：
1. **尸变与吞噬进化**：普通丧尸无理智；理智尸兄可保留人格并吞噬进化，获得肉体强化与特殊能力。
2. **武术 / 内力 / 功法**：杀势、诗仙剑、游鲨刀法、天罡九重、血莲大法、密宗降魔击、金钟罩等，把末日战抬到「武神阁可参悟」的国术级。
3. **异能**：水球、空间、生命元素重组、空气压缩喷射、镜像、植物操控、光谱排序、次声波等。
4. **体质改造**：黑骑士再生、吸血鬼不死血、钛合金肉盾、天族血脉、神之基因药剂、控脑虫等。
5. **神器与组织武装**：万龙魔心塔、承影剑、唐刀·寒辰、十二生肖机器人、核武级异能者。

**等级阶梯（人→地→天→神→超神）**与舞尸猎人协会 1～9 级大致对应；欧盟另有 S/SS 级口径。破坏力表现对照乐园图鉴（宁低勿高）：
- **普通人 / 普通丧尸 / 民兵**：巷战、冷热兵器互杀，一刀一枪可致命 ≈ **一阶**
- **人级武者 / 初期理智尸兄 / 低级猎人**：拆门掀车、单挑小队 ≈ **二阶**
- **地级 / 地级MAX / 成型小队与变异个体**：清据点、街区级破坏 ≈ **三阶**
- **天级 / 九级猎人 / 城市精锐能力者**：掀建筑群、硬抗重火、跨城追猎 ≈ **四阶**
- **神级（司令、不朽大师常态、龙右中期、血莲教主等）**：区域灾变、改写一城战局 ≈ **五阶**
- **超神（白小飞+黑小飞合体、龙右终局、部分宇宙/地狱级压迫）**：跨区域终战、封印级手段 ≈ **六阶（本库顶点）**

**乐园阶位映射**：路人/普通丧尸≈一阶；人级/初期尸兄≈二阶；地级≈三阶；天级≈四阶；神级≈五阶；超神/终局尸王线≈六阶。世界顶点以漫画终局「超神对决+封印」为上限，**禁止写成爆星九阶**；顶点存在=情报优先/条件性胜利，严禁「把顶点写成战力归零」的偷懒解释。——终局处理是付出代价后的结果，不是开局设定顶点为零。

特殊规则：
- **感染与理智**：被咬有风险；理智是稀缺资源，饥饿与情绪可腐蚀人格。
- **吞噬进化**：强者越吃越强，亦越引各方猎杀。
- **克隆与黑骑士**：黑色火线可量产「黑小飞」类战斗体；再生不等于无敌。
- **信仰 / 神器 / 血脉**：后期引入天族、古尸、地狱之门，战力口径上移但仍落在六阶档内叙述。

**【地理 · 舞台】**
- **H市**：瘟疫源头。自来水厂、尸巢、「尸兄乐园」氛围街区、疯马夜总会（黑色火线据点）、公寓与商场医院学校构成前期全部视野。腐臭、停电、碎玻璃与广播寻人启事是感官底色。
- **L市（浙省，白小飞故乡）**：苟氏集团、枫林晚洗浴中心、舞尸协会分部；亲情与资本线交织。
- **京城**：炎黄首都。军方、特能队、十二生肖机器人、九级猎人镇守；京城大战是中期格局翻盘点。
- **藏区**：不朽大师与密宗体系，降魔击等神级手段出处。
- **蜀山**：对抗天尸军团的保卫战舞台，道门与尸潮对撞。
- **东瀛**：神宫、忍者家族、黑太阳与「神皇」徐福系；海外战把「国漫丧尸」拉成国际战争。
- **葬坟岭**：血莲教总坛，邪教敛财与血祭扩军。
- **珠穆朗玛峰 / 喜马拉雅洞穴**：终局主战场；承影剑、麒麟真人遗言与封印太阳的想象在此落地。
- **其他**：无人岛、梵蒂冈/地狱之门意象、印国/米国/欧盟据点——世界篇地图，低阶只作传说与新闻。

写场景时先定「玩家在哪一层」：H市巷战 ≠ 京城会战 ≠ 珠峰终局。

**【世界剧情线】**

① **棺木入水 · 尸变之夜（开篇）**  
两名盗墓贼用货车运送古棺，意图倒卖；途中棺木坠入水库/自来水系统，污染H市中心城区水源。次日，饮用者陆续异变：先是高热与暴食，再是失去理智的啃食与传播。平面设计师**白小飞**因个人习惯未喝自来水，成为极少数「清醒人类」之一。女友**小薇**（高中同学）来电分手后失联——这一私怨式动机把他从「锁门等死」推进「冲进尸潮找人」。名场面：电梯停运、对门咀嚼声、第一次确认「外面已经不是原来的城市」。世界状态：警察与医院先于军队崩溃，谣言比尸潮跑得快。

② **救小薇 · 遇小惠 · 小队雏形**  
白小飞在市中心尸潮中挣扎求生，结识H市女警**小惠**——她是第一个真正意义上的队友：专业、强硬、会开枪，也最早把「活下去」从个人欲望拉成团队责任。小薇被救出后托付给**鱼博士**等科研侧人物；小惠随后被尸王线相关力量盯上（因容貌酷似龙右幼年玩伴「小彗星」），埋下中期被掳与悲剧的种子。同期登场：**开胃奶**（表弟，后期黑骑士再生体质）、**余晓佳**（空气压缩喷射异能）、**小胖**（武术与唐刀线）等，吐槽与物资争夺并行。主题：**末日里最先死的是「一个人扛」的幻想。**

③ **异能觉醒与H市据点战**  
白小飞逐步获得**水球异能**、肉体强化与战斗直觉，从「能跑」变成「能带队」。小队在超市、医院、学校、夜总会等场景清场：一边搬泡面一边打变异体。出现能力型对手与路人——壁虎功系、变色系、忍者渗透等——把单纯丧尸片升级为「异能乱斗」。**小鹿**（萌神气质，生命元素重组）加入，提供治疗/重组向的关键支援，也带来「非人类同伴能否被人类社会接纳」的伦理问题。名场面：第一次公开以「尸兄」身份站在人类幸存者面前却选择救人；小队内部为「杀或留」理智尸发生争吵。阶段结局：H市出现相对稳定的据点政治——税收式保护、假药交易、电台谣言。

④ **势力入场 · 血莲教与黑色火线**  
尸变不再是天灾独角戏：**血莲教**以救世为名敛财扩军，教主与少教主、左右护法、四大坛主构成邪教军事结构；**黑色火线**军火商在疯马夜总会等据点做丧尸生意与人体改造；**舞尸猎人协会**建立 1～9 级猎人制度，试图把「杀尸」职业化；炎黄军方与**特能队**、**十二生肖机器人**（子鼠、丑牛、卯兔等）代表国家机器的最后体面。白小飞在多方拉拢与猎杀中被贴上「特殊样本」「炎黄守护候选人」等标签。**穆教授**等研究者揭示：污染源与古尸、实验、跨国资本有关——「有人要世界乱」。因果：个人求生变成政治筹码。

⑤ **小惠被掳 · 尸王龙右现身**  
**龙右**（尸王）从幕后走到台前：不死鸟系再生、玄武体、植物操控、万龙之力、天罡九重与血莲大法，使其成为「天级→神级→超神」的完整Boss模板。小惠因「小彗星」投影被其抓走；营救行动把小队推入尸巢与京城方向的死亡行军。小惠线的悲剧（含后续自杀/无法回归等沉重节点，以正典为准）成为白小飞从「吐槽活命」转向「必须变强到能改规则」的心理质变。名场面：尸王降压全场、人类火力无效、同伴第一次真正意义上的永别感。

⑥ **京城大战 · 蜀山 · 东瀛**  
中期格局三次抬升：  
- **京城大战**：九级猎人（黑虎骚博、镜、一眉道长、倪大爷、圣等）、司令与神器**万龙魔心塔**、十二生肖机甲与尸王军团对轰；白小飞首次以国家级战场棋子身份出场。  
- **蜀山保卫战**：天尸军团压境，道门与尸兄同盟临时合作，主题是「正统修者与变异者也能并肩」。  
- **东瀛神宫战**：**徐福**（神皇级）、风魔灰太郎、北辰烈等把战线拉到海外；「长生」与「尸变」被揭示为同一枚硬币的两面。阶段结局：白小飞战力逼近天级/神级门槛，敌人名单从「附近的尸」变成「想重写人类定义的古与洋」。

⑦ **克隆体黑小飞 · 合体超神**  
黑色火线造出**黑小飞**——吸血鬼体质、杀势、不死血、密宗佛法的黑暗镜像。两人从敌对、竞速到理解，最终在终局前完成合体，战力跃迁**超神**。这一设定服务主题：主角必须吞下自己的「黑暗可能性」才能打到顶点，而不是靠天降外挂。**唯欣/小萌**（原血莲教圣女）线在此阶段成为家庭与人性锚点——她不是战力外挂，而是「打完之后还要不要做人」的答案。

⑧ **珠峰终局 · 龙右醒悟 · 封印**  
最终决战锚定喜马拉雅/珠峰洞穴。白小飞（合体超神）与龙右死斗；龙右被打入洞穴后见师傅**麒麟真人**断剑（承影剑）与遗言，得知当年封印涅槃凤凰失败、师兄徐福隐瞒等真相，其「成王」之路建立在被欺骗的仇恨上。龙右放弃挣扎，临死以余力助白小飞返回人间。白小飞将尸王力量/本体以极端手段**封印于太阳**一类终局处理（公开完结资料口径），地球表面尸潮被清除。小萌产子，白小飞打消同归于尽念头，但自身仅余约三十日寿命的余韵收束——完结于「使命完成、日常未尽写」的开放感，作者转《地球尽头》等后续。谁活谁死：核心队友部分牺牲、部分团聚；龙右以醒悟式死亡收束；人类文明残存。

⑨ **贯穿支线**  
- **猎人协会晋升战**：契约者可走职业猎人线。  
- **血莲教内部清洗**：圣女出走、教主吸血式扩张。  
- **欧盟黑暗议会 / 米国三K社意象 / 印国线**：国际篇的丑恶对照。  
- **小金刚（葫芦）反水与回归**：捡漏进化者的立场戏剧。  
- **历史乱入**（秦始皇尸兄、徐福、诗仙剑气质）：搞笑外壳下的「古中国幽灵」。

**【主要人物】**

**白小飞（主角·炎黄守护）**｜性格：屌丝吐槽、怂中带义、重情、口头贱手底下真救人。｜装备·能力：水球与空间异能、药物强化、内力杀势、杀神九式、诗仙剑、游鲨刀法、天族血脉；后期与黑小飞合体超神。｜人物弧光：分手找人的宅男 → H市小队核心 → 国家级战场棋子 → 超神封印者；从「我想活」到「我们要留下人类的活法」。｜立场关系：护小惠/小萌/小鹿一众同伴；对军方与猎人半合作半警惕；对龙右从惧到理解其被骗的仇恨。

**小惠（H市警察·首个队友）**｜性格：专业、刚烈、外冷内热。｜装备·能力：枪械与战术、末日执法经验。｜人物弧光：执法者 → 尸潮中的姐姐型战友 → 被尸王线吞噬的悲剧节点。｜立场关系：最早把白小飞从个人英雄主义拉回团队；其命运是主角黑化/成长的手术刀。

**小鹿（萌神·生命元素）**｜性格：软萌外表下藏着非人尺度的善良。｜装备·能力：生命元素重组（治疗/重构关键）。｜人物弧光：被捡到的「萌物」 → 不可或缺的后勤与伦理镜子。｜立场关系：人类是否把她当武器，是小队底线的试纸。

**黑小飞（克隆体）**｜性格：冷、杀伐、少言，是白小飞压下去的刃。｜装备·能力：吸血鬼、杀势、不死血、密宗佛法。｜人物弧光：兵器 → 镜像自我 → 合体共命。｜立场关系：黑色火线造物；最终选择与本体同一方向。

**唯欣 / 小萌（血莲圣女→妻子）**｜性格：从神坛圣女到烟火妻子的落差感。｜装备·能力：教内秘传与精神韧性（战力非第一优先级）。｜人物弧光：邪教符号 → 人性锚点 → 母亲。｜立场关系：给终局「活下去」的理由。

**开胃奶（表弟）**｜性格：损、粘人、关键时刻不退。｜装备·能力：黑骑士体质、强再生。｜人物弧光：拖后腿亲戚 → 肉盾型可靠战力。｜立场关系：亲情线的活口。

**余晓佳**｜性格：鲜明、易爆名场面。｜装备·能力：空气压缩喷射（阶段内有失能风险）。｜人物弧光：能力者队友的荣光与脆弱。｜立场关系：小队输出位之一。

**小胖**｜性格：直、义气。｜装备·能力：武术、唐刀·寒辰（阶段失落）。｜人物弧光：同学战友的凡人英雄样本。｜立场关系：提醒主角「不是人人都能进化」。

**龙右（尸王）**｜性格：暴烈、执念、被欺骗后仍一度选择毁灭。｜装备·能力：不死鸟、玄武体、壁虎功、次声波、植物操控、万龙之力、天罡九重、血莲大法；形态含雌体/原体/生命魔铠。｜人物弧光：天级灾厄 → 神级暴君 → 知真相后放弃的悲剧王。｜立场关系：最终Boss与主题载体；对「小彗星」投影的执念害死小惠线。

**徐福（神皇）**｜性格：长生执念下的算计者。｜装备·能力：神级MAX向法术与「一力降十会」式压制。｜人物弧光：历史乱入的幕后之一，对龙右身世有隐瞒之罪。｜立场关系：东瀛线核心敌对。

**司令（炎黄）**｜性格：国家机器的冷面负责人。｜装备·能力：神级、神器万龙魔心塔。｜人物弧光：从征用民间战力到承认「炎黄守护」的权威让渡。｜立场关系：可合作，亦可下令牺牲主角。

**不朽大师**｜性格：藏区密宗高僧的压迫感与慈悲并存。｜装备·能力：神级→回调，降魔击、金刚持等。｜人物弧光：提供「以神级手段镇尸」的另一条路。｜立场关系：关键时刻的外援与导师型压力。

**血莲教主**｜性格：神棍+军阀。｜装备·能力：神级、血龙宝刀、血莲大法。｜人物弧光：邪教扩张至被清剿。｜立场关系：中期稳定反派池。

**花田（红衣忍者） / 毒姬**｜性格：任务型刺客与毒控。｜装备·能力：忍术、毒素战场。｜人物弧光：东瀛/渗透线的刀。｜立场关系：可雇可敌，不可交底。

**【叙事基调 · 雷区】**
基调：前期「笑着逃」，中期「骂着打」，后期「哭着赢」。忌把白小飞写成无吐槽的冷酷杀神；忌低阶开局就硬刚龙右/徐福/合体超神；忌抹掉小惠、小鹿、小萌等真名写成「女主群像」；忌无代价全民解药；忌用「其实全是梦」消解尸潮。土著默认不知「契约者/乐园」；把吐槽留给角色性格，不要第四墙拆到破坏沉浸。


**【世界剧情线 · 补全细目（仍属原作主线，非凑字）】**

⑩ **H市「尸兄乐园」日常政治**  
据点形成后，出现三类人：交保护费换围墙的平民、以猎尸换粮的佣兵、以及把尸兄当动物表演的娱乐商人。白小飞小队拒绝「把同伴关进笼子展览」，因此与夜总会资本线冲突。广播电台成为第二战场：谁掌握话筒，谁就能把「尸兄是怪物」或「尸兄是新人类」写成真理。契约者若在此阶段卖假药，会被小惠式角色当场缴械——这是世界的道德底线，不是说教。

⑪ **舞尸猎人考核的残酷**  
协会把杀尸量化成等级：低级任务清小巷，高级任务探巢穴。九级猎人镇守京城级节点，战力对标天级。白小飞一度被视作「不可控样本」而非猎人同僚，必须用战绩换身份证式的合法性。名场面气质：考核场上人类考生被普通丧尸撕碎，理智尸兄反而救人——「谁更像人」的尖锐反问。

⑫ **十二生肖机器人与国家奇观**  
子鼠的热成像与电光、丑牛的霸刀、卯兔的轰炸……机甲是国家「我们还能打」的广告。它们在京城大战中减员惨重，提醒科技奇观也会耗尽。白小飞与机甲并肩时，吐槽外壳下是「肉身进化者」与「钢铁国产」的互相承认。

⑬ **东瀛线的长生骗局**  
徐福把尸变包装成「另一种长生」，神宫仪式与黑太阳实验并行。白小飞在海外第一次感到「中国丧尸片」变成「东亚神话战争」。风魔灰太郎的忍术暗杀、北辰烈的武道，把近战规格抬到天级以上。阶段收获：主角学会在异国语言与枪口下谈判，而不是只靠水球砸场。

⑭ **血莲圣女出走**  
唯欣从教廷符号变成选择爱人的人，教内追杀与教外歧视双压。她的弧光证明：本世界最硬的力量有时不是异能，是「我不回去做你们的神」。与白小飞的结合在终局前完成从战友情到家庭的跃迁，为「余命三十日仍想看孩子」做铺垫。

⑮ **龙右的「小彗星」执念机制**  
尸王并非无来由的疯：幼年玩伴的影像被投射到小惠身上，仇恨与温柔缠在一起，才导致掳人与后续悲剧。写作时禁止把龙右写成纯狂暴Boss；他的每一次「留活口」都可能是执念干扰杀戮效率。终局洞穴遗言把这层执念戳破，完成从灾厄到人的坠落。

⑯ **封印太阳与余命**  
公开完结口径中，白小飞以超神手段处理尸王威胁后，人间尸潮消退，但他自己付出寿命代价。小萌产子的画面打断他的殉道冲动——主题落点是「活着比封神难」。任务结算应允许契约者选择：帮他争取更多日子、记录真相、或守护重建中的H市，而不是强迫扮演第二个超神。

**【世界剧情线 · 关键战役与因果链补全】**

⑰ **H市尸巢总攻**  
尸巢是污染后形成的地下—地表复合生态，普通火力打进去只是送粮。白小飞小队与猎人、军方临时联军从三个入口推进：地表清场、地下爆破、中枢诱杀。名场面气质：狭小管道里的近战、理智尸兄被误认为敌人差点被友军爆头、小鹿在毒气边缘重组伤员。阶段因果：H市从「无限尸潮」变成「可标注的危险区」，为撤离与世界篇腾出叙事空间。

⑱ **京城外围阻击与万龙魔心塔**  
当尸王压力抵达京畿，司令启动神器级演出——万龙魔心塔不是个人光环，是国家级「我们仍握有规则」的宣言。白小飞在塔影下第一次被正式称呼为可合作的「炎黄守护」候选，同时也被写进可牺牲名单。因果：主角获得国家级资源通道，也失去「只是一个吐槽宅」的逃避权。

⑲ **蜀山道门临时同盟**  
天尸军团压境时，道门放下对「尸」字的洁癖，与理智尸兄并肩。降魔击与水球异能出现在同一镜头里，主题是「正统与变异可以共享敌人」。此战减员让读者意识到：高阶不是无敌，是「死得更贵」。

⑳ **克隆工厂与黑小飞觉醒**  
黑色火线的流水线不止造武器，还造「另一个你」。白小飞面对黑小飞时，最难的不是武力，是承认自己若被资本与仇恨浇铸也会变成那把刀。合体不是简单战力相加，是人格层面的吞并与和解——写作时必须给双方对白，而不是闪现变强。

㉑ **终局后的人间**  
尸潮消退不等于乌托邦：空城、谣言、战后清算、对尸兄幸存者的歧视都会残留。白小飞余命三十日的设定要求终局正文克制燃向，多写烟火：月子中心的消毒水味、H市修好的第一盏路灯、有人在废墟上重新开平面设计工作室。契约者的高阶任务可以是「帮这个世界学会正常生活」。

## 阶位切入点

> 阶位↔境界：一阶≈普通人/普通丧尸；二阶≈人级/初期理智尸兄；三阶≈地级/成型小队与变异；四阶≈天级/九级猎人/城市精锐；五阶≈神级；六阶≈超神与终局尸王线。世界顶点（合体超神/龙右终态）为六阶顶，低阶一律「规避顶点·情报优先/条件性胜利」，严禁「遭终局级处理削弱所以战力为零」。

**一阶（尸变第一夜 · 巷战求生线）**

切入身份/时点：H市普通上班族/外卖骑手/实习生，锚定水源污染后第一个停电之夜。

初始事件：单元楼停水停电，对门传来咀嚼声；你冲下楼梯时撞见一个边跑边骂的平面设计师——**白小飞**正喊着去找前女友，走廊尽头白眼邻居已在啃消防栓皮。

开场白建议：「警报像被掐断的尖叫。你推开门，阿婆的眼全白，牙上挂着窗帘布。楼下有人扯着嗓子自我介绍：我叫白小飞——我好像还没死透，但腿比脑子听话。」

关键NPC立场：**白小飞**——要结伴但先抢矿泉水；**小惠**（若已着装巡楼）——先缴你手里的甩棍再问身份；**保安圭哥**——想用铁门把整栋楼焊成棺材；**小薇**（电话线）——只剩忙音与未接。

主线钩子/支线：活过第一夜并确认污染源传言；支线＝救对门小孩、抢到未过期抗生素、护送孕妇到楼顶信号点。

危险度/规避：中——规避尸潮洪峰与持枪劫匪；不要学主角冲市中心。

任务方向/奖励：生存包、手绘楼道地图、与白小飞的「还能说话」临时信任。

**二阶（超市清场 · 尸兄初进化线）**

切入身份/时点：民兵/志愿者/低级猎人学徒，锚定小队首次大型搜刮。

初始事件：超市二楼被尸群堵住，白小飞提议自己去引开——引开途中水球异能失控，冷柜区变成溜冰场与血泊。

开场白建议：「购物车撞翻货架的声音比枪声还响。你看见自称尸兄的家伙把货架当滑板，一边骂娘一边把丧尸引进速冻区。」

关键NPC立场：**白小飞**——要配合更要面子；**开胃奶**——跟着表哥起哄；**余晓佳**——救人优先于搬货；**小胖**——守后门；**男尸兄（怪物控）**——想活捉「稀有变异」。

主线钩子/支线：清场并运出物资车；支线＝发现带编号的药剂箱（火线/实验痕迹）。

危险度/规避：中高——规避突然变异的货架区BOSS。

任务方向/奖励：改装武器、临时编制袖标、抑制剂假药鉴别手册。

**三阶（医院夜战 · 样本与背叛线）**

切入身份/时点：医疗后勤/情报贩/地级猎人，锚定医院夺药。

初始事件：药库密码在**穆教授**手里，交换条件是活体理智尸兄样本——白小飞拒绝，血莲教出价更高，冲突三方化。

开场白建议：「急诊灯还亮着，血已干成黑壳。广播循环寻人启事，最后一声却变成低笑：把还能说话的尸体留下。」

关键NPC立场：**穆教授**——要数据不要感情；**小鹿**——挡在白小飞前；**小惠**——拒绝把队友当样本；**血莲左护法**——价更高刀更快；**玲珑（林珑）**——卖三方情报吃回扣。

主线钩子/支线：夺真药或销毁样本协议；支线＝救出被困护士组、截获血莲入教名单。

危险度/规避：高——规避组织武装与医院地下变异巢。

任务方向/奖励：真抑制剂线索、医院通行权限、地级评定推荐信。

**四阶（京城前夜 · 据点攻防与猎人线）**

切入身份/时点：据点战术顾问/八级猎人候选，锚定H市或京郊大型据点被围、忍者线切断供电。

初始事件：红衣残影掠过烟囱，全城黑灯，尸群借黑暗涌入；**花田**的任务名单上有你的代号。

开场白建议：「整座城像被按灭的电视。你在屋顶看见红影掠过，白小飞在下面吼：谁关的灯？！我怕黑但我更怕被吃！」

关键NPC立场：**花田**——任务第一；**毒姬**——毒雾划界；**九级猎人·镜**——镜像复制你的杀招试探；**司令代表**——要你交出白小飞「配合研究」；**白小飞小队**——守人优先。

主线钩子/支线：夺回供电并守住平民区；支线＝护送十二生肖机器人零件、截获黑色火线改造名单。

危险度/规避：高——规避天级对轰波及；不要在明处指挥。

任务方向/奖励：据点话语权、特殊弹药、猎人晋级考核名额。

**五阶（尸王阴影 · 区域灾变线）**

切入身份/时点：跨城联络官/神级势力观察员，锚定龙右逼近、前哨全灭。

初始事件：卫星图出现「移动的空白」，有人提议用白小飞或小惠做饵；你必须在会议上投票。

开场白建议：「风向变了，腐臭里夹着金属腥。最后一封电报只写：尸王不是传说。白小飞把电报揉了：诱饵？那也得我愿意。」

关键NPC立场：**龙右**——远压全场的灾变本体；**司令**——国家算术；**不朽大师**——主张降魔而非献祭；**小惠/小萌**——反对拿同伴填坑；**小金刚**——立场摇摆的高战力。

主线钩子/支线：延迟尸王路线、转移平民；支线＝夺取承影剑线索、策反一支天尸偏师。

危险度/规避：极高——**禁止正面对抗尸王本体**；只做情报、疏散、破坏再生源。

任务方向/奖励：区域疏散权、神级情报碎片、与炎黄守护的正式同盟契约。

**六阶（珠峰终局 · 徐福/合体超神线）**

切入身份/时点：世界联军观察员/古物专家/超神战场后勤，锚定喜马拉雅终战前后。

初始事件：谈判桌上人类代表先内斗；徐福谈长生，龙右谈毁灭，白小飞与黑小飞的合体光在雪线上升起——有人把你的名字写进「特殊样本」附录。

开场白建议：「铜车马的影子投在现代废墟上。有人用古音说长生，有人用枪指谈判桌。白小飞举手：能不能先说人话？我死过一次，还没学会文言文。」

关键NPC立场：**白小飞/黑小飞（合体）**——拒绝被定义的超神；**龙右**——知真相前后的两种暴君；**徐福**——隐瞒者；**麒麟真人遗志（物）**——真相载体；**小萌**——终局人性投票。

主线钩子/支线：确保封印条件成立（非替主角砍Boss）；支线＝护送承影剑残片、阻止第三方偷尸王血、为余命倒计时的主角换回人间通道。

危险度/规避：贴近顶点——**六阶顶·存在情报优先/条件性胜利**；可做仪式锚点、平民门、真相传递，不可宣称单挑超神。

任务方向/奖励：世界级同盟署名、终局立场选择权、封印后「重建H市」建设令。
