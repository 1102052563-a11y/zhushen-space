## 剧情

**【作品来源】**
《灵剑尊》为云天空所著东方玄幻长篇，2016 年 5 月 31 日起连载于神起中文网（阅文系电子版权），2021 年 8 月完结，全书约 5421–5425 章、总字数约 1213 万至 1460 万字量级（公开资料口径不一，以终章「大结局（下）」为准）。2017 年入选中国作家协会重点作品扶持名单。衍生：iCiyuan 动漫等改编同名漫画（约 530 回完结向）、杭州索以文化 3D 动画（腾讯视频等平台连载，覆盖武府选拔、兽潮、天演门等早期主线）。文风：重生打脸、丹剑双修、后宫与多界升级并行；口号句「待到灵剑长啸之时，天地三界，我为至尊」。本档案以小说设定为骨架，漫画/动画早期章节（提亲、百宝楼、九寒宫、家主）作低阶场景锚点。全本检索入口：[搜笔趣阁 keyword=灵剑尊](https://www.sobqg.com/searchBook.html?keyword=%E7%81%B5%E5%89%91%E5%B0%8A)（站内收录情况以搜索结果为准，可与百科、漫画目录交叉核对）。

**【世界定位】**
真灵大陆西风城起步的武灵剑修宇宙：强者楚行云遭袭后重返十六岁「废物」少主之身，以《吞神灵诀》《混沌剑灵决》等逆修，从楚家、水家婚约与五大武府选拔，一路打到乾坤世界、星辰之海、太古战场与混沌之海，终证道境、成灵剑尊。一句话：重生剑修清算前世、护短成尊、多界称霸。

**【世界观 · 力量体系】**
力量本源为武灵、灵气、剑道与更高层的天道/大道法则。凡人亦可淬体入门，真正分水岭在武灵觉醒与品级：武灵分品（一品起步，可吞噬、晋级至大周天、混沌武灵等），决定资质上限与神通方向。楚行云武灵本为剑，自一品经《吞神灵诀》不断吞灵改造。功法品级自人阶、灵阶至帝级、天帝级；武器品阶常见表述：灵器 → 宝器 → 法器 → 王器 → 皇器 → 无上帝兵 → 无上天兵。丹药一品起（如火灵散），炼器炼丹与剑修并行。特殊系统：轮回石（星辰仙石）与《轮回天书》支撑重生；法身（万象虚空法身、混沌黑龙战体、灵剑战体等）；魂兽/魂骨/战魂（太古战场）；先天帝兵与混沌之源。死亡：低阶身死即灭；中高阶有护身、再生、夺舍与轮回手段，但复仇逻辑允许致命清算；顶点非无限复活。

**基础境界**（淬体至武皇，每境约一至九重）：
- **淬体境**：锻体开筋，西风城少主层常见；战力观感=巷斗、破门、护短立威。
- **聚灵境**：纳天地灵气成灵漩，可外放灵劲、入武府门槛。
- **地灵境 / 天灵境**：区域强者与武府真传层，可镇一城坊市、屠兽潮前排。
- **阴阳境 / 涅槃境**：大宗/宫级中坚，一战可毁城一隅、左右城邦政局。
- **武皇境**（九重）：真灵大陆巅峰向，八阶武皇等表述出现于中后期；一剑裂云，可定一域归属。

**帝境及以上**：
- **帝尊**：掌天地之力，渡天地劫；六劫以上可称天帝，九劫可称天尊（亦与所掌世界数挂钩）。
- **天帝 / 天尊**：一方或多方世界主宰；离掌控范围则难尽全力。
- **祖境**：炼混沌元神（五行元神），祖境高手/大能分档。
- **圣境**：无量级法力与千万法身；自初阶圣尊至古圣（天罡/地煞/归真/大周天/千道等分支，公开资料有细分）。
- **道境**：圣境之上，完美掌握三千大道法则之向；楚行云终局所达，灵剑尊名号落点。更高是否存在书中未完全钉死。

乐园阶位映射（对照《阶位战力图鉴》，宁低勿高）：淬体~聚灵前段≈一阶；聚灵~地灵≈二阶；天灵~阴阳≈三阶；涅槃~武皇初≈四阶；武皇圆满~帝尊≈五阶；天帝/天尊≈六阶；祖境~圣境前≈七阶；高阶圣境~道境≈八阶顶点；道境巅峰/多界主宰向=**超阶巅峰：存在·情报优先/条件性胜利**，严禁「把顶点写成战力归零」。

**【地理 · 舞台】**
- **西风城 / 楚家 / 水家**：开篇主舞台。楚家昔日荣光与衰败、水家婚约、提亲打脸、家族大会、家主之位皆在此。感官：喜堂如刑场、药香与剑鸣、拍卖木槌、血毒青砖。
- **百宝楼与拍卖场**：丹方、墨灵鼎、炎心果、假货与灭口；「神秘高人」报价打脸水千月；秦雨烟线入口。
- **皇朝五大武府选拔场**：一年之约兑现地；天才云集、事故与兽潮、天演门暗杀。
- **九寒宫**：寒脉、九寒绝脉、宫主夜雪裳体系；夜千寒出身地；顾城主嚣张逼宫、水家出手；后期有进攻九寒宫、救流香等高张力节点。
- **流云武府 / 云腾商会 / 万剑阁 / 齐天峰 / 弥天山**：楚行云真灵世界建制与收拢势力后的地盘。
- **大楚皇朝（真灵世界）**：统一/主宰真灵后的政治壳；北荒域、真灵世界之子等身份叠层。
- **乾坤世界**：中期开图，再立大楚、更高资源与帝兵线。
- **星辰之海**：死灵界、玄天仙门、妖莲城、幽冥海、魔祖创魔、北方星域人族圣祖等；三大妖皇级冲突。
- **初级/高级太古战场**：魂兽、战魂、传承魔技；天魔岛、苍水阁、横宇商会、冰霜/烈焰/崩坏战场等副本式版图。
- **混沌之海**：圣魔两族、大道魔神、终局剑道大圆满舞台。
地图逻辑：城邦家族 → 武府皇朝 → 大陆主宰 → 星海 → 太古 → 混沌，契约者切入须认清本阶地图天花板。

**【世界剧情线】**

① **重生提亲与一年之约（开篇·漫画约 1–10 回气质）**  
强者楚行云遭恶人偷袭，灵魂借轮回石等机缘重返十六岁西风城楚家少主之身：体弱、一品武灵、懦弱废柴名。十六年前楚父楚星辰与水家有婚约；今生楚行云赴水家提亲，家主水崇贤推脱，大小姐水千月（淬体九重、四品武灵、西风城第一美女向）当众嘲讽「对不起，你自作多情」。楚行云反击：要娶的不是你，而是地位低下的二小姐水流香（青楼血脉、至阴至寒之体与九寒绝脉），立下**一年之约**——将在皇朝五大武府选拔中证明自己。名场面：喜堂打脸、轮回石异宝确认、灵阶武技初显、「你可以滚了」。主题：信息差复仇、护短定情、隐忍蓄力。世界状态：淬体为主，聚灵已是高人。

② **百宝楼、丹药与家族大会（漫画约 7–30 回）**  
楚行云以「神秘高人」姿态入百宝楼拍卖，打脸水千月；得火灵散丹方、联合百宝楼、武灵「桃花杵」、风雷剑诀、墨灵鼎、炎心果等。秦雨烟登场。家族大会上无耻污蔑与「楚行云，受死吧」式围杀，千钧一发后**家主之位非他莫属**，雷厉风行整顿楚家。黯然血毒（阎毒线）险杀，水流香体贴与护短名场面「流香是我的女人」。阶段结局：废物少主→楚家主；资源网（丹、剑、商）成型。

③ **九寒宫、顾城主与绝脉线（漫画约 44–50 回及延伸）**  
水流香九寒绝脉爆发，九寒宫入局；嚣张跋扈的顾城主逼宫，水家出手。楚行云立誓保护，寒宫规矩与寒脉功法成为中期副本。因果：情线绑定加深，宫级势力成为仇友双刃。可介入：送药、闯宫外围、顾城情报、绝脉续命。

④ **武府选拔、兽潮与天演门（动画主推早期主线）**  
一年之约到期，五大武府选拔天才云集、事故频发；赛后出现人为操控兽潮，神秘暗杀宗派**天演门**连环狙杀。楚行云凭吞神灵诀、伐生淬体诀、剑诀披荆斩棘，重塑肉身与名声。阶段结局：从城邦后起之秀跃入皇朝视野，仇敌清单从家族延伸到暗杀组织。

⑤ **云腾商会、流云武府与真灵称霸**  
收拢商会成立/执掌云腾商会，建流云武府、万剑阁等；唤醒重伤昏睡的父亲楚星辰（醒神草等），营救母亲柳梦烟（星辰古宗第一天才向）。建立**大楚皇朝**，成真灵世界之子与主宰、霸天武皇向。前世仇人今生逐个清算：信息差+资源+剑，而非无脑送头。夜千寒（九寒宫宫主夜雪裳亲传）情线展开，后有多世轮回（狐丽、狐云、苏柳儿等公开资料记述）。

⑥ **乾坤世界与星辰之海（中后期）**  
入乾坤再拓版图；入星辰之海历沧桑，觉醒/坐实魔祖（熬云）等前世身份，创魔族以平衡诸族，与三大妖皇等强者斗争。身份叠层：死灵界主宰、玄天仙门门主、幽冥海之主、北方星域人族圣祖、灵剑天尊等。南宫花颜（东方血脉）、洛澜等情线与幻阵、凝固武灵等名场面并进。武灵向混沌武灵进化，多重法身成型。

⑦ **太古战场与混沌之海（后期）**  
初级太古战场制霸、太古意志之子；高级太古战场多岛主/商会/魔皇身份；混沌之海成圣魔两族主宰、大道魔神。剑道三千、大道法则、帝天弈等强敌与轮回因果收束。斩空剑经王器、皇器直至天帝神兵北斗七星剑等升级链。

⑧ **终局：灵剑长啸，我为至尊**  
约第 5424–5425 章「大结局」上下收束：楚行云问鼎道境巅峰，成灵剑尊；后宫主要妻子（水流香、夜千寒、南宫花颜、洛澜、水千月等）多圆满向；子女楚无意（女）、楚无情（子）等公开资料有载。主题落地：前世债今生还，护短与称尊一体。具体谁死谁服以原作为准，卡片终局宜写「见证成尊+清算余孽」，勿凭空改结局。

⑨ **微观政治贯穿**  
家族贡献点、拍卖假货、丹方版权、联姻退婚、血毒暗杀、武府名额、商会账本、兽潮责任归属、暗杀名单——低中阶正文优先写这些，而不是一上来灭界。


**【世界剧情线·分卷细目】**

**卷一：西风城废柴与提亲（淬体主舞台）**  
楚行云重生后第一要务不是公开称霸，而是活着把「废物」身份用到极致：对外懦弱，对内盘点前世仇人、资源点与婚约漏洞。水家提亲是公开打脸场，也是信息差第一课——水崇贤要的是推责，水千月要的是面子，楚行云要的是水流香与一年之约。轮回石在漫画/小说早期以异宝形态确认重生资本，任何试图染指异宝的旁支、牙人、城中混混，都会在这一卷被当成「试刀石」。  
名场面链：喜堂嘲讽 → 「我要娶的人不是你」 → 轮回石微光 → 灵阶武技初显 → 家族冷眼反转。  
可介入：婚宴宾客、楚家杂役、水家外姓、黑市异宝掮客。危险：聚灵以上强者=规避。胜利定义：不被灭口、换到楚行云人情、摸清西风城权力结构。

**卷二：百宝楼丹剑起家（聚灵~地灵）**  
百宝楼是资源与阴谋的交汇：拍卖大会上「神秘高人」报价打脸水千月；火灵散丹方、墨灵鼎、炎心果、风雷剑诀、武灵桃花杵相继入局。假货与真方并行，灭口队在后巷等鉴定结果。秦雨烟作为商路关键人，可合可算。楚行云用丹养剑、用剑护丹，把「废物少主」洗成「能开价的人」。  
名场面链：木槌与天价 → 假火灵散 → 联合百宝楼 → 武灵晋级 → 再入百宝楼。  
可介入：鉴定学徒、护镖、假丹线人、柜后账房。危险：灭口与城主级插手。

**卷三：家族大会与黯然血毒（地灵~天灵）**  
家族大会是权力绞肉：无耻污蔑、「楚行云受死吧」式围杀、千钧一发后家主之位落地。雷厉风行整顿楚家，贡献点、家法、联姻账本被重写。紧接着黯然血毒（阎毒线）险杀，水流香体贴与「流香是我的女人」把情线从婚约推到生死。斩空剑在此阶段成为立威符号。  
名场面链：罪证木简 → 剑光打断 → 家主之位非他莫属 → 黑血青砖 → 解药争夺。  
可介入：长老会书记、护卫队长、下毒链末端、解药掮客。危险：阴阳境老怪围观即退。

**卷四：九寒宫·绝脉·顾城主（天灵~阴阳~涅槃）**  
水流香九寒绝脉爆发，把舞台从城邦家族拉到宫级。九寒宫规矩森严，宫主夜雪裳体系下的夜千寒冷眼入场；顾城主嚣张逼宫，水家出手，楚行云护短不惜开战。寒脉功法、续命之法、宫内密辛成为中期副本货币。后期漫画另有进攻九寒宫、救流香失控等高张力回声，说明该势力贯穿长线。  
名场面链：哈气成冰 → 宫阶要人 → 「流香是我的女人」 → 九寒绝脉 → 我会保护你。  
可介入：寒宫外门、城邦使节、送药人、罪证收集者。危险：宫主本体=极高规避。

**卷五：五大武府选拔·兽潮·天演门（涅槃~武皇）**  
一年之约兑现。选拔场天才云集、事故频发；赛后人为兽潮冲垮外围，天演门暗杀名单展开。楚行云以吞神灵诀、伐生淬体诀与剑道披荆斩棘，名声从西风城扩到皇朝。此卷主题是「从城邦棋子变成皇朝变量」：考官要政绩，杀手要人头，商会要股份，楚行云要活着赢。  
名场面链：选拔擂台 → 兽潮失控 → 暗杀名单 → 反杀与立威。  
可介入：外聘执事、巡察、商会代理人、兽潮调查。危险：帝尊与天演门总部。

**卷六：云腾商会·流云武府·大楚皇朝（武皇~帝尊）**  
楚行云收拢商路成立/执掌云腾商会，建流云武府、万剑阁、齐天峰、弥天山等据点，统一真灵相关势力为**大楚皇朝**，号真灵世界之子与主宰。父亲楚星辰以醒神草等被唤醒，母亲柳梦烟营救线并行，昇龙会与星辰古宗前史浮出。前世仇人清单按信息差逐个清算：丹、剑、商、政四线并进。夜千寒情线深化，为后文多世轮回埋钩。  
名场面链：商会会长身份确认 → 武府府主 → 父子重聚 → 母线营救 → 皇朝立国。  
可介入：边军司马、商会账房、皇朝使节、旧仇线人。危险：天帝级外来者。

**卷七：乾坤世界与星辰之海（天帝/天尊）**  
乾坤世界再拓版图与帝兵资源；星辰之海则是沧桑主场：死灵界、玄天仙门、妖莲城、幽冥海、八方世界、幽灵战队等身份叠层。楚行云坐实魔祖熬云等前世，创魔族以平衡人妖深渊诸族，与三大妖皇级强者对峙。南宫花颜、洛澜等情线展开，武灵向混沌武灵进化，多重法身成型。北方星域人族圣祖、灵剑天尊等称号落地。  
名场面链：星海裂隙 → 妖皇战书 → 魔祖创族 → 法身显化 → 圣祖认证。  
可介入：引航员、舰队参谋、仙门先遣、妖皇使者谈判。危险：天尊本体与妖皇真身。

**卷八：太古战场（祖境~圣境）**

**【任务世界执行备注（给 AI）】**
写本世界时优先用「武灵/品级/武府/皇朝/星海」术语，勿把境界写成修仙凝气筑基。低阶正文以家族、拍卖、血毒、宫门口角为主；中阶以武府、暗杀、商会、皇朝立国为主；高阶才开星海太古混沌。契约者对楚行云的正确姿态是「可结盟的变数」而非「可碾压的主角光环NPC」。任何「直接抢走轮回石/上帝兵」的剧情须付出血价与追杀链。


**【势力与经济运行细则】**
西风城经济靠坊市、家族贡赋与百宝楼拍卖三角支撑：灵石是硬通货，丹方与武技残页是软通货，武灵品级是隐性信用。楚家衰败时贡赋被水家与城中豪强分食；楚行云夺家主后第一刀往往砍在「假账与白养闲人」上，而非立刻灭敌。云腾商会阶段，账本比剑重要——护送、分成、假货索赔、兽潮保险，都是可写正文的任务。皇朝立国后税收与武府名额挂钩，契约者可做税吏、巡察或黑市对冲。星海阶段货币换成航线权、魂骨与战场通行证；太古以后则以战魂与法则碎片计价。任何越级发财都会引来天演门式追杀或妖皇使者「收购」。

**【修炼与战斗观感（写斗戏用）】**
淬体：拳脚与兵器，破门裂桌，血溅喜堂。聚灵：灵劲外放，可断坊市旗杆。地灵/天灵：一击塌屋，兽潮前排可屠。阴阳/涅槃：毁城一隅，宫门对峙。武皇：一剑裂云，定一域归属。帝尊以上：天地之力、劫光、世界权柄，正文应写「远压」与「布局」而非互殴几十页。剑修特色：剑鸣、剑意、人剑合一气质；楚行云后期更偏法则与法身同显。吞噬武灵不是无代价：反噬、仇家追杀、品级不稳都是可写风险。

**【情线与后宫边界】**
本世界后宫向明确，但低阶切入应优先水流香与水千月对照，中阶引入夜千寒，高阶才并进南宫花颜、洛澜。忌把所有女主同时塞进西风城开局。护短戏要具体：送药、挡剑、拒交人质、当众宣示归属，而不是空喊「我的女人」凑句。夜千寒多世线适合高阶「重逢认人」任务，不适合一阶强行剧透魔祖全史。

**【分卷战役与因果补全】**
西风城提亲受辱立一年之约，是全书情绪发动机：楚行云要的不是脸面，是把水流香从「卑贱二小姐」叙事里拽出来。百宝楼、兽潮、武府选拔把一品剑武灵与吞神灵诀推到台前；九寒宫与夜千寒线把多世轮回从背景板变成刀子。家主之争与楚家内部清洗证明重生者仍会被亲情与利益绑架。出西风城后真灵大陆只是第一张地图，乾坤世界规则改写战力表，星辰之海用航行与资源战争取代城邦斗殴，太古战场用魂骨战魂把死亡变成可交易材料，混沌之海则逼主角直面「尊」字是否值得。帝天奕不是单纯终极怪，是镜像强者；治愈水流香九寒绝脉是情感线的技术解，也是道境权柄的人性用法。五位道侣各有弧光：水流香至阴至寒与救赎，夜千寒轮回与白发代价，南宫花颜战场并肩，洛澜智计，水千月从嘲讽到软化。弟子陆凌、古玄青与太虚噬灵蟒构成传承侧写。动画/漫画只覆盖早期，中后期以小说为准；搜笔趣阁检索页必须保留在来源。映射维持淬体聚灵一阶至道境八阶，宁低勿高，顶点条件性胜利。

吞神灵诀、混沌剑灵决、轮回石与轮回天书、先天帝兵、混沌之源、火灵散等丹、无上帝兵/天兵、魂骨战魂、九寒相关秘药。

**【势力图谱】**
楚家、水家、五大武府、九寒宫、各大陆皇朝、魔祖余绪、道境对手集团。宗旨从家族脸面到界域主宰层层抬升。

## 阶位切入点

> 阶位↔境界：一阶≈淬体~聚灵前；二阶≈聚灵~地灵；三阶≈天灵~阴阳；四阶≈涅槃~武皇初；五阶≈武皇圆满~帝尊；六阶≈天帝/天尊；七阶≈祖境~圣境前；八阶≈高阶圣境~道境。世界顶点（道境灵剑尊/多界主宰）为**超阶巅峰：存在·情报优先/条件性胜利**，低阶切入一律规避顶点。

**一阶（淬体 · 西风城提亲线）**
切入身份/时点：契约者以楚家外姓旁支杂役/水家婚宴宾客身份切入，锚定楚行云重返少年、水家提亲打脸当日。
初始事件：喜堂上水千月当众羞辱楚行云，水崇贤要你「作证他不配」；楚行云却改口提亲水流香，并瞥见你袖中（或身后角落）与轮回石共鸣的微光——有人要你交出「异宝线索」。
开场白建议：「喜堂的红绸像刑场的绳。水千月的声音很脆：对不起，你自作多情了。楚行云笑了一下，那笑让你后背发冷——他说要娶的人不是她。你还来不及起哄，就有人在你耳边压低声音：把你看见的玉光，卖个好价钱。」
关键NPC立场：**楚行云**——需要沉默盟友与不泄密的眼线；**水千月**——要面子与联姻体面；**水崇贤**——要推掉婚约；**水流香**——边缘位置却被突然点名，警惕中带希望；**楚家老一辈**——要联姻资源。
主线钩子/支线：提亲现场不作伪证、护送水流香离场；支线＝追查轮回石流言、偷出被藏的楚家旧账。
危险度/规避：中——规避水家打手群殴与城主级来客；淬体以上强者出现即退。
任务方向/奖励：楚行云人情、基础灵石、灵阶入门武技残页。

**二阶（聚灵~地灵 · 百宝楼拍卖线）**
切入身份/时点：以百宝楼学徒/鉴定学徒身份切入，锚定「神秘高人」拍卖、火灵散丹方与假货灭口前后。
初始事件：楚行云以不可能的价格喊拍，水千月再被打脸；你发现拍品里有假火灵散，柜后老板要你改鉴定结果——今晚有人会杀人灭口。
开场白建议：「木槌落下前，有人喊出不可能的价格。柜后老板擦汗：那少年眼熟。你捏着假丹的蜡封，知道今晚的账，要用血来平。」
关键NPC立场：**楚行云**——借势打脸与收资源；**水千月**——羞愤寻机报复；**秦雨烟**——商路与情报；**百宝楼掌柜**——逐利保命；**阎毒线眼线**——准备下毒或灭口。
主线钩子/支线：揭假货、护送真丹方；支线＝墨灵鼎垫石下落、联合百宝楼分成谈判。
危险度/规避：中高——规避灭口队与地灵以上强者；拍卖后巷是死亡区。
任务方向/奖励：鉴定术、丹方抄本分成、低阶灵器。

**三阶（天灵~阴阳 · 血毒与家主线）**
切入身份/时点：以楚家护卫队长/长老会书记身份切入，锚定黯然血毒与家族大会夺位。
初始事件：楚行云中黯然血毒，水流香要你找解药；下毒者要你改口「是情敌所为」。与此同时，反对派长老要你宣读「叛族罪证」，斩空剑出鞘打断宣读。
开场白建议：「黑血从袖口滴到青砖上。水流香的声音稳得可怕：救他。祠堂香火灭了一半，罪证读到一半，剑光削断木简——楚行云说：家主之位，非他莫属。」
关键NPC立场：**楚行云**——要合法继位与解毒时间；**水流香**——护夫优先；**反对派长老**——诬叛夺权；**阎毒**——完成暗杀；**楚虎**（认主型战力，原作/改编有收服气质）——可被招安。
主线钩子/支线：解血毒、助其继位；支线＝清内部奸细、反杀下毒链。
危险度/规避：高——规避武灵高品围杀与阴阳境老怪。
任务方向/奖励：解药残方、武技残篇、家臣身份与中阶资源。

**四阶（涅槃~武皇初 · 九寒宫与顾城主线）**
切入身份/时点：以寒宫外门使者/城邦使节身份切入，锚定九寒绝脉爆发、顾城主嚣张逼宫。
初始事件：水流香绝脉反噬，顾城主踩在宫阶上要人；九寒宫门内剑鸣，楚行云要闯宫夺续脉之法，水家同时出手——你被要求选边：交人、护人，或两头骗。
开场白建议：「哈气成冰。顾城主的靴子踩在宫阶上：把人交出来。宫门内剑鸣一声，像有人回答：滚。你夹在寒气与剑气之间，第一次明白宫级规矩不是请客。」
关键NPC立场：**顾城主**——要脸与人质；**夜雪裳**——宫规与传承；**夜千寒**——冷眼观察外来者；**楚行云**——护短不惜开战；**水流香**——命悬绝脉；**水家出手代表**——家族颜面。
主线钩子/支线：护绝脉者、闯宫外围送药；支线＝盗寒宫典籍副本、顾城罪证。
危险度/规避：极高——规避宫主夜雪裳本体与武皇级出手。
任务方向/奖励：寒脉功法残卷、结盟信物、城邦避战权。

**五阶（武皇圆满~帝尊 · 武府选拔与天演门线）**
切入身份/时点：以五大武府外聘执事/选拔场巡察身份切入，锚定一年之约兑现、兽潮与天演门暗杀。
初始事件：选拔事故频发后，人为兽潮冲垮外围营地；天演门名单上出现楚行云——也出现你。有人悬赏「带走楚行云人头换上帝兵线索」，武皇只需一剑。
开场白建议：「云被剑气切开。传令兵的腿在抖：兽潮是被人赶的。名单上第三个名字是你。楚行云擦血：那就让他们试试第二剑——你站哪边？」
关键NPC立场：**楚行云**——要活着赢选拔并反杀名单；**天演门杀手**——完成暗杀；**武府考官**——保场与政绩；**云腾商会早期代理人**——资源置换；**水流香**——后勤与寒宫支援（时点允许时）。
主线钩子/支线：拖住暗杀、护住选拔场；支线＝兽潮源头、商会收拢第一笔大单。
危险度/规避：极高——规避帝尊与天演门总部；上帝兵线索多为饵。
任务方向/奖励：武府推荐信、武皇见闻、商会股份残券。

**六阶（天帝/天尊 · 真灵主宰与星海入口线）**
切入身份/时点：以大楚皇朝边军司马/星海航线引航员身份切入，锚定真灵称霸完成、星辰之海初航。
初始事件：楚行云以真灵世界之子身份召开北荒域会盟，同时一道星海裂隙打开——三大妖皇之一的使者投下战书，魔祖名号在暗处回响。你被点名为「引航人」，也有人要你把航线卖给妖皇。
开场白建议：「真灵的风第一次吹出大陆。会盟鼓未歇，星海裂隙里落下妖皇的战书。有人叫他皇朝主，有人叫他魔祖——楚行云只问你：船，开不开。」
关键NPC立场：**楚行云**——要航线与会盟；**楚星辰**——家族后盾（若已团聚）；**柳梦烟**——母系资源；**三大妖皇使者**——试探与收买；**南宫花颜**——中后期情线入口；**洛澜**——关键助力。
主线钩子/支线：护航首航、会盟不被拆台；支线＝魔族创生情报、父母旧案收尾。
危险度/规避：贴近顶点外围——规避天尊本体与妖皇真身。
任务方向/奖励：星海通行令、天帝级功法残页、皇朝封赏。

**七阶（祖境~圣境前 · 太古战场线）**
切入身份/时点：以初级太古战场登记魂修/横宇商会客卿身份切入，锚定太古意志试炼与魂兽传承争夺。
初始事件：魂兽潮与战魂拍卖同日开场，楚行云以「太古意志之子」迹象吸引追杀；有人用你的名册换一张高级战场通行证——卖，或不卖，都可能死。
开场白建议：「太古的风里没有人味，只有魂骨的腥。拍卖槌落下时，你的名册被人抽走一页。楚行云在高台上看你：卖我，还是卖他们——选快一点。」
关键NPC立场：**楚行云**——要制霸初级战场；**魂兽领主**——领地与传承；**横宇商会竞品**——吞并；**夜千寒**——情缘试炼（多世时点允许时）；**帝天弈棋子**——布局夺轮回线索。
主线钩子/支线：助其拿下关键战魂/传承；支线＝通行证黑市、魂骨鉴定反杀。
危险度/规避：极高——规避高级太古战场主宰与圣境来客。
任务方向/奖励：战魂残片、太古功法拓本、商会特许。

**八阶（高阶圣境~道境 · 混沌终局线）**
切入身份/时点：以混沌之海见证修士/圣魔两族调停使身份切入，锚定灵剑长啸、大道魔神与道境成尊前夜。
初始事件：灵剑长啸，前世仇敌余孽与失控神兵同时现世；楚行云问你：要旁观，还是把名刻上剑冢。帝天弈余党抛出「以你换轮回石完整权柄」的条件。
开场白建议：「三界的风都往一把剑上收。有人跪，有人逃，有人笑。楚行云把剑尖抵地：至尊不是称呼，是还债的方式。你——签名吗？」
关键NPC立场：**楚行云**——成尊与还债；**水流香**——终局并肩；**夜千寒**——多世情缘收束；**南宫花颜**——护持子嗣；**洛澜**——法则侧助力；**水千月**——后宫收束；**帝天弈余党**——翻盘；**圣魔两族长老**——站队。
主线钩子/支线：见证成尊并清算余孽；支线＝封印失控神兵、护送子嗣楚无意/楚无情离场（若已出世）。
危险度/规避：顶点——**超阶巅峰：存在·情报优先/条件性胜利**；严禁硬撼道境本体，胜利须来自布局、代价与情报。
任务方向/奖励：终局见证权、剑道残篇、三界通行令、混沌法则碎片（不可无代价）。
