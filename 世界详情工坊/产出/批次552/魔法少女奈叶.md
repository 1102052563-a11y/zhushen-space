# 魔法少女奈叶

## 剧情

**【作品来源】**
《魔法少女リリカルなのは》（Magical Girl Lyrical Nanoha）由都筑真纪原作，Seven Arcs 制作，属「奈叶宇宙」核心动画系列。主轴为 2004 年第一期、2005 年《A's》、2007 年《StrikerS》；后有《ViVid》《Force》漫画与剧场版《The Movie 1st》《The Movie 2nd A's》《Reflection》《Detonation》等。本档案以动画主三部（第一期 / A's / StrikerS）为叙事正典，剧场版作「平行历史」对照，不覆盖漫画后期未动画化的全部支线。文风：魔法少女外壳 + 硬核魔导科技与军事编制；战斗台词（Divine Buster、Starlight Breaker）与友情羁绊并重，强调「保护」与「对话」而非单纯歼灭。检索冲突时优先动画本篇设定与公开百科词条。

**【世界定位】**
多维管理文明「时空管理局」（TSAB）统合的次元治安宇宙。地球是未管理世界（编号约 #97），日常与超常并行；行政中枢在管理世界 Mid-Childa（贝卡逻辑）。主线从地球上的「宝石之种」回收，经「闇之书／夜天之书」事件，再跨越十年到 Mid-Childa 的「圣王的摇篮」危机。主角高町奈叶由普通小学女生成长为 S 级空战魔导师、机动六课核心；菲特·T·哈拉温由被虐待的克隆体成长为执务官；八神疾风由闇之书宿主成为夜天之书主人与部队指挥官。一句话给 AI：这是「魔导器 + Lost Logia + 管理局编制」下，以友情与责任重写灾难循环的次元治安世界。

**【世界观 · 力量体系】**
力量本源是魔力（Mana）与链接核心（Linker Core）：Linker Core 是魔导师体内将魔力成形为术式的「器官／演算核」，可被检测、抽取、再生（再生需时日，期间几乎无法施法）。施法依赖「魔导器（Device）」——从储杖型智能魔导器、武装型、存储型，到合一型（Unison Device）。主流体系：
1. **Mid-Childa 式（贝卡逻辑系）**：程序化、弹幕与远程射击、飞行与结界并重；代表为奈叶、菲特、克罗诺等。
2. **古代贝卡／现代贝卡骑士系**：近战斩击、装甲、骑士礼仪与「骑士」称谓；代表为云骑士团（Signum / Vita / Shamal / Zafira）与部分圣教会系统。
3. **古代 Belka 圣王系遗物**：Lost Logia 级兵器与战舰（圣王的摇篮、圣铠等），可越阶改写战场。
4. **人工魔导师／战斗生化人（Combat Cyborg）与使魔**：Project Fate、Numbers、Type-Zero 等；使魔（如阿尔夫、尤诺的兽形）可共享或辅助主核。

**魔导师阶位（Mage Rank，TSAB 官方）**——衡量「完成任务能力」而非纯火力，但与战斗力强相关（自弱到强）：
F → E → D → C → B → A → AA → AAA → S → SS → SSS  
可带 +/−。武装部队基层多在 D–C；B 是多数人的坎；军官常见 AA–S。**SSS 无正式持有者**；古代 Belka 圣王或可对标。专长分：空战（Air）、陆战（Ground）、综合／支援（Synthetic / Composite）。军衔与魔导师阶位分离：军衔管编制（执务官、舰长、少将等），阶位管任务资格与考核。

**魔导器类型（简）**：
- **智能魔导器（Intelligent Device）**：可对话、自主辅助演算（高町奈叶的 Raising Heart、菲特的 Bardiche）。
- **武装魔导器（Armed Device）**：偏骑士近战（Signum 的 Laevatein、Vita 的 Graf Eisen）。
- **存储／支援型**：治疗、侦察、结界（Shamal 的 Klarwind 等）。
- **合一装置（Unison Device）**：与宿主融合增幅（Reinforce II、Agito 等）。
- **存储装置／书本型**：夜天之书本体系列。

**死亡与复活**：低阶魔导师中弹／重创可死；Linker Core 被抽空会暂时废魔但不等于立刻死亡。Lost Logia 暴走可造成次元崩坏级灾难（闇之书防御程序、宝石之种失控）。高阶存在少有「无限复活」；云骑士团可被书重写召唤，但主人死亡／书完结会重塑命运。强化限制器（Limiter）可人为压低输出以保护身体或合规；解除限制器（如奈叶 Blaster Mode）换取爆发，伴随身体负荷。

**Lost Logia 规则**：古代文明（尤其古代 Belka、传说中的 Al-Hazard）遗留的超规格魔导遗产。管理局核心使命之一即回收、封存监管、研究，防止落入犯罪者或失控。几乎每部主线都围绕一件／一组 Lost Logia 展开。

战力表现（按原作战斗场面，宁低勿高）：
- **未觉醒普通人／地球市民**：徒手与冷兵器，无魔。
- **F–E 级见习**：初级护盾与弱弹，毁木门、制服普通人。
- **D–C 级武装队员**：稳定射击／近战，可毁车辆级目标、压制小规模怪物。
- **B–A 级**：正式前线；可掀翻建筑局部、高速空战，对常规军械有优势。
- **AA–AAA 级**：一城级强者；可清扫街区战斗、对舰支援侧翼。
- **S 级**：战略级个人；全功率弹幕可夷平街区级区域、对 Lost Logia 本体构成威胁（奈叶／菲特典型）。
- **SS 级综合**：大范围结界／炮击与指挥一体（八神疾风后期）。
- **古代 Belka 圣王遗物／摇篮全功率**：准世界级兵器；可威胁管理世界整片区域乃至轨道级。

乐园阶位映射（对照《阶位战力图鉴》破坏力，宁低勿高）：无魔普通人≈一阶初中；F–E 见习／弱使魔≈一阶后～二阶初；D–C 武装队员／初级魔导师≈二阶；B–A 正式前线≈三阶；AA–AAA 精英执务／骑士≈四阶；S 级（受限器常态）≈五阶；S+／解除限制器全力与 SS 综合≈六阶；古代 Belka 圣王遗物全功率、圣王的摇篮级战略兵器、闇之书防御程序暴走态≈七阶（世界顶点·人形／舰体战略武器）。**本世界峰值落在七阶，不设超阶。** 低阶切入一律规避顶点·情报优先。

**【地理 · 舞台】**
- **地球（未管理世界 #97）**：日本上町家附近的日常街区、学校、河川与夜空是第一期与 A's 主舞台。市民不知次元管理局；大规模战斗须结界遮蔽。适合一～三阶「日常中突然卷入超常」。
- **Mid-Childa（管理世界 1 号）**：管理局核心行政星；首都 Cranagan 一带有总局、机场、学院、圣教会自治区。StrikerS 主舞台。城市科技高度发达，魔力与民用并存。适合三～七阶编制任务。
- **Arthra 号等管理局舰艇**：次元航行的武装舰；可展开大型结界、搭载 Arc-en-ciel 等舰载战略武器。A's 终局关键舞台。
- **古代 Belka 相关遗迹／圣王的摇篮**：失落文明战场与战舰内部迷宫；Relic 碎片散落多维。StrikerS 高潮。
- **传说 Al-Hazard**：Precia 追求的「可复活死者」的神话故乡；本篇未完整落地，作危险执念与 Lost Logia 来源传说。
- **Infinity Library、监狱、训练设施、Long Arch（机动六课本部 Southfort）**：编制生活与任务调度节点。
势力地盘：管理局占管理世界治安主权；地球属「不公开介入」；犯罪者（Jail 一党）潜伏地下工场；圣教会在 Mid-Childa 有自治区与骑士传统。

**【世界剧情线】**
① **第一期：宝石之种事件（地球·开篇）**  
上町家三女高町奈叶过着普通小学生生活。某日她救助受伤的雪貂——实为来自 Mid-Childa 的考古／调查系少年魔导师**尤诺·斯克莱亚**。尤诺为回收 21 枚 Lost Logia「宝石之种（Jewel Seeds）」而来：种子可赋予生物异常力量，常令宿主怪物化，并威胁次元稳定。尤诺将智能魔导器 **Raising Heart（高町なのは用）** 托付奈叶；奈叶展现惊人魔力适性，在上学与家务间隙学会变身、射击、结界与飞行。

同期，金发少女**菲特·T·泰斯塔罗萨**与使魔**阿尔夫**亦在地球收集种子。双方多次空战：奈叶坚持「想理解对方」的对话型战斗，菲特则执行「为母亲收集种子」的冷酷任务。真相逐步揭开——菲特是**普蕾希亚·泰斯塔罗萨**以亡女**阿丽西亚**为蓝本制造的克隆体，被植入记忆却遭受虐待；普蕾希亚意图用宝石之种抵达传说 Al-Hazard 以「真正复活」阿丽西亚，本人是管理局通缉级危险人物。

管理局舰 **Arthra** 介入：舰长**琳迪·哈拉温**、执务官**克罗诺·哈拉温**等因破坏规模扩大而登场。奈叶在多次交锋后击败菲特；菲特在被母亲抛弃后转向与奈叶并肩，阻止种子失控。普蕾希亚强行次元跳跃，生死成谜。事件收束：奈叶与菲特结为挚友；菲特随管理局返回 Mid-Childa 接受审查与收养（后姓哈拉温）。世界状态：地球日常恢复表面平静；管理局记下「地球出现 S 级苗子」；21 种子部分回收、部分失控后果成为后续技术隐患（StrikerS 的 Gadget Drone 等）。

名场面因果：Raising Heart 启动的「Raise」；Divine Buster 对轰；Starlight Breaker 集束；菲特被弃时的崩溃；奈叶伸手「我是你的朋友」。主题：**力量是为了保护，战斗是为了对话。**

② **A's：闇之书／夜天之书事件（约半年后·地球）**  
半年后，奈叶与菲特以影像联络维系友情。菲特再访地球之夜，两人遭遇古代 Belka 风格的骑士：**西格纳姆、薇塔、夏玛尔、扎菲拉**——合称**云骑士团（Wolkenritter）**。他们为主人、坐轮椅的少女**八神疾风**收集 Linker Core 魔力以填满「闇之书（Book of Darkness）」的书页。该书本名「夜天之书（Tome of the Night Sky）」，本是知识与术式存储用 Lost Logia，后因污染／防御程序异化，成为会吸干历代主人、完成后暴走毁灭周围、再异地重生的诅咒循环。云骑士团是书的守护程序具现，真心爱护疾风，却不知完整填页将杀死主人。

奈叶、菲特、尤诺与管理局（克罗诺、琳迪、舰员）调查骑士与书的真相。薇塔与奈叶多次近战碰撞，结下「对手兼理解」的羁绊；西格纳姆与菲特剑戟对轰；夏玛尔负责抽取与治疗；扎菲拉守护本阵。疾风本人温柔善良，对骑士视如家人，长期不知自己被书侵蚀寿命。克罗诺一系牵出旧案：约管理局历 0054 年，克罗诺之父**克莱德·哈拉温**曾为阻止闇之书暴走与舰同沉。

终局：书页将满，防御程序（NachtWal 等）暴走；疾风作为真正主人觉醒，在奈叶、菲特、管理局与骑士协助下重编程本书——驱逐污染防御程序，将主程序命名为 **Reinforce（Reinforce Eins）**，并与骑士建立新契约。为永久切断防御程序再生，Reinforce 选择自我消解牺牲。疾风以新「夜天之书」为魔导器重生体系；云骑士团得以作为「人」继续陪伴她。奈叶在战斗中 Linker Core 一度被抽，仍以预存 Starlight Breaker 完成关键一击，之后数日无法施法——体现核心被夺的真实代价。世界状态：闇之书诅咒循环在主线上被终结；疾风、骑士与奈叶菲特结为家人式羁绊；管理局获得处理顶级 Lost Logia 的实绩与伤痕。

主题：**即使是「为了保护主人」的杀戮，也必须被真相与爱纠正；牺牲可以是选择，不是宿命无限重演。**

③ **StrikerS：圣王的摇篮事件（A's 十年后·Mid-Childa）**  
十年后，奈叶、菲特、疾风均已成为管理局中坚：奈叶与菲特为 S 级空战／执务向王牌（常佩限制器保护身体与合规），疾风为综合 SS 级、军衔攀升并推动特种编制。因机场大火等公共安全事件与 Lost Logia 对策需求，疾风主导组建**机动六课（Riot Force 6 / 遗失物管理部机动六课）**，本部 Long Arch。新世代「前卫」加入：**斯巴鲁·中岛、缇安娜·兰斯特、艾利欧·蒙迪亚尔、卡罗·鲁·鲁西**；云骑士团亦在编制内（西格纳姆等）。训练严苛，限制器与考核日常化；部队同时面对官僚掣肘与媒体视线。

主敌**杰尔·斯卡杰蒂（Jail Scaglietti）**：由管理局高层秘密「无限欲望」类计划催生的天才犯罪者／人工魔导师，目标是回收 Lost Logia 级兵器与古代 Belka／Al-Hazard 技术。他制造**Numbers（战斗生化人，Uno、Due、Tre、Quattro…）**、嵌入宝石之种技术的 **Gadget Drone**，并收集 **Relic** 碎片以启动古代 Belka 次元战舰 **圣王的摇篮（Saint's Cradle）**。他与 Project Fate 历史有牵连，一度试图认领菲特为「女儿」；对 Numbers「女儿们」有扭曲亲情。

关键转折：幼女**薇薇奥**被植入 Relic，作为启动摇篮的「钥匙／圣王因子」容器；容貌酷似幼年奈叶。机动六课救出并收养薇薇奥（奈叶与菲特成为其监护／母亲角色）。斯卡杰蒂发动对 Long Arch 与公开活动的袭击：Numbers 与被操纵的召唤师**露特西亚**、骑士**泽斯特**、合一装置 **Agito** 等多线开战；斯巴鲁之姐**银河·中岛**曾被改造成 Number；斯巴鲁与银河身世揭为 Type-Zero 生化人先驱、由中岛家收养。

高潮：摇篮启动并升空，意图利用 Mid-Childa 双月充能至近乎不可阻挡。多线决战——奈叶与薇塔突入舰内，奈叶对上被圣铠强化、意志受控的薇薇奥与 Quattro，解除限制器进入 Blaster 等全力模式；菲特突袭斯卡杰蒂工场；疾风外线指挥与炮击；前卫队地面夺回人质与同伴；西格纳姆对决泽斯特／Agito。奈叶以「母亲」的呼唤与压倒性却克制的力量打通薇薇奥意志；舰体在克罗诺舰队等配合下被阻止并摧毁。斯卡杰蒂与年长 Numbers 入狱；年幼者走更生；机动六课完成历史使命后解散，成员各奔新职。世界状态：管理世界避免了轨道级灾难；Lost Logia 监管加严；奈叶家（奈叶·菲特·薇薇奥）与疾风家（疾风·骑士·Reinforce II 等）进入「战后日常与责任」阶段。

主题：**编制与个人羁绊可以并存；人造生命同样值得被当作人；S 级的真正资格是「在能摧毁一切时选择保护」。**

④ **支线与互文（择要）**  
- **哈拉温家线**：克莱德殉职 → 琳迪收养菲特 → 克罗诺成长为舰长级执务核心。  
- **高町家线**：地球家庭不知全貌但提供情感锚；奈叶「普通女孩」身份与 S 级身份双轨。  
- **圣教会／Belka 传统**：疾风与骑士的信仰与骑士道；自治区政治。  
- **剧场版平行史**：Movie 1st / 2nd A's 重述种子与闇之书，细节与人物互动可有差异，本档案不强制覆盖。  
- **Reflection / Detonation、ViVid、Force**：后续威胁与下一代（薇薇奥成长等）；若玩家世界卡延伸可作后日谈，本常青档以三部主轴为时间脊梁。

时间顺序总览：地球宝石之种 → 半年后闇之书 → 十年后摇篮 → 战后编制重组。任意时点判断世界状态：种子是否回收中、闇之书是否已完结、机动六课是否已成立／解散、薇薇奥是否已被收养、摇篮是否已升空。

**【主要人物】**
- **高町奈叶（主角·地球出身空战魔导师）**｜性格：阳光坚韧、先伸出手再出拳；战斗中仍想理解敌人；对「保护重要之人」偏执到可自毁性训练。｜装备·能力：Raising Heart（智能魔导器，后多次升级；Divine Buster、Starlight Breaker、Blaster Mode 等）；S 级空战；限制器常态。战力曲线：小学生意外觉醒 → 种子战后准 S → A's 与骑士死斗 → StrikerS 成人教育官兼王牌。｜人物弧光：普通小学生 → 「白色魔导师」传说 → 收养薇薇奥的母亲。｜立场关系：菲特挚友／伴侣向羁绊；疾风家人；尤诺初期搭档；对敌人亦给改过之路。

- 薇薇奥的圣王因子：古代 Belka「王」血脉／设计在现代克隆伦理中的回声。

- 剧场版「平行历史」标签：与 TV 细节冲突时，本档案以 TV 三部为主，电影作可选分支。

**【叙事基调 · 雷区】**
画风：明亮魔法少女用色 + 军事电台与战术呼叫；友情戏真挚，战斗戏讲弹道、结界、魔力负荷与限制器。口吻可热血可温柔，忌无脑黑暗虐杀日常角色。  
忌把奈叶写成只会喊「友谊」的软萌花瓶——她是顶级炮击手与教育官。  
忌把菲特写成永远被害妄想——她有成长后的强硬与温柔。  
忌把云骑士团写成无脑反派；他们护主逻辑完整。  
忌用「顶点被莫名废力／战力归零」式解释——摇篮／闇之书暴走态是**七阶战略威胁**，低阶应规避、情报、协作管理局，条件性胜利靠编制与正确目标（断核心／救钥匙／舰炮授权），不是主角一个人肉硬抗世界。  
忌在地球随便对市民暴露魔导与 TSAB（结界与保密是默认）。  
忌低阶发放完整圣王的摇篮控制权或无限制 S 级魔导器。  
最早可切入锚点：尤诺坠落地球、奈叶获得 Raising Heart 前后；或闇之书骑士初现之夜；或机动六课招募期。

## 阶位切入点

> 阶位↔原作：一阶≈地球普通人／F–E 见习边缘；二阶≈D–C 武装队员／初级魔导师；三阶≈B–A 正式前线；四阶≈AA–AAA 精英执务／骑士；五阶≈S 级受限器常态；六阶≈S+ 解除限制／SS 综合；七阶≈古代 Belka 圣王遗物全功率·摇篮级·闇之书防御程序暴走。世界顶点为七阶战略级，低阶切入一律「规避顶点·情报优先」，严禁「顶点战力归零」式解释。

**一阶（地球日常·见习边缘 · 卷入超常）**
切入身份/时点：契约者以「上町附近转校生／邻居家亲戚小孩」身份切入，锚定**尤诺刚坠落、奈叶尚未稳定使用 Raising Heart** 的第一期最早期。
初始事件：放学路上目击受伤雪貂（尤诺）与异光；随后一枚宝石之种激活，路边野猫怪物化袭击行人。奈叶第一次变身赶到，契约者必须在「逃跑隐瞒」与「帮忙疏散市民／递上被种子污染的碎片情报」之间抉择——管理局结界尚未完全展开。
开场白建议：「你刚把书包甩到长椅上，河堤边的风就变了味道。一只像雪貂的小兽跌进草丛，远处柏油路鼓起裂纹，橘色的结晶从地里钻出来。有人喊『变身』，白光里一个和你差不多大的女孩举起红色魔杖——而那只怪物已经朝还在拍视频的路人扑了过去。」
关键NPC立场：**高町奈叶**（警惕但优先救人，对帮忙者道谢并要求保密）；**尤诺·斯克莱亚**（急需人手却怕暴露次元信息）；**菲特·T·泰斯塔罗萨**（若稍晚登场则视你为种子争夺干扰项）；**阿尔夫**（敌意警戒）；路人教师与高町家成员（完全不知情）。
主线钩子/支线：宝石之种首波失控；支线——帮奈叶圆谎不让家人发现战斗伤痕；在学校散布「煤气爆炸」说辞时被好奇同学追问。
危险度/规避：低～中。规避：正面对抗已启动的多种子共鸣；勿企图私藏完整种子。
任务方向/奖励：疏散与情报；奖励取向为初级护符、一次性通讯石、管理局「观察对象」级匿名援助，不可获完整智能魔导器。

**二阶（D–C · 种子争夺的武装边缘）**
切入身份/时点：契约者以「管理局派遣的见习记录员／地球协力者」或「被菲特一方短暂雇佣的本地向导」切入，锚定奈叶与菲特**多次空战、Arthra 尚未全面介入**阶段。
初始事件：废弃工厂区两枚种子同时反应；奈叶与菲特对峙，弹幕撕开屋顶。克罗诺的远程通讯要求「尽量活捉、减少建筑损害」。契约者被要求在交火带架设简易结界柱——柱子却被第三方拾荒者偷走一枚反应碎片。
开场白建议：「工厂的铁皮屋顶像纸一样掀开，蓝与金的弹道在雨里对切。耳机里少年军官的声音冷静得近乎冷酷：『见习，结界柱钉在第三支柱——不要试图劝架。』你刚把柱子插进水泥，就看见一个扛麻袋的影子夹着发光碎片跑向巷子。」
关键NPC立场：**克罗诺·哈拉温**（执法优先，可给临时权限）；**高町奈叶**（希望你别受伤）；**菲特·T·泰斯塔罗萨**（任务中不留情，但厌恶连累平民）；**琳迪·哈拉温**（通讯中的温和施压）；**普蕾希亚**侧使魔（夺碎片）。
主线钩子/支线：种子对轰的降损任务；支线——追查拾荒者黑市；护送尤诺回收数据晶体。
危险度/规避：中。规避：站在 Starlight Breaker 射界正中；向普蕾希亚主动献媚献种。
任务方向/奖励：降损与回收；奖励为制式手杖型低阶魔导器零件、D–C 级防护服、小额管理局点数。

**三阶（B–A · 闇之书骑士初现）**
切入身份/时点：契约者以「地球侧协力魔导师／私立调查事务所助手」切入，锚定 **A's 开局、云骑士团首次与奈叶菲特冲突**之夜。
初始事件：公园夜战，薇塔的铁锤砸碎结界；夏玛尔的空间门一闪，同伴 Linker Core 被点名扫描。疾风在远处公寓窗口咳嗽，骑士为「保护主人」加大吸取力度。契约者被夹在「协助奈叶保护市民」与「被夏玛尔判定为优质核心」之间。
开场白建议：「铁锤落地的震动从脚底传到牙关。粉色双马尾的骑士咬着糖，却用杀气说『把核心交出来』。你护住胸口那团发烫的魔力源时，轮椅上的黑发少女在公寓灯影里轻轻说了句『不要打架』——可书页，正在夜空里自行翻开。」
关键NPC立场：**薇塔**（好战但护疾风）；**西格纳姆**（骑士礼，败则认）；**夏玛尔**（温柔执行抽取）；**八神疾风**（不知全貌，对冲突愧疚）；**高町奈叶／菲特·T·哈拉温**（试图停战对话）。
主线钩子/支线：阻止无差别抽取；支线——潜入八神宅确认疾风病情；向克罗诺递交「骑士非恶意」的证词。
危险度/规避：中高。规避：单独追杀疾风；用平民当盾。
任务方向/奖励：停战谈判与治疗药剂；奖励为 Belka 式护甲碎片、B 级考核推荐信（非直接升 S）。

**四阶（AA–AAA · 书页将满的深水区）**
切入身份/时点：契约者以「Arthra 临时配属执务辅官／骑士对话特使」切入，锚定闇之书**页数接近完成、克莱德旧案被揭开**前后。
初始事件：舰桥警报：防御程序预启动征兆。克罗诺下令准备最坏情况下的舰载兵器授权流程；奈叶坚持最后一次对话。契约者需在「执行清除预案」与「护送疾风到可重编程的仪式阵」间做现场决断，同时维塔与奈叶的死斗余波未平。
开场白建议：「Arthra 的走廊红灯连闪，少年舰长的手套捏得咯吱响。『我父亲就是死在这本书上——但我仍希望有别的办法。』舷窗外，地球的夜云裂开书页形状的黑洞，你听见有人用几乎哭腔喊『疾风』。」
关键NPC立场：**克罗诺·哈拉温**（正义与私怨拉扯）；**琳迪·哈拉温**（授权与否的最终重量）；**八神疾风**（愿担罪）；**Reinforce（将显）**；**云骑士团**（护主优先可违令）。
主线钩子/支线：重编程仪式护卫；支线——稳定奈叶被抽核后的魔力空洞；阻止恐慌军官提前开火。
危险度/规避：高。规避：刺激防御程序完全觉醒；私自尝试「夺书」。
任务方向/奖励：断循环关键助攻；奖励为 AAA 级防护结界模块、夜天之书残页复制知识（只读）、骑士认可的纹章。

**五阶（S 受限 · 机动六课编制期）**
切入身份/时点：契约者以「机动六课应募前卫／外派教官助理」切入，锚定 **StrikerS 前期、Long Arch 训练与 Gadget Drone 灾害**阶段。
初始事件：机场货站 Drone 群暴动；斯巴鲁与缇安娜考核中的「B 级墙」曝光。奈叶戴着限制器指挥降损，同时疾风在通讯里要求「活口与数据」。契约者被派去保护媒体隔离线——却发现记者中有 Due 系渗透者。
开场白建议：「Long Arch 的模拟弹在你耳边炸成白雾，粉发教官的声音却温柔得可怕：『限制器开着也能救人，才是合格的 S。』货站方向黑烟升起，机械虫群爬过广告牌，你的新队友已经冲了出去——而镜头后面，有人在笑。」
关键NPC立场：**高町奈叶**（严教官＋保护者）；**菲特·T·哈拉温**（抓捕渗透）；**八神疾风**（政治与作战双线）；**斯巴鲁·中岛／缇安娜·兰斯特**（争气与互怼）；**杰尔·斯卡杰蒂**（尚未正面，只闻其名）。
主线钩子/支线：Drone 回收与 Relic 线索；支线——帮前卫过考核；追查中岛家旧档案。
危险度/规避：高。规避：擅自解除他人限制器；对媒体泄露六课名单。
任务方向/奖励：编制积分与制式武装；奖励为限制器兼容型魔导器插件、前卫小队临时指挥权、S 级观摩席位（非直接授予 S）。

**六阶（S+／SS · 摇篮启动前夜）**
切入身份/时点：契约者以「六课外勤队长／管理局联合打击组成员」切入，锚定 **薇薇奥被救后、斯卡杰蒂对 Long Arch 发动总攻前后**。
初始事件：发布会现场 Numbers 突袭；银河被改写身份暴露；薇薇奥遭再绑架。菲特杀向工场，奈叶准备突入轨道。契约者负责地面夺回通道——露特西亚的召唤兽压境，泽斯特要求「骑士对骑士」的决斗规则。
开场白建议：「闪光弹熄灭后，会场只剩警报和小孩子的哭声。金发执务官的背影已经化作雷光：『我去抓博士，摇篮不能升起来。』你抬头，Mid-Childa 的双月之间，有什么巨大的影子正在苏醒。」
关键NPC立场：**菲特·T·哈拉温**（斩首斯卡杰蒂）；**高町奈叶**（救女）；**西格纳姆**（应泽斯特之决斗）；**露特西亚**（可谈条件救母）；**Numbers 个体**（敌我混杂，部分可劝降）。
主线钩子/支线：夺回薇薇奥的地面战；支线——说服露特西亚停火；保护更生意向的年幼 Number。
危险度/规避：极高。规避：让摇篮完成双月充能；对薇薇奥本体无差别致命打击。
任务方向/奖励：战役勋章与 SS 级协同认证；奖励为合一辅助短时契约、高阶武装维修权、司法特赦申请票（用于劝降对象）。

**七阶（圣王遗物／战略级 · 摇篮与书之顶点）**
切入身份/时点：契约者以「联合舰队特准观察打击员／疾风直属战略辅佐」切入，锚定 **圣王的摇篮升空决战**或（可选时间支）**闇之书防御程序完全暴走**的顶点窗口——两者皆为世界峰值灾厄，只写「条件性胜利」。
初始事件：摇篮冲破大气，舰体装甲排斥常规弹幕；奈叶在舰内与圣铠薇薇奥对峙，克罗诺舰队等待 Arc-en-ciel 级授权窗口。契约者的任务不是「单挑摇篮」，而是在充能完成前：**切断外部充能阵列／护送爆破组／保证奈叶通讯链路／阻止第二次 Relic 注入**。
开场白建议：「双月的光被战舰吞成一条白河。通讯里粉发女人的呼吸很重，却仍在喊孩子的名字。舰长倒计时已经念到两位数——你手中的起爆密钥冰得像骨头：不是你去推翻世界，是你让世界还有机会被她们救回来。」
关键NPC立场：**高町奈叶**（全力但仍克制杀意）；**薇薇奥**（意志争夺中）；**克罗诺·哈拉温**（战略开火权）；**八神疾风**（外线总指挥）；**杰尔·斯卡杰蒂**（困兽）；若闇之书线则 **Reinforce** 与防御程序互斥。
主线钩子/支线：充能阻断与舰内协同；支线——疏散轨道电梯平民舱；保全证据链以审判高层黑计划。
危险度/规避：贴近顶点。**超阶不适用；本阶即顶点。** 必须「存在·情报优先／条件性胜利」：摧毁／阻止的是摇篮机能或防御程序，不是幻想「肉身硬抗满功率圣王兵器」。严禁把顶点写成「莫名战力归零」。
任务方向/奖励：世界级危机参战印记；奖励为战略协作勋、残骸级 Belka 材料（受监管）、战后家庭线人情（奈叶家／八神家信任），**禁止**直接授予完整摇篮控制权。
