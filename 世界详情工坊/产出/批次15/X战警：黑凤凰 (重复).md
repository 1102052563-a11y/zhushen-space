# X战警：黑凤凰 (重复)

## 剧情

**【作品来源】**
《X战警：黑凤凰》（Dark Phoenix / X-Men: Dark Phoenix）是二十世纪福斯出品、漫威娱乐参与的超级英雄电影，西蒙·金伯格（Simon Kinberg）执导并编剧，2019年6月北美上映，片长约114分钟。属福斯版「X战警系列电影」主线第十二部、前传时间线收官作，承接《X战警：天启》后约九年（主戏锚定1992年）。故事灵感源自克里斯·克莱蒙特与约翰·伯恩漫画《黑凤凰传奇》（The Dark Phoenix Saga），在《X战警：未来昔日》重置时间线后重讲琴·葛蕾成为凤凰的起源；人物命运以**本片电影正典**为准，不倒灌《X战警：最后战役》旧版结局。文风：家庭创伤、变种人权、失控力量与救赎；外星威胁（德巴利人）为前台催化。本条目为世界库**重复档**，与主档可重叠高阶视角，但镜头固定在**八阶战力舞台**：近地轨道、德巴利舰队级集群、凤凰形能量覆盖半空。卡片锚点：**外星舰队毁灭**——琴·葛蕾在地球轨道上单方面屠杀德巴利人的外星舰队，凤凰之力形成的火鸟覆盖了半个天空。

**【世界定位】**
1992年福斯X战警宇宙的「行星/星空级切片」：琴·葛蕾在营救奋进号时吸收宇宙级能量（凤凰之力），经雷德胡克误杀瑞雯、吉诺沙驱逐、纽约内战与政府收容后，在押送列车遭遇德巴利总攻；她完全掌控力量，粉碎外星集群，携领袖伍克冲出大气层，于轨道抹杀对手并以凤凰光体爆发。一句话：当变种人内战与国家收容把所有棋手铐上同一列车时，真正的终局在**轨道**——火鸟覆盖半空，舰队级入侵被一人清空。契约者默认以**八阶**权限入世：可参与轨道通讯、能量光谱记录、舰队残骸回收与「牺牲叙事」定义，不可无代价抹杀瑞雯之死与太空爆发核心因果。本世界库**仅覆盖八阶**。

**【世界观 · 力量体系】**
力量来源为**X基因变种能力**与**外星/宇宙能量（凤凰之力）**并行。本档不写一～七、九阶可玩切片，只把低中阶当作八阶战场的**气压与后勤背景**（宁低勿高）：

1. **凡人军警、低阶变种、学院学员**：枪械、小队能力、封锁线——在八阶战场是「需疏散的质量」与「被碾压的目击者」，非对等战力。  
2. **史考特·桑玛斯光学光束、彼得·麦西莫夫超音速、瑞雯·达克霍姆变形指挥、稳定态琴**：城市压力级——可掀建筑、乱军取首，但在完全释放凤凰面前只能侧翼。  
3. **艾瑞克·蓝歇尔磁场、查尔斯·泽维尔阿尔法心灵、德巴利精锐个体**：地区至准世界级——可瘫痪航空、冻结人群、渗透政府，仍非轨道清场者。  
4. **失控凤凰态琴、伍克抽力仪式、列车联军**：世界级战略节点——一怒可定一城生死，为八阶爆发的直接前夜。  
5. **八阶主落点·行星/星空级**：琴完全掌控后粉碎列车外围德巴利集群（观感等同**舰队级单位被单体清空**）、肉身/能量态横渡近地虚空、携伍克离轨、收回被夺之力并抹杀领袖；太空中凤凰形能量爆发，火鸟覆盖半天空——破坏力对齐「摧毁行星表面/近地轨道战场」、机动对齐「肉身横渡虚空」边缘。  
6. **顶点阴影**：凤凰之力本体曾毁灭德巴利母星及途经星球；琴化为光体消失——对八阶仍为**超阶情报事件**，条件性胜利，严禁战力归零式削弱。

**乐园阶位映射（本世界仅覆盖八阶）**：凡人～城市精英变种≈一～四阶（不覆盖）；艾瑞克/查尔斯/德巴利精锐≈五阶（不覆盖）；吉诺沙与Cerebro战略网≈六阶（不覆盖）；失控凤凰与伍克抽力≈七阶（不覆盖）；**完全释放·轨道清场·火鸟半空≈八阶**（唯一覆盖）；凤凰之力本体/光体存续≈九阶及**超阶**（不写切入点，剧情中仅作远景）。世界顶点＝完全释放之凤凰之力（及与之合一的琴·葛蕾）；八阶切入仍须**规避把顶点当血条硬刚**，以护众、引导离轨、定义叙事、截获舰队残骸为合法胜利。

死亡规则：瑞雯·达克霍姆为真死；多数变种与凡人身死即死；琴终局「消失于凤凰爆发」被学院当作牺牲，片尾火焰凤凰暗示能量态存续——**非廉价满血复活**。特殊系统：脑波增强器Cerebro、X战警战机与黑黄战衣、政府变种人抑制镣铐与押送列车、德巴利变形渗透与舰队/集群投送、凤凰之力（宿主情绪决定可控度）。

**【地理 · 舞台】**
八阶镜头优先下列坐标，低空舞台只作因果前置：

- **近地轨道·奋进号（STS-49）**：凤凰之力入体原点；能量风暴灌入宿主，为日后轨道屠杀埋下「容器」。  
- **地球轨道战场（卡片锚点）**：琴携伍克离轨对决处；德巴利舰队/集群在此被凤凰之力单方面清空；火鸟能量体覆盖半天空，地面可见白昼级光幕。  
- **政府押送列车上空至大气层交界**：八阶爆发的「发射台」——她选择离轨是为保护车厢内所有人，包括曾要杀她的艾瑞克与汉克。  
- **纽约楼宇顶层与收容发车站**：舰队总攻前的地面集结点；德巴利人由此渗透、锁定宿主。  
- **吉诺沙岛沿海**：艾瑞克的隔离家园；琴求庇护被拒，美军直升机被撕碎的画面是「行星级力量失控」的预演。  
- **雷德胡克**：瑞雯之死的情感核爆点——八阶战力若无此因果，只是毁星特效。  
- **泽维尔资优青少年学校 → 琴·葛蕾资优青少年学校**：战前政治秀与战后纪念碑；八阶遗产被建制收编为校名。  
- **巴黎·和平街咖啡馆**：查尔斯与艾瑞克终局对弈；天空再现金色/火焰凤凰剪影——八阶事件的余波观测点。

写场景先定契约者在：**轨道通讯站 / 列车顶层 / 大气层交界观测舱 / 舰队残骸坠落走廊 / 巴黎远眺**，再写冲突；禁止把八阶开局写成小镇枪战。

**【世界剧情线】**
① **前史锁死：1975车祸与精神屏障**  
八岁琴·葛蕾念力失控致车祸，母伊莲·葛蕾身亡，父约翰·葛蕾生还却将她交托查尔斯·泽维尔。查尔斯在其脑海安置**精神屏障**，压抑记忆与峰值——这是日后八阶失控的火药线，不是可有可无的心理描写。

② **1992英雄年：奋进号与宇宙能量入体**  
天启覆灭九年后，X战警（琴、史考特·桑玛斯、奥罗罗·梦洛、寇特·华格纳、彼得·麦西莫夫、汉克·麦考伊、瑞雯·达克霍姆等）被总统征召营救轨道受损的奋进号。琴入舱以念力稳住解体结构，为寇特瞬移疏散宇航员争取时间，自己独自吸收全部耀斑式宇宙能量（凤凰之力）后奇迹生还。地面庆功是查尔斯的政治资本峰值；瑞雯私下不满年轻人被当筹码。因果：容器已成，轨道终局的燃料已加满。

③ **屏障崩解与雷德胡克质变**  
琴力量飙升，险伤学生。查尔斯承认屏障；Cerebro修复失败，反被琴反制。她回雷德胡克见「应死」的父亲，读出被遗弃真相。X战警劝返时，她重创彼得，并在失控念动中**误杀瑞雯·达克霍姆**。汉克叛离，将怒火指向查尔斯。此段破坏力已逼近七阶观感，功能是把「可救的女孩」钉成「必须被轨道级力量自己赎回的罪人」。

④ **吉诺沙驱逐与德巴利接触**  
琴逃往艾瑞克·蓝歇尔的吉诺沙难民营。美军搜捕直升机被她撕碎；艾瑞克救下人类士兵后**驱逐琴**——隔离主义≠屠城。变形外星种族**德巴利人（D'Bari）**渗透地球，领袖**伍克**以银发女子玛格丽特·史密斯形象接近：声称愿教她运用曾毁灭德巴利母星的宇宙力量「造福」——实则要夺力重塑家园并弃杀宿主。琴在孤立中投靠。德巴利人的舰队/集群投送能力在此埋线：他们不是无组织丧尸，而是带着毁星仇恨的流亡军事文明。

⑤ **纽约内战与国家收容**  
汉克联艾瑞克追杀琴；查尔斯携史考特、奥罗罗阻止。艾瑞克被琴扔出摔昏；查尔斯请寇特瞬移近前，让琴读到收养真相，唤回部分良知。伍克抽力被史考特打断。政府变种人管制部队一网打尽双方，镣铐押上列车。所有八阶前夜的棋手——英雄、隔离派、罪犯叙事——被装进同一条金属走廊。

⑥ **列车总攻 → 轨道舰队毁灭（卡片锚点·八阶主舞台）**  
伍克率德巴利人袭击押送编组：破窗、夺钥、优先杀守卫。解锁后，艾瑞克磁场、史考特光束、奥罗罗雷暴、寇特瞬移、彼得残影、汉克蓝兽近战协同，仍难敌伍克。查尔斯以心灵深入琴意识，道歉并承担；琴原谅他，**第一次真正控制凤凰之力**。  
她先护住列车上的同伴免遭撞击与屠戮，再释放潜能——车厢外围、轨道侧翼、低空增援的德巴利**舰队级集群**在火鸟状能量中被单方面清空：金属外壳气化、变形体来不及换皮、指挥链路断裂。此即卡片锚点「外星舰队毁灭」：不是地面巷战的延长，而是**行星表面至近地轨道的不对称屠杀**。  
面对伍克，她深知全面释放会波及友军，于是**将伍克带离大气层**。近地轨道上，她收回被夺力量，抹杀领袖；力量以**凤凰形状**爆发，火鸟覆盖半个天空，地表白昼错乱，雷达与光学观测同时饱和。琴消失于光体，被认定为英勇牺牲。

⑦ **终局遗产**  
学院更名为琴·葛蕾资优青少年学校；查尔斯退休，汉克接任校长。巴黎咖啡馆，艾瑞克与查尔斯再下棋；天空现火焰凤凰。德巴利入侵失败，母舰/舰队战力随伍克覆灭，残余渗透个体（如琼斯体系）仅余清扫级威胁。结局锁死：瑞雯真死、琴牺牲/升维、查尔斯退位、火鸟半空成为变种人史诗级传说。

⑧ **八阶日常可介入（非低阶枪战）**  
轨道碎片再入大气走廊、能量光谱黑市、白宫「非敌对光柱」通稿战、吉诺沙对「是否公开承认被一人救下」的羞辱政治、学院更名典礼安保、巴黎棋局的远距观测权限、德巴利残骸是否交给军方解剖。AI优先这些**行星级事件的后勤与定义权**，而非让八阶契约者去抢小镇面包。

⑨ **蝴蝶效应边界**  
可改：列车钥匙归属、离轨前是否向地面发布「非攻击」信号、舰队残骸坠落点、是否多争取数秒让琴稳定瞄准、巴黎是否公开「凤凰仍存」。不可无代价改写：瑞雯之死的叙事功能、凤凰入体主轴、轨道清场与火鸟爆发形态、查尔斯退位与学院更名。改局部后因果以「更高平民伤亡、核武级误判还击、德巴利第二波渗透、琴更早屠城」回流。

**【主要人物】**
- **琴·葛蕾 / 凤凰（黑凤凰）**｜性格：罪恶感深重，被爱也怕失控；完全掌控后冷静得可怕。｜装备·能力：顶级心灵感应与念动；八阶态可清空德巴利舰队级集群、肉身/能量态出大气层、太空抹杀伍克、火鸟覆盖半空。｜人物弧光：屏障优等生→奋进号容器→误杀瑞雯→投靠伍克→列车原谅查尔斯→轨道清场并消失。｜立场关系：史考特为情感锚；对查尔斯爱恨交织；对契约者先测「是否把她当武器/舰队」。**完全释放态：规避硬刚，条件性胜利/情报优先。**
- **伍克（德巴利领袖） / 玛格丽特·史密斯形象**｜性格：临床式算计，毁星仇恨披「教导」糖衣。｜装备·能力：变形、心灵诱导、吸取凤凰之力；指挥德巴利舰队/集群总攻。｜人物弧光：渗透→收编琴→抽力失败→列车总攻→轨道被抹杀。｜立场关系：八阶前台终局反派；合法目标是切断其与宿主的连接，非在母星仇恨叙事里讲道理。
- **查尔斯·泽维尔 / X教授**｜性格：理想主义与政治算计并存，真心爱学生却替人做主。｜装备·能力：阿尔法级心灵、Cerebro；八阶战场上是「安抚窗口」而非毁舰输出。｜人物弧光：英雄政治峰→信誉崩盘→列车心灵救赎→退休。｜立场关系：艾瑞克宿敌兼挚友；对契约者纳入「如何定义牺牲」的棋盘。

- **汉克·麦考伊 / 野兽**｜性格：科学家外壳下重情，怒转蓝兽。｜装备·能力：超人力量速度、科学；战后掌校。｜人物弧光：忠于查尔斯→瑞雯死后联艾瑞克→列车再联合→校长。｜立场关系：涉瑞雯之死极度敏感。
- **史考特·桑玛斯 / 独眼龙**｜性格：克制，死守「琴可被拯救」。｜装备·能力：光学破坏光束；纽约打断抽力的关键输出。｜人物弧光：恋人→内战希望持有者→列车侧翼。｜立场关系：主张杀琴者与其敌对。

**【势力图谱】**
- **X战警 / 泽维尔学校**：从政治英雄秀崩盘为「救赎一人」；战后以校名纪念琴。  
- **吉诺沙变种人难民营（艾瑞克派）**：隔离家园；被一人救下是战略羞辱也是活命事实。  
- **美国政府·变种人管制体系**：收容列车与可能的误判还击；八阶要处理的是**国家级开火权**。  
- **德巴利人（D'Bari）**：母星毁于凤凰之力的变形流亡军事文明；舰队/集群投送+渗透网；目标夺宿主之力重塑家园。  
- **媒体与全球观测网**：火鸟半空把「神秘」变成全球直播；定义权=战后谁是英雄谁是末日。

敌友逻辑：暴露「可单体清空舰队」=政府与外星双重猎杀；瑞雯之死使「杀琴」与「救琴」同时正义；轨道爆发后所有人必须回答「那算攻击还是护盾」。

**【贵重物品】**
- **凤凰之力（宇宙能量）**：第一至宝/至危；毁星级背景；八阶可引导不可「长期持有复制」。  
- **德巴利舰队指挥链路/残骸核心**：轨道屠杀后的科学与军工争夺物；谁先回收谁定义「外星入侵证据」。

**【大事记时间线】**
1975 车祸·屏障 → 1983 天启覆灭（背景）→ 1992 奋进号·凤凰入体 → 学院失控 → 雷德胡克·**误杀瑞雯** → 吉诺沙驱逐 → 伍克收编 → 纽约内战·政府收容 → **德巴利袭列车·舰队级集群被凤凰清空·轨道抹杀伍克·火鸟覆盖半空** → 学院更名·查尔斯退休 → 巴黎下棋·天空现凤凰。

**【叙事基调 · 雷区】**
八阶舞台是「行星/星空级悲剧」，不是无脑毁舰爽片。忌把琴写成无来由的舰队杀手或无代价无敌女神；忌无代价救活瑞雯；忌用修仙境界词替换X基因/凤凰之力；忌把德巴利写成无组织乌合之众；忌把万磁王写成无脑灭世——他护吉诺沙、拒滥杀。忌写一～七、九阶切入点。最早八阶锚点：**政府押送列车遭德巴利舰队级总攻、琴即将离轨的数分钟**；或**轨道爆发光谱刚被全球观测网捕获的同一小时**。

**【八阶战力舞台·轨道清场细则】**
德巴利人投送不只「几个变形特工」：列车侧翼同时出现低空载具、车顶渗透组、轨道监视体与增援波次——合称舰队级集群。琴释放时，能量不以直线炮击表现，而以**火鸟形场**吞没编队：先断指挥，再气化壳体，最后让变形体在换皮中途失去质量锚定。地面观测者看到的是半边天空被点燃；轨道观测者看到的是「一个生物级光源正在删除一支入侵编队」。契约者若在通讯链，合法动作是广播「非核、非对地攻击、宿主正在带离威胁」；非法动作是向光柱发射拦截弹——会把叙事锁成「人类先动手」。

**【八阶战力舞台·离轨伦理】**
琴看列车一眼再离轨：那不是逃跑，是把毁星级余波带离友军。艾瑞克可能不抬手，史考特可能喊名字，查尔斯的心灵触感从「控制」变成「送行」。契约者若打抑制剂赌她留下当人，可能换来：力量在车厢内提前全开、全员焦化、德巴利反而抢到残存宿主。条件性胜利=让她带着伍克走、让地面不误判开火、让光谱证据证明护众。

**【八阶战力舞台·火鸟半空的政治】**
半空火鸟是军事目标也是宗教符号。军方要拦截参数；媒体要末日标题；吉诺沙要「变种人救了你们」的通稿；学院要「她是牺牲者」的校史。八阶任务奖励常常是**定义权**与**残骸归属**，不是又一件能复刻的神器。谁掌握原始光谱带，谁能在听证会上挡住「对变种人先发制人」法案。

**【分线档案·奋进号作为八阶燃料】**
轨道任务表面是英雄秀：总统电话、媒体待命、战机升空。琴入舱稳住结构时，能量强制灌入——容器化瞬间，团队还在庆祝。八阶回顾此段应强调：没有奋进号，就没有后来的舰队清空；低空角色永远碰不到能量本体，只能碰通讯与着陆隔离。

**【分线档案·瑞雯之死作为八阶债】**
小镇尺度的死，决定轨道尺度的救赎是否成立。彼得以速度接近仍被重创；瑞雯用「回家」而非杀招，死于失控。汉克的恨必须具体到尸体温度，才能解释他为何在列车上仍可能先想动手。契约者在八阶提「早该杀了她」会立刻失去史考特与部分学院线；提「她欠瑞雯一条命所以必须死在轨道」会被汉克短暂认同却被查尔斯反对——站队即阵营。

**【分线档案·吉诺沙与舰队阴影】**
艾瑞克驱逐琴时救美军，校准「恨压迫≠屠城」。德巴利选择在国家收容最虚弱时总攻，是军事时机：宿主与所有强变种被铐在同一金属走廊，磁场英雄变成囚徒，瞬移点有限。八阶写吉诺沙，应写「被一人救下的耻」与「活下来的账」并行。

**【分线档案·纽约抽力失败】**
伍克在楼内对琴说「把力量给我」：抽力仪式若成功，地球会失去容器、德巴利获得毁星权柄。史考特打断不是恋爱脑，是战略阻断。政府收容是第三股收割者。八阶回顾：抽力失败迫使德巴利改为舰队级强攻列车——锚点战的直接起因。

**【组织运行·八阶权限层】**
能插话近地轨道通讯链的人极少：北美防空、光学望远镜网、总统应急小组、Cerebro残余接口、德巴利叛逃频率。契约者身份应落在这些接口之一，而不是「又一个街头变种人」。学院更名典礼需要的是防刺杀与防圣物化，不是再开一次光学光束教学。

**【战力对照·仅供八阶写作】**
列车联军全力≈世界级侧翼，仍清不掉伍克指挥的集群。琴完全释放≈行星/星空级：单体对舰队。伍克≈可威胁宿主的吸取与指挥节点，非毁星本体。凤凰之力本体≈超阶背景：曾毁德巴利母星。写八阶输出用「清空编队、覆盖半空、离轨抹杀」，禁用「开山裂石」等低阶修辞。

**【情感锚点·八阶仍禁止说教】**
史考特的爱是「我见过她选择救人」的证据链；汉克的恨来自瑞雯；艾瑞克的不抬手是承认有比磁场更重的东西；查尔斯的道歉必须晚到列车才有重量。契约者用「你该原谅」说教会被讨厌；用行动挡住一次对地误射、保住一段光谱，才可能换一句真话。

**【终局入世状态】**
学院走廊有琴的肖像；汉克校长令更务实；巴黎棋局可旁观；天空凤凰是高阶彩蛋。德巴利母舰级力量已随伍克覆灭；残余渗透适合作为八阶「清扫第二波」支线，但不可再写成与火鸟对等的舰队。契约者战后最大资本：你是否在半空燃烧时选择相信「那是护盾」。

**【德巴利军事文化】**
母星葬礼记忆共享；对凤凰之力既恐惧又饥渴。战术层级：渗透组夺钥→低空载具压制→轨道监视→领袖抽力/斩首。他们不恋战普通变种人，核心始终是宿主。写作勿丧尸化；应有伍克口令、琼斯换皮、佯攻车头真攻车锁的节奏。

**【光谱与残骸·可玩物】**
原始光谱可证：能量曲线非核弹、轨迹离地而去、峰值在轨道而非城市。残骸含变形组织与合金，军方要武器化，学院要封印，吉诺沙要作为「人类欠变种人」的物证。八阶交易这三份东西，每一笔都改写下一任总统的变种人政策。

**【人物关系在八阶桌上的重估】**
史考特要救；汉克要债；奥罗罗复杂；寇特劝和；彼得抢秒；查尔斯送行；艾瑞克沉默；伍克要容器。契约者的每一句「开火授权」建议，都会被当成站队。伍克的糖衣永远是「我教你用力量」——识破毁星仇恨是情报任务，不是辩论赛。

**【媒体直播与神秘泄露】**
半空火鸟无法用「煤气爆炸」掩盖。神秘大规模泄露已成事实；八阶胜利条件包括「让泄露指向外星入侵而非变种人宣战」。伪造视频与销毁视频同样危险：前者可煽动清洗，后者可包庇第二波渗透。

**【巴黎观测尾声】**
糖霜未化，天空先烧。查尔斯抬眼，艾瑞克不看棋盘。那不是流星。八阶契约者若此时在场，可选择：公开「她还在」、沉默、或把观测数据卖给想复制凤凰的实验室——第三项会触发新一轮全球猎巫，且拿不到稳定复制体。

**【与主档重叠声明】**
本重复档可与《X战警：黑凤凰》主档共享人物与因果，但**玩法镜头锁八阶**：轨道、舰队、火鸟半空、定义权与残骸。不提供一～七、九阶开局剧本。

**【轨道屠杀名场面·分秒因果】**
德巴利第一波破车顶时，押运官的钥匙还在血泊里；第二波低空载具压来时，艾瑞克才把断裂钢梁焊成弧盾；第三波——真正的舰队感——是雷达上连续亮起的增援图标与轨道监视体的同步俯冲。琴在查尔斯心灵触感里从「罪人」切到「保护者」的瞬间，火鸟场不是向外炸，而是先向内收：护住车厢生命体征，再向外抹平编队。伍克试图再次抽力，发现宿主已不再是容器而是意志；她被带离大气层时，地面只看见一道反向的流星。抹杀发生在真空与薄气交界，没有声音，只有光学饱和。火鸟展开的半空面积足以让东西半球部分城市同时进入「白昼错乱」，防空系统把这误读为多弹头再入——这就是契约者必须抢的那九十秒。

**【八阶后勤账本】**
清空一支舰队级集群之后，世界并不会自动和平。需要处理：① 残骸毒性与变形组织扩散；② 全球直播导致的变种人恐慌挤兑与反向圣物崇拜；③ 白宫与吉诺沙对「谁才是拯救者」的叙事战；④ 学院更名是否等于承认国家曾错误收容全体英雄；⑤ 巴黎棋局上空的剪影要不要写入官方年表。八阶奖励若只有「更强能量」是失败的设计；正确的奖励是**权限、证据、人脉与不被清洗的身份**。契约者若在爆发夜选择沉默，可能活得更久，但下一任总统会用「未知高能体」法案合法化更残酷的收容。

**【伍克的母星叙事·可拆穿点】**
伍克对琴说的每一句「我教你用力量造福」都嵌着德巴利葬礼：母星如何在凤凰之力路过时蒸发，幸存者如何学会变形潜入其他文明，如何把「夺回力量」写成种族正义。可拆穿点不在道德辩论，而在行动顺序——她总是先要宿主服从，再提家园；总是先杀约翰·葛蕾一类「多余目击者」，再谈教导；总是在国家机器把变种人铐死时发动总攻。八阶情报任务：把这三条时间线叠给史考特与查尔斯看，比在轨道上对伍克喊话有用一百倍。

**【列车金属学与磁场伦理】**
封闭金属车厢是艾瑞克的主场：枪械、镣铐、钢轨皆为延伸肢体。但当凤凰场展开，金属会先「歌唱」后「服从另一种更重的力」。艾瑞克不抬手，不一定是认输，可能是判断任何磁场拉扯都会干扰琴的瞄准，把毁星余波拽回地表。八阶写他，忌「怂」，宜「算完了」。彼得的速度在此只能抢人不能抢轨道；寇特的瞬移点被烟与白光淹没，每多送一人出爆心就少一次容错。史考特的光束在火鸟背景下像划火柴——仍有用，用于切断德巴利残体靠近钥匙，而不是与光柱比亮度。

**【全球观测网反应链】**
光学望远镜、军用相控阵、业余天文直播、民航避让指令，在同一小时内串联。谁先把「能量峰值在轨道」说清楚，谁就能挡住「对纽约预授权反击」。琼斯类渗透者会抢在你之前把数据改成「对地俯冲」。八阶冲突经常发生在服务器机房与加密频道，而不是拳脚。Cerebro残余接口若仍可用，可定位德巴利变形体的心灵噪声——但扫描全国变种人也会重演收容政治，汉克接任校长后对此极度敏感。

**【学院更名政治】**
「琴·葛蕾资优青少年学校」五个字是纪念也是招魂。激进人类至上团体会来砸牌；崇拜凤凰的变种子弟会来献血式朝圣；军方会要求在校园地下设监测站。八阶入世若落在战后数周，主线不是再打一场舰队，而是**不让纪念碑变成下一场内战的引信**。查尔斯的退休不是遁世，是把政治责任从自己身上卸下，让汉克用更少演讲、更多制度活下去。

**【契约者禁术清单（剧情向）】**
禁止：用乐园外挂稳定复制凤凰之力；无代价改瑞雯生死；把伍克写成可感化的苦命移民领袖而忽略舰队屠城意图；让低阶路人「碰巧」扛住火鸟余波；用修仙飞升解释离轨。允许：牺牲通讯站换取不开火；用自己的八阶抗性挡一次边缘灼烧救下黑匣子；与艾瑞克做「残骸换居留」的肮脏交易；对光体只对话不索取许愿。

**【时间线锚点速查·八阶专用】**
T-9年：天启覆灭，英雄年启动。  
T-0日：奋进号，凤凰入体。  
T+数日：屏障崩、雷德胡克、瑞雯死、吉诺沙逐、伍克收编。  
T+内战日：纽约抽力失败、全体收容上列车。  
T+锚点时：德巴利舰队级总攻、清空、离轨、火鸟半空、琴消失。  
T+战后：更名、巴黎棋、天空剪影。  
契约者默认落点＝T+锚点时前后一小时；次选＝残骸雨与光谱争夺的六小时窗口。

## 阶位切入点

> 阶位↔原作强度：完全释放之琴·葛蕾清空德巴利舰队级集群、离轨抹杀伍克、火鸟覆盖半空≈**八阶**（行星/星空级）；艾瑞克/查尔斯/列车联军≈五～七阶观感背景；凤凰之力本体毁星级≈超阶。宁低勿高。世界顶点（完全释放之凤凰之力及光体琴·葛蕾）为跨阶存在，切入一律「规避顶点·情报优先」；**超阶巅峰：存在·情报优先/条件性胜利**，严禁战力归零式削弱。本世界**仅覆盖八阶**。

**八阶（行星/星空级 · 轨道舰队毁灭与火鸟半空）**

切入身份/时点：契约者以「近地轨道通讯链协作席 / 北美防空光学观测主控顾问 / 总统应急小组变种人事务特派 / Cerebro残余接口守密员 / 德巴利频率破译官（叛逃线）」等高权限身份切入，锚定 **政府押送列车遭德巴利舰队级总攻、琴·葛蕾已粉碎车厢外围集群、正决定是否携伍克冲出大气层的三至十分钟**（卡片：外星舰队毁灭——火鸟覆盖半天空）。亦可同期切入「爆发后一小时内光谱原始带争夺」。

初始事件：你的耳机同时灌进三路声音——① 列车押运频道里金属在尖叫，艾瑞克的磁场把断裂车厢焊成临时盾；② 光学望远镜网报警「非核高能体正在删除低空编队」；③ 一条银冷的外来频率用人类嗓音说「打开她，否则整颗行星只是下一座母星坟」。雷达屏上，德巴利增援波次仍在爬升；车窗反光里，琴的红发已变成火的边缘。有人把冷却抑制剂推进你手里，要你赌她留下当人；史考特的频道在喊「别拦她」；白宫要你在九十秒内判定：那光柱算攻击还是护盾。你必须在「广播非敌对并封锁对空开火权」「打抑制剂赌人格残留」「把最后带宽让给查尔斯的心灵安抚」「抢先锁定伍克真身坐标供琴瞄准」中选主路径——选错，不是死一个小队，是半个天空的误判核武链。

开场白建议：「铁轨的震动还没离开你的骨头，天空已经先白了。你看见编队图标一格格熄灭，像有人用拇指抹掉蚁群。有人对讲机里喊『开火』，有人喊『那是鸟』，有人只是哭。琴的声音从很远的地方传来，轻得像告别：『别跟着。』你的八阶资本刚够改写一条开火授权，而授权外面是整颗行星的白天。」

关键NPC立场：**琴·葛蕾**——已选离轨护众，不请求许可，只确认「他们会没事」；对试图控制她的人可瞬间升为敌；**伍克**——末路反扑，要宿主与力量，视契约者为可吞噬的坐标噪声；**查尔斯·泽维尔**——心灵安抚窗口，要你别把她逼回失控；**艾瑞克·蓝歇尔**——可能不抬手，厌恶你拿变种人当拦截靶；**史考特·桑玛斯**——死守救琴叙事，拦截离轨=宣战；**汉克·麦考伊**——恨与债未消，却可能在光谱证据上与你合作；**奥罗罗·梦洛**——可用天气干扰低空德巴利残波；**寇特·华格纳**——唯一能把人送出爆心的机动点，带宽有限；**彼得·麦西莫夫**——抢秒救人，拦不住轨道；**琼斯（德巴利）**——渗透体制，要你的开火权文件；**美国总统幕僚**——要可控通稿，可灭口；**地面防空指挥官**——手指在红色按钮上发抖。

主线钩子/支线：  
主线钩子＝**外星舰队毁灭 / 轨道清场**：确保琴能携伍克离轨并完成抹杀；同时阻止人类防空把火鸟当敌对目标；让「清空德巴利舰队」被记录为护众而非宣战。  
支线（本阶独有）：① 抢夺/保护爆发光谱原始带，决定听证会叙事；② 引导德巴利残骸坠落无人区，避免砸中城市；③ 识破琼斯伪装，切断第二波渗透频率；④ 在巴黎观测窗出现前，决定是否公开「光体仍存」；⑤ 阻止军方启动复制凤凰实验的紧急拨款；⑥ 与吉诺沙交易「被救事实」换战后变种人居留条款；⑦ 抢救重伤押运员与断裂车厢，换取钥匙与黑匣子；⑧ 在火鸟覆盖半空的直播里植入一句可验证的「非对地攻击」时间戳。

危险度/规避：极高/贴近顶点。死于德巴利破车、友军误伤、对空误射反噬、能量场边缘气化、政治灭口。须规避：拦截凤凰光柱、宣称能永久控制/复制凤凰之力、无代价复活**瑞雯·达克霍姆**、在车厢内强迫琴全开。**轨道段本体不可跟随硬刚**；**凤凰之力本体 / 光体琴＝超阶巅峰：存在·情报优先/条件性胜利**，严禁战力归零解释。

任务方向/奖励：让舰队级入侵在轨道被清空，且地表不因误判开战；奖励取向＝行星级事件档案与光谱原始带、残骸核心（高危不可稳定武器化）、学院更名决议影响力、与查尔斯/艾瑞克的战后直接对话权、国家级通缉撤销或「开火权一票否决」临时权限。禁止八阶直接发放「可复刻的凤凰之力本体」或毁星级恒定外挂。

**【八阶补充·通讯链夺权】**  
切入身份补充：以光学主控/防空副席切入时，初始事件可改为——你的上级已下达「默认敌对」预令，你有四十八秒用光谱曲线证明能量轨迹离地而去。开场白补充：「屏幕上的半边天在烧，光标却还在问你：友军，还是目标。」关键NPC补充：**总统幕僚**要你签字；**琼斯**要你把签字改成开火。支线补充：伪造/保全时间戳、黑掉煽动清洗的直播推流。危险度补充：签字即战争罪或英雄，取决于火鸟落地后的叙事。奖励补充：永久进入应急小组通讯白名单（有监控）。

**【八阶补充·残骸坠落走廊】**  
切入身份补充：以再入大气监测官/吉诺沙密使切入，时点锁爆发后残骸雨。初始事件：三块德巴利指挥核心分别砸向城区、海域与吉诺沙外海——你只能优先偏转一枚。开场白补充：「天在掉铁，铁里还在蠕动。你得在三十秒里决定：救哪一座城，以及是否让艾瑞克看见人类又一次把锅甩给变种人。」关键NPC补充：**艾瑞克**要残骸作证；**军方**要解剖；**汉克**要封印。任务奖励补充：一块惰性化残骸样本、一条吉诺沙永久靠港权、一次「人类欠条」的公开听证席位。
