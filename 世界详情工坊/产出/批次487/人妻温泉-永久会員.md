# 人妻温泉-永久会員

## 剧情

**【作品来源】**
《人妻温泉-永久会員》为轮回乐园休闲库收录的**预约制秘汤旅馆·常客与人妻职员情感经营**情景档案。标题点出「人妻温泉／永久会员」母题，**无与标题逐字对应的单一商业长篇**；气质整合 DLsite 同人圈常见的「人妻温泉」「NTR 秘汤」「一泊二日不倫旅行」系公开作品惯例，并吸收商业 CG／音声作品对「常客制度、问诊式入浴、可叫停的亲密」的写法。可公开对照的氛围源包括：Sage《人妻NTR温泉旅館》（RJ279511）、Alter Maschine《人妻温泉-寝取られの秘湯-》（RJ01209928）、以及 DLsite 词条「人妻温泉」相关短篇／音声系列。媒介印象：同人 CG／ADV／音声跨媒介；搜笔趣阁以本条目标题检索无对应长篇小说书页。本档案以**永久会员卡**为专属锚点，写**预约制、禁忌登记、家庭边界与是否把「九十分钟」谈成「一整段呼吸」**，不写强制掠夺美化，不写数值比拼或通关式推进。

**【世界定位】**
山阴线雾见台站步行十二分钟的完全预约制秘汤「霧乃湯」。契约者以**见习仲居／永久会员新卡持有人**身份进入，在四季会期与人妻职员、常客之间经营距离与心动。一句话：这是「永久会员续约周」里的秘汤人情恋爱世界——核心不是谁征服谁，而是**谁记得你的禁忌、谁肯在你说停时立刻松手**。

**【世界观 · 舞台设定】**
时代为当代日本地方温泉街：巴士末班、乡土料理、地方电视台天气预报、周末东京来的短期客。社会氛围温和却带着口碑压力——雾见台靠旅游与口耳相传活着，「霧乃湯」不做广告，只靠旧名片与介绍状。没有超能力，没有系统面板；唯一接近「超常」的是馆内时间感：暖光、白噪音、硫磺味，会让人误以为外面世界暂停。

软规则（写作必须遵守）：
1. **任何亲密推进前须有明确可观察的同意**（语言确认、主动换门牌、主动留下续约签）；醉酒无意识、沉睡中动手＝危险结局，不当 HE。
2. **「停」永远有效**：客人／职员任一方说停、丈夫来电、门锁从内反锁，都必须立刻中止场景。
3. **禁止偷拍／装监控／录音要挟**；发现隐藏设备即进入黑线并应被叙事谴责。
4. **丈夫与家庭不是纸片背景板**：每位人妻的婚姻有名字、有疲惫、有失败，不是纯为「戴绿帽」而存在的道具。
5. **永久会员不是特权掠夺证**：会员权益＝优先预约、熟悉禁忌史、可申请「延长十分钟静默」；不等于可越过同意。

世界的温度来自：谁总把你的拖鞋摆到左边、谁记得你不喝冰的柚子茶、谁在你肩膀僵硬时先问「今天可以碰吗」、以及关馆铃响三下后还肯留下来收毛巾的人。治愈感不靠奇迹，靠重复：每两周一次的会期、同一句「先问状态再开始」、同一杯课后温果汁。

**【地理 · 生活舞台】**
- **霧乃湯本馆**：二层木造。一楼前台与鞋柜、内汤「薄雾」、露天「夜蝉」；二楼四人房与一间「静间」（只给需要长谈的人）。玻璃门贴「完全予約制・少人数」。
- **内汤「薄雾」**：初诊、正规入浴、公开示范式放松；呼吸交叠是升温主舞台，也是最容易被旁人（其他职员）看见边界的地方。
- **露天「夜蝉」与静间**：加钟、私密对谈、慢热告白多发生于此；适合练习「说不」。
- **茶水角／里间办公室**：排班表贴冰箱门；加班崩溃与和好常在这里。
- **更衣室与回廊**：会后散步与秘密短谈的默认舞台；雨天屋顶会滴水，成为共撑伞的借口。
- **雾见台商店街与周末朝市**：采购柑橘、花与小礼物；节日灯笼下可发生告白级对白。
- **駅前连锁「速汤」对照店**：便宜、快速、没人记得你的禁忌史——衬托霧乃湯的「慢」。
- **各人自宅玄关一角**：个人线后期「上门送恢复卡／还遗忘物」——只到玄关或客厅垫子，尊重家庭边界。
- **山道观景台**：真正离开门牌号的对谈场；在这里才能谈「我们到底在预约什么」。

每个场景的情感功能：初遇＝临时空位被硬塞的体验班；升温＝固定时段续约；冲突＝家庭日程与营业撞车、或被邻居看见共伞；和解＝雨天加开一场修复入浴；告白＝关馆后只剩一盏灯与未收的问诊卡。

**【故事主线 · 情感线】**

**共通线·永久会员续约周**
你因介绍状或临时工签到「霧乃湯」。馆主**柊木沙织**把你塞进见习排班：叠巾、记禁忌、学「先问状态」。第一周你只被允许待在茶水角与更衣室外侧，却已经听见足够多的生活碎片——**白峰玲子**在电话里对孩子说「妈妈今晚加班」时声音发紧；**七瀬真由**把迎客词说得很亮，关灯后却坐在回廊发呆；**遠野あかり**话少，总把拖鞋对齐成一条线；**水無月香**把工作手套洗得比谁都白；**橘美咲**用日程表把自己钉成「可靠」；外派仲居**神崎涼**从合作馆回来时，袖口总有别馆的皂香。

共通线不是「征服全馆」，而是学会三件事：① 记住禁忌比记住外貌重要；② 续约签必须本人自愿；③ 永久会员卡背面印着的不是特权，是「可被拒绝」的提醒。中段出现三次可选择的门：送恢复卡到玄关为止、雨天共伞到站口为止、关馆后是否接受「只喝一杯茶」的邀请。后段把选择题摊开：地方小报闲话、某位丈夫突然回国、连锁店挖角、有人要求「秘密施术」越过同意边界被当场拒客。没有大反派，只有口碑、自尊与家庭角色的拉扯。

**时间节拍参考（写作可用，非死板日程）**
- 第 1 周：见习、禁忌三问、第一次被允许进薄雾外侧。
- 第 2–3 周：固定续约、茶水角崩溃戏、第一次「可以停吗」被当真。
- 第 4 周：送恢复卡／遗忘物上门窗口。
- 第 5–6 周：闲话、丈夫回国、挖角与拒客事件。
- 第 7 周后：观景台对谈、分线收束（专情／共同经营／离开／崩盘）。

**柊木沙织线（馆主·核心）**
外柔内韧的经营者，把温柔都给了客人与家庭，把自己活成永远的提供者。丈夫**黒川健吾**常年在外地工程现场，婚姻不响不破。她的魅力不在「寂寞可乘」，而在她仍努力维持体面：问诊卡折法认真、黑名单用退色蓝墨水、拒客时声音比谁都稳。个人线分三段：
- **规矩期**：你只被她当劳动力；她用店规保护所有人，包括你。
- **卸壳期**：关馆后她肯说「今天我也累」；第一次主动把续约本推到你面前，不是要你签，是要你看「谁被记住了」。
- **选择期**：HE——共同挂名经营或礼优先婚内修复，你成为可被介绍的伴侣而非秘密；BE——你把馆内隐私当谈资或逼她在丈夫回国夜越界，关系冻结，她把你降为仅前台不可触客。
True 倾向：她重新学会为自己预约一场「九十分钟」，而你只是坐在外侧倒茶的人。

**白峰玲子线（夜班骨干）**
两个孩子的母亲，把「坚强」练成习惯，静室会让她情绪决堤。攻略关键是接受沉默，不把落泪当突破口消费。HE：晨间私人入浴，静默依偎型；她开始在家庭与会期之间用诚实而非表演平衡。BE：因闲话暂停会期，只留下一张未用完的果汁券。

**七瀬真由线（元气前台）**
用大声欢迎掩饰迷茫，其实怕被看成「只会笑的人」。攻略：请她决定主题歌单与季节装饰，把决策权交还。HE：转正企划与阳光恋爱；BE：被当成气氛组工具人，笑容冻住后离职。

**遠野あかり线（害羞更衣室担当）**
镜子是敌人，B 室／静间对她像考场。攻略：从不展览她，等她主动点「夜蝉」。HE：公园草地上第一次不躲眼神；Bittersweet：独自整理，留下叠好的毛巾与「谢谢你让我学会说不」。

**水無月香线（清洁与耗材）**
话少、手套永远干净，用劳动代替表达。心结是「被需要＝被使唤」。攻略：在她收工后请她坐下来喝热的，而不是再塞工作。HE：细水长流的婚外心动（氛围向）或干净友情；BE：你把她的劳动当理所当然。

**橘美咲线（排班与会计）**
职场夹缝里只剩六十分钟是自己的。攻略：不追问公司与家庭账本，只把那六十分钟做到安全精准。HE：细水长流；BE：调职离开，留下未用完的会期卡。

**神崎涼线（外派仲居）**
两头跑，家中情感空缺，用工作填满不想回家的夜晚。攻略：不把同情当交换，不把外派当猎场。HE：减少夜宿、建立稳定约会并协商婚姻边界；Bittersweet：她选择先把自己安顿好，与你保持通信。

**黒川健吾线（沙织丈夫·可互动配角）**
不是反派。工程监理，话少，用加班逃避不会吵架的家。与你在商店街居酒屋偶遇后可能成为酒友：他喝两杯后只重复「她把店看得比自己还要紧」。线的意义是让你看见「被叙事简化的那个人」也有体重。互动 HE：你促成一次夫妻对谈而非偷情；BE：你利用他的信任完成背刺，线以他沉默搬家结束。

**共通后日谈**
无论选谁，霧乃湯继续四季会期；未攻略角色保持友好，不恶意拆家。True 氛围＝「一馆多人、各得其所」或「一人专属静间」——由玩家择。永久会员条款被写清：必须双方同意、可随时叫停、禁止代约与强制加钟。

**微观日常事件池**
连锁店挖角时，沙织只回「我们慢一点」；雨天共伞的规矩：谁先开口谁撑伞；黑名单写在签到本最后一页；拒客事件后店内空气清一周；冬日加会总被约满，却仍留一个体验空位；外派日的便当由神崎涼打包，永远多一双筷；关馆铃响三下：第一下收物，第二下道别，第三下只留给还想说的人。耗材不够、温控失灵、雨天屋顶湿滑、过敏报备、会员睡着打鼾、丈夫电话打来时的屏息——每一件都可转化为：谁先蹲下收拾、谁替谁圆场、谁在事后单独道谢。

**永久会员专属流程备忘（本世界独有）**
1. 开场：前台核名 → 禁忌三问（过敏／疲劳／「今天可不可以碰」）→ 选薄雾／夜蝉／静间。
2. 中段：正规流程进行到一半，允许一次「暂停重来」而不扣费——这是沙织立下的馆规。
3. 收尾：果汁、问诊卡归还、预约本只写自愿的下一格。
4. 永久会员特别项：与「人妻温泉」相关的私密道具必须双人清点；遗失一件，当晚全员留下复盘，不罚钱只谈「谁在慌」。
5. 外派日：跟神崎涼走合作路线时，禁止把馆内黑话带到外场；违者由沙织亲自停班一周（情感上等于冷处理）。
6. 雨天预案：更衣室关闭时改走茶水角长谈；共伞只送到站口，不送进玄关。
7. 月末复盘会：每人说一句「这周我被谁记住了」——比业绩数字更重要。

**结局方向**
- **馆内 HE**：专情或共同经营；关系公开到可被介绍的程度；永久会员卡成为「可被拒绝的亲密」象征。
- **修复 HE**：协助某对夫妻沟通，你退回干净的邻人／同事位置。
- **离开 HE**：你选择离开雾见台，留下只谈天气与柚子茶的短信。
- **BE 冷**：偷拍、代约、逼迫加钟、把丈夫当笑话；口碑崩盘，全员拒你。
- **True**：观景台或关馆灯下，至少一次拒绝在「丈夫在家／本人不清醒」时的越界；永久会员条款被重印成更清楚的同意书；霧乃湯明年仍开。

- **主角视点（契约者）**｜姓名外貌自定；默认见习仲居或永久会员新卡持有人｜成长体现为：从「想被特别对待」到「敢被拒绝」；最强能力不是攻略话术，是**在该停的时候停**
