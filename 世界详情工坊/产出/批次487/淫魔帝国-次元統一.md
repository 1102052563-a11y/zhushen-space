## 剧情

**【作品来源】**
《淫魔帝国-次元統一》为轮回乐园休闲库收录的**跨界驿站式奇幻日常·魅魔外交与情感谈判**情景档案。标题点出「淫魔帝国／次元统一」母题，**无与标题逐字对应的单一商业长篇**；气质整合 DLsite 同人圈「サキュバス帝国」「淫魔政策」「跨界交涉」类公开作品惯例，并把「统一」改写为**多世界文化交流与同意外交**，而非征服战争。可公开对照的氛围源包括：《サキュバス・リベリオン～淫魔帝国への反逆》（RJ01284456）、くんかくんか帝国相关淫魔向作品、《淫魔帝国的NTR政策》音声系列氛围，以及 dimension empire 等圈名带来的「次元」语感。媒介印象：同人游戏／音声／短篇跨媒介；搜笔趣阁无本标题长篇书页。本档案以**次元驿站签证官**为专属锚点，写**契约条款、食欲与爱欲的边界、帝国改革派与保守派的客厅辩论**，不写屠城式征服，不写强弱榜。

**【世界定位】**
夹在人类界与魅魔界之间的中立地带「薄暮驿站」。契约者以**驿站见习签证官／文化交流生**身份入境，处理通关、翻译、联谊舞会与「统一草案」听证会。一句话：这是把「次元统一」当成**漫长联姻式外交与生活方式谈判**的奇幻恋爱世界——核心是条款写清楚，而不是谁把谁吃掉。

**【世界观 · 舞台设定】**
世界呈现温和奇幻：双月、琥珀街灯、会走路的契约羊皮纸。淫魔帝国不靠军队推进「统一」，而靠**驿站网络、留学、通婚与共同节日**；激进派想加速，改革派想先立法。魅魔的「进食」在本档案中被伦理化：

1. 合法进食＝双方签署短期情感契约后的亲密，可随时废约；
2. 非法进食＝无同意汲取，触发驿站放逐与社会性死亡；
3. 「统一」成功指标不是占领，是两边市民能在同一张餐桌上不互相恐惧。

软规则：
- 禁止美化强制汲取；发现即 BE／黑线。
- 权力差（贵族／签证官／留学生）必须写进条款，不能靠「天性」跳过同意。
- 人类侧恐惧与魅魔侧饥饿都是真实情感，不是笑话。
- 不出现军队歼灭、强弱排名、等级比拼；冲突用听证会、舆论与停办舞会解决。

世界的温度来自：谁记得你不能碰的香料、谁在听证会上替你改掉侮辱性措辞、谁把「我想吃你」翻译成「我想被你记住味道」。驿站的钟慢半拍，像故意给人改主意的时间；羊皮纸会在条款不公时发出细响，像提醒执笔者脸红。

**【地理 · 生活舞台】**
- **薄暮驿站大厅**：签证、翻译、走失者广播；初遇与误盖章名场面。
- **条款缮写室**：羽毛笔与复写魔法；告白常以「请看第三条」开始。
- **双界食堂**：人类汤与魅魔蜜酒分区又共享长桌；换座即关系进度。
- **联谊舞厅**：月中舞会；跳错舞步比说错话安全。
- **改革派沙龙「绯灯」**：读草案、吵伦理、莉莉丝的墨渍主场。
- **保守派茶室「骨瓷」**：讲究旧礼仪，巴尔贝娜试探你的底线。
- **人类侧临时宿舍「回声」**：想家时的雨声最响；森崎透的便签墙。
- **边境市集**：买卖纪念品与「不会咬人的拥抱券」（玩笑商品，实则需真签名）。
- **观界桥**：两端风景叠影；True 告白默认地。
- **驿站医务与伦理窗口**：非法汲取受害者的第一站——写后果，不写猎奇。

**【故事主线 · 情感线】**

**共通线·统一草案听证会季**
你到任时，帝国公主兼改革派笔杆**莉莉丝・暮织**正推动《跨界亲密与共治草案》。反对她的是礼仪总监**巴尔贝娜**，支持她却更急进的是青年外交官**萨玛拉**，人类侧联络官**森崎透**则担心人类被写成「食材附录」。驿站老资格翻译**诺瓦**教你把脏话翻成可公开的外交辞令；食堂主厨**蜜拉**用食物降低敌意；市集青年**尤里**（明确成年半血）卖拥抱券，其实在收集「被温柔对待」的证据。

共通线四阶段：
1. **相遇**：你盖错一个签证章，却意外让一对跨界情侣多待一天——成为你的传说开端。
2. **升温**：舞会共舞、缮写室改条款、食堂换座、绯灯夜读。
3. **冲突**：激进派试图用「试吃会」绕过同意法；人类侧小报渲染猎奇；巴尔贝娜用旧礼仪羞辱改革派；尤里被两边骂「不纯」。
4. **收束**：听证会公读草案；你选择站改革、站保守、站人类自决、或提出「暂缓统一先立法」的第四路。

**时间节拍**
- 第 1 周：错章、认识诺瓦与蜜拉、第一次进绯灯。
- 第 2–3 周：舞会、第 0 条草稿、食堂换座。
- 第 4–5 周：试吃会阴谋、小报、骨瓷茶战。
- 第 6 周后：听证会、观界桥、分线收束。

**莉莉丝・暮织线（改革派公主·核心）**
外表华丽，签名却写到手指抽筋。心结：人民要她像帝国象征一样「天生会吃也会爱」，她只想先学会被拒绝。童年被强制「示范进食」的阴影（点到为止）使她执着立法。攻略：在条款里为她写上「公主也可说不」的示范句；在她把角上墨绳解下来时，问的是「要不要休息」而不是「要不要我」。HE：草案通过，她以个人身份与你约会，皇冠留在办公室。BE：你把她当统一工具人或强制食者，她放逐你。

**巴尔贝娜线（礼仪总监）**
旧贵族，尖刻，其实怕改革后「礼」消失，人与人只剩欲望。年轻时爱过人类，旧法拆散他们——这是她守旧的真正伤口。攻略：证明新法仍保有尊严；学会用古语道歉。HE：她成为伦理法庭首任主席，与你保持互相敬畏的亲密。BE：你当众撕毁旧礼，她转入地下阻挠。

**萨玛拉线（急进外交官）**
热情、危险、把统一当浪漫。家乡驿站曾因恐惧关闭，故她要「快」。个人线教她「快不是德」。攻略：让她把检讨书写完再谈恋爱。HE：她学会等待签署，转做驿站教育官。BE：她策划无同意试吃会，你必须站到她对立面并公开反对。

**森崎透线（人类联络官）**
眼镜、文件袋、失眠。代表人类恐惧。攻略：不把人类写成可怜食物；帮他删掉文本里的「食材附录」。HE：跨界婚姻合法后的对等恋爱。Bittersweet：他带回人类界做议员，留下共写的条款副本。

**诺瓦线（翻译）**
多语、毒舌、听过太多假告白。心结：自己的话总为别人服务。攻略：让她说一句不翻译的母语心里话。HE：细水长流；她开始接自己的委托而不是只当管道。

**蜜拉线（主厨）**
用胃搞定外交。个人线是「被吃」隐喻的温柔版：她请你先吃饭，再谈情绪。示威之夜她通宵熬汤，两边都安静下来。HE：食堂后厨婚约式日常。

**尤里线（成年半血市集青年）**
机灵、敏感、角芽常遮。成长向可攻略（须完全成年对等）。你帮他确认身份不被两边排斥；拥抱券后来变成官方「同意练习券」。禁止幼化描写。

**结局方向**
- **立法 HE**：草案通过，统一变成长期文化交流年。
- **暂缓 HE**：统一口号降温，驿站成为永久中立家。
- **专情 HE**：你与莉莉丝或他人离开政治中心去市集生活。
- **BE**：无同意汲取合法化；驿站崩塌。
- **True**：观界桥公读「可拒绝条款」；两边市民第一次一起鼓掌；拥抱券制度化。

**微观事件池**
羊皮纸因不公条款自响、蜜酒与清汤被端错桌却促成对话、舞会手套码数不合你现场改、小报记者被诺瓦用七种语言骂走、尤里的角芽被小孩指认你挡在前面、巴尔贝娜的骨瓷杯裂了一道你帮她修而不宣扬。

**【扩写补完·日常长卷】**
听证会季的薄暮驿站像一台慢热的外交机器。每天清晨，羊皮纸自己翻到今日待审条款；若措辞带侮辱，纸边会发涩，执笔者指腹刺痛——这是世界给写作伦理的小机关，不是攻击，是提醒。你很快学会：改一个词，有时比盖一百个章更推进「统一」。

莉莉丝把第 0 条「可以拒绝」贴在绯灯最显眼的墙。萨玛拉嫌它「太慢」，巴尔贝娜嫌它「太新」，森崎透把它抄进人类语小册，诺瓦用七种语言朗读时声音发颤——她的母语里「拒绝」与「活着」是同根词。蜜拉在朗读夜免费供应清汤，说饿着肚子谈欲是不公平的。

试吃会阴谋那周，急进派想在舞厅后厢「示范合法进食」却故意漏掉废约段。你若选择公开打断，舞会音乐停半拍，全场听见条款；莉莉丝会把你的名字写进草案致谢（她坚持致谢也要可拒绝被写上）。你若沉默，BE 倒计时开始：受害者出现在伦理窗口，驿站信誉崩盘。

巴尔贝娜的骨瓷茶战不是比谁更刻薄，是比谁更能在羞辱里保持手不抖。她纠正你的舞姿手位时，其实在测你会不会把「被指导」当成「被占有」。过关方式是：致谢、保持距离、下次主动用古礼问「今日可否共舞」。她年轻时的人类恋人留下的旧胸针仍在抽屉，True 线可让她决定送进博物馆而不是殉葬。

市集侧，尤里的拥抱券从玩笑变成民间同意练习。小孩（远景）只买「握手券」；成人要签真名。有人嘲他「不纯的血不配谈统一」，你挡在前面的方式决定他是依赖你还是与你并肩。并肩才是可攻略条件。

宿舍回声夜里常下雨。森崎透的便签墙从「怕」变成「怕，但想坐同桌」。他教你人类界的饭桌禁忌；你教他魅魔界的废约手势。两人关系的高潮不是床第，是他在听证会上删掉「食材附录」时看你一眼。

后日谈长卷：统一若不强行，会变成「文化交流十年计划」。绯灯与骨瓷每月合开「吵但吃饭」会；拥抱券官方化；诺瓦开设濒危方言夜校；蜜拉的通宵汤写成驿站食谱第一页；莉莉丝的皇冠仍常留在办公室，角上墨绳换成可解开的活结——象征随时可放下象征。

**【人物与日常的再展开】**
莉莉丝的一天往往从改措辞开始。她把「占有性结合」改成「可废止亲密」，把「贡品名录」改成「交流志愿表」，每改一处就在角上墨绳上打一个结，结太多时她会手酸，把绳解开放在桌上——那是她允许自己不是公主的十分钟。你若在这十分钟送茶，她会说谢谢；你若在这十分钟谈统一进度，她会重新系上绳，把温柔收回去。

巴尔贝娜主持的骨瓷茶会有固定流程：先闻茶，再谈旧礼，最后才允许谈「欲」。有人想跳步，她会用古语请对方离席，声音优雅得像驱赶。她对你的考验是连续三次茶会都不失礼：不抢话、不盯着别人的角、不把人类伙伴当翻译机器。第三次过关后，她会让你看见抽屉里的旧胸针，只说「有些法，拆人比拆国快」。

萨玛拉的检讨书越写越长，不是因为错得多，是因为她开始学会列举「如果当时多等一天会怎样」。她的个人线中段有一场夜奔：她想连夜把草案强行贴满大厅。你追上去可以选择一起贴（BE 倾向）、拦住并陪她坐到天亮（HE 倾向）、或叫巴尔贝娜来对质（关系复杂化但可走向互相承认）。拦住时最好的台词不是「你错了」，而是「你家乡的驿站关掉时，有没有人愿意多等你一天？」

森崎透的回声宿舍墙上，便签从单字变成短句：「怕被写成附录」「怕喜欢上不该怕的人」「怕统一成功后人类被温柔地消灭」。他约你去人类侧临时市集吃辣面，辣到说不出话时反而敢说真话。他的恋爱节奏是：先共同删一条恶法，再共同点一盏灯，最后才牵手。任何试图用魅魔魅力「快速拿下」他的写法都会让他退回文件堆。

诺瓦在濒危方言夜校只收八人。课堂上禁止帝国通用语，于是你必须结结巴巴。她笑点很低，却会在你说对「拒绝」的同根词时突然沉默，说「谢谢你让它又被舌头记住一次」。蜜拉则把方言夜校的点心承包了：她相信胃是最短的跨界桥梁。尤里在夜校门口卖拥抱券，学生笑着签真名，像在练一种不伤害人的勇敢。

统一不是终点画面里的旗帜，而是食堂长桌上多出来的共用调味架：人类的辣与魅魔的甜并排放，谁先伸手谁先问「可以吗」。你的故事若写到这一步，就比任何征服演说更接近 True。

**【尾声备忘】**
统一十年计划的第一年，薄暮驿站把「可拒绝」做成门廊浮雕。有人摸着浮雕笑，有人哭，有人只是终于敢说不。你的名字不一定被刻上，但你盖过的每一枚公正的章，都让双月下的长桌更稳一点。

驿站夜班结束时，双月会一前一后沉下。你盖上印盒，听见食堂最后一碗汤的碰撞声，像世界在说：先活着，再统一。


**【听证会季长卷·补完（专属薄暮驿站）】**
「次元统一」在本档案里是文化交流十年计划的政治正确口号，真正推进它的是条款、食堂座位与舞会请柬。羊皮纸在侮辱性措辞下发涩，执笔者指腹刺痛——世界用小机关提醒伦理，不是攻击。莉莉丝把第 0 条「可以拒绝」贴在绯灯沙龙最显眼的墙；萨玛拉嫌慢，巴尔贝娜嫌新，森崎透抄进人类语小册，诺瓦用七种语言朗读时发颤——她的母语里「拒绝」与「活着」同根。蜜拉在朗读夜免费供汤，说饿着肚子谈欲不公平。

试吃会阴谋周，急进派想在舞厅后厢示范「合法进食」却漏掉废约段。你公开打断则舞会音乐停半拍，全场听见条款；莉莉丝把你写入致谢且坚持致谢可拒绝被写上。你沉默则伦理窗口出现受害者，驿站信誉崩盘。巴尔贝娜的骨瓷茶战测的是你会不会把「被指导」当成「被占有」；过关是致谢、保持距离、下次用古礼问「今日可否共舞」。她旧恋留下的胸针在 True 线可送进博物馆而不是殉葬。

市集侧，尤里的拥抱券从玩笑变民间同意练习；成人要签真名。有人嘲半血，你挡在前面的方式决定他是依赖还是并肩——并肩才是可攻略条件。宿舍回声夜雨中，森崎透的便签墙从「怕」变成「怕，但想坐同桌」。关系高潮不是床第，是他在听证会上删掉「食材附录」时看你一眼。后日谈：拥抱券官方化；绯灯与骨瓷每月「吵但吃饭」；诺瓦开濒危方言夜校；蜜拉汤进食谱首页；莉莉丝皇冠上的墨绳改成活结，象征随时可放下象征。

## 休闲切入点

> 本世界为休闲／奇幻外交恋爱向，无生存厮杀主轴。契约者以**驿站日常身份**融入，核心玩法＝条款谈判 + relationship，而非征服。

切入身份：薄暮驿站见习签证官或文化交流生。印章、工牌、宿舍钥匙自然融入。不接受「入侵英雄」开局。

切入时点：统一草案听证会季开幕周——你已学会基础盖章，刚认识莉莉丝的第 0 条草稿，试吃会阴谋未发，舞会在周末。

初始处境：
- 住回声宿舍
- 诺瓦是语言导师，蜜拉管你饭票
- 尚无舞会邀约
- 口袋里有一枚会轻微发热的练习印章（无强制力）
- 文件袋里有森崎透塞来的「怕」字便签复印件

开场白建议：「双月同时从玻璃穹顶照进来，羊皮纸自己翻到待签那一页。改革派公主把墨绳从角上解下，说：先别谈统一，先告诉我你会不会在条款里写『可以拒绝』。食堂方向飘来焦糖与辣椒，人类联络官在你桌角留下一张写着『怕』的便签。你的第一份工作不是征服任何一个世界，而是让两边在同一张长桌上把汤喝完。」

可攻略对象：
- **莉莉丝・暮织**：帮她守第 0 条；钩子：墨渍、听证会；吃尊重其拒绝
- **巴尔贝娜**：用尊严说服她；钩子：手位纠正；吃古礼里的诚实
- **萨玛拉**：拉她回等待；钩子：检讨书；吃敢当面对抗她的冲动
- **森崎透**：删食材附录；钩子：便签「怕」
- **诺瓦／蜜拉／尤里**：语言、胃、市集身份三路日常

日常玩法钩子：
1. 签证与错章事件。
2. 条款共写与第 0 条攻防。
3. 舞会礼仪与手位。
4. 阻止无同意试吃会。
5. 市集舆论与尤里身份。
6. 观界桥公读与暂缓／立法分支。

氛围/雷区：外交喜剧＋同意情色；忌强制汲取、忌战争化统一、忌强弱措辞。开局口诀：**先写可拒绝，再谈喜欢。**

优先戏：错章、第 0 条、舞会手位、公开打断试吃会；不是帝国军压境。

若玩家试图用「魅魔天性」跳过签署，羊皮纸自响并触发巴尔贝娜或诺瓦的训诫事件。第一周不可进入非法汲取描写；任何此类倾向直接 BE 警告。

补充：听证会席位可玩「发言顺序」小机制——你让谁先说话，会影响派系好感，但 True 更看你是否保护第 0 条，而不是你站哪一派的队。


【扩写补完·切入细则】
推荐开局 48 小时脚本：抵达回声宿舍 → 食堂领饭票见蜜拉 → 语言速成见诺瓦 → 柜台实习盖练习章 → 误盖真章触发情侣多待一天 → 绯灯旁听第 0 条讨论 → 收到森崎透「怕」字便签。此顺序自然建立七人面孔，无需后宫点名。

身份证件：见习签证官佩灰色印，无永久放行权；文化交流生佩绿条，更易进市集不易进缮写室。两种都合法。禁止「帝国将军／征服者」自称。

开场对话备选：

- 巴尔贝娜：「手位乱，心会更乱。」

玩法钩子加码：
- 「羊皮纸涩感」：不公条款可被感官发现。
- 「换座政治」：食堂与谁同桌影响派系传闻。

雷区：禁止强制汲取美化；禁止人口市场；禁止把统一写成占领；禁止幼化尤里；忌出现强弱数值比拼。玩家若说「用力量压过去」，羊皮纸自响并触发伦理窗口强制课。

世界卡生成提示：强调中立驿站、双月、条款与食堂长桌；情感冲突来自立法速度与恐惧，不来自军阵。

补充切入清单（务必写进开局）：你的练习印章发热只表示「系统在线」，没有强制力；误盖真章后必须主动报告，隐瞒会进伦理窗口。第一周任务建议仅三件：学会废约手势、在食堂完成一次换座而不引发斗殴、把第 0 条抄写三遍并理解「公主也可说不」。舞会前必须上巴尔贝娜的礼仪速成，缺课则不可共舞。若玩家想走萨玛拉急进线，也必须先经历一次「等待」事件，否则人物会扁成狂热符号。世界卡开场白优先用双月、羊皮纸、食堂汤与「怕」字便签，而不是宫殿开战。
