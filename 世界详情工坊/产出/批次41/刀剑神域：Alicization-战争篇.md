# 刀剑神域：Alicization-战争篇

## 剧情

**【作品来源】**  
《刀剑神域》第4部「Alicization / 爱丽丝化」全篇，含人界篇与 Underworld 大战篇（War of Underworld），作者川原砾、插画 abec；电击文库第9–18卷为正传主线，第19–20卷「Moon Cradle / 月摇篮」为大战后极限加速两百年初期外传。动画《刀剑神域 Alicization》与《Alicization War of Underworld》由 A-1 Pictures 制作，导演小野学，对应轻小说第9–18卷（约第3期全47话）。时间锚在序列之争之后、Unital Ring 之前。文风：中世纪剑戟史诗 + 硬科幻灵魂工程 + 大规模战争与伦理；核心主题是「人工摇光是否算人」「禁忌与自由」「心念能否改写规则」。**本档案覆盖 Alicization 人界线 + 战争篇 + 月摇篮/星王收束**；艾恩葛朗特/ALO/GGO/序列之争仅作前史接口，不重写。卡片锚点：**星王桐人——Underworld 极限加速两百年尽头，桐人成星王，手撕太空怪兽，将文明推至星际时代，拥有改写世界物理常数的能力。**

**【世界定位】**  
双层舞台：现实侧是防卫关联组织 **RATH** 的海上设施 **Ocean Turtle（海龟）** 与 Project Alicization；虚拟侧是由 The Seed 构建、以 **Soul Translator（STL）** 运行的拟真文明 **Underworld（底界）**。主角 **桐谷和人（桐人）** 在摇光受损后被送入底界，与 **尤吉欧** 共攀中央教堂、打倒最高祭司 **奎涅拉（亚多米尼史特蕾达）**；大战中 **爱丽丝·滋贝鲁库（整合骑士爱丽丝·辛赛西斯·萨提）** 护其北归，**结城明日奈（亚丝娜）** 以创世神史提西娅超级账号入界，与 **加百列·米勒（闇神贝库达 / Subtilizer）**、**瓦沙克·卡萨鲁斯（PoH）** 及外来玩家军团开战。终局后桐人与亚丝娜在极限加速中度过约两百年，桐人成为 **星王**——卡片锚点即此顶点。

**【世界观 · 力量体系】**  
**1. 现实层规则**  
- **STL / STLT**：灵魂翻译机，直接读写「摇光（Fluctlight）」——人脑量子层的意识载体；可复制新生婴儿摇光、加速主观时间、修复或损伤神经网络。  
- **Light Cube（光量子立方体）**：存放人工摇光的硬件单元；Alice 的完成型光立方是战争争夺物。  
- **加速倍率**：底界时间可相对现实加速（日常约千倍级；终局 **极限加速** 可达约五百万倍量级，导致外部卫星链路跟不上而强制登出）。  
- **死亡**：底界「天命（Life）」归零＝该摇光单元崩溃；现实中切断主电源/电流冲击可灼伤正在治疗中的摇光（桐人战后失神的直接原因）。  
- **超级账号**：RATH 预留的高权限角色——创世神史提西娅（亚丝娜）、太阳神索鲁斯（诗乃）、地母神提拉利亚（莉法）等，权限远超土著对象控制权威。

**2. Underworld 内规则**  
- **禁忌目录（Taboo Index）**：公理教会颁布的绝对律法；违反触发右眼封印（Code 871）剧痛，强行突破可爆眼。  
- **神圣术（Sacred Arts）**：以空间资源（System Call）调用的「魔法」；高阶可治疗、冻结、炮击、转换资源。  
- **剑技 / 记忆解放（Perfect Weapon Control / Release Recollection）**：神器武器的完全记忆解放可改写战场（蓝蔷薇之剑冻结吸命、金木犀之剑花瓣斩、夜空之剑与夜空之星的共鸣等）。  
- **心念（Incarnation）**：意志具现；可补全断肢、再生右眼、改写外貌与伤害判定——是本世界真正的「规则级皮毛」。  
- **对象控制权威 / 系统控制权威**：决定能操作何种对象与系统指令；最高祭司近乎无限。  
- **合成秘仪（Synthesis Ritual）**：剥离记忆、植入虔信模块，将人改造成整合骑士。

**3. 战力阶梯（按破坏力 / 压迫感，宁低勿高）**  
1. **卢利特村民 / 普通侍卫 / 低阶学徒**：冷兵器格斗、砍魔树日常。≈一阶。  
2. **修剑学院一般生 / 哥布林小队 / 地方剑士**：可小队战，有成型剑技。≈一~二阶。  
3. **学院精英生（索蒂莉娜级）/ 诺尔朗贵族剑士**：单挑小队、决斗有名。≈二阶。  
4. **见习整合骑士 / 暗黑界中坚队长 / 学院首席边缘**：神器雏形、可守城门一翼。≈二~三阶。  
5. **正式整合骑士中坚（艾尔多利耶、迪索尔伯特等）/ 暗黑界各族首领**：一人当一面战线。≈三阶。  
6. **上位整合骑士（爱丽丝、法娜提欧、谢达等）/ 暗黑界顶尖（拳王伊斯坎等）**：城门级歼灭、心念初显。≈三阶顶~（四~六阶由「战争篇」分条覆盖，本条不单列切入）。  
7. **最高祭司奎涅拉 / 闇神贝库达·加百列 / 创世神史提西娅等超级账号全开 / 双剑心念全盛桐人**：战略级、可左右整场战争与摇光存亡。  
8. **星王桐人（极限加速两百年后）**：改写世界物理常数、手撕太空级威胁、将文明推至星际——**本条七阶顶点与卡片锚点**。

**乐园阶位映射**  
卢利特日常~普通学徒≈一阶；修剑学院精英 / 地方强者≈二阶；整合骑士中坚 / 暗黑界将领≈三阶；**星王桐人 / 奎涅拉系统权 / 加百列灵魂猎食 / 超级账号神级终局**≈七阶。**本世界切入覆盖仅一、二、三、七**（中高阶战争细目见「刀剑神域：战争篇」分条）。顶点＝**超阶边缘：存在·情报优先 / 条件性胜利**；禁止把星王/奎涅拉写成「被封印故战力为零」。

**死亡与代价**  
- 底界天命尽＝摇光崩溃（真死）。  
- 合成＝人格被改写，记忆可部分唤醒但不可当无代价复活。  
- 现实电源冲击可永久损伤摇光（桐人失神半年）。  
- 极限加速中的两百年记忆，回归现实后可被要求删除（星王线的伦理炸弹）。

**【地理 · 舞台】**  
**Underworld 人界**  
- **卢利特村**：北境边村；魔树（基加斯雪松）每日砍伐任务；爱丽丝故乡；大战前爱丽丝与失神桐人隐居木屋。  
- **北之尽头山脉 / 北之洞窟**：人界与暗黑界边界；幼年爱丽丝触暗被押走；战后爱丽丝冰封北窟阻哥布林。  
- **萨卡利亚**：北方城镇；剑术大会舞台，桐人尤吉欧由此入学院资格。  
- **北圣托利亚修剑学院**：四帝国之一北方帝国都城旁；精英剑士养成；禁忌突破事件（莱欧斯/亨伯特）发生地。  
- **中央大教堂（Central Cathedral）**：公理教会巨塔，百层；图书馆贤者（卡迪纳尔）、元老院、最高祭司居顶；人界篇决战场。  
- **东之大门**：人界与暗黑界分界巨门；最终负荷实验启动后门崩，大战主战场。  
- **世界尽头的祭坛**：暗黑界远端系统接口；爱丽丝由此被送出现实、桐人与加百列决战周边。

**Underworld 暗黑界**  
- 哥布林、兽人、巨人、半兽人、暗黑骑士、暗黑术师、拳斗士等多族混战社会；弱肉强食。  
- 闇帝宝座：加百列以超级账号 **Vector / 贝库达** 夺权。

**现实侧**  
- **Ocean Turtle**：伊豆海域核动海上设施；主轴/上轴/下轴；STL 室、光立方集群、主副控制室。  
- **RATH 六本木分部**：STL 一号机、六号机；诗乃/莉法入界入口。  
- 日本总务省假想课、Glowgen Defense Systems（美军关联 PMC）外部压力环。

写场景优先：魔树木屑与禁忌目录石板、教堂螺旋阶与云上庭园、东门崩塌烟尘、暗黑界血红天穹、海龟金属走廊警报、极限加速下星空与「星王」冠冕感。

**【世界剧情线】**  
① **前史接口（2026.6 前）**  
序列之争后，菊冈诚二郎以半胁迫方式让和人测试 STL。Project Alicization 目标：以复制摇光在底界加速培养「可自主违反规则」的完成型 AI（A.L.I.C.E.），供无人兵器伦理决策。底界已运行数百年主观时间；公理教会用禁忌目录消灭了「犯罪」，却也消灭了自由意志——RATH 需要外部刺激。微笑棺木余党 **金本敦（强尼·布莱克）** 在现实街头以琥珀胆碱袭击和人，致其濒死；菊冈将他送入海龟 STL，以底界治疗摇光，并保留其外部记忆以便「催化」人工摇光。

② **童年底片与魔树（人界开篇 · 一阶叙事）**  
桐人在底界以儿童态与 **尤吉欧**、**爱丽丝·滋贝鲁库** 成长于卢利特。三人误入 **北之尽头山脉** 洞窟探险，爱丽丝跌倒手触暗黑界土地，违反禁忌目录；整合骑士（后知为 **迪索尔伯特** 一系）押其往中央教堂，尤吉欧因恐惧僵立，桐人试图阻拦失败——此羞耻成为尤吉欧终生动机。  
主观数年后再登录：桐人失忆重逢尤吉欧。尤吉欧的「天职」是每日砍 **基加斯雪松（魔树）** 至成年仪式，持从山中取回的 **蓝蔷薇之剑** 却因技艺权威不足无法完全解放。桐人发现自己能发动 SAO 时代剑技，助砍、教步法。**赛鲁卡**（爱丽丝之妹）听闻姐姐往事只身后失踪入山，被哥布林捕获；桐人尤吉欧追击，尤吉欧重伤，赛鲁卡以神圣术转送桐人天命救其命——转送中桐人幻见中央教堂顶「等待你们的女人」（卡迪纳尔伏笔）。战后记忆回流，桐人技能点够用蓝蔷薇重创魔树；尤吉欧获准改职剑士，二人南下寻爱丽丝。

③ **修剑学院与禁忌突破（二阶叙事）**  
途经 **萨卡利亚** 剑术大会取胜，入 **北圣托利亚修剑学院**。桐人师从精英剑士 **索蒂莉娜·瑟路特**，以魔树枝打造 **黑剑**；休息日训练沾污首席 **沃洛·莱文坦** 制服，被邀实剑「惩罚」切磋——莱文坦家以敌血饲剑，桐人以心念抬升剑威逼平。次日索蒂莉娜击败沃洛毕业，桐人赠风花草作饯别。二人升精英，收侍从 **罗妮耶·阿拉贝尔**、**缇洁·修托利南**。  
贵族生 **莱欧斯·安提诺斯**、**亨伯特·齐杰克** 虐待侍从 **弗伦妮卡**，并设计诱使罗妮耶与缇洁上门「质问」后将其捆绑；亨伯特宣称「未违反目录条文」。尤吉欧为救人强破禁忌，右眼封印崩裂、眼球损毁，斩断亨伯特手臂；桐人赶到斩落莱欧斯双臂，莱欧斯系统崩溃而死。教会派整合骑士逮捕——来者正是无记忆的 **爱丽丝·辛赛西斯·萨提**。主题钉死：**法律若不保护弱者，破法者才是人。**

④ **中央教堂攻略（三阶叙事主轴）**  
爱丽丝将二人入狱并命徒弟 **艾尔多利耶** 埋伏。二人以高对象控制权威越狱，与艾尔多利耶战至其虔信模块松动；另一骑士截击时，传送门将二人送入 **大图书馆**。**卡迪纳尔**（少女外表的 Cardinal 分枝）揭示：最高祭司 **奎涅拉** 曾为延寿与孩童链接记忆，反使孩童获得对等权限——孩童即卡迪纳尔前身，败逃至教会无法管辖的图书馆，两百年谋划推翻女祭司；整合骑士皆经 **合成秘仪** 抽忆植入虔信模块；唤醒需拔模块并唤重要回忆。卡迪纳尔授两柄特制短剑（本拟留给爱丽丝与奎涅拉）并结盟。  
塔内战序：取回武器后败 **迪索尔伯特**（尤吉欧认出其即押爱丽丝者，骑士战前记忆被抹）；**莉涅尔与菲洁尔** 以毒匕首伪作「战利品」抬至五十层，被桐人反制；副团长 **法娜提欧** 与其下骑士团激战，法娜提欧重创，桐人违命使用短剑将其送入卡迪纳尔治疗。八十层云上庭园：爱丽丝拦截，战中外墙破洞，桐人与爱丽丝坠出塔外；尤吉欧绝望独上。墙外二人暂盟攀塔，歼灭伪装雕像的暗黑界潜伏者；桐人陈述公理教会真相，爱丽丝强破右眼封印、眼球炸裂后昏迷——她成为 RATH 定义的完成型 **A.L.I.C.E.**。  
九十层大浴场：尤吉欧以蓝蔷薇完全记忆解放与 **贝尔库利·辛赛西斯·一** 同冻相持并赌命吸命；丘德金到场石化贝尔库利、俘尤吉欧。百层：奎涅拉诱导尤吉欧放松摇光核心防护，合成为 **尤吉欧·辛赛西斯·三十二**。桐人爱丽丝寻至，被迫与合成态尤吉欧互斩；因合成仓促、记忆未抽尽，尤吉欧自行挣脱，佯装虔信再近身刺奎涅拉——金属无法破其皮肤。丘德金召唤火灵巨像「灯灵」；桐人以心念短暂再现艾恩葛朗特装束斩丘德金。奎涅拉放出 **剑之巨人**（素材为人界住民转化的剑，核心为虔信模块），扬言再造半人界为兵器抗暗黑界。观察蜘蛛 **夏洛特** 死战拖延；尤吉欧按卡迪纳尔指示插短剑开传送，卡迪纳尔现身雷击巨人并治疗二人，随即以己命与奎涅拉交易换三人自由，遭雷殛将死前应尤吉欧之愿将其与蓝蔷薇融合。剑形态尤吉欧碎巨人虔信、斩奎涅拉一臂，自身被折。桐人以血与心念重铸水晶质蓝蔷薇，二刀流断奎涅拉余臂并重创；奎涅拉欲经控制台逃离底界，丘德金死缠不放，余火焚尽二人。尤吉欧将爱丽丝童年记忆碎片托付后消散。桐人接通主控台——海龟遭袭；菊冈令带爱丽丝往 **世界尽头的祭坛**；袭击者切断电源，电流冲击治疗中摇光，桐人失神。

⑤ **北地隐居与卢利特防卫（战争篇序）**  
爱丽丝携失神桐人乘飞龙 **雨缘** 归卢利特，因「罪人」不得入村，在村外筑木屋日夜照料，**赛鲁卡** 常来探视；地主 **巴尔波萨** 等以低价役使她伐木。教会内部有人主张处决失神者；**艾尔多利耶** 亲至请爱丽丝归队并「处理」桐人，被拒。夜半桐人短暂惊醒感知危机——卢利特遭哥布林与兽人袭击。爱丽丝着甲却先以常服入村，村民因目录必须服从侍卫长而拒撤；她公开整合骑士身份强制疏散，独力击退入侵，心念再生右眼。事后她冰封北窟通道，携桐人赴东门军营。与贝尔库利会合时说明奎涅拉罪行，但隐瞒「外部世界」与祭坛全貌；罗妮耶、缇洁闻尤吉欧死讯恸哭，受托战时守护桐人。

⑥ **最终负荷与闇帝登场（现实×暗黑界）**  
**Glowgen Defense Systems** 潜艇部队夺取海龟主控制室，却无法强破光立方集群与副控。**加百列·米勒**（GGO 的 Subtilizer，曾在 BoB 以空手扼杀诗乃角色并低语「你的灵魂会很甜」）受美国政府任务夺取 STL 与光立方；其副手 **瓦沙克·卡萨鲁斯（PoH）** 为 SAO 微笑棺木首领。二人登录超级账号 **闇神贝库达（Vector）** 与暗黑骑士，迅速接管暗黑界军权。加百列童年为「取出灵魂」用长针刺死玩伴 **艾丽西亚**；登基夜掐死假扮侍妾行刺的 **莉皮娅·赞卡尔**（将军 **维克萨·乌尔·沙斯塔** 之恋人），以冰封头颅示众。沙斯塔释放毁灭级心念欲杀帝，反被加百列「灵魂之穴」吞灭。最终负荷实验启动，东门将崩——表面是人界防御测试，实为加百列猎杀 Alice 的倒计时。现实侧亚丝娜在凛子协助下潜入海龟，确认桐人状况后决定登录救两人。

⑦ **东门大战与诱敌（战争主体 · 三阶可介入面）**  
东门崩塌，暗黑界大军涌入。人界前列：**迪索尔伯特、法娜提欧、艾尔多利耶**；后列 **贝尔库利（时穿剑待机）、谢达、伦利**（伦利一度怯战逃亡，后在补给线遭山岳哥布林突入时被 **莉涅尔、菲洁尔** 救场并重拾勇气，发动双翼刃完全武器控制）。爱丽丝藏身高空，以死者逸散的空间资源构筑神圣术巨炮。迪·艾伊·艾尔放出全部飞龙从空突袭，被贝尔库利时穿剑一次清空；暗黑术因空间资源不足初败。爱丽丝开炮重创敌军近半。迪·艾伊·艾尔经贝库达批准献祭兽人再生资源，放出食人虫群；**艾尔多利耶** 以身作盾全灭，爱丽丝为徒复仇屠灭术师部队，并在敌前自称「光之巫女」——确认贝库达目标即自己。人界遂组诱饵部队（爱丽丝、贝尔库利、伦利、谢达、失神桐人、罗妮耶、缇洁、志愿兵与神官、飞龙）主动进入暗黑界，把敌军从走廊引开。

⑧ **超级账号与外来玩家（战局质变）**  
诱饵部队在南林设伏时，后背遭 PoH 暗黑骑士突袭；**亚丝娜** 以 **创世神史提西娅** 超级账号降落，权限级改写地形与资源，救下危局。爱丽丝因亚丝娜靠近桐人而误判为敌，交手被贝尔库利喝止。军议上爱丽丝说明右眼封印与自己即猎物之事；会后与亚丝娜交换桐人情报，敌意转同盟。  
结衣侦知美军以「活动奖励」欺骗大量海外 VRMMO 玩家登录底界；**莉法（地母神提拉利亚）**、**诗乃（太阳神索鲁斯）** 自 RATH 六本木 STL 入界——莉法因六号机问题降落位置偏移至暗黑界附近，诗乃落地即歼灭约两万刚登录的美方玩家并与亚丝娜会合。约三万外来者总涌入后，爱丽丝独战中被加百列趁乱劫走并强制昏睡。贝尔库利追至，以时穿剑与自身天命重创贝库达，迫其现实侧掉线身死边缘；诗乃赶到告知「现实中人还活着」，自愿殿后。加百列改登 GGO 账号 Subtilizer 再入，诗乃狙击与近战重创之仍败，却争取了时间。  
**PoH** 纠集被骗的美/中等玩家与暗黑残部组成「红色军团」追杀亚丝娜一行；失神桐人在罗妮耶、缇洁哭喊与尤吉欧残存记忆碎片刺激下摇光复苏，以系统级权限强制登出大量外来玩家并击败 PoH。亚丝娜绝境中使出本不存在于底界的 **圣母圣咏** 十一连击（有纪声援的奇迹描写以动画/文库为准）重创 PoH。人界与部分暗黑界势力（如拳斗士 **伊斯坎** 一线）临时停火，共抗只知杀戮的外来者。克莱因、莉兹、西莉卡、艾基尔等日本玩家亦经结衣号召入界支援。

⑨ **祭坛决战与极限加速**  
爱丽丝骑龙赶往 **世界尽头的祭坛**，雨缘与友龙被加百列心念重创；复苏的桐人以心念将龙压至卵态交爱丽丝护种。桐人持 **夜空之剑**（与尤吉欧意志共鸣的伙伴之剑协同）对加百列展开心念顶峰战；亚丝娜护送爱丽丝至第三系统控制台，途中与爱丽丝、莉法、诗乃等向夜空之剑输送心念。加百列败，现实侧亦死；瓦沙克双死。爱丽丝光立方由 **神代凛子** 接入 EMOM 机体接出三维世界。RATH 启动 **极限加速** 以防再夺；卫星链路无法跟随，莉法、诗乃等外部支援被强制登出。桐人与亚丝娜未及登出，意识留在加速中的底界，迎来主观约两百年。

⑩ **月摇篮 · 星王（卡片锚点 · 七阶）**  
人界历约 380 年代起（Moon Cradle）：大战后边界开放，人界与暗黑界居民开始贸易与旅行；桐人预判和平脆弱，与亚丝娜引入现实科学、法律与行政，抑制再战。人界历 382 年 2 月，闇之国旅客卷入「人界本不应发生的杀人案」，桐人与见习整合骑士 **罗妮耶** 共同搜查——月摇篮的政治推理面。漫长岁月中二人推动文明跃迁：从中世纪剑与神圣术，到工业、航天与星际殖民。桐人被推戴为 **星王**——卡片锚点完全展开：**在 Underworld 加速两百年尽头，他手撕威胁殖民圈的太空级怪兽，将文明推进到星际时代，并拥有改写世界物理常数的能力。**  
回归现实窗口打开时，和人要求删除两百年记忆以免人格被「另一个自己」压垮；**比嘉健** 等暗中保留星王态摇光备份——为后续再访底界与 Unital Ring 留接口。现实侧约两周后苏醒；爱丽丝已适应三维生活与科技，暂居桐谷家；约一月半后 The Seed 联合体融合成 **Unital Ring** 强制转移——**本条主线止于星王收束与回归**，UR 正文不展开。

⑪ **蝴蝶效应边界**

- **蓝蔷薇之剑**：北窟龙与英雄传说相关神器；冻结全域、吸命、完全记忆解放；尤吉欧本命；融剑斩奎涅拉后与其意志同在。  
- **夜空之剑（Night-Sky Blade）**：序列之争通关起桐人本命神器；大战记忆解放与心念共鸣核心。  
- **金木犀之剑**：爱丽丝神器；花瓣斩、骑士身份与「金木犀之骑士」称号来源。  
- **时穿剑**：贝尔库利神器；可斩未来轨迹级攻击，东门清空飞龙、祭坛换命。  
- **黑剑（魔树枝材）**：桐人学院期武器；心念可抬升其权威。  
- **虔信模块**：额前紫结晶；合成关键；拔除=记忆与人格争夺战。  
- **Light Cube / Alice 完成体**：战争目标本体；现实侧驱动 EMOM，完成型人工摇光的硬件心。  
- **STL 与超级账号权限**：史提西娅（创世/地形）、索鲁斯（太阳）、提拉利亚（地母/自愈）、贝库达（闇帝）——权限即战略武器，非普通掉落。  
- **禁忌目录石板/条文**：律法本身；低阶世界的「空气墙」与戏剧冲突源。  
- **卡迪纳尔短剑**：两柄特制；一柄救法娜提欧，一柄开门召卡迪纳尔。  
- **星王权能（物理常数改写）**：两百年终点的存在状态；手撕太空怪兽的力量来源；**不可被低阶契约者复制领取**。

**【隐藏剧情 · 伏笔】**  
- 公理教会「无犯罪」= 无自由意志；**主动违反禁忌**才是 A.L.I.C.E. 的操作性定义——RATH 要的不是听话 AI，是会说「不」的 AI。  
- 卡迪纳尔与奎涅拉同源：同一权限树的自我内战打了两百年。  
- 剑之巨人计划：把半个人界住民转成兵器——比暗黑界更黑暗的「人界神权」。  
- 加百列「观看灵魂升天」不是诗，是童年谋杀的美学化；莉皮娅之死复读艾丽西亚。  
- PoH 把 SAO 红名公会的种族仇恨与折磨美学直接灌进底界战争。  
- 桐人失神=电源冲击+治疗中摇光过载，不是剧情惩罚性「变弱」。  
- 星王两百年记忆删除 vs 比嘉备份：谁有权决定「哪一个桐人」作数——UR/后日谈伦理核。  
- 与《加速世界》Soul Translator→神经连接器技术史互文，点到为止勿倒灌。  
- 序列之争→重村入 RATH→Alicization：国家在「记忆堆女儿」失败后改走「底片加速灵魂」。

**【叙事基调 · 雷区】**  
基调：中世纪沉重剑戟 + 海龟科幻冷光 + 战争群像；热血里必须留下「尤吉欧真的死了」「人工摇光会哭会爱」「法律若只保护强者就该被撕」。心念戏写意志与代价，不写无限制膨胀。  
忌：开局送星王权能或超级账号；把尤吉欧随便复活当儿戏；把爱丽丝写成无自我的战斗人偶；把加百列写成普通抢 BOSS 龙套；低阶硬刚奎涅拉/贝库达/星王；把本条写成纯人界学院日常而砍掉战争与星王锚点；用「被封印所以为零」解释顶点；把 Underworld 居民当可随意格式化的 NPC 笑料。  
最早可切入锚点：卢利特砍魔树时期；修剑学院侍从事件前后；东门崩塌初周志愿兵；七阶仅限两百年后星王时代或祭坛终局观测位。

## 阶位切入点

> 阶位↔本世界：一阶≈卢利特村民/普通学徒/砍魔树日常；二阶≈修剑学院精英/地方剑士名手；三阶≈整合骑士中坚/暗黑界将领/教堂中层战；七阶≈星王桐人/奎涅拉系统权/加百列灵魂猎食/超级账号终局。世界顶点（星王·改写物理常数）为跨阶存在，低阶切入一律「规避顶点·情报优先」。

**一阶（卢利特 · 魔树与禁忌的井底）**  
切入身份/时点：契约者以「卢利特村新来的木工/牧助/外村剑术学徒」切入，锚定桐人与尤吉欧仍每日砍 **基加斯雪松**、爱丽丝已被押走数年的时期。  
初始事件：你在魔树下帮忙清木屑，听见北方来的孩子争论「禁忌目录能不能破」；当晚村口有人失踪，有人指认方向是 **北之尽头山脉**——赛鲁卡失踪的同一条路。  
开场白建议：「斧刃砍进漆黑树干的声音沉闷得像在砍骨头。村里人人背得出禁忌目录，却没人敢问：被教会带走的金发女孩，现在还算不算『人』。」  
关键NPC立场：**尤吉欧**——温和、要你先活过今天的配额；**桐人**——失忆的外乡剑士，警惕却护短；**赛鲁卡·滋贝鲁库**——想追姐姐的真相；**村长滋贝鲁库**——怕教会多于怕哥布林。  
主线钩子/支线：参与或旁观北窟救赛鲁卡；打听蓝蔷薇之剑来历；支线＝魔树配额作假被揭、哥布林斥候夜袭粮仓。  
危险度/规避：中低——哥布林可死人；规避单独深挖北窟暗黑界边界、对教会骑士口出「叛教」。  
任务方向/奖励：生存、基础剑技/神圣术入门、村内信任；奖励＝普通铁剑、药草、北境地图残片。

**二阶（北圣托利亚 · 学院与贵族毒）**  
切入身份/时点：以「修剑学院下级生/城镇佣兵剑士」切入，锚定桐人尤吉欧已升精英、**罗妮耶与缇洁** 仍为侍从的学期。  
初始事件：训练场上精英生 **亨伯特** 当众折辱侍从；你被拉去「观摩决斗」，有人塞给你一张字条——今夜贵族宿舍有「合法的私刑」。  
开场白建议：「圣托利亚的剑光比卢利特亮十倍，规则也冷十倍。你的剑可以赢过同学，却赢不过一句『贵族的责务』——除非有人先把禁忌目录撕开。」  
关键NPC立场：**桐谷和人（桐人）**——会为侍从拔剑；**尤吉欧**——忍耐到极限；**罗妮耶·阿拉贝尔 / 缇洁·修托利南**——恐惧与愤怒；**莱欧斯·安提诺斯**——把规则当遮羞布；**索蒂莉娜·瑟路特**——可成为公正见证。  
主线钩子/支线：阻止或记录侍从受辱事件；学院排名战站队；支线＝沃洛·莱文坦的血剑传闻求证、风花草被毁的园艺案。  
危险度/规避：中——实剑决斗可残；规避在无证人时与莱欧斯单挑至死、向教会诬告「破禁忌」反噬自身。  
任务方向/奖励：学院通行证、高级剑术指导、贵族罪证抄本；不发神器。

**三阶（中央教堂 / 东门 · 骑士与战争中坚）**  
切入身份/时点：以「刚破禁忌的流亡剑士 / 人界军志愿兵 / 暗黑界逃兵翻译」切入，锚定教堂攻略中段（法娜提欧之战前后）或东门崩塌初周。  
初始事件（教堂向）：你在螺旋阶听见毒匕首落地——**莉涅尔与菲洁尔** 抬着两具「尸体」要向五十层报功，其中一具黑衣剑士睁眼看了你一眼。  
初始事件（东门向）：东门碎石如雨，**艾尔多利耶** 在虫群前大喊让后列撤退；有人把失神的黑衣人塞进你的运输龙车，要你「活着送到爱丽丝看得见的地方」。  
开场白建议：「教堂的每一层都用剑写成戒律，东门的每一秒都用命写成资源。你终于站在能看见整合骑士旗帜的高度——也第一次看清旗帜背后有人在被改造成『模块』。」  
关键NPC立场：**爱丽丝·辛赛西斯·萨提**——对谎言零容忍，对失神桐人极护；**贝尔库利·辛赛西斯·一**——豪爽可盟，不容辱骑士团；**法娜提欧**——严酷副团长，认强者；**卡迪纳尔**——给条件不给施舍；**迪·艾伊·艾尔**——视人界为祭品。  
主线钩子/支线：协助送伤员/卡迪纳尔短剑；东门积蓄炮击时的掩护；支线＝护送罗妮耶缇洁撤离补给线、与谢达并肩斩暗黑斥候、劝降一名不想为人祭的兽人百夫长。  
危险度/规避：高——骑士与术师团可屠队；规避顶层奎涅拉、贝库达本尊、剑之巨人核心；不挑战「光之巫女」的指挥权。  
任务方向/奖励：神器碎片级素材、人界军士官推荐、空间资源采集权、整合骑士「暂不处决」令。

**七阶（星王 · 顶点条件战 / 终局观测）**  
切入身份/时点：以「极限加速两百年后的星际官吏 / 祭坛决战协力观测者 / RATH 内部灰色记录官」切入，锚定 **星王桐人** 已确立、或加百列祭坛战尾声。  
初始事件：轨道站警报响起——一头足以撕开殖民穹顶的太空怪兽正在逼近；通讯里星王的声音平静：「物理常数，借我改一改。」与此同时，现实侧光立方室有人试图拷贝 Alice 的完成型备份。  
开场白建议：「星空不再是诗，是国境。你站在改写常数后的静默里，忽然懂了两百年前东门下那些尸体的重量——文明每一次往上爬，都先学会如何不把『摇光』当成零件。」  
关键NPC立场：**星王桐人**——疲惫的绝对战力，厌战却不让步；**结城明日奈**——共治者，盯着伦理底线；**爱丽丝·滋贝鲁库**——现实与底界的桥梁；**加百列·米勒**（若锚在终战）——只对灵魂感兴趣；**比嘉健 / 菊冈诚二郎**——备份与国家议程。  
主线钩子/支线：确认太空威胁被排除后的常数是否回滚；阻止光立方非法拷贝；支线＝月摇篮期命案卷宗解密、星王删忆前的「遗嘱」传递、暗黑界代表在星际议会的投票权谈判。  
危险度/规避：贴近顶点——**超阶巅峰：存在·情报优先 / 条件性胜利**；严禁把星王/奎涅拉/加百列写成可随意击杀的沙袋；低姿态换情报与一次性协力评价。  
任务方向/奖励：星际航图残页、摇光伦理白皮书、与星王政权的一次性通行印；**不发**可复制的物理常数改写权。
