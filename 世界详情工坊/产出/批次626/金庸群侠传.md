# 金庸群侠传
<!--meta lib=主库 tiers=一、二、三、四-->

## 剧情

**【作品来源】**
《金庸群侠传》（Heroes of Jin Yong）是河洛工作室（Heluo Studio）于1996年开发、智冠科技发行的DOS平台经典中文角色扮演游戏。玩家扮演误入金庸武侠世界的现代人（默认存档名常见「小虾米」「金迷」「徐小侠」），在开放大地图上收集金庸十四部小说（江湖中被当作「十四天书」）、招募群侠、修炼武功，最终成为武林盟主并通过圣堂时空裂缝返回现实。文风：高自由度、碎片化金庸名场面拼贴、回合制战斗与道德值双线；幽默彩蛋（南贤北丑、野球拳、孔八拉）与原著武侠气质并存。续作与衍生包括《武林群侠传》《三国群侠传》、网游《金庸群侠传Online》、半瓶神仙醋《金庸群侠传2/3》、汉家松鼠《金庸群侠传X》及河洛原班人马的《侠客风云传》系列等；**本档案以1996年DOS原版逻辑与通行攻略脉络为准**，重制/MOD仅作对照，不替代原版因果。

**【世界定位】**
「飞雪连天射白鹿，笑书神侠倚碧鸳」——金庸十四部小说人物与场景被并置在同一张开放江湖地图上的武侠RPG世界。主角**小虾米**（玩家可改名）是二十世纪末愤世嫉俗的电脑游戏迷，一觉醒来坠入此界；为回家必须集齐十四天书、经华山论剑成盟主，再入霹雳堂（本名圣堂）地下室战胜十大高手。一句话：这是「现代穿越者在金庸群像江湖里练野球拳、收群侠、抢天书、打东方不败」的开放式武侠任务世界。

**【世界观 · 力量体系】**
力量本源：内力（阴/阳/调和）、拳掌剑刀特技、轻功、用毒、医疗、暗器与声望；无灵气飞升，无系统面板「一键满级」。成长靠：①装备秘籍在战斗中升级（武功多可练至约十级）；②资质影响学习与升级速度（资质偏低反而更易悟**左右互搏**）；③名师点拨、天书任务、生死战与吃药炼体；④道德值影响可学武学、可收队友与终局对手阵容。死亡规则：战斗败北可重来/重载（游戏机制），叙事上江湖一刀可致命，顶尖高手亦会因围攻、剧毒、阴谋与心魔落败——**无复活外挂**。特殊系统：①**道德值**——助人、放生、高品德选择上升；偷窃、滥杀、邪派路线下降；≥50左右终战遇十大恶人，≤50左右遇十大善人（后者更难）；②**左右互搏**——低资质路线核心，攻击变两次，配合满级野球拳近乎破局；③**声望**——影响部分对话与事件；④**队伍**——最多约五名同伴，正邪混搭会拉扯道德；⑤**韦小宝式流动商人**——买卖奇物与霹雳刀法等；⑥**罗盘坐标探索**——大地图自由行走，场景内对话/开箱/踢馆。

战力表现梯（按原作/游戏可观察破坏力，宁低勿高；对应人间武学而非毁城超凡）：
1. **江湖新人 / 三流**：市井混混、镖局走卒、门派记名弟子、刚穿越的小虾米初期。战力：冷兵器格斗、踹破木门、单挑平民；一刀一枪可致命。
2. **二流 / 门派骨干**：普通少林武僧、武当弟子、五岳剑派中坚、丐帮中低袋、药王庄护卫级。战力：空手碎砖裂石边缘、单挑一小队、内力雏形；可清山寨外围。
3. **一流高手**：胡斐、令狐冲（成长中后期）、段誉（六脉未稳前）、苗人凤、狄云（神照后）、程灵素（医毒向）、蓝凤凰、田伯光等一方成名者。战力：裂石断金、乱军取首级边缘、可端据点/屠山寨级个人战。
4. **绝顶 / 宗师**：东方不败、张三丰、乔峰、郭靖、黄药师、洪七公、周伯通、任我行、小虾米终局（满级野球拳+左右互搏+九阳/九阴等叠满）、华山论剑与圣堂十大高手序列。战力：左右一域武林格局、以一敌众、可改写「谁为盟主」；**仍属人间武学顶点**，不能夷平城池、不能爆星。

乐园阶位映射（对照《阶位战力图鉴》，宁低勿高）：江湖新人~三流≈**一阶**；二流~门派骨干≈**二阶**；一流高手≈**三阶**；绝顶宗师/小虾米终局/东方不败/五绝级≈**四阶**。世界顶点＝圣堂终战十大高手与满状态小虾米所代表的宗师天花板；**无超阶**。低阶切入严禁硬刚东方不败、张三丰、乔峰巅峰与圣堂终局群战，情报与周旋优先。

**【地理 · 舞台】**
舞台是一张可罗盘导航的开放「金庸拼图地图」：中原门派、江南水乡、塞外沙漠、海岛秘地与古墓冰火并存，玩家不必按小说时间线推进。
- **主角居 / 南贤居 / 北丑棚**：开局锚点。南贤给罗盘与智慧果提示，北丑提示常颠倒却关键；回家客厅在集齐天书后出现华山论剑邀请帖。
- **药王庄 / 苗人凤居 / 阎基相关点**：飞狐线起点；七心海棠、程灵素、胡斐单挑苗人凤。
- **闯王宝藏 / 田伯光居 / 东海无名岛**：雪山飞狐与鸳鸯刀线；军刀开锁、鸳鸯双刀合璧。
- **江南唐诗山洞 / 天宁寺**：连城诀线；唐诗选辑、北丑泡水、佛像后天书。
- **燕子坞 / 绝情谷 / 铁掌峰 / 丐帮**：天龙线与神雕交叉点；玉玺、大燕谱、单挑乔峰。
- **黑龙潭 / 一灯居 / 百花谷 / 桃花岛**：射雕线；周伯通、瑛姑、郭靖黄蓉、道德门槛或硬刚。
- **高昌迷宫 / 沙漠废墟**：白马啸西风线；玄铁剑、隔世石、杨过相关。
- **神龙岛 / 五毒教**：鹿鼎记线；韦小宝商路、洪安通。
- **梅庄 / 黑木崖 / 五岳剑派据点 / 福威镖局 / 破庙蝴蝶谷**：笑傲线；四友字画、吸星大法、黑木令、东方不败。
- **回族部落 / 金轮寺**：书剑线；可兰经、金轮法王（兼练野球拳热门点）。
- **古墓 / 绝情谷后山**：神雕线；玉蜂浆、小龙女、石棺天书。
- **凌霄城 / 侠客岛 / 蜘蛛洞**：侠客行线；赏善罚恶令、石破天、太玄。
- **冰火岛 / 光明顶 / 明教分舵 / 灵蛇岛 / 蝴蝶谷**：倚天线；谢逊、六大派、成昆、光明圣火阵。
- **渤泥岛 / 金蛇秘洞 / 华山旁**：碧血剑线；金蛇剑、袁承志、高道德和平得书。
- **华山**：论剑成盟主、获武林神杖（圣杖）。
- **霹雳堂（圣堂）**：孔八拉坐镇；绿钥匙、厨房地洞、十四天书书架、十大高手与时空裂缝终局。
- **林厨子海边小屋**：彩蛋向补给与人情点。

地理逻辑：一~二阶冲突在主角居周边、客栈、小门派与支线山寨；三阶转入天书主线副本与名门踢馆；四阶锁定黑木崖、光明顶、华山论剑与圣堂终战。

**【世界剧情线】**

① **穿越开局：南贤、罗盘与「回家」契约（游戏序章）**  
二十世纪末，愤世嫉俗的年轻人在电脑前睡着，误入金庸武侠拼图世界。醒来后拜访**南贤**，得罗盘开启大地图导航，并被告知：江湖上散落着被误认作武功秘籍的**十四天书**（对应金庸十四部小说），集齐者可问鼎武林、接近回家之路。**北丑**提供颠三倒四却常指向关键坐标的提示；智慧果可换更明确线索。此阶段世界状态：小虾米武功近乎白板，野球拳尚是笑话，任何一流高手都能碾压他。主题：**你不是原著主角，你是外来变量。** 名场面：南贤居对话、第一次打开罗盘、客栈被嘲「三脚猫」。因果：①回家动机锁死主线；②提示NPC永久存在；③道德初始中立，第一次选择开始记账。

② **飞狐线：七心海棠、胡斐与苗人凤（《飞狐外传》《雪山飞狐》）**  
典型流程：阎基处得刀法残页→收**胡斐**→**苗人凤**居→药王庄摘蓝花入、先杀**阎基**取七心海棠交**程灵素**制解药→带胡斐单挑苗人凤胜出，得《飞狐外传》。胡斐加入可获闯王军刀与宝藏地图，入闯王宝藏开锁得《雪山飞狐》。此阶段名场面：药王庄毒攻防、胡斐苗人凤对刀、宝藏开锁。世界状态：中原北方一线活起来，玩家第一次体会「原著名场面可介入可改写」。因果：胡斐成早期强力队友；飞狐双书进包；道德因杀阎基等选择微调。

③ **连城诀线：狄云、唐诗与天宁寺（《连城诀》）**  
救**狄云**（常关联大轮寺等事件）→江南山洞取唐诗选辑→北丑处泡水→天宁寺战斗→佛像后取《连城诀》。此线相对短平，适合早期补书。名场面：唐诗谜题、佛像机关。因果：狄云可加入；神照经相关武学线索露头；玩家学会「提示NPC+场景道具」解谜范式。

④ **天龙线：燕子坞、玉玺与单挑乔峰（《天龙八部》）**  
燕子坞、绝情谷取传国玉玺、铁掌峰取大燕世族谱等条件齐备后，可至丐帮挑战**乔峰**（或用带头大哥书信等旁门，常降道德）换《天龙八部》。途中可遇**段誉**、**虚竹**、**阿紫**、**丁春秋**、**游坦之**、**王语嫣**、**慕容复**、**薛慕华**等。名场面：单挑乔峰的压迫感、燕子坞世家气场、星宿用毒。世界状态：北宋武林顶尖战力正式进入玩家视野。因果：天书入手；乔峰可成标尺或队友（视版本事件）；道德与正邪队友选择开始显著分叉。

⑤ **射雕线：黑龙潭、周伯通与桃花岛（《射雕英雄传》）**  
黑龙潭—**一灯**居—**百花谷**—再回黑龙潭—**桃花岛**；道德足够高（通行攻略约90档）可较和平从**郭靖**/**黄蓉**处得书，否则需击败郭靖。**周伯通**、**瑛姑**、**黄药师**、**洪七公**、**欧阳锋**、**欧阳克**、**丘处机**等在此线高频。名场面：老顽童玩闹、桃花岛奇门、东邪北丐气场。因果：《射雕英雄传》入手；降龙、九阴、空明等武学环境打开；高道德路线价值凸显。

⑥ **白马与神雕交叉：杨过、玄铁与古墓（《白马啸西风》《神雕侠侣》）**  
**杨过**加入常需绝情谷断肠草等事件；得玄铁剑后入沙漠高昌迷宫，以武力/内力开隔世石得《白马啸西风》。神雕线：百花谷送玉蜂浆予周伯通→绝情谷后山寻**小龙女**（宜带杨过）→古墓石棺得《神雕侠侣》。名场面：玄铁重剑、古墓寒玉、杨龙并肩。因果：杨过/小龙女可成核心队友；黯然、玄铁、玉女素心等气质进入队伍；玩家战力跨入一流门槛。

⑦ **鹿鼎与书剑：神龙岛、五毒与可兰经（《鹿鼎记》《书剑恩仇录》）**  
神龙岛与五毒教战斗链后得《鹿鼎记》；**韦小宝**多作流动商人而非传统队友，**洪安通**为敌或事件BOSS。书剑线：回族部落→金轮寺击败**金轮法王**取可兰经→归还得《书剑恩仇录》；金轮寺亦是练**野球拳**的热门练功点。名场面：神龙教仪式感、金轮五轮、还经得书。因果：两书入手；野球拳从笑话变核心培养；商路与塞外地图打开。

⑧ **笑傲线：梅庄四友、吸星与东方不败（《笑傲江湖》）——最高难天书之一**  
悦来客栈请**令狐冲**→备齐《溪山行旅图》（福威镖局）、《率意帖》（嵩山）、《呕血谱》（燕子坞玉玺相关）、《广陵散》（破庙/蝴蝶谷铁铲挖墓）→单挑**梅庄四友**→出庄再进得钥匙→地牢取吸星大法与黑木令→**黑木崖**挑战**东方不败**→出后再经五岳剑派事件得《笑傲江湖》。沿途**平一指**、**蓝凤凰**、**林平之**、**田伯光**、**余沧海**、**左冷禅**、**岳不群**、**风清扬**、**任我行**等悉数登场。名场面：四友书画局、地牢吸星、黑木崖绣花针与葵花。世界状态：邪派天花板现身；未练成九阳/左右互搏/满级主力武功时极易团灭。因果：笑傲天书到手；吸星/辟邪相关风险；令狐冲常成队伍灵魂。

⑨ **侠客行与倚天：侠客岛、光明顶（《侠客行》《倚天屠龙记》）**  
侠客：山洞/蜘蛛洞玄冰碧火酒→收**石破天**→凌霄城**白万剑**处得赏善罚恶令→带石破天赴侠客岛得《侠客行》（太玄气质）。倚天：冰火岛金毛**谢逊**→昆仑/**张无忌**→六大派与明教分舵**杨逍**→地道—光明顶大战得铁焰令→再经冰火岛、成昆、灵蛇岛紫衫龙王（蝴蝶谷**王难姑**提示）→破光明圣火阵得《倚天屠龙记》。名场面：侠客岛壁上太玄、光明顶围攻、屠龙倚天传说。因果：两书入手；九阳、乾坤大挪移、太极拳剑环境全开；张无忌/谢逊线改变明教格局观感。

⑩ **碧血与鸳鸯：金蛇、袁承志与双刀（《碧血剑》《鸳鸯刀》）**  
华山旁金蛇秘洞拔**金蛇剑**（攻击门槛约83档）→渤泥岛；道德够高和平得《碧血剑》并可收**袁承志**，武力硬夺大降道德且袁承志不加入。鸳鸯：田伯光居取鸳刀+闯王宝藏取鸯刀→东海无名岛开门得《鸳鸯刀》。名场面：金蛇剑出鞘、高道德劝说、双刀合门。因果：十四天书凑齐的最后拼图；道德终局分支被锁死在「已选人格」上。

⑪ **华山论剑：武林盟主与神杖（中后期总高潮）**  
十四天书齐备后回主角居，客厅出现华山论剑武林邀请帖。赴华山击败群雄，成为武林盟主，获**武林神杖（圣杖）**。此阶段是「身份质变」：江湖从「你在找书」变为「你是盟主」。名场面：华山群英、授杖、声望爆表。因果：圣杖=打开圣堂终局的钥匙中介。

⑫ **霹雳堂·圣堂终局：十四书架与十大高手（通关）**  
持杖见**孔八拉**换绿钥匙→厨房开里间、推方桌入地洞→将十四天书放入规定书架→深入触发武林十大高手。分支完全由道德值决定：道德≥50左右战**十大恶人**（**欧阳锋、金轮法王、裘千仞、慕容复、何太冲、唐文亮、左冷禅、岳不群、任我行、洪安通**等序列，通行名单以原版数据为准）；道德≤50左右战**十大善人**（**郭靖、黄蓉、丘处机、张三丰、乔峰、玄慈、苗人凤、洪七公、黄药师、周伯通**等，周伯通/郭靖左右互搏极难）。击败后四大天王对话，入时空裂缝返回现实；临别江湖一句「有缘再会」。结局气质：过程即传说——《武林群侠传》时间线上洛阳有小虾米雕像、其消失百年后仍被少年英雄会追忆。谁「死」谁「活」：终战是挑战而非原著全员团灭；原著角色命运以玩家是否击杀/是否收入队伍的游戏选择为准，写正文时勿无故屠杀原作善终者。

⑬ **贯穿支线：道德、招募与练功哲学**  
高道德易收正派（如袁承志需极高道德、虚竹等高门槛）、和平取射雕/碧血等书；低道德易收邪派（如阿紫等）但终战更虐。野球拳初期弱、后期满级逆天；左右互搏是低资质的神技；九阳神功、九阴真经、降龙十八掌、太极剑、吸星大法、北冥、六脉、葵花/辟邪等构成「天书武学池」。主题：**养成与选择比线性剧情更重要。**

**【隐藏剧情 · 伏笔】**
- **圣堂=霹雳堂前史**：孔八拉继承后改名；地下室时光传送机才是「游戏世界」的元设定出口——呼应主角本是现代人。

- **野球拳梗**：初期垃圾技能、后期破局神技，与「小虾米」草根名形成主题：最不起眼的拳头打上黑木崖。

- **角色归属错位彩蛋**：如蓝凤凰、金轮、韦小宝用途与原著不完全一致，属游戏改编，写正文时以游戏逻辑优先并保留金庸名场面气质。
- **四大天王原型**：南贤、北丑、孔八拉、林厨子原型为河洛成员，是「开发者现身」式隐藏层。
- **续作时间线**：《武林群侠传》设在小虾米消失约百年后，洛阳雕像与少年英雄会=其传说化；勿在本世界通关前强行并入逍遥谷主线。

- 笑傲线 → 梅庄、黑木崖、东方不败。  
- 侠客/倚天线 → 侠客岛、光明顶、九阳体系。  
- 碧血/鸳鸯线 → 金蛇、双刀、十四书齐。  
- 华山论剑 → 盟主、神杖。  
- 霹雳堂地下室 → 安放天书、十大高手、时空裂缝归乡。  
- 传说期 → 洛阳雕像、百年后仍被追忆。

**【叙事基调 · 雷区】**
画风：90年代单机武侠RPG的自由、恶趣味与金庸群像致敬；正文应有江湖客栈烟火、门派踢馆张力与「天书是书也是命」的收集感，可带轻度穿越吐槽，但武学对决须认真。  
忌：①把金庸原著时间线强行统一（本世界是并置沙盒，不是编年史）；②无视道德写「无代价邪功」；③低阶秒杀东方不败/张三丰/乔峰；④把野球拳全程写成废物或开局就满级；⑤用《金庸群侠传X》门派剧情替换DOS原版天书主线。  
最早可切入锚点：小虾米刚得罗盘、尚未完成任何天书之时；或任一单线（如药王庄前）的「群侠尚未被收编」窗口。

## 阶位切入点

> 阶位↔原作战力：一阶≈江湖新人/三流；二阶≈二流门派骨干；三阶≈一流成名高手；四阶≈绝顶宗师（东方不败/五绝级/满状态小虾米）。世界顶点为宗师天花板，**无超阶**；低阶切入一律规避顶点·情报优先。

**一阶（江湖新人·罗盘初开与市井求生）**

切入身份/时点：契约者以「刚穿越的落魄青年 / 客栈跑堂 / 药童 / 镖局新丁」身份切入，锚定**小虾米刚从南贤处取得罗盘、野球拳未成形、十四天书零收集**前后。

初始事件：夜宿中原客栈，听人议论「有人睡一觉就从现代掉进武林」；清晨被小混混勒索，南贤居方向有人发放「智慧果换线索」的传闻。你必须在被卷进踢馆私斗前，选择：跟去南贤居、随商队离开、或偷听两位老者争论「天书到底是不是武功秘籍」。

开场白建议：「你醒来时，窗户外是旌旗和马粪味，不是空调声。柜上罗盘的指针乱转，跑堂的说南贤居的老头专收『异乡人』。街口有人喊劫道，有人喊卖刀谱——你还打不过他们，但你已经听见了『十四天书』四个字。」

关键NPC立场：**南贤**（愿指点、换智慧果，试探你是否真是异乡人）；**北丑**（胡言乱语里塞坐标，笑你三脚猫）；**小虾米**（同期竞争者或临时队友，抢线索时会翻脸）；**韦小宝**（路过兜售劣质刀与情报，价码黑心）；**田伯光**（可早期接触的怪侠，好色却守约）；客栈刀客（装备压力，逼你表态正邪）。

主线钩子/支线：钩子——护送小虾米或抢先抵达药王庄外围，目击阎基与七心海棠传闻。支线：①为南贤跑腿换第一枚智慧果；②在林厨子小屋帮厨换补给；③阻止低级山贼劫镖，攒第一笔声望。

危险度/规避：危险度低~中。规避苗人凤、任何门派掌门、毒术对决；莫进黑木崖方向。一阶硬闯金轮寺/梅庄=送死。

任务方向/奖励：活过第一周、建立罗盘导航习惯、接触飞狐线入口；奖励向：基础内功残页、小还丹、商路人情、道德初始偏移权（一次关键选择）。

**二阶（门派骨干·天书初猎与道德分叉）**

切入身份/时点：契约者以「已入门的武当/少林记名弟子、丐帮九袋以下、药王庄助手、或小虾米队中的第二把刀」切入，锚定**飞狐双书已得或进行中，连城/射雕线刚打开**的中早期。

初始事件：药王庄战后余波未平，胡斐要单挑苗人凤，程灵素缺一味药引；同时北丑说天宁寺佛像后「有纸香」。你被请求：做胡斐的陪练公证、去江南取唐诗选辑、或替郭靖黄蓉传一封「查品德」的信。

开场白建议：「刀声从苗人凤居传来时，你已经能听出胡家刀法的进退。程灵素把解药瓶塞进你掌心：『别让他死。』罗盘指向江南，南贤却说桃花岛不欢迎缺德的人——你第一次发现，拳头之外还有道德这根绳。」

关键NPC立场：**胡斐**（认能挡刀的人）；**程灵素**（用医术换你守诺）；**狄云**（沉默报恩）；**郭靖**（高道德才正眼看你）；**黄蓉**（出题考机变）；**周伯通**（拉你玩游戏，赢了给线索）；**欧阳克**（找茬，试你毒抗）。

主线钩子/支线：钩子——介入胡斐 vs 苗人凤公证战，或完成连城诀佛像解谜。支线：①黑龙潭送瑛姑口信；②替丘处机送信全真；③在金轮寺外围「路过练拳」不深入BOSS房。

危险度/规避：危险度中。可挑战山寨头目与门派弟子，规避东方不败、乔峰全力、张三丰；桃花岛硬闯需道德或战力评估。

任务方向/奖励：拿下1~3本天书相关进度、学会一门成型拳掌/剑法；奖励向：铁掌拳谱级、医疗用毒入门、中阶兵器、队伍位推荐信。

**三阶（一流高手·名场面介入与黑木前夜）**

切入身份/时点：契约者以「成名游侠、明教分舵客卿、令狐冲酒友、杨过并行的独行客」切入，锚定**天龙/神雕/笑傲前置进行中，梅庄四友可挑战、乔峰可单挑、杨过已得玄铁**的中期。

初始事件：燕子坞送来玉玺消息，丐帮传来「有人要挑战乔峰换天书」；另一边福威镖局《溪山行旅图》被三方争抢。你被卷入：为令狐冲凑齐四样书画、或作为公证人登上乔峰的擂台边缘。

开场白建议：「乔峰把酒碗放在桌上时，周围没人敢出声。你知道自己已能一掌裂石，可仍不敢接他那一掌。令狐冲在背后嘀咕：『四样宝贝凑齐，梅庄的门才开。』黑木崖的风还没吹到，你已经闻到葵花香。」

关键NPC立场：**乔峰**（敬勇士、厌阴谋）；**令狐冲**（酒与义气结盟）；**杨过**（认强者，嫌伪君子）；**小龙女**（寡言，只问是否伤杨过）；**慕容复**（利用你复国）；**岳不群**（伪善招揽）；**任我行**（地牢交易吸星风险）；**蓝凤凰**（五毒人情）。

主线钩子/支线：钩子——梅庄四友连环单挑，或高昌迷宫开隔世石。支线：①绝情谷断肠草助杨过；②冰火岛见谢逊听屠龙往事；③拒绝带头大哥书信的降道德捷径。

危险度/规避：危险度高。可与一流对刀，仍须规避东方不败全力、郭靖+黄蓉合击、圣堂终局；吸星大法反噬与辟邪诱惑写成真正代价。

任务方向/奖励：推进笑傲/天龙/神雕关键天书，收1~2名一流队友；奖励向：吸星残篇（慎）、玄铁剑线索、六脉/北冥机缘、成名声望。

**四阶（宗师·华山神杖与圣堂终战）**

切入身份/时点：契约者以「武林一方宗师、小虾米结义兄妹、盟主候选人、或圣堂编外护法」切入，锚定**十四天书集齐前后、华山论剑至霹雳堂地下室**。

初始事件：主角居客厅金光一闪，华山论剑帖摊开；孔八拉放话「无杖者莫入厨房地洞」。道德榜已定：你或站在小虾米身侧准备战十大恶人，或在低道德世界里听见郭靖要「替天行道」。

开场白建议：「华山风烈，神杖比想象中沉。孔八拉只看杖，不看脸：『天书归位，你们才能回家。』地洞书架空着十四格，每一格都是一部江湖。你听见上方传来两种脚步——恶人索要秘籍，或善人要你的命。」

关键NPC立场：**小虾米**（视你为终局搭档或夺杖竞争者）；**孔八拉**（流程执行者，无私情）；**东方不败**（若未倒，仍是黑木余悸）；**张三丰**（劝你向善再入圣堂）；**郭靖**（低道德线最强阻击）；**周伯通**（终战互搏，玩心杀机并存）；**欧阳锋**（高道德线恶人潮头）；**南贤**/**北丑**（送行与最后谜语）。

主线钩子/支线：钩子——华山论剑夺杖、地下室安放十四天书、十大高手团战。支线：①调整道德临界改变终战名单；②满级野球拳+左右互搏的极限养成；③通关前与林厨子最后一餐（人情收束）。

危险度/规避：危险度极高/贴近顶点。世界顶点即本阶团战本身：**存在·情报优先/条件性胜利**（依赖养成、走位、药与互搏），**严禁**用“顶点战力归零/被压制成凡人”式解释。未满状态强闯善人终战极易全灭。

任务方向/奖励：成盟主或助盟主归乡、见证时空裂缝；奖励向：宗师级秘籍合集、武林神杖仿制荣誉、天书抄本（剧情向）、归乡卷轴级「任务完成证明」、洛阳立像级声望（叙事）。
