# 仙剑奇侠传3外传
<!--meta lib=主库 tiers=一、二、五、六-->

## 剧情

**【作品来源】**
《仙剑奇侠传3外传》（全称常见为《仙剑奇侠传三外传·问情篇》，英文常作 Chinese Paladin 3 Plus / Chinese Paladin 3 Gaiden: Wenqing）由大宇资讯旗下上海软星开发、大宇发行，2004 年上市的 PC 角色扮演游戏，是《仙剑奇侠传三》的外传，时间上接在三的锁妖塔事件约十八年后，约在一代故事前三十二年。主题定为「问情」：亲情、友情、爱情与天道使命如何互相撕扯。引擎沿用仙三 3D，新增五灵轮、摄灵、兽化变身等系统；迷宫密度高，玩家戏称「问路篇」。本档案以**游戏原作正典**为准，写总览与前中段可玩舞台；同库若另有「问情篇」分条，可互补中后期与更高阶，但不互相覆盖终局人名与核心因果。标签锚点：南宫煌、温慧、雷元戈、星璇、王蓬絮、五灵轮、地脉、里蜀山、燎日火魔兽。

**【世界定位】**
神州仙侠世界中，以**蜀山仙境—地脉十二经络—里蜀山妖界**为立体舞台的人情与天道剧。蜀山因地脉失衡濒临崩坏，掌门徐长卿遣半路出身的南宫煌取回五灵轮、疏通五灵地脉；煌在旅途中结识逃婚的镇北王郡主温慧、五毒兽化形的王蓬絮、里蜀山少主星璇、滞留人间的鬼差雷元戈，揭开自己与星璇同为赤炎、丝缎双生子的身世，并最终面对燎日以祭品唤醒火魔兽、打通里蜀山火脉的阴谋。一句话：问的是情，走的是地脉，赢的是代价，不是免死金牌。

**【世界观 · 力量体系】**
本界力量不可硬套网文「炼气筑基」阶名，应按**实际破坏力、门派地位与血脉代价**落位。

1. **力量本源**
   - **蜀山仙术 / 御剑**：御剑飞行、剑气、阵法、锁妖塔相关禁制；正道中枢战力来源。
   - **五灵之力（水火雷风土）**：经**五灵轮**收灵、平衡、变身与摄灵法阵；打通地脉的关键权柄，也是南宫煌成长轴。
   - **妖力 / 血脉变化**：狼妖赤炎一脉、燎日体系、里蜀山妖众；南宫煌可兽化（狼人形态），星璇身负父尸身合法术与剧毒。
   - **鬼道 / 阴司权柄**：雷元戈一系勾魂、纸钱、腹语与灵兽「风」「雅」「颂」；在人间滞留即有规则代价。
   - **仙兽 / 毒系**：王蓬絮本为五毒兽，可产五毒珠、读心、化形；五毒珠关乎解毒与情线分岔。
   - **凡俗武学与旁门**：温慧天生怪力与洞冥宝镜；南宫煌早年的看相批命、飞龙探云手等江湖旁门——战力有限，叙事浓度极高。
   - **神界 / 天道侧影**：殊明等已近神界者、锁妖塔与魔界之门相关布局；高阶压迫感来源，低阶只闻其名。

2. **死亡与代价**
   - 凡人与低阶弟子身死即止；地脉瘴气可克制蜀山仙术，使探脉长老功力尽失而死——**环境本身会杀人**。
   - 高阶可借身合、毒防腐、祭品唤醒、招魂等手段延命或召回，皆有剧毒、献祭、失忆、离队等代价；原作多结局中星璇、思堂、雷元戈等皆可能付出生命。
   - 五灵轮爆发、摄灵法阵次数、好感度会改写牺牲名单——胜利常是条件性的，不是战力归零后的无痛通关。
   - 燎日化为火魔兽（燎火炙日兽）后，地脉崩塌、岩浆喷涌，战场本身即地区级灾难。

3. **战力阶梯（按破坏力表现）**
   - **村镇江湖 / 未入门杂役**：徒手、冷兵器、小术；≈一阶。
   - **蜀山外门与游方修士 / 成名侠客雏形**：碎石裂木、单挑小队妖物、初阶仙术；≈一~二阶。
   - **正式弟子与一域好手**：成型剑气、御剑短途、端据点级妖将；≈二阶。
   - **蜀山中坚 / 里蜀山将领 / 地脉 BOSS 级**：掀建筑群、瘴气战场中作战、跨城追杀；≈四阶向（本档案不单开四阶切入，写中景时作背景压迫）。
   - **掌门 / 少主 / 清冷仙人级与领域感**：一城至多城压迫、五灵大成、阵法显化；≈五阶。
   - **燎日火魔兽 / 里蜀山火脉崩塌级**：地区级灾难、岩浆吞战场、需五灵轮与同伴合力条件性取胜；≈六阶。
   - **神界侧影 / 天道布局者**：非常规顶点边缘，本档案六阶切入仅作情报与规避，不写可日常修炼的「成神课表」。

4. **乐园阶位映射**
   村镇江湖与未入门≈一阶；蜀山外门/初成五灵/一域好手≈二阶；蜀山掌门与清冷级、里蜀山少主稳固战力、五灵大成≈五阶；燎日火魔兽与火脉崩塌级灾难≈六阶。宁低勿高。本库仅覆盖**一、二、五、六**阶。世界顶点以**燎日化火魔兽后的地区级灾难**与相关高阶仙妖为参照；低阶切入一律规避顶点·情报优先。**超阶巅峰**若触及殊明神界侧影，只写条件性胜利与情报战，严禁「战力归零式」解释。

**【地理 · 舞台】**
舞台以**表蜀山（人界仙门）—地脉迷宫（十二经络意象）—里蜀山（妖界）**三层套叠，辅以神州城镇行旅。

- **蜀山仙剑派 / 蜀山仙境**：开篇危机中心。仙境崩坏、瘴气侵门、弟子探脉折损；徐长卿议事、常纪养父日常、酒剑仙司徒钟客串、锁妖塔阴影皆在此。
- **地脉门户与阳性地脉链**：少阳三焦（雷）、厥阴心包、阳名百纳、太阴归尘、太阳华池、少阴凝碧、少阳参天、厥阴蔽日等——名称取自经络，实为五灵属性迷宫；瘴气、机关、妖将与「问路」式走法是日常。
- **绿萝嶂 / 绿萝山**：清冷仙人隐居地；五灵轮交接、收徒、风灵与旧怨（殊明线）的关键驿站。
- **里蜀山外城 / 内城**：妖界政治舞台；星璇少主、思堂忠仆、燎日支配；人族久留危险，星璇因身合尸身与毒更难久居人间。
- **盘古之心**：深层地脉与创世意象相关迷宫；高难度与结局前资源点。
- **唐家堡、登云路、胜州 / 室韦方向、京城、石村、双溪、渝州一带、夜光城等**：行旅与支线；温慧逃婚线、雷元戈室韦出身线、父母往事闪回（京城/石村）、隐居结局的石村铁铺皆锚定于此。
- **魔界之门相关层**：殊明布局与可选挑战；揭示「削弱妖族」的上层意图。
- **锁妖塔（本时期）**：仙三余波；徐长卿因理念与错误认知而隐退的心理战场，亦是殊明等可再挑战的符号地。

写场景时：底层是买酒、逃婚、捉小妖、坊市骗子；中层是地脉开图、里蜀山外交与毒线；高层是燎日祭品与火魔兽。本地人若听闻「一阶弟子单挑火魔兽」，会当疯子——正确反应。

地理动线建议：蜀山危机受命 → 下山遇温慧 → 救王蓬絮 → 绿萝山合五灵轮 → 逐段阳性地脉入里蜀山 → 结识星璇/思堂 → 胜州等处遇雷元戈 → 身世与父母闪回 → 打通诸脉、蜀山暂稳 → 身份暴露与逐山风波 → 再入里蜀山决战燎日火魔兽。

**【世界剧情线】**

① **前史：锁妖塔后的十八年与双生子悲剧**
仙三邪剑仙之乱与紫萱、徐长卿相关事件落幕约十八年后，徐长卿为蜀山掌门（因紫萱内丹等机缘成仙体）。狼妖**赤炎**与人类**丝缎**生下双生子：兄星璇、弟周煌（后名南宫煌）。赤炎曾被紫萱相关事件分出「无妖力的人」与「无记忆的妖」等形态纠葛，**燎日**一脉与记忆/力量追寻导致赤炎之死、丝缎自尽；兄入里蜀山为燎日侧少主，弟被蜀山弟子**常纪**收养，在蜀山长大却**始终未正式入门**。世界状态：人妖旧账未清，地脉暗流已在。

② **开篇危机：蜀山仙境崩坏（一~二阶舞台）**
未知原因（实与火属性地脉被打通、阴阳失衡相关）导致蜀山仙境崩坏，地脉门户大开。探查长老弟子遭**克制蜀山仙术的瘴气**化去功力，苦战妖众而亡。徐长卿临危授命：令南宫煌携书信往**绿萝山**求**清冷仙人**，取回蜀山秘宝**五灵轮**，疏通五灵，使蜀山「否极泰来」。煌玩世不恭、嘴贫、爱看相戏法，实则自卑于「养在山门却不算弟子」。下山买酒途中结识逃婚的**温慧**（镇北王之女，母系与蜀山有旧，天生神力，持洞冥宝镜），二人合作捉妖上山；又救下化形少女**王蓬絮**（实为五毒兽「小桃子」）。世界状态：一阶叙事为主，强者如天，任务像跑腿却死人。

③ **五灵轮合身与地脉启程**
清冷收煌为徒，将五灵轮与身相合，开启地脉入口。打通之法：以五灵轮开阳性地脉入里蜀山，再经对应阴性地脉回人界，逐一疏通。五灵收灵、平衡变身、摄灵法阵成为成长与结局变量。地脉内：少阳三焦追波罗大王、厥阴心包等层机关杀、瘴气逼退蜀山式打法——**换武器思维比硬刚仙术重要**。

④ **里蜀山与兄弟线：星璇、思堂、燎日**
里蜀山外城到内城，煌遇见少主**星璇**：冷傲、会做饭却因尸身与毒尝不出味，受众妖尊重，实为煌之孪生兄长。忠仆**思堂**护主至死不渝。燎日以养父/支配者姿态经营妖界，暗图修魔、以祭品唤醒**火魔兽**、打通火脉。星璇身中剧毒，五毒珠与王蓬絮线交织。人情主题在此加重：同父同母、一在仙门边缘一在妖界巅峰，却都缺「完整的家」。

⑤ **雷元戈与胜州/室韦线**
鬼差**雷元戈**因勾魂失败（与徐长卿等前史相关）滞留人间，口不能言，以腹语与灵兽说话，吝啬、烧纸钱、冷幽默；出身室韦方向，曾被当妖怪。与温慧有隐线情愫。队伍编制随好感与支线在「温慧 / 王蓬絮 / 星璇 / 雷元戈」间重组——**编制即结局伏笔**。

⑥ **身世揭露、父母闪回与京城/石村人情**
阳名百纳等段触发赤炎、丝缎往事：相国府、订亲、人妖不容、蜀山介入的阴影。煌确认半妖血统与狼人变身；身份一旦在蜀山暴露，将引发逐出山门等政治后果（酒剑仙亦可能被牵连）。石村、京城支线提供凡人尺度的「问情」对照：不是仙术高低，是谁肯为谁留下。

⑦ **地脉尽通、蜀山暂稳与理念崩裂**
诸脉打通后蜀山暂时恢复。徐长卿若发现锁妖塔与人妖相处的理念错误，可能选择隐退——正道领袖的「正确」本身成问题。殊明仙人线揭示上层对魔界之门、削弱妖族的布局，清冷旧怨（修为、妻子惨剧等说法）把「仙」字撕出裂纹。世界从救灾副本升级为**立场审判**。

⑧ **质变副本：里蜀山火脉与燎日火魔兽（卡片锚点）**
卡片锚点：**燎日变身火魔兽，打通里蜀山地脉，岩浆喷涌而出，南宫煌需在炼狱般的环境中阻止地脉崩塌。** 决战舞台在里蜀山深层/火脉相关战场。燎日以**温慧或王蓬絮**为祭品（视结局分支），唤醒火魔兽（燎火炙日兽）。星璇可能与燎日身合互搏自杀；思堂或雷元戈可能为救主/救人牺牲；五灵轮在隐藏结局中可突然发威减少牺牲。名场面必须含：祭品倒计时、岩浆与地脉崩响、兄弟对峙、同伴默认可死不可弃、出战后的「情归何处」。此卷是中高阶核心可玩副本：规则杀（毒、瘴、火脉）与人心杀并行。

⑨ **结局三岔（写终局勿乱改核心人名因果）**
- **温慧向结局**：王蓬絮好感更高时常见路径——絮以五毒珠解毒后化兽离去，雷元戈入队；燎日祭温慧，雷元戈护她牺牲；星璇与燎日合体赴死；温慧直面和亲责任，煌寻回兽形絮后赴室韦履约。
- **王蓬絮向结局**：温慧好感不低且摄灵法阵使用较少——絮在队，雷元戈离队；燎日祭絮，思堂救星璇而死，星璇与燎日互尽；絮守墓，煌修失传招魂术召回兄魂，人称「煌仙人」。
- **隐藏/团圆向**：温慧好感条件满足且摄灵法阵使用达高阈值（约 360 次量级）——五灵轮发威战胜燎日，主要牺牲可免；煌依常纪之意携家人隐居石村，絮与星璇走到一起（思堂可以黑猫等形态陪伴）。
主笔倾向与玩家呼声不必统一；AI 写终局须标明分支，不可捏造第四种「全员无代价成神」。

⑩ **微观政治与日常可介入**
买酒赊账、逃婚追捕、蜀山门规与「编外弟子」歧视、地脉告示牌走法情报买卖、里蜀山通行证与人族税、五毒珠黑市、鬼差纸钱汇率、清冷与殊明旧案的街谈、锁妖塔参观禁令、石村铁铺学徒——AI 优先这些，而不是一上来火魔兽。

⑪ **蝴蝶效应边界**
契约者可改：某段地脉减员、某次逃婚追捕成败、非核心宝箱归属、临时救下思堂一次、影响一次好感礼物。不可无代价改写：双生子身世结构、燎日火魔兽锚点、五灵轮必须合身疏脉的主轴、三结局牺牲逻辑的骨架。改了局部，因果以毒发提前、蜀山再崩、祭品换成另一人、兄弟相残更惨等方式回流。

**【主要人物】**
- **南宫煌（周煌）**｜性格：玩世、贫嘴、圆滑，内里自卑于半妖与编外身份；对情懵懂，先义后色。｜装备·能力：五灵轮、蜀山仙术与旁门、狼人变身、后期招魂可能。｜弧光：跑腿少年→地脉行者→问情之人→煌仙人或石村隐者。｜关系：常纪养父子；星璇孪生；温慧/絮情线；徐长卿任务链；燎日血仇。

**【世界剧情线·因果长链】**
煌不是天选超人，是**蜀山养大的半妖编外**。徐长卿把五灵轮任务交给他，既是信任也是无奈：正式弟子在瘴气里先死光了。温慧的逃婚拳头敲开「情」的第一种形状——伙伴。絮儿的读心敲开第二种——被看见的软弱。星璇的饭菜敲开第三种——血缘却隔着人妖与毒。雷元戈的纸钱敲开第四种——死生簿上的幽默。燎日把所有情收成祭品柴火；火魔兽是情的反面教材：只要力量与地脉，不要人。三结局不是「选老婆界面」，是问：你肯为谁死、肯让谁活、肯不能拥有谁。

火脉若止：蜀山仙境可居，人妖外交短暂窗口，半妖英雄或被歌颂或被忌。火脉若崩：岩浆改写地理，双方难民，任务世界进入灾后篇。隐藏结局石村：小历史胜利，大历史仍在锁妖塔与天道线上滚动——为一代故事留白，不剧透一代正文。

**【世界剧情线·与「问情篇」分条的分工说明】**
本条清单名「仙剑奇侠传3外传」承担总览、前中段地脉与一/二/五/六阶可玩层；同库「问情篇」分条可深写中高阶与另一套阶位覆盖。两边共用真名与终局骨架，禁止写成两个无关世界。卡片锚点火魔兽在两条皆应可指认，本条在六阶完整可玩。

**【主要人物·台词气质（非原文引语，是文风锚）】**
煌：半真半假的笑。温慧：直球。絮：小心翼翼的试探与突然爆发。星璇：短句。雷元戈：别人的嘴（风雅颂）说自己的话。燎日：宏大叙事腔。清冷：霜。徐长卿：疲倦的责任。思堂：行动代替形容词。

**【贵重物品·伪造与骗局】**
假五灵轮模型、假抗瘴香、假少主令、假和亲撤约文书、假招魂符——坊市与奸细爱用。验真手段：洞冥镜、清冷印、星璇血纹、鬼差阴司印。二阶正文「假货案」比秘境更常见。

**【势力·情报价码】**
地脉走法>里蜀山内城地图>燎日祭品名单>殊明行程>蜀山门规漏洞。钱不够时，可用「帮疏一条小脉」「替温慧挡一队追兵」「替雷元戈销一桩阴司小案」交换。

**【世界剧情线·兄弟相认夜】**
同脸、同血、不同山门气味。星璇递菜，煌贫嘴掩饰哭意，思堂在门外杀刺客，絮读到两人心跳同频。此夜之后，「杀燎日」从蜀山任务变成家事。五阶切入可卡在此夜前后，信息密度最高。

**【世界剧情线·祭品是谁的伦理战】**
队伍若投票牺牲谁，即已经输了问情。燎日要的就是把情算成燃料。正确打法是毁祭坛、断火脉、条件性封印/击破兽体，而不是「选一个朋友去死更优」。六阶正文禁止把祭品选择写成最优解算术题。

**【地理·里蜀山火脉层感官】**
空气烫到语言变短；石板下有心跳一样的脉动；岩浆光把人脸照成仇人色；五灵轮在怀中发烫像第二颗心；远处兽吼与地鸣分不清。

**【主要人物·对契约者的第一反应】**
煌：套话+看相（多半不准）。温慧：打量你能不能并肩。絮：读心，撒谎扣好感。星璇：验你对妖民的态度。雷元戈：估价你的装备值多少纸钱。燎日：评估你当不当得成柴。徐长卿：你是否增加山门死亡率。清冷：你是否配碰五灵概念。

**【贵重物品·五灵轮使用纪律】**
开脉前校准属性；战中平衡再变身；摄灵有次数命运账；对火魔兽优先「压脉」而非「秀伤害」。把轮借给外人≈宣判蜀山再崩。

**【隐藏剧情·酒剑仙与门规】**
司徒钟的醉是保护色；当半妖身份案爆发，他是否护短决定蜀山「还有没有人味」。支线价值高于战力数字。

**【世界剧情线·清冷收徒的条件】**
不只是送信成功，而是煌肯把「玩世」暂时放下，承认怕与在乎。五灵轮认主认的是心的缺口，不是功法年级。

**【世界剧情线·室韦路上的风】**
若走温慧责任线，草原风比蜀山剑气更磨人。煌若以国师/顾问身份随行，战力戏减少，问情戏增加——五阶后也可以是政治与陪伴，不是永无止境的迷宫。

**【主要人物·再增群像】**
- 追温慧的王府武护｜公务脸｜二阶｜可打可骗。
- 里蜀山厨役｜敬星璇｜低｜情报。
- 地脉亡魂残识｜怨｜环境｜给走法或诅咒。
- 石村孩童｜慕剑｜凡｜隐结局镜子。
- 神界传令残影｜冷｜六阶边缘｜只宣读不解释。

**【贵重物品·终局账户】**
胜利战利品往往不是神器，而是：一条还活着的命、一碗星璇再也做不了的菜的记忆、一面不再需要逃婚的镜子、一枚烧剩的纸钱、一轮不再轰鸣的五灵。写奖励时优先这些，再写装备。

**【世界剧情线·收束句】**
这世界的剑会生锈，地脉会再震，门规会再写；问情的答案每次只能用命付一次。契约者若只带走五灵轮的「功能说明」而带走不了任何一段情，则任务成功也算失败。

## 阶位切入点

阶位↔原作对照：一阶≈村镇江湖/蜀山编外跑腿/地脉外围小妖与追兵；二阶≈五灵初合、外门级仙术、一域行旅与地脉中层妖将；五阶≈蜀山掌门/清冷级、里蜀山少主稳固、五灵大成与跨域布局；六阶≈燎日火魔兽与里蜀山火脉崩塌级地区灾难。宁低勿高；本世界不覆盖三、四、七、八、九阶切入（勿硬写）。

**一阶（凡俗新人 · 蜀山编外与逃婚路上）**

切入身份/时点：以蜀山杂役、坊市术士、唐家堡跑腿、温慧逃婚路上的护卫/追捕方小卒、或刚入地脉门户外围的新弟子切入；锚定徐长卿已授命、煌尚未合五灵轮或刚下山之时。

初始事件：仙境裂缝落下碎瓦，门房弟子嘲「编外也配送信」；你被派去给南宫煌带酒钱/带路，半道撞上砸花轿的**温慧**与官兵冲突。选择：帮郡主、帮官兵、或趁乱偷洞冥宝镜的仿品——仿品会害你被真镜照破。

开场白建议：「蜀山的云裂开一条缝，像谁把天幕撕了个口子讨债。南宫煌把酒葫芦倒过来，一滴都没有，笑着问你借三文钱买醉。山门石阶上，有人正用仙术的边角料吓唬逃婚的姑娘——她一拳比你的符纸响。一阶的你若想活，先学会：这里死的不是英雄，是跑腿。」

关键NPC立场：**南宫煌**——贫嘴背后观察你是否可靠；**温慧**——只认拳头与公道；**常纪**——护短儿子；**蜀山势利弟子**——踩你换绩效；**王蓬絮**（若已遇）——读心，撒谎即凉。

主线钩子/支线：送信/带路到绿萝方向的第一段；帮温慧甩掉王府武护；买真抗瘴香囊；支线＝酒坊赊账、假五灵轮骗局、地脉告示牌残页。

危险度/规避：低阶——远离锁妖塔深层与里蜀山内城；见瘴气浓到仙术失灵就撤；禁止妄称能单挑「山里那头火」。

任务方向/奖励：凡级灵材、抗瘴劣香、温慧的人情、煌的半真半假情报；奖励取向＝活命道具与门路，非法宝爆炸。

**二阶（超凡起点 · 五灵初合与地脉中层）**

切入身份/时点：以清冷记名外围、蜀山外门、地脉向导、里蜀山外城牙人翻译、或队伍临时佣兵切入；锚定五灵轮已合身、少阳三焦等初段地脉开启、星璇尚未完全认弟。

初始事件：**南宫煌**请你协助采集某属性灵以稳轮；同层**波罗大王**级妖将堵路，瘴气让蜀山式输出打折。**王蓬絮**突然说你心里有私货——你要在「交还偷藏的灵珠」与「被读心当众撕破」之间选。

开场白建议：「五灵轮在南宫煌掌心转了一圈，雷火水土风像五条不听话的蛇。地脉石壁上的告示牌写着死人的走法，走错一格就有人变成墙上的新字。二阶的你已经能碎石裂木，可在这里，方向比拳头贵。」

关键NPC立场：**清冷仙人**——考你心性是否配碰五灵；**星璇**（初见）——冷，称重你的立场；**思堂**——防刺客也防你；**雷元戈**（若已入队）——用纸钱跟你谈价；**地脉妖将**——领地意识。

主线钩子/支线：协助疏通一条阳性脉段；护送队伍过里蜀山外城市集；替星璇找续毒/五毒珠线索；支线＝假少主令案、胜州鬼差销假。

危险度/规避：中——可打妖将，不可闯燎日内城火脉；对殊明只听不惹；半妖话题在蜀山弟子前慎提。

任务方向/奖励：属性灵珠、中阶兵刃、地脉完整走法抄本、外城通行证；奖励取向＝生存与情报，不发终局神器。

**五阶（城市级 · 掌门桌与少主桌）**

切入身份/时点：以可与徐长卿对坐的隐世高人、清冷同辈客卿、里蜀山可受星璇接见的外使、或五灵大成的同行者切入；锚定地脉大半已通、蜀山暂稳、身份暴露危机与殊明布局并行。

初始事件：议事殿上，**徐长卿**问你：锁妖塔究竟是镇妖还是害妖；窗外仙境刚稳又颤。**星璇**密信至，称**燎日**已在选祭品。你要在「向蜀山全盘托出」与「先帮兄弟截祭品名单」之间选——两选都会树敌。

开场白建议：「蜀山的云第一次像云，又像随时会裂的疤。徐长卿的茶凉了，清冷的风从绿萝吹到议事窗。里蜀山的信只有一行字：火要醒了。五阶的你坐得进这张桌，是因为你能移开一座城的重量——但移不开别人心里的山门。」

关键NPC立场：**徐长卿**——要山门与公道的两全，知道不可能；**清冷**——要传人与旧案真相；**星璇**——要弟与族的两全；**殊明**——要你做棋子；**温慧/絮**——要你别把人算成棋。

主线钩子/支线：截获祭品名单；安排半妖身份公关或假死；与殊明谈判换时间；支线＝酒剑仙护短案、门规修订阴谋。

危险度/规避：高——可与少主/掌门级交手边缘周旋，仍禁止solo 火脉核心；对神界侧影情报优先。

任务方向/奖励：五阶兵刃/护身罡、内城短时令、祭坛结构图、政治人情；奖励取向＝布局权与撤退权。

**六阶（地区级灾难 · 燎日火魔兽与火脉崩塌）**

切入身份/时点：以可参与终局决战的顶尖战力、五灵轮共鸣协作者、或妖界反燎日联盟代表切入；锚定祭品已锁定、燎日即将化兽、岩浆将灌地脉。

初始事件：里蜀山火脉层，**燎日**点名**温慧**或**王蓬絮**为祭；**星璇**拔刀指向养父亦指向自己；**思堂**已站到必死的位置。**五灵轮**烫得发亮——你若抢输出不顾压脉，地脉仍崩；你若只顾救人，兽化燎日会先撕开队伍。

开场白建议：「岩浆把夜空映成冶炼厂。燎日的影子在火里站起来，比山门更高，比门规更旧。南宫煌的轮光碎成五色，温慧的拳上是血，絮儿的毒香里是哭，星璇的脸像要说一句终于说出口的哥哥。六阶不是让你单挑天灾，是让你决定：这场问情的账单，你付哪一角，又绝不让谁被算成柴。」

关键NPC立场：**燎日**——只要火与魔，不要谈判；**星璇**——可死不可辱族与弟；**思堂/雷元戈**——牺牲窗口已开；**南宫煌**——需要你补位压脉而非抢最后一击的名；**温慧/絮**——拒绝被「为了大局」卖掉。

主线钩子/支线：毁祭坛；合力压制火魔兽；守护非祭品队员；触发或协助五灵轮爆发条件（叙事化，不写外挂指令）；战后处理蜀山叙事与石村/室韦分岔。

危险度/规避：贴近顶点——火魔兽为六阶地区级灾难本体；胜利必须是条件性：断祭+压脉+同伴协作（及可能的轮光爆发/身合互尽等原作分支），严禁写成「兽体无故失去战力后任打」或用任务规则取消其破坏力。殊明神界侧影仍以情报与规避为主，不在此阶单独加戏成可练级目标。

任务方向/奖励：终局见证者地位、六阶以下顶尖防具/心法、火脉稳定残识、主题级信物（一枚纸钱、一颗五毒珠壳、一轮裂痕、一块石村铁渣）；奖励取向＝命与情的剩余，而非无代价神器。
