# 対魔忍RPG-最终决战

## 剧情

**【作品来源】**
《対魔忍RPG／対魔忍RPGX》（LiLiTH，2018年9月25日运营开始）长线主线的「合流终局」档案。系列原作监修笹山逸刀斎；世界观承接《対魔忍アサギ3》再创造、《決戦アリーナ》与RPGX学园／深渊两线。本档案锚定副标题「最终决战」——诺玛德始祖**埃德温·黑**、脑吸食者残存、内阁与米连G、五车对魔忍与独立游撃队在同一棋盘上摊牌的终局战役。资料综合日文维基对魔忍RPG条目群与系列通史。文风：国家机密战争＋群像牺牲，不再保留学园喜剧的安全网。

**【世界定位】**
深渊入侵被压成「门级裂隙」之后的现在。五车仍在，但全国对魔忍进入战时编制；黄泉原不再只是犯罪都市，而是终战前线之一；东京王国、米连、魔界九贵族同时派代理人入场。主角**风魔小太郎**的右目邪眼在终战前夜彻底觉醒边缘，独立游撃队从「学生特遣」升格为「可影响国运的刀」。本篇回答三个问题：黑要什么、阿尔萨尔如何「从现在内部出生」、五车还能不能在终战后继续当学校。

**【世界观 · 力量体系】**
终战期力量分层：
1. **对魔忍忍法全开**：限制器解除成为常态，副作用是寿命与理性。
2. **诺玛德血盟与始祖权能**：黑可赋予干部「夜之律法」，在夜间域内改写物理。
3. **脑吸食者内部化**：不再靠大型裂隙，而靠「被说服的人」当移动巢点。
4. **国家战略兵器**：神田旅团非法部队、G的沙耶系量产、内阁情报局的清洗名单。

战力：见习～下忍≈一～二阶；中忍≈三阶；上忍≈四阶；**五车精锐／游撃队终战编制≈五阶**；诺玛德干部／地区灾厄≈六阶；黑与阿尔萨尔完全体≈七阶～超阶。

乐园阶位映射：未觉醒～下忍≈一～二阶；中忍≈三阶；上忍≈四阶；终战精锐≈五阶；干部级≈六阶；始祖／巢主≈七阶及超阶。顶点情报优先；**超阶巅峰**条件性胜利。

**【地理 · 舞台】**
- **五车终战司令部**：学园地下扩建为全国指挥节点。
- **黄泉原全开战场**：魔界之门周边三十公里成无人区。
- **东京王国上空**：浮空火力与义体空降场。
- **旧皇居／政府避难层**：黑的「日宴」与谈判桌。
- **裂隙残核室**：深渊篇留下的主裂隙被关在五车最深层。
- **魔界临时桥**：九贵族观战席，偶尔投子。

**【世界剧情线】**
① **终战宣言**  
浅葱向全体对魔忍发布非公开战时令：诺玛德将在「日蚀夜」开启大规模门；同步，阿尔萨尔录音中的「内部出生」被证实——多名高官与对魔忍干部已成移动巢点。政府要求五车「可牺牲」，浅葱要求「先救人再谈面子」。小太郎被授予跨部队指挥权，时子接管后勤总账。

② **三线并进**  
A线黄泉原：紫、樱、英格丽德（敌或临时同盟）争夺门控制权。B线东京王国：雪风、凛子、假面对魔忍切断G的量产沙耶。C线五车深层：小太郎、契约者、桐生守护裂隙残核，防止内部出生完成。三线必须在同一夜成功两条以上，否则黑的日蚀仪完成。

③ **日蚀夜总攻**  
黑现身不是为了立刻毁灭世界，而是举行「新契约」：以对魔忍血脉为墨，重写人魔互不干涉的旧约。胧忍军、骑士军、菲鲁斯特残党同时出动。浅葱以杀阵华对黑的夜域，第一次正面交锋以「双方试探」结束——浅葱重伤，黑袖口破损。证明五阶精锐可伤始祖，但不能击杀。

④ **内部出生**  
阿尔萨尔选中的「容器」在终战中段揭晓：不是路人，而是某位被信任的五车关系者（AI可在凛花／某教师／某毕业生中按玩家线索落点，但不可是浅葱本体）。容器觉醒时裂隙残核暴走，未来锈斑吞没司令部一层。双雪风（若未来雪风仍可被召唤）再次同调，短暂压制。小太郎右目完全看见「杀死容器／保住世界」与「救下容器／世界半毁」的分叉。

⑤ **条件性胜利的定义**  
终战没有无代价的S。原作逻辑下的可达结局包括：①斩容器＋重创黑，五车残存，黄泉原门半毁；②与英格丽德交易放黑撤退，换取摧毁容器与裂隙；③小太郎以邪眼短暂对等黑，换取同伴撤离，自己失明或沉睡。AI必须让玩家看见代价清单再选。

⑥ **终战后的世界**  
对魔忍身份边缘公开化；学园是否重建成悬念；诺玛德转入地下；脑吸食者被标记为「可复发传染病」。雪风把枪托上的裂痕留下当纪念；蛇子的汤变成葬礼与庆生共用；时子把死亡账本封存。小太郎若存活，会在废墟教室写下：下一学期——如果还有。

⑦ **蝴蝶效应边界**  
可改：容器人选、三线哪条成功、是否与诺玛德交易、小太郎是否失明。不可无代价改：黑的始祖级本质、终战需要牺牲的结构、对魔忍作为国家刀具的定位。

⑧ **与前两档档案的关系**  
学园篇给人；深渊篇给恐惧；本篇把两者烧成同一把刀。未经历前两档的契约者会被终战信息密度压垮——建议先读前档。

五车对魔忍；诺玛德；米连G；内阁清洗派；魔界观战贵族；脑吸食者残党；东京王国黑帮临时民兵。

**【贵重物品】**
日蚀仪核心；裂隙残核；邪眼抑制眼罩；杀阵华全开催化剂；雷核最终片；逆同化终极剂（一支）；英格丽德的停战纹章；死亡账本。

**【隐藏剧情 · 伏笔】**
黑的新契约可能是为了对抗更大虚空威胁；容器人选与小太郎血脉有关；再创造世界并非最终层；紫水若再现将指向「下一轮再创造」。

**【大事记时间线】**
深渊冷战 → 终战令 → 三线部署 → 日蚀夜 → 容器觉醒 → 条件性胜利 → 残世界重建。

**【叙事基调 · 雷区】**
史诗残酷、牺牲可见；忌无代价斩杀始祖；忌把终战写成后宫收官；忌浅葱战力归零。最早切入：终战令下达当夜。

**【三线战术详述】**
A线黄泉原：目标是门控室。紫开路，樱负责平民隔离，契约者可选择炸门或夺控。炸门会让魔界愤怒，夺控要与英格丽德谈判。英格丽德条件通常是「不追击黑的本体」。B线东京王国：雪风与凛子切断沙耶量产线，假面对魔忍可能背后开枪。成功标志是工厂停转与数据焚毁。C线五车深层：最像恐怖片。灯管逐个熄灭，同事突然微笑劝降。桐生的终极剂只能用一次，用在容器还是用在小太郎，是终战最大抉择之一。

**【日蚀仪机制】**
日蚀仪以人血与门坐标为燃料，进度条分五格。每失败一条主线亮一格；五格满则关东进入永久夜域，五阶契约者无法以常规手段翻转，只可转「流亡线」。AI应把进度条可视化。

**【终战对话核】**
黑：「你们杀妖魔，像牧人剪羊毛。」浅葱：「那你就是要剥皮的狼。」小太郎：「狼也怕陷阱。」雪风：「少说话，多给坐标。」时子：「伤亡数字我来写，你们负责让数字停涨。」

**【战后重建模块】**
可玩：搜救、审判叛忍、与媒体周旋、决定学园是否公开、安葬名单朗读。奖励不再是神器，而是「还能一起吃饭的人」。

**【人物补完段落】**
小太郎在终战中必须连续做不完美决定，其弧光是「接受自己会恨自己」。浅葱的弧光是「把学生当成士兵后仍记得他们名字」。雪风的弧光是「从好胜到承担」。蛇子的弧光是「用生活细节对抗史诗」。桐生的弧光是「第一次真心希望某人活」。黑的弧光不是被感化，而是重新评估人类刀具的价值。阿尔萨尔的弧光是「从外敌变成内寄生」。

**【地理补完】**
终战夜的五车天空会出现双重月：一月是真月，一月是神经云反射。黄泉原的排水隧道变成运兵道。东京王国的赌场屏幕被黑强制播放「新契约宣言」。旧皇居谈判桌铺着对魔忍套装碎片——黑的品味。

**【物品使用例】**
邪眼抑制眼罩可让小太郎延迟觉醒副作用，但摘下时预知会爆发。停战纹章可让诺玛德部队停火十分钟，对脑吸食者无效。死亡账本在终战可被朗诵为「反精神武器」：念出名字能短暂唤醒轻度同化者。

**【隐藏补完】**
若玩家在深渊篇保留锚定钉碎片，终战可额外关闭一格日蚀仪。若学园篇救下特定同学，终战多一条民兵支援。若与英格丽德交恶，A线难度上升。

**【叙事执行】**
节奏快、镜头碎、伤亡点名。每死一个具名角色，应用真名与一句遗言。禁止用「若干人牺牲」糊弄。

**【终战全景补充】**
日蚀夜不是单点爆破，而是全国同步的「门压」。北海道到九州的对魔忍驻屯地同时收到浅葱的战时令，但只有五车掌握裂隙残核与阿尔萨尔录音。这意味着终战的信息差本身就是武器：地方部队在打诺玛德外围时，并不知道真正的胜负在五车地下。契约者若向地方泄密，会换来援军，也会让黑提前调整仪程。

三线并进要求AI同时维护三张小地图。A线黄泉原的空气有硫与香水混合的怪味——那是魔界门与诺玛德夜总会叠加的气味。紫的斧每砍断一条导管，英格丽德就记录一次「人类的无理」。樱用影遁带走平民时，会把孩子的眼睛蒙上，说「等天亮再看」。B线东京王国的工厂灯光像手术无影灯，沙耶量产体的笑容一模一样，雪风必须学会「对同一张脸开第七枪」。凛子的空遁在金属丛林里受限，需要鹿之助先短路雷达。C线五车深层最安静，安静到能听见锚定钉的金属响。桐生在那里准备终极剂，嘴里说笑话，手却稳。

黑的「新契约」内容在谈判桌上被宣读：对魔忍承认夜域治权，换取普通人免于无差别猎杀；拒绝则日蚀仪走完，关东变牧场。浅葱的回答是把契约书砍成两半——杀阵华的分身在桌面上同时落下。黑笑了，说「刀具有了自己的意见」。这一笑不是胜利，是兴趣。终战的核心戏剧是：始祖开始认真对待人类。

容器觉醒时，司令部的广播会自动切换成温柔的劝降。认识的声音劝你放下武器。对抗方法是时子准备的「噪声磁带」：内容是学园祭的吵闹声、蛇子的汤沸腾声、雪风骂人的声音——生活噪声能短暂对抗精神殖民。这是学园篇细节在终战的回收，AI应让玩家感到「日常原来是武器」。

条件性胜利的三种主结局必须写清代价：结局一，斩容器＋黑撤退，五车半毁，小太郎右目失明；结局二，交易放黑，裂隙永封，英格丽德成为监视者，五车失去对黄泉原的主动权；结局三，小太郎以邪眼对等黑十秒，全员撤离，小太郎沉睡，时子摄政风魔。没有「全都要」的第四结局。

战后审判是终战的第二战场。内阁要清算「谁把学生送上战场」；媒体要英雄；家属要尸体。浅葱可能被要求辞职；小太郎若存活，会成为符号而非人。契约者可选择站法庭、站阴影、站重建工地。重建工地上，紫把碎斧埋在校门，樱种树，雪风教新生「先学会停火再学会开枪」。

**【终战人物微观】**
小太郎在终战夜写了三封信：给时子、给蛇子、给「可能成为容器的人」。信不一定送出。雪风把Lightning Shooter的限制器拆掉，说「活不到需要限制的时候」。凛子把空遁坐标设成「全队优先」而不是「自己优先」。桐生第一次拒绝采集某具尸体，说「让她完整」。黑在撤退时留下一枚棋子给小太郎，棋子上刻着「下次」。阿尔萨尔的残响在裂隙关闭后仍会在梦里出现，但只能问问题，不能下命令——深渊变成了心理病，而不是天灾。

**【终战地理微观】**
黄泉原的排水口在终战夜喷出荧光的魔血，像反向的瀑布。东京王国赌场的轮盘被子弹卡死在「00」。五车钟楼的指针在日蚀最高点停摆七分钟，全国时间服务出现跳秒。魔界临时桥上，某九贵族举杯：「人类今晚很好看。」这些细节用于让终战「可闻可见」，而不是抽象的数值对撞。

**【终战物品微观】**
日蚀仪核心被夺取后会灼伤持有者灵魂，必须每小时换人。邪眼抑制眼罩摘下时会有血泪。停战纹章激活时诺玛德部队会单膝跪地，非常羞辱，也非常有效。死亡账本朗诵到一半可能被黑打断：「别念了，我记得那些名字。」——这句话让账本从武器变成哀悼。

**【终战与乐园衔接】**
五阶契约者在终战后可带走：终战权限残章、一份半公开的对魔忍名录、一枚未解的黑之棋子。不可带走：始祖血、完整日蚀仪、阿尔萨尔本体。世界状态定格为「残胜」，供后续轮回或二周目。

## 阶位切入点

> 阶位↔原作：一～四阶略；**五阶≈终战精锐编制**；六阶≈干部；七阶～超阶≈黑／阿尔萨尔。顶点情报优先。

**五阶（终战精锐 · 日蚀夜三线）**

切入身份／时点：以「终战编制特别战术官」切入，锚定**终战令下达后六小时内**，三线尚未出发。

初始事件：司令部警报三色同亮。浅葱点名契约者：选A黄泉原、B东京王国或C五车深层。时子发终战套装与死亡备忘升级版。雪风在天台试枪，说：「这次没有B级胜利。」小太郎把死亡账本拍在桌上：「看完再选线。」

开场白建议：「五车的地下比任何一夜都亮，灯却冷得像手术室。浅葱把三枚任务章推到你面前：黄泉原的门、东京王国的工厂、正下方的裂隙残核。时子说：『五阶的终战很简单——你选一条线，就对那条线的死人负责。』远处雷声滚动，分不清是雪风还是未来。」

关键NPC立场：**井河浅葱**——要结果；**风魔小太郎**——要可执行计划；**风魔时子**——要账目；**水城雪风**——要一起冲；**八津紫**——要正面；**桐生佐马斗**——要样本与活口；**英格丽德**——要交易；**埃德温·黑**——要新契约；**容器**——要被爱或被杀。

主线钩子／支线：主线＝日蚀仪进度压制；支线①夺停战纹章；支线②唤醒轻度同化者；支线③保护学园地上残存平民。

危险度／规避：贴近顶点。规避与黑长时间单挑、对容器感情用事超过战术时限。超阶巅峰：存在·情报优先／条件性胜利。

任务方向／奖励：压日蚀仪、重创或逼退顶点、保全核心同伴。奖励＝终战勋章（权限）、抑制眼罩、重建学园的决策权、人脉。禁止无代价击杀黑本体的奖励。

补充：终战五日压缩为「一夜＋余波三日」。第一夜三线；第二日清场；第三日审判与葬礼；第四日决定公开与否。契约者若在第一夜死亡，可转「残响视角」旁观，但不可再获S。与黑谈判的桌边必须有浅葱或英格丽德在场，否则视为自杀事件。

终战开局六小时清单（五阶）：确认三色警报；领取终战套装；阅读死亡账本最新页；与小太郎同步三线优先级；选择A/B/C；与时子确认通讯死码；与桐生确认终极剂归属协议；与雪风约定「若我被同化先打膝盖」。任何一项跳过，都会在日蚀仪进度上+0.5格隐藏惩罚。

谈判桌规则：黑只接受「血与门」为筹码；英格丽德只接受「面子与停战」；浅葱只接受「人活」。契约者若提出「乐园公证」，会被全场视为精神异常并解除武装。

撤退优先级终战版：普通人＞未污染同伴＞情报＞敌首＞ equip。与学园篇不同，终战允许遗弃装备，不允许遗弃名字——阵亡者必须被账本记录。

终战字段加细：选线后立即获得该线专属NPC绑定（A英格丽德/紫，B雪风/凛子，C小太郎/桐生）。换线需时子批准并扣日蚀仪进度。开场白可按线替换。危险度贴近顶点时，每回合强制一次「精神检定或弹药检定」。任务奖励分「权限／人脉／残章」三类，禁止神杀级武器。
