# 谍影重重
<!--meta lib=主库 tiers=一、二-->

## 剧情

**【作品来源】**
美国环球影业动作惊悚系列《谍影重重》（英语：*Bourne* film franchise；台译《神鬼认证》、港译《叛谍追击》），概念源自罗勃·陆德伦（Robert Ludlum）1980–1990 年杰森·伯恩三部曲小说，电影剧本与小说情节大幅改写，仅保留失忆杀手、Treadstone 等核心意象。正传五部：道格·李曼执导《谍影重重》（*The Bourne Identity*，2002）、保罗·格林格拉斯执导《谍影重重2》（*The Bourne Supremacy*，2004）与《谍影重重3》（*The Bourne Ultimatum*，2007）、托尼·吉尔罗伊执导外传《谍影重重4：伯恩的遗产》（*The Bourne Legacy*，2012）、格林格拉斯再执导《谍影重重5》（*Jason Bourne*，2016）。另有 2019 年美剧《绊脚石》（*Treadstone*）探索计划起源。文风：后九一一写实间谍片——手持摄影、快速剪辑、近身格斗与城市追车；核心命题是「被国家制造的武器能否夺回自己的名字」。本档案以**电影宇宙正传+外传**为主线；陆德伦原著「卡洛斯」线、续写小说线不并入。搜笔趣阁等小说站未以影视改编本收录完整对照文本，资料以维基百科、公开剧情与系列设定页核对。

**【世界定位】**
2000 年代至 2010 年代中期的当代地球：苏黎世银行保险柜、巴黎安全屋、柏林地铁、莫斯科雪夜、纽约东河、拉斯维加斯赌场——都在同一套卫星监控与黑行动电话链下运转。主角杰森·伯恩（Jason Bourne，本名戴维·韦布 David Webb）是中央情报局（CIA）黑计划「绊脚石」（Treadstone）训练出的顶级刺客，任务失败后失忆漂流，在追杀与自救中撕开 Treadstone / 黑荆棘（Blackbriar）等非法暗杀体系。一句话：这是一个用护照、枪火与记忆碎片，把「国家秘密杀手」逼回「人」的写实谍战世界。

**【世界观 · 力量体系】**
本世界无超自然修炼、无魔法、无公开超能力。战力落在**凡人特工网络**：近身格斗（菲律宾武术/拳击/环境武器）、枪械、驾驶、反侦察、假身份、卫星与通信截听、组织层级与政治掩护。死亡为硬伤；「失忆」是心理/创伤性解离，非复活机制。成长逻辑=训练植入的肌肉记忆与战术直觉 + 情报优势 + 道德抉择（是否扣扳机）。执法与黑行动共用同一物理世界：一发步枪、一场车祸、一次跳楼都能终结「伯恩神话」。

力量层级（按实际破坏力与组织位阶，宁低勿高）：
1. **平民 / 记者 / 游客 / 银行职员**：徒手与口舌，无武装战力；一枪可致命。≈一阶初期（T0）。
2. **地方警察 / 宪兵 / 领事馆保安 / 普通探员**：手枪、程序权限、人数优势；能压制平民，难抗顶级刺客单挑。≈一阶中期。
3. **街头打手 / 雇佣枪手 / 低级资产**：手枪、跟踪、突袭；对普通人构成威胁，仍惧专业反制。≈一阶后期。
4. **Treadstone / Blackbriar 训练杀手**（如「教授」The Professor、卡斯特尔 Castel、帕兹 Paz、德什·布克萨尼 Desh Bouksani、基里尔 Kirill）：专业近战与枪法、跨国执行、灭口与善后；可在车站、公寓、公路上单挑小队。≈二阶初期～中期。
5. **巅峰资产 / 计划核心执行者**（杰森·伯恩、亚伦·克罗斯 Aaron Cross、拉斯维加斯「资产」The Asset）：多国语言、即时威胁评估、环境杀、车辆与屋顶机动；个人近身战力为本世界天花板，仍惧重火力包围与卫星锁定。破坏力=据点/多条人命/城市追逐级，**未达夷平街区的超凡档**。≈二阶后期。
6. **增强型计划（Outcome / LARX）**：国防部「成果」（Outcome）以「化学片」强化体能与认知；LARX 为后续更冷酷资产线。仍属**药物/训练强化的凡人**，非魔法，不升超阶。
7. **情报官僚层**（亚历山大·康克林、沃德·阿博特、诺亚·沃森、罗伯特·杜威等）：个人枪法未必顶尖，威胁来自授权、监控网与「一通电话派杀手」。
8. **顶点警示**：伯恩、克罗斯、顶级杀手与 CIA 黑行动火力仍属凡人枪火与组织；**严禁**写成超阶、领域或核武级单兵。

**乐园阶位映射**（对照《阶位战力图鉴》，宁低勿高）：  
街头平民～入门执法/雇佣枪手≈**一阶**；职业杀手/黑计划资产/伯恩级近战与跨国逃杀≈**二阶**；世界顶点（杰森·伯恩巅峰、Blackbriar 指挥网、杜威时代全监控）=**二阶后期·当代黑行动天花板**。无三阶及以上真实战力。低阶切入一律「规避顶点·情报与生存优先」；伯恩的威胁来自训练、情报与枪火，**禁止**写成超凡存在或「因其缺席而自动无威胁」的伪设定。

**【地理 · 舞台】**
- **地中海 / 马赛附近海域**：开场渔船捞起两枪穿背的漂流者；激光植入物指向苏黎世保险箱。
- **苏黎世**：银行保险柜（多国护照、现金、手枪）、公园夜宿与警察冲突、美国领事馆追逐。
- **巴黎**：玛丽·克罗伊茨公寓、地铁与小巷、约翰·迈克尔·肯恩（John Michael Kane）假身份、停尸房、乡间农场（玛丽异母兄埃蒙 Eamon 处）、Treadstone 安全屋攀爬与终局交火。
- **希腊米科诺斯岛**：玛丽出租摩托艇的落脚点；第一部重逢与短暂喘息。
- **印度果阿**：伯恩与玛丽隐居；基里尔刺杀导致玛丽身亡，第二部复仇引擎启动。
- **那不勒斯 / 慕尼黑 / 柏林 / 莫斯科**：第二部主舞台——领事馆复制 SIM、雅尔达（Jarda）公寓煤气爆炸、亚历山大广场绑架妮基、涅斯基酒店记忆回收、莫斯科出租车雪地追车、向伊莲娜·涅斯基（Irena Neski）道歉。
- **伦敦滑铁卢车站 / 马德里 / 丹吉尔 / 纽约**：第三部——记者西蒙·罗斯（Simon Ross）被狙、丹尼尔斯站长线、屋顶与街道追逃、沃森办公室窃取黑荆棘文件、东河一跃。
- **阿拉斯加 / 马尼拉**：第四部 Outcome 训练与「病毒化」脱药、LARX 追杀、货船逃离菲律宾。
- **雅典 / 雷克雅未克 / 柏林 / 伦敦 / 拉斯维加斯**：第五部——抗议中的会合、黑客泄密、Deep Dream 社交平台监控阴谋、赌场与下水道终局。
- **华盛顿兰利 / 纽约深掩局**：Treadstone 与 Blackbriar 的神经中枢；沃森指挥室多屏、兰迪调查组、委员会听证室把「合法」与「非法」隔一张门。
- **空间气质**：旧货店伪装、安全屋白板、指挥室多屏监控、手持镜头下的真实街景——任何「干净身份」都紧贴着「一枪灭口」的同一座城。巴黎小巷与莫斯科雪道同样致命，差别只在口音与车牌。

**【世界剧情线】**

**第一部《谍影重重》（约 2002 · 身份 · Treadstone 崩裂）**  
地中海渔船救起失忆美国人：背有两处枪伤，臀下植入微型激光，显示苏黎世银行箱号。意大利船长詹卡洛（Giancarlo）给钱让他上岸。男子苏醒后多语流利、格斗本能完好，却不知自己是谁。与此同时，兰利：CIA 副局长沃德·阿博特获知暗杀流亡非洲独裁者尼匡纳·旺博西（Nykwana Wombosi）失败，要求康克林补救。男子打开保险柜，取出多国货币、多本护照与手枪，采用美国护照名「杰森·伯恩」却留下枪——这一细节标志他尚不想以杀手自居；银行职员报警触发 Treadstone。行动主管亚历山大·康克林启动三名杀手（卡斯特尔、曼海姆 Manheim、「教授」等）追杀。伯恩在苏黎世公园夜宿时被警察盘查，本能制伏二人后自己也震惊于身体的记忆。他逃入美国领事馆，付钱两万美元给德国女子玛丽·海伦娜·克罗伊茨（Marie Helena Kreutz）送他去巴黎护照地址。公寓中卡斯特尔从阳台闯入，被制服后咬破氰化物跳楼自杀；玛丽在其包中发现印有二人照片的通缉令，房东老太太已死，她呕吐后仍选择同行。康克林为掩盖旺博西案，在停尸房伪造「约翰·迈克尔·肯恩」尸体；旺博西不信，威胁把 CIA 捅给媒体，遭「教授」灭口。伯恩以肯恩身份调查游艇资料，确认自己即刺客：潜入时见旺博西孩子熟睡而拒绝扣扳机，撤离时中两枪坠入海中——这是他第一次「作为人」的选择。乡间农场，「教授」先杀狗再设伏；伯恩用猎枪重创对方，临死前对方承认同属 Treadstone，并说出那句主题回响：「看看我们，看看他们让你付出什么。」伯恩送走玛丽与埃蒙一家，用教授手机约见康克林，反跟踪至巴黎安全屋，持枪宣告退出计划并警告勿再追踪。阿博特下令关闭 Treadstone，并派曼海姆枪杀康克林灭口。委员会听证上，阿博特把 Treadstone 说成「已退役的理论推演」，同时把讨论引向新项目代号「黑荆棘」。伯恩在米科诺斯岛租船摊找到玛丽，拥抱重逢。名场面：保险柜身份拼图、领事馆逃生、巴黎小车追逐、农场猎枪对决、安全屋徒手攀墙。

**第二部《谍影重重2》（约 2004 · 至尊 · 玛丽之死与莫斯科道歉）**  
伯恩与玛丽在果阿隐居约两年；他仍用笔记本记录闪回中的柏林酒店与枪声。柏林：副局长帕梅拉·兰迪（Pamela Landy）的探员用三百万美元向俄方线人以物易物「涅斯基文件」（七年前两千万美元失窃相关），遭俄罗斯联邦安全局腐败资产基里尔（Kirill）灭口并栽赃伯恩指纹。兰迪追查 Treadstone 与已故康克林，怀疑伯恩即涅斯基案凶手；阿博特表面配合，实则掩饰自己与寡头尤里·格雷特科夫（Yuri Gretkov）的盗款合谋——正是这笔钱让他必须让「伯恩」当替罪羊。基里尔在果阿追杀，摩托艇追逐中玛丽中弹身亡（座位调换后仍未能幸免）；伯恩抱尸上岸，复仇与自责同时点燃。他认定 CIA 再启猎杀，经那不勒斯故意暴露身份，复制外交安全局特工 SIM 卡，截听得知兰迪与栽赃。慕尼黑：他造访最后一名 Treadstone 杀手雅尔达，得知计划已关，近战后勒死对方并以煤气爆炸毁证。柏林：他从屋顶打电话给兰迪（「我能看见你」），在亚历山大广场绑架妮基·帕森斯（Nicky Parsons）问讯，得知阿博特曾是康克林上司后放走她。酒店闪回完整回收：伯恩曾奉康克林之命枪杀俄国政客弗拉基米尔·涅斯基（Vladimir Neski），其妻闯入后亦被枪杀并伪装成谋杀—自杀。丹尼·佐恩（Danny Zorn）向阿博特指出栽赃破绽，反被阿博特灭口。伯恩闯入阿博特酒店房间，录下其与格雷特科夫通话；阿博特承认盗款、派基里尔取文件并灭口伯恩。伯恩拒绝处决他（「玛丽不会要我这么做」），把枪放在桌上离开，另将录音留给兰迪；阿博特在兰迪对质后自杀。伯恩赴莫斯科向涅斯基之女伊莲娜坦白并道歉——这是全片道德高潮，不是胜利宣言。基里尔雪地出租车追车重伤，格雷特科夫被捕。纽约：兰迪电话告知伯恩原名戴维·韦布与出生信息；伯恩从对面楼顶说「你看起来很累，休息吧」后消失。主题从「我是谁」转为「我做过什么、如何偿还」。

**第三部《谍影重重3》（约 2007 · 最后通牒 · 黑荆棘曝光）**  
时间线与第二部莫斯科收尾部分重叠并向前推进：开场即莫斯科枪伤逃亡与更多改造闪回。兰迪把阿博特录音呈交局长埃兹拉·克莱默（Ezra Kramer）。英国《卫报》记者西蒙·罗斯调查伯恩与「黑荆棘」——Treadstone 升级版、可绕过部分许可对目标（含美国公民）动手。ECHELON 截获罗斯通话；黑荆棘主管诺亚·沃森（Noah Vosen）下令灭口，兰迪被迫协助却全程反对滥杀。伯恩先在欧洲通知玛丽之弟马丁·克罗伊茨（Martin Kreutz）姐姐死讯，再赴伦敦滑铁卢车站与罗斯接头：他用人群、出口与反监视走位助记者躲避，罗斯仍因恐慌跑出掩护被帕兹狙杀。笔记指向马德里站长尼尔·丹尼尔斯（Neal Daniels）；伯恩突入站部，与妮基重逢——她已调离 Treadstone 后勤，却选择帮他逃出沃森小队，暗示失忆前的情感线索。丹尼尔斯逃往丹吉尔；沃森派德什·布克萨尼杀丹尼尔斯并连坐妮基与伯恩。伯恩未能阻止汽车炸弹/狙杀丹尼尔斯，却在屋顶与街巷追杀中勒死德什，放妮基逃亡。伯恩从丹尼尔斯公文包找到纽约深掩局地址，利用兰迪「4-15-71」暗语（实为东 71 街地址编码，与第二部结尾电话场景无缝衔接并超越）与沃森对峙，盗取保险柜黑荆棘文件交给兰迪。追车后伯恩放过帕兹，质问「你知道他们让你付出什么吗」（呼应「教授」遗言）。医院楼：他面对行为改造负责人阿尔伯特·赫希博士（Dr. Albert Hirsch），确认自己曾自愿加入 Treadstone，并杀死陌生人证明忠诚——记忆闭环完成。沃森开枪，伯恩跃入东河。三日后新闻：黑荆棘曝光，赫希与沃森被捕，克莱默遭调查；妮基微笑于「尸体未找到」的报道——伯恩生还，游入黑暗。三部曲在此完成「身份—罪责—体系」闭环：名字找回，计划被公开，人仍选择消失。

**第四部《谍影重重4：伯恩的遗产》（约 2012 · 外传 · Outcome 清洗）**  
伯恩本人未出场，但其在第三部的曝光引发参议院与 FBI 调查。退役空军上校埃里克·拜尔（Eric Byer）负责监管从中孵化 Treadstone/Blackbriar 的「贝塔」系计划，决定终结国防部「成果」（Outcome）并灭口所有强化特工。亚伦·克罗斯（化名肯尼斯·J·基特索姆 Kenneth J. Kitsom）在阿拉斯加训练中幸存，依赖蓝/绿化学片维持体能与认知。他找到药剂师玛塔·希尔灵博士（Dr. Marta Shearing）：实验室同事遭清洗后她成为最后链接。希尔灵揭示认知强化可「病毒化」永久固化；克罗斯已在八个月前被永久固化体能强化，却仍需认知片——停药将跌回极低智商。二人赴马尼拉工厂完成危险固化，遭警方与 LARX-3 资产追杀，最终以货船乘客身份逃离菲律宾。本线证明：伯恩撕开的不是单一计划，而是一整棵黑行动树；每一片「遗产」都在流血。

**第五部《谍影重重5》（约 2016 · 杰森·伯恩 · 父亲与 Deep Dream）**
