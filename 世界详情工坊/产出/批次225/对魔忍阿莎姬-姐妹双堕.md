# 对魔忍阿莎姬-姐妹双堕
<!--meta lib=主库 tiers=四-->

## 剧情

**【作品来源】**
对魔忍阿莎姬系列（Lilith）中「井河姐妹同时落入敌手」的任务线口径。**本档案专线：姐妹双堕——井河阿莎姬与井河樱被同一条俘获链／竞技场／改造课绑定；焦点亲缘人质、双目标救援伦理与编制切割；只写任务与制度，不写色情细节。** 与「最强陷落」区分：彼以阿莎姬单点名号为主；本线＝**姐救妹／妹救姐的双死结**。

**【世界定位】**
最强对魔忍与见习妹同时成为诺玛德—胧联合项目的「姐妹套装」标本。敌方战术核心：让一方看见另一方屈服以完成精神崩坏。契约者四阶任务常在「先救谁」的伦理定时器上。


**【世界观 · 力量体系】**
现代日本（东京）＋魔界渗透：对魔忍以忍者血脉与忍法对抗魔族与人魔犯罪组织。无修仙境界，按个人武勇、忍法完成度、编制与改造度分层。
1. 普通警员、黑帮打手、见习对魔忍≈一阶：巷战级。
2. 正式对魔忍、企业安保精英、中级妖魔≈二阶：可清据点。
3. 精锐对魔忍、妖魔干部、改造体≈三阶：毁楼级交锋。
4. **井河阿莎姬级「最强对魔忍」／胧／诺玛德干部／混沌竞技场冠军级**≈**四阶**：可屠小队到掀平建筑群；本库覆盖。
5. 吸血鬼始祖埃德温·布莱克等魔界上位≈更高/超阶向：情报优先。
乐园阶位映射：见习≈一阶；正式对魔忍≈二阶；精锐≈三阶；最强对魔忍/胧/诺玛德高层≈**四阶**；布莱克等≈超阶向。死亡：凡人一死即终；高阶妖魔可有契约再生，代价极高。顶点条件性胜利，严禁削弱归零。


**【地理 · 舞台】**
井河宅；对魔忍总部隔离审讯室；姐妹分押的双监区（可视玻璃相连）；混沌竞技场「双人场」；诺玛德姐妹改造舱并联室；东京水道救援廊。

**【世界剧情线】**
① **血脉诱饵**：樱潜入／被诱出，阿莎姬无法只当任务机器。  
② **双捕**：胧刻印链＋假令，使姐弟同落网；监区玻璃让双方可见。  
③ **双人场**：竞技场推出「最强与见习」组合赛，观众要的是伦理崩溃而非单纯流血。  
④ **并联改造**：科学班用姐数据校准妹、用妹哭声校准姐——互相为触媒。  
⑤ **编制反应**：总部讨论「弃妹保姐」与「双灭口」；两案都可能盖章。  
⑥ **终局**：双救／救一弃一／双工具化；无完美。

**【主要人物】**
- **井河阿莎姬**｜姐｜最强｜护妹破绽被计算。  
- **井河樱**｜妹｜见习｜想变强却成饵。  
- **胧**｜要姐妹同台。  
- **泽木恭介**｜伦理第二炸弹（可在本线作胁或已失）。  
- **埃德温·布莱克**｜要双份数据。  
- **内鬼课长**｜推动弃子案。  
- **竞技场经纪人**｜双人场溢价。  
- **科学班双人舱技师**｜并联关键。  
- **同级救援队队长**｜执行弃或救。  
- **信息屋**｜卖双监区地图。  
- **水道向导孩**｜知玻璃墙检修口。  
- **诺玛德法务**｜「自愿改造」伪文件。

**【势力图谱】**对魔忍两派；诺玛德项目组；胧；竞技场；民间「救樱会」小组织。

**【贵重物品】**双监区钥匙卡；并联舱紧急分离栓；姐妹忍刀一对；伪自愿书；玻璃墙检修密码；弃子盖章文件。

**【隐藏剧情 · 伏笔】**分离栓只能用一次；伪自愿书有樱的练习签名被盗用；布莱克要的是「血脉对」而非个体。

**【大事记时间线】**诱妹 → 双捕 → 双人场 → 并联舱 → 弃子令草案 → 终局。

**【叙事基调 · 雷区】**亲缘伦理极限。忌把樱写成无脑拖油瓶；忌无代价双全。最早锚点：樱失踪夜或双人场彩排。


**【姐妹线·玻璃墙声学】**监区玻璃传声单向可调；敌方常开「只让姐听妹」模式。破坏调音盒可恢复双向。

**【姐妹线·弃子令格式】**需三名课长联署；缺一则无效。内鬼常伪造第三署。

**【姐妹线·双人场规则】**一方投降则另一方「加时受罚」；故阿莎姬更不愿投降。

**【姐妹线·并联触媒】**姐的神经数据灌妹会致魔力暴走；分离栓在暴走前90秒有效。

**【姐妹线·井河宅暗号】**旧门牌后藏「先救樱」遗嘱式纸条——阿莎姬自己写的，与她口头「任务优先」矛盾。

**【姐妹线·救樱会】**民间狂热粉丝组织，易被胧渗透；真伪接头用井河家剑印。

**【姐妹线·水道检修口】**玻璃墙下水管可通，但每夜有妖魔承包巡游。

**【姐妹线·伪自愿书笔迹】**樱练字「河」末笔上挑；盗用版下压。

**【姐妹线·布莱克双份】**他要对照数据证明「血脉可复制最强」；毁一份服务器不够。

**【姐妹线·救援队队长】**曾受阿莎姬恩，却收了弃子令——可策反点是「令缺第三署」。

**【姐妹线·条件性胜利】**双人离场＋分离栓未耗尽＋弃子令作废，占二即胜。

**【姐妹线·忍刀一对】**姐刀开锋、妹刀未开；同台时未开之刀可作信号弹材料。

**【姐妹线·经纪人溢价表】**姐妹同台票价公式公开贴后台——可作舆论爆料。

**【姐妹线·法务伪文件】**「自愿」条款引用过期法律编号；律师型契约者可当场撕。

**【姐妹线·樱潜力】**本线暗示樱忍法与姐不同系；过早逼她开眼会爆。

**【姐妹线·恭介变量】**若恭介仍在，双人质变三人伦理；敌方更稳。

**【姐妹线·终局舆论】**只救姐会被民间骂；只救妹会被总部骂；双救则诺玛德全面追杀。

**【姐妹线·检修密码】**密码是姐妹生日逆序拼接；樱自己可能不知道。
