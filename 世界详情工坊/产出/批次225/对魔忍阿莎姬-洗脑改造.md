# 对魔忍阿莎姬-洗脑改造
<!--meta lib=主库 tiers=四-->

## 剧情

**【作品来源】**
对魔忍阿莎姬系列中「洗脑·肉体／神经改造」任务线口径（含完全版与后续作品常见的改造 captives 结构）。**本档案专线：洗脑改造——井河阿莎姬在俘获后进入神经重塑、记忆覆写与忠诚重定向的工厂流程；焦点科学班课表、洗脑阶段可逆窗口、身份夺回战；只写制度与任务，不写色情细节。** 与「最强陷落」区分：彼重俘获政治；本线重**改造流水线与逆洗脑**。

**【世界定位】**
阿莎姬已失去自由后的「第二战场」：不是越狱那么简单，而是与自己被改写的记忆作战。诺玛德科学班目标＝把最强对魔忍变成可控兵器；胧要的是人格羞辱式服从。


**【世界观 · 力量体系】**
现代日本（东京）＋魔界渗透：对魔忍以忍者血脉与忍法对抗魔族与人魔犯罪组织。无修仙境界，按个人武勇、忍法完成度、编制与改造度分层。
1. 普通警员、黑帮打手、见习对魔忍≈一阶：巷战级。
2. 正式对魔忍、企业安保精英、中级妖魔≈二阶：可清据点。
3. 精锐对魔忍、妖魔干部、改造体≈三阶：毁楼级交锋。
4. **井河阿莎姬级「最强对魔忍」／胧／诺玛德干部／混沌竞技场冠军级**≈**四阶**：可屠小队到掀平建筑群；本库覆盖。
5. 吸血鬼始祖埃德温·布莱克等魔界上位≈更高/超阶向：情报优先。
乐园阶位映射：见习≈一阶；正式对魔忍≈二阶；精锐≈三阶；最强对魔忍/胧/诺玛德高层≈**四阶**；布莱克等≈超阶向。死亡：凡人一死即终；高阶妖魔可有契约再生，代价极高。顶点条件性胜利，严禁削弱归零。


**【地理 · 舞台】**
诺玛德地下改造翼；洗脑椅与神经液槽；记忆备份库；忠诚测试场；对魔忍总部「叛忍对策室」；东京地下冷库式安全屋（逆洗脑用）。

**【世界剧情线】**
① **收容**：俘获后第一周外改造，记忆仍在。  
② **神经重塑**：液槽阶段，感度与痛觉阈值被改（制度描述到此为止）。  
③ **记忆覆写**：植入「对魔忍背叛人民」伪史；泽木恭介面孔被替换。  
④ **忠诚测试**：令其执行对旧同僚的模拟击杀；通过则编入诺玛德编制。  
⑤ **逆洗脑战**：救援队需在备份库取出「原阿莎姬」镜像，与现身体同步风险极高。  
⑥ **终局**：完全工具／镜像夺回人格／人格分裂双轨（最难）。

**【主要人物】**
- **井河阿莎姬**（改造中／后）｜记忆与忠波动。  
- **科学班主刀**｜课表制定者。  
- **胧**｜要服从表演。  
- **布莱克**｜要完美兵器。  
- **井河樱**｜持有部分童年记忆可作密钥。  
- **备份库管理员**｜可贿。  
- **叛忍对策室军官**｜倾向灭口「不干净的回收」。  
- **逆洗脑医师**｜民间／叛逃。  
- **旧任务搭档**｜测试场假想敌。  
- **泽木恭介**（真或伪记忆）｜锚点。  
- **信息屋**｜卖课表。  
- **诺玛德政委**｜编制接收人。

**【势力图谱】**科学班；胧演出系；布莱克军；对策室灭口派；民间逆洗脑网。

**【贵重物品】**记忆镜像碟；神经液槽紧急排空键；伪史剧本原件；童年密钥（樱的回忆物）；忠诚项圈主控；逆洗脑药剂（不稳定）。

**【隐藏剧情 · 伏笔】**镜像碟有三份，一份在布莱克私库；伪史引用真实丑闻作底；对策室军官收诺玛德钱。

**【大事记时间线】**收容 → 外改造 → 液槽 → 覆写 → 忠诚测试 → 逆洗脑／灭口令 → 终局。

**【叙事基调 · 雷区】**身份恐怖＋冷战救人。忌把洗脑写成一巴掌醒；忌无代价完美复原。最早锚点：液槽前夜或忠诚测试彩排。


**【改造线·阶段时钟】**外改造7日可逆；液槽开始后每24小时可逆率降半；覆写完成后需镜像碟。

**【改造线·伪史写法】**剧本混入阿莎姬真实任务黑料（误伤、弃子令执行）以增可信；全假易被识破。

**【改造线·樱密钥】**「姐第一次教我握刀」的触感可作逆洗脑口令；无樱则用道场木刀。

**【改造线·排空键】**液槽排空会损数据也救人；科学班主刀口袋常备。

**【改造线·忠诚测试脚本】**目标是旧搭档模拟体；若契约者替换成假体可减罪疚永久化。

**【改造线·对策室灭口】**回收标准「人格完整度＞80%」；低于则杀。镜像同步风险会压低完整度读数。

**【改造线·三镜像】**A公库、B私库、C胧保险柜；只取A不够。

**【改造线·项圈主控】**主控在政委而非主刀；杀主刀不停项圈。

**【改造线·药剂不稳】**逆洗脑药可能恢复记忆但损杀阵华同步；阿莎姬或拒药。

**【改造线·条件性胜利】**取至少两镜像＋排空液槽＋停灭口令，占二即胜。

**【改造线·布莱克私库】**入私库＝超阶风险；可用交易换拷贝而非硬闯。

**【改造线·旧搭档】**愿当假想敌演员以换阿莎姬不记恨；需预签谅解。

**【改造线·信息屋课表】**课表有「覆写日」红字；那天备份库戒备最低（全员观礼）。

**【改造线·政委接收仪式】**公开给阿莎姬新代号；打断仪式可舆论反噬诺玛德。

**【改造线·人格双轨】**最难结局：任务人格与井河人格并存，需长期锚定物。

**【改造线·恭介锚】**真恭介声音可破伪史一层；伪恭介录音会加固伪史。

**【改造线·主刀性格】**视阿莎姬为杰作而非人；可利用其「怕作品被毁」逼停灭口。

**【改造线·终局读数】**完整度仪表可被技术员做假；契约者需独立验。
