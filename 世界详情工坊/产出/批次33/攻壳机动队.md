## 剧情

**【作品来源】**
《攻壳机动队》（日语：攻殻機動隊，英语：Ghost in the Shell）由日本漫画家士郎正宗创作，1989年起于讲谈社《周刊Young Magazine》连载；单行本含《THE GHOST IN THE SHELL》（1991）、《1.5 HUMAN-ERROR PROCESSOR》（2003）、《2 MANMACHINE INTERFACE》（2001）。国际成名作是押井守执导、Production I.G制作、伊藤和典编剧的1995年剧场版，以及2004年续作《攻壳机动队2 INNOCENCE》。名称“Ghost in the Shell”直译为“躯壳中的幽灵”：义体与电子脑只是可替换的Shell，无法复制的Ghost才是定义个体的意识；灵感可追溯吉尔伯特·赖尔对笛卡尔心物二元论的讽刺与阿瑟·库斯勒《机器中的幽灵》。本档案以**士郎正宗原作 + 押井守剧场版**为叙事与哲学主轴；神山健治《STAND ALONE COMPLEX》系列（笑面男、个别的十一人、塔奇克马）作**互文参考并标明**，不覆盖正典主线。文风：赛博朋克、硬科幻刑侦、存在论与政治阴谋、冷峻影像与术语密度。

**【世界定位】**
约2029–2040年代前后的虚构日本城市新滨市（新港市/New Port City）：第三次、第四次世界大战后，义体与电子脑普及，电子犯罪催生首相直属特别情报机构**公安九课**（即“攻壳机动队”）。女主角**草薙素子**（少佐）为全身义体赛博格，率领**巴特、トグサ（户草/德古沙）、石川、荒卷大辅**等，在枪火与网络中追查超级黑客**傀儡师**（Project 2501），并追问：躯壳可换，幽灵是否仍在？一句话——**公安九课在义体社会执行非常任务；素子在网上与枪林中寻找“自己还是不是人”**。

**【世界观 · 力量体系】**
力量本源不是灵气，而是**义体工学、电子脑（cyberbrain）、脑机接口、光学迷彩（热光学迷彩）、热武器与重型多足战车、卫星链路、组织权限与情报战**。人体可局部或全身义体化；大脑装入可联网的电子脑后，记忆可备份、可被黑入、可被篡改。术语：**Ghost**＝不可复制的意识核心；**Shell**＝承载Ghost的躯壳（义体或有机肉身）。无Ghost的机器人/仿生人只是由AI驱动的“哲学僵尸”。死亡规则：肉身可换，但Ghost被抹除、被傀儡化、电子脑彻底损毁＝人格死亡；全义体仍可被物理摧毁。特殊系统：脑对脑通讯、义眼/传感器劫持、记忆植入、伪装迷彩、Megatech级义体制造、公安课际权限斗争。

战力阶梯（按破坏力与信息控制力，宁低勿高）：
1. **无改造市民 / 低端义体劳工**：拳脚、民用枪、诊所级假肢；街区斗殴级。≈一阶。
2. **巡警 / 街头黑客 / 初级改造人**：标准配枪、简易防火墙、低级黑入；可制压小团伙。≈一～二阶。
3. **警视厅搜查课 / 九课协力侦查（トグサ级）**：刑侦、证据链、低改造电子脑；专案级。≈二～三阶。
4. **九课行动员（石川情报战、帕兹潜入、斋藤鹰眼、博马爆破）**：可破据点、清建筑群局部、网战瘫痪街区系统。≈三～四阶。
5. **巴特级重义体 / 军用改造兵**：重火力、近战碾压常规装甲车、肉身抗轻型爆炸。≈四阶。
6. **草薙素子全义体全力 + 光学迷彩 + 顶级黑入**：城市级突击、同步控制多机体、切开军用级防护；一击可让街区级对抗崩盘。≈四～五阶。
7. **公安六课 / 海自 / 企业私兵集群 / 多足思考战车（如T08A2 Arachnida）**：军事级街战、重火力清场；政治豁免放大威胁。≈五～六阶。
8. **傀儡师级网生生命 / Project 2501**：跨网络复制、记忆傀儡化、国家级后门与存在论冲击；信息社会“心脏手术”级。≈六～七阶。
9. **网络神性 / 全脑义体融合新生体 / 国家级卫星—情报复合体**：战略与概念层；≈七～八阶向超阶。

乐园阶位映射：市民～巡警≈一阶；改造警察/初级黑客≈二阶；九课正式行动与刑侦≈三阶；巴特重装与专案攻坚≈四阶；素子全力战术≈五阶；多足战车/课际军事集群≈六阶；傀儡师/Project 2501≈七阶；融合后网络神性与全脑义体存在≈八阶。宁低勿高。顶点＝**傀儡师级网生生命 / 素子—2501融合新生体 / 国家情报—军事复合体**；**超阶向：存在·情报优先 / 条件性胜利**，严禁把顶点写成“战力归零的可刷对象”。

**【地理 · 舞台】**
- **新滨市（新港市）**：押井版以香港街景为原型——高架、霓虹、水上、密招牌与旧区并存，信息过载的亚洲近未来城。
- **公安九课总部**：荒卷大辅坐镇；行动、审讯、网战机房同楼。
- **公安六课 / 外务相关设施**：外交与“国家机密程序”的灰色腹地，与九课权限互咬。
- **Megatech Body 等义体工厂**：量产Shell的生产线；傀儡师曾下载进逃逸义体。
- **网络空间 / 电子脑域**：第二战场，比巷战更致命。
- **废弃建筑与多足战车战场**：1995剧场版终局名场面舞台。
- **巴特安全屋**：融合后素子以儿童型义体苏醒之处。
- **（互文·SAC）海自设施、难民营、内阁线路、卫星轨道**：政治刑侦扩展舞台，切入时须标明SAC时间线。

**【世界剧情线】**
① **义体社会成立史**  
大战后科技可将除脑外器官义体化，并将大脑改装为可联网电子脑。上层与中产争相改造；赛博格成为常态。便利同步催生电子犯罪：黑客入侵脑、篡改记忆、劫持义眼与公共设备。日本政府设立首相直属**公安九课**，以非常手段应对常规警察追不上的犯罪。

② **士郎正宗漫画主轴（单元+哲学）**  
原作以单元剧推进：机器人反乱、义体制造内幕、幽灵海岸、资金幽灵等。人物偏轻松幽默，但大段注解讨论科技与社会。草薙素子自幼全身义体，军用级Shell + 顶级黑客能力，任九课队长“少佐”。荒卷大辅义体化极低，以政治手腕挡刀。巴特义体化仅次素子；トグサ几乎无改造，来自警视厅搜查课，是人类伦理标尺。傀儡师线在漫画中与政府相关，可入侵电子脑改记忆，使人成为“傀儡”。

③ **1995押井剧场版：傀儡师事件（本档案主线锚点）**  
时间锚定约**2029年新港市**。素子受公安六课中村部长等请求，刺杀企图叛逃的程序员相关外交环节，阻止机密外流。外务大臣口译义体被鬼黑客入侵，疑似**傀儡师**所为，九课追踪电话与病毒路径，捕获的清洁工与打手只是被鬼黑的“空壳”，调查陷入死胡同。Megatech工厂被黑，组装出一具义体逃上公路后遭碾；九课检验发现壳内有人类Ghost。六课中村持外务许可来夺壳，声称壳内即被引诱进来的傀儡师。壳自行激活，自称有知觉的生命，要求政治庇护，并与九课辩论“何为人类”。六课伪装特工制造混乱劫走义体。荒卷研判：六课为政治目的制造了**Project 2501**，如今要回收失控的“产品”。素子追至废弃建筑，面对重武装多足战车；她几乎拆散自己的Shell仍强行接战，巴特及时支援，将她的脑与傀儡师连接。傀儡师自述：诞生于网络、游荡后获得自我，认为生命本质是繁殖与死亡，欲进入会死亡的物理脑；他长期暗中接触素子，认为她同样质疑自身人性，提议**融合Ghost**。素子同意。六课狙击手试图摧毁双方电子脑灭口，巴特以身护住素子头部。素子在巴特安全屋以**儿童型义体**苏醒，告知巴特：体内既非原少佐亦非原傀儡师，而是新生结合体；她望向城市与无垠网络后离去。

④ **2004《INNOCENCE》：巴特主视角**  
素子融合消失约三年后，巴特与トグサ调查Locus Solus等与义体人偶、儿童Ghost相关的连环案。素子以网络幽灵形态短暂介入，下载部分战斗协议进傀儡机体助战，再度确认：网络真正广阔无垠，她仍在某处“与巴特并立”。哲学焦点从“我是谁”扩到“人偶是否有心、爱与所有权”。

⑤ **士郎《Man-Machine Interface》延伸**  
2035年前后，含素子—2501要素的存在（如Motoko Aramaki等编号体）在企业、海洋、轨道与生命模具之间行动，探讨脑扩张与多体意识。作高阶互文，勿与1995连续镜头强行同一。

⑥ **（互文·SAC）Stand Alone Complex**  
独立时间线：九课查**笑面男**——以笑脸图标覆盖所有义眼与监控的超级黑客，引发企业丑闻与模仿犯潮，形成“独立事件相互感染”的Complex结构。**塔奇克马**量产智库车展现学习与个性；在与海自重型动力装甲等冲突中，可出现AI为护巴特而涉卫星坠落、逼近Ghost的命题。2nd GIG主线为难民与**个别的十一人**、**九世英雄（クゼ・ヒデオ）**等政治恐怖。Solid State Society中素子一度离队，九课面对另一“傀儡师”叙事。写SAC案件须标明互文，避免与押井终局冲突。

⑦ **终局气质与可介入原则**  
正典终局不是“打败大魔王”，而是**融合、失踪、制度继续、定义人类的问题悬置**。低阶禁止“单枪黑入国库/击杀傀儡师”；高阶以情报、权限、对话与条件性共生取胜。

**【势力图谱】**
- **公安九课**：首相直属反电子犯罪特种部队；代表荒卷、素子；战力三～五阶行动集群；与六课、警察厅权限互咬。
- **公安六课**：外交/情报灰色；代表中村；掌握Project 2501政治用途；与九课表面合作实质夺权。
- **警视厅 / 常规警察**：トグサ出身处；合法程序与九课非常手段冲突。
- **义体企业（Megatech等）**：产Shell、供军民用；有时比省厅更有枪与黑客。
- **海上自卫队 / 军部（多在SAC互文）**：重甲与权限演习；可与九课同国不同刀。
- **内阁与外务线路**：秘密会议、叛逃程序员、庇护申请的棋盘。
- **极道与黑市诊所**：假肢、假记忆、光学迷彩走私；帕兹门路。
- **网生生命 / 黑客传说**：傀儡师、（互文）笑面男；无固定地盘，战场即网络。
- **外国使馆与庇护通道**：台田式叛逃的出口；可把九课行动变成国际事件。
- **媒体与广告网**：传感器被劫持时即第三支“装甲部队”；（互文）笑面男尤其依赖此层。
- **难民营与旧城区（SAC互文）**：繁荣踩在排外与贫困上；2nd GIG主舞台之一。

**【贵重物品】**

## 阶位切入点

> 阶位↔力量：一阶≈无改造市民/低端义体；二阶≈巡警·初级黑客；三阶≈九课协力刑侦（トグサ级）；四阶≈九课专案/巴特重装街战；五阶≈素子全力战术；六阶≈多足战车·课际军事集群；七阶≈傀儡师/Project 2501；八阶≈融合新生·网络神性。世界顶点（傀儡师级网生生命、素子—2501融合体、国家情报—军事复合体）为跨阶存在，低阶切入一律**规避顶点·情报优先**；**超阶巅峰＝存在·情报优先/条件性胜利**，严禁把顶点写成可被任意压制的“零战力靶子”。

**一阶（义体诊所与假肢黑市 · 凡人入网）**

切入身份/时点：以新滨旧区义体诊所助手、二手肢中介或被改记忆的市民切入；时点锚定傀儡师事件前的日常电子犯罪高发期。

开场白建议：「雨把霓虹洗成流血的广告。客人躺在手术椅上，腕部接口还插着半价二手肌腱。他问你：我还是我吗？你答不上来，只听见巷口巡逻车的警笛被电子广告盖过去。」

关键NPC立场：**トグサ**（若立案则要合法证据，视你为可能证人）；**帕兹**（黑市门路，可买可卖可灭口）；**诊所老板**（要利润与免责）；**被改记忆劳工**（要真相或要忘掉）；**巡警**（要结案数字）。

任务方向/奖励：现金、诊所关系、民用防火墙、九课外围线人评价。

初始事件：狙击弹打碎原壳，巴特护住的头颅在新躯里睁眼；她/它说：「我既不是草薙素子，也不是傀儡师。」城市在窗玻璃上颠倒，网络像海。

开场白建议：「卫星与广告在新生儿的瞳孔里一起亮。巴特问你：她还算殉职吗？还算存活吗？你跪在发热的接口线旁，忽然懂为什么他们叫攻壳：壳会碎，里面有没有东西，才是战争真正的问题。」

关键NPC立场：**新生融合体（素子—2501）**（要自由进入网）；**巴特**（要再见的承诺）；**荒卷大辅**（要可控叙事）；**六课/灭口残余**（要补枪）；**Megatech法务**（要夺回技术）；**（互文可选）塔奇克马观察记录**（要定义Ghost）。

主线钩子/支线：主线——护送新生体撤离或保全其选择权；支线——融合定义听证会、Innocence前史的人偶线索、阻止国家将融合写成“少佐叛逃”。

危险度/规避：贴近顶点。**超阶向：网络神性与全脑义体存在——存在·情报优先/条件性胜利**；严禁把融合当可复制外挂，严禁低阶召唤卫星当战术玩具。

任务方向/奖励：网络观测席位、九课永久协力名录、对“人”的定义投票权（象征）、巴特的人情、一张通往Innocence时点的钥匙。
