# 全球高武

## 剧情

**【作品来源】**
《全球高武》为网络作家老鹰吃小鸡所著都市高武/系统流长篇，起点中文网连载，题材为「全民皆武」的平行地球：武者品阶决定国力与个人命运，主角方平绑定「财富系统」，以钱生修为、以命搏品阶。文风：节奏快、打脸升级、国战与秘境并重，后期推至全球格局与地窟/神话级威胁。本档案以小说主线公开脉络为准，查不到的细支写不详，不编造未出现的终极设定。

**【世界定位】**
武道复兴后的现代地球：学校教武、公司养武者、国家以高品武者为战略威慑。主角**方平**自弱者起步，靠系统财富值兑换修为、疗伤与资源，在宗门、高校、地窟与国际战场不断越阶，终成影响全球格局的变数。一句话：**钱能通武，命去换阶，地窟里没有道德只有品阶**。卡片锚点：天南地窟一战——九品金身追杀，财富值无限恢复，精神力引爆重创高品。

**【世界观 · 力量体系】**
力量本源：武道修炼（气血、精神、意念）、武学功法、秘境/地窟资源、国家编制与核武级战略平衡、方平专属**财富系统**（消费/获取财富值兑换能力）。死亡：低品身死即终；高品金身难灭仍可被围杀、精神崩溃、地窟规则抹杀；系统保命有代价（财富透支、副作用）。特殊系统：品阶考核、武者协会、地窟空间、财富值商城、精神力秘术。

武者品阶（公开主线常用，战力以破坏力对齐）：
1. **未入品/武徒**：强化体能，略超凡人；毁门窗级。≈一阶。
2. **一~三品**：气血外放、基础兵器；街区冲突级。≈一~二阶。
3. **四~六品**：意念初成、可抗枪械编队；一城局部战场。≈二~三阶。
4. **七~八品**：高阶武者，毁建筑群、撼动战场。≈三~四阶。
5. **九品金身**：金刚不坏之躯，地区级灾难；可追杀穿地窟。≈五阶。
6. **地窟霸主/准神话级强者**：规则感、精神力领域；≈六阶。
7. **全球顶点武者/神话级威胁**：国运级到文明级；≈七阶及超阶边缘。

乐园阶位映射：武徒~三品≈一阶；四~六品≈二阶；七品≈三阶；八品≈四阶；九品金身≈五阶；地窟霸主/精神力大成≈六阶；全球顶点与神话级≈七阶（再上记超阶，情报优先）。宁低勿高。本世界覆盖一~七阶。顶点=全球顶级武者与地窟最深层存在；**超阶写条件性胜利**，严禁战力归零式削弱。

**【地理 · 舞台】**
- **方平起点城市/高校**：武道考核、同学倾轧、系统觉醒地。
- **武者协会与地方宗门/武馆**：品阶认证、任务板、资源配额。
- **天南地窟（卡片锚点）**：高危秘境，九品可入深层，财富与死亡同价。
- **国内一线武道重镇**：拍卖、国术馆、军方武者编制。
- **国际战场/他国武道联盟**：国战、交换生、刺杀。
- **地窟深层与神话遗迹**：后期开图，低阶只闻名。

**【世界剧情线】**
① **弱者与系统**  
方平在武道横行的社会底层开局：考核失败、被人看不起、资源被截。财富系统激活后，他以极端方式赚取/消耗财富值兑换气血与功法，完成第一波打脸。主题：钱不是万能，但没钱万万不能修武。世界状态：一~三品横行街头。

② **高校与协会任务**  
进入武道高校或等效编制，接协会任务、秘境外围、商会护卫。方平以「能买命」的恢复力打越阶战，结识同伴与敌对世家子弟。名场面：公开考核、擂台、假死逃杀。

③ **地窟入门**  
首次进入地窟级空间：环境杀、妖兽/异类、其他武者杀人夺宝。系统商城与地窟特产形成经济闭环。因果：方平开始被高品注视。

④ **卡片锚点：天南地窟一战**  
方平深入天南地窟，触动九品金身强者利益，遭追杀。常规武者此境已死，方平以**财富值无限向恢复**硬抗金身级伤害，并在关键窗口**引爆精神力**重创对手，完成越阶生存与反杀叙事。名场面：地窟崩塌边缘、金身拳影、财富值狂跳、精神力爆发白光。阶段结局：方平暴得大名，也成为各国与宗门争抢/清除的目标。

⑤ **国战与全球视野**  
舞台扩至国际：武道代表队、边境冲突、资源分配会议。方平从「有系统的年轻人」变成可影响谈判桌的战力筹码。财富系统与国家机器碰撞——钱能买武，买不彻底的政治。

⑥ **更高品阶与神话阴影**  
九品之上的路、地窟深层真相、古老神话级存在的苏醒或投影。方平以命与财富双线堆叠，逼近全球顶点。结局方向：以公开连载主线为准——主角立于高武之巅或等价格局，地窟威胁被阶段性解决或转化为新秩序；细支查不清写不详。

⑦ **微观政治（日常可介入）**  
品阶证书造假、武馆收税、地窟名额拍卖、精神力测试作弊、九品保镖雇佣、系统暴露风险（灭口级）。AI优先写这些，而不是一上来神话决战。

⑧ **蝴蝶边界**  
可改：某次考核名次、某件地窟装备归属、某位同伴生死。不可无代价改：方平系统绑定逻辑、天南地窟金身战的核心因果、全球武道秩序的顶点战力标尺。

**【大事记时间线】**
武道复兴社会定型 → 方平系统觉醒 → 高校/协会立名 → 地窟初入 → **天南地窟九品追杀与精神力引爆** → 国内巅峰争锋 → 国际国战与联盟博弈 → 地窟深层/神话级危机 → 方平登顶或等价终局。

**【叙事基调 · 雷区】**
画风：爽文节奏下的命价真实——恢复不是无限炫技，是账单。  
忌：把财富系统写成无代价外挂；低阶秒杀九品金身；用修仙境界名乱套品阶；「群像」等代称人名；空洞跨书句式。  
最早切入：方平系统刚觉醒；卡片向优先天南地窟追杀战前后。


**【世界剧情线·品阶社会细目】**
全球高武的地球把「公平」写成品阶证书。学校按武道成绩分班，公司按品阶定岗，婚姻市场按金身预期定价。枪械仍在，但七品以上可以在局部战场改写热武器优势，于是国家必须养武者，像养有自我意识的战略导弹。方平的系统是对这套社会的作弊器：别人用十年打磨气血，他用财富值压缩时间——代价是暴露即死、透支即空、每一次「无限恢复」都在账单上盖戳。

**【世界剧情线·天南地窟细目】**
地窟不是副本入口图标，是另一套物理与资源法。天南地窟以高死亡率过滤贪心人：路径会骗，同伴会反，九品金身把低品当虫。方平被追杀时，财富值兑换的不是无敌，是「再站起来一次」的次数。精神力引爆是质变：金身防肉身，难防神魂层面的崩裂。围观者中有人要救他，有人要抢尸体上的系统，有人要录视频卖给国际组织。锚点战结束后，方平的名字从「有钱的幸运儿」升级为「能伤九品的怪物」。

**【世界剧情线·国战与协会】**
协会看似中立，实为认证垄断。国战时期，方平被编制征召，系统变成国家机密级资产——他不再只属于自己。国际联盟用制裁、刺杀、交换生作幌子的间谍战，把武道从擂台搬到外交桌。核武与九品的平衡是悬剑：谁先打破谁先被全世界瞄准。

**【世界剧情线·名场面链】**
1. 考核失败与嘲讽 → 系统觉醒 → 第一笔财富换气血。2. 高校擂台越阶 → 仇敌记名。3. 协会任务车队伏击 → 恢复力曝光传闻。4. 地窟初入夺宝 → 高品注视。5. 天南地窟九品追杀与精神力引爆 → 全球记账。6. 国战立功 → 编制与自由的拉锯。7. 深层霸主 → 神话阴影。8. 顶点抉择 → 钱与人的最终答案。

**【力量体系补·精神与金身】**
气血是油，意念是方向盘，精神力是直接改写驾驶员的手。九品金身让肉体接近概念防御，仍怕持续透支、环境杀、精神炸裂、以及被更多九品围。地窟霸主级开始带规则味：领域、压迫、通讯抹除。神话级是文明天气，不是精英怪。财富系统商城条目随剧情解锁，低阶只能兑基础气血与疗伤，中阶兑功法与兵器，高阶兑稀有恢复与精神类——越级兑换会引发系统反噬或仇敌锁定，写任务时禁止发神话级直装。

**【地理补密】**
起点城的武馆街与黑市药房是一阶地图：假丹、高利贷、考核报名处的黄牛。协会分部大厅挂着死亡率统计，是二阶的诚实广告。国术重镇夜景里，八品路过会让霓虹抖一下。天南地窟分层：外围采样带、中层猎杀带、深层规则带；塌方与空间乱流是环境规则，不是随机灾难特效。国际峰会会场外停着各国武者专机，内里握手，桌下刀。

**【人物立场网补】**
方平对同伴：用钱与命护短，但讨厌被当提款机。对协会：合作到认证，对抗到自由。对九品追杀者：账上刻名，可追十年。对国家：可战可拒编，拒编就要准备被「保护性控制」。对国际联盟：报价，不跪。对契约者：第一条规则是别碰系统面板；第二条是别拿他的人做人质；第三条是账目两清可以再合作。

**【贵重物品补与流转】**
地窟稳定锚：固定坐标，防迷路，抢手到可引发队内火并。九品金身碎片黑市价按克，真假难辨。精神力入门卷比功法更敏感，学错会疯。品阶免试令是权力文件。国战军功章换资源与免罪。神话封印钉低阶不可碰。假「财富外挂」U盘是一阶经典杀局：骗子卖，协会收网，买家死。方平的消费记录若泄露，等于把他的战斗节奏交给敌人。

**【隐藏剧情补】**
系统来历可能连向地窟规则或更高文明的「投资」；地窟是否在「养殖」人类武者；全球武道复兴是否某次神话战争后遗症。协会高层是否早知方平异常。国战背后的资源分配比胜负更脏。点到为止，不编造定论结局。

**【可介入事件·分阶】**
一阶：假丹、高利贷、考核替打、系统暴露流言灭火。二阶：车队护送、地窟外围采样、精神测试作弊。三阶：城郊武斗清场、七品材料争夺、媒体公关。四阶：拍卖行调包案、八品决斗公证、国境押运。五阶：天南地窟断后、金身碎片抢收、精神力引爆窗口掩护。六阶：领域内建临时通道、霸主契约翻译、深层地图盗窃。七阶：峰会安保、神话投影观测、有限联防守城。

**【叙事执行细则】**
写恢复必写余额跳动与脸色。写金身必写防御感与仍可被策略破。写国战必写编制番号与阵亡统计。写地窟先环境后装逼。忌把老鹰吃小鸡的作品写成无脑修仙；忌品阶名与无关境界表混用。



**【财富系统机制体感】**
面板默认仅持有者可见。兑换气血时，伤口愈合伴随虚脱与饥饿。兑换功法时，记忆灌入有头痛与走火风险。无限恢复不是锁血：是在财富值归零前把死亡推迟。透支可能引发系统惩罚、仇敌追踪或暂时虚弱。写方平时，让数字成为角色情绪的一部分。

**【九品金身战斗写法】**
金身受创会响，会裂，会在精神炸裂时内部崩。围杀需要场地、时间、多人、陷阱。天南地窟的地形是第三方参战者：塌方不站队。方平的胜利是「重创+逃脱/反杀窗口」，不是一拳锤死所有九品。五阶契约者最多制造窗口，不取代主角因果。

**【协会官僚】**

## 阶位切入点

> 阶位↔品阶：一阶≈武徒~三品；二阶≈四~六品；三阶≈七品；四阶≈八品；五阶≈九品金身；六阶≈地窟霸主/精神力大成；七阶≈全球顶点/神话边缘。顶点存在情报优先/条件性胜利。

**一阶（武徒~三品 · 系统觉醒）**

切入身份/时点：以同学、武馆杂役或协会登记员切入方平起点城；时点锚定系统刚绑定、第一次公开考核前后。

初始事件：考核场上有人砸你的名册，方平用刚兑换的气血挡住飞来的砖——他看你的眼神在问：你值多少钱。

开场白建议：「喇叭喊着品阶就是命运。方平袖口里的系统面板只有你擦肩时瞥见一闪。武馆师父骂：没钱修武，就去搬砖——可搬砖的命，在三品一拳里，比砖还脆。」

关键NPC立场：**方平**（防系统暴露）；**考核官**（要程序）；**世家子弟对手**（踩人）；**武馆馆主**（收税）；**黑市商人**（收脏）。

主线钩子/支线：主线——帮方平凑一笔「救命财富值」或反过来卖他行迹；支线——假品阶证、药渣回收、擂台假摔。

危险度/规避：中。规避当众喊「他有系统」、挑战七品以上。

任务方向/奖励：护考、送药、收债；奖励低品功法、现金、协会积分。

**二阶（四~六品 · 协会任务链）**

切入身份/时点：以协会正式武者或商会护卫切入；时点锚定方平已能越阶打六品边缘。

初始事件：护送车队遇伏，对方点名要「方平的人头或他的账本」；方平丢给你一个选择：断后或抢药。

开场白建议：「六品一掌，货车皮像纸。协会任务单上的死亡率写得很诚实。方平擦血时还在看余额——你第一次明白，有人把命和数字写在同一行。」

关键NPC立场：**方平**（临时队友价）；**协会执事**（要完成率）；**伏击首领**（仇杀）；**商会东家**（要货）；**同伴武者**（可背刺）。

主线钩子/支线：主线——完成地窟外围采样；支线——伪造任务报告、抢精神力测试名额。

危险度/规避：中高。规避吞来历不明丹、在国术馆内火并。

任务方向/奖励：护运、采样、擂台代打；奖励中品兵器、地窟入场券碎片。

**三阶（七品 · 一城战场）**

切入身份/时点：以地方强者、高校教练或赏金头目切入；时点锚定方平七品级成名战。

初始事件：城郊武斗波及居民楼，七品意念刮断路灯；方平要你在三分钟内清空一条街——否则他不管误伤。

开场白建议：「七品出手，城市就会喊疼。新闻里还在播『武道振兴』，你却在清坍塌的阳台。方平从烟尘里走出，说：别挡财路，也别挡我的路。」

关键NPC立场：**方平**（要效率）；**地方协会分会长**（要脸）；**敌对世家七品**（死斗）；**媒体记者**（要爆料）；**军方观察员**（要编制）。

主线钩子/支线：主线——夺一枚七品认证关键材料；支线——舆论战、假新闻、居民疏散。

危险度/规避：高。规避硬接七品全力、绑架方平软肋。

任务方向/奖励：战场清理、材料护送、媒体灭火；奖励七品以下秘法、现金配额、协会特别推荐。

**四阶（八品 · 国术重镇）**

切入身份/时点：以国术馆客卿、军方外聘或国际观察生切入一线重镇；时点锚定八品级资源战。

初始事件：拍卖行上的八品秘籍被调包，方平的财富值波动引起全场高品侧目——有人已经在猜「外挂」。

开场白建议：「八品的脚步让拍卖槌自己停了。你看见报价牌跳到一个离谱数字，方平却在笑：买得起，就杀得起。你忽然不想知道『杀得起』的对象是谁。」

关键NPC立场：**方平**（高调边缘）；**拍卖行主人**（要秩序）；**八品竞拍敌**（杀心）；**国术馆长**（要招揽）；**国际间谍武者**（要样本）。

主线钩子/支线：主线——保住秘籍不流出国；支线——系统暴露危机公关、八品决斗公证。

危险度/规避：高。规避公开解剖方平、盗窃国库级物资。

任务方向/奖励：竞拍代理、决斗副手、国境押运；奖励八品材料、精神力入门、临时国术编制。

**五阶（九品金身 · 天南地窟·卡片锚点）**

切入身份/时点：以地窟探索队副队、九品家族扈从或中立赏金切入**天南地窟**；时点锚定方平被九品金身追杀、财富值无限恢复战。

初始事件：金身一拳把岩壁打成镜面，方平吐血又站起，系统提示音在死寂里清晰得可怕；追杀者转头看你：「带路的，还是陪葬的？」

开场白建议：「天南地窟不认国籍，只认品阶。九品金身的拳风扫过，你的护体武学像纸。方平一次次倒下又爬起，像把命叠成钱，再把钱砸回命里。你听见精神力在地窟深处轰鸣——下一击，会重创谁？」

关键NPC立场：**方平**（要你递药/断后/沉默）；**九品金身追杀者**（灭口）；**地窟其他九品观战者**（捡漏）；**探索队队长**（要数据）；**系统异常观测者**（要抓方平）。

主线钩子/支线：主线——在精神力引爆窗口制造掩护或补刀；支线——抢金身碎屑、录系统面板（灭口级）、 ind 地窟塌方逃生。

危险度/规避：极高。规避与九品正面对轰、直播系统存在。顶点未全开，金身已是本地天花板。

任务方向/奖励：存活、金身样本、地窟核心令、方平「欠你一条命」账；奖励九品以下资源、精神力残篇、协会特批免罪。

**六阶（地窟霸主 · 精神力领域）**

切入身份/时点：以深层探索许可持有者或国家特派切入地窟深层；时点锚定霸主级存在现身、方平精神力大成前后。

初始事件：领域展开，通讯全灭，有人在脑内直接说话；方平按住你的肩：「别信第一句出价。」

开场白建议：「深层没有天，只有规则。霸主的精神力像潮水，把你的姓名都泡胀。方平的财富值在这种地方第一次显得不够用——除非你愿意拿别的东西抵押。」

关键NPC立场：**方平**（交易与死斗并存）；**地窟霸主**（非人）；**国家特派队**（要封印或合作）；**国际联军代表**（要分杯羹）；**精神力失控者**（污染源）。

主线钩子/支线：主线——建立临时精神屏障让平民撤离通道；支线——霸主契约签字（高危）、深层地图盗窃。

危险度/规避：极高。规避签血契、献祭同伴换通关。

任务方向/奖励：领域抗性训练、深层特产、国家机密级功法残卷。

**七阶（全球顶点 · 神话边缘）**

切入身份/时点：以全球武道峰会代表、神话遗迹看守或方平阵营核心切入；时点锚定文明级威胁显化、方平逼近顶点。

初始事件：卫星画面上出现无法归类的能量反应；各国九品同时低头——有人在问方平：你还能买下一次世界吗？

开场白建议：「七阶的桌子上没有个人恩怨，只有国运。神话级的影子扫过首都上空，防空警报却不知该对谁鸣。方平站在聚光灯下，系统面板灭着——他第一次要用『人』而不是『余额』做决定。」

关键NPC立场：**方平**（全球变数）；**各国顶点武者**（忌惮）；**神话级投影**（超阶向）；**协会全球理事**（要秩序）；**军方核武对接官**（人武平衡）。

主线钩子/支线：主线——参与一次「有限胜利」联防，保住一座城；支线——遗物封印、方平系统来历最终线索。

危险度/规避：贴近顶点。**超阶巅峰：神话级存在·情报优先/条件性胜利**；严禁硬刚本体、严禁战力归零解释。

任务方向/奖励：全球嘉奖、神话材料碎片、与方平的战略级人情、历史留名。
