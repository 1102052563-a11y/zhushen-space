## 剧情

**【作品来源】**
《多罗罗》（どろろ）为手塚治虫所绘日本少年漫画，属战争、人性、黑暗奇幻与惊悚。题名来自作者童年听友将「どろぼう（小偷）」念作「どろろ」的记忆。1967–1968 年连载于小学馆《周刊少年Sunday》，中断后于 1969 年在秋田书店《冒险王》续完至结局；单行本常见秋田书店全 4 卷、讲谈社文库全集等版本；英文版 Vertical 出版并获艾斯纳奖相关荣誉。台湾早年曾以《妙小子》译名出版。改编谱系：1969 年虫制作公司电视动画全 26 话（杉井仪三郎总导演，后半改名《多罗罗与百鬼丸》，亦为后来「世界名作剧场」源流相关条目）；2004 年 SEGA 出品 PS2 游戏《Blood Will Tell》（日版仍称《どろろ》，角色设计含沙村广明等）；2007 年盐田明彦执导真人电影《多罗罗：天下之战》（妻夫木聪、柴崎幸主演）；2019 年 MAPPA×手塚制作公司电视动画全 24 话（古桥一浩监督、小林靖子系列构成、浅田弘幸角色原案，bilibili 等平台播出）。另有志水明重制漫画《百鬼丸与多罗罗》等衍生。本档案以 **2019 动画版主线骨架 + 原作核心因果** 为准（卡片锚点「十二魔神」「地狱堂」与 2019 版一致）；原作「四十八魔神」规模差异在隐藏段注明。文风：血腥与童声并列、残酷乱世不美化、夺体即夺人性。

**【世界定位】**
战国乱世的日本：饥馑、疫病、战火与鬼神交易并存。武将**醍醐景光**为求国运，在**地狱堂**与魔神立约，献出长子身体。残缺儿被医生**寿海**以义肢与兵器养大，取名**百鬼丸**，为夺回身体讨伐魔神；途中遇女扮男装的神偷**多罗罗**。一句话：**被父亲卖给鬼神的孩子，用刀把身体一块块讨回来，并学会不要变成自己所斩的东西**。卡片锚点：醍醐之国魔神讨伐——百鬼丸夺回身体，十二魔神，地狱堂业火。世界气质不是「升级打怪」，而是**夺体即夺人性**：每多一块器官，就多一分痛、一分恨、一分「像人」的资格。

**【世界观 · 力量体系】**
力量本源不在「修仙升级」，而在**鬼神契约、魂与业、武士刀技、义肢机关、灵视力、佛与邪的表里**。凡人一刀即死；鬼神可被斩而**归还**其所据身体部位；斩神则其眷顾之地丰饶崩坏——国运与土地被契约记账。死亡规则残酷：乱世人命贱于米；器官回归伴随痛觉、听觉、情感的回归，等于重新成为「人」。特殊系统：百鬼丸义肢藏刀；寿海义体术；琵琶法师（琵琶丸）灵魂视；三面鬼神可从其他魔神处**偷取**器官；地狱堂总契约绑定醍醐国运。

**灵魂颜色**：有害者异色，亲近者可辨；染红＝复仇欲吞噬主体。百鬼丸与琵琶丸皆能视之；多宝丸授眼后亦能。染红过深则斩神不归器官——规则本身惩罚「成魔」。

**契约会计**：贷方＝醍醐之米、医、安；借方＝百鬼丸身体与他处饥荒。斩神＝逼债；村荒＝破产。四阶任务常是「救了人却被扔石头」。

**义肢战斗美学**：刀从腕出，关节咔哒，木与铁的声音先于喊杀。回归血肉后，刀伤第一次让百鬼丸理解他人的痛。完整体：剑技仍在，但会眨眼、会泪、会握多罗罗的手而发抖。

战力阶梯（以破坏力对齐《阶位战力图鉴》，宁低勿高）：
1. **流民 / 小偷 / 足轻 / 逃兵**：冷兵器、石子、饥饿与人海；踹门、抢粮、被一刀结果。≈一阶。
2. **熟练武士 / 山贼头 / 百鬼丸器官残缺期（中前期）**：斩人破门、小队战；义肢刀与敏捷可屠中小妖。≈一～二阶。
3. **一国精锐 / 具名中妖 / 妖刀级灾厄**：毁村寨、端据点、一战搅乱战场。≈三阶。
4. **具名魔神本体（单神）/ 器官大成期百鬼丸**：毁城寨级妖祸、动摇一地国运条账。≈四阶。
5. **十二魔神主力 / 器官近满的百鬼丸 / 三面鬼神授眼后的多宝丸**：一击可夷平街区至城塞级冲突、国运明显崩坏。≈五阶。
6. **醍醐景光 + 契约总盘 / 三面鬼神水晶真身 / 地狱堂业火清算**：地区至国灭级灾难；军与民同时失「丰饶 buff」。≈六阶。

乐园阶位映射：流民～足轻≈一阶；武士～百鬼丸初行≈二阶；中妖与精锐≈三阶；具名魔神≈四阶；十二神主力与近满器官的百鬼丸≈五阶；地狱堂总契约与国运终战≈六阶。世界顶点＝十二魔神契约体系 + 醍醐国运 + 三面真身；**条件性胜利 / 情报优先**，严禁把顶点写成「被封印故战力为零」。本世界切入仅覆盖一～六阶，以上再上记超阶（本档不写）。

**【地理 · 舞台】**
舞台是手塚笔下的战国日本，不按真实战国编年严格复刻，而按「饥馑—契约繁荣—再崩坏」展开。地理即伦理：哪里丰饶，哪里就有人在付身体利息。
- **乱世野道与饥馑村落**：多罗罗成长与偷窃的主舞台；兵匪、腐尸、被吃光的田是日常。一阶契约者最常落地处：抢粮、逃兵、假路条。
- **寿海隐居工房**：义肢与义眼工坊、养育百鬼丸之处；表面安全屋，窗外仍有被烙印吸引的妖怪。工房规则：修身体，不修仇恨；不接「把囚犯改造成兵器」的单。
- **鬼神眷顾之村（各话不同）**：白天似桃源，夜里账房是祭坛；斩神后一夜荒芜，是主题剧场。写时先给村民脸，再给刀光。
- **醍醐之国（醍醐领）**：景光治下「米香刺鼻」的繁荣，实为契约利息；城墙内军心含丰饶 buff。国运裂时，米香先散，军心后散。
- **地狱堂**：立约与业火终局之地；石阶凉、火从内部烧，是国运账单的金库。六阶卡片锚点核心。
- **板门、无常岬、妖刀之地、白面不动石像一带等**：2019 版逐话讨伐锚点，器官账本的地理页；板门问「脸」，无常岬问「足与海」，妖刀之地问「耳与心」。
- **战场与朝仓前线（后期）**：外部压境迫使景光更不敢废约；国运裂时军与民同失血。战报可作四～五阶情报任务。
- **彼岸花海 / 雪原（父母死地）**：自夜之死的记忆坐标；童声旁白下的尸体美学，写时残酷不美化。
- **骨岬一类藏宝坐标**：火袋藏金处；原作有迁藏与信，追图者常死于半路——图是生路也是杀机。
- **骨岬一类藏宝坐标**：火袋藏金处；原作有迁藏与信，追图者常死于半路——图是生路也是杀机。

**【世界剧情线】**
本线按时间顺序写，便于任意时点判断「世界处于什么状态」。以 2019 动画 24 话骨架为主，原作差异随文标注。

① **地狱堂立约 · 弃子（前史 / 醍醐之卷）**
醍醐景光屡向神佛祈愿无应，领内饥馑疫病与战乱不止。国家存亡之际，他在地狱堂与魔神立约：献出即将出生的长子身体，换取醍醐繁荣与「统一乱世」的国运。婴儿出生即被分走身体部位——原作为四十八魔神分体；2019 版缩为十二魔神，出世时雷击下被分走十一部位（右足、皮肤、神经、耳、声带、眼、鼻、手、脊、足等），**身体**由本人持有。母亲**缝之方**（缝）不忍见子被处决，将其放入竹篮/木桶弃河。世界状态：契约生效，醍醐丰饶，他处仍乱；繁荣是租来的，租金是孩子的身体。低阶切入时，醍醐城内米香是「正常」，野道饥民是「正常」——两套正常靠契约并置。

② **寿海收养 · 义肢成「人」（寿海之卷前后）**
医生**寿海**于河上发现残缺婴儿。他不忍弃之，以木头、瓦片与机关制假手假脚假五官，训练知识与身手，令其长大。2019 版补完寿海前史：原为斯波氏家臣，因无法忍受对违法者施酷刑凌迟而投海，被明朝商船救起，在明土习得义肢术后归国，免费治疗贫困伤残。百鬼丸身上的魔神烙印不断招妖；寿海最终在义肢中装入兵器（2019：双手各藏刀；再失右足时右足亦装刀），放他上路夺体。原作中义肢设定更「机关兵器化」（拆下手中之刀、足喷碱水、鼻藏炸药等）；2019 收敛为藏刀美学。世界状态：百鬼丸以「兵器」自处，尚不会以人的方式恨与爱；工房是安全屋幻觉。

③ **多罗罗同行 · 神偷与刀（初遇至同行确立）**
女扮男装的小偷**多罗罗**，父母为盗贼首领**火袋**与**自夜**（只杀武家人），因部下「地鼠」等投降武家而家破：火袋被箭伤膝后惨死，自夜饿死/冻死于彼岸花海或雪原，死前以血在多罗罗背上画**藏宝图**（体温升高才显形）。多罗罗自称天下第一神偷，敏捷、石击、大叫吓退敌人。遭鬼神攻击时得百鬼丸相救，发誓跟随——原作偏「要偷他的刀」；2019 偏「看上身手、接下除魔活计」。二人在乱世斩妖，见「鬼神之村」繁荣：斩神后村即荒，多罗罗质问：「这些村民真的希望我们破坏这种和平吗？」百鬼丸最初不答，因为他没有回答的习惯与完整感官。世界状态：器官开始一件件归还；伦理冲突成为主线第二引擎。

④ **器官归还账本 · 2019 讨伐序列（分话主线）**
每斩一神，对应部位归还（若复仇欲过盛、灵魂染红，则部位不归）。2019 版公开对应关系可作讨伐清单：
- **皮肤**←沼和尚（泥吕）——万代之卷级开场妖祸；
- **神经**←招徕鬼（狮面女村长一类）——「和平村」表象下的神经支配；
- **右足**←罗丈门 / 脏奇虫等流转——舌头怪与鬼甲虫连环；
- **双耳**←妖刀「似蛭」——妖刀之卷，刀要耳也要心性；
- **声带**←脏奇虫——摇篮曲上下卷，百鬼丸第一次真正「说话」的门槛；
- **鼻子**←四化入道（蜈蚣）——阿猿之卷前后；
- **左手**←百面不动（夺脸石像）——板门/白面不动之卷，脸与身份被夺的恐怖；
- **脊椎**←魔魔蛾——地狱变之卷，身体支柱回归；
- **左足**←白神龙（附身鲨鱼）——无常岬之卷；
- **右手**←鵺（蛇尾吐火怪）——鵺之卷，五灵合体级灾厄；
- **双眼**←九尾/狐火等，终局经**三面鬼神**再流转——鬼神之卷至终话；
- **身体**←本人持有。
三面鬼神因神像庇佑未直接分得「身体」，只能从其他鬼神处偷取，成为契约体系内鬼。世界状态：百鬼丸逐步获得痛、触、听、声与泪；也逐步学会恨。写任意一话时，先查「此时他还缺什么」，再写他如何感知世界。

⑤ **美绪与战争儿童切片（摇篮曲之卷）**
**美绪**收留孤儿，以歌声安抚；她向两边军营乞食（含身体交易）养活孩子。百鬼丸初得听觉时，世间噪音难耐，唯美绪歌声可入。醍醐斥候视她为间谍（出入敌我营），下令袭击。百鬼丸因除魔归来不及，美绪为护孩而死。此段把「器官回归带来的感情」第一次打痛：百鬼丸开始理解爱与失去。世界状态：战争儿童与被卖掉的孩子主题并置——多罗罗是前者，百鬼丸是后者；多罗罗也在此学到「大人的和平」可以吃人。

⑥ **板门 · 无残帐 · 多宝丸初逢（中盘）**
板门之卷上下：夺脸、身份与「人是什么脸」的拷问。无残帐之卷：乱世账本式杀戮与生存记录。多宝丸之卷：继承人出场，知兄事后质问父亲，却在「兄」与「国与民」之间开始倾斜。世界状态：醍醐内部政治从「繁荣神话」滑向「抓害国运者」；通缉画像上出现无脸少年与小贼。

⑦ **灵魂染红 · 近魔危机（中后期）**
**琵琶法师（琵琶丸）**可观察灵魂颜色，数次警告百鬼丸：复仇欲染红则与鬼神无异。百鬼丸一度被仇恨蒙蔽，骑白马怨灵所化之鬼，疯狂斩杀醍醐兵，要夺回身体。多罗罗在断交与重聚之间摇摆：她决定分道时被山贼绑架；百鬼丸为救她斩鲨神，看清「鬼神的和平只是一瞬」。蛇尾吐火怪之战中多罗罗险死，被琵琶法师所救；百鬼丸怨恨自己无手，内心更魔化，直杀醍醐国。多罗罗跟他一起面对因果。世界状态：主题句落地——**不要变成你所斩的东西**。契约者若在此阶段只推「加速夺器」，会把正典线推向屠城坏结局的气压。

⑧ **多宝丸与国 · 兄弟之刀（逆流 / 缝 / 鬼神）**
弟**多宝丸**被立为继承人。原作中他不择手段、与兄决战而死；2019 版扩写为关心百姓的郡主：知兄事后质问父亲，却在「兄」与「国与民」之间选择国，维护魔神约定。第一次对战中多宝丸右眼被砍瞎，战力下滑，愈发认定百鬼丸是乱国之源。后期向鬼神祈愿，被三面鬼神授予与百鬼丸同质之眼，可看灵魂颜色。缝之卷：生母**缝之方**线回收，母性、弃子与终局救多罗罗的因果闭合。兄妹/兄弟刀兵相见，是国运与血缘的撕裂。世界状态：醍醐内部「正确」的爱国开始要求刺向血亲；外有朝仓压境，内有业火账单到期。

⑨ **卡片锚点：醍醐魔神讨伐 · 地狱堂业火（鬼神之卷 · 终话）**
百鬼丸器官将近，直取醍醐与十二魔神契约核心。朝仓等外压与国运反噬同步：醍醐在战与业中崩坏。多罗罗被抓后得百鬼丸之母相救，得知经历，在牢中劝：「如果没有器官，我愿做你的手或眼，但千万不要变成魔神。」终局：百鬼丸夺回全部身体与感官，斩断因果；对景光或杀或恕——2019 偏破坏头盔选择原谅，景光悔「若立长子或许不同」；多宝丸力竭或死（2019：献出器官后伤重而逝）；三面鬼神水晶真身被灭。多罗罗愿在醍醐等他旅行归来。百鬼丸再踏人生之路。世界状态：身体完整，问题未完；繁荣会计清零，遗民与废墟留下。

⑩ **原作终局对照（四十八魔神线）**
原作漫画中醍醐景光可被怪物结合体依附，百鬼丸斩结合体后景光野心不改，终被农民赶走；多宝丸在决战中死去，百鬼丸事后才知是弟。1969 动画另有完结处理。本档案卡片锚点以 2019 地狱堂业火为准；若玩「四十八完整账本」模式，须标注魔神数量与终局政治差异，禁止混成一锅。

⑪ **可介入微观事件（任意时点）**
村税与人柱、妖刀传说、藏宝图争夺（鼬追图、骨岬迁藏）、山贼绑多罗罗、白马怨灵工具化、朝仓战报、斩神善后的瘟疫与土匪接收、伪造国运报告、反抗军接头、寿海工房守门送药、假扮祭官毁一半祭品。斩神前先问：**此村靠神活着吗？** 斩神后先问：**逃难的人今夜睡哪里？**

⑫ **伦理核心（贯穿全线，非凑字）**
被压迫者是否有权拆掉别人的「恶的和平」？多罗罗的质疑不是拖剧情，是主题。百鬼丸最初不回答，因为他没有「回答」的声带与习惯；有了感情后，恨比答更快。琵琶丸的红色警告等于：你正在变成你所恨的契约。景光的逻辑是「国重于子」；多宝丸的逻辑是「民重于兄」；多罗罗的逻辑是「人重于国与神」；百鬼丸最终逻辑是「我要完整，但不以成魔为价」。器官回归带来的不仅是战力，是痛、触、泪、性与耻的全部人类负担——他曾以「兵器」自处，完整后必须学「人」。

醍醐之卷 → 万代之卷 → 寿海之卷 → 妖刀之卷 → 摇篮曲上/下 → 络新妇之卷 → 阿猿之卷 → 无残帐之卷 → 多宝丸之卷 → 板门上/下 → 白面不动之卷 → 鲭目之卷 → 地狱变之卷 → 不知火之卷 → 问答之卷 → 无常岬之卷 → 天邪鬼之卷 → 鵺之卷 → 逆流之卷 → 缝之卷 → 鬼神之卷 → 终话「多罗罗与百鬼丸」。每一话都是器官账本的一页，也是乱世人命账本的一页；写支线时可挂在对应话标题下，勿打乱器官回归顺序的因果。

**【主要人物】**
- **百鬼丸**（主角·被献祭的长子）｜性格：寡言、执念、从「物」学会「人」，恨与爱交战；初期近于兵器，后期险成魔；名字取自百鬼夜行意象。｜装备·能力：寿海制义肢（双手藏刀，可再装足刀；原作更机关化）；剑术与敏捷；灵魂视色（判断有害）；器官回归后的完整肉身与痛觉；无眼期用心眼与传念沟通。｜人物弧光：残缺弃儿→讨伐者→灵魂染红近魔→在多罗罗与琵琶丸的拉扯下拒绝成魔→完整之人再上路。｜立场关系：对多罗罗羁绊最深；对景光仇与恕并存；对多宝丸是血缘与刀锋；对契约者先当刀、障或同行。写他初期：少台词，多动作与灵魂色；写他后期：痛觉与泪比喊杀重要。
- **多罗罗**（女主·神偷）｜性格：嘴硬命硬、善良藏在偷窃里、童言利刃；女扮男装以在乱世活命。｜装备·能力：神偷、敏捷、石击、大叫吓退；背上藏宝图（体温显形）。｜人物弧光：战争孤儿→同行者→道德质问者（斩神村荒）→劝百鬼丸勿成魔→在醍醐等归人。｜立场关系：可带路、可吵、可救命；离开时因伦理，重聚因认清鬼神和平虚妄。禁止恋物化；强调生存与选择。

**【势力图谱】**
- **醍醐之国**：宗旨是「繁荣与存续优先于一子」；地盘为契约丰饶核心；代表**醍醐景光、多宝丸、缝之方**；战力一～六阶随国运涨落；与朝仓等外敌对立，与十二魔神是交易而非臣属。军吃契约米，精锐含丰饶 buff；国运裂则军心裂。主线作用：繁荣会计的贷方与崩坏现场；五～六阶政治战场。
- **十二魔神 / 原作四十八魔神**：交易丰饶、分食身体；代表各具名魔神与**三面鬼神**；战力三～六阶；彼此可偷器官。主线作用：器官账本与伦理困境源。写讨伐时必须具名对应，禁止「随便一个妖」糊弄。
- **山贼与盗贼团（地鼠、鼬等）**：乱世寄生两端难民；代表**火袋旧部、鼬**；战力一～二阶。可投武家、可再被贵族当诱饵。主线作用：多罗罗身世与藏宝图支线。
- **他国（朝仓等）**：后期压境；迫使景光不敢废约。战力以凡人军阵为主，含精锐≈三阶。战报与细作是四阶情报货。
- **佛寺 / 琵琶法师游方**：灵视、葬礼、警告；可收醍醐香火而沉默，亦可救人。代表**琵琶丸**。是「染红」监测网的节点。
- **寿海工房圈**：非政权，是修复伦理的孤岛；吸引妖怪却拒绝酷刑逻辑。一～二阶安全屋与维修点。
- **鬼神村自治体（各话）**：表面自治、实为魔神佃户；斩神后权力真空，土匪与瘟疫接手。主线作用：伦理剧场。

敌友法则：斩神＝可能救一人却饿一村；利益与业纠缠。契约者夹在「斩」与「不斩」之间，最稳的活是救人、修义肢、藏名册，而不是宣称一刀清十二神。只斩不救→多罗罗离你；只救不斩→百鬼丸视你为障。

- **归还之器官（耳目手足皮肤神经声带脊等）**：斩神归还物；不是无来源宝箱，写讨伐须对应魔神。灵魂染红时可不归——规则本身就是主题。

- **琵琶藏刀**：路遇援；游方僧的杀器。

**【隐藏剧情 · 伏笔】**
- 繁荣是租来的，租金是孩子的身体；醍醐米香的借方是他处的饥荒。斩神后会计崩溃，村民会恨「拯救者」。
- 灵魂染红则与鬼神无异——主题是「不要变成你所斩的东西」。琵琶丸的红是温度计，多罗罗的劝是刹车。
- 原作四十八神 vs 2019 十二神：舞台规模差，因果同构；本档清单以十二为主，四十八作「更残酷完整账本」的传闻。
- 三面鬼神未获「身体」、只能偷器官：契约体系内鬼，终局必须碎的核。
- 景光「若立长子」的悔是终章盐，不是洗白全部罪行。
- 游戏版「多罗罗为魔神之首」、百鬼丸「天之子」等分支不采信为本档案主线。
- 百鬼丸完整后仍选择旅行：身体回来了，世界的问题没结束。
- 多罗罗女扮男装与「我愿做你的手眼」：爱与防成魔的双重结构，禁止成人恋物化写法。
- 原作景光可被怪物结合体依附、终被农民赶走：与 2019「业火+悔」不同终局政治，写时须标注版本。

**【叙事基调 · 雷区】**
画风：血腥美、童声与地狱并列；战争残酷不美化；器官回归写痛与泪，不写升级快感。

最早可切入锚点：寿海工房期，或百鬼丸初遇多罗罗；卡片向优先醍醐终战前后。写百鬼丸初期：少台词，多动作与灵魂色；写斩神：先写村民脸，再写刀光；写终局：痛觉回归的第一口空气比胜利呐喊重要。写多罗罗：童言利刃，禁止傻白甜。

## 阶位切入点

> 阶位↔战力：一阶≈流民盗贼；二阶≈武士与百鬼丸初期；三阶≈精锐与中妖；四阶≈具名魔神；五阶≈十二神主力/器官近满；六阶≈地狱堂总契约终战。世界顶点（十二魔神契约+三面真身+业火）为条件性胜利；低阶切入一律「规避顶点·情报优先」。

**一阶（流民·盗贼 · 多罗罗独行期）**

切入身份/时点：以饥民、小偷、足轻逃兵或货郎切入乱世野道；时点锚定**多罗罗父母已死、独自行窃**、尚未稳定跟随百鬼丸。

初始事件：你与多罗罗抢同一个粮袋，她亮石子大喊「天下第一神偷」；远处村庄却飘着不该有的稻香——「鬼神村」的味道，税吏正押着人柱名单走过。

开场白建议：「乱世里姓名比米贱。小孩嗓门却大得像要吓退鬼：我是天下第一神偷！你还没笑出来，就听见林子里有东西用『不是人』的步子走近——而那稻香，甜得像谁刚付过一笔不该付的账。」

关键NPC立场：**多罗罗**——防又好奇，可临时合伙，怕被卖；**鼬**等山贼——抓壮丁与藏宝图；**鬼神村村民**——排外，靠神活着；**足轻队**——征粮抓逃；**火袋旧部残党**——认装备、要图。

主线钩子/支线：主线——与多罗罗临时合伙偷出**人柱名册**或逃过征粮；支线——假粮交易、逃兵路条、藏宝图传闻（温泉/体温显形）、给饥村送半袋米换一句情报。

危险度/规避：中。规避夜闯地狱堂方向；规避对村民宣布「我来斩你们的神」；规避与醍醐正规军硬碰。

任务方向/奖励：粮、药、路引、多罗罗的半句信任、人柱名单残页；奖励勿越阶发妖刀/器官。

**二阶（百鬼丸同行·初斩）**

切入身份/时点：以浪人、药卖、寿海信使或临时除魔佣兵切入；时点锚定**百鬼丸与多罗罗刚组队、首次归还器官前后**（皮肤/神经/右足等初回期）。

初始事件：泥沼里伸出不是人的手臂，百鬼丸义肢咔哒出刀；多罗罗塞给你一块布：「帮盯着，别让他灵魂变红。」远处琵琶声起，**琵琶丸**在测颜色。

开场白建议：「他没有眼睛，却看你。木制关节里藏着寒光。多罗罗在旁边嚷着要偷他的刀——你知道那不是刀，是他还没长回来的命。沼气里有东西在笑，像在收下一笔利息。」

关键NPC立场：**百鬼丸**——视你为刀、障或短同行；**多罗罗**——嘴炮队友，管伦理；**寿海**——可修义肢，拒酷刑单；**沼和尚/招徕鬼等初妖**——敌；**琵琶丸**——观察染红。

主线钩子/支线：主线——助斩一神并处理**村荒善后**（逃难/瘟疫/土匪接收）；支线——义肢零件采购、灵魂色观察训练、送药守工房。

危险度/规避：中高。规避偷器官、煽动百鬼丸仇恨、只斩不救（多罗罗会离开你）。

任务方向/奖励：妖残骸、医术、短期同行位、寿海的一枚铆钉；奖励取向＝义肢维修知识、低阶刀技，勿发国运契约。

**三阶（精锐与妖刀）**

切入身份/时点：以醍醐斥候、妖刀猎人、税吏随员或逃民首领切入；时点锚定**似蛭等具名事件、双耳归还前后**，国运仍稳但裂痕初现。

初始事件：妖刀要你的耳，百鬼丸要刀里的神；战场第三边是景光的税吏——丰年要谢殿下，乞丐却说丰年是小孩的肉换的。

开场白建议：「刀有舌头，会舔血认主。税吏说丰年要谢殿下，乞丐却说丰年是小孩的肉换的。你站在刀与税之间，没有第三种沉默能活命——而百鬼丸的灵魂，已经开始有一点红。」

关键NPC立场：**百鬼丸**—— debatable 盟友；**多罗罗**——质问斩神；**醍醐税吏**——要粮要人；**妖刀执/似蛭相关**——敌；**村民长老**——怕荒。

主线钩子/支线：主线——夺妖刀归耳、阻止错误献祭；支线——税册调包、逃民护送、假扮斥候送假战报。

危险度/规避：高。规避把妖刀献给景光换官；规避在鬼神村当众拆祭坛却不安排后路。

任务方向/奖励：刀技、情报、器官回归目击证明（奇货）、税册副本；奖励贴合三阶，勿给三面之眼。

**四阶（具名魔神·国运裂）**

切入身份/时点：以国侍、反抗军接头人、魔神祭官或朝仓细作切入醍醐边境；时点锚定**多神已斩、国运不稳、通缉百鬼丸与多罗罗**。

初始事件：稻田一夜焦黑，景光下令抓「害国运者」；通缉画像上是无脸的少年与小贼。反抗村民把你拉进地窖：「救我们的孩子，或去告诉殿下——繁荣在退潮。」

开场白建议：「丰饶退潮，露出底下的白骨。殿下要祭品，鬼神要利息，百姓要活。百鬼丸从焦田走过，每一步都像在收回自己的骨头——而你手里的假国运报告，能救人，也能把你钉在城门上。」

关键NPC立场：**醍醐景光**——要杀百鬼丸保约；**多宝丸**——动摇中；**百鬼丸**—— debatable；**多罗罗**——要救人；**魔神祭官**——要利息；**反抗村民**——要活。

主线钩子/支线：主线——救被献祭者或阻一次错误斩神（防村灭）；支线——伪造国运报告、引朝仓细作、藏人柱名单、接头寿海运义肢。

危险度/规避：高。规避在地狱堂门口叫阵；规避向三面献祭任何器官。

任务方向/奖励：武士身份、避战符、百鬼丸「不杀之恩」、反抗军路标；奖励＝人脉与情报优先。

**五阶（十二神主力·兄弟刀）**

切入身份/时点：以多宝丸侧近侍、百鬼丸临时军、牢卒或城门武者切入；时点锚定**兄弟决战、三面授眼前后、多罗罗被囚**。

初始事件：多宝丸右眼已盲又复「神视」，要你指认百鬼丸弱点；多罗罗在牢里喊：「别让他变成鬼！」三面鬼神在笑，用别人的器官当筹码。

开场白建议：「兄弟的刀比国界锋利。三面神在笑，用别人的器官当筹码。你看见两种灵魂的颜色在城墙上撞——红的是恨，青的是还没说完的话。多罗罗的嗓子已经哑了，她还在喊：不要变成魔神。」

关键NPC立场：**百鬼丸**——近魔边缘；**多宝丸**——择国；**多罗罗**——劝诫核心；**三面鬼神**——敌；**醍醐军**——人海；**琵琶丸**——最后警告；**缝之方**——可救多罗罗。

主线钩子/支线：主线——在决战中保住「可醒悟」的缝（阻错误致命、递话、毁错误人柱）；支线——救多罗罗出牢、截白马怨灵、毁授眼仪式一半祭品。

危险度/规避：极高。规避献祭百鬼丸器官给三面；规避煽动屠城。

任务方向/奖励：鬼神之眼情报、城门钥匙、终战入场资格、多罗罗的信任；奖励勿让契约者直接持有「完整十二神之力」。

**六阶（地狱堂业火·卡片锚点）**

切入身份/时点：以契约见证人、业火祭司、终战特使或遗民首领切入地狱堂；时点锚定**十二魔神讨伐终局、业火吞噬、身体完整前后**。

初始事件：堂内业火起，神像崩，景光解盔待死；百鬼丸刀停在他额前——你的一句话可能偏转「杀」或「恕」。三面水晶真身未碎前，国运仍可能反噬全场。

开场白建议：「地狱堂的火不是照明，是账单。十二神的名字在火焰里一个个灭掉，像收回的借条。百鬼丸的身体终于完整，痛觉第一次教会他何谓活着。景光跪着，国在烧——你要见证屠父，还是见证一个人拒绝变成魔神？」

关键NPC立场：**百鬼丸**——终局选择；**醍醐景光**——悔/死；**多罗罗**——求他别成魔；**多宝丸**——力竭；**三面真身**——最后一击；**业火**——规则清算本身。

主线钩子/支线：主线——助斩三面水晶真身并护多罗罗、阻止百鬼丸成魔；支线——抢出契约原本、护送遗民、在业火中保一条逃命路。

危险度/规避：贴近顶点。**超阶向：总契约+业火——存在·情报优先/条件性胜利**；严禁把景光或三面写成无代价可秒；严禁用「封印削弱」解释顶点战力。

任务方向/奖励：完整因果见证、百鬼丸的人情、多罗罗藏宝图一角、乱世重建名义；奖励取向＝人脉/见证/重建权，勿发「随意支配国运」的外挂。
