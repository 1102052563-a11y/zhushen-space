## 剧情

**【作品来源】**
《一世之尊》作者**爱潜水的乌贼**，起点中文网东方玄幻长篇，已完结，约444.79万字。融合江湖武道、诸天轮回、佛道神话与布局流，是乌贼早期代表作之一，后接《诡秘之主》宇宙前史气质。主题句：「我这一生，不问前尘，不求来世，只轰轰烈烈，快意恩仇，败尽各族英杰，傲笑六道神魔。」**以小说正典为准**。  
**全本检索**：[搜笔趣阁·一世之尊](https://www.sobqg.com/book/s2-yishizhizun.html)（已完结）+ [起点作品页](https://m.qidian.com/book/3249362/) 核对人物与分卷。

**【世界定位】**
大周王朝江湖与朝廷并行，少林、六扇门、隐皇堡、九幽神道与诸天轮回交织。现代人**孟奇**穿越为大户弃子，入少林法号**真定**，借「六道轮回」任务世界收集绝学神兵，从杂役院一路杀到传说、造化，终问「一世为尊」。一句话：机心少年在轮回棋盘上活成自己的一世。卡片锚点：**少林杂役院·首次隐皇堡任务前后**——真定初入轮回、绝学兑换、江湖仇杀与六扇门秩序同时压上。

**【世界观 · 力量体系】**
武道与道法双轨，核心是**窍穴—内外景—法身—传说—造化**递进，绝学、神兵、因果、轮回积分共同构成成长资源。  
- **练气/百日筑基**：开窍前体能与内息奠基，能打江湖三流。  
- **开窍**：打通窍穴，内力成型，可施展门派绝技雏形，一域好手。  
- **外景**：外景现世，借天地之势，毁屋裂地，镇一方。  
- **内景**：内景世界自成，法术与武道质变，可抗军阵与小规模神通。  
- **法身**：法身凝聚，近乎「半神」位格，移山填海初现，一城难敌。  
- **传说**：传说级强者，改写局部规则、跨域追杀，名动诸天。  
- **造化**：造化级，掌大道一角，可开宗立派、干涉轮回棋盘。  
- **传说之上/准道祖**：触及六道轮回之主、元始、灵宝、道德等阴影。  
- **道祖/天尊级**：元始、灵宝、道德、如来等顶点；**六道轮回之主**为轮回体系幕后。  
死亡：低境身死即死；高境有轮回、化身、因果债与「再来一世」可能，皆有代价。神兵有灵、绝学有门槛、轮回任务有积分与死亡风险。  
**乐园阶位映射（宁低勿高）**：练气~开窍≈一～二阶；外景≈三阶；内景≈四阶；法身≈五阶；传说≈六阶；造化≈七阶；传说之上≈八阶；道祖/六道轮回之主＝**超阶**（情报优先/条件性胜利）。顶点严禁「被封印所以战力为零」。

**【地理 · 舞台】**
- **少林寺**：孟奇起点。杂役院（苦役、机心弃子）、武僧院（正规资源）、演武殿（名利与仇）、玄字辈院务、黄墙黑瓦鎏金匾。写戏先定：你在哪个院、有无法号、戒尺离你多远。  
- **大周江湖与京畿**：州府、码头、茶楼通缉令、门派山门、世家庄园。六扇门衙署与编外暗桩并存。江东等地缘节点在中期频繁出现，适合写盟约与名局。  
- **隐皇堡世界（轮回本）**：堡、刀墙、走廊、堡主所在核心区；规则杀优先于人数杀。  
- **诸天任务世界群**：每本任务一个小舞台（古城、雪原、宫城、封印之地等），共同规则是积分与死亡；「四大封印之兵」类名场面属于高危本。  
- **玉虚相关**：玉虚宫、门下据点、异变污染带；越靠近核心越「世间美好、玉虚幽暗」的反差。  
- **九幽神道路径**：可被斩断的神道脉络与残路；斩路后是废墟政治。  
- **神话上层**：道祖投影、造化会盟虚空、轮回棋盘边缘——低阶禁止硬闯。  
写场景五感：粥香与戒尺、刀锈与雨、积分榜的冷光、神兵出鞘的寒意、玉虚异变的气味（说不清的香或腥）。

**【世界剧情线】**
公开分卷脉络（据目录与正典可核）：少林机心 → 隐皇堡轮回 → 江湖六扇门 → 内外景扩张 → 法身密会 → 玉虚/九幽/元始线 → 传说再续 → 造化云集 → 真定如来与一世收束。以下按阶段写清因果，供 AI 判断任意时点世界状态。

① **机心与少林（开局·练气到开窍）**  
现代人孟奇穿越，睁眼即被马脸护卫安顿，随黄衣僧**玄藏**入**少林寺**。询名时他答「既入少林，俗名已忘」，**玄苦**断「机心太重」，法号**真定**，入**杂役院**而非武僧院。同期孩童有**真慧（方阿七）**、**真德（刘治）**等。  
百日筑基、小玉佛坠、罗汉拳、演武殿争霸与「恶人小孟」外号：他用机心换资源，也在青灯下炼出真勇气。同门压迫、少林规矩（心空无玄真排序）、离开少林的办法成为早期钩子。此阶段世界状态=江湖入口级，朝廷与神话皆远。

② **六道轮回开启·隐皇堡（开窍到外景门槛）**  
真定触及**六道轮回**，首次任务世界即**隐皇堡**：第九章起刀、第一战、援手、自杀、堡主、兑换与回归。轮回规则：任务目标、积分、死亡风险、绝学神兵兑换。  
名场面链：奇诡再聚 → 隐皇堡刀墙 → 堡主试炼 → 积分兑换 → 回归少林实力跃迁。江湖与**六扇门**开始记名。主题句在此落地：勇气、秘闻、祸福相依。此阶段=「轮回新人」，假兑换单与假地图可致命。

③ **江湖扩张·六扇门与同伴（外景—内景）**  
带艺行走：离开少林、化缘式任务、围杀与血战、重逢。结识**顾小桑**（目的成谜的灵动女主线）、**江芷微**（剑道，屠鸡亦可斩神气质）、**张遇贤**、**方多病**、**阮盐**等。  
「六扇门中好修行」阶段：朝廷秩序与江湖仇杀并行，真定在捕快、门派、轮回队员多重身份间切换。**轮回符**、更多任务世界开启，绝学与神兵数量上升，内外景稳固。  
因果：同伴不是挂件——每个人有自己的轮回与债；出卖同伴换积分会在后续被追讨。

④ **法身之路·玉虚与九幽（法身）**  
法身密会、顶级法身、离开法身后的两年空白感：位格从「人上人」进「半神话」。**玉虚门下/玉虚之异/玉虚之事/玉虚宫前/玉虚异变**连成中期主轴；**一朝斩断九幽路**是神道线硬节点。  
元始相关绝学与概念开始频繁：元始九印、不灭元始身、太上无极元始庆云、观想元始、人皆元始——写高阶时用这些真名，禁止用「某神功」糊弄。  
此阶段世界状态=城邦/地区级冲突可被法身一人改写；低阶契约者应规避正面法身对决。

⑤ **传说再续·诸天棋盘（传说）**  
「传说再续」后，跨域追杀与规则级手段常态化。各自的轮回、直面顾小桑、查阅卷宗：情报战与情感线同步升温。孟奇在「快意恩仇」与「布局一世」间取舍，棋手与棋子身份反复翻转。  
危险来源从门派仇家转为**造化级布局**与**六道轮回之主**的棋盘微震。低阶只闻名见痕迹。

⑥ **造化云集·道祖阴影（造化）**  
造化云集：同阶巅峰并立，绝学终极（如来神掌、截天七式等）与神兵认主条件公开化。真定如来等名场面标志佛道顶峰手段介入人间。  
大劫再启伏笔兑现：如来金身 vs 元始道体的比较不再是酒肆传言，而是可观测的天边投影与城邦级恐慌。

⑦ **结局方向（传说之上收束）**  
主题兑现：不问前尘、不求来世、一世为尊。孟奇完成「纵横一生，谁能相抗」的自我定义；同伴线（顾小桑、江芷微等）与神兵、绝学下落以正典收束。  
档案写公开完结向：谁活谁走、棋盘是否翻盘，以完结章与番外为准，不扩编未核细目。任意时点判定：看孟奇境界（开窍/内外景/法身/传说/造化）+ 是否已触玉虚/九幽/元始线 + 轮回积分是否已成负资产。

**【分卷因果速查（AI时点）】**  
未触轮回＝少林杂役与机心时代；已入隐皇堡＝轮回新人；六扇门阶段＝江湖名号；法身密会后＝半神话；玉虚/九幽线开启＝中期主轴；传说再续＝跨域；造化云集＝终局前；真定如来后＝收束可写。

**【主要人物】**
- **孟奇/真定（主角）**｜性格：机心重、自嘲幽默、重恩仇与同伴、敢在绝境下赌命｜装备·能力：罗汉拳与少林底子起家，后藉六道轮回兑换诸天绝学（含元始、如来等体系相关）、神兵与法身—传说—造化递进；轮回积分是第二生命｜人物弧光：杂役院「机心太重」的弃子 → 隐皇堡证明勇气 → 江湖与六扇门双面人 → 法身玉虚九幽杀出 → 传说造化上反将棋盘 → 一世为尊｜立场关系：对顾小桑深缠；对江芷微剑心相惜；对张遇贤/方多病/阮盐等同伴共债；对六道轮回之主从棋子到对弈。

- **玄苦**｜性格：方正严厉、厌机心｜能力：玄字辈黄衣僧，戒尺与院务｜弧光：把真定打入杂役院的关键判定者｜立场：秩序方，可敬可惧。  
- **玄藏**｜性格：颓丧外表、行事干脆｜能力：带孟奇入寺｜弧光：少林入口NPC｜立场：中立偏门派。

- **六道轮回之主**｜性格：棋手、超然残酷｜能力：轮回体系幕后，超阶｜弧光：全书最大阴影之一｜立场：顶点——情报优先/条件性胜利。  
- **元始/灵宝/道德/如来等道祖侧**｜能力：道祖级阴影与绝学源头｜弧光：终局神话标尺｜立场：低阶只闻名，高阶条件胜。

**【势力图谱】**
- **少林寺**：宗旨=佛武双修、镇一方；地盘=少林山门与下院；代表=玄苦、玄藏、真字辈；战力=开窍到内外景常见，传说级稀有；与六扇门合作多于开战；作用=孟奇起点与部分绝学来源。  
- **六扇门/大周朝廷**：宗旨=律法与稳定；地盘=京畿与州府；代表=具名捕头/主事；战力=编制武力+江湖编外；与门派既合作又监视；作用=通缉、修行便利、「六扇门中好修行」舞台。  
- **江湖门派与世家**：宗旨=绝学与地盘；代表随卷出现（写时用正典真名）；战力=开窍到法身为主；互相仇杀联姻；作用=日常冲突与资源。  
- **六道轮回体系**：宗旨=任务—积分—兑换—筛选强者；代表=轮回队员与幕后**六道轮回之主**；战力=跨世界；作用=全书成长引擎与最大阴谋容器。  
- **玉虚相关（道门线）**：宗旨=道统与绝学传承；关键地点=玉虚宫等；绝学=玉虚宫三大绝学之首等；作用=中期主轴与元始线接口。  
- **九幽神道**：宗旨=神道权柄与路；名场面=一朝斩断九幽路；作用=法身—传说过渡的硬冲突。  
- **造化云集层**：造化级个人与小团体并立，临时同盟极脆。  
敌友随境界与因果瞬变；写站队必须真名。

**【贵重物品】**
- **小玉佛坠**：入寺前既得佛缘信物，早期护身与机缘钩子。  
- **轮回符/轮回积分与兑换权**：比散装法宝更常决定生死；假符与假兑换单是黑市杀手。  
- **隐皇堡相关兵器与刀意传承**：早期任务核心掉落向。  
- **诸天绝学真名**：如来神掌、截天七式、元始九印相关、不灭元始身、太上无极元始庆云、玉虚宫三大绝学之首等——写下落、门槛与代价，不编未核绝学。  
- **神兵**：任务世界神兵与兑换神兵；有灵、认主、因果；低阶最多触「认主失败样本」。  
- **法身/传说/造化本命物**：中高阶才可染指；道祖级神器不对低阶发放。  

**【隐藏剧情 · 伏笔】**
- **六道轮回之主**的棋盘：积分异常、任务目标互相矛盾、队友突然「合理死亡」。  
- 孟奇「机心」与「勇气」：玄苦的判定是伏笔，隐皇堡是第一次兑现，终局是总兑现。  
- **顾小桑**目的层：中前期只给碎片，直面章才摊牌。  
- **元始**概念扩散：从绝学名到「人皆元始」级现象，标志神话层渗透人间。  
- 大劫再启、如来金身与元始道体比较：酒肆流言→投影→终局。  
- 与乌贼后续作品宇宙气质互文：点到为止，不抢《诡秘之主》正典。

**【叙事基调 · 雷区】**
热血布局 + 江湖快意 + 神话棋盘；信息差、因果债、积分代价优先于无脑开挂。文风可带乌贼式吐槽，但杀伐与恩情要落地。  
忌：孟奇纯圣母或纯屠戮；忌低阶硬刚道祖/六道轮回之主；忌轮回积分无限刷无死亡风险；忌「被封印所以顶点战力为零」；忌「群像/红颜/牙人」代称；忌把玉虚/九幽/元始线提前剧透给一阶开局。  
最早切入：少林杂役院，或首次隐皇堡任务前后。报酬/死亡率/污染（因果）率分阶写清；映射宁低勿高。

**【境界与绝学执行细则（写战斗用）】**

主线钩子/支线：主线=第一次完整轮回任务外围；支线=绝学残本鉴定、仇家名单。  
危险度/规避：中高；规避任务世界内死亡机制。  
任务方向/奖励：开窍丹药级资源、一门入门绝学、轮回积分碎片。

**三阶（外景·隐皇堡阴影）**  
切入身份/时点：外景修士/轮回编外队员，锚定隐皇堡线高潮前后。  
初始事件：堡内刀阵开启，你与**孟奇**被分到同一条「自杀式」试炼走廊，只能合作破局。  
开场白建议：「走廊尽头有人在笑。刀墙上的锈味像旧债。孟奇压低声音：别信第一个让你先走的人。」  
关键NPC立场：**孟奇**、隐皇堡**堡主**、任务中的刀客/自杀者（真名以正典）、**张遇贤**（若已入队）。  
主线钩子/支线：主线=隐皇堡结算与兑换；支线=神兵认主失败样本、同伴背叛测试。  
危险度/规避：高；规避堡主本尊与规则杀。  
任务方向/奖励：外景稳固资源、任务级兵器、积分。

**四阶（内景·江湖名号）**  
切入身份/时点：内景高手/门派客卿，锚定孟奇名动江湖期。  
初始事件：六扇门悬赏与门派灭口令叠发，你被夹在「交人」与「藏人」之间，藏的人是**江芷微**相关线索的持有者。  
开场白建议：「通缉令贴到茶楼门板上时，剑鸣先响。你手中的信只写了一句：见剑如见我。」  
关键NPC立场：**江芷微**、**阮盐**、**方多病**、六扇门主事。  
主线钩子/支线：主线=朝廷—江湖平衡点；支线=绝学真伪鉴定、神兵夺还。  
危险度/规避：高；规避法身以上路过碾压。  
任务方向/奖励：内景辅助秘法、具名神兵线索、势力人情。

**五阶（法身·半神话）**  
切入身份/时点：法身修士/轮回资深队员，锚定法身到传说门槛。  
初始事件：九幽边缘出现「因果债清单」，你的名字与**孟奇**写在同一行，债主未现身。  
开场白建议：「清单是血写的，风一吹却不干。孟奇看你一眼：别还——先查谁有权写债。」  
关键NPC立场：**孟奇**、**顾小桑**、九幽神道使者（真名）、传说级旁观者。  
主线钩子/支线：主线=法身稳固与传说仪式外围；支线=道侣线危机、门派存亡。  
危险度/规避：极高；规避传说本尊决斗。  
任务方向/奖励：法身材料、传说级情报、不越级神器。

**六阶（传说·诸天棋子）**  
切入身份/时点：传说级强者/轮回高层合作者，锚定传说期诸天任务。  
初始事件：轮回积分榜异常跳动，有人用「造化」级诱饵换你出卖**孟奇**的位置。  
开场白建议：「积分像活物一样爬。诱饵很香：一门造化绝学的封面。你知道封面背后是绞索。」  
关键NPC立场：**孟奇**、**六道轮回之主**的代行痕迹（非本尊硬刚）、造化级旁系。  
主线钩子/支线：主线=传说巩固与反棋；支线=神兵归位、同伴救赎。  
危险度/规避：极高；顶点阴影只露边。  
任务方向/奖励：传说级资源、棋盘级情报、存活本身。

**七阶（造化·大劫将启）**  
切入身份/时点：造化级/道门客卿，锚定造化期大劫前夜。  
初始事件：如来金身与元始道体的「比较」流言变成实体投影，城邦级恐慌，你需撤离或夺投影核心。  
开场白建议：「天边两道影子谁也不肯先灭。有人哭有人跪有人记账。你只想问：投影碎了会砸死谁。」  
关键NPC立场：**孟奇**、道门/佛门具名强者、**顾小桑**等核心同伴。  
主线钩子/支线：主线=造化站位；支线=绝学终极残篇、因果清账。  
危险度/规避：贴近顶点；情报优先。  
任务方向/奖励：造化材料线索、终局人情、不发道祖神器。

**八阶（传说之上·一世收束）**  
切入身份/时点：准道祖边缘的见证者/条件胜利执行者，锚定终局主题兑现期。  
初始事件：六道轮回之主棋盘微震，你领到的不是斩杀令，而是「记录一世」的公证任务——写错一字即因果反噬。  
开场白建议：「笔比刀重。轮回之主的影子只在纸角停了一息。孟奇在远处笑：写我，别写神。」  
关键NPC立场：**孟奇**、**六道轮回之主**（超阶：存在·情报优先/条件性胜利）、道祖侧投影。  
主线钩子/支线：主线=终局见证与条件胜；支线=同伴结局安顿、神兵封存。  
危险度/规避：超阶压力；严禁硬刚本尊。  
任务方向/奖励：纪元级记录权、传承线索、存活与「被记住」。
