# 姫骑士诸神黄昏-神器觉醒
<!--meta lib=主库 tiers=六-->
## 剧情

**【作品来源】**
本条目锚定日本 BLACK Lilith 作品《姫騎士リリア 〜魔触の王城に堕つ〜》（2004 年发售，2005 年完全版；PIXY 改编 OVA 六卷，2006–2010）公开世界观，以及乐园侧「姫骑士诸神黄昏」六阶切片组中的**神器觉醒**支线。原作舞台为边境国**ルーフェン王国（鲁芬王国）**，女主**リリア・エーベルヴァイン（莉莉娅·埃贝尔瓦因）**以「白之姫骑士」闻名。原作主线是王子**ディルク・エーベルヴァイン（迪尔克·埃贝尔瓦因）**借魔族之力夺权、以「灵魂能量」偿债；本切片不写成人向陷落细目，而把冲突上移到**王族圣剑认主、王陵试炼、神殿认证权、魔族竞夺与条件性弑神预备**。文风：冷硬黑暗奇幻、宫廷军务、圣物政治、骑士誓约。标签：莉莉娅圣剑、王陵祭坛、铭文拓片、认主试炼、反见证、净刃、克洛维斯战史、白之御使遗产。

**【世界定位】**
大陆西北边境，鲁芬以圣血王族与守护神信仰立国。当神恩反转、信仰政治开裂之后，唯一还能「直接对话神权」的器物是王族圣剑。本切片时点：神堕已被部分确认、供奉草约尚未终裁、圣剑持续烫手。一句话：剑醒了，人必须决定它砍谁——砍神、砍政敌，还是先砍掉自己的幻觉。

**【世界观 · 力量体系】**
世界基本规则：中世纪偏黑暗奇幻。力量来源分四条可并存轨道——**血脉圣恩**（埃贝尔瓦因圣血可共鸣圣器）、**骑士修炼**（剑术、战阵、圣痕刻印）、**神殿权柄**（祝祷、圣水、结界、神谕）、**魔族仪式力**（以堕落之魂能强化魔力；公开设定中魔族战士一骑当千）。死亡：凡人一击即终；正式骑士可凭圣痕续战半刻；神格容器可「神化再生」但自我记忆消融。复活昂贵，通常意味着人格被改写。

本切片特化规则——**神器共鸣**：
- 圣剑在神恩反转环境下会「烫手」：刃温与污染等级正相关，是可量化仪表，不是迷信。
- 觉醒后可短时发动**净刃**：切断局部「神恩反转」效果（敌魔被圣光治愈、友军被圣光灼伤的现象），每次消耗持剑者神智稳定度。
- 觉醒试炼会投影旧日「弑神残影」：要求持有者回答「你杀神，还是成为神」。答「先救人／先守誓」可通过；答「成为神」易滑入容器线；答「无条件杀神」易反噬。
- 认主不等于无限弑神权：净刃有次数与冷却；无锚（无白骑士人间锚、无誓约墙联署）强行全开，会先烧穿持剑者。

成长路径（人类侧）：
1. **民兵·侍从**：冷兵器、盾墙、弓弩。≈一阶。
2. **正式骑士·边境斥候**：小队战、破寨。≈二阶。
3. **圣骑士·团级军官**：端据点、圣痕挡小型魔咒。≈三阶。
4. **王族骑士·姫骑士营精锐**：建筑群级破坏；白之姫骑士常态属此档顶尖。≈四阶。
5. **圣女团长·神使级容器·魔参谋城级网**：一城信仰/情报崩盘级。≈五阶。
6. **堕落神祇本体·完全神化容器·弑神级神器全开**：地区级灾难，凡俗军队近乎无效。≈**六阶（本档）**。

魔族侧：普通魔兵→魔将→大魔将→魔王代理→魔王本体（冻结大地深处，情报优先）。魔族「灵魂能量」经济是边境黑市与政治交易隐形轴心——本切片只写其**竞夺神器与债务改口**后果。

乐园阶位映射（对照破坏力标尺，宁低勿高）：民兵≈一阶；正式骑士≈二阶；圣骑士≈三阶；王族精锐/白之姫骑士常态≈四阶；圣女/神使/城级信仰崩≈五阶；**神器完全觉醒、净刃对神格、弑神预备战、堕落神祇近身对峙≈六阶（本档覆盖）**。世界顶点＝堕落态守护神完全体与冻结大地魔王级存在，属跨阶/贴近超阶，**情报优先、条件性胜利**，严禁战力归零式削弱。

**【地理 · 舞台】**
- **ルーフェン王国（鲁芬）**：大陆西北边境农耕国，暖流使气候偏暖；西临海路，北接**凍結大地（冻土）**，东南邻**ザクセン王国（萨克森）**。建国传说：约千年前**辺境戦争**中，圣人**クリストフ・エーベルヴァイン（克里斯托夫·埃贝尔瓦因）**号「白之御使」，驱魔族入北土并建鲁芬为「封盖」。
- **王陵与圣剑祭坛**：本切片第一现场。陵道有旧血槽；插入圣剑可唤醒古语回音。祭坛法要求：王族血＋白骑士见证＋神殿反见证。
- **王都圣殿主堂**：神像裂痕与圣剑对鸣的名场面可能发生处；神殿宣称唯己可认证神器。
- **姫骑士营与誓约墙**：莉莉娅直属精锐驻防；认主后剑名可登记于此，抗「王室武库令」。
- **王城议政厅／武库**：迪尔克要把觉醒剑登记为「王室武器」实控于己；钥匙与编制令在此争夺。
- **神殿地下圣骸库／仪式厅**：旧日封存残片、记谱与伪神谕抄本可能出处。
- **黒の楽園館（黑乐园馆）**：雾子据点；魔族竞价、拓片黑市、私兵调度（仅作情报与竞价节点）。
- **克洛维斯峡谷**：上一年五千破三万的战场；战史与军报抄本支撑「谁有权持剑」的舆论。
- **修道院侧殿**：纯信者通道；若认主失败，容器神谕会改口「剑已弃她」。
- **萨克森边境诸堡**：若鲁芬公开「可伤神之刃」，邻国既恐又馋。

**【世界剧情线】**

① **前史：白之御使与圣剑**  
千年前辺境戦争，克里斯托夫以「奇迹」与团结反胜，驱魔族入冻结大地，建鲁芬为封盖。教义称王族圣剑为「白之御使余火」：平时只是锋利的王族佩剑；在神权崩坏时才会「烫、鸣、问」。神殿旧派把剑解释为「神恩导管」；骑士传统把剑解释为「守门人的刃」。两种解释在本切片正面冲突。

② **近代：克洛维斯与人望**  
上一年克洛维斯峡谷，莉莉娅率约五千破萨克森逾三万，斩名将**アドラー（阿德勒）**。战后民谣把她写成「白之御使再临」。人望压过被讥为「废物皇子」的迪尔克，也让神殿更急于把圣血叙事收归教义。圣剑在此役中未「觉醒」，只完成军事胜利——说明觉醒触发器不是胜仗，而是**神权病态**。

③ **并行暗线：王病、魔族信贷与神堕**  
父王病倒（游戏线）或宫廷变故（OVA 线存在弑父夺权描写）。迪尔克经**キリコ（雾子）**接触魔族，换权柄与私兵，欠下须以「灵魂能量」偿的债。诸神黄昏组前切片中，神恩反转、神像黑金、容器神谕已陆续出现。本切片承接：圣剑刃温持续上升，宫廷医师**弥撒**记录「刃温—黑金样本等级」正相关曲线。

④ **本切片起点：圣剑异变（T0）**  
莉莉娅夜巡时发现：剑出鞘半寸，主堂神像裂痕收缩又扩张——**剑与神互相定义**。刃身浮现古王铭文，短暂可切断「神恩污染」，也会切断施术者神智。米娅制表，弗兰克核签字：只收可验证战报，不收酒馆神迹。迪尔克把同一批战报改成「考验成功」送神殿。契约者若持有未改原件，可在议政厅一锤定音。

⑤ **法理战：谁有权让剑醒（T+12h）**  
古法：王陵试剑须王族血＋白骑士见证＋神殿反见证。大主教**赛尔**派祭司**约书**阻拦：「唯神殿可认证神器，否则为邪物。」圣骑士团长**加雷斯**派米娅护送：「剑是军务，不是夜仪。」议长**赫尔曼**提出「双认证」——神殿＋骑士团，给契约者斡旋位。迪尔克签发**武库登记预备令**：觉醒后剑归王室武库，钥匙在摄政。雾子同时收购两种版本的拓片行情：「能让剑恨谁，就能卖谁。」

⑥ **王陵试剑夜（T+24h）**  
陵道血槽见旧锈。回音用古语提问，先问莉莉娅，也问在场契约者。若迪尔克在场，回音会刺他的名——血脉不纯且有魔契。白骑士四人立四方作「人间锚」：东弗兰克、西沙姆洛克、南梅查、北卢滕（伤愈后仍争北门）；米娅掌誓约与真名。试炼投影弑神残影：画面是旧日封盖之战的碎片，不是全知史书。答错则刃温暴涨、持剑者神智眩；答「先守人、后问神」可过。

⑦ **名场面：剑与神像对鸣（T+36h，可与陵试交错）**  
主堂内圣剑出鞘半寸，神像裂痕一缩一扩。莉莉娅选择**收回剑**，不在无计划时砍神像——这是纪律，不是懦弱。赛尔据此指控「畏神」；加雷斯据此赞「懂战场节奏」。弥撒当众读刃温曲线：收回后刃温下降，证明「不拔」是保留净刃次数的战术。

⑧ **魔族竞夺（T+36~48h）**  
魔王使者**サンド（桑德）**上报：人间出现可伤神格的刃。报价单出现「优先获取」类条目（公开设定中桑德为催债/传令将官级）。雾子开价收购拓片与护送权。萨克森间谍散布「鲁芬有弑神刃」小册子——既想吓退主战派，又想诱使鲁芬内斗。正确应对：拓片仅两份（莉莉娅＋可信保险箱），第三份出现＝泄密。

⑨ **认主政治三终局（T+72h）**  
- **A. 剑认主莉莉娅**：登记在白骑士誓约墙；迪尔克武库令作废；为后续弑神/拒神位线铺路。默认优结局。  
- **B. 剑被迪尔克封印武库**：钥匙在摄政，净刃变成「需批文的武器」，神智消耗转嫁给执行骑士。  
- **C. 剑被神殿宣布为神物收走**：夜仪合法化，剑成为新神容器的配套圣物。  
条件性胜利清单（占多数即优）：①认主完成且在莉莉娅侧；②拓片仅两份可互校；③武库令作废或附则被当众拆穿；④拒桑德「护送」；⑤莉莉娅神智稳定度未崩；⑥双认证或等值法理通过。禁止写「剑无限弑神、神已死透」。

⑩ **与原作主线的衔接**  
原作中莉莉娅以武人自尊与绝望摆动；OVA 终局有离反魔王、率残存白骑士开战的方向。诸神黄昏线把「离反」放大为**拒容器／可伤神的刃在谁手里**。神器觉醒不替代迪尔克偿债线，但会改写其杠杆：若剑在莉莉娅侧，雾子更难把「新秩序」卖成唯一解；若剑在武库或神殿，神化线与制度化供奉会更容易。

**【主要人物】**
- **リリア・エーベルヴァイン / 莉莉娅·埃贝尔瓦因（白之姫骑士·王女）**｜性格：刚直、重誓、战场果断，蔑视迪尔克的软弱与暴虐；白衣姫骑士的自尊极强。｜装备·能力：王族圣剑、圣血共鸣、白骑士指挥权；克洛维斯战绩属四阶顶尖；觉醒后获短时净刃与弑神预备资格，但受神智稳定度约束。｜人物弧光：信剑如信神 → 疑神而问剑 → 认主后在「守人」与「弑神」之间立誓。｜立场关系：对契约者先试誓言与物证；对迪尔克敌对；对蕾拉孝且痛；对白骑士托命。
- **ディルク・エーベルヴァイン / 迪尔克·埃贝尔瓦因（王子）**｜性格：劣等感驱动的享乐与恐惧并存；公开资料中游戏版丑角感强，OVA 更近小坏蛋。｜装备·能力：王权名分、魔族信贷、武库令与全国动员（OVA）。｜人物弧光：被嘲废物皇子 → 借外力夺权 → 企图把神器登记为私兵权杖。｜立场关系：要登记权；可交易情报，不可信誓。
- **レイラ・エーベルヴァイン / 蕾拉·埃贝尔瓦因（王妃·辺境之至宝）**｜性格：温雅、信性善、对继子亦不忍弃。｜装备·能力：宫廷地位、人望缓冲、外交礼仪渠道。｜人物弧光：和平象征 → 试剑夜的情感软肋与舆论盾。｜立场关系：保护对象；家书可让莉莉娅剑尖垂三寸，也可被迪尔克当软肋。
- **キリコ / 雾子（魔族·黒の楽園館主人）**｜性格：娼馆女主人的冷笑与参谋的算计，把道德当可计价商品。｜装备·能力：魔族身份、贵族人脉、数千私兵传闻、拓片黑市。｜人物弧光：扶迪尔克 → 神器竞价商 → 可能因报价失败被魔王侧清算。｜立场关系：最危险的「可结盟灰」；卖版本不卖真相。
- **サンド / 桑德（魔王使者）**｜性格：绷带下的锐眼，谈判优先于嗜杀。｜装备·能力：将官级交涉权、对价回收、优先获取令。｜人物弧光：收债人 → 神器竞夺表上的魔族标尺。｜立场关系：情报优先；勿无准备开战；勿接受其「护送」。
- **ミーア / 米娅（白骑士·政委型副手）**｜性格：忠、共情、敢在莉莉娅崩溃时拉住她。｜装备·能力：白骑士从属战力、誓约墙法令、真名管理。｜人物弧光：好友 → 试剑见证 → 认主联署关键人。｜立场关系：可靠盟友；陵门对峙时用法理挡神殿军。
- **フランク / 弗兰克、シャムロック / 沙姆洛克、ルッテン / 卢滕、メッツァー / 梅查（白骑士四人）**｜性格：各异但共誓；东西南北位人间锚。｜装备·能力：团战、阵型、陵试站位。｜人物弧光：从战友变为「剑是否仍有可护之人」的活证据。｜立场关系：可征召作战单元；缺一则试炼锚不稳。
- **クリストフ・エーベルヴァイン / 克里斯托夫（白之御使·建国圣人·传说）**｜性格：传说中的团结者与奇迹执行者。｜装备·能力：历史权柄、圣血源头、圣剑神话合法性。｜人物弧光：已成教义符号；其「余火」是否本就可反噬神恩，是隐藏线。｜立场关系：所有派系都要借他的名解释剑。
- **堕落神祇（原鲁芬守护神·顶点之一）**｜性格：优雅的病、索要容器与供奉，厌恶「可伤己的刃」落在不听话者手中。｜装备·能力：信仰改写、神恩反转、裂痕共鸣。｜人物弧光：裂痕 → 对鸣 → 企图让剑改认「神物」。｜立场关系：不可默认斩杀；净刃只换时间与条件。
- **大主教·赛尔**｜性格：表象维护者，词频里「考验」过高。｜装备·能力：神殿军、认证权叙事、圣歌谱系。｜人物弧光：反见证派 → 可能转向「神物收缴」。｜立场关系：法理敌手。
- **改革派祭司·约书**｜性格：真心信新神更诚实，危险在于真诚。｜装备·能力：夜仪、变调圣歌、陵门阻拦令。｜人物弧光：反见证执行者 → 可辩可抓的证人。｜立场关系：审讯靠逻辑。

**【叙事基调 · 雷区】**
基调：神权冷硬、骑士誓、军务会计、圣物政治、悲剧英雄。对话要短，物证要硬，胜利要条件化。  
忌：把莉莉娅写成无脑斩神机器；把迪尔克写成纯丑角而无恐惧；把觉醒写成无限外挂；把成人向写成细目展览；把顶点写成可秒杀。  
忌：战力归零削弱顶点；跨世界拍卖护送套话；用无名群像替代具名 NPC；用「安涅罗泽／米琪可／巴尔德」等外世界名污染本库。  
最早锚点：圣剑烫手被医师记录之夜，或王陵试剑公告日。契约者入场应已能摸到刃温表的数字。

**【神器觉醒专项 · 刃温与污染仪表盘】**
圣剑烫手、神纹跳闪、黑金烟色，三者构成污染仪表盘。宫廷医师弥撒要求：每次出鞘记录刃温、在场者、法令编号。赛尔若篡改记录使「刃温突然正常」，那是政治不是奇迹。莉莉娅规定非战时不拔，是为了保留神智稳定度与净刃次数。AI 写战斗前先写仪表读数。

**【神器觉醒专项 · 王陵试炼问答库】**
残影提问核心只有一句：「你杀神，还是成为神？」可扩展为三连问：①你要保护的名字；②你肯付的代价；③你是否接受「剑也会恨你」。标准通过句式强调**守人优先**。答「成为神」会推容器进度；答「无条件杀神」触发反噬小震。契约者若抢答，回音会要求其出示人间锚（白骑士在场证明）。

**【神器觉醒专项 · 人间锚与四方站位】**
弗兰克、沙姆洛克、梅查、卢滕不是龙套，是试炼几何学：剑要确认持有者仍有可护之人。任一方缺席，试炼难度上升；两人以上缺席，建议改期。米娅掌真名名册，防止「替身骑士」混入。加雷斯若强行军事化接管站位，会把试炼变成阅兵——残影不认阅兵。

**【神器觉醒专项 · 拓片水印与泄密链】**
合法拓片仅两份：一份入莉莉娅军务匣，一份入第三方保险箱（弥撒或契约者联署匣）。水印互校：缺角、纤维、圣血微点三要素。第三份出现即启动泄密链：印刷所→商会→雾子→桑德。玛丽亚擅长查水印；赫尔曼擅长查谁投了「公开拓片」的票。

**【神器觉醒专项 · 武库令与附则战】**
迪尔克的武库登记预备令正文写「保护国器」，附则写「摄政可指定持剑执行人」。赫尔曼可被买通朗读时跳过附则。条件性胜利要求：**当众完整朗读**并双份书记记录，或直接表决作废。只撕正文不撕附则＝伪胜。

**【神器觉醒专项 · 双认证投票算术】**
临时法示例：骑士团三票、神殿两票、王室观察一票（无否决）。赛尔会要求对调；约书会要求「神谕加票」。契约者斡旋目标是保住骑士侧多数，而不是羞辱神殿——羞辱会逼出神物收缴的 C 终局。

**【神器觉醒专项 · 净刃使用纪律】**
净刃可短时切断神恩反转：战地表现为「敌魔不再被圣光治愈、友军不再被圣光灼伤」。每次使用后弥撒测神智：眩晕、失名、短暂把战友看成「容器候选」均为红线。战场节奏：先疏散平民，再精锐突入，再计数净刃，最后才考虑对神像或神影的条件打击。禁止连斩炫技。

**【神器觉醒专项 · 魔族合同法视角】**
桑德报告不写「人类勇敢」，写「能量通量／风险／可替代抵押」。神器出现会让他评估：是摧毁、买下、还是诱使持剑者神智崩溃后回收。与魔族谈判要带赫尔曼（合同）与加雷斯（武力）同时在场；只带骑士语言会被笑死。拒绝「护送」是红线——护送＝押运。

**【神器觉醒专项 · 条件性胜利判定表】**
1. 认主完成且剑登记在莉莉娅／誓约墙侧。  
2. 拓片合法双份，无第三份流通。  
3. 武库令作废或附则当众拆穿。  
4. 拒绝桑德护送与雾子独家收购。  
5. 莉莉娅神智稳定度可战（未失名）。  
6. 双认证或等值法理通过，神殿不能单方面称邪物。  
满足四项以上为优；三项为可接受；两项以下为劣稳态（B/C 终局倾向）。劣稳态仍可玩：剑在武库则转「盗钥／再试剑」；剑在神殿则转「神物夺还／防神化」。

**【神器觉醒专项 · 失败态转进】**
- 武库封印：净刃变批文武器，白骑士执行时神智消耗由基层承担，军心裂。  
- 神物收缴：夜仪半公开，剑成为容器配套，后续「莉莉娅神化」难度上升。  
- 拓片泄密：萨克森与魔族同步加价，王都粮价与圣水汇票双崩。  
- 试炼反噬：莉莉娅短暂失名，米娅必须代持誓约——代持超过时限会触发「剑恨主」事件。  
失败不是删档，而是更糟的可玩稳态。

**【战争与骑士团运行细则】**
姫骑士营以白骑士为核，克洛维斯后兵员未补满，却被要求同时守主堂、守陵、防萨克森假旗劫粮。加雷斯若把觉醒剑军事化，会与赛尔的神物化对撞。契约者可提供的不是替他们练兵，而是**可验证的认主程序与敌我识别**：谁能碰剑匣、谁能看拓片、谁能在净刃窗口指挥疏散。

**【信仰经济与拓片行情】**
圣水、祝祷券、战伤治愈是鲁芬信任货币。神器觉醒新闻会让黑市同时出现「求购拓片」与「求购伪神谕」。议会若禁拓，雾子改走口头拍卖；若公开拓，桑德直接上门。正确是**双份密封＋公开摘要**：摘要只写「刃可短时净反恩」，不写次数与站位细节。

**【迪尔克线与神器线的交叉火力】**
若只帮莉莉娅试剑不碰武库令，迪尔克会在认主次日用附则把剑「借走」。若只废武库令不完成认主，神殿会称邪物收缴。正确路径往往是**同场闭环**：试剑→联署→废令→封拓，四步在同一听证日照完。雾子最怕同一日照完，因为她来不及改价。

**【萨克森机会窗口】**
阿德勒死后主战派短暂失势，「鲁芬有弑神刃」可重启预算。边境假冒白骑士旗的劫粮队会试探：真莉莉娅是否还敢持剑露面。露面是舆论必需，也是容器与竞夺的诱饵——露面必须带人间锚与弥撒随行。

**【冻结大地与封盖国】**
鲁芬合法性来自守门。神器觉醒被魔族解读为「盖子上出现了钥匙」。冻土线若在试剑周南移，说明门外在加压。六阶时间压力不是残影台词，而是**封盖失效速度与净刃次数的赛跑**。

**【宫廷文书战与语言选择】**
莉莉娅与迪尔克对峙很少用脏字，用编制与法令：谁有权调白骑士，谁有权签武库，谁有权反见证。赛尔说话像念经，约书像写论文，赫尔曼像算账，加雷斯像点兵。契约者插入对话须先选一种语言——经、论文、账、兵——否则会被当外行赶出陵门会议。

**【情感操作与战略投资】**
救蕾拉、护米娅、给白骑士真名空间，是战略投资：直接回稳莉莉娅神智与人间锚。公开羞辱至亲会同时喂迪尔克对价与神格容器进度。成人向危险以「俘获／胁迫／国破」呈现，不写细目。

**【与乐园契约者的交互边界】**
土著默认不知「契约者」体系；对外身份用王陵史官／试炼见证人／编制外军事顾问。六阶战力可碾压普通骑士，但**不可**靠一人无限净刃通关——顶点规则是情报与条件。向莉莉娅证明可信的方式是物证、誓言与守人优先的答词，不是展示跨世界神通。向雾子交易时拒绝「灵魂」报价，改为情报／通道／物资。

加雷斯：「给我站位图，我给你活门。」  
赫尔曼：「附则第三条，摄政可指定持剑人……」——此处必须被打断或完整朗读。  
弥撒：「刃温与黑金同级。别让我重复第三次。」  
雾子：「能让剑恨谁，就能卖谁。」  
桑德：「优先获取。护送另计。」  
残影：「你杀神，还是成为神？」  
莉莉娅（通过句）：「我先守还活着的人。」

**【社会纹理 · 可观察细节】**
粮价随「剑醒」新闻波动；娼馆与神殿在同一条街上收不同的税；白骑士的剑油比圣油紧俏。孩子玩「公主砍神」游戏会被神殿禁，却在巷弄里越禁越传。契约者走进酒馆先听歌——军歌多还是圣歌多，决定今晚拓片情报价。

**【白骑士军制补记】**
四人不是背景板：缺东位，陵试易被神殿军从晨光侧压门；缺北位，卢滕旧伤会成为迪尔克攻击「营务废弛」的口实。米娅是政委型副手，管士气与真名。没有他们，莉莉娅的剑只是个人英雄主义，残影不认。

**【神恩反转战例（可回调）】**
某次边境冲突中，敌魔族伤势愈合，我方骑士被「圣光」灼伤。战报被赛尔压下，被玛丽亚式记录官抄出。觉醒后的净刃第一次实战建议选这类「可验证小战场」，而不是主堂斩神秀。


## 阶位切入点

> 阶位↔原作：民兵≈一阶；正式骑士≈二阶；圣骑士≈三阶；王族精锐/白之姫骑士常态≈四阶；圣女·神使·城级信仰崩≈五阶；**圣剑完全觉醒·净刃对神格·认主政治·弑神预备≈六阶**。本世界仅覆盖：**六阶**。世界顶点（堕落神祇完全体 / 冻结大地魔王级）为跨阶存在，**超阶巅峰：存在·情报优先/条件性胜利**，严禁硬刚本体或战力归零式削弱。

**六阶（地区灾难 · 圣剑认主与神器政治）**

切入身份/时点：契约者以乐园编制外「王陵史官 / 试炼见证人 / 姫骑士营外援顾问」切入，锚定 **T0 圣剑异变已记录、T+12h 法理战开打、T+24h 王陵试剑夜将临** 的窗口。本地核验编制外徽章；无徽章先被当间谍拷问三轮，拷问题库＝本切片物证名录（刃温编号／拓片水印／武库令条文／联署格式／报价单抬头）。

初始事件：王陵外三方对峙。**约书**带神殿军堵陵门，宣称未认证即为邪物；**米娅**以誓约墙法令要求放行见证；**迪尔克**使者宣读武库登记预备令；侧廊烟味里**雾子**的人在收拓片预付款。陵内回音已开始用古语试探门槛。你必须在开门前表态：见证认主、支持反见证、还是「先查武库附则」。

开场白建议：「剑在问你要砍谁。神像在裂，刃温表上的数字比圣歌响。陵门左边是神殿的反见证，右边是摄政的武库令，中间是白骑士的誓约。你若把拓片卖给侧廊的烟味女人，整座王都明天会改价——改的不是粮，是谁配活在剑的保护里。」

关键NPC立场：**莉莉娅·埃贝尔瓦因**试炼主体，先听你答「守人还是成神」；**米娅**见证与法理盾；**约书**反见证执行；**赛尔**要认证垄断；**迪尔克·埃贝尔瓦因**要登记权与钥匙；**雾子**要收购与版本；**桑德**要优先获取与护送；**加雷斯**要军事化站位；**赫尔曼**要双认证算术；**弥撒**要曲线与药剂；**玛丽亚**要伪拓水印；堕落神祇只给对鸣条件不给斩杀。

主线钩子/支线：主线＝**完成认主并使剑留在莉莉娅／誓约墙侧**，同时废武库令或拆附则。支线＝①双份拓片水印互校；②当众朗读武库附则；③拒桑德护送并曝光报价单；④四方人间锚到齐；⑤试炼答词记录归档；⑥弥撒神智稳定药剂分配；⑦防萨克森假白骑士旗逼莉莉娅单独露面；⑧刃温—黑金曲线写入军务日志。

危险度/规避：极高 / 贴近顶点。必须规避：无锚强行试炼；把剑「暂交」迪尔克保管；接受桑德护送；无计划在主堂砍神像；向雾子出售第三份拓片；让莉莉娅失名后仍连开净刃。失败死亡率：无锚试炼反噬≈持剑者失能；陵门武装冲突无认证≈团灭级政治化屠杀指控。

任务方向/奖励：自然目标是认主闭环＋拓片双份＋武库令失效。奖励取向：铭文拓片（合法份）、短时净刃权（非无限）、誓约墙联署、王陵通行、刃温鉴定权、六阶神器危机档案、编制外顾问转正令（本地）。不发「弑神成功」或「剑无限」。

【六阶执行细则·身份核验】入场须乐园任务印＋本地徽章。答错物证名＝敌对标记，白骑士先缴械再问话；拷问可转情报交易。

【六阶执行细则·物证优先】每幕至少一件可验证物证。纯砍杀不推进认主状态。拓片要水印，令要双份，答词要记录。

【六阶执行细则·顶点交互】与堕落神祇／残影对话默认记录原话、条件、索取；净刃只换时间与局部反恩切断，不默认斩杀完全体。

【六阶执行细则·队友红线】四方锚到齐再试剑；不把米娅单独留在陵门与神殿军对峙过夜；药剂优先莉莉娅与失名风险者。

【六阶执行细则·失败态】主线失败则转入 B/C 稳态（武库封印或神物收缴），仍可玩但奖励降档，并抬高后续神化线难度。

【六阶支线池】伪拓追查；附则朗读；报价单原件；假白骑士旗；冻土线测绘；克里斯托夫圣物真伪；克洛维斯军报舆论战；神智药剂防篡改。

【六阶时序锚】T0 异变；T+12h 法理战；T+24h 陵试；T+36h 竞夺与对鸣；T+72h 三终局。

【六阶结算关键词】回合总结须出现：认主、拓片、武库、净刃、人间锚、反见证（至少四项）。

【六阶战场节奏】开场三十分钟内卷入陵门或主堂；中段物证推进；末段条件性胜利判定而非斩首秀。

【六阶禁令】禁止战力归零削弱顶点；禁止跨世界套话任务；禁止无名群像；禁止外世界角色名；禁止色情细目。
