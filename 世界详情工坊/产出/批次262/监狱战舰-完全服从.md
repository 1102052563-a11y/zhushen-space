# 监狱战舰-完全服从
<!--meta lib=主库 tiers=二-->

## 剧情

**【作品来源】**
《監獄戦艦 〜非道の洗脳改造航海〜》（中译常见「监狱战舰」）为 LiLiTH／ANIME LiLiTH 于 2007年3月发售的太空歌剧向 18禁 ADV；原画カガミ，监督笹＠，系列后有 IF、2、3、アカデミア 等。本档为分锚条目「**完全服从**」，以第1作主线公开设定为准：宇宙联邦军少佐**ドニ・ボーガン**舰长要人运输舰**ジャサント号**，以「正常意识时段／暗示异常时段」交替推进洗脑，时限约一至二周。气质：硬核太空歌剧＋洗脑时限赛跑。成人向只写舰务、阵营、时限与任务介入，不写色情过程。

**【世界定位】**
太阳系已部分テラフォーミング；**ネオ・テラーズ**（地球系）与**ニュー・ソラル**（外惑星系宇宙人类）对立未消。本档锚点：**完成度末段女军官可被口令驱动但仍有短暂清醒窗**。一句话：在ジャサント号蓝光夜航里，女军官的军阶与自我在时限表上赛跑。

**【世界观 · 力量体系】**
力量本源：宇宙战舰火力、フェイザー／鱼雷、单兵ブラスター、洗脑手术室（药物＋ナノマシン＋触发暗示）、装甲与舰内权限。死亡：凡人与士兵战死即灭；芯片保存人格有「魂の消失」期限（约最长三年级设定），非无限复活。特殊系统：
- **洗脑完成度 0–100%** 与**时限日数**双轨；
- **触发器**：第1作舰内 22 时后自动切**蓝色照明**；
- **正常时段必须保留**——否则无法完成舰务与欺骗上层，这是可利用的规则缝。

力量阶梯（宁低勿高）：
1. **舰内杂役／新兵**：近战与门禁，≈一阶。
2. **正规士兵／宪兵／本档女军官初始可战态**：破门、舷廊枪战、单兵火力，≈**二阶**（本档主覆盖）。
3. **精锐陆战队／洗脑技术班**：舰内攻坚，≈三阶边缘。
4. **舰长ドニ・ボーガン及其亲信指挥网**：权限＋情报＋手术室，≈三～四阶边缘（作压迫者）。
5. **主力舰队会战级火力／装甲骑后期系列**：跨档，本档二阶不发。
6. **阵营战争全局／后续ディ・エルデ级**：超阶棋盘，情报优先。

**乐园阶位映射**：杂役≈一阶；女军官与正规士兵舷廊战≈**二阶**；技术班与精锐≈三阶边缘；ボーガン指挥网≈三～四阶边缘；舰队会战与后期组织≈更高或超阶。本任务世界仅覆盖**二阶**；规避舰队决战与后期顶点组织。

**【地理 · 舞台】**
- **ジャサント号**：要人运输新锐舰；舰首フェザー炮与鱼雷；舰长室、洗脑手术室、蓝色夜航走廊、弹药库、通信室。
- **土星～地球航路**：公开「护送要人」、实际绑架／暗杀任务航线。
- **ニュー・ソラル势力圈（土星侧）**：**リエリ・ビショップ**等所属阵营后方。
- **ネオ・テラーズ地球侧**：ボーガン表面效忠的联邦军体系。
- **接舷／拿捕点**：BAD线中ニュー・ソラル舰队接舷处（时限失败的公共后果）。
写场景：蓝光、金属回声、权限门嘀声、手术室消毒味、舰钟与时限表。

**【世界剧情线】**
① **接舷与俘获（女军官被俘锚点前史）**  
ドニ・ボーガン以护送为名接近ニュー・ソラル女军官团。**リエリ・ビショップ**（冷静、军纪、对地球侧高度警惕）与**ナオミ・エヴァンス**（豪爽、战斗型）等被设计俘上ジャサント号。公开日志写「要人护送」，内部日志写「改造对象收容」。

② **时限启动：蓝光与正常时段**  
22 时后蓝光触发异常人格窗口；白天必须维持可出舰务报告的「正常」。时限约 1–2 周，完成度未满则女军官可能反杀或引来友军接舷。

③ **本档锚点：完全服从**  

本档聚焦时限**末段**：完成度接近 100%，公开日志已可把女军官写成「自愿协助」。卡片锚点「完全服从」＝**军阶自我被权限系统与触发器压到可远程口令驱动，但仍可能残留 3 秒战术清醒**。可玩：在 3 秒清醒窗注入反向口令、抢芯片人格备份、呼叫接舷前伪造「舰将自毁」逼ボーガン分神。禁止写「永久无自我」。

④ **多结局分轨（公开结构）**  
Normal／Good／Bad：完成度与是否被ニュー・ソラル拿捕、女军官是否反制、舰是否被接舷。Bad 线暗示洗脑仍可在狱中继续——续作设定中ボーガン可再出。

⑤ **蝴蝶效应边界**  
契约者可改：藏起触发器密钥、伪造完成度报告、救出一名未 saturation 的军官、切断手术室供电。不可无代价改：太阳系两大阵营战争总盘、ボーガン在系列中的长期棋手位。

**【主要人物】**
- **ドニ・ボーガン（ジャサント号舰长／联邦军少佐）**｜性格：冷酷、算计、把洗脑当任务工具。｜装备·能力：舰长权限、手术室、亲信班、单兵火力平庸但信息权极高。｜人物弧光：俘获女军官→时限内完成改造→（系列）越狱与更大组织。｜立场：对契约者先利用后灭口；软肋是完成度报表与舰队审查窗口。
- **リエリ・ビショップ（ニュー・ソラル女军官）**｜性格：冷静、自尊、军纪铁。｜装备·能力：军用小火器、战术素养、舰内近战≈二阶。｜人物弧光：被俘→抵抗蓝光暗示→完成度攀升或反杀窗口。｜立场：对外来者先审后用；软肋是同伴与军旗名誉。
- **ナオミ・エヴァンス（ニュー・ソラル女军官／战斗型）**｜性格：豪爽、好战、护同伴。｜装备·能力：近战与枪斗≈二阶。｜人物弧光：被俘→以武力突围未遂→洗脑压力测试。｜立场：可结盟突围；冲动是把柄。
- **ハンス／ウーログ／インゲボリ／マグヌス（舰员班，公开资料点名）**｜性格：各异，执行层。｜能力：舰务与看守。｜弧光：Bad线爆炸死亡与续作「部下损失」叙事差异——任务中以「亲信班」职能写入，勿强行锁死存亡。｜立场：可贿可袭。
- **アマダ（リエリ相关姻缘线，IF设定）**｜性格：不详从略。｜能力：后方人事。｜弧光：IF「如果结婚」支线；本档正史不强制。｜立场：情报侧。
- **ニュー・ソラル接舷指挥官（舰队侧）**｜性格：军务优先。｜能力：舰队火力远超二阶。｜弧光：时限失败时的拿捕方。｜立场：条件性胜利靠信号不靠击沉己方目标舰硬刚。
- **ネオ・テラーズ审查官**｜性格：要脸与报表。｜能力：行政权。｜弧光：白天「正常时段」的外部压力。｜立场：可被伪造报表欺骗。
- **洗脑手术室技官**｜性格：职业冷漠。｜能力：药物与ナノマシン操作。｜弧光：完成度的手。｜立场：断电／抢药剂可拖时限。
- **被俘勤务兵／情报士官（具名不详则写职能）**｜性格：怕。｜能力：无。｜弧光：世界残酷标尺。｜立场：可护可弃。
- **契约者可结盟的灰市零件商（靠港时）**｜性格：见钱。｜能力：假证件与屏蔽贴片。｜弧光：支线池。｜立场：给钱办事。

**【隐藏剧情 · 伏笔】**
- ジャサント号「护送」是假皮，暗杀／绑架才是真任务。
- Bad 线拿捕后洗脑仍可继续——「入狱≠安全」。
- 续作中ボーガン可再出，本档二阶只结算本舰时限，不结算太阳系战争。
- IF 线婚姻与正史情报需求冲突：正史里リエリ公开再婚被禁令约束（情报便利）。

**【大事记时间线】**
两大阵营对立固定 → ボーガン就任ジャサント舰长 → 女军官被俘上舰 → 蓝光时限启动 → 本档锚点（公开／服从／俘获阶段） → 完成度结算或接舷 → 多结局分轨 → （系列）越狱与更大棋盘。

**【叙事基调 · 雷区】**
冷硬太空歌剧、时限焦虑、军阶与自我的撕裂。忌写成无脑肉搏超级英雄；忌二阶击沉主力舰队；忌「被封印所以舰长战力为零」；忌把女军官写成无意志道具——完成度未满时必须保留战术判断力。最早切入：被俘后第一次蓝光前夜，或完成度过半、审查官要报表的早晨。

**【服从·3秒窗】**蓝光峰值后常有 3 秒回拉；预埋指令必须短：如「关主阀」「扔密钥」。

**【服从·芯片】**元人格可进芯片；「魂の消失」长期风险——本档二阶只抢舱，不修三年。

**【服从·自愿协助文件】**电子签可伪；纸质签要指纹——双轨验证。

**【服从·接舷倒计时】**发出真坐标后舰队仍需时间；完全服从阶段呼叫＝赌ボーガン是否毁证。

**【服从·条件性胜利】**芯片备份离舰＋至少一次 3 秒窗成功指令＋自愿文件双轨作废。

**【服从·失败态】**芯片「魂消失」倒计时被加速（药物）；不可逆数据化——正文以「认不出军靴敲击节奏」表现。

**【服从·Naomi／Lieri差异】**Naomi 窗更短更爆；Lieri 窗更稳可传完整句。

**【服从·ボーガン分心】**他最怕审查与接舷同时；双线报警可造窗。

**【服从·契约者皮】**手术室见习、黑匣子技术、叛变亲信。

**【服从·禁杀】**击杀女军官以「免其服从」＝任务失败（护人原则）。

**【服从·蓝光主控】**毁主控会全舰节律崩；可作终局前威胁筹码。

**【服从·续作钩】**即使本档胜，ボーガン系列可再出——结算只到本舰。
