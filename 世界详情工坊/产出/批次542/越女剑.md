# 越女剑

## 剧情

**【作品来源】**
《越女剑》为金庸（查良镛）武侠短篇，约一万六千至一万九千字，是金庸全部武侠作品中**历史背景最早**（春秋末期吴越争霸）、**篇幅最短**的一部。1970年1月起在香港《明报晚报》创刊连载，时《鹿鼎记》尚在连载，作者将其视为独立短篇，常被并称「封笔前后」的收束之作之一。后收入明河社、广州出版社、远流等金庸作品集；英文通译 *Sword of the Yue Maiden*。题材取自《吴越春秋》《越绝书》等古籍中「越女试剑」「南林处女」传说，与勾践、夫差、范蠡、西施等史传人物交织，金庸原创阿青、白猿「白公公」情感与剑术戏核。文风：史诗短制、家国复仇与个人爱恨并置，剑气清冷而悲剧收束；无长篇门派江湖结构。衍生：1986年香港亚视二十集《越女剑》（李赛凤饰阿青、岳华饰范蠡）等。**本档案以小说正文为准**，历史背景仅作舞台扩写，不另编原作没有的情节。全本检索可参考百科与作品集页；剧情为公开梗概与分场总结，不整章粘贴原文。

**【世界定位】**
春秋末年东南水国：吴强越弱、卧薪尝胆、美人计与铸剑争锋的人间史剧。叙事核心是牧羊女**阿青**以竹棒悟出的无双剑术，被大夫**范蠡**发现并注入越军，最终助**勾践**灭吴；同时阿青单恋范蠡，在灭吴后欲杀情敌**西施**却被其绝色震慑，以劲气误伤成就「西子捧心」，黯然隐退。一句话给AI：**吴越争霸史台上的「越女剑」——天下最强剑术属于最不通世故的少女，复仇成功之日即是个人悲歌落幕之时。**

**【世界观 · 力量体系】**
本世界**无仙道、无内丹境界表、无系统、无复活**。力量来源为：**兵刃锋利度 + 剑术身法 + 战阵协同 + 极少数近乎本能的「劲气」外放**。生死残酷：一剑断腕、戳目、劈羊、乱军取首皆可当场致命；重伤靠医治与休养，气绝不可回。吴越两国皆以水师、步卒、青铜/精钢剑戟为常；「剑利」与「术精」同等重要——宝剑可断轻纱而不折，劣剑再好招式也难敌锋。

战力分层（原作无官方境界名，按破坏力与地位归纳，宁低勿高）：
1. **庶民 / 牧羊 / 农渔 / 普通卫士**：持农具、短刀或常规兵器，护家看门。破坏力≈徒手制民、冷兵器格斗；遇精锐剑士即溃。≈一阶。
2. **军中精锐 / 宫门锦衫卫 / 地方剑手**：受过军训或门路剑法，能列阵、守宫、护卫大夫。可小队作战，难敌真正名剑手。≈一阶后期～二阶初期。
3. **吴越两国剑术高手 / 青衣剑士级**：如吴王所派**八名青衣剑士**——剑利术精，善用「以众击寡」群战，能在越宫连杀多名锦衫卫士，街头一剑劈羊、断人掌。破坏力：裂石断金边缘、乱军中取首、一堂震慑；仍属人间精锐，非毁城。≈二阶～三阶初期。
4. **越国剑室八十高手级 / 千名剑手卫队**：经阿青「影子」浸润后的越军剑术中坚与宫卫，可支撑灭吴两次大战的剑锋优势。单人难敌阿青三招，群体战力可碾压同级吴卒。≈三阶。
5. **人间剑术顶点 · 阿青 / 白猿「白公公」**：阿青以竹棒四招戳瞎两名吴士、八人围攻仍尽落兵刃各伤一目；四日之内连败八十名越国剑术高手（无人能挡三招）；灭吴后可破千名卫士直抵馆娃宫，竹棒尖劲气可隔空伤人。白猿通人性，以竹棒与阿青对刺数年，是其剑术神髓来源，曾三度欲杀范蠡，被阿青挡下乃至断臂遁山。破坏力：一人可乱军取将、破千人卫队，近「毁据点 / 掀平建筑群」的精英Boss感，**未达夷平城池、移山填海**。≈**四阶**（世界峰值）。

关键「武学」逻辑（非门派体系，而是本世界可观察规则）：
- **越女剑法（阿青）**：轻妙灵活、变化繁复，来如雷霆收震怒、罢如江海凝清光；只可意会不可言传。核心不是招式名录，而是与白猿对刺中悟出的距离、角度、虚实与杀机。武器可是**竹棒**亦可是剑——锋利与否次于「人」。
- **剑法影子**：越国高手无人真正学会阿青招式，但与之对招时捕捉到的「影子」层层下传，已足以使越军剑术无敌。这是原作明确的「群体战力跃迁」机制。
- **吴国剑法**：利剑 + 精湛剑术 + 孙武一脉「以众击寡」战法；宫中比武与街头作恶皆体现其专业杀伤。
- **铸剑一脉**：名匠**欧冶子**已死，曾铸湛卢、纯钧、胜邪、鱼肠、巨阙等；弟子**薛烛**（相剑/铸剑）被投吴的师兄**风胡子**斩断四指成残，仍可**督工**铸出大批利剑；锋利兵器与剑术影子并列为灭吴硬件。
- **劲气 / 内劲雏形**：阿青竹棒尖可发劲气刺伤西施心口——本世界极少数近超凡的表现，仍锚定人身与兵器，不上升为法术。

乐园阶位映射（对照《阶位战力图鉴》，宁低勿高）：庶民/牧羊/普通卫士≈**一阶**；军中精锐/宫卫/地方剑手≈**一～二阶**；吴国八剑士/越国一流剑手≈**二～三阶**；经影子浸润的越军剑术中坚/千名剑手≈**三阶**；阿青巅峰与白猿≈**四阶**（人间剑术极限、毁据点级精英Boss，未达城市夷平）。**世界峰值≈四阶**；无五阶以上，无超阶。顶点为**阿青**（及白猿），低阶切入一律规避硬刚顶点·情报优先。

**【地理 · 舞台】**
舞台为春秋末期东南吴越两国及毗邻水乡，尺度小于后世金庸长篇的「全中原江湖」，但史感更重：
- **越国都会稽一带 / 越王宫**：勾践议政、吴使献剑比武、阿青入宫剑室连败八十高手的政治与剑术中心。宫中锦衫卫、剑室、议政堂是一～三阶日常舞台。
- **越国城中街市 / 范蠡府邸**：范蠡归途、吴八剑士醉酒作恶、阿青牧羊遇辱与一战成名处；范蠡赔羊、邀阿青至家、陪其牧羊的情感起点。
- **山野牧地 / 白猿出没的山林**：阿青十三岁起放羊、与「白公公」竹棒对刺数年的修炼场；白猿三刺范蠡、断臂遁山亦在此。氛围：雾、羊、竹、兽啼——剑术神髓不在宗门而在山野。
- **浣纱溪 / 西施故里一线**：范蠡寻访绝色**夷光**（后改名西施）及**郑旦**之处；美人计的地理起点。山水灵秀与国仇并置。
- **吴国姑苏 / 馆娃宫**：夫差沉湎酒色之所；灭吴后范蠡率剑手攻入、与西施重逢、阿青破窗欲杀西施的终局舞台。「西子捧心」定格于此。
- **太湖**：终局范蠡弃官与西施泛舟隐退的终点，象征功成身退与情债两讫。
- **吴越交战原野 / 水师战场**：两次大战大败吴兵的宏观战场；剑术影子与薛烛利剑在此兑现为军国胜利。契约者低阶切入多见斥候、运粮、刺客，而非一上来馆娃宫。
- **周边楚地意象**：范蠡、文种本楚人投越——可作外来谋士身份切入的背景板，勿写成楚国主线。

势力地盘：越王勾践—范蠡/文种大夫府—剑室军中；吴王夫差—太宰伯嚭等佞臣—青衣剑士与水师；山野则近乎「法外」——阿青与白猿不属于任何朝廷编制。

**【世界剧情线】**

### 一、前史·吴越血仇（史台铺底，小说默认已知）
春秋末，吴越世仇。吴王阖闾攻越，槜李之战负伤，临终嘱子**夫差**必报。夫差即位后大败越军，**勾践**困于会稽，几乎亡国。勾践以厚赂吴太宰**伯嚭**等求和，入吴为质服役约三年，受尽屈辱；归国后「**卧薪尝胆**」——坐卧薪草、口尝苦胆，日夜不忘会稽之耻。大夫**文种**献「灭吴九术」（或称伐吴九术），**范蠡**（字少伯，楚人）与之同心辅政：垦田练兵、结好邻国、收买佞臣、离间忠臣（如针对伍子胥一系）、进献美女与良材等。史传与小说交汇的关键节点：**西施（夷光）、郑旦**被选为美人计棋子；范蠡亲自寻访浣纱溪，已与夷光互生情愫，却为复国忍痛将其改名「西施」送入吴宫，使夫差沉湎酒色、疏远忠谏。此阶段世界状态：越国表面恭顺、暗中备战；吴国兵强剑利而政治开始腐化；「剑」与「术」仍是越国短板——八计可成，兵器与剑士仍惧吴。

### 二、吴使八剑·宫中血光（小说开篇冲突）
复仇大计推进至关键瓶颈：铸剑与剑术。名匠**欧冶子**已死；其徒**薛烛**尚在越，却因拒绝随师兄**风胡子**投吴，被风胡子斩断四指，无法亲手锻打，仅能相剑、督工。吴王夫差派**八名青衣剑士**持宝剑至越「送礼试探」。越宫比武：吴士剑锋断轻纱而不折，剑术与群战法连杀越国多名锦衫卫士，勾践、范蠡、文种俱惊——若开战，越卒兵器与剑技皆处下风。名场面：宫门血、宝剑寒光、越卫仆地、君臣无言。阶段结果：越国确认「利剑 + 剑术」双缺口；薛烛受命督工铸剑；范蠡心系西施又忧国事，归途情绪复杂。世界状态：吴越表面礼尚往来，剑尖已抵咽喉。

### 三、牧羊女与竹棒（阿青登场）
范蠡归途，街头吴国八剑士醉酒作恶：斩断范蠡两名卫士手掌，又将路过的牧羊女**阿青**（约十六七岁，浅绿衫、瓜子脸、长睫、纤弱而秀丽）的一头山羊一剑劈成两半，并出言戏耍。阿青大怒，手持**竹棒**上前——四招戳瞎两名剑士眼睛；八人围攻，兵刃尽被打落，各被刺瞎一目（或伤腕）。阿青要他们赔羊，否则再戳另一只眼。范蠡赔羊劝止，心中狂喜：此等剑术，正是越国所缺。他邀阿青至家，温言询问「师父」——阿青答是「**白公公**」，一只通晓人性的**白猿**。她十三岁起牧羊，白猿常以竹棒与她玩耍对刺；白猿刺来，她从抵挡中学会反击，日久竟悟出天下无双、只可意会不可言传的剑术神髓。范蠡陪她放羊、照料羊群，阿青对这位温和大夫生出感激与爱慕。白猿忌人，曾**三次**欲刺死范蠡，皆被阿青挡下；甚至断白猿两臂，白猿负痛遁入山林。名场面：街头竹棒如飞、羊血与人目、范府问师、山野白猿三刺。阶段结果：阿青被纳入范蠡视野；白猿线揭示剑术来源非宗门而近「天授」；阿青情愫暗生而范蠡不觉。世界状态：越国第一次摸到「可压过吴剑」的活棋。

### 四、剑室四日·影子下传（教剑与不辞而别）
范蠡请阿青入宫，希望她教练越国剑士。前后**四天**，阿青在剑室与**八十名**越国剑术高手对招，**无人能挡三招**。勾践大喜，愈信复国有望。阿青并不能把剑法「讲」会——其术本不可言传——但高手们在死里逃生的对招中捕捉到「**剑法影子**」，再层层传授军中，已足以使越军剑术质变。与此同时，薛烛督工加紧铸利剑，硬件与软件并进。阿青在与范蠡相处中爱意日深，却渐渐察觉：范蠡心里只有吴宫中的西施。得知真情后，阿青**不辞而别**，消失于山野；范蠡寻访无果。阶段结果：越国剑术与兵器双线补齐；阿青人去影留；范蠡仍一心复国与迎回西施。世界状态：备战进入最后数年，灭吴倒计时开始。

### 五、兴兵伐吴·两次大战（国仇了结）
约**三年后**，越国兴兵伐吴。凭借薛烛督造之利剑与阿青剑法影子浸润的精妙剑术，越军经**两次大战**大败吴兵，兵临姑苏。吴王夫差兵败势穷，**自杀**；吴国宗庙倾覆，越国会稽之耻得雪。文种、范蠡等谋臣之功与战士之血共同铸成这一刻——原作明确：剑术与利剑是战场胜负的关键技术因素，美人计与离间是战略因素。阶段结果：吴亡越霸；范蠡亲领剑手攻入政治与情感的双重终点——馆娃宫。世界状态：春秋东南格局改写；个人情债尚未结算。

### 六、馆娃宫·西子捧心（终局悲歌）
范蠡攻入**馆娃宫**，与西施重逢，恩爱无边。阿青突然再现——她已深深爱上范蠡，视西施为必须除去的情敌。**千名**卫士无法阻挡，阿青轻易破阵，竹棒抵住西施心口。正要下手时，西施**无与伦比的绝色**令阿青杀气层层剥落：失望、沮丧、惊奇、羡慕，终至崇敬。她下不了杀手，黯然破窗离去，从此再未出现于范蠡身侧。然而竹棒尖端的**劲气**已刺伤西施心口；西施微蹙眉、伸手捧心，美得夺魂——金庸借此锚定千古成语「**西子捧心**」的小说起源。范蠡忧阿青再来，亦看透功高震主之危，**弃官舍利**，与西施**泛舟太湖**，悠游终生。阿青剑法的「影子」则流入后世武林（金庸宇宙中《射雕英雄传》江南七怪**韩小莹**一派「越女剑法」，有「枝击白猿」等招，为跨书互文，非本篇正文出场）。阶段结果：国仇已报、情债未偿；最强者退出历史前台；范蠡西施归隐。世界状态：越国进入胜利后的权力整合（文种命运等史传余波可作背景，勿强行写成阿青线续集）；山野再度沉默。

### 七、可介入的微观事件链（日常与副本感）
除大场面外，正文与史台可写的高频事件：宫中试剑与献宝、街头吴士滋事、牧羊与赔羊纠纷、剑室点将比武、督工铸剑与矿炭运输、浣纱溪选美与护送入吴、细作往来吴越、水师演练、灭吴前夜的暗杀与反间、馆娃宫破城混乱、太湖隐退前的追杀与护送。AI写日常优先这些，而非一上来改写夫差生死。

### 八、蝴蝶效应边界
契约者可改局部：救一名被吴士欺辱的越卒、参与一次铸剑督工护送、在剑室抢先与阿青过招而留下「另一道影子」、助范蠡寻访失踪的阿青（未必找到）、灭吴之役中保下某支偏师。不可无代价改写：欧冶子已死、薛烛四指已残、西施入吴的战略棋、阿青剑术来自白猿而非某大宗师、馆娃宫「西子捧心」的悲剧定格、世界峰值不超过阿青/白猿的四阶逻辑。强行「提前杀死西施」或「让阿青与范蠡成婚」会与原作主题（家国压过私情、最强者孤独退场）硬撞，须付相应因果（勾践猜忌、越军剑术线断裂、吴国反扑等）。

### 九、历史长线补述（忠史扩写，不另造小说情节）
为档案可写厚度，补吴越争霸史台中与小说咬合、可作背景注射的公开史脉（**非**金庸新编情节，正文仍以小说人物戏核为准）：

**阖闾—夫差复仇链。** 吴王阖闾在越王允常死后攻越，槜李之战中负伤，临终嘱太子夫差必报此仇。夫差使人立于庭，出人呼「夫差，而忘越王之杀而父乎？」以耻励志，终大败越军。此链解释小说开局「吴强越弱、吴剑压人」的政治军事前提：吴不是突然霸道，而是带着父仇与训练有素的剑士传统压境。

**会稽之困与为质三年。** 勾践兵败，退保会稽山，遣使卑辞厚礼，并贿吴太宰伯嚭。夫差不听伍子胥「尽灭越」之谏，许越求和。勾践夫妇入吴为臣妾，驾车养马、尝粪问疾等屈辱叙事在史传与话本中反复出现；金庸小说将其压缩为已知背景，重点放在归国后的卧薪尝胆与九术。契约者若在「为质期」切入，舞台在吴都，风险是吴卒与伯嚭一系，阿青线尚未开启。

**文种九术与范蠡定倾。** 史载文种献伐吴之策，范蠡有「持盈、定倾、节事」一类治国框架。小说把可戏剧化的部分落到：铸剑补短板、美人计（西施、郑旦）、离间（针对伍子胥等忠臣）、收买佞臣。写正文时九术不必逐条念经，但应让读者感到——灭吴是**多年系统工程**，阿青是工程里突然出现的「剑术奇点」，不是唯一变量。

**伍子胥之死与吴政癌变。** 伍子胥屡谏夫差勿纵越、勿北上争霸中原，终被赐属镂剑自尽。此事件在小说中可作远景与吴国细作对话的谈资：吴国剑再利，朝堂已开始自毁免疫。馆娃宫的奢靡与忠臣血，是同一枚硬币的两面。

**姑苏台、馆娃宫与太湖。** 姑苏台象征夫差好大喜功；馆娃宫是西施入吴后的空间锚点，也是终局阿青破窗处；太湖是范蠡扁舟五湖的归宿。地理三点连成「骄奢—情债—身退」的空间叙事。

**灭吴后的兔死狗烹阴影。** 史传范蠡致书文种，谓飞鸟尽、良弓藏，狐狸烹、走狗烹；文种终未能全退。金庸让范蠡在馆娃宫情债了却后立即泛舟，既是爱情收束，也是政治本能。契约者若立大功，应感到勾践的猜忌目光——这是史台正确反馈，不是 OOC。

写正文时：**小说出场人物与情节以金庸为准**；史传人物未在小说展开者（如伍子胥、伯嚭、允常、阖闾）可作远景名与对话谈资，**勿抢阿青—范蠡—西施主戏**，勿编造三人与上述史传人物的原作没有的私密互动。

### 十、人物关系与情感结构（可写进对白的动力学）
本篇人物少、关系紧，AI 最易写歪的是「三角」力度：

1. **范蠡—西施**：先有情，再有国；献美是自我凌迟式的战略，重逢时的恩爱是多年压抑的回弹。范蠡对西施的专一，导致他对阿青的爱完全「视线外」。
2. **范蠡—阿青**：恩主与兵器、长者与少女、被爱者与爱人。范蠡给的是照料、赔羊、尊重其剑，不是承诺。阿青解读为可托付终身的温情——错位即悲剧。
3. **阿青—西施**：从未共事，却在馆娃宫完成一次「以美破杀」的对决。西施无招式，却让四阶剑术失效于心；劲气仍伤其心口，美与伤互相成就。
4. **阿青—白猿**：师、伴、情敌（白猿忌范蠡）。断臂遁山后，阿青在人间再无同类。
5. **勾践—诸大夫**：工具理性。阿青是剑，范蠡是脑，薛烛是炉；用尽则危。

写对白时：阿青句短、意象近羊与山；范蠡句稳、常双关国事与相思；勾践句冷、问结果不问心情；西施出场宜少言，以目与姿驱动场面。

### 十一、跨作品互文（金庸宇宙边缘）
《越女剑》在金庸内部时间线最早，却写作最晚之一。其剑法「影子」在《射雕英雄传》中体现为江南七怪之一**韩小莹**所习「越女剑」，招式名含「枝击白猿」，与白猿传剑遥相呼应。这证明：阿青本人从历史蒸发，她留下的「可观察残影」却改变了后世武林技能树。契约者若从本世界「带走」剑法影子，应表现为身法轻妙、竹木亦可伤人、以巧破力的风格，而非突然多出完整九阴九阳式内功体系——本世界无那种内功天花板，硬塞会破坏阶位映射的宁低勿高原则。

### 十二、舞台氛围与感官锚点（供正文沉浸）
- **越地**：潮气、竹青、羊膻、炉炭与铸剑的金属腥、会稽山影、水田蛙鸣。
- **吴地**：锦绣、酒甜、姑苏台高风、馆娃宫纱与脂粉、败兵甲叶碰撞。
- **剑室**：汗、木人桩、兵器架、四日不停的拆招喘息。
- **山野**：白猿啼、竹响无人影、断臂兽血、雾中牧径。
感官优先于形容词堆砌；阿青的杀招应用「快、准、停」三拍写，而非金光特效。

**【势力图谱】**
- **越国王室与大夫府**：宗旨——复国会稽之耻、图存图霸。地盘——会稽及越地。代表——**勾践、范蠡、文种**；剑术侧一度仰赖**阿青**与剑室；军工侧**薛烛**。战力：备战期中下，灭吴前因剑术影子与利剑跃升至可灭吴。与吴不共戴天；对内功高者（范蠡）有兔死狗烹隐患（史传余韵）。
- **吴国王室与青衣剑士**：宗旨——维持霸权、震慑越人。地盘——姑苏及吴地。代表——**夫差**；武力象征——**八剑士**与水师；技术侧——**风胡子**等。战力：故事前中期明显强于越；政治腐化后战场崩盘。
- **山野「白猿—阿青」无编制力量**：无宗旨、无地盘行政，只有牧地与山林。代表——**阿青、白猿**。战力顶点四阶，不听王命，只应情义与本能。可被范蠡短期借力，不可被勾践真正豢养。
- **铸剑匠人谱系（欧冶子门）**：跨吴越的技术势力。欧冶子死后分裂为投吴（风胡子）与留越致残（薛烛），直接造成两国兵器代差与越国补课线。

**【贵重物品】**
- **吴国试探宝剑（八剑士所持）**：吴王遣使所携，锋利可断轻纱而不折，宫中比武用以羞辱越剑。下落：示威后仍属吴使一系；象征吴国兵器优势。
- **薛烛督造之越国利剑（批量）**：欧冶子传人督工三年所出，与剑术影子并列灭吴物质基础。非单件神兵，而是「军阵级」利器群。
- **欧冶子五剑传说（湛卢、纯钧、胜邪、鱼肠、巨阙等）**：前史名剑，部分流入吴楚传说；本篇多作背景名号，契约者「寻五剑」须标为史传/传说线，勿写成阿青主线必掉落。
- **阿青之竹棒**：看似凡竹，在其手中等于无双剑，可发劲气伤人。灭吴后馆娃宫一役仍持此；人去则棒亦失其神。
- **西施（作为战略「人」的棋子）**：严格说不是「物品」，但是灭吴九术中最关键的「美人计载体」；原作中其美貌甚至化解阿青杀意，价值超越寻常宝兵。
- **羊群（阿青所牧）**：情节催化剂——羊被劈催生街头一战；范蠡赔羊建立信任。可作一阶日常钩子。

**【隐藏剧情 · 伏笔】**
- **剑法不可言传**：阿青无法「教会」任何人完整剑法，只能留下影子——暗示本世界最高武学近乎天赋与自然，而非门派课程；也解释为何后世只余「越女剑」碎片（韩小莹线）。
- **白猿三刺范蠡**：白猿忌的是「抢走阿青的人」；范蠡始终未真正理解自己在阿青与白猿关系中的位置，情感盲区是悲剧根源之一。
- **西子捧心**：以武侠劲气解释成语来源——美与伤、爱与杀在同一瞬间完成；阿青「杀不下去」比「杀掉」更残酷。
- **范蠡泛舟**：功成身退既是情，也是政治生存；与史传「兔死狗烹」互文，文种命运可作暗线警示。
- **跨书伏笔**：越女剑法流入南宋武林，证明阿青的影子改变了长远武学史，尽管本人从历史中蒸发。
- **美人计的道德黑洞**：范蠡爱西施而献西施，家国大义合法化了情感施暴；阿青的纯粹之爱在此结构里无处安放——这是主题，不是可有可无的恋爱番外。

最早可切入锚点：吴八剑士入越前后（宫中试剑或街头滋事），一阶即可卷入；亦可从浣纱溪护送、铸剑督工、剑室点将等支点进入。

## 阶位切入点

> 阶位↔原作战力：一阶≈庶民/牧羊/普通卫士与新卒；二阶≈军中精锐/宫卫骨干/地方剑手；三阶≈吴国八剑士级与越国一流剑手/经影子浸润的剑术中坚；四阶≈阿青巅峰与白猿（人间剑术顶点）。世界顶点为**阿青**（及白猿），属四阶峰值；低阶切入一律「规避顶点·情报优先」，顶点须按原作全力与条件性胜利处理。

**一阶（庶民·牧羊·新卒 · 街头与牧场线）**

切入身份/时点：契约者以越国边邑牧民、城中小贩、新入伍卒或范府杂役身份切入，锚定**吴国八剑士入越前后**——宫中试剑消息刚传出，或街头吴士醉酒滋事当日。

初始事件：市集/城门口，青衣吴士当众劈羊或折辱越人；人群惊散。你或护住邻人羊只，或被范府差遣送物恰遇冲突；竹棒少女阿青即将或正在出手——你必须选择：逃跑、助拳、报官，还是趁乱接近范蠡随从。

开场白建议：「会稽的风里有铜腥。吴人青衣在街心一剑把山羊剖成两半，血溅到你草鞋上。围观的越人不敢出声，只有个浅绿衫的牧羊女握紧竹棒往前走。你听见有人低呼大夫范蠡的车驾将近——下一息，要么你被当成多余的牲口，要么你被卷进一场改变吴越气数的竹影。」

关键NPC立场：**阿青**（尚不识你，护羊第一，对善意赔羊者会记住）；**范蠡**（忧国忧西施，愿赏识胆色与有用之人，无暇儿女私情）；范府卫士（刚被断掌的幸存者惊恐敌视吴人）；路人越卒（怕事，可买通带路）。

主线钩子/支线：主线——见证或协助「街头一战」的信息传到范蠡耳中；支线——替牧户追偿失羊、替薛烛工匠运炭矿时遭遇吴国细作、在宫门外听闻试剑惨败的民谣。

危险度/规避：危险度中。规避与吴八剑士正面对决；规避对阿青出手抢棒（她一阶视角已是不可理解的快）。白猿此时在山，勿主动寻猿。

任务方向/奖励：活着把「有牧女胜吴士」的消息交给正确的人；奖励取向——越地人情、范府杂役身份、一柄薛烛工坊流出的寻常利刃、阿青若受你助可能的口头承诺（非剑法真传）。

**二阶（军中精锐·宫卫·剑室边缘 · 试剑与督工线）**

切入身份/时点：契约者为越宫锦衫卫骨干、剑室记名剑手，或铸剑工坊护卫头目；时点在**宫中吴使比武前后**至**阿青入宫教剑前夜**。

初始事件：你奉命观摩或替补上场对吴士——眼见同袍被利剑洞穿；勾践震怒，文种、范蠡连夜议薛烛督工。你或得令护送薛烛前往矿场，或奉范蠡密令出城寻访「传闻中的竹棒少女」。

开场白建议：「剑室的灯油烧到见底。吴人的剑断纱不折，你师兄的血还在青石砖上。范大夫说越要的不是勇，是『另一种剑』。城外牧场有人听见白猿啼——你握紧新配的脊，不知自己要找的是救国的剑，还是一只不该惹的兽。」

关键NPC立场：**范蠡**（需要可靠武力执行寻人与护工）；**文种**（以国事考核你，功则用、过则弃）；**薛烛**（残指匠人，多疑，只信肯护炉火的人）；**勾践**（多疑的君王，赏罚酷烈）；吴使余党（可能街头报复）。

主线钩子/支线：主线——护铸或寻阿青成功，使「剑术+利剑」双线不中断；支线——识破风胡子安插的破坏炉火细作、在浣纱溪方向护送美人计相关车仗的外围警戒、安抚因宫中惨败而哗变的新卒。

危险度/规避：危险度中高。规避单独挑战八剑士全员；规避在勾践面前夸口「可替阿青」；白猿若现身，**情报优先，不硬刚**（四阶峰值）。

任务方向/奖励：稳定的军中或工坊职衔；一柄督造序列的良剑；范蠡短暂信任；可能换取剑室观摩阿青对招的席位（仍学不会完整剑法，只得影子皮毛）。

**三阶（一流剑手·影子中坚 · 教剑四日与备战线）**

切入身份/时点：契约者为越国剑术一流、可入剑室与阿青对招的八十高手之一，或范蠡亲信剑士队长；时点锚定**阿青剑室四日**至**不辞而别后的备战三年**。

初始事件：你抽签与阿青过招——三招内兵器脱手，生死一线却瞥见「影子」；当夜范蠡命你把影子拆解教给百人队。或：阿青失踪，你奉命沿线追访，在山缘听见白猿啼与竹响。

开场白建议：「她只是个牧羊的，竹棒却让你腕骨发麻。第三招你跪在剑室，听见自己的呼吸像破风箱。范大夫不看你的脸，只说：把你看见的『影子』留下。门外有人报——阿青走了。你不知该追人，还是追那道再也抓不准的剑光。」

关键NPC立场：**阿青**（对能挨过一招半式且不辱她的人有朴素好感，对打探范蠡与西施隐私者厌恶）；**范蠡**（视你为可执行灭吴的剑；对你报告阿青情愫可能迟钝或拒听）；**勾践**（要结果：三年内能战）；**西施**（远在吴宫，你可能接密信/接应任务而不得见面）；白猿残影（山中威胁）。

主线钩子/支线：主线——影子下传不走样、维持越军剑术优势至兴兵；支线——追查阿青去向却不惊动白猿杀机、保护薛烛最后一批利剑运阵、参与对吴细作的反间（呼应九术）。

危险度/规避：危险度高。与阿青真比不可存胜心；馆娃宫级场面未到勿抢入；白猿为**四阶峰值存在：情报优先/条件性胜利**，须按其原作全力来写，不得用「战力异常下降」糊弄。

任务方向/奖励：剑术影子的稳定掌握（身法轻妙、竹木亦可伤人的风格）、百人队指挥权、灭吴役中的先锋名额、范蠡亲赐的玉佩信物（人脉向）。

**四阶（近峰值·破城与馆娃 · 灭吴终局线）**

切入身份/时点：契约者为越军破吴前锋剑士统领、范蠡亲率入馆娃宫的千名剑手之一，或与阿青短暂并肩过的山野强者；时点锚定**两次大战至馆娃宫西子捧心**。

初始事件：姑苏城破，你随范蠡冲入馆娃宫走廊——西施与范蠡重逢的笑声未落，竹影已至，千人卫士如割草。你必须在几息内判断：拦阿青（几乎必死）、护西施、或护范蠡侧翼并制造「让阿青看清西施容颜」的空间。

开场白建议：「吴王的血还在城楼上干。馆娃宫的纱帐被剑风撕开，你看见范大夫握着一个女人的手——然后竹棒到了。千人挡不住她。你平生第一次明白什么叫『剑』：不是刃，是她眼里那一点要熄灭又熄不灭的火。西施回眸的刹那，连杀气都跪了下去。」

关键NPC立场：**阿青**（杀意只向西施，对无意义拦路者斩而不顾；若你曾护羊、赔羊、不辱她，或可换一声「让开」）；**范蠡**（此刻只要西施活与阿青不再来）；**西施**（无武力，美与伤是结局关键）；**勾践**（城外收权，可能已开始疑功臣）；残吴死士（同归于尽式偷袭）。

主线钩子/支线：主线——促成「西子捧心」定格而非西施惨死或阿青被围杀，使原作悲剧与国运收束成立；支线——护送范蠡西施出城泛舟太湖、拦截欲杀功臣的暗令、在太湖岸留下或不留阿青的消息。

危险度/规避：危险度**极高 / 贴近顶点**。阿青为本世界**四阶峰值**：**存在·情报优先 / 条件性胜利**（例如借西施之美化解杀意、引其破窗离去）。必须按原作全力与主题收束处理，不得用「临时战力异常下降」让契约者硬刚取胜。白猿若在终局山缘再现，同等处理。

任务方向/奖励：灭吴首功级声望（亦招勾践疑）、范蠡离去前的密件与太湖航线、阿青消失前遗落的半截竹（信物非功法全书）、越女剑法影子的「最完整一代」体悟（仍非阿青本体）。
