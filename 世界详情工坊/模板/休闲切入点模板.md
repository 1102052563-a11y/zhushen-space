# 休闲·切入点条目模板（`<世界名>·休闲切入点`，正文 ≥1500 字）

> 用途：**仅世界选择/生成世界卡时读取**（玩家入世后不再注入）。
> 休闲世界**没有阶位/战力档**——不写「一阶…九阶」「阶位↔境界」。切入点写的是**契约者以什么身份融入这个日常/恋爱世界、能展开哪些情感线与日常玩法**。
> 段落标题一字不差保留；关键人物**加粗原作真名**，禁代称。

---

开头固定写一行「切入基调」，例：

> 本世界为休闲/恋爱向，无生存压力与战斗。契约者以**日常身份**融入，核心玩法＝relationship 攻略 + 日常事件，而非任务厮杀。

然后写以下**七个字段**（一个不能少，字段名照抄）：

---

切入身份：契约者以什么日常身份进入（转校生 / 新邻居 / 社团新人 / 打工店员 / 归乡的幼驯染 / 借住亲戚家…），说明为何自然融入不突兀。

切入时点：锚定原作哪个时点（开学季 / 文化祭前 / 暑假 / 主角与女主刚相遇之际…），要具体。

初始处境：住在哪、日常节奏、所在的班级/社团/圈子、手头的社交起点（认识谁、和谁同桌/同社团）。

开场白建议：「一段 60~120 字的开场白范文，第二人称，恋爱日常向，有画面有生活气息，直接可用作该世界开局正文的定调（如樱花飘落的转校日、社团招新、雨天屋檐下的相遇）。」

可攻略对象：列 3~6 个原作可攻略角色，**加粗真名**，各附一句「攻略切入方式 + 好感起点 + 情感钩子」（在哪认识、她/他吃哪套、有什么心结可介入）。禁「女主/红颜」等代称。

日常玩法钩子：本世界能展开的日常与情感活动——社团活动、一起打工、约会地点、节日祭典、共同兴趣、家庭日常、季节事件；2~4 条可推进关系的「事件线」。

氛围/雷区：应保持的基调（治愈 / 青涩 / 甜宠 / 致郁纯爱）+ 必须规避的写法（**忌强行加入战斗 / 任务厮杀 / 力量体系**；忌让角色 OOC；NSFW 尺度按世界原作把握）。

---

（休闲世界只有一份整合切入点，不按阶位分段。低门槛、重情感与日常代入。）
