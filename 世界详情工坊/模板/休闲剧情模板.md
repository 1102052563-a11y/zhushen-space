# 休闲·剧情条目模板（`<世界名>·剧情`，正文 ≥6000 字）

> 用途：**休闲世界**（galge / 恋爱 / 日常 / 治愈 / 校园 / 后宫）专用。世界卡生成 + 入世后正文常读的「常青档案」。
> **与战斗世界完全不同**：不写力量体系 / 战力 / 阶位 / 巅峰战力 / 危险度 / 贵重法宝。核心是**人物魅力 + 情感线 + 日常舞台**。
> **真名铁则**：可攻略角色、配角、社团一律原作真名；禁群像/女主A等代称。**禁凑字**：字数不够去百科/维基/galge攻略站检索后总结再写。
> 下方【】段落标题必须一字不差保留（机检按它验收）；配额是下限。

---

**【作品来源】**（~200 字）
作者/原作媒介（galge·视觉小说·恋爱动画·少女/少年漫画·乙女游戏·轻小说）/所属品牌（Key/型月/Leaf/枕…）/连载完结状态/整体气质（纯爱·催泪·治愈·后宫·搞笑·致郁·百合·乙女）。

**【世界定位】**（~150 字）
一句话锚定这是个什么样的日常/恋爱世界 + 主角与核心舞台。

**【世界观 · 舞台设定】**（~800 字）
背景设定（现代校园 / 都市 / 小镇 / 温和奇幻日常 / 架空学园）；时代与季节基调；社会与校园氛围；**若有超自然设定也只写其"日常/情感"侧**（如"幻想世界的另一边""能看见幽灵"），不写成战斗力。世界的"温度"与治愈感来自哪里。

**【地理 · 生活舞台】**（~500 字）
主要生活场景逐个写：学校 / 教室 / 天台 / 社团活动室 / 打工的店 / 主角住所 / 约会地点（河堤 / 车站 / 祭典会场 / 海边）/ 街区。每个场景在故事/情感中承载什么（初遇、告白、名场面发生地）。

**【故事主线 · 情感线】**（~1800 字 · 大头）
按篇章/时间顺序详述原作主线与情感发展脉络：相遇 → 日常相处 → 情感升温 → 冲突 / 误会 / 分离 → 和解 / 告白 → 结局。写清关键事件、催泪/名场面、情感转折的因果。
**共通线 + 各角色个人线（route）分别交代**；每条 route 的走向与结局都写明白（HE/BE/True End）。**结局必须写清楚**（在一起 / 分离 / 治愈 / 成长）。

**【可攻略角色 / 主要人物】**（~2200 字 · 核心 · ≥6 人）
每个可攻略角色一段，字段固定：
`姓名（身份/年级/所属）｜外貌：…（发色瞳色·穿着·标志特征）｜性格：…｜角色类型：…（傲娇/天然呆/大和抚子/病娇/元气/文静/女王/妹系/幼驯染…）｜萌点/魅力：…｜个人线剧情：…（她/他的故事与心结、如何攻略、结局）｜与主角关系：…`
- **姓名必须原作真名**；主角 300~400 字，每个女主/男主 250~350 字，重要配角 100~150 字。
- 可攻略对象写全（女主群 / 男主群）；旁白/家人/损友等重要配角也各一条。

**【人际关系网 / 社团势力】**（~500 字）
朋友圈 / 社团（如文艺部、篮球社）/ 家庭 / 情敌 / 师长；各人物之间的关系与三角/多角关系，用相对链接互链。谁和谁是青梅竹马、谁暗恋谁、谁是情敌。

**【情感事件 · 名场面】**（~500 字）
原作最经典的情感名场面逐条列：告白、约会、误会、生病照顾、雨中、祭典烟花、毕业、重逢、催泪高潮。写清何时何地、谁与谁、为何动人。

**【隐藏剧情 · 真结局 · 伏笔】**（~400 字）
True Route / 最终线的隐藏真相；埋在日常里的伏笔与回收；官方FD/续作补充的后日谈；世界的隐藏设定（若为奇幻日常）。

**【氛围基调 · 雷区】**（~300 字）
画风与叙事口吻（温暖治愈 / 青涩 / 致郁 / 搞笑日常）；BGM/演出氛围；NSFW 尺度（全年龄 / 有 H 场景则注明但不展开露骨）；最容易 OOC 的写法（列 2~4 条"忌…"，如"忌把治愈日常写成打斗""忌让角色崩坏人设""忌强行加战斗任务"）；最适合切入的时点。
