# 剧情条目模板（`<世界名>·剧情`，正文 ≥10000 字）

> 用途：世界卡生成 + 玩家入世后正文 API 常读的「常青档案」。
> 原则：**剧情优先**（世界剧情线是最大头）；一切内容忠于原作、可溯源；写给 AI 读——信息密度优先，不写空话。
> **真名铁则**：人物、势力代表、宝物一律原作真名；禁止群像/红颜/牙人等代称。
> **禁凑字**：禁止扩写·补密·跨世界套话；字数不够去百科/维基检索后总结再写。
> 下方【】段落标题必须一字不差保留；配额是下限，超出无妨。

---

**【作品来源】**（~200 字）
作者/原作媒介（小说·动漫·游戏·影视）/所属系列与关联作品/连载与完结状态/整体文风气质（热血·黑深残·日常·克苏鲁……）。

**【世界定位】**（~150 字）
一句话锚定这是个什么世界 + 主角是谁、走了条什么线。

**【世界观 · 力量体系】**（~1200 字）
世界基本规则（力量来源、修炼/成长逻辑、死亡与复活规则、特殊系统）。
力量体系**逐级列全**：每个境界/等级的名称、达成条件、典型战力表现（能开山？能灭国？）。
最后一行给**乐园阶位映射**：`原作境界X~Y≈一阶；…；顶点=N阶/超阶`。映射**必须对照附件《阶位战力图鉴》逐档对齐**——拿各境界的实际破坏力表现（毁街区？夷平城？爆星？）匹配图鉴各阶的破坏力描述，宁低勿高；「绝强」及以上一律记「超阶」；原作真实顶点忠实标注，不许压低。

**【地理 · 舞台】**（~600 字）
世界地图层级（国家/大陆/位面/星域…）、主要地点及其在剧情中的意义、势力地盘分布。

**【世界剧情线】**（~4000 字 · 最大头）
按卷/篇/季**分阶段详述**原作主线：每阶段写清——起因、关键事件链、名场面、重要战役/博弈、阶段结局、对后续的因果。
支线与番外择重要的写。**时间顺序写**，让 AI 能判断"任意时点世界处于什么状态"。
结局必须写明白（谁活谁死、世界变成什么样）。

**【主要人物】**（~2500 字 · ≥10 人）
每人一段，字段固定：
`姓名（身份）｜性格：…｜装备·能力：…（招牌武器/技能/血脉，含品级强度）｜人物弧光：…（从哪到哪）｜立场关系：…（与主角/各势力的亲疏敌友）`
- **姓名必须是原作真名**（可含道号/称号作括号补充，如「纪宁（北冥）」）。
- **禁止**：群像、红颜、牙人、眼线、核心道侣线、相关存在、标尺角色、代行者（单独成条）等代称。
- 主角 300~500 字，核心配角/反派 150~300 字，次要人物 50~100 字。
- 搜不到某人真名 → 不写该条或写「不详」，**不许用占位符凑人数**。

**【势力图谱】**（~600 字）
每个势力：宗旨/性质、地盘、**具名**代表人物、大致战力层级、与其他势力的敌友关系、在主线中的作用。

**【贵重物品】**（~600 字）
神器/秘宝/功法/血脉/系统道具等：**原作真名**、真实来历、能力、品级感、原作中的下落。（契约者最垂涎的部分，来历必须查证，不许编、不许用「某神器」糊弄。）

**【隐藏剧情 · 伏笔】**（~500 字）
幕后黑手与真相、重大伏笔及其回收、官方设定集/访谈补充设定、跨作品互文（若属于系列宇宙）。

**【大事记时间线】**（~400 字）
编年式一行一事：`时点 → 事件`，从故事前史写到结局。

**【叙事基调 · 雷区】**（~250 字）
画风与叙事口吻应该是什么样；最容易 OOC 的写法（列 2~4 条"忌…"）；最早可切入的时间线锚点。
