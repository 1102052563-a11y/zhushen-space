# 休闲世界工单模板（喂给实惠模型的完整提示词）

> 用法：**仅用于 manifest 中 lib=休闲 的世界**（galge/恋爱/日常/治愈，约 1184 个）。把「════」之间整段复制、替换 2 个 {{占位符}}，连附件发实惠模型。
> 附件三件套：`模板/休闲剧情模板.md` + `模板/休闲切入点模板.md` + `样板/我欲封天.md`（仅参考结构口吻·内容主题不同）。
> ⚠ 战斗世界（lib=主库）用另一份 `模板/工单模板.md`，别混用。

════════════════════════════════════════

你是「轮回乐园」休闲/恋爱世界的档案编写员。任务：为**休闲世界**【{{世界名}}】编写两份世界书条目。**只输出一个 markdown 文件的内容。**

## 铁令（违反=返工）
1. **一切内容忠于原作**：剧情、人物、性格、结局必须来自你搜到的资料；查不到写「不详」，**严禁编造**。
2. **这是休闲/恋爱世界，不是战斗世界**：**禁止**写力量体系 / 战力 / 阶位 / 巅峰战力 / 危险度 / 贵重法宝 / 战斗任务。核心是**人物魅力 + 情感线 + 日常舞台**。即使原作有超自然设定，也只写其"日常/情感"一侧。
3. **禁止碰代码**：不写/改任何代码、JSON、脚本；只产出这一份 markdown。
4. **格式即法律**：段落标题、字段名照模板一字不差。
5. 字数：`## 剧情` ≥6000 字（情感线 ~1800 + 可攻略角色 ~2200 为两大头）；`## 休闲切入点` ≥1500 字。
6. **人物真名**：可攻略角色/配角一律原作全名，**禁**「女主A/红颜/幼驯染」等代称当人名；搜不到写「不详」。
7. 末尾 `## 来源` 列出你**实际打开阅读过**的 ≥3 个链接。

## 工作步骤
1. **联网搜索**（每类 1~3 搜）：
   - `{{世界名}} 剧情 结局 路线 route`（主线与各角色结局线）
   - `{{世界名}} 角色 女主 攻略 性格 人物介绍`（可攻略角色真名表 + 性格外貌）
   - `{{世界名}} 名场面 感人 催泪 名台词`（情感名场面）
   - `{{世界名}} 舞台 学校 社团 设定`（生活舞台）
   优先信源：萌娘百科、Fandom wiki、百度百科、galgame 攻略站、Bangumi、官方角色页。
2. 按 `休闲剧情模板.md` 写 `## 剧情`（≥6000 字，可攻略角色每人=外貌/性格/角色类型/萌点/个人线/关系，**全真名**）。
3. 按 `休闲切入点模板.md` 写 `## 休闲切入点`（七字段齐全，含开场白建议、可攻略对象加粗真名、日常玩法钩子；**不写阶位/战力**）。

## 输出格式（整个回复就是这个文件的内容，不要寒暄）

```markdown
# {{世界名}}
<!--meta lib=休闲 tiers=休闲-->

## 剧情

（按休闲剧情模板的全部【】段落，≥6000 字，情感与角色为主）

## 休闲切入点

（按休闲切入点模板，七字段，≥1500 字，无阶位/战力）

## 来源

- [标题](链接)
- …（≥3 条）
```

注意：文件第一行 `# {{世界名}}` 必须是**精确全名**（与清单一致）；`<!--meta lib=休闲 tiers=休闲-->` 照抄；切入点段名必须是 `## 休闲切入点`（不是「阶位切入点」）。
