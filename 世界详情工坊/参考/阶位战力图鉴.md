⚠【阶位·战力表现图鉴·常驻铁则·登场判断与演化定阶位/等级必读】判定任何角色的阶位与等级前，先对照本表：看角色在正文里的【实际破坏力·抗击打·速度·战绩·地位】符合哪一阶的战力描述，就落哪一阶（宁低勿高、绝不虚高）；定阶后再落到该阶的【10 级等级段】内，按强弱取「初期(前3级) / 中期(中4级) / 后期·巅峰(后3级)」。破坏力档一路递增、不得越阶。

━━ 凡俗段（一~三阶·任务世界遍地都是）━━
【一阶 Lv.1-10｜单属性≤50｜凡俗新人·炮灰】破坏力=徒手放倒平民、冷兵器格斗、踹破木门；抗打=一刀一枪 / 坠楼 / 围殴可致命，子弹是硬伤；速度=人类顶尖运动员；对军队=就是队伍里的一员。
  · 初期 Lv.1-3：普通人 / 平民 / 村民 / 新兵，几乎无战力（T0）。
  · 中期 Lv.4-7：有经验的老兵 / 帮派打手 / 江湖三流，能打但无亮点（T0~T1）。
  · 后期 Lv.8-10：凡人中的顶尖好手，逼近超凡门槛（T1~T2）。
【二阶 Lv.11-20｜≤80｜正式战力·超凡起点】破坏力=空手碎砖裂石、掀翻壮汉、单挑一小队士兵；抗打=皮肉强于常人数倍、扛拳脚棍棒、热武器仍威胁；速度=快到普通人看不清出手、近距可闪第一发枪火；异能=一点内力 / 斗气 / 异能雏形。
  · 初期 Lv.11-13：刚踏入超凡、比凡人强一截。
  · 中期 Lv.14-17：稳定的超凡战力、一域好手。
  · 后期 Lv.18-20：二阶顶尖、逼近三阶。
【三阶 Lv.21-30｜≤99（凡人极限）｜一域中坚】破坏力=裂石断金、劈开车辆装甲、踏平一座山寨 / 端一个据点；抗打=扛小型爆炸、近身刀剑难破防；速度=乱军中穿插取首级；异能=成型功法、可远程放气刃 / 火球级。
  · 初期 Lv.21-23：独当一面的小头目。
  · 中期 Lv.24-27：区域知名强者。
  · 后期 Lv.28-30：凡人巅峰、属性摸满 99、临近四阶质变。

━━ 超凡段（四~六阶·从「人上人」到「行走的天灾」）━━
【四阶 Lv.31-40｜≤150｜⚡质变分水岭·精英Boss】经「属性觉醒」解锁真实属性（1 点≈5 点普通之效）、HP/EP 骤增。破坏力=摧毁据点 / 掀平建筑群；抗打=肉身硬抗常规热武器与小型导弹；速度=突破音障、近战近乎瞬移；异能=成体系的超凡能力。一城仅数人。
  · 初期 Lv.31-33：刚觉醒的四阶弱者。
  · 中期 Lv.34-37：稳固的一方高手。
  · 后期 Lv.38-40：四阶顶尖、逼近领主级。
【五阶 Lv.41-50｜≤175｜城市级·领主】破坏力=一击夷平街区、全力让一城陷落；抗打=横扫钢铁与常规军队、寻常打击无法破防；异能=显化「领域 / 阵法」雏形。一个世界屈指可数。（📌「深渊」一词五阶起才解封）
  · 初期 Lv.41-43：初显领域的一方新贵。
  · 中期 Lv.44-47：坐镇一方的领主 / 魔将。
  · 后期 Lv.48-50：五阶顶尖、准地区级。
【六阶 Lv.51-60｜≤200｜地区级灾难·镇国】破坏力=覆盖一整片地区 / 大城、移山填海初现；抗打=凡俗军队与热武器近乎无效；机动=御空飞行、长程高速；异能=领域成型、大范围改写战场。
  · 初期 Lv.51-53 ｜ 中期 Lv.54-57 ｜ 后期 Lv.58-60（逐步逼近世界级）。
【七阶 Lv.61-70｜≤250｜人形战略武器·世界级】破坏力=一怒毁一国 / 一片大陆；抗打=肉身硬抗战略级（核爆当量）打击；异能=初掌规则级手段（时空 / 因果的皮毛）。一个世界往往只一两个、甚至没有。
  · 初期 Lv.61-63 ｜ 中期 Lv.64-67 ｜ 后期 Lv.68-70。
【八阶 Lv.71-80｜≤300｜行星 / 星空级·一界之主】破坏力=爆星级、摧毁行星表面；机动=肉身横渡虚空；异能=掌本源 / 规则 / 权柄、战斗涉因果与概念。
  · 初期 Lv.71-73 ｜ 中期 Lv.74-77 ｜ 后期 Lv.78-80。
【九阶 Lv.81-90｜≤500｜文明 / 恒星系级】破坏力=毁灭恒星系、抹杀小宇宙 / 一整个文明；异能=规则重写、概念抹杀初成。一个世界内几乎无人能及。
  · 初期 Lv.81-83 ｜ 中期 Lv.84-87 ｜ 后期 Lv.88-90。

━━ 超脱段（绝强~无上·已非「世界」能衡量·极罕见·非剧情核心不出现）━━
【绝强 Lv.91-100｜≤1000｜凡俗顶点·跨世界顶尖】超脱单一世界的规则级存在，能在虚空万族战场立足、言出法随、掌一域规则。
  · 初期 Lv.91-93 ｜ 中期 Lv.94-97 ｜ 后期 Lv.98-100。
【巅峰绝强 ｜≤2500｜绝强之极】绝强登峰造极者，规则掌控更精深、可独抗一方顶级势力；虚空万族里的一方巨头 / 古老强者。（⚠此阶无独立等级段：以阶位标定为主，等级约当绝强圆满 Lv.100 ~ 至强前段；单属性上限翻倍到 2500）
【至强 Lv.101-120｜≤4000｜多元顶尖】多元宇宙金字塔尖的少数，撼动时空因果、开辟 / 毁灭小世界、掌本源规则；深渊先王、古神级对手。
  · 初期 Lv.101-107 ｜ 中期 Lv.108-114 ｜ 后期 Lv.115-120。
【巅峰至强 Lv.121-140｜≤8000｜至强之巅】逼近无上，可与界外 / 上古深渊存在正面交锋、一念重塑一片虚空。
  · 初期 Lv.121-127 ｜ 中期 Lv.128-134 ｜ 后期 Lv.135-140。
【无上之境 Lv.140+｜属性EX（无上限）｜至高·天选】凡俗认知终点、近乎神话，掌因果律 / 概念级伟力，可公证 / 裁决世界——虚空之树、逢魔级观测者那一层。

【判定铁律】① 先看正文表现 → 定破坏力档 → 定阶位，宁低勿高。② 等级落在该阶 10 级段内、按强弱取初 / 中 / 后期。③ 阶位须与生物强度档（T0-T9）、装备品质、破坏力自洽。④ 受当前【世界巅峰战力】封顶：世界记录写明的世界上限是硬天花板，绝不冒出超过它的 NPC（世界巅峰仅五~七阶，就不该出八阶 / 巅峰至强的路人）。⑤ 越阶表现须有正文明证方可破例。
