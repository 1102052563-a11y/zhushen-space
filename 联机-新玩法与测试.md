# 联机新玩法 · 玩法说明 + Live 测试清单

> 本轮联机大改一览。**全程纯前端、不用重 deploy worker、对普通联机/单机零影响。**
> 改动均已 build + 类型门禁 + 单元实测通过；下面是 **2 端真实联机** 的体感测试清单。

---

## 0. 前置准备

- [ ] `cd zhushen-space/zhushen-space` → `.\node_modules\.bin\vite build` → `commit + push`（Cloudflare 自动部署）。也可本地 `npm run dev` 测。
- [ ] **worker 不用动**（新玩法全走现有通用 relay）。
- [ ] 开 **2 个浏览器会话**（或拉个朋友）：一个当房主、一个当来宾。测完整版双视角/多人建议 3 端。
- [ ] **每位测试者各配好自己的「正文生成」API key**：双视角渲染、脱队单走、隐藏结局生成都用**各自的 key**（没配 key 的来宾：双视角由房主代渲染，但不能脱队单走）。

---

## 1. 三个基础修复（最简单，先测；多数不需要 key）

| 修复 | 操作 | 期望 |
|---|---|---|
| **幻影队友名字** | 房主+来宾各提交一回合行动，房主发送 | 房主正文里队友用**真实角色名**（如「稗田铃」），不再乱入开房花名「道友」 |
| **技能泄漏** | 房主角色在正文里用某技能 | 来宾的 B1 **不会**凭空冒出房主的技能 |
| **NPC 装备好感门禁** | 进某 NPC 详情→储存空间→「取走→我」 | 好感 < 60 且非随从 → 提示「好感不足」拦截；好感 ≥ 60 或随从/宠物/召唤物 → 正常取走 |

- [ ] 幻影名字已修
- [ ] 技能不再泄漏
- [ ] 好感门禁生效（低好感拦 / 高好感放 / 随从免门禁）

---

## 2. 双视角（两套，二选一体验）

### 2a. 完整版双视角（主控-分支-对齐）
- **开启**：房主建房时勾选 **🎭 完整版双视角**。
- **流程**：房主+来宾各提交行动 → 房主发送 → 房主侧出现进度条 `🎬主控推演 → ✍️分配大纲 → 🎨各自渲染 → 🧭主控对齐`。

- [ ] 房主看到**自己视角**的正文（不是客观群像稿）
- [ ] 来宾收到**本人视角**的正文（与房主**客观事实一致、措辞/视角不同**）
- [ ] 没配 key 的来宾也能收到（房主代渲染）
- [ ] 来宾角色的技能/物品演化正确（基于自己的专属大纲，不串房主的）

### 2b. 双视角轻量版（来宾自助改写）
- **开启**：来宾在房间工具栏点 **✍️ 我的视角**（需自己的 key）。普通模式即可，不必开完整版。
- [ ] 房主广播正文后，来宾端的那条正文被**改写成本人视角**（失败则保留原文，不报错）

---

## 3. 分头行动 → 回援 → 化学反应 → 隐藏结局（你的核心愿景闭环）

### 3a. 🚶 脱队单走（需自己的 key）
- 来宾工具栏点 **🚶 脱队单走** → 确认。
- [ ] 来宾出现「🚶 脱队单走中」横幅；房主/全房的队伍名单里该座位显示 **🚶单走中**
- [ ] 来宾发消息 = **本地独立正文**（用自己的 key 跑支线，不等房主、不收主线广播）
- [ ] 房主这边的主线**不包含**该来宾的行动

### 3b. 🎁 跨线支援（现成的转移）
- 脱队的人在支线打到装备/道具 → 用 **赠予 / 交易行 / 私信** 给主线队友。
- [ ] 主线队友**收到**该物品（世界快照不碰个人背包 → 支线掉落随你回主线）

### 3c. 🤝 归队汇流（化学反应）
- 脱队的人点 **归队** → 自动用自己 key 把支线经历**浓缩成见闻摘要**回传房主。
- [ ] 房主**下一回合正文**里出现联动：「X 刚结束独自行动归队，带回……」并与主线呼应（而非无视）

### 3d. 🔮 隐藏结局（集齐剧情道具触发）
- 房主点工具栏 **🔮 隐藏结局**（需房主的 key）→ AI 编织 1~2 个跨人条件，全房看板显示 🔒 目标（如「集齐【断罪之刃】【幽冥司南】」，奖励 ???）。
- 队伍分头去支线搜集这些道具、互传凑齐。
- [ ] 看板出现 🔒 隐藏条件（标题 + 需集齐的道具，奖励为 ???）
- [ ] 全队合计集齐所需道具后，房主下次发送 → 条件变 **🔓 已解锁**，正文**触发隐藏剧情/合体大招/隐藏结局**

---

## 4. 已知小限制（不是 bug，心里有数即可）

- 没配 key 的来宾**不能脱队单走**（支线是完整生成，必须自己的 key）。
- 纯**旁观者**在「完整版双视角」下暂不收正文（没有专属座位）。
- 脱队期间对**共享 NPC** 的改动，归队后会被主线世界快照覆盖（支线视为「别处」）；但你的**角色/装备/道具会保留**。
- 隐藏条件判定用的是队员**上报卡快照**里的物品（提交/归队时刷新），可能有一回合的轻微滞后。

---

## 5. 出问题怎么排查

- 浏览器 Console 看 `[POV] / [汇流] / [隐藏结局]` 前缀的 warn 日志。
- 双视角/脱队不动 → 多半是该端**没配正文 key**（看设置→正文生成→API）。
- relay 不通 → 确认两端连的是**同一个房间码**、worker 已部署。
