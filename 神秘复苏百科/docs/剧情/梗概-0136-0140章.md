# 第136-140章梗概

## 第136章·有身份的鬼

赵磊担心落单的上官云，众人正准备去找人，上官云却先回到包厢并要带苗小善离开。杨间从其衬衫口袋、皮带字母左右颠倒及伤口异常止血看出不对，判断回来的是披着上官云身份的鬼。他以王珊珊控制住苗小善，连开两枪验证，假上官云中弹不死；赵磊叫破其鬼的身份后，对方连同伪装直接消失，只留下黄金子弹。杨间认为鬼需借活人身份显现，随即带众人前往厕所找真正的上官云；包厢玻璃映出六人，实际离开者仅五人，异常仍未解除。（第136章）

## 第137章·被困

真正的上官云洗脸后发现镜中没有自己，转身离开时又被浓黑般的边界拦在镜内厕所，遂拨给苗小善求助。杨间谨慎要求先确认来电者身份，得知上官云所见厕所外一片黑暗后赶到现场；现实厕所有人影全无，巨大镜面内却映出持手机的上官云。双方借电话互相确认位置，杨间指出不是众人在镜中，而是上官云被困镜中；上官云受惊失手摔坏手机，通讯中断。杨间由此确认假上官云消失并不等于危机结束，镜中空间与现实的分隔才是救援难题。（第137章）

## 第138章·镜子里面的声音

杨间分析镜中鬼虽能替换身份，却并非无差别杀人，因此暂时保持冷静；为查明上官云如何被困，他决定继续沟通。众人不敢离开去找纸笔，苗小善便用水在镜面写字，代为安慰并询问表哥。上官云只记得自己洗脸，无法解释进入镜内的过程，恐惧使其难以提供线索。随后他写称听见有人要进厕所，现实侧却毫无声响；杨间据此判断声音只能来自镜内，鬼已进入上官云所在空间。救援从信息搜集转为必须防范镜中鬼直接接近被困者。（第138章）

## 第139章·现身

苗小善先在镜中厕所拐角看见半露的苍白人影，回头确认现实无人，杨间据此推断鬼真身存在镜内、现实中只是不可见影子，且可借镜面寻找出口。他警告众人远离镜子，张伟也由此想起自己曾对镜猜拳。赵磊、张伟、王珊珊先后离开，杨间因同学情分留下协助苗小善。二人继续写字交流果然引鬼现身：它初显为与张伟七八分相似、穿工作服的男子，旋即开始变成苗小善。杨间确认它正以模仿镜前活人为条件进行身份替换，救援进入最危险的观察阶段。（第139章）

## 第140章·成功营救

镜中鬼完全化成苗小善后，紧跟她的写字动作学习并逐步消除延迟；镜内的上官云因看见第二个“苗小善”而崩溃。杨间判断动作完全同步即是内外交换的触发点，提前张开鬼域保护苗小善，但鬼域起初无法深入镜中。当鬼与苗小善同步，镜面产生拉扯，鬼要替代她、她则将进入镜内；杨间立即将苗小善移到身后，以厕所门板顶替原位。鬼因未取得活人身份而消失，交换通道却短暂打开，杨间以红光伸入镜内带出上官云。三人立刻撤离KTV，杨间选择不冒险抓鬼，优先结束这次救援。（第140章）
