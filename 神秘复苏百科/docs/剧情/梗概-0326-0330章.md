# 第326-330章梗概

## 第326章·传教士的录音

保罗以损坏磁带作最后筹码，播放自称“传教士”的男子录音。录音宣称听者会在第二声“嘟”后死亡，保罗借此要求杨间撤人、恢复双腿。杨间判断这更像国外驭鬼者制作的诅咒，承认其可威胁普通人，却不认为足以杀死已半人半鬼的自己。他早已用鬼域把保罗与办公室隔开，转移到严力鬼血复苏留下的高速血池上空；磁带未及第二声即被投入血池压住。保罗试图以一只鬼换命失败，被杨间投入血池，遭血手拖没。刘小雨记录其先发动灵异袭击，总部认可处置；杨间认为迁走附近村民、封禁血池可将严力遗留危害控制在有限范围。（第326章）

## 第327章·三口箱子

杨间回尚通大厦时，章华已被密室内雇佣兵开枪击中两次，保险室门也被同事紧急关回。章华担心自己撑不到救护车，仍要求杨间直接击毙持枪者；杨间以鬼影伸入伤口取出两颗子弹、拼接伤口止血，令其可等待输血恢复。得知密室有黄金箱后，杨间迎着枪火进入黄金贴覆的保险室，鬼域使子弹无效，并瞬间杀死三名外籍雇佣兵。五口箱子中两口为空、三口有物，他带走三箱作为鬼橱交易筹码；另有两名女子获救。章华确认窝藏罪犯、持枪伤人和私藏军火已足以查封尚通科技，杨间则要求保留大楼保险室供日后办公。（第327章）

## 第328章·鬼橱里的照片

杨间把三口黄金箱带到钟塔鬼橱前，决定用其中一只鬼交换赵磊位置，以免篡改记忆的鬼继续潜伏。打开首箱后，他发现箱中不是普通厉鬼，而是一名被金线捆缚、身中弹孔的半具腐烂驭鬼者尸体；尸体抽动表明其体内鬼尚未复苏。杨间谴责保罗一方猎杀驭鬼者夺鬼的做法，随即以鬼影将尸体送入鬼橱。鬼橱交出写有“中山市凯撒大酒店”的13号门牌，继而提出一年内找到照片中一名无眼、尸斑、红旗袍女子的条件。杨间从红灯笼老宅背景联想到鬼车见过的干尸新娘，却无法确认是否同一目标，也不知鬼橱目的。（第328章）

## 第329章·中山市事件

杨间因门牌信息询问中山市负责人，秦媚柔答复为童倩，却同时告知童倩处理灵异事件时已失联三日。杨间立刻怀疑赵磊与篡改记忆鬼正是冲童倩的鬼脸而去，要求撤回两名刑警和三名外援，避免队伍有人被改写记忆后反噬同伴。赵建国却称五人当天也已失联八小时，且总部不能放弃正恶化的A级事件。他提出调用熊文文、鬼烛与其他资源，杨间认为普通事件可能只是记忆鬼布下的陷阱，但因赵磊线索与此直接相连，仍决定次日赴中山市，只带熊文文同行。张丽琴此后查明鬼橱瓷盘为七十年前最后生产的一批景镇瓷器，杨间再度联想到民国线索。（第329章）

## 第330章·两条信息

杨间阅读档案得知失联支援者有冯全、郭凡及灵异论坛三名外援罗素一、黄飞、林落梅；档案记录的中山市人头气球事件会在夜间杀人，把死者人头升至空中，因伤亡增长和规则未知由B级升为A级。为应对记忆篡改鬼，他准备手枪、鬼绳、鬼烛、哭坟鬼遮脸布及深埋的人皮纸。江艳拼好赵磊笔记后，杨间从每页角落字迹拼出“中山市凯撒大酒店31号房”，与鬼橱给出的13号房相冲突。赵磊死前留下的信息与需要他继续履约的鬼橱都似有可信理由，杨间却无法判断哪条为鬼设的陷阱，甚至不能排除两者皆假；尚未出发，行动已被两条矛盾线索笼罩。（第330章）
