# 第216-220章梗概

## 第216章·保护任务

老王的出租车因高速漂移后刹车失灵冲入人工河，翻覆沉底、车窗陷入淤泥。杨间以鬼域把两人移至人行道，独自回小强俱乐部确认王小强已死。大昌市随即被封锁，赵开明提议暂时和解；刘小雨传达总部保护王小明的任务，并以可调用全市银行黄金储备为报酬，杨间接受。叶枫追捕和赵开明旧怨均因此暂缓。（第216章）

## 第217章·入夜

杨间在青黑阴霾下前往警局领取缴获的特制枪械和弹药，并要求刘小雨整理全市灵异档案，防备大事件打破既有平衡。刘队告知被镜中鬼困住的警员张元已在镜内渴死，恳请杨间不要让牺牲毫无意义。杨间无法承诺承担全城责任，只以敬礼回应；城市灯光和阴霾交织，已如鬼城。（第217章）

## 第218章·喜当爹

张丽琴在观江花园别墅等待杨间，称自己短时间内出现异常孕相。杨间触摸后确认其腹部有拳头大小胎儿，且鬼婴面孔隔肚皮咬人；刘小雨否认这与杨间有关。杨间判断张丽琴是大昌市事件酝酿期的受害者，鬼婴继续成长将吃空内脏、破腹而出，决定暂留她并亲自设法取出。（第218章）

## 第219章·不可控的鬼婴

杨间确认张丽琴尚未被鬼婴啃食内脏，仍有救援窗口；她无法控制体内鬼婴，因此只是被附身而非驭鬼者。杨间拒绝去医院，准备在五楼用黄金盒、干枯手指和鬼烛处置。直接以无头鬼影抓取可能连同内脏撕出，他转而设想以鬼域虚化人体、再用鬼影分离鬼婴，并决定先做能力试验。（第219章）

## 第220章·尝试的驾驭

杨间先确认鬼婴可进入鬼域，再令无头鬼影伸入张丽琴腹部；因人体与鬼婴一同虚化，抓取失败。为让人体虚化而鬼婴保留实体，他将鬼影一条手臂融入自己的左臂，以鬼眼压制。鬼影夺取身体的本能导致手臂皮肤开裂、扭曲，杨间最终调动第七只鬼眼暂时压下异动。局部驾驭是否成功、鬼婴能否安全取出仍未揭示。（第220章）
