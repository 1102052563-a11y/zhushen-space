# 第221-225章梗概

## 第221章·成功取出

杨间确认左臂局部驾驭鬼影成功，但须以两只鬼眼压制，手臂冰冷僵硬且裂痕未愈。他以鬼域虚化张丽琴身体、用附着鬼影的手臂追抓逃窜鬼婴，最终无伤扯出。鬼婴试图跳逃，杨间用干枯手指钉住，再连手指装入黄金盒、以金箔密封。张丽琴提到不孕邻居近日也突然怀孕，杨间判断若有同类寄生，整座城市将面临灾难。（第221章）

## 第222章·黑暗的一天

刘小雨为张丽琴的鬼婴案建档，记录其并非个例。次日大昌市早上仍无日出，青黑阴霾持续，恐慌开始蔓延。别墅外枪声吵醒杨间与张丽琴，杨间确认灵异事件已进入爆发阶段；市民暂未大规模失序，但异常仍在扩张。（第222章）

## 第223章·话不能乱说

张显贵带亲属到观江小区寻求杨间庇护。杨间说明自己无法确认灾难能否解决，只能估计全城幸存者或不超过三成；张显贵姐姐轻视其身份，杨间以局部驾驭鬼影的左手掐住她警告，后放人并允许张家人住进小区。张伟拾到黄金手枪，杨间带他去练枪；张丽琴则决定先离开处理自身事务。（第223章）

## 第224章·赴会

张伟展现异常精准枪法，杨间练枪仍难命中远距离目标。刘小雨通知王小明召开紧急会议，并确认市内类似张丽琴的鬼婴寄生至少120例、其中40名男性；杨间立即停止练枪，携人皮纸赴体育馆。会议前叶枫以王小明取消通缉为由出现，双方同意在危机期间暂不清算；张韩和精神不稳的何川也到场。（第224章）

## 第225章·鬼婴超生

王小明在体育馆向九名驭鬼者说明：周正死后逃出的鬼婴经历婴儿、孩童、青年三阶段，推测通过吃人成长。医院资料显示城市内已确认鬼婴从120只增至138只，数量仍可能上升。面对群体恐慌，王小明要求众人先捕获一只鬼婴供其研究，作为处理本次事件的第一步；具体抓捕方案尚未提出。（第225章）
