# 第366-370章梗概

## 第366章·坟土

杨间带鬼报纸穿过鬼雾，循电话来到凯撒大酒店四楼005号房，见到被柴刀鬼腰斩、只剩半截身体的冯全。冯全断口被暗褐色泥土堵住，称此物名为坟土，能压制其鬼雾却会逐渐把自己变成坟丘；他希望杨间从雾中找回被带走的下半身。杨间因冯全存在记忆遭篡改的可能，先断开双方通讯并让无头鬼影侵入其身体，准备以鬼报纸试验。冯全却在鬼影入体后喊出赵磊之名，随即以坟土凝成双腿、抱住杨间发动反击，将两人一同埋进新坟丘。杨间早维持鬼域防备而脱身，听见冯全提赵磊后确认其记忆确受过鬼袭篡改；坟丘持续吞没冯全，救援转为先压制其复苏。（第366章）

## 第367章·取脸

冯全反击失败后，坟土形成的土丘仍在把他吞没。杨间不近身，先以鬼影操控鬼绳套住冯全头部，试图强拉出坟丘，却发现坟土内存在难以撼动的束缚，强行只会扯坏其颈部。为争取压制窗口，杨间点燃上一战后仅余的一小节红色鬼烛；鬼烛面对附近灵异瞬间烧尽，却给出不足两秒的空隙，让他把冯全拖出。坟丘塌成尸臭泥土，内部出现泥塑人形，冯全又有坟土复苏迹象，杨间以鬼影暂压。随后他将鬼报纸覆到冯全面上，约一分钟后取下报纸时连同整张脸一并撕下；纸上显出赵磊留下的“忘记被袭击”“杀了杨间”等未完成指令，证实冯全仅部分记忆被植入恶念。（第367章）

## 第368章·新的冯全

杨间擦去鬼报纸上原有字迹，在冯全脸旁写下忘却恩怨、感激杨间、听从命令、远离挑拨者等内容，又以称呼“腿哥”作为验证标记。他试图直接把脸贴回冯全头上，却发现鬼报纸只能取脸，不能装脸；杨间据此判断完整的取脸、改写、装脸应是那只篡改记忆之鬼的能力，手中仅是残缺灵异。他改用无头鬼影，一部分继续压制冯全，一部分进入报纸，成功让脸脱落并将其拼回头部。约两分钟后冯全醒来，第一句话便称杨间“腿哥”，还认为杨间救了自己，杨间遂判断这次改写成功。他收回鬼绳与报纸，决定离开酒店，并把尚未处理的童倩相关鬼物列为下一目标。（第368章）

## 第369章·收回雾鬼

杨间背着仍只有半截身体的冯全离开酒店，询问同行失散的郭凡。冯全只知郭凡关闭通讯、可能仍躲在酒店，杨间因交情不佳暂不搜救。二人进入鬼雾后，冯全说明雾鬼会按雾中动静追人，普通人应静止等待鬼雾离开；鬼靠近时雾气会侵蚀身体，受害者会化为浓雾，死亡还会使雾更浓、范围更大。杨间收起鬼域、故意行走引鬼，以鬼眼看见雾鬼的上身由浓雾凝成、下身却是冯全失去的身体，残余坟土正暂时束缚它。他待目标进入范围后以无头鬼影抓住其腿，雾鬼和周围浓雾迅速溃散，再把下半身接回冯全；但撤去压制后两只鬼能否自行恢复平衡，仍须当场验证。（第369章）

## 第370章·新的平衡

杨间撤回无头鬼影后，冯全体内完全复苏的坟土压过半复苏的鬼雾，泥土持续涌出、雾气消失，冯全眼眶也被坟土填满，主动要求杨间立刻关押自己。杨间认为单靠持续用鬼影压制无意义，又无第三只鬼可供驾驭，便取出白色引鬼鬼烛，点在冯全身旁。黑色烛火把坟土吸出体外，约两分钟后冯全抓住两鬼力量短暂相平的时机坐起催促熄烛。蜡烛熄灭后，外部坟土试图回流，杨间以鬼影截断其路径，只让部分回到冯全体内，留下渗血的小土丘并装入黄金制装尸袋。冯全称自身恢复此前状态，杨间则确认本次救援、记忆改写和灵异分割均暂告完成。（第370章）
