# 第156-160章梗概

## 第156章·对付的方法

警局监控锁定假张元进入五楼档案室，刘建国立即封锁现场，要求普通警员只警戒、不接近。假张元穿着文字反向的制服，面对命令反复以固定句式提问，刘建国开枪击中其手臂后既无鲜血也无痛觉，确认对方不是人。杨间赶到制止继续射击，指出枪械对鬼无效；他先问姓名，再直接拆穿“你根本不是张元”，假张元便由带笑的正常面孔转为惨白僵硬，像被抹去般消失。杨间说明此法只能处理现实伪装，特殊事件难以根除。警员虽然首次亲眼见到鬼消失，却不能据此认定危机结束，镜内的真张元也仍未获救。（第156章）

## 第157章·紧跟

刘建国以为假张元已经被解决，杨间却让众人看向走廊玻璃：每个人的倒影间多出一名工作服男子，现实里并没有这个人。杨间由此说明镜中鬼并未死亡，只是从被识破的张元身份退回镜内；玻璃能映出它，而其离开镜面进入现实的具体条件尚未明。他拒绝承诺救出张元，因为只有鬼本身能进入镜内，先前救上官云还依赖同镜位置与苗小善成为诱饵，本次无法复现。杨间沿玻璃倒影追至一楼后失去目标，随即想到鬼只会在张伟出现处活动，决定改带走拘留所中的张伟，把风险从警局人群中移开。刘建国以特殊事件优先命警员配合放人。（第157章）

## 第158章·自己看

杨间接出张伟后，告知其已被鬼盯上，张伟却先沉迷于杨间车内黄金手枪，提出用三栋别墅和三间店铺换一把枪及子弹，杨间同意交易。张伟只记得最初在家中遇到异常，杨间便决定回别墅查源头。路上，杨间看见与张伟相貌相同的工作服人影站在车前，车辆碾过却毫无碰撞；再看后视镜，镜中鬼已坐在后座，张伟仍毫无察觉。刘建国此前已以天然气泄漏为由封锁别墅，杨间进入后判断房屋本身无鬼，问题在于鬼跟着张伟。他拿起化妆镜要求张伟自己环照四周，令其亲眼面对危险。（第158章）

## 第159章·场面凝固

张伟起初不敢照镜，最终转动化妆镜时在沙发后看见穿脏工作服、脸色惨白而眼神空洞的人影，对方还似乎转头看向自己；他当场摔碎镜子。杨间以肉眼和鬼眼都看不到鬼，确认它暂时只能借镜面显现，且目前无法直接干预现实。他认为镜中鬼已跟随张伟约一周，别墅区人口稀少使其较少有机会替换身份，而上官云和张元在人多处遇袭显示风险会增加。两人随后听见二楼声音，以为另有鬼物，张伟踹门后却发现父亲张显贵与黄姓家政阿姨在房内，四人因意外撞见私事而陷入尴尬；灵异源头依旧没有线索。（第159章）

## 第160章·张显贵

张显贵与黄姓家政阿姨整理后在客厅接待杨间。他对第七高中传闻和杨间处理特殊事件的身份仍保留怀疑，但承认曾调查学校、发现其中有疑点。杨间说明警察封锁别墅并非真因天然气泄漏，而是避免灵异风险扩散；张显贵则坦言自己通过装修时留下的逃生通道进入。为取信对方，杨间让他拿镜子查看别墅。张显贵先在张伟指出的沙发处一无所获，正以为受鬼故事影响，镜中却映出二楼护栏旁站着一人；他抬头看现实空无一物，再看镜中人影已转身消失。张显贵至此恐惧失语，确认镜中鬼确实存在，但张伟被跟随的原因和救张元的方法仍无答案。（第160章）
