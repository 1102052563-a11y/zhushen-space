# 第396-400章梗概

## 第396章·代号鬼掐人

杨间检查洗手间女尸，确认尸臭来自曾在机舱徘徊的厉鬼，而非刚死不久的受害者。乘警王东先按普通凶案与他交涉，总部随即命令王东无条件服从杨间；杨间借枪给王东稳住乘客，并从死者扭转多圈的脖颈及左右成形的女性手掌淤青判断，死者像是被一双手逼着掐死自己。他为事件建立恐怖级别C的“鬼掐人”档案，又查知半年内该航班无类似死亡，推断鬼很可能由某位乘客带上飞机。面对以鬼域疏散乘客或让飞机继续飞行、在机上解决厉鬼的取舍，总部最终选择就地处理。杨间接受命令，但航班乘客由此成为处置风险的承担者。（第396章）

## 第397章·被入侵的客舱

总部决定不疏散后，杨间判断机上除自己外已排除驭鬼者，厉鬼可能伪装成乘客或随乘客而来；因装备有限、身体恶化且没有帮手，他先与王东巡视，未发现明确目标。一名浓妆女乘客因被查看而大声指控杨间，王东出面解释，杨间却宣布暂停半小时行动。他直言自己要等鬼再次出现，在绝望中救人以取得能开展工作的威信，秦媚柔认为这会害死乘客，双方因此争执。等待期间，后舱灯光和信号被阴冷气息压暗，王东察觉异常，万德路和西装男子也亲眼见手机光无法照亮黑暗区域。杨间确认鬼已进入客舱，仍选择等待其杀人以锁定目标。（第397章）

## 第398章·扭转的脖子

为配合机上处置，客机改在无人旷野上空盘旋，乘客尚不知道灵异事件。王东限制乘客周浩去厕所，周浩回座后注意到前三排中年乘客后脑多出一只惨白、带尸斑的手；该乘客自己的双手按住脸颊，额外的前后两只手随之扭转其脖子。周浩眼见颈骨断裂、头颅被连续拧转，直到中年人脸色发黑发紫、明显死亡才崩溃尖叫逃开。其他乘客被尖叫惊动，杨间在头等舱平静判断“鬼开始杀人了”。本章确认厉鬼会以实体手掌参与杀人，但触发条件、手掌来源及如何阻止仍未明朗；第二名死者出现也使原本可维持的平静正式破裂。（第398章）

## 第399章·第二位死者

第二起死亡令头等舱乘客开始相信飞机闹鬼。杨间告知万德路等人，航班未飞往大京市而是在空中盘旋，若事件不能解决，众人只能死于厉鬼或坠机；他强调自己处理的是灵异事件，不承诺保护每个乘客。西装男子指责杨间冷血、催其履职，却不敢接过可攻击厉鬼的特制手枪，杨间遂以耳光、掐颈和鬼眼震慑，迫使头等舱乘客安静留座。秦媚柔转来周浩的口供：第二位死者头上曾有一双死人手。杨间据此认为厉鬼存在实体，决定下次出手即行动，并拒绝秦媚柔继续干预，要求以后改由刘小雨联络。随后他进入昏暗客舱，准备查看第二具尸体。（第399章）

## 第400章·多出的手

杨间抵达第二名死者旁，见其脖子扭成麻花、身上留有多处手掌淤青；周浩虽惊恐失常，仍说出死者脸前脸后各有一只不属于本人的手。杨间比较两起死状后怀疑这可能是杀人后会变化的成长性厉鬼，却仍不知道其选择目标和杀人规律。他命王东公开灵异事件、让所有人回座，自己站到乘客前方，故意不开鬼域，以乘客及自身作诱饵等待第三次袭击。乘客因恐惧要求立即降落，王东没有权限答应；此时杨间以鬼眼发现一名女乘客僵住，带尸斑和淤青的惨白手掌从其身后伸出、搭上肩膀。杨间已锁定新的袭击征兆，但手掌下一步如何杀人及能否成功限制，尚未写明。（第400章）
