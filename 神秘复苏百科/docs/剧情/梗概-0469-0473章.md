# 第469-473章梗概

## 第469章·热情过头

杨间脱离培训基地后回到平安酒店清洗身体，却没有把鬼差离开视为终结。他想到自己改变鬼差数量判定后，驭鬼者虽可能不再是目标，体内没有鬼的普通人反而会被视作落单，若鬼差游荡入城市，后果不亚于大昌市旧事。因此他准备维持与刘小雨的通讯，至少提供已知情报。黄子雅因获救前来致谢，并直言希望借短暂亲密关系得到顶尖驭鬼者照应；杨间称仍要处理事务，拒绝其提议，只收下号码。张雷、王江在走廊旁观并以结果打赌，令黄子雅恼怒离开。杨间随后主动联络刘小雨，询问王小明与鬼差在实验室是否相遇。（第469章）

## 第470章·无限重启

实验室外，鬼差形成覆盖数里、棺材形状的浓黑鬼域。王小明没有撤走，认为大京市附近的普通人无法逃离，遂与李军、童倩等顶尖驭鬼者正面应对。众人依已知数量规律本不应成为目标，却发现鬼差每次都追向没有驾驭厉鬼的王小明；他们先后以压制、特制箱和分解关押尝试处理，鬼差都在受限后重启，以同样姿态发动下一轮袭击。王小明推断目标可能因被带出的鬼棺而来，决定改为让鬼棺与队伍分离。酒店内的杨间从刘小雨处得知僵局，判断王小明短期不至死亡，却不会无偿主动参战。（第470章）

## 第471章·奇货可居

李军经刘小雨接通杨间，询问鬼差为何在每次限制后重启。杨间说明自己在培训基地改变了鬼差规则：它如今不再把驭鬼者当作袭击目标，普通人王小明才会持续遇袭；这既解释当前现象，也意味着普通城市仍有巨大风险。被问及关押方法时，他以黄岗村经历提出，让鬼回到鬼棺或许可平息事件。李军立刻把情报转给队伍，王小明认为方案与自己判断一致。杨间明确只提供信息，保留一个回棺失败时才有价值的未公开方案，继续在酒店等待。鬼域内，王小明提出自己独自入棺引鬼，李军则怀疑其想借事故自杀而反对。（第471章）

## 第472章·解决？

众人因培训基地伤亡和王小明的状态劝其不要冒险，王小明仍认为唯有自己作为落单普通人能引鬼回棺，并要求队伍撤开。连续重启让他记录了鬼差出现和靠近的时间；第七次袭击来临时，他没有亲身躺入棺中，而是以枪打穿装有替死布娃娃的黄金盒，将血滴入其中，再把盒子抛入破损鬼棺。布娃娃替代王小明成为鬼差目标，逼近的阴冷气息随即转向棺内，撞击声响起。鬼差进入鬼棺后，笼罩四周的黑暗开始消散，冯全等人看见与黄岗村相似的一幕。众人一度认为S级鬼差已被王小明独自解决，但棺盖破损是否会留下隐患尚未得到答案。（第472章）

## 第473章·鬼棺杀人

阳光重现、鬼差未再立即重启，姜尚白因此轻视事件危险；童倩和曹洋则指出，眼前结果依赖杨间改变规则、已有情报和王小明掌握的鬼棺联系。王小明观察残破鬼棺，发现覆盖棺口的黑色“棺盖”并非实体，子弹可以穿过，遂推断棺材形鬼域或许就是鬼棺本体，身在其中无法完成关押；卫景所化部分也可能对应棺盖。由此他意识到回棺未必是限制，反而可能让鬼逐步完整，立即命令以黄金尽快处置。棺内敲击却愈发剧烈，黑暗棺盖溃散外涌；两名正在操作的未具名驭鬼者尚未来得及退离，脖颈便被扭断死亡，回棺造成的短暂平静彻底破裂。（第473章）
