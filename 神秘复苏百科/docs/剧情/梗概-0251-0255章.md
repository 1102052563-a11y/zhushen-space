# 第251-255章梗概

## 第251章·恐怖继续

王彬一家入住观江花园后，杨间重申小区只相对安全，不能保证其免于鬼婴；他因九眼复苏后的疲累和失衡，决定暂不再随意动用厉鬼。洗浴时，他发现肩部被第四阶段鬼婴搭过的位置留下无法冲掉的青黑大手印，确认自己也受了诅咒；虽想到鬼镜复活和人皮纸记载的凶险驾驭法，仍选择先等安全屋完成。医院内，遗留黄金箱遭持续敲击后变形破开，叶枫外貌的第四阶段鬼婴从中脱出，吞食干枯手指后穿回寿衣，连叶枫残头也被阴霾吞没；孙义尸体与断臂则仍被遗弃。此前的暂封彻底失效，城市恐怖继续扩散。（第251章）

## 第252章·收割

医院行动失败四五天后，大昌市被阴霾封锁已十余日，网络上开始流传孕妇短期孕育鬼婴、城市闹鬼的消息，但外界仍多将其当作化工泄漏或谣言。王小明完成报告，主张彻底隔离城市并凭四阶段鬼婴和源头鬼情报等待后续针对性支援；城市街头的人气却迅速消失。观江花园的安全屋因工人恐惧、不断离开而减慢，张显贵把日薪提到一万元仍难维持进度。杨间、张伟和江艳以游戏暂时排遣，随后全城停电；王小明告知市中心出现大量鬼婴，判断源头鬼已开始主动“收割”整城。杨间承认安全屋仅能容十余至二十人，拒绝以自己的复苏风险去承担全城救援。（第252章）

## 第253章·队长计划

王小明以大昌市和张伟死而复生的异常催杨间再试，承诺鬼烛、驾驭第三只鬼的协助及更多秘密，并提出国家将集中资源培养顶尖驭鬼者的“队长计划”：若杨间解决饿死鬼事件，可获推荐。杨间承认或许能再次正面接触源头鬼，却估计成功率仅一两成，且仍缺少限制目标以完成关押的第二条件。王小明提出把安全屋作为引鬼容器，被杨间立即拒绝并以枪警告；杨间随后要求刘小雨派两名武警看守王小明、限制其行动和进入五楼。收到短信后，杨间改变主意，称要出去“扮耶稣”，安排张韩处理事务、张伟阻止王小明上五楼，独自离开小区。（第253章）

## 第254章·死去的城市

杨间驶入死寂的大昌市，见街道只余阴霾、惨叫、鬼婴啼哭与残尸；他撞到一只第三阶段鬼婴却始终不出声，借“听见声音才袭击”的规律安全离开，印证王小明的阶段判断。刘小雨以卫星电话提示有人要与他通话，杨间未理会，转到南城区一处有尸臭的旧小区赴短信约。上楼时他想起第七高中厕所门后的白骨树、被棺材钉钉住的巨大黑影，推测棺材钉或能替代失去的干枯手指限制源头鬼；但取钉将放出未知黑影，故只作为未实施的危险设想。到四楼住宅后，他发现血迹、拖痕、尸体残骸和一只第二阶段鬼婴，谨慎避开接触后继续搜寻。（第254章）

## 第255章·杨间的决心

杨间用椅子砸开挡路的第二阶段鬼婴，在阳台找到躲藏两三日的张丽琴并直接带离。小区门口一名女记者认出他、不断求救；杨间以文字提醒她三分钟内不可说话，记者仍连番开口，引来多只第三阶段鬼婴并被拖入阴霾。杨间载张丽琴走绕城高速，得知屋内死者是其同事，又发现她胸口已出现青黑诅咒印，坦言自己也未必能救她。张丽琴因恐惧和绝望试图以亲近安慰他，杨间没有接受；望着阴霾中的城市，他压下怒意后决定不再只等死，告诉张丽琴要“和命运赌一把”，驾车继续前进。具体计划与她的诅咒结果尚未揭示。（第255章）
