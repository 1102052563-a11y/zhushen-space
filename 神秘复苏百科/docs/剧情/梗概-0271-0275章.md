# 第271-275章梗概

## 第271章·无解的源头鬼

第七中阴霾浓到鬼眼失去视野，杨间以自身为饵等待源头鬼，却先听到一双自行行走的皮鞋在一米外停留、绕圈。他没有贸然把棺材钉刺向脚步，因而避开了诱饵；真正的鬼始终站在身后，趁其注意力被吸引时以青黑手掌掐住脖子、以干枯手指钉住持钉手腕，并张开鬼牙欲咬头。杨间张开鬼域、放出鬼绳暂缓袭击，再令无头鬼影接钉反击，眼看将成时却恍惚片刻。目标与伤口同时消失，校园时间倒回四十分钟前，赵开明预留的录音又重新播放。杨间保留了先前记忆，确认源头鬼能在局部重置时间，近身计划已无法重用。（第271章）

## 第272章·五秒计划

杨间向远程监听的王小明报告时间回退，二人对照时间后确认重置并未影响校园外；王小明承认张韩诱饵方案错误，并在五秒内提出远距狙击：让鬼眼鬼域凝成细线，从重置范围外照到目标，再把棺材钉直接转移进其体内。该方案只有一次钉子机会，还须解决穿透阴霾和锁定源头的难题，但杨间仍决定实施。他离开第七中，到一千余米外通讯大楼信号塔观察学校，以死机鬼影覆盖身体来压制鬼眼复苏，准备把力量推至极限。此时他尚未突破阴霾，也未确认能否完成狙击。（第272章）

## 第273章·限制成功

通讯塔上，杨间先将极限鬼域压成红光，仍只能抵至七中门口。他由白骨树黑影仅有一眼想到叠加的应是鬼眼本身，便将两眼、再将额头一眼与掌中双眼排成一线，连续在鬼域内张开三层鬼域。红光遂撕开学校阴霾，逐处搜索操场、教学楼和实验楼，最终在北门门卫室发现穿寿衣、顶着周正面孔的饿死鬼。目标被红光照到后转头，杨间立刻把身旁棺材钉转移至一千余米外，贯穿其头颅。鬼物随即如普通尸体倒地，全城阴霾开始消散；这次攻击快到目标来不及再重置时间，饿死鬼事件暂告结束。（第273章）

## 第274章·支援进城

城外指挥中心见阴霾消散，王小明报告杨间已限制源头，曹延华却先派赵建国、李军率驭鬼者与特战人员入城确认，再安排后勤、医疗和物资后续进入。赵建国带十五名刑警、李军带十名特战人员乘防爆车越过警戒线，并分队前往观江小区接王小明。杨间则守在第七中北门，确认头部被棺材钉贯穿的饿死鬼仍无反应，并判断衍生鬼婴已随源头受限而失去力量。赵开明以阴霾幻影现身，称计划成功、本人已远离学校，又提醒杨间功劳可能招来敌意；幻影很快溃散。赵建国抵达后带来金箱，准备正式接收尸体。（第274章）

## 第275章·接收

赵建国见尸体面貌近周正虽感震惊，仍依杨间要求不触动棺材钉。杨间警告饿死鬼是多种厉鬼的综合体，限制成功取巧而来，最好永久封箱；支援刑警郭凡则认为团队、有王小明计划和棺材钉后不难应对，遭杨间反驳，赵建国也制止其轻视事件。接收组以金箔包裹尸体、金绳打死结后装入厚金箱，准备焊死并防范外围国外势力。杨间回观江花园，见小区已有警戒和发电；李军代表各方致谢，江艳确认安全屋人员脱险。饿死鬼造成的危机虽暂时结束，杨间仍担心鬼镜被王小明取走，且总部后续封存能否维持尚未验证。（第275章）
