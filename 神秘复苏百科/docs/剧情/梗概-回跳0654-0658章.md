# 原文回跳第654-658章剧情梗概

> 本文件对应原文在第663章后回跳出现的第654-658章，与此前已整理的正常第654-658章不是同一段剧情。

## 原文回跳第654章·撕碎的诅咒

杨间带鬼童撤离木屋时，壁橱尸体鬼追来，令整栋房屋的距离继续错乱。杨间判断鬼童原本被压在壁橱中，恰好维持了厉鬼无法对付自己的平衡；鬼童脱困后，他已满足袭击条件。穿满旧衣的尸体显形，诅咒随即落到杨间衣物上，衣服被撕裂时，他的皮肉和骨骼也同步产生被撕开的剧痛。鬼影只能暂时抵挡，脱衣也无效。杨间以鬼手压住受侵蚀的外套，终止袭击，再令鬼童用死人头压制尸体鬼，成功离屋。衣物掉落后露出除灵社制服，证明该鬼是死去社员复苏所致；杨间不愿偏离敲门鬼目标，留下它继续前进。（原文回跳第654章）

## 原文回跳第655章·骤至的凶险

美岛继续带队深入神户，街区因敲门鬼长期游荡而老化、破败，沿路出现死者和鬼奴。杨间判断鬼奴可被轻易清除，却足以耗死普通人；随后美岛手机收到未知来电，她感到电话另一端不是活人并将其摔毁。杨间说明这是鬼奴拨号传播敲门诅咒，接通会听见敲门声、招来敲门鬼，命众人丢弃普通手机。队伍逐渐接近目标时，美岛只能感知“就在这里”。酒井突然倒地，头颅无声脱离脖子升上天空，杨间未察觉任何前兆，只能怀疑他满足人头气球的未知杀人规律。协助者首次确认死亡，人头气球与敲门鬼的双重威胁同时逼近。（原文回跳第655章）

## 原文回跳第656章·点燃的蜡烛

酒井被斩首后，尸体有复苏迹象。杨间为不让其影响后续，把无头遗体丢进附近房屋暂时隔离；美岛受持续感应折磨，只能确定鬼在周边，无法给出位置。人头气球已开始杀人，杨间不愿再等待，转而带队到十字路口，拿出白色鬼烛，让穿鬼寿衣的鬼童持烛。黑色火焰燃起，街道立即陷入更深黑暗，鬼童则成为吸引厉鬼的靶子。杨间以鬼眼观察，解释白烛会吸引周边的鬼，自己不能承担持烛风险。最先靠近的是鬼奴，长泽只听到四周脚步，恐惧地意识到越来越多东西正向他们聚集。（原文回跳第656章）

## 原文回跳第657章·涌来的凶险

白烛燃烧的同时，敲门鬼仍在神户一栋公寓内杀人。长衫老人对开着的门继续敲击，屋中男女和接过诅咒电话的幸存男子都在听见敲门声后无伤死亡；老人行过处建筑加速老化。白烛的牵引最终改变其路线，使它向十字路口走去。另一边，杨间用鬼影清掉靠近的鬼奴，但空中死人头正向烛光汇集，鬼童也被鬼奴围住，只靠寿衣和尸水抵挡。长泽请求立即吹灭白烛，美岛因感知到四周全是鬼而崩溃，杨间拒绝中止，认为敲门鬼尚未出现、计划必须继续。（原文回跳第657章）

## 原文回跳第658章·源头的出现

鬼烛持续引来鬼奴、木屋穿衣鬼和酒井复苏的无头尸体，杨间担心若敲门鬼再不来，全队会先被其他灵异围死。他观察到空中腐烂人头坠落后，总有新的活人头颅飞起补位，推测所有人均可能被人头气球选中。为防止自己也被斩首，他取一尊鬼瓷，以头发作身份媒介启动，留下临战保险。随即街边店铺老化，穿长衫、满身尸斑的敲门鬼从黑暗中出现。杨间命鬼童吹灭白烛、抱死人头靠近，自己也伸出鬼手，以死人头注视和鬼手压制同时攻击老人。敲门鬼能否被成功限制，尚未揭示。（原文回跳第658章）
