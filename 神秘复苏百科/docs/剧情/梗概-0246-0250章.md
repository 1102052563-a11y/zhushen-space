# 第246-250章梗概

## 第246章·消失之人

鬼烛熄灭后，杨间先前发声已使附近第三阶段鬼婴锁定自己。他解除五眼对无头鬼影的压制，鬼眼连续复苏至第八只，再吞下一眼刺激出第九只；红光鬼域在阴霾压制下仍扩展至约一千米，将医院内普通鬼婴隔绝在外。无头鬼影则立即试图摘取杨间头颅，张韩以背后无皮鬼手将其压住，自身却也开始遭鬼影侵入。正当杨间赌最后一眼复苏前源头鬼会出现时，一只青黑手臂闯入鬼域搭上其肩；他用干枯手指刺去，却被一件绣寿字的旧寿衣挡住。抬头所见竟是失踪多日、面色青黑的叶枫，杨间由寿衣和外貌判断来者并非普通鬼婴，计划第一次钉击落空。（第246章）

## 第247章·最后的烛火

杨间认出叶枫外貌的目标拥有寿衣，又从其裂开的口中看见能咬开鬼域的怪牙，确认它继承了叶枫两只鬼的力量。目标一口咬开红光鬼域，黑青阴霾随裂口侵入；张韩也因压制鬼影而近乎被背后鬼手掐死。杨间无法从失控鬼域逃离，只能点燃仅余约十厘米的鬼烛争取最后机会。该目标竟直接吃掉近半鬼烛，余火却仍让它短暂停滞。杨间绕开寿衣，赌其颈部无法被及时遮挡，以干枯手指刺入喉咙。目标随即静止，鬼烛火势亦缓和，但杨间没有确认是否真正成功，只能立即转救已濒临失控的张韩。（第247章）

## 第248章·不动的尸体

杨间趁鬼烛最后火苗尚在，把五只鬼眼重新转入无头鬼影，令其像五枚钉子压回地面，并从张韩体内剥离；张韩得以停止被自己和背后鬼手掐颈。鬼烛熄灭后，杨间仅余四只鬼眼，复苏暂被两鬼平衡承担，但已接近极限。叶枫外貌的尸体仍被手指钉住，二人试图以鬼影脱下寿衣，刚解一粒扣子便见尸体手、眼异动。杨间判断寿衣与鬼牙可能互相平衡，脱衣会使鬼牙彻底复苏，遂放弃分离，连同寿衣将其装进黄金箱。杨间随后找到仍守在孙义尸旁的王小明，却发现赵开明已在混乱中消失；三人决定先离开医院。（第248章）

## 第249章·绝望的结果

杨间以鬼域把王小明和张韩带入红光范围，承认钉住的不是源头鬼，行动只暂时限制一只高危目标。王小明将其定为第四阶段鬼婴，认为它能吞食并驾驭叶枫遗留的厉鬼；他又推测第四阶段会因吸入黑青阴霾而袭击全城人，前三阶段则各自按看见、接触、听见触发，首次袭击挡住后暂可避开。医院手术室方向忽有浓郁阴霾如潮吞没墙地，压缩杨间鬼域，三人仓促转移至数百米外楼顶。黄金箱却在转移瞬间被医院内伸出的长青黑手抓住，连手指、寿衣和目标一同遗失。王小明认为缺少限制源头鬼的手段、鬼烛又耗尽，行动已从局部得手变为全局失败。（第249章）

## 第250章·意外的客人

杨间不再停留市中心，以鬼域带王小明和张韩回观江花园。小区施工仍在连夜赶建安全屋，少量楼房已有住户；张韩打算接家人避难，杨间赠其一栋别墅，王小明则要电脑撰写行动报告。杨间表示总部要求自己保住王小明，暂不追究王小强之死，却以此次救援要求日后补偿一件实验室新产品。别墅内，江艳早已接待王彬、王海燕和王珊珊一家。王珊珊脸色苍白、情感冷淡，手腕鬼眼烙印更清晰，并称能看见异常画面，故带家人找到此处。王彬亲见街头有人被吞食后求安置；杨间承认全城危险，仍因同学情分让江艳为其家人准备一套房，观江花园暂成避难点而非绝对安全处。（第250章）
