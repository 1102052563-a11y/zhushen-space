# 第550-554章剧情梗概

## 第550章·室友

苗小善的室友张霞在去美术馆带回老旧油画后失踪，苗小善终于拨通杨间电话。寝室里孙于佳、王悦等人把失踪当作普通外出，也取笑苗小善借题联络旧同学；只有刘紫因男友曾提及杨间身份而有所动摇。杨间赶到后确认油画令鬼眼无法延伸进寝室，判断其属于鬼画诅咒，带走已做好撤离准备的苗小善。苗小善仍想救室友，刘紫也因杨间不可能短时间到达寝室而恐惧逃离，孙于佳与王悦却留在画旁。寝室人员由此分散，油画的袭击风险落到未撤离者身上。（第550章）

## 第551章·有问题

杨间以鬼域把苗小善带到空无一人的篮球馆，说明鬼画只要出现就会杀人、对普通人而言与七中鬼一样危险；他坚持不能为救所有人先赔上身边人，苗小善则请求至少疏散整栋宿舍。杨间通过刘小雨联系总部，报告女生宿舍有鬼画并要求撤离人员，总部答应派队协助，他也接受了处理现场的责任。另一边，寝室仅余孙于佳和王悦，她们重新推想杨间出现速度和刘紫离开后的异常。孙于佳查看张霞床铺时发现油画已经自行移到王悦床头，二人终于决定逃离，但危险已先一步进入寝室。（第551章）

## 第552章·凶画

孙于佳和王悦看见油画自行换位后开始收拾撤离。画中原本五官模糊、穿红色欧式礼服的女人被放大，苍白双手伸到画框外；孙于佳因恐惧无法喊出警告，王悦则为取包返回床边。她被手掌抓住手腕，倒下的油画压住身体并像无底洞般将其吞入，孙于佳最后只看见画外挣扎的双腿逐渐静止。此时刘紫已跑到操场，发现杨间和苗小善不可思议地消失，转而拨打苗小善电话求助，却不知道寝室内已发生袭击。王悦的生死、张霞所在和孙于佳能否逃出均未在本章确认。（第552章）

## 第553章·准备工作

李瑶在贺天雄别墅外等待杨间时撞见仍存活的贺天雄，险被对方因泄密而杀。贺天雄因惧怕杨间追杀，夺走李瑶车辆准备逃离，并留下别墅钥匙和赔礼，称不再找杨间麻烦；杨间在电话中认为他已被吓破胆，威胁较方世明更小，拒绝立刻追杀。篮球馆内，苗小善得知自己的求助耽误了杨间原行动，杨间仍以其安全为先。刘紫随后抵达，杨间确认自己将苗小善留在空旷处，实际是在观察她是否已被鬼画盯上。总部车辆与火警疏散警报迅速抵校，杨间转而开始处理校园事件。（第553章）

## 第554章·坠落

总部支援队负责人王泉与涂校长在篮球馆向杨间报到。杨间让直接接触油画的刘紫接受观察，把苗小善留在馆内并交给她红色鬼烛和打火机，随后要求排查美术馆，警告必要时可用破坏手段降低搜寻风险。宿舍楼疏散缓慢，四楼窗外却出现哭泣发抖的孙于佳；杨间由其宁可翻窗逃命判断楼内出现了比坠楼更可怕的事，拒绝贸然派人入楼。孙于佳最终松手坠下，被杨间单手接住。杨间命人问清楼内经过，只带王泉和两名人员进入宿舍楼调查，油画本体、王悦和张霞的结局仍未揭示。（第554章）
