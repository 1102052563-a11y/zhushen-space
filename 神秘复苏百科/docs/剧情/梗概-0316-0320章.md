# 第316-320章梗概：离开鬼车与鬼橱首笔交易

## 第316章·消失的鬼车

灵异公交驶入正常城市后，杨间与许峰仍因熄火事件互相威胁；杨间明确表示会在离车后要求总部撤许峰职，许峰则不愿以离车后的复苏风险同他决斗。早晨七点半，车辆终于在正常站台开门，杨间因自身没有许、林二人的复苏限制而立即离开。临别前，他答应替林北向家人报平安，却又以五千万挑拨对方在车内对付许峰。杨间警告排队上车的普通人鬼车危险，回头却发现面前已经换成另一辆普通公交，灵异公交与车内的许峰、林北在极短时间内消失。他由此确认鬼车并非可随时回返的避难所，许峰遗失的卫星定位手机成为日后定位车辆的唯一线索。（第316章）

## 第317章·X档案

杨间回顾半夜鬼车经历，认为正常行人虽无知却暂免灵异威胁；他判断鬼车对复苏驭鬼者价值极高，却只能依总部全球定位追踪许峰手机。联络秦媚柔后，他要求建立代号“鬼车”的X级档案，即无法按常规危害等级定义、但极重要的神秘事件档案，并按亲历过程说明哭坟鬼、干尸新娘、不可见鬼、熄火与离车。通话末，他请总部向林北家人报平安、以正当名义汇款一千万，继续追踪许峰手机，并上报撤许峰职、通缉许峰的请求。档案使鬼车信息进入总部体系，但具体追踪和处置尚未展开。（第317章）

## 第318章·五句话

五日后杨间回到大昌市，重读程雄自鬼橱原主人记事本压痕解析出的五句话：询问身份、交易耗尽、索回被橱子拿走之物、寻找某只鬼，以及“我想离开这里”。他推断原主人至少使用过鬼橱四次，鬼橱可能以不对等交易形式提供帮助，却仍因密室、黄铜门与警示被原主人深深忌惮。章华带来尚通科技搜查令、总部配给的一根鬼烛及五百公斤黄金已送达的消息；赵建国则告知总部将哭坟鬼调为B级、干尸新娘定为C级，鬼车档案秘密移交，并批准撤许峰职但拒绝通缉。杨间拒绝新的出差，决定先休整并继续研究鬼橱。（第318章）

## 第319章·另类的回答

回到观江花园后，杨间向江艳简略说明鬼车险情，安排未来让张丽琴担任秘书、江艳管财务，随后带纸笔进入荒宅第二密室。为免工人被灵异波及，他先让张显贵停工疏散。杨间模仿旧主人，将写着“你是谁”的纸塞进鬼橱上层；纸被橱内力量拉入，十秒后从下层缝隙落回，纸上多出歪斜黑字“鬼”。他由此确认橱内存在可书面回应的东西，却没有因好奇打开橱门，转而准备以问题和交易测试其规则。（第319章）

## 第320章·和鬼橱的交流

杨间先询问鬼橱能否找到赵磊，获得“你给我眼睛，我给你赵磊”的回答，判断对方索要的是鬼眼，拒绝直接交换；但“给你赵磊”而非仅告知位置，也显示鬼橱或有转移目标的能力。他改要一碗蛋炒饭，鬼橱索一碗血。杨间看出回答未指定人血，取大狼狗血试探。鬼橱上门自行打开并取走狗血，下层门随后开启，交付一碗仍冒热气的蛋炒饭；饭碟下压着新纸条，要求“帮我把橱子搬出去”。交易成功证明其可进行实物交换，却同时提出未被杨间答应的额外要求，橱外后果尚未写明。（第320章）
