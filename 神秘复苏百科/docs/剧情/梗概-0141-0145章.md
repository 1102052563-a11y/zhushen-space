# 第141-145章梗概

## 第141章·分别

杨间以鬼域把苗小善和上官云带到距KTV三公里外的人行道。上官云虽被救出却惊魂未定，杨间说明镜中鬼并非最凶险的一类，仍提醒两人远离镜面、及时回家；苗小善在父母到场前感谢杨间，并说自己将离开大昌市。镜中鬼则在KTV镜内恢复张伟的样子，走回镜中黑暗，表明它仍会寻找镜前替身。张伟逃离后迷路，误把色情场所理解为可“吃鸡”的地方。此次聚会的救援结束，但镜中鬼未被关押，普通人仍可能成为其目标。（第141章）

## 第142章·被抓的张伟

杨间回到江艳公寓，确认江艳已在五天内花完四亿元、购得四百公斤黄金；他将黄金视为应对灵异事件的资源，决定避免继续接触卖家，并考虑搬到人少的市郊。无头鬼影一度自行立起，杨间向江艳说明自己需持续压制第二只鬼，留在身边存在危险，却仍让她继续处理事务、筹备车辆。张伟电话中得知上官云获救，随后说起父亲工地失踪数名工人，杨间准备查看。张伟在误会中被警察带走，杨间暂认定普通拘留不会出事。（第142章）

## 第143章·复苏的鬼血

夜里杨间察觉天花板滴落腐血，循血脚印找到濒临复苏的严力。严力告知小强俱乐部已盯上杨间，黄金盒也被抢走，并托杨间通知妻儿离开大昌市；他不愿在市区复苏，遂驾跑车高速北去。途中严力身体完全融化为鬼血，失控车辆翻入田野，鲜血持续涌出形成不断扩大的血池，最后有染血手臂和人形从中爬出、朝最近村庄移动。严力以远离城市保护他人的意愿结束，但其厉鬼已摆脱压制并成为新的外部威胁。（第143章）

## 第144章·残忍的现实

严力的异常死亡令杨间怀疑有人刻意加速其复苏。他按托付前往严力别墅通知家人，却在紧闭、断电的屋内闻到尸臭，破窗进入后发现严力妻子及一儿一女已被杀害多日并吊在客厅水晶灯上。杨间由尸体腐烂时间与严力复苏时间推断，凶手先灭其家人，再逼严力动用厉鬼直至死亡，动机可能涉及黄金盒、仇杀或针对自己的布局。面对尸体，他意识到驭鬼者冲突会波及亲属，决定不能坐等危险降临，随即开车前往小强俱乐部调查。（第144章）

## 第145章·会议安排

小强俱乐部内，王小强向股东和成员说明严力死后鬼血已在市外扩张，并把贺胜、张一鸣、叶俊、欧阳天和严力等人的损失归咎于杨间，提议将其抹除。少数人担心无故招惹驭鬼者会遭报复，但多数仍投票支持，王小强决定十日内动手。杨间此时持枪来到庄园，以无头鬼影威慑保镖进入会议厅，正撞见投影中的本人照片和相关资料。会议众人见他上门后拔枪、争相报价杀人，王小强承诺每人一亿元；杨间则要求他们先把话说清，冲突在行动前一刻公开化。（第145章）
