# 第411-415章梗概

## 第411章·存疑的人

魂瓶左臂把郭凡往瓶口拖拽，杨间判断此前限制的是右手，瓶中对象会随复苏从手臂成长为完整厉鬼，至少可能达到A级；更危险的是，郭凡体内的鬼已被瓶中鬼压制。陈义徒手掰瓶却受黄金隔绝能力所阻，钟山也没有能安全入侵人体的手段。杨间为不让魂瓶和郭凡死后厉鬼失控，终于用无头鬼影进入郭凡身体，发现其腹中藏有嵌遗像的木质灵位，并压退鬼臂救下郭凡。解除压制后，郭凡破碎的脸由灵位遗像替换恢复，遗像中的男子还转头对杨间鬼眼微笑。杨间因此怀疑郭凡与遗像中人之一可能是鬼。（第411章）

## 第412章·带走

郭凡醒后称瓶中鬼试图把自己体内厉鬼转移进魂瓶，确认失去一鬼会造成复苏；杨间没有公开对其身份的怀疑。陈义兑现不再争案的承诺，把魂瓶交杨间，杨间立即盖瓶、用金箔包裹，暂时不打算动其中的鬼。陈义因郭凡引发意外离场，沈良虽希望魂瓶由总部处理，仍默认杨间先做调查。杨间随后带走万德路，说明他会决定其是否交给大京市审问，实则要追查魂瓶线索，为自己的身体恶化寻找可用灵异资源。万德路认清驭鬼者的权力，因害怕陈义而选择完全配合杨间；古董来源尚未交代。（第412章）

## 第413章·刘小雨

总部安排专车与接线员刘小雨接杨间、万德路去酒店，车内檀香用于遮尸臭。杨间立刻发现车顶暗藏黄金钢板，认定这是可一键封锁的移动收容装置，遂掐住刘小雨并以鬼影确认；刘小雨承认总部为防来京驭鬼者复苏而预设此类措施。二人又谈到总部曾入侵张伟电脑、掌握杨间私生活，杨间警告她应提前说明风险。刘小雨提起中山市灵异入侵导致三十余接线员死亡，自己因杨间获得单独通讯室而幸存，称应感谢他；杨间否认自己是救命恩人，并拒绝听她未说完的个人打算。（第413章）

## 第414章·你是个好人

杨间趁检查车顶天窗时已用鬼域把刘小雨、万德路移出车外，确认没有立即的收容威胁后才继续乘车。刘小雨安排其休息两日、接受七天培训、总部嘉奖和研究院参观；杨间嫌培训太久欲拒绝，她则以队长计划已启动劝其暂时服从。刘小雨虽直言杨间冷血、自私又变态，却因他在大昌市和中山市的行动、赵建国对其评价，认为他有资格成为队长且应活下去。车到平安大酒店后，她留下私人电话、提醒勿闹事；杨间带万德路入住，对总部全面监控仍保持戒备。培训是否参加、会面王小明何时实现仍未定。（第414章）

## 第415章·食鬼者张雷

万德路入住总统套房后试图讨好杨间，杨间却提醒平安大酒店冷清异常、工作人员似受训人员，推断来客多为驭鬼者，并告诫普通人可能被精神失常的驭鬼者伤害。敲门声响起时，他先从猫眼看见无人，开门后才遇到自称张雷的国际刑警。张雷档案代号“食鬼者”，仅驾驭一鬼却能吃鬼，因饿死鬼事件想向杨间请教；杨间没有立即答应，约其次日再谈。杨间推测张雷要么被厉鬼利用来吞鬼、要么频繁用能力已近复苏，因此决定保持距离。张雷离开后，杨间转而准备审问万德路，魂瓶来源依旧是当前关键。（第415章）
