# 第186-190章梗概

## 第186章·无形之帽

张丽琴在售楼部醒来后，杨间说明驭鬼者越接近复苏，人的情感和生理欲望越会被厉鬼压低，自己近期才感到某些失去的东西回来。他为前夜冲动道歉，张丽琴要求保密，又坦白自己已有家庭，拒绝杨间提出的雇佣与安全庇护。她办完购车合同、发票、保险和车牌代办手续后离开，约定数日后送车牌。江艳因独自住在闹鬼房屋一夜未眠，张伟则通宵打游戏；杨间没有解释与张丽琴的关系，转而要求张伟中午等张显贵到来，以调查其被镜中鬼跟随的问题。张丽琴离开后仍对后续上门交牌感到不安，是否接受杨间工作邀请未获改变。（第186章）

## 第187章·古宅

张显贵带杨间、张伟乘小区巡逻车巡视观江花园，前部别墅区平静，杨间的鬼眼也未发现异常。车辆进入仍停工的D区商品房工地后，杨间却对后方一片荒地里的民国青砖旧宅产生强烈不安。张显贵查得该地原是十余户小村，因原房主未找到而未彻底拆迁；四十多年前有一名老人住在宅中，病死后房屋便荒废。旧宅无论正侧都没有窗户，外墙爬满藤蔓，屋顶腐朽，杨间认为其结构异常且鬼眼躁动，决定不再靠近、先搜集资料。江艳拒绝随行，杨间则指定张伟必须到场，因为张伟可能是镜中鬼线索的源头。（第187章）

## 第188章·直播鬼宅

为确保住处和四成楼盘权益，杨间带张伟进入荒宅探查，并携鬼烛与鬼域作撤退准备。宅内一楼是堆有发霉农具的前堂和大堂，真正令鬼眼不安的是二楼。此时，直播者虎哥和同伴闯入，声称要直播村中鬼屋；张伟认出虎哥曾拍到自己野外如厕，追着对方打架，杨间趁机独自上楼。二楼有三道异常门：外覆铁皮的黄金门、无把手的厚铜门，以及锁被剪断的木门。木门后的房间狭小却有至少三米厚砖墙，杨间判断其并非居住空间，可能存放极危险或极重要之物。他尚未打开其余两门，旧宅与工人失踪的关联仍待查明。（第188章）

## 第189章·神秘的镜子

杨间从木门房的脚印、工地工具和钻孔推断，失踪工人为盗取古宅财物而来，搬开镜子并试图钻墙进入铜门房。房内有一面铜架锈蚀、镜面却崭新的民国更衣镜，旁边留有遮镜油布；镜子照不出杨间，且地面显示曾被移位。杨间点燃鬼烛后以手试探，手臂穿入镜内冰冷黑暗，随即被多只不同大小的手抓住拉扯；他及时抽回，却留下青黑手印，判断镜内有多只鬼。杨间不敢继续触动，猜测原主人把镜子和三道门留在显眼老宅，并非单纯封死危险，而是可能想让后人利用。他关门下楼，鬼镜和黄金、铜门后的秘密均未打开。（第189章）

## 第190章·社会我腿哥

杨间下楼时，张伟仍和虎哥在直播镜头前扭打，直播同伴不断煽动观众。杨间拉开两人后，虎哥拿起木棍要以一敌二；张伟立即退到后方，实际把冲突交给杨间。杨间以特制手枪一枪打碎木棍，木屑令虎哥受伤、耳鸣，虎哥见枪后跪地求饶。杨间没有继续动手，而让他向张伟道歉；面对仍在直播的手机，他公开说明自己是有持枪资格的大昌市国际刑警，开枪属于面对袭击的反击。冲突暂被压下，但虎哥同伴仍在现场直播，杨间也尚未返回二楼处理鬼镜、调查工人失踪或验证张伟与荒宅的关联。（第190章）
