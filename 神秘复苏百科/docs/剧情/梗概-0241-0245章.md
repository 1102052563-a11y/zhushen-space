# 第241-245章梗概

## 第241章·被包围的众人

杨间独自抵达医院二十四层的王小明办公室，取回钉住鬼婴的干枯手指；手指一离盒，盒中鬼婴立即挣扎，但黄金盒仍能将其隔绝。因迟迟听不见王小明一队上楼的脚步，他判断同伴多半受困，暂留顶层恢复体力并观察动静。与此同时，王小明等人在楼梯口遇到一只第二阶段鬼婴；张韩绕行未受袭，王小明据此暂判衍生鬼婴的触发条件未必叠加，第二阶段可能须接触才会杀人。众人尚未来得及利用这一发现，楼上、楼下及两部电梯同时传来僵硬脚步，数十个青黑人影从三面合围。队伍退入空病房封门、接床单预备翻窗逃离，鬼婴却使墙壁黑化并穿墙显形，首轮正面冲突开始。（第241章）

## 第242章·第一次袭击

王小明一行困在病房时，楼梯和电梯中的鬼婴已封死三向去路；王小明坚持真正源头鬼只有一只，衍生鬼婴并非完整厉鬼，要求众人撑过袭击，等待取手指的杨间回援。病房内的门锁、杂物和床单只能拖延，众人仍须动用自身厉鬼抵挡。外面脚步停在门外后，墙面被黑青阴霾同化，手电无法照亮；随后墙上凸出一张张青黑死人脸，数十只鬼婴无视实体阻隔穿入。此时王小明也承认这种改造墙壁的表现不似单纯鬼域，行动可能走向惨败。第242章停在驭鬼者被迫合力拦截的瞬间，未写明首轮死伤与抵挡结果。（第242章）

## 第243章·压迫的烛光

顶层久候无声后，杨间决定携干枯手指下楼支援。他循体内鬼眼感应与电梯口汇集的黑青脚印找到受困楼层，明知前方黑暗内有鬼，仍点燃鬼烛进入。烛光起初可照五六米，越接近病房越被阴霾压缩至两米左右，燃烧速度也骤增；他估计照此速度最多十分钟便会烧尽。途中一条带金表的断臂伸入烛光，杨间尝试救援却只拉下被齿痕咬断的手臂，原主人已不见踪影。继续深入后，他见到血迹、残肉及潜伏阴霾的人形，终于同王小明会合。王小明据此前袭击确认第三阶段鬼婴会杀死听见其声音者，且各阶段条件不叠加；同时他称队伍已几近全灭，疑似第四阶段或源头鬼就在附近，必须趁鬼烛尚存寻找并钉住它。（第243章）

## 第244章·伤亡惨重

杨间与王小明退入病房厕所，幸存者只剩赵开明、张韩和重伤的孙义。张韩背后渗血，血淋淋尸体持续试图脱离却受限制；孙义下半身被咬去，体内驾驭的鬼亦被剥离，临时包扎无法阻止其失血。赵开明称更多同行已在第一波袭击中直接死亡，遗留厉鬼若落入鬼婴手中仍可能扩大风险，并趁绝境试图向张韩提出合作。杨间持鬼烛进入后，发现赵开明身后有一片烛光始终无法照亮的阴影，既有的隐秘厉鬼怀疑更深。孙义在杨间询问遗愿时只念及未还房贷，旋即死亡。面对人手和烛身皆将耗尽的困境，杨间决定主动令体内厉鬼复苏，目标是以干枯手指钉住源头鬼；他警告赵开明不得干预，王小明则要求尽快行动。（第244章）

## 第245章·解除压制

医院外的黑青阴霾未散，通往大昌市的道路已在五里内划为军事禁区；总部在“解决事件”与“隔离全城、防止扩散”之间同步准备。赵建国告知李军，参与医院行动的驭鬼者已大多死亡，事件被定义为S级，现有刑警无法突入；两人虽希望保住城市，仍只能将最后机会寄托在杨间，并为保全王小明预留后手。医院内，王小明向杨间重申一至三阶段鬼婴的条件各自触发，源头鬼则具有条件叠加特性；他判断鬼烛熄灭后，源头鬼会首先袭击杨间。为保留最后一截鬼烛供逃离，王小明亲自吹灭烛火，命张韩协助杨间。杨间随即准备收回压制鬼影的鬼眼、再张开鬼域，让张韩全力压住失控鬼影，以复苏的风险换取钉住源头鬼的一次机会；行动结果尚未揭示。（第245章）
