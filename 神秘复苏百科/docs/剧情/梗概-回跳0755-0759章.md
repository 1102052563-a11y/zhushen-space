# 原文回跳第755-759章剧情梗概

## 原文回跳第755章·死亡的敲击声

罗素一名字对应的老坟被挖开，露出其死人脸。孙瑞发现其眼眶、嘴巴和身体内部都塞满坟土，坟土已压制体内厉鬼，使他失去维持生命的力量；罗素一确认死亡。林落梅试图继续挖掘，被杨间以鬼手强行制止。杨间命人重新填坟，放弃救援。刻碑声随即再度停止，意味着又一个名字已被刻好，但队伍暂时无人被埋。杨间不等下一次死亡结果，继续朝78号方向行动。（原文回跳第755章）

## 原文回跳第756章·刘老太的出现

刻碑声近在身后，杨间一度考虑折返压制鬼，却因已接近78号墓而选择先送信。第二次刻名完成后，一名论坛成员被泥土拖走；杨间尝试拆下其手臂，确认刻名坟允许“缺斤少两”地埋人，却救不回目标。众人找遍77、79等相邻编号仍无78号坟。刘浩和刘欣悦却看见刘老太穿黑殓服、双手沾满坟土，僵立在老坟间，且手中握有一根锈钉；她随后向刻碑声方向消失。（原文回跳第756章、原文回跳第757章）

## 原文回跳第757章·冲我来的

杨间重新分析红信任务，判断普通信使不可能在坟场中顶着连环刻名找到78号墓，因此目标应仍是原福寿园的刘老太，而非坟场编号。他准备暂时撤离时，刘欣悦提供刘老太出现方向，正好指向刻碑声。杨间决定再赌一次，独自追击声音，约定若第五次刻名结束前仍找不到鬼就撤退。与此同时，刻碑鬼正在以棺材钉在墓碑上缓慢刻写“杨间”，大汉市酒店内绑定杨间的鬼瓷同步开裂。（原文回跳第757章）

## 原文回跳第758章·园中的漏网之鱼

杨间赶到声源，看见刘老太以疑似棺材钉敲击空白墓碑，确认她是刻名行为的直接载体。刻名完成后，鬼瓷在大汉市炸裂，杨间脚下坟土塌陷，鬼眼、鬼影被压制沉睡，身体迅速被埋。替死布娃娃破土逃跑后被坟场吞没，替杨间承受埋葬诅咒，杨间才从坟中爬出。他意识到已刻名的坟以布娃娃为墓主，自己成为坟场“漏网之鱼”，暂未再被直接拖入。他又判断刘老太手中钉子可能对应张雷记忆中的坟场棺材钉线索。（原文回跳第758章）

## 原文回跳第759章·诅咒的归属

明珠大厦外，叶真因看动漫暂不亲自处理福寿园，论坛将事件建档为A级。坟场内，杨间认为刻名诅咒不一定来自棺材钉，更可能来自能容纳姓名的空白墓碑；刘老太可能只是被坟场操控的刻字工具。杨间决定回到有空白碑的区域，以刘老太本名“刘兰”反向刻碑，尝试用坟场规则把她埋葬并取得钉子。孙瑞虽焦虑，仍选择服从；红信尚未送达，坟场鬼域也未解除。（原文回跳第759章）
