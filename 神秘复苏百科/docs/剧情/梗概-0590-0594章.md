# 第590-594章剧情梗概

## 第590章·笼罩众人

杨间离开总部宿舍后并未立刻离城，而是潜入地下灵异物品库。此前他因鬼画救援放弃队长职位，认定预先看中的裹尸布本该归自己，便砸开不能隔绝自身力量的墙体，取走印有尸体轮廓的脏旧裹尸布，准备在八音盒诅咒失效时处理后事或限制厉鬼。离开地下后，秦老以未知方式阻隔其鬼域；秦老没有索回物品，只说自己曾开公交撞死一个姓杨的人。杨间联想到早年车祸身亡的父亲却不信任对方，双方未动手。杨间随后到平安大厦，以五层鬼域红光突袭正在议论撤离的朋友圈成员，会议室众人和顶层三层建筑开始被消磨、送走。（第590章）

## 第591章·消失的朋友圈

红光持续消磨会议室，许多驭鬼者连同体内鬼一起消失，平安大厦顶端三层也被送走。姜尚白靠从饿死鬼身上取得的鬼寿衣挡下五层鬼域，方世明则以未知方式让红光在身边消失；吴云借墓碑诅咒流出的血字幸存。刘栋只留下持续腐烂、眼珠转动的人头，显示其鬼未被送走。姜尚白认出袭击来自杨间鬼眼，却仍把它当作鬼眼复苏；方世明虽怀疑可能是刻意报复，又因杨间已被鬼剪刀杀死而难作定论。三人决定先转往安全屋，吴云负责以衣物携带刘栋人头，朋友圈核心成员却已在一次袭击中接近覆灭。（第591章）

## 第592章·第二波来袭

杨间在高楼观察第一次袭击的结果，确认姜尚白的鬼寿衣挡住五层鬼域，吴云只是侥幸，最令他在意的是毫发无损的方世明。他没有冒险测试可能影响八音盒诅咒的六层鬼域，转而决定让鬼手直接侵入现场。平安大厦内，方世明拒绝立刻逃离，称应先去安全屋确认，姜尚白因鬼眼压制无法开鬼域，只能走楼梯。三人尚未离开，青黑鬼手已经从脚踝、肩背、口鼻甚至体内伸出；方世明的鬼也被压制。第二波袭击证明红光并非偶发灵异，幸存者却仍未看见杨间本人。（第592章）

## 第593章·各自抵挡

鬼寿衣暂时隔开姜尚白身上的鬼手，却加速将他侵蚀成尸体。吴云无法扯掉从体内长出的鬼手，只能拿刘栋腐烂人头与自己对视；人头目光聚焦后逼退鬼手，但也使他脸部开始腐烂。真正被重点攻击的是方世明，鬼手在三层鬼域内压住其体内鬼，从口鼻、耳朵和腹内肆意撕扯，令他无法反抗。方世明这才把连续袭击归因于杨间仍有意识并在报复，却在发出名字前被折断颈骨死亡。姜尚白难以相信能挡住红光的方世明会如此死去，也未能确认尸体背后另有秘密。（第593章）

## 第594章·第二个方世明

杨间检查方世明尸体，发现其中没有厉鬼复苏，判断事情异常，仍继续优先清除吴云。姜尚白从两轮袭击推断杨间未死，试图用队长身份、朋友圈条件和鬼剪刀谈判；杨间不回应，直接以鬼手掐死吴云，并压住其墓碑诅咒。刘栋腐烂人头滚落后，姜尚白想抢作保命物，也被鬼域中的手掌先一步取走。只剩姜尚白时，无头鬼影已成为下一轮威胁。与此同时，平安大厦安全屋中走出第二个方世明：他是枯瘦尸体般的本体，称先前死亡者只是没有鬼能力的“照片里的我”，并以阴冷尸臭风侵入杨间第二层鬼域。其失去最近记忆，准备再次杀杨间，双方冲突仍未结束。（第594章）
