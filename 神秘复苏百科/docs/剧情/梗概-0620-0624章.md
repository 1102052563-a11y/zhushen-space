# 第620-624章剧情梗概

## 第620章·又一幅画

杨间为赶在八音盒诅咒期限前寻找鬼画源头，低调排查自己曾以鬼域覆盖过、却没有负责人封锁的异常地点。他抵达一处老旧住宅小区，鬼眼不敢看向四层401室，确认那里有另一幅鬼画，但小区与上下住户尚未受鬼域影响。室内油画画着红衣、五官模糊的女子，背景正是小区；画中还有一具恐惧蜷缩的住户尸体。杨间据自己站在画前未受袭的事实推测，看见画或许只是杀人条件之一，决定以画布遮住画面并搬走，作为进入鬼画、寻找源头的突破口。下楼时，曾在航班获救的苏晴向他道谢，杨间冷淡回应；苏晴随后因意识到杨间带走异常物而怀疑小区闹鬼，急忙与男友离开。（第620章）

## 第621章·王野

杨间搬着鬼画离开时，鬼眼和鬼域均被画压制，只能步行前往贺天雄原有的八号别墅。王野安排房车主动接近，自称来自亚洲以外某驭鬼者总部，想在杨间与总部、朋友圈冲突后谈合作，并拿欧洲鬼画绝密资料换取一件A级灵异事件。杨间认为资料可能很快贬值，且王野可能一份多卖，拒绝交易并以鬼影威胁其闭嘴。抵达别墅后，王野改把档案免费当作赔礼，杨间却认定免费物不可信，当面烧掉。王野事后承认档案副本有错误、自己轻率低估杨间；随即两只发青手掌缠住他的脖颈，将其扭断。王野死于车内，手掌具体来源与后续影响未明。（第621章）

## 第622章·画中的城市

杨间回到八号别墅，把鬼画放入可隔绝鬼域的安全屋般房间，试图以观看和触碰触发进入条件，却未成功。他将其与鬼镜作比，判断鬼画同样存在未明杀人规则，原计划另找一幅已形成鬼域的画。正要离开时，油画消失，只剩能望见昏暗都市的画框，画中红衣鬼却未现身。杨间据此推测，现实中移动画作会令画内背景暂未同步，鬼会因不识路而无法立即从出口侵入；一旦停留足够久，鬼画事件仍会爆发。他看见画内城市与大京市相似、拼合进度已过半，担心数日后会形成整座城市背景，最终留下一枚黄金子弹作出口记号，跨入画框并沿新出现的扭曲道路深入死寂都市。（第622章）

## 第623章·苟活的人

杨间进入鬼画城市后发现街道、汽车和店铺触感近似现实，随后追踪一名逃跑的金发女子，发现地上的矿泉水真实存在。他据此认为鬼画内既依赖灵异又能供普通人生存。城市背景很快由大京市拼接到国外区域，他在一栋大楼中循荧光箭头找到幸存者。亚洲男子李阳说明，自己与女友旅游时被卷入鬼画，众人已在此苟活很久；原有三十多人，两日前被鬼找到后只剩十余人。李阳得知杨间知道出口，急忙叫来同伴留住他。杨间确认他们只是长期避鬼的普通人，且情报有限，拒绝立即带人离开。幸存者的求生冲动由此把初次接触推向紧张对峙。（第623章）

## 第624章·威胁

杨间通过李阳翻译得知，国外鬼画事件被卷入的幸存者已被鬼杀得所剩无几，并怀疑国外档案刻意隐瞒幸存者与死亡人数。白人大卫因恐惧失控，用枪顶住杨间，逼其立刻带人离开；杨间警告无效后扭断其手腕、将其踢昏，走火的子弹则被鬼影挡下。幸存者目睹其毫发无伤，反把杨间当作恶灵。杨间不追究其他人，只提出需要两名熟悉画中城市的向导，找到鬼后才可能让他们离开；他也指出继续躲藏只会等来下一轮团灭。李阳和皮肤黝黑的吉米最终追出大楼，自愿承担带路风险，为余下幸存者争取生路。杨间报出大昌市与“鬼眼”代号，行动尚未展开。（第624章）
