# 第166-170章梗概

## 第166章·请求支援

钱总、秦总仍困在杨间以鬼域制造的地下十层，张显贵趁其恐惧提出把两人楼盘连同资产以十亿元打包出售。秦总认为张显贵资金不足，钱总却在保镖被黑暗中的人形轮廓拖走后先行松口。此时刘小雨来电称赵开明已被卷入玫瑰酒吧的鬼敲门事件、下落不明，请杨间到胜利路支援。杨间以鬼敲门属无解级别、普通驭鬼者进入鬼域只会耗死，以及自己尚未正式任职为由拒绝；赵建国接手后接受其警告，却仍因市中心风险安排邻市刑警乘直升机临时增援，并批评刘小雨不应以办公室命令逼迫一线人员。钱、秦的合同尚未签字，赵开明与王岳的生死亦未明确。（第166章）

## 第167章·合同搞定

杨间得知赵开明陷入危机后并未改变立场，转而协助张显贵收尾观江花园交易。张显贵称钱总、秦总已经同意十亿元出售楼盘，拿出合同交给杨间；杨间进入仍被自己维持的地下十层，要求二人当场签字。秦总试图以五千万、两亿乃至四亿元收买杨间毁约，杨间始终以“张总出的价格你们出不起”拒绝，并以一分钟倒计时、黑暗脚步施压。钱总先签又漏签，秦总最终补齐，杨间才带众人离开。受困者不知异象人为制造，反而因突然回到小区门口而敬畏、感激杨间；交易已签，工地鬼物和张伟的镜中鬼问题仍没有调查。（第167章）

## 第168章·增援

钱总、秦总死里逃生后留下名片、电话和一张谢礼卡，杨间估计卡内至少有数百万元，却没有揭穿鬼域幻象的真相。张显贵确认转让合同到手，准备把部分楼盘转给杨间；杨间只要四成最终利润、数栋楼和别墅，并要求利润尽量兑换黄金。售楼部以五千万元从其利润扣除，张显贵答应清空人员物品，供杨间作为住处。张伟的跟随鬼则因天晚暂不处理，杨间要求他远离镜子和一切反光物、留在售楼部等候次日调查。杨间回市区搬黄金并查看玫瑰酒吧时，看到上空盘旋的直升机，判断赵建国已调来其他驭鬼者；他认为这些人面对敲门鬼仍凶多吉少。（第168章）

## 第169章·条件和答应

赵建国再次联络杨间，承认先前派入鬼敲门鬼域的两名刑警已经失联，市中心道路封锁，至少数百群众被卷入。杨间认为继续派人只会延长厉鬼在城内停留，却在思考时接到江艳求救：她开车经过胜利路、玫瑰酒吧附近后误入黑暗空城，只能不停行驶。为履行保护江艳的约定并维持后续事务，杨间答应尝试救人，但不保证救出多少人或刑警，条件是先获得大昌市国际刑警特权。赵建国请示上级后获准，杨间随即带上金盒中的鬼烛、打火机和特制手枪，通过警戒路障驶向黑暗。鬼域仍在意味着可能仍有幸存者；杨间进入前只准备救援，尚未承诺解决敲门鬼。（第169章）

## 第170章·再进鬼域

杨间进入大昌市中心鬼域，外界仍是夏日傍晚，域内却黑得如同空城；鬼眼张开后，红光令附近景物可见。另一边，江艳带女销售驾驶新宾利持续行驶，副驾驶门却在落锁状态下自行打开，车窗外人影紧随不舍，一只惨白僵硬的手伸进车厢抓向女销售。杨间到胜利路时见大量车辆堵路，破败的玫瑰酒吧招牌只剩闪烁的“鬼”字，二楼悬有一具叼雪茄的未具名尸体；他判断敲门鬼已不在酒吧，优先继续寻找江艳。驶至另一处，鬼眼忽被火锅店二楼的尖叫与灵异刺激牵引，他判断那里有鬼也可能有活人。本章止于其准备接近，江艳、女销售和鬼域内其他人的下落均未揭晓。（第170章）
