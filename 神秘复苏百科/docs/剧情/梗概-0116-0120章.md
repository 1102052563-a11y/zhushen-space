# 第116-120章梗概

## 第116章·交易正在进行

王小明接受杨间以吴越性命换取信任的条件，三次开枪杀死吴越；赵建国和李军均未阻止，众人各自以吴越死亡换取杨间加入、保护王小明或交代责任。杨间同意日后参加国际刑警考核，并开始出售鬼棺。他按鬼棺及其中厉鬼开价五亿元，另以搬运和拆车成本加五千万，王小明均立即同意。王小明只急于带走鬼棺，把其余交易交孙俪红，杨间则意识到冯全未必如自己预想般值钱。（第116章）

## 第117章·全都要

孙俪红因吴越之死畏惧杨间，面对装尸袋内的冯全不敢强硬议价。杨间展示冯全头颅，强调其既是两只鬼的载体也是失踪国际刑警，提出“全都要”作为价码。另一边，王小明认为鬼棺有改变灵异研究的价值，赵建国提醒其不应轻易打开，但王小明坚持危险研究值得。得知冯全仍活着后，王小明要求务必买回，孙俪红解释杨间不只要钱；赵建国判断其真正所求应是驭鬼者相关资源，王小明遂取来实验室金盒。（第117章）

## 第118章·鬼烛

金盒内是一根猩红蜡烛。王小明将其命名为鬼烛，称点燃后烛光范围内的人在火焰熄灭前不会被厉鬼杀死，已经三起大型事件检验；面对越恐怖的鬼，燃烧越快，但至少可维持半小时。杨间将其视为重要保命底牌，接受以冯全交换，王小明一行带走冯全和装尸袋。赵建国批评他贩卖同僚，杨间则反称这比背后被假档案陷害更仁慈。杨间暂不加入国际刑警，表面理由是等待公司方案成熟，实际隐瞒自己已通过人皮纸驾驭第二鬼。（第118章）

## 第119章·回市

封锁撤除后，张韩开车送杨间回市。张韩因背后伤势和复苏加重，准备尽快接受公司测试；杨间以成功率不足百分之二十为由延后，实则已有两鬼制衡。两人谈及鬼烛，杨间警告不得泄露，并以日后消息泄露便找张韩追责相威胁。张韩承认若早听杨间安排，黄岗村或许不会死这么多人；杨间仍称即便叶俊等生还也会找机会杀掉他们。抵达市区后，他联络江艳接人，独自去炸鸡店排队，无头鬼影始终随其脚下而行。（第119章）

## 第120章·被追求的江艳

江艳接到杨间回市电话后精心打扮，准备开车去接他，却遭大学同学钱枫在门口送花追求。江艳称已有男友拒绝，钱枫甚至租下其对门公寓继续纠缠。驾车途中，钱枫又在红灯处公开表白；江艳为摆脱他谎称怀有杨间的孩子，钱枫仍称愿意抚养。围观者起哄，一名出租车司机因看不下去而撞钱枫车辆并斥责他，江艳趁绿灯离开。她对杨间的依赖和钱枫的后续纠缠均未解决。（第120章）
