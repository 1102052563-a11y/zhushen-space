# 第560-564章剧情梗概

## 第560章·完整的限制

红衣画中鬼站在402寝室门口，没有继续行动。杨间不愿为多出一个压制名额而放弃对无头鬼影的绝对控制，因为鬼影仍可能夺取新身体、破坏三鬼平衡，继而让濒临复苏的鬼眼杀死自己；苗小善既已脱险，他也不肯为陌生人冒险。王泉带装画的大袋绕鬼而行，成功越过后却因回头确认而被鬼盯上。杨间等鬼靠近，抓住苍白手掌将其压制，红衣鬼化为纸灰；鬼域却未消失。画框随即再现红衣女子，杨间由此确认画是源头、画中鬼可被重生。待画与鬼同时存在，他再一次施加限制，昏暗与阴冷尽退，鬼域终于消散；单独限制任何一方都不能完成处置。（第560章）

## 第561章·猜测

鬼画恢复平常油画后，王泉和队员以多层袋子密封画框，另派人将滞留学生救出。杨间要求王泉复盘自己被盯上的动作，因破解鬼画规律比一次暂时限制更有价值。王泉先说自己回头看了鬼，杨间指出402女学生从未见鬼仍受袭，排除单纯回头或看见；王泉又猜测恐惧，或看见后产生恐惧，杨间仍以其他同样恐惧却幸存者否定其为完整答案，命其将所有细节报总部。总部催运鬼画后，杨间留下黄金工作手套套住鬼手，确认其能隔绝异常接触，也遮住腐黑、带尸臭的手臂。他把押送与扫尾交还王泉，离开宿舍楼。（第561章）

## 第562章·小聪明

杨间回到篮球馆，发现工作人员把所有接触过鬼画的学生集中于此，表面是方便管理，实际可能把未知风险聚集，并把苗小善置于其中以迫使自己继续处理。提出主意的姚队到场后，杨间摘下手套用鬼手将其扇倒，警告总部人员不得借自己与苗小善的关系耍手段；姚队虽想靠总部约束杨间，仍因鬼手后续造成的身体失调而恐惧。杨间随后安抚苗小善，说明鬼画暂被带走却未破解规律，不能认定危险已尽。他坚持送她回发生过事件的寝室，认为鬼已排查过的楼反而更安全，并决定让刘紫陪她。杨间离馆时，姚队拒绝上报挨打一事，只求送医。（第562章）

## 第563章·离去

杨间回到寝室等候刘紫、孙于佳，接到刘小雨电话，被通知次日早上八点赴总部参加鬼画事件会议。他对频繁开会不耐，仍因刘小雨判断事态不简单而答应。两名室友送到后，杨间说明留在苗小善身边才能获得自己关注，要求她们不要再把灵异警告视为玩笑；他也提醒苗小善，紧急联络不等于每次都能救到。杨间以鬼域迅速离开，苗小善追到走廊只见灯光闪烁。刘紫询问其身份，苗小善说起第七高中曾被厉鬼屠戮；孙于佳发现自己所见红衣鬼与尸斑老人不同，终于理解世上存在多种厉鬼。三人是否安稳过夜尚未写明。（第563章）

## 第564章·回来的记忆

深夜离校后，杨间因校园鬼画耽搁而放弃继续追杀贺天雄，转而约李瑶到僻静处。李瑶为自己遭贺天雄威胁、杨间又放跑目标而愤怒；杨间却宣布不再需要她，并以她从未真正看过自己的档案、却无端爱上陌生敌人为证，揭开其记忆被改写的事实。他问出“1+1等于几”这个预设暗号，李瑶立刻恢复记忆：她曾在平安大厦外被杨间以染血报纸取脸、篡改记忆，随后才误把自己对他的厌恶变作爱意并协助对付朋友圈。恢复真相令她尖叫崩溃；杨间将如何处理失去控制的内线及朋友圈风险，仍无答案。（第564章）
