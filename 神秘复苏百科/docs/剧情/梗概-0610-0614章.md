# 第610-614章梗概

## 第610章·卫景传讯

杨间在平安大厦内收起刚取得的柴刀，维持三层鬼域等待方世明是否会因远程斩击而报复。方世明始终没有现身，杨间一面逐渐相信目标已死，一面又因鬼剪刀可能落入未知驭鬼者手中而不能安心；八音盒诅咒和自身厉鬼平衡开始成为比追杀更紧迫的问题。凌晨五时，浓黑鬼域侵入红光，外貌近似鬼差的卫景到达顶楼，自称受王小明之命传讯，并交出方世明在地下室被劈成两半的照片。卫景约杨间当日十时见王小明，临走补充对方或有八音盒诅咒的想法。杨间虽对总部与卫景都保持警惕，仍决定为诅咒一事听一听条件。（第610章）

## 第611章·放鸽子

总部为杨间举行个人会议，曹延华、王小明、沈良、卫景、李军和刘小雨等待至十时半，杨间始终未现身。王小明判断他已把总部视为可能清除自身的鸿门宴，且坦承自己确实评估过提前除掉杨间、避免其死后演变为大型灵异事件的可能，曹延华遂改带人前往平安酒店。杨间则在酒店收拾死人头箱、绣花鞋、柴刀、寿衣、替死布娃娃及鬼烛等物，准备视谈判结果离开。黄子雅和张雷来问组队，黄子雅展示黑发能力带来的外貌变化，杨间却告知自己受八音盒诅咒、几日内可能死亡，只能把冯全、童倩、张雷等人的组队安排延后。双方一方防备、一方求稳，未赴会使旧有信任进一步破裂。（第611章）

## 第612章·抉择

曹延华一行只带少数人进入平安酒店，以免杨间误会；卫景提醒他们已身处杨间鬼域。杨间将王小明单独移至楼顶，先追问八音盒诅咒。王小明承认自己曾放弃研究，却提出可将诅咒与杨间意识剥离，并要求两天准备时间；他同时警告杨间，灵异公交车若压住诅咒和体内鬼，可能令其意识直接消失。谈判随即转向昨夜冲突：王小明拟把姜尚白责任归于死者、把杨间叙为有功者，并让总部接收朋友圈项目和资产。交换条件是杨间不正式辞职，改无限期休假、放弃队长职位而保留大昌市负责人及六个组队名额。杨间识破这仍会以城市和队员绑定自己，但为求生暂时同意，决定不把希望押在单一路径。（第612章）

## 第613章·改名

杨间拒绝同王小明握手，强调双方只是交易，并追问方世明遗体和鬼剪刀。王小明称曹洋正在收尸，若曹洋取得剪刀会尝试要求转交，却无法保证；杨间保留亲自取回的立场。王小明回到曹延华等人身边，指出杨间没有索要额外好处，反而说明双方只剩名义合作，总部必须让诅咒方案成功。人群离开时，杨间单独留下刘小雨，要求她秘密进入总部身份系统改掉自己的名字，且新名字连他也不要知道，以尝试规避可能以姓名为媒介的鬼剪刀诅咒；刘小雨答应保密。杨间随后向张雷、黄子雅、王江留下新地址，告知与他组队意味着脱离总部体系，最终带着物品迁往贺天雄原有的城郊八号别墅。（第613章）

## 第614章·错误的方案

八号别墅内，杨间把解决八音盒诅咒的路径排为鬼橱交易、人皮纸、灵异公交车，并认为重返凯撒大酒店追查香兰和八音盒秘密太过危险。他重新分析鬼橱所要照片女子，猜测其可能与鬼画或鬼新娘有关，却承认没有证据。为取得线索，他戴隔绝灵异的手套打开黄金盒，直接向人皮纸索要解法。纸上先显示二十五日零时将死于诅咒，又给出八音盒旋律乐谱与一段未来：未来的杨间会到凯撒大酒店某钢琴房，听见三种旋律并猜测合奏有效，却因不会弹琴、浪费时间而死亡。杨间由此识别该路线为错误方案。旧字消失后，纸面改写为自己“差点”死于诅咒，只说明未来已受当前选择影响，尚未给出确证生路。（第614章）
