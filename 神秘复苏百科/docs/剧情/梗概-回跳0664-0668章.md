# 原文回跳第664-668章剧情梗概

> 本文件对应原文在第663章后回跳出现的第664-668章，与此前已整理的正常第664-668章不是同一段剧情。

## 原文回跳第664章·平静下的凶险

二次行动前，杨间观察失去死人头的鬼童仍会躲避活人，只是活动更频繁，暂未失控。他判断敲门鬼至少有尸斑、鬼域、敲门声等多种复苏特征，正面压制并不稳妥。三岛按约送来重型箱体，杨间将其埋入地下，只留开口并在上方布置临时小屋。长泽带着敲门录音进入屋内充当唯一听者，杨间与鬼童远离等待。与此同时，一郎私下煽动田野，安排田野和山崎绕过三岛进入神户，意图在杨间成功时抢功、失败时借机问责。敲门鬼鬼域逐渐覆盖陷阱区，但天空所有人头已消失，杨间怀疑人头气球源头完成了未知变化。（原文回跳第664章）

## 原文回跳第665章·被断的手臂

长泽在小屋中承受录音诅咒，墙壁随鬼域靠近而腐烂破损，皮肉也不断剥落，只能勉强贴回身体。敲门鬼走到门口、抬手欲敲时，杨间从屋顶发动鬼域入侵。三层、四层鬼域都被尸斑灵异腐蚀，无法将老人从鬼域拉入现实；他被迫开启可承受上限的五层鬼域，才以红光限制老人鬼域外溢，使其从覆盖陷阱的鬼域道路坠回现实、掉进箱体。盖板却被老人举起的手臂卡住，杨间、长泽与巨石均压不严缝隙。尸斑开始从缝内侵蚀二人，杨间最终直接挥柴刀斩断手臂，箱体才闭合。（原文回跳第665章）

## 原文回跳第666章·打主意的人

敲门鬼被封箱后，杨间推测柴刀直接砍鬼同样能肢解厉鬼，但暂未显出代价。他命长泽通知三岛加固运走箱体，同时感觉附近有未知窥视。山崎、田野和一名皮包骨男子赶到，山崎先称赞杨间，再以除灵社名义要求接手关押物。杨间明确箱内厉鬼归自己，强硬威胁后，山崎评估杨间状态尚佳而退走。为不空手而归，三人到十字路口调查残余灵异，田野喊住一个背对人影；目标回头显出半腐烂头颅，田野瞬间被斩首。厉鬼目光转向山崎，山崎感到能力受压制，只能逃离。（原文回跳第666章）

## 原文回跳第667章·新的伤口

机场内，除灵社因敲门鬼被关押而短暂庆祝，三岛却得知一郎擅派山崎、田野进入市区，担心二人抢鬼或破坏与杨间的合作，亲自赶往神户。杨间在箱旁等待加固时发现右臂出现迟发腐烂伤口，位置与自己斩下敲门鬼右臂完全相同。他由此推断柴刀直接砍鬼会让使用者对应部位受伤，区别于远距媒介挥刀消耗压制名额的代价。伤口正深入骨头，杨间让无头鬼影融入身体尝试驱散。三岛一行尚未到场，关押箱和神户残余灵异仍需看守。（原文回跳第667章）

## 原文回跳第668章·混进人群的东西

三岛向杨间解释山崎未经授权，杨间要求追加二十吨黄金，并由除灵社安全把箱体送回大昌市，否则会放出敲门鬼。三岛立即答应，安排两小时内加固、拖运及专机。三岛随后发现田野无头尸体和山崎遗车，推测神户仍有高危残余事件。山崎正被半腐烂头颅厉鬼跨街追杀，发现对方会借不同男女老少的身体出现，目光能压制驭鬼者能力；他仍在逃亡。最后，一具穿西装、面容近似活人的冰冷尸体带着灰白眼睛混入城市幸存者，显示人头气球相关灵异已具备更隐蔽的伪装风险。（原文回跳第668章）
