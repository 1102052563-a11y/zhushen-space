# 第650-654章剧情梗概

## 第650章·最可怕的陷阱

王小明根据郭凡早先灵位失控又恢复的经历，推断郭凡可能早已被灵位中的鬼入侵；他担心杨间同样已变成鬼，却又不能放任身体里的厉鬼复苏。王小明决定赌一次，让卫景以黄金手套把染血灵位放回杨间身体。卫景刚取起灵位，外表如杨间的异常存在便抓住他；卫景也露出相同诡异笑容。王小明意识到卫景同样被灵位之鬼入侵，实验陷阱从意识交换扩大到两名顶尖异类。（第650章）

## 第651章·反制的手段

基地一楼出现尸水脚印，鬼童正追逐一只留下恶臭水渍的未知厉鬼；陈博士等人躲在二楼。高明在楼外看见第二只鬼后选择避开。地下五层中，被入侵的杨间和卫景走出实验房，王小明以安全屋试探，发现二者尚未完全使用原有规律，转而想到杨间留下的鬼童可能是破局关键。他取走灵位和杨间遗照，联系总部追查鬼童能听谁命令。此时杨间鬼域已侵入整楼，基地内至少有杨间、卫景、鬼童和尸水鬼四个危险存在。（第651章）

## 第652章·拦下一切

鬼域补足鬼童短板，它在五楼抓住试图翻窗逃离的尸水鬼。鬼寿衣挡住尸水侵蚀，鬼童将目标吞下，仅留下暗褐人皮。大门处，死人头持续压住被入侵的杨间；卫景则用步行方式触发出楼规则。鬼童暂时无法同时处理两名目标，最终让死人头继续盯住杨间、自己拦在卫景前。王小明赶到一楼，意识到鬼童正让两只顶尖异类无法顺利离开，却仍无法单靠它取得胜利。（第652章）

## 第653章·高下立判

总部通过刘小雨锁定王珊珊与鬼童的关联，王小明电话确认她可下令。与此同时，卫景跨出大门，鬼童两次攻击都被其鬼差压制，寿衣只能让鬼童在松手后复原，无法阻止卫景离开。王小明判断仅由王珊珊远程决策不足以应对，要求她将鬼童临时控制权交出。王珊珊最终同意，愿让鬼童听从王小明至次日九点。（第653章）

## 第654章·扭转的局势

王珊珊的转授权生效后，王小明命鬼童吞下无遗照的灵位，鬼童成长至约九、十岁并获得意识对调表现。王小明再命其把杨间遗照贴回杨间额头，杨间意识恢复，八音盒音乐消失。杨间发现鬼手变黑、鬼眼被压制，身体达到新的平衡；他命鬼童收走压住自己的死人头。得知卫景仍失控，杨间以三个条件为价接受救援，带鬼童追去。灵位之鬼和郭凡的结局仍未揭示。（第654章）
