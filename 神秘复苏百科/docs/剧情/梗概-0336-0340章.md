# 第336-340章梗概

## 第336章·微笑的尸体

白色鬼烛点燃后，餐厅外先出现被吸引来的高大黑影，餐桌旁又出现持报男子；杨间确认后者像控制赵磊记忆的目标，却未及出手，二三十张餐椅上又同时出现持报尸体。杨间吹灭白烛、释放受鬼域限制的鬼绳，草绳吊起全部尸体，报纸落下后露出带诡异笑容的死人脸。尸体被吊仍转眼盯住杨间并一起怪笑，笑容与童倩后脑鬼脸一致。杨间据此推断篡改记忆鬼已夺走童倩鬼脸，事件危险上升；他命熊文文发动预知，阴风随即卷起带人脸轮廓的血报纸，尸体的笑声持续扩散。（第336章）

## 第337章·恶化制止

笑声会像哭坟鬼哭声一样致命，杨间用三层鬼域争取时间，让熊文文预知真正带鬼脸的尸体；但熊文文看见鬼域将在三秒内被侵入。血报纸果然飞入红光、不断撕开缺口，杨间依预知连续尝试侵入多具尸体，均不能止住笑声。鬼域将尽时，熊文文又预见餐厅外被白烛引来的鬼已入内，二人继续硬撑会死亡。杨间收回鬼域，在血报纸、脚步和笑声同时逼近时点燃红色鬼烛；绿火阻住血报纸、隔绝笑声，也令高大黑影退出餐厅，危局暂止。（第337章）

## 第338章·二楼的鬼

杨间带熊文文准备撤离，却先以鬼烛保护自己回收鬼绳源头，避免复苏鬼绳在酒店扩散。黄金标记的草绳被收回后，吊尸落地，但尸体仍微笑怪笑，血报纸仍随阴风围绕。熊文文发现餐厅地毯密布黑脚印，杨间判断被白烛引来的二楼未知鬼原本一直在附近徘徊，幸而鬼域阻隔才未正面接触。两人持烛退回大厅，火焰燃速下降后熄烛；冯全、郭凡已不见踪影，杨间由停在四楼的电梯判断二人可能深入四层，暂不追随。（第338章）

## 第339章·被证实的身份

杨间在大厅重新制定计划，确认二楼有鬼、控制赵磊的鬼已得童倩鬼脸且能压制鬼域，决定不再轻用白烛。他以卫星定位手机搜到三个信号：冯全、郭凡在头顶附近，第三个则在大厅不足十米处。杨间让刘小雨拨打童倩手机，并以接线室播放儿歌作提示音，声音从始终微笑的女前台体内传出。杨间据女前台身材、手机与换脸鬼的规律，判断其可能是被换脸后的童倩；为核实，他让鬼影先侵入女前台身体接管行动，自己才靠近检查。（第339章）

## 第340章·被人脸覆盖

杨间从被鬼影控制的女前台腹中取出童倩卫星定位手机，发现手机温热，遂检查其脸部并撕下阴冷的人皮面具，露出童倩原脸。童倩仍有微弱心跳，却被夺走头顶鬼脸、头皮血肉模糊；杨间无法确认她未死原因，只推测可能不满足继续袭击条件或鬼只能取一张脸。他让熊文文通知张高，自己抱童倩出酒店交给救护车；医护发现她失血、内脏出血、多处骨折且心率低。杨间决定当天撤离酒店，等待童倩能否醒来并重拟方案，冯全、郭凡则仍留在酒店内。（第340章）
