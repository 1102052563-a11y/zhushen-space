# 第479-483章梗概

## 第479章·一缕红光

鬼差重启复制出的数百口鬼棺在黑暗中铺开，棺内鬼影开始迈步，王小明、李军、童倩、冯全等人已无脱身手段。李军鬼火将熄、童倩鬼脸受压，王小明也承认没有底牌，只能担忧他们死后谁来处置S级事件。就在鬼影靠近时，黑暗被一条迅速扩大的红色裂口撕开，杨间以第五层鬼眼鬼域从外部伸入带眼的惨白鬼手，抓住众人肩膀并以红光隔绝黑暗。他要求所有人不要反抗，随即把行动队转移出鬼域，临走又抓走一口鬼棺。鬼差失去袭击目标后，复制棺材和鬼影陆续消失，黑暗中仅留本体；其另一只脚仍未迈出棺外，复苏尚差一步。救援成功，但鬼差没有被关押。（第479章）

## 第480章·临时处理

钟山、姜尚白等外围人员等待杨间行动结果，原拟五分钟无动静便请求总部另派支援。红光过后，王小明、李军、冯全等人突然出现在空地，证实杨间救人成功，但行动队已有人员折损。杨间说明自己未完全进入鬼差鬼域，只让带鬼眼的鬼手侵入并抓人；鬼棺则被一并带出。李军因燃烧鬼火过度半边身体焦黑、虚弱倒地，王小明判断其可恢复。众人决定用金箔包封鬼棺后撤离，杨间却先以鬼域改变周围真实地貌，制造四面围合的山谷把黑暗留在其中；这只能令普通人难见难入，不能阻止鬼差出来。王小明承认带队失败源于无法让人按计划行动，杨间则完成临时遮掩。（第480章）

## 第481章·新的路

杨间以鬼域形成真实山谷震动众人。姜尚白认为鬼域只是虚幻空间，曹洋、冯全却从现场结果判断其已能影响现实；王小明解释可借鬼域与现实交替移走真实物体。杨间就“能否令整栋楼消失”与王小明约定赌一个条件，姜尚白另押一千万元，随后众人抬着金箔封住的鬼棺往培训基地转移。远离鬼差黑暗后，杨间以第五层鬼域演示：红光集中于额头鬼眼，路边树木和其后泥土一并消失，只留下延伸土沟。王小明认输，并提出若能逆转此结果或许可恢复物体、接近重启能力；杨间认为自己尚不能理解或做到。现场由此确认第五层鬼域能改变现实，消失物去向和逆转条件仍不明。（第481章）

## 第482章·熟悉的人

鬼棺封存、鬼差黑暗暂留山谷后，行动队在培训基地修整，后续封锁交由总部。杨间不愿再被卷入任务或会议，独自返回大京市；姜尚白在曹洋介绍下暴露自己属驾驭三鬼但实战较少的一类，曹洋又提醒其朋友圈早期排斥杨间，如今已因后者实力感到棘手。杨间走入普通人群却感到自己因接触灵异和体内厉鬼而与正常生活脱轨，苍白鬼手还使手机黑屏，令他警惕无意影响现实。准备回酒店时，他偶遇高考后到大京市读大学的苗小善。苗小善不信他只是打工，杨间仍不解释灵异事务，并因不愿影响其生活试图离开；苗小善却拉住他去买日用品，二人重新同行。（第482章）

## 第483章·逛街

杨间虽想与普通生活保持距离，仍因旧同学情分陪苗小善逛商场。苗小善试衣时，室友刘紫以杨间“打工、没大学文凭”为由劝其不要影响苗小善学业，实则轻视其身份与条件；杨间指出双方本不应有交集，也拒绝让对方替苗小善作决定。苗小善返回后，杨间称衣服好看，却再次以夜间安全和自己有事为由劝她回去。她听出其对灵异危险的暗示，坚持要联系方式；杨间最终留下卫星定位手机号码。苗小善拒绝与刘紫同走，要求杨间送她回学校并让其付账，还提到张伟直播曾说杨间家五楼有鬼。杨间未解释，只意识到自己存放异常物品的地点不再安全。二人有限恢复联络，灵异风险未向苗小善公开。（第483章）
