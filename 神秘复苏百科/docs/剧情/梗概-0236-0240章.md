# 第236-240章梗概

## 第236章·一家人

赵开明回到私有仓库，为十二具亲属遗体准备餐食、烧纸祭拜，透露妻女、父母等人均已死亡，自己正等待某项计划完成。他对疑似闯入仓库的鬼开枪泄愤，接线员随后通知医院灵异事件爆发。赵开明威胁接线员后赶往医院，仓库与其家庭死亡原因、复活计划及所关联厉鬼仍不明。（第236章）

## 第237章·医院的鬼

医院被黑青气息吞没，王小明召集剩余驭鬼者与杨间会合，携黄金收容箱再次进入。王小明承认无有效情报，仍执意同行以寻找第七街道和源头。八人进入后大厅空间被拉长、退路模糊；医院内尚有二三十名寄生病人，众人必须向王小明办公室前进。（第237章）

## 第238章·突然的死亡

队伍进入门诊通道，王小明通过步数确认空间被厉鬼改写。门诊室前的人影与灯光反复消失，一名驭鬼者踢开门试探，立即被黑青手臂掐断脖子、拖入黑暗。王小明仍要求前进，杨间判断此举也在以死亡试探规律；第二阶段或其他鬼婴已在医院内活动。（第238章）

## 第239章·熄灭的鬼烛

杨间点燃鬼烛带队，绿光照出通道两侧所谓门诊室实际全是闭眼静立的成长鬼婴。鬼烛随即被未知力量从近处吹灭，杨间怀疑跟随赵开明的高危厉鬼涉入；王小明趁短暂照明找到路线，众人转入安全通道。杨间决定离队，鬼烛在独处时恢复正常燃烧，进一步坐实未知鬼不在其身边。（第239章）

## 第240章·分开行动

王小明解释第七街道可能已把鬼群释放入医院，队伍继续上楼；杨间以赵开明的隐秘厉鬼风险为由单独前往顶楼取回干枯手指。杨间脱队后，原本沉睡在门诊通道的鬼婴群睁眼行动；王小明一队则在楼梯口被上下脚步和电梯内鬼影围住，退入病房用床单准备逃生，鬼群开始穿墙进入。（第240章）
