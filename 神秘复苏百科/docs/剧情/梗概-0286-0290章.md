# 第286-290章梗概

## 第286章·交个朋友吧

杨间在饿死鬼事件报告中继续把功劳推给王小明，赵建国、曹延华等人看出他刻意降低影响力，却不能当众点破。会后，国际刑警林龙以十亿元、一年黄金配额和日后援助为条件，要求收购杨间可能得到的红色鬼烛；杨间以鬼烛是保命资源为由拒绝。林龙随即握住其肩膀施压，以不做朋友就会成为敌人相威胁。杨间不再维持谦逊，额头鬼眼睁开，直接令林龙一条手臂脱落。林龙意识到杨间另有第二只鬼，准备反击，会议室内新老刑警的冲突正式爆发。（第286章）

## 第287章·会议室内的碰撞

林龙释放腐尸恶臭的厉鬼力量，试图以两鬼之力反制杨间。杨间召出无头鬼影侵入其身体，使林龙体内厉鬼沉寂、身体无法动弹，实力差距立刻显现。赵建国担心折损刑警而出声制止，其余人却多在观望。杨间张开红色鬼域，从体育馆取来黄金关押箱，操控林龙自行走向箱内。赵建国请李军救人，李军的青绿色鬼域侵入后，杨间以两只鬼眼叠出第二层鬼域将其压过。曹洋因此判断，杨间的鬼域和实力已与在场多数刑警拉开差距。（第287章）

## 第288章·熟悉的操作

李军无法突破杨间鬼域，王小明判断继续救援会把李军也拖入冲突。林龙被无头鬼影操控着进入黄金箱，杨间把人和箱子移出体育馆；他称林龙未死，只在地下约一万米处。赵建国无法派人挖掘，只能谈判。杨间开价一根鬼烛，赵建国提出增加黄金配额，王小明则承诺两个月内交付鬼烛。杨间接受赊账，却规定林龙不得再踏入大昌市，并要赵建国担保。王小明说明交易并非因林龙价值，而是总部需要杨间持鬼烛处理灵异事件，且若杨间遇险会优先救援。杨间在总部的实际地位由此被众人重新评估。（第288章）

## 第289章·坑妈的孩子

林龙在密闭黑暗中被困约半小时后，仍未出箱便被送离会场；会议继续进行。王小明客观说明饿死鬼事件并非单人可解，参与者据林龙与李军的败退重新确认杨间的实力。临别前，杨间要求王小明两个月内补足两根鬼烛和一次驾驭厉鬼的机会，并交出白骨树、黑影和棺材钉的照片，要求其评估潜在危险；王小明收下资料，表示会上报。曹延华完成会议总结。熊文文忽然试图把母亲介绍给杨间，杨间以其父亲态度调侃；赵建国说明熊文文父亲已死于灵异事件，这场插曲未改变会议散场的走向。（第289章）

## 第290章·散场

杨间拒绝熊文文的撮合，说明驭鬼者随时会死于灵异事件或厉鬼复苏，不适合成家。曹洋也参与玩笑，随后因负责城市出现异常而先行离开；各地刑警陆续返回辖区，林龙式的觊觎者因杨间展示的实力不敢再表露。王小明交给杨间一份身体修复计划，针对其尸斑、身体机能和结构提出处理办法，杨间收下并离开体育馆。曹延华担忧杨间性格偏激，赵建国则认为危急局势下应优先培养能解决S级事件的人。王小明携李军返回大京市继续研究。杨间驾车回观江小区时被同学赵磊拦下，赵磊语无伦次说出方镜、人皮纸和白骨树等信息，令杨间察觉他知晓不应知晓的事。（第290章）
