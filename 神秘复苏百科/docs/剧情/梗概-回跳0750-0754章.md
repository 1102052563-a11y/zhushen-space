# 原文回跳第750-754章剧情梗概

## 原文回跳第750章·消失的尸体

罗素一和林落梅到福寿园后，将灵堂、花圈和尸体视作灵异风险，强令手下驱赶葬礼宾客、抬走刘老太棺材。杨间为保留红信收信目标，开枪阻止罗素一。棺材意外落地、棺盖打开后，杨间先前确认的刘老太尸体已经消失。墓园暮色加深，尸体去向无法确定：可能被人搬走、自身异变，或被墓园内未知厉鬼带走。原本的送信等待转为即时灵异危机。（原文回跳第750章）

## 原文回跳第751章·诡变的墓园

杨间用三层鬼域搜索福寿园，地面未见尸体或厉鬼，却发现部分坟墓地下阻挡鬼域。收回红光后，现代墓园已被诡异坟场覆盖，城市消失、柏油路变泥路、旧坟不断增加。杨间推测红信可能是让鬼从另一鬼域渗透的坐标或诱饵，但尚不能断言信使在替鬼做事。罗素一向叶真求援的念头被杨间压下，众人决定继续寻找78号坟并送信。刘元请求带走子女，罗素一同意让刘欣悦、刘浩随队。（原文回跳第751章）

## 原文回跳第752章·拦路的坟

五名驭鬼者、论坛成员和刘家姐弟深入坟场。坟碑数字以旧式计数逐渐接近78号，路上却出现裂开的空坟、新近埋入普通宾客的坟与露在土外的死人手臂，说明鬼已开始杀人。刻碑般的敲击声从前方传来，杨间暂不撤退，判断声音可能是鬼活动预兆，决定接近查明。坟场看似无边，任何一座坟都无法被鬼眼透视，危险来源始终隐藏在地下或另一层空间。（原文回跳第752章）

## 原文回跳第753章·埋下一人

杨间冲向刻碑声源，赶到时声音停止，只看见新碑以血迹刻下“罗素一”。罗素一随即踩空、被泥土中的死人手抓住，体内厉鬼似被抽离；林落梅鬼音无法影响泥土，杨间鬼手也未能接触目标。罗素一被拖入未知空间，只剩与其名字相连的老坟。众人由此确认刻碑鬼会先刻名、再把目标埋走，但其获知姓名方式和能否阻止仍不明。（原文回跳第753章）

## 原文回跳第754章·挖坟

杨间分析刻碑声是锁定预兆，认为罗素一或许未死、可能埋在刻名老坟中。孙瑞担心挖坟会放出更危险的鬼，李阳则提出应验证能否救回被埋的驭鬼者。杨间最终决定挖坟，以风险换取救援可能和对规则的确认。他枪杀一名在紧要关头喊出同伴名字的论坛成员，警告众人不要让刻碑鬼从交谈中获取目标。论坛成员与刘家姐弟被迫挖土，刻碑声又一次响起；刘欣悦从坟土中挖出一张死人脸，本批在此结束。（原文回跳第754章）
