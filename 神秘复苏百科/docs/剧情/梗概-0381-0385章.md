# 第381-385章梗概

## 第381章·特殊的布娃娃

沈良无权即时报销杨间索要的两根鬼烛，改以特制手提箱内的布娃娃抵偿中山市行动损失。此物以脏旧布缝制、带尸臭，六根金针固定其四肢、腹部和头部；拔掉头针便会挣动，复钉后静止。沈良称只要滴血认主、拔针放走，袭击使用者的鬼会转攻布娃娃，娃娃未死前使用者绝对安全，且总部已试验过。杨间怀疑它由鬼制造，沈良以高度保密拒绝解释，仍称价值胜过两根鬼烛。交涉中，总部还希望借童倩前往大京市研究其复活；杨间认为不可复制而反对，童倩却愿配合。杨间最终收下布娃娃，暂认行动损失得到报销，但其来历、真实效果和王小明欠烛均未解决。（第381章）

## 第382章·郭凡的出现

杨间离开中山市回大昌市后，沈良没有破坏其封锁凯撒大酒店的安排，只准备从外墙救出仍有定位信号的郭凡。他命人定点爆破一层墙体、架设云梯直达四楼，并派年轻驭鬼者钟山入内。钟山反锁通往深处的房门，循手机信号在卧房见到郭凡的遗照嵌在木质灵位上，而郭凡身体僵跪地面；郭凡借遗照发声，称自己多次遭鬼盯上，只能用旧方法躲避。其起身取走灵位时，遗照模糊为空白。正要撤离，走道传来沉重脚步，钟山确认不可能是活人，郭凡则表示自己已多次听见。二人没有深入探查，选择沿云梯撤退；脚步声所属鬼及灵位能力尚未说明。（第382章）

## 第383章·个人灵异档案

钟山与郭凡在一墙之隔的脚步声压迫下撤离四楼，反锁房门未被打开，二人顺利回到酒店外。沈良随即命张高封死爆破口、堵塞可能的门窗和下水道，并将警戒线扩大至一千米；三天后，他带郭凡、童倩等人赴大京市，中山市人头气球事件暂以封锁收束。杨间回到观江小区，确认黄金箱、鬼镜、鬼骷髅和钟塔鬼橱未出现异常，便把凯撒大酒店持刀高大尸体定为“砍人鬼”、标为个人危险等级A，录入离线电脑。他因记忆可能被灵异改变或丢失，开始建立个人灵异档案。见小区保安、监控和巡逻已由张显贵加强后，他进一步计划以大昌市和自身安全利益为核心建立合作团体。（第383章）

## 第384章·身价

杨间回家后从江艳、张丽琴处得知，大昌市把尚通大厦转让给他作为奖励。张丽琴愿代办手续，杨间却不急签字，认为驭鬼者随时可能死亡，资产应留给更可能活下来的人，避免亲友失去依靠或招来麻烦。他改定次日上午九点在尚通大厦召集张伟、张显贵、王珊珊一家和张韩处理事务。张丽琴还希望亲属入住观江小区；杨间说明小区不再卖普通房产，若售则每平方米一百万元，所卖的是自己鬼域能迅速覆盖小区、必要时转移住户的安全服务，而不是房屋本身。二人关于赠房、工资扣款和救援价值的调侃，没有改变张丽琴继续依赖杨间、参与其日常起居的状态。（第384章）

## 第385章·残留的诅咒

清冷的大昌市中，杨间带江艳、张丽琴到尚通大厦。两人称旧租户以未到期合同为由索求高额补偿，还曾言行冒犯；杨间决定先依正规手续清退，不轻易动用负责人权力。王彬、王海燕和王珊珊已在大厦等候，王彬因女儿变化求助。杨间观察到王珊珊面色苍白、情感冷淡，鬼影入侵时又在其腹内感到已成形的鬼婴，确认饿死鬼被限制后本应消散的遗留诅咒仍在。王彬夫妇早知腹部异常，只因其成长过快才确认问题；杨间凭曾从张丽琴体内取出鬼婴的经验，判断可以处理，带众人上楼。鬼婴为何残留、最终如何处置及尚通大厦会议后续均未写。（第385章）
