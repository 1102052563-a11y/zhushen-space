# 第630-634章剧情梗概

## 第630章·呈现真实的画

杨间在四楼确认楼梯鬼尚在附近徘徊，鬼眼被压制而无法直接看见对方，只能根据自己刚被杀死一次、又由八音盒诅咒恢复意识的经历，推测脚步声会引来楼梯鬼，距离过近便会致死。他警告留在三楼的李阳和吉米不可踩楼梯离开，独自以无头鬼影搜索四楼。地上金色弹壳令他怀疑此前覆灭的国外驭鬼者来过此处。杨间推开一扇门，先发现一幅会显出手掌、人脸和黑瞳轮廓的黑画，随后见到房内墙壁与天花板挂满昏暗画作。他从楼梯画、客厅画与实际场景的对应关系判断，这些画会呈现画外真实情况及看不见的鬼；其中一幅村庄画还显示被困在鬼画内的鬼差。最后，他在七层楼画中看到红衣女子正站在大楼一楼门口。（第630章）

## 第631章·危险的楼

杨间继续观察七层楼画，发现一楼门口的红衣鬼转身进入大楼，认定鬼已向上靠近，而楼内除红衣鬼与楼梯鬼外还可能藏有未知厉鬼。画室内的画又从深处逐渐消失，杨间认为鬼画鬼域开始改写环境、阻止自己接近画中源头；即使鬼画可能就在十米内，他仍因鬼眼、鬼域受压和八音盒诅咒期限的风险放弃强搜。离开画室后，门已变成墙壁，他与楼下的李阳约定一口气冲楼撤离。杨间先从四楼冲至三楼，再等待李阳、吉米跟上，避免二人落后被脚步鬼追死。三人接着赶向二楼，身后空无一人的木梯却传来愈发急促的嘎吱声；杨间判断被追上便是必死，仍只能向唯一出口推进。（第631章）

## 第632章·楼梯上的相遇

杨间抵达二楼通往一楼的木梯时，发现穿红衣、五官模糊而双手苍白的鬼正从下方无声上行，令出口被彻底堵住；后方则是会致死的楼梯脚步。李阳与吉米赶到后因前后夹击几近崩溃，杨间认为停下必死，决定依靠八音盒诅咒冒险冲过红衣鬼。他以鬼影覆盖自身、再用鬼手尝试推开对方，却发现鬼手的一个压制名额和鬼影力量都无法撼动其身体，只得翻越鬼身、踏下最后台阶脱离木梯。李阳、吉米却因恐惧不敢效仿。考虑到李阳已成驭鬼者、吉米也曾提供重要地点，杨间没有独自离去，而以无头鬼影覆盖二人、将其抬起带回。楼梯脚步和红衣鬼即将在狭窄楼道相撞，二人的生死仍未确定。（第632章）

## 第633章·脚步的阻拦

鬼影强行带李阳从红衣鬼侧面挤过，又把吉米从鬼头顶越过；李阳接触红衣鬼时只觉像碰到残缺的旧画，而非实在尸体。就在脚步鬼距二人约三个台阶时，红衣鬼和脚步同时停止，似乎在木梯内互相阻拦，鬼影也被无形灵异干扰得扭曲失控，李阳和吉米摔下台阶但未致命。杨间率二人退至马路，见红衣鬼轮廓和双脚逐渐清晰，推测楼梯鬼可能成为其拼图之一。幸存者按约赶来后，鬼画又补入第四幅背景，使原本不长的道路被诡异拉远；红衣鬼在数百米外追来。杨间让其余幸存者继续逃，自己与李阳、吉米分列路旁以确认目标，并取出柴刀准备在鬼锁定自己时反击。（第633章）

## 第634章·离画

红衣鬼接近杨间、李阳和吉米时，脚下带着与其动作不一致的急促脚步声；杨间判断若两种灵异同时袭来，普通驭鬼者也难以承受。他铺开无头鬼影、握住锈迹柴刀，准备在鬼锁定自己时以脚印为媒介肢解对方。红衣鬼却在距杨间约五米处停下并转向吉米，杨间由此推断三人中有人暂时不再符合杀人规律，随即带李阳撤离。吉米留在原地闭眼祈祷、哭泣，临终前托李阳转告母亲自己很爱她，之后死亡。杨间发现原路已被城市建筑覆盖，却循预留的金色子弹进入小巷，在死胡同无玻璃的窗边找到第二枚标记；他率先跨窗，回到八号别墅的房间。李阳和幸存者随后陆续离画，杨间命他们立即远离仍被鬼域入侵的房间。（第634章）
