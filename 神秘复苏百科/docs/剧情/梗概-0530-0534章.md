# 第530-534章剧情梗概

## 第530章·正确的时间

黄岗村内，李军一行在先前险境后确认自己被困鬼域，并与外界完全失联。李军说明他们必须在总部约定的正确时刻打开正确的门，才可能借鬼门离开；首次机会已错过，十分钟后还有第二次，之后只余更遥远的第三次。鬼画对鬼差鬼域的侵蚀正打破原有平衡，遗像不断掉落，苏凡的手机又显示“您已死亡”，队伍只能继续深入村路寻找生机。徐一平发现掉落画像旁出现有五官的扭曲黑影，李军遂带人前进。外部的钟山同时面临白烛将尽，陈义必须冒险为郭凡之鬼更换烛火。内外两支队伍都在倒计时中行动，任何一次失误都可能令鬼失控或彻底失去撤离窗口。（第530章）

## 第531章·杨间的打算

杨间没有参与外围救援，独自在大京市街头判断鬼画与鬼差鬼域消失或许意味着事件已获控制，却仍意识到S级事件的危险在于厉鬼数量、无法逃出的鬼域和驭鬼者人数劣势。他回到平安酒店后，李瑶主动带来新情报：朋友圈已决定在队长计划结束前除掉他，因总部当前行动影响而暂缓，倾向在他处理下一起灵异事件时动手以掩盖谋杀。李瑶担心杨间先发制人会引来总部与多方势力反制；杨间则要求她搜集详细成员名单，认为自己体内三只鬼足以令常规权势失效，并警告若生命仍受人控制，求生毫无意义。李瑶受震动后答应继续卧底整理资料。（第531章）

## 第532章·转身的人

钟山持有的白色鬼烛接近燃尽，陈义为完成交接靠近郭凡之鬼。他明知两根鬼烛重叠可能造成更大失控，故先掐灭郭凡肩上的旧烛，准备立即点燃新烛再插回原处。五名异常目标并未远离，而是倒立般停在近处；更关键的是，打火机火焰在触及新烛时无故熄灭。失去烛光影响后，原本背对陈义的郭凡之鬼转身，以空洞眼睛望向他，阴冷气息侵入身体。钟山在前方无法回头协助，交接人员虽已候命，原本依赖两层牵引维持的队列却在数秒内失去核心。陈义能否重新点烛、郭凡之鬼是否已选定目标，均未揭示。（第532章）

## 第533章·徘徊的选择

郭凡之鬼抓住陈义手臂，令其身体失控、意识近乎冻结。陈义被迫动用体内“鬼皮”，皮下出现人脸和四肢轮廓，暂时恢复知觉，却付出皮肤迅速死灰、尸斑扩大的复苏代价，仍无法挣脱鬼手。五名异常目标此时转向钟山，郭凡之鬼也被白烛吸引，钟山为避免成为唯一诱饵吹灭残烛，双烛引鬼计划至此崩解。黄岗村里，李军队伍绕回村头，发现最后一栋未被鬼画侵蚀的砖瓦房，其中吊着鬼绳和尸体；离第二次开门只剩三分钟，他们只能赌房中还有可用的门。两条线都从暂时平衡转为直接与鬼对抗。（第533章）

## 第534章·放落的绳子

总部等待第二次鬼门开启时，曹延华追问王小明的后手。王小明承认自己以鬼给出的信息设置了撤退保障，并透露阿红是行动队中的第一道后手，却未说明完整安排。黄岗村内，李军带队进入最后的砖瓦房，发现吊尸、鬼绳和一扇受潮木门；时间临近，他要求众人先找门离开。柳三却突然攀上吊尸，称受王小明指派另有任务；阿红默契地取出并点燃计划外的红色鬼烛。鬼绳随即落地，尸体出现活动迹象，烛火急速燃烧，阿红半张脸显出陌生男性面孔。秘密任务究竟要取得什么、能否与第二次开门配合撤离，章节末未给答案。（第534章）
