# 原文回跳第730-734章剧情梗概

## 原文回跳第730章·错误的路

杨间以鬼眼第一层制造的幻觉将王晗单独隔开，既向聚会宾客展示灵异真实，也避免真正鬼域波及普通人。王晗称驭鬼者将因无法被普通手段消灭而成为未来主导者，且总部被入侵、鬼画未完和队长计划都说明官方已难压住顶尖驭鬼者。他又透露认识一名至少活过十年的老驭鬼者，邀请杨间接触一批疑似掌握更优驾驭方式的“异类”。杨间从父亲和民国线索判断其说法未必全假，却拒绝加入，认为人性被侵蚀后的长寿不是正确道路。（原文回跳第730章）

## 原文回跳第731章·解惑

杨间回到聚会后，鬼眼幻觉让更多富商接受灵异现实，十余人表示愿意投资。面对投资人提出的安全问题，杨间承诺在大昌市范围内提供救援，不承诺跨市或出国。马有财询问队长含义，杨间说明自己只守大昌市是因为能力有限，不会贪图扩大辖区。王晗在警告下提前离场，钱总留下。唐燕燕尝试通过应聘接近杨间，被交给张伟处理。杨间表面继续应酬，心中却开始反复思考王晗所称“错误的驾驭道路”。（原文回跳第731章）

## 原文回跳第732章·窥视和异动

聚会后，杨间、江艳和张丽琴回到观江小区。张丽琴担心自己被抛下，杨间说明她能提醒自己仍保有人性，却拒绝她的亲密请求。窗外鬼童窥视后离开，杨间发现王珊珊在路灯下等候。王珊珊称民国古宅内、从梅山村运回的木棺当天曾晃动，还命鬼童攻击棺中目标却无效。杨间意识到尸体或鬼童均可能出现异常，跟她前往查看。（原文回跳第732章）

## 原文回跳第733章·尸体的异变

杨间开棺，发现被鬼手压制腐烂的噩梦鬼尸体已成覆满黑毛的腐肉，黑毛爬满棺底和内壁，肉团会抽动、起伏、扭曲，鬼眼也看不穿原因。鬼童不攻击使目标性质更难判断。杨间为避免小区受波及，暂将整口木棺转入尚通大厦办公室安全屋，与熊文文专属纸人隔离，计划日后建专门收容灵异物的安全屋。回家后，他要求张丽琴手写记录梅山村“鬼梦”经过，江艳在旁睡去。（原文回跳第733章）

## 原文回跳第734章·孙瑞

数日后，大昌市公司投资充足，杨间因鬼橱交易仅剩七十五天，决定调查大汉市建设路869号坐标，带新人李阳同行，并让张韩暂看大昌市。章华转达熊文文母亲陈淑美要求救援，杨间让她到大昌市面谈，因纸人复活涉及异类状态和监护人同意，暂不执行。章华继续追查孙仁但无结果。另一边，大汉市负责人孙瑞得知杨间来访，担心这意味着辖区已有灵异问题，决定亲自接机确认目的。（原文回跳第734章）
