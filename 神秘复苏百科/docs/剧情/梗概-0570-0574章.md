# 第570-574章剧情梗概

## 第570章·画中背景

总部转而讨论鬼画事件。曹延华说明鬼画虽覆盖鬼差鬼域，却在郭凡、钟山、陈义引鬼收尾时走失两只分别模仿苏凡、柳三的鬼，市区也随即出现鬼画杀人。四张画都画有红衣模糊女子，却对应尚峰小区、森林公园、大京大学和天桥大道等不同背景。杨间说明普通鬼画必须先限制游荡鬼再关押画，否则鬼会再生。王小明据不同背景与画内尸体推断，多幅鬼画内部可能连通，进入其中或能寻找源头；背景若全部拼合，出口可能消失并令鬼画扩张。总部限三日确认源头，直接解决者可破格进入队长计划。杨间因鬼画压制鬼眼且线索杂乱，决定暂不冒险，转而提防朋友圈反扑。（第570章）

## 第571章·索要

会议结束后，曹延华调解杨间与姜尚白的冲突，希望双方以大局为重并由总部保证停止报复。杨间拒绝把险些丧命之事当作普通教训，最终提出唯一和解条件：朋友圈交出鬼剪刀。姜尚白看到照片后神色改变，只称需半日答复，曹延华以封杀产业施压。杨间又借放弃队长位置谈判，要求组队、举荐下任大昌市负责人、退出鬼画行动与资源。曹延华仅同意最多六个组队名额、可举荐负责人、正常任务有拒绝权，以及灵异物和三根鬼烛二选一；杨间要求写入合同。双方尚未签约，朋友圈也未交出剪刀，杨间仍不能退出鬼画事件。（第571章）

## 第572章·决定

杨间回到平安酒店，向张雷、黄子雅、王江、冯全和熊文文说明自己推掉队长名额却保留六人小队资格。众人虽惋惜仍准备日后组队，冯全接下联络童倩的地址，熊文文则致电征求母亲同意；杨间决定暂缓正式定队。另一边，姜尚白回平安大厦说明杨间索要鬼剪刀及总部施压。总裁方世明否定妥协，认定曹延华借杨间打压朋友圈，决定六点亲自杀死杨间，并禁止参会者泄露消息。姜尚白由此确认双方无法和解。方世明认为杨间死后即可成为最直接的答复，具体袭击手段未在本章揭露。（第572章）

## 第573章·见家长

熊文文带杨间回家见母亲陈淑美。杨间从档案得知熊文文父亲死于灵异事故，熊文文此后有短时预知能力但使用会令厉鬼复苏；他认为这项能力适合队伍。陈淑美礼貌招待杨间，却在得知他仅二十岁后更忧虑。她要求杨间保证熊文文安全，杨间坦言谁都无法保证存活，也不愿作虚假承诺；陈淑美因此拒绝儿子入队。杨间认为她不了解驭鬼者处境，仍未继续劝说。与此同时，方世明从档案室取出杨间全身照，准备在六点发动袭击。杨间对此不知情，组队与和解均未取得结果。（第573章）

## 第574章·突然的伤口

杨间为避免被临时拉入鬼画行动，在市区闲逛至傍晚，准备返回酒店。平安大厦内，方世明打开黄金盒取出鬼剪刀，在杨间全身照写下真实姓名，于六点剪向照片颈部。剪刀发动时出现腐臭、黑暗和残缺尸影等诅咒现象。杨间正与曹延华通话，先感到危险，随即颈部无征兆裂开。他紧急张开四层鬼域，以无头鬼影拼接伤口，刚取出鬼烛又遭第二次切割，头颅与身体分离。方世明见照片变红且杨间未立刻消失，认为第一、二剪仍不足以彻底杀死目标，决定再剪一次头部；结果待后续。（第574章）
