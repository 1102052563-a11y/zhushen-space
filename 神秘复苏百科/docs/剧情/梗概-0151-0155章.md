# 第151-155章梗概

## 第151章·通风报信

杨间离开小强俱乐部后，没有贸然前往玫瑰酒吧找王岳，而是准备等待俱乐部遭袭的消息扩散，再看王岳会站向哪边。他重新推敲严力一家遇害、王小强借机与自己冲突以及赵开明被卸腿后始终隐忍的先后关系，怀疑赵开明才是借刀杀人的布局者。马有财随即来电，称被杨间斩首的王小强已经复活，还强令其在三天内筹十亿元重建俱乐部、设法对付杨间。杨间禁止马有财出资，并让他试探王小强对赵开明的态度，由此确认严力并非死于俱乐部。张伟又发来自己被拘留的短信；杨间想到拘留所归刘队管辖，决定先去接人，并借此取得赵开明的联络方式。（第151章）

## 第152章·个个都是人才

张伟在拘留所内与其他被拘者互相讲述“无辜”经历，把自己误入色情场所、踢倒脱衣女子后被捕的经过仍解释成去“吃鸡”，等待杨间救援。杨间到警局时，刘建国正被各类案件与特殊事件压得难以喘息，却仍将刑警事务置于优先位置。他承认赵开明难以相处，仍交出其卫星定位手机号码；对于放出张伟则坚持先核实案情。警方调查后认定张伟无案底，疑为受引诱进入场所，办理手续即可离开。杨间表态会在避开赵开明的前提下继续处理灵异事件、积累官方档案功劳，刘建国则希望他能为大昌市出力。双方尚未去拘留所，警局内便传来警员出事的报告。（第152章）

## 第153章·反派死于话多

杨间拨通赵开明电话，直接以严力一家四口的死质问对方。他指出严力的鬼血本可成为刑警助力，杀害严力既让自己失去帮手，又能把怀疑引向小强俱乐部，使双方冲突后真正挑事者坐收结果。赵开明不再否认，承认自己杀害严力一家，意图就是借两方死斗清除杨间，并表示王小强所驾驭的特殊鬼足以令其保命。赵开明还打算在双方分胜负前躲藏，避免承受两面压力。杨间却让通话中响起鬼敲门声，称这是一封挑战书。赵开明查阅第七高中档案，得知听见声音者会被敲门鬼找上，才意识到自己因炫耀布局而被杨间以厉鬼反制；诅咒何时发作仍不得而知。（第153章）

## 第154章·出事的警员

刘建国查明张伟无重大案情、可以释放时，警员小朱报告大厅反光墙面里困着一名同事。杨间立即联想到KTV遭遇的镜中鬼，判断这是新的特殊事件，答应协助却不承诺冒险。刘建国随即疏散群众、隔离现场。被困者名为张元，二十七岁；监控显示其可能早上便已出事，但同事未察觉。杨间据此推断除镜内张元外，警局里还存在一个现实的“张元”，否则日常工作不可能无人发现其失踪。镜中的张元写下不少文字，却无法直接听见刘建国的呼喊；杨间认为单纯与他沟通治标不治本，必须先找出现实替身，也就是镜中鬼。至本章末，镜中鬼尚未现身，警局陷入搜查前的紧张状态。（第154章）

## 第155章·跟在身边

杨间说明镜中鬼的目标并非直接杀人，而是用活人身份替换：张元进入镜内，鬼便以张元身份留在现实，只有把鬼赶回镜中才可能救人。刘建国准备调动监控，杨间则离开嘈杂人群，在警局外留意出入口以避免不明情况下暴露。随后他把张伟家、张伟请客的饭店和KTV、以及张伟被拘留后的警局串联，终于推定镜中鬼只能出现在张伟附近，至于对谁下手则不受限制。鬼眼感应随即令他抬头看见四楼玻璃后的假张元：对方穿警服、低头缓行，映在玻璃上的脸惨白无血，正往五楼去。杨间通知刘建国停止大范围查监控，改从四楼封锁楼道、电梯并逐层排查；鬼、真张元与张伟的后续风险均留待下章。（第155章）
