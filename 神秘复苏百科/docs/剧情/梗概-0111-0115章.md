# 第111-115章梗概

## 第111章·厉鬼回棺

杨间把昏迷的张韩、冯全头颅和身体聚在一起，避免再次触发落单规则。村鬼这次以刘根荣老人形象进入灵堂，未攻击三人，而是走向空出的鬼棺并自行合棺；杨间只能推测此举或因三人体内厉鬼已超出其压制上限，或因回棺优先。观察近一小时后，他确认鬼未再出棺，判断黄岗村事件暂时结束。冯全仍认为杨间毁掉成为救世主的机会，杨间则坚持只求活下去。为避免冯全体内厉鬼复苏又取得行动资源，杨间将其身体装入黄金尸袋，未杀死他；张韩醒来后也被杨间以“出售”打趣威胁。（第111章）

## 第112章·出村

村鬼回棺后天色转亮，黄岗村的建筑、道路等景象却如海市蜃楼般逐渐消失。杨间认为这只是暂时消退，若鬼再次出棺，村子仍可能重现且更危险；他将原本B级的事件评价为规律不明时足以令驭鬼者必死、并会随首名驭鬼者死亡滚雪球增强的厉鬼。杨间把装有冯全的尸袋和鬼棺装车，与受伤的张韩离村。村外已被公司、国家单位、研究人员和军方人员戒严，负责人吴越原以为杨间难以生还，得知其带棺归来后只能出面应对。（第112章）

## 第113章·吴越

张韩背后鬼复苏加重，自认不动用力量也活不过十天，仍盼借交易延寿。两人抵达封锁线，吴越代表公司祝贺任务完成并提出庆祝，杨间则指出六名行动者死四人、张韩濒临复苏，质问其是否轻视驭鬼者性命。吴越称公司会提供第二鬼驾驭方案，但须采集信息、实验室匹配，约一周给出结果，成功率仅约百分之十五；杨间已靠人皮纸成功驾驭第二鬼，对其方案深感不信。吴越回避黄岗村档案问题，并以交易作筹码阻止杨间追责，张韩为保住延寿机会劝杨间克制。（第113章）

## 第114章·我不听

吴越试图以杨间家庭处境和一点五亿元溢价收购安抚他，并承认隐瞒信息是出于更大利益、六人全活概率不足百分之一、解决事件概率不足百分之十。杨间将交易报酬与删档责任区分，认为不完整档案使自己几近死亡，必须有人负责，否则未来仍会有人以假档案陷害他。吴越以必要牺牲为由辩解，杨间则拒绝继续听取，取枪抵住其头部，明确自己不必先找出幕后者，只需让足够高层的人为后果付出代价。吴越试图继续解释时，杨间扣下扳机。（第114章）

## 第115章·布鲁斯·皮

子弹穿过吴越头部却未造成伤害，白大褂研究员王小明与赵建国、李军等人出现，承认删档由自己指使、吴越只是执行者。王小明以国内灵异事件增长、鬼棺回收价值和国家利益解释牺牲，李军则以保护国宝级研究员警告杨间。赵建国从中调和，并告诉杨间王小明正是国际网站上发表黄金不受鬼力影响论文的“布鲁斯·皮”。杨间不接受口头大义，要求王小明依其价值逻辑亲手杀死吴越：若能做到，自己将加入国际刑警并不再追究删档；否则他将认定对方只是空谈。王小明尚未回应。（第115章）
