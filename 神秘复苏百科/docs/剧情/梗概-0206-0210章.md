# 第206-210章梗概

## 第206章·庇护

观江小区枪击现场清出二十三具行动人员尸体、境外身份材料、枪支和未引爆炸弹。杨间要求刘队只追查受雇枪手，驭鬼者层面的王小强、叶枫由自己处理，并提醒赵开明同样涉入借刀杀人。王小强回到已人去楼空的小强俱乐部，王小明电话要求他前往实验室避难，称上层已默许杨间行动；王小强因怨恨兄长曾把自己当实验素材而砸毁电话，仍留在原地恐惧杨间追杀。（第206章）

## 第207章·完美驾驭厉鬼的方法

杨间意识到叶枫寿衣与鬼牙难以正面克制，转而威胁人皮纸给出增强力量的办法。人皮纸提出“自杀”并解释目标是完美驾驭无头鬼影：需让鬼影完全融入身体、在鬼镜前用鬼绳上吊，使复苏鬼影与鬼镜争夺身体，再由镜中复活完成死机平衡。杨间承认此法可能失败为真正吊死，因风险过高暂时封回人皮纸，未付诸实施。（第207章）

## 第208章·任命

人皮纸进一步列出自杀法的三重必死环节，杨间仍决定不轻易依赖。次日赵建国通知孙义已到、任命可办，完成后全市力量将支持通缉王小强和叶枫。杨间抵达市政接待室，孙义以录音考核提问身份、婚恋与灵异处置假设；杨间的回答始终以驭鬼者生存和有限牺牲为优先。考核尚未结束，孙义身份与任命结果暂未完全落定。（第208章）

## 第209章·无意义的考核

孙义继续以五人中藏鬼、子弹有限的假设审问杨间。杨间拒绝为救少数普通人轻易牺牲刑警，主张用最小代价排除鬼的可能，理由是所有人都想救反会团灭。两小时问答后，孙义宣布杨间正式成为大昌市国际刑警，交付新的卫星定位手机、证件和制服，收回周正旧机；杨间得知赵开明也会参加随后会议，立即警惕。（第209章）

## 第210章·第二位刑警

孙义向杨间介绍大昌市各部门人员，赵开明作为同级刑警到场，双方在公开场合维持表面客气。杨间拒绝立刻另建特别行动小组，要求全市警力先追捕王小强、叶枫，并强调两人是驭鬼者，只能击毙。东城区负责人承诺三日内破案、最快当日下午行动。会后孙义与杨间吃饭，确认孙义也是驭鬼者但因不适合一线转做联络和考核；他称考核资料会影响未来安排，国际刑警制度或将改革。（第210章）
