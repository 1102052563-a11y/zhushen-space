# 第484-488章剧情梗概

## 第484章·改变

苗小善与室友分别后，由杨间陪她步行回大学。她谈及选择冷门历史系、上官云平安无事，又询问七中幸存同学近况；杨间如实说张磊已死、王珊珊近似活死人，孙仁和刘奇下落不明，张伟虽直播也曾死过一次。他因此劝苗小善远离自己，认为灵异会拖累普通人；苗小善却指出是他救过众人，并感到他半年间已变得陌生。二人都不愿这种疏离，却只能沉默前行。另一边，刘紫被男友李洋接走，提起杨间和平安酒店。李洋由酒店住客资格、杨间姓名和大昌市经历，推断此人很可能是鬼眼刑警，警告刘紫绝不可再招惹或深究。杨间与苗小善的关系未获解决，李洋所知内幕也未展开。（第484章）

## 第485章·第一驭鬼者？

杨间将苗小善送至校门，仍用读书和工作玩笑劝她好好生活，自己则决定保持距离，以免把她卷入危险。夜里回到平安酒店，张雷等候数小时，报告同批幸存驭鬼者张辉已收拾行李、神情异常地离店，且没有向总部说明。张雷怀疑张辉是否发现酒店不安全或遭遇灵异；杨间认为更可能是鬼差事件造成的精神创伤，但答应留意。次日刘小雨通知杨间十点赴总部开会，内容可能是鬼差事件善后；与会者包括部分队长计划候选、驭鬼者中的异类，以及被称为大京市第一驭鬼者的人。张雷、王江和黄子雅未获通知。杨间乘专车出发，既想了解总部秘密，也想借会议确认自己在顶尖驭鬼者中的位置。（第485章）

## 第486章·第一次总部会议

专车穿过多道岗哨，沿未标示道路驶入大京市郊山区的总部；普通手机信号被屏蔽，杨间的卫星定位手机验证“鬼眼”身份后才切入内网。刘小雨按专属接线员制度接待他，将其带入布置有食物、电脑和玩具的独立休息室等待会议。她提醒早期存活至今的老牌驭鬼者可能已难分人鬼，杨间因此更为警惕。休息室中，杨间翻阅重大灵异事件记录本，看到欧洲幽灵列车、岛国鬼井或鬼寺、已解决的饿死鬼和未知的鬼湖等资料。刘小雨说明国际间会共享重大事件情报，并转述不同事件可能由同一只鬼造成的观点。会议尚未开始，杨间只获得有限警示；总部完整布局、记录本信息的真实性及与会者身份仍未揭示。（第486章）

## 第487章·刘小雨的危机

会议前，刘小雨收到同事“快来帮我”的短信，认为尚有半小时便可处理，独自前往十二号休息室。开门的高志强以同事在卧室为由让她进入；她发现同事目光呆滞、麻木脱衣，随即意识到对方受鬼影响，却被高志强夺去手机。刘小雨认出他是代号鬼骗人的特招驭鬼者，高志强承认以能力控制原接线员，并要利用同一能力令她十天内以为自己心甘情愿。刘小雨试图逃跑呼救，仍被控制得失去神采，按命令关门躺下。高志强准备实施侵害时，角落出现红光，杨间由鬼域现身；他看到刘小雨受控，未接受高志强私下和解，只盘算在总部杀人会否有影响。两名接线员如何恢复、高志强能否受惩均未写明。（第487章）

## 第488章·疑心病

杨间将高志强一脚踹出十二号休息室，后者胸骨凹陷、身体扭曲，却在十余秒内如重启般恢复。高志强试图以总部会议和厉鬼复苏风险劝杨间止手，杨间则说明自己动手的关键不是替普通人出头，而是刘小雨作为专属接线员已被高志强的能力影响，今后可能不可信；为避免同类风险，他宁愿杀掉高志强。警报引来其他驭鬼者围观，有人要求顾及总部，有人明确赞同处置高志强，但无人直接介入。围观者也由近期战绩判断杨间的重要性高于高志强。高志强听到“杨间”身份后想起鬼眼刑警的名声，立即转身逃跑，意图拖到总部人员赶来。本章止于他穿过休息室走廊奔逃，杨间是否追上、两名接线员状态和会议后续均未揭示。（第488章）
