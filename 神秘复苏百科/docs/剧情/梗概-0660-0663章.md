# 第660-663章剧情梗概

## 第660章·拒绝

陈淑美为失联的熊文文不惜提出金钱和关系交换，杨间仍拒绝进入鬼画，只留下副部长曹延华电话及一条画内出口线索。随后他到平安酒店正式处理组队：黄子雅立刻选择随他回大昌市；张雷胸前食鬼者复苏加剧，杨间转让王小明三个条件中的一个，让他争取驾驭第二鬼；王江因舍不得负责人身份而被杨间排除。得知冯全、童倩失联，杨间决定带黄子雅先找冯全。（第660章）

## 第661章·离市

杨间和黄子雅乘出租离开大京市，路过商业楼时发现第十三层窗内有模糊女子，判断鬼画中的目标已经走入现实。杨间认为总部应派顶尖驭鬼者应对，自己则因鬼画压制鬼眼鬼域、八音盒诅咒又已解除，拒绝再参与。他联络苗小善，要求她和室友尽快离校离市并保存红色鬼烛。车辆离开市郊后，他的鬼眼预警才消失，表明鬼画影响范围极广。（第661章）

## 第662章·大雾

出租车抵达小安县后，路上突起白雾。黄子雅认出这近似冯全的鬼雾，杨间判断不是复苏而是冯全遇险，开启三层鬼域压开浓雾，在宾馆房间发现冯全被坟土埋住、仅剩头颅在外。杨间以变黑鬼手压制坟土救出冯全。冯全称自己受童倩手机信号诱骗、遭陌生驭鬼者袭击，只能以坟土与鬼雾自保；三人决定继续追踪手机。（第662章）

## 第663章·老一批的人

冯全独自取起荒地水泥柱上的童倩卫星定位手机，立刻被拉入赤红天空的陌生鬼域。旧一批负责人裴东现身，冯全认出其代号“鬼摸头”。裴东承认己方利用童倩手机制造失联假象，猎杀驭鬼者后让新人继承遗鬼；童倩尚未死亡，却因特殊价值被控制。冯全鬼雾不断被压制，周边出现五名驭鬼者。杨间和黄子雅在外目睹冯全消失，杨间准备入侵鬼域救人，但围困尚未展开结果。（第663章）
