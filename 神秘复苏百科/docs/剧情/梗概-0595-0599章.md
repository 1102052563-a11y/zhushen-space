# 第595-599章剧情梗概

## 第595章·姜尚白的鬼

大半市区仍在红色鬼域中，总部向所有携卫星定位手机的驭鬼者发出平安大厦求援。郭凡、钟山从朋友圈突然求助及鬼域只定向袭击大厦，推测杨间可能未死；曹洋、李军等人也开始赶往现场。总部会议室内，王小明承认刻意延后求援以收集信息，并准备必要时调卫景支援；卫景现身后直接断言杨间未死，王小明判断其应是开启八音盒。顶层的姜尚白无法等到救援，面对无头鬼影亮出隐藏的鬼踩人，以尸斑鬼脚踩住影子，短暂令其停滞。杨间因而确认姜尚白的三鬼配置和鬼脚限制效果，却决定继续挖掘鬼影能力，双方谈判未成。（第595章）

## 第596章·铺开的黑影

姜尚白踩住无头鬼影后自觉有了谈判资本，要求杨间收手并愿意补偿；杨间现身，指出朋友圈曾要在六点前杀死自己，如今只是在重演其做法。交谈间，原本被踩住的影子已铺满残缺顶层，黑暗中隆起多具尸体般轮廓，多只影手抓住姜尚白四肢和脖颈，欲把他拖入地下。姜尚白的鬼脚无法压住整片鬼影，鬼寿衣也仅暂隔侵入。杨间不接受其“无冤无仇”说法，亲手将其按入黑影，并用鬼手压制其体内鬼；鬼寿衣需三个压制名额才失效。姜尚白彻底被吞没，杨间取得寿衣，朋友圈最后留在顶层的队长级成员由此死亡。（第596章）

## 第597章·方世明再现

郭凡、钟山、李军、曹洋、陈义、苏凡等人聚到平安大厦楼下，因百层大楼电梯停运而暂不贸然上楼。姜尚白尸体突然从顶层坠下，李军辨认身份并准备先处理其体内复苏的鬼。顶层中，杨间把姜尚白尸体扔下，只取回鬼寿衣，继而怀疑未见鬼剪刀的方世明替身有异，准备探查地下安全屋。方世明本体却从阴影出现，承认先前鬼剪刀袭击，并以背尸鬼压住杨间。尸体依“范围内身高最高者”锁定目标，三层鬼域亦无法隔绝，杨间胸骨碎裂、体内鬼受压。方世明以为胜局已定，杨间却在无法起身时叠开两只鬼眼，首次开启第六层鬼域。（第597章）

## 第598章·停滞和对抗

第六层鬼域红光覆盖顶层，杨间发现此前五层鬼域送走的建筑和人员并未死亡，而是被隔绝；更直接的效果是周围一切连同他自身、方世明和背尸鬼都停滞，唯有鬼眼和意识仍可活动。八音盒铃声变慢，说明诅咒也受影响。约三十秒后厉鬼逐渐恢复，杨间把握鬼域仍能改变位置的机会脱离背尸鬼，放弃先取鬼剪刀，改以鬼手折断方世明脖颈并压制其鬼。方世明以阴冷怪风挡住鬼影，携杨间从近百层高楼坠下；他抛出的首只替死布娃娃被杨间五层鬼域毁掉，但那只是诱饵。两人继续下落，方世明暗藏的第二只布娃娃和杨间的八音盒状态都成为胜负变量。（第598章）

## 第599章·坠落和未死

楼下众人刚处理姜尚白的风险，笼罩市区的红光便暂时消失，平安大厦前恢复正常夜景；杨间与方世明随即从高空砸落，令地砖碎裂。苏凡的手机预警危险，众人不敢接近。检查确认坠落者是杨间和方世明后，钟山等结合姜尚白死亡、杨间特意出现在大厦的事实，推断这是对朋友圈的报复。方世明先靠第二只替死布娃娃爬起，说明先前那只只是诱饵；两只娃娃耗尽、厉鬼消耗过多，又误认总部人员来援杨间，使他准备撤退。杨间则在八音盒铃声中由无头鬼影拼接扭曲身体，确认自己未死。二人再度对峙，方世明发现街道仍在杨间鬼域中，冲突没有结束。（第599章）
