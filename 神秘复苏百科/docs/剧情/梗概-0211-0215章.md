# 第211-215章梗概

## 第211章·亡命逃亡

大昌市启动对王小强、叶枫的追捕，杨间选择先抓较易处理的王小强，赵开明和孙义仅按安排支援。俱乐部被近百武警包围，王小强接王小明电话后遵指示驾跑车冲破路障北逃。杨间判断北方有人接应，以鬼域把覆盖范围压成狭长路径，将单次移动距离由约三十米拉至约三百米，开始追击。（第211章）

## 第212章·化虹

王小明在直升机上通过卫星定位观察追逐，原以为杨间鬼域有停顿、难追跑车。杨间则不断尝试把移动与前伸鬼域衔接，最终消除间隔，以红光高速飞行，速度提升到每秒数百米。王小强看见兄长直升机后以为得救，杨间却已坐进副驾驶并把跑车拖入鬼域；王小强持枪分神，车辆撞毁，他虽重伤仍因体内鬼未死。（第212章）

## 第213章·王小强之死

鬼域内杨间以鬼影勒住王小强，将汽油转移到其头顶。王小明在鬼域外愿意开价求情，杨间强调暗杀者此前未曾放过自己，点火焚毁王小强外层皮肉。王小强死亡后留下会活动的骷髅鬼；王小明说明应拆分黄金关押，且不能让其接触活人血肉。杨间以鬼影拆骨、用金箔逐根包裹，王小明压住丧弟之痛离开。大昌市天空同时出现青黑阴霾。（第213章）

## 第214章·大型灵异事件

王小强被击毙后，亚洲国际刑警分部讨论是否撤换杨间。赵建国承认其性格与报复风险，却以第七高中、福仁商场、黄岗村和鬼敲门救援战绩主张重用，会议最终保留杨间。随即卫星图显示大昌市异常，赵建国命刘小雨、赵开明接线员保持通讯，初判可能是能危及数城人口的S级大型灵异事件。（第214章）

## 第215章·老司机

王小明的直升机因青黑阴霾无法飞离大昌市，确认是灵异事件并要求总部隔离城市、叫李军携鬼烛支援。杨间也察觉全城光线变暗、空气阴冷，要求出租车司机老王加速返回。老王先怀疑刑警身份，见杨间开枪后启动改装车高速行驶，准备以驾驶技术赶路；阴霾的来源、范围和城市内具体伤亡尚未揭示。（第215章）
