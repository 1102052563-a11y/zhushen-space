# 第361-365章梗概

## 第361章·踩脚印

罗素一在两点十五分未按熊文文预见被腰斩后，杨间不把地毯本身当作答案，而是回看持柴刀高大尸体行走时留下的腐臭脚印。他用一只鬼眼制造小范围红色鬼域，将众人先前进入走道、尸体折返的路线和时间重演出来：尸体折返后留下的脚印，恰在两点十分覆盖熊文文原先的脚印，随后便在其先前站位高度挥刀。杨间据此判断，柴刀砍的不是现在的人，而是留下脚印时的人；但触发点不是单纯踩到，而是尸体脚印完整覆盖目标脚印。熊文文的下一次死亡将至，杨间立刻翻转脚下地毯，把众人脚印藏到反面，时间到后熊文文未死。罗素一因此明白杨间靠墙走也避过了中间路线，众人暂时获得应对走道鬼的明确方法。（第361章）

## 第362章·墙壁上的脸

杨间命罗素一和林落梅翻转走道地毯，遮住所有已留下的脚印；他认为柴刀鬼一旦离开酒店，便可能踩中大量人的旧脚印，故暂不能放任其外出。为避开正走来的尸体，四人强行进入三十一号房。房间与十三号房不同，灯光明亮，四壁贴满带血旧报纸，每张报纸中央都挂着闭眼的人脸，数量至少二三百张。杨间由十三号房无脸身体、三十一号房人脸的对应关系，推测这是一场早有准备的陷阱：来者为追查赵磊、人脸或源头鬼而深入酒店，便会在走道留下脚印，最终被柴刀鬼杀死。熊文文发现一张人脸与童倩完全相同，杨间因此怀疑曾被送入医院、仍以童倩面貌出现的对象可能是鬼，而真正童倩只余墙上人脸；但鬼脸究竟何在、这一判断能否证实，尚无答案。（第362章）

## 第363章·鬼报纸

杨间再次搜查三十一号房，未找到鬼，只能将墙上人脸及其背后的红色报纸全部取下，准备带走后再找出源头。罗素一被安排动手，林落梅和熊文文休息；杨间明言报纸可能有危险，却以罗素一先前获救为由要求其承担风险。他留意到熊文文脸色惨白，判断反复预知已接近厉鬼复苏，而卫星定位手机在这个灵异空间中没有信号，更加深了郭凡求援信息是诱饵的判断。二十分钟后，罗素一发现一张报纸写满赵磊字迹，内容涉及监视杨间、忘却过去与成为鬼奴；纸张随即鼓起、试图覆盖罗素一脸部。杨间赶到后以鬼影压住报纸，墙面其余人脸纷纷落下，确认这张就是源头鬼报纸。他推测其通过夺脸、写字、贴脸篡改记忆，遂用金纸和黄金容器收容；赵磊原脸仍未找到，童倩身份被冒用的风险也未解除。（第363章）

## 第364章·冯全的电话

在三十一号房停留约两小时后，杨间决定离开，暂缓处理酒店内的人头气球事件与柴刀鬼。罗素一主张趁掌握踩脚印规律关押尸体，杨间则认为知道规律只够自保，若没有限制手段、足够黄金和直接接触的条件，仓促关押只会使高危险厉鬼日后脱困；他还想取得柴刀，故要求众人保密。杨间以鬼域带人离开异常走道，不让他们留下新脚印，却发现鬼域不能探入周围任何房间，只能沿撤离路线延伸。他将酒店所有地毯都转移走，连一楼可能留脚印的地砖也撬离，准备日后封锁酒店。就在一行人回到四楼窗边时，卫星电话接通了冯全。冯全称自己状态极差、鬼雾已脱离控制，请杨间带他离开；杨间虽从定位确认对方在四楼，却仍怀疑这个本应已牺牲的人及其电话的可信度。（第364章）

## 第365章·杨间的救援

杨间接到冯全电话后，没有立刻信任对方。他一面把同伴送到酒店一楼，一面追问冯全是否遇到持柴刀尸体；冯全承认勉强承受过一次袭击，身体残缺使鬼雾被强行脱离，而非完整复苏。赵建国随即接入电话，称自杨间进入酒店后中山市未再增加人头气球死亡，要求其带冯全撤离，并以功劳相劝。杨间承认自己只解决了部分问题，最终答应救援，但先把熊文文、林落梅、罗素一送出鬼雾，把赵磊无脸身体交他们看守，同时警告医院里的童倩可能是鬼。为换取罗素一继续协助，他重申会将自己取得的第二鬼驾驭名额交给林落梅。安排妥当后，杨间带着新得的鬼报纸重返鬼雾，准备把仍不可信的冯全当作试验对象，验证报纸能否篡改记忆；他只承诺让冯全活着，并未承诺救援的方式。（第365章）
