# 原文回跳第745-749章剧情梗概

## 原文回跳第745章·灵异论坛的强势

孙瑞专机带杨间、李阳飞往大海市送红信。因忌惮灵异论坛，杨间先绕经大昌市，以鬼域从观江小区带走鬼童作为底牌；王珊珊能看见鬼域，默许其带走。专机落地后，灵异论坛管理员阿武带人强行拦截，要求孙瑞不得下机、立即返航。阿武发现杨间同机后态度迟疑，杨间则明确不愿仅因叶真地盘规矩就放弃送信行动。（原文回跳第745章）

## 原文回跳第746章·退让

阿武坚持论坛规矩，杨间给出十秒期限，要求其撤人或全员死亡。阿武判断自己无法同时对抗杨间、孙瑞、李阳，选择退让上报。明珠大厦内，叶真得知杨间进入大海市，一度想维护地盘威严；经理提醒双方无直接利益冲突，且杨间可能持有总部失窃棺材钉，贸然交手风险很高。叶真暂不亲自出面，但开始密切关注福寿园。（原文回跳第746章）

## 原文回跳第747章·古怪的墓地

杨间三人抵达福寿园，这座民国起源、现代化改建的墓园表面正常。78号墓穴已挖好却无棺材，墓碑显示刘老太前天死亡。杨间判断他们到得太早，尸体尚未入葬，红信应在其成为78号墓主后再投递。三人发现园内有多块无字碑，也认为红信既然送给刚死老人，危险多半来自墓主而非路程。李阳向保安确认下午将举行刘老太入葬仪式。（原文回跳第747章）

## 原文回跳第748章·墓主的身份

叶真得知福寿园将葬刘氏公司老板刘元的母亲，仍无法判断杨间目的。他担心杨间有棺材钉，却接受经理建议，先观察而非试探交战，并安排人员关注墓园。福寿园内，杨间和孙瑞未查到直接灵异，只发现更多无字墓碑。李阳确认刘老太棺材将在下午运至78号墓，杨间决定等待下葬，避免利用文字漏洞过早送信。装有刘老太遗体的黑色欧式棺材随即被运往墓园。（原文回跳第748章）

## 原文回跳第749章·齐聚

刘老太葬礼在福寿园举行，近百名宾客到场，刘元安排临时灵堂和告别仪式。杨间以鬼眼确认棺内原本只是普通尸体，仍坚持等入土后投信；孙瑞驱走试图赶他们离开的助理张鑫。灵堂内，刘老太悄然睁眼，一号墓园同时被诡异暮色覆盖，异变开始。叶真最终派罗素一与林落梅前往调查，墓园即将同时汇集杨间一行、刘家宾客和灵异论坛人员。（原文回跳第749章）
