# 第489-493章剧情梗概

## 第489章·真假

总部收到十二号休息室附近斗殴警报后，曹延华担心顶尖驭鬼者冲突失控，命沈良先去处理、请秦老震场，并准备提前开会。休息区内，高志强表面逃跑，杨间却发现普通视野所见的逃身不在鬼眼红光中，判断是假象。高志强实际藏在杨间三米内，以鬼骗人欺骗众人视线；他自称还能欺骗自身厉鬼不复苏、让重伤身体相信无伤而恢复。杨间叠开第二只鬼眼，终于找到本体，放出无头鬼影侵入高志强。高志强身体随即被拆下四肢和头颅，意识仍被鬼影维持。假身使围观者一度误判杨间未追击，实际能力已被看穿；高志强能否再次恢复、总部人员何时到场仍未写。（第489章）

## 第490章·高志强的能力

围观驭鬼者由黑影肢解高志强的表现，意识到杨间除鬼眼外还驾驭危险的无头鬼影。高志强的鬼被压制，无法再欺骗身体复原，便在死亡威胁下实施极端尝试：让自己的鬼相信它是高志强，同时让自己相信是鬼。杨间以鬼域将其头颅埋入地下后，高志强却以无实体的阴冷声音出现，称厉鬼已取代自己并会继续自我欺骗，维持人的思维。他随即令杨间相信自己不存在，使杨间身体开始消失。杨间判断真正的高志强已经死去，留下的是拥有其思维的鬼；新鬼能否长久维持人格、能否击杀杨间，均未得到验证。（第490章）

## 第491章·启发

鬼骗人令杨间肉身和意识逐渐模糊，但鬼影形成黑色躯体、鬼手和鬼眼仍未受影响，阻止他彻底消失。围观者由此看出杨间驾驭鬼眼、鬼影和鬼手三只鬼，部分人准备救援，担心总部内发生同僚杀死S级事件功臣的后果。杨间却从高志强自我取代得到重启启发，认为这只鬼可能影响更多厉鬼，并决定收走目标。鬼手抓住已化鬼的高志强，使其无法行动；杨间随后令鬼影反向侵入，试图强行驾驭该鬼、取回自身身体。王小明、李军、曹延华及秦老这时到场，秦老引发众驭鬼者体内鬼的极度危险感。高志强意识将如何处置、杨间能否恢复仍悬而未决。（第491章）

## 第492章·不可思议的鬼

曹延华到场时误以为杨间在总部欺压同事，王小明指出高志强已死、放开只会帮助鬼反击；曹洋说明高志强先控制两名女接线员。工作人员检查十二号休息室后确认刘小雨和高志强的接线员张乐乐均思维异常，曹延华由此接受冲突起因。与此同时，杨间以无头鬼影覆盖高志强之鬼，成功逆转自己身体消失的影响，并进一步让该鬼欺骗自己的身体恢复健康。其人体部分恢复血色、健硕和生机，鬼手所在手臂不受影响。王小明认为杨间已迅速理解现实影响的用法；杨间则发现这并非单纯幻觉，而可能是能将愿望化为现实的鬼。修复是否永久、目标是否可长期驾驭及接线员能否恢复，均未写。（第492章）

## 第493章·无法驾驭

杨间尝试真正驾驭高志强之鬼后，发现体内多出沉默意识，且鬼眼、鬼影和鬼手产生排斥，判断该鬼不是不复苏，而是会逐渐与人意识融合、最终以鬼取代人格。他承认长期纳入会破坏三鬼平衡，只能偶尔用其修复身体；王小明认为他虽未掌握重启，方向仍可研究。曹延华要求总部接收目标，担心杨间杀同事后再私取厉鬼会形成坏例。杨间转而申请未来驾驭资格，接受程序处理，却把高志强外形的鬼封入黄金箱，再以鬼域送进总部地下深处暂存。曹延华只能默认这一安排，鬼的正式归属、杨间申请及长期封存效果均未揭示。（第493章）
