# 第625-629章剧情梗概

## 第625章·楼梯上的动静

杨间向李阳、吉米说明自己不是为带走幸存者而来，而是要寻找鬼画内的鬼；两人若想给其余人争得离开机会，便须以熟悉城市的经验带路。吉米回忆曾在一栋国外居民楼门口见到红衣女人，三人遂向那里进发。杨间从城市背景仍未变化判断自己仍在较完整的一幅鬼画内，又由幸存者能活数月推测鬼画杀人效率未必极高、真正危险在于会不断扩张的鬼域。抵达七层居民楼后，他要求二人同行以补足鬼眼受压制后的视野。李阳说服原本拒绝的吉米上楼；杨间刚踩响木质台阶，楼上即传来相同嘎吱声。众人停步时声音也停止，杨间确认楼上有会随楼梯脚步活动的未知鬼，人鬼继续上下楼终会相遇。（第625章）

## 第626章·第三楼的异常

杨间退到二楼水泥地面后，发现木梯脚步消失，便让李阳和吉米谨慎上楼，判断声音是楼梯鬼的关键。二楼搜索无果，他找到室外铁梯，借其不会发声的特点绕开楼内木梯，三人从窗进入三楼一户房间。屋内有一具腐烂数月的普通尸体，走道却密集躺着十余具不同腐烂程度的尸体；杨间据一、二楼没有尸体而死者集中三楼，推断这些人被困于此，鬼画或许就在本层。此时吉米发现来时的窗和室外楼梯已变为墙面，鬼画改变布局封死出口。杨间认为目标既然在近处，危险也在近处；三人只能留在三楼，继续寻找藏起来的鬼画。（第626章）

## 第627章·沙发上的人

三楼出口消失令杨间进一步判断鬼画为了避免被找到而改变环境。他翻查走道遗体，在部分尸体手中发现黄金子弹手枪等驭鬼者装备，点出五具疑似驭鬼者和十二具普通人尸体，推测曾有国外队伍在此寻找鬼画后覆灭；楼梯鬼也可能是其中一人死后遗留，但均未获证实。李阳看见一间未锁房内有黑影，杨间遂入内查看。屋中保留一扇能望见楼外的窗，吉米一度想逃，杨间却指出其可能是另一只鬼的干扰或陷阱。客厅单人沙发上坐着一具腐烂尸体，杨间本想不惊动地退出，余光却见人影掠过窗前；回头时沙发已空，身后尸臭逼近，房门也被某个看不见的存在猛然关上。（第627章）

## 第628章·李阳的改变

杨间被关在房内后，以无头鬼影撞门，只从门缝看见腐烂手臂撑住门外；李阳、吉米则在外面看见腐尸跑出并堵在门前，无法逃离。杨间观察到鬼改堵窗时门便会打开，得出其一次只能封住一个出口的判断。他让李阳进入房内盯住窗外鬼，自己趁门未堵时离开，再准备在外限制鬼。李阳依约奔向门口时仍被鬼抢先关门，几乎成为替代品；杨间立即以鬼手压住腐尸，尸体散架、门重新开启，李阳得以逃出。杨间没有工具也不愿耗时关押此鬼，选择关门离开。脱险的李阳却脸色如死人、浮现尸斑，杨间据此判断附在腐尸上的鬼转到了他身上，李阳从幸存者意外成为驭鬼者。（第628章）

## 第629章·必死的脚步声

李阳得知被鬼附身后惊恐追问，杨间解释他不是变成鬼，而是被鬼侵蚀；初期未必有感觉，终究可能变成腐尸，驾驭第二只鬼或可延缓死亡却更危险。李阳仍接受不能一味躲藏的现实，和吉米继续配合搜索。杨间确认三楼其余房间没有鬼画，判断目标很可能在四楼；木梯是唯一仍在的道路，他认为其未被鬼画改变或因楼梯鬼存在，遂让刚成为驭鬼者的李阳与普通人吉米留守三楼，独自上行。踏上木梯后，五楼方向的沉重脚步随之回应。杨间抵达四楼、尚未看见鬼便骤然昏厥倒地，阴冷脚步在身边徘徊；八音盒铃声将其意识拉回，鬼才远去。他确认这是第三次被诅咒从必死袭击中救回，恢复后仍决定在四楼继续找鬼画。（第629章）
