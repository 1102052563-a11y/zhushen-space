# 第406-410章梗概

## 第406章·抵达大京市

源头鬼手被限制后，秦媚柔向总部汇报本次航班确认死亡两名飞行员、三名乘客，曹延华认为五人伤亡已远低于通常灵异事件，但飞机仍在坠落。杨间以金箔和黄金盒收容第三层鬼域中才显现的鬼手，由此理解黄金能跨鬼域限制厉鬼，并意识到王小明早已掌握更深研究。他没有让客机坠毁，而是以一层鬼域托住失控飞机、站上机头把全机送往大京市机场。机场的驭鬼者提前接应，有人忌惮也有人讥讽其高调到来；杨间则将接触王小明列为下一目标。航班事件暂告结束，但鬼手与黄金原理的后续价值仍未展开。（第406章）

## 第407章·再见沈良

飞机落地后，乘客仍因停电与坠机经历崩溃，王东确认抵达大京市，归还借枪并获杨间建议迁往大昌市。沈良带郭凡、钟山接应，称赞杨间一小时内解决事件；杨间不接受空泛褒奖，先索取出差奖励，最终改要沈良安排与王小明见面。郭凡因中山市事件指责杨间未救援，杨间反驳凯撒酒店的求救信号可能是陷阱，沈良暂时调停。杨间本想休息，却因沈良称航班源头可能可调查而留下半小时，要求获得调查结果。救援已开始，但魂瓶为何上机、鬼手如何被放出仍是后续核心。（第407章）

## 第408章·奇怪的邀请

乘客和尸体被转移后，钟山在临时棚提醒杨间，外界把C级“鬼掐人”视作易事，实则总部的就地处置要求是在迫使驭鬼者冒险。他提出联合五名同类争取队长计划席位，以资源、情报和集体威慑对抗总部对民间驭鬼者的管制；郭凡也劝杨间听取意见。杨间认为信息不足，拒绝立即加入，只保留观察态度。随后调查人员称已找到事件源头，杨间起身查看。本章未给出队长计划的最终名单或钟山其余同伴身份，但显示驭鬼者与总部之间的权力矛盾开始被明确提出。（第408章）

## 第409章·魂瓶

调查人员在破损行李箱中发现古董和黑色陶瓷器。大京市国际刑警陈义因找不到源头而暴躁，郭凡却感到黑色瓶子异常；白教授学生判断其是约百年历史的近代魂瓶。钟山摔碎外层陶瓷，露出内藏的黄金瓶，杨间据此推断百年前驭鬼者曾将厉鬼装入黄金容器，瓶盖在航班颠簸中脱落，才造成鬼手逃出。调查随即找到行李箱主人万德路，杨间认出这位头等舱相识者。魂瓶把事件指向民国时期灵异遗留，但瓶中是否还藏有别物、万德路从何取得古董均未揭示。（第409章）

## 第410章·突然的袭击

万德路因行李箱被武警押来，猜到古董引发闹鬼后几近崩溃。陈义以灵异力量将其提起审问，杨间为掌握魂瓶线索与陈义争夺审问权，并以红光鬼域展示威慑；陈义最终放人，沈良将万德路和报告交由杨间。郭凡不满事件好处落入杨间手中，随手抛出黄金魂瓶，却意外触发瓶内惨白手臂。手臂抓住郭凡脸部，试图把他拉入巴掌大的瓶口，现场众人才意识到此前只限制了逃出的鬼手，黄金瓶内还有未知厉鬼。航班危机刚结束，魂瓶的新袭击立刻开启更高风险。（第410章）
