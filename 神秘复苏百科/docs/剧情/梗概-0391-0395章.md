# 第391-395章梗概

## 第391章·养小鬼

晚饭后，杨间带王珊珊上楼，从观江花园别墅的备用黄金盒和鬼绳准备处置其腹内鬼婴。他先承认曾把王珊珊变为半人半鬼，又说明她一旦失去自己的力量维持便会再度濒死；王珊珊则表示不愿等死。杨间以鬼眼锁定胎儿，借鬼域和无头鬼影将其无痛剥离，取出一只巴掌大、肤色淡青、红眼似鬼眼的婴儿。它没有按既有规律攻击最先见到的杨间，证实其为异常个体。面对灭杀或豢养两种选择，杨间因其可吞鬼成长、价值极高而决定冒险养起；王珊珊自愿照看，接手后鬼婴亦未显凶相。杨间同意其搬往小区后方新庙独住，并计划以鬼报纸再加控制。鬼婴是否真能长期听令、寄养是否安全均未揭示。（第391章）

## 第392章·说服

杨间将鬼影的脸取下覆上染血鬼报纸，为鬼婴写入三条命令：完全听从杨间和王珊珊、未获吩咐不得杀人、不得随意进入人群；再将脸拼回，改写是否永久有效尚无验证。王珊珊坚持以养鬼婴保护父母和小区，杨间遂交给她特制手枪、黄金盒及半截白色鬼烛，说明白烛可在失控时将鬼婴引入盒内，却也可能引来老宅黄金门后的未知危险，要求她记录过程。下楼后，王珊珊公开鬼婴和寄养决定；王海燕恐惧反对，江艳亦担心其会伤人。杨间称风险已被考虑、决定出于王珊珊本人意愿，王珊珊则以杨间曾救一家人为由拒绝退让。王海燕最终未能说服女儿，鬼婴寄养由此确定。（第392章）

## 第393章·登机

杨间回忆五日前医院检查：专家称其身体机能衰减、局部坏死，普通人本应抢救，预计只可维持三个月至一年。他据此确认半人半鬼身体难以长存，且无头鬼影更换身体的本能可能随条件不足而影响思维，决定赴大京市寻求驾驭第三只鬼的资源；寿衣虽可能使身体转鬼，却随已封存饿死鬼而无法轻动。他携人皮纸、鬼绳及武器从邻市机场出发，并由秦媚柔确认其母亲仍在大京市受照顾。安检时，一名中年男子携古瓷器和黑色陶罐先行通过；杨间因武器触发警报后获特殊通道放行，鬼眼却对该男子产生不安感应。飞机货仓内，陶罐盖因运输颠簸裂开，阴冷气息溢出且灯光闪烁，罐内之物开始脱困。（第393章）

## 第394章·货仓的动静

杨间被安排进头等舱，因翻书、使用平板遭一名西装男子拐弯嘲讽，双方言语冲突；杨间要求道歉后，万德餐饮董事长万德路主动递名片劝和并称他为“腿哥”。西装男子查到万德路经营的企业市值百亿，因而不敢还嘴；杨间只承认张伟是同学，对张伟直播挥霍的事感到意外。飞机起飞后，货仓中黑色陶罐的盖缝完全打开，阴冷气息加重，一个模糊之物从罐内探出并撞破行李箱逃离；气息随后向客舱侵蚀。客舱中表面的口角没有立即酿成冲突，但货舱异常已使航班处于未被乘客察觉的灵异风险中。陶罐所装何物、携带者是否知情及其具体规律均未说明。（第394章）

## 第395章·飞机有鬼

万德路由杨间的持枪安检待遇判断其来历不凡，继续邀请饭局并取得号码。秦媚柔忽然来电，称杨间所乘飞机十分钟前出现异常，希望他先确认是否为特殊事件；杨间虽抱怨情报不足，仍带两把金色手枪随空姐离席，万德路因担忧也偷偷跟随。空姐只知有乘客死在洗手间，杨间在途中批评总部未先收集情报，却为争取后续资源决定先确认。抵达机上隔间后，他见一具女尸倒地，脸覆衣物，四周有阴冷感和尸臭，无需验尸便判定飞机存在灵异事件。其他乘客仍被瞒在鼓里，机上厉鬼的身份、死亡规律、货仓陶罐与尸体的直接关系及杨间能否处理，均留待后续。（第395章）
