# 原文回跳第720-724章剧情梗概

## 原文回跳第720章·寻来的两人

柳三与李阳从总部赶到观江小区，未进小区便被鬼童拦下。柳三由鬼童的饿死鬼外貌和不主动袭击的异常，判断其受杨间控制；杨间随后以鬼域现身，命鬼童离开。面对柳三和李阳的来意，杨间保持敌意，警告不得随意靠近观江小区。李阳是鬼画行动幸存者，此行希望加入杨间队伍；柳三则带来鬼画行动后续与总部局势情报。杨间虽接受谈话，却把二人直接转移至尚通大厦，以自己主场控制接触风险。（原文回跳第720章、原文回跳第721章）

## 原文回跳第721章·局势的骤变

柳三转述鬼画行动惨败：陈义已死，李阳被李军送出，李军则多半留在鬼画中死亡；秦老出手后暂将鬼画限制在一栋大楼内。更严重的是，秦老进入鬼画期间，总部遭拥有高阶鬼域者入侵，多只关押鬼被带走，饿死鬼与棺材钉也可能失窃。柳三称叶真是当前嫌疑人，却没有证据。杨间拒绝替总部试探叶真，也意识到柳三的问题可能想促成自己与叶真的冲突。总部提出三根鬼烛、棺材钉等报酬寻人，杨间仍选择不趟浑水，只守大昌市。（原文回跳第721章、原文回跳第722章）

## 原文回跳第722章·特别的纸人

柳三带来的另一目的是熊文文。王小明认为熊文文被鬼相机关进照片并非必死，若使用鬼童吞下的灵位、熊文文照片和特殊身体，可尝试意识转移。柳三主动提供一具仿真纸人，撕下自己外层纸皮后露出内部仿熊文文的小纸人，称外层只是一具无意识替身。杨间意识到条件已经齐备，却担心照片中同时有鬼，贸然释放会让熊文文死亡、厉鬼复苏，因此暂不实施。李阳则被杨间接纳入队，前往章华处登记。（原文回跳第722章、原文回跳第723章）

## 原文回跳第723章·反应

柳三纸人自行撕毁，杨间将熊文文专属纸人存进办公室安全屋。张丽琴因饿死鬼可能失窃而恐惧，杨间告诉她总部失窃与顶尖鬼域者有关，也怀疑柳三试图挑拨自己和叶真。杨间回乡八天期间，公司积压章华坐标调查、张显贵聚会、陌生同事预约等信息；他决定先去找章华、次日参加张显贵聚会，并让张丽琴、江艳同行。鬼童因感知观江小区附近有驭鬼者而跑出，杨间暂时离开与张伟吃饭后返回办公室查看。（原文回跳第723章）

## 原文回跳第724章·聚会

张显贵在五星级酒店举办聚会，受邀者多为因“鬼眼杨间”而来的富商和豪门。杨间与张伟却躲在角落吃烤串，查看章华调查出的敲门鬼木板坐标：大汉市建设路869号，一栋反复易主、不断出事的烂尾楼。张伟凭直觉认定烂尾楼闹鬼，杨间未下定论。江艳看到其他女性接近杨间而吃醋。此时杨间以鬼眼察觉聚会门口一群人中有驭鬼者气息，危害暂不强，但来客身份和目的仍未揭示。（原文回跳第724章）
