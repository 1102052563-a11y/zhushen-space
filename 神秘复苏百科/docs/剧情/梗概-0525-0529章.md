# 第525-529章剧情梗概

## 第525章·引路的人

鬼画与鬼差鬼域消散后，五个疑似行动队成员的鬼正向城市靠近。大京市外围的陈义、钟山和郭凡接到消息，得知目标数量为五、规律全然未知，若放入城区可能造成更大灾难。陈义说明总部不以人多取胜，而要挑选能存活、精神稳定且有经验的少数驭鬼者；既定方案是把鬼引向鬼宅。郭凡虽接近复苏，仍因灵位替换能力被指定为引路关键。他取出嵌有相片的灵位，以自身与其中鬼交换，外表逐渐呈现尸体特征，走向五个僵硬同步的人影。五鬼没有形成鬼域，令计划尚存实施条件，但郭凡与目标首次接触前的真正规律、行动队原人下落和能否安全引离均未揭示。（第525章）

## 第526章·意外的出现

郭凡以灵位中的鬼替代自身后，本可借鬼的身份承受风险，却也因替换加速而不断被照片中的鬼侵蚀。五个迎面而来的对象被陈义、钟山认作李军、柳三、苏凡、阿红和徐一平，众人遂推测行动队可能已团灭、被操控或厉鬼复苏，但尚不能确定。更紧急的是，郭凡没有按计划点燃白色鬼烛：外面的鬼继续前行，照片里的郭凡则伤痕增加、意识似被困住。暗中观察的杨间看见外围三人冒险接近五鬼，却不知其布置。引鬼方案原本依赖郭凡安全持烛，如今关键媒介失控，陈义与钟山必须在五鬼接触郭凡前决定是否救援、放弃或改用其他办法。（第526章）

## 第527章·引路人

郭凡确认自己与灵位中鬼的联系断开，身体和照片的身份仿佛已经调转。钟山想靠近压制他体内的鬼，陈义认为五鬼当前逼近、救援代价不可承受，决定把失控的郭凡也纳入引离方案。他追上捧灵位的鬼，将带金针的烛台刺入其肩，点燃鬼烛后撤退。黑色烛火形成阴影道路，五个从鬼画中走出的鬼果然被吸向郭凡，第一层牵引成立。为使郭凡继续前进，钟山再持白色鬼烛站到前方，陈义负责防止郭凡肩上的烛火熄灭，形成钟山引郭凡、郭凡引五鬼的三段队列。杨间认可应变使鬼不入城，未介入行动；但钟山从此须在六只鬼身前行走，任何转头或失误都可能触发未知规律。（第527章）

## 第528章·身边的危险

钟山顶着白色鬼烛沿总部以鬼域开出的直路离城，始终不敢回头或多做动作，陈义则维系郭凡肩头鬼烛，六只鬼被暂时带离城区。总部会议确认引离计划初步成功，曹延华、王小明、沈良和赵建国转而讨论郭凡灵位与黄岗村鬼棺前灵位可能存在的关联；王小明并未否认自己期待出现特殊现象。与此同时，他私下向曹延华提出赵建国可能遭鬼入侵脑部的怀疑，但没有确凿证据。为救回仍可能困在异处的李军行动队，曹延华、王小明和秦老前往地下收容区，准备以鬼门按时开门接人。行动队是否仍活、赵建国的异常是否属实，以及引路队列能维持多久，均未解决。（第528章）

## 第529章·打开的门

曹延华为失联行动队保留三次鬼门救援机会，命秦老在首个约定节点打开这扇被锁链束缚的木门。门后本无实体空间，开启后却显现不可照透的黑暗，并迅速干扰房内灯光。青黑手指密集扣住门框，腐烂人头从黑暗伸出、轮廓逐渐像秦老，另一侧又有苍白脚底板探出，证明门后首先涌来的并非李军等人。时间到后，秦老以拐杖令异象退回、关闭鬼门，门框留下抓痕，后两次机会也因此被压缩。章节末转回黄岗村：李军等五人实际仍在村内，虽从险境勉强活下，却被满墙遗像和两扇挂遗像的民宅门拦住；阿红惊惧发问，队伍无安全路径可走。外界首次救援失败，内外两线都未脱离危险。（第529章）
