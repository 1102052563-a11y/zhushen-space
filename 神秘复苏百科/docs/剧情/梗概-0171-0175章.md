# 第171-175章梗概

## 第171章·鬼脸童倩

杨间抵达火锅店二楼前，外市国际刑警童倩正带数十名鬼域幸存者藏身其中。她发现敲门鬼逼近后组织人群由侧门撤离，却看见门窗、桌椅全化作腐烂墙壁，确认此处只是鬼域伪造的火锅店；敲门声响起，已有幸存者倒地死亡。童倩不得不解开马尾，露出脑后会怪笑的惨白鬼脸，鬼笑能令门外老人暂时停手，但鬼脸也沿其头皮向正面爬行，随时会覆盖她本人。她只能再以头发遮住鬼脸，敲门声便恢复。她决定继续拖延、让人群设法逃走时，杨间带着已点燃、泛绿火光的鬼烛从角落出现。童倩、杨间和幸存者能否利用这段空隙脱身，至本章未决。（第171章）

## 第172章·鬼烛之火

童倩认出额生鬼眼的杨间，杨间则表明自己是以赵建国答应条件为代价来救人，并非来解决敲门鬼。他以鬼烛靠近玻璃门，烛火逼得老人后退，却发现越接近厉鬼蜡烛烧得越快，只愿给众人约一分钟。童倩立即指引幸存者向杨间所称出口逃离，但很多人已崩溃、瘫地或不肯行动；杨间到时便撤，不愿再以鬼烛冒险。童倩最终拉着两三名能动的人追上，余下者留在黑暗中，敲门声、哭喊声渐少直至死寂。老人后来离去，但鬼域依旧存在，获救者也没有真正脱离危险。（第172章）

## 第173章·争执

离开火锅店后，童倩认为杨间既能以鬼烛压制敲门鬼，就该多争取几分钟救出其余幸存者；杨间以烛火消耗、鬼域未散及无人能保证全员存活回应，强调她的牺牲只会制造又一只厉鬼。二人围绕刑警责任和救援取舍争执：童倩坚持成为驭鬼者便应尽力守城、让普通人少受牵连，杨间则坦言自己只求活着，救援也是赵建国拿利益换来。前方路灯下出现僵立人影时，他们暂停冲突，杨间判断可能是鬼奴，选择绕行。随后他致电江艳，得知她仍在鬼域内开车、路途无尽，只能由杨间定位救出；童倩与幸存者暂时留在原处。（第173章）

## 第174章·路灯下的鬼

杨间执意骑共享单车独自寻找江艳，童倩担忧他离开后幸存者再遇敲门鬼，试图劝他先送人离域。杨间以手枪压制童倩，明确自己不会接受指挥，令她留守等待，随后骑车离开。杨间用鬼眼发现一盏仍亮的路灯及灯下背对人影，点燃鬼烛靠近后，蜡烛数秒猛烧五厘米，速度远胜面对敲门鬼。他据此后退并推断人影不在敲门鬼鬼域内，鬼域反而隔开了这只更危险的现实厉鬼。另一边，赵开明已从鬼域出来，路灯下的鬼却仍缠住他；他接连以特制手枪自杀，子弹均卡壳，得知父亲也出事后决定继续活下去。杨间继续搜索，尚未救到江艳。（第174章）

## 第175章·活着的代价

赵开明在路灯鬼阻止自杀、父亲死讯传来后离开现场，路灯下人影消失；他的具体损失和鬼物规律仍未说明。鬼域内的杨间一边骑车、一边以鬼眼警戒，终于发现一辆持续亮灯的豪车，判断是按其先前指示保持发动的江艳座驾。车旁站着已在第七高中事件中死亡的钱万豪，其头颅残缺、身体腐烂，已成敲门鬼鬼奴；鬼烛的光将其逼退，杨间未与之纠缠。透过车窗，他看见江艳仍维持紧张开车的姿势，显然被鬼域幻觉困住，车辆实际早已停下。杨间拨通电话，命令她开门，章节在命令传达后结束，江艳是否恢复清醒、车内女销售去向及救援结果皆未揭晓。（第175章）
