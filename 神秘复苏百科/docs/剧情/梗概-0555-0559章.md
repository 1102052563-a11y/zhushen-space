# 第555-559章剧情梗概

## 第555章·被影响的楼层

涂校长和王泉的人手在宿舍楼外疏散学生，杨间则带王泉及两名人员入楼，约定救人必须服从灵异处置优先。杨间确认一至三层正常，王泉便调人逐层排查；走到三楼通往四楼的拐角，灯光忽然被无形界线削暗，杨间体内鬼眼也像死机般受压制。他判断鬼画已不只是单独杀人，四、五层都开始受到鬼域影响，风险远超先前寝室里的油画。考虑鬼眼沉寂也让自己多出压制厉鬼的名额，加上八音盒仍可保命，他决定继续上楼。四层回望时楼梯已显作通向黑暗深处的无尽台阶，黄金子弹却在第八级下停住，证明现实楼梯仍在，只是人会被鬼域诱入未知处。杨间警告少说、少看、少走后继续搜查，黑暗楼梯上则已有模糊人影和学生遗留手机。（第555章）

## 第556章·错开

杨间认为源头仍是五楼苗小善寝室的油画，拒绝在退路已断的四层优先搜救滞留学生，带王泉直往五楼。四层表面安静，他未发现孙于佳跳窗的原因；其离开后，走廊实则多出一间暗红木门、门内窗口面对飘灰的灰暗世界，显示鬼域已在不易察觉处改写环境。杨间到达记忆中的寝室，却发现门上脚印消失、陈设全非；鬼手取下门牌后，“503”褪为“603”，证实四、五层已重叠，众人并未算错却走进六层。鬼画借鬼域隐藏了原目标，王泉担忧会先遇上真正的鬼；杨间认为现实尚未被完全覆盖，鬼画作为画也未必能随意移动，仍可搜寻。他们不知道的是，多出的四层寝室正敞向灰白世界，源头入口已经留在身后。（第556章）

## 第557章·出现的身影

杨间准备回四层重新寻找，却见下行楼梯已被平整水泥墙取代，只有通往未知上方的阶梯尚存。王泉提出用特制黄金绳从六楼下降，杨间则令无头鬼影渗入地面，拆开钢筋水泥、塑成黑色台阶，带三人安全返回标有401、402的真实四层。杨间说明鬼域再精巧也应有真实连接点，要求王泉三人戴黄金手套沿墙摸索，自己以鬼手从另一头排查；他们分开行动，王泉小队保持谨慎，杨间则逐间踹门确认。与此同时，先前黑暗楼梯内的身影已走上走廊：它穿模糊红衣、没有真实双腿，只有苍白双手异常清晰。楼下观察人员先发现目标并急报，却已无法阻止它在拐角后消失。队伍虽回到正确楼层，画中鬼已先一步进入搜查区域。（第557章）

## 第558章·画中人

杨间搜房时发现两名滞留女学生，只告诫她们留在寝室、不要期待随行救援；王泉一侧也登记并安抚学生，继续以寻找鬼画为第一任务。红衣女子却突然出现在走廊，队员险些惊叫，王泉及时捂住其嘴，判断普通人无法对抗厉鬼，只能目送目标进入已有学生的402寝室，再后撤通知杨间。杨间因鬼眼沉寂未能察觉，仍以鬼手触墙搜寻，终于摸到异常门框。虚假寝室门被压制后露出暗红画框，门板化作苗小善寝室的油画；画内地砖上趴着一名缺失双脚、已无生气的女学生，中央另有原应画着鬼的人形空白。杨间由此推断画中鬼已走出，而王泉的卫星电话同步确认鬼就在402。源头画框和离画目标分处两端，处置选择被迫摆在杨间面前。（第558章）

## 第559章·熟悉第一眼

杨间按住已找到的鬼画画框，没有立刻赶赴402：鬼手在鬼眼沉寂时只能限制一只鬼，若放开画去追画中鬼，源头可能再度消失；若先关押画，则王泉与队员近旁的目标无人应对。队员误以为杨间可能拿他们试探规律，王泉却看出他正同时面对两个目标，严令众人保持观察和纪律。等待中402门缓缓打开，红衣女子走出，面容、双腿和头发如油画般朦胧，唯有白皙双手真实清晰。杨间将其红衣、双手和鞋履与灵异公交车的干尸新娘、鬼橱交易照片中的女子及自己收容的红色绣花鞋联系起来，又立即承认这可能只是没有证据的拼凑。鬼已现身，杨间仍须在限制画框和限制画中人之间作出选择；两者真实关联、402学生遭遇及本轮行动结果均未在本章揭示。（第559章）
