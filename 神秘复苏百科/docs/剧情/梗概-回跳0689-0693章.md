# 原文回跳第689-693章剧情梗概

## 原文回跳第689章·第一件东西

杨间为避免再次进入梦境，收回无头鬼影、主动承受厉鬼影响，整夜不眠。清晨他发现父亲遗像房门半掩，以鬼域搜查整村和房间无果，转而调查父亲留下的鱼塘。池水异常冰冷、鱼少且死寂，鬼眼在池底淤泥中受阻。杨间用鬼手搅开淤泥，捞出一个腐烂却完好的木盒，盒内装着两条浸泡十余年仍不腐的惨白人腿。木盒离水后鱼塘水位下沉，水中倒影开始出现异样；杨间未发现自己取走的第一件封藏物已引来未知跟随者。（原文回跳第689章）

## 原文回跳第690章·养狗的树林

杨间将木盒装入装尸袋，远离村子后确认其中人腿介于驭鬼者和厉鬼之间，怀疑来自已摆脱复苏、近似异类的存在。他到村西树林，找到父亲当年养狗的小木屋，以鬼域挖出几十条健壮狗骨，确认所谓“一夜卖光”的狗其实被集体杀死。树下另有红漆木箱，装着一具无头无脚上半身，与人腿属于同一分尸。杨间不敢拼合，转往第三处小庙。路上，水洼中的非同步倒影持续跟随他，却未被察觉。（原文回跳第690章）

## 原文回跳第691章·离奇的事情

小庙两尊神像各藏一颗惨白人头。杨间以鬼手触碰第一颗时被咬出浅牙印，强行取出后发现其面容陌生、五官有不协调的拼凑感。第二颗则酷似父亲遗像上的面孔，使杨间意识到这里只有一具身体却有两颗头，且分尸封藏时父亲仍在世。杨间无法解释父亲是否早已死亡、记忆是否被改写，也不敢将头颅拼回躯体，只能把尸体各部收走。梅山村表面平静下的历史因而更加诡异。（原文回跳第691章）

## 原文回跳第692章·问路

返村途中，杨间遇李跃驾驶吉普车带三名同伴问路。李跃展示十余年前树林小木屋的照片、以旧版钱请杨间指路，谎称要迁亲属坟墓。杨间看出四人并非普通访客，仍指方向并暗中跟随。李跃离开后告诉同伴，杨间提装尸袋、手掌冰冷、没有常人情绪，极可能是驭鬼者；他们虽知道大昌市负责人叫杨间，却不知道刚才遇见的就是本人。其目的、旧照片来源和与分尸的关系仍被隐瞒。（原文回跳第692章）

## 原文回跳第693章·同样的目的

李跃一行进入树林，按照片找到小木屋，挖出已被打开的红木箱，发现尸体已失踪。杨间现身，承认尸体在自己手中，要求他们说明为何知道封藏地点；持枪威胁无效，李跃听到“杨间”姓名后确认其为大昌市负责人。李跃取出一封黄色信封，称正因这封信才来找尸体，并自称信使；他拒绝杨间查看，称非信使读信没有好处。双方围绕尸体和信封信息僵持，信件体系与任务目标尚未揭示。（原文回跳第693章）
