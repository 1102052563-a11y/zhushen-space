# 第640-644章剧情梗概

## 第640章·谁会活下来

刘小雨因鬼童近在别墅而恐惧，杨间承认自己在控制饿死鬼残留鬼奴，请她暂时保密。刘小雨起初以工作准则拒绝，杨间则以无头鬼影说明驭鬼者本质上都在用鬼对付鬼，并承认无法保证鬼童永不失控；刘小雨最终答应，却被限制至次日不得离开别墅。与此同时，王小明在郊外秘密实验室同陈博士检查残缺鬼棺与灵位。陈博士指出郭凡若接管杨间身体，未必能驾驭三只鬼；王小明却明确无论诅咒杀死谁，杨间身体都应保留为可用战力。基地内已准备棺材、灵位和郭凡遗照，王小明还提到次日将有针对鬼画的关押行动。杨间休息前相信八音盒与鬼橱交易规则会使自己暂不死亡，等待王小明方案。（第640章）

## 第641章·准备

杨间在八音盒旋律中浅眠，梦见密闭昏暗空间里有残指黑手停在残缺钢琴上，钢琴缺失的键已有五个被新补上；黑暗阴影靠近时，脑内铃声阻止其袭击。杨间怀疑这是诅咒或体内鬼传来的信息，却无法确定地点，只排除凯撒大酒店71号房。刘小雨随后通知王小明已准备完成。杨间为防方案出问题，决定增强鬼童：将鬼寿衣交给它穿上，寿衣自行缩小且未与鬼童冲突；又交给它装尸袋内的死人头，并带一只替死娃娃。杨间命鬼童无令不得袭人，随后让刘小雨留在别墅、自己按定位前往基地。鬼童遵守旧报纸命令而避开刘小雨视线，在杨间周边暗中跟随；杨间将面对能否解除诅咒的最后一场赌局。（第641章）

## 第642章·危险的抉择

杨间以鬼域抵达地图上登记为平安科技研究基地的郊外研究所，先让鬼童潜入楼内窗边待命，自己现身进入基地。卫景在门口接待，双方互相质疑对方危险性后，卫景带他经过无电梯、可逐层封闭的地下楼梯，抵达第五层。王小明解释方案：杨间意识进入郭凡灵位，由郭凡意识暂时取代身体；八音盒诅咒理论上不会跟入灵位，待爆发后再还回意识。身体同时躺入残缺鬼棺，避免三只鬼复苏。王小明也直言失败时杨间会死、郭凡可能获得身体，自己只保证最大化保存杨间这具三鬼平衡的战力。杨间认为方案可能暗藏人皮纸陷阱，却已无其他选择，只能在有限时间内决定是否接受。（第642章）

## 第643章·诅咒继承

杨间质问王小明会否借机永久囚禁自己意识，王小明承认郭凡只是替代者、却不提供更多保障。杨间为防王小明、卫景或外界在赌局中抽身，向鬼童下达新令，要求杀死所有进出实验大楼者；鬼童穿寿衣、抱死人头在楼内游荡，封锁命令覆盖其原先待命状态。杨间随后打开八音盒、走进隔绝房，躺入鬼棺并捧起灵位。鬼棺压住其体内厉鬼却未令其死机，灵位则抽离意识：照片逐渐换成杨间，郭凡照片消失，郭凡意识出现在杨间身体中。郭凡误以为自己获救，迫不及待想拥有三鬼平衡身体，却听见反复音乐。王小明由此确认八音盒诅咒已继承给郭凡，杨间意识暂避灵位。（第643章）

## 第644章·滚动的异物

王小明实验期间，陈博士担忧多个不稳定因素，命助手整理资料后撤离并放假三天。刘助理与同事在走道讨论研究风险时，先察觉昏暗拐角有东西经过，又听到沉闷滚动声。两人准备联系安保科时，一颗带黑发、惨白腐烂的死人头从拐角滚来，还转向他们，灯光随之闪烁。二人逃跑，随后见到身穿脏旧寿衣、皮肤青黑的鬼童捡起人头；刘助理回头时吓得跌倒流血。鬼童因被他们看见而躲入视野外，没有继续追击；死人头也未正面盯住刘助理，使二人暂时幸存。鬼童仍在楼内抱头游荡，刘助理决定通知陈博士和王小明，封楼命令已制造新的灵异恐慌。（第644章）
