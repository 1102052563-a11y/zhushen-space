# 第585-589章剧情梗概

## 第585章·猜测

黄子雅担心杨间只剩半个头颅、身体随时会失控，杨间却从袭击只发生两次这一点反推：若是厉鬼按规律杀人，不会在两次后停止；更像某名驭鬼者先斩首、发现他未死后再补刀，并在确认尸体状态后离开。他意识到此人能无视四层鬼域和体内厉鬼，曾考虑取凯撒大酒店持柴刀尸体的诡异柴刀作反制，但决定先查清真相。黄子雅黑发复苏后下肢瘫痪，杨间将她抱回平安酒店，承认她救了自己，却未说明八音盒与自身死而复动的全部缘由。二人都面对复苏和死亡的压力，杨间则准备动用高志强遗留的鬼处理身体。（第585章）

## 第586章·制造诡异之物

回到房间后，杨间以鬼手压住黄金箱内高志强遗留的鬼，令其对周围身体状态产生影响：自己被鬼剪刀毁坏、但未受厉鬼侵蚀的头部和身体重新完整，黄子雅也恢复知觉和健康外观；鬼手未恢复，显示效果并非万能。为让黄子雅自行压制黑发复苏，杨间以第五层鬼域将模糊人形扭成水晶内的黑点，制成水滴形挂件。黄子雅握住挂件反复自我暗示后，黑发果然停止生长，但黑点随即变大，说明使用越多、里面的鬼越快复苏。杨间将挂件及剩余大半红色鬼烛暂交她保管，黄子雅带着短暂活下去的希望离开。（第586章）

## 第587章·杨间的复苏？

杨间维持四层鬼域和死亡假象，回到遇袭街道拾起卫星定位手机与八音盒。他发现总部已发出自己疑似死于灵异事件的通报，又切断手机信号、封锁附近街区，由此判断自己已被作为死亡驭鬼者及潜在复苏源处理。为暗中调查，他决定先让各方继续以为自己是死人，继而睁开九只鬼眼，以三片三层鬼域迅速覆盖大半市区。李军、张雷、曹洋及平安大厦的朋友圈成员均认出红光，却只当鬼眼已经全面复苏。杨间没有公开现身，而是以异常覆盖为掩护，把调查目标指向总部的刘小雨。（第587章）

## 第588章·惊怒

大范围红光被总部认定为杨间厉鬼复苏，曹延华因此前低估杨间的鬼而惊怒；沈良又报告驭鬼者和朋友圈可能趁机撤离，鬼画危机本已脆弱的协作局面更趋崩溃。曹延华一度命请秦老前往平安大厦，强令朋友圈不得离开并承担后果，却很快发现鬼域已经把总部切开：李军电话中称自己站在办公室，曹延华和沈良却完全看不见他。无法理解局势的曹延华只得暂缓调动秦老，决定转交王小明制定方案，必要时允许动用饿死鬼棺材钉。与此同时，杨间在高处完成分层布局，第一层隔开普通人、第二层困住驭鬼者、第三层供自己潜行，整座城市暂被其接管。（第588章）

## 第589章·杨间下线

杨间在第三层鬼域中发现城内安全屋、总部及若干无法侵入的空白区域，却无暇深查，转而潜入总部宿舍寻找刘小雨。刘小雨仍为他的死亡和通讯室关闭难过，见到恢复外观的杨间时一度以为是鬼；在杨间追问下，她交代朋友圈因期限要求杀害杨间，曹延华原想追责，却被其他人以鬼画行动和队长安全为由劝止，部长最终决定将暗杀伪装成未知灵异事件。杨间理解总部的取舍，却认为规矩不应因死者是自己而失效。他告诉刘小雨“鬼眼杨间”此后下线，随即将她打昏后离开。其是否辞职、如何报复朋友圈和八音盒诅咒还能维持多久均未写明。（第589章）
