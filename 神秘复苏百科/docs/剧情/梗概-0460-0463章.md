# 原文第460-463章梗概

> 原文在本段连续出现两个“第四百六十二章”标题。以下按原文标题出现顺序记录，不将其擅改为连续章号。

## 第460章·汇合的凶险

杨间持白色鬼烛带张雷、王江、黄子雅等人向钱毅靠近，鬼差与队内逃出的魂瓶鬼同时随行，队伍只能以数量平衡暂保性命。资源几近耗尽时，人皮纸改写未来：钱毅一旦靠近，鬼差会舍弃烛火改杀落单者，鬼数逆转后全员皆死；但接触也是唯一机会，需取走鬼差身上的棺材钉、削减其数量，不能关押。杨间据此决定冒险，向同伴说明钱毅遇袭时自己会肢解厉鬼。黄子雅却报告定位已至五十米，钱毅也正主动接近；鬼差的高大尸影随即扑向前方人影，杨间只能带队冲入黑暗。（第460章）

## 第461章·可怕的接触

钱毅不知会合即是死关，只因基地无法走出、独处更危险，硬着头皮循定位寻找杨间，也担心对方追究自己先前背约。信号重合后，他闻到尸臭、手机屏幕被僵黑手掌遮住，随即被老旧草绳吊起；厉鬼力量沉寂，黑手又撕开皮肉搜寻体内之鬼。杨间携鬼烛赶到，见鬼差仍是卫景尸体的腐烂血污外形，立即带同伴冲近。钱毅被重新纳入队伍范围后，鬼差停止杀戮，虽已将其身体扭曲流血，尚未夺走其鬼。杨间确认规则切换只给一次出手机会，准备趁鬼停顿削减其能力。（第461章）

## 原文首个第462章·鬼回头

杨间不能关押鬼差，否则它会舍弃卫景身份重现；他看见王小明制成的黑色棺材钉已融成鬼差肩至心口的痕迹，仍以鬼手和无头鬼影侵入，欲把带钉部分肢解。碰撞使局部黑暗暂退，众人终于看清钱毅被吊与鬼差尸身，但杨间手臂、鬼影立刻遭压制，无法抽离。鬼差随即向后转头，杨间的颈椎也被牵动扭曲。张雷认出这是赵昊遗留的鬼回头，主动上前准备承受袭击；杨间被困在原地，只能依赖同伴能否挡住这次反击。（首个第462章·鬼回头）

## 原文重复第462章·食鬼之人

鬼回头继续扭动时，杨间宁可不舍手逃命，也不愿失去唯一削弱鬼差的机会。张雷撕开衣服，显出胸口腐烂人头和正在生成躯干的轮廓，抱住鬼差后令死人头张口吞住其半个头颅；鬼头若再转动便会被吞食。张雷承受全身如遭碾碎的疼痛，胸前厉鬼也逐渐沉寂，却终于迫使回头停止，杨间的身体恢复。张雷由此察觉鬼差夺鬼后会获能力，却会在压制其他鬼时失去自身能力，随即准备以数量让其彻底失效；这条看似可行的生路尚未经过杨间判断。（重复第462章·食鬼之人）

## 第463章·最后一种能力

张雷提出王江、黄子雅等人共同接触鬼差，以人数压掉其全部能力；杨间立即制止，说明完整压制会触发鬼差放弃卫景尸体、换身份发动第二波袭击，且钱毅仍在压制范围，人数也被算错。他只让王江接触，使鬼差在压住八只鬼后保留一项能力。天空随即垂下无数老旧鬼绳，绳套准确寻找众人脖颈；王江不能松手，黄子雅和未具名同伴又难久撑，队伍陷入更短的死局。杨间趁自身仍能行动，将染血报纸盖上鬼差面部，企图改写记忆，但鬼绳已逼近，结果未明。（第463章）
