# 第331-335章梗概

## 第331章·进入中山市

杨间乘低调安排的直升机进入中山市，在机上听完童倩失联前录音，确认她循城南最初的人头线索到凯撒大酒店调查后失踪。他命刘小雨转告赵建国，让冯全和郭凡远离酒店，理由是记忆可能遭篡改者本人无法察觉；总部虽以通话分析判断二人遇袭可能低于一成，杨间仍拒绝冒险。飞机避开上空漂浮的人头降至在建公园临时指挥区，杨间目测全市至少有一两百颗人头。中山市联络员张高接应并确认熊文文已到，杨间决定立即前往酒店。（第331章）

## 第332章·出乎意料

张高带杨间见到正在打游戏的熊文文，并转达总部尽量保障灵童刑警撤离的要求；杨间只承诺尽力，要求备皮卡、封锁酒店及周边、继续核查空中人头身份。杨间向熊文文说明可在危险时用鬼域送他离城，又警告未经自己允许不可随意预知，以免被假危险诱导厉鬼复苏。两人抵达已封锁的凯撒大酒店，郭凡和冯全却已在门口等候，违背了杨间希望二人远离的安排。郭凡指责杨间独揽功劳，杨间则因二人突然出现而更怀疑其记忆可能已被鬼改写。（第332章）

## 原文第334章·酒店的前台

郭凡、冯全拒绝撤离，冯全提议进入酒店后各自调查、不正面接触，杨间为避免同驭鬼者内斗暂时接受。熊文文也因二人毫无准备进入闹鬼酒店而判断其可能出事；杨间嘱咐他非必要不动用预知。酒店大厅灯火通明、奢华如常，与已封锁停业的状态矛盾，一名女前台机械微笑接待众人。郭凡欲开枪试探，冯全以黄岗村的教训阻止，认为未知规律下不能先破坏平衡。杨间见前台暂未攻击，绕开郭、冯，注意到通往二楼红地毯上留有带尸臭的黑脚印，转而带熊文文上楼。（原文第334章·酒店的前台）

## 原文第334章·酒店的二楼

杨间沿脚印确认其通向被封死的二楼，判断数日前已有一只鬼进入酒店，可能关联赵磊、童倩或人头气球事件。他开枪破开玻璃门，带熊文文进入无灯走廊，以额头、后脑鬼眼戒备，并准备以鬼域、鬼烛、鬼绳和装尸袋快速关押目标。走廊尽头传出刀叉碰瓷盘的用餐声，二人抵达封锁餐厅外；杨间不再等待，发动三层鬼域直扑声源，企图在不试探规律的前提下直接收容。这次突袭却在声音中止后失败，餐厅内的实际目标仍未被看见。（原文第334章·酒店的二楼）

## 第335章·黑色的烛火

鬼域撤开后，杨间只见餐桌上有新鲜白玫瑰、装着带热血人脸的瓷盘和染血刀叉，说明某种残酷进食刚发生，却能在他发动鬼域的一秒内消失。他据此前遭遇推测篡改记忆的鬼没有固定实体，只能借他人身份存在；但赵磊已死，无法补足情报。熊文文主张先撤，杨间仍决定确认目标是否在附近，取出王小明留下的白色鬼烛。熊文文因白烛会引鬼而极力阻止，杨间权衡后点燃三十秒；黑色烛光立刻侵蚀周围鬼域，如黑雾般连向未知区域，后果尚未揭示。（第335章）
