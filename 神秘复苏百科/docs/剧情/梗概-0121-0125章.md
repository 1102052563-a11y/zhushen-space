# 第121-125章梗概

## 第121章·第一次见面

江艳精心打扮到炸鸡店见杨间，仍被他称作“大姐”。钱枫循车找到店内，江艳称杨间是男友；杨间却说二人只是雇佣关系。钱枫轻视杨间衣着和职业，杨间以不可能上班、靠敲诈勒索生活回应。双方围绕江艳关系争执，钱枫暂离接电话，江艳仍未摆脱其纠缠。（第121章）

## 第122章·骚动

钱枫回桌继续贬低杨间，杨间将黄金手枪、子弹摆上桌；钱枫确认并非玩具后立刻改口奉承，否认租住江艳对门。杨间此时感到鬼眼被高楼异常刺激，判断附近发生灵异事件，带江艳离店。钱枫因误以为杨间要杀自己跪地求饶，未再阻拦。（第122章）

## 第123章·偶遇事件

市中心写字楼十五层人员砸窗、跳楼或被拖回昏暗楼层。杨间观察到青黑阴霾，判断厉鬼已能影响环境、接近鬼域，十五层人员难以生还；他不愿贸然介入，只记录事件并意识到灵异事件数量正在增加。警车赶到时，跳楼死亡仍持续。（第123章）

## 第124章·熟悉的鬼

刘建明到场疏散群众，请杨间协助，杨间以新任负责人赵开明在位为由拒绝。杨间看见十五层青黑皮肤、黑眼无瞳孔的人影，认出是从周正腹中脱出的鬼婴，且它已由婴儿、鬼孩成长为青年形态，鬼域雏形也更强。杨间决定带江艳离开，却有防爆车直冲自己。（第124章）

## 第125章·交锋

防爆车急停后，赵开明现身，以撞车玩笑和火锅邀约试探杨间。杨间不接受道歉，借黄岗村“土特产”发动无头鬼影，取下赵开明一条腿，再将其原话反还。赵开明意识到杨间可能已驾驭第二鬼，仍以大昌市事务威胁；杨间警告其先处理鬼婴事件，随后带江艳离开，双方冲突公开化。（第125章）
