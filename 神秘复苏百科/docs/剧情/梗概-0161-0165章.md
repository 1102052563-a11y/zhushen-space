# 第161-165章梗概

## 第161章·鬼敲门事件重演

张显贵用镜子确认别墅有鬼后，杨间判断镜中鬼跟着张伟而非藏在房内，追查重点转向张伟首次见鬼的观江花园工地。张显贵称近期停工的工地有人失踪，镜中人影又穿工服，愿出价请杨间调查，三人随即离开别墅。与此同时，赵开明在玫瑰酒吧逼王岳加入对付杨间的联手：他以黄金、商业街利诱，又开枪杀死一名女子，用查封和通缉相威胁，王岳最终屈服。交易刚成，酒吧包间灯灭、墙壁腐烂，街道被黑暗吞没，黑长衫人影在门外敲门。赵开明认出是A灾难级鬼敲门事件，怀疑杨间借其反制自己，但鬼是否真的由杨间操控尚不能确认。（第161章）

## 第162章·观江小区

杨间、张显贵和张伟抵达依江而建的观江花园。楼盘规模庞大、已大部竣工，张显贵原想建成富人别墅区，却因工人失踪而停工停售。杨间看中其远离高密度人群、交通便利的条件，提出以五千万元购买五层售楼部；张显贵本想相赠，杨间坚持把救张伟和买卖分开。两人进一步谈到项目另有钱总、秦总两名开发商，若灵异事件和杨间入住能使房产升值，便可趁对方恐慌低价收购份额。杨间索要全小区四成收益，张显贵虽认为过高，仍先达成合作意向，条件是杨间证明自身能力。张伟只顾提出低价送房、同住等建议，未提供新的遇鬼线索。（第162章）

## 第163章·事实说话

售楼部内，张显贵得知钱总、秦总已在三楼会客，便带杨间、张伟上楼商谈。两名合作开发商只关心恢复销售，不信工人失踪与灵异有关；钱总还以烟头轻慢杨间，秦总把其“国际刑警”身份当成玩笑。张显贵不替杨间出面，既让其自行处理，也借机检验合作价值。杨间说明售楼部已有鬼，二人仍不接受，遂以额头睁开的鬼眼和扩散的红光回应。钱、秦看见异眼后神色改变，却还未完全相信；本章止于鬼域笼罩会客区域。杨间选择动用会加重风险的厉鬼能力，不是为立即调查工地源头，而是为了用可见证据逼两名开发商面对异常，收购谈判由此转入强制展示阶段。（第163章）

## 第164章·好戏上演

鬼域中，钱总、秦总分别看见沙发、墙面、天花板与茶杯充满新鲜血迹，惊惧逃离会客室；张显贵和张伟却未见同样景象。杨间承认售楼部只有跟随张伟的鬼，血迹是自己额外制造的幻觉，目的在于让两人产生足够恐惧。为证明能力，他又使桌上杯子消失后出现在张显贵手中，并展示黄金手枪，说明鬼域能制造或改变感知，但对付人的真正手段不必依赖鬼域。张显贵因而相信杨间具备未来接任大昌市刑警的价值，回去准备收购合同。杨间则继续集中精力操控鬼域，让逃走的钱、秦两人经历一场“恐怖电影”，只打算令其破财，不直接取命。（第164章）

## 第165章·趁火打劫

钱总、秦总带保镖乘电梯离开售楼部，仍把手上血迹和工人失踪当成撞鬼证据。鬼域随即使电梯楼层不断降到不存在的地下十层，门外只有吞没光线的黑暗；手机光无法照远，远处脚步靠近。保镖把手机丢出后，众人只看见垫脚站立、惨白枯瘦的一双脚，又见其走入黑暗，终于确信遭遇鬼物。报警救援至少需半小时，电梯按钮也无法返回，他们便转向唯一知道异常的杨间与张显贵求救。张显贵接通电话后没有立刻施救，而是提出打包收购两人楼盘。秦总看穿这是趁火打劫并怒骂，章节至此结束；收购是否成功、鬼域何时解除和工地真正源头均未写明。（第165章）
