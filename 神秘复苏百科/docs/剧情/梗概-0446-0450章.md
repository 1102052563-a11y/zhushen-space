# 第446-450章梗概

## 第446章·与厉鬼比赛

杨间已确认正面关押鬼差只会重演鬼棺事件的后续袭击，决定先活下去并逃离封闭空间。他带着只剩两只鬼的状态靠近宿舍楼，鬼眼在黑暗里只能看清五米，判断鬼差不能同时分身，便要赶在其杀人夺鬼前找到其他受训者。杨间先在楼外发现迷失的黄子雅，让她搭着自己的肩膀、用卫星电话与总部通话确认幸存者位置；到楼下又看见尚未凝固的血和尸体碎片，计算鬼差至少已能压制五只鬼，自己与黄子雅合计仍只有三只。为保存仅剩鬼烛，他不立刻点燃，只进入宿舍楼继续搜寻。双方由此变成一场集结速度与鬼差成长速度的竞赛。（第446章）

## 第447章·摸黑前进

杨间逐间踢开宿舍门，找到一名抱腿躲在角落的未具名驭鬼者。对方原想靠静止躲鬼，杨间解释鬼差专挑落单者，强行带其与黄子雅同行；黄子雅则从总部得知钱毅正由另一通道下楼，杨间决定前往会合。与此同时，四楼的张雷、王江和两名未具名驭鬼者已先聚成一队。王江担心不少新人没接到总部的第二条规律通知，张雷却主张继续搜人，避免出楼后更难集结。总部传来杨间仍在楼内的消息，众人正要赴安全通道会合，张雷却在墙上摸到冰冷僵硬的手，确认鬼差就在前方，四人随即掉头逃跑。（第447章）

## 第448章·错误的选择

张雷一行在黑暗中因撞见鬼差而分散。张雷逃跑、撞墙后才重新计算：自己体内实际有三只鬼，四人合计六只鬼，鬼差当时只能压制五只，他们原本即使正面遇鬼也未处于落单状态。恐惧使他先喊逃跑，亲手拆散本可安全同行的队伍；他只能改往安全通道寻找杨间。另一名未具名驭鬼者则独自迷失在阳台，鬼差从黑暗里抓住其双臂、压制体内厉鬼并从身体取走某物，受害者重伤后死亡，鬼差可压制的鬼数升至六。杨间在二楼看到从四楼滴下的新鲜血，确认又有驭鬼者死亡，必须更快集结。（第448章）

## 第449章·失败的汇合

杨间在楼梯口等待钱毅，却只得到钱毅关闭通讯的消息。总部定位显示钱毅正朝基地出口移动；杨间判断他故意以汇合为名拖住其他人、独自逃离，但也指出鬼棺空间内单独向外并无真正生路。杨间放弃等待，改去三楼寻找原称有四人的张雷一行，在楼道截住独自下行的王江。王江说明队伍在四楼遇鬼后随张雷逃散，并不确定是否有人已被袭击。杨间拒绝继续下楼避险，要求众人折返四楼搜张雷和其余两人，认为逃跑只会让鬼差继续夺鬼成长。王江和余下同伴接受这一高风险方案，队伍带着仍不足以抗衡鬼差的鬼数重新上楼。（第449章）

## 第450章·黑暗中的人

杨间带王江、黄子雅等人进入鬼差所在的四楼，命令所有人抓紧彼此、未经指令不得逃跑。他已因第四层鬼域逼近复苏极限，不能再开鬼域，只以五米鬼眼视野沿走廊搜索。众人先发现一具体内厉鬼已被取走的残破尸体，随后近处又响起一名幸存者的惨叫并骤停，意味着鬼差再次夺鬼、压制数量升至七。杨间依据惨叫位置避开鬼差，赌另一名幸存者会从北侧安全通道撤离，带队加速搜寻。前方终于出现人形轮廓，他尚不能判定是人还是鬼，只能握住鬼烛缓慢靠近；此时黄子雅接到总部消息，确认张雷仍在四楼，且其体内实际有三只鬼。杨间尚未辨明前方目标，事件在新的接触前停住。（第450章）
