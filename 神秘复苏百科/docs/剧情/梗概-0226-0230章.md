# 第226-230章梗概

## 第226章·饿死鬼事件

王小明说明人体内鬼婴平时无实体，普通剖腹无法取得；杨间交出已钉住的小鬼婴，却被判定样本太小。王小明提出优先捕获第三阶段青年鬼婴、理想情况则是第四形态，并将事件正式命名为“饿死鬼事件”。杨间以鬼婴、干枯手指和黄金盒的研究使用权，换取大昌市银行余下约两吨黄金；九名驭鬼者随即按三人一队分区巡查。（第226章）

## 第227章·异样

杨间、张韩、孙义负责东城区，巡查中未发现鬼婴。张韩说明驾驭第二只鬼延长复苏，本质也会限制厉鬼能力成长，令杨间想到人皮纸主动促成复苏、再设法控制的另一条路径。西城区的赵开明、叶枫、何川则进入异常安静区域，后路消失；赵开明提议叶枫用鬼牙逃离，叶枫拒绝协助其逃跑并表示会先按王小明安排行动。（第227章）

## 第228章·消失的路

东城区三人仍无收获，杨间遇到借阴霾囤货抬价的小卖部老板，未继续追究后回车巡查。西城区里何川突然消失，赵开明也在叶枫眼前不见；叶枫肩膀留下穿透寿衣的青黑手印，听到鬼婴啼哭后进入凭空出现的小巷，试图寻路离开。（第228章）

## 第229章·无法抵挡的恐怖

叶枫在小巷发现鬼牙无法咬开环境，确认自己并非困在普通鬼域；随后见到被啃食的何川头颅和第二阶段鬼婴。店铺陆续走出大量不同形态的青黑鬼婴，寿衣只能短暂抵挡不断增加的手掌袭击。叶枫被鬼群围死前发出一条求救短信，之后与鬼影一同消失，只留被踩碎的手机。（第229章）

## 第230章·安全屋

杨间结束东城区巡查后，银行人员将约两吨黄金送到别墅。张显贵已囤积物资、改造地下避难所；杨间提出在其中以黄金打造密闭安全屋，双方约定七三权益和施工分工，并要求每日检查小区住户腹部异常。夜间刘小雨通报叶枫、何川失踪，赵开明称其遭遇灵异事件，王小明将再次召开会议。（第230章）
