# 第201-205章梗概

## 第201章·我在这边等你

叶枫穿寿衣硬闯鬼绳，杨间则以鬼烛和鬼域幻象不断换位，让他耗尽力气仍无法接近。鬼域外，王小强重新套上被撕下的脸皮后复活，却遇见从鬼镜放出的、无眼无口鼻的鬼女孩；其受货车鸣笛吸引，沿路离开观江小区，王小强因此暂避一劫。叶枫终于走到原位置时才发现杨间只是幻象，伏杀行动已彻底失去主动。（第201章）

## 第202章·叶枫的第二只鬼

叶枫承认无法在消耗中杀死杨间，提出罢手并退出小强俱乐部；杨间拒绝和解。鬼绳首次套住叶枫，寿衣令其一只手尸化，他随即揭开寿衣露出胸腹游走的鬼牙。鬼牙会咬掉被寿衣尸化的血肉，也能强行咬开杨间鬼域，代价是使用者身体如被撕裂。叶枫借此打开缺口，准备离开鬼域。（第202章）

## 第203章·平手

杨间确认鬼牙的攻击性后，趁叶枫破域，持鬼烛离开鬼域取回干枯手指；他开枪击倒王小强，却无法阻止其再度借体内鬼复活。杨间让无头鬼影拔出手指，鬼域恢复自由；叶枫也从缺口出来。双方认定此战没有赢家：叶枫暴露两鬼并付出复苏代价，杨间烧掉近半截鬼烛。叶枫再提停战，杨间只允许其离开，并宣告日后不死不休。（第203章）

## 第204章·保大保小

叶枫和王小强撤退后，杨间在鬼域内凭预留金线找到真正鬼绳，将其重新装回黄金袋关押。随后他向刘队报备持枪人员死亡，再向赵建国说明王小强、叶枫暗杀自己的经过。赵建国确认暗杀国际刑警的驭鬼者应按反人类罪、优先于灵异事件处理；杨间据此要求官方不要包庇王小强。赵建国最终决定取缔小强俱乐部、派孙义协助杨间正式任职，却仍想由部门先处理王小强以避免杨间私下杀人。（第204章）

## 第205章·孙义

国际刑警文职孙义在相亲餐厅接到赴大昌市的紧急命令。他一面把工作自嘲为“挖掘人才”，一面在相亲对象面前夸大未来生活；对方离开后，孙义发现三万八千元餐费无力支付，网络贷款也因信用过低失败。赵建国电话催促其半小时内集合，并在得知餐费后表示会调查是否受骗、再考虑报销。孙义尚未启程抵达大昌市，任职协助仍停留在调度阶段。（第205章）
