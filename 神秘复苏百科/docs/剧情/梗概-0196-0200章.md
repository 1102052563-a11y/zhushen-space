# 第196-200章梗概

## 第196章·从鬼镜当中走出来的人

观江花园内的枪声惊动杨间，鬼镜随即发生异变：黑布隆起，镜内张伟被多只腐烂手掌拖向深处。杨间确认对方手臂有活人温度后抓住不放，却连鬼眼鬼域也无法影响抓住张伟脚踝的一只血手；他冒险将无头鬼影投入镜内，才令手掌松开、把张伟拉回现实。镜子恢复原状，而张伟只记得自己上厕所时突然进入黑暗、遇到鬼。人皮纸解释他已死后由鬼镜复活，复活者是第一次照镜时的张伟，记忆未包含刚才遇刺，且一只镜内鬼因此获释。杨间未继续验证代价，命张伟避开镜子躲藏，自己循枪声出门，判断有人正在小区内针对自己行动。（第196章）

## 第197章·诡异的手指

孔峰的行动组在河对岸布置多处狙击点，探测确认杨间离开别墅后先行射击；杨间因救张伟时仍维持鬼域而避过子弹，直接转至观测员与狙击手身后，以鬼眼将二人肢解。补枪和快枪手偷袭均不能改变结果，他用鬼域幻象诱敌、反手射杀快枪手，再把孔峰、黛西等人困进红光范围逐一清除。王小强见普通人员失败，取出金箱内干枯手指刺入地面，红光鬼域被钉住、拖拽缩小。叶枫随即现身，以谈话拖延限制生效；杨间不接受招揽，转至王小强面前，逼其撕开头皮、露出皮下腐烂人头。叶枫阻止王小强放鬼并抓住无头鬼影，伏杀转为驭鬼者正面冲突。（第197章）

## 第198章·鬼衣

叶枫承认自己已驾驭两只鬼，试图以国际刑警只是卖命、俱乐部才可抱团的说法劝杨间让步；杨间则指出对方杀害张伟、侵入自己地盘，拒绝妥协。干枯手指持续把鬼域拉向远处，叶枫以此拖延，杨间却仍转移到王小强面前，令鬼影扯下其整张人皮脸，想让皮下鬼物脱困。叶枫及时抓住鬼影，鬼影无法侵占他的身体；他脱去大衣，露出染血、写满福字的寿衣，说明自己是“鬼衣叶枫”。杨间认出寿衣能隔绝其他厉鬼，却判断它必有反噬，且叶枫寿衣下还藏着滴血蠕动的第二只鬼。鬼影和鬼域暂未能取胜，双方情报与能力差距让杨间转而寻求别墅中的鬼绳。（第198章）

## 第199章·复苏的鬼绳

叶枫凭鬼衣抵住鬼影后逼近，寿衣下不断滴出黑血；杨间判断对方第二只鬼尚未亮明，且被手指钉住的鬼域无法发挥原本优势。他转回别墅取出黄金袋，拿出此前从王岳处关押的鬼绳，决定只在鬼域内释放已经完全复苏的厉鬼。草绳被叶枫接住后，鬼域内立刻从头顶、前后及四周垂下大量绳索，叶枫与杨间都成了索命目标。叶枫的鬼衣暂时弹开绳套，认为杨间会先死；杨间则提醒寿衣并非保护，而是在持续把叶枫变成尸体。双方都无法轻易离开鬼域，杨间以复苏鬼绳把战斗改成谁先被厉鬼与复苏代价拖垮的消耗战。（第199章）

## 第200章·互相消耗

被手指压缩后的鬼域仅余一条狭窄道路，杨间和叶枫隔着百余米、面对无差别飘落的鬼绳。鬼衣使叶枫暂时不受绳索袭击，但他的手掌已开始失血、僵硬；他承认寿衣穿久会将人变成尸体，只能靠第二只鬼延缓。杨间取出余量约三分之二的鬼烛点燃，绿火把靠近的鬼绳全部荡开，并估计可维持一小时。鬼烛使叶枫的消耗优势逆转，叶枫仍不愿等死，强行挤开密集绳索向杨间靠近。第200章结束时，叶枫尚未接近烛火范围，王小强和干枯手指去向不明，鬼绳也仍在鬼域中持续复苏，胜负与后续收容均未揭晓。（第200章）
