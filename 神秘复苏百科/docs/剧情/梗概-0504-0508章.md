# 第504-508章剧情梗概

## 第504章·保守的选择

朋友圈讨论杨间威胁。贺天雄提议绑架其母诱杀，被费总否决；费总认为伤害亲属无法消除杨间本人，还会招来其鬼域报复。姜尚白也顾忌杨间可能厉鬼复苏，届时或成S级事件。组织因队长计划和生存资源压力暂不掀桌，决定先令李瑶以条件招揽，失败后再交董事会决定。杨间同时整理物品库资料，排除有失灵风险的鬼相机、缺失鬼火的人皮灯笼和危险的死人幡，在八音盒和裹尸布间权衡。他认为八音盒虽可长期保命却带必死诅咒，最终更偏向能限制凯撒酒店柴刀鬼、可替代鬼绳的裹尸布；物品尚未取回。（第504章）

## 第505章·李瑶的情报

万德路来电称魂瓶调查有收获，约杨间在平安酒店会面。李瑶则先到，以朋友圈派她公关、又须掩盖与杨间秘密关系为由进入房间。她透露朋友圈已连夜讨论对付杨间，曾有驭鬼者提出直接打掉他，但姜尚白认为动手会造成惨重代价，组织暂未采纳。杨间并不因这一克制放心，认为对方只是知晓自己也有掀桌能力，若他较弱早已遭袭。李瑶确认组织另有招揽打算；杨间因她能迅速取得总部机密般的内部消息，进一步意识到其作为情报渠道的价值，准备听取具体条件并继续利用她监控朋友圈。（第505章）

## 第506章·带话

李瑶转达朋友圈两百亿元招揽条件，并试图以势力依靠和避免冲突劝杨间接受。杨间认为驭鬼者只能依靠自身，加入朋友圈会失去总部资源和个人功劳分配，金钱无法弥补关系生死的损失；他还判断组织可借招揽使其失去队长资格，待局势稳定后再排挤自己。故他要求成为队长后才考虑加入，并索要朋友圈三成股份，同时警告若为高志强报仇便尽管来。李瑶答应带话，也接受持续留意组织动向、整理详细成员资料、发现行动立即通知杨间的任务。万德路恰在此时到访，李瑶为避免暴露双方关系离开。（第506章）

## 第507章·五年前的照片

万德路带来调查档案：魂瓶等古董来自中安市外陵镇一座罗姓民国老宅的地下室，后人准备推倒旧宅重建时将其挖出并售入古玩街。照片中的老宅青砖无窗、木门高窄且年久失修；五年前游客拍摄时，门顶缝隙伸出一只模糊手掌。杨间无法确认手是否为魂瓶鬼，却发现旧宅拆除时没有死亡，而疑似鬼物五年前已出现又未扩散，与魂瓶关押状态相互矛盾。万德路还查到另一只疑似成对魂瓶仍在古玩街，但街区已紧急疏散、封锁停业。杨间据未关店门判断发生灵异事件，决定会见被请来大京市的原宅户主罗永，寻找民国线索。（第507章）

## 第508章·张辉的死

杨间与万德路离开平安酒店前，遇到张雷、王江和黄子雅接受总部询问，得知培训基地幸存驭鬼者张辉于早晨约六点死在外部小宾馆，尸体刚被发现不久。陈义已接管现场并处理张辉体内的鬼；因只死一人、没有灵异事件扩散，众人不排除他杀，但死因未定。杨间认为此事特殊却未介入，继续随万德路出发。车上，万德路安排女明星作陪，杨间拒绝并警告大京市可能因鬼差与鬼画在城郊碰撞而不太平。他建议万德路尽快离开、迁往自己所在的大昌市；万德路接受判断，计划迁走公司总部、家人与亲属。（第508章）
