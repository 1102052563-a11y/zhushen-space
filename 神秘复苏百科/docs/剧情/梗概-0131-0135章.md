# 第131-135章梗概

## 第131章·有问题

杨间以第五副碗筷为由提议离开饭店，之后向众人说明包厢原有四人却多出一副使用过的餐具，担心存在鬼物。张伟等人相信学校经历，上官云仍以科学为由嘲笑。众人转往KTV，饭店关门时，第五副碗筷位置出现一名坐在暗处的人影，确认异常并未消失。（第131章）

## 第132章·混进来的一人

KTV内，王珊珊承认靠近杨间会舒服、远离时身体异常，杨间判断鬼奴印已侵入精神。赵磊追问驭鬼者，杨间说明这是驾驭厉鬼、以命换力量的人，不建议普通人追求。服务员按人数送来七瓶啤酒，实际在场仅六人，杨间确认饭店的第五人一直随队而来。（第132章）

## 第133章·看不见的鬼

杨间推断多出的人更可能跟随张伟或赵磊，暂不激发冲突，先用鬼眼搜索却未见异常。鬼眼开启时，他发现自己可借王珊珊手腕烙印直接操控其身体。服务员坚称包厢有七人、其中两女，说明普通人能看见该鬼；杨间据此确认多出者为男性，但尚不知其显现条件。（第133章）

## 第134章·阿飞哥

阿飞等混混因张伟唱歌闯入勒索，误伤穿名牌的上官云。杨间回来后被曾遭其鬼域戏弄的混混认出，阿飞立刻求饶。杨间把黄金枪交上官云鼓励报复，但苗小善阻止；上官云最终只要道歉，混混离开。杨间仍未查明第七人。（第134章）

## 第135章·镜中人

杨间向众人说明服务员确实看见第七人，建议先确定鬼跟随对象再撤离。上官云去厕所洗鼻血时，镜面中出现现实不存在的人影；他获得纸巾后抬头，镜中自己脸色惨白、动作开始不同步。镜中异常人影逼近，其危险已直接落到上官云身上。（第135章）
