# 原文回跳第775-779章剧情梗概

## 原文回跳第775章·熊文文的存在

杨间的鬼眼复苏暂被鬼手和鬼影压下，眼睛却仍猩红、失去正常触觉。他无法停止调查鬼邮局，也不能回避鬼橱交易，开始考虑驾驭第四只鬼。返回尚通大厦后，他得知战胜叶真引来多笔巨额投资，又见到专程来大昌市的陈淑美。杨间带她查看熊文文照片和专属纸人，说明儿子可能从照片转入纸人，也可能因放出厉鬼而彻底失败；陈淑美在恐惧中请求救子，杨间暂让她自行决定。（原文回跳第775章）

## 原文回跳第776章·七人队

冯全、童倩、李阳、黄子雅、张韩齐聚尚通大厦，杨间宣布组队名额只有七个，目前自己与五人共六人。队伍以杨间为首，冯全、童倩依序作为继任队长；成员可入住观江小区、使用安全屋，并得到不可转卖、可继承分红的公司股权。杨间强调队伍不是只为处理事件，更要让成员死后家属仍有保障。张雷和熊文文都被列为最后空位候选，但尚未加入。（原文回跳第776章）

## 原文回跳第777章·诡怪

组队会议后，冯全提出追查裴东的摆钟诅咒，杨间同意后续可与童倩协作；鬼邮局调查暂仍由杨间、李阳、孙瑞推进。陈淑美最终同意救熊文文，也接受儿子若复活便须加入杨间团队的条件。杨间解释普通活人身体无法承载当时接近复苏的熊文文，只能用柳三纸人。午餐中，木棺传出动静，杨间称里面正在孕育“鬼怪”，与鬼梦有关，队员不应触碰。（原文回跳第777章）

## 原文回跳第778章·复活的纸人

众人转至二号安全屋。杨间命鬼童把熊文文黑白照片贴到纸人额头，凭其吞灵位后保留的能力转移照片内意识与厉鬼。照片中熊文文先后呈现惊恐、阴冷、挣扎，随后褪为白纸并粉碎；纸人开始活动、呼吸般动作，却一时无法说话。冯全和黄子雅以算术试探，纸人分别竖中指、交叉成十字，确认了熊文文式的反应。杨间仍将其封在房内，由张韩看守适应纸人身体。（原文回跳第778章）

## 原文回跳第779章·镜子和狗

杨间带八音盒、旧报纸、替死娃娃、鬼烛与鬼瓷等资源准备下一次行动，并决定把鬼镜公开给团队。童倩接手管理，黄子雅、张韩先后在镜前留影，获得潜在复活机会；杨间说明代价是放出镜内鬼，且镜中影子与本人完全重合会导致活人被替换。鬼镜转向木棺时，呈现出沉睡的黑毛狼狗。杨间推测父亲遗留的鬼梦驾驭计划可能与这条看不见的狗有关，但没有触碰木棺。三日后，熊文文已学会说话，杨间确认其人格后放出，纸人载体和预知鬼的长期平衡仍未知。（原文回跳第779章）
