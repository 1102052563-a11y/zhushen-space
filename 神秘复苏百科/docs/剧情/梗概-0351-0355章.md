# 第351-355章梗概：鬼音寻路与十三号房

## 第351章·凶险降低

杨间从鬼雾中返回第二道警戒线，称自己只在酒店内做了准备，要求熊文文、罗素一和林落梅先到高处等待，若布置生效便立刻行动。酒店广播持续播放敲门鬼音频后，二层餐厅内原本保持僵坐、带诡异笑容的饿死尸体突然全部倒下，笑容消失；原文以敲门鬼诅咒覆盖原有诅咒解释这一变化。二层厨房的门随之打开，一名持缺口锈菜刀的高大黑影拖着腐臭、似带血的脚印走出。四层则显出一条不存在的走廊，摆有郭凡遗照的空白牌位倒地，跪在牌位前的人抬头，十三号与三十一号房又先后开门、关闭。敲门声能否真正带走酒店危险、各处异动者身份及郭凡下落仍不详。（第351章）

## 第352章·鬼音的探索

杨间等人在酒店外楼顶等待时，笼罩酒店的环境突然如入黑夜。杨间判断这是敲门鬼已循广播到来，认为其鬼域会把酒店内大部分危险带入另一空间，反而是进入调查的短暂时机。约十五分钟后鬼雾消失、酒店重新显现，他仍提醒没有被卷入鬼域的存在通常更危险。队伍第三次进入大厅后，杨间不冒险直接寻找十三号或三十一号房，而命林落梅以鬼音搜寻赵磊。林落梅的声音可沿固定方向扩散、穿墙遍历附近区域，听到者可能有致命后果；一层、二层搜索均未得到赵磊回应，厨房内却留有通向三层的黑脚印。熊文文补充自己曾预见二层某鬼会将他和杨间切尸，杨间将预知视作可规避死亡的保险。（第352章）

## 第353章·不存在的路

林落梅的鬼音越过三层进入酒店四层，在正常客房逐间寻找赵磊；其中一扇门缝曾露出窥看的眼睛，但未与声音正面冲突。声音最终抵达一片额外出现、门牌样式不同的走廊，在十三号房得到僵硬的“我在这里”回应，故林落梅确认可带路定位。鬼音撤回时却被一个高大黑影拦住退路，她惊觉四层另有鬼物，杨间因此决定趁敲门鬼鬼域尚未移开危险、立刻带人上楼。到四层后，他以鬼眼和鬼域护住队伍，按林落梅记忆行进，却在应右转继续的地方只看到窗户。杨间判断自己身处鬼域才看不见那条路，遂收回鬼域、改用手电照明；窗户随即消失，原处出现一条不属于酒店的深邃走道。（第353章）

## 第354章·走道里的危机

杨间带罗素一、林落梅和熊文文越过不存在走道的入口。他留一只鬼眼观察，发现两侧老式客房门均紧闭、墙面异常干净，暂未见血迹或尸臭；一至数号房以蛮力都无法打开，故没有为无关房间继续试探。到第九号房附近，走道形成十字口，地毯上散布大量新鲜浅黑脚印，分别延向十三号房和两条岔路，说明与鬼音碰撞的鬼不久前经过，却无法判明去向。众人赶至一扇被取走门牌的门前，杨间取出鬼橱交易所得的铜制“13”号牌比对，发现孔位严丝合缝，确认线索对应此门。他以无头鬼影的拼合能力开门；同时罗素一照见十字口有一名眼睛腐烂、滴尸水的高大男子行来，对方在路口中央停住，危险逼近。（第354章）

## 第355章·死亡预知

罗素一惊呼后，杨间以多只鬼眼确认高大腐臭男子停在十字路口，距离队伍约十米，尚未攻击，故判断众人暂未触发其规律。他不愿同时处理走道鬼和赵磊线索，仍选择进入已打开的十三号房暂避；房内陈旧有霉味，鬼眼初查未见赵磊或厉鬼，却不能排除暗处异常。杨间此时命熊文文发动预知，罗素一催促时被其斥责，不得再给十岁孩子增加压力。预知画面中，众人先查厕所无异常，继而开卧室门，见到多名坐、站在房内且面孔平坦、没有五官的人；随后熊文文自己头颅如被切断般坠落死亡，预知也就此中止。杨间只能确认危机与开门后出现的无脸人有关，具体杀人条件、赵磊是否在房内及如何改写结果均未揭示。（第355章）
