# 第431-435章梗概

## 第431章·新的人格

李瑶在平安大酒店醒来，鬼报纸植入的记忆使她把前夜理解为自己为继续招揽杨间而主动约见、继而因好感与其相恋。杨间没有立刻信任结果，而以仍会拒绝加入、为何向他透露秘密等问题逐项试探。李瑶不但没有察觉记忆断层，反而称若杨间拒绝招揽又成为队长候选，背后势力会毫不犹豫将他打掉，并愿意转为替他做事。原同伴电话质问她未上班、定位又在杨间房间时，她以私下争取杨间、化解双方误会为由应答，称杨间只是不加入任何一方，也要求对方别再制造麻烦；对方暂时接受。杨间确认篡改至少能经受苏醒、问答与电话质询，正式将李瑶视为情报渠道，但仍未判断其长期稳定性。（第431章）

## 第432章·朋友圈

杨间继续从李瑶处追问势力背景，得知其内部称为“朋友圈”。李瑶称这并非单一驭鬼者团体，而是早期接触真实灵异的资本家形成的圈子；他们曾建实验室、招揽驭鬼者、收购被关押厉鬼，王小明早期也曾为其工作，平安大厦和平安大酒店则有其出资和股份。国家介入后双方改以合作延续，但灵异事件升级、总部针对和民间势力出现已削弱其优势。李瑶认为队长名额决定后续利益，故朋友圈会拉拢候选人或排斥不受控者。杨间不认同危机下仍争权逐利的做法，只要求她留在圈内收集动向、照看张芬并避免露出破绽。张雷到门外提醒培训开始，杨间带走鬼报纸和人皮纸离开，李瑶仍把二人关系当作真实恋情。（第432章）

## 第433章·第二根棺材钉的信息

前往培训基地的车上，张雷再度请求杨间提供解决厉鬼复苏的方法。杨间以自己曾限制饿死鬼的棺材钉举例：若能得到同类物并钉住体内鬼，理论上可使复苏问题受到限制，但这种能够压制S级厉鬼的东西极难取得。他用鬼域显出一根假钉时，张雷忽然想起自己曾吞食的一只鬼残留的记忆。那段记忆中没有光亮，只有无边坟场、一扇木柱和破败铁门构成的大门，另一根棺材钉钉在门旁木柱上。张雷不能说明地点，也不能确定自己为何保留该段鬼的记忆；杨间却联想到灵异公交曾经过的无尽坟场，怀疑两处相同。二人仅取得可能的第二根钉子线索，尚未验证地点，也没有解决张雷复苏问题。（第433章）

## 第434章·约见王小明

杨间抵达大京市郊区培训基地，按总部规矩须在此接受至少七天培训。他先嘱咐张雷保密第二根棺材钉的线索，避免对方因掌握限制物信息遭人挖掘秘密；沈良随即依先前承诺，带他到附近招待所单独会见王小明，李军与安保人员分布在周围戒备。谈话开始，王小明便从杨间复活后身体机能不断恶化推断其厉鬼正在躁动，又以“它”的称呼试探协助杨间复活的未知存在。杨间没有正面回应。王小明随后提出把身体转化为有意识的鬼看似可规避侵蚀，却承认鬼棺实验没有留下人：受试者失去感情和自我，最终只孕育出新的鬼。杨间因此判断这条修复路径不能直接采用，双方尚未达成交易。（第434章）

## 第435章·特别的交易

赵建国与李军在楼顶谈及队长计划。赵建国担心把希望集中到少数人会流失杨间这类优秀驭鬼者，李军则认为灵异事件升级、刑警持续死亡、新人减少，集中资源守住部分区域已没有更好替代方式。遮阳棚下，王小明进一步解释鬼棺实验失败的原因：人失去感情后，厉鬼本能会吞噬自我，鬼棺实际是孕育鬼的棺材。他又从杨间自杀复活录像推断，杨间曾主动触发多鬼规则冲突，使其中至少一只死机。王小明遂提出交易，以杨间身边的“它”为筹码，为其制定新方案，并交出一枚鬼棺孕育之鬼死机后制成的黑色钉状物，称其可使三只鬼能力失效、价值近似棺材钉。杨间尚未接受，灵异物由鬼制成的推论反而留下更多来历疑问。（第435章）
