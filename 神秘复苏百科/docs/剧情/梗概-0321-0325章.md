# 第321-325章梗概：鬼橱代价与尚通搜查

## 第321章·无限循环的交易规则

杨间没有立即答应鬼橱“搬出去”的纸条要求，而在密室观察后果。十五分钟后木橱渗出超出狗血分量的暗黑血液，再过一刻钟，下层柜门打开，湿发、肿胀手掌和疑似泡水女尸试图钻出。杨间由此确认鬼橱提出条件后若未完成，会按时间升级异常；他以鬼域把鬼橱移至小区高楼楼顶，渗血和女尸立刻消失。检查下层仅余湿痕与黑发，证明异象并非幻觉。他推导出鬼橱交易会形成循环：人提出要求后要支付条件，完成后鬼橱又提出要求；若不断索取，代价会逐步扩大，人终会承受不住。（第321章）

## 第322章·第二次交易

杨间把恢复平静的鬼橱放在附近钟塔顶层，既便于观察也避免密室再度失控。他没有放弃研究，改将问题从“交付赵磊”缩小为询问赵磊位置；鬼橱要求“一只鬼”。杨间认为条件尚可操作，却不愿交出顺手的鬼绳或能制造驭鬼者的鬼骷髅，决定另找价值较低的受限厉鬼。此时鬼橱没有因杨间暂缓完成自己的条件而产生异变，令他观察到交易中人提出要求时可暂不履行、鬼橱反向提出要求时却会带有胁迫。张丽琴同时带行李搬入杨间住处，江艳与她因杨间产生明显敌意；二人接下鉴定鬼橱蛋炒饭瓷盘来历的事务。（第322章）

## 第323章·大厦

杨间准备随章华搜查尚通大厦前，张丽琴发现他按王小明计划改善外表尸斑；杨间说明身体平衡一旦失效仍会腐烂。章华提醒尚通科技虽被怀疑雇佣境外人员、参与非法活动，却须依法取得证据以免灵异真相外泄引发社会动荡；杨间只同意先走搜查程序，并看上可能被查封的大厦作为未来公司办公地点。进入大厦后，他以鬼域覆盖楼层，发现只有顶层一间房被黄金隔绝，遂跳过常规逐层搜查，带章华一行直赴四十五层。尚通人员切断电梯电源并试图远程阻拦，仍被杨间以鬼域恢复、驱动电梯越过限制。（第323章）

## 第324章·唱片

顶层走道灯光异常闪烁后又恢复，杨间与章华进入金发外籍男子保罗的办公室。保罗自称公司老板，早已认出杨间，并以酒和拥抱试图拖入谈话。杨间确认他不是驭鬼者，却感觉办公室有异常，直接撬开办公桌抽屉取出一张美国风格唱片；保罗称其为“问候”，杨间为避免未知声音类风险当场掰断。唱片碎裂后异常感消失，但保罗没有表现出真正惊讶。办公室背后的黄金隔绝和保罗手里的资料仍显示公司另有准备，搜查与对峙继续。（第324章）

## 第325章·声音

杨间指出酒柜后存在黄金隔绝的密室，章华依法要求开启，保罗仍以隐私和律师拖延。杨间将保罗隔入局部鬼域，警告离开沙发便会消失；保罗迈步后双腿被鬼影力量取下，只能承认观江小区行动者是公司员工，并以公司、钱财乃至替杨间处理王小强作交换。杨间告知王小强已死，拒绝让公司仅以转让存续。保罗最后播放一盒带异样气息的磁带，杨间在录音展开前捏碎播放器和磁带，但碎片仍发出沙沙声。杨间将事件警戒提升至可能的灵异层面，录音内容和密室证据尚未揭示。（第325章）
