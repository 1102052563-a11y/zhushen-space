# 第101-105章梗概

## 第101章·驾驭第二只鬼

人皮纸持续吞噬附身老婆婆的无头鬼影及叶俊尸水，老婆婆虽挣扎仍被完全拉入纸中。杨间意识到交易原本只约定一只鬼，人皮纸却至少取得两只，担心承诺失效，也警惕它为何能吞鬼却未吞自己的鬼眼。鬼域已接近极限，鬼眼在身体各处乱窜；他不敢撤域，因黄岗村落单必死的风险仍在，甚至准备钻进黄金尸袋、给母亲留下遗产。此时无头鬼影从人皮纸中沿地面回流，连接纸与杨间的黑线显现，纸上写明交易成功、将开始驾驭第二只厉鬼。影子开始侵入其身体，人皮纸再显示具体方法，杨间只能继续冒险。（第101章）

## 第102章·鬼影

杨间确认无头鬼影能从人皮纸出现并非自行脱困，而是纸为履行交易主动释放并暂时压制它。他明知直接融合第二只鬼等于抢夺身体，仍按人皮提示将一只鬼眼交给鬼影；鬼眼转移后，复苏躁动减轻。他连续交出五只鬼眼，鬼眼复苏得到延缓，自己也能操控无头鬼影，但人皮警告鬼影一旦获得其他身体，就会掌控鬼眼并反客为主。连接人皮的黑线断开后，鬼影果然贴上杨间夺身；杨间用鬼影身上的五只鬼眼将其钉回地面，把复苏压力转给鬼影。鬼影最终融入他的影子，杨间拥有代号“无头鬼影”的第二只鬼。（第102章）

## 第103章·打开的棺材

杨间认为两鬼平衡只能暂缓复苏，仍决定保留人皮纸并把它塞入旧黄金盒。他再次展开鬼域，范围已达约五十米，说明鬼驾驭鬼能发挥更强力量。他从跑车黄金行李箱放出躲藏的张韩，解释危机只暂时解除，二人必须同行以免触发落单必死。杨间判断黄岗村一鬼游荡杀害落单驭鬼者，另一鬼在灵堂棺材，要求张韩共同开棺，并准备以黄金行李箱和尸袋收容。两人刚要撬棺，棺内传出咳嗽，原本难以开启的棺材自行裂开，一只惨白手掌伸出，张韩惊恐呼喊杨间。（第103章）

## 第104章·病鬼的身份

棺盖自行推动，杨间和张韩以鬼眼、无头鬼影、背后鬼和黄金尸袋戒备。一个脸色死灰、目光痛苦麻木的男子从棺中坐起，穿着国际刑警制服；杨间认出他可能是失踪的大昌市第一任刑警冯全，便暂缓肢解。冯全回应身份，称众人本不该来救他。面对质问，他承认此前咳嗽来自自己，原意是将分散驭鬼者聚在一起抵抗会杀落单者的鬼；但众人未理解，连续死亡令村鬼增强并接管村子。杨间认为删减档案和模糊暗示使自己险些团灭，责怪冯全没有直接说明；冯全则称夜晚以后已无力挽回。（第104章）

## 第105章·村子的真相

冯全解释自己不能离开鬼棺，是为阻止外面厉鬼进入棺中，否则其恐怖程度足以毁灭城市。杨间此前开棺时未见的厉鬼，原来就是供桌遗像里原先的年轻男子；相片此时已空白，冯全称该鬼一直守棺。杨间曾独坐灵堂未遭袭，是因冯全在棺中使他不算落单，灵堂由此兼具危险与安全。冯全又说明村民是因鬼棺存在的“不存在的人”，类似鬼奴却能受棺中自己影响；村中除三人外早无活人。行动表面为解灵异事件，实际也涉及救冯全和鬼棺，但冯全拒绝泄露鬼棺机密。谈话结束时，恢复明亮的村子再次昏暗，新的危机逼近。（第105章）
