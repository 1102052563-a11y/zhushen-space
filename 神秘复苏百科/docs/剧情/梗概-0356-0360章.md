# 第356-360章梗概：柴刀鬼与死亡节点

## 第356章·死亡时间

熊文文再次预知十三号房内的死亡，确认自己会在两点十分被斩首，即使杨间把头拼回去也不能复活；随后罗素一于两点十五分腰斩，林落梅也会被拦腰杀死，杨间则独自开门逃离。杨间认为预知可能受复苏影响而有偏差，却仍把时间当作可利用的信息：既然熊文文尚未到死亡时刻，眼下应有数分钟行动空间。他把红色鬼烛和打火机交给林落梅，令她保护熊文文、约三分钟后点燃；又强迫罗素一监视走道鬼，自己进入卧室寻找源头。杨间判断篡改记忆鬼的已知手段不符合暴力斩首，故怀疑杀人者另有其鬼。罗素一虽恐惧，仍探头观察走道，调查转为以死亡倒计时反查厉鬼规律。（第356章）

## 第357章·转身回来

杨间在卧室见到不同性别、衣着的无脸人，有些尸体腐烂，有些仍有体温心跳；他认为它们的脸并非被直接剥下，而是被取走五官，只剩平坦人皮。他未在其中发现带童倩笑脸的真正鬼，却找到背对窗户、体貌为赵磊的活体。赵磊脸上装着腐烂陌生人脸，杨间以鬼影撕下后令其重成无脸状态，并未找到赵磊原本的脸，遂推测赵磊身体在十三号房、脸或源头鬼可能与三十一号房有关。门外罗素一发现高大尸体背向队伍离开，随即又在出口前停下转身走回；他由此意识到死亡预告仍会发生，紧急报告杨间。杨间只能在带走赵磊身体、继续追查三十一号房和应对走道鬼之间取舍。（第357章）

## 第358章·瞬间燃空的蜡烛

杨间背起赵磊身体离开十三号房，决定不在原地等死，并让林落梅持燃着的鬼烛带熊文文同行。走道尽头的高大尸体持锈蚀缺口柴刀正面逼近；杨间由脚印确认它是此前二层的未知鬼，并因其手持柴刀联想到弘法寺棺材钉，却不能确认两件铁器关联。他根据中山市既有伤亡和目击情况推测该鬼并非无差别杀人，或一次只会杀一人，但不敢接触验证。两点十分到来时，尸体停步、把柴刀抬到与熊文文颈部等高处挥向空中；鬼烛立刻如爆燃般耗去大半，熊文文幸存。杨间由此确认死亡预告与持刀鬼有关，但唯一可靠防护已几乎耗尽。（第358章）

## 第359章·搜寻规律

鬼烛仅剩的小段又因林落梅惊吓脱手而熄灭，杨间带队向走道深处撤退、寻找三十一号房。持柴刀尸体在后方停住不动，杨间试图从其走路、转身、停步和挥刀的行为推导规律，却只确定杀人不依赖简单的视线或朝向。熊文文本该死亡却因鬼烛活下，杨间无法判断尸体会重试还是跳过目标；他坦言下一次袭击几乎无烛可挡，众人应做好死亡准备。到三十一号房前，他命已多次使用能力、明显越发冰冷死灰的熊文文再次预知。结果显示尸体会跳过被鬼烛保护的熊文文，仍在两点十五分杀死罗素一；杨间以鬼域包住罗素一也无法改变结局，证明其鬼域不能防御这次必杀。（第359章）

## 第360章·大力出奇迹

杨间让罗素一尝试不回头看鬼，预知仍显示其会死亡，故排除单纯方位条件；他认定高大尸体每次停步即代表已经锁定一名符合条件者，却无法在短时间解出具体触发方式。鬼绳直接绑住尸体双腿的方案也被熊文文预见为失败：杨间会死于鬼脚边、鬼绳被柴刀斩断。罗素一接近两点十五分而崩溃，欲以自身厉鬼搏命，被杨间阻止。最后十秒，杨间注意到尸体正沿一整条相连地毯行走，便以鬼影强化后的巨力猛扯地毯，试图将其绊倒。尸体没有摔倒却未在预定时刻停步挥刀，罗素一反而活下来；杨间只确认干扰地毯改变了结果，规律的真正条件仍未查明。（第360章）
