# 第535-539章剧情梗概

## 第535章·第二次开门

黄岗村最后一栋砖瓦房内，距第二次鬼门窗口仅余一分钟。柳三放落鬼绳后，鬼绳与吊尸原有的相互限制被打破；李军判断尸体本身也是鬼，房内两鬼一旦在鬼烛熄灭后活动，队伍几乎不可能撑住。鬼画同时侵入墙面和地面，纸灰蔓延，阿红手中红色鬼烛加速燃烧，半张脸也逐渐变成陌生人的样子。时间到达后，李军把总部木门把手置于房门，在约定时刻推门，门后出现一片黑暗。苏凡、柳三、阿红和徐一平先后穿过黑暗并抵达总部地下房间；李军最后进入时被尸体冰冷手掌抓住肩膀。秦老准备关门时，门后传来两个人的脚步，救援虽接回四人，李军的安危和另一个跟随者的身份仍未揭示。（第535章）

## 第536章·走出的卫景

鬼门仅能开启三十秒，门后却传出两道脚步。曹延华见一条发黑腐烂的人腿先踏出，立刻要求关门；王小明却以愿意承担责任为代价阻止，秦老也挡住试图自行关门的队员。腐臭尸体最终走入总部，柳三认出它正是自己从房梁放下的卫景。李军随即浑身焦黑地走出，因持续使用鬼火而倒地，证明第二次开门仍把他带回。王小明确认卫景尚有意识，并解释这是根据人皮纸信息所做的实验：卫景可能随鬼差重启回到仍存活的状态，又因鬼画压制而维持下来。卫景自称身体已经死去腐坏、意识却始终存在，认为自己更像鬼棺的一部分；秦老将其称为“异类”。曹延华随即把卫景情况列为机密，秦老关门，卫景的稳定性与危险仍无结论。（第536章）

## 第537章·遗失的两只鬼

城区外围，陈义在郭凡之鬼袭击和钟山熄烛后勉强活下，并由总部接应者续燃第二根鬼烛。队列没有恢复原状：原来从鬼画走出的五只鬼只剩三只继续跟烛，两只失去踪迹；鬼画与鬼差鬼域也一同消失。陈义认为此前利用两鬼互相压制、再引走鬼画的主方向仍成功，却因郭凡环节失控留下可能在国内复发的巨大隐患。赶来的姜尚白未能找到两鬼，确认郭凡已和灵位中的鬼重新换回，暂时没有复苏。陈义则因脱皮逃生失去鬼皮，露出血肉，只能忍受更剧烈痛苦重新穿回已会自行站立、等待活人的人皮衣，以维持体内平衡。清晨城市恢复日常，杨间从平安酒店观察到平静，判断总部暂时处理住事件后，收到李瑶的消息，决定转为主动处理朋友圈。（第537章）

## 第538章·瓦解核心

李瑶亲自到平安酒店接杨间，交出通宵整理的朋友圈成员资料，同时带来新消息：鬼画走出的五只鬼中有两只失踪，接手处置的姜尚白及其朋友圈必须投入搜查。杨间没有因成员资料多而放弃反制，判断多数挂名驭鬼者不会为组织拼命，决定直接打掉少量核心人员，再以“鬼眼杨间”公开警告震慑外围。李瑶筛出平安大厦总裁费总，称此人负责制定对付杨间的计划，虽然是普通人，却有保镖张建暗中保护。她设计先找张建、再借其车辆接近费总的路线，自己则不在现场露面。杨间开车到张建的私人酒吧，见大门暂停营业，便按李瑶指引独自从后门进入。费总、张建的防卫安排和杨间能否按计划接近费总，尚未写出。（第538章）

## 第539章·酒吧中的红光

杨间进入私人酒吧后，发现白天仍有大量年轻男女在音乐中狂欢，怀疑闭门营业只是掩饰。包厢内的张建正以打牌、灌酒和散钱取乐，见杨间闯入后认出其身份，先踢飞两名保安撇清关系，又以加入朋友圈、享受普通人恐惧等说辞招揽。张建认为驭鬼者活不长，因而应尽情践踏他人；他掐住女伴逼问爱意，再用一袋钱引得众人争抢，显露出失控和残酷。杨间没有回应招揽，只以鬼域令包厢、人群和光源消失，留下红光覆盖的封闭空间。张建立即认出鬼域，意识到杨间的目的不是偶遇；杨间则说自己赶时间，要顺手把他埋掉，免得制造麻烦。本章结束在对峙时刻，张建会否反抗、杨间是否杀人以及费总后续处境均未揭示。（第539章）
