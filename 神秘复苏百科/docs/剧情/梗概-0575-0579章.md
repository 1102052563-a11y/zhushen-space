# 第575-579章剧情梗概

## 第575章·释放的诅咒

曹延华误以为杨间故意不接电话，让刘小雨继续安抚，实际杨间已在平安酒店附近遭鬼剪刀袭击，四层鬼域覆盖整条街道。他头身分离、头颅又被横向剪开，无头鬼影无法覆盖脑部，意识即将消失。方世明使用两张完整复印照连续发动鬼剪刀，照片一旦剪坏便不能再次触发；连续两次使用也令剪刀诅咒显化为办公室内的残肢、鬼影和脚步，使他不敢继续冒险。杨间在鬼烛、替死布娃娃和八音盒间选择最后者，借鬼影勉强操控尸体打开八音盒一角。额头再被剪裂时，他听见空灵铃声，八音盒诅咒被正式释放，能否存活仍待验证。（第575章）

## 第576章·情绪崩溃

刘小雨持续呼叫杨间未果，循定位发现手机停在平安酒店附近，遂强制开启卫星定位手机摄像头。她先只见红光覆盖的街道和倒地手机，转动视角后认出血泊中的杨间衣物、鬼烛、无头尸体及裂开的头颅，情绪当场崩溃。她赶往曹延华办公室报告杨间死亡；恰逢李军已报告平安酒店附近整条街道消失，曹延华据鬼域、六点答复期限和中断电话判断，这很可能是朋友圈的报复。刘小雨与杨间长期接线形成的依赖感令她无法接受事实，曹延华则带人赶往通讯室确认，并在惊怒中准备追究责任。（第576章）

## 第577章·诅咒的效果

总部通过手机影像确认杨间身首分离，判断其死后鬼域与厉鬼复苏可能引发连锁危机；曹延华要求向姜尚白核实。方世明回到平安大厦后承认杀死杨间，并威胁总部若报复便可继续打掉李军、曹洋等队长，提出让总部当作无事发生、朋友圈仍协助鬼画事件。曹延华愤而欲半日内打掉朋友圈，却被多数人以鬼画、杨间复苏风险和避免再损失队长为由阻止。街道上，八音盒音乐只被杨间听见，已死的意识逐渐恢复；他确认八音盒以诅咒维持其思维，却使他失去视觉和身体控制，必须尽快自救。（第577章）

## 第578章·危险正在降临

杨间确认八音盒暂保意识不灭，但身体已死、无法说话或行动，视觉听觉尽失，只能在脑内听见循环音乐。他预判四层鬼域长期维持会令鬼眼复苏，若尸体上的鬼失控，可能覆盖整座城市，形成比鬼画更难处理的事件。总部影像显示尸体双手、双腿开始不规则活动，确认厉鬼复苏迹象；部长随即要求吴组长封存刘小雨通讯室，隐瞒杨间被朋友圈杀死，对外改称死于灵异事件。曹延华虽不甘仍同意，王小明只询问尸体能否取回。杨间则在黑暗中回忆后脑仍有鬼眼，开始感应它，寻找打破活死人状态的机会。（第578章）

## 第579章·站立起来

被四层鬼域困住的路人发现杨间无头尸体站起，惊恐躲避。杨间终于感应并控制后脑残留的一只鬼眼，恢复视野，发现自己的意识寄在被横切后的上半头颅；无头尸体则由无头鬼影接管。鬼影暂时因死机而不主动杀人，却会本能寻找新的头颅，一旦接触路人便可能把所有人变作拼接材料。杨间想直接用鬼域把头移回脖子却失败，单只鬼眼不能在既有四层鬼域上再开一层，只能寻求可信驭鬼者帮助。尸体随即开始向人群行走，鬼影本能迅速增强；杨间无暇继续思考，控制四层鬼域朝平安酒店方向扩散，后续结果未写。（第579章）
