# 第301-305章梗概：被困鬼公交与哭坟站

## 第301章·隔绝的车厢

杨间原本只想替总部捡回许峰的卫星定位手机，发现公交改道、久不停站并驶向小春市郊后，决定跳窗离开，以免卷入未知灵异事件。然而车窗无法开启，连他足以打碎钢化玻璃的力气也无效；掌心鬼眼的红光鬼域同样被车厢隔绝。他由此确认车辆异常，怀疑许峰正是在车上出事。背包青年张浩等人仍把路线问题当成普通改道，张浩欲找司机询问，又被一名年长婆婆劝回。杨间虽未能判断车内哪些人有问题，却借其余五人刚上车的表现暂认他们正常，继续观察；车门、车窗与鬼域同时失效，使他无法用惯常能力直接离开。（第301章）

## 第302章·死去的司机

杨间向张浩等人说明车辆长时间不进站且不易下车，部分人仍不相信。郑耀文到驾驶位呼喊、推动司机后，司机倒地露出已死亡至少一天且轻微腐烂的尸体，车内恐慌立刻爆发。更异常的是，仪表盘全灭的公交仍以约五十码行驶，方向盘自行转动，普通乘客既不能转向也无法刹车。杨间没有把司机尸体视为核心线索，而是推测可能有无形鬼驾驶、公交本身是鬼，或厉鬼伪装成乘客；他暂时倾向车辆本身有鬼性，只能等待它主动停车开门。救援电话虽可拨通，却无法解决被封闭在车内的现实处境。（第302章）

## 第303章·车上的乘客

乘客拨通求救电话、砸窗和掰门均无效，杨间则趁车厢内暂时没有触犯禁忌的迹象检查驾驶位。他发现未通电的黑色显示屏亮着“当前乘客数量：2”，面对车内二十余人，推测数字实际记录车上厉鬼而非活人，因而重点怀疑司机尸体、提醒张浩的婆婆，以及末排低头不动的风衣男子和鸭舌帽男子。程雄此时短信确认记事本压痕已有分析结果，杨间因被困未作处理。正当张浩求助时，公交开始减速靠站，杨间准备利用开门后可延伸的鬼域脱身，却仍不能确认下车是否安全。（第303章）

## 第304章·停站下车

公交无人操控地停在荒野电线杆和临时站牌旁，车门一开，普通乘客争相下车。杨间察觉外界昏暗陌生而保持警惕，此时末排风衣男子出声，承认自己是失踪刑警许峰，并说明晚六点后每过一站会有厉鬼登车，显示屏数的是鬼数；此刻数字由二变为一，意味着有一只鬼下车。许峰已在车上撑了七八日，称每站仅停五分钟、下车要等时机。远处随即出现无碑土坟和孝服哭坟者，哭声令下车乘客跪地死亡，许峰的警告得到印证，杨间因而没有立即踏入车外。（第304章）

## 第305章·哭坟鬼

面对哭坟声，杨间认定车外是鬼的世界，提醒张浩等人回车；张浩和两名女生成功返回，郑耀文因回头救摔倒的冯晓露而同她一起不受控制地哭泣、跪向土坟并死亡。短时间内十余名下车乘客遇害，杨间据此将厉鬼暂定为A级的哭坟鬼。哭声停止后，哭坟鬼从坟前来到公交车尾并走向车门；杨间权衡鬼影、资源与后续风险，没有阻拦。它上车坐下便静止不哭，显示屏鬼数重回二，公交五分钟后关门驶离，车内仍有另一只鬼身份未明。（第305章）
