# 第266-270章梗概

## 第266章·第七中

杨间检查王珊珊遗体，发现她虽无心跳却没有尸臭、皮肤柔软，判断鬼奴印失去力量后她只是介于生死之间。他把红光注回其手腕黯淡的鬼眼图案，王珊珊随即苏醒，并说昏死时见过一只鬼站在“七中”门口。杨间由此补全叶枫遗言的“七”字，认定源头鬼在第七中学；他拒绝王小明让张韩充当诱饵的七成成功率方案，独自进校。浓阴霾把鬼域压至十米，校内又有上千鬼婴，令他不敢贸然接触源头，只先寻找当年厕所后的棺材钉入口。（第266章）

## 第267章·有内鬼

指挥中心得知杨间拒绝诱饵计划，曹延华不满，赵建国则认为只能相信行动中的杨间。总部已以数吨黄金制成厚五厘米的金棺，准备封入饿死鬼后焊死。杨间手机却检测到失联刑警赵开明的信号，曹延华查出其精神评估早已不合格，只因周正死后人手不足才被临时任用。校园内，杨间在鬼域保护下穿过上千静止鬼婴，发现三楼厕所已失去那扇门，意识到原入口随旧日鬼域迁至步行街。他收到“有内鬼、赵开明在附近”的警告后，赵开明便直接闯入鬼域。（第267章）

## 第268章·被附身的赵开明

赵开明称自己并非通常驭鬼者，而是与体内无形鬼合作；鬼让他肢体被拆后仍可复原。他承认一直放纵鬼婴，目的在让它吞吃叶枫、何川等驭鬼者的厉鬼，为体内鬼拼合更完整的“拼图”，还想夺取杨间鬼眼。杨间虽能扯下其手脚和头，却杀不死人与鬼融合的目标。源头鬼回到学校并看穿鬼域后，杨间不愿无限制物硬拼，便把赵开明头颅扔向源头鬼，借其被吸引脱身。赵开明却暗中打算在附身鬼成功、复活家人后，再利用杨间关押两只鬼。（第268章）

## 第269章·锈迹斑斑的棺材钉

杨间回到步行街，以鬼域重演当年逃离第七中的路线，发现三楼厕所对应弘法寺大雄宝殿前；他在寺庙地下室找到被金箔封住的木门。住持认出其中物品是代代相传不可动的“降魔杵”，却在得知大昌市危机后不再阻拦。门后灵异空间内，白骨树钉着一具三米高、缺一只眼的实体黑影，胸口锈蚀棺材钉不断滴下黑血。杨间拍照留档、以金箔包住钉身后拔出，决定暂不处理失去限制的黑影，立即携钉返回七中。（第269章）

## 第270章·引诱厉鬼

杨间十分钟后回七中，发现密集鬼婴全部消失，赵开明留下录音称计划成功。走廊出现鬼域看不见的脚步和黑脚印，阴霾迅速把他的鬼域从百米侵蚀至五十米；他判断赵开明附身鬼与饿死鬼结合后已成更难定义的目标。手握棺材钉仍是唯一胜机，他主动把鬼域缩到一米，等待厉鬼进入反击范围，却久未等到动静。为遮蔽钉子的威胁，他用金箔彻底包裹并继续缩小鬼域。此时目标已站在他背后两米，外形竟近周正、穿叶枫寿衣并持干枯手指，杨间仍未察觉。（第270章）
