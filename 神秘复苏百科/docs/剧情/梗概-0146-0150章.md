# 第146-150章梗概

## 第146章·开始的战斗

杨间闯入小强俱乐部会议厅后，追问王小强为何要清除自己及谁杀了严力一家。王小强否认俱乐部杀害严力家人，却承认杨间若成为国际刑警将触动大昌市既有利益，又把黄岗村中多名成员的损失归咎于他。雷虎等武装人员奉命以黄金制特殊弹药开火，子弹击中的却是鬼域中的假杨间；杨间真身始终隐在域内，以鬼影侵入先前偷袭者，并在枪火停止后以手枪处决两名人员。普通股东试图撇清投票责任，杨间认为其既举手支持杀人便须承担后果，会议由暗中表决正式变成死亡冲突。（第146章）

## 第147章·王小强的鬼

王小强识破眼前杨间为幻象，雷虎便投放含黄金成分的烟雾弹，试图让看不见的鬼显形；武装人员却在烟中误把一名股东射死。杨间借鬼域变换视听，反从身后枪杀两名持弩者，表明进入鬼域的人员无法分辨真假。王小强见普通人已无法抵抗，脱下衣物露出由青黑肉块、尸斑老皮和惨白皮肤以金线缝合的身体。他每走一步都有血肉掉落，烟雾里的模糊人影便拾起血肉跟随；杨间的鬼域随之被压缩，确认王小强放出了真正的鬼。双方尚未直接接触，王小强的异常身体和厉鬼已使战局升级。（第147章）

## 第148章·改变位置

杨间判断王小强体内放出的鬼过于危险，没有正面接触，而是张开鬼眼鬼域，把王小强、雷虎、股东和自己移到庄园外部般的红光空间，单独留下那只鬼。王小强由烟雾、尸体和环境同时消失，意识到杨间可将人和鬼分隔；雷虎等人仍企图循枪声寻找真身，却被鬼域幻象和无头鬼影逐个瓦解。杨间命影子侵入雷虎，借其手取下王小强头颅，再补枪确认击杀。王小强临死前察觉杨间至少驾驭两只鬼；其头颅内没有头骨的异常则未获解释。杨间以避开厉鬼、控制活人为策略结束主要抵抗。（第148章）

## 第149章·拷问

战斗前后不足五分钟，杨间清除王小强和武装人员后，继续处置支持过表决的富商股东。他以鬼域消去虚假的会议桌，将不知情或试图以金钱、公司求生者送到地下十米活埋，认定放过有资本的股东只会留下雇凶报复的风险。马有财为保命供出线索：王小明常去玫瑰酒吧，酒吧老板王岳是曾到过俱乐部、可收钱办事的驭鬼者，或许知道严力死亡内情。杨间虽怀疑其真假，仍以无头鬼影将马有财头颅换到雷虎身体上作为约束，并离开调查王岳。与此同时，留在原地的王小强之鬼正朝尸体走去。（第149章）

## 第150章·死而复活

庄园内，换头后幸存的马有财目睹王小强放出的腐烂厉鬼走到尸体旁，依次掏取内脏、撕下皮肉填补自身骨架，最后套上没有头骨的王小强头颅。约半小时后，残缺拼凑的王小强恢复意识，承认自己已又死过一次，并担忧身体再损坏便会被厉鬼舍弃。他从马有财的换头情况推测杨间能随时再取其头，误认杨间因看中资本才留其活命；马有财则隐瞒王岳线索。为重建遭清洗的俱乐部，王小强要求唯一幸存股东三日筹十亿元资助，自己也开始怀疑杨间在黄岗村取得了第二鬼驾驭方法。冲突未因王小强一次死亡而结束。（第150章）
