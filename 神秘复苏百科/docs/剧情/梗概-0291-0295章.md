# 第291-295章梗概：赵磊与报纸鬼的线索

## 第291章·赵磊的经历

杨间离开总结会后被旧同学赵磊拦下。赵磊因自认全家死于饿死鬼事件，质问杨间为何救下王珊珊、张伟等人却没有救自己家；杨间指出大型灵异事件中不可能照顾所有人，也察觉赵磊的愤怒和陌生感并不正常。赵磊承认自己成为驭鬼者，并说曾在杨间旧居见到与杨间相同、坐着读报纸的人。杨间因此认出住处旧鬼的线索，欲以鬼域限制赵磊；赵磊却化作留下人脸的血红纸片消失。漫天带脸红纸形成荒废街道，尽头长椅上坐着持报的僵硬人影，杨间被迫与这只未知鬼物对峙。（第291章）

## 第292章·寻找脸的鬼

持报鬼未被饿死鬼事件清除，红纸飞舞时会遮挡杨间鬼眼，使其无法张开鬼域；带脸红纸贴上无头鬼影，也能令鬼影局部停滞。杨间改用死机鬼影覆盖身体，发现头部以下几乎化作黑影，随后拆下鬼影手臂作为标枪投向长椅目标。手臂击穿数张红纸和报纸，鬼域随之消退，但杨间只找到破衣与陌生女人的人皮脸，确认那只是替身，赵磊已逃脱。他让刘小雨发布全国通缉，并因红纸持续收集人脸而联想到童倩的鬼脸，担忧两者是否存在关联，但未得到答案。（第292章）

## 第293章·最后的笔记

杨间重新回到旧租屋追查赵磊。屋内散落的碎纸记录了赵磊遭鬼袭击后的挣扎：他不断遗忘自身记忆、混入其他人的记忆，曾用笔记提醒自己，鬼又利用笔记制造相反指令。赵磊最终意识到无法驾驭该鬼，便自我催眠植入“杨间害死自己全家”的假记忆，希望厉鬼因仇恨找到杨间并被解决；但鬼吸收了赵磊全部记忆，也看穿了计划，所以此前只是试探便撤走。杨间确认赵磊的人格很可能已消失，却也因其绝境中的自救设计而愤怒，决意继续追查。（第293章）

## 第294章·再次对话

杨间理解赵磊并非主动仇视自己，而是失败后被篡改记忆的受害者，收走碎纸和旧物、留下房租后告别旧居。他巡逻大昌市，刘小雨查明赵磊父母其实仍活着，只有部分亲属失踪，印证其记忆失真；市内暂未发现新灵异事件。北方高速附近则有严力鬼血复苏留下的血红水塘，杨间认为其平静且风险不明，暂不触碰。回到观江花园后，他让江艳、张丽琴整理笔记，又记录鬼影投掷和覆盖身体的试验结论，最后取出人皮纸寻求赵磊线索。（第294章）

## 第295章·信息指引

人皮纸先称需一个月寻找赵磊，杨间认为它在拖延，便以自己已能长期压制厉鬼、可将其永久深埋为筹码谈判。人皮纸最终指向观江花园民国荒宅的第二间房；杨间虽怀疑这是借刀杀人，仍决定白天前往探索，同时把人皮纸放进黄金盒，以鬼域送至地下约一万米。出门时，他遇张韩遭伪装记者队纠缠。苏珊持续追问鬼眼刑警，摄影师特瑞等人又在警告枪声后露出武器，杨间判断采访只是接触自己的掩护，拒绝其舞会邀请并报警。第二间密室和这支采访队的来历均留待后续。（第295章）
