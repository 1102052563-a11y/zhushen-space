# 原文回跳第694-698章剧情梗概

## 原文回跳第694章·古怪的东西

杨间追问李跃信使身份，得知丢失或不能送达信件会遭厉鬼袭击。杨间认为替未知鬼物办事会给大昌市带来隐患，决定强看信件；持枪信使以嵌入手指骨的特殊子弹击伤其胸口，却无法阻止无头鬼影拼补伤口。杨间随即以鬼手扭断枪手颈骨，借此试探信使的对抗手段。李跃承认自己两个月前因拆信成为信使，本次只是第三次送信。面对杨间持续威胁，他交出信封，试图以信件背后的真相换取生机。（原文回跳第694章、原文回跳第695章）

## 原文回跳第695章·诅咒的信

杨间拆开黄色信封，信纸依次出现“千万不要打开这封信”“打开这封信的人会被厉鬼杀死”“鬼就要来了”“鬼就在你的身边”。看信的短暂过程对李跃等人却像过去数分钟，树林骤暗、阴风和腐臭出现。李跃以骨灰站位自保，赵丽取出圆形纸钱，周林持最后一发骨制子弹等待。杨间意识到信可能原本没有这些文字，或拆信本身触发诅咒；无论原因，信件已把一只未知厉鬼引入树林。鬼眼受树木和灵异影响无法完全穿透，众人只能在盲区中等待鬼接近。（原文回跳第695章、原文回跳第696章）

## 原文回跳第696章·徘徊在树林中

树林外仍是白天，内部却被局部黑暗笼罩。杨间在树间短暂看见人影，推断鬼在不断移动并接近众人；周林、赵丽听见脚步却难以定位。周林感到后颈被冰冷手指触及，惊慌中打出最后一发骨制子弹，却击中树木。鬼没有立即杀他，使所有人误以为暂有缓和。杨间则继续观察，认为信件引来的鬼可能不按常规规律杀人，贸然靠近或动用高层鬼域都存在风险。周林失去唯一武器，心理防线随之崩溃。（原文回跳第696章）

## 原文回跳第697章·不寻常的移动

杨间以三只鬼眼寻找树林盲区，周林因恐惧逃跑。鬼在其背后现形，像刚从坟土挖出的尸体贴在身上；周林却毫无察觉，走出不远便倒地死亡。杨间靠近时发现周林尸体半张脸已变为鬼脸，尸体又抬手试图触碰，显示厉鬼可借活人身体行动或取代活人。杨间立刻用鬼域退开，不敢用鬼手强压，因为一旦厉鬼反向侵入自身，多鬼平衡会立即崩溃。树林中的信件危机由此从普通追杀转成高风险入侵事件。（原文回跳第697章）

## 原文回跳第698章·放走的三人

杨间派无头鬼影接近周林时，树林黑暗却突然消散，周林如从昏睡中醒来，否认记得死亡或被鬼入侵。杨间确认尸体曾失温，却无法解释其复活，决定不再冒险压制。黄色信件化为纸灰，李跃认为任务结束，并说明信使只能靠继承诡异邮局摆脱诅咒。杨间将李跃、赵丽和周林逐出大昌市，留下字条要求出境后再看；李跃读到“鬼在另外一个男子的身上”后，从后视镜看见周林半张泥土惨白鬼脸。危机没有被关押，只被带离辖区。（原文回跳第698章）
