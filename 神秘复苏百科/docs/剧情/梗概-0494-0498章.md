# 第494-498章剧情梗概

## 第494章·民国时期的老人

高志强之鬼被杨间封入黄金箱、送往总部地下后，曹延华虽不满其先斩后奏，仍承认目标已留在总部、杨间也答应走申请程序，决定十分钟后开会。杨间注意到秦老令自己的鬼眼本能回避，向曹洋探问其来历。曹洋只知卫景曾提及某位老人将过百岁生日，由此猜测秦老可能活过三年乃至生于民国。杨间联想弘法寺地下封印、观江小区民国老宅、鬼镜和驭鬼者笔记，意识到民国时期或许仍留有解决厉鬼复苏的线索，决定日后申请单独询问秦老。秦老是否真是民国驭鬼者、知晓何种历史仍未证实。（第494章）

## 第495章·总部大会

已恢复的刘小雨向杨间说明，自己被高志强控制时仍有意识，只是人格像被扭曲，因亲身经历而理解普通人面对厉鬼的无助；杨间则称高志强针对的是自己，未把救她视为目的。二人前往山体防空基地的总部会议室，顶尖驭鬼者围坐圆桌，杨间与冯全相邻，并遇到相貌无法被记住的李乐平。李乐平试探杨间的驾驭方式，杨间只以巧合和运气作答。姜尚白随后以高志强属于朋友圈为由要求交代，杨间强硬威胁敢挑事便打掉朋友圈；姜尚白虽被点明身份却未承认。曹延华、王小明和秦老尚未入场，会议暂未开始。（第495章）

## 第496章·没一个正常

等待会议时，杨间观察到与会驭鬼者各有尸化、纸钱折物、尸臭妆容或异常肢体，确认无人正常。会前王小明提出取消杨间的队长推荐名额、改给李军，理由是杨间更适合继续成长和处理他需要的事务；曹延华担忧此举损害总部利益，要求他给出足以说服人的解释，结果未揭示。会议开始后，曹延华说明鬼差已能压制至少十六只鬼、在一间房大小范围内会杀死鬼数不足的一方，并具鬼域内无法关押、会持续重启的特性。多数与会者因此畏惧。姜尚白提议拔出饿死鬼身上的棺材钉，杨间立即反对，认为即使成功压住鬼差，也无人能承担饿死鬼脱困的后果。（第496章）

## 第497章·调停

杨间以饿死鬼由自己关押为由，表示总部若采纳姜尚白的拔钉方案，自己就辞职且不再主动参与灵异事件；曹延华遂暂时搁置提议。姜尚白仍试图强调取钉成本低，杨间当场警告其必须独自承担失控后果，朋友圈数人继而声援姜尚白。杨间以椅子砸歪一名驭鬼者头颅，冯全、李军、曹洋和钟山先后表态支持杨间，双方形成对峙。秦老敲拐杖要求众人坐下，所有人竟不自主复位，连破碎椅子也恢复原状，冲突被强行压平。之后李乐平提出鬼差本质是鬼域，必须在鬼域外关押，令会议回到处置讨论。（第497章）

## 第498章·新的方法

李乐平的分析获王小明认可：鬼差的棺形鬼域就是目标本体，内部关押必会重启，唯有以另一片更强鬼域从外部压缩其范围，才可能装入容器。王小明据鬼棺能容纳大范围鬼域的现象说明鬼域间存在压制，并以此否决棺材钉能稳定钉住一片鬼域的设想。面对缺少压制目标的难题，他提出引来国外S级“鬼画”事件源头，让两件S级灵异相撞；若压制成功即可关押鬼差，失败也可能让鬼差解决鬼画。冯全介绍鬼画曾从画中走出杀人、疑似与艺术馆参观者有关，且身上被留下黄金烛台。杨间认出那或许是自己曾在大昌市见过的鬼域源头，认为计划极其危险；方案仍未执行。（第498章）
