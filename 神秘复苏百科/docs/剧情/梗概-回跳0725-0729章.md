# 原文回跳第725-729章剧情梗概

## 原文回跳第725章·突然的拜访

钱总带一群不请自来的客人进入张显贵聚会，其中西装驭鬼者王晗感应到杨间的位置，横冲直撞接近角落。江艳正要求杨间陪自己参加聚会，王晗抬手准备推开她时，杨间提前将她拉倒避开。张伟拔出两把黄金手枪指住王晗。王晗自称只是想认识杨间，却公开轻视普通人；杨间要求他下跪道歉，双方开始以态度和灵异能力互相试探。聚会宾客尚不清楚眼前的冲突已涉及驭鬼者与厉鬼力量。（原文回跳第725章）

## 原文回跳第726章·聚会上的试探

王晗称江艳只是可替换的“货色”，认为杨间应与自己一样漠视普通人。杨间反驳，即使自身冷漠也保留人类观念；张伟随即开枪命中王晗额头，伤口流出腥臭黯血却未使其倒下。王晗以灯光闪烁显露鬼域能力，判断若同杨间在人群中开战，宾客会瞬间死亡，遂选择暂退。杨间也因张显贵、王彬、张丽琴等人安全而没有压上鬼域。双方确认彼此危险，却暂时不撕破脸。（原文回跳第726章）

## 原文回跳第727章·入座

张显贵等人赶到，杨间将冲突称作圈内人打招呼，安抚宾客。马有财以鬼影换身近半年无异常的状态出席，令杨间确认活人头身拼接可长期生效。杨间随后不再扮作角落食客，以负责人姿态命众人入座；江艳和张丽琴分别跟在身侧，显示二人都与他关系亲近。王晗暂回钱总一方，张伟虽想继续动枪仍被制止。聚会从娱乐场转为杨间面对富商传达灵异时代信息的正式场合。（原文回跳第727章）

## 原文回跳第728章·背景

张显贵简短开场后把主导权交给杨间。杨间公开称世界将因灵异失控而改变，要求富商提前参与公司建设。樊宏嘲笑鬼故事、报警举报杨间，反被办案系统警告接受调查，随即道歉离场。杨间以此压下质疑，说明自己是大昌市负责人，公司不只是商业项目，而是建立安全屋、农业、仓储、商场和能源等城市生存网络。他把投资门槛定为十亿元，要求参与者以现实资产和资源换取未来的安全位置。（原文回跳第728章）

## 原文回跳第729章·破财免灾

十亿元门槛令多数富商犹豫。杨间指出黄金已被各国管制，十亿元甚至难建一处安全屋，现金购买力会在灵异时代快速下跌；王彬解释项目以分散建设防止单个农场、仓储被灵异毁掉。马有财率先投二十亿元，证明自己已把未来押在大昌市生存网络。王晗判断自己和钱总若不示好难以安全离开，遂让钱总同样投二十亿元。杨间以鬼域清空宾客、单独留下王晗，称投资买下二人的命，并开始追问王晗真正来访目的。（原文回跳第729章）
