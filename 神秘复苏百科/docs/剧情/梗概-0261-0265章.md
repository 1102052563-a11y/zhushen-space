# 第261-265章梗概

## 第261章·异动

杨间死亡近四小时后，尸体仍被鬼镜吸附，镜内却冒出大小不一、腐烂浮肿或焦黑的鬼手，不断拍击镜面并想把尸体拖入。镜中与杨间同形的鬼也等待借替换脱困，但无头鬼影占据尸体，令镜内鬼无法完成规则。黑色液体扩满房间后，鬼影爬上支架覆盖镜面，却同样不能穿入镜中；两边只能借杨间尸体拔河。尸体一度沉入镜面，鬼影又回缩将其往外扯，现场无人见证。安全屋内，王小明从这场碰撞想到新的驾驭鬼方式；张伟保管的人皮纸则撑开金盒，留下杨间可能成功却未带上它的字迹后重新沉寂。（第261章）

## 第262章·诡异的复活

僵持持续数日。鬼影先从房间退回杨间尸体，鬼镜则缓慢吞入尸身；第六日尸体半沉镜中，鬼影接管的鬼眼从皮肉钻出，以红光逼退镜内鬼手和同形鬼。第七日红光与鬼影先后沉寂，鬼镜出现裂纹，尸体落地时头部却已被镜面吸入。无头鬼影无法夺取头部，遂与鬼镜一同死机；杨间的头在镜内醒来，焦黑鬼手接近时被一只半温半冷的手折断。头部拉回镜外后方向反转，他以反关节手臂活动，确认自己复活，却没有完整身体和正常倒影。鬼镜内其他脚步与鬼物仍在，复活代价和后患未除。（第262章）

## 第263章·背尸

杨间失联七日，大昌市黑暗与阴霾已由中心扩至全城。王小明此前公开第三阶段鬼婴规律，广播让市民躲藏、静默并避开接触，暂留一批幸存者；但水电、食物短缺使拖延无从根治。城外指挥中心因杨间定位不动，开始讨论封锁、放弃城市的最坏选择。别墅中，杨间发现自己正面偏人、背面偏鬼，头颅反向、尸斑和尸臭不退；他掰正头部，取下暂时沉寂的鬼绳，遮住鬼镜回观江花园。另一边，王珊珊仍无心跳，王彬决定背女儿下楼火化，冒险进入楼道。杨间尚未追源头鬼，先准备确认留下的人是否还活着。（第263章）

## 第264章·鬼眼刑警上线

杨间回到观江花园，见整片小区同样断电、被黑青阴霾吞没，却从门窗紧闭判断别墅未被直接入侵。他锁好鬼镜、洗去部分尸臭，发现张丽琴仍在四楼独守七日，便用一只鬼眼开启红光鬼域检查楼内外，确认小区有不少鬼婴。穿上刑警制服后，他重新接通卫星定位手机，以“鬼眼刑警”身份上线。刘小雨确认信号后上报，总部与曹延华依王小明预案准备救援、安置灾民和刑警进城。杨间在鬼域中看见王彬背王珊珊尸体下楼、被两只第三阶段鬼婴接近，决定用这场遭遇检验自己复活后的力量。（第264章）

## 第265章·恐怖在消散

王彬背着王珊珊尸体走楼梯，先后听见上下夹来的脚步，见到青黑死人脸的第三阶段鬼婴后几近崩溃。杨间以鬼域将他、尸体和两鬼婴转到路面，徒手拆下其中一只头颅，却发现单纯肢解无法杀死衍生鬼。随后他让鬼影力量侵入，鬼婴立即化为黑青阴霾消散；第二只也同样被处理。杨间因此判断衍生鬼依赖鬼婴力量，鬼影可将其驱散，并以鬼域把全小区数十只第一至三阶段鬼婴集中清除。他虽暂时恢复观江小区安全，却知道全城鬼婴无数，决定停止局部扫荡，直接寻找真正的源头鬼。（第265章）
