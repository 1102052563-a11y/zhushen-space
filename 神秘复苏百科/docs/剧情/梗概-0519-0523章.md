# 第519-523章剧情梗概

## 第519章·询问

杨间在培训基地屋顶等待鬼差行动结果，近一小时没有消息，担心鬼画与鬼差失控后大京市重演大昌市灾难。柳三以纸人替身从墙外爬上屋顶，请教黄岗村鬼棺与鬼差资料；杨间说明自己曾强行取走鬼棺、使二者分离，却不能确定现今鬼差位置。柳三披露行动队已在鬼画鬼域内发现被分割成黄岗村的鬼差鬼域，村中每间房都可能有被鬼差夺走后重现的鬼，甚至包括杨间的鬼绳；他们去过原鬼棺房屋并关押一只鬼，却未找到鬼差。杨间提醒其不能只盯着鬼差而忽略鬼画。纸人离开后，杨间才确认来访者并非柳三本人。村内的李军等人据地图发现黑房连成棺形、灰白道路分割两种鬼域，明白平衡随时会因鬼画规律破裂，只能继续寻找目标。（第519章）

## 第520章·来过的地方

朋友圈在平安大厦监看鬼画行动，费总认为可趁杨间在培训基地时杀人并嫁祸事件，方世明却拒绝仓促动手，要求等杨间真正卷入灵异事件、再借厉鬼与自身手段确保击杀；姜尚白则期待行动失败后重提拔饿死鬼棺材钉。黄岗村内，李军带五人已排查多栋房屋，虽然未减员，鬼火过度使用令他身体焦黑、濒临复苏。面对苏凡劝退，他承认鬼差外逃责任与自己有关，仍决定为唯一关押机会继续。苏凡游戏提示二楼危险，柳三先行探查，在门口看见带灰白灰烬的凌乱脚印和一批破烂纸人；纸人全是柳三容貌且像由他亲手制作。柳三意识到自己可能已在没有记忆的情况下来过此地，并已被杀死多次，黄岗村的搜索由危险变成无法验证记忆的异常。（第520章）

## 第521章·掉落的画像

柳三清点二楼房内残破纸人，确认共有五具，按其“被杀一次便由一具纸人代死”的能力，推测队伍可能已经五次到过此处、失败后被某种力量重置记忆；苏凡与李军虽不认同，却也无法解释纸人来源。苏凡主张在记忆已不可信且鬼画仍在的情况下撤退，李军则认为鬼差鬼域已被压制，若此时放弃，背后的大京市将再无可承受的关押机会，决定继续拼命。隔壁传出僵硬脚步，柳三原想独自开门，李军为改变可能重复的行动规律，先点燃鬼烛并率五人破门而入。房内没有鬼，只剩家具农具；墙上一张老旧老人遗像忽然掉落。遗像老人半脸在黑暗中、嘴角像带微笑，旁有模糊字迹。脚步来源、纸人与遗像的关系及此前五次到访是否真实，均未解开。（第521章）

## 第522章·诡异的背景

众人认定掉落遗像并非档案中的鬼差，鬼烛又燃烧平稳，柳三便按李军安排以黄金手提袋收容。苏凡游戏角色已无端死亡，浓妆女子也指出行动面对的可能不只是鬼差。柳三擦去遗像灰尘后，发现老人身后背景正是他们所在的黑暗民房，画中还站着左二右三共五个模糊人影：蜡黄脸、绿火鬼烛和手机等特征对应行动队。李军命立即封袋离开，避免继续研究。五人走出大门，却看到前方一名僵硬老人背影，黑房与五人站位同遗像完全一致；鬼烛随即暴燃，显示鬼已出现。苏凡由此理解自己的游戏为何每次都会死亡。鬼域外，杨间等待三小时后判断行动大概失败，准备离开做最坏准备；但他尚未得知队伍遭遇何种异常，老人是否从遗像而来、遗像能否收容均无答案。（第522章）

## 第523章·走出的五人

鬼差行动失去联系三个多小时，总部会议收到报告：李军、柳三、阿红、苏凡、徐一平五人已失联。沈良建议取消行动、带走鬼画以减少损失；赵建国认为王小明应有后手，王小明则称撤退方案取决于李军执行，并要求总部更注意鬼域外是否有扩散风险。沈良已安排钟山、郭凡与本市负责人陈义在外围戒备，曹延华决定继续等待。与此同时，杨间察觉灰白鬼画鬼域迅速消散，连棺形鬼差鬼域也一并消失，起初以为关押成功。中心处却走出五道人影，杨间认出李军与柳三，却发现每人手掌都有相同灰白空洞，像被利器贯穿。他以白色鬼烛验证，五人同时僵硬转头望向烛火，只能立刻吹灭，并判断走出的五个已全部是鬼。原行动者的生死、鬼画和鬼差的实际去向、五个鬼的杀人规律均未揭示。（第523章）
