# 第545-549章剧情梗概

## 第545章·苗头

李瑶因无头尸体起身而逃往杨间所在楼栋。围观居民只把异状理解为行为艺术或仿真人，直到有人摇落尸体颈上的画框，才发现尸体头颅消失、断口持续流血。掉落油画画着昏暗都市、五官模糊而双手清晰的女子，角落还多出一颗现代发型的人头。杨间隔楼看见现场时，鬼眼自行闭合，和先前不敢直视鬼画的反应相同；他据李瑶所见“油画吞掉头颅”确认是鬼画。陈义、姜尚白、张雷随后带队隔离现场。陈义称这并非首幅同类油画，已收集三四幅，且有扩散趋势；杨间判断真正源头尚未现身，选择不参与处置。（第545章）

## 第546章·贺天雄的家

鬼画在市区出现令杨间加快处置朋友圈核心的计划，他带李瑶前往贺天雄的一处别墅。刘小雨来电希望他留在平安酒店、应对可能爆发的鬼画事件；杨间以不应连续承担外辖事件为由拒绝无条件参战，最后承诺留五日，五日内爆发才会帮忙。别墅内外大量使用黄金装修，令杨间鬼域不能穿墙搜全屋；贺天雄又靠养犬与暗门躲藏。杨间以鬼域使屋体摇晃相逼，贺天雄才在三楼现身。杨间明说费总、张建已死，自己是来杀他；贺天雄由此意识到朋友圈机密情报已经泄露。（第546章）

## 第547章·肩膀上的人

杨间借档案与鬼眼看见贺天雄肩上站着一具只露下半身的尸体，确认其代号“鬼踩人”：只要目标作出抬头看鬼的动作，鬼便会从贺天雄肩上走下杀人。杨间避免抬头，使贺天雄无法主动指定鬼袭击自己；贺天雄因此认定朋友圈有内鬼。面对杨间提出的五分钟生死试验，他表面同意、立即逃入黄金隔断的房间，企图拖延。杨间不愿让肩上鬼脱离控制，又顾虑方世明及未知第二鬼，遂以鬼域追踪而不直接拼命。贺天雄进入安全屋前丢下某物，杨间以鬼影和鬼眼锁定其位置，准备发动五层鬼域。（第547章）

## 第548章·一次替死

杨间以五层鬼域红光覆盖贺天雄，原本足以送走张建的攻击却被一只破旧布娃娃承受：布娃娃大半躯体消失，贺天雄本人无损。杨间据此确认他消耗了替死布娃娃，而贺天雄也意识到再无第二次保命物。杨间准备再用五层鬼域时，鬼眼连续高负荷使用出现刺痛与不稳，判断短时间两次已近极限，第三次可能导致三鬼平衡被破坏。贺天雄躲进安全屋，露出其腐烂胸腔：第二只鬼是一块会替换自身血肉的死人肉，他以金钉勉强维持身体。杨间停止维持鬼域，改以鬼影破门逼近。（第548章）

## 第549章·关键的电话

贺天雄回忆自己先驾驭腐肉鬼，虽可在坠楼、火化后复活，也能使鬼不袭击腐烂部位，但代价是全身不断腐烂；肩上鬼只能延缓腐烂，不能由他指定杀人。安全屋门被鬼影侵入后，贺天雄双腿先失去知觉，身体逐渐被鬼影夺取和肢解，胸腔腐肉则暂时挡住侵蚀。五分钟将尽时，苗小善来电称室友当日去美术馆带回一幅油画后失踪；杨间立即要求她远离油画、到空旷处等待。为赶往学校，他撤走鬼影并离开贺宅。贺天雄瘫坐存活，明确知道自己逃过一劫并非靠实力，而是电话带来的新灵异风险。（第549章）
