# 第615-619章剧情梗概

## 第615章·新的陷阱

杨间从人皮纸新浮现的字迹得知，八音盒诅咒的解法竟指向王小明。纸面称他在绝望时会被王小明救下，杨间却没有把这当作保障：公交车与凯撒酒店的既有路线已被否定，鬼橱仍未完成，而人皮纸过去给出的真实信息常藏有致命代价。他由王小明曾借纸半小时、卫景后来死而复现，推测王小明或许已借某件应验之事开始相信纸面，但这只是推测，无法确定卫景复现是否相关。杨间担心纸既可能诱使自己误解王小明，也可能借王小明实施更深陷阱，遂将纸埋在八号别墅附近地下，转向鬼橱。另一边，王小明以准备解咒为由带卫景离开曹延华车队，目的地和所需物品未说明。（第615章）

## 第616章·小明的问候

杨间以鬼域秘密回到大昌市观江小区钟塔，向鬼橱塞入“我找到了”的纸条，试图用模糊回答提前完成一年内寻找照片女子的交易，再要求解除八音盒诅咒。鬼橱立刻否认，回纸称他“没有找到，只是看到了”，追问后也只提示还剩九个月零三天。杨间据此将候选收窄为鬼画中的鬼和灵异公交车上的干尸新娘：二者都像他见过却未真正找到的对象。他无法承受登车追查的风险，便决定赌鬼画方向，并把行动期限从诅咒剩余三天压缩为两天，以给王小明方案或公交车留下余地。他确认母亲仍在附近后没有相见，立即折返大京市。与此同时，王小明定位郭凡住处，发出约见信息。（第616章）

## 第617章·借东西

王小明到郭凡住处时，郭凡正与钟山喝茶。王小明没有追究此前引鬼失败，而是先指出郭凡厉鬼复苏状态极差，提出可帮忙解决，条件是借其腹内灵位。郭凡说明灵位内有鬼，自己靠定期放出它抵抗另一只鬼侵蚀；若失去灵位便会迅速死亡。王小明进一步要求他再次进入灵位，待关押其中鬼后再换一副身体，郭凡因无法确认意识转移与返回可能而拒绝。王小明随即表明会自行取物，屋外中午骤暗，黑暗从缝隙侵入。钟山意识到这是预先布置的鬼域，郭凡则看出退路已被封死；王小明不掩饰强制意图，只表示郭凡若肯交出灵位，仍可能有活下去的机会。（第617章）

## 第618章·最好选择

郭凡面对强取威胁想挟持王小明逃往国外，钟山先联系曹延华求援。曹延华判断灵位关系杨间的解咒方案，未叫停行动，只请郭凡顾全大局、又要求王小明尽量保命；王小明切断通话后称这是私人行动。郭凡动手瞬间，卫景从黑暗中以发黑鬼绳勒住其颈部，压制体内灵异并将其拖向黑暗。看见外貌近似鬼差的卫景后，郭凡放弃抵抗，取出腹内木质灵位。灵位中的郭凡意识与鬼发生替换，照片内留下惊恐的郭凡，鬼则占据肉身；卫景再以鬼绳吊住鬼化身体，王小明取走掉落的灵位。钟山无法阻止，只得到郭凡大概率会死的答复。（第618章）

## 第619章·未知的隐患

杨间回到大京市，准备检查曾被自己鬼域隔绝、疑似存在鬼画的地点。他在胡同内一座反锁四合院发现鬼眼没有受压制，屋内却有一名死亡数日、缺失一只手的住户，因而判断并非鬼画，而是未被记录的另一场灵异。杨间让路过的遛鸟老人报案，本欲离开，却由断手联想到曾被自己释放的魂瓶内鬼和此前离奇死亡的张辉，推测可能有鬼已潜入市区杀人；考虑到已有驭鬼者遇害，他私下估作至少B级，但为赶在八音盒期限前找鬼画而没有处置。陈义接报后封锁现场，并得知杨间曾来过；无人察觉阴冷气息可能已随入屋者离开，新的隐患由此未被控制。（第619章）
