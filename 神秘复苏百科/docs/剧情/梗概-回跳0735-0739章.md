# 原文回跳第735-739章剧情梗概

## 原文回跳第735章·负责人的碰面

杨间带李阳抵达大汉市，孙瑞亲自以高规格接机。孙瑞表面热情，实际担忧杨间跨市是为找麻烦；杨间明确只调查线索，暂缓冲突。孙瑞介绍自己曾与冯全同批负责城市，得知冯全将加入杨间队伍后认可其价值。杨间携带旧报纸、鬼烛、替死娃娃、鬼瓷、黄金容器和柴刀等装备，李阳则只有配枪和普通行李。二人没有直接赴坐标，而是先借孙瑞熟悉大汉市的优势了解当地异常。（原文回跳第735章）

## 原文回跳第736章·烂尾楼

孙瑞承认大汉市并非绝对平静，约三四个月来每天约有九人神秘失踪，受害者初期尚能联系、随后彻底消失，故被内部列为C级“大汉市神秘失踪事件”。杨间判断大概率是灵异问题，拿出木板坐标，孙瑞带他前往建设路。坐标对应一栋曾多次施工事故、无人能解释的烂尾楼。孙瑞此前多次查过该楼却没有发现，杨间仍认为普通调查不足以排除灵异。（原文回跳第736章）

## 原文回跳第737章·一切的起点之地

杨间发现烂尾楼共五层，与旧论坛“雷电法王”所述敲门鬼老人五层坠楼相符，推测这里是鬼敲门最初爆发地点，也可能关联民国驭鬼者遗留秘密。他要求孙瑞封锁街区、征用酒店、布置观察者和狙击手。孙瑞接受，并将隔离范围扩大。等待封锁期间，杨间以两尊鬼瓷分别绑定自己和李阳身份证，作为进入楼内前的额外保护。傍晚封锁完成，三人准备进入烂尾楼。（原文回跳第737章）

## 原文回跳第738章·强行闯入

烂尾楼内表面正常，孙瑞一度认为没有问题。杨间却从墙壁、水泥地和天花板取出金戒指、项链和手镯，证明失踪者到过此处但黄金被留在现实。他判断烂尾楼只是与另一鬼域重叠，不是鬼本体所在。杨间连开五层鬼域：第三层令烂尾楼消失，第四层显出迷雾中的民国建筑，第五层强行入侵，牌匾“鬼邮局”现形。三人进入时，顶楼窗户打开、黑长衫阴影与诡异脚步回应闯入。（原文回跳第738章）

## 原文回跳第739章·楼内

鬼邮局一楼陈旧阴冷，鬼眼受压制，杨间至少须维持五层鬼域才能行动。楼内有五层回字走廊、每层七间编号房、不同年代黑白人物像，却没有通往二楼的楼梯。11号房走出持枪男子万兴，称杨间是新人；他开枪后被杨间轻易控制，孙瑞以金杖压住。房内又走出一名未具名女子。杨间从万兴对邮局的熟悉判断二人是信使，意识到此前李跃所说、无法定位的诡异邮局终于被找到，遂以生死威胁要求万兴交代情报。（原文回跳第739章）
