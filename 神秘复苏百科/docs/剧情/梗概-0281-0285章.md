# 第281-285章梗概

## 第281章·叶真

林落梅的喊话刚触及杨间，罗素一三人便被他以鬼绳、鬼影和鬼域迅速制住。黄飞解释三人来自灵异论坛，只是受管理员委托见面；杨间不接受夜访理由，借电话向管理员叶真限时十分钟索要十亿元。大海市的叶真起初欲报复，中年下属却看出这是对论坛财力、情报与决断的试探，遂劝其付款。杨间扣住三人为筹码，意在为饿死鬼事件后处于风口的自己树立强硬边界。论坛最终在八分四十二秒把钱汇入其母亲账户，证明组织能快速定位保密亲属资料并调动巨资。杨间警告不可再夜访，随后放人。（第281章）

## 第282章·不丢人

杨间离开后，罗素一、林落梅和黄飞恢复行动，却发现自己已在大昌市外郊区。黄飞判断杨间早将三人转移到远离城区的位置，若谈判失败便会直接杀人，避免驭鬼者死亡后的厉鬼在城内复苏；罗素一也因此确认杨间拥有鬼域。三人承认接触计划失败，决定暂不再招揽这个危险的刑警。杨间遵守收到十亿元后放人的承诺，回到别墅用“快递”掩饰夜间冲突。张丽琴因无法继续做汽车销售，提出在杨间未来的公司任职；杨间认可应建立自己的势力，准备向张显贵请教，并暂定她为秘书。（第282章）

## 第283章·灵异论坛

杨间在体育馆会议前向赵建国追问灵异论坛。赵建国说明其五年前只是鬼故事兴趣站，灵异事件爆发后成为幸存者与早期驭鬼者的交流地，继而发展成涉及金融、建筑、医疗、运输的亚洲顶尖势力。总部曾试图收编，被管理员拒绝后选择以约定默许其暗中发展。现任管理员叶真已成为驭鬼者一年以上，曾独自处理A级事件，实力位列亚洲前列。赵建国劝杨间不必加入民间组织，只要立功提升权限便能接触更多档案；他也预警会议中的刑警可能不服杨间。杨间与童倩重逢，发现她鬼脸复苏加剧。（第283章）

## 第284章·熊孩子

总结会议开场前，灵童刑警熊文文以预知未来一小时的能力自傲，辱骂未能进入大昌市的刑警。曹洋故意以言语激他，熊文文不顾保护人员阻止而发动厉鬼能力，脸色死灰、会场灯光昏暗。他预见某个可怕结果后被吓哭、退离门口，众人误以为有鬼接近；进门的却是杨间。曹洋从反应推测，若争执继续，杨间会危险介入，便未再挑衅。赵建国介绍熊文文的预知限制，王小明补充其只能看一小时内身边的事。曹延华随后召集众人入座，宣布由王小明和杨间总结饿死鬼事件。（第284章）

## 第285章·大忽悠

曹延华强调S级事件处理能力关系国家国际地位，并将报告交给杨间。杨间怀疑总部想借会议探查自己死而复生和人皮纸的秘密，故刻意谦称全靠王小明计划、自己只是幸运地活到最后。曹洋试探两鬼驭鬼者能否接触饿死鬼，杨间反称只要懂规律和有特殊能力便能轻松关押，还把饿死鬼淡化为最多A级的“鬼小孩事件”。部分刑警因此轻信，认为危机只是前期扩大所致；曹洋则以叶枫死亡为例看出其话中有坑。王小明不拆穿杨间，会议在杨间继续隐藏底牌时尚未结束。（第285章）
