# 第256-260章梗概

## 第256章·电话会议

杨间回观江花园时，安全屋主体虽成，工人却因恐惧逃散，配套未完只能维持约十日。市中心鬼婴预计三日内可能逼近小区，杨间遂确定十九个避难名额：按原承诺留出三个给总部，另给王小明、江艳，自己不入内，以免复苏害死所有人。随后他在五楼要求总部召开紧急会议，并单独叫王小明上楼。江艳因杨间把名额给她却不自保而哭求他进入安全屋，二人谈话被王小明打断。杨间未说明完整计划，只表示会议后王小明应老实进入安全屋；他已决定把能否救城的判断交由总部参与，同时准备承担不进安全屋的风险。（第256章）

## 第257章·最后的夜

赵建国接到会议要求后，迅速组织刑警总部副部长曹延华、李军及多名保密电话参会者连线。杨间先说明解决饿死鬼须具备“接触”和“关押”两条件：他可用不足两成把握再次正面接触源头鬼，但鬼烛已尽，仍缺能够限制目标的物品。王小明替其补充限制物失去的经过，杨间则坦言大昌市有一件可能替代物，却被用来钉住另一只未知高危鬼；若取走它，等同放出可能不低于A乃至S级的恐怖。总部须决定是否承担这一后果。杨间挂断电话后告知王小明，明日避难者入安全屋时自己将开始计划，并以手机视频让其观看结果；他召开会议是要让国家承担释放未知鬼的后果，而非独自背负。（第257章）

## 第258章·上吊自杀

安全屋完工后，十九名名单人员开始进入，张伟代杨间保管人皮纸，约定若第九日仍未归来便交给王小明。杨间把鬼镜和复苏鬼绳带到张伟市中心别墅的空房，布置手机视频，让王小明远程观看；他将鬼绳一端系上吊灯、另一端套在自己颈部，黑布遮住鬼镜并避免手机摄像头照到镜面。刘小雨转达曹延华已批准行动、国家承担后果，杨间仅让她转告母亲自己安好。倒数后，他揭开鬼镜黑布、放无头鬼影进入身体并解开鬼绳金箔。鬼绳立即收紧，他在鬼影令身体失去心跳、血流和行动力的状态下被吊起，镜中同形鬼也随之模仿上吊；杨间最终死亡，计划转入鬼与鬼之间的未知阶段。（第258章）

## 第259章·死去的杨间

王小明从安全屋视频中看见杨间死亡，却不知鬼镜规则，只能推断绳子、无头鬼影和镜子被刻意构成死局。镜内同形鬼逐渐完整，杨间让鬼影侵入身体、鬼绳勒杀自身，试图同时满足多种厉鬼规律；鬼绳吊住尸体，使鬼影暂时无法取得身体，二者开始无形较量。上吊约三小时后，尸体脚尖滴下浓黑液体，显示鬼影复苏并压制鬼绳；鬼镜也开始介入，试图排除尸体中的鬼影以按其规则复活死人。王小明只察觉尸体位置出现微小变化，无法确认过程是否通向复活，仍持续守着电量不断下降的手机观察。（第259章）

## 第260章·尸体的异常

杨间死亡约四小时后，观江花园内的王珊珊因鬼奴印关联骤然倒下，体温和心跳均消失，王彬夫妇据她先前“杨间死我也会死”的说法推断杨间已死。张伟别墅中，杨间尸体滴落的浓黑鬼影越来越多，尸体开始摇摆，最后双脚触及鬼镜并被镜面吸住，形成斜挂姿势。王小明由此看出杨间是在制造鬼间冲突：鬼影复苏后覆盖鬼绳，使其失去力量并反受鬼影控制；鬼绳完成吊死任务后垂下，尸体随即被鬼镜吸附。杨间企图让鬼影与鬼镜形成逻辑死循环、同时死机，继而以鬼镜规则从死亡中复活。手机恰在尸体撞向镜面、关键过程开始时耗尽电量，复活是否成功未写。（第260章）
