# 第346-350章梗概

## 第346章·找人

杨间为寻找篡改记忆鬼，来到中山市城郊的罗素一住处。罗素一、黄飞和林落梅均为灵异论坛会员，原受邀支援却因童倩获救、郭凡冯全被困而判断事件恶化，准备连夜离开。杨间以鬼域潜入别墅，说明只需林落梅用能寻人的鬼音找出目标，自己负责处理；罗素一以能力使用次数有限、连杨间也感到棘手为由拒绝。杨间随即提出征用林落梅体内厉鬼的能力，明确这是命令而非协商。三人曾在大昌市被杨间压制，对其手段心存忌惮，却仍未立刻答应。杨间的要求是否会获总部授权、林落梅是否被迫参与，至本章末未定。（第346章）

## 第347章·征调和变化

面对罗素一和黄飞的抗拒，杨间打开卫星定位手机让总部监听，赵建国随即确认征调有效，称若三人强拒可特事特办，由杨间处理后果。杨间说明自己不是逞强，而是把手中资源用于求生；林落梅见同伴无力保护，借播音器答应同行。罗素一最终为保护林落梅加入，黄飞则按三人至少留一人存活的约定留在郊区别墅。杨间以鬼域带罗、林到临时安置点，要求二人在行动前不要接触他人、只信自己，以防鬼奴渗透。与此同时，第三医院的童倩在无人察觉时睁开空洞双眼，嘴角出现僵硬职业笑容；她是否恢复、是否被改写记忆或将做何事均未揭示。（第347章）

## 第348章·鬼雾笼罩

次日八时，杨间带熊文文、罗素一、林落梅驶往凯撒大酒店，准备第二次行动。酒店却被只覆盖建筑本身的浓雾笼罩，雾内灰冷厚重，能遮蔽视听并使人失去方位；杨间由冯全“鬼雾”代号及其黄岗村表现，怀疑冯全已死、体内厉鬼复苏。张高报告，凌晨一点四十分至两点酒店曾持续传出呼救，之后二至四楼全亮且窗前有人影；四时起鬼雾出现并持续扩散，二十四小时内可能越过警戒线。杨间认定多起厉鬼问题叠加，要求外部准备最坏结果。罗素一提议撤退，杨间虽更谨慎，仍未放弃处置。（第348章）

## 第349章·鬼雾中的人

杨间没有让四人直接闯入鬼雾，判断现有资源若一次失败便难再重组，决定先独自进入降低酒店危险，再视结果决定是否集体行动。他令熊文文、罗素一和林落梅退至警戒线，无论听见自己求救也不得进入，并警告罗素一不可趁机逃离。雾内如鬼域般隔绝方位，杨间以一层、两层鬼眼鬼域分别驱开不足三米、约十米范围，确认自身鬼域受压制；后脑鬼眼发现有模糊人形一路尾随，目标又随雾消失，他未为不明对象偏离寻找源头鬼的计划。穿行近十分钟后，他终于抵达凯撒大酒店门前。（第349章）

## 第350章·诅咒对诅咒

杨间进入酒店后发现二楼尸水脚印增加，判断一楼暂时相对安全、二楼活动鬼和四楼失联者仍构成威胁；随深入，外部鬼雾反而变淡，他据此警惕酒店内有更强存在限制鬼雾。他用鬼域穿墙抵达监控室，准备以酒店广播播放保留的敲门鬼敲门音频半小时，引敲门鬼进入，让厉鬼对抗酒店内鬼物，并放弃为郭凡冯全破坏计划。播放后他迅速撤出，鬼雾隔绝声音，故诅咒未向外扩散；雾中人形又接近，他仍保留鬼雾作为封锁手段而未接触。敲门鬼是否到达、酒店内鬼物会受何影响，均留待后续。（第350章）
