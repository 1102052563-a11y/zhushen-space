# 第176-180章梗概

## 第176章·救人到底

杨间电话命令江艳开门后，以点燃鬼烛站在她的车外，烛光驱散黑暗，江艳与女销售才发现车辆早停在路边，先前的无尽行驶全是鬼域幻觉。女销售手腕留有被鬼抓出的淤血掌印，暂未死亡；江艳因获救抱住杨间，杨间将两人安排进后座，熄灭鬼烛后开车折返童倩所在区域。路上有腐烂尸体接近，杨间以威胁制止女销售尖叫。他决定为向赵建国交代再带一批幸存者离域，限三分钟集合；童倩带十余人从商铺中出现，接受其安排。幸存者数量比先前更少，余人因鬼域路径变化而失散。杨间尚未说明离开鬼域的方法，救援队在本章末仍停留在黑暗街道。（第176章）

## 第177章·王岳出现

杨间把救出童倩视为任务底线，仍与她争论刑警责任和普通人价值。一个背包大妈想携带鬼域中搜来的“商品”同行，杨间以鬼眼看出其背后实为鬼物，警告无效后她被突然出现的麻绳吊起。王岳从店铺走出，自称同在鬼域受困，请杨间带他离开；杨间认出他可能是马有财供出的玫瑰酒吧老板。王岳承认自己从敲门鬼手中幸存，提议互相帮助，却未说明能交换什么。大妈遭吊起后生死未明，童倩、江艳和十余幸存者也仍困于王岳与杨间的对峙现场。（第177章）

## 第178章·鬼绳

杨间由王岳吊起大妈的手法联想到严力一家被吊死，逼问后，王岳承认杀人却推给赵开明指使，并称赵开明曾计划让自己与王小强联手对付杨间；他改口愿投靠杨间以换取离域。杨间认为王岳杀害严力一家、又参与谋害自己，拒绝和解并胁迫童倩协助。王岳逃跑时被特制手枪击倒，露出颈部打死结的老旧麻绳，正式显出“鬼绳王岳”的能力。他令鬼绳套上杨间颈部，鬼眼暂时抵抗后仍难完全脱身；王岳不断收紧自身绳结，企图在厉鬼复苏前拉杨间同死。冲突进入生死相搏。（第178章）

## 第179章·王岳之死

鬼绳在杨间颈部收紧，令他几乎窒息；他以两只鬼眼暂缓绞杀，再召无头鬼影扯住绳索。王岳孤注一掷继续催动鬼绳，自己却先被绳结吊起，尸体逐渐苍白悬空；杨间判断鬼绳已开始脱离控制，遂要求童倩提前杀死王岳。童倩解开头发，以后脑鬼脸怪笑令王岳带着同样笑容断气。鬼影将杨间颈部绳索扯落，但王岳尸体仍被悬挂，随即天空垂下无数绳套，已有四五名幸存者被吊起。王岳死后鬼绳彻底复苏，敲门鬼鬼域内又多出一场大规模灵异危机。（第179章）

## 第180章·关押鬼绳

王岳死亡令鬼绳在街道、商铺与车厢大量垂下，数量至少数万，普通人一触即被吊起。杨间认定应立刻关押真正鬼绳，既消除危险也取得可绑鬼的灵异物；他点燃鬼烛，烛光荡开绳索并令被吊幸存者落地。童倩交出黄金密封袋并接过鬼烛守护众人，杨间带无头鬼影接近王岳尸体。鬼影先破坏仍被鬼操控的尸体，再取下其颈部鬼绳；此时敲门鬼黑暗逼近，鬼绳试图勒住无头鬼影，却因其无头而滑落。杨间将鬼绳收入黄金袋封死，完成关押，但众人尚未离开鬼域。（第180章）
