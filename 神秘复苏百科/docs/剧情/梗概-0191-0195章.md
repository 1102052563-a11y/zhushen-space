# 第191-195章梗概

## 第191章·引鬼入镜

虎哥在直播中向张伟屈辱道歉后离开，杨间关闭直播，转而带张伟回荒宅二楼处理镜中鬼。杨间通过木门房墙上的小孔、远处大楼玻璃和张伟曾在工地野外如厕的位置，将零散线索串起：鬼镜原先正对洞口，借大楼反光照到张伟，镜中才出现其模样。张伟站到鬼镜前仍无倒影，片刻后镜中角落出现惨白手臂，与张伟相同的鬼从镜内大门方向缓慢走来。杨间命张伟不要动，意图让这只鬼完整回到鬼镜，验证它如何离开；张伟虽害怕，却以快速结手印故意令镜中鬼难以同步。至章末，鬼已逼近镜中张伟，杨间仍在观察，尚未证明镜内外联系能否切断。（第191章）

## 第192章·未斩断的联系

杨间确认跟随张伟的鬼来自鬼镜，判断其杀人需要与镜外人动作一致、完成替换，条件虽苛刻却不能说明整面镜子安全。他让张伟以快速不同手势破坏同步后将其带离镜前，鬼却留在镜内不再跟随。杨间由此推断鬼镜中的鬼必须借另一面镜子或反光媒介离开，原先小孔正对远处玻璃才把张伟的镜中鬼送出；油布遮镜亦是为了防止这种转移。因人与镜内鬼仍一模一样，他认为联系只是受限未被斩断，遂盖回油布、扛走鬼镜，并警告张显贵工人可能已被镜子吞入。杨间要求加固荒宅、封住出入并可建庙遮掩，避免其余两门因房屋倒塌而暴露。（第192章）

## 第193章·心中的疑问

回程中，杨间告诉江艳灵异全面爆发，观江花园因人少、位置适合且由本地驭鬼者守护，仍比市中心更适宜居住；他命江艳到张显贵售楼部管理自己四成股权。三日间，他补充特制子弹、尸袋、金箔与熔金工具等物，花费近六千万，江艳开始上班，荒宅也在张显贵监工下昼夜维修并筹建庙宇。杨间通过观察确认鬼镜内的张伟鬼始终未再离镜，强化了“不能以另一镜面照它”的判断。他把鬼镜安置在别墅五楼，换支架、罩厚黑布，并将墙窗涂黑防反光。为查鬼镜及剩余两间房，他从黄金盒取出人皮纸；纸只重复已知发现，未给关键答案，杨间怀疑它拒绝合作。（第193章）

## 第194章·死人复活？

杨间威胁要把消极的人皮纸焊进盒子沉入深海后，纸面改写，称鬼镜有一种匪夷所思的能力，能让死去的人“复活”。它随即限定这不是真正复活，而须付出释放镜中一只鬼的代价；杨间既看到绝境求生时可能值得交换的诱惑，也警惕这是借人求生欲放鬼的陷阱，决定不能尽信也不能不信。人皮纸又建议永封荒宅另两间房，称门后有未知恐怖，杨间认为其仍可能隐瞒。与此同时，复生后的王小强带叶枫和车队抵达观江花园；叶枫坦言鬼域难以正面处理，只能突袭。王小强打开研究所带来的金箱，展示一根比例失衡、黑指甲的干枯手指，称它是鬼烛的衍生失败品，并决定展开行动。（第194章）

## 第195章·猎杀计划

王小强带叶枫及二十余名受训人员进入观江花园，要求找到杨间前不得惊动目标，泄密或引起注意者可直接杀死；人员带特制武器，伪装游客、购房者和工人分散搜索。孔峰、混血雇佣兵黛西等人到售楼部，借“杨间介绍”套出杨间住在旧售楼部改成的庄园别墅，遂通知队伍聚集。江艳察觉来客像保镖而非买家，却暂未采取行动。张伟开启直播，计划带观众看鬼镜，跟拍孔峰一行而被视为干扰；阿海摔碎其手机、以匕首刺入其颈部并拖入绿化带。张伟却带伤持特制手枪反击，近距离击中阿海后再次倒下。孔峰误判张伟只是回光返照，命人藏尸并继续三段猎杀计划，枪声和张伟安危均成未解变数。（第195章）
