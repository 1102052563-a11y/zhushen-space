# 第605-609章梗概：柴刀媒介与方世明之死

## 第605章·影子的身份

杨间被柴刀拦腰斩断后身体迅速冰冷、心跳停歇，却仍因八音盒诅咒保持意识。他看见持刀高大尸体远去的背影与无头鬼影轮廓几乎完全相同，回头又发现无头鬼影也随自己从腰间断成两截。鬼影颈部原有断口和新腰部断口同样平整，令他推测完整鬼影曾被柴刀肢解：腐烂尸体可能是身体、无头鬼影是残躯，头部则遗失在别处；这只是现场推测。杨间以鬼域拼接身体和影子，但鬼影恢复很慢，双腿一时不能动。他冒险从十米逼近至五米，尸体始终未袭击，因而暂将规律判断为与直接接触有关，决定改用随身裹尸布限制目标、夺取柴刀。（第605章）

## 第606章·裹住不动

杨间戴黄金手套取出裹尸布，布上原本的人形轮廓在展开时仿佛扭曲、伸臂，显示其本身另有危险。他将布从高大尸体头顶放下，布中人形如附身般爬行并缓慢收紧，尸体仍能移动，杨间只能跟随等待。尸体带他走到敞开的82号房前，门把手留有鲜红指痕；房内民国式装修与71号房近似，却没有钢琴，墙壁和天花板布满血手印。尸体终被裹尸布完全包住而停下，杨间虽不能用鬼域覆盖它，仍亲手扛起。此时82号房血手印已蔓延至走道墙壁，正向他靠近；他携带被限制尸体，陷入未知鬼追来的危局。（第606章）

## 第607章·黯淡的灯

杨间扛尸折返，选择直接穿过布满血手印的走道，八音盒诅咒令他敢于承担风险。手印一路尾随到出口附近，他先隔黄金手套强取柴刀失败，改以鬼手触碰，裹尸布内尸体剧烈挣扎，柴刀却自行脱手。为验证媒介，他以手掌覆盖墙上血印，眼前浮现不同年代的无眼腐烂人群：他们贴墙摸索、以血手留下印记，接触后又集体伸手抓他。杨间挥柴刀砍向最近手臂，众多手臂同时断裂，走道血印迅速消散回82号房；房门猛然关闭，门灯闪烁后近乎熄灭。他获得柴刀的媒介用法，却未确认房中鬼是否真正被关押。（第607章）

## 第608章·堵路

血手印消失后，杨间没有立刻被柴刀的威力冲昏头脑，检查发现融合鬼差部分血肉的黑色手臂已褪白近半，意味着首次挥刀消耗了一个压制厉鬼名额。他原靠三个名额压住鬼眼和鬼手，如今只剩两个，且这种损失不会自行恢复。考虑异常走道内还有可能开门外出的鬼，他没有搬走被裹尸布限制的高大尸体，而将其竖在出口，用鬼影拆取水泥填严周围，形成鬼域无法穿透的嵌尸墙。封好走道后，他以鬼域离开凯撒大酒店，转向平安大厦；方世明虽已逃离，他仍准备利用对方留下的脚印作柴刀媒介，即使再耗一个名额也要结束追杀。（第608章）

## 第609章·媒介触发

总部内，曹洋向王小明汇报朋友圈近乎覆灭及方世明逃走，王小明希望尽快定位方世明、以其死亡结束冲突。方世明则藏在城外乡镇地下室，身体濒临复苏，要求正在飞往国外的贺天雄在杨间死后接管公司；贺天雄畏惧杨间，只能勉强应下。杨间返回平安大厦，在方世明私人办公室找到两张被鬼剪刀剪断的高中照片，由姓名和照片确认鬼剪刀的精确媒介。他坐上方世明坐过的椅子，以鬼影覆盖地面，显出所有旧脚印对应的人影，再对方世明影像从头劈至尾。远处本体和背后尸体同步裂成两半；杨间为此再失一个压制名额，只余一个。（第609章）
