# 第645-649章剧情梗概

## 第645章·高明

陈博士向王小明报告实验楼内见鬼，得到暂不离楼、可派一人试探的答复。助理小张认为留守等死，主动沿楼梯下至一楼并顺利抵达大门；但他跨出楼外时，鬼童封锁大楼的“进出者死”命令生效，小张听见小孩脚步、感觉死人头窥视后当场倒地死亡。基地保安队长高明在外目击，结合自己此前险遭鬼童袭击的经验，确认大楼已经被鬼封锁。实验室内的陈博士等人尚不知小张死状，只能等待其短信。地下五层中，郭凡在杨间身体内听见八音盒音乐，仍误以为那是三只鬼的副作用；王小明拒绝解释，并决定暂不处理大楼异常。（第645章）

## 第646章·猜测

高明认出鬼童身上穿的是杨间从姜尚白处取回的鬼寿衣，判断封楼鬼由杨间留下；他不愿与能无视其他鬼影响的鬼童硬抗，选择离开，将陈博士等人困境留给总部。十分钟未收到小张消息后，陈博士确认试探失败，命剩余人员留在办公室。与此同时，鬼画行动在临时咖啡厅启动：陈义带来新入口，李军准备带柳三、李阳和陈义进入画内。柳三指出关押源头后出口可能一并消失，在鬼画鬼域内与鬼交手也可能重演鬼差惨剧；他主张等鬼画爆发、画中鬼进入现实再关押。李军仍以鬼相机、木门把手和总部命令为依据推进，行动风险被明确保留。（第646章）

## 第647章·熊孩子的参与

为提升先遣队避险能力，李军到熊文文家征调其预知能力。陈淑美听闻姜尚白已死、杨间也快死后更加担忧，却因任务关系城市安全且李军只给十分钟考虑，最终同意儿子参加。熊文文认为任务无法推脱，留下杨间私人电话给母亲，称若自己出事可尝试求助；他因厉鬼影响预感杨间未死，却无法得知详情。下楼后，他与李阳初见便粗鲁相骂，仍被李军带入队伍。此时队伍由李军、柳三、陈义、李阳、熊文文构成，尚未进入鬼画；熊文文预知能力还未使用，五人即将面对杨间曾险些死在其中的画内世界。（第647章）

## 第648章·棺材的异常

晚上八点，王小明从高明和监控档案确认鬼童携带鬼寿衣与死人头，外貌又近似幼年杨间。他认为鬼童受杨间控制，否则陈博士早已被饿死鬼规律杀死，决定保留研究价值并封存档案。地下房间内，控制杨间身体的郭凡五小时前睡入鬼棺，棺材却开始剧烈震动、开裂，先后伸出两只右手；黑暗、人形轮廓和发黑手掌随之出现。杨间遗像从灵位中渗出鲜血，王小明仍拒绝提前干预，等待八音盒时限。鬼棺逐渐破碎、房间被灵异侵蚀，鬼童也突然向一楼跑去，似乎感知到有东西入侵。（第648章）

## 第649章·回归平静的时刻

距午夜仅十分钟时，鬼棺已残破不全，缺口被黑手堵住，染血灵位在血泊中浮沉。王小明与卫景只观察不介入，直至房内彻底黑暗、杨间身体疑似撞门、鲜血从门缝流出。十二点整，所有动静骤停，八音盒自行关闭，鬼棺化为碎木，杨间身体安静留在室内。按既有规律，郭凡应已被诅咒杀死，但杨间体内三只鬼没有复苏，违背王小明预期。王小明由染血灵位和异常平衡推断郭凡可能仍在控制身体，并可能已经成为能承受诅咒的鬼；他进一步怀疑灵位意识交换本身是鬼设置的陷阱。杨间意识是否还在灵位、郭凡是否存活，均未揭示。（第649章）
