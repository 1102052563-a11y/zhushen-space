# 第580-584章剧情梗概

## 第580章·三选一

李军见平安酒店附近街道、车辆和行人消失，又发现异常区域正向外延伸，误判为鬼域扩大，立即令工作人员撤退；实际杨间正以仅存鬼眼把四层鬼域定向伸往酒店。无头鬼影接管的尸体已朝人群移动，杨间知道尸体一旦先夺得头颅，自己和整条街都会失控，因而只能从酒店内选择一名援手。他排除经验不足的王江，又顾虑张雷体内鬼多、复苏风险高且可能成为尸体的拼图，最终锁定只驾驭一只鬼、状态较稳定的黄子雅。选择并非因张雷不可靠，而是杨间担心无头鬼影会优先夺取张雷这具多鬼身体，反令自己被拼成厉鬼身体。黄子雅当时仍在房内看电视，只觉窗外红光是熟悉的杨间鬼域，尚不知道求援将至。（第580章）

## 第581章·黄子雅的行动

黄子雅起初不知鬼域为何出现，却从电视上的杨间红字得知其脑袋已离开身体，需要她拼回。她没有等待总部通知，循鬼域门径赶到血迹处，看到红色鬼烛、打开的八音盒和裂成上下两半、仅残一只鬼眼的头颅。她把两半头颅简单合拢、用外衣包起，并沿染血脚印追踪无头尸体。杨间无法听见她说话，只能继续用红字催促；黄子雅仍主动承担搜索。脚印在街道中段消失后，她判断尸体可能进入店铺，且被困路人四散会增加搜索难度。杨间的意识仍依附残头，八音盒只能令其思考，不能让他自行行动；两人只能在尸体彻底复苏前争取时间。（第581章）

## 第582章·好与坏

黄子雅按杨间提示点燃红色鬼烛，以烛火燃速寻找尸体；杨间同时确认鬼烛不能影响脑中八音盒铃声。烛火在店铺附近猛增，两人看见无头尸体正抓住一名中年男子。尸体戴着的黄金手套隔绝鬼影拆头能力，未能取走头颅，却把人捏碎；黄子雅冲入店内时，四层鬼域已制造错位，眼前只剩尸体，证明鬼影开始利用鬼域藏身。杨间判断尸体下次袭击必会显形，黄子雅遂吹灭鬼烛准备等待。另一边，姜尚白转达总部的妥协条件：朋友圈交出部分产业和项目、参与鬼画并负责杨间尸体；方世明认为总部承受不起驭鬼者内斗，决定暂且接受。双方都未知道杨间尚存意识。（第582章）

## 第583章·意识回归

总部把杨间疑似死亡的消息发送给市区驭鬼者，张雷、陈义等人为此震动，郭凡却看到队长位置空缺；曹洋则判断总部在掩盖朋友圈暗杀。四层鬼域内，无头尸体越发熟练地借鬼域杀人，黄子雅放弃盲搜，在空地等待自己成为目标。尸体从背后抓住她头部时，她释放浓密黑发缠住双臂，趁短暂空隙将杨间两半头颅放回脖颈。鬼影只拼回无意识的上半头，黑发反噬却将黄子雅拖入其中。杨间为救她，以残眼强行折叠四层鬼域，把承载意识的半头粗略重叠回身体；他终于抬手抓住黄子雅手腕，显示意识已经回归，但能否完全控制身体仍未确定。（第583章）

## 第584章·无法理解的异类

黄子雅被黑发吞没时，杨间抓住她并压制其体内鬼，令黑发退去；但他仅剩半个头颅、尸体僵冷，仍像复苏厉鬼。杨间意识回归并非正常复活，而是强行以鬼域把半头塞回身体，正与体内厉鬼争夺控制权。八音盒诅咒使他的意识未被鬼消灭，约半小时后逐渐压过鬼的本能；他以鬼眼替换死灰眼球恢复视野，也能以怪异声音回应黄子雅。杨间据此判断自己已在死亡后成为受诅咒维持、保有人类思维的异类：若诅咒持续，或能承受厉鬼复苏，但诅咒不会永久存在，且更强的鬼仍可能压过它。鬼剪刀袭击来源、八音盒时限和这种状态的结局均未写明。（第584章）
