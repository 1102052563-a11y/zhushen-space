# 第371-375章梗概

## 第371章·灵异入侵

杨间带冯全离开凯撒大酒店，熊文文、罗素一在警戒线外确认冯全仍活，随即被带往第三中医院。杨间说明目标能篡改记忆、已取得童倩后脑笑脸，且医院里的童倩就是鬼；他把鬼报纸的存在隐去，只把记忆能力归给目标。冯全顾虑对同事动手，仍向赵建国例行申报；赵建国担心误判，杨间却认为一旦暴露、鬼换身份潜伏，自己便再无机会锁定。总部为核实情况致电医院，电话接通却无人回应，随后笑声经耳机传出，联络员带笑死亡，通讯室也被笑声入侵。赵建国紧急撤离人员，终于确认杨间判断大概率正确；杨间一行则已抵近医院，准备直接限制。（第371章）

## 第372章·童倩的危急

总部通讯室的骚动被暂时切断，但多名接线员已死。独立办公的刘小雨得知电话引来灵异，看见防护人员不断抬出面带诡异笑容的同事尸体，原有的安全感彻底瓦解；她回到办公室拼命呼叫杨间，杨间为留下行动记录而恢复通讯。中山市第三中医院外，杨间没有制定复杂计划，只以目标恐怖等级不高、多人联手可压制为由带队进入。门口保安也露出同样的生硬笑容，显示被改记忆者可能很多，杨间遂张开鬼域覆盖全院，放弃逐层潜入。病房内的童倩早已看着他到来，鬼域落下时怪笑从其口中扩散到全院；杨间确认这将是双方最后一次接触，结果只能是自己死亡或目标被关押。（第372章）

## 第373章·顺利限制

冯全听见笑声后，杨间以第二层鬼域暂作屏蔽，说明声音会随停留时间增强，必须抢在失控前完成限制。他带冯全、罗素一直接抵达顶楼病房，见病服、缠头绷带的童倩身体正发笑；鬼影从后侵入，冯全带坟土的双手掐颈捂嘴，罗素一的血线又自脚下缠住全身，三重灵异瞬间切断笑声。杨间原想用鬼报纸取下笑脸，却发现报纸对同源目标无效，只得让鬼影强行剥走笑脸并放入黄金盒。失去笑脸的童倩身体仍有活动能力，说明冒名鬼未被消灭，冯全只得把它装尸袋交总部。杨间宣布中山市可暂告一段落，却承认人头气球源头和酒店残留仍未调查。（第373章）

## 第374章·禁忌重生

童倩鬼身被限制后，冯全负责联系总部善后；总部内，曹延华以通讯室死亡三十二名接线员追责赵建国，指出他忽略杨间预警才是主因，并安排沈良赴中山市接收。杨间因城市里或仍有被改记忆者，避到罗素一郊区别墅，先让总部专机送走熊文文。别墅外罗素一、黄飞庆祝生还，林落梅握着王小明的联络方式考虑第二鬼方案；杨间则在房内分拣从酒店带来的人脸。他确认真正童倩已死、身体已被鬼夺取，只剩可用人脸；赵磊无脸身体尚有生命特征却只能维持一两天。因不愿重入酒店找女尸或伤害无辜者，他决定将童倩脸拼到赵磊身体上，令两名受害者以极端方式合并延续。（第374章）

## 第375章·三张脸的人

杨间明知赵磊男身与童倩女脸的拼接违背常理，仍认为两人的原有生路都已断绝，且脸与身体再拖延便会彻底失效。他依人皮纸给出的驾驭方法操作，先将童倩人脸贴上鬼报纸，发现鬼曾留下“杀死杨间”的记忆指令，便擦去旧字，写下信任杨间、忘掉植入记忆、谨慎善良等内容。随后他摆开哭坟鬼遮脸布上的哭脸、从童倩鬼身取下的笑脸及童倩本人脸，让无头鬼影把笑脸贴在赵磊右脸、哭脸贴左脸、童倩脸置中，意在使两张鬼脸同时形成平衡。三脸黏合后，房中同时响起笑声和哭声，两只鬼开始复苏；杨间无法承受声音，立即以鬼域撤离，拼接结果留待后续。（第375章）
