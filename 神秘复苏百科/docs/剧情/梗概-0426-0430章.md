# 第426-430章梗概

## 第426章·助理李瑶

杨间原本要去平安大厦二十层看望母亲张芬，电梯却被人为直送一百层。他在空旷楼层见到自称助理的李瑶，对方承认早作安排，以队长计划余下名额有限、杨间不符合“各方利益”为由提出招揽，要求他进入即将改变局势的新规矩。杨间认为李瑶既不报势力名称也不谈报酬，只以可能被打掉负责人位置相威胁，故拒绝合作；李瑶则声称最初一批驭鬼者或许已不惧灵异。杨间以民国驭鬼者覆灭、诸多未解灵异为例反驳，双方对危险程度和控制权的判断彻底冲突，谈话未达成任何协议。（第426章）

## 第427章·安排

杨间结束五分钟谈话后离开一百层，李瑶身边空地显出由鬼域构成的会议室，数名未具名者现身评估结果。李瑶认为杨间强势、难以屈服，建议改接触与他有过摩擦、资历也足够的冯全；会议中又得知赵建国和王教授可能举荐杨间成为队长。众人因此约定，若杨间队长名额确定便将其“打掉”，把他视作总部一方的危险阻碍。杨间则来到二十层，见张芬在看似轻松、实为保护家属的工作环境中使用电脑，确认其待遇与照看尚可；他没有暴露自己驭鬼者身份，只准备另作安排。（第427章）

## 第428章·猛鬼路？

杨间以停学、同学创业和出差拉投资等说法向张芬遮掩大昌市的真实情况，又考虑队长计划带来的危机，私下要求经理日后以辞退为借口协助母亲离开平安大厦、返回大昌市；但他保留了自身问题无法解决时不带母亲离开的后手。傍晚，李瑶下班驾车离开，渐觉道路车辆减少、行程拉长，停车后发现只剩自己。她按自己认知发出求救并等待三分钟，仍无救援、车辆亦消失，误以为落入A级灵异事件“猛鬼路”。这实际是杨间为等待她而布置的鬼域，李瑶尚未察觉操作者身份。（第428章）

## 第429章·改掉你的记忆

李瑶在无人的路上失车后赤脚逃跑，体力耗尽时发现前路被黑暗截断；路灯下坐着一个以血报纸遮脸的人，她认定厉鬼已出现并绝望瘫倒。持报者放下报纸，露出杨间的脸，说明白天在李瑶地盘时有所克制，如今则在自己的鬼域内行动。他坦言李瑶掌握大京市局势信息，虽然厌恶她，仍需要她替自己工作；李瑶先以死亡后果威胁，后试图用妥协拖延。杨间拒绝征求其同意，宣告会让她忘掉当晚经历、改变记忆成为忠于自己的员工，冲突由招揽失败转为强制控制。（第429章）

## 第430章·这就是爱情？

李瑶推测杨间掌握了中山市篡改记忆鬼的能力，暗中录音自保；杨间却以鬼域令其所谓黄金手机显成拖鞋，指出她一路所见的车、物品和衣物皆可能是假象。李瑶因恐惧和羞辱崩溃，仍威胁日后报复；杨间随即用染血鬼报纸覆住她的脸，取走其面孔与记忆。为避免植入不合现实的亲属或旧友关系引起破绽，他筛选后选择爱情，写入李瑶忘记八点后经历、从见到自己时便有好感、醒后对他形成至死不渝爱情且愿意帮助其亲友的记忆。李瑶尚未苏醒，改写的实际效果、代价和后续反应均未揭示。（第430章）
