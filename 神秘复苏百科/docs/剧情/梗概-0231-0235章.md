# 第231-235章梗概

## 第231章·数字7

叶枫失踪前只留下数字“7”，各人无法判断是鬼数、剩余人数还是其他信息。杨间主张先记录而不据此贸然行动，并指出若叶枫、何川死亡，其体内厉鬼会使失踪地点更危险。王小明设下七日期限，若仍无法捕获第三阶段鬼婴，所有人包括他本人将前往失踪点探查。（第231章）

## 第232章·手术

三日内鬼婴寄生案例突破两百，城外支援仍无法进入；杨间隐瞒自己可取出鬼婴，避免被迫处理全部个案。王小明在医院召见他，以一名男性寄生者进行研究，要求杨间用鬼域模拟大昌市地图、在取出鬼婴后观察其行动，以寻找叶枫失踪地点。杨间只同意本次协助，并取得赵建国七日后可自行行动的承诺。（第232章）

## 第233章·再次研究

杨间以局部鬼域和左臂鬼影从男性患者腹内抓出第一阶段鬼婴，患者获救。王小明放开鬼婴后观察其优先攻击附近可见目标，杨间以鬼域瞬间收容。王小明据已有资料提出：第一阶段杀看见者，第二阶段杀接触者，第三阶段可能杀听到声音者；鬼域可同时规避这些条件，杨间因而被视作关键行动者。（第233章）

## 第234章·各阶段变化

王小明以婴儿感知世界的顺序解释“看见、触碰、听见”三阶段规律，并拒绝对第四阶段作无证推断。鬼域中的鬼婴沿虚假城市向北城区移动，最终朝叶枫、何川失踪区域靠近。杨间与王小明确认其大致位置在一家药店对面，准备继续观察。（第234章）

## 第235章·被触碰的禁忌

鬼婴在目标位置徘徊时，杨间鬼域内凭空出现地图不存在的第七街道；他收回鬼域后，街道投影仍在医院出现，证明未知力量已反向侵入现实。街道深处青黑人影和脚步逼近，杨间、王小明撤离并疏散医院。人影随后推开手术室门，青黑气息开始侵蚀整座医院，源头鬼或第四阶段威胁正式显形。（第235章）
