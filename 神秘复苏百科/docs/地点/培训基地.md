# 培训基地

## 概述

培训基地位于大京市郊区，是总部安排新任国际刑警接受至少七日培训的场所。杨间、张雷、王江等驭鬼者在此参与课程和枪械训练；基地内有教室、靶场、宿舍楼和出入口。第441至445章，王小明鬼棺实验失控后，目标自附近实验室进入或影响此地，基地成为鬼差事件的现场。（第413章、第436章、第441章）

## 培训活动

第436至438章，白学文在教室以历史、神鬼志异和黄金棺椁授课，张雷、王江、杨间等人到场。第439章课程结束后，杨间到靶场练枪，张雷返回宿舍；这些记录仅能确认该处承担集中培训和射击训练，管理编制、常驻人数及防护设施均不详。（第436章至第439章）

## 鬼差入侵

第441章，靶场周围被黑暗吞没，杨间遭一双僵硬手掌袭击。刘小雨随后通过卫星定位手机告知，实验室有未知级别的鬼脱困，总部要求基地人员立即撤离、能走一人是一人。杨间先前已动用三只鬼勉强抵住袭击，意识到留在基地的驭鬼者若遭杀害会让目标继续成长。（第441章、第442章）

宿舍楼内，王江、张雷和其他未具名驭鬼者听见惨叫并接到总部警示，原计划在一楼集合冲出基地。屋顶却传来不应存在的脚步，一名从五楼跳下逃离的驭鬼者被从屋顶垂下的鬼绳套住脖子、吊在半空。其姓名、是否死亡以及其余人员是否成功集合均未写明。（第442章）

## 封闭状态

杨间由大门离开后，无论怎样在黑暗中绕行都会回到培训基地，举鬼烛也不能脱离。他将鬼域叠至四层后发现阻隔空间的是陈旧棺材板，因而推断整个基地及周边区域都在鬼棺内。该结论解释了出口循环，却仍是杨间的现场判断；鬼棺本体位置、黑暗覆盖边界和普通人能否从其他路径离开均不详。（第443章）

第444章，杨间因无法离开而折返宿舍楼，准备把仍存活的驭鬼者聚在一起，以免他们落单遭杀并推动鬼差成长。总部会议在基地外进行，至第445章末尚未抵达现场；基地中的确切幸存者、伤亡人数及封锁何时解除均不详。（第444章、第445章）

## 第446-450章·宿舍楼搜救与伤亡扩大

杨间确认基地出口循环无法靠鬼烛或鬼域破除后，从外侧进入黑暗覆盖的宿舍楼搜人。他先在楼外找到黄子雅，再在宿舍内找到一名未具名驭鬼者；楼下新鲜血迹和尸体碎片则表明已有人员遇害，鬼差至少能压制五只鬼。宿舍楼空间宽大、设有八个安全出口，原本利于疏散，在黑暗中却使受训者难以辨向和汇合。黄子雅通过总部通话确认钱毅、张雷等人的位置，杨间要求以实际通话而非手机定位判断幸存者，避免把队伍带向遗失手机或死亡者所在处。（第446章、第447章）

四楼成为鬼差与多名驭鬼者同处的高危区域。张雷、王江及两名未具名者先在此摸到鬼差后逃散；一人随后在阳台被夺走体内鬼并死亡，另一人在第450章再次惨叫后失声。杨间在三楼接回走散的王江，虽知鬼仍在四楼，仍带黄子雅等人折返搜救，以阻止鬼差继续成长。第450章末，杨间一行已在四楼看见无法确认身份的人影，张雷也仍在该层迷失；基地伤亡至少包括本批两名未具名驭鬼者，钱毅是否已抵出口、其余受训者人数和救援能否抵达均不详。（第448章至第450章）

## 第451-455章·操场暂歇与白烛诱鬼

杨间在四楼以耗尽红色鬼烛为代价救回张雷后，五人撤至基地操场。五人合计八只鬼，暂时高于鬼差当时可压制的六或七只，鬼差没有继续追击；但基地出口仍受鬼棺式空间封闭，杨间已从一楼搜索到四楼、判断宿舍楼剩余人员不多，转而带队绕基地外侧寻找走散者。王小明通过实验室监控确认鬼棺仍在特殊房间、棺盖打开，令“棺在实验室而人如在棺内”的关系仍无法解释。（第451章、第452章）

搜寻期间，人皮纸称鬼差已可压制八只鬼，基地水泥路旁又出现半具新尸，说明鬼差持续猎杀落单者。杨间认为五人搜救速度无法胜过鬼差夺鬼速度，遂不再只靠搜索，实际点燃白色鬼烛，尝试以其引鬼特性把鬼差留在五人附近；若队伍仍不落单，目标或许无法动手，其余幸存者则可获得短暂安全期。白烛燃起灰黑烛光后，基地环境再被黑暗压住；诱导是否成功、鬼差是否已到来和新的伤亡均未写明。（第453章至第455章）

## 原文重复第456章至第459章·白烛牵引下的双重危机

白色鬼烛实际把鬼差从食堂和钱毅所在处引向杨间五人：食堂内一名未具名驭鬼者因躲藏、没有卫星定位手机且处于落单状态，被鬼差杀死；钱毅独自出基地又绕回，黑暗将至时因白烛转移目标而幸存。鬼差接近五人后压住杨间鬼眼，杨间为补足数量释放鬼手、打开魂瓶；本方能力恢复，却同时使鬼手复苏、魂瓶内鬼逃出。基地内五人之外仍有多少幸存者、食堂死者身份和钱毅能否立即脱险均不详。（原文首个第456章·厉鬼来袭、原文重复第456章·数量上的比较、第457章、第458章）

第459章，杨间以持续燃烧的白烛带队移动，鬼差黑暗和脚步始终随行，暂未再杀其他人；魂瓶中逃出的鬼可能藏在五人中。总部另组成十余名顶尖驭鬼者的队伍，由李军鬼域护送王小明赴实验室研究并取回鬼棺等物，但此时尚不能实际支援被困基地。黄子雅定位到钱毅在前方约五百米，杨间决定优先会合。白烛范围、总部队伍与基地空间的距离，以及钱毅会合是否安全均未写明。（第459章）

## 出处
