# 71号房

## 概述

71号房位于凯撒大酒店四层那条不属酒店正常布局的异常走道内。房门使用铜制门牌，编号清晰为“71”；走道两侧客房由昏黄门灯标示，墙壁却阻隔杨间鬼域，因而他无法在门外看见房中情形。第602章，杨间在走道深处听到钢琴声，循声抵达这间门灯昏暗的房间，确认音乐来自室内。（第602章）

## 室内与活人

杨间开门后看见民国风格、但十分整洁的客厅，中央放着钢琴。香兰身穿红色旗袍坐在琴前，钢琴旋律与杨间脑中八音盒诅咒的铃声相同，却因她紧张而不断出现错拍；隔壁还藏着阿南。二人都被杨间确认是活人，但凯撒大酒店外部早已封死，二人又把杨间称为可能来自“出口”的外来者，使这间房与异常走道另一端或存在未知连接的可能浮现。此关联只是杨间依据现象作出的推测，房间本身是否为出入口、二人何时进入均未获确认。（第602章、第603章）

## 第604章·钢琴诅咒

香兰与阿南离开后，71号房再次响起钢琴声，旋律已变为杨间未听过、近似摇篮曲却带黑暗压抑感的音乐。杨间在门外没有重新进入，因他推测二人急于离开是知道房内有鬼。其后木门重重关闭，钢琴声骤止；杨间立刻被阴冷寒意笼罩，意识消失十余秒，直到八音盒诅咒的旋律将他拉回。杨间认为这是近乎必杀的诅咒，若非八音盒保护，即使驾驭三只鬼也会死亡；这只是他从当次遭遇作出的判断，房内究竟是钢琴、客房还是未知鬼在发动规律，触发是否必须听琴或关门，以及对其他人的效果均不详。（第604章）
