# 赵开明仓库

## 概述

赵开明仓库是赵开明购下并自住的一处小物流公司库房。他将库房改作私人空间，表面存放大量高档食材，内设冰棺室和灵堂；其具体位置未说明。（第236章）

## 冰棺与灵堂

库房内有十二口冰棺，分别安置赵开明的妻女、父母、舅舅等亲属遗体；尸体均经专业整理，仍保持冰冷僵硬。赵开明每日为遗体准备多份餐食，又在十二个牌位前烧纸磕头。他称会让家人“团聚”，并把对杨间的杀意与计划联系起来，但复活或计划的具体方式不详。（第236章）

## 风险

赵开明在仓库内对空处和疑似闯入的“鬼东西”说话、开枪，墙上弹孔隐约构成人形；其精神状态明显异常。该仓库是否有厉鬼进入、冰棺遗体是否受灵异影响，均未获确认。（第236章）
