# 第七街道

## 概述

第七街道是王小明对叶枫、何川失踪处所对应异常地点的暂定称呼。现实大昌市地图中不存在这条街道；其外观却是两侧商铺紧闭、通向青黑阴霾深处的街道。叶枫在西城区最先误入类似区域，王小明后通过鬼婴在杨间鬼域地图中的移动反应锁定其现实邻近位置。（第232章至第235章）

## 已知表现

- 叶枫进入的区域中，正常退路消失，鬼牙不能咬开环境；何川残头和大量不同阶段鬼婴在其中出现，叶枫随后失踪。（第228章、第229章）
- 王小明判断该地点不属于大昌市任何现实街道，而是类似鬼域的特别空间；叶枫失踪点实地勘查未发现异常。（第232章）
- 杨间以鬼域模拟全市地图时，第一阶段鬼婴在北城区药店对面徘徊，原本不存在的街道突然出现，显示该处能修改或侵入杨间鬼域。杨间收回鬼域后，街道投影仍于医院显现。（第234章、第235章）
- 街道深处有青黑人影伴随脚步逼近，最终推开手术室门进入医院；其身份、是否为第四阶段鬼婴或饿死鬼源头均未确认。（第235章）

## 风险

王小明和杨间均认为第七街道存在高度禁忌：发现它可能意味着被它同时发现。其可把虚假鬼域中的街道强行显化到现实并放出未知厉鬼，超出两人既有认知；医院已在其出现后被青黑气息侵蚀。（第235章）

## 第266-269章·定位修正

王珊珊苏醒后称曾在梦中看见源头鬼站在大昌市第七中学门口，杨间据此进入学校，发现该处阴霾最浓、校内有上千只各阶段鬼婴。可是他到三楼厕所后发现原先通向灵异空间的厕门消失，意识到第七中校园只是当年鬼域中的位置，入口随鬼域迁移。杨间回到步行街，以鬼域重演从校门逃至三楼厕所的旧路径，最终定位到弘法寺地下室的木门；木门后正是当初灵异空间。第七街道与七中、步行街及弘法寺之间是否为同一鬼域的不同投影，原文未作确认。（第266章、第267章、第269章）
