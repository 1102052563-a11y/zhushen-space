# 鬼邮局

## 概述

鬼邮局是一栋处于独立鬼域中的民国风格五六层旧建筑，现实中与大汉市建设路869号烂尾楼重叠，却通常不和现实直接交集。其大门牌匾写有“鬼邮局”三字，旁有红绿霓虹灯；建筑墙壁发黑、覆有青苔，内部旧木地板、发黄灯光和回字形走廊都保留着旧时代特征。（原文回跳第738章、原文回跳第739章）

## 与现实的重叠

杨间在烂尾楼内发现被藏进墙壁、水泥地和天花板的金戒指、项链、手镯，判断这里是失踪者经过之处。孙瑞据此指出黄金不受灵异力量影响，失踪者被带走时金饰才会留在现实。杨间进一步推测，烂尾楼并非鬼本体所在，而是建在鬼邮局鬼域与现实重叠的位置，因而长期发生施工事故、无法完工。（原文回跳第738章）

该推测尚不能解释两处空间为何重叠、烂尾楼最初何时受影响，或所有大汉市失踪者是否都经由此地进入邮局。孙瑞此前虽多次经过或派人查过烂尾楼，普通人均未发现邮局存在。（原文回跳第736章、原文回跳第738章）

## 进入与压制

杨间先以第三层鬼域令现实烂尾楼消失，第四层鬼域才显出被迷雾笼罩的民国建筑轮廓，第五层鬼域方能撕开迷雾并带孙瑞、李阳强行进入。顶层窗户在入侵时猛然开启，窗口出现疑似穿黑长衫的阴影，楼上同时传来尸体踩木板般的脚步，表明闯入已触动未知灵异。（原文回跳第738章）

一楼内，杨间的鬼眼受压制，四层以下鬼域无法展开，须维持五层鬼域才可无视压制。这说明邮局对较低层级灵异具有约束或隔绝能力，但对具体厉鬼、压制范围和能否安全离开均未验证。（原文回跳第739章）

## 内部结构

一楼为大厅和回字形天井，沿层有七间木门房，门牌以“11”“12”等格式标示楼层和房号；五层均可看见相似走廊与房门，总数疑为三十五间。墙上挂有不同年代陌生人的黑白画像，楼内没有通向二楼的可见楼梯。杨间注意到三十五间房与灵异公交车三十五个乘客座位加一名司机的结构接近，但仅为联想，未证实关联。（原文回跳第739章）

## 信使关联

李跃曾说明，信使会在开门或行走时偶然进入一座无法定位的邮局，并相信掌控邮局可摆脱送信诅咒。杨间进入后遇到万兴等活人，判断他们可能是邮局信使；这使此前信使所说的“邮局”首次对应到实体灵异空间。邮局如何挑选信使、信件从何处发出、每层规则和掌控条件仍不详。（原文回跳第698章、原文回跳第739章）

## 原文回跳第740-744章·层级、熄灯与红信任务

万兴、王善称每层最多七名信使，完成三次送信后会被带往上一层，越高层的送信间隔越长。杨间认为这更像逐层淘汰、选拔精锐的机制；二至五层并非普通楼梯可抵达的实体空间，鬼眼向上只能看见黑暗，强行入侵可能迷失于嵌套鬼域。（原文回跳第740章）

邮局每天晚六点至早六点自动熄灯，黑暗中有夜游鬼活动；信使必须回房避险。杨间等四人同住17号房后，夜游鬼连续敲门、熄灭室内灯光并试图开门，李阳、杨间和孙瑞共同将其逼退。房间如何隔绝夜游鬼、多人同住为何会引来异常、夜游鬼是否由邮局控制均不详。（原文回跳第740章至原文回跳第742章）

翌晨，一楼柜台出现红色集体信。杨间取信后得到“七天内送往福寿园78号主人”的任务，决定与孙瑞、李阳赴大海市。王善、万兴和钱蓉也从邮局出发，却未和三人出现在同一处现实出口；邮局出口与不同信使原地点间的对应方式尚未明确。（原文回跳第743章、原文回跳第744章）

## 原文回跳第780-784章·信使召回、二楼开放与红信撕毁

第二次任务开始时，杨间、李阳在各自所在地都看见一条只有信使可见的小路，路尽头直接连到鬼邮局。杨间判断邮局会主动靠近不愿归队的信使，将其强行拉入灵异空间；小路能跨大昌市与大汉市的现实距离，但具体覆盖范围和拒绝进入的后果未验证。（原文回跳第780章）

二人重返一楼后，原有七间房消失、通往二楼的木楼梯出现，说明红信成功投递使一楼信使满足上楼条件。楼梯空间与距离受灵异扭曲，中途有一具穿旧中山装的尸体；杨间、李阳越过后抵达二楼，楼梯随即消失，暂时只能向上而不能返回一楼。（原文回跳第780章、原文回跳第781章）

二楼仍为回字走廊，设21至27号七间房。晚六点至早六点熄灯后，走廊滞留的陈星失踪，强化了夜间不得留在房外的风险判断；但熄灯是否必死、失踪是否由夜游鬼造成尚未直接确认。（原文回跳第781章、原文回跳第782章）

次日，二楼全部信使被红色集体信召集，任务地址为大川市明月小区7栋301室。杨间为防止红信任务在外界引发新的大规模事件，直接将信撕毁，触发二楼七房内未知厉鬼对全层信使的袭击；具体袭击规律另见[鬼邮局二楼红信袭击](鬼邮局二楼红信袭击.md)。（原文回跳第782章至原文回跳第784章）
