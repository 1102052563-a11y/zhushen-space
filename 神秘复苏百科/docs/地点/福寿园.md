# 福寿园

## 概述

福寿园是位于大海市近郊、历史可追溯至民国时期的墓园，后经多次翻修成为道路整洁、绿化良好的现代墓园。大海市土地昂贵，在此安葬费用高；园内一号墓园采用无坟丘的墓碑编号布局，78号墓地位于其中。（原文回跳第744章、原文回跳第747章）

## 鬼邮局任务地点

鬼邮局红色信件要求七天内送至“福寿园78号主人”。杨间、孙瑞、李阳到达时，78号墓穴刚挖好、棺材尚未下葬；墓碑显示墓主是前天去世的刘老太。杨间认为只有刘老太正式入土、成为78号墓主后再投信，才可能符合任务文字，故没有直接前往殡仪馆塞信。（原文回跳第744章、原文回跳第747章）

## 异常迹象

墓园白天仍有普通人扫墓、车辆进出，杨间以鬼眼检查时没有发现直接灵异。园内却存在多块无姓名、无遗像的空白墓碑，与通常已埋葬者应有的标记不符；其是否只是预留墓位、习俗选择或灵异线索均不详。（原文回跳第747章、原文回跳第748章）

刘老太葬礼进行时，棺内尸体悄然睁眼，一号墓园同时出现诡异暮色。该异变说明红信任务和刘老太尸体存在异常关联，但福寿园整体是否已成灵异事件范围、无字碑与尸体异变是否有关均未证实。（原文回跳第749章）

刘老太棺材被灵异论坛人员抬动后，尸体消失。杨间以三层鬼域搜索地面无果，发现鬼域不能渗入部分墓地下方；随即现代墓园被旧式坟场覆盖，城市景象消失、柏油路变泥土、老坟不断出现。此处目前已成为诡异坟场鬼域的一部分，刘老太尸体去向及78号墓是否仍在原位置均不详。（原文回跳第750章、原文回跳第751章）

## 所属与管控

福寿园位于灵异论坛控制的大海市，叶真因杨间等人到访而派罗素一、林落梅调查。墓园本身未被确认由灵异论坛直接经营；工作人员仅按普通管理要求登记访客、规定晚五点半闭园。（原文回跳第746章、原文回跳第747章、原文回跳第749章）
