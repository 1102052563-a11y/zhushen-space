# 黄岗村

## 概述

黄岗村位于大昌市郊山脚，表面是半新半旧的普通村庄：前半有新建小楼，后半保留旧土屋、木屋、菜地与鸡群。第82章杨间为取得第二鬼驾驭方案来到此处，发现村中异常冷清，却有村民、灵堂和朱漆棺材等看似正常的生活景象。（第82章、第83章）

## 已知历史

刘小雨提供的档案显示，黄岗村曾在半年前一夜消失，建筑、村民与家禽全部不见，只剩村中心一口打开的棺材和模糊人影；三天后村子重现，原村民没有回来。前任大昌市刑警冯全进入调查后留下“别动那口棺材”的残缺报告，最终下落不明。当前村民与旧村民是否同一批、村庄是否原本地点，均不能确认。（第93章）

## 灵异表现

- 村口灵堂内有朱漆棺材和不断变化的男子遗像。白天村民披麻守灵，夜间灵堂无人、棺材疑有鬼物。（第82章至第92章）
- 村中会出现虚弱咳嗽与脚步声，脚步可将驭鬼者引向同一地点；落单的贺胜、欧阳天先后死亡。（第83章至第90章）
- 张韩等人驾车离村却回到村牌前，说明黄岗村存在阻止离开的异常；平衡崩坏后，村民和灵堂房屋会消失，村庄陷入反常黑暗。（第91章、第96章）
- 杨间按夜间脚步汇合点绘制地图，发现三个地点围绕村口灵堂；贺胜和欧阳天先后死亡，二人遗体内的鬼均被认为已经离开。（第86章至第90章）
- 张一鸣等离村后遭鬼打墙回到村牌，证明村庄不能正常离开；杨间查档得知半年前村庄曾连人和家禽整体消失，三日后重现而原村民未归。（第91章、第93章）
- 杨间判断凶鬼会杀害落单驭鬼者，张一鸣三人聚坐整夜未受袭击印证此推测；村民身份仍无法确认。（第93章、第94章）
- 第96章，老婆婆鬼奴被收容后村庄由白昼骤暗，村民、灵堂及整栋灵堂房屋消失，显示两鬼平衡被打破；无头鬼影附身老婆婆后，部分村民逐步消失、灵堂和朱漆棺材重现，黑暗开始退去。（第96章、第100章）
- 第103章，杨间驾驭无头鬼影后带张韩进入重现的灵堂。二人得知棺内并非单纯厉鬼，而是失踪的大昌市第一任刑警冯全；第105章冯全称黄岗村早已没有正常活人，所见村民因鬼棺而存在，自己可在棺中有限影响其思维。（第103章至第105章）
- 第105章，冯全确认供桌遗像中的年轻男子是守在鬼棺旁的厉鬼；杨间此前在灵堂未被落单规则杀死，是因冯全也在棺中。冯全称厉鬼一旦进入鬼棺，可造成一座城市沦陷；本章末村中天色再度转暗，平衡仍在恶化。（第105章）
- 第106至108章，遗像男子随夜色来到灵堂，先在杨间鬼域外停步；杨间的鬼域、无头鬼影与张韩背后鬼都被其压制。冯全称该鬼可使三只厉鬼能力失效，且夺取更多厉鬼后压制数目会增加。（第106章至第108章）
- 第109至110章，冯全将遗像男子的崩解躯体装入黄金箱后，村中脚步仍重现。杨间据入鬼棺后意识可感知全村的经历，推测村鬼并无固定形体、黄岗村整体与鬼棺相连；此为杨间当时判断，尚未获确认。（第109章、第110章）

## 第517-518章·在鬼画鬼域中重现

总部引鬼画与鬼差碰撞后，李军等行动者在市郊原本隔绝鬼差的山谷处见到一座黄岗村，村口牌上仍写着“黄岗村”。行动队已从档案得知鬼差源头来自此处，因此在原山谷被村庄取代、鬼差黑暗也消失时立刻选择后撤；但他们穿过灰白树林后又回到同一村口，证明鬼画鬼域内的黄岗村会阻碍离开。这个村庄是否为真实黄岗村被搬入、鬼画制造的景象，或鬼差力量重现，原文未能确定。（第517章）

第518章，行动队推测村内建筑和村庄整体都可能属于鬼的一部分，故先在村外停留；苏凡认为进村是唯一暂时不死的路线，李军最终带队进入。村中静得没有虫鸣，首间砖瓦房内有一具被鬼绳吊起的尸体，鬼绳另一端可见黄金；第三间房内又有一具头转一百八十度、已腐烂的尸体。队伍没有接触两者，只由档案推向鬼棺所在的村前房屋。此次重现与第82至110章黄岗村原有村民、灵堂和鬼棺关系如何对应，均不详。（第518章）

## 第519-523章·房屋搜索、重复痕迹与行动队失联

行动队进入后发现，灰白道路可暂时通行，但每一栋黑暗房屋都可能有鬼；他们已陆续排查多栋，第七栋三层民房内又有沉重脚步。李军绘制的临时地图显示，若撇开灰白道路，全部黑房会连成一口黑色棺材；纸灰铺成的道路向黑房侵蚀，却在边缘被黑暗吞没，行动者判断鬼画与鬼差处于暂时平衡。这个村庄的房屋是否都是鬼差的一部分、道路是否属于鬼画，均是现场判断，未获确认。（第519章、第520章）

柳三进入该民房二楼，发现门口带有灰白灰烬的凌乱脚印，以及五具由他制作、外貌皆为他的残破纸人，遂怀疑队伍或曾反复到访且记忆被影响。隔壁房间只掉下一张老人遗像，遗像背景与民房相同，且将五名行动者映入画中；遗像被黄金袋收容后，门外即出现与画面构图相同的老人背影，鬼烛剧烈燃烧。行动队不久后整体失联，随后鬼画与鬼差鬼域均消散。黄岗村是否随之消失、遗像与老人背影的关系、五次纸人死亡和失联的因果，均不详。（第520章、第521章、第522章、第523章）
