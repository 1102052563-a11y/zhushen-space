# 弘法寺

## 概述

弘法寺位于大昌市，是一处有住持和大雄宝殿的寺庙。对普通幸存者而言，饿死鬼事件后这里成为烧香、求符和寻求心理安慰的地点；对杨间而言，寺内地下室却连接着曾封存白骨树、三米高黑影与棺材钉的灵异空间。寺庙的日常宗教活动与地下封印之间的具体历史关系，至第280章仍不详。（第267章、第269章、第277章）

## 地下室与灵异空间

杨间为寻找能够限制饿死鬼的物品，通过鬼域重演旧日路线，确认第七中厕所对应弘法寺大雄宝殿前的位置。他在寺庙地下室尽头发现被金箔覆盖的老木门；住持称门后遗物由前代高僧留下，称作“降魔杵”，只知不可随意开启。杨间进入后见到无生命的黑暗空间、青砖墙、白骨树及被棺材钉钉住的无五官三米黑影，遂取走钉子处理大昌市危机。其后住持查看发现门后已只剩光秃青砖墙，原有空间、白骨树和黑影均告消失。（第267章、第269章、第277章）

## 第277章·幸存者聚集与历史追问

饿死鬼阴霾消散五日后，大昌市恢复水电、武警驻防和医疗救援，大量幸存居民因确认厉鬼存在而到弘法寺烧香拜佛。住持没有出面念经或开光，直言面对厉鬼，普通符法不能救人，拿无效符纸去对抗厉鬼反而可能害人。杨间以鬼域直接进入其后院住处，追问地下空间和其中厉鬼的来历。

住持只能转述师门所知：其师父称地下室内由旧时高僧关押厉鬼，以降魔杵镇压；年轻时他本人曾见门后白骨树、三米黑影和一只诡异红眼。经查寺内修建地下室的记录约在民国时期，彼时其师父的师父仍在世，但再早的信息已断绝。杨间因此将弘法寺地下室与观江小区民国老宅联系起来，推测百余年前或已有驭鬼者与厉鬼的对抗；这只是其推测，寺中没有留下可验证答案。（第277章）

## 当前状态

第277章后，弘法寺对外仍接纳前来求安心的居民，地下木门后却只余墙面。棺材钉已被杨间带走并用于限制饿死鬼，原本被钉住的黑影及白骨树不在寺内，也未被确认在大昌市复苏。寺庙能否继续保管相关秘密、消失的灵异空间是否会重现，均不详。（第269章、第277章）

## 出处

- 杨间在寺内地下空间取出棺材钉。（第269章）
- 杨间向住持追查地下室记录，确认其约建于民国时期。（第277章）
