# 陵镇老宅

## 概述

陵镇老宅位于中安市外的陵镇附近，原属罗姓人家，按万德路收集的资料建于民国时期。第507章，杨间因魂瓶古董来源委托万德路调查，得知后人本计划推倒久无人住的祖宅重建，却在地下室发现一批古董；这些古董后被收购至中安市古玩街，魂瓶即在该批物品之中。宅主家系与民国驭鬼者是否有关、地下室是否原为关押设施及古董最初由谁放入，均不详。（第507章）

## 建筑与废弃状态

资料照片显示，老宅使用青砖修建，左右无窗，仅有通风口与一扇窄高木门；墙体年久失修，半面墙已倒塌，砖石间长满野草。杨间将其与大昌市民国老宅比较，推测其废弃时间至少约六十年，而非确定史实。他又推测其原主人可能在民国时期因厉鬼复苏中期死亡，但该判断只基于建筑废弃情况和现时灵异线索，不能确认原主身份或死因。（第507章）

## 五年前的异常照片

万德路取得一张五年前游客在宅前拍摄的照片：木门有十余厘米门缝，内部漆黑，而门顶缝隙处伸出一只模糊手掌，四根僵硬手指搭在门沿。木门约三米高，宅内又早已无人居住，杨间与万德路均无法以普通人解释该手；但照片像素低，杨间不能确认它是否就是魂瓶鬼手。五年前拆旧房时未报人员死亡，若异常手属于鬼，为什么没有扩散、又如何与后来被关进魂瓶的鬼对应，皆构成未解问题。（第507章）

## 与魂瓶事件的联系

万德路另查到一只疑与此前魂瓶成对的瓶子仍留在中安市古玩街，但古玩街已紧急疏散、封锁停业，部分店铺甚至未关门。杨间据现场航拍判断这不是普通封锁，而可能是魂瓶中厉鬼被放出后引发的灵异事件。该事件是否来自陵镇老宅、另一个魂瓶是否确实已破封、当地负责人如何处置，均未写明。杨间决定见一见原宅户主罗永，期望取得民国线索。（第507章）

## 第513章·户主补充地下室出土物

罗永到大京市接受杨间询问后，确认老宅户主原为其祖父、自己只是继承人，并补充地下室实际挖出五只瓶子及盘、杯等物。五瓶的售卖对象分散：两只低价售出，三只被陌生人以现金高价一并买走，买方未留下身份或联系方法。该口供扩大了已知古董规模，但不能确认每个瓶子都关押厉鬼，亦不能证明买方与古玩街封锁或魂瓶鬼有关。（第513章）
