# 观江花园

## 概述

观江花园是大昌市市区不远处、依江建设的大型楼盘，也是张显贵带杨间调查张伟所遇镜中鬼线索时所到的工地。小区入口有圆形拱门，内部已建成多排楼房，按杨间目测至少有数十栋高楼和上百栋别墅；大部分楼盘已完成并开始销售。张显贵称自己希望将其打造为富人区，别墅区占地超过一半，且未采用预售制。原文同时写明工地已有工人失踪，张显贵为封锁消息而停工、停售，因此其环境、建设规模与“闹鬼”风险并存。（第161章、第162章）

## 已知区域

- **工地与楼盘**：张显贵确认张伟首次撞见跟随鬼的地点不是别墅，而是自己带儿子去过的工地；他以鬼穿工地工作服、近期又有工人失踪为由，认为异常来自此处。这只是张显贵和杨间依据线索作出的可能性判断，鬼的来源尚未查明。（第161章）
- **售楼部**：售楼部临江而建，共五层，张显贵称由国外知名设计师设计，外观和装修如庄园。第163章杨间在三楼会客区域向钱总、秦总展示鬼眼并以鬼域制造血迹幻象；第165章两人带保镖乘电梯离开时，又在杨间的鬼域幻象中抵达并不存在的地下十层。现实楼层和地下结构是否存在对应区域，原文未写。（第162章至第165章）

## 开发与处境

观江花园并非张显贵独资开发：他负责前部楼盘，另有钱总、秦总负责后部，三人曾共同竞得土地。工地失踪与停工使项目资金和销售受压，钱、秦二人起初主张照常销售、以促销化解风险，并不相信灵异说法。（第162章、第163章）

杨间认为灵异事件正在增多，人口较少、交通便利且临江的观江花园适合作为住处；他提出以五千万元购买售楼部，并与张显贵初步谈定合作：若能让另两名开发商低价售出其份额，自己取得全小区四成收益。张显贵要求先看到实际能力，双方仍未在本批完成具体收购或签约。（第162章）

第167至168章，钱总、秦总在转让合同上签字，张显贵以十亿元完成对二人名下楼盘的收购。杨间选择不把楼盘直接过户到自己名下，只取得最终利润、数栋楼和别墅；售楼部则按五千万元从其利润扣除，销售人员与物品将搬离，以供其居住。张伟因镜中鬼风险暂住售楼部，杨间计划次日调查整个小区，并要求其暂避镜子和反光物。交易结束不等于工地失踪、镜中鬼源头或楼盘安全已经解决。（第167章、第168章）

第163至165章，杨间为说服钱总、秦总，以鬼眼和鬼域令二人相信售楼部闹鬼，并把他们困入虚构的地下环境。两人终于向张显贵求救时，张显贵借救援谈判提出收购其楼盘；截至第165章末，秦总只认定这是趁火打劫，尚未写明两人是否接受交易，工地的真正鬼患也仍未调查。（第163章至第165章）
