# 观江花园避难所

## 概述

观江花园避难所是张显贵在大昌市青黑阴霾事件发生后，利用小区地下空间改造的避难设施。张显贵原本为应对可能的长期混乱而囤积生活物资、推进施工；杨间随后提议在避难所内增设以黄金隔绝灵异力量的安全屋。（第230章）

## 安全屋方案

杨间提出将黄金熔成板材，结合钢板和密闭结构制作小房间，作为遭遇灵异事件时的最后庇护。张显贵认为两吨多黄金不足以单独建造完整设施，但可在既有避难所中隔出安全屋，复用通风、用水、物资储备等配套。双方约定以杨间提供的黄金为核心材料，避难名额或权益按七三分配；具体面积、厚度、可避难人数和完工时间尚未确定。（第230章）

## 管理与风险

张显贵负责设计、施工、人手与既有避难所配套，杨间要求不得为扩大面积而削薄黄金层，并要求江艳记账、监工。杨间还要求小区居民每日检查腹部异常，发现突然怀孕等鬼婴寄生迹象立即报告。项目尚在筹划或施工阶段，尚未实际验证对厉鬼的隔绝效果。（第230章）

第256章，工人因恐惧逃离，主体虽已完成，通风、用水等配套未完，张显贵估计安全屋只可维持约十日，远低于杨间原先两三个月的预期。第258章，地下安全屋勉强容纳十九人，张显贵家属、张韩家属中一名婴儿、王小明、江艳、两名武警及其他已定名单人员陆续进入；仍有王彬一家和张显贵其他亲属留在外面。杨间不进安全屋，称驭鬼者复苏会害死其中所有人。安全屋在本批未遭厉鬼验证，十九人名单全貌和十日后补给方案不详。（第256章、第258章）
