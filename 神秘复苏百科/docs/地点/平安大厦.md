# 平安大厦

## 概述

平安大厦位于大京市市中心的繁华地段，周边高楼密集、人流拥挤。第425章，杨间到此看望在楼内工作的母亲张芬；他由大京市人口密度异常偏高，推测在全球灵异事件频发的背景下，知情者正向这类相对安全的城市集中。这一人口成因是杨间的判断，平安大厦是否由总部直接管理、具体职能和建筑规模均未正式说明。（第425章）

## 出入与安保

大厦入口由两名体型、气质均显经专门训练的保安把守，无通行证者不得随意进入。杨间出示国际刑警证件后，保安仍须核验身份，数分钟后才发放仅一天有效的临时通行证；此举表明证件可作为内部识别依据，却不免除现场核查。杨间进入后察觉多道隐晦目光，认为楼内另有暗中安保力量。安保人员是否持枪、具体编制和面对灵异事件的处置预案，原文未写。（第425章）

## 家属保护功能

客服系统能依据杨间资料查询张芬身份、所在楼层，并将其引导至二十层。杨间想到其他国际刑警的家属也可能在此工作，结合严格门禁和暗中安保，推断平安大厦实际是国际刑警家属的保护基地：一旦此处出事，可能波及大量刑警及其所属地区的稳定。该“保护基地”定位为杨间的推断，不应视作已获总部明示的组织架构；楼内家属人数、其他楼层用途及日常工作内容均不详。（第425章）

## 目击反应

杨间离开客服台前往电梯时，一名客服与同事因察觉他身后的无头鬼影而恐惧。两人所见影子在地面倒映、周围没有对应人物，且随电梯上升被拉向天花板后消失；她们由此确认杨间体内鬼的存在。该事件没有造成直接伤亡，但显示即使在有组织保护的场所，普通工作人员仍会因近距离接触驭鬼者而承受心理压力。大厦是否因此采取额外措施、两名工作人员身份及后续情况，原文均未写。（第425章）

## 第520章·朋友圈以顶层监看鬼画行动

鬼画、鬼差事件发生时，朋友圈成员在平安大厦顶层开会，并通过数台无人机传回的画面观察市郊灰白鬼域。费总、方世明和姜尚白在此讨论杨间位置、鬼差关押失败的可能和后续对策；方世明决定暂不趁杨间在培训基地屋顶时动手。此处说明大厦顶层可被朋友圈用于秘密议事及接收外部影像，但并不证明整栋建筑由朋友圈控制，设备来源、楼层产权、无人机操作人和安保是否参与均不详。（第520章）

第425章。

## 第590-599章·顶层遭袭与楼下对峙

杨间为清除朋友圈成员，以五层鬼域袭击大厦顶层会议室；多人、部分厉鬼和顶端三层建筑被暂时隔绝。其后姜尚白在残缺顶层以鬼踩人反制无头鬼影，却仍被杨间铺开的黑影吞没；杨间取回鬼寿衣并把其尸体从高空扔至楼前。大厦此时一百层、高逾四百米，电梯已经全部停运，赶到楼下的李军等人若欲进入只能徒步攀登。（第590章至第597章）

方世明本体和杨间在顶层交战时均处于杨间三层鬼域内，楼下无法看见经过。第599章，杨间的鬼域一度消失，城市夜景恢复；随后两人从高楼坠至大厦门前，地砖炸裂、地面震动，路人聚集。方世明和杨间均未死亡，后者身体又在鬼域内恢复，方世明则发现街道仍被鬼域覆盖。大厦被隔绝的三层能否恢复、建筑内部是否还有其他鬼或幸存者、楼下人员之后是否进入均不详。（第597章至第599章）
