# 第三中医院

## 概述

中山市第三中医院是童倩在凯撒大酒店获救后接受治疗、又被篡改记忆鬼冒用身份的地点。杨间在第371章确认，正在该院治疗的“童倩”实为鬼；他认为该鬼夺取童倩后脑笑脸后，也盗用了她的身份，因此不能等待总部漫长核实。医院在第372章成为直接限制行动的现场，杨间以鬼眼鬼域覆盖整栋大楼，进入顶楼特殊病房找到目标。该院此前如何收治童倩、病房具体安全措施及常规医疗状况，本批未再说明。（第371章、第372章）

## 电话入侵

赵建国为确认医院内童倩情况，命总部联络员拨打第三中医院电话。电话接通后对方没有说话，赵建国起初即要求挂断；但耳机内随即传出怪异笑声，联络员身体僵硬、脸上露出笑容后倒地死亡，笑声还逐渐在通讯室中清晰。赵建国据此断定医院已经出事，杨间对童倩身份的判断大概率无误。电话是否由病房目标主动接起、院内其他人员是否也因笑声死亡，以及电话线路如何成为媒介，均不详。（第371章）

## 顶楼病房限制

杨间抵院后没有逐层搜查。门口一名保安同样带着怪异笑容，他判断中山市可能存在许多被更改记忆的鬼奴，故将目标锁定为尽快控制顶楼病房中的童倩。鬼域覆盖医院时，病房中的目标已站在窗前观察杨间进入，随即从口中放出扩散全院的笑声；杨间以第二层鬼域暂时隔绝声音，再带冯全、罗素一直接进入病房。（第372章、第373章）

病房内，目标穿病服、头缠绷带，外表仍是童倩身体。杨间的无头鬼影从后侵入，冯全的坟土手臂掐住其颈部并捂嘴，罗素一的染血细线再由脚下缠至全身，三重压制令笑声中止。杨间随后取走笑脸并黄金收容，把失脸后仍会活动的身体装进尸袋。冯全联系总部善后，曹延华安排沈良前来接收；尸袋何时离开医院、医院是否继续封锁及被篡改者如何处置均未写。（第373章、第374章）
