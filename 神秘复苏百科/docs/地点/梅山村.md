# 梅山村

## 概述

梅山村是杨间父亲生前居住、杨间返乡过年的村庄，位于阳镇附近，距县城与大昌市不远。村中没有梅花和山，名称来历不详；年轻人大多外出，返乡季也显得冷清。杨间白天以鬼眼巡查，没有发现直接灵异异常，但表妹梁圆、附近湿尸和父亲留下产业的线索令他认定此地存在待查风险。（原文回跳第680章、原文回跳第681章）

## 父亲遗留线索

杨间母亲与亲戚提到，杨父生前在村南有鱼塘、村后有一块曾建小庙的地、村西树林里曾养过一群狗并建木屋；狗又在某个夜晚全部消失。杨间认为这些持续亏本、看似无关的行为不像普通养家者会做的事，结合秦老曾暗示父亲与灵异有关，开始怀疑父亲早已接触灵异。此判断尚属推测，鱼塘、小庙、树林的现况与灵异联系均未证实。（原文回跳第681章）

## 老宅与遗像

杨家老宅为两层水泥房，长年空置而有青苔、老化痕迹，张芬返乡后已打理通风。二楼一间常年关闭的房间摆有杨父黑白遗像；杨间用鬼眼检查未见异常，仍认为遗像是父亲死后设置，未必是正确调查方向。夜间杨间离村进入梦境后，房内遗像与木桌消失，出现单人床，遗像中的青年疑似从房间走至阳台；这一段发生于梦境或现实、青年身份均不详。（原文回跳第680章、原文回跳第682章）

## 夜间异常

杨间在夜雨中被梁圆叫出老宅，离村后发现道路与白天不同，进入一处使其鬼眼、鬼影和鬼手都无法使用的灵异梦境。村口有白衣黑头罩尸体、村外荒野有无头尸体，镇上悬挂草绳吊尸；杨间怀疑这是自身既有厉鬼经历影响的梦境，而非梅山村白天固定景象。梦境是否依托梅山村、村民是否也被卷入、现实中的村庄安危均不详。（原文回跳第682章、原文回跳第683章）
