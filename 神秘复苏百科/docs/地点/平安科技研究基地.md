# 平安科技研究基地

## 概述

平安科技研究基地位于远离城市的郊区山区，地图上登记为一家正规科研公司，实际是承担总部灵异研究和实验的隐秘基地。第642章，杨间抵达时见新修柏油路由高速公路通向山区，基地外观近似研究所；其真实研究项目、隶属部门、建设时间、人员规模和完整安保体系均不详。（第642章）

基地大楼地下共五层，楼内不设电梯，只以宽阔楼梯连接；每层楼梯有数道大门，关闭后可单独封锁。杨间以鬼眼观察到多处无法窥见的区域，后又发现地下层之间有能隔绝鬼域的隔层。他据此认为整个建筑类似可分层封闭的大型黄金盒子，用于限制不可控灵异和防止其扩散；这是杨间基于结构作出的判断，材料是否确为黄金、每层实际关押物和完整封锁能力均未证实。（第642章）

## 地下第五层与实验设施

第640章，基地最高防范区域隔着玻璃陈列一口老旧棺材和一块带黑白照片的灵位。第642章，卫景带杨间通过三层封闭大门抵达地下第五层；王小明、陈博士和助手在多层特殊玻璃墙后的实验室工作。杨间将接受意识进入灵位、身体进入鬼棺的八音盒诅咒处置方案，因此第五层被作为本次实验核心地点。（第640章、第642章）

鬼棺与灵位所在小房间四壁厚重，鬼域难以覆盖，入口关闭后同外界几乎隔绝，只余一面小玻璃窗与通话设备。王小明称该设计既用于防备杨间失去意识后体内鬼复苏，也用于提高鬼棺压制效果。空间能否真正隔绝所有灵异、鬼棺压制是否足够，以及若实验失控基地能否承担关押功能，均不详。（第642章、第643章）

## 第644章·鬼童封楼后的异常

杨间在第643章命鬼童“杀死进出这栋楼的所有人”，令其在基地大楼内游荡。鬼童身穿鬼寿衣、捧带腐烂死人头，因新命令覆盖原先蛰伏指令而获得自主活动；这使基地从实验场所额外出现鬼童造成的灵异风险。陈博士事先已组织大部分工作人员撤离，仍有数名助理留在楼内整理资料。（第643章、第644章）

第644章，刘助理和同事在走道听到异响，目睹死人头滚动和鬼童出现后逃跑。死人头在本次没有正面盯住二人，鬼童又因被看见而暂时躲避，故未造成确认死亡；但两人已决定通知陈博士、王小明与安保科。鬼童命令覆盖范围、楼内实际剩余人数、基地是否已完全封锁及后续伤亡，至本批末均不详。（第644章）
