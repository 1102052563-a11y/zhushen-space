# 福仁商场

## 概述

福仁商场位于大昌市一条繁华主干道旁，周边有楼盘、小区和人流。第32章，杨间在电线杆广告上见到此处“重金求大师做法、驱邪，有效重赏一百万”的地址后前来查看，并以应聘保安身份进入。商场外仍有警方警戒线，内部大多熄灯且冷清歇业；公开信息称有人在商场失踪，警方最后获得的录像也显示失踪者在此出现。（第32章、第33章）

## 失踪与异常

- 杨间进入时闻到若有若无的尸臭，认出与第七高中灵异事件中的腐尸气味相同，并觉得黑暗深处似有东西窥视。（第32章）
- 保安刘强称失踪案可能是绑架或抢劫，且商场歇业后案件几乎不再发生；杨间则认为人在有限的商场内失踪且找不到，更符合灵异异常的可能。（第33章）
- 商场已请过大师处理，效果不佳，老板又去外地请更有名的人；丽姐虽承认此事，却拒绝向杨间说明细节。（第32章）
- 其中一间黑暗商铺存放十余具残缺人体模特。刘强将突然出现在巡逻通道的模特搬回店内后，原文显示该模特又自行立在店门口、头部朝向二人离去方向，但二人当时没有发现。（第33章）

## 人员与现状

商场门口张贴月薪六千、包食宿的招聘广告。丽姐负责招工，刘强与新来的杨间是第33章巡逻时的两名保安；刘强称先前另有一名保安已经失踪。杨间进入此处并非单为谋职，而是希望确认驱邪悬赏是否对应真实灵异事件，以取得能留给母亲养老的报酬。他暂定试用三天观察，尚未见到商场老板。（第32章、第33章）
