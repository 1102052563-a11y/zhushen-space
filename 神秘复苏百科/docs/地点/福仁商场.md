# 福仁商场

## 概述

福仁商场位于大昌市一条繁华主干道旁，周边有楼盘、小区和人流。第32章，杨间在电线杆广告上见到此处“重金求大师做法、驱邪，有效重赏一百万”的地址后前来查看，并以应聘保安身份进入。商场外仍有警方警戒线，内部大多熄灯且冷清歇业；公开信息称有人在商场失踪，警方最后获得的录像也显示失踪者在此出现。（第32章、第33章）

## 失踪与异常

- 杨间进入时闻到若有若无的尸臭，认出与第七高中灵异事件中的腐尸气味相同，并觉得黑暗深处似有东西窥视。（第32章）
- 保安刘强称失踪案可能是绑架或抢劫，且商场歇业后案件几乎不再发生；杨间则认为人在有限的商场内失踪且找不到，更符合灵异异常的可能。（第33章）
- 商场已请过大师处理，效果不佳，老板又去外地请更有名的人；丽姐虽承认此事，却拒绝向杨间说明细节。（第32章）
- 其中一间黑暗商铺存放十余具残缺人体模特。刘强将突然出现在巡逻通道的模特搬回店内后，原文显示该模特又自行立在店门口、头部朝向二人离去方向，但二人当时没有发现。（第33章）
- 第40章夜间，无人使用的感应手扶电梯自行启动，并从五楼通往四楼、三楼、二楼，最后运行到一楼；第41章商场白日看似正常，仍有尸臭反复出现。（第40章、第41章）
- 第42至45章，商场停电后先有无头腐尸坠下又消失，五楼出现无头模特人影，一楼女员工魏晓红的腐烂头颅滚出。江艳称四日前见魏晓红自行取下头部；无头模特可自行移动，却被严力击散后未显示异常。厉鬼本体、尸体与模特的关系仍不详。（第42章至第45章）
- 第46至50章监控揭示，未知中年女子从背后取下魏晓红头颅、换上自己的头；魏晓红随后又可给丽姐等换头。被换头者会保留原有外观与部分生活状态，脖颈留环形伤口并逐步腐烂，且会在停电后行动。换头鬼的杀人规律初步为必须从背后取头，正面对其可暂避攻击；商场内已不止一名被换头者。（第46章至第50章）
- 第51至54章，杨间以“背贴地面、不背对任何换头者”的规则带部分幸存者从五楼滑至商场外；逃跑者被鬼群取头并转化。警方随后破门封锁现场，但真正的换头鬼与鬼群仍留在商场，江艳尚在监控室。（第51章至第54章）
- 第56至60章，江艳的监控显示五楼服装店是最早异变处：黑影可拆拼假模特身体、融入无头模特并从背后换取活人头颅。杨间与严力重返此处，发现换头鬼会将假模特、尸体作为旧身体或诱饵，真正鬼体藏在尸群中。（第56章至第60章）
- 第61至63章，杨间以鬼域困住无头鬼，严力的鬼血将其限制并装入焊死的黄金盒。杨间确认商场内有数十具此前失踪者尸体，唯一幸存者为江艳；刘建明随后接管善后。至此换头鬼不再在商场活动，是否永久安全仅能以本批关押状态判断。（第61章至第63章）

## 人员与现状

商场门口张贴月薪六千、包食宿的招聘广告。丽姐负责招工，刘强与新来的杨间是第33章巡逻时的两名保安；刘强称先前另有一名保安已经失踪。杨间进入此处并非单为谋职，而是希望确认驱邪悬赏是否对应真实灵异事件，以取得能留给母亲养老的报酬。他暂定试用三天观察，尚未见到商场老板。（第32章、第33章）

第41章，老板[唐安](../人物/唐安.md)带罗大师及员工进入商场；第42章，驭鬼者[严力](../人物/严力.md)也以五百万元处理费争取委托。罗大师要求锁住所有出口做法，导致停电后在场人员受困；杨间已被李经理辞退，拒绝无偿协助。会计[江艳](../人物/江艳.md)则在男厕所躲藏四天，未被现场人员发现。（第41章至第45章）
