# 82号房

## 概述

82号房位于凯撒大酒店四层异常走道深处，门旁有微黄门灯，装修与71号房同为老旧民国风。第606章，杨间跟随被裹尸布逐步包住的持柴刀高大尸体抵达此处时，发现房门向走道外敞开；门把手上留有数道鲜红指痕。尸体高度腐烂、只流尸水而没有鲜血，杨间据此判断痕迹不可能由它留下，怀疑另一个房间中的鬼已经开门外出。（第606章）

## 房内与血手印

杨间冒险经过门口，见客厅空无钢琴，墙壁、天花板却布满鲜血手掌印，部分鲜艳得像要滴落，颜色使他联想到严力鬼血；他当时未见实体。持柴刀尸体被裹尸布完全包住、杨间扛其离开后，血手印从房内蔓延到走道两侧墙壁，并持续跟随他折返。杨间再次路过时，房内原有手印已经消失，说明手印位置会变动；这不能单独证明手印就是厉鬼本体。（第606章、第607章）

## 第607章·诅咒显形与退回

杨间以鬼手覆盖走道血手印、同时持有柴刀作为媒介后，看见大量无眼、衣着跨越不同时代的腐烂人影沿墙摸索；他们以染血手掌留下手印，并在接触被触发后同时伸出另一只手抓向杨间。该画面是否为媒介呈现的源头、幻象或鬼域景象，原文没有定论。杨间挥柴刀斩向最近手臂，众多人影手臂同时断开，走道手印迅速消散至82号房；敞开的门随即猛然关闭，门灯闪烁后变得近乎熄灭。房内厉鬼、人影与手印的确切关系、是否已被永久限制均不详。（第607章）
