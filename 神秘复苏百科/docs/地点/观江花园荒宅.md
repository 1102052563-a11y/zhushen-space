# 观江花园荒宅

## 概述

观江花园荒宅位于小区后方一片未完成拆迁的荒地，是一栋两层高、青砖砌成的民国风旧宅。它周边有破败木屋、被推倒的土屋与杂草，外墙被茂密爬墙虎覆盖，屋顶支撑瓦片的木头腐烂塌陷，长期无人居住而阴森。杨间巡视观江花园时，鬼眼产生越来越强的不安，因而把它列为工人失踪和镜中鬼线索的重点调查地点。（第187章）

## 建筑与旧闻

荒宅一、二楼外墙均未修窗，侧面也没有窗户；杨间认为这种没有采光结构的民居不合常理。张显贵从拆迁户处查得，宅子所在位置原是一个仅十余户的小村，后由秦总拆迁，因未找到原房主而未完全动工。当地人只知四十多年前有一名老人住在宅中，老人某个冬天病死，村民为其办过丧事，房屋此后锁起荒废；老人是否有亲属、旧宅更早的建造者及无窗目的均无从查证。（第187章）

## 室内布局

旧宅门口杂草丛生，两扇腐烂木门半开半闭；因无窗，即便盛夏日光强烈，屋内仍昏暗阴冷。一楼前堂和大堂较宽，堆有已受潮发霉的农具，显示后来曾被当作仓库。二楼走道尽头是砖墙，旁侧则有三间狭小房间：第一扇门外覆铁皮、实际为黄金打造且无把手；第二扇是厚重无把手铜门，与墙相融；第三扇是有七八道锁的木门，锁已被工具剪断。木门房内部看似仅约五平方米，青砖墙厚至少三米，杨间判断其建造用途并非住人。（第188章）

## 鬼镜发现与风险

杨间从已被打开的木门房进入更深处，发现一面铜架锈蚀、镜面却历经百年仍崭新的民国更衣镜，旁有原本用于遮盖的油布。房内另有工地安全帽、钳子、电钻等新近工具，墙上被钻出可望见对面在建高楼的小孔，镜子也有被移位痕迹。他据此推测失踪工人曾为找财物进入旧宅、搬开镜子后试图钻墙进入铜门房，而穿工服的镜中鬼线索亦指向这里。镜面照不出杨间；其戴鬼烛触摸后，手臂被镜内多只大小不同的手掌拉扯，留下青黑手印，遂判断其中有多只鬼。荒宅是否由早期驭鬼者修建、黄金门与铜门后藏有什么，至第190章不详。（第188章、第189章）

## 当前状态

第192章，杨间确认张伟的镜中鬼已被限制回鬼镜，遂用原油布遮镜并将镜子带走，认为失踪工人可能已被吞入其中。他要求张显贵维修加固荒宅、不得开启二楼剩余两门，以免房屋倒塌使未知之物暴露；张显贵又准备圈地建庙遮掩古宅。三日后，维修正日夜进行，庙宇地基也已开工。人皮纸则称剩余两房应彻底封存，其中有未知恐怖；杨间怀疑其仍有隐瞒，尚未开门验证。荒宅的建造者、工人下落及两房内容依然未解。（第192章至第194章）

## 出处

- 荒宅位置、外观、无窗结构与拆迁旧闻。（第187章）
- 室内三门、工地工具与鬼镜试探。（第188章、第189章）
- 直播冲突后暂未继续处置。（第190章）
