# 时间线

按批次逐章记录事件与其结果或意义。

- **第1章**：杨间阅读“雷电法王”的论坛帖，得知一名本应死亡多日的黑衫老人自行走动、敲门并进入发帖者家中。（敲门老人异象首次出现）
- **第2章**：周正在大昌市第七高中夜自习讲授“鬼无法被杀死”“能对付鬼的只有鬼”。（灵异生存原则首次明确）
- **第3章**：黑衫老人于学校形成鬼域，王老师开门进入黑暗后失踪；周正决定留守救人。（高中陷入灵异危机）
- **第4章**：杨间推测敲门声是规律，周正以腹中鬼手短暂压制老人并令学生撤离。（获得短暂逃生窗口）
- **第5章**：学生在楼梯鬼域中迷失，苏蕾因楼板坍塌被钢筋贯穿颈部，方镜把杨间推向厕所内另一只鬼。（逃生路线中断）
- **第6章**：方镜煽动同学将杨间推入厕所，杨间播放敲门音频逼退惨白手掌。（首次借鬼的声音自救）
- **第7章**：段鹏和郑飞先后被厕所鬼袭击，杨间循滴水声寻找出口。（厕所黑暗空间持续封锁）
- **第8章**：红色眼珠钻入杨间手背，令其看见白骨树、被钉黑影和厕所门。（鬼眼能力首次显现）
- **第9章**：周正说明鬼奴后因鬼婴破腹而出死亡；杨间以鬼眼看破鬼域，开始带同学下楼。（鬼婴成为新威胁）
- **第10章**：鬼婴攻击杨间时被其后脑鬼眼阻止，随即转而勒住王珊珊。（撤离再度受阻）
- **第11章**：杨间救下王珊珊，手臂新增三只鬼眼并逼退鬼婴。（鬼眼复苏加快）
- **第12章**：幸存者抵达操场后发现时间错乱、外界无反应；杨间判断操场仍在鬼域内。（未真正脱困）
- **第13章**：杨间与方镜通话，以暂时指路为条件索取羊皮纸。（交易开始）
- **第14章**：方镜交出羊皮纸；杨间发现其为会自行删除旧主信息、重写新主信息的人皮。（羊皮纸易主）
- **第15章**：杨间诱导方镜播放敲门音频，黑衫老人出现并袭击方镜。（验证音频引鬼的推测）
- **第16章**：幸存者抵达校门仍被鬼域黑暗封锁；杨间拨打雷电法王号码，铃声在校内响起。（发现通讯异常）
- **第17章**：诡异号码回拨并群发来电，钱万豪接听后听见敲门声。（电话陷阱再度引鬼）
- **第18章**：黑衫老人逼近校门，羊皮纸提示吞眼增加数量可尝试张开杨间的鬼域。（获得脱困方法）
- **第19章**：杨间吞眼生出第六只眼并释放红光鬼域，带八人离开学校。（首次成功脱离高校鬼域）
- **第20章**：八名幸存者抵达大昌市步行街；杨间警告论坛敲门音频危险，众人各自准备回家。（事件暂时告一段落）
- **第21章**：杨间通过周正卫星手机与接线员刘小雨通话，周正死亡消息传回国际刑警体系。（后续调查启动）
- **第22章**：赵建国建立“鬼敲门”A灾难级档案，并拟招揽已成为驭鬼者的杨间。（官方体系接触杨间）
- **第23章**：杨间送王珊珊回家，张伟因拒绝返校被警方找到协查。（幸存者回归日常受阻）
- **第24章**：杨间经历鬼眼复苏剧痛，登录保密网站了解全球灵异事件。（生存压力转向体内厉鬼）
- **第25章**：刘小雨揭示杨间父亲早已死亡，杨间发现家中“父亲”与遗像矛盾。（住处出现新异常）
- **第26章**：杨间确认无脸“父亲”为鬼，以多只鬼眼撕碎贴脸的染血报纸逃离住处。（报纸留下杨间面容）
- **第27章**：杨间发现红纸碎片可压制一只鬼眼，并以五眼首次发动鬼域。（获得暂缓复苏的线索）
- **第28章**：杨间用阿飞等人测试鬼域的循环、换位、扭曲距离与穿墙能力，记录维持极限约五分钟。（初步掌握五眼鬼域）
- **第29章**：王珊珊因颈侧手印和窗外异象崩溃，王彬在浴室发现密集小孩手印。（鬼婴威胁延续至其家）
- **第30章**：杨间等待报纸鬼未果，王彬来电请其处理女儿异常；杨间索价五十万，交易暂未成立。（杨间开始将灵异处置视作拿命换钱）
- **第31章**：杨间以鬼域将手机置入奸商脑中、以铃声逼其转账，获得资金后离开。（为母亲养老筹钱）
- **第32章**：杨间循百万驱邪广告进入福仁商场，闻到尸臭并应聘保安。（开始调查商场失踪案）
- **第33章**：杨间与刘强巡逻，发现假模特被搬回后又自行出现；王彬再度来电。（商场出现灵异征兆）
- **第34章**：鬼婴进入王珊珊家、封堵大门，王彬一家被困并向杨间求救。（危机由印记发展为直接袭击）
- **第35章**：杨间以鬼域覆盖出租车、穿越道路障碍，一分钟赶到王家小区。（以复苏代价争取救援时间）
- **第36章**：杨间击退成长中的鬼婴并使其坠楼，带王家撤离；王彬将报酬增至一百万。（鬼婴显出鬼域趋势）
- **第37章**：王家支付一百万后随杨间离开小区，杨间判断鬼域可隔绝其他鬼的追寻。（王珊珊追杀问题未解）
- **第38章**：刘小雨判定王珊珊颈印为鬼奴印，提出以驭鬼者印记覆盖的未验证方案。（获得暂缓追杀的理论方法）
- **第39章**：杨间以鬼眼印覆盖王珊珊的鬼婴印，鬼婴失去目标后改为随机跟随路人。（风险转为无差别袭人）
- **第40章**：杨间返回福仁商场夜巡，无人感应电梯从五楼依次运至一楼。（商场异常加深）
- **第41章**：杨间发现红纸出现裂纹、羊皮纸无法烧毁；唐安请罗大师入商场，尸臭再现。（商场调查转入公开做法）
- **第42章**：严力带无头腐尸现身，索价五百万处理商场厉鬼。（第二名驭鬼者介入）
- **第43章**：商场停电，尸体消失、无头人影现于五楼，严力上楼追查。（厉鬼开始显露活动）
- **第44章**：魏晓红的腐烂头颅滚出，罗大师锁门使众人受困；严力打散无头模特。（死亡和模特异常并存）
- **第45章**：杨间在男厕救出躲藏四天的会计江艳，得知魏晓红曾自行取头。（获得商场鬼目击线索）
- **第46章**：杨间发现厕鬼只在江艳背对时撞门，拍到保安服身影。（初步锁定换头规律与刘强嫌疑）
- **第47章**：杨间决定到五楼监控室查明身份，并答应江艳听令后带其离开。（调查转向监控证据）
- **第48章**：严力以血液暂封杂物间大门；杨间在监控室遇见刘强。（幸存者与疑似鬼分处五楼）
- **第49章**：监控显示陌生女子、魏晓红和丽姐连续换头，商场内存在多名被换头者。（确认鬼群）
- **第50章**：刘强确认自己已被换头并因头颅脱落死亡；江艳留守监控协助杨间。（获得远程预警）
- **第51章**：杨间穿过鬼群抵达杂物间，揭露丽姐为鬼并迫使幸存者出门。（失踪者全部现身）
- **第52章**：杨间令众人躺地避开换头，严力说明鬼血压制与黄金关押方法。（规律成功救人）
- **第53章**：唐安等人为逃生支付劳务费，杨间取得超千万元并带人撤离。（交易完成）
- **第54章**：幸存者背贴地面滑出商场，警方破门接管现场；严力提出合作抓鬼。（商场危机暂时外溢受控）
- **第55章**：李经理举报杨间敲诈，警方因其特殊档案和上级警告无法擅自拘捕。（身份权限首次生效）
- **第56章**：杨间决定同严力重返商场；江艳查出换头鬼源于五楼服装店无头模特。（锁定源头）
- **第57章**：杨间与严力达成收益和情报对半的合作，进入服装店。（首次主动合作抓鬼）
- **第58章**：店内假模特集体异动，真鬼以两米模特从背后袭击严力。（鬼设伏反击）
- **第59章**：尸群涌入服装店，杨间张开鬼域争取三分钟找出真鬼。（行动进入限时搜索）
- **第60章**：杨间识破诱饵，真鬼摘下严力手臂欲换鬼血；杨间以鬼域锁定它。（抓捕目标确认）
- **第61章**：严力以鬼血限制无头鬼影，将其收进黄金盒并以金箔封裹。（换头鬼初步关押）
- **第62章**：杨间以鬼域幻象夺取黄金盒保管权，双方约定严力找买家、杨间谈交易。（合作暂续）
- **第63章**：杨间带江艳离开商场，向刘建明报告数十人死亡和事件结束。（福仁商场事件收束）
- **第64章**：杨间焊死黄金盒并向刘小雨询问国际刑警权责与待遇。（开始评估身份选择）
- **第65章**：红纸裂开，巨大灰色鬼域掠过江艳小区，杨间看见绿烛与神秘女子手部幻象。（遭遇S级征兆）
- **第66章**：杨间决定以提高身份、交换情报为活路，江艳以工作能力换取其庇护。（建立互惠关系）
- **第67章**：江艳操作黄金杠杆令资金翻倍；张伟约杨间见面。（日常资源与同学线并进）
- **第68章**：严力警告黄金盒消息泄露，吴峰率人接触杨间收购盒子。（夺盒势力显现）
- **第69章**：杨间骗取吴峰预支工资并报警，将其团伙交给刘建明。（未动用鬼力化解威胁）
- **第70章**：郝少文囚禁严力逼问盒子，杨间以鬼域和敲门声进入其藏身处。（营救冲突开启）
- **第71章**：杨间以鬼域反制郝少文黄金弩箭，开始倒计时审问。（夺盒团伙失去主动）
- **第72章**：郝少文枪杀手下仍无法破解鬼域，杨间迫使其供出俱乐部雇主。（获得幕后线索）
- **第73章**：郝少文说出双鬼平衡可延缓复苏，杨间在倒计时结束后移走团伙。（得到极端续命方案）
- **第74章**：红纸彻底失效，杨间长出第七只鬼眼；他与严力约定次晚交易。（复苏期限缩短）
- **第75章**：杨间破解郝少文通讯录，向名单人播放敲门音频。（人为定向传播诅咒）
- **第76章**：杨间进入小强俱乐部，叶俊以鬼手印索取交易五成抽成。（俱乐部勒索新人规则暴露）
- **第77章**：杨间用黄金子弹重创叶俊，以暴力压下成员反弹。（确立自身规矩）
- **第78章**：王小强现身调停；孙俪红到俱乐部洽谈收购无头鬼。（交易进入正式谈判）
- **第79章**：杨间以双鬼驾驭法为交易条件，公司要求先解决黄岗村灵异事件。（获得下一行动目标）
- **第80章**：杨间交严力假盒作后路，并花两千余万购黄金装尸袋等装备。（为黄岗村孤注一掷）
- **第81章**：赵开明夜访并以江艳施压，杨间持黄金枪与其对峙。（新旧大昌市驭鬼者冲突显现）
- **第82章**：杨间独自进入黄岗村，叶俊、张韩等俱乐部驭鬼者也随后入村。（调查队伍集结）
- **第83章**：村口灵堂出现陌生死者；杨间与张韩听见废木屋外的虚弱咳嗽。（首个灵异线索出现）
- **第84章**：杨间租住刘根荣家，鬼眼复苏瘫痪时有脚步和咳嗽进入二楼。（夜间危机降临）
- **第85章**：未知鬼物敲击关押无头鬼的黄金盒，人皮纸覆盖盒子后敲击停止。（无头鬼是否脱困不明）
- **第86章**：杨间确认黄金盒未开，追脚步时误伤欧阳天。（脚步会将驭鬼者误导相撞）
- **第87章**：贺胜被杀、体内鬼疑似离体，其情妇存活。（确认厉鬼优先袭击驭鬼者）
- **第88章**：杨间将咳嗽目标命名为病鬼，以手绘地图指向村口灵堂。（队伍决定查棺）
- **第89章**：灵堂内咳嗽与脚步再次诱导众人汇合，欧阳天失踪。（灵堂异常加剧）
- **第90章**：欧阳天尸体在棺材旁被发现；其余驭鬼者撤离，杨间独留开棺。（调查转为孤身冒险）
- **第91章**：张一鸣三人离村后被送回黄岗村；杨间窥见棺内鬼手并守棺。（确认棺中有鬼）
- **第92章**：遗像被查出为疑似合成图且发生变化，黄岗村旧档案送达杨间。（历史线索补全）
- **第93章**：档案揭示村庄曾整体消失，杨间确认病鬼与凶鬼并存。（双鬼关系成为核心）
- **第94章**：杨间推断凶鬼规则为落单驭鬼者必死，老婆婆咳嗽暴露病鬼线索。（锁定可行动目标）
- **第95章**：叶俊腐蚀老婆婆，张韩制住目标，杨间以黄金装尸袋收容。（病鬼疑似被关押）
- **第96章**：杨间确认老婆婆为鬼奴，黄岗村平衡破裂；他放出无头鬼影应急。（村庄进入失控黑暗）
- **第97章**：无头鬼影附身老婆婆，杨间躲入黄金装尸袋等待平衡重建。（以放鬼换取生机）
- **第98章**：杨间在尸袋中向赵建国施压，要求追查删档与公司负责人。（黄岗村交易背景受质疑）
- **第99章**：杨间发现村民已成无眼鬼物，与羊皮纸达成第二鬼驾驭交易。（人皮纸交易启动）
- **第100章**：无头鬼影令村庄部分恢复，杨间以其为筹码，人皮纸开始吞噬该鬼。（交易结果未明）
- **第101章**：人皮纸吞噬老婆婆及其体内至少两只鬼，再放出无头鬼影侵入杨间。（第二鬼交易启动）
- **第102章**：杨间交出五只鬼眼压制无头鬼影，正式驾驭第二只鬼。（两鬼形成暂时制衡）
- **第103章**：杨间放出张韩并共同到灵堂，鬼棺自行打开。（棺中存在即将现身）
- **第104章**：冯全从鬼棺坐起，承认咳嗽是聚集驭鬼者的提示。（黄岗村误解得到部分澄清）
- **第105章**：冯全说明村民、遗像男子和鬼棺的关系，村内天色再次昏暗。（平衡仍不稳定）
- **第106章**：遗像年轻男子在灵堂外现身，冯全要求三人守棺并准备关押。（守棺之战开始）
- **第107章**：杨间鬼域无法覆盖遗像男子，男子入域后反令鬼域失控；三人联手阻截。（厉鬼显出压制能力）
- **第108章**：冯全以浓雾和咳嗽压制男子，并说明其可令三只厉鬼能力失效。（杨间怀疑冯全借人试探）
- **第109章**：杨间入鬼棺暂失实体并感知全村；冯全收容男子形象后，鬼的脚步仍重现。（关押未解决本体）
- **第110章**：杨间推断黄岗村整体与鬼棺相连，摘下冯全头颅并主张送村鬼回棺。（提出离村方案）
- **第111章**：村鬼以刘根荣形象回到鬼棺，杨间收容冯全身体。（黄岗村事件暂止）
- **第112章**：黄岗村随鬼回棺消失，杨间与张韩运走鬼棺并抵达公司封锁线。（回收谈判开启）
- **第113章**：吴越承诺约一周提供成功率约百分之十五的第二鬼驾驭方案，回避删档责任。（交易与追责冲突）
- **第114章**：吴越承认以高伤亡概率隐瞒信息，杨间向其开枪。（杨间要求有人负责）
