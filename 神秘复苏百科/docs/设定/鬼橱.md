# 鬼橱

## 概述

鬼橱是杨间在观江花园荒宅二楼第二间密室中发现的老旧木制橱柜，也是他为橱内存在暂定的名称。第296章，他穿过封死的黄铜门进入密室，室内只有一张普通木桌和这只上下两层的旧橱柜。杨间拍碎木桌后确认桌子没有异常，便将怀疑集中到橱柜；名称只是杨间基于目击所作命名。（第296章、第297章）

## 外观与表现

第二间密室入口是一扇无钥匙孔、直接封死的厚重黄铜门。室内橱柜已脱漆，分上下两层：上层门为木格栅，格内始终漆黑；下层为两扇紧闭柜门。鬼眼无法看穿上层格栅后的黑暗，手机手电光也只能深入数厘米。杨间隔着格栅观察时，曾见一双眼睛贴在门后与其对视；退开后眼睛消失。他确认橱内有东西，却不能确认其是何种鬼或是否真为孩童大小。（第296章）

## 处置与线索

第297章，杨间试探后确认橱内存在没有从上层出来，也没有在他靠近时发动袭击。他称其像一只待在柜内的“自闭”鬼，但这只是当次现象，不能推导为无害。因下层柜门没有开启、杀人方式和用途皆未知，他决定不动橱柜，只取走顶部的干墨水瓶、民国钢笔和留有压痕的记事本，准备寻找原主人线索。第318章，程雄从记事本压痕恢复五句话，其中包括“可恶的橱子，把东西还我”“我已经没有东西可以给你了”及“我想要离开这里”。杨间据此推测旧主人至少进行过四次交易、曾被取走某物并对鬼橱既依赖又忌惮；这只是基于残句的推断，原主人的身份、交易总数和结局均不详。（第297章、第299章、第318章）

## 纸条问答

第319章，杨间在疏散古宅周边工人后，将写有“你是谁”的纸条塞进上层格栅。纸条被橱内力量拉入黑暗，约十秒后从下层缝隙落回，下面多出黑色歪斜的“鬼”字。他因此确认橱内存在能理解并以纸条作答的东西，但未打开上层柜门；“鬼”是该物的自述，不能据此确认其形体、数量或是否等同于一般厉鬼。（第319章）

## 交易与交付

第320章，杨间询问能否找到赵磊，得到“你给我眼睛，我给你赵磊”的回答。他判断“眼睛”指鬼眼而拒绝交易，并注意到答复承诺直接交付赵磊、而不只是提供位置。为测试条件能否随要求变化，他改要一碗蛋炒饭，获得“给我一碗血，我给你蛋炒饭”的条件。鬼橱未指定人血，杨间以狗血试验；上层柜门自行打开、取走装血的碗，上门关闭后下层柜门开启，交付一碗仍冒热气的蛋炒饭。饭碟下另有纸条“帮我把橱子搬出去”。

本次证明鬼橱至少能以书面提出交易、收取物品并从下层交付实物，但“血”的合格范围、蛋炒饭来源、交付是否必附额外要求，以及是否会因杨间拒绝搬橱而产生后果均不详。（第320章）

## 出处

第296章、第297章、第318章至第320章。
