# 摆钟诅咒

## 概述

摆钟诅咒是裴东、周杰峰等人遭受的一种时间型灵异影响。周杰峰称诅咒来自一台老旧摆钟，会为受害者定下死亡时间，期限抵达便会死亡；团伙长期寻找摆钟却未能找到本体，只在特殊地点的老照片中见过其外观。（原文回跳第637章）

## 已知表现

- **时间即生命**：摆钟会为受诅咒者设定时间，时间到达后受害者死亡。时间具体长短、是否可被延长、死亡方式及是否所有人期限相同均不详。（原文回跳第637章）
- **压制厉鬼复苏**：周杰峰称诅咒也能压制驭鬼者体内厉鬼复苏，使仅驾驭一只鬼的裴东团伙成员能够长期活动。这种压制并非没有代价，且只基于团伙经验，不能确认适用于所有驭鬼者。（原文回跳第637章）
- **三鬼阈值**：按周杰峰说法，受诅咒者一旦驾驭第三只鬼，诅咒会消失，本人也会立即死亡；团伙因此不敢以通常的驾驭更多厉鬼方式延寿。该因果和“生命诅咒”的机制尚未由其他来源证实。（原文回跳第637章）

## 来源地点

周杰峰称摆钟所在之地范围不大却极其特殊，受诅咒者只能听见到点后的钟声，始终找不到摆钟本体。他们无法以黄金关押摆钟，因为不知其具体位置；该地点名称、进入方法、照片来源和是否仍可进入均不详。（原文回跳第637章）

## 与童倩事件的关系

裴东团伙把童倩曾更换赵磊身体视为摆脱诅咒与厉鬼复苏的希望，故用安眠药控制她并对其身体进行实验。杨间认为更换身体未必没有后遗症，也未认可团伙的推论；童倩获救后，摆钟诅咒本身仍未被处理。（原文回跳第637章、原文回跳第638章）
