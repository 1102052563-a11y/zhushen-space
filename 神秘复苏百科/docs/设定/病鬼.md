# 病鬼

## 概述

“病鬼”是杨间对黄岗村中发出虚弱咳嗽声的灵异目标所取的暂用代号。它的咳嗽像病入膏肓之人，伴随可将驭鬼者引至同一点的脚步声。第96章修正：第95章被装入黄金装尸袋的无名老婆婆不是病鬼本体，而是被厉鬼操控的鬼奴；其被收容后，病鬼败退，黄岗村原有平衡随之彻底破裂。（第88章、第95章、第96章）

## 已知表现

- 虚弱咳嗽：杨间、张韩在废木屋旁首次听见贴近身后的咳嗽，循声进入屋内却未发现目标；灵堂内空房亦曾出现同样声音。（第83章、第89章）
- 脚步诱导：脚步声会令不同方向的驭鬼者追至同一位置，曾使杨间、张韩相遇，也在灵堂内把张韩与张一鸣等引至楼道。杨间据此怀疑该目标有聚集驭鬼者的作用。（第86章、第89章）
- 疑似鬼域：目标能在查无实体的屋内发出咳嗽、再从现场消失，杨间据此推测可能拥有鬼域；该推测尚未得到证实。（第89章）
- 老婆婆鬼奴：第95章，村中一名老婆婆咳嗽声与病鬼一致。叶俊以尸水使其严重腐烂后，她仍能起身关电视，被张韩背后鬼制住并由杨间装袋；第96章，杨间据其被轻易收容及随后失衡，修正判断为鬼奴而非病鬼本体。（第95章、第96章）

## 与黄岗村事件的关系

杨间认为黄岗村至少有两只鬼：一只通过脚步聚人、另一只杀害落单驭鬼者；该判断来自贺胜、欧阳天遇害与张一鸣三人聚坐整夜未遭袭的对照。病鬼可能对应前者，但尚无直接证明。第96章，老婆婆鬼奴被收容后，村庄骤然变暗、村民与灵堂消失，杨间判断病鬼已彻底败退，另一只凶鬼开始接管黄岗村，证实两者此前存在某种平衡。（第90章、第93章至第96章）
