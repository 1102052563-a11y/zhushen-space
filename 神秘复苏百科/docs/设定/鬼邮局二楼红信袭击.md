# 鬼邮局二楼红信袭击

## 概述

鬼邮局二楼红信袭击，是杨间在原文回跳第783章撕毁二楼集体红色信件后出现的灵异反应。红信任务原地址为“大川市明月小区7栋301室”；杨间认为若按任务投递，可能重演福寿园式外部灵异事件，因此选择以撕信换取全层上楼机会，并承担对二楼信使发动的厉鬼袭击。（原文回跳第783章）

撕信后，二楼21至27号七间房门同时打开，阴冷气息自楼下蔓延，房内成为未知厉鬼的袭击媒介。袭击并非在一处固定房门持续发生，目标会被靠近房门的无形力量拖入黑暗房间，随后房门关闭、惨叫中断。（原文回跳第783章、原文回跳第784章）

## 已见规律

- **红信撕毁触发全层袭击**：红信被撕碎后，二楼所有信使均被卷入本次代价，不能以未拿信、未参与讨论为由避开。（原文回跳第783章）
- **房门为拖拽媒介**：一名信使在23号房门开启时被吸入，钱蓉在21号房卧室被吊死，万兴及其他信使也被不同房门拖入。厉鬼本体是否藏在某一间房、在七房间转移，或只是借门发动规律，均未确认。（原文回跳第784章）
- **关门后的短时空档**：杨小花观察到每次有人被拖入、门重新关闭后，后续受害者都靠近尚未关上的房间，因而推测刚关闭的房门前可暂时安全。该结论来自短暂观察，持续时间、是否适用于全部门及能否主动关门均未验证。（原文回跳第784章）
- **走廊四角假设**：蔡玉认为回字形走廊的四个角位于房间分界线中央，可能避开房门袭击。该判断与杨小花的“闭门安全”推测冲突，两者均未验证。（原文回跳第784章）

## 已知伤亡

陈星在二楼熄灯后因手脚被杨间打断、无法进入房间，次日消失，疑已死于邮局夜间规律；死因未亲见。红信袭击期间，钱蓉死于21号房卧室吊杀，万兴被拖入房内，另有至少两名未具名二楼信使被门内力量带走。被拖入者是否全部即时死亡、尸体是否留在房内均待后文。（原文回跳第782章、原文回跳第784章）
