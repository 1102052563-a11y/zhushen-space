# 鬼踩人

## 概述

鬼踩人又称踩人鬼，是姜尚白隐藏、未被总部记录的一只鬼。第595章，姜尚白面对无头鬼影时双脚同时离地，脚下阴影中随即伸出一双布满尸斑、如死人般的脚底板代替双脚落地；他因此仿佛踩高跷般悬空，却仍能正常行走。姜尚白说明，这只鬼并非他自行偶遇后驾驭，而是有人主动为他“按上去”的；朋友圈内部知道其存在。安排者、驾驭过程、获得时间、使用代价及其是否有正式档案均不详。（第595章）

## 已见规律

- **踩中活人可致死**：姜尚白称，死人脚掌行走时一旦踩中活人，目标必死；即使是驭鬼者也未必能挡住一脚。该效果没有在本批直接以活人受害者验证，能否被哪些灵异能力抵抗不详。（第595章）
- **可以接触厉鬼**：这双脚可以同真正的鬼触碰。姜尚白曾在一次未说明的灵异事件中踹走一只鬼；被踹目标会停滞数秒、数十秒或数分钟不等，期间绝对不会行动。旧事件的地点、目标、停滞精确时长和是否有其他条件均不详。（第595章）
- **对无头鬼影形成短暂压制**：第595章，姜尚白让鬼脚踩中地面的无头鬼影。鬼影抓住他的手臂立刻松开、退回影子状态，并开始摇晃挣扎；杨间也确认自己暂时无法驾驭鬼影，认为效果近似令鬼“死机”。此压制不是永久的，杨间随后扩大鬼影范围后，一双脚无法继续压住整片黑影。（第595章、第596章）

## 与姜尚白的关系

姜尚白以鬼寿衣抵御袭击、以鬼域移动、以鬼踩人限制厉鬼，自认三只鬼配合足以面对S级以下灵异事件。面对杨间时，他曾因踩住无头鬼影而认为双方有谈判余地；但杨间将鬼影铺开后，他无法用单一落点稳定压制整层黑暗，最终被影手拖入鬼影。姜尚白死亡后，原文没有写鬼踩人是否复苏、遗留在尸体何处、由谁处理或能否被杨间取得。（第595章、第596章、第597章）
