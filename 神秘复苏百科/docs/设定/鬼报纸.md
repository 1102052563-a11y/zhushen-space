# 鬼报纸

## 概述

鬼报纸是杨间在凯撒大酒店异常走道三十一号房取得的染血红色报纸。房间四壁贴满同类旧报纸，每张报纸中央都挂着一张闭眼、五官齐全的人脸，数量约有两三百张；杨间据此判断，真正具备灵异力量的报纸应混在其中。它与夺脸、改写记忆的鬼有关，但鬼报纸是否就是该鬼本体、二者如何分离，原文尚未说明。（第362章、第363章）

## 外观与所在

三十一号房灯光明亮，墙上的旧报纸均带血迹，人脸则有男女老少之分。杨间估计墙上人脸存在时间最多约十日。房中发现童倩的人脸后，他进一步推断，遇害者的身体被放在十三号房，人脸被放在三十一号房；这只是杨间针对该酒店布置作出的判断，不能据此确认所有受害者均遵循相同处置方式。（第362章）

罗素一撕取墙面人脸和报纸时，发现一张染血红报纸上密布赵磊字迹，旁边挂着一张杨间不认识的中年男子脸。这些字包括“找到杨间”“监视他”“你是鬼”“忘记你的过去，你是鬼奴”，显示该物至少曾被用于写入与记忆、身份有关的内容；字迹的书写者虽被杨间认定为赵磊，具体是在何种受控状态下写下仍不详。（第363章）

## 已见能力与规律

该报纸会主动袭击近距离目标。罗素一撕下有赵磊字迹的报纸后，纸面鼓起并向其面部覆盖；杨间以无头鬼影压住报纸后，报纸和其上人脸仅短暂挣扎便恢复平静，墙上其余人脸也随之掉落，其他报纸失去诡异表现。由此，杨间将这张写字报纸判断为源头，其余纸张则是其灵异力量的衍生物。（第363章）

杨间推测鬼报纸篡改记忆的过程是先夺取人脸、在报纸上写入内容，再把人脸贴回；普通人即便未被夺脸，靠近时也可能受到记忆影响。他强调这只是根据酒店现场、赵磊字迹和自己先前接触经历得出的推断，不是已被完全验证的规则。鬼报纸的完整袭击范围、能否一次影响多人、写入内容是否必定生效，均不详。（第363章）

## 收容与关联

杨间认为报纸恐怖程度不高，无头鬼影即可限制，遂将其卷起、以金纸包裹并放入黄金容器收走。他取得报纸后，三十一号房的其余人脸和普通报纸不再显示原有异常；赵磊原脸却未在墙上找到，赵磊身体能否恢复仍无答案。（第363章）

酒店墙上童倩的人脸使杨间怀疑，当时医院病房中以童倩身份出现的对象可能是鬼；但本批没有直接查验病房对象，也没有确认真正童倩的意识、身体与其被夺人脸之间的全部关系。因此，鬼报纸与童倩身份被冒用一事确有关联，具体链条仍待后续原文证实。（第362章、第363章）

## 第367-368章·取脸、书写与鬼影拼接试验

杨间为确认冯全是否受记忆篡改，将鬼报纸覆到其脸上。约一分钟后，报纸松动，杨间撕下时连同冯全整张脸一并取走；纸上随即显现赵磊字迹，包括“忘记自己被袭击”“找到合适机会杀了杨间”等未完成指令。杨间据此认为，该物呈现的更像植入恶念而非任意重写全部记忆；冯全原先的改写只完成一部分，这是针对本次纸面文字的判断。（第367章）

杨间擦去旧字，写入忘却恩怨、感激和听令等内容，又以“喜欢称杨间为腿哥”检验是否生效。报纸不能把人脸装回，说明取脸、写字、装脸的完整流程可能属于原篡改记忆之鬼，单独持有的报纸是残缺灵异；杨间以无头鬼影压住报纸后使冯全面脸脱落，再将脸拼回。冯全醒来称杨间“腿哥”，表现符合写入内容，故杨间判断本次改写成功。书写内容的长期稳定性、可改范围及对其他对象是否同样有效，均不详。（第368章）

## 第373-375章·同源失效与童倩脸的改写

杨间在第三中医院限制冒用童倩身份的鬼后，试图以鬼报纸取走目标脸上的笑脸；报纸贴住数分钟仍无效果。他据此意识到，报纸只是这只篡改记忆鬼的一部分，不能拿同源部分去夺取鬼自身的脸，改由无头鬼影强行完成分离。这次失败说明报纸的取脸能力至少不适用于其同源目标，但不能据此推定对其他厉鬼或所有人脸均无效。（第373章）

第375章，杨间将真正童倩的人脸放上报纸，纸旁浮出“杀死杨间”的字样，显示该脸所承载的记忆曾被植入杀意。他涂掉旧字，写入信任杨间、忘掉厉鬼植入记忆、谨慎善良等内容；随后才把该脸拼到赵磊身体中央。此处只记录纸面文字和杨间的处理，童倩是否会按新内容苏醒、改写是否完整或永久，均未写。（第375章）

## 第392章·对异常鬼婴写入三项命令

杨间为控制从王珊珊腹内剥离的异常鬼婴，再次用染血鬼报纸取下无头鬼影的脸，并写入三条简单命令：完全听从杨间和王珊珊；没有二人吩咐不得杀人；不得随意出现在人群中。写完后他把脸拼回鬼婴，认为若改写有效，这三项已足够，若无效则更多文字也无济于事。鬼婴在此之前已未按既有规律攻击杨间，王珊珊接手后也未显凶相，但这些表现不能单独证明报纸命令已经生效。命令的持续时间、能否约束其成长后状态及失败时的后果均不详。（第392章）

## 第430章·对李瑶植入爱情记忆

杨间在鬼域内限制李瑶后，以鬼报纸覆盖其脸部。李瑶虽剧烈挣扎，却无法撕破报纸；约十秒后失去动静，杨间取纸时发现纸上已粘住一张保留绝望神情的李瑶脸。他判断李瑶并未死亡，而是脸和原有记忆被报纸取走。这次表现再次确认报纸可使普通人失去脸与记忆，但取脸的完整条件、受害者身体在无脸状态下的生理状况，以及对驭鬼者是否适用仍不详。（第430章）

杨间认为遗忘不利记忆较易，真正困难的是植入与现实不矛盾的新记忆：把李瑶设为亲属或旧友会因可查证的现实经历产生破绽。他最终写入李瑶忘记当晚八点后发生的事、初见杨间时已有好感、醒来后形成至死不渝爱情，并愿意帮助杨间和其亲友等内容。此做法只记录了杨间的设计和书写，李瑶尚未醒来，爱情、忠诚和遗忘是否生效、能维持多久及是否会被外人察觉均未验证。（第430章）
