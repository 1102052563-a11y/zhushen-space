# 重大灵异事件记录本

## 概述

重大灵异事件记录本是总部休息室内供杨间翻阅的一本厚图册式资料，记录影响重大或神秘未知的灵异事件。第486章，刘小雨称其为“重大灵异事件的记事本”，允许杨间在会议前查阅，以增加其对外界事件的了解。原文没有交代编纂者、更新周期、保密等级、是否每名驭鬼者均可阅读，不能把一次休息室内的阅览视为普遍公开制度。（第486章）

## 已见记录

杨间翻到的“幽灵列车事件”发生在欧洲，配图为一辆穿行城市、引发周边人群恐慌的老旧列车，事件级别标作未知。另有岛国的“鬼井事件”，又称“鬼寺事件”，资料标为S级未解决，配图显示一座被灰蒙雾霾笼罩、若隐若现的寺庙。两项记录只提供名称、地点或级别与画面，起因、伤亡、处理者及后续均未写。（第486章）

记录本也收录大昌市饿死鬼事件，图片是一座被青黑色鬼域覆盖的城市，状态标为“已解决”。杨间还看到国内的“鬼湖事件”，其无人机高空配图是一片漆黑如墨、平静异常的小湖；他由此联想到曾在平安酒店死亡的林山体内与水有关的鬼，但二者是否有关只是他的联想，资料没有作出关联结论。（第486章）

## 信息用途与关联说法

刘小雨解释，大型灵异事件难以完全隐瞒，各国有记录并会共享通报，信息有助于解决事件。她又转述一名驭鬼者的观点：不同地点的两起灵异事件或许并非偶然单体事件，可能由同一只鬼造成。杨间因此想到鬼拼图假设，但这是一种未被当场验证的理论，不能据记录本把不同事件直接认定为同源。（第486章）

杨间认为处理灵异事件时，情报差异会直接影响处置者是否能作出基本正确判断：有情报者较能从容，无情报者则须步步谨慎。该判断解释他认真阅读记录本的原因，却不证明记录内容完整或可靠；他翻阅后仍看到许多国内事件只标为未知级别，显示总部当时也未掌握其全貌。（第486章）
