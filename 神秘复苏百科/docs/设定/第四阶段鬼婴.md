# 第四阶段鬼婴

## 概述

第四阶段鬼婴是王小明在医院行动失败后，对更高阶段衍生鬼所作的暂定称呼。第246章，杨间在九只鬼眼复苏、鬼域扩大至约一千米时，隔绝了医院内第三阶段鬼婴；仍能闯入其鬼域的目标却呈叶枫外貌、穿叶枫的寿衣并长有叶枫体内第二只鬼的牙齿。杨间据此否认目标是叶枫本人，王小明随后将它归入“第四阶段鬼婴”。该称呼、成长方式和规律均出自本次观察与王小明判断，尚非完整档案结论。（第246章至第249章）

## 已见表现

- **使用死亡驭鬼者的厉鬼**：该目标外貌如失踪的叶枫，寿衣阻挡干枯手指刺入身体；其口中怪牙又能咬开杨间鬼域。杨间确认这两项皆曾属于叶枫，故推断鬼婴能驾驭被其吞食的驭鬼者体内厉鬼。（第246章、第247章）
- **吞食鬼烛**：杨间点燃最后约十厘米鬼烛逼近时，目标直接吃去近一半烛身。残余烛光仍使其短暂停滞，杨间才得以绕过寿衣将干枯手指刺进其颈部；这只能说明鬼烛在本次仍有局部压制，不代表可稳定克制该类鬼。（第247章）
- **受限与收容**：手指刺入颈部后，叶枫外貌的目标暂不行动。杨间尝试用鬼影脱下寿衣时，其手和眼又有异动；他判断寿衣与鬼牙可能形成平衡，贸然脱衣会使手指压不住完全复苏的鬼牙，遂将尸体连同寿衣、手指一并装入黄金箱暂封。（第248章）

## 风险推断

王小明认为第四阶段鬼婴的触发条件可能是吸入大昌市黑青阴霾。其依据是自己连续呼吸阴霾后手背渐呈青黑，如王珊珊曾受鬼婴留下印记一般；他进而推断全城吸入阴霾者都可能成为目标。此为个人推测，未在本批由其他个体直接验证。王小明还认为前三阶段鬼婴靠普通进食已停止成长，第四阶段须“吃鬼”，被吃掉的厉鬼会被其驾驭；已死六名驭鬼者可能意味着更多高阶段个体出现，但数量和转化条件均不详。（第249章）

## 当前处境

杨间、张韩和王小明虽暂时离开医院，却因更浓阴霾压缩鬼域，未能带走黄金箱。箱子被一只两三米长的青黑手从医院窗内抓住，仍悬在窗外；手指、寿衣和目标均随箱遗留。王小明因此认定先前钉住的是错误目标，鬼烛也已耗尽，行动未能限制源头鬼。第四阶段鬼婴是否仍被手指压制、会否脱离黄金箱和其后的行动，至第250章均不详。（第249章、第250章）

第251章，医院十六层的黄金箱被持续敲打至变形、箱口撕开，叶枫外貌目标已离开。病房内仅见该目标咀嚼后掉下干枯手指的黑色弯曲指甲，继而取回并穿上寿衣；叶枫残头也被阴霾吞没。此过程证实目标已脱离黄金箱，且干枯手指未能持续限制它。它之后在城内的去向、是否仍保有鬼牙和实际攻击对象不详。（第251章）

第266至270章，杨间确认自己尚可清除第一至三阶段鬼婴，却不敢断言能处理第四阶段或源头；第七中曾聚集上千只不同阶段鬼婴，后在赵开明的计划成功后全部消失。杨间遭遇的变化鬼无法被鬼眼鬼域看见，却能侵蚀其鬼域；其外貌近似周正，穿叶枫寿衣、持干枯手指，腹部有不明蠕动。赵开明称体内鬼已借饿死鬼收集“拼图”，但该目标是否就是第四阶段鬼婴进一步成长、饿死鬼或另一只未知鬼，尚无定论。（第266章至第270章）

第271至275章，周正外貌、穿寿衣并持干枯手指的目标先以皮鞋脚步为诱饵、再以双手和鬼牙近身袭击杨间；近身棺材钉将命中时，局部时间被重置。第273章，杨间在第七中北门门卫室以远距棺材钉贯穿其头颅，目标倒地后全城阴霾消散，衍生鬼婴也失去力量。赵建国等人将连钉尸体以金箔、金绳和金箱封存。该目标与第四阶段鬼婴、饿死鬼或赵开明附身鬼的关系仍无定论；本批只能确认被钉对象是造成阴霾的源头，不能据此完成身份归类。（第271章至第275章）
