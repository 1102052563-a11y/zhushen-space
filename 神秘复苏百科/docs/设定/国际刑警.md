# 国际刑警

## 概述

本书第21至25章所见“国际刑警”是一套处理超自然灵异事件的跨国体系。周正曾以国际刑警身份赴大昌市第七高中讲课并处理敲门鬼事件；其死亡后，亚洲分区的接线员刘小雨、队长赵建国接手调查。该体系可建立厉鬼档案、派人取证、提供保密网站和卫星定位手机，并尝试招揽驭鬼者承担地方治安。（第2章、第21章、第22章）

## 本批组织构成

- **接线员**：刘小雨原为周正专属接线员，负责通过卫星定位手机接收报告、记录信息和联系调查人员。赵建国说明，接线员在特殊事件中应安慰、容忍和体谅精神异常的幸存者。（第21章、第22章）
- **队长与亚洲分区**：赵建国是刘小雨称为队长的中年人员，代表亚洲分区与杨间交换情报、决定档案代号和人员安排。（第22章）
- **驭鬼者人员**：周正是负责大昌市治安的一线刑警，死后赵建国认为大昌市治安存在空缺，拟将当地新驭鬼者杨间纳入体系，先作为临时工亦可。（第9章、第22章）

## 档案与分级

赵建国根据杨间报告建立黑衫老人档案，代号定为“鬼敲门”，恐怖程度直接定义为A灾难级。第25章杨间从保密网站查到，厉鬼分级以对人类的危害程度划分，不能直接代表个体恐怖程度：C级为限制级、B级为危险级、A级为灾难级、S级为毁灭级。网站还存在全球厉鬼悬赏、事件请求及恐怖程度排行榜。（第22章、第25章）

## 设备与保密网站

周正使用的卫星定位手机被赵建国允许暂时由杨间保管。赵建国称设备特别打造、不易受厉鬼力量侵蚀，电量可维持一年，并能在严重事件中保持通话。杨间也获周正编号代码，用于登录各国政府联合搭建、高度保密的特殊网站，网站显示世界各地紧急事件、驭鬼者支援请求、悬赏与厉鬼资料。（第9章、第22章、第24章）

## 第298-300章补充

第298章，总部为杨间新增第二接线员秦媚柔，与原接线员刘小雨轮流提供服务，理由是其工作重要、单人无法保证二十四小时接听。第299章，刘小雨说明国际刑警的卫星定位手机之间可相互定位，原用途是方便联手处理灵异事件时支援；追踪断续信号需要同类设备。小春市失踪刑警许峰的手机因信号不稳定而无法由当地普通警力有效追查，杨间遂以自己的手机追踪。设备定位与卫星定位共同修正时，杨间称误差几乎不超过两米；这只是他当次使用中的说明，完整技术边界不详。（第298章至第300章）

## 第371-374章·通讯风险与中山市善后

赵建国为核实第三中医院中疑似童倩的目标，命联络员以电话联系医院；电话接通后无人回应，耳机传出的笑声却使联络员死亡，并在通讯室内扩散。总部随后切断通讯、撤离人员，但第374章曹延华确认已有三十二名接线员死亡。此事表明卫星定位与常规通讯可被中山市目标利用传播灵异，具体安全程序、死者名单和恢复措施均未写。（第371章、第374章）

冯全报告目标已被限制后，总部由曹延华安排沈良前往接收；曹延华同时追责赵建国忽略杨间预警，并称不再放心其继续负责杨间。杨间则为保留行动记录而在限制医院目标前主动恢复与刘小雨的通讯。谁最终接收尸袋、赵建国与杨间的联络安排是否变更，原文未写。（第372章至第374章）
