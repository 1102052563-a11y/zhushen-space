# 鬼童

## 概述

鬼童是杨间在第638章后对王珊珊寄养的第二阶段鬼婴所用的称呼。它原本从王珊珊腹内被剥离，杨间曾判断其由自身死而复生的诅咒延续而来，既有饿死鬼衍生鬼的特征，也与杨间鬼眼存在同源般联系；其具体来源和完整本质仍不详。第638章，杨间看见它已成长到约五六岁孩童大小，认为继续称为“鬼婴”不贴切，故改称“鬼童”。（第386章、第391章、第638章）

鬼童皮肤青黑、浑身阴冷，双眼泛红且没有瞳孔；外形虽穿衣服，仍给人被厉鬼盯住的强烈恐惧。王珊珊以新鲜血肉喂养它，试过牛肉、猪肉、鸡肉和羊肉，其中羊肉较为适合，但她认为那也只是替代品；鬼童真正所需食物未写明。它在没有杀人的情况下已停在第二阶段一段时间，杨间认为若再以鬼喂养、继续成长，可能失控，故尚未实施。（第637章、第638章）

## 已知命令与约束

杨间此前以染血旧报纸改写鬼童记忆，写入三项内容：完全听从杨间和王珊珊的命令；没有两人命令禁止杀人；禁止随意出现在人群中。第639章，鬼童在刘小雨看见它后立刻躲到杨间身后，表现与第三条相符；但它此前已悄然站到刘小雨背后、显出袭击意图，杨间据此指出第二条只禁止杀人、未禁止伤人，仍留有危险漏洞。（第391章、第392章、第639章）

杨间在鬼域内直接测试命令效果。“过来”的指令下达后，鬼童可立刻从远处消失并出现在他身边；在未接到新指令时，约十分钟后开始东张西望，约二十分钟后在其周围活动，约三十分钟后恢复自行玩耍般的游走状态。因此，当次观察中单一命令的约束不足三十分钟。杨间又命令它不准离开十米，它在范围内活动；随后改令它到约三十米外的路灯下，新命令直接覆盖旧命令，鬼童没有发生厉鬼规则冲突或死机。（第639章）

## 鬼域与行为表现

鬼童可自由进入杨间的鬼域。杨间试验发现，无论鬼域开启至第几层，它都能在其中来去，既不会掉队，也不受鬼域阻挡；这使杨间警惕，一旦鬼童失控，自己的鬼域不仅难以隔绝它，反可能成为其快速接近自己的路径。它能听懂“叫爸爸”这类语言指令并尝试开口，但嘴中漆黑、未见舌头、喉咙或牙齿，最终无法发声。杨间因此怀疑它身体部位尚不完整，可能缺少某种拼图，但这只是推测。（第639章）

它具有第二阶段鬼婴袭击活人的本能。王珊珊曾命令它赶走未获允许进入观江小区的驭鬼者，五天后它回来时嘴里带着数根活人手指；杨间推测有陌生驭鬼者夜间潜入小区、遭其咬伤后退走，未能确认对方身份、伤势和去向。刘小雨在八号别墅被它从背后靠近时，杨间也察觉袭击意图逐渐增强并及时喝止。鬼童会因“禁止随意出现在人群中”的旧报纸命令避开被活人看见的场面，却不能据此视为安全。（第637章、第639章）

## 杨间的处置计划

杨间因次日将面对八音盒诅咒的处置，担忧自己与王珊珊若一同死亡后鬼童失控，故第637章提出将其带走或暂时封入特制箱。王珊珊选择让他带走，认为鬼童可保护小区且可能帮到杨间；杨间随后将它带回大J市八号别墅。杨间承认鬼童危险，却把它视为提升生存机会的可控风险，暂不杀死或以厉鬼喂养。（第637章、第638章）

他考虑过用手中关押的鬼测试喂养效果，包括王小强死后遗留的鬼骷髅和朋友圈交战后得到的死人头；前者危险较低却可用于挽救濒死活人，后者则具有压制厉鬼、令活人短时死亡的表现但风险较高。杨间没有在本批实际投喂，鬼童吞鬼后会被侵蚀、驾驭其为拼图，还是发生其他变化，均不详。（第639章）
