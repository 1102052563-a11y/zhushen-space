# 背尸鬼

## 概述

背尸鬼是方世明驾驭、以一具冰冷沉重尸体压迫目标的厉鬼。第597章，方世明与杨间在平安大厦顶层对峙时，杨间忽觉背后一沉，一具死寂尸体已趴到他背上。三层鬼域不能隔绝这次袭击，尸体的重量压碎杨间胸骨、使内脏受创，并连同无头鬼影一并受到压制。鬼的本名、来历、是否曾由方世明以外的人驾驭及尸体真实身份均不详；“背尸鬼”为据其表现所作的条目称呼。（第597章）

## 袭击条件与效果

- **以身高锁定**：方世明说明，这只鬼会在一定范围内袭击身高最高的人。他在现场坐下，而范围内只余自己和杨间，于是杨间成为最高者并遭袭。范围大小、多人同高时的选择、是否能靠持续坐下规避及其他触发条件均不详。（第597章）
- **以重量压制人和鬼**：尸体趴上目标后可将普通人直接压死；对驭鬼者则能压制其体内的鬼。方世明称该压制没有数量限制，只要目标无法背动，驾驭再多厉鬼也会在一定程度上被削弱。该说法来自方世明当场说明，未在更多对象上验证，压制是否能彻底关押厉鬼不详。（第597章）
- **不受三层鬼域隔绝**：杨间身处三层鬼域仍被尸体直接压中，认为这类似于被鬼盯上后无论躲在哪里都无法避开。第598章开启六层鬼域后，尸体与其袭击一并停滞，杨间才以改变自身位置脱离；鬼域层数与避开此鬼的完整关系不详。（第597章、第598章）

## 使用记录

方世明用背尸鬼压制杨间后，杨间没有挣扎，而是开启第六层鬼域。约三十秒停滞后，杨间可控制鬼域变换位置，尸体没有继续跟随出现；他转而接近方世明，以鬼手折断其颈部。方世明之后以阴冷怪风带杨间坠楼，原文未再写背尸鬼是否仍在其背上、是否被鬼手压制、坠落后能否再次发动。方世明仍存活，故不能据这次反制判断背尸鬼已被关押或失效。（第598章、第599章）
