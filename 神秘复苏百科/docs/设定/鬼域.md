# 鬼域

## 概述

鬼域是第3章在大昌市第七高中出现的灵异范围。方镜首先指出学校中“出现鬼域”，周正与电话中的接线员据此判断事态升级。鬼域不是单纯停电或封闭区域：它带来浓重黑暗、环境异常老化、空间路线失真，并能使人难以离开原本熟悉的教学楼。根据本批内容，鬼域与黑衫老人直接同时出现，但其完整形成机制、范围边界与持续时间均不详。（第3章至第5章）

## 已见表现

- **黑暗侵蚀**：黑衫老人站在教室外时，走廊灯光仿佛熄灭，浓黑如水银般向教室内侵蚀。王老师开门进入后，门外黑暗又像黑雾一样渗入教室；第5章学生下楼时，楼梯灯熄灭，窗外也没有一丝光亮。（第3章、第5章）
- **环境老化与腐败**：走廊墙壁以肉眼可见速度老化，墙皮发黑、长出青苔；教室书本泛黄、发霉、腐烂，墙面、地面和座椅同样衰败，水泥风化后露出锈钢筋，局部发生坍塌。原文以“仿佛一瞬间经历几十年”描述这种变化。（第3章、第4章）
- **死亡与压迫**：鬼域内有学生突然呈现冰冷、僵直、死灰和腐臭的尸体状态。敲门声响起时，人会感到灵魂被掐住、无法呼吸般压抑；但本批原文未能确认所有死亡均由鬼域本身直接造成。（第3章、第4章）
- **空间迷失**：学生从五楼向下撤离，灯灭前已到三楼、正往二楼半走；黑暗后继续下行三至四层，似乎走进教学楼并不存在的地下区域。后来众人又回到二楼半厕所，继续下行也无法完成离校，呈现路线循环或错乱。（第5章）
- **光照受限**：学生手机仍可照明，但十几道灯光只能照出不足一米，之后仍被浓黑遮蔽，无法恢复正常视野。（第5章）

## 与敲门鬼的关系

周正报告时说黑衫老人已经形成鬼域，且鬼域范围可能很大；接线员因此认为事件不再是他能够处理的程度，并称能产生鬼域的鬼甚至可定义为灾难级别。第4章周正用腹中灰青怪手抓住老人后，教室黑暗暂时消失、灯光恢复，显示鬼域至少会受对老人限制的影响；但老人很快重新站起，第5章楼梯黑暗再现，表明这次压制未能终止鬼域。（第3章至第5章）

## 第6-10章补充

厕所作为鬼域中的区域，在第6章表现为门被关闭后即与外部隔绝：段鹏和郑飞沿墙寻找出口，面对的却是没有尽头的斑驳冰冷墙壁。杨间在其中行走二十分钟，未再碰到墙壁、马桶或正常出口，判断这里已不是通常意义上的厕所。（第6章、第7章）

第7章，杨间循滴水声前行，最终在第8章看见白骨树与被钉黑影，并在鬼眼进入手背后找到其实很近的厕所门。第9章他穿门回到五楼教室外，周正解释黑衫老人已离开，但鬼域不想让学生离开；第9至10章，张伟等普通学生仍只能在黑暗中反复绕路，杨间则能以鬼眼看破幻象带路。这显示同一鬼域对普通人的路径感知与对杨间的视野不同，但具体机制仍不详。（第7章至第10章）

第12章，幸存学生抵达操场后，手表和手机显示凌晨四点左右，与先前八点半的时间不同；校外无车辆灯光、无报警回应，天空也无月星。杨间据此判断鬼域范围不止教学楼，连操场乃至绿化树林也在其中，且可能随黑衫老人移动。第13章方镜困在操场外绿化树林，面积不大的数十棵树在黑暗中表现得如无尽森林，进一步印证鬼域对空间的影响。（第12章、第13章）

第16章，杨间带十三名学生到校门，马路两端仍被无尽黑暗封住，学校外也没有车辆或行人，说明跨出校门并不能脱离鬼域。第18至19章，黑衫老人带来的黑暗从前后包围校门，周围建筑腐朽、保安室和道路出现鬼物。杨间吞眼张开的红光鬼域约覆盖五米，能短暂驱退黑暗中的手掌和令老人后退；红光闪灭后他与七名学生到达步行街，证实这一次转移离开了学校鬼域。（第16章、第18章、第19章、第20章）

第27至28章，杨间在红纸压住一只鬼眼后以五眼发动自身红光鬼域。他先令阿飞等人所见街景皆被红光覆盖，逃跑许久仍回到公共厕所前；又可在域内从一端直接出现在另一端。抓人撞墙时，鬼域使通风口与人的头部出现可被强行重合的距离变化；他将范围收缩至自身后，身体可穿过居民楼墙壁。杨间认为域内景象兼具虚实、能隔绝外界，却只记录到五分钟左右的维持上限，不能据此认定全部鬼域皆有相同规则。（第27章、第28章）

第35章，杨间为赶赴王珊珊家，将五眼鬼域扩至整辆出租车。他在域内和司机换位，令车辆穿过前车、绿化带与房屋，以一分钟抵达小区；司机所见只是正常高速驾驶，说明杨间可用幻觉遮掩部分异常。杨间明确五分钟后鬼眼会再被刺激复苏一分，故把已开启的鬼域视作有限时机。（第35章）

第146至148章，杨间先以鬼域覆盖小强俱乐部会议厅，使雷虎等武装人员把其幻象当作真身，黄金制子弹和含黄金烟雾弹都未能找出其真实位置；鬼域还能消去虚假的会议桌、混淆枪声来源。王小强放出体内厉鬼后，杨间感到自身鬼域范围被压缩，具体是该鬼能力还是其他作用尚不能确认。为避开厉鬼，他将自己、王小强、雷虎及股东移到另一处红光笼罩的环境，却把厉鬼留在原会议厅；这次操作只证明鬼域可在当时转移其中的活人并形成分隔，不能推知所有厉鬼均会受同样限制。（第146章至第148章）

第163至165章，杨间在观江花园售楼部为说服钱总、秦总张开鬼域。两人先看到现实中不存在的满室血迹和血茶；杨间后来说明这是自己加入的幻觉。二人逃入电梯后，楼层显示连续降至不存在的地下十层，门外是黑暗和疑似鬼物的垫脚惨白双足，手机光只能照亮极小范围。他又明确二人未经允许无法离开会客范围。该批事实可证明鬼域能在本次对两人制造环境、视觉和路径异常，不能据此确认那双脚为真实厉鬼或其完整作用范围。（第163章至第165章）

第169至170章，敲门鬼的黑暗鬼域已覆盖大昌市中心部分道路。外部警戒逐步消失后，前方道路由正常光线没入深黑，杨间据鬼域仍在判断其中尚有幸存者；域内时间仍显示下午六点，夏日应有的日光却被完全遮蔽，城市人车活动也如骤然清空。江艳驾驶的车辆在域内无法脱离黑暗，车窗外的人影能持续跟随高速行驶的车辆，锁住的副驾驶门还会自行打开。杨间的鬼眼能在本次黑暗中显示清晰景物，但普通车灯仅能照亮数米。鬼域的确切边界、车辆异象是否由鬼域本身或其中鬼物造成，以及幸存者能否正常离开，均未写明。（第169章、第170章）

## 出处

第3章至第5章。
