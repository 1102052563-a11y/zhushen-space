# 鬼眼

## 概述

鬼眼是第8章由白骨树上巨大黑影凹口中脱出的红色眼珠。杨间以为红光是灯泡或玻璃珠而触碰，眼珠黏在手上并钻入手背，带来剧痛和一段异常视野。第9章起，他以其看破鬼域黑暗和迷路现象；第10章它又在后脑显露并阻止鬼婴袭击。杨间据周正的遭遇推测鬼眼也是会逐渐复苏、最终反噬自己的鬼，但本批不能确认其完整规则。（第8章至第10章）

## 来历与进入过程

- 杨间在厕所黑暗空间里循滴水声前行，见到一棵苍白如骨的树。树上挂有人皮、布条、干尸头颅和纸幡，树干上倒挂一个约四米高的黑影，其胸口被锈蚀棺材钉钉在树上。（第8章）
- 黑影没有面孔，眼睛位置只有凹口；黑血从被钉之处滴下，正是杨间一路听见的滴水声。红光来自该凹口中的“眼珠子”。（第8章）
- 杨间触摸红光后手掌剧痛，红色眼珠钻入手背。手背皮肉裂开时，眼珠可转动显出，并把一个不同的视角传入其脑海。（第8章）

## 已见能力

- **看见黑暗中的环境**：眼珠进入后，杨间周围黑暗退去，他能看清白骨树、黑影及无尽空间，并在回头时发现原本找不到的厕所门其实离自己很近。（第8章）
- **看破鬼域迷路**：第9章，杨间走出厕所门后看见正常走廊，并意识到不是鬼域消失，而是自己能看破鬼域。他据此找到在第二层来回绕路的张伟、苗小善等人，并带领他们下楼。（第9章、第10章）
- **后方视野与阻拦**：第10章鬼婴的手臂从黑暗袭向杨间时，其后脑皮肉裂开并显出第二只红眼。该眼发红光盯住鬼婴手掌，使攻击停止；显眼后，杨间也能接收身后的视野。（第10章）
- **多眼反击**：第11章鬼婴吞咬杨间手臂时，手臂皮肉裂开并出现三只猩红眼珠。手背一只、后脑一只、手臂三只共五只眼睛的红光使鬼婴吐出手臂并逃离。（第11章）
- **红光鬼域**：第19章杨间吞下一只手臂鬼眼后，体内出现第六只眼。红光从皮肤裂缝扩散约五米，驱散周围黑暗、迫使惨白手掌缩回，并让黑衫老人先退一步；红光闪灭时带走杨间与七名学生。（第19章）

## 代价与存疑

每次显现都伴随疼痛。第11章眼睛数量由两只增至五只；第19章吞眼后第六只眼出现并张开鬼域。杨间在发动时皮肤龟裂、衣物染血，且担心自己会像周正一样被体内鬼物占据。红光最终成功让八人离校，但老人进入范围后红光即闪灭，显示该力量尚不稳定。鬼眼与白骨树黑影的具体关系、为何能带人转移、是否有其他能力，均不详。（第8章至第19章）

第24章，杨间睡眠中经历意识清醒却无法行动的状态，感觉巨大鬼眼在天花板窥视，脸颊皮肉被撑开并显出红眼，体内还有东西游走、几乎撕裂皮肤。痛苦持续两三个小时后才消失。羊皮纸称这属于厉鬼复苏，杨间由此确认使用鬼眼后的反噬已经进入日常身体状态。（第24章）

第26章，杨间遭染血报纸贴脸时，为撕开报纸先后令脸部、额头与脖颈出现红眼；原有和新增显眼共提供红光，使报纸出现裂口并被扯碎。第27章他发现红纸碎片可压住手背一眼，五眼仍能张开鬼域，且身体副作用较六眼状态轻。第28章，他以五眼鬼域测试出可令范围内的人循环返回、让自身在域内换位、扭曲距离和穿墙；连续维持约五分钟后，体内鬼眼再次躁动。以上皆为杨间当次使用经验，完整规则仍不详。（第26章至第28章）

第39章，杨间以手掌显出的鬼眼向王珊珊手腕注入红光，令鬼婴留下的青黑印记消退，并将部分鬼眼力量留在她体内，形成红色鬼眼烙印。杨间认为这使王珊珊成为自己仍有理智的活人鬼奴；具体能力、寿命和副作用均未能确认。（第39章）

第74章，杨间因在郝少文住所张开鬼域，红纸压制彻底失效，胸口再裂出第七只鬼眼，超过他此前以六眼为限的判断。他估计鬼眼复苏可能在一个月内失控；这也促使他重视第二只鬼相互制衡的情报。（第74章）

第246章，杨间为对抗医院鬼群解除五眼对无头鬼影的压制，使鬼眼从七只连续显出至八只；他再咬下一眼吞入，刺激出第九只。红光鬼域在黑青阴霾压制下仍扩至约一千米，隔绝第三阶段鬼婴，但第十只眼若出现即可能完全复苏。第248章，他趁鬼烛余火将五只鬼眼重新转入鬼影，以此压制鬼影并分担复苏，身体仅余四眼。第九眼后的极限、四眼可使用次数与下一次复苏后果均不详。（第246章、第248章）

第273章，杨间以死机无头鬼影覆盖全身，令鬼眼在极限使用下不复苏。他先把两只移到掌心并隔手背重叠，让第二只在第一层鬼域内张开；红光遂从通讯大楼穿入第七中。再将一眼移至额头，与掌中两眼排成直线后，第三层鬼域使光束彻底撕开校内阴霾并搜索目标。杨间据此认为一只鬼眼对应一层鬼域，但十层是否能叠加、叠加后的能力及脱离鬼影压制的复苏代价，均未由本批验证。（第273章）

第477章，杨间凭鬼手、无头鬼影与三鬼棺材钉形成的暂时平衡，连续将鬼眼鬼域叠开至第四层：第一层可规避普通人感知，第二层令域内景象虚幻，第三层令现实景物消失，第四层红光能强行驱散部分鬼差黑暗，却仍无法深入源头。他准备开启第五层以反入侵鬼差鬼域；是否成功、第五层具体表现与复苏代价均未写明。（第477章）

第479至481章，第五层鬼眼鬼域得到本次现场表现：杨间以红光裂口切开鬼差黑暗，令带鬼眼的鬼手越过边界抓走多名被困者及鬼棺；他称自己仅以一只手侵入，未完全进入鬼域。随后，他先用鬼域塑造山谷并把现实泥土填入其中，再将红光汇集至额头鬼眼，使一棵树及其后泥土消失，留下一道远延土沟。王小明据此提出或可逆转消失、接近重启的理论，杨间否认目前能做到；第五层的发动条件、可影响范围、消失物去向、能否恢复及复苏代价均不详。（第479章至第481章）

## 出处

第8章至第10章。
