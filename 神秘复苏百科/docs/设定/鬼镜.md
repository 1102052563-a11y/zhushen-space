# 鬼镜

## 概述

鬼镜是杨间为观江花园荒宅木门房内一面异常更衣镜所作的称呼。镜子铜制支架已有锈迹，镜面却在至少百年后仍光亮如新；其原本被油布遮住，后被人挪动。杨间进入房间时，镜中没有映出他的身影，因而将其视为与工地失踪、张伟身边工服鬼相关的灵异物。（第189章）

## 所在与环境

鬼镜放在荒宅二楼第三间房的深处。外层木门曾有七八道锁，已被工具剪断；房内从青石通道通向近乎密封的小空间，四壁均为青砖，没有窗户与天花板。镜旁留有旧油布，地上则有新近出现的工地安全帽、钳子、锤子、电钻和被移动的痕迹。杨间依据墙上钻孔、对面在建楼房和工具，推测工人曾为寻找古宅财物进入，搬开镜子并试图钻入相邻铜门房。（第188章、第189章）

## 已知表现

- **无倒影**：杨间靠近时，镜面不能映出他的人影。此现象与先前镜中鬼需借镜面显现的情况有关联，但两者是否为同一厉鬼或同一规则，原文未直接确认。（第189章）
- **可伸入的镜面**：杨间以点燃的鬼烛防护后伸手触镜，手掌穿入镜内，感到黑暗、冰冷，如同置入冰窖。（第189章）
- **镜内拉扯**：他伸入的手臂随即被多只手掌抓住并往镜内拉。鬼烛或其他未明原因使他成功抽回，手臂却留下小孩、老人、女人等不同大小的青黑掌印；杨间据此判断镜内存在多只鬼。（第189章）
- **借反光媒介转移**：第191至192章，杨间推断鬼镜原先通过木门房墙孔、远处大楼玻璃照到张伟，才使镜内与张伟相同的鬼离开。张伟在鬼镜前干扰同步后离去，镜中鬼仍留在镜内；杨间据此认为镜中鬼须经另一面镜子或反光物转移。这只是针对张伟同形鬼的当次推断，不能确认所有镜内鬼均遵循同一条件。（第191章、第192章）
- **某种意义的复活**：人皮纸称鬼镜可令死人“复活”，随后限定并非真正复活，代价是释放镜内一只鬼。信息由人皮纸在受威胁后给出，杨间认为既可能有用也可能是引诱放鬼的陷阱，具体复活对象、结果和操作条件都不详。（第194章）

## 风险与推测

杨间认为若镜内的鬼全部走出，可能令大昌市陷入极大危险，故没有继续试验或移动镜子。他又认为原主人既把镜子置于明显的老宅内、又以油布遮盖，并非单纯要永久隐藏或毁灭世界，而可能想在未来留给后人利用；这只是他根据建筑、摆放和三道特殊门作出的推测。第194章的人皮纸把镜子描述为可用一只镜内鬼交换“某种意义复活”的工具，进一步提高了其被人主动使用的风险。黄金门、铜门和鬼镜间的确切关系、镜内鬼如何出入、能否与张伟身边的镜中鬼画等号，均不详。（第189章、第194章）

## 当前状态

第192章，杨间以老油布遮住鬼镜后将其带出荒宅，以免好奇者触发灵异事件；他要求张显贵加固古宅、避免另两间房暴露。三日后，鬼镜被安置在杨间别墅五楼，已换稳固支架、罩厚黑布，房内墙窗均涂黑以避免产生反光媒介。镜中张伟同形鬼仍留在其中，但人鬼联系未获确认已完全斩断。（第192章、第193章）

第196章，张伟遭行动组刺杀后，镜内出现一个有体温、心跳的张伟，被多只手掌向内拖拽；杨间以无头鬼影协助将其拉出，镜面随即恢复光洁。人皮纸称鬼镜复活的是张伟首次照镜时被记录的版本，记忆停在当时；代价是放出镜内一只鬼。该叙述未经其他证据验证，且未交代放出的鬼是什么、去了哪里。杨间仍命张伟避开五楼和镜子，未再次尝试复活。（第196章）

第201章，王小强在观江花园遇见一名穿破连衣裙、约七八岁、无眼无口鼻的鬼女孩；其皮肤青黑、带尸臭，与人皮纸所称复活代价放出的镜内鬼相符。鬼女孩在听到货车鸣笛后高速沿路往南离开，王小强据此推测其对声音敏感；这只鬼是否确由鬼镜放出、完整杀人规律及去向，原文均未确认。（第201章）

第258至260章，杨间将鬼镜移至张伟市中心别墅空房，避开手机镜头与镜面相照，令王小明只能远程看到镜子支架和黑布。他先以鬼绳吊死自己，又让无头鬼影侵入尸体；揭开黑布后，镜内逐渐出现与杨间完全相同的黑西装人影，并在镜外尸体上吊时同步模仿。约四小时后，鬼影压制鬼绳，尸体摆动至双脚接触镜面，随即被镜子吸附。王小明据这些过程推测杨间意在让鬼影与鬼镜发生逻辑冲突、令其死机后借镜子复活；手机在尸体被镜面吸住时断电，鬼镜是否完成复活、是否释放新的镜内鬼以及杨间去向均未揭示。（第258章至第260章）

第261至263章，镜内无数不同腐坏形态的鬼手与变作杨间的镜中鬼试图把尸体拖入镜内，无头鬼影则从外覆盖镜面、再退入尸体，双方以尸体为媒介拉扯。第七日鬼影以鬼眼红光暂退镜内鬼，却与鬼镜先后沉寂；镜面出现细小裂纹，头部因鬼影无法控制而被吸入镜内。杨间的头在镜内复活并避开焦黑鬼手，随后从镜中拉出，但身体仍在镜外，形成不完整的反向复活。镜面此后不再映出杨间倒影；他以黑布重新盖住鬼镜并将其带回观江花园。镜中裂纹、焦黑断手和其余鬼物是否会脱困不详。（第261章至第263章）

第277章，杨间在观江花园五楼再次检查被黑布覆盖的鬼镜，确认镜面顶端仍留有此前鬼影与鬼镜对抗造成的细小裂纹，底部则有一条镜中厉鬼留下的冰冷焦黑手臂；两者均未在本章表现新异动。该手臂的完整身份、裂纹是否会扩大及镜内其余鬼物状态仍不详。（第277章）

## 原文回跳第779章·改作团队复活资源与显鬼媒介

杨间把鬼镜移至安全屋，向冯全、童倩、李阳、黄子雅和张韩公开其用途，并交由童倩管理。照镜者若在镜中留下影子，死后可由镜中复活；但每次复活会放出一只镜内鬼，复活者记忆和状态停在留影时，且驭鬼者复活后能否保有体内鬼尚无先例。（原文回跳第779章）

黄子雅和张韩先后在镜前留影。留影过程中，镜中会出现由远及近、逐渐模仿照镜者的僵硬鬼影；若镜内外形象完全重叠，活人将被拉入镜中并由镜内鬼替代。杨间及时用黑布遮住镜面中止黄子雅的同步变化，证明该禁忌真实存在。（原文回跳第779章）

本次还发现鬼镜可呈现肉眼不可见的灵异目标：镜面朝向杨间木棺时，映出一条沉睡的黑毛狼狗。杨间认为该异象与棺内鬼梦孕育有关，但鬼镜是“看见”还是“映照”此目标、黑毛狗是否实际存在均未确认。（原文回跳第779章）
