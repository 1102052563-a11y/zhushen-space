# 鬼奴

## 概述

鬼奴是周正于第9章对杨间说明的灵异存在：人在鬼域内被厉鬼杀死后，死者会形成鬼奴，受厉鬼“鬼主”无条件支配，如同古代地主家的奴仆，因此得名。周正强调鬼奴不如厉鬼本体恐怖，但对普通人仍有极大威胁。第10章楼梯间出现的赵强等僵硬尸体，与鬼奴规则可能有关，但本批未由周正或其他人明确确认其身份。（第9章、第10章）

## 形成与归属

- 周正说鬼奴形成于鬼域中：被厉鬼杀死的人死后成为这类东西。该解释对应学校鬼域里已出现的多名死者，但不代表本批每名死者都已经被证实为鬼奴。（第9章）
- 鬼奴受制于“鬼主”，会无条件执行厉鬼的吩咐。周正以古代地主和奴仆作比喻，说明鬼奴不是独立行动的普通亡者。（第9章）
- 周正认为一个鬼域中只会有一只厉鬼，并称黑衫老人是该鬼域中的厉鬼；他据此判断杨间在厕所所遇者应为鬼奴，而非另一只与鬼域并列的厉鬼。（第9章）

## 本批相关记录

厕所内出现的惨白手掌会抓住人并拖入黑暗。它先抓杨间肩膀，杨间播放敲门音频后手掌退回；第7章它又先后拖走段鹏、抓住郑飞颈部。第9章周正将杨间所遇对象判断为鬼奴，但没有单独说明其外观、姓名或由哪位死者转化。（第6章、第7章、第9章）

第10章，赵强等此前跟随方镜的学生僵硬站在一楼楼梯口，赵强的头能不动身体而转一百八十度，眼睛死灰，随后又恢复背对姿态。杨间将赵强称为尸体，且怀疑方镜可能丢下这些人拖住厉鬼；然而原文没有直接称他们是鬼奴，也没有写出他们受谁指挥，故只能保留为可能相关的现象。（第10章）
