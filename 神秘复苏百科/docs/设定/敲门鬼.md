# 敲门鬼

## 概述

“敲门鬼”是第4章标题对黑衫老人的称呼。本批原文所见个体是一名穿黑色长衫、身形干瘦、皮肤或脸上有尸斑的老人；其眼睛死灰、空洞，行动僵直而迟缓，却能在医院、论坛发帖者家中和大昌市第七高中出现。它无法被普通方式杀死，具有鬼域，并以固定节奏敲门。其真正来历、完整规则与最终目标在第5章前均不详。（第1章至第5章）

## 外观与活动记录

- 某省第三中医院的帖子称，它原是从五楼坠下的老人尸体。发帖医生认为尸体已死亡不止一天，之后尸体却从停尸房失踪，并被拍到穿黑色连体长衫、自行经过急诊室。（第1章）
- 照片中的老人浑身干瘦、皮肤褐色有斑点，一只眼睛死灰空洞；杨间在学校窗外所见老人同样穿黑长衫、脸有尸斑，确认与照片为同一形象。（第1章、第3章）
- 它在论坛发帖者家门外敲门，随后又在发帖者房门外敲门；学校中则站在教室门外敲门，并未立即推开没有上锁的门。（第1章、第4章）

## 已知特性

- **疑似与敲门有关的规则**：其敲门声为“一长、两短”的固定频率。杨间结合论坛帖与现场，推测敲门方式可能是杀人方式、计时条件或行动条件；该说法在本批仅为推测，未获完整验证。周正依据此推测强行制止其敲门，曾让教室黑暗暂退。（第3章、第4章）
- **鬼域能力**：它在学校周围形成黑暗鬼域，伴随环境迅速腐败、多人死亡及空间迷失。接线员认为能产生鬼域的鬼可能被定义为灾难级别；周正则在事发初期称其至少为危险级别，后认为其绝不只是危险级别。（第3章、第4章）
- **难以杀死**：周正明确说鬼无法被杀死，即使碾成肉泥、烧成灰也会以匪夷所思的方式出现。第4章他撞倒老人并用腹中鬼手抓住后，老人仍能缓缓站起，现场印证普通压制不能消灭它。（第4章）
- **受鬼手短暂限制**：周正以腹中伸出的灰青怪手抓住老人后，笼罩教室的黑暗消失、灯光重现，说明鬼手有短暂限制效果；老人重新起身后，周正立刻让学生逃走。（第4章）

## 已知影响

第3章至第4章，教室中已有多名学生以死灰、腐臭尸体状态死亡，王老师开门进入黑暗后失踪。第5章学生撤离时，鬼域又在楼梯出现并令路径错乱；同时厕所中还有另一只鬼出现，但本批原文不能确认这只鬼是否与敲门鬼为同一个体或由其直接造成。（第3章至第5章）

## 第6-10章补充

第9章，周正明确说一个鬼域中只会有一只厉鬼，并指认黑衫老人是学校鬼域的厉鬼；他还称老人此前已经离开。因此，他把杨间在厕所所遇的惨白手掌判作鬼奴，而不是鬼域中的第二只厉鬼。该解释也使第10章赵强等僵硬尸体是否与鬼奴有关成为待确认问题。（第9章、第10章）

本批没有让黑衫老人再次直接出现，也未新增其敲门或本体攻击记录。敲门音频仍能让厕所鬼手暂时收缩，但这只证明该声音对鬼奴有效，不能据此确定它对已离开的黑衫老人本体仍有何种限制。（第6章、第9章）

## 第11-15章补充

第15章，杨间让方镜先保存论坛中老人的照片，观察数分钟并未见老人出现，暂时排除“看照片”是老人出现的触发条件。接着方镜按杨间要求播放帖子中的敲门音频，黑衫老人随即出现在绿化带小路、朝方镜步行。杨间据此推测，自己先前听过敲门声，可能才是老人进入学校并锁定自己的原因；这是基于本次试验作出的推测。（第15章）

方镜重复播放音频时并不知道老人已在身后。杨间离开绿化树林后告知其真相，方镜转身撞到老人，老人以冰冷枯瘦手掌覆盖其脸部，造成面骨被碾碎般的痛感。第15章只写至方镜在电话中尖叫，尚未写明敲门鬼是否杀死他或之后如何行动。（第15章）

## 第16-20章补充

杨间拨打雷电法王电话时，手机铃声在大昌市第七高中内回响，接通后教学楼三楼出现手机光与人影；该号码随后反拨，并显示在多名学生手机上。钱万豪接听显示为王老师的号码，听筒传出敲门声，杨间判断这会再次引来老人。第18章黑衫老人携黑暗抵达校门，证明电话与敲门声能跨距离把其引向听者所在处。（第16章至第18章）

第19章老人直接袭击钱万豪，并踏入杨间约五米的红光鬼域。老人先退一步，随后又向前踏入；红光随即闪灭，杨间等八人脱离学校。该过程只能证明红光短暂影响老人行动，不能证明已经限制或伤害其本体。（第19章）

## 出处

第1章至第5章。
