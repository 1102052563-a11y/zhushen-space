# 中山市人头气球事件

## 概述

中山市人头气球事件是总部在第330章提供给杨间的一份灵异事件档案中所记载的事件名称。档案照片显示，城市上空漂浮着大量静止的黑色物体；近距离照片确认它们是一颗颗脸色苍白的人头，如同被放飞的气球，因此得名。每天夜里，城内都会有人诡异死亡，死者头颅会升至城市上空，且数量每天增加。事件的灵异原因、杀人方式和人头为何漂浮，档案均未查明。（第330章）

## 等级与风险

总部起初依已知伤亡把事件定义为B级：恐怖程度暂按B级判断，但死亡在持续增加，且无法迅速解决。中山市国际刑警童倩先行处理后失联，最后通话显示她疑似被困在某处无法离开；其后两名支援刑警和三名外援也在同日失去联络。由于伤亡扩大、童倩失联和处置停滞，总部将档案升为A级，要求继续派人支援。（第329章、第330章）

## 与记忆篡改鬼的重合

杨间由鬼橱得到赵磊位于“中山市凯撒大酒店13号房”的信息，又从赵磊遗留笔记拼出同一酒店“31号房”的信息。他认为控制赵磊、能够篡改记忆的鬼已前往中山市，并警告其他驭鬼者一旦被篡改记忆，可能在不自知时成为鬼的工具。事件与该鬼是否同源、酒店两条房号中何者真实、童倩及失联人员是否已遭篡改，均不详。（第328章至第330章）

## 已知支援与处置

第330章时，已知进入中山市者包括童倩、冯全、郭凡及灵异论坛外援罗素一、黄飞、林落梅；杨间决定次日与熊文文同行，赵建国承诺安排专机，并允许调用鬼烛。杨间拒绝增加更多队友，理由是记忆篡改风险下人数越多，身份越难辨认。其出发、现场规则、死者数量及支援结果截至本批末均未写出。（第329章、第330章）

## 出处

第329章、第330章。

## 第331章·抵达后的目击与追查

杨间乘直升机进入中山市上空时，机组险些撞上一颗漂浮人头。他近看确认人头脸色惨白、闭眼不动，未立即发现其他异常；下降途中目测城市上空至少有一两百颗人头。童倩的最后录音则说明，她通过全市录像发现最初数颗人头浮于城南上空，沿线索来到凯撒大酒店前，称“这里有问题”后失联。人头是否直接出自酒店、何时杀人、为何暂未攻击直升机及人头身份截至本批均不详。（第331章）

## 第346-350章·处置风险扩大

郭凡、冯全在凯撒大酒店失联后，原先驻中山市的灵异论坛外援罗素一、黄飞、林落梅认为事件已非普通B级处置，曾准备连夜离开；赵建国确认征调，杨间遂带林落梅、罗素一筹备第二次酒店行动。第348章，酒店又出现持续扩大的鬼雾，张高称若无有效遏制，约二十四小时可能越过第二、三警戒线，周边街道和小区已开始扩大封锁、撤离。杨间认为人头气球本身与篡改记忆鬼、酒店内其他厉鬼及疑似冯全复苏的鬼雾交织，风险已不能仅以原档案单独评估；这些事件是否同源、鬼雾会否造成新的伤亡和最终处置结果均不详。（第346章至第350章）
