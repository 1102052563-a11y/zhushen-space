# 鬼棺

## 概述

鬼棺是黄岗村村口灵堂内的朱漆棺材。第91章，杨间从棺盖缝隙见到棺内有一双发黑的手；第103至105章，冯全从棺中坐起，并称外面的厉鬼不能回到棺中，否则会造成极严重后果。第106至110章，杨间与冯全围绕其作用冲突：冯全把它视为可令自己成为真正厉鬼的机会，杨间则依据亲身进入后的感受，提出它实际孕育黄岗村之鬼的判断。该判断来自本批观察，尚未获独立证实。（第91章、第103章至第110章）

## 入棺表现

第109章，杨间为躲避灵堂浓雾和遗像男子，躺入棺中并盖严棺盖。开始时棺内只有黑暗和压迫感，随后他的身体无痛地逐渐消失，意识仍留存且能越过浓雾看见灵堂对抗，感知范围还延伸至村内；离棺时，身体又在黑暗中重新凝聚。他因此只确认入棺者会经历暂时失去实体、意识仍存在的状态，不能据此断定具体原理或永久效果。（第109章）

## 与黄岗村的关系

杨间根据入棺体验、村鬼被肢解后脚步仍重现，以及冯全长期居棺能影响部分村民等现象，推测鬼棺不是单纯关押容器，而是为黄岗村之鬼准备的孕育之物；村鬼离棺、冯全占棺，才形成双方互相牵制的僵局。他还推测遗像男子只是该鬼的形象，整个黄岗村及其中村民可能都是该鬼的一部分。冯全未接受这一判断，故截至第110章，这些关系仍为杨间的推断。（第109章、第110章）

## 当前争议

冯全认为回到鬼棺可成为有意识的鬼，从而无惧厉鬼复苏、驾驭更多厉鬼；杨间指出冯全若占棺，村鬼便会持续守在外面，外来救援者也可能反复遇害。杨间主张将村鬼送回鬼棺以令黄岗村事件结束，并为阻止冯全回棺，先以无头鬼影摘下其头颅。能否让村鬼回棺、回棺后的结果及鬼棺是否会继续孕育危险，至本批末均未揭晓。（第110章）

## 出处

第91章、第103章至第110章。
