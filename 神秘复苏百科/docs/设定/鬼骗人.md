# 鬼骗人

## 概述

鬼骗人是驭鬼者高志强的代号及其所驾驭能力的称呼。第487章，刘小雨在十二号休息室认出高志强档案后，称其因驾驭的鬼极其特殊而被特招到大京市。高志强本人以“我连鬼都能骗”概括该能力，但原文未披露厉鬼形态、驾驭过程、是否只有一只鬼或其受总部特招的具体理由。（第487章）

## 认知与行为影响

高志强先用能力影响一名女接线员，使她双目茫然、表情麻木，站在床边自行脱衣；该接线员仍能行走和听从指令，却没有表现出自主判断。刘小雨发现异常后试图以总部手机求救，高志强夺走手机并抓住她，随后她仅迈出一步便停住，神采消失、神情转为麻木，按命令关门、走到床上躺下。原文借两人的现场表现确认这种控制至少可作用于普通人。（第487章）

高志强称，受影响者会在当下以为自己心甘情愿，甚至主动行事，十天后才会察觉异常，同时仍记得期间发生的事情。他以此计划在会议结束后离开大京市，规避追究。这是施术者为威胁刘小雨而作的说法，十天是否准确、受害者能否提前恢复、是否可被外部人员解除以及能力是否必须近距离发动，均未获得验证。（第487章）

## 已见限制与风险

杨间通过鬼眼察觉休息区有人动用厉鬼力量，随即以鬼域进入房间；这表明鬼骗人当次发动并未完全遮蔽鬼眼的异常感知，但并不能推出所有探查能力均可发现它。杨间没有尝试直接解除控制，而是在高志强身上施暴并以其可能污染接线员可信度为由追杀；刘小雨和另一名接线员是否在本批内恢复自我意识未写。（第487章、第488章）

高志强在第488章遭重创后数十秒内恢复，表现近似厉鬼重启。该复原是否属于鬼骗人的一部分没有得到确认，故不能将“控制他人”和“重置伤势”视为同一条已证实规则。其完整规则、反制方法、复苏代价与能否影响厉鬼，均不详。（第488章）

## 第489-493章·隐藏、取代与现实影响

第489章，高志强以鬼骗人制造一个正在逃跑的假身，令在场驭鬼者及杨间的普通视线都确认假身存在，本体则留在原地不足三米处。杨间额头的一只鬼眼未见假身，叠至两层鬼域后才找到本体。高志强称该鬼不只可骗活人，也可骗自己体内的鬼，使其相信不会复苏；他此前重伤恢复，是让身体相信没有受伤的结果。该说法由高志强提出，具体发动条件和可欺骗对象范围未被独立验证。（第489章）

杨间以无头鬼影压制高志强后，高志强让鬼相信自己是高志强、又让自身意识相信自己是鬼，主动让厉鬼取代人身。第490章，他以无实体、鬼眼两层鬼域也看不见的状态发声，并令杨间身体和意识逐步相信自己不存在；杨间推断留下的是拥有高志强思维的鬼，而高志强本人已死。该鬼能否永远以反复欺骗维持人类思维，高志强虽作肯定回答，但没有得到验证。（第490章）

第491至493章，杨间以鬼手限制取代者，再以无头鬼影强行覆盖，借其能力恢复自己被影响的身体并让人体部分恢复血色、健康和健硕状态。杨间随后发现该鬼会形成独立意识、排斥其体内的鬼眼、鬼影和鬼手，不能长期加入现有平衡；他把它装入黄金箱并存至总部地下深处。此页所称鬼骗人现以高志强外形存在，但是否具有固定实体、能否被普通人驾驭及正式收容结果均不详。（第491章至第493章）

## 第502-503章·批准归杨间暂持

第502章，杨间以高志强遗留的鬼作为对总部表彰的要求；曹延华考虑到事件已告一段落、杨间有功且总部即使拒绝也未必找得到其先前藏匿的黄金箱，批准由杨间带走。第503章，杨间离开总部时取走装有该鬼的箱子，并在平安酒店将其记录为接触过的灵异物品之一。此处只确认总部默许其由杨间带离，并不表示杨间已经成功驾驭、稳定关押或掌握完整规则。（第502章、第503章）
