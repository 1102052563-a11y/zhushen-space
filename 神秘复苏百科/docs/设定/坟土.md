# 坟土

## 概述

坟土是冯全体内第二只鬼相关的暗褐色泥土。第366章，冯全被柴刀鬼腰斩后，断口没有血液和内脏，而被坟土堵住；他称坟土能压制鬼雾复苏，使雾中鬼无法凝形，但自身会遭它逐渐侵蚀。其来源、完整驾驭过程及与既有坟类灵异的关系均不详。（第366章）

## 侵蚀与坟丘

冯全一只手已近乎变为暗褐色泥土，他称坟土会不断增多，最终形成坟丘将自己埋葬。第366章，他在无头鬼影侵入时引发坟土复苏，泥土先聚成双腿，继而黏住杨间和他本人，仅约五秒便形成坟丘；冯全的头露在丘顶却仍持续下沉。该坟丘能困住目标，杨间以鬼绳和自身力量无法拉出冯全，判断若自己被埋也难以脱身；但坟土对其他厉鬼的完整压制范围不详。（第366章、第367章）

## 本批处置

杨间点燃仅余的一小节红色鬼烛，得到不足两秒的压制窗口，以鬼绳将冯全拉出坟丘。坟丘随即坍成带尸臭的泥土，内部留有一具由泥土堆成、五官可辨的人形轮廓；冯全身体仍继续冒出坟土，杨间只能以无头鬼影暂压复苏。第369章，残存坟土又束缚带着冯全下半身的雾鬼；其能限制雾鬼多久，杨间仅按泥土会在一两日内脱落作出现场判断。（第367章、第369章）

## 分割后的平衡

冯全接回下半身后，完全复苏的坟土压过尚属半复苏状态的鬼雾，险些再次形成坟丘。杨间以白色鬼烛将部分坟土引到烛旁，使冯全抓住两鬼力量短暂相平的时机；熄烛后，他用无头鬼影阻断这部分坟土回流，将留在外面的、渗出鲜血的小土丘装入黄金制装尸袋。冯全称体内剩余两鬼已恢复此前平衡。被分离坟土是否仍会复苏、冯全新平衡的期限及装尸袋外的处置，均不详。（第370章）
