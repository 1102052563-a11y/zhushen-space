# 鬼雾

## 概述

鬼雾是冯全驾驭的两只鬼之一所形成的灰色浓雾。第369章，冯全说明雾中存在由浓雾凝成的人形鬼，鬼会依雾内动静锁定目标；他此前失去下半身后，雾鬼携带那半截身体脱离控制，在凯撒大酒店周围形成鬼雾。它是否与先前酒店外全部浓雾完全等同、除冯全所述外还有何种规律，均不详。（第369章）

## 已见规律与危险

人在雾中乱跑、乱喊或以动作搅动浓雾，大概率会被雾鬼循动静找来；站在原地不动可降低遇袭可能，却不能保证安全，因为雾鬼仍会在雾中移动，碰上同样会杀人。雾鬼靠近时，呼吸会带来寒意，雾气会像诅咒般侵蚀人体；普通人与其接触后身体会迅速被同化成浓雾，只剩衣物。冯全称每多一名死者，鬼雾会更浓、覆盖范围会扩大，因此不能放任其继续成长；该增长的速度、上限和是否有其他触发条件不详。（第369章）

## 形态与本批限制

杨间以鬼眼看见雾鬼上半身由较周围更浓的雾组成，具四肢五官却极不稳定；其下半身仍是冯全的身体。冯全下半身残存的坟土使雾鬼行走时不断掉落泥土，暂时束缚它；杨间判断一两日后坟土掉尽，雾鬼即可脱离身体、重新融入鬼雾，届时覆盖范围可能扩大。杨间主动在雾中行走将其引近，再以无头鬼影抓住双腿，雾鬼上半身和周围浓雾随即溃散；这说明本次可被鬼影限制，但不能据此推定所有状态下均可如此处置。（第369章）

## 与冯全的新平衡

杨间将被限制的雾鬼及冯全下半身接回其上半身后，撤去鬼影。最初坟土力量占上风，雾鬼被坟土压制而消失，冯全濒临被埋；杨间点燃白色鬼烛引走部分坟土，令两鬼短暂达到平衡。熄烛后，他以鬼影截断被引走坟土的回流，只让剩余坟土回到冯全体内，冯全称已恢复此前状态。这个平衡能维持多久、雾鬼能力是否完整恢复，原文未写。（第370章）
