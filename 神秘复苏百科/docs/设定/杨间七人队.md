# 杨间七人队

## 概述

杨间七人队是杨间在大昌市尚通大厦组织的驭鬼者小队。总部仅允许杨间保留七个组队名额；名额可使成员不被总部直接调离，但不等于成员脱离自身负责城市的职责。杨间将小队定位为共同处理灵异事件、抱团生存并保障成员家属的长期组织，而非临时行动组合。（原文回跳第776章）

## 初始成员与空位

小队初步成员为杨间、冯全、童倩、李阳、黄子雅、张韩，共六人，尚余一个名额。杨间原考虑状态不稳的张雷，也考虑若纸人转移成功便吸纳熊文文；截至本批，张雷未加入，熊文文虽已苏醒，其加入仍以杨间后续安排为准。（原文回跳第776章、原文回跳第778章、原文回跳第779章）

成员均为驭鬼者，但能力、复苏风险与职责不同。杨间强调，人员稀少不代表可随意补充，驭鬼者必须具备存活能力、理智和协作素质；普通人侥幸成为驭鬼者后快速死亡或失控的情况并不少见。（原文回跳第776章）

## 继任与保障

杨间担任第一任队长，并预先规定自己死亡后由冯全继任、童倩列为第三任，之后依存活顺序继续负责队伍。该顺位是本批会议的初步共识，后续是否调整未写。（原文回跳第776章）

为保障成员和家属，他承诺将观江小区别墅、安全屋使用权提供给成员居住，成员亲友可迁入普通商品房；又计划向每名成员转让尚通大厦公司一成股权，作为可由父母、后代继承的分红权益。股份不能转卖，具体法律手续、最终估值与亲属安置范围待后文。（原文回跳第776章）

## 行动原则

小队建立目的包括共同抵御灵异事件、避免成员死亡后家属失去保障，以及在队长更替时维持组织延续。杨间认为利益绑定和规则都不可缺少，单靠临时聚集无法形成稳定团队。（原文回跳第776章）

鬼邮局调查由杨间、李阳和孙瑞已先行展开，暂未要求小队全员参与；冯全则准备追查裴东所涉摆钟诅咒，并被建议视情况与童倩同行。各成员最终固定分工、总部是否认可全部安排及队伍实战协作效果均不详。（原文回跳第777章）
