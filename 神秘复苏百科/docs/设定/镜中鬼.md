# 镜中鬼

## 概述

“镜中鬼”是为便于指代第136至140章中出现的未知厉鬼所用的暂称，原文未给出正式代号、来源或关押记录。它最初混入杨间等人的同学聚会，能在现实中借用活人的外貌、衣着与身份显现；第136章，它伪装为苗小善的表哥上官云，连面部既有伤痕也一并保留，但衬衫口袋、皮带英文左右颠倒，且中弹不死，因此被杨间识破。伪装被赵磊叫破后，它舍弃“上官云”身份消失，只留下两颗黄金子弹。（第136章）

## 已知表现

- **镜内真身与现实影子**：杨间在第139章推断，鬼的本体存在于镜中厕所，现实中可见者更像无形影子；此前服务员能看见混入包厢的第七人，说明其并非完全不能影响现实，但现阶段的显现依赖身份。（第136章、第139章）
- **夺取并替代身份**：真正的上官云被困镜内后，鬼以其形貌回到包厢。杨间据此判断，它需要活人身份才能在现实中出现；识破身份会使其失去该伪装并不可见。（第136章、第137章）
- **镜面囚禁**：上官云在镜中没有倒影，且镜像厕所之外只有黑暗；现实厕所有人影全无，镜中却能看见他。镜内人听不见现实声音，双方只能借电话或镜面文字沟通。（第137章、第138章）
- **模仿作为交换条件**：鬼在镜内先呈现与张伟七八分相似的男子样貌，后变成苗小善，并模仿她对镜写字的动作。动作由有延迟逐渐同步；杨间判断完全一致时，鬼会从镜中取代镜前活人，而活人进入镜内。（第139章、第140章）
- **空间隔绝与短暂通道**：杨间的鬼域能覆盖镜子表面，却不能直接深入镜内。鬼与苗小善动作同步、交换开始时，镜面产生拉扯，鬼域才得以短暂进入镜内；杨间藉此救出上官云。（第140章）

## 本批结果

杨间以鬼域先移开苗小善，再以厕所门板取代她原先的位置。鬼未能取得活人身份，因而没有成功显现在现实；但镜内外通道在此过程中开启，杨间以红光带出上官云。救援完成后，杨间因没有黄金容器、亦不愿为抓鬼继续冒险，选择立即带苗小善和上官云离开KTV。该鬼是否仍在原处、之后是否会继续袭击，原文截至第140章不详。（第140章）

## 第154-155章·警局替身与活动线索

警局大厅的反光墙面中出现被困的警员张元，现实里则有另一名外貌相同的“张元”继续在警局活动。杨间以KTV中上官云遭替代的经历判断，这仍是镜中鬼所为：活人被送入镜内，鬼借该活人的身份停留现实。监控显示假张元中午在食堂出现、约半小时前还经过大厅，说明其已混入警局一段时间；杨间随后在四楼玻璃后看见其惨白面孔，并通知刘建国自四楼开始封锁搜索。（第154章、第155章）

杨间又把张伟家中、张伟请客的饭店与KTV、以及张伟被拘留后的警局相连，推测镜中鬼并非随意出没，而是只能出现在张伟附近；它对上官云和张元动手，显示受害者并不限定为张伟本人。该推测基于地点重合，镜中鬼为何跟随张伟、是否有距离或持续时间限制均不详，不能据此当作完整规则。（第155章）

## 第156-160章·伪装破除与随张伟移动

假张元在档案室现身时，制服文字反向、重复问答机械，且被子弹击中后无血无痛。杨间直接说破其不是张元而是鬼，伪装随即僵硬、褪白并从现实消失；这只能使它回到镜内或玻璃倒影中，并非关押。杨间随后在窗玻璃中继续看见它下楼，也说明镜内张元无法仅靠点破伪装获救。该鬼可通过特定条件从镜内走出，但本批没有写明该条件究竟为何。（第156章、第157章）

杨间带张伟离开警局时，先见鬼在车前以张伟相貌显作幻影，又从后视镜看见其坐在后座。张伟回家后用化妆镜看见鬼站在沙发后，张显贵也在第160章从镜中看见二楼护栏旁的鬼影；现实直接观察均看不到它，鬼眼也不能直接看见。杨间据此认为它跟随张伟而非栖身别墅，且低密度的别墅环境使其较少取得替换身份的机会；“机会较少”仅是杨间据上官云、张元均在人多环境遇袭所作的判断，不代表绝对安全。（第157章至第160章）

## 第184-185章·售楼部内的避镜措施

张伟入住观江花园售楼部后，房内镜子与楼内玻璃、其他反光处均以黑布遮盖；他还以纸袋罩头，认为即使照镜子也不会使镜中鬼出现。杨间仅提醒鬼可能就在张伟所在房间，并未认可纸袋能规避袭击。张丽琴到二楼休息时发现所有反光处均被遮住，未擅自揭开；杨间随后将该鬼风险告知她，并以其体温、心跳等生命特征确认她尚未遭替换。遮镜是否能阻止鬼活动、鬼是否仍在售楼部、判断活人特征是否足以识别所有替身，原文均未验证。（第184章、第185章）
