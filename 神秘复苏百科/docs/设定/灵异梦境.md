# 灵异梦境

## 概述

灵异梦境是杨间在梅山村夜间被梁圆喊出老宅后进入的异常空间。它复现村外道路和县城，却与白天环境不一致，距离会被压缩、场景充满白衣头罩尸体、无头尸体和草绳吊尸。杨间在其中清醒地意识到自己可能在做梦，却无法唤醒或主动离开；截至本批，梦境来源、覆盖范围和谁被卷入均不详。（原文回跳第682章、原文回跳第683章）

## 对驭鬼者的影响

杨间进入后无法感知或使用鬼眼、无头鬼影、鬼手，甚至左手在梦中消失，身体状态近似尚未接触灵异时的普通学生。他发现自己对断手异常起初毫不在意，说明梦境会同步影响认知与警觉。杨间推测无论多强的驭鬼者进入这里都会失去厉鬼能力；这只是依据自身状态的推测，未获其他驭鬼者验证。（原文回跳第682章、原文回跳第683章）

## 已见景象与规则线索

- 白衣黑头罩尸体被梁圆连续刺中三处后倒下，却会重新站起，身份是鬼、鬼奴或梦境产物不详。（原文回跳第682章）
- 荒野无头尸体靠近三人时，杨间建议不要背对，三人转身后尸体停止并退回。杨间认为此景象可能受无头鬼影经历影响，不能确认现实中无头鬼影本体进入梦境。（原文回跳第683章）
- 小镇上空悬有草绳吊尸，草绳令杨间联想到鬼绳；他由多个熟悉元素推测这可能是自己的噩梦。该关联是推测。（原文回跳第683章）

## 风险与存疑

杨间记起“梦魇”档案中人若在梦中死亡、现实亦会死亡的说法，因而决定以躲避危险、活到天亮为临时策略。此说法在本批没有直接验证。若杨间意识在梦中死亡，其现实身体可能厉鬼复苏并危及梅山村周边；这同样是其风险判断，不是已经发生的结果。（原文回跳第683章）

梁圆和林小夕看似熟悉梦境、携现实购买的雨衣和刀具进入，但二人是现实活人、梦境投影还是受灵异操纵的对象尚不详。杨间已开始怀疑梁圆喊他出门可能是梦中鬼的诱导，不能把她们的言行视作可靠信息。（原文回跳第682章、原文回跳第683章）
