# 灵异梦境

## 概述

灵异梦境是杨间在梅山村夜间被梁圆喊出老宅后进入的异常空间。它复现村外道路和县城，却与白天环境不一致，距离会被压缩、场景充满白衣头罩尸体、无头尸体和草绳吊尸。杨间在其中清醒地意识到自己可能在做梦，却无法唤醒或主动离开；截至本批，梦境来源、覆盖范围和谁被卷入均不详。（原文回跳第682章、原文回跳第683章）

## 对驭鬼者的影响

杨间进入后无法感知或使用鬼眼、无头鬼影、鬼手，甚至左手在梦中消失，身体状态近似尚未接触灵异时的普通学生。他发现自己对断手异常起初毫不在意，说明梦境会同步影响认知与警觉。杨间推测无论多强的驭鬼者进入这里都会失去厉鬼能力；这只是依据自身状态的推测，未获其他驭鬼者验证。（原文回跳第682章、原文回跳第683章）

## 已见景象与规则线索

- 白衣黑头罩尸体被梁圆连续刺中三处后倒下，却会重新站起，身份是鬼、鬼奴或梦境产物不详。（原文回跳第682章）
- 荒野无头尸体靠近三人时，杨间建议不要背对，三人转身后尸体停止并退回。杨间认为此景象可能受无头鬼影经历影响，不能确认现实中无头鬼影本体进入梦境。（原文回跳第683章）
- 小镇上空悬有草绳吊尸，草绳令杨间联想到鬼绳；他由多个熟悉元素推测这可能是自己的噩梦。该关联是推测。（原文回跳第683章）

## 风险与存疑

杨间记起“梦魇”档案中人若在梦中死亡、现实亦会死亡的说法，因而决定以躲避危险、活到天亮为临时策略。此说法在本批没有直接验证。若杨间意识在梦中死亡，其现实身体可能厉鬼复苏并危及梅山村周边；这同样是其风险判断，不是已经发生的结果。（原文回跳第683章）

梁圆和林小夕看似熟悉梦境、携现实购买的雨衣和刀具进入，但二人是现实活人、梦境投影还是受灵异操纵的对象尚不详。杨间已开始怀疑梁圆喊他出门可能是梦中鬼的诱导，不能把她们的言行视作可靠信息。（原文回跳第682章、原文回跳第683章）

## 原文回跳第684-688章·时间限制、现实映射与梦中鬼

梁圆和林小夕称，梦中真正的鬼会主动接近三人，出现得越晚越危险；她们因而试图在早期主动寻找、杀死它以结束噩梦。杨间据此推测，梦中杀死真正的鬼可能是醒来的方法，但鬼不会被永久杀死，梦境也会反复出现。该规律来源于二人的经验，尚未由杨间完整验证。（原文回跳第684章）

梦中人物死亡会映射现实。梁圆在阳镇梦境错杀一名中年女子后，现实县城一名熟睡女子同步以双手撕裂颈部、胸口死亡，证明梦境中的伤害并非只影响意识。杨间因此将梦中陌生人视为可能被卷入的真实活人，不能随意以试错方式攻击。（原文回跳第684章、原文回跳第685章）

阳镇集市先出现鬼绳吊尸和追击尸体，后又化为人潮。林小夕被人群拖走，梁圆随后冲入人群失散。杨间在集市中看见敲门鬼、饿死鬼、鬼棺、鬼新娘等与自身经历相关的恐惧投影，但它们被刀刺杀后并未终止梦境；真正的梦中鬼则伪装成抬棺人，以棺材钉刺穿杨间腹部后保持距离等待其死亡。（原文回跳第685章至原文回跳第687章）

杨间重伤时被未知者拖离集市，最终在村口醒转，继而在现实老宅床上惊醒。江艳确认自己没有做梦，只见杨间整夜出冷汗。杨间据此只能确认自己脱离当次梦境，未知拖行者身份、梁圆和林小夕现实下落、梦中鬼是否仍在村周活动均不详。（原文回跳第688章）

## 原文回跳第699-703章·喊名入梦与整村扩散

林小夕在梦中被鬼拖走后，现实中以刀反复斩断自己颈部死亡，杨间据此确认梦中被鬼杀害会令现实身体自杀式死亡。梁圆因身沾血、同住而被误认凶手，办案人员却初步判定自杀。梦境杀人已从先前的推测获得直接个案印证。（原文回跳第699章、原文回跳第700章）

杨间清醒守夜时，听见陌生男子在楼下喊“杨间”，随即在未睡状态被强制拉入梦境；江艳也听到仿冒杨间的声音喊名而入梦。杨间据两次经历推断，被喊出名字可能是进入梦境的触发条件，清醒和不睡并不能阻止。触发是否必须听见声音、能否由鬼模仿任意人声、能否拒绝均不详。（原文回跳第702章）

本次梦境复现十余年前的梅山村并卷入大量村民，杨间认为危险已从个体噩梦扩展为区域事件。梦中鬼能伪装成普通村民混入人群：它先以惨叫和血脚印引走梁圆，后从老宅走出、改变惨白面容为普通村民。鬼的真实相貌、杀人规律与如何从人群中辨别仍不详。（原文回跳第702章、原文回跳第703章）

杨间在梦中仍将失去驭鬼能力，无法凭鬼眼识别大量村民，只能限制江艳独处并试图寻找鬼。梁圆仍主张先杀人缩短鬼出现时间，杨间认为整村场景不适用该策略；二人未形成解决方案。梦境是否已影响邻村、是否与父亲分尸封藏或遗像房脚步有关均不详。（原文回跳第703章）
