# 鬼血

## 概述

鬼血是严力所驾驭厉鬼的表现形式。第44章，他曾以从双手渗出的猩红鲜血限制无头假模特；第52章则说明，鬼血能限制厉鬼行动，但频繁使用会加快自身复苏。第143章，严力的身体已被鬼血侵蚀至面容消失、仅余血液轮廓；他主动驾车离开大昌市，试图避免复苏后的鬼血在市区扩散。（第44章、第52章、第143章）

## 已知表现

- **限制作用**：严力以鬼血困住无头鬼影时，鬼影会被血泊限制，随后可连同血液流入黄金盒中关押。（第61章）
- **复苏与扩张**：严力在高速旁彻底失去人形后，鬼血从翻覆跑车持续涌出，先填满凹坑形成血池，继而扩大为猩红池塘。第145章王小强称，严力死亡后不足一天，鬼血已覆盖约两千平方米、深至少两米，增长尚未停止。（第143章、第145章）
- **复苏后的行动性**：血池中伸出染血手臂与人形轮廓，并朝最近村庄爬行，显示严力意识消失后鬼血并非静止污染物。（第143章）

## 风险与处置

严力认为若在大昌市内复苏，鬼血可能顺地下管道侵入水管、波及全市；因此他主动远离市区。王小强在第145章也据其扩散速度认可这一后果的危险性。原文截至第145章未写出鬼血后续是否被关押、村庄是否遭袭，不能确认其完整杀人规律或处理结果。（第143章、第145章）

## 原文回跳第673章·杨间的血池死亡影像

杨间在大阪使用除灵社手摇式放映机时，机器首次放出血红投影并令画面中的血滴实际渗到宴会厅地面。画面指向大昌市郊高速附近的严力鬼血池：血池内浸泡尸体，最后一具外貌、体型和穿着近似杨间的尸体浮出水面。该画面是放映机所展示的未来死亡片段，不是截至本批已经发生的事实。（原文回跳第673章）

杨间认为鬼血具有压制厉鬼作用，推测自己未来也可能是在无路可退时主动进入血池，尝试压制体内厉鬼或搏取生机；这只是他根据自身行事方式作出的推测。放映机因血池附近灵异而画面模糊、残缺并中断，未显示杨间进入池中的原因、死亡时间、尸体是否真正死亡或血池后续状态。（原文回跳第673章）
