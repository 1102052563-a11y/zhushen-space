# 厉鬼分级

## 概述

厉鬼分级是各国政府按“对世界或人类的危害程度”制定的灾害参考标准，不直接等同于厉鬼本身的恐怖程度。杨间在第25章通过国际刑警保密网站查到C、B、A、S四级定义，并注意到同样恐怖的厉鬼可能因影响范围不同获得不同级别。（第25章）

## 四级定义

- **C级：限制级**：固定在一个区域内，危害程度低于一百人；可被驭鬼者限制、收容，不具传播性。（第25章）
- **B级：危险级**：具游走能力，危害程度低于一万人；无法收容但仍可被驭鬼者限制，具一定扩散性。（第25章）
- **A级：灾难级**：若放任，会在一定时间内杀死一座城市人口；目前无法限制、收容，具相当大扩散性。（第25章）
- **S级：毁灭级**：具摧毁数个城市人口的能力；若无法遏制，甚至能毁灭一个国家。（第25章）

## 使用记录

赵建国在第22章将敲门鬼直接定义为A灾难级。第25章网站中，米国“幽灵教堂”被悬赏资料定为C级，但评论者依据其影响环境、接近形成鬼域等现象质疑至少应为B级甚至A级；这显示网站分级和实际风险可能不一致。（第22章、第24章）
