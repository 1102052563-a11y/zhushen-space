# 信使

## 概述

信使是被不明灵异诅咒卷入、定期负责递送黄色信件的人。李跃称自己两个月前仍是普通新人，因在家听见敲门、开门后发现落地的信封，好奇拆开便成为信使；其后约每月送一次信。信使不是普通职业称谓，而是与某个诡异邮局和送信规则绑定的身份。（原文回跳第694章、原文回跳第698章）

## 送信与失败代价

信件会把信使引向危险地点，李跃一行本次便按黄色信封线索到梅山村树林寻找尸体。李跃称丢失、未送达信件会招致厉鬼袭击，偶有信使能躲过第一次，但下一次袭击的恐怖程度会翻倍；死亡者身上的信也会消失。这些规则来自信使经验，具体的失败判定、翻倍方式和是否存在例外均不详。（原文回跳第694章、原文回跳第695章）

杨间强行撕开李跃的信后，树林被黑暗和入侵型厉鬼笼罩，周林一度死亡，说明非正常开信会中断任务并引来危险。信纸上出现“千万不要打开这封信”“打开这封信的人会被厉鬼杀死”“鬼就要来了”“鬼就在你的身边”等字句；信件随后化灰。该袭击是否只针对拆信者、也会波及在场信使，或与信件本身原有任务如何关联均不详。（原文回跳第695章至原文回跳第698章）

## 邮局与脱离方式

李跃称信使会偶然在开门、行走等过程中进入“邮局”，无法靠定位确定其位置；他不知道邮局所在何处。其所知唯一脱离办法是继承、掌控那座诡异邮局，送信过程中可逐步获得更多秘密。邮局是否实体建筑、继承需满足什么条件、是否真有人成功摆脱诅咒均未证实。（原文回跳第698章）

## 原文回跳第739章·鬼邮局实体被发现

杨间在大汉市建设路烂尾楼重叠的鬼域中，强行进入一栋牌匾写有“鬼邮局”的民国建筑，并遇见万兴等疑似信使。建筑内存在带编号的房间、不同年代人物画像和受压制的灵异环境，说明李跃所称邮局并非纯粹传闻或单次幻觉，而至少存在可被强行进入的实体灵异空间。（原文回跳第738章、原文回跳第739章）

万兴将杨间称作“新人”，且熟悉进入邮局后难以存活的风险。其是否正在送信、是否所有信使均居住于此、邮局与黄色信封的完整机制仍未披露；不能据此确认李跃已抵达或死亡。（原文回跳第739章）

## 原文回跳第740-744章·层级任务与红色集体信

一楼信使万兴、王善称每层最多七人，每成功送出三封信就会被带到上一层；越高层送信周期、休整时间越长。这是信使间流传的经验，万兴尚未上过二楼，王善也只完成两次任务，不能视为已验证制度。杨间将其理解为筛选精锐的可能机制。（原文回跳第740章）

王善另推测第三封信是“上楼信”，若有人抢走并成功投递，可能取代原信使升级、令原信使次数清零。该说法没有成功先例，仅是他的推理，杨间暂时同意等待其和万兴的第三封信以验证。（原文回跳第741章）

信件位置可能决定任务范围：床头柜信为个人任务、房门夹信为同房信使任务、柜台信则疑为全层集体任务。原文回跳第743章，一楼柜台出现前所未见红信，杨间取信后获得七日内投递至大海市福寿园78号主人的任务；红色信的失败代价、奖励和是否会让全层信使共同受罚均不详。（原文回跳第743章、原文回跳第744章）

## 风险

信使缺少稳定的驭鬼者能力，往往使用骨制子弹、纸钱、骨灰等粗糙灵异物自保。李跃小队本次一人被杨间杀死，周林又遭厉鬼入侵，显示信使身份不能保证安全；李跃仍视单次任务仅死一人为相对幸运。信使组织规模、管理者、信件投递对象和与普通民众的关联均不详。（原文回跳第694章至原文回跳第698章）
