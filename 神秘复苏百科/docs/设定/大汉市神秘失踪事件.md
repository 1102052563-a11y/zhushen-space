# 大汉市神秘失踪事件

## 概述

大汉市神秘失踪事件是孙瑞将持续人口失踪升级为C级灵异事件后使用的暂定代号。大汉市表面长期没有正式灵异事件记录，但孙瑞助手统计到约三四个月内几乎每天失踪九人；单日数量在千万级人口城市中不显眼，持续稳定发生则明显异常。（原文回跳第736章）

## 已知模式

失踪者初期常仍可和家属联系，因此难以界定为失踪或短期外出；其后会再次失踪、彻底消失，既找不到尸体也查不到出行记录。孙瑞据此怀疑他们被鬼盯上、暂未立即死亡，但没有掌握具体位置、目标筛选条件或失踪后的状态。（原文回跳第736章）

该事件在本批之前未上报总部。孙瑞曾调查城市内各类可疑区域而无结果，只将其作为低级灵异事件内部建档；这也解释了大汉市“没有灵异事件”的表面记录与持续失踪数据间的矛盾。（原文回跳第736章）

## 与鬼邮局的关联

杨间到建设路烂尾楼后，从墙体、水泥地和天花板取出失踪者遗留的黄金饰品。由于黄金不受灵异影响，孙瑞判断被带走的是人、金饰则被留在现实，烂尾楼很可能是失踪者进入另一灵异空间的交汇点。杨间强行进入与烂尾楼重叠的鬼邮局后又发现万兴等活人，进一步支持失踪者可能被卷入邮局的推断。（原文回跳第738章、原文回跳第739章）

此关联仍非完整结论。万兴和未具名女子是否属于统计失踪者、每日九人是否全部进邮局、鬼邮局为何需要或容纳普通人，均未揭示；C级定级也可能随内幕发现而变化。（原文回跳第736章、原文回跳第739章）
