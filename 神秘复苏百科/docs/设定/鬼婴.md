# 鬼婴

## 概述

鬼婴是周正腹中驾驭的厉鬼，周正称其档案命名为“鬼婴”。第9章，周正因连续使用厉鬼力量到达极限，鬼婴破开其腹部脱出，周正随即死亡。鬼婴浑身青黑、眼睛漆黑无瞳孔，体型像四五岁孩子般大小，却能以蜘蛛般方式沿墙快速爬行，并在第10章连续攻击杨间与王珊珊。（第9章、第10章）

## 与周正的关系

- 周正说鬼婴最初进入他腹中时只有拇指大小；每使用一次厉鬼力量，鬼婴便成长一分，即使不使用也会继续成长。（第9章）
- 鬼婴已吃空周正的肠子和内脏。周正的肚皮膨大而近乎透明，能看见青黑色婴儿在内蠕动、撕扯；他称自己每天承受内脏被啃食的痛苦，换作常人应已死亡，却因驾驭关系仍无法死去。（第9章）
- 周正用腹中力量拖延敲门鬼后，鬼婴开始复苏。他把卫星定位手机交给杨间，要求其离开，称若鬼婴跑出，学校将有两只厉鬼，幸存者更难逃离。（第9章）

## 脱出与能力表现

- 周正倒地后，鬼婴以青黑手臂撕开其肚皮，探头观察四周。它从周正身体爬出后，周正腹部迅速干瘪，未流鲜血或露出内脏，只剩发黑皮肉；周正随后失去生机。（第9章）
- 它会发出“哇嘎嘎”的怪叫，并以常人不及的怪异速度沿墙爬行。第10章，它从楼梯口迅速爬下，锁定正在带人撤离的杨间。（第10章）
- 它能把细长手臂诡异拉长，隔着黑暗抓人。其手臂袭向杨间时，杨间后脑显出的红色鬼眼盯住它，令其停止并收回手臂。（第10章）
- 攻击杨间失败后，它转而从后抱住王珊珊，双臂勒住颈部并发生皮肤黏连。第10章未写出这次袭击结果，也未展示可彻底限制鬼婴的方法。（第10章）
- 第11章，杨间为救王珊珊击中其头部，鬼婴未受实质伤害，随即张口吞咬杨间半截手臂。杨间手臂暴增的三只鬼眼发红光后，它吐出手臂、失去活力般落地并迅速爬走，属于暂时退避而非被消灭。（第11章）
- 王珊珊获救后颈后留下两个青黑手掌印，颜色继续向外扩散。原文没有确认该印记的具体效果，不能直接认定为鬼婴造成的诅咒或标记。（第11章）
- 第29章，王珊珊颈侧的两个青黑小手印仍无法洗除，且她自称鬼婴般的东西来到十六层窗外找她。其父在浴室发现小孩手印从窗外延至墙壁与吊顶；这些现象是否均由鬼婴造成，原文尚未确认。（第29章）
- 第34章，王彬夫妇看见一个青黑皮肤、无衣、似死婴的小孩进入王珊珊房间；其后房内留有小孩脚印。王彬从猫眼看见同样特征、黑眼无瞳孔的小孩在门外顶门，门锁全上后仍传来大门被打开的声音。杨间据电话内容判断堵门者就是鬼婴，正赶往现场。（第34章、第35章）
- 第36章，鬼婴已能站立行走、沿墙高速移动，口腔张成无舌无腔的漆黑洞口，试图吞食王珊珊；杨间发现其体型已较学校时增长，称其更像“鬼孩”。它还放出青黑气息、开始影响周围，杨间判断为鬼域雏形。杨间令其坠下十六层，鬼婴摔扁后仍能恢复站起，证明未被杀死。（第36章）
- 第38至39章，刘小雨推断它以“吃”为杀人方式，触碰可能是优先触发条件。王珊珊的鬼婴印记被杨间以鬼眼印覆盖后，鬼婴失去首选目标，转而在街上随机跟随一名撞到它的年轻人。（第38章、第39章）
- 第124章，杨间在市中心写字楼十五层看见青黑皮肤、黑眼无瞳孔的人影，认出是鬼婴。它已从第36章的鬼孩长成青年形态，青黑阴霾覆盖楼层，杨间认为其鬼域已接近形成；楼内职员被逼跳楼或遭拖回，伤亡持续。（第124章）
- 第263章，王小明以全市广播告知幸存者第三阶段鬼婴的避险方式为躲藏、不看、不出声、不接触；这救下一部分人，但并未阻止城市停水停电后的生存危机。（第263章）第264至265章，杨间以鬼眼鬼域在观江小区定位第一至三阶段鬼婴，发现第三阶段会因听到他的声音主动袭击。他用鬼影侵入两只第三阶段鬼婴后，目标均化作黑青阴霾消散；之后把小区数十只各阶段衍生鬼婴集中，同法全部驱散。杨间据此认为衍生鬼依赖鬼婴力量，鬼影能驱散该力量；源头鬼和第四阶段鬼婴未被清除。（第264章、第265章）

## 出处

第9章、第10章。

## 第391-392章·王珊珊腹内异常个体

杨间从王珊珊腹内取出一只巴掌大小的鬼婴：它通体淡青、双眼猩红如鬼眼，既不像死婴，也不同于他此前见过的凶恶鬼婴。其最先看见杨间，却没有按杨间所知的出生后袭击首见者规律攻击，也没有攻击身旁王珊珊；杨间据此前核查，将其判断为自身复活诅咒延续形成的特殊个体，而非受关押饿死鬼控制的衍生物。这是杨间的判断，不能据此归为周正腹中脱出的原鬼婴或确认完整规则。（第391章）

杨间认为该个体可被杀死，也可通过吞食厉鬼成长；他推测若成功豢养、并多次喂鬼，可能成长为比饿死鬼级别更高的恐怖存在，但同时承认会有失控风险。王珊珊接手时它显得自在，杨间随后用鬼报纸写入听从二人、不得擅杀、不得进入人群三项命令，并安排王珊珊带黄金盒、白色鬼烛和特制手枪到新庙独住照看。此处的成长上限、三项命令的效力、是否会吃人或吞鬼以及与既有各阶段鬼婴的关系均未证实。（第391章、第392章）
