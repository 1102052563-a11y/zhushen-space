# 笑脸

## 概述

笑脸原为童倩后脑的鬼脸，表现为会发出怪笑的诡异人脸。杨间在中山市凯撒大酒店的持报尸体上已见过同类笑容，并据此判断篡改记忆鬼可能夺走了童倩鬼脸；第371章至第373章，他确认医院中冒用童倩身份的鬼已携带这张笑脸。它既能以笑声杀人，也使冒名目标的危险显著上升；原文没有给出独立档案名或完整规则。（第336章、第371章至第373章）

## 笑声与危险

杨间向冯全、罗素一说明，笑脸和童倩原有的哭脸能力相似，都是以声音杀人：不听见笑声便不会受直接影响。第371章，总部拨通第三中医院电话后，笑声从耳机传出，一名联络员身体僵硬、脸露笑容并倒地死亡，声音还在通讯室扩散。第372章，病房目标放出的笑声传遍医院；第373章，杨间以第二层鬼域屏蔽时仍称声音会在停留中越来越清晰、越来越强，长久后连第三层鬼域也可能挡不住。这些判断主要来自杨间，笑声的最大范围、听见多久会死亡及各类隔绝方式的稳定性均不详。（第371章至第373章）

## 分离与收容

第三中医院顶楼病房中，杨间、冯全、罗素一以三种灵异同时压制冒用童倩身体的鬼，笑声随之中断。杨间先把鬼报纸贴上笑脸，想以此前取冯全脸的方式分离，却发现数分钟后毫无作用；他认为报纸与冒名鬼属于同一整体，不能用其一部分对付它自身。随后无头鬼影化出黑色手掌，从童倩身体脸上强行把笑脸取下，杨间立即放入黄金盒关好。（第373章）

失去笑脸后，童倩身体仍被鬼影压制且有行动能力，表明笑脸并非篡改记忆鬼的全部。杨间将笑脸带往罗素一别墅，并在第375章依人皮纸方案把它贴到赵磊身体右脸；与此同时，哭坟鬼遮脸布上的哭脸贴到左脸，二者立刻同时复苏。杨间为避开笑、哭两种声音而以鬼域离开，笑脸是否已与哭脸达成平衡、是否仍在黄金盒外受控及最终归属，均未写。（第373章、第375章）
