# 刻碑鬼

## 概述

刻碑鬼是福寿园诡异坟场鬼域内的未知厉鬼。它未在众人面前显形，只以“叮，叮叮”的凿刻墓碑声活动；杨间循声赶到时，只看见新立墓碑上出现的姓名和渗血刻痕。其外貌、位置、是否独立于坟场鬼域、与红信或刘老太尸体的关系均不详。（原文回跳第752章、原文回跳第753章）

## 已知规律

### 刻名预兆

罗素一被袭击前，坟场先响起凿击墓碑声。杨间追至声源时，墓碑已刻上“罗素一”并渗出血迹；罗素一旋即陷入泥土、被未知死人手抓住，身体与体内厉鬼像被一同埋入未知空间。由此可暂定：凿刻声是其锁定或完成杀人准备的明显预兆。（原文回跳第752章、原文回跳第753章）

这不等于鬼只要听见姓名就能刻名。它如何获知罗素一姓名尚有多种可能：预先记录名单、通过众人对话获知，或以其他方式识别入域者；杨间没有确认任一种。（原文回跳第754章）

### 坟墓埋入

名字刻完后，罗素一踩空并迅速沉入松软黑泥，林落梅鬼音无法命令泥土内的鬼，杨间鬼手也未能接触目标。杨间判断罗素一并非被实体泥土掩埋，而是被拖入未知灵异空间；对应写有姓名的老坟可能是其当前所在或死亡标记，但尚待挖掘验证。（原文回跳第753章、原文回跳第754章）

刻碑鬼一次似乎只处理一个名字，效率较低却对两鬼驭鬼者也具高度致命性。其是否必须按姓名笔画完成、能否在刻完前打断、一次刻名后间隔多久再次行动均不详。（原文回跳第753章、原文回跳第754章）

## 应对与风险

杨间认为刻碑鬼杀人过程存在从敲击声到姓名完成的短暂窗口，但罗素一姓名笔画较少，众人未能及时阻止。鬼眼无法窥视坟中内容，挖坟可能救回未死的驭鬼者，也可能放出更危险对象；杨间仍决定挖写有罗素一姓名的坟，以验证埋入是否存在救援可能。（原文回跳第753章、原文回跳第754章）
