# 鬼奴印

## 概述

鬼奴印是第38章由刘小雨说明的厉鬼标记。她称，能够使用鬼域的厉鬼可把死者变为鬼奴；在人成为鬼奴前，厉鬼可能先留下相当于“商标”的标记。王珊珊颈部的两只青黑小手印被刘小雨判断为鬼婴留下的鬼奴印：她尚未死亡，却已被鬼婴视为目标，一旦死亡便可能被其控制。（第38章）

## 鬼婴印记

- 鬼婴在第10章勒住王珊珊时留下两个青黑手掌印；第29章该印记难以洗去，鬼婴随后循此再度找上王家。（第10章、第29章、第34章）
- 刘小雨根据档案推断，鬼婴的杀人方式是“吃”，触发条件则可能是触碰；王珊珊与鬼婴有过接触，因而成为其优先目标。即使不触发明确条件，鬼仍可能随机袭击周边人，发现规律只能提高生存率。（第38章）
- 杨间发现王珊珊手腕原有一个接近消失的淡红眼印。该印为他在学校救人时无意留下，正因其暂时压过鬼婴印，鬼婴数日未直接杀死王珊珊；鬼婴成长后，红印逐渐失效。（第38章）

## 替换方案

刘小雨提出未获实践验证的方案：能使用鬼域的驭鬼者，也许可用自己的鬼奴印覆盖厉鬼印记。驭鬼者是活人，其活着时的“鬼奴”据称仍保有理智，不同于厉鬼控制的死者；但若驭鬼者死于厉鬼复苏，其鬼奴将被体内鬼接收，风险没有消失。（第38章）

第39章，杨间以手掌鬼眼红光侵入王珊珊手腕，令颈部青黑手印褪去，并在其手腕留下清晰、微发红光的鬼眼烙印。他感到部分鬼眼力量留在王珊珊体内，不能确认后续副作用；鬼婴则因失去明确目标，转向路人随机跟随。该处理只改变目标关系，并未消灭鬼婴或根除风险。（第39章）
