# 鬼婴寄生

## 概述

鬼婴寄生是第218至220章在张丽琴身上出现的异常：其腹部在极短时间内隆起，内部有拳头大小、会蠕动并隔肚皮张口咬人的鬼婴。杨间判断这不是普通怀孕，也不能由一般医学手段处理；其与周正体内鬼婴、以及笼罩大昌市的青黑阴霾之间的具体关联尚未证实。（第218章、第219章）

## 已知风险

- 鬼婴尚未开始啃食张丽琴内脏时，她仍有机会被救；一旦开始吃内脏，宿主会依赖鬼婴维持生命，取出鬼婴或其自行破腹都会导致宿主死亡。（第219章）
- 张丽琴无法控制鬼婴，杨间因此认定她不是驭鬼者，而是单纯遭到附身寄生。（第219章）
- 杨间推测寄生个案可能是大昌市灵异事件爆发前的受害表现，但该推测缺乏直接证实。（第219章）

## 处置尝试

杨间先以鬼域覆盖张丽琴，确认鬼婴可随人体一同进入鬼域；再用无头鬼影手臂穿入其腹部，发现鬼域虚化人体时鬼婴也会虚化，鬼影无法接触目标。为在不伤及宿主的情况下分离两者，他开始尝试局部驾驭无头鬼影，让鬼眼与鬼影力量配合；截至第220章末，试验尚未完成。（第220章）

第221章，杨间以局部驾驭后的左臂配合鬼域，追抓并无伤扯出张丽琴体内鬼婴；他用干枯手指钉住目标，再与手指一并密封进黄金盒。第224至225章，市内已确认同类寄生由一百二十例增至一百三十八例，其中包括四十名男性；王小明将其视为鬼婴群体事件，并要求先取得样本研究。（第221章、第224章、第225章）

第232至234章，已确认寄生案例超过两百。王小明通过医院男性患者的第一阶段鬼婴总结：第一阶段会袭击看见者，第二阶段会袭击接触者，第三阶段推测会袭击被其听见声音者，即“不见、不碰、不听”；第三阶段规律仍未获直接验证。杨间以局部鬼域、左臂鬼影取出样本，再把其放入覆盖全市的鬼域地图观察行动。（第232章至第234章）

第241章，王小明一队在医院楼梯口近距离绕开第二阶段鬼婴，张韩被其注视却未遭袭；王小明据此暂判衍生鬼婴各阶段的条件不叠加，第二阶段只在接触时杀人。鬼婴群随后由楼梯、电梯及墙壁合围，能使墙壁与黑青阴霾同化并穿墙进入病房。第243章，王小明依据首轮交战确认第三阶段鬼婴会攻击被其听见声音者，因而一至三阶段分别对应看见、接触与听见；这项“单独触发”结论仅针对本次出现的衍生鬼婴。第245章，他同时判断源头鬼具有条件重复叠加特性，并预言鬼烛熄灭后会优先袭击杨间；源头鬼身份、攻击结果及判断是否正确，均未揭示。（第241章至第245章）

## 出处
