# 鬼差

## 概述

鬼差是王小明在第445章为培训基地失控目标确定的S级灵异事件代号。事件源自其鬼棺实验：受试者卫景被尝试转化为保持意识的真正的鬼，实验失败后目标离开实验室并侵入培训基地。王小明不能完全确认目标的性质，只要求行动者把它视为“被鬼操控了的驭鬼者”应对；卫景是否还保有意识、鬼棺是否与目标完全融合均不详。（第444章、第445章）

## 已知规律与能力

- **三鬼压制起步**：王小明说明，鬼棺中的鬼能压制一名驭鬼者体内三只鬼的能力；驾驭数量低于三只者面对目标近似普通人。杨间在靶场遭袭时，一层鬼域被压至不足十厘米，无头鬼影的拼接压制也未能取下夺纸手掌，印证其面对该目标时的失效表现。（第441章、第444章）
- **夺鬼成长**：目标若杀死驭鬼者并夺走其体内鬼，压制数量会由三增长为四，并可能继续提高，上限不明。王小明因此认为培训基地每多一名驭鬼者死亡，后续处置就会更危险。（第442章、第444章）
- **区域数量判定**：冯全依据黄岗村经历说明，早期同类目标并非按人头判定，而是以约一间正常房间大小范围内的鬼数量为条件；区域中合计不少于三只鬼可避免触发其对落单者的袭击。该规律是否仍适用于已成长的鬼差，原文未确认。（第445章）
- **重复出现**：冯全称当年曾将同类目标关押，但第二波袭击又以另一身份、方式出现，像会反复重启，直到杀死所有人。鬼差是否必然具备同样的完整机制、重启条件和次数均不详。（第445章）
- **成长后的数量对抗**：第446章，杨间由楼下尸体与已丢失的鬼绳推算，鬼差至少可压制五只鬼；第448章，一名走散驭鬼者被取走体内鬼后，数量升为六；第450章又一人惨叫后，杨间判断其至少为七。张雷一行四人实际合计六只鬼，曾在五对六的安全条件下遇到鬼差，却因逃跑走散。数量是否只按局部范围计算、被压制或吞入体内的鬼是否一律计入、七只之后的上限均不详。（第446章、第448章、第450章）
- **八鬼压制与距离**：第451章，杨间为救张雷而靠近时发现，一间房间大小的数量判定距离约在三米内而非其原先以为的五米。第453章，人皮纸称鬼差已能压制八只鬼，并指向又有落单驭鬼者在杨间一行离开宿舍楼后遇害；第454章，杨间又在基地道路旁见到半具新尸。人皮纸信息可能带有陷阱，但杨间认为其既有信息并不直接说谎；八只是否为即时精确数、遗体对应人数及距离规则是否固定均不详。（第451章、第453章、第454章）
- **至少九鬼压制与白烛牵引**：原文重复第456章“数量上的比较”中，杨间五人本有八只鬼，放出鬼手后鬼手也被压制，故推断鬼差至少压制九只鬼；王江仍受袭，说明鬼手是否完整计为一只、鬼差是否已达十只或袭击另有来源均未明。打开魂瓶后，鬼眼和鬼影恢复使用，显示本方数量已超过鬼差压制上限；但王江手臂仅留鬼绳勒痕，杨间不能确认此前攻击由鬼差直接发动。白色鬼烛已成功令鬼差放弃钱毅、持续留在杨间队伍附近，具体牵引条件和持续时长不详。（原文首个第456章·厉鬼来袭、原文重复第456章·数量上的比较、第457章、第459章）
- **压制的代价与第二波风险**：张雷在正面接触后判断，鬼差每夺取一只鬼即可获得其部分能力、增加可压制数量；但每压制一只鬼，也会令自身失去一项诡异能力。第463章，杨间据钱毅、张雷和自身等被压制的数量，只让王江再接触一次，使鬼差维持八鬼压制、仅余一种能力。杨间明确不能使其完全失能：若形成完整压制，它会舍弃卫景尸体、换身份重现并发动第二波袭击。该规律是张雷和杨间基于本次现场形成的判断，完整换身条件、是否必然发生与新身份表现均不详。（重复第462章·食鬼之人、第463章）
- **棺材钉融合与鬼绳外放**：杨间在卫景尸体肩部发现棺材钉已化作通向心口的黑痕、与皮肉融合，试图肢解取出时被鬼差压制。王江再接触后，鬼差剩下的能力表现为鬼绳：大量老旧草绳从上方垂下，绳套会找向每人颈部，具无差别复苏、难以靠少数人长久抵挡的风险。鬼差是否一直控制曾被夺走的杨间鬼绳、棺材钉能否在不触发第二波袭击下取出及鬼绳是否是唯一残余能力，均未写明。（首个第462章·鬼回头、第463章）

## 现场表现

第441章，目标在黑暗中以僵硬双手争夺杨间的人皮纸，持续抽离无头鬼影，后被复苏鬼绳牵制而松手；鬼绳却随目标离开。第442章，培训基地宿舍楼上方出现沉重脚步，跳楼逃离的一名未具名驭鬼者被鬼绳套颈吊住。第443章，杨间两次离开基地均回到大门，第四层鬼域看见阻路棺材板，认定自身位于鬼棺内；目标是否同时在棺内、基地外空间是否真实消失均未确定。（第441章至第443章）

第447至450章，鬼差在宿舍楼四楼以黑暗遮蔽视野。张雷先摸到其冰冷僵硬的手，四人并未即时遭袭；走散后，一名未具名驭鬼者在阳台被双手抓住、压制体内鬼并从身体取走某物，死亡后留下残破尸体。第450章，附近再有一名幸存者惨叫并迅速停息，杨间据此判断鬼差又完成一次夺鬼。鬼差随后的精确位置、它是否已经锁定杨间队伍、两名受害者姓名和具体鬼物均不详。（第447章至第450章）

第451章，鬼差以死黑脸、腐烂的国际刑警制式服装和僵硬手掌现身张雷面前，红色鬼烛靠近时极快燃尽。杨间借烛光将张雷拉走后，鬼没有追击五人，显示队伍合计八只鬼时暂不满足落单条件。第455章，杨间主动点燃白色鬼烛，以灰黑烛光吸引鬼差，意图利用五人不落单的状态让其只能跟随、暂停猎杀其他目标；鬼差是否接受诱导、鬼烛是否会优先改变其规律或使其直接袭击，均未写明。（第451章、第452章、第455章）

原文首个第456章，白烛实际使逼近钱毅的黑暗退去、转向杨间五人。重复第456章至第459章，鬼差到场后压住杨间鬼眼，直至杨间增加本方鬼数才使鬼眼与鬼影恢复；它始终以黑暗和沉重脚步伴随白烛队伍，没有再离队猎杀。鬼差是否还控制被夺走的鬼绳、是否故意不立刻袭击、白烛熄灭后会发生什么，均未写明。（原文首个第456章·厉鬼来袭、原文重复第456章·数量上的比较、第457章至第459章）

第460至463章，鬼差放弃白烛牵引、转而袭击靠近的落单钱毅，显示烛火不足以在该条件下维持原有优先级。它仍穿卫景死尸的破旧血污衬衫，先以鬼绳吊住钱毅，再以黑手探入其体内搜寻厉鬼；杨间一行靠近后，攻击暂止。杨间侵入其肩部棺材钉时，鬼差使局部黑暗退去，却全面压制杨间鬼手与鬼影，继而转头发动赵昊遗留的鬼回头；张雷阻断回头后，王江接触使其只余鬼绳能力。鬼差最终能否被削弱、钱毅是否仍在其压制范围内及鬼烛对下一次袭击的影响，均未写明。（第460章、第461章、首个第462章·鬼回头、重复第462章·食鬼之人、第463章）

第474章，鬼差回棺后已可选择压制李军的鬼域，并在两名未具名驭鬼者逃离时令其自行扭断脖颈；二人身体内的鬼也未能反抗。第477章，鬼差在棺内仍能显出鬼绳、渗血、阴风及夺取其他鬼后遗留的灵异，童倩的笑声可暂退一波袭击，但其哭脸后来又被压制。原有压制范围、每种灵异的来源与完整条件均不详。（第474章、第477章、第478章）

第478章，鬼差借未知媒介把一名未具名驭鬼者拖入鬼棺，突破了团队不落单时原有的距离限制；王小明据此判断它在学习成长。李军接触目标时，鬼差发动一种不再让本体消失的重启：周围连续出现与原物相同的鬼棺，每口棺中都有站立鬼影，最终形成数百口棺材包围队伍。复制鬼棺是否皆为实体、重启能否停止及其后果均不详。（第478章）

第479章，复制鬼棺内的鬼影开始迈步逼近王小明等人，显示其并非立即消失的纯粹景象；李军所抓本体仍维持一脚迈出棺外的姿势，原有规律似仍在影响它。杨间以第五层鬼域带走全部可见目标及一口鬼棺后，鬼差失去袭击对象，复制鬼棺和鬼影陆续消失，黑暗中仅余一个人影；该人影另一脚尚未迈出棺外，复苏未被原文写为完全完成。第480章，鬼差黑暗仍留在杨间制造的山谷内，山谷只能遮掩和阻止普通人进入，不能阻止其离开；其下一次行动、复制物是否可再现及封锁时效均不详。（第479章、第480章）

第496章，总部向顶尖驭鬼者说明，鬼差当前至少可无解压制十六只鬼；在约一间房大小的平面范围内，若其拥有鬼的数量大于目标区域内鬼的总数，便会杀死其中驭鬼者。第497至498章，李乐平与王小明进一步把鬼差分析为棺形整片鬼域：从内部关押会触发重启，须在外部以更强鬼域压缩其范围，才可能装入容器。鬼域间压制是否足以胜过鬼差、压缩后如何安全收容均未验证。（第496章至第498章）

## 定级与处置争议

王小明考虑到压制能力会成长、目标优先夺取驭鬼者体内鬼，且可能具有卫景智慧，正式将事件定为S级并命名鬼差。姜尚白建议动用饿死鬼身上的棺材钉，由顶尖团队处理；王小明拒绝，理由是棺材钉未必能限制成长后的目标、不能应对后续袭击，且取钉会造成饿死鬼脱困风险。至第445章末，总部尚未形成完整处置方案。（第445章）

第496至497章，姜尚白再提取饿死鬼棺材钉，杨间以饿死鬼失控风险和辞职表态反对，曹延华暂时搁置。第498章，王小明认为棺材钉无法保证钉住一片鬼域，提出引来国外S级鬼画，用其鬼域压制鬼差；失败时也可能由鬼差压制鬼画。该计划尚未执行，不能视为总部已确定的处置方案。（第496章至第498章）

## 第516-518章·与鬼画碰撞后的黄岗村异变

总部引鬼画进入市郊后，鬼差所在的山谷被灰白鬼域覆盖。李军等人抵达原定位点时，棺材状黑暗与山谷均已消失，取而代之的是标有“黄岗村”的村庄；行动队试图撤离，又穿过树林回到同一村口。行动者据此认为两起S级事件的碰撞已经造成无法预测的变化，原本等待鬼画压制鬼差、再行关押的方案无法按原步骤继续执行。（第517章）

第518章，苏凡依据其游戏预警推测鬼差可能已经受压制，黄岗村呈现的是早期鬼棺事件的形态；这是行动者为选择路线作出的推测，并非确认结论。李军等人进入村内后，在首间房看见一具被鬼绳吊起的尸体，草绳另一端仍缠着黄金；第三间房又有头部扭转一百八十度、腐烂并带尸斑的尸体。行动队至少辨认出鬼绳、吊尸和扭头尸体三只鬼，却观察到它们暂未行动，推测存在某种平衡或压制。鬼差本体、鬼棺位置和这种平衡能维持多久，均不详。（第518章）

## 第519-523章·搜索无果、两片鬼域消失与异常人影

柳三向杨间转述，行动队曾到过原本放置鬼棺的房屋，遭到厉鬼袭击后成功关押其中一只，却没有发现鬼差；村内每间房屋都可能有鬼，逐间寻找的试错代价过高。李军绘图所见黑房连成棺形，灰白鬼画道路将其分割，双方边界相互侵蚀又消失，行动者只能判断两种鬼域存在微妙平衡。柳三五次替死纸人与将五人收入背景的遗像，则显示村中出现不符合既有鬼差档案的异常；其是否属于鬼差新能力，不能确认。（第519章、第520章、第521章、第522章）

第523章，行动队失联后，鬼画灰白鬼域与鬼差棺形鬼域同时消散。五个人影从中心走出，手掌都有大小相同的灰白空洞；杨间点燃白色鬼烛，五者齐齐转头，遂判定它们都是鬼。人影是否即李军、柳三、阿红、苏凡、徐一平的鬼化状态、鬼差是否已被关押或消失、两鬼域如何结束，原文均未确认。（第523章）
