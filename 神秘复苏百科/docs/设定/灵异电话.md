# 灵异电话

## 概述

第16至18章出现的异常电话，与论坛网名“雷电法王”留下的“138”开头号码及黑衫老人的敲门声有关。电话能在鬼域中拨通或显示接通，却可能无声；接听后听到敲门声，会招来黑衫老人。杨间据论坛发帖时间、错字和现场电话现象判断，论坛后段的发帖者已不是雷电法王本人，而是鬼借其手机和帖子散播诅咒；这一判断基于本批信息。（第16章至第18章）

## 本批记录

- 赵磊称自己此前拨打求救电话时，电话可以打通却没有声音，且会在无声状态下显示接通。苗小善补充，教室内电话曾可正常接通，鬼域再次出现后才出现异常。（第16章）
- 杨间用周正交给的卫星定位手机拨打论坛中雷电法王的号码。电话铃声没有从远方传来，而是在寂静学校中回响；接通后，教学楼三楼窗户出现手机光和模糊人影，促使杨间立刻挂断。（第16章）
- 该号码随后回拨杨间，又同时显示在王珊珊、苗小善、赵磊等人的手机上。钱万豪接到显示为王老师的电话并接听，听筒先有电流声，后传出“一长、两短”的敲门声；杨间虽关机，却认为声音已会引来老人。（第17章）

## 规则判断

杨间结合论坛内容推断：雷电法王在家门外第一次被敲门时可能已死亡，因此之后留下手机号、音频和急促错字的帖子或许由鬼继续发出；听到网上或手机播放的敲门声的人，会像被无形坐标标记般遭老人寻找。现实中现场听见敲门声则可能立刻致死。厕所内杨间播放音频却未遇老人，他从羊皮纸看到的说明推测，当时厕所所在区域有另一处更可怕的鬼域及隔绝效果。这些都是杨间在第17至18章的推断，尚非完整证实的规则。（第17章、第18章）

## 出处

第16章至第18章。
