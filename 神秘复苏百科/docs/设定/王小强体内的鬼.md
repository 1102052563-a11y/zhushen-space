# 王小强体内的鬼

## 概述

王小强体内的鬼是他在第147至150章战斗中放出的厉鬼。杨间只能在含黄金成分的烟雾里看见其模糊人形，无法看清相貌；王小强则称这只鬼“很特殊”。它与王小强由金线缝合的拼凑身体存在直接联系：王小强行走时，身上脱落的他人血肉会被该鬼捡起；王小强被斩首和补枪后，鬼又用其尸体的内脏、皮肉和无头骨架重组躯体，使其恢复意识。其正式名称、来历、杀人规律、是否能完全受王小强控制均不详。（第147章、第150章）

## 已见表现

- **跟随与拾取血肉**：王小强脱去衣服后露出由多种尸体部位以金线缝合的身体，行走中不断掉落血肉。烟雾内随即出现一道人形轮廓，拾起掉落肉块并跟在其后；杨间据此确认那不是人，而是一只真正的鬼。（第147章）
- **压制鬼域范围**：该鬼出现后，杨间感到自身鬼域被强行压缩、覆盖范围无法扩大。原文没有明确说明这是否为该鬼的独立能力，或只是其出现时与王小强身体关系造成的现象，故不能断言完整规则。（第147章）
- **无法随人一同被移走**：杨间以鬼域把王小强、雷虎等活人和自己转移到另一位置，刻意将此鬼留在原会议厅。王小强因此失去与鬼直接接触的机会，并担忧厉鬼在外过久会失控；这只能证明杨间当次转移没有带走该鬼，不代表鬼永久无法跨越鬼域。（第148章）
- **拆取尸体并重组成王小强**：王小强死亡后，该鬼走向尸体，取出心脏、肝、胃和肠子塞入自己的腐烂骨架，再撕取皮肉黏合；最后把没有头骨的王小强头颅套到自己头部。约半小时后，拼凑出的王小强睁眼复苏，且保有记忆和言语能力。（第150章）

## 与王小强的限制

王小强称自己的身体本就是王小明实验制造的拼合产物，并非通常意义上驾驭一只鬼的驭鬼者；他认为自己能复生，是这只鬼尚未抛弃其身体。复生后的身体更加破烂，他无法确定还能使用多少次，并承认多次死亡终有可能无法复活。以上是王小强在复生后的自我判断，不能据此确认复活次数、条件或该鬼最终选择身体的标准。（第147章、第150章）
