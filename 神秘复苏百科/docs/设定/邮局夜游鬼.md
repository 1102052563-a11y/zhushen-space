# 邮局夜游鬼

## 概述

邮局夜游鬼是鬼邮局在晚六点熄灯至次日早六点间出现的未知厉鬼。它的脚步声如尸体踩木板，能在不同楼层间跳跃出现；其位置在黑暗中飘忽不定，白天则未在一楼现身。王善称信使在熄灯后若仍留在房外，会被这只鬼杀死。（原文回跳第740章、原文回跳第741章）

## 已知规律

### 敲门灭灯

夜游鬼靠近17号房后连续以“咚，咚咚”的节奏敲门。第一次敲门，客厅吊灯熄灭；第二次敲门，厕所灯熄灭；第三次敲门时，卧室最后一盏灯开始闪烁。杨间据此推测其规律是每次敲门熄灭房内一盏灯，全部灯灭后，室内人员可能触发“人死如灯灭”的必死袭击。（原文回跳第741章、原文回跳第742章）

此推测尚未被全灯熄灭后的死亡结果验证。李阳以堵门鬼隔绝灵异后，已熄灯重新亮起、最后一盏灯未灭，说明堵门能中止或逆转当次熄灯过程。（原文回跳第742章）

### 开门入侵

三次敲门后，夜游鬼转而试图打开17号房门。它令木门严重凹陷、门把手剧烈转动，却被李阳以堵门鬼、杨间无头鬼影共同挡住。杨间判断“开门”是区别于敲门灭灯的第二种杀人规律，打开门后会造成何种必死后果尚未直接发生。（原文回跳第742章）

杨间将鬼手从门缝探出，摸到门外一只枯瘦、手指异常长的阴冷手掌；孙瑞发动重叠咳嗽声后，鬼收回手掌并离去。鬼瓷同步开裂承受李阳压力，表明这次对抗已造成强灵异冲击。夜游鬼的真实外貌、所在楼层、白天藏匿位置与是否另有更多规律均不详。（原文回跳第742章）

## 危险评估

杨间等人根据脚步、敲门、熄灯和开门行为判断该鬼至少有两种疑似必死规律，危险程度极高；若按总部事件分级建立档案，杨间认为至少可达A级。该判断只是现场评估，未形成正式档案，也不能据此确认它与敲门鬼是同一只或关联鬼。（原文回跳第742章）
