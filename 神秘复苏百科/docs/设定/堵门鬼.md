# 堵门鬼

## 概述

“堵门鬼”是杨间在鬼画内部一栋国外居民楼三层房间中遭遇的未知厉鬼的暂用称呼。第629章，杨间根据其会附着于尸体、封堵离开房间的门窗、又在原宿主被压制后转而附着李阳等表现，称其可叫“堵门鬼”或“鬼堵门”；李阳没有作出选择，因此本页采用前一名称作便于检索的暂称。它原先附着的腐烂外国男性尸体，是否属于先前进入鬼画而覆灭的国外驭鬼者队伍，杨间只作出猜测，尚未证实。（第627章至第629章）

## 已见表现

- **以尸体封堵出口**：杨间进入三层一间保留窗户的房间后，先见单人沙发上有一具腐烂尸体；尸体随后消失、门自行关闭。无头鬼影撞门时，门只能开出缝隙，杨间从缝中看见一条腐烂手臂横撑门前。门外的李阳和吉米则看见腐尸从房内跑出、立在门前。由此只能确认鬼能借该尸体堵住门，尸体是否就是鬼本体不详。（第627章、第628章）
- **一次封堵一个出口**：杨间尝试开窗时，堵门的尸体消失，大门随即微开；带尸臭的黑影移至窗外，以双手按住玻璃阻止开窗。杨间据门与窗不能同时被堵的现象推断，鬼一次只能选择一个位置封堵。该规律仅在这间房、这两个出口中观察到，能否封堵墙壁、是否存在距离或数量限制均不详。（第628章）
- **以困住目标作为袭击方式**：杨间判断鬼没有立即杀人，而是借封死门窗将人困在室内；普通人或驭鬼者若长期无法进食、饮水、睡眠，仍会死亡。这是杨间基于现场封堵作出的规律判断，鬼是否还有直接杀人方式、困住多久才会致死均不详。（第628章）
- **附身与转移疑点**：杨间以鬼手压制门前尸体后，尸体失去灵异支撑而散架，门重新打开。李阳随后出现尸斑并被杨间判断已被鬼附身。杨间提出两种未证实可能：鬼选择了最近的李阳，或最后从房内走出的人会承接这类诅咒；确切转移条件不能确定。（第628章、第629章）

## 本批经过

杨间原本在三楼寻找鬼画，因李阳看见房门内的黑影而进入查探。房内窗户未被鬼画抹去，吉米一度想由窗逃离；杨间认为这可能是另一只鬼干扰鬼画，或是刻意留下的陷阱，故独自入内。发现沙发尸体会活动、房门又被关上后，他没有以鬼域强行脱身，因为鬼眼与鬼域在鬼画内受到压制，只能以无头鬼影试门并观察封堵规律。（第627章、第628章）

杨间发现门窗只能择一封堵后，让李阳进入房间看住窗外鬼影，自己趁门未被堵时离开；但李阳跑向门口时，鬼先一步再度关门。杨间在门外用鬼手抓住腐尸，压住其灵异并令尸体散架，才使房门开启、李阳逃出。他没有关押该鬼，理由是没有关押工具且不愿在寻找鬼画时耗费时间；原房间被关门隔开，鬼的后续状态不详。（第628章）

## 与李阳的关系

李阳为协助杨间脱困而进入房间，在鬼由腐尸处失去灵异支撑后，成为最直接的附身对象。杨间告知他这意味着成为驭鬼者：初期未必感到控制，之后体内鬼会逐渐侵蚀身体，最终可能变成先前腐烂尸体的模样；驾驭第二只鬼或能以对抗延缓死亡，但也会增加风险。以上关于驭鬼者后果是杨间对李阳的解释，不构成堵门鬼完整规律的证实。（第628章、第629章）

## 未明事项

- 堵门鬼的本体、来历、原宿主身份及是否属于覆灭的国外驭鬼者队伍均不详。
- 它的准确附身条件、转移顺序、能否脱离鬼画内部及对李阳的长期影响均不详。
- 杨间离开后，房间内是否仍留有可活动的鬼、其与鬼画及楼梯上的鬼是否有关，均不详。
