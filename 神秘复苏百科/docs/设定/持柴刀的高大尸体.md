# 持柴刀的高大尸体

## 概述

“持柴刀的高大尸体”是凯撒大酒店内出现的未知厉鬼暂称。它身材高大魁梧，脸部发黑腐臭，双眼持续流出恶臭尸水，穿老旧大衣；行走动作僵硬沉重，所经地面留下尸水腐烂后形成的浅黑脚印。杨间据脚印形状确认，它就是自己此前在酒店二层见过、却未能确认身份的鬼。第351章，敲门鬼广播传入二层厨房后，它从黑暗厨房持锈蚀缺口的刀具走出；第354章，它又出现在四层异常走道十字口。它与二层高大黑影是否必然为同一个体、来源与档案代号均不详。（第351章、第354章、第358章）

## 柴刀与袭击

其手中所持物形近菜刀，杨间更倾向称为农村砍柴用的柴刀：刀身锈迹斑斑、满是缺口，似久埋土中，仍留有特殊泥土。杨间仅以棺材钉同为锈蚀铁器作联想，不能据此断定二者来源、等级或效果相同。第358章，两点十分熊文文预定死亡时，该尸体停步，将柴刀抬至约熊文文颈高的位置后对空挥砍；林落梅手中的鬼烛瞬间剧烈燃烧至不足十分之一，熊文文本人未死。第360章，熊文文预见杨间以鬼绳直接限制时会被斩首、鬼绳也会断裂；杨间据此推测柴刀可能斩断鬼绳，但尚未实际验证。（第358章、第360章）

## 已见规律与限制

尸体最初在十字口停住却未攻击，后来背向队伍似欲离开，临近走道出口又转身往回走。杨间注意到它每次发动前都会异常停步，因而推测“停下”代表已有目标符合杀人条件；熊文文预见的死亡先后为两点十分熊文文、两点十五分罗素一、其后林落梅。鬼烛挡住针对熊文文的第一次挥砍后，预见显示它会跳过该目标、转向罗素一，说明袭击失败未必持续锁定同一人。杨间试验不正面看鬼仍不能改变罗素一死亡，故仅能排除方位是唯一条件，完整触发规则不详。（第356章、第359章、第360章）

## 第360章·地毯干预

罗素一临近两点十五分时，杨间未能以鬼域保护他，也放弃直接用鬼绳接触。因尸体正沿整条相连地毯走来，杨间以鬼影强化后的力量猛扯地毯，原意是将鬼绊倒、令其无法继续行走。尸体没有倒下，仍踩着地毯前进，但它没有如预见般停步挥刀，罗素一因此未死。该次只能确认地毯被扯动时改变了当次死亡结果；是否因脚下环境变化、行走状态被干扰或另有未见条件，均不能确认。（第360章）

## 出处

第351章、第354章、第356章至第360章。

## 第383章·个人档案代号

杨间回到大昌市后，以录音笔和未联网电脑整理个人灵异档案，将凯撒大酒店内“拿刀的高大男尸”记录为“砍人鬼”，并按个人判断列为危险程度A。该代号和评级是杨间的个人档案用语，不等同于总部正式命名或等级认定；原文也没有新增其杀人规律、来源或限制状态的信息。（第383章）
