# 篡改记忆鬼

## 概述

篡改记忆鬼是中山市事件中夺取他人脸孔、身份与记忆的灵异源头。杨间在凯撒大酒店三十一号房取得源头鬼报纸后，根据赵磊字迹、墙上人脸和被夺脸者的状况，推测它可先取下人脸、在报纸上写入内容，再把脸贴回以改写受害者记忆；这在当时仍是未完全验证的推断。第371章至第373章，杨间确认冒用童倩身份、住在第三中医院的对象就是该鬼，并将其与童倩后脑笑脸分离、分别收容。鬼是否有独立本体、改记忆与占据身份是否必须依赖鬼报纸，仍不详。（第363章、第371章至第373章）

## 已见表现

- **篡改记忆与植入指令**：冯全被鬼报纸取脸后，纸面显出赵磊留下的“忘记自己被袭击”“找到合适机会杀了杨间”等内容。杨间据此确认冯全曾受篡改，亦认为纸上文字更像在记忆中植入恶念而不是任意重写全部人格；这一边界未获完整验证。（第367章、第368章）
- **夺取身份**：杨间确认医院中的“童倩”是鬼，认为该鬼既已夺走童倩鬼脸，也能夺走童倩身份。目标以童倩的活体身体、病服和外貌出现，却在脸上发出笑声；真正童倩的人脸则被杨间保留在别墅内。鬼如何进入、何时夺取身体及是否能同时维持其他身份均不详。（第371章、第373章、第374章）
- **笑声远程杀人**：总部联络员拨通第三中医院电话后，电话无人应答却传出怪笑。联络员听见后身体僵硬、面露笑容并倒地死亡，笑声还在通讯室内逐渐清晰；赵建国据此下令撤离。笑声经电话传播的距离、持续时间、是否必须接听及完整防护方式均不详。（第371章）
- **无脸状态仍可行动**：杨间从冒用童倩身体的脸上取走笑脸后，无头鬼影仍未被排出，表明失去笑脸的身体仍具活动能力。他将该身体装入尸袋，推测这只鬼也许没有固定实体，谁继承其灵异谁就是鬼；这只是杨间针对本次收容作出的推测。（第373章）

## 限制与收容

第三中医院顶楼病房内，杨间先以第二层鬼域争取时间，再与冯全、罗素一同时发动三种灵异：无头鬼影侵入身体、坟土手臂掐住颈部并捂嘴、染血细线自脚下缠住全身。三重压制令笑声中断、目标失去行动能力。杨间尝试以鬼报纸取下笑脸失败，意识到报纸与这只鬼属于同一整体，不能以其一部分夺取其自身脸孔；他遂用鬼影强行取脸，放入黄金盒，将无脸身体交冯全装入尸袋，等待总部接收。（第373章）

收容并未解决中山市全部风险。杨间仍未查明人头气球源头，凯撒大酒店内的郭凡、柴刀尸体及其他异常也没有处理；被写入记忆者的数量、能否自行恢复、尸袋与黄金盒能维持多久均不详。（第373章）

## 与童倩的关联

杨间在第374章确认真正童倩已经死亡，其身体被该鬼夺走，只剩原本的人脸。鬼报纸贴住童倩脸后，纸上浮出“杀死杨间”，显示鬼还曾在其记忆中留下针对杨间的命令。杨间擦去该字，写入新的记忆内容，再将童倩脸用于赵磊身体的拼接；这属于杨间对既有灵异物的使用，不代表篡改记忆鬼本身已被彻底理解或消灭。（第374章、第375章）
