# 鬼门

## 概述

鬼门是总部地下特殊收容区内的一扇老旧木门，门上平日缠有锁链，门后在现实中本为空无一物，却会传来震动和刮木声。第501章，杨间在物品库中已发现这件被称为“极度危险”的物品；第528章，曹延华、王小明和秦老为救援失联的李军行动队，进入存放处准备启用它。总部将其称作“代号鬼门”，但其来源、最初事件、关押方式和完整规则均未公开。（第501章、第528章）

## 启用目的与限制

第528章，行动队在鬼画、鬼差事件中失联，外部虽出现五名疑似已成鬼的同形者，曹延华仍不能确认那就是李军等人本人。他与王小明安排秦老在约定节点开门，意图把李军、苏凡、柳三等人从未知位置带回。第529章补充，救援最多只有三次开门机会：首次后十分钟再开第二次，第三次在第二次后半小时进行；若三次均无人归来，救援将停止。该次数安排是本次救援方案，并未说明是否为鬼门的固定规则。（第528章、第529章）

连续开门会加重危险。首次开门计划最多维持一分钟；曹延华随后说明，第二次仅能开三十秒，第三次仅十秒，因为首次打开后门后的鬼似乎已被吸引到门边。长期关闭可能重置可开启时间，但需关闭多久无人知晓，且原文未写有专人完成过该研究。（第529章）

## 第529章·首次开启

秦老在曹延华指令下拉开无把手的门板。门只开一道缝便被后方力量撞开，随即显出浓墨般黑暗，强光亦不能照透；现实与黑暗原由门板隔开，开门后分界被打破。十余秒内，房内灯光迅速变暗、闪烁，显示门后影响正在侵蚀环境；但该收容房间有专门隔离，外部暂未受影响。（第529章）

门后先出现青黑、僵硬的手指扣住门框，数量随后增至密布边缘；又有浓发遮脸的腐烂人头探出，面貌在头发垂落时越来越像秦老，同时另一侧出现一双苍白脚底板，似有尸体被推向门外。秦老未确认这些目标身份或规律。曹延华判断首次开启没有接回行动队，时间到后命令关闭；秦老以拐杖敲地，手指、人头和脚底板均退回黑暗，木门随即重重合拢，门框仍留有抓痕。（第529章）

## 风险与现状

鬼门打开后可能放出鬼而非救援对象。秦老称自己对它没有把握，因无人能预知门后走出的是什么；王小明则在首次开启时观察黑暗变化。曹延华为保存失联队员的救援机会，仍按时限维持开门，而非在异象初现时立刻终止，表明此物既有救援价值也有直接扩散风险。第529章末木门已关闭，计划仍保留两次更短的开启机会；第二、三次是否执行、门后对象与李军等人是否相关均不详。（第529章）

## 第534章·第二次开启前的准备

首次关闭八分钟后，曹延华要求秦老为第二次开门准备，并将其视作决定行动队生死的关键窗口。王小明此时承认另有撤退后手，称若第二次失败才会令阿红带队离开；这不改变鬼门的既定时限，只说明总部未把全部希望放在门上。第534章结束时秦老尚未实际拉开木门，第二次门后情形、开门是否接应成功均不详。（第534章）

## 出处
