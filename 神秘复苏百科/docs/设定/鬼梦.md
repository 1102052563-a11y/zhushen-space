# 鬼梦

## 概述

鬼梦是杨间对梅山村梦境事件源头及其遗留异常所用的称呼。它能把活人拖入与现实重叠的旧村梦境，并通过梦中鬼仿冒熟人、造成致命袭击；其完整规律、现实与梦境伤害的对应关系和原始来源均未确认。（原文回跳第699章至原文回跳第713章）

杨间在梅山村结束当夜梦境后，从老家木棺中带走一具腐烂鬼尸。尸体曾长出黑发、肉身腐败，鬼童拒绝攻击，杨间据此将其视为高危灵异目标，移入尚通大厦安全屋看守。它与鬼梦是否为同一厉鬼、拼图或仅存在关联，原文未作定论。（原文回跳第713章、原文回跳第733章）

## 梦境表现

梦境会以熟悉环境和熟人声音诱导目标。江艳听到“杨间”呼唤后被拖入旧村，随后遭另一名外貌相同的鬼袭击；杨间曾用钢筋刺伤江艳以辨别活人与仿冒者。梦中鬼被杀后梦境结束，但梦中伤害是否必然反映到现实、死亡者的完整死因仍有待核实。（原文回跳第700章、原文回跳第707章、原文回跳第708章）

## 鬼怪孕育线索

杨间把带回的尸体称为正在孕育的“鬼怪”，意指既非人类驭鬼者、也非普通厉鬼的未知异类。他称该过程已完成第一阶段、正处于第二阶段，并认为结果可能可控，但控制方法和最终成品均不详；其他驭鬼者若擅自触碰可能遭鬼梦杀害。（原文回跳第777章）

鬼镜在照向木棺时，镜面呈现出一条沉睡、浑身黑毛的狼狗，而非现实中可见的腐烂尸体。杨间据此联想到父亲遗留的记忆可能意在让一条狗驾驭鬼梦；若对抗成功，狗可能苏醒、成为看不见的异常存在。该狗是否真实存在于棺中、是否已驾驭鬼梦，以及杨父的记忆具体内容均未证实。（原文回跳第779章）

## 风险与现状

杨间未打开木棺，只把它置于安全屋中看守。鬼镜显现黑毛狗时，目标仍处沉睡状态；杨间要求队友不要继续以鬼镜照向棺材，等待其自然孕育结果。鬼怪能否受控、会否释放鬼梦、与鬼童是否相克及其苏醒时机均不详。（原文回跳第777章、原文回跳第779章）
