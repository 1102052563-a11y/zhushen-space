# 楼梯鬼

## 概述

楼梯鬼是杨间在鬼画内部一栋国外居民楼木质楼梯间遇到的未知厉鬼的暂用称呼。它没有在原文中得到正式代号；本页按其活动地点命名，不能据此认定其本名。杨间、李阳与吉米进入大楼后，只要有人踩响木质台阶，楼上便传来对应的嘎吱声，仿佛有东西沿楼梯下行。杨间由此确认楼上不是活人，而有一只会随楼梯脚步活动的鬼。（第625章）

## 已见规律

- **脚步声会引动活动**：杨间在二楼前踩响一节木质台阶，楼上立即传来同样的嘎吱声；他停下或退至二楼水泥地面、声音消失后，楼上的脚步也停止。李阳、吉米踩木梯上二楼时，楼上脚步再次向下传来。现有记录只表明木梯脚步声与鬼的活动有关，是否必须由活人发出、轻重差异是否影响目标、无声通过能否永久规避均不详。（第625章、第626章）
- **会沿楼梯接近，位置难以直接看见**：杨间以楼层与声音判断，鬼起初大约在四、五楼之间；他后来从三楼走向四楼时，判断鬼正由五楼往下，在相邻楼梯上同向接近。鬼实际出现前，杨间始终没有看见实体，只能以声音估算位置，因此其外貌、是否有固定形体均不详。（第625章、第626章、第629章）
- **近乎必死的袭击**：杨间抵达四楼后，明明未看见鬼或异常，却突然眩晕、意识消失并倒在楼梯口；阴冷感与沉重脚步随后在身边徘徊。八音盒铃声将他的意识拉回，脚步才逐渐远去。杨间将这视为一次必杀袭击，认为若没有八音盒诅咒自己必死；该结论仅适用于本次接触，完整杀人条件和能否抵抗均不详。（第629章）

## 行动记录

杨间原为寻找鬼画内的红衣鬼，接受吉米所见地点的指引进入这栋七层居民楼。发现木梯异响后，他不让李阳、吉米继续在原处停留，而是先带二人到二楼水泥地面；三人搜索二楼无果，遂经不会发出木梯声的室外铁楼梯进入三楼。此举暂时没有引动楼梯鬼，但三楼窗户和外楼梯随后被鬼画改变为墙面，三人只能留在鬼画可能所在的楼层。（第625章、第626章）

三楼走道集中有十余具不同腐烂程度的尸体，杨间据此推测此处可能是国外驭鬼者队伍覆灭地点；他又认为木梯鬼可能是其中一名驭鬼者死亡后遗留的厉鬼。这是对尸体、楼梯鬼和国外队伍之间关系的推断，并未获得直接证据。杨间随后经由三楼房间事件确认楼梯仍是唯一上四楼的通路，便让李阳、吉米留在三楼，独自踏上木梯试探。（第626章、第627章、第629章）

他在四楼被袭后依靠八音盒诅咒恢复意识，暂未再听见脚步声；鬼是已经离开，还是因杨间一度如同死亡而不再把他视为目标，原文未确认。李阳与吉米当时仍留在三楼，是否听见袭击、何时能安全上楼也未写明。（第629章）

## 与鬼画的关系

楼梯所在大楼位于鬼画世界，杨间发现三楼大量尸体、出口消失及布局改变后，判断鬼画可能就在附近；他也认为楼梯未被鬼画抹去，是因其本身有鬼，从而对鬼画影响存在一定抵抗。该解释是杨间的现场分析，不能确定楼梯鬼由鬼画生成、被鬼画困住，或本就与鬼画源头有直接从属关系。（第626章、第629章）

## 未明事项

- 楼梯鬼的真实身份、来源、形体、原宿主及与三楼尸体的对应关系均不详。
- 其完整触发与杀人规律、是否会锁定多名踩楼梯者、能否借其他声音行动均不详。
- 八音盒诅咒为何能使杨间从本次袭击中恢复，以及该鬼之后的行踪均不详。

## 第630-634章·画中定位、阻拦撤离与疑似拼图变化

杨间进入四楼画室后，看见一幅画内的老旧楼梯与现实木楼梯一致，楼梯上还显出一个模糊阴影。他借画面判断楼梯鬼当时在由四楼往三楼的第三阶台阶附近，与自己此前遭袭的位置相差无几。该画室画作会依现实更新的判断来自杨间对多幅画的比对，故此定位只能说明当时画面反映的位置，不能确定画面对楼梯鬼的显示是否始终准确。（第630章）

杨间带李阳、吉米下撤时，三人的急促脚步重新引来楼梯鬼。它在四楼至三楼的木梯上快速追近，杨间认为一旦被脚步追上便会立刻死亡，自己也无法抵挡。李阳和吉米因前方红衣鬼而停滞，杨间以无头鬼影将二人从红衣鬼身侧和头顶带过；楼梯鬼此时已接近红衣鬼约三个台阶，二鬼在狭窄楼梯上相遇。楼梯鬼的脚步与红衣鬼的动作均停下，形成暂时僵持，鬼影也因不明灵异干扰而失控，二人摔下台阶后未受致命伤。（第631章至第633章）

杨间撤出楼外后，看见红衣鬼原本模糊的身形、双脚逐渐变得清晰，便推测楼梯鬼的脚步可能成为红衣鬼的拼图之一。此后追向人群的红衣鬼脚下确实传出与木梯上相同、却和其缓慢步伐不协调的急促脚步声；杨间因此判断目标一旦被接近，或会同时遭受脚步与鬼画两次袭击。该“拼图”推测、两种袭击是否必定同时发生、楼梯鬼是否已彻底融入红衣鬼及其后续状态，均未获原文直接证实。（第633章、第634章）
