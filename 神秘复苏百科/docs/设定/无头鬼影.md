# 无头鬼影

## 概述

无头鬼影是杨间驾驭的一只黑色影子状厉鬼。第489章，围观驭鬼者看见高志强被黑影覆盖、肢解，影子不像属于高志强或杨间，而像多出的第三个存在；杨间能操控它。围观者据此认定它是杨间未记入常规档案的第二只鬼，但其来历、完整形态、杨间最初驾驭经过和常态外观，本批不作补充。（第489章、第490章）

## 已见表现

杨间以两层鬼眼鬼域找到被鬼骗人隐藏的高志强后，令黑影沿地面侵入目标身体。高志强随即全身僵住，双脚、双手和头颅相继脱落，却仍保持意识；围观者因此认为直接触碰该影子对驭鬼者也很危险。第490章，杨间说明高志强能靠头颅存活是鬼影维持的结果，撤回鬼影后，对方将因失去厉鬼力量而死亡。该次肢解的触发距离、能否作用于其他厉鬼和维持时限不详。（第489章、第490章）

高志强以鬼骗人令杨间身体逐渐消失时，杨间的鬼影形成黑色躯体，阻止其身体完全消失。杨间解释，鬼影本能是占据一具身体，不会轻易放弃自身这具仍可占据的躯体；此机制令鬼骗人不能彻底抹去杨间。该表现只确认本次对抗中鬼影能与杨间身体争夺或维持躯体，不能确认其对其他消失、夺身或幻觉类灵异都有效。（第491章）

## 驾驭与限制

第491章，杨间以鬼手压住已取代高志强的鬼，再令无头鬼影从自身黑色躯体反向侵入该鬼，意图强行驾驭并取回被影响的身体。第492章，鬼影完全覆盖目标后，杨间成功借鬼骗人力量恢复消失的身体，继而让自己的身体变得更健康；这一结果依赖鬼影对目标的强行覆盖和限制。第493章，杨间发现该鬼会作为意识进入自身、破坏与鬼眼和鬼手的平衡，故不能长期驾驭。鬼影能否稳定驾驭其他鬼、是否有复苏代价及它与杨间现有平衡的完整关系不详。（第491章至第493章）

## 第557章·破开鬼域阻断的楼层

女生宿舍内的鬼画鬼域抹去六层至下层的楼梯后，杨间没有让王泉小队使用登山绳冒险外降，而令无头鬼影从身后渗入地面。影子覆盖处的水泥和钢筋被拆开，扭曲成向下延伸的黑色台阶，队伍得以从错位楼层返回真实四层。（第557章）

王泉踩上影子时感觉阴冷活物似要侵入身体、令双脚失控；杨间说明这是鬼影本能替换身体的迹象，但在其身体未腐烂、鬼影仍受控制时不会失控。本次仅显示鬼影可破坏并重组建筑结构、在杨间控制下暂供他人通行，不能确认其承载人数、作用范围或普通人长期接触的后果。（第557章）

## 第579-584章·接管无头尸体与尸体拼接

杨间被鬼剪刀断头后，无头鬼影接管其无头尸体，初时只能依本能走向人群寻找新鲜头颅，不能控制鬼眼或鬼手。尸体仍戴着杨间此前取得的黄金手套，手套隔绝了鬼影拆取头颅的能力，致使它每次袭击只能把受害者头部捏碎；同时它逐步掌握鬼眼形成的四层鬼域，以错位画面藏身并连续杀人。（第579章、第582章、第583章）

黄子雅以黑发缠住尸体双臂、把杨间头颅放回后，鬼影先自行拼接上半头颅与脖颈伤口，却无法直接拼回承载意识的另一半。杨间随后以扭曲鬼域令半头与身体重叠，才夺回有限控制；鬼影的拼接本能仍参与了头身连接。鬼影此后是否仍受八音盒状态压制、能否再夺取杨间身体及黄金手套能限制其多久均不详。（第583章、第584章）

## 原文回跳第682-683章·梦境中的无头尸体投影

杨间在梅山村夜间灵异梦境中失去对鬼影的感知和使用，荒野却出现大量无头尸体。尸体原先静立，靠近杨间三人后转身、开始活动；三人不背对它们时，尸体又停止并退回。杨间认为这一现象像是无头鬼影相关规则在梦境中的投影，并非确认鬼影本体已进入梦境。（原文回跳第683章）

梦境如何提取杨间既有灵异经历、无头尸体和鬼影真实能力是否存在对应关系均不详。杨间本身在梦中没有鬼影、鬼眼和鬼手，不能把梦中规避方式直接当作现实中无头鬼影的完整规律。（原文回跳第682章、原文回跳第683章）
