# 福寿园诡异坟场

## 概述

福寿园诡异坟场是红色信件出现后，在大海市福寿园内显现并逐步覆盖现实墓园的未知鬼域。最初刘老太尸体失踪，杨间收回三层鬼域后，现代墓园没有恢复，天空转为昏暗、城市建筑消失，柏油路化作泥土路，现代墓地混入大量年代久远的老坟。（原文回跳第750章、原文回跳第751章）

## 鬼域特征

杨间的鬼域可覆盖福寿园地面，却无法渗入一些墓地地下；这些缺口散布各处，说明异常关键不在整个园区地表，而在特定坟墓。坟场扩张时，老坟数量增加、部分坟碑使用“廿一”“卌五”等旧式数字，显示至少有部分坟墓具民国或更久远的年代特征。（原文回跳第751章、原文回跳第752章）

有些老坟无碑，有些碑后坟丘已经从内部裂开、空无棺木或尸体；一座被孙瑞手杖戳过的坟丘则渗血并逐渐塌陷。以上现象能证明坟场内的坟墓并非普通墓园陈设，却不能确认每座均埋有厉鬼。（原文回跳第752章）

## 与红信的关系

杨间推测红信可能不是单纯任务物，而是引子、诱饵或坐标：鬼域内的鬼感知信件后，从另一灵异空间向福寿园渗透。这个假设可解释信件为何准确指向事件地点，却无法解释鬼邮局如何选择送信人、为何指定78号墓主，以及信件最终是否能终止或加剧坟场。（原文回跳第751章）

## 当前风险

坟场内已有普通葬礼宾客死亡，尸体被埋入新坟，只露出手臂；刻碑鬼又将罗素一拖入未知空间。杨间认为若不能完成送信或离开，众人将持续面临鬼域内厉鬼袭击。鬼域能否被五层鬼眼鬼域强行脱离、源头鬼位置和坟场最终范围均不详。（原文回跳第752章、原文回跳第753章、原文回跳第754章）

杨间以柴刀在空白墓碑刻下“刘兰”，反用刻碑诅咒将刘老太埋入，刻字杀人声暂时停止。他又确认坟土必须接触肢体才能继续吞没目标，因此可通过离地、截去已陷入部分暂时救出被刻名者；该方法不保证能保住完整身体。（原文回跳第760章、原文回跳第761章）

红信送入刘老太手中、杨间取走其疑似棺材钉后，尸体吐出大量粘稠黑血。黑血能令无头鬼影退避，尸体受钉后仍爆裂；随后多座老坟塌陷、坟头出现覆满坟土的人影。杨间推测坟场中原本沉睡的多只厉鬼可能因某种平衡被破坏而复苏，具体触发点究竟是红信、取钉、尸体黑血还是其他机制，均未确认。（原文回跳第763章、原文回跳第764章）
