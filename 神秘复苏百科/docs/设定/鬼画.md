# 鬼画

## 概述

鬼画是冯全在总部鬼差会议中提到的一件国外S级灵异事件的国内称呼；国外档案译名为“艺术馆事件”。王小明计划寻找一只能以鬼域压制鬼差的对象时，意图引来的正是此事件源头。原文只交代与会者讨论时掌握的简略档案，鬼画的真实形态、出现地点、现状和完整规律均不详。（第498章）

## 已知经过

冯全称，国外一只鬼曾从画中走出并无差别杀人；死者似乎大多参观过那家艺术馆，但“看过画会被盯上”只是当时的推测，不能据此视为已验证的杀人条件。由于参观者数量很多、死亡又较分散，准确死伤数无法统计，只能确定死亡规模远超过大昌市事件。原文未说明该艺术馆所在国家、发生时间、死者身份或国外处置过程。（第498章）

## 鬼域与限制信息

王小明计划以鬼画的鬼域去压制鬼差鬼域，而不是让驭鬼者进入鬼差内部关押。该设想建立在鬼域彼此存在压制关系、鬼棺能把大范围鬼域缩入其中的分析上；若鬼画压制成功，鬼差活动范围或可缩小至能以容器关押。王小明同时承认这只是唯一可讨论的方向，尚未找到可保证成功的操作方式。（第498章）

冯全还转述，国外曾以一支驭鬼者队伍近乎团灭为代价，把黄金打造的烛台留在鬼画源头身上。按其理解，黄金存在时该鬼应无法再进入画中杀人；这不是现场验证，且冯全也提醒国外档案可能隐瞒关键资料。黄金烛台是否仍在、实际限制效果、鬼画是否还能经其他媒介行动，均不详。（第498章）

## 与鬼差处置的关系

王小明提出让鬼画“顺路”经过鬼差所在区域：成功则以鬼域压住鬼差，失败则可能由鬼差压制或解决鬼画，等于两件S级事件碰撞后至少消除其中一件。姜尚白认为这比取棺材钉更疯狂；王小明则以鬼画尚未被关押、饿死鬼一旦拔钉就可能失控为由坚持比较。该方案截至第498章仍只是会议讨论，未见引鬼、交战或任何处置结果。（第498章）

## 第516-518章·引至大京市与灰白鬼域

总部当晚执行引鬼计划，柳三以飞机将鬼画引向大京市市郊鬼差所在山谷；飞机在鬼域内无法久飞，最后坠在附近树林。鬼画抵达时，天空持续飘落灰白纸灰，路灯、道路和所有可见物都失去色彩，形成死寂的灰白世界。李军等行动者将此确认作鬼画鬼域，因他们附近没有画作、也未看过画，暂判断自身大概率不会成为鬼画目标；这只是队伍依据已有资料作出的判断，并未排除袭击风险。（第516章）

柳三的一个纸人进入鬼画鬼域后死亡，显示纸人这一类似厉鬼的替身也未能抵御其袭击；柳三因此认为国外档案隐瞒重要内容。第517章，鬼域内出现现实中不存在的道路、废城、老宅、坟地及人影，原山谷的位置更被黄岗村取代，行动队反复折返仍回到村口。鬼画如何制造这些景物、黄岗村与鬼差的关系、鬼画本体是否仍在坠机树林及是否已压制鬼差，均未获证实。（第516章、第517章、第518章）

## 第519-523章·黄岗村搜索与鬼域同时消散

柳三向杨间说明，行动队进入黄岗村后，正常走在灰白道路上暂未遇袭，一旦进入黑暗房屋便会遭厉鬼攻击；他推测鬼差曾夺走的鬼都在村中重现。李军临时绘图发现黑色房屋连成棺材形，灰白道路由持续飘落的纸灰构成，向房屋侵蚀时又被黑暗吞没，显示鬼画与鬼差形成暂时平衡。平衡随时可能因触发鬼画杀人规律而破裂，形成机制及其是否为鬼画主动控制均不详。（第519章）

第520至522章，行动队在一栋民房内发现柳三五具残破纸人和一张将五人收入背景的老人遗像；后者被黄金袋收容后，门外出现与遗像构图对应的老人背影，鬼烛骤燃，队伍不久失联。第523章，灰白鬼画鬼域与棺形鬼差鬼域一同迅速消失，中心走出五个具有相同手掌空洞的人影；杨间以白色鬼烛确认五者均呈鬼的反应。两片鬼域为何同时结束、鬼画是否被鬼差解决、五个鬼与原行动队的关系及鬼画本体去向，均未写明。（第520章至第523章）
