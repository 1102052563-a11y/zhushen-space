# 灵异公交车

## 概述

灵异公交车是杨间为追踪许峰卫星定位手机而误上的异常车辆。第301章，车辆在傍晚六点二十分后离开小春市区，路线已经被更改、长时间不停站；杨间试图开窗跳车，发现车窗坚硬且无法开启，掌心鬼眼放出的红光鬼域也被车厢阻隔，车厢内外如同两个世界。他因此判断失踪的许峰很可能已经栽在车上，而非仅仅遗失手机。（第301章）

## 已知表现

第302章，驾驶位上的司机被乘客推动后露出已死亡至少一天、开始腐烂的尸体；仪表盘熄灭、车辆未发动，公交仍以约五十码行驶，方向盘自行转动，刹车和方向盘均不能由乘客控制。杨间据此推测可能是看不见的鬼驾驶、整辆车本身是鬼，或鬼伪装成乘客；这三种解释均只是其当时推测，尚未获证实。（第302章）

车内黑色电子显示屏在未通电状态下显示“当前乘客数量：2”。第303至304章，许峰说明该数字记录的是厉鬼数量，而非活人；晚六点后车辆每经过一站，可能有一只或两只厉鬼上车。停车时数字由二降为一，哭坟鬼上车后又回到二，与杨间的观察相符，但车内另一只鬼的身份未明。车辆每次停靠约五分钟，时间到后会无视车内人员和无人驾驶状态继续行驶。（第303章至第305章）

## 停站与风险

第304章，公交在荒野电线杆旁的临时站牌停靠，车门开启后车外并非正常道路环境。许峰警告杨间，前门下车会进入不明鬼地，自己已困在车上七八日，未等到合适离车时机。多名普通乘客下车后，坟前哭声令其跪地哭泣、死亡；这证明下车并不等于脱险。第305章，哭坟鬼自行登车后不再哭泣，杨间推测它可能只能在坟前哭，或车辆能压制上车厉鬼，二者均不详。（第304章、第305章）

第306章，公交又先后停在有红灯笼荒宅与无人的十字路口：干尸新娘从荒宅登车，后一次停车虽无人可见，显示屏鬼数仍由三变四，证明至少有一只不可见的鬼上车。第307章，车辆没有进站便急刹停在路中，三只先前上车的鬼从后门依次离开；随即车灯、过道灯由前向后熄灭。许峰和鸭舌帽男子称这属于少见的“熄火”突发状况，车内人和鬼必须离开，强留车内会遭遇近乎必死的危险；同时，平时无法开启的后门和车窗会解除限制。杨间只在当时借开门后的鬼域离开，公交熄火的原因、再次发动时间及上述规律是否存在例外均不详。（第306章、第307章、第308章）

## 座位与乘客

第306章，杨间从许峰所说的“满载”得知车内共有三十六个座位，并推断满载的危险不在活人增多，而在活人死去、厉鬼持续留下后占满座位。许峰没有说明满载会发生什么，只提醒应当祈祷车辆不要满载。第307章，许峰又暴露自己希望车上保留驭鬼者占座的意图，但座位如何影响公交规则、普通人是否也能占座、驭鬼者死后会否计入鬼数均未说明。（第306章、第307章、第308章）

## 压制与离车

第312至313章，熄火的公交重新鸣笛、亮灯发动后，车外的厉鬼不再直接构成车内威胁；五分钟后前后门和林北打开的车窗自行闭锁，车辆再次行驶。第315章，林北与许峰说明二人长期留车是为了利用公交对厉鬼的强力压制：即使体内厉鬼已经复苏，鬼在车内也难以杀死驾驭者。二人称停留越久，复苏鬼离车后可维持的时间越长；林北六天内曾下车两次，离车存活时间由约三分钟延至约十分钟。这是两人的经验说法，压制机制、时间增长上限与是否会最终使厉鬼沉睡均未验证。（第312章、第313章、第315章）

公交并非绝对安全。林北列举熄火、满载和鬼拦车为曾遭遇的大危机，并说前一名能驾驭复苏鬼的幸存者仍因自保不足而死；另有一名据说在车上活了两个半月后下车，最终状况不详。第314章，重启后的显示屏仍为“1”，证明至少一只未知鬼留在车厢；车辆随后穿过无边坟场却未停靠，最终驶向有正常灯火的城市。杨间据此将正常城市停站视为离车机会，而不是将车内压制视作长期选择。（第314章、第315章）

第316章早晨七点半，车辆在一座正常城市的站台开门，杨间成功下车；他转身警告普通乘客勿上车时，灵异公交已在近处消失，原位置变成另一辆外观相近的普通公交。车内的许峰、林北及许峰手机都随之失去直接联系。杨间认为这解释了长期被困者不敢随意离车：车辆不能保证再次找到或重新登上。第317章，他以许峰卫星定位手机仍在车上的事实，请总部持续追踪信号；其能否精确定位、车辆消失机制及再次出现的条件均不详。（第316章、第317章）

## 档案

第317章，杨间要求总部建立代号“鬼车”的X级档案，并说明X级代表无法定义危害程度、却极其重要的神秘未知事件，而非通常C至S级分类。他报告首次遭遇中四鬼上车、熄火时人鬼均须离车、约十分钟后车辆重启及自身在早晨离车的事实。第318章赵建国告知该档案已被秘密封存移交，王小明对车辆感兴趣；这只记录总部已接收资料，不表示车辆已被控制或其机制已解明。（第317章、第318章）

## 出处

第301章至第318章。

## 第612-614章·被列作最后的诅咒生路

杨间在八音盒诅咒逼近时仍考虑重新登上灵异公交车，因车内强力压制体内鬼的经验或可同时处理其厉鬼复苏问题。（第612章）王小明则警告，杨间的意识当前由八音盒诅咒维持，若公交车把身体里的鬼和诅咒一并压住，意识可能在瞬间消失；这是王小明基于研究给出的判断，未在杨间身上验证。（第612章）

第614章，杨间把公交车方案排在鬼橱交易和人皮纸信息之后，理由是自己也认为上车即死的可能很大。公交车现处位置、是否能再次寻到、是否真能压制八音盒诅咒及意识消失后能否恢复，本批均不详。（第614章）
