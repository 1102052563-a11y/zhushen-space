# 干尸新娘

## 概述

干尸新娘是灵异公交车在第306章停靠红灯笼荒宅时登车的厉鬼。它穿红色旗袍、头盖红色头巾，露出的双手干枯僵硬，暗褐色尸皮紧贴骨头；杨间由鬼眼预警判断其威胁高于哭坟鬼，但没有正式代号，故以外观暂称“干尸新娘”。它上车后没有坐近车门，而是在杨间前排落座，显示屏鬼数从二变三。其来历、是否与荒宅有关及完整规律均不详。（第306章）

## 张浩与招鬼

公交熄火后，干尸新娘没有远离，站在道路对面。第310至311章，它隔着杨间两层鬼域令张浩僵硬走出鬼域，靠近自己；张浩先在哭坟鬼哭声中幸存，杨间怀疑是这只鬼的力量使其暂避死亡。干尸新娘伸手牵住张浩后，张浩立即失去生命气息，却没有倒下，而是牵着它的手以悲伤哭容挤出笑意、不断招手。杨间随后确认张浩招向的是哭坟鬼，哭坟鬼便从树林走出；公交内的鬼也开始拍打车身，显示张浩的异常动作会引来周边厉鬼。张浩如何被选中、牵手是否为必要条件及“招鬼”能影响哪些厉鬼均不详。（第310章、第311章）

## 接触风险与短暂压制

杨间不敢直接攻击干尸新娘，担心自己或无头鬼影被它牵手后遭遇同样结果，遂以处于躺尸状态的许峰身体撞向目标。第311章，许峰压住干尸新娘后，张浩松手倒地，招鬼动作暂停；但第312章显示，这种压制不足三分钟，许峰半边腐烂身体逐渐干枯、仿佛被同化，干尸新娘重新站起并再次牵住张浩。许峰尚未死亡，却已受其影响，说明躺尸状态只能暂时抵抗，不能安全接触。杨间据本次表现判断这只鬼不可接触、不可被盯上，但此为他的风险判断，未能推出完整杀人规则。（第311章、第312章）

## 停留与去向

公交重新发动后，干尸新娘仍牵着张浩尸体站在路边，没有追上车辆或再次登车。杨间、林北和许峰因此暂时回到车内；车行五分钟后门窗重新锁死，目标仍留在远处，像两尊没有动作的雕像。该鬼为何没有上车、张浩尸体是否持续受其操控及它会否再次出现，均未说明。（第313章、第314章）

## 出处

第306章、第310章至第314章。

## 第559章·红衣外观的待核对线索

杨间首次看见鬼画中走出的女子时，注意到其模糊的红色衣物与自己在灵异公交车上见过的干尸新娘衣着相似。鬼画女子的面容、头发和双腿均不清楚，干尸新娘则曾以红色旗袍、红盖头和干枯双手出现；两者外观细节并不相同，杨间没有确认是同一件衣物或同一只鬼。（第559章）

该相似性仅促使杨间把鬼画、鬼橱照片、公交荒宅和干尸新娘列为可能相关的线索，原文未揭示它们的关系，不能据此改变干尸新娘的已知去向或能力判断。（第559章）
