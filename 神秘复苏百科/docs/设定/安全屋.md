# 安全屋

## 概述

安全屋是以黄金为核心隔绝材料建造的避难空间，用于在灵异事件中保护普通人。杨间认为普通人若拥有食物、水和合格安全屋，可在多数灵异灾害中提高生存机会；黄金能隔绝鬼域等灵异影响，因而比现金、普通建筑材料更具战略价值。（第256章至第258章、原文回跳第729章）

## 观江小区的早期安全屋

饿死鬼事件扩大时，张显贵组织在观江小区赶建地下安全屋，主体虽完成，配套不足，只能让人员短期维持。杨间为总部、王小明、江艳和张显贵家属分配有限席位，表明安全屋并非能无条件容纳所有人；其面积、黄金用量和完整防护范围均未写明。（第256章至第258章）

饿死鬼事件后，张显贵继续推进小区安全屋建设，外界愿意高价购买附近房屋的原因之一，正是杨间居住与安全屋带来的保护预期。安全屋是否能抵挡所有等级厉鬼、是否仍需驭鬼者看守、普通人能否自行维护均不详。（第277章）

## 黄金限制与成本

杨间在私人聚会中说明，各国已开始管制黄金，普通资金很难再购买到足够材料；即使十亿元，现阶段也未必能建立一处完整安全屋。黄金价格和获取难度上升，意味着现金购买力在灵异时代下降，安全屋建设须提前筹备而非等到灾难爆发后再行动。（原文回跳第729章）

黄金虽能隔绝灵异，却不是万能庇护。安全屋仍需要位置、物资、人员和外部供应链支撑；若建筑周围长期被封锁、食物耗尽，或内部本就存在灵异物，黄金隔绝也不能自动解决问题。上述限制部分来自杨间和王彬在公司规划中的考虑，详细工程标准不详。（第386章至第390章、原文回跳第729章）

## 公司计划中的定位

杨间公司计划将安全屋与农业、仓储、超市、商场和能源项目一起布局，目标是让大昌市在灵异频发、物流和金融动荡时仍能维持基本自给。王彬强调养殖、粮食基地等设施需分散建设：一处若被灵异事件毁掉，其他地点仍可供应。这使安全屋不只是富人躲避设施，而是城市生存网络的一环。（第386章至第390章、原文回跳第729章）
