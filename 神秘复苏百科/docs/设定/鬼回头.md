# 鬼回头

## 概述

鬼回头是培训基地受训驭鬼者赵昊所驾驭厉鬼的能力名称。赵昊与张雷等人为同批培训人员，但在鬼差事件中已经死亡；张雷在第462章的正面接触中认出，鬼差夺走赵昊体内的鬼后，已把这项能力拼入自身。赵昊的外貌、经历、死亡过程、原本如何驾驭此鬼及该鬼是否有独立形态，原文均未写明。（重复第462章·食鬼之人）

## 已知表现

鬼差以卫景尸体为载体时，身体背对杨间；杨间将鬼手和无头鬼影侵入其肩部、持续接触后，鬼差僵硬的脖颈开始向后缓慢转动，并发出骨节断裂般的声响。与此同时，杨间的脖子和整条脊椎被无形力量牵动，跟着发生无法抗拒的扭曲，身体近乎会被拧断。张雷据此称，这不是鬼差原有的单纯压制，而是赵昊遗留的鬼回头能力。（首个第462章·鬼回头、重复第462章·食鬼之人）

## 触发与结果

本次袭击发生在杨间直接接触鬼差、且无法迅速抽回手臂时。杨间推测，原本以一片区域计算的“不落单”规则在近身接触后改变，谁持续触碰鬼差，谁就会成为局部意义上的落单目标；因此鬼差回头的对象是杨间。张雷判断若鬼头完整转至背面，身后被盯住的人必死，但这只在本次现场由其说明，是否还需要位置、持续接触或其他条件不详。（首个第462章·鬼回头、重复第462章·食鬼之人）

## 对抗与限制

张雷为救杨间抱住鬼差，用胸前能吞鬼的腐烂人头咬住其半个头盖骨。鬼差若继续回头，便会把自身头部送入人头口中，回转因而被卡住；张雷则承受胸前鬼被碾压般的疼痛，并被鬼差压制至逐渐沉寂。此次对抗使鬼回头中止，也让杨间恢复行动。原文只确认张雷以此挡下这一轮袭击，不能确认腐烂人头能否真正吞掉鬼回头、再次使用是否仍有效，或鬼差换身份后会否保留该能力。（重复第462章·食鬼之人、第463章）

## 出处

首个第462章·鬼回头、重复第462章·食鬼之人、第463章。
