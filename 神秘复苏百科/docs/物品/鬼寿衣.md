# 鬼寿衣

## 概述

鬼寿衣是一件黑色、脏旧、近似死人入殓所穿的寿衣。它原由姜尚白持有，杨间在平安大厦交战中取回，之后一直封存。衣物来历、制作方式、与饿死鬼的完整关系及此前持有者均不详。（第591章至第597章）

## 已知效果与代价

活人穿上鬼寿衣后，可抵挡其他厉鬼的袭击。姜尚白曾借其挡住杨间第五层鬼域红光和鬼手侵入；但寿衣会持续侵蚀穿戴者，把活人逐步变成尸体，且活人穿上后不能自行脱下。杨间最终以鬼手三个压制名额压住姜尚白体内灵异，才取回寿衣。防护上限、侵蚀速度是否随目标而异及脱下的通用方法均不详。（第591章至第597章）

杨间据寿衣对活人的侵蚀判断，它更适合由鬼穿戴：鬼不怕被侵蚀，或可把衣物作为自身拼图。该判断在第641章得到一次当场表现支持，但“任何鬼均可完美驾驭”仍是杨间推测，不能作为已验证规则。（第641章）

## 第641章·由鬼童穿戴

杨间决定赴平安科技研究基地处理八音盒诅咒前，为加强鬼童，把封存的鬼寿衣取出、装入盒中送进关押鬼童的房间，命其穿好后出来。鬼童出现时，原本尺寸更大的寿衣已自动缩至合身大小，双方没有显出冲突；杨间据此判断鬼童可驾驭寿衣。鬼童穿衣后仍受杨间命令，可被安排携死人头潜伏。（第641章）

其后鬼童带着寿衣进入平安科技研究基地，杨间命其封锁整栋大楼出入口。第644章，刘助理等人在走道目击的正是身穿脏旧寿衣、抱着死人头的鬼童。寿衣是否因此提高鬼童防护、是否会改变其袭击规则，以及能否被杨间重新取回，均不详。（第643章、第644章）
