# 熊文文专属纸人

## 概述

熊文文专属纸人是柳三派往大昌市的纸人躯壳内藏着的小型纸人，外貌仿照被困黑白照片中的熊文文。柳三称这具纸人本身没有其意识，王小明则判断它可作为特殊载体，配合鬼童吞下的灵位和鬼相机照片，让被困照片的熊文文以另类方式重新存在。（原文回跳第722章、原文回跳第723章）

## 形态与来源

柳三当着杨间、李阳和张丽琴的面撕下自己蜡黄的纸皮，显露出层层仿真纸张；外层纸人残破后，空腔中放有一个更小、面容接近熊文文的纸人。其触感柔软冰冷、近似死人皮肤，重量却异常轻。柳三说明该躯壳是专门为熊文文准备，不是临时由普通纸张制作。（原文回跳第722章、原文回跳第723章）

## 预定用途

熊文文因鬼相机失控被困在黑白照片内，照片中同时可能纠缠着厉鬼。杨间曾通过灵位实现意识转移，王小明据此判断，只要有可承载厉鬼的特殊身体、熊文文照片和鬼童腹中的灵位，便有机会让熊文文从照片中出来。柳三主动提供纸人，称复活后的熊文文不再依附自己的意识，纸人将成为其独立存在的载体。（原文回跳第722章、原文回跳第723章）

这不是普通复活。杨间指出，若载体无法压制照片中的厉鬼，可能只会释放厉鬼、导致熊文文真正死亡；纸人是否足以承受其预知鬼、意识转移后是否保留原有能力、会否出现纸人化或失控均未验证。（原文回跳第722章、原文回跳第723章）

## 原文回跳第778-779章·成功作为熊文文载体

杨间在二号安全屋命鬼童把黑白照片贴到纸人额头，照片中熊文文的意识和与其一同封印的厉鬼随即被转入纸人。失去灵异维持的照片化为粉末；纸人先是出现眼睛、手指和呼吸般动作，后能以手势回应问题，三日后又恢复语言交流。（原文回跳第778章、原文回跳第779章）

杨间据孩子气反应暂判苏醒者是熊文文，但仍把其留在封闭房间适应纸人身体，并安排张韩看守。纸人没有正常口腔、内脏和呼吸需求，熊文文是否能长期适应、纸人能否压制预知鬼及柳三死亡是否影响载体均不详。（原文回跳第778章、原文回跳第779章）

## 当前状态

柳三纸人自行撕毁后，杨间取出熊文文专属纸人，将其放入尚通大厦办公室的临时安全屋，暂未立刻实施转移。该纸人目前由杨间保管，是否会被其他灵异影响、何时用于熊文文，均不详。（原文回跳第723章）
