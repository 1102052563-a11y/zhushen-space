# 郭凡灵位

## 概述

郭凡灵位是一块陈旧发黑的木质灵位，中间嵌有黑白相片。第382章，钟山在凯撒大酒店四楼救出郭凡时，见郭凡的卫星定位手机旁摆着这件物品，照片里是郭凡本人，而郭凡肉身僵跪在地；其声音却从遗照中传出。郭凡称自己遭厉鬼盯上时会用这种“老方法”躲避袭击。灵位的获得时间、原始持有人、是否由总部收容物转来，原文均未说明。（第382章）

## 外观与替换现象

第411章魂瓶内鬼袭击郭凡时，杨间以无头鬼影压制其体内灵异，发现郭凡腹中藏有木质灵位和遗像。郭凡先前被鬼臂抓伤、脸部皮肉残破；压制解除后，灵位照片上的脸也显出破损，郭凡本人的伤脸却随之恢复，照片中的男子甚至转头朝杨间鬼眼微笑。这说明灵位、相片中形象与郭凡身体存在可见的伤害对应或替换现象，但原文没有给出完整规则，也不能据此确认照片中的男子身份。（第411章）

第525章，郭凡在外围引鬼时将灵位从衣物内取出。其将灵位捧在手中前进后，照片起初显示半张脸皮垂落、受伤的郭凡；随着他前行，照片变成脸部完整的郭凡，路上的郭凡反而脸皮垂下。第526章又明确，照片中的郭凡会逐步面色发黑、伤痕增加，而捧灵位者的伤痕相应消失。此时原文将外面行走者称作真正的鬼，而郭凡意识被困于相片中；该称呼用于描述当次失控状态，不足以穷尽灵位全部能力。（第525章、第526章）

## 使用效果与代价

郭凡可与灵位相片中的“人”替换，原文解释更接近与一只鬼替换。替换后，行走者带有尸臭和死灰眼神，能够以鬼的身份靠近未知厉鬼；第525章外围计划正是因此由他承担持烛引鬼的高风险位置。第526章还写明，郭凡过去可借替换把部分受伤部位转给灵位中的鬼，从而避免立即死亡。灵位能否抵抗所有袭击、替换距离和次数、是否需要主动收回，均不详。（第525章、第526章）

代价是鬼也会反向替换郭凡。第526章，郭凡意识到每次使用，灵位中的鬼都更接近自己，最终可能由他被困在相片内、鬼走出灵位取代其位置；他此前在机场受鬼袭留下的脸伤也无法由灵位修复。第527章，这种失控在行动中加剧：照片内郭凡无法再联系外部，外面的鬼继续前进，陈义和钟山只能把他按作失控目标处置。郭凡是否能在此次失控后重新取得身体、灵位是否已永久改变，至第529章末未写明。（第526章、第527章）

## 归属与使用记录

灵位明确由郭凡持有并随身携带。第382章，他从凯撒大酒店撤离时亲自取走灵位，照片随即模糊为空白；第411章杨间在救援中只观察到其藏于郭凡腹内，未收缴；第525章郭凡又能从衣物中取出使用。因此截至第529章，物品仍在郭凡一方，但其当前是否仍由能自主行动的郭凡控制、总部是否知道其风险，均不详。（第382章、第411章、第525章）

## 出处

- 第382章、第411章、第525章至第527章。
