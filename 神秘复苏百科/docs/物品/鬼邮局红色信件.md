# 鬼邮局红色信件

## 概述

鬼邮局红色信件是一封摆在一楼柜台、无署名且未封口的染血色信封。它不同于信使常见的黄色信件，也不同于床头或房门出现的个人、房间任务信；王善依据柜台位置判断，这是一楼全体信使共同承担的集体任务。红色含义、邮局为何在杨间等人闯入后出现此信、任务奖励均不详。（原文回跳第743章）

## 任务内容

杨间取起红信后，柜台灰尘在阴风中显出任务：七天内将信送至“福寿园78号主人”手中。孙瑞离开邮局后查明福寿园位于大海市、为墓地名称，78号大概率指第七十八座坟，因此收信对象可能并非活人。（原文回跳第743章、原文回跳第744章）

信件没有明示失败代价或奖励，但王善、万兴和钱蓉均认为拒绝全层任务会带来死亡风险。王善曾见不接信者次日死在床上、死时仍紧抓信件；这是其个人见闻，红信是否适用同一规则、失败是否会杀死所有一楼信使均未验证。（原文回跳第743章）

## 风险与处置

孙瑞提议撕毁红信以避免进入未知墓地，杨间认为撕毁本身也可能是陷阱，决定先送信、再借过程判断鬼邮局究竟是在压制灵异还是替厉鬼办事。杨间接下信时表示由自己承担风险，但普通信使是否能因此免除集体任务仍不详。（原文回跳第743章、原文回跳第744章）

杨间、孙瑞、李阳决定乘孙瑞专机前往大海市送信。王善、万兴和钱蓉也走向邮局出口，但三人未与杨间等人同时回到大汉市烂尾楼；他们是否被分别送回原来地点、能否及时抵达福寿园，均待后文。（原文回跳第744章）

## 原文回跳第750-759章·疑为坟场入侵的坐标

刘老太尸体消失后，福寿园被旧坟鬼域覆盖。杨间推测红信可能是引子、诱饵或坐标，使原本存在于另一灵异空间的坟场鬼域向信件所在位置入侵；这一推测可解释邮局为何将任务指定在事发地，却尚不能证明红信主动制造了事件或信使在替鬼送信。（原文回跳第750章、原文回跳第751章）

坟场中找不到鬼域内的78号坟，刘欣悦又目击刘老太尸体游荡。杨间重新判断信件的目标仍可能是原福寿园78号墓主刘老太，而非坟场编号；信件未送达、内容未公开，任务是否会因坟场变化而改变不详。（原文回跳第757章、原文回跳第759章）

## 原文回跳第763章·送入刘老太手中后消失

杨间在刘老太被坟场埋入后挖开其坟，将红色信件塞进尸体手中。信件随即像突然腐朽般破碎消失，没有留下信纸内容、奖励或失败提示；杨间和孙瑞暂将此视作任务成功。（原文回跳第763章）

信件消失后刘老太尸体很快出现黑血和爆裂异变，坟场多座老坟也开始塌陷。二者是否为任务完成的直接后果、红信究竟是完成封印还是打破平衡，均不详。（原文回跳第763章、原文回跳第764章）
