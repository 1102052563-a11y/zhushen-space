# 鬼相机

## 概述

鬼相机是刑警总部地下灵异物品库一号房间内封存的一台老旧相机。第500章，杨间透过钢制房门上淡黄色玻璃看见它被放在约五平方米小单间中央的普通桌子上：相机年代久远，外漆剥落、脏旧得像废品。房门外层为钢铁、内部疑有黄金夹层，杨间无法以鬼域入侵，说明总部以隔绝灵异的方式单独保存其中物品。其前任主人是一名摄影师，但姓名、取得经过及相机何时被总部收缴均不详。（第500章）

## 已知效果

曹延华称，鬼相机可给鬼拍照：若成功拍到一只鬼，会将该鬼关押进吐出的照片中。前任摄影师曾拍到一只鬼，目标随即消失，相机则吐出一张呈现那只鬼形象的照片，这是总部认定其具有关押效果的已知案例。该物的关键并不是普通摄影，而是把厉鬼完整摄入画面；曹延华明确说明，只要少拍到一条腿、一只手臂等任何身体部分，关押便不能完成。它是否能同时拍摄多只鬼、照片能保存多久、照片破损后会发生什么，原文均未说明。（第500章）

## 风险与封存原因

使用过程会触发闪光，而闪光存在失控风险；一旦失控，使用者会被关进照片里。曹延华还只掌握不完整记录：每次拍摄时镜头可能被不存在的手、第二个人影或其他东西遮挡，成片也可能模糊，像有未知存在干扰拍摄。因此即便拍摄者愿承担被关押的风险，也未必能拍清厉鬼全貌。总部因成功率不高、潜在恐怖尚未查明而没有常规启用它，而非单纯否定其关押价值。（第500章）

## 处置定位

杨间认为，若遇到无法解决的厉鬼，这件物品可成为以命换取关押机会的底牌；但他也判断自己需要的是保命物而非频繁赌命的物品，故当时没有选择。王小明已考虑把鬼相机与相邻房间的八音盒共同启用：八音盒的短时不死或可抵消相机失控时对使用者的伤害，从而在牺牲一人的条件下争取关押一只鬼。该组合只是总部的预案与讨论，未在本批实际使用，不能视为已证实的安全方案。（第500章）

## 原文回跳第722-723章·熊文文被困照片后的转移方案

鬼画行动中，熊文文用鬼相机拍摄目标时相机失控，将熊文文本人关进黑白照片。柳三与李阳转述王小明判断：照片中的熊文文可能未彻底死亡，若以灵位转移意识、再提供能承受厉鬼的专属纸人身体，有机会将其从照片中释放。该方案建立在杨间曾借灵位完成意识转移的经历上。（原文回跳第642章、原文回跳第722章、原文回跳第723章）

杨间提醒照片内可能同时困着熊文文和厉鬼，错误释放会让熊文文死亡、厉鬼复苏。专属纸人已被送至杨间办公室安全屋，但鬼相机本体、照片的完整状态及实际转移步骤均未在本批执行。（原文回跳第722章、原文回跳第723章）
