# 黄铜钥匙

## 概述

黄铜钥匙是鬼橱在七日保护期结束、开启新交易时从下层柜门交给杨间的老旧钥匙。它外表近似普通老物件，没有明显灵异气息，但与一张木制老宅照片同时出现；照片背面要求杨间在三个月内“打开那扇上锁的门”。杨间因此判断钥匙对应的并非普通房门，而是老宅中某个被特意锁住的高危区域。（原文回跳第676章）

## 来历与关联任务

午夜十二点，鬼橱上下柜门自行开启并渗出猩红鲜血，杨间脚下也浮现血脚印；血液汇合后退回柜内，鬼橱才从下层交付老宅照片和这把钥匙。照片里的老宅以木材建造、年代久远，大门紧闭；杨间认出它与曾见过的鬼新娘照片背景、灵异公交车途经地点可能相同，但尚不能确认。照片背面的任务和三个月期限表明，黄铜钥匙是鬼橱新一轮交易的必要物品。（原文回跳第676章）

## 已知特征与风险

杨间观察钥匙后认为，它表面没有异常，也不像可直接发动的灵异物品；但鬼橱不会为普通门特意交付钥匙，故其特殊性只能在对应老宅的上锁房门处验证。他推测鬼橱、鬼新娘、鬼画、木制老宅和上锁房门之间存在连续关联，鬼橱可能借交易者之手完成某个目的；此为基于多次交易线索的推测，并非已证实事实。（原文回跳第676章）

任务要求本身风险极高：锁门可能关押更恐怖的厉鬼，开门后杨间可能当场死亡。杨间没有立即前往，而是收下钥匙，打算在三个月内调查老宅和相关线索；若任务过于危险，他也保留终止与鬼橱交易的打算。钥匙能否打开照片中的门、开门后会出现何物及逾期后果均不详。（原文回跳第676章）

## 现归属

杨间在鬼橱钟塔内收下黄铜钥匙后带回住处，未交给总部、公司或其他人。其后续保管方式、是否会被鬼橱远程影响及使用时间均不详。（原文回跳第676章）
