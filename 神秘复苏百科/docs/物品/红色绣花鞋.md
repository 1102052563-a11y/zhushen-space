# 红色绣花鞋

## 概述

红色绣花鞋是一双藏在钱毅皮鞋里的老旧女式绣花鞋。钱毅死亡后，尸体被体内灵异驱使站起，上半身折断般后仰，双腿却能自行行走。杨间以无头鬼影压住尸体后，脱去外层皮鞋，才发现真正灵异是这双小巧的红鞋。鞋呈民国风格，红色纹样怪异；钱毅的大脚被强行挤入鞋内，已扭曲畸形。（第468章）

## 表现

尸体穿鞋后仅走几步，便从原地出现在百余米外。杨间判断效果近似鬼域、却不同于普通鬼域，可能可以无视距离行走；这是仅由本次移动得出的判断，触发条件、距离、是否可带人均不详。（第468章）

鞋在杨间的鬼域中显形，尸体被鬼影压住后倒地；脱鞋后的灵异暂被压制。杨间推测其若复苏，或能走出自己的鬼域，但没有实战验证，不能视作已证实能力。（第468章）

## 来历与收容

原文仅知钱毅驾驭或持有此物，来历、完整驾驭条件和复苏风险均不详。杨间因鞋码和钱毅双足畸形，认为其更适合小脚女性，这只是现场判断。杨间将鞋放入行李箱中的黄金盒并锁好；后续归属和长期关押效果未写明。（第468章）

## 与钱毅死亡的关系

钱毅在鬼差袭击中被草绳吊起、身体遭黑手探入搜寻，随后因流血过多死亡。绣花鞋在尸体死亡后才显露并驱使双腿活动，但原文没有说明它是否在鬼差袭击时被压制、是否曾参与钱毅求生，或其复苏是否与钱毅失血死亡直接相关。杨间仅依据无头鬼影可压住尸体、取鞋后异动停止，判断这是一件能被处理的普通层级灵异物，不等同于当时的鬼差。（第461章、第467章、第468章）

该物被发现时，钱毅的外层皮鞋掩盖了它的存在；杨间是在自身鬼域内才看清真正灵异。这一经过说明外观正常的鞋层不能排除其中藏有灵异物，但不能据此推广为通用鉴别规则。钱毅是否知晓鞋的能力、是否主动把红鞋藏进皮鞋，以及其遗体在黄金收容后如何处置，均不详。（第468章）

## 第559章·与画中女子鞋履的未证实联想

杨间在鬼画画中人身上看见一双如油画绘出的、不够真实的鞋子，联想到自己在鬼差事件后收容的红色绣花鞋；同时，鬼橱照片中的红衣女子也穿有精致绣花鞋。此为杨间由外观拼接出的线索，画中鞋的颜色、纹样和是否为实体均未明确，不能确认就是该收容物或具备相同能力。（第559章）

红色绣花鞋仍只确认由杨间以黄金盒收容；它与鬼画画中人、鬼橱照片女子及干尸新娘是否有关，均不详。（第559章）
