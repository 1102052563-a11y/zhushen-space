# 死人头

## 概述

死人头是一颗带黑发、面色惨白且高度腐烂的人头，为朋友圈驭鬼者刘栋在杨间第五层鬼域袭击后留下的灵异物。刘栋身体在红光中消失，现场只剩这颗眼皮会眨动、眼球能转动并持续腐烂的人头；其原本属于何种厉鬼、与刘栋如何结合及完整称谓均不详。（第591章、第592章）

## 已知表现

人头会以死灰、瞳孔发散的眼睛盯视目标。吴云曾以它与自身对视来逼退侵入身体的鬼手，但代价是自己脸部开始腐烂；杨间也认为它能压制被其盯住的鬼，并使被盯住的活人在短时间内死亡。上述效果分别来自现场使用和杨间的判断，具体距离、锁定条件、压制时长和对高恐怖厉鬼的稳定性均不详。（第593章、第594章、第639章）

杨间在平安大厦杀死吴云后收走尸体与这颗人头，未立刻把它交总部。第639章，他将其列为可供鬼童吞食的候选：此物危险较高，却可能保留压制厉鬼和杀人的能力；他当时没有投喂，也未确认鬼童吞食后会获得能力还是失控。（第594章、第639章）

## 第641-644章·交由鬼童携带

杨间为应对平安科技研究基地内的八音盒诅咒方案，让穿上鬼寿衣的鬼童带上装尸袋内的死人头，意图以寿衣抵住人头的风险并给自己留下后手。鬼童接到“没有命令不准袭击任何人”的要求后随杨间前往基地，实际能否完全遵守未获验证。（第641章）

第643章，杨间命鬼童杀死所有进出基地大楼的人，鬼童遂带人头在楼内游荡。它因六岁孩童体型抱不稳成年头颅，常在奔跑时令其落地滚远，再捡起继续玩耍。第644章，刘助理和同事目击人头从拐角滚出并转向自己；二人逃跑后，鬼童将头捧起。刘助理未被人头正面盯住，故没有当场死亡。人头是否已主动锁定任何人、此后是否造成伤亡及杨间能否收回，均不详。（第643章、第644章）
