# 鬼瓷

## 概述

鬼瓷是岛国除灵社制造或掌握的人造灵异物品，外形为黑红相间、呈痛苦求救人形的怪异瓷器。杨间以鬼眼观察时，看见瓷器上有阴冷黑气凝成人形，判断它不是完整厉鬼，而是截取部分灵异力量后封存的产物，类型近似鬼烛、替死娃娃。（原文回跳第644章）

## 启动方式

使用者须以能表明自身身份的媒介触发鬼瓷。王信把写有本名“长谷川信”的证件压在瓷器下，证件随即以无法理解的方式进入瓷器内部并被灵异浸染，鬼瓷才开始对他生效。媒介可被杨间鬼手从瓷器内取出，取出后保护立即消失。证件之外能否使用照片、指纹或其他身份媒介均不详。（原文回跳第644章）

## 已知效果与代价

- **抵挡厉鬼袭击**：王信使用鬼瓷后，无头鬼影最初无法拆下其手臂；其后鬼影加深入侵，王信仍未立刻死亡，说明鬼瓷能在一定程度上转移或抵消灵异伤害。（原文回跳第644章）
- **同步损伤**：无头鬼影令王信手臂扭曲时，鬼瓷同一条瓷手臂出现裂纹；杨间以水果刀砍断瓷手，王信手臂随即如瓷器般碎裂。鬼瓷遭实体损毁会把对应伤害转移给绑定使用者。（原文回跳第644章）
- **媒介解除**：媒介被取出后，鬼瓷停止保护，王信原本被鬼影扭断的手臂立即被拆下，显示此前伤害并未真正消除，只是在灵异保护下维持相对完整。（原文回跳第644章）

## 归属与交易

王信以一尊鬼瓷作为王野死亡的赔礼送给杨间。杨间试验后收下残缺个体，认为其比需要持续燃烧的鬼烛更省耗，普通人持有也可能在普通灵异事件中存活；但它存在媒介绑定和同步受损风险。回跳第648章，除灵社又承诺以五尊鬼瓷作为杨间赴神户市处理鬼敲门事件的部分报酬，尚未交付。（原文回跳第644章、原文回跳第648章）

## 原文回跳第649章至原文回跳第658章·神户订金与临战启用

杨间抵达神户后，除灵社已交付本次报酬中的三尊鬼瓷，另有二十五吨黄金作为半数订金；故第648章约定的五尊并非尚未交付，而是本批已先收三尊，余数、最终结算和此前残缺鬼瓷是否单独计算均不详。（原文回跳第649章）

人头气球斩首酒井、空中死人头又持续补位后，杨间推测现场所有活人可能都已被选为潜在目标。他取其中一尊完整鬼瓷，拔下一小束头发作身份媒介并压在瓷器下，头发随即进入瓷器；杨间感到全身仿佛被厉鬼笼罩般闭塞压抑，仍将其留在街边小店，作为即将接触敲门鬼前的额外保险。本批未发生鬼瓷实际代挡袭击，媒介是否能抵抗人头气球或敲门鬼、使用后损耗与能否取回头发均不详。（原文回跳第658章）

## 原文回跳第659-661章·抵挡尸斑后完全破碎

杨间靠近敲门鬼后，鬼瓷先以压抑闭塞感抵消部分阴冷侵蚀，瓷身却迅速被黑影浸染、浮现细小裂纹。老人抓住杨间戴黄金手套的手腕时，鬼瓷相同位置的手腕裂纹扩散、手掌断裂，证明它仍会将绑定者遭受的灵异伤害转移或同步呈现；但杨间并未因此完全免受尸斑影响。（原文回跳第659章）

死人头失控、压制崩解后，敲门鬼对杨间的侵蚀继续增强，鬼瓷最终满布裂纹并化为碎片，头发媒介的去向未写明。杨间脱离鬼域后，尸斑仍在手臂、肩膀和胸口蔓延，须靠无头鬼影融入身体才逐步清除；这表明该尊鬼瓷可延缓但不能完全抵挡敲门鬼尸斑诅咒。一尊完整鬼瓷在本次彻底消耗，剩余数量不详。（原文回跳第660章、原文回跳第661章）

## 原文回跳第737章·以身份证保护大汉市调查者

杨间进入大汉市烂尾楼调查前，将自己和李阳的身份证分别压在两尊鬼瓷下，作为可识别个人身份的媒介。房内灯光随之闪烁，李阳感到阴冷压迫；杨间说明鬼瓷会持续提供保护直至破碎，未遭灵异袭击时可长时间维持，但副作用仍不清楚。（原文回跳第737章）

这次使用说明身份证也可作为身份媒介，且杨间手中至少仍有两尊可用鬼瓷。两尊瓷器被留在酒店房间并要求封锁，不代表它们已在本批抵挡袭击或已经损坏。（原文回跳第737章）

## 原文回跳第757-758章·为刻名诅咒破碎

坟场中的刘老太以疑似棺材钉刻写杨间名字时，大汉市酒店内、用杨间身份证绑定的一尊鬼瓷同步出现裂纹；裂纹先形成“杨”字，待墓碑姓名完成后鬼瓷彻底炸裂。杨间据刻字时间变慢与鬼瓷破碎，推测鬼瓷替他抵消了部分刻名诅咒，为其争取到应对时间。（原文回跳第757章、原文回跳第758章）

鬼瓷最终无法完全阻挡坟场埋葬，杨间仍被坟土压制；其后依靠替死布娃娃脱险。该尊瓷器确认耗尽，李阳绑定的另一尊状态未写。（原文回跳第758章）
