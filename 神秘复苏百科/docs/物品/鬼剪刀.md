# 鬼剪刀

## 概述

鬼剪刀是方世明以黄金盒密封保存的灵异剪刀。杨间曾从朋友圈资料室取得其照片，照片显示剪刀老旧生锈、把手缠着黑发；资料只确认方世明持有它，未说明能力。（第544章）第571章，杨间将交出鬼剪刀设为和解条件，姜尚白见照片后明显知晓其重要性。（第571章）

## 外观与收容

第574章，方世明打开焊死的黄金盒，盒内剪刀为旧式样，表面有难辨锈迹或污渍，手柄缠满黑发，部分还连着头皮。开盒后办公室变冷并出现腐臭。方世明称黄金能隔绝灵异，故长期密封随身携带。剪刀只知由他在某次灵异事件中从鬼身上取得，来历不详。（第574章）

## 已知规律与代价

方世明摸索出的使用方法是持有目标完整全身照并写下真实姓名。第574章，他在杨间全身照上写“杨间”后剪向照片颈部，现实中的杨间随即颈部裂开；第二次剪动后其头颅与身体分离。照片、姓名以外是否另有条件不详。（第574章）

使用会加深诅咒。方世明拿起剪刀后，办公室黑暗加深，出现滴水声、腐臭血腥味和残缺尸影；照片亦开始陈旧褪色。他认为剪颈部消耗较小，却仍忌惮反噬。诅咒如何伤害持有者、黄金盒是否完全压制均不详。（第574章）

## 使用记录

- 第571章：杨间索要鬼剪刀作为和解条件。（第571章）
- 第574章：方世明以写有杨间姓名的全身照发动袭击。（第574章）

## 原文回跳第770-771章·总部作为调停补偿交还

杨间钉住叶真后，总部为促成双方停战，经沈良提出三根鬼烛与已由曹洋回收的鬼剪刀作为补偿。杨间在卫景将到、自己鬼眼状态恶化的局面下接受调停；原文只确认总部承诺事后交付，未写他在本批实际收到剪刀或重新使用。（原文回跳第770章、原文回跳第771章）
