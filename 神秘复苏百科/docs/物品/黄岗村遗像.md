# 黄岗村遗像

## 概述

黄岗村遗像是一张从鬼画鬼域内黄岗村三层民房二楼掉落的老旧老人遗像。行动队起初只听见隔壁房间的脚步、看见木门被推开，却在点燃鬼烛后发现房内没有实体厉鬼，只有老旧家具、农具和墙上的遗像掉到地上。画中是一名男性老者，脸有皱纹、神态死气沉沉，却睁着眼并带着微笑；众人从不同角度看去，都觉得老人正在观察自己。其是否本身就是鬼、承载厉鬼、或仅是某种灵异现象留下的媒介，均不详。（第521章、第522章）

## 外观与异常

遗像掉落时，上半张老人脸被黑暗遮住，下半张脸像带着和蔼笑容，旁边另有难以辨认的字迹，似为生卒日期和儿孙信息。柳三擦去灰尘后，看出背景是一栋一楼大门敞开、内部漆黑的房子，与行动队当时所在房屋完全相同；老人背后还站着五道模糊人影，左二右三。其中一人面色蜡黄，能对应柳三；一人持有燃着绿色火焰的蜡烛，另一人手持手机，李军、柳三、苏凡据此判断画中正是自己五人的形象。画面形成时间、是否记录未来或重复经历，以及其余两人身份如何辨认，均未得到解释。（第522章）

## 收容与后续表现

李军因鬼烛燃烧平稳、周围暂未显示有鬼，决定先把遗像关押，避免意外；柳三取出黄金制手提袋，将遗像装入并封好。五人离开房间时，门外不远处立着一名背对众人的模糊黑影，身体僵硬，轮廓像老人；其身后是黑暗房屋与五名行动者，场景和遗像构图完全重合。此时鬼烛骤然剧烈燃烧，证明附近出现鬼。老人背影是否由遗像中走出、遗像是否仍在黄金袋内产生影响、其杀人规律及五人是否遭袭，章节均未写明。（第522章）

## 归属

遗像由柳三在李军指挥下放入随身黄金手提袋，行动队随即离开房屋。第523章行动队先失联，后有五名被杨间判定为鬼的人影走出鬼域；原文没有写黄金手提袋、遗像或柳三本人是否仍在其中，因此遗像现存位置与收容是否有效均不详。（第522章、第523章）
