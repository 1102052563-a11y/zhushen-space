# 裹尸布

## 概述

裹尸布是刑警总部地下灵异物品库中单独封存的灵异物品。第501章，杨间隔着金色盒子看到它，曹延华说明此物可包住厉鬼，形成从未失败过的压制，但需要相应时间；杨间当时考虑用它处理凯撒大酒店中持柴刀的尸体，因适用范围较窄而未立即选择。（第501章）其来历、前任使用者、材质、压制时间的精确长度、如何解除包裹及对活人的作用均不详。

总部以逐间独立房间保存库中物品，钢门和疑有黄金夹层的结构能阻隔鬼域；但第590章，杨间从地下潜入后没有尝试穿过特制大门，而是砸开相邻墙体，进入预先选中的裹尸布房间。该过程只说明墙体未能像门体一样抵抗其力量，不能说明任何人都可用此法进入物品库。（第590章）

## 已知形态

第590章，杨间打开盒子后见到的裹尸布脏旧，布面印有一具尸体轮廓，脸部位置还隐约可见一张模糊人脸。他据此只能看出该物曾包裹过一具可怕尸体；尸体身份、人脸是否仍具灵异、布上轮廓是否会变化均未说明。（第590章）

## 归属与用途

杨间此前为取得鬼画救援报酬而放弃队长职位，认为裹尸布本该属于自己。第590章，他将其取走，打算在八音盒诅咒无法抵挡时处理自己的后事，也保留将其用于某只厉鬼的可能。此时他正筹备对付朋友圈，未实际展开裹尸布或验证其压制效果。物品是否获总部正式交付、曹延华等人何时发现遗失及杨间最终对谁使用均不详。（第590章）

## 使用记录

#### 第590章·杨间强行取走

杨间离开刘小雨宿舍后进入总部地下物品库，略过鬼相机、招鬼幡、人皮灯笼等其他房间，直奔裹尸布所在处。他砸开附近墙体，开盒确认布面尸体轮廓后立即带走。秦老随后以阻隔鬼域的方式拦下他，但未收回物品；杨间保持警惕离开总部。此次只记录取得行为，裹尸布尚未被打开或使用。（第590章）

## 出处
