# 噩梦鬼尸体

## 概述

噩梦鬼尸体是杨孝生前分藏于梅山村鱼塘、树林和小庙的鬼躯：陌生惨白头颅、无头无脚躯干与双腿拼合而成。杨间按杨孝梦中残留方案，以柴刀在双手留“X”记号、以无头鬼影拼合、再用鬼手压制其闭眼，借此推进黑色大狼狗在梦中取代原鬼的循环。（原文回跳第689章至原文回跳第710章）

## 腐烂与压制

鬼手压制期间，尸体逐步腐烂，眼睛最终腐烂消失，杨间据此认为它已失去再次睁眼、从梦中醒来的条件。其后他将尸体装入普通木棺，先放于梅山村老宅，再委托运回观江小区民国古宅；普通木棺不是灵异关押容器，只是短期承载物。（原文回跳第710章、原文回跳第717章至原文回跳第718章）

## 原文回跳第733章·黑毛异变

尸体在观江小区古宅木棺中发生异常撞击。杨间开棺后发现，原本高度腐烂的尸体已化为覆满黑色浓密毛发的腐肉，黑毛蔓延至棺底和内壁，肉团会抽动、起伏、扭曲，近似要重新活动。鬼眼无法看出这种异变的规律或来源。（原文回跳第733章）

王珊珊命鬼童袭击棺内目标时，鬼童没有行动。杨间因此提出三种未定解释：鬼童失控、棺内目标不符合鬼童袭击条件，或棺内已不再是鬼。该现象不能证明原鬼已消失，也不能证明黑毛腐肉安全。（原文回跳第733章）

为避免风险波及观江小区居民，杨间以鬼域将整口木棺转移到尚通大厦办公室的小型安全屋，与熊文文专属纸人同处隔离空间。此处安全屋可暂困异常目标，但尸体异变是否会继续、黑色大狼狗取代计划是否被影响均不详。（原文回跳第733章、原文回跳第734章）
