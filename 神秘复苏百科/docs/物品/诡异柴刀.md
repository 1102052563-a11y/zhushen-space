# 诡异柴刀

## 概述

诡异柴刀原由凯撒大酒店四层异常走道内的持柴刀高大尸体握持。刀身锈迹斑斑，杨间先前看见其有泥土和疑似凝固血污；第605章，他又由无头鬼影与持刀尸体背影几乎重合、鬼影颈部和新腰部断口完全平整，推测柴刀曾将一只完整的鬼肢解成腐烂身体、无头鬼影与鬼影头等部分。这是杨间基于现场的推测，不等同于已证实的拼图来历。（第604章、第605章）

## 取得与媒介

第606章，杨间以裹尸布完全包住高大尸体后，隔黄金手套强拽柴刀未果；改以鬼手接触，尸体虽在裹尸布中激烈挣扎，柴刀却自行脱手。第607章，他将手掌覆到82号房蔓延出的血手印上，媒介触发后才看见手印源头般的无眼人影；挥刀斩击一条手臂，众多人影手臂同时断开，血手印退回并使82号房门关闭。完整脚印和完整手印均能作为已见媒介，但其他媒介、锁定范围及一次攻击能波及多少目标均不详。（第606章、第607章）

## 代价与现归属

第608章，杨间检查首次挥刀后的身体，发现融合鬼差部分血肉的黑色手臂已有近半变白，推断为抵销柴刀诅咒失去一个压制厉鬼名额；该名额原本用于维持鬼眼与鬼手平衡。他用柴刀经鬼影覆盖办公室地面锁定方世明遗留脚印，再斩其影像，手臂黑色又褪去一大截，只剩一个压制名额。柴刀由杨间装入原本盛放裹尸布的金盒携带，使用后果是否必定消耗压制名额、名额耗尽后的代价均不详。（第608章、第609章）

## 原文回跳第665-667章·近距离断臂与同位腐烂代价

杨间在神户市以五层鬼域将敲门鬼拉回现实、令其坠入箱体后，盖板被老人举起的手臂卡住，巨石与两人重量均无法闭合。他没有经由手印或脚印媒介触发柴刀诅咒，而是用鬼域让刀身直接砍中敲门鬼的右臂；枯瘦手臂当场折断，盖板随即关闭，证明柴刀近距离直接接触厉鬼仍可造成肢解或拆分效果。（原文回跳第665章）

当时杨间未立即感到代价，但约十分钟后，自己的正常右臂出现与所砍位置相同、贯穿皮肉的迟发腐烂伤口，黑肉深入骨头且有继续恶化可能。他据此推断，柴刀直接砍鬼会将相应伤害回馈给使用者；而此前远距利用手印、脚印媒介挥刀，则以消耗压制厉鬼名额为代价。此为杨间的现场归纳，近距离伤害是否必定同位、能否砍头或砍躯干、不同鬼的回馈程度均未验证。（原文回跳第667章）

杨间以无头鬼影融入身体尝试驱散右臂诅咒，理由是只有鬼能对付鬼。本批尚未写出黑肉是否完全消除、柴刀是否还有延迟代价，也不能据此认定近距离使用比媒介方式更安全。（原文回跳第667章）

## 原文回跳第676章·右臂腐烂伤口消退

杨间与鬼橱完成新的三个月开门交易后洗澡，发现柴刀斩敲门鬼后右臂留下的腐烂伤口已不再坏死，只留下一道类似刀疤的痕迹。他推测鬼橱新交易的诅咒或保护覆盖、消除了柴刀的负面影响；这只是推测，不能确认伤口是自然痊愈、鬼影先前处理生效，还是鬼橱直接造成。（原文回跳第676章）

杨间据此警惕，下一次近距离用柴刀后未必还能得到相同结果，鬼橱也可能只是以新的诅咒覆盖旧诅咒。他没有再次挥刀验证，柴刀的长期代价和右臂疤痕是否会复发均不详。（原文回跳第676章）
