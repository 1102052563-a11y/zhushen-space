# 鬼绳

## 概述

鬼绳是一根老旧麻绳，原由玫瑰酒吧老板王岳驾驭，平时在其脖颈打成死结。王岳动用其力量时，鬼绳能在目标颈部无征兆出现并勒杀；第179章王岳死于厉鬼复苏后，鬼绳失控，化为遍布街道的大量索命绳套。杨间于第180章将其收入童倩携带的黄金密封袋，完成关押。（第178章至第180章）

## 效果与限制

- **颈部绞杀**：王岳令鬼绳复苏时，杨间颈部突然出现草绳，如同无形之手拉紧，造成窒息、眩晕和颈部失去知觉。杨间判断鬼绳内存在无实体、不可见的鬼；这一判断仅基于本次交手。（第179章）
- **复苏反噬**：王岳每次加强鬼绳，自己脖子上的绳结也同步收紧、发出颈骨碎裂声。其欲以自身复苏来杀死杨间，最终先被鬼绳吊起；说明驾驭者无法免除该反噬，但完整规律不详。（第178章、第179章）
- **失控扩散**：王岳死亡后，鬼绳在鬼域内垂下成千上万根绳套，可进入街道、店铺和车厢；普通人一被套住便被吊起。杨间认为这些分散绳索是厉鬼力量的显化，鬼影抓住后会化灰消散，真正鬼绳仍系在王岳尸体颈上。（第179章、第180章）
- **对抗与关押**：杨间以鬼眼短暂松开自身绳套，再用无头鬼影将其扯下；鬼影因无头，鬼绳套住其颈部后会自行滑落。杨间最终命鬼影把从王岳尸体上取下的本体装进黄金密封袋封死。（第179章、第180章）

## 归属

第180章前，鬼绳由王岳驾驭。王岳死后，杨间使用童倩提供的黄金手提密封袋收容，实际保管归属、后续是否转移均未写明。（第180章）

## 使用记录

#### 第178-179章·王岳的反击与复苏

王岳为求杨间带其离开鬼域，承认杀害严力一家并提出反过来对付小强俱乐部；杨间拒绝，开枪后双方交手。王岳脖子显出鬼绳，随即将草绳套到杨间颈部。杨间先以脖颈新生的两只鬼眼令绳索稍松，再由无头鬼影拉扯抵抗；王岳继续催动能力，自己被绳索吊起。童倩用鬼脸笑声令王岳死亡，但鬼绳因此彻底复苏。（第178章、第179章）

#### 第180章·黄金袋关押

王岳死后，鬼绳将多名幸存者吊起。杨间点燃鬼烛逼退绳索，取得童倩的黄金密封袋；无头鬼影先毁去妨碍的王岳尸体部分，再扯下颈部鬼绳。敲门鬼黑暗逼近时，鬼绳试图套住无头鬼影却因无头滑脱，杨间随即命鬼影将其装入黄金袋并封死。街道绳索是否立刻全部消失，本批未写。（第180章）

#### 第199-200章·鬼域内主动释放

杨间与叶枫对峙时，鬼域已被王小强的干枯手指钉住，鬼影又无法穿透叶枫的鬼衣。他从别墅取来黄金袋，说明鬼绳已彻底复苏、不能在现实中随意打开，否则可能祸及整座城市，遂仅在受限鬼域内撕开封口。鬼绳被抛向叶枫后迅速从头顶、前后和四周垂落，杨间和叶枫都成为目标；叶枫认出这是王岳的鬼绳，试图丢弃已来不及。鬼衣可暂时荡开绳套，杨间则点燃鬼烛自保。此次使用证明鬼绳在未被驾驭时会无差别索命，杨间只能借鬼域限制其活动范围，战后是否重新收容尚未写明。（第199章、第200章）

#### 第204章·依金线重新关押

叶枫和王小强撤离后，杨间返回仍有密集草绳垂落的鬼域。他曾在本体鬼绳留下金线作记号，遂通过鬼眼穿过显化绳索找到真正本体，再以无头鬼影取下并装入袋中密封，鬼绳本轮复苏由此被制止。杨间称这是借鉴此前关押经验完成的回收；黄金袋的具体归属和日后使用条件仍不详。（第204章）

#### 第258-260章·用于吊死杨间并受鬼影压制

杨间将重新关押的鬼绳以金箔包裹带到张伟市中心别墅，先把一端系在吊灯、另一端套上自己的脖子，再在鬼镜前解开金箔。鬼绳立刻收紧，将已被无头鬼影侵入、失去行动和心跳的杨间吊死；它由此暂时阻止鬼影完全取得尸体。约四小时后，尸体滴出的浓黑鬼影逐渐覆盖绳索，鬼绳失去力量、从顶部垂下，并反被鬼影控制。王小明认为鬼绳已完成杀死杨间的环节；其本体是否仍在现场、鬼影控制能维持多久以及之后会否再次复苏，至第260章不详。（第258章至第260章）

#### 第263章·复活后暂时沉寂

杨间复活并离开张伟别墅时，鬼绳仍勒在其颈部，留下深刻痕迹；他徒手将绳扯下，鬼绳在手中未作反应。杨间判断它不是失去灵异能力，而是因落在自己手中而暂时平静，一旦脱离仍会成为危险厉鬼。他随后携带鬼绳与遮住的鬼镜驱车回观江花园，未写重新以黄金容器封存的过程。（第263章）

#### 第271章·近身关押中的短暂失效

第七中内，饿死鬼以干枯手指限制杨间手腕、鬼牙即将咬下时，草绳从杨间身上落下并缠住目标脖颈，试图将其吊起。鬼绳只能令目标踮脚，未能完全吊起或终止袭击，却把鬼牙与杨间头部拉开少许距离，使杨间撑过第一波攻击、放出无头鬼影接过棺材钉。该次效果显示鬼绳面对变化后的源头鬼压制有限；时间重置后，绳索的具体位置和状态未再交代。（第271章）

## 第277章·作为可控武器携带

杨间将复苏鬼绳继续以金箔缠裹，外观近似金绳，并称自己如今可以完美压制它。他把绳子系在腰间、准备如鞭子般作为对付厉鬼的武器；但他仍强调一旦解除限制，鬼绳会无限蔓延，连厉鬼也可能被吊起，普通敌人更难幸免。该次携带只说明杨间自认能控制，尚未写出实际交战效果或失控代价是否改变。（第277章）

## 出处

- 王岳的鬼绳能力与复苏代价。（第178章、第179章）
- 失控扩散、黄金收容及无头鬼影的应对。（第180章）
