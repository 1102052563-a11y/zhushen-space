# 灵位

## 概述

灵位是一块老旧木质牌位，表面有干枯发黑的血污，并嵌有黑白遗照。它最迟在郭凡被困凯撒大酒店时已由其持有，可让使用者的意识进入照片、以灵位内鬼替换现实身体，从而躲避厉鬼袭击；其来历、原主人、制造者与完整规则均不详。（第382章、第383章）

第615至618章，王小明为准备杨间八音盒诅咒方案强取郭凡的灵位。郭凡意识被困入遗照后，灵位内鬼占据其肉身；卫景以鬼绳压制该鬼，王小明收走灵位。郭凡因此失去肉身，灵位成为王小明处理杨间诅咒的关键物品。（第615章至第618章）

## 已知效果

- **意识转移与替身**：接触灵位会令活人意识进入照片，灵位中的鬼或另一意识取代现实身体。郭凡至少四次借此躲避凯撒大酒店内的厉鬼；第643章，杨间与灵位中的郭凡再次完成该种身份互换。（第382章、第643章）
- **照片随身份变化**：第643章，杨间捧起灵位后，原有郭凡照片逐渐褪去，杨间遗像显现；同时郭凡面孔出现在杨间肉身上。该次说明照片会随交换对象改变，不能确认照片是否总是唯一身份凭据。（第643章）
- **长期停留风险**：郭凡称在灵位中停留太久会受到鬼侵蚀、可能无法返回；第643章杨间也感到意识仿佛被抽离、逐步死亡。完整停留时限与能否自主返回均不详。（第382章、第643章）

## 第642-643章·用于八音盒诅咒方案

王小明向杨间提出：先让杨间意识进入灵位、由其中郭凡意识临时接管身体；若八音盒诅咒不随意识进入灵位，则可等诅咒爆发后再把杨间意识取回。为防接管身体者无法维持杨间体内三只鬼平衡，身体同时放入残缺鬼棺压制。王小明承认若诅咒随杨间意识转移，杨间可能死亡、郭凡则占有身体；该方案不是保证救援，而是二人之间的生死赌局。（第642章）

第643章，杨间进入鬼棺后捧起灵位，完成与郭凡的意识、身体交换。郭凡在杨间身体中听见八音盒音乐，王小明据此判断诅咒被郭凡继承，而杨间意识留在灵位照片内暂避风险。郭凡尚未意识到音乐含义，杨间能否在诅咒爆发后顺利返回身体、灵位能否被外部人员控制和交换是否可逆，均不详。（第643章）

## 第648-649章·遗像渗血与陷阱推断

郭凡在鬼棺中异常后，灵位上的杨间遗像不断渗出猩红黏稠鲜血，最终照片被血遮没，灵位本身也被染成鲜红，周围形成血泊。鬼棺震动、黑暗扩散和遗像渗血几乎同步出现，王小明无法用既有实验解释。午夜八音盒关闭后，灵位仍浸在血泊中保持安静。（第648章、第649章）

杨间身体内厉鬼没有如预期复苏，王小明由此把异常重点指向灵位，推测郭凡可能仍活着且已成为能承受八音盒诅咒的鬼，并认为灵位意识交换可能是鬼针对活人的陷阱。该结论只是王小明的推断；血液来源、杨间意识是否仍在遗像内、灵位能否使郭凡鬼化及交换如何收束均不详。（第649章）

## 第650-654章·吞入鬼童并完成一次意识回归

王小明决定把灵位放回杨间身体以赌其意识回归，卫景戴黄金手套取位时却遭灵位之鬼入侵；此后杨间和卫景均呈鬼化状态。王小明从灵位上分离出代表杨间意识的遗照，带走牌位本体，试图利用杨间留下的鬼童反制。（第650章、第651章）

王珊珊暂时令鬼童听从王小明后，王小明命其“吃掉这东西，驾驭它”。鬼童将失去遗像的染血灵位吞下，身体随即成长至约九、十岁；王小明据此判断其获得灵位的意识对调能力。随后鬼童接过杨间遗照并贴到杨间额头，遗照上血迹剥落、杨间脸上诡异笑容消失，杨间意识恢复身体。遗照最终呈现诡异微笑，郭凡、灵位之鬼和被替换意识的最终归属均不详。（第654章）
