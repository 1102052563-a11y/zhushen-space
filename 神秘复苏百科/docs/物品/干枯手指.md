# 干枯手指

## 概述

干枯手指是王小强从其兄王小明研究所带到观江花园的异常物。它被放在金色箱子内，外观比例失衡：手指干瘦、指甲漆黑，长度几乎接近人的小臂；王小强将其称为鬼烛实验中衍生出的失败品，且明确承认此物“不受控制”。叶枫在行动前并未见过它，王小强则将其视作针对杨间鬼域的依仗。原文没有说明研究所的制造过程、完整来源或王小明是否知晓王小强取走该物。（第194章）

## 已知效果

- **钉住鬼域**：第197章，王小强把手指黑色指甲刺入地面后，原本对外不可见的红光鬼域显露出来。手指像一枚钉子，能固定鬼域的一处位置，使杨间不能按原先方式自由移动整个鬼域。（第197章）
- **拉扯并压缩范围**：王小强抓住刺入地面的手指不断后退，鬼域随之被拉成长线，杨间周边可用范围迅速缩小。手指并未直接消灭鬼域，也没有令杨间失去鬼眼；杨间仍可在受限范围内转移到王小强面前，说明它是限制而非绝对封禁。（第197章、第198章）
- **战术价值有限**：叶枫借手指争取到与杨间正面交手的时间，但杨间仍能以鬼影攻击王小强，并在后续鬼域内释放鬼绳、点燃鬼烛。手指不能阻止鬼域中其他灵异力量运作，也不能保证王小强不受攻击。（第197章至第200章）

## 使用记录

#### 第194章·作为伏杀底牌出现

王小强为对付拥有鬼域的杨间，向叶枫展示金箱内的干枯手指，并称它是鬼烛的失败衍生品。叶枫此前只判断突袭才可能处理杨间鬼域，未确认手指的效果；王小强则以此确信行动能够成功。此时手指尚未被启用，控制条件、反噬和能否多次使用都不详。（第194章）

#### 第197章·限制杨间鬼域

孔峰等雇佣行动人员被杨间困入鬼域、接近全灭后，王小强在鬼域边缘打开金箱，将干枯手指钉进地面。鬼域立即显形，王小强带着手指向后退去，红光范围被拉扯缩小。该举动为叶枫现身拦截创造机会，却未能阻止杨间转移而至；王小强随后被鬼影扯开头皮，手指也未再有单独描写。（第197章、第198章）

## 当前状态

第203章，王小强守在仍插于地面的手指旁，杨间以无头鬼影将其拔出，鬼域随即恢复自由。王小强未能夺回，手指由杨间取走；杨间此后如何保存、研究所能否追索，以及持续使用的消耗和反噬仍不详。其名称、研究所背景和“鬼烛失败品”说法均来自王小强，尚无其他来源印证。（第194章、第203章）

第241章，杨间从王小明顶层办公室取回手指，原本被其钉住的鬼婴在黄金盒中立即挣扎。第246至247章，他以手指攻击可闯入九眼鬼域的叶枫外貌目标，寿衣先挡住刺击；待最后鬼烛令目标短暂停滞，他才绕开寿衣将手指刺入颈部，目标随即不动。第248章，杨间担心脱下寿衣会破坏其与鬼牙的平衡，故将手指连同目标和寿衣封入黄金箱。第249章撤离医院时，长青黑手截走黄金箱，手指现仍留在医院；其是否持续限制该目标及能否再次取回不详。（第241章、第246章至第249章）
