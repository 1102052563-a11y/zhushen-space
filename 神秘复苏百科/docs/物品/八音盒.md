# 八音盒

## 概述

八音盒是刑警总部地下灵异物品库二号房间内封存的一个小巧木盒。第500章，杨间在观察鬼相机后隔着房门玻璃看到此物，曹延华将其称为“八音盒”。它与鬼相机一样被单独置于编号房间内，外部的钢铁与疑有黄金夹层的门用于阻隔灵异力量；其来历、前任持有者、木盒内音乐的名称和实际运作结构均不详。（第500章）

## 已知效果

曹延华称，打开八音盒后会播放音乐，持有它的人在音乐持续期间“绝对不会死”。这一效果不等同于鬼烛：鬼烛是使厉鬼不袭击烛光范围内的人，八音盒却不能阻止厉鬼发动袭击，只保证持有者即使遭遇恐怖厉鬼也不死亡。该说法已有测试和实际使用者作为依据；持续时长的极限未经测试，但曹延华能确认至少可轻易维持数小时，甚至可能达到一整天。音乐是否需要持有者亲自开启、离开持有者后效果如何、能否同时保护多人，原文没有交代。（第500章）

## 已知后果

总部并不知道八音盒即时不死效果背后的完整风险，却掌握一项共同结果：所有曾以它续命的人，包括驭鬼者，最终都死亡，且死因诡异。最后一名使用者出现后，总部曾试图保住其性命，结果仍告失败。曹延华据此将后续死亡归为难以理解的诅咒，并以该物容易害死人为由封存。诅咒在何时发生、具体如何杀人、是否存在避免方式，以及此前使用者人数，均不详。（第500章）

## 与鬼相机的组合设想

杨间听取风险后提出以八音盒配合鬼相机：短期不死或许可以抵消相机闪光失控时把使用者关入照片的风险，换取拍摄关押厉鬼的机会。王小明表示总部此前也作过相同考虑，故将两件物品分别放在一号、二号房间；若不得不启用，应该一同启用，至少可在牺牲一人的情况下保证关押一只鬼。该方案仅为推演，没有本批实战或成功记录，八音盒诅咒也仍可能在事后夺命。（第500章）

## 第514-515章·杨间改选后的交付与封存

朋友圈决定对杨间动手的警报传来后，杨间放弃原先较倾向的裹尸布，改向曹延华索取八音盒作为与朋友圈接触前的保命底牌。第515章，总部以金色盒严密包装八音盒，并与两根红色鬼烛、一根白色鬼烛及替死布娃娃一同秘密送至平安酒店。杨间确认其余物品后，才打开金色外盒查看八音盒；物品现由杨间暂时持有，是否已完成正式调拨手续不详。（第514章、第515章）

木盒为正方形、外观老旧，木制外壳上的红漆斑驳脱落，传出腐烂尸臭。杨间由红漆木质外观联想到鬼橱，推测两者可能同一时期、或同为旧时驭鬼者以厉鬼制成的物品；这只是他的现场推测，不能确认来历关联。未开启时，八音盒周围已出现阴冷气息、电灯闪烁、光线黯淡和电路异响，盒体甚至自行颤动似将开启。杨间立刻盖回黄金盒，异常随即消失；他因而决定只在关键时刻启用。八音盒是否真的自行开启、不开启时的影响范围与长期风险均不详。（第515章）

## 第575-579章·死亡后维持意识的实战表现

方世明以鬼剪刀切断杨间头颅、又横向切开其脑部后，杨间在鬼烛、替死布娃娃和八音盒中选择后者，以无头鬼影勉强操控尸体打开木盒一角。空灵铃声随即只在其意识中响起，杨间已死亡的意识逐渐恢复，证明本次开启确能在致命伤后维持使用者不死；但效果不是修复身体。杨间的头仍裂成两半、身体仍是无头尸体，起初更失去视觉、听觉和行动能力。（第575章、第577章、第578章）

第579章，杨间以残留在后脑的一只鬼眼恢复视野，确认八音盒保住的是寄于上半头颅的意识。他能清晰思考，却仍须依赖外力将头拼回尸体；脑内铃声持续循环，并开始轻微干扰正常思维。诅咒会持续多久、何时以何种方式杀死使用者、是否会阻止无头鬼影复苏及能否帮助恢复肉体，均不详。（第577章至第579章）

## 第580-584章·诅咒维持意识回归

黄子雅点燃红色鬼烛寻找无头尸体时，杨间确认八音盒铃声没有受烛火影响，至少本次鬼烛不能隔绝其诅咒。（第582章）头颅被放回尸体后，杨间以鬼域将承载意识的半个头颅强行与身体重叠，八音盒诅咒使意识没有被体内鬼消灭，并逐步压过鬼的本能；约半小时后，他恢复身体控制、视野和说话能力。（第583章、第584章）

杨间据此前使用者情况推测诅咒持续时间应不超过十天，但这是他的推测，不是已验证规则。他同时推想，若诅咒持续，已死的自己或可承受厉鬼复苏，成为保有思维的异常存在；但任何更强复苏鬼若能压过诅咒，他仍会立刻死亡。八音盒的实际时限、压制层级和最终杀人方式均不详。（第584章）
