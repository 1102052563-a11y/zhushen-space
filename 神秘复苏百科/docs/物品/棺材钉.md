# 棺材钉

## 概述

棺材钉是一根锈迹斑斑、可穿透厉鬼身体的钉状物。杨间最初在第254章想到第七高中灵异空间白骨树上钉住黑影的物品可能可代替干枯手指；第269章，他经鬼域重演定位至弘法寺地下室后的空间，隔着金箔将其取出，准备限制大昌市饿死鬼源头。（第254章、第269章）

## 来历与封存环境

第269章，弘法寺住持称此物为代代相传秘密中的“降魔杵”，其师父只告知门内有高僧关押的恶鬼。寺庙地下室尽头的老木门原先被金箔覆盖并上锁；门后是一处无生命的黑暗空间，内有发霉青砖墙、白骨树、悬挂的干枯人头和破布。三米高、无五官的实体黑影被棺材钉从胸口钉在树上，伤口不断滴落黑血；杨间还发现黑影头部位置似乎缺少一只眼睛。门、寺庙与空间的历史及封印者身份不详。（第269章）

## 已知效果与使用准备

- **长期限制**：棺材钉穿透黑影胸口后使其长期贴在白骨树上；杨间取钉时黑影暂未行动，但不能据此认定其已失去危险。（第269章）
- **金箔隔离**：杨间先以撕下的金箔包住钉身再触碰，顺利拔出且未直接感到异常；金箔是否必要或能完全隔绝其作用不详。（第269章）
- **关押希望**：杨间将其视为限制饿死鬼的最后手段，认为只需命中一次便可能结束大昌市事件；此时效果尚未在源头鬼身上验证。（第270章）

## 现状

杨间第269章拔钉后立即离开空间，要求弘法寺住持关门远离。住持随后发现门后只剩青石砖墙，原有灵异空间已消失；白骨树黑影是否脱离、棺材钉是否会带来新的风险均不详。第270章，杨间已把棺材钉带回第七中，并在等待变化后的源头鬼靠近时以金箔包裹，避免目标察觉威胁。（第269章、第270章）

#### 第271-275章·远距贯穿饿死鬼并随总部接收

第271章，杨间近身反击时，青黑鬼手以干枯手指钉住其持钉手腕，无头鬼影才接过棺材钉继续刺向源头；行动将成却遭局部时间重置，钉子仍回到杨间手中。第273章，他从一千余米外以三层鬼眼鬼域锁定第七中北门门卫室，把棺材钉直接转移进穿寿衣、面貌近周正的饿死鬼头颅，几乎贯穿头部。目标随即失去异动、阴霾消散，说明该次命中可限制它。第275章，杨间拒绝拔钉或取走寿衣；接收人员以金箔、金绳与厚金箱封装尸体，赵建国下令焊死押运。钉子现随目标进入总部接收程序，之后归属、封存效果与能否再次使用均不详。（第271章、第273章、第275章）

#### 第289章·照片交由王小明评估

会议结束前，杨间把白骨树、被钉黑影和棺材钉的照片交给王小明。王小明根据照片判断白骨树与三米高的人形轮廓很可能都是高等级厉鬼，棺材钉则是限制它们的重要工具，并表示会作危险评估上报。杨间称照片中的东西已不在原处；这与先前钉子随饿死鬼进入总部接收程序并不矛盾，但照片所示黑影去向、评估结论和总部后续行动均未写明。（第289章）

#### 第433-435章·第二根线索与鬼制钉状物

第433章，张雷见到杨间以鬼域显出的棺材钉形象后，回忆起自己吞食过的一只鬼残留的记忆：一片黑暗坟场内，一扇由木柱和破败铁门组成的大门旁，有一根棺材钉钉在木柱上。张雷不能确定地点，也不能说明该物是否与杨间曾使用的钉子同源；杨间只因灵异公交曾经过无尽坟场而产生联想。该线索尚未被实地确认，不能据此认定第二根钉子存在或可取得。（第433章）

第435章，王小明展示另一枚形近钉子的黑色菱形尖锥，称其并非杨间用于限制饿死鬼的锈蚀棺材钉，而是鬼棺中孕育之鬼死机后、由李军制成的物件。王小明称该物能使三只鬼能力失效，若不计副作用价值近似棺材钉；杨间尚未接受以未知对象交换该物的交易。两种钉状物是否同类、黑色尖锥的副作用和实际限制效果均未由本批验证。（第435章）

#### 第445章·鬼差会议中的取用争议

姜尚白面对鬼差事件，建议取出仍限制饿死鬼的棺材钉，集合顶尖团队由其带队处置。王小明没有否认该物可用于第一波袭击，却指出鬼差可能已成长到钉子无法压制，且即使一次得手也不能阻止其后续以其他方式再现；拔钉还会使饿死鬼脱困，形成第二件S级事件。该建议被王小明拒绝，棺材钉仍未被取出。（第445章）

#### 首个第462章·鬼差体内的黑色尖锥痕迹

杨间按人皮纸提示，确认王小明制成、原被鬼差夺走的黑色菱形尖锥仍在卫景尸体载体内。鬼眼所见不再是可直接拔出的钉子，而是一道自鬼差肩部深入、似通向心脏的黑色痕迹；钉体已与冰冷皮肉融为一处。杨间认为若直接取走其他被夺的鬼，脱离鬼棺限制后可能反杀队伍，故仍以取出这枚由三只死机鬼和鬼棺能力制成的钉状物为目标。（首个第462章·鬼回头）

他以覆盖无头鬼影的鬼手伸入黑痕，试图结合鬼影拼接和鬼手侵入三层鬼域的能力，把带钉的部分从鬼差身体肢解下来。鬼差压制令鬼手和鬼影停滞，整条手臂失去知觉，杨间未能拔出或确认钉子是否松动。此物仍在鬼差体内，融合状态、能否分离及取出后是否触发鬼差第二波袭击均不详。（首个第462章·鬼回头、重复第462章·食鬼之人）

#### 第477章·三鬼钉维持的临时平衡

杨间以从鬼差体内取出的三鬼棺材钉与鬼手、无头鬼影形成暂时平衡后，确认即使叠开三层鬼眼鬼域也未出现此前的复苏躁动。第477章，他据此继续叠开至第四层，并准备尝试第五层鬼域反入侵鬼差；原文只将这一稳定状态视为当次尝试的条件，不能确认钉子本身赋予鬼域层数，亦不能确认第五层的代价或长期压制效果。（第466章、第477章）

#### 原文回跳第757-758章·福寿园坟场出现疑似同类钉

福寿园坟场内，刘老太尸体以一根锈迹斑斑的大钉敲击空白墓碑、刻写活人姓名。杨间认为其外形与自己曾用来限制饿死鬼的棺材钉高度相似，又联想到张雷从鬼记忆中见过、钉在坟场大门旁的同类物，故推测这可能是未确认的第二或第三根棺材钉。（原文回跳第433章、原文回跳第757章、原文回跳第758章）

钉子的刻名效果究竟来自钉子本身、空白墓碑还是坟场鬼域，尚未确认；杨间没有取得该物，只计划在送信后尝试从刘老太手中夺走。不能据此认定张雷线索已获实证，或确认棺材钉总数。（原文回跳第758章）

#### 原文回跳第763-764章·从刘老太手中实际取得疑似第二根

杨间以柴刀刻下“刘兰”后，刘老太被坟场埋入。挖开其坟、将红色信送入尸体手中后，他从尸体另一只紧握的手里掰出一根锈迹斑斑钉子。该钉外观与杨间曾用于限制饿死鬼的棺材钉相同，故可确认杨间实际取得一根疑似同类物。（原文回跳第763章）

刘老太尸体在钉子离手后发生黑血、膨胀和爆裂异变，杨间又将该钉钉入其头颅，仍未完全遏止坟场异常。钉子是否因被拔出破坏封印、能否限制其他厉鬼、是否就是张雷见过的坟场钉均未验证；本页不以“第二根”作为定论。（原文回跳第763章、原文回跳第764章）

#### 原文回跳第768-769章·六层鬼域中钉住叶真

杨间与叶真交战时，以六层鬼域使叶真短暂停滞，随后将这根疑似同类钉直接转移进其额头。六层鬼域解除后，叶真灵异反应消失、从空中坠落且无法行动；灵异论坛成员认定拔钉后他大概率会恢复。该次效果说明钉子至少能压制叶真当时展现的能力，但叶真是否死亡、压制是否长期有效和替死能力能否绕开钉子仍未验证。（原文回跳第768章、原文回跳第769章）

杨间因鬼眼复苏加重，无法继续稳固控制钉住叶真的局面；灵异论坛成员随后围拢，试图夺回叶真和钉子。该钉的最终归属待后文，不可视为杨间已永久持有。
