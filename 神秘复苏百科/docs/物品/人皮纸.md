# 人皮纸

## 概述

人皮纸是杨间持有的一张诡异人皮，可写出未来方案与线索，却会在方案中留下致命漏洞。第658章，杨间根据其利用尸水进入平安科技研究基地、介入灵位实验的现象，认为它并非只会被动书写，而可能借曾吞入的厉鬼活动；其完整能力、来源、意识与最终目的均不详。（第658章）

## 第657-658章·借尸水进入基地的推测

杨间在基地五楼窗边发现暗褐色死人皮和尸水痕迹，鬼童又差点把地上的人皮纸吞掉。人皮纸上留有鬼童的黑青手印，使杨间判断鬼童曾因第二阶段规律触碰到它；尸水则被认出是人皮纸早年在黄岗村吞入、原先由叶俊驾驭的灵异。杨间据此推测，人皮纸能借吞入的尸水从其埋藏地下的位置活动到基地，后被鬼童追逐并吞掉尸水，自己则遗留在楼内。（第657章、第658章）

此推测令杨间重新串联卫景复现、郭凡遭灵位鬼入侵、自己与卫景在灵位实验中失控等事件，认为人皮纸可能预知并利用实验布置了更大的陷阱。它之所以未得逞，或因鬼童这个不在人皮纸预料中的异类介入；这是杨间的判断，不能确认人皮纸是否真的操控尸水、参与灵位异常或存在自主意识。（第658章）

## 使用风险与保留决定

杨间承认人皮纸的方案危险，且每次看似给出活路时都可能留下日后才会发挥作用的漏洞；但他也认为在走投无路时，人皮纸至少提供赌命的机会。八音盒事件中，纸上“相信王小明”的提示在结果上成立，杨间因此没有立刻以柴刀毁掉它。（第658章）

第659章，杨间用市郊别墅地下工作室的模具和黄金制成空心金球，将人皮纸装入后焊死，再以金链挂到鬼童颈上。他打算以鬼童约束人皮纸活动，若出现异常便让鬼童吞掉它；这种“以鬼制鬼”的安排是否可靠、金球能否长期阻止人皮纸活动及鬼童会否反而吞食人皮纸，均不详。（第659章）
