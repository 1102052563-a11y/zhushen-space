# 腐烂人头

## 概述

腐烂人头是朋友圈驭鬼者刘栋在第591章遭杨间五层鬼域袭击后留下的灵异遗留物。刘栋的身体和会议室上层建筑一并消失，现场没有伤口或完整尸体，唯独这颗干瘪、凹陷的人头未被送走。人头眼珠持续转动，随后皮肉迅速腐烂，杨间据此认定它是无法被五层鬼域送走、正在复苏的鬼。（第591章）刘栋如何驾驭此物、头颅原本是否为其身体一部分及完整来历均不详。

## 外观与风险

人头干瘪且被撞凹，散发恶臭，内部看不见骨头；眼球死灰、空洞，却会无规律转动。吴云短暂把它举到面前时，眼睛会停下并聚焦注视对方。该注视虽可逼退侵入人体的鬼手，却同时令吴云脸部皮肤腐烂、成片剥落，且没有疼痛感。故它不是安全防护物，使用者只能以衣物遮住并避免持续对视。（第592章、第593章）

## 已知效果

第593章，吴云遭杨间鬼手从眼眶、脚踝等处侵入，无法自行剥离，遂以人头对准自己。人头目光聚焦后，鬼手如受灼般退回并消失；杨间据此判断人头的恐怖程度高于这次鬼手。该效果只在吴云本次遭袭中得到验证，不代表能压制其他厉鬼，也不能确认是否需要人与人头对视、能维持多久或会否反过来杀死持有者。（第593章）

## 归属变动

刘栋死亡后，吴云受姜尚白建议以衣物将人头包起，准备带入安全屋；吴云借它挡过第二次袭击后仍未能摆脱杨间。第594章，吴云死亡时人头滚落，姜尚白试图抢夺，杨间先以鬼域中的手掌提走人头并同时取走吴云尸体。此后持有者、封存地点、是否仍在复苏及能否与刘栋消失的身体重合均不详。（第592章至第594章）

## 出处

第591章至第594章。
