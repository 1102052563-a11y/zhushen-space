# 黄色信封

## 概述

黄色信封是李跃在梅山村树林内寻找分尸时携带的老式信封。信封外观崭新、款式略旧，杨间仅靠近便感到不安，怀疑它是灵异物品。李跃称自己一行人正是依照信件信息抵达小木屋寻找尸体，并自称“信使”。（原文回跳第693章）

## 已知作用

李跃没有展示信内具体内容，只说明信件使他们知道梅山村树林与尸体的存在。该信息准确指向杨父旧木屋下的红木箱，说明至少具备提供地点或任务线索的作用；信封是否自行投递、内容能否变化、为何能知晓十余年前封藏物均不详。（原文回跳第693章）

杨间要求查看时，李跃明确拒绝，称非信使不能看、强看对杨间没有好处。该说法只来自李跃，尚不能确认信封会主动伤害非信使、是否只是隐瞒任务信息，或杨间是否实际无法读取。（原文回跳第693章）

## 归属与风险

本批末，信封仍由李跃保管，杨间没有抢夺或打开。李跃一行与信件关联的任务目标原为树林尸体，而杨间已将尸体带走，后续是否发生交付、追索或灵异反应不详。（原文回跳第693章）

信封与“信使”的完整体系、信件来源、递送条件及失败代价均未披露。现阶段只能把它视为携带灵异风险、能引导持有人前往目标的可疑物，不能断言其具体规则。（原文回跳第693章）

## 原文回跳第694-698章·被杨间拆开后化灰

李跃补充，信件丢失、未送达或信使死亡后均可能引来厉鬼；杨间仍以鬼手撕开信封。信纸依次显示“千万不要打开这封信”“打开这封信的人会被厉鬼杀死”“鬼就要来了”“鬼就在你的身边”，树林随即转暗并出现入侵活人身体的厉鬼，周林一度死亡。信纸内容是否原本就如此、是否因杨间强拆而改变不详。（原文回跳第694章、原文回跳第695章）

入侵型厉鬼的异常消散后，信纸和信封在杨间手中化为纸灰。李跃因此判断本次送信任务结束，但信使诅咒没有解除。该信件的原收件目标、原本要递交给无头分尸的内容、化灰是否等于完成投递均不详。（原文回跳第698章）
