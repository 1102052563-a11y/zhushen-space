# 手摇式放映机

## 概述

手摇式放映机是除灵社保存的老旧灵异物品，至少有八十年、可能近百年历史。机器油漆斑驳、老化脏旧，配有一卷看似没有图像的黑色胶卷；使用者摇动侧面握把后，放映机会把其死亡前的一段影像投到墙上。三岛称许多普通人和驭鬼者使用后，影像均与后来死法相符；但杨间认为它也可能是在制造死亡诅咒，预测性质尚不能完全确认。（原文回跳第670章）

## 使用与表现

- **死亡影像**：山崎摇动后看到自己在大阪街头被斩首；金川、藤原、石田及另一名未具名成员都被预示会以相同方式死亡，且画面附近反复出现皮鞋、西裤目标。放映机无法由使用者选择角度、位置或播放时长，只能被动接收有限片段。（原文回跳第671章、原文回跳第672章）
- **灵异干扰**：影像通常为黑白色。画面中的未知厉鬼靠近山崎尸体时，投影扭曲、闪烁后消失，说明放映机对部分厉鬼的窥视会受影响。其干扰是否来自厉鬼压制、胶卷限制或其他机制不详。（原文回跳第671章）
- **杨间的异常影像**：杨间用鬼手摇动后，投影转为血红，画面内鲜血实际从墙上滴落并扩散；影像指向大昌市郊严力鬼血池，最后出现杨间尸体浮出血池。机器此前未出现过现实渗血和血色投影，原因不详。（原文回跳第673章）

## 限制与风险

三岛称曾有社员看见自己死于车祸，为验证而被送往远离车辆的安全屋，却在前往安全屋的路上死亡；另有使用者看见病床老死，符合一般人生轨迹。除灵社因此禁止普通人随意使用、主要允许驭鬼者尝试借影像规避死局。杨间指出“预知”也可能本身就是诅咒，使用次数、是否会加速死亡及能否改变画面未获验证。（原文回跳第670章）

放映机只能显示死亡前一小段画面，常无法看见完整厉鬼或死亡原因。它在山崎、金川等人的画面中显示同一类目标，却未说明该鬼身份；杨间画面则因血池附近灵异而逐渐残缺。故影像只能视为风险线索，不能替代实际调查。（原文回跳第671章至原文回跳第673章）

## 归属

放映机由大阪除灵社保存在金属箱中，三岛在庆功宴上命下属取出。杨间没有带走或关押该物，仅以鬼手使用一次；其平时存放位置、取得来源和是否会继续用于成员预测均不详。（原文回跳第670章、原文回跳第673章）
