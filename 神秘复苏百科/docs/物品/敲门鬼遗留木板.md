# 敲门鬼遗留木板

## 概述

敲门鬼遗留木板是杨间在神户市首次对抗敲门鬼时，从老人长衫腰间口袋取出的金漆木盒内之物。木板老旧、略有腐朽，材质令杨间联想到鬼棺，但他没有感到直接灵异气息；其上以指甲刻有扭曲数字“114，2230，30”。杨间起初无法理解数字，后在惠子提示下以经纬度坐标定位，地点落在大汉市附近。（原文回跳第660章、原文回跳第662章）

## 来历与获取

杨间曾从敲门鬼照片中发现老人长衫腰间有不合旧式服装的鼓起口袋，怀疑其中藏有秘密。回跳第659章，他以鬼手、无头鬼影和死人头共同压制老人，虽未能彻底限制其行动，仍趁鬼影入侵时从口袋取回一个巴掌大的四方木盒。木盒外刷有薄金漆，封得没有缝隙，鬼眼无法窥视；杨间撤离后以更大的黄金盒临时封装，直至在温泉旅店打开。（原文回跳第659章、原文回跳第660章、原文回跳第662章）

## 已知内容与意义

木盒打开后未触发明显灵异，内仅有该块木板。杨间认为敲门鬼将其长期放在口袋中，说明它可能关联老人死前留下的秘密；但这是他的判断，木板本身是不是灵异物、是否直接指向敲门鬼源头，均未证实。数字被惠子识别为坐标后，定位到大汉市附近；杨间结合此前郭涛反复返回大汉市、当地长期无灵异上报却有人失踪的线索，推测该地点可能重要。这些关于大汉市与老人秘密的关联均为推测，不能视为已知结论。（原文回跳第662章）

## 现归属

杨间取出木板后将木盒和木板收回行李袋，准备记录坐标、返回后再与大昌市队友共同研究。原文未写他交给总部或除灵社，也未写坐标地点的实际内容；截至本批，物品仍由杨间保管。（原文回跳第662章）
