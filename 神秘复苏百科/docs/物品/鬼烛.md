# 鬼烛

## 概述

鬼烛是王小明实验室制造、以金色长盒保存的一根猩红蜡烛。第118章，王小明以它交换被杨间控制的冯全，称其为实验室战略级资源，目前无法用钱买到，价值可与五只厉鬼相比。（第118章）

## 效果与限制

王小明称点燃鬼烛后，在烛火熄灭前，烛光覆盖范围内的人不会被厉鬼杀死，处于绝对安全；实验室已在三起大型灵异事件中检验其有效，连A级厉鬼也不能在火焰熄灭前杀死范围内的人。燃烧速度随厉鬼恐怖程度增加而加快，无法给出固定时长，但王小明估计至少能维持半小时。上述效果均为王小明的说明，杨间尚未亲自验证。（第118章）

## 归属

杨间以冯全交换鬼烛后立即收起，认为钱无法买命、鬼烛则是面对无解厉鬼时的重要底牌。他在第119章要求张韩不要泄露其持有鬼烛的消息，担心其他驭鬼者杀人夺取。（第118章、第119章）

## 第169章使用准备

大昌市中心的鬼敲门事件扩大、赵建国接受杨间救援条件后，杨间从车内金色长盒取出猩红鬼烛，确认其带有腐烂尸臭，并与打火机、特制手枪一同带入黑暗鬼域。他重述王教授对鬼烛的命名与“点燃后、熄灭前处于绝对安全”的说明，将其作为重返敲门鬼鬼域的保险。至第170章末，原文只写杨间携带，未写他点燃鬼烛、烛火实际抵御厉鬼或燃烧速度。（第169章、第170章）

## 第171-175章·鬼敲门事件实战记录

杨间在火锅店二楼点燃鬼烛，绿色烛光约覆盖周围三米；他靠近门外敲门鬼时，老人停止敲门并缓慢后退，童倩与幸存者得以趁隙撤离。烛火并非无代价：接近老人时，原本几乎不见消耗的烛身开始明显下降，杨间估计只能维持约一分钟，离开时实际已超时。第174章，他骑车接近仍亮路灯下的未知人影，鬼烛数秒燃去约五厘米，燃烧速度被他观察为敲门鬼时的数倍，遂立即后退；这只鬼不在敲门鬼鬼域内，具体身份不详。第175章，烛光又迫使钱万豪化成的鬼奴退开。故截至本批，鬼烛可逼退不同厉鬼或鬼奴，但燃耗会随接近对象显著变化，不能据此确认固定安全时长。（第171章至第175章）
