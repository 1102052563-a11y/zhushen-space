# 染血报纸

## 概述

染血报纸是第26章由杨间住处内、伪作其父亲的无脸男子持有的诡异物品。它表面是一份被猩红血液浸透的报纸，却能贴覆人的脸部并与皮肉般难以剥离，使受害者窒息；杨间以多只鬼眼的红光将其撕碎后，报纸又被男子收集、复原。第27章，残留在杨间手背的一张红纸碎片能压制一只鬼眼，成为他延缓鬼眼复苏的临时手段。（第26章、第27章）

## 来历与使用记录

- 杨间发现房内男子始终以报纸遮脸，报纸一角被血浸透。他确认父亲早已死亡后欲逃离，报纸立刻贴到脸上，黏连如皮肤，撕扯会引发剧痛。（第26章）
- 报纸收缩并在男子拉扯下加重窒息。杨间先以手背鬼眼、再以脸部及脖颈显出的鬼眼撕开多处裂口，最终将其撕碎脱困。（第26章）
- 杨间离开后，男子将碎纸逐片拼回，报纸恢复原状，血纸上多出杨间面容轮廓；数小时后男子脸也呈现杨间面容，但原应有鬼眼的位置一片空白。（第26章）
- 一块红纸碎片黏在杨间手背。撕下它会使鬼眼张开、全身鬼眼躁动；重新贴住该眼后，眼睛闭合、异样消失。杨间据此判断红纸能不完全压制鬼眼复苏。（第27章）

## 效果与限制

杨间观察到，红纸贴住一只眼后，他仍可用其余五只眼发动鬼域，且身体承受的副作用较六眼状态轻；红纸和鬼眼会反复显出轮廓，说明压制并不稳定。红纸来自未知厉鬼，杨间推测两只鬼的力量互相克制，但不知道为何此前鬼婴没有产生相同压制效果，也未取得整份报纸。（第27章）

## 现归属

完整染血报纸及持报男子在第28章前已离开杨间住处，去向不明；杨间仅暂持一块黏在手背、可压制鬼眼的红纸碎片。（第27章、第28章）

第74章，杨间连续动用鬼域后，红纸碎片彻底裂开、化作纸灰消散，第27章以来的压制失效。他判断真正具有压制作用的是报纸鬼留在碎片上的力量而非纸张本身，若要再获压制只能找到完整染血报纸或其所属厉鬼。（第74章）

第291至292章，赵磊留下印有自己脸孔的血红纸，随即引出持报纸、坐在长椅上的未知鬼物。其鬼域中有大量带不同人脸的飞舞红纸；纸张遮住杨间视野，令鬼眼无法展开鬼域，贴到无头鬼影上又能局部使其静止，显出对两种力量的压制。红纸还会扑向人的脸，疑似要以纸上脸孔换脸；后果未经实际证实。杨间以鬼影手臂击穿报纸后，散落物显示为一张张血淋淋、近似自活人剥下的人皮脸，但被击中的只是陌生女子脸制成的替身，持报鬼与赵磊均逃离。完整报纸现由该未知鬼物掌控，能否与第26章住处持报男子完全视作同一存在，原文未明说。（第291章、第292章）

## 第454章·作为误导鬼差的未实施设想

人皮纸提示“让鬼误判数量”后，杨间联想到自己持有的染血旧报纸：报纸曾改写李瑶等人的记忆，他因此猜测若能以同类手段改掉鬼差对鬼数量的判断，或可令其错误选择目标。这个思路仅是杨间基于人皮纸残句的推演；他没有接触鬼差，也没有实际以报纸尝试。完整厉鬼是否有记忆、报纸能否影响其判断、接触是否会使报纸被夺走或导致团灭，均不详。（第454章）

## 第463章·覆盖鬼差面部的未完成改写

鬼差经王江接触后只余鬼绳能力，草绳已开始向队伍全员垂下。杨间虽被压制而不能使用体内厉鬼，仍取出染血旧报纸，直接盖在鬼差的脸上；他明言时间不足以完成改写，却仍要执行这一动作。原文没有写明报纸是否成功贴附、是否已改变鬼差记忆，或其与压制数量、鬼绳复苏之间产生何种关系。该次是第454章设想首次实际接触鬼差的尝试，不可据此把效果视作已验证。（第463章）

## 第464-466章·取脸、写入数量与完成欺骗

黄子雅带白烛引走鬼绳后，覆盖鬼差面部的报纸缓缓滑落，同时取下其安详发黑的死人脸；这使杨间可在报纸上改写记忆。人皮纸提示必须让鬼误判自身鬼数为三，待其重启时再取走三鬼制成的棺材钉，令鬼从自身认知中得出数量为零。杨间反复写入相关内容，再将脸与报纸覆盖回鬼差头部。（第464章、第465章）

无头鬼影受压制而不能把脸拼回时，杨间让王江短暂松开鬼差，使其自行取回脸，之后立即重新压制。鬼差恢复行动和本方能力时，杨间取出棺材钉，鬼差随即舍弃卫景尸体、化作黑暗离去。该次说明报纸的篡改与取脸过程参与了数量骗局；写入持续多久、未在压制中完成是否仍会被鬼抹除，以及能否影响其他完整厉鬼，均不详。（第466章）

## 第564章·李瑶记忆的预设恢复暗号

杨间与李瑶对质时，以她无法找到“读过杨间资料”的具体位置为证，指出她对自己的爱意和相关记忆并非自然形成。随后他问出“1+1等于几”，李瑶立刻恢复被遗忘的记忆：她曾在平安大厦外的道路上遇见杨间和染血报纸，随后被改写记忆；醒来后才误以为自己读过其档案、主动爱上他，并协助其对付朋友圈。杨间说明该问题是自己改写李瑶记忆时预先留下的暗号，但没有描述报纸文字如何设置该触发条件。（第564章）

这一结果表明，至少对李瑶而言，染血报纸造成的记忆缺失可因预设口令被恢复，且恢复内容包括被遮蔽的相遇、篡改和其后行为的原因。李瑶恢复后情绪崩溃，显示此前植入的解释被推翻；但报纸篡改能否对其他人设置同类口令、能否由不同说话者触发、恢复是否完整永久，以及李瑶之后是否仍受其影响均不详。（第564章）
