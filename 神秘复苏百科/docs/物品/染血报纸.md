# 染血报纸

## 概述

染血报纸是第26章由杨间住处内、伪作其父亲的无脸男子持有的诡异物品。它表面是一份被猩红血液浸透的报纸，却能贴覆人的脸部并与皮肉般难以剥离，使受害者窒息；杨间以多只鬼眼的红光将其撕碎后，报纸又被男子收集、复原。第27章，残留在杨间手背的一张红纸碎片能压制一只鬼眼，成为他延缓鬼眼复苏的临时手段。（第26章、第27章）

## 来历与使用记录

- 杨间发现房内男子始终以报纸遮脸，报纸一角被血浸透。他确认父亲早已死亡后欲逃离，报纸立刻贴到脸上，黏连如皮肤，撕扯会引发剧痛。（第26章）
- 报纸收缩并在男子拉扯下加重窒息。杨间先以手背鬼眼、再以脸部及脖颈显出的鬼眼撕开多处裂口，最终将其撕碎脱困。（第26章）
- 杨间离开后，男子将碎纸逐片拼回，报纸恢复原状，血纸上多出杨间面容轮廓；数小时后男子脸也呈现杨间面容，但原应有鬼眼的位置一片空白。（第26章）
- 一块红纸碎片黏在杨间手背。撕下它会使鬼眼张开、全身鬼眼躁动；重新贴住该眼后，眼睛闭合、异样消失。杨间据此判断红纸能不完全压制鬼眼复苏。（第27章）

## 效果与限制

杨间观察到，红纸贴住一只眼后，他仍可用其余五只眼发动鬼域，且身体承受的副作用较六眼状态轻；红纸和鬼眼会反复显出轮廓，说明压制并不稳定。红纸来自未知厉鬼，杨间推测两只鬼的力量互相克制，但不知道为何此前鬼婴没有产生相同压制效果，也未取得整份报纸。（第27章）

## 现归属

完整染血报纸及持报男子在第28章前已离开杨间住处，去向不明；杨间仅暂持一块黏在手背、可压制鬼眼的红纸碎片。（第27章、第28章）

第74章，杨间连续动用鬼域后，红纸碎片彻底裂开、化作纸灰消散，第27章以来的压制失效。他判断真正具有压制作用的是报纸鬼留在碎片上的力量而非纸张本身，若要再获压制只能找到完整染血报纸或其所属厉鬼。（第74章）
