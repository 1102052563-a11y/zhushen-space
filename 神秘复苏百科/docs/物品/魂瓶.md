# 魂瓶

## 概述

魂瓶是杨间在赴大京市航班上取得的灵异容器。外层为近代样式的黑色陶瓶，破损处露出内层黄金瓶；杨间据黄金能隔绝厉鬼的既有经验，判断旧代驭鬼者曾把关押物藏入陶瓶，以避免外部察觉。行李主人万德路称该瓶出自乡镇古玩街附近、拆迁祖宅流出的古董，三万元购得；原始关押者、制造年代和瓶中厉鬼身份均不详。（第409章、第410章、第416章）

## 来历与归属

第409章，航班异常调查发现魂瓶外壳损坏、黄金内瓶暴露；杨间推测飞机颠簸令瓶盖脱落，放出或影响了瓶中厉鬼。陈义本想继续审问万德路，杨间因魂瓶可能指向更多旧关押物、也可能为自身身体修复提供线索而接管调查。第411章，郭凡被瓶内惨白手臂抓脸、试图拉入瓶口，杨间以无头鬼影把手臂逼回，再盖上瓶口并以金箔包裹，暂时持有。魂瓶由此随杨间进入培训基地，但瓶内是否仅有一只鬼，当时未获确认。（第409章至第412章）

## 容纳目标与限制

魂瓶曾显现一只惨白手臂，其从瓶口抓住郭凡面部，且能在压制郭凡体内厉鬼时施加影响；杨间以无头鬼影才迫其退回。此后瓶内的完整目标一直未显形，杨间只能确认其中仍有鬼。第456章，鬼差到来、杨间为增加本方鬼数量而打开黄金瓶；该鬼未在常规视野中出现。第458章，杨间发现空瓶后推测其已逃出，可能和鬼手一样藏在第三层鬼域，甚至混在五人队伍中。其外形、规律、是否与先前鬼臂为同一目标及能否被鬼差压制，均不详。（原文重复第456章·数量上的比较、第458章）

## 第456-459章·应急打开后的失控

白色鬼烛把鬼差引到杨间五人身边后，王江遭袭。杨间先释放黄金盒内鬼手，再打开魂瓶，意在把本方鬼数增至十、恢复不落单状态；鬼眼和无头鬼影随后确实恢复活动，但王江的勒痕显示袭击可能来自被鬼差带走的鬼绳，而非数量判定本身。鬼手继而复苏、被无头鬼影强制拼到杨间身上；与此同时魂瓶落地且瓶内已空，说明另一只鬼未被妥善限制。杨间没有停下搜救，改以白烛持续吸引鬼差，赌鬼差会压住逃出的瓶鬼；这是他的即时判断，不代表已完成收容。（第457章至第459章）

## 现状

至第459章，魂瓶瓶口打开、容器为空，瓶中鬼去向不明。杨间判断其仍在白烛范围附近，因鬼差被引到队伍周边而暂受压制或未能远离，但没有实际看见或关押该鬼。魂瓶是否还能再次使用、鬼是否会攻击队伍、其与鬼手和鬼差的数量计算关系均未写明。（第458章、第459章）

## 第513章·确认出土数量与失联去向

原宅后人罗永称，陵镇老宅地下室最初挖出的是五只瓶子，而非已查到的两只：两只分别卖出一万、三万元，另三只由未留联系方式的陌生人以现金每只十万元收购。杨间据此怀疑现金收购者可能是了解瓶中物的民间势力或驭鬼者；这是他的推测，不能确认买方身份、五只瓶子是否都为魂瓶，或其中是否均关押厉鬼。三只高价售出的瓶子以及低价售出的另一只的现状均不详。（第513章）
