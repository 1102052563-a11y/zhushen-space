# 小强俱乐部

## 概述

小强俱乐部是大昌市郊庄园内的民间驭鬼者聚集地，由董事长[王小强](../人物/王小强.md)负责。本地富商资助俱乐部及成员，驭鬼者收钱后为资助人提供一定保护；严力直言这实质上接近富商为自身买保镖。庄园环境豪华，有迎宾、酒吧和包厢，但杨间注意到迎宾的笑容与气息阴冷异常。（第76章）

## 成员与潜规则

- 严力带杨间进入时，张韩、叶俊等驭鬼者已在大厅。严力警告这些人“不正常”，并称自己曾在张韩手上吃亏。（第76章）
- 叶俊以鬼手印逼杨间承认所谓新人规矩：新人第一笔交易须被所有成员抽取五成利润，之后新人可从后来者交易中享同类抽成。（第76章）
- 杨间认为此规则是接近复苏的老成员不愿冒险处理灵异事件，转而勒索新人的手段。叶俊遭其重创后，其余成员因不愿拼命而沉默，说明俱乐部内部并无稳定的强制秩序。（第77章、第78章）

## 与无头鬼交易

严力在俱乐部约来孙俪红代表的公司，拟以一亿元收购黄金盒内无头鬼。杨间将交易改为以黄岗村灵异事件的解决结果，交换“驾驭第二只厉鬼、延缓复苏”的方法。消息公开后，张韩等成员被生存希望驱动；杨间留下黄岗村资料，约三日后行动。俱乐部是否支持交易、是否泄露严力消息，至第80章均未得到原文确认。（第79章）

## 第145章·针对杨间的表决

严力在市外死于鬼血复苏后，王小强召集十余名俱乐部相关人员开会，展示鬼血扩张的卫星图，并把十日内贺胜、张一鸣、叶俊、欧阳天、严力等五名成员的死亡损失归因于杨间。王小强提议抹除杨间；一名马姓股东认为俱乐部初衷是为资助者提供灵异保障、主动招惹驭鬼者会招致报复，但其他人以损失和未来利益为由支持，最终除其外的多数举手，他也随即附议。王小强称将在十日内动手。杨间持枪进入会议厅后，与会者拔枪并有人开价杀人，王小强答应每人一亿元，俱乐部与杨间的冲突由暗中计划转为当面危机。（第145章）

## 第146-150章·会议厅覆灭与重组意图

杨间进入会议厅后，王小强承认俱乐部清除他的一个目的，是维护不愿让其接管大昌市的既有利益。雷虎等武装人员动用黄金制特殊子弹、烟雾弹及手弩，却始终被杨间的鬼域幻象误导；多名武装人员和一名误中箭矢的股东死亡。王小强放出体内厉鬼后，杨间把在场活人转移、将厉鬼留在原会议厅，再借受无头鬼影控制的雷虎取下王小强头颅，随后清除大部分支持表决的股东。马有财成为唯一暂存活者，但被杨间换头控制。第150章，王小强体内厉鬼以尸体拼合令其复生；王小强判断其他股东死亡会使后续遗产分配妨碍资助，转而逼马有财三日筹集十亿元。俱乐部是否还能维持、现有成员和产业如何处理，截至本批均不详。（第146章至第150章）

## 出处

第76章至第80章、第145章至第150章。
