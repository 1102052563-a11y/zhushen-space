# 灵异论坛

## 概述

灵异论坛最早出现于全球性灵异事件爆发前约五年，起初是民间自发建立的兴趣论坛，内容多为各地鬼故事和网友亲历的怪事。灵异事件变为现实后，幸存者会通过这类网站寻找相同经历者与帮助，论坛遂聚集最早一批驭鬼者，并由交流平台逐步形成组织与势力。（第283章）

## 发展与规模

赵建国称，灵异论坛因建立早，笼络了亚洲最早的一批驭鬼者，现已是亚洲顶尖驭鬼者势力之一，涉及建筑、金融、医疗和运输等行业；亚洲财富排行榜前五十名富商中，近半与论坛成员存在不同程度关系。上述规模说明来自赵建国，并未公开成员总数、资金归属、分支架构或实际指挥方式。（第283章）

刑警总部建立亚洲分部初期，部长曾试图收编论坛，以取得早期驭鬼者力量；当时管理员拒绝，双方未能达成收编。总部其后以约定为条件默许其存在，论坛一年多来主要暗中发展、少有公开挑事；随着各地俱乐部和小团体增多，总部也未再集中处理。约定的具体内容、论坛是否始终遵守及总部的监管方式均不详。（第283章）

## 第281-282章·夜访杨间

论坛会员罗素一、林落梅和黄飞夜入观江小区，称管理员希望与刚解决饿死鬼事件的杨间会面。林落梅用诡异喊话寻找杨间，触发杨间反制：鬼绳限制林落梅，鬼影侵入三人身体，三人被鬼域转移至大昌市外郊区。黄飞说明论坛已向刑警总部备案，杨间则以十分钟内支付十亿元为条件测试组织的财力、情报和执行能力。

管理员叶真最终在八分四十二秒内将钱汇入杨间母亲账户，显示论坛能在信息保密情况下定位其亲属资料并快速调动巨额资金。杨间据此判断论坛是受官方默许的庞大组织，警告此类夜间接触若再发生即视为开战，随后放走三人。此次交锋只确认双方暂未结仇，论坛是否仍会招揽杨间及后续关系不详。（第281章、第282章）
