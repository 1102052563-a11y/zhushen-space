# 尚通科技

## 概述

尚通科技有限公司是第297章大昌市侦查队怀疑与国外势力渗透有关的公司，办公地点在尚通大厦。章华称它一年前作为招商引资项目进入大昌市，表面主营网络软件开发，背景来自国外一家知名企业。原文没有公开实际控制人、注册资料、雇员规模、所谓国外企业名称或正式违法事实，不能直接将其定性为灵异组织或袭击主谋。（第297章）

## 侦查线索

公司一直处于试营业状态，银行查账中资金长期亏损，侦查方因而怀疑它可能是皮包公司。警方在公司早期监控资料中通过人脸识别发现，曾在观江小区发动武装袭击的几名嫌疑人有进出该公司的记录。这个记录只证明相关人员曾出入，照片中的人并非公司在职员工，不足以证明公司直接组织或指挥袭击。（第297章）

## 与伪采访队的关联

特瑞率领的采访队在观江小区接触杨间后被武装特警控制。苏珊称自己受尚通科技资助参加采访；章华同时指出，混入队伍的三名雇佣兵在表面上与公司无关。杨间要求查封和抓捕，章华说明仍须取得确凿证据和搜查令。杨间要求以其名义申请搜查令，原文截至第300章没有写出申请结果或公司是否被搜查。（第296章、第297章）

第296章、第297章。

## 第323-325章·尚通大厦搜查

第318章，章华取得搜查令；第323章，他带杨间及公务人员进入尚通大厦。杨间以鬼域覆盖大厦后，发现仅有四十五层一间房因黄金隔绝而无法探查，遂直接带人前往。公司人员切断电梯、试图远程阻拦，仍未能阻止杨间以鬼域驱动电梯。顶层办公室内，外籍男子保罗自称公司老板；杨间确认他不是驭鬼者，却发现办公桌唱片、酒柜后黄金隔绝的保险室与磁带均有异常感。

保罗承认观江小区行动人员是公司员工，却称公司只接业务、不管伤亡；此说法是其自述，不能单独证明公司指挥袭击。他试图转让公司、赔偿和谈判换取存续，杨间拒绝，章华一行则开始开启酒柜后保险室。密室内容、实际违法证据、公司是否被查封及磁带声音的性质，在本批均未揭示。（第318章、第323章至第325章）
