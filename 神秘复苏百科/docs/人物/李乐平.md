# 李乐平

## 简介

李乐平是参加总部鬼差后续会议的驭鬼者，座位在杨间附近。其相貌平庸、身体恶化表面不明显，但人一旦离开旁人视线，旁人虽记得有这个人和其姓名，却无法在脑中留下具体面容；杨间确认自己并未感到灵异入侵或思维异常。原文未交代他的驻守地、刑警级别、加入总部时间和其所驾驭厉鬼的名称。（第495章）

## 能力

- **难以记住相貌的影响**：李乐平离开视野后，杨间与在场者无法回忆其具体脸孔，只能借桌前“李乐平”铭牌重新确认身份。该影响对驭鬼者同样有效，但原文未说明是他主动使用、长期常驻的能力，还是某只鬼的被动表现。（第495章、第498章）
- **灵异事件分析**：他听完鬼差资料后，判断所谓鬼差并非单独一只鬼，而是形如棺材的整片鬼域；既然从鬼域内部关押会触发重启，处置须从外部压缩并关押鬼域本身。王小明确认其结论正确，但这属于基于资料的分析，不等同于已经执行成功的方案。（第497章、第498章）

## 经历

#### 第495章·参加总部大会并询问杨间

鬼差后续会议在总部防空基地的圆桌举行时，李乐平已就座。他主动向杨间说明，与会者只是全国部分顶尖驭鬼者，其他区域仍需负责人驻守；杨间借铭牌确认其名字，却发现无法记住其样貌。李乐平见杨间已驾驭三只鬼，询问其是否未走总部的常规驾驭途径；杨间只答是巧合和运气，未交出具体方法。李乐平随即停止追问，未与杨间发生冲突。（第495章）

#### 第497-498章·提出从鬼域外关押

姜尚白取棺材钉的提议被暂搁后，李乐平指出鬼差在自身鬼域中会不断重启，不能在内部关押，须从外部动手。他进一步把鬼差界定为“鬼域就是鬼”，要求考虑如何收容数公里范围的鬼域。第498章，他从资料中准确指出鬼棺形状对应鬼域，并顺着王小明的提问推得：要将目标缩至可装箱的程度，必须用更强鬼域压制它。王小明赞同分析，却说明可用目标尚未找到；方案没有立即实施。（第497章、第498章）

## 人际关系

- [杨间](杨间.md)：会议相邻而坐；李乐平试探其驾驭路径，未获具体答案。（第495章）
- [王小明](王小明.md)：二人对鬼差本体及外部关押方向得出一致结论，王小明认可其分析。（第498章）
