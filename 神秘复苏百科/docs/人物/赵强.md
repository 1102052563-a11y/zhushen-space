# 赵强

## 简介

赵强是杨间的同班学生，也是第6章参与将杨间推往厕所的学生之一。第10章，杨间带张伟等人下楼时，在一楼楼梯口看见赵强和数名此前跟随方镜离开的学生僵硬站立。赵强身体不动，头却能以诡异姿势转过一百八十度，露出死灰眼睛；杨间将其认作尸体，并认为方镜可能已把这些人抛下拖延厉鬼。（第6章、第10章）

## 出处

- 第6章中，方镜号召三名男学生协助把杨间推入厕所；第10章杨间明确把赵强列为与方镜、段鹏、郑飞共同实施此事的“凶手之一”。（第6章、第10章）
- 一楼楼梯口的赵强笔直僵硬站立。杨间拍他后，其脑袋在身体不动的情况下转向身后，死灰眼睛麻木看人，随后又将头缓缓转回，继续背对众人站着。（第10章）
- 赵强没有继续攻击，杨间判断其暂时没有威胁，并带人绕开。其具体死亡时点、死因以及是否已成为鬼奴，均未在本批原文中确认。（第10章）
