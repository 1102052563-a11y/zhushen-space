# 金川

## 简介

金川是参加大阪除灵社宴会的骨干成员，年轻、西装革履，皮肤黯淡发黑，头形像遭外力扭曲，外貌显示其可能受到灵异长期影响。他在山崎的死亡影像出现后主动使用手摇式放映机，画面预示其将在居酒屋被同一双皮鞋、西裤所代表的未知厉鬼斩首；截至本批，预示尚未成为现实，他仍存活。其驾驭厉鬼、职位、年龄与具体能力均不详。（原文回跳第671章、原文回跳第672章）

## 经历

#### 原文回跳第671章·参与死亡影像宴会

山崎使用手摇式放映机后，画面显示其会在大阪街头被未知厉鬼斩首，金川在席间理性分析影像，指出街道尚有车辆和行人，说明山崎死亡时城市未必发生大规模灵异失控。他没有立即操作放映机，只在山崎死象结束后起身表明自己也想知道未来，成为第二名主动参与者。（原文回跳第671章）

#### 原文回跳第672章·获得居酒屋斩首死象

金川摇动放映机后，看见自己死在居酒屋：身体倒地、头颅离颈，普通人惊慌离开，角落里则有穿皮鞋、西裤的目标静立。死状与山崎完全相同，令在场者判断同一厉鬼正在针对多名除灵社成员。金川因影像愤怒又恐惧，暂时回到座位；真实死亡时间、是否能改变预示、该鬼为什么选中他均不详。（原文回跳第672章）

#### 原文回跳第673章·观察杨间的血池死象

杨间使用放映机后，影像没有出现皮鞋厉鬼，而指向大昌市郊外严力鬼血形成的血池。金川据此指出杨间未和除灵社成员遭遇同一结局，却无法判断杨间如何进入血池或具体死因。他没有借此直接与杨间冲突；山崎威胁惠子、双方爆发冲突时，金川等人只在旁观察。山崎被杨间杀死后，金川是否继续留在宴会、会否因死象采取行动不详。（原文回跳第673章）
