# 费总

## 简介

费总是第504章参与朋友圈临时会议的一名中年男子。面对杨间杀死高志强、队长计划可能改变资源分配的局面，他主张以现实风险而非帮会式报复评估杨间。原文只以“费总”称呼他，未给出完整姓名、企业、驭鬼者身份、能力或在朋友圈的正式职务。（第504章）

## 经历

#### 第504章·否决绑架并决定尝试招揽

贺天雄提出绑架杨间母亲时，费总认为这会激化矛盾而不能消除杨间本人的威胁；他指出杨间拥有类似李军的鬼域，一旦报复可迅速波及城市和各方亲属。费总承认此前接触李瑶与杨间时错估了后者的地位与价值，因而询问能否以足够价码补救、重新公关。会议最终要求李瑶接触杨间，条件由内部另议；若招揽失败，再讨论应对，且针对杨间的行动须经董事会同意。该次决定不代表朋友圈已放弃敌对准备。（第504章）

#### 第512章·推动表决并负责拟定方案

李瑶报告杨间拒绝两百亿元招揽后，费总说明这一价码本意是买下队长计划与鬼差事件关键期的一个月安稳，而杨间要求成为队长后再谈，说明已看穿安排。费总据此将选择归为放任杨间成为队长，或在计划前除掉他；他率先举手赞成后，会议多数通过。面对姜尚白的质疑，他以维持朋友圈亚洲第一驭鬼者势力、提高成员生存率为由说服对方，并提出请方世明执行。方世明同意把杨间引入下一场灵异事件后，费总承担准备具体方案的任务；方案内容、是否落实及其后果均不详。（第512章）
