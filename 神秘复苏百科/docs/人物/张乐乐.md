# 张乐乐

## 简介

张乐乐是高志强的专属接线员。第492章，副部长曹延华派女性工作人员查看十二号休息室后，确认她与刘小雨都躺在床上、思维异常，像被某种力量控制。原文没有描写她的外貌、年龄、接线经历、如何成为高志强的接线员，亦未说明她是否在高志强操控前发送了求助短信；第487章中向刘小雨求助的女同事是否就是张乐乐，至本批才由后续场景可对应为高志强的接线员，但短信实际由谁发送的细节仍未明说。（第487章、第492章）

## 经历

#### 第487-492章·在十二号休息室受控制

刘小雨到十二号休息室时，看见一名认识的女同事双目茫然、表情麻木，正站在床边脱衣；高志强承认先以鬼骗人影响一名女接线员，再把刘小雨诱来。第492章，工作人员实地查看后确认该受害者是高志强的接线员张乐乐。曹延华由此接受高志强确曾利用鬼能力侵害总部女性的说法，认为其行为是在找死。张乐乐当时能否理解自身处境、其身体是否受伤，以及高志强被限制后她是否立即解除控制，均未写。（第487章、第492章）
