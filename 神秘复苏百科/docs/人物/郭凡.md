# 郭凡

## 简介

郭凡是饿死鬼事件结束后，随赵建国进入大昌市第七中的一名国际刑警。第275章，他与十余名刑警一起抵达北校门，参与接收被棺材钉限制的饿死鬼。原文未交代他的驻地、驾驭厉鬼、具体职务或此前履历，只能确认他属于本次由总部派入城内的支援人员。（第275章）

## 经历

#### 第275章·接收现场的争执

赵建国率人用金箔、金绳和金箱关押饿死鬼时，郭凡听见杨间警告该鬼一旦苏醒，现场所有人都可能死亡。他认为支援队伍中任一刑警均属全国数一数二的人员，团队协作加上王小明计划与棺材钉后，处理S级事件并不困难；杨间则以众人此前无法进入大昌市、事件实际造成的伤亡反驳。郭凡继而批评杨间不听命令、不服管教，认为其负责大昌市会成为隐患。赵建国当场制止，指出郭凡不了解事件机密，也不应因杨间独自立功而流露不满。争论未影响当次关押完成，郭凡此后的任务、对杨间的态度是否改变均不详。（第275章）
