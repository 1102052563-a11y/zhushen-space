# 郭凡

## 简介

郭凡是饿死鬼事件结束后，随赵建国进入大昌市第七中的一名国际刑警。第275章，他与十余名刑警一起抵达北校门，参与接收被棺材钉限制的饿死鬼。原文未交代他的驻地、驾驭厉鬼、具体职务或此前履历，只能确认他属于本次由总部派入城内的支援人员。（第275章）

## 经历

#### 第275章·接收现场的争执

赵建国率人用金箔、金绳和金箱关押饿死鬼时，郭凡听见杨间警告该鬼一旦苏醒，现场所有人都可能死亡。他认为支援队伍中任一刑警均属全国数一数二的人员，团队协作加上王小明计划与棺材钉后，处理S级事件并不困难；杨间则以众人此前无法进入大昌市、事件实际造成的伤亡反驳。郭凡继而批评杨间不听命令、不服管教，认为其负责大昌市会成为隐患。赵建国当场制止，指出郭凡不了解事件机密，也不应因杨间独自立功而流露不满。争论未影响当次关押完成，郭凡此后的任务、对杨间的态度是否改变均不详。（第275章）

#### 第332章至原文第334章·拒绝撤离并进入酒店

郭凡在凯撒大酒店门口与冯全等待杨间，因杨间要求总部调离二人而指责其独揽功劳；他提到成为队长才有资格管辖其他刑警，透露自己希望通过处理事件取得功劳的倾向。面对杨间对记忆篡改的警惕，他拒绝在外等待，提出与冯全组队进入酒店。大厅内，郭凡见异常女前台即欲持枪试探，遭冯全阻止后仍主张优先清除可疑目标。其是否被篡改记忆、驾驭厉鬼、实际调查路线及之后安危均不详。（第332章、原文第334章·酒店的前台）

#### 第341-345章·四楼探查与三十一号房求援

郭凡与冯全在凯撒大酒店四楼迷路四小时，认为普通步行无法离开特殊鬼域。他曾因没有鬼烛而不愿依赖杨间，并把酒店内对象按“危害程度只有B级、因影响才列A级”判断，遂主张两名驭鬼者可不先摸清规则、直接关押；冯全同意后，两人循开门声进入与酒店房号不符的走廊，面对半开的三十一号房决定动手。第345章，郭凡通过卫星定位手机向总部求援，自称被困凯撒大酒店三楼三十一号房，冯全已失联。酒店存在人为信号干扰器，杨间据此怀疑求援可能是陷阱，未前往；郭凡求救时的实际位置、状态及信息是否出于本人均不详。（第341章、第345章）

#### 第369章·仍在酒店的推测

杨间带冯全离开酒店时询问郭凡下落。冯全只知二人此前分散自保，郭凡关掉通讯、躲藏起来，估计其未死且十有八九仍在酒店；杨间因与郭凡交情不佳、其生死不影响当前处置，决定暂不搜寻。该说法只是冯全基于分散时情况作出的估计，郭凡实际位置、生死及此前求援真假均未确认。（第369章）

#### 第382-383章·借灵位避袭并获外部救援

沈良根据仍在凯撒大酒店四楼的定位信号，命钟山从爆破后的外墙沿云梯入内。钟山在套房卧室找到郭凡的卫星手机、嵌有郭凡黑白遗照的木质灵位，以及僵跪在地的郭凡身体；郭凡的声音从遗照中传出，称自己被困期间至少四次被厉鬼盯上，故以这种“老方法”躲避袭击。原文没有说明灵位来历、能力名称或郭凡何时取得。（第382章）

得知冯全已由杨间救出后，郭凡虽对杨间会救冯全感到意外，仍决定离开灵位状态；他起身取走灵位，遗照随即化为无人的轮廓。走道脚步声靠近时，他判断不可能是活人，却以自己曾多次听见该声为由，与钟山直接撤向云梯。第383章，反锁房门未被打开，二人安全离开酒店。三日后沈良带他与童倩等人去大京市；其受困期间的完整遭遇、灵位后续和赴京任务均不详。（第382章、第383章）

#### 第523章·被列入鬼差行动外围支援

鬼差行动队失联后，沈良向总部说明已把郭凡与钟山、大京市负责人陈义安排在事件附近，作为鬼域外发生突发情况时的支援。该段仅有总部会议中的调度说明，郭凡没有出场、发言或展示能力；他是否抵达现场、是否接触鬼画或鬼差、是否看见鬼域消散和五个人影，均不详。（第523章）

#### 第525-527章·灵位失控后成为引鬼媒介

五名异常人影向城区接近时，郭凡按陈义安排取出木质灵位，以相片中的鬼替换自己，原拟借鬼的身份持烛引走目标。他前行后相片与身体的伤脸互换，随后意识被困在照片内，外面捧灵位者失去回应，未能按约点燃鬼烛。陈义没有接受钟山临时救援的建议，而将烛台刺入其肩、点燃鬼烛，以失控的郭凡吸引五鬼；钟山再以白烛把郭凡和五鬼向城外牵引。郭凡是否仍保有意识、能否重新控制灵位、其实际生死均不详。（第525章至第527章）

#### 第537章·确认与灵位之鬼重新换回

陈义换烛失败后，姜尚白确认郭凡已重新和灵位中的鬼交换回来，恢复为本人状态；陈义此前以为郭凡已在失控边缘，故对这一变化感到意外。姜尚白判断郭凡状态并不好，但距厉鬼复苏尚有距离。原文只记录了结果，没有交代交换是何时、如何自行完成，郭凡本人没有出场说明，也没有写他是否记得被困照片内的经历、是否仍能使用灵位或是否参与后续引鬼。（第537章）

#### 第567章·被列为引鬼计划失误者

王小明在总部会议复盘鬼差、鬼画联动计划时，指出郭凡、钟山、陈义负责引鬼环节；最终引走鬼画的计划因郭凡出现纰漏而失败，鬼画中走出的五只鬼有两只消失。该段是王小明对行动结果的归因，未说明郭凡当时的具体失误、是否主动违令或其后承担何种责任；郭凡本人未在会上回应。（第567章）

#### 第583章·对杨间疑死的反应

郭凡与钟山在住处巡查鬼画无果后收到杨间疑似死亡的总部短信。他把杨间死亡视作队长位置空出的机会，并以其一贯强势而流露幸灾乐祸；钟山提醒只是“疑似”、且少一名队长会增加鬼画行动危险，他仍未改变态度。杨间实际未死，郭凡是否知情及其后行动不详。（第583章）

#### 第587章·因鬼域异变收起庆幸

杨间把鬼域覆盖大半市区后，郭凡从窗外红光认出其特征，先前因杨间疑似死亡而生出的轻松随即消失。钟山提醒他不应过早高兴，郭凡面对可能全面复苏的鬼眼未再提出反驳；两人是否离开住处、是否接到总部命令及郭凡灵位状态均不详。（第587章）
