# 胡凯

## 简介

胡凯是章华的下属、负责梅山村特殊案件善后的年轻办案人员，未满三十岁，做事干练。他率数十名人员进入村内处理噩梦死亡、自杀尸体和村民冲突，知晓杨间负责人身份并对其压制中的厉鬼保持高度警惕。胡凯未展示驭鬼能力，具体职务与权力范围不详。（原文回跳第711章）

## 经历

#### 原文回跳第711章·率队进入梅山村处理善后

杨间通知章华后，胡凯因乡镇道路拥堵稍晚抵达梅山村，先向杨间道歉并确认灵异危险暂时解决。杨间要求其带更多人处理死亡善后，又命其将围堵江艳、张芬索赔的村民以涉灵异、可能异常为由带走隔离。胡凯理解车后备箱内可能压制着鬼，随即下令带走闹事者、封锁村口且不得让外人靠近。（原文回跳第711章）

他目睹杨间持枪鸣枪震慑村民，未干预其处置，而是优先配合保护压制现场。之后杨间要求建立梅山村事件档案，代号“鬼梦”，并调取杨孝和梁圆档案；胡凯安排下属查档，具体尸体善后、隔离期限和档案上报过程不详。（原文回跳第711章）

#### 原文回跳第713章·查出杨园园旧档

胡凯初查找不到梁圆档案，擅自为她补办户口和身份证；杨间要求改查梅山村旧人档案。胡凯随后找到四十年前出生、后来溺水死亡的杨园园资料：其外貌与梁圆完全相同，且是杨孝表妹。胡凯仍倾向于认为死人无法复活，不敢断言杨园园与梁圆的关系；杨间则认为灵异可以造就另类存活。（原文回跳第713章）
