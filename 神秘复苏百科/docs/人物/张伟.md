# 张伟

## 简介

张伟是杨间的同班同学，身材瘦高。因入学时老师把其姓名中的“张伟”误念成与杨间姓名连在一起的“杨伟”，他被同学取了“阳痿”的绰号，且这个绰号伴随其三年。第2章他与杨间、苗小善谈论网络灵异故事，第5章随学生从鬼域教室撤离，因楼板坍塌掉到下一层但自称无事。（第2章、第5章）

## 能力

- 本批未展示特殊能力。张伟保有一张小区集体上吊事件的照片，称来自朋友路过拍摄，但无法验证照片真伪。（第2章）
- 在楼板坍塌后跌落约三米至下一层教室，仍能自行起身并跟随逃离。（第5章）

## 经历

#### 第2章·课堂上的灵异闲谈

夜自习时，张伟见杨间整天打哈欠，先以玩笑追问其是否有“资源”，得知杨间前晚在看鬼故事后，又参与同学对市内灵异传闻的讨论。他拿出一张小区照片，照片中多个尸体挂在防盗窗上，头在防盗窗内、身体在外，姿势怪异；他说照片由朋友路过拍下，如今地点已经封锁。杨间没有立即相信灵异传闻，苗小善则询问世上是否真会闹鬼。周正进入教室后，张伟同样听见关于鬼的讲课，却表示完全听不懂。（第2章）

#### 第5章·坍塌楼板后的撤离

周正命学生逃走时，张伟随人群从五楼下楼。风化地面被踩塌后，他与苗小善掉到下一层教室，约三米的高度没有立刻致命；他摸着屁股抱怨有人推他。杨间随后发现苏蕾颈部被钢筋贯穿，催促仍能行动的人离开。张伟听到杨间催促后继续逃跑，并在杨间先走时以往日玩笑抱怨对方以后别想向自己要“资源”。原文未交代他在厕所鬼出现后的具体位置。（第5章）

#### 第9-10章·在鬼域中获杨间带路

张伟与苗小善等人被困在第二层走廊，拿手机灯从走廊一端走到另一端又折返，已重复数次而无法离开。杨间因鬼眼能看破鬼域，找到他们并说可带路。张伟没有立刻相信，要求杨间回答自己的英文名；杨间答出“沃德吉尔·英邦邦”后，张伟才确认对方不是鬼。第10章他跟杨间手牵手下楼，仍以玩笑缓和恐惧，称手上液体是“童子尿”。听见鬼婴怪叫后，他立刻随队加快撤离；第10章末鬼婴袭击王珊珊，张伟后续不详。（第9章、第10章）

#### 第16-20章·校门危机与离校

张伟在校门处清点一班幸存人数，称四十多人和老师仅余十三人；他虽焦躁害怕，仍随杨间继续寻找出口。众人接到诡异电话、黑衫老人逼近时，他一度哭求杨间想办法，惨白手掌也抓到其肩膀。杨间张开红光鬼域后，张伟随七人一同离开学校，抵达步行街才确认路灯、车辆和太阳均已恢复正常。杨间提醒音频危险时，张伟因恐惧不要自己的手机，并称可换用更旧的通讯设备；第20章末他仍存活，准备回家休息。（第16章至第20章）

#### 第29章·被父亲责罚后的疑虑

张伟从警方处回家后，父亲张显贵以为他犯了事，抽打并不准他回房睡觉。他站在阳台思考整夜，没有公开学校灵异经历，只反复怀疑究竟是谁陷害了自己。原文未说明他此后是否再接受警方问询。（第29章）

## 关系

- [杨间](杨间.md)：同班同学，互相以玩笑交谈；危机中杨间曾呼喊确认张伟情况。（第2章、第5章）
- [苗小善](苗小善.md)：同班同学，两人在楼板坍塌时共同掉到下一层。（第2章、第5章）
