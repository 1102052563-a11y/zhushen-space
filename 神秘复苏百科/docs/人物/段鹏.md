# 段鹏

## 简介

段鹏是杨间的同班学生。第6章，方镜以“牺牲一个人”拖住厕所鬼为由煽动其他学生协助，段鹏是抓住杨间并将其推往厕所的三名男同学之一。杨间被惨白手掌拖入黑暗时，抓住段鹏与另一名同学的手臂，把两人一同拉入厕所。第7章，段鹏在无尽黑暗中被惨白手掌抓住手腕并拖走；截至第10章，其后续不明。（第6章、第7章）

## 出处

- 方镜要求同学协助将杨间推进厕所，称若厕所鬼出来所有人都会死。段鹏在厉鬼威胁和这一理由下与另外两名男同学上前，和方镜共同推杨间；四人合力使杨间退到厕门。（第6章）
- 被拖入厕所后，他和郑飞试图沿墙摸索大门，却发现左右都是没有尽头的冰冷墙壁。杨间再出现时，段鹏反过来质问杨间为何把自己拖进来；杨间指出最初是他们先谋害自己，并打了他一拳。（第7章）
- 段鹏承认自己不想死，又称是方镜让他这样做，认为死杨间一人好过全员死亡。随后惨白手掌抓住其手腕，把他拖向黑暗；他向杨间呼救，杨间没有播放音频救援，只看着他和手掌一同消失。原文未交代其尸体或结局，状态记为不明。（第7章）
