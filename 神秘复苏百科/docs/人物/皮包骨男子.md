# 皮包骨男子

## 简介

皮包骨男子是与山崎、田野一同进入神户市的除灵社成员，身形消瘦至皮包骨，杨间和长泽都认为其异常体态可能受厉鬼影响。他未报出姓名、未说明驾驭厉鬼或实际能力，也没有直接参与与杨间的冲突。山崎在十字路口遭半腐烂头颅厉鬼追杀后，皮包骨男子未再出现在原文，生死和去向不详。（原文回跳第666章、原文回跳第668章）

## 经历

#### 原文回跳第666章·随山崎前往索取关押物

敲门鬼被杨间关入箱体后，皮包骨男子与山崎、田野驾车抵达现场。长泽认出山崎，却不认识此人；杨间根据其瘦骨嶙峋的外貌判断他很可能受厉鬼影响。山崎试图以除灵社名义接手箱内敲门鬼，杨间拒绝并警告三人不要抢夺，山崎随后选择暂时退让。皮包骨男子全程没有发言或展示能力。（原文回跳第666章）

#### 原文回跳第666章·进入十字路口调查后下落不明

山崎为避免空手而归，带田野和皮包骨男子到杨间此前点燃鬼烛的十字路口，称先处理一件残余灵异事件。遇见背对行走的半腐烂头颅厉鬼后，皮包骨男子建议“就拿它交差吧”；田野随即被厉鬼瞬间斩首，山崎感到自己被盯上而逃走。原文没有说明皮包骨男子是否同行逃离、是否被袭击或动用能力。（原文回跳第666章、原文回跳第668章）
