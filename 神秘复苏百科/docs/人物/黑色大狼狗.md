# 黑色大狼狗

## 简介

黑色大狼狗是杨孝在长期灵异试验中留下的成果，能出现在梅山村噩梦的树林和未损坏的小木屋前。它全身乌黑、体型高大，眼神没有普通野兽的凶性而显得诡异；杨孝的梦中残留计划让它取代噩梦鬼、成为新的梦境源头。它当前尚未取代任何鬼，具体灵异能力和是否保有自主意识均不详。（原文回跳第705章、原文回跳第706章）

## 来历

杨间从现实树林中挖出数十条狗骨，推测杨孝生前曾在此饲养、杀死大量狗进行灵异尝试。梦中杨孝称这些尝试最终得到黑色大狼狗，显示它并非普通幸存动物；但杨孝没有说明具体试验方法、其余狗死亡原因和狼狗如何进入梦境。（原文回跳第690章、原文回跳第705章、原文回跳第706章）

## 取代计划

杨孝残留要求杨间在现实给噩梦鬼分尸留显著灵异记号、将尸体拼合并在不隔绝感应的情况下看守至半腐烂。若计划成功，杨孝残留会消失，黑色大狼狗将取代噩梦鬼成为新噩梦源头；杨孝认为动物比人稳定，拒绝以梁圆等人作为替代对象。（原文回跳第706章）

这个方案意味着狼狗可能承接噩梦能力，但原文没有说明它会否伤人、能否被杨间控制或取代后的规则。杨间仅得到方案，尚未开始现实操作，不能确认黑色大狼狗是否值得信任。（原文回跳第706章）
