# 王泉

## 简介

王泉是总部派往大学处理鬼画异常的支援小队负责人，体形略发福、面容严肃。他带领穿西装的特殊人员和车辆快速抵达篮球馆，表示已经看过鬼画档案，并接受杨间对现场工作的指导。（第553章、第554章）

## 出处

- 王泉到场后主动介绍身份，并带来校方涂校长协调事务；他确认了解鬼画档案，故杨间没有重复说明既有背景。（第554章）
- 他按杨间要求安排直接接触过寝室油画的刘紫离开观察，又接下排查美术馆的任务，并要求涂校长十五分钟内疏散宿舍楼学生。（第554章）
- 孙于佳从四楼坠下获救后，杨间命现场人员询问其楼内经过，并只带王泉和两人进楼查看。王泉实际入楼后的行动、伤亡和是否发现油画，均未写明。（第554章）

## 第555-559章·随行排查与维持队伍纪律

王泉带两名人员随杨间进入宿舍楼，先按杨间确认安全的结果安排一至三层排查；当四层以上显出鬼画影响后，他立即以对讲机禁止任何人靠近四楼。面对杨间提出“解决灵异事件优先于救人”的判断，他认可普通人进入已无把握区域只是送死，并在鬼眼受压、楼梯异变时服从杨间的专业判断，没有以外行意见干预。（第555章）

楼层重叠使队伍错入六层、下行楼梯消失后，王泉提出用特制黄金登山绳从楼外下降，但杨间以无头鬼影开出通道，队伍由此回到真实四层。王泉同意分头寻找鬼域与现实的连接点，带两名人员戴黄金手套摸墙查房；他发现滞留学生后只记录寝室号、劝其留在原处，未强行带离已失去退路的楼层。红衣画中人进入402寝室时，他捂住队员口鼻避免惊叫，命其后撤并用卫星定位手机向杨间报告目标位置；在杨间同时按住画框、尚未决定先限制何物时，又制止队员妄测负责人会弃人试探，并以配枪维持可能失控者的纪律。402内学生结局、王泉小队后续伤亡及他能否离楼均不详。（第557章至第559章）

## 第560-561章·冒险送袋、获救与押送鬼画

杨间等待红衣鬼的反应期间，王泉接受其指令，带着可装画的大型折叠袋独自绕过堵在走道的鬼。他明知普通人被近距离盯上无从获救，仍在杨间催促下跑过鬼身侧，最初未受袭；回头确认时，红衣鬼却转向并跟随他。王泉没有停下，按杨间命令继续前行，最终在苍白手掌将碰到肩膀前由杨间抓住鬼手、压制目标，因而幸存。此事说明他曾触发鬼画相关袭击，但触发原因尚未确认。（第560章）

鬼与画重新合一后，王泉和两名队员按杨间要求将巨幅鬼画装进袋子，并在外层反复包裹，确保不留缝隙；他同时派两人救出楼内滞留学生。杨间询问袭击经过时，王泉先以自己回头看鬼作答，后又提出恐惧、或“看见鬼后恐惧”的猜测；杨间逐一指出仍有不合条件的幸存者，要求他把全部细节作为分析材料上报总部。总部下令立刻运走鬼画后，王泉带队接手押送。离开前他把一双金黄色工作手套交给杨间，以供其遮住并隔绝鬼手；押送是否安全到达、总部对规律的结论均未写明。（第561章）
