# 阿飞

## 简介

阿飞是第27至28章在公共厕所外拦住杨间的青年混混之一。他以“交朋友”和借钱为名，伙同数人持刀勒索杨间；杨间反过来以鬼眼与鬼域制造的异常吓退他们，并从其身上取走钱款，称作借用。（第27章、第28章）

## 出处

- 阿飞在厕所外搭住杨间肩膀，自报姓名后要求借钱；同伴以小刀威胁，显示所谓借钱实为勒索。（第27章）
- 杨间发动红光鬼域，令阿飞等人奔跑后仍回到厕所前，又在他们面前突然出现。阿飞将搭在肩上的冰冷身影误认为鬼，和同伴逃散。（第27章、第28章）
- 鬼域散去后，阿飞吓得躲在垃圾桶后。杨间以其原话要求“借点钱”，取走其一百余元；阿飞不敢要求归还，并发誓不再讹人。（第28章）
