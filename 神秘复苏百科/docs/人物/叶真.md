# 叶真

## 简介

叶真是灵异论坛现任管理员，约二十出头，常穿休闲装，活动地点为大海市一栋摩天大楼顶层会议室。刑警总部调查显示，他成为驭鬼者已超过一年，曾独自解决一件A级灵异事件；赵建国认为，若非王小明推动总部实力迅速扩张，他称作亚洲第一驭鬼者也不为过。其驾驭厉鬼、事件细节与能力代价均不详。（第281章、第283章）

## 经历

#### 第281章·接受杨间的试探

罗素一三人在大昌市被杨间制住后，以电话联系叶真。杨间自报身份并限其十分钟内支付十亿元，随后捏碎电话；叶真起初将此视为无端威胁，甚至想订票前往大昌市镇杀杨间。身旁中年男子则解释，杨间未给账户却要求限时付款，是在测试论坛能否查到其个人信息、调动财力，并以三名会员作为筹码展示强硬。叶真接受解释后取消敌对打算，称欣赏杨间并愿交朋友，命人汇款。

论坛最终在八分四十二秒内把十亿元汇入杨间母亲账户。叶真将此定为给杨间的面子，而非畏惧；他仍以“前方绝无敌手”自许。叶真是否知道罗素一三人被鬼域转移、是否会亲自赴大昌市及其与杨间后续关系，均不详。（第281章）

#### 原文回跳第721-722章·被列为总部失窃案嫌疑对象

柳三向杨间转述：秦老进入鬼画期间，总部遭拥有高阶鬼域者入侵，多只关押厉鬼与饿死鬼疑被带走，叶真因具有相应能力而成为当前怀疑对象。柳三没有提供现场证据，也明确只是“怀疑对象”；叶真是否入侵总部、是否带走饿死鬼和棺材钉均未证实。（原文回跳第721章、原文回跳第722章）

杨间拒绝就此判断或替总部试探叶真，认为柳三的提问可能意在挑拨两名顶尖驭鬼者冲突。叶真本批没有出场、没有回应，也不能把柳三转述当作对其行为的定论。（原文回跳第722章）
