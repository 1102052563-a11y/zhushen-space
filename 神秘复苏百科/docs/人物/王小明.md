# 王小明

## 简介

王小明是某研究院主任，约二十五岁，穿白大褂；他也是小强俱乐部王小强的哥哥。第115章，他在黄岗村外公司营地现身，自称可为删减档案一事负责，并被赵建国指出是国际网站研究灵异事件的论文作者“布鲁斯·皮”。（第115章）

## 立场与作为

王小明承认国际刑警档案由自己下令删除，而非吴越越权操作。他称此举是为防情报泄露、回收鬼棺，并以国内灵异事件与死亡不断增加、灾难可能全面爆发为理由，把行动中驭鬼者的伤亡视为值得的个人牺牲。他的判断以国家、民族和长期压制灵异事件为优先，明确认为鬼棺回收所得价值大于死去行动者的代价。（第115章）

## 已知信息

- 王小强：王小明称小强俱乐部负责人王小强是自己“不成器的弟弟”。（第115章）
- 布鲁斯·皮：赵建国称王小明在国际网站使用该笔名，曾发表《厉鬼的力量无法影响黄金》等论文，属早期研究灵异事件并提出重要结论的专业人士。（第115章）
- 价值立场：面对杨间的追问，他回答能解决灵异事件的国际刑警比维持公司运转的总经理更重要，并称若杀死普通人可换来灵异事件解决，自己会毫不犹豫。（第115章）

## 当前冲突

王小明要求杨间放下枪并接受其为国家所作的解释；杨间则要求他按自己的价值衡量标准亲手杀死吴越。杨间以是否执行该要求为条件，决定自己是否加入国际刑警并是否追究删档。至第115章末，王小明未给出答复。（第115章）

## 第116-118章·履行交易与回收鬼棺

王小明在第116章直接向吴越开枪，称杨间处理大昌市灵异事件的价值远高于公司总经理，最终杀死吴越。杨间据此接受其删档是为大局的说法，承诺日后参加国际刑警考核。王小明随后以五亿五千万收购鬼棺，对冯全则拿出实验室鬼烛交换；他急于将鬼棺和冯全带回实验室，坚称即使棺中有鬼、研究存在风险，也值得寻求研究突破。（第116章至第118章）
