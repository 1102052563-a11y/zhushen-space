# 阿海

## 简介

阿海是王小强雇佣行动组成员，受孔峰命令在观江花园内排除跟拍行动组的张伟。他以匕首袭击张伟后遭到反击。（第195章）

## 经历

#### 第195章·袭击张伟后中枪

张伟在售楼部外直播并跟上孔峰、黛西等人，行动组担忧样貌和计划泄露。孔峰经耳机确认杨间在别墅后，命阿海“解决”张伟。阿海先笑着拍张伟肩膀，抢走并摔碎手机，继而捂住其嘴，将锋利匕首刺入其颈部，把人拖进绿化带；他以为普通人已被轻易杀死，甩去刀上血迹返回。张伟却带着颈部伤口和手枪追出，近距离连续开枪击中阿海。阿海倒地、冒汗颤抖，黛西判断伤势严重；孔峰称三枪均命中胸膛要害，决定不救他。其后是否死亡不详。（第195章）

#### 第197章·行动组覆灭后的状态

阿海在第195章中三枪后被孔峰放弃救治。第197章，杨间将孔峰等行动人员困入鬼域并逐一杀死，但原文未点名阿海是否仍在场，也没有写到其尸体或脱离现场的经过。阿海胸膛要害中弹且未获救治，其生还可能很低；不过仅凭行动组大部分人员覆灭不能确认其已死亡，故状态保留为不明。（第195章、第197章）
