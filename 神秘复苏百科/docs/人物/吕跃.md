# 吕跃

## 简介

吕跃是鬼邮局二楼的男性信使，身材高大健壮，携带枪械，在二楼信使中具有一定威慑力。他反对杨间撕毁集体红色信件，认为这样会强迫全层信使承受厉鬼袭击；其房间号、送信次数、进入邮局前身份和是否驾驭厉鬼均不详。（原文回跳第783章、原文回跳第784章）

## 经历

#### 原文回跳第783章·反对撕毁红信

杨间取得红色信件、得知任务地址是大川市明月小区7栋301室后，吕跃与其他二楼信使均主张先讨论任务。他担心红信难度极高，却更不愿撕信直接触发厉鬼袭击，称自己可通过拖延送信争取一个月的安全期。杨间无视反对，仍将红信彻底撕碎。（原文回跳第783章）

#### 原文回跳第784章·开枪袭击杨间后被杀

红信碎裂、七间房门开启后，吕跃从背后近距离连续枪击杨间，试图先杀死使全层陷入危机的人。子弹未能使杨间受伤，他随即被杨间以发黑鬼手掐住脖子、折断颈骨，再从二楼走廊抛入下方黑暗，坠落后无声无息。（原文回跳第784章）

吕跃当场死亡的判断来自颈骨断裂、坠入黑暗与后续无回应；坠落空间是否为一楼、尸体是否会受邮局灵异影响均不详。他的死亡使二楼剩余信使进一步放弃正面反抗杨间。（原文回跳第784章）
