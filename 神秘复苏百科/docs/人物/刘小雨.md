# 刘小雨

## 简介

刘小雨是国际刑警亚洲分区的接线员，原为周正专属接线员。第21章她通过周正遗留的卫星定位手机与杨间通话，得知周正死于厉鬼复苏。她因杨间以“张伟”假名和玩笑式回答而失去冷静；第22章经队长赵建国接手后，被安排继续担任杨间的专属接线员。（第21章、第22章）

## 能力

- 接线与记录：可通过卫星定位手机接听一线人员报告，追问事件、身份和位置，并按程序联系当地警方或安排调查。（第21章）
- 档案核验：第25章，她通过杨间档案确认其真实姓名、出生地、生日和单亲家庭资料，识破其先前冒充张伟的说法。（第25章）

## 经历

#### 第21章·接通周正遗留手机

杨间在公交车上接通卫星定位手机后，刘小雨依程序要求“周正报告情况”。杨间告诉她周正死于鬼婴破腹而出，却故意用“生孩子生死了”等说法描述。刘小雨起初不接受，质问卫星定位手机为何在陌生人手中，并以国际刑事案件和拘留四十八小时要求其如实报告。杨间不断以“呵呵”回应，并报出张伟的名字挑衅，刘小雨最终失控回骂，通话被赵建国注意到。（第21章）

#### 第22章·接线失误与专属任务

赵建国批评刘小雨把个人情绪带入特殊事件，并要求她理解幸存者可能精神异常。她随后记录杨间复述的高校事件，内容包括鬼域、敲门声、厕所鬼和电话诅咒。赵建国判断杨间已成为驭鬼者，考虑招揽其负责大昌市治安，要求刘小雨后续跟紧杨间、担任专属接线员。刘小雨对杨间未经训练便可能被招揽表示担忧，但执行队长安排。（第22章）

#### 第25章·识破杨间身份并警告异常

刘小雨派人找到张伟后，查阅档案确认通话者应为杨间，并在卫星电话中逐项念出其资料，批评假名耗费调查资源。杨间正和身旁自称父亲的中年男子说话时，她从档案发现杨间父亲在其小学五年级已因车祸去世，母亲没有再婚，立即判断住处异常，催促杨间离开。她的档案信息使杨间注意到黑白遗像与身旁男子的矛盾。（第25章）

## 人际关系

- [周正](周正.md)：原专属接线员与一线刑警关系；周正死亡后，她通过其手机首次获知高校事件。（第21章）
- [赵建国](赵建国.md)：队长，曾批评其失控，并安排其继续负责杨间。（第22章）
- [杨间](杨间.md)：后被指定为其专属接线对象；双方因假名和沟通方式冲突，但她在第25章提供关键风险提醒。（第21章、第25章）
