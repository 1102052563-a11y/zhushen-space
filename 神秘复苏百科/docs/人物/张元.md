# 张元

## 简介

张元是警局警员，二十七岁。第154章，他被同事发现困在警局大厅一处可映照人影的外墙面板内；从监控回看，他最早在当天早上已出事。其余同事直到十分钟前才从镜中发现他，说明现实中的异常并未立即被察觉。（第154章）

## 经历

#### 第154-155章·被镜中鬼替代

刘建国赶到大厅后，张元仍坐在镜内，已写下许多提示信息，但来往人员此前或因匆忙、或只把他当作不起眼的影子，没有理解求救内容。杨间判断他听不到现实交谈、却能看见外界，且其处境与上官云曾被镜中鬼困住时相同。杨间进一步指出，张元被困的同时，现实中必有另一名“张元”在警局活动；监控随后证实该替身中午曾在食堂露面，半小时前也在大厅出现。第155章，杨间在四楼玻璃后见到一名穿制服、面容惨白且酷似张元的人正走向五楼，确认镜中鬼尚未离开。张元能否被救出、镜中替身之后如何处置，截至本批均不详。（第154章、第155章）

#### 第156-157章·替身被识破后仍未获救

假张元被锁定在档案室后，刘建国开枪击中其手臂却未见流血或痛苦；杨间随后以“你根本不是张元，你是鬼”点破身份，使这个现实替身当场消失。鬼并未被消灭，而是重新显现在窗玻璃倒影里，继续向楼下移动。杨间说明镜中人只能由镜中鬼进入其所在位置，无法锁定鬼便无法救人；上官云上次获救依赖鬼与人同处一面镜子、苗小善又成为交换诱饵，本次没有相同条件。张元仍被困在镜内，获救机会被杨间判断为很小，状态维持不明。（第156章、第157章）
