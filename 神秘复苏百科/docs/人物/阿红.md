# 阿红

## 简介

阿红是总部关押鬼差行动队的五名成员之一，与李军、柳三、苏凡和徐一平一同进入鬼画鬼域内重现的黄岗村。第519章柳三向杨间转述队内人员时明确点出其姓名；该次行动原计划利用鬼画鬼域压制鬼差后实施关押，但黄岗村内每栋黑暗房屋都可能有鬼，队伍改为逐栋排查。原文未说明阿红的外貌、性别、职务、所驾驭厉鬼、能力或此前履历。（第519章）

## 经历

#### 第519-522章·作为黄岗村行动队成员参加搜索

阿红随五人进入黄岗村后，行动队已查探多栋有鬼房屋，并在一栋三层民房继续搜索。李军鬼火复苏加重，柳三先上二楼探路，苏凡则以手机游戏判断二楼危险；随后队伍发现五具外貌皆为柳三的残破纸人，并在另一房间取得一张背景映有五人身影的老人遗像。原文将阿红计为照片中右侧三道模糊轮廓之一，却没有写她在这些行动中的单独动作、发言或是否接触遗像。她是否持有保命手段、是否受纸人和遗像异变影响，均不详。（第520章、第521章、第522章）

#### 第523章·失联后以异常形态走出鬼域

总部会议收到报告时，阿红与其余四人已失去联系约三分钟；鬼画与鬼差的鬼域随后一同消散，远处有五个人影缓缓走出。杨间最初只能辨认李军和柳三，未能逐一指认其余三人，因此不能直接确认其中一人必为阿红；但总部失联名单与行动队名单均包含她。五人手掌均有大小一致的灰白空洞，杨间点燃白色鬼烛后，五人同步停步看向烛火，他据此判断五个走出的人“全部都是鬼”。阿红本人是否已死亡、被替代，或仍处于其中某种状态，原文均未确认。（第523章）
