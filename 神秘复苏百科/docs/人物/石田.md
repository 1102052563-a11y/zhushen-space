# 石田

## 简介

石田是大阪宴会中的除灵社骨干成员。藤原的死亡影像出现后，三岛继续指定石田操作手摇式放映机；放映出的画面显示石田将在飞机机舱内被斩首，头颅滚到一名穿皮鞋、西裤乘客脚下。该影像把未知厉鬼的威胁延伸至飞行途中，但截至本批并非既成事实。石田的外貌、能力、职务和乘坐航班的时间均不详。（原文回跳第672章）

## 经历

#### 原文回跳第672章·获得飞机内的斩首死象

三岛让石田接替藤原摇动放映机，结果仍是相同的无头尸体和滚落头颅，但场景变为飞机机舱。画面中石田的尸体坐在座位上，头颅在机舱内滚动，最终停到穿西裤、皮鞋的乘客脚旁；未知厉鬼再度以局部肢体进入画面。石田没有在本批采取自救措施，影像后也未显露灵异能力。（原文回跳第672章）

这一死象说明该鬼可能在非固定地点出现，甚至能进入正常飞行环境；但放映机只能展示有限片段，无法确认它如何登机、是否依赖石田自身行程，或影像是否会因未来改变而失效。石田在本批结束时仍然存活。（原文回跳第672章）

#### 原文回跳第674章·被山崎遗鬼锁定

山崎被杨间杀死后，其体内长发尸体鬼立刻复苏，能无视杨间三层鬼域、短暂抵抗鬼手压制。杨间不愿替除灵社处理山崎遗留问题，直接离开大阪；尸体鬼随即转头面向石田，腐烂脸下的眼睛转动并露出诡异笑意，灯光开始变暗。它以极快速度向石田爬去，原文至本章末仅写到除灵社骨干出手应对，未写石田是否受伤或死亡。（原文回跳第674章）
