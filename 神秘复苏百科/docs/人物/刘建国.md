# 刘建国

## 简介

刘建国是张伟被拘留所在地的警局队长，警员多称其“刘队”。杨间为联络赵开明并接出张伟到访警局时，他正处理大量刑事、治安与需要上报的特殊事件，认为刑警相关事务应优先办理。（第152章）

## 经历

#### 第152章·协助联络与核查张伟

刘建国向杨间说明自己有赵开明的卫星定位手机号码，但平时不敢贸然联系这位难以相处的刑警；在杨间提出请求后，他仍交出号码。对于张伟的释放，他先明确若涉及重大刑案不能通融，随后命人核查。警员查明张伟是在扫黄行动中被带走、没有案底，笔录显示是受人引诱，遂可办理手续离开。刘建国同时请求杨间在能够处理时继续协助大昌市的特殊事件，并承认赵开明并非认真办事的人。（第152章、第154章）

#### 第154-155章·组织警局镜中鬼应对

获知张元被困在大厅反光墙面后，刘建国接受杨间将其定性为特殊事件的判断，立即疏散大厅人员、拉起警戒并禁止警员靠近。他先询问张元出事时间，得知监控显示事件可能从当日早上开始；听到杨间判断现实中还有假张元时，他命人调监控、把该“张元”视为敌人并要求全员谨慎。杨间在四楼发现目标后，他又同意以四楼为起点封锁楼道、电梯并逐层排查。其组织措施旨在减少接触与伤亡，但假张元和张元本人的最终结果截至本批未明。（第154章、第155章）

#### 第156-157章·封锁档案室与放走张伟

警员通过监控锁定假张元进入五楼档案室后，刘建国命人封锁房间、警戒四楼并留意出口。他面对从档案室走出的假张元，以配枪警告无效后开枪命中其手臂；弹孔无血、对方也无痛感，使其确认对方并非正常警员。杨间到场阻止继续射击，点破伪装令鬼暂时消失，但说明这不是彻底解决，张元也未必能被救回。其后杨间要将张伟带出拘留所、把镜中鬼引离警局，刘建国命警员按特殊事件优先配合，并承认此类事件中自己也须让步。（第156章、第157章）
