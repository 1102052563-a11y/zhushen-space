# 孙瑞

## 简介

孙瑞是大汉市负责人，二十六七岁，驾驭两只厉鬼。其脸色蜡黄发黑、常咳嗽，走路需拄金色拐杖且一只脚跛，身体状态接近病人；但谈吐清醒、对辖区风险保持警惕。队长计划前，他自认两鬼驭鬼者尚属顶尖，现因杨间等队长级人物崛起而选择安守大汉市。（原文回跳第734章）

## 经历

#### 原文回跳第734章·得知杨间将赴大汉市

孙瑞通过卫星定位手机得知杨间和李阳将抵达大汉市调查。他担心杨间不会无故跨市，判断其来访可能与大汉市既有异常有关，决定亲自接机、确认目的，并要求助手提前安排。其所称“大汉市一定出了问题”只是基于杨间行动的判断，尚未指明具体事件。（原文回跳第734章）
