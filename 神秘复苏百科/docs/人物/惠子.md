# 惠子

## 简介

惠子是除灵社翻译组实习生，短发、外表年轻，穿水手服式服装。三岛称她是大学外语系学生，安排其从杨间抵达神户起贴身翻译、协助行动并照料生活；杨间判断这既是公关和拉拢，也可能包含监视意图。她不是驭鬼者，只是被普通人身份直接投入高危灵异行动，具体语言水平、家庭和进入除灵社经过均不详。（原文回跳第649章）

## 性格与表现

惠子谨慎、服从且求生欲强。她初见杨间时拘谨行礼，知道自己命运可能与外援绑定；田野利用鬼域杀害陪同女子时，她抓住杨间衣角求保护。杨间没有承诺额外救援，只说明她的生存取决于运气；她仍在事后尽力翻译命令、安慰美岛并跟随队伍进入神户市。（原文回跳第649章、原文回跳第650章、原文回跳第651章）

## 经历

#### 原文回跳第649章·被安排贴身协助杨间

三岛在机场欢迎会中让惠子向杨间行礼，宣布她从此负责协助工作与生活起居。惠子接受命令，意识到自身已被交给陌生外援。杨间没有明确拒绝，却指出普通人卷入灵异事件很容易死亡；惠子仍以“请多关照”回应。她未在本章展示灵异能力。（原文回跳第649章）

#### 原文回跳第650章·田野袭击中的幸存者

田野在机场小店的鬼域中将两名作陪女子分别摔断脖颈、嵌入水泥地，并威胁下一位目标是惠子。惠子惊恐求助，杨间当时没有立即出手救人，但在田野进一步挑衅后以鬼手压制对方，令其鬼域消失。惠子因此活下，见证杨间无需起身便可制伏田野，之后更依赖其保护。（原文回跳第650章）

#### 原文回跳第651-652章·翻译并进入神户鬼域

杨间带三名除灵社协助者进入神户市后，惠子负责逐句翻译其命令和他人回答。她首次见到鬼童从行李袋走出，被其红眼注视后瘫坐在地；杨间说明鬼童受自己控制、不会伤害同伴后，她强迫自己恢复工作。美岛不敢靠近感应到鬼的房屋时，惠子又代为安慰，强调只有展现价值才可能获杨间认可。木屋探查开始后，她留在外面，是否能在房屋异变中保持安全不详。（原文回跳第651章、原文回跳第652章）

杨间从木屋平安撤出后，惠子立刻上前确认其情况，并继续翻译他对美岛的带路命令。队伍进入敲门鬼影响的老城区后，酒井在她面前遭人头气球斩首，她惊恐捂嘴避免尖叫；面对鬼奴电话、空中人头和白色鬼烛造成的黑暗，她始终紧跟杨间，保留三岛配发的卫星定位手机、丢弃自身普通手机。（原文回跳第654章至原文回跳第656章）
