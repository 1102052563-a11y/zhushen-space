# 田野

## 简介

田野是岛国除灵社驭鬼者，具备鬼域能力。三岛称他曾在此前鬼敲门行动中失败、害死多人，却因自尊扭曲仍坚信自己足以处理神户事件；本次未获带队资格后，他断开通讯、擅自闯入机场，以杀害普通人和挑衅杨间的方式试图争夺行动主导权。其年龄、所驾驭厉鬼名称、鬼域范围和过去失败细节均不详。（原文回跳第650章）

## 能力

- **低层鬼域**：田野能使机场小店、候机大厅光线瞬间昏暗，并把自身隐藏在模糊人影中；杨间评价其鬼域不强，却仍属于较少见的能力。田野在鬼域中可隔空掀飞目标、将人摔死或嵌入水泥，但完整杀人媒介和限制不详。（原文回跳第650章）
- **鬼域内隐匿**：田野认为鬼域可隔绝杨间鬼眼窥视，但杨间仍能以鬼眼锁定其移动位置。随后鬼手从脚下抓住田野、令鬼域消失，说明其鬼域无法抵御更高层压制。（原文回跳第650章）

## 经历

#### 原文回跳第650章·擅闯机场并杀害两名女子

除灵社准备让杨间中午进入神户市时，田野因未被任命带队，强行赶到机场并断绝通讯。三岛称其能力不足、过去失败害死多人，命其离开；田野却在鬼域中先后把两名作陪女子摔断脖子、嵌入水泥，威胁下一位是惠子，意在逼杨间出手或证明外援无能。（原文回跳第650章）

杨间起初不救两名女子，认为这是除灵社内部问题；田野据此嘲讽其为胆小外援。杨间反指田野只敢伤害普通女性、回避真正强敌，并公开要求其直接动手。田野因被激怒继续维持鬼域，却无法摆脱鬼眼注视。（原文回跳第650章）

#### 原文回跳第650章·被杨间压制并受筷子警告

杨间以鬼手从田野脚下拉倒目标，驱散其鬼域并使其体内厉鬼沉寂。田野被拖入小店、无法使用能力或反抗，杨间用两根筷子分别钉穿其双手，警告若再犯会令其直接消失。三岛为保住田野性命向杨间道歉；田野被留下后是否接受除灵社处罚、手伤如何处理及会否再参与神户行动均不详。（原文回跳第650章）

#### 原文回跳第664-666章·受一郎鼓动越权入城后死亡

部长一郎被三岛逐出会议后，私下联系田野，称三岛过度依赖杨间、若由田野和山崎带队必能解决神户异常。田野因先前被杨间压制而心怀怨气，接受一郎越过三岛安排的私自行动，声称自己与山崎一定能成功；但一郎实际只希望田野进入神户市制造可供抢功或问责的局面，并未把他的生存置于首位。（原文回跳第664章）

杨间关押敲门鬼后，田野随山崎和一名皮包骨男子试图接手箱体，被杨间当面喝止，不敢继续挑衅。三人转往曾点燃鬼烛的十字路口调查时，田野喊住一个背对他们行走的人影；目标猛然回头，露出半腐烂死人头和灰白眼睛。田野立刻倒地，头颅脱离脖颈滚向厉鬼，未能发动鬼域或进行抵抗，当场死亡。（原文回跳第666章）
