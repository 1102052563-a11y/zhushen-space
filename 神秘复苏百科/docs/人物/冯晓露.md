# 冯晓露

## 简介

冯晓露是与张浩、郑耀文同行的女性公交乘客。她在此前车内鬼故事争论中的具体发言、职业和与同行者的关系均未交代。第304章，她与张浩等人随大批乘客在荒野临时站下车，亲见田野土坟前的哭声使若干乘客跪地、哭泣并死亡；杨间提醒众人车外属于鬼的世界，要求不想死的人回到公交车。（第304章）

## 经历

#### 第305章·返车途中死亡

冯晓露在逃回公交车途中因穿高跟鞋扭伤、失去平衡而摔倒。郑耀文回头试图拉她时，两人同时出现不受控制的悲伤哭泣表情和泪水，随后向田野间土坟跪下。原文明确说明两人意识尚在，却无法控制身体；他们没有回到车内。杨间将坟前厉鬼命名为“哭坟鬼”，并根据此前遇害者和此时结果认定其哭声会杀死下车乘客。冯晓露死亡后的遗体处置不详。（第305章）
