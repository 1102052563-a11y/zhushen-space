# 周杰峰

## 简介

周杰峰是裴东团伙成员，也是该团伙鬼域的实际使用者。原文回跳第636章，他在鬼域内同裴东等人讨论是否交出童倩；杨间据鬼域表现判断其仅驾驭一只鬼，却依靠未知的摆钟诅咒压制复苏，得以长期存活。其过往身份、负责城市、加入团伙时间及从何处获得厉鬼均不详。（原文回跳第636章、原文回跳第637章）

## 能力

- **高层鬼域**：周杰峰的鬼域可压制冯全鬼雾，按杨间以鬼眼划分的判断，接近四层鬼域水准。但他无法长期承受与杨间鬼域对耗，红光入侵后被锁定位置；鬼域具体范围、移动能力、杀人规律与发动代价均不详。（原文回跳第634章、原文回跳第635章、原文回跳第636章）
- **摆钟诅咒的延缓作用**：周杰峰称团伙成员受某台找不到的老摆钟施加时间诅咒，诅咒会在期限到达时杀人，却也能压制一、二只鬼的复苏；若驾驭到三只鬼，诅咒会消失且本人立即死亡。此为周杰峰提供的团伙经验，摆钟位置、规则完整性和是否可验证均不详。（原文回跳第637章）

## 经历

#### 原文回跳第634-636章·鬼域支援与被杨间擒获

裴东围困冯全时，周杰峰以鬼域压制鬼雾、协助隐藏其余成员。杨间进入后没有马上全力压制，而是以红光持续消耗；周杰峰看出杨间有意“钝刀割肉”，劝裴东尽快决定。裴东同意交出童倩时，杨间已通过鬼眼锁定周杰峰，将真正身体转移到其身边，以黑化鬼手掐住脖颈并压住体内鬼，鬼域立即崩溃。（原文回跳第635章、原文回跳第636章）

周杰峰在最后关头把裴东等人送离鬼域，自己留下被俘。他承认杨间一瞬间就令六人队伍崩溃，并在无法自杀、无法动弹的状态下同意带路寻找童倩。（原文回跳第636章）

#### 原文回跳第637章·供出囚禁童倩与摆钟诅咒

被杨间、冯全和黄子雅押送时，周杰峰说明团伙发现童倩曾换用赵磊身体，误以为更换身体可摆脱厉鬼复苏和时间诅咒，故以安眠药趁其睡眠时控制并长期研究。他还供出团伙受老摆钟诅咒、诅咒同时压制复苏的情报，并承认童倩被关在小安县医院地下二层。（原文回跳第637章）

#### 原文回跳第638章·死亡与遗鬼收容

杨间在地下二层发现被实验后的童倩，并认为周杰峰等人越过不伤害无关者的底线。确认不再需要情报后，杨间在鬼手持续压制状态下扭断周杰峰颈骨，将其杀死。冯全和黄子雅把尸体放进原本囚禁童倩的特制玻璃箱；杨间认为该箱可隔绝灵异，且周杰峰仅驾驭一只鬼却有强鬼域，遗鬼有收容价值。其后谁接管玻璃箱及厉鬼是否复苏均不详。（原文回跳第638章）
