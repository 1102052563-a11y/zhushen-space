# 吉米

## 简介

吉米是被困在鬼画世界中的一名皮肤黝黑的外国男性幸存者。杨间进入鬼画内由大京市背景拼接到国外城市后，发现包括吉米在内的十余名普通人依靠荧光箭头、建筑躲避与画中可用食水生存。吉米的国籍、被卷入事件的缘由、在幸存者中的身份和掌握的具体路线均不详。（第623章、第624章）

## 经历

#### 第623章·作为幸存者群体的一员出现

杨间通过李阳得知，这批普通人于半年前国外鬼画事件中被鬼域卷入，最初人数很多，之后不断被鬼杀死，到当时只剩十余人。他们在国外风格大楼内以荧光物质涂出箭头，既为求援，也方便活人进出而不至迷路。吉米在本章未单独发言，原文只将他包含在从各房间走出、急切想获知出口的幸存者中。（第623章）

#### 第624章·自愿成为鬼画向导

大卫持枪威胁失败后，杨间没有立即迁怒其他幸存者，而是提出需要两名熟悉鬼画城市的合格向导，以寻找画中的鬼，并把幸存者获得离开的机会作为条件。李阳与吉米在杨间走出大楼后追上来；李阳说明两人愿意带路，吉米没有另行补充。杨间报出姓名与“鬼眼”代号后，向导行动尚未展开；吉米能提供哪些路线、是否认识鬼的活动规律及其最终下落均不详。（第624章）

#### 第625-629章·指认居民楼并留守三楼

吉米向李阳说明自己曾在一栋七层居民楼门口看见穿红衣的女人，李阳转译后，三人因此入楼寻找。听到木楼梯上未知鬼随脚步回应时，他极度恐惧，一度摇头拒绝进楼；李阳和杨间说明带路是换取离画机会的条件后，他还是与二人同行。进入二楼、经室外铁梯到三楼的过程中，他负责留意杨间身后；发现三楼入口消失时，由他先看到翻入的窗户已变成墙面。（第625章、第626章）

杨间检查尸体和房间时，吉米在房外看见窗户，曾激动想从该处逃离，后被李阳拉住并接受杨间对陷阱的判断。堵门鬼现身时，他与李阳在房门外近距离看见腐尸跑出、堵在门前，因此不敢移动；杨间压住鬼后，李阳出现异常并成为驭鬼者。三楼未找到鬼画后，杨间认为通往四楼的木梯会直接遭遇未知鬼，便让吉米和李阳继续留在三楼，自己独自上楼。吉米是否能安全等待、能否继续提供路线及最终能否离开鬼画，均不详。（第627章至第629章）

#### 第631-634章·楼梯突围与被鬼画杀死

杨间从四楼下撤时，吉米与李阳在三楼会合并一同往下跑。木梯前有红衣鬼上行、后有楼梯鬼脚步逼近，他因恐惧不敢如杨间般接触厉鬼，撞上后退的李阳后僵在原地。杨间出于李阳已成驭鬼者、吉米曾提供地点的考虑，以无头鬼影将二人抬起救援；吉米被带过红衣鬼头顶。两鬼相持令鬼影失控后，他从台阶跌下，未受致命伤，仍随杨间撤出居民楼。（第631章至第633章）

鬼画背景扩展、红衣鬼追向人群后，杨间将吉米置于马路一侧以判断目标。红衣鬼先接近杨间，继而改向吉米；杨间据此离开，吉米没有逃跑，持续闭眼祈祷并哭泣。杨间、李阳离远后，吉米用英文托李阳转告母亲自己很爱她；李阳翻译完毕，杨间已判断他死亡。原文未描写直接袭击过程、遗体或其国籍，不能补定死因细节。（第634章）
