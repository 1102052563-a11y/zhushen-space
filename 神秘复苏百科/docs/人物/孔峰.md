# 孔峰

## 简介

孔峰是王小强雇佣行动组的指挥者之一，负责针对杨间的“三段猎杀计划”。他认为杨间只是实战经验不多的学生，并预估行动目标活下来的几率不超过百分之三十。（第195章）

## 经历

#### 第195章·安排猎杀计划

孔峰随黛西等人进入售楼部，要求同伴按步骤打探杨间位置，不许在行动前轻率行事。他知道目标是危险的驭鬼者，警告若计划失败，行动人员都可能死亡。得到杨间可能在庄园别墅的消息后，他命散开的人员汇聚；张伟拿直播手机跟拍时，孔峰起初要求不理会，后来从耳机获知杨间在别墅，便下令阿海处理张伟，以免暴露或干扰计划。阿海以匕首刺伤张伟，却被濒死反击的张伟开枪击倒；孔峰误以为这只是回光返照，命人稍作藏尸、不救中弹的阿海，继续准备猎杀计划。（第195章）

#### 第197章·鬼域内被杀

孔峰的狙击和金黄色烟雾计划相继失效，杨间在烟雾扩散前张开鬼域，把孔峰和余下行动人员困在红光之中。孔峰知道鬼域会让人分不清真实、也无法按正常方式离开，仍试图带队撤退。杨间先后杀死行动者并判断幕后者是王小强，孔峰的头颅随后被其拎在手中、随意丢下。孔峰在第195章以三段猎杀计划指挥普通武装人员围杀驭鬼者，最终未获王小强和叶枫直接支援，死于杨间鬼域内。（第197章）
