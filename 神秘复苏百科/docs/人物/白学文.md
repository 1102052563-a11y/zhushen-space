# 白学文

## 简介

白学文被参加总部培训的驭鬼者称为“白教授”。他双鬓发白、留短发，表情严肃却精神十足，在大京市郊区培训基地为新到的驭鬼者讲授第一堂课。张雷转述，他自大学时期便研究各国历史，尤其专研神鬼志异，并称他是知名教授、大文学家，至少会五国语言；这些履历均出自张雷的介绍，白学文本人未在本批逐项证实。（第436章）

## 观点与讲授

白学文认为，“鬼”这一称呼伴随人类历史发展而存在，只是不同国家的文献使用幽灵、邪灵、亡灵等不同名称；中国正史、野史与民间传说也都有相关记述。他没有把这些材料当作直接证明，而是以历史记载构成一套解释当前灵异事件的说辞。他还把负责一座城市灵异事件、保护居民的国际刑警，类比为古代“缉拿恶鬼，庇护一方”的城隍。（第437章）

他的讲课另以国外古代法老金棺和金字塔为例，提出黄金棺椁及特殊建筑也许不只服务于墓葬习俗，还可能承担关押厉鬼的目的。王江据此直接说出“关押厉鬼么”，白学文只回答“不排除有这个可能”，未将其确认为史实。（第438章）

## 经历

#### 第436-437章·在培训基地讲授第一课

杨间到课时，白学文正在讲授“鬼、历史、起源、猜想”等内容。部分学员不认可用古代神鬼传说解释自己所遭遇的厉鬼，其中一名未具名驭鬼者两度起身斥责其为歪门邪说。白学文没有反骂，只说明这只是个人见解、希望众人耐心听完，并以自己的讲授可能误人子弟而接受责骂。张雷因尊敬教授与该学员争执，杨间随后以威胁制止学员继续打断，课堂才恢复安静；白学文向杨间致意后继续授课。（第436章、第437章）

#### 第499章·继续跟进民国资料

王小明向杨间谈及民国时期灵异线索时，称白教授是历史学家、文学家，正对这件事持续跟进。原文未写白学文在第499章实际进行了何种调查，也未说明王小明所称“白教授”是否有其他同名人物；结合既有培训基地讲师身份，此处仅记录王小明的转述，不能补充其研究结论。（第499章）
