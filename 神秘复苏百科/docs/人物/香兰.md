# 香兰

## 简介

香兰是杨间在凯撒大酒店四层异常走道的71号房内遇到的女性。她穿红色旗袍，盘着乌黑头发，外表健康、面容白皙；杨间以鬼眼反复观察，确认她有正常活人的血色和气息，并未感到厉鬼特征。她却身处已被水泥封死、外部长期无人可入的酒店深处，这一事实令杨间无法解释其来路。（第602章、第603章）

## 经历

#### 第602-603章·在71号房弹奏钢琴并逃向深处

杨间循与八音盒铃声相同的钢琴旋律进入71号房时，香兰背对门口坐在钢琴前。她演奏时多次错拍，神情紧张、身体发颤，却仍勉强继续弹奏；杨间判断她并非因兴致演奏，而是受某种未明理由驱使。杨间现身发问后，香兰受惊停琴。阿南自隔壁房冲出，她立刻拉着对方逃走；得知杨间是从“出口”进入后，她急切追问出口在哪里，请求带两人离开。杨间尚未解释，阿南已怀疑他不正常并带她离开。二人没有向酒店出口跑，而是朝异常走道更深处逃去。香兰是否来自另一端入口、为何必须弹琴、是否知道71号房内有鬼及最终去向均不详。（第602章、第603章）

## 关系

- [阿南](阿南.md)：阿南听到动静后从隔壁冲出，香兰抓住他一同逃离71号房。（第603章）
- [杨间](杨间.md)：杨间在71号房发现她是活人并询问来路；她则把杨间视为可能来自出口的外来者。（第603章）
