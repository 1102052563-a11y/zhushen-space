# 张芬

## 简介

张芬是杨间的母亲。杨间成为驭鬼者后，总部把她安排到大京市平安大厦工作，提供正式单位、六险一金与相对稳定的工资待遇；杨间接受这一安排，原因是自己若因厉鬼复苏死亡，母亲至少仍有维持生计的工作和较安全的环境。她在本批正文中未直接出场，仅由平安大厦客服按系统资料确认其身份和工作楼层。（第425章）

## 经历

#### 第425章·被确认在平安大厦工作

杨间离开平安大酒店后前往市中心平安大厦看望张芬。大厦客服根据其国际刑警身份确认，张芬正在第二十层上班，并为杨间指示电梯和接待安排。杨间由大厦外松内严的安保、暗中监视力量及家属集中工作的情况，推断这里实际兼具国际刑警家属保护基地性质；这一判断是杨间的现场推断，并非总部正式说明。杨间是否见到张芬、她是否知晓灵异事件及其后续工作情况，原文未写。（第425章）

#### 第427-428章·与杨间见面及返乡安排

杨间在二十层找到张芬时，她正坐在办公桌前使用电脑，因不习惯操作而动作迟缓；同层还有年长员工，杨间据此进一步确认这里的工作主要是为家属提供保护与安置，而非追求效率。经理准许她休息一天与儿子见面。张芬从大昌市化工泄漏、学校关闭等传闻问起，杨间以停课避难、暂时辍学及和同学开公司拉投资等说法掩饰灵异事件和自己的身份；她相信儿子仍会继续学习，也叮嘱他小心受骗、注意安全。（第427章、第428章）

杨间因预感队长计划将改变局势，提出让张芬过几年辞掉这里的工作、回大昌市；张芬认为这份抽中而来的工作工资高、福利好，暂时拒绝。待她返回岗位后，杨间私下请经理将来以辞退为由协助自己带母亲离开，经理称须向上申请。杨间计划在本次大京市行程结束且自身问题能够解决时再执行，故张芬是否会离职、何时回到大昌市均未定。（第428章）
