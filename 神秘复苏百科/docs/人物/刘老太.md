# 刘老太

## 简介

刘老太是大海市刘氏公司老板刘元之母，七十多岁，死亡时间为杨间抵达福寿园前两日。她的78号墓位正是鬼邮局红色信件指定的“福寿园78号主人”，但红信为何选择一名刚去世的普通老人、她生前是否接触灵异，均不详。（原文回跳第747章、原文回跳第748章）

## 葬礼与异常

刘家为她举行较大规模土葬告别仪式，近百名宾客到场。杨间以鬼眼确认棺内老人身穿黑色殓服、脸部化妆、双手交叠，起初只是一具无灵异痕迹的普通尸体，因此暂未投递红信，准备等其正式下葬成为78号墓主。（原文回跳第749章）

葬礼进行时，刘老太尸体在棺内悄然睁眼，凹陷眼眶中的眼珠开始转动；一号墓园同时有暮色降临。此时尚未写明她是否起身、是否已经变为厉鬼、红信能否令其恢复意识，不能直接把“异常复苏中”视作普通死而复生。（原文回跳第749章）

灵异论坛人员抬动棺材时，棺盖意外打开，刘老太尸体已不在其中。杨间确认此前鬼眼看见的确为普通尸体，故失踪可能来自人为搬走、尸体自身异变或福寿园内厉鬼；其后墓园被坟场鬼域覆盖，刘老太去向仍不明。（原文回跳第750章、原文回跳第751章）

#### 原文回跳第757-759章·在坟场中作为刻碑载体出现

刘欣悦、刘浩在坟场中看见刘老太穿黑色殓服、双手沾满坟土，僵立于老坟间；其一只手还握有锈迹斑斑的大钉。杨间随后亲眼见其以该钉敲击空白墓碑、刻写姓名，确认她已被坟场灵异操控。她究竟是复苏厉鬼、鬼奴还是尸体被其他力量驱动，仍不能定论。（原文回跳第757章、原文回跳第758章）

杨间认为钉子外形近似棺材钉，并联想到张雷记忆中坟场大门上的钉子，推测它可能是未确认的第二或第三根棺材钉；此为线索比对，尚未实际取得或验证同源。刘老太在刻完杨间名字后消失，红信尚未投递。（原文回跳第758章）

#### 原文回跳第760-764章·被反向刻名、送信后爆裂异变

杨间以柴刀在空白墓碑刻下“刘兰”，坟场诅咒随即将刘老太埋入，刻碑声暂止。挖开坟土后，杨间将红信塞入其手中，信件腐朽消失，并从尸体另一手取走锈蚀钉子。刘老太随即吐出大量可驱退无头鬼影的粘稠黑血，鬼手压制和棺材钉穿头均未能稳定尸体，最终尸体爆裂，只余被钉住的头颅较完整。（原文回跳第760章、原文回跳第763章、原文回跳第764章）

刘老太的异变后多座老坟开始塌陷，疑有其他坟中目标苏醒。她是平衡关键、红信收信工具还是单纯受害尸体均不详；当前不能确认其是否已彻底被关押或死亡。
