# 林北

## 简介

林北是被困灵异公交车的驭鬼者，原先因始终戴鸭舌帽、坐在末排而被杨间误判为可能是车内厉鬼。第307章公交熄火后，他从后排车窗离开，承认与许峰已在车上共同滞留一段时间；第313章他取下帽子，露出光头，自称是尚未毕业的佛教学院学生，勉强算和尚。他不以“救人”为己任，却主张把前人以死亡摸索出的公交线索留给可能下车的人，认为自己只能做力所能及之事。（第307章、第313章、第315章）

## 能力

- **睡眠自保**：林北称自己睡着后会进入一种特别状态，通常能使自身非常安全；公交熄火后，他在路边睡眠，以此规避周边厉鬼的直接威胁。（第310章、第313章）
- **梦中受袭**：熄火期间，他在睡眠中不断抽搐、挣扎、掐住自己脖子，称梦里有无时无刻接近的恶鬼，若在梦中被杀，现实也会死亡。他推测纠缠自己的鬼能入梦，但杨间仅怀疑其可能是“梦魇”，并无确证。（第311章、第313章）
- **复苏救醒**：第313章，他称自己在梦中濒死时因厉鬼复苏而醒来，才得以回到公交；这只说明本次复苏中断了梦境袭击，不能确认其是否可主动重复。（第313章）
- **车内压制下的复苏鬼驾驭**：林北说明公交会压制厉鬼，令已复苏的厉鬼在车上不能杀死其驾驭者。他自称上车第六日、曾下车两次；最初离车后厉鬼约三分钟会复苏，如今可延至约十分钟，期间可不顾忌地驾驭一只复苏鬼。具体鬼物、延长条件与该时间是否稳定均不详。（第315章）

## 经历

#### 第307-312章·熄火后与梦中鬼对抗

公交熄火时，林北从末排车窗跳到车外，与许峰留在车辆附近等待重启。他认为熄火后车内比外部更凶险，便说明平时无法打开的窗门会在此时解除限制；但他未告诉杨间完整规则，仍与许峰讨论后者留新人占座的打算。哭坟鬼出现后，他选择睡眠自保，身体却越来越剧烈抽搐、乱蹬和自掐，显然在梦中遭遇袭击。杨间据此判断还有一只鬼与他纠缠，并怀疑是能入梦杀人的厉鬼。公交重启后，林北自行翻窗登车，称已从噩梦中醒来；梦中鬼是否随他上车或仍在其体内，他无法确定。（第307章、第308章、第311章至第313章）

#### 第313章·与杨间互通身份

林北在车窗外重新出现后，承认杨间在熄火期间的存活方式和压制哭坟鬼的表现远超“新人”印象，主动询问姓名。得知杨间是大昌市国际刑警后，他报出林北之名及佛教学院学生身份，并坦言自己的安全方式并非绝对有效：睡眠虽能进入特殊状态，却也成为入梦厉鬼可袭击的入口。杨间没有因其身份放松戒备，继续观察仍在车外牵着张浩尸体的干尸新娘；此时林北、杨间与许峰均回到车内，普通乘客已全部死亡。（第313章、第314章）

#### 第315章·说明留车求生的理由

车辆接近有路灯、货车通行的正常城市时，林北提醒杨间一旦停车应立即离车，并承认自己与许峰不是找不到机会，而是不能轻易离开。按他转述的前辈经验，灵异公交可强力压制厉鬼，令已经复苏的鬼暂不能杀死驾驭者；林北和许峰因此把长期留车当作延缓复苏的求生方法。林北仍承认这不是无风险庇护：熄火、满载、鬼拦车都会造成大危机，而前一名同样能驾驭复苏鬼的幸存者已因自保能力不足死亡。他坚持将信息交给杨间，希望线索不会随被困者死亡而断绝。（第315章）

#### 第316章·留车并托付家讯

公交在正常城市开门时，林北没有跟随杨间离开。他没有提出物资要求，只请杨间查找佛教学院学生林北的档案、向家人报平安，理由可由杨间自行编造；杨间当场答应。车辆随杨间下车后不久消失，林北仍留在车内，后续状态不详。第317章，杨间将托付交给秦媚柔，要求以正当名义向其家人汇款一千万元；该款是否最终送达，原文未写。（第316章、第317章）

## 人际关系

- **杨间**：熄火后首次正式相识。林北认可杨间的能力，并提醒其在正常城市停站时离车；双方仅交换了有限信息，未形成合作承诺。（第313章、第315章）
- **许峰**：同为长期被困公交的驭鬼者。二人都以车内压制延缓自身复苏，但林北反对许峰向杨间隐瞒生存信息、借新人占座的做法。（第307章、第315章）
