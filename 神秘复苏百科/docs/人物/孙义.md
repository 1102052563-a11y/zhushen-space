# 孙义

## 简介

孙义是国际刑警部门的文职人员，自称工作内容是为体系挖掘人才。他在接到赵建国命令后被派往大昌市，任务是协助杨间完成正式任职。其对工作有保密要求，生活中则正处相亲、欠费的窘境。（第205章）

## 经历

#### 第205章·受命前往大昌市

孙义在餐厅相亲时接到命令，起初以休假和相亲为由推迟，得知可获介绍接线员的承诺后立即同意出发。他因相亲餐费三万八千元无力结账，又无法通过网络贷款审核，只能向赵建国求助。赵建国在电话中催促其半小时内前往集合点，并说明会有人送他去大昌市。至本章末，孙义尚未抵达大昌市，也未与杨间直接见面。（第205章）

#### 第208-210章·考核杨间并完成任命

孙义在大昌市接待室负责杨间的刑警考核，问题涵盖身份、婚恋、隐私与数个灵异事件处置假设，过程全程录音录像。他虽然表现得贫困、散漫，却是驭鬼者；因不适合一线刑警工作而转任联络和考核人员。第209章，他宣布杨间通过程序、交付新卫星定位手机和证件并回收周正旧机。第210章，他带杨间与市内部门人员见面，之后说明本次考核虽带例行性质，记录仍会影响未来安排，国际刑警制度也可能调整。（第208章至第210章）

#### 第241-244章·医院行动中死亡

孙义随王小明进入被饿死鬼侵蚀的医院，在鬼婴群从楼梯、电梯及墙壁发动首轮袭击后幸存片刻，却被未知高阶段鬼婴咬去下半身，体内驾驭的鬼也被剥离。赵开明判断他失去厉鬼力量维持后，普通人体质无法承受失血；众人将其带入病房厕所临时包扎，但缺少血浆和医疗器材。杨间赶到后询问遗愿，孙义只提及未还完的房贷，随即吐血、失去神采而死。他的遗书内容、所驾驭厉鬼和尸体后续处置，原文未说明。（第242章至第244章）
