# 万峰

## 简介

万峰是大昌市第七高中、与方镜一同在鬼域中逃亡的学生。第13章方镜在操场外绿化树林中被黑衫老人追逐时，回想自己一路为拖延厉鬼而抛下过多名同学，并明确点出万峰的名字。原文未交代万峰被抛下时所在的具体地点、遭遇或是否死亡。（第13章）

## 出处

- 方镜逃进学校操场外的绿化树林后，因树林在黑暗中如同无尽原始森林，和剩余学生无法离开。他回想从把杨间推入厕所起，已通过各种手段丢下赵强、万峰、贺秋燕等同学，意在用他们的性命拖住身后的厉鬼。（第13章）
- 此时仍跟着方镜的学生只剩五六人，其他人一部分被抛下，一部分自行逃入黑暗后再未遇见。万峰属于前者或后者中的具体情形不详，故不能确认其下落。（第13章）
