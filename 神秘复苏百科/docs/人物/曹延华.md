# 曹延华

## 简介

曹延华是第257章主持大昌市饿死鬼事件电话会议的刑警总部副部长。总部部长此前已出国求援，因此由他在大昌市外临时指挥中心负责会议准备、线路保密和现场指挥；赵建国、李军与他同在现场。（第257章）

## 第257章·主持电话会议

杨间要求总部所有能决策者参与事关大昌市生死的会议后，曹延华在赵建国协助下紧急接入多部保密电话，并要求警卫戒严、会议结束后销毁手机、录音只留一份。他面对杨间与一名军区首长的争吵未立即插话，但在杨间说明“接触”和“关押”源头鬼的两个条件后，表示总部会尽力解决可解决的部分。杨间提出为取得替代限制物，可能需释放一只未知高危鬼，要求总部决定是否承担后果；曹延华未在通话中立刻给出答复。第258章，刘小雨转达曹延华已批准杨间行动，无论成败由国家承担。其具体决策依据、与其他参会者的意见及之后指挥不详。（第257章、第258章）

## 第263-264章·准备封锁与救援

杨间失联七日、卫星定位手机未移动时，曹延华在大昌市外指挥中心因市内联络尽失而日益急躁。他听赵建国说明安全屋最多只能再维持两日，若杨间计划没有结果，幸存者和王小明均面临物资断绝与外出风险；李军则指出城市将成为全球首个沦陷大城市。曹延华因此开始考虑封锁周边、撤离并设为禁区的最坏方案。第264章收到杨间重新上线的可靠消息后，他立即按王小明预先方案召开紧急会议，命救援中心准备灾民安置、赵建国征调的刑警准备入城，并称相关风险由自己负责；行动尚未实际展开。（第263章、第264章）

## 第267章·确认赵开明异常并备好金棺

杨间在七中检测到赵开明刑警信号后，曹延华调取其档案和精神评估，发现报告已差到不宜担任刑警；赵建国说明周正死亡、合格人手匮乏且自己曾同意沈良临时顶替，曹延华遂要求杨间警惕赵开明，待大昌市事件后再处置。与此同时，指挥中心已由武警看守一口耗用数吨黄金、壁厚五厘米的金色大箱，计划在封入饿死鬼后焊死，作为本次关押容器。（第267章）
