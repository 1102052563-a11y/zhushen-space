# 陈星

## 简介

陈星是鬼邮局二楼21号房的男性信使，与老许及另一名未具名男子同住、携带枪械。他在钱蓉被赶出21号房后负责威胁和驱逐，对刚上二楼的杨间表现出敌意；其成为信使的经过、已送信次数、是否驾驭厉鬼和真实身份均不详。（原文回跳第781章）

## 经历

#### 原文回跳第781章·因驱逐钱蓉与杨间冲突

钱蓉被21号房赶出、濒临熄灯时，杨间为给自己、李阳和钱蓉取得房间，命李阳开门进入。陈星从卧室冲出准备取枪，承认21号房内有多把武器，且认为外来信使无资格占房；他见杨间用疑似鬼手控制自己后，判断对方是远超普通信使的驭鬼者。（原文回跳第781章）

杨间没有向他索要二楼情报，而是以鬼手扭断其手臂和腿，将其拖至走廊，命李阳把他留在门外过夜。陈星曾以部分信件需要人数为由求饶，称减少信使可能使送信失败；杨间以自己人数足够回应。该说法是否准确、陈星是否掌握任务经验均未核实。（原文回跳第781章）

#### 原文回跳第782章·熄灯后失踪

晚六点，陈星因四肢重伤不能进入其他房间，被熄灯后的黑暗吞没。次日走廊上已无其人，杨间、李阳未听到整夜惨叫或门外异动，只能以熄灯后不能滞留走廊推测其死亡。尸体去向、夜间厉鬼身份及其是否被转移而非死亡均不详。（原文回跳第782章）
