# 吴越

## 简介

吴越是与孙俪红同属一家公司、负责管理公司的总经理，也是黄岗村鬼棺回收事务的现场负责人。第112章，公司人员在黄岗村外封锁道路并监视村庄时，他在临时帐篷内主持应对会议；他本认为杨间在黄岗村中生还的概率很低，得知其带着鬼棺和张韩驶出后，被迫出面谈判。（第112章）

## 经历

#### 第112章·等候回收结果

吴越在公司临时营地中与负责人商议如何应对离村驭鬼者。他称公司派人到场主要是给赵建国面子，认为若有人解决事件，只须履行此前交易；他亦把信息不透明称为正常商业操作，认为民间驭鬼者自愿接下高危任务，失败并不会让公司损失什么。观察员报告黄岗村消失、杨间与张韩车辆驶来后，他只能中断判断、出去会面。（第112章）

#### 第113-114章·回避档案责任

吴越以公司主要负责人身份欢迎杨间和张韩，承诺会交付第二鬼驾驭方案，却称要先采集数据、交实验室匹配，成功率约百分之十五。他否认知道档案细节，以交易可能作废劝阻杨间动手；随后又以一点五亿元收购和“必要牺牲”安抚，承认公司评估六名行动者全体生还率不足百分之一。杨间认定他拒绝对删档后果负责，拔枪指向其头部。（第113章、第114章）

#### 第115章·被李军救下

杨间开枪后，子弹穿过吴越头部而未伤及其身体。王小明随即承认删档命令出自自己，称吴越只是为公司办事。吴越此前强撑的镇定消失，见王小明一行到场才放松；杨间仍未接受其解释，现场责任被转向王小明。（第115章）

#### 第116章·被作为交易代价杀死

杨间要求王小明亲手杀死吴越，以验证其所称大义是否适用于身边普通人。王小明衡量两人价值后认定杨间更重要，持枪向吴越连续开枪；吴越逃跑、受伤倒地后，仍被王小明第三枪杀死。赵建国与李军均在场而未阻止，吴越最终成为促成杨间承诺参加国际刑警考核的代价。（第116章）
