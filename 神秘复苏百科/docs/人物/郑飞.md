# 郑飞

## 简介

郑飞是杨间的同班学生。第6章，他在方镜煽动下参与推杨间去拖住厕所鬼，后被杨间抓住手臂，和段鹏一起被拖进厕所黑暗。第7章，他曾与段鹏试图寻找消失的厕门；段鹏失踪后，杨间拒绝带他离开。郑飞在迷路、道歉时被一只惨白手掌从背后抓住颈部，尖叫后下落不明。（第6章、第7章）

## 出处

- 方镜提出“想活命就得牺牲一个人”后，郑飞与段鹏及另一名男同学上前抓住杨间。杨间停止抵抗时抓住两人手臂，使背后鬼手的拖拽把他们一并带向厕所。（第6章）
- 厕所内没有正常出口。郑飞与段鹏沿着冰冷、斑驳且似乎无尽的墙壁寻找门，却一直找不到；杨间被音频救下后拿手机灯照到两人，郑飞因杨间还活着感到震惊。（第6章、第7章）
- 杨间认为郑飞也参与了害人，不肯带他同行。郑飞在黑暗里跟丢杨间，试图道歉并摸索路线；他听到身后脚步后误以为可能是段鹏，随即被惨白手掌从背后抓住颈部，发出尖叫。该尖叫没有传到杨间处，原文未写其后的状态。（第7章）
