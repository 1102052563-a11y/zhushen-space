# 苗小善

## 简介

苗小善是杨间同班的女学生。第2章课前，她凑到杨间和张伟身边，表示自己在网上听过许多像真的一样的灵异传闻，害怕到不敢再看，并询问是否真有地方闹鬼。杨间答复多数网络故事是假的，但真碰到这类东西应离远一些；苗小善接受了这一说法。第5章，她在教室撤离时因楼板坍塌与张伟一同掉到下一层，之后仍能行动。（第2章、第5章）

## 出处

- 在第2章，她作为学生参与灵异话题讨论，所关心的是传闻真假及现实中是否会闹鬼。（第2章）
- 在第5章，她的名字被杨间在楼板坍塌后呼喊；张伟回应自己没事，原文未单独描写苗小善伤势或后续去向。（第5章）
- 她参与第2章的讨论时，张伟展示了小区防盗窗上悬挂多具尸体的照片。面对这类传闻，她没有断言照片为真，而是说网上许多人讲得像真的一样，自己已不敢再看；这也是本批原文中她唯一明确的态度。（第2章）
- 撤离发生在周正要求学生趁鬼域暂退逃走之后。她和张伟跌落至下一层教室，约三米高度未被原文写为立即致命；杨间没有停下来救援，而是因苏蕾伤势和追击风险催众人继续离开。（第5章）
- 第9章，她与张伟等人困在第二层走廊，已经来回走了半小时，担心会永远被困在学校。杨间因鬼眼看破鬼域后找到众人，承诺带路；她随后跟随杨间下楼。（第9章、第10章）
- 她亲眼听到鬼婴的怪叫，杨间确认是从周正身上脱出的鬼婴后要求所有人快走。第10章在鬼婴转而攻击王珊珊时结束，苗小善没有受伤记录，后续不明。（第10章）
- 第16至17章，她和其他幸存者到校门后仍被黑暗阻隔。她提醒杨间，电话异常是在众人离开教室、鬼域再度出现后才发生；黑衫老人逼近时，她也收到“138”号码来电，但没有接听。（第16章、第17章）
- 第19章惨白手掌抓住她的脚踝，她与其他人一同受鬼攻击；杨间红光鬼域张开后，她随幸存者离开学校。第20章，她在步行街确认安全并支持杨间回家等待官方调查的建议。（第19章、第20章）
