# 苗小善

## 简介

苗小善是杨间同班的女学生。第2章课前，她凑到杨间和张伟身边，表示自己在网上听过许多像真的一样的灵异传闻，害怕到不敢再看，并询问是否真有地方闹鬼。杨间答复多数网络故事是假的，但真碰到这类东西应离远一些；苗小善接受了这一说法。第5章，她在教室撤离时因楼板坍塌与张伟一同掉到下一层，之后仍能行动。（第2章、第5章）

## 出处

- 在第2章，她作为学生参与灵异话题讨论，所关心的是传闻真假及现实中是否会闹鬼。（第2章）
- 在第5章，她的名字被杨间在楼板坍塌后呼喊；张伟回应自己没事，原文未单独描写苗小善伤势或后续去向。（第5章）
- 她参与第2章的讨论时，张伟展示了小区防盗窗上悬挂多具尸体的照片。面对这类传闻，她没有断言照片为真，而是说网上许多人讲得像真的一样，自己已不敢再看；这也是本批原文中她唯一明确的态度。（第2章）
- 撤离发生在周正要求学生趁鬼域暂退逃走之后。她和张伟跌落至下一层教室，约三米高度未被原文写为立即致命；杨间没有停下来救援，而是因苏蕾伤势和追击风险催众人继续离开。（第5章）
- 第9章，她与张伟等人困在第二层走廊，已经来回走了半小时，担心会永远被困在学校。杨间因鬼眼看破鬼域后找到众人，承诺带路；她随后跟随杨间下楼。（第9章、第10章）
- 她亲眼听到鬼婴的怪叫，杨间确认是从周正身上脱出的鬼婴后要求所有人快走。第10章在鬼婴转而攻击王珊珊时结束，苗小善没有受伤记录，后续不明。（第10章）
- 第16至17章，她和其他幸存者到校门后仍被黑暗阻隔。她提醒杨间，电话异常是在众人离开教室、鬼域再度出现后才发生；黑衫老人逼近时，她也收到“138”号码来电，但没有接听。（第16章、第17章）
- 第19章惨白手掌抓住她的脚踝，她与其他人一同受鬼攻击；杨间红光鬼域张开后，她随幸存者离开学校。第20章，她在步行街确认安全并支持杨间回家等待官方调查的建议。（第19章、第20章）
- 第136章，镜中鬼伪装成她的表哥上官云返回包厢，杨间识破后开枪验证，鬼舍弃该身份消失。她先因亲属关系阻止杨间开枪，后亲眼见到中弹不死与伪装消失，确认表哥遭遇异常。（第136章）
- 第137至140章，真正的上官云被困镜内，她以水在镜面写字安慰并转述杨间的问题。镜中鬼靠近并改作模仿她时，她明知危险仍继续写字以争取救表哥的机会；杨间在交换发生前用鬼域将她移到身后，最终带出上官云。她没有进入镜内，但本批未交代事件后去向。（第137章至第140章）
- 第141章，杨间将她和上官云带离KTV后，她因恐惧和获救的情绪抱着杨间痛哭；父母到场后，她告知杨间自己过两天将离开大昌市，并提醒自己会远离镜面。临别时她说大学毕业后若杨间仍无女友，愿考虑做其女友，之后随父亲和表哥离开。（第141章）

## 第482-483章·大京市重逢与同游

苗小善高考后到大京市读大学，在街头偶遇杨间。她称曾在张伟直播间告知自己的去向，杨间因近期忙碌未曾得知；得知其手机损坏后，她提出送一部手机，杨间以需要特制手机谢绝。她怀疑杨间所谓“打工”只是对秘密事务的掩饰，却没有得到灵异事件的解释。（第482章）

杨间起初因不愿让她接触危险而试图离开，她仍拉住对方去买日用品。第483章，她让杨间陪同逛街、试穿连衣裙，并在刘紫要求同走时选择让杨间送回学校。她从张伟直播得知杨间在大昌市有房产及五楼“有鬼”的说法，杨间未解释，只意识到应把自己存放的异常物品转移至更安全处。临别时，苗小善取得杨间留下的卫星定位手机号码，号码可接通但并非常规号码；此后双方联络与她是否会接触灵异不详。（第483章）

#### 第484-485章·谈及旧同学并返回学校

回校途中，苗小善告诉杨间自己读历史系，也提到表哥上官云希望请杨间吃饭；她询问七中幸存者近况后，得知张磊已死、王珊珊状态异常、孙仁和刘奇不知去向，以及张伟也曾遭遇危险。她没有把这些结果归咎于杨间，反而强调七中众人是被他救下，并察觉杨间正逐渐失去某些情感。杨间仍以灵异会带来死亡为由劝她少联系，她表示不愿如此，双方却未能改变分歧。（第484章）

第485章，杨间送她到校门，拿她考上大学与自己、张伟和王珊珊的生活作玩笑，仍要求她好好读书、珍惜生命。苗小善反过来劝他少接触危险、好好活着；杨间称自己贪生怕死后离开。她是否使用卫星定位号码、是否再联系杨间或将被卷入灵异，均未写。（第485章）
