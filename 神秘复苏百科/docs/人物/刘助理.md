# 刘助理

## 简介

刘助理是陈博士实验室中一名仅以姓氏称呼的助理。第644章，陈博士要求人员撤离并整理资料时，他与一名未具名同事在基地走道交谈。其全名、专业、工作年限、是否负责具体灵异项目及之后去向均不详。（第644章）

## 经历

#### 第644章·目击滚动死人头与鬼童

刘助理和同事离开陈博士后，在昏暗走道讨论王小明的激进研究及驭鬼者危险性。同事先称看见不明物从前方经过，刘助理起初认为是看错，后又听见拐角有沉闷物体滚动的声音，才同意准备联系保安科。两人尚未完成上报，一颗带黑发、面容惨白且高度腐烂的死人头从拐角滚出，并在地上改变方向向他们靠近；灯光受影响闪烁，二人立即逃离。（第644章）

死人头在墙边停下后，穿脏旧寿衣、皮肤青黑的鬼童出现并将其捧起。刘助理回头确认时被吓得跌倒、鼻子磕破流血，随后继续逃跑；鬼童因被活人看见而躲避，其与死人头均消失在二人视野中。刘助理没有被死人头正面盯住，故逃过袭击；他与同事停下后决定通知陈博士、王小明和安保人员。通知是否成功、他是否离开大楼和之后是否再遇鬼童，均不详。（第644章）

## 出处

#### 第645-646章·目睹小张死亡后停止撤离

鬼童出没被确认后，刘助理与其他人先主张撤出大楼，陈博士允许小张分批试探。十分钟过去，小张既未发来短信也未返回，刘助理由此庆幸先行者不是自己，并接受陈博士“先留在楼内”的决定。第648章总部仍确认陈博士等人存活，刘助理是否获救、是否再看见鬼童及其后工作状态均不详。（第645章、第646章、第648章）
