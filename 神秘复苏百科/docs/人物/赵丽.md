# 赵丽

## 简介

赵丽是李跃的女性信使同伴，约三十岁。她随李跃到梅山村树林送信，随身持有一张圆形旧纸钱作为灵异自保物；信件被杨间拆开后，她与周林、李跃一同面对入侵型厉鬼。本批末她随李跃和周林驾车离开大昌市，尚未察觉周林已被鬼入侵。（原文回跳第695章至原文回跳第698章）
