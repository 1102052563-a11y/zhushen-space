# 上官云

## 简介

上官云是苗小善的表哥，在第128章陪苗小善参加第七高中幸存同学的聚会。他对杨间等人所说的灵异经历抱持怀疑，认为现实中不应存在鬼。第134章KTV冲突中，他被阿飞等人误伤流鼻血；第135章独自到洗手间处理伤口时，镜中出现与现实不一致的影像，随后被镜中鬼拖入镜内空间。杨间利用鬼域在鬼试图替换苗小善身份的瞬间救出上官云，三人随即离开KTV。（第128章、第134章至第140章）

## 经历

#### 第134章·冲突后的洗手间

阿飞等混混闯入KTV包厢时，上官云因穿着名牌被误当成有钱人而受伤流鼻血。杨间将黄金手枪交给他，让他自行决定是否报复；苗小善阻止后，他只要求对方道歉，冲突暂止。为处理鼻血，他之后独自前往洗手间，这次与众人分开成为镜中鬼制造身份替换的起点。（第134章、第135章）

#### 第135-137章·被困镜内

上官云洗脸时在镜中发现自己没有倒影，转身离开洗手间又被镜内空间外的浓黑阻断去路。他虽开始恐慌，仍先以常理解释异常，并拨通苗小善电话求助。现实中的杨间等人进入洗手间后发现现场无人，却在镜子里看见上官云；他所在的镜像厕所只能映出该处，外面一片黑暗，双方需借电话确认彼此位置。手机落地导致通话中断后，苗小善以水在镜面写字与他沟通，但上官云已因无法离开和不明声音而接近崩溃。（第137章、第138章）

#### 第139-140章·镜中鬼逼近与获救

镜内厕所拐角出现现实中不存在的苍白身影，上官云却听不到镜外警告；鬼随后模仿苗小善进入镜内，令他在回头看见“表妹”后意识到异常，惊恐逃到墙边。杨间判断鬼真正目标已经转向镜外的苗小善，要求她继续写字以维持鬼的模仿。鬼的动作与苗小善完全一致时，镜内外产生交换；杨间以鬼域将苗小善移走、以门板代替其原位，使鬼未能取得身份，并抓住通道开启的一瞬以红光把上官云从镜内带回现实。上官云脱困后，杨间没有尝试关押厉鬼，而是带苗小善与上官云立即远离KTV。（第139章、第140章）

#### 第141章·确认灵异事件后离开

上官云被带到KTV外三公里处后仍瘫坐在绿化带旁，反复追问镜中鬼去向；杨间确认他未被吓傻，并直言这次只是较特殊的灵异事件。他因此接受世上确有鬼、普通人平时被蒙在鼓里的事实。苗小善父亲开车来接后，他与表妹一同离开；杨间提醒他珍惜活下来机会，也明确表示下次遇到麻烦不要再来求助。（第141章）

## 关系

- [苗小善](苗小善.md)：表妹；她在镜外冒险写字安慰并配合杨间救援，使杨间得以观察镜中鬼的替换规律。（第136章至第140章）
- [杨间](杨间.md)：同学聚会中的参与者；杨间识破厉鬼假扮上官云，并在镜内外交换的一瞬使用鬼域将其救出。（第136章、第140章）
