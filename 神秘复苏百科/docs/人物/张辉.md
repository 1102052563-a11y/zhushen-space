# 张辉

## 简介

张辉是参加大京市培训、并在培训基地鬼差事件中幸存的一名驭鬼者。第475章时，他住在平安酒店的一间客房，原文没有交代他的驻地、国际刑警职务、驾驭了几只鬼、能力表现或进入培训基地前的履历。鬼差事件让他第一次直面无法逃离、无法反抗的灵异，他此前认为灵异主要威胁普通人、驭鬼者纵然无法处理也能逃走的想法被打破；此后反复失眠，并自觉精神已接近崩溃。（第475章）

## 经历

#### 第475章·房门倒影与离开平安酒店

培训基地事件结束后，张辉回到平安酒店休息，却因不断回想黑暗中的遭遇多次从睡眠中惊醒。他知道不少前任驭鬼者会因恐惧、失眠与精神问题走向发疯、极端甚至自杀，因而也意识到自己的状态不正常。一次准备入睡时，他从余光看见客房门外地面有一道静止的人影倒影；他出门查看，客厅中无人，房门也没有被打开的痕迹，回头时倒影已经消失。他无法确认是精神紧张导致误见、酒店残留灵异，还是有驭鬼者暗中窥视，遂收拾物品离开平安酒店，想回到热闹都市寻求安全感。其他人未察觉其临时出行，其后去向、房门倒影来源和精神状态均不详。（第475章）

#### 第485章·离店情况被张雷报告

张雷向杨间转述酒店工作人员所见，称张辉是在前一夜收拾好物品、慌张离开，而非普通外出，且始终没有向总部说明。杨间认为其经历鬼差这类S级事件后，可能只是害怕留在大京市、产生辞职想法；这只是杨间的推测，不能排除张辉察觉危险或遭遇灵异。张雷决定提醒众人留意，原文没有写张辉收到联络、回到酒店或下落被确认。（第485章）

#### 第508章·死于外部小宾馆

张辉在早晨约六点死亡，尸体约在第508章前两小时被发现于平安酒店外的一家小宾馆。陈义已接管现场并处理其体内厉鬼；张雷、王江和黄子雅正接受总部人员询问。现场未爆发灵异事件、目前只死张辉一人，故张雷等人不排除他杀，但凶手、死因、张辉离开酒店的真正目的和陈义的调查结论均不详。（第508章）

#### 第509-510章·死因被陈义判断为疑似灵异

陈义复查小宾馆后称现场没有可疑痕迹，否定有人特意在大京市杀害张辉的解释，转而怀疑是灵异事件；这是陈义的判断，并未公开确证。杨间据张辉从培训基地逃出、在平安酒店察觉异常后匆忙换住处、最终仍死亡的过程推测，他可能早被一只看不见、摸不着、也听不见的鬼跟随；杨间又联想到自己在鬼差事件中打开魂瓶后失踪的瓶中鬼。该关联及张辉是否因此被杀均为杨间的推测，不能替代调查结论；第二、第三名受害者、该鬼的规律与陈义后续处置均不详。（第509章、第510章）

## 出处

- 张辉的身份、培训后精神状态、房门倒影及离开酒店。（第475章）
