# 郑耀文

## 简介

郑耀文是与张浩同行的小春市公交乘客。第302章，他因不满车内对异常的猜测，直接到驾驶位呼喊并推动司机，令已死亡至少一天、开始腐烂的司机尸体倒下，从而使全车乘客确认车辆处于异常状态。原文第301章中其姓名未被写出；第302至305章均作“郑耀文”，但第302章一处同伴呼喊写作“郑文耀”，此页以多数写法记录，并保留原文异写。（第302章）

## 经历

#### 第304-305章·下车后遇害

车辆在荒野临时站停靠后，郑耀文随张浩等人下车。坟前哭声响起，先下车者陆续跪地哭泣而死；他本可随同伴返回，却在冯晓露跌倒后回头相助。两人随即不受控制地露出悲伤表情、流泪并朝土坟跪下。杨间随后将这类哭声视为厉鬼的无差别袭击；郑耀文未能回到车内，且原文记述其与冯晓露均已死亡。（第304章、第305章）
