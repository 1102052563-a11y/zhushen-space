# 王海燕

## 简介

王海燕是王珊珊的母亲。第23章，她因女儿整夜未归且电话不通，在高档小区门口等候。她见杨间送王珊珊回家后，怀疑二人关系，质问其是否带女儿“鬼混”，又以两百元早饭钱试探和羞辱杨间，希望其不要影响王珊珊学习。（第23章）

## 出处

- 王海燕因王珊珊整夜未归而责骂女儿，并称若当天仍不回来，自己和丈夫会去报案。她没有接受学校发生灵异事件的说法，且以王老师电话打不通的事实试探杨间。（第23章）
- 她强调女儿家教严格、高三应以学习为重，要求杨间不要影响女儿；随后塞给杨间两百元。杨间收钱后离开，王珊珊认为母亲是在羞辱救命恩人而哭泣。王海燕则判断杨间收钱说明其“没志气”，坚持让女儿远离他。（第23章）
- 杨间没有解释高校内的死亡和灵异经历，只说晚自习后全班被留在学校、为王珊珊安全才送她回来。王海燕在第25章前未再出场，未知她是否得知事件真相。（第23章）
- 第29章，她发现女儿从学校回来后不吃饭、怕黑且反复洗澡，因担心其遭欺负或精神受创而询问丈夫学校情况。王珊珊在浴室尖叫后，她撞门进入，以浴巾遮住满身鲜血的女儿；她看不见女儿所称的窗外之物，只能安慰并同意先包扎伤口。（第29章）
- 第34章，她与王彬均看见一个青黑无衣的小孩跑进王珊珊房间，后见到满地、满墙和满天花板的脏小孩脚印。她在丈夫发现大门无法打开后收拾东西欲离开，最终随丈夫和女儿退入主卧避险。（第34章）
- 第36至39章，她目睹鬼婴扑向女儿及杨间以红光击退它。逃离后她一度因鬼婴未死而质疑付费，后在杨间欲离开时道歉；知晓女儿会被鬼婴追杀后与王珊珊痛哭，并协助解开颈部绷带，亲见杨间抹去手印、留下红色眼印。（第36章至第39章）
- 第250章，王海燕随王彬、王珊珊来到观江花园向杨间求助。再见这名曾被她轻视的女儿同学时，她因一家人须求人安置而尴尬、不知所措；杨间同意安排房屋后，她未再提出异议。她在阴霾中所见、对女儿手腕鬼眼烙印变化的理解及之后的安置情况不详。（第250章）

- 第385章，王海燕与丈夫、女儿在尚通大厦等候杨间处理王珊珊的异常。杨间以鬼影确认王珊珊腹内有成形鬼婴后，称可带众人上楼处置；王海燕未在本章发表处理意见。她对鬼婴残留的知情时间、心理反应与处置后的状况均不详。（第385章）

- 第386章，王海燕在顶层办公室催促杨间尽快取出鬼婴，担心其害死女儿；杨间以鬼眼查验后称其暂时稳定、不伤王珊珊，决定在会议结束后再到家中处理。她听到杨间将其判断为与自身诅咒有关的特殊存在，但原文未写其是否理解、接受该判断或参与后续公司会议。（第386章）

- 第391至392章，王海燕在杨间家等待女儿的处置结果。王珊珊展示被剥离的淡青色鬼婴并宣布要自行寄养后，她受惊尖叫、后退，反复要求杨间劝阻，又担心女儿与鬼相处会丧命。她虽听到杨间称目标已受控制、寄养可保护小区与家人，仍只强调自己仅有一个女儿，不能接受这一危险选择；王珊珊坚持后，她未能说服女儿。她对鬼婴的三项指令、王珊珊实际迁居及其后是否仍同住均不详。（第391章、第392章）
