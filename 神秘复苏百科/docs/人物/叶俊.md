# 叶俊

## 简介

叶俊是小强俱乐部成员，代号“鬼手叶俊”。他能通过握手在目标手掌留下漆黑手印；张韩称一个手印三日内可使人烂成泥，多个手印可迅速致死。第76章，他借此勒索杨间交易利润，反遭杨间用黄金子弹重创。（第76章）

## 出处

- 叶俊先以握手在杨间掌心留下黑手印，再要求交易利润五成，称这是俱乐部新人规矩。（第76章）
- 杨间立即以黄金子弹打碎其头部，后以黄金甩棍打断其肢体；叶俊因体内厉鬼仍保有意识，承认自己“就差口气”。（第76章、第77章）
- 第78章，他称自己鬼很特殊、难以被杀，仍未解释鬼手印的完整规则与代价。（第78章）
- 第91章，他与张一鸣、张韩驾车离村却回到黄岗村入口，因鬼打墙被迫返回。（第91章）
- 第94章，他一度欲枪击村民验证身份，被杨间制止；第95章，他从屋顶发动能力，令腐臭尸水滴落并使老婆婆躯体腐烂，随后参与关押。（第94章、第95章）
- 第96章，黄岗村双鬼平衡被打破后，他在老婆婆屋外失踪，杨间判断其已遇害。此后无头鬼影附身的老婆婆可流出尸水并抵挡鬼域，表明叶俊体内的鬼已被夺走；杨间将其计入黄岗村死亡的三名驭鬼者。（第96章、第98章、第100章）

- 第658章，杨间根据基地内尸水痕迹辨认出这正是叶俊生前驾驭的尸水，并称活人接触后数日内会全身腐烂、未复苏时可使驾驭者承受袭击而不死。杨间同时推测该尸水早已被人皮纸吞入，后被其借来活动；尸水实际是否受人皮纸控制、叶俊原有鬼手能力与尸水关系均不详。（第658章）
