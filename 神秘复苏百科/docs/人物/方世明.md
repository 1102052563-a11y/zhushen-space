# 方世明

## 简介

方世明是朋友圈在第512章会议中公认的顶尖驭鬼者，也是最早一批驭鬼者之一，并兼任一座城市的负责人。外表看似不到三十岁，实际资历被与会者称为比冯全更早；他穿休闲衣、姿态懒散，会议中架脚躺在椅上休息。其具体辖区、驾驭的厉鬼、能力、加入朋友圈的时间及实际职务，原文均未公开。（第512章）

## 外貌·性格

方世明在朋友圈讨论杨间时没有立即发言，待表决形成结果后才放下腿回应，表现得像是对会议和冲突都缺乏兴致。但费总及与会者明显忌惮其决定：费总称只有请他亲自出马才可能确保行动，且在方世明要求计划出纰漏便自行消失时立刻冒冷汗答应。方世明把消灭驭鬼者称为对自己“没什么影响”，又认为这个世界尚未准备接纳他们这类异类，显示其对普通社会及杀人后果的态度冷漠；这只是本次言行，不能据此推断其全部立场。（第512章）

## 经历

#### 第512章·同意借灵异事件除掉杨间

李瑶报告两百亿元招揽失败后，朋友圈多数成员表决同意在队长计划开始前“打掉”杨间。费总认为公开袭击会引来总部猜忌，提出须由方世明出手；方世明没有接受贺天雄的半途截杀建议，而称应让杨间死于下一场灵异事件，使死者无法说明真相。他判断鬼画与鬼差两件S级事件的处置必有一件失控，只要稍作引导便能把杨间卷入。费总接下制定方案的任务，方世明则以严厉后果约束其不得失手。行动尚未实施，所谓引导方式、方世明是否亲自参与及杨间能否获悉计划均不详。（第512章）

## 关系

- [费总](费总.md)：费总提出应请方世明出手并负责制定具体方案；方世明接受方向后要求其不得出错。（第512章）
- [杨间](杨间.md)：方世明同意利用未来灵异事件使杨间死亡，双方本批尚未直接接触。（第512章）
