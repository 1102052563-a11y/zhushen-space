# 王悦

## 简介

王悦是苗小善在大京市大学宿舍的室友，睡在对面床铺。张霞失踪、苗小善要求撤离时，她认为杨间夜闯女生寝室、带走苗小善的行为可疑，并未相信油画会引发灵异事件。她在油画自行移动后才改变主意，但因折返取包直接遭遇从画中伸出的手掌。（第550章至第552章）

## 经历

#### 第550-551章·不信警告后决定撤离

苗小善称寝室油画有问题、可能涉及灵异事件时，王悦只认为杨间在吓人，且对他踹门、抱走苗小善的行径感到愤怒，声称再有此类闯入便要报案。杨间离去后，孙于佳指出他不可能在数分钟内赶到寝室，王悦仍未立即接受这种异常。直到张霞留下的油画自行移到她床头，她才感到毛骨悚然，并联想到苗小善和刘紫先后离开的反常，决定与孙于佳离开宿舍。（第550章、第551章）

#### 第552章·被油画吞没

王悦穿好衣物准备离开时，因想拿回放在床头的包、手机和钱包而折返。她看不到油画中已伸出的苍白手掌，踩上台阶伸手取包时被手掌抓住手腕；她挣扎回缩，油画随之倒下并压在身上。画框内像无底洞般吞没她的上身，画中手掌又抓住露在外面的双腿向内拉扯。孙于佳最终只看见一双扭曲变形的腿露在画外，停止挣扎后偶有抽动。原文没有确认王悦是否死亡、是否被带入鬼画，或能否被救出，因此状态只能记为不明。（第552章）
