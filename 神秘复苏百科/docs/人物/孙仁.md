# 孙仁

## 简介

孙仁是杨间、张伟的第七高中同班同学，也是鬼敲门事件后幸存的七名学生之一。他原与杨间、张伟交往不深；半年后因被一具女尸灵异盯上、又无法进入真正灵异圈求助，逐渐变得偏激，为换取被关押的鬼而绑架张伟。其家庭、职业、失踪前经历与是否已成为驭鬼者均不详。（原文回跳第646章）

## 状态与异常

孙仁反复在睡眠中见到一具惨白女尸：最初女尸立在床头，后逐次靠近，回跳第646章已倒在床边并以冰冷手臂压住他。他曾安装摄像头，虽无法拍到女尸，却录到多重门锁自行开启，因而确认自己被鬼盯上。他右手缺失数根手指，称曾试图进入灵异圈寻找解决方法，具体断指原因与女尸规律关系均不详。（原文回跳第646章）

## 经历

#### 原文回跳第646章·以张伟逼杨间交鬼

孙仁以请吃饭为名在张伟饮料中下药，将其囚于小城市出租房的焊死铁门房间，借张显贵留下的电话联系杨间。他承认针对张伟只是为找到杨间，提出以张伟安全交换“一只被关押的鬼”。杨间认为他已走向极端，仍为确保张伟生还暂时答应，并要求视频确认人质状态。（原文回跳第646章）

视频中张伟精神尚可，还向孙仁吐出蓄积半小时的浓痰；孙仁没有当场伤害他，却要求杨间按其给出的地址交付厉鬼。孙仁明确表示自己“烂命一条”，不在乎报复，反映女尸压床和断指已使其濒临崩溃。（原文回跳第646章）

#### 原文回跳第646章·女尸逼近与换鬼动机

孙仁挂断电话后准备入睡，再次遭遇女尸靠近；他惊醒后发现房门多道锁均已被破坏，确认并非噩梦。他判断女尸每次都比上次更接近，继续下去必死，因此把成为驭鬼者、取得一只鬼视为唯一生路。张伟为其异常取了“鬼压床”代号；这只是张伟的玩笑称呼，不是总部正式代号。（原文回跳第646章）

#### 原文回跳第647章·交换前等待与被通缉风险

孙仁收到杨间将交付厉鬼的承诺后，发出地址、关机并丢弃手机，试图切断追踪。杨间则命章华在张伟安全回来后通缉并直接击杀孙仁，认定其已涉灵异事件、不能按普通犯罪处理。孙仁是否知道通缉、是否真的能取得厉鬼及张伟能否按约获释均不详。（原文回跳第647章）

#### 原文回跳第678章·逃离追捕且疑已成为驭鬼者

章华向杨间汇报，张伟已被救回，但孙仁在交易后逃脱，并取走原定交付的灵异物。追踪行动因现场发生无法理解的灵异现象失败，章华据此判断孙仁大概率已经成为驭鬼者；该判断没有说明他取得哪只鬼、是否仍受女尸压床影响或具体逃亡地点。杨间认为张伟安全更重要，暂未要求继续强追。（原文回跳第678章）

- [杨间](杨间.md)：高中同学。孙仁因无法求助其解决女尸异常，转而绑架张伟逼其交换厉鬼。（原文回跳第646章）
