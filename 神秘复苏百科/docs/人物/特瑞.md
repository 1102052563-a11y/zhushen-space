# 特瑞

## 简介

特瑞是第295章进入观江花园小区采访队的一名外国男子，最初负责扛摄影机。他能以不算流畅的中文与杨间交谈；在杨间摔毁摄影机、鸣枪警告后，特瑞和同伴显露出受过武器训练的反应，令杨间认定其并非普通记者。（第295章）

## 经历

#### 第295章·摄影采访中的武装暴露

特瑞随苏珊等人采访张韩和杨间，负责以摄影机近距离拍摄。杨间要求小区内人员离开时，特瑞仍在拍摄；杨间随后抢走摄影机、抛出数十米摔碎。枪声响起后，特瑞和另外两名外国男子迅速摸向腰间武器，动作使杨间判断他们具备训练背景，也使这支采访队的记者身份受到直接质疑。（第295章）

#### 第295章·转达舞会邀请

杨间怀疑采访只是接触自己的掩护后，特瑞承认“不借用记者身份”难以在大昌市活动，称有人想邀请杨间参加舞会。杨间追问姓名后，他报出“特瑞”；杨间旋即开枪使子弹从其裤裆附近飞过，并拒绝邀请、要求其放下武器。特瑞没有立即反击，只称杨间应当了解这件事会对其有好处；杨间决定报警，称要让刘队审问众人。特瑞此后是否投降、舞会主办者身份和采访队实际归属均不详。（第295章）

## 关系

- [苏珊](苏珊.md)：同属以采访为掩护进入小区的队伍；苏珊负责提问，特瑞负责摄影和转达邀请。（第295章）
- [杨间](杨间.md)：特瑞试图邀请杨间参加舞会，遭拒绝并被杨间以枪警告。（第295章）
