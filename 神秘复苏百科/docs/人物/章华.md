# 章华

## 简介

章华是大昌市一名自称侦查队长的公职人员。第297章，他带同事到杨间别墅报告伪装记者一案，自报姓名并出示证件；杨间核验后确认无误。原先跟进杨间案件的刘队已在三日前被证实失踪，章华表示此后若无意外将由自己负责跟进相关案件。刘队全名、失踪经过及章华的具体职级均不详。（第297章）

## 经历

#### 第297章·报告渗透线索

特瑞等人被武装特警带走后，章华到杨间住处汇报侦查结果。他说明近期渗入大昌市的国外势力不只白天那一队，市内至少已抓捕六起类似事件、武力驱逐证件不齐者二十余批。特瑞曾参加海外战争、退役后进入多家国外安保公司；更重要的是，大昌市存在一家雇有大量外国安保人员的公司。这些线索使杨间认定白天接触不是普通采访。（第297章）

#### 第297章·提交尚通科技材料

章华出示尚通大厦照片，称尚通科技一年前以招商引资进入大昌市，表面主营网络软件开发，背景来自国外知名企业。公司长期试营业、账目亏损，早期监控又拍到观江小区武装袭击嫌疑人出入。可是照片中的人并非公司员工，苏珊也只承认受其资助采访，三名雇佣兵在表面上与公司无关，警方仍缺少查封证据。杨间要求以自己名义申请搜查令，章华接受安排。（第297章）

#### 第297章·接收字迹恢复委托

杨间把荒宅鬼橱顶部取出的老旧记事本交给章华，希望解析第一页残留字痕。章华确认存在叠加笔画，却说本地擅长此项工作的老前辈已无法使用，调附近市同事需要时间；他介绍小春市新城区警局的程雄。杨间决定亲自前往，不等待调人。章华是否取得搜查令、如何处理尚通科技，至第300章不详。（第297章、第298章）

第297章、第298章。

#### 第318-325章·执行尚通大厦搜查

章华取得搜查令后，带公务人员在第323章按约到杨间住所接人，并说明若无确凿证据不能公开越权查封跨国企业，以免在灵异消息仍受控制时造成额外社会动荡。杨间以鬼域锁定顶层黄金隔绝的房间，章华遂接受其判断，跳过普通楼层直上四十五层。面对公司断电阻拦、保罗主张隐私与律师权利，章华仍以搜查令要求配合，并组织人员开启酒柜后保险室；杨间则负责限制保罗。保险室内物、证据结果及公司处置尚未写出。（第318章、第323章至第325章）

#### 第389-390章·接手尚通大厦租户纠纷

尚通大厦旧租户因索取高额赔偿而聚众进入顶层办公室，杨间原计划叫章华处理。第389章，杨间在压住刘驰等人后以电话通知章华自己遭勒索、涉额数亿元；第390章章华带队到场，将被视为勒索大昌市负责人、聚众闹事和妨碍其工作的人员接手。部分老板选择私了，刘驰则在医院醒来后仍要求赔偿。章华具体采取的法律程序、被带走人数和案件结论均不详。（第389章、第390章）

#### 第327章·中弹获救与查封依据

杨间离开尚通大厦期间，章华带人开启保险室门，遭内中两名雇佣兵开枪；他共中两枪，失血严重，一度认为撑不到救护车，仍授权杨间直接击毙持枪者。杨间以鬼影从其体内取出两颗子弹、拼接伤口止血，章华得以站起并等待输血。保险室内雇佣兵死亡后，他确认窝藏罪犯、持枪伤人和私藏军火足以查封尚通科技；具体查封程序、其伤势恢复和证据处理均不详。（第327章）
