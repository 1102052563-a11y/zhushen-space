# 乐乐

## 简介

乐乐是中山市居民戴勇的五岁儿子。人头气球事件恶化的夜晚，他独自在家看动画片，母亲外出打麻将，父亲戴勇回家时发现无人照料。他年幼，不理解传闻和高层阳台的危险，只因声称看见母亲在窗外，便搬凳子将大半身体伸出十八楼窗外，试图开窗让对方进入。（第341章）

## 经历

#### 第341章·看见窗外母亲

戴勇将乐乐从窗边抱下并关窗，认为母亲不可能出现在十八楼外。乐乐所见随后被证实：窗外漂浮的是戴勇妻子的头颅，头下没有身体，仍带着熟悉耳环。该人头随风穿过窗户、进入室内时，戴勇试图带乐乐逃离，却发现乐乐已经站在原处，颈上没有头颅；他的头如气球般离开身体升起。原文以戴勇的绝望和尖叫写明此事，未再交代乐乐遗体、头颅去向或他是否在此前被何物袭击。（第341章）
