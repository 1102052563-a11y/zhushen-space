# 小张

## 简介

小张是陈博士实验室的一名助理，只以姓氏称呼。鬼童在平安科技研究基地大楼内游荡后，他认为留在有鬼出没的大楼等同等死，主张应在事态恶化、形成鬼域前迅速撤离。其全名、专业、进入基地时间和是否知晓具体实验项目均不详。（第645章）

## 经历

#### 第645章·自愿试探离楼路线

陈博士接到王小明要求暂不离楼的答复后，小张认为灵异事件中等待救援最危险，提出以最快速度冲出大楼；其他助理也倾向撤离。陈博士最终允许分批试探，要求首人安全离开后再通知其余人。小张因尚未亲眼见到鬼童，自认生还概率较高，主动承担先行者角色。他离开办公室后沿楼梯下至一楼，途中没有遇到鬼，因而确认大厅和敞开大门在表面上都没有异常。（第645章）

#### 第645章·触发出楼必死规律

小张跨出大楼时，身后传来小孩急促奔跑般的脚步；他尚未来得及回头，便感到阴冷并似乎看见死人头在暗中盯视自己。鬼童此前已接到杨间“杀死所有进出这栋楼的人”的新命令，小张的出楼举动因此触发杀人规律。他瞬间失去生命、面带离楼的喜悦倒在地上。高明在楼外目击结果，确认大楼已被鬼童封锁。遗体后续处置、是否有厉鬼复苏及其亲属情况均不详。（第645章）
