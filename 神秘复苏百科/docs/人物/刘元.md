# 刘元

## 简介

刘元是大海市刘氏公司老板，约五十岁、身材偏矮胖，具有商业人士的威严。他为两日前去世的母亲刘老太举办规模较大的土葬告别仪式，近百名亲友和社会人士到场。公司规模年盈利不足一亿元，具体行业、家庭情况和是否知晓灵异均不详。（原文回跳第748章、原文回跳第749章）

## 经历

#### 原文回跳第749章·举行刘老太入葬仪式

刘元在福寿园为母亲搭建临时灵堂，安排棺材、亲友告别和后续入葬78号墓地。他发现杨间、孙瑞、李阳在灵堂附近停留，命助理张鑫客气驱离；得知三人来头可能很大后，选择暂不冲突。刘元没有发现棺内母亲已睁眼，也未察觉墓园暮色的出现。（原文回跳第749章）
