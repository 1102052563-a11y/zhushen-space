# 林辉

## 简介

林辉是林小夕的哥哥。妹妹被发现自斩头颅死亡后，他误认梁圆是凶手，拿木凳闯入梁家准备报复；杨间以鬼手制止并说明此案属于特殊情况。办案人员初步认定林小夕自杀，林辉最终接受五百万元私了、带走尸体。其对灵异梦境和妹妹真正死因并不了解。（原文回跳第700章）

## 经历

#### 原文回跳第700章·误认梁圆杀妹并接受私了

林辉冲入梁家时看见身上沾血、神情失常的梁圆，随即抄起木凳砸向她头部。杨间徒手接住木凳，又以鬼手抓住其手腕，使他身体暂时无法动弹；林辉在恐惧中停止暴力。杨间检查林小夕尸体后称死者为自杀，办案人员也初步作出相同结论，但林辉认为梁家仍应负责。（原文回跳第700章）

杨间为避免村中纠纷继续扩大，提出五百万元赔偿和解。林辉先坚持“杀人偿命”，得知附近另有类似自杀死亡、且办案人员不会按凶案处理后转为接受私了，收款后离开。赔偿是否实际用于安葬、他是否仍怀疑梁圆、日后会否知晓灵异真相均不详。（原文回跳第700章）
