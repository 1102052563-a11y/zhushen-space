# 赵小雅

## 简介

赵小雅是赵开明的六岁女儿，也是其早年请身边厉鬼办事所害死的亲属之一。赵开明完成利用饿死鬼的计划后，她先于其他家人从仓库冰棺中复活；其复活使赵开明一度相信厉鬼的承诺并非骗局，却也成为诅咒转移的起点。（第276章）

## 经历

#### 第276章·冰棺复活与诅咒承接

赵开明吞枪重伤、准备自杀时，赵小雅所在冰棺传出拍打玻璃的声音。赵开明爬到冰棺旁推开棺盖后，她哭着爬出、呼喊父亲，并与满身鲜血的赵开明相抱。赵开明因她是最早死亡的亲属，判断其余家人可能会依死亡顺序复活，便让她去打开存放母亲、爷爷和奶奶的冰棺。赵小雅虽听话照做，却因冰棺为赵开明定制、开启方式特殊而无法打开。

其后赵开明昏迷又苏醒，发现其他亲属都已复活。赵小雅称是一位“叔叔”帮她开门，指向背后却没有活人；赵开明由此意识到跟随自己的厉鬼仍在。该鬼替赵小雅完成开棺，代价是必须有一名亲属死亡。赵开明察觉代价落在自己身上后死亡，并在临终前认定自身的悲剧会在女儿身上重演。赵小雅是否知道厉鬼存在、是否会被迫继续请鬼办事及其后续处境，均不详。（第276章）

## 出处

- 赵开明之女，六岁；从冰棺复活后获未知厉鬼协助开棺。（第276章）
