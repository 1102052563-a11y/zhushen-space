# 刘欣悦

## 简介

刘欣悦是刘元之女、刘老太孙女，二十三四岁，参加祖母在福寿园的葬礼。她与弟弟刘浩在灵异论坛人员试图抬走棺材时阻拦，后来最先发现棺内刘老太尸体消失。（原文回跳第750章）

## 经历

#### 原文回跳第750-754章·卷入福寿园诡异坟场

罗素一手下强行驱赶葬礼宾客、抬走刘老太棺材时，刘欣悦与弟弟阻拦未果。棺材落地、棺盖打开后，她发现祖母尸体不见，随即卷入福寿园坟场鬼域。父亲刘元担心子女安全，请杨间等人带走二人；罗素一允许两人随队，但不保证存活。（原文回跳第750章、原文回跳第751章）

进入坟场后，她强忍恐惧照顾十五六岁的弟弟。杨间命灵异论坛成员和刘家姐弟共同挖掘刻有罗素一姓名的老坟时，她因目睹杨间射杀一名论坛成员而惊惧，仍为求生参与挖掘；本批末她从坟土中挖出一张死人脸，脸的身份和后续结果未写。（原文回跳第752章、原文回跳第754章）

#### 原文回跳第755-759章·目击祖母刻碑并提供身份

罗素一被确认死亡后，刘欣悦与弟弟随杨间队伍继续行动。刘浩先看见祖母刘老太穿黑殓服在坟间出现，她随后亲眼确认其僵立、双手沾满坟土，因恐惧一度无法行动；刘老太消失方向与刻碑声来源一致。（原文回跳第757章）

杨间返回后询问刘老太原名，刘欣悦答为“刘兰”。她本人仍受困坟场，杨间准备利用空白墓碑和刻名规则对付刘老太，刘欣悦及弟弟能否因此生还不详。（原文回跳第759章）

#### 原文回跳第761-763章·弟弟获救并完成红信投递

刘浩被坟场刻名后陷入泥土，杨间以鬼影卸下其双腿、将上半身提离地面，使其脱离坟土袭击。刘欣悦随后扶弟弟坐在墓碑上，虽失去双腿，刘浩仍存活。杨间允许她专心照顾弟弟，不再强令她挖坟。（原文回跳第761章、原文回跳第763章）

杨间以其提供的“刘兰”姓名反向刻碑后，挖出并向刘老太尸体投递红信。刘欣悦仍在坟场内，祖母尸体爆裂和老坟塌陷后，她与弟弟的撤离结果未写。（原文回跳第763章、原文回跳第764章）
