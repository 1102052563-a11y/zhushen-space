# 钱万豪

## 简介

钱万豪是鬼域中跟随杨间逃到校门附近的学生。第17章，同学手机接连收到诡异来电时，他见自己的来电显示为王老师号码，误以为王老师可能来救人，违背杨间的警告接听。电话里先是电流声，继而传出与黑衫老人相同的敲门声。第19章被鬼包围后，他试图逃跑却撞上黑衫老人，随即遭老人手掌袭击并发出临死尖叫。（第17章、第19章）

## 出处

- 钱万豪接听王老师号码前，杨间已明确要求所有人不要接诡异来电。赵磊和杨间先后试图阻止，但他认为王老师可能来救人，仍接通电话。（第17章）
- 听筒中没有王老师说话，只有“嗤嗤”电流声；随后一长两短的敲门声传出。钱万豪惊恐后把手机丢到地上，杨间迅速捡起并关闭电话，却判断敲门声已经会把老人引来。（第17章）
- 第19章黑暗包围校门，钱万豪已失去理智，转身逃离时撞到黑衫老人。他发现自己不能动，老人抬起枯瘦手掌伸向其面部，原文接着写出其“临死前”的凄厉惨叫，因此状态记为死亡。（第19章）
- 第175章，杨间在鬼敲门鬼域内找到江艳车辆时，看见钱万豪站在车旁。其头颅缺失半片、身体发黑腐烂，却仍穿着第七高中时的衣服；杨间判断他已成为鬼奴。鬼烛烛光靠近后，钱万豪被迫后退，不敢进入光照范围。（第175章）
