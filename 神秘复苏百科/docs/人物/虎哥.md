# 虎哥

## 简介

“虎哥”是第188章在观江花园荒宅直播探险的年轻男子，真实姓名不详。他与持手机拍摄的同伴把将被拆除的旧宅当作直播题材，白天踩点、计划夜间再探，并在镜头前以“老铁”“双击666”等话术吸引观众。（第188章）

## 经历

#### 第188章·闯入荒宅直播

杨间进入荒宅一楼后，虎哥和同伴也从后方进入。他在直播中将此处称为村中有名的鬼屋，声称要带观众探索未知世界；杨间喝问时，他被吓得想逃，得知张伟也在场后又留在屋内。杨间要求二人离开，虎哥反以直播为由顶撞，并用碰瓷式言语威胁。张伟认出虎哥曾直播拍到自己野外如厕，立即追打他；二人纠缠期间，杨间独自上楼发现鬼镜。（第188章）

#### 第190章·持棍冲突后求饶

虎哥与张伟在楼下互殴，直播同伴持续拍摄。被杨间拉开后，他拿起废旧农具上的木棍，扬言要以一敌二并打断杨间的腿；杨间以特制手枪击碎木棍，木屑划伤其脸，使他因枪声和持枪者身份恐惧跪地求饶。杨间没有继续伤害他，只让他向张伟求饶，并借直播说明自己是拥有持枪资格的大昌市国际刑警。虎哥此后是否停止直播、是否离开荒宅或得知鬼镜危险，原文未写。（第190章）

## 出处

- 户外直播闯入荒宅并与张伟冲突。（第188章）
- 持棍威胁后被杨间开枪震慑。（第190章）
