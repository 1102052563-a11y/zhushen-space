# 一郎

## 简介

一郎是岛国除灵社部长，约五十岁，秃顶、身材微胖。在杨间首次神户行动受挫、酒井死亡且三尊鬼瓷被消耗后，他闯入三岛与杨间的视频会议，以黄金损失和行动失败为由公开要求三岛辞职，并主张派山崎取代杨间处理鬼敲门事件。其全名、具体部门、驭鬼者身份、是否拥有灵异能力及与三岛的长期权力关系均不详。（原文回跳第663章）

## 性格与处事

一郎把首次行动视为管理和资源损失，而未表现出对神户现场风险的理解。他称五十吨黄金、酒井和三尊鬼瓷的损失都应由三岛负责，并把杨间定为“失败的人”，要求其离开；这与亲历行动风险的长泽及坚持继续委托的三岛直接冲突。其言论是否源于真实的处置方案、派系斗争，或单纯借机夺权，原文未能确认。（原文回跳第663章）

## 经历

#### 原文回跳第663章·闯入视频会议逼三岛辞职

杨间在温泉旅店与三岛视频通话时，一郎突然在镜头外出现，指责三岛的外援决定令除灵社蒙受损失，要求其立即引咎辞职并称会推荐更合适的继任者。三岛要求他会后再谈，一郎拒绝并继续施压；惠子将争吵翻译给杨间，令杨间提出可退还报酬、让一郎自行处理事件。此事险些中断委托谈判。（原文回跳第663章）

#### 原文回跳第663章·主张派山崎接替并被逐出会议

一郎称杨间未能一次关押敲门鬼，便应“滚蛋”，并断言山崎前往一定能成功。三岛反驳山崎进入神户鬼域只会死亡，指出杨间已经带酒井之外的人员生还、保护普通人惠子，首次受阻是多重灵异干扰而非无能。一郎仍以权位争论影响会谈，最终三岛命社员将其带离会议室；一郎叫骂离开，是否因此失职、后续能否继续影响行动不详。（原文回跳第663章）

#### 原文回跳第664章·私下鼓动田野、安排抢功行动

被赶出会议室后，一郎在机场外发泄对三岛的不满，私下召见仍不服杨间的田野。他以三岛“过分相信外人”、自身和田野被轻视为由煽动对方，又称田野和山崎带队必能解决神户异常；田野因此接受一次越过三岛安排的私自行动。一郎表示会擅自进行安排，实际只要求田野进入神户市走一趟。（原文回跳第664章）

一郎的盘算并非真以田野成功为前提：若杨间关押敲门鬼，他可将山崎、田野的行动包装为己方功劳；若杨间失败，则可把私自进入理解为有魄力的支援、借机逼三岛下台。这是原文直接揭示的利益计算。山崎后来未夺得关押物，田野死亡、山崎被未知厉鬼追杀，一郎是否获知后果或受何处分不详。（原文回跳第664章、原文回跳第666章至原文回跳第668章）
