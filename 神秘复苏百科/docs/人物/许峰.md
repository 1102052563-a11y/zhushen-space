# 许峰

## 简介

许峰是小春市国际刑警。第300章，杨间为追查一部属于失踪刑警的卫星定位手机而调阅本地负责人档案，得知该刑警名为许峰。档案显示许峰已失踪十天；第304章，杨间在异常公交末排认出仍存活的许峰。许峰已在车上坚持七八日，神情憔悴、满眼血丝，手机却不在其身上。第307至310章，他在公交熄火后离车，以身体腐烂、如尸体般平躺的状态抵住哭坟鬼哭声，仍能保持意识；杨间推测此状态下他无法行动。其能力名称、实际失踪经过、躺尸状态的条件与能否离车均不详。（第300章、第304章、第307章至第310章）

## 档案信息

- **信息隐藏**：许峰档案没有代号，也没有驭鬼者能力介绍，相关栏目注明其故意隐藏信息。杨间据此评价其对总部也保持谨慎，但不能补充为已证实动机。（第300章）
- **精神评估**：档案记录为“良好”，不能代表其失踪时或当前精神状态。（第300章）
- **手机信号**：刘小雨定位到其卫星定位手机的断续移动信号，怀疑手机摔坏后被人捡走。杨间确认它在一辆公交车上；第304章许峰称手机在出事时已在车上遗失，后来未能找回，因而怀疑它可能落到某只鬼手里，但这只是其推测。（第299章、第300章、第304章）

## 经历

#### 第299-300章·失踪手机追踪开始

刘小雨没有要求杨间立即调查许峰失踪案，只请求取回可能装有重要线索的卫星定位手机，因为当地缺乏追踪断续信号的设备。杨间接受委托，以自己的卫星定位手机锁定信号源。第300章，他在公交站确认目标不在乘客或垃圾桶中，登上一辆公交车；车辆随后偏离路线驶向市郊，许峰、手机持有人和车辆异常的关系均不详。（第299章、第300章）

#### 第304章·在异常公交现身

公交停入荒野站点后，许峰从末排风衣男子身份出声，认出杨间是国际刑警，也承认自己正是小春市失踪的许峰。他说明晚六点后公交每经一站会有一至两只厉鬼上车，电子屏“乘客数量”记录厉鬼数；此前为二、停车后降为一。他称已观察到车辆每次只停五分钟，且下车要等待时机，自己困在车上七八日仍未等到。杨间认为其说法有漏洞而未全信；许峰能否安全离车、其掌握规则的来源均不详。（第304章）

#### 第307-310章·熄火离车与躺尸自保

公交在非站点急停、车灯熄灭时，许峰立刻下车；他后来与鸭舌帽男子说明，熄火后强留车内几乎必死，平日锁住的车窗和车门也会解除限制。他同时承认曾想把不明熄火规则的杨间留在车内死亡，以替自己占住一席驭鬼者座位，后因杨间脱险才改口提出合作。哭坟鬼哭声重现后，许峰没有逃离公交附近，而是躺在路面，身体冰冷腐烂、多处流出尸水，外观近似遗弃多日的尸体；其眼睛仍能转动，显示意识未失。杨间由其无法起身和对手枪的反应，推测这种自保状态能硬抗哭声但会使其不能动；这只是杨间的判断，未获许峰说明。（第307章至第310章）

#### 第311-315章·被迫压鬼与留车理由

张浩招来周边厉鬼时，杨间无法用鬼域覆盖许峰的躺尸身体，便用鬼影包住自身后将许峰当作重物抛向干尸新娘。许峰短暂压住目标，令张浩停止招手，却在不足三分钟内被其寒意侵蚀，半边腐烂躯体逐渐干枯；他没有死亡，并在公交重启后自行返回车内。许峰认定杨间借压鬼之名想杀自己，双方敌意加深。第315章，他承认长期留在车上是为压制自己已复苏的厉鬼，认为只要能活着，永远受困也值得；对林北向杨间透露规则十分不满。其完整能力、在车外能维持多久及是否会离车仍不详。（第311章至第315章）
