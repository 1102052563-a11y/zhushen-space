# 陈义

## 简介

陈义是负责大京市安全的国际刑警之一。第409章，他参与“鬼掐人”航班事件的现场调查，身体已有尸臭、僵硬手指和尸斑，且以厚口罩遮住嘴部；杨间据此判断其厉鬼复苏状态比自己更糟，却仍具危险性。陈义说话直接、情绪暴烈，重视辖区内灵异事件的责任与审问权。（第409章、第410章）

## 能力

- 灵异身体征兆：陈义身上有淡淡尸臭，手指僵硬、手背生出尸斑，面部以口罩遮掩；这些现象说明其身体已接近厉鬼状态，但其具体驾驭数量、厉鬼能力和复苏期限均不详。（第409章）
- 异常力量与阴冷压制：他单手提起万德路时表现出不似常人的力量，并使万德路感到阴冷侵蚀、身体失控；该效果的具体规律、是否可致死和限制方法均不详。（第410章）

## 经历

#### 第409章·参与魂瓶调查

航班平安降落后，陈义与沈良、杨间、郭凡等人检查破损行李箱。得知鬼可能被乘客带上飞机时，他主张揪出托运者，语气激烈地表示要追究责任；行李箱先被查出装有古董，陈义因迟迟未见灵异源头而失控发怒。郭凡随后从物品中辨出黑色陶瓷器的异常，研究人员确认其为近代魂瓶；钟山摔碎外层瓷器，露出内藏黄金瓶。陈义未在本章展示处理瓶中鬼的手段，但其复苏征兆和激烈反应使杨间提高警惕。（第409章）

#### 第410章·与杨间争夺万德路审问权

行李箱主人万德路被武警带到现场后，陈义先以灵异力量将其提起，要求查清魂瓶来历。杨间认为万德路是重要线索，提出由自己审问；陈义强调这里是自己的辖区、犯错者不得因熟人关系例外，并质疑杨间干预。杨间展开红光鬼域施压后，陈义没有直接冲突，选择放下万德路并要求杨间公事公办。沈良遂将万德路与后续报告交给杨间。郭凡随后抛出魂瓶，瓶内又伸出惨白手臂袭击郭凡，陈义与在场者才意识到黄金瓶中另有鬼；袭击结果和陈义的应对尚未写明。（第410章）

#### 第411-412章·协助脱险并交出魂瓶

魂瓶鬼臂抓住郭凡后，陈义直接抓住黄金瓶，试图掰开扣脸的鬼指，却因黄金隔绝自身能力而未能完成救援；他指出若郭凡死亡，至少两只鬼复苏会令事件更复杂。杨间以鬼影击退鬼臂后，陈义取下瓶子、以手掌堵住瓶口。郭凡恢复时，陈义仍认为其能力不足且可能接近复苏，建议按尸体处理；沈良担保后，他没有继续干预。陈义遵守先前不再争案的承诺，将魂瓶交给杨间，要求其负责后续安全，随后离场。其为何能徒手堵住瓶口、之后是否追查事件不详。（第411章、第412章）

#### 第508章·接管张辉死亡现场

培训基地幸存驭鬼者张辉被发现死于平安酒店外的小宾馆后，陈义已接管现场并处理其体内的鬼。张雷等人转述，现场没有灵异事件扩散且目前只有张辉死亡，因此不排除他杀；陈义是否认同该判断、如何处理体内厉鬼、尸检结论和凶手线索均未公开。（第508章）

#### 第509章·怀疑灵异并请求杨间协查

陈义在张辉死亡小宾馆附近封路时拦下杨间车辆，称自己已查过现场、没有发现可疑痕迹且已处理善后，却认为案件不简单。他否定杨间提出的人为杀害可能，理由是没有人会在大京市无故杀害张辉这类新人并招来总部关注，转而认定更可能是灵异事件；这仍是他的判断，未有现场证据公开。陈义因杨间拥有鬼域而要求其短时间协助搜寻附近目标，并希望找到后共同处理。杨间只愿在总部正式申请后考虑确认情况，不答应替陈义处理其辖区事件，双方争执后陈义令其离开。陈义之后的调查、是否发现第二名受害者及事件是否扩散均不详。（第509章）

#### 第523章·作为本市负责人列入外围调度

鬼差行动队失联后，沈良在总部会议中说明，自己已让钟山、郭凡与本市负责人陈义前往鬼画、鬼差事件附近。该安排表明陈义仍负责大京市辖区的突发风险，但原文没有写他何时抵达、具体负责何处、是否动用灵异能力，也没有写他是否看到两片鬼域消散或五名异常人影。因此不能把这次点名视作陈义已参与关押或已完成现场处置。（第523章）
