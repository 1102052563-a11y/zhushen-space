# 王珊珊

## 简介

王珊珊是跟随杨间撤离教学楼的女学生。第10章末，周正腹中脱出的鬼婴在攻击杨间失败后，沿楼梯快速爬下，从后抱住王珊珊并以细长手臂勒住其脖子。第11章杨间冒险救下她，鬼婴暂时退走；她颈后留下会扩散的青黑手掌印，随后由杨间扶到操场。（第10章至第12章）

## 出处

- 鬼婴抱住王珊珊时，双臂勒住其颈部，整个身躯挂在她身上。她惊恐尖叫，试图掰开鬼婴手臂，但已难以说话，抬头挣扎且面色憋红，接近窒息。（第10章）
- 同学用手机照向她后，看到鬼婴的婴儿脑袋歪扭看向众人；鬼婴手臂具有异常怪力，且其皮肤与王珊珊接触的位置开始逐渐黏连。第10章在此处结束，无法确认她是否被救或死亡。（第10章）
- 第11章，她向杨间伸手求救。杨间因不愿像方镜一样以同学性命拖鬼，转身一拳打向鬼婴头部；鬼婴松开她、吞咬杨间手臂后被暴增的鬼眼逼退。王珊珊因此脱险，但已瘫坐、咳嗽且浑身无力。（第11章）
- 鬼婴离开后，她颈后留下两个青黑手掌印，颜色向周围皮肤蔓延。杨间发现此异常但没有告诉她，担心鬼婴折返，扶她继续撤离；她因恐惧依靠杨间，并在第12章随他抵达操场。（第11章、第12章）
- 第16至19章，她始终把杨间当作唯一依靠。校门处被黑暗阻隔时，她拉住杨间请求救人；鬼物包围后，她抱紧杨间手臂，随其红光鬼域离开学校。（第16章、第19章）
- 第20章抵达步行街后，她仍不敢独处，向杨间提出到其家借住数日。杨间理解她并非爱慕，而是因鬼婴袭击留下恐惧；原文在此未交代杨间是否接受。（第20章）
- 第21至23章，她在公交车上仍抱住杨间手臂，请他送自己回家。到小区后，母亲王海燕误会杨间带她“鬼混”并试图用钱令杨间远离；王珊珊解释杨间救过自己，却未公开学校灵异细节。该章后她已回到家中。（第21章、第23章）
- 第29章，她返家后因颈侧两个青黑小手印反复洗澡、用刷子刷破皮肤仍不能去除。她害怕黑暗，在浴室看见鬼婴似的东西贴在十六层窗外，要求父母送自己去杨间家，并说苏蕾、赵燕、刘佳怡等人已死。父亲后来在浴室发现从窗外延入室内的密集小孩手印。（第29章）
- 第34章，鬼婴般的青黑小孩进入其房间并令灯熄灭，她尖叫着撞开父亲逃出房门；父亲在门外看见鬼婴堵门后，她精神近乎崩溃，反复要求杨间。她随后随父母躲进主卧，未能离开十六层住所。（第34章）
- 第36章，鬼婴破门后扑向她、试图张口吞食，杨间以鬼域赶到击退对方。第37章她随父母离开小区，并向杨间道歉、致谢。（第36章、第37章）
- 第38至39章，她得知颈部手印是鬼婴追杀的标记，决定接受杨间以自身鬼奴印覆盖。红光消除颈印后，其手腕留下红色鬼眼烙印；她自述感觉好转却有说不出的异常。杨间拒绝其同行要求，她留在父母身边。（第38章至第40章）
- 第128至129章，她参加幸存同学聚会时脸色苍白、体温低且死气沉沉。王彬称她先会发愣并念杨间名字，后在午夜梦游出门；医院检测其心率由每分钟四十余次降至二十次左右，却仍能正常活动。杨间判断这与其手腕鬼眼烙印有关：她尚保有意识和人性，但若杨间死亡将彻底转化为鬼奴；当前是否还会继续变化不明。（第128章、第129章）
- 第136章，苗小善阻止杨间向假上官云开枪时，杨间直接操控王珊珊抱住苗小善。她双臂冰冷、力量异常，且神色茫然，显示杨间可借鬼眼烙印支配其身体。第139章杨间让她离开洗手间并通知父亲来接，她平静照做；原文未写其后去向。（第136章、第139章）
- 第250章，大昌市阴霾危机中，她与父母来到观江花园别墅等候杨间。她脸色苍白、坐姿僵直，眼神显得冷淡，仍保有转动眼睛和回答的能力；手腕红色鬼眼烙印更清晰，近似要形成一只眼睛。她称自己有时会看见奇怪画面，并据此知道杨间住处。杨间推测是自身鬼眼接近复苏影响了她，未作进一步确认。她与父母获安排在小区住处暂避，鬼奴印后续变化不详。（第250章）
- 第260章，杨间以鬼绳上吊死亡数小时后，王珊珊在观江花园住处突然闭眼倒地。王海燕触及其身体时发现体温彻底消失，胸口也没有心跳；王彬随即想起女儿曾称“杨间死了，自己也会死”，据此推测杨间已经死亡。她是否真的死亡、是否会随杨间布局变化而恢复及鬼奴印的完整联系机制，至本章均不详。（第260章）
- 第266章，杨间复活后检查王珊珊，发现她尸体未腐、皮肤仍柔软有弹性，判断并非正常死亡而是缺少维持鬼奴状态的力量。他重新向其手腕黯淡的鬼眼印注入红光后，她立即睁眼坐起，自称状态与从前相近却又不同。她称昏死期间看见一只鬼站在“七中”门口，补全叶枫临死前未发完的“七”字信息；杨间据此前往大昌市第七中学。她的感知来源、复活后的长期状态与是否仍受杨间控制不详。（第266章）

#### 第385章·腹内残留鬼婴

王珊珊随父母在尚通大厦等候杨间时，脸色苍白、坐姿笔直，眼神比以往更显漠然；她走近时周围温度下降。杨间以无头鬼影尝试侵入其身体，发现腹内已有一只成形鬼婴，随即收回鬼影。该鬼婴被确认是饿死鬼事件中孕育的残留物，但饿死鬼既已受限，按杨间所知本应失去力量而消散；它却已存在一段时间且成长很快。杨间曾从张丽琴体内取出鬼婴，故称此事不算难题，带她与父母上楼处理。残留原因、她本人的身体影响及处理结果均未写。（第385章）

#### 第386章·鬼婴归属改判

杨间向总部核实饿死鬼仍被严密关押后，以鬼眼透视王珊珊腹内鬼婴。鬼婴蜷缩安静，却因鬼眼注视睁开猩红眼睛；杨间感到双方如同同源鬼眼般熟悉，并能借它感知另一个身体。他因此回忆王珊珊曾受自身死而复生影响、死去数日后又被复活，判断鬼婴由那次诅咒延续生命，现已与饿死鬼无关，成为自己的鬼奴。因其短期稳定、未伤害她，杨间没有立即取出，只约定会后到家中处理。鬼婴能否真正受控、是否会继续成长及对王珊珊的后续影响不详。（第386章）

#### 第391-392章·自愿寄养剥离鬼婴

杨间在别墅楼上将她腹内鬼婴以鬼域和无头鬼影剥离，她没有感到疼痛；目标被取出后呈淡青色、红眼，未攻击杨间或她。杨间说明她正向鬼靠拢、自己死亡会令她再度濒死，她则称自己不愿死，也认为活下来已胜过死在学校的同学。杨间决定豢养鬼婴后，王珊珊主动提出代为照看；她接过目标时，鬼婴没有凶相，杨间因她与目标似有联系、亦相信她的意愿，最终同意。（第391章）

鬼报纸写入三项命令后，她仍坚持寄养，理由是灵异事件会增加，养大鬼婴既可能保护父母，也不是只让杨间独自冒险。杨间给她特制手枪、黄金盒和半截白色鬼烛，安排她住进小区后方新庙并要求记录过程。王海燕和江艳均恐惧反对，她则在客厅展示盒中鬼婴，宣布将其“收养”，并向母亲重申杨间曾救全家、自己不愿只受保护；王海燕未能说服她。她最终带着鬼婴离开，但保证在家中不放出；居所是否已实际迁入、三条指令是否生效及她的安全均不详。（第392章）
