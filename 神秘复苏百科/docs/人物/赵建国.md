# 赵建国

## 简介

赵建国是国际刑警亚洲分区内、刘小雨称为“队长”的中年男子。他在第22章接手与杨间的卫星电话通话，记录大昌市第七高中灵异事件，建立黑衫老人档案并将其代号定为“鬼敲门”、恐怖程度定义为A灾难级。他同时认为杨间已成为有潜力的驭鬼者，拟招揽其负责大昌市治安。（第22章）

## 能力

- 事件统筹：在刘小雨与杨间冲突后接管通话，要求记录事件细节并建立新厉鬼档案。（第22章）
- 情报交换：以高度保密的国际网站访问权限交换杨间掌握的老人定位与敲门诅咒信息。（第22章）
- 人员安排：安排刘小雨跟进杨间，考虑以临时工方式先将其招进国际刑警体系，并另行为大昌市安排人员。（第22章）

## 经历

#### 第22章·建立鬼敲门档案

赵建国先批评刘小雨应当安慰和容忍特殊事件幸存者，随后以沉稳口吻向杨间道歉并换取其合作。杨间说明周正于第七高中五楼楼道死于鬼婴复苏，又复述黑衫老人、鬼域、电话敲门声等情况。赵建国要求刘小雨建立新厉鬼档案，听到“敲门声杀人、电话传出敲门声”等线索后神色凝重。杨间承认自己以驭鬼者身份离开鬼域，赵建国则提供特殊网站访问流程，换取雷电法王帖子与敲门诅咒的关键信息。（第22章）

#### 第22章·定义灾难级与招揽杨间

通话后，赵建国决定卫星定位手机暂由杨间保管，理由是该手机特别打造、不易受厉鬼力量侵蚀且电量可维持一年。他要求刘小雨成为杨间专属接线员，并指出周正牺牲后大昌市没有新的治安刑警；杨间又是当地人，若能负责当地治安最合适。面对刘小雨关于杨间未经培训的疑问，他提出可先做临时工、先上岗后培训。最后他将老人档案代号定为“鬼敲门”，恐怖程度直接定义为A灾难级。（第22章）

## 人际关系

- [杨间](杨间.md)：高校事件幸存者与新驭鬼者；赵建国试图以情报和职位将其纳入体系。（第22章）
