# 吴组长

## 简介

吴组长是刑警总部一名仅以姓氏和职务出现的管理人员。第578章，他在杨间死亡、无头尸体疑似厉鬼复苏的影像仍由刘小雨通讯室传回时，直接按部长命令切断通讯，并宣布封存通讯室及其中与杨间有关的资料。其全名、具体部门、职权范围、是否参与此前同朋友圈的处置讨论均不详。（第578章）

## 经历

#### 第578章·执行封存与隐瞒命令

总部人员通过卫星定位手机画面看见杨间无头尸体出现异常活动后，吴组长中断视频，称这是部长的意思。他要求封存刘小雨的私人通讯室和其中资料，不准任何人再查看，并强调不能让其他人知道杨间是被人杀死，之后对外将传播其死于灵异事件的消息。曹延华最初对该决定感到惊愕，最终以“大局为重”同意执行。吴组长只是传达并执行命令，不能据此确定该决定由其提出或其个人立场。（第578章）

## 关系

- [曹延华](曹延华.md)：在曹延华及总部人员在场时，传达部长对杨间事件的封存命令。（第578章）
- [刘小雨](刘小雨.md)：其通讯室被要求封存，卫星定位手机的监控画面亦由其处切断。（第578章）
