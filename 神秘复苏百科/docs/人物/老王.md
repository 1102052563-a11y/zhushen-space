# 老王

## 简介

老王是载杨间返回小强俱乐部的出租车司机，自称有二十年车龄，并以私房钱改装出租车性能。他起初不信杨间的刑警证件，见到真枪后才配合加速。（第215章）

## 经历

#### 第215章·载杨间赶路

大昌市天空出现青黑阴霾时，杨间要求老王加速赶路。老王担心超速处罚，直到杨间开枪证明身份才驾驶改装出租车高速行驶。他自称车辆性能接近赛车、擅长弯道漂移，并得知先前逃跑的跑车已翻车。第215章止于他准备展示漂移技术，车辆是否安全抵达尚未写明。（第215章）

#### 第216章·翻车落水后获救

老王漂移后车辆部件掉落、刹车失灵，仍试图驾驶并向妻子告别。出租车冲入人工河后翻覆沉底，车窗被淤泥封死；他一度放弃逃生，向杨间展示女儿照片。杨间以鬼域将两人移至人行道，老王脱险后未能理解转移经过。杨间随即离开，老王后续去向不详。（第216章）
