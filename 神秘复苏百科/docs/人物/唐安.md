# 唐安

## 简介

唐安是福仁商场老板，身材较矮的中年男子，早年做房地产，近年改行经营商场。商场连续发生人员失踪后，他请罗大师做法，又在第42章遇到开价五百万元的严力。（第41章、第42章）

## 出处

- 第41章，他带罗大师进入商场，希望找出近期频繁失踪的原因；罗大师要求改从送货门进入，他照办。（第41章）
- 第42至44章，他先把严力视作可疑的勒索者并让丽姐报警，停电、尸体消失和员工死亡后才确认商场异常真实。他被困于罗大师先前要求锁住的商场内，尚未与严力或杨间谈妥委托。（第42章至第44章）
- 第51至54章，杨间揭露丽姐为换头鬼并以躺地规则救他脱困后，唐安支付五百万元劳务费，和幸存者背贴地面滑出商场。他在门外获警方接应，商场厉鬼尚未被解决。（第51章至第54章）
