# 费小宝

## 简介

费小宝是费总与婉月的儿子，原文写其约六七岁、相貌清秀。他在父亲回家时从房内走出，先说自己肚子饿、想吃饭；彼时杨间已在其家中，而费总正因认出杨间而陷入恐惧。原文未说明其学业、性格、是否理解父亲的工作，以及是否接触过任何灵异信息。（第542章）

## 经历

#### 第542章·与母亲一同遗忘费总

费总因担心妻儿遭杨间杀害，跪求放过家人并向他们喊话离开本市。费小宝随婉月走出后，却在记忆影响下对费总表现出害怕和陌生；婉月拉着他躲至杨间身后，把杨间视为可依靠的朋友，并询问是否要报案。杨间随后说明，婉月和费小宝的记忆里不再有丈夫和父亲费总，只留下一个早已死亡、不负责任的男人。费小宝未在场理解或回应这一说法，只由母亲带回房内。记忆修改对其成长、是否会因周围物品或他人叙述恢复认知，以及费总死讯会否传至其耳中均不详。（第542章）

#### 第543章·父亲死亡后的处境未明

杨间离开住处后，费总从高处坠下死亡。费小宝当时仍与婉月在屋内，原文没有写他是否听见坠落声、看见小区骚乱，或得知死者身份。由于其对费总的记忆已被改写，费总死亡对他而言将如何被解释亦无从确认。此后的监护、居住安排和是否再卷入朋友圈与杨间冲突，均留待后文。（第543章）
