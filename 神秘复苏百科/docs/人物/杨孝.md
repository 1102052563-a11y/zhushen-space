# 杨孝

## 简介

杨孝是杨间已故父亲，生前与灵异事件和秦老的灵异公交车有关。他在厉鬼侵蚀后试图成为完美驾驭厉鬼者而失败，死亡后有一段带其记忆的“不完整异类”残留在梅山村噩梦中；该残留明确否认自己仍是完整的杨孝，只能视为死者记忆与灵异结合后的存在。（原文回跳第681章、原文回跳第705章）

## 经历

#### 原文回跳第681章·生前遗留的三处异常地点

杨间从母亲和亲戚处得知，杨孝生前经营村南鱼塘、村后小庙和村西树林养狗，却都没有正常经济收益；树林中的狗后来一夜消失。杨间据此和秦老旧话推测，父亲早已涉入灵异，三处地点可能用于封存或维持某种平衡。此时具体真相尚未揭示。（原文回跳第681章）

#### 原文回跳第689-691章·分尸封藏与身份谜团

杨间在鱼塘、树林、小庙取到两条惨白人腿、无头无脚躯干和两颗人头，其中一颗酷似杨孝。分尸在杨孝仍在世时已被封藏，令杨间无法理解其是否早已死亡、记忆是否遭篡改。第一颗陌生头颅与躯体同属噩梦鬼，第二颗为杨孝头颅；他将两颗头分置以保留自身在梦中的残留。（原文回跳第689章至原文回跳第691章、原文回跳第705章）

#### 原文回跳第705-706章·死后记忆残留与替代计划

杨间在梦中追逐皮鞋脚步，遇到外貌与杨孝遗像相同的存在。对方称自己是拥有杨孝记忆、却不完整的鬼或异类，记忆停留在十余年前；他死于厉鬼侵蚀和成为完美驾驭者的尝试失败。秦老曾驾灵异公交车撞击噩梦鬼，使其进入不彻底的沉睡；杨孝原有机会成为下一任公交车司机，但因失败未能接替。（原文回跳第705章）

杨孝生前将噩梦鬼身体拆分、分藏在三处，令鬼在多个感应目标间徘徊，借此保护村子并让自己的梦中残留躲避追杀。他又以长期试验留下黑色大狼狗，计划在外力协助下让狼狗取代噩梦鬼成为新源头；当杨间取得分尸并打破平衡后，残留向其交代现实中留记号、拼尸、看守尸体至半腐烂的方案。计划能否成功、杨孝残留最终是否消失不详。（原文回跳第705章、原文回跳第706章）
