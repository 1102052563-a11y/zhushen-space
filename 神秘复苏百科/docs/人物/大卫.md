# 大卫

## 简介

大卫是被困在鬼画世界中的一名高大白人男性幸存者。他与其余幸存者一样在国外鬼画事件后长期躲避鬼的袭击，得知杨间知道出口时急于离开。半年余的恐怖生活使其精神状态明显失常：他认定时间已耗尽，也不再相信温和交涉能换取生机。其全名、国籍、进入鬼画前经历及同伴关系均不详。（第624章）

## 经历

#### 第624章·持枪逼迫杨间

杨间经李阳翻译了解幸存者处境后，一名高大白人抓住他的衣襟，以手枪顶住其额头，要求立刻带众人离开。旁人称其为大卫并试图制止，指出杨间是唯一知道出口的人；大卫却认为鬼仍在城市游荡、所有人随时会死，坚持持枪威胁最有效。李阳也翻译杨间的最后警告，并说明杨间不是普通人，但大卫仍拒绝放手。（第624章）

#### 第624章·袭击失败后昏厥

杨间未再交涉，直接扭断大卫持枪手腕，将这个百余公斤的成年男子踢飞。枪在反击中走火，子弹却被鬼影挡住，杨间从额头取下变形弹头；大卫则撞上墙壁、手腕扭曲并因痛苦昏厥。其他幸存者因此恐惧杨间、把他称作恶灵。大卫是否苏醒、伤势如何，以及能否随幸存者离开鬼画，均未写明。（第624章）
