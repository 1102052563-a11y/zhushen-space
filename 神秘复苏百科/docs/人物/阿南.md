# 阿南

## 简介

阿南是凯撒大酒店四层异常走道内的年轻男子，约二十多岁，穿中山装。他原在71号房隔壁，听见香兰因杨间出声而停止弹琴后立即冲出，显然十分在意香兰的安危。杨间在鬼域中未能隔墙发现他，说明异常走道墙体能阻断当时的鬼域探查；阿南如何进入、被困多久及原本身份均不详。（第603章）

## 经历

#### 第603章·将杨间误认作异常存在并带香兰逃离

阿南听见房内异动后赶到香兰身边，先询问她是否受伤；他见杨间凭空现身，惊讶对方并非鬼。杨间追问二人为何会在走道、是否知道外面有踩脚印杀人的鬼时，阿南没有提供来历。他注意到杨间不认识自己和香兰，认为其像不正常的“外来者”；香兰则从杨间话中猜测对方由出口而来，请求带路离开。阿南不愿相信杨间，称不能继续浪费时间，拉开香兰后绕过杨间向外逃走。二人的方向却是异常走道更深处，显示他们未把酒店方向当作可靠退路。第604章71号房重传琴声后，杨间推测二人急走或因知晓房中存在鬼，但该推测未获二人证实。阿南的真实目的地、对房间规则的了解和生死均不详。（第603章、第604章）

## 关系

- [香兰](香兰.md)：阿南听到她受惊后立即赶来，并带她一同逃向走道深处。（第603章）
- [杨间](杨间.md)：他不信任突然出现的杨间，认为其不像正常活人，拒绝停留说明情况。（第603章）
