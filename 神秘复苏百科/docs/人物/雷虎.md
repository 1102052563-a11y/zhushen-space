# 雷虎

## 简介

雷虎是小强俱乐部会议厅内听命于王小强的光头壮汉，携带枪械、黄金制特殊子弹、烟雾弹和手弩等装备。他把自身及同伴视作足以对付驭鬼者的专业武装人员，面对杨间时主张直接开枪，并以五六把枪指向对方为依仗；但他不了解杨间鬼域的范围和幻觉能力，实际始终被困在鬼域中。（第146章、第147章）

## 经历

#### 第146章·率武装人员开火

小强俱乐部决定清除杨间后，雷虎在会议厅内先以言语催促王小强动手，称普通人也能杀死驭鬼者。王小强示意后，他命令同伴开枪，并取出黄金制成的特殊武器；众人以特制子弹连续射击坐在会议桌旁的杨间，却只打到鬼域制造的假象。杨间借幻象掩护，以手枪先后击杀两名武装人员，雷虎等人仍未察觉已在鬼域内。（第146章）

#### 第147-148章·识破幻象失败并遭操控

王小强指出眼前杨间可能是幻觉后，雷虎投掷含黄金成分的烟雾弹，意图显出鬼或锁定真身；烟雾中他又命同伴以手弩射向人影，结果射杀一名俱乐部股东。杨间随即在其背后用枪处决两名持弩者。第148章，雷虎继续依照枪声寻找杨间真实位置，其同伴却被无头鬼影肢解；鬼影随后融入雷虎脚下并进入其身体，他本人没有立即察觉。杨间命令已被控制的雷虎杀死王小强，雷虎取下王小强头颅；该动作之后的雷虎下场，原文截至本批未另行交代。（第147章、第148章）
