# 山崎

## 简介

山崎是除灵社部长一郎在视频会议中提出的替代人选。一郎认为杨间首次进入神户鬼域后未能关押敲门鬼，便声称只要派山崎前往就“一定会成功”。三岛当场否定，认为山崎若进入现场必死。山崎本人没有出场、没有说话，也未展示能力，除灵社是否真的考虑派遣他不详。（原文回跳第663章）

## 已知定位

山崎至少被一郎视为有资格处理灵异事件的人，可能是除灵社驭鬼者或行动成员；但这只是由一郎提出其名字所能推知，不能据此确认其驾驭厉鬼、战绩或真实实力。三岛以“必死”评价他的现场生存可能，反映的是三岛对神户多重灵异风险的判断，不等于已经发生的结果。（原文回跳第663章）

## 性格与能力判断

山崎光头、身形强壮，外表接近普通人，却给杨间、长泽以明显诡异感；长泽称他是岛国少数顶尖驭鬼者之一，被评为有资格成为“玉”的人物，地位很高。山崎没有展示具体厉鬼能力，且自认没有鬼域；当半腐烂头颅厉鬼盯上他时，体内鬼迅速受压制，令其无法正面抵抗，只能逃跑。其“玉”称号的具体评价体系和真实战力均不详。（原文回跳第666章、原文回跳第668章）

## 经历

#### 原文回跳第664章·受一郎私下驱使进入神户市

一郎为争夺三岛权位，私下煽动田野并安排山崎参与未经社长同意的神户行动。山崎与田野、皮包骨男子在杨间二次关押敲门鬼后赶到现场，时机恰好落在关押物尚未运走之时；其具体出发命令、是否完全知晓一郎意图不详。（原文回跳第664章、原文回跳第666章）

#### 原文回跳第666章·试图接手敲门鬼关押物后退让

山崎到场后先称赞杨间的处置，随即要求三人接手脚下箱体、请杨间回去接受惠子照料，实则试图取得敲门鬼及事件功劳。杨间声明箱内厉鬼归自己，严厉警告后，山崎阻止田野继续顶撞，评估杨间两次对抗厉鬼后状态仍好、强行交手可能两败俱伤，最终表面道歉撤离。山崎此举是理智规避冲突，还是为另找机会夺取利益，尚不能完全确认。（原文回跳第666章）

#### 原文回跳第666-668章·遭半腐烂头颅厉鬼追杀

山崎为不空手而归，带人去十字路口调查残余灵异，田野喊住背对行走的人影后被其回头瞬杀。厉鬼随后把灰白眼睛转向山崎，他感到生命威胁、立即逃离；在神户市连续逃过三条街后仍被不同体型的同一头颅厉鬼找到。他确认目标会借男女老少等不同身体出现，自己一被盯中就遭受厉鬼压制，无法使用手段反击。第668章末他仍在逃亡，生还并非安全。（原文回跳第666章、原文回跳第668章）

#### 原文回跳第669-673章·死象显现后挑衅杨间并被杀

山崎从神户逃回大阪后参加除灵社宴会，脸色发白且对杨间抱有明显敌意。他使用手摇式放映机，影像预示自己会在大阪街头被头颅厉鬼斩首；该死象令他更加急于寻找自救办法。其后金川、藤原、石田等人画面出现相同厉鬼，山崎把杨间未被预示同类死亡视作其掌握躲避方法的证据。（原文回跳第669章至原文回跳第672章）

杨间的血池死象出现后，山崎要求其交出抵御厉鬼的方法，并以杀死惠子威胁。他试图借除灵社其他成员在场压迫外来者杨间，掀翻餐桌、释放体内鬼反抗五层鬼域。杨间以鬼域压制其寄生鬼，再用鬼手捏碎其颈骨、将尸体转移消失；山崎体内一具蜷缩长发女尸形态的鬼无法被鬼域送走，留在原位置。山崎当场死亡，其先前放映出的头颅鬼斩首死象未成为实际死因。（原文回跳第673章）
