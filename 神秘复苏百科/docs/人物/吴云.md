# 吴云

## 简介

吴云是第591章在平安大厦顶层会议室出现的朋友圈驭鬼者。杨间从费总留下的资料中得知，他是朋友圈的重要人物之一，与此前逃离的贺天雄同被列为组织重要人物；资料没有公开其具体职务、负责区域、驾驭厉鬼的名称、加入组织的经过或与方世明、姜尚白的上下级关系。（第591章）

他成为驭鬼者源于一次户外探险。资料称其或因和朋友打赌、或为证明胆量，曾去农村坟地，在一块废弃墓碑上刻下自己的名字；此后便像是被厉鬼纠缠。该段来历存在两个并列说法，不能确定他当时的实际动机、墓地位置、碑文内容和所遇厉鬼身份。（第591章）

## 能力

- **墓碑诅咒**：吴云身上会不断渗出鲜血，血迹在体表勾画出彼此未连成整体的字迹。第591章，杨间五层鬼域的红光消磨会议室时，这种流血和字迹替他抵住袭击，使其没有同其他人一样被直接送走；杨间将其认作一种诅咒。诅咒形成的完整文字、触发条件、能否主动使用及代价均不详。（第591章）
- **有限的生存抵抗**：第593章，鬼手从吴云眼眶、脚踝等身体内部伸出时，他无法自行扯掉，只能拿刘栋遗留的腐烂人头与自己对视。人头逼退鬼手后，他脸部皮肤仍开始腐烂、剥落，且感觉不到疼痛，显示其自身诅咒并未消除。该抵抗依靠的是刘栋遗留之鬼，并非吴云本人的能力。（第593章）

## 经历

#### 第591章·在五层鬼域袭击中幸存

平安大厦顶层会议室讨论是否撤离时，红光突然照入，吴云与其他人一同发现身体和体内鬼都像字迹被擦除。大多数与会者被送走，顶层三层建筑也随之消失；吴云身上流血并形成不完整字迹，因墓碑诅咒替他抵消部分影响而留下。他随即看见只剩姜尚白、方世明及刘栋腐烂人头的现场，承认无法理解为何自己等人会遭疑似鬼眼袭击，并担心下一次来袭会先使自身诅咒复苏。（第591章）

#### 第592-593章·借腐烂人头抵御鬼手

第一次袭击后，吴云主张立即离开，方世明则决定先去安全屋确认情况；吴云被迫用衣物包起刘栋的人头同行。三人准备下楼时，鬼手从其身体各处钻出，甚至从眼眶抓住脸颊。他把腐烂人头举向自己，灰白眼珠聚焦后，鬼手立即退回体内消失。吴云立刻以衣物重新遮住人头，避免长期对视；短暂接触仍令其脸部皮肤剥落。他因此确认人头可挡鬼手，却无法确认其规律和是否还可再次使用。（第592章、第593章）

#### 第594章·被鬼手杀死

方世明的“照片”替身死后，吴云听姜尚白判断杨间可能未死，转而责骂方世明行动失手。他尚未获得脱离红光鬼域的方法，杨间已在第三层鬼域中以鬼手直接掐住其颈部；墓碑诅咒被压制，吴云无法抵抗，颈骨折断后身体悬在半空死亡。其尸体随后和刘栋腐烂人头一并被杨间以鬼域取走。吴云死亡后，诅咒文字的具体内容、尸体是否会因厉鬼复苏而产生异变及杨间如何处置遗体均不详。（第594章）

## 关系

- [方世明](方世明.md)：方世明决定转往安全屋时，吴云服从并携带刘栋人头；后者死亡后，他指责方世明未能杀死杨间。（第592章、第594章）
- [姜尚白](姜尚白.md)：二人共同经历红光与鬼手袭击；吴云死亡后，姜尚白失去最后一名现场同伴。（第591章至第594章）
- 刘栋：吴云以刘栋死后遗留的腐烂人头暂时逼退鬼手，但不能长期与人头对视。（第592章、第593章）
