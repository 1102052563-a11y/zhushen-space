# 钱志强

## 简介

钱志强是中山市第三医院负责童倩抢救的主治医生。童倩获救后，医院为这名中山市国际刑警集中医疗资源，病房所在楼层由警卫戒严；钱志强与数名专家持续观察其病情。他初见前来探望的杨间时，注意到对方呼吸沉重、心率不稳，握手又感到异常阴冷和不似常人的力量，但未据此作出灵异判断。（第342章、第343章）

## 经历

#### 第342-343章·说明童倩病情

经张高出示证件，钱志强向杨间说明童倩送医前有多处骨折、内脏出血和严重头皮外伤，但伤势已暂时稳定。真正令医院束手无策的是脑部检测：设备多次查不到任何脑电波，按医学常规应近似脑死亡，童倩身体却仍保持正常机能。面对其他医生提出深度休眠的可能，他说明医院已反复检测而无结果。杨间要求独处尝试处理后，钱志强与专家离开病房；原文未写他获知灵异原因、参与后续治疗或童倩是否苏醒。（第342章、第343章）
