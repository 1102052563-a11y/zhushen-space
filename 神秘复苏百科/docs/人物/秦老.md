# 秦老

## 简介

秦老是刑警总部内被曹延华称作“秦老”的老人。第489章，总部得知休息区驭鬼者发生冲突后，曹延华认为普通工作人员难以压住队长计划候选人和特殊异类，遂命人请秦老前往，以震慑并压低现场冲突。原文未给出其完整姓名、职务、是否为驭鬼者、拥有何种能力，不能仅凭曹延华的安排判断其具体权限。（第489章）

## 经历

#### 第489章·受命前往休息区

曹延华获知高志强所在十二号休息室附近发生争斗后，担心顶尖驭鬼者在总部互相冲突会造成损失，先让工作人员通知秦老走一趟，并称应由他“杀杀这些人的锐气”。此时秦老是否立即收到通知、如何赶到、是否实际采取制止手段，原文尚未写明。（第489章）

#### 第491章·随总部人员现身

杨间以鬼手压住已化为鬼的高志强、正尝试借无头鬼影驾驭目标时，王小明、李军、曹延华等人来到现场。秦老拄着拐杖、满脸老人斑，站在人群中；在场驭鬼者的鬼对他产生感应，并共同生出“危险、极度危险”的判断。他没有说话或出手，危险感来自何种灵异、是否已发现杨间的行动及其后续态度均不详。（第491章）

#### 第494-497章·引出民国猜测并强行调停

第494章，杨间发现鬼眼持续回避秦老的身影，曹洋则转述卫景曾提及一位将过百岁生日的老人。杨间据此推测秦老可能是从民国活到现在的人，甚至可能是那个时代的驭鬼者；曹洋明确只是在猜测，秦老何时成为驭鬼者、是否真的经历民国均未证实。（第494章）

第497章，杨间与姜尚白及朋友圈成员在会议桌前对峙时，秦老敲拐杖要求所有人坐下。八名站起的驭鬼者立刻不自主坐回椅子，杨间此前砸碎的实木凳也恢复在其身下；众人随后感到体内鬼都沉寂。秦老没有解释这是否为灵异能力、如何同时影响人和物，亦未参与鬼差方案讨论。（第497章）

#### 第499章·被说明为天生驭鬼者

曹延华带杨间、王小明和秦老前往总部地下物品库途中，杨间继续询问民国时期的信息。王小明称秦老出生于民国时期，是孤儿，曾被一户秦姓人家收养；养父母在他五岁时死亡，他不知养父母名字，也未进入那个时代的驭鬼者圈子。秦老仅以动乱年代能活下来已属不易回应，未自行补充经历。（第499章）

王小明继而说明，秦老在胎中就已与鬼融为一体，体内的鬼永久死机，故属于天生驭鬼者，可近似视为“鬼”。他猜测秦老母亲怀孕时可能使用某种方法令鬼在胎儿中死机，也承认这只是自己的推测，秦老也可能是不可复制的特殊个体。杨间因不知秦老具体灵异而认为其不是自己当前能对抗的对象；秦老对该判断未置评。其所驾驭之鬼、死机方式、是否真可复制及完整能力仍不详。（第499章）

#### 第528-529章·开启鬼门并迫退门后异象

曹延华为救出失联行动队，带秦老到总部地下收容处解开鬼门锁链。首次开门时，秦老拉开无把手门板，面对门后黑暗、密集扣框手指和逐渐显出类似自己面孔的腐烂人头，仍留在原位观察；同时另有苍白脚底板从黑暗一侧探出。时间到后，他以拐杖敲地，令手指、人头和脚底板退回，木门重重关闭。此举的具体灵异机制、他是否压制鬼门本体和后续能否再次开门均不详。（第528章、第529章）

#### 第534章·准备第二次开启鬼门

首次开门关闭八分钟后，秦老按曹延华要求在第二次约定时间前重新走到鬼门前准备开启。曹延华称此次尤为关键，秦老只答应会留意；原文尚未写他实际拉门、门后是否再有异象或能否接回行动队。因此不能将准备动作当作第二次开启已经完成。（第534章）

#### 第535-536章·第二次开门接回行动队并关闭通道

秦老在约定时刻开启鬼门，使黄岗村砖瓦房的木门与总部地下房间连通。苏凡、柳三、徐一平、阿红依次经黑暗通道到达；第二次开门时限只有三十秒，秦老原本准备在五秒后关闭，以防门后厉鬼进入。两道脚步和腐烂尸体先后出现时，他拦住苏凡等人自行关门的意图，并依王小明要求让尸体通过；其后李军带伤走出。卫景自称仍有意识后，秦老以“异类”描述其状态。曹延华要求保密、讨论停止时，秦老立即关门，阻止更多异常从通道进入。开门与关门的具体机制、卫景是否受其压制及鬼门第三次能否使用均不详。（第535章、第536章）

#### 第588章·被拟派往平安大厦后暂缓

曹延华误以为杨间鬼眼已经全面复苏、又担心朋友圈借机撤离时，命沈良请秦老前往平安大厦，要求组织成员不得离开并继续承担后果。沈良提醒总部本身也处在鬼域范围，秦老离开或使总部失去保障；李军随后确认总部人员已被鬼域隔绝，曹延华遂暂缓这项调动。秦老是否收到命令、是否已经出发或对鬼域作出判断均不详。（第588章）
