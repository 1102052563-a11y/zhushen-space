# 黄飞

## 简介

黄飞是灵异论坛会员，与罗素一、林落梅一同受命到观江小区接触杨间。三人遭杨间反制时，他被有意留下解释，称论坛成员已向刑警总部备案、并非非法组织；其自身驾驭厉鬼和在论坛内的职位均不详。（第281章）

## 经历

#### 第281-282章·夜访失败后的交涉

林落梅的诡异喊话进入杨间住处后，杨间在不足一分钟内以鬼绳、鬼影和鬼域制住三人。黄飞请求杨间放人，提醒驭鬼者死亡可能导致体内厉鬼复苏，又急忙否认这是威胁。因杨间坚持追问来意，他交出手机，说明管理员想与杨间交谈；杨间却借电话限管理员十分钟汇十亿元，并将三人继续扣作筹码。

论坛完成付款后，杨间解除鬼影限制并将三人留在大昌市外郊区。黄飞由环境变化和地上的木头判断，杨间先前以鬼域制造假象并预先选好郊外位置，若谈判失败便会在远离城市处杀死三人，以免厉鬼失控危及城区。他认为夜间喊话的做法错误，也判断杨间不会加入论坛；其后续任务和是否再接触杨间不详。（第281章、第282章）

## 出处

- 灵异论坛会员，负责向杨间解释来意并联系管理员。（第281章）
- 判断杨间已用鬼域把三人移至市外，以防死亡后厉鬼在城内复苏。（第282章）

#### 第346-347章·留守中山市郊外

黄飞与罗素一、林落梅同住中山市城郊豪宅。罗素一提出离开时，他担心违背赵建国的支援征调会遭通缉，起初不愿连夜逃走；杨间到来征调林落梅后，他也以灵异论坛身份和杨间强势态度表示不满，但无法改变总部支持征调的结果。罗素一决定陪同林落梅后，黄飞本想一起前往，罗素一以三人约定“总得活一个下来”为由要他留守，他遂坐回原处。原文未写他是否离开中山市、与总部联络或再收到同伴消息。（第346章、第347章）

#### 第374章·别墅庆生

中山市目标被限制后，黄飞仍在罗素一城郊别墅。他与罗素一架起烧烤架、饮啤酒，庆祝从事件中活下来；罗素一发现杨间回房时，黄飞说明杨间已把刚烤好的肉串全部拿走。黄飞没有参与医院限制、童倩拼接或总部善后，其后是否继续随罗素一、是否离开中山市均不详。（第374章）

#### 第376-378章·避开别墅复活事故

杨间从别墅内突然出现、称童倩鬼脸复苏而准备撤离时，黄飞与罗素一、林落梅听见混杂的哭笑声，马上随其通过鬼域转至溪边。杨间说明以赵磊身体和三张脸尝试复活童倩后，黄飞认为这种颠覆生死、操弄厉鬼的做法可怕；他没有参与后续确认。童倩醒来、三张脸能开口且影响旁人情绪后，他仍同罗素一保持警惕。其是否接受杨间的保密要求、何时离开中山市均不详。（第376章至第378章）
