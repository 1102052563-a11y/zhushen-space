# 欧阳天

## 简介

欧阳天是小强俱乐部驭鬼者。第86章夜间，他被杨间误认为厉鬼而中枪，伤及下体；之后以手枪回击却被鬼眼视野下的杨间避开。第90章，他在灵堂内与同伴被脚步分开后死亡。（第86章、第90章）

## 出处

- 杨间、张韩追逐脚步声时，他从巷中出现，杨间开枪误伤其下体；他提出以还击一枪抵消，向杨间射击后未击中，双方在张韩劝阻下停火。（第86章）
- 贺胜死亡后，他虽然与杨间有旧怨，仍选择留下争取第二鬼驾驭方法。（第88章）
- 灵堂搜索期间，脚步声把其他人引到楼道，欧阳天失踪；众人回到大堂时发现其尸体呈扭曲姿势靠在棺材边，体内厉鬼已不见。（第89章、第90章）
