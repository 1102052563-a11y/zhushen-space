# 李跃

## 简介

李跃是驾吉普车到梅山村外树林寻找尸体的男子，约三十岁，穿羽绒服、面容阴沉。他与两男一女同行，自称因一封黄色信封得知树林中有尸体；他称自己和杨间不是同一路人，却承认是“信使”，并警告杨间不能看信。信使的组织、能力、信件来源和任务规则截至本批不详。（原文回跳第692章、原文回跳第693章）

## 经历

#### 原文回跳第692章·向杨间打听树林位置

李跃驾车在梅山村外向提着装尸袋的杨间问路，先以找亲人、迁坟为由掩饰，后展示一张拍有树林和小木屋的旧照片，出一千元请杨间指路。他交付的钱是早已淘汰却崭新的旧版纸币，且车内同伴目光异常，使杨间判断一行人目的不单纯。杨间仍指出树林方向，并暗中跟随。（原文回跳第692章）

李跃离开后向同伴坦言，杨间提着装尸袋、手掌冰冷、情绪缺乏波动，极可能是驭鬼者；他主张避开这种人，先完成自己的目标。他知道大昌市负责人名为杨间，却起初不相信路边相遇者就是本人。（原文回跳第692章）

#### 原文回跳第693章·发现尸体被取走并承认信使身份

李跃一行按照片找到小木屋，发现地下红木箱已被打开、尸体不见，随即确认杨间先到。杨间现身后直接承认尸体在自己手中，要求说明来历；持枪同伴威胁无效，李跃转为交涉，听到“杨间”姓名后与同伴震惊、确认对方是大昌市负责人。（原文回跳第693章）

他最终拿出一封黄色信封，称自己等人只因信件才知道尸体位置；面对杨间索看，他拒绝并称“不是信使看不了，对你没好处”。李跃是否继续争取尸体、信件如何指向梅山村、同行者身份及信使任务后续均未写明。（原文回跳第693章）
