# 酒井

## 简介

酒井是岛国除灵社安排给杨间的三名神户行动协助者之一，为中年男性驭鬼者。他与长泽共同在木屋外顶住受灵异影响的大门，表现出一定服从和求生判断；但进入敲门鬼鬼域后不久即遭人头气球无预兆袭击，头颅脱离身体升上天空，成为本次行动第一个确认死亡的协助者。其全名、所驾驭厉鬼、能力效果、加入除灵社经历均不详。（原文回跳第654章、原文回跳第655章）

## 经历

#### 原文回跳第654章·木屋外协助守门

杨间在木屋内以鬼手和死人头压制穿衣尸体鬼时，酒井与长泽留在门外。木屋大门被未知力量强行关闭，长泽几乎被门夹住，酒井也上前帮忙；待屋内厉鬼暂受压制，大门异力消失，他将门拉开、险些跌坐在地。酒井坦言若再持续下去，自己不得不动用厉鬼能力，而以普通驭鬼者的复苏代价难以在灵异对抗中长期支撑。（原文回跳第654章）

杨间带鬼童撤出后，认可酒井和长泽没有逃走，命二人继续随行。酒井没有表现出反对，随队由美岛继续指引，深入受敲门鬼影响而老化的神户街区。此时他仍未展示自身驾驭的厉鬼或攻击能力。（原文回跳第654章）

#### 原文回跳第655章·被人头气球斩首身亡

队伍靠近敲门鬼所在区域时，酒井最先发现窗边死者，后与众人遭遇未知来电和空中人头异常。美岛称敲门鬼似在附近后，酒井突然无声倒地，面色红润、闭着眼的头颅从脖颈断口飞起，逐渐升向空中；杨间未能察觉袭击前兆，只能推测酒井在说话间满足人头气球的杀人规律。酒井没有机会动用能力或自救，当场确认死亡。（原文回跳第655章）

酒井头颅离体后，尸体因驭鬼者死亡而出现继续活动迹象，体内厉鬼开始快速复苏。杨间为避免其干扰追踪，把无头尸体丢入附近房屋暂时隔离；白色鬼烛点燃后，复苏尸体仍自行走出并朝十字路口靠近。该复苏鬼的能力、能否被控制和后来结局均不详。（原文回跳第656章、原文回跳第658章）
