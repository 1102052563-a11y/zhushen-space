# 王晗

## 简介

王晗是随钱总进入张显贵聚会的驭鬼者，约三十岁，常穿整洁西装，脸色麻木、情感观念已受厉鬼侵蚀。他具备鬼域，能令大厅灯光闪烁并准备与杨间对抗；张伟以黄金手枪击中其额头后，他仍能行动。其驾驭厉鬼数量、鬼域规律、来源和原本目的均不详。（原文回跳第725章、原文回跳第726章）

## 表现与立场

王晗认为普通人如同可随意踩死的对象，称江艳为“货色”“狗”，反映其虽有活人意识却已缺乏正常情感与伦理。杨间判断他被厉鬼侵蚀过深，观念正脱离人的范畴；这一判断不等于王晗已成为异类或无法恢复。（原文回跳第725章、原文回跳第726章）

他知道杨间、朋友圈覆灭和杨间与总部的矛盾，显然掌握部分灵异圈情报。与杨间初次试探后，他承认杨间鬼眼令自己产生随时触发死亡规律的压力，估计正面对抗撑不过十秒，因而未在聚会人群中升级冲突。（原文回跳第726章）

## 经历

#### 原文回跳第725-726章·冒犯江艳并与杨间试探

王晗感知到杨间所在位置后横冲直撞穿过宾客，试图推开挡路的江艳。杨间及时拉开江艳，张伟以两把黄金手枪指住他；王晗自称想认识杨间，认为普通人不值得在意。杨间要求其下跪道歉，双方一度以鬼眼和闪烁灯光示警。张伟开枪击中其额头，王晗头部留下腥臭血孔却未倒下。（原文回跳第725章、原文回跳第726章）

王晗判断鬼域碰撞会令在场普通人死亡，选择暂退。钱总随后在其建议下向杨间公司投资二十亿元，以此化解冒犯；杨间认为这笔钱买下了二人的命，并将王晗拉入鬼域单独谈话。谈话目的、王晗接下来提出什么要求、是否真正与杨间和解均待后文。（原文回跳第729章）

#### 原文回跳第730-731章·提出异类道路并被逐出大昌市

王晗称总部被入侵、鬼画未彻底结束与队长计划，都说明驭鬼者会成为未来主导者；他又以一名存活至少十年的老驭鬼者为例，邀请杨间接触疑似解决复苏、近似异类的隐秘群体。杨间拒绝，认为活得久而人性被侵蚀并非正确道路，反指王晗自身已在进行精神层面的另类复苏。（原文回跳第730章）

王晗承认杨间比自己掌握更多历史灵异线索，接受其警告，在聚会结束前离开大昌市。杨间明确下次若王晗继续观念失控或在大昌市闹事，可能直接杀死他；王晗所称群体、十年驭鬼者身份和异类方法均未揭示。（原文回跳第730章、原文回跳第731章）
