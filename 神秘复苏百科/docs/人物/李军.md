# 李军

## 简介

李军是随王小明到黄岗村外营地的李少将，穿军装，中年，神情不怒自威。杨间见到他时，鬼眼出现本能睁开的反应，因而判断其是危险程度很高的驭鬼者；原文未直接说明其驾驭的厉鬼和能力。（第115章）

## 经历

杨间在第114章向吴越开枪时，子弹穿过吴越头部却未伤人。王小明说明是自己让李军救下吴越，杨间据此判断这可能是鬼域能力，但未获确认。其后李军始终注视杨间，在杨间因删档问题与王小明冲突、鬼眼准备张开时，警告王小明是国宝级人物，若杨间动手，自己会亲自开枪杀死杨间，即使认为杀死人才可惜。（第115章）

第245章，大昌市外道路因阴霾被划入军事禁区，李军身着军装守在警戒线外，目睹阴霾如活物蠕动且无法被风或军事打击驱散。赵建国告知医院内行动者大多死亡、事件已定为S级后，李军提出自己复苏拼命进入；赵建国以其死亡后体内鬼可能加剧灾情劝阻。李军虽不满一座城市难获救援，仍接受维持秩序、等待杨间最后一搏及上级命令的安排。其驾驭厉鬼、能否进入阴霾和之后行动不详。（第245章）

第274章，饿死鬼受限、阴霾消散后，李军随赵建国率领先遣队乘防爆车进入大昌市。他带十名特战队员，与赵建国所带十五名刑警在警戒位置确认道路和建筑轮廓恢复可见；随后依赵建国安排，分出人员前往观江花园接王小明。第275章，杨间回到小区时，李军代表王小明、大昌市市民及国家向其致谢，并称若无杨间事件将一发不可收拾。先遣队之后的巡查、救援成果及李军是否直接参与金箱押运均不详。（第274章、第275章）

第287章，赵建国为救被杨间关入黄金箱的林龙，请李军以鬼域介入。李军身上放出阴森青绿色光芒，尝试侵入红色鬼域；杨间随即以两只鬼眼叠出第二层鬼域，将其鬼域全数覆盖。李军无法把林龙带出，其他刑警据此确认双方鬼域碰撞中李军落败。第290章，他继续作为王小明的护卫，随其乘专机返回大京市；其后是否参与研究和林龙救援不详。（第287章、第290章）
