# 秦媚柔

## 简介

秦媚柔是国际刑警方面为杨间新增的第二位接线员。第298章，杨间通过卫星定位手机联系刘小雨时由她接线。她说明杨间之后将由她和刘小雨轮流服务，原因是杨间工作重要，刘小雨一人不能保证二十四小时接听。其具体编制、此前经历和轮班安排不详。（第298章）

## 经历

#### 第298章·作为第二接线员上线

秦媚柔协助连线小春市警局，并向杨间说明接线轮值安排，承诺专业素养和服务不会耽误工作。杨间以检验心理素质为由提出与公务无直接关联的声音要求，并声称紧急事件中犹豫可能危及城市；她先沉默，后按其要求回应，杨间随即结束通话。该次互动不能证明她已被调离或失职。（第298章）

#### 第299章·暂未继续接线

第299章，原文提到秦媚柔在通讯室单间内对杨间又羞又气；同章由刘小雨重新联系杨间。原文只称杨间的操作似乎把新接线员吓退，没有记录她正式调离、受处分或不再轮值。因此她之后是否仍为杨间接线不详。（第299章）

## 关系

- [刘小雨](刘小雨.md)：两人被安排轮流为杨间提供接线服务，具体交接规则不详。（第298章）
