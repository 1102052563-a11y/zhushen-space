# 秦媚柔

## 简介

秦媚柔是国际刑警方面为杨间新增的第二位接线员。第298章，杨间通过卫星定位手机联系刘小雨时由她接线。她说明杨间之后将由她和刘小雨轮流服务，原因是杨间工作重要，刘小雨一人不能保证二十四小时接听。其具体编制、此前经历和轮班安排不详。（第298章）

## 经历

#### 第298章·作为第二接线员上线

秦媚柔协助连线小春市警局，并向杨间说明接线轮值安排，承诺专业素养和服务不会耽误工作。杨间以检验心理素质为由提出与公务无直接关联的声音要求，并声称紧急事件中犹豫可能危及城市；她先沉默，后按其要求回应，杨间随即结束通话。该次互动不能证明她已被调离或失职。（第298章）

#### 第299章·暂未继续接线

第299章，原文提到秦媚柔在通讯室单间内对杨间又羞又气；同章由刘小雨重新联系杨间。原文只称杨间的操作似乎把新接线员吓退，没有记录她正式调离、受处分或不再轮值。因此她之后是否仍为杨间接线不详。（第299章）

#### 第329章·通报中山市失联事件

杨间查询中山市负责人时，秦媚柔查到童倩负责该市，并立即补充童倩已在灵异事件中失联三天，最后通话疑似说明被困。她还确认总部已派两名国际刑警和三名外援支援。杨间警告中山市另有能篡改记忆的鬼，要求撤人；秦媚柔意识到情报复杂，随即转报赵建国。她能否参与后续联络、对该事件掌握多少额外信息均不详。（第329章）

#### 第345章·转达郭凡求救

秦媚柔通过卫星定位手机通知杨间，郭凡发出紧急求救，称自己困在凯撒大酒店三楼三十一号房，冯全已失去联系；她按信号位置判断，杨间动用鬼域约三十秒即可赶到，并请求尽快支援。杨间指出童倩旧录音提到酒店内有人为信号干扰器，若郭凡仍能发信号，可能正是鬼许可，因而怀疑这是围点设伏。秦媚柔据此说出“围点打援”的可能，仍担心郭凡、冯全死亡会使事件恶化。她在杨间坚持拒援后只能转达其要求总部配合、勿擅自指挥；她能否采取其他救援措施及二人后续消息均不详。（第345章）

#### 第393-395章·联络杨间赴京及航班异常

杨间赴大京市前以卫星定位手机联络秦媚柔，她查到其登机信息，称总部会安排接机并要求抵达后配合安排；她还确认杨间母亲在大京市的正规企业工作，待遇良好，总部会照顾其家属。第395章，她又通知杨间所乘飞机十分钟前出现异常，因飞机在空中而请他先确认，强调尚只属怀疑、不排除特殊事件；杨间批评情报不足，她解释事发突然、总部来不及收集，并就此道歉。她未亲见机上尸体，之后能否取得情报、派出支援或参与处置均不详。（第393章、第395章）

#### 第396-400章·转达就地处置决定并失去航班接线资格

杨间确认航班灵异事件后，秦媚柔按其要求调查该机近半年航班记录，确认没有类似死亡，仅有乘客突发疾病且未出意外；她据此听取其“厉鬼可能被乘客带上飞机”的分析。面对以鬼域疏散、日后再追查，或在机上解决、可能造成当场伤亡的两种方案，她参与总部电话讨论并传达“解决飞机上的灵异事件”的决定。杨间暂停行动、准备等厉鬼再杀人时，她认为乘客会受害并出言劝阻；杨间反驳其未亲历现场、无权指责。第二名死者出现后，她转达王东取得的周浩口供，却再次追问杨间此前为何不行动，杨间随即拒绝今后在灵异事件中继续与她通话，要求由刘小雨替代。她只能接受，后续是否仍参与此次事件不详。（第396章、第397章、第399章）

#### 第417章·转达酒店求援

平安大酒店凌晨枪响后，秦媚柔强行接通杨间的卫星定位手机，说明一名驭鬼者精神失常、已枪杀酒店服务人员，且有继续作案风险。她承认酒店不属陈义辖区、高明又不在工作时间，总部若另派专人至少需半小时，因而请求杨间就近处置。杨间认为总部借辖区与值班理由让自己无偿卖命，开价十根鬼烛；秦媚柔没有接受，随即中断通讯。她之后如何改派张雷、是否参与现场后续不详。（第417章）
