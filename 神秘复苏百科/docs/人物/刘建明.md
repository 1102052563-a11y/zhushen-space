# 刘建明

## 简介

刘建明是负责福仁商场案件的区警队长。第54章，他在商场外组织警员破锁救援，闻到尸臭后意识到失踪案可能比普通刑事案件复杂。（第54章）

## 出处

- 他接到商场有疑似杀人犯的报案，得知唐安曾请风水大师封锁商场，遂命警员开锁破门并谨慎观察。（第54章）
- 李经理举报杨间敲诈后，刘建明核查其档案，发现权限不足；上级告知杨间档案已转特殊部门、不得继续追查，若故意诬陷国际刑警可涉危害国家安全。他因此亲自接手后续处理。（第55章）
- 第69章，杨间报警称吴峰等人企图绑架、袭击自己；刘建明率配枪便衣警员在杨间安排的空街埋伏，将吴峰一行控制。（第69章）
