# 刘紫

## 简介

刘紫是苗小善在大京市读大学期间的室友。第483章，杨间陪苗小善在商场逛街时，她以高考、大学和找工作为话题评价杨间。原文未交代她的学校、专业、家庭具体情况或是否知道灵异事件。（第483章）

## 第482-483章·对杨间的轻视与离开

刘紫以为杨间高三辍学后找不到工作、可能准备露宿街头，称自己只是好心提醒其继续读书。她又以苗小善学业为由，要求杨间不要频繁约人外出，实际担忧苗小善与一个她认定条件较差的人交往会令自己难堪。杨间指出其是在拿自身优势比较他人短处，并表示苗小善的选择不应由室友说教；刘紫没有承认这一判断。（第483章）

苗小善换衣服回来后，刘紫邀其同走并称已叫朋友接人；苗小善选择由杨间送回学校，刘紫便独自离开，临走仍对杨间表现出厌恶。她没有接触杨间的鬼手、鬼眼或任何灵异现场，也不知道杨间的国际刑警身份；离开后的去向和二人关系不详。（第483章）

#### 第484章·从李洋处得知平安酒店的警告

刘紫被男友李洋接走后，抱怨商场遇到的杨间，并提出要去手机搜索到的平安酒店住宿。李洋闻言立刻制止，称酒店不对外开放；当她说苗小善的朋友住在那里，又说出杨间姓名及苗小善曾在大昌市读书，李洋推断那人是大昌市鬼眼刑警。刘紫因此才知道杨间并不如自己所见普通，却仍只答应以后离他远些。她没有直接向杨间道歉或联系苗小善，之后是否理解李洋所说危险不详。（第484章）
