# 刘紫

## 简介

刘紫是苗小善在大京市读大学期间的室友。第483章，杨间陪苗小善在商场逛街时，她以高考、大学和找工作为话题评价杨间。原文未交代她的学校、专业、家庭具体情况或是否知道灵异事件。（第483章）

## 第482-483章·对杨间的轻视与离开

刘紫以为杨间高三辍学后找不到工作、可能准备露宿街头，称自己只是好心提醒其继续读书。她又以苗小善学业为由，要求杨间不要频繁约人外出，实际担忧苗小善与一个她认定条件较差的人交往会令自己难堪。杨间指出其是在拿自身优势比较他人短处，并表示苗小善的选择不应由室友说教；刘紫没有承认这一判断。（第483章）

苗小善换衣服回来后，刘紫邀其同走并称已叫朋友接人；苗小善选择由杨间送回学校，刘紫便独自离开，临走仍对杨间表现出厌恶。她没有接触杨间的鬼手、鬼眼或任何灵异现场，也不知道杨间的国际刑警身份；离开后的去向和二人关系不详。（第483章）

#### 第484章·从李洋处得知平安酒店的警告

刘紫被男友李洋接走后，抱怨商场遇到的杨间，并提出要去手机搜索到的平安酒店住宿。李洋闻言立刻制止，称酒店不对外开放；当她说苗小善的朋友住在那里，又说出杨间姓名及苗小善曾在大昌市读书，李洋推断那人是大昌市鬼眼刑警。刘紫因此才知道杨间并不如自己所见普通，却仍只答应以后离他远些。她没有直接向杨间道歉或联系苗小善，之后是否理解李洋所说危险不详。（第484章）

#### 第550-554章·逃离寝室后接受观察

张霞失踪、寝室留下老旧油画时，刘紫先制止其他室友打趣苗小善；她因男友李洋曾提醒杨间身份特殊，开始相信杨间的警告。杨间带走苗小善后，她发现对方不可能在短短数分钟内准确到达女生寝室，便穿鞋追出；走廊却空无一人，进一步放大了她对闹鬼的恐惧。她独自跑到操场，再向苗小善求助并依短信找到篮球馆。（第550章、第552章、第553章）

杨间认为刘紫直接接触过油画，且不能和苗小善留在同一处，要求她接受总部安排。王泉随即让人员带她离开观察，并称事件结束后会使她恢复正常上学；刘紫虽恐惧不安，仍被带走。她是否受到鬼画影响、是否得知王悦和孙于佳的经历及观察结果，均不详。（第554章）

#### 第563章·返回寝室陪伴苗小善

鬼画被杨间限制并由王泉小队运走后，工作人员把刘紫与孙于佳送回寝室。刘紫看见杨间以不合常理的方式迅速离开走廊，仍感到惊恐；她询问苗小善杨间究竟是谁，又不敢相信对方只是高中同学。苗小善说明除杨间外，自己其余高中同学都是普通人，并说当年一所学校只剩少数幸存者。刘紫从前未相信灵异，如今听见这一经历后不再否认世界存在鬼。她是否因此留在苗小善身边、恢复正常上学或再受鬼画牵连，均不详。（第563章）
