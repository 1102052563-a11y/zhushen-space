# 贺秋燕

## 简介

贺秋燕是大昌市第七高中、与方镜一同在鬼域中逃亡的学生。第13章方镜被黑衫老人追逐、困在操场外绿化树林时，承认自己曾为拖延厉鬼而抛下贺秋燕等同学。原文没有描写贺秋燕的外貌、言行、被抛下的位置或最终结局。（第13章）

## 出处

- 方镜在第13章回顾逃亡过程时，称自己一路上留下了赵强、万峰、贺秋燕等十几名同学，以他们的性命为代价拖住身后的厉鬼。（第13章）
- 当时其他幸存学生已不敢接近方镜，却也不敢独自离开；先前自行逃走的人消失在黑暗中。贺秋燕被提及后未再出场，故页面只记录其与方镜行动和下落不明的关系。（第13章）
