# 张显贵

## 简介

张显贵是张伟的父亲。第23章原文确认张伟家住别墅、确为富二代，张显贵平日认为儿子较低调、未沾染败家习性。张伟从高校鬼域回家后不敢说明真实经历，只称和朋友出去玩；张显贵要求他吃早点后照常上学，引发父子争执。（第23章）

## 出处

- 张显贵见张伟整夜未归、电话不通后询问情况，听到“和朋友出去玩”后只要求以后不准如此，并提醒其不要上学迟到。（第23章）
- 张伟称学校闹鬼、同学死了几十个，拒绝再进第七高中。张显贵不信这些说法，要带他去学校查看，并以不给零用钱要求其继续上学。（第23章）
- 张伟表面拿书离家，实际准备去网吧。原文没有写张显贵发现儿子逃学或得知灵异事件；他是第23章中张伟家庭经济状况的直接来源。（第23章）
