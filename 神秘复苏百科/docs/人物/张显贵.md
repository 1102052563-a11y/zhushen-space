# 张显贵

## 简介

张显贵是张伟的父亲。第23章原文确认张伟家住别墅、确为富二代，张显贵平日认为儿子较低调、未沾染败家习性。第160章，他从封锁别墅的杨间处得知张伟被镜中鬼跟随，起初仍认为灵异传闻可疑，直至亲自从镜中看见二楼人影才相信异常存在。（第23章、第160章）

## 经历

#### 第160章·在别墅确认镜中人影

张显贵经别墅装修时留下的逃生通道，与黄姓家政阿姨进入已被警察封锁的住处，意外被张伟和杨间撞见。尴尬过后，他询问杨间处理的“特殊事件”及第七高中传闻，表示自己曾调查学校事件，承认存在疑点却仍要求亲眼证明。杨间说明天然气泄漏只是封锁幌子，并让其用镜子寻找别墅内的鬼。张显贵起初在沙发后未见异常，还认为自己受鬼故事影响；待镜子放在茶几上，他却看见二楼护栏旁站有人影，抬头现实处却空无一物，镜中人影随即走向走廊消失。他至此恐惧而结巴，确认儿子所述并非玩笑；镜中鬼源头与张伟的风险仍未解除。（第160章）

#### 第161-165章·以楼盘线索换取调查与收购机会

张显贵根据镜中鬼穿着工地工作服、观江花园工地近期有人失踪、张伟曾被自己带去工地等线索，认为异常可能从工地带回，随即请求杨间协助调查，并承诺报酬可以商量。他带杨间、张伟到自己参与开发的观江花园，说明项目另有钱总、秦总两名合作方，提出若杨间入住并能保证楼盘安全，便合作低价吃下另两方份额。杨间要求四成收益后，他虽认为条件过高，仍先同意以实际能力为准。钱、秦不信闹鬼，杨间以鬼域让其恐惧并困住；二人求救时，张显贵顺势提出收购。第165章末秦总指责他趁火打劫，未写明交易是否成立或工地风险是否解除。（第161章至第165章）

#### 第166-168章·完成收购并交出售楼部

钱总、秦总在杨间制造的地下环境中答应出售后，张显贵拿出预先草拟的合同，委托杨间带入其中取得签字。杨间带人脱离异常环境后，张显贵确认十亿元收购两人名下楼盘的合同已到手；他认为此价既能低价取得项目，又不会压得对方日后必然报复。面对杨间原先提出的四成利润，他提出把若干楼盘转到杨间名下，杨间只要最终利润和数栋楼、别墅，双方遂约定由他继续处理楼盘。杨间另以五千万元购买售楼部，张显贵答应安排销售人员和物品搬离，并介绍售楼部可直接居住的设施。此笔交易虽完成，工地失踪、镜中鬼源头及张伟的风险仍未调查。（第167章、第168章）

## 出处

- 张显贵见张伟整夜未归、电话不通后询问情况，听到“和朋友出去玩”后只要求以后不准如此，并提醒其不要上学迟到。（第23章）
- 张伟称学校闹鬼、同学死了几十个，拒绝再进第七高中。张显贵不信这些说法，要带他去学校查看，并以不给零用钱要求其继续上学。（第23章）
- 张伟表面拿书离家，实际准备去网吧。原文没有写张显贵发现儿子逃学或得知灵异事件；他是第23章中张伟家庭经济状况的直接来源。（第23章）
- 他在镜中看见二楼现实不存在的人影，首次直接确认别墅内存在异常。（第160章）
- 他是观江花园三名合作开发商之一，负责前部楼盘；项目工人失踪后曾停工停售以封锁消息。（第161章、第162章）
- 他与杨间初步讨论收购售楼部和合作吞下另两方份额，具体合同与分配尚未完成。（第162章）
- 钱总、秦总已在转让合同签字，张显贵以十亿元完成对其名下楼盘的收购；他与杨间约定由自己继续管理楼盘、再按利润分账。（第167章、第168章）
