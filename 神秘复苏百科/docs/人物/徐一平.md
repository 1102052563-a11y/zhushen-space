# 徐一平

## 简介

徐一平是总部鬼差行动队中被明确点名的成员，与李军、柳三、阿红、苏凡共同进入鬼画鬼域内的黄岗村，目标是寻找已被鬼画干扰的鬼差。第523章总部报告将他列入与外界失去联系的五人；此前原文一直把队内两名同行者写作未具名驭鬼者，因此除姓名、所属队伍和参与本次行动外，没有交代他的外貌、性格、驾驭厉鬼、能力、职务或过往经历。（第519章、第523章）

## 经历

#### 第519-522章·随队在黄岗村排查房屋

徐一平随李军等人在黄岗村内搜索，队伍已经进入多栋有鬼房屋，面对第七栋疑有厉鬼的三层民房仍决定继续排查。李军因鬼火使用过度接近复苏，仍坚持把握鬼差鬼域被压制的机会；柳三率先上楼，苏凡通过游戏角色死亡提示二楼危险。众人随后在二楼发现五具柳三纸人，柳三由此怀疑队伍曾在没有记忆的情况下重复到访；之后又在另一房间收容掉落的老人遗像，画面背景中有五个模糊人影。原文只确认徐一平属于五人之一，未写其在搜索、收容或决策中的个人作为。（第520章、第521章、第522章）

#### 第523章·列入失联名单与异常归来者范围

行动持续三个多小时后，总部确认李军、柳三、阿红、苏凡、徐一平五人全部失联。鬼画鬼域与鬼差鬼域不久后同时消散，中心位置走出五个人影；杨间仅认出李军和纸人柳三，故不能以目击直接确认徐一平的身份。五个人影的手掌均有相同灰白空洞，白色鬼烛点燃后又同时转头望向杨间，令他判断走出者均非活人、而是鬼。徐一平是否在失联时死亡、是否被鬼画力量替代，或其本人是否仍在鬼域中，原文没有给出结论。（第523章）
