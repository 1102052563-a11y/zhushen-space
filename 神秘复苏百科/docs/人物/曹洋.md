# 曹洋

## 简介

曹洋是参加大昌市饿死鬼事件总结会议的国际刑警，已驾驭两只鬼。他言语尖刻、善于试探与激将，会借熊文文的年幼和预知能力诱其消耗厉鬼；面对杨间时又能从熊文文的惊恐反应判断风险，未令冲突继续恶化。其两只鬼的能力、驾驭时间和驻守城市均不详。（第284章、第285章）

## 经历

#### 第284章·激将灵童刑警

熊文文在会前辱骂众人后，曹洋以更尖刻的言辞回击，并故意要求其预知一小时内谁会获胜。旁观刑警看出这是诱导熊文文使用厉鬼、提前承受复苏代价，保护人员亦立刻制止；熊文文仍受激发动能力，旋即被未来所见吓得退缩。杨间随后进门，曹洋从熊文文的反应推断，未来中杨间会因冲突介入且后果危险，因而收敛行动。

#### 第285章·试探饿死鬼行动经验

曹洋在会议中以自己同样驾驭两只鬼为由，询问是否有资格正面接触饿死鬼。杨间故意称只要洞察规律和有特别能力便可关押，曹洋联想到同样驾驭两鬼的叶枫已在行动中死亡，判断回答是挖坑而未轻信。其他刑警却有人接受了杨间淡化风险的说法。曹洋没有揭穿杨间，后续是否仍会与其冲突不详。（第285章）

## 出处

- 两只鬼的国际刑警，在会议前与熊文文争执。（第284章）
- 识破杨间淡化饿死鬼风险的说法，不接受其试探。（第285章）
