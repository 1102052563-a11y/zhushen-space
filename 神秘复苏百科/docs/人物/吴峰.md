# 吴峰

## 简介

吴峰自称某公司业务经理，第68章率数名西装男子在咖啡厅包场接触杨间。他持有福仁商场现场照片，意图以招聘、收购为名取得杨间保管的黄金盒。（第68章、第69章）

## 出处

- 他先以月薪十万元招揽杨间，再称可一千万元收购黄金盒；其名片只有姓名与电话，刻意隐藏公司信息。（第68章、第69章）
- 杨间假称盒子已上交、后又称仍在手中，诱使吴峰先转一百二十万元工资。杨间同时报警，刘建明率便衣警员将吴峰一行以涉嫌绑架、袭击刑警控制；其后下落未写。（第69章）
