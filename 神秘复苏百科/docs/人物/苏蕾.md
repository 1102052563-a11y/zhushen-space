# 苏蕾

## 简介

苏蕾是杨间的同班女学生。第3章黑衫老人引发鬼域后，她最先因课桌上的书本迅速泛黄、发霉、腐烂而尖叫，张伟以玩笑方式称桌上像有一坨东西。第5章学生逃出教室时，已被风化的楼板因人群踩踏坍塌，苏蕾掉到下一层并被裸露钢筋贯穿颈部，伤势极重；第5章结束前原文没有明确交代她是否死亡。（第3章、第5章）

## 出处

- 她在第3章目睹自己的书本变质为黑漆漆腐烂物，是教室内鬼域异变的直接受害者之一。（第3章）
- 第5章楼板坍塌后，她睁着眼躺在地上，颈部冒血、口中发出“咯咯”声音，说明当时尚未断气；钢筋从颈部穿过。杨间认出她，并认为在眼下环境中即使呼叫救护车也来不及，随即催促仍能行动的人离开。（第5章）
- 事故发生在五楼教室外的逃离途中。楼板此前已受鬼域风化，露出锈迹斑斑的钢筋；人群同时踩踏后坍塌，张伟和苗小善也掉到下一层，但原文只明确写出苏蕾被钢筋贯穿。她此后未再出场，故状态记为不明。（第5章）
