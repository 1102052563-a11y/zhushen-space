# 王岳

## 简介

王岳是大昌市玫瑰酒吧的老板。马有财曾称其是可收钱办事的驭鬼者；第161章他本人出现，承认自己经历过数次灵异事件，但本批没有展示其驾驭的鬼或具体能力。他与赵开明的谈话确认曾替赵开明杀死严力及其一家三口，却仍未写明其亲自动手的经过、动机之外的内情或其与小强俱乐部的完整关系。（第149章、第161章）

## 已知线索

#### 第149章·被供作调查对象

杨间在小强俱乐部杀死多数股东后追问严力一家遇害的凶手。马有财为换取活命，称王小明常去玫瑰酒吧喝酒，酒吧老板王岳可能知道什么；他又强调王岳是难办事的驭鬼者，并以其拥有多处娱乐场所、收钱办事来解释为何此人可能卷入事件。杨间怀疑该线索可能是临时编造，却决定前往询问，并以换头后的马有财作为担保。至第150章末，杨间尚未与王岳接触，线索真假和其与严力之死的关系均不详。（第149章）

#### 第161章·受赵开明威胁并遭鬼敲门事件波及

赵开明到玫瑰酒吧二楼包间要求王岳加入王小强、叶枫对杨间的行动。王岳称杨间不是自己的麻烦，已替赵开明杀死严力一家，拒绝再被利用；赵开明则以杨间将来会为严力报仇、自己可用刑警身份查封酒吧和通缉他施压，并开枪杀死包间内一名女子。王岳考虑后答应联手，却在赵开明离开前见到包间灯灭、墙壁腐烂、街道被黑暗吞没，门外还出现黑长衫高瘦人影和敲门声。赵开明称这是A灾难级的鬼敲门事件，认为杨间可能操控其针对二人；王岳当场只能确认这不是自己的能力，二人如何脱困尚未写明。（第161章）

#### 第177-179章·求援、交手与死亡

王岳从鬼敲门袭击中暂时活下后，在鬼域内用麻绳吊起一名背包大妈，现身向杨间求带离。他承认严力一家三口由自己下手，却称赵开明逼迫、意在让杨间误会小强俱乐部；又表示赵开明已可能死亡，愿倒向杨间、协助其对付王小强和叶枫。杨间认为王岳杀人且曾同意谋害自己，拒绝交易，并以幸存者性命要挟童倩共同出手。王岳逃跑时被特制手枪击中，露出颈部打死结的老旧麻绳，因而被称为“鬼绳王岳”。（第177章、第178章）

王岳以鬼绳无征兆勒住杨间颈部，自己脖颈绳结也随之收紧。他见杨间以鬼眼、无头鬼影抵抗后，仍继续催动厉鬼，试图在复苏时让真正鬼物与杨间同归于尽。鬼绳最终先将王岳吊至半空，童倩以鬼脸怪笑使其断气；其尸体保留与鬼脸相同的僵硬笑容。王岳死亡后，鬼绳脱离其控制并在鬼域中大规模复苏。（第178章、第179章）
