# 张浩

## 简介

张浩是第301章与同伴乘坐小春市公交车的背包青年，自称二十二岁、练有“童子功”，并在车内关于世上是否有鬼的争论中支持有鬼一方。起初他把车辆改道当作普通路线问题，想向司机询问；年长婆婆劝他等待下一站后，他接受劝阻返回座位。司机尸体被发现、车辆失控后，他仍一度不信杨间的判断，直至公交停入荒野站点才同郑耀文及三名女生下车。第311章，他被干尸新娘牵手后死亡，尸体仍受其操控。（第301章、第302章、第304章、第311章）

## 经历

#### 第304-305章·返回异常公交

张浩在荒野中目睹坟前哭声令多名下车乘客跪地死亡，又听从杨间“回车上”的提醒，与两名未具名女同伴先后逃回车内。郑耀文和冯晓露在返回途中受哭声影响跪向土坟，未能同行返回；张浩作为三名回车者之一存活至公交再次发车。他此后是否离开车辆、能否获救均不详。（第304章、第305章）

#### 第307-310章·熄火脱车后被带离

公交熄火、车厢灯光熄灭时，张浩与两名女同伴仍在车上；杨间以鬼域转移离车时把三人一并带到路旁。哭坟鬼哭声持续增强后，张浩在第二层鬼域内表情僵硬地倒地，但杨间确认他仍有生命气息；同行两名女生则已死亡。第310章末，张浩不受控制地站起，以僵硬步伐走出杨间鬼域，朝道路对面出现的干尸新娘靠近。原文未揭示他是否受该鬼操控、是否死亡或此后去向。（第307章、第309章、第310章）

#### 第311-314章·被牵手后成为招鬼媒介

干尸新娘牵住张浩时，他立即失去生命气息，却没有倒下，而是继续牵手、以悲伤哭容挤出微笑并向树林招手。杨间确认他招来的是哭坟鬼，公交内的鬼也因这一动作开始拍打车身；杨间以许峰身体压住干尸新娘后，张浩一度倒地，招鬼动作停止。目标很快摆脱压制，再次牵住张浩，尸体又恢复招手。公交重启后，干尸新娘仍牵着张浩尸体留在路边，未再上车。尸体是否仍保有意识、会否继续招鬼及最终去向均不详。（第311章至第314章）
