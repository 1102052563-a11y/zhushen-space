# 程雄

## 简介

程雄是小春市新城区警局一名四十出头、入职超过二十年的老警员。第299章，杨间到警局后见到本人，确认其相貌普通、略显老态，办案经验丰富。原文没有说明他的完整警衔、家庭情况、常驻岗位或是否参与灵异案件。（第299章）

## 能力

- **字痕恢复**：程雄能辨认纸面压痕所用的繁体字和笔画深浅，计划用电脑扫描、软件分析，再由人工排序来复原内容。他认为旧记事本的书写者下笔很重，纸上磨损尚可，可恢复大部分文字。（第299章）

## 经历

#### 第299章·接受紧急委托

杨间以大昌市刑警身份到小春市新城区警局，说明由章华介绍，要求程雄尽快处理一本老旧记事本的残留字迹。程雄已接到上级通知、处于待命状态，得知并非出任务而是需要专业协助后立即答应。杨间强调不愿拖延，因为内容可能关系到鬼橱秘密和赵磊去向。（第299章）

#### 第299章·评估与保密处理

程雄检查记事本后指出第一页为繁体字笔画，书写力量大，纸面磨损尚可。他说明将以扫描、软件分析和人工排序进行恢复，而非直接读出完整文字。杨间要求整个工作和内容保密，并留下卫星定位手机号码；程雄承诺三小时内完成。杨间随即离开，至第300章没有写出任何恢复结果或意外。（第299章）
