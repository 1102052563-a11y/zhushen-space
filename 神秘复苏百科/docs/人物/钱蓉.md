# 钱蓉

## 简介

钱蓉是鬼邮局一楼13号房的年轻女性信使，约二十四五岁，浓妆掩不住憔悴和恐惧。她与万兴同批进入邮局、原本彼此依靠，但同批信使已大量死亡；她目前只成功送过一封信。姓名以万兴在杨间控制后称呼为准，其进入邮局前身份和首封信经历均不详。（原文回跳第740章、原文回跳第743章）

## 经历

#### 原文回跳第739-740章·从13号房出现并接受控制

万兴被杨间和孙瑞控制后，钱蓉从11号房内走出，听从万兴提醒站到一旁。她目睹杨间无视手枪、孙瑞以手杖威胁万兴，未试图反抗。万兴称钱蓉与自己同批进入邮局，一楼原有七人但多数死于此前送信任务。（原文回跳第739章、原文回跳第740章）

#### 原文回跳第743-744章·卷入红色集体信任务

次日她确认自己未收到个人信件，随后柜台出现红色信封。王善判断这是一楼集体任务，钱蓉因只送过一封、又无法对抗杨间等人，只能随众人等待决定。杨间取信、决定前往大海市福寿园78号送信后，她与万兴、王善一同走向邮局大门；但杨间一行回到大汉市烂尾楼时，钱蓉没有出现。她是否回到进入邮局前的地点、会否自行前往福寿园均不详。（原文回跳第743章、原文回跳第744章）

#### 原文回跳第781-784章·二楼受辱后死于房门袭击

红信投递成功后，钱蓉随一楼信使进入二楼，却因房间不足尝试投靠21号房，被陈星等人以欺骗和暴力赶出。杨间为补偿一楼信使被自己提前上楼的异常、也为取得房间，清除21号房原有人员并让她暂入房内。（原文回跳第781章）

杨间撕毁二楼红信后，七间房内出现未知厉鬼。钱蓉没有从21号房卧室走出，杨间、李阳检查时发现她已被吊在床头、指甲和血迹留在房门附近，确认死亡。她的强烈求生欲未能改变普通信使面对灵异袭击的结果。（原文回跳第784章）
