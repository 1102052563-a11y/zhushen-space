# 高明

## 简介

高明是大京市的一名国际刑警。鬼差事件紧急会议扩大后，他作为赶到总部的顶尖驭鬼者之一在场；副部长曹延华决定让所有已驾驭两只鬼的与会者集体护送王小明前往实验室，高明随队由李军的鬼域带离会议室。原文没有交代其外貌、所驾驭厉鬼、具体职务、过往履历或是否已驾驭两只鬼。（第459章）

## 经历

#### 第459章·加入实验室护送队

王小明认为实验室内仍开启的鬼棺可能与鬼差存在联系，提出亲赴研究；童倩、冯全、李军先后表态同行，曹延华随后下令让在场所有已驾驭两只鬼的驭鬼者集体行动，既护送王小明，也带走鬼棺和其他重要实验物资。原文在列举赶到会议的人员时点出高明是大京市另一名国际刑警，说明他属于此次临时集结的力量之一。李军展开青绿色鬼域，将除曹延华外的与会者一并转移向实验室；高明是否实际抵达、在队中的职责、能力表现及随后生死，均未写明。（第459章）
