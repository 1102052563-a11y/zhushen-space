# 赵磊

## 简介

赵磊是从大昌市第七高中鬼域中逃出的学生之一。第16章，他向杨间说明自己此前曾拨打求救电话，电话可以打通却没有声音，并且会在无声状态下显示接通。第17章，他的手机也收到以“138”开头号码显示的来电；第19章他被周围鬼物和死亡场景吓至瘫坐，但最终随杨间的红光鬼域离开学校。（第16章至第20章）

## 出处

- 在校门附近，杨间询问是否有人尝试电话求救。赵磊称电话打得通，却没有任何声音而且被接通，因此不敢再拨打；这使杨间开始调查论坛发帖者雷电法王留下的号码。（第16章）
- 钱万豪接听显示王老师号码的电话并传出敲门声后，赵磊先制止其接听，又因恐惧抓住钱万豪衣襟责骂。他和王珊珊、苗小善一样收到“138”号码的来电，但没有接听。（第17章）
- 黑衫老人和多个鬼奴包围众人时，赵磊已失去逃跑力气，瘫坐在地。杨间吞下鬼眼、张开红光鬼域后，赵磊与张伟、王珊珊等人一同消失；第20章他在步行街确认已离校，却仍难以相信瞬间从学校来到此处。（第19章、第20章）
