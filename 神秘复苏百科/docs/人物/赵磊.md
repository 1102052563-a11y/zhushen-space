# 赵磊

## 简介

赵磊是从大昌市第七高中鬼域中逃出的学生之一。第16章，他向杨间说明自己此前曾拨打求救电话，电话可以打通却没有声音，并且会在无声状态下显示接通。第17章，他的手机也收到以“138”开头号码显示的来电；第19章他被周围鬼物和死亡场景吓至瘫坐，但最终随杨间的红光鬼域离开学校。（第16章至第20章）

## 出处

- 在校门附近，杨间询问是否有人尝试电话求救。赵磊称电话打得通，却没有任何声音而且被接通，因此不敢再拨打；这使杨间开始调查论坛发帖者雷电法王留下的号码。（第16章）
- 钱万豪接听显示王老师号码的电话并传出敲门声后，赵磊先制止其接听，又因恐惧抓住钱万豪衣襟责骂。他和王珊珊、苗小善一样收到“138”号码的来电，但没有接听。（第17章）
- 黑衫老人和多个鬼奴包围众人时，赵磊已失去逃跑力气，瘫坐在地。杨间吞下鬼眼、张开红光鬼域后，赵磊与张伟、王珊珊等人一同消失；第20章他在步行街确认已离校，却仍难以相信瞬间从学校来到此处。（第19章、第20章）
- 第136章，他建议寻找落单的上官云，也在杨间连开两枪后率先叫破假上官云是鬼，伪装随即消失。第138至139章，他因恐惧不愿单独去取纸笔；杨间允许普通人撤离后，他先行离开洗手间。原文未说明他之后是否离开KTV。（第136章、第138章、第139章）

#### 第291-293章·遭报纸鬼篡改记忆后的求救

第291章，赵磊拦住杨间，先以家人皆死为由质问杨间为何只救王珊珊、张伟等人，又称自己已成为驭鬼者，并说在杨间旧居见过一个与杨间相同、坐着看报纸的人。杨间因其情绪、神态与记忆异常，怀疑他已被住处厉鬼影响；赵磊随即留下印有本人脸孔的红纸后消失。第293章，杨间回到旧居，从赵磊留下的碎纸笔记得知：赵磊原本为向杨间求助而进入未锁的房屋，遭袭后不断遗忘自身记忆并混入徐帆、张霞等人的记忆。他曾试图用笔记维持清醒，又在最后清醒时植入“杨间害死全家”的虚假仇恨，想诱使吸收记忆的厉鬼找到杨间并遭其解决；但厉鬼也吸收了这份计划，故只作试探便离去。赵磊本人是否仍保有清醒意识、其身体和人格还能否分离，均不详。（第291章、第293章）

#### 第294章·家属生还信息与通缉持续

刘小雨核查后告知杨间，赵磊父亲在外地出差、母亲留在家中，二人均幸存，另有数名亲属失踪；这与赵磊所信“全家皆死”不符，进一步印证其记忆已被篡改。杨间没有撤销全国通缉，认为被厉鬼操纵的赵磊即使看见父母也未必能识认。赵磊此时的实际行踪、外貌及是否仍在大昌市均不详。（第294章）
