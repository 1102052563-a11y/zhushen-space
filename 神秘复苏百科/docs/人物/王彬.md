# 王彬

## 简介

王彬是王珊珊的父亲，与妻子王海燕同住在大昌市一处高档小区。女儿从第七高中灵异事件返家后，他发现她不吃饭、怕黑、反复洗澡，且学校已被封锁、许多学生失踪；他没有立刻接受闹鬼的说法，却将异常与学校事件联系起来。（第29章）

## 经历

#### 第29章·察觉女儿异状

王珊珊在浴室停留两个多小时并不断刷洗颈侧时，王彬与王海燕在客厅担忧等候。他说明自己去过学校，见其已封锁，又从其他家长处得知不少学生夜自习后失踪。他认为女儿必然经历过某事，但顾虑其精神状态而没有直接追问。浴室尖叫后，他撞开反锁的门，看见女儿满身血迹、恐惧窗外的东西；待妻女离开，他又发现雾气窗户、墙壁和吊顶上都有小孩手印，因而意识到事情不对。（第29章）

#### 第30章·向杨间求助

王彬通过王珊珊手机联系杨间，希望见面询问学校经过。杨间直言学校闹鬼，并从“有东西跟着王珊珊”的描述猜到可能是婴儿；王彬想到浴室手印，却仍要求对方认真说明。杨间提出可处理但须付费，先后拒绝两万、五万，报出五十万元。王彬认为价码荒唐而发火，双方未成交；杨间提醒拖延可能导致王珊珊或自己死亡，并留下卫星定位手机号码。（第30章）

## 出处

- 王彬确认学校封锁和学生失踪，亲见王珊珊浴室的异常手印。（第29章）
- 他首次向杨间询问并就处理王珊珊危机的报酬议价。（第30章）
- 第34章，王彬亲见青黑无衣的小孩进入女儿房间、房内小孩脚印遍布墙顶，又从猫眼看见鬼婴顶住大门，使全家无法离开。他向杨间承诺价钱可谈并求救，随后用门栓和保险拖延，却听见本不可能打开的大门被开启，只能带妻女退入主卧等待。（第34章）
- 第36至39章，王彬见杨间以鬼域进入卧室、击退鬼婴并带全家离开。他为救女儿将报酬提至一百万元并立即转账；得知替换印记只能延缓死亡后，仍决定让杨间施行。颈印消失后，他看到女儿手腕留下红色鬼眼烙印，暂时接受后果。（第36章至第39章）
