# 王老师

## 简介

王老师是杨间班级的教师。第2章晚自习时，他宣布校方临时通知，将原本晚自习改为学生安全防范知识宣传课，并请周正上台讲课。他在课堂上负责维持纪律，先后批评杨间突然站起、方镜突然站起等行为。第3章敲门鬼已在教室外形成黑暗鬼域，王老师仍因好奇开后门外出查看；周正出言制止时已晚，王老师在跨出门的瞬间失踪。（第2章、第3章）

## 出处

- 王老师引入周正，要求学生课堂安静，并把周正的讲座称作安全宣传课。（第2章）
- 鬼域出现后，他未理解危险，听到学生称有人敲门，便打开教室后门。门外黑暗浓重，他刚走出去就不见踪影，之后没有声音传回；原文第5章前未说明其下落。（第3章）
- 此前杨间因看见黑衫老人而惊恐站起时，王老师只将其视为扰乱课堂纪律，要求其坐下；方镜因鬼域异象站起时，他也先以课堂秩序质问。鬼域刚出现时，他没有掌握周正和方镜所知的灵异信息。（第3章）
- 他打开的是教室后门，而非正在被老人敲击的教室门。周正看到其行动后马上喊“别出去”，但王老师已跨出门外；其失踪使学生看到门外黑暗能直接吞没进入者，教室恐慌随之加剧。（第3章）
