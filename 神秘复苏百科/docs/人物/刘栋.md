# 刘栋

## 简介

刘栋是第591章平安大厦顶层会议室中的朋友圈驭鬼者。原文只在杨间五层鬼域袭击后的残余现场点出其姓名，未交代他加入朋友圈的时间、组织职务、外貌、性格、此前经历或与其他成员的具体关系。（第591章）

他驾驭的厉鬼以一颗干瘪、腐烂的人头形式遗留。杨间的五层鬼域可令一般人、建筑乃至部分厉鬼暂时消失，但刘栋身体被红光送走后，头颅没有消失且继续眨眼、转动眼球并迅速腐烂；杨间据此判断该头颅是无法被送走、正在复苏的鬼。刘栋生前如何驾驭这只鬼、头颅是否原本就与身体分离及其完整杀人规律均不详。（第591章）

## 经历

#### 第591章·身体被送走并遗留复苏人头

杨间以五层鬼域的红光攻击会议室，刘栋未能抵挡，身体在红光中消失。现场没有留下尸体或伤口，只有一颗干瘪凹陷、死灰眼睛的死人头；其眼皮随后眨动、眼球转动，脸部皮肉开始腐烂。姜尚白、吴云和方世明将该头认作刘栋遗留物，吴云只能用普通衣物临时包裹，准备带往安全屋处理。刘栋具体如何死亡、身体与厉鬼分别被送往何处及人头能否被安全收容均不详。（第591章、第592章）

#### 第593章·遗留人头逼退鬼手

吴云遭鬼手侵入眼眶、脚踝和全身时，将包裹中的腐烂人头举起与自己相对。人头的眼睛立即聚焦，使侵入吴云的鬼手快速退去，显示其恐怖程度足以压过杨间此时放出的鬼手；但吴云短暂对视后脸部皮肤开始腐烂，故立即重新以衣物遮盖。该表现只能确认人头在这次接触中可驱退鬼手，不能确认其是否受刘栋遗志控制、能否辨别目标或长期使用。（第593章）

#### 第594章·被杨间取走

吴云被鬼手杀死后，包裹人头滚落在地。姜尚白想抢走它作为最后的自保手段，杨间却先以鬼域显出手掌将人头提走，并一同带走吴云尸体。人头之后的收容地点、是否继续复苏及刘栋身体会否重新出现均未写明。（第594章）
