# 林落梅

## 简介

林落梅是与罗素一同行、约二十岁、穿连衣裙的年轻女子。她外表可爱，却能借扩音器发出异常声音；该声音不以普通声波扩散，而会像幽灵般自行搜寻被呼叫的目标。她与罗素一及另一名男子到观江小区寻找杨间，三人的关系和所属组织均不详。（第279章）

## 能力

- **诡异喊话**：林落梅闭口时，扩音器仍会发出“杨间”“鬼眼刑警杨间”的呼叫。每一次声音会自行向前搜寻；她喊六次后停止。声音可绕过小区路径、穿过保安亭并进入别墅，在一楼、二楼逐层徘徊，而不在该层的旁人听不见。此处仅记录其寻找效果，声音的杀人条件、覆盖范围和能否被抵抗均不详。（第279章、第280章）

## 经历

#### 第279-280章·受命搜寻杨间

罗素一一行夜入观江小区后，罗素一请林落梅“喊个话”。林落梅没有拒绝，取出扩音器呼叫杨间；罗素一抱怨小区很大、六次呼喊不足以寻遍，林落梅则反问他是否愿意自己喊。同行男子担心这种能力可能误杀旁人，罗素一也只以“应该没有这么容易死”回应。随后，声音越过大门钻入房屋，开始在楼层间以不合常理的方式搜索，搜寻尚未结束。（第279章、第280章）

## 出处

- 以扩音器放出会自行寻人的声音，夜间在观江小区寻找杨间。（第279章、第280章）
