# 李洋

## 经历

#### 第484章·制止刘紫前往平安酒店

刘紫因在手机上查到平安酒店，提出当晚入住。李洋听见酒店名称后立刻急刹车，并称该处不向外人开放，要求她换一家酒店；当刘紫说苗小善的朋友正住在里面时，他更加紧张。他没有说明酒店的实际职能，只警告刘紫，大京市有资格入住平安酒店的人不应轻易接触，即使这类人在街头杀人也可能无人追究。此说法是李洋基于所知内幕作出的警告，不等同于酒店规则已被正式公开。（第484章）

#### 第484章·确认杨间身份后的不安

刘紫说出“杨间”之名并说明苗小善曾在大昌市读书后，李洋将其与大昌市事件联系起来，认定刘紫遇到的很可能是代号“鬼眼”的杨间。他因此出冷汗、感到忐忑，担心刘紫此前已在不知情下得罪一名危险且特殊的人。李洋只劝她今后远离杨间、遇见时客气，并让她不要再追问平安酒店；他没有联系杨间、总部或苗小善，之后是否继续担忧或采取行动不详。（第484章）
