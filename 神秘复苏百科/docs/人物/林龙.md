# 林龙

## 简介

林龙是参与大昌市饿死鬼事件总结会的国际刑警，驾驭了两只鬼。他在杨间完成报告后，主动以十亿元和一年黄金配额收购杨间可能获得的红色鬼烛，又提出协助处理大昌市灵异事件作为附加条件。杨间拒绝后，他没有收手，反而握住杨间肩膀施力，以“做不成朋友”可能成为敌人相威胁，显露出以力量强买资源的态度。（第286章）

## 经历

#### 第286-288章·强购鬼烛后被关押交易

体育馆会议室内，林龙把杨间报告时的谦逊误作软弱，要求取得鬼烛未果后直接威胁。杨间先令林龙搭在肩上的整条手臂脱落，再让无头鬼影侵入其身体，使体内厉鬼沉寂、身体无法行动。杨间以两层鬼域压制前来救人的李军鬼域，操控林龙走入黄金箱，并把箱子送至地下约一万米处。（第286章、第287章、第288章）

赵建国为救人提出补偿，王小明则承诺两个月内提供一根鬼烛，杨间才同意将林龙取出；同时要求林龙此后远离大昌市，赵建国为此作担保。林龙在黑暗密闭环境中被困约半小时，随后连同黄金箱被抬上运输机离开会议室，具体送往地点、两个月后是否获释及其驾驭厉鬼能力均不详。（第288章、第289章）

## 出处

- 强购鬼烛并威胁杨间。（第286章）
- 被无头鬼影和鬼域压制，成为鬼烛交易的筹码。（第287章、第288章）
