# 林小夕

## 简介

林小夕是梁圆的同住好友，性格怯生、内向。她白天随梁圆在县城与杨间相识，夜间则穿雨衣、拿着水果刀进入同一灵异梦境。她对梁圆的奇怪购物和夜间活动似乎有所适应，却会因无头尸体、白衣头罩人等异常害怕。其家庭、年龄、为何与梁圆同住及是否长期遭受梦境影响均不详。（原文回跳第679章、原文回跳第682章）

## 经历

#### 原文回跳第679章·县城相识

梁圆因骑车撞人被青年追赶时，林小夕先因跑不动而停下，随后和梁圆被围住。杨间替梁圆赔钱后邀请二人吃饭，她礼貌但拘谨地答应。回梅山村途中，梁圆称林小夕现在与自己住在一起、是最好的朋友；杨间暂未确认这一说法背后的情况。（原文回跳第679章、原文回跳第680章）

#### 原文回跳第682-683章·持刀进入夜间梦境

夜间，林小夕随梁圆到杨间老宅外叫人，穿雨衣并持白天购得的水果刀。她自报姓名，听梁圆说要在村外玩捉迷藏；遇白衣黑头罩尸体时，她紧张询问是否绕路，梁圆上前连刺后她继续同行。荒野无头尸体活动时，她先感到恐惧，后按杨间建议与众人转身，发现异动停止。她在梦境中能否独立自保、是否了解规则不详。（原文回跳第682章、原文回跳第683章）
