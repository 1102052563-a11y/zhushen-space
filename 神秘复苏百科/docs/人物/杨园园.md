# 杨园园

## 简介

杨园园是四十年前出生的女子，旧档案记录其为杨孝的表妹，后因溺水死亡。她十七八岁时的黑白照片与现今梁圆外貌完全相同，只是照片中留长辫、梁圆为短发。胡凯无法在现行档案中找到梁圆，杨间因此怀疑梁圆可能与已死亡的杨园园存在灵异关联；二者是否同一人、替身或记忆复现，原文未证实。（原文回跳第713章）

## 档案线索

胡凯按杨间要求翻查梅山村旧档，发现杨园园四十年前出生、身份关系久远且多已遗失。档案同时记载其死因为溺水。由于梁圆在现代村民中没有正常出生、入学等档案记录，杨园园旧照成为目前唯一能解释其外貌和杨孝亲缘关系的线索。（原文回跳第713章）

杨间知道灵异手段可能将人的记忆、面孔或身体以非正常方式延续，因而不接受“死人不能复活”的普通结论；但他没有证据证明杨园园被复活。杨园园死亡时间、溺水地点、当年是否接触灵异及她与梁圆的具体关系均不详。（原文回跳第713章）
