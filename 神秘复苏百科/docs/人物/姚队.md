# 姚队

## 简介

姚队是大学鬼画事件现场的一名中年总部支援人员，身穿西装，体重近两百斤。其完整姓名、具体职务、所属单位和是否具有驭鬼者身份均未交代。鬼画暂被杨间限制后，他负责或参与留守篮球馆内学生的安置，却将所有曾与鬼画有接触的学生集中在同一处；这一安排表面理由是便于统一管理，实际包含借苗小善迫使杨间在场兜底的盘算。（第562章）

## 经历

#### 第562章·以集中安置试图绑定杨间

杨间返回篮球馆，发现馆内多出许多学生，留守工作人员解释这些学生都不同程度接触过鬼画，集中看管是为防止其四散、扩大风险。杨间认为未知灵异规律下把接触者聚在一起，反而可能引发集体死亡，并要求提出该方案的人到场。姚队随即由工作人员带来，先以恭敬语气询问吩咐；杨间没有追问姓名，当面脱下手套，以鬼手扇在其脸上。姚队被打飞、牙齿脱落并流血，杨间说明自己因总部才没有杀他，警告其不要耍小手段。（第562章）

姚队事后拒绝让工作人员向总部报告，要求先送医。他明白杨间看穿了自己的考虑：篮球馆内有与杨间关系亲近的苗小善，若接触鬼画的学生在此出事，杨间很可能不得不处理；若不出事，自己亦可借集中安置宣称没有失职。他此前预期总部约束会使杨间不敢严厉报复，却因鬼手造成身体逐渐失调而转为恐惧。原文只写他感觉四肢协调性不断恶化，未确认伤势是否危及性命、总部是否得知、是否受到处分或其后还参与现场处置。（第562章）
