# 王善

## 简介

王善是鬼邮局一楼17号房的年轻信使，约十八九岁，成为信使前仍在读高中。他已成功送出两封信，即将迎来第三次任务；在同批信使大多死亡的情况下仍能保持冷静，主动与杨间等人谈判。其家庭、如何收到首封信、前两次送信内容和实际自保手段均不详。（原文回跳第740章、原文回跳第741章）

## 邮局认知

王善称一楼信使每送三封信可进入二楼，之后每三封再上升一层；越高层送信间隔越长、休整期越久，但这一规则主要来自楼上传下的经验，未获他亲身验证。他还说明晚六点至早六点邮局熄灯，留在房外会遭游荡鬼袭击，房间内相对安全；一间房多人过夜可能触发异常，同样只是他听来的规则。（原文回跳第740章、原文回跳第741章）

他见到柜台出现红色信件后，说明床头信属于个人任务、门上信属于房间成员任务、柜台信疑为全层任务，且危险程度可能依次提高。红信的性质超出其经验，他没有提供失败代价或奖励的确证。（原文回跳第743章）

## 经历

#### 原文回跳第740章·提出熄灯规则与第五层传闻

杨间等人控制万兴时，王善从17号房现身，自称比万兴早入邮局，愿提供不按常规慢慢送信、快速抵达五楼的方法。他先提醒邮局将于晚六点熄灯，黑暗中有鬼游荡，劝众人回房避险。杨间强行要求与他共住17号房后，他才继续交代五楼传闻：据楼上信使说，第五层存在通向外界的楼梯，离开者可摆脱信使身份；该传闻未验证。（原文回跳第740章、原文回跳第741章）

#### 原文回跳第741-743章·提出第三封信越级假设并接受合作

王善推测第三封信是关键性的“上楼信”：若抢走即将完成第三次任务者的信并成功投递，可能由投递者替代原信使上楼，而原信使送信次数归零。他没有成功案例，只愿以自己即将出现的第三封信和万兴的第三封信供杨间、孙瑞、李阳尝试，交换条件是三人若能离开邮局便带他同行。杨间接受这一约定，但未承诺已验证上楼方法。（原文回跳第741章）

夜游鬼在17号房外连续敲门、试图关灯和开门时，王善留在房内观察三名驭鬼者对抗，未直接出手。次日红色信件被杨间取走后，他决定跟随送信队伍，担心不参与会被视为未完成一楼集体任务；离开邮局时他未与杨间等人同时出现在烂尾楼，去向暂不明。（原文回跳第741章至原文回跳第744章）

#### 原文回跳第762章·在福寿园封锁线外等待

王善自行赶到大海市福寿园送红信，却因灵异论坛多层封锁无法进入。他与万兴会合后，认为杨间一行进入封锁区、并能逼退邮局夜游鬼，或许有机会处理鬼邮局并让信使摆脱诅咒；他选择在外围等待，不尝试硬闯。（原文回跳第762章）

他没有确认红信是否已送达、福寿园内鬼域情况或钱蓉去向。其所称杨间等人“可能比高层信使更可怕”只是个人判断。（原文回跳第762章）
