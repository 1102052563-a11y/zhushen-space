# 钱枫

## 简介

钱枫是江艳大学同学。第120章，他在江艳住处门口送玫瑰表白，又租下她对门公寓，称要与其所谓男友公平竞争；江艳认为他是看中自己的车和经济条件。（第120章）

## 经历

江艳开杨间留下的奔驰参加同学聚会后，钱枫开始持续追求。她多次称已有男友、要求他远离，钱枫仍在红灯处驾车追上并高声表白。江艳为拒绝他谎称自己怀有男友孩子，钱枫仍称可抚养；出租车司机看不下去，驾车撞其车辆并责骂，江艳趁机离开。其后是否继续纠缠，至本批末不详。（第120章）
