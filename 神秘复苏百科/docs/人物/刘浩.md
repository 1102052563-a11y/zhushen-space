# 刘浩

## 简介

刘浩是刘元之子、刘老太孙子，约十五六岁，随家人参加祖母在福寿园的葬礼。灵异论坛人员与刘家冲突时，他和姐姐刘欣悦试图阻止棺材被抬走；后在福寿园鬼域中因恐惧难以行走。（原文回跳第750章、原文回跳第753章）

## 经历

#### 原文回跳第750-754章·跟随驭鬼者队伍求生

刘老太尸体消失、福寿园化为诡异坟场后，父亲刘元请求杨间等人带刘浩和刘欣悦同行。罗素一同意二人跟随，但没有承诺保护；刘浩在坟场中恐惧发抖、称快走不动，只能依靠姐姐带路。（原文回跳第751章、原文回跳第753章）

杨间决定挖掘刻有罗素一姓名的老坟时，刘浩被命令与姐姐共同挖土。他未展示灵异能力，也无法确认能否活过坟场后续袭击。（原文回跳第754章）

#### 原文回跳第761章·被刻名后失去双腿而存活

刘浩被刻碑鬼盯上后陷入坟土。杨间根据坟土必须接触身体才能吞没目标的规律，以鬼影卸下他的双腿、将其上半身拉离地面；双腿被留在坟土中，他本人因此保住性命。姐姐刘欣悦按杨间要求扶其坐上墓碑，避免再次接触地面。（原文回跳第761章）

其后坟场老坟继续塌陷、刘老太尸体异变，刘浩是否能离开、失去双腿后的长期状态均不详。（原文回跳第763章、原文回跳第764章）
