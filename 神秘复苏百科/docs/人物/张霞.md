# 张霞

## 简介

张霞是苗小善的大学室友，也是寝室油画异常最早可确认关联的失踪者。她当天早晨去过学校美术馆，带回一幅带历史感的老旧油画后没有离开学校，电话也留在寝室。苗小善因此怀疑她并非普通外出失联，并向杨间求助。（第549章、第550章）

## 出处

- 油画最初放在张霞床铺靠墙的位置。苗小善接到杨间“油画有问题”的提醒后，看见画仍在原处；杨间进入寝室时，鬼眼反应表明画与此前油画异常一致，但他没有直视画面。（第550章）
- 张霞没有在寝室内出现。孙于佳在第551章发现油画已自行移到王悦床头，说明原放置地点已改变；这不能证明张霞的位置或处境。（第551章）
- 张霞是否已被油画吞没、是否仍存活、美术馆取得油画的具体过程及学校中是否还有其他接触者，原文均未写明。（第549章至第554章）
