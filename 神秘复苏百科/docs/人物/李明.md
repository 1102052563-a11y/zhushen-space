# 李明

## 简介

李明是杨间班级的一名男学生。第4章，敲门鬼仍在教室外以固定频率敲门，已有数名学生趴在课桌上不动、散出尸臭。李明此前仍站在人群中，随后突然浑身一颤、身体僵直地栽倒在地，成为现场新增死亡者。（第4章）

## 出处

- 同学试图扶起李明时，接触到的是冰冷、僵直、如尸体般的身体；他睁大眼睛和嘴巴，脸色死灰，并逐渐散发腐烂臭味。（第4章）
- 原文将他与数名同样死灰、趴在桌上不动的学生并列，说明教室内死亡正在持续发生；第5章前没有交代其亲属、经历或其他身份信息。（第4章）
- 李明倒下前，周围学生已看见若干同学趴在课桌上一动不动。方镜指出门外有鬼后，敲门声再次响起，李明随即浑身一颤、身体僵直栽倒；原文没有写明是否有人触碰敲门声、是否存在其他触发条件。（第4章）
- 他的死亡让班内学生从害怕环境异变转为直接确认同学正在死亡。有人询问“李明你怎么了”，有人伸手去扶后尖叫退开；这一过程发生在周正尚未依杨间推测出手之前。（第4章）
