# 罗素一

## 简介

罗素一是饿死鬼事件结束后前往观江小区的年轻男子，与一名约三十岁的男子及林落梅同行。他身材修长、相貌俊朗，同行者默认由其主事；三人通过内部可查的国际刑警资料确认杨间住在小区，目的为“拜访”，具体意图未明。（第278章、第279章）

## 经历

#### 第278-279章·夜访观江小区

赵建国离开杨间住处后，罗素一与同伴已在观江小区河对岸观察。他们掌握杨间“鬼眼刑警”的身份和住址，认为亚洲分部虽试图掩盖饿死鬼事件的细节，国际刑警的基础资料仍可在内部查到。罗素一没有立即入内，称应当拜访杨间但不急于白天行动；他也知晓大昌市至少聚集二十余名顶尖驭鬼者，李军亦在城中，因而表面上表示要顾及这些人的面子。

当夜，罗素一带两名同伴进入小区。他以“做鬼就要有做鬼的风格”为由选择夜间低调行动，并让林落梅使用扩音器呼叫杨间；他本人自称唱歌不错却不适合喊话，担心误把别人喊死。林落梅的声音随后以无规律方式自行寻找目标，保安等旁人却听不见。罗素一的驾驭厉鬼、来访目的和他是否已与杨间接触，均不详。（第278章至第280章）

## 出处

- 与两名同伴在观江小区外确认杨间住址，并于夜间入内寻找杨间。（第278章、第279章）
- 指使林落梅以诡异喊话搜寻杨间。（第279章、第280章）
