# 卫云

## 简介

卫云是灵异论坛成员中的驭鬼者，身材高瘦、瘦至近似干尸，常以风衣遮身。他在叶真被杨间以疑似棺材钉压制后，按论坛经理命令试图拖住杨间，却被鬼童扑中并留下尸水侵蚀痕迹。（原文回跳第770章）

## 经历

#### 原文回跳第770-772章·试图拖延杨间后被鬼童侵蚀

灵异论坛准备抢回被钉住的叶真时，经理命卫云出手拖延杨间。鬼童随即抱住他的腿向上攀爬，尸水留下黑色湿痕并令身体开始腐烂；卫云的反击对鬼童未能生效，短暂纠缠后由林落梅鬼音和总部调停中断冲突。（原文回跳第770章）

战后卫云称伤势暂未失控，但认为再被鬼童纠缠片刻就会被活活吃掉。他所驾驭厉鬼、试图使用的能力及尸水侵蚀能否彻底恢复均不详。（原文回跳第772章）
