# 周林

## 简介

周林是李跃的信使同伴，接替被杨间杀死的同伴持有老式手枪。他没有驾驭厉鬼，依靠一发嵌入手指骨的特殊子弹尝试击退信件引来的鬼，却在树林中被鬼贴身入侵、死亡后又重新站起。其被鬼控制、被鬼取代还是异常复活均未证实；本批末他随李跃离开大昌市，后视镜中已显露与本来面容不符的泥土惨白半脸。（原文回跳第696章至原文回跳第698章）

## 经历

#### 原文回跳第694章·同伴死亡后持枪对抗杨间

杨间要求查看黄色信封并准备处置信使时，周林的持枪同伴先被鬼手扭断颈骨。周林接手遗下的老式手枪，手枪射出的不是普通子弹，而是一小节带灵异力量的手指骨；这一枪在杨间胸口造成短暂伤口，却被无头鬼影迅速拼补。周林此后不敢再正面挑战杨间。（原文回跳第694章）

#### 原文回跳第696-697章·遭信件厉鬼入侵

杨间撕开信件后，树林变暗、冷风和脚步出现。周林紧张中感觉后颈被冰冷手指触碰，仓促射出最后一发骨制子弹却打空。逃离时，杨间用鬼眼看到一具从坟土挖出的鬼趴在其背上；周林本人无法察觉，走出不远便倒地死亡。杨间靠近检查时，尸体半张脸已变成鬼的泥土惨白侧脸，并抬手试图触碰杨间，迫使杨间以鬼域退避。（原文回跳第696章、原文回跳第697章）

#### 原文回跳第698章·异常醒转并离开大昌市

杨间准备让无头鬼影压制时，树林异象反而消散，周林重新站起，自称像昏睡过、没有死亡期间记忆。杨间确认他先前尸体已冷，仍无法解释这种醒转；为避免直接触碰入侵型厉鬼打破自身平衡，杨间不再出手，故意让李跃一行把周林带离大昌市。李跃按杨间字条回望后，见周林半张脸陌生、惨白且带泥土，证明异常尚未结束。（原文回跳第698章）
