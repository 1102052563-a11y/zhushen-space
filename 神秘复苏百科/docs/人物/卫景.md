# 卫景

## 简介

卫景是最早一批驭鬼者之一，也曾是国际刑警。王小明在鬼差事件的紧急会议中展示其档案，说明卫景早年处理过多起灵异事件，在灵异事件爆发初期立过功劳，因此在场的陈义等人认得其名。原文未写明他的具体驻地、所驾驭厉鬼、曾处理的事件名称和亲属关系。（第444章）

## 经历

#### 第444章·早期驾驭失败后的幸存

王小明说明，早期第二只鬼的驾驭方案尚不成熟、成功率偏低，卫景没能成功驾驭第二只鬼。其后他没有立刻死于厉鬼复苏，而是经某些特殊抢救措施被压制下来，以半死不活的状态幸存。王小明未说明抢救者、压制手段、卫景当时体内的鬼和此后是否仍保有清醒意识，不能据此把他视为正常存活。（第444章）

#### 第444章·成为鬼棺实验对象

王小明为验证人能否转为真正的鬼而仍保持意识，选择卫景作为实验对象。实验失败后，培训基地出现的鬼被王小明暂判为卫景彻底厉鬼复苏后的结果；但他同时担忧卫景可能继承鬼棺能力，或已成为鬼棺的一部分。该判断尚未经过现场确认，卫景本人是否尚存意识、身体位置及是否可恢复都不详。（第444章）

## 与鬼差的关系

王小明在第445章进一步说明，当前目标虽按厉鬼本能行动，却疑似拥有卫景的一部分智慧：可能是鬼占据上风、控制了卫景，而非卫景成功控制鬼。因卫景原为经验丰富的国际刑警，若该判断属实，目标的威胁会高于单纯按本能行动的厉鬼；这仍只是王小明的推测，不能确认目标是否会使用卫景的完整记忆或策略。（第445章）
