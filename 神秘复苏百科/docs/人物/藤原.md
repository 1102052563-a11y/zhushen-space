# 藤原

## 简介

藤原是大阪宴会中的除灵社骨干成员。三岛为确认放映机中反复出现的死亡模式，安排藤原作为第三名使用者；影像预示他会在一栋房屋内无头死亡，门口则出现与山崎、金川死象相同的皮鞋和西裤。藤原当前没有遭遇实际袭击，身份、能力、外貌和与除灵社的具体关系均不详。（原文回跳第672章）

## 经历

#### 原文回跳第672章·第三次死亡影像

山崎、金川的影像连续显示同类斩首死象后，三岛没有中止放映，转而要求藤原继续操作，以判断事件是否偶然。藤原迟疑后走到手摇式放映机前摇动握把，画面显示他倒在一栋屋内，头颅滚在尸体旁，敞开的大门外有一双穿西裤、皮鞋的腿。影像再次出现同一未知目标，强化了该厉鬼会连续追杀除灵社成员的判断。（原文回跳第672章）

藤原使用结束后没有发表明确判断，也未展示驾驭厉鬼能力。影像没有给出房屋位置、死亡时间、该鬼如何进入室内或能否通过改变行程规避；藤原在本批末仍属死亡预示中的潜在受害者，实际状态为存活。（原文回跳第672章）
