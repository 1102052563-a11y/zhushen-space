# 郝少文

## 简介

郝少文是第70章负责抢夺黄金盒行动的冷漠青年。他带人以掺黄金的锁链和钉子囚禁严力，试图逼问盒子位置；得知吴峰等人失联后，准备亲自前往杨间处。（第70章）

## 出处

- 他以甩棍殴打严力，认为驭鬼者仍有人类弱点，不惧严力死后厉鬼复苏会由国际刑警善后。（第70章）
- 房中连续响起敲门声、门锁失灵时，他误以为遭遇灵异事件，准备放弃严力逃走；最后房门打开，杨间从中走出，后续冲突未写。（第70章）
- 第71至73章，他以黄金弩箭、黄金弹头攻击杨间，却在鬼域幻象下误杀手下。面对倒计时，他供出雇主来自严力接触的俱乐部、此行报酬五千万美元，并说明第二只鬼可能与原鬼互相制衡、延缓复苏。杨间随后将其及手下移出现场，去向不明。（第71章至第73章）
