# 马有财

## 简介

马有财是大昌市有名的企业家，也是资助小强俱乐部的股东之一。俱乐部会议中他曾随多数人举手支持清除杨间；战斗后则成为现场唯一被杨间暂留的股东。杨间以无头鬼影把他的头颅换到雷虎身体上，将其性命和身体状况作为获取情报的约束。马有财因畏惧杨间，没有向王小强透露自己供出王岳的事。（第145章、第149章、第150章）

## 经历

#### 第149章·以王岳线索换命

杨间清除王小强及有反抗能力的人员后，逐一盘问躲在虚假会议桌下的股东，马有财在其余股东因答不出严力家人遇害一事而被杀后，承认知道一条消息。他称王小明常去大昌市的玫瑰酒吧喝酒，酒吧老板王岳曾到过俱乐部，是可收钱办事的驭鬼者，可能知道严力之死的内情。杨间不能确认线索真伪，仍以鬼影取下他的头颅、同一具头部被击碎的尸体互换身体，警告若消息无用，新身体会杀死他。（第149章）

#### 第150章·目睹复生并受资助逼迫

杨间离开庄园后，马有财因颈部缝合伤口和目睹厉鬼而难以行动。他看到王小强放出的腐烂厉鬼从尸体中取出内脏、皮肉和头颅，拼成残缺的王小强，使其恢复意识。王小强误以为杨间因贪图资本才放过马有财，又据其换头结果判断杨间可随时再取其头；随后要求马有财在三日内筹集十亿元继续资助俱乐部。马有财没有明确应允，也没有供出王岳线索，仍受杨间和王小强两方威胁。（第150章）

## 关系

- [杨间](杨间.md)：马有财以王岳的线索换得暂时存活，头颅被其换到雷虎身体上。（第149章）
- [王小强](王小强.md)：同为俱乐部相关者；王小强复生后强令他筹资十亿元。（第150章）
- [王岳](王岳.md)：马有财向杨间供出其玫瑰酒吧老板及驭鬼者身份。（第149章）
