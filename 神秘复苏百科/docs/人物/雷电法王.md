# 雷电法王

## 简介

“雷电法王”是论坛网名。其本人在帖子中自称某省第三中医院坐诊医生，称自己夜间坐诊时接收一名从五楼坠下的老人；他依据尸体没有生命症状、尸体特征、胸口温度低于当夜常温至少十度等情况，判断老人已死不止一天。其真实姓名、身份及帖中内容的完整真实性，在本批原文中均无外部佐证。（第1章）

## 出处

- 他称该老人尸体在停尸房失踪，有病人说见到老人自行从停尸间方向走出；他后来亲眼拍到老人穿黑色长衫经过急诊室。（第1章）
- 他在家中发帖时称老人站在门外敲门，客厅灯熄灭后又听见脚步；帖中上传了敲门录音并留下电话号码，最后一行写“那老人进来了”。帖子到此中断，第5章前原文未交代其下落。（第1章）
- 他在帖子里曾称自己是无神论者，因此先把病人所说的尸体行走归为不可信；但亲眼看见老人经过急诊室后仍拿手机拍照，后来又因老人到家门外而向论坛求助。其发帖节奏在末段不足三十秒且出现错字，杨间据此判断发帖者已十分惊慌。（第1章）
- 本批只能确认“雷电法王”是其网名。帖子评论区有网友质疑其打广告或虚构故事，杨间开始也有此怀疑；不过第3章杨间在学校窗外看见外观相同的老人，使论坛帖中的老人形象得到现场对应，但不能据此确认发帖者个人的全部遭遇。（第1章、第3章）
