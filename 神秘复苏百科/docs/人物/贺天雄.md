# 贺天雄

## 简介

贺天雄是第504章出席平安大厦朋友圈临时会议的强壮矮个男子。会议围绕杨间杀死高志强、可能成为队长后的威胁展开；贺天雄将杨间视为凭解决一件S级事件而获得资本的外来者，主张在其成为队长前主动打掉。他的职务、驾驭厉鬼、与高志强的具体关系及是否属于董事会均不详。（第504章）

## 经历

#### 第504章·提出绑架亲属方案

贺天雄提议先绑架杨间母亲，引杨间现身后再联手杀死。费总当场反对，指出杨间才是威胁本身，伤害其亲属只会招来更严重报复；杨间拥有鬼域，可能在短时间内对城市人口和敌方亲属造成灾难。贺天雄仍建议联合总部人员并预先安置亲属，未能改变会议倾向。姜尚白又以杨间厉鬼复苏后可能成为S级事件为由，说明不能轻易掀桌。最终会议没有采纳贺天雄的方案，转而决定先派李瑶尝试招揽杨间。（第504章）

#### 第512章·赞成动手并再提截杀

费总提出表决是否在队长计划前打掉杨间时，贺天雄立即赞成，并称早该听取自己的主意。方世明同意行动后，贺天雄又提出把杨间引回大昌市、半路截杀的方案；方世明没有回应该提议，转而指定利用下一场灵异事件制造死亡假象。贺天雄是否参与后续计划或其截杀建议是否被保留，均不详。（第512章）

#### 第543-544章·被杨间列为下一目标

费总死亡后，李瑶指出若要扩大先手成果，应再处理一名有分量的顶尖驭鬼者，遂将姜尚白与贺天雄的资料交给杨间。杨间顾虑姜尚白已加入总部，转而选定贺天雄；李瑶称其案底很多、手不干净，局势不严峻时本应被处理。此处只是杨间确定的下一步目标，并非已经找到、交手或处置贺天雄；其两只鬼的具体信息和行踪均不详。（第543章、第544章）

#### 第546-549章·别墅遇袭与逃过处置

杨间到贺天雄市郊高档别墅时，贺天雄正在书房看兵法；他在屋外养狗、屋内布置暗门，并用黄金装饰墙壁和房间以阻碍鬼域。犬吠骤停、大门被开后，他没有贸然下楼，而是藏进暗门，通过播音器与杨间交谈。杨间以鬼域摇晃整屋，威胁拆毁其黄金防护，迫使他现身三楼；听到张建、费总已死后，贺天雄确认朋友圈的核心情报已泄露。（第546章）

杨间以鬼眼看见贺天雄肩上有一具穿旧式黑布裤的尸体，档案称其代号“鬼踩人”：目标只要作出抬头看鬼的动作，肩上鬼便会走下杀人。贺天雄不能替鬼选择目标，只能等待条件触发；杨间已知规则且始终不抬头，使其防备失效。贺天雄提出退出行动、赠送别墅换和解，杨间则以五分钟为限，称不能杀死才谈和解；贺天雄立即借黄金隔断和熟悉地形逃入安全屋。（第547章）

安全屋外，杨间的五层鬼域先被贺天雄预留的替死布娃娃转移，布娃娃随即残缺，贺天雄本人躲过一次必死攻击。他没有第二只布娃娃，继续逃进安全屋；杨间鬼眼连续使用受限后转以鬼影破门。贺天雄露出腐烂胸腔：其最早驾驭的是会逐渐取代自身血肉的死人肉，曾使他在坠楼和火化后恢复，也会让鬼避开腐烂部位，但代价是身体不断腐烂；肩上鬼能延缓腐烂，却不能由他指定攻击。（第548章、第549章）

鬼影渗入安全屋后，贺天雄双腿、腰部逐渐失去知觉，身体被分开拉扯，胸腔内腐肉暂时阻碍侵蚀。杨间接到苗小善关于油画与室友失踪的电话后撤走鬼影离开，贺天雄在五分钟届满后存活。他认定自己不是凭实力逃脱，而是因来电带来更紧急的灵异事件；之后的身体恢复、是否继续执行朋友圈计划和两只鬼的完整规律均不详。（第549章）

#### 第553章·逃离别墅并威胁李瑶

杨间离开后，贺天雄带着行李走出别墅准备暂离城市，因不敢赌杨间会否折返处理自己。他发现李瑶驾张建的车进入小区，立即认定她泄露住址，砸窗将其拖下并一度考虑按“鬼踩人”规律杀人；刚受杨间教训的他没有贸然放鬼。贺天雄最终顾忌李瑶是杨间的人，改为威胁日后报复、夺车离开。他留下别墅钥匙和屋内赔礼，称房子输给杨间、双方扯平，且不再主动找麻烦。该表态是否可信、他将去向何处和是否脱离朋友圈，均不详。（第553章）

#### 第609章·在飞机上被要求接管公司

方世明藏入乡镇地下室后，以视频电话联系正在飞往国外的贺天雄，要求其在自己离开期间打理朋友圈公司。贺天雄见方世明形销骨立，先以国外业务为由推辞，实际顾忌接手已遭杨间清洗的组织会成为目标；他直言杨间若还活着，自己既不敢招惹也打不过。方世明以威胁后果迫他答应在杨间死亡时接管，贺天雄只能表示尽力。此通话后方世明即遭柴刀媒介斩杀，贺天雄是否得知死讯、是否继续出国或实际接管事务均不详。（第609章）

#### 第621-622章·八号别墅被用作鬼画临时收容处

杨间在第621章搬走小区内发现的鬼画后，选择前往贺天雄耗费巨资建造的八号别墅。该处墙壁和地面贴有金箔，另有小型安全屋，杨间认为可在短时间内减少鬼画对外界的影响；他随后把画放入安全屋般房间，揭开画布并由此进入画中城市。别墅此时由杨间使用，贺天雄本人未在场；其是否知情、是否仍拥有该处产权及金箔防护能否长期约束鬼画均不详。（第621章、第622章）
