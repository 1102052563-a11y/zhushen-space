# 何川

## 简介

何川是原小强俱乐部成员，长期少在俱乐部露面，叶枫曾猜测他已死于厉鬼复苏。他在大昌市鬼婴事件会议现身，对杨间杀死王小强抱有强烈敌意，精神状态被叶枫评价为不稳定。（第225章）

## 经历

#### 第225章·参加鬼婴事件会议

王小明召集大昌市驭鬼者开会时，何川到场并直接质问杨间杀死王小强一事，声称想杀杨间。张韩、叶枫先后要求他冷静，避免在大型灵异事件期间内斗，他才暂时收敛。王小明公布市内已确认鬼婴由一百二十只增至一百三十八只后，何川同其余参与者接到捕获一只鬼婴供研究的任务。其驾驭厉鬼、具体能力和是否参与任务均未写明。（第225章）

#### 第227-229章·西城区失踪

何川与赵开明、叶枫负责西城区巡查，进入道路消失、环境安静异常的区域后，先在叶枫身边失去踪影。第229章，叶枫在小巷发现一颗被啃食得残缺的头颅，面貌可辨为何川，附近另有第二阶段鬼婴；但头颅之外的身体、死亡经过和其体内厉鬼去向均未写明。赵开明向总部只报告叶枫、何川失踪，故状态保留不明。（第227章至第230章）
