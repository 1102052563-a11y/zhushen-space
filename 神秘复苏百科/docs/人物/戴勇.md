# 戴勇

## 简介

戴勇是中山市一处居民小区的住户，三十出头，事业略有成就。他因人头气球传闻影响工作、收入下降，对灵异传闻非常抵触，认为只是骗老人的怪谈；即使朋友以凯撒大酒店封锁、武警警戒和抓鬼部门抵达的照片劝他离城，他也仅将其当作小道消息，没有带儿子离开。（第341章）

## 经历

#### 第341章·家中遇袭

戴勇回家后发现妻子外出打麻将，只留下五岁的乐乐独自在客厅。他做饭时收到朋友再次示警，仍以工作和吃饭为由结束对话。饭菜做好后，乐乐站到阳台凳子上，说母亲在窗外，戴勇先认为孩子看错；关窗时却看见妻子的人头悬浮在十八楼外，耳环仍是其熟悉的那一副。人头随风越窗进入室内，戴勇想带乐乐逃走，却发现儿子的头颅也已离开颈部飞起。他在恐惧中尖叫，原文此后未记载他是否逃离、是否受害或妻子此前遭遇。（第341章）
