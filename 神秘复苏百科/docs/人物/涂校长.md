# 涂校长

## 简介

涂校长是发生油画异常大学的校长，约五十岁，由王泉带到篮球馆协助总部处理校内事务。他在现场与杨间首次见面，负责学校方面的疏散和协调，但原文未交代其全名、学校名称或此前是否知道灵异事件。（第554章）

## 出处

- 王泉介绍涂校长可负责处理校内问题。鬼画出现地点不在篮球馆，杨间要求先把接触过寝室油画的刘紫交由总部观察，并让苗小善留在馆内由自己照看。（第554章）
- 宿舍楼火警铃响后仍有不少学生滞留，涂校长解释时间已晚、许多学生可能熟睡；王泉因此要求他在十五分钟内完成全部撤离，并安排人员协助。（第554章）
- 杨间建议若美术馆排查危险，可爆破或焚烧以减少搜寻风险；涂校长以馆内有名人真迹、供学生临摹学习为由反对。美术馆是否排查、最终采取何种方案及学生是否全部疏散，原文未写。（第554章）
