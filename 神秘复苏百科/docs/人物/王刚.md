# 王刚

## 简介

王刚是杨间班级学生。第16章，几名曾和方镜同行、后被杨间从绿化树林带出的同学向杨间道歉时，提到方镜一路留下赵强、王刚，甚至女同学断后。原文没有写明王刚何时与方镜分开、被何种鬼追击，或是否已死亡。（第16章）

## 出处

- 杨间回到操场后，四名此前跟随方镜的学生为未阻止方镜把杨间推进厕所道歉，并称后来才知道方镜会把同学留在后方拖延鬼物。（第16章）
- 在这段指控中，学生明确点名赵强和王刚，也提到方镜连女同学都留下断后。王刚没有直接出场、没有台词，且此后第20章前未再被提及。（第16章）
- 方镜在第13章已承认会用同学性命拖住厉鬼，但没有在本批明确叙述王刚的具体遭遇。因此只能记录其曾被方镜留下和目前下落不明，不能据此认定死亡。（第13章、第16章）
