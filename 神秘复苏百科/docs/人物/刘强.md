# 刘强

## 简介

刘强是福仁商场的青年保安，今年刚大学毕业，近期才到商场工作。商场此前已有一名保安失踪，他仍因自己是外地人、这份工作包吃住且工资尚可而选择留下。第33章，他与新来应聘保安的杨间共同巡逻。（第33章）

## 经历

#### 第33章·带杨间巡逻

杨间换上保安服后，刘强按商场规定带他从五楼向下巡逻，计划最后检查地下停车场；因停电，二人只能步行。他承认偶尔闻到臭味，却把它解释为冰柜冷冻食品变质，并不相信商场闹鬼或失踪者就在其中。杨间指出黑暗中另有站立人影时，刘强用手电确认那是人体假模特，便将其搬回摆着十余残缺模特的黑暗商铺。杨间追问模特为何出现在路中间，刘强只猜是同事试衣服所致。二人离开后，原文写明该模特又立在商铺门口；刘强对此没有察觉。巡逻未发生直接事故，他与杨间回到保安室。（第33章）

## 出处

- 刘强说明商场此前一名保安失踪，自己因工作条件而未离职。（第33章）
- 他带杨间完成一次巡逻，却未发现模特自行复位的异常。（第33章）
