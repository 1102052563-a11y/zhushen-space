# 苏珊

## 简介

苏珊是第295章出现在观江花园小区的一名艳丽女记者，自称某电台记者。她与数名随行者以采访大昌市迷雾事件为名接触张韩和杨间，持续追问灵异事件、“鬼眼刑警”及居民保持沉默的原因。杨间认为其采访具有明确目的，且她所属小队的真实身份并不符合普通新闻队特征。（第295章）

## 经历

#### 第295章·借采访接近观江花园住户

苏珊穿白衬衫、妆容精致，在张韩带妻子于小区行走时上前采访。张韩已表示不接受采访，她仍要求回答迷雾是否化工污染、网上灵异传闻是否真实；杨间到场后要求一行人离开私人物业，她又改以“新闻自由”和公众知情权施压，并提及神秘论坛所说的“鬼眼刑警”。她拿着录音器继续录下杨间砸毁摄影机后的反应，自报“我是记者苏珊，正在持续为你们报道”。（第295章）

#### 第295章·身份受到怀疑

杨间鸣枪警告后，随队数名外国男子迅速摸向腰间武器。苏珊本人被枪声吓到，但原文没有写她拔枪或参与武装动作；杨间据随行人员的反应判断，这一行人以采访为掩护，目的可能是接触、搜集“鬼眼刑警”信息。摄影师特瑞转称有人邀请杨间参加舞会，苏珊和同伴的真实委托者、是否确属记者以及被警方带走后的结果均不详。（第295章）
