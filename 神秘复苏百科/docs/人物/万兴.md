# 万兴

## 简介

万兴是鬼邮局一楼11号房走出的消瘦男子，面颊消瘦、神情枯槁、目光阴沉，随身持普通手枪。他看见杨间后称其为“新人”，又警告外界再横的人进入邮局也活不了几天，显示其已在此生活或执行任务一段时间；其成为信使的经过、在邮局停留多久和是否具备驭鬼能力均不详。（原文回跳第739章）

## 经历

#### 原文回跳第739章·在一楼与杨间冲突后被控制

万兴从11号房走出，原欲前往其他房间，发现杨间后以戏谑口吻称其是倒霉新人。杨间为了解邮局情况要求他停下，万兴拔枪射击；子弹未能阻止杨间，反被杨间以鬼手掐住、抛倒在地。孙瑞用金色手杖抵住其眼眶，令其不得反抗。（原文回跳第739章）

万兴报出姓名后，孙瑞试图用卫星定位手机查询身份，但邮局内信号无法连接外界。11号房又走出一名未具名女子，万兴因畏惧杨间一行而提醒她服从。杨间据其对邮局和“新人”的熟悉，判断他疑似信使，并以保命为条件要求其交代全部情报。万兴尚未开始具体供述，姓名和背景也未获外部核验。（原文回跳第739章）

#### 原文回跳第762章·抵福寿园却被封锁阻挡

万兴离开鬼邮局后赶赴大海市，希望完成红信集体任务，却发现福寿园已被灵异论坛封锁，无法进入。他与王善会合，承认自己畏惧杨间、孙瑞和李阳，得知三人曾在邮局内逼退夜游鬼后，选择不再硬闯、在外围等待。（原文回跳第762章）

钱蓉没有与他同行，是否放弃任务、被困其他地点或已遇险均不详。万兴能否因杨间投递红信而被判定完成任务，也待后文。（原文回跳第762章）

#### 原文回跳第781-784章·进入二楼后被房门拖走

福寿园红信投递成功后，万兴与王善等一楼信使一同进入鬼邮局二楼，暂住在王善所在房间。他仍畏惧杨间，亲见对方占下21号房、撕毁二楼红信后，没有试图阻止。（原文回跳第781章、原文回跳第782章、原文回跳第783章）

二楼红信撕毁引发七间房内厉鬼袭击时，万兴被无形力量拖入一间房内，房门关闭后动静中断。王善当场认出目标是万兴，后文未给出遗体或存活证据；本页按现场表现改为死亡，是否存在特殊脱身可能待后文验证。（原文回跳第784章）
