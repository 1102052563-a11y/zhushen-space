# 婉月

## 简介

婉月是费总的妻子，住在费总安排的一处居民楼三层住所。费总回家前，她正在厨房做饭，并招呼儿子费小宝给屋内的“叔叔”盛饭；她当时不知道杨间已经先进入屋内，也不知道丈夫正因朋友圈与杨间的冲突遭到清算。原文仅以“婉月”称呼她，没有交代年龄、职业、与费总相识经过、是否知晓朋友圈或灵异事件。（第541章、第542章）

## 经历

#### 第542章·被遗忘记忆影响

费总进门后因看见陌生男鞋和听见男子声音而冲入屋内，婉月随后从厨房走出。费总担心妻儿受害，要求他们立刻离开本市；婉月却拉着费小宝躲到杨间身后，把费总当作突然闯入的陌生人，并询问杨间是否需要报案。杨间称只是走错门、会自行处理，她因记忆中杨间是朋友而暂时接受说法、带儿子回房。费总由此得知，杨间已令婉月和费小宝忘记他的存在，记忆中只留下一个早已死亡且不负责任的男人。记忆被改变的具体方法、能否恢复、婉月之后是否会发现异常均不详。（第542章）

#### 第543章·察觉现实与记忆不协调

杨间离开前，婉月仍邀请他留下吃饭。她虽按被改写后的记忆把杨间视为熟人，却觉得先前闯入者似认识自己和费小宝，又疑惑杨间如何持有住处钥匙、为何记忆里的他与现实中的冷漠感不同。她没有追查，而是因仍在做饭返回厨房；原文以此显示她即使察觉不协调，仍选择相信既有记忆。费总随后坠楼死亡，但婉月是否听闻、是否将事件与住处异状相连，均不详。（第543章）

- [费小宝](费小宝.md)：费小宝之母，事发时带其远离费总并返回房内。（第542章）
