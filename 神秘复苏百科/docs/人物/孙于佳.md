# 孙于佳

## 简介

孙于佳是苗小善在大京市大学宿舍的室友，睡在苗小善上铺。张霞失踪、寝室内留下老旧油画时，她起初和其他室友一样，把苗小善要求撤离的举动视为疑神疑鬼，也认为杨间所说的灵异事件不可信。她没有见过第七高中事件，不理解苗小善对油画的恐惧；直到油画自行从张霞床铺移到王悦床头，她才承认寝室出了无法以常理解释的问题。（第550章、第551章）

## 经历

#### 第550章·拒绝随苗小善先行撤离

苗小善给杨间打电话求助后，孙于佳在寝室里打趣她是借口联络高中同学，并称张霞也许只是出去约会、次日会回来。苗小善说明张霞失踪和油画可能关联灵异事件，催促所有人离开；孙于佳仍认为“真有鬼”只是杨间故意吓胆小的苗小善，选择留在被窝。她不了解苗小善曾在第七高中经历大规模死亡，也未接受杨间到场后的警告。杨间带走苗小善、刘紫随后自行逃离时，孙于佳仍与王悦留在原寝室，因而继续处在油画附近。（第550章）

#### 第551章·发现油画转移

杨间和苗小善离开后，孙于佳逐渐意识到杨间不可能在通话数分钟内准确抵达女生寝室，刘紫又反常追出且未返回，故重新想起油画问题。她掀开被子查看原在张霞床上的画，发现画已不在原位；经过寻找，油画竟出现在王悦床头。她因此惊恐呼喊王悦，并明确相信油画存在异常。此时油画尚未发动可见袭击，孙于佳与王悦决定穿衣离开；两人是否能从门口安全撤出，仍取决于随后变化。（第551章）

#### 第552章·目击王悦被油画吞没

油画移至王悦床头后，孙于佳先劝其立即离开。王悦却返回床边取包，画中五官模糊、穿红色欧式礼服的女子被放大，两只苍白手掌伸出画框。孙于佳看见王悦正会触及那双手，因极度恐惧无法发声或移动，未能及时阻止。王悦被画中手掌抓住、连人带画摔倒，身体逐步陷入画中；孙于佳只看见画外挣扎的双腿，最后腿部不再动弹。她被吓瘫并哭泣，却没有被画直接袭击。（第552章）

#### 第554章·翻窗逃离并获救

孙于佳没有走宿舍楼梯，而是从四楼窗户翻到外侧，浑身发抖、哭泣着试图沿外墙下爬。杨间从楼下认出她，判断其宁愿冒险坠楼也不回寝室，说明楼内可能已有直接灵异威胁；他拒绝让人员贸然入楼救她，以免触发未知条件。孙于佳因寒冷、恐惧和体力不足松手坠下，杨间单手稳稳接住，随即把她交给现场人员询问寝室发生的事。她获救后是否完整说出王悦、油画和画中手掌的经过，及是否受灵异影响，原文未写。（第554章）

## 人际关系

- [苗小善](苗小善.md)：同寝室室友；孙于佳起初不信其撤离警告，后来因油画异动才承认异常。（第550章、第551章）
- [王悦](王悦.md)：同寝室室友；二人一起准备撤离时，孙于佳目击王悦被油画吞没。（第551章、第552章）
- [杨间](杨间.md)：杨间在宿舍楼下接住从四楼坠落的孙于佳，并要求人员向她问明楼内情况。（第554章）

#### 第563章·获送回寝室后的恐惧

孙于佳在鬼画暂被限制后由工作人员送回寝室，与刘紫一同按杨间安排陪伴苗小善。她先前亲眼见到的红衣画中鬼与苗小善回忆中第七高中的尸斑老人并不相同，因而直接意识到灵异事件不是单一特例，世界上存在不止一只鬼。她仍有明显恐惧，原文没有写她接受心理安抚、是否能休息，或其后是否再次受到鬼画影响。（第563章）
