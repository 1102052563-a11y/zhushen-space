# 魏晓红

## 简介

魏晓红是福仁商场女收银员。监控显示她在五天前被陌生中年女子从背后取下头颅，身体随即失去生命；陌生女子换上她的头后，以魏晓红身份在商场活动。原本的魏晓红头颅逐渐腐烂，并在第44章滚至一楼。（第44章、第49章）

## 经历

#### 第49章·被换头与身份伪装

监控中，陌生中年女子在二楼楼梯口停留半小时后靠近正在收银台活动身体的魏晓红。趁魏晓红背对时，女子从其颈部取下头颅，过程无血、无惨叫；无头身体僵立。女子随后摘下自己的头，换上魏晓红头颅，并抱走女子原头和魏晓红身体，成为外观如常的“魏晓红”。此后她继续上班，并在四天前趁丽姐背对时再度换头。（第49章）

#### 第44-45章·腐烂头颅与目击误解

停电后，魏晓红的腐烂头颅滚到唐安等人面前，杨间检查其颈部无血、皮肉已像死去多日。江艳四天前曾看见“魏晓红”自行取头，因不知道身份已被替换而将其误认作魏晓红本人。真相由第49章监控确认：当时行动者是戴着魏晓红头颅的换头鬼，而魏晓红已死。（第44章、第45章、第49章）

## 出处

- 魏晓红为福仁商场收银员，成为换头事件首个由监控明确展示的受害者。（第49章）
- 她的死亡推动丽姐等更多员工被换头、商场鬼群形成。（第49章）
