# 钱毅

## 简介

钱毅是参加培训基地培训的驭鬼者。鬼差侵入基地后，总部经黄子雅向杨间转达：他原在宿舍楼三楼，正从另一条安全通道下至一楼，双方本应在楼道汇合。原文没有写出他的外貌、所驾驭厉鬼、过往事件或是否真正拥有脱离鬼棺空间的手段。（第447章）

## 经历

#### 第447章·约定向杨间会合

杨间在宿舍楼搜到黄子雅和另一名未具名受训驭鬼者后，黄子雅从总部得到钱毅的位置：钱毅已由三楼另一条安全通道回到一楼。杨间认为集结速度必须快过鬼差夺鬼成长的速度，遂要求钱毅折返上楼，自己则带着两人赶往另一安全通道。此时钱毅尚未出场，也没有直接回应杨间；关于他是否已获悉鬼差以落单者为目标的完整规则，原文未写明。（第447章）

#### 第449章·关闭通讯并独自逃离

杨间在楼梯口等待约定会合的钱毅，却始终没有见到人。黄子雅联系总部后得知钱毅失去回应，似乎主动关闭通讯；总部定位其卫星定位手机后又发现，他正在快速朝培训基地门口移动。杨间据此判断，钱毅借通话把自己一行留在离鬼较近的宿舍楼，试图以其他人的停留拖住鬼差、为独自逃离争取时间。杨间并不反对有人能实际离开，但认为这种做法耽误了争分夺秒的集结，并指出钱毅不了解基地处于鬼棺封闭空间，单独向外逃走未必可行。这个判断是杨间的现场推断；钱毅关闭通讯的真实动机、是否成功到达出口、是否遭鬼差袭击及最终生死均未写明。（第449章）

#### 原文首个第456章·逃离失败后因白烛幸存

钱毅独自离开宿舍楼并走出培训基地后，仍在黑暗中绕回原地，确认此地不是能够靠自身能力脱离的普通鬼域。他承认先前故意以杨间拖住宿舍楼内的鬼，为自己争取逃离时间，如今后悔没有会合；在独自落单、黑暗向他侵蚀时，他判断鬼差已选中自己而崩溃。黑暗靠近约三四米后却突然退向另一方向，实际是杨间点燃的白色鬼烛将鬼差引走。钱毅不知道原因，也不能确认杨间五人是否能抗住被引去的鬼差。（原文首个第456章·厉鬼来袭）

#### 第459章·重新开启信号请求会合

杨间一行维持白烛诱导后，黄子雅通过总部确认钱毅仍活着，卫星定位信号在前方约五百米。钱毅是否主动重新开启手机、是否知道杨间一行已有白烛方案及当时具体处境均未写明。杨间虽因其旧事愤怒，仍接受张雷的劝说，以增加幸存者和鬼数量为先，决定向定位点前进；第459章末尚未实际会合，但钱毅此时仍存活。（第459章）

#### 第460-461章·会合时遭鬼差袭击

钱毅在黑暗中循卫星定位向杨间一行靠近。他知道单独行动最危险，也担心杨间会追究自己此前关闭通讯、借同伴拖延鬼差的行为；但培训基地无法走出、周围愈发异常，使他只能选择会合。定位信号重合时，他先闻到尸臭、听见沉重脚步，手机屏幕又被发黑僵硬的手遮住，随即认定真正的鬼已到近处。（第460章、第461章）

鬼差以老旧草绳勒住钱毅脖颈，将他吊起；他的厉鬼力量当场沉寂，无法反抗。紧接着一双冰冷黑手撕开衣物、探入血肉，似在搜寻体内的鬼，令其身体撕裂变形、吐血濒死。杨间带队冲到鬼差与钱毅近处，使其重新被计入队伍的鬼数，袭击才停下。钱毅仍被草绳吊着、浑身流血扭曲，却在本批末存活；伤势、体内鬼是否受损及能否继续行动均不详。（第461章、首个第462章·鬼回头）

#### 第467-468章·伤重死亡与遗留绣花鞋

鬼差离开后，钱毅仍昏迷在血泊中。杨间以无头鬼影检查，确认其尚有微弱呼吸，却因失血过多，即使立即送医输血也仅有存活可能；杨间因其此前背约，没有亲自用鬼影拼合伤口，只令张雷通知总部。救援抵达前，钱毅在昏迷中断气。（第467章）

尸体随即被体内灵异驱使站起并异常远行。杨间以鬼影压住尸体，发现根源是一双藏在皮鞋内的红色绣花鞋，脱下后收入黄金盒。钱毅驾驭能力、鞋的来历与是否加剧伤势均不详；状态据第467章修正为已死亡。（第468章）

## 关系

- 杨间：原约定在宿舍楼汇合，钱毅却在约定后关闭通讯并向出口移动；杨间据此认定其以同伴作拖延，并表示若其存活会追究。（第447章、第449章）
- 黄子雅：黄子雅作为联络者接收总部关于钱毅位置、失联和定位结果的信息。（第447章、第449章）
