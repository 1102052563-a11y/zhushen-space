# 黛西

## 简介

黛西是带有外国面孔的混血雇佣兵，因国外环境不适合发展而来到亚洲，加入专门猎杀驭鬼者的公司。她受王小强雇佣，参与观江花园内针对杨间的行动。（第195章）

## 经历

#### 第195章·售楼部探问住处

行动组伪装进入观江花园后，黛西和孔峰等人到售楼部打听杨间位置。她以熟练中文向销售人员谎称自己由杨间介绍而来，套出杨间住在原售楼部改成的庄园别墅、未必正在其中的消息，并立刻把情报传给分散人员。张伟直播跟上孔峰一行时，她先提醒孔峰此人可能是杨间朋友；阿海刺伤张伟后又遭张伟以特制手枪击中，她确认阿海伤势严重。至本章结束，她未被售楼部人员识破，也未说明是否知道王小强携带物的作用。（第195章）

#### 第197章·鬼域内逃亡与死亡

狙击和第二阶段计划相继失效后，黛西与其他行动人员被杨间的红光鬼域困住。她看见孔峰和多数同伴已死，认定这项任务不该接，杨间在她看来已不再像普通人。逃跑中，杨间直接出现在她前方，让她自行结束生命；她试图以陪睡交换生机，又趁机开枪偷袭，均未奏效。杨间转至其身后，以手枪抵住她后背心脏位置射击。黛西倒地流血死亡，至死仍无法相信自己会死在不足二十岁的杨间手中。（第197章）
