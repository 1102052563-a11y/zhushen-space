# 王野

## 简介

王野是自称来自亚洲以外某地区驭鬼者总部的组长，暂时负责接待杨间。他在杨间搬运一幅鬼画、需要车辆前往城郊八号别墅时主动现身，显然已预先注意到杨间的行踪。王野以豪华房车、陪同人员和两名女公关接待，表面称只想提供车辆，实际意在趁杨间与总部、朋友圈冲突后建立合作关系，并以一份欧洲鬼画事件的绝密档案作为交易筹码。（第621章）

## 经历

#### 第621章·借送行接触杨间

杨间带着白布遮盖的鬼画走到路口，正打算雇货车时，一名西装男子先急忙跑来代为安排车辆。王野随即在房车内自报姓名，称自己只是负责接待；车辆的路线原本已有绕行意图，杨间以走错路口便杀人的警告迫其照导航行驶。王野解释自身属于亚洲以外的总部，否认是要杨间“跳槽”，改称想建立合作关系。杨间认为双方实际无异，且不愿在八音盒诅咒尚存时再受其他势力约束，直接拒绝。（第621章）

#### 第621章·以鬼画档案议价失败

为改变杨间态度，王野提出持有欧洲方面未公开的鬼画情报，开价是一件A级灵异事件，并以黄金、资产和女性加码。杨间指出鬼画事件可能很快由总部解决，资料可能贬值，且王野可能把同一份资料卖给自己和曹延华，因而认定其报价与诚意都不足。王野仍试图讨价还价，杨间则放出鬼影警告其不得再说话。抵达八号别墅后，王野把一份档案当作赔礼免费交出；杨间认为免费物品不可信，当面将其烧毁并离开。（第621章）

#### 第621章·交易后死亡

杨间离车后，王野的下属称赠送档案没有得到部长同意。王野承认送出的只是存在错误的副本，也承认自己低估了杨间的谨慎，并把此次接触的后果视为自身轻率的代价。话未说完，两只冰冷发青的手掌出现在他的颈部，将其颈部扭断；他倒在车厢内，双眼睁开且已无气息。手掌的具体来源、是否由杨间先前鬼影留下，以及王野所属总部将如何处理其死亡，原文未说明。（第621章）
