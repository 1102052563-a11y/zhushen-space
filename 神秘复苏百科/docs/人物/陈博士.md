# 陈博士

## 简介

陈博士是平安科技研究基地的一名实验室负责人，约三十出头，戴眼镜。他原先负责的实验室隶属朋友圈旗下，朋友圈覆灭后由总部接管，相关项目一并交接。第640章，他与王小明在基地内讨论残缺鬼棺、灵位和郭凡的意识替换方案；第644章，他在方案实施期间先行组织大部分工作人员撤离地下实验室。其姓名全称、具体专业、职务层级和此前负责项目的完整内容均不详。（第640章、第642章、第644章）

## 研究与观点

- **鬼棺压制风险判断**：陈博士确认此前鬼棺实验“很顺利”，却不能确定残缺鬼棺是否足以压制杨间身体内全部厉鬼；他担心郭凡没有驾驭三只鬼的经验，替换身体后可能引发失控。这是他对王小明方案的风险评估，并非已验证结果。（第640章）
- **反对激进实验**：他认为应按发现、了解、控制的顺序研究未知灵异，而非直接把活人变成鬼、试图窃取厉鬼能力。他将王小明的做法归为因害怕灵异全面爆发而抢时间的激进路线。（第644章）
- **基地危险认知**：他向助理强调，实验室内看似不起眼的灵异物只是被关押才显得安静，一旦外泄也可能持续造成数百、上千人死亡。（第644章）

## 经历

#### 第640章·参与灵位与鬼棺方案讨论

王小明在基地最高防范隔间外询问鬼棺实验情况时，陈博士答复实验顺利，却提出残缺鬼棺未必能压住杨间三只鬼的风险。他进一步点破意识进入灵位后，王小明可以让任何人取代杨间身体，询问方案最终是让杨间还是郭凡活下；王小明则表示只要八音盒诅咒能从杨间意识剥离，郭凡便可代为承受死亡，否则郭凡占用杨间身体也属最坏情形下的备用结果。陈博士没有改变方案，仅记录并提出疑问。（第640章）

#### 第642章·撤离第五层前的警示

卫景带杨间抵达地下第五层后，陈博士仍在王小明身边。他警告此次实验不只涉及杨间，还存在其他未知因素，灵异规则远比研究者已掌握的复杂；王小明决定继续后，陈博士带助手撤出该层，将现场交给王小明、卫景与杨间。此举令多数研究人员暂时远离意识替换和鬼棺压制的直接风险。（第642章）

#### 第644章·组织撤离并遭遇鬼童活动

方案开始后，陈博士带数名助手从地下实验室撤离，要求各人回办公室整理资料后离开，并放假三天。他因担心实验涉及多个不稳定因素而提醒保密，却也承认王小明不会因风险停止寻求研究突破。其后刘助理等人在走道见到滚动的腐烂死人头与穿鬼寿衣的鬼童，准备通知陈博士和王小明；本批末未写陈博士是否收到通知、能否组织处置或是否已安全撤离。（第644章）

#### 第645-646章·试探撤离失败后留守

陈博士向王小明报告楼内见鬼，得到“暂不离开、可先派一人试探”的答复。他虽知道灵异案例通常应尽快远离现场，却接受助手分批撤离的提议，让小张先行、其他人等待短信。小张跨出大楼即死且十分钟没有联络，陈博士据此确认封楼鬼会杀死离开者，命剩余人员暂时留在办公室并设法继续同王小明沟通。第648章王小明获知陈博士仍活着；其后能否获救不详。（第645章、第646章、第648章）

#### 第657章·获准离楼并提供痕迹线索

杨间沿尸水痕迹上二楼时，鬼童拍开陈博士等人躲藏的办公室门。陈博士听见杨间声音并认出其身份后制止助手跳窗，确认鬼童受杨间控制；杨间告知楼内灵异已解除，允许他们随时离开。陈博士随后说明曾听见一只入侵物与鬼童在楼内追逐，最后活动地点大致在四、五楼窗边，为杨间追查尸水残留提供方向。（第657章）
