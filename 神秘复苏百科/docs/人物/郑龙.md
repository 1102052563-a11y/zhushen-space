# 郑龙

## 简介

郑龙是裴东猎杀驭鬼者团伙的成员之一。他在小安县针对冯全的鬼域陷阱中埋伏，原文未交代其背景、年龄、何时加入团伙或是否从他人身上继承厉鬼。杨间进入鬼域后将其作为第一名击杀目标，称他只驾驭一只鬼却模仿他人参与驭鬼者战斗，判断其迟早会因自身能力或冲突死亡。（原文回跳第635章）

## 能力

- **鬼叫**：郑龙死亡后嘴巴仍异常张大，口腔深处传出厉鬼般凄厉怪叫；裴东合上其嘴后叫声暂止。该声音的杀人规律、能否影响鬼域及郑龙生前如何使用均不详。（原文回跳第635章）

## 经历

#### 原文回跳第635章·在鬼域中被杨间杀死

裴东以“一对六”威胁杨间时，杨间已通过鬼域悄然锁定郑龙位置。杨间随后将郑龙尸体从鬼域中踢到裴东面前，尸体张口发出可怕怪叫，表明郑龙已被瞬间杀死。裴东等人无法确认杨间何时出手，这一死亡使团伙人数从六人减至五人，也直接动摇了裴东继续对抗的信心。（原文回跳第635章）

郑龙死后体内鬼尚未被收容，尸体由裴东暂时合嘴压住怪叫。其遗鬼最终由谁处理、尸体是否复苏或是否被团伙带走均不详。（原文回跳第635章）
