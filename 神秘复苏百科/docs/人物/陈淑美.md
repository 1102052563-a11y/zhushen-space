# 陈淑美

## 简介

陈淑美是熊文文的母亲，住在平安酒店附近高档住宅小区十层1001户。熊文文的父亲死于一次与灵异事件有关的出游事故；事故细节、她是否亲历过程及如何获知总部事务均不详。她知道儿子的预知能力存在复苏代价，并希望尽量让他远离危险。（第573章）

## 外貌·性格

杨间初见她时，她约三十岁，穿居家服与围裙，眉宇带着伤感忧愁。她礼貌留杨间吃饭，也谨慎确认其是否可靠；在杨间拒绝保证熊文文能活下来后，立即反对儿子入队，显示其将儿子的安全置于首位。（第573章）

## 经历

#### 第573章·拒绝熊文文加入杨间队伍

熊文文为加入杨间的小队带杨间回家。陈淑美得知杨间只有二十岁后更感意外，要求他保障熊文文安全。杨间坦言驭鬼者随时可能死于灵异事件或厉鬼复苏，也不能作虚假保证；陈淑美因此请他离开，要求熊文文不要组队。熊文文虽认为杨间没有说错，仍因母亲生气而未再争辩。她的决定是否会改变不详。（第573章）

## 关系

- [熊文文](熊文文.md)：其子；她担忧其预知能力的复苏代价，反对其加入杨间队伍。（第573章）
- [杨间](杨间.md)：熊文文带来商谈组队的对象；她因杨间不能保证儿子安全而拒绝组队。（第573章）
