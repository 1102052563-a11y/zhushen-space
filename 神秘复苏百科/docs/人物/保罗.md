# 保罗

## 简介

保罗是尚通科技在尚通大厦顶层办公室内现身的外籍中年男子，自称公司“boss”。他金发、穿西装，中文不甚流利，却在杨间带搜查人员抵达前已掌握杨间资料。杨间确认保罗本人不是驭鬼者，但其办公室有无法被鬼域侵入的黄金隔层和疑似异常物品。其国籍、完整姓名、公司职务登记与是否为实际控制人均不详。（第324章）

## 经历

#### 第324章·顶层办公室相见

尚通大厦被搜查时，保罗没有逃离，而在四十五层办公室取酒等待。杨间直接指出办公桌抽屉有异常，保罗承认唱片是“问候”，却没有说明内容；杨间立刻将唱片掰断。面对杨间追问观江小区武装袭击，他没有正面解释，只称公司可随时关门，并自称早期待与“鬼眼刑警”见面。杨间不接受谈判，认为公司可能与王小强雇佣行动有关；保罗是否参与雇佣、知晓程度和唱片原内容均未证实。（第324章）

#### 第325章·黄金密室与强制谈判

杨间以鬼域发现酒柜后的房间无法被探查，章华一行移开酒柜后露出厚重保险室门。保罗否认密室存在并试图以律师、隐私和谈判拖延；杨间则以鬼域把保罗困在两张沙发和茶几构成的局部空间，警告不得离席。保罗试图离开时双腿脱落，被迫爬回沙发；他承认观江小区行动人员是公司员工，却声称公司只是接活、不管伤亡，并称可交出公司、赔偿或替杨间处理王小强。杨间告知王小强已死，要求的不只公司而是“全部”，拒绝保罗以退让换取存续的提议。（第325章）

#### 第325章·播放磁带未果

被拒绝后，保罗取出随身小播放器和磁带，称其中录音可能改变杨间决定。杨间感到磁带附有异样力量，仍在录音展开前以鬼域取走并捏碎播放器与磁带；碎片中却持续传出沙沙声。杨间据此警告，一旦事件涉及灵异，他可不等常规搜查证据直接处置保罗和公司。录音内容、磁带来源、沙沙声是否构成灵异袭击及保罗的后续处境，本批均未揭示。（第325章）

## 人际关系

- [杨间](杨间.md)：尚通科技受搜查时首次正面相见。保罗试图谈判与让步，杨间拒绝并将其限制在办公室局部鬼域内。（第324章、第325章）
- [章华](章华.md)：章华持搜查令带人进入办公室，要求保罗配合开启黄金隔绝的密室；搜查结果未在本批写出。（第324章、第325章）
