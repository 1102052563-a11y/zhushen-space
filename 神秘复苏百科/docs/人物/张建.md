# 张建

## 简介

张建是费总身边的保镖，按李瑶交给杨间的资料，他平日不在平安大厦内露面，只有费总下班回家时才会开车暗中随行保护。李瑶认为费总虽是普通人，却负责朋友圈大量事务，若要处理费总，张建是必须先解决的护卫；她建议杨间先在费总下班前处理张建，再利用其车辆接近费总。张建是否为朋友圈正式成员、何时成为驭鬼者、驾驭了什么鬼及实际受谁指挥，原文均未说明。（第538章）

## 外貌·性格

杨间在酒吧包厢见到张建时，他约三十岁，脸色苍白浮肿，正抽烟并与数名年轻男女打牌。他把钱袋放在脚边，定下赢一张牌给一万元、输一张牌喝一杯酒的规则，以灌酒和散钱取乐；赢钱者和女伴争抢钞票时，他只旁观并称这种场景有趣。面对保安把杨间当闯入者拦下，他先撇清关系，随后一脚将两名成年保安踢出包厢、令其吐血，显示其力量异于常人。张建又把杨间认作“鬼眼杨间”，一面招揽，一面把普通人贬为应当恐惧、下跪的对象，言行呈现出明确的傲慢与失控倾向。（第539章）

## 能力

- **异常力量**：张建踢开两名成年保安时，两人直接飞出包厢并吐血；这证明他至少具备超出常人的力量。该力量是否来自厉鬼、是否还有其他灵异能力、使用代价及限制均不详。（第539章）
- **未知鬼域应对**：杨间以红光覆盖包厢、使人群与原有环境消失后，张建立刻认出这是鬼域，并据此判断杨间来意不善。这只能说明他具备辨认鬼域的常识，不能证明其自身拥有鬼域或反制能力。（第539章）

## 经历

#### 第538章·被列为接近费总的首要阻碍

鬼画事件暂时平静后，杨间从李瑶取得部分朋友圈成员资料，并决定不逐一攻击挂名者，而是先瓦解少数核心人员。李瑶筛出费总，说明此人是平安大厦总裁、负责为朋友圈制定对付杨间的计划；但费总本人是普通人，其身边真正的障碍是保镖张建。她掌握到张建在平安大厦对面街上一间不对外营业的私人酒吧活动，便留在车内避免暴露，让杨间独自从后门进入。杨间尚未在本章与张建接触，李瑶的路线、费总是否按时下班及计划能否执行均不详。（第538章）

#### 第539章·在私人酒吧遭杨间锁定

杨间进入挂着暂停营业牌的酒吧，发现内部仍聚集大量年轻男女，遂进入张建所在的豪华包厢。张建认出杨间后没有立即抵抗，先以女伴、消费和加入朋友圈等说辞试探招揽；他声称驭鬼者寿命有限，应趁活着把普通人踩在脚下。为证明自己的观念，他掐住一名女伴的脖子，逼其回答是否爱他，之后又把一袋现金踢给对方，引得其他人争相附和。杨间没有接受招揽，而是用鬼域隔绝所有旁人、把包厢变成红光覆盖的空间。张建认出鬼域后问杨间是否来杀自己；杨间只说自己赶时间、要顺手把他“埋了”。本章至此没有写两人交手、张建死亡或杨间是否完成处置。（第539章）

## 关系

- 费总：按李瑶掌握的资料，张建负责在费总下班后开车暗中保护他；费总是否直接向张建下令、二人私下关系不详。（第538章）
- [杨间](杨间.md)：杨间为瓦解朋友圈核心人员而前往酒吧锁定张建，并在包厢展开鬼域；冲突结果未写。（第538章、第539章）
- [李瑶](李瑶.md)：李瑶向杨间提供张建的身份、活动地点和护卫费总的情报，但未进入酒吧，避免暴露内线身份。（第538章）
