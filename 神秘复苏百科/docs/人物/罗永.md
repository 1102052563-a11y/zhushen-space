# 罗永

## 简介

罗永是中安市外陵镇罗姓老宅的后人，约五十岁，皮肤略黑、身形偏瘦，日常以务农和外出做工为生。他从父辈继承祖宅后原拟推倒旧房重建，施工时在地下室挖出五只瓶子及盘、杯等古物；他把瓶子先后卖出，其中三只被一名带现金上门的人以每只十万元收走。罗永没有灵异知识，也不知道魂瓶及老宅异常已引发事件，只将万德路派人接他到大京市、预付劳务费的安排视为可疑骗局。（第510章、第513章）

## 外貌·性格

杨间初见罗永时，见其外表像普通工地工人，满脸皱纹、黑瘦朴素，没有可见灵异异常；但他的面貌与敲门鬼照片中的老人约有六七分相似。罗永本人不知这种相似的意义，因陌生人以高额报酬把自己带到私人会所、又有保镖看守而始终不安，反复要求离开。他担心对方先以好处诱骗、再骗回钱财；万德路追加五万元且当面付款后，他才同意再等两个小时。面对杨间出示证件和配枪的询问，他因误以为自己涉案而发抖、说话结巴，但仍如实回忆祖宅、古董和祖父的有限信息。（第510章、第511章、第513章）

## 经历

#### 第510-511章·被请至大京市等候问话

万德路为追查魂瓶来源，付给罗永五万元出差费并将其带到大京市私人会所。罗永原只答应停留半天，等待近一天后认为受骗，试图离开；两名保镖以已收劳务费和老板尚在路上为由阻拦。万德路赶到后说明只想询问祖宅、承诺不伤害他，又加付五万元，才令他留在休息室。杨间此时首次看清其脸，因其与敲门鬼高度相似而震惊，却未立刻询问，决定先由万德路安置。（第510章、第511章）

#### 第513章·交代祖宅古董与祖父线索

下午约两点，杨间以大昌市负责人身份出示证件、把配枪放在桌上，要求罗永配合调查。罗永确认照片中的老宅原属其祖父、自己只是继承者，并承认地下室共挖出五只瓶子，而非先前线索中的一只：两只分别以一万、三万元卖出，余下三只由陌生人携现金按十万元一只收购，未留下联系方式。谈及祖父，他只记得幼年生病时，长期住在村外老宅的祖父曾穿黑色旧式长衫送来医药钱；此后祖父离奇不见，父亲也从未说明其下落。杨间展示敲门鬼照片后，罗永凭同一件长衫确认照片中人为祖父。祖父身份、失踪原因、收购者身份及其余瓶子的去向均不详。（第513章）

#### 第514章·获安置费并离开会所

杨间因收到朋友圈将对自己动手的警报而结束问询，明确告诉罗永古董一事牵涉其失踪祖父、要求他对当天谈话保密；若他人追问，只能称古董均已卖掉、祖父之事一概不知。杨间又让他在受逼问时把安排推到自己名下，并留下联系方式，理由是罗永及祖宅仍是追查真相的唯一线索之一。（第514章）

罗永仍担忧自己是否可走，杨间让万德路给他另找住处并支付二百万元安家费，劝其暂时不要回老家、改住别处。罗永离开包厢时既受惊又因意外获得巨款而高兴，但原文没有写他是否接受住处、是否实际迁离、后续是否遭人追问或再向杨间提供线索。（第514章）
