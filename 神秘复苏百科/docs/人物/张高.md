# 张高

## 简介

张高是中山市负责特殊事件联络的工作人员。第331章，杨间乘总部安排的直升机抵达中山市一处在建公园的临时指挥区域时，张高主动表明身份，说明在杨间执行任务期间由自己负责联络与配合。他不是驭鬼者，原文未交代其完整编制、此前经历、能调度的具体资源及是否知晓记忆篡改鬼的详情。（第331章）

## 经历

#### 第331章·接应杨间与安排熊文文

张高在临时指挥区接到杨间后，确认熊文文已提前十分钟抵达，可随时参与行动，并带杨间进入帐篷会面。他转达总部要求：若中山市事件无法解决，应尽可能保证熊文文活着离开；杨间只承诺尽力，不接受必须完整带回的保证。张高随后按杨间要求准备皮卡、封锁凯撒大酒店及附近区域、限制人员出入，并继续进行上空人头的身份识别，承诺发现身份不明死者即通知杨间。具体封锁效果、人脸识别结果及其之后是否仍与杨间联络均不详。（第331章、第332章）

## 关系

- [杨间](杨间.md)：张高作为中山市联络员接应杨间，并按其要求调配车辆、警戒和信息核查。（第331章、第332章）
- [熊文文](熊文文.md)：张高向杨间转达总部优先保障熊文文撤离的安排，并确认其已抵达临时指挥区。（第332章）

#### 第340章·接应童倩抢救

熊文文按杨间要求联系张高后，张高将救护车和医护人员带至凯撒大酒店警戒线外。杨间交出被换脸、仍有微弱生命特征的童倩，医护随即确认其心率过低、失血、内脏出血和多处骨折，安排输血、手术准备与转运。张高认出失联三日的童倩，对其获救感到意外，并为杨间和熊文文安排临时住处。童倩抢救结果及张高后续协调情况均不详。（第340章）

#### 第379章·确认复活并整理事件资料

童倩恢复联络后，张高向总部确认其确已复活，使刘小雨得以向赵建国报告该消息。随后他整理人头气球事件遇害者资料，发现死者都曾入住凯撒大酒店；杨间、童倩、冯全据此认为酒店内灵异与事件有关，但仍无法确定真正源头。张高是否参与凯撒大酒店封锁、向沈良完成何种交接及后续是否继续担任联络员，原文未写。（第379章）
