# 王信

## 简介

王信是岛国除灵社的工作人员、组长，真实证件姓名为长谷川信。他约四十余岁，身材矮瘦、光头、脸有深皱纹，初见给人威严凶悍之感，却能在杨间面前迅速转为热情谦卑。除灵社为岛国对应总部的灵异组织；王信此行以王野死亡赔礼为表面理由，实际意在请杨间处理神户市的鬼敲门事件。（原文回跳第644章）

## 性格与处事

王信在强者面前极能忍让。杨间为验证鬼瓷效果，以无头鬼影扭断其手臂、最终让手臂碎裂，他仍不敢当面反抗，次日带五亿元补偿再次登门，并向社长请示接受高额委托条件。这种屈从并不等于软弱：离开大厦后，他立刻掌掴曾拔枪的下属，并命其回国后自尽以平息杨间不愿再见此人的要求，显示其对下属冷酷且重视组织利益。（原文回跳第644章、原文回跳第645章）

## 经历

#### 原文回跳第644章·携鬼瓷赔礼并接受试验

王信带数名除灵社人员至尚通大厦，承认王野是社团第二组组长，称王野得罪杨间、己方愿道歉。他带来鬼瓷作为赔礼，说明该物可保护使用者抵挡厉鬼袭击。杨间要求以王信本人试验，王信被迫以身份牌作媒介启动鬼瓷，承受无头鬼影侵入；鬼瓷替其分担灵异伤害，但小人手臂破裂时，王信手臂也随之碎裂。杨间取走媒介后终止保护，收下残缺鬼瓷，王信则带伤离开。（原文回跳第644章）

#### 原文回跳第645章·鬼敲门情报谈判失败

王信为取得杨间掌握的鬼敲门情报，率属下跪地求助；杨间提出再给一尊鬼瓷，王信以无权决定为由改开十五亿元注资条件。杨间认为其仍在试探和占便宜，终止谈判。王信离开时接受“下次不想见拔枪属下”的要求，随后命该属下自行处理后果。王信未得到情报，但仍保留重新请示和继续拉拢杨间的意图。（原文回跳第645章）

#### 原文回跳第648章·接受高额委托并安排神户行动

王信以五亿元先行赔礼重返尚通大厦，转达社长“只要能做到均可答应”的授权。杨间提出一百吨黄金，王信请示后报出五十吨黄金、五尊鬼瓷的组合，并承诺再给三名成员协助；杨间接受，约定其前往神户市处理鬼敲门事件。王信随后紧急安排专机、三名协助者与翻译，并把自己前途押在杨间行动上。酬劳交割、协助者身份和行动结果均不详。（原文回跳第648章）

#### 原文回跳第649-650章·交付订金并启动行动

王信随杨间乘专机抵达神户附近机场，确认二十五吨黄金和三件鬼瓷已作为订金送往大昌市，又在机场为行动提供三名协助者名单。田野擅闯机场后，王信奔走报告、转交卫星定位手机和档案；杨间以鬼域带惠子前往集合点，王信留在机场等待定位信息。订金是否已由杨间公司签收、他如何处理田野和行动后续不详。（原文回跳第649章、原文回跳第650章）
