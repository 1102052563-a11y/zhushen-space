# 第一部分 · 必讀

## 1. 這是什麼遊戲

這是一款 **AI 驅動的文字角色扮演遊戲（RPG）**，題材是「**輪迴樂園 · 無限流**」——你是一名被神秘「主神空間」選中的**契約者**，會被不斷送進各種各樣的世界（原著世界、影視世界、你自訂的世界……）裡冒險、變強、活下去，再回到主神空間休整，如此輪迴。

它和普通遊戲最大的不同：**故事正文是 AI 即時寫出來的**。你不是點按鈕觸發固定劇情，而是**用文字描述你想做什麼**，AI 扮演「導演 + 旁白 + 所有配角」，把接下來發生的事寫成一段小說給你看。

**核心玩法循環**（記住這張圖就夠了）：

```
你輸入行動（想做什麼）
        ↓
AI 寫出這一段的故事正文（戰鬥/對話/探索……）
        ↓
後台「演化系統」自動讀正文，幫你更新資料
（你的屬性、物品、技能、NPC 好感、任務進度、地圖……全部自動記帳）
        ↓
你看著新劇情，繼續輸入下一步行動 → 循環
```

你幾乎不用手動填數值。**你負責「演」，AI 和系統負責「記帳」。** 你砍死一個敵人、撿到一把劍、學會一招——只要正文裡寫了，系統就會自動把劍放進你背包、把技能記到你身上。

**幾個反覆出現、必須先認識的名詞：**

| 名詞 | 通俗解釋 |
|---|---|
| **契約者** | 就是「你」這個身分。被主神空間選中、能穿越各世界的人。 |
| **主神空間** | 你的大本營 / 老家。世界之間休息、買東西、強化裝備的地方（也叫「輪迴樂園」）。 |
| **階位** | 你的實力等級（類似修仙裡的境界）。從低到高：一階 → 二階 …… 越往上越強，每階還分 Lv.1~10。 |
| **天賦** | 你的天生資質 / 特殊能力，評級 D → C → B → A → S → SS → SSS（越靠後越逆天）。 |
| **六維屬性** | 力量 / 敏捷 / 體質 / 智力 / 魅力 / 幸運。角色的基礎數值，AI 會根據劇情給你成長。 |
| **樂園幣 / 魂幣** | 兩種貨幣。樂園幣是日常通用幣；魂幣（靈魂錢幣）更值錢，`1 魂幣 ≈ 15 萬樂園幣`。 |
| **演化** | 「正文寫完後，後台自動更新資料」這一整套機制的統稱。物品演化、NPC 演化…都是它。 |

> 💡 **一句話總結**：你寫行動 → AI 寫故事 → 系統自動更新一切資料。剩下的功能都是圍繞這個循環，讓世界更豐富、角色更立體。

## 2. 第一步：設定 AI 介面（最重要）

> ⚠️ **這是全篇最重要的一章。不設定好介面，遊戲一個字都生成不出來。** 請務必讀完 2.1 ~ 2.5。

### 2.1 為什麼必須先配介面

遊戲本身**不自帶 AI**。它只是一個「殼」，負責把你的行動、世界設定、記憶等打包好，發給**你自己的 AI 大模型**去生成故事。所以你需要先告訴遊戲：**「用哪個 AI、怎麼連上它」**。

你需要準備的東西（三選一，看你用什麼）：
- 一個 **OpenAI / Google Gemini / Claude / DeepSeek** 等官方或中轉的 **API Key（金鑰）**；
- 或任何**相容 OpenAI 格式**的中轉站 / 反代位址 + Key；
- 或本機跑的模型（如 Ollama / LM Studio，相容 OpenAI 介面即可）。

> 如果你完全沒有 Key：需要自行去對應平台註冊申請。本指南只講**遊戲裡怎麼填**，不涉及去哪買 Key。

### 2.2 新增你的第一條介面（API 介面庫）

遊戲裡所有介面都集中在一個叫「**API 介面庫**」的地方統一管理。步驟：

1. 點右側導覽列最下面的 **⚙ 設定**。
2. 進入 **綜合設定**（第一個大項）。
3. 找到 **「API 介面庫」** 板塊，點 **「+ 新增介面」**。
4. 點開這條新介面，把下面幾欄填好：

| 欄位 | 填什麼 | 例子 |
|---|---|---|
| **名稱** | 隨便取個你認得的名字 | `我的GPT` / `Gemini主力` |
| **介面位址** | 你的 API 根位址（一般以 `/v1` 結尾） | `https://api.openai.com/v1` |
| **API Key** | 你的金鑰 | `sk-xxxxxxxx...` |
| **模型 ID** | 要用的模型名 | `gpt-4o` / `gemini-2.5-pro` / `claude-sonnet-4` |
| **溫度 / Top-P / Max Tokens** | 生成參數，新手用預設即可 | 溫度 0.7~1.0；Max Tokens 建議 ≥ 4096 |

> 🔑 **你的 Key 只存在你自己的瀏覽器裡**，不會上傳到任何伺服器，可以放心填。
>
> 📌 **模型 ID 不知道填什麼？** 填好位址和 Key 後，點旁邊的 **「重新整理模型」** 按鈕，遊戲會自動從你的介面拉取可用模型清單，直接下拉選一個即可。

### 2.3 測試連線與重新整理模型

填完後，點這條介面裡的 **「🔌 測試連線」**：
- 顯示 **✅** = 連通成功，可以用了。
- 顯示 **❌** = 有問題，看提示訊息（常見：位址寫錯、Key 失效、被地區限制）。

**「重新整理模型」**：自動拉取該介面下所有可用模型，幫你確認模型 ID 沒寫錯。

> 別忘了：介面預設是「**已啟用**」狀態。如果顯示「已停用」，點一下「啟用」。**沒啟用 / 沒填位址 / 沒填 Key 的介面不會被真正呼叫。**

### 2.4 連不上？閘道代理與一鍵閘道

有時候會遇到「**本機能用、部署到線上就 403**」，或者瀏覽器**跨網域（CORS）**報錯、或者你的中轉是 `http` 裸 IP。這類問題可以用「**閘道代理**」解決：

- 在介面的編輯區，點 **「🛡 經閘道代理」**：遊戲會把請求包一層，透過它自帶的 Cloudflare 反代轉發，繞開跨網域 / IP 鎖定 / HTTPS 問題。想還原就點「↩ 取消代理」。
- 如果你要用 **Google AI Studio** 或 **Vertex** 的 Gemini，點介面庫底部的 **「⚡ 一鍵填入 AI Studio / Vertex 閘道」**，會自動幫你建好兩條設定好的介面，你只要填自己的 AI Studio Key 即可（Vertex 需要在伺服器端設定好金鑰，進階玩家用）。
- 還有個「**本機閘道位址**」選填框：給會折騰的玩家用（本機跑一個 worker 中轉，讓中轉看到的是你自己家的 IP）。新手可忽略。

### 2.5 整合路由（介面路由）怎麼設定

> 這是本作介面系統的**精髓**，也是你特別想了解的功能。理解了它，你就能讓遊戲「**一個介面掛了自動換下一個，永不中斷**」，還能給不同功能分配不同的模型。

**它解決什麼問題？**
- 你可能有好幾個 Key（比如兩個 GPT 中轉 + 一個 Gemini），想讓它們**輪流用 / 一個壞了自動切換**（fallback）。
- 你可能想**給不同功能用不同模型**：比如「寫正文」用最貴最強的模型，「更新物品資料」用便宜快速的模型省錢。

**「介面路由」就是幹這個的**：為每一個功能，指定**一串**介面，按你排的**優先順序從上往下呼叫**——排最上面的先用，它失敗了自動切下一條，再失敗切再下一條。

**怎麼配（以「正文生成」為例）：**

1. 先確保你已經在 **2.2** 裡往「API 介面庫」加好了至少一條（最好兩三條）介面。
2. 進 **設定 → 正文生成**，最上方會看到一個 **「⚡ 介面路由」** 的框。
3. 點框裡的下拉選單 **「+ 新增介面到路由…」**，把你想用的介面一條條加進來。
4. 加進來後，每條右邊有 **↑ ↓** 箭頭調順序（**上面的先呼叫**），有 **✕** 移除。
5. 排好即可，**自動生效**。以後這個功能就會：先試第 1 條 → 失敗自動試第 2 條 → 以此類推。

**幾個要點：**
- **留空 = 回退**：如果某功能的介面路由一條都沒選，它會退回去用該功能下方的「單獨設定 / 共用正文 API」。所以你也可以完全不碰路由，只配一個主介面，遊戲照樣能跑——路由是「進階增強」，不是「必填」。
- **只有「可用」的介面才算數**：路由裡的介面必須**同時滿足**「已啟用 + 有介面位址 + 有 Key」才會被真正呼叫。如果某條缺東西，介面會用紅字標出來「缺金鑰·不生效」之類，並提醒你去補。
- **指向已刪除介面的紀錄**：如果你把介面庫裡某條刪了，但它還留在某個路由裡，介面會提示「有 X 條指向已刪除介面」，點「清理」一鍵清掉即可。

> ✅ **新手最省事的推薦配法**：介面庫裡加 2 條能用的介面 → 在「正文生成」的介面路由裡把這 2 條都加上、把最強的排第一。其它功能先不管（留空自動複用）。這樣既有主力又有備胎，一條掛了自動頂上。

### 2.6 各個功能都可以單獨掛介面

本作把 AI 呼叫拆成了很多**獨立功能**，每個都能配自己的介面路由（都在各自的設定頁頂部，用法和 2.5 一樣）。你**不需要**每個都配，絕大多數留空即可（自動複用正文介面）。列出來只是讓你知道「原來這些都能單獨換模型」：

- **正文（text）** —— 寫主線故事，最重要，建議配最強模型。
- **世界（world）** —— 生成新世界 / 樂園的設定。
- **物品 / 主角 / NPC / 勢力 / 領地 / 冒險團 / 萬族 / 雜項** —— 各種「演化」階段，負責更新資料。想省錢可以給它們配便宜快速的模型。
- **記憶（memory）/ 敘事記憶（nm）** —— 整理長期劇情記憶。
- **頻道（channel）** —— 公共頻道、私訊裡那些虛擬契約者的發言。
- **生圖正文（image_story_llm）** —— 決定正文裡哪段配圖。

> 💡 典型省錢玩法：**正文**掛 Claude / GPT-4 級別的強模型；**物品 / NPC / 雜項**等演化統統掛便宜的 `gemini-flash` / `deepseek` 之類。既保證故事品質，又不心疼 token。

### 2.7 各功能推薦用什麼模型（省錢又好用）

不同功能對模型的要求天差地別：寫故事要文筆最強的，更新資料用便宜快的就夠。**核心原則：好鋼用在刀口上——文筆靠強模型，記帳靠便宜模型。** 下表是一份穩妥的搭配建議（模型名只是舉例，用你手上同檔次的即可）：

| 功能 | 推薦檔次 | 舉例 | 為什麼這樣配 |
|---|---|---|---|
| **正文生成（text）** | 🔥 最強 | Claude Sonnet / Opus、GPT-4o、`gemini-2.5-pro`、DeepSeek | 沉浸感和文筆全靠它，最值得下血本 |
| **世界選擇（world）** | 💪 強（Pro 級） | **`gemini-2.5-pro`** | 生成世界觀設定量大、要嚴謹自洽，Pro 級更穩、不容易崩設定 |
| **各種演化**（物品 / 主角 / NPC / 勢力 / 領地 / 冒險團 / 萬族 / 雜項） | ⚡ 快而便宜（Flash 級） | **`gemini-2.5-flash`**、DeepSeek | 只是「照著正文更新資料」，不需要文筆，Flash **又快又省 token**，完全夠用 |
| **記憶 / 敘事記憶（memory / nm）** | ⚡ Flash 級 | `gemini-2.5-flash` | 抽取、壓縮長期事實，重邏輯不重文采 |
| **頻道 / 私訊（channel）** | ⚡ Flash 級 | `gemini-2.5-flash` | 一堆虛擬契約者的閒聊和報價，量大、求快 |
| **細綱 / 劇情指導 / 資料庫推進** | 💪 中～強 | `gemini-2.5-pro` 或與正文同款 | 直接影響正文的品質和走向，別配太弱的 |
| **生圖正文（image_story_llm）** | ⚡ Flash 級 | `gemini-2.5-flash` | 只是判斷「哪一段該配圖」，很輕 |

**怎麼落地？** 就是用 **2.5 / 2.6** 講的「介面路由」：
1. 在 API 介面庫裡把你的強模型、Flash 模型各建一條介面。
2. 進「正文生成」的介面路由 → 掛強模型；進「世界選擇」→ 掛 `gemini-2.5-pro`；進各個「演化」功能（物品/NPC/勢力…）的介面路由 → 掛 Flash。
3. 懶人版：只在「正文生成」掛強模型，其餘全部留空——它們會自動複用正文模型，也能玩，只是沒那麼省錢。

> 💡 **一句話**：**演化用 Flash（快 + 省），世界選擇用 2.5 Pro（穩），正文用你能負擔的最強模型（好看）。**
>
> ⚠️ 注意：這裡說的是「寫文字/更新資料」的**語言模型**。真正**畫圖**的「🖼 生圖」用的是另一套畫圖服務（NAI / OpenAI / Gemini 畫圖 / ComfyUI），在生圖設定裡單獨選，和上面這張表無關。

## 3. 第二步：建立角色，進入遊戲

配好介面後，就能開局了。

**開始畫面（封面）**：一張全螢幕封面圖，上面有三個**透明熱區**（看不到明顯按鈕，直接點對應位置即可）：
- **開始** —— 新建角色開新遊戲。
- **讀檔** —— 載入之前的存檔。
- **設定** —— 打開設定（就是剛才配介面的地方）。

**角色創建**分兩步（填寫 → 確認）：

1. **選難度** —— 難度決定你的**初始屬性點總數**（簡單 50 點，越難越少，最難「無用之人」只有 10 點）。新手建議**簡單**。
2. **選所屬樂園** —— 你的「陣營 / 出身樂園」，影響一些身分設定。
3. **填基本資料** —— 姓名、年齡、外觀、背景故事等（都可自訂，AI 會參考這些來寫你）。
4. **分配六維** —— 把難度給的屬性點分到力量/敏捷/體質/智力/魅力/幸運上（每項一般 ≤ 10，總和 ≤ 難度點數）。
5. **寫天賦** —— 你的開局天賦（天生能力），有固定格式的編輯器，可即時預覽。
6.（進階）還能選「**攜帶開局物品**」和「**開局隨行隨從**」，讓 AI 幫你把它們也生成好。

點**確認**後，遊戲會自動發出**開場白**——也就是你的第一段故事正文。開場白內容可以在設定裡自訂。之後你就正式進入遊戲主畫面了。

**主畫面三欄佈局：**
- **左欄（主角欄）**：你的頭像、姓名、階位、六維、HP/EP 血條、狀態。**點上面的欄位可以直接編輯。**
- **中間（正文區）**：故事在這裡一段段顯示。最下面是**輸入框**，你在這裡打字行動。
- **右欄（導覽列）**：一長串功能圖示（裝備、背包、技能、NPC、勢力……）。**所有功能都從這裡進。**

> 📱 手機版：右欄會收進一個抽屜按鈕裡，點開即是同一份導覽。

## 4. 第三步：一個回合怎麼玩（核心循環）

這是你每天在做的事，學會這個就算入門了。

**① 輸入你的行動。** 在中間底部的輸入框裡，用文字描述你想做什麼。可以很簡單（`我推開門走進去`），也可以很詳細（帶對話、帶心理、帶具體招式）。寫得越具體，AI 演得越貼合你的想法。

**② 傳送。** 點傳送（或 Enter 鍵，具體看設定）。AI 開始「思考」，然後**串流**地把這一段故事一個字一個字寫出來。

**③ 看正文 + 後台自動演化。** 正文寫完後，後台的「演化系統」會自動讀這段故事，更新你的資料：撿到的東西進背包、受的傷扣血、NPC 的態度變化、任務推進……這些都是**自動**的，你能在狀態列看到「物品演化中…」「NPC 演化中…」等進度提示。

**④ 繼續下一步。** 循環往復。

### 介面底部按鈕逐個說明（截圖裡那些小按鈕）

正文區下方從上到下一共**四條**，每條都有用，逐個說清楚：

**第 1 條 · 📋 狀態命令列**（顯示為 `📋 本回合狀態命令 · N 回合`）
- **點它展開/收起「選擇世界 · 結算任務」欄**——想去新世界、或給當前任務世界做**任務結算**，都從這裡進。
- 這條列平時還兼當**演化進度條**：物品管理、主角/NPC/勢力/領地/冒險團/萬族/雜項演化、記憶回溯、生圖……正在跑哪一步，會在這條上**即時滾動**顯示（如「物品管理處理中…」「NPC 演化處理中…」）。看到它在轉，就說明系統正在後台幫你記帳。
- 最右邊（生成過一次後才出現）有幾個**除錯按鈕**：`🛠 開發者`（看這回合實際發給 AI 的完整提示詞）、`🧠 查看注入記憶`、`查看輸入`、`查看返回`。想知道 AI 到底收到了什麼、排查問題時很有用。

**第 2 條 · 本回合操作列**（傳送過至少一回合後才出現；手機上收在「⚙ 本回合操作 ▾」裡）
- **⟳ 重新生成** —— 對本回合正文不滿意，撤銷本回合（含所有演化）並用同一條輸入**重寫**（會重新整理頁面，需確認）。
- **↩ 回退上一回合** —— 撤銷本回合的正文和所有演化，**回到上一回合結束時的狀態**（重新整理頁面，需確認）。這才是安全的「撤銷」。
- **♻ 重算變數** —— 正文滿意、但**某項資料錯了 / 漏了**？點它彈出「重 ROLL 選單」，可以**只重跑某一個演化**（物品、主角屬性、NPC、勢力、領地、冒險團、萬族、任務、記憶、生圖），也可以選「全部變數」整體重算。其它資料不動。
- **🔊 朗讀 / ⏹ 停止朗讀**（旁邊的 ⚙ 是語音設定）—— 朗讀本回合正文（詳見第 10 章 TTS）。
- **⛔ 停止生成** —— 一鍵叫停**正在跑的全部演化和生圖**（卡住、或不想等時用）。
  > 🧭 **鐵律**：⟳ / ↩ / ♻ 這三個依賴「本回合剛生成的原文」，**一旦重新整理頁面或讀檔就失效**（原文只存在記憶體裡）。要用趁早用，別先關頁面。

**第 3 條 · 📌 前置提示詞欄**（點 `▶ 📌 前置提示詞` 展開）
- 你寫在這裡的話會**每回合都注入到故事最深處、權重最高**，用來長期約束 AI（比如「保持黑暗風格」「別讓配角圍著我轉」「一律用第二人稱寫」）。**一直生效，不用每回合重打。**
- 你的**場外操作**（加點 / 合成 / 強化 / 花錢等）也會自動追加在它下面，作為「本回合場外」通報，讓正文和你在面板裡做的事保持一致、不重複結算。

**第 4 條 · 輸入框那一列**（從左到右）
- 🔴 **紅色 ↺ 按鈕 = 清空對話** —— ⚠️ **小心**：它會清掉當前螢幕上顯示的對話訊息，讓正文區變空。**它不是「回退」、也不刪存檔**，但清空後當前對話就不顯示了。想安全撤銷請用第 2 條的 **↩ 回退**，別誤點這個紅按鈕。
- **輸入框** —— 打字的地方（`在此輸入你的行動…`）。Enter 傳送、`Shift+Enter` 換行（可在設定裡改成「Enter 換行、按鈕傳送」）。
- **↵** —— 手動插入換行（和 `Shift+Enter` 一樣，手機上方便）。
- ⏩ **綠色·推進劇情** —— **不用打字**，讓劇情順著當前局勢、鋪墊和伏筆**自然往前走一拍**。懶得想怎麼寫、或只想推進節奏時超好用（你輸入框裡的草稿會保留）。
- 🔁 **循環自動推進** —— 自動連推好幾拍（次數 / 間隔在設定裡調）；你一傳送新內容它就自動停。掛機「看戲」用。
- ▶ **神色·傳送** —— 傳送你輸入的行動（生成中會變成轉圈圖示）。

### 骰子、人稱、三種「推進」模式

- **🎲 搖骰子** —— 需要「命運判定」時（能不能撬開鎖、說服能不能成功），手動搖骰，結果連同你的屬性/技能修正一起注入給 AI，讓成敗更有隨機性和張力。可切「確定性引擎」或「AI 裁判」兩種模式。
- **敘事人稱** —— 一個開關，切換正文用第一 / 第二 / 第三人稱來寫你。
- **正文前的三種「推進」模式**（三選一，互斥，可在設定裡開）：
  - **細綱** —— 生成正文前，先彈一個「本回合劇情大綱」讓你**預覽、編輯、確認**，再據此寫正文。想掌控走向時很有用。
  - **劇情指導** —— 生成正文前，AI 先給一段「本回合怎麼寫更好」的建議（可連網搜原作設定），讓切入更合理。
  - **資料庫推進** —— 進階的規劃層，先「召回 + 推進」再寫正文，劇情連貫性更強。

### 頂部狀態列

頂欄會顯示**雙時間**（主神空間的「輪迴曆」時間 + 當前任務世界的時間）、當前**世界名**、**天氣**（任務世界裡還有下雨/下雪等視覺效果）。

### 快速跳轉：命令面板

按 **⌘K / Ctrl+K**（或點頂欄的 🔍）打開**命令面板**，輸入關鍵詞（支援拼音、首字母、別名）就能模糊搜尋、一鍵跳到任何功能面板，不用在長長的導覽列裡找。**強烈推薦養成用它的習慣。**

## 5. 角色與成長

### 主角面板（左欄身分檔案）
你的核心資訊面板，就在螢幕**左側**。包含姓名、等級、階位、稱號、職業、身分、契約者 ID、**生物強度**（如 `T3·勇士`，衡量種族/血脈的天生強度）、六維、外觀、位置、HP/EP。**點任意欄位即可編輯。** 物理攻擊/防禦等**衍生屬性**由六維+等級+裝備自動算出，不用你填。

### ⚔ 裝備
管理你**穿在身上**的裝備。打開是一個裝備槽面板：主武器、副手、頭/上身/下身/手/腳/肩/腰等防具部位、飾品、法寶等。點槽位可以穿脫。裝備會即時影響你的攻防和六維。裝備卡右上角的 **`+N`** 是強化等級（見 6.2）。

### 🎒 儲存空間（背包）
你的**倉庫**——所有沒穿在身上的物品都在這。每件物品有品質顏色、效果、數量、詞綴、評分等。底部有**貨幣欄**（樂園幣 / 魂幣 / 技能點 / 黃金技能點）和一個**貨幣兌換器**（`1 魂幣 = 15 萬樂園幣`，可雙向換）。在背包裡點物品的「使用」會把「使用XX」自動填進主輸入框，方便你發出去。

> 物品分為裝備、消耗品、材料、重要物品等類別。它們的名字、效果、價格全部由 AI 根據劇情生成，並遵循一套「顏色品質定價」體系（白→綠→藍→紫→淡金→金→暗金→……越稀有越貴）。

### ✨ 技能
你會的所有**技能**。每個技能有：名稱、等級（Lv.1→Lv.10→Lv.EX 滿級）、類型、**品級**（7 檔：普通→精良→稀有→史詩→傳說→奧義→極境）、消耗、目標、效果、傷害、描述等。技能靠**劇情證據**成長——正文裡你練成/領悟了什麼，系統就更新什麼，**不消耗任何點數**。也可以手動編輯。

### 🎖 稱號
你一路**掙來的名號牆**（如「屠龍者」「競技場冠軍」）。每個稱號有品級、來源、效果。**同時只能佩戴 1 個**（佩戴的那個會加成並被 AI 記住）。稱號只增不減、多多益善——贏來的榮譽都會保留。

### 🏆 成就
類似其它遊戲的成就系統（**僅主角有**）。達成特定條件解鎖，有隱藏成就（帶🔒）。純紀念性質，不影響數值。

### 🛠 副職業
非戰鬥的**生活 / 製造手藝**（煉金、鍛造、烹飪……名字全自訂）。有兩層熟練度：總熟練度（新手→……→宗師，五檔）和每個配方的進度（0~100）。做東西時熟練度會累積。**僅主角可用。**

### 🌳 技能樹
**徑向星圖**式的職業技能樹 🌳。花「潛能點」解鎖節點，每個節點給你灌一個技能或天賦。內建了「劍士」等範本，也能讓 AI 幫你生成新樹，還能手動加點、匯出分享（`.tree.json`）。**僅主角。**

### 🎴 體系（流派裝備欄）
一個「**替補席**」模型的裝備/技能設定台 🎴。你可以把不同的技能、天賦、裝備存成多套「體系/流派」範本，一鍵切換出戰組合（像換 build）。切換是「零丟失」的——沒上場的會退到替補席，不會消失。也能上傳到創意工坊分享。

### HP / EP 與自訂能量條
- **HP（生命）/ EP（精力/法力）** 的上限由你的**六維按係數算出**（預設「體質×20 → HP，智力×15 → EP」，可在血條面板的 6×2 矩陣裡自訂係數）。當前血量**忠於正文**，不會自動回滿。
- 你還能自訂血條的**稱呼和皮膚**（比如把 HP 叫「血池」、EP 叫「靈力」，換成血怒條外觀）。
- **⚡ 自訂能量條**：在 HP/EP 之外，自建額外資源條（怒氣 / 墮落值 / 靈力……）。它可以純劇情展示，也能被技能消耗 / 當施放門檻，甚至在戰鬥中累積。**僅主角。**

## 6. 製作、強化與樂園設施

### 🎡 樂園設施（總入口）
點右側導覽 **🎡 樂園設施**，會彈出一個小選單，聚合了主神空間裡的功能與娛樂設施：**裝備強化、技能升級、歡愉宮（成人向，可關）、競技場、賭場**。下面分別介紹。

> ⚠️ 強化、技能升級等設施**只能在主神空間（樂園/專屬房間）裡用**，在任務世界裡會置灰。

### ⚒ 裝備強化
把裝備從 **+0 強到 +16** ⚒。這是一套**純前端、確定性**的玩法（不花 AI token）：
- 每級有固定成功率（+1→+2 是 100%，越往上越難，+15→+16 只有約 10%）。
- **失敗有懲罰**，且分等級：+0~+2 必成；+3~+6 失敗降 1 級；+7~+9 失敗歸零（回 +0）；**+10 以上失敗裝備會碎掉消失（爆裝）**。
- 有**保底機制**：真的爆裝了會累加「墊子」，墊滿必成一次。可以拿便宜裝故意去危險區刷保底。
- 有**看板娘（老闆）**：不同看板娘有不同的成功率加成、費用倍率、性格，還有**分階段立繪**（越強化立繪越……你懂的）。點立繪她會根據實況**吐槽**。
- **收尾刷裝備**：一輪強化結束時，AI 會根據你漲的等級重算裝備的攻防、詞綴、外觀、評分（每跨 4 級 +1 詞綴）。

### 🔼 技能升級
類似裝備強化，但升的是**技能** 🔼。用「技能點」給技能升級（跨 10 級會加新效果），用「黃金技能點」升技能**品級**（質變）。同樣在樂園設施裡。

### 🧰 合成工坊
**全世界可用**的合成台 🧰。資料驅動的 10 大門類，前端確定性護欄（擲品質、鎖品級、算詞綴預算）+ AI 填風味。有配方圖鑑。想自己造裝備/道具時用。

### 🎁 開箱
**全世界可用** 🎁。選一個寶箱，開出與其品級相稱的物品。玩法類似合成，鎖定產出檔位後由 AI 決定具體開出什麼。

### 🎰 賭場（輪迴賭坊）
純前端確定性的**賭博小遊戲** 🎰：猜大小、轉盤、21 點、翻倍梯子、角鬥場共 5 種玩法，外加福袋扭蛋。用樂園幣或魂幣下注，有荷官立繪。圖個樂子、也能搏一把。

### 🏟 競技場
**階位分支的排行榜** 🏟。選一個名次去挑戰，勝利就取代對方名次，前 100 名有獎勵。複用真實戰鬥系統結算，榜單有記憶。一階是普通競技場，五階起有「強者爭霸戰」。

### 💗 歡愉宮（成人向 · 預設可關）
成人向內容 💗。從樂園設施進入，選看板娘 → 進包廂，有情慾值驅動的分階段立繪和快捷動作。**介意的話可以在設定裡直接關掉**，關了導覽裡就不顯示。

## 7. 戰鬥系統

### ⚔️ 戰鬥
點右側導覽 **⚔️ 戰鬥** 發起一場**結構化戰鬥**（區別於純正文打鬥）。選擇對手 NPC 後進入戰鬥面板，用**標籤式**的前端結算 + 本機敵人 AI 打回合制戰鬥，打完由 AI 潤色成戰報。支援 buff/減益/護盾/冷卻/蓄力大招/領域陣法等機制。戰鬥結果（傷害、勝負）只進資料、由 AI 寫成好看的戰報。

> 連線房間裡，戰鬥由房主發起。

### 🎲 搖骰子判定
不是獨立面板，而是**輸入框旁的工具**（見第 4 章）。用於需要運氣/技巧檢定的場合：手動搖骰 → 結果連同你的屬性/技能/裝備修正一起注入給 AI → AI 據此判定成敗。可切「確定性引擎」或「AI 裁判」兩種模式。判定結果會以卡片形式顯示在對話裡。

## 8. NPC 與社交

### 📇 NPC
你遇到的所有**角色檔案庫** 📇。每個 NPC 有詳細的 11 欄資料：性別、階位、性格、狀態、對你的稱呼、背景、內心想法、關係網、好感、外觀、動機等。NPC 會**自動演化**——即使不在你面前，後台也會按身分推進他們自己的生活、關係、成長。**離場 ≠ 死亡**（離場的角色被歸檔，但被正文點名時還能救回記憶）。死亡的 NPC 不再顯示。

> NPC 詳情裡可以看/編輯他的裝備、技能、生平，還能給他生成頭像立繪、發起私聊。

### 👥 好友
把 NPC 加為好友後 👥，他會**每回合參與 NPC 演化**（關係更活躍）。加好友的入口有三處：NPC 面板點 ☆、頻道裡點「⭐加好友」、私訊裡點「⭐加好友」。加好友時若對方還沒建檔，系統會自動補全他的檔案和隨身物品。

### ✉ 私訊
和單個 NPC 一對一**私聊** ✉。除了聊天，還能**交易**：買 / 賣 / 索取 / 贈予 / 換物，AI 會扮演對方報價、討價還價，成交是確定性結算（東西真的會轉移）。契約者、隨從、寵物可私訊；土著、召喚物不可。

### 📡 頻道（契約者廣場）
七個**公共頻道** 📡，模擬一個熱鬧的契約者社群：
- **氛圍閒聊** —— AI 模擬一堆虛擬契約者發帖。
- **交易** —— 別人掛的出售帖你可一鍵買；你也能掛求購單讓 AI 報價，確定性成交。
- **系統商店** —— 買賣物品。
- **主角發言** —— 你自己發帖上牆，AI 扮演的契約者會陸續回覆，你能定向「↩回覆」。
- **臨時組隊** —— 加入別人的組隊帖、或邀請別人，組成臨時隊伍。

### 🤝 隊伍
你當前的**臨時隊伍** 🤝（從頻道組來的隊友）。隊友是臨時的 NPC，世界結束會自動解散，也可以「轉正」成正式夥伴。和「冒險團」（9 節，你自己的固定團隊）不是一回事。

## 9. 世界與探索

### 🌍 世界選擇 / 進入樂園
在設定或相關入口裡，可以**生成一個新世界**去闖（AI 會讀你的階位、忠於你給的世界設定卡來生成世界觀）。進入後世界觀注入正文最深處，你就在新世界裡冒險；離開時 AI 會總結，同名世界再進可選擇「繼承」之前的進度或重置。

### 🗺 世界記錄
記錄你**闖過的世界誌** 🗺。每個世界的世界觀、你在裡面的經歷、離世總結都在這。

### 📖 世界百科
當前所在世界的**百科/設定庫** 📖，方便你查這個世界的背景、勢力、規則等資訊。

### 📚 輪迴 WIKI
「輪迴樂園」這個大宇宙本身的**世界觀百科** 📚（一個獨立的知識站）。介紹主神空間、階位體系、貨幣、術語等設定。**漸進式防劇透**——只展示你已經讀到 / 經歷過的部分。

### 🏛 勢力
世界裡的**各方勢力/組織**檔案 🏛（門派、家族、公司、國家……）。像 NPC 一樣會自動演化：勢力對你的好感、目標、地盤會隨劇情變。分「當前世界」「非當前世界」「已覆滅」三類。換世界時舊勢力會自動清出當前世界。

### 🏯 領地
你在主神空間的**個人基地** 🏯（跨世界保留、絕對安全）。等級隨你的階位走，可以蓋各種**全自訂的建築**，有建設進度條和被動產出（產出會自動進領地倉庫）。你可以把 NPC 安排進領地當成員。

### 🛡 冒險團
**你自己創立的固定團隊** 🛡（唯一一個，你是團長）。有等級（E~SSS）、經驗、活躍度雙計量、成員、團隊效果、考核試煉。經驗滿且活躍度夠會自動晉級，高階晉級要過考核。只有正文裡明確「建團」了才會成立。NPC 也能發起加入你的團。

### 🌌 萬族
**宇宙背景層** 🌌——頭頂那片「七樂園 / 萬族 / 深淵」的大勢力格局，會自轉演化。屬於烘托世界宏大感的背景設定層，獨立演化。

### 🕳 深淵地牢（墮落流 Roguelike）
一個**墮落流的 Roguelike 玩法** 🕳：進主神空間的多層地牢，奪取「原罪物」，以「腐蝕值」為核心資源，越深越危險。有三層 Build 和多種加成卡流派。喜歡肉鴿 / 暗黑風的可以玩。

## 10. 資訊、記憶與體驗工具

### 🔍 回合洞察
每回合的**精簡資料快照** 🔍：主角六維/狀態/技能/稱號 + 全部 NPC 的好感/狀態/動機 + 勢力好感/目標/地盤……捲動保留 14 份。面板能顯示「最新 vs 上一份」的**結構化差異（diff）**，讓你一眼看出這回合資料變了什麼。純本機比對，不花 AI。

### 🧾 審計
查看「**綜合對帳糾錯**」的結果 🧾。系統在每回合正文後會自動跑一次對帳：拿「套用後的真實資料 + 最近兩回合正文」逐項檢查，糾正演化時漏掉/更新錯的資料。這個面板讓你看它做了什麼。

### 📋 任務
**任務 / 世界大事 / 總結**面板 📋。顯示你的主線/支線任務、世界發生的大事件、分段劇情總結，以及「雙時間」等世界級雜項資訊。

### 🧠 記憶
長期劇情**記憶中樞** 🧠。這裡其實涉及三套配合的機制：
- **敘事記憶**（設定裡開）：用關鍵詞從歷史劇情裡召回「相關」的片段注入正文，讓 AI 記得很久以前的事（無需向量，輕量）。
- **生平壓縮**：每個角色的經歷攢多了會自動讓 AI 壓縮成「長期事實」，防止遺忘。
- **結構化檔案召回**（預設開）：每回合自動把主角 + 在場/相關 NPC 的完整檔案塞給寫正文的 AI，保證它不會「失憶」寫錯你的資料。

### 向量資料庫（語義記憶）
進階記憶方案（設定 → 向量記憶）：用**語義向量**檢索長期記憶，比關鍵詞更聰明（需要一個 embedding 介面）。還內建了「原著小說 + 世界書」的向量庫——每回合按語義相關度把原著片段/世界書條目注入，讓劇情更貼合原作。**開啟後接管召回。**

### 🖼 生圖
給遊戲配圖 🖼（設定 → 生圖設定）。支援多種服務（NAI / OpenAI / Gemini / ComfyUI / 自訂），分三條獨立線：**角色肖像、裝備圖、正文插圖**，可各自選服務和自動開關。開了自動後，外觀變化會自動重繪肖像、正文關鍵處會自動插圖。圖片存在瀏覽器的 IndexedDB 裡（不佔存檔體積）。

### 🔊 語音（TTS）
**朗讀**功能 🔊。支援本機 Web Speech 或 Edge-TTS 閘道（免 Key 的微軟神經語音）。可以給每個 NPC 分配不同音色、旁白和台詞分離朗讀，正文裡還有行內 🔊 按鈕逐句唸。

### 🎨 介面外觀美化
（設定 → 介面外觀美化）**護眼與換主題** 🎨：8 套主題皮膚、護眼色調、暗角、正文字體（含「霞鶩文楷」）、字號行距調節。看久了眼睛累就來這調。

### 多語言
遊戲支援**簡體 / 繁體 / 英文 / 越南語**切換（設定內）。這是執行時翻譯層，正文本身不會被翻譯。

## 11. 連線與社群

> 這一類大多需要連網服務支援，是**可選的社交擴充**。單機玩不碰它們也完全 OK。

### 🌐 連線
**線上組隊連線** 🌐：房主權威 + 中繼伺服器同步回合。隊友零門檻加入（不用自己的 Key），可以觀戰戰鬥、控制角色、共享連線存檔。

### 💬 聊天室
一個**獨立的全域即時聊天室** 💬（真人玩家）。可以分享你的技能/天賦/裝備/NPC 卡，支援 Discord 登入、未讀紅點、自訂頭像和氣泡皮膚。

### 🛒 交易行
**全域交易行** 🛒：掛牌賣東西、公開還價看板。看板型（不自動成交，避免丟物），帶託管機制。

### 🏪 產業
你可以經營的**產業** 🏪（商店 / 娼館 / 鐵匠鋪），供其他玩家來消費。有獨立店面頁、AI 生成的貨品、立繪輪播、雙幣記帳、伺服器端收入帳本和待領橫幅。

### 🏰 公會
**非同步玩家公會** 🏰（原「家族」）。建立/加入公會、名冊軍銜、捐獻換福利、公會倉庫、編年史、週任務、貢獻榜、公會戰、據點建設等。帳號級、跨存檔。

### 🆘 助戰
**助戰大廳** 🆘：把你的主角或 NPC 上傳成公開卡，別人可以邀請你的卡物化成他的臨時隊友（反之亦然）。有排行榜。

### 🏆 世界競技場
**上傳參賽卡的線上競技場** 🏆：伺服器端權威裁判（按戰力 + 隨機種子的純函數判定），自動戰鬥，佔位取代排名。每帳號可放 3 張卡。（和 6 節樂園設施裡的單機「競技場」不同，這個是**連線**的。）

### ⏱ 遊玩時長
統計你的**線上遊玩時長** ⏱並上榜（凡 Discord 登入者）。前端心跳累計「頁面活躍時長」，伺服器端防刷。

### 🪦 紀念豐碑
**跨存檔的英靈殿** 🪦：把你過往的主角銘刻成「英靈卡」，之後能召喚入隊。跨存檔、雲端同步。

### 🏦 帳戶倉庫
**跨存檔的保險箱** 🏦（Discord 雲端）：把物品存進去，換個存檔也能取出來。
> ⚠️ 存取時會帶**完整物品快照**，防止「塞進去再拿出來詞綴全沒了」。

### 🧩 創意工坊
**社群共享中心** 🧩：上傳/下載別人分享的技能樹、體系、看板娘、世界卡等 10 類內容。

## 12. 存檔與資料管理

### 💾 存檔
**多存檔系統** 💾（存在瀏覽器的 IndexedDB 裡）。一個存檔 = 你全部資料 + 對話歷史 + 圖片的完整快照。
- 支援**存檔 / 讀檔 / 重新命名 / 刪除 / 匯出 / 匯入**。
- **讀檔靠重新整理頁面**恢復（這是正常機制，別慌）。
- 有**自動存檔**（每回合後延時約 20 秒自動覆蓋存一份）。
- 「**新遊戲**」會清空進度但保留你的設定（介面、預設等）。
- 存檔管理裡還能 **📖 匯出小說**：把你的對話歷史清洗成分章的小說 TXT。

### ⚙ 變數管理（演化功能啟動台）
（設定 → 變數管理）這是所有「**演化功能的中心啟動台**」🧬。物品、主角、NPC、勢力、領地、冒險團、萬族、雜項、記憶、生圖、裝備強化……每一套演化系統的開關、頻率、預設、專用介面，都在這裡分門別類地管理。**新手預設不用動**——演化開箱即用；等你想精調「多久更新一次 NPC」「給物品演化換個便宜模型」時再進來。

### 全域設定匯出 / 匯入
在變數管理頁最下方的「**設定備份 · 遷移**」欄，可以一鍵**打包匯出**你所有的設定（介面、預設、世界書、正則、生圖、向量、範本……**不含遊戲進度**），換裝置時一鍵匯入。適合備份和搬家。

## 13. 新手推薦流程 & 常見問題

### 推薦上手流程

1. **配介面**（第 2 章）：介面庫加 1~2 條能用的介面 → 測試 ✅ → 在「正文生成」介面路由裡掛上。
2. **建角色**（第 3 章）：選**簡單**難度 → 分好六維 → 寫個天賦 → 確認。
3. **玩幾個回合**（第 4 章）：隨便輸入行動，熟悉「你演、AI 寫、系統記帳」的節奏。看看左欄資料怎麼變、背包怎麼多出東西。
4. **逛一圈功能**：打開 📇NPC、🎒背包、✨技能、📋任務 看看系統都幫你記了些什麼。
5. **想強化 / 娛樂**：回主神空間，進 🎡樂園設施 玩強化、賭場、競技場。
6. **想要圖 / 語音**：去設定開 🖼生圖、🔊語音。
7. 覺得 AI「記性差」或資料老出錯：去設定開 **🧠敘事記憶**、檢查 **🧾審計**，或用 **♻重算變數** 修。

### 常見問題（FAQ）

**Q：點了傳送，一直轉圈 / 報錯生成不出來？**
A：99% 是介面問題。回**設定 → 綜合設定 → API 介面庫**，點你的介面「🔌 測試連線」看是否 ✅。紅字提示會告訴你是位址錯、Key 失效還是被限制。也確認「正文生成」的介面路由裡掛的介面是「可用」狀態（已啟用 + 有位址 + 有 Key）。

**Q：資料沒更新 / 更新錯了（比如砍死了敵人血沒扣、撿的東西沒進包）？**
A：用輸入框旁的 **♻ 重算變數**，選對應的那一項（物品 / 主角 / NPC…）單獨重跑一次。注意這功能重新整理頁面後就失效，要趁本回合用。也可以去 **🧾審計** 看系統自己糾了哪些。

**Q：AI 忘記很久以前的劇情 / 記錯我的設定？**
A：開 **設定 → 敘事記憶**（關鍵詞召回長期記憶）。想更強可開 **向量記憶**。另外「結構化檔案召回」預設就開著，會保證 AI 記得你**當前**的資料。

**Q：花了 5000 卻記成花了 10000 之類的數值錯亂？**
A：本作有「場外操作自動通報」和「前置提示詞」機制來緩解。實在不對就用 ♻ 重算變數修，或在**前置提示詞**裡寫明約束。

**Q：遊戲卡頓 / 打字有延遲？**
A：可以在設定裡**關掉全息卡特效**、**調低歷史樓層限制**。圖片多了也會拖慢——用背包/設定裡的清理工具清掉孤兒圖片。

**Q：想徹底重來？**
A：存檔面板裡「新遊戲」會清進度保設定。想連設定都清空，需要清掉瀏覽器裡以 `drpg-` 開頭的本機儲存（進階操作，慎用）。

**Q：換了電腦，設定和存檔怎麼帶走？**
A：**設定**用變數管理頁底部的「全域設定匯出/匯入」。**存檔**用存檔面板的「匯出/匯入」。（登入 Discord 的話，雲端存檔 / 帳戶倉庫也能跨裝置。）
