# 第一部分 · 必读

## 1. 这是什么游戏

这是一款 **AI 驱动的文字角色扮演游戏（RPG）**，题材是「**轮回乐园 · 无限流**」——你是一名被神秘「主神空间」选中的**契约者**，会被不断送进各种各样的世界（原著世界、影视世界、你自定义的世界……）里冒险、变强、活下去，再回到主神空间休整，如此轮回。

它和普通游戏最大的不同：**故事正文是 AI 实时写出来的**。你不是点按钮触发固定剧情，而是**用文字描述你想做什么**，AI 扮演「导演 + 旁白 + 所有配角」，把接下来发生的事写成一段小说给你看。

**核心玩法循环**（记住这张图就够了）：

```
你输入行动（想做什么）
        ↓
AI 写出这一段的故事正文（战斗/对话/探索……）
        ↓
后台「演化系统」自动读正文，帮你更新数据
（你的属性、物品、技能、NPC 好感、任务进度、地图……全部自动记账）
        ↓
你看着新剧情，继续输入下一步行动 → 循环
```

你几乎不用手动填数值。**你负责「演」，AI 和系统负责「记账」。** 你砍死一个敌人、捡到一把剑、学会一招——只要正文里写了，系统就会自动把剑放进你背包、把技能记到你身上。

**几个反复出现、必须先认识的名词：**

| 名词 | 通俗解释 |
|---|---|
| **契约者** | 就是「你」这个身份。被主神空间选中、能穿越各世界的人。 |
| **主神空间** | 你的大本营 / 老家。世界之间休息、买东西、强化装备的地方（也叫「轮回乐园」）。 |
| **阶位** | 你的实力等级（类似修仙里的境界）。从低到高：一阶 → 二阶 …… 越往上越强，每阶还分 Lv.1~10。 |
| **天赋** | 你的天生资质 / 特殊能力，评级 D → C → B → A → S → SS → SSS（越靠后越逆天）。 |
| **六维属性** | 力量 / 敏捷 / 体质 / 智力 / 魅力 / 幸运。角色的基础数值，AI 会根据剧情给你成长。 |
| **乐园币 / 魂币** | 两种货币。乐园币是日常通用币；魂币（灵魂钱币）更值钱，`1 魂币 ≈ 15 万乐园币`。 |
| **演化** | 「正文写完后，后台自动更新数据」这一整套机制的统称。物品演化、NPC 演化…都是它。 |

> 💡 **一句话总结**：你写行动 → AI 写故事 → 系统自动更新一切数据。剩下的功能都是围绕这个循环，让世界更丰富、角色更立体。

## 2. 第一步：配置 AI 接口（最重要）

> ⚠️ **这是全篇最重要的一章。不配好接口，游戏一个字都生成不出来。** 请务必读完 2.1 ~ 2.5。

### 2.1 为什么必须先配接口

游戏本身**不自带 AI**。它只是一个「壳」，负责把你的行动、世界设定、记忆等打包好，发给**你自己的 AI 大模型**去生成故事。所以你需要先告诉游戏：**「用哪个 AI、怎么连上它」**。

你需要准备的东西（三选一，看你用什么）：
- 一个 **OpenAI / Google Gemini / Claude / DeepSeek** 等官方或中转的 **API Key（密钥）**；
- 或任何**兼容 OpenAI 格式**的中转站 / 反代地址 + Key；
- 或本地跑的模型（如 Ollama / LM Studio，兼容 OpenAI 接口即可）。

> 如果你完全没有 Key：需要自行去对应平台注册申请。本指南只讲**游戏里怎么填**，不涉及去哪买 Key。

### 2.2 添加你的第一条接口（API 接口库）

游戏里所有接口都集中在一个叫「**API 接口库**」的地方统一管理。步骤：

1. 点右侧导航栏最下面的 **⚙ 设置**。
2. 进入 **综合设置**（第一个大项）。
3. 找到 **「API 接口库」** 板块，点 **「+ 添加接口」**。
4. 点开这条新接口，把下面几栏填好：

| 栏位 | 填什么 | 例子 |
|---|---|---|
| **名称** | 随便起个你认得的名字 | `我的GPT` / `Gemini主力` |
| **接口地址** | 你的 API 根地址（一般以 `/v1` 结尾） | `https://api.openai.com/v1` |
| **API Key** | 你的密钥 | `sk-xxxxxxxx...` |
| **模型 ID** | 要用的模型名 | `gpt-4o` / `gemini-2.5-pro` / `claude-sonnet-4` |
| **温度 / Top-P / Max Tokens** | 生成参数，新手用默认即可 | 温度 0.7~1.0；Max Tokens 建议 ≥ 4096 |

> 🔑 **你的 Key 只存在你自己的浏览器里**，不会上传到任何服务器，可以放心填。
>
> 📌 **模型 ID 不知道填什么？** 填好地址和 Key 后，点旁边的 **「刷新模型」** 按钮，游戏会自动从你的接口拉取可用模型列表，直接下拉选一个即可。

### 2.3 测试连接与刷新模型

填完后，点这条接口里的 **「🔌 测试连接」**：
- 显示 **✅** = 连通成功，可以用了。
- 显示 **❌** = 有问题，看提示信息（常见：地址写错、Key 失效、被地区限制）。

**「刷新模型」**：自动拉取该接口下所有可用模型，帮你确认模型 ID 没写错。

> 别忘了：接口默认是「**已启用**」状态。如果显示「已禁用」，点一下「启用」。**没启用 / 没填地址 / 没填 Key 的接口不会被真正调用。**

### 2.4 连不上？网关代理与一键网关

有时候会遇到「**本地能用、部署到线上就 403**」，或者浏览器**跨域（CORS）**报错、或者你的中转是 `http` 裸 IP。这类问题可以用「**网关代理**」解决：

- 在接口的编辑区，点 **「🛡 经网关代理」**：游戏会把请求包一层，通过它自带的 Cloudflare 反代转发，绕开跨域 / IP 锁定 / HTTPS 问题。想还原就点「↩ 取消代理」。
- 如果你要用 **Google AI Studio** 或 **Vertex** 的 Gemini，点接口库底部的 **「⚡ 一键填入 AI Studio / Vertex 网关」**，会自动帮你建好两条配置好的接口，你只要填自己的 AI Studio Key 即可（Vertex 需要在服务端配好密钥，进阶玩家用）。
- 还有个「**本地网关地址**」选填框：给会折腾的玩家用（本机跑一个 worker 中转，让中转看到的是你自己家的 IP）。新手可忽略。

### 2.5 集成路由（接口路由）怎么设置

> 这是本作接口系统的**精髓**，也是你特别想了解的功能。理解了它，你就能让游戏「**一个接口挂了自动换下一个，永不中断**」，还能给不同功能分配不同的模型。

**它解决什么问题？**
- 你可能有好几个 Key（比如两个 GPT 中转 + 一个 Gemini），想让它们**轮流用 / 一个坏了自动切换**（fallback）。
- 你可能想**给不同功能用不同模型**：比如「写正文」用最贵最强的模型，「更新物品数据」用便宜快速的模型省钱。

**「接口路由」就是干这个的**：为每一个功能，指定**一串**接口，按你排的**优先级从上往下调用**——排最上面的先用，它失败了自动切下一条，再失败切再下一条。

**怎么配（以「正文生成」为例）：**

1. 先确保你已经在 **2.2** 里往「API 接口库」加好了至少一条（最好两三条）接口。
2. 进 **设置 → 正文生成**，最上方会看到一个 **「⚡ 接口路由」** 的框。
3. 点框里的下拉菜单 **「+ 添加接口到路由…」**，把你想用的接口一条条加进来。
4. 加进来后，每条右边有 **↑ ↓** 箭头调顺序（**上面的先调用**），有 **✕** 移除。
5. 排好即可，**自动生效**。以后这个功能就会：先试第 1 条 → 失败自动试第 2 条 → 以此类推。

**几个要点：**
- **留空 = 回退**：如果某功能的接口路由一条都没选，它会退回去用该功能下方的「单独配置 / 共用正文 API」。所以你也可以完全不碰路由，只配一个主接口，游戏照样能跑——路由是「进阶增强」，不是「必填」。
- **只有「可用」的接口才算数**：路由里的接口必须**同时满足**「已启用 + 有接口地址 + 有 Key」才会被真正调用。如果某条缺东西，界面会用红字标出来「缺密钥·不生效」之类，并提醒你去补。
- **指向已删除接口的记录**：如果你把接口库里某条删了，但它还留在某个路由里，界面会提示「有 X 条指向已删除接口」，点「清理」一键清掉即可。

> ✅ **新手最省事的推荐配法**：接口库里加 2 条能用的接口 → 在「正文生成」的接口路由里把这 2 条都加上、把最强的排第一。其它功能先不管（留空自动复用）。这样既有主力又有备胎，一条挂了自动顶上。

### 2.6 各个功能都可以单独挂接口

本作把 AI 调用拆成了很多**独立功能**，每个都能配自己的接口路由（都在各自的设置页顶部，用法和 2.5 一样）。你**不需要**每个都配，绝大多数留空即可（自动复用正文接口）。列出来只是让你知道「原来这些都能单独换模型」：

- **正文（text）** —— 写主线故事，最重要，建议配最强模型。
- **世界（world）** —— 生成新世界 / 乐园的设定。
- **物品 / 主角 / NPC / 势力 / 领地 / 冒险团 / 万族 / 杂项** —— 各种「演化」阶段，负责更新数据。想省钱可以给它们配便宜快速的模型。
- **记忆（memory）/ 叙事记忆（nm）** —— 整理长期剧情记忆。
- **频道（channel）** —— 公共频道、私信里那些虚拟契约者的发言。
- **生图正文（image_story_llm）** —— 决定正文里哪段配图。

> 💡 典型省钱玩法：**正文**挂 Claude / GPT-4 级别的强模型；**物品 / NPC / 杂项**等演化统统挂便宜的 `gemini-flash` / `deepseek` 之类。既保证故事质量，又不心疼 token。

## 3. 第二步：创建角色，进入游戏

配好接口后，就能开局了。

**开始界面（封面）**：一张全屏封面图，上面有三个**透明热区**（看不到明显按钮，直接点对应位置即可）：
- **开始** —— 新建角色开新游戏。
- **读档** —— 载入之前的存档。
- **设置** —— 打开设置（就是刚才配接口的地方）。

**角色创建**分两步（填写 → 确认）：

1. **选难度** —— 难度决定你的**初始属性点总数**（简单 50 点，越难越少，最难「无用之人」只有 10 点）。新手建议**简单**。
2. **选所属乐园** —— 你的「阵营 / 出身乐园」，影响一些身份设定。
3. **填基本信息** —— 姓名、年龄、外观、背景故事等（都可自定义，AI 会参考这些来写你）。
4. **分配六维** —— 把难度给的属性点分到力量/敏捷/体质/智力/魅力/幸运上（每项一般 ≤ 10，总和 ≤ 难度点数）。
5. **写天赋** —— 你的开局天赋（天生能力），有固定格式的编辑器，可实时预览。
6.（进阶）还能选「**携带开局物品**」和「**开局随行随从**」，让 AI 帮你把它们也生成好。

点**确认**后，游戏会自动发出**开场白**——也就是你的第一段故事正文。开场白内容可以在设置里自定义。之后你就正式进入游戏主界面了。

**主界面三栏布局：**
- **左栏（主角栏）**：你的头像、姓名、阶位、六维、HP/EP 血条、状态。**点上面的字段可以直接编辑。**
- **中间（正文区）**：故事在这里一段段显示。最下面是**输入框**，你在这里打字行动。
- **右栏（导航栏）**：一长串功能图标（装备、背包、技能、NPC、势力……）。**所有功能都从这里进。**

> 📱 手机端：右栏会收进一个抽屉按钮里，点开即是同一份导航。

## 4. 第三步：一个回合怎么玩（核心循环）

这是你每天在做的事，学会这个就算入门了。

**① 输入你的行动。** 在中间底部的输入框里，用文字描述你想做什么。可以很简单（`我推开门走进去`），也可以很详细（带对话、带心理、带具体招式）。写得越具体，AI 演得越贴合你的想法。

**② 发送。** 点发送（或回车，具体看设置）。AI 开始「思考」，然后**流式**地把这一段故事一个字一个字写出来。

**③ 看正文 + 后台自动演化。** 正文写完后，后台的「演化系统」会自动读这段故事，更新你的数据：捡到的东西进背包、受的伤扣血、NPC 的态度变化、任务推进……这些都是**自动**的，你能在状态栏看到「物品演化中…」「NPC 演化中…」等进度提示。

**④ 继续下一步。** 循环往复。

### 输入框周边的实用按钮

- **⟳ 重新生成** —— 对这一回合的正文不满意？点它**重写本回合**（会先撤销本回合再用同样的输入重发，数据不会叠加）。
- **↩ 回退** —— 想撤销上一回合、回到发送前的状态，点它（会刷新页面并恢复）。
- **♻ 重算变量（重 ROLL 菜单）** —— 正文满意、但**某项数据更新错了 / 漏了**？点它弹出菜单，可以**只重跑某一个演化**（物品、主角属性、NPC、势力、领地、冒险团、万族、任务、记忆、生图……），其它数据不动。也可以选「全部变量」整体重算。
  > 🧭 **铁律**：这三个工具（回退 / 重新生成 / 重算变量）依赖「本回合刚生成的原文」，**刷新页面或读档后就用不了了**（原文只存在内存里）。要用趁早用。

- **前置提示词** —— 输入框上方有个可展开的小栏。你写在这里的话会**每回合都注入到故事最深处**，用来长期约束 AI（比如「保持黑暗风格」「不要让配角围着我转」）。它一直生效，不用每回合重打。
- **叙事人称** —— 一个开关，切换正文用第一/第二/第三人称来写你。
- **🎲 摇骰子** —— 需要「命运判定」时（比如能不能撬开锁），可以手动摇骰子，结果会注入给 AI，让成败更有随机性和张力。
- **正文前的三种「推进」模式**（三选一，互斥，可在设置里开）：
  - **细纲** —— 生成正文前，先弹一个「本回合剧情大纲」让你**预览、编辑、确认**，再据此写正文。想掌控走向时很有用。
  - **剧情指导** —— 生成正文前，AI 先给一段「本回合怎么写更好」的建议（可联网搜原作设定），让切入更合理。
  - **数据库推进** —— 进阶的规划层，先「召回 + 推进」再写正文，剧情连贯性更强。

### 顶部状态栏

顶栏会显示**双时间**（主神空间的「轮回历」时间 + 当前任务世界的时间）、当前**世界名**、**天气**（任务世界里还有下雨/下雪等视觉效果）。

### 快速跳转：命令面板

按 **⌘K / Ctrl+K**（或点顶栏的 🔍）打开**命令面板**，输入关键词（支持拼音、首字母、别名）就能模糊搜索、一键跳到任何功能面板，不用在长长的导航栏里找。**强烈推荐养成用它的习惯。**

# 第二部分 · 功能详解

> 下面按类别介绍**每一个**功能。用不到的可以跳过，需要时再回来查。每个功能都标了**从哪进**（右侧导航的图标 + 名字）。

## 5. 角色与成长

### 主角面板（左栏身份档案）
你的核心信息面板，就在屏幕**左侧**。包含姓名、等级、阶位、称号、职业、身份、契约者 ID、**生物强度**（如 `T3·勇士`，衡量种族/血脉的天生强度）、六维、外观、位置、HP/EP。**点任意字段即可编辑。** 物理攻击/防御等**衍生属性**由六维+等级+装备自动算出，不用你填。

### ⚔ 装备
管理你**穿在身上**的装备。打开是一个装备槽面板：主武器、副手、头/上身/下身/手/脚/肩/腰等防具部位、饰品、法宝等。点槽位可以穿脱。装备会实时影响你的攻防和六维。装备卡右上角的 **`+N`** 是强化等级（见 6.2）。

### 🎒 储存空间（背包）
你的**仓库**——所有没穿在身上的物品都在这。每件物品有品质颜色、效果、数量、词缀、评分等。底部有**货币栏**（乐园币 / 魂币 / 技能点 / 黄金技能点）和一个**货币兑换器**（`1 魂币 = 15 万乐园币`，可双向换）。在背包里点物品的「使用」会把「使用XX」自动填进主输入框，方便你发出去。

> 物品分为装备、消耗品、材料、重要物品等类别。它们的名字、效果、价格全部由 AI 根据剧情生成，并遵循一套「颜色品质定价」体系（白→绿→蓝→紫→淡金→金→暗金→……越稀有越贵）。

### ✨ 技能
你会的所有**技能**。每个技能有：名称、等级（Lv.1→Lv.10→Lv.EX 满级）、类型、**品级**（7 档：普通→精良→稀有→史诗→传说→奥义→极境）、消耗、目标、效果、伤害、描述等。技能靠**剧情证据**成长——正文里你练成/领悟了什么，系统就更新什么，**不消耗任何点数**。也可以手动编辑。

### 🎖 称号
你一路**挣来的名号墙**（如「屠龙者」「竞技场冠军」）。每个称号有品级、来源、效果。**同时只能佩戴 1 个**（佩戴的那个会加成并被 AI 记住）。称号只增不减、多多益善——赢来的荣誉都会保留。

### 🏆 成就
类似其它游戏的成就系统（**仅主角有**）。达成特定条件解锁，有隐藏成就（带🔒）。纯纪念性质，不影响数值。

### 🛠 副职业
非战斗的**生活 / 制造手艺**（炼金、锻造、烹饪……名字全自定义）。有两层熟练度：总熟练度（新手→……→宗师，五档）和每个配方的进度（0~100）。做东西时熟练度会累积。**仅主角可用。**

### 🌳 技能树
**径向星图**式的职业技能树 🌳。花「潜能点」解锁节点，每个节点给你灌一个技能或天赋。内置了「剑士」等模板，也能让 AI 帮你生成新树，还能手动加点、导出分享（`.tree.json`）。**仅主角。**

### 🎴 体系（流派装备栏）
一个「**替补席**」模型的装备/技能配置台 🎴。你可以把不同的技能、天赋、装备存成多套「体系/流派」模板，一键切换出战组合（像换 build）。切换是「零丢失」的——没上场的会退到替补席，不会消失。也能上传到创意工坊分享。

### HP / EP 与自定义能量条
- **HP（生命）/ EP（精力/法力）** 的上限由你的**六维按系数算出**（默认「体质×20 → HP，智力×15 → EP」，可在血条面板的 6×2 矩阵里自定义系数）。当前血量**忠于正文**，不会自动回满。
- 你还能自定义血条的**称呼和皮肤**（比如把 HP 叫「血池」、EP 叫「灵力」，换成血怒条外观）。
- **⚡ 自定义能量条**：在 HP/EP 之外，自建额外资源条（怒气 / 堕落值 / 灵力……）。它可以纯剧情展示，也能被技能消耗 / 当施放门槛，甚至在战斗中累积。**仅主角。**

## 6. 制作、强化与乐园设施

### 🎡 乐园设施（总入口）
点右侧导航 **🎡 乐园设施**，会弹出一个小菜单，聚合了主神空间里的功能与娱乐设施：**装备强化、技能升级、欢愉宫（成人向，可关）、竞技场、赌场**。下面分别介绍。

> ⚠️ 强化、技能升级等设施**只能在主神空间（乐园/专属房间）里用**，在任务世界里会置灰。

### ⚒ 装备强化
把装备从 **+0 强到 +16** ⚒。这是一套**纯前端、确定性**的玩法（不花 AI token）：
- 每级有固定成功率（+1→+2 是 100%，越往上越难，+15→+16 只有约 10%）。
- **失败有惩罚**，且分等级：+0~+2 必成；+3~+6 失败降 1 级；+7~+9 失败归零（回 +0）；**+10 以上失败装备会碎掉消失（爆装）**。
- 有**保底机制**：真的爆装了会累加「垫子」，垫满必成一次。可以拿便宜装故意去危险区刷保底。
- 有**看板娘（老板）**：不同看板娘有不同的成功率加成、费用倍率、性格，还有**分阶段立绘**（越强化立绘越……你懂的）。点立绘她会根据实况**吐槽**。
- **收尾刷装备**：一轮强化结束时，AI 会根据你涨的等级重算装备的攻防、词缀、外观、评分（每跨 4 级 +1 词缀）。

### 🔼 技能升级
类似装备强化，但升的是**技能** 🔼。用「技能点」给技能升级（跨 10 级会加新效果），用「黄金技能点」升技能**品级**（质变）。同样在乐园设施里。

### 🧰 合成工坊
**全世界可用**的合成台 🧰。数据驱动的 10 大门类，前端确定性护栏（掷品质、锁品级、算词缀预算）+ AI 填风味。有配方图鉴。想自己造装备/道具时用。

### 🎁 开箱
**全世界可用** 🎁。选一个宝箱，开出与其品级相称的物品。玩法类似合成，锁定产出档位后由 AI 决定具体开出什么。

### 🎰 赌场（轮回赌坊）
纯前端确定性的**赌博小游戏** 🎰：猜大小、转盘、21 点、翻倍梯子、角斗场共 5 种玩法，外加福袋扭蛋。用乐园币或魂币下注，有荷官立绘。图个乐子、也能搏一把。

### 🏟 竞技场
**阶位分支的排行榜** 🏟。选一个名次去挑战，胜利就取代对方名次，前 100 名有奖励。复用真实战斗系统结算，榜单有记忆。一阶是普通竞技场，五阶起有「强者争霸战」。

### 💗 欢愉宫（成人向 · 默认可关）
成人向内容 💗。从乐园设施进入，选看板娘 → 进包间，有情欲值驱动的分阶段立绘和快捷动作。**介意的话可以在设置里直接关掉**，关了导航里就不显示。

## 7. 战斗系统

### ⚔️ 战斗
点右侧导航 **⚔️ 战斗** 发起一场**结构化战斗**（区别于纯正文打斗）。选择对手 NPC 后进入战斗面板，用**标签式**的前端结算 + 本地敌人 AI 打回合制战斗，打完由 AI 润色成战报。支持 buff/减益/护盾/冷却/蓄力大招/领域阵法等机制。战斗结果（伤害、胜负）只进数据、由 AI 写成好看的战报。

> 联机房间里，战斗由房主发起。

### 🎲 摇骰子判定
不是独立面板，而是**输入框旁的工具**（见第 4 章）。用于需要运气/技巧检定的场合：手动摇骰 → 结果连同你的属性/技能/装备修正一起注入给 AI → AI 据此判定成败。可切「确定性引擎」或「AI 裁判」两种模式。判定结果会以卡片形式显示在对话里。

## 8. NPC 与社交

### 📇 NPC
你遇到的所有**角色档案库** 📇。每个 NPC 有详细的 11 栏资料：性别、阶位、性格、状态、对你的称呼、背景、内心想法、关系网、好感、外观、动机等。NPC 会**自动演化**——即使不在你面前，后台也会按身份推进他们自己的生活、关系、成长。**离场 ≠ 死亡**（离场的角色被归档，但被正文点名时还能救回记忆）。死亡的 NPC 不再显示。

> NPC 详情里可以看/编辑他的装备、技能、生平，还能给他生成头像立绘、发起私聊。

### 👥 好友
把 NPC 加为好友后 👥，他会**每回合参与 NPC 演化**（关系更活跃）。加好友的入口有三处：NPC 面板点 ☆、频道里点「⭐加好友」、私信里点「⭐加好友」。加好友时若对方还没建档，系统会自动补全他的档案和随身物品。

### ✉ 私信
和单个 NPC 一对一**私聊** ✉。除了聊天，还能**交易**：买 / 卖 / 索取 / 赠予 / 换物，AI 会扮演对方报价、讨价还价，成交是确定性结算（东西真的会转移）。契约者、随从、宠物可私信；土著、召唤物不可。

### 📡 频道（契约者广场）
七个**公共频道** 📡，模拟一个热闹的契约者社区：
- **氛围闲聊** —— AI 模拟一堆虚拟契约者发帖。
- **交易** —— 别人挂的出售帖你可一键买；你也能挂求购单让 AI 报价，确定性成交。
- **系统商店** —— 买卖物品。
- **主角发言** —— 你自己发帖上墙，AI 扮演的契约者会陆续回复，你能定向「↩回复」。
- **临时组队** —— 加入别人的组队帖、或邀请别人，组成临时队伍。

### 🤝 队伍
你当前的**临时队伍** 🤝（从频道组来的队友）。队友是临时的 NPC，世界结束会自动解散，也可以「转正」成正式伙伴。和「冒险团」（9 节，你自己的固定团队）不是一回事。

## 9. 世界与探索

### 🌍 世界选择 / 进入乐园
在设置或相关入口里，可以**生成一个新世界**去闯（AI 会读你的阶位、忠于你给的世界设定卡来生成世界观）。进入后世界观注入正文最深处，你就在新世界里冒险；离开时 AI 会总结，同名世界再进可选择「继承」之前的进度或重置。

### 🗺 世界记录
记录你**闯过的世界志** 🗺。每个世界的世界观、你在里面的经历、离世总结都在这。

### 📖 世界百科
当前所在世界的**百科/设定库** 📖，方便你查这个世界的背景、势力、规则等信息。

### 📚 轮回 WIKI
「轮回乐园」这个大宇宙本身的**世界观百科** 📚（一个独立的知识站）。介绍主神空间、阶位体系、货币、术语等设定。**渐进式防剧透**——只展示你已经读到 / 经历过的部分。

### 🏛 势力
世界里的**各方势力/组织**档案 🏛（门派、家族、公司、国家……）。像 NPC 一样会自动演化：势力对你的好感、目标、地盘会随剧情变。分「当前世界」「非当前世界」「已覆灭」三类。换世界时旧势力会自动清出当前世界。

### 🏯 领地
你在主神空间的**个人基地** 🏯（跨世界保留、绝对安全）。等级随你的阶位走，可以盖各种**全自定义的建筑**，有建设进度条和被动产出（产出会自动进领地仓库）。你可以把 NPC 安排进领地当成员。

### 🛡 冒险团
**你自己创立的固定团队** 🛡（唯一一个，你是团长）。有等级（E~SSS）、经验、活跃度双计量、成员、团队效果、考核试炼。经验满且活跃度够会自动晋级，高阶晋级要过考核。只有正文里明确「建团」了才会成立。NPC 也能发起加入你的团。

### 🌌 万族
**宇宙背景层** 🌌——头顶那片「七乐园 / 万族 / 深渊」的大势力格局，会自转演化。属于烘托世界宏大感的背景设定层，独立演化。

### 🕳 深渊地牢（堕落流 Roguelike）
一个**堕落流的 Roguelike 玩法** 🕳：进主神空间的多层地牢，夺取「原罪物」，以「腐蚀值」为核心资源，越深越危险。有三层 Build 和多种加成卡流派。喜欢肉鸽 / 暗黑风的可以玩。

## 10. 信息、记忆与体验工具

### 🔍 回合洞察
每回合的**精简数据快照** 🔍：主角六维/状态/技能/称号 + 全部 NPC 的好感/状态/动机 + 势力好感/目标/地盘……滚动保留 14 份。面板能显示「最新 vs 上一份」的**结构化差异（diff）**，让你一眼看出这回合数据变了什么。纯本地对比，不花 AI。

### 🧾 审计
查看「**综合对账纠错**」的结果 🧾。系统在每回合正文后会自动跑一次对账：拿「应用后的真实数据 + 最近两回合正文」逐项检查，纠正演化时漏掉/更新错的数据。这个面板让你看它做了什么。

### 📋 任务
**任务 / 世界大事 / 总结**面板 📋。显示你的主线/支线任务、世界发生的大事件、分段剧情总结，以及「双时间」等世界级杂项信息。

### 🧠 记忆
长期剧情**记忆中枢** 🧠。这里其实涉及三套配合的机制：
- **叙事记忆**（设置里开）：用关键词从历史剧情里召回「相关」的片段注入正文，让 AI 记得很久以前的事（无需向量，轻量）。
- **生平压缩**：每个角色的经历攒多了会自动让 AI 压缩成「长期事实」，防止遗忘。
- **结构化档案召回**（默认开）：每回合自动把主角 + 在场/相关 NPC 的完整档案塞给写正文的 AI，保证它不会「失忆」写错你的数据。

### 向量资料库（语义记忆）
进阶记忆方案（设置 → 向量记忆）：用**语义向量**检索长期记忆，比关键词更聪明（需要一个 embedding 接口）。还内置了「原著小说 + 世界书」的向量库——每回合按语义相关度把原著片段/世界书条目注入，让剧情更贴合原作。**开启后接管召回。**

### 🖼 生图
给游戏配图 🖼（设置 → 生图设置）。支持多种服务（NAI / OpenAI / Gemini / ComfyUI / 自定义），分三条独立线：**角色肖像、装备图、正文插图**，可各自选服务和自动开关。开了自动后，外观变化会自动重绘肖像、正文关键处会自动插图。图片存在浏览器的 IndexedDB 里（不占存档体积）。

### 🔊 语音（TTS）
**朗读**功能 🔊。支持本地 Web Speech 或 Edge-TTS 网关（免 Key 的微软神经语音）。可以给每个 NPC 分配不同音色、旁白和台词分离朗读，正文里还有行内 🔊 按钮逐句念。

### 🎨 界面外观美化
（设置 → 界面外观美化）**护眼 & 换肤** 🎨：8 套主题皮肤、护眼色调、暗角、正文字体（含「霞鹜文楷」）、字号行距调节。看久了眼睛累就来这调。

### 多语言
游戏支持**简体 / 繁体 / 英文 / 越南语**切换（设置内）。这是运行时翻译层，正文本身不会被翻译。

## 11. 联机与社区

> 这一类大多需要联网服务支持，是**可选的社交扩展**。单机玩不碰它们也完全 OK。

### 🌐 联机
**在线组队联机** 🌐：房主权威 + 中继服务器同步回合。队友零门槛加入（不用自己的 Key），可以观战战斗、控制角色、共享联机存档。

### 💬 聊天室
一个**独立的全局实时聊天室** 💬（真人玩家）。可以分享你的技能/天赋/装备/NPC 卡，支持 Discord 登录、未读红点、自定义头像和气泡皮肤。

### 🛒 交易行
**全局交易行** 🛒：挂牌卖东西、公开还价看板。看板型（不自动成交，避免丢物），带托管机制。

### 🏪 产业
你可以经营的**产业** 🏪（商店 / 娼馆 / 铁匠铺），供其他玩家来消费。有独立店面页、AI 生成的货品、立绘轮播、双币记账、服务端收入账本和待领横幅。

### 🏰 公会
**异步玩家公会** 🏰（原「家族」）。创建/加入公会、名册军衔、捐献换福利、公会仓库、编年史、周任务、贡献榜、公会战、据点建设等。账号级、跨存档。

### 🆘 助战
**助战大厅** 🆘：把你的主角或 NPC 上传成公开卡，别人可以邀请你的卡物化成他的临时队友（反之亦然）。有排行榜。

### 🏆 世界竞技场
**上传参赛卡的在线竞技场** 🏆：服务端权威裁判（按战力 + 随机种子的纯函数判定），自动战斗，占位取代排名。每账号可放 3 张卡。（和 6 节乐园设施里的单机「竞技场」不同，这个是**联机**的。）

### ⏱ 游玩时长
统计你的**在线游玩时长** ⏱并上榜（凡 Discord 登录者）。前端心跳累计「页面活跃时长」，服务端防刷。

### 🪦 纪念丰碑
**跨存档的英灵殿** 🪦：把你过往的主角铭刻成「英灵卡」，之后能召唤入队。跨存档、云同步。

### 🏦 账户仓库
**跨存档的保险箱** 🏦（Discord 云）：把物品存进去，换个存档也能取出来。
> ⚠️ 存取时会带**完整物品快照**，防止「塞进去再拿出来词缀全没了」。

### 🧩 创意工坊
**社区共享中心** 🧩：上传/下载别人分享的技能树、体系、看板娘、世界卡等 10 类内容。

## 12. 存档与数据管理

### 💾 存档
**多存档系统** 💾（存在浏览器的 IndexedDB 里）。一个存档 = 你全部数据 + 对话历史 + 图片的完整快照。
- 支持**存档 / 读档 / 重命名 / 删除 / 导出 / 导入**。
- **读档靠刷新页面**恢复（这是正常机制，别慌）。
- 有**自动存档**（每回合后延时约 20 秒自动覆盖存一份）。
- 「**新游戏**」会清空进度但保留你的配置（接口、预设等）。
- 存档管理里还能 **📖 导出小说**：把你的对话历史清洗成分章的小说 TXT。

### ⚙ 变量管理（演化功能启动台）
（设置 → 变量管理）这是所有「**演化功能的中心启动台**」🧬。物品、主角、NPC、势力、领地、冒险团、万族、杂项、记忆、生图、装备强化……每一套演化系统的开关、频率、预设、专用接口，都在这里分门别类地管理。**新手默认不用动**——演化开箱即用；等你想精调「多久更新一次 NPC」「给物品演化换个便宜模型」时再进来。

### 全局配置导出 / 导入
在变量管理页最下方的「**配置备份 · 迁移**」栏，可以一键**打包导出**你所有的配置（接口、预设、世界书、正则、生图、向量、模板……**不含游戏进度**），换设备时一键导入。适合备份和搬家。

## 13. 新手推荐流程 & 常见问题

### 推荐上手流程

1. **配接口**（第 2 章）：接口库加 1~2 条能用的接口 → 测试 ✅ → 在「正文生成」接口路由里挂上。
2. **建角色**（第 3 章）：选**简单**难度 → 分好六维 → 写个天赋 → 确认。
3. **玩几个回合**（第 4 章）：随便输入行动，熟悉「你演、AI 写、系统记账」的节奏。看看左栏数据怎么变、背包怎么多出东西。
4. **逛一圈功能**：打开 📇NPC、🎒背包、✨技能、📋任务 看看系统都帮你记了些什么。
5. **想强化 / 娱乐**：回主神空间，进 🎡乐园设施 玩强化、赌场、竞技场。
6. **想要图 / 语音**：去设置开 🖼生图、🔊语音。
7. 觉得 AI「记性差」或数据老出错：去设置开 **🧠叙事记忆**、检查 **🧾审计**，或用 **♻重算变量** 修。

### 常见问题（FAQ）

**Q：点了发送，一直转圈 / 报错生成不出来？**
A：99% 是接口问题。回**设置 → 综合设置 → API 接口库**，点你的接口「🔌 测试连接」看是否 ✅。红字提示会告诉你是地址错、Key 失效还是被限制。也确认「正文生成」的接口路由里挂的接口是「可用」状态（已启用 + 有地址 + 有 Key）。

**Q：数据没更新 / 更新错了（比如砍死了敌人血没扣、捡的东西没进包）？**
A：用输入框旁的 **♻ 重算变量**，选对应的那一项（物品 / 主角 / NPC…）单独重跑一次。注意这功能刷新页面后就失效，要趁本回合用。也可以去 **🧾审计** 看系统自己纠了哪些。

**Q：AI 忘记很久以前的剧情 / 记错我的设定？**
A：开 **设置 → 叙事记忆**（关键词召回长期记忆）。想更强可开 **向量记忆**。另外「结构化档案召回」默认就开着，会保证 AI 记得你**当前**的数据。

**Q：花了 5000 却记成花了 10000 之类的数值错乱？**
A：本作有「场外操作自动通报」和「前置提示词」机制来缓解。实在不对就用 ♻ 重算变量修，或在**前置提示词**里写明约束。

**Q：游戏卡顿 / 打字有延迟？**
A：可以在设置里**关掉全息卡特效**、**调低历史楼层限制**。图片多了也会拖慢——用背包/设置里的清理工具清掉孤儿图片。

**Q：想彻底重来？**
A：存档面板里「新游戏」会清进度保配置。想连配置都清空，需要清掉浏览器里以 `drpg-` 开头的本地存储（进阶操作，慎用）。

**Q：换了电脑，配置和存档怎么带走？**
A：**配置**用变量管理页底部的「全局配置导出/导入」。**存档**用存档面板的「导出/导入」。（登录 Discord 的话，云存档 / 账户仓库也能跨设备。）
