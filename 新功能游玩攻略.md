# 新功能游玩攻略（小白向 · 手把手）

> 面向完全没玩过的人。每个功能都告诉你：**在哪进 → 怎么玩 → 花什么 → 新手怎么不亏**。
> 通用前提：很多新玩法**只在主神空间（轮回乐园 / 专属房间）里开放**，在任务世界里会提示「仅主神空间可用」。先回乐园再玩。

---

## 0. 先认路：新东西都在哪

新功能的入口分两类：

- **玩法入口**：游戏界面**右侧导航栏**的图标按钮。
  - 🌳 技能树 ｜ 🕳 深渊（深渊地牢）｜ 💗 欢愉宫 ｜ 🏟 竞技场
  - 🎰 **赌场**藏在右导航「**乐园设施**」里：点乐园设施 → 里面有「🎰 赌场」「🏟 竞技场」。
- **设置/调参入口**：右导航「**设置**」→「**变量管理**」，里面一排启动台按钮：
  - 🎰 赌场（玩法/API/限红抽水/荷官）｜ 🕳 深渊地牢（API/门票/进度/重置）｜ 🌳 技能树（建树/节点编辑/模板）等。

> 一句话：**想玩**就去右导航；**想调规则/接 AI**就去 设置 → 变量管理。

## 1. 🎰 轮回赌坊（最大的新玩法）

主神空间里的赌场，**8 种玩法**。前 4 个是纯本地秒结算、不花 AI；后面几个用 AI 生成剧情/对战。

### 1.1 两个厅 + 怎么换筹码

- **普通厅**：用 **乐园币** 换 **筹码** 下注。人人能玩。
- **魂币贵宾厅**：用 **魂币（灵魂钱币）** 换 **魂筹**，限红更高、还解锁两个高级玩法（灵魂决斗场、魂赌）。**需要达到「五阶」阶位**才能进，没到会显示「贵宾厅🔒」。
- **换筹码**：顶部输入数字 → 点「**买筹码**」把钱换成筹码；点「**兑现**」把筹码换回钱。中间可能有「抽水 X%」的手续费（庄家抽成，设置里可调）。

> 顶栏右上角随时显示你的**筹码余额**；下面一行显示你还有多少**乐园币/魂币**。

### 1.2 荷官（陪聊的发牌员）

赌桌上方有个头像，是**荷官**。用 `‹ ›` 换人，点头像他会说一句话（赢了恭喜、输了拱火）。纯气氛，不影响输赢。内置玄夜（兔族）、巧姐（魂族）、老李（魔鬼族）、魔笼等。

### 1.3 下注额怎么调

中间「下注」框填数字，旁边 `+10 / +50 / +100 / +500` 快捷加注，「全押」一把梭。**限红 X~Y** 是本厅单注的最小/最大额度。

### 1.4 八个玩法逐个讲

| 玩法 | 怎么玩 | 赔率/要点 |
|------|--------|-----------|
| 🎲 **猜大小（骰宝）** | 三颗骰子，押「大(11~17)」「小(4~10)」或「豹子(三个一样)」，也可押某个单点 | 大/小 **1:1**；豹子 **30:1**（爆冷高赔） |
| 🎡 **轮回转盘** | 押红/黑、单/双、大(19-36)/小(1-18)，或押某个具体号码 | 红黑单双大小 **1:1**；押单号 **35:1**；有个绿色 0 是庄家优势 |
| 🃏 **夺命 21 点** | 跟荷官比谁的牌更接近 21、不能超。点「发牌」后用「要牌/停牌/加倍」 | 荷官停在 17；天生 21（黑杰克）赔 **1.5×**；前两张可加倍 |
| 🪜 **翻倍梯子** | 押底注后反复开盅，每级约 50% 概率翻倍，翻错血本无归 | **随时可「收手」落袋**！贪心翻到底大概率破产，见好就收 |
| 🏟 **角斗场** | AI 安排两名 1~4 阶角斗士对打，你押谁赢 | 见下方详解 |
| ⚜️ **灵魂决斗场** | 角斗场的高阶版，5~7 阶强者，**仅贵宾厅（五阶）** | 用魂筹下注 |
| 🎁 **命运福袋** | 花魂币抽奖（扭蛋） | 见下方详解 |
| 🩸 **魂赌** | 押上珍贵之物跟「魔笼」赌命，**仅贵宾厅（五阶）** | 见下方详解 |

#### 🏟 角斗场怎么玩（重点）

1. （可选）指定两边的**种族**，留空就随机。点「**⚔ 安排一场对战**」，AI 生成两名角斗士。
2. 出现两张角斗士卡，各有一个**赔率**（系统按双方战力差实算）。点卡片可看**完整档案 + 专家战略评估**（优点 / 弱点 / 锐评），帮你判断押谁。
3. 选一边点「**押 X · 开战**」。然后是**逐回合动画战报**：双方血条、buff、技能进 CD、物品消耗一回合一回合演给你看，还有角斗士的心理活动。
4. 嫌慢就点「⏩」加速或「跳到结果」。结束后弹**结算**：押中按赔率赔付，押错没收注金。

> 小白提示：**先点开档案看专家锐评再下注**。赔率越高的越可能是弱势方，赌冷门回报高但风险大。

#### 🎁 命运福袋（扭蛋）怎么玩

- 花 **魂币**，「单抽」或「十连」。可能开出**装备 / 宝石 / 材料 / 技能书 / 乐园币 / 魂币**，奖励直接进背包和钱包。
- 稀有度从低到高：**普通 → 精良 → 稀有 → 史诗 → 传说**。
- **两条保底**：① 顶部进度条 **60 抽内必出史诗+**（出了就清零重新算）；② **十连保证至少一个稀有**。
- 想要史诗/传说就攒着十连抽，性价比比单抽稳。

#### 🩸 魂赌（高风险剧情局）

贵宾厅深处，古老赌笼「魔笼」主持。押上 **魂币 / 本命装备 / 自身天资**，AI 把这一局演成一段宿命剧情：

- **赢**：厚报。**输**：真损——魂币被没收 / 装备直接销毁 / 六维属性受损。
- 每种押注有固定胜率，UI 会写「胜率 X%」。**这是真会亏装备和属性的玩法，慎入。**

### 1.5 输光了怎么办（破产保护）

普通筹码和魂筹都归零、又没在玩梯子时，底部会出现「💸 **领取破产保护**」按钮，白送你一笔普通筹码，不至于卡死。

### 1.6 战绩

面板下方有「战绩」（总局数 / 累计赢输 / 最大单局 / 连胜）和「最近流水」。打出好成绩还会解锁称号成就。

## 2. 🌳 职业技能树 / 星图

右导航 **🌳 技能树**。一张星图，花「**潜能点**」点亮节点，给主角加六维属性、灌技能和天赋。**目前只对主角（B1）开放。**

### 2.1 潜能点（PP）从哪来

- **出生送 4 点，每升 1 级 +4 点**（纯确定性，不靠 AI 施舍）。
- **乐园币兑换**：面板右上「⇄ 兑换潜能点」。**越买越贵**（每买一点下一点 ×1.25，且阶位越高基础价越贵），防止前期囤点。
- 世界结算 / 任务也可能额外奖励潜能点。

右上角实时显示「可用潜能」和明细（升级得多少·额外多少·已用多少）。

### 2.2 怎么点

1. 顶部「**职业树**」下拉选一棵（会自动匹配你的职业，内置「剑士」星图）。
2. 在画布上**点节点**看详情，点「**解锁 / ＋点一次**」消耗潜能点。**滚轮缩放、拖空白处平移。**
3. **两类节点**：
   - **小节点（星点）**：每个能点 **3 次（3 个豆子）**，每点给一份**六维属性加成**。
   - **大节点（核心 / 终极）**：点亮直接**灌一个技能或天赋**进你的技能栏（不花 AI）。
4. **解锁条件**：要先点亮**前置节点**，有的还要求**阶位**或**累计已投点数**达标。条件没满足会用黄字提示。

### 2.3 进阶玩法（会花 AI / 计费）

- **⬆ 升级技能**：大节点点亮后再投点（第 2、3 豆），会**调用 AI 把那个技能/天赋大幅强化**（数值跃升 + 新增效果）。有**二次确认**防手滑烧 token。
- **✦ 星座觉醒**：点亮一组节点拼成「星座」图案后，顶部会亮「★ 星座名」，点「✨觉醒」让 AI 炼一个**质变级觉醒奖励**。
- **💠 星核镶嵌位**：某些节点是镶嵌位，把**背包里一件物品**炼成「星核」（半径内每颗已点亮的微星都吃额外加成），还会生成一条带技能的「脉络链」。计费。
- **↺ 洗点**：只退**小节点**的点数（每点花 200 乐园币），**大节点（技能/天赋）不可洗**、永久保留。
- **传承·提前解锁**：如果主角已经从别处学到了某条分支的**终极技能/天赋**，那条线会**提前全解锁**，且每个节点只要 **1 点**——白嫖捷径。

### 2.4 建自己的树

设置 → 变量管理 → 🌳 技能树（编辑器）：可创建/编辑节点、连线、按层套阶位、导入分享别人的 `.tree.json` 模板。

> 小白建议：先沿一条主分支往下点小节点堆属性，凑齐前置就拿大节点的技能；潜能点早期很紧，别乱兑换乐园币（很贵），多靠升级自然攒。

## 3. 🕳 深渊地牢（堕落流 Roguelike）

右导航 **🕳 深渊**。主神空间限定的**多层闯关**：一层层往下打，越打越强，死了或通关就结束、下次重来（Roguelike）。**进度总览在玩法面板顶部**。

### 3.1 五大「险地」（从浅到深）

| 顺序 | 险地 | 区主（Boss） |
|---|------|------|
| 1 | **黑渊** | 黑雾母体·腐渊之核 |
| 2 | **古战场** | 枯骨统帅·无尽亡灵将军 |
| 3 | **渊龙底** | 极寒渊龙先祖 |
| 4 | **死亡屋** | 黑暗神教主教·死亡屋主 |
| 5 | **界之底** | 深渊先王眷属 |

越深的险地怪越强（阶位、血量、攻击全面上涨），掉落也越好。

### 3.2 开局选项

进面板后开局可选：

- **起始险地**：从第几个险地开打。
- **极限单人**：硬核模式，只带主角一个人。
- **带随从**：把**当前在场的契约者 NPC**（非组队的）拉进来当队友一起打。
- **无尽**：通关后继续无限往下刷，挑战极限。
- **门票**：每局花一笔乐园币（设置里可调）。

### 3.3 房间类型（地图上的格子）

🚪 入口 ｜ ⚔ 战斗 ｜ 💀 精英 ｜ 👑 区主 ｜ ❓ 事件 ｜ 🔥 休整 ｜ 💎 宝藏 ｜ 🌀 信标 ｜ 🩸 原罪。一路选格子前进，战斗自动结算（精英/区主的六维和技能由 AI 现场生成）。

### 3.4 三层「变强」系统（核心乐趣）

1. **战后加成卡（本局内，3 选 1）**：每场战斗后弹三张卡选一张，分 **6 个流派**：
   - **腐蚀** · **武技** · **守护** · **亡灵** · **领域** · **赌徒**
   - 例：`渊怒`攻击+12%、`战狂`攻+30% 防-10%、`孤注`攻+45% 命-10%、`不动壁垒`防+35%……越赌的卡收益越高但有代价。卡只在**本局**有效。
2. **原罪物 / 原罪武器**：地牢里随机掉的特殊装备，数值由前端定骨架、描述由 AI 配文，本局内变强用。
3. **堕落星图（永久 meta 层）**：跨局攒的永久强化（如「深渊烙印」生命上限+5%），**死了也不丢**，越玩底子越厚。

**腐蚀值**是堕落流的主资源轴——很多最猛的加成都要你「沾染更深的腐蚀」来换力量。

### 3.5 死亡 / 通关

死亡会保留一部分收益（约 50%），不是一无所有；通关给结算。设置 → 变量管理 → 🕳 深渊地牢 可调门票、难度、查看/重置进度。

> 小白建议：第一次从**黑渊**起步、**带 1~2 个在场 NPC**当队友、别开极限。加成卡早期优先拿「**回血 / 生命上限 / 防御**」活下来，后期再赌「孤注」这类高攻卡。

## 4. ⚙️ 升级方式变了：世界结算（重要！）

**旧的「进阶点数」「击杀升级」已经全部取消了。** 现在靠「**世界结算**」升级：

1. 在 **🌍 选择世界**的界面，找到「**📊 结算任务**」按钮。
2. 点它 → 会往聊天输入框塞一个 `【结算任务】`。**你手动发送这条**。
3. 正文 AI 会判定你这个世界的**主线完成度**，并按「**世界之源 %**」给你**评级（E ~ SSS 八档）**，然后发奖：乐园币、属性点、技能点 / 黄金技能点、潜能点、升 1~3 级、灵魂结晶等。
4. 结果会渲染成一张**金边结算卡**，奖励真正入账。

> 货币也扩成了 4 种：**乐园币 / 魂币 / 技能点 / 黄金技能点**（技能点是给技能升级用的钱，别和潜能点搞混）。
> 一句话：**想升级 = 把一个世界的主线推到一定程度 → 点「📊 结算任务」发送**。

## 5. 💗 欢愉宫世界书（成人向，可选）

欢愉宫（💗）新增了**专用世界书**系统，让私密互动更贴合你想要的风格。

- **管理**：欢愉宫里的世界书管理（JoyManager）可**导入 / 开关 / 编辑 / 导出**。内置 5 本（NSFW 写作指导、BDSM、姿势、淫语、服装肉体描写等）。
- **怎么生效**：常驻类一直挂着；关键词类在你**发消息时**按词触发注入，影响 AI 的描写。
- **包间快捷键**：进包间后有 **🤸 姿势 / ⛓ BDSM** 快捷按钮，点一下把对应标题填进输入框，**由你发送**触发那段玩法。

## 6. 🎨 血条换皮 + 20 款皮肤（纯装饰）

给主角的 HP/EP 条「换皮」，**只是好看，不改任何数值**。

- 在主角**侧栏**，**点血条本身**、或点血条下方「**✎ 自定义血条**」展开面板。
- **改称呼**：把「生命 HP」改成任意名字（玩吸血鬼可叫「血池」、EP 叫「血怒」）。
- **选皮肤**：20 款血条外观（经典 / 霓虹 / 熔岩 / 冰晶 / 赛博 / 翠盟 / 魔渊 / 街机 / 魂界…），点格即换。
- 血量低于 20% 时血条会**变红急促闪烁**告警（任何皮肤都会）。

> 这些设置随存档保存。改了名字只影响显示和 AI 的叙述措辞，战斗/数值照常用真实 HP/EP。

## 7. 🤝 冒险团：加入契约者 NPC 的团

以前只能自己当团长，现在**可以加入别的契约者 NPC 的冒险团**（你不是团长）。

1. 打开某个**契约者 NPC** 的详情 → 「💬 对话」进私聊。
2. **真诚、明确地表达**你想加入「他所属的那个冒险团」。
3. NPC 会**按自己的人设、立场、对你的好感**来决定。同意时他会发出 `<加入冒险团>` 信号。
4. 弹窗确认后，系统**全量生成**那支团的信息，你就入团了。

> 前提：那个 NPC 得**本身隶属某支冒险团**（详情页「隶属冒险团」字段有内容）。

## 8. 🎲 NPC 机械生成六维（治 AI 乱填属性）

NPC 详情页有个「**🎲 机械生成六维**」按钮。以前 NPC 的力/敏/体/智/魅/运经常被 AI 拍脑袋乱给，现在改成：**AI 只给几个描述性元字段，前端用公式反推出合理六维**，稳定不漂。

- 生成后会双显「**资质档**」（基础底子，稳定）和「**战力档**」（当前有效战力，会越阶浮动）。
- 是个**校准工具**，觉得某个 NPC 属性不对劲点一下重算即可。

## 9. ⚔️ 战斗系统增强（打架更有花样）

外置「⚔️ 战斗」按钮的回合制战斗，这次加了更多机制（结果照旧写进输入框由你发送）：

- **状态**：buff / debuff、持续伤害(DoT) / 持续回血(HoT)、护盾、技能冷却。
- **群体**：群攻(AoE)、控制（眩晕等）、手动控队。
- **大招**：蓄力大招（攒几回合放大招，法宝类重型技能）。
- **场地**：领域 / 阵法（关键词触发，展开后每回合笼罩全场）。
- **保命**：不死（保底 1 血）/ 锁血（限时完全不掉血），靠关键词触发。

> 这些在普通战斗、竞技场、深渊地牢、角斗场里都会体现。

## 🚀 给新手的上手顺序

1. **先把主线推一推，用「📊 结算任务」升级**——这是现在唯一的升级途径，先把等级和资源搞起来。
2. **点技能树**：升级攒的潜能点拿去 🌳 点节点，先堆属性再拿技能。
3. **回乐园玩赌坊**：缺钱去 🎰 玩骰子/转盘小赌怡情，攒够魂币抽 🎁 福袋出装备；**翻倍梯子记得见好就收**。
4. **攒到五阶**解锁贵宾厅，再碰灵魂决斗场 / 魂赌这些高风险玩法。
